# 17.16.9 使用错误对话框放松约束





有时，如果种子被过度约束，网格生成会失败。 当Abaqus/CAE由于过度约束的种子而无法创建网格时，会出现以下对话框：

![](../graphics/mgn-error-nls.png)

此外，过度约束的种子会在视口中高亮显示。 您可以选择以下选项之一：
- 单击**Yes**放松种子约束并继续对区域进行网格划分。
- 单击**No**保存种子约束并取消网格划分过程。
