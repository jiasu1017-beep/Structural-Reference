# 67.1 Abaqus/CAE 中自由体切割上的合力和力矩







Abaqus/CAE 中的自由体横截面是模型中您想要显示合力和力矩的区域。定义横截面后，Abaqus/CAE 会显示向量，以显示您所选区域上的合力和力矩的大小和方向。力向量以单个箭头显示，力矩向量以双箭头显示，如[图 67-1](pt06ch67s01.md#usi-fbd-intro-example) 中的示例所示。

**图 67-1** 在机身模型上显示的合力和力矩。

![](../graphics/rnb68-out-freebody-nls.png)

您可以使用与显示组定义非常相似的过程来定义构成横截面的节点和单元。Abaqus/CAE 使您能够通过包含曲面、显示组以及按编号的单元或节点来指定横截面的分量；并且您可以从视口中按特征角度或单独选择项目进行选择。参见["创建或编辑自由体切割，" 第 67.2 节"](pt06ch67s02.md)获取详细说明。定义自由体横截面的物理分量后，您可以设置求和点（围绕该点计算合力矩）的位置，并且可以指示在以分量形式显示向量时应用的坐标系变换。

横截面只能沿网格边界创建；您不能沿任意平面指定横截面。您只能在可视化模块中创建和显示自由体切割。
