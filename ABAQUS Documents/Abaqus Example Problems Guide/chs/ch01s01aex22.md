# 1.1.22 油井筒中的材料侵蚀（出砂）

**产品：** Abaqus/Standard

本例展示了在Abaqus/Standard中使用自适应网格划分和自适应网格约束来建模油提取过程中砂生产等大规模材料侵蚀的方法。在Abaqus/Standard中，外部表面的材料侵蚀通过将表面声明为自适应网格域的一部分，并规定退入材料中的表面网格运动来建模。然后Abaqus/Standard将使用相同的网格拓扑对自适应网格域进行重新网格划分，但考虑表面的新位置。所有材料点和节点量都将被输运到它们的新位置。本例还展示了在需要新网格拓扑以继续超过某一阶段的分析时使用网格到网格解映射的方法。

### 问题描述

优化油井生产价值的过程很复杂，但可以简化为油回收率（以油的体积流量衡量）、回收可持续性（以最终回收的油量衡量）和油井运营成本之间的平衡。在实践中，实现这种平衡需要考虑井筒中岩石的侵蚀，这种现象发生在以足够高的压力梯度提取油时。这种侵蚀现象通常被称为"出砂"。根据流速，砂可能会积累在井中，影响回收的可持续性，或者可能随油被带到地表。油中的砂会导致管道系统及其组件（如节流阀和弯头）的侵蚀，增加油井运营成本。因此，过度出砂是不可取的，会限制油回收率。对这些回收率限制的典型衡量标准是所谓的"无砂速率"。无砂速率可能基于出砂造成的直接损害，或基于限制砂损害的砂处理系统的成本。前者称为最大无砂速率（MSR），后者称为可接受无砂速率（ASR）。ASR概念的实现得益于许多商业砂处理系统的可用性以及管道组件的新设计。ASR概念也产生了预测出砂速率的需求，以正确选择和确定砂处理系统和管道组件的尺寸。在本例中，我们专注于可以从Abaqus获得用于预测这些速率的方法。

### 几何与模型

油井的几何形状有两个主要组成部分。第一部分是通过岩石钻出的井筒。第二部分是一系列垂直于井筒轴线的射孔隧道。这些隧道由炸药成形弹形成，有效增加了用于油提取的井筒表面积。射孔隧道通常以螺旋方式围绕井筒展开，在垂直间距和方位角上均匀偏移。

#### 三维模型

三维例题中考虑的问题域是一块203毫米（8英寸）厚的含油岩石圆切片，建模了井筒和射孔隧道。域的直径为10.2米（400英寸）。射孔隧道从井筒径向发出，间隔90度。每个射孔隧道直径为43.2毫米（1.7英寸），长度为508毫米（20英寸）。井筒半径为158.8毫米（6.25英寸）。由于对称性，仅建模包含一个射孔隧道的域的四分之一。图1.1.22-1显示了有限元模型。岩石用C3D8P单元建模，井套管用M3D4单元建模。

#### 平面模型

平面模型是三维模型的简化版本，其中忽略了射孔隧道和井筒套管。岩石用CPE4P单元建模。模型域由长度为10.2米（400英寸）的四分之一对称方形域组成，具有半径为158.8毫米（6.25英寸）的单一井筒。图1.1.22-5显示了有限元模型。图1.1.22-6详细显示了井筒附近的区域。

### 材料

岩石选择具有硬化的线性Drucker-Prager模型，三维模型中的套管为线性弹性。

### 载荷

井筒分析的载荷序列通常包括：
- 根据上覆载荷建立地静力平衡；
- 材料移除操作的模拟，包括钻井筒和形成射孔隧道；以及
- 在井筒中施加降压以模拟抽汲。

该序列在三维和平面模型中的建模略有不同。

#### 三维模型

分析包括五个步骤。第一步进行地静力步骤，在施加初始孔隙压力、初始应力和代表射孔隧道上方土壤的分布载荷后达到平衡。第二步代表钻井操作，其中使用Abaqus中的单元移除功能移除代表井筒和射孔隧道的单元（见Abaqus Analysis User's Guide第11.2.1节"单元和接触对的移除与重新激活"）。在第三步中，更改边界条件以在射孔隧道表面施加孔隙压力。在第四步中，进行稳态土分析，其中将射孔隧道表面的孔隙压力降低到所关注的期望降压。第五步包括进行四天的土固结分析，期间发生侵蚀。

#### 平面模型

分析包括三个步骤。第一步进行地静力步骤，在施加初始孔隙压力和初始应力后达到平衡，代表井筒上的欠平衡状态。第二步代表钻井操作，其中使用单元移除功能移除代表井筒的单元，并更改边界条件以在射孔隧道表面施加孔隙压力。第三步包括进行32小时的土固结分析，期间发生侵蚀。如下面"平面模型的重新网格划分"中所讨论的，第三步被中断以重新划分模型网格。

### 侵蚀准则

井筒中有两个主要的侵蚀材料来源。一种是体积源，是由于高应力而破碎并通过孔隙中的流体输送的材料。另一种是表面源，是由于流动对表面的水动力学作用而破碎的材料。根据含油地层的特性和流速，其中一种可能是主要的侵蚀材料来源。描述井筒侵蚀的方程式的开发是一个活跃的研究领域。在本例中我们仅考虑表面侵蚀，但选择了与Papamichos和Stavropoulou用于体积侵蚀的形式相似的侵蚀方程形式。这种近似是合理的，因为高应力仅存在于围绕井筒的非常薄的层中。侵蚀方程为：

### 自适应网格域

侵蚀在每个分析的最后一步中进行建模。侵蚀方程将材料消退速度描述为解量的局部函数。Abaqus/Standard通过自适应网格划分提供了强制此表面速度、保持其相对于移动表面的法向进展以及调整表面下节点以应对大量侵蚀材料损失的功能。

侵蚀本身通过空间自适应网格约束来描述，该约束应用于射孔隧道表面上的所有节点。自适应网格约束只能应用于自适应网格域；在本例中，井筒和射孔隧道表面附近足够大范围的有限元网格被声明为自适应网格域。三维模型的自适应网格域的切割截面（包括射孔隧道）如图1.1.22-2所示。平面模型的自适应网格域是井筒附近的常规网格（参见图1.1.22-6和图1.1.22-8）。自适应网格域的识别将导致近表面网格的光顺，这对于使侵蚀能够进展到任意深度是必要的。自适应网格域边界上与常规网格相遇的所有节点必须被视为拉格朗日的，以尊重相邻的非自适应单元。

定义了速度自适应网格约束。侵蚀方程的通用性和解相关性通过在用户子程序`UMESHMOTION`中描述侵蚀方程来处理（见Abaqus Analysis User's Guide第12.2.6节"在Abaqus/Standard中定义ALE自适应网格域"）。用户子程序`UMESHMOTION`在每个网格光顺扫描期间针对给定节点调用。Abaqus/Standard网格算法为该节点计算的网格速度被传递到`UMESHMOTION`，后者修改它们以考虑在该节点计算的侵蚀速度。修改后的速度根据../graphics/exa_eqn00198.gif的方程确定，其中需要../graphics/exa_eqn00086.gif、n和../graphics/exa_eqn00199.gif的局部结果。为获得这些值，我们分别请求输出变量PEEQ、VOIDR和FLVEL的结果，注意到孔隙比与孔隙率的关系为../graphics/exa_eqn00209.gif。由于这些输出变量都在单元材料点可用，因此使用工具程序`GETVRMAVGATNODE`来获取外推到表面节点的结果（见Abaqus User Subroutines Reference Guide第2.1.8节"获取在节点处平均的材料点信息"）。

### 平面模型的重新网格划分

重新网格划分是在模型变形配置中创建新网格的过程，是解决侵蚀问题中单元畸变的有用技术。

本例中的侵蚀模型是解相关的，因为它将侵蚀速度定义为等效塑性应变PEEQ和流体速度FLVEL局部值的函数。由于这些节点定位结果值是从相邻单元的材料点结果外推的（使用工具程序`GETVRMAVGATNODE`），侵蚀模型中使用的结果质量取决于局部单元质量。实际上，随着近表面单元的变形，侵蚀模型中存在不稳定的趋势。

通过重新划分模型并提供更规则的网格来继续分析可以缓解这种不稳定性。重新划分的模型在Abaqus/CAE中创建，状态变量的传递使用Abaqus/Standard中的网格到网格解映射进行。重新网格划分发生在平面模型32小时侵蚀分析中的55000秒（15.3小时）。

#### 使用Abaqus/CAE提取二维轮廓和重新网格划分

重新划分的模型是通过从原始分析的结果数据库中提取变形岩石区域的二维轮廓来创建的。您可以通过在Abaqus/CAE主窗口底部的命令行界面中输入命令来执行此操作。要将变形几何形状作为孤单元零件从结果数据库中提取，请使用命令`PartFromOdb`，它采用以下参数：

| *name* | 要创建的孤单元零件的名称。 |
| --- | --- |
| *odb* | 从命令`openOdb`返回的结果数据库对象。 |
| *instance* | 初始模型中零件实例的名称（大写）。 |
| *shape* | 确定以未变形还是变形形状导入零件。 |

命令`PartFromOdb`返回一个零件对象，该对象被传递到命令`Part2DGeomFrom2DMesh`。此命令从先前导入的孤单元创建几何零件对象。它采用以下参数：

| *name* | 要创建的零件的名称。 |
| --- | --- |
| *part* | 从命令`PartFromOdb`返回的零件对象。 |
| *featureAngle* | 一个浮点数，指定触发几何形状中断的线段之间的角度（度）。 |

一旦创建了变形零件的轮廓，您将准备输入文件用于侵蚀分析的后续期间，如下所示：
- 重新建立与变形零件几何形状相关的所有属性。这些属性包括载荷和边界条件定义，以及集和表面定义。
- 重新划分零件网格。
- 创建单个固结步骤，完成您预期的侵蚀期。
-写出新的输入文件。

#### Abaqus/Standard中的网格到网格解映射

解映射中使用的插值技术是一个两步过程。首先，通过从积分点外推到每个单元的节点，然后对所有邻接每个节点的单元进行平均，获得旧网格节点上所有解变量的值。第二步是定位新网格中相对于旧网格的每个积分点。然后将变量从旧网格中单元的节点插值到新网格中的位置。所有解变量都以这种方式自动插值，以便可以在新网格上继续分析。每当映射模型时，由于网格的变化，解决方案中会出现一些不连续性是可以预期的。为解决这种不连续性，重新网格划分分析包括一个地静力步骤，在侵蚀过程在后续步骤继续之前重新建立平衡。

### 结果与讨论

三维和平面分析的分析场景和结果不同。

#### 三维分析

进行侵蚀的固结分析运行四天，以观察出砂的开始并预测其初始速率。图1.1.22-3显示了四天结束时的射孔隧道，可以看到最大量的材料在井筒和射孔隧道交界处被侵蚀。离井筒边界越远，侵蚀量逐渐减少。这种行为是预期的，因为在井筒和射孔隧道交界处存在高应变，而侵蚀准则仅在等效塑性应变高于阈值时才激活。

自适应域中侵蚀引起的体积变化量可以使用历史输出变量VOLC获得。实际侵蚀的固体材料量取决于岩石的孔隙率，通过将VOLC乘以../graphics/exa_eqn00210.gif获得。图1.1.22-4显示了固结步骤期间产生的砂的总体积变化（立方英寸）。随着材料固结，侵蚀速率减慢。射孔孔中的应力随时间减少并稳定。从图1.1.22-4可以得出结论，由降压、流体速度、井筒和套管几何形状以及初始射孔隧道几何形状产生的应力使得该射孔隧道将在油回收继续时以更稳定的速率产生砂；然而，该速率可能随侵蚀引起的射孔隧道的进一步变化而变化。在设计阶段，可以修改任何这些参数以限制出砂速率。许多射孔隧道从井筒发出，井筒的总出砂量将是所有射孔隧道的总和。

#### 平面分析

进行侵蚀的欠平衡固结分析运行32小时，以观察出砂的开始并预测其初始速率。图1.1.22-7显示了15小时时（第一个分析作业结束时）的井筒。基于此配置创建了新网格，如图1.1.22-8所示。该模型的最终配置，代表32小时的侵蚀，如图1.1.22-9所示。正如预期的，结果表明侵蚀继续从最大等效塑性应变位置向外扩展。图1.1.22-10显示了固结步骤期间产生的砂的总量（每英寸深度的立方英寸）。
