# 20.19.4 删除 Sketcher 对象


使用 Sketcher 工具箱中的删除工具移除 Sketcher 对象:线、圆弧、圆、圆角、样条曲线、点、约束、尺寸或参数。

**删除 Sketcher 对象的步骤:**

1. 从 Sketcher 工具箱中,选择删除工具 ![](../graphics/ico_delete.png)。有关 Sketcher 工具箱中工具的图示,请参见["Sketcher 工具," 第 20.4.1 节](pt03ch20s04s01.md)。Abaqus/CAE 会在提示区域显示提示,引导您完成操作。
2. 选择要删除的所有对象。从提示区域选择一个过滤器,将可用选择限制为**几何**、**尺寸**(包括参数)或**约束**;默认过滤器**任意**不会限制您的选择。草图和[构造几何](pt03ch20s05s02.md)都可以被删除;[参考几何](pt03ch20s05s01.md)不能被删除。**提示:**要选择多个对象,请在单击每个对象时按住 **[Shift]** 键,或围绕对象拖动一个矩形。要取消选择对象,请使用 **[Ctrl]****+单击**。更多信息,请参见[第 6 章,"在视口中选择对象"](pt01ch06.md)。
3. 完成选择对象后,单击鼠标按钮 2。Abaqus/CAE 会删除所选对象以及任何相关的尺寸或约束。**提示:**要恢复意外删除的对象,请单击 Sketcher 工具箱中的[撤销](pt03ch20hla04.md)工具 ![](../graphics/ico_undo.png)。
4. 要删除更多对象,请从第 2 步开始重复上述步骤。
5. 完成删除对象后,执行以下操作之一: - 在 Abaqus/CAE 窗口任意位置单击鼠标按钮 2。 - 选择 Sketcher 工具箱中的任何工具。 - 单击提示区域中的取消按钮 ![](../graphics/afxI_cancel.png)。
