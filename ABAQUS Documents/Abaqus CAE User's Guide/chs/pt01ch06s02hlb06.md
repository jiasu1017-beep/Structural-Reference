# 6.2.7 将相邻对象添加到选择





如果你已经选择一个或多个对象，你可以扩展你的选择以包含所有相同类型的相邻对象。添加相邻对象是使用拖动选择或角度方法（分别请参阅["拖动选择多个对象，" 第 6.2.2 节](pt01ch06s02hlb02.md)和["使用角度和特征边方法选择多个对象，" 第 6.2.3 节](pt01ch06s02hlb03.md)）来快速选择多个对象的替代方法。选择相邻对象允许你向所有方向扩展选择，而不管周围特征的形状或对象连接的角度如何。它也允许你拾取模型中的多个感兴趣区域，并同时在每个区域中扩展选择集。

要将相邻对象添加到当前选择，请在现有选中对象上单击鼠标按钮 3 并选择**添加相邻实体**。Abaqus/CAE 将你的选择扩展到所有相同类型的相邻对象，包括不包含在当前显示中的对象。如有需要，Abaqus/CAE 会将新选中的对象添加到当前显示组以使它们可见。相邻对象的定义如下（基于当前选中的实体）：
- 与一条或多条选中边共享公共顶点的边
- 与一个或多个选中顶点共享公共边的顶点
- 与一个或多个选中面共享公共顶点或边的面
- 与一个或多个选中节点共享公共边的节点
- 与选中单元共享公共单元边或节点的单元
