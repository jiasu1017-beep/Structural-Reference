# 11.21.4 添加实体放样特征





从主菜单栏选择**形状**![](../graphics/images/arrow.gif)**实体**![](../graphics/images/arrow.gif)**放样**，向当前视口中的零件添加实体放样特征。您只能向三维零件添加实体放样特征。

您通过从选定边缘创建两个或多个截面并定义一个或多个放样路径来添加实体放样特征。

您可以允许Abaqus/CAE使用平滑路径定义单个放样路径，以连接每个放样截面的中心。如果您允许Abaqus/CAE定义路径，可以对放样的起始和结束截面应用切线方法。或者，可以通过选择连接每个放样截面上一点到下一放样截面对应点的曲线来定义一个或多个放样路径。

**注意：**添加放样特征时，不使用草图编辑器。因此，定义放样截面和放样路径的所有边缘必须在创建放样之前存在于零件几何中。

**添加实体放样特征：**

1. 从主菜单栏，选择**形状**![](../graphics/images/arrow.gif)**实体**![](../graphics/images/arrow.gif)**放样**。
2. 通过从视口中的零件选择边缘来创建放样截面。
3. 切换**保持内部边界**以保持放样实体特征与现有零件之间生成的任何面或边。
4. 完成创建放样截面后，单击**编辑放样**对话框中的**过渡**选项卡。
5. 执行以下操作之一：单击**选择路径**或**指定切线**。
6. 单击**预览**按钮。Abaqus/CAE显示将使用当前设置创建的放样的线框表示。
7. 单击**完成**创建放样并关闭**编辑放样**对话框。
