# 3.4.2 什么是步骤相关管理器？







与[「什么是基本管理器？，第 3.4.1 节」](pt01ch03s04s01.md) 中描述的基本管理器一样，步骤相关管理器包含您创建的所有某种类型对象的列表，以及可用于操作现有对象和创建新对象的**创建**、**编辑**、**复制**、**重命名**和**删除**按钮。

但是，步骤相关管理器中出现的对象类型是您可以创建的对象类型，在某些情况下，这些对象可以在特定分析步骤中进行修改、抑制和停用。因此，与基本管理器不同，步骤相关管理器包含有关列表中每个对象历史的附加信息。步骤相关管理器显示这些对象在 Abaqus 分析过程中如何从一个步骤传播到另一个步骤。（有关步骤和多步骤分析的信息，请参阅[「定义分析，Abaqus Analysis User's Guide 第 6.1.2 节」](../usb/usb-link.md#usb-anl-aover)。）

Abaqus/CAE 中存在以下步骤相关管理器：

**在 Load 模块中：**

- **载荷管理器**
- **边界条件管理器**
- **预定义场管理器**

**在 Interaction 模块中：**

- **相互作用管理器**

**在 Step 模块中：**

- **场输出请求管理器**
- **历史输出请求管理器**
- **自适应网格约束管理器**

例如，[图 3-16](pt01ch03s04s02.md#lbi-loadmanager2) 显示了**载荷管理器**。

**图 3-16** **载荷管理器**。

![](../graphics/lbi-loadmanager-nls.png)

此管理器沿对话框左侧显示现有载荷的字母顺序列表。分析中所有步骤的名称按执行顺序显示在对话框顶部。这两个列表形成的表格显示了每个步骤中每个载荷的状态。（有关创建和删除步骤的信息，请参阅[第 14 章，「Step 模块」](pt03ch14.md)。）

如果您点击表格中的某个单元格，该单元格会高亮显示，并且在管理器底部的图例中会出现与该单元格相关的以下信息：
- 该列中步骤执行的解析过程类型。
- 该行中步骤相关对象的信息。
- 该步骤中步骤相关对象的状态（与表格单元格中显示的信息相同，在某些情况下更详细）。

您可以使用管理器左侧列中的图标来抑制对象或恢复之前被抑制的对象以进行分析。有关更多信息，请参阅[「抑制和恢复对象，第 3.4.3 节」](pt01ch03s04s03.md)。

管理器右侧的按钮允许您操作所选步骤中的对象。例如，如果您在上面显示的载荷管理器中点击**编辑**，则会出现一个编辑器，您可以在其中修改 `Step-1` 中名为 `Force` 的载荷。其他按钮（**向左移动**、**向右移动**、**激活**和**停用**）允许您更改对象在特定步骤中的状态。

**注意：****激活**和**停用**按钮在**预定义场管理器**中不可用。

有关更多信息，请参阅[「修改步骤相关对象的历史，第 3.4.6 节」](pt01ch03s04s06.md)；[「更改步骤中对象的状态，第 3.4.12 节」](pt01ch03s04hlb04.md)；以及[「编辑步骤相关对象，第 3.4.13 节」](pt01ch03s04hlb05.md)。

您可以通过将列标题之间的分隔线向左或向右拖动来调整表格列的大小。您也可以通过拖动框的侧面来增加对话框的大小。如果分析包含许多步骤或许多步骤相关对象，增加对话框的大小允许您查看更多行和列，而不必使用滚动条。
