# 10.7.12 从输出数据库导入零件





从主菜单栏选择**文件**![](../graphics/images/arrow.gif)**导入**![](../graphics/images/arrow.gif)**零件**，可以导入以网格零件形式存储在输出数据库中的零件实例。如果输出数据库包含多个零件实例，您可以选择要导入的零件实例。Abaqus/CAE将每个零件实例作为单独的零件导入。您可以导入未变形形状或变形形状。如果导入变形形状，可以指定要导入的步骤和帧。

要验证网格质量，您可以在网格模块中显示零件，并从主菜单栏选择**网格**![](../graphics/images/arrow.gif)**验证**。此外，您可以使用网格模块更改分配给网格的元素类型，并编辑原始网格定义。更多信息，请参见"我可以使用编辑网格工具集做什么？"，第64.1节，以及"分配Abaqus元素类型"，第17.5节。

**从输出数据库导入零件：**

1. 从主菜单栏，选择**文件**![](../graphics/images/arrow.gif)**导入**![](../graphics/images/arrow.gif)**零件**。出现**导入零件**对话框。
2. 从**导入零件**对话框底部的**文件过滤器**菜单中，选择**输出数据库 (*.odb)**。Abaqus/CAE列出所选目录中所有具有`.odb`文件扩展名的文件。
3. 选择包含要导入的零件的输出数据库，然后单击**确定**。出现**从输出数据库创建零件**对话框。该对话框列出输出数据库中的每个零件实例及其类型（可变形体或离散刚体表面）。
4. 从对话框中，选择要导入的实例。
5. 如果只选择了一个零件实例，Abaqus/CAE使用该实例的名称命名结果零件，尽管您可以更改名称。相反，如果选择了多个零件实例进行导入，Abaqus/CAE使用每个实例的名称命名每个零件，您不能更改它们的名称。Abaqus/CAE确定零件实例的建模空间（三维、二维或轴对称）。您不能更改建模空间或类型。
6. 默认情况下，Abaqus/CAE导入零件的未变形配置。要导入变形零件，请单击**导入变形配置**。从输出数据库中可用的步骤和帧中选择包含变形形状的步骤和帧。
7. 单击**确定**从输出数据库导入孤儿网格并关闭对话框。
8. 如果您输入的名称与模型中现有零件的名称相同，Abaqus/CAE会询问您是否要覆盖现有零件或替换网格。如果选择替换网格，Abaqus/CAE会用导入的孤儿网格的节点和元素替换现有零件的节点和元素。引用原始零件的集合和截面分配将保持不变。但是，由于集合和截面分配引用节点和元素编号，因此导入零件的网格应与原始零件的网格相似。例如，您可以用变形网格替换未变形网格。Abaqus/CAE进入零件模块，导入的零件替换当前视口的内容，零件出现在上下文栏中模型的零件列表中。
