# 19.9.4 在数据检查后继续自适应过程







当您对自适应过程执行数据检查时，会为自适应过程的第一次迭代创建一个作业（如["对自适应过程执行数据检查，"第 19.9.2 节](pt03ch19s09hlb02.md)中所述）。此外，Abaqus 创建并保存所有必要的文件，以便稍后完成此作业。您也可以使用这些文件从第一次迭代的当前状态继续整个自适应过程。

要仅完成与自适应过程第一次迭代关联的作业，请使用["在数据检查后继续分析作业，"第 19.7.5 节](pt03ch19s07hlb05.md)中列出的过程。不会发生重新网格划分和进一步迭代。

要继续整个自适应过程，请选择****自适应**![](../graphics/images/arrow.gif)**继续分析**![](../graphics/images/arrow.gif)***自适应过程名称*****。您也可以通过在**自适应过程管理器**中选择过程名称，然后点击**继续分析**来继续自适应过程。自适应过程按照["提交自适应过程，"第 19.9.3 节](pt03ch19s09hlb03.md)中所述进行。

**继续分析**选项只能与数据检查结合使用；它不用于在自适应过程完成后继续重新网格划分。一旦自适应过程运行完成，您可以通过重新提交完整过程来继续对模型进行重新网格划分。您还可以考虑使用["使用自动和手动网格自适应的组合，"第 19.3.4 节](pt03ch19s03s04.md)中描述的组合重新网格划分技术。
