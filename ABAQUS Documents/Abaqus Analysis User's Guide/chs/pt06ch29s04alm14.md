**产品：** Abaqus/Standard

### 概述

框架截面行为：
- 需要定义截面的形状及其材料响应；
- 在框架单元内部使用线弹性行为；
- 可以在单元端部包含"集中"塑性以模拟塑性铰的形成；
- 可以是单轴的，响应由现象学屈曲支柱模型以及线弹性和拉伸塑性屈服控制；
- 仅适用于管道截面，可在分析过程中切换到屈曲支柱响应。

### 定义弹塑性截面行为

要包括弹塑性响应，请直接将*N*、![](../graphics/usb_eqn04588.gif)、![](../graphics/usb_eqn04589.gif)和*T*指定为其共轭塑性变形变量的函数，或使用基于材料屈服应力的*N*、![](../graphics/usb_eqn04588.gif)、![](../graphics/usb_eqn04589.gif)和*T*的默认塑性响应。Abaqus/Standard使用指定值或默认值来定义非线性运动硬化模型，该模型"集中"在单元端部的塑性铰中。由于塑性集中在单元端部，铰没有关联长度。广义力与广义塑性位移相关，而非应变。实际上，塑性铰将具有由结构构件长度和载荷决定的有效尺寸，这将影响硬化速率，但不影响极限载荷。例如，纯弯曲下的屈服（构件上恒定的弯矩）将产生等于构件长度的铰长度，而具有横向尖端载荷的悬臂梁（构件上线性变化的弯矩）的屈服将产生更为局部的铰。因此，如果特定载荷下的硬化速率和由此产生的塑性变形很重要，您应该对不同长度和不同载荷情况适当校准塑性响应。

在塑性范围内，唯一可用的塑性面是椭球面。此屈服面仅对管道横截面相当准确。箱形、圆形、I形和矩形横截面可由您自行决定使用，但须理解椭圆形屈服面可能无法准确近似弹塑性响应。通用横截面类型不能与塑性一起使用。

### 定义可选的单轴支柱行为

框架单元可选地允许仅单轴响应（支柱行为）。在这种情况下，单元的两端都不支撑垂直于轴线的弯矩或力；因此，只存在沿单元轴线的力。此外，此轴向力沿单元长度为常数，即使切向分布载荷作用于单元轴线。单元的单轴响应是线弹性的或非线性的，在后者情况下包括压缩时的屈曲和后屈曲以及拉伸时的各向同性硬化塑性。

#### 定义线弹性单轴行为

线弹性单轴框架单元的行为类似于具有恒定刚度![](../graphics/usb_eqn04649.gif)的轴向弹簧，其中*E*是杨氏模量，*A*是横截面积，*L*是原始单元长度。应变度量是单元长度变化量除以单元原始长度。

### 在分析过程中切换到可选的单轴支柱行为

框架单元允许在分析过程中切换到单轴屈曲支柱响应。切换的标准是"ISO"方程和"强度"方程（见["框架单元的屈曲支柱响应，" Abaqus理论指南第3.9.3节](../stm/stm-link.md#stm-elm-strut)）。当满足ISO方程时，弹性或弹塑性框架单元会一次性切换到屈曲支柱响应。强度方程的引入是为了在没有显著轴向力的情况下防止切换。

当框架单元切换到屈曲支柱响应时，会发生显著的結構剛度损失。切换后的单元不再支撑弯曲、扭转或剪切载荷。如果切换导致全局结构不稳定（即结构在施加载荷下会坍塌），分析可能无法收敛。

要允许单元响应切换，请使用默认屈曲包络线或定义屈曲包络线并提供屈服应力，但不要为框架单元激活线弹性单轴行为。

ISO方程是基于细长管道状（管状）构件的实验得出的经验关系。由于方程用管道外径和厚度明确表示，因此只有管道截面允许使用屈曲支柱响应。ISO方程包含几个可以定义的因子。有效长度和附加长度因子考虑了单元端部固定度，屈曲折减因子考虑了弯矩对屈曲的影响。您可以为每个局部横截面方向定义这些因子的非默认值。

| **输入文件用法：** | 要允许切换到具有ISO方程默认系数和默认屈曲包络线的屈曲支柱响应，请使用以下选项： |
| --- | --- |
|  | ``` [*FRAME SECTION](../key/key-link.md#usb-kws-mframesection), SECTION=PIPE, ELSET=*name*, BUCKLING, YIELD STRESS=![](../graphics/usb_eqn00310.gif) ``` 要允许切换到具有ISO方程非默认系数和默认屈曲包络线的屈曲支柱响应，请使用以下所有选项： ``` [*FRAME SECTION](../key/key-link.md#usb-kws-mframesection), SECTION=PIPE, ELSET=*name*, BUCKLING, YIELD STRESS=![](../graphics/usb_eqn00310.gif) [*BUCKLING LENGTH](../key/key-link.md#usb-kws-mbucklinglength) [*BUCKLING REDUCTION FACTORS](../key/key-link.md#usb-kws-mbucklingreductfact) ``` 要允许切换到具有ISO方程非默认系数和用户定义屈曲包络线的屈曲支柱响应，请使用以下所有选项： ``` [*FRAME SECTION](../key/key-link.md#usb-kws-mframesection), SECTION=PIPE, ELSET=*name*, YIELD STRESS=![](../graphics/usb_eqn00310.gif) [*BUCKLING ENVELOPE](../key/key-link.md#usb-kws-mbucklingenvelope) [*BUCKLING LENGTH](../key/link.md#usb-kws-mbucklinglength) [*BUCKLING REDUCTION FACTORS](../key/key-link.md#usb-kws-mbucklingreductfact) ``` |
