# 21.4 定义粘聚层与周围散体材料之间的绑定约束





如果您想使用比相邻散体材料网格更细的网格来对粘聚层进行建模，则应将粘聚层生成为单独的网格，并使用绑定约束绑定到散体材料（请参阅["定义绑定约束，" 第15.15.1节](pt03ch15s15hlb01.md)）。 您应该创建壳几何模型来表示粘聚层一侧的表面，并以所需的网格密度对该模型进行网格划分。 您可以使用此网格从可以生成偏移网格的网格零件创建网格。 有关更多信息，请参阅["使用粘聚单元进行建模，" Abaqus Analysis User's Guide第32.5.3节](../usb/usb-link.md#usb-elm-ecohesiveusage)。

将零厚度粘聚层绑定到周围散体材料网格时，您应小心。 使用以下过程来避免问题：

1. 使用实体偏移网格工具沿顶层和底层创建粘聚单元层。 Abaqus/CAE在创建粘聚单元层两侧的两个表面时，会将**TopSurf**和**BottomSurf**附加到表面名称前缀。
2. 使用堆栈方向查询工具区分粘聚单元的顶面（棕色）和底面（紫色）。
3. 将周围散体材料网格的表面绑定到相应的顶层和底层粘聚表面。 当提示您从粘聚表面选择表面时，从提示线右侧单击**Surfaces**并选择相应的表面。 例如，选择***surface name*-BottomSurf**将粘聚层的底面（紫色）绑定到散体材料网格的相邻表面。
