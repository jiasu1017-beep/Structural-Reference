# 4.3.5 最小化、最大化、恢复或删除视口





最小化、最大化和删除按钮位于每个视口右上角，视口标题旁边，如下图所示：

![](../graphics/cnv-maxbutton-nls.png)

如果需要，**平铺**或**层叠**视口以显示最小化、最大化和删除按钮。

**最小化按钮**

当您单击最小化按钮时，视口会缩小为绘图区域左下方默认位置的简短标题栏。最小化按钮会变为恢复按钮，并与最大化和删除按钮一起保留在标题栏中。如果您移动最小化的视口，新位置将保存为其最小化位置，持续整个Abaqus/CAE会话。

**最大化按钮**

当您单击最大化按钮时，视口会更改大小和位置以填充画布。最大化按钮会变为恢复按钮，并与最小化和删除按钮一起移动到主菜单栏的右侧。最大化的视口成为当前视口，并移动到画布的前面，隐藏绘图区域中的任何其他视口。视口标题栏中的信息会显示在Abaqus/CAE标题栏中。

**删除按钮**

当您单击删除按钮或从主菜单栏选择****视口（Viewport）**![](../graphics/images/arrow.gif)**删除当前（Delete Current）****时，视口会从画布中删除。您无法恢复已删除的视口。如果画布上只有一个视口，Abaqus/CAE不允许您删除它。如果您已在画布上创建了其他视口，Abaqus/CAE会删除当前视口并选择另一个视口作为当前视口。您无法控制Abaqus/CAE选择哪个视口作为当前视口。

**恢复按钮**

恢复按钮会替换您最近使用的最小化按钮或最大化按钮。当您单击恢复按钮时，Abaqus/CAE会将视口返回到之前的大小和位置。恢复按钮会恢复为它所替换的最小化或最大化按钮。恢复的视口成为当前视口，并移动到画布的前面，隐藏绘图区域中的任何其他视口。
