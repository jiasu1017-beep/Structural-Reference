# 70.3.2 作为特征的分区







每个部件的几何都由一组特征构建；分区只是您添加到部件中的附加几何特征。因为分区是特征，所以可以使用特征操作工具集进行编辑、删除、抑制和恢复。

如果您创建分区然后修改底层部件或装配的几何，Abaqus/CAE 会与其他所有特征一起再生分区。此外，再生分区的几何依赖于您用于创建分区的方法。以下示例说明了底层几何的变化可能导致分区在装配再生时移动或改变形状。

1. 用户使用两个选定顶点之间的线（两个圆的中心）对面进行分区，如图 70-3](pt06ch70s03s02.md#par-regenface1) 所示。**图 70-3** 两个选定顶点之间的分区。![](../graphics/par-regenface1-nls.png)
2. 用户修改装配，定义分区的选定顶点位置发生变化。当 Abaqus/CAE 再生装配时，分区仍定义为两个圆中心之间的线，如图 70-4](pt06ch70s03s02.md#par-regenface2) 所示。**图 70-4** 再生后的分区。![](../graphics/par-regenface2-nls.png)

您可以使用特征操作工具集对某些分区进行有限修改：
- 您可以输入参数直接分区边并定位划分面的 Bézier 曲线。参数必须介于零和一之间，表示边长度的分数；特征操作工具集允许您编辑提供的值。
- 您可以在面上草绘分区；特征操作工具集允许您编辑草图。
