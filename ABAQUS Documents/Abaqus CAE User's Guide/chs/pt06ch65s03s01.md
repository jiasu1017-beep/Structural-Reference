# 65.3.1 什么是再生？







每当您修改部件的特征时，Abaqus/CAE 必须再生该部件。类似地，要将修改后的特征合并到部件的任何实例中，Abaqus/CAE 会在您进入装配相关模块时再生装配中的特征。但是，为节省时间，Abaqus/CAE 会确定需要再生的最少特征数量，并仅再生受修改影响的特征。如果您的部件和装配相对简单，您不会注意到 Abaqus/CAE 正在执行此再生。然而，在更复杂的模型中，再生可能导致性能下降。触发再生的特征修改包括使用特征操作工具集编辑、抑制、恢复和删除特征。

模型树使用黄色勾号指示特征需要再生，如["使用模型树管理特征，" 第 65.2 节"](pt06ch65s02.md)中所述。您始终可以通过从主菜单栏选择 ****Feature**![](../graphics/images/arrow.gif)**Regenerate****（特征→再生）或在模型树中的 **Features**（特征）容器上点击鼠标按钮 3 来强制 Abaqus/CAE 再生部件或装配。
