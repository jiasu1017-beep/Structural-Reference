# 23.2 创建复合铺层





Abaqus/CAE允许您为三种类型的单元定义复合铺层：常规壳、连续壳和实体。 有关详细说明，请参阅以下章节：
- ["创建常规壳复合铺层，" 第12.14.2节](pt03ch12s14s02.md)
- ["创建连续壳复合铺层，" 第12.14.3节](pt03ch12s14s03.md)
- ["创建实体复合铺层，" 第12.14.4节](pt03ch12s14s04.md)

您在Property模块中创建复合铺层。 复合铺层编辑器提供了一个表格，您可以使用它来定义铺层中的铺层。 对于每个铺层，您指定铺层的名称、材料、厚度和方向，以及积分点的数量以及铺层被分配到的模型区域。 复合铺层编辑器的铺层表格如[图 23-3](pt04ch23s02.md#adv-layups-plytable-nls)所示。

**图 23-3** 复合铺层编辑器中的铺层表格。

![](../graphics/adv-layups-plytable-nls.png)

铺层表格提供了几个选项，使您可以更轻松地定义包含许多铺层的铺层； 例如，铺层表格允许您执行以下操作：
- 在表格中向上或向下移动或复制选定的铺层。
- 抑制或删除选定的铺层。
- 对选定的铺层进行图案化。
- 从ASCII文件读取铺层数据或向其写入铺层数据。 数据可以定义铺层中的所有铺层或仅定义铺层的子集。

抑制铺层的能力使您可以轻松试验复合铺层中不同配置的铺层，并查看对模型分析结果的影响。 有关更多信息，请参阅["定义复合铺层时使用铺层表格，" 第12.14.1节](pt03ch12s14s01.md)。

如果您定义的复合铺层的铺层关于中心芯对称，您只需要在铺层表格中输入铺层的下半部分。 当您应用对称选项时，Abaqus会自动通过以相反顺序重复输入的铺层（包括中心铺层）在生成的截面中创建额外的铺层。

您可能必须对模型进行划分以创建要分配铺层的区域。 您应该在创建铺层并开始定义铺层之前创建划分。 您可以直接从当前视口中的零件中选择区域，也可以创建引用该区域的命名集。 Abaqus/CAE会保留您决定在定义复合铺层后添加划分和铺层时任何现有的铺层区域； 例如，您可以向建模为区域提供额外刚性的肋的铺层添加铺层。 您为铺层分配的区域可以是Abaqus/CAE几何或网格。 您可以在单个复合铺层中合并分配给几何的铺层和分配给原生网格的铺层。

连续壳和实体复合铺层预期在整个厚度方向上在复合铺层中指定的所有区域组合中具有单个单元。 穿过厚度的每个单个单元包含您在铺层表格中定义的多个铺层。 如果您分配连续壳或实体复合铺层的区域包含穿过厚度的多个单元，每个单元将包含您在铺层表格中定义的所有铺层，分析结果将不符合预期。

如果您的模型在区域的整个厚度方向上包含多个连续壳或实体单元，您可以通过为每个单元层定义单独的复合铺层来获得正确的结果。 您可以通过在指定铺层区域时选择原生Abaqus/CAE网格的单元或孤儿单元来为每个单元层定义复合铺层。 您必须为每个单元层创建一个铺层。

如果您的复合铺层中有重叠的铺层，必须按重叠区域中出现的顺序在铺层表格中输入铺层。 [图 23-4](pt04ch23s02.md#adv-layups-plyorder) 展示了一个重叠区域和铺层表格中相应铺层顺序的简单示例。 铺层表格中的第一个铺层代表铺层中的底层铺层。

**图 23-4** 铺层表格中铺层的顺序。

![](../graphics/adv-layups-plyorder-nls.png)

在某些情况下，Abaqus无法确定复合铺层常规壳的铺层方向。 例如，如果您的铺层通过90度的角度急剧过渡，和/或您的零件与全局坐标系统的一个或多个平面对齐，则可能发生这种情况。 为帮助您诊断问题区域，Abaqus/CAE绘制了一个塌陷的坐标系统，指示用户选择的坐标系统和几何法线无法解析为有效方向以用于显示目的的区域。 如果出现这种情况，您可能可以通过在过渡处将铺层分割成多个铺层并为每个新铺层分配方向来完成分析。

如果您应用复合铺层并将截面分配分配给同一区域，Abaqus/CAE在分析期间仅使用复合铺层中的属性。 如果您将两个或多个复合铺层应用到重叠的区域，Abaqus/CAE使用最后一个铺层的属性，其中"最后"指的是复合铺层名称的字母顺序。 Property模块中的默认颜色编码使用黄色着色来指示具有重叠复合铺层和截面分配的区域，或指示具有多个重叠复合铺层的区域。
