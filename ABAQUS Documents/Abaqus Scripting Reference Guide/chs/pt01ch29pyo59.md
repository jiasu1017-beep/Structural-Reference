# 29.59 HeatGeneration 对象

HeatGeneration 对象在热传导分析中包含体积热生成。

**访问**

```
import material
mdb.models[*name*].materials[*name*].heatGeneration
import odbMaterial
session.odbs[*name*].materials[*name*].heatGeneration
```

### 29.59.1 HeatGeneration()

此方法创建 HeatGeneration 对象。

**路径**

```
mdb.models[*name*].materials[*name*].HeatGeneration
session.odbs[*name*].materials[*name*].HeatGeneration
```

**返回值**

HeatGeneration 对象。

**异常**

无。

### 29.59.2 成员

HeatGeneration 对象没有成员。
