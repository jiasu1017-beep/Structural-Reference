# 4.4.4 创建组合箭头和文本注释





您可以同时创建箭头和文本注释，以将文本与视口中的特定特征或结果相关联。此方法会创建一个箭头注释，锚点位于箭头头部，单独的文本注释锚定在同一位置并位于箭头尾部。在创建过程中，您的注释选项有限；例如，您无法将箭头头部与锚点偏移，也无法为箭头尾部（文本位置）选择第二个锚点。

创建后，Abaqus/CAE会独立保存箭头和文本注释，这样您就可以使用["编辑箭头注释属性，"第4.4.6节"](pt01ch04s04hlb05.md)和["编辑文本注释属性，"第4.4.7节"](pt01ch04s04hlb06.md)分别为每种注释类型提供的所有选项来编辑它们。由于它们没有链接，您也可以独立移动箭头和文本。

**提示：**要在保持相同相对位置的同时移动箭头和文本注释，请同时选择两个注释，然后将文本定位到所需位置，再仅选择箭头来编辑箭头位置。

**要创建组合箭头和文本注释：**

1. 从主菜单栏，选择****视口（Viewport）**![](../graphics/images/arrow.gif)**创建注释（Create Annotation）**![](../graphics/images/arrow.gif)**箭头和文本（Arrow & Text）****。**提示：**您也可以通过点击[**视口**工具栏](pt01ch04s02hlb02.md)中的![](../graphics/ico_annotationTextArrow.png)来创建箭头和文本注释。
2. 在视口中选择模型节点或顶点，或在提示区域输入模型坐标来选择箭头目标。Abaqus/CAE会在所选点显示锚点符号和箭头，默认文本字符串`Text--*n*`出现在光标位置。
3. 在视口中定位文本；将光标移动到所需的文本位置，然后单击鼠标按钮1。Abaqus/CAE会显示**箭头和文本编辑器（Arrow & Text Editor）**对话框。
4. 完成**箭头和文本编辑器**对话框中的信息以最终确定两个注释。随时单击**预览（Preview）**以在视口中查看您的更改；单击**确定（OK）**保存箭头和文本注释并关闭**箭头和文本编辑器**。

**要编辑文本：**

1. 如果**箭头和文本编辑器**中的**文本（Text）**选项卡尚未选中，请单击它。
2. 要显示或取消显示勾勒文本的边框，请切换**显示边框框（Show bounding box）**。边框框在视觉上将文本与底层模型或结果分开。
3. 输入所需的文本。输入文本注释时，您可以使用标准的鼠标和键盘编辑技术，如退格、复制和粘贴；按**[Enter]**开始新行。
4. 要自定义背景，请单击以下选项之一：- **匹配视口（Match viewport）**以匹配视口颜色作为背景。- **透明（Transparent）**以消除背景并仅显示文本。- **其他颜色（Other color）**以显示其他背景颜色选项。如果您选择了**其他颜色**：1. 单击颜色样本![](../graphics/usi-colorsample.png)。Abaqus/CAE会显示**选择颜色（Select Color）**对话框。2. 使用**选择颜色**对话框中的方法之一来选择新颜色。有关更多信息，请参阅["自定义颜色，"第3.2.9节"](pt01ch03s02s09.md)。3. 单击**确定（OK）**关闭**选择颜色**对话框。颜色样本会变为所选颜色。
5. 自定义文本字体。1. 单击**设置字体（Set Font）**。出现**选择字体（Select Font）**对话框。2. 使用**选择字体**对话框来选择您想要的字体特征。有关更多信息，请参阅["自定义字体，"第3.2.8节"](pt01ch03s02s08.md)。3. 完成后，单击**确定（OK）**实施您的更改并关闭**选择字体**对话框。
6. 选择文本颜色。1. 单击颜色样本![](../graphics/usi-colorsample.png)。Abaqus/CAE会显示**选择颜色**对话框。2. 使用对话框中的方法之一来选择新颜色。有关更多信息，请参阅["自定义颜色，"第3.2.9节"](pt01ch03s02s09.md)。3. 单击**确定（OK）**关闭**选择颜色**对话框。颜色样本会变为所选颜色。
7. 选择**左对齐（Left）**、**居中（Center）**或**右对齐（Right）**来排列边框区域内文本的位置。

**要编辑箭头：**

1. 如果**箭头和文本编辑器**中的**箭头（Arrow）**选项卡尚未选中，请单击它。
2. 选择所需的线条样式；您可以为箭头选择实线或几种虚线样式。
3. 选择所需的线条粗细。**注意：**更改线条粗细也会更改箭头的大小。
4. 选择所需的箭头符号。
5. 选择箭头颜色。1. 单击颜色样本![](../graphics/usi-colorsample.png)。Abaqus/CAE会显示**选择颜色**对话框。2. 使用对话框中的方法之一来选择新颜色。有关更多信息，请参阅["自定义颜色，"第3.2.9节"](pt01ch03s02s09.md)。3. 单击**确定（OK）**关闭**选择颜色**对话框。颜色样本会变为所选颜色。
