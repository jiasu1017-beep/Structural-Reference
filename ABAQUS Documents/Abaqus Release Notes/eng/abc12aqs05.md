# 12.5 Magnitude output of vector nodal variables







**Product: **Abaqus/Explicit  

**Benefits: **You can now output the magnitude of translational vector nodal quantities for displacement, velocity, acceleration, and reaction force.

**Description: **Four new nodal output variables have been introduced: spatial displacement magnitude UMAG, spatial velocity magnitude VMAG, spatial acceleration magnitude AMAG, and reaction force magnitude RFMAG. These new output variables are available for history output requests.
