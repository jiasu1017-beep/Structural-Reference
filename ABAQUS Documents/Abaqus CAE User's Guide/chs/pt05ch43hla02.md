# 43.6 叠加变形和未变形模型绘图





您可以将变形和未变形模型形状组合在单个绘图中。组合形状为显示和解释等值线、符号或材料方向提供了背景。[图43-1](pt05ch43hla02.md#viw-superim-online)中显示了一个叠加模型形状的示例。

**图43-1** 叠加在未变形形状上的变形形状。

![](../graphics/viw-superim-nls.png)

 要生成叠加绘图，从主菜单栏选择****绘图**![](../graphics/images/arrow.gif)***等值线、符号或材料方向***![](../graphics/images/arrow.gif)**两种形状上****，或使用工具箱中的![](../graphics/ico_plotContourDefUndef.png)、![](../graphics/ico_plotSymbolDefUndef.png)和![](../graphics/ico_plotMatOrientDefUndef.png)工具。要叠加未变形和变形形状而不显示等值线、符号或材料方向，或为相同结果显示任何组合的绘图类型，请使用![](../graphics/ico_plotMultipleStates.png)工具并从工具箱中选择所有所需的绘图类型（有关详细信息，请参见["显示多个绘图状态，" 第55.6节](pt05ch55s06.md)）。

**自定义叠加绘图：**

1. 从主菜单栏，选择****选项**![](../graphics/images/arrow.gif)**叠加****来自定义未变形形状的外观。**提示：**您也可以使用工具箱中的![](../graphics/ico_plotSuperimposeOptions.png)工具来自定义未变形绘图形状。
2. 点击以下选项卡来修改选项：- **基本**：选择渲染样式和边缘可见性。- **颜色和样式**：控制模型边缘颜色和样式、模型面颜色、边缘样式和边缘厚度。- **标签**：控制单元、面和节点标签以及节点符号。- **法线**：控制单元和表面法线。- **其他**：**其他**页面包含以下选项卡：- **缩放**：控制模型缩放和收缩。- **半透明度**：控制着色和填充渲染样式的半透明度。- **偏移**：控制未变形和变形形状之间的偏移。有关自定义绘图显示特征的更多详细信息，请参见[第55章，"自定义绘图显示"](pt05ch55.md)。Abaqus/CAE使用叠加绘图选项代替常用绘图选项来在视口中显示未变形绘图。
3. 要自定义变形形状的类似选项，请选择****选项**![](../graphics/images/arrow.gif)**常用****或使用工具箱中的![](../graphics/ico_plotCommonOptions.png)工具。
4. 如果等值线、符号或材料方向处于活动状态，请使用关联的绘图状态相关选项来自定义未变形和变形绘图的显示。
5. 在每个选项对话框中点击**应用**以在视口中实现您的更改。

您的叠加绘图选项更改将保存整个会话期间，并将影响所有后续显示两种模型形状的绘图中的未变形形状。
