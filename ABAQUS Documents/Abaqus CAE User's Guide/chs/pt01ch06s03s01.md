# 6.3.1 选择选项概述





当提示你从视口中选择一个对象时，Abaqus/CAE 提供了可以使你更轻松高效地进行所需选择的选择工具。

使用**选择**工具栏配置选择选项。[图 6-15](pt01ch06s03hlb01.md#obj-selection-toolbar-nls) 显示了选择工具的布局。如果选择工具对当前过程无效，则会变暗。
