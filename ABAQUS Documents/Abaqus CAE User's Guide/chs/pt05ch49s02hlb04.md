# 49.2.4 自定义比例因子或谐波动画





您可以通过在**半循环**和扩展范围**全循环**之间选择来自定义动画化模型的表观运动。扩展范围用于可视化例如通过模态分析获得的振动模式。您还可以更改动画中的帧（单独图像）数量。更多的帧会产生更平滑的动画。

在比例因子动画中，Abaqus/CAE通过帧数除以比例因子范围来计算动画比例因子值。这些值被应用于每个坐标方向中的[变形比例因子](pt05ch55s04hlb01.md)和[场输出变量值](pt05ch42s03hlb01.md)以产生动画序列中的每个图像。

在谐波动画中，Abaqus/CAE通过帧数除以角度范围来计算动画角度值。这些值被应用于计算该角度的场输出变量值并产生动画序列中的每个图像。

**自定义比例因子或谐波动画：**

1. 找到"比例因子/谐波"选项。从主菜单栏中，选择****选项**![](../graphics/images/arrow.gif)**动画****；然后在出现的对话框中单击"比例因子/谐波"选项卡。"比例因子/谐波"选项出现。
2. 要控制范围，请单击您想要的**相对缩放**：- **全循环**：对于比例因子动画（实数），此选择对应于范围1到1；对于谐波动画（具有实部和虚部的复数），此选择对应于范围180到180。- **半循环**：对于比例因子动画（实数），此选择对应于范围0到1；对于谐波动画（具有实部和虚部的复数），此选择对应于范围0到180。
3. 要选择动画中单独图像的数量，请单击"帧"箭头。指定数量的帧出现在"帧"框中。
4. 单击"应用"以实现您的更改。所有动画化视口中的比例因子或谐波动画会更改以反映您指定的相对缩放和帧数。您的更改在会话期间保存，并将影响谐波或比例因子动画状态中的所有后续绘图。
