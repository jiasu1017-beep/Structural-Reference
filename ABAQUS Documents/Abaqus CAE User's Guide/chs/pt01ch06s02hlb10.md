# 6.2.10 循环浏览有效选择





在某些情况下，Abaqus/CAE 无法区分你选择的对象和其他附近或相关的对象。这种模糊性可能由以下原因引起：
- 设想一个包围光标的小方块。当你点击一个对象时，落在此方块内的任何其他相同类型的有效对象也会被视为可能的选择。例如，如果你选择了一条位于另一条边附近的边，Abaqus/CAE 可能会认为两条边都是可能的选择。方块的大小与显示器大小、视口大小和模型尺寸无关。当你放大或缩小时，它也保持不变。因此，通过放大模型来增加对象之间的距离，你可以更精确地在视口中选择特定对象。
- 如果你的模型是三维的，设想一条垂直于屏幕并穿过光标延伸到模型的直线。当你选择一个对象时，任何与该直线相交的相同类型的有效对象都被视为可能的选择。（旋转模型可能会消除一些模糊性。）

Abaqus/CAE 尽可能通过针对当前过程过滤你的选择来减少模糊的可能性。例如，如果你正在分割一个单元体，Abaqus/CAE 会提示你选择要分割的单元体。当你进行选择时，Abaqus/CAE 仅认为单元体是有效选择。相反，如果你正在创建几何集，Abaqus/CAE 认为单元体、面、边和顶点都是有效选择，模糊的可能性增加。此外，预选高亮允许你在进行选择之前准确看到将选择哪个对象。在视口中移动光标可能会消除选择中的模糊性，并导致 Abaqus/CAE 高亮显示你选择的对象。如果模糊性仍然存在，Abaqus/CAE 会更改光标，在箭头右侧添加省略号 (...)，并高亮显示所有可能的选择。

当你的选择模糊时，Abaqus/CAE 会在提示区域显示按钮，允许你循环浏览所有可能的选择，如下所示：

![](../graphics/pic-next-feature-nls.png)

使用**下一个**和**上一个**按钮在视口中所有可能的选择对象之间向前和向后循环；每个对象会依次高亮显示。当你选择的对象高亮显示时，单击**确定**或单击鼠标按钮 2 确认你的选择。（你也可以在当前视口中单击鼠标按钮 3 以显示提示区域中选项的菜单。）
