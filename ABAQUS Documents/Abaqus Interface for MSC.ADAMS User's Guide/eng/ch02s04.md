# 2.4 Supported Abaqus elements







The Abaqus Interface for MSC.ADAMS is designed to support most Abaqus elements that have displacement degrees of freedom at any node. However, there are a few limitations and considerations. Infinite elements (for example, CIN3D8), coupled thermal-structural elements (for example, C3D8T), generalized axisymmetric elements (for example, CGAX3), and frame elements (for example, FRAME3D) are not supported.
