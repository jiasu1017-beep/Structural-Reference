# 14.9.4 替换步骤





您可以用Abaqus分析产品允许的任何过程替换现有过程；**替换步骤**对话框中的过程列表会更新以仅显示修订后步骤的可用过程。例如，您可以将**静态，通用**过程更改为**静态，Riks**过程。Abaqus/CAE将兼容的步骤相关对象复制到新步骤，替换等效对象（如果可能），并删除剩余对象。

替换步骤后，您应验证初始步骤中先前定义的属性、单元类型、作业和边界条件及场对模型仍然有效。更多信息，请参见["什么是步骤替换？，"第14.3.4节](pt03ch14s03s04.md)。

**要替换步骤：**

1. 从主菜单栏，选择**步骤**![](../graphics/images/arrow.gif)**替换**![](../graphics/images/arrow.gif)***步骤名称***。**替换步骤**对话框出现。**提示：**您也可以在**步骤管理器**中选择步骤名称并点击**替换**。
2. 点击**新过程类型**字段旁边的箭头，并从出现的列表中选择**通用**或**线性扰动**。对话框的下半部分显示可用过程列表。
3. 选择新过程，并点击**继续**。**编辑步骤**对话框出现。
4. 使用**编辑步骤**对话框从默认值修改设置，并为可选设置提供值。（有关特定编辑器功能的详细帮助，请从主菜单栏选择**帮助**![](../graphics/images/arrow.gif)**上下文帮助**，然后点击感兴趣的功能。）
5. 点击**确定**。如果步骤相关对象与新步骤不兼容，Abaqus/CAE会在消息区域显示步骤替换期间删除的对象列表，并关闭**编辑步骤**对话框。
