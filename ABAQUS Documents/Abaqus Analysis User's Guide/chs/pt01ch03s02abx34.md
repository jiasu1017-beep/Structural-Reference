# 3.2.34 将 LS-DYNA 数据文件转换为 Abaqus 输入文件





**产品：**Abaqus/Explicit

##### **参考**

- [『Abaqus 执行程序：概述』，第 3.1.1 节](pt01ch03s01abo02.md)

### 概述

LS-DYNA 到 Abaqus 的转换器将 LS-DYNA 输入文件中支持的关键字集合转换为 Abaqus 中的等效项。

### 使用转换器

转换器支持由 LS-DYNA Version 971 Rev 5 或更早版本创建的输入文件的转换。输入文件可以是任何名称和可选扩展名。

支持转换的 LS-DYNA 关键字列于下表中。其他 LS-DYNA 关键字和数据将被跳过并在日志文件中注明。

转换器创建一个包含模型数据和历史数据的 Abaqus 输入文件。但是，转换器不会为节点输出、单元输出和接触输出创建特定输出量的精确 Abaqus 等效项；而是使用预选的变量。您可以提供其他输出实体来完成请求。

#### 单元编号和分组

生成的 Abaqus 输入文件中的所有单元都具有唯一的单元编号。LS-DYNA 输入中具有非唯一单元编号的单元会自动分配新的单元编号；所有单元编号重新分配都会在日志文件中注明。

分配了相同 PART 标识号的单元被分组到一个单元集中。具有不同材料或属性的单元应分配不同的 PART 标识号；即，相同的材料和属性必须适用于分组在同一单元集中的所有单元。

当 PART 引用刚性材料时，该零件被视为刚体。与该零件对应的单元集用于刚性体定义。

#### 材料模型

转换器仅支持下表中所示的材料模型（第 3.2.34-1 表）。所有不支持的材料模型在需要应力-应变定律定义时都作为线弹性材料进行转换。在这些情况下，转换器会提供材料属性的标称值。

#### 映射以 _ID 或 _TITLE 结尾的 LS-DYNA 单元

许多 LS-DYNA 关键字包含选项 _ID、_TITLE 或这两个选项。除非 LS-DYNA 关键字与 _ID 或 _TITLE 在本文档的映射表中指定，否则转换器会将这些选项的数据映射到为主 LS-DYNA 关键字指定的相同 Abaqus 关键字。

### 命令行选项

**job**

此选项用于指定转换器输出的 Abaqus 输入文件的名称。Abaqus 输入文件的名称必须不带 `.inp` 扩展名。转换器创建的诊断信息写入名为 `*job-name*.log` 的文件。

**input**

此选项用于指定包含 LS-DYNA 关键字数据的文件的名称。LS-DYNA 输入文件可以有扩展名。

**splitFile**

此选项指定是否将 Abaqus 输入文件拆分为多个文件。如果 **splitFile**=`ON`，则输出以下文件：
- `job-name_nodes.inc`：包含节点数据的包含文件
- `job-name_elements.inc`：包含单元数据的包含文件
- `job-name_model.inc`：包含剩余模型数据的包含文件
- `job-name.inp`：Abaqus 输入文件，包含所有上述包含文件和历史数据
