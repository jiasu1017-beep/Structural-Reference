# 2.1.3 刚性弹丸撞击侵蚀板









**产品：** Abaqus/Explicit  

本示例模拟了刚性球形弹丸以1000 m/sec的速度撞击平板装甲板的斜向冲击。对板使用失效模型，从而允许弹丸穿透板。该示例说明了冲击、渐进失效和无限单元的使用。

### 问题描述

装甲板厚度为10 mm，与弹丸相比被假设为半无限尺寸。这通过在板周围使用CIN3D8无限单元来实现。板用4480个C3D8R单元建模。装甲板材料具有206.8 GPa的杨氏模量、0.3的泊松比、7800 kg/m³的密度、1220 MPa的屈服应力和1220 MPa的恒定硬化斜率。材料定义还包括渐进失效模型，这导致Abaqus/Explicit在单元失效时将其从网格中移除。假设失效发生在100%的等效塑性应变时，此时单元被瞬时从模型中移除。（失效应变值的选择有些任意，并不旨在建模任何特定材料。）

球体直径为20 mm，被假设为刚性的，质量对应于密度为37240 kg/m³的均匀材料。球体的旋转惯性在模型中不需要，因为我们假设球体和板之间没有摩擦。施加边界条件以约束球体在*y*方向的运动。测试了两种建模球体表面的方法：使用解析刚性表面和使用R3D4刚性单元。解析刚性表面是在准确性和计算性能方面表示这种简单刚性几何形状的首选方法。然而实践中出现的更复杂的三维表面几何形状必须用由单元面形成的表面来建模。这里给出了面表示的结果。C3D8R单元的单元公式用截面控制修改。此问题推荐使用的公式使用质心运动学公式以及刚度和粘性沙漏控制的线性组合。提供了运动学公式和沙漏控制的其他组合用于比较。

只对板的一半进行建模，在*x*–*z*平面使用适当的对称边界条件。模型如图2.1.3-1所示。为了可视化目的对完整球体进行建模。模型中有17094个自由度。

由于板中的单元将失效并从模型中移除，板内部的节点将暴露于与刚性球体表面的接触。因此，必须在球体表面（定义为基于单元的表面）和包含板中撞击点20 mm半径内所有节点的基于节点的表面之间建模接触。（有关使用基于单元的表面建模侵蚀的示例，请参见["侵蚀弹丸撞击侵蚀板，"第2.1.4节](ch02s01aex65.md)。）在主要输入文件中，接触对用于定义球体表面与节点集中任何节点之间的接触。还提供了使用通用接触算法的输入文件。

### 结果和讨论

球形弹丸以1000 m/sec的速度在距板法向30°的角度撞击板。不同时刻的变形形状如图2.1.3-2到图2.1.3-4所示，适用于质心运动学和组合（粘性-刚度形式）沙漏截面控制（分析情况pl3d_erode_ccs）。在分析早期，如图2.1.3-2所示，板表面被侵蚀的材料量相对较小，板仍在球体下变形。在图2.1.3-3中，板已被穿透，弹丸仍与孔边缘接触。在图2.1.3-4中，弹丸已离开板并以恒定速度移动。图2.1.3-5和图2.1.3-6显示了弹丸速度的历史（表2.1.3-1显示了用于获得这些结果的分析选项）。结果显示高度一致。

在图2.1.3-2到图2.1.3-4中，失效单元通过在Abaqus/CAE中创建一个仅包含活动单元的显示组来消除。
