# 42.5.2 使用场输出工具栏





您可以使用**Field Output**工具栏访问**Field Output**对话框的基本功能。从工具栏中，您可以：
- 选择要操作的场输出变量类型（[**Primary**](pt05ch42s03hlb03.md)、[**Deformed**](pt05ch42s03hlb04.md)或[**Symbol**](pt05ch42s03hlb05.md)）；
- 从可用场输出变量列表中选择变量名；
- 选择所选主变量的细化级别（如不变量和分量）（如果有）；以及
- 选择视口绘图状态是否应与工具栏选择同步。

**Status**和**Stream**变量类型是唯一无法从工具栏选择的场输出。**Status**和**Stream**变量类型在**Field Output**对话框中可用；状态变量允许您指定Abaqus/CAE用于从模型显示中移除失败单元的标准，流变量决定流体流动数据分析中流线显示的场输出。要打开**Field Output**对话框，请点击**Field Output**工具栏左侧的![](../graphics/ico_fieldOutput.png)。

**图42-2****Field Output**工具栏。

![](../graphics/viw-fieldout-toolbar-nls.png)

当您从工具栏进行选择时，Abaqus/CAE会更新当前视口以显示输出；如果需要更改绘图状态且绘图状态同步开关（![](../graphics/ico_synchLayers.png)）已启用，绘图状态也会更新。例如，选择**Primary**作为变量类型会将绘图状态更改为在变形模型上显示云图（如果视口尚未包含云图）。如果禁用同步，Abaqus/CAE仍会在当前视口中显示新选择的场输出变量，但不会更改绘图状态。
