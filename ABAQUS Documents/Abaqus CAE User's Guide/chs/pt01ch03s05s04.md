# 3.5.4 更改模型的视图







如果您从容器的弹出菜单中选择**设为主项**，Abaqus/CAE 会在模型树或结果树中显示所选容器下的所有内容，并在树上方的菜单中显示容器名称。例如，如果您有一个复杂的模型或输出数据库，以及相应复杂的模型树或结果树，此选项很有用。您可以使用**设为主项**通过仅显示您正在处理的部分来简化模型树或结果树。例如，[图 3-29](pt01ch03s05s04.md#uss-int-tree-setasroot) 显示了左侧模型树的视图，并将其与将**材料**容器设为主项的效果进行对比。

**图 3-29** **设为主项**对模型树的影响。

![](../graphics/uss-int-tree-setasroot-nls.png)

当您更改默认根容器时，您可以使用模型树和结果树上方的菜单在其级别之间向上移动。此外，Abaqus/CAE 会在模型树或结果树上方激活两个图标，如[图 3-29](pt01ch03s05s04.md#uss-int-tree-setasroot)所示：模型树中的**设为主项为模型数据库**图标，结果树中的**设为主项为会话数据**图标，以及两个树中的**向上一级**图标。
- **设为主项为模型数据库**图标![](../graphics/uss-int-modeltree-home.png)将模型树返回到默认视图，在树顶部显示**模型数据库**。**设为主项为会话数据**图标![](../graphics/uss-int-modeltree-home.png)将结果树返回到默认视图，在树顶部显示**会话数据**。
- **向上一级**图标![](../graphics/uss-int-modeltree-uponelevel.png)将模型树或结果树的根向上移动一级；例如，从**材料**容器向上一级到包含**材料**容器的**梁**模型。

如果您在模型树或结果树的背景上点击鼠标按钮 3，Abaqus/CAE 会显示带有以下选项的弹出菜单：

**显示空容器**

默认情况下，Abaqus/CAE 会在模型树和结果树中显示所有容器，无论它们是否有内容。通过关闭**显示空容器**选项，您可以禁止显示没有任何内容的容器。如果您在 Abaqus/CAE 中执行将项目添加到先前空容器的操作（例如，使用 Interaction 模块工具箱创建相互作用），则容器和项目将重新出现在模型树或结果树中。要从视图中抑制容器，容器必须完全为空；即使容器中的所有项目都因过滤器而隐藏（请参阅[「在模型树和结果树中使用弹出菜单，第 3.5.3 节」](pt01ch03s05s03.md)），该容器也不会被**显示空容器**选项抑制。

**显示空容器**选项的状态在 Abaqus/CAE 会话之间保持。

**全部展开**

如果选择**全部展开**，Abaqus/CAE 会展开模型树或结果树中的所有容器和项目。

**全部折叠**

如果选择**全部折叠**，Abaqus/CAE 会折叠模型树或结果树中的所有容器，仅使顶级容器和项目可见。

**设为主项为显示对象**

如果选择**设为主项为显示对象**，则与当前视口中可见零件对应的容器将成为模型树的根（如上所述）。如果在当前视口中可见装配，则相应模型的**装配**容器将成为模型树的根。

此选项仅在模型树中可用。

**设为主项为模型数据库**

如果选择**设为主项为模型数据库**，Abaqus/CAE 会将模型树返回到默认视图，在树顶部显示**模型数据库**。此选项与**设为主项为模型数据库**图标![](../graphics/uss-int-modeltree-home.png)具有相同的效果。

此选项仅在模型树中可用。

**设为主项为会话数据**

如果选择**设为主项为会话数据**，Abaqus/CAE 会将结果树返回到默认视图，在树顶部显示**会话数据**。此选项与**设为主项为会话数据**图标![](../graphics/uss-int-modeltree-home.png)具有相同的效果。

此选项仅在结果树中可用。
