# 40.2 进入和退出可视化模块







您可以在Abaqus/CAE会话的任何时候通过单击上下文栏中**模块**列表中的**可视化**来进入可视化模块。主菜单栏上会出现**结果**、**绘图**、**动画**、**报告**、**选项**和**工具**菜单；如果存在当前输出数据库，则当前视口的标题栏会显示当前输出数据库的名称。

您也可以通过打开现有的输出数据库来进入可视化模块。要打开输出数据库，请从主菜单栏中选择**文件**![](../graphics/images/arrow.gif)**打开**（更多信息，请参见"打开模型数据库或输出数据库"，第9.7.2节），或者当某个选定作业的结果可用时，在**作业管理器**中单击**结果**。当您使用这些方法之一进入可视化模块时，Abaqus/CAE会在当前视口中显示输出数据库中模型的绘图。

要退出可视化模块，请从**模块**列表中选择任何其他模块，或者通过从主菜单栏中选择**文件**![](../graphics/images/arrow.gif)**退出**来结束会话。当您结束会话时，Abaqus/CAE会关闭所有文件和窗口。Abaqus仅在会话期间保存您的绘图选项。
