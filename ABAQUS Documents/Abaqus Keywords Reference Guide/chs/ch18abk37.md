# *SUBCYCLING





### *SUBCYCLING定义子循环区域。

此选项用于定义子循环区域。

**产品：**Abaqus/Explicit

**类型：**模型数据

**级别：**模型

##### **参考：**

- ["Explicit dynamic analysis," Section 6.3.3 of the Abaqus Analysis User's Guide](../usb/usb-link.md#usb-anl-aexpdynamic)
- ["Fully coupled thermal-stress analysis," Section 6.5.3 of the Abaqus Analysis User's Guide](../usb/usb-link.md#usb-anl-acouptempdisp)
- ["Selective subcycling," Section 11.7.1 of the Abaqus Analysis User's Guide](../usb/usb-link.md#usb-anl-aselectivesubcycling)

### **必需参数：**

ELSET

将此参数设置为定义子循环区域的单元集名称。

**此选项没有关联的数据行。**
