# 12.13.4 创建复合实体截面





复合实体截面用于定义由不同材料层以不同取向组成的三维区域的截面属性。请参见Abaqus Analysis User's Guide的第28.1.1节"实体（连续体）元素"中的"在Abaqus/Standard中定义复合实体元素"。

复合实体截面只能用于Abaqus/Standard。复合实体截面只能分配给仅具有位移自由度 的三维砖元素。复合实体元素主要用于建模方便。在大多数情况下，您应该将复合截面建模为壳或连续壳。但是，在以下情况下应使用复合实体截面：
- 当横向剪切效应占主导时。
- 当您不能忽略正应力时。
- 当需要精确的层间应力时，例如在复杂加载或几何形状的局部区域附近。

如果您分配复合实体截面的区域包含穿过厚度的多个元素，则每个元素将包含数据表中定义的所有材料层，分析结果将不如预期。

**创建复合实体截面：**

1. 从主菜单栏，选择****Section**![](../graphics/images/arrow.gif)**Create****。出现**Create Section**对话框。
   **提示：**您也可以在**Section Manager**中点击**Create**，或在Property模块工具箱中选择创建截面工具![](../graphics/ico_section.png)。
2. 输入截面名称。有关命名对象的更多信息，请参见["使用基本对话框组件，"第3.2.1节"](pt01ch03s02s01.md)。
3. 选择**Solid**作为截面**Category**，选择**Composite**作为截面**Type**，然后点击**Continue**。出现复合实体截面编辑器。
4. 可选地，输入铺层名称。Abaqus/CAE此名称显示在铺层堆叠图中。有关命名对象的更多信息，请参见["使用基本对话框组件，"第3.2.1节"](pt01ch03s02s01.md)。
5. 如果截面中的材料层关于中心芯对称，请切换**Symmetric layers**。在数据表格中输入材料层，从第一行的底层开始，到中心层结束。在分析过程中，Abaqus通过以相反顺序重复输入的层（包括中心层）将层附加到截面定义到截面的顶部。如果为材料层命名，则每个生成的层通过在重复层的原始名称前面添加`Sym_`来在铺层堆叠图和输出数据库中标记。
6. 复合实体截面的每一层由数据表格中的一行表示。**注意：**复合层的堆叠方向由分配给实体区域的材料取向决定。请参见["分配材料取向"位于"分配材料取向或钢筋参考取向，"第12.15.4节"](pt03ch12s15s01.md#usi-prp-assign-matorient)，了解更多。要向表格添加行，请在行上点击鼠标按钮三，并从出现的菜单中选择**Insert Row Before**或**Insert Row After**。对于每一层，输入以下数据：
   **Material** 形成该层的材料名称。点击**Material**列，然后点击出现的箭头以显示可用材料列表，并选择形成该层的材料。
   **Element Relative Thickness** 该层的相对厚度。Abaqus根据元素几何确定整个截面厚度，并且对于给定截面定义，厚度可能在模型中变化。因此，您指定的厚度值仅是每层的相对厚度。层的实际厚度是元素厚度乘以每层占总厚度的分数。您不必使用物理单位来指定层的厚度比，并且层相对厚度的总和不必总和为一。
   **Orientation Angle** 取向。取向可以指定为角度或取向名称。取向角相对于截面取向定义，沿法线正向逆时针测量。如果您指定取向名称，Abaqus/CAE假定为用户定义的取向。您必须提供包含指定取向名称的用户定义取向定义的用户子程序[`ORIENT`](../sub/sub-link.md#sub-xsl-orient)。您不能使用离散场定义可变取向角；要为复合实体定义逐层取向分布，必须使用复合铺层编辑器（请参见["创建和编辑复合铺层，"第12.14节"](pt03ch12s14.md)）。
   **Integration Points** 穿过厚度的积分点数量。您只能指定奇数。
   **Ply Name** 该层的名称。Abaqus/CAE在Visualization模块中查看复合铺层时以及在铺层堆叠图中显示此名称。为复合实体截面中的层命名是可选的。但是，如果您为任何层提供名称，则必须为截面中的所有层提供名称。
7. 点击**OK**保存更改并关闭复合实体截面编辑器。
