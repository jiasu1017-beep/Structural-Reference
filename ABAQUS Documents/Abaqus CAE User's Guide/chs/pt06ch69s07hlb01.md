# 69.7.1 移除面







为了使部件有效或精确，您可以使用移除面工具从三维实体或壳或二维平面部件中移除所选的面。 选择要移除的面后，Abaqus/CAE 查找定义特征的相邻面。您可以移除整个特征，也可以仅移除所选的面。当您从实体部件中移除一个或多个面时，Abaqus/CAE 将部件中的所有胞转换为壳，如下图所示：

![](../graphics/prt-hlp-shell-solid-nls.png)

查询工具集提供了一组几何诊断工具，使您能够定位无效和不精确几何的区域。 有关更多信息，请参阅["在 Part 模块中使用查询工具集，" 第 11.16.4 节"](pt03ch11s16s04.md)。移除面的操作作为部件的特征存储；因此，您可以使用模型树恢复您删除的面。

**移除面：**

1. 从主菜单栏中选择 ****Tools**![](../graphics/images/arrow.gif)**Geometry Edit****（工具→几何编辑）。Abaqus/CAE 显示 **Geometry Edit**（几何编辑）对话框。**提示：**您也可以使用位于 Part（部件）模块工具箱中编辑工具处的 ![](../graphics/ico_repairRemoveFace.png) 工具来移除面。 有关工具箱中编辑工具的图表，请参阅["使用几何编辑工具集，" 第 69.1 节"](pt06ch69s01.md)。
2. 从对话框中选择 **Face**（面）类别和 **Remove**（移除）方法。
3. 选择要移除的面。您可以从视口选择面或选择包含面的现有集合。**注意：**默认选择方法基于您最近使用的选择方法。要恢复为其他方法，请点击提示区域右侧的 **Select in Viewport**（在视口中选择）或 **Sets**（集合）。您可以使用拖动选择、**[Shift]****+点击**、**[Ctrl]****+点击** 和角度方法的组合来选择多个要移除的面。 有关更多信息，请参阅["在当前视口中选择对象，" 第 6.2 节"](pt01ch06s02.md)。**提示：**如果您无法选择所需的面，可以使用 **Selection**（选择）工具栏更改选择行为。 有关更多信息，请参阅["使用选择选项，" 第 6.3 节"](pt01ch06s03.md)。
4. 点击鼠标按钮 2 表明您已完成选择面。**注意：**如果您要从实体部件中移除一个或多个面，Abaqus/CAE 会指示移除面将导致实体转换为壳，并提示您继续。Abaqus/CAE 移除所选的面。
