# 39.8 Spectrum 对象

Spectrum 对象定义轮廓显示的颜色光谱。

**访问**

```
import visualization
session.spectrums[*name*]
```

### 39.8.1 Spectrum(...)

此方法创建 Spectrum 对象并将其放入 spectrums 存储库。

**Path**

```
session.Spectrum
```

**必需参数**

*name*

字符串，指定光谱的名称。

*colors*

字符串序列，指定光谱的颜色。

**可选参数**

无。

**返回值**

Spectrum 对象。

**异常**

无。

### 39.8.2 成员

Spectrum 对象没有成员。
