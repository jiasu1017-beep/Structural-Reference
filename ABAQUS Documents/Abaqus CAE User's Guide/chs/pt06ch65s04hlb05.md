# 65.4.5 删除特征







选择 ****Feature**![](../graphics/images/arrow.gif)**Delete****（特征→删除）从主菜单栏删除所选特征。您还可以通过在模型树中的特征上点击鼠标按钮 3 并从出现的菜单中选择 **Delete**（删除）来删除特征。

当您删除父特征时，也会删除其所有子特征。您无法恢复已删除的特征。

**删除特征：**

1. 从主菜单栏中选择 ****Feature**![](../graphics/images/arrow.gif)**Delete****（特征→删除）。**提示：**您也可以使用模块工具箱中位于的 ![](../graphics/ico_featureDelete.png) 工具来删除特征。
2. 选择一个或多个要删除的特征。您可以从当前视口或从模型树中选择特征。有关更多信息，请参阅[第 6 章，"在视口中选择对象"](pt01ch06.md)"和["使用模型树和结果树，" 第 3.5 节"](pt01ch03s05.md)。Abaqus/CAE 高亮显示所选特征。如果您选择了具有子特征的特征，Abaqus/CAE 也会高亮显示子特征。如果您从视口选择特征，请在完成选择后点击提示区域中的 **Done**（完成）。
3. 如果您从视口选择了特征，请在提示区域中点击 **Yes**（是）。如果您从模型树选择了特征，请在特征上点击鼠标按钮 3，并从出现的菜单中选择 **Delete**（删除）。所选特征和任何子特征被删除。
4. 要退出删除过程，请执行以下操作之一 - 点击提示区域中的取消按钮 ![](../graphics/fts-cancel.gif)，或 - 在 Abaqus/CAE 窗口中的任意位置点击鼠标按钮 2，或 - 从 **Feature**（特征）菜单中选择其他操作。
