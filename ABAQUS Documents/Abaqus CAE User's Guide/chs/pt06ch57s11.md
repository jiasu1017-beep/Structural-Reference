# 57.11 定义频谱







使用频谱方法定义要在响应谱分析中使用的频谱。有关更多信息，请参见 Abaqus Analysis User's Guide](../usb/usb-link.md#usb-anl-aresponsespectrum-spec) 的"响应谱分析"第 6.3.10 节中的"指定频谱"。

**定义频谱：**

1. 如 ["选择要定义的振幅类型，" 第 57.3 节](pt06ch57s03.md) 中所述显示 **编辑振幅** 对话框。
2. 点击 **规格单位** 字段旁边的箭头，并指定您希望如何定义频谱的单位。- 选择 **位移**、**速度** 或 **加速度** 分别以位移单位、速度单位或加速度单位定义频谱。- 选择 **重力** 以 *g* 单位指定加速度频谱。
3. 如果您选择了 **重力**，请输入重力加速度的值。
4. 在数据表中，输入 **幅值**（频谱的幅值）、**频率**（使用此幅值的频率，以周期/时间表示）和 **阻尼**（关联阻尼，以临界阻尼比表示）值。有关如何输入数据的详细信息，请参见 ["输入表格数据，" 第 3.2.7 节](pt01ch03s02s07.md)。
5. 点击 **确定** 保存您的振幅定义并关闭 **编辑振幅** 对话框。
