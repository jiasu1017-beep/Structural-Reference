# 17.16.1 定义整个零件或零件实例的种子密度





您可以从主菜单选择 ****Seed**![](../graphics/images/arrow.gif)**Part**** 或 ****Seed**![](../graphics/images/arrow.gif)**Instance**** 来定义尚未具有品红色边种子的零件或零件实例所有边的近似单元尺寸。 以这种方式定义的种子称为零件种子或实例种子，显示为白色。 （有关更多信息，请参阅["控制种子密度，" 第17.4.3节](pt03ch17s04s03.md)。） 您应该对所有边施加种子。 如果均匀的种子分布足够，建议的方法是对整个零件或零件实例进行种子布置。

**创建零件或实例种子：**

1. 从主菜单栏，选择 ****Seed**![](../graphics/images/arrow.gif)**Part**** 或 ****Seed**![](../graphics/images/arrow.gif)**Instance****。 Abaqus/CAE在提示区域显示提示以指导您完成该过程。 **提示：**您也可以使用Mesh模块工具箱中种子工具所在的 ![](../graphics/ico_meshSeedInstance.png) 工具对零件或零件实例进行种子布置。 （有关更多信息，请参阅["使用Mesh模块工具箱，" 第17.15节](pt03ch17s15.md)。）
2. 如果您正在对零件实例进行种子布置，并且您的装配包含多个零件实例，请选择要种子布置的零件实例，然后单击鼠标按钮2。 **注意：**边种子始终覆盖零件和实例种子； 因此，如果您已经单独对零件或零件实例的所有边进行了种子布置，则实例种子未使用且不显示。 如有必要，使用种子删除工具（请参阅["删除边种子，" 第17.16.8节](pt03ch17s16hlb08.md)）删除任何不需要的边种子； 如果您已分配零件或实例种子，这些种子会自动出现在删除边种子的边上。
3. 在出现的**Global Seeds**对话框中，输入近似单元尺寸。
4. 默认情况下，Abaqus/CAE对零件或零件实例的种子布置应用曲率控制，以便在网格中正确近似小孔或高曲率区域。 曲率控制允许Abaqus/CAE根据边的曲率以及目标单元尺寸计算种子分布。 要控制曲率对种子布置的影响，请输入**Maximum deviation factor**的值。 偏差因子是单元边缘偏离原始几何形状的程度的一种度量。 为了帮助您可视化偏差因子的影响，Abaqus/CAE会显示它将在圆周上创建的单元数量，对应于您输入的设置。
5. 如果需要，更改**Minimum size control**。 指定最小尺寸控制有助于防止Abaqus/CAE在高曲率区域创建不必要的细密网格（如果这些区域对分析意图不重要）。 选择以下选项之一： - 将最小值指定为全局单元尺寸的分数。 Abaqus/CAE使用此方法，默认最小值为`0.1`（10%）。 - 输入绝对最小单元尺寸。 该尺寸必须大于`0.0`且小于近似全局单元尺寸。
6. 单击**Apply**查看Abaqus/CAE将使用的种子布置，并根据需要调整**Global Seeds**对话框中输入的值。
7. 单击**OK**提交单元尺寸并关闭对话框。 白色种子出现在零件或零件实例的所有边上，但已分配品红色边种子的边除外。
8. 要退出零件或实例种子布置过程，请按**[Enter]**或单击鼠标按钮2。
