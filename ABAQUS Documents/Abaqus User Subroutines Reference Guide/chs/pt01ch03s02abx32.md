# 3.2.32 将RADIOSS输入文件翻译为部分Abaqus输入文件





**产品：** Abaqus/Explicit  

##### **参考**

- ["Abaqus执行程序：概述，" 第3.1.1节](pt01ch03s01abo02.md)

### 概述

从RADIOSS到Abaqus的翻译器将RADIOSS输入文件中的某些关键字转换为其在Abaqus/Explicit中的对应物。

### 使用翻译器

翻译器需要由RADIOSS版本4.4或5.1创建的块格式输入文件。输入文件可以有任意名称和可选扩展名。

翻译的RADIOSS数据条目列在下面的表格中。其他RADIOSS关键字和数据被跳过并记录在日志文件中。

翻译器创建一个部分Abaqus输入文件，其中仅包含模型数据和时间历史输出数据。您可以提供额外的输出数据来完成输入。

#### 元素编号和分组

生成Abaqus输入文件中的所有元素将具有唯一的元素编号。将自动为RADIOSS输入中具有非唯一元素编号的元素分配新的元素编号。分配了相同PART标识号的元素被分组在一个元素集中。

具有不同材料或属性的元素必须分配不同的PART标识号；即，相同的材料和属性必须适用于组合在同一元素集中的所有元素。

如果从SPRING翻译而来的元素具有不同的元素属性（如用于定义局部方向的歪斜系统）并被分配了相同的PART标识号，翻译器会自动将它们分离到不同的元素集中。

#### 材料模型

翻译器仅支持[表3.2.32-1](pt01ch03s02abx32.md#table-usb-int-dradiossproc-materialdata)中所示的材料模型。所有不支持的材料模型在需要应力-应变律定义时被翻译为线性弹性。在这些情况下，翻译器会提供材料属性的标称值。

### 命令行选项

**job**

此选项用于指定要由翻译器输出的Abaqus输入文件的名称。必须不带`.inp`扩展名给出Abaqus输入文件的名称。翻译器生成的诊断信息被写入名为`*job-name*_fromradioss.log`的文件。

**input**

此选项用于指定包含RADIOSS数据的文件的名称。文件扩展名是可选的。

**splitAirbagElements**

此选项用于指定将4节点气囊膜元素拆分为两个3节点气囊膜元素。默认值为`ON`。气囊膜元素由具有厚度方向一个积分点的SHELL或SH3N的翻译产生。此选项仅在RADIOSS输入文件中指定了关键字MONVOL/AIRBAG时有效。

**readAbaqusDat **

此选项允许使用先前Abaqus分析的数据(`.dat`)文件来重新制定点焊定义。数据文件应标识无法形成的点焊。使用此选项，使用分布式耦合约束翻译所识别点焊的连接点。

**userDefaultMass**

此选项用于指定在翻译期间需要非零质量的额外节点分配的节点质量。此值应该很小（通常为整个模型质量的10⁻⁶倍）。如果省略此选项，默认质量设置为10⁻⁴。

**userDefaultInertia**

此选项用于指定在翻译期间需要非零转动惯量的额外节点分配的转动惯量。此值应该很小（通常为整个模型转动惯量的10⁻⁶倍）。如果省略此选项，默认转动惯量设置为10⁻³。

**userHistoryTime**

此选项用于指定用于时间历史输出的时间间隔。如果省略此选项，时间历史间隔设置为10⁻⁵。
