# *SUBSTRUCTURE PATH





### *SUBSTRUCTURE PATH进入子结构以获取输出或从先前进入的子结构返回。

此选项用于在子结构的"级别"间导航以获取结果输出。

**产品：**Abaqus/Standard

**类型：**历史数据

**级别：**此选项在以部件实例装配定义的模型中不受支持。

##### **参考：**

- ["Using substructures," Section 10.1.1 of the Abaqus Analysis User's Guide](../usb/usb-link.md#usb-anl-asuperelements)

### **至少需要以下参数之一：**

ENTER ELEMENT

如果要在子结构内跟随输出请求，则包含此参数。将此参数设置为在此选项出现之前Abaqus/Standard操作的使用级别处用于子结构的单元号。

LEAVE

包含此参数以离开子结构并返回到其使用级别。

**此选项没有关联的数据行。**
