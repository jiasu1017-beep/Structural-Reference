# 10.7 示例：轴对称支架







您被要求找到如图 [Figure 10--39](ch10s07.md#gss-axi-mount) 所示橡胶支架的轴向刚度，并识别可能限制支架疲劳寿命的任何高最大主应力区域。支架两端粘合到钢板上。它将承受高达 5.5 kN 的轴向载荷，均匀分布在钢板上。横截面几何形状和尺寸如图 [Figure 10--39](ch10s07.md#gss-axi-mount) 所示。

**图 10–39** 轴对称支架。

![](../graphics/gss-axi-mount-nls.png)

由于结构和加载都是轴对称的，您可以使用轴对称单元进行此模拟。因此，您只需要对穿过组件的平面建模：每个单元代表完整的 360° 环。您将检查支架的静态响应；因此，您将使用 Abaqus/Standard 进行分析。

### 10.7.1 对称性

您不需要对整个轴对称组件进行建模，因为问题关于支架中心的水平线是对称的。通过仅对一半进行建模，您可以使用一半的单元，从而大约减少一半的自由度。这显著减少了分析的运行时间和存储需求，或者允许您使用更细化的网格。

许多问题包含一定程度的 symmetry。例如，镜像对称、循环对称、轴对称或重复对称（如图 [Figure 10--40](ch10s07.md#gss-symmetry) 所示）是常见的。结构或组件中可能存在多种类型的对称。

**图 10–40** 各种对称形式。

![](../graphics/gss-symmetry-nls.png)

当仅对对称组件的一部分进行建模时，您必须添加边界条件以使模型表现得好像正在对整个组件进行建模。您可能还需要调整施加的载荷以反映实际建模的结构部分。考虑如图 [Figure 10--41](ch10s07.md#gss-portal-frame) 所示的门框。

**图 10–41** 对称门框。

![](../graphics/gss-portal-frame-v-nls.png)

门框关于图中所示的垂直线对称。为了在模型中保持对称，必须约束对称线上任何节点，使其不能沿 1 方向平移，也不能绕 2 或 3 轴旋转（自由度 5 和 6）。因此，对称约束为

```
[*BOUNDARY*](../key/key-link.md#usb-kws-hboundary)
*<node>*,1
*<node>*,5,6
```

在门框问题中，载荷沿模型的对称平面施加；因此，应该只对您建模的部分施加总值的一半。

在使用轴对称单元（如本橡胶支架示例）的轴对称分析中，我们只需要对组件的横截面进行建模。单元公式自动包含轴对称的影响。

### 10.7.2 坐标系

在此模拟中，此模型使用默认的 *r–z*（1–2）轴对称坐标系。其原点放置在板底部的高度处，如图 [Figure 10--39](ch10s07.md#gss-axi-mount) 所示。

### 10.7.3 网格设计

此示例中的网格使用一阶轴对称混合实体单元（CAX4H）的 30 × 15 网格用于橡胶支架。仅指定了模型中支架的下半部分，如图 [Figure 10--42](ch10s07.md#gss-mesh) 所示。

**图 10–42** 橡胶支架的网格。

![](../graphics/gss-mesh-v-nls.png)

### 10.7.5 检查输入文件——模型数据

我们审查模型数据，包括几何定义（节点和单元）和材料特性。

**模型描述**

输入文件应包含分析的合适描述。

```
*HEADING
Axisymmetric mount analysis under axial loading
S.I. Units (m, kg, N, sec)
```

**节点坐标和单元连接性**

输入文件中至少会有两个 [*ELEMENT*](../key/key-link.md#usb-kws-melement) 选项块，因为模拟中使用了两种不同类型的单元。检查单元类型是否正确以及包含单元的单元集是否具有描述性名称是个好主意。您的输入文件中的 [*ELEMENT*](../key/key-link.md#usb-kws-melement) 选项应类似于

```
*ELEMENT, TYPE=CAX4I, ELSET=PLATE
*ELEMENT, TYPE=CAX4H, ELSET=RUBBER
```

**节点集**

检查节点集 `MIDDLE` 是否已创建。如果没有，请使用编辑器添加。

**特性定义**

需要两个单元特性定义：一个用于对橡胶进行建模的单元，一个用于对钢板进行建模的单元。您的模型中应有以下单元特性定义：

```
*SOLID SECTION, MATERIAL=RUBBER, ELSET=RUBBER
*SOLID SECTION, MATERIAL=STEEL, ELSET=PLATE
```

**材料特性：橡胶的超弹性模型**

您已经获得了一些用于支架中橡胶材料的实验测试数据，如图 [Figure 10--46](ch10s07.md#gss-mat-testdata) 所示。有三组不同的测试数据——单轴测试、双轴测试和平面（剪切）测试——可用。您决定让 Abaqus 根据测试数据计算适当的超弹性材料常数。您不确定橡胶支架中的应变有多大，但您怀疑它们小于 2.0。

**图 10–46** 橡胶材料的材料测试数据。

![](../graphics/gss-mat-testdata-v-nls.png)

双轴和平面测试的测试数据远远超过了此量级，因此您决定执行实验测试的单单元模拟，以确认 Abaqus 从测试数据计算的系数是足够的。

使用一阶多项式应变能函数来对橡胶材料进行建模。通过在 [*HYPERELASTIC*](../key/key-link.md#usb-kws-mhyperelast) 选项上使用 N=1 和 POLYNOMIAL 参数来指示这些选择。使用 TEST DATA INPUT 参数指示 Abaqus 应从您将提供的测试数据中找到材料常数。测试数据在紧跟 [*HYPERELASTIC*](../key/key-link.md#usb-kws-mhyperelast) 选项的选项上输入。数据应作为公称应力和相应的公称应变输入，负值表示压缩。您可能能够直接使用预处理器输入数据（例如，如果您使用 Abaqus/CAE）；否则，您将不得不使用编辑器将其添加到输入文件中。橡胶的材料定义类似于

**材料特性：钢的弹性特性**

钢仅使用线弹性特性建模（![](../graphics/gsk_eqn00072.gif) = 200 GPa，![](../graphics/gsk_eqn00073.gif) = 0.3），因为载荷应该不足以引起非弹性变形。因此，钢的材料选项块为

```
*MATERIAL, NAME=STEEL
*ELASTIC
2.0E11, 0.3
```

### 10.7.6 检查输入——历史数据

我们现在讨论与此问题相关的历史数据，包括时间增量参数、边界条件、载荷和输出请求。

**包含非线性几何并指定初始增量大小**

当在模型中使用超弹性材料时，Abaqus 假定它可能经历大变形。但是，只有在 [*STEP*](../key/key-link.md#usb-kws-hstep) 选项上设置 NLGEOM=YES 时，才会包含大变形和其他非线性几何效应。因此，您必须在此模拟中包含它，否则 Abaqus 将终止分析并出现输入错误。[*STEP*](../key/key-link.md#usb-kws-hstep) 选项应类似于

```
*STEP, NLGEOM=YES
```

模拟将是静态分析，总步骤时间为 1.0。将初始时间增量指定为总步骤时间的 1/100。程序选项块应类似于

```
*STATIC
.01, 1.0
```

**边界条件**

在位于对称平面上的节点上指定对称边界条件。在此模型中，对称条件阻止节点在自由度 2（轴向）中移动，如图 [Figure 10--50](ch10s07.md#gss-boundarycond) 所示。

**图 10–50** 橡胶支架上的边界条件。

![](../graphics/gss-boundarycond-v.png)

可以使用 YSYMM 类型边界条件施加约束全局 2 方向运动的对称条件，或者您可以简单约束 2 方向。在这种情况下，[*BOUNDARY*](../key/key-link.md#usb-kws-hboundary) 选项块具有以下格式：
```
*BOUNDARY
MIDDLE, 2, 2, 0.0
```

径向方向（全局 1 方向）不需要边界约束，因为模型的轴对称性质不允许结构作为刚体在径向方向移动。Abaqus 将允许节点在径向方向移动，甚至那些最初在对称轴上（即径向坐标为 0.0 的节点），只要没有对其径向位移（自由度 1）施加边界条件。由于您希望在此分析中让支架径向变形，不要施加任何边界条件；再次，Abaqus 将自动防止刚体运动。

**加载**

支架必须承受 5.5 kN 的最大轴向载荷，均匀分布在钢板上。因此，在钢板底部施加分布载荷。压力大小由

![](../graphics/gsk_eqn00289.gif)

给出。

如果您使用预处理器生成了压力载荷，则输入文件中可能存在具有许多数据行的 [*DLOAD*](../key/key-link.md#usb-kws-hdload) 选项块。

```
*DLOAD
 1, P1, 0.50E6
 2, P1, 0.50E6
...
29, P1, 0.50E6
30, P1, 0.50E6
```

对于此处讨论的单元和节点编号，压力施加到单元集 `PLATE` 中所有单元的面 1。这允许我们使用更紧凑的格式用于 [*DLOAD*](../key/key-link.md#usb-kws-hdload) 选项块的数据行。

```
*DLOAD
PLATE, P1, 0.50E6
```

**输出请求**

将预选变量和公称应变写为场输出到输出数据库文件。此外，将钢板底部一个节点的位移写入输出数据库文件，以便计算支架的刚度。您需要创建一个包含该节点的节点集。模型中的输出选项块应类似于以下内容：

```
*NSET, NSET=OUT
1,
*OUTPUT, FIELD, VARIABLE=PRESELECT
*ELEMENT OUTPUT
NE,
*OUTPUT, HISTORY
*NODE OUTPUT, NSET=OUT
U,
```
确保使用 [*END STEP*](../key/key-link.md#usb-kws-hendstep) 选项清楚标记步骤定义的结束。

### 10.7.7 运行分析

将输入选项存储在名为 `mount.inp` 的文件中。上述部分讨论的模型的输入选项可在 ["轴对称支架，" 第 A.10 节](ap01s10.md) 中找到。由于模拟的非线性性质意味着它可能需要一些时间才能完成，请使用以下命令在后台运行分析：

```
abaqus job=mount
```

作业完成后，检查数据文件 `mount.dat` 是否有错误。如果有，纠正输入文件并重新运行分析。如有必要，将您的输入与 ["轴对称支架，" 第 A.10 节](ap01s10.md) 中所示的进行比较。

### 10.7.9 后处理

通过在操作系统提示符下发出以下命令使用 Abaqus/Viewer 查看分析结果：

```
abaqus viewer odb=mount
```

**计算支架的刚度**

通过创建钢板上节点位移作为施加载荷函数的 *X–Y* 图来确定支架的刚度。您将首先为您写入输出数据库文件数据的钢板节点创建垂直位移图。数据是为集合 `OUT` 中的节点写入的。

**创建垂直位移历史曲线并交换 *X* 和 *Y* 轴：**

1. 在结果树中，展开名为 `mount.odb` 的输出数据库下的 **History Output** 容器。
2. 找到并选择集合 `OUT` 中节点的垂直位移 U2。
3. 点击鼠标按钮 3，并从出现的菜单中选择 **Save As** 保存 *X--Y* 数据。出现 **Save XY Data As** 对话框。
4. 在 **Save XY Data As** 对话框中，将曲线命名为 `SWAPPED`，并选择 **swap(XY)** 作为保存操作；点击 **OK**。时间-位移图出现在视口中。

您现在有了时间-位移曲线。您需要的是力-位移曲线。这很容易创建，因为在此模拟中，施加到支架的力与总分析时间成正比。您所要做的就是将 `SWAPPED` 曲线乘以载荷的大小（5.5 kN）来绘制力-位移曲线。

**将曲线乘以常数：**

1. 在结果树中，双击 **XYData**。出现 **Create XY Data** 对话框。
2. 选择 **Operate on XY data**，然后点击 **Continue**。出现 **Operate on XY Data** 对话框。
3. 在 **XY Data** 字段中，双击 `SWAPPED`。表达式 `"SWAPPED"` 出现在对话框顶部的文本字段中。您的光标应在文本字段的末尾。
4. 输入 `*5500` 将数据对象乘以施加载荷的大小。
5. 点击底部的 **Save As** 保存相乘的数据对象。出现 **Save XY Data As** 对话框。
6. 在 **Name** 文本字段中，输入 `FORCEDEF`；然后点击 **OK** 关闭对话框。
7. 要查看力-位移图，点击 **Operate on XY Data** 对话框底部的 **Plot Expression**。

您现在创建了一条具有支架力-挠度特性的曲线（轴标签没有反映这一点，因为您没有更改实际绘制的变量）。要获得刚度，您需要区分曲线 `FORCEDEF`。您可以使用 **Operate on XY Data** 对话框中的 `differentiate( )` 运算符来执行此操作。

**获得刚度：**

1. 在 **Operate on XY Data** 对话框中，清除当前表达式。
2. 从列出的 **Operators** 中，点击 **differentiate(X)**。`differentiate( )` 出现在对话框顶部的文本字段中。
3. 在 **XY Data** 字段中，双击 `FORCEDEF`。表达式 `differentiate( "FORCEDEF" )` 出现在文本字段中。
4. 点击底部的 **Save As** 保存相乘的数据对象。出现 **Save XY Data As** 对话框。
5. 在 **Name** 文本字段中，输入 `STIFF`；然后点击 **OK** 关闭对话框。
6. 点击 **Plot Expression** 绘制刚度-位移曲线。
7. 点击 **Cancel** 关闭对话框。
8. 打开 **Axis Options** 对话框，切换到 **Title** 选项卡页面。
9. 自定义轴标题，使其如图 [Figure 10--51](ch10s07.md#gss-stiff) 所示。**图 10–51** 支架的刚度特性。![](../graphics/gss-stiff-v-nls.png)
10. 点击 **Dismiss** 关闭 **Axis Options** 对话框。

支架的刚度随着支架变形增加近 100%。这是橡胶非线性和支架变形时形状变化的结果。或者，您可以通过将上述所有运算符组合成一个表达式直接创建刚度-位移曲线。

**直接定义刚度曲线：**

1. 在结果树中，双击 **XYData**。出现 **Create XY Data** 对话框。
2. 选择 **Operate on XY data**，然后点击 **Continue**。出现 **Operate on XY Data** 对话框。
3. 从列出的 **Operators** 中，点击 **differentiate(X)**。`differentiate( )` 出现在对话框顶部的文本字段中。
4. 在 **XY Data** 字段中，双击 `SWAPPED`。表达式 `differentiate( "SWAPPED" )` 出现在文本字段中。
5. 将光标放在 `"SWAPPED"` 数据对象之后的文本字段中，然后输入 `*5500` 将交换的数据乘以常数总力值。`differentiate( "SWAPPED"*5500 )` 出现在文本字段中。
6. 点击底部的 **Save As** 保存相乘的数据对象。出现 **Save XY Data As** 对话框。
7. 在 **Name** 文本字段中，输入 `STIFFNESS`；然后点击 **OK** 关闭对话框。
8. 点击 **Cancel** 关闭 **Operate on XY Data** 对话框。
9. 如果您还没有这样做，请自定义 *X* 和 *Y* 轴标签，使其如图 [Figure 10--51](ch10s07.md#gss-stiff) 所示。
10. 在结果树中，在 **XYData** 容器下方点击 `STIFFNESS` 的鼠标按钮 3，并从出现的菜单中选择 **Plot** 查看 [Figure 10--51](ch10s07.md#gss-stiff) 中的图，该图显示了支架变形时轴向刚度的变化。

从主菜单栏选择 ****Plot**![](../graphics/images/arrow.gif)**Deformed Shape****，或使用 ![](../graphics/ico_plotDeformed.png) 工具绘制支架的变形模型形状（参见 [Figure 10--53](ch10s07.md#gss-deformed-v)）。

板已被推起，导致橡胶向侧面鼓起。使用 **View Manipulation** 工具栏中的 ![](../graphics/ico_viewZoomRectangle.png) 工具放大网格的左下角。点击鼠标按钮 1 并按住以定义新视图的第一个角；移动鼠标以创建包围您想要查看区域的框；然后释放鼠标按钮。或者，您可以通过从主菜单栏选择 ****View**![](../graphics/images/arrow.gif)**Specify**** 来缩放和平移绘图。

您应该有一个类似于如图 [Figure 10--54](ch10s07.md#gss-distortion) 所示的图（在此图及后续图中，抑制了之前应用的扫过和镜像操作）。

**图 10–54** 橡胶支架模型左角的变形。

![](../graphics/gss-distortion-v-nls.png)

模型此角中的一些单元变形严重，因为该区域的网格设计对于发生的变形类型来说是不够的。虽然单元形状在分析开始时很好，但随着橡胶向外鼓起，它们会变形严重，尤其是角落的单元。如果进一步增加载荷，单元变形可能变得过大，分析可能会中止。[第 10.8 节"大变形网格设计"](ch10s08.md) 讨论了如何改进此问题的网格设计。

右下角变形单元呈现的梯形模式表明它们正在锁定。在这些单元的静水压力应力等值线图中（不跨共享公共节点的单元平均），相邻单元之间的压力应力变化很快。这表明这些单元正在遭受体积锁定，这在 ["为弹塑性问题选择单元，" 第 10.3 节](ch10s03.md) 中从塑性不可压缩性的背景下讨论过。在此问题中，体积锁定源于过度约束。钢相对于橡胶非常硬。因此，沿着粘合线，橡胶单元无法侧向变形。由于这些单元还必须满足不可压缩性要求，它们受到高度约束并发生锁定。解决体积锁定的分析技术在 ["减少体积锁定的技术，" 第 10.9 节](ch10s09.md) 中讨论。

**最大主应力等值线**

在模型中绘制最大面内主应力。按照以下过程在实际变形形状上创建带有抑制绘图标题的填充等值线图。

**对最大主应力进行等值线：**

1. 默认情况下，Abaqus/Viewer 显示 **S**，**Mises** 作为主场输出变量。在 **Field Output** 工具栏中，选择 **Max. Principal** 作为不变量。Abaqus/Viewer 自动将当前绘图状态更改为在变形模型形状上显示最大面内主应力等值线。
2. 打开 **Contour Plot Options** 对话框。
3. 将均匀等值线间隔滑块拖动到 `8`。
4. 点击 **OK** 查看等值线图并关闭对话框。创建一个仅显示橡胶支架单元的显示组。
5. 在结果树中，展开名为 `Mount.odb` 的输出数据库下的 **Materials** 容器。
6. 点击 **RUBBER** 的鼠标按钮 3，并从出现的菜单中选择 **Replace**，以选定的元素替换当前显示。
7. 视口显示更改，仅显示橡胶支架单元，如图 [Figure 10--55](ch10s07.md#gss-contours) 所示。**图 10–55** 橡胶支架中最大主应力等值线。![](../graphics/gss-contours-v-nls.png)

模型中报告在等值线图例中的最大主应力为 136 kPa。虽然此模型中的网格相当细化，因此外推误差应该最小，但您可能希望使用查询工具 ![](../graphics/ico_info.png) 来确定最大主应力的更准确的积分点值。

当您查看积分点值时，您会发现最大主应力的峰值出现在模型右下部分变形单元之一中。此值可能不可靠，因为单元变形和体积锁定的程度。如果忽略此值，则对称平面附近有一个区域的最大主应力约为 88.2 kPa。

显示模型中主应变范围的 最简单方法是显示等值线图例中的最大值和最小值。

**检查主公称应变幅度：**

1. 从主菜单栏，选择 ****Viewport**![](../graphics/images/arrow.gif)**Viewport Annotation Options****。出现 **Viewport Annotation Options** 对话框。
2. 点击 **Legend** 选项卡，并切换 **Show min/max values**。
3. 点击 **OK**。最大值和最小值出现在视口中等值线图例的底部。
4. 在 **Field Output** 工具栏中，如果尚未选择，请选择 **Primary** 作为变量类型。Abaqus/Viewer 自动将当前绘图状态更改为在变形模型形状上显示最大面内主应力等值线。
5. 从输出变量列表中，选择 **NE**。
6. 从 **Field Output** 工具栏的不变量列表中，如果尚未选择，请选择 **Max. Principal**。等值线图更改为显示最大主公称应变的数据。从等值线图例中记下最大主公称应变值。
7. 从不变量列表中，选择 **Min. Principal**。等值线图更改为显示最小主公称应变的数据。从等值线图例中记下最小主公称应变值。

最大和最小主公称应变值表明模型中的最大拉伸公称应变约为 100%，最大压缩公称应变约为 56%。由于模型中的公称应变保持在 Abaqus 超弹性模型与材料数据拟合良好的应变范围内，因此您可以相当确信支架预测的响应是合理的材料建模。
