# *INELASTIC HEAT FRACTION






### *INELASTIC HEAT FRACTION定义表现为热源的不可逆耗散率分数。

此选项用于在绝热热机械问题中提供不可逆能量耗散作为热源。当在 [*DYNAMIC](ch04abk43.md) 或 [*STATIC](ch18abk31.md) 选项上包含 ADIABATIC 参数时，它是相关的。当使用 [*COUPLED TEMPERATURE-DISPLACEMENT](ch03abk81.md) 或 [*DYNAMIC TEMPERATURE-DISPLACEMENT](ch04abk44.md) 选项进行耦合热-电-结构或完全耦合热-应力分析时，它也是相关的。

**产品：** Abaqus/Standard  Abaqus/Explicit  Abaqus/CAE  

**类型：** 模型数据  

**级别：** 模型  

**Abaqus/CAE：** Property 模块
