# 8.1.5 硬拷贝图像质量





当你直接将所选视口的快照打印到 PostScript 打印机或保存到 PostScript 或封装的 PostScript (EPS) 文件时，Abaqus/CAE 会创建图像的矢量或光栅表示（有关更多信息，请参阅["打印图像格式，" 第 8.1.1 节](pt01ch08s01s01.md)）。

矢量表示图像是分辨率无关的，因此其质量仅取决于打印机的分辨率。

对于 PostScript 和 EPS 图像，你可以使用相应选项对话框中的**分辨率**字段来指定你保存或打印的图像的分辨率。在较高分辨率下，光栅图像看起来更平滑，锯齿更少。低分辨率矢量图像如果缩放到较大尺寸可能会有孔。

虽然高分辨率图像质量更高，但定义图像需要更多数据；结果文件会消耗大量磁盘空间。低分辨率图像通常打印和显示更快。一般来说，你应该选择仍然产生可接受图像的最低分辨率。你可能想要在制作草稿时保存较低分辨率的图像，然后在最终版本切换到较高分辨率。

打印机的分辨率设定了打印图像分辨率的上限。例如，如果你在 600 点/英寸 (dpi) 的分辨率下保存图像，但在 300 dpi 分辨率的打印机上打印，打印图像的分辨率将仅为 300 dpi。

光栅表示图像质量也可能受到你在图像创建后使用外部软件对图像进行的更改（如缩放和旋转）的影响。缩放和旋转可能会扭曲光栅图像。因此，在打印图像的光栅表示之前，你应该调整画布上的视口以匹配最终应用程序中的尺寸和方向。缩放和旋转不会扭曲或降低矢量表示图像的质量。

对于矢量表示的 PostScript 和 EPS 图像，你可以使用相应选项对话框中的**着色质量**字段来指定图像中曲面上照明的质量。此选项不影响图像分辨率或文件大小。更精细的着色质量会产生更接近光栅表示的图像。更粗糙的着色质量通常打印和显示更快。与光栅图像的分辨率类似，你应该选择仍然产生可接受图像的最粗糙着色质量。

矢量 PostScript 和 EPS 图像不支持透明度；所有半透明或透明对象在使用矢量 PostScript 或 EPS 格式打印时都将显示为不透明。
