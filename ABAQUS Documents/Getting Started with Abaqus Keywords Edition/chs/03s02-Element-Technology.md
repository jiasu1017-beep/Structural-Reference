# 3.2 刚性体

在Abaqus中，刚性体是由节点和单元组成的集合，其运动由单个节点控制，该节点称为**刚性体参考节点**，如图3-7所示。

**图3-7** 形成刚性体的单元

![图3-7 形成刚性体的单元](../graphics/gss-rigidelem-nls.png)

刚性体的形状既可以是通过旋转或拉伸二维几何轮廓得到的解析曲面，也可以是通过对物体进行网格划分得到的离散刚性体。刚性体的形状在模拟过程中不会改变，但可以发生大的刚体运动。离散刚性体的质量和惯性可以根据其单元的贡献进行计算，也可以直接指定。

刚性体的运动可以通过在刚性体参考节点上施加边界条件来规定。作用于刚性体上的载荷来自施加到组成刚性体的节点上的集中载荷和施加到组成刚性体的单元上的分布载荷，或者来自施加到刚性体参考节点上的载荷。刚性体通过与变形单元的节点连接以及与变形单元的接触来与模型的其他部分相互作用。

刚性体的应用在[第12章，"接触"](../key/key-link.htm#usb-kws-mrigidbody)中进行了说明。

## 3.2.1 确定何时使用刚性体

刚性体可用于模拟非常硬的部件，这些部件可以是固定的或正在发生大的刚体运动。它们也可用于模拟变形部件之间的约束，并提供了一种方便的方法来指定某些接触相互作用。当Abaqus用于准静态成形分析时，刚性体非常适合模拟模具（如冲头、凹模、拉延筋、压边圈、辊轮等），也可以作为一种有效的约束方法。

出于验证目的，将模型的某些部分设为刚性可能是有用的。例如，在复杂模型中，远离感兴趣区域的单元可以作为刚性体的一部分，从而在模型开发阶段获得更短的运行时间。当您对模型满意后，可以删除刚性体定义，并在整个模型中采用精确的变形有限元表示。

与变形有限元相比，用刚性体表示模型部分的主要优势是计算效率。刚性体组成部分的单元不进行单元级计算。虽然需要一些计算工作来更新刚性体的节点运动以及组装集中载荷和分布载荷，但刚性体的运动完全由刚性体参考节点最多六个自由度决定。

在Abaqus/Explicit中，刚性体对于模拟结构中相对较硬的部分特别有效，而跟踪这些部分的应力波和分布并不重要。硬区域中单元的稳定时间增量估计可能导致非常小的全局时间增量。由于刚性体和作为刚性体一部分的单元不影响全局时间增量，在硬区域使用刚性体而不是变形有限元表示可以获得更大的全局时间增量，而不会显著影响解的整体精度。

在Abaqus中用解析刚性曲面定义的刚性体比离散刚性体的计算成本略低，并且可能产生更平滑的结果。例如，在Abaqus/Explicit中，与解析刚性曲面的接触往往比与离散刚性体的接触噪音更小，因为解析刚性曲面可以是光滑的，而离散刚性体本质上是分面化的。然而，解析刚性曲面可以定义的形状是有限的。

## 3.2.2 刚性体的组成

要创建离散刚性体，请使用`*RIGID BODY`选项作为形成刚性体的单元的属性参考。使用`REF NODE`参数为刚性体分配一个刚性体参考节点。刚性体参考节点具有平移和旋转自由度，每个刚性体都必须定义一个。刚性体参考节点的位置并不重要，除非对物体施加旋转或需要关于穿过物体某轴的约束反力矩。在这两种情况下，节点应放置在穿过物体的所需轴上。

```abaqus
*RIGID BODY, REF NODE=<节点>, ELSET=<单元集名称>,
PIN NSET=<节点集名称>, TIE NSET=<节点集名称>
```

除了刚性体参考节点外，离散刚性体还包括通过将单元和节点分配给刚性体而生成的节点集合。这些节点称为**刚性体从节点**（参见图3-7），它们提供与其他单元的连接。构成刚性体的节点有两种类型：

- 销节点，只有平移自由度。
- 绑定节点，具有平移和旋转自由度。

刚性体节点类型由节点所附着的刚性体上的单元类型决定。在将节点直接分配给刚性体时，也可以指定或修改节点类型。对于销节点，只有平移自由度属于刚性体，这些自由度的运动受刚性体参考节点运动的约束。对于绑定节点，平移和旋转自由度都属于刚性体，并受刚性体参考节点运动的约束。

定义刚性体的节点不能施加任何边界条件、多点约束或约束方程。但是，可以在刚性体参考节点上施加边界条件、多点约束、约束方程和载荷。

## 3.2.3 刚性单元

Abaqus中的刚性体功能允许大多数单元（不仅仅是刚性单元）成为刚性体的一部分。例如，如果`*RIGID BODY`选项引用包含形成刚性体的单元的单元集，则可以使用壳单元或刚性单元来模拟相同的效果。管理刚性体的规则（如如何施加载荷和边界条件）适用于形成刚性体的所有单元类型，包括刚性单元。

所有刚性单元的名称都以字母"R"开头。接下来的字符表示单元的维度。例如，"2D"表示单元是平面的；"AX"表示单元是轴对称的。最后一个字符表示单元的节点数。

**刚性单元库**

三维四边形（R3D4）和三角形（R3D3）刚性单元用于模拟三维刚性体的二维表面。Abaqus/Standard中还提供了另一种单元——双节点刚性梁单元（RB3D2），主要用于模拟必须施加流体阻力和浮力载荷的海洋结构部件。

对于平面应变、平面应力和轴对称模型，可使用双节点刚性单元。Abaqus/Standard中还提供了平面双节点刚性梁单元，主要用于二维模拟海洋结构。

**自由度**

只有刚性体参考节点具有独立自由度。对于三维单元，参考节点具有三个平移自由度和三个旋转自由度；对于平面和轴对称单元，参考节点具有自由度1、2和6（绕3轴的旋转）。

附着在刚性单元上的节点只有从自由度。这些节点的运动完全由刚性体参考节点的运动决定。对于平面和三维刚性单元，唯一从自由度是平移。Abaqus/Standard中的刚性梁单元具有与相应变形梁单元相同的从自由度：三维刚性梁为1-6，平面刚性梁为1、2和6。

**物理特性**

所有刚性单元必须引用`*RIGID BODY`选项。对于平面和梁单元，可以在数据行上定义横截面积。对于轴对称和三维单元，可以在数据行上定义厚度；仅在向刚性单元施加体力时才需要这些数据。默认厚度为零。或者，`NODAL THICKNESS`参数可以根据节点处的厚度定义平均面元厚度。在施加体力或接触定义需要厚度时需要这些数据。

**公式和积分**

由于刚性单元是不可变形的，它们不使用数值积分点，也没有可选的公式。

**单元输出变量**

刚性单元没有单元输出变量。刚性单元的唯一输出是节点的运动。此外，还可以在刚性体参考节点上获取反力和反力矩。
