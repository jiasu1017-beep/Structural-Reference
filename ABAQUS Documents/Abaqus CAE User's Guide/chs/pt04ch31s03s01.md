# 31.3.1 扩展有限元方法（XFEM）概述





您可以使用扩展有限元方法（XFEM）来研究准静态问题中的裂缝萌生和扩展。XFEM允许您研究沿任意、依赖解的路径的裂缝扩展，而无需对模型进行重新网格划分。XFEM仅适用于三维实体和二维平面模型；不支持三维壳模型。您可以使用XFEM来研究包含几何体、孤立网格单元或两者组合的部件中的裂缝。您可以选择研究任意穿过模型的裂缝扩展或静止裂缝。您在Interaction模块中定义XFEM裂缝。您可以指定裂缝的初始位置。或者，您可以允许Abaqus在分析过程中基于裂缝域中计算的最大主应力或应变值来确定裂缝位置。更多信息，请参见["使用扩展有限元方法将不连续性建模为富集特征"，Abaqus Analysis User's Guide第10.7.1节](../usb/usb-link.md#usb-anl-aenrichment)。在Abaqus/CAE中创建的XFEM模型示例请参见["使用XFEM对不连续性进行建模"，Abaqus Benchmarks Guide第1.19节](../bmk/bmk-link.md#bmkxfem)。

要执行XFEM裂缝分析，必须指定以下内容：

**裂缝域**

要定义裂缝域，您可以从三维部件选择一个或多个单元或从二维平面部件选择一个或多个面。如果在孤立网格或包含孤立网格和原生网格单元组合的部件上定义裂缝域，可以选择单元。裂缝域包括包含任何现有裂缝的区域以及可能萌生裂缝并可能扩展到其中的区域。

**裂缝扩展**

您可以允许裂缝沿任意、依赖解的路径扩展，或者可以指定裂缝是静止的。

**初始裂缝位置**

要定义初始裂缝位置，可以从三维实体或二维平面模型中选择面。初始裂缝位置必须位于裂缝域内。选定的面可以是实体的面、由分区创建的面或平面部件实例。类似地，选定的边缘可以是实体的边缘、由分区创建的边缘或线部件实例；您不应选择裂缝裂缝。您不应为您选择用于定义初始裂缝位置的面或边缘划分网格。[图31-14](pt04ch31s03s01.md#usi-eng-xfem-crack)显示了两个和三维几何体以及孤立网格的裂缝域和裂缝位置示例。

或者，您可以选择不定义初始裂缝位置。无论您是否定义初始裂缝位置，Abaqus都会通过搜索正在经历大于由牵引-分离法则指定的 最大损伤值的主应力和/或应变区域来在模拟期间启动裂缝创建。

**富集半径**

富集半径是从裂缝尖端开始的一小段距离，在此范围内单元将用于计算静止裂缝的裂缝奇异性。富集半径内的单元必须包含在您选择表示裂缝域的单元或面中。您可以允许Abaqus计算半径（富集区域中典型单元特征长度的三倍），或者可以指定其值。

**接触相互作用属性**

您可以选择将接触相互作用属性与XFEM裂缝关联，该属性定义裂缝单元表面的接触。详细信息请参见["为XFEM指定接触相互作用属性"，第31.3.6节](pt04ch31s03hlb03.md)。

**损伤萌生**

您必须通过在材料定义中指定损伤萌生标准来指定将萌生裂缝的条件。您可以指定基于最大主应力或最大主应变的标准。更多信息，请参见["定义损伤"中的"最大主应力或应变损伤"，第12.9.3节](pt03ch12s09s03.md#usi-prp-mechanical-damage-maxpsmaxpe)。

**分析过程**

您可以在静态分析过程中包含XFEM裂缝。或者，您可以在隐式动态分析过程中包含XFEM裂缝，以模拟高速冲击载荷下结构中的断裂和失效。隐式动态过程中模拟的基于XFEM的裂缝扩展也可以在静态过程之前或之后进行，以对整个加载历史中的损伤和失效进行建模。

详细说明请参见["创建XFEM裂缝"，第31.3.4节](pt04ch31s03hlb01.md)。
