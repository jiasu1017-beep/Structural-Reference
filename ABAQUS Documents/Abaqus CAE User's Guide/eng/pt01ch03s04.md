# 3.4 Managing objects







Managers are dialog boxes you use to manage all objects of a given type associated with the current model or session; examples of such objects include materials, parts, steps, display groups, and *X–Y* data objects. In addition, you can use the **Model Manager** to manage the models contained in the current model database.
