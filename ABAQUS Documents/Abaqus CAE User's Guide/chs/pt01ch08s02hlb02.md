# 8.2.2 选择要打印的图像部分





当你直接将图像打印到打印机或文件时，你可以使用**打印**对话框选择要包含在打印图像中的画布上的视口。你可以选择的选项如下：

**所有或当前视口**

选择**所有视口**打印画布上的所有视口。在画布上但由于超出绘图区域而不可见的视口仍会被打印。如果视口重叠，打印的图像将再现画布上的层级；也就是说，如果一个视口遮挡了另一个，被遮挡的部分将不会出现在打印图像中。默认情况下，Abaqus/CAE 打印画布上的所有视口。

选择**当前视口**仅打印最近使用的视口。有关选择视口的更多信息，请参阅["选择视口，" 第 4.3.2 节](pt01ch04s03hlb02.md)。

**注意：**无论你在**打印**对话框中做何选择，最小化的视口都不会被打印。

**视口装饰**

切换**打印视口装饰（如果可见）**以选择你的图像是否包含视口装饰。装饰定义为视口边框和视口标题。

**视口背景**

切换**打印视口背景**以控制打印图像中视口背景的外观。当此选项被选中且视口背景图像或电影处于活动状态时，Abaqus/CAE 在打印图像的背景中包含视口背景图像或电影的当前帧。如果**打印视口背景**处于活动状态但视口背景图像或电影处于非活动状态，Abaqus/CAE 会在背景中显示视口背景颜色。

**打印视口背景**选项仅在[为图像选择灰度或颜色](pt01ch08s02hlb03.md)时可用；当你选择黑白时，Abaqus/CAE 始终在白色背景上打印黑色图像。

**注意：**打印时不带视口背景（使背景显示为透明或白色）通常会产生最具吸引力的硬拷贝图像。

**视口罗盘**

切换**打印视口罗盘（如果可见）**以在打印图像中包含 3D 罗盘。如果罗盘在视口中当前不可见（请参阅["自定义 3D 罗盘，" 第 5.3.4 节](pt01ch05s03s04.md)），则无论此选项的设置如何，罗盘都不会出现在打印图像中。

**要选择要打印的图像部分：**

1. 从主菜单栏中，选择****File**![](../graphics/images/arrow.gif)**Print****。**提示：**你也可以单击**文件**工具栏中的![](../graphics/ico_filePrint.png)。**打印**对话框出现。
2. 从对话框顶部的**打印**字段中，选择：
   - **所有视口**打印所有视口，即使它们位于绘图区域之外。
   - **当前视口**打印你最近操作的视口。
3. 切换**打印视口装饰（如果可见）**。当**打印视口装饰（如果可见）**开启时，画布上可见的所有视口标题和边框都会被打印。当**打印视口装饰（如果可见）**关闭时，不会打印任何视口标题或边框。要打印当前视口但不带边框，你必须关闭此选项。
4. 切换**打印视口背景**。当**打印视口背景**开启时，你的图像将继承显示器上视口的背景颜色或背景图像。当**打印视口背景**关闭时，视口背景的外观取决于你为图像选择的格式：
   - 当你选择 **PS**（PostScript）或 **EPS**（封装的 PostScript）格式或直接打印到 Windows 打印机时，图像中的视口将具有白色背景。
   - 当你选择 **TIFF**、**PNG** 或 **SVG** 格式时，图像中的视口将具有透明背景。
5. 切换**打印视口罗盘（如果可见）**。当**打印视口罗盘（如果可见）**开启时，当前在视口中可见的任何 3D 罗盘将出现在你的打印图像中。当**打印视口罗盘（如果可见）**关闭时，3D 罗盘不会出现在你的打印图像中的任何位置。
6. 当你完成**打印**对话框时，单击**确定**生成所需的输出。Abaqus/CAE 生成输出并关闭**打印**对话框。你的**打印**对话框设置会在会话期间保存。
