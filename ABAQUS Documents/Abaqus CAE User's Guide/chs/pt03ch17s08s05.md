# 17.8.5 在凹边界附近使用结构化网格





当您使用任何网格技术对区域进行网格划分时，网格边界上的节点始终位于几何区域的边界上。但是，当Abaqus/CAE使用结构化网格技术创建网格时，网格内部的节点可能会落在区域几何形状之外，从而导致扭曲的无效网格。此问题通常发生在凹边界附近。

例如，图17-59中的区域有五条边；因此，当Abaqus/CAE使用结构化网格技术对该区域进行网格划分时，它会将正五边形的网格图案应用于该区域。

**图17-59** 正五边形的网格图案应用于该区域。

![](../graphics/mgn-concaveseed.png)

但是，如果您对区域进行布种以减少单元数量，如图17-60所示，则由于高度弯曲边处的凹面，会产生扭曲的网格。网格图案内部的节点（在图17-61中以实心圆表示）落在区域几何形状之外，而网格边界上的节点（在图17-61中以空心圆表示）保留在区域几何形状的边界上。

**图17-60** 指定较粗网格的种子。

![](../graphics/mgn-concave.png)

当内部节点落在区域几何形状之外时，您可以尝试以下技术来改进网格：
- 更改网格种子并重新网格划分。例如，图17-59中高度弯曲边上的单元数量大于图17-61中的数量。
**图17-61** 网格内部的节点落在区域几何形状之外。
![](../graphics/mgn-concaveprop.png)
- 将零件实例分割成更小、形状更规则的区域。例如，模型在图17-62中被分割成三个区域。
**图17-62** 分割区域。
![](../graphics/mgn-concavepart-nls.png)
- 选择不同的网格技术。此选项对二维区域最有用，您可以从结构化网格切换到自由网格，同时在网格中保留四边形单元。（三维自由网格仅限于四面体单元。更多信息，请参见["自由网格，"第17.10节](pt03ch17s10.md)。）图17-63展示了使用自由网格技术对该区域进行网格划分。
**图17-63** 使用自由网格技术对区域进行网格划分。
![](../graphics/mgn-concavefree.png)
图17-63中的网格不是对称的，这是自由网格的典型特征。
