# 16.8.4 编辑应用规定条件的区域





您可以在创建载荷、边界条件或预定义场的步骤中编辑应用规定条件的区域。如果规定条件的定义引用原始区域内的子区域（例如材料赋值预定义场），则无法编辑该区域。

**注意：**重力载荷可以应用于模型中的区域，但不能应用于Abaqus/CAE中的单个点质量。

**编辑应用规定条件的区域：**

1. 从主菜单栏中的**载荷**、**边界条件**或**预定义场**菜单中，选择**管理器**以显示**载荷管理器**、**边界条件管理器**或**预定义场管理器**。
2. 单击位于要修改的规定条件所在行和感兴趣步骤所在列交叉处的单元格，然后单击**编辑**。或者，您可以直接双击该单元格。**提示：**您也可以通过在上下文栏中的**步骤**列表中单击创建规定条件的步骤来启动此过程。从主菜单栏中的**载荷**、**边界条件**或**预定义场**菜单中，选择****编辑**![](../graphics/images/arrow.gif)***规定条件*****。例如，要编辑载荷，您可以选择****载荷**![](../graphics/images/arrow.gif)**编辑**![](../graphics/images/arrow.gif)***您选择的载荷*****。将出现一个编辑器。
3. 在编辑器顶部，单击![](../graphics/ico_selectBlue.png)编辑区域选择。
4. 使用以下方法之一编辑区域：- 在视口中选择和取消选择对象。完成编辑区域后，单击鼠标按钮2。（更多信息，请参见第6章"在视口中选择对象"。）**提示：**您可以通过在**选择**工具栏中指定过滤选项来限制可以在视口中选择的对象类型。请参见"使用选择选项"，第6.3节。- 若要从现有集合或表面列表中进行选择，请执行以下操作：1. 单击提示区域右侧的**集合**或**表面**。（按钮的名称取决于您正在编辑的对象类型。例如，如果您正在编辑压力载荷，则会出现**表面**按钮。）Abaqus/CAE显示**区域选择**对话框，其中包含可用集合或表面的列表。2. 选择感兴趣的集合或表面，然后单击**继续**。**注意：**默认选择方法基于您最近使用的选择方法。若要恢复为另一种方法，请单击提示区域右侧的**在视口中选择**或**集合**或**表面**。
5. 在编辑器中，根据需要完成编辑规定条件定义，然后单击**确定**。视口中代表规定条件的符号将更改为显示在新编辑的区域上。
