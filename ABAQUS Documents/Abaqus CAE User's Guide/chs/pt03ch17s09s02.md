# 17.9.2 表面的扫掠网格





Abaqus/CAE只能将扫掠网格技术应用于可以通过沿扫掠路径从源侧扫掠到目标侧来复制的表面区域。扫掠路径始终是一条边；但是，对于表面区域，源侧和目标侧也是边。表面区域可以是挤出的、旋转的、扫掠的或平面的。此外，挤出的表面区域可以包含扭转，旋转的表面区域可以包含平移。您可以使用**四边形**或**四边形主导**单元形状选项对表面区域应用扫掠网格技术。

如果您正在创建旋转扫掠网格，Abaqus/CAE会对源侧进行网格划分，并将该网格旋转到目标侧。扫掠路径可以旋转完整的360度。当源侧在一点接触旋转轴时，您必须使用**四边形主导**单元形状选项，因为会在该点生成一层三角形单元。当单个表面在两点接触旋转轴时，Abaqus/CAE无法生成旋转表面网格，除非旋转表面是球体。

例如，在图17-67所示的模型顶部，源侧接触旋转轴，并会在该点生成一层三角形单元。
