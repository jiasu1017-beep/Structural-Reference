# 47.4.31 平滑X–Y数据对象







### 47.4.31 平滑X–Y数据对象

使用 **smooth** 函数或 **smooth2** 函数对[之前保存的](pt05ch47s02hlb07.md) *X–Y* 数据对象（有序对的集合）进行运算，生成具有更平滑曲线的新 *X–Y* 数据对象。新的 *X–Y* 数据对象具有与原始 *X–Y* 数据对象相同的 *X* 坐标值。在每个 *X* 坐标处，Abaqus/CAE 将新 *Y* 坐标计算为原始数据对象相邻 *Y* 坐标的平均值。这种计算称为"移动平均"。这两个函数使用不同的算法来计算移动平均值：
- **smooth** 函数使用简单移动平均来计算新 *Y* 坐标值，这意味着每个新 *Y* 坐标是多个相邻点的未加权均值。您可以指定要包含在平均值中的相邻点数；较大的值会产生更平滑的曲线。Abaqus/CAE 将您指定的值解释为在当前 *X* 坐标两侧使用的点数。默认值为2，这意味着 Abaqus/CAE 平均5个值来计算每个新 *Y* 坐标。
- **smooth2** 函数使用指数移动平均来计算新 *Y* 坐标值，这意味着相邻点对结果 *Y* 坐标值的影响呈指数递增。您可以为曲线指定0到1之间的平滑因子；较小的值会产生更平滑的曲线。默认平滑因子为0.75。

**平滑X–Y数据对象：**

1. 找到 **Operate on XY Data** 对话框。从主菜单栏，选择 ****Tools**![](../graphics/images/arrow.gif)**XY Data**![](../graphics/images/arrow.gif)**Create****. 在出现的对话框中点击 **Operate on XY data**；然后点击 **Continue**。**Operate on XY Data** 对话框出现。
2. 从列出的 **Operators** 中，点击 **smooth(X, I)** 或 **smooth2(X, F)**。**smooth** 或 **smooth2** 函数出现在表达式窗口中。
3. 从 **XY Data** 选项中，点击要操作的 *X–Y* 数据对象的名称，然后点击 **Add to Expression**。您可以从本次会话中之前保存的所有 *X–Y* 数据对象中选择（在 **XY Data** 字段中按字母顺序列出）。*X–Y* 数据对象名称出现在表达式窗口中 **smooth** 或 **smooth2** 函数的括号内。
4. 将光标定位在表达式窗口中第二个逗号之前，然后输入以下值：- 对于 **smooth** 函数，输入一个整数值，表示在给定点的任一侧包含在平均值中的点数。- 对于 **smooth2** 函数，输入0到1之间的小数值来确定曲线的平滑因子。较小的值会产生更平滑的曲线。
5. 要继续构建表达式，将光标定位在表达式窗口中，然后输入或选择要包含的函数、运算符和 *X–Y* 数据。
6. 要评估并显示表达式，点击 **Plot Expression**。
7. 要[保存](pt05ch47s02hlb07.md)您的新 *X–Y* 数据对象，点击 **Save As**，然后在出现的对话框中提供名称。保存数据对象使其可用于本次会话中的未来操作以及包含多个数据对象的 *X–Y* 曲线图。
8. 完成后，点击 **Cancel** 关闭对话框。
