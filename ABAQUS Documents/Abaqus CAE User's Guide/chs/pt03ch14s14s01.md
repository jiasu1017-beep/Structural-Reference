# 14.14.3 指定ALE自适应网格的控制





您可以为应用于自适应网格域的自适应网格和平流算法指定控制。更多信息，请参见「Abaqus/Explicit中的ALE自适应网格和重映射」，Abaqus Analysis User's Guide第12.2.3节，以及「Abaqus/Standard中的ALE自适应网格和重映射」，Abaqus Analysis User's Guide第12.2.7节。

以下章节提供指定ALE自适应网格控制的说明：
- 「为Abaqus/Standard分析指定ALE自适应网格控制」，见「指定ALE自适应网格的控制」，第14.14.3节
- 「为Abaqus/Explicit分析指定ALE自适应网格控制」，见「指定ALE自适应网格的控制」，第14.14.3节

### 为Abaqus/Standard分析指定ALE自适应网格控制

您可以使用「Edit ALE Adaptive Mesh Controls」对话框来自定义应用于自适应网格域的自适应网格和平流算法的各个方面。更多信息，请参见「Abaqus/Standard中的ALE自适应网格和重映射」，Abaqus Analysis User's Guide第12.2.7节。

**指定ALE自适应网格控制的步骤：**

1. 从主菜单栏选择 ****Other**![](../graphics/images/arrow.gif)**ALE Adaptive Mesh Controls**![](../graphics/images/arrow.gif)**Create****。此时会出现「Create ALE Adaptive Mesh Controls」对话框。
2. 在对话框中为ALE自适应网格控制输入名称，然后点击「Continue」。此时会出现「Edit ALE Adaptive Mesh Controls」对话框。
3. 点击「Smoothing algorithm」字段右侧的箭头，并选择计算自适应网格平滑的方法：- 选择「Determined by analysis product」可以接受您正在使用的分析产品的默认值（在本例中为Abaqus/Standard）。- 选择「Enhanced algorithm based on evolving geometry」可以使用几何增强形式的基本平滑算法作为缓解畸变的技术。这些形式是启发式的，仅基于节点位置。由于其启发式特性，几何增强可能并不总是能改善网格平滑。- 选择「Conventional smoothing」可以使用常规形式的平滑算法。有关自适应网格平滑的更多信息，请参见「Abaqus/Standard中的ALE自适应网格和平流」，Abaqus Analysis User's Guide第12.2.7节中的「ALE自适应网格扫描算法」。
4. Abaqus/Standard中新网格的计算基于两种基本平滑方法的某种组合：原始构型投影平滑和体积平滑。您必须在「Volumetric」和「Original configuration projection」字段中为每种方法指定权重因子。更多信息，请参见「Abaqus/Standard中的ALE自适应网格和平流」，Abaqus Analysis User's Guide第12.2.7节中的「组合平滑方法」。
5. 在「Initial feature angle」字段中，输入初始几何特征角，![](../graphics/usi_eqn00434.gif)，以度为单位（![](../graphics/usi_eqn00435.gif)）。此角度用于检测几何边缘和角落。默认值为![](../graphics/usi_eqn00436.gif)。设置![](../graphics/usi_eqn00437.gif)将确保不检测或强制执行任何几何边缘或角落。更多信息，请参见「在Abaqus/Standard中定义ALE自适应网格域」，Abaqus Analysis User's Guide第12.2.6节中的「控制几何边缘和角落的检测」。
6. 在「Transition feature angle」字段中，输入过渡几何特征角，![](../graphics/usi_eqn00438.gif)，以度为单位（![](../graphics/usi_eqn00439.gif)）。此角度用于确定何时应停用几何边缘和角落以允许跨它们进行重网格划分。默认值为![](../graphics/usi_eqn00440.gif)。设置![](../graphics/usi_eqn00441.gif)将确保不停用任何几何边缘或角落。更多信息，请参见「在Abaqus/Standard中定义ALE自适应网格域」，Abaqus Analysis User's Guide第12.2.6节中的「控制几何边缘和角落的激活和停用」。
7. 点击「OK」保存命名控制集并关闭编辑器。

### 为Abaqus/Explicit分析指定ALE自适应网格控制

您可以使用「Edit ALE Adaptive Mesh Controls」对话框来自定义应用于自适应网格域的自适应网格和平流算法的各个方面。更多信息，请参见「Abaqus/Explicit中的ALE自适应网格和重映射」，Abaqus Analysis User's Guide第12.2.3节。

**指定ALE自适应网格控制的步骤：**

1. 从主菜单栏选择 ****Other**![](../graphics/images/arrow.gif)**ALE Adaptive Mesh Controls**![](../graphics/images/arrow.gif)**Create****。此时会出现「Create ALE Adaptive Mesh Controls」对话框。
2. 在对话框中为ALE自适应网格控制输入名称，然后点击「Continue」。此时会出现「Edit ALE Adaptive Mesh Controls」对话框。
3. 选择网格平滑的「Priority」：- 选择「Improve aspect ratio」可以执行自适应网格划分，以最小化单元畸变并改善单元长宽比，代价是扩散初始网格梯度。此目标推荐用于具有中等到大整体变形的问题。更多信息，请参见「Abaqus/Explicit中的ALE自适应网格和平流」，Abaqus Analysis User's Guide第12.2.3节中的「指定均匀网格平滑目标」。- 选择「Preserve initial mesh grading」可以执行自适应网格划分，在分析演进过程中尝试保持初始网格梯度同时减少畸变。此目标仅推荐用于具有合理结构化梯度网格的自适应网格域，这些网格经历低到中等整体变形。更多信息，请参见「Abaqus/Explicit中的ALE自适应网格和平流」，Abaqus Analysis User's Guide第12.2.3节中的「指定梯度网格平滑目标」。
4. 点击「Smoothing algorithm」字段右侧的箭头，并选择计算自适应网格平滑的方法：- 选择「Determined by analysis product」可以接受您正在使用的分析产品的默认值（在本例中为Abaqus/Explicit）。- 选择「Enhanced algorithm based on evolving geometry」可以使用几何增强形式的基本平滑算法作为缓解畸变的技术。- 选择「Conventional smoothing」可以使用常规形式的平滑算法。有关自适应网格平滑的更多信息，请参见「Abaqus/Explicit中的ALE自适应网格和平流」，Abaqus Analysis User's Guide第12.2.3节中的「网格平滑方法」。
5. 选择「Meshing predictor」选项：- 选择「Current deformed position」可以基于当前节点位置执行自适应网格划分。此方法推荐用于所有类Lagrangian问题和具有非常大畸变的问题。更多信息，请参见「Abaqus/Explicit中的ALE自适应网格和平流」，Abaqus Analysis User's Guide第12.2.3节中的「在Lagrangian域中定位节点」。- 选择「Position from previous ALE adaptive mesh increment」可以基于上一个自适应网格增量结束时的节点位置执行自适应网格划分。此技术推荐用于Eulerian类问题，其中材料流动相对于整体变形是显著的。更多信息，请参见「Abaqus/Explicit中的ALE自适应网格和平流」，Abaqus Analysis User's Guide第12.2.3节中的「在Eulerian域中定位节点」。
6. 在「Curvature refinement」字段中，输入曲率细化权重值，![](../graphics/usi_eqn00442.gif)。合适的值允许您确保在高度弯曲的边界附近有足够的网格细化。默认值，![](../graphics/usi_eqn00443.gif)，适用于各种问题。更多信息，请参见「Abaqus/Explicit中的ALE自适应网格和平流」，Abaqus Analysis User's Guide第12.2.3节中的「基于凹边界曲率的相关网格」。
7. Abaqus/Explicit中新网格的计算基于三种基本平滑方法的某种组合：体积平滑、Laplacian平滑和等势平滑。您必须在「Volumetric」、「Laplacian」和「Equipotential」字段中为每种方法指定权重因子。更多信息，请参见「Abaqus/Explicit中的ALE自适应网格和平流」，Abaqus Analysis User's Guide第12.2.3节中的「网格平滑方法」。
8. 在「Initial feature angle」字段中，输入初始几何特征角，![](../graphics/usi_eqn00434.gif)，以度为单位（![](../graphics/usi_eqn00435.gif)）。此角度用于检测几何边缘和角落。默认值为![](../graphics/usi_eqn00436.gif)。设置![](../graphics/usi_eqn00437.gif)将确保不检测或强制执行任何几何边缘或角落。
9. 在「Transition feature angle」字段中，输入过渡几何特征角，![](../graphics/usi_eqn00438.gif)，以度为单位（![](../graphics/usi_eqn00439.gif)）。此角度用于确定何时应停用几何边缘和角落以允许跨它们进行重网格划分。默认值为![](../graphics/usi_eqn00440.gif)。设置![](../graphics/usi_eqn00441.gif)将确保不激活任何几何边缘或角落。
10. 在「Mesh constraint angle field」中，输入网格约束角，![](../graphics/usi_eqn00444.gif)，以度为单位（![](../graphics/usi_eqn00445.gif)）。默认值为![](../graphics/usi_eqn00446.gif)。当自适应网格约束应用于Lagrangian或滑动边界区域上的节点时，如果边界区域法线与规定约束方向之间的角度变得小于![](../graphics/usi_eqn00444.gif)，分析将终止。当自适应网格约束应用于属于Lagrangian或活动几何边缘一部分的节点时，如果规定约束与垂直于边缘的平面之间的角度变得小于![](../graphics/usi_eqn00444.gif)，分析将终止。
11. 选择在执行自适应网格划分后对解变量进行重映射的算法：- 选择「First order」使用基于供体单元差分的一阶方法。- 选择「Second order」使用基于Van Leer工作的二阶方法。更多信息，请参见「Abaqus/Explicit中的ALE自适应网格和平流」，Abaqus Analysis User's Guide第12.2.3节中的「单元变量的平流方法」。
12. 选择平流动量的方法：- 选择「Element center projection」作为最便宜的方法。- 选择「Half-index shift」作为一种计算成本更高但可能表现出更好色散特性的方法。更多信息，请参见「Abaqus/Explicit中的ALE自适应网格和平流」，Abaqus Analysis User's Guide第12.2.3节中的「动量平流」。
13. 点击「OK」保存命名控制集并关闭编辑器。
