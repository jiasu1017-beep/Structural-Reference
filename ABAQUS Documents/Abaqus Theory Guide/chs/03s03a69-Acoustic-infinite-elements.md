# 3.3.2 声学无限单元

### 3.3.2 声学无限单元

**产品：** Abaqus/Standard  Abaqus/Explicit

在声学分析中，在无界域中的问题，或者感兴趣区域相对于周围介质较小的问题，是常见的。例如，深水中的潜艇可能经历来自流体的载荷，并将声音辐射到流体中，就好像海洋是无限大的。外部流体可以被认为是"无界"或"无限"的程度取决于物体或感兴趣区域与最近边界之间的波长数量：这个数字越高，这些边界的影响越小到可以忽略的可能性越大。例如，周围流体对相对较浅港口中的潜艇的影响在高频时可能类似于无限介质，但在较低频率时港口底部和自由表面可能产生影响。类似地，空气中的扬声器可能从高频扬声器辐射声音，就好像周围的空气是无限的一样，但房间墙壁的影响可能影响低频扬声器的辐射模式。

Abaqus提供了一系列功能来模拟外部流体效应。周围流体都可以用有限元建模；显然，这仅在周围介质的范围较小时才实际可行。在下一级复杂程度中，用户可以对一小部分流体进行建模，并在终止表面施加简单的辐射边界条件。这些辐射边界条件使用通过边界的波的简单模型推导。Abaqus提供了这些边界条件的几种替代公式（参见"耦合声学-结构介质分析，" 第2.9.1节，以及"Abaqus Analysis User's Guide"第6.10.1节"声学、冲击和耦合声学-结构分析"）。这些辐射条件不会向系统添加自由度，也不会影响矩阵的对称性。

最后，提供了声学无限单元，允许保留的有限元流体区域更小，同时具有类似的精度。声学无限单元公式在几个关键方面不同于辐射边界条件公式。在无限单元中，无限外部被细分为单元，并以与常规有限元方法完全类似的方式对单元强制执行加权残差声明。自由度，对应于无限方向上的插值函数，被添加到整体矩阵系统中。Abaqus中使用的加权残差方法导致非对称无限单元矩阵，因此这些单元的相对成本高于简单辐射边界条件。然而，无限单元的精度足够高，使得有限元区域可以大大减小，在许多应用中抵消了成本。
