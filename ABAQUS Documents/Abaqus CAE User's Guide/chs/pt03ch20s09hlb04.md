# 20.9.4 自定义工作表大小和网格







使用**Sketcher 选项**对话框来控制工作表大小和网格。您可以自定义以下内容：

****工作表大小****

工作表的边界始终为正方形，其高度和宽度等于工作表大小。如果您发现工作表大小太大或太小，可以使用**Sketcher 选项**对话框更改大小。

当您创建零件或独立草图时，您需要定义新零件或草图的大致大小；Abaqus/CAE 根据您提供的大致大小确定初始工作表大小。大小必须在 10–3 到 105 单位之间。Abaqus/CAE 不使用特定单位，但单位在整个模型中必须一致。**自动**切换按钮位于**工作表大小**字段旁边，用于控制 Abaqus/CAE 是否可以自动更改当前草图和您创建的新草图的工作表大小。

****网格间距****

您可以使用此选项使用定义**工作表大小**的相同单位来更改主网格线的网格间距。如果[**捕捉到网格**](pt03ch20s09hlb02.md)已启用，光标将捕捉到每个网格点。**自动**切换按钮位于**网格间距**字段旁边，用于控制 Abaqus/CAE 是否可以自动更改当前草图和您创建的新草图的网格大小。

****次间隔****

次网格线将主网格线之间的空间细分为间隔，使您在将 Sketcher 设置为高放大倍数时能够更精确地定位草图中的项目。您可以使用此选项更改每个主网格线之间的次间隔数。

次网格线的显示是动态的：Abaqus/CAE 会在默认放大倍数下隐藏这些线，并在您放大时显示它们。

****显示网格线****

您可以在 Sketcher 中切换主网格线和次网格线的显示。次网格线仅在主网格线也显示时才能显示。

工作表大小和网格自定义选项适用于当前草图，并与草图一起存储。当您创建新草图时，Abaqus/CAE 使用最新设置来确定是否重新计算工作表大小和网格间距以及是否显示网格。

**自定义工作表大小和网格：**

1. 从 Sketcher 工具箱底部，选择 Sketcher 自定义工具![](../graphics/ico_sketchOptions-nls.png)。出现**Sketcher 选项**对话框。
2. 从**常规**选项卡页面，关闭**工作表大小**文本框旁边的**自动**。
3. 在文本框中，输入将包含正在创建或编辑的特征的方形工作表的尺寸。以与用于描述模型其余部分的单位一致的单位指定工作表大小。
4. 关闭**网格间距**文本框旁边的**自动**。
5. 在文本框中，输入网格点之间所需的间距。以与定义工作表大小相同的单位指定网格间距。
6. 点击**次间隔**字段右侧的箭头增加或减少每个主网格线之间显示的次网格间隔数。
7. 切换**主**以显示或隐藏主网格线。如果显示了主网格线，您也可以切换**次**以显示或隐藏次网格线。
8. 选择完所需的自定义选项后，点击**确定**应用您的更改并关闭**Sketcher 选项**对话框。
