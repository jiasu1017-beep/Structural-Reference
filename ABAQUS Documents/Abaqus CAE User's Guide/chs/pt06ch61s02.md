# 61.2 配置键盘快捷键





键盘快捷键是您可以用来在 Abaqus/CAE 中启动功能的单个按键或组合键；例如，**[Ctrl]**+S 保存当前模型。默认情况下为常见功能定义了几个键盘快捷键。您可以使用 Customize 工具集来编辑这些默认快捷键，为 Abaqus/CAE 中的其他功能添加新的快捷键，以及显示当前定义的所有快捷键的列表。从主菜单栏选择 ****Tools**![](../graphics/images/arrow.gif)**Customize****；显示出现的对话框的 **Functions** 选项卡页面以访问 **Keyboard** 编辑器。

**提示：**键盘快捷键仅在其功能在当前模块中允许时有效，并且仅在应用程序焦点不在模型树、结果树、消息区域或命令行界面中时有效。如果快捷键似乎不起作用，请确认该功能在当前模块中允许，或通过单击视口标题栏将焦点移至视口。

Abaqus/CAE 允许将以下按键和按键组合指定为键盘快捷键：
- 除了 **F1** 键之外的任何功能键，
- **[Alt]**+[Shift]**+任意键**，或
- **[Ctrl]**+任意键。您还可以在任何包含 **[Ctrl]** 键的键盘快捷键中添加 **[Alt]** 或 **[Shift]**。

但是，并非所有键都符合加入键盘组合的条件；您不能指定包含功能键或字母数字键盘上键的键盘组合。
