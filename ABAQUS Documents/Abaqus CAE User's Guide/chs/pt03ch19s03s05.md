# 19.3.4 结合使用自动和手动网格自适应





在许多情况下，您会希望在自适应重新网格划分的自动使用和手动使用之间进行切换。一些示例包括：
- 您必须保持当前的 Abaqus/CAE 会话处于活动状态，以便在分析作业之间进行自动重新网格划分。如果您退出 Abaqus/CAE 并启动新会话，则必须使用手动重新网格划分来重新网格划分模型。
- 您启动了一系列自动作业，但机器问题（例如磁盘已满）提前中断了分析。您可以手动重新启动分析，然后以完全自动的方式继续分析。这种情况的变体包括因收敛失败而结束的分析，或因您在特定迭代期间终止作业而结束的分析。
- 您可以使用尺寸函数参数进行实验，并手动重新网格划分模型，直到找到一组似乎正在积极响应您感兴趣变量的重新网格划分规则。当您对重新网格划分规则感到满意后，可以切换到自动网格自适应并完成指定数量的分析迭代。
- 自动网格自适应过程生成的网格收敛到可接受的配置。您现在可以修改应用于模型的载荷，并执行额外的手动迭代来进行小的网格调整以补偿新的载荷。

您可以在自动和手动网格自适应之间直接切换。要从自动网格自适应切换到手动网格自适应，您进入网格模块并从主菜单栏选择 ****自适应**![](../graphics/images/arrow.gif)**手动自适应重新网格划分**** 并重新网格划分模型。然后返回作业模块并提交作业进行分析。相反，要从手动网格自适应切换到自动网格自适应，您可以在作业模块中创建自适应过程并提交进行分析。
