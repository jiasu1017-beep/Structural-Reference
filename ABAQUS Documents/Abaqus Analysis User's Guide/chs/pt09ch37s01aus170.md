### 概述

使用接触对模拟的压力渗透载荷：
- 模拟流体在两个接触结构之间的渗透；并且
- 允许流体从表面多个位置渗透。

### 定义接触体之间的压力渗透载荷

分布压力渗透载荷允许模拟流体渗透到两个接触体之间的表面，并施加垂直于表面的流体压力。使用基于单元的接触表面来模拟物体之间的相互作用（参见["接触相互作用分析：概述，" 第36.1.1节](pt09ch36s01abo33.md)）。表面被建模为从属接触表面和主控接触表面（参见["Abaqus/Standard中的接触公式，" 第38.1.1节](pt09ch38s01aus177.md)）。

可以使用任何接触公式。

形成接头的物体可以都是变形的，如螺纹连接器的情况；或者一个是刚性的，如使用软垫圈作为较硬结构之间的密封的情况。您需要指定暴露于流体压力的节点、流体压力的大小以及流体渗透开始发生的临界接触压力。更多信息请参见["基于表面的接触的压力渗透加载，" Abaqus Theory Guide 第6.4.1节](../stm/stm-link.md#stm-ldc-presspenetrationsurf)。

| **输入文件用法：** | ``` [*PRESSURE PENETRATION](../key/key-link.md#usb-kws-hpressurepenetration), SLAVE=*slave1*, MASTER=*master1* *slave surface node or node set*, *master surface node or node set*, *magnitude*, *critical contact pressure* ``` |
| --- | --- |
|  | 如果指定了节点集，在二维中只能包含一个节点；在三维中可以包含任意数量的节点。 |

| **Abaqus/CAE用法：** | Interaction模块：**Create Interaction**：**Surface-to-surface contact (Standard)**，**Name：** *contact_interaction_name*；选择主表面和从表面**Create Interaction**：**Pressure penetration**；**Contact interaction：** *contact_interaction_name*，**Region on Master：** 选择面、边或点，**Region on Slave：** 选择面、边或点，**Critical Contact Pressure：** *critical contact pressure*，**Fluid Pressure：** *magnitude* |
| --- | --- |

#### 指定压力渗透准则

使用单个基于从属节点的渗透准则。流体会从一个或多个暴露于流体的位置渗透到接触体之间的表面，直到达到接触压力大于指定临界值的点，从而切断流体的进一步渗透。

#### 指定流体压力的渗透时间

当流体压力渗透准则满足时，流体压力被施加到垂直于表面的位置。如果立即施加完整的当前流体压力，则接触表面附近应变的大变化可能导致收敛困难。对于大应变问题，也可能发生严重的网格扭曲。为确保解的平滑性，流体压力在从零压力渗透载荷到当前完整大小的时间段内线性增加。

您可以指定流体压力渗透载荷在新渗透的表面段上达到当前完整大小所需的时间。如果在渗透后立即测量的累积增量大小大于渗透时间，则将施加完整的当前流体压力渗透载荷；否则，新渗透表面段上的流体压力将在渗透时间内线性增加到当前大小，可能在多个增量中。当渗透时间等于0时，一旦满足流体压力渗透准则，立即施加当前流体压力。默认渗透时间为总步长时间的0.001。在线性扰动分析中忽略渗透时间。

| **输入文件用法：** | ``` [*PRESSURE PENETRATION](../key/key-link.md#usb-kws-hpressurepenetration), PENETRATION TIME=*n* ``` |
| --- | --- |

| **Abaqus/CAE用法：** | Interaction模块：**Create Interaction**：**Pressure penetration**；**Penetration time：** *n* |
| --- | --- |

#### 指定暴露于流体压力的节点

流体可以从表面的一个或多个位置渗透。您必须识别接触体从表面上的节点或节点集，以定义表面暴露于流体压力的位置。在二维中，如果主表面不是解析刚性表面（参见["解析刚性表面定义，" 第2.3.4节](pt01ch02s03aus19.md)），您还必须在主表面上识别节点或节点集，以定义表面暴露于流体压力的位置。如果表面有多个位置暴露于流体，您可以指定多个节点或节点集。这些节点或节点集始终受到压力渗透载荷的影响，无论其接触状态如何。然后，流体开始从这些节点或节点集渗透到两个接触体之间的表面。

#### 指定施加的流体压力

您必须定义流体压力的参考大小。您可以通过参考振幅曲线来定义步中流体压力的变化。默认情况下，参考大小在步开始时立即施加或在线性增加，取决于分配给步的振幅变化（参见["定义分析，" 第6.1.2节](pt03ch06s01abo05.md)）。

流体压力渗透载荷将基于增量开始时的压力渗透准则施加到单元表面，即使在该增量期间流体进一步渗透，也将保持在该增量期间不变。在二维中使用节点积分方案将分布流体压力渗透载荷集成到单元上，而在三维中使用高斯积分方案；分布流体压力在单元上的变化将由单元节点上的载荷大小决定。

##### 移除或修改压力渗透载荷

将压力渗透载荷施加到单元表面后，即使重新建立表面之间的接触，也不会自动移除它们。然而，在每个新步中，可以以类似于定义分布载荷的方式修改或完全重新定义流体压力渗透载荷（参见["施加载荷：概述，" 第34.4.1节](pt07ch34s04aus120.md)）。

| **输入文件用法：** | 使用以下选项修改在前面的步中施加的流体压力渗透载荷： |
| --- | --- |
|  | ``` [*PRESSURE PENETRATION](../key/key-link.md#usb-kws-hpressurepenetration), OP=MOD (default) ``` 在这种情况下，数据行上必须指定暴露于流体压力的从属节点。如果主表面不是解析刚性表面，对于平面或轴对称模型，数据行上还必须指定暴露于流体压力的主节点。使用以下选项移除所有流体压力渗透载荷，并可选择指定新的流体压力渗透载荷： ``` [*PRESSURE PENETRATION](../key/key-link.md#usb-kws-hpressurepenetration), OP=NEW ``` 当使用OP=NEW移除所有流体压力渗透载荷时，不需要数据行。但是，当使用OP=NEW指定新的流体压力渗透载荷时，数据行上必须指定暴露于流体压力的节点。定义新的暴露节点时必须使用OP=NEW。此外，当使用OP=NEW重新指定先前定义的压力渗透载荷时，流体压力载荷将首先恢复到最后已知的配置，即使接触状态随后已更改。 |

|  | Interaction模块：**Interaction Manager**：选择相互作用，**Edit** 使用以下选项移除在前面的步中施加的流体压力渗透：Interaction模块：**Interaction Manager**：选择相互作用，**Deactivate** |

#### 指定临界机械接触压力

为了考虑接触表面上的微凸体，引入了一个临界接触压力，低于该压力时流体渗透开始发生。该值越高，流体越容易渗透。临界接触压力的默认值为零，在这种情况下，只有在失去接触时才会发生流体渗透。

#### 在线 性扰动分析中的使用

可以通过在常规分析步之间包含线性扰动步来随时执行完全非线性分析中的线性扰动分析。因为接触条件在线性扰动分析期间不能改变，所以流体不会进一步渗透到表面，它保持为在基态中定义的样子。然而，在先前的常规分析步中施加的流体压力大小可以在线性扰动分析步中修改。在矩阵生成（参见["生成矩阵，" 第10.3.1节](pt04ch10s03at32.md)）和稳态动态分析（直接或模态——参见["直接解稳态动态分析，" 第6.3.4节](pt03ch06s03at09.md)，和["基于模态的稳态动态分析，" 第6.3.8节](pt03ch06s03at13.md)）中，您可以指定载荷的实部（同相）和虚部（异相）。

#### 压力渗透载荷的限制

每个承受压力渗透载荷的从属表面必须是连续的，不能是闭合环。压力渗透载荷不能与基于节点的从属表面一起使用。在任何增量中施加的压力渗透载荷基于该增量开始时的接触状态。因此，在接触状态发生变化的增量结束时解释结果时应小心。建议使用小时间增量以获得准确的结果。

当流体在不同压力下同时从表面上多个位置渗透到接触体之间时，单元上施加流体压力载荷的最大值。

当流体在解析刚性表面和变形表面之间渗透到接触体时，不会向解析刚性表面施加压力渗透载荷。因此，解析刚性表面上的参考节点应在所有方向上受到约束。为了考虑流体压力渗透载荷对刚性表面的影响，解析刚性表面应用基于单元的刚性表面替换。

在大位移分析中，压力渗透载荷引入非对称载荷刚度矩阵项。使用分析步的非对称矩阵存储和求解方案可以提高平衡迭代的收敛率。参见["定义分析，" 第6.1.2节](pt03ch06s01abo05.md)，了解非对称矩阵存储和求解方案的更多信息。

仅支持实体、壳、圆柱和刚性单元进行三维压力渗透。
