# 16.9.6 定义管道压力载荷





您可以创建此类载荷以在管道或弯头中指定内部或外部压力。

**创建或编辑管道压力载荷：**

1. 使用以下方法之一显示管道压力载荷编辑器：- 若要创建新的管道压力载荷，请遵循"创建载荷"，第16.8.1节中概述的过程（**类别**：`Mechanical`；**所选步骤的类型**：`Pipe pressure`）。- 若要使用菜单或管理器编辑现有的管道压力载荷，请参见"编辑依赖于步骤的对象"，第3.4.13节。若要编辑应用载荷的区域，请参见"编辑应用规定条件的区域"，第16.8.4节。
2. 选择您选择的**侧面**选项：- 选择**内部**以在管道内指定内部压力。- 选择**外部**以在管道上指定外部压力。
3. 在**有效直径**字段中，输入适当的管道直径：- 如果您在上一步中选择了**内部**，请输入管道的内径。- 如果您在上一步中选择了**外部**，请输入管道的外径。**注意：**您输入的有效直径在整个分析过程中保持不变。即使在打开**Nlgeom**设置时，它也不会随管道在压力下膨胀或收缩而缩放。（有关**Nlgeom**设置的更多信息，请参见"考虑几何非线性"，第14.9.6节。）
4. 单击**分布**字段右侧的箭头，并从出现的列表中选择您的选项：- 选择**均匀**以定义在管道表面上均匀的载荷。- 选择**静水压**以定义管道上或管道内的静水压力。- 选择**用户定义**以在用户子程序[`DLOAD`](../sub/sub-link.md#sub-xsl-dload)中定义载荷幅值。有关更多信息，请参见以下章节：- "指定常规作业设置"，第19.8.6节- "Abaqus User Subroutines Reference Guide"第1.1.5节"DLOAD"- 选择分析场以定义空间变化载荷。只有对该载荷类型有效的分析场才会在选择列表中显示。或者，您可以单击![](../graphics/ico_analyticalField.png)创建新的分析场。（更多信息，请参见第58章"分析场工具集"。）
5. 如果您选择了**均匀**或分析场分布选项，请执行以下步骤：1. 在**幅值**文本字段中，输入压力幅值（单位[FL<sup>-2</sup>](../popups/usb-int-iconventions-unitsym.md)）。2. 如果需要，单击**幅值**字段右侧的箭头，并从出现的列表中选择您选择的幅值。或者，您可以单击![](../graphics/ico_amplitudeSmall.png)创建新的幅值。（更多信息，请参见第57章"幅值工具集"。）3. 单击**确定**保存您的数据并退出编辑器。
6. 如果您选择了**静水压**分布选项，请执行以下步骤：1. 在**幅值**文本字段中，输入压力幅值（单位[FL<sup>-2</sup>](../popups/usb-int-iconventions-unitsym.md)）。2. 在**零压力高度**字段中，输入压力为零处的高度*Z*坐标。3. 在**参考压力高度**字段中，输入压力为**幅值**字段中指定数值处的高度*Z*坐标。（更多信息，请参见"Abaqus Analysis User's Guide"第34.4.3节"Distributed loads"中的"Abaqus/Standard中的二维、三维和轴对称单元上的静水压力载荷"。）
7. 如果您选择了**用户定义**分布选项，请执行以下步骤：1. 如果需要，在**幅值**字段中输入压力幅值（单位[FL<sup>-2</sup>](../popups/usb-int-iconventions-unitsym.md)）。您在编辑器中输入的幅值数据被传递到用户子程序中。2. 单击**确定**保存您的数据并退出编辑器。3. 进入作业模块并显示感兴趣的分析作业的作业编辑器。（更多信息，请参见"创建、编辑和操作作业"，第19.7节。）4. 在作业编辑器中，单击**常规**选项卡，并指定包含用户子程序[`DLOAD`](../sub/sub-link.md#sub-xsl-dload)的文件。更多信息，请参见"指定常规作业设置"，第19.8.6节。**注意：**您只能在作业编辑器中指定一个用户子程序文件；如果您的分析涉及多个用户子程序，您必须将用户子程序合并到一个文件中，然后指定该文件。
