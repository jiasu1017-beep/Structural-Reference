# *NODE









### *NODE指定节点坐标。

此选项用于通过指定其坐标直接定义节点。如果在使用此选项时[*SYSTEM](ch18abk59.md)选项生效，则此选项中给出的节点坐标在局部系统中。

**产品：**Abaqus/Standard  Abaqus/Explicit  Abaqus/CFD  Abaqus/CAE  

**类型：**模型数据  

**级别：**部件、部件实例、装配  

**Abaqus/CAE：**网格模块

##### **参考：**

- ["节点定义，" Abaqus Analysis User's Guide第2.1.1节](../usb/usb-link.md#usb-int-inode)

### **可选参数：**

INPUT

将此参数设置为包含此选项数据行的备用输入文件名称。如果省略此参数，则假定数据在关键字行之后。

NSET

将此参数设置为其节点将被分配到的节点集名称。使用此选项创建或修改的节点集始终是排序的。

SYSTEM

设置SYSTEM=R（默认）以在矩形笛卡尔坐标系中给出坐标。设置SYSTEM=C以在圆柱坐标系中给出坐标。设置SYSTEM=S以在球坐标系中给出坐标。

SYSTEM参数完全局部于此选项。当读取数据行时，给出的坐标会立即转换为矩形笛卡尔坐标。如果[*SYSTEM](ch18abk59.md)选项也生效，则这些是局部矩形笛卡尔坐标，然后立即转换为全局笛卡尔坐标。
