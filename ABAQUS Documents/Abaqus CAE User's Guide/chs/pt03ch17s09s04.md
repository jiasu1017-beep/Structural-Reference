# 17.9.4 圆柱实体的扫掠网格





您可以使用扫掠网格来对圆柱几何体创建网格，使用多种类型的实体单元，包括来自实体圆柱单元库中的单元。如果您对原生网格使用实体圆柱单元，Abaqus/CAE使用所选扫掠/旋转区域的扫掠路径作为生成圆柱单元的周向方向。在创建网格之前，您应该确认扫掠路径设置正确。

当您尝试使用圆柱单元对实体几何体进行网格划分时，Abaqus会考虑圆柱单元的以下要求：
- 圆柱单元圆形边上的中边节点必须恰好放置在边缘节点的中间位置。
- 横截面上的所有节点必须位于同一径向平面内。
- 单元内所有圆形边必须跨越相同的角度。
- 单元内圆形弧的中心必须位于一条公共轴线上。
- 一个单元不能跨越超过180度。

这些要求意味着您只能在旋转零件或可以分割成旋转域的零件上创建有效的圆柱单元。图17-81中S形挤出物在分割成旋转区域后可以使用圆柱单元进行网格划分。

**图17-81** 实体圆柱网格的可接受几何体。

![](../graphics/mgn-sweep-cyl-solids-s-shaped.png)

图17-82展示了三种类型的几何体——螺旋弹簧、带切割的旋转零件和锥形零件——由于这些网格划分要求，无法使用实体圆柱单元进行网格划分。

**图17-82** 无法使用圆柱单元进行扫掠网格的几何体形式。

![](../graphics/mgn-sweep-cyl-solids-unavail.png)

如果由于约束网格的内部边，连接旋转区域中源面到目标面的连接边之一被分割成多个面，则生成的单元可能过于扭曲而无法用作圆柱单元。您必须将实体区域分割成更简单的旋转区域，如图17-83右侧示例所示。

**图17-83** 不适合使用圆柱单元进行网格划分的实体区域（左），以及通过使用分割使其适合使用圆柱单元进行网格划分的相同区域（右）。

![](../graphics/mgn-sweep-cyl-partition.png)

当您通过分配圆柱单元来编辑网格零件时，Abaqus/CAE认为每个圆柱单元的面1和面2是沿其径向平面的面。图17-84指示了几种圆柱单元类型这些面的位置。在分配圆柱单元类型之前，您应该为零件中的单元定向栈方向；更多信息，请参见["定向栈方向，"第64.6.4节](pt06ch64s06s04.md)中的描述。
