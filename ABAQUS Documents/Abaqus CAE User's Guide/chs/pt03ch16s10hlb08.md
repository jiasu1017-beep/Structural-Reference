# 16.10.9 定义次级基础运动边界条件





您可以在频率提取步骤中创建次级基础运动边界条件。如果后续的模态动态或稳态动态（基于模态或基于子空间）步骤有多个独立基础运动，则驱动节点必须除了主基础外还要分组到次级基础中。关于基础运动边界条件的完整讨论，请参见《Abaqus Analysis User's Guide》第6.3.5节"自然频率提取"中的"定义模态叠加过程的基和次级基"。

**要创建或编辑次级基础边界条件：**

1. 使用以下方法之一显示边界条件编辑器：- 要创建新的次级基础边界条件，请按照"创建边界条件"中的步骤操作（第16.8.2节）（**类别**：`Mechanical`；**所选步骤的类型**：`Secondary base`）。- 要使用菜单或管理器编辑现有边界条件，请参见"编辑依赖于步骤的对象"（第3.4.13节）。
2. 选择要应用边界条件的区域。双击**Region**列下的空表格单元格，然后使用以下方法之一：- 在视口中选择一个区域。您可以使用角度方法从几何体中选择一组面或边，或从网格中选择一组单元面。有关更多信息，请参见"使用角度和特征边方法选择多个对象"（第6.2.3节）。完成选择后，单击鼠标按钮2。**提示：**您可以通过在**Selection**工具栏中指定过滤选项来限制可以在视口中选择的对象类型。有关更多信息，请参见"使用选择选项"（第6.3节）。如果模型包含网格和几何体的组合，您必须选择要应用边界条件的区域类型。从提示区域，选择以下之一：- 单击**Geometry**将边界条件应用于几何体或参考点。- 单击**Mesh**将边界条件应用于本机或孤立网格选择。- 要从现有集合列表中进行选择，请执行以下操作：1. 单击提示区域右侧的**Sets**。Abaqus/CAE显示**Region Selection**对话框，其中包含可用集合的列表。2. 选择感兴趣的集合，然后单击**Continue**。**注意：**默认选择方法基于您最近使用的选择方法。要恢复为另一种方法，请单击提示区域右侧的**Select in Viewport**或**Sets**。
3. 通过单击一个或多个复选框选择表中的**Constrained-degrees-of-freedom**。
4. 单击**OK**保存数据并退出编辑器。
