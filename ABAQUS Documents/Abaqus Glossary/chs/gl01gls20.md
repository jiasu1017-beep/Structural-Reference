# S（续）







### superelasticity

材料在卸载时恢复其原始形状的能力，即使产生了永久变形（超弹性行为）。形状记忆合金表现出超弹性。
更多信息：- [「超弹性，」Abaqus Analysis User's Guide 第 22.1.2 节](../usb/usb-link.md#usb-mat-sesupersuplastic)

- [「超弹性行为，」Abaqus/CAE User's Guide 第 12.8.1 节](../usi/usi-link.md#usi-prp-hyperelasticity)

### superelement

一种矩阵被缩减以最小化计算成本的元素类型。子结构就是超单元的示例。
更多信息：- [「超单元分析，」Abaqus Analysis User's Guide](../usb/usb-link.md)

### superposition

线性问题的求解技术，其中总响应计算为施加在不同负载情况下的线性独立载荷的响应之和。
更多信息：- [「线性扰动步骤，」Abaqus Analysis User's Guide 第 6.1.4 节](../usb/usb-link.md#usb-anl-alinearnonlinear)

### surface-based cohesive behavior

使用接触公式定义粘聚区域界面的本构行为。表面基粘聚行为可用于模拟粘聚区模型，其中粘聚力根据相对位移和牵引力之间的关系定义。
更多信息：- [「表面基粘聚行为，」Abaqus Analysis User's Guide 第 30.1.1 节](../usb/usb-link.md#usb-mat-cohesiveoverview)

### surface-based failure

使用接触公式定义粘聚区域界面的失效。表面基失效可用于模拟界面失效，其中基于牵引力-分离响应的失效准则用于确定界面何时失效。
更多信息：- [「表面基粘聚行为，」Abaqus Analysis User's Guide 第 30.1.1 节](../usb/usb-link.md#usb-mat-cohesiveoverview)

### surface-based fluid cavity

一种腔体，其中密封表面定义明确的边界。腔内的压力分布是体积和腔室表面温度的函数。腔体可以与外部流体域或仅与结构相互作用。
更多信息：- [「流体填充腔体的压力穿通，」Abaqus Analysis User's Guide 第 14.2.1 节](../usb/usb-link.md#usb-anl-afluidcavitypressureventing)

- [「定义表面流体腔，」Abaqus/CAE User's Guide 第 15.25 节](../usi/usi-link.md#usi-itn-hlp-fuidcav-srf)

### surface fluid exchange

两个表面之间的流体交换，这些表面形成间隙并形成流体流动的通道。
更多信息：- [「表面流体交换，」Abaqus Analysis User's Guide 第 14.2.2 节](../usb/usb-link.md#usb-anl-asurfacefluid)

### surface interaction

两个接触表面之间的相互作用，包括法向行为（如硬接触）和切向行为（如摩擦）。
更多信息：- [「接触相互作用分析：概述，」Abaqus Analysis User's Guide 第 36.1.1 节](../usb/usb-link.md#usb-cni-acontactoverview)

### surface normal

在接触公式中使用的几何属性，定义每个接触面的法线方向。
更多信息：- [「接触公式，」Abaqus Analysis User's Guide 第 38.1.1 节](../usb/usb-link.md#usb-cni-acontactpairform)

### surface smoothness

在接触公式中使用的几何属性，定义接触面在节点处的平滑度。
更多信息：- [「接触公式，」Abaqus Analysis User's Guide 第 38.1.1 节](../usb/usb-link.md#usb-cni-acontactpairform)

### survey

在 Abaqus/CAE 中，指的是 viewport 中当前显示内容的概述。 survey 显示模型或零件在 viewport 中的位置和范围。
更多信息：- [「调查模型，」Abaqus/CAE User's Guide 第 77.1 节](../usi/usi-link.md#uss-dsp-dynamic-survey)

### symmetry

边界条件的一种类型，允许您通过仅对模型的一半（或四分之一或八分之一）进行建模来利用对称性。必须对对称平面施加对称边界条件。
更多信息：- [「对称边界条件，」Abaqus/CAE User's Guide 第 16.5.3 节](../usi/usi-link.md#usi-lbi-hlp-sym)

- [「对称，」Abaqus Analysis User's Guide 第 34.2.3 节](../usb/usb-link.md#usb-prc-psymmetry)

### symmetry boundary condition

一种边界条件类型，通过将适当的边界条件应用于对称平面，指定模型关于对称平面是对称的。
更多信息：- [「对称边界条件，」Abaqus Analysis User's Guide 第 34.2.3 节](../usb/usb-link.md#usb-prc-psymmetry)

### symmetric keys

一组四个以菱形排列的 Abaqus/CAE viewport 键。这些键用于平移视图。
更多信息：- [「使用键进行视图操作，」Abaqus/CAE User's Guide 第 5.2.3 节](../usi/usi-link.md#uss-viw-keyboardshortcuts)
