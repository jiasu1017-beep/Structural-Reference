# 20.2.6 aStudy.generate()





### 20.2.6 ***aStudy*.generate()**：生成参数化研究的分析作业数据。

**产品：** Abaqus/Standard  Abaqus/Explicit  

此命令用于为参数化研究生成分析输入文件。

##### **参考**

- ["参数化研究脚本，" 第20.1.1节](pt04ch20s01aus108.md)

### **命令：**

***aStudy*.generate** (**template**= )

### **必需数据：**

**template**

将**template**设置为要从中生成每个参数化研究变体的输入文件的模板输入文件名称；此名称必须用匹配引号括起来。
