# 1.6.18 具有规定间隙或过闭合值的小滑动接触





**产品：**Abaqus/Standard  Abaqus/Explicit  

### 单元测试

C3D8    CPE4    CPE4R    R2D2    R3D4    S4R  

### 功能测试

接触干涉

初始间隙

小滑动接触对

### 问题描述

Abaqus/Standard模型由两个接触表面初始过闭合的物体组成。通过将初始间隙值指定为零来在整个分析过程中保持此初始过闭合，但使用接触干涉的情况除外。在这些情况下，初始过闭合在施加载荷之前被解析。

**模型：**

| 二维实体单元尺寸 | 1.0×1.0×1.0 |
| --- | --- |
| 三维实体单元尺寸 | 每个方向1.0单位 |
| 三维壳单元尺寸 | 直径20个单位 |

**材料：**

| 杨氏模量 | 30×10⁶ |
| --- | --- |
| 泊松比 | 0.0 |
| 摩擦系数 | 0.0 |

Abaqus/Explicit模型由与刚性表面接触的三个可变形物体组成。使用了两种不同的方法来定义初始间隙值：指定间隙值，或指定从节点及其在跟随关键字行或从输入文件读取的数据行上的相应初始间隙值。考虑了一个二维模型，其中可变形物体使用CPE4R单元建模，刚性物体使用刚性单元R2D2建模。

### 结果与讨论

在所有情况下，结果在整个分析过程中都是正确的。
