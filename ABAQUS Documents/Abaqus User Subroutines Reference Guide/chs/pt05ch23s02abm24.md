### 概述

["延性金属的损伤和失效：概述，" 第24.2.1节"](pt05ch24s02abm41.md)中描述的逐步损伤和失效模型是在Abaqus中建模材料损伤和失效的推荐方法；这些模型适用于准静态和动态情况。Abaqus/Explicit提供了两种额外的仅适用于高应变率动态问题的单元失效模型。剪切失效模型由塑性屈服驱动。拉伸失效模型由拉伸加载驱动。这些失效模型可用于在达到应力极限后限制单元的后续承载能力（直至移除单元）。两种模型可以用于同一材料。

剪切失效模型：
- 专为包括大多数金属在内的高应变率变形设计；
- 使用等效塑性应变作为失效度量；
- 提供两种失效后处理选择，包括从网格中移除单元；
- 可与Mises或Johnson-Cook塑性模型结合使用；和
- 可与拉伸失效模型结合使用。

拉伸失效模型：
- 专为包括大多数金属在内的高应变率变形设计；
- 使用静水压力应力作为失效度量来模拟动态层裂或压力截止；
- 提供多种失效后处理选择，包括从网格中移除单元；
- 可与Mises或Johnson-Cook塑性模型或状态方程材料模型结合使用；和
- 可与剪切失效模型结合使用。

### 剪切失效模型

剪切失效模型可与Abaqus/Explicit中的Mises或Johnson-Cook塑性模型结合使用来定义材料的剪切失效。

#### 剪切失效准则

剪切失效模型基于单元积分点处的等效塑性应变值；当损伤参数超过1时，假定失效发生。损伤参数D通过以下方式定义：

D = Σ (Δε¯^p^/ε¯^p^f)

其中ε¯^p^~0~是等效塑性应变的任何初始值，Δε¯^p^是等效塑性应变增量，ε¯^p^f是失效时的应变，且求和遍及分析中的所有增量。

失效时的应变ε¯^p^f假定依赖于塑性应变率ε¯˙^p~；无量纲压力-偏量应力比p/q~（其中p是压力应力，q是Mises应力）；温度；和预定义场变量。有两种方法可以定义失效时的应变ε¯^p^f。一种是使用直接表格数据，其中依赖性以表格形式给出。或者，可以调用Johnson和Cook提出的分析形式（更多详情见["Johnson-Cook塑性，" 第23.2.7节"](pt05ch23s02abm23.md)）。

当使用直接表格数据定义剪切失效模型时，失效时的应变ε¯^p^f必须给出为等效塑性应变率、静水压力-偏量应力比、温度和预定义场变量的表格函数。此方法需要使用Mises塑性模型。

对于Johnson-Cook剪切失效模型，您必须指定失效参数d~1~ - d~5~（更多详情见["Johnson-Cook塑性，" 第23.2.7节"](pt05ch23s02abm23.md)）。剪切失效数据必须在["Johnson-Cook塑性，" 第23.2.7节"](pt05ch23s02abm23.md)中定义的转变温度θ~t~或以下进行校准。此方法需要使用Johnson-Cook塑性模型。

| **输入文件用法：** | 对Mises塑性模型使用以下两个选项： |
| --- | --- |
|  | ``` [*PLASTIC](../key/key-link.md#usb-kws-mplastic), HARDENING=ISOTROPIC [*SHEAR FAILURE](../key/key-link.md#usb-kws-mshearfailure), TYPE=TABULAR ``` 对Johnson-Cook塑性模型使用以下两个选项： ``` [*PLASTIC](../key/key-link.md#usb-kws-mplastic), HARDENING=JOHNSON COOK [*SHEAR FAILURE](../key/key-link.md#usb-kws-mshearfailure), TYPE=JOHNSON COOK ``` |

#### 单元删除

当在积分点满足剪切失效准则时，所有应力分量将设置为零，并且该材料点失效。默认情况下，如果单元任何截面上的所有材料点都失效，则从网格中移除该单元；不需要单元中的所有材料点都失效。例如，在一阶减缩积分实体单元中，一旦其唯一的积分点失效，就移除单元。但是，在壳单元中，必须所有穿透厚度的积分点都失效才能从网格中移除单元。对于二阶减缩积分梁单元，默认情况下，沿梁轴线的两个单元积分位置中任一位置的整个截面内所有积分点失效都会导致单元移除。类似地，在改进的三角和四面体实体单元中，默认情况下任何单个积分点失效都会导致单元移除。单元删除是默认的失效选择。

另一种失效选择（不删除单元）是指定当在材料点满足剪切失效准则时，该点的偏量应力分量将设置为零，并在计算的剩余部分保持为零。然后要求压力应力保持为压缩状态；也就是说，如果在增量中在失效材料点计算出负压力应力，则将其重置为零。当使用平面应力、壳、膜、梁、管道和桁架单元时，不允许此失效选择，因为可能违反结构约束。

| **输入文件用法：** | 使用以下选项允许在满足失效准则时删除单元（默认）： |
| --- | --- |
|  | ``` [*SHEAR FAILURE](../key/key-link.md#usb-kws-mshearfailure), ELEMENT DELETION=YES ``` 使用以下选项允许在满足失效准则时单元仅承受静水压缩应力： ``` [*SHEAR FAILURE](../key/key-link.md#usb-kws-mshearfailure), ELEMENT DELETION=NO ``` |

#### 确定何时使用剪切失效模型

Abaqus/Explicit中的剪切失效模型适用于惯性很重要的高应变率动态问题。剪切失效模型的不当使用可能导致不正确的模拟。

对于可能需要单元删除的准静态问题，建议使用逐步损伤和失效模型（[第24章"逐步损伤和失效"](pt05ch24.md)"）或多孔金属塑性模型（["多孔金属塑性，" 第23.2.9节"](pt05ch23s02abm25.md)）。

### 拉伸失效模型

拉伸失效模型可与Abaqus/Explicit中的Mises或Johnson-Cook塑性模型或状态方程材料模型结合使用来定义材料的拉伸失效。

#### 拉伸失效准则

Abaqus/Explicit拉伸失效模型使用静水压力应力作为失效度量来模拟动态层裂或压力截止。拉伸失效准则假定当压力应力p变得比用户指定的静水截止应力p~cutoff~更具拉伸性时发生失效。静水截止应力可能是温度和预定义场变量的函数。此应力没有默认值。

拉伸失效模型可与Mises或Johnson-Cook塑性模型或状态方程材料模型一起使用。

| **输入文件用法：** | 对Mises或Johnson-Cook塑性模型使用以下两个选项： |
| --- | --- |
|  | ``` [*PLASTIC](../key/key-link.md#usb-kws-mplastic) [*TENSILE FAILURE](../key/key-link.md#usb-kws-mtensilefailure) ``` 对状态方程材料模型使用以下两个选项： ``` [*EOS](../key/key-link.md#usb-kws-meos) [*TENSILE FAILURE](../key/key-link.md#usb-kws-mtensilefailure) ``` |

#### 失效选择

当在单元积分点满足拉伸失效准则时，材料点失效。为失效材料点提供了五种失效选择：默认选择（包括单元删除）和四种不同的层裂模型。这些失效选择如下所述。

##### 单元删除

当在积分点满足拉伸失效准则时，所有应力分量将设置为零，并且该材料点失效。默认情况下，如果单元任何截面上的所有材料点都失效，则从网格中移除该单元；不需要单元中的所有材料点都失效。例如，在一阶减缩积分实体单元中，一旦其唯一的积分点失效，就移除单元。但是，在壳单元中，必须所有穿透厚度的积分点都失效才能从网格中移除单元。对于二阶减缩积分梁单元，默认情况下，沿梁轴线的两个单元积分位置中任一位置的整个截面内所有积分点失效都会导致单元移除。类似地，在改进的三角和四面体实体单元中，默认情况下任何单个积分点失效都会导致单元移除。

| **输入文件用法：** | ``` [*TENSILE FAILURE](../key/key-link.md#usb-kws-mtensilefailure), ELEMENT DELETION=YES (default) ``` |
| --- | --- |

##### 层裂模型

也提供了基于层裂（材料碎裂）而非单元删除的替代失效选择。此类别中有四种失效组合可用。当在材料点满足拉伸失效准则时，偏量应力分量可能不受影响或可能需要为零，压力应力可能受到静水截止应力的限制或可能需要为压缩状态。因此，有四种可能的失效组合（见图23.2.8-1，其中"O"是如果不使用拉伸失效模型将存在的应力）。

**图23.2.8-1** 拉伸失效选择。

![](../graphics/ctensilefail.png)

- 延性剪切和延性压力：此选择对应于图23.2.8-1中的点1，模拟偏量应力分量不受影响且压力应力受到静水截止应力限制的情况；即 p = min(p, p~cutoff~)。 | **输入文件用法：** | ``` [*TENSILE FAILURE](../key/key-link.md#usb-kws-mtensilefailure), ELEMENT DELETION=NO, SHEAR=DUCTILE, PRESSURE=DUCTILE ``` | | --- | --- |
- 脆性剪切和延性压力：此选择对应于图23.2.8-1中的点2，模拟偏量应力分量设置为零并在计算的剩余部分保持为零，且压力应力受到静水截止应力限制的情况；即 S = 0，p = min(p, p~cutoff~)。 | **输入文件用法：** | ``` [*TENSILE FAILURE](../key/key-link.md#usb-kws-mtensilefailure), ELEMENT DELETION=NO, SHEAR=BRITTLE, PRESSURE=DUCTILE ``` | | --- | --- |

- 延性剪切和脆性压力：此选择对应于图23.2.8-1中的点4，模拟偏量应力分量不受影响且压力应力需要为压缩状态的情况；即 p = max(p, 0)。 | **输入文件用法：** | ``` [*TENSILE FAILURE](../key/key-link.md#usb-kws-mtensilefailure), ELEMENT DELETION=NO, SHEAR=DUCTILE, PRESSURE=BRITTLE ``` | | --- | --- |

对于层裂模型，没有默认的失效组合。如果您选择不使用单元删除模型，则必须明确指定失效组合。如果未定义材料的偏量行为（例如，使用没有偏量行为的状态方程模型），则组合的偏量部分没有意义将被忽略。当使用平面应力、壳、膜、梁、管道和桁架单元时，不允许使用层裂模型。

#### 确定何时使用拉伸失效模型

Abaqus/Explicit中的拉伸失效模型适用于惯性效应重要的高应变率动态问题。拉伸失效模型的不当使用可能导致不正确的模拟。

### 将失效模型与钢筋结合使用

可以将剪切失效和/或拉伸失效模型用于也定义了钢筋的单元。当根据失效准则此类单元失效时，根据所选失效类型，基础材料对单元应力承载能力的贡献被移除或调整，但对单元应力承载能力的钢筋贡献不会被移除。但是，如果您还在钢筋材料定义中包含失效，则如果满足为钢筋指定的失效准则，对单元应力承载能力的钢筋贡献也将被移除或调整。

### 单元

具有单元删除的剪切和拉伸失效模型可用于Abaqus/Explicit中包含力学行为（具有位移自由度的单元）的任何单元。没有单元删除的剪切和拉伸失效模型只能在Abaqus/Explicit中的平面应变、轴对称和三维实体（连续体）单元中使用。

| STATUS | 单元状态（如果单元是活动的，则为1.0；如果单元不是活动的，则为0.0）。 |
