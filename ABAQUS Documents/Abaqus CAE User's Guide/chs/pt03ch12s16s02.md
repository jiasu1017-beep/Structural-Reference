# 12.17.2 创建和编辑校准数据集







您可以从文本（`.txt`）文件导入材料数据，并在数据表中自定义这些数据。您还可以指定数量类型来描述数据，如应力和应变或力和位移。

**定义校准数据集：**

1. 从模型树中，展开**校准**并双击**数据集**。**创建数据集**对话框出现。
2. 要从文件导入数据集，请执行以下操作：1. 单击**导入数据集**。**从文本文件读取数据**对话框出现。2. 导航到您要导入的文件。有关文件选择的更多信息，请参见"使用文件选择对话框"，第3.2.10节。3. 如果您导入的文件中的数据由空格、制表符或逗号以外的字符分隔，请切换**其他**并在**分隔符**字段中输入分隔字符。4. 从**数据集类型**选项中，选择描述您导入数据的数据输出变量配对。默认类型为**应力/应变**。5. 如有需要，指定Abaqus/CAE应从中导入数据的字段编号。默认情况下，Abaqus/CAE从数据集中的第一和第二个字段读取数据。6. 如有需要，指定**数据集形式**为**真实**以真实形式导入材料数据。默认情况下，材料数据以名义（或工程）形式导入。Abaqus/CAE在对话框的表中显示导入的数据和所选数量类型。
3. 如有需要，通过添加、删除或修改行来自定义表中显示的数据。有关表格数据输入的更多信息，请参见"输入表格数据"，第3.2.7节。
4. 单击**确定**创建新数据集并关闭**创建数据集**对话框。Abaqus/CAE在模型树的**数据集**容器中显示新数据集，并在视口中绘制数据。
