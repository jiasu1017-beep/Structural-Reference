# 69.4 修复几何的策略





几何编辑工具集中的工具通常用于修复导入到Abaqus/CAE中的几何。修复操作的目标是创建一个没有无效实体的零件，并创建一个具有最少边和面的零件以避免影响网格生成。如果您无法创建有效的零件，可以忽略无效性。有关更多信息，请参见["使用无效零件," 第10.2.3节](pt02ch10s02s03.md)。理想情况下，零件应该没有小边或小面。您不必创建一个精确的零件，如果可能的话，应该继续使用不精确的零件。具有不精确几何的零件可以进行网格划分。但是，在极少数情况下，其他操作可能在具有不精确几何的零件上失败；例如，其他网格划分功能、添加几何特征和分割。如果您无法使用不精确的零件，也无法使零件精确，则应返回生成原始文件的CAD应用程序并提高精度。有关更多信息，请参见["什么是有效和精确的零件？," 第10.2.1节](pt02ch10s02s01.md)。

Abaqus/CAE提供了许多不同的编辑工具，决定使用哪些工具以及按什么顺序使用通常很困难。一般来说，修复导入零件的几何是一门需要实验和经验才能掌握的艺术。最佳方法取决于零件的复杂性以及您在网格划分和分析零件时希望保留多少细节。

以下准则将帮助您制定有效的零件修复方法：

**使用几何诊断工具**

使用几何诊断工具查询零件的无效和不精确实体、自由边、短边、小面和尖角。

**使用选择组**

您可以使用鼠标按钮3创建临时选择组，使几何诊断工具和修复工具的组合更易于使用。您可以使用几何诊断工具高亮显示无效和不精确几何的区域，然后将该区域复制到选择组中。然后，在指定要修复的区域时，可以将选择组粘贴到几何编辑工具集中。

**使用显示组**

使用显示组有选择地移除面和单元，以帮助您查看复杂零件内部并确定几何诊断工具高亮显示的实体。

**采取增量方法**

对选定的小组面和边使用修复工具，并在继续之前检查修复过程的结果是否正确。

**保持实体完整**

尽量不要通过从实体移除面来将实体转换为壳。为了避免移除面，请对面使用**替换**工具，对边和面使用**修复小**工具。

**避免创建线**

如果您的零件是实体或壳，尽量不要执行导致创建线的操作。Abaqus/CAE很难从线重构面。

**移除圆角和倒角**

使用**替换**工具和**扩展相邻面**选项从零件中移除圆角和倒角，并扩展相邻面以封闭产生的间隙。

**移除相切相遇的面**

使用**替换**工具或**修复小**工具来修复近似相切的面。您应该使用**修复小**工具来移除相对较小的面。**修复小**工具只是扩展剩余的面直到它们相遇。在此过程中会删除小面，尽管结果可能不精确。对于相对较大且相切相遇的面，请使用**替换**工具。**替换**工具使用底层点用近似曲面替换原始面。

**检查零件的有效性**

如果您的零件无效，您可以单击模型树中零件上的鼠标按钮3并从出现的菜单中选择**更新有效性**来检查修复操作是否使零件有效。检查有效性可能是一个耗时的操作，您应该在完成编辑零件后再检查其有效性。如果零件已经有效，则没有任何修复操作会使它无效。

**修复后创建分割**

如果您的零件包含分割，几何编辑工具集可能会在修复操作期间删除它们。为避免此问题，您应该等到完成编辑后再分割零件。
