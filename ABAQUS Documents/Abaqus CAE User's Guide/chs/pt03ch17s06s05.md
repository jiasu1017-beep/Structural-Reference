# 17.6.5 修改几何后重新生成分割





分割是与零件或零件实例关联的特征；因此，您可以像处理任何其他特征一样修改和重新生成它们。

例如，考虑图17-29中零件实例右侧的分割。

**图17-29** 已分割的零件实例。

![](../graphics/mgn-regenerate.png)

如果您返回"零件"模块并加宽模型的右侧，分割也会扩展并继续将面划分为两个区域，如图17-30所示。

**图17-30** 分割已重新生成。

![](../graphics/mgn-regenerate2.png)

有时分割的重新生成会创建无法网格化的区域。在这种情况下，只需添加、修改或删除分割，直到零件实例可以再次进行网格划分为止。
