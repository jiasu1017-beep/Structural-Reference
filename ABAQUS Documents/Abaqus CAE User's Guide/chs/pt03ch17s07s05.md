# 17.7.5 什么是网格过渡？





网格过渡是网格从粗（大单元）到细（小单元）的过渡区域，如图17-33所示。

**图17-33** 从粗单元到细单元过渡的网格。

![](../graphics/mgn-transition-nls.png)

Abaqus/CAE为以下类型的网格提供网格过渡控制：
- 使用结构化网格技术或使用中轴算法的自由网格技术创建的二维纯四边形网格。
- 通过从二维网格扫掠创建的三维纯六面体网格。有关更多信息，请参见["什么是扫掠网格？"第17.9.1节](pt03ch17s09s01.md)。

当网格过渡控制适用于您创建的网格类型时，**网格控制**对话框右侧会出现一个切换按钮，允许您最小化网格过渡。默认情况下，Abaqus/CAE最小化网格过渡，在某些情况下会减少网格畸变。相反，如果关闭最小化网格过渡选项，网格可能会更接近指定的网格种子。要显示**网格控制**对话框，请从主菜单栏选择****网格**![](../graphics/images/arrow.gif)**控制****。有关更多信息，请参见["设置网格算法，"第17.18.5节](pt03ch17s18s05.md)。
