# 62.5.1 创建datum点的方法概述







当您从**Create Datum**对话框中选择**Point**时，**Method**列表显示以下创建datum点的方法：

**![](../graphics/ico_dtmPtCoordinates.png) 输入坐标**

输入datum点的*X*、*Y*和*Z*坐标，如[图62--6](pt06ch62s05s01.md#dtm-hptc)所示。有关详细说明，请参见["通过输入坐标创建datum点，" 第62.6.1节](pt06ch62s06hlb01.md)。

**图62–6** 通过输入坐标创建datum点。

![](../graphics/dtm-hptc-nls.png)

**![](../graphics/ico_dtmPtOffsetPoint.png) 从点偏移**

以相对于选定点的*X*、*Y*和*Z*坐标偏移形式输入datum点的位置，如[图62--7](pt06ch62s05s01.md#dtm-hptop)所示。有关详细说明，请参见["在距选定点的偏移处创建datum点，" 第62.6.2节](pt06ch62s06hlb02.md)。

**图62–7** 通过从点偏移创建datum点。

![](../graphics/dtm-hptop-nls.png)

**![](../graphics/ico_dtmPtMidpoint.png) 两点之间的中点**

在模型上选择两点；Abaqus/CAE在两个选定点的中点处创建datum点，如[图62--8](pt06ch62s05s01.md#dtm-hpmid)所示。有关详细说明，请参见["在两点之间的中点处创建datum点，" 第62.6.3节](pt06ch62s06hlb03.md)。

**图62–8** 通过选择两个端点创建datum点。

![](../graphics/dtm-hpmid-nls.png)

**![](../graphics/ico_dtmPtOffsetEdges.png) 从两条边偏移**

在模型上选择两条边；并输入datum点到每条边的距离，如[图62--9](pt06ch62s05s01.md#dtm-hpted)所示。有关详细说明，请参见["在距两条边的指定距离处创建datum点，" 第62.6.4节](pt06ch62s06hlb04.md)。

**图62–9** 在距两条边指定距离处定位datum点。

![](../graphics/dtm-hpted-nls.png)

**![](../graphics/ico_dtmPtParameter.png) 输入参数**

在模型上选择一条边，并输入以边长度百分比表示的参数值形式的datum点位置。沿边的箭头表示参数值从起点（对应于边参数值为零）到终点（对应于值为一）增加的方向，如[图62--10](pt06ch62s05s01.md#dtm-hpted2)所示。有关详细说明，请参见["通过输入边参数创建datum点，" 第62.6.5节](pt06ch62s06hlb05.md)。

**图62–10** 沿边指定距离处定位datum点。

![](../graphics/dtm-hpted2-nls.png)

**![](../graphics/ico_dtmPtProjectOnFace.png) 将点投影到面/平面**

选择一个点以及要投影该点的面或平面。Abaqus/CAE在面或平面与通过选定点的法线相交的交点处创建datum点，如[图62--11](pt06ch62s05s01.md#dtm-hptpf)所示。datum点也标记了选定点和选定面或平面之间的最短距离。有关详细说明，请参见["通过将点投影到面或平面上创建datum点，" 第62.6.6节](pt06ch62s06hlb06.md)。

**图62–11** 通过将点投影到面上创建datum点。

![](../graphics/dtm-hptpf-nls.png)

**![](../graphics/ico_dtmPtProjectOnLine.png) 将点投影到边/datum轴**

在模型上选择一个点，以及要投影该点的边或datum轴。Abaqus/CAE在边或datum轴与通过选定点的法线相交的交点处创建datum点，如[图62--12](pt06ch62s05s01.md#dtm-hptpe)所示。datum点也标记了选定点和选定边或datum轴之间的最短距离。有关详细说明，请参见["通过将点投影到边或datum轴上创建datum点，" 第62.6.7节](pt06ch62s06hlb07.md)。

**图62–12** 通过将点投影到边上创建datum点。

![](../graphics/dtm-hptpe-nls.png)
