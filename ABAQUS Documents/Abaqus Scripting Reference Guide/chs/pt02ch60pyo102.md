# 60.102 UserDefinedField 对象





UserDefinedField 对象在材料点上重新定义场变量。

**访问**

```
materialApi.materials()[*name*].userDefinedField()
```

### 60.102.1 UserDefinedField()

此方法定义一个 UserDefinedField 对象。

**路径**

```
materialApi.materials()[*name*].UserDefinedField
```

**原型**

```
odb_UserDefinedField&
UserDefinedField();
```

**返回值**

一个 UserDefinedField 对象。

**异常**

无。

### 60.102.2 成员

UserDefinedField 对象没有成员。
