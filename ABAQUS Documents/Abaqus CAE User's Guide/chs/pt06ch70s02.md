# 70.2 使用分区工具集







您可以通过从菜单栏中选择 ****Tools**![](../graphics/images/arrow.gif)**Partition****（工具→分区）来访问分区工具集。出现 **Create Partition**（创建分区）对话框，您可以从对话框顶部 **Type**（类型）区域的按钮中选择要分区的几何类型——边、面或胞。**Method**（方法）列表更改为反映您可以创建的分区。从 **Method**（方法）列表中选择所需的分区工具，然后按照提示区域中的提示创建分区。大多数工具允许您在一个操作中分区多个边、面或胞。您可以使用拖动选择、**[Shift]****+点击**、**[Ctrl]****+点击** 和角度方法的组合来选择要分区的边、面或胞。 有关更多信息，请参阅["在当前视口中选择对象，" 第 6.2 节"](pt01ch06s02.md)。

["分区技术概述，" 第 70.4 节"](pt06ch70s04.md)提供了每种分区类型可用方法的概述。在 ["在 Part 模块中使用分区工具集，" 第 11.16.3 节"](pt03ch11s16s03.md)和["分区装配，" 第 13.8.3 节"](pt03ch13s08s03.md)中提供了关于在部件和装配上创建分区的更多信息。

您还可以从模块工具箱访问分区工具集；[图 70-2](pt06ch70s02.md#par-toolbox) 显示了模块工具箱中所有分区工具的隐藏图标。
