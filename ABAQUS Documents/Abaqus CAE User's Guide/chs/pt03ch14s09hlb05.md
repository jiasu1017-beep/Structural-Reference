# 14.9.5 重置步骤编辑器中的默认值





当您创建、编辑或替换步骤时，您使用步骤编辑器来配置分析过程设置。您可以使用替换功能通过用相同过程类型的步骤替换现有步骤来将步骤编辑器中的设置重置为默认值。

**要重置过程设置的默认值：**

1. 从主菜单栏，选择**步骤**![](../graphics/images/arrow.gif)**替换**![](../graphics/images/arrow.gif)***步骤名称***。**替换步骤**对话框出现，当前过程在可用过程列表中高亮显示。**提示：**您也可以在**步骤管理器**中选择步骤名称并点击**替换**。
2. 点击**继续**。**编辑步骤**对话框出现，其中包含过程设置的默认值。
3. 使用**编辑步骤**对话框从默认值修改设置，并为可选设置提供值。（有关特定编辑器功能的详细帮助，请从主菜单栏选择**帮助**![](../graphics/images/arrow.gif)**上下文帮助**，然后点击感兴趣的功能。）
4. 点击**确定**。Abaqus/CAE将步骤相关对象复制到新步骤，并关闭**编辑步骤**对话框。
