# 13.11.2 用平行平面约束两个实例





从主菜单栏选择**约束**![](../graphics/images/arrow.gif)**平行面**来施加约束，将可动实例定位，使所选面平行于固定实例的所选面。所有位置约束都是装配的特征，可以使用特征操作工具集来抑制或删除。

**要用平行平面约束两个零件或模型实例：**

1. 从主菜单，选择**约束**![](../graphics/images/arrow.gif)**平行面**。**提示：**您也可以使用装配模块工具箱中的![](../graphics/ico_constraintParallelFace.png)工具来施加平行面约束。有关装配工具箱中工具的图表，请参见["使用装配模块工具箱，"第13.9节](pt03ch13s09.md)。Abaqus/CAE在提示区域显示提示来引导您完成过程。
2. 从将移动的实例中选择一个平面，并从将保持固定的实例中选择一个平面，如下图所示：![](../graphics/asm-help-parallelface-select-nls.png)Abaqus/CAE显示垂直于所选面的箭头。当Abaqus/CAE提示您从固定实例中选择面时，您可以选择在零件或装配模块中创建的基准平面。相比之下，当您从可动零件实例中选择面时，您只能选择在零件模块中创建的基准平面。
3. 从提示区域中的按钮，执行以下操作之一：
   - 点击**确定**接受可动实例面上箭头的方向。
   - 点击**翻转**反转可动实例面上箭头的方向，然后点击**确定**。Abaqus/CAE将可动零件或模型实例定位，使两个面平行且箭头指向相同方向。固定实例的方向保持不变。更改箭头方向的效果如下图所示：![](../graphics/asm-help-parallelface.png)
   如果平行面约束与现有约束冲突，Abaqus/CAE显示错误消息并取消操作。要避免冲突，您可以尝试反转将移动的实例和将保持固定的实例的选择。或者，您可以删除现有的相对位置约束，施加绝对位置约束，然后重新施加平行面约束。
