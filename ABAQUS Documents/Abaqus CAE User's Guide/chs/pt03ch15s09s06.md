# 15.9.6 约束编辑器





要创建约束，从主菜单栏选择****Constraint**![](../graphics/images/arrow.gif)**Create****。出现**创建约束**对话框，你可以在其中指定约束的名称和类型。点击**继续**指定要应用约束的区域（如果适用）并显示你可以输入定义约束所需数据的编辑器。

每个约束编辑器在对话框顶部面板显示你正在定义的约束的名称和类型。编辑器其余部分的格式因你定义的约束类型而异。例如，[图15-19](pt03ch15s09s06.md#itn-tiededitor)显示了绑定约束编辑器。

**图15-19** 绑定约束编辑器。

![](../graphics/itn-tiededitor-nls.png)

你可以通过从主菜单栏选择****Help**![](../graphics/images/arrow.gif)**On Context****然后点击感兴趣的编辑器功能来显示有关特定编辑器功能的信息。有关创建约束的详细说明，请参见["使用约束编辑器，"第15.15节](pt03ch15s15.md)。
