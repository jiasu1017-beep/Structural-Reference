# 11.9.5 混合特征





混合特征可平滑三维实体零件的边缘。要创建混合特征，请从主菜单栏选择**形状**![](../graphics/images/arrow.gif)**混合**，或从零件模块工具箱中选择一个混合工具。您可以使用混合工具通过执行以下操作之一来创建混合特征：
- 用指定半径的圆形混合平滑边缘。- 用指定长度的倒角混合对边缘进行斜切。

您可以使用混合工具在三维建模空间中创建的可变形或离散刚体零件的边缘上添加混合特征。您不能向二维或轴对称零件添加混合特征；但是，您可以通过编辑零件的草图来混合其角落。
