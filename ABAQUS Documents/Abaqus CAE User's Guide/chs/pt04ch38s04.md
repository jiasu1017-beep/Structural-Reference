# 38.4 创建子模型边界条件





最常见的子建模技术是基于节点的子建模，它使用节点结果场（包括位移、温度或压力自由度）将全局模型结果插值到子模型节点上。基于节点的子建模也是一种更通用的技术。要使用基于节点的子建模，您需要创建子模型边界条件。

如果您将子模型边界条件应用到在先前步骤中由位移/旋转边界条件或连接器位移边界条件约束的节点，并且全局模型的边界条件使用**Fixed at Current Position**方法固定，Abaqus/CAE会忽略这些节点的子模型边界条件，并保留全局模型上边界条件中的规范。Abaqus/CAE会在分析的数据文件中报告此边界条件的替换。

**要创建子模型边界条件：**

1. 进入Load模块并从主菜单栏中选择****BC**![](../graphics/images/arrow.gif)**Create****。
2. 从步骤列表中，选择将应用子模型边界条件的步骤。
3. 从**Category**字段中，选择**Other**。
4. 从**Types for Selected Step**字段中，选择**Submodel**，然后单击**Continue**。
5. 从模型中选择要将边界条件应用到的区域。在大多数情况下，您将边界条件应用到在从全局模型切割区域时创建的面和边缘。您可以向相同区域指定其他边界条件；例如，对称边界条件。指定的边界条件优先于子模型边界条件。
6. 从出现的**Edit Boundary Condition**对话框中，执行以下操作：
   1. 在**Driving region**字段中，执行以下操作之一：
      - 选择**Automatic**以允许Abaqus/CAE通过搜索全局模型中位于子模型附近的的所有区域来创建驱动区域。
      - 选择**Specify**以指定将用作驱动区域的集名称。您必须给出集的完整名称。集名称的语法是`*assembly_name*.*part_name*-1.*set_name*`，假设您在部件的第一个实例上定义驱动区域。
   2. 如果您正在用壳全局模型驱动实体子模型，则必须在**Shell thickness**字段中输入全局模型中壳厚度的最大值。
   3. 在**Exterior tolerance**字段中，执行以下操作：
      - 输入**absolute** exterior tolerance。这是子模型的驱动节点可能位于全局模型单元外部的绝对值。默认值是相对exterior tolerance。
      - 输入**relative** exterior tolerance。这是全局模型中平均单元尺寸的分数，子模型的驱动节点可能位于全局模型单元外部。默认值为0.05。更多信息，请参见["子建模：概述"，Abaqus Analysis User's Guide第10.2.1节](../usb/usb-link.md#usb-anl-asubmodeloverview)。
   4. 如果您正在用实体全局模型驱动实体子模型，或者正在用壳全局模型驱动壳子模型，则必须输入逗号分隔的列表，指示被驱动的自由度；例如，`1,2,3`。您不能将此字段留空。
   5. 如果您正在用壳全局模型驱动实体子模型，则可以提供壳中面周围中心区域大小的厚度。默认值是全局模型中**Shell thickness**字段中定义的最大壳厚度的10%。
   6. 在**Global step number**字段中，输入一个整数，表示将读取驱动变量值的全局分析中的步骤编号。
   7. 如果您在静态、线性扰动步骤中创建了边界条件，则可以指定全局分析步骤中的增量，这将作为计算驱动变量值的基础。默认零值对应前一步骤的最后一个增量。
   8. 如果子模型分析的时间段与全局分析的时间段不同，您可以选择将全局步骤的时间段缩放以匹配子模型步骤的时间段。例如，Abaqus在全局步骤中时间的20%处确定全局模型的位移，并将其应用到子模型步骤中时间的20%处。如果您不选择将全局步骤的时间段缩放以匹配子模型步骤的时间段，Abaqus会在相同时间应用全局模型的位移。例如，Abaqus在全局步骤中一秒后确定全局模型的位移，并将其在子模型步骤中的一秒后应用。如果两个时间段不同，此行为可能不是您想要的。您可以通过切换**Scale time period of global step to time period of submodel step**来选择缩放时间段。
