# 17.18.5 设置网格算法





可用的网格算法选项取决于您选择的单元形状和网格划分技术。 如果网格算法选项适用于您正在创建的网格类型，**Mesh Controls**对话框右侧会出现一个**Algorithm**字段。

Abaqus/CAE提供以下网格算法选项：

**选择网格算法**

选择**Medial axis**或**Advancing front**。 很难预测哪个算法会为特定区域产生更好的网格；您可能需要尝试两种算法设置。 有关更多信息，请参阅["内侧轴算法和进阶前沿算法有什么区别？" 第17.7.6节](pt03ch17s07s06.md)。

**最小化网格过渡**

您可以控制Abaqus/CAE在从粗网格过渡到细网格时是否尝试最小化网格过渡。 在大多数情况下，切换**Minimize the mesh transition**会减少网格畸变。 但是，如果关闭**Minimize the mesh transition**，网格可能会更接近指定的网格种子。 有关更多信息，请参阅["什么是网格过渡？" 第17.7.5节](pt03ch17s07s05.md)。

**在适当的地方使用映射网格划分**

一些看起来非常复杂的模型实际上包含几何相对简单的面。 默认情况下，Abaqus/CAE在适当的地方使用映射网格划分技术在简单面上生成单元。 您可以关闭**Use mapped meshing where appropriate**以阻止Abaqus/CAE使用映射网格划分。 但是，如果您使用简单面进行网格划分且不允许映射网格划分，简单面上产生的单元质量可能较差。 有关更多信息，请参阅["什么是映射网格划分？" 第17.8.2节](pt03ch17s08s02.md)和["Abaqus/CAE何时可以应用映射网格划分？" 第17.8.6节](pt03ch17s08s06.md)。

**插入边界层**

当您创建四面体单元的自由网格时，可以沿区域边界的面添加楔形单元边界层。 边界层由一系列垂直于区域壁的楔形单元组成，最薄的单元靠在壁上。 边界层创建的网格在壁处具有高密度，随着向区域内部四面体网格的推进，密度逐渐降低。 楔形单元适用于Abaqus/CFD，当您分析管道壁和其他结构附近的流速和温度时。 有关创建边界层的详细说明，请参阅["在四面体网格边界添加楔形单元层，" 第17.18.12节](pt03ch17s18s10.md)。

**使用默认算法**

当您创建四面体单元的自由网格时，您可以选择默认网格生成算法或Abaqus/CAE 6.4及更早版本中包含的算法。 在大多数情况下，默认算法更稳健，特别是在网格划分复杂形状和薄壁实体时。 有关更多信息，请参阅["使用三角形和四面体单元的自由网格划分，" 第17.10.3节](pt03ch17s10s03.md)。

**增加内部单元的尺寸**

如果您选择默认网格生成算法来创建四面体单元的自由网格，您可以切换**Non-standard interior element growth**并使用滑块控制或文本字段指定内部单元的生长速率。 生长速率必须在`1.0`（无或最小生长）和`2.0`（最大生长）之间。

如果模型的分析网格密度足够，且感兴趣区域位于网格边界上，增加内部单元的尺寸将提高计算效率。 要查看Abaqus/CAE生成的内部单元，您可以使用视图切割或使用显示组从视图中移除外部单元。

**设置网格算法：**

1. 从主菜单栏，选择 ****Mesh**![](../graphics/images/arrow.gif)**Controls****。 Abaqus/CAE在提示区域显示提示以指导您完成该过程。 **提示：**您也可以单击Mesh模块工具箱中的 ![](../graphics/ico_meshControls.png) 工具。 （有关更多信息，请参阅["使用Mesh模块工具箱，" 第17.15节](pt03ch17s15.md)。）
2. 如果您的零件或装配包含多个区域，请选择感兴趣的区域，然后单击鼠标按钮2。 **Mesh Controls**对话框出现。 如果网格算法选项适用于所选单元形状和网格划分技术，**Mesh Controls**对话框右侧会出现一个**Algorithm**字段。
3. 选择所需的算法选项，然后单击**OK**保存数据并关闭对话框。
