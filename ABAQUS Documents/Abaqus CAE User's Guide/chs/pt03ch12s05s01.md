# 12.5.1 材料库概述





材料库工具位于Property模块的Model Tree区域中。材料库提供了一种便捷的方式来存储材料属性数据，以用于多个分析。您可以创建一个或多个库来保存和组织与项目、项目组或整个公司相关的材料数据。使用材料库允许您维护一组一致的材料的属性，并快速将材料添加到模型中。

材料库被视为Abaqus/CAE的插件，尽管它们会自动加载而无需使用**Plug-ins**菜单。材料库具有`.lib`文件扩展名，并存储在位于Abaqus父目录、您的主目录或当前目录（启动Abaqus/CAE的目录）中的`abaqus_plugins`目录中。您还可以使用`abaqus_v6.env`文件中的`plugin_central_dir`环境变量来指定一个额外的目录路径。（有关插件目录位置的更多信息，请参见["插件文件存储在哪里？，"第81.6.1节"](pt08ch81s06s01.md)。"）Abaqus/CAE搜索指定插件位置的所有子目录中的材料库文件。

默认情况下，Abaqus/CAE以树格式显示材料库，类似于Model Tree。在此格式中，您可以展开和折叠类别以帮助定位所需的材料。[图12-14](pt03ch12s05s01.md#prp-matlibrary-nls)显示了一个包含金属材料的简单材料库。它按铝、铜和钢分类组织；铝类别已被展开。您也可以将材料库查看为材料名称的字母列表，忽略类别。

**图12-14** **Material Library**。

![](../graphics/prp-matlibrary-nls.png)

您可以使用材料列表上方的**Filter**来搜索材料名称中的字符列表。过滤器可以包含Abaqus材料名称中允许的任何字符（有关对象命名的更多信息，请参见["使用基本对话框组件，"第3.2.1节](pt01ch03s02s01.md)）。过滤器不应用于类别名称，因此过滤可能导致"空"类别。

![](../graphics/ico_manager.png)工具打开**Material Library Manager**，您可以在其中创建、编辑、重命名和重组材料库。您还可以使用管理器来创建、编辑、重命名和删除**Specification Tags**以帮助识别库中的材料。

[图12-14](pt03ch12s05s01.md#prp-matlibrary-nls)显示了预定义的**Specification Tags**。![](../graphics/ico_materialAdd.png)工具将所选材料从库复制到当前模型。
