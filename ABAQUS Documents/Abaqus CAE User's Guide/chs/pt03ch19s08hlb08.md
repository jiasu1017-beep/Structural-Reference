# 19.8.9 控制作业精度







要找到作业编辑器，请从主菜单栏中选择****作业**![](../graphics/images/arrow.gif)**编辑**![](../graphics/images/arrow.gif)***作业名称*****。使用**精度**选项卡页面控制 Abaqus/Explicit 分析的精度或 Abaqus/Standard 或 Abaqus/Explicit 分析的节点输出精度。您可以执行以下操作：
- 为 Abaqus/Explicit 精度选择单精度、强制单精度（在必要时覆盖环境变量 **double_precision** 设置）或双精度（仅分析、仅约束或分析和打包程序）设置。此选项等同于 Abaqus/Explicit 执行过程中的 `double` 选项。
- 选择单精度或完整节点输出精度。此选项等同于 Abaqus 执行过程中的 `output_precision` 选项。
