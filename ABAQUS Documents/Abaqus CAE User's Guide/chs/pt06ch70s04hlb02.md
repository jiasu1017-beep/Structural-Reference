# 70.4.2 分区面的方法概述







当您从 **Create Partition**（创建分区）对话框中选择 **Face**（面）时，**Method**（方法）列表显示以下分区面的方法：

**![](../graphics/ico_ptnCellSketch.png) 草绘平面分区**

使用草图绘制器草绘分区来分区所选面（如图 70-8 所示）。详细说明，请参阅["使用草图方法分区面，" 第 70.6.1 节"](pt06ch70s06hlb01.md)。

您可以直接在要分区的面上草绘，或者您可以在第二个面或基准平面上草绘，然后将草图投影到要分区的面上。关于从基准平面投影草图的示例，请参阅["在 Part 模块中使用基准工具集，" 第 11.16.1 节"](pt03ch11s16s01.md)。

**图 70-8** 使用草图绘制器分区面。

![](../graphics/par-face-sketch-nls.png)

**![](../graphics/ico_ptnFaceTwoPoints.png) 两点之间的最短路径**

沿连接两个所选点的最短路径分区面；如果被分区的面是曲面的，则 resulting 分区将是弯曲的（如图 70-9 所示）。您可以选择与被分区面无关的点；例如，这些点可以位于不同的面甚至不同的部件实例上。详细说明，请参阅["使用最短路径方法分区面，" 第 70.6.2 节"](pt06ch70s06hlb02.md)。

**图 70-9** 使用两点之间的最短路径分区面。

![](../graphics/par-face-shortest-nls.png)

**![](../graphics/ico_ptnFaceDatumPlane.png) 使用基准平面**

使用与基准平面扩展的交点来分区面（如图 70-10 所示）。详细说明，请参阅["使用基准平面方法分区面，" 第 70.6.3 节"](pt06ch70s06hlb03.md)。

**图 70-10** 使用基准平面分区面。

![](../graphics/par-face-datum-nls.png)

**![](../graphics/ico_ptnFaceCurvedPath.png) 垂直于两条边的曲线路径**

沿一条垂直于面两条边的 Bézier 曲线分区面（如图 70-11 所示）。通过沿两条边任意位置选择两个点来定位曲线。两条边所对的弧必须小于 180°。详细说明，请参阅["使用曲线路径方法分区面，" 第 70.6.4 节"](pt06ch70s06hlb04.md)。

**图 70-11** 使用 Bézier 曲线分区面。

![](../graphics/par-face-curve-nls.png)

**![](../graphics/ico_ptnFaceExtendFace.png) 扩展另一个面**

使用与另一个面扩展的交点来分区面（如图 70-12 所示）。被扩展的面可以是平面、圆柱形、圆锥形或球形的；它不必属于包含要分区的面的部件。详细说明，请参阅["使用扩展面方法分区面，" 第 70.6.5 节"](pt06ch70s06hlb05.md)。

**图 70-12** 使用另一个面的扩展分区面。

![](../graphics/par-face-extend-nls.png)

**![](../graphics/ico_ptnFaceIntersectFace.png) 通过其他面交叉**

使用目标面与一个或多个其他面的交点来分区面（如图 70-13 所示）。这些面可以相交或相切。详细说明，请参阅["使用交叉方法分区面，" 第 70.6.6 节"](pt06ch70s06hlb06.md)。

**图 70-13** 使用面交叉分区面。

![](../graphics/par-face-intersect-nls.png)

**![](../graphics/icoR_ptnFaceProjectEdges.png) 投影边**

通过在模型中投影边来分区面（如图 70-14 所示）。分区使用从被分区面到分区边的垂直投影创建。您可以选择仅使用投影，或者在必要时延伸投影边的端点以完成面分区。详细说明，请参阅["使用投影边方法分区面，" 第 70.6.7 节"](pt06ch70s06hlb07.md)。

**图 70-14** 通过投影边分区面。

![](../graphics/prt-face-projedge-nls.png)

**![](../graphics/ico_ptnFaceAuto.png) 自动分区**

当您使用自由网格划分技术使用四边形单元对面进行网格划分时，Mesh（网格）模块在对面进行网格划分之前会内部将面分区为三到五个逻辑边的区域。有关更多信息，请参阅["使用四边形和四边形主导单元的自由网格划分，" 第 17.10.2 节"](pt03ch17s10s02.md)。但是，如果您想在生成网格之前查看并可能修改自动生成的区域，可以使用自动分区工具对面进行分区而无需对其进行网格划分。此工具仅在 Mesh（网格）模块中可用。详细说明，请参阅["使用自动生成方法分区面，" 第 70.6.8 节"](pt06ch70s06hlb08.md)。
