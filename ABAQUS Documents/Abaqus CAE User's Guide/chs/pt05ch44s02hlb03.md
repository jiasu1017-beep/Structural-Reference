# 44.5.2 美化线型云图





Abaqus/CAE将线型云图显示为模型表面上的自定义彩色线。您可以单独美化每条云图线以便于区分。当颜色不可用时（例如，用于黑白硬拷贝绘图），此选项特别有用。[图44-5](pt05ch44s02hlb02.md#usv-cont-linestyle)显示了一个具有自定义云图线的云图。

**图44-5** 具有自定义云图线的线型云图。

![](../graphics/usv-cont-linestyle.png)

您一次只能按一种颜色（[区间](pt05ch44s02hlb12.md)）自定义彩色线，方法是引用每个区间在云图图例中的位置。区间号1是从云图图例底部开始的第一条彩色线。有关云图图例的信息，请参见["自定义图例，" 第56.1节](pt05ch56s01.md)。

**要美化线型云图：**

1. 找到**Line**选项。从主菜单栏中选择****Options**![](../graphics/images/arrow.gif)**Contour****，或单击工具箱中的![](../graphics/ico_plotContourOptions.png)。在出现的对话框中单击**Color & Style**选项卡；然后单击**Line**选项卡。**Line**选项出现。
2. 要选择您想要自定义的区间，请单击**Interval**箭头，直到**Interval**框中出现所需的区间号。**注意：****Style**和**Thickness**按钮更改以显示此区间各自的当前样式和厚度。上方和下方**Interval**框的数字更改以对应相邻区间号。**Style**和**Thickness**按钮上方和下方的区域更改以显示这些相邻区间的样式和厚度。
3. 选择区间线的样式：1. 单击**Style**按钮显示样式选择。2. 单击您想要的样式。指定样式出现在**Style**按钮上。
4. 选择区间线的厚度：1. 单击**Thickness**按钮显示厚度选择。2. 单击您想要的厚度。指定厚度出现在**Thickness**按钮上。
5. 重复步骤2到4以自定义其他区间。
6. 单击**Apply**实施您的更改。当前视口中线型云图的云图线更改为反映您的样式和厚度规格。如果激活，云图图例也会更改以显示您的选择。有关云图图例的更多信息，请参见["自定义图例，" 第56.1节](pt05ch56s01.md)。此外，如果存在刻度线轴，它们也会更改以显示您的选择。有关刻度线云图的更多信息，请参见["使用刻度线云图绘制线形单元，" 第44.5.4节](pt05ch44s02hlb04.md)。默认情况下，您的更改在会话期间保存，并将影响所有后续云图。如果您想在后续会话中保留更改，请将它们保存到文件。有关更多信息，请参见["保存自定义设置以供后续会话使用，" 第55.1.1节](pt05ch55s01s01.md)。
