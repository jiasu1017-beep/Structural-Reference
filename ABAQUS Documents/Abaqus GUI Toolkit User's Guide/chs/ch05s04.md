# 5.4 选择壳单元







- 线性、有限膜应变、完全积分的四边形壳单元（S4）可在需要更高求解精度时、对于容易发生膜或弯曲模式沙漏的问题、或预期面内弯曲时使用。
- 线性、有限膜应变、减缩积分的四边形壳单元（S4R）稳健，适用于广泛的应用。
- 线性、有限膜应变三角形壳单元（S3/S3R）可用作通用单元。可能需要细化网格以捕获弯曲变形或高应变梯度，因为单元中是常应变近似。
- 要考虑层压复合壳模型中剪切柔度的影响，请使用适合建模厚壳的壳单元（S4、S4R、S3/S3R、S8R）；检查平面截面保持平面的假设是否满足。
- 二次壳单元（四边形或三角形）对于一般、小应变、薄壳应用非常有效。这些单元不易受剪切或膜锁定影响。
- 如果必须在接触模拟中使用二阶单元，请勿使用二次三角形壳单元（STRI65）。请改用 9 节点四边形壳单元（S9R5）。
- 对于将仅经历几何线性行为的非常大的模型，线性薄壳单元（S4R5）通常比通用壳单元更具成本效益。
- 小膜应变单元对涉及小膜应变和任意大旋转的显式动力学问题有效。
