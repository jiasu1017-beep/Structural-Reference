# 4.1.29 VDLOAD：非均匀载荷





### 4.1.29 [`VDLOAD`](../sub/sub-link.md#sub-xsl-vdload)：非均匀载荷

**产品：**Abaqus/Explicit  

### I. 非均匀体力

### 测试单元

CPE3、CPE4R、CPS3、CPS4R、CAX3、CAX4R、C3D4、C3D6、C3D8R

M3D3、M3D4R、SAX1、S3R、S4R、T2D2、T3D2

### 测试功能

非均匀体力。

### 问题描述

在此验证测试中，通过非均匀体力加载所有可用的单元类型来测试。所有单元节点的位置都是固定的，节点产生的反作用力用于验证单元载荷计算。此测试的目的是验证单元载荷计算，而不是测试用户子程序[`VDLOAD`](../sub/sub-link.md#sub-xsl-vdload)的所有功能。

材料模型是各向同性线弹性。使用的材料特性定义如下：弹性模量 = 193.1×10^9，泊松比 = 0.3，密度 = 785。

在第一步中，除了轴对称单元（沿r方向施加）外，对所有单元沿x方向施加1.0×10^5的非均匀体力。对于轴对称单元，载荷沿r方向施加。此非均匀体力的幅值函数定义使得载荷在步骤的前半段斜升，然后在分析的剩余部分保持恒定。在第二步中，对所有单元（除了轴对称单元，其中沿z方向施加）沿y方向施加另一个1.0×10^5的非均匀体力。此载荷使用与第一步相同的幅值函数施加。对于C3D4、C3D6、C3D8R、S3R、S4R、M3D3和M3D4R单元，在第三步中沿z方向施加另一个1.0×10^5的非均匀体力。此载荷也使用与第一步相同的幅值函数。

### 结果与讨论

所有单元的结果与分析值一致，分析值包含在输入文件的顶部。

### 输入文件

[element_nbody.inp](../eif/element_nbody.inp)

用于此测试的输入数据。

[element_nbody.f](../eif/element_nbody.f)

用于此测试的用户子程序。

### II. 非均匀压力载荷

### 测试单元

**分布式基于单元的载荷**
B21、B31、PIPE21、PIPE31、CPE3、CPE4R、CPS3、CPS4R、CAX3、CAX4R、C3D4、C3D6、C3D8R、M3D3、M3D4R、R2D2、RAX2、R3D3、R3D4、SAX1、S3R、S4R

**分布式表面载荷**
CPE3、CPE4R、CPS3、CPS4R、CAX3、CAX4R、C3D4、C3D6、C3D8R、M3D3、M3D4R、R2D2、RAX2、R3D3、R3D4、SAX1、S3R、S4R

### 测试功能

使用分布式基于单元的载荷和分布式表面载荷规定的非均匀压力载荷。

### 问题描述

在此验证测试中，通过非均匀压力载荷加载所有可用的单元类型来测试。所有单元节点的位置都是固定的，节点产生的反作用力用于验证单元载荷计算。此测试的目的是验证单元载荷计算，而不是测试用户子程序[`VDLOAD`](../sub/sub-link.md#sub-xsl-vdload)的所有功能。

材料模型是各向同性线弹性。使用的材料特性定义如下：弹性模量 = 193.1×10^9，泊松比 = 0.3，密度 = 785。

在第一步中，在单元边缘（对于CPE3、CPE4R、CPS3、CPS4R、CAX3、CAX4R、SAX1、R2D2和RAX2单元）或单元面（对于C3D4、C3D6、C3D8R、S3R、S4R、M3D3、M3D4R、R3D3和R3D4单元）施加1.0×10^5的非均匀压力。对于梁和管道单元（B21、B31、PIPE21和PIPE31），沿y方向施加1.0×10^5的单位长度非均匀力。此非均匀压力载荷的幅值函数定义使得载荷在步骤的前半段斜升，然后在分析的剩余部分保持恒定。在第二步中，在不同的单元边缘（对于CPE3、CPE4R、CPS3、CPS4R、CAX3和CAX4R单元）或单元面（对于C3D4、C3D6和C3D8R单元）施加1.0×10^5的非均匀压力。对于梁单元（B21和B31），沿x方向施加1.0×10^5的单位长度非均匀力。这些载荷使用与第一步相同的幅值函数施加。

### 结果与讨论

所有单元的结果与分析值一致，分析值包含在输入文件的顶部。
