# 19.11.3 提交共同执行







要提交共同执行，请从主菜单栏中选择****共同执行**![](../graphics/images/arrow.gif)**提交**![](../graphics/images/arrow.gif)***共同执行名称*****。（您也可以通过在**共同执行管理器**中选择共同执行名称，然后点击**提交**来提交共同执行。）Abaqus/CAE 使用共同执行编辑器或各个作业编辑器中定义的作业设置提交作业进行分析。

共同执行中涉及的作业出现在模型树中**共同执行**容器内的**作业**容器中。您可以通过在模型树中右键点击作业并选择**监视器**来显示作业监视器对话框，以监视已提交的作业。有关更多信息，请参见["监视分析作业的进度，"第 19.2.6 节](pt03ch19s02hlb01.md)。
