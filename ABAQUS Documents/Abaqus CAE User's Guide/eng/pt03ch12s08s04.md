# 12.8.4 Defining constants for a user material







The following user subroutines allow you to create user-defined material models:
- [`UMAT`](../sub/sub-link.md#sub-xsl-umat) for mechanical material models in Abaqus/Standard.
- [`VUMAT`](../sub/sub-link.md#sub-xsl-vumat) for mechanical material models in Abaqus/Explicit.
- [`UMATHT`](../sub/sub-link.md#sub-xsl-umatht) for thermal material models in Abaqus/Explicit.

You can enter constants in the **Edit Material** dialog box for use in these subroutines. See the following sections for more information:- ["User-defined mechanical material behavior," Section 26.7.1 of the Abaqus Analysis User's Guide](../usb/usb-link.md#usb-mat-cusermat)
- ["User-defined thermal material behavior," Section 26.7.2 of the Abaqus Analysis User's Guide](../usb/usb-link.md#usb-mat-cthusermat)

**To define material constants for a user material:**

1. From the menu bar in the **Edit Material** dialog box, select ****General**![](../graphics/images/arrow.gif)**User Material****. (For information on displaying the **Edit Material** dialog box, see ["Creating or editing a material," Section 12.7.1](pt03ch12s07hlb01.md).)
2. If you are performing an Abaqus/Standard analysis, click the arrow to the right of the **User material type** field, and select the type of user material for which you are defining constants.
3. If you are performing an Abaqus/Standard analysis, toggle on **Use unsymmetric material stiffness matrix** if the material stiffness matrix, ![](../graphics/usi_eqn00008.gif), is not symmetric or, in the case of a thermal constitutive model, if ![](../graphics/usi_eqn00009.gif) is not symmetric. This parameter causes Abaqus/Standard to use its unsymmetric equation solution procedures.
4. If you are defining a mechanical or thermomechanical material, enter the **Mechanical Constants** in the **Data** table. For detailed information on how to enter data, see ["Entering tabular data," Section 3.2.7](pt01ch03s02s07.md).
5. If you are defining a thermal or thermomechanical material, enter the **Thermal Constants** in the **Data** table. For detailed information on how to enter data, see ["Entering tabular data," Section 3.2.7](pt01ch03s02s07.md).
6. Click **OK** to close the **Edit Material** dialog box. Alternatively, you can select another material behavior to define from the menus in the **Edit Material** dialog box (see ["Browsing and modifying material behaviors," Section 12.7.2](pt03ch12s07hlb02.md), for more information).
