# 2.1.2 退出 Abaqus/CAE 会话







您可以随时通过从主菜单栏选择****文件**![](../graphics/images/arrow.gif)**退出****来退出 Abaqus/CAE 会话。如果您对当前模型数据库进行了任何更改，Abaqus/CAE 会在退出会话之前询问您是否要保存更改。然后 Abaqus/CAE 关闭当前模型或输出数据库以及所有窗口并退出会话。

Abaqus/CAE 会保存您的 GUI 设置；例如，主窗口的大小和对话框的大小和位置。有关更多信息，请参阅[「使用 `abaqus_v6.14.gpr` 文件，第 2.1.3 节」](pt01ch02s01s03.md) 和[「理解 Abaqus/CAE GUI 设置，第 3.6 节」](pt01ch03s06.md)。此外，Abaqus/CAE 会自动创建一个名为 `abaqus.rpy` 的文件来记录您在会话期间的操作；您可以使用此文件来重现您的操作。有关重现操作和恢复中断会话的更多信息，请参阅[「重新创建未保存的模型数据库，第 9.5.3 节」](pt02ch09s05s03.md)。
