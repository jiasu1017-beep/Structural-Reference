# 69.8.2 转换为精确







当您将部件转换为精确时，Abaqus/CAE 尝试更改相邻实体以使其几何完全匹配。转换为精确表示通常会减少部件中不精确实体的数量，尽管对于某些部件，转换可能会增加不精确实体的数量。即使不精确实体的数量增加，愈合部件的最大容差通常仍会减少；但是，当不精确实体的数量增加时，您应该考虑抑制或删除转换为精确操作。

转换为精确操作可能很耗时，并且会增加导入部件的复杂性；因此部件的后续处理可能会更慢。此操作也可能导致无效实体。

**转换为精确：**

1. 从主菜单栏中选择 ****Tools**![](../graphics/images/arrow.gif)**Geometry Edit****（工具→几何编辑）。Abaqus/CAE 显示 **Geometry Edit**（几何编辑）对话框。**提示：**您也可以使用位于 Part（部件）模块工具箱中编辑工具处的 ![](../graphics/ico_repairConvertToPrecise.png) 工具来转换为精确。 有关工具箱中编辑工具的图表，请参阅["使用几何编辑工具集，" 第 69.1 节"](pt06ch69s01.md)。
2. 从对话框中选择 **Part**（部件）类别和 **Convert to precise**（转换为精确）方法。
3. 从出现的警告对话框中点击 **OK**（确定）。
4. 选择以下转换方法之一：**Tighten Gaps**（收紧间隙）Abaqus/CAE 尝试减少部件中节点和边之间的间隙，但不执行几何的完整计算。此选项提供更快但不太完整的转换。**Recompute Geometry**（重新计算几何）Abaqus/CAE 尝试更改相邻实体以使其几何完全匹配。重新计算几何通常会产生精确几何；但是，此操作可能很耗时，并会增加导入部件的复杂性，这意味着部件的后续处理会更慢。此外，如果部件包含许多复杂曲面，转换为精确表示可能会失败。如果可能，您应该返回生成原始文件的 CAD 应用程序并提高精度。选择方法后，Abaqus/CAE 尝试将部件转换为其精确表示。如果转换成功，Abaqus/CAE 会提示您更新部件的有效性。
5. 如果您已完成编辑部件，您应该通过在模型树中的部件上点击鼠标按钮 3 并从出现的菜单中选择 **Update validity**（更新有效性）来检查其有效性。
