# 26.3 为协同仿真链接和排除零件实例


对于协同仿真,您可能希望在单个视口中看到参与协同仿真的"整个"模型。为此,您可以将一个模型中的零件实例链接到另一个模型中的零件实例。例如,您可以将 Abaqus/Standard 模型中的零件实例链接到 Abaqus/Explicit 模型中的零件实例。在这种情况下,Abaqus/Standard 模型中链接的零件实例无法编辑,它们的位置仅由 Abaqus/Explicit 模型中零件实例的位置决定,并且仅用于显示目的。对 Abaqus/Explicit 模型中零件实例的任何更改都会自动更新到链接的零件实例中。您必须将链接的零件实例从相应的分析中排除;例如,链接到 Abaqus/Standard 模型的 Abaqus/Explicit 模型中的零件实例必须从 Abaqus/Explicit 分析中排除。更多信息,请参见["链接模型之间的零件实例," 第 13.3.5 节](pt03ch13s03s05.md),和["从分析中排除零件实例," 第 13.3.6 节](pt03ch13s03s06.md)。
