# 3.7.1 Abaqus使用的FORTRAN单元号





**产品：**Abaqus/Standard  Abaqus/Explicit  

##### **参考**

- ["Abaqus执行程序：概述，" 3.1.1节](pt01ch03s01abo02.md)

### 概述

Abaqus使用下面概述的FORTRAN单元号。除非另有说明，否则不应尝试从用户子程序写入这些FORTRAN单元。

对于Abaqus/Standard，您应指定单元号15-18或大于100的单元号。

对于Abaqus/Explicit，请指定单元16-18或以5到9结尾的大于100的单元号，例如105、268等。您不能写入`.sta`文件。
