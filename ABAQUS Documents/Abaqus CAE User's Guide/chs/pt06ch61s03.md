# 61.3 自定义工具栏





Customize 工具集允许您创建包含 Abaqus/CAE 中大多数已定义功能快捷方式的自定义工具栏。自定义工具栏可用于将常用工具收集在一个位置，为没有现有快捷方式的功能创建工具，或为常见过程提供易于访问的控件（例如，创建零件、实例零件和网格零件功能都可以位于同一工具栏中）。自定义工具栏在所有模块中都可见。如果工具充当模块专用功能的快捷方式，则 Abaqus/CAE 在调用该工具时会自动切换到适当的模块。

自定义工具栏的内容和位置在会话之间保存。

要创建自定义工具栏，请从主菜单栏选择 ****Tools**![](../graphics/images/arrow.gif)**Customize****。有关创建自定义工具栏的详细说明，请参见["创建和修改自定义工具栏，" 第61.5.4节](pt06ch61s05hlb04.md)。
