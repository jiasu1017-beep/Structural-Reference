# 1 Overview







This guide contains information about installing and customizing Abaqus 6.14. This guide is also accessible from the **Documentation** section of the **Support** page at [www.3ds.com/simulia](http://www.3ds.com/simulia).

The installation of Abaqus involves the following tasks:
- Installing the Abaqus documentation.
- Installing the Abaqus license server.
- Installing Abaqus products.

For a description of the installation procedures, see [Chapter 2, "Installing Abaqus](ch02.md).”

Network licensing and procedures to administer the license server are described in [Chapter 3, "Abaqus licensing](ch03.md).”

The environment file provides options for customizing an Abaqus installation. Specifying the file system to be used to store scratch files is particularly important. For details on how to set local defaults and adjust the system configuration to run Abaqus jobs efficiently, see [Chapter 4, "Customizing the Abaqus environment](ch04.md).”

For more information on customizing the Abaqus user interface, printer configuration, and graphic card tuning, see [Chapter 5, "Customizing Abaqus/CAE and Abaqus/Viewer](ch05.md).”
