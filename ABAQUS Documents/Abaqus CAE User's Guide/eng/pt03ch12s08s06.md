# 12.8.6 Specifying the number of user variables







You can allocate space at each material calculation point for user-defined output variables defined in user subroutine [`UVARM`](../sub/sub-link.md#sub-xsl-uvarm).

**To specify the number of user variables:**

1. From the menu bar in the **Edit Material** dialog box, select ****General**![](../graphics/images/arrow.gif)**User Output Variables****. (For information on displaying the **Edit Material** dialog box, see ["Creating or editing a material," Section 12.7.1](pt03ch12s07hlb01.md).)
2. Click the arrows to the right of the **Number of user-defined variables at each material point** field to specify how many user-defined variables you want to allocate space for at each material calculation point. Any number of user-defined output variables can be used.
3. Click **OK** to close the **Edit Material** dialog box. Alternatively, you can select another material behavior to define from the menus in the **Edit Material** dialog box (see ["Browsing and modifying material behaviors," Section 12.7.2](pt03ch12s07hlb02.md), for more information).
