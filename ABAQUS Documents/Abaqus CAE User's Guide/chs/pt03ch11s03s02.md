# 11.3.2 基础特征





构建零件时创建的第一个特征称为基础特征；您通过添加更多特征来构建零件的其余部分，这些特征可以修改或向基础特征添加细节。使用零件模块中的工具构建Abaqus/CAE本机零件的过程遵循类似于在机械车间中构建零件的操作序列。例如，您从一块坯料（基础特征）开始，然后执行以下操作：
- 将其他部件连接到坯料（应用实体拉伸、旋转壳或草图线）。
- 从坯料上切掉材料（应用拉伸切割、旋转切割或圆孔；或对边缘进行圆角或倒角）。

创建新零件时，必须描述基础特征。您可以通过指定基础特征的两个属性来做到这一点：其形状和类型。形状表示特征的基本拓扑；即它是实体、壳、线框还是点。类型表示将使用以下哪种方法来生成基础特征：

**平面**

您在二维草图平面上绘制特征。

**拉伸**

您绘制特征轮廓，然后通过指定距离进行拉伸。

**旋转**

您绘制特征轮廓，然后绕轴旋转指定角度。

**扫掠**

您绘制两个形状：扫掠路径和扫掠轮廓。然后轮廓沿路径扫掠以创建特征。

**坐标**

您在提示区域输入单个点的坐标。

在创建零件并选择基础特征的形状和类型之前，您应该知道用于构建所需零件的序列。选择正确的基础特征类型和形状非常重要。

[表11-1](pt03ch11s03s02.md#asm-base)显示了基于零件建模空间和类型可以选择的基础特征。

**表11-1** 选择基础特征。
| 零件类型 | 建模空间 |
| --- | --- |
| 三维 | 二维或轴对称 |
| 可变形 | 任意 | 平面壳、平面线框或点 |
| 离散刚体 | 任意（必须先将3D实体离散刚体零件转换为壳才能实例化） | 平面线框或点 |
| 分析刚体 | 拉伸或旋转壳 | 平面线框 |
| 欧拉 | 拉伸、旋转或扫掠实体 | 不适用 |
| 流体 | 拉伸、旋转或扫掠实体 | 不适用 |
| 电磁 | 拉伸、旋转或扫掠实体 | 平面壳 |
