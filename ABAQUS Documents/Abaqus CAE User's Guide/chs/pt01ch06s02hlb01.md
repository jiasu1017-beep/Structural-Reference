# 6.2.1 选择和取消选择单个对象





在当前视口中选择和取消选择对象是使用标准方法的简单操作。有关选择视口的更多信息，请参阅["选择视口，" 第 4.3.2 节](pt01ch04s03hlb02.md)。

预选高亮允许你预览如果点击当前光标位置 Abaqus/CAE 将选择哪个对象。此外，草图器中的预选使用辅助光标来指示将被选择实体的确切位置和类型。（有关草图器预选的更多信息，请参阅["草图器光标和预选，" 第 20.4.5 节](pt03ch20s04s05.md)。）

你将最频繁地使用以下三种选择操作：

**点击选择一个对象**

要从当前视口中选择单个对象，请将光标移到对象上，然后单击鼠标按钮 1。
- 要选择一个点，请点击相应的点标记。选中后点标记会改变颜色。你可以选择的顶点用小实心圆标记；基准点用小空心圆标记。（有关基准点的信息，请参阅["理解基准几何的作用，" 第 62.1 节](pt06ch62s01.md)。）你可以选择的边中点和圆弧中心用小菱形标记。**注意：**使用草图模块时出现的一些选择标记与此处描述的不同。有关使用草图模块时选择对象的信息，请参阅["草图器光标和预选，" 第 20.4.5 节](pt03ch20s04s05.md)。
- 要选择一条边，请在远离任何顶点的位置点击边。选中的边会高亮显示。
- 要选择一个面，请在远离任何边或顶点的位置点击面。选中的面会以网格图案高亮显示。（网格图案与网格单元位置无关。）
- 要选择一个单元体，请点击它的任意面。选中单元体的所有边都会高亮显示。

如果你无法选择所需的对象，可以使用**选择**工具栏更改选择行为。有关更多信息，请参阅["使用选择选项，" 第 6.3 节](pt01ch06s03.md)。

一旦你选择一个对象，当前视口中之前选择的任何对象都会自动取消选择。

如果你的当前过程、选项和光标位置没有明确指定一个对象进行预选，Abaqus/CAE 会高亮显示所有潜在选择，并在光标箭头旁边添加省略号 (...) 以指示模糊的预选。如果你接受模糊的预选或以其他方式进行模糊选择，请使用提示区域中的按钮进行最终选择。更多信息，请参阅["循环浏览有效选择，" 第 6.2.10 节](pt01ch06s02hlb10.md)。

****[Shift]****+点击**选择其他对象**

要选择其他对象，请将光标移到对象上，然后 **[Shift]****+点击**。你的原始选择保持高亮显示，新选择的对象变得高亮显示。

选择多个对象的替代方法是围绕对象拖动一个矩形。更多信息，请参阅["拖动选择多个对象，" 第 6.2.2 节](pt01ch06s02hlb02.md)。

****[Ctrl]****+点击**取消选择对象**

要取消选择一个对象，请将光标移到对象上，然后 **[Ctrl]****+点击**。要取消选择所有对象，请在当前视口中未使用的区域点击。

当你完成在视口中选择和取消选择项目后，单击鼠标按钮 2 确认你的选择。你可以使用选择选项工具调整拖动选择区域的形状。你也可以选择拖动选择区域选择哪些对象。选择选项工具位于**选择**工具栏中。更多信息，请参阅["修改拖动选择区域的形状，" 第 6.3.5 节](pt01ch06s03s04.md)和["选择拖动选择区域选择哪些对象，" 第 6.3.6 节](pt01ch06s03s05.md)。
