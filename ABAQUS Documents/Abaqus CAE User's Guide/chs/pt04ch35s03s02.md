# 35.3.2 为中面建模使用延伸面工具





您可以使用Geometry Edit工具集中的延伸面工具来创建中面。延伸面工具延伸现有的壳面，因此对于中面建模，它必须在创建了一个或多个壳面之后使用。延伸面工具的详细使用在["延伸面"，第69.7.8节](pt06ch69s07hlb08.md)中讨论。

您可以通过选择单个边缘来延伸面以在一个或多个方向延伸面，或者您可以指定面以沿所有外部边缘延伸面。还有几种确定延伸距离的方法；**Up to reference representation**选项专门针对中面建模。

当您使用**Specify edges of faces to extend**选项沿一组边缘延伸面时，延伸面操作可能会失败。如果发生这种情况，请尝试减少所选边缘的数量，然后使用第二次操作沿剩余边缘延伸面。
