# 61.5.3 Displaying existing keyboard shortcuts







The **Shortcut Listing** dialog box displays a listing of all keyboard shortcuts currently assigned in Abaqus/CAE and the functions to which they map.

**To display a listing of keyboard shortcuts:**

1. From the main menu bar in any module, select ****Tools**![](../graphics/images/arrow.gif)**Customize****.
2. From the **Customize** dialog box that opens, click the **Functions** tab.
3. To display a list of existing keyboard shortcuts, click **Show all assignments**. From the **Shortcut Listing** dialog box that appears, you can review all currently defined keyboard shortcuts.
