# 47.4.29 对X–Y数据对象应用SAE滤波







### 47.4.29 对X–Y数据对象应用SAE滤波

使用一个SAE滤波函数对[之前保存的](pt05ch47s02hlb07.md) *X–Y* 数据对象（有序对的集合）应用SAE滤波运算，生成一个新的 *X–Y* 数据对象。例如，您可以使用这些函数去除高频噪声。

SAE滤波操作执行双通道、零相移、二阶Butterworth滤波。Abaqus/CAE中允许的SAE滤波类别为60、100、180、600和1000，如SAE Standard J211 (OCT88)所定义。Abaqus/CAE中的Visualization模块使用来自美国国家公路交通安全管理局（NHTSA）的信号分析软件。更多信息，请参见NHTSA网站（http://www.nhtsa.gov/ 上搜索NVS Software Applications）上信号分析软件安装包的文档。

Abaqus/CAE提供以下两种类型的SAE滤波函数：
- 通用运算符 **saeGeneralFilter** 使用通用滤波类别，可以为用户指定的截止频率过滤数据。
- 类别特定运算符过滤每个支持的SAE滤波类别：**sae60Filter**、**sae100Filter**、**sae180Filter**、**sae600Filter** 和 **sae1000Filter**。由于它们使用特定的滤波类别，这些运算符不支持指定截止频率。

您的 *X–Y* 数据对象必须具有恒定的时间步长才能被滤波。如果时间步长不是常数，Abaqus/CAE 通过[插值](pt05ch47s03s02.md)以恒定间隔计算额外点。SAE滤波的恒定时间步长由要滤波的 *X–Y* 数据对象中的最小时间步长定义。

**saeGeneralFilter** 函数需要两个参数：*X–Y* 数据对象的名称（*name*）和截止频率（*cutoffFrequency*），即滤波器衰减至少一半输入信号的频率。类别特定滤波器也需要两个参数：*X–Y* 数据对象的名称（*name*）和时间比例因子（*timeScaleFactor*），这是要过滤数据集的 *X* 值的乘数。例如，值为1表示以秒为单位的时间比例，值为0.001表示以毫秒为单位的时间比例。

可选参数的说明如下：
- 指定在数据信号起点处计算的投影和预充电方法的符号常数（*startCondition*）。此参数的有效值为 ZERO（应用零常投影和预充电）；CONSTANT（应用等于 *X–Y* 数据对象中第一个数据点的常投影和预充电）；MIRROR（应用相当于围绕第一个数据点处的垂直线反射 *X–Y* 数据对象的投影和预充电）；REVERSE_MIRROR（应用相当于围绕第一个数据点处的垂直线和水平线反射 *X–Y* 数据对象的投影和预充电）；以及 TANGENTIAL（应用与前两个数据点相切的线性投影和预充电）。默认值为 CONSTANT。
- 指定在数据信号终点处计算的投影和预充电方法的符号常数（*endCondition*）。此参数的有效值为 ZERO（应用零常投影和预充电）；CONSTANT（应用等于 *X–Y* 数据对象中最后一个数据点的常投影和预充电）；MIRROR（应用相当于围绕最后一个数据点处的垂直线反射 *X–Y* 数据对象的投影和预充电）；REVERSE_MIRROR（应用相当于围绕最后一个数据点处的垂直线和水平线反射 *X–Y* 数据对象的投影和预充电）；以及 TANGENTIAL（应用与最后两个数据点相切的线性投影和预充电）。默认值为 CONSTANT。
- 指定插值方案的符号常数（*interpolation*）。此参数的有效值为 QUADRATIC（指定拉格朗日二阶插值方案）；CUBIC_SPLINE（指定三次样条插值方案）；和 LINEAR（指定线性插值方案）。默认值为 QUADRATIC。
- 到达第一个数据点之前的原始数据曲线的斜率（*startslope*）。此参数的默认值为0.0（水平斜率），仅在 interpolation=CUBIC_SPLINE 时使用。
- 越过最后一个数据点之后的原始数据曲线的斜率（*endslope*）。此参数的默认值为0.0（水平斜率），仅在 interpolation=CUBIC_SPLINE 时使用。
- 指定是否对滤波后数据执行反向传递（*backwardPass*）的布尔值。此参数的默认值为 True。当此参数设置为 False 时，*endCondition* 参数将被忽略。

Abaqus/CAE参考下表确定类别特定SAE滤波器运算符的截止频率乘数（CM）。

| SAE类别 | 截止频率乘数（CM） |
| --- | --- |
| 60 | 100 |
| 100 | 165 |
| 180 | 300 |
| 600 | 1000 |
| 1000 | 1650 |

**对X–Y数据对象应用SAE滤波：**

1. 找到 **Operate on XY Data** 对话框。从主菜单栏，选择 ****Tools**![](../graphics/images/arrow.gif)**XY Data**![](../graphics/images/arrow.gif)**Create****. 在出现的对话框中点击 **Operate on XY data**；然后点击 **Continue**。**Operate on XY Data** 对话框出现。
2. 从列出的 **Operators** 中，点击 **saeGeneralFilter(X,F)** 进行通用SAE滤波，或点击一个类别特定的SAE滤波函数。您选择的滤波函数出现在表达式窗口中。
3. 从 **XY Data** 选项中，点击要操作的 *X–Y* 数据对象的名称，然后点击 **Add to Expression**。您可以从本次会话中之前保存的所有 *X–Y* 数据对象中选择（在 **XY Data** 字段中按字母顺序列出）。*X–Y* 数据对象名称出现在表达式窗口中函数括号内。
4. 如果您选择了 **saeGeneralFilter(X,F)**，请将光标定位在表达式窗口中并输入截止频率的值。
5. 如果您选择了类别特定的SAE滤波器，请将光标定位在表达式窗口中并输入时间比例因子的值。
6. 要继续构建表达式，将光标定位在表达式窗口中，然后输入或选择要包含的函数、运算符和 *X–Y* 数据。
7. 要评估并显示表达式，点击 **Plot Expression**。
8. 要[保存](pt05ch47s02hlb07.md)您的新 *X–Y* 数据对象，点击 **Save As**，然后在出现的对话框中提供名称。保存数据对象使其可用于本次会话中的未来操作以及包含多个数据对象的 *X–Y* 曲线图。
9. 完成后，点击 **Cancel** 关闭对话框。
