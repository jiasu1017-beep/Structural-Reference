# 17.18.7 对轮廓接触旋转轴的实体旋转区域进行扫掠网格划分





在大多数情况下，除非您创建了策略性的划分，否则如果用于创建区域的旋转轮廓接触旋转轴，则无法使用所有六面体单元对旋转实体区域进行扫掠网格划分。 例如，下图中的圆柱形零件无法进行扫掠网格划分。

![](../graphics/mgn-revpart.png)

使零件可进行扫掠网格划分的划分技术涉及将零件或零件实例划分为以下两个区域：
- 可使用拉伸扫掠网格划分技术进行网格划分的圆柱芯部区域。
- 可使用旋转扫掠网格划分技术进行网格划分的外部区域。

（有关扫掠网格划分实体区域的详细信息，请参阅["三维实体的扫掠网格划分，" 第17.9.3节](pt03ch17s09s03.md)。）

**提示：**以下说明引用了仅在选择**Mesh defaults**颜色映射时出现的颜色提示。 在尝试为扫掠网格划分划分实体旋转区域之前，将此颜色映射应用到您的视口。

**为扫掠网格划分划分实体旋转区域：**

1. 使用Partition工具集在区域中心创建一个圆柱芯部。 （有关划分的详细信息，请参阅[第70章，"The Partition toolset"](pt06ch70.md)。） 创建圆柱芯部的划分在下图中以红色勾勒。 ![](../graphics/mgn-revpartition.png) 圆柱芯部可以使用拉伸扫掠网格划分技术进行网格划分，因此变为黄色。 外部区域保持橙色，因为仍然无法进行网格划分。 ![](../graphics/mgn-revcore.png) 在下一步中，您将创建额外的划分，使Abaqus/CAE能够将外部区域识别为轮廓不接触旋转轴的旋转实体。
2. 使用Partition工具集创建任何必要的划分，以勾勒出外部区域的旋转轮廓。 该轮廓将作为旋转网格的源面，并将沿由圆柱芯部定义的圆形边缘扫掠，以创建实体网格。 例如，使用划分来勾勒下图中外部区域的轮廓（以红色显示）。 ![](../graphics/mgn-revprofile.png) 定义轮廓后，内部圆柱区域和外部区域都变为黄色，准备好进行扫掠网格划分。 ![](../graphics/mgn-revready.png) 生成的网格如下图所示。 ![](../graphics/mgn-revmesh.png) 即使源面和目标面不是平面的，您也可以应用类似的技术对旋转区域进行网格划分，如下图所示。 ![](../graphics/mgn-char-axis.png)
