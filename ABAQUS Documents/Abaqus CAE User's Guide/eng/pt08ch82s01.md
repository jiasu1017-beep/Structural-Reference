# 82.1 Running the Getting Started with Abaqus examples







This plug-in allows you to run the examples described in [Getting Started with Abaqus: Interactive Edition](../gsa/gsa-link.md#gsa). The plug-in creates a model that reproduces each example. The plug-in also creates a job associated with the model; however, the plug-in does not submit the job for analysis. If desired, you can go to the Job module and submit the job for analysis. You can then can go to the Visualization module to view the results of the analysis.

The plug-in fetches files associated with the examples and places those files in the current directory. You can run the following examples from the plug-in:
- ["Example: creating a model of an overhead hoist," Section 2.3 of Getting Started with Abaqus: Interactive Edition](../gsa/gsa-link.md#gsa-abs-model)
- ["Example: connecting lug," Section 4.3 of Getting Started with Abaqus: Interactive Edition](../gsa/gsa-link.md#gsa-ctm-connectlug)
- ["Example: skew plate," Section 5.5 of Getting Started with Abaqus: Interactive Edition](../gsa/gsa-link.md#gsa-shl-example-skewplate)
- ["Example: cargo crane," Section 6.4 of Getting Started with Abaqus: Interactive Edition](../gsa/gsa-link.md#gsa-bms-exacargocrane)
- ["Example: cargo crane under dynamic loading," Section 7.5 of Getting Started with Abaqus: Interactive Edition](../gsa/gsa-link.md#gsa-dyn-exacargodyn)
- ["Example: nonlinear skew plate," Section 8.4 of Getting Started with Abaqus: Interactive Edition](../gsa/gsa-link.md#gsa-nln-skewplate)
- ["Example: stress wave propagation in a bar," Section 9.4 of Getting Started with Abaqus: Interactive Edition](../gsa/gsa-link.md#gsa-ovw-exastresswave)
- ["Example: connecting lug with plasticity," Section 10.4 of Getting Started with Abaqus: Interactive Edition](../gsa/gsa-link.md#gsa-mat-connectlug)
- ["Example: blast loading on a stiffened plate," Section 10.5 of Getting Started with Abaqus: Interactive Edition](../gsa/gsa-link.md#gsa-mat-exablastload)
- ["Example: axisymmetric mount," Section 10.7 of Getting Started with Abaqus: Interactive Edition](../gsa/gsa-link.md#gsa-mat-exaximount)
- ["Example: vibration of a piping system," Section 11.3 of Getting Started with Abaqus: Interactive Edition](../gsa/gsa-link.md#gsa-stp-exapipingsys)
- ["Abaqus/Standard 3D example: shearing of a lap joint," Section 12.8 of Getting Started with Abaqus: Interactive Edition](../gsa/gsa-link.md#gsa-cnt-exalapjoint)
- ["Abaqus/Explicit example: circuit board drop test," Section 12.11 of Getting Started with Abaqus: Interactive Edition](../gsa/gsa-link.md#gsa-cnt-circuitboard)
- ["Example: forming a channel in Abaqus/Explicit," Section 13.5 of Getting Started with Abaqus: Interactive Edition](../gsa/gsa-link.md#gsa-qsi-exadeepdraw)
