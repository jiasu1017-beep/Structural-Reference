# 10.9.7 将模型数据导出为Nastran批量数据文件格式





在作业模块中，从主菜单栏选择**作业**![](../graphics/images/arrow.gif)**导出**![](../graphics/images/arrow.gif)**Nastran输入文件**![](../graphics/images/arrow.gif)***作业名***，可以将与Abaqus作业关联的模型数据和作业数据导出为Nastran批量数据文件（`.bdf`）格式。

**提示：**您也可以从模型树启动此导出。展开**作业**容器，在作业上单击鼠标按钮3，然后选择**导出**![](../graphics/images/arrow.gif)**Nastran输入文件**。

您创建的Nastran批量数据文件包括Abaqus模型和作业中的选定数据；有关支持的Abaqus关键字及其映射到的Nastran实体的更多信息，请参见"Abaqus文件到Nastran批量数据文件的转换"，Abaqus分析用户指南第3.2.29节。批量数据文件包括Abaqus模型中选定的历史数据，但不包括Nastran不支持的零件和装配体。![](../graphics/images/black4rule.gif)有关相关信息，请单击以下任一项：- "使用文件选择对话框"，第3.2.10节- "导入模型"，第10.8节- "将Abaqus文件转换为Nastran批量数据文件"，Abaqus分析用户指南第3.2.29节
