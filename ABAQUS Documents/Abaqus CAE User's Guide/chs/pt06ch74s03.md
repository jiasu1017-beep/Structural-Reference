# 74.3 显示和隐藏流







您可以通过在**Stream Manager**中切换其**Show**复选框来在当前视口中单独显示或隐藏流，您可以通过单击**Set All On**或**Set All Off**来显示或隐藏会话中的所有流。该管理器显示会话中定义的所有流，并指示每个流当前是否显示。此外，您可以通过切换其**Upstream**或**Downstream**复选框来显示或隐藏每个流的上游或下游部分。

**Stream Manager**还显示会话中每个流的耙上的点数。您可以直接从管理器中编辑流的此数字，但以下情况除外：在圆形路径上定义的流或使用三个或更多点或节点的路径定义定义的流。
