# 68.1 自定义内存限制和再生选项







几何状态是部件或独立部件实例的内部表示的快照。在内存缓存中保存几何状态可以加快再生性能；但是，保存的状态会消耗大量内存并降低整体性能。

**Options**（选项）对话框的 **Memory**（内存）选项卡页面上的选项使您能够控制分配给 Abaqus 内核的内存量，指定将提示 Abaqus/CAE 以减少内存模式运行的内存使用百分比，并调整保存状态的便利性与增加内存消耗导致的性能下降之间的平衡。从主菜单栏中选择 ****Tools**![](../graphics/images/arrow.gif)**Options****（工具→选项）以显示 **Options**（选项）对话框，然后点击 **Memory**（内存）选项卡。

在大多数情况下，默认设置将产生可接受的再生性能。只有在 Abaqus/CAE 再生部件或装配花费很长时间时，您才应该修改特征选项有关更多信息，请参阅["调整特征再生，" 第 65.3 节"](pt06ch65s03.md)。

**自定义会话的内存限制和内存缓存设置：**

1. 从任何模块的主菜单栏中选择 ****Tools**![](../graphics/images/arrow.gif)**Options****（工具→选项）。出现 **Options**（选项）对话框。
2. 显示 **Memory**（内存）选项卡页面。
3. 如果需要，在 **Kernel memory limit**（内核内存限制）中指定新的值（以 MB 为单位）。如果此设置为零，则不限制内核内存。
4. 切换开启 **Run in reduced memory mode when memory reaches**（当内存达到时以减少内存模式运行），并指定内核内存限制的百分比，超过该百分比时 Abaqus/CAE 以减少内存模式运行。Abaqus/CAE 通过将几何数据分页到磁盘或删除可以自动再生（需要时）的非活动部件中的数据来减少内存。
5. 如果需要，更改 Abaqus/CAE 在内存中自动缓存的几何状态数量。理想情况下，为了获得最大的再生性能，您会在每次特征修改后保存几何状态。但是，存储大量几何状态会消耗大量内存并阻碍整体性能。决定保存多少个状态是在再生速度和内存消耗之间的权衡。有关更多信息，请参阅["什么是几何状态？" 第 65.3.2 节"](pt06ch65s03s02.md)。
6. 如果您发现 Abaqus/CAE 在您的工作站上消耗了大量内存并影响了系统性能，您可以使用 **Options**（选项）对话框中的 **Clear**（清除）按钮来擦除与部件和装配缓存关联的内存。清除此内存将提高性能；但是，几何状态的快照将被擦除，再生将变慢。 有关更多信息，请参阅["什么是缓存？" 第 65.3.3 节"](pt06ch65s03s03.md)。
7. 如果您想在内存中保存当前状态，请点击 **Options**（选项）对话框中的 **Cache current state**（缓存当前状态）。Abaqus/CAE 可以返回此状态并再生任何后续的添加或修改。**Options**（选项）对话框指示缓存中存储了多少几何状态。点击 **Query**（查询）以确定几何状态相对于当前部件缓存或装配缓存中特征序列的位置。当 Abaqus/CAE 以减少内存模式运行时，如果内存消耗高于指定阈值，几何状态可能会被删除。
8. 点击 **OK**（确定）更改当前模型的内存限制和再生性能选项。这些选项仅适用于当前会话，不会保存在模型数据库中。
