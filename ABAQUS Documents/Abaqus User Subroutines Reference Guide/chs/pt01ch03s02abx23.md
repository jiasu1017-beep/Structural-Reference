### 概述

**abaqus odbcombine**实用工具将两个或多个Abaqus输出数据库文件(`.odb`)中的结果数据组合成单个输出数据库(`.odb`)文件。**abaqus odbcombine**实用工具旨在组合包含不同结果的输出数据库。如果您想在重启前后组合同一分析的输出数据库，请改用**abaqus restartjoin**执行程序。有关更多信息，请参阅["连接重启分析的输出数据库(`.odb`)文件，" 第3.2.21节](pt01ch03s02abx21.md)。

Abaqus在组合输出数据库中包含来自所选输出数据库的所有模型数据；但是，对于结果数据，您可以选择包含所指定输出数据库中的数据子集。Abaqus/CAE根据两个因素确定组合输出数据库中包含哪些结果数据：您指定的过滤选项和您选择的主输出数据库。

### 过滤器

您可以过滤实用工具包含在组合输出数据库中的数据，以仅包含所选步骤或帧中的结果、所选输出变量中的结果，或这些选项的组合。例如，过滤器可以仅让您包含所指定输出数据库的最后一步和最后帧的结果，并且同一过滤器可以规定仅将Mises应力结果包含在组合输出数据库中。如果要为第一步设置与第二步不同的过滤条件，您也可以建立多个过滤器。

**abaqus odbcombine**实用工具还提供两级过滤：输出数据库特定过滤器（仅过滤单个输出数据库的结果）和默认过滤器（适用于整个作业）。输出数据库特定过滤器的优先级高于默认过滤器，因此Abaqus/CAE仅在默认过滤器定义的内容与某个单独输出数据库的过滤器不冲突时才使用默认过滤器中的设置。

过滤语法足够灵活，允许您指定多个步骤、帧或输出变量值。您可以在逗号分隔列表中指定多个步骤名称，例如`Step-1, Step-2, Step-4`。对于帧，您可以包含范围或单独的值；例如，输入`1, 3, 5, 7:9`返回帧1、3、5、7、8和9到组合输出数据库。

您还可以使用符号常量`'ALL'`、`'FIRST'`和`'LAST'`作为快捷方式，指定要包含的数据。这些选项使您能够包含所有步骤或帧的结果数据和所有输出变量的数据，而不是一个或多个所选变量。

### 主输出数据库

每个组合操作中的输出数据库被指定为主输出数据库。实用工具首先将所有字段输出数据（根据过滤选择）从主输出数据库传输到组合输出数据库。然后，实用工具在后续输出数据库中找到匹配步骤和帧的结果数据，并仅将这些数据复制到组合输出数据库。此策略为组合结果数据提供了更连贯的结构。

### 配置文件使用

**abaqus odbcombine**实用工具使用配置文件中的数据来确定要组合哪些输出数据库、要指定为主输出数据库的文件，以及要强制执行的默认过滤选项和每个输出数据库的过滤选项。配置文件必须为`.xml`格式，它可以按以下顺序包含三种类型的元素：
- `<DefaultFilters>`元素指定一个或多个默认过滤定义。此部分是可选的，但如果要为组合操作设置默认过滤，则必须包含它。
- `<MasterOdb>`元素指定主输出数据库的位置，如果需要，还可以指定该输出数据库中数据的一个或多个过滤定义。此部分是必需的。
- 每个要包含在组合操作中的其他输出数据库需要一个`<Odb>`元素。

然后，您可以通过在`<DefaultFilters>`元素内嵌入`<Filter>`元素或在某个输出数据库元素内嵌入`<Filter>`元素来为输出数据库特定过滤器设置默认过滤器。

### 配置文件模板

以下示例说明了**abaqus odbcombine**实用工具配置文件的结构。

```
<?xml version='1.0' encoding='UTF-8'?>
*Your XML file declaration may differ from this one.*
<OdbInput>    <DefaultFilters>
    *The default filtering element is optional. If you include this element in the configuration file,*
    *you must include at least one <Filter> element within this section. Filter elements can*
    *use the Steps or Frames attributes to refer to symbolic constants or the StepName or*
    *FrameIndex attributes to refer to individual steps or frames, as shown in the following examples:*
        <Filter Steps='*step names or symbolic constants*'
            Frames='*frame numbers or symbolic constants*'
            VariableName='*variable*' />
        <Filter StepName='*full step name*'
            FrameIndex='*individual frame number*'
            VariableName='*variable*' />      </DefaultFilters>
    <MasterOdb Name='*path to master output database*'>
    *Filtering elements for the master output database are optional. If you want to filter*
    *the data from this output database, include a <Filter> element within this section*
    *for each filtering option you want to define.*
    </MasterOdb>
    <Odb Name='*path to output database*'>
    *Filtering elements for the output database are optional. If you want to filter*
    *the data from this output database, include a <Filter> element within this section*
    *for each filtering option you want to define.*
    </Odb>
    *Append an <Odb> element for each additional output database you want to include.*
</OdbInput>
```

### 组合输出数据库中不包含的数据

组合输出数据库文件时，以下类型的输出数据不包含：
- 历史输出。
- 表面数据。
- 解析刚体部件实例的数据。
- 与字段输出数据关联的局部坐标系。

### 命令行选项

**job**

此选项指定生成的组合输出数据库的名称和日志文件的名称。Abaqus也按此名称搜索配置文件。

如果命令行上省略了此选项，Abaqus将提示您输入其值。

**input**

此选项指定配置文件的名称，该文件指定要组合的输出数据库以及要包含在组合中的步骤、帧和输出变量。配置文件必须为`.xml`格式。

**verbose**

此选项指定Abaqus写入日志文件的消息详细级别。可能的值为`1`或`2`。如果指定`1`，Abaqus仅向日志文件写入错误和警告；如果指定`2`，Abaqus还记录您选择的过滤选项，并列出成功复制到组合输出数据库的模型数据和字段输出数据。
