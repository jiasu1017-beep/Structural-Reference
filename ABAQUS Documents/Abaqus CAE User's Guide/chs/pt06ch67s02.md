# 67.2 创建或编辑自由体切割







要创建自由体切割，请选择构成自由体横截面的节点和单元；即选择模型中您想要查看合力和力矩的曲面。Abaqus/CAE 提供了几种选择横截面节点和单元的方法：
- 对于三维物体，您可以选择要包含在横截面中的单独单元面。所有连接到所选面节点的单元的贡献都将被考虑。
- 对于二维或三维物体，您可以选择要包含在横截面中的单独单元边。每条边为 Abaqus/CAE 提供一个单元定义和两个或三个要包含在横截面中的节点。所有连接到所选边节点的单元的贡献都将被考虑。
- 对于任何类型的模型，您可以单独选择要包含的节点和单元。此方法推荐用于从几何区域（如梁的端部）选择节点和单元，在这些区域中，使用它们的边或面选择节点和单元可能很别扭或困难。仅考虑所选单元的贡献。

您还可以在视图切割上显示自由体切割。有关更多信息，请参阅["理解视图切割，" 第 80.1 节"](pt07ch80s01.md)。

当您使用单元面或单元边定义横截面时，Abaqus/CAE 提供了三种额外选择方法，使您能够选择此定义的节点和单元：
- 您可以通过在视口中选择组件（单独或按角度）来选择面或边。
- 您可以选择在输出数据库中定义的曲面集。
- 您可以选择一个或多个显示组用作曲面定义的一部分。

当您通过单独选择节点和单元来定义横截面时，您可以通过直接在 **Free Body Cross-Section**（自由体横截面）对话框中输入它们的标签来指定要包含在定义中的节点或单元。

Abaqus/CAE 还使您能够自定义自由体切割的求和点，这是计算合力矩的三维位置。默认情况下，求和点放置在贡献自由体计算的单元面的面积质心处，但您可以将求和点移动到横截面节点的质心位置，或移动到模型中的任意三维位置。

您还可以调整自由体切割的分量向量显示方向。在许多情况下，自由体切割最理想的坐标系是一个轴垂直于模型曲面、另一个轴切向于曲面的坐标系。此"法向和切向"坐标系在内部创建，默认用于分量向量的显示。但是，您可以更改方向以使用会话中可用的任何自定义坐标系。

当您编辑自由体切割时，Abaqus/CAE 使您能够更改包含在自由体横截面中的分量、求和点设置和分量解析设置。但是，您无法更改用于定义自由体切割的选择方法；例如，如果您使用三维单元面定义了自由体切割，则对自由体切割的后续编辑也将使用此方法。

默认情况下，自由体切割仅在会话期间保留。如果您希望保留已定义的自由体切割以供后续会话使用，可以将其保存到 XML 文件、模型数据库或输出数据库。有关更多信息，请参阅["管理会话对象和会话选项，" 第 9.9 节"](pt02ch09s08.md)。

**创建或编辑自由体切割：**

1. 找到用于创建或编辑自由体切割的选项： - 要创建自由体切割，请在可视化模块中从主菜单栏选择 ****Tools**![](../graphics/images/arrow.gif)**Free Body Cut**![](../graphics/images/arrow.gif)**Create****（工具→自由体切割→创建）。**提示：**您也可以使用可视化模块工具箱中的 ![](../graphics/ico_freeBodyDiagram.png) 工具来创建自由体切割。 - 要编辑自由体切割，请在可视化模块中从主菜单栏选择 ****Tools**![](../graphics/images/arrow.gif)**Free Body Cut**![](../graphics/images/arrow.gif)**Edit****（工具→自由体切割→编辑），然后选择要编辑的自由体切割。**提示：**您也可以使用自由体切割管理器来编辑自由体切割。从主菜单栏选择 ****Tools**![](../graphics/images/arrow.gif)**Free Body Cut**![](../graphics/images/arrow.gif)**Manager****（工具→自由体切割→管理器），然后从出现的对话框中点击 **Edit**（编辑）。打开 **Free Body Cross-Section**（自由体横截面）对话框，您可以在其中自定义构成横截面的边、面或单元和节点。
2. 如果您正在创建自由体切割，请从 **Selection Method**（选择方法）选项中选择以下之一，然后点击 **OK**（确定）： - 选择 **Based on view cut**（基于视图切割）以基于当前视图切割计算和显示自由体切割。如果您选择此选择方法，Abaqus/CAE 会打开 **View Cut Manager**（视图切割管理器），您可以从该管理器显示和定位模型的视图切割，并自定义关联自由体切割的显示。有关更多信息，请参阅["自定义活动视图切割上合力和力矩的计算和显示，" 第 80.2.8 节"](pt07ch80s02hlb08.md)。 - 选择 **2D element edges**（2D 单元边）通过选择单元边隐式（或间接）指定节点和单元。此方法推荐用于二维模型。 - 选择 **3D element faces**（3D 单元面）通过选择单元面隐式（或间接）指定节点和单元。此方法推荐用于三维模型。 - 选择 **Elements and nodes**（单元和节点）以单独选择自由体横截面的单元或节点。打开 **Free Body Cross-Section**（自由体横截面）对话框，您可以在其中选择构成横截面的边、面或单元和节点。
3. 从对话框左上角的 **Item**（项目）列表中，选择 **Surfaces**（曲面）、**Display groups**（显示组）、**Elements**（单元）或 **Nodes**（节点）作为用于自由体横截面的模型组件类型。Abaqus/CAE 刷新 **Method**（方法）列表。
4. 从 **Method**（方法）列表中选择一种选择方法；和/或通过从视口中选择、从对话框右侧出现的列表中选择项目，或在对话框右侧输入数据来选择自由体横截面的特定项目。有关选择选项的更多信息，请参阅["自由体横截面的选择方法，" 第 67.3 节"](pt06ch67s03.md)。可以在视口中高亮显示某些项目以验证您的选择。如果可用，请切换 **Highlight items in viewport**（在视口中高亮显示项目）。
5. 完成选择后，点击 **OK**（确定）关闭 **Free Body Cross-Section**（自由体横截面）对话框。打开 **Edit Free Body Cut**（编辑自由体切割）对话框。
6. 如果需要，自定义所选自由体切割的求和点或坐标系变换选项。 1. 从 **Summation point**（求和点）选项中，选择自由体切割中向量起源的三维位置。 - 选择 **Centroid of cut**（切割质心）以将求和点自动放置在贡献自由体计算的单元面的面积质心处。 - 选择 **Nodal average**（节点平均）以将求和点自动放置在包含在自由体切割中的所有节点的平均变形坐标处。 - 在 **User-defined**（用户定义）字段中指定自定义三维位置。 2. 从 **Component resolution**（分量解析）选项中，您可以指定在以分量形式显示向量时发生的坐标系变换。（有关以分量形式显示力和力矩向量的更多信息，请参阅["自定义自由体切割的常规显示选项，" 第 67.5.1 节"](pt06ch67s05s01.md)。） - 选择 **Normal and tangential**（法向和切向）以使分量向量与您选择的曲面的法向和切向对齐。 - 选择 **CSYS** 和一个坐标系以将分量向量变换到自定义坐标系。**Component resolution**（分量解析）选项仅当在 **Free Body Plot Options**（自由体绘图选项）对话框中选择了分量向量显示时才会影响自由体切割的显示。
7. 点击 **OK**（确定）。Abaqus/CAE 创建自由体切割并将其显示在视口中。
