# 35.3 Creating shell faces for the midsurface model







Once you have a reference representation, you must manually build a shell model to replace the solid cells that have been removed from the model. You can use the shell feature tools and the tools in the Geometry Edit toolset (for more information, see ["Adding a shell feature," Section 11.22](pt03ch11s22.md), and [Chapter 69, "The Geometry Edit toolset](pt06ch69.md),” respectively) to create and edit shell faces. 

This section discusses some special conditions that may apply when you use these tools to create faces for a midsurface model. The following topics are covered:
- ["Using the offset face tool for midsurface modeling," Section 35.3.1](pt04ch35s03s01.md)
- ["Using the extend faces tool for midsurface modeling," Section 35.3.2](pt04ch35s03s02.md)
