# 2.6.3 查找在线文档的特殊章节





以下**帮助**菜单项允许您显示您可能觉得有用的HTML文档部分：

**关于模块**

选择****帮助**![](../graphics/images/arrow.gif)**关于模块****可显示[Abaqus/CAE用户指南](book01.md)，其中包含描述当前模块的章节的开头。如果您尚未进入模块，指南将打开到模块概念描述。在任一情况下，您都可以自由阅读额外需要的信息，并可以对整个指南进行文本搜索。

**关于帮助**

选择****帮助**![](../graphics/images/arrow.gif)**关于帮助****可显示[Abaqus/CAE用户指南](book01.md)，其中包含描述如何使用帮助系统的部分。您也可以自由阅读额外需要的信息，并可以对整个指南进行文本搜索。

**入门**

选择****帮助**![](../graphics/images/arrow.gif)**入门****可显示提供有关在Abaqus/CAE窗口中工作基本信息的部分。此部分还包含指向[Abaqus入门：交互版](../gsa/gsa-link.md#gsa)指南中有帮助教程的链接。

**发布说明**

选择****帮助**![](../graphics/images/arrow.gif)**发布说明****可显示[Abaqus发布说明](../rnb/rnb-link.md#rnb)。发布说明详细说明了软件的新功能，并提供了更新和增强功能的列表。
