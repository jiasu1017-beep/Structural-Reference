# 20.8.1 通过拖动对象修改草图







您可以将顶点、边、构造几何和尺寸文本拖动到草图中的新位置。您一次只能拖动一个对象，当**Sketcher 选项**对话框中的**捕捉到网格**选项处于打开状态时，您拖动的对象会捕捉到网格线。此选项默认处于打开状态。

当您拖动顶点或边时，Abaqus/CAE 重新定位选择并使用**拖动期间**约束求解方法来修改任何连接的边（有关更多信息，请参见["在 Sketcher 中自定义约束的使用，"第 20.9.10 节](pt03ch20s09hlb10.md)）。草图约束可能导致多个对象与您选择的对象一起移动。约束也可能限制或阻止拖动某些对象。例如，考虑如图 20-8](pt03ch20s08s01.md#ske-modobj)所示的草图。

**图 20-8** 原始草图。

![](../graphics/ske-modobj.png)

草图包括 Abaqus/CAE 在草绘过程中创建的默认水平、垂直和垂直约束以及一个添加的长度尺寸。有了这些约束，拖动草图的左边与拖动左边任一端的顶点具有相同的效果，如图 20-9](pt03ch20s08s01.md#ske-modobj-move1)所示。

**图 20-9** 在有约束的情况下拖动左边或顶点。

![](../graphics/ske-modobj-move1.png)

移除约束和尺寸后，拖动边会产生与图 20-9](pt03ch20s08s01.md#ske-modobj-move1)类似的结果，但拖动顶点会修改草图，如图 20-10](pt03ch20s08s01.md#ske-modobj-move2)所示。图 20-10](pt03ch20s08s01.md#ske-modobj-move2)中的结果使用默认的**最小移动**约束求解方法实现。对于此草图，使用任何其他约束求解方法拖动顶点都会重新定位整个草图。

**图 20-10** 在无约束的情况下拖动左下顶点。

![](../graphics/ske-modobj-move2.png)

拖动边和顶点是一种快速但不精确的移动方法，因为运动基于光标位置而不是精确的数值变化。要更精确地移动对象，请使用["通过更改尺寸或添加参数修改对象，"第 20.8.2 节](pt03ch20s08s02.md)和["通过选择边修改或复制对象，"第 20.8.3 节](pt03ch20s08s03.md)中描述的方法。

拖动尺寸文本是在使用**自动尺寸**工具后清理草图的一种方法。由于尺寸文本的确切位置并不重要，因此精确度无关紧要。拖动尺寸文本时，您可以进行以下更改：
- 将线性尺寸移近、移远或移至尺寸量的另一侧。
- 将角度尺寸移近或远离角度的顶点。
- 将角度尺寸移到定义角度的一条边上，以标注原始角度的补角。

拖动角度尺寸的可能位置如图 20-11](pt03ch20s08s01.md#ske-modobj-angledim)所示。

**图 20-11** 角度尺寸的四个可能位置。

![](../graphics/ske-modobj-angledim.png)

尺寸文本不受任何其他约束的控制，仅受其附加到尺寸对象的约束。约束求解方法也不会影响草图中尺寸文本的拖动。![](../graphics/images/black4rule.gif)有关相关主题的信息，请点击以下任一项目：- ["指定精确几何，"第 20.6 节](pt03ch20s06.md)
