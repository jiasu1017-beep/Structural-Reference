# 2.6.1 显示上下文相关帮助





您可以使用主菜单栏上的帮助工具![](../graphics/ico_help.png)来显示Abaqus/CAE中任何图标、菜单或对话框的详细HTML帮助。当您单击帮助工具然后单击Abaqus/CAE窗口中的项目时，会出现一个包含在线文档中与该项目相关部分的帮助窗口。

**显示主窗口或对话框中项目的帮助：**

1. 单击主菜单栏上的帮助工具![](../graphics/ico_help.png)。**提示：**您也可以从主菜单栏选择****帮助**![](../graphics/images/arrow.gif)**上下文帮助****。光标会变为问号。
2. 将光标放在需要帮助的项目上，然后单击鼠标按钮1。会出现一个帮助窗口。该窗口包含相关的在线文档和指向相关主题的链接。

或者，您可以使用**[F1]**键显示特定项目的帮助。在大多数情况下，您可以使用**帮助**菜单、帮助工具图标或**[F1]**键访问上下文相关帮助。但是，如果您正在寻找有关不允许访问帮助工具的菜单项或对话框的信息，则必须使用**[F1]**。

**使用[F1]键显示帮助：**

1. 单击Abaqus/CAE窗口中需要帮助的功能。如果该功能是菜单的一部分，请不要释放鼠标按钮。
2. 按**[F1]**。会出现一个帮助窗口。该窗口包含相关的在线文档和指向相关主题的链接。如果您选择了菜单项而没有释放鼠标按钮，该菜单会消失。

**注意：**Abaqus/CAE还提供简短的"工具提示"来描述工具箱和工具栏中工具的功能。要查看"工具提示"，请将光标放在工具上并保持一小段时间不动。
