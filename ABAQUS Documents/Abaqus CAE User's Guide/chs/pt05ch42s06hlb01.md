# 42.8.2 创建固定坐标系





您可以通过指定原点位置和两个点来定义固定坐标系。从原点到点1的线是*X*轴（对于直角坐标系）或*R*轴（对于圆柱坐标系或球坐标系）；包含原点、点1和点2的平面是*X-Y*平面（对于直角坐标系）或*R-*![](../graphics/usi_eqn00065.gif)平面（对于圆柱坐标系或球坐标系）。无论模型如何运动，坐标系在空间中保持固定。

您可以在视口中选择单个节点，或在提示区域中输入点的*X*、*Y*和*Z*坐标。当您在视口中选择节点时，将使用当前绘图状态的节点坐标来定向坐标系；也就是说，使用未变形或变形形状上节点的真实坐标（变形形状上应用的任何变形比例因子都会被忽略）。

**创建固定坐标系：**

1. 从主菜单栏，选择****工具**![](../graphics/images/arrow.gif)**坐标系**![](../graphics/images/arrow.gif)**创建****。此时将出现**创建坐标系**对话框。**提示：**您也可以使用工具箱中的![](../graphics/ico_dtmCsys.png)工具或当前输出数据库结果树中的**会话坐标系**容器来创建坐标系。
2. 在**创建坐标系**对话框中：1. 接受坐标系的默认名称，或在文本字段中输入您选择的名称。2. 切换**固定系统**以指定坐标系的运动。3. 选择以下之一作为坐标系类型：- **直角**：*X*、*Y*和*Z*轴分别解释为1、2和3轴。- **圆柱**：*R*、![](../graphics/usi_eqn00065.gif)和*Z*轴分别解释为1、2和3轴。- **球面**：*R*、![](../graphics/usi_eqn00065.gif)和![](../graphics/usi_eqn00226.gif)轴分别解释为1、2和3轴。4. 点击**继续**。Abaqus/CAE在提示区域显示提示，以帮助您定义坐标系轴。
3. 在视口中选择一个节点作为原点；或在提示区域中输入原点的*X*、*Y*和*Z*坐标。
4. 在视口中选择一个节点位于*X*轴（对于直角坐标系）或*R*轴（对于圆柱坐标系或球坐标系）上；或在提示区域中输入该点的*X*、*Y*和*Z*坐标。
5. 在视口中选择一个节点位于*X-Y*平面（对于直角坐标系）或*R-*![](../graphics/usi_eqn00065.gif)平面（对于圆柱坐标系或球坐标系）上；或在提示区域中输入该点的*X*、*Y*和*Z*坐标。坐标系出现在视口中、**坐标系管理器**的列表中，以及当前输出数据库结果树中的**会话坐标系**容器下。Abaqus仅将用户定义的坐标系保存到输出数据库关闭为止，除非您将它们保存到输出数据库文件。坐标系一旦创建就不能编辑；您可以重命名或删除尚未保存到输出数据库的坐标系。
