# 17.1 理解网格模块的作用







网格模块允许您在 Abaqus/CAE 中创建的零件和装配体上生成网格。提供各种级别的自动化和控制，以便您可以创建满足分析需求的网格。与创建零件和装配体一样，为模型分配网格属性（如种子、网格划分技术和单元类型）的过程是基于特征的。因此，您可以修改定义零件或装配体的参数，而您在网格模块中指定的网格属性会自动重新生成。

网格模块提供以下功能：
- 用于在局部和全局级别规定网格密度的工具。
- 模型着色，用于指示分配给模型中每个区域的网格划分技术。
- 多种网格控制，例如： - 单元形状 - 网格划分技术 - 网格划分算法 - 自适应重网格划分规则
- 用于为网格单元分配 Abaqus/Standard、Abaqus/Explicit 或 Abaqus/CFD 单元类型的工具。这些单元可以属于您创建的模型，也可以属于孤立网格。
- 用于验证网格质量的工具。
- 用于细化网格和改进网格质量的工具。
- 用于将网格化装配体或所选零件实例保存为网格零件的工具。
