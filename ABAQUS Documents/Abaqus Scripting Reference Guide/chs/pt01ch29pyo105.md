# 29.105 VelocityDependence 对象

VelocityDependence 对象指定材料渗透率对流体流速的依赖性。

**访问**

```
import material
mdb.models[*name*].materials[*name*].permeability.velocityDependence
import odbMaterial
session.odbs[*name*].materials[*name*].permeability.velocityDependence
```

### 29.105.1 VelocityDependence(...)

此方法创建 VelocityDependence 对象。

**路径**

```
mdb.models[*name*].materials[*name*].permeability.VelocityDependence
session.odbs[*name*].materials[*name*].permeability.VelocityDependence
```

**必需参数**

*table*

Float 元组序列，指定下述项目。

**可选参数**

无。

**表格数据**

- ![](../graphics/ker_eqn00095.gif)。仅允许 ![](../graphics/ker_eqn00383.gif) ≥ 0.0。
- 孔隙比，![](../graphics/ker_eqn00289.gif)。

**返回值**

VelocityDependence 对象。

**异常**

RangeError。

### 29.105.2 setValues(...)

此方法修改 VelocityDependence 对象。

**必需参数**

无。

**可选参数**

`setValues` 的可选参数与 [VelocityDependence](pt01ch29pyo105.md#ker-velocitydependence-velocitydependence-pyc) 方法的参数相同。

**返回值**

无

**异常**

RangeError。
