# 12.1 应用程序对象





应用程序对象管理消息队列、计时器、任务、GUI更新和其他系统设施。每个应用程序都会有一个应用程序对象，你通常在应用程序的启动文件中创建它。有关更多信息，请参见["启动脚本，"第11.2节"](pt06ch11s02.md)。应用程序对象的构造函数接受以下参数：

```
AFXApp(appName, vendorName, productName, majorNumber, minorNumber, 
       updateNumber, prerelease)
```

应用程序名称和供应商名称旨在作为注册表的键。注册表是存储在应用程序会话之间持久化的设置的地方；例如，应用程序启动时在桌面上的大小和位置。当前Abaqus不使用注册表，但这些键作为占位符包含以备将来使用。注册表将有各种部分，允许你对设置进行分组。某些设置可能适用于特定供应商的所有产品，某些设置可能仅适用于供应商的特定产品。

默认情况下，Abaqus在主窗口标题栏中显示产品名称和发布编号；有关更多信息，请参见["标题栏，"第13.2节"](pt06ch13s02.md)。
