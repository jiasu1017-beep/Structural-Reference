# 12.2.2 Abaqus/Explicit 与 Abaqus/Standard 之间的结果传递

**产品：** Abaqus/Standard 和 Abaqus/Explicit

##### **参考资料**

- ["结果传递概述，" 第 12.2.1 节](pt04ch09s02aus54.md)
- [*MODEL CHANGE](../key/key-link.md#usb-kws-hmodelchange)
- ["Abaqus/Standard 和 Abaqus/Explicit 的比较，" 第 3.5 节](../gen/gen-link.md#gen-anl-explicit-standard)

### 概述

Abaqus 允许在 Abaqus/Explicit 和 Abaqus/Standard 之间传递结果。这种能力使得可以利用每种产品的优势：

- **Abaqus/Explicit**：适合处理复杂接触、大变形和高度非线性问题
- **Abaqus/Standard**：适合处理线性静力、模态分析和稳态传热问题

### 支持的传递方向

| **从** | **到** | **用途** |
| --- | --- | --- |
| Abaqus/Explicit | Abaqus/Standard | 详细 Explicit 分析后的 Standard 后续分析 |
| Abaqus/Standard | Abaqus/Explicit | Standard 分析后的 Explicit 后续分析 |

### 传递的内容

结果传递包括：

- 位移和旋转
- 应力分量
- 应变分量
- 温度场（如果适用）
- 速度场（仅 Abaqus/Explicit）
- 加速度场（仅 Abaqus/Explicit）

### 传递方法

使用 `*MODEL CHANGE, ADD` 或 `*INITIAL CONDITIONS, TYPE=FIELD` 传递结果：

| **输入文件用法：** | ``` [*MODEL CHANGE](../key/key-link.md#usb-kws-hmodelchange), ADD, ELSET=*element_set* ``` |

### 要求和限制

1. **网格兼容性**：两个模型的网格必须匹配或可映射
2. **单元类型**：必须兼容
3. **材料模型**：必须一致
4. **单位制**：必须相同
5. **时间/增量匹配**：目标分析的初始时间应与传递状态的时间对应

### 应用示例

#### 落锤冲击分析（Abaqus/Explicit）后的屈曲分析（Abaqus/Standard）

1. 在 Abaqus/Explicit 中执行落锤冲击模拟
2. 将冲击后的应力状态传递到 Abaqus/Standard
3. 在 Abaqus/Standard 中执行线性屈曲分析

#### 成形分析（Abaqus/Explicit）后的回弹分析（Abaqus/Standard）

1. 在 Abaqus/Explicit 中执行成形过程
2. 将成形后的状态作为初始条件
3. 在 Abaqus/Standard 中执行回弹分析

### 注意事项

1. **显式到隐式传递**：通常较直接
2. **隐式到显式传递**：需要确保状态适合显式求解
3. **接触状态**：接触定义需要在接收模型中重新定义
4. **初始条件**：显式分析中的速度场可以传递

### 执行

```
# 从 Explicit 传递到 Standard
abaqus job=standard_job oldjob=explicit_job

# 从 Standard 传递到 Explicit
abaqus job=explicit_job oldjob=standard_job
```

### 输入文件模板

#### Explicit 到 Standard 传递

```
*-heading
Explicit to Standard transfer
**
*RESTART, READ, STEP=1, INC=50
**
*STEP
*STATIC
Initial state from Explicit analysis
*END STEP
```

#### Standard 到 Explicit 传递

```
*-heading
Standard to Explicit transfer
**
*RESTART, READ, STEP=1, INC=50
**
*STEP
*DYNAMIC, EXPLICIT
Initial state from Standard analysis
*END STEP
```
