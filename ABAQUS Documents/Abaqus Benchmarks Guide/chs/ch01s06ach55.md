# 1.6.3 耦合温度-位移分析：一维间隙传导和辐射






**产品：** Abaqus/Standard  Abaqus/Explicit

本示例展示了一维、完全耦合、热传递和应力分析的两个基本非线性案例。这些问题足够简单，可以轻松获得精确解，从而验证使用 Abaqus 获得的数值解。

### 问题描述

模型如图 1.6.3-1](ch01s06ach55.md#sxmtempdisp-specs)所示。单位面积的导电杆一端固定在 A 点，另一端 ![](../graphics/bmk_eqn00655.gif) 自由。在自由端和相邻固定墙 C 之间有一个间隙，热量将通过该间隙传导或辐射。

在案例 1 中，考虑了两种形式的间隙依赖热传导：第一种，间隙的电导率随间隙增大而线性降低；第二种，间隙辐射视角因子随间隙增大而线性降低。杆的固定端 A 和墙 C 都保持在固定温度 ![](../graphics/bmk_eqn00656.gif) 和 ![](../graphics/bmk_eqn00657.gif)。最初间隙是开放的，因此 B 和 C 之间的距离是 ![](../graphics/bmk_eqn00658.gif)（![](../graphics/bmk_eqn00659.gif)）。目标是预测杆自由端的稳态位移 ![](../graphics/bmk_eqn00660.gif) 和温度 ![](../graphics/bmk_eqn00661.gif)。我们假设应变很小，杆的行为是线性弹性的，具有恒定的模量和热膨胀系数。在这种情况下，间隙永远不会关闭，因此杆始终是无应力的。

在案例 2 中，假定封闭间隙的电导率随通过间隙传递的压力 p 的增加而线性增加。杆的固定端 A 和墙 C（位置也是固定的）都保持在固定温度 ![](../graphics/bmk_eqn00656.gif) 和 ![](../graphics/bmk_eqn00657.gif)。由于在这种情况下间隙永远不会打开，杆中的轴向应力将为非零。我们求解间隙上的压力 p 和杆端部的温度 ![](../graphics/bmk_eqn00661.gif)，假设应变很小，杆的行为是具有恒定模量和热膨胀系数的线性弹性。

在 Abaqus/Standard 中，杆使用二维或三维单元建模；杆端部和墙之间的接触以三种方式之一建模：作为间隙单元（GAPUNIT）或由 T2D2T、S4RT、S4T 或 S8RT 单元组成的基于单元的刚性表面。在 Abaqus/Explicit 中，杆使用二维或三维单元建模；墙以两种方式之一建模：要么作为解析刚性表面，要么作为基于单元的刚性表面。杆和墙之间采用基于表面的接触；同时考虑运动和罚机械接触。
