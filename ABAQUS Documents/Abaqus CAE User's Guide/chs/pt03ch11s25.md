# 11.25 使用编辑特征对话框







### 11.25 使用**编辑特征**对话框

首次选择要编辑的特征时，会出现**编辑特征**对话框。

本节描述**编辑特征**对话框中的选项；可用选项取决于所选的特征。您使用**编辑特征**对话框进行以下更改：

****深度****

使用**深度**参数更改盲孔拉伸特征的深度。输入拉伸深度的新值，然后单击**应用**以在视口中查看修改后的特征。您无法更改首次创建特征时定义的拉伸距离方式；例如，从**盲孔**更改为**至面**。有关更多信息，请参见"定义拉伸距离"，第11.13.1节。

****半径****

使用**半径**参数更改圆角/倒圆角特征的半径。输入圆角半径的新值，然后单击**应用**以在视口中查看修改后的特征。有关更多信息，请参见"圆化边缘"，第11.27.1节和"圆化线框中的顶点"，第11.23.3节。

****翻转拉伸方向****

切换**翻转拉伸方向**可更改所选特征的拉伸方向。单击**应用**以在视口中查看修改后的特征。此选项仅适用于添加到基特征的拉伸特征。有关更多信息，请参见"控制拉伸特征的方向"，第11.13.2节。

**保持内部边界**

切换**保持内部边界**可维护在特征和现有零件之间生成的任何面或边缘。内部边界可以创建区域，这些区域可以进行结构化或扫掠网格划分，而无需求助于分区。此选项仅适用于添加到基特征的拉伸、旋转和放样实体及壳特征。

****拔模角****

使用**拔模角**参数更改拉伸特征的拔模。输入拔模角的新值，然后单击**应用**以在视口中查看修改后的特征。零度拔模角将创建直拉伸。如果在创建特征时未定义拔模，则无法向拉伸添加拔模。有关更多信息，请参见"在拉伸中包含拔模"，第11.13.4节。

****节距****

使用**节距**参数更改拉伸特征的扭曲或旋转特征的节距。输入节距的新值，然后单击**应用**以在视口中查看修改后的特征。如果在创建特征时未定义节距，则无法向特征添加节距。

在拉伸特征中，节距定义轮廓将扭曲360度的拉伸距离。有关更多信息，请参见"在拉伸中包含扭曲"，第11.13.3节。同样，在旋转特征中，节距是轮廓在360度旋转期间平移的距离。有关更多信息，请参见"为轴对称零件和旋转特征定义旋转轴"，第11.13.5节。

****角度****

使用**角度**参数更改旋转特征的旋转角度。输入新值，然后单击**应用**以在视口中查看修改后的特征。

****翻转旋转方向****

切换**翻转旋转方向**可反转旋转特征的旋转方向。单击**应用**以在视口中查看修改后的特征。有关更多信息，请参见"控制旋转特征的方向"，第11.13.6节。

****翻转节距方向****

切换**翻转节距方向**可更改带节距旋转特征中的平移方向。单击**应用**以在视口中查看修改后的特征。此选项仅适用于旋转特征，并与**节距**参数配合使用。有关更多信息，请参见"控制旋转特征的方向"，第11.13.6节。

****扫掠草图法向于路径****

切换**扫掠草图法向于路径**可更改用于创建带节距旋转特征的初始草图方向。单击**应用**以在视口中查看修改后的特征。此选项仅适用于带节距的旋转特征。有关更多信息，请参见"控制带节距旋转特征的横截面"，第11.13.7节。

****编辑截面草图****

使用**编辑截面草图**按钮更改所选特征的轮廓。进行更改，关闭草图编辑器，然后单击**应用**以在视口中查看修改后的特征。

****编辑扫掠路径草图****

使用**编辑扫掠路径草图**按钮更改扫掠特征的路径。进行更改，关闭草图编辑器，然后单击**应用**以在视口中查看修改后的特征。有关更多信息，请参见"定义扫掠路径和扫掠轮廓"，第11.13.8节。

****缝合公差****

使用**缝合公差**参数增加或减少在缝合零件中自由边缘之间的间隙时Abaqus/CAE修复的间隙大小。有关更多信息，请参见"缝合边缘以创建面"，第69.6.1节。

****确定时重新生成****

切换**确定时重新生成**可控制特征重新生成。**确定时重新生成**默认处于开启状态。如果您不想在单击**确定**时重新生成特征，请关闭此选项。有关更多信息，请参见"重新生成零件或装配"，第65.4.6节。
