# 19.10.1 导航自适应过程自定义选项







使用自适应过程编辑器在提交之前自定义您的自适应过程。这些设置类似于 Abaqus 执行过程的参数。有关更多信息，请参见["Abaqus/Standard、Abaqus/Explicit 和 Abaqus/CFD 执行，"《Abaqus 分析用户手册》第 3.2.2 节](../usb/usb-link.md#usb-int-danalysisproc)。

要找到自适应过程编辑器，请从主菜单栏中选择****自适应**![](../graphics/images/arrow.gif)**编辑**![](../graphics/images/arrow.gif)***自适应过程名称*****。自适应过程编辑器包含以下选项卡页面：

****自适应****

使用**自适应**选项卡页面指定最大分析迭代次数并选择运行模式。有关更多信息，请参见["配置自适应过程属性，"第 19.10.2 节](pt03ch19s10hlb02.md)。

****常规****

使用**常规**选项卡页面配置预处理器打印输出并指定临时目录。您还可以使用**常规**选项卡页面指定模型引用的任何用户子程序以及结果输出格式。有关更多信息，请参见["指定常规自适应过程设置，"第 19.10.3 节](pt03ch19s10hlb03.md)。

****内存****

使用**内存**选项卡页面配置分配给每个自适应分析的内存量。有关更多信息，请参见["控制自适应过程内存设置，"第 19.10.4 节](pt03ch19s10hlb04.md)。

****并行化****

使用**并行化**选项卡页面配置 Abaqus 分析作业的并行执行，如处理器数量和并行化方法。有关更多信息，请参见["控制自适应过程并行执行，"第 19.10.5 节](pt03ch19s10s01.md)。

****精度****

使用**精度**选项卡页面指定每次分析期间写入输出数据库的节点输出精度。有关更多信息，请参见["控制自适应过程精度，"第 19.10.6 节](pt03ch19s10hlb05.md)。

您在自适应过程编辑器中指定的设置适用于每次自适应重新网格划分迭代后运行的分析。

您也可以使用 Abaqus 环境文件（`abaqus_v6.env`）来控制自适应过程编辑器选项卡页面中许多设置的默认值。有关更多信息，请参见["使用 Abaqus 环境设置，"《Abaqus 分析用户手册》第 3.3.1 节](../usb/usb-link.md#usb-int-denvfile)。
