# 3.2.33 将Abaqus输出数据库文件翻译为Nastran Output2结果文件





**产品：** Abaqus/Standard  

##### **参考**

- ["Abaqus执行程序：概述，" 第3.1.1节](pt01ch03s01abo02.md)

### 概述

翻译器将Abaqus输出数据库`.odb`文件中的某些结果转换为Nastran Output2文件格式。

### 使用翻译器

toOutput2翻译器只能用于翻译[*STATIC*](../key/key-link.md#usb-kws-hstatic)或[*FREQUENCY*](../key/key-link.md#usb-kws-hfrequency)过程的Abaqus输出数据库。使用[*OUTPUT*](../key/key-link.md#usb-kws-houtput)选项将Abaqus分析的结果写入Abaqus输出数据库。应在Abaqus输入文件中包含以下选项以确保要翻译的结果在Abaqus输出数据库中可用：

```
[*OUTPUT*](../key/key-link.md#usb-kws-houtput), FIELD
[*NODE OUTPUT*](../key/key-link.md#usb-kws-hnodeoutput)
U,
RF,
CF,
[*ELEMENT OUTPUT*](../key/key-link.md#usb-kws-helementoutput)
S,
E,
SF,
NFORC,
```

除上述内容外，Abaqus输出数据库中的其他结果将在翻译期间被跳过。仅翻译弹簧元素和三维连续体、壳、膜、梁和桁架元素的结果。

对于壳元素，翻译器将最低编号截面点处的应力和应变视为底面处的应力和应变，将最高编号截面点处的应力和应变视为顶面处的应力和应变。翻译到Output2文件的中面应力和应变计算为底面和顶面应力和应变的平均值。

节点结果始终位于全局坐标系中。元素张量结果位于Abaqus元素坐标系中。

输出数据库中的模型数据（节点坐标、元素拓扑、材料属性和元素属性）在适用记录存在时被写入Output2文件。

|  | **[**[**odb**](pt01ch03s02abx33.md#odb2-usb-int-dodbtonasproc)=*odb-name***]** **[**[**step**](pt01ch03s02abx33.md#step3-usb-int-dodbtonasproc)=*step-number***]** **[**[**increment**](pt01ch03s02abx33.md#increment4-usb-int-dodbtonasproc)=*increment-number***]** **[**[**slim**](pt01ch03s02abx33.md#slim5-usb-int-dodbtonasproc)**]** **[**[**quad4corner**](pt01ch03s02abx33.md#quad4corner6-usb-int-dodbtonasproc)**]** |

### 命令行选项

**job**

此选项指定翻译器创建的Nastran Output2文件的名称。它也是Abaqus输出数据库的默认名称。

**odb**

此选项指定Abaqus输出数据库的名称（如果与*job-name*不同）。

**step**

此选项指定翻译器要翻译的Abaqus输出数据库的步骤号。如果指定步骤包含多个荷载情况，所有荷载情况都会被翻译。默认值为分析的最后一步。

**increment**

此选项仅在与**step**选项结合使用时有效。它用于指定翻译器要翻译的Abaqus输出数据库中步骤的增量号。默认值为指定步骤的最后增量。

**slim**

此选项用于在Output2文件中包含SLIM/VISION软件（可从Third Millennium Productions, Inc.获得）后处理所需的数据块。

**quad4corner**

此选项用于请求在角节点而不是在质心处的壳输出。此选项与应力、应变和截面力输出相关。
