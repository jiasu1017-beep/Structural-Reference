# 47.5.5 自定义X–Y图轴刻度模式





### 47.5.5 自定义X–Y图轴刻度模式

点击**Scale**选项卡来自定义*X–Y*图的*X*轴和*Y*轴的刻度模式。刻度模式选项使您能够控制沿任一轴出现的主刻度线和次刻度线的数量。默认情况下，Abaqus/CAE根据图中包含的数据点的最小值和最大值计算最小轴值和最大轴值。

**要自定义X–Y图的轴范围：**

1. 找到**Tick Mode**选项。选择****Options**![](../graphics/images/arrow.gif)**XY Options**![](../graphics/images/arrow.gif)**Axis****；或点击![](../graphics/ico_xyAxisOptions.png)，它位于Visualization模块工具箱中的*X–Y*绘图工具中。在出现的对话框中点击**Scale**选项卡。**Tick Mode**选项出现在**Scale**页面的底部。
2. 从**X Axis**或**Y Axis**字段中，高亮显示一个或多个轴。
3. 选择计算所选轴上主刻度线之间间隔的方法。从**Major**选项中，选择以下之一：**Automatic**Abaqus计算所选轴上主刻度线之间的间隔，并四舍五入以提供合理的间隔。**By increment****Tick increment**文本字段变得可用，您可以在其中输入所选轴上主刻度线之间的增量。**By count****Number of ticks per decade**字段变得可用，您可以增加或减少沿所选轴的最小值和最大值之间出现的主刻度线的总数。
4. 指定次刻度线的数量。从**Minor**刻度选项中，点击**Ticks per increment**箭头（对于线性或分贝轴）或**Ticks per decade**箭头（对于对数轴）来指定每个主刻度线之间出现的次刻度线数量。
5. 点击**Dismiss**关闭**Axis Options**对话框。
