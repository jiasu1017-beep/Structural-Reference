# 15.13.4 为通用接触指定和修改接触稳定化分配







接触稳定化在两个表面之间引入粘性阻尼，以在建立接触之前稳定刚体运动。接触稳定化定义独立于相互作用编辑器中**属性分配**部分的接触域分配给表面。

除了初始步骤外，接触稳定化定义可以在Abaqus/Standard分析的任何步骤中分配。由于通用接触相互作用必须在初始步骤中创建，因此稳定化分配要求您编辑先前创建的通用接触相互作用。

稳定化效果通常仅在被应用的步骤中强制执行。但是，如果选择了非默认幅值作为接触稳定化定义的一部分，则稳定化效果可能会按照Abaqus Analysis User's Guide的第36.2.5节"Abaqus/Standard中的通用接触稳定化"中所述延续到后续步骤。要在此情况下移除稳定化效果，您必须使用**从先前步骤重置值**选项创建单独的接触稳定化定义，并在后续步骤中应用该稳定化定义。

有关接触稳定化定义的更多信息，请参见第15.12.5节"创建接触稳定化定义"。

**指定接触稳定化分配：**

1. 从主菜单栏中，选择****相互作用**![](../graphics/images/arrow.gif)**编辑**![](../graphics/images/arrow.gif)***相互作用名称****。
2. 在相互作用编辑器的**属性分配**部分中单击**接触属性**选项卡（如果尚未选择）。
3. 如有必要，在相互作用编辑器的**接触属性**部分中单击![](../graphics/icoR_contactStabsSmall.png)以创建接触稳定化定义；有关更多信息，请参见第15.12.5节"创建接触稳定化定义"。
4. 单击**稳定化分配**旁边的![](../graphics/ico_edit.png)。将出现**编辑稳定化分配**对话框。默认情况下，当您从列表或表中选择表面时，Abaqus/CAE会在视口中高亮显示该表面；但是，默认表面标记为**（全局）**和**（自）**时不适用高亮显示。您可以切换关闭**高亮显示选定区域**以关闭选择高亮显示（位于对话框底部）。
5. 从左侧第一列中的现有表面列表中选择一个或多个表面。选择**（全局）**可在整个接触域和单个表面之间引入稳定化效果。**提示：**您可以单击![](../graphics/ico_surface.png)定义新表面并将其添加到列表中。有关定义表面的说明，请参见第73.3.3节"创建表面"。
6. 从第二列中的现有表面列表中选择第二个或多个表面以定义表面配对。- 当在任一列中选择了多个表面时，会生成所有可能的组合。- 要为自接触分配稳定化定义，请在第二列中选择相同的表面名称或**（自）**。- 接触域之外区域的任何稳定化分配都将被忽略。
7. 从第三列中的现有稳定化定义列表中，选择要分配的稳定化。您可以在**编辑稳定化分配**对话框中创建新的接触稳定化定义（请参见第15.12.5节"创建接触稳定化定义"）。
8. 单击对话框中间的箭头![](../graphics/usi-itn-contactattribarrow.png)将您的选择转移到稳定化分配列表中。对话框右侧的表格将更新以反映您的选择。
9. 根据需要重复上述步骤以完成稳定化分配。如果要删除稳定化分配，请选择行并单击![](../graphics/ico_delete.png)。**注意：**分配的顺序可能是相关的；当稳定化分配重叠时，最后的分配优先。
10. 单击**确定**保存您的选择并关闭**编辑稳定化分配**对话框。相互作用编辑器重新出现，其中包含有关单个稳定化分配数量的更新信息。
11. 单击**确定**保存您的更改并关闭编辑器。

**修改接触稳定化分配：**

1. 从主菜单栏中，选择****相互作用**![](../graphics/images/arrow.gif)**编辑**![](../graphics/images/arrow.gif)***相互作用名称****。将出现**编辑相互作用**对话框。
2. 在相互作用编辑器的**属性分配**部分中单击**接触属性**选项卡（如果尚未选择）。
3. 单击**稳定化分配**旁边的![](../graphics/ico_edit.png)。将出现**编辑稳定化分配**对话框。
4. 从对话框右侧表格中选择要修改的稳定化分配所在的行或行。
5. 从对话框左侧第三列的列表中选择新的稳定化定义。
6. 单击对话框中间的箭头![](../graphics/usi-itn-modcontactprop.png)以用您的新选择替换稳定化定义。
7. 单击**确定**保存您的选择并关闭对话框。相互作用编辑器重新出现。
8. 单击**确定**保存您的更改并关闭相互作用编辑器。
