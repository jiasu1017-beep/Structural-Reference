# 20.22.3 将当前草图保存为独立草图


在绘制时,您可以将当前草图保存为独立草图。独立草图独立于任何特征进行维护;它们随后可以被检索到 Sketcher 中,并会覆盖任何现有几何。

Abaqus/CAE 仅保存草图内的项目。[参考几何](pt03ch20s05s01.md)不会被保存。同样,顶点与参考几何线条之间的尺寸也不会被保存。

**将当前草图保存为独立草图的步骤:**

1. 从 Sketcher 工具箱中,选择另存为工具 ![](../graphics/ico_sketchSaveAs.png)。有关 Sketcher 工具箱中工具的图示,请参见["Sketcher 工具," 第 20.4.1 节](pt03ch20s04s01.md)。
2. 在提示区域出现的文本框中,输入草图的名称。Abaqus/CAE 会保存草图并返回 Sketcher。
