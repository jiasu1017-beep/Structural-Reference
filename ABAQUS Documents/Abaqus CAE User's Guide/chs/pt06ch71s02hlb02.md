# 71.2.2 获取模型的一般信息







要获取模型的一般信息，请从主菜单栏选择****Tools**![](../graphics/images/arrow.gif)**Query****或单击**Query**工具栏中的![](../graphics/ico_info.png)工具。从出现的**Query**对话框中的**General Queries**字段中选择以下之一：

**Point/Node**

选择一个点或节点。Abaqus/CAE显示该点的*X*、*Y*和*Z*坐标。

**Distance**

选择两个点或节点。在Part、Property和Mesh模块中，您可以选择点、边或面。Abaqus/CAE显示以下内容：
- 每个实体的*X*、*Y*和*Z*坐标。
- 实体之间的绝对距离。
- 两个实体之间向量的*X*、*Y*和*Z*分量。

非点实体之间的距离始终是两个实体内最近两点之间的距离。

**Angle**

选择两条边或两个面，或选择一条边和一个面。Abaqus/CAE根据您的选择显示以下内容之一：
- 两条边之间的角度。
- 两个面法线之间的角度。
- 边与面法线之间的角度。

当您选择一条边时，Abaqus/CAE沿该边显示一个箭头。当您选择一个面时，Abaqus/CAE在该面上显示一个指示面法线方向的箭头。角度定义为将一个边或面扫过以使两个箭头对齐所需的角度。角度始终为正且小于或等于180。

**Feature**

在模型树中的特征上单击鼠标按钮3，并从出现的菜单中选择**Query**。Abaqus/CAE显示有关所选特征以下信息：
- 名称和描述；例如，实体拉伸。
- 状态（如果特征被抑制或无法重新生成）。
- 其父特征的名称（如果有）。
- 其子特征的名称（如果有）。
- 定义特征的参数值。

**Shell element normals**

- 对于具有壳区域的部件，Abaqus/CAE使用阴影渲染样式显示部件或装配体。面法线与壳法线重合的壳侧（顶面）被染成棕色；相反侧（底面）被染成紫色。
- 对于具有线区域的轴对称部件，Abaqus/CAE显示青色箭头以指示法线方向。

您可以使用Property模块中的**Assign**菜单或Mesh模块中的****Mesh**![](../graphics/images/arrow.gif)**Orientation**![](../graphics/images/arrow.gif)**Normal****菜单来反转法线方向。有关更多信息，请参见["分配壳/膜法线方向，"第12.15.5节](pt03ch12s15hlb04.md)。

**Beam element tangents**

Abaqus/CAE显示青色箭头以指示梁切线方向。

您可以使用Property模块中的**Assign**菜单或Mesh模块中的****Mesh**![](../graphics/images/arrow.gif)**Orientation**![](../graphics/images/arrow.gif)**Normal****菜单来反转切线方向。有关更多信息，请参见["分配梁/桁架切线方向，"第12.15.6节](pt03ch12s15hlb05.md)。

**Mesh stack orientation**

您只能使用此工具查询六面体、楔形和四边形单元，因为这些是唯一可以堆叠形成连续壳、内聚、圆柱形或垫片网格的单元。对于六面体和楔形单元，Abaqus/CAE将顶面染成棕色，底面染成紫色。类似地，箭头指示四边形单元的方向。此外，Abaqus/CAE会高亮显示方向不一致的任何单元面和边。有关更多信息，请参见["使用几何和网格工具创建具有内聚元素的模型，"第21.3节](pt04ch21s03.md)；["连续壳，"第25章](pt04ch25.md)";["圆柱形实体的扫掠网格划分，"第17.9.4节](pt03ch17s09s04.md)；和["垫片，"第32章](pt04ch32.md)。

如果区域是原生Abaqus网格，您可以通过更改扫掠路径的方向来更改网格堆栈方向。对于实体区域，您可以独立于扫掠路径分配新的网格堆栈方向。如果区域是孤立网格，您可以使用"编辑网格"工具集更改网格堆栈方向。有关更多信息，请分别参见["指定扫掠路径，"第17.18.6节](pt03ch17s18s06.md)；["应用网格堆栈方向，"第17.18.8节](pt03ch17s18hlb01.md)；和["定向堆栈方向，"第64.6.4节](pt06ch64s06s04.md)。

**注意：**查询结果不考虑在Property模块中定义固体复合铺层或复合壳铺层时对网格堆栈方向所做的更改。

****Mesh****

对于装配体、部件或部件实例、几何区域或单元，Abaqus/CAE显示以下内容：
- 所选区域中的节点和单元总数
- 每种单元形状的单元数量

默认情况下，Abaqus/CAE在消息区域显示网格信息，但您可以通过在提示区域切换**Display detailed report**在**Mesh statistics**对话框中以表格格式显示此信息。**Mesh statistics**对话框还允许您按部件实例或按单元类型显示网格信息。

**Element**

- 单元标签
- 单元拓扑；例如，线性六面体
- Abaqus单元名称；例如，C3D8I
- 节点连通性

**Mesh gaps/intersections**

选择网格部件或部件实例，并输入节点与单元面之间的最大距离。Abaqus/CAE高亮显示与边界面相交的单元边以及距边界面小于指定距离的单元边。

**Mass properties**

此查询显示有关所选内容的表面积和面积质心、体积和体积质心、质量和质心以及关于质心或关于指定点的惯性矩的信息。有关更多信息，请参见["查询质量属性，"第71.2.3节](pt06ch71s02hlb03.md)。

**Geometry diagnostics**

- 无效、不精确或小几何体
- 拓扑
