# 4.1 布局管理器概述







布局管理器将其子组件以特定方式排列在其内部。布局管理器使用布局提示和排列样式的组合来决定如何放置和调整其子组件的大小。Abaqus GUI Toolkit 中的布局管理器计算相对大小和相对位置，而不是绝对坐标。这种相对方法会自动适应变化，例如不同的字体大小和窗口调整大小。

Abaqus GUI Toolkit 中提供以下布局管理器：

**`FXHorizontalFrame`**

水平排列组件。如需更多信息，请参阅"水平和垂直框架"，第 4.3 节。

**`FXVerticalFrame`**

垂直排列组件。如需更多信息，请参阅"水平和垂直框架"，第 4.3 节。

**`AFXVerticalAligner`**

垂直对齐其子组件中的第一个子组件。如需更多信息，请参阅"复合子组件的垂直对齐"，第 4.4 节。

**`FXPacker`**

以通用方式排列组件。如需更多信息，请参阅"通用布局管理器"，第 4.5 节。

**`AFXDialog`**

与 `FXPacker` 具有相同的功能。如需更多信息，请参阅"通用布局管理器"，第 4.5 节。

**`FXGroupBox`**

提供与 `FXPacker` 相同的功能，但允许带标题边框。如需更多信息，请参阅"通用布局管理器"，第 4.5 节。

**`FXMatrix`**

以行和列的方式排列组件。如需更多信息，请参阅"行和列布局管理器"，第 4.6 节。

**`FXSplitter`**

垂直或水平分割区域，并允许调整区域大小。如需更多信息，请参阅"可调整大小的区域"，第 4.7 节。

**`FXSwitcher`**

将子组件堆叠在一起（轮换区域）。如需更多信息，请参阅"轮换区域"，第 4.8 节。

**`FXTabBook`**

一次显示一个选项卡或一页组件。用户通过单击选项卡按钮来选择要查看的选项卡。如需更多信息，请参阅"选项卡簿"，第 4.9 节。
