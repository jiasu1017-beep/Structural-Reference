# 5.2.7 循环工具





当你选择循环工具 ![](../graphics/ico_viewCycle.png) 和要工作的视口时，Abaqus/CAE 进入循环模式，如双向箭头形式的光标所示。你可以在每个视口中循环浏览最近的八个视图。

要循环浏览之前的视图，请在你要更改视图的视口中点击。要控制循环方向，请在提示区域中单击**向后**或**向前**。默认是向后循环。向后循环到最早的可用视图后，继续点击无效。同样，向前循环到最新的视图后，继续点击无效。
