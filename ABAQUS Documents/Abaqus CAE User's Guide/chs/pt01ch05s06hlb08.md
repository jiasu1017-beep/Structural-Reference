# 5.6.8 应用自定义视图





使用**视图**工具栏将视图定向、缩放和定位到七个预定义或四个用户定义设置之一。要显示**视图**工具栏，请从主菜单中选择****View**![](../graphics/images/arrow.gif)**Toolbars**![](../graphics/images/arrow.gif)**Views****；**视图**工具栏如下图所示：

![](../graphics/viw-viewtoolbox-nls.png)

以下自定义视图可用：
- **前**、**后**、**顶**、**底**、**左**和**右**：相当于从立方体六个侧面观察模型。
- **等轴测**：等轴测视图。这是三维模型的默认方向。
- **用户1**、**用户2**、**用户3**和**用户4**：四个用户定义视图。请参阅["保存用户定义的视图，" 第 5.6.9 节](pt01ch05s06hlb09.md) 获取有关如何保存用户定义视图的描述。

如果当前视口链接到其他视口，Abaqus/CAE 也会将会话中所有链接视口中的相同自定义视图应用。有关更多信息，请参阅["链接视口以进行视图操作，" 第 4.5 节](pt01ch04s05.md)。

**要应用自定义视图：**

1. 如果尚不可见，请通过从主菜单中选择****View**![](../graphics/images/arrow.gif)**Toolbars**![](../graphics/images/arrow.gif)**Views****来显示**视图**工具栏。Abaqus/CAE 显示**视图**工具栏。
2. 从**视图**工具栏中，单击所需的工具。如果你只有一个视口，Abaqus/CAE 会立即应用所选视图并从**视图**工具栏中取消选择它。如果你有多个视口，请将光标放在要更改视图的视口上。光标变为 triad；点击，Abaqus/CAE 会将该视图应用到该视口。**注意：**当你应用使用**自动拟合**选项保存的视图时，视图采用保存视图的方向，并立即重新缩放以填充视口。当你应用使用**保存当前**选项保存的视图时，视图采用保存视图的方向、缩放因子和位置。
3. 根据需要重复步骤 2 多次，以达到所需的视图。**提示：**使用[循环](pt01ch05s06hlb07.md)视图操作工具返回上一个视图。
