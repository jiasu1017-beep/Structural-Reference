# 81.6.4 向插件工具箱添加内核插件的示例





您使用 `registerKernelToolButton()` 命令使内核插件从插件工具箱可用。当您向工具箱添加插件时，****Plug-ins**![](../graphics/images/arrow.gif)**Toolboxes**** 菜单中会出现一个项。当您选择此项目时，Abaqus/CAE 会显示一个插件工具箱。虽然您可以在工具箱中显示文本，但您可能希望提供一个图标而不是使用文本（有关创建图标的更多信息，请参阅 [Abaqus GUI Toolkit User's Guide 附录A "Icons"](../cus/cus-link.md#cus-apx-icons)）。

`registerKernelToolButton` 注册命令使用与 `registerKernelMenuButton` 命令相同的参数；但是，它还需要一个工具箱名称。您可以使用 *buttonText* 参数仅指定工具提示，方法是在提示文本前加 "\t"，如以下版本的 `myUtils_plugin.py` 所示：

```
from abaqusGui import getAFXApp, FXXPMIcon
from myIcons import vpIconData
vpIcon = FXXPMIcon(getAFXApp(), vpIconData)

toolset = getAFXApp().getAFXMainWindow().getPluginToolset()
toolset.registerKernelToolButton(toolboxName='My Utils',
    buttonText='\tPrint Current Viewport', icon=vpIcon,
    moduleName='myUtils', functionName='printCurrentVp()' )
```

此示例在 ****Plug-ins**![](../graphics/images/arrow.gif)**Toolboxes**** 菜单下创建一个 **My Utils** 项。当用户点击 **My Utils** 时，Abaqus/CAE 会显示一个工具箱，其标题栏中显示 **My Utils**。当您点击工具箱中的图标时，Abaqus/CAE 会向内核发送以下命令：

```
myUtils.printCurrentVp()
```
如果我们在 **Plug-ins** 菜单中注册了插件，则会发送相同的命令到内核。

如果 **Toolboxes** 项存在，它总是首先出现在 **Plug-ins** 菜单中；**About Plug-ins** 项总是最后出现。**Plug-ins** 菜单中的其他项目按字母顺序列出。
