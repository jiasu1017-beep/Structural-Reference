# *PRESTRESS HOLD







### *PRESTRESS HOLD在初始平衡求解期间保持钢筋预应力恒定。

此选项在[*STATIC](ch18abk31.md)步骤（["静态应力分析，" Abaqus Analysis User's Guide第6.2.2节](../usb/usb-link.md#usb-anl-astatic)）中使用，以在初始平衡求解期间保持部分或全部钢筋中的应力恒定。

**产品：**Abaqus/Standard  

**类型：**历史数据  

**级别：**步骤  

##### **参考：**

- ["定义钢筋，" Abaqus Analysis User's Guide第2.2.3节](../usb/usb-link.md#usb-int-erebarlayer)

**此选项没有关联的参数。**

### **保持预应力恒定的数据行：**

**第一行：**

根据需要重复此数据行。每行给出四对数据。
