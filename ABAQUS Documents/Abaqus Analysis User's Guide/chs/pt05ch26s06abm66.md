### 概述

吸湿性：
- 定义多孔材料在部分饱和流动条件下的吸收/解吸行为；以及
- 用于耦合润湿液体流动和多孔介质应力的分析（["耦合孔隙流体扩散和应力分析，" 第6.8.1节](pt03ch06s08at26.md)）。

### 吸湿性

当总孔隙液体压力 ![](../graphics/usb_eqn01163.gif) 变为负值时，多孔介质变为部分饱和（参见["多孔介质的有效应力原理，" Abaqus理论指南第2.8.1节](../stm/stm-link.md#stm-anl-poreffstress)）。![](../graphics/usb_eqn01163.gif) 的负值表示介质中的毛细效应。已知对于 ![](../graphics/usb_eqn01188.gif)，饱和度位于一定范围内，该范围取决于毛细压力 ![](../graphics/usb_eqn04379.gif) 的值（参见["多孔介质中润湿液体相的连续性声明，" Abaqus理论指南第2.8.4节](../stm/stm-link.md#stm-anl-porcontstate)）。这些限值的典型形式如图26.6.4-1所示。我们将这些限值写为 ![](../graphics/usb_eqn04380.gif)，其中 ![](../graphics/usb_eqn04381.gif) 是发生吸收的限值（因此 ![](../graphics/usb_eqn04382.gif)），![](../graphics/usb_eqn04383.gif) 是发生解吸的限值（因此 ![](../graphics/usb_eqn04384.gif)）。吸收和解吸之间的转换以及相反过程沿"扫描"曲线进行讨论（见下文）。这些曲线由图26.6.4-1中所示的单条直线近似。

**图26.6.4-1** 典型的吸收和解吸行为。

![](../graphics/csorption-absorp-exsorp-nls.png)

当在多孔介质流动分析中包含部分饱和时，应分别定义吸收行为、解吸行为和扫描行为（在吸收和解吸之间）。以下将讨论每种行为。如果完全未定义吸湿性，Abaqus/Standard假定对于 ![](../graphics/usb_eqn01163.gif) 的所有值都处于完全饱和流动状态（![](../graphics/usb_eqn01203.gif)）。

吸湿性的定义会产生强非对称的部分饱和流动耦合方程。因此，如果您请求部分饱和分析（即定义了吸湿性），Abaqus/Standard会自动使用其非对称矩阵存储和求解方案（参见["定义分析，" 第6.1.2节](pt03ch06s01abo05.md)）。

### 定义吸收和解吸

通过将孔隙液体压力 ![](../graphics/usb_eqn01163.gif)（负"毛细张力"）指定为饱和度的函数来定义吸收和解吸行为。在大多数物理情况下，润湿液体不能被驱动到零饱和度；要达到零饱和度，数据必须将 ![](../graphics/usb_eqn04385.gif) 定义为 ![](../graphics/usb_eqn04386.gif)。吸收和解吸数据可以以表格形式或分析形式定义。

### 定义吸收和解吸之间的行为

吸收和解吸之间的行为由用户指定的恒定斜率 ![](../graphics/usb_eqn04391.gif) 的扫描线定义。此斜率应大于吸收或解吸行为任何段的斜率。

如果定义了吸收和解吸行为而没有扫描线，则扫描线的斜率取为吸收和解吸行为定义中给出的最大 ![](../graphics/usb_eqn04392.gif) 值的1.05倍。

| **输入文件用法：** | ``` [*SORPTION](../key/key-link.md#usb-kws-msorption), TYPE=SCANNING ``` |
| --- | --- |
|  | 这必须是同一材料的 [*SORPTION](../key/key-link.md#usb-kws-msorption) 选项的重复使用。 |
