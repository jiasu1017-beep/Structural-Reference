# 17.18.3 选择网格划分技术





Abaqus/CAE根据区域的几何形状和该区域当前选择的单元形状，为模型的每个可网格划分区域分配默认自顶向下网格划分技术。 Abaqus/CAE使用分配给区域的网格技术为该区域生成网格。 您可以使用**Mesh Controls**对话框选择替代网格划分技术。

**选择网格划分技术：**

1. 从主菜单栏，选择 ****Mesh**![](../graphics/images/arrow.gif)**Controls****。 Abaqus/CAE在提示区域显示提示以指导您完成该过程。 **提示：**您可以使用Mesh模块工具箱中的 ![](../graphics/ico_meshControls.png) 工具设置网格划分技术。
2. 选择**faces of solid regions**以为将进行四面体网格划分或自底向上网格划分的实体区域的边界面分配网格控制。
3. 如果您的零件或装配包含多个区域，请选择要查看或修改其网格划分技术的区域，然后按鼠标按钮2。 选定区域必须全部具有相同的维度。 **Mesh Controls**对话框出现。
4. 从**Technique**选项列表中，选择您想要的网格划分技术。 （某些技术仅在它们对选定区域有效时才可用。） - 如果在上一步中您选择了已分配不同网格划分技术的多个区域，Abaqus/CAE会选择**As is**。 - 选择[**Free**](pt03ch17s10s01.md)创建自由网格。 - 选择[**Structured**](pt03ch17s08s01.md)创建结构化网格。 - 选择[**Sweep**](pt03ch17s09s01.md)创建扫掠网格。 - 选择[**Bottom-up**](pt03ch17s11s01.md)创建自底向上网格。 - 如果单元形状分配的变化导致自动分配多种技术，Abaqus/CAE会选择**Multiple**。 例如，假设自由网格划分技术应用于实体零件实例的所有区域。 如果您将这些区域的单元形状分配从**Tet**更改为**Hex**或**Hex-dominated**，Abaqus/CAE会自动将分配给每个区域的网格划分技术从自由网格划分技术更改为适合每个区域的技术； 例如，某些区域使用结构化网格划分，其他区域使用扫掠网格划分。 **注意：**必须手动为区域分配或从区域移除自底向上网格划分技术。 当您修改分配了自底向上网格划分技术的区域的几何形状时，生成的区域也将被分配自底向上网格划分技术。 您可以单击**Mesh Controls**对话框中的**Defaults**为自底向上区域分配自顶向下技术。 有关每种网格划分技术的详细信息，请参阅["理解网格生成，" 第17.7节](pt03ch17s07.md)。 在为区域面分配网格控制时，Abaqus/CAE会根据将要在面上使用的网格技术为面添加颜色代码。 面颜色可能与区域颜色不同。 例如，自底向上区域的面默认显示为粉红色，因为它们是自由网格划分的。 如果为某些面分配了结构化网格技术，它们将显示为绿色。 自底向上区域的实体区域颜色为浅棕色，将进行四面体网格划分的实体区域颜色为粉红色。
5. 单击**OK**关闭对话框并保存您的网格划分技术选择。 下次在选定的零件实例或区域上生成网格时，您的选择将被遵守。 如果选定区域已经包含网格，系统会提示您删除网格或取消网格控制过程。
