# 48.2.3 创建或编辑边缘列表路径





路径是通过在模型中指定一系列线段来定义的线。在边缘列表路径中，线段由单元边缘和连接单元边缘端点的线定义。有关在路径中包含边缘以外的位置的信息，请参阅["创建穿过模型的路径，"第48.2节](pt05ch48s02.md)。

您可以通过将部件实例名称和边缘标签输入表格来创建边缘列表路径。每个边缘标签由单元编号、面编号、边缘编号和沿边缘的方向组成。使用三种方法的组合从视口中选择边缘以输入表格：
- 选择单个单元边缘。
- 选择一个单元边缘和沿特征边缘的方向，让Abaqus/CAE自动添加特征边缘末端之间的剩余边缘。
- 选择一个单元边缘和一个节点，让Abaqus/CAE选择它们之间沿单元边缘的最短距离路径。

您也可以直接编辑表格，使用视口跟踪您的进度。但是，边缘标签的某些组成部分取决于单元类型、方向和所需的连接性，这些通过从视口中进行选择更容易确定。

**创建或编辑边缘列表路径：**

1. 找到路径创建和编辑选项。**要创建新路径：**1. 从主菜单栏中，选择****工具**![](../graphics/images/arrow.gif)**路径**![](../graphics/images/arrow.gif)**创建****。"创建路径"对话框出现。2. Abaqus在"名称"文本字段中显示路径的默认名称。要提供更有意义的名称，请将此默认值替换为您选择的名称（包括空格如果您希望的话）。3. 在"创建路径"对话框中，单击"边缘列表"；然后单击"继续"。**要编辑现有路径：**- 从主菜单栏中，选择****工具**![](../graphics/images/arrow.gif)**路径**![](../graphics/images/arrow.gif)**编辑****。从出现的菜单中，选择要编辑的路径。**提示：**您也可以使用"路径管理器"来编辑路径。从主菜单栏中，选择****工具**![](../graphics/images/arrow.gif)**路径**![](../graphics/images/arrow.gif)**管理器****以显示管理器。选择要编辑的路径，然后单击管理器右侧的"编辑"按钮。"编辑边缘列表路径"对话框出现。
2. 要通过在视口中选择边缘来创建或编辑路径，请单击"添加之前"或"添加之后"开始进行选择。Abaqus/CAE提示您**选择要插入路径的边缘**，并关闭"编辑边缘列表路径"对话框。
3. 使用提示区域中的选择方法选择边缘。三种边缘选择方法可用：****单独****按它们在路径中出现的顺序选择单个单元边缘；所选边缘不必相邻。如有必要，单击提示区域中的"翻转"以反转您选择的最后一个边缘的连接性。****按特征边缘****选择沿特征边缘的起始边缘；Abaqus/CAE将自动选择您的选择与特征边缘末端之间的所有边缘。单击"翻转"以反转您选择的边缘和用于自动选择的方向。****按最短距离****选择起始边缘和端点（节点）；Abaqus/CAE将自动选择您的选择之间沿单元边缘定义最短路径的单元边缘。单击"翻转"以反转您选择的边缘和用于自动边缘选择的方向。如果存在具有相同距离的多条路线，Abaqus/CAE选择其中一条。**注意：**您选择的边缘和节点必须在同一部件实例上。如果所需边缘存在于多个部件实例中，请在"选择"工具栏中关闭"选择最接近屏幕的实体"工具。如果进行模糊选择，部件实例名称和边缘标签（不带方向值）将出现在视口左下角，格式为`高亮单元边缘 *单元*.*面*.*边缘*`。使用"下一个"和"上一个"按钮循环浏览可能的选择，然后单击"确定"确认您的选择。Abaqus在您选择时高亮显示边缘，并显示节点标签和路径连接。此外，Abaqus将路径中的第一个节点标记为"起点"，最后一个节点标记为"终点"。
4. 完成后，在提示区域中单击"完成"。Abaqus/CAE显示"编辑边缘列表路径"对话框；路径定义表中列出了您的路径中的部件实例和边缘。
5. 如果需要，重复前述步骤以添加其他边缘。"添加之前"和"添加之后"按钮决定了您的选择将添加在路径定义表中当前高亮行的什么位置。
6. 如果需要，您可以使用标准键盘和鼠标编辑技术在"路径定义"表中插入、修改或删除部件实例名称和边缘标签。边缘标签格式为（`*单元编号*, *面编号*, *边缘编号*, *方向*`）。**提示：**使用查询工具集查询视口中的节点以获取部件实例名称。使用绘图状态独立选项在视口中显示单元和面标签。（有关详细信息，请参阅["自定义模型标签，"第55.5节](pt05ch55s05.md)。）边缘编号从元素外部观看时按顺时针方向围绕面排列。方向（1或-1）取决于您所需的路径连接性。有关特殊表编辑选项或从ASCII文件读取数据，请按鼠标按钮3。（有关详细信息，请参阅["输入表格数据，"第3.2.7节](pt01ch03s02s07.md)。）
7. 编辑"路径定义"表后，按**[Enter]**。Abaqus更新视口中显示的路径。
8. 完成选择边缘后，单击"确定"。如果创建了新路径，Abaqus/CAE会将路径名称添加到"路径管理器"列表中。默认情况下，Abaqus/CAE保存您在会话期间指定的路径。如果要保留路径以供后续会话使用，请将其保存到XML文件、模型数据库或输出数据库；有关详细信息，请参阅["管理会话对象和会话选项，"第9.9节](pt02ch09s08.md)。
