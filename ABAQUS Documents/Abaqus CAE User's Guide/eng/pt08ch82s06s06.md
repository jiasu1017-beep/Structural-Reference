# 82.6.6 Ranking criterion for modal contribution factors







If you select **Rank** as the variable to plot, the plots generated by the plug-in allow you to identify and rank significant modes based on the change in total response magnitude that would result from a fixed percentage increase in the real and imaginary components of the MCF (or partial response).  The plug-in implements the ranking criterion as follows:
- ![](../graphics/usi_eqn00007.gif) = mode number
- ![](../graphics/usi_eqn00806.gif) = partial response for mode ![](../graphics/usi_eqn00007.gif)
- ![](../graphics/usi_eqn00807.gif) = real portion of partial response for mode ![](../graphics/usi_eqn00007.gif)
- ![](../graphics/usi_eqn00808.gif) = imaginary portion of partial response for mode ![](../graphics/usi_eqn00007.gif)
- ![](../graphics/usi_eqn00809.gif) = magnitude of partial response for mode ![](../graphics/usi_eqn00007.gif)
- ![](../graphics/usi_eqn00810.gif) = phase of partial response for mode ![](../graphics/usi_eqn00007.gif)
- *T* = total response *P*
- ![](../graphics/usi_eqn00811.gif) = real portion of total response
- ![](../graphics/usi_eqn00812.gif) = imaginary portion of total response
- ![](../graphics/usi_eqn00813.gif) magnitude of total response
- ![](../graphics/usi_eqn00814.gif) phase of total response
- ![](../graphics/usi_eqn00815.gif) = delta in total response magnitude for 10% increase in any given partial response (i.e. sensitivity)

You can determine the top *n* modes based on the absolute value of ![](../graphics/usi_eqn00816.gif) and output ![](../graphics/usi_eqn00817.gif)  and ![](../graphics/usi_eqn00818.gif) for these modes.
