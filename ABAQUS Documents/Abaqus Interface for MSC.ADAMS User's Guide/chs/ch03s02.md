# 3.2 执行 adams 命令以创建不包含应力或应变的模态中性文件





### 3.2 执行 **adams** 命令以创建不包含应力或应变的模态中性文件

**adams** 命令用于读取多步骤 Abaqus 分析生成的结果文件，并生成 MSC.ADAMS 模态中性（`.mnf`）文件。结果文件格式有几个具体要求。创建满足这些要求的输入文件在 ["设置 Abaqus 模型以创建不包含应力或应变的模态中性文件" 第 2.2 节](ch02s02.md) 中描述。

| **abaqus adams** | **job**=*job-name* |
| --- | --- |
|  | **[****input**=*input-file***]** **[****units**=`mmks` | `mks` | `cgs` | `ips`**]** **[****length**=*length-units-name***]** **[****mass**=*mass-units-name***]** **[****time**=*time-units-name***]** **[****force**=*force-units-name***]** |

****job****

此选项指定在结果文件转换期间使用的输入和输出文件名。*job-name* 值用于构造默认输入文件名 `*job-name*.fil`。输出模态中性文件被命名为 `*job-name*.mnf`。

如果此选项从命令行中省略，系统将提示您输入此值。

****input****

此选项指定结果文件的名称（如果与 `*job-name*.fil` 不同）。

****units****

此选项指定模型的单位系统。可能的值为 `mmks`、`mks`、`cgs` 或 `ips`，对应于同名的 ADAMS/View 选项。默认值为 `mks`。

****length****

此选项指定模型的长度单位。如果指定此选项，它将覆盖指定单位系统的长度单位。

****mass****

此选项指定模型的质量单位。如果指定此选项，它将覆盖指定单位系统的质量单位。

****time****

此选项指定模型的时间单位。如果指定此选项，它将覆盖指定单位系统的时间单位。

****force****

此选项指定模型的力单位。如果指定此选项，它将覆盖指定单位系统的力单位。

可以在 Abaqus 环境文件（`abaqus_v6.env`）中定义单位选项的默认值。可以通过 **adams_units_family** 参数定义 **units** 选项的默认值。可以通过 **adams_length_units**、**adams_mass_units**、**adams_time_units** 和 **adams_force_units** 参数分别定义 **length**、**mass**、**time** 和 **force** 选项的默认值。
