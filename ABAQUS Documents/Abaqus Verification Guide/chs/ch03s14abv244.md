# 3.14.4 使用通用梁截面传递结果





**产品：**Abaqus/Standard  Abaqus/Explicit

### 测试的单元

B21    B31

### 问题描述

以下测试验证了使用通用梁截面定义来指定梁单元截面属性的用法。B21和B31单元承受组合递增载荷。分析包括从Abaqus/Standard到Abaqus/Explicit再回到Abaqus/Standard的顺序传递。非线性非弹性截面行为通过设置非线性广义截面属性来定义，并从定义通用梁截面的轴向、弯曲和扭转行为的选项中省略线性和弹性参数。

非线性非弹性轴向、弯曲和扭转行为由以下数据行定义：

| 0., 0. |
| --- |
| 2.E7, 0.001 |
| 2.5E7, 0.002 |
| 3.0E7, 0.003 |

### 结果与讨论

该分析的结果表明，使用通用梁截面定义指定的截面属性在Abaqus/Explicit和Abaqus/Standard之间正确传递。
