# 19.5.1 什么是优化过程？





[图 19-5](pt03ch19s05s01.md#ana-optprocess-flow-nls) 说明了优化过程如何迭代更新设计变量、修改有限元模型，并在搜索优化解时运行 Abaqus 分析。

**图 19-5** 优化过程迭代搜索优化解。

![](../graphics/ana-optprocess-flow-nls.png)

优化过程读取您在优化模块中定义的优化任务，并基于您在优化任务中定义的目标函数和约束迭代搜索优化解。每个迭代称为一个设计周期。在每个设计周期中，拓扑优化、形状优化或尺寸优化修改有限元模型，并对修改后的模型执行 Abaqus 分析。优化过程读取分析结果，并决定是结束优化（因为解是最优的或已达到指定的停止条件）还是继续优化并在另一个设计周期上进行迭代。

优化过程生成优化结果和分析结果。您必须将优化结果和分析结果组合成单个输出数据库文件，以在可视化模块中查看优化结果，如[「后处理优化，」第 19.5.3 节](pt03ch19s05s03.md)中所述。优化模块不支持在 Abaqus 输入文件中使用零件和装配。当您运行优化任务时，优化模块会生成一个扁平的输入文件，不使用零件和装配，无论您的 Abaqus 模型属性如何。

优化过程出现在模型树的**分析**部分，并包含在优化期间运行的 Abaqus 作业，如图 19-6 所示（[图 19-6](pt03ch19s05s01.md#ana-optprocess-modeltree)）。

**图 19-6** 模型树中的优化过程。

![](../graphics/ana-optprocess-modeltree-nls.png)

您可以检查优化过程的有效性；但是，验证不会检查您的 Abaqus 模型。您应该运行模型的完整分析并确保它运行完成，然后再尝试运行优化过程。
