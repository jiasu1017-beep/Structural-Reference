# 16.11.15 定义流体空腔压力场







您可以为流体填充空腔定义初始流体空腔压力（请参见《Abaqus Analysis User's Guide》中的"Surface-based fluid cavities: overview"第 11.5.1 节）。您只能在初始步骤中创建或修改流体空腔压力场。有关更多信息，请参见《Abaqus Analysis User's Guide》中的"Initial conditions in Abaqus/Standard and Abaqus/Explicit"第 34.2.1 节。要在初始步骤以外的步骤中修改流体空腔压力，您必须定义流体空腔压力边界条件（有关更多信息，请参见"定义流体空腔压力边界条件"第 16.10.15 节）。

**创建或编辑流体空腔压力场：**

1. 使用以下方法之一显示流体空腔压力场编辑器： - 要创建新的流体空腔压力场，请按照"创建预定义场"第 16.8.3 节中概述的程序进行操作（**类别**：`Other`；**所选步骤的类型**：`Fluid cavity pressure`）。您只能在初始步骤中创建流体空腔压力场。 - 要使用菜单或管理器编辑现有的流体空腔压力场，请参见"编辑依赖于步骤的对象"第 3.4.13 节。您只能在初始步骤中修改流体空腔压力场。

2. 点击 **Fluid cavity interaction** 字段右侧的箭头，然后选择流体空腔压力所适用的相互作用。

3. 输入 **Fluid cavity pressure**。

4. 点击 **OK** 创建流体空腔压力场并关闭对话框。绿色菱形出现在流体空腔参考点处，表示您刚刚创建的初始流体空腔压力场。
