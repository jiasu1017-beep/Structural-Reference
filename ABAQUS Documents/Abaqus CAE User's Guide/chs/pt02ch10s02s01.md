# 10.2.1 什么是有效和精确零件？





当您导入实体零件时，Abaqus/CAE会尝试创建封闭的实体零件。同样，当您导入壳零件时，Abaqus/CAE会尝试创建连通的壳零件。如果零件成功导入，则该零件被视为有效和精确的。但是，如果原始零件的精度低于Abaqus/CAE使用的精度，则零件可能不精确或无效。在大多数情况下，您可以继续使用不精确的零件。如果修复或选择忽略无效状态，您也可以使用无效零件。

"不精确"和"无效"术语在下面有更详细的描述。

**不精确**

有效零件可以是精确的或不精确的。如果Abaqus/CAE必须在某些区域使用更宽松的容差来从导入的零件重新创建封闭体积，则该零件被视为不精确。您可以使用不精确的零件完成大多数建模操作。

您应该尝试使用不精确的零件。如果Abaqus/CAE无法继续，您可以抑制不精确区域或使用几何编辑工具来尝试使零件精确。但是，如果零件包含许多复杂曲面，几何编辑工具可能无法使零件精确，使用这些工具可能非常耗时。如果您无法使用不精确的零件且无法使零件精确，则应返回生成原始文件的CAD应用程序并提高精度。

**无效**

如果错误太大以至于Abaqus/CAE无法从导入的零件重新创建封闭体积，则该零件被视为无效。例如，边缘之间的大间隙会导致零件无效。类似地，边缘上远离底层曲面的点会导致零件无效。

如果零件无效，您可以使用几何编辑工具集来尝试使其有效。如果您无法修复零件，可以指示要忽略无效零件状态并继续使用该零件，就像它是有效的一样（有关更多信息，请参阅["处理无效零件，"第10.2.3节"](pt02ch10s02s03.md))。但是，无效几何体上的操作可能会失败、产生不一致的结果或导致Abaqus/CAE中的不稳定。如果在忽略零件无效性后遇到模型问题，请考虑尝试在生成原始文件的CAD应用程序中修复几何体。

如果您不修复或忽略无效零件的状态，则在Abaqus/CAE中使用它的唯一方法是在"相互作用（Interaction）"模块中向零件施加显示体或刚体约束。显示体仅出于显示目的而包含在模型中。如果您施加显示体约束，则不必对实例进行网格划分，并且可以继续分析模型。更多信息，请参阅[第27章，"显示体"](pt04ch27.md)。
