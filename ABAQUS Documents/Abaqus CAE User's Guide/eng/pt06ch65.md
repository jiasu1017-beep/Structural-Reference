# 65 The Feature Manipulation toolset







You build a model in Abaqus/CAE by creating a series of features. For detailed discussions of features and feature-based modeling, see ["The relationship between parts and features," Section 11.3.1](pt03ch11s03s01.md), and ["Manipulating features in the Assembly module," Section 13.8.2](pt03ch13s08s02.md). This chapter explains how to use the Feature Manipulation toolset to modify and manage the existing features in your model and how to tune the performance of feature regeneration. The following topics are covered:
- ["Using the Feature Manipulation toolset," Section 65.1](pt06ch65s01.md)
- ["Using the Model Tree to manage features," Section 65.2](pt06ch65s02.md)
- ["Tuning feature regeneration," Section 65.3](pt06ch65s03.md)
- ["Modifying and manipulating features," Section 65.4](pt06ch65s04.md)
