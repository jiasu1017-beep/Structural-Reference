# 51.2.4 探测X-Y曲线图





### 51.2.4 探测X-Y曲线图

您可以使用"查询"工具集获取有关X-Y图中曲线的信息。当您在当前视口中将光标定位在X-Y曲线上时,Abaqus/CAE会高亮显示最近的曲线点,并在"探测值"对话框中显示曲线图例文本、点的序列标识以及该点的X和Y坐标。Abaqus/CAE还使用符号高亮显示所选的曲线点,并在该符号旁边显示该曲线点的X和Y坐标。这些符号和坐标会一直保留在X-Y曲线图上,直到您退出探测模式。您可以选择仅显示组成X-Y数据对象的曲线点,或者通过插值显示曲线上的任意点。

在"探测"模式下选择一个点时,Abaqus/CAE会将当前点的图例文本、序列标识以及X和Y坐标添加到数据表中,该表出现在"探测值"对话框的底部。您可以清除或排序此表、删除单个行,以及将这些数据写入文件。

当"探测"模式处于活动状态时,Abaqus/CAE还会沿着X轴和Y轴用箭头跟踪光标在X-Y曲线图中的位置。这些箭头分别跟踪光标的当前X和Y位置,使您能够更精确地选择曲线点。

为了帮助定位X和Y数据点,您可能希望在使用了查询工具集之前生成一个带有可见曲线符号的X-Y曲线图。有关详细信息,请参阅["自定义X-Y曲线上使用的符号," 第47.6.5节](pt05ch47s05hlb04.md)。

**探测X-Y曲线图的步骤:**

1. 找到"探测值"对话框。在可视化模块工具箱中,点击 ![](../graphics/ico_probe.png)。"探测值"对话框出现。
2. 在对话框顶部,切换"**点间插值**"以启用或禁用曲线上任意点的探测值显示。当启用插值时,Abaqus/CAE通过在括号中列出包围X-Y数据对象点来显示任意点的序列标识。
3. 选择要包含在查询中的信息类型。在数据表任意位置点击鼠标按钮3,从出现的列表中选择"**编辑可见列**",然后切换您想要包含在查询中并显示为数据表列的信息类型。当某种信息类型被切换打开时,Abaqus/CAE可以在两个位置为所选元素显示该类型数据:在鼠标悬停或选择视口中元素时的数据表中,以及在探测值输出文件中。
4. 移动光标以探测当前视口中的X-Y曲线图。当您将光标定位在曲线点上时,Abaqus/CAE会在曲线本身上显示当前点的X和Y坐标,并在X-Y曲线的X轴和Y轴上显示光标的当前位置。
5. 要选择并累积感兴趣的值并允许随后将这些值写入文件,请在将光标定位在曲线点上时点击鼠标按钮1。Abaqus/CAE将当前点的图例文本、序列标识以及X和Y坐标存储到数据表中。Abaqus/CAE还通过在您选择的曲线点处放置一个符号并在那个点旁边显示X和Y坐标来标记您在视口中选择的点。
6. 要从数据表中删除数据或清除/排序表,请点击鼠标按钮3。有关详细信息,请参阅["输入表格数据," 第3.2.7节](pt01ch03s02s07.md)。
7. 要保存您选择的数据,点击"**写入文件**"。有关详细信息,请参阅[第54章,"生成表格数据报告](pt05ch54.md)。"
8. 要退出探测模式,请执行以下操作之一: - 点击提示区域的取消 ![](../graphics/afxI_cancel.png) 按钮。 - 点击"探测值"对话框中的"**取消**"。 - 点击鼠标按钮2。 - 以其他(例如,未变形、变形)模式生成曲线图。
