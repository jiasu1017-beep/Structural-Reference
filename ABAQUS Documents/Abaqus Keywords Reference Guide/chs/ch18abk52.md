# *SURFACE PROPERTY





### *SURFACE PROPERTY定义腔体辐射的表面属性。

此选项用于定义腔体辐射分析的表面属性。它必须紧跟在[*EMISSIVITY](ch05abk15.md)选项之前。

**产品：**Abaqus/Standard  Abaqus/CAE

**类型：**模型数据

**级别：**模型

**Abaqus/CAE：**Interaction模块

##### **参考：**

- ["腔体辐射," Section 41.1.1 of the Abaqus Analysis User's Guide](../usb/usb-link.md#usb-cni-acavityradiation)
- [*EMISSIVITY](ch05abk15.md)

### **必需参数：**

NAME

将此参数设置为一个标签，用于引用属性名称，以便交叉引用正在定义属性的表面定义。只能使用基于元素的表面。

**此选项没有关联的数据行。**
