# 1.9.2 Sinusoidal excitation of a damped spring-mass system







**Products: **Abaqus/Standard  Abaqus/Explicit  

### Elements tested

CONN2D2    CONN3D2    

### Problem description

Translational and rotational basic connector components as well as translational-rotational combination connections are tested for sinusoidal excitation of a damped spring-mass connector system. A series of simple, 2-node connector elements simulating a damped spring-mass system verify the connections described in [Table 1.9.2--1](ch01s09abv120.md#ver-elm-sinusoid-contypes). The kinetic behavior in the connector elements is modeled using the connector elasticity, connector damping, and connector friction behaviors.

In Abaqus/Standard a direct-integration implicit dynamic step is used with a small- or large-displacement formulation. In Abaqus/Explicit the usual explicit dynamic procedure is used with the default large-displacement formulation. The spring-mass system in each case is driven by assigning a connector element load.
