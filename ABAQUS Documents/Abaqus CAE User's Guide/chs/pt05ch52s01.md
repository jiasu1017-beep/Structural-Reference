# 52.1 理解应力线性化





应力线性化是将截面上的应力分离为恒定的膜应力和线性弯曲应力。应力线性化仅适用于输出数据库。要在Abaqus/CAE中对应力进行线性化,使用以下过程:
- 您定义模型中的一个截面。您通过在模型中选择节点、空间中的坐标或保存的路径来指定截面的端点。
- Abaqus/CAE通过在两点之间进行插值来定义应力线,以便沿着直线上获得用户指定的等间隔数。
- 您可以将应力线保存为路径;保存的路径包括应力线上的端点和每个间隔点。如果您将应力线保存为路径,Abaqus/CAE总是创建一个点列表路径,无论您使用什么方法来定义端点。
- 在由线定义的局部坐标系中,沿着线在等间隔的位置获得指定的应力结果。
- Abaqus/CAE执行应力线性化计算,并以X-Y曲线图的形式显示结果。您可以选择保存数据和/或将其写入文件。

如果您跨多个部件实例对应力进行线性化,对于存在于多个部件实例中的任何点,结果将被平均。

要沿着一线获取线性化应力,请从主菜单栏选择 ****Tools**![](../graphics/images/arrow.gif)**Query****,然后从"可视化模块查询"选项中选择"**应力线性化**"。
