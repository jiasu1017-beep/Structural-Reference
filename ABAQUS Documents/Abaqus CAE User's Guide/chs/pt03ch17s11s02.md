# 17.11.2 自底向上网格的域





当您开始创建自底向上网格时，第一个要求是定义将创建网格的域。该域决定自底向上网格是作为原生网格还是作为孤立节点和单元的集合创建。要创建原生网格，您选择一个三维几何模型区域，该区域已在**网格控制**对话框中分配了自底向上网格技术。（更多信息，请参见["分配网格控制，"第17.18.1节](pt03ch17s18s01.md)。）如果没有分配自底向上技术的几何区域，您必须创建一个或使用孤立单元域。

选择三维区域作为域也表明您打算将自底向上网格与几何体完全关联。网格与几何体之间的关联对于将载荷、边界条件和其他信息从几何体传递到网格是必要的。网格-几何体关联在["网格-几何体关联，"第17.12节](pt03ch17s12.md)中讨论。

选择孤立单元作为域表明自底向上网格不会与几何体完全关联。创建的单元可能与几何体没有关联，或者您可以将创建的孤立网格实体与相邻的几何面关联。将孤立网格面与相邻几何面关联允许Abaqus/CAE在您对几何体进行网格划分时创建兼容的网格。
