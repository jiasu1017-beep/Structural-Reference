# 11.6 什么是孤儿节点和单元？





孤儿节点和单元是不存在任何相关几何的有限元网格组件。实际上，网格信息已与其父几何分离。孤儿节点和单元可以通过多种方式创建；它们可以：
- 从输出数据库导入（更多信息，请参见"从输出数据库导入零件"，第10.7.12节）
- 从Abaqus输入文件导入（更多信息，请参见"导入模型"，第10.8节）
- 作为网格零件创建（更多信息，请参见"创建网格零件"，第17.20节）
- 在自底向上网格划分过程中创建（更多信息，请参见"自底向上网格划分"，第17.11节）
- 通过某些网格编辑操作创建，例如创建单元和偏移（更多信息，请参见"网格编辑工具概述"，第64.4节）
- 通过删除与其父几何的关联性创建（更多信息，请参见"删除网格-几何关联性"，第64.7.12节）

上述前三种方法将孤儿网格特征创建为新零件的基础特征。其余方法是编辑网格工具集的一部分；这些方法编辑现有网格，因此不作为零件的特征存在。更多信息，请参见第64章，"编辑网格工具集"。

您可以选择孤儿单元的面作为草图平面来添加几何特征。此外，在网格模块中，您可以更改分配给孤儿单元的单元类型，并可以验证和编辑网格。
