# 75.6.5 恢复被虚拟拓扑替换的实体





您可以选择 ****Tools**![](../graphics/images/arrow.gif)**Virtual Topology**![](../graphics/images/arrow.gif)**Restore Entities**** 从主菜单栏来恢复先前被虚拟拓扑替换的边和顶点。使用恢复工具允许您选择要恢复的单个实体，而不是删除或抑制虚拟拓扑特征（这会恢复所有实体）。

**恢复实体的步骤：**

1. 从上下文栏的 **Object** 字段中，选择一个零件或装配。
2. 从主菜单栏，选择 ****Tools**![](../graphics/images/arrow.gif)**Virtual Topology**![](../graphics/images/arrow.gif)**Restore Entities****。**提示：**您也可以使用位于 Mesh 模块工具箱中的 ![](../graphics/ico_virtTopoRestore.png) 工具来恢复实体。Abaqus/CAE 以蓝色高亮显示所有已被虚拟拓扑替换的实体。
3. 选择要恢复的实体组合。您可以使用拖选、**[Shift]****+点击** 和 **[Ctrl]****+点击** 的组合来选择多个要恢复的实体。有关更多信息，请参见 ["在当前视口中选择对象，" 第6.2节](pt01ch06s02.md)。
4. 点击鼠标按钮2，或在提示区域点击 **Done**。Abaqus/CAE 在模型树中创建一个 **Restore ignored geometry** 特征来恢复选定的实体——虚拟拓扑特征不会改变。如果还有更多可以恢复的实体，Abaqus/CAE 继续选择过程。
5. 当您完成恢复实体后，点击鼠标按钮2或提示区域中的取消按钮 ![](../graphics/afxI_cancel.png)。
