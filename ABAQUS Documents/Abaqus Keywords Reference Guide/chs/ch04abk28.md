# *DISTRIBUTING COUPLING





### *DISTRIBUTING COUPLING为分布式耦合单元指定节点和加权。

此选项用于定义力和质量根据指定加权分布到的节点集，并指定相关分布式耦合单元的质量。定义分布式约束的首选方法是使用 [*COUPLING](ch03abk83.md) 选项与 [*DISTRIBUTING](ch04abk27.md) 选项结合使用。如果还需要分布参考节点处的点质量，则必须使用 DCOUP* 单元以及 [*DISTRIBUTING COUPLING](ch04abk28.md) 选项。

**产品：**Abaqus/Standard  Abaqus/CAE  

**类型：**模型数据  

**级别：**部件、部件实例、装配  

**Abaqus/CAE：**不支持；此选项已被与分布式选项结合使用的耦合约束取代。

##### **参考：**

- ["分布式耦合单元，" Abaqus分析用户指南第32.4.1节](../usb/usb-link.md#usb-elm-edistcoupling)

### **必需参数：**

ELSET

将此参数设置为包含与耦合节点相互作用的分布式耦合单元的单元集合名称。此单元集合可以包含多个单元，尽管这不是典型情况。

### **可选参数：**

MASS

将此参数设置为要分布到耦合节点的质量。

### **指定耦合节点和分配权重因子的数据行：**

**第一行：**

根据需要重复此数据行。每个分布式耦合定义必须至少指定两个耦合节点。
