# 12.10.2 Including volumetric heat generation in heat transfer analyses







User subroutine [`HETVAL`](../sub/sub-link.md#sub-xsl-hetval) allows you to define a heat flux due to internal heat generation for heat transfer, coupled thermal-electric, or coupled temperature-displacement analyses. 

Abaqus/Standard calls user subroutine [`HETVAL`](../sub/sub-link.md#sub-xsl-hetval) at each point for which the material definition includes a reference to the user subroutine. For more information, see ["HETVAL," Section 1.1.13 of the Abaqus User Subroutines Reference Guide](../sub/sub-link.md#sub-rtn-uhetval).

**To include a reference to user subroutine [`HETVAL`](../sub/sub-link.md#sub-xsl-hetval) in a material definition:**

1. From the menu bar in the **Edit Material** dialog box, select ****Thermal**![](../graphics/images/arrow.gif)**Heat Generation****. (For information on displaying the **Edit Material** dialog box, see ["Creating or editing a material," Section 12.7.1](pt03ch12s07hlb01.md).)
2. Click **OK** to close the **Edit Material** dialog box. Alternatively, you can select another material behavior to define from the menus in the **Edit Material** dialog box (see ["Browsing and modifying material behaviors," Section 12.7.2](pt03ch12s07hlb02.md), for more information).
3. Enter the Job module and display the job editor for the analysis job of interest. (For more information, see ["Creating, editing, and manipulating jobs," Section 19.7](pt03ch19s07.md).)
4. In the job editor, click the **General** tab, and specify the file containing the user subroutine [`HETVAL`](../sub/sub-link.md#sub-xsl-hetval). For more information, see ["Specifying general job settings," Section 19.8.6](pt03ch19s08hlb06.md). **Note:**You can specify only one user subroutine file in the job editor; if your analysis involves more than one user subroutine, you must combine the user subroutines into one file and then specify that file.
