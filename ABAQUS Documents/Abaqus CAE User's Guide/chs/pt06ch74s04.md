# 74.4 自定义流显示







本节描述如何使用**Stream Plot Options**对话框中的设置自定义流线的颜色、线型和箭头。您可以通过在Abaqus/CAE的Visualization模块的主菜单栏中选择****Options**![](../graphics/images/arrow.gif)**Stream****来打开**Stream Plot Options**对话框。

您在**Stream Plot Options**对话框中所做的更改仅适用于当前视口中的活动流。每个视口都有独立的流选项，这使您能够在每个视口中以不同方式显示流线。例如，您可以在一个视口中使用均匀颜色显示流线，在另一个视口中显示使用当前主变量的带状轮廓的较粗流线。

您还可以通过在Abaqus环境文件(`abaqus_v6.env`)中的`onCaeStartup()`函数中编辑设置来控制**Stream Plot Options**对话框中出现的默认设置。更多信息，请参见["在环境文件中使用Abaqus Scripting Interface命令," 第6.9节（Abaqus Scripting User's Guide）](../cmd/cmd-link.md#cmd-int-acl-environment)。
