# 13.1 显式动力学类比





为了给您提供对慢速准静态加载情况和快速加载情况之间差异更直观的理解，我们使用如图 13-1 所示的类比。

**图 13-1** 慢速和快速加载情况的类比。

![](../graphics/gsx-fast-case-nls.png)

该图显示了一个满载乘客的电梯的两种情况。在慢速情况下，门打开，您走进去。为了腾出空间，门旁边的乘客慢慢推动他们的邻居，邻居再推动他们的邻居，依此类推。这种扰动穿过电梯，直到靠近墙壁的人表示他们无法移动。一系列波穿过电梯，直到每个人都达到新的平衡位置。如果您稍微提高速度，您将比之前更用力地推开您的邻居，但最终每个人都会与慢速情况下处于相同的位置。

在快速情况下，门打开，您高速冲进电梯，不给乘客时间重新安排以容纳您。您将伤害门口的两个人，而其他乘客将不受影响。

对准静态分析的想法也是如此。分析速度通常可以大幅提高，而不会严重降低准静态解的质量；慢速情况和稍微加速的情况的最终结果几乎相同。但是，如果分析速度提高到惯性效应占主导地位的程度，解往往局部化，结果与准静态解相当不同。
