# 29.3.1 定义将连接紧固件的表面





您可以使用**Domain**标签页编辑定位点并选择Abaqus/CAE将投影点和向量的目标表面。

**要定义紧固件的连接位置：**

1. 从**Edit Fasteners**对话框中，显示**Domain**标签页。
2. 选择定位点将投影到目标表面的方向向量。
   - 默认情况下，Abaqus/CAE沿着最近面的法线方向创建定位点将被投影的方向向量。
   - 要更改方向向量，切换**Specify**，单击![](../graphics/ico_selectBlue.png)，然后选择表示方向向量起点和终点的点。Abaqus/CAE显示方向向量。
3. 选择目标曲面的方法。
   - 选择**Whole model**以使用模型中的所有曲面作为目标曲面。
   - 选择**Fasten specified surfaces by proximity**以指定要连接的曲面。紧固点连通性由曲面沿方向向量的相对位置定义。
   - 选择**Fasten in specified order**以指定目标曲面的特定顺序。紧固点连通性由方向向量确定。
4. 双击目标曲面表中的第一行，然后在视口中选择曲面。单击提示区域中的**Done**以表示您已完成曲面选择。您可以从视口中选择多个曲面以定义单个曲面区域。
5. 在目标曲面表上的任意位置单击鼠标按钮3，以编辑所选曲面、添加新曲面或删除曲面。
