# 13.8.6 在装配模块中使用集和面





通过从装配中选择几何创建的集称为装配集，您可以使用集工具集来创建和管理装配集。例如，您可以选择装配集来指示施加载荷、边界条件和相互作用的位置。您也可以使用装配集来定义Abaqus/CAE将在分析期间从中生成输出的模型区域；例如，所选的顶点或面。装配集可以包含来自多个零件实例的区域。

相反，零件集是通过从零件模块或属性模块中的零件选择几何创建的。当您在装配模块中实例化一个零件时，您可以引用您之前创建的任何零件集；但是，Abaqus在装配相关模块中仅提供对这些零件集的只读访问。此外，您无法从装配相关模块中的**集管理器**访问零件集；但是，您可以在过程中通过点击**集**按钮并从出现的**区域选择**对话框中选择集来访问符合条件的零件集。更多信息，请参见["理解集和面，"第73.2节](pt06ch73s02.md)。

您通过从装配中选择面或边来创建面，并使用面工具集来创建和管理面。通常当过程需要面时您选择面；例如，当您施加分布载荷（如压力载荷）和定义接触相互作用时。更多信息，请参见["什么是面？，"第73.2.3节](pt06ch73s02s03.md)。

对于模型实例，在原始模型中定义的任何集或面都会被引入模型实例，保持模型树特征层次结构。
