# 16.3 管理规定条件







规定条件管理器是用于组织和操作与给定模型关联的规定条件的对话框。您可以在载荷模块中定义的每种规定条件都有单独的管理器。您可以通过从主菜单栏上的相应菜单中选择**管理器**来访问这些管理器。载荷模块提供以下管理器：
- **载荷管理器**
- **边界条件管理器**
- **预定义场管理器**

规定条件管理器包含您创建的所有某种类型规定条件的字母顺序列表。例如，所示的**载荷管理器**包含载荷列表。

**图16-1** **载荷管理器**。

![](../graphics/lbi-loadmanager-nls.png)

管理器中的**创建**、**编辑**、**复制**、**重命名**和**删除**按钮允许您创建新的规定条件或编辑、复制、重命名和删除现有规定条件。您还可以使用主菜单栏上的**载荷**、**边界条件**和**预定义场**菜单来启动创建、编辑、复制、重命名和删除过程。从主菜单栏选择管理操作后，过程与您单击管理器对话框内相应按钮完全相同。

规定条件管理器是依赖于步骤的管理器，这意味着它们包含模型中每个载荷、边界条件和预定义场历史的附加信息。您可以使用管理器左侧列中的图标来抑制规定条件或恢复先前抑制的规定条件以进行分析。抑制和恢复过程也可从主菜单栏上的**载荷**、**边界条件**和**预定义场**菜单获得。更多信息，请参见"抑制和恢复对象"，第3.4.3节。

您可以使用管理器对话框中的**复制**按钮、相应的菜单命令或模型树来复制载荷、边界条件或预定义场。您可以将这些对象从任何步骤复制到任何有效步骤，但有一些限制。更多详细信息，请参见"使用管理器对话框复制依赖于步骤的对象"，第3.4.11节。

**向左移动**、**向右移动**、**激活**和**停用**按钮允许您操作规定条件的逐步历史。更多信息，请参见"修改依赖于步骤的对象的历史"，第3.4.6节。

**注意：****激活**和**停用**按钮在**预定义场管理器**中不可用。
