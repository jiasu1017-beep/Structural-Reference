# 20.12.2 自动标注草图尺寸


向草图添加尺寸有助于确定绘制特征的最终大小和形状。Abaqus/CAE 可以自动为您的草图应用尺寸。您可以使用 sketcher 选项来选择 Abaqus/CAE 可以添加到草图的尺寸类型。(更多信息,请参见["自定义 Sketcher 中尺寸的格式和使用," 第 20.9.9 节](pt03ch20s09hlb09.md)。)

**自动标注草图尺寸的步骤:**

1. 从 Sketcher 工具箱中的工具中,选择自动标注尺寸工具 ![](../graphics/ico_sketchDimensionAuto.png)。有关 Sketcher 工具箱中工具的图示,请参见["Sketcher 工具," 第 20.4.1 节](pt03ch20s04s01.md)。Abaqus/CAE 会在提示区域显示提示,引导您完成操作。
2. 选择要标注尺寸的实体。**注意:**您可以选择整个草图;Abaqus/CAE 不会复制任何现有尺寸。
3. 如果您还希望 Abaqus/CAE 为所选实体自动创建约束,请切换打开**自动约束**。
4. 在提示区域单击**完成**。Abaqus/CAE 会在草图上创建尺寸(如果选择了约束,还会创建约束)。
5. 如果需要,使用拖动工具移动出现在绘制特征内的尺寸,并使用删除工具删除任何不需要的尺寸或约束。更多信息,请分别参见["拖动 Sketcher 对象," 第 20.17.1 节](pt03ch20s14hlb01.md)和["删除 Sketcher 对象," 第 20.19.4 节](pt03ch20s16hlb04.md)。
