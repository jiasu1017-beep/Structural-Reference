# 15.9.4 接触控制编辑器





要为面-面接触和自接触相互作用创建接触控制，请从主菜单栏选择**相互作用**![](../graphics/images/arrow.gif)**接触控制**![](../graphics/images/arrow.gif)**创建**。会出现**创建接触控制**对话框，您可以在其中指定接触控制的名称以及您想要创建的接触控制类型。一旦您指定了此信息，点击**继续**以显示接触控制编辑器。

**警告：**接触控制面向高级用户。这些控制的默认设置适用于大多数分析。使用这些控制的非默认值可能会大大增加分析的计算时间，或产生不准确的结果。在Abaqus/Standard分析中更改这些设置也可能导致收敛问题。

每个接触控制编辑器在对话框顶部面板显示您正在定义的接触控制的名称和类型。编辑器其余部分的格式取决于您是为Abaqus/Standard还是Abaqus/Explicit分析定义控制。

您可以通过从主菜单栏选择**帮助**![](../graphics/images/arrow.gif)**上下文帮助**，然后点击感兴趣的功能来显示编辑器特定功能的帮助。更多信息，请参见["在Abaqus/Standard分析中指定接触控制，"第15.13.9节](pt03ch15s13hlb07.md)，和["在Abaqus/Explicit分析中指定接触控制，"第15.13.10节](pt03ch15s13hlb08.md)。
