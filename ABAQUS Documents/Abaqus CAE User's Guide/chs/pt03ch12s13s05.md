### 选择轮廓类型

**Create Profile**对话框允许您指定要定义的轮廓类型。您可以通过提供几何数据来定义基于形状的轮廓，以便Abaqus可以计算截面的工程特性。或者，您可以通过直接提供截面的工程特性来定义广义轮廓。请参见[Abaqus Analysis User's Guide的第29.3.5节，"梁截面行为"](../usb/usb-link.md#usb-elm-ebeamsectionbehavior)。

将梁截面和梁取向分配给部件后，您可以使用部件显示选项来查看基于形状或广义的梁轮廓的理想化表示。显示梁轮廓有助于检查正确的轮廓已被分配给特定区域，以及分配的梁取向是否导致预期的轮廓方向。有关更多信息，请参见["控制梁轮廓显示，"第76.7节"](pt07ch76hla06.md)。

**选择轮廓类型：**

1. 从主菜单栏，选择****Profile**![](../graphics/images/arrow.gif)**Create****。出现**Create Profile**对话框。
   **提示：**您也可以在**Profile Manager**中点击**Create**，或在Property模块工具箱中选择创建轮廓工具![](../graphics/ico_profile.png)。
2. 输入轮廓名称。有关命名对象的更多信息，请参见["使用基本对话框组件，"第3.2.1节"](pt01ch03s02s01.md)。
3. 选择轮廓形状，然后点击**Continue**。出现您所选轮廓形状的**Edit Profile**对话框。
4. 在**Edit Profile**对话框中，输入所需的轮廓数据。请参阅以下章节获取详细信息：
   - ["定义箱形轮廓"](pt03ch12s13s05.md#usi-prp-section-profiles-box)

### 定义箱形轮廓

通过提供矩形空心箱的几何数据来定义箱形轮廓。

**定义箱形轮廓：**

1. 如["选择轮廓类型"](pt03ch12s13s05.md#usi-prp-section-profiles-create)中所述显示**Edit Profile**对话框。
2. 在**Width (a)**字段中，输入平行于局部1轴的线段长度。
3. 在**Height (b)**字段中，输入平行于局部2轴的线段长度。
4. 点击**Thickness**字段右侧的箭头，并指示您希望如何定义每个线段的厚度：
   - 如果箱的四个线段具有相同的厚度，请选择**Uniform**。如果选择此选项，请在下面的**Thickness**字段中输入线段厚度。
   - 选择**Individual**以分别输入每个线段的值。如果选择此选项，请输入以下内容：
     - 在**t1**字段中，输入图中标记为t1的线段厚度。
     - 在**t2**字段中，输入图中标记为t2的线段厚度。
     - 在**t3**字段中，输入图中标记为t3的线段厚度。
     - 在**t4**字段中，输入图中标记为t4的线段厚度。
5. 点击**OK**保存轮廓并关闭**Edit Profile**对话框。

### 定义管道轮廓

通过为空心圆提供几何数据并选择薄壁管道或厚壁管道的积分方案来定义管道轮廓。

**定义管道轮廓：**

1. 如["选择轮廓类型"](pt03ch12s13s05.md#usi-prp-section-profiles-create)中所述显示**Edit Profile**对话框。
2. 从**Integration scheme**选项中，指定薄壁管道或厚壁管道。
3. 在**Radius**字段中，输入从管道中心到管道壁外边缘的半径。
4. 在**Thickness**字段中，输入管道壁的厚度。
5. 点击**OK**保存轮廓并关闭**Edit Profile**对话框。

### 定义任意轮廓

您可以创建任意轮廓来建模简单薄壁开口或闭口截面的形状。您通过输入由直线段连接的一系列点来定义轮廓。请参见Abaqus Analysis User's Guide的第29.3.9节"梁横截面库"中的"任意薄壁开口和闭口截面"。

**定义任意轮廓：**

1. 如["选择轮廓类型"](pt03ch12s13s05.md#usi-prp-section-profiles-create)中所述显示**Edit Profile**对话框。
2. 在数据表格中，为**Point 1**输入局部坐标。
3. 为**Point 2**输入局部坐标。连接**Point 1**和**Point 2**的线在数据表格中标记为**Segment 1-2**。
4. 为**Segment 1-2**输入**Thickness**。
5. 为**Point 3**输入局部坐标。**Point 2**和**Point 3**之间的线在数据表格中标记为**Segment 2-3**。
6. 为**Segment 2-3**输入**Thickness**。
   **注意：**对于任意截面的每个单独线段，连接线段端点的线没有弯曲刚度。因此，任意截面必须包含至少两个线段。
7. 如有需要，输入其他点的局部坐标，并为生成的线段提供厚度。
8. 点击**OK**保存轮廓并关闭**Edit Profile**对话框。

### 定义广义轮廓

创建广义轮廓时，您直接在**Edit Profile**对话框中提供截面的工程特性。请参见[Abaqus Analysis User's Guide的第29.3.5节，"梁截面行为"](../usb/usb-link.md#usb-elm-ebeamsectionbehavior)。

**定义广义轮廓：**

1. 如["选择轮廓类型"](pt03ch12s13s05.md#usi-prp-section-profiles-create)中所述显示**Edit Profile**对话框。
2. 在**Area**字段中，输入轮廓形状的面积。
3. 在**I11**字段中，输入关于截面1轴弯曲的惯性矩。
4. 在**I12**字段中，输入交叉弯曲的惯性矩。
5. 在**I22**字段中，输入关于截面2轴弯曲的惯性矩。
6. 在**J**字段中，输入扭转常数。
7. 如果轮廓描述了开口截面梁，请输入以下内容：
   - 在**Gamma O**字段中，输入扇形矩，![](../graphics/usi_eqn00392.gif)。
   - 在**Gamma W**字段中，输入翘曲常数，![](../graphics/usi_eqn00393.gif)。
8. 点击**OK**保存轮廓并关闭**Edit Profile**对话框。
