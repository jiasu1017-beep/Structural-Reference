# 1.6.29 Abaqus/Standard中的通用接触







**产品：**Abaqus/Standard  

### 单元测试

S3    S4R    C3D4    C3D8    C3D8R    CAX4R    CPE4H    CPE8H    GK3D6

### 功能测试

Abaqus/Standard中的通用接触公式。

### 问题描述

本节涉及Abaqus/Standard中的通用接触功能，它使用有限滑动、表面-表面接触公式。这些示例使用不同类型的单元并测试通用接触中使用的不同功能。

### 结果与讨论

这些结果说明了Abaqus/Standard中通用接触公式的准确性。
