### 概述

求解控制参数可用于控制：
- 非线性方程求解精度；
- 时间增量调整；以及
- Abaqus/CFD到Abaqus/Standard或Abaqus/Explicit协同仿真中的FSI稳定性和网格畸变。

对于大多数分析，无需更改这些求解控制参数。但是，在困难的情况下，求解过程可能不会使用默认控制收敛，或者可能使用过多增量和迭代。在确定此类问题不是由建模错误引起之后，更改某些控制参数可能是有用的。

本节提供了更重要的求解控制参数的简要概述，以及可以有效使用它们的情况描述。

求解控制参数的值将在分析的剩余部分保持有效，直到被重置。用户可以将所有求解控制参数恢复为默认值（参见["收敛与时间积分准则：概述，" 第7.2.1节](pt03ch07s02abo11.md)）。

### 术语

在本节中，"通量"一词指的是正在寻求其离散平衡的变量，以及可能非线性平衡方程所针对的变量：力、力矩、热通量、浓度体积通量或孔隙液体积通量。"场"一词指的是系统的基本变量，例如连续体应力分析中的位移分量或传热分析中的温度。上标(i)指的是此类方程的一种。["非线性问题的收敛准则，" 第7.2.3节](pt03ch07s02aus51.md)中列出了Abaqus/Standard中可用的场和相应的通量。

#### 修改残差控制

α是最大残差与相应平均通量范数的比值用于收敛的收敛准则。的定义在["非线性问题的收敛准则，" 第7.2.3节](pt03ch07s02aus51.md)中。默认值为α = 5×10⁻³，按照工程标准这是相当严格的，但在除特殊情况外的所有情况下都将保证复杂非线性问题得到准确解。如果可以牺牲一些精度来提高计算速度，则可以将此比率的值增加到一个更大的数字。

#### 修改解修正控制

β是最大解修正与相应最大增量解值的比值的收敛准则。默认值为β = 10⁻²。除了足够小的残差外，Abaqus/Standard还要求解值的最大修正相对于最大相应增量解值较小。某些分析可能不需要这种精度，因此允许增加此比率。

#### 指定平均通量

是Abaqus/Standard用于检查残差的平均通量值。默认值是Abaqus/Standard计算的时间平均通量，如["非线性问题的收敛准则，" 第7.2.3节](pt03ch07s02aus51.md)中所定义。然而，用户可以为平均通量定义一个常数值，在这种情况下，在整个步骤中 = 常数。

用户可能希望对残差检查使用绝对容差。绝对容差值等于平均通量与比率的乘积。为避免检查解修正的大小，可以将β设置为1.0。

#### 修改初始时间平均通量

是当前步骤的时间平均通量的初始值。默认值是前一步骤的时间平均通量，或者是10²（如果是步骤1）。在分析耦合问题时，如果问题中的某些场在第一步不活跃，重新定义可能是有帮助的；例如，如果在完全耦合的热应力步骤之前执行静力步骤。

如果第一步本质上是空步骤，则重新定义也可能是有用的；例如，在接触问题中在任何接触发生之前，生成的初始通量（力）为零。在这种情况下，应将指定为场(i)首次活跃时将发生的典型通量大小。

的初始值将保留，直到完成满足的迭代，此时我们重新定义。对于相对于的零通量的判据为（参见["非线性问题的收敛准则，" 第7.2.3节](pt03ch07s02aus51.md)）。

如果用户直接指定平均通量，则指定的的值将被忽略。

### 控制时间增量方案

求解控制参数可用于改变收敛控制算法和时间增量方案。时间增量参数和是最重要的，因为它们对收敛有直接影响。如果收敛是（最初）非单调的或收敛是非二次的，则可能需要修改它们。

如果各种非线性相互作用，可能会出现非单调收敛；例如，摩擦、非线性材料行为和几何非线性的组合可能导致残差非单调递减。

如果Jacobian矩阵不是精确的，则会出现非二次收敛，这可能在复杂材料模型中发生。如果Jacobian矩阵是非对称的但使用了对称方程求解器，也可能会发生这种情况。在这种情况下，应为该步骤指定非对称方程求解器（参见["定义分析，" 第6.1.2节](pt03ch06s01abo05.md)）。

### 激活"线搜索"算法

在强非线性问题中，Abaqus/Standard中使用的Newton算法有时可能在平衡迭代期间发散。线搜索算法（在["使用线搜索算法提高求解效率"中讨论，见"非线性问题的收敛准则，" 第7.2.3节](pt03ch07s02aus51.md#usb-anl-aconvergcriteria-linesearch)）自动检测这些情况，并将一个比例因子应用于计算出的解修正，这有助于防止发散。当使用准Newton方法时（参见["求解方法"中"非线性问题的收敛准则，" 第7.2.3节](pt03ch07s02aus51.md#usb-anl-aconvergcriteria-solutionmethod)），线搜索算法特别有用。

#### 施加周期性条件

用户可以指定首次施加周期性条件的迭代次数。默认值为 = 1，在这种情况下，从分析开始就为所有迭代施加周期性条件。此求解控制参数很少需要从其默认值重置。

#### 定义稳态和塑性棘轮检测的容差

用户可以指定稳态检测准则和。是Fourier级数中任意项上最大残差系数与相应平均通量范数的最大允许比值，而是Fourier级数中任意项上位移系数的最大修正与最大位移系数的最大允许比值。默认值分别为 = 5×10⁻³和 = 5×10⁻³。如果同时满足这两个准则，则解收敛到稳态。

如果发生塑性棘轮，应力-应变曲线的形状保持不变，但一个周期内塑性应变的平均值在迭代之间继续偏移。在这种情况下，希望对Fourier级数中的常数项使用单独的容差来检测塑性棘轮。

用户还可以指定塑性棘轮检测准则和。是Fourier级数常数项上最大残差系数与相应平均通量范数的最大允许比值，而是Fourier级数常数项上位移系数的最大修正与最大位移系数的最大允许比值。默认值分别为 = 5×10⁻³和 = 5×10⁻³。如果任何周期项上的残差系数和位移系数修正分别在和设定的容差范围内，但常数项上的最大残差系数和常数项上位移系数的最大修正分别超过和设定的容差，则预期会发生塑性棘轮。

### 在Abaqus/CFD变形网格分析中控制求解精度和网格质量

求解控制参数可用于控制网格运动，并在涉及移动边界或变形几何的变形网格问题中保持网格质量。它们还可用于控制在执行Abaqus/CFD到Abaqus/Standard或Abaus/Explicit协同仿真时控制FSI稳定性。

#### 控制网格平滑和FSI稳定性

当使用网格平滑的隐式算法（默认）时，用户可以指定执行收敛检查之前的迭代次数、最大迭代次数、FSI罚函数比例因子、固体/流体密度比、线性收敛准则和刚度比例因子，以控制网格运动和FSI稳定性。

隐式算法使用无矩阵迭代方法求解伪弹性问题。迭代次数和线性收敛准则在FSI或变形网格问题的ALE过程中求解线性弹性方程时控制精度。减少迭代次数或放宽线性收敛准则有助于减少计算时间。同样，增加迭代次数或线性收敛准则有助于确保网格质量保持良好。刚度比例因子可用于缩放弹性刚度。降低弹性刚度会产生具有更多局部变形的ALE网格。

当使用网格平滑的显式算法时，用户可以指定最小网格平滑增量数、最大网格平滑增量数、FSI罚函数比例因子、固体/流体密度比和刚度比例因子，以控制网格运动和FSI稳定性。

最小和最大网格平滑增量数控制在FSI或变形网格问题的ALE过程中采取的网格平滑步骤数。减少最小和最大网格平滑增量数有助于减少计算时间。同样，增加最小/最大平滑增量数有助于确保网格质量保持良好，并避免变形网格问题演化过程中潜在的单元崩溃。

FSI罚函数比例因子用于控制FSI稳定性，默认值为1.0。对于高密度流体中的极柔性结构，当结构加速度较高时，可能需要以0.1的增量增加此参数。

固体/流体密度比也用于控制FSI稳定性。默认情况下，如果未指定其值，则忽略固体/流体密度比。当存在多个固体-流体界面时，用户应选择最小的固体/流体密度比。

#### 控制网格畸变

与Abaqus/Explicit中使用的畸变控制类似（详见["截面控制，" 第27.1.4节](pt06ch27s01aus113.md)），Abaqus/CFD提供畸变控制，以在使用显式网格平滑算法时防止单元在流体网格移动中反转或过度畸变。默认情况下，在协同仿真期间畸变控制关闭，如果使用隐式网格平滑算法则忽略。

| **输入文件用法：** | 使用以下选项在隐式网格平滑算法使用时停用畸变控制（默认）： |
| --- | --- |
|  | ``` [*CONTROLS](../key/key-link.md#usb-kws-hcontrols), TYPE=FSI, MESH SMOOTHING=EXPLICIT,DISTORTION CONTROL=OFF ``` 使用以下选项激活畸变控制： ``` [*CONTROLS](../key/key-link.md#usb-kws-hcontrols), TYPE=FSI, MESH SMOOTHING=EXPLICIT,DISTORTION CONTROL=ON ``` |

| **Abaqus/CAE用法：** | 在Abaqus/CAE中不支持控制Abaqus/CFD到Abaqus/Standard或Abaqus/Explicit协同仿真中的网格畸变。 |
| --- | --- |
