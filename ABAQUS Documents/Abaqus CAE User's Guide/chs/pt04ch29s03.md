# 29.3 创建基于点的紧固件





您可以在Interaction模块中通过从视口拾取每个定位点或从文件读取定位点坐标来创建基于点的紧固件。您可以允许Abaqus/CAE沿着垂直于表面的方向将定位点投影到最近的面，或者您可以指定点投影到最近面的方向向量。更多信息，请参见["关于紧固件"，第29.1节](pt04ch29s01.md)。

**要创建基于点的紧固件：**

1. 从Interaction模块的主菜单栏中，选择****Special**![](../graphics/images/arrow.gif)**Fasteners**![](../graphics/images/arrow.gif)**Create****。Abaqus/CAE显示**Create Fasteners**对话框。**提示：**您也可以使用Interaction模块工具箱中的![](../graphics/ico_fastener.png)工具创建基于点的紧固件。
2. 在出现的**Create Fasteners**对话框中，选择**Point-based**紧固件，然后单击**Continue**。
3. 选择定位点，然后单击**Done**。您可以选择连接点、参考点或来自孤立网格的节点。更多信息，请参见["通过拾取或从文件读取创建连接点"，第59.3节](pt06ch59hla01.md)；["通过选择方向和间距创建连接点"，第59.4节](pt06ch59hla02.md)；以及["基于边缘创建连接点图案"，第59.5节](pt06ch59hla03.md)。Abaqus/CAE显示**Edit Fasteners**对话框。
4. 在**Domain**、**Criteria**、**Property**、**Formulation**和**Adjust**标签页上，配置设置，如将创建点的区域、连接器截面、耦合类型和影响半径，如下述过程所述：
   - ["定义将连接紧固件的表面"，第29.3.1节](pt04ch29s03hlb01.md)
   - ["定义紧固件连接区域"，第29.3.2节](pt04ch29s03hlb02.md)
   - ["定义紧固件属性"，第29.3.3节](pt04ch29s03hlb03.md)
   - ["定义紧固件公式"，第29.3.4节](pt04ch29s03hlb04.md)
   - ["调整紧固件定义"，第29.3.5节](pt04ch29s03hlb05.md)
5. 单击**OK**创建基于点的紧固件。符号出现在视口中，代表您刚刚创建的基于点的紧固件的定位点。
