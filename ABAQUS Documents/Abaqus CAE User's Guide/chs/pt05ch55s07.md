# 55.7 显示单元和表面法线







创建未变形或变形图时，您可以选择在图中显示指示单元或表面法线方向的箭头。您可以为梁、管、框架单元、接触表面、膜、刚体单元、刚体表面、壳、三维实体和桁架显示法线方向。（有关特定类型单元或表面的法线方向的信息，请参见[Abaqus Analysis User's Guide的"单元"第六部分](../usb/usb-link.md#usbechapter)。）

**显示和自定义单元或表面法线的外观：**

1. 找到通用或叠加的法线选项。通用法线选项适用于在单独绘制时未变形和变形形状。当未变形形状与变形形状一起绘制时，叠加法线选项适用。从主菜单栏选择****选项**![](../graphics/images/arrow.gif)**通用****或**叠加**；然后单击出现的对话框中的"法线"选项卡。
2. 切换"显示法线"。出现"法线"选项。
3. 切换"在单元上"以显示模型中单元的法线，或"在表面上"以显示表面上法线。
4. 在标记为"颜色"的区域中，选择您希望不同类型法线显示的颜色。
5. 在标记为"样式"的区域中：
   1. 单击"长度"字段旁边的箭头，然后选择您想要的箭头长度。默认选择是"中等"。
   2. 单击"线条粗细"字段旁边的箭头，然后选择您想要的箭头粗细。
   3. 单击"箭头"字段旁边的箭头，然后选择您想要的箭头样式。
6. 单击"应用"以实现您的更改。箭头出现在视口中，指示模型中单元或表面的法线方向。通常，箭头出现在外部单元面的形心处。
