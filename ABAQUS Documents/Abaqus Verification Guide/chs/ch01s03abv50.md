# 1.3.45 内聚元件





**产品：**Abaqus/Standard  Abaqus/Explicit  

### 测试的功能

本节提供以下验证：
- 内聚元件的元件属性。
- 定义损伤起始的材料和接触属性。
- 定义损伤演化的材料属性。

还验证了Abaqus/Standard中未损伤孔隙压力内聚元件的压力连续性。

### I. 元件运动学

### 测试的元件

COH3D8    COH3D6    COH2D4    COHAX4    

### 问题描述

在本测试中验证内聚元件的以下三种本构响应类型：
- 用作垫片或小粘接斑块的内聚元件。
- 使用基于连续体的本构响应建模的有限厚度粘接层。
- 使用基于牵引-分离响应的建模的可忽略不计的薄粘接层。

通过施加适当的位移边界条件，验证每种响应在纯正法和两个纯剪切模式（二维和轴对称元件为一个剪切模式）下的变形。

![](../graphics/ver-cohesivebc.png)

**模型：**

本测试包括单元件模型，其几何形状被定义为每种情况下初始厚度为1.0。对于所有元件，厚度方向设置为全局1方向，但对于COH3D6，厚度方向设置为默认方向。

**材料：**

针对以下材料模型测试内聚元件的响应：
- 线弹性
- 超弹性
- 超泡沫
- Mises塑性
- Drucker-Prager塑性

### II. 损伤建模验证

### 测试的元件

COH3D8    COH3D6    COH2D4    COHAX4    COH3D8P    COH3D6P    COH2D4P    COHAX4P    

### 问题描述

本测试使用不同的损伤起始标准和损伤演化规律来验证内聚元件的损伤建模，以模拟内聚层的失效。使用线弹性材料模型验证基于内聚元件最大名义应变的损伤起始标准和基于二次牵引-相互作用的损伤起始标准。分别使用Mises和Drucker-Prager塑性验证基于延性应变和基于剪切失效应变的损伤起始标准。

损伤演化基于有效位移或消耗的能量来定义。定义了线性、指数和表格软化规律来指定损伤变量演化的性质。每种损伤模型在纯正法和两个纯剪切模式（二维和轴对称元件为一个剪切模式）下验证损伤。本测试还考虑了损伤演化对以表格、幂律或Benzeggagh-Kenane形式指定的开模式混合测量的依赖性。此外，测试验证了当同一材料定义有多个损伤起始标准活动时内聚元件的整体损伤。

### 结果与讨论

当满足指定的损伤起始标准时，内聚元件的响应开始退化。损伤变量根据以位移或能量耗散规定的演化规律演化。

### III. 孔隙压力内聚元件的压力连续性

### 测试的元件

COH3D8P    COH3D6P    COH2D4P    COHAX4P    

### 问题描述

本测试验证无损伤的孔隙压力内聚元件的压力连续性。

模型包含两个使用孔隙压力实体元件网格划分的块。一个块在顶部，另一个在底部。它们通过一层孔隙压力内聚元件相互连接。在测试中，没有向孔隙压力内聚元件引入损伤。当在模型的顶部和底部指定不同压力时，驱动流体平滑地流过内聚元件层，在各处产生相同的压力梯度。在一些测试中，通过为孔隙压力内聚元件定义渗漏系数来构建"滤饼"从而引入流动阻力。在一些测试中，实体和内聚元件具有不同的网格密度；因此，将使用基于表面的绑定约束来将它们相互连接。

### 结果与讨论
