# 45 区域命令

区域命令用于从元素、节点和几何创建零件和装配集或曲面。有关更多信息，请参阅["指定区域，"《Abaqus 脚本用户指南》第 6.6 节](../cmd/cmd-link.md#cmd-int-acl-regions)。零件和装配对象具有以下成员，即 Set 对象存储库：
- *sets*

Set 对象可以包含以下任一类型的实体：
- 元素
- 节点
- 几何

Set 对象可以包含多个单一类型（节点、元素或几何）或节点和元素类型的组合的实体。但是，除节点和元素外，Set 对象不能包含类型组合。

Set 对象的成员如下：
- nodes
- elements
- cells
- faces
- edges
- vertices
- referencePoints

区域命令还用于在装配上创建曲面。曲面是具有额外"面方向"信息的集。

零件集包含零件的区域。您可以为通过选择零件区域创建的集分配截面定义。零件集可以从实例访问；但是，您分配给区域的截面定义会自动复制到该零件在装配中的所有实例。

装配集包含装配的区域，用于许多在装配上运行命令。例如，您可以将负载或边界条件应用于通过选择装配区域创建的集。集可以包括来自多个零件实例的区域。
