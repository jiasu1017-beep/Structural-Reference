# 70.2 使用分割工具集





您可以通过从菜单栏中选择****工具**![](../graphics/images/arrow.gif)**分割****来访问分割工具集。将出现**创建分割**对话框，您可以从**类型**区域顶部的按钮中选择要分割的几何类型——边、面或单元。**方法**列表会更改以反映您可以创建的分割。从**方法**列表中选择所需的分割工具，然后按照提示区域中的提示创建分割。大多数工具允许您在一个操作中分割多个边、面或单元。您可以使用拖动选择、**[Shift]**+点击、**[Ctrl]**+点击和角度方法的组合来选择要分割的边、面或单元。有关更多信息，请参见["在当前视口中选择对象," 第6.2节](pt01ch06s02.md)。

["分割技术概述," 第70.4节](pt06ch70s04.md)提供了每种类型分割可用方法的概述。关于在零件和装配体上创建分割的更多信息，请参见["在零件模块中使用分割工具集," 第11.16.3节](pt03ch11s16s03.md)和["分割装配体," 第13.8.3节](pt03ch13s08s03.md)。

您也可以从模块工具箱访问分割工具集；[图70-2](pt06ch70s02.md#par-toolbox)显示了模块工具箱中所有分割工具的隐藏图标。
