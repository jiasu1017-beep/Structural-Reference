# 11.3 用户控制的区域分解







**产品：**Abaqus/Explicit  

**优点：**当局部区域计算密集时，可以实现更好的并行扩展性。

**说明：**您可以通过指定要独立分解的区域，或指定特定元素集的所有单元应约束到同一并行域来影响区域分解。此选项不推荐广泛使用。对于某些情况，当局部区域需要比其他区域更多的接触或其他计算时，此选项可以有益于并行扩展性，但此选项可能会降低性能。例如，将鸟撞模型中的撞击区域指定为独立的区域分解区域可能会改善并行扩展性。具有局部自适应网格细化的耦合欧拉-拉格朗日模拟（其中细化单元会增加计算成本）也可能从定义局部区域分解区域中受益。使用这种方法，每个处理器将处理多个域：例如，一个与局部区域关联的域和一个与模型其余部分关联的域。

可以指定多个区域分解区域。总域数是区域分解区域数乘以每个区域的用户指定域数。区域分解区域的数量通常比用户定义的区域分解区域数多一个，因为未明确定义为区域分解区域的单元将形成一个单独的区域分解区域。
