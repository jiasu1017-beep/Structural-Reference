### 概述

集中载荷：
- 将集中力和力矩施加到节点自由度上；并且
- 可以固定方向；或者
- 可以随节点旋转（称为跟随力），导致载荷刚度的额外且可能非对称的贡献

在稳态动力分析中，可以施加实部和虚部集中载荷（详见["直接求解稳态动力分析，" 第6.3.4节"](pt03ch06s03at09.md)和["基于模态的稳态动力分析，" 第6.3.8节"](pt03ch06s03at13.md)）。

可以在随机响应分析中定义多个集中载荷情况（详见["随机响应分析，" 第6.3.11节"](pt03ch06s03at16.md)。

集中载荷还用于在声学分析中在具有压力自由度的节点上施加压力共轭（见["声学和冲击载荷，" 第34.4.6节"](pt07ch34s04aus125.md)），以及为不可压缩流动指定流体参考压力（见["不可压缩流体动力学分析，" 第6.6.2节"](pt03ch06s06aus48.md)）。

连接器单元中的驱动载荷可以作为连接器载荷定义，类似于集中载荷。详见["连接器驱动，" 第31.1.3节"](pt06ch31s01alm26.md)。

这些载荷可以使用的过程在["规定条件：概述，" 第34.1.1节"](pt07ch34s01abo31.md)中概述。详见["施加载荷：概述，" 第34.4.1节"](pt07ch34s04aus120.md)了解适用于所有载荷类型的一般信息。

### 集中载荷

在Abaqus/Standard和Abaqus/Explicit分析中，集中力或力矩可以施加在任何节点自由度上。

您不应在圆柱坐标系的原点施加力矩载荷；这样做会使径向和切向载荷不确定。

| **输入文件用法：** | ``` [*CLOAD*](../key/key-link.md#usb-kws-hcload) *node number or node set, degree of freedom, magnitude* ``` |
| --- | --- |

| **Abaqus/CAE用法：** | 载荷模块：**创建载荷**：为**类别**选择**机械**，为**所选步的类型**选择**集中力**、**力矩**或**广义平面应变** |
| --- | --- |

#### 指定集中跟随力

您可以指定集中力的方向应随其施加的节点旋转。此规格仅应在大位移分析中使用，并且只能用于具有活动转动自由度（如梁和壳单元的节点，或在Abaqus/Explicit中刚性体上的tie节点）的节点，不包括广义平面应变单元的参考节点。如果您指定跟随力，则集中力的分量必须相对于参考配置指定。

跟随载荷导致刚度矩阵的非对称贡献，这通常称为载荷刚度。与载荷刚度贡献相关的一些问题在["改善大位移隐式分析收敛速率"](pt07ch34s04aus121.md#usb-prc-ploadgeneral-rateofconvergence)中讨论。

| **输入文件用法：** | ``` [*CLOAD*](../key/key-link.md#usb-kws-hcload), FOLLOWER ``` |
| --- | --- |

| **Abaqus/CAE用法：** | 载荷模块：**创建载荷**：为**类别**选择**机械**，为**所选步的类型**选择**集中力**或**力矩**：**跟随节点旋转** |
| --- | --- |

#### 从用户指定文件定义集中节点力的值

您可以使用先前Abaqus分析输出数据库（`.odb`）文件中特定步和增量的节点力输出来定义节点力。原始分析的零件（`.prt`）文件在从输出数据库文件读取数据时也需要。在这种情况下，先前模型和当前模型必须一致定义，包括节点编号，在两个模型中必须相同。如果模型基于零件实例的装配定义，则零件实例命名必须相同。

| **输入文件用法：** | ``` [*CLOAD*](../key/key-link.md#usb-kws-hcload), FILE=file, STEP=step, INC=inc ``` |
| --- | --- |

| **Abaqus/CAE用法：** | Abaqus/CAE不支持从用户指定文件定义集中节点力的值。 |
| --- | --- |

#### 指定流体参考压力

对于Abaqus/CFD中的不可压缩流体动力学分析，当未规定其他压力条件时，您必须在某个节点上指定流体参考压力以设置静水压力水平。可以指定多个参考压力，但仅应用最后指定的静水压力载荷。更多信息，请参阅["不可压缩流体动力学分析，" 第6.6.2节"](pt03ch06s06aus48.md)和["Abaqus/CFD中的边界条件，" 第34.3.2节"](pt07ch34s03aus119.md)。

| **输入文件用法：** | ``` [*CLOAD*](../key/key-link.md#usb-kws-hcload) *node number or node set*, HP, *magnitude* ``` |
| --- | --- |

| **Abaqus/CAE用法：** | 载荷模块：**创建载荷**：为**类别**选择**流体**，为**所选步的类型**选择**流体参考压力** |
| --- | --- |

### 定义随时间变化的集中载荷

集中载荷的规定幅值可以随步中时间按照幅值定义变化，如["规定条件：概述，" 第34.1.1节"](pt07ch34s01abo31.md)中所述。如果不同载荷需要不同的变化，每个载荷可以引用自己的幅值。

### 修改集中载荷

如["施加载荷：概述，" 第34.4.1节"](pt07ch34s04aus120.md)中所述，可以添加、修改或移除集中载荷。

### 改善大位移隐式分析收敛速率

当在几何非线性静态和动力学分析中指定集中跟随力时，通常应使用非对称矩阵存储和求解方案。见["定义分析，" 第6.1.2节"](pt03ch06s01abo05.md)，了解非对称矩阵存储和求解方案的更多信息。
