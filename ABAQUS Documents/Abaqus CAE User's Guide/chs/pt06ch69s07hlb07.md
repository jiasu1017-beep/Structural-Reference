# 69.7.7 偏移面







您可以通过从现有面偏移来创建面。Abaqus/CAE 将新面创建为壳。偏移操作作为部件的特征存储；因此，您可以使用模型树删除您创建的面。您可以在创建中面模型时使用偏移面； 有关信息，请参阅[第 35 章，"中面建模"](pt04ch35.md)。

您可以使用目标面计算偏移距离（如图 69-9 所示），或者您可以指定偏移距离。

**图 69-9** 发散面之间的恒定偏移距离。

![](../graphics/rep-faceoffset-nls.png)

无论您选择哪种偏移距离方法，Abaqus/CAE 都以与所选面的恒定偏移距离创建所有新面（如图所示）。但是，如果您使用目标面计算偏移距离，Abaqus/CAE 会使用要偏移的面和目标面来计算偏移面中每个节点处的厚度和偏移距离，使得偏移面的行为近似于原始截面的行为。您可以通过显示原生网格并在 **Part Display Options**（部件显示选项）对话框中切换开启 **Render shell thickness**（渲染壳厚度）和 **Scale factor**（比例因子）为 1 来在部件网格划分后查看这些计算的结果。

您可以使用 **Assign thickness and Offset**（分配厚度和偏移）工具查看和编辑壳厚度值（有关更多信息，请参阅["分配厚度和偏移，" 第 35.5.2 节"](pt04ch35s05s02.md)），或者在为新面分配截面时可以为它们指定厚度。

**通过偏移面创建新面：**

1. 从主菜单栏中选择 ****Tools**![](../graphics/images/arrow.gif)**Geometry Edit****（工具→几何编辑）。Abaqus/CAE 显示 **Geometry Edit**（几何编辑）对话框。**提示：**您也可以使用位于 Part（部件）模块工具箱中编辑工具处的 ![](../graphics/ico_offsetFaces.png) 工具来偏移面。 有关工具箱中编辑工具的图表，请参阅["使用几何编辑工具集，" 第 69.1 节"](pt06ch69s01.md)。
2. 从对话框中选择 **Face**（面）类别和 **Offset**（偏移）方法。
3. 选择要偏移的面。您可以单独、按面角度或按面曲率选择面；并且您可以选择包含面的现有集合。 有关在视口中选择对象的更多信息，请参阅["在当前视口中选择对象，" 第 6.2 节"](pt01ch06s02.md)。**注意：**默认选择方法基于您最近使用的选择方法。要恢复为其他方法，请点击提示区域右侧的 **Select in Viewport**（在视口中选择）或 **Sets**（集合）。
4. 从提示区域点击 **Done**（完成）。Abaqus/CAE 显示 **Offset Faces**（偏移面）对话框。
5. 选择以下可用选项之一来确定偏移距离：**注意：**无论计算方法如何，Abaqus/CAE 都会对面应用恒定偏移（如图 69-9 所示）。您必须使用其他方法在收敛或发散面之间居中创建新面。**Use target faces to compute distance**（使用目标面计算距离）选择一个或多个目标面。Abaqus/CAE 通过在多个点采样来计算要偏移的面与目标面之间的距离。使用以下距离选项之一： - 选择 **Half the average distance**（平均距离的一半）以按此量偏移新面。 - 选择 **Fraction distance to closest point on face**（到面上最近点的分数距离），然后输入一个值。Abaqus/CAE 按目标面上最近点距离乘以输入的值偏移新面。 - 选择 **Fraction distance to farthest point on face**（到面上最远点的分数距离），然后输入一个值。Abaqus/CAE 按目标面上最远点距离乘以输入的值偏移新面。您可以使用与第 3 步相同的方法选择目标面。如果您的原始面是中面模型的参考表示的一部分，您可以点击 **Auto Select**（自动选择）让 Abaqus/CAE 从参考表示的相对侧选择适当的面。潜在目标面之间的变化距离可能导致仅部分匹配原始面的选择。如有必要，点击 **Edit**（编辑）以在视口中修改选择。**Distance**（距离）输入偏移距离，或选择 ![](../graphics/ico_measure.png) 来测量并输入视口中对象之间的距离。分数或实际距离的负值会在面法向方向创建偏移；正直在相反方向创建偏移。如果你不确定法向方向，使用 **Shell element normals**（壳元素法向）常规查询来查看模型中壳的法向方向。 有关更多信息，请参阅["获取模型的一般信息，" 第 71.2.2 节"](pt06ch71s02hlb02.md)。实体的法向指向外部，因此，正偏移将在部件内部创建一个面，除非偏移距离大于部件厚度。
6. 如果模型包含参考表示，切换 **Auto trim to reference representation**（自动修剪到参考表示）以控制 Abaqus/CAE 是否将偏移面匹配到参考表示。如果此选项开启，Abaqus/CAE 会沿其自由边扩展偏移面，并在与参考表示相交处修剪它们。**注意：**对于复杂面，面扩展可能会失败。如果面扩展失败，Abaqus/CAE 会指示错误并修剪与参考表示相交的任何未扩展面。
7. 点击 **OK**（确定）。Abaqus/CAE 在距原始面规定距离处创建新面，过程在第 3 步重新开始。
8. 要退出偏移过程，请执行以下操作之一 - 点击提示区域中的取消按钮 ![](../graphics/fts-cancel.gif)，或 - 在 Abaqus/CAE 窗口中的任意位置点击鼠标按钮 2，或 - 从几何编辑工具集或 Part（部件）模块中的工具中选择其他操作。
