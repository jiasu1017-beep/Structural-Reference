# 11.9.4 切割特征





切割是从零件中移除材料的特征。切割可以是圆孔或任意形状。任何切割的草图轮廓必须是闭合的。在许多情况下，整个轮廓会影响切割特征的形状，即使它最初不与被切割的表面接触。要创建切割特征，请从主菜单栏选择**形状**![](../graphics/images/arrow.gif)**切割**，或从零件模块工具箱中选择一个切割工具。

绘制初始轮廓后，执行以下操作之一来创建切割特征：
- 要创建拉伸切割，请通过指定距离(*d*)拉伸轮廓。- 要创建切割放样特征，您可以从初始放样截面过渡到不同形状或方向的结束截面。- 要创建旋转切割，请通过指定角度(*α*)旋转轮廓。- 要创建扫掠切割，请沿指定路径扫掠轮廓。- 要创建圆孔，请输入孔的直径及其中心与两条选定边缘的距离。

绘制拉伸、旋转或扫掠切割的轮廓时，您可以在单个草图中绘制多个轮廓。

您可以使用切割工具向任何可变形或离散刚体零件添加切割特征。您不能向分析刚体零件添加切割特征；只能修改定义该零件的原始草图。
