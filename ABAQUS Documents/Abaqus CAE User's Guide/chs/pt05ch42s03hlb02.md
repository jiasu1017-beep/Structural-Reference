# 42.5.1 场输出变量选择概述





云图、模型探测、基于等值面的视图切割以及沿模型路径的X-Y结果图都显示分析特定步骤和帧中特定场输出变量的值。类似地，当您基于结果形成显示组或指定颜色编码，或当您从当前模型数据库中的数据显示载荷或预定义场时，这些结果也与特定场输出变量相关。显示其值的变量称为[主场输出变量](pt05ch42s03hlb03.md)。

[变形](pt05ch43.md)图根据分析特定步骤和帧中节点变量（如位移）值显示模型的形状。显示其值的变量称为[变形场输出变量](pt05ch42s03hlb04.md)。

符号图在分析的特定步骤和帧显示特定向量或张量变量的大小和方向。符号图也可以显示当前模型数据库中模型中选定载荷或预定义场的大小和方向。显示其值的变量称为[符号场输出变量](pt05ch42s03hlb05.md)。

您可以为选定场输出变量指定基于结果的标准用于单元失败，并从模型图中移除满足失败标准的单元。定义失败标准的变量称为[状态场输出变量](pt05ch42s03hlb06.md)。

您可以指定流数据显示的速度或涡量数据。显示这些数据的变量称为[流场输出变量](pt05ch42s03hlb07.md)。

您可以选择在未变形或变形模型形状上显示云图和符号图。当使用变形模型形状时，轮廓或符号分别表示主场输出变量或符号场输出变量的值，而底层模型的形状由变形场输出变量的值决定。

**选择场输出：**

1. 找到**Field Output**选项。从主菜单栏，选择****Result**![](../graphics/images/arrow.gif)**Field Output****。**Field Output**对话框出现。**提示：**您也可以通过点击任何出现此按钮的对话框中的**Field Output**按钮，或通过点击**Field Output**工具栏中的![](../graphics/ico_fieldOutput.png)来访问这些选项。
2. 选择您想要的主场、变形、符号和状态场输出变量，如以下各节所述：
   - ["选择主场输出变量，" 第42.5.3节](pt05ch42s03hlb03.md)
   - ["选择变形场输出变量，" 第42.5.4节](pt05ch42s03hlb04.md)
   - ["选择符号场输出变量，" 第42.5.5节](pt05ch42s03hlb05.md)
   - ["选择状态场输出变量，" 第42.5.6节](pt05ch42s03hlb06.md)
   - ["选择流场输出变量，" 第42.5.7节](pt05ch42s03hlb07.md)
   - ["选择复数结果，" 第42.5.8节](pt05ch42s03hlb08.md)
   - ["选择截面点数据，" 第42.5.9节](pt05ch42s03s01.md)
   - ["选择接触输出，" 第42.5.10节](pt05ch42s03hlb09.md)

   当前视口中的模型图会更改以显示您选择的变量。如果处于活动状态，图例和状态块中的文本会更改以标识与图关联的变量。有关图例和状态块的更多信息，请参见["自定义图例，" 第56.1节](pt05ch56s01.md)和["自定义状态块，" 第56.3节](pt05ch56s03.md)。此外，Abaqus会刷新所有标识当前变量的对话框。
