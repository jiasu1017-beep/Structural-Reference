# 31.2.5 控制小应变分析中裂缝尖端的奇异性





如果裂缝区域的几何体定义了尖锐裂缝，则应变场在裂缝尖端变得奇异，如["传统有限元方法小应变分析的断裂力学网格构建"中的"轮廓积分评估"，Abaqus Analysis User's Guide第11.4.2节](../usb/usb-link.md#usb-anl-acontintegral-singularity)中所述。在模型中包含小应变分析的奇异性可提高轮廓积分以及应力和应变计算的准确性。

要在轮廓积分估计中包含奇异性，请在**Crack**编辑器的**Singularity**标签页上单击并选择中节点和退化单元控制所需的位置。详细说明请参见["控制裂缝尖端的奇异性"，第31.2.8节](pt04ch31s02hlb01.md)。

此外，您必须在Mesh模块中执行以下操作：
- 如果装配或部件是二维的，您必须使用一圈三角形对裂缝前缘进行建模，并为轮廓积分区域的其余部分分配四边形单元。
- 如果装配或部件是三维的，您必须使用一圈楔形单元对裂缝前缘进行建模，并为轮廓积分区域的其余部分分配六面体单元。

当您对裂缝前缘进行网格划分时，Abaqus执行以下操作：
- 将裂缝前缘中的单元转换为退化的四边形或六面体单元。
- 如果您使用一圈二阶三角形或楔形单元对裂缝前缘进行建模，Abaqus会将中间节点移动到沿从裂缝尖端或裂缝线辐射出的单元边缘上指定的位置。（如果您使用一圈一阶三角形或楔形单元对裂缝前缘进行建模，则Abaqus忽略您为中间节点指定的位置。）

包含孤立网格单元的部件实例始终是依赖实例。因此，您无法在装配中调整这些实例的中间节点。更多信息，请参见["在装配中编辑孤立网格、网格化部件和网格化部件实例有什么区别？"，第64.2节](pt06ch64s02.md)。您必须显示原始孤立网格并使用Edit Mesh工具集来调整中间节点的位置。更多信息，请参见["调整中间节点的位置"，第64.5.7节](pt06ch64s05s07.md)。此外，您无法从孤立网格的三角形和楔形单元创建退化单元。
