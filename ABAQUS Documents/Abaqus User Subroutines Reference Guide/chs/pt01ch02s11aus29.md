# 2.11.1 定义矩阵





**产品：** Abaqus/Standard

##### **参考文献**

### 概述

矩阵：
- 可用于表示模型一部分或整个模型的刚度、质量、粘性阻尼或结构阻尼；
- 通过赋予唯一名称和指定矩阵数据（可以缩放）来定义；
- 可以是对称的或非对称的；
- 可以以文本格式给出（下三角、上三角或方阵形式），或从矩阵生成过程生成的二进制`.sim`文件读取；
- 可用于提供具有大平移但没有大旋转的线性弹性响应；
- 可用于静态和固有频率提取过程；
- 可用于矩阵生成和子结构生成过程；
- 可用于使用SIM架构的瞬态模态动力学、基于模态的稳态动力学、基于子空间的稳态动力学、随机响应、响应谱和复特征值提取过程；
- 可以直接对任何矩阵节点自由度施加载荷、边界条件和约束；
- 可用于子模型分析；以及
- 不能用于不使用SIM架构的直接稳态动力学或基于模态的分析。

### Abaqus/Standard中的矩阵是什么？

设计汽车等结构的复杂模型通常涉及将工作分包给各个部件。当必须将整个模型放在一起时，需要在不同供应商之间交换部件信息。通常，为了避免交换专有信息，这些信息以矩阵形式交换，表示每个部件的刚度、质量和阻尼。在分析过程中，这些矩阵被添加到相应的全局有限元矩阵中，以完成整个模型的装配。

Abaqus/Standard提供了直接输入刚度、质量、粘性阻尼和结构阻尼矩阵的能力。您可以定义构建模型所需的任意多个不同矩阵。

### 在模型中包含矩阵

您必须为矩阵分配一个名称以将其包含在矩阵使用模型中。

| **输入文件用法：** | ``` [*MATRIX INPUT](../key/key-link.md#usb-kws-mmatrixinput), NAME=*name* ``` |
| --- | --- |

#### 指定矩阵类型

对于以文本格式给出的矩阵，您可以指定矩阵类型为对称的（默认）或非对称的。如果是对称的，可以以、下三角、上三角或方阵形式输入。

对于从`.sim`文件读取的矩阵，矩阵类型根据SIM数据库上存储的矩阵数据自动设置。

| **输入文件用法：** | 使用以下选项之一指定文本格式矩阵的类型： |
| --- | --- |
|  | ``` [*MATRIX INPUT](../key/key-link.md#usb-kws-mmatrixinput), NAME=*name*, TYPE=SYMMETRIC ``` ``` [*MATRIX INPUT](../key/key-link.md#usb-kws-mmatrixinput), NAME=*name*, TYPE=UNSYMMETRIC ``` |

#### 缩放矩阵数据

您可以为所有矩阵条目定义乘以比例因子。

| **输入文件用法：** | ``` [*MATRIX INPUT](../key/key-link.md#usb-kws-mmatrixinput), NAME=*name*, SCALE FACTOR=*sval* ``` |
| --- | --- |

#### 直接提供矩阵数据

您可以直接指定数据来以下三角、上三角或方阵格式定义对称矩阵。对于要对称的方阵，对角线上方和下方的相应条目必须具有完全相同的值。您可以通过为每个矩阵条目提供数据来直接定义非对称矩阵。

| **输入文件用法：** | ``` [*MATRIX INPUT](../key/key-link.md#usb-kws-mmatrixinput) *row node label*, *degree of freedom for row node*, *column node label*, *degree of freedom for column node*, *matrix entry* ``` |
| --- | --- |
|  | 重复此数据行以为每个矩阵条目指定数据。 |

#### 从备用文件读取文本格式的矩阵数据

文本格式的矩阵数据可以包含在备用文件中。通常，备用文件用于大型矩阵。为确保可接受的性能，备用文件中的数据行在读取时不进行详细的数据格式检查。您应确保数据条目以正确的格式指定，不包含任何注释或空行。文本格式输出的矩阵数据可以在矩阵生成过程中生成（请参阅["生成矩阵"中的"输出"第10.3.1节](pt04ch10s03at32.md#usb-anl-amtxgenerate-output)。

| **输入文件用法：** | ``` [*MATRIX INPUT](../key/key-link.md#usb-kws-mmatrixinput), NAME=*name*, INPUT=*input_file_name* ``` |
| --- | --- |

#### 从SIM数据库读取矩阵数据

二进制格式的矩阵数据可以从矩阵生成过程生成的`.sim`文件读取（请参阅["生成矩阵"中的"引言"第10.3.1节](pt04ch10s03at32.md#usb-anl-amtxgenerate-sim)。`.sim`文件可以包含刚度、质量、粘性阻尼和结构阻尼矩阵。您指定要从`.sim`文件读取的每个矩阵。

| **输入文件用法：** | 使用以下选项： |
| --- | --- |
|  | ``` [*MATRIX INPUT](../key/key-link.md#usb-kws-mmatrixinput), NAME=*stif_name*, INPUT=*sim_file_name*, MATRIX=STIFFNESS ``` ``` [*MATRIX INPUT](../key/key-link.md#usb-kws-mmatrixinput), NAME=*mass_name*, INPUT=*sim_file_name*, MATRIX=MASS ``` ``` [*MATRIX INPUT](../key/key-link.md#usb-kws-mmatrixinput), NAME=*dmpv_name*, INPUT=*sim_file_name*, MATRIX=VISCOUS DAMPING ``` ``` [*MATRIX INPUT](../key/key-link.md#usb-kws-mmatrixinput), NAME=*dmps_name*, INPUT=*sim_file_name*, MATRIX=STRUCTURAL DAMPING ``` |

### 在模型中定义刚度、质量和阻尼以及包含的矩阵

您可以将已指定的刚度、质量、粘性阻尼和结构阻尼矩阵组装到模型的相应全局有限元矩阵中。可以定义和组装多个具有不同名称的矩阵。

| **输入文件用法：** | 使用以下选项组装从相同原始模型生成的矩阵： |
| --- | --- |
|  | ``` [*MATRIX ASSEMBLE](../key/key-link.md#usb-kws-mmatrixassemble), STIFFNESS=*stif_name*, MASS=*mass_name*, VISCOUS DAMPING=*dmpv_name*, STRUCTURAL DAMPING=*dmps_name* ``` 要组装从不同原始模型生成的矩阵，请为每个模型重复[*MATRIX ASSEMBLE](../key/key-link.md#usb-kws-mmatrixassemble)选项。 |

#### 连接由矩阵表示的模型部分

由用户定义矩阵表示的模型部分通过共享节点连接到其他部分和有限元。您必须直接在模型中定义这些节点（请参阅["节点定义，" 第2.1.1节](pt01ch02s01aus05.md)）。此外，可能有仅由矩阵使用但未共享的节点。您不需要定义未共享且没有与其关联的载荷、边界条件或约束的节点；这些节点将为您定义并放置在全局坐标系的原点。

| **输入文件用法：** | 使用以下选项直接定义共享节点： |
| --- | --- |
|  | ``` [*NODE](../key/key-link.md#usb-kws-mnode) ``` |

#### 在组装矩阵中重新映射用户定义节点

组装矩阵中定义的节点可以重新映射（重命名）为矩阵使用模型中的不同节点标签。您必须在矩阵使用模型中定义所有新节点标签，从它们创建节点集，并在组装矩阵时指定此节点集。节点集的大小和集中节点的顺序必须与所有组装矩阵的节点组合集完全对应。假设矩阵节点按其原始标签的升序排序，这些标签是在生成时定义的或在矩阵数据中指定的。

| **输入文件用法：** | 使用以下选项为矩阵节点创建节点集： |
| --- | --- |
|  | ``` [*NSET](../key/key-link.md#usb-kws-mnset), NSET=*nset_name*, UNSORTED ``` 使用以下选项通过节点重新映射组装矩阵： ``` [*MATRIX ASSEMBLE](../key/key-link.md#usb-kws-mmatrixassemble), STIFFNESS=*stif_name*, MASS=*mass_name*, VISCOUS DAMPING=*dmpv_name*, STRUCTURAL DAMPING=*dmps_name*, NSET=*nset_name* ``` |

#### 矩阵的多次实例化

通过节点重新映射功能，相同矩阵可以在矩阵使用模型中多次使用。您定义矩阵一次，然后多次组装它，指定相关的节点集进行重新映射。

| **输入文件用法：** | ``` [*MATRIX INPUT](../key/key-link.md#usb-kws-mmatrixinput), NAME=*name* [*MATRIX ASSEMBLE](../key/key-link.md#usb-kws-mmatrixassemble), STIFFNESS=*name* [*MATRIX ASSEMBLE](../key/key-link.md#usb-kws-mmatrixassemble), STIFFNESS=*name*, NSET=*nset1_name* [*MATRIX ASSEMBLE](../key/key-link.md#usb-kws-mmatrixassemble), STIFFNESS=*name*, NSET=*nset2_name* ``` |
| --- | --- |

#### 矩阵数据中的内部节点

内部节点是具有与之关联的内部自由度的节点（例如，拉格朗日乘子和广义位移），由Abaqus/Standard内部创建。根据定义，用户定义节点具有正节点标签，内部节点具有负节点标签。您可以使用矩阵生成过程将某些用户定义节点指定为内部节点，以在矩阵使用模型中隐藏它们（请参阅["生成矩阵"中的"引言"第10.3.1节](pt04ch10s03at32.md#usb-anl-amtxgenerate-sim)。

当使用包含内部节点的矩阵数据时，这些节点会自动重新映射到矩阵使用模型中的唯一内部节点标签。对于从同一模型生成的组装矩阵，内部节点是共享的。对于从不同模型生成的组装矩阵，即使内部节点具有相同的负节点标签，它们也会映射到矩阵使用模型中的不同内部节点。

### 在非线性分析中使用矩阵

当您在非线性分析过程使用矩阵时，非线性不被考虑。由于矩阵数据在分析过程中保持不变，因此只能表示线性弹性材料行为，在几何非线性分析中只能正确建模大平移。在几何非线性分析中不会计算由于大旋转或载荷刚度导致的矩阵变化。

### 在线性能扰动分析中使用矩阵

矩阵可用于静态扰动分析以及使用Lanczos或AMS特征解算器的固有频率提取分析。为了正确计算某些量（如参与因子和全局惯性属性），与矩阵关联的节点的坐标应在模型中使用矩阵定义。矩阵也可用于使用高性能SIM架构的模态分析过程；即，稳态动力学、模态动力学、随机响应、响应谱和复频率提取分析。矩阵也可用于子结构生成和矩阵生成过程。

矩阵不能用于直接解稳态动力学分析过程和不使用高性能SIM架构的模态过程。

### 约束和转换

运动约束（例如，耦合约束、线性约束方程、多点约束或基于表面的绑定约束）可以施加到包含矩阵的模型中的任何节点。由于Abaqus/Standard中的运动约束通常通过消除从属节点的自由度来施加，因此矩阵节点不应被用作从属节点。

要将接触约束施加到矩阵节点，必须在这些节点上定义基于节点的表面，并且该表面应用作接触对定义中的从属表面。

在矩阵中出现的节点处定义的节点转换不会影响矩阵。与这些节点对应的矩阵条目假定位于由节点转换定义的局部坐标系中。

### 初始条件

初始条件可以照常指定；但是，只能将基于节点的初始条件应用于出现在矩阵中的节点。请参阅["Abaqus/Standard和Abaqus/Explicit中的初始条件，" 第34.2.1节](pt07ch34s02aus116.md)。

### 边界条件

边界条件可以照常指定。请参阅["Abaqus/Standard和Abaqus/Explicit中的边界条件，" 第34.3.1节](pt07ch34s03aus118.md)。矩阵节点可以在子模型分析中定义为被驱动节点（请参阅["子模型：概述，" 第10.2.1节](pt04ch10s02aus60.md)）；它们不能在全局模型中定义为驱动节点。对于壳到实体的子模型，定义为被驱动节点的矩阵节点被视为位于中心区域，无论它们离壳参考表面多远。

### 载荷

集中节点力可以像往常一样施加在任何节点的位移自由度（1-6）上。分布压力载荷可以施加到在矩阵节点上定义的表面单元上（请参阅["表面单元，" 第32.7.1节](pt06ch32s07alm52.md)）。体力不能施加到由矩阵表示的模型部分。用户定义的载荷可以与上述分布压力载荷和体力的相同限制一起施加。

像往常一样，可以在任何节点上施加预定义场（请参阅["预定义场"中的"预定义场变量"第34.6.1节](pt07ch34s06aus128.md#usb-prc-pfields-fieldvariables)和["预定义场"中的"预定义温度"第34.6.1节](pt07ch34s06aus128.md#usb-prc-pfields-temp)）；但是，矩阵数据不受预定义场的影响。例如，如果在出现在矩阵上的节点上将温度指定为预定义场，则只有与矩阵共享这些节点的单元在该单元指定了热膨胀时才会经历热应变。矩阵不会经历任何热应变，但它可能由于共享节点处的位移而经历线性弹 l力学力。

### 单元

所有可用于静态应力分析的单元都可用（请参阅["为分析类型选择适当的单元，" 第27.1.3节](pt06ch27s01aus112.md)。

### 输出

适用于静态分析的所有节点输出变量都可用（请参阅["Abaqus/Standard输出变量标识符，" 第4.2.1节](pt02ch04s02abv01.md)。

### 限制

使用矩阵已知的限制如下：
- 矩阵不能用于包含部件和装配的模型。
- 包含声压和机械自由度的矩阵将禁用耦合声-结构特征值提取。
- 默认情况下，不支持使用文本格式包含内部节点的矩阵数据。对于某些特殊情况，可以允许使用此类文本格式矩阵。此功能应谨慎使用。
- 在使用质量矩阵矩阵输入数据的Abaqus/Standard分析中，数据（`.dat`）文件中报告的全局模型的惯性量（包括质心坐标和惯性矩）可能计算不正确。
- 矩阵不能用于具有惯性 relief 载荷的分析。
- 矩阵不能用于直接稳态动力学分析或不使用基于SIM的架构的基于模态的分析。
