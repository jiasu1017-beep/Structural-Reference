# 4.4.1 FV2：销接双十字：面内振动







**产品：** Abaqus/Standard  

### 测试单元

B22    B23  

### 问题描述

![](../graphics/bmkfv2.png)

**材料：**

弹性模量 = 200 GPa，泊松比 = 0.3，密度 = 8000 kg/m³。

**边界条件：**

梁在A、B、C、D、E、F、G、H处销接。

### 参考解

这是英国国家有限元方法与标准机构（NAFEMS）推荐的测试：NAFEMS出版物TNSB第3版"The Standard NAFEMS Benchmarks"（1990年10月）中的测试FV2。

### Abaqus预测的振型

![](../graphics/bmkfv2-modes-nls.png)

模态2到8是具有相同特征值的振动模态。由于特征值相同，模态2到8的任何线性组合仍然是有效的模态。因此，这些模态的形状是图中所示模态形状的任意线性组合，特别是会因计算机而异。对于模态10到16也可以观察到相同的行为。
