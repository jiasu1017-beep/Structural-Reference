# 与 Abaqus/CAE 交互







## 第一部分：与 Abaqus/CAE 交互

本指南是 Abaqus/CAE（包括 Abaqus/Viewer）的主要参考文档。

**Abaqus/CAE**

Abaqus/CAE 是一个完整的 Abaqus 环境，为创建、提交、监控和评估 Abaqus/Standard 和 Abaqus/Explicit 仿真结果提供了一致简单的工作界面。Abaqus/CAE 分为多个模块，每个模块定义了建模过程的一个逻辑方面；例如，定义几何、定义材料属性和生成网格。当您从一个模块移动到另一个模块时，您构建的模型由 Abaqus/CAE 生成输入文件，然后您将该文件提交给 Abaqus/Standard 或 Abaqus/Explicit 分析产品。分析产品执行分析，向 Abaqus/CAE 发送信息以允许您监控作业进度，并生成输出数据库。最后，您使用 Abaqus/CAE 的 Visualization 模块（也可作为 Abaqus/Viewer 单独获得许可）读取输出数据库并查看分析结果。

**Abaqus/Viewer**

Abaqus/Viewer 提供 Abaqus 有限元模型和结果图形显示。Abaqus/Viewer 作为 Visualization 模块集成到 Abaqus/CAE 中。
