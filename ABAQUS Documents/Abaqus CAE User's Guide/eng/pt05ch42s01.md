# 42.3 Selecting the results step and frame







You can control the step and frame at which Abaqus/CAE obtains model data and results from an output database. You can select a specific step and frame using the **Step/Frame** dialog box, or you can step through frames using controls available in the context bar. 

**Note:**If you are displaying model data from the current model database in the Visualization module, you can select a specific step using the **Step/Frame** dialog box, but the frame controls are disabled. Model database data does not include individual frames.
