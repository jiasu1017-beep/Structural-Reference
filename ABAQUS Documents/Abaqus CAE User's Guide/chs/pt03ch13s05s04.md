# 13.5.4 使用平移至工具定位零件或模型实例







### 13.5.4 使用**平移至**工具定位零件或模型实例

**平移至**工具通过沿用户定义的定义运动方向的向量平移一个实例来定位两个零件或模型实例，直到可移动实例的所选面或边缘与固定实例的所选面或边缘相距指定距离。

当您使用**平移至**工具在三维建模空间中定位实例时，您选择面进行接触；对于二维或轴对称建模空间中的实例，您选择边缘进行接触。此外，当您使用**平移至**工具定位轴对称实例时，平移向量必须平行于旋转轴。

当您使用**平移至**定位工具时，您可以从固定实例和可移动实例中选择多个面或边缘。如果您不确定可移动实例沿所选向量移动时模型的哪一部分会接触，则选择多个面或边缘很有用。但是，为加快处理速度，您应尽可能少地选择面或边缘。

要将可移动零件或模型实例平移到固定实例，请执行以下操作：
- 从将移动的实例和将保持静止的实例中选择面或边缘。
- 通过定义平移向量来指定可移动实例的运动。图13-14说明了所选边缘和平移向量。**图13-14** 选择要接触的边缘并定义平移向量。![](../graphics/asm-help-contact-select-nls.png)
- 定义所选面或边缘之间的所需间隙。图13-15显示了指定间隙值为零和间隙值为*d*后接触约束的效果。**图13-15** 应用接触约束并指定间隙值为零和*d*的效果。![](../graphics/asm-help-contact-nls.png)要测量间隙*d*，Abaqus/CAE首先沿平移向量移动实例，直到任何一对所选面或边缘接触。然后Abaqus/CAE沿平移向量移动实例，移距为间隙值。间隙可以为零或正数或负数；间隙为负值会导致所选面或边缘之间产生过盈。当您使用**平移至**工具时，Abaqus/CAE根据其大小在容差范围内计算可移动实例的位置。如果您想避免任何过盈的可能性，应该指定一个小的间隙值，而不是简单地指定零。如果所选面或边缘沿平移向量不可能接触，Abaqus/CAE会显示错误消息并不移动实例。

即使您平移可移动实例直到与固定实例接触，所选表面的物理接近度也不足以表明它们之间有任何类型的相互作用。您必须使用交互模块来指定表面之间的机械接触。**平移至**定位工具仅在根据模型大小的容差范围内满足。因此，除非在两个平面之间应用，否则接触可能不精确。

Abaqus/CAE使用一组平面面来近似曲线面。同样，Abaqus/CAE使用一组平面边缘来近似曲线边缘小平面的数量取决于您在零件模块中创建零件时指定的曲线细化程度。使用框缩放工具![](../graphics/ico_viewZoomRectangle.png)查看应用于装配中曲线面或曲线边缘的平面化。当您平移曲线面或曲线边缘时，Abaqus/CAE使用此平面化表示法计算接触位置。您可能希望根据您知道将接触的面或边缘的曲率将曲线细化设置为更细的设置。请参见"控制曲线细化"，第76.4节。
