# 51.1.1 模型或模型图的探测





如果当前视口包含模型图（未变形、变形、等值线、符号或材料方向图）或来自当前模型数据库的模型，Abaqus/CAE允许您探测所选模型或输出数据库的结果。"探测值"对话框标识Abaqus/CAE将获取值的步骤、帧和场输出变量。要了解如何更改步骤、帧或场输出变量，请参阅["选择结果步骤和帧，"第42.3节](pt05ch42s01.md)和["选择主场输出变量，"第42.5.3节](pt05ch42s03hlb03.md)。

当您探测三维壳或复合实体的模型图时，Abaqus/CAE返回的结果取决于您在"截面点"对话框中选择的活动位置。当"顶部"是活动位置时，Abaqus/CAE从"顶部位置"截面点返回结果；当活动位置设置为"底部"或"顶部和底部"时，从"底部位置"截面点返回结果。有关截面点的更多信息，请参阅["选择截面点数据，"第42.5.9节](pt05ch42s03s01.md)。

对于模型图，您可以选择获取基于节点或基于单元的数据和结果，如下所示：

**探测节点**

当您选择探测节点并将光标置于当前视口中的节点位置时，Abaqus/CAE会高亮该节点，并在数据表中显示以下关于该节点的信息：
- 节点的标签，
- 节点的原始坐标，
- 节点的变形坐标（使用当前变形场输出变量），
- 共享该节点的单元，以及
- 当前场输出变量（包括分量值，如果当前场输出变量是张量或向量）。

如果当前视口中选择了模型，当前场输出变量反映模型属性（如载荷）在所选节点处的值。

无论当前主变量最初是在节点、质心还是积分点位置保存到输出数据库，您都可以显示该节点处的场输出。Abaqus/CAE通过向外推至节点并根据您选择的方式进行平均来计算质心和积分点数据的节点值。如果节点引用未平均的单元数据，将显示与探测节点关联的所有单元数据。有关平均的更多信息，请参阅["理解结果值平均，"第42.6.2节](pt05ch42s04s02.md)。

您还可以显示或隐藏为每个选择的节点出现在视口中的注释。探测注释显示节点的标签和该节点处当前场输出变量的结果。您可以自定义探测注释文本的字体和颜色。

**探测单元**

当您选择探测单元并将光标置于当前视口中的单元上时，Abaqus/CAE会高亮该单元，并在数据表中显示以下关于该单元的信息：
- 单元的标签，
- 单元的连接性，以及
- 当前场输出变量（包括分量值，如果当前场输出变量是张量或向量）。

如果当前视口中选择了模型，当前场输出变量反映模型属性（如载荷）在所选单元处的值。

仅当当前主变量是基于单元的变量（如应力或应变）时，才能显示该单元处的场输出。您可以选择场输出结果的输出位置：积分点、质心、单元节点或单元面。对于每个这些输出位置，Abaqus/CAE在单元基础上计算探测结果，不进行平均。

您还可以显示或隐藏为每个选择的单元出现在视口中的注释。探测注释显示单元的标签和该单元处当前场输出变量的结果。您可以自定义探测注释文本的字体和颜色。

无论模型以何种渲染样式显示，仅可见单元可以被探测。如有必要，您可以创建显示组或使用视图切割来显示原本不可用的内部单元。

当单元被裂纹穿透时，裂纹单元分成两部分，每部分由实域和虚域形成。当您探测裂纹单元时，仅报告来自单元一部分的贡献。更多信息，请参阅["可视化"中的"使用扩展有限元方法将不连续性建模为 enrichment特征，"Abaqus Analysis User's Guide第10.7.1节](../usb/usb-link.md#usb-anl-aenrichment-vis)。
