# 10.2.2 精度和容差如何相关？





精度和容差是成功将零件导入Abaqus/CAE的重要因素。对于修剪曲面，容差定义了边缘与边界边缘曲面之间的最大允许偏差。定义修剪曲面边缘的多项式的精度取决于CAD系统的容差。Abaqus/CAE使用ACIS来表示零件或装配。ACIS使用10^6的精度来定义几何实体。

要成功导入由修剪曲面或B-rep定义的实体，原始文件的容差和Abaqus/CAE的容差必须在可接受的限制内匹配。如果原始文件的容差比Abaqus/CAE的容差大得多，则导入可能会失败，因为Abaqus/CAE无法从修剪曲面和B-rep信息重建实体。

将零件导入Abaqus/CAE后，修复过程旨在提高零件的准确性。Abaqus/CAE尝试更改相邻实体，使其几何形状完全匹配。转换为精确表示通常会产生精确的几何形状。但是，这可能是一个漫长的操作，会增加导入零件的复杂性。因此，零件的后续处理和分析可能会更慢。而且，如果零件包含许多复杂曲面，转换为精确表示可能会失败。如果可能，您应该返回生成原始文件的CAD应用程序并提高精度。

您可以使用"零件"模块中的"查询"工具集来高亮显示导入零件中具有几何精度和有效性错误的区域。不精确的顶点可以认为是围绕虚拟球体的顶点，其中球体的直径等于局部精度。当您修复零件时，ACIS假定虚拟球体内的任何点都与顶点重合，如图[图10-6](pt02ch10s02s02.md#dbs-conc-tolerance-sphere)所示。

**图10-6** 虚拟球体定义不精确顶点。

![](../graphics/dbs-conc-tolerance-sphere-nls.png)
