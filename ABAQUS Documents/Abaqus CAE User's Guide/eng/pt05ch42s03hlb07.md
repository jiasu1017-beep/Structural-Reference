# 42.5.7 Selecting the stream field output variable







You can specify the variable for which stream data are computed; this variable is called the stream field output variable. For most analyses velocity (**V**) can be used for the stream data; however, for some analyses you can use **VORTICITY** as the stream field output variable instead. Only node-centered vector results can be associated with streamline traces.

When selecting a stream field output variable, Abaqus/CAE lists all of the vector variables stored at nodal locations and available at the current [step and frame](pt05ch42s01hlb01.md) of your [output database](pt02ch09s06hlb02.md). For information on individual output variable identifiers, see ["Abaqus/CFD output variable identifiers," Section 4.2.3 of the Abaqus Analysis User's Guide](../usb/usb-link.md#usb-out-hcfdoutputvar).

**To select the stream field output variable:**

1. Locate the options that control the stream field output variable. From the main menu bar, select ****Result**![](../graphics/images/arrow.gif)**Field Output****. Click the **Stream Variable** tab in the dialog box that appears. **Tip:**You can also access these options by clicking the **Field Output** button in any dialog box in which it appears. To see the complete descriptions of the variables listed, increase the width of the dialog box by dragging one corner.
2. From the **Name** and **Description** lists, click the name of the analysis variable that you want. The selected variable is highlighted.
3. Click **Apply** to implement your changes. Abaqus/CAE recomputes the streamlines in the current viewport based on the selected stream output variable. Your selection for stream output variable is also saved for the duration of the session.
