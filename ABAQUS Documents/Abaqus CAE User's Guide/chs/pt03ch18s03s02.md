# 18.3.2 关于设计响应





优化的输入称为设计响应。 设计响应可以从Abaqus输出数据库文件读取； 例如，刚度、应力、固有频率和位移。 或者，Optimization模块可以从模型中节点的位置或单元的布局中提取设计响应； 例如，其重量、重心或惯性矩。

设计响应与模型的区域相关联； 但是，它由单个标量值组成，例如区域中的最大应力或模型总体积。 此外，设计响应可以与特定步骤或载荷工况相关联。 有关更多信息，请参阅["配置设计响应，" 第18.7节](pt03ch18s07.md)和["设计响应，" Abaqus Analysis User's Guide第13.2.1节](../usb/usb-link.md#usb-anl-aoptdesignresponses)。
