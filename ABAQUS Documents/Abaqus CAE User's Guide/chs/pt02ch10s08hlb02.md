# 10.9.1 将草图导出为ACIS、IGES或STEP格式文件





从主菜单栏选择**文件**![](../graphics/images/arrow.gif)**导出**![](../graphics/images/arrow.gif)**草图**，可以将当前草图导出为ACIS、IGES或STEP文件。许多计算机辅助建模和绘图应用程序可以读取和写入ACIS、IGES和STEP文件；因此，您可以在Abaqus/CAE和这些应用程序之间传输草图。

在草图编辑器中保存草图后，您可以在Abaqus/CAE会话期间随时导出草图

**将草图导出为ACIS、IGES或STEP格式文件：**

1. 从主菜单栏，选择**文件**![](../graphics/images/arrow.gif)**导出**![](../graphics/images/arrow.gif)**草图**。出现**导出草图**对话框。
2. 从**导出草图**对话框底部的**文件过滤器**菜单中，选择以下之一：- **ACIS SAT (*.sat)**- **IGES (*.igs)**- **STEP (*.stp)** Abaqus/CAE列出所选目录中所有具有相应文件扩展名的文件。
3. 选择要导出草图的文件，或在**文件名**文本字段中输入新文件的名称。
4. 从**导出草图**对话框中，单击**确定**。Abaqus/CAE关闭对话框并显示**选择草图**对话框，其中列出了所有已保存的草图。
5. 从**选择草图**对话框中，选择要导出的草图，然后单击**确定**。
6. 如果要将草图导出为IGES文件，Abaqus/CAE会显示**IGES风格**对话框。执行以下操作：1.选择用于写入IGES文件的以下风格之一：-**标准**-**AutoCAD**-**Solid Works**-**JAMA**2.单击**确定**。Abaqus/CAE调整IGES文件的内部表示以匹配所选应用程序期望的格式。默认情况下，Abaqus/CAE以**标准**格式导出草图。Abaqus/CAE将所有几何数据写入IGES文件中的单个图层。IGES文件仅包含几何数据，Abaqus/CAE不会从草图导出构造线和尺寸。
