# 60.103 UserMaterial 对象





UserMaterial 对象定义用于子程序 [`UMAT`](../sub/sub-link.md#sub-xsl-umat)、[`UMATHT`](../sub/sub-link.md#sub-xsl-umatht) 或 [`VUMAT`](../sub/sub-link.md#sub-xsl-vumat) 的材料常数。

**访问**

```
materialApi.materials()[*name*].userMaterial()
```

### 60.103.1 UserMaterial(...)

此方法创建一个 UserMaterial 对象。

**路径**

```
materialApi.materials()[*name*].UserMaterial
```

**原型**

```
odb_UserMaterial&
UserMaterial(const odb_String& type,
             bool unsymm,
             const odb_SequenceDouble& mechanicalConstants,
             const odb_SequenceDouble& thermalConstants);
```

**必需参数**

无。

**可选参数**

*type*

一个 odb_String，指定由命令定义的材料行为类型。可能的值为 "MECHANICAL"、"THERMAL" 和 "THERMOMECHANICAL"。默认值为 "MECHANICAL"。

*unsymm*

一个布尔值，指定材料刚度矩阵是否不对称，或者当使用热本构模型时，![](../graphics/ker_eqn00381.gif) 是否不对称。默认值为 false。

此参数仅对 Abaqus/Standard 分析有效。

*mechanicalConstants*

一个 odb_SequenceDouble，指定材料的机械常数。此参数仅在 *type*="MECHANICAL" 或 "THERMOMECHANICAL" 时有效。默认值为空序列。

*thermalConstants*

一个 odb_SequenceDouble，指定材料的热常数。此参数仅在 *type*="THERMAL" 或 "THERMOMECHANICAL" 时有效。默认值为空序列。

**返回值**

一个 UserMaterial 对象。

**异常**

RangeError。
