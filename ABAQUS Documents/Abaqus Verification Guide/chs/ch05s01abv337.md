# 5.1.21 释放旋转自由度





**产品：**Abaqus/Standard  

### 测试单元

B21    B22    B23    B31    B32    B33

### 测试功能

通过释放一个或多个旋转自由度来测试各种类型的铰接连接。包含使用多点约束的等效模型进行比较。

### 问题描述

![](../graphics/verrelease-model.png)

两个梁单元与*x*轴对齐，在中心连接，在节点1和4处被夹紧。旋转自由度在中心节点2处被释放。等效的MPC定义用于连接中心两个独立的节点，节点2和3。

**载荷：**

步骤1：模型的左半部分承受每单位长度的力，PY = 1000和PZ = 1000。

模型的右半部分承受每单位长度的力，PY = 1000和PZ = 1000。

步骤2：再次施加第一步中施加的载荷，但使用NLGEOM进行大位移分析。

### 结果与讨论

等效多点约束模型的结果相同。

### 输入文件

[xreleasepinx2.inp](../eif/xreleasepinx2.inp)

静态步骤，释放在平面中梁的所有旋转自由度。

[xmpcpinx2.inp](../eif/xmpcpinx2.inp)

静态步骤，平面中梁的等效MPC类型PIN。

[xreleasepinx3.inp](../eif/xreleasepinx3.inp)

静态步骤，释放在空间中梁的所有旋转自由度。

[xmpcpinx3.inp](../eif/xmpcpinx3.inp)

静态步骤，空间中梁的等效MPC类型PIN。

[xreleaserevo2.inp](../eif/xreleaserevo2.inp)

静态步骤，释放在空间中梁的一个旋转自由度。

[xmpcrevo2.inp](../eif/xmpcrevo2.inp)

静态步骤，空间中梁的等效MPC类型REVOLUTE加MPC类型PIN。

[xreleaseuniv2.inp](../eif/xreleaseuniv2.inp)

静态步骤，释放在空间中梁的两个旋转自由度。

[xmpcuniv2.inp](../eif/xmpcuniv2.inp)

静态步骤，空间中梁的等效MPC类型UNIVERSAL加MPC类型PIN。

[xreleaseuniv3.inp](../eif/xreleaseuniv3.inp)

频率、稳态动力学、模态动力和响应谱步骤；释放在空间中梁的两个旋转自由度。

[xmpcuniv3.inp](../eif/xmpcuniv3.inp)

频率、稳态动力学、模态动力和响应谱步骤；空间中梁的等效MPC类型UNIVERSAL加MPC类型PIN。
