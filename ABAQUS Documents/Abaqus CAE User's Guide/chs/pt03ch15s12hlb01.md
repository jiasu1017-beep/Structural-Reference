# 15.12.1 创建相互作用





当您创建相互作用时，必须指定相互作用的名称、激活相互作用的步骤、相互作用的类型，以及要应用相互作用的装配区域。区域选择不适用于通用接触相互作用。可用的相互作用类型取决于为该步骤选择的过程。例如，您只能在热传递、耦合温度-位移或耦合热电步骤中在表面上定义热通量。类似地，您只能在初始步骤中定义具有用户定义致动器/传感器的相互作用。

如果您正在创建表面对表面接触相互作用，您可以使用接触检测工具使以下过程中的许多步骤自动化。更多信息，请参见「使用接触和约束检测」，第15.16节。

**创建相互作用的步骤：**

1. 从主菜单栏选择 ****Interaction**![](../graphics/images/arrow.gif)**Create****。此时会出现「Create Interaction」对话框，「Name」文本字段中显示默认名称。**提示：**您也可以使用「相互作用」模块工具箱中的![](../graphics/ico_interaction.png)工具创建相互作用。
2. 为相互作用输入名称。有关命名对象的更多信息，请参见「使用基本对话框组件」，第3.2.1节。
3. 选择激活相互作用的步骤。点击「Step」文本字段旁边的箭头，并从出现的列表中选择。 「Types for Selected Steps」列表会变为所有可用相互作用类型的列表。
4. 从「Types for Selected Steps」列表中选择相互作用类型，然后点击「Continue」。
5. 如果您要创建除通用接触之外的相互作用，请使用以下方法之一选择要应用相互作用的区域：- 在视口中选择一个区域。您可以使用角度方法从几何中选择一组面或边，或从单元面中选择一组。更多信息，请参见「使用角度和特征边方法选择多个对象」，第6.2.3节。完成选择后，点击鼠标按钮2。**提示：**您可以通过在「Selection」工具栏中指定过滤选项来限制可以在视口中选择的对象类型。请参见「使用选择选项」，第6.3节，以获取更多信息。如果模型包含网格和几何的组合，请从提示区域点击以下之一：- 点击「Geometry」将相互作用应用于几何区域或参考点。- 点击「Mesh」将相互作用应用于本机或孤儿网格选择。默认情况下，对于大多数相互作用，会创建一个包含所选对象的集合或表面。您可以通过关闭提示区域中创建集合或表面的选项来更改此行为。提示区域中提供了默认名称，但您可以输入新名称。- 要从现有集合或表面列表中进行选择，请执行以下操作：1. 点击提示区域右侧的「Sets」或「Surfaces」。（按钮的名称取决于您正在创建的对象类型。例如，如果您正在创建表面对表面接触相互作用，则会出现「Surfaces」按钮。）Abaqus/CAE会显示包含可用集合或表面列表的「Region Selection」对话框。2. 选择您感兴趣的集合或表面，然后点击「Continue」。**注意：**默认选择方法基于您最近使用的选择方法。如需恢复为另一种方法，请点击提示区域右侧的「Select in Viewport」或「Sets」或「Surfaces」。此时会出现相互作用编辑器。您正在应用相互作用的区域会在视口中高亮显示。
6. 如果您要创建通用接触相互作用，则会出现相互作用编辑器。
7. 输入定义相互作用所需的所有数据，然后点击「OK」。有关编辑器特定功能的详细信息，请从主菜单栏选择 ****Help**![](../graphics/images/arrow.gif)**On Context****，然后点击感兴趣的功能，或参见「使用相互作用编辑器」，第15.13节。符号会出现在视口中，表示您刚刚创建的相互作用。更多信息，请参见「理解表示相互作用、约束和连接器的符号」，第15.10节。
