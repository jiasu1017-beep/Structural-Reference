# 70.6.3 使用基准平面方法分区面







您可以通过用基准平面切割所选面来分区面（如下图所示）：

![](../graphics/par-face-datum-nls.png)

**使用基准平面方法分区面：**

1. 从主菜单栏中选择 ****Tools**![](../graphics/images/arrow.gif)**Partition****（工具→分区）。出现 **Create Partition**（创建分区）对话框。Abaqus/CAE 在提示区域显示提示以指导您完成过程。**提示：**您也可以使用位于模块工具箱分区面工具处的 ![](../graphics/ico_ptnFaceDatumPlane.png) 工具来使用基准平面方法分区面。 有关工具箱中分区工具的图表，请参阅["使用分区工具集，" 第 70.2 节"](pt06ch70s02.md)。
2. 从对话框顶部的 **Type**（类型）单选按钮中选择 **Face**（面）。**Method**（方法）列表显示可用于分区面的方法。
3. 从方法列表中选择 **Use datum plane**（使用基准平面）。
4. 如果部件或装配包含多个面，选择要分区的面。您可以使用拖动选择、**[Shift]****+点击**、**[Ctrl]****+点击** 和角度方法的组合来选择多个要分区的面。 有关更多信息，请参阅["在当前视口中选择对象，" 第 6.2 节"](pt01ch06s02.md)。**提示：**如果您无法选择所需的面，可以使用 **Selection**（选择）工具栏更改选择行为。 有关更多信息，请参阅["使用选择选项，" 第 6.3 节"](pt01ch06s03.md)。Abaqus/CAE 高亮显示所选面。
5. 选择一个现有基准平面。如果所需位置不存在基准平面，请使用基准工具集创建基准。Abaqus/CAE 高亮显示所选基准平面。
6. 在提示区域中点击 **Create Partition**（创建分区）。Abaqus/CAE 创建分区。
