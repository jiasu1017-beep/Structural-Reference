# 12.5.3 模型的循环对称分析

**产品：** Abaqus/Standard 和 Abaqus/Explicit

##### **参考资料**

- ["循环对称分析，" 第 3.5 节](../stm/stm-link.md#stm-anl-cyclic-symmetry)
- [*CYCLIC SYMMETRY](../key/key-link.md#usb-kws-hcyclicsymmetry)

### 概述

循环对称分析：
- 利用循环对称性减少分析规模
- 只需建模完整结构的一个扇区
- 适用于具有重复模式旋转对称的结构
- 可用于静力和动力分析

### 循环对称性定义

循环对称结构由围绕中心轴重复的多个扇区组成：

| **参数** | **描述** |
| --- | --- |
| 扇区角 | 每个重复扇区的角度 |
| 重复数 | 完整圆周中的扇区数 |
| 对称轴 | 旋转对称的中心轴 |

| **输入文件用法：** | ``` [*CYCLIC SYMMETRY](../key/key-link.md#usb-kws-hcyclicsymmetry), MODEL=SEGMENT, SECTOR ANGLE=*angle* ``` |

### 扇区建模

只需建模完整结构的一个扇区：

| **输入文件用法：** | ``` [*CYCLIC SYMMETRY](../key/key-link.md#usb-kws-hcyclicsymmetry), MODEL=SEGMENT ``` |

### 对称边界条件

扇区边界上需要定义循环对称边界条件：

| **输入文件用法：** | ``` [*BOUNDARY](../key/key-link.md#usb-kws-hboundary) *节点集*, 11, 11, 0.0 *节点集*, 22, 22, 0.0 ``` |

### 模式数

指定要提取的循环对称模式数：

| **输入文件用法：** | ``` [*CYCLIC SYMMETRY](../key/key-link.md#usb-kws-hcyclicsymmetry), MODEL=SEGMENT, HARMONIC MODES=*n* ``` |

### 循环对称模态分析

可以执行循环对称模态分析：

| **输入文件用法：** | ``` [*BUCKLE](../key/key-link.md#usb-kws-hbuckle) [*CYCLIC SYMMETRY](../key/key-link.md#usb-kws-hcyclicsymmetry), MODEL=SEGMENT ``` |

### 动力分析

循环对称结构可以执行动力分析：

| **分析类型** | **描述** |
| --- | --- |
| 频率提取 | 提取循环对称固有频率 |
| 瞬态响应 | 计算循环对称瞬态响应 |
| 响应谱 | 执行响应谱分析 |

### 结果解释

循环对称分析的结果解释：

| **结果类型** | **描述** |
| --- | --- |
| 单扇区结果 | 显示一个扇区的结果 |
| 完整结构 | 通过复制扇区结果获得 |
| 谐波分量 | 各谐波模式的贡献 |

### 应用

循环对称分析常用于：

1. **涡轮机械**：叶片轮盘组件
2. **齿轮**：齿轮和轴承
3. **法兰**：管道和容器法兰
4. **对称结构**：任何具有旋转重复模式的结构

### 优势

| **优势** | **描述** |
| --- | --- |
| 模型缩小 | 显著减少建模工作量 |
| 计算效率 | 分析规模大幅减少 |
| 精度保持 | 保留完整结构精度 |
| 参数研究 | 便于修改扇区参数 |

### 局限性

| **局限性** | **描述** |
| --- | --- |
| 几何要求 | 结构必须具有循环对称性 |
| 载荷要求 | 载荷必须与对称性一致 |
| 边界条件 | 扇区边界条件必须正确 |
| 接触问题 | 接触定义可能复杂 |

### 输入文件模板

```
*-heading
Cyclic symmetry analysis
**
*CYLINDRICAL SECTION, POINT=1, NORMAL=2
*CYCLIC SYMMETRY, MODEL=SEGMENT, SECTOR ANGLE=30
*NODE, NSET=ROOT
*ELEMENT, TYPE=C3D8, ELSET=SECTOR
*BOUNDARY
ROOT, ENCASTRE
*BOUNDARY
SYMMETRY_I, 11, 11, 0.0
SYMMETRY_J, 22, 22, 0.0
*STEP
*STATIC
*CLOAD
*END STEP
**
*STEP
*BUCKLE
*CYCLIC SYMMETRY, HARMONIC MODES=5
*END STEP
```
