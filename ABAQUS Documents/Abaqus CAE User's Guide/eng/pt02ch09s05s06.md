# 9.5.6 Customizing your Abaqus/CAE environment







You use the Abaqus environment file (`abaqus_v6.env`) to specify parameters that control Abaqus/Standard and Abaqus/Explicit. In addition, you can use the environment file to specify a set of commands that are executed when you start an Abaqus/CAE session. Examples of commands that configure how you want a job to run on a remote host computer are given in ["Submitting a job remotely," Section 19.2.7](pt03ch19s02s06.md).
