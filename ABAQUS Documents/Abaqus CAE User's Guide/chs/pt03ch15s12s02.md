# 15.12.14 在相互作用模块中使用Special菜单





您可以使用「相互作用」模块中的「Special」菜单来定义以下工程特征：
- **惯性**。您可以在装配中的点上定义集中质量、旋转惯性和热容。在Abaqus/Standard分析中，您还可以定义质量和惯性比例阻尼以及复合阻尼。更多信息，请参见第33章，「惯性」。
- **裂纹**。您可以使用以下技术研究裂纹的萌生和扩展：- 具有重复重叠节点的嵌入裂缝- 轮廓积分分析- 扩展有限元方法（XFEM）- 虚拟裂缝闭合技术（VCCT）此外，您可以使用裂纹在Abaqus/CFD分析中对零厚度壳进行建模，以使用壳/膜进行流固耦合分析。更多信息，请参见「建模裂纹和裂缝」。
- **弹簧/阻尼器**。您可以定义表现出相同线性行为的弹簧和阻尼器，该行为与场变量无关。您也可以在同一组点上同时定义弹簧和阻尼器行为。在Abaqus/Explicit或Abaqus/Standard分析中，您可以建模沿两点之间的作用线连接两点的弹簧和阻尼器。在Abaqus/Standard分析中，您还可以建模沿固定方向连接两点的弹簧和阻尼器，或连接点到地面的弹簧和阻尼器。更多信息，请参见第37章，「弹簧和阻尼器」。
- **紧固件**。您可以使用基于点或离散的紧固件对两个或多个面之间的点对点连接进行建模。基于点的紧固件可以使用附件点、参考点或孤儿节点来定义。离散紧固件可以使用附件线来定义。更多信息，请参见第29章，「紧固件」。

### 建模裂纹和裂缝

在对裂纹进行建模时，您可以将裂缝分配给模型的区域。Abaqus/CAE在生成网格时会在裂缝上放置重叠的重复节点。裂缝不能沿部件边界延伸，必须嵌入二维部件的面内或实体部件的单元内。因为裂缝会修改网格，所以您不能在从属部件实例上创建裂缝。

对于断裂力学，裂缝定义了可以沿分析分离的具有重叠节点的边缘或面。您可以在模型中包含裂缝裂纹。或者，您可以在创建轮廓积分时引用裂缝；但是，您不能将裂缝裂纹与扩展有限元方法（XFEM）一起使用。更多信息，请参见第31章，「断裂力学」。

对于涉及壳/膜流固耦合的Abaqus/CFD分析，裂缝在网格中定义零厚度壳。您可以创建表示裂缝的双侧表面，并将该表面选为流固耦合相互作用边界的区域。更多信息，请参见「定义流固耦合仿真相互作用」，第15.13.15节。

**分配裂缝的步骤：**

1. 在「相互作用」模块中，从主菜单栏选择 ****Special**![](../graphics/images/arrow.gif)**Crack**![](../graphics/images/arrow.gif)**Assign seam****。
2. 从视口中的模型中，选择表示裂缝的实体。实体必须是嵌入在二维部件面内的边缘或嵌入在实体部件单元内的面；您不能选择位于部件边界上的任何实体。
3. 点击鼠标按钮2表示您已完成选择裂缝。Abaqus/CAE会创建裂缝。
