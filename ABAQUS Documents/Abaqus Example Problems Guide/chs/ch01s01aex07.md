# 1.1.7 循环载荷下的缺口梁





**产品：** Abaqus/Standard   

本示例说明了使用非线性各向同性/运动硬化材料模型来模拟循环载荷下缺口梁的响应。该模型具有两个在循环载荷条件下模拟塑性硬化的特征：屈服面中心在应力空间中移动（运动硬化行为），以及屈服面大小随非弹性变形而发展（各向同性硬化行为）。引入运动和各向同性硬化分量的这种组合是为了模拟包辛格效应以及其他现象，如塑性安定、棘轮效应和平均应力松弛。

本示例中研究的部件是承受循环四点弯曲载荷的缺口梁。结果与 Benallal 等（1988）和 Doghri（1993）发表的有限元结果进行比较。没有实验数据。

### 几何和模型

几何和网格如[图 1.1.7-1](ch01s01aex07.md#sxmnotched-meshinit)所示。[图 1.1.7-2](ch01s01aex07.md#sxmnotched-magview) 显示了缺口附近离散化的细节，这是本分析中感兴趣的 region。由于几何和载荷相对于 *x* = 0 平面对称，因此只建模了梁的一半。所有尺寸均以毫米给出。梁厚 1 mm，用平面应变、二阶、减缩积分单元（类型 CPE8R）建模。网格选择得与 Doghri（1993）使用的网格相似。没有进行网格收敛研究。

### 载荷和边界条件

梁承受四点弯曲载荷。由于只建模了梁的一半，模型在距对称平面 26 mm 处包含一个集中载荷（见[图 1.1.7-1](ch01s01aex07.md#sxmnotched-meshinit)）。枢轴点距对称平面 42 mm。模拟运行 3.5 个周期，历时 7 个时间单位。在每个周期中，载荷从零线性增加至 675 N，然后回到零。使用幅值曲线来描述加载和卸载。增量大小限制在最大 0.125 以迫使 Abaqus 紧密跟随规定的加载/卸载模式。

### 结果和讨论

[图 1.1.7-3](ch01s01aex07.md#sxmnotched-meshdef) 显示了在 3.5 个加载周期后梁的最终变形形状；梁上的最终载荷为 675 N。

变形在缺口根部附近最为严重。[图 1.1.7-4](ch01s01aex07.md#sxmnotched-stressvstrain) 和[图 1.1.7-5](ch01s01aex07.md#sxmnotched-backstress)中报告的结果是在该区域（单元 166，积分点 3）测量的。[图 1.1.7-4](ch01s01aex07.md#sxmnotched-stressvstrain) 显示了应力与应变的时间演化。使用此材料模型预测了几个重要效应。首先，在第一次卸载时屈服的开始发生在比第一次加载时更低的绝对应力水平，这是包辛格效应。其次，应力-应变循环趋于移动和稳定，因此平均应力从周期到周期减小，趋向于零。这种行为称为平均应力松弛，在非对称应变值之间规定应变的单轴循环测试中最明显。第三，屈服面随循环沿应变轴移动，而应力-应变曲线的形状在从一个周期到下一个周期时趋于保持相似。这种行为称为棘轮效应，在非对称应力值之间规定应力的单轴循环测试中最明显。最后，与其他循环的硬化曲线相比，第一次半循环的硬化行为非常平坦，这是加工硬化金属的典型特征，其初始硬化状态是由轧制等成型过程引起的大单调塑性变形的结 果。低硬化模量是背应力初始条件的结果，这使屈服面中心位于距应力空间原点 ![](../graphics/exa_eqn00094.gif) 228 MPa 的距离。由于这个距离接近最大可能距离（257 MPa），第一次循环的大部分硬化是各向同性的。

这些现象主要通过背应力的非线性演化在本示例中建模，因为各向同性硬化速率非常小。可以通过在整个分析中保持弹性域固定的分析来验证此行为。

[图 1.1.7-5](ch01s01aex07.md#sxmnotched-backstress) 显示了背应力张量偏斜部分直接分量的演化。背应力分量在第一次循环中演化最大，因为包辛格效应克服了初始硬化配置。只显示背应力的偏斜分量，以便将从 Abaqus 获得的结果与 Doghri（1993）报告的结果进行比较。由于 Abaqus 使用 Ziegler 演化定律的扩展，产生具有非零压力的背应力张量，而 Doghri 使用的定律（线性 Prager 定律的扩展）产生的背应力张量是偏斜的。由于塑性模型只考虑背应力的偏斜部分，因此这种定律差异不影响其他解变量。

[图 1.1.7-4](ch01s01aex07.md#sxmnotched-stressvstrain) 和[图 1.1.7-5](ch01s01aex07.md#sxmnotched-backstress) 中显示的结果与 Doghri（1993）报告的结果非常一致。
