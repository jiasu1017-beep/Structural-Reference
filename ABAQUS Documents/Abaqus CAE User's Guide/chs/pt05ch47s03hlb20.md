### 47.4.22 对X–Y数据对象应用三角函数

使用三角函数 **cos**、**cosh**、**acos**、**sin**、**sinh**、**asin**、**tan**、**tanh** 和 **atan** 对[之前保存的](pt05ch47s02hlb07.md) *X–Y* 数据对象（有序对的集合）进行运算，生成一个新的 *X–Y* 数据对象。新的 *X–Y* 数据对象具有与原始 *X–Y* 数据对象相同的 *X* 坐标值，但新的 *Y* 坐标计算为每个原始 *Y* 坐标的三角函数值。三角函数的参数假定以弧度为单位。

**对X–Y数据对象应用三角函数：**

1. 找到 **Operate on XY Data** 对话框。从主菜单栏，选择 ****Tools**![](../graphics/images/arrow.gif)**XY Data**![](../graphics/images/arrow.gif)**Create****. 在出现的对话框中点击 **Operate on XY data**；然后点击 **Continue**。**Operate on XY Data** 对话框出现。
2. 从列出的 **Operators** 中，点击以下三角函数之一：| **sin(A)** | 计算 *X–Y* 数据对象的正弦。 | | --- | --- | | **sinh(A)** | 计算 *X–Y* 数据对象的双曲正弦。 | | **asin(A)** | 计算 *X–Y* 数据对象的反正弦。您的 *Y* 数据值必须在 [--1, 1] 范围内。 | | **cos(A)** | 计算 *X–Y* 数据对象的余弦。 | | **cosh(A)** | 计算 *X–Y* 数据对象的双曲余弦。 | | **acos(A)** | 计算 *X–Y* 数据对象的反余弦。您的 *Y* 数据值必须在 [--1, 1] 范围内。 | | **tan(A)** | 计算 *X–Y* 数据对象的正切。 | | **tanh(A)** | 计算 *X–Y* 数据对象的双曲正切。 | | **atan(A)** | 计算 *X–Y* 数据对象的反正切。 | 您选择的三角函数出现在表达式窗口中。
3. 从 **XY Data** 选项中，点击要操作的 *X–Y* 数据对象的名称，然后点击 **Add to Expression**。您可以从本次会话中之前保存的所有 *X–Y* 数据对象中选择（在 **XY Data** 字段中按字母顺序列出）。*X–Y* 数据对象名称出现在表达式窗口中三角函数括号的内部。
4. 要继续构建表达式，将光标定位在表达式窗口中，然后输入或选择要包含的函数、运算符和 *X–Y* 数据。
5. 要评估并显示表达式，点击 **Plot Expression**。
6. 要[保存](pt05ch47s02hlb07.md)您的新 *X–Y* 数据对象，点击 **Save As**，然后在出现的对话框中提供名称。保存数据对象使其可用于本次会话中的未来操作以及包含多个数据对象的 *X–Y* 曲线图。
7. 完成后，点击 **Cancel** 关闭对话框。
