# 12.13.12 创建桁架截面





桁架截面用于定义二维和三维中提供轴向强度但没有弯曲刚度的细长杆状结构的截面属性。请参见[Abaqus Analysis User's Guide的第29.2.1节，"桁架元素"](../usb/usb-link.md#usb-elm-etruss)。

**指定桁架截面属性：**

1. 从主菜单栏，选择****Section**![](../graphics/images/arrow.gif)**Create****。出现**Create Section**对话框。
   **提示：**您也可以在**Section Manager**中点击**Create**，或在Property模块工具箱中选择创建截面工具![](../graphics/ico_section.png)。
2. 输入截面名称。有关命名对象的更多信息，请参见["使用基本对话框组件，"第3.2.1节"](pt01ch03s02s01.md)。
3. 选择**Beam**作为截面**Category**，选择**Truss**作为截面**Type**，然后点击**Continue**。出现桁架截面编辑器。
4. 为桁架截面选择一种材料。如有需要，点击![](../graphics/ico_material.png)创建材料；有关更多信息，请参见["创建或编辑材料，"第12.7.1节"](pt03ch12s07hlb01.md)。
5. 为**Cross-sectional area**输入值。
6. 点击**OK**保存更改并关闭桁架截面编辑器。
