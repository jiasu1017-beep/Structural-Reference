# 62.5.3 An overview of the methods for creating a datum plane







When you choose **Plane** from the **Create Datum** dialog box, the **Method** list displays the following methods for creating a datum plane:

**![](../graphics/ico_dtmPlaneOffsetPrincipal.png) Offset from principal plane**

Select one of the three principal planes; and provide the location of the datum plane in the form of an offset from the selected plane, as shown in [Figure 62--22](pt06ch62s05s03.md#dtm-hpopp). A positive value indicates an offset in the positive direction along the axis normal to the selected plane; for example, along the *X*-axis normal to the *Y*–*Z* plane. For detailed instructions, see ["Creating a datum plane offset from a principal plane," Section 62.8.1](pt06ch62s08hlb01.md).

**Figure 62–22** Creating a datum plane by offsetting from one of the three principal planes.

![](../graphics/dtm-hpopp-nls.png)

**![](../graphics/ico_dtmPlaneOffsetPlane.png) Offset from plane**

Select any plane on the model; and provide the location of the datum plane by specifying the direction of the normal and an offset from the selected plane along the normal, as shown in [Figure 62--23](pt06ch62s05s03.md#dtm-hpop). You can specify the offset by entering a value or selecting a point. For detailed instructions, see ["Creating a datum plane at an offset from a selected plane," Section 62.8.2](pt06ch62s08hlb02.md).

**Figure 62–23** Creating a datum plane by offsetting from any plane.

![](../graphics/dtm-hpop-nls.png)

**![](../graphics/ico_dtmPlaneThreePoints.png) 3 points **

Select three points through which the datum plane must pass, as shown in [Figure 62--24](pt06ch62s05s03.md#dtm-hp3pts). For detailed instructions, see ["Creating a datum plane passing through three points," Section 62.8.3](pt06ch62s08hlb03.md).

**Figure 62–24** Creating a datum plane by selecting three points.

![](../graphics/dtm-hp3pts-nls.png)

**![](../graphics/ico_dtmPlaneLinePoint.png) Line and point**

Select an edge and a point through which the datum plane must pass, as shown in [Figure 62--25](pt06ch62s05s03.md#dtm-hplp). For detailed instructions, see ["Creating a datum plane through a line and a point," Section 62.8.4](pt06ch62s08hlb04.md).

**Figure 62–25** Creating a datum plane by selecting a point and an edge.

![](../graphics/dtm-hplp-nls.png)

**![](../graphics/ico_dtmPlanePointNormal.png) Point and normal**

Select a point and an edge; the datum plane passes through the point and is normal to the selected edge, as shown in [Figure 62--26](pt06ch62s05s03.md#dtm-hppn). For detailed instructions, see ["Creating a datum plane passing through a point and normal to an edge," Section 62.8.5](pt06ch62s08hlb05.md).

**Figure 62–26** Creating a datum plane by selecting a point and a normal edge.

![](../graphics/dtm-hppn-nls.png)

**![](../graphics/ico_dtmPlaneMidpoint.png) Midway between 2 points**

Select two points. Abaqus/CAE creates a datum plane midway between the two selected points and normal to the line connecting them, as shown in [Figure 62--27](pt06ch62s05s03.md#dtm-hpmid2). For detailed instructions, see ["Create a datum plane midway between two points and normal to the line connecting the two points," Section 62.8.6](pt06ch62s08hlb06.md).

**Figure 62–27** Positioning a datum plane midway between two points.

![](../graphics/dtm-hpmid2-nls.png)

**![](../graphics/ico_dtmPlaneRotatePlane.png) Rotate from plane**

Select a face and an axis of rotation, and specify the angle through which the face will be rotated. Abaqus/CAE creates a datum plane by rotating the face about the axis through the specified angle, as shown in [Figure 62--28](pt06ch62s05s03.md#dtm-hprot). For detailed instructions, see ["Creating a datum plane by rotating an existing face through a specified angle," Section 62.8.7](pt06ch62s08hlb07.md).

**Figure 62–28** Defining a datum plane by rotating a face through a specified angle.

![](../graphics/dtm-hprot-nls.png)
