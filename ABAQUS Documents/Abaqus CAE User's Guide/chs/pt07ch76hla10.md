# 76.11 控制基准点显示






可以使用 ****视图**![](../graphics/images/arrow.gif)**零件显示选项**** 和 ****视图**![](../graphics/images/arrow.gif)**装配显示选项**** 菜单项来控制当前视口中零件或装配上基准点的显示。关闭基准点显示有助于生成干净的零件或装配打印图像。即使您选择不显示基准点，它们仍然是零件或装配的特征要素。有关基准点的更多信息，请参见["基准点，" 第11.8.1节](pt03ch11s08s01.md)。

**控制基准点显示：**

1. 找到 **基准** 显示选项。从主菜单栏，选择 ****视图**![](../graphics/images/arrow.gif)**零件显示选项**** 或 ****视图**![](../graphics/images/arrow.gif)**装配显示选项****。在出现的对话框中点击 **基准** 选项卡。
2. 切换相应的按钮来控制： - 基准点标签 - 基准点符号
3. 点击 **确定** 实施更改并关闭对话框。更改仅适用于当前视口，并在会话期间保存。
