# 43.2 理解变形形状绘图





变形绘图根据节点变量（如位移）的值显示模型的形状。您可以选择Abaqus/CAE显示结果的节点变量（称为变形场输出变量），以及这些结果的步骤和帧。

如果您不选择变形场输出变量，Abaqus/CAE会尝试选择默认值。Abaqus/Standard和Abaqus/Explicit中的大多数程序默认将位移写入输出数据库；在这些情况下，Abaqus/CAE选择位移作为用于默认变形变量的节点向量量。一些程序（例如热传递）默认不将位移写入输出数据库；因此，Abaqus/CAE不选择默认变形变量。如果输出数据库不包含任何可用于计算变形形状的变量，Abaqus/CAE无法显示变形绘图。

在从输出数据库获取生成变形绘图所需的信息后，Abaqus/CAE通过根据以下内容调整每个节点的坐标来计算变形模型的形状：
- 变形场输出变量（请参见"选择变形场输出变量"，第42.5.4节）。
- 分析步骤和帧（请参见"选择特定结果步骤和帧"，第42.3.1节）。
- 均匀或非均匀变形比例因子（请参见"缩放变形"，第55.4.1节）。
- 应用于变形场输出变量的任何用户定义的刚体变换（请参见"将结果转换到新坐标系"，第42.6.8节）。

与未变形绘图形状一样，您可以使用不依赖于绘图状态的选项来自定义变形形状绘图。选择主菜单栏中的**选项**![](../graphics/images/arrow.gif)**通用**来选择不依赖于绘图状态的自定义选项。

您也可以使用Abaqus/CAE将您的未变形和变形模型形状组合成单个绘图。将形状组合可以提供评估变形的上下文。如果您选择叠加未变形和变形模型形状，可以使用**叠加**绘图选项来单独控制未变形形状的显示以区别于变形形状。

**提示：**如果您想显示一个或两个形状而不带云图、符号或材料方向，请使用**允许多个绘图状态**![](../graphics/ico_plotMultipleStates.png)工具来显示两种模型形状。您也可以使用此工具从可视化模块工具箱中的工具选择任何组合的绘图状态（更多信息，请参见"显示多个绘图状态"，第55.6节）。
