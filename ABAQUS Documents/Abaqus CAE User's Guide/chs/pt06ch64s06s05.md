# 64.6.5 折叠四边形或三角形单元的边缘





从主菜单栏选择 ****网格**![](../graphics/images/arrow.gif)**编辑**** 可以折叠三角形或四边形单元的所选边缘。实际上，您是将单元边缘两端的两个节点替换为单个节点。您可以在两种定位单个节点的方法之间进行选择：

**方向**

选择 **方向** 以沿指定方向折叠单元边缘。从提示区域的按钮中，选择 **翻转** 以更改 Abaqus/CAE 折叠单元的方向。以下图形显示了 **方向** 方法如何折叠单元边缘以及更改方向的效果：

![](../graphics/edt-meshedit-element-direction-nls.png)

**平均**

选择 **平均** 以折叠单元边缘并调整剩余单元边缘，使其汇聚到折叠边缘的中点。以下图形显示了 **平均** 方法如何折叠单元边缘：

![](../graphics/edt-meshedit-element-average-nls.png)

**折叠四边形或三角形单元边缘的步骤：**

1. 进入"网格"模块。
2. 从上下文栏的 **对象** 字段中选择 **零件** 并从列表中选择一个零件。
3. 从主菜单栏选择 ****网格**![](../graphics/images/arrow.gif)**编辑****。Abaqus/CAE 显示 **编辑网格** 对话框。 **提示：** 您也可以使用位于"网格"模块工具箱底部的 ![](../graphics/ico_meshEdit.png) 工具显示 **编辑网格** 对话框。
4. 在对话框中执行以下操作： 1. 在 **类别** 字段中选择 **单元**。 2. 从 **方法** 列表中选择 **折叠边缘（三角/四边）**。
5. 从提示区域的菜单中，选择 Abaqus/CAE 用于折叠边缘的方法。 - 选择 **方向** 以沿指定方向折叠单元边缘。 - 选择 **平均** 以折叠单元边缘并调整剩余单元，使其汇聚到折叠边缘的中点。
6. 选择要折叠的单元边缘。您只能选择一个边缘。Abaqus/CAE 会高亮显示所选边缘，并提示您确认要折叠所选边缘。
7. 从提示区域的按钮中，点击 **是** 以折叠所选边缘。Abaqus/CAE 会提示您选择下一个要折叠的边缘。
8. 根据需要重复上述步骤以折叠其他边缘。
9. 完成边缘折叠后，点击鼠标按钮2或提示区域中的取消按钮 ![](../graphics/afxI_cancel.png) 退出过程。 **提示：** 如果在折叠边缘时出错，点击 **编辑网格** 对话框中的 **撤销** 可以撤消更改。Abaqus/CAE 会将网格恢复到操作之前的状态。
