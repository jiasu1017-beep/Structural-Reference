# 1.8.3 压电材料的稳态动态分析

**产品：** Abaqus/Standard

### 问题描述

模型是["压电材料的静态分析"，《Abaqus 验证指南》第 3.7.1 节](../ver/ver-link.md#ver-prc-piezostatic)中描述的圆柱体。有九个输入文件。输入文件 [ppzossd1.inp](../eif/ppzossd1.inp)、[ppzossd3.inp](../eif/ppzossd3.inp)、[ppzossd4.inp](../eif/ppzossd4.inp) 和 [ppzossd4a.inp](../eif/ppzossd4a.inp) 有 16 个 CAX4E 单元。输入文件 [ppzossd2.inp](../eif/ppzossd2.inp) 有 4 个 CAX8E 单元。输入文件 [ppzossd7.inp](../eif/ppzossd7.inp)、[ppzossd8.inp](../eif/ppzossd8.inp) 和 [ppzossd9.inp](../eif/ppzossd9.inp) 分别有 32 个 CAX3E、8 个 CAX6E 和 4 个 CAX8RE 单元。

基于模态的稳态动态分析使用在["压电材料的频率提取分析"，《Abaqus 验证指南》第 3.7.2 节](../ver/ver-link.md#ver-prc-piezofreqextract)中生成的重新启动文件中的特征数据。输入文件 [ppzossd1.inp](../eif/ppzossd1.inp) 和 [ppzossd2.inp](../eif/ppzossd2.inp) 说明了无阻尼的稳态动态分析，旨在与 Mercer 等人（1987）的结果进行比较。在该分析中，压力载荷施加在顶面上。在输入文件 [ppzossd3.inp](../eif/ppzossd3.inp)、[ppzossd7.inp](../eif/ppzossd7.inp)、[ppzossd8.inp](../eif/ppzossd8.inp) 和 [ppzossd9.inp](../eif/ppzossd9.inp) 中，向前述稳态动态分析引入模态阻尼项以说明阻尼的影响。在输入文件 [ppzossd1.inp](../eif/ppzossd1.inp)、[ppzossd2.inp](../eif/ppzossd2.inp)、[ppzossd3.inp](../eif/ppzossd3.inp)、[ppzossd7.inp](../eif/ppzossd7.inp)、[ppzossd8.inp](../eif/ppzossd8.inp) 和 [ppzossd9.inp](../eif/ppzossd9.inp) 中执行了模态和直接计算稳态分析。输入文件 [ppzossd4.inp](../eif/ppzossd4.inp) 说明了带分布电荷的稳态分析，而输入文件 [ppzossd4a.inp](../eif/ppzossd4a.inp) 执行带集中电荷的稳态分析代替压力载荷。仅使用直接计算选项，因为基于模态的过程无法将电荷载荷充分转换为模态载荷。

除了基于模态和直接解分析外，还在输入文件 [ppzossd1.inp](../eif/ppzossd1.inp)、[ppzossd3.inp](../eif/ppzossd3.inp)、[ppzossd7.inp](../eif/ppzossd7.inp)、[ppzossd8.inp](../eif/ppzossd8.inp) 和 [ppzossd9.inp](../eif/ppzossd9.inp) 中执行了基于子空间的稳态动态分析。额外的频率步骤提取所有可用的特征模态，然后在基于子空间的稳态动态步骤中使用这些模态来计算响应。由于使用了所有特征模态，结果与直接解分析获得的结果相同。

对于所有这些分析，选择 100000 rad/sec（15.9 kHz）的单个正弦频率以与["压电材料的模态动态分析"第 1.8.2 节](ch01s08ach64.md)中的模态动态结果进行比较。

### 结果与讨论

在稳态动态过程中，正弦载荷的频率由用户定义，对于达到稳态时发生的各种量（如应力和位移）具有复值解。模态过程的准确性取决于先前提取的特征数据中包含的表示，而直接过程利用所有可用的自由度。对于模态过程，两个无阻尼模型对于正弦施加压力载荷在顶面中心处的垂直挠度，对于使用 CAX4E 单元的模型为 1.56×10⁻⁷，对于使用 CAX8E 单元的模型为 1.55×10⁻⁷。对于直接过程，对于使用 CAX4E 单元的模型，挠度为 1.80×10⁻⁷，对于使用 CAX8E 单元的模型为 1.79×10⁻⁷。如果在使用所有可能的模态进行频率提取，模态过程给出与直接过程相同的结果。

这些结果与 Mercer 等人（1987）对于类似模型报告的结果在相同范围内。由于模型中未包含阻尼，所有变量的相位角为 0 或 180°。顶面中心处 2-位移的相位角为 180°。从[图 1.8.3-1](ch01s08ach65.md#bmkpiezo-dispnodamp) 可以明显看出这种相位偏移。输出了 CAX8E 模型约束底边上的总力；垂直方向的总力与边上反力的总和相匹配。

对于使用 CAX4E 单元（包括 Rayleigh 阻尼）的分析（输入文件 [ppzossd3.inp](../eif/ppzossd3.inp)），顶面中心处的挠度为 1.21×10⁻⁷，相位角为 133°。从[图 1.8.3-2](ch01s08ach65.md#bmkpiezo-disps) 可以明显看出幅值的减小和相位偏移。使用 CAX3E、CAX6E 和 CAX8RE 单元分别在输入文件 [ppzossd7.inp](../eif/ppzossd7.inp)、[ppzossd8.inp](../eif/ppzossd8.inp) 和 [ppzossd9.inp](../eif/ppzossd9.inp) 中分析了包括 Rayleigh 阻尼的问题。在这些分析中，顶面中心的挠度和相位角与使用 CAX4E 单元获得的结果一致。此外，直接和模态动态解的结果在每种情况下都良好一致。

使用输入文件 [ppzossd4.inp](../eif/ppzossd4.inp) 和 [ppzossd4a.inp](../eif/ppzossd4a.inp) 的分析具有正弦变化的表面电荷。由于模态过程无法将这些载荷充分转换为模态载荷，仅使用直接过程。第一步使用直接计算分析过程的 REAL ONLY 特征。第二步包括实部和虚部，载荷使用 IMAGINARY 参数施加在虚部平面上。顶面中心处的垂直挠度为 5.5×10⁻⁸。当电荷施加在实部平面时，该位置垂直位移的相位偏移为 180°（无阻尼），当电荷施加在虚部平面时为 90°。该节点位置的电势为 158V，第一步相位偏移为 0，第二步相位偏移为 90°。

### 输入文件

[ppzossd1.inp](../eif/ppzossd1.inp)

CAX4E 单元。

[ppzossd2.inp](../eif/ppzossd2.inp)

CAX8E 单元。

[ppzossd3.inp](../eif/ppzossd3.inp)

CAX4E 单元，包含阻尼。

[ppzossd4.inp](../eif/ppzossd4.inp)

带分布电荷的 CAX4E 单元（仅直接计算）。

[ppzossd4a.inp](../eif/ppzossd4a.inp)

带集中电荷的 CAX4E 单元（仅直接计算）。

[ppzossd7.inp](../eif/ppzossd7.inp)

CAX3E 单元，包含阻尼。

[ppzossd8.inp](../eif/ppzossd8.inp)

CAX6E 单元，包含阻尼。

[ppzossd9.inp](../eif/ppzossd9.inp)

CAX8RE 单元，包含阻尼。
