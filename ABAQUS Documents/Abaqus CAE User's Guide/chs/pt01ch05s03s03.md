# 5.3.4 自定义 3D 罗盘





要自定义 3D 罗盘的外观和方向，请在罗盘上单击鼠标按钮 3 并从出现的菜单中选择选项。你可以执行以下自定义：

****编辑****

选择**编辑**显示**指定视图**对话框并数值指定自定义视图方向。有关指定自定义视图方向的详细信息，请参阅["数值指定视图，" 第 5.2.9 节](pt01ch05s02s09.md)。

**提示：**你也可以双击 3D 罗盘以显示**指定视图**对话框。

**更改特权平面**

罗盘的底部（包含罗盘操作手柄）称为特权平面。默认情况下，*X*–*Z* 平面是 Abaqus/CAE 中的特权平面。特权平面有助于确定模型在视口中的"正确"方向。在罗盘的默认等轴测方向中，特权平面出现在底部，自由旋转手柄出现在顶部；从特权平面到自由旋转手柄的轴实际上指示了模型的"上"方向。

如果 *Y* 轴不对应于模型中的"上"方向，你可以将特权平面更改为罗盘中的三个主平面之一。例如，选择**使 XY 成为特权平面**将 *X*–*Y* 平面设置为特权平面。更改特权平面只会重新配置 3D 罗盘的形状，如图 [图 5-17](pt01ch05s03s04.md#usi-compass-privileged-plane) 所示；视图方向和**视图**工具栏中的预定义视图不会改变。

**图 5-17** 将特权平面从 *X*–*Z* 平面（左）更改为 *X*–*Y* 平面（右）。

![](../graphics/usi-compass-privileged-plane.png)

****隐藏****

选择**隐藏**从视口显示中移除 3D 罗盘。要恢复 3D 罗盘的显示，你必须使用视口标注选项（从主菜单中选择****Viewport**![](../graphics/images/arrow.gif)**Viewport Annotation Options****）。有关控制视口标注可见性的信息，请参阅["视口标注选项概述，" 第 56.4 节](pt05ch56hla01.md)。

****帮助****

选择**帮助**显示包含使用 3D 罗盘文档的帮助窗口。
