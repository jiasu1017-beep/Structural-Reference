# 15.6 理解接触和约束检测





Abaqus/CAE中的接触检测工具提供了一种快速简便的方法来在三维模型中定义接触相互作用和绑定约束。您无需单独选择表面并定义它们之间的相互作用，您可以指示Abaqus/CAE自动定位模型中基于初始接近很可能相互作用的所有表面。您可以调整接近设置并指定控制活动搜索域、表面定义和默认相互作用或约束设置的各种选项。搜索适用于几何和网格模型。

每个检测到的相互作用或约束涉及两个已识别的表面，也称为接触对候选。接触检测对话框以表格格式列出每个接触对候选及其默认参数。默认接触对候选参数与Abaqus/CAE传统相互作用编辑器中使用的默认参数略有不同；特别是，接触检测工具最初为每个接触对候选分配表面对表面离散化，而不是节点对表面离散化。

使用表格界面，您可以审查接触对候选以确保表面定义完整、主从分配适当以及参数正确。如有必要，您可以在表格中修改参数或表面分配，也可以根据需要创建新的接触对。一旦接触对候选配置符合您的规格，Abaqus/CAE会同时定义所有接触相互作用和绑定约束。
