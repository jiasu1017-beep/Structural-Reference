# 15 Summary of changes







This section summarizes the changes and the additions that have been made to the items that define an Abaqus model, including elements, keywords, user subroutines, and output variables. For more information on these modifications, refer to the preceding chapters.

The following identifiers are used:

| new | New in 6.14. |
| --- | --- |
| mod | Existed in 6.13 but has been modified or enhanced in 6.14. |
| rem | Existed in 6.13 but has been removed in 6.14. |
| (S) | New, modified, or removed in Abaqus/Standard. |
| (E) | New, modified, or removed in Abaqus/Explicit. |
| (C) | New, modified, or removed in Abaqus/CFD. |
