# 20.10.7 绘制与线相切的圆弧


使用 Sketcher 工具箱中的切线圆弧工具,在选定点处绘制与线或曲线相切、并结束于第二个选定点的圆弧,如下图所示:

![](../graphics/ske-arc2-nls.png)

在绘制时,您应该注意点的位置,因为这会影响网格的质量。草图中的点会成为您正在创建或修改的零件的顶点。反过来,当您在 Mesh 模块中对模型进行网格划分时,Abaqus/CAE 会将这些顶点转换为完全约束的种子,并在其位置放置节点。有关如何移动顶点(包括定义圆弧的顶点)的信息,请参见["拖动 Sketcher 对象," 第 20.17.1 节](pt03ch20s14hlb01.md)。

**绘制与线相切的圆弧的步骤:**

1. 从 Sketcher 工具箱的圆弧工具中,选择切线圆弧工具 ![](../graphics/ico_sketchArcTangent.png)。有关 Sketcher 工具箱中工具的图示,请参见["Sketcher 工具," 第 20.4.1 节](pt03ch20s04s01.md)。Abaqus/CAE 会在提示区域显示提示,引导您完成操作。
2. 选择沿线条或曲线的点。圆弧将在此点处保持切线连续。如果您从线条、样条曲线或第二个圆弧的端点开始圆弧,则新圆弧在端点处保持切线连续。如果您从空间中的一个点开始(该点没有端点),则圆弧与正 *X* 轴相切。如果有多条线或曲线在该点相交,则圆弧将与最早创建的线或曲线相切。
3. 随着移动光标,圆弧的半径会发生变化。在所需的端点处单击。**提示:**如有必要,您可以使用提示区域中的文本框输入切点和圆弧端点的精确坐标。有关创建所需圆弧的更多信息,请参见["指定精确几何," 第 20.6 节](pt03ch20s06.md)。
4. 要创建更多圆弧,请从第 2 步开始重复上述步骤。
5. 完成创建切线圆弧后,执行以下操作之一: - 在 Abaqus/CAE 窗口任意位置单击鼠标按钮 2。 - 选择 Sketcher 工具箱中的任何其他工具。 - 单击提示区域中的取消按钮 ![](../graphics/afxI_cancel.png)。
