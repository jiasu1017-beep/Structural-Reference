# 6.3.2 基于对象类型过滤你的选择





为了帮助你从当前视口中选择所需的实体（顶点、边、面、单元体、节点和单元），Abaqus/CAE 提供了一套过滤器，你可以用它们基于对象类型来限制你的选择。例如，如果你正在创建一个仅包含表面的集合，你可以将选择限制为仅面——顶点、边和单元体将不会被选择。

对象过滤器列在**选择**工具栏中。Abaqus/CAE 根据当前过程配置过滤器列表。如果你尚未开始选择过程，Abaqus/CAE 会列出一些常用过滤器（有关在过程之外选择对象的更多信息，请参阅["在选择过程之前选择对象，" 第 6.3.7 节](pt01ch06s03s06.md)）。

如果当前视口包含 Abaqus/CAE 部件或部件实例，你可以选择以下过滤器之一：

**全部**

除表皮和纵向加筋件之外的所有对象。

**顶点**

所有点对象，如顶点、基准点和节点。

**边**

所有边对象，如边、基准轴和单元边。

**面**

所有平面对象，如面、基准平面和单元面。

**单元体**

所有体积，如单元体和单元。

**表皮**

所有表皮增强件。

**纵向加筋件**

所有纵向加筋增强件。

**参考点**

所有参考点。

默认情况下，Abaqus/CAE 从所有顶点、边、面、单元体和参考点中选择，但排除表皮和纵向加筋件。只有在选择适当的过滤器后，你才能从视口中选择表皮或纵向加筋件。可用过滤器列表会随着你切换模块或选择过程而更新。

类似地，如果你正在当前视口中选择孤立单元（例如，要分配单元类型），你可以选择以下过滤器之一：
- 全部
- 零维单元
- 一维单元
- 二维单元
- 三维单元
- 表皮
- 纵向加筋件

默认情况下，Abaqus/CAE 从除表皮和纵向加筋件之外的所有单元中选择。
