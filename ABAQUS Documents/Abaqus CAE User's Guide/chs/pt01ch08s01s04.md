# 8.1.2 Windows 和 PostScript 图像布局





当你直接将所选视口的快照打印到 Windows 打印机或 PostScript 打印机或文件时，图像布局由可用页面大小、方向和视口纵横比决定：

**可用页面大小**

可用页面大小是根据你提供的总页面大小和边距信息计算的，如图 [图 8-1](pt01ch08s01s02.md#pri-target) 中所示的示例尺寸所示。

**图 8-1** 可用页面大小。

![](../graphics/pri-target-nls.png)

总页面大小由纸张大小决定。对于 Windows 打印机，请在**打印**对话框中单击**打印机属性**按钮打开**文档属性**对话框并更改纸张大小。对于 PostScript 打印机或文件，请在**打印**对话框中单击 **Postscript 格式选项**按钮更改纸张大小。

**方向**

页面方向可以是纵向或横向。

**纵横比**

纵横比是你为打印选择的视口整体宽度与整体高度之比。Abaqus/CAE 始终保持对象的纵横比，如图 [图 8-2](pt01ch08s01s02.md#uss-pri-results) 所示。你可以通过在打印前在画布上操作视口来控制纵横比。

**图 8-2** 缩放对象以保持纵横比。

![](../graphics/pri-results-nls.png)

确定 Windows 打印机上图像大小的默认方法——也是确定 PostScript 打印机和文件上图像大小的唯一方法——将图像缩放以适应纵横比和可用页面大小。

详细说明，请参阅["自定义发送到 Windows 打印机的图像，" 第 8.2.5 节](pt01ch08s02hlb05.md)和["自定义发送到 PostScript 打印机或文件的图像，" 第 8.2.6 节](pt01ch08s02hlb06.md)。
