# 29.4 创建离散紧固件





您可以通过选择连接所需表面的附着线在相互作用模块中创建离散紧固件。对于每条附着线，Abaqus/CAE确定紧固点，并在紧固点及其对应表面之间应用分布耦合。

您必须为选择的每条附着线分配连接器截面。您可以使用任何Abaqus连接类型来建模紧固件的复杂行为，例如包含弹性、损伤、塑性和摩擦影响的变形连接器。此外，您可以为不同的附着线分配不同的连接器截面。

**创建离散紧固件：**

1. 从相互作用模块的主菜单栏中，选择****特殊**![](../graphics/images/arrow.gif)**紧固件**![](../graphics/images/arrow.gif)**创建****。Abaqus/CAE显示**创建紧固件**对话框。**提示：**您也可以使用相互作用模块工具箱中的![](../graphics/ico_fastener.png)工具创建离散紧固件。
2. 从出现的**创建紧固件**对话框中，选择**离散**紧固件，然后单击**继续**。
3. 选择表示离散紧固件位置的附着线，然后单击**完成**。更多信息，请参见["通过投影点创建附着线，" 第59.6节](pt06ch59hla04.md)。Abaqus/CAE显示**编辑紧固件**对话框。
4. 选择将紧固点的运动耦合到影响半径内表面上节点平均运动的方法。
   - 选择**连续体分布**（默认）将每个紧固点的位移和旋转耦合到节点的平均位移。
   - 选择**结构分布**将每个紧固点的位移和旋转耦合到节点的平均位移和旋转。更多信息，请参阅《Abaqus分析用户指南》第35.3.2节"耦合约束"中的["定义表面耦合方法"](../usb/usb-link.md#usb-cni-pcoupling-method)。
5. 如果需要，关闭旋转自由度以移除其约束。Abaqus/CAE在分布耦合中始终约束所有平动自由度。
6. 选择Abaqus用于修改耦合节点处默认权重分布的权重方法：
   - 选择**均匀**（默认）以选择均匀且等于1.0的权重分布。
   - 选择**线性**以选择随距紧固点距离线性递减的权重分布。
   - 选择**二次**以选择随距紧固点距离按二次多项式函数递减的权重分布。
   - 选择**三次**以选择随距紧固点距离按三次多项式单调函数递减的权重分布。更多信息，请参阅《Abaqus分析用户指南》第35.3.2节"耦合约束"中的["权重方法"](../usb/usb-link.md#usb-cni-pcoupling-weighting)。
7. 如果需要，选择或创建指定约束自由度定义局部系统初始方向的数据坐标系。默认情况下，约束自由度的方向由全局坐标系定义。
8. 指定以紧固点为中心的影响半径。您可以输入值或允许Abaqus使用整个表面来定义耦合约束。
9. 单击**确定**关闭**编辑紧固件**对话框。Abaqus/CAE显示离散紧固件在紧固点和连接的表面之间的分布耦合。圆的半径反映了指定的影响半径。如果耦合约束定义使用整个表面，则来自紧固点的线将延伸超出圆。对于孤立网格模型，您可以通过更改装配显示选项中的面和边缘密度来修改分布耦合的外观；更多信息，请参见["控制属性的显示，" 第76.15节](pt07ch76hla14.md)。
10. 如["创建和修改连接器截面分配，" 第15.12.12节](pt03ch15s12hlb11.md)中所述，通过为附着线分配连接器截面来完成离散紧固件定义。Abaqus/CAE显示离散紧固件的连接器符号。
