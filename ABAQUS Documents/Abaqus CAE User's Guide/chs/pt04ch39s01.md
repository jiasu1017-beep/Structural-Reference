# 39.1 Abaqus/CAE 中子结构概述





子结构是将已分组的元素集合起来，以便为分析消除内部自由度。使用子结构可以使模型定义更简单，当您分析包含多次出现的相同零件（如齿轮的齿）的模型时，分析速度更快，因为您可以在模型中重复使用子结构。子结构通过保留节点上的保留自由度连接到模型的其余部分。决定应保留哪些节点和自由度以及保留多少的因素在["定义子结构，"Abaqus Analysis User's Guide第10.1.2节](../usb/usb-link.md#usb-anl-asuperelementdef)中讨论。模型中的子结构定义遵循两组步骤：
- ["在模型数据库中创建子结构，"第39.1.1节](pt04ch39s01s01.md)
- ["在分析中包含子结构，"第39.1.2节](pt04ch39s01s02.md)
