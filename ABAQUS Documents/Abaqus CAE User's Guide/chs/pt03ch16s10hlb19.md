# 16.10.19 定义声压边界条件





您可以创建声压边界条件来为声学介质的所选区域指定压力。（更多信息，请参见"边界条件"章节中的["声学、冲击和耦合声学-结构分析"，《Abaqus Analysis User's Guide》第6.10.1节](../usb/usb-link.md#usb-anl-aacoustic)。）

**指定声压的步骤：**

1. 使用以下方法之一显示声压边界条件编辑器： - 要创建新的声压边界条件，请按照["创建边界条件"章节第16.8.2节](pt03ch16s08hlb02.md)中概述的程序进行操作（**类别**：`其他`；**所选步骤的类型**：`声压`）。 - 要使用菜单或管理器编辑现有的声压边界条件，请参见["编辑依赖于步骤的对象"章节第3.4.13节](pt01ch03s04hlb05.md)。要编辑应用边界条件的区域，请参见["编辑应用规定条件的区域"章节第16.8.4节](pt03ch16s08hlb04.md)。
2. 如果编辑器顶部出现**方法**字段，请单击字段右侧的箭头，然后选择以下之一： - 如果要为声压指定值，请选择**指定幅值**。 - 如果要将声压幅值固定为前一个通用步骤中的最终值，请选择**固定为当前幅值**。只有当所选步骤同时支持这两种方法时，**方法**选项才会出现在编辑器中。
3. 如果编辑器中出现**分布**字段，请单击字段右侧的箭头，然后选择以下之一： - 选择**均匀**来定义均匀边界条件。 - 选择**用户自定义**来在用户子程序[`DISP`](../sub/sub-link.md#sub-xsl-disp)中定义边界条件。有关更多信息，请参见以下章节： - ["指定常规作业设置"章节第19.8.6节](pt03ch19s08hlb06.md) - ["DISP"，《Abaqus User Subroutines Reference Guide》第1.1.4节](../sub/sub-link.md#sub-rtn-udisp) - 选择解析场来定义空间变化的边界条件。只有对该边界条件类型有效的解析场才会显示在选择列表中。或者，您可以单击![](../graphics/ico_analyticalField.png)来创建新的解析场。（有关更多信息，请参见[第58章，"解析场工具集"](pt06ch58.md)。）**分布**选项仅在创建边界条件或在其创建的步骤中编辑边界条件时才会出现在编辑器中。如果您选择**固定为当前幅值**方法，此选项不可用。
4. 如果您选择了**均匀**或解析场分布选项，请执行以下步骤： 1. 如果适用，请在**幅值**字段中输入声压幅值。 2. 如果适用，请单击**幅值**字段右侧的箭头，然后从出现的列表中选择所需的幅值。或者，您可以单击![](../graphics/ico_amplitudeSmall.png)来创建新的幅值。（有关更多信息，请参见[第57章，"幅值工具集"](pt06ch57.md)。）**注意：**如果您在初始步骤中创建边界条件（此时幅值始终为零），或者将其幅值固定为前一个分析步骤中的最终值，则**幅值**和**幅值**字段不可用。 3. 单击**确定**保存数据并退出编辑器。
5. 如果您选择了**用户自定义**分布选项，请执行以下步骤： 1. 如果需要，请在**幅值**字段中输入声压幅值。您在编辑器中输入的幅值数据将传递到用户子程序中。（如果您在初始步骤中创建边界条件，则**幅值**字段不可用。） 2. 单击**确定**保存任何输入的数据并退出编辑器。 3. 进入作业模块，并显示感兴趣分析作业的作业编辑器。（有关更多信息，请参见["创建、编辑和操作作业"章节第19.7节](pt03ch19s07.md)。） 4. 在作业编辑器中，单击**常规**选项卡，并指定包含定义边界条件的用户子程序的文件。有关更多信息，请参见["指定常规作业设置"章节第19.8.6节](pt03ch19s08hlb06.md)。**注意：**您只能在作业编辑器中指定一个用户子程序文件；如果您的分析涉及多个用户子程序，则必须将用户子程序合并到一个文件中，然后指定该文件。
