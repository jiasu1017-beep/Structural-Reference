# 17.16.4 沿边施加有偏种子布置





您可以通过定义沿边最粗和最细单元的尺寸，或通过定义单元数量和两个尺寸的比值，来定义沿所选边的不均匀单元分布。 您可以定义从边的一端到另一端改变网格密度的单一偏置。 或者，您可以定义从边的中心到每一端改变网格密度的双向偏置。 例如，[图 17-138](pt03ch17s16hlb04.md#mgn-bias-double-nls) 显示了单一偏置和双向偏置种子布置边的组合。

**图 17-138** 单一偏置和双向偏置种子布置。

![](../graphics/mgn-bias-double-nls.png)

所有边种子工具都会生成边种子，以品红色显示。 边种子覆盖您指定的任何零件或实例种子。 您应该对所有边施加种子。

**沿边施加有偏种子布置：**

1. 从主菜单栏，选择 ****Seed**![](../graphics/images/arrow.gif)**Edges****。 Abaqus/CAE在提示区域显示提示以指导您完成该过程。 **提示：**您也可以单击Mesh模块工具箱中种子工具所在的 ![](../graphics/ico_meshSeedEdges.png) 工具。 （有关更多信息，请参阅["使用Mesh模块工具箱，" 第17.15节](pt03ch17s15.md)。）
2. 选择从视口中拾取的方法： - 对于单一偏置种子布置，切换**Use single-bias picking**并选择要种子布置的边。 对于单一偏置种子布置，您只能选择边，而且必须在您期望网格更密的那一端附近选择每条边。 - 对于双向偏置种子布置，关闭**Use single-bias picking**并选择要种子布置的边、面或单元。 您的选择位置不会影响种子布置。 默认情况下，Abaqus/CAE仅允许您选择边进行种子布置。 要选择面或单元进行种子布置，请使用**Selection**工具栏将可选择的对象类型更改为**Face**、**Cells**或**All**。 有关更多信息，请参阅["根据对象类型过滤您的选择，" 第6.3.2节](pt01ch06s03s01.md)。
3. 完成选择边、面或单元后，在提示区域单击**Done**。
4. 从出现的**Local Seeds**对话框中，选择偏置控制（**Single**或**Double**）。 **注意：**如果您选择的边之前使用多种偏置种子布置方法的组合进行了种子布置，Abaqus/CAE会提供**As Is**选项，允许您保留所选边上的偏置种子布置方法。 如果您选择了单一偏置种子布置，Abaqus/CAE会在每条所选边上显示箭头，指示单元密度将增加的方向。 如果您选择了双向偏置种子布置，Abaqus/CAE会在每条边的中心显示箭头，指示单元密度将增加的方向。
5. 选择尺寸方法（**By size**或**By number**）。 1. 如果您选择了**By size**，请输入元素的最小和最大尺寸——有偏种子布置每一端近似单元尺寸。 您不能对有偏种子布置应用曲率控制。 2. 如果您选择了**By number**，请输入单元数量和偏置比。 偏置比是边上最大和最小单元尺寸的近似比，必须大于1。 3. 如果需要，单击**Select**反转种子布置偏置的方向。 （当您选择要施加有偏种子布置的边时，单元密度朝着距离您的拾取位置最近的边的端部增加。） Abaqus/CAE反转每条所选边上的箭头方向。 **注意：**如果您选择的边之前使用多种单元尺寸或数量参数的组合进行了种子布置，Abaqus/CAE会提供**As Is**选项，允许您保留所选边上的尺寸参数。
6. 如果需要，通过在提示区域单击**Constraints**按钮并响应出现的对话框来更改默认种子约束。 有关设置种子约束的更多信息，请参阅["对边种子施加约束，" 第17.16.5节](pt03ch17s16hlb05.md)。
7. 如果需要，切换**Create set with name**以创建包含您选择的边、面和单元的集合，并输入集合的名称。 如果您随后想要更改种子布置，可以在步骤2中选择该集合，而无需重新选择边、面和单元。
8. 单击**Apply**查看Abaqus/CAE将使用的种子布置。 品红色种子出现在所选边上。
9. 如果需要，调整**Local Seeds**对话框中输入的值。
10. 单击**OK**提交单元种子布置并关闭对话框。
