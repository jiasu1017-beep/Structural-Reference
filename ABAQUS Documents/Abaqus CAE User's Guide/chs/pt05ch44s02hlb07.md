# 44.5.7 设置云图限制





默认情况下，Abaqus/CAE自动[计算](pt05ch44s02hlb09.md)云图中显示值的限制。您可以控制Abaqus/CAE显示的最小值和最大值；例如，要消除极值或在固定范围内检查变化。

一旦设置，云图限制在会话期间保持有效。要了解如何显示与云图关联的最小值和最大值，请参见["自定义图例，" 第56.1节](pt05ch56s01.md)。

如果您使用自定义云图区间值，可以覆盖云图限制；有关更多信息，请参见["自定义云图区间，" 第44.5.12节](pt05ch44s02hlb12.md)。当您使用自定义云图区间时，Abaqus/CAE使用**Edit Intervals**对话框中指定的最大和最小区间值作为云图限制。

**要设置云图限制：**

1. 找到**Contour Limits**选项。从主菜单栏中选择****Options**![](../graphics/images/arrow.gif)**Contour****，或单击工具箱中的![](../graphics/ico_plotContourOptions.png)；然后在出现的对话框中单击**Limits**选项卡。**Limits**选项变得可用。
2. 在标记为**Max**的区域中，选择以下选项之一：- 选择**Auto-compute**以请求Abaqus计算最大云图值。- 选择**Specify**以自己指定最大值；然后在**Specify**文本字段中输入您选择的最大值。
3. 在标记为**Min**的区域中，选择以下选项之一：- 选择**Auto-compute**以请求Abaqus计算最小云图值。- 选择**Specify**以自己指定最小值；然后在**Specify**文本字段中输入您选择的最小值。
4. 单击**Apply**实施您的更改。在当前视口中的云图中，Abaqus显示您在限制范围内指定的云图值。如果激活，云图图例会更改以显示您指定的限制。如果您指定的最大值小于实际最大值，或者如果您指定的最小值大于实际最小值，Abaqus会在图例中显示实际最大值或最小值（分别）。有关云图图例的更多信息，请参见["自定义图例，" 第56.1节](pt05ch56s01.md)。要了解如何控制超出限制的值的颜色，请参见["自定义云图颜色，" 第44.5.10节](pt05ch44s02hlb10.md)。默认情况下，您的更改在会话期间保存，并将影响所有后续云图。如果您想在后续会话中保留更改，请将它们保存到文件。有关更多信息，请参见["保存自定义设置以供后续会话使用，" 第55.1.1节](pt05ch55s01s01.md)。
