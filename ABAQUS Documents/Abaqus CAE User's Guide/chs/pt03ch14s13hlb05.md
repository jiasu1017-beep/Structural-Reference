# 14.13.5 定义时间点





您可以定义命名的时间点集合，用于描述分析过程中需要Abaqus写入场输出或历史输出数据的所有时刻。通过在指定的时间点请求输出，您可以创建不规则采样，重点收集分析关键时刻的数据。

「Create Time Points」对话框提供了两种定义时间点集合的方法。您可以通过在对话框表格部分的单独行上分别输入每个值来指定每个时间点。或者，您可以选择「Specify using delimiters」来定义多个具有不同增量（定义每个范围的采样率）的时间范围。例如，您可以在0到1之间每0.1秒创建一个时间点，然后在1到2之间将增量更改为0.05。单独定义和时间范围定义这两种方法互斥；例如，您不能将单独的时间点添加到使用时间范围方法定义的系列时间点中。

**注意：**您还可以从ASCII文件将时间点集合（单独或使用时间范围）导入模型数据库。导入数据后，您可以编辑这些时间点。

「Create Time Points」对话框以表格格式显示时间点数据。您可以通过添加、删除和清除数据行，或编辑单个数据点或时间范围来修改这些集合。创建时间点集合后，您可以编辑其内容或复制、重命名或删除该集合。如果您重命名或删除已请求场输出或历史输出的时间点集合，Abaqus/CAE会显示错误消息。

**创建时间点集合的步骤：**

1. 打开「Create Time Points」对话框。从主菜单栏选择 ****Output**![](../graphics/images/arrow.gif)**Time Points**![](../graphics/images/arrow.gif)**Create****。
2. 输入此时间点集合的名称，然后点击「OK」。此时会出现「Edit Time Points」对话框。
3. 要从ASCII文件读取时间点数据，请执行以下步骤：1. 在对话框的表格部分点击鼠标按钮3；然后从出现的列表中选择「Read from File」。此时会出现「Read Data from ASCII File」对话框。2. 点击![](../graphics/ico_fileOpen.png)，并使用「ASCII File Selection」对话框导航到要导入的文件。选中文件后点击「OK」。3. 如有需要，您可以修改数据导入在时间点表中的起始位置。默认情况下，导入的起始行和列值都等于1；如果您想用额外的行和列值自定义导入的数据，可以修改任一值。4. 点击「OK」将值导入时间点表。
4. 选择两种时间点定义方法之一：- 要单独定义时间点，请在第1行输入一个时间点，然后为集合中的每个额外时间点添加新行和时间点。单独输入的时间点必须按顺序输入。- 要使用分隔符指定时间点，请选择「Specify using delimiters」，然后为您要创建的每个时间点系列输入起始时间、结束时间和增量。有关表格数据输入的更多信息，请参见「输入表格数据」，第3.2.7节。
5. 点击「OK」保存您的时间点集合。
