# 12.11.1 定义电导率







如果您在耦合热-电或耦合热-电-结构分析中使用材料，必须定义材料的电导率。对于时间谐波涡流分析，材料的电导率必须用于定义导体的电磁响应。您可以指定各向同性、正交异性或完全各向异性的电导率。有关更多信息，请参见以下章节：
- "电导率"，Abaqus分析用户指南第26.5.1节
- "耦合热-电分析"，Abaqus分析用户指南第6.7.3节
- "完全耦合热-电-结构分析"，Abaqus分析用户指南第6.7.4节
- "涡流分析"，Abaqus分析用户指南第6.7.5节

**定义电导率：**

1. 从**编辑材料**对话框的菜单栏中，选择**电气/磁性**![](../graphics/images/arrow.gif)**电导率**。（有关显示**编辑材料**对话框的信息，请参见"创建或编辑材料"，第12.7.1节。）
2. 单击**类型**字段右侧的箭头，指定电导率的方向依赖性。
3. 切换**使用频率相关数据**定义随频率变化的电导率。**数据**表中会出现一列标为**频率**。
4. 切换**使用温度相关数据**将电导率定义为温度的函数。**数据**表中会出现一列标为**温度**。
5. 单击**场变量数量**字段右侧的箭头增加或减少电导率所依赖的场变量数量。
6. 在**数据**表中输入适用数据：**电导率**各向同性电导率。（单位为[CT1L1![](../graphics/usi_eqn00317.gif)1](../popups/usb-int-iconventions-unitsym.md)。）**s11(E)、s22(E)和s33(E)**正交异性电导率的三个值，![](../graphics/usi_eqn00318.gif)、![](../graphics/usi_eqn00319.gif)和![](../graphics/usi_eqn00320.gif)。（单位为[CT1L1![](../graphics/usi_eqn00317.gif)1](../popups/usb-int-iconventions-unitsym.md)。）**s11(E)、s12(E)、s22(E)、s13(E)、s23(E)和s33(E)**各向异性电导率的六个值，![](../graphics/usi_eqn00318.gif)、![](../graphics/usi_eqn00321.gif)、![](../graphics/usi_eqn00319.gif)、![](../graphics/usi_eqn00322.gif)、![](../graphics/usi_eqn00323.gif)和![](../graphics/usi_eqn00320.gif)。（单位为[CT1L1![](../graphics/usi_eqn00317.gif)1](../popups/usb-int-iconventions-unitsym.md)。）**频率**频率，单位为循环/时间。**温度**温度。**场变量* n ***预定义的场变量。您可能需要展开对话框以查看**数据**表中的所有列。有关如何输入数据的详细信息，请参见"输入表格数据"，第3.2.7节。
7. 单击**确定**关闭**编辑材料**对话框。或者，您可以从**编辑材料**对话框的菜单中选择其他材料行为进行定义（有关更多信息，请参见"浏览和修改材料行为"，第12.7.2节）。
