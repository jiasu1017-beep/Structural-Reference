# 29.106 Viscoelastic 对象

Viscoelastic 对象指定与弹性一起使用的耗散行为。

**访问**

```
import material
mdb.models[*name*].materials[*name*].viscoelastic
import odbMaterial
session.odbs[*name*].materials[*name*].viscoelastic
```

### 29.106.1 Viscoelastic(...)

此方法创建 Viscoelastic 对象。

**路径**

```
mdb.models[*name*].materials[*name*].Viscoelastic
session.odbs[*name*].materials[*name*].Viscoelastic
```

**必需参数**

*domain*

SymbolicConstant，指定域定义。可能的值为：
- FREQUENCY，指定频率域。此域仅适用于 Abaqus/Standard 分析。
- TIME，指定时间域。

*table*

Float 元组序列，指定下述项目。

**可选参数**

*frequency*

SymbolicConstant，指定频率域定义。此参数仅在 *domain*=FREQUENCY 时需要。可能的值为 FORMULA、TABULAR、PRONY、CREEP_TEST_DATA 和 RELAXATION_TEST_DATA。默认值为 FORMULA。

*type*

SymbolicConstant，指定类型。此参数仅在 *domain*=FREQUENCY 且 *frequency*=TABULAR 时需要。可能的值为 ISOTROPIC 和 TRACTION。默认值为 ISOTROPIC。

*preload*

SymbolicConstant，指定预载荷。此参数仅在 *domain*=FREQUENCY 且 *frequency*=TABULAR 时需要。可能的值为 NONE、UNIAXIAL、VOLUMETRIC 和 UNIAXIAL_VOLUMETRIC。默认值为 NONE。

*time*

SymbolicConstant，指定时间域定义。此参数仅在 *domain*=TIME 时需要。可能的值为 PRONY、CREEP_TEST_DATA、RELAXATION_TEST_DATA 和 FREQUENCY_DATA。默认值为 PRONY。

*errtol*

Float，指定数据点在最小二乘拟合中的允许平均均方根误差。Float 值对应于百分比；例如，0.01 为 1%。默认值为 0.01。

此参数仅在 *time*=CREEP_TEST_DATA、RELAXATION_TEST_DATA 或 FREQUENCY_DATA 时有效；或仅在 *frequency*=CREEP_TEST_DATA 或 RELAXATION_TEST_DATA 时有效。

*nmax*

Int，指定 Prony 级数中项的最大数量 ![](../graphics/ker_eqn00384.gif)。最大值为 13。默认值为 13。

此参数仅在 *time*=CREEP_TEST_DATA、RELAXATION_TEST_DATA 或 FREQUENCY_DATA 时有效；或仅在 *frequency*=CREEP_TEST_DATA 或 RELAXATION_TEST_DATA 时有效。

*volumetricTable*

Float 元组序列，指定下述项目。默认值为空序列。

**表格数据**

如果 *frequency*=FORMULA，则 *table* 的表格数据指定以下内容：
- ![](../graphics/ker_eqn00385.gif) 的实部，![](../graphics/ker_eqn00386.gif)。
- ![](../graphics/ker_eqn00385.gif) 的虚部。
- ![](../graphics/ker_eqn00278.gif) 的值。
- ![](../graphics/ker_eqn00387.gif) 的实部，![](../graphics/ker_eqn00388.gif)。如果材料不可压缩，则忽略此值。
- ![](../graphics/ker_eqn00387.gif) 的虚部。如果材料不可压缩，则忽略此值。
- ![](../graphics/ker_eqn00038.gif) 的值。如果材料不可压缩，则忽略此值。

如果 *frequency*=TABULAR 且 *type*=ISOTROPIC 且 *preload*=NONE，或 *time*=FREQUENCY_DATA，则 *table* 的表格数据指定以下内容：
- ![](../graphics/ker_eqn00389.gif) 的实部，![](../graphics/ker_eqn00390.gif)。
- ![](../graphics/ker_eqn00389.gif) 的虚部，![](../graphics/ker_eqn00391.gif)。
- ![](../graphics/ker_eqn00392.gif) 的实部，![](../graphics/ker_eqn00393.gif)。如果材料不可压缩，则忽略此值。
- ![](../graphics/ker_eqn00392.gif) 的虚部，![](../graphics/ker_eqn00394.gif)。如果材料不可压缩，则忽略此值。
- 每单位时间的频率 ![](../graphics/ker_eqn00395.gif)（周期/时间）。

如果 *frequency*=TABULAR 且 *type*=ISOTROPIC 且 *preload*=UNIAXIAL，则 *table* 的表格数据指定以下内容：
- 损耗模量。
- 储能模量。
- 频率。
- 单轴应变。

如果 *frequency*=TABULAR 且 *type*=TRACTION 且 *preload*=NONE，则 *table* 的表格数据指定以下内容：
- 归一化损耗模量。
- 归一化剪切模量。
- 频率。

如果 *frequency*=TABULAR 且 *type*=TRACTION 且 *preload*=UNIAXIAL 或 *preload*=UNIAXIAL_VOLUMETRIC，则 *table* 的表格数据指定以下内容：
- 损耗模量。
- 储能模量。
- 频率。
- 闭合。

如果 *time*=PRONY 或 *frequency*=PRONY，则 *table* 的表格数据指定以下内容：
- ![](../graphics/ker_eqn00396.gif)，剪切松弛模量 Prony 级数展开中第一项的模量比。
- ![](../graphics/ker_eqn00397.gif)，体积松弛模量 Prony 级数展开中第一项的模量比。
- ![](../graphics/ker_eqn00262.gif)，Prony 级数展开中第一项的松弛时间。

如果 *frequency*=TABULAR 且 *type*=ISOTROPIC 且 *preload*=VOLUMETRIC 或 *preload*=UNIAXIAL_VOLUMETRIC，则 *volumetricTable* 的表格数据指定以下内容：
- 损耗模量。
- 储能模量。
- 频率。
- 体积比。

**返回值**

Viscoelastic 对象。

**异常**

RangeError。

### 29.106.2 setValues(...)

此方法修改 Viscoelastic 对象。

**必需参数**

无。

**可选参数**

`setValues` 的可选参数与 [Viscoelastic](pt01ch29pyo106.md#ker-viscoelastic-viscoelastic-pyc) 方法的参数相同。

**返回值**

无

**异常**

RangeError。

### 29.106.3 成员

Viscoelastic 对象的成员与 [Viscoelastic](pt01ch29pyo106.md#ker-viscoelastic-viscoelastic-pyc) 方法的参数具有相同的名称和描述。

此外，Viscoelastic 对象可以具有以下成员：

*combinedTestData*

[CombinedTestData](pt01ch29pyo17.md) 对象。

*shearTestData*

[ShearTestData](pt01ch29pyo90.md) 对象。

*trs*

[Trs](pt01ch29pyo100.md) 对象。

*volumetricTestData*

[VolumetricTestData](pt01ch29pyo110.md) 对象。
