### 概述

接触属性：
- 定义当表面接触时控制表面行为的表面相互作用模型；和
- 可以选择性地应用于一般接触域内的特定区域。

### 分配接触属性

Abaqus/Standard中的默认接触属性模型假定法向"硬"接触、无摩擦、无热相互作用等。您可以为一般接触域的指定区域分配非默认接触属性定义（表面相互作用）。

Abaqus/Standard中一般接触的接触属性在分析开始时分配，不能跨步修改，以下述摩擦模型的更改除外。

用于指定应分配非默认接触属性的区域的表面名称不必与用于指定一般接触域的表面名称对应。在许多情况下，接触相互作用将为大域定义，而将非默认接触属性分配给该域的子集。任何落在一般接触域之外的区域的接触属性分配将被忽略。如果指定区域重叠，最后分配优先。

| **输入文件用法：** | ``` [*CONTACT PROPERTY ASSIGNMENT*](../key/key-link.md#usb-kws-hcontpropassign) *表面_1*, *表面_2*, *相互作用属性名称* ``` |
| --- | --- |
| | 此选项必须与[*CONTACT*](../key/key-link.md#usb-kws-hcontact)选项结合使用，最多应出现一次；数据行可以根据需要重复，以将接触属性分配给不同区域。如果省略第一个表面名称，则假定为包含整个一般接触域的默认表面。如果省略第二个表面名称或与第一个表面名称相同，则假定为第一个表面与其自身的接触。表面可以定义为跨越多个未连接的物体，因此自接触不限于单个物体与自身的接触。如果省略相互作用属性名称，则假定为Abaqus/Standard中默认接触属性的未命名集。如果指定了相互作用属性名称，它也必须作为输入文件模型部分中[*SURFACE INTERACTION*](../key/key-link.md#usb-kws-hsurfaceinteraction)选项上NAME参数的值出现。 |

| **Abaqus/CAE用法：** | 使用以下选项将全局接触属性分配给整个一般接触域： |
| --- | --- |
| | 相互作用模块：**创建相互作用**：**一般接触（Standard）**：**接触属性**：**全局属性分配**：*相互作用属性名称* 使用以下选项将接触属性分配给单个表面对：相互作用模块：**创建相互作用**：**一般接触（Standard）**：**接触属性**：**单个属性分配**：**编辑**：在左侧列中选择表面和接触属性，点击中间的箭头将它们转移到接触属性分配列表。在Abaqus/CAE中，您必须分配全局接触属性；Abaqus/CAE不假定默认接触相互作用属性。分配给单个表面对的接触属性覆盖全局分配。 |

#### 在分析期间更改摩擦属性

与给定命名表面相互作用定义关联的摩擦属性可以在Abaqus/Standard分析的任何特定步中修改，如["在Abaqus/Standard分析期间更改摩擦属性" in "摩擦行为，" 第37.1.5节"](pt09ch37s01aus169.md#usb-cni-afriction-change-std)中所讨论。

#### 示例

以下接触属性分配在一般接触分析中作为模型数据指定：
- 将`contProp1`全局分配给整个一般接触域；
- 将`contProp2`局部分配给`surf1`的自接触；
- 将默认Abaqus接触属性局部分配给`surf2`和`surf3`之间的接触；和
- 将`contProp3`局部分配给整个接触域与`surf4`之间的接触。`contProp3`的摩擦系数在第二步中从初始值0.20重置为0.05。
