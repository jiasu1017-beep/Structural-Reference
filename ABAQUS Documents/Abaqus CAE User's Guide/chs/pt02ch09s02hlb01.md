# 9.2.2 模型属性是什么？





模型属性描述模型的特征，并与模型一起存储在模型数据库中。以下列表描述了Abaqus/CAE模型的属性：
- 描述。如果模型数据库中有许多相似的模型，您可以使用描述来区分模型。您输入的描述会与模型属性一起存储；描述写在输入文件的标题上方，但不会写入输出数据库。有关更多信息，请参阅["为Abaqus/CAE模型添加描述，"第9.10.2节"](pt02ch09s09hlb02.md)。
- 类型。您可以在**Standard和Explicit模型**（默认）、**CFD模型**或**电磁模型**之间进行选择。一旦您选择了模型类型，Abaqus/CAE会过滤主菜单栏、工具箱和模型树中可用的选项集，使它们适合您的模型类型选择。
- 模型的物理常数。您可以为**绝对零温度**和**斯特藩-玻尔兹曼常数**输入值。这些值是指定热传递分析中的表面发射率和辐射条件所必需的。您还可以为**通用气体常数**输入值，并且可以从**指定声波公式**列表中选择选项。
- 重启信息，将使用先前分析的数据启动分析。您可以指定以下内容：- Abaqus/CAE将从中读取重启信息的作业名称。- Abaqus/CAE将从中重启分析的步骤名称。- Abaqus/CAE将从中重启分析的增量或间隔。有关更多信息，请参阅["重启分析，"第19.6节"](pt03ch19s06.md)和["重启分析，"Abaqus Analysis User's Guide第9.1.1节"](../usb/usb-link.md#usb-anl-arestart)。

- 子模型信息，将用于驱动模型中的子模型边界条件或载荷。您可以指定以下内容：- 将用于驱动子模型边界条件或载荷的全局解决方案的作业。- 是否将使用壳全局模型来驱动实体子模型。有关更多信息，请参阅[第38章，"子模型"](pt04ch38.md)。
