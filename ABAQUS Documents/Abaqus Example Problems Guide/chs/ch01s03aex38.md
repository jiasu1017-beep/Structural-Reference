# 1.3.7 圆筒杯的轴对称成形





**产品：** Abaqus/Standard  Abaqus/Explicit  

本示例说明了使用轴对称模型进行圆筒杯液压成形的工艺。在这种情况下，使用两级成形序列，两级之间进行退火。使用了两种分析方法：一种是完全使用Abaqus/Explicit分析整个过程；另一种是使用Abaqus/Explicit分析成形序列，而回弹分析在Abaqus/Standard中运行。这里，使用导入功能在Abaqus/Explicit和Abaqus/Standard之间传输结果，反之亦然。

### 问题描述

模型由可变形坯料和三个刚性模具组成。坯料半径为150.0 mm，厚度为1.0 mm，使用轴对称壳单元SAX1进行建模。坯料与模具之间的摩擦系数取为0.1。模具1和2相对于坯料偏移了坯料厚度的一半，因为接触算法考虑了壳厚度。为了避免在第二成形阶段将模具3放入位置时坯料被夹住，模具2和3之间的径向间隙设置为比初始壳厚度大20%。[图1.3.7-1](ch01s03aex38.md#exxaxiform-config1)和[图1.3.7-2](ch01s03aex38.md#exxaxiform-config2)显示了模型的初始几何形状。

三个模具使用二维解析刚性表面或RAX2刚性单元进行建模。解析刚性表面可以更准确地表示二维曲线冲头几何形状，并可以节省计算成本。可以在试样表面上查看接触压力，反应力在刚性体参考节点处可用。此外，还测试了运动学（默认）和罚函数接触公式。此处提供了使用刚性单元的运动学接触公式的结果。

坯料由铝镇静钢制成，假定满足真应力和对数应变之间的Ramberg-Osgood关系：

![](../graphics/exa_eqn00281.gif)

参考应力值（*K*）为513 MPa，加工硬化指数（*n*）为0.223。假定为各向同性弹性，杨氏模量为211 GPa，泊松比为0.3。使用这些数据获得91.3 MPa的初始屈服应力。应力-应变行为由匹配Ramberg-Osgood曲线的分段线性段定义，直至107%的总（对数）应变水平，具有Mises屈服、各向同性硬化和无率依赖性。

完全在Abaqus/Explicit中进行的分析包括六个步骤。在第一步中，定义了坯料与模具1和2之间的接触。两个模具保持固定，而将10 MPa的分布载荷在负*z*方向上斜坡加载到坯料上。然后在第二步中斜坡卸载该载荷，允许坯料弹回到平衡状态。

第三步是退火步骤。Abaqus/Explicit中的退火程序将所有适当的状态变量设置为零。这些变量包括应力、应变（不包括壳、膜和平面应力单元的变薄应变）、塑性应变和速度。退火步骤没有时间关联。该过程瞬时发生。

在第四步中，定义了坯料与模具3之间的接触，并移除了坯料与模具1之间的接触。模具3垂直向下移动，为下一次压力加载做准备。

在第五步中，另一个分布载荷在正*z*方向上施加到坯料上，迫使坯料进入模具3。然后在第六步中斜坡卸载该载荷以监控坯料的回弹。

为了获得准静态响应，进行了调查以确定施加压力载荷和移除它们的最佳速率。最佳速率在计算时间和结果准确性之间取得平衡；增加加载速率将减少计算机时间，但导致较不准确的准静态结果。

使用导入功能的分析包括四次运行。第一次运行与之前描述的Abaqus/Explicit分析的第一步相同。在第二次运行中，将第一成形阶段的Abaqus/Explicit结果导入Abaqus/Standard，不更新参考配置，并导入材料状态以进行第一次回弹分析。第三次运行将第一次回弹分析的结果导入Abaqus/Explicit，用于随后的退火过程和第二成形阶段。通过更新参考配置且不导入材料状态，此运行开始时没有初始应力或应变，有效地模拟了退火过程。最后一次运行将第二成形阶段的结果导入Abaqus/Standard进行第二次回弹分析。

### 结果和讨论

[图1.3.7-3](ch01s03aex38.md#exxaxiform-deform1)到[图1.3.7-5](ch01s03aex38.md#exxaxiform-shllthck-cntr)显示了完全在Abaqus/Explicit中使用刚性单元方法和运动学接触公式进行的分析结果。[图1.3.7-3](ch01s03aex38.md#exxaxiform-deform1)显示了第2步结束时的变形形状，在弹性回弹之后。[图1.3.7-4](ch01s03aex38.md#exxaxiform-configfinal)显示了分析结束时的变形形状，在第二次弹性回弹之后。虽然此处未显示，但在卸载步骤期间观察到的回弹量可以忽略不计。[图1.3.7-5](ch01s03aex38.md#exxaxiform-shllthck-cntr)显示了分析结束时壳厚度（STH）的等值线图。杯中心处的材料厚度减少了约20%，而杯边缘处的厚度增加了约10%。

使用导入功能在Abaqus/Standard中进行回弹分析获得的结果几乎相同，使用解析刚性表面和/或罚函数接触公式的结果也是如此。

您可以使用**abaqus restartjoin**执行过程从重新启动分析创建的输出数据库中提取数据，并将数据追加到第二个输出数据库。有关更多信息，请参见["连接重新启动分析的输出数据库（`.odb`）文件，"Abaqus分析用户指南第3.2.21节](../usb/usb-link.md#usb-int-drestartjoinproc)。
