# *PLASTIC M1







### *PLASTIC M1定义框架单元的第一塑性弯曲矩行为。

此选项只能与[*FRAME SECTION](ch06abk34.md)选项结合使用。它描述框架单元中的弯曲矩作为关于第一截面方向塑性旋转的函数。

**产品：**Abaqus/Standard  

**类型：**模型数据  

**级别：**零件，零件实例  

##### **参考：**

- ["框架截面行为，" Abaqus Analysis User's Guide第29.4.2节](../usb/usb-link.md#usb-elm-eusingframesection)
- [*FRAME SECTION](ch06abk34.md)

**此选项没有关联的参数。**

### **定义屈服矩*M10*随塑性旋转变化的数据行：**

**第一行：**

根据需要重复此数据行以定义弯曲矩与关于第一截面轴塑性旋转之间的关系。至少需要三对数据。
