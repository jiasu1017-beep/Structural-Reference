# 20.10.5 使用圆心和两个端点草绘圆弧







使用 Sketcher 工具箱中的带两个端点的圆心圆弧工具绘制基于圆心和两个端点的圆弧。下图显示了生成的圆弧：

![](../graphics/ske-arcbtn-nls.png)

构成分析[刚性表面](pt03ch11s04s02.md)一部分的圆弧不能包含大于 180 度的角度。如有必要，请连接两个圆弧以创建包含大于 180 度角度的圆弧。对于可变形体或离散刚性表面没有这种限制。

在草绘时，您应该注意定位点，因为这会影响网格的质量。草图中的点会成为您正在创建或修改的零件的顶点。反过来，当您在网格模块中对模型进行网格划分时，Abaqus/CAE 会将这些顶点转换为完全约束的种子，并将节点放置在其位置。有关如何随后移动顶点的信息，请参见["拖动 Sketcher 对象，"第 20.17.1 节](pt03ch20s14hlb01.md)。

**使用圆心和两个端点草绘圆弧：**

1. 从 Sketcher 工具箱中的圆弧工具中，选择带两个端点的圆心圆弧工具![](../graphics/ico_sketchArcCenterEnds.png)。有关 Sketcher 工具箱中工具的图表，请参见["Sketcher 工具，"第 20.4.1 节](pt03ch20s04s01.md)。Abaqus/CAE 在提示区域显示提示以指导您完成该过程。
2. 点击所需圆弧的中心。
3. 点击第一个端点以定义圆弧的半径。**提示：**如有必要，您可以使用提示区域中的文本框输入圆弧中心和端点的精确坐标。有关精确定义圆弧的更多信息，请参见["指定精确几何，"第 20.6 节](pt03ch20s06.md)。当您将光标从圆弧中心移动到其第一个端点时，Abaqus/CAE 会绘制一个圆来显示圆弧的半径。
4. 从第一个端点沿顺时针方向移动光标以顺时针方向绘制圆弧。从第一个端点沿逆时针方向移动光标以逆时针方向绘制圆弧。点击第二个端点以定义圆弧的长度。如果您开始绘制圆弧然后决定更改其方向，您必须返回第一个端点并将光标朝第二个端点的所需方向移动。第 2 步和第 3 步分别定义圆弧的圆心和圆弧的半径。您在第 4 步中选择的点仅定义圆弧的长度，该点可能不在圆弧上。如果希望圆弧通过草图的顶点，您应该在第 3 步中点击第一个端点时选择该顶点。**提示：**如有必要，点击**上一步**按钮![](../graphics/afxI_backup.png)可以反转端点的选择。
5. 要创建更多圆弧，重复从第 2 步开始的上述步骤。
6. 完成创建圆弧后，执行以下操作之一：- 在 Abaqus/CAE 窗口中的任意位置点击鼠标按钮 2。- 在 Sketcher 工具箱中选择任何其他工具。- 点击提示区域中的取消按钮![](../graphics/afxI_cancel.png)。
