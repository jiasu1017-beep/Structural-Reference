# 15.1.5 输出辐射视角因子和面元面积：FRAD





**产品：** Abaqus/Standard

本示例说明了使用FORTRAN程序从结果文件读取辐射视角因子和面元面积。

### 概述

该程序展示了如何从结果文件检索视角因子和面元面积。使用Abaqus Benchmarks Guide中["轴对称单元腔体辐射视角因子计算，"第1.6.6节](../bmk/bmk-link.md#bmk-anl-radviewelemcalcs-axisym)中详述的基准问题创建的结果文件来验证输出记录已被正确读取和输出。此程序将假定结果文件具有默认文件扩展名`.fil`，对应于FORTRAN单元8。

### 编程细节

在继续之前，请审查["Abaqus结果文件用户后处理概述，"第15.1.1节](ch15s01abo02.md)中关于编程概念和Abaqus FORTRAN接口的一般讨论，以及Abaqus Analysis User's Guide第5章"文件输出格式"中给出的后处理详细讨论。

运行程序`FRAD`时（此程序在Abaqus发布媒体上命名为`frad.f`），用户将收到提示输入初始化`FNAME`的文件名。其他变量（如`LOUTF`、`NRU`、`LRUNIT(1,NRU)`和`LRUNIT(2,NRU)`）在程序内部初始化。然后调用`INITPF`和`DBNRU`来完成必要的初始化和文件连接。默认情况下，结果文件会处理结果文件中的所有步和增量。用户可以通过将`LSTEPA`和`LINCA`设置为所需的步和增量来限制输出，并取消注释简单的`IF - END IF`块。数据处理从`DO`循环开始，通过调用`DBFILE`逐一读取所有要处理的记录。每条记录由其记录键标识，记录键存储在记录的第二条目中。当`FRAD`处理记录`1922`和`2000`时，会写出标题和当前步和增量号，以便提供一种识别每个分析中数据开始的方法。然后读取记录类型`1605`、`1606`、`1607`和`1609`；并将所需的输出写入输出文件`vfout`。

### 程序编译和链接

在程序可以执行之前，必须编译和链接FORTRAN程序。**abaqus make**工具的单一执行执行这两个操作以及`aba_param.inc`文件的包含：

```
abaqus make job=frad
```

此过程可能需要重复，直到所有FORTRAN错误被纠正。成功编译后，程序的目标代码会自动与存储在共享程序库和接口库中的Abaqus目标代码链接，以构建可执行程序。请参阅[Abaqus Installation and Licensing Guide](../sgb/sgb-link.md#sgb)，获取有关特定计算机的编译和链接命令的信息。

### 程序执行

在程序执行之前，必须创建结果文件，并将所需的输出写入该文件。本示例使用运行Abaqus Benchmarks Guide中["轴对称单元腔体辐射视角因子计算，"第1.6.6节](../bmk/bmk-link.md#bmk-anl-radviewelemcalcs-axisym)讨论的输入文件`xrvda4n1.inp`创建的结果文件`xrvda4n1.fil`。当使用命令`abaqus frad`执行程序时，会出现提示

```
Enter the name of the input file (w/o .fil):
```

输入`xrvda4n1`以定义`FNAME`。程序处理数据并生成名为`vfout`的文件，其中包含所需的信息。

### 结果与讨论

辐射视角因子和面元面积被读取并输出到`vfout`。输出与预期结果一致。
