# 55.5.6 显示节点平均取向







对于线型和带状轮廓图，元素结果会从元素质心或积分点外推并有条件地平均到节点，以创建图形。您可以使用 **结果选项** 对话框来控制 Abaqus/CAE 如何对模型应用平均。一个选项允许您在平均每个节点的结果后计算不变量（标量）并提取分量；当您选择按此顺序计算结果时，Abaqus/CAE 会将相邻元素之一的向量或张量取向应用为每个节点的取向。在 **轮廓图选项** 对话框的 **其他** 选项卡下，切换 **在平均后计算标量时显示节点平均向量/张量取向** 可在轮廓图上显示或隐藏平均取向。启用此选项后，Abaqus/CAE 会在每个节点处绘制一个三轴图标，表示所绘制向量或张量的节点平均取向。默认情况下，1 轴显示为青色，2 轴显示为黄色，3 轴显示为红色。您可以使用 ["自定义材料取向图三轴，" 第 46.4.1 节](pt05ch46s02hlb01.md) 中描述的相同过程来自定义这些三轴的外观。

如果您选择在平均节点结果之前计算标量，则取向不会被平均，也不会出现在您的轮廓图中。
