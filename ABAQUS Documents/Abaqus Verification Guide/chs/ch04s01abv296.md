# 4.1.21 UMAT 和 UHYPER





### 4.1.21 [`UMAT`](../sub/sub-link.md#sub-xsl-umat) 和 [`UHYPER`](../sub/sub-link.md#sub-xsl-uhyper)

**产品：**Abaqus/Standard  

### 测试功能

用于定义各向同性Mises塑性、Mooney-Rivlin和neo-Hookean超弹性材料特性的用户子程序。

### I. 用于各向同性Mises塑性模型的用户子程序

### 测试单元

C3D8

### 问题描述

这组验证问题测试传入[`UMAT`](../sub/sub-link.md#sub-xsl-umat)的许多变量，如材料特性、步长时间和应变增量数据。这些测试还验证用户定义的应力、与解相关的变量和雅可比矩阵被正确传递到求解过程。这些测试是["与速率无关的塑性，" 第2.2.10节](ch02s02abv148.md)中描述的测试的修改。在那些测试中定义弹塑性材料的地方，用用户定义的材料代替。正在分析的结构是由单个C3D8单元组成的立方体。

**材料：**

**弹性**

弹性模量，*E* = 200.0E3

泊松比，![](../graphics/ver_eqn00181.gif) = 0.3

**塑性**

硬化：

| 屈服应力 | 塑性应变 |
| --- | --- |
| 200. | 0.0000 |
| 220. | 0.0009 |
| 220. | 0.0029 |

### 结果与讨论

本节中的测试设置为单位尺寸单元素的均匀变形情况。因此，结果在元素内所有积分点相同。在每种情况下，本构路径以20个固定大小的增量进行积分。

### 输入文件

[umatmst3.inp](../eif/umatmst3.inp)

Mises塑性，单轴拉伸，三维实体。

[umatmst3.f](../eif/umatmst3.f)

umatmst3.inp中使用的用户子程序[`UMAT`](../sub/sub-link.md#sub-xsl-umat)。

[umatmss3.inp](../eif/umatmss3.inp)

Mises塑性，简单剪切，三维实体。

[umatmss3.f](../eif/umatmss3.f)

umatmss3.inp中使用的用户子程序[`UMAT`](../sub/sub-link.md#sub-xsl-umat)。

### II. 用于超弹性模型的用户子程序

### 测试单元

C3D8、CPE4

### 问题描述

这组验证问题主要用于测试传入[`UMAT`](../sub/sub-link.md#sub-xsl-umat)的变形梯度。以变形梯度为函数的子程序[`UHYPER`](../sub/sub-link.md#sub-xsl-uhyper)中的变量也被测试。对于二维情况，正在分析的结构是由三个重合的CPE4单元组成的单位正方形。三维情况由三个重合的C3D8单元组成的单位尺寸立方体组成。对于两种情况，第一个单元的材料特性直接用一般超弹性材料特性指定。第二个和第三个单元的材料特性分别通过用户子程序[`UMAT`](../sub/sub-link.md#sub-xsl-umat)和[`UHYPER`](../sub/sub-link.md#sub-xsl-uhyper)定义。节点的位移是规定的，因此每个单元中引起的应力将是相同的。

**材料：**

**超弹性：Mooney-Rivlin**

![](../graphics/ver_eqn00068.gif) = 80.0

![](../graphics/ver_eqn00069.gif) = 0.0

![](../graphics/ver_eqn00472.gif) = 2.013E4

### 结果与讨论

本节中的测试设置为三个单位尺寸元素的均匀变形情况。因此，结果在每个元素内的所有积分点相同。在每种情况下，本构路径以10个固定大小的增量进行积分。

超弹性，简单剪切，二维实体。

[umathrs2.f](../eif/umathrs2.f)

umathrs2.inp中使用的用户子程序[`UMAT`](../sub/sub-link.md#sub-xsl-umat)和[`UHYPER`](../sub/sub-link.md#sub-xsl-uhyper)。

![](../graphics/ver_eqn01446.gif) = –5.116E-03

### 结果与讨论
