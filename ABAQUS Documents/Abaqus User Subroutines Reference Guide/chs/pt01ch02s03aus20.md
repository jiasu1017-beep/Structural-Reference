### 概述

欧拉表面：
- 必须是三维的；
- 必须定义为模型数据；
- 可用于Abaqus/Explicit中的通用接触算法；以及
- 通过指定欧拉材料实例的名称创建。

### 什么是欧拉表面，为什么使用它们？

欧拉表面表示Abaqus/Explicit分析中特定欧拉材料实例的外表面。由于欧拉材料流过欧拉网格，它们的表面不能由简单的单元面列表定义。相反，这些表面通常位于欧拉单元内部，必须在每个时间增量中使用单元体积分数数据计算。

您可以使用欧拉表面来定义与Abaqus/Explicit通用接触算法中拉格朗日表面的特定相互作用。定义后，您可以在包含、排除和相互作用定义中引用欧拉表面。您不能组合或裁剪欧拉表面。

使用欧拉-拉格朗日接触不需要欧拉表面定义。如果您为整个模型指定"自动"接触，所有欧拉材料的外表面将自动被考虑用于接触。

#### 创建欧拉表面的优点

您可以使用欧拉表面来：
- 为涉及特定欧拉材料实例的接触相互作用分配接触属性。
- 排除欧拉材料与不太可能接触的拉格朗日体之间的相互作用，简化接触问题并降低计算成本。

### 创建欧拉表面

要创建欧拉表面，必须指定模型中存在的材料实例的名称。材料实例名称在欧拉截面中定义（请参阅["欧拉单元，" 第32.14.1节](pt06ch32s14alm59.md)）。Abaqus/Explicit计算指定材料实例的外边界，并定义对应该边界的表面。随着材料变形，表面在每个时间增量重新计算。

| **输入文件用法：** | ``` [*SURFACE](../key/key-link.md#usb-kws-msurface), TYPE=EULERIAN MATERIAL, NAME=*name* *material instance name*, ``` |
