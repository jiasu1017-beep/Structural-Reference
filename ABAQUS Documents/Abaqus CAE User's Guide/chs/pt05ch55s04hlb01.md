# 55.4.1 缩放变形





变形是变形场输出变量的值;例如,位移或速度。Abaqus/CAE通过将变形应用于未变形节点坐标来计算变形形状。您可以缩放变形以放大、缩小或以其他方式扭曲变形模型形状。例如,[图55-5](pt05ch55s04hlb01.md#viw-scaledef)左侧显示变形形状云图,右侧显示变形放大15倍的相同曲线图。

**图55-5** 显示默认和放大变形值的云图。

![](../graphics/viw-scaledef.png)

变形缩放是曲线图状态无关的;也就是说,您为所有变形、云图、符号和材料取向曲线图中的变形形状设置一次。对于大位移分析,默认缩放是1.00的均匀因子。对于小变形分析——例如,扰动分析——Abaqus/CAE缩放变形,使得最大变形是最大模型尺寸的10%。

**缩放变形的步骤:**

1. 从主菜单栏,选择 ****Options**![](../graphics/images/arrow.gif)**Common****。在出现的对话框中点击"**Basic**"选项卡。"**Deformation Scale Factor**"选项在"**Basic**"页面左下角。
2. 选择以下比例因子选项之一: - 点击"**Auto-compute**"请求Abaqus自动计算并均匀应用单个比例因子到所有X-、Y-和Z-分量变形值。 - 点击"**Uniform**"指定并均匀应用单个比例因子到所有X-、Y-和Z-分量变形值。当"**Uniform**"打开时,"**Value**"规格框变得可用。点击"**Value**"框,输入您的比例因子。 - 点击"**Nonuniform**"指定要应用于X-、Y-和Z-分量变形值的各个比例因子。当"**Nonuniform**"打开时,X-、Y-和Z-分量比例因子规格框变得可用。对于每个要缩放的分量,点击该分量(X、Y或Z)比例因子框并输入您的比例因子。
3. 点击"**Apply**"实施您的更改。变形形状根据您的规格进行更改。如果处于活动状态,[状态块](pt05ch56s03.md)会更改以显示当前变形比例因子。您的更改会在会话期间保存,并会影响所有后续变形形状曲线图。默认情况下,您的更改会在会话期间保存,并会影响所有后续变形形状曲线图。如果要为后续会话保留您应用的更改,请将它们保存到文件。更多信息,请参阅 ["为后续会话保存自定义," 第55.1.1节](pt05ch55s01s01.md)。
