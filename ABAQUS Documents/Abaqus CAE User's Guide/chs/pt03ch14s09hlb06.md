# 14.9.6 考虑几何非线性





步骤的**Nlgeom**设置决定Abaqus是否在该步骤中考虑几何非线性。对于Abaqus/Explicit步骤，**Nlgeom**设置默认开启；对于Abaqus/Standard步骤，默认关闭。

步骤序列和当前的**Nlgeom**设置决定您是否可以更改特定步骤中的**Nlgeom**设置。例如，如果Abaqus已经在考虑几何非线性，则**Nlgeom**设置对所有后续步骤都处于开启状态，您无法将其关闭。同样，您无法在线性扰动步骤期间更改**Nlgeom**设置。更多信息，请参见["线性和非线性过程，"第14.3.2节](pt03ch14s03s02.md)。

**注意：**当您创建步骤时，您可以点击**步骤编辑器**中的**基本**选项卡，并选择**开**或**关**作为**Nlgeom**设置。

**要更改现有步骤的**Nlgeom**设置：**

1. 要显示**编辑Nlgeom**对话框并在适用处更改设置，请执行以下操作之一：
   - 从主菜单栏，选择**步骤**![](../graphics/images/arrow.gif)**Nlgeom**。
   - 从主菜单栏，选择**步骤**![](../graphics/images/arrow.gif)**编辑**![](../graphics/images/arrow.gif)***步骤名称***。**步骤编辑器**出现。从**基本**选项卡上的**Nlgeom**字段中，点击![](../graphics/ico_edit.png)。
   - 从主菜单栏，选择**步骤**![](../graphics/images/arrow.gif)**管理器**。**步骤管理器**出现。从管理器底部的按钮中，点击**Nlgeom**。
2. 从**编辑Nlgeom**对话框中，点击感兴趣步骤的名称以开启或关闭该步骤的**Nlgeom**。如果步骤的**Nlgeom**已开启，则**Nlgeom**列中出现复选标记。如果步骤的**Nlgeom**已关闭，则不出现勾选标记。
3. 点击**确定**关闭**编辑Nlgeom**对话框。
