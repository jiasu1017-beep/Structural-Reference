# 47.2.1 从输出数据库历史输出读取X–Y数据









### 47.2.1 从输出数据库历史输出读取X--Y数据

您可以从输出数据库中的历史输出读取*X–Y*数据。可用数据由在分析期间保存的历史输出组成。从这些输出中，您可以选择以下内容：
- 某位置处的变量——例如，节点1处的U1（*X*方向的位移）；
- 感兴趣的步或多个步；
- 这些步内的帧模式——例如，每10帧；以及
- 时间基准。

然后，Abaqus/CAE会在您指定的帧处从输出数据库读取*X–Y*数据对。如果您选择多个变量，默认情况下，Abaqus/CAE会读取数据对并为每个变量创建单独的*X–Y*数据对象。当保存历史输出变量时，您可以选择先对数据执行运算；然后Abaqus/CAE将运算结果保存为新的*X–Y*数据对象。

对于基于时间的分析，*X*值取为从分析开始的总时间，或者在重启分析的情况下，取为从上次继续分析开始的总时间。或者，您可以使用步时间作为时间基准。*Y*值取为指定位置处该时间点的变量值。

Abaqus/Standard分析的历史输出在向输出数据库写入10个增量后可用；Abaqus/Explicit分析的历史输出在向输出数据库写入100个增量后或触发基于CPU时间的Abaqus/Explicit心跳后（以先发生者为准）可用。

**要从输出数据库历史输出读取X–Y数据：**

1. 找到**历史输出**选项。从主菜单栏选择****工具**![](../graphics/images/arrow.gif)**XY数据**![](../graphics/images/arrow.gif)**创建****，然后选择**ODB历史输出**。**提示：**您也可以通过从主菜单栏选择****结果**![](../graphics/images/arrow.gif)**历史输出****来指定历史输出。出现**历史输出**对话框。**注意：****历史输出**对话框列出了分析期间保存的所有历史输出，给出了保存变量的变量名和位置。如果此列表为空，则历史输出不可用。
2. 如果尚未选择，点击**变量**选项卡。选择要读取的一个或多个变量。变量及其位置按字母顺序列出。默认是读取列表中的第一个条目。有关在对话框中选择多个项目的更多信息，请参见["从列表和表格中选择多个项目，" 第3.2.11节](pt01ch03s02s11.md)。如果您的输出数据库包含许多历史输出变量，可以使用过滤器减少显示的输出变量名称数量。点击**名称过滤器**字段旁边的![](../graphics/ico_tipSmall.png)查看有效过滤语法的示例。指定的变量和位置高亮显示。
3. 点击**步/帧**以更改Abaqus/CAE从中读取*X--Y*数据的步、帧或时间基准。有关更多信息，请参见["激活和停用步和帧，" 第42.4.1节](pt05ch42s02s01.md)。
4. 如果数据包含复数，可以使用**结果选项**对话框选择显示形式。有关Abaqus/CAE中复数形式的更多信息，请参见["控制复数结果的形式，" 第42.6.9节](pt05ch42s04hlb05.md)。
5. 要评估和显示数据，点击**绘图**。当前视口中出现*X--Y*图。该图表示您在对话框中配置的数据，Abaqus认为这是临时数据，无论您是否点击**另存为**保存它。
6. 要保存您配置的数据，点击**另存为**。任何复数数据都保存为当前的复数形式，而不是完整的复数。**注意：**要绘制保存的*X--Y*数据，请从主菜单栏选择****工具**![](../graphics/images/arrow.gif)**XY数据**![](../graphics/images/arrow.gif)**绘图****，然后从弹出菜单中选择*X--Y*数据。在出现的**将XY数据另存为**对话框中，您可以在保存之前对数据执行任何数学、三角、对数、指数或其他运算（有关更多信息，请参见["保存X--Y数据对象，" 第47.2.7节](pt05ch47s02hlb07.md)）。
7. 完成后，点击**关闭**以关闭**历史输出**对话框。
