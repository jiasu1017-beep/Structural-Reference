# L







### Lagrangian analysis

一种有限元技术，其中材料固定在元素内，材料随元素网格变形。
更多信息：- [「Eulerian 分析，」Abaqus Analysis User's Guide 第 14.1.1 节](../usb/usb-link.md#usb-anl-aeuleriananalysis)

- [第 28 章，「Eulerian 分析，」Abaqus/CAE User's Guide](../usi/usi-link.md#usi-adv-eulerian)

### LEFM（linear elastic fracture mechanics）

一种裂纹扩展技术，假定材料是弹性的或材料承受小规模屈服。该技术适用于脆性断裂分析。
更多信息：- [「基于线性弹性断裂力学（LEFM）和虚节点原理的移动裂纹建模」中的「使用扩展有限元方法将不连续性建模为富集特征，」Abaqus Analysis User's Guide 第 10.7.1 节](../usb/usb-link.md#usb-anl-aenrichment-lefm)

### load

任何力；力矩；集中或分布牵引；通量；或者更一般地说，影响——包括施加在结构或物体上的预定义场——导致变形、位移或使结构从卸载状态改变。
更多信息：- [「施加载荷：概述，」Abaqus Analysis User's Guide 第 34.4.1 节](../usb/usb-link.md#usb-prc-ploading)

- [第 16 章，「The Load module，」Abaqus/CAE User's Guide](../usi/usi-link.md#usi-lbi)

### load case

在一组负载、边界条件和基础运动中，用于定义模型中步骤内特定载荷条件的内容。可以在单个步骤中定义多个载荷案例。
更多信息：- [「多载荷案例分析，」Abaqus Analysis User's Guide 第 6.1.4 节](../usb/usb-link.md#usb-anl-amultiloadcase)

- [第 34 章，「Load cases，」Abaqus/CAE User's Guide](../usi/usi-link.md#usi-adv-loadcase)

### Load module

Abaqus/CAE 模块，用于定义预设条件（如载荷、边界条件和预定义场）。
更多信息：- [第 16 章，「The Load module，」Abaqus/CAE User's Guide](../usi/usi-link.md#usi-lbi)
