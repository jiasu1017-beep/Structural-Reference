# 42.5.5 选择符号场输出变量





您可以为符号图选择向量或张量变量分量，并可选择选择要显示的分量；这个变量称为符号场输出变量。

当选择输出数据库时，Abaqus/CAE默认列出当前[步骤和帧](pt05ch42s01hlb01.md)的[输出数据库](pt02ch09s06hlb02.md)中所有可用的向量和张量变量供您选择；在向量变量符号图中显示合成值，在张量变量符号图中显示所有主分量。描述左侧的星号表示该变量包含复数结果。

当选择当前模型数据库中的模型时，Abaqus/CAE默认列出模型当前步骤中所有可用的载荷、预定义场、边界条件和相互作用供您选择。所有这些可选项目的名称前都有括号中的字母来区分类别：**(L)**表示载荷，**(P)**表示预定义场，**(B)**表示边界条件，**(I)**表示相互作用。

使用**Field Output**对话框中的**Symbol Variable**选项来选择您想要的变量和特定分量。有关单个输出变量标识符的信息，请参见["输出变量，" Abaqus Analysis User's Guide第4.2节](../usb/usb-link.md#usbovsect)。

在符号图中显示向量变量时，您可以从以下选项中选择：
- 显示表示变量合成值的箭头。例如，总位移的符号图出现在[图42-3](pt05ch42s03hlb05.md#viw-sym1)的左侧。
- 显示表示变量特定分量值的箭头。例如，1方向位移的符号图出现在[图42-3](pt05ch42s03hlb05.md#viw-sym1)的右侧。

**图42-3**显示总位移（左）和1方向位移（右）的符号图。

![](../graphics/viw-sym1.gif)

类似地，在符号图中显示张量变量时，您可以从以下选项中选择：
- 显示表示变量每个主分量的箭头。例如，最大、中间（mid）和最小主应力的符号图出现在[图42-4](pt05ch42s03hlb05.md#viw-sym3)的左侧。
- 显示表示变量特定主分量的箭头。例如，仅显示最大主应力的符号图出现在[图42-4](pt05ch42s03hlb05.md#viw-sym3)的右侧。

**图42-4**显示所有三个主分量（左）的符号图，以及仅显示最大主应力（右）的符号图。

![](../graphics/viw-sym3.png)

- 显示表示变量每个直接分量的箭头。例如，所有直接分量的符号图出现在[图42-5](pt05ch42s03hlb05.md#tensor-direct)的左侧。
- 显示表示变量特定直接分量的箭头。例如，仅显示S22的符号图出现在[图42-5](pt05ch42s03hlb05.md#tensor-direct)的右侧。

**图42-5**显示所有直接分量（左）的符号图，以及仅显示S22（右）的符号图。

![](../graphics/usv-sym-tensor-direct.png)

**选择符号场输出变量：**

1. 找到控制符号场输出变量的选项。从主菜单栏，选择****Result**![](../graphics/images/arrow.gif)**Field Output****。在出现的对话框中点击**Symbol Variable**选项卡。**Symbol Variable**选项出现。**提示：**您也可以通过点击任何出现此按钮的对话框中的**Field Output**按钮来访问这些选项。要查看列出变量的完整描述，请通过拖动角落来增加对话框的宽度。
2. 要控制**Name**和**Description**列表中显示哪些变量：
   1. 切换**List only variables with results**以按变量的存储位置限制显示列表。限制列表有助于您通过例如仅显示积分点量来选择变量。当**List only variables with results**开启时，下拉菜单中会出现筛选选项。
   2. 点击**List only variables with results**箭头显示筛选选项。
   3. 点击**Name**和**Description**列表中要包含的变量位置的文本。文本出现在**List only variables with results**框中，**Name**和**Description**列表会刷新以仅包含具有该位置的变量。
3. 从**Name**和**Description**列表中，点击您想要的分析变量名。列表中描述左侧的星号表示该变量包含复数结果。选中的变量高亮显示。对话框底部的**Vector Quantity**、**Tensor Quantity**和**Component**列表会刷新以分别显示可用的向量量、张量量和分量。
4. 如果您要创建向量变量符号图，请选择要绘制的分量。
   - 选择**Resultant**显示表示变量合成值的箭头。
   - 选择**Selected Component**及您想要显示其特定分量的分量，以显示表示变量特定分量的箭头。
5. 如果您要创建张量变量符号图，请选择要绘制的分量。
   - 选择**All Principal Components**显示表示所有可用主分量的箭头。
   - 选择**Selected Principal Component**及您想要显示的分量，以仅显示表示特定主分量的箭头。
   - 选择**All Direct Components**显示表示所有三个直接分量的箭头。
   - 选择**Selected Direct Component**及您想要显示的直接分量，以仅显示表示特定直接分量的箭头。
6. 点击**Apply**实现您的更改。当前视口中的符号图会更改以显示您指定的分析变量的值。如果处于活动状态，图例和状态块中的文本会更改以标识与图关联的变量。有关图例和状态块的更多信息，请参见["自定义图例，" 第56.1节](pt05ch56s01.md)和["自定义状态块，" 第56.3节](pt05ch56s03.md)。您的更改将保存到会话结束。
