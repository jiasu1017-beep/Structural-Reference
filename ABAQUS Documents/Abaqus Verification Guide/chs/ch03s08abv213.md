# 3.8.10 热传递子模型





**产品：**Abaqus/Standard  Abaqus/Explicit  

### 测试的功能

子模型功能应用于Abaqus/Standard中的热传递单元和Abaqus/Explicit中的耦合温度-位移单元。在Abaqus/Explicit分析中，热膨胀系数设置为零，并使用虚拟机械特性，因为只关注热响应。三个全局模型网格取自单元片测试，第四个是规则网格。在Abaqus/Standard中测试了稳态和瞬态程序，而在Abaqus/Explicit中，所有模拟都使用动态温度-位移程序。

### I. DC2D4、CPE4RT、CPS4RT单元

### 问题描述

**模型：**

几何取自[ec24dfp4.inp](../eif/ec24dfp4.inp)中的片测试。

全局模型在x-y平面中的尺寸为0.24×0.12，厚度为1.0。子模型的一边沿全局模型的右侧面，而子模型的其他三边完全位于全局模型内部。

**材料：**

| 传导率 | 4.85×10^-4 |
| --- | --- |
| 密度 | 0.283 |
| 比热 | 0.116 |

**载荷：**

均匀膜，参考汇温度为75，膜系数为0.103，施加在全局模型的左边缘。分别为48和60的节点温度施加在全局模型的右下节点和右上节点。

### 结果与讨论

执行瞬态热传递分析。子模型中驱动节点的节点温度从全局模型结果正确插值。

### II. DC2D6、CPE6MT、CPS6MT单元

### 问题描述

**模型：**

全局模型在x-y平面中的尺寸为4×3，厚度为1.0。子模型占据全局模型的右上角。

**材料：**

| 传导率 | 4.85×10^4 |
| --- | --- |
| 密度 | 0.283 |
| 比热 | 0.116 |

**载荷：**

在整个全局模型和子模型上施加0.3的体通量。分别为200和400的节点温度规定在全局模型的左边缘和底边缘沿线。

### 结果与讨论

在Abaqus/Standard中执行稳态热传递分析。在Abaqus/Explicit中，执行瞬态分析一段达到稳态 regime 的时间。子模型中驱动节点的节点温度从全局模型结果正确插值。在[图3.8.10-1](ch03s08abv213.md#verheatsubmodel-contours)中，子模型的温度等值线图覆盖在全局模型的等值线图上。温度等值线在全局模型和子模型之间的边界处匹配，表明子模型上驱动的节点温度是正确的。

**图3.8.10-1** 全局模型和子模型中DC2D6单元的温度等值线。

![](../graphics/verheatsubmodel-contours.png)
