# 76.13 控制模型照明




您可以使用 ****视图**![](../graphics/images/arrow.gif)**光源选项**** 菜单项来控制模型的照明。当使用 **着色** 渲染样式时，光源会影响模型在当前视口中的外观。您可以控制 **环境光** 的强度和模型表面的 **光泽度**。您还可以控制多达八个附加位置光源的位置和强度。光设置的组合效果可用于模拟模型表面的不同表面光洁度和光照条件。默认设置可在大多模型中提供良好的对比度以查看所有特征。

默认设置可在大多模型中提供良好的对比度以查看所有特征。对于镶嵌式、分层云图，使用强光可能会使云图小平面中的颜色变淡，并显示与图例颜色不匹配的误导性结果。如果将分层云图打印为EPS、PostScript或SVG格式的文件，这些三种格式默认使用镶嵌等值线，则打印输出中也可能出现等值线颜色的这些变化。要将分层云图打印为这些任何格式而不产生误导性的等值线颜色，请在打印前关闭着色。

**全局设置**

环境光均匀应用于整个场景，并从所有方向照亮模型。低强度环境光允许位置光源创建阴影，以区分模型中的小特征和表面轮廓与模型的其他部分。高强度环境光会消除阴影，可能使某些模型特征难以看清。

如果您的计算机图形卡支持OpenGL着色语言（GLSL），Abaqus/CAE 还会显示全局 **着色** 选项，您可以为会话启用Phong着色。与默认选项Gouraud着色相比，Phong着色在三维表面上渲染更真实的着色，但这可能会在某些系统上造成明显的性能影响。此性能影响仅在隐藏网格时发生；如果显示网格（无论是在建模还是后处理期间），Abaqus/CAE 渲染Phong光照时不会有明显的性能影响。

光泽度是模型表面的反射率，用于控制位置光源产生的镜面高光的大小。非常光的表面像镜子一样反射光——光根据光源的入射角从一个方向反射。因此，表面必须相对于光源正确定位，才能将光反射到您的视点。光泽度较低的表面更随机地反射光，因此更广范围的表面角度可以将光反射到您的视点。与高强度环境光照一样，低光泽度可能会遮挡模型中的小特征和轮廓，使其不可见。

**视点** 选项控制如何计算镜面光照效果。**无限** 视点假设从相机到每个点的方向向量为常数，用于计算点在处的反射和镜面高光。**局部** 视点根据每个点在视口中的位置为每个点计算单独的方向向量。**局部** 视点创建更真实的光照效果，但可能会降低整体性能。

**位置光源设置**

位置光源提供灯泡效果。位置光源从指定位置投射到模型上，其效果会随着您旋转模型视图而改变。位置光源与光泽度协同工作，以确定模型如何反射光。

光源与模型的距离等于相机与模型的距离。您可以通过指定它们在模型周围的半球上的 **纬度** 和 **经度** 来定位光源。您还可以指定所使用的光源 **类型**。**方向** 光源是平面光源；模型上平行表面的入射光角度将是等效的。**点** 光源是点光源；入射光的角度取决于表面或点相对于光源的位置。**点** 光源创建更真实的光照效果，但可能会降低整体性能。

您可以控制位置光源的两个不同质量：**漫反射** 强度和 **镜面反射** 强度。**漫反射** 设置控制位置光源的强度。与环境光不同，当增加位置光源的漫反射强度时，面向光源位置的表面将比其他表面更快速地变亮。**镜面反射** 设置控制从光源向视点反射的表面的亮度。您应该首先设置位置光源的位置和漫反射强度以达到所需的着色。然后您可以使用 **镜面反射** 滑块调整反射光的亮度。

**控制模型照明：**

1. 找到照明选项。从主菜单栏，选择 ****视图**![](../graphics/images/arrow.gif)**光源选项****。
2. 如果显示了 **着色** 选项，请选择 **Gouraud** 或 **Phong** 着色。
3. 从 **视点** 字段，选择 **无限** 或 **局部** 来定义视点类型。
4. 使用滑块设置所需的 **环境光** 强度。
5. 使用滑块设置模型表面所需的 **光泽度**。数字越高，表面越亮。
6. 切换 **光源** 字段中的数字以将位置光源添加到场景。
7. 要更改位置光源的设置，请点击 **光源** 字段中相应的数字选项卡。 - 从 **类型** 字段，选择 **方向** 或 **点** 来定义光源类型。 - 使用滑块设置光源位置的所需 **纬度** 和 **经度**。 - 使用滑块设置所需的 **漫反射** 强度。 - 使用滑块设置所需的 **镜面反射** 强度。
8. 点击 **默认值** 恢复所有照明到默认设置。
9. 点击 **关闭** 关闭对话框。更改在会话期间保存。要保存供将来会话使用的设置，请从主菜单栏选择 ****文件**![](../graphics/images/arrow.gif)**保存选项****（请参见["保存显示选项设置，" 第76.16节](pt07ch76hla15.md)）。
