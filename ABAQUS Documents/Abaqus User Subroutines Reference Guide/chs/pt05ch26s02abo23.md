# 26.2.1 热属性：概述

以下属性描述了材料的热行为，可用于热传递和热应力分析（参见["热传递分析程序：概述，" 第6.5.1节](pt03ch06s05abo08.md)）：

**热传导率**：当热量通过传导流动时，必须定义热传导率（["热传导率，" 第26.2.2节](pt05ch26s02abm55.md)）。

**比热**：在瞬态热传递分析以及绝热应力分析中，必须定义材料的比热（["比热，" 第26.2.3节](pt05ch26s02abm56.md)）。

**潜热**：当材料发生相变时，内部能量的变化可能很显著。可以通过为材料经历的每个相变指定潜热来定义释放或吸收的能量（["潜热，" 第26.2.4节](pt05ch26s02abm57.md)）。
