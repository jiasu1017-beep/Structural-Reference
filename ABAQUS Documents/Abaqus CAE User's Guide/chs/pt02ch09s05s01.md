# 9.5.1 重放Abaqus/CAE会话





您在Abaqus/CAE中执行的几乎所有操作都会以Abaqus Scripting Interface命令的形式自动记录到重放文件（`abaqus.rpy`）中。执行重放文件相当于重放原始操作序列，包括任何冗余过程以及您犯的任何错误和随后的更正。重放文件还包括画布操作，如创建新视口。

Abaqus/CAE保留最近五个版本的重放文件。最近版本的重放文件名为`abaqus.rpy`；它在您启动会话时创建。四个较旧的版本在文件名末尾附加了数字；数字最小的文件名表示最早的重放文件，数字最大的文件名表示第二最近的重放文件。

您可以在启动Abaqus/CAE时或在会话期间执行重放文件中的命令；但是，如果重放文件生成错误，结果可能会有所不同。

**从Abaqus执行过程**

要从Abaqus执行过程运行重放文件，请输入`abaqus cae`（或`abaqus viewer`）`replay=`*replay_file_name*.rpy``。如果执行重放文件生成错误，Abaqus/CAE会忽略该错误并继续执行重放文件中的下一个命令。因此，Abaqus/CAE始终尝试执行重放文件中的每个命令。您不能使用`replay`选项来执行带有控制流语句的脚本。有关更多信息，请参阅["Abaqus/CAE执行，"Abaqus Analysis User's Guide第3.2.6节"](../usb/usb-link.md#usb-int-dcaeproc)。

**在Abaqus/CAE会话期间**

要在会话期间运行重放文件，请从主菜单栏选择****文件（File）**![](../graphics/images/arrow.gif)**运行脚本（Run Script）****。如果重放文件生成错误，Abaqus/CAE会停止执行重放文件，并在命令区域显示错误消息。建议您从Abaqus执行过程运行重放文件。

您也可以使用Abaqus Python开发环境（Abaqus PDE）执行重放文件。重放文件中的Abaqus Scripting Interface命令必须在Abaqus PDE的内核工作区中运行。有关Abaqus PDE的更多信息，请参阅[Abaqus Scripting User's Guide第7章，"使用Abaqus Python开发环境"](../cmd/cmd-link.md#cmd-pde-using)。
