# 14.15.1 自定义通用求解控制





通用求解控制编辑器允许您修改用于收敛和时间积分精度的算法。更多信息，请参见「分析收敛控制」，Abaqus Analysis User's Guide第7.2节。

**注意：**此选项仅适用于通用Abaqus/Standard分析步骤。

**自定义通用求解控制的步骤：**

1. 从主菜单栏选择 ****Other**![](../graphics/images/arrow.gif)**General Solution Controls**![](../graphics/images/arrow.gif)**Manager****。Abaqus/CAE会显示「General Solution Controls Manager」，其中显示模型中定义的步骤以及与每个步骤关联的通用求解控制。
2. 在「General Solution Controls Manager」中，选择您感兴趣的分析步骤，然后点击「Edit」。此时会出现「General Solution Controls Editor」。
3. 选择用于指定求解控制的选项：- 选择「Propagate from previous step」如果您希望前一步骤的求解控制设置在当前步骤中保持有效。- 选择「Reset all parameters to their system-defined defaults」如果您希望编辑器中所有选项卡页面上的所有参数恢复为默认值。- 选择「Specify」如果您想为该步骤输入特定参数的新值。
