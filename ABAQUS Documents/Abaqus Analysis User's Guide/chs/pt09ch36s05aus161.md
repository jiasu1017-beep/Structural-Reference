### 概述

本节描述如何修改在 Abaqus/Explicit 中使用接触对算法定义的接触相互作用的表面属性，包括表面厚度和偏移。

### 壳、膜或刚体单元厚度以及壳或刚体偏移

要在壳、膜或刚体单元上定义表面，使得它们在分析开始时接触，在定义节点坐标时必须考虑单元厚度；否则，接触对中的表面将过盈。表面厚度和表面偏移是默认情况下从底层壳和膜单元继承的属性。对于基于刚体单元的表面，默认表面厚度和偏移对应于属于该刚体的单元定义的厚度和偏移（见 ["刚体单元，" 第 30.3.1 节](pt06ch30s03alm23.md)）。对于基于实体单元的表面，表面厚度和偏移为零。

默认情况下，基于壳、膜或刚体单元的表面的节点厚度等于周围单元的最小厚度（见 [图 36.5.2-1](pt09ch36s05aus161.md#adefsurf-cont-thick1) 和 [表 36.5.2-1](pt09ch36s05aus161.md#table-adefsurf-thick1)）。面片内的表面厚度从节点值插值；插值表面厚度永远不会超过指定的单元或节点厚度，这可能对于初始过盈量很重要。

如果为底层单元定义了空间变化的节点厚度（见 ["节点厚度，" 第 2.1.3 节](pt01ch02s01aus07.md)），则节点表面厚度可能与指定的节点厚度不完全对应（见 [图 36.5.2-2](pt09ch36s05aus161.md#adefsurf-cont-thick2) 和 [表 36.5.2-2](pt09ch36s05aus161.md#table-adefsurf-thick2) 中的节点 4）。节点表面厚度分布将比指定的节点厚度分布更分散（因为指定的节点厚度被平均以计算单元厚度，而节点表面厚度是周围单元厚度的最小值）。

下面讨论表面厚度和偏移的影响，以及修改表面厚度和避免表面偏移的方法。

#### 表面厚度和偏移的影响

在接触对算法中考虑厚度将导致表面在单元平面内超出父单元边界，延展量等于其厚度的一半。例如，这种半圆形形状的表面延伸将导致在壳边界上的节点到达相对表面之前就建立了与相对表面的接触。延伸适用于单侧和双侧表面。[图 36.5.2-3](pt09ch36s05aus161.md#adefsurf-edge-contact) 演示了这个概念。当使用通用接触算法（["在 Abaqus/Explicit 中定义通用接触相互作用，" 第 36.4.1 节](pt09ch36s04aus155.md)）时，可以避免这种"牛鼻"延伸。壳或刚体偏移对表面的影响如图 [图 36.5.2-4](pt09ch36s05aus161.md#adeform-offset1) 所示。如果存在大偏移，拐角附近可能会出现定义不当的表面，如图 [图 36.5.2-5](pt09ch36s05aus161.md#adeform-offset2) 所示。定义模型时应考虑这一点。如果偏移幅度大于任何底层壳单元边缘长度的一半，将发出警告消息。然而，在尖角处，偏移小于父单元尺寸的一半也可能导致接触表面定义不当（在这种情况下不会给出警告）。

#### 控制接触计算中表面厚度和偏移的影响

您可以控制仅在接触计算中使用的厚度和偏移；它们不影响基于表面的约束。这些设置主要用于自接触表面，因为您不能强制这些表面为零厚度，如下所述。

自接触表面不应包含比其边缘或对角线长度更厚的面片。非常大的厚度将导致节点即使在平坦自接触表面中也看起来穿透附近的面片，这是由于算法在每个面片边缘周围使用半径为接触厚度一半的半圆形管的算法使用（见 [图 36.5.2-6](pt09ch36s05aus161.md#adeform-offset4)）。

**图 36.5.2-6** 自接触表面中大厚度导致的不期望穿透。

![](../graphics/adeform-offset4-nls.png)

您可以通过单个因子 *f* 缩放表面上所有面片使用的有效厚度。或者，您可以仅调整厚度与最小边缘或对角线长度比超过指定值 *r* 的表面面片的厚度；由于面片尺寸的变化，面片厚度调整量可能在分析过程中变化。如果在初始配置中自接触表面的厚度与单元尺寸比超过 1.0，将发出错误消息，建议您调整厚度。

对于双侧表面或将参与自接触的表面，不应为 *f* 或 *r* 指定极小值，因为这些表面必须具有与面片尺寸相比较大的接触厚度。对于仅参与两个表面接触的表面，设置 *f*=0.0 是可接受的；但是，使用下面描述的方法强制零表面厚度在计算上更有效。还可以通过将比例因子 *f* 设置为零来强制执行偏移但不强制执行表面模型中的厚度。

| **输入文件用法：** | 使用以下选项通过单个因子缩放表面厚度： |
| --- | --- |
|  | ``` [*SURFACE](../key/key-link.md#usb-kws-msurface), NAME=*name*, SCALE THICK=*f* ``` 使用以下选项调整厚度与单元尺寸的比值： ``` [*SURFACE](../key/key-link.md#usb-kws-msurface), NAME=*name*, MAX RATIO=*r* ``` |

| **Abaqus/CAE 用法：** | 您不能在 Abaqus/CAE 中缩放接触表面的厚度。 |
| --- | --- |

#### 强制零表面厚度和偏移

您可以强制表面厚度和偏移为零，而不是继承底层壳、膜或刚体单元的厚度和偏移。在这种情况下，接触表面被当作参考表面（见 [图 36.5.2-7](pt09ch36s05aus161.md#adeform-no-thick)）。

**图 36.5.2-7** 具有零厚度和偏移的接触表面。

![](../graphics/adeform-no-thick-nls.png)

您不能忽略用作单侧（自）接触的接触表面的厚度。如果接触对中的一个表面是双侧表面，只能在一个表面上强制零厚度：涉及双侧表面的接触对中至少有一个表面必须具有非零厚度。强制零表面厚度的能力对于对模型厚度或偏移进行参数研究很有用，因为您可以更改厚度和偏移，而不必移动网格来控制表面之间的初始分离。

| **输入文件用法：** | ``` [*SURFACE](../key/key-link.md#usb-kws-msurface), NAME=*name*, NO THICK ``` |
| --- | --- |

| **Abaqus/CAE 用法：** | 您不能在 Abaqus/CAE 中强制表面厚度为零。 |
| --- | --- |

接触计算通常在默认的厚度和偏移处理下最准确。但是，当指定了半个原始壳厚度的壳偏移时，强制零表面厚度将给出表面一侧的准确表示。如图 [图 36.5.2-8](pt09ch36s05aus161.md#adeform-offset3) 所示，这种方法在拐角附近（尤其是在拐角的外侧）可能比强制执行表面的偏移和厚度更准确。

#### 强制零表面偏移

在需要忽略偏移影响但仍需要在接触计算中对厚度进行建模的情况下，您可以强制仅将表面偏移为零而不影响表面厚度。在这种情况下，接触表面是假想壳、膜或刚体单元的外表面，其中面位于参考表面上（见 [图 36.5.2-9](pt09ch36s05aus161.md#adeform-no-offset)）。

**图 36.5.2-9** 具有零偏移的接触表面。

![](../graphics/adeform-no-offset-nls.png)

如果偏移强制执行会导致自接触表面定义不当，则可以使用此方法（对于自接触表面必须强制执行厚度）。

| **输入文件用法：** | ``` [*SURFACE](../key/key-link.md#usb-kws-msurface), NAME=*name*, NO OFFSET ``` |

### 为接触对相互作用定义附加接触厚度

除了已经为接触对表面底层单元定义的任何单元厚度或面偏移外，您还可以为接触对相互作用指定接触偏移。对于小滑动，这包括由初始间隙定义的接触偏移（见 ["在 Abaqus/Explicit 中为接触对调整初始表面位置和指定初始间隙"中的"精确指定初始间隙值，" 第 36.5.4 节](pt09ch36s05aus163.md#usb-cni-aexpadjustsurfaces-clearance)）。指定的偏移值将被施加为分离两个表面的层的附加厚度，而不是接触对中每个表面的附加厚度。此值可以是正的或负的。此技术通常与软化行为结合使用（见 ["接触压力-过盈量关系，" 第 37.1.2 节](pt09ch37s01aus166.md)）来建模两个接触表面之间薄层的厚度。

| **输入文件用法：** | ``` [*SURFACE INTERACTION](../key/key-link.md#usb-kws-hsurfaceinteraction), PAD THICKNESS=*value* ``` |
