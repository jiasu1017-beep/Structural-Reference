# 55.2.3 选择整体填充颜色





填充颜色是 Abaqus/CAE 显示模型面的颜色。此颜色是绘图状态无关的；也就是说，您为所有未变形、变形、符号和材料取向绘图设置一次。当变形形状叠加在未变形形状上时，您还可以设置 Abaqus/CAE 将应用于未变形形状的填充颜色。填充颜色仅适用于填充和光源着色 [渲染样式](pt05ch55s02hlb01.md) 绘图；它不适用于云图绘图。

您可以为整个模型选择一种显示颜色，或者可以选择覆盖所选项目（例如特定单元）的此显示颜色。例如，您可以绿色显示模型，同时以红色显示一组单元。有关单个项目着色的更多信息，请参阅 ["自定义单个对象的显示颜色，" 第 77.8 节"](pt07ch77s08.md)。

**选择模型填充颜色：**

1. 找到通用或叠加填充颜色选项。通用填充颜色始终适用于变形形状；当未变形形状以任何绘图状态（未变形、变形、符号或材料取向）单独绘制时，它适用于未变形形状。当未变形形状与变形形状一起绘制时，叠加填充颜色适用。从主菜单栏中，选择 ****选项（Options）**![](../graphics/images/arrow.gif)**通用（Common）**** 或 **叠加（Superimpose）**；然后在出现的对话框中点击 **颜色和样式（Color & Style）** 选项卡。**填充/着色绘图中的填充颜色（Fill color in filled/shaded plots）** 选项位于 **颜色和样式（Color & Style）** 页面的中央。
2. 选择填充颜色：1. 点击 **填充/着色绘图中的填充颜色（Fill color in filled/shaded plots）** 颜色样本 ![](../graphics/usi-colorsample.png)。Abaqus/CAE 显示 **选择颜色（Select Color）** 对话框。2. 使用 **选择颜色（Select Color）** 对话框中的方法选择新颜色。有关更多信息，请参阅 ["自定义颜色，" 第 3.2.9 节"](pt01ch03s02s09.md)。3. 点击 **确定（OK）** 关闭 **选择颜色（Select Color）** 对话框。颜色样本变为所选颜色。在您点击 **绘图选项（Plot Options）** 对话框中的 **确定（OK）** 或 **应用（Apply）** 之前，新颜色不会出现在视口中。
3. 切换 **允许颜色编码选择覆盖此对话框中的选项（Allow color code selections to override options in this dialog）** 关闭，以所有填充颜色显示上述指定的颜色。将其打开以使用单个项目颜色选择覆盖此颜色。默认是打开。有关更多信息，请参阅 ["自定义单个对象的显示颜色，" 第 77.8 节"](pt07ch77s08.md)。
4. 点击 **应用（Apply）** 实施您的更改。填充颜色会更改以反映您的规范。默认情况下，您的更改将在会话期间保存，并将影响当前视口和从当前视口创建的任何新视口中的所有后续绘图。如果您希望保留为后续会话应用的更改，请将它们保存到文件。有关更多信息，请参阅 ["保存自定义以供后续会话使用，" 第 55.1.1 节"](pt05ch55s01s01.md)。





# 55.2.2 自定义透明度





透明度选项控制模型的透明程度，它们适用于除线框外的所有渲染样式绘图中的绘图。透明度是绘图状态无关的；也就是说，您为所有未变形、变形、云图、符号和材料取向绘图设置一次。当变形形状叠加在未变形形状上时，您还可以设置 Abaqus/CAE 将应用于未变形形状的透明度。

对于着色符号和材料取向绘图，如果您打开透明度，位于模型内部的符号绘图箭头和材料取向三轴更容易查看。

**自定义透明度：**

1. 找到通用或叠加 **透明度（Translucency）** 选项。通用透明度设置始终适用于变形形状；当未变形形状以任何绘图状态（未变形、变形、云图、符号或材料取向）单独绘制时，它适用于未变形形状。当未变形形状与变形形状一起绘制时，叠加透明度样式适用。从主菜单栏中，选择 ****选项（Options）**![](../graphics/images/arrow.gif)**通用（Common）**** 或 **叠加（Superimpose）**。在出现的对话框中点击 **其他（Other）** 选项卡；然后点击 **透明度（Translucency）** 选项卡。出现 **透明度（Translucency）** 选项。
2. 切换 **应用透明度（Apply translucency）** 打开以激活透明度选项。
3. 选择透明度百分比。拖动滑块到您想要的透明度百分比。值为 .00 表示透明绘制；1.00 表示不透明绘制。默认透明度是 .30。
4. 点击 **应用（Apply）** 实施您的更改。透明度会更改以反映您的规范。默认情况下，您的更改将在会话期间保存，并将影响当前视口和从当前视口创建的任何新视口中的所有后续绘图。如果您希望保留为后续会话应用的更改，请将它们保存到文件。有关更多信息，请参阅 ["保存自定义以供后续会话使用，" 第 55.1.1 节"](pt05ch55s01s01.md)。

# 55.2.1 选择渲染样式





渲染样式是 Abaqus/CAE 显示模型的方式。渲染样式是绘图状态无关的；也就是说，您为所有未变形、变形、云图、符号和材料取向绘图设置一次。您还可以设置当变形形状叠加在未变形形状上时 Abaqus/CAE 将应用于未变形形状的第二渲染样式。可能的渲染样式是 **线框（Wireframe）**、**隐藏（Hidden）**、**填充（Filled）** 和 **着色（Shaded）**；这些样式如图 [图 55–1](pt05ch55s02hlb01.md#viw-render) 所示。以下是对这些选择的说明。

**图 55–1** 显示渲染样式选项的模型。从左到右：线框、填充、隐藏和光源着色渲染样式。

![](../graphics/viw-render.png)

**线框（Wireframe）**

显示模型边；内部和外部边都可能可见。线框绘图产生类似框架的视觉效果，模型面不显示。

**填充（Filled）**

以均匀颜色"绘制"模型面。填充绘图产生实体而非类似框架的外观，其中仅外部面可见。

**隐藏（Hidden）**

显示模型中被遮挡的边不可见的线框绘图。隐藏绘图产生实体而非类似框架的外观。

**着色（Shaded）**

显示似乎有光源指向模型的填充绘图。着色绘图产生高度三维的视觉效果。

**选择绘图的渲染样式：**

1. 找到通用或叠加 **渲染样式（Render Style）** 选项。通用渲染样式始终适用于变形形状；当未变形形状以任何绘图状态（未变形、云图、符号或材料取向）单独绘制时，它适用于未变形形状。当未变形形状与变形形状一起绘制时，叠加渲染样式适用。从主菜单栏中，选择 ****选项（Options）**![](../graphics/images/arrow.gif)**通用（Common）**** 或 **叠加（Superimpose）**；然后在出现的对话框中点击 **基本（Basic）** 选项卡。**渲染样式（Render Style）** 选项变得可用。
2. 从 **渲染样式（Render Style）** 列表中，选择您想要的渲染样式。**提示：**您还可以通过点击 **渲染样式（Render Style）** 工具栏中的线框 ![](../graphics/ico_displayWire.png)、隐藏 ![](../graphics/ico_displayHidden.png)、填充 ![](../graphics/ico_displayFilled.png) 或着色 ![](../graphics/ico_displayShaded.png) 图标来为当前绘图状态（不包括显示体和叠加时的未变形形状）选择渲染样式。**注意：**如果您激活 [透明度](pt05ch55s02hlb02.md)，以着色、填充和隐藏渲染样式显示的符号和材料取向绘图更容易查看。
3. 点击 **应用（Apply）** 实施您的更改。渲染样式会更改以反映您的规范。默认情况下，您的更改将在会话期间保存，并将影响当前视口和从当前视口创建的任何新视口中的所有后续绘图。如果您希望保留为后续会话应用的更改，请将它们保存到文件。有关更多信息，请参阅 ["保存自定义以供后续会话使用，" 第 55.1.1 节"](pt05ch55s01s01.md)。
