# 11.3 许可和命令行选项





前面章节描述的启动脚本通过在命令行上将脚本名称指定为*–custom*选项的参数来运行。要启动应用程序，请输入以下命令之一：

```
abaqus cae -custom *startupScript*
abaqus viewer -custom *startupScript*
```
其中*startupScript*是应用程序启动脚本的名称，不包括文件扩展名。你负责确保脚本存在于`abaqus.aev`文件中定义的`PYTHONPATH`环境变量指定的目录中。`abaqus.aev`文件在["安装，"第11.4节"](pt06ch11s04.md)中有描述。

`abaqus`命令的第一个参数指定要检出的许可证类型。指定`abaqus cae`将检出名为`cae`的令牌，该令牌使你能够访问所有Abaqus/CAE kernel模块。指定`abaqus viewer`将检出名为`viewer`的令牌，该令牌使你只能访问可视化kernel模块。因此，如果你的应用程序需要导入除可视化模块之外的任何Abaqus模块，你必须检出`cae`令牌。
