# 3.2.8 拓扑优化执行





**产品：** Abaqus/CAE  

##### **参考**

- ["Abaqus执行程序：概述，" 第3.1.1节](pt01ch03s01abo02.md)

### 概述

使用TOSCA进行拓扑优化通过运行Abaqus执行程序并提供参数文件的名称来执行，该参数文件又引用Abaqus输入文件。您可以使用Abaqus/CAE定义您的优化模型，并使用**优化过程管理器**创建参数文件和Abaqus/CAE输入文件。

### 命令摘要

| **abaqus optimization** | [**task**](pt01ch03s02abx08.md#task1-usb-int-dtopooptimizeproc)=*parameter file* |
| --- | --- |
|  | [**job**](#job2-usb-int-dtopooptimizeproc)=*results folder* |

### 命令行选项

**task**

此选项用于指定包含用于执行优化的参数的文件。参数文件还包含与优化关联的输入文件的信息。

如果命令行上省略了此选项，系统将提示您输入此值。

**results folder**

此选项用于指定存储优化结果的文件夹的名称。

如果命令行上省略了此选项，系统将提示您输入此值。

### 示例

要使用参数文件`WeightReduction`和结果文件夹`WeightReductionResults`运行优化，请使用以下命令：

```
abaqus optimization task=WeightReduction job=WeightReductionResults
```
