# 14.13.4 配置监视器请求

使用**自由度监视器**对话框来在分析过程中监视某点的自由度。Abaqus将该点的自由度值报告给Job模块、状态文件（`*jobname*.sta`），对于Abaqus/Standard分析，还报告给消息文件（`*jobname*.msg`）。更多信息，请参见["自由度监视器请求，"第14.5.4节"](pt03ch14s05s04.md)。

**要监视自由度：**

1. 从主菜单栏，选择**输出**![](../graphics/images/arrow.gif)**自由度监视器**。**自由度监视器**对话框出现。
2. 在对话框中，开启**在整个分析过程中监视自由度**。如果**在整个分析过程中监视自由度**已开启，则对话框中的监视选项变得可用。
3. 点击**区域**字段旁边的![](../graphics/ico_selectBlue.png)，并使用以下方法之一选择要监视的点：
   - 使用现有集来定义点。在提示区域右侧，点击**点**。从出现的**区域选择**对话框中选择现有集，然后点击**继续**。
   - 使用鼠标在视口中选择一个点。（更多信息，请参见["在当前视口中选择对象，"第6.2节"](pt01ch06s02.md)。）您选择的点在视口中以红色高亮显示。
4. 在**自由度**字段中，输入您想要监视的自由度。
5. 如果您正在执行Abaqus/Standard分析，请在**每n个增量打印到消息文件**字段中，输入您希望Abaqus写入消息文件中自由度值的频率。
6. 点击**确定**保存您的请求并退出对话框。
