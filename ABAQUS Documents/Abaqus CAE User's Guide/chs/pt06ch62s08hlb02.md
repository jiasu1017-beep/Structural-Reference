# 62.8.2 在距选定平面的偏移处创建datum平面







您可以通过在模型上选择任何平面（包括另一个datum平面）并指定从选定平面的偏移来定位datum平面；您可以通过输入值或选择一个点来指定偏移。

![](../graphics/dtm-hpop-nls.png)

**在距选定平面的偏移处创建datum平面：**

1. 从主菜单栏中，选择****Tools**![](../graphics/images/arrow.gif)**Datum****。此时将出现**Create Datum**对话框。该对话框概述了您可以创建的datum几何体类型。Abaqus/CAE在提示区域中显示提示以指导您完成此过程。**提示：**您也可以使用模块工具箱中的![](../graphics/ico_dtmPlaneOffsetPlane.png)工具创建datum平面。有关工具箱中datum工具的图示，请参见["使用Datum工具集，" 第62.2节](pt06ch62s02.md)。
2. 从对话框顶部的类型列表中，选择**Plane**。**Method**列表指示您可以使用的方法来创建datum平面。
3. 从**Method**列表中，选择**Offset from plane**。
4. 从当前视口中的零件或装配中，选择一个平面。
5. 从提示区域中的按钮中，选择以下之一： ****输入值**** 1. 会出现一个箭头，指示偏移方向。 2. 如果需要，单击**Flip**反转箭头。单击**OK**接受指示的偏移方向。 3. 在提示区域的文本框中，输入距选定平面的偏移。偏移可以是正数或负数。正值沿箭头指示的方向偏移datum平面。 ****选择点**** 从当前视口中的零件或装配中，选择一个点来定义距选定平面的偏移。datum平面会出现。您可以通过从主菜单栏中选择****Feature**![](../graphics/images/arrow.gif)**Edit****并修改偏移值来修改datum平面的位置。
