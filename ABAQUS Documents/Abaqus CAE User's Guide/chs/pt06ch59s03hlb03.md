# 59.7.3 编辑通过投影点创建的附件线





您可以编辑通过投影点创建的附件线。

**编辑通过投影点创建的附件线的步骤：**

1. 从主菜单栏，选择 ****Feature**![](../graphics/images/arrow.gif)**Edit****。
   **提示：**您也可以通过单击模块工具箱中的 ![](../graphics/ico_featureEdit.png) 工具来编辑特征。

2. 选择要修改的附件线特征。您可以直接从当前视口或从模型树中选择特征。会出现**编辑附件线特征**对话框。

3. 选择确定投影向量长度的方法。
   - 选择 **Maximum projections along direction**，然后输入附件线将相交的目标面的最大数量。
   - 选择 **Maximum length of projection vector**，然后输入附件线的最大长度。

4. 如果需要，切换 **Regenerate on OK** 以控制在编辑过程完成时是否重新生成模型。
