# *CLOAD






### *CLOAD指定集中力和力矩。

此选项用于向模型中的任何节点施加集中力和力矩。[*CLOAD](ch03abk22.md) 选项还可用于在 Abaqus/CFD 分析中指定不可压缩流动的流体参考压力，并在 Abaqus/Aqua 分析中指定集中浮力、阻力和惯性载荷。

**产品：**Abaqus/Standard  Abaqus/Explicit  Abaqus/CFD  Abaqus/CAE  Abaqus/Aqua  

**类型：**历史数据 

**级别：**步骤

**Abaqus/CAE: **载荷模块

### 应用集中载荷

### **从输出数据库文件读取集中节点力的必需参数：**

FILE

将此参数设置为要从中读取数据的输出数据库文件名。文件扩展名是可选的。

### **稳态动力学分析中循环对称模型的必需参数：**

CYCLIC MODE

将此参数设置为在当前稳态动力学过程中施加载荷的循环对称模式号。

### **可选参数：**

AMPLITUDE

将此参数设置为在步骤中定义载荷大小的幅值曲线名称。

如果在 Abaqus/Standard 分析中省略此参数，则根据 [*STEP](ch18abk36.md) 选项上 AMPLITUDE 参数分配的值，参考大小将在步骤开始时立即应用或在线性分布于整个步骤（请参阅 ["定义分析，" Abaqus 分析用户指南第 6.1.2 节](../usb/usb-link.md#usb-anl-aover)）。如果在 Abaqus/Explicit 分析中省略此参数，参考大小将在步骤开始时立即应用。

FOLLOWER

如果假定载荷方向随该节点处的旋转而旋转，则包含此参数。

此参数仅应在大位移分析中使用，并且只能用于具有活动旋转自由度的节点（如梁或壳单元的节点）。

Abaqus/Aqua 分析中的集中浮力、阻力和流体惯性载荷自动被视为随动载荷，因此在这种情况下不需要此参数。

一般来说，在 Abaqus/Standard 的 [*DYNAMIC](ch04abk43.md) 和 [*STATIC](ch18abk31.md) 分析中，UNSYMM=YES 应与 [*STEP](ch18abk36.md) 选项上的 FOLLOWER 参数结合使用。在特征值分析（如 [*BUCKLE](ch02abk16.md) 或 [*FREQUENCY](ch06abk35.md)）中，UNSYMM 参数被忽略，因为 Abaqus/Standard 只能对对称矩阵执行特征值提取。

INC

仅当使用 FILE 参数时此参数才相关。

将此参数设置为将从中读取集中节点力的先前分析所选步骤中的增量数。默认情况下，集中节点力将从 STEP 参数指定的步骤的最后增量读取，如果省略 STEP 参数，则从最后步骤读取。

LOAD CASE

此参数仅适用于 Abaqus/Standard 分析。

将此参数设置为载荷工况号。此参数在 [*RANDOM RESPONSE](ch17abk07.md) 分析中使用（["随机响应分析，" Abaqus 分析用户指南第 6.3.11 节](../usb/usb-link.md#usb-anl-arandomresponse)），此时它是 [*CORRELATION](ch03abk77.md) 选项上载荷工况的交叉引用。该参数的值在所有其他过程中被忽略。

OP

设置 OP=MOD（默认）以保留现有的 [*CLOAD](ch03abk22.md)，此选项修改现有的集中载荷或定义额外的集中载荷。

设置 OP=NEW 以移除模型上所有现有的 [*CLOAD](ch03abk22.md)。可以定义新的集中载荷。

REGION TYPE

此参数仅适用于 Abaqus/Explicit 分析。

此参数仅适用于施加在自适应网格域边界上的集中载荷。如果集中载荷施加在自适应网格域内部的节点上，这些节点将始终跟随材料移动。

设置 REGION TYPE=LAGRANGIAN（默认）以将集中载荷施加到跟随材料（非自适应）的节点。

设置 REGION TYPE=SLIDING 以将集中载荷施加到可以在材料上滑动的节点。通常对节点施加网格约束以将其空间固定。

设置 REGION TYPE=EULERIAN 以将集中载荷施加到可以独立于材料移动的节点。此选项仅用于材料可以流入或流出自适应网格域的边界区域。必须沿欧拉边界区域使用网格约束法向，以允许材料流过该区域。如果没有施加网格约束，欧拉边界区域的行为将与滑动边界区域相同。

STEP

仅当使用 FILE 参数时此参数才相关。

将此参数设置为将从中读取集中节点力的先前分析的步骤号。默认情况下，集中节点力将从先前分析的最后步骤读取。

### **矩阵生成和稳态动力学分析的可选、互斥参数：**

IMAGINARY

包含此参数以定义载荷的虚部（异相）部分。

REAL

包含此参数（默认）以定义载荷的实部（同相）部分。

### 在 Abaqus/CFD 中应用流体参考压力

### **可选参数：**

AMPLITUDE

将此参数设置为在步骤中定义流体参考压力大小的幅值曲线名称。如果省略此参数，则根据 [*STEP](ch18abk36.md) 选项上 AMPLITUDE 参数分配的值，参考大小将在步骤开始时立即应用或在线性分布于整个步骤（请参阅 ["定义分析，" Abaqus 分析用户指南第 6.1.2 节](../usb/usb-link.md#usb-anl-aover)）。

OP

设置 OP=MOD（默认）以保留现有的 [*CLOAD](ch03abk22.md)，此选项修改现有的流体参考压力。

设置 OP=NEW 以移除模型上所有现有的 [*CLOAD](ch03abk22.md)。

### **用于在 Abaqus/CFD 中定义不可压缩流动流体参考压力的数据行：**

**第一行：**

根据需要重复此数据行以定义参考压力；但是，在给定流体域中仅应用最后指定的静水压力载荷。
