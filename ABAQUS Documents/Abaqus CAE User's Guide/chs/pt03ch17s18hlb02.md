# 17.18.9 更改先前网格划分区域的网格控制





如果您更改了先前网格划分区域分配的任何网格控制，该区域的网格可能会失效。 在这些情况下，当您在**Mesh Controls**对话框中单击**OK**时，会出现以下警告对话框。

![](../graphics/mgn-controlwarn-nls.png)

您可以通过单击**Delete Meshes**删除网格，或者通过单击**Cancel**保持您的网格并取消**Mesh Controls**对话框中的新设置。

您也可以通过切换**Automatically delete meshes invalidated by mesh control changes**来避免在当前会话的剩余时间内显示此警告消息。 下次尝试更改已包含网格的区域的控制时，网格将被立即删除而不会出现警告对话框。
