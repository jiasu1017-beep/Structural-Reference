# 12.12.1 定义声学介质





您可以定义声学介质来在纯声学分析或耦合声学-结构分析中建模声音传播问题。有关更多信息，请参见[Abaqus Analysis User's Guide的第26.3.1节，"声学介质"](../usb/usb-link.md#usb-mat-cacousticmed)。

**定义声学介质：**

1. 从**Edit Material**对话框的菜单栏中，选择****Other**![](../graphics/images/arrow.gif)**Acoustic Medium****。（有关显示**Edit Material**对话框的信息，请参见["创建或编辑材料，"第12.7.1节"](pt03ch12s07hlb01.md)。）
2. 显示**Bulk Modulus**标签页。
3. 切换**Use temperature-dependent data**将体积模量定义为温度的函数。**Data**表格中出现一个标为**Temp**的列。
4. 点击**Number of field variables**字段右侧的箭头以增加或减少体积模量所依赖的场变量数量。
5. 在**Data**表格中输入以下数据：
   **Bulk Modulus** 体积模量，![](../graphics/usi_eqn00338.gif)。（[FL2](../popups/usb-int-iconventions-unitsym.md)单位。）
   有关更多信息，请参见Abaqus Analysis User's Guide的第26.3.1节"声学介质"中的"定义声学介质"。
   **Temp** 温度。
   **Field *n*** 预定义的场变量。
   有关如何输入数据的详细信息，请参见["输入表格数据，"第3.2.7节"](pt01ch03s02s07.md)。
6. 如有需要，显示**Volumetric Drag**标签页，并切换**Include volumetric drag**。有关更多信息，请参见Abaqus Analysis User's Guide的第26.3.1节"声学介质"中的"体积拖曳"。如果不想包含体积拖曳数据，请跳至第11步。
7. 切换**Use temperature-dependent data**将体积拖曳数据定义为温度的函数。**Data**表格中出现一个标为**Temp**的列。
8. 点击**Number of field variables**字段右侧的箭头以增加或减少体积拖曳数据所依赖的场变量数量。
9. 在**Data**表格中输入以下数据：
   **Volumetric Drag** 体积拖曳系数。（[FTL4](../popups/usb-int-iconventions-unitsym.md)单位。）
   **Frequency** 频率。（循环/时间。）频率相关性仅在Abaqus/Standard中的频域过程期间处于活动状态。
   **Temp** 温度。
   **Field *n*** 预定义的场变量。
   有关如何输入数据的详细信息，请参见["输入表格数据，"第3.2.7节"](pt01ch03s02s07.md)。
10. 点击**OK**关闭**Edit Material**对话框。或者，您可以从**Edit Material**对话框的菜单中选择另一种材料行为进行定义（有关更多信息，请参见["浏览和修改材料行为，"第12.7.2节"](pt03ch12s07hlb02.md)）。
