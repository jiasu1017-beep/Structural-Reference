# 42.8.6 将坐标系保存到输出数据库文件





默认情况下，后处理期间定义的坐标系仅在关联的输出数据库文件关闭之前可用。如果希望这些坐标系在以后的Abaqus/CAE会话中可用，必须将它们保存到输出数据库文件中。分析期间定义的局部坐标系会自动写入输出数据库。您只能向输出数据库追加数据；您不能以任何其他方式修改文件内容（即，一旦坐标系已保存到输出数据库，就不能修改、重命名或删除）。

**注意：**默认情况下，Abaqus/CAE将输出数据库文件作为只读打开。如果您想将任何数据（如坐标系）保存到其中，必须选择用写权限打开输出数据库文件（有关详细信息，请参见["打开模型数据库或输出数据库，" 第9.7.2节](pt02ch09s06hlb02.md)）。

**将用户定义的坐标系保存到输出数据库文件：**

1. 从主菜单栏，选择****工具**![](../graphics/images/arrow.gif)**坐标系**![](../graphics/images/arrow.gif)**管理器****。此时将出现**坐标系管理器**。默认情况下，**当前会话**被切换，所定义的坐标系列在当前输出数据库的当前会话期间。
2. 选择您希望保存的用户定义坐标系，然后点击对话框右侧的**移动到ODB**按钮。如果当前输出数据库文件是用写权限打开的，所选坐标系将从当前会话列表移动到当前输出数据库列表。要查看已保存到当前输出数据库文件的坐标系列表，请切换**当前ODB**。
