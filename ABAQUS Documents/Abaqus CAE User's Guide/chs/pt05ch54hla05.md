# 54.8 报告数据值、最小值、最大值和总计





对于 X–Y 数据或场输出的表格报告，您可以包含或抑制 X–Y 数据或场输出值、列最小值和最大值摘要以及列总计。默认情况下，Abaqus/CAE 仅在 X–Y 数据报告中包含实际数据值。在场输出值报告中，默认情况下，Abaqus/CAE 包含数据值以及列摘要和总计。

对于探针值的表格报告，Abaqus/CAE 始终在报告中包含实际数据值。您可以包含或抑制任何有意义的列的列最小值和最大值摘要以及列总计。

**报告数据值、最小值、最大值和列总计：**

1. 找到 **数据（Data）** 或 **数据值（Data Values）** 选项。**对于 X–Y 数据或场输出的报告：** 从主菜单栏中，选择 ****报告（Report）**![](../graphics/images/arrow.gif)**XY**** 或 ****报告（Report）**![](../graphics/images/arrow.gif)**场输出（Field Output）****；然后在出现的对话框中点击 **设置（Setup）** 选项卡。**数据（Data）** 选项位于页面底部。**对于探针值的报告：** 从主菜单栏中，选择 ****工具（Tools）**![](../graphics/images/arrow.gif)**查询（Query）****，然后在出现的对话框中点击 **探针值（Probe values）**。出现 **探针值（Probe Values）** 对话框。将光标移到当前视口中的模型或 X–Y 绘图上探测，然后点击鼠标按钮 1 将感兴趣的值存储在对话框中。完成后，点击 **写入文件（Write to File）**。出现 **报告探针值（Report Probe Values）** 对话框；**数据值（Data Values）** 选项位于对话框底部。
2. 对于 X–Y 数据报告或场输出报告，您必须选择以下一个或多个项目包含在报告中：- 切换 **XY 数据（XY data）** 或 **场输出（Field output）** 分别包含实际的 X–Y 数据或场输出值。关闭此选项会导致报告仅由您请求的列总计或最小/最大数据组成。- 切换 **列总计（Column totals）** 包含每列的总计。- 切换 **列最小/最大（Column min/max）** 包含每列中出现的最小值和最大值的摘要。

要生成报告，请配置对话框中的剩余选项；然后点击 **应用（Apply）**（对于 X–Y 报告或场输出报告）或 **确定（OK）**（对于探针值报告）。完成后，点击 **取消（Cancel）** 关闭对话框。

# 55.1.1 保存自定义以供后续会话使用





默认情况下，绘图显示自定义仅在当前会话期间保持，但您可以将自定义保存到文件以应用于后续会话。Abaqus/CAE 提供以下方法来保留绘图显示自定义：

**保存自定义到显示选项文件**

从主菜单栏中选择 ****文件（File）**![](../graphics/images/arrow.gif)**保存显示选项（Save Display Options）**** 将所有绘图状态相关和绘图状态无关的自定义选项保存为显示选项。显示选项存储在名为 `abaqus_v6.14.gpr` 的文件中，Abaqus/CAE 在启动时自动加载此文件中的设置作为整个会话的新默认设置。对于绘图状态自定义，将您的显示选项保存到文件使您能够存储设置，如渲染样式、规定条件和各种类型基准几何体的可见性，以及网格、单元和节点标签的显示。

有关更多信息，请参阅 ["保存您的显示选项设置，" 第 76.16 节"](pt07ch76hla15.md)。

**保存自定义到 XML 文件、模型数据库或输出数据库**

从主菜单栏中选择 ****文件（File）**![](../graphics/images/arrow.gif)**保存会话对象（Save Session Objects）**** 将选定的绘图状态相关和绘图状态无关的自定义选项保存到 XML 文件、模型数据库或输出数据库。当您使用此方法时，您可以保存所有自定义选项，也可以仅保存与一个或多个绘图状态相关的自定义选项。例如，您可以将您在通用绘图选项和云图绘图选项中进行的自定义保存到文件。

如果您将自定义保存到模型数据库或输出数据库，这些设置将在您打开文件时使用。当您从 XML 文件加载自定义时，这些自定义将成为会话的新设置。您可以通过选择 ****文件（File）**![](../graphics/images/arrow.gif)**加载会话对象（Load Session Objects）**** 从文件加载设置。

有关更多信息，请参阅 ["管理会话对象和会话选项，" 第 9.9 节"](pt02ch09s08.md)。





# 55.1 绘图显示自定义概述





您可以自定义模型在未变形、变形、云图、符号和材料取向绘图中显示的方式。对于未变形和变形绘图状态，应用于模型形状的所有自定义选项都是绘图状态无关的。也就是说，Abaqus/CAE 将自定义应用于所有未变形和变形绘图，包括也显示云图、符号或材料取向的绘图。绘图状态无关选项控制渲染样式和填充颜色；单元和表面边；单元颜色；模型标签、节点符号和显示体的外观；相机移动；各种模型实体的显示；弯曲边的平滑度；无结果的单元颜色；以及模型扫略和挤出。使用主菜单栏中的 **选项（Options）** 菜单从中选择这些绘图状态无关自定义选项。

从主菜单栏中选择 ****工具（Tools）**![](../graphics/images/arrow.gif)**颜色编码（Color Code）**** 为单个单元着色。从主菜单栏中选择 ****选项（Options）**![](../graphics/images/arrow.gif)**显示体（Display Body）**** 自定义模型中所有显示体的外观。从主菜单栏中选择 ****视图（View）**![](../graphics/images/arrow.gif)**视图选项（View Options）**** 自定义相机的移动。要选择剩余的绘图状态无关选项，请从主菜单栏中选择 ****视图（View）**![](../graphics/images/arrow.gif)**ODB 显示选项（ODB Display Options）****。如果您选择为同一绘图状态叠加未变形模型形状和变形模型形状，请从主菜单栏中选择 ****选项（Options）**![](../graphics/images/arrow.gif)**叠加（Superimpose）**** 独立于变形形状自定义未变形形状。与其他绘图状态无关自定义一样，这些绘图选项也应用于所有绘图状态中的叠加绘图。

**注意：**在视口中显示模型标签和/或符号会增加刷新绘图和动画显示所需的时间。

除了绘图状态无关自定义选项外，您还可以为云图、符号和材料取向绘图状态选择绘图状态相关选项。绘图状态相关选项对于每种绘图类型都是唯一的。例如，您可以自定义云图颜色谱、符号的大小和样式以及材料取向三轴的显示。如果可以使用独立和相关绘图选项进行类似的自定义（如通用和云图选项中的模型边缘颜色），则绘图状态相关选项将覆盖独立选项。





# 55.1.2 可用的自定义选项





Abaqus/CAE 提供以下自定义选项：

**渲染样式**

渲染样式是 Abaqus/CAE 显示模型的方式。渲染样式是绘图状态无关选项；也就是说，您为所有未变形、变形、云图、符号和材料取向绘图设置一次。但是，您可以为叠加绘图中未变形形状设置不同的渲染样式值。渲染样式选项包括线框、填充、隐藏和光源着色。有关更多信息，请参阅 ["自定义渲染样式、透明度和填充颜色，" 第 55.2 节"](pt05ch55s02.md)。

**单元和表面边样式和可见性**

边样式是单元和表面边的线条样式和粗细；边可见性是 Abaqus/CAE 显示这些边的程度。边样式和可见性选项是绘图状态无关的，但您可以为叠加绘图中未变形形状设置不同的值。有关更多信息，请参阅 ["自定义单元和表面边，" 第 55.3 节"](pt05ch55s03.md)。

**单元和表面边颜色**

您可以使用绘图状态无关的 **颜色和样式（Color & Style）** 选项控制单元和表面边的整体颜色。此外，您可以使用绘图状态无关的 **颜色编码（Color Code）** 对话框自定义单个单元和表面的边缘颜色。默认情况下，颜色编码选择覆盖整体颜色。有关整体边缘着色的更多信息，请参阅 ["选择整体单元和表面边缘颜色，" 第 55.3.3 节"](pt05ch55s03hlb03.md)。要了解更多关于单个项目边缘着色的信息，请参阅 ["自定义单个对象的显示颜色，" 第 77.8 节"](pt07ch77s08.md)。单个项目边缘着色仅适用于线框和隐藏渲染样式绘图。

**单元面和表面填充颜色**

您可以使用绘图状态无关的 **颜色和样式（Color & Style）** 选项控制单元面和表面的整体填充颜色。此外，您可以使用绘图状态无关的 **颜色编码（Color Code）** 对话框自定义单个单元的面颜色。默认情况下，颜色编码选择覆盖整体颜色。有关整体填充颜色的更多信息，请参阅 ["选择整体填充颜色，" 第 55.2.3 节"](pt05ch55s02s01.md)。要了解更多关于单个项目颜色填充的信息，请参阅 ["自定义单个对象的显示颜色，" 第 77.8 节"](pt07ch77s08.md)。单个项目填充着色仅适用于填充和着色渲染样式绘图。

**模型标签和节点符号**

您可以更改单元、节点和面标签的可见性并自定义其颜色和字体，以及节点符号的颜色和样式。所有这些选项都是绘图状态无关的。有关更多信息，请参阅 ["自定义模型标签，" 第 55.5 节"](pt05ch55s05.md)。

**单元和表面法线**

您可以选择显示表示模型中单元或表面法线的箭头。箭头在所有绘图状态中都可见，您可以控制箭头大小、颜色和箭头样式。有关更多信息，请参阅 ["显示单元和表面法线，" 第 55.7 节"](pt05ch55s07.md)。

**显示体外观**

您可以自定义模型中所有显示体的渲染样式；边缘可见性、颜色和样式；填充颜色；缩放；和透明度。这些选项在应用于显示体时是绘图状态无关的。有关更多信息，请参阅 ["自定义显示体的外观，" 第 55.8 节"](pt05ch55s08.md)。

**相机移动**

您可以选择坐标系并将相机移动与该坐标系匹配。您还可以使相机跟随所选坐标系的旋转。如果您正在使用电影模式下的相机，您还可以将相机定位在模型中坐标系的原点处。有关更多信息，请参阅 ["自定义相机移动，" 第 55.9 节"](pt05ch55s09.md)。

**实体显示**

您可以控制表示各种模型实体的符号的显示，包括边界条件、连接器、坐标系和点单元。实体显示选项是绘图状态无关的。有关更多信息，请参阅 ["控制模型实体的显示，" 第 55.10 节"](pt05ch55s10.md)。

**约束显示**

您可以控制显示在 Interaction 模块中应用的分析约束；例如，绑定约束或运动耦合约束。有关更多信息，请参阅 ["在可视化模块中控制约束的显示，" 第 55.11 节"](pt05ch55s11.md)。

**常规模型显示选项**

Abaqus/CAE 提供其他几个绘图状态无关的常规模型显示选项。这些选项包括：
- **扫略/挤出（Sweep/Extrude）** 控制二维模型的三维显示。有关更多信息，请参阅 ["扫略和挤出您的模型，" 第 55.12.1 节"](pt05ch55s12hlb01.md)。
- **镜像/图案（Mirror/Pattern）** 当仅分析模型的代表性部分时，模拟完整模型的显示。有关更多信息，请参阅 ["使用镜像和图案显示您的结果，" 第 55.12.2 节"](pt05ch55s12hlb02.md)。
- **弯曲线和面（Curved Lines & Faces）** 控制二次和三次单元边和面的显示方式。有关更多信息，请参阅 ["细化弯曲边和面，" 第 55.12.3 节"](pt05ch55s12hlb03.md)。
- **无结果的单元（Elements with No Results）** 控制云图中无结果（例如，刚性表面）的单元的颜色。有关更多信息，请参阅 ["为无结果的单元着色，" 第 55.12.4 节"](pt05ch55s12hlb04.md)。
