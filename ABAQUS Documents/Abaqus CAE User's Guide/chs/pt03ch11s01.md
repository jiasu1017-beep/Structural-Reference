# 11.1 理解零件模块的作用





在Abaqus/CAE中创建零件有几种方法：
- 使用零件模块中可用的工具创建零件。
- 从包含第三方格式几何数据的文件导入零件。
- 从输出数据库导入零件网格。
- 从Abaqus输入文件导入网格化零件。
- 在装配模块中合并或切割零件实例。
- 在网格模块中创建网格化零件。

使用零件模块工具创建的零件称为本机零件，具有基于特征的表示。特征捕获您的设计意图，包含几何信息以及控制几何行为的一组规则。例如，圆形通孔是一个特征，Abaqus/CAE存储切割的直径以及它应贯穿零件的信息。如果您增加零件的尺寸，Abaqus/CAE会识别切割深度应增加，以便继续贯穿零件。

您使用零件模块创建、编辑和管理当前模型中的零件。Abaqus/CAE以有序特征列表的形式存储每个零件。定义每个特征的参数——拉伸深度、孔径、扫掠路径等——组合起来定义零件的几何。

零件模块允许您执行以下操作：
- 创建可变形、离散刚体、分析刚体或欧拉零件。零件工具还允许您编辑和操作当前模型中定义的现有零件。
- 创建定义零件几何的特征——实体、壳、线框、切割和圆角。
- 使用特征操作工具集编辑、删除、抑制、恢复和重新生成零件的特征。
- 为刚体零件分配参考点。
- 使用草图编辑器创建、编辑和管理形成零件特征轮廓的二维草图。这些轮廓可以拉伸、旋转或扫掠以创建零件几何；或者它们可以直接用于形成平面或轴对称零件。
- 使用集合工具集、分区工具集和基准工具集。这些工具集对当前视口中的零件进行操作，允许您分别创建集合、分区和基准几何。
