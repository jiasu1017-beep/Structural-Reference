**产品：** Abaqus/Standard

### 概述

非对称轴对称问题中的接触建模：
- 需要使用接触单元（ISL 或 IRS）；
- 需要在每个周向平面上使用独立的接触单元；以及
- 只能在某些周向平面上进行。

### 非对称轴对称问题中的接触建模

CAXA 或 SAXA 单元（见 ["具有非线性非对称变形的轴对称实体单元，" 第 28.1.7 节](pt06ch28s01ael06.md)，和 ["具有非线性非对称变形的轴对称壳单元，" 第 29.6.10 节](pt06ch29s06ael20.md)）用于建模最初轴对称的结构可能经历非对称变形的问题。这些非对称变形可能包括非对称接触条件。基于表面的接触功能不能用于建模此类问题；必须使用接触单元（ISL 或 IRS）。

必须为 CAXA 或 SAXA 单元中的每个周向平面创建独立的二维接触单元集。您必须指定每个接触单元集所关联的周向平面的角度 ![](../graphics/usb_eqn00123.gif)，以及与底层 CAXA 或 SAXA 单元一起使用的傅里叶模式数 *n*。

| **输入文件用法：** | 使用以下两个选项： |
| --- | --- |
|  | ``` [*INTERFACE](../key/key-link.md#usb-kws-minterface), ELSET=*element_set_name* [*ASYMMETRIC-AXISYMMETRIC](../key/key-link.md#usb-kws-masymmaxisymm), MODE=*n*, ANGLE=![](../graphics/usb_eqn00123.gif) ``` 其中 ELSET 参数指一组 ISL- 或 IRS 型接触单元。 |

#### 非对称轴对称问题中接触的限制

如果非对称轴对称问题中的周向平面旋转超过几度，Abaqus/Standard 只能正确建模 ![](../graphics/usb_eqn00123.gif)=0 和 180 周向平面上的接触条件。非对称轴对称单元具有用于周向平面旋转和面外运动的内部自由度，但这些自由度在接触单元中没有被考虑。忽略这些自由度意味着 Abaqus/Standard 将接触方向固定在初始周向平面中，并且节点的位置在接触计算中被投影回这些初始平面。如果节点从这些初始平面的旋转和运动很小，则这种方法引起的误差很小。如果很大，误差将变得非常大，使结果不现实。
