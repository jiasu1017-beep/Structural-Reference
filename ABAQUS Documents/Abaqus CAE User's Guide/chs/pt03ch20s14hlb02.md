# 20.17.1 拖动 Sketcher 对象


使用 Sketcher 工具箱中的**拖动**工具快速移动或调整 Sketcher 对象的大小,或重新定位尺寸文本。您一次只能拖动一个顶点、线条或尺寸。当选择**对齐网格**选项时,您拖动的对象或尺寸文本会在对齐网格时捕捉;有关更多信息,请参见["打开或关闭捕捉," 第 20.9.2 节](pt03ch20s09hlb02.md)。

如果您拖动顶点或线条,Abaqus/CAE 会根据应用的约束(包括尺寸和参数)以及应用于拖动操作的约束求解方法来修改已连接的 Sketcher 对象。默认情况下,使用**最小移动**约束求解方法,其中 Abaqus/CAE 移动最少数量的实体以满足约束(更多信息,请参见["自定义 Sketcher 中约束的使用," 第 20.9.10 节](pt03ch20s09hlb10.md))。

**拖动 Sketcher 对象的步骤:**

1. 从 Sketcher 工具箱的修改工具中,选择**拖动**工具 ![](../graphics/ico_sketchDragEntities.png)。有关 Sketcher 工具箱中工具的图示,请参见["Sketcher 工具," 第 20.4.1 节](pt03ch20s04s01.md)。所有顶点的句柄会出现在视口中。Abaqus/CAE 会在提示区域显示提示,引导您完成操作。
2. 将选定的顶点、边缘或尺寸文本单击并拖动到新位置。选定的对象会随光标移动,除非草图中的约束不允许所要求的移动。
3. 完成拖动对象后,执行以下操作之一: - 在 Abaqus/CAE 窗口任意位置单击鼠标按钮 2。 - 选择 Sketcher 工具箱中的任何工具。 - 单击提示区域中的取消按钮 ![](../graphics/afxI_cancel.png)。
