### 概述

Abaqus/Standard分析的诊断可用于：
- 检查模型中的初始接触条件；和
- 跟踪分析过程中接触状态的变化。

诊断信息可在多个位置获取：
- 输出数据库
- Abaqus/CAE可视化模块中的作业诊断工具
- 数据（`.dat`）文件
- 消息（`.msg`）文件

### 审查初始过闭合表面的调整

Abaqus/Standard在各种情况下执行节点位置的初始无应变调整，以消除接触过闭合（参见["在Abaqus/Standard中控制初始接触状态，" 第36.2.4节](pt09ch36s02aus142.md)，和["在Abaqus/Standard接触对中调整初始表面位置和指定初始间隙，" 第36.3.5节](pt09ch36s03aus149.md)），或消除基于表面的绑定约束表面之间的过闭合或间隙（参见["网格绑定约束，" 第35.3.1节](pt08ch35s03aus132.md)）。模型的初始配置在这些无应变调整应用后确定。关于过闭合表面调整的信息有两个来源：数据（`.dat`）文件和输出数据库（`.odb`）文件。

#### 向数据文件输出无应变调整信息

默认情况下，数据（`.dat`）文件中会提供有限数量的无应变节点调整信息。请求更详细的接触约束输出将为所有无应变调整提供信息，无论调整的节点数量如何。

| **输入文件用法：** | ``` [*PREPRINT](../key/key-link.md#usb-kws-mpreprint), CONTACT=YES ``` |
| --- | --- |

| **Abaqus/CAE用法：** | 作业模块：作业编辑器：**常规**：**预处理器打印输出**：**打印接触约束数据** |
| --- | --- |

#### 可视化无应变调整

输出变量STRAINFREE（参见["Abaqus/Standard输出变量标识符，" 第4.2.1节](pt02ch04s02abv01.md)）包含表示初始无应变调整的节点向量。默认情况下，如果Abaqus/Standard进行了任何无应变调整，此输出变量会被写入零时刻原始场输出帧的输出数据库（`.odb`）文件。在Abaqus/CAE的可视化模块中，此变量的符号图显示表示各个节点如何被调整的向量，而等值线图则显示调整量的大小分布（您必须在Abaqus/CAE的可视化模块中选择零时刻的原始输出帧，然后选择STRAINFREE输出变量）。Abaqus/Standard写入输出数据库的初始节点位置包含无应变调整的效果，因此初始配置的绘图显示调整后的节点位置。

### 审查初始接触条件

在进行分析之前，对模型进行数据检查以审查初始接触条件（参见["Abaqus/Standard、Abaqus/Explicit和Abaqus/CFD执行，" 第3.2.2节](pt01ch03s02abx02.md)）。数据检查会创建一个输出数据库，并根据模型的初始配置计算每个从表面上的COPEN（接触开口）。您可以在Abaqus/CAE的可视化模块中创建COPEN的等值线图，以检查模型装配中的过闭合表面（过闭合对应于COPEN的负值）。

此外，您可以指示Abaqus在数据检查期间将初始接触条件的详细信息打印到数据文件（默认情况下不打印此信息）。数据文件列出从表面上每个约束点的状态（打开或闭合）和间隙距离、与每个从节点或小面关联的内部生成的接触元素编号，以及接触相互作用属性的摘要。内部生成的接触元素不是用户定义的，不会出现在输入文件中，因此如果错误或警告消息引用它们，可能很难找到。数据文件中的信息可用于在模型中定位这些接触元素。

数据文件还列出模型中每个接触相互作用的关键参数。这些参数包括：
- 从表面和主表面名称；
- 相互作用属性；
- ![](../graphics/usb_eqn06367.gif) 的值（参见["在未收敛迭代中基于穿透距离控制增量大小" "Abaqus/Standard中接触建模的常见困难，" 第39.1.2节](pt09ch39s01aus184.md#usb-cni-acontacttrouble-hcrit)）；
- 主表面上的平滑度（参见["有限滑动、节点到表面公式的主表面平滑" "Abaqus/Standard中的接触公式，" 第38.1.1节](pt09ch38s01aus177.md#usb-cni-acontactpairform-smoothing)）；
- 穿透容差计算中使用的特征长度（参见["Abaqus/Standard中的接触约束 enforcement方法，" 第38.1.2节](pt09ch38s01aus178.md#usb-cni-acontactconstraints-auglagrange)中的"增广拉格朗日方法"）；
- 应用于主表面边缘的延伸率（参见["延伸主表面和滑移线，" 第36.3.8节](pt09ch36s03aus152.md)）；和
- 接触公式。

参数仅针对其适用的相互作用列出。例如，![](../graphics/usb_eqn06367.gif)、表面平滑和延伸率不用于表面到表面接触计算（包括通用接触），因此Abaqus不会报告这些参数在表面到表面相互作用中的值。

| **输入文件用法：** | 使用以下选项将初始接触条件信息打印到数据文件： |
| --- | --- |
|  | ``` [*PREPRINT](../key/key-link.md#usb-kws-mpreprint), CONTACT=YES ``` |

| **Abaqus/CAE用法：** | 作业模块：作业编辑器：**常规**：**预处理器打印输出：** **打印接触约束数据** |
| --- | --- |

### 在模拟过程中跟踪接触状态

Abaqus提供了两种在分析过程中跟踪接触相互作用状态的方法：Abaqus/CAE可视化模块中可用的诊断工具以及到数据（`.dat`）文件的接触输出。跟踪接触状态有助于您确保接触表面正确定义、对已终止的接触分析进行故障排除，并验证接触相互作用的行为是否符合实际。

Abaqus/CAE中的诊断工具提供了接触条件如何在整个模拟过程中演变的良好概览。它对于审查已终止的分析非常有用，因为它报告每次迭代中的接触变化计算。数据文件提供了更详细的接触条件总体摘要和驱动这些条件的力。但是，它仅对成功完成的增量提供输出。

#### Abaqus/CAE可视化模块中的接触诊断

Abaqus/CAE可视化模块中的诊断工具可用于以下程序类型：
- 静态应力/位移；
- 耦合热/应力；和
- 耦合孔隙流体流/应力。

诊断工具跟踪分析期间接触的所有变化。每次约束点的接触状态从闭合变为打开时，它被记录为"开口"。每次状态从打开变为闭合时，它被记录为"过闭合"。如果接触相互作用涉及摩擦效应，诊断会记录约束点何时开始沿主表面滑动（"滑移"）以及运动中的约束点何时停在主表面上（"粘滞"）。诊断工具列出涉及状态变化的约束点，并允许您在模型中突出显示约束点的位置。还显示计算的间隙或过闭合距离，当超过增广拉格朗日接触的穿透容差时报告最大穿透（参见["Abaqus/Standard中的接触约束 enforcement方法，" 第38.1.2节](pt09ch38s01aus178.md#usb-cni-acontactconstraints-auglagrange)中的"增广拉格朗日方法"）。

对于默认的接触收敛标准，诊断工具显示最大穿透误差和最大估计接触力误差；这些决定了接触条件是否已收敛（详细信息，参见["Abaqus/Standard中的严重不连续" "定义分析，" 第6.1.2节](pt03ch06s01abo05.md#usb-anl-aover-sdiconvert)）。如果您选择使用传统接触收敛标准，则不会报告这些误差值。对于涉及拉格朗日摩擦的分析，诊断显示应处于粘滞状态的点的最大滑移误差（参见["摩擦行为中的剪切应力与弹性滑移，" 第37.1.5节](pt09ch37s01aus169.md#usb-cni-afriction-slip)）。

关于使用诊断工具的详细说明，参见[Abaqus/CAE用户指南，第41章，"查看诊断输出"](../usi/usi-link.md#usv-output)。Abaqus/CAE中可用的接触诊断信息也可以打印到Abaqus消息文件。详细信息，参见["Abaqus/Standard消息文件" "输出，" 第4.1.1节](pt02ch04s01aus38.md#usb-out-ooutput-message-std)。

#### 数据文件中的接触输出

当您请求接触输出到数据文件（参见["Abaqus/Standard的表面输出" "输出到数据和结果文件，" 第4.1.2节](pt02ch04s01aus39.md#usb-out-oprintfile-surface)）时，Abaqus会在分析的每个增量中列出每个约束点的接触状态。默认情况下，还会报告每个约束点上CPRESS、CSHEAR、COPEN和CSLIP的值。

#### 示例：形成通道

接触诊断通常有助于确认模型中的相互作用是否符合实际并按预期运行。诊断还提供了一种逐节点跟踪接触状态演变的方法。在此示例中，诊断基于通道形成模型。通道由一块相当厚度的钢板（或坯料）形成。坯料用二维平面应变单元建模；成形工具（模具、夹持器和冲头）建模为解析刚体表面。模型的初始和最终配置显示在[图39.1.1-1](pt09ch39s01aus183.md#acontact-blank-original-example)中。

**图39.1.1-1** 通道形成示例模型。（为了可视化目的，坯料已被挤出。）

![](../graphics/acontact-blank-original-example-nls.png)

如果您在模型中包含旨在建立两个表面之间接触的步骤或规定条件，Abaqus/CAE中的诊断工具可以确认此建模技术的成功。在此示例中，成形过程开始前，必须在坯料、模具和夹持器之间牢固地建立接触。坯料表面节点上小但一致的过闭合表明接触条件适合开始形成通道（参见[图39.1.1-2](pt09ch39s01aus183.md#acontact-blank-step1)）。

**图39.1.1-2** 诊断确认坯料、模具和夹持器之间的接触条件。

![](../graphics/acontact-blank-step1-nls.png)

您还可以使用接触条件来审查整个成形过程中接触状态的变化。[图39.1.1-3](pt09ch39s01aus183.md#acontact-blank-slipping)描述了坯料上两个节点开始滑动的时刻。

**图39.1.1-3** 开始滑动的诊断。

![](../graphics/acontact-blank-slipping-nls.png)

此信息可用于确认摩擦或材料效应。例如，您可以对通道成形分析中的这些诊断得出以下结论：
- 如果滑动直到成形过程后期才发生，摩擦力可能将坯料保持在模具和夹持器之间。
- 由于坯料上的所有节点并非同时滑动，坯料很可能发生轻度拉伸和非均匀变形。

要更深入了解滑动节点，请参考数据文件。以下节录列出了与[图39.1.1-3](pt09ch39s01aus183.md#acontact-blank-slipping)中相同增量中坯料-模具相互作用的部分内容：

```
    NODE  FOOT-   CPRESS      CSHEAR1     COPEN       CSLIP1  
          NOTE 

      290  OP    0.000       0.000      4.1155E-07 -2.8783E-07
      295  SL   4.4632E+06 -4.4632E+05   0.000     -5.1137E-06
      300  ST   9.5643E+06 -9.3177E-05   0.000     -4.8711E-06
      305  ST   2.9421E+06 -2.7867E-05   0.000     -4.7359E-06
```

接触状态在"脚注"列中指示：打开（OP）、闭合且切向粘滞（ST）或闭合且切向滑动（SL）。在没有摩擦属性的情况下，两种接触状态是打开（OP）和闭合（CL）。

在上面的输出中，节点290是打开的；因此，接触压力变量CPRESS为零。COPEN变量报告此节点距离主表面4.1155×10⁻⁷长度单位。节点295的SL脚注表明它与主表面（模具）接触并且正在"滑动"。临界剪切应力，![](../graphics/usb_eqn06214.gif)，可以通过方程![](../graphics/usb_eqn06368.gif)确定，其中*p*是CPRESS下显示的接触压力值，![](../graphics/usb_eqn00434.gif)是接触相互作用的摩擦系数。在此模型中![](../graphics/usb_eqn00434.gif)=0.1；临界剪切应力（4.4632×10⁶×0.1=4.4632×10⁵）等于摩擦剪切应力CSHEAR1，因此节点正在滑动。对于节点300，临界剪切应力（9.5643×10⁶×0.1=9.5643×10⁵）大于摩擦剪切应力，因此节点正在粘滞。同样对于节点305。

CSLIP1变量是从节点处的总累积（积分）滑移。累积滑移和局部切向方向的详细信息在["定义Abaqus/Standard中的接触对，" 第36.3.1节](pt09ch36s03aus145.md#usb-cni-acontactpair-tangent)中的"切向结果输出"中讨论。

### 诊断已终止的接触分析

接触诊断在尝试解析已终止分析中的错误时提供了宝贵的资料。诊断让您可以审查模型接触状态的趋势、视觉识别模型中涉及接触困难的区域，并数值量化错误的严重程度。

有关在Abaqus/Standard分析中使用接触相关常见错误的更一般性讨论，请参阅["Abaqus/Standard中接触建模的常见困难，" 第39.1.2节](pt09ch39s01aus184.md)。

#### 过多的严重不连续迭代

在隐式静态接触分析中，建立接触条件是常见的困难来源。如果分析因超过最大严重不连续迭代次数而终止（参见["Abaqus/Standard中的严重不连续" "定义分析，" 第6.1.2节](pt03ch06s01abo05.md#usb-anl-aover-sdiconvert)），接触诊断可以深入了解如何解决问题。您可以绘制尝试过程中接触状态变化的数量，如[图39.1.1-4](pt09ch39s01aus183.md#usb-contact-plot)所示。

**图39.1.1-4** 尝试过程中接触状态的变化。

![](../graphics/usb-contact-plot-nls.png)

如果变化趋向于零，增加允许的严重不连续迭代次数或调整SDI转换设置可能允许Abaqus解决接触条件。如果变化不趋向于零，您需要修改模型或研究其他选项。

使用可视化工具，您可以看到模型中参与接触变化的区域。如果特定的接触对或表面区域导致大部分状态波动，您可能需要修改相关相互作用的特性。例如，对于使用小滑动跟踪方法的接触对（如果适用）通常比使用有限滑动跟踪方法的接触对更容易解决接触条件。

#### 跳动

接触诊断工具使检测模型中的跳动变得非常容易。在这种情况下，同一节点或约束出现在每次迭代的诊断摘要中，交替作为过闭合或开口。典型的跳动场景产生的诊断图由于振荡的接触状态而趋向于零，但在低数值处趋于平稳（参见例如[图39.1.1-4](pt09ch39s01aus183.md#usb-contact-plot)）。解决接触跳动问题的技术讨论在["接触模拟中的过多迭代" "Abaqus/Standard中接触建模的常见困难，" 第39.1.2节](pt09ch39s01aus184.md#usb-cni-acontacttrouble-iters)中。

#### 不现实且严重的过闭合

在审查诊断时，您可能会注意到位于收敛状态中接触区域之外的节点或约束点在未收敛迭代中出现过闭合。这些节点的报告过闭合值将显著大于接触区域内的节点的过闭合，如[图39.1.1-5](pt09ch39s01aus183.md#usb-contact-excessive-overclosure)中突出显示的约束点所示。

**图39.1.1-5** 一个约束点处的过闭合显著高于其他约束点处的过闭合。

![](../graphics/usb-contact-excessive-overclosure-nls.png)

这表示模型中存在物理或数值不稳定。您应该采取措施在继续模拟之前更牢固地建立接触，或向模型添加某种稳定化（参见["解决非线性问题，" 第7.1.1节](pt03ch07s01aus49.md)；["缓冲器，" 第32.2.1节](pt06ch32s02alm38.md)；和["Abaqus/Standard中接触控制的调整，" 第36.3.6节](pt09ch36s03aus150.md#usb-cni-acontacttrouble-stabilize)中的"接触问题中刚体运动的自动稳定化"）。在某些情况下，使用较小的增量可以有时使获得解成为可能。

#### 不收敛的力方程

接触诊断并不总是涉及严重不连续迭代。定义不良的接触可能导致分析中力方程的不收敛（参见[图39.1.1-6](pt09ch39s01aus183.md#usb-contact-enclosure-residuals)）。

**图39.1.1-7** 在力方程不收敛区域中的两个表面。

![](../graphics/usb-contact-overhang-nls.png)

诊断突出显示从表面周边上的"问题节点"。在这个节点附近仔细查看，发现从表面网格太粗糙。从表面周边的节点接触到主表面，但下一排节点"悬挂在"主表面的边缘上。如果此接触对使用节点到表面接触离散化，主表面可以几乎无阻力地穿透节点之间的从表面。这种穿透可能导致诊断中看到的力方程不收敛。

主表面可以自由穿透从表面的任何情况都可能阻止分析收敛。可能的解决方案包括：
- 切换主从分配；
- 使用表面到表面离散化（但是，如果不细化粗糙的从网格而使用表面到表面离散化，即使分析确实收敛，也可能产生不准确的结果）；或
- 细化从表面上的网格。
