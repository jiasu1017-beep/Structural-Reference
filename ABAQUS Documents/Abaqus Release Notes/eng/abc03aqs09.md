# 3.9 Switching the context for model instances in Abaqus/CAE







**Product: **Abaqus/CAE  

**Benefits: **You can now use the Model Tree to switch the context for model instances and their child part instances, making it easier to access the original model or part for editing.

**Description: **You can select a model instance or a child part instance from the Model Tree and switch the context to the original model or part from which the selected instance was created.

**Abaqus/CAE Usage: **
```
Assembly module or Mesh module:
    Select model or child part instance from the Model Tree, then click mouse button 3 and 
    select **Switch to part/model context** from the menu. 
```
