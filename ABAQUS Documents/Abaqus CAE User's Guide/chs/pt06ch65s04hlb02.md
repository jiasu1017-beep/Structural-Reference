# 65.4.2 重命名特征







您可以通过在模型树中的特征上点击鼠标按钮 3 并从出现的菜单中选择 **Rename**（重命名）来重命名特征。

**重命名特征：**

1. 在模型树中的特征上点击鼠标按钮 3，然后从出现的菜单中选择 **Rename**（重命名）。出现 **Rename Features**（重命名特征）对话框。
2. 输入特征的新名称。
3. 点击 **OK**（确定）完成重命名特征。模型树显示新名称。
