# 8.1 理解打印





Abaqus/CAE 允许你获取一个或多个视口及其内容的快照，并将图像直接发送到打印机或文件以供以后使用；例如，包含在演示文稿中、嵌入打印报告中或显示在 HTML 文档中。附加选项允许你选择结果图像中视口的外观，以及图像的颜色、分辨率和大小。

本节描述在将输出发送到打印机或文件之前应理解的基本概念。涵盖以下主题：
