### 概述

低循环疲劳损伤模型可与损伤演化模型结合使用，以模拟疲劳裂纹形成后的裂纹扩展。该组合模型：
- 首先预测疲劳裂纹形成（基于应变-life 方法）；
- 然后模拟裂纹扩展（基于损伤演化）；以及
- 最终导致元素删除。

### 建模方法

组合建模方法包括：
1. 定义低循环疲劳损伤模型
2. 定义损伤起始准则
3. 定义损伤演化定律
4. 元素删除

| **输入文件用法：** | ``` [*FATIGUE](../key/key-link.md#usb-kws-mfatigue) [*DAMAGE INITIATION](../key/key-link.md#usb-kws-mdamageinitiation) [*DAMAGE EVOLUTION](../key/key-link.md#usb-kws-mdamageevolution) ``` |
| --- | --- |

### 与塑性模型的结合

组合模型可与各向同性硬化塑性模型结合使用：

| **输入文件用法：** | ``` [*PLASTICITY](../key/key-link.md#usb-kws-mplasticity) [*FATIGUE](../key/key-link.md#usb-kws-mfatigue) [*DAMAGE INITIATION](../key/key-link.md#usb-kws-mdamageinitiation) [*DAMAGE EVOLUTION](../key/key-link.md#usb-kws-mdamageevolution) ``` |
| --- | --- |

### 元素

组合损伤模型可与 Abaqus/Standard 和 Abaqus/Explicit 中的连续体元素和壳元素一起使用。
