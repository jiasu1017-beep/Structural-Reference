### 概述

Abaqus/Standard提供空腔辐射功能，用于建模由于空腔中辐射导致的热传递效应。此空腔辐射功能：
- 可包含在无变形的热传递分析问题中（["非耦合热传递分析，" 第6.5.2节](pt03ch06s05at18.md)，和["耦合热-电分析，" 第6.7.3节](pt03ch06s07at22.md)）；
- 提供用于二维、三维和轴对称情况；
- 考虑空腔内的对称性、表面阻塞和表面运动；和
- 可包括封闭空腔或开放空腔（意味着某些辐射发生在外部介质中）。

空腔辐射方程是非对称的；因此，自动调用包含空腔辐射的模型中的非对称矩阵存储和求解方案（参见["空腔辐射，" Abaqus理论指南第2.11.4节](../stm/stm-link.md#stm-anl-cavradiation)，和["定义分析，" 第6.1.2节](pt03ch06s01abo05.md)）。每个空腔定义一个涉及封闭中表面之间几何关系的视角因子矩阵。这些矩阵可能在分析过程中多次更新（由于空腔中移动的表面）。因此，大型空腔辐射问题可能在计算上昂贵。相反，您应该考虑使用：
- 间隙辐射（参见["热接触属性，" 第37.2.1节](pt09ch37s02aus174.md)）用于建模紧密间隔表面之间的辐射；
- 平均温度辐射条件用于建模近似等温且发射率恒定、不需要阻塞或反射考虑的封闭（参见["热载荷，" 第34.4.4节](pt07ch34s04aus123.md)）；或
- 用于视角因子并行计算和辐射热传递方程求解的空腔并行分解（参见下面的["并行分解大型空腔"](pt09ch41s01aus187.md#usb-cni-acavityradiation-parallel)）。

### 定义空腔辐射问题

由于空腔辐射效应仅在热传递和耦合热-电程序中计算，因此可以包含这些效应的唯一热-应力分析类型是顺序耦合热-应力分析（参见["顺序耦合热-应力分析，" 第16.1.2节](pt04ch16s01at39.md)）。此外，除非您允许空腔并行分解（参见下面的["并行分解大型空腔"](pt09ch41s01aus187.md#usb-cni-acavityradiation-parallel)"），否则Abaqus/Standard中有16,000个节点和小面的软件限制。

#### 模型定义

当您定义空腔辐射问题的模型时，必须：

1. 定义空腔中的所有表面（参见["定义表面"](pt09ch41s01aus187.md#usb-cni-acavityradiation-definesurfaces)）；
2. 定义每个表面的辐射属性（即发射率）和物理常数（参见["定义表面辐射属性"](pt09ch41s01aus187.md#usb-cni-acavityradiation-surfprops)）；和
3. 从表面构建空腔（参见["构建空腔"](pt09ch41s01aus187.md#usb-cni-acavityradiation-constructcavity)）。

#### 历史定义

在空腔辐射分析的第一步中，您必须将视角因子定义与每个空腔关联，该定义控制空腔视角因子的计算。然后您可以：

1. 定义空腔对称性（如果有）（参见["定义空腔对称性"](pt09ch41s01aus187.md#usb-cni-acavityradiation-cavsymm)）；
2. 规定表面运动（参见["在空腔辐射分析期间规定运动"](pt09ch41s01aus187.md#usb-cni-acavityradiation-prescribemotion)）；
3. 定义边界条件，如温度和强制对流（参见["边界条件"](pt09ch41s01aus187.md#usb-cni-acavityradiation-bc)）；
4. 控制每个步骤中的空腔辐射和视角因子计算（如果未在步骤中重新定义，则使用前一步骤的规范；参见["在分析期间控制视角因子计算"](pt09ch41s01aus187.md#usb-cni-acavityradiation-viewfactorcalc)）；
5. 请求将热传递变量输出到数据和结果文件（参见["请求表面变量输出"](pt09ch41s01aus187.md#usb-cni-acavityradiation-outputvars)）；和
6. 请求辐射视角因子矩阵的输出（参见["将视角因子矩阵写入结果文件"](pt09ch41s01aus187.md#usb-cni-acavityradiation-viewfactoroutput)）。

如果您的分析中包含上述任何一项，必须在热传递或耦合热-电步骤定义中定义它们。

### 定义表面

#### 限制

与空腔辐射关联的表面除了["基于单元的表面定义，" 第2.3.2节](pt01ch02s03aus17.md)中概述的一般表面定义限制外，还须遵守以下限制：
- 表面不能重叠，因为会导致相关属性定义和阻塞规范中的歧义。
- 表面只能用于一个空腔定义（同一表面不能出现在两个不同的空腔中）。

此外，三维四边形小面应尽可能接近平面；否则，视角因子计算的质量将受到影响。

#### 控制虚假空间振荡

每个小面的辐射通量基于该小面上节点温度的平均值计算（参见["空腔辐射，" Abaqus理论指南第2.11.4节](../stm/stm-link.md#stm-anl-cavradiation)）。然后将该辐射通量值按比例分配给每个节点，其比例与其面积成正比。因此，网格必须足够精细，使得跨单元的温度差异很小。否则，高于小面平均温度的节点处计算的通量将过低，而低于平均温度的节点的通量将过高。这往往会导致空间振荡解。可以通过在高温梯度附近减小单元尺寸来消除此效应。

### 定义表面辐射属性

空腔辐射问题本质上是非线性的，因为辐射通量依赖于小面温度的四次方。此外，通过将发射率描述为温度的函数，可以引入非线性。

您必须为定义发射率的表面属性指定名称。

##### 创建具有多个开口或复杂环境条件的空腔

开放空腔定义允许具有通向具有单一恒定温度值的环境的单个开口的空腔。如果空腔有多个开口或环境温度不是恒定的，您应该以不同方式对周围环境进行建模。

您应该用单元封闭任何空腔开口，并在这些单元上规定外部介质的温度。由于空腔现在是封闭的，您不应在空腔定义中指定环境温度。您用于封闭单元的温度定义提供环境温度，并且允许您在空腔开口处指定不同的温度，包括可变温度。建模外部介质的单元不应与空腔单元共享节点（以便它们之间不会发生传导）。由外部介质单元定义的表面应具有1的发射率。

### 并行分解大型空腔

默认情况下，Abaqus/Standard在视角因子矩阵计算和辐射热传递方程求解（参见["空腔辐射，" Abaqus理论指南第2.11.4节](../stm/stm-link.md#stm-anl-cavradiation)）中使用单个工作线程。此方法对于由数百个小面组成的较小空腔是稳健且有效的，但对于由数千个小面组成的大型空腔，它变得低效且计算昂贵。此外，这些空腔的内存需求对于单个计算节点来说可能非常大（视角因子矩阵的大小是小面数量的平方）。在这些情况下，您应该考虑允许Abaqus/Standard在视角因子计算和辐射热传递方程求解期间将空腔分解到所有CPU。

| **输入文件用法：** | 使用以下选项激活空腔并行分解： |
| --- | --- |
|  | ``` [*CAVITY DEFINITION](../key/key-link.md#usb-kws-mcavitydef), NAME=*cavity_name*, PARALLEL DECOMPOSITION=ON ``` |

| **Abaqus/CAE用法：** | Abaqus/CAE不支持空腔并行分解。 |
| --- | --- |

#### 并行求解辐射热传递方程

当启用空腔并行分解时，Abaqus/Standard使用迭代求解技术来获得辐射热通量。此技术基于Krylov方法，采用预处理器，并且仅使用基于MPI的并行化（详细信息参见["Abaqus/Standard中的并行执行，" 第3.5.2节](pt01ch03s05aus33.md)）。此迭代技术仅用于求解空腔辐射方程，不需要用户干预。您仍可以选择使用迭代或直接稀疏求解器来求解热传递有限元方程。

#### 具有分解空腔的模型的收敛性

无论是否允许并行分解，都会求解精确的空腔辐射方程；但是，当并行分解处于活动状态时，Abaqus/Standard可能需要更多迭代来获得解。更慢的收敛速度来自于对Jacobian（辐射通量的线性化）的近似，该近似基于照射（表面发出的任何部分除外）的微小变化。具有低发射率的表面和稳态分析涉及的模型可能特别受影响。如果您遇到并行分解空腔的收敛问题，您可以考虑
- 将分析从稳态更改为瞬态（["非耦合热传递分析，" 第6.5.2节](pt03ch06s05at18.md)）；或
- 允许更多每次时间增量的求解器迭代（["非线性问题的收敛标准，" 第7.2.3节](pt03ch07s02aus51.md)）。

#### 具有分解空腔的模型的运动约束

运动约束（例如，耦合约束、线性约束方程、多点约束或基于表面的绑定约束）可以应用于允许并行分解的空腔所属的任何节点或表面。但是，节点或表面必须是约束定义中的独立（主）节点或表面。

### 定义空腔对称性

利用几何对称性可以减少计算模型大小和模拟时间。您无需对对称装配中的所有零件或组件进行建模，而是可以对较小的重复组件进行建模，并在空腔辐射相互作用的定义中考虑对称性。在Abaqus/Standard中，定义了对称性的空腔定义考虑每个空腔小面与空腔及其所有对称图像之间所有小面的辐射相互作用。Abaqus/Standard不检查使用空腔对称性创建的模型在物理上是否真实。您必须仔细检查输入和结果以确保创建了有效模型。

您必须为每个辐射对称性定义分配一个名称，以供辐射视角因子定义引用。辐射视角因子定义和相应的辐射对称性定义必须出现在同一步骤中。

可以定义循环、周期和/或反射对称性，如下所述。

#### 周期对称性

您可以沿给定方向通过周期重复来定义空腔对称性。从物理上理解，周期对称性是在周期间隔处相同图像的无限重复。数值上，周期对称性必须由有限数量的重复图像表示。您可以定义数值计算中使用的重复次数*n*。

周期对称性将导致由用户定义空腔加上两倍*n*个相似图像组成的空腔，因为周期对称性假定在正方向和负方向都适用。默认情况下，*n*=2。

虽然对称性不会增加视角因子矩阵的大小，但它们确实使计算更加昂贵。因此，应最小化重复次数，但*n*的值应足够大以准确计算视角因子。输出变量VFTOT可用于检查对称性暗示的闭合量。（参见下面的["控制视角因子计算的准确性"](pt09ch41s01aus187.md#usb-cni-acavityradiation-viewfactor-vtol)。）用于定义空腔辐射视角因子矩阵的周期对称性不会自动在热传递分析中施加对称条件。可能需要在周期对称平面上的节点处对温度和载荷条件施加适当的约束，以从底层热传递分析中获得有意义的解。

### 在空腔辐射分析期间规定运动

在许多空腔辐射问题（如制造序列模拟）中，视角因子由于表面在分析过程中移动而发生变化。您可以在热传递或耦合热-电分析期间规定表面运动。

规定的运动仅影响热传递中由于空腔辐射导致的视角因子（以及因此的辐射通量）的计算。它们不影响热传导、存储或分布式通量贡献。

您可以独立定义步骤中运动的平移和旋转分量。例如，您可以根据某个幅值函数规定节点集的平移运动，然后根据不同的幅值函数规定节点集的旋转运动。在每个步骤中，每个运动分量只能为任何特定节点指定一次。

也可以在与空腔辐射关闭的步骤中规定运动，如下所述。

#### 平移运动

平移![](../graphics/usb_eqn06405.gif)根据全局*x*-、*y*-和*z*分量指定，除非在为指定运动的节点定义了局部坐标系；然后根据局部*x*-、*y*-和*z*分量指定（参见["变换坐标系，" 第2.1.5节](pt01ch02s01aus09.md)）。

平移位移始终被规定为平移运动的总量值。这种处理平移的方式与应力/位移分析中位移边界条件使用的处理方式一致（["Abaqus/Standard和Abaqus/Explicit中的边界条件，" 第34.3.1节](pt07ch34s03aus118.md)）。默认是施加平移运动。

也可以规定平移速度。平移速度始终指当前步骤；因此，规定为速度的平移运动速率仅在定义它的步骤期间有效。此行为与速度边界条件不同，在后续步骤中，如果未重新定义，速度将保持有效。

|  | ``` [*MOTION](../key/key-link.md#usb-kws-hmotion), TRANSLATION, TYPE=DISPLACEMENT [*MOTION](../key/key-link.md#usb-kws-hmotion), TRANSLATION, TYPE=VELOCITY ``` |

| **Abaqus/CAE用法：** | Abaqus/CAE不支持空腔辐射的表面运动。 |

#### 旋转运动

由刚体旋转引起的位移，![](../graphics/usb_eqn06406.gif)，可以通过指定旋转量值和旋转轴来定义。在三维中，旋转轴通过指定轴上的两点定义，![](../graphics/usb_eqn06384.gif)和![](../graphics/usb_eqn06407.gif)。在二维中，旋转轴假定为垂直于模型平面，并通过指定一点定义，![](../graphics/usb_eqn06384.gif)。

定义旋转轴的点的坐标必须在定义刚体旋转的步骤开始时的配置中定义。

步骤期间的刚体旋转导致的运动被规定为仅在该步骤期间发生的旋转量。因此，在步骤中规定的刚体旋转是局部的；如果后续步骤中未规定刚体旋转，则不会发生进一步旋转。

刚体旋转的处理与平移的处理不同：刚体旋转从步骤到步骤增量规定，而平移被规定为总量值。

##### 规定大旋转运动

超过![](../graphics/usb_eqn00093.gif)弧度的大旋转运动或在三维模型中不同方向上复杂旋转序列的旋转最简单地通过指定旋转速度来定义，这允许以角速度而不是总旋转来给出定义。Abaqus/Standard将旋转计算为增量开始和结束时角速度的平均值乘以时间增量。（参见["惯例，" 第1.2.2节](pt01ch02s02aus02.md)。）

##### 示例

例如，如果需要绕*z*轴旋转![](../graphics/usb_eqn05969.gif)，且绕*x*-和*y*轴无旋转，并假定步骤时间为1.0，按如下方式指定恒定角速度![](../graphics/usb_eqn05969.gif)：

```
[*MOTION](../key/key-link.md#usb-kws-hmotion), TYPE=VELOCITY, ROTATION
 node (node set), 18.84955592, 0., 0., 0., 0., 0., 1.
```

角速度将是恒定的，因为热传递或耦合热-电步骤（稳态和瞬态）中使用预定义速度场的运动的默认变化是阶跃函数（参见["定义分析，" 第6.1.2节](pt03ch06s01abo05.md)）。可以使用幅值引用来指定角速度的其他变化。

如果在下一步中，相同节点（或节点集）应该绕全局*x*轴额外旋转![](../graphics/usb_eqn04487.gif)弧度，同样假定步骤时间为1.0，按如下方式规定恒定角速度：

```
[*MOTION](../key/key-link.md#usb-kws-hmotion), TYPE=VELOCITY, ROTATION
 node (node set), 1.570796327, 0., 0., 0., 1., 0., 0.
```

如果您在规定运动定义中规定了平移的量值，它将被幅值曲线修改（如果有），并传递到子程序[`UMOTION`](../sub/sub-link.md#sub-xsl-umotion)，在那里可以重新定义。

### 在分析期间控制视角因子计算

空腔辐射功能可用于制造序列模拟等应用，其中辐射视角因子在模拟过程中发生变化。因此，辐射视角因子定义在步骤中为控制视角因子计算提供了显著的灵活性。

如果不同类型的辐射和不同空腔的视角因子计算需要不同的视角因子计算，则可以在步骤定义中指定多个辐射视角因子定义。可以在分析的不同步骤中为同一空腔指定不同类型的视角因子计算。

默认情况下，视角因子在包含辐射视角因子定义的第一步开始时计算。视角因子仅在后续步骤开始时重新计算（如果该步骤中视角因子定义发生变化；例如，如果为同一空腔指定了不同的表面阻塞检查）。在重启分析中，Abaqus/Standard从用户指定的重新启动步骤和增量读取辐射视角因子，并且仅在视角因子定义发生变化时才重新计算视角因子。

您可以指定为其规定辐射视角因子控制的空腔名称。如果不指定空腔名称，则辐射视角因子定义适用于模型中的所有空腔。

| **输入文件用法：** | ``` [*RADIATION VIEW FACTOR](../key/key-link.md#usb-kws-hradviewfactor), CAVITY=*cavity_name* ``` |
| --- | --- |

| **Abaqus/CAE用法：** | 辐射视角因子分别为每个空腔辐射相互作用定义，并适用于该相互作用活跃的所有步骤。 |
| --- | --- |

#### 激活和停用空腔辐射

在某些实际情况下，在分析过程中打开和关闭空腔辐射效应可能很有用。例如，辐射可能发生在随后充满流体的空腔中，因此辐射不再显著；稍后在分析中，当流体从空腔排出时，辐射可能会恢复。在这种情况下，您可以使用辐射视角因子定义在分析的一个或多个步骤期间打开和关闭任何特定空腔的辐射。

当在关闭后重新打开空腔辐射时，Abaqus/Standard将使用最后活动空腔辐射步骤中计算的最新视角因子。但是，如果在空腔辐射关闭期间规定运动，并且指定节点集中任何节点的位移分量超过了关闭空腔辐射的步骤中指定的最大允许运动*max*，则在重新打开空腔辐射的步骤开始时将重新计算视角因子。

##### 具有对称性的空腔中的视角因子计算

视角因子计算考虑空腔对称性暗示的闭合。对于没有周期或循环对称性的空腔，视角因子对二维几何精确计算，但对轴对称和三维几何进行近似。随着表面之间的距离减小，这些近似变得不那么准确。定义热辐射以建模紧密间隔的表面（参见["热接触属性，" 第37.2.1节](pt09ch37s02aus174.md)）。

##### 开放空腔中的视角因子计算

如果开放空腔（通过指定环境温度值定义）中表面的视角因子之和与一的偏差大于指定的视角因子容差，则会发生与环境的辐射。在这种近乎封闭的空腔中，这种偏差可能很小。如果容差未被违反，则不包括与外部介质的辐射，即使空腔被定义为开放；会发出关于此的警告消息。您可以放宽视角因子容差以包括此类辐射。

#### 空腔辐射分析中的内存使用

Abaqus中空腔辐射小面之间的热传递建模使用完整的非对称矩阵，定义空腔中每个节点与所有其他节点的相互作用。对于具有大量节点的面，此矩阵可能很大，导致内存需求显著大于没有空腔辐射相互作用的有限元部分分析的内存需求。

为最小化空腔辐射热传递分析的内存需求和计算成本，空腔可以使用更粗网格的热传递壳单元定义，每个节点只有一个自由度。覆盖单元应具有最小的热容和传导，并且应用于定义空腔而不是物理多自由度壳。覆盖单元应用于在绑定耦合约束（["网格绑定约束，" 第35.3.1节](pt08ch35s03aus132.md)）中定义主表面；多自由度、物理、热传递壳单元形成从表面。

### 初始条件

默认情况下，所有节点的初始温度为零。您可以在空腔辐射分析中指定非零初始温度；参见["在Abaqus/Standard和Abaqus/Explicit中定义初始温度" "初始条件，" 第34.2.1节](pt07ch34s02aus116.md#usb-prc-pinitialcond-temp)。

在涉及通过网格强制对流的热传递分析中，您可以定义强制对流/扩散热传递单元模型节点处的非零初始质量流率（参见["非耦合热传递分析，" 第6.5.2节](pt03ch06s05at18.md)）。

### 边界条件

您可以规定温度（自由度11）的边界条件在节点处（参见["Abaqus/Standard和Abaqus/Explicit中的边界条件，" 第34.3.1节](pt07ch34s03aus118.md)）。壳单元在厚度方向上有额外的温度自由度12、13等（参见["惯例，" 第1.2.2节](pt01ch02s02aus02.md)）。边界条件可以通过引用幅值曲线指定为时间的函数（["幅值曲线，" 第34.1.2节](pt07ch34s01aus115.md)）。

对于纯扩散单元，没有规定边界条件（自然边界条件）的边界对应于绝缘表面。对于强制对流/扩散单元，仅与传导相关的通量为零；能量可以在未加载表面自由对流。这种自然边界条件正确地模拟了流体穿过表面的区域（例如，在网格的上游和下游边界）并防止能量虚假反射回网格。

### 载荷

### 单元

Abaqus/Standard中任何热传递或耦合热-电单元都可用于空腔辐射分析，包括强制对流/扩散热传递单元（参见["为分析类型选择适当的单元，" 第27.1.3节](pt06ch27s01aus112.md)；["非耦合热传递分析，" 第6.5.2节](pt03ch06s05at18.md)；和["耦合热-电分析，" 第6.7.3节](pt03ch06s07at22.md)）。耦合温度-位移和耦合热-电-结构单元不能用于空腔辐射分析。

除了您定义的单元外，Abaqus/Standard还使用根据辐射空腔定义自动生成的内部单元。

##### 打印输出

由辐射输出请求到数据文件生成的输出表按表面对基础组织。将出现在特定表中的行通过选择空腔、表面或单元集来定义：表的每一行对应于属于所选空腔、表面或单元集的单个单元面。如果表行中的所有变量都为零，则不打印该行。

每个表的第一列是单元编号，第二列是单元面标识符。您选择要出现在剩余列中的变量。表的数量没有限制。

例如，考虑一个包含名为`CAV1`的空腔的热传递模型，该空腔又由表面`SURF1`和`SURF2`组成。如果您为此模型请求将辐射通量（RADFL）和小面温度（FTEMP）输出到数据文件，数据文件中将出现两个表。一个表将包含组成表面`SURF1`的所有单元面的RADFL和FTEMP输出，另一个表将包含组成表面`SURF2`的所有单元面的相同输出变量。

默认情况下，Abaqus/Standard写入表中每列最大值和最小值的摘要。您可以选择抑制此摘要。此外，您可以选择打印表中每列的总计，这例如对于求和跨构成辐射表面的所有小面的辐射通量很有用。默认情况下，不打印这些总计。

| **输入文件用法：** | 使用以下选项控制输出到数据文件的摘要信息： |
| --- | --- |
|  | ``` [*RADIATION PRINT](../key/key-link.md#usb-kws-hradprint), SUMMARY=YES *or* NO ``` 使用以下选项控制输出到数据文件的总计： ``` [*RADIATION PRINT](../key/key-link.md#usb-kws-hradprint), TOTALS=YES *or* NO ``` |

| **Abaqus/CAE用法：** | Abaqus/CAE不支持输出到数据文件的空腔辐射输出。 |
| --- | --- |
