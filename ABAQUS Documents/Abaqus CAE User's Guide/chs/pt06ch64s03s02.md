# 64.3.2 在Abaqus/CAE中使用偏移网格





网格编辑工具集中的偏移网格工具在Abaqus/CAE中有许多用途，如下所示：

**连续壳网格**

您可以使用实体偏移网格工具来网格化基本上是加厚壳的薄实体，如金属板、复合材料或注塑塑料组件。实体偏移网格工具对于创建连续壳单元非常有用，因为Abaqus/CAE在厚度方向上一致地定向单元。有关更多信息，请参见["用连续壳单元网格化零件，" 第25.2节](pt04ch25s02.md)。

**粘聚单元**

您可以为二维模型的四边形单元分配粘聚单元。对于三维模型，您可以使用实体偏移网格工具在网格上生成一层粘聚单元。偏移网格工具生成的单元在厚度方向法向的堆叠方向上保持一致，它是快速创建一层粘聚单元的便捷工具。有关更多信息，请参见["在现有三维网格中嵌入粘聚单元，" 第21.2节](pt04ch21s02.md)。

**皮肤加强件**

如果您的 skin 可以定义为单层，则在"属性"模块中创建 skin 是首选方法。但是，您可以使用壳偏移网格工具创建使用多层加强层的 skin 。有关更多信息，请参见["使用偏移网格创建皮肤加强件，" 第36.6节](pt04ch36s06.md)。

您还可以通过以下方式将实体偏移网格工具与"网格"模块中的网格生成工具结合使用：

1. 在"网格"模块中创建部分网格。
2. 使用实体偏移网格工具完成网格。

您可以从偏移网格创建包含单元的集合。您可以创建包含偏移单元的单个集合，也可以为每层偏移单元创建单独的集合。此外，如果您正在创建实体偏移网格，则可以从偏移网格的顶部和底部创建曲面，前提是第一层单元未嵌入原始网格中。您可以在后续过程中使用这些集合和曲面来帮助您从偏移网格中选择单元。例如，您可以在为偏移单元层分配截面时选择集合。类似地，当您将一层粘聚单元的一侧绑定到周围的 bulk 材料时，可以选择曲面。
