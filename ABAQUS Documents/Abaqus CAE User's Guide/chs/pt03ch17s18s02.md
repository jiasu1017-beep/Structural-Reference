# 17.18.2 选择单元形状





您可以通过从主菜单栏选择 ****Mesh**![](../graphics/images/arrow.gif)**Controls**** 来控制网格中单元的形状。 出现的**Mesh Controls**对话框顶部有**Element Shape**选项。

**指定区域中使用的单元形状：**

1. 从主菜单栏，选择 ****Mesh**![](../graphics/images/arrow.gif)**Controls****。 Abaqus/CAE在提示区域显示提示以指导您完成该过程。 **提示：**您可以使用Mesh模块工具箱中的 ![](../graphics/ico_meshControls.png) 工具设置单元形状。
2. 如果您的零件或装配包含多个区域，请选择要查看或修改其单元形状的区域，然后按鼠标按钮2。 所有选定区域必须具有相同的维度。 **Mesh Controls**对话框出现。
3. 从**Element Shape**选项列表中，选择您想要的单元形状。 如果您选择了二维区域，可以选择以下单元形状选项： ****Quad**** 仅使用四边形单元。 下图显示使用此设置构建的网格示例： ![](../graphics/mgn-corners3.png) ****Quad-dominated**** 主要使用四边形单元，但在过渡区域允许三角形。 此设置为默认设置。 下图显示使用此设置构建的网格示例： ![](../graphics/mgn-char2.png) ****Tri**** 仅使用三角形单元。 当您将网格控制应用于实体区域的面时，此设置是唯一可用的选项，因为三角形面网格将用于生成四面体实体网格。 下图显示使用此设置构建的网格示例： ![](../graphics/mgn-freemesh.png) 如果您选择了三维区域，可以选择以下单元形状选项： ****Hex**** 仅使用六面体单元。 此设置为默认设置。 下图显示使用此设置构建的网格示例： ![](../graphics/mgn-char1.png) ****Hex-dominated**** 主要使用六面体单元，但在过渡区域允许一些三角形棱柱（楔形）。 下图显示使用此设置构建的网格示例： ![](../graphics/mgn-hexdom.png) ****Tet**** 仅使用四面体单元。 下图显示使用此设置构建的网格示例： ![](../graphics/mgn-char3.png) ****Wedge**** 仅使用楔形单元。 下图显示使用此设置构建的单单元网格示例： ![](../graphics/mgn-char-wedge.png)
4. 单击**OK**。 下次在选定区域上生成网格时，您的选择将被遵守。 如果选定区域已经进行网格划分，系统会提示您删除网格或取消网格控制过程。
