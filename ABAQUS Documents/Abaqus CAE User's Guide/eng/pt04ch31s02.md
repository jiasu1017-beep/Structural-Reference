# 31.2 Using contour integrals to model fracture mechanics







You can study the onset of cracking in a quasi-static model by selecting regions from your model that will be used to compute contour integral estimates. During the analysis Abaqus/Standard writes the values of the contour integrals to the output database as history output. For a detailed description of contour integrals, see ["Contour integral evaluation," Section 11.4.2 of the Abaqus Analysis User's Guide](../usb/usb-link.md#usb-anl-acontintegral).
