# 26.2 什么是协同仿真?


协同仿真技术是一种多物理场能力,提供 Abaqus 分析程序的运行时耦合。您可以将模型划分为多个域,并使用不同的分析程序获取每个域的解。对于 Abaqus/Standard 到 Abaqus/Explicit 的协同仿真,每个 Abaqus 分析在模型域的互补部分上运行,在该部分中它有望提供计算效率更高的解。例如,Abaqus/Standard 为轻而硬的组件提供更高效的解,而 Abaqus/Explicit 在求解复杂接触相互作用方面更高效。

您可以定义交换场的接口区域和耦合方案。更多信息,请参见["结构到结构协同仿真," Abaqus Analysis User's Guide 第 17.3.1 节](../usb/usb-link.md#usb-anl-acosimabqtoabq),和["流体到结构和共轭热传递协同仿真," Abaqus Analysis User's Guide 第 17.3.2 节](../usb/usb-link.md#usb-anl-acosimcfdtoabq)。
