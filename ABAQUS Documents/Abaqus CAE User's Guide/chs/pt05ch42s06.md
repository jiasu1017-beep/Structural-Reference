# 42.8 后处理期间创建坐标系





您可以在可视化模块中创建本地（或"会话"）坐标系用于后处理目的：场输出结果可以转换到指定的用户定义坐标系。这些用户定义的坐标系可以在空间中固定，也可以在模型变形时随指定节点移动。会话坐标系仅在会话期间持续有效，并且仅对单个输出数据库可用；但是，您可以将会话坐标系保存到其关联的输出数据库文件中，以便在后续的Abaqus会话中使用。

您可以从**坐标系管理器**或当前输出数据库结果树中的**会话坐标系**容器重命名或删除会话坐标系。您还可以通过从结果树使用其快捷方式或将其添加到显示组来显示、高亮和隐藏各个坐标系。有关将基于会话的和基于输出数据库的坐标系添加到显示组的详细信息，请参见["管理显示组，" 第78.2节](pt07ch78s02.md)。
