# 13.10.9 旋转零件或模型实例





从主菜单栏选择****Instance**![](../graphics/images/arrow.gif)**Rotate****可以绕选定轴旋转所选零件或模型实例。要旋转三维实例，必须选择定义实例将绕其旋转的轴的两个点。要旋转二维实例，必须选择一个实例将绕其旋转的点。你不能旋转轴对称实例。如果旋转与先前的位置约束冲突（例如，约束对齐两个面），Abaqus/CAE显示错误消息，旋转失败。

当你在装配中创建第一个实例时，Abaqus/CAE显示一个图形，指示装配全局坐标系的原点和平移方向。你可以使用此图形来帮助你决定如何旋转零件和模型实例。此外，你可以使用查询工具集查看先前应用于实例的平移和旋转总和，以及选定顶点之间的距离。旋转和平移不被视为装配的特征，不能被编辑或删除。

**要旋转零件或模型实例：**

1. 从主菜单栏，选择****Instance**![](../graphics/images/arrow.gif)**Rotate****。
   **提示：**你也可以使用装配模块工具箱中的![](../graphics/ico_instanceRotate.png)工具来旋转实例。有关装配工具箱中工具的图表，请参见["使用装配模块工具箱，"第13.9节](pt03ch13s09.md)。Abaqus/CAE在提示区域显示提示来指导你完成过程。
2. 从装配中选择要旋转的零件或模型实例。你也可以点击提示区域右侧的**Instance List**按钮并从出现的**Instance List**对话框中选择要旋转的实例。
   **提示：**如果你无法选择所需的实例，可以使用**Selection**工具栏更改选择行为。更多信息，请参见["使用选择选项，"第6.3节](pt01ch06s03.md)。你可以使用**[Ctrl]****+Click**和**[Shift]****+Click**的组合来选择多个实例。Abaqus/CAE高亮显示选定的实例。
3. 选择定义旋转轴的矢量的起点。你可以选择的任意现有顶点或基准点，或者可以在提示区域的文本框中输入坐标。
4. 选择定义旋转轴的矢量的终点。同样，你可以选择任意现有顶点或基准点，或者可以在提示区域的文本框中输入坐标。
5. 在提示区域出现的文本框中，输入旋转角度。正角度表示逆时针旋转；负角度表示顺时针旋转。Abaqus/CAE显示临时图像，指示将应用于选定实例的旋转。应用后你不能编辑或删除旋转。尝试旋转实例可能会与现有位置约束冲突。如果发生冲突，Abaqus/CAE显示错误消息，旋转失败。
6. 执行以下操作之一：
   1. 如果你确认旋转正确，点击提示区域中的**OK**按钮。Abaqus/CAE旋转实例并将其定位在与实例临时图像相同的位置。
   2. 如果你对旋转不满意，点击**Previous**按钮（![](../graphics/afxI_backup.png)）并指定新的旋转。
   3. 通过点击取消按钮（![](../graphics/afxI_cancel.png)）中止旋转。
