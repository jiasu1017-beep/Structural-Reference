# 1 Introduction to Abaqus 6.14







This document introduces features in Abaqus that have been added, enhanced, or updated since the Abaqus 6.13 release. 

Chapter 1 provides a brief overview of the Abaqus products included in this release. Chapters 2–14 provide short descriptions of new Abaqus 6.14 features in Abaqus/Standard, Abaqus/Explicit, Abaqus/CFD, and Abaqus/CAE, categorized by subject:
- [Chapter 2, "General enhancements](abc02.md)": general changes to the Abaqus interface.
- [Chapter 3, "Modeling](abc03.md)": features related to creating your model.
- [Chapter 4, "Analysis procedures](abc04.md)": features related to defining an analysis.
- [Chapter 5, "Analysis techniques](abc05.md)": features related to analysis techniques in Abaqus.
- [Chapter 6, "Materials](abc06.md)": new material models or changes to existing material models.
- [Chapter 7, "Elements](abc07.md)": new elements or changes to existing elements.
- [Chapter 8, "Prescribed conditions](abc08.md)": loads, boundary conditions, and predefined fields.
- [Chapter 9, "Constraints](abc09.md)": kinematic constraints.
- [Chapter 10, "Interactions](abc10.md)": features related to contact and interaction modeling.
- [Chapter 11, "Execution](abc11.md)": commands and utilities for running any of the Abaqus products.
- [Chapter 12, "Output and visualization](abc12.md)": obtaining, postprocessing, and visualizing results from Abaqus analyses.
- [Chapter 13, "User subroutines, utilities, and plug-ins](abc13.md)": additional user programs that can be run with Abaqus.
- [Chapter 14, "Abaqus Scripting Interface](abc14.md)": using the Abaqus Scripting Interface to write user scripts.

Each entry in these chapters clearly indicates the Abaqus product or products to which the feature applies and includes cross-references to more detailed information. [Chapter 15, "Summary of changes](abs01.md),” summarizes in tabular format the changes to Abaqus elements, keyword options, user subroutines, and output variable identifiers.
