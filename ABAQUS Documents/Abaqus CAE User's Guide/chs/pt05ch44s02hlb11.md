# 44.5.11 创建新的颜色光谱





Abaqus/CAE提供七种预定义颜色光谱，您可以使用它们来自定义云图中的颜色。如果您希望使用不同的颜色集或不同的排列，您可以创建新的颜色光谱并将其颜色应用于云图。

创建和编辑颜色光谱是类似的过程。当您创建新的颜色光谱时，Abaqus/CAE使用您上次修改或应用于云图的颜色光谱内容填充**Create Spectrum**对话框。如果您在该会话中尚未修改光谱或创建云图，Abaqus/CAE使用预定义的**Rainbow**颜色光谱中的颜色填充新的颜色光谱。然后，您可以通过添加、修改、移动和删除颜色来编辑此新颜色光谱，以达到您想要的内容和排列。

光谱必须包含至少两种颜色，最多可包含任意数量的颜色。在许多情况下，您选择的颜色光谱中的颜色数量与云图区间数量不匹配，因此Abaqus/CAE无法将颜色一对一地分配给云图区间。在这种情况下，Abaqus/CAE执行以下操作之一：
- 如果您的光谱颜色多于当前云图区间数量，Abaqus/CAE从光谱中采样颜色来创建云图。例如，当光谱包含24种颜色而只选择了12个区间时，云图将显示光谱中每隔一种颜色。
- 如果您的光谱颜色少于当前云图区间数量，Abaqus/CAE在光谱颜色之间插值以确定用于云图区间的其他颜色。所有用户定义的颜色光谱遵循HSV插值方案，而**Red to blue**预定义光谱遵循RGB插值方案。这意味着**Red to blue**预定义光谱和用户定义的光谱可能包含相同的两种颜色（红色和蓝色），但由于它们的插值方案会创建不同的中间区间值，因此行为可能不同。一般来说，HSV插值比RGB插值更符合人类直觉。

颜色光谱是特定于会话的定义；因此，它们在您会话期间的任何视口中可用，但不会保存到输出数据库（`.odb`）。您可以编辑任何颜色光谱的颜色和排列，也可以重命名、复制或删除任何用户创建的颜射光谱。Abaqus/CAE不允许您重命名或删除七种预定义颜色光谱。Abaqus/CAE从主菜单栏和Results Tree两者提供对颜色光谱的访问。

**要创建新的颜色光谱：**

1. 找到**Create Spectrum**对话框。从主菜单栏中，选择****Tools**![](../graphics/images/arrow.gif)**Spectrum**![](../graphics/images/arrow.gif)**Create****。**Create Spectrum**对话框出现。
2. 为这个新的颜色光谱指定**Name**。
3. 要将新颜色插入颜色光谱：1. 突出显示您希望新颜色在其之前或之后的颜色。2. 单击**Insert Before**或**Insert After**以在指定位置添加新颜色。**Select Color**对话框打开。3. 选择颜色，然后单击**OK**关闭**Select Color**对话框。Abaqus/CAE在所选位置添加新颜色。
4. 要更改颜色光谱中的某种颜色，请双击该颜色并从出现的**Select Color**对话框中选择新颜色。
5. 要在颜色光谱内移动颜色，请突出显示该颜色，然后单击**Move Up**或**Move Down**以沿所选方向移动一个步长。
6. 要从光谱中删除颜色，请突出显示该颜色并单击**Delete**。
7. 继续添加、更改、移动和删除颜色，直到颜色光谱包含您以所需顺序想要的颜色。
8. 单击**OK**。Abaqus/CAE添加新的颜色光谱，使其可用于自定义云图。
