# 20.10.2 草绘直线和多边形







使用 Sketcher 工具箱中的直线工具绘制直线、连续直线或多边形。下图显示了如何通过按所示顺序点击位置来绘制直线、连续直线和多边形：

![](../graphics/ske-linebtn-nls.png)

在草绘时，您应该注意定位点，因为这会影响网格的质量。草图中的点会成为您正在创建或修改的零件的顶点。反过来，当您在网格模块中对模型进行网格划分时，Abaqus/CAE 会将这些顶点转换为完全约束的种子，并将节点放置在其位置。有关如何随后移动顶点的信息，请参见["拖动 Sketcher 对象，"第 20.17.1 节](pt03ch20s14hlb01.md)。

**草绘直线和多边形：**

1. 从 Sketcher 工具箱中的直线工具中，选择连续直线工具![](../graphics/ico_sketchLines.png)。有关 Sketcher 工具箱中工具的图表，请参见["Sketcher 工具，"第 20.4.1 节](pt03ch20s04s01.md)。Abaqus/CAE 在提示区域显示提示以指导您完成该过程。
2. 要构造简单直线，请点击两个端点。要构造连续直线或多边形，请点击每个顶点。**提示：**如有必要，您可以使用提示区域中的文本框输入直线顶点的精确坐标。有关精确定义直线的更多信息，请参见["指定精确几何，"第 20.6 节](pt03ch20s06.md)。当您点击顶点或输入坐标时，直线或多边形会出现。
3. 要完成直线或多边形，请点击鼠标按钮 2。**提示：**如果在构造连续直线或多边形时出错，请点击 Sketcher 工具箱中的**撤销**工具![](../graphics/ico_undo.png)删除最近的线段。如果较早的线段出错，您可以使用**删除**工具![](../graphics/ico_delete.png)删除错误的线段，然后使用直线工具重绘。
4. 要创建更多直线或多边形，重复从第 2 步开始的上述步骤。
5. 完成创建直线和多边形后，执行以下操作之一：- 在 Abaqus/CAE 窗口中的任意位置点击鼠标按钮 2。- 在 Sketcher 工具箱中选择任何其他工具。- 点击提示区域中的取消按钮![](../graphics/afxI_cancel.png)。
