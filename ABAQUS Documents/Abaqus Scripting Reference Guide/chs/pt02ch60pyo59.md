# 60.59 HeatGeneration 对象

HeatGeneration 对象在热传递分析中包含体积热生成。

**访问**

```
materialApi.materials()[*name*].heatGeneration()
```

### 60.59.1 HeatGeneration()

此方法创建一个 HeatGeneration 对象。

**路径**

```
materialApi.materials()[*name*].HeatGeneration
```

**原型**

```
odb_HeatGeneration&
HeatGeneration();
```

**返回值**

一个 HeatGeneration 对象。

**异常**

无。

### 60.59.2 成员

HeatGeneration 对象没有成员。
