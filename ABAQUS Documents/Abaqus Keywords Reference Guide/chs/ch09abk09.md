# *INCIDENT WAVE FLUID PROPERTY






### *INCIDENT WAVE FLUID PROPERTY定义与入射波相关的流体特性。

定义入射波流体特性的首选界面是使用 [*INCIDENT WAVE INTERACTION](ch09abk10.md) 选项的 [*INCIDENT WAVE INTERACTION PROPERTY](ch09abk11.md) 选项。替代界面使用 [*INCIDENT WAVE FLUID PROPERTY](ch09abk09.md) 选项来定义用于定义入射波的流体特性。

[*INCIDENT WAVE FLUID PROPERTY](ch09abk09.md) 必须与 [*INCIDENT WAVE PROPERTY](ch09abk12.md) 选项一起使用。

**产品：** Abaqus/Standard  Abaqus/Explicit  Abaqus/CAE  

**类型：** 模型数据  

**级别：** 模型  

**Abaqus/CAE：** 不支持；此选项已被入射波相互作用特性取代。
