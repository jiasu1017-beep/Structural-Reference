# 64.6.1 创建单元





从主菜单栏选择****网格**![](../graphics/images/arrow.gif)**编辑****可以从所选节点创建单元。

**创建单元的步骤：**

1. 进入"网格"模块。
2. 从上下文栏的**对象**字段中选择**零件**并从列表中选择一个零件。
3. 从主菜单栏选择****网格**![](../graphics/images/arrow.gif)**编辑****。Abaqus/CAE显示**编辑网格**对话框。 **提示：**您也可以使用位于"零件"模块和"网格"模块工具箱底部的![](../graphics/ico_meshEdit.png)工具显示**编辑网格**对话框。
4. 在对话框中执行以下操作： 1. 在**类别**字段中选择**单元**。 2. 从**方法**列表中选择**创建**。Abaqus/CAE显示现有节点。在隐藏和着色模式下，仅显示可见节点。
5. 在提示区域，点击**单元形状**字段旁边的箭头，从出现的列表中选择您选择的单元形状。
6. 在视口中选择将定义单元的节点。您必须为您在上一步中指定的单元形状选择所需数量的节点。此外，您必须按特定顺序选择节点。从提示区域右侧点击**提示**可查看单元形状和连接性的图形。例如，当您创建10节点四面体单元时，Abaqus/CAE显示以下图形： ![](../graphics/prt-element-shapetip-nls.png) 选择所需数量的节点后，Abaqus/CAE会创建新单元。 **提示：**如果在创建单元时出错，点击**编辑网格**对话框中的**撤销**可以移除最近创建的单元。
7. 根据需要重复上述步骤以创建其他单元。如果更改单元形状，请再次点击**提示**按钮以更新单元形状和连接性图形。
8. 完成单元创建后，点击鼠标按钮2或提示区域中的取消按钮![](../graphics/afxI_cancel.png)退出过程。
