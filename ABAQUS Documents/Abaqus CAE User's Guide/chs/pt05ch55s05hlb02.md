# 55.5.2 自定义单元标签





单元标签是标识每个单元的数字标签(单元号)。例如,[图55-7](pt05ch55s05hlb02.md#viw-labelelem)中显示了单元标签。

**图55-7** 显示单元标签的模型。

![](../graphics/viw-labelelem.gif)

单元标签是曲线图状态无关的;也就是说,您为所有未变形、变形、云图、符号和材料取向曲线图设置一次。您也可以设置Abaqus/CAE在变形形状叠加在未变形形状上时将显示在未变形形状上的单元标签。切换"**Show element labels**"以显示或抑制单元标签并选择其颜色。

**自定义单元标签的步骤:**

1. 找到公共或叠加单元标签选项。公共单元标签选项始终应用于变形形状;它们适用于在任何曲线图状态(未变形、变形、云图、符号或材料取向)中单独绘制未变形形状时。当未变形形状与变形形状一起绘制时,应用叠加单元标签选项。从主菜单栏,选择 ****Options**![](../graphics/images/arrow.gif)**Common**** 或 **Superimpose**;然后在出现的对话框中点击"**Labels**"选项卡。
2. 切换"**Show element labels**"以显示或抑制数字单元标签。当"**Show element labels**"打开时,单元标签颜色选项变得可用。
3. 选择单元标签的颜色: 1. 点击颜色样本 ![](../graphics/usi-colorsample.png)。Abaqus/CAE显示"选择颜色"对话框。 2. 使用"选择颜色"对话框中的方法之一来选择新颜色。更多信息,请参阅 ["自定义颜色," 第3.2.9节](pt01ch03s02s09.md)。 3. 点击"**OK**"关闭"选择颜色"对话框。颜色样本变为所选颜色。
4. 点击"**Apply**"实施您的更改。标签根据您的规格进行更改。默认情况下,您的更改会在会话期间保存,并会影响当前视口和从当前视口创建的任何新视口中的所有后续曲线图。如果要为后续会话保留您应用的更改,请将它们保存到文件。更多信息,请参阅 ["为后续会话保存自定义," 第55.1.1节](pt05ch55s01s01.md)。
