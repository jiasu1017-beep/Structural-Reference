### 概述

对 Abaqus/Explicit 接触对中从节点位置的调整：
- 对于所有具有过盈从节点且没有指定初始间隙的接触对执行，除了当刚体节点充当从节点时；
- 可以消除使用图形预处理器（如 Abaqus/CAE）时由数值舍入引起的小间隙或穿透；
- 在模拟的第一步中不在模型中产生任何应变或动量；
- 在模拟的后续步骤中产生应变和动量；
- 不应用于纠正网格设计中的重大错误；和
- 不应用于解决涉及被困在两个主表面之间的从节点的初始过盈量。

如果使用小滑动接触 formulation（见 ["在 Abaqus/Explicit 中接触对的接触 formulations，" 第 38.2.2 节](pt09ch38s02aus181.md)），则调整表面位置的替代方法是精确地定义表面之间的初始间隙，包括大小和方向。

### 模拟第一步中过盈表面的调整

Abaqus/Explicit 将自动调整表面位置以移除在模拟第一步中定义接触对时存在的任何初始过盈量，除非刚体节点充当从节点或使用了用户子程序 [`VUINTER`](../sub/sub-link.md#sub-xsl-vuinter)。调整是通过对表面上的从节点进行无应变初始位移来进行的。因此，当定义平衡主-从接触对时，两个表面上的节点都可能进行调整。这种自动表面位置调整旨在仅纠正与网格生成相关的小不匹配。您可以在状态（`.sta`）文件、消息（`.msg`）文件和输出数据库（`.odb`）文件中查看表面调整；请参见 ["Abaqus/Explicit 分析中的接触诊断，" 第 39.2.1 节](pt09ch39s02aus185.md) 了解更多信息。

一些软化接触模型在零过盈量时具有非零接触压力（见 ["接触压力-过盈量关系，" 第 37.1.2 节](pt09ch37s01aus166.md)）。对于这些模型，在分析开始时可能存在一些初始不平衡接触压力，因为调整是为了满足零过盈量而不是零接触压力。大的初始接触压力可能导致接触表面附近单元过度变形。

来自单独接触对的冲突调整将导致初始过盈量的不完全解决，并导致嘈杂的解决方案或严重的单元变形。这可能发生在从节点被夹在两个主表面之间时。

由于双侧面片缺乏唯一的向外方向，双侧表面的大初始穿透解决可能很困难。仅当从节点位于底层单元厚度内时，才会检测到初始穿透，并且初始穿透将通过把从节点移动到最近的自由表面来解决（见 [图 36.5.4-1](pt09ch36s05aus163.md#adefsurf-bad-init-two-dual))。

**图 36.5.4-1** 涉及两个双侧表面的接触对的初始过盈量校正。

![](../graphics/adefsurf-bad-init-two-dual-nls.png)

如果检测到两个相邻从节点（由面片边缘连接）位于涉及接触对算法定义接触的双侧主表面的相对两侧，将向状态（`.sta`）文件发出警告消息。对于位于双侧主表面相对两侧的基于节点的表面节点，不会发出此类警告，因为无法确定基于节点的表面节点之间的邻接性。如果主表面是单侧表面，初始过盈量将使用主表面的表面法线解决，如 [图 36.5.4-2](pt09ch36s05aus163.md#adefsurf-bad-init-reg-dual) 所示。

**图 36.5.4-2** 涉及单侧和双侧表面的接触对的初始过盈量校正。

![](../graphics/adefsurf-bad-init-reg-dual-nls.png)

被困在双侧主表面相对两侧的从节点通常会导致严重问题，这些问题可能直到分析后期才会变得明显。因此，建议在运行大型接触对分析之前进行数据检查分析（见 ["Abaqus/Standard、Abaqus/Explicit 和 Abaqus/CFD 执行，" 第 3.2.2 节](pt01ch03s02abx02.md)），以便您可以检查状态文件（`.sta`）中的警告消息，并检查主表面相对两侧的错位相邻从节点。

调整仅影响表面上的节点。如果使用此功能纠正初始几何中的重大错误，可能会导致相邻单元过度变形，导致分析以错误消息结束。

刚体上的节点只能作为惩罚接触对的从节点。作为刚体一部分的从节点的初始穿透不会用无应变校正解决；即，不调整从节点。这些穿透可能在分析的第一个增量中导致人为大的接触力，因此应在网格定义中避免。

### 模拟后续步骤中过盈表面的调整

如果在后续步骤中定义了具有初始过盈表面的接触对，Abaqus/Explicit 不会采取任何特殊操作来逐步解决这些初始穿透：接触力将根据所使用的任何接触约束强制方法施加。这些接触力可能非常大，导致大的加速度和速度以及可能的单元变形。如果使用 [`VUINTER`](../sub/sub-link.md#sub-xsl-vuinter) 用户子程序，则初始穿透有可能在任何步骤引入的接触对中导致问题；但在这种情况下，您控制接触力的应用。

#### 最小化与初始过盈表面调整相关的噪声

当为初始过盈量调整不是非常小的情况使用平衡主-从接触对时，调整后的几何中可能会持续存在不可忽略的误差，并可能导致接触过程中的嘈杂振荡（或"振铃"）。这个问题有时可以通过修改接触对为使用加权因子的纯主-从关系来缓解；详见 ["在 Abaqus/Explicit 中接触对的接触 formulations"中的"接触表面加权，" 第 38.2.2 节](pt09ch38s02aus181.md#usb-cni-acontactpair-exppair)。

### 精确指定初始间隙值

当从节点坐标计算不够准确时，您可以精确地为从表面上的节点定义初始间隙和接触方向；例如，如果初始间隙相对于坐标值非常小。初始间隙和接触方向只能在微小滑动接触分析中定义（["在 Abaqus/Explicit 中接触对的接触 formulations，" 第 38.2.2 节](pt09ch38s02aus181.md)）。

基于从节点坐标和主表面计算的每个从节点处的初始间隙值会被您指定的值覆盖。此过程不改变从节点的坐标。

当为接触对调用平衡主-从接触算法时，可以在一个或两个表面上定义初始间隙。在仅作为主表面接触的接触表面上定义的初始间隙将被忽略。

#### 为表面指定空间变化的间隙

或者，您可以通过识别接触对和指定表格数据来为接触对指定空间变化的间隙，表格数据指定属于从表面的单个节点或一组节点的间隙。未识别的任何从表面节点将使用 Abaqus/Explicit 从表面初始几何计算出的间隙。

##### 为接触计算指定表面法线

通常 Abaqus/Explicit 根据离散化表面的几何形状计算用于接触计算的表面法线，使用 ["在 Abaqus/Explicit 中接触对的接触 formulations，" 第 38.2.2 节](pt09ch38s02aus181.md) 中描述的算法。在指定空间变化的间隙时，您可以通过指定此向量的分量来重新定义 Abaqus/Explicit 与每个从节点一起使用的接触方向。该向量必须定义主表面向外法线的全局笛卡尔分量。

| **输入文件用法：** | ``` [*CLEARANCE](../key/key-link.md#usb-kws-mclearance), SLAVE=*surface_name*, MASTER=*surface_name*, TABULAR *node number or node set label, clearance value, first normal component, second normal component, third normal component* ``` |

|  | 根据需要重复数据行。 |

##### 自动生成螺纹螺栓连接的接触法线方向

或者，对于单螺纹螺栓连接，可以通过指定螺纹几何数据和用于定义螺栓/螺栓孔轴线上向量的两点来自动生成每个从节点的接触法线方向。当受拉时，轴向量应从螺栓尖端指向螺栓头部；当受压时，从头部指向尖端。
