# 6.2.8 组合选择技术





有时组合使用选择和取消选择对象的方法会很方便。例如，你可以在使用集合工具集创建节点集时拖动选择一组节点。然后，你可以 **[Ctrl]****+点击**各个节点以取消选择它们，并 **[Shift]****+点击**其他节点以将它们添加到你的选择。以下是三种技术的组合说明：

1. 首先，你使用拖动选择来选择一组节点。
![](../graphics/pic-combo1.png)
2. 然后，你使用 **[Ctrl]****+点击**取消选择各个节点。
![](../graphics/pic-combo2.png)
3. 最后，你使用 **[Shift]****+点击**将节点添加到你的集合，然后单击鼠标按钮 2 表示你已完成选择。
![](../graphics/pic-combo3.png)

你可能会发现调整视图方向会使视口中的特定项目更容易访问。你可以在选择过程中的任何时刻调整视图方向。有关视图操作工具的信息，请参阅[第 5 章，"操作视图和控制透视"](pt01ch05.md)。

**提示：**要取消选择所有对象，请在当前视口中未使用的部分点击。
