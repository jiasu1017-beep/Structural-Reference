# 10.5.3 从输出数据库导入模型





您可以通过从主菜单栏选择****文件（File）**![](../graphics/images/arrow.gif)**导入（Import）**![](../graphics/images/arrow.gif)**模型（Model）****使用输出数据库将模型导入Abaqus/CAE。

以下功能可以从输出数据库导入到模型中：
- 节点和单元
- 曲面、节点和单元集合以及接触节点集合
- 材料和截面定义
- 梁轮廓

虽然截面会被导入，但将截面分配给适当单元集合的操作不会被导入。类似地，将梁轮廓分配给适当梁截面的操作也不会被导入。当您提交输入文件进行分析时，Abaqus不会将某些材料定义写入输出数据库。因此，如果您将输出数据库导入Abaqus/CAE，这些材料将缺失。如果发生这种情况，您可以从输入文件而不是从输出数据库导入模型。

Abaqus/CAE通过读取定义装配中每个零件实例的节点和单元来从输出数据库导入零件。如果创建输出数据库的输入文件是使用零件和装配构建的，则从输出数据库导入的每个零件实例在Abaqus/CAE中显示为单独的零件。此外，装配及其每个零件实例一起显示在Abaqus/CAE中。因此，导入Abaqus/CAE的模型包含对应于输出数据库中每个零件实例的零件和零件实例。

定义输出数据库中零件实例的节点和单元已被平移和旋转到其在装配中的位置。导入Abaqus/CAE的生成零件的名称与输出数据库中的原始零件实例相同，其方向反映了输出数据库中零件实例的方向。因此，导入零件的方向可能与创建输出数据库的输入文件中零件的方向不同。

如果创建输出数据库的输入文件不是使用零件和装配构建的，则Abaqus/CAE将网格定义作为单个零件和单个零件实例写入输出数据库。如果输出数据库中的单个零件包含解析刚体表面，则Abaqus/CAE尝试导入的零件将包含可变形零件和解析刚体零件的无效组合。因此，无法导入输出数据库。
