# 24.10 施加连接器载荷和连接器边界条件






在"载荷"模块中，您可以对连接器的可用相对运动分量施加连接器力或连接器力矩，以模拟连接器驱动。类似地，您可以规定连接器的可用相对运动分量的连接器位移、连接器速度或连接器加速度。创建这些连接器载荷或边界条件时，您选择要施加规定条件的线框。选择线框的最佳方法是使用线框特征的默认几何集名称（请参见["为多个连接器创建或修改线框特征，" 第 15.12.9 节"](pt03ch15s12hlb09.md)，了解更多详细信息）。您选择的线框必须与连接器截面分配相关联。如果您选择多个线框，必须确保分配给线框的连接器截面在连接器截面分配中具有您要定义力或力矩的可用相对运动分量。如果连接器力或连接器力矩的可用相对运动分量不足，会出现一条消息，要求您选择不同的线框或更改连接类型。

您还可以对与连接器截面分配关联的线框端点施加连接器材料流边界条件。有关更多信息，请参见["创建载荷，" 第 16.8.1 节"](pt03ch16s08hlb01.md)，和["创建边界条件，" 第 16.8.2 节"](pt03ch16s08hlb02.md)。
