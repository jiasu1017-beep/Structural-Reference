# 6 使用梁单元







使用梁单元对结构建模，其中一个维度（长度）明显大于其它两个维度，且纵向应力最重要。梁理论基于这样的假设：结构的变形可以完全由沿结构长度位置函数的变量确定。为了使梁理论产生可接受的结果，横截面尺寸应小于结构典型轴向尺寸的 1/10。以下是典型轴向尺寸的示例：
- 支撑之间的距离，
- 横截面大变化之间的距离，以及
- 感兴趣最高振动模式的波长。

Abaqus 梁单元假设垂直于梁轴的平面截面在变形过程中保持为平面。

不要混淆认为横截面尺寸应小于典型 *单元* 长度的 1/10。高度细化的网格可能包含长度小于其横截面尺寸的梁单元，尽管通常不推荐这样做——在这种情况下连续体单元可能更合适。
