# 6.3.1 Overview of the selection options







When you are prompted to select an object from the viewport, Abaqus/CAE provides selection tools that can make it easier and more efficient for you to make the desired selection. 

Use the **Selection** toolbar to configure the selection options. [Figure 6--15](pt01ch06s03hlb01.md#obj-selection-toolbar-nls) shows the layout of the selection tools. Selection tools appear dimmed if they are not valid for the current procedure.
