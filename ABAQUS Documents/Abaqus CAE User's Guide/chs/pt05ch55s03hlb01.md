# 55.3.1 控制单元和表面边缘可见性





您可以控制Abaqus/CAE显示单元和表面边缘的程度。边缘可见性是曲线图状态无关的;也就是说,您为所有未变形、变形、符号和材料取向曲线图设置一次。您也可以设置Abaqus/CAE在变形形状叠加在未变形形状上时将应用于未变形形状的边缘可见性。以下选项可用于控制边缘显示:**All edges**、**Exterior edges**、**Feature edges**、**Free edges**和**No edges**。前四个选项如[图55-2](pt05ch55s03hlb01.md#viw-edgeviz)所示。

**图55-2** 显示边缘显示选项的模型。

![](../graphics/viw-edgeviz-nls.png)

**注意:**如果您的输出数据库同时包含均匀实体单元和复合实体单元或连续壳单元,则Abaqus/CAE会在均匀实体几何与复合几何或连续壳几何之间的边界处显示自由边缘或特征边缘。

**控制边缘可见性的步骤:**

1. 找到公共或叠加"**Visible Edges**"选项。公共可见边缘设置始终应用于变形形状;当在任何曲线图状态(未变形、变形、云图、符号或材料取向)中单独绘制未变形形状时,它也适用于未变形形状。当未变形形状与变形形状一起绘制时,应用叠加可见边缘设置。从主菜单栏,选择 ****Options**![](../graphics/images/arrow.gif)**Common**** 或 **Superimpose**;然后在出现的对话框中点击"**Basic**"选项卡。"**Visible Edges**"选项在"**Basic**"页面右侧。
2. 要选择您想要显示的单元和表面边缘,点击以下之一:**All edges** 显示所有单元和表面边缘。要查看模型内部的单元边缘,您还必须将[渲染样式](pt05ch55s02hlb01.md)设置为线框。 **Exterior edges** 仅显示模型外部的边缘。 **Feature edges** 仅显示模型外部被计算为特征边缘的边缘。特征边缘位于法线相差超过"特征角度"的单元之间的边缘。特征角度在 ****View**![](../graphics/images/arrow.gif)**ODB Display Options**![](../graphics/images/arrow.gif)**General**** 对话框中设置;更多信息,请参阅 ["定义模型特征边缘," 第55.3.2节](pt05ch55s03hlb02.md)。 **Free edges** 仅显示属于单个单元的边缘。自由边缘显示对于定位网格中潜在的孔或裂缝特别有用。 **No edges** 抑制所有边缘的显示。此选择仅适用于[填充或着色](pt05ch55s02hlb01.md)渲染样式中的曲线图。 **注意:**当单元被复制时(即共享所有相同的角节点),Abaqus/CAE不会在自由或特征边缘曲线图中显示单元边缘。
3. 点击"**Apply**"实施您的更改。边缘可见性根据您的规格进行更改。默认情况下,您的更改会在会话期间保存,并会影响当前视口和从当前视口创建的任何新视口中的所有后续曲线图。如果要为后续会话保留您应用的更改,请将它们保存到文件。更多信息,请参阅 ["为后续会话保存自定义," 第55.1.1节](pt05ch55s01s01.md)。
