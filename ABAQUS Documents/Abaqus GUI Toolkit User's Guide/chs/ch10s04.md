# 10.4 示例：带塑性的连接耳







您被要求调查如果第 4 章"使用连续体单元"](ch04.md) 中的钢连接耳承受由事故引起的极端载荷（60 kN）会发生什么。线性分析的结果表明耳会屈服。您需要确定耳中塑性变形的程度和塑性应变的大小，以便评估耳是否失效。在此分析中不需要考虑惯性效应；因此，您将使用 Abaqus/Standard 来检查耳的静态响应。

钢的唯一非弹性材料数据是屈服应力（380 MPa）和失效时的应变（0.15）。您决定假设钢是理想塑性的：材料不会硬化，应力永远不会超过 380 MPa（参见[图 10--8](ch10s04.md#gss-steel)）。

**图 10–8** 钢的应力-应变行为。

![](../graphics/gss-steel-nls.png)

在现实中可能会发生一些硬化，但这个假设是保守的；如果材料硬化，塑性应变将小于模拟预测的值。

以下步骤假设您可以访问此示例的完整输入文件。此输入文件 `lug_plas.inp` 在 ["带塑性的连接耳，" 第 A.8 节](ap01s08.md)中提供。获取和运行脚本的说明在 [附录 A，"示例文件"](ap01.md) 中给出。

如果您希望使用 Abaqus/CAE 创建整个模型，请参阅 ["示例：带塑性的连接耳，" Getting Started with Abaqus: Interactive Edition 第 10.4 节](../gsa/gsa-link.md#gsa-mat-connectlug)。

### 10.4.1 修改输入文件——模型数据

在此示例中，材料定义使用 [*PLASTIC*](../key/key-link.md#usb-kws-mplastic) 选项指定材料的后屈服行为。材料的杨氏模量为 200 GPa，零塑性应变时的初始屈服应力为 380 MPa。由于您将钢建模为理想塑性，[*PLASTIC*](../key/key-link.md#usb-kws-mplastic) 选项上没有给出其他屈服应力。

完整的材料定义为：

```
*MATERIAL, NAME=STEEL
*ELASTIC
200.E9, 0.3
*PLASTIC
380.E6,0.0
```
 模型定义部分中的所有其他选项块保持不变。

### 10.4.2 修改输入文件——历史数据

此分析需要一般的非线性模拟，因为模型中存在非线性材料行为。因此，必须从 [*STEP*](../key/key-link.md#usb-kws-hstep) 选项中移除 PERTURBATION 参数。[*STATIC*](../key/key-link.md#usb-kws-hstatic) 程序选项块中的总步骤时间已设置为 1.0，初始增量大小为总步骤时间的 20%。此模拟是耳在极端载荷下的静态分析；您事先不知道此模拟可能需要多少个增量。但是，默认的最大 100 个增量相当大，应该足以满足此分析。此外，我们假设几何非线性的影响在此模拟中不重要，因此 NLGEOM 参数从 [*STEP*](../key/key-link.md#usb-kws-hstep) 选项中省略。输入文件的此部分如下所示。

```
*STEP
*STATIC
0.2, 1.0
```

**加载**

此模拟中施加的载荷是线性弹性模拟中施加的载荷的两倍（60 kN vs. 30 kN）。因此，此模型将施加到耳上的压力大小加倍。修改后的 [*DLOAD*](../key/key-link.md#usb-kws-hdload) 选项块如下：

```
*DLOAD
PRESS, P6, 1.E+08
```

**输出请求**

您将使用 Abaqus/Viewer 查看此模拟的所有结果，因此所有打印输出请求都已删除。输入文件中的结果输出请求选项如下：

```
*OUTPUT, FIELD, FREQUENCY=1, VARIABLE=PRESELECT
```

您需要将一些历史数据保存到输出数据库文件中，以用于 Abaqus/Viewer 中的 *X–Y* 绘图功能。节点集 `HOLEBOT`（应该已经存在）的位移使用以下选项存储：

```
*OUTPUT, HISTORY, FREQUENCY=1
*NODE OUTPUT, NSET=HOLEBOT
U,
```

您还需要耳内置端沿一个单元的详细结果（参见[图 10--9](ch10s04.md#gss-elem206)）。

**图 10–9** 单元 206。

![](../graphics/gss-elem206-v-nls.png)

这是 ["带塑性的连接耳，" 第 A.8 节](ap01s08.md) 中找到的命令生成的网格中的单元 206；此位置的单元在您的模型中可能有不同的编号。选择此单元是因为它的应力最有可能具有最大量值。在本示例的输入文件中，已创建了一个包含该单元的单元集，并将该单元集的应力（S）、应力不变量（SINV）、塑性应变（PE）和应变（E）保存到输出数据库文件中。必要的选项块如下所示：
```
*ELEMENT OUTPUT, ELSET=EL206
S, SINV, PE, E
*ELSET, ELSET=EL206
206,
```

**注意：**用于定义单元集的 [*ELSET*](../key/key-link.md#usb-kws-melset) 选项出现在输出请求之后，以免打断与 [*OUTPUT*](../key/key-link.md#usb-kws-houtput) 选项关联的子选项块块。

### 10.4.4 后处理结果

在 Abaqus/Viewer 中查看结果以了解是什么导致了过度塑性。通过在操作系统提示符下输入以下命令来运行 Abaqus/Viewer：

```
abaqus viewer odb=lug_plas
```

**绘制变形模型形状**

创建模型变形形状的图，并检查此形状是否合理。

默认视图是等轴测。您可以使用 **View** 菜单中的选项或 **View Manipulation** 工具栏中的工具设置[图 10--10](ch10s04.md#gss-hardening) 中所示的视图；在此图中，还关闭了透视。

**图 10–10** 使用无硬化模拟结果时的变形模型形状。

![](../graphics/gss-hardening-v.png)

图中显示的位移，特别是耳的旋转，很大，但似乎不足以引起模拟中看到的所有数值问题。仔细查看图的标题信息以获取解释。此图中使用的变形比例因子为 0.02；即，位移按其实际值的 2% 缩放。（您的变形比例因子可能不同。）

对于几何线性模拟，Abaqus/Viewer 总是缩放位移，使模型的变形形状适合视口。（这与几何非线性模拟形成对比，在几何非线性模拟中，Abaqus/Viewer 不缩放位移，而是通过放大或缩小来调整视图以将变形形状适合图。）要绘制实际位移，将变形比例因子设置为 1.0。这将产生模型的图，其中耳已变形直到几乎平行于垂直（全局 *Y*）轴。

施加的 60 kN 载荷超过了耳的极限载荷，当材料在通过其厚度的所有积分点屈服时，耳会发生坍塌。由于钢的理想塑性后屈服行为，耳没有刚度来抵抗进一步变形。这与之前观察到的关于大应变增量位置和最大位移校正位置的情况一致。

### 10.4.5 向材料模型添加硬化

使用理想塑性材料行为进行连接耳模拟预测耳会因结构坍塌而遭受灾难性失效。我们已经提到钢在屈服后可能会表现出一些硬化。您怀疑包含硬化行为会允许耳承受 60 kN 载荷，因为它会提供额外的刚度。因此，您决定在钢的材料特性定义中添加一些硬化。假设屈服应力在塑性应变 0.35 时增加到 580 MPa，这代表了此类钢的典型硬化。修改后材料模型的应力-应变曲线如图 [Figure 10--11](ch10s04.md#gss-modified) 所示。

**图 10–11** 钢的修改后应力-应变行为。

![](../graphics/gss-modified-nls.png)

按如下方式修改您的 [*PLASTIC*](../key/key-link.md#usb-kws-mplastic) 选项块以包含硬化数据：

```
*PLASTIC
380.E6, 0.00
580.E6, 0.35
```

### 10.4.6 使用塑性硬化运行分析

将带有塑性硬化的模型保存到名为 `lug_plas_hard.inp` 的新输入文件中，并使用以下命令运行分析

```
abaqus job=lug_plas_hard
```

**Mises 应力等值线图**

对模型中的 Mises 应力进行等值线处理。在实际变形形状上使用十个等值线间隔创建填充等值线图（即，将变形比例因子设置为 1.0），并抑制绘图标题和状态块。使用视图操作工具定位和调整模型大小，以获得类似于[图 10--13](ch10s04.md#gss-von-mises) 所示的图。

**图 10–13** Mises 应力等值线。

![](../graphics/gss-von-mises-v-nls.png)

等值线图例中列出的值是否让您感到惊讶？最大应力大于 580 MPa，这不应该发生，因为假定材料在此应力幅度下是理想塑性的。产生这种误导性结果是因为 Abaqus/Viewer 用于为单元变量（如应力）创建等值线图的算法。等值线算法需要在节点处的数据；但是，Abaqus/Standard 在积分点计算单元变量。Abaqus/Viewer 通过从积分点到节点外推数据来计算单元变量的节点值。外推顺序取决于单元类型；对于二阶减缩积分单元，Abaqus/Viewer 使用线性外推来计算单元变量的节点值。为了显示 Mises 应力等值线图，Abaqus/Viewer 从积分点外推应力分量到每个单元内的节点位置，并计算 Mises 应力。如果 Mises 应力值之间的差异在指定平均阈值内，则从每个周围单元的不变量应力值计算节点平均 Mises 应力。外推过程可能产生超过弹性极限的不变量值。

尝试绘制应力张量每个分量（S11、S22、S33、S12、S23 和 S13）的等值线。您将看到这些应力在内置端周围的单元之间存在显著变化。这导致外推的节点应力高于积分点处的值。因此，从这些值计算的 Mises 应力也会更高。

**注意：**单元类型 C3D10I 不受此 外推问题的影响。此单元类型的积分点位于节点上，从而改善了表面应力可视化。

积分点处的 Mises 应力永远不会超过单元材料当前屈服应力；但是，等值线图中报告的外推节点值可能会超过。此外，单独的分量应力可能具有超过当前屈服应力值的量级；只有 Mises 应力被要求具有小于或等于当前屈服应力值的量级。

您可以使用 Visualization 模块中的查询工具来检查积分点处的 Mises 应力。

**查询 Mises 应力：**

1. 从主菜单栏，选择 ****Tools**![](../graphics/images/arrow.gif)**Query****；或使用 **Query** 工具栏中的 ![](../graphics/ico_info.png) 工具。出现 **Query** 对话框。
2. 在 **Visualization Module Queries** 字段中，选择 **Probe values**。出现 **Probe Values** 对话框。
3. 确保选择了 **Elements** 和输出位置 **Integration Pt**。
4. 使用光标选择靠近耳约束端的单元。Abaqus/Viewer 默认报告单元 ID 和类型，以及从第一个积分点开始每个积分点处 Mises 应力的值。Mises 应力值都低于等值线图例中报告的值，也低于 580 MPa 的屈服应力。您可以点击鼠标按钮 1 来存储探测值。
5. 完成后点击 **Cancel**。

外推值与积分点值差异如此之大表明，应力在单元之间变化很快，网格太粗，无法进行准确应力计算。如果细化网格，这种外推误差会不那么显著，但始终会存在一定程度。因此，始终谨慎使用单元变量的节点值。

**等效塑性应变等值线图**

材料中的等效塑料应变（PEEQ）是一个标量变量，用于表示材料的非弹性变形。如果此变量大于零，则材料已屈服。可以通过在 **Field Output** 工具栏左侧的变量类型列表中选择 **Primary**，然后从输出变量列表中选择 **PEEQ**，在 PEEQ 等值线图中识别已屈服的耳的那些部分。在 Abaqus/Viewer 中以深色绘制的区域仍然具有弹性材料行为（参见[图 10--14](ch10s04.md#gss-plastic-strain)）。

**图 10–14** 等效塑性应变（PEEQ）等值线。

![](../graphics/gss-plastic-strain-v.png)

从图中可以明显看出，耳在与其母结构连接处发生了整体屈服。等值线图例中报告的最大塑性应变约为 10%。但是，此值可能包含来自外推过程的误差。使用查询工具 ![](../graphics/ico_info.png) 检查具有最大塑性应变的单元在积分点处的 PEEQ 值。您会发现模型中最大的等效塑性应变在积分点处约为 0.067。这并不一定表示大的外推误差，因为峰值塑性变形附近存在应变梯度。

**创建变量-变量（应力-应变）图**

Abaqus/Viewer 中的 *X–Y* 绘图功能在本指南前面介绍过。在本节中，您将学习如何创建显示一个变量作为另一个变量函数变化的 *X–Y* 图。您将使用保存到输出数据库（`.odb`）文件的应力和应变数据，为靠近耳约束端的单元中某个积分点创建应力-应变图。

在讨论使用的模型中，应力-应变数据是针对单元 206 保存的。您可能在模型中指定了不同的单元号；如果是这样，请在以下输入示例中使用该单元号代替 206。另外，使用最接近耳顶表面但不在约束节点旁边的积分点数据。因此，您需要确定单元的标签及其节点连接，以确定要使用哪个积分点。

**确定积分点编号：**

1. 在 **Display Group** 工具栏中，选择 **Create Display Group** ![](../graphics/ico_displayGroupCreate.png) 工具；在 **Create Display Group** 对话框中，选择 **Elements** 作为项目，选择 **Element sets** 作为方法。从可用单元集列表中，选择 **EL206** 并切换 **Highlight items in viewport** 以确认其选择。点击 **Replace**。
2. 使用可见的节点标签绘制此单元的未变形形状。点击自动拟合工具 ![](../graphics/ico_viewFit.png) 以获得类似于[图 10--15](ch10s04.md#gss-integration) 的图。**图 10--15** 单元 206 中积分点 1 的位置。![](../graphics/gss-integration-v-nls.png)
3. 使用 **Query** 工具获取此角单元的节点连接（选择 **Element**，然后在视口中点击单元）。节点连接将打印到消息区域；您只需要前四个节点。
4. 将节点连接列表与未变形模型形状图进行比较，以确定您的 C3D20R 单元上的 1--2--3--4 面，如 ["三维实体单元库，" Abaqus Analysis User's Guide 第 28.1.4 节](../usb/usb-link.md#usb-elm-e3delem) 中所定义。例如，在[图 10--15](ch10s04.md#gss-integration) 中，2841、3241、3245、2845 面对应于 1--2--3--4 面。比较这些后，我们将发现我们感兴趣的是对应于积分点 1 的点。

**创建沿耳单元 206 中应力和平动应变的历史曲线：**

1. 在结果树中，点击名为 `lug_plas_hard.odb` 的输出数据库下的 **History Output** 的鼠标按钮 3。从出现的菜单中选择 **Filter**。
2. 在过滤器字段中，输入 `*MISES*` 以将历史输出限制为仅 Mises 应力。
3. 点击单元 206、积分点 1 处的 MISES 应力的鼠标按钮 3。从出现的菜单中选择 **Save As**。输入名称 `MISES` 并点击 **OK**。
4. 使用 `*E11*` 过滤历史输出，并在同一积分点保存 **E11** 应变分量。将曲线命名为 `E11`。

使用 **MISES** 应力而不是真应力张量的分量，因为塑性模型根据 Mises 应力定义塑性屈服。**E11** 应变分量被使用是因为它是总应变张量在此点最大的分量；使用它可以清楚地显示此积分点处材料的弹性和塑性行为。

曲线出现在 **XYData** 容器中。每条曲线都是历史（变量与时间）图。您必须组合这些图，消除时间依赖性，以产生所需的应力-应变图。

**组合历史曲线以产生应力-应变图：**

1. 在结果树中，双击 **XYData**。出现 **Create XY Data** 对话框。
2. 选择 **Operate on XY data**，然后点击 **Continue**。出现 **Operate on XY Data** 对话框。展开 **Name** 字段以查看每条曲线的全名。
3. 从列出的 **Operators** 中，选择 **combine(X,X)**。`combine( )` 出现在对话框顶部的文本字段中。
4. 在 **XY Data** 字段中，选择应力和应变曲线。
5. 点击 **Add to Expression**。表达式 `combine("E11", "MISES")` 出现在文本字段中。在此表达式中，`"E11"` 将决定 *X* 值，`"MISES"` 将决定组合图中 *Y* 值。
6. 点击底部的 **Save As** 保存组合数据对象。出现 **Save XY Data As** 对话框。在 **Name** 文本字段中，输入 `SVE11`；然后点击 **OK** 关闭对话框。
7. 要查看组合应力-应变图，点击对话框底部的 **Plot Expression**。
8. 点击 **Cancel** 关闭对话框。
9. 点击提示区域的 ![](../graphics/afxI_cancel.png) 取消当前过程。

如果更改 *X* 和 *Y* 轴的限值，此 *X–Y* 图会更清晰。

**自定义应力-应变曲线：**

1. 双击任一轴以打开 **Axis Options** 对话框。
2. 将 *X* 轴（E11 应变）的最大值设置为 `0.09`，*Y* 轴（MISES 应力）的最大值设置为 `500` MPa，最小值设置为 `0.0` MPa。
3. 切换到 **Title** 选项卡页面，并按[图 10--16](ch10s04.md#gss-e11) 中所示自定义 *X* 和 *Y* 轴标签。**图 10--16** 角单元中沿耳的 Mises 应力与平动应变（E11）的关系。![](../graphics/gss-e11-c-nls.png)
4. 点击 **Dismiss** 关闭 **Axis Options** 对话框。
5. 在曲线的每个数据点显示符号也会有所帮助。打开 **Curve Options** 对话框。
6. 在 **Curves** 字段中，选择应力-应变曲线（`SVE11`）。`SVE11` 数据对象高亮显示。
7. 切换 **Show symbol**。接受默认值，然后点击对话框底部的 **Dismiss**。应力-应变图出现在曲线的每个数据点带有符号。

您现在应该有一个类似于[图 10--16](ch10s04.md#gss-e11) 中所示的图。应力-应变曲线表明，在模拟的前两个增量期间，此积分点的材料行为是线弹性的。在此图中，材料在第三个增量期间似乎保持线性；但它确实在此增量期间屈服。这种错觉是由图中显示的应变范围引起的。如果将显示的最大应变限制为 0.01，最小值设置为 0.0，可以更清楚地看到第三个增量中的非线性材料行为（参见[图 10--17](ch10s04.md#gss-mises-stress)）。

**图 10–17** 角单元中沿耳的 Mises 应力与平动应变（E11）的关系。最大应变 0.01。

![](../graphics/gss-mises-stress-c-nls.png)

此应力-应变曲线包含另一个表观误差。材料似乎在 250 MPa 时屈服，远低于初始屈服应力。但是，此误差是由 Abaqus/Viewer 使用直线连接曲线上的数据点这一事实引起的。如果限制增量大小，图形上的额外点将更好地显示材料响应，并显示屈服恰好发生在 380 MPa。

第二次模拟的结果表明，如果钢在屈服后硬化，耳将承受 60 kN 载荷。综合两个模拟的结果，表明确定钢的实际后屈服硬化行为非常重要。如果钢硬化很少，耳可能在 60 kN 载荷下坍塌。而如果它具有中等硬化，耳可能会承受载荷，尽管耳中会有广泛的塑性屈服（参见[图 10--14](ch10s04.md#gss-plastic-strain)）。但是，即使有塑性硬化，此载荷的安全系数也可能非常小。
