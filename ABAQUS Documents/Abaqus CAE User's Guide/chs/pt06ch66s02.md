# 66.2 对场和历史数据应用边界值







过滤器定义中的边界值使您能够调查特定输出请求的最大值或最小值，并建立返回给输出请求的变量值的上限或下限。您可以为过滤和未过滤的数据建立边界值；要仅在过滤器定义中指定这些值而不对数据执行 Butterworth 过滤或任一类型的 Chebyshev 过滤，请定义一个 **Operator**（算子）过滤器，它使您能够在不以任何其他方式更改输出数据的情况下对输出请求实施边界值。

Abaqus/CAE 提供三种类型的边界值：
- **Maximum**（最大值）边界值返回输出请求中变量在每个输出区间内的最高值。
- **Minimum**（最小值）边界值返回输出请求中变量在每个输出区间内的最低值。
- **Absolute maximum**（绝对最大值）边界值返回每个输出区间内输出请求中变量的最高绝对值。

此外，您可以设置边界值的限制，它决定 Abaqus/CAE 为使用此过滤器的变量记录的最高或最低值。如果需要，当输出请求中的任何变量达到此限制值时，您可以停止分析。

默认情况下，张量或矢量量的每个分量分别被过滤，最大值、最小值或绝对最大值以及限制值分别报告给每个分量。但是，您可以直接将过滤器应用于不变量。
