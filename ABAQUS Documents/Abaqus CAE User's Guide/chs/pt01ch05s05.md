# 5.5 控制透视





透视表示准确地描绘了二维平面中三维对象的相互关系。换句话说，当透视开启时，屏幕上的三维模型看起来更真实。或者，当透视关闭时，模型中的平行线看起来是平行的。透视影响除 *X–Y* 绘图之外的所有绘图，适用于所有模块，默认情况下是开启的。

- 要开启透视，请选择**视图选项**工具栏中的![](../graphics/ico_viewPerspective.png)图标，或从主菜单栏中选择****View**![](../graphics/images/arrow.gif)**Perspective****。
- 要关闭透视，请选择**视图选项**工具栏中的![](../graphics/ico_viewParallel.png)图标，或从主菜单栏中选择****View**![](../graphics/images/arrow.gif)**Parallel****。

你的更改仅适用于当前视口，并在会话期间保存。
