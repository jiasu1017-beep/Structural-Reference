# 20.2.4 aStudy.execute()





### 20.2.4 ***aStudy*.execute()**：执行参数化研究设计的分析。

**产品：** Abaqus/Standard  Abaqus/Explicit  

此命令用于执行参数化研究生成的设计的分析。

##### **参考**

- ["参数化研究脚本，" 第20.1.1节](pt04ch20s01aus108.md)

### **命令：**

***aStudy*.execute** (*token*, **execOptions**= , *additional data*)

### **符号：**

ALL

使用此符号顺序执行参数化研究所有设计的分析。此选项是默认值。

DISTRIBUTED

使用此符号使用本地和/或远程计算机的指定队列接口执行所有设计的分析。类似数量的分析将被分配到每个指定的队列。

INTERACTIVE

使用此符号在交互模式下顺序执行参数化研究所有设计的分析。在这种情况下，进程暂停以提示您输入进一步执行指令。提示允许您指定要执行的分析数量、执行剩余分析、指定要跳过的执行数量或跳过所有剩余分析。

### **可选数据：**

**execOptions**

将**execOptions**设置为在执行参数化研究设计的分析时添加到Abaqus执行命令的Abaqus执行选项字符串（参阅["Abaqus/Standard、Abaqus/Explicit和Abaqus/CFD执行，" 第3.2.2节](pt01ch03s02abx02.md)）；此字符串必须用匹配引号括起来。

##### **DISTRIBUTED的额外数据：**

### **必需数据：**

**queues**

将**queues**设置为队列接口名称或队列接口名称序列。如果给出单个名称，则必须用匹配引号括起来。如果给出名称序列，则必须将其括在括号或方括号内，并包含用匹配引号括起来并用逗号分隔的队列接口名称。

### **可选数据：**

**files**

将**files**设置为符号常量或符号常量序列，用于标识远程执行后必须返回到本地计算机的文件。序列项必须用逗号分隔，序列必须用括号或方括号括起来。

允许的符号常量为：`DAT`、`LOG`、`FIL`、`SEL`、`MSG`、`STA`、`ODB`、`IPM`、`RES`、`ABQ`和`PAC`。默认值为**files**` = (DAT, FIL, LOG, ODB, SEL)`。

##### **定义队列和队列接口：**

在为分布式参数化研究使用之前，必须在Abaqus环境文件的**design_startup**部分内定义队列接口。例如，要为远程计算机`server`上的现有队列`short`定义队列接口，环境文件中需要以下条目：

```
def onDesignStartup():
    from par_Session import Queue
    import os

    # convenience assignment
    SCRATCH = '/scratch/' + os.environ['USER']

    # create remote queue interface
    Queue(name='short_interface', hostName='server',
          driver='abaqus', queueName='short', directory=SCRATCH)
```

如果还需要本地队列，则必须扩展条目为：
```
def onDesignStartup():
    from par_Session import Queue
    import os

    # convenience assignment
    SCRATCH = '/scratch/' + os.environ['USER']

    # create remote queue interface
    Queue(name='short_interface', hostName='server',
          driver='abaqus', queueName='short', directory=SCRATCH)

    # create local queue interface
    Queue(name='local_interface', driver='abaqus',
          queueName='local')

queue_name="local"
local="echo "./%S 1>%L 2>&1" | batch"
```
