# 17.3.7 网格细化







网格模块提供了一组用于细化网格的工具。
- 您可以使用分区工具集将几何区域划分为较小的区域。生成的分区会引入新的边缘，您可以在这些边缘上布置种子；因此，您可以将分区和种子结合使用，以获得对网格生成过程的额外控制。您还可以使用分区来创建可以分配不同单元类型的区域。例如，您可能希望为模型的某些部分分配减缩积分单元，为其他部分分配完全积分单元。 有关更多信息，请参见第 70 章"分区工具集"。
- 在某些情况下，几何形状包含非常小的面和边缘等细节。虚拟拓扑工具集允许您通过将小面与相邻面合并或将小边缘与相邻边缘合并来移除这些小细节。引入虚拟拓扑是创建干净、形状良好的网格的便捷方法。 有关更多信息，请参见第 75 章"虚拟拓扑工具集
