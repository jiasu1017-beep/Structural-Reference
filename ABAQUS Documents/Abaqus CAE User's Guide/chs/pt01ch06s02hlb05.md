# 6.2.5 使用拓扑方法选择多个单元





你可以基于单元行或层的连接来选择多个单元。当你执行允许你拾取多个单元的任务时，Abaqus/CAE 会在提示区域显示一个字段。该字段允许你在四种选择方法之间进行选择——**逐个**、**按角度**、**按特征边**和**按拓扑**，如图 [图 6-9](pt01ch06s02hlb05.md#pic-elemtopo) 所示。

**图 6-9** 从提示区域的字段中选择选择方法。

![](../graphics/pic-elemtopo-nls.png)

角度和特征边选择方法在["使用角度和特征边方法选择多个对象，" 第 6.2.3 节](pt01ch06s02hlb03.md)中有描述。

拓扑方法在大多数选择单元的过程中可用。如果过程接受除单元之外的其他对象类型，你可以在**选择**工具栏中将对象类型更改为**单元**以访问拓扑方法。

拓扑方法专为二维和三维结构化网格设计。从网格中选择一个单元面，Abaqus/CAE 选择通过网格与其连接的所有单元行。从网格中选择一个单元边，Abaqus/CAE 选择从共享所选边的单元面开始的一层中的所有单元。[图 6-10](pt01ch06s02hlb05.md#pic-faceelemtopomethod) 左侧显示选择内部行，右侧显示选择内部层。

**图 6-10** 使用拓扑方法选择单元行或层。

![](../graphics/pic-faceelemtopomethod.png)

你可以从其他网格类型中使用拓扑方法选择单元，但没有结构化网格的明确定义的行或层，选择可能会不可预测。在某些情况下，例如四面体网格，拓扑选择可能仅限于共享你选择的面的单元。

使用拓扑方法后，你可以选择提示区域中的其他方法，并 **[Shift]****+点击**将更多单元附加到你的选择。你也可以 **[Ctrl]****+点击**项目以取消选择它们。此外，你可以继续使用拓扑方法，并使用 **[Shift]****+点击**将单元附加到你的选择。更多信息，请参阅["组合选择技术，" 第 6.2.8 节](pt01ch06s02hlb08.md)。
