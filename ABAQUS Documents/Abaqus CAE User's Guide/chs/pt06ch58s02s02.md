# 58.2.2 表达式中的运算和函数概述







本节列出了表达式场编辑器中可用的运算和函数。假设三角函数的参数以弧度为单位，范围从 ![](../graphics/usi_eqn00792.gif) 到 ![](../graphics/usi_eqn00793.gif)。有关更多信息，请参阅官方 Python 主页上的 [math 模块文档](http://www.python.org/doc/current/lib/module-math.md)。 

**数学运算：**

| **+** | 加 |
| --- | --- |
| **** | 减 |
| ***** | 乘 |
| **/** | 除 |
| **%** | 返回整数除法的余数。 |
| **1/A** | 返回 **A** 的倒数。 |
| **ceil(A)** | 返回 **A** 的上限，即大于或等于 **A** 的最小整数值。 |
| **fabs(A)** | 返回绝对值。 |
| **floor(A)** | 返回 **A** 的下确界，即小于或等于 **A** 的最大整数值。 |
| **fmod(A,B)** | 返回由平台 C 库定义的 fmod(**A**, **B**)。C 标准的目的在于 fmod(**A**, **B**) 在数学上（无限精度）等于 **A** *n* * **B**（对于某个整数 *n*，使得结果与 **A** 同号且幅值小于 abs(**B**)）。 |
| **frexp(A)[ ]** | 返回 **A** 的尾数和指数作为对 (*m*, *e*)。「m」是浮点数，「e」是整数，使得 **A** = *m* * 2 ** *e* 精确。使用括号指定要在表达式评估中使用的返回值索引。 |
| **modf(A)[ ]** | 返回 **A** 的小数部分和整数部分。使用括号指定要在表达式评估中使用的返回值索引。 |

**三角函数：**

| **acos(A)** | 返回反余弦。 |
| --- | --- |
| **asin(A)** | 返回反正弦。 |
| **atan(A)** | 返回反正切。 |
| **cos(A)** | 返回余弦。 |
| **cosh(A)** | 返回双曲余弦。 |
| **hypot(A,B)** | 返回欧几里得范数，sqrt(**A*****A** + **B*****B**)。这是从原点到点 (**A**, **B**) 的向量长度。 |
| **sin(A)** | 返回正弦。 |
| **sinh(A)** | 返回双曲正弦。 |
| **tan(A)** | 返回正切。 |
| **tanh(A)** | 返回双曲正切。 |

**幂和对数函数：**

| **exp(A)** | 返回 **A** 的自然指数，![](../graphics/usi_eqn00794.gif)。 |
| --- | --- |
| **ldexp(A,B)** | 返回 **A** * (![](../graphics/usi_eqn00795.gif))。 |
| **log(A)** | 返回自然对数。 |
| **log10(A)** | 返回以 10 为底的对数。 |
| **pow(A,B)** | 将变量提升到幂。 |
| **sqrt(A)** | 返回平方根。 |

**常数：**

| **pi** | 数学常数 π。 |
| --- | --- |
| **e** | 数学常数 *e*。 |
