### 概述

Abaqus/Standard提供了用于模拟由于封闭腔内辐射而产生的热传递效应的空腔辐射功能。此空腔辐射功能：
- 可包含在无变形的热传递分析问题中（["非耦合热传递分析，" 第6.5.2节"](pt03ch06s05at18.md)，和["耦合热-电分析，" 第6.7.3节"](pt03ch06s07at22.md)）；
- 提供二维、三维和轴对称情况；
- 考虑腔内的对称性、表面阻塞和表面运动；以及
- 可包括封闭腔或开放腔（意味着一些辐射发生到外部介质）。

空腔辐射方程是非对称的；因此，在包含空腔辐射的模型中自动调用非对称矩阵存储和解决方案方案（参见["空腔辐射，" Abaqus理论指南第2.11.4节](../stm/stm-link.md#stm-anl-cavradiation)，和["定义分析，" 第6.1.2节"](pt03ch06s01abo05.md)）。每个空腔定义了一个视角系数矩阵，涉及封闭腔中表面之间的几何关系。这些矩阵可能在分析过程中多次更新（由于空腔中移动的表面）。因此，大型空腔辐射问题可能计算成本高昂。相反，您应该考虑使用：- 间隙辐射（参见["热接触属性，" 第37.2.1节"](pt09ch37s02aus174.md)）用于建模紧密间隔表面之间的辐射；
- 平均温度辐射条件，用于建模近似等温的空腔，具有恒定发射率，不需要阻塞或反射考虑（参见["热载荷，" 第34.4.4节"](pt07ch34s04aus123.md)）；或
- 并行空腔分解，用于视角系数和辐射热传递方程求解的并行计算（参见下文["并行分解大型空腔"](pt09ch41s01aus187.md#usb-cni-acavityradiation-parallel)）。

### 定义空腔辐射问题

由于空腔辐射效应仅在热传递和耦合热-电过程中计算，因此可以包含这些效应的唯一热-应力分析类型是顺序耦合热-应力分析（参见["顺序耦合热-应力分析，" 第16.1.2节"](pt04ch16s01at39.md)）。此外，除非允许空腔并行分解（参见下文["并行分解大型空腔"](pt09ch41s01aus187.md#usb-cni-acavityradiation-parallel)），否则Abaqus/Standard中的软件限制为16,000个节点和面。

#### 模型定义

定义空腔辐射问题的模型时，必须：

1. 定义空腔中的所有表面（参见["定义表面"](pt09ch41s01aus187.md#usb-cni-acavityradiation-definesurfaces)"）；
2. 定义每个表面的辐射属性（即发射率）和物理常数（参见["定义表面辐射属性"](pt09ch41s01aus187.md#usb-cni-acavityradiation-surfprops)"）；和
3. 从表面构建空腔（参见["构建空腔"](pt09ch41s01aus187.md#usb-cni-acavityradiation-constructcavity)"）。

#### 历史定义

在空腔辐射分析的第一步中，必须将辐射视角系数定义与每个空腔关联，该定义控制空腔视角系数的计算。然后您可以：

1. 定义空腔对称性（如果有）（参见["定义空腔对称性"](pt09ch41s01aus187.md#usb-cni-acavityradiation-cavsymm)"）；
2. 规定表面的运动（参见["在空腔辐射分析期间规定运动"](pt09ch41s01aus187.md#usb-cni-acavityradiation-prescribemotion)"）；
3. 定义边界条件，如温度和强制对流（参见["边界条件"](pt09ch41s01aus187.md#usb-cni-acavityradiation-bc)"）；
4. 控制每个步骤中的空腔辐射和视角系数计算（如果未在步骤中重新定义，则使用上一步骤的规范；参见["控制分析期间的视角系数计算"](pt09ch41s01aus187.md#usb-cni-acavityradiation-viewfactorcalc)"）；
5. 请求将热传递变量输出到数据和结果文件（参见["请求表面变量输出"](pt09ch41s01aus187.md#usb-cni-acavityradiation-outputvars)"）；和
6. 请求辐射视角系数矩阵的输出（参见["将视角系数矩阵写入结果文件"](pt09ch41s01aus187.md#usb-cni-acavityradiation-viewfactoroutput)"）。

如果您的分析中包含上述任何一项，则必须在热传递或耦合热-电步骤定义中定义它们。

#### 限制

与空腔辐射关联的表面除了["基于单元的表面定义，" 第2.3.2节"](pt01ch02s03aus17.md)中概述的一般表面定义限制外，还受以下限制：
- 表面不能重叠，因为这会导致相关属性定义和阻塞规范产生歧义。
- 一个表面只能在一个空腔定义中使用（同一表面不能出现在两个不同的空腔中）。

此外，三维四边形面应尽可能接近平面；否则，视角系数计算的质量将受到影响。

#### 控制虚假空间振荡

每个面的辐射热通量基于该面上节点温度的平均值计算（参见["空腔辐射，" Abaqus理论指南第2.11.4节](../stm/stm-link.md#stm-anl-cavradiation)）。然后根据每个节点的面积比例将辐射热通量分配给每个节点。因此，mesh必须足够精细，使得单元之间的温度差异很小。否则，温度高于面平均值的节点计算的热通量将过低，低于平均温度的节点的热通量将过高。这往往会引起空间振荡解。可以通过在高温梯度附近减小单元尺寸来消除这种效应。

### 定义表面辐射属性

空腔辐射问题本质上是非线性的，这是由于辐射热通量对面温度四次方的依赖性。此外，通过将发射率描述为温度的函数，可以引入非线性。

##### 创建具有多个开口或复杂环境条件的空腔

开放空腔定义允许空腔向具有单一恒定温度值的环境进行单一开口。如果空腔有多个开口或环境温度不是恒定的，则应以不同的方式对周围环境进行建模。

您应该用单元封闭任何空腔开口，并在这些单元上规定外部介质的温度。由于空腔现在是封闭的，您不应在空腔定义中指定环境温度。您用于封闭单元的温度定义提供了环境温度，它允许您在空腔开口处指定不同的温度，包括可变温度。建模外部介质的单元不应与空腔单元共享节点（以便它们之间不会发生传导）。由外部介质单元定义的表面应具有1的发射率。

### 并行分解大型空腔

默认情况下，Abaqus/Standard在视角系数矩阵计算和辐射热传递方程求解（参见["空腔辐射，" Abaqus理论指南第2.11.4节](../stm/stm-link.md#stm-anl-cavradiation)）中使用单个工作线程。此方法对于由数百个面组成的小型空腔是稳健且有效的，但对于由数千个面组成的大型空腔，它变得效率低下且计算成本高昂。此外，这些空腔的内存需求对于单个计算节点来说可能非常大（视角系数矩阵的大小是面数的平方）。在这些情况下，您应该考虑允许Abaqus/Standard在视角系数计算和辐射热传递方程求解期间将空腔分解到所有CPU上。

| **输入文件用法：** | 使用以下选项激活空腔并行分解： |
| --- | --- |
| | ``` [*CAVITY DEFINITION](../key/key-link.md#usb-kws-mcavitydef), NAME=*cavity_name*, PARALLEL DECOMPOSITION=ON ``` |

| **Abaqus/CAE用法：** | Abaqus/CAE不支持空腔并行分解。 |
| --- | --- |

#### 并行求解辐射热传递方程

启用空腔并行分解时，Abaqus/Standard使用迭代求解技术来获得辐射热通量。此技术基于Krylov方法，采用预处理器，仅使用基于MPI的并行化（详细信息请参见["Abaqus/Standard中的并行执行，" 第3.5.2节"](pt01ch03s05aus33.md)）。此迭代技术仅用于求解空腔辐射方程，不需要用户干预。您仍可以选择使用迭代求解器或直接稀疏求解器来求解热传递有限元方程。

#### 具有分解空腔的模型的收敛性

无论是否允许并行分解，都求解精确的空腔辐射方程；但是，当并行分解处于活动状态时，Abaqus/Standard可能需要更多迭代来获得解。这种较慢的收敛率来自对Jacobian（辐射热通量的线性化）的近似，该近似基于照射（小值（非由于表面发射的部分）的变化）。具有低发射率的表面和稳态分析涉及的模型可能特别受影响。如果您在使用并行分解空腔时遇到收敛问题，可以考虑
- 将分析从稳态改为瞬态（["非耦合热传递分析，" 第6.5.2节"](pt03ch06s05at18.md)）；或
- 允许更多求解器迭代每次时间增量（["非线性问题的收敛准则，" 第7.2.3节"](pt03ch07s02aus51.md)）。

#### 具有分解空腔的模型的运动约束

运动约束（例如，耦合约束、线性约束方程、多点约束或基于表面的绑定约束）可以应用于允许并行分解的空腔所属的任何节点或表面。但是，节点或表面必须是约束定义中的独立（主）节点或表面。

### 定义空腔对称性

利用几何对称性可以减少计算模型大小和模拟时间。您不必对对称组装中的所有零件或组件进行建模，而是可以对较小的重复组件进行建模，并在空腔辐射相互作用的定义中考虑对称性。在Abaqus/Standard中，具有定义对称性的空腔定义考虑每个空腔面与空腔中所有面及其所有对称图像之间的辐射相互作用。Abaqus/Standard不检查使用空腔对称性创建的模型在物理上是否真实。您必须仔细检查输入和结果以确保创建了有效的模型。

您必须为每个辐射对称性定义分配一个名称，以供辐射视角系数定义引用。辐射视角系数定义和相应的辐射对称性定义必须出现在同一步骤中。

可以按如下所述定义循环、周期和/或反射对称性。

#### 周期对称

您可以通过给定方向的周期重复来定义空腔对称性。从物理上讲，周期对称被理解为在周期间隔处相同图像的无限重复。从数值上讲，周期对称必须由有限数量的重复图像表示。您可以定义数值计算中使用的重复次数，*n*。

周期对称将导致空腔由用户定义的空腔加上两倍*n*个相似图像组成，因为假定周期对称在正负两个方向上都适用。默认情况下，*n*=2。

尽管对称性不会增加视角系数矩阵的大小，但它们确实会使其计算更加昂贵。因此，应最小化重复次数，但*n*的值应该足够大，以准确计算视角系数矩阵。输出变量VFTOT可用于检查对称性所暗示的封闭程度。（参见下文["控制视角系数计算的准确性"](pt09ch41s01aus187.md#usb-cni-acavityradiation-viewfactor-vtol)。）用于定义空腔辐射视角系数矩阵的周期对称性不会自动在热传递分析中施加对称条件。可能需要在周期对称平面上的节点上施加适当的温度和载荷条件约束，以从底层热传递分析中获得有意义的解决方案。

### 在空腔辐射分析期间规定运动

在许多空腔辐射问题中（如制造序列模拟），辐射视角系数会因表面在分析过程中移动而发生变化。您可以在热传递或耦合热-电分析期间指定表面运动。

规定的运动仅影响由于空腔辐射导致的热传递中的视角系数（因此是辐射热通量）计算。它们不影响热传导、存储或分布式通量贡献。

您可以独立定义步骤中运动的平移和旋转分量。例如，您可以根据某个幅值函数规定节点集的平移运动，然后根据不同的幅值函数规定节点集的旋转运动。在每个步骤中，任何特定节点的每个运动分量只能指定一次。

也可以在关闭空腔辐射的步骤中规定运动，如下所述。

#### 平移运动

平移，![](../graphics/usb_eqn06405.gif)，以全局*x*-、*y*-和*z*分量指定，除非在为指定运动的节点处定义了局部坐标系；然后平移以局部*x*-、*y*-和*z*分量指定（参见["变换坐标系，" 第2.1.5节"](pt01ch02s01aus09.md)）。

平移位移始终指定为平移运动的总量值。这种对平移的处理方式与应力/位移分析中的位移边界条件（["Abaqus/Standard和Abaqus/Explicit中的边界条件，" 第34.3.1节"](pt07ch34s03aus118.md)）一致。默认是施加平移运动。

也可以指定平移速度。平移速度始终指当前步骤；因此，规定为速度的平移运动速率仅在定义它的步骤期间有效。这与速度边界条件不同，速度边界条件如果未重新定义，则会在后续步骤中保持有效。

| **输入文件用法：** | 使用以下任一选项规定平移运动： |
| --- | --- |
| | ``` [*MOTION](../key/key-link.md#usb-kws-hmotion), TRANSLATION, TYPE=DISPLACEMENT [*MOTION](../key/key-link.md#usb-kws-hmotion), TRANSLATION, TYPE=VELOCITY ``` |

| **Abaqus/CAE用法：** | Abaqus/CAE不支持空腔辐射的表面运动。 |

定义旋转轴的点的坐标必须在定义刚性体旋转的步骤开始时的构型中定义。

步骤中刚性体旋转引起的运动被指定为仅在该步骤期间发生的旋转量。因此，步骤中规定的刚性体旋转是局部于该步骤的；如果下一步骤中未规定刚性体旋转，则不会发生进一步旋转。

在第二步中，物体仅在*y*方向上平移3个长度单位。此运动将物体放置在位置*D*，无额外旋转。最后，在第三步中，物体同时沿与*y*方向成53.13度角的方向平移5个长度单位，并以恒定角速度再次顺时针旋转180度围绕*z*轴。此运动将物体返回到其原始位置。

### 控制分析期间的视角系数计算

空腔辐射功能可用于制造序列模拟等应用，其中辐射视角系数在模拟过程中会发生变化。因此，辐射视角系数定义为在步骤中控制视角系数计算提供了显著的灵活性。

如果不同的空腔需要不同类型的辐射和视角系数计算，则可以在步骤定义中指定多个辐射视角系数定义。可以在分析的不同步骤中为同一空腔指定不同类型的视角系数计算。

默认情况下，视角系数在包含辐射视角系数定义的第一步开始时计算。视角系数仅在后续步骤开始时重新计算（如果视角系数定义在该步骤中发生了变化；例如，如果为同一空腔指定了不同的表面阻塞检查）。在重新启动分析中，Abaqus/Standard从用户指定的重新启动步骤和增量中读取辐射视角系数，仅在视角系数定义发生变化时才重新计算。

您可以指定正在为其指定辐射视角系数控制的空腔名称。如果不指定空腔名称，则辐射视角系数定义适用于模型中的所有空腔。

| **输入文件用法：** | ``` [*RADIATION VIEW FACTOR](../key/key-link.md#usb-kws-hradviewfactor), CAVITY=*cavity_name* ``` |
| --- | --- |

| **Abaqus/CAE用法：** | 辐射视角系数分别为每个空腔辐射相互作用定义，并适用于该相互作用处于活动状态的所有步骤。 |
| --- | --- |

#### 激活和停用空腔辐射

在某些情况下，在分析过程中打开和关闭空腔辐射效应可能很有用。例如，辐射可能正在空腔中进行，然后充满流体，因此辐射不再显著；稍后在分析中，当流体从空腔中排出时，辐射可能恢复。在这种情况下，您可以使用辐射视角系数定义在分析的一个或多个步骤中关闭和打开特定空腔中的辐射。

当空腔辐射在关闭后重新打开时，Abaqus/Standard将使用最后一步中计算的最后视角系数计算其中空腔辐射处于活动状态。但是，如果在空腔辐射关闭期间规定了运动，并且指定节点集中某个节点的位移分量超过了关闭空腔辐射的步骤中指定的最大允许运动*max*的值，则视角系数将在重新打开空腔辐射的步骤开始时重新计算。

| **Abaqus/CAE用法：** | 无法为选定步骤关闭或打开辐射视角系数。您可以使用以下选项关闭或打开空腔辐射相互作用： |

##### 具有对称性的空腔中的视角系数计算

视角系数计算考虑了任何空腔对称性所暗示的空腔封闭。对于没有周期或循环对称性的空腔，视角系数针对二维几何形状精确计算，但针对轴对称和三维几何形状进行近似。这些近似值随着表面之间距离的减小而变得不那么准确。定义热辐射以模拟紧密间隔的表面（参见["热接触属性，" 第37.2.1节"](pt09ch37s02aus174.md)）。

##### 开放空腔中的视角系数计算

如果开放空腔（通过指定环境温度值定义）中面的视角系数之和与1的偏差大于指定的视角系数容差，则会发生向环境的辐射。在近乎封闭的空腔中，此偏差可能很小。如果未违反容差，即使空腔被定义为开放，也不会包括向外部介质的辐射；将发出有关此的警告消息。您可以放宽视角系数容差以包括此类辐射。

#### 空腔辐射分析中的内存使用

Abaqus中空腔表面面之间的空腔辐射热传递使用完整的非对称矩阵建模，该矩阵定义空腔中每个节点与所有其他节点之间的相互作用。对于具有大量节点的面，此矩阵可能很大，导致内存需求明显大于没有空腔辐射相互作用的有限元部分分析的内存需求。

为了最小化空腔辐射热传递分析的内存需求和计算成本，可以使用更粗mesh的热传递壳单元定义空腔，每个节点只有一个自由度。覆盖单元应具有最小的热容和传导，且应用于定义空腔而不是物理的多自由度壳。覆盖单元应用于在绑定耦合约束中定义主表面（["mesh绑定约束，" 第35.3.1节"](pt08ch35s03aus132.md)）；多自由度、物理、热传递壳单元形成从表面。

### 初始条件

默认情况下，所有节点的初始温度为零。您可以在空腔辐射分析中指定非零初始温度；请参见["在Abaqus/Standard和Abaqus/Explicit中定义初始温度"第34.2.1节"](pt07ch34s02aus116.md#usb-prc-pinitialcond-temp)中的"定义初始温度"。

在涉及通过mesh进行强制对流的热传递分析中，您可以定义强制对流/扩散热传递单元模型节点处的非零初始质量流率（参见["非耦合热传递分析，" 第6.5.2节"](pt03ch06s05at18.md)）。

### 边界条件

您可以指定边界条件来规定节点的温度（自由度11）（参见["Abaqus/Standard和Abaqus/Explicit中的边界条件，" 第34.3.1节"](pt07ch34s03aus118.md)）。壳单元在厚度方向上有额外的温度自由度12、13等（参见["约定，" 第1.2.2节"](pt01ch01s02aus02.md)）。边界条件可以通过引用幅值曲线指定为时间的函数（["幅值曲线，" 第34.1.2节"](pt07ch34s01aus115.md)）。

对于纯扩散单元，没有任何规定边界条件的边界（自然边界条件）对应于绝热表面。对于强制对流/扩散单元，仅与传导相关的通量为零；能量可以自由跨越未加载表面。这种自然边界条件正确地模拟了流体穿过表面的区域（例如，在mesh的上游和下游边界处），并防止能量虚假反射回mesh。

#### 请求表面变量输出

对于空腔辐射相互作用，您可以请求将空腔、基于单元或基于表面的辐射输出（如辐射热通量、面的视角系数总和和面温度）写入数据、结果和/或输出数据库文件。输出请求可以根据需要重复，以请求不同变量、不同空腔、不同表面、不同单元集等的输出。可以请求的表面变量列于上方。

您可以指定正在为其请求输出的特定空腔、单元集或表面。如果不指定空腔、单元集或表面，则将为模型中的所有空腔提供输出。相同的空腔、单元集或表面可以出现在多个辐射输出请求中。

默认情况下，不会提供空腔辐射数据输出。如果您定义辐射输出请求而未指定所需的输出变量，则将输出所有六个空腔辐射表面变量。

您可以通过指定所需的输出频率（以增量为单位）来控制辐射输出的频率。默认输出频率为1。指定输出频率为0以抑制输出。除非您指定输出频率为0，否则输出将始终在每个步骤的最后增量写入。

##### 打印输出

通过辐射输出请求到数据文件生成的输出表按表面对表面的方式组织。将出现在特定表中的行由选择空腔、表面或单元集定义：表的每一行对应于组成所选空腔、表面或单元集的单个单元面。如果表中一行的所有变量都为零，则不打印该行。

每个表的第一列是单元编号，第二列是单元面标识符。您选择要出现在其余列中的变量。可以定义的表数量没有限制。

例如，考虑一个包含名为`CAV1`的空腔的热传递模型，该空腔又由表面`SURF1`和`SURF2`组成。如果您请求将此模型的辐射热通量（RADFL）和面温度（FTEMP）输出到数据文件，数据文件中将出现两个表。一个表将包含组成表面`SURF1`的所有单元面的RADFL和FTEMP输出，另一个表将包含组成表面`SURF2`的所有单元面的相同输出变量。

默认情况下，Abaqus/Standard写入表中每列最大值和最小值的摘要。您可以选择抑制此摘要。此外，您可以选择打印表中每列的总和，例如，用于对组成辐射表面的所有面的辐射热通量求和。默认情况下，不打印这些总和。

| **输入文件用法：** | 使用以下选项控制写入数据文件的摘要信息输出： |
| --- | --- |
| | ``` [*RADIATION PRINT](../key/key-link.md#usb-kws-hradprint), SUMMARY=YES *or* NO ``` 使用以下选项控制写入数据文件的总和输出： ``` [*RADIATION PRINT](../key/key-link.md#usb-kws-hradprint), TOTALS=YES *or* NO ``` |

| **Abaqus/CAE用法：** | Abaqus/CAE不支持写入数据文件的空腔辐射输出。 |
