# 8.1.2 向弹性裂纹尖端的扩散







**产品：** Abaqus/Standard

这个简单的二维问题验证了Abaqus中顺序耦合的应力辅助质量扩散能力。Abaqus中使用的质量扩散 formulations 在 ["质量扩散分析，" Abaqus Analysis User's Guide 第6.9.1节](../usb/usb-link.md#usb-anl-amassdiffusion) 和 ["质量扩散分析，" Abaqus Theory Guide 第2.13.1节](../stm/stm-link.md#stm-anl-massdiffusion) 中描述。

一块中心带有裂纹的2 1/4 Cr-1 Mo钢合金板在富氢环境中承受端部加载。氢被吸引到裂纹尖端区域的高静水应力处，可能有助于氢脆导致的裂纹扩展。本例关注的是问题的氢扩散方面。

### 几何与模型

问题几何和边界条件如图8.1.2-1所示（[ch08s01aex128.md#sxmh2odifftip-geom](ch08s01aex128.md#sxmh2odifftip-geom)）。试样厚度为10毫米，宽度为20毫米，高度为80毫米，中心有4毫米的裂纹。裂纹附近的网格集中在裂纹尖端，单元尺寸随到裂纹尖端距离的平方增长（第一个有界节点集表示裂纹尖端）。使用非常细的网格（参见图8.1.2-2）来准确捕获裂纹尖端附近的浓度和应力梯度。给出了四种应力与质量扩散分析组合：
- 应力分析，使用裂纹尖端的二次单元和四分之一节点间距，随后进行质量扩散分析，使用线性单元。
- 应力分析，使用裂纹尖端的二次单元和四分之一节点间距，随后进行质量扩散分析，使用裂纹尖端的二次单元和四分之一节点间距。
- 应力分析，使用二次单元（无四分之一节点间距），随后进行质量扩散分析，使用二次单元（无四分之一节点间距）。
- 应力分析，使用线性单元，随后进行质量扩散分析，使用线性单元。

在断裂力学分析中使用四分之一节点间距技术在裂纹尖端强制执行 ![](../graphics/exa_eqn00720.gif) 奇异性，其中*r*是到裂纹尖端的距离。

从图中可以看出，除了第一个单元外，所有四种单元组合的有限元结果都是相同的，在那里结果预计无效。结果表明，当到裂纹尖端的距离*r*接近零时，等效压力应力和浓度与解析预测的解决方案都有良好的一致性。离裂纹尖端更远，解析解和有限元解之间的偏差增加。这种偏差与线弹性裂纹尖端解仅在*r*接近零时有效这一事实一致。

关于裂纹尖端区域单元数量的网格收敛研究尚未进行。为了与这里给出的解进行比较，使用了接近裂纹尖端的等间距单元进行分析。结果（此处未显示）表明，有必要将单元偏向裂纹尖端，以充分捕获浓度和等效压力应力的梯度。此外，等效压力应力结果表明，只要网格足够细化，在裂纹尖端使用四分之一节点定位节点的影响在该问题中不显著。

有限元和解析预测浓度之间的差异直接源于有限元和解析预测的压力应力值之间的差异。如果使用等效压力应力的解析值来驱动Abaqus浓度解，结果曲线与所示的解析浓度无法区分。

### 输入文件

[difftocrack_quarterpstress.inp](../eif/difftocrack_quarterpstress.inp)

裂纹尖端具有四分之一节点间距的二次应力分析。该分析写入用于 difftocrack_linearmassdiff1.inp 和 difftocrack_quarterpmassdiff.inp 的结果文件。

[difftocrack_linearmassdiff1.inp](../eif/difftocrack_linearmassdiff1.inp)

从 difftocrack_quarterpstress.inp 读取结果文件数据的线性质量扩散分析。

[difftocrack_stress.inp](../eif/difftocrack_stress.inp)

使用二次单元（无四分之一节点间距）的应力分析。该分析写入用于 difftocrack_massdiff.inp 的结果文件。

[difftocrack_massdiff.inp](../eif/difftocrack_massdiff.inp)

使用二次单元的质量扩散分析，从 difftocrack_stress.inp 写入的结果文件读取等效压力应力。

[difftocrack_quarterpmassdiff.inp](../eif/difftocrack_quarterpmassdiff.inp)

使用裂纹尖端四分之一节点间距的二次单元进行质量扩散分析。该分析从 difftocrack_quarterpstress.inp 写入的结果文件读取等效压力应力。

[difftocrack_linearstress.inp](../eif/difftocrack_linearstress.inp)

使用线性单元的应力分析。该分析写入用于 difftocrack_linearmassdiff2.inp 的结果文件。

[difftocrack_linearmassdiff2.inp](../eif/difftocrack_linearmassdiff2.inp)

使用线性单元的质量扩散分析，从 difftocrack_linearstress.inp 写入的结果文件读取等效压力应力。

[difftocrack_node.inp](../eif/difftocrack_node.inp)

所有分析的节点数据。

[difftocrack_quad_elements.inp](../eif/difftocrack_quad_elements.inp)

使用二次单元的分析的单元数据。

[difftocrack_linear_elements.inp](../eif/difftocrack_linear_elements.inp)

使用线性单元的分析的单元数据。
