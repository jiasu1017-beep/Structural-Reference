# 9.8.1 在模型数据库中操作模型





模型数据库可以包含多个模型。虽然您一次只能打开一个模型数据库，但可以同时打开多个模型。主窗口的标题栏显示模型数据库的名称，每个视口的标题栏显示与视口关联的模型的名称。当前视口由深灰色标题栏指示；与当前视口关联的模型称为当前模型。当前模型的名称也显示在上下文栏中的**模型（Model）**列表中。

要创建新模型，请从主菜单栏选择****模型（Model）**![](../graphics/images/arrow.gif)**创建（Create）****，并在出现的**编辑模型属性（Edit Model Attributes）**对话框中输入模型的名称。

要打开模型并将其与当前视口关联，请从上下文栏中的**模型（Model）**列表中选择所需的模型。**模型（Model）**列表包含当前模型数据库中的所有模型。

要复制、重命名或删除模型，请从主菜单栏上的**模型（Model）**菜单中选择**复制模型（Copy Model）**、**重命名（Rename）**或**删除（Delete）**项目。**复制模型（Copy Model）**、**重命名（Rename）**和**删除（Delete）**项目包含列出当前模型数据库中所有模型的子菜单。有关如何使用这些菜单的一般信息，请参阅["使用管理器菜单管理对象，"第3.4.10节"](pt01ch03s04hlb02.md)。

您也可以使用**模型管理器（Model Manager）**创建、复制、重命名和删除模型。要显示**模型管理器（Model Manager）**，请从主菜单栏选择****模型（Model）**![](../graphics/images/arrow.gif)**管理器（Manager）****。**模型管理器**对话框包含与**模型（Model）**菜单下列出的相同功能，但带有一个方便的浏览器，列出当前模型数据库中所有可用的模型。有关如何使用管理器的一般信息，请参阅["管理对象，"第3.4节"](pt01ch03s04.md)。

您可以将模型复制到模型数据库中的新模型。此外，您可以在模型数据库中的模型之间复制对象，如草图、零件和材料；有关更多信息，请参阅["在模型之间复制对象，"第9.8.3节"](pt02ch09s07hlb03.md)。您也可以从另一个Abaqus/CAE模型数据库将模型复制到当前模型数据库中的新模型；有关更多信息，请参阅["从Abaqus/CAE模型数据库导入模型，"第10.5.1节"](pt02ch10s05s01.md)。
