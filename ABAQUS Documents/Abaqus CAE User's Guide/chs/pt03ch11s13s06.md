# 11.13.6 控制旋转特征的方向





创建具有旋转基础特征的零件或向三维零件添加旋转特征时，可以控制旋转方向。如果在旋转特征中包含pitch，也可以控制平移方向。以下描述提供了控制Abaqus/CAE中创建的任何旋转特征的旋转方向以及适用时的平移方向的详细信息：

**创建旋转特征时选择方向**

完成草图以创建旋转特征时，Abaqus/CAE会显示包含指示旋转方向箭头的草图轮廓。如果您正在创建具有旋转基础特征的零件，也会显示旋转轴。此外，Abaqus/CAE会显示**编辑旋转**对话框。

如果需要，旋转视图直到可以辨别指示旋转方向的箭头方向。您可以通过在对话框中单击**旋转方向**的来反转旋转方向。视口中的箭头会更改方向以显示新的旋转方向。

如果您为旋转特征选择**包含平移**，第二个箭头会出现在视口中，指示沿旋转轴的平移方向。单击对话框中**Pitch方向**的来反转平移方向。

**编辑现有旋转零件或特征的方向**

当您选择要编辑的旋转特征时，Abaqus/CAE会在视口中高亮显示选定的特征，并出现**编辑特征**对话框。

要反转旋转方向，请在**编辑特征**对话框中切换**翻转旋转方向**。单击**应用**查看更改。

如果正在编辑的旋转特征包含pitch，您可以编辑平移方向。切换**翻转pitch方向**以反转平移方向。单击**应用**查看更改。

单击**确定**接受更改。
