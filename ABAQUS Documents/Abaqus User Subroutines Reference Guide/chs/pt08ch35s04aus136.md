### 概述

埋入单元技术：
- 用于指定一个单元或一组单元埋入到宿主单元组中，宿主单元的响应将用于约束埋入节点（即埋入单元的节点）的平动自由度；
- 可用于几何线性或非线性分析；
- 不适用于具有旋转自由度的宿主单元；
- 可用于模拟一组钢筋增强的膜单元、壳单元或面单元埋入到一组三维实体（连续体）单元中；一组桁架或梁单元埋入到一组实体单元中；或一组实体单元埋入到另一组实体单元中；
- 当壳单元或梁单元埋入实体单元时，不会约束埋入节点的旋转自由度；以及
- 可以从Abaqus/Standard导入到Abaqus/Explicit，反之亦然。

### 引言

埋入单元技术用于指定一个单元或一组单元埋入到"宿主"单元中。例如，埋入单元技术可用于模拟钢筋加固。Abaqus搜索埋入单元节点与宿主单元之间的几何关系。如果埋入单元的节点位于宿主单元内，则该节点的平动自由度被消除，节点成为"埋入节点"。埋入节点的平动自由度被约束到宿主单元相应自由度的插值。埋入单元可以具有旋转自由度，但这些旋转不受埋入约束。允许多个埋入单元定义。

### 可用的埋入单元类型

埋入单元和宿主单元的单元集可以使用不同的单元类型。但是，所有宿主单元只能有平动自由度，埋入单元节点处的平动自由度数量必须与宿主单元节点处的平动自由度数量相同。

提供以下"埋入单元-宿主单元"的一般类型：
- 二维模型： - 梁-实体 - 实体-实体 - 桁架-实体
- 轴对称模型： - 膜-实体（仅Abaqus/Standard） - 壳-实体 - 实体-实体 - 面-实体（仅Abaqus/Standard）
- 三维模型： - 梁-实体 - 膜-实体 - 壳-实体 - 实体-实体 - 面-实体 - 桁架-实体

### 指定宿主单元

默认情况下，搜索埋入单元附近的单元，以找到包含埋入节点的单元；然后这些宿主单元的响应约束埋入节点。为了排除某些单元约束埋入节点，可以定义宿主单元集；搜索将限于模型中宿主单元的这个子集。如果埋入节点接近模型中的不连续处（裂缝、接触对等），强烈建议使用此功能。

| **输入文件用法：** | ``` [*EMBEDDED ELEMENT](../key/key-link.md#usb-kws-membeddedelement), HOST ELSET=*name* ``` |
| --- | --- |
|  | [*EMBEDDED ELEMENT](../key/key-link.md#usb-kws-membeddedelement)选项必须包含在输入文件的模型定义部分。允许有多个[*EMBEDDED ELEMENT](../key/key-link.md#usb-kws-membeddedelement)选项。 |

| **Abaqus/CAE用法：** | 相互作用模块：**创建约束**：**埋入区域**：从提示区域选择宿主区域时选择**选择区域** |
| --- | --- |

### 指定埋入单元

必须指定埋入单元。可以指定单个单元或单元集。默认情况下，如果Abaqus未能成功将所有指定的埋入单元完全埋入宿主单元，将发出错误消息。可选地，可以允许部分埋入，即只有埋入单元中位于宿主单元内的那些节点将被约束。

埋入单元可能与宿主单元共享一些节点。但是，这些节点不会被视为埋入节点。

| **输入文件用法：** | 使用以下选项完全埋入单元（默认）： |
| --- | --- |
|  | ``` [*EMBEDDED ELEMENT](../key/key-link.md#usb-kws-membeddedelement), PARTIAL EMBED=NO *埋入单元* ``` 使用以下选项部分埋入单元：``` [*EMBEDDED ELEMENT](../key/key-link.md#usb-kws-membeddedelement), PARTIAL EMBED=YES *埋入单元* ``` |

| **Abaqus/CAE用法：** | 在Abaqus/CAE中只能完全埋入单元。 |
| --- | --- |
|  | 相互作用模块：**创建约束**：**埋入区域**：选择埋入区域 |

### 定义几何容差

几何容差用于定义埋入节点可以远离模型中宿主单元区域的距离。默认情况下，埋入节点必须位于模型中所有非埋入单元平均尺寸乘以0.05计算的距离范围内；但是，您可以更改此容差。

可以将几何容差定义为模型中所有非埋入单元平均尺寸的分数。或者，可以将几何容差定义为模型所选长度单位的绝对距离。如果同时指定了两个外部容差，Abaqus使用两者中更严格的容差。计算所有非埋入单元的平均尺寸，乘以分数外部容差，然后与绝对外部容差进行比较，以确定两者中更严格的容差。宿主单元中埋入单元的外部容差由[图35.4.1-1](pt08ch35s04aus136.md#embed-exteriortol)中的阴影区域表示。

**图35.4.1-1** 埋入单元的外部容差。

![](../graphics/embed-exteriortol-nls.png)

如果埋入节点位于指定容差区域内，则该节点被约束到宿主单元。该节点的位置将被调整以精确定位到宿主单元上。如果埋入节点位于指定容差区域外，将发出错误消息。

| **输入文件用法：** | 使用以下选项将容差定义为分数： |
| --- | --- |
|  | ``` [*EMBEDDED ELEMENT](../key/key-link.md#usb-kws-membeddedelement), EXTERIOR TOLERANCE=*容差* ``` 使用以下选项将容差定义为绝对距离：``` [*EMBEDDED ELEMENT](../key/key-link.md#usb-kws-membeddedelement), ABSOLUTE EXTERIOR TOLERANCE=*容差* ``` |

| **Abaqus/CAE用法：** | 相互作用模块：**创建约束**：**埋入区域**：**分数外部容差**或**绝对外部容差** |
| --- | --- |

### 调整埋入节点的位置

如果埋入节点位于宿主单元内的单元边缘或单元面附近，为了计算效率，可以对埋入节点的位置进行小幅调整，使节点精确定位在宿主单元的边缘或面上。定义了一个小容差，低于该容差时，与埋入节点关联的宿主单元节点上的权重因子将被清零。小的权重因子将按其初始权重的比例重新分配给宿主单元上的其他节点，埋入节点的位置将根据新的权重因子进行调整。此调整仅在分析开始时执行，不会对模型产生任何应变。它最适用于进行小幅调整，使埋入节点位于宿主单元的边缘或面上。如果使用较大的非默认舍入容差值对埋入节点位置进行重大调整，您应该仔细检查调整后获得的网格。

| **输入文件用法：** | ``` [*EMBEDDED ELEMENT](../key/key-link.md#usb-kws-membeddedelement), ROUNDOFF TOLERANCE=*容差* ``` |
| --- | --- |

| **Abaqus/CAE用法：** | 相互作用模块：**创建约束**：**埋入区域**：**权重因子舍入容差** |
| --- | --- |

### 与其他多点运动约束的使用

如果埋入节点也被多点约束、方程约束、运动耦合、基于表面的绑定或刚体约束绑定，则会引入过约束，并将发出错误消息。如果边界条件应用于埋入节点，则埋入单元定义始终优先。边界条件将被忽略，并将发出警告消息。

### 在埋入单元上定义面

埋入单元由于埋入而没有外部（自由）表面。因此，它们的面不包含为使用通用接触建模的相互作用自动定义的全包括表面。此外，基于这些单元的任何表面定义必须明确指定面标识符（参见["基于单元的表面定义，" 第2.3.2节](pt01ch02s03aus17.md)）。

### 限制

埋入单元技术存在以下限制：
- 具有旋转自由度的单元（具有扭转的轴对称单元除外）不能用作宿主单元。
- 埋入节点的旋转、温度、孔隙压力、声压和电势自由度不受约束。
- 宿主单元本身不能被埋入。
- 在积分点相同位置，宿主单元定义的材料不会被埋入单元定义的材料替换。
- 埋入单元带来的额外质量和刚度被添加到模型中。
- 如果使用改进的四面体单元作为宿主单元，则仅使用角节点来约束适当的埋入节点。

### 示例

考虑[图35.4.1-2](pt08ch35s04aus136.md#embedded-elements)中的示例。

**图35.4.1-2** 单元埋入在宿主单元中。

![](../graphics/embedded-elements-nls.png)

单元3（桁架）和单元4（膜）埋入在单元1和单元2中。单元1由节点a、b、c、d、e、f、g和h形成；单元2由节点e、f、g、h、i、j、k和l形成；单元3由节点A和B形成；单元4由节点C、D、E和F形成。如果宿主单元集包含单元1和单元2，埋入单元集分别包含单元3和单元4，Abaqus将尝试查找是否有任何埋入节点（A、B、C、D、E和F）位于宿主单元1或2内。如果发现节点A接近单元1的a-b-f-e面，则节点A的所有自由度被约束到节点a、b、f和e，适当的权重因子基于节点A在单元1中的几何位置确定。同样，如果发现节点B位于单元1内部，且节点E接近单元2的g-k边缘，则节点B的所有自由度被约束到节点a、b、c、d、e、f、g和h，节点E的所有自由度被约束到节点g和k，适当的权重因子分别基于节点B在单元1中的几何位置和节点E在单元2的g-k边缘上的几何位置确定。

您应确保所需埋入单元上的所有节点都正确约束到宿主单元上的节点。可以通过执行数据检查分析来验证这一点（参见["Abaqus/Standard、Abaqus/Explicit和Abaqus/CFD执行，" 第3.2.2节](pt01ch03s02abx02.md)）。对于每个埋入节点，在数据检查分析期间，会将用于约束该节点的节点列表及相关权重因子输出到数据文件。如果埋入节点未受约束且使用完全埋入，则会发出错误消息。
