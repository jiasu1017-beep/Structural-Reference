# 5.5 Viewport 对象





以下命令对 Viewport 对象进行操作。有关 Viewport 对象的更多信息，请参见["Viewport 对象"，第 11.4 节](pt01ch11pyo04.md)。

**访问**

```
import annotationToolset
```

### 5.5.1 plotAnnotation(...)

此方法在视口中绘制一个 [Annotation](pt01ch05pyo01.md) 对象。

**必需参数**

*annotation*

一个 [Annotation](pt01ch05pyo01.md) 对象要绘制。

**可选参数**

*index*

一个 Int，指定 [Annotation](pt01ch05pyo01.md) 对象在要绘制的注释序列中的索引。默认值为零。

**返回值**

无

**异常**

无。
