# 9.7.7 Closing the current output database







Select ****File**![](../graphics/images/arrow.gif)**Close ODB**** from the main menu bar to close an output database. Closing an output database releases computer resources, such as memory.

**To close an output database:**

1. From the main menu bar, select ****File**![](../graphics/images/arrow.gif)**Close ODB****. The **Close Output Database** dialog box appears with a list of all the output databases that are open, the date they were last updated, and the viewports that reference each open output database.
2. Select the output database to close, and click **OK** to close the dialog box. Abaqus/CAE closes the selected output database and clears any viewports that were displaying data from that output database.
