# 4.5.9 Test 21: Simply supported thick square plate: frequency extraction







**Product: **Abaqus/Standard  

### Elements tested

S4    S4R    S4R5    S8R    

### Problem description

![](../graphics/bmkfv21.png)

**Model: **

Plate thickness = 0.05 m.

**Material: **

Young's modulus = 200 GPa, Poisson's ratio = 0.3, density = 8000 kg/m3.

**Boundary conditions: **

![](../graphics/bmk_eqn01715.gif) at all nodes. ![](../graphics/bmk_eqn01281.gif) along all four edges. ![](../graphics/bmk_eqn01733.gif) along edges ![](../graphics/bmk_eqn00104.gif) and ![](../graphics/bmk_eqn01734.gif), ![](../graphics/bmk_eqn01735.gif) along edges ![](../graphics/bmk_eqn00105.gif) and ![](../graphics/bmk_eqn01736.gif).

Frequency extraction is performed in Step 1.

### Reference solution

This is a test recommended by the National Agency for Finite Element Methods and Standards (U.K.): Test 21 from NAFEMS “Selected Benchmarks for Forced Vibration,” R0016, March 1993.
