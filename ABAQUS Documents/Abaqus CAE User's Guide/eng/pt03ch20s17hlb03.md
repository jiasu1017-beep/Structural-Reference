# 20.22.3 Saving the current sketch as a stand-alone sketch







While you are sketching, you can save the current sketch as a stand-alone sketch. Stand-alone sketches are maintained independent of any features; they can be subsequently retrieved into the Sketcher and will overlay any existing geometry.

Abaqus/CAE saves only items within the sketch. [Reference geometry](pt03ch20s05s01.md) is not saved. Similarly, dimensions between vertices and lines from reference geometry are not saved.

**To save the current sketch as a stand-alone sketch:**

1. From the Sketcher toolbox, select the save as tool ![](../graphics/ico_sketchSaveAs.png). For a diagram of the tools in the Sketcher toolbox, see ["The Sketcher tools," Section 20.4.1](pt03ch20s04s01.md).
2. In the text field that appears in the prompt area, enter the name of the sketch. Abaqus/CAE saves the sketch and returns to the Sketcher.
