# 1.3 Abaqus 文档

Abaqus 的文档内容丰富且完整。以下文档和出版物可通过 SIMULIA 获得，包括 Abaqus HTML 文档和 PDF 格式。如需了解访问 HTML 指南的更多信息，请参阅《Abaqus 分析用户指南》中关于执行程序的讨论。如需了解打印指南的更多信息，请参阅《Abaqus 在线文档使用指南》第 5.3 节"从 PDF 图书打印"。

**Abaqus 分析用户指南**

本指南包含对单元、材料模型、过程、输入规范等的完整描述。这是 Abaqus/Standard、Abaqus/Explicit 和 Abaqus/CFD 的基本指南；它同时提供输入文件使用和 Abaqus/CAE 使用信息。本指南经常引用《Abaqus 分析用户指南》，因此在阅读示例时应该将其作为参考。

**Abaqus/CAE 用户指南**

本指南包括如何使用 Abaqus/CAE 进行模型生成、分析以及结果评估和可视化等详细描述。Abaqus/Viewer 用户应参阅本指南中关于可视化模块的信息。

**使用 Abaqus 在线文档**

本指南包含浏览、查看和搜索 Abaqus HTML 和 PDF 文档的说明。此外，本指南还解释了如何使用 PDF 文档制作高质量的打印副本，以及如何使用除《Abaqus 脚本参考指南》和《Abaqus GUI 工具包参考指南》之外的所有 PDF 图书中的 ![](../graphics/hhp-pdf-print-exa-button.gif) 图标打印图书的选定部分。

**其他 Abaqus 文档：**

**Abaqus 示例问题指南**

本指南包含详细的示例，旨在说明执行有意义的线性和非线性分析所需的方法和决策。许多示例使用几种不同的单元类型、网格密度和其他变体进行了计算。典型案例包括：弹性塑性管撞击刚性墙的大运动；薄壁弯头的非弹性屈曲坍塌；弹性粘塑性薄环的爆炸载荷； footing 下的固结；带孔复合壳的屈曲；以及金属薄板的深拉。当开始处理一类新问题时，查阅本指南中相关的示例并回顾它们通常很有帮助。

当您想使用以前未使用过的功能时，应该查找使用该功能的一个或多个示例。然后，使用该示例来熟悉该功能的正确用法。要查找使用某个功能的示例，请搜索在线文档或使用 **abaqus findkeyword** 实用程序（有关更多信息，请参阅《Abaqus 分析用户指南》第 3.2.15 节"查询关键词/问题数据库"）。

与示例相关的所有输入文件都作为 Abaqus 安装的一部分提供。使用 **abaqus fetch** 实用程序可以从发布提供的压缩档案文件中提取样本 Abaqus 输入文件（有关更多信息，请参阅《Abaqus 分析用户指南》第 3.2.16 节"获取样本输入文件"）。您可以获取任何示例文件，以便自己运行模拟并查看结果。您还可以通过《Abaqus 示例问题指南》中的超链接访问输入文件。

**Abaqus 基准测试指南**

本指南包含用于评估 Abaqus 性能的基准问题和分析；这些测试是对简单几何形状或实际问题的简化版本进行的多单元测试。本指南中包含了 NAFEMS 基准问题。

**Abaqus 验证指南**

本指南包含基本测试用例，提供对每个单独程序功能（过程、输出选项、MPC 等）相对于精确计算和其他已发布结果的验证。在学习使用新功能时，运行这些问题可能很有用。此外，提供的输入数据文件为检查单元、材料等的行为提供了良好的起点。

**Abaqus 理论指南**

本指南包含对 Abaqus 所有理论方面的详细而精确的讨论。它的编写目的是让具有工程背景的用户能够理解。

**Abaqus 关键词参考指南**

本指南包含对 Abaqus/Standard、Abaqus/Explicit 和 Abaqus/CFD 中所有可用输入选项的完整描述。

**Abaqus 用户子程序参考指南**

本指南包含对可用于 Abaqus 分析的所有用户子程序的完整描述。它还讨论了在编写用户子程序时可以使用的实用程序例程。

**Abaqus 术语表**

**Abaqus 发布说明**
