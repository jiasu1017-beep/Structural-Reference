# 17.8.6 Abaqus/CAE 什么时候可以应用映射网格划分？






Abaqus/CAE 在满足以下条件时，认为四边区域适合应用映射网格划分：
- 可能生成具有合理质量的规则网格模式，且
- 对网格播种的任何微小调整都不会违反用户的意图（如任何现有播种约束所指示的）。

当 Abaqus/CAE 应用映射网格划分时，它会对网格播种进行微小调整，以确保矩形区域的相对边具有相同数量的播种点。如果您的模型很大且包含许多简单区域，Abaqus/CAE 检查矩形区域并调整播种以生成映射网格的时间可能略长于生成不带映射网格的网格的时间。但是，对于大多数模型，包含映射网格的时间差异与网格质量的提升相比并不显著。

[图 17-64](pt03ch17s08s06.md#mgn-mapped-threemeshes) 展示了一个使用 Abaqus/CAE 中以下网格技术和算法进行三角形网格划分的壳零件：
- 自由网格划分
- 在适当的地方使用映射网格划分的自由网格划分
- 结构网格划分

**图 17-64** 以三种不同方式进行三角形网格划分的壳零件。

![](../graphics/mgn-mapped-threemeshes-nls.png)

在许多情况下，使用映射网格划分的三角形自由网格与三角形的结构网格相同。在[图 17-64](pt03ch17s08s06.md#mgn-mapped-threemeshes) 中网格不同，是因为 Abaqus/CAE 尊重原始播种，并确定在零件的边上不适合应用映射网格划分。相比之下，当 Abaqus/CAE 创建结构网格时，它会显著调整播种以创建用户请求的结构网格。
