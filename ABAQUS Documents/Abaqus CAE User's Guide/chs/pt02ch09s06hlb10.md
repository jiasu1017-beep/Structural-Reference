# 9.7.10 在没有许可证的情况下保存当前模型数据库





如果您的系统在会话期间失去与许可证服务器的连接或以其他方式丢失许可证，您可以以其当前状态保存模型数据库。Abaqus/CAE会显示一个消息对话框，其中包含以下选项：
- 尝试重新获取许可证或检查服务器是否可用
- 保存模型数据库
- 在不保存模型数据库的情况下退出Abaqus

默认情况下，Abaqus/CAE会尝试重新获取许可证或重新连接到许可证服务器。要保存模型数据库，请选择第二个选项并提供文件名；如果您在当前会话期间已经保存了模型，Abaqus/CAE会将上一个文件名和路径显示为保存当前模型数据库的默认值。当您单击**确定（OK）**时，Abaqus/CAE会保存模型。

**注意：**如果您正在执行某个过程（如创建草图或编辑材料），则只能保存该过程开始之前完成的模型部分。

如果您选择在不保存的情况下退出，Abaqus/CAE会在下次启动会话时尝试恢复模型信息。一旦您保存了模型，保存选项会从对话框中删除——您可以继续尝试获取许可证或退出会话。

您应该定期保存模型数据库。除非您执行明确的保存操作，否则Abaqus/CAE永远不会保存模型数据库；例如，没有基于计时器的自动保存。如果您尝试保存未修改的模型数据库，则不会采取任何操作。
