### 概述

表面单元：
- 像膜单元一样定义——作为空间中的表面；
- 没有固有刚度；
- 可能具有单位面积质量；
- 可用于定义刚体；
- 可用于定义表面和基于表面的绑定约束；
- 像零厚度膜单元一样行为；
- 可与钢筋层一起使用；
- 可以嵌入实体单元中；
- 只能传递面内力；并且
- 没有弯曲刚度或横向剪切刚度。

### 典型应用

表面单元在几种特殊建模情况下很有用：
- 它们用于承载钢筋层，以表示实体结构中的薄加强组件。钢筋层的刚度和质量被添加到表面单元（请参阅["定义钢筋，" 第2.2.3节](pt01ch02s02aus13.md)）。加强的表面单元也可以嵌入"宿主"实体单元中（请参阅["嵌入单元，" 第35.4.1节](pt08ch35s04aus136.md)）。
- 它们用于将额外的质量带入模型，形式为单位面积质量；例如，将燃料罐中燃料的质量分散到罐表面上，特别是当罐用实体单元建模时。
- 它们用于指定在约束中使用的表面，当该表面不具有结构特性时。
- 当与基于表面的绑定约束结合使用时，它们用于在梁单元上指定分布表面载荷，如入射波载荷。
- 在Abaqus/Explicit中（当与基于表面的绑定约束结合使用时），它们可用于在梁单元上指定复杂表面以用于通用接触。用于强制执行接触约束的惩罚弹簧刚度与表面节点的质量大致成正比。如果表面节点没有质量，则不会强制执行接触。
- 在Abaqus/Explicit中，它们可用于定义基于表面的流体腔的全部或部分边界（例如，请参阅["流体静力学流体单元：建模空气弹簧，" Abaqus示例问题指南第1.1.9节](../exa/exa-link.md#exa-sta-hydrofluidairspring)）。
- 在Abaqus/Aqua分析中，它们可用于可视化重力波。

### 选择合适的单元

除了在Abaqus/Standard和Abaqus/Explicit中都可用的通用表面单元外，圆柱表面单元和轴对称表面单元仅在Abaqus/Standard中可用。

#### 通用表面单元

通用表面单元应用于结构变形可以沿三个维度发展的三维模型。

#### 圆柱表面单元

圆柱表面单元在Abaqus/Standard中可用于精确建模具有圆形几何形状的结构区域，例如轮胎。这些单元利用三角函数沿圆周方向插值位移，并在平面内方向使用规则等参插值。它们沿圆周方向使用三个节点，可以跨越0到180度的段。平面内方向有一阶和二阶插值的单元可用。

单元的几何形状通过在全局笛卡尔系统中指定节点坐标来定义。

这些单元可以与规则表面单元在同一网格中使用。它们也可以嵌入通用实体单元和圆柱单元中。

#### 轴对称表面单元

Abaqus/Standard中可用的轴对称表面单元分为两类：不允许绕对称轴扭转的单元和允许扭转的单元。这些单元分别称为常规和广义轴对称表面单元。

广义轴对称表面单元（带扭转的轴对称表面单元）允许载荷的周向分量，这可能导致绕对称轴的扭转。周向载荷分量独立于周向坐标![](../graphics/usb_eqn00123.gif)。由于载荷不依赖于周向坐标，变形是轴对称的。

广义轴对称表面单元不能用于动力学或特征频率提取过程。

### 单元法线定义

表面单元的"顶"表面是沿正法线方向（定义如下）的表面，称为用于接触定义的SPOS面。"底"表面沿法线负方向，称为用于接触定义的SNEG面。

对于通用表面单元，正法线方向由沿单元节点按元素定义中指定的顺序排列的右手定则定义。请参见[图32.7.1-1](pt06ch32s07alm52.md#esurface-gen-normal)。

### 定义单元的截面属性

你必须将表面截面属性与模型的某个区域关联。

| **输入文件用法：** | ``` [*SURFACE SECTION](../key/key-link.md#usb-kws-msurfacesection), ELSET=*name* ``` |
| --- | --- |
|  | 其中ELSET参数引用一组表面单元。 |

| **Abaqus/CAE用法：** | 属性模块：**创建截面**：选择**壳**作为截面**类别**和**表面**作为截面**类型** ****分配**![](../graphics/images/arrow.gif)**截面****：选择区域 |
| --- | --- |

#### 使用表面单元承载钢筋层

你可以定义由表面单元承载的加强层。钢筋层导致的刚度和质量被添加到表面单元。

| **输入文件用法：** | 同时使用以下两个选项： |
| --- | --- |
|  | ``` [*SURFACE SECTION](../key/key-link.md#usb-kws-msurfacesection), ELSET=*name* [*REBAR LAYER](../key/key-link.md#usb-kws-mrebarlayer) ``` |

| **Abaqus/CAE用法：** | 属性模块：**创建截面**：选择**壳**作为截面**类别**和**表面**作为截面**类型**，**钢筋层** |
| --- | --- |

你可以定义由表面单元承载的单位面积质量。

| --- | --- |

| **Abaqus/CAE用法：** | 属性模块：**创建截面**：选择**壳**作为截面**类别**和**表面**作为截面**类型**，切换打开**密度**：*number* |
| --- | --- |

#### 在约束中使用表面单元

表面单元可用于在Abaqus中定义表面，此表面可用于基于表面的绑定约束（请参阅["网格绑定约束，" 第35.3.1节](pt08ch35s03aus132.md)）。

| **输入文件用法：** | 使用以下选项： |
| --- | --- |
|  | ``` [*SURFACE](../key/key-link.md#usb-kws-msurface), NAME=*surface_name* [*TIE](../key/key-link.md#usb-kws-mtie), NAME=*name* *surface_name*, *master_name* ``` |

| **Abaqus/CAE用法：** | 在Abaqus/CAE中，当提示选择表面时，你可以在视口中直接选择一个或多个面。此外，你可以使用表面工具集将面和边定义为表面的集合。 |
| --- | --- |
|  | 相互作用模块：**创建约束**：**绑定** |

| **Abaqus/CAE用法：** | 在Abaqus/CAE中不支持指定可视化波面。 |
