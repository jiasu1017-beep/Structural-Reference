# 64.7.12 删除网格-几何关联性







您可以通过从主菜单栏选择 ****Mesh**![](../graphics/images/arrow.gif)**Delete Mesh Associativity****（网格→删除网格关联性）并选择顶点、边或面来删除节点、单元边或单元与底层几何之间的关联性。如果您删除 Abaqus/CAE 几何与网格之间的关联性，受影响的网格组件将变为孤网格节点和单元。如果您正在处理复杂部件并希望保持某些部分的网格不变而更改其他区域的几何，这将非常有用。

**删除网格-几何关联性：**

1. 进入 Mesh（网格）模块。从上下文栏的 **Object**（对象）字段中选择 **Part**（部件），然后从列表中选择一个部件。
2. 从主菜单栏中选择 ****Mesh**![](../graphics/images/arrow.gif)**Delete Mesh Associativity****（网格→删除网格关联性）。Abaqus/CAE 在提示区域显示提示以指导您完成过程。**提示：**您也可以点击 Mesh 模块工具箱中的 ![](../graphics/icoR_meshDeleteAssociativity.png) 工具，或者点击 ![](../graphics/ico_meshEdit.png) 工具显示 **Edit Mesh**（编辑网格）对话框——删除关联性方法位于对话框内的 **Mesh**（网格）类别中。（有关 Mesh 模块工具箱的更多信息，请参阅["使用 Mesh 模块工具箱，" 第 17.15 节"](pt03ch17s15.md)。）
3. 选择一个或多个要删除关联性的几何实体——顶点、边、面或整个区域。您可以使用拖动选择、**[Shift]****+点击** 和 **[Ctrl]****+点击** 的组合来选择几何实体。有关更多信息，请参阅["在当前视口中选择对象，" 第 6.2 节"](pt01ch06s02.md)。**注意：**要删除原生网格区域的关联性并创建孤网格，您必须选择整个区域。
4. 如果需要，您可以在提示区域中切换关闭 **Delete association from bounding entities**（从边界实体删除关联性）。默认情况下，当您删除区域的关联性时，Abaqus/CAE 也会删除节点、单元边和面与相应顶点、边和面之间的关联性。
5. 完成选择几何实体后，点击 **Done**（完成）。Abaqus/CAE 将几何实体高亮显示为红色，将关联的网格实体高亮显示为黄色，以便您确认选择。
6. 点击 **Yes**（是）删除网格关联性，或点击 **No**（否）取消选择。
7. 重复第 3 步到第 6 步以删除其他区域的关联性，或点击提示区域中的取消按钮 ![](../graphics/afxI_cancel.png) 结束过程。
