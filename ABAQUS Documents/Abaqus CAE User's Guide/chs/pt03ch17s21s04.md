# 17.21.4 选择重新网格划分规则约束





您可以定义 Abaqus/CAE 在自适应重新网格划分过程中应用的单元尺寸约束。您还可以指定控制较大和较小单元引入的速率限制，并调节 Abaqus/CAE 应用尺寸方法的激进程度。

您指定的最小和最大单元尺寸限制网格尺寸函数中使用的尺寸，Abaqus/CAE 使用该函数引导内部网格划分算法。您指定的尺寸不是单元尺寸的绝对约束——生成网格中的单元尺寸仅近似于尺寸函数。因此，某些单元边缘可能大于或小于您指定的最大和最小单元尺寸。

您指定的最大单元数限制网格尺寸函数生成的单元数。网格尺寸函数调整目标误差和单元尺寸，使重新网格划分区域中生成的单元数不超过指定值。生成网格中的单元数（与单元尺寸一样）仅近似于尺寸函数。因此，生成的单元数可能大于您指定的最大单元数。

**选择单元尺寸约束的步骤：**

1. 在"创建重新网格划分规则"对话框中，单击**约束**选项卡。
2. 指定最小和最大**单元尺寸**。- 指定**最小**值以为重新网格算法计算的单元尺寸设置下限。您可以选择 Abaqus/CAE 计算的最小单元尺寸，也可以输入最小单元尺寸。- 指定**最大**值以为重新网格算法计算的单元尺寸设置上限。您可以选择 Abaqus/CAE 计算的最大单元尺寸，也可以输入最大单元尺寸。
3. 如果需要，指定**近似最大单元数**以为重新网格算法计算的单元总数设置上限。
4. 指定**速率限制**。- **细化**速率限制调节尺寸方法的攻击性，并控制较小单元的引入。选择以下之一：- **使用默认值**指定介于**高**和**低**之间的细化限制。- **指定**并拖动**细化**滑块以指定速率限制。指定**低**表示单元尺寸的最小减少；指定**高**表示原始单元尺寸的最大减少。细化因子对自适应网格程序的收敛有显著影响，可以帮助您实现更快、更有效的网格收敛。- **不细化**以防止单元尺寸减小。- **粗化**速率限制调节尺寸方法的攻击性，并控制较大单元的引入。选择以下之一：- **使用默认值**使用默认限制。- **指定**并拖动**粗化**滑块以指定速率限制。指定**低**表示单元尺寸的最小增长；指定**高**表示单元尺寸的最大增长。- **不粗化**以防止单元尺寸增长。
