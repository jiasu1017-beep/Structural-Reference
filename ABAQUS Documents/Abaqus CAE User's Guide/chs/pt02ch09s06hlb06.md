# 9.7.6 管理网络ODB连接器





从主菜单栏选择****文件（File）**![](../graphics/images/arrow.gif)**网络ODB连接器（Network ODB Connector）**![](../graphics/images/arrow.gif)**管理器（Manager）****来管理您的网络连接器。该管理器还监控您的网络ODB连接器的状态。

您可以使用**网络ODB连接器管理器（Network ODB Connector Manager）**创建网络ODB连接器。有关更多信息，请参阅["创建网络ODB连接器，"第9.7.4节"](pt02ch09s06hlb04.md)。您还可以执行以下操作：
- 编辑网络ODB连接器。
- 将网络ODB连接器复制到具有不同名称的另一连接器。
- 重命名网络ODB连接器。
- 删除网络ODB连接器。
- 启动网络ODB连接器。创建网络ODB连接器后，必须启动它才能使其处于活动状态。**注意：**您也可以通过从**打开数据库（Open Database）**对话框中的**目录（Directory）**字段选择**网络连接器（Network connectors）**来启动连接器。Abaqus/CAE会显示网络连接器列表，您可以双击连接器来启动它。您可以通过从主菜单栏选择****文件（File）**![](../graphics/images/arrow.gif)**打开（Open）****来显示**打开数据库**对话框。
- 停止您之前启动的网络ODB连接器。您必须先停止连接器才能编辑、重命名或删除它。
