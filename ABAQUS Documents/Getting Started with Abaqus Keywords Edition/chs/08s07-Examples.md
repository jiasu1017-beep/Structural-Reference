# 8.7 小结

- 结构问题中的非线性有三个来源：材料非线性、几何非线性和边界（接触）非线性。Abaqus分析中可能同时存在任意组合的这几种非线性。
- 当位移的大小影响结构响应时，就会产生几何非线性。它包括大位移和大旋转效应、跳跃失稳以及载荷刚化效应。
- 在Abaqus/Standard中，非线性问题采用牛顿-拉普森方法迭代求解。非线性问题所需的计算机资源通常是线性问题的许多倍。
- Abaqus/Explicit不需要迭代即可获得解；但是，由于几何形状的大变化导致稳定时间增量减小，计算成本可能会受到影响。
- 非线性分析步骤被划分为若干个增量。
  - Abaqus/Standard通过迭代来找到每个新载荷增量结束时的近似静态平衡。Abaqus/Standard在整个模拟过程中使用收敛控制来控制载荷增量。
  - Abaqus/Explicit通过从一个增量到下一个增量推进运动状态来确定解，使用比隐式分析中通常使用的时间增量更小的时间增量。增量的大小受稳定时间增量限制。默认情况下，Abaqus/Explicit中的时间增量完全自动化。
- 状态文件允许在分析运行时监控其进度。消息文件包含载荷增量和迭代的详细信息。
- 可以在每个增量结束时保存结果，以便在Abaqus/Viewer中可视化结构响应的演化过程。也可以用*X–Y*曲线图的形式绘制结果。
