# 6.2.2 拖动选择多个对象





大多数提示要求你从当前视口中仅选择一个对象。但是，某些任务允许你选择一个或多个对象；例如，集合工具集允许你选择多个相同类型的对象并将它们分组到集合中。你可以使用 ["选择和取消选择单个对象，" 第 6.2.1 节](pt01ch06s02hlb01.md) 中描述的 **[Shift]****+点击**方法选择多个对象。另一种选择多个对象的方法是围绕这些对象拖动一个矩形。你可以使用选择选项工具调整拖动选择区域的形状。你也可以选择拖动选择区域选择哪些对象。选择选项工具位于**选择**工具栏中。更多信息，请参阅["修改拖动选择区域的形状，" 第 6.3.5 节](pt01ch06s03s04.md)和["选择拖动选择区域选择哪些对象，" 第 6.3.6 节](pt01ch06s03s05.md)。

**要拖动选择多个对象：**

1. 设想一个只包含你要选择对象的矩形。
2. 在矩形的一个角落点击，然后继续按住鼠标按钮，拖动直到包含所有对象。
3. 释放鼠标按钮。矩形内部或交叉的所有有效对象都会高亮显示。
4. 单击鼠标按钮 2 表示你已完成选择对象。

有时组合使用 **[Shift]****+点击**和拖动选择技术会很方便。更多信息，请参阅["组合选择技术，" 第 6.2.8 节](pt01ch06s02hlb08.md)。

**提示：**如果你选择了多个对象，然后想要取消选择其中一些，请 **[Ctrl]****+点击**你要取消选择的对象。要取消选择所有对象，请在视口中未使用的区域点击。
