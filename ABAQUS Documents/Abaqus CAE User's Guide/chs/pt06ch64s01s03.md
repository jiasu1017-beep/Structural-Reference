# 64.1.3 操作网格





网格编辑工具集提供以下工具，允许您操作网格：
- 从现有网格的所选单元面法向偏移创建实体单元层，如[图64-8](pt06ch64s01s03.md#edt-bracket-solidoffset)所示。此工具对于用实体单元网格化壳状零件非常有用。您可以指定所有层的总厚度和要创建的层数。第一层单元可以与现有单元合并，也可以按指定距离偏移。所选单元面可以来自壳单元或实体单元，但不能同时来自两者。如果您选择了壳单元面，则可以在Abaqus/CAE生成实体单元层后选择是否删除壳单元。 **图64-8** 创建实体单元层。 ![](../graphics/edt-bracket-solidoffset-nls.png) 您应该使用此工具生成连续壳和粘聚单元，因为Abaqus/CAE生成的实体单元沿其偏移的表面法向堆叠。
- 创建从现有网格的所选单元面法向偏移的壳单元层，如[图64-9](pt06ch64s01s03.md#edt-shell-offset)所示。 **图64-9** 创建壳单元层。 ![](../graphics/edt-shell-offset-nls.png) 此工具与前一个工具类似；但是，您不是指定层的总厚度，而是指定层之间的距离。您应该使用此工具创建现有壳网格的偏移副本，并创建多层复合模型或增强件。
- 包裹网格。您可以在指定半径处将平面孤兒网格绕全局Z轴包裹。包裹过程会将平面网格上位于点(![](../graphics/usi_eqn00797.gif), ![](../graphics/usi_eqn00798.gif))的节点重新定位到(![](../graphics/usi_eqn00799.gif))，其中![](../graphics/usi_eqn00270.gif)是指定半径，![](../graphics/usi_eqn00065.gif) = ![](../graphics/usi_eqn00800.gif)，且![](../graphics/usi_eqn00801.gif)=![](../graphics/usi_eqn00798.gif)。
- 折叠边缘。您可以指定网格中所需的最小单元边缘尺寸。Abaqus/CAE会合并短于指定长度的单元边缘的端节点。默认情况下，Abaqus/CAE会检查网格零件中的所有单元边缘，但您可以指定不同的单元域。对于结构化网格，您可以指定厚度方向或参考边缘，以限制考虑折叠的边缘方向。如果您选择厚度方向，Abaqus/CAE使用方向向量而不是仅测量节点之间的距离来测量单元。如果单元的方向使得方向在拓扑上不明确，Abaqus/CAE会高亮显示这些单元，以便您选择排除它们或添加参考边缘以进一步定义所需的边缘。当您选择带或不带厚度方向的参考边缘时，Abaqus/CAE仅在拓扑上与参考边缘平行的边缘才考虑折叠。参考边缘只能应用于单个结构化网格部分；如果需要参考边缘，则必须在单独的回缩操作中处理多个独立部分。您不能在包含二次单元的单元域中折叠小边缘。
- 生长边缘。您可以指定希望出现在模型中的最小单元边缘尺寸，Abaqus/CAE会通过向外调整节点来将较小的边缘生长到指定的最小长度，从而有效地从相邻边缘获取片段。如果调整节点会导致相邻单元边缘太小，Abaqus/CAE不能增加边缘的长度。但是，如果添加厚度方向，Abaqus/CAE会在该方向上生长所有单元，以满足最小尺寸。此工具的选项与折叠边缘工具相同；您可以指定除整个网格零件之外的不同单元域，并且可以在结构化网格中使用厚度方向或参考边缘，仅生长符合您方向标准的边缘。您不能在包含二次单元的单元域中生长边缘。 **注意：**指定除整个网格之外的不同单元域时要小心。Abaqus/CAE仅考虑所选域内的单元边缘。如果短单元边缘存在于域边界附近，Abaqus/CAE在生长域内边缘之前不会检查域外相邻边缘的长度。域边界处边缘的生长可能导致域外出现短边缘或倒置单元。
- 将三角形单元转换为四面体单元。您可以将完全由三角形单元组成的三维壳封闭面转换为四面体实体网格。三角形壳单元必须完全包围可以用实体单元填充的体积。
- 将实体孤兒网格单元转换为壳孤兒网格单元。此工具在零件的自由面上创建三角形或四边形壳单元。对于具有创建裂纹特征的未合并节点或具有不兼容网格的连接面的网格零件，Abaqus/CAE会在界面两侧创建壳单元。
- 合并层。您可以选择一组相邻单元，并沿指定方向将它们连接起来。每个合并后的单元将具有父单元的组合形状和相同的单元类型。选择要合并的单元后，您选择单元内的一条边缘以指示合并方向，如[图64-10](pt06ch64s01s03.md#edt-merge-layer)左侧所示。元素顶部的两层使用**按拓扑**选择方法选择（有关更多信息，请参见["使用拓扑方法选择多个单元，" 第6.2.5节](pt01ch06s02hlb05.md)）；为清晰起见，图中未显示选择。您不需要选择整个单元层进行合并。但是，如果您没有选择整个层，Abaqus/CAE会警告您新单元将与周围网格不兼容。然后您可以选择继续或取消合并过程。您不能合并二次单元。
- 细分层。您可以选择一组单元，并通过等分单元边缘来分割它们。Abaqus/CAE会提示您选择一条边缘、一个面或**所有方向**，以指示将创建新层的方向，并输入要从每个单元边缘进行的分割数。您不能细分二次单元。
- 复制网格图案。您可以将二维网格图案应用到同一零件或装配上类似的几何目标。选择源网格和目标几何体后，您选择几个节点并将它们映射到几何体上的点，然后Abaqus/CAE完成映射过程以将网格图案应用到目标。
- 将网格与几何体关联。您可以将孤兒或自底向上网格实体与选定的几何实体相关联。Abaqus/CAE会提示您选择顶点、边缘或面，然后选择要与几何体关联的节点、单元边缘或单元面。关联可用于将载荷应用到网格，或在孤兒网格单元和几何体之间的界面上创建兼容网格。
- 删除网格关联。您可以删除顶点、边缘、面或整个几何区域的网格-几何体关联。您还可以删除原生网格的关联，以创建部分模型的孤兒网格。

**提示：**网格编辑撤销功能可以回滚您对网格所做的任何更改。有关更多信息，请参见["撤销或重做网格编辑工具集中的更改，" 第64.9节](pt06ch64s09.md)。

此外，[第17.18.11节"更改所有节点和单元的标签"](pt03ch17s18s09.md)描述瞭如何对零件或装配中所选零件实例的所有节点和/或单元重新编号。
