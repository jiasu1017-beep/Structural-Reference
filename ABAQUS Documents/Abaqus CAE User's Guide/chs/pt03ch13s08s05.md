# 14.2 进入和退出Step模块





您可以在任何时候通过点击上下文栏中**模块**列表中的**Step**来进入Step模块。**Step**、**输出**、**其他**和**工具**菜单会出现在主菜单栏上。如果当前视口包含的不是装配，则在启动Step模块时视口的内容会消失。

要退出Step模块，请从**模块**列表中选择任何其他模块。您无需在退出模块前保存您的步骤或输出请求；当您通过从主菜单栏选择**文件**![](../graphics/images/arrow.gif)**保存**或**文件**![](../graphics/images/arrow.gif)**另存为**来保存模型数据库时，它们会自动保存。
