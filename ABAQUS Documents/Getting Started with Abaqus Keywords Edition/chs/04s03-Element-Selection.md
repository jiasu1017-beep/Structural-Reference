# 4.3 示例：连接销

在本例中，您将使用三维连续体单元对如图4-14所示的连接销进行建模。

![图 4–14]((../graphics/gss-connecting-lug-nls.png))

连接销的一端牢固地焊接在大型结构上。另一端带有孔洞。在使用时，螺栓将穿过销的孔洞。您需要确定当30 kN的载荷沿2方向负向施加到螺栓上时销的静态挠度。因为此分析的目的是检验销的静态响应，所以应使用Abaqus/Standard作为分析产品。您决定通过以下假设简化此问题：

* 与其在模型中包含复杂的螺栓-销相互作用，您将使用孔洞下半部分的分布压力来加载连接销（参见图4-14）。
* 您将忽略孔洞周围压力大小的变化，并使用均匀压力。
* 所施加均匀压力的大小将为50 MPa：30 kN / (2 × 0.015 m × 0.02 m)。

在检验销的静态响应之后，您将修改模型并使用Abaqus/Explicit来研究销突然加载时产生的瞬态动力学效应。

## 4.3.1 坐标系

在模型中，定义全局1轴沿销的长度方向，全局2轴为垂直方向，全局3轴沿厚度方向。将全局坐标系的原点（![img](../graphics/gsk_eqn00038.gif)）置于![img](../graphics/gsk_eqn00039.gif)面孔洞中心处（参见图4-14）。

## 4.3.2 网格设计

在开始为特定问题构建网格之前，您需要考虑将要使用的单元类型。适合使用二次单元的网格设计在改用线性、减缩积分单元时可能并不适合。对于本例，使用20节点六面体减缩积分单元（C3D20R）。选择单元类型后，您可以为连接销设计网格。关于此应用网格设计最重要的决策是在销孔圆周上使用多少个单元。图4-15显示了该连接销的可能网格；您应构建与此相似的模型。

![图 4–15]((../graphics/gss-c3d20r.png))

在设计网格时还需要考虑的另一件事是您希望从仿真中获得何种结果。图4-15中的网格相当粗糙，因此不太可能产生精确的应力。对于此类问题，每90°使用四个二次单元是应考虑的最小数量；建议使用两倍的数量以获得合理的应力结果。但是，此网格应足以预测销在施加载荷下的总体变形程度，这正是您被要求确定的内容。本模拟中增加网格密度的影响将在4.4节"网格收敛"中讨论。

您需要决定在模型中使用何种单位制。建议使用米、秒和千克的SI制，但如果您愿意也可以使用其他单位制。

本示例使用图4-16所示的网格、节点集和单元集，以及图4-14所示的压力载荷和边界条件。

![图 4–16]((../graphics/gss-elementsets-nls.png))

后续步骤将添加模型所需的额外数据，以描述Abaqus输入文件的格式。如果您希望调整网格且没有预处理器，请使用A.2节"连接销"中的Abaqus网格生成选项。如果您希望使用Abaqus/CAE创建整个模型，请参阅《Abaqus入门：交互版》第4.3节"示例：连接销"。

## 4.3.4 审核输入文件——模型数据

模型数据（包括节点和单元定义、集定义、截面和材料属性）将在以下各节中讨论。

**模型描述**

Abaqus输入文件始终以`*HEADING`选项开头。通常，预处理器在此选项中提供的描述不太具有信息性，尽管它可能给出文件生成时的日期和时间。您应在此选项的数据行上提供合适的标题，以便查看此文件的人能够了解正在建模的内容以及所使用的单位。`lug.inp`中使用的`*HEADING`选项块如下所示：

```
*HEADING
44  Linear Elastic Steel Connecting Lug
45  S.I. Units (N, kg, m, s)
```

**节点坐标和单元连接**

在预处理器创建的输入文件中，模型的节点坐标通常在一个大的`*NODE`选项块中，为每个节点单独指定坐标。

预处理器生成的单元定义通常包含在多个`*ELEMENT`选项块中。通常，每个块包含具有相同单元截面和材料属性的单元。在连接销模型中只使用了一种单元类型，所有单元具有相同的属性。因此，输入文件中可能只有一个`*ELEMENT`选项块。它将类似于：

```
*ELEMENT, TYPE=C3D20R, ELSET=LUG
46     1,     1,   401,   405,     5, 10001, 10401, 10405, 10005,
47   201,   403,   205,     3, 10201, 10403, 10205, 10003,
48 5001,  5401,  5405,  5005
49     2,     5,   405,   409,     9, 10005, 10405, 10409, 10009,
50   205,   407,   209,     7, 10205, 10407, 10209, 10007,
51 5005,  5405,  5409,  5009
52  .......
```

此处使用三行数据来完全定义一个C3D20R单元的连接（最少需要两行）。如果`*ELEMENT`选项块中的数据行以逗号结尾，则表示下一行数据包含定义当前单元的更多节点。参数`ELSET=LUG`表示接下来数据行中定义的所有单元将被存储在一个名为`LUG`的单元集中。如果您的模型在`*ELEMENT`选项中没有描述性的单元集名称，请将其更改为`LUG`。

**节点集和单元集**

节点集和单元集是Abaqus输入文件的重要组成部分，因为它们允许您有效地分配载荷、边界条件和材料属性。它们还在定义仿真将产生的输出方面提供了极大的灵活性，并使输入文件更容易理解。

一些预处理器（如Abaqus/CAE）允许您在构建模型时选择和命名实体组（如节点和单元）；当创建Abaqus输入文件时，会从这些组生成节点集和单元集。

您可以使用输入文件中的`*NSET`和`*ELSET`选项定义集。集的名称使用`NSET`或`ELSET`参数指定。数据行列出包含在该集中的节点或单元。每个数据行最多可包含16个数字，并且可以根据需要包含任意数量的数据行。例如，节点集`LHEND`（参见图4-16）可以定义为：

```
*NSET, NSET=LHEND
53  3241,  3243,  3245,  3247,  3249,  3251,  3253,  3255, 3257,
54  8241,  8245,  8249,  8253,  8257, 13241, 13243, 13245, 13247,
55 13249, 13251, 13253, 13255, 13257, 18241, 18245, 18249, 18253,
56 18257, 23241, 23243, 23245, 23247, 23249, 23251, 23253, 23255,
57 23257
```

如果您正在使用编辑器向输入文件添加节点集或单元集，且标识号遵循规律模式，则`GENERATE`参数允许通过指定起始节点号、结束节点号和节点号增量来包含一系列节点。例如，节点集`LHEND`可以如下定义：

```
*NSET, NSET=LHEND, GENERATE
58  3241,  3257, 2
59  8241,  8257, 4
60 13241, 13257, 2
61 18241, 18257, 4
62 23241, 23257, 2
```

集也可以通过引用其他集来创建。如果您的预处理器没有创建图4-16所示的单元集`BUILTIN`或节点集`HOLEBOT`，请使用编辑器将它们添加到输入文件中；它们在限制仿真期间的输出时至关重要。您还应创建图4-16所示的单元集`PRESS`。请记住使用模型中的节点和单元编号，而不是图中所示的编号。

**截面属性**

在《Abaqus分析用户指南》第28章"连续体单元"中查找C3D20R单元，以确定正确的单元截面选项以及必须为此单元指定的数据。您会发现C3D20R单元使用`*SOLID SECTION`选项来定义单元的截面属性。因为这是三维单元，Abaqus不需要单元截面的额外几何数据。

单元集`LUG`包含所有单元，因此该单元集适用于本示例。以下单元截面选项语句用于本示例：

```
*SOLID SECTION, ELSET=LUG, MATERIAL=STEEL
```

如果您没有使用名称`LUG`定义单元集，请使用模型中包含所有单元的任何单元集的名称作为`ELSET`参数的值。模型中的材料属性名称为`STEEL`。

**材料**

连接销由软钢制成，因此在所施加的载荷下具有各向同性、线性弹性材料行为。假设*E* = 200 GPa，且![img](../graphics/gsk_eqn00028.gif) = 0.3。这些值在`*ELASTIC`选项的数据行上给出（请记住第2章"Abaqus基础"中的架空起重机示例）。以下材料属性定义在输入文件中指定这些属性：

```
*MATERIAL, NAME=STEEL
63  *ELASTIC
64  200.E9, 0.3
```

`*MATERIAL`选项上`NAME`参数的值必须与`*SOLID SECTION`选项上`MATERIAL`参数的值相匹配。

## 4.3.5 审核输入文件——历史数据

输入文件的历史数据部分从第一个`*STEP`选项开始。许多预处理器默认在输入文件中创建线性静力步骤。本示例将使用通用静力步骤。以下选项定义该步骤：

```
*STEP
65  <possibly a title describing this step>
66  *STATIC
```

如果这些选项不在您的输入文件中，请将它们添加到现有数据的末尾。如果您在`*STEP`选项后面的数据行上添加描述该步骤中正在模拟的事件的合适标题，则其他人更容易理解您的模型。

**边界条件**

在连接销模型中，销连接到母结构的所有节点都需要在左端在所有三个方向上受到约束（参见图4-19）。

![图 4–19]((../graphics/gss-lug.png))

在本例中，每个约束自由度都在`*BOUNDARY`选项块中单独指定，如下所示：

```
*BOUNDARY
67  3241, 1,1
68  3241, 2,2
69  3241, 3,3
70  ......
```

如果约束了大量节点，这些数据可能会占用大量计算机内存。如果许多节点具有相同的边界条件，则将约束直接应用于包含所有节点的节点集会更有效。因此，在销模型中我们更倾向于创建节点集`LHEND`来指定约束：

```
*BOUNDARY
71  LHEND, ENCASTRE
```

如果您认为边界条件定义不正确，可以在Abaqus/Viewer中显示它们，并与图4-19所示的边界条件进行比较。2.3.9节"后处理"末尾给出的后处理说明讨论了如何执行此操作。

**载荷**

销在孔洞下半部分承受50 MPa的分布压力。压力载荷可以通过选择施加载荷的单元面来方便地使用预处理器定义。在连接销输入文件中，这些载荷将显示为`*DLOAD`选项块。例如，连接销的`*DLOAD`选项块可能如下所示：

```
*DLOAD
72     1,  P6, 5.E+07
73     2,  P6, 5.E+07
74     3,  P6, 5.E+07
75     4,  P6, 5.E+07
76    13,  P6, 5.E+07
77  ...
78  1015,  P6, 5.E+07
79  1016,  P6, 5.E+07
```

每行数据的格式为：

```
<element or element set name>, <load ID>, <load magnitude>
```

在此情况下，"load ID"由字母"P"后跟施加压力的单元面编号组成。面编号取决于单元的连接性，并在《Abaqus分析用户指南》中为每种单元类型定义。对于本示例中使用的三维六面体单元，面编号如图4-20所示。

![图 4–20]((../graphics/gss-hexahedral-nls.png))

在模型中（如A.2节"连接销"中所定义），压力施加到孔底周围单元的面6上，因此load ID为"P6"。

对于使用预处理器生成的网格，面编号，从而load ID将取决于网格的生成方式。某些预处理器（如Abaqus/CAE）可以自动确定正确的load ID；这使得在复杂网格上指定压力载荷变得非常容易。但是，这种方法往往会在输入文件中产生很长的数据行列表。在模型中对每个单元使用相同的load ID和载荷大小的情况下，您可以使用单元集（效率更高）来施加压力载荷。例如，在本模型中`*DLOAD`选项块可能显示为：

```
*DLOAD
81  PRESS,  P6, 5.E+07
```

其中我们利用了图4-16所示成员组成的单元集`PRESS`。

**输出请求**

默认情况下，许多预处理器会创建包含大量输出请求选项的Abaqus输入文件。这些请求是Abaqus自动生成的输出数据库文件请求之外的附加请求。如果在编辑输入文件时发现创建了这些额外的输出选项，请删除它们，因为它们通常会产生过多不必要的输出。

您被要求确定施加载荷时连接销的挠度。获取此结果的简单方法是打印模型中的所有位移。但是，销上最大挠度的位置可能在载荷施加的孔底。此外，只有2方向（U2）的位移才是您感兴趣的。您应该已经创建了一个名为`HOLEBOT`的节点集，其中包含这些节点。使用该集将请求的位移限制为仅孔底的五个节点，并将输出限制为仅垂直位移。

```
*NODE PRINT, NSET=HOLEBOT
82  U2,
```

检查约束处的反作用力是否与施加的载荷相平衡是一种很好的做法。可以通过指定变量`RF`来打印节点处的所有反作用力。我们再次使用节点集`LHEND`将输出限制为那些被约束的节点。

```
*NODE PRINT, NSET=LHEND, TOTAL=YES, SUMMARY=NO
83  RF,
```

您可以定义多个`*NODE PRINT`和`*EL PRINT`选项。参数`TOTALS=YES`导致打印节点集中所有节点的反作用力之和。`SUMMARY=NO`参数防止打印表中最小值和最大值。

以下命令打印约束端（单元集`BUILTIN`）单元的应力张量（变量S）和Mises应力（变量`MISES`）：

```
*EL PRINT, ELSET=BUILTIN
84  S, MISES
```

您将使用`NFORC`输出变量在4.3.8节"后处理——可视化结果"中创建和显示自由体切割。以下选项在写入默认输出的同时，将由于单元应力产生的节点力写入输出数据库：

```
*OUTPUT, FIELD, VARIABLE=PRESELECT
85  *ELEMENT OUTPUT
86  NFORC,
87  *OUTPUT, HISTORY, VARIABLE=PRESELECT
```

步骤的结束由以下选项表示：

```
*END STEP
```

此输入选项必须是模型中的最后一个选项。

## 4.3.6 运行分析

如果修改了任何输入数据，请将输入存储在名为`lug.inp`的文件中（示例文件列在A.2节"连接销"中）。然后，使用以下命令运行仿真：

```
abaqus job=lug interactive
```

作业完成后，检查数据文件`lug.dat`中是否有任何错误或警告。如果有错误，请更正输入文件并再次运行仿真。如果在更正任何错误时遇到问题，请尝试将您的输入文件与A.2节"连接销"中给出的文件进行比较。检查每个输入选项的参数是否正确。


**更改视图**

默认视图是等轴测视图。您可以使用"View"菜单中的选项或"View Manipulation"工具栏中的视图工具更改视图。您还可以通过输入旋转角度、视点、缩放因子或视口平移比例来指定视图。也可以使用3D指南针进行直接视图操作。

**使用3D指南针操作视图：**

* 单击并拖动3D指南针的直线轴之一可沿轴平移。
* 单击并拖动3D指南针上任一四分之一圆形面可沿平面平移。
* 单击并拖动3D指南针周边上的三条弧之一可绕垂直于包含该弧的平面的轴旋转模型。
* 单击并拖动3D指南针顶部附近的自由旋转手柄可绕其支点自由旋转模型。
* 单击3D指南针上任一轴的标签可选择预定义的视图（所选轴垂直于视口平面）。
* 双击3D指南针上的任意位置可指定视图。

本指南中的大多数视图是直接指定的。这是为了让您能够通过与指南中的图像进行对比来确认模型的状态。但是，我们鼓励您根据需要练习使用上述方法来操作视图。

**指定视图：**

1. 从主菜单栏中，选择**View → Specify**（或双击3D指南针）。

   将出现"Specify View"对话框。

2. 从可用方法列表中，选择**Viewpoint**。

   在"Viewpoint"方法中，您输入三个值，表示观察者的*X*、*Y*和*Z*位置。您还可以指定一个向上向量。Abaqus将您的模型放置在此向量指向上方的位置。

3. 输入视点向量的*X*、*Y*和*Z*坐标为`1, 1, 3`，向上向量的坐标为`0, 1, 0`。

4. 单击**OK**。

   Abaqus/Viewer以指定视图显示您的模型，如图4-25所示。

**为模型子集生成表格数据报告**

先前使用打印输出请求为此模型生成了表格输出数据。但是，对于复杂模型，使用Abaqus/Viewer为所选区域写入这些数据很方便。这是通过将显示组与报告生成功能结合使用来实现的。对于连接销问题，我们将生成以下表格数据报告：

* 销约束端的单元应力（确定销中的最大应力）
* 销约束端的反作用力（检查约束处的反作用力是否与施加的载荷相平衡）
* 孔底的垂直位移（确定施加载荷时销的挠度）

每个报告都将使用在视口中选择的显示组内容来生成。因此，首先为每个感兴趣的区域创建并保存显示组。

**创建并保存包含约束端单元的显示组：**

1. 在Results Tree中，双击**Display Groups**。

2. 从"Item"列表中选择**Elements**，从选择方法中选择**Pick from viewport**。

3. 恢复选择屏幕上最近的实体的选项。

4. 在提示区域中，将选择方法设置为**by angle**；然后单击销的约束面。当销约束面上所有单元在视口中高亮显示时，单击**Done**。在"Create Display Group"对话框中，单击**Save Selection As**。将显示组保存为`built-in elements`。

**创建并保存包含约束端节点的显示组：**

1. 在"Create Display Group"对话框中，从"Item"列表中选择**Nodes**，从选择方法中选择**Pick from viewport**。

2. 在提示区域中，将选择方法设置为**by angle**；然后单击销的约束面。当销约束面上所有节点在视口中高亮显示时，单击**Done**。在"Create Display Group"对话框中，单击**Save Selection As**。将显示组保存为`built-in nodes`。

**创建并保存包含孔底节点的显示组：**

2. 在提示区域中，将选择方法设置为**individually**；然后选择销中孔底的节点，如图4-35所示。当孔底所有节点在视口中高亮显示时，单击**Done**。在"Create Display Group"对话框中，单击**Save Selection As**。将显示组保存为`nodes at hole bottom`。

## 4.3.9 使用Abaqus/Explicit重新运行分析

现在您将评估在同一载荷突然施加时销的动力响应。特别感兴趣的是销的瞬态响应。您将需要修改Abaqus/Explicit分析的模型。在继续之前，请将现有输入文件复制到名为`lug_xpl.inp`的输入文件。随后所有更改都针对`lug_xpl.inp`输入文件进行。在运行显式分析之前，您需要更改单元类型、在材料模型中添加密度，以及更改步骤类型。此外，您应该修改场输出请求。

**更改单元类型**

二阶六面体单元在Abaqus/Explicit单元库中不可用。因此，您需要将`*ELEMENT`选项上指定的单元类型从C3D20R更改为C3D8R。此更改还需要您编辑单元节点连接，以便为每个单元仅指定八个节点。例如，以下`*ELEMENT`选项块用于在`lug.inp`中定义其中一个单元：

```
*ELEMENT, TYPE=C3D20R
227     1,     1,   401,   405,     5, 10001, 10401, 10405, 10005,
228   201,   403,   205,     3, 10201, 10403, 10205, 10003,
229 5001,  5401,  5405,  5005
```

在`lug_xpl.inp`中，此选项块有两处更改：单元类型已更改为C3D8R，节点连接由原始列表中的前八个节点组成，这些节点定义了单元的角节点。

```
*ELEMENT, TYPE=C3D8R
230     1,     1,   401,   405,     5, 10001, 10401, 10405, 10005
```

因为C3D8R单元采用减缩积分，所以使用增强应变算法来控制沙漏效应。您可以使用`*SECTION CONTROLS`选项指定增强沙漏：

```
*SOLID SECTION, ELSET=LUG, MATERIAL=STEEL, CONTROLS=EC-1
*SECTION CONTROLS, NAME=EC-1, HOURGLASS=ENHANCED
```

**编辑材料定义**

因为Abaqus/Explicit执行动态分析，所以完整的材料定义需要您指定材料密度。对于此问题，假设密度等于7800 kg/m³。

您可以通过将`*DENSITY`选项添加到材料选项块来修改材料定义。连接销的完整材料定义为：

```
*MATERIAL, NAME=STEEL
234  *ELASTIC
235  200.E9, 0.3
236  *DENSITY
237  7800.,
```

**用动态显式步骤替换静力步骤**

修改步骤定义以在0.005 s期间检查销的动力响应。此更改要求您更改`*STEP`选项块，静力分析中该选项块如下所示：

```
*STEP
238  Apply uniform pressure to the hole
*STATIC
```

将此选项块替换为：

```
*STEP
239  Dynamic lug loading
240  *DYNAMIC, EXPLICIT
241  , 0.005
```

**请求以均匀间隔的场输出和默认历史输出**

以125个均匀间隔写入场输出，并写入默认历史输出。您可以通过将`NUMBER INTERVAL`参数附加到`*OUTPUT`选项块来指定均匀间隔的输出。将现有输出请求替换为：

```
*OUTPUT, FIELD, NUMBER INTERVAL=125
242  *NODE OUTPUT
243  RF, U
244  *ELEMENT OUTPUT, DIRECTIONS=YES
245  S,
246  *OUTPUT, HISTORY, VARIABLE=PRESELECT
```

将更改保存到名为`lug_xpl.inp`的输入文件。然后，使用以下命令运行仿真：

```
abaqus job=lug_xpl
```

## 4.3.10 动态分析结果的后处理**

在使用Abaqus/Standard进行的静力分析中，您检查了销的变形形状以及应力和位移输出。对于Abaqus/Explicit分析，您同样可以检查销中的变形形状、应力和位移。由于突然加载可能导致瞬态动力学效应，您还应该检查内能和动能、位移以及Mises应力随时间的变化。

打开此作业创建的输出数据库（`.odb`）文件。

**绘制变形形状**

从主菜单栏中，选择**Plot → Deformed Shape**；或使用工具箱中的![img](../graphics/ico_plotDeformed.png)工具。图4-36显示了分析结束时模型的变形形状。

![图 4–36]((../graphics/gsk-displaced-exp.png))

如前所述，Abaqus/Explicit默认采用大变形理论；因此，变形比例因子自动设置为1。如果位移太小而看不见，可以应用缩放以帮助研究响应。

为了更清楚地看到销中的振动，将变形比例因子更改为50。此外，制作销变形形状的时间历史动画，并降低时间历史动画的帧速率。

销变形形状的时间历史动画表明，突然施加的载荷会在销中引起振动。通过绘制内能、动能、位移和应力随时间变化的图，可以获得有关销在这种加载条件下行为的更多见解。需要考虑的一些问题是：

1. 能量是否守恒？
2. 本次分析是否需要大变形理论？
3. 峰值应力是否合理？材料会屈服吗？

**X–Y绘图**

X–Y图可以显示变量随时间的变化。您可以从场输出和历史输出创建X–Y图。

**创建内能和动能随时间变化的X–Y图：**

1. 在Results Tree中，展开名为`lug_xpl.odb`的输出数据库下的**History Output**容器。

2. 出现历史部分输出数据库中所有变量的列表；这些是您可以绘图的唯一历史输出变量。

   从可用输出变量列表中，双击**ALLIE**绘制整个模型的内能。

   Abaqus从输出数据库文件中读取曲线数据，并绘制如图4-37所示的图。

![图 4–37]((../graphics/gsk-lugexp-allie.png))

3. 重复此过程绘制整个模型的动能**ALLKE**（如图4-38所示）。

![图 4–38]((../graphics/gsk-lug-allke-nls.png))

内能和动能都显示出反映销振动的振荡。在整个仿真过程中，动能转化为内能（应变能），反之亦然。因为材料是线性弹性的，所以总能量守恒。可以通过将系统的总能量`ETOTAL`与`ALLIE`和`ALLKE`一起绘制来看到这一点。`ETOTAL`的值在整个分析过程中大约为零。动态分析中的能量平衡在第9章"非线性显式动力学"中进一步讨论。

我们将检查销孔底部的节点位移，以评估几何非线性效应在此仿真中的重要性。

**生成位移-时间图：**

1. 绘制销的变形形状。在Results Tree中，双击**XY Data**。

2. 在出现的"Create XY Data"对话框中，选择**ODB field output**作为源，然后单击**Continue**。

3. 在出现的"XY Data from ODB Field Output"对话框中，选择**Unique Nodal**作为读取*X–Y*数据的位置类型。

4. 单击**U: Spatial displacement**旁边的箭头，然后切换打开**U2**作为*X–Y*数据的位移变量。

5. 选择**Elements/Nodes**选项卡。选择**Pick from viewport**作为识别要为其获取*X–Y*数据的节点的选择方法。

6. 单击**Edit Selection**。在视口中，选择孔底部的一个节点，如图4-39所示（如果需要，更改渲染样式以便于选择）。在提示区域中单击**Done**。

![图 4–39]((../graphics/gsk-lugexp-node-nls.png))

7. 在"XY Data from ODB Field Output"对话框中单击**Plot**，绘制节点位移随时间变化的图。

   振荡历史（如图4-40所示）表明位移很小（相对于结构尺寸）。

![图 4–40]((../graphics/gsk-lugexp-nodaldisp.png))

因此，此问题可以使用小变形理论充分解决。这将降低仿真成本，而不会显著影响结果。几何非线性效应在第8章"非线性"中进一步讨论。

我们还对连接销的应力历史感兴趣。销靠近约束端的区域特别值得关注，因为预计在那里发生的峰值应力可能导致材料屈服。

**生成Mises应力-时间图：**

1. 再次绘制销的变形形状。

2. 在"XY Data from ODB Field Output"对话框中选择**Variables**选项卡。取消选择**U2**作为*X–Y*数据图的变量。

3. 将"Position"字段更改为**Integration Point**。

4. 单击**S: Stress components**旁边的箭头，然后切换打开**Mises**作为*X–Y*数据的应力变量。

5. 选择**Elements/Nodes**选项卡。选择**Pick from viewport**作为识别要为其获取*X–Y*数据的单元的选择方法。

6. 单击**Edit Selection**。在视口中，选择销约束端附近的一个单元，如图4-41所示。在提示区域中单击**Done**。

![图 4–41]((../graphics/gsk-lugexp-elem.png))

7. 在"XY Data from ODB Field Output"对话框中单击**Plot**，绘制所选单元处Mises应力随时间变化的图。

峰值Mises应力约为550 MPa，如图4-42所示。该值大于钢的典型屈服强度。因此，材料在经历如此大的应力之前就会屈服。材料非线性在第10章"材料"中进一步讨论。
