# 20.12.6 创建参数方程


参数方程是关联草图中不同数量的数学表达式。**表达式构建器**帮助您使用草图中的参数创建参数方程。例如,如果您希望 Sketcher 对象的宽度是其长度的两倍,您可以[将尺寸与参数关联](pt03ch20hla01.md),并使用**表达式构建器**创建参数方程 ![](../graphics/usi_eqn00694.gif)。

您可以使用以下运算符来创建参数方程:

**数学运算:**

| **+** | 加。 |
| --- | --- |
| **** | 减。 |
| ***** | 乘。 |
| **/** | 除。 |
| **1/A** | 用参数、值或表达式除 1。 |
| **abs(A)** | 取参数、值或表达式的绝对值。 |
| **sqrt(A)** | 取参数、值或表达式的平方根。 |

**三角运算:**

| **cos(A)** | 取参数、值或表达式的余弦。 |
| --- | --- |
| **acos(A)** | 取参数、值或表达式的反余弦。 |
| **cosh(A)** | 取参数、值或表达式的双曲余弦。 |
| **sin(A)** | 取参数、值或表达式的正弦。 |
| **asin(A)** | 取参数、值或表达式的反正弦。 |
| **sinh(A)** | 取参数、值或表达式的双曲正弦。 |
| **tan(A)** | 取参数、值或表达式的正切。 |
| **atan(A)** | 取参数、值或表达式的反正切。 |
| **tanh(A)** | 取参数、值或表达式的双曲正切。 |

**对数和指数运算:**

| **exp(A)** | 取参数、值或表达式的指数。 |
| --- | --- |
| **log(A)** | 取参数、值或表达式的自然对数。 |
| **log10(A)** | 取参数、值或表达式以 10 为底的对数。 |

**创建表达式的步骤:**

1. 当**参数管理器**中的表达式高亮显示时,单击**表达式构建器**。Abaqus/CAE 会打开**表达式构建器**对话框。
2. 从**运算符**列表中,选择所需的运算。运算符会出现在表达式窗口中。
3. 从**参数名称**选项中,单击要对其操作的参数的名称,然后单击**添加到表达式**。您可以选择**参数管理器**中位于您正在定义的参数上方的所有参数。参数名称会出现在表达式窗口中。
4. 根据需要重复第 2 步和第 3 步来完成表达式。在某些情况下,您可能需要重新定位光标以正确放置表达式的下一部分。
5. 完成后,单击**确定**保存更改并关闭对话框。Abaqus/CAE 会更新**参数管理器**中的表达式和当前值。
