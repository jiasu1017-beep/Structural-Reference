### 概述

织物材料行为：
- 可与线弹性模型（["Linear elastic behavior," Section 22.2.1](pt05ch22s02abm02.md)）结合使用；
- 专门为织物材料设计；
- 支持各向异性弹性响应；
- 考虑面内剪切刚度；以及
- 适用于复合材料层合板中的织物层。

### 定义织物材料参数

织物材料需要以下参数：
- 经向弹性模量
- 纬向弹性模量
- 面内剪切模量
- 泊松比

| **输入文件用法：** | ``` [*FABRIC](../key/key-link.md#usb-kws-mfabric) ``` |
| --- | --- |

### 与其他材料模型的结合

织物材料行为可与线弹性模型结合使用。

### 元素

织物材料行为可与 Abaqus/Standard 和 Abaqus/Explicit 中的膜元素和壳元素一起使用。
