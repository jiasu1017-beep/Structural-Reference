# 52.1.2 应力结果的提取





沿着应力线的每个点处的应力是通过使用"结果选项"对话框中定义的平均标准对唯一节点应力值进行插值获得的(有关指定平均标准的详细信息,请参阅["控制结果平均," 第42.6.6节](pt05ch42s04hlb02.md))。Abaqus/CAE使用由应力线定义的局部坐标系来获取应力结果。局部x轴沿着应力线;它起源于应力线上的第一个点,由连接应力线上第一个点和第二个点的向量定义。局部y轴和z轴的确定方式如下:

**对于二维模型:**

局部y轴通过将全局z轴与局部x轴进行向量乘法获得。局部z轴与全局z轴相同。

**对于三维模型:**

**如果局部x轴平行于全局y轴:**

如果正局部x轴沿着正全局y轴,则局部y轴将是全局x轴的负方向。如果正局部x轴沿着负全局y轴,则局部y轴将是全局x轴。局部z轴通过将局部x轴与局部y轴进行向量乘法获得。

**如果局部x轴不平行于全局y轴:**

局部z轴通过将局部x轴与全局y轴进行向量乘法获得。局部y轴通过将局部z轴与局部x轴进行向量乘法获得。

然后对应力进行线性化。非轴对称模型和轴对称模型的应力线性化计算不同。有关这些计算的更多信息,请参阅["应力线性化," Abaqus理论指南第2.17.1节](../stm/stm-link.md#stm-anl-stresslinear)。

Abaqus以X-Y曲线图的形式显示产生的线性化应力。每个应力分量显示三条独立的曲线,分别表示:

1. 整个应力线上的实际应力分布。
2. 线性化膜应力。
3. 线性化膜应力和线性化弯曲应力之和。

此外,每个应力分量的线性化应力输出以及线性化应力的应力不变量计算都被写入报告文件。
