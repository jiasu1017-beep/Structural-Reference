# 3.2.29 将Abaqus文件翻译为Nastran bulk数据文件





**产品：** Abaqus/Standard  Abaqus/Explicit  

##### **参考**

- ["Abaqus执行程序：概述，" 第3.1.1节](pt01ch03s01abo02.md)
- ["将Nastran bulk数据文件翻译为Abaqus输入文件，" 第3.2.28节](pt01ch03s02abx28.md)

### 概述

从Abaqus到Nastran的翻译器将Abaqus文件中的某些实体转换为其在Nastran中的对应物。只能翻译"平面"Abaqus文件；即，Abaqus文件不能包含部件和装配。

### 使用翻译器

Abaqus输入数据必须在扩展名为`.inp`或`.sim`的文件中。如果您指定`.inp`文件，执行程序会翻译选定的关键字并创建扩展名为`.bdf`的Nastran bulk数据文件。如果您使用**substructure**选项并指定子结构`.sim`文件，执行程序会将子结构数据翻译为Nastran DMIG系数，并创建扩展名为`.bdf`的Nastran bulk数据文件。

### 命令行选项

**job**

此选项用于指定要由翻译器输出的Nastran bulk数据文件的名称。它也是Abaqus文件的默认名称。翻译器生成的诊断信息被写入名为`*job-name*.log`的文件。

**input**

此选项用于指定包含Abaqus数据的文件的名称（如果与*job-name*不同）。

**substructure**

此选项用于将Abaqus `.sim`文件中的子结构翻译为Nastran bulk数据文件(`.bdf`)格式。

**complex**

此选项用于决定结构阻尼项如何表示。如果**complex**=`YES`（默认），结构阻尼项被写为刚度矩阵的虚部；如果**complex**=`NO`，结构阻尼项被写为单独的实矩阵。
