### 概述

本节提供Abaqus/Aqua中可用的弹塑性关节单元的参考。

### 单元类型

| JOINT2D | 二维弹塑性关节单元 |
| --- | --- |
|  |

| JOINT3D | 三维弹塑性关节单元 |
| --- | --- |
|  |

##### 激活的自由度

1, 2, 6 用于JOINT2D

1, 2, 3, 4, 5, 6 用于JOINT3D

##### 额外求解变量

无。

### 需要的节点坐标

无。

### 单元属性定义

| **输入文件用法：** | ``` [*EPJOINT](../key/key-link.md#usb-kws-mepjoint) ``` |
| --- | --- |

### 基于单元的载荷

无。

### 单元输出

与下面的力和力矩对应的相对位移和旋转通过请求相应的"应变"来选择。可获得弹性和塑性应变。对于插销靴，垂直（塑性）嵌入量从分析开始给出为PE11；总垂直嵌入量可用PEEQ获得。

#### JOINT2D

| S11 | 第一个局部方向的总直接力。 |
| --- | --- |

| S22 | 第二个局部方向的总直接力。 |
| --- | --- |

| S12 | 第三个局部方向的总力矩。 |
| --- | --- |

#### JOINT3D

| S11 | 第一个局部方向的总直接力。 |
| --- | --- |

| S22 | 第二个局部方向的总直接力。 |
| --- | --- |

| S33 | 第三个局部方向的总直接力。 |
| --- | --- |

| S12 | 第三个局部方向的总力矩。 |
| --- | --- |

| S13 | 第二个局部方向的总力矩。 |
| --- | --- |

| S23 | 第一个局部方向的总力矩。 |
| --- | --- |

### 单元关联的节点

两个节点。
