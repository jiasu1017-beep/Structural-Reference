# 9.1 什么是Abaqus/CAE模型数据库？





模型数据库（文件扩展名`.cae`）存储模型和分析作业。（有关分析作业的更多信息，请参阅["理解分析作业，"第19.2节"](pt03ch19s02.md)。）您可以在工作站或网络上存储多个模型数据库，但Abaqus/CAE一次只能处理其中一个。模型数据库可以包含多个模型；如果您计划同时处理多个模型，它们必须存储在一个模型数据库中。当前使用的模型数据库称为当前模型数据库；Abaqus/CAE会在主窗口顶部显示当前模型数据库的名称，如图[图9-1](pt02ch09s01.md#dbs-models)所示。

**图9-1** Abaqus/CAE显示模型数据库名称和模型名称。

![](../graphics/dbs-models-nls.png)

首次启动Abaqus/CAE时，**开始会话（Start Session）**对话框允许您创建新的空模型数据库或打开现有的模型数据库。您在Abaqus/CAE中创建或定义的所有内容都存储在此模型数据库中。您可以通过从主菜单栏选择****文件（File）**![](../graphics/images/arrow.gif)**保存（Save）****或****文件（File）**![](../graphics/images/arrow.gif)**另存为（Save As）****来保存内容。

除非您执行明确的保存操作，否则Abaqus/CAE永远不会保存模型数据库；例如，没有基于计时器的自动保存。但是，当您处理模型时，Abaqus/CAE会维护所有更改模型数据库的操作的记录。尽管您可能没有保存模型数据库，但您始终可以重放复制其当前状态的操作。有关重新创建模型数据库的更多信息，请参阅["重新创建未保存的模型数据库，"第9.5.3节"](pt02ch09s05s03.md)。Abaqus/CAE向后兼容，可以打开由先前版本的Abaqus/CAE创建的模型数据库。

开始Abaqus/CAE会话后，您可以通过从主菜单栏选择****文件（File）**![](../graphics/images/arrow.gif)**打开（Open）****来打开现有的模型数据库，或者通过选择****文件（File）**![](../graphics/images/arrow.gif)**新建（New）****来创建新的模型数据库。如果您在当前模型数据库做了更改后打开或创建另一个模型数据库，Abaqus/CAE会在关闭当前模型数据库之前询问您是否要保存更改。

您可以在可视化模块中打开模型数据库来探测或查询其节点和元素，并为选定的属性绘制轮廓或符号。有关更多信息，请参阅["理解可视化模块的角色，"第40.1节"](pt05ch40s01.md)。
