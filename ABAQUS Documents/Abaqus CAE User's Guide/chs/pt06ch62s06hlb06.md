# 62.6.6 通过将点投影到面或平面上创建datum点







您可以通过在模型上选择一个点以及要投影该点的面或平面来定位datum点。Abaqus/CAE在选定的面或平面与通过选定点且垂直于该面或平面的直线的交点处创建datum点。

![](../graphics/dtm-hptpf-nls.png)

**通过将点投影到面或平面上创建datum点：**

1. 从主菜单栏中，选择****Tools**![](../graphics/images/arrow.gif)**Datum****。此时将出现**Create Datum**对话框。该对话框概述了您可以创建的datum几何体类型。Abaqus/CAE在提示区域中显示提示以指导您完成此过程。**提示：**您也可以使用模块工具箱中的![](../graphics/ico_dtmPtProjectOnFace.png)工具创建datum点。有关工具箱中datum工具的图示，请参见["使用Datum工具集，" 第62.2节](pt06ch62s02.md)。
2. 从对话框顶部的类型列表中，选择**Point**。**Method**列表指示您可以使用的方法来创建datum点。
3. 从**Method**列表中，选择**Project point on face/plane**。
4. 从当前视口中的零件或装配中，选择一个点。
5. 选择要投影该点的面或平面。选区可以是平面、非平面或datum平面。datum点会出现。您无法修改使用此方法创建的datum点；您必须删除旧点并创建一个新的。
