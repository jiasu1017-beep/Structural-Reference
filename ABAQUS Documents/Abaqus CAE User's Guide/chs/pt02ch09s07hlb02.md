# 9.8.2 打开现有模型





要打开模型并将其与当前视口关联，请从上下文栏中的**模型（Model）**列表中选择所需的模型。**模型（Model）**列表包含当前模型数据库中的所有模型。

Abaqus/CAE会切换到所选模型并将其与当前视口关联（由红色边框指示）。新模型会出现在上下文栏中的模型列表中。

您可以同时打开多个模型；视口的标题栏指示与当前视口关联的模型。在打开现有模型之前，您不必保存当前模型，因为Abaqus/CAE会将所有模型存储在模型数据库中。
