# 20.2.9 aStudy.report()





### 20.2.9 ***aStudy*.report()**：报告参数化研究结果。

**产品：** Abaqus/Standard  Abaqus/Explicit  

此命令用于报告跨参数化研究设计收集的结果。

##### **参考**

- ["参数化研究脚本，" 第20.1.1节](pt04ch20s01aus108.md)

### **命令：**

***aStudy*.report** (*token*, **results**= , **par**= , **designSet**= , **variations**=, **truncation**= , *additional data*)

### **符号：**

FILE

使用此符号指定结果作为带相关标题的表写入ASCII文件。

PRINT

使用此符号指定结果作为带相关标题的表打印。由于结果被打印到默认输出设备（屏幕），您可能希望限制表中的列数以便使表可读。

XYPLOT

使用此符号指定结果作为不带标题的表写入ASCII文件。此表随后可被Abaqus/CAE的可视化模块读取，以显示结果和参数值的X-Y曲线图。

### **必需数据：**

**results**

将**results**设置为要报告的结果名称序列；此序列必须用括号或方括号括起来。例如，`results=('e33_sinv.1', 'e52_strain', 'n25_u.3')`，其中`'e33_sinv.1'`是单元33的Mises应力（Mises是SINV记录的第一个分量），`'e52_strain'`是单元52的所有应变分量，`'n25_u.3'`是节点25的位移第三分量。此示例假定上述三个结果是通过分别请求SINV、E和U变量标识符键在先前的**gather**命令中收集的。

### **可选数据：**

**par**

将**par**设置为要包含在报告表中的参数或参数序列的名称。如果指定单个参数，则必须用匹配引号括起来；例如，`'par1'`。如果指定参数序列，则必须将其放在括号或方括号内，并且必须包含用匹配引号括起来并用逗号分隔的参数名称；例如，`('par1', 'par2', 'par3')`或`['par1', 'par2', 'par3']`。

如果省略**par**，参数化研究中的所有参数都包含在报告表中。

**designSet**

将**designSet**设置为要包含在报告表中的结果的设计集名称；此名称必须用匹配引号括起来。

如果省略**designSet**，参数化研究中的所有设计集的结果都包含在报告表中。

**variations**

将**variations**设置为ON，以指示报告表的第一列必须显示正在报告的设计名称。将**variations**设置为OFF，以指示正在报告的设计名称不显示在报告表的第一列中。

如果省略**variations**，报告中不包含设计名称列。

**truncation**

将**truncation**设置为ON，以指示必须以有限精度报告报告表的数据。将**truncation**设置为OFF，以指示必须以完整精度报告报告表的数据。

如果省略**truncation**，报告表的数据以有限精度报告。

##### **FILE和XYPLOT的额外数据：**

### **必需数据：**

**file**

将**file**设置为要将报告表写入的文件名称。文件名必须用匹配引号括起来。
