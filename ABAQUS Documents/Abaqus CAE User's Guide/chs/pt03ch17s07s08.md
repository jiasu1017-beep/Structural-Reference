# 17.7.8 Abaqus/CAE何时会删除网格？





零件、零件实例或区域的以下属性会影响网格的生成方式：
- 布种
- 单元形状
- 网格技术
- 网格算法
- 二维结构化区域的逻辑角
- 过渡控制
- 扫掠区域的扫掠路径

如果您更改列出的任何属性，则现有网格将不再与其属性一致。因此，Abaqus/CAE会删除网格，您可以重新创建与新属性匹配的新网格。（单元阶次和单元族是更改时不需要删除和重新创建网格的唯一属性。）

每当您更改会影响自顶向下网格区域这些属性的任何内容时，Abaqus/CAE都会显示一个类似于图17-39所示的对话框。

**图17-39** 警告对话框。

![](../graphics/mgn-controlwarn-nls.png)

您可以通过单击**删除网格**按钮删除网格，或者通过按**取消**按钮保持网格并退出过程。

如果切换**自动删除因网格控制更改而失效的网格**，则可以在当前会话的剩余时间内避免此警告消息。下次您尝试更改已包含网格的零件、零件实例或区域的属性时，网格将立即删除，不会显示任何警告。

如果在删除网格之前将模型保存到模型数据库，则如果对以后的网格划分尝试不满意，可以恢复到该网格。

由于自底向上网格的创建可能非常耗时，Abaqus/CAE在许多会导致自顶向下网格删除的情况下会尝试保留它们。自底向上网格在布种、分隔和虚拟拓扑更改期间保留。当您对区域进行分割或创建虚拟拓扑特征时，网格将被保留，但分割或虚拟拓扑操作产生的几何关联性将丢失。

**警告：**当您更改上述网格属性列表时，无法避免删除自顶向下网格。同样，如果您更改零件几何形状，Abaqus/CAE将始终无警告地删除自顶向下和自底向上网格。由于对于大型或复杂模型，重新网格划分可能非常耗时，因此在更改这些属性时应谨慎。

有关删除网格的详细分步说明，请参见["删除网格，"第17.17.2节](pt03ch17s17s02.md)。
