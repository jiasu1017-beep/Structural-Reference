# 15.13.19 定义循环对称







您可以定义循环对称相互作用，以仅使用单个重复扇区对整个360结构建模。选择****相互作用**![](../graphics/images/arrow.gif)**创建****，选择**循环对称**，然后通过选择主表面或节点区域、从表面或节点区域以及结构的对称轴来指定几何区域。您还可以控制创建完整360结构的扇区数，选择将执行特征频率分析的循环对称节点直径，以及选择将为稳态动力学步骤激发的循环对称节点直径。

您只能在初始步骤中创建循环对称相互作用，一个模型中只能有一个循环对称相互作用处于活动状态。

有关Abaqus中循环对称的更多信息，请参见Abaqus Analysis User's Guide的第10.4.3节"分析具有循环对称性的模型"。

**为模型定义循环对称：**

1. 从主菜单栏中，选择****相互作用**![](../graphics/images/arrow.gif)**创建****。**提示：**您也可以使用相互作用模块工具箱中的![](../graphics/ico_interaction.png)工具创建循环对称相互作用。
2. 在出现的**创建相互作用**对话框中，执行以下操作：- 命名该相互作用。有关命名对象的信息，请参见第3.2.1节"使用基本对话框组件"。- 选择初始步骤。- 选择**循环对称**类型的相互作用。
3. 单击**继续**关闭**创建相互作用**对话框。
4. 选择主表面。1. 在提示区域中，选择以下之一：- 如果要选择表面，请选择**表面**。- 如果要选择节点区域，请选择**节点区域**。2. 使用以下方法之一选择主表面：- 使用现有表面定义区域。在提示区域右侧单击**表面**。从出现的**区域选择**对话框中选择现有表面，然后单击**继续**。**注意：**默认选择方法基于您最近使用的选择方法。要恢复为另一种方法，请单击提示区域右侧的**在视口中选择**或**表面**。- 使用鼠标在视口中选择区域。（更多信息，请参见"在当前视口中选择对象"，第6.2节。）如果模型包含网格和几何的组合，请从提示区域单击以下之一：- 如果要从几何区域选择表面，请单击**几何**。- 如果要从本机或孤立网格选择表面，请单击**网格**。您可以使用角度方法从几何或网格中选择一组面或边或一组单元面。更多信息，请参见"使用角度和特征边缘方法选择多个对象"，第6.2.3节。您选择的主表面在视口中以红色高亮显示。
5. 选择从表面。1. 在提示区域中，选择以下之一：- 如果要选择表面，请选择**表面**。- 如果要选择节点区域，请选择**节点区域**。2. 使用与上一步骤中描述的相同方法选择从表面或区域。您选择的从表面或区域在视口中以洋红色高亮显示。Abaqus/CAE会提示您定义对称轴。
6. 使用以下方法之一选择对称轴上的第一个点：- 使用现有集合定义区域。该集合必须包含单个点或顶点。在提示区域右侧单击**集合**。从出现的**区域选择**对话框中选择现有集合，然后单击**继续**。**注意：**默认选择方法基于您最近使用的选择方法。要恢复为另一种方法，请单击提示区域右侧的**在视口中选择**或**集合**。- 使用鼠标在视口中选择节点或顶点。（更多信息，请参见"在当前视口中选择对象"，第6.2节。）如果模型包含网格和几何的组合，请从提示区域单击以下之一：- 如果要从几何区域选择表面，请单击**几何**。- 如果要从本机或孤立网格选择表面，请单击**网格**。您选择的点在视口中以红色高亮显示。
7. 使用与上一步骤中描述的相同方法，选择对称轴上的第二个点。您选择的点在视口中以洋红色高亮显示，并且打开**编辑循环对称**对话框。
8. 从**位置容差**选项中，指定Abaqus将把从表面上的节点连接到主表面的距离。选择以下选项之一：- 选择**使用计算的距离**以使用基于所用约束的元素formulations和表面类型自动计算的距离。- 选择**指定距离**，然后输入一个值以设置从节点将被连接到主表面的距离。
9. 切换关闭**调整从表面初始位置**以防止Abaqus将，从表面上的所有连接节点移动到主表面上。默认情况下，此选项处于打开状态，所有连接节点在初始配置中被移动而不在模型中施加应变。
10. 指定组成完整360结构的**总扇区数**。
11. 从**提取的节点直径**选项中，指定将执行特征频率分析的循环对称节点直径的范围。您只能在初始步骤或频率提取步骤中定义此范围。使用以下选项之一指定此范围：- 选择**所有可能的节点直径**以提取每个可能的循环对称节点直径。- 选择**指定范围**以提取您在**最低节点直径**和**最高节点直径**之间指定的每个循环对称节点直径。最高节点直径必须小于或等于扇区数的一半。
12. 指定与载荷定义中的载荷相关联的**激发的节点直径**。您可以在初始步骤或基于模态的稳态动力学步骤中选择激发的节点直径。只能激发一个循环对称节点直径。
13. 单击**确定**。Abaqus/CAE创建循环对称相互作用。
