# 39.8 可视化子结构输出





Abaqus/CAE 为分析中使用的每个子结构零件实例创建单独的输出数据库(`.odb`)文件，因此如果您想在装配其余部分的上下文中显示子结构结果，必须执行一些额外的步骤。Visualization 模块提供了以下工具，使您能够将子结构结果整合到模型的其余部分中：
- 您可以使用叠加绘图在同一个视口中显示子结构数据和装配其余部分的绘图。
- 您可以使用**组合 ODB** 插件将一个或多个子结构输出数据库文件中的数据与装配其余部分的数据组合在一起。
