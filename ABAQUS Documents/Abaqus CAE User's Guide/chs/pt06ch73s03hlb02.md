# 73.3.2 创建集







"集工具集"允许您创建以下类型的集：
- 几何集
- 节点集和单元集

当您在Abaqus/CAE创建的原生部件上工作时，您创建几何集。当您在网格上工作时，您创建节点集和单元集。有关更多信息，请参见["什么是集？," 第73.2.1节](pt06ch73s02s01.md)。

此外，您可以创建部件集或装配体集。当您在Part模块或Property模块中使用部件时，您创建部件集。当您在Assembly模块中对部件进行实例化时，您可以引用先前创建的任何部件集。最后，当您在装配体相关模块（Assembly、Interaction、Load和Mesh）中工作时，您创建装配体集。有关更多信息，请参见["部件集和装配体集有何不同？," 第73.2.2节](pt06ch73s02s02.md)。

**创建集的步骤：**

1. 从主菜单栏，选择****Tools**![](../graphics/images/arrow.gif)**Set**![](../graphics/images/arrow.gif)**Create****。**提示：**您也可以在**Set Manager**中单击**Create**。将出现"创建集"对话框。该对话框根据当前视口中显示的部件或装配体列出您可以创建的集类型。
2. 在对话框中：
   1. 输入集的名称。在不同部件上创建的部件集可以使用相同的名称；但是，装配体集名称必须唯一。
   2. 如果需要，选择集类型。
   3. 单击**Continue**。如果您在原生部件上创建节点或单元集，Abaqus/CAE会自动显示网格（如果尚未可见）。如果您在部件实例或包含原生和孤立网格组件的部件上创建节点或单元集，必须使用**Visible Objects**工具栏中的**Show native mesh**![](../graphics/ico_mesh.png)工具来显示和选择网格原生部分中的项目。
3. 在视口中选择要包含在集中的实体。**提示：**您可以使用**Selection**工具栏限制可以在视口中选择的实体类型。更多信息，请参见[第6章，"在视口中选择对象"](pt01ch06.md)。选择完实体后，单击鼠标按钮2。
