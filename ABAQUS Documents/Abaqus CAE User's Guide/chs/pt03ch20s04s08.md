# 20.4.8 在 Sketcher 中使用查询工具集







从主菜单栏中选择****工具**![](../graphics/images/arrow.gif)**查询****，或点击**查询**工具栏中的![](../graphics/ico_info.png)工具启动查询工具集。

您可以使用查询工具集请求常规信息或特定于模块的信息。有关常规查询显示的信息的讨论，请参见["获取模型的一般信息，"第 71.2.2 节](pt06ch71s02hlb02.md)。

以下查询特定于 Sketcher：

**约束**

Abaqus/CAE 在消息区域显示约束类型和受约束实体的名称，并在草图中高亮显示这些实体。

**详情**

Abaqus/CAE 在消息区域显示几何、顶点、约束、尺寸和未约束自由度数量的信息。
