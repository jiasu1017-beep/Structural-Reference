# 15.17.10 定义锁定

您可以为任何相对运动分量指定锁定准则。锁定准则可以依赖于相对运动可用分量的运动，以及相对运动可用分量和约束分量中的力/力矩。当满足任何一个准则时发生锁定，而不是当满足所有准则时发生锁定。有关更多信息，请参见["连接器止动和锁定，"Abaqus分析用户指南第31.2.8节](../usb/usb-link.md#usb-elm-econnstoplockbehav)。

**要创建或编辑锁定：**

1. 使用以下方法之一显示连接器截面编辑器：
   - 要创建新的连接器截面，请按照["创建连接器截面，"第15.12.11节"](pt03ch15s12s01.md)中概述的过程进行操作。
   - 要编辑现有的连接器截面，请从主菜单栏选择**连接器**![](../graphics/images/arrow.gif)**截面**![](../graphics/images/arrow.gif)**管理器**，从出现的列表中选择连接器截面，然后点击**编辑**。
2. 在**编辑连接器截面**对话框中，执行以下操作之一：
   - 要定义新的锁定选项，请点击**添加**并从出现的选项菜单中选择**锁定**。
   - 要编辑现有的锁定选项，请从**行为选项**列表中选择该选项以显示该选项的关联数据字段。
3. 在**分量**字段中，打开用于定义锁定准则的相对运动分量的开关。如果您想为不同分量指定不同的限制，则必须定义多个锁定选项。
4. 点击**锁定**字段右侧的箭头，并从出现的列表中选择一个选项：
   - 选择**全部**以在满足任何锁定准则时锁定所有相对运动分量。
   - 选择**指定**以选择在满足任何锁定准则时要锁定的可用相对运动分量。
5. 要基于运动定义锁定准则，请在编辑器的**位置锁定准则**部分输入将触发锁定的可用相对运动分量的相对位置限制。
6. 要基于力/力矩定义锁定准则，请在编辑器的**力/力矩锁定准则**部分输入将触发锁定的可用或约束相对运动分量的力/力矩限制。
7. 选择以下之一：
   - 如果您想继续定义行为选项，请点击**添加**，选择所需的行为选项，并继续连接器截面定义。有关定义其他行为选项的说明，请参见["使用连接器截面编辑器，"第15.17节"](pt03ch15s17.md)。
   - 如果您想查看或编辑现有的行为选项，请从**行为选项**列表中选择该选项。有关编辑行为选项的说明，请参见["使用连接器截面编辑器，"第15.17节"](pt03ch15s17.md)。
   - 如果您想保存连接器截面定义并退出编辑器，请点击**确定**。
