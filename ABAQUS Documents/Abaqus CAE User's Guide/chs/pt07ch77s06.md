# 77.6 在可视化模块中对节点或单元进行着色





当在当前视口中选择了输出数据库时，您可以对选定的**节点**和**单元**进行颜色编码来自定义视口中结果的显示。有关使用**颜色编码**对话框的分步说明，请参见["对几何图形和网格元素进行着色，"第77.4节](pt07ch77s04.md)。要按选定项目属性（而不是节点或单元）进行颜色编码，请参见["自定义单个对象的显示颜色，"第77.8节](pt07ch77s08.md)。

对于**单元**和**节点**项目类型，您的项目选择会因从**颜色编码**对话框顶部**依据**列表中选择的方法而异。某些选择方法要求您完成**颜色映射**表中的信息。无论**颜色映射**表是由Abaqus/CAE完成还是由您完成，一旦完成，您可以选择多个单元格来更改节点和单元颜色、节点符号形状以及节点符号大小（有关更多信息，请参见["从列表和表中选择多个项目，"第3.2.11节](pt01ch03s02s11.md)）。

更改节点和单元颜色时，必须从**选择颜色**对话框中选择您想要的颜色。Abaqus/CAE不支持节点和单元的自动颜色编码。此外，当您检查**节点**和**单元**的颜色映射时，**颜色映射**表中的列不可排序；如果需要从大型列表中查找节点或单元名称，请考虑使用筛选器。

关闭对话框或切换视口时，**颜色编码**对话框不会保留节点和单元的颜色选择。在更改节点和单元颜色时，您应该考虑经常保存颜色宏；有关更多信息，请参见["保存和恢复自定义颜色映射，"第77.11节](pt07ch77s11.md)。

选择以下选择方法来对模型中的单元和节点进行着色：

**从视口选择**

选择**从视口选择**可直接从视口中通过选择来指定单元或节点。分别点击**编辑选择**或**添加选择**来编辑现有行或向**颜色映射**表添加新行。Abaqus/CAE自动进入选择模式，提示区域显示**选择要进行颜色编码的*项目***。有关在视口中选择项目的更多信息，请参见[第6章，"在视口中选择对象](pt01ch06.md)。"点击**删除选择**可从表中删除高亮显示的行。

**单元标签（节点标签）**

选择**单元标签**或**节点标签**可按编号指定单元或节点。对于**颜色映射**表中的每一行，从**创建显示组**对话框右侧**零件实例**列的列表中选择节点或单元所属的零件实例名称，然后在**标签**字段中输入用逗号分隔的单元或节点编号列表，或输入编号范围，例如`1:4`。如果需要，您可以使用1以外的其他数字作为运算符来指定范围；例如，`1:21:5`选择标签为1、6、11、16和21的项目。

**结果值**

选择**结果值**可指定包含在给定值范围内的结果的单元或节点。

要考虑的输出变量显示在**颜色映射**表底部**场输出**按钮的右侧。要选择新的结果变量，请点击**场输出**；有关**场输出**对话框的更多信息，请参见["选择要显示的场输出，"第42.5节](pt05ch42s03.md)。从**类型**单元格中的筛选方法中进行选择；符号表示结果低于![](../graphics/dgtG_ResRangeBelow.png)、在...内部![](../graphics/dgtG_ResRangeInside.png)、在...外部![](../graphics/dgtG_ResRangeOutside.png)或高于![](../graphics/dgtG_ResRangeAbove.png)所选值或范围。在**最小值**和**最大值**单元格中输入所需的值或值，以指定所选筛选器类型的范围。您可以向表中添加行并选择不同的筛选方法和结果范围，但所有行都引用相同的场输出变量。模型中的每个单元（或节点）都用于筛选过程中，无论当前活动显示组如何。

**注意：**基于单元或节点输出变量进行筛选的边界始终基于节点处变量的值。因此，在将用户定义的边界与单元或节点输出量进行比较之前，单元和曲面输出量会在节点处进行外推和平均。**结果选项**对话框中的平均设置决定了单元和曲面变量在节点处的计算方式。例如，考虑一个使用75%的默认平均阈值基于Mises应力筛选单元的案例。外推至节点后，根据此阈值对值进行平均。这种条件平均可能导致节点处基于节点所属各个单元的贡献而产生几个不同的Mises应力值。任何Mises应力贡献落在用户定义边界内的单元都包含在颜色编码选择中。

**所有单元（所有节点）**

选择**所有单元**或**所有节点**可选择模型中指定类型的所有项目。不需要进一步指定项目。

**节点集**

选择**节点集**可指定模型中保存的节点集的新颜色。**颜色映射**表列出所有可用的集合名称。您可以使用**全部**项目类型来选择单元集。

**内部集合**

选择**内部集合**可指定Abaqus/CAE创建的内部节点或单元集。**颜色映射**表列出所有可用的集合名称。

**显示组**

选择**显示组**可指定保存的显示组。**颜色映射**表列出所有可用的显示组名称。

**零件实例**

选择**零件实例**可为所选零件实例中的所有节点指定新颜色。**颜色映射**表列出模型中的所有零件实例。

要选择零件实例中的所有单元，请对**全部**项目类型使用**零件实例**方法。
