# 53.2.4 自定义层压板堆叠绘图中的参考面





点击 **参考面（Reference Plane）** 选项卡来选择是否在层压板堆叠绘图中显示参考面，并控制参考面的外观。传统壳单元使用参考面来定义零件的几何形状。因此，您只能在外壳复合层上显示参考面；您不能在连续壳复合叠层或实体复合叠层上显示参考面。此外，如果层的偏移由分布定义，Abaqus/CAE 无法显示参考面。

**自定义层压板堆叠绘图中参考面的外观：**

1. 找到参考面选项。1. 从提示区域点击 **层压板堆叠绘图选项（Ply Stack Plot Options）** 按钮，或执行以下操作之一：- 在属性模块中，选择 ****视图（View）**![](../graphics/images/arrow.gif)**层压板堆叠绘图选项（Ply Stack Plot Options）****。- 在可视化模块中，选择 ****选项（Options）**![](../graphics/images/arrow.gif)**层压板堆叠绘图（Ply Stack Plot）**** 或点击工具箱中的 ![](../graphics/ico_plotPlyStackOptions.png) 工具。2. 在出现的 **层压板堆叠绘图选项（Ply Stack Plot Options）** 对话框中点击 **参考面（Reference Plane）** 选项卡。
2. 执行以下操作：- 切换 **显示参考面（Show reference surface）** 以在层压板堆叠绘图中显示参考面。- 选择参考面的颜色和半透明度。- 切换 **显示参考轮廓（Show reference outline）** 以显示参考面边缘周围的线条。选择线条的颜色、样式和粗细。
3. 点击 **应用（Apply）** 实施您的更改。层压板堆叠绘图会更改以反映您的规范。默认情况下，您的更改将在会话期间保存，并将影响该视口中所有后续层压板堆叠绘图。如果您想在后续会话中保留您的更改，请将它们保存到文件。有关更多信息，请参阅 ["保存自定义以供后续会话使用，" 第 55.1.1 节"](pt05ch55s01s01.md)。

# 53.2.3 自定义层压板堆叠绘图中纤维的外观





点击 **纤维（Fiber）** 选项卡来选择是否在层压板堆叠绘图中显示纤维以及纤维的外观。纤维始终绘制在 1–2 平面中，与 1 方向成一定角度。有关更多信息，请参阅 ["理解复合叠层和取向，" 第 23.3 节"](pt04ch23s03.md)。

如果您使用坐标系来定义层的取向，Abaqus/CAE 无法在不知道单元的空间取向的情况下确定单元内的取向。因此，Abaqus/CAE 在层纤维的位置显示星号（`*`）来表示它无法在 1–2 平面中绘制表示纤维方向的线。出于同样的原因，如果层的取向使用离散场分布定义，Abaqus/CAE 无法绘制准确表示纤维方向的线；层纤维位置的脱字号（`^`）表示层取向中的附加旋转是使用离散场定义的。

在实体复合叠层中，一层中的纤维并不总是平行于 1–2 平面运行（例如，如果层取向的 3 方向与单元堆叠方向未对齐）。由于层压板堆叠绘图中的纤维始终绘制在 1–2 平面中，它们并不是实体复合层中纤维的真实描绘。相反，纤维是复合截面或叠层定义中旋转角度的图形表示：层压板堆叠绘图中绘制的角度是层表中指定的相对于单元堆叠方向轴测量的旋转角度。

**自定义层压板堆叠绘图中纤维的外观：**

1. 找到纤维选项。1. 从提示区域点击 **层压板堆叠绘图选项（Ply Stack Plot Options）** 按钮，或执行以下操作之一：- 在属性模块中，选择 ****视图（View）**![](../graphics/images/arrow.gif)**层压板堆叠绘图选项（Ply Stack Plot Options）****。- 在可视化模块中，选择 ****选项（Options）**![](../graphics/images/arrow.gif)**层压板堆叠绘图（Ply Stack Plot）**** 或点击工具箱中的 ![](../graphics/ico_plotPlyStackOptions.png) 工具。2. 在出现的 **层压板堆叠绘图选项（Ply Stack Plot Options）** 对话框中点击 **纤维（Fiber）** 选项卡。
2. 执行以下操作：- 切换 **显示纤维（Show fibers）** 以在层压板堆叠绘图中显示纤维。- 选择在层压板堆叠绘图中表示纤维的线条的颜色、样式和粗细。- 拖动 **间距（Spacing）** 滑块来更改显示的纤维数量。
3. 点击 **应用（Apply）** 实施您的更改。层压板堆叠绘图会更改以反映您的规范。默认情况下，您的更改将在会话期间保存，并将影响该视口中所有后续层压板堆叠绘图。如果您想在后续会话中保留您的更改，请将它们保存到文件。有关更多信息，请参阅 ["保存自定义以供后续会话使用，" 第 55.1.1 节"](pt05ch55s01s01.md)。





# 54.4 指定报告文件名





Abaqus/CAE 将您的表格数据报告写入您选择的文件名。如果您选择将报告写入现有文件，您可以进一步指定是新信息应该追加到当前文件内容还是覆盖文件。默认是追加新信息。

**指定报告文件名：**

1. 找到 **文件（File）** 选项。**对于 X–Y 数据或场输出的报告：** 从主菜单栏中，选择 ****报告（Report）**![](../graphics/images/arrow.gif)**XY**** 或 ****报告（Report）**![](../graphics/images/arrow.gif)**场输出（Field Output）****；然后在出现的对话框中点击 **设置（Setup）** 选项卡。**文件（File）** 选项位于页面顶部。**对于自由体切割的报告：** 从主菜单栏中，选择 ****报告（Report）**![](../graphics/images/arrow.gif)**自由体切割（Free Body Cut）****。**文件（File）** 选项位于页面顶部。**对于探针值的报告：** 从主菜单栏中，选择 ****工具（Tools）**![](../graphics/images/arrow.gif)**查询（Query）****，然后在出现的对话框中点击 **探针值（Probe values）**。出现 **探针值（Probe Values）** 对话框。将光标移到当前视口中的模型或 X–Y 绘图上探测，然后点击鼠标按钮 1 将感兴趣的值存储在对话框中。完成后，点击 **写入文件（Write to File）**。出现 **报告探针值（Report Probe Values）** 对话框；**文件（File）** 选项位于对话框顶部。
2. 要使用标准文件浏览器选择文件名，请点击 **选择（Select）**。出现 **文件选择（File Selection）** 对话框。使用该对话框过滤和浏览现有文件。有关使用此对话框的更多信息，请参阅 ["使用文件选择对话框，" 第 3.2.10 节"](pt01ch03s02s10.md)。完成后，点击 **确定（OK）** 关闭对话框。
3. 在 **名称（Name）** 字段中输入要将报告写入的文件名。
4. 切换 **追加到文件（Append to file）** 以将此报告追加到您指定的文件的当前内容（如果有）。当 **追加到文件（Append to file）** 关闭时，现有文件内容将被覆盖。

要生成报告，请配置对话框中的其余选项；然后点击 **应用（Apply）**（对于 X–Y、场输出或自由体切割）或 **确定（OK）**（对于探针值报告）。完成后，点击 **取消（Cancel）** 关闭对话框。
![](../graphics/images/black4rule.gif)有关相关信息，请点击以下项目：- ["表格报告选项概述，" 第 54.2 节"](pt05ch54s02.md)
