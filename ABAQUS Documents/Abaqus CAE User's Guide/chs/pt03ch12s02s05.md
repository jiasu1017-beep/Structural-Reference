# 12.2.5 理解壳截面中的钢筋





您可以通过为每个钢筋层指定唯一的层名称来在壳截面中定义一层或多层增强（钢筋）。您还可以选择形成每个钢筋层的材料名称，并指定每个钢筋的横截面积、间距和每层中钢筋的取向。

要为每个钢筋层定义取向，您可以指定取向角度或取向名称。钢筋层的角度取向是相对于钢筋参考取向定义的。您使用**Assign**菜单为壳区域分配钢筋参考取向。如果您指定取向名称，则必须提供用户子程序[`ORIENT`](../sub/sub-link.md#sub-xsl-orient)。有关更多信息，请参见["定义钢筋层，"第12.13.19节](pt03ch12s13hlb15.md)和["分配钢筋参考取向"位于"分配材料取向或钢筋参考取向，"第12.15.4节](pt03ch12s15s01.md#usi-prp-assign-rebarorient)。

在Step模块中，您必须为钢筋请求输出，以便在Abaqus写入输出数据库的数据中包含钢筋输出，并在Visualization模块中查看钢筋取向的绘图。在Visualization模块中，Abaqus/CAE将钢筋层视为用于输出的截面点，您可以创建材料取向图来显示钢筋取向。

有关钢筋的更多信息，请参见[Abaqus Analysis User's Guide的第2.2.3节，"定义增强"](../usb/usb-link.md#usb-int-erebarlayer)。
