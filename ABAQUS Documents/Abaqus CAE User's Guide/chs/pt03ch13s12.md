# 13.12 使用查询工具集查询装配







从主菜单栏选择**工具**![](../graphics/images/arrow.gif)**查询**以启动查询工具集。您可以使用查询工具集请求常规信息或特定于模块的信息。关于常规查询显示的信息讨论，请参见"获取关于模型的一般信息"，第71.2.2节。此外，您可以使用查询工具集中的装配模块特定工具来确定所选零件或模型实例的属性和位置。

**查询装配：**

1. 从主菜单栏，选择**工具**![](../graphics/images/arrow.gif)**查询**。**提示：**您也可以在**查询**工具栏中选择![](../graphics/ico_info.png)工具。Abaqus/CAE显示**查询**对话框。
2. 从**查询**对话框中，单击以下选项之一：**实例属性**选择一个零件或模型实例。Abaqus/CAE在消息区域显示以下内容：- 实例的名称、建模空间和类型（可变形或刚性、从属或独立）**实例位置**选择一个零件或模型实例。Abaqus/CAE在消息区域显示以下内容：- 实例原点相对于全局坐标系的位置- 相对于装配全局坐标系应用于实例的旋转总和- 应用于实例的约束列表
3. 查询完成后关闭**查询**对话框。
