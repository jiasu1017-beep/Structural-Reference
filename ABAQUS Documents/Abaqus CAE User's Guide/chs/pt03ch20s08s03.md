# 20.8.3 通过选择边修改或复制对象







移动边时，Abaqus/CAE 重新定位所选边并使用**编辑期间**约束求解方法来修改任何连接的边（有关更多信息，请参见["在 Sketcher 中自定义约束的使用，"第 20.9.10 节](pt03ch20s09hlb10.md)）。草图约束和约束求解方法可能会限制您可以进行的移动类型，或导致 Abaqus/CAE 重新定位您未选择的边。复制边时，副本根据复制方法定位——复制的边不受任何现有草图约束的影响。您不能在草图之间复制对象。Abaqus/CAE 提供以下方法来移动或复制所选边：

**平移**

您可以通过指定平移向量的起点和终点坐标来平移所选边。平移向量可以从任何点开始；但是，您可能会发现从所选边一端的顶点开始并在其新位置结束更有意义。[图 20-12](pt03ch20s08s03.md#ske-tech-trans)显示了所选的边、平移向量和平移结果。为了实现所示的结果，添加了**固定**约束以防止上方矩形的两个顶点移动。

**图 20-12** 平移所选边。

![](../graphics/ske-tech-trans-nls.png)

**旋转**

您可以通过指定旋转中心的坐标并输入旋转角度来旋转所选边；正角度表示逆时针旋转。[图 20-13](pt03ch20s08s03.md#ske-tech-rotate)显示了所选边、所选旋转中心和 90 度旋转的结果。所选边未连接或以其他方式约束到草图中的其他边，因此约束求解方法对旋转没有影响。

**图 20-13** 旋转所选边。

![](../graphics/ske-tech-rotate-nls.png)

**镜像**

您可以通过选择草图中的一条直线作为镜像线来镜像所选边。[图 20-14](pt03ch20s08s03.md#ske-tech-mirror)显示了所选边、镜像线和结果副本，其中包括四个所选边中每个边的镜像约束。镜像线可以是草图中任何直线对象线或构造线。

**图 20-14** 镜像所选边。

![](../graphics/ske-tech-mirror.png)

**缩放**

您可以通过指定中心点和缩放因子来缩放所选边。小于 1.0 的缩放因子会减小边之间的间距，大于 1.0 的缩放因子会增加边之间的间距。[图 20-15](pt03ch20s08s03.md#ske-tech-scale)显示了所选边、所选中心点以及缩放因子为 1.5 时的结果。

**图 20-15** 缩放所选边。

![](../graphics/ske-tech-scale-nls.png)
