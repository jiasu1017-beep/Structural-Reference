# 47.5.10 Customizing the X–Y plot axis color and style







### 47.5.10 Customizing the X--Y plot axis color and style

Click the **Axes** tab to customize the color and thickness of the graph axes of an *X–Y* plot. Your selections apply to both the *X*- and *Y*-axes; it is not possible to customize the axes individually.

**To customize the color and style of an axis in an X–Y plot:**

1. Locate the **Style** options. Select ****Options**![](../graphics/images/arrow.gif)**XY Options**![](../graphics/images/arrow.gif)**Axis****; or click ![](../graphics/ico_xyAxisOptions.png), which is located with the *X--Y* plotting tools in the Visualization module toolbox. Click the **Axes** tab in the dialog box that appears. The **Style** options for the axes appear at the bottom of the **Axes** page.
2. From the **X Axis** or **Y Axis** fields, highlight one or more axes.
3. Choose the line style. From the **Style** list, click the desired line style (solid, dashed, etc.). The specified line style appears for the selected axis and in the **Style** list.
4. Choose the line thickness. From the **Thickness** list, click the desired line thickness. The specified line thickness appears for the selected axis and in the **Thickness** list.
5. Choose the line color: 1. Click the color sample ![](../graphics/usi-colorsample.png). Abaqus/CAE displays the **Select Color** dialog box. 2. Use one of the methods in the **Select Color** dialog box to select a new color. For more information, see ["Customizing colors," Section 3.2.9](pt01ch03s02s09.md). 3. Click **OK** to close the **Select Color** dialog box. The axis line changes to the selected color.
6. Click **Dismiss** to close the **Axis Options** dialog box.
