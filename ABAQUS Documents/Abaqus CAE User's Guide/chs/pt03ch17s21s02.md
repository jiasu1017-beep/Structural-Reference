# 17.21.2 为重新网格划分规则选择步骤和误差指示器输出变量






创建重新网格划分规则时，必须选择将应用该规则的步骤。您还必须选择分析过程中 Abaqus/CAE 将写入输出数据库的误差指示器输出变量，以及写入它们的频率。误差指示器输出变量是自适应重新网格划分过程的基础。它们向 Abaqus/CAE 提供信息，描述在哪里需要细化网格以接近或达到所需的误差指示器目标。此外，Abaqus/CAE 使用误差指示器输出变量来确定可以在哪里粗化网格而不会引入不可接受的误差。[「影响自适应重新网格划分的误差指示器选择，」Abaqus Analysis User's Guide 第 12.3.2 节](../usb/usb-link.md#usb-anl-aadperrorindicators)。

您可以使用自适应绘图仪插件来查看自适应分析期间所选误差指示器和单元计数的历史记录。有关更多信息，请参阅[「显示自适应重新网格划分误差指示器的历史，」第 82.7 节](pt08ch82s07.md)。

**为重新网格划分规则选择步骤和误差指示器输出变量：**

1. 从**创建重新网格划分规则**对话框中，单击**步骤和指示器**选项卡。
2. 单击**步骤**字段右侧的箭头，从出现的列表中选择您选择的步骤。重新网格划分规则将仅在此步骤期间应用；但是，您可以将具有相同设置的不同重新网格划分规则应用到模型中的另一个步骤。自适应重新网格划分仅适用于使用 Abaqus/Standard 的过程。此外，Abaqus/CAE 无法在频率提取和稳态动态过程中应用自适应重新网格划分。有关更多信息，请参阅[「哪些过程可以与自适应重新网格划分一起使用？，」第 17.13.3 节](pt03ch17s13s03.md)。
3. 选择误差指示器输出变量写入输出数据库的频率。您可以选择在每个增量之后写入变量，也可以在步骤的最后增量结束之后写入。Abaqus/CAE 根据步骤最后增量中的误差指示器值对模型进行重新网格划分。但是，如果您的分析无法收敛，并且您保存了每个增量之后的误差指示器输出变量，您可以使用最新的值手动对模型进行重新网格划分并继续。有关更多信息，请参阅[「手动调整大小和重新网格划分，」第 17.21.6 节](pt03ch17s21s06.md)。
4. 从**误差指示器变量**列表中，选择一个或多个将写入输出数据库的变量。对于您选择的每个变量，Abaqus/CAE 也会将一个伴随的基础解变量写入输出数据库。
