# 59.7.2 编辑基于边缘图案创建的附件点





您可以编辑基于边缘图案创建的附件点。

**编辑基于边缘图案创建的附件点的步骤：**

1. 从主菜单栏，选择 ****Feature**![](../graphics/images/arrow.gif)**Edit****。
   **提示：**您也可以通过单击模块工具箱中的 ![](../graphics/ico_featureEdit.png) 工具来编辑特征。

2. 选择要修改的附件点特征。您可以直接从当前视口或从模型树中选择特征。会出现**编辑从边缘偏移附件点特征**对话框。

3. 执行以下操作之一来指定沿边缘的点：
   - 选择 **by number** 以定义沿边缘选择的附件点数量。Abaqus/CAE 会自动沿边缘分布点。
     1. 输入沿边缘的点数。
     2. 输入从起始点的第一个附件点的所需偏移量。
     3. 输入从终点的最后一个附件点的所需偏移量。
   - 选择 **by spacing** 以定义附件点之间的距离。
     1. 输入点之间的间距。
     2. 输入从起始点的第一个附件点的所需偏移量。
     3. 指定沿边缘的点数，或选择 **Auto-fit** 以让 Abaqus/CAE 确定附件点的数量。如果您指定点数，Abaqus/CAE 将仅创建起点和终点之间的点；延伸到终点之外的任何点都不会被创建。

4. 如果需要，通过指定以下内容来定义附件点行的图案：
   - 输入图案中的行数。值为 1 会创建单行附件点。
   - 输入间距值以定义图案行之间的距离。
   - 输入图案第一行从边缘的偏移距离。值为 0 会在边缘上创建图案的第一行。

5. 如果需要，切换 **Regenerate on OK** 以控制在编辑过程完成时是否重新生成模型。
