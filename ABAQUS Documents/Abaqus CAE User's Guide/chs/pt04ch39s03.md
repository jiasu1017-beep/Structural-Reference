# 39.4 将子结构导入Abaqus/CAE




您可以通过将子结构作为新部件定义导入来在模型数据库中包含子结构定义并开始使用它们进行建模。子结构数据可在`.sim`文件中获得，子结构标识符包含在文件名中；例如，在子结构名为`FAN`且子结构标识符为`Z400`的分析中，子结构数据库文件名为`FAN_Z400.sim`。

您从中导入子结构的`.sim`文件必须位于`.sim`数据库所引用的支持Abaqus文件的同一目录中；这些支持文件可能包括`.prt`、`.mdl`、`.stt`或`.sup`格式的数据。

子结构导入还需要输出数据库(`.odb`)文件用于网格显示。
