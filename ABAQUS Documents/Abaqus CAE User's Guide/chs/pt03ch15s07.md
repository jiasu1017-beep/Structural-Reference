# 15.7 理解连接器





连接器允许您对装配中两点之间或装配中一点与地面之间的连接进行建模。要在Abaqus/CAE中对连接器进行建模，您必须创建一个装配级线特征、一个连接器截面，以及一个将连接器截面与所选线关联的连接器截面分配。

线特征包含一个或多个定义底层连接器几何的线。连接器截面指定连接类型、连接器行为和截面数据。类似于在「属性」模块中将截面分配给模型区域的方式，您创建一个连接器截面分配以将连接器截面分配给模型区域；具体来说，您将连接器截面分配给线。您还需要在连接器截面分配定义中为线的端点指定局部方向。

有关Abaqus/CAE中连接器的更多信息，包括连接器建模概述和示例，请参见第24章，「连接器」。
