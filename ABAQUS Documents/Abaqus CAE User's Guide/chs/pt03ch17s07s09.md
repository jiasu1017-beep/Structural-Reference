# 17.7.9 我是否必须一次性对整个模型进行网格划分？





Abaqus/CAE允许您以增量方式对模型进行网格划分，每次网格划分操作对模型的不同区域进行网格划分。您可以使用增量网格划分来微调模型中所选区域的网格，而无需重新对整个模型进行网格划分。

当您对选定区域进行网格划分时，Abaqus/CAE会在可能的情况下保留模型其他区域中的现有网格。但是，增量网格划分可能会迫使现有网格边界上的节点移动，并可能降低区域边界处的网格质量。在某些情况下，Abaqus/CAE无法继续增量网格划分操作，必须在继续之前删除所有现有网格：
- 如果现有网格与所选区域之间的布种无法被遵守，则无法继续增量网格划分。您必须允许Abaqus/CAE删除现有网格，并重新对原始区域和所选区域进行网格划分。例如，考虑图17-40中的零件实例。
**图17-40** 无法对中心区域进行增量网格划分。
![](../graphics/mgn-incremental-central.png)
中心区域无法进行增量网格划分，因为一端有4×4网格图案，另一端有3×3网格图案。如果您只尝试对中心区域进行网格划分，Abaqus/CAE将检测到问题，并允许您选择以下选项：
- 重新对已网格化的区域和中心区域进行网格划分，以生成兼容网格。
- 取消操作，不对中心区域进行网格划分。
- 如果现有网格需要从您尝试创建的网格中派生，则无法继续增量网格划分。例如，考虑图17-41中的零件实例。
**图17-41** 必须按正确顺序对区域进行网格划分。
![](../graphics/mgn-incremental-order-nls.png)
为了在区域1和区域2之间创建兼容网格，区域2的网格从区域1中圆柱体的网格派生。同样，区域3的网格从区域2派生，而区域2又从区域1中的圆柱体的网格派生。因此，如果您首先对区域3进行网格划分，Abaqus/CAE无法对区域1和2进行增量网格划分。您必须允许Abaqus/CAE在重新对已网格化的区域进行网格划分之前先对区域1进行网格划分。

如果增量网格划分无法继续，Abaqus/CAE会在删除现有网格之前显示警告消息。

如果您想增量对零件或装配进行网格划分，您可以遵循一种策略，以尽量减少Abaqus/CAE必须删除现有网格的次数。网格划分策略取决于区域的拓扑、单元形状、网格技术和网格布种。
- 布种的变化总是向外传播到边界。因此，您应该从零件或零件实例的内部开始网格划分，并继续到零件或实例的边界。
- 但是，如果您可以识别将使用扫掠方法进行网格划分的一组相邻三维区域，则应该从零件或零件实例的一侧开始网格划分，并通过内部继续到另一侧。
- 使用三角形或四面体单元进行网格划分的区域在增量网格划分期间永远不会强制删除整个网格。使用前进波算法使用四边形主导单元进行网格划分的区域也是如此。Abaqus/CAE始终可以重新对这些区域进行网格划分，您可以随时对它们进行网格划分。
