### 概述

Abaqus/Standard中的经典偏量金属蠕变行为：
- 可以使用用户子程序[`CREEP`](../sub/sub-link.md#sub-xsl-creep)定义，也可以通过提供一些简单蠕变定律的参数作为输入来定义；
- 可以模拟各向同性蠕变（使用Mises应力势）或各向异性蠕变（使用Hill各向异性应力势）；
- 仅在使用耦合温度-位移过程、瞬态土壤固结过程和准静态过程的步骤中激活；
- 要求将材料的弹性定义为线性弹性行为；
- 可以修改为实现核标准NEF 9-5T"一级高温核系统组件设计指南和程序"中规定的辅助蠕变硬化规则；这些规则通过橡树岭国家实验室开发的本构模型来执行（["ORNL -- 橡树岭国家实验室本构模型，" 第23.2.12节"](pt05ch23s02abm28.md)）；
- 可与蠕变应变率控制结合使用，在分析中必须将蠕变应变率保持在一定范围内；和
- 如果各向异性蠕变和塑性同时发生，可能导致计算蠕变应变错误（见下文讨论）。

Abaqus/Standard中率相关垫片行为：
- 将单向蠕变用作垫片厚度方向行为模型的一部分；
- 可以使用用户子程序[`CREEP`](../sub/sub-link.md#sub-xsl-creep)定义，也可以通过提供一些简单蠕变定律的参数作为输入来定义；
- 仅在使用准静态过程的步骤中激活；和
- 要求使用弹塑性模型来定义垫片厚度方向行为的率无关部分。

Abaqus/Standard中的体积肿胀行为：
- 可以使用用户子程序[`CREEP`](../sub/sub-link.md#sub-xsl-creep)定义或提供表格输入；
- 可以是各向同性或各向异性的；
- 仅在使用耦合温度-位移过程、瞬态土壤固结过程和准静态过程的步骤中激活；和
- 要求将材料的弹性定义为线性弹性行为。

### 蠕变行为

蠕变行为由等效单轴行为（蠕变"定律"）指定。在实际案例中，蠕变定律通常非常复杂以拟合实验数据；因此，这些定律使用用户子程序[`CREEP`](../sub/sub-link.md#sub-xsl-creep)定义，如下所述。或者，Abaqus/Standard中提供了五种常见蠕变定律：幂律、双曲正弦律、双幂律、Anand定律和Darveaux定律。这些标准蠕变定律用于模拟二次或稳态蠕变。蠕变通过在材料模型定义中包含蠕变行为来定义（["材料数据定义，" 第21.1.2节"](pt05ch21s01aus109.md)）。或者，蠕变可以与垫片行为结合定义，以定义垫片的率相关行为。

#### 选择蠕变模型

幂律蠕变模型因其简单性而吸引人。然而，它的应用范围有限。幂律蠕变模型的 time-hardening版本通常仅在应力状态基本保持恒定的情况下推荐使用。幂律蠕变的strain-hardening版本应在分析期间应力状态发生变化时使用。在应力恒定且没有温度和/或场依赖性的情况下，幂律的 time-hardening和strain-hardening版本是等效的。对于任一版本的幂律，应力应该相对较低。

在高应力区域（如裂纹尖端附近），蠕变应变率通常表现出与应力的指数依赖性。双曲正弦蠕变定律在 高应力水平(σ¯/σ~0~ > 1.5，其中σ~0~是屈服应力)下表现出与应力σ¯的指数依赖性，在低应力水平下简化为幂律（没有明确的时间依赖性）。

双幂律、Anand和Darveaux模型特别适合模拟电子封装中使用的焊料合金的行为，已被证明可在各种温度和应变率下产生准确的结果。

上述模型都不适合模拟循环加载下的蠕变。ORNL模型（["ORNL -- 橡树岭国家实验室本构模型，" 第23.2.12节"](pt05ch23s02abm28.md)）是不锈钢的经验模型，对循环加载给出近似结果，而无需数值执行循环加载。通常，循环加载的蠕变模型很复杂，必须使用用户子程序[`CREEP`](../sub/sub-link.md#sub-xsl-creep)或用户子程序[`UMAT`](../sub/sub-link.md#sub-xsl-umat)添加到模型中。

#### 同时模拟蠕变和塑性

如果蠕变和塑性同时发生且隐式蠕变积分生效，两种行为可能相互作用，需要求解耦合的本构方程组。如果蠕变和塑性是各向同性的，Abaqus/Standard正确地考虑这种耦合行为，即使弹性是各向异性的。然而，如果蠕变和塑性是各向异性的，Abaqus/Standard在积分蠕变方程时不考虑塑性，这可能导致蠕变应变的重大错误。这种情况仅在塑性 和蠕变同时激活时发生，例如在长期加载增加期间发生；在短期预加载阶段以塑性为主，然后是不再发生屈服的蠕变阶段的蠕变阶段，不会有问题。蠕变定律和率相关塑性的积分在["率相关金属塑性（蠕变），" Abaqus理论指南第4.3.4节](../stm/stm-link.md#stm-mat-ratedepplast)中讨论。

#### 幂律模型

幂律模型可以以其"time hardening"形式或相应的"strain hardening"形式使用。

##### 时间硬化形式

"time hardening"形式是幂律模型两种形式中较简单的一种：

ε¯˙^c^ = A (σ¯)^n t^m

其中ε¯˙^c^是单轴等效蠕变应变率，σ¯是单轴等效偏量应力，t是总时间或蠕变时间，A、n和m由您定义为温度的函数定义。

σ¯是根据是否定义各向同性或各向异性蠕变行为（见下文）的Mises等效应力或Hill各向异性等效偏量应力。对于物理上合理的行为，A和n必须为正，且m ≥ 0。

##### 数值困难

根据幂律形式单位的选择，A的值对于典型蠕变应变率可能非常小。如果A小于10^27，数值困难可能导致材料计算中的错误；因此，使用另一单位系统来避免蠕变应变增量计算中的此类困难。

#### 双曲正弦定律模型

双曲正弦定律采用以下形式：

ε¯˙^c^ = A [sinh(B σ¯)]^n exp(-Q/RT)

其中σ¯和σ¯如上定义，θ是温度，θ~0~是所用温度标尺上用户定义的绝对零度值，Q是活化能，R是通用气体常数，A、B和n是其他材料参数。

该模型包含温度依赖性，这在上述表达式中是明显的；然而，参数A、B、n、Q和R不能定义为温度的函数。

| --- | --- |
|  | ``` [*CREEP](../key/key-link.md#usb-kws-mcreep), LAW=HYPERB [*PHYSICAL CONSTANTS](../key/key-link.md#usb-kws-mphysicalconsts), ABSOLUTE ZERO=θ~0~ ``` |

| **Abaqus/CAE用法：** | 定义以下两项： |
| --- | --- |
|  | 属性模块：材料编辑器：****机械**![](../graphics/images/arrow.gif)**塑性**![](../graphics/images/arrow.gif)**蠕变****：****定律：**双曲正弦**** 任何模块：****模型**![](../graphics/images/arrow.gif)**编辑属性**![](../graphics/images/arrow.gif)***模型名称***：****绝对零温度**** |

##### 数值困难

与幂律一样，A对于典型蠕变应变率可能非常小。如果A非常小（例如小于10^27），使用另一单位系统来避免蠕变应变增量计算中的数值困难。

#### Anand模型

Anand模型采用以下形式：

ε¯˙^c^ = A exp(-Q/RT) [sinh(B σ¯)]^n

其中σ¯、σ¯、R、θ和θ~0~如上定义，Q是活化能，s是变形阻力，A~0~、n和m是材料参数。

变形阻力的演化方程，s（初始为s~0~）是：

d s/dt = h (1 - s/s^*^) |ε¯˙^c|^a * sign(σ¯ - s)

其中：

s^*^ = s~ref~ [ε¯˙^c^/A exp(-Q/RT)]^n

且s~ref~、h~0~、a、m、s~0~、Q/R和n是材料参数。

此外，初始变形阻力是温度的函数，形式为：

s~0~ = s~0~ref~ exp(-Q/Rθ)

其中s~0~ref~、Q/R和n是材料参数。

| --- | --- |
|  | ``` [*CREEP](../key/key-link.md#usb-kws-mcreep), LAW=ANAND [*PHYSICAL CONSTANTS](../key/key-link.md#usb-kws-mphysicalconsts), ABSOLUTE ZERO=θ~0~ ``` |

| **Abaqus/CAE用法：** | Abaqus/CAE不支持指定Anand定律。 |
| --- | --- |

#### Darveaux模型

Darveaux模型涉及初级和次级蠕变。次级蠕变（稳态）分量用标准双曲正弦定律描述：

ε¯˙^c^ = K~0~ [sinh(B σ¯)]^n

稳态定律通过以下方式修改以包含初级蠕变效应：

ε¯˙^c^ = K~0~ [sinh(B σ¯)]^n / (1 + K~1~ (ε¯^c^)^K~2^)

其中σ¯、σ¯、R、Q、θ和θ~0~如上定义，K~0~是稳态蠕变前因子，K~1~是稳态蠕变幂律 breakdown，m、K~1~和K~2~是其他材料参数。

| **Abaqus/CAE用法：** | Abaqus/CAE不支持指定Darveaux定律。 |
| --- | --- |

### 积分

显式积分、隐式积分或两种积分方案可用于蠕变分析，这取决于所使用的过程、为过程指定的参数、塑性的存在以及是否请求几何非线性。

#### 显式和隐式方案的应用

非线性蠕变问题通常通过非弹性应变的向前差分积分（"初始应变"方法）有效地求解。这种显式方法计算效率高，因为与隐式方法不同，不需要迭代。虽然该方法仅条件稳定，但显式算子的数值稳定性限制通常足够大，以允许在少量时间增量中发展解。

Abaqus/Standard使用显式或隐式积分方案，或在同一步骤中从显式切换到隐式。这些方案首先概述，然后描述哪些过程使用这些积分方案。

1. 积分方案1：以显式积分开始，基于稳定性或塑性是否激活切换到隐式积分。显式积分中使用的稳定性限制在下一节中讨论。
2. 积分方案2：以显式积分开始，当塑性激活时切换到隐式积分。稳定性准则在此不起作用。
3. 积分方案3：始终使用隐式积分。

上述积分方案的使用由过程类型、您选择的要使用的积分类型以及是否请求几何非线性决定。对于准静态和耦合温度-位移过程，如果您不选择积分类型，几何线性分析使用积分方案1，几何非线性分析使用积分方案3。您可以强制Abaqus/Standard在耦合温度-位移或准静态过程中的蠕变和肿胀效应使用显式积分，当塑性在整个步骤中不激活时（积分方案2）。无论是否请求几何非线性（见["通用和线性扰动过程，" 第6.1.3节"](pt03ch06s01aus44.md)），都可以使用显式积分。

对于瞬态土壤固结过程，无论执行几何线性还是几何非线性分析，始终使用隐式积分方案（积分方案3）。

#### 显式积分期间稳定 性限制的自动监测

Abaqus/Standard在显式积分期间自动监测稳定性限制。如果在模型中的任何点，蠕变应变增量Δε¯^c^大于总弹性应变，问题将变得不稳定。因此，每增量计算一个稳定时间步长Δt~s~：

Δt~s~ = (2 ε¯^e^~t~) / (3 ε¯˙^c^~t~)

其中ε¯^e^~t~是时间t（增量开始）处的等效总弹性应变，ε¯˙^c^~t~是时间t处的等效蠕变应变率。此外，

σ¯~t~ = √(3/2 S~ij~ S~ij~)~t~

其中S~ij~是时间t处的偏应力，且

E~eff~ = √(E × E~e~) 或 √(E/(3(1-2ν))

其中E~e~ = (3/2)(E/(1+ν))是偏量弹性模量，D~ijkl~是弹性矩阵，E~eff~是有效弹性模量——对于各向同性弹性，E~eff~可以近似为杨氏模量。

对于执行显式积分的每个增量，稳定时间增量Δt~s~与临界时间增量Δt~crit~比较，如下计算：

Δt~crit~ = errtol / (dσ¯/dt)~max~

量errtol是您定义的错误容忍度，如下所述。如果Δt~s~小于Δt~crit~，则使用Δt~s~作为时间增量，这意味着稳定性准则比精度要求更严格地限制了时间步长。如果Δt~s~连续九个增量小于Δt~crit~，您没有如上所述限制Abaqus/Standard使用显式积分，且分析中有足够的时间剩余（时间剩余 > Δt~s~），Abaqus/Standard将自动切换到向后差分算子（隐式方法，无条件稳定）。如果使用隐式算法，将在每次迭代时重构刚度矩阵。

#### 指定自动增量的容忍度

必须选择积分容忍度以准确计算应力增量Δσ。考虑一个一维示例。应力增量Δσ是：

Δσ = E (Δε - Δε^c^) = E Δε - σ Δt

其中Δε^e^、Δε和Δε^c^分别是单轴弹性、总和蠕变应变增量，E是弹性模量。为了准确计算Δε^c^，蠕变应变增量中的误差Δε^c^ - (Δε^c^)^true^必须与Δε^e^相比很小；即：

|error| ≤ errtol

将Δε^c^中的误差测量为error = (Δε^c^ - (Δε^c^)^true^)/Δε^e^，导致：

errtol ≤ 1 - (Δε - σ Δt/E) / Δε^e^

errtol ≤ 1 - Δε/Δε^e^ + σ Δt/(E Δε^e^)

errtol ≤ 1 - σ Δt/(E Δε^e^) - Δε^c^/Δε^e^

errtol ≤ 1 - Δε/Δε^e^

errtol ≤ Δε^e^/Δε

您为适用过程定义errtol，方法是选择可接受的应力误差容忍度并将其除以典型弹性模量；因此，它应该是典型应力与问题中有效弹性模量之比的一小部分。重要的是要认识到，选择errtol值的这种方法通常是相当保守的，通常可以获得具有更高值的可接受解决方案。

### 使用蠕变应变率控制加载

在超塑性成形中，施可控压力使物体变形。超塑性材料可以变形到非常大的应变，前提是变形的应变率保持在非常严格的容差范围内。超塑性分析的目的是预测如何控制压力以尽可能快地成形组件，同时不超过材料中任何地方的超塑性应变率。

要在Abaqus/Standard中使用此功能，控制算法如下。在增量期间，Abaqus/Standard计算R~max~，即指定单元集中任何积分点上等效蠕变应变率与目标蠕变应变率之比的最大值。如果给定增量中R~max~小于0.2或大于3.0，则放弃该增量，并使用以下负载修改重新启动：

p~new~ = 0.9 p~old~

其中p是新负载幅值，p~old~是旧负载幅值。如果R~max~ ≤ 0.2，增量被接受；并且在下一时间增量开始时，负载幅值修改如下：

p~new~ = p~old~ (3.0/R~max~)^1/2^

当您激活上述算法时，蠕变和/或肿胀问题中的加载可以基于在定义单元集中找到的最大等效蠕变应变率来控制。作为最低要求，此方法用于定义目标等效蠕变应变率；但是，如果需要，也可以用作等效蠕变应变（测量为对数应变）、温度和其他预定义场变量的函数定义目标蠕变应变率。每个温度下的蠕变应变依赖曲线必须始终从零等效蠕变应变开始。

使用解相关振幅来定义加载的最小和最大限制（见["在"振幅曲线，" 第34.1.2节中定义用于超塑性成形分析的解相关振幅""](pt07ch34s01aus115.md#usb-prc-pamplitude-data-sol)）。可以使用任意数量或组合的负载。R~max~的当前值可供输出，如下所述。
