# 2.1.9 带通用接触的膝部加强件冲击





**产品：** Abaqus/Explicit   

本示例说明了在涉及潜在接触表面之间大相对运动的模拟中使用通用接触能力。

### 问题描述

该模型代表了汽车膝部加强件总成——在碰撞中乘客腿部撞击的仪表板部分。该总成由硬塑料盖（膝部加强件）和刚性钢子结构支撑。正确的设计确保乘客能量以最小的伤害力消散。在该模拟中，腿部以6 m/s的速度接近膝部加强件，代表了从15 mph到完全停止的碰撞事件后的无约束运动。

仪表板组件使用S3R和S4R壳单元建模。加强件由2690个壳单元组成，材料建模为von Mises弹塑性应变硬化塑料，杨氏模量为2.346 GPa，泊松比为0.4，密度为1140 kg/m³，屈服应力为11.7 MPa。钢子结构由1648个单元组成，材料建模为应变硬化钢，杨氏模量为207 GPa，泊松比为0.3，密度为7700 kg/m³，屈服应力为207 MPa。图2.1.9-1显示了撞击前从膝部加强件后部看的模型几何形状，图2.1.9-2显示了撞击前从驾驶员外侧和后方看的膝部加强件和膝部/腿部总成。

腿部表示为具有周围刚性表面的结构构件。代表骨骼的结构构件使用B31梁单元和T3D2桁架单元建模，材料建模为弹性，杨氏模量为207 GPa，泊松比为0.3，密度为7.7 kg/m³。代表膝盖和小腿的刚性表面使用R3D4刚性单元建模。身体质量通过在结构单元节点上的各个位置分布质量单元来建模。

在输入文件[knee_bolster_nsm.inp](../eif/knee_bolster_nsm.inp)中，上身质量作为非结构质量分布在髋骨（B31梁单元）上，而不是点质量。单元的非结构质量贡献增加了单元质量和单元旋转惯性，从而导致稳定时间增量增加。当使用非结构质量功能时，此分析以大约少40%的增量完成，因为问题的时间增量由梁单元控制。在输入文件[knee_bolster_massadjust.inp](../eif/knee_bolster_massadjust.inp)中，上身质量使用质量调整分布在髋骨上，而不是点质量。质量分布和分析结果与使用非结构质量的结果相似。包括了使用管道单元（PIPE31）代替梁单元的类似分析的输入文件。

仪表板总成组件之间以及仪表板和腿部之间的潜在接触使用通用接触能力建模。通用接触包含选项用于自动定义全包容表面，这是定义模型接触的最简单方法。此外，提供了使用替代接触对算法的模型；使用接触对算法定义接触更繁琐。

在腿部组件上定义初始速度以近似15 mph（6 m/s）的碰撞条件。臀部被约束为在座椅平面内平移。脚踝被约束与脚固定在汽车地板上一致。仪表板子结构在会焊接到汽车车架的位置固定；假设此冲击引起的变形仅限于明确建模的结构。

### 结果与讨论

图2.1.9-3显示了100.0 ms后加强件总成的变形形状。

图2.1.9-4显示了整个模型能量随时间的变化：内能、动能、可恢复应变能和塑性耗散。此图显示几乎所有身体初始动能在此模拟结束时已被传递。在被传递的能量中，很少量被转移到仪表板结构和骨骼中的弹性变形，其余部分在塑性耗散中损失。

图2.1.9-5显示了相对于侵入加强件位移测量的总膝盖和小腿接触力（用SAE 600滤波器过滤）。与能量观察一致，碰撞事件明显已完成。使用管道单元获得的解与使用梁单元获得的解一致。
