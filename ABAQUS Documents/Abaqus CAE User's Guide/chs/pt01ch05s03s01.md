# 5.3.2 使用 3D 罗盘平移视图





3D 罗盘允许你使用两种不同方法平移模型视图：你可以沿指定轴平移，或者你可以在指定平面内平移。

使用 3D 罗盘执行的平移操作与使用标准平移视图工具执行的操作不同（请参阅["平移视图工具，" 第 5.2.2 节](pt01ch05s02s02.md)）。标准平移工具在模型前方横向移动相机；相机只能在与视口平行的平面内移动。3D 罗盘的平移平面不一定与视口平行。结果是，使用罗盘平移的行为类似于标准平移和缩放工具的替代模式的组合：相机既横向穿过模型移动，也垂直于模型移近或移远。在平移操作期间，罗盘和模型视图的方向不会改变。

**注意：**与标准平移视图工具的替代模式一样，使用 3D 罗盘平移视图也会将视口的旋转中心与相机一起平移。关于旋转中心的信息，请参阅["旋转视图工具，" 第 5.2.3 节](pt01ch05s02s03.md)。

**沿轴平移**

要沿轴平移视图，请点击并拖动 3D 罗盘上的任意直轴：

![](../graphics/usi-compass-axis-nls.png)

当你拖动鼠标时，视口平面中指针的路径被投影到所选的罗盘轴上（上例中的 *Z* 轴）。相机根据此投影的线性路径移动。

**沿平面平移**

要沿平面平移视图，请点击并拖动 3D 罗盘上的任意四分之一圆面：

![](../graphics/usi-compass-plane-nls.png)

当你拖动鼠标时，视口平面中指针的路径被投影到所选的罗盘平面上（上例中的 *Y*–*Z* 平面）。相机根据此投影的路径移动。
