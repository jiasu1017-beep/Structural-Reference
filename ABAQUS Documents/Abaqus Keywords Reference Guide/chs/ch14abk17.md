# *NSET









### *NSET将节点分配到节点集。

此选项将节点分配到节点集。

**产品：**Abaqus/Standard  Abaqus/Explicit  Abaqus/CFD  Abaqus/CAE  

**类型：**模型或历史数据  

**级别：**部件、部件实例、装配、模型、步骤

**Abaqus/CAE：**集合工具集

##### **参考：**

- ["节点定义，" Abaqus Analysis User's Guide第2.1.1节](../usb/usb-link.md#usb-int-inode)

### **必需参数：**

NSET

将此参数设置为其节点将被分配到的节点集名称。

### **可选参数：**

ELSET

将此参数设置为先前定义的元素集名称。属于此元素集的元素（在遇到此选项时属于该元素集）定义的节点将被分配到指定的节点集。UNSORTED参数不能与此参数一起使用。

ELSET和GENERATE参数是互斥的。

GENERATE

如果包含此参数，则每个数据行应给出一个起始节点、一个终止节点以及这些节点之间节点编号的增量。然后将从到的所有节点（以的步长）添加到集合中。必须是使得为整数的整数。

ELSET和GENERATE参数是互斥的。

INSTANCE

将此参数设置为包含数据行上列出的节点的部件实例名称。此参数只能在装配级别使用，旨在用作命名约定的快捷方式。它只能用于基于部件实例装配定义的模型。

INTERNAL

Abaqus/CAE使用INTERNAL参数来标识内部创建的集合。INTERNAL参数仅用于基于部件实例装配定义的模型。默认是省略INTERNAL参数。

UNSORTED

如果包含此参数，则此节点集中的节点将按其给出顺序分配到集合中（或如果已存在则添加到集合中）。如果使用了ELSET参数，则此参数将被忽略。

如果省略此参数，则集合中的节点按其节点编号升序排序，并消除重复。

### **如果省略GENERATE参数时的数据行：**

**第一行：**

根据需要重复此数据行。每行最多允许16个条目。

### **如果包含GENERATE参数时的数据行：**

**第一行：**

根据需要重复此数据行。
