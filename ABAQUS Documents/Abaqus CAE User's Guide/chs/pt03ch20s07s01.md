# 20.7.1 使用约束控制草图几何







约束创建逻辑关系来控制几何的位置或大小。约束的定义不使用数值。另一类几何控制——尺寸和参数——是使用值或计算表达式来定义草图中数值关系的专门约束。Abaqus/CAE 草图中不需要约束；但是，它们可以帮助您保留设计意图。当您修改受约束的草图时，您必须要么在现有控件内工作，要么修改它们，或删除它们以完成更改。

[表 20-5](pt03ch20s07s01.md#skt-constraints)列出了每种约束类型及其符号和简要使用说明。

**表 20-5** Sketcher 约束。
| 约束 | 符号 | 描述 |
| --- | --- | --- |
| 重合 | o | Abaqus/CAE 移动所选实体，使它们共享一个公共点。该点可能是两个实体上的点，也可能是仅在其中一个实体延伸时才存在的投影点。如果选择两条直线，它们将变为共线。 |
| 同心 | ![](../graphics/skt_concentric.png) | Abaqus/CAE 移动所选圆弧或圆以共享一个公共中心。 |
| 等长 | / | Abaqus/CAE 使所选直线具有相同的长度。 |
| 等半径 | --- | Abaqus/CAE 使所选圆弧或圆具有相同的半径。 |
| 固定 | ![](../graphics/skt_fixed.png) | Abaqus/CAE 固定圆弧或圆的半径、直线的角度或顶点的位置。 |
| 水平 | H | Abaqus/CAE 将所选直线与草图的 *X* 轴对齐。水平约束不受网格方向影响。 |
| 等距 | |![](../graphics/usi_eqn00693.gif)| | Abaqus/CAE 移动所选顶点，使其与用于定义约束的两个顶点或直线保持相同距离。 |
| 平行 | // | Abaqus/CAE 使所选直线平行。 |
| 垂直 | ![](../graphics/ske-square.png) | Abaqus/CAE 使所选直线垂直。 |
| 对称 | ![](../graphics/usi_eqn00693.gif)|![](../graphics/usi_eqn00693.gif) | Abaqus/CAE 使所选实体相对于您选择的对称线对称。 |
| 相切 | || | Abaqus/CAE 使所选实体在其最近点相切。至少其中一个实体必须是曲线。 |
| 垂直 | V | Abaqus/CAE 将所选直线与草图的 *Y* 轴对齐。水平约束不受网格方向影响。 |

**注意：**[表 20-5](pt03ch20s07s01.md#skt-constraints)中描述的行为假设没有其他约束应用于所选实体。其他约束的存在以及**Sketcher 选项**中选择的约束求解方法可能会影响 Abaqus/CAE 移动实体以符合约束的方式。

您可以在草图中创建同一类型约束的多个实例。您可以使用草图模块查询来定位约束控制的几何或列出整个草图的约束详细信息（有关更多信息，请参见[第 71 章，"查询工具集](pt06ch71.md)）。

您可以使用以下技术向草图添加约束：

**预选几何**

如果预选处于活动状态，Abaqus/CAE 可以在您创建几何时添加一些约束。重合、水平、垂直、相切和垂直约束可以使用预选光标添加。有关更多信息，请参见["Sketcher 光标和预选，"第 20.4.5 节](pt03ch20s04s05.md)。

**选择直线或曲线的中点**

如果您选择直线或曲线的中点作为附加草图几何的顶点，Abaqus/CAE 还会在该中点和直线或曲线端点之间创建等距约束。

**自动添加约束**

创建草图后，您可以使用自动约束工具![](../graphics/ico_sketchConstraintAuto.png)向整个草图或选定的实体组添加约束。使用草图选项控制 Abaqus/CAE 可以自动添加的约束类型以及添加约束的允许容差。有关更多信息，请参见["在 Sketcher 中自定义约束的使用，"第 20.9.10 节](pt03ch20s09hlb10.md)。

**添加单个约束**

创建草图几何后，您可以选择约束类型并选择要应用约束的实体。这种方法给您最大的控制权，但也最耗时。有关更多信息，请参见["添加单个约束，"第 20.12.3 节](pt03ch20s12hlb03.md)。

添加约束时，不得使草图几何过度约束。当多个控件应用于单个自由度时，几何是过度约束的（有关更多信息，请参见["完全约束几何，"第 20.7.4 节](pt03ch20s07s04.md)）。Abaqus/CAE 将冲突约束显示为洋红色以指示过度约束条件。在使用草图创建特征或将其与模型数据库一起保存之前，必须解决任何过度约束问题。
