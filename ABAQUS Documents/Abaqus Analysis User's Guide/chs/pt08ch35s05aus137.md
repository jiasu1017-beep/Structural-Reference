# 35.5.1 单元端点释放







**产品：** Abaqus/Standard

##### **参考文献**

- ["运动约束：概述，" 第35.1.1节](pt08ch35s01abo32.md)
- [*RELEASE](../key/key-link.md#usb-kws-mrelease)

### 概述

单元端点释放：
- 允许在一个或两个单元或单元集的端部释放一个旋转自由度或多个旋转自由度的组合；
- 可用于几何线性或非线性分析；以及
- 仅适用于Abaqus/Standard中的梁单元和管道单元。

### 引言

单元端点释放用于模拟单元一端或两端的铰接连接（沿一个、两个或三个正交方向铰接）。通过释放旋转自由度，单元端部可以相对于节点沿选定的自由度自由旋转。任何未被释放的旋转自由度将与节点共享。您必须小心，不要在共享某个节点的所有单元的该节点上释放给定的自由度；否则，该节点在该自由度上没有刚度，Abaqus/Standard会发出零主元警告消息。

单元端点释放作用于单元局部自由度。有关梁类型单元的局部轴（![](../graphics/usb_eqn06151.gif)、![](../graphics/usb_eqn06152.gif)、*t*）的定义，请参见["梁单元横截面方向，" 第29.3.4节](pt06ch29s03alm09.md)。释放影响的旋转自由度为：空间梁关于局部![](../graphics/usb_eqn06151.gif)轴的旋转、关于局部![](../graphics/usb_eqn06152.gif)轴的旋转，以及关于局部*t*轴的旋转。对于平面梁，只有关于局部![](../graphics/usb_eqn06151.gif)轴的旋转是活动的（与关于负全局*z*轴的旋转重合）。

#### 等效MPC

如果只释放一个旋转自由度，则运动约束等效于MPC类型REVOLUTE加上MPC类型PIN在两个节点之间。如果释放两个旋转自由度，则等效于MPC类型UNIVERSAL加上MPC类型PIN。如果释放所有旋转自由度，则等效于MPC类型PIN。有关详细信息，请参见["通用多点约束，" 第35.2.2节](pt08ch35s02aus130.md)。

### 确定释放所涉及的单元端部

可以为释放定义指定单元集或单个单元。可以在单元的第一端、第二端或第一和第二端释放自由度。单元的第一端S1是单元定义中定义的节点1；第二端S2是最后的节点（节点2或3，视情况而定）。有关梁单元的节点排序定义，请参见["梁单元库，" 第29.3.8节](pt06ch29s03ael14.md)。

### 确定释放所涉及的局部旋转自由度

使用旋转组合代码（而不是自由度）来指定释放所涉及的旋转自由度。

| M1 | 指关于![](../graphics/usb_eqn06151.gif)轴的旋转， |
| --- | --- |
| M2 | 指关于![](../graphics/usb_eqn06152.gif)轴的旋转， |
| M1-M2 | 指关于![](../graphics/usb_eqn06151.gif)轴和![](../graphics/usb_eqn06152.gif)轴的旋转自由度的组合， |
| T | 指关于*t*轴的旋转， |
| M1-T | 指关于![](../graphics/usb_eqn06151.gif)轴和*t*轴的旋转自由度的组合， |
| M2-T | 指关于![](../graphics/usb_eqn06152.gif)轴和*t*轴的旋转自由度的组合，以及 |
| ALLM | 代表所有旋转自由度的组合（即M1、M2和T）。 |

| **输入文件用法：** | ``` [*RELEASE](../key/key-link.md#usb-kws-mrelease) *单元编号或单元集, 单元端部ID, 释放组合代码* ``` |
| --- | --- |
|  | 例如，要在单元10的第一端释放关于![](../graphics/usb_eqn06151.gif)轴的旋转自由度，并在单元的第二端释放所有旋转自由度，请使用以下输入： ``` [*RELEASE](../key/key-link.md#usb-kws-mrelease) 10, S1, M1 10, S2, ALLM ``` |

### 与转换坐标系统一起使用

应用于释放节点的转换（["转换坐标系统，" 第2.1.5节"](pt01ch02s01aus09.md)）对释放没有影响。释放作用于单元的局部自由度。

### 从备用输入文件读取数据

释放定义的数据可以包含在单独的文件中。

| **输入文件用法：** | ``` [*RELEASE](../key/key-link.md#usb-kws-mrelease), INPUT=*file_name* ``` |
| --- | --- |
|  | 如果省略INPUT参数，则假定数据行跟随在关键字行之后。 |
