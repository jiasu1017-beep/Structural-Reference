# 3.4.10 使用管理器菜单管理对象







与管理器一样，主菜单栏中的下拉菜单允许您管理当前模型或会话中定义的所有对象。

**使用菜单管理对象：**

1. 从主菜单栏，选择以下之一：
   - 要管理与模块关联的对象，请在主菜单栏的相应菜单中选择管理器菜单项。例如，要在 Property 模块中编辑材料，您可以从主菜单栏选择****材料**![](../graphics/images/arrow.gif)**编辑**![](../graphics/images/arrow.gif)***您选择的材料*****。
   - 要管理与工具集关联的对象，请在**工具**菜单中选择相应的管理器菜单项。例如，要删除集合，您可以从主菜单栏选择****工具**![](../graphics/images/arrow.gif)**Set**![](../graphics/images/arrow.gif)**删除**![](../graphics/images/arrow.gif)***您选择的集合*****。
   - 要管理当前模型数据库中定义的所有模型，请在主菜单栏的**模型**菜单中选择管理器菜单项。例如，要复制模型，您可以从主菜单栏选择****模型**![](../graphics/images/arrow.gif)**复制**![](../graphics/images/arrow.gif)***您选择的模型*****。
   
   在大多数情况下，会出现一个对话框；例如，当您重命名对象时，会出现一个对话框，询问对象的新名称。

2. 如果出现对话框，请提供请求的信息并点击**确定**。

**提示：**您也可以使用管理器对话框来管理对象。有关更多信息，请参阅[「使用管理器对话框管理对象，第 3.4.9 节」](pt01ch03s04hlb01.md)。
