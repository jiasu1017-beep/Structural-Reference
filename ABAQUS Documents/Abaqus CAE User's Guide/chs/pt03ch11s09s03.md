# 11.9.3 线框特征





线框在Abaqus/CAE中表示为线条，用于对厚度和深度相对于长度都很小的实体进行理想化。要创建线框特征，请从主菜单栏选择**形状**![](../graphics/images/arrow.gif)**线框**，或从零件模块工具箱中选择一个线框工具。您可以在线框模块中使用线框工具执行以下操作之一来创建线框特征：
- 在选定的平面或基准平面上绘制线框以创建草图线框特征。- 用直线或样条曲线连接两个或多个点。

您可以使用线框工具向任何可变形或离散刚体零件添加线框特征。您不能向分析刚体零件添加线框特征；只能修改定义该零件的原始草图。

您使用属性模块创建指定横截面几何的截面，并将该截面分配给线框特征。您可以使用Abaqus/Standard或Abaqus/Explicit中可用的任何梁、桁架或轴对称壳单元对线框特征进行建模。

**注意：**虽然您可以创建梁单元网格，但当前版本的Abaqus/CAE只允许您为线框分配以下截面：
- 梁截面
- 桁架截面
