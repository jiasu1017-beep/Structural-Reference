# 15.14.2 定义薄膜条件相互作用属性





您可以将薄膜系数定义为温度和场变量的函数。薄膜条件相互作用属性只能由表面薄膜条件相互作用或集中薄膜条件相互作用引用。有关更多信息，请参见"定义表面薄膜条件相互作用"，第15.13.22节，和"定义集中薄膜条件相互作用"，第15.13.23节。

**定义薄膜条件相互作用属性的步骤：**

1. 从主菜单栏选择 ****Interaction**![](../graphics/images/arrow.gif)**Property**![](../graphics/images/arrow.gif)**Create****。
2. 在出现的「Create Interaction Property」对话框中，执行以下操作：- 为相互作用属性命名。有关命名对象的更多信息，请参见"使用基本对话框组件"，第3.2.1节。- 选择「Film condition」类型的相互作用属性。
3. 点击「Continue」关闭「Create Interaction Property」对话框。
4. 切换「Use temperature-dependent data」以定义随温度变化的薄膜系数。数据表中会出现一列标为「Temp」。
5. 要定义依赖于场变量的薄膜系数，点击「Number of field variables」字段右侧的箭头以增加或减少场变量的数量。场变量列会出现在数据表中。
6. 在数据表中，输入薄膜系数*h*（单位为[JT1L2![](../graphics/usi_eqn00605.gif)1](../popups/usb-int-iconventions-unitsym.md)），作为温度和场变量的函数。您可以使用键盘输入数据。或者，您可以在表格中任意位置点击鼠标按钮3，以查看用于指定表格数据的选项列表。有关每个选项的详细信息，请参见"输入表格数据"，第3.2.7节。
7. 点击「OK」创建薄膜条件相互作用属性并退出编辑器。
