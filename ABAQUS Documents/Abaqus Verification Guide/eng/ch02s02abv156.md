# 2.2.18 Drucker-Prager plasticity







**Product: **Abaqus/Explicit  

### Elements tested

C3D8R    CPE4R    CPS4R    

### Feature tested

Extended Drucker-Prager plasticity with third stress invariant.

### Problem description

This problem contains 16 one-element verification problems that are all run in one input file. The problem exercises the extended Drucker-Prager plasticity material model using associated and nonassociated flow rules. Three different element types are tested (C3D8R, CPE4R, CPS4R). [Figure 2.2.18--1](ch02s02abv156.md#exxdrucker-deform) shows the 16 elements used in the analysis in their original and deformed shapes. The dashed lines illustrate the original mesh. The 8-node brick element (C3D8R) appears twice in each row: in one case boundary conditions are applied to constrain the out-of-plane displacement so that the C3D8R element generates plane strain results, in the second case no out-of-plane displacement boundary conditions are specified so that the C3D8R element generates plane stress results. The original length of each side of the elements is ![](../graphics/ver_eqn00779.gif) 1.

This example problem is designed to test the following features:
- plane strain, plane stress, and three-dimensional cases
- compression and simple shear deformations
- associated and nonassociated flow
- third invariant dependence (the value of *K*)

This is accomplished as described below.

In rows (a) and (c) in [Figure 2.2.18--1](ch02s02abv156.md#exxdrucker-deform), the bottom and top nodes of each element are given equal and opposite prescribed constant velocities in the *y*-direction to generate a compressive loading. For rows (b) and (d) in [Figure 2.2.18--1](ch02s02abv156.md#exxdrucker-deform), the top nodes of each element are given prescribed velocities in the *x*-direction, while the bottom nodes are fixed, thereby leading to simple shear modes.

The elements in rows (a) and (c) in [Figure 2.2.18--1](ch02s02abv156.md#exxdrucker-deform) are assigned material definitions with an associated flow rule: the friction angle (![](../graphics/ver_eqn00621.gif)) and dilation angle (![](../graphics/ver_eqn00732.gif)) are each 40. The elements in rows (b) and (d) in [Figure 2.2.18--1](ch02s02abv156.md#exxdrucker-deform) are assigned material definitions with a nonassociated flow rule: the friction angle (![](../graphics/ver_eqn00621.gif)) is 40, while the dilation angle (![](../graphics/ver_eqn00732.gif)) is 20.

The value of *K* is a measure of the strength differences in tension and compression. The yield surface will not be circular in the deviatoric stress space if *K* does not have a value of 1.0. In the first two rows of elements in [Figure 2.2.18--1](ch02s02abv156.md#exxdrucker-deform), a value of ![](../graphics/ver_eqn00780.gif) 0.8 is assigned. In rows (c) and (d) in [Figure 2.2.18--1](ch02s02abv156.md#exxdrucker-deform), a value of ![](../graphics/ver_eqn00780.gif) 1.0 is assigned to the elements.

The plastic stress-strain relationship is defined using the extended Drucker-Prager plasticity model with hardening. Perfect plasticity is assumed, with the yield stress in uniaxial compression ![](../graphics/ver_eqn00772.gif) 40  103. The elastic properties are ![](../graphics/ver_eqn00187.gif) 20  106, ![](../graphics/ver_eqn00188.gif) 0.3. Material densities are 1000.

### Results and discussion

[Figure 2.2.18--2](ch02s02abv156.md#exxdrucker-ys-assoc-plstrn) shows the plot of Mises stresses versus pressure in the plane strain cases (elements C3D8 and CPE4R) with an associated flow rule. This demonstrates the pressure-dependent nature of the material. In the case of ![](../graphics/ver_eqn00780.gif) 1.0, the slope of the curve corresponds to the tangent of 40.

[Figure 2.2.18--3](ch02s02abv156.md#exxdrucker-ys-assoc-plstrss) shows the plot of Mises stresses against pressure in the plane stress cases (elements C3D8 and CPS4R) with an associated flow rule.

[Figure 2.2.18--4](ch02s02abv156.md#exxdrucker-ys-na-plstrn) shows the plot of Mises stresses versus pressure in the plane strain cases (elements C3D8 and CPE4R) with a nonassociated flow rule. In the case of ![](../graphics/ver_eqn00780.gif) 1.0, the slope of the curve corresponds to the tangent of 40.

[Figure 2.2.18--5](ch02s02abv156.md#exxdrucker-ys-na-plstrss) shows the plot of Mises stresses against pressure in the plane stress cases (elements C3D8 and CPS4R) with a nonassociated flow rule.

When *K* is less than 1.0, the slope of the Mises stress versus pressure curve will be less than or equal to the frictional angle. This depends on the plastic strain path in the noncircular deviatoric space.

[Figure 2.2.18--6](ch02s02abv156.md#exxdrucker-eps-08) contains eight curves of the time history response of equivalent plastic strain for the eight elements that have a value of ![](../graphics/ver_eqn00781.gif)  0.8. [Figure 2.2.18--7](ch02s02abv156.md#exxdrucker-eps-10) contains eight history curves of equivalent plastic strain for the eight elements that have a value of ![](../graphics/ver_eqn00781.gif) 1.0. Only four curves are visible in [Figure 2.2.18--6](ch02s02abv156.md#exxdrucker-eps-08) and [Figure 2.2.18--7](ch02s02abv156.md#exxdrucker-eps-10) because the three-dimensional results for the C3D8R elements reproduce the plane strain and plane stress results. As discussed above, boundary conditions were applied to the C3D8R elements to achieve this correspondence. This serves as a check that both the two- and three-dimensional material models achieve the same results.

The results that are obtained with Abaqus/Explicit are the same as those obtained with Abaqus/Standard.
