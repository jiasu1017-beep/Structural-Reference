# 69.6.1 缝合边以创建面







您可以通过自动缝合自由边之间的任何间隙来创建新面。缝合间隙允许您替换先前删除的小面、圆角或倒角。缝合间隙操作作为部件的特征存储；因此，您可以使用模型树删除缝合并恢复到原始部件。 有关更多信息，请参阅["什么是缝合？" 第 69.3 节"](pt06ch69s03.md)。

您可以在流形部件（每个边仅被一个或两个面共享的部件）或非流形部件中缝合间隙。

**缝合边：**

1. 从主菜单栏中选择 ****Tools**![](../graphics/images/arrow.gif)**Geometry Edit****（工具→几何编辑）。Abaqus/CAE 显示 **Geometry Edit**（几何编辑）对话框。**提示：**您也可以使用位于 Part（部件）模块工具箱中编辑工具处的 ![](../graphics/ico_repairStitch.png) 工具来缝合边。 有关工具箱中编辑工具的图表，请参阅["使用几何编辑工具集，" 第 69.1 节"](pt06ch69s01.md)。
2. 从对话框中选择 **Edge**（边）类别和 **Stitch**（缝合）方法。
3. 从出现的警告对话框中点击 **OK**（确定）。
4. 如果需要，从提示区域为缝合操作指定缝合公差。Abaqus/CAE 仅会缝合小于您指定的公差值的间隙。
5. 执行以下操作之一： - 如果您想单独选择要缝合操作的边，请切换开启 **Pick edges**（选择边），点击鼠标按钮 2，然后指定要选择的边。您选择的所有边必须是自由边，因为当选择 **Pick edges**（选择边）时，Abaqus/CAE 仅尝试流形缝合。如果您选择任何被多个面共享的边，缝合操作将失败。**提示：**您可以单独选择边，指定包含边的现有集合，或使用角度方法选择边。 有关在视口中选择几何的更多信息，请参阅["在当前视口中选择对象，" 第 6.2 节"](pt01ch06s02.md)。 - 如果您想在整个模型中缝合边，请关闭 **Pick edges**（选择边）并点击鼠标按钮 2。当 **Pick edges**（选择边）关闭时，Abaqus/CAE 可以尝试在整个模型上进行流形和非流形缝合。如果模型的任何边被多个面共享，则 resulting 模型将是非流形的。Abaqus/CAE 根据您的规范尝试缝合部件。如果修复成功，Abaqus/CAE 会提示您更新部件的有效性。
6. 如果您编辑了部件，您应该通过在模型树中的部件上点击鼠标按钮 3 并从出现的菜单中选择 **Update validity**（更新有效性）来检查其有效性。
