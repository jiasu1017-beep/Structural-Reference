# 19.4.2 协同执行管理器





**协同执行管理器**与 Abaqus/CAE 中的其他管理器类似，允许您执行以下操作：
- 创建协同执行并将其与所选分析关联。
- 编辑所选协同执行。
- 复制、重命名或删除所选协同执行。

此外，**协同执行管理器**允许您执行以下操作：- 在提交协同执行之前对模型执行[数据检查](pt03ch19s11hlb02.md)。
- [提交](pt03ch19s11hlb03.md)协同执行。
- [查看](pt03ch19s11hlb04.md)协同执行的结果。
- [终止](pt03ch19s11hlb05.md)当前正在运行的协同执行。

您可以通过从主菜单栏选择 ****协同执行**![](../graphics/images/arrow.gif)**管理器**** 来显示**协同执行管理器**。[图 19-4](pt03ch19s04s02.md#ana-coexman) 显示了**协同执行管理器**的布局。

**图 19-4** **协同执行管理器**。

![](../graphics/ana-coexman-nls.png)

**协同执行管理器**的三列显示以下内容：

**名称**

**名称**列显示协同执行的名称。单击**重命名**可以重命名所选协同执行。

**分析**

**分析**列显示与协同执行关联的分析类型；例如，**Abaqus-Abaqus**。

**状态**

**状态**列显示协同执行中作业的状态，并在协同执行运行时持续更新。状态可以是以下之一：

**无**

协同执行尚未提交进行分析。

**已提交检查**

分析正在提交进行数据检查。

**正在检查**

分析的数据检查正在运行。

**检查完成**

分析的数据检查已成功完成；您现在可以继续协同执行。

**已提交**

协同执行正在提交。

**正在运行**

协同执行已提交并正在运行。

**已完成**

协同执行已运行完成。

**中止**

由于某个模型中的致命错误或磁盘空间不足等问题，协同执行已中止。

**已终止**

协同执行已被用户终止。
