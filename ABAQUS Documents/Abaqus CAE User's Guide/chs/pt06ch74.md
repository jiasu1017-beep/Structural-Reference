# 74 流工具集







流线是瞬时与流动速度向量相切的曲线，流是由一组流线组成的，使您能够可视化流体流动分析中的速度或涡量数据。"流工具集"仅在Visualization模块中可用。

本章解释如何使用"流工具集"创建、修改和删除流；在视口中显示或隐藏它们；以及自定义其外观的多个方面。涵盖以下主题：

- ["创建流," 第74.2节](pt06ch74s02.md)
- ["显示和隐藏流," 第74.3节](pt06ch74s03.md)
- ["自定义流显示," 第74.4节](pt06ch74s04.md)
