# 18.9 创建约束





约束限制Optimization模块在优化期间对模型拓扑进行更改。 如果约束无法满足，则优化不可行。 要配置约束，请选择设计响应或设计响应组合并指定约束的值；例如，优化模型的体积可以约束为原始体积的50%。

**创建或编辑约束：**

1. 从主菜单栏，选择 ****Constraint**![](../graphics/images/arrow.gif)**Create****。 出现**Create Constraint**对话框。 **提示：**您可以通过另外两种方式启动**Create**过程： - 单击**Constraint Manager**中的**Create**。 （您可以通过从主菜单栏选择 ****Constraint**![](../graphics/images/arrow.gif)**Manager**** 来显示**Constraint Manager**。） - 单击Optimization模块工具箱中的 ![](../graphics/icoR_optConstraint.png) 工具。
2. 从出现的**Create Constraint**对话框中，输入约束的名称。 出现**Edit Constraint**对话框。
3. 从菜单中选择将用于定义此约束的设计响应。 或者，您可以选择 ![](../graphics/icoR_optDesResp.png) 创建新的设计响应并将其添加到设计响应列表。 约束编辑器显示设计响应的类型，这是您正在约束的变量。 如果您正在执行基于条件的优化，必须选择体积约束。
4. 定义约束。 编辑器提供不同的选项，取决于您选择的优化任务类型。 - 如果您正在为基于条件的拓扑或形状优化定义约束，体积约束可以是以下之一： - 等于固定值。 - 等于优化开始前值的一部分。 - 如果您正在为一般拓扑优化定义约束，约束可以是以下之一： - 小于、大于或等于固定值。 - 小于、大于或等于优化开始前值的一部分。 **注意：**在基于条件的拓扑或形状优化期间，体积会增量减少，直到在最后一步满足体积约束。 相比之下，一般拓扑优化在第一个设计周期就满足体积约束。 因此，一般拓扑优化从更弱的结构开始，单元密度被缩放以满足剩余的约束和目标。
5. 单击**OK**保存数据并退出编辑器。
