# 3.8 报告 FLEXnet 许可证使用数据



本节介绍创建许可证使用报告的方法。如果您的 Abaqus FLEXnet 许可证服务器位于广域网 (WAN) 上，且不同地区的 Abaqus 用户从多个区域 SIMULIA 办事处获得技术支持和服务，则需要向 SIMULIA 提交许可证使用报告。有关报告要求的更多信息，请参阅 Dassault Systèmes 知识库中的"多地区帐户的许可证使用报告"，网址为 [www.3ds.com/support/knowledge-base](http://www.3ds.com/support/knowledge-base)。

报告工具可供所有 Abaqus 许可证持有者使用。Abaus 提供了两个实用程序来生成许可证使用报告：
- `usage` 实用程序（请参阅["手动创建 FLEXnet 许可证使用报告，"第 3.8.1 节](ch03s08.md#sgb-chp-net-lic-report-man)）可用于为指定日期范围手动生成许可证使用报告。
- 计划的许可证报告实用程序（`schLicRpt`，请参阅["计划自动 FLEXnet 许可证使用报告，"第 3.8.2 节](ch03s08.md#sgb-chp-net-lic-report-sched)）可用于设置每月或每季度自动生成许可证使用报告。

许可证使用报告实用程序使用 FLEXnet 调试日志文件中的信息。您可以通过在使用 `lmgrd` 启动许可证服务器时包含 `-l` 选项来确保创建此日志（请参阅["FLEXnet 许可证服务器管理器 lmgrd，"第 3.5 节](ch03s05.md)）。要按用户位置细分报告，必须设置 **computer_location** 环境文件参数。有关环境文件参数的更多详细信息，请参阅["许可证管理参数，"第 4.1.6 节](ch04s01.md#sgb-chp-env-net)。

### 3.8.1 手动创建 FLEXnet 许可证使用报告

`usage` 实用程序允许您生成详细说明 Abaqus 许可证使用情况的 CSV 格式报告。生成两种类型的用法报告：汇总使用报告和每日使用报告。汇总使用报告提供按用户位置或单个用户细分的每个 Abaqus 许可证功能的使用信息。汇总使用报告列出在指定时间段内请求的许可证总数、授予的许可证数量、拒绝的许可证数量以及许可证使用总小时数。每日使用报告列出指定时间段内每天检出的分析和 Abaqus/CAE 许可证令牌的峰值和平均值。每日使用报告提供所有用户和位置的汇总数据。

许可证使用报告将令牌使用计算为两位有效数字。如果使用小时数（汇总报告中）或检出的平均令牌数（每日报告中）小于 0.005，则使用报告将这些值列为零。

如果 FLEXnet 调试日志文件（或日志文件）位于远程服务器上，则必须在运行 `usage` 实用程序之前将其复制到本地计算机可访问的目录。

**语法和选项**

`abaqus licensing usage` {`-summary` *report_file* | `-daily` *report_file*} [`-nouser`] `-log` {*debug_log_file* | *debug_log_directory*} [`-start` *start_date*] [`-end` *end_date*]

**`-summary`**

使用此选项创建汇总每个许可证功能按用户或位置的使用情况的报告。报告将写入由 *report_file* 指定的路径和文件名。

**`-daily`**

使用此选项创建提供分析和 Abaqus/CAE 许可证令牌峰值和每日使用情况的报告。报告将写入由 *report_file* 指定的路径和文件名。

**`-nouser`**

此选项抑制汇总报告中关于个别用户的使用数据。它报告整个许可证服务器的汇总数据。如果设置了 **computer_location** 环境文件参数，它还会报告每个位置的汇总数据。此选项与 `-daily` 选项结合使用时无效；每日报告不包括有关个别用户或位置的信息。

**`-log`**

使用此选项指定调试日志文件或包含多个调试日志文件的目录的路径；实用程序会在指定路径中检查所有日志文件，以获取指定报告期内的使用数据。

**`-start`**

使用此选项指定报告期的开始日期和时间。如果未指定 `-start` 选项，则报告期从调试日志文件中记录的最旧项目开始。日期和时间必须按以下格式输入：

```
*dd*-*mmm*-*yyyy*[_*hh*[:*mm*[:*ss*]]]
```
 例如，`-start 01-jan-2008_09:00:00` 表示 2008 年 1 月 1 日上午 9:00 的开始时间。时间指定是可选的；默认开始时间是 `00:00:00`。如果指定了时间，则必须以 24 小时格式输入小时字段 (*hh*)。

**`-end`**

使用此选项指定报告期的结束日期和时间。如果未指定 `-end` 选项，则报告期从调试日志文件中记录的最新项目结束。日期和时间必须按以下格式输入：

```
*dd*-*mmm*-*yyyy*[_*hh*[:*mm*[:*ss*]]]
```
 例如，`-end 31-mar-2008_18:00:00` 表示 2008 年 3 月 31 日下午 6:00 的结束时间。时间指定是可选的；默认结束时间是 `23:59:59`。如果指定了时间，则必须以 24 小时格式输入小时字段 (*hh*)。

### 3.8.2 计划自动 FLEXnet 许可证使用报告

Abaqus 提供了一个实用程序，可根据每月或每季度的计划自动生成汇总和每日 FLEXnet 许可证使用报告。该实用程序创建一个包含报告生成命令的脚本，然后安装一个计划任务（在 Windows 平台上）或 `crontab` 条目（在 Linux 平台上），以每月或每季度执行该脚本。调试日志文件可以位于与计划自动许可证使用报告的计算机不同的服务器上。如果需要，您可以在报告生成后立即指示实用程序自动将报告通过电子邮件发送到指定地址。

您应该创建一个专用于许可证使用报告的新目录，并从该目录内运行计划的许可证使用报告实用程序。报告生成脚本（`usage_report.bat` 或 `usage_report.sh`）、实用程序配置文件（`schLicRpt.cfg`）和生成的报告都保存到您运行实用程序的目录中。生成的报告使用以下文件命名约定：

```
abaqus_usage*n*_*siteID*_*date*.csv
```
其中 *siteID* 是 SIMULIA 分配给您站点的客户编号，*date* 以 *yymmdd* 格式数字编写。*n* 指示使用报告的类型：`1` 表示汇总使用报告，`2` 表示每日使用报告。例如，`abaqus_usage2_01ABCD_080515.csv` 是 2008 年 5 月 15 日在站点 01ABCD 创建的每日使用报告的文件名。

要运行该实用程序，请使用以下命令：

```
abaqus schLicRpt
```
该实用程序将提示您输入所需信息，包括调试日志文件的位置、报告生成的频率，以及如果您选择通过电子邮件发送报告，则需要 SMTP 服务器的完全限定域名和发送报告的电子邮件地址。如果您从同一目录再次运行该实用程序，则在配置过程中将根据保存的配置文件提供默认设置。因此，您可以对小范围的报告配置设置进行修改，而无需重新输入所有必需信息。

您必须有权访问 `schTasks.exe` 实用程序（在 Windows 上）或 `crontab` 实用程序（在 Linux 上）才能运行计划的许可证报告实用程序。为了获得最佳性能，建议您在 Abaqus 许可证服务器主机计算机上执行该实用程序，这只有在已在此计算机上安装 Abaqus 产品时才有可能。该实用程序只能在整个站点的一台计算机上执行。否则，您将生成重复的使用报告，并在生成报告时可能降低许可证服务器的性能。

使用以下过程从系统中删除自动生成的报告：

**Linux 平台**

要列出当前的 crontab 条目，请运行以下命令：

```
crontab -l
```
要编辑当前的 crontab 条目，请运行以下命令：
```
crontab -e
```
要删除所有当前的 crontab 条目，请运行以下命令：
```
crontab -r
```
有关 crontab 的更多详细信息，请在任何提示符下键入 `man crontab` 以查看系统的 crontab 手册页。

**Windows 平台**

要列出当前计划的任务，请运行以下命令：

```
schTasks /query
```
要删除自动 Abaqus 许可证使用报告，请运行以下命令：
```
schTasks /delete /tn abaqus_usage_report
```
