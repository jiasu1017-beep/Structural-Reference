# 72.2 参考点的用途是什么？







您可以将参考点用于以下用途：
- 如果部件是使用广义平面应变单元建模的可变形平面部件，您必须创建一个参考点来指示所需的参考节点。有关更多信息，请参见["创建广义平面应变截面," 第12.13.2节](pt03ch12s13hlb02.md)。有关广义平面应变单元的更多信息，请参见["选择单元的维度," 第27.1.2节（Abaqus Analysis User's Guide）](../usb/usb-link.md#usb-elm-edimension)。
- 如果部件是离散或解析刚体，您可以使用参考点来指示刚体参考点。您应用于参考点的约束或运动将应用于整个刚体。刚体参考点的位置会影响您如何指定力矩或运动；此外，刚体参考点的位置会影响力矩反力的解释。如果您的模型包含涉及旋转的动态分析，刚体的转动惯性规格必须与其刚体参考点的位置一致。
- 在Interaction模块中创建刚体约束时，您必须引用装配体上的一个参考点。刚体约束将装配体区域的运动约束到参考点的运动。您可以创建并命名一个包含参考点的集，然后引用该集，或者您可以直接从当前视口中选择参考点。您还可以在Load模块中将载荷和边界条件应用于参考点。
- 在某些情况下，您可能希望将参考点定位在质心处，您可以使用查询工具集找到质心。您可以使用Property模块将质量和转动惯性截面属性分配给参考点。该截面还可以包含可选的阻尼数据。
- 您可以将约束应用于参考点；例如，方程、耦合和显示体约束。
- 当您在Assembly模块或Interaction模块中创建装配体级线特征时，您可以使用参考点。当您想将连接器连接到空间中的一个点时，参考点非常有用。您可以在所需位置创建参考点，然后使用刚体约束将部件实例附加到参考点。当您将连接器附加到这样的参考点时，连接器实际上附加到了部件实例。
