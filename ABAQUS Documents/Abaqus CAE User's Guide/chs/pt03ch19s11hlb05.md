# 19.11.5 终止共同执行







要终止共同执行，您应该使用**共同执行管理器**显示正在运行的共同执行并终止分析作业。终止作业将终止共同执行。Abaqus/CAE 会要求您确认，终止作业，并将**共同执行管理器**中共同执行的状态更新为`已终止`。请注意，共同执行终止可能导致各种错误消息写入消息区域，反映共同执行作业间消息终止的不可预测方式。共同执行终止后立即出现的这些错误消息通常可以忽略。
