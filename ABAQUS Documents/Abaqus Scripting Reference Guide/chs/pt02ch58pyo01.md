# 58.1 ConnectorBehaviorOption 对象







ConnectorBehaviorOption 对象是其他 ConnectorBehaviorOption 对象的抽象基类型。ConnectorBehaviorOption 对象没有显式构造函数。ConnectorBehaviorOption 对象的成员是所有派生自 ConnectorBehaviorOption 的对象的通用成员。

**访问**

```
sectionApi.sections()[*name*].behaviorOptions(*i*)
```

### 58.1.1 成员

ConnectorBehaviorOption 对象没有成员。
