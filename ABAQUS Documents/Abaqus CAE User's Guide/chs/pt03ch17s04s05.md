# 17.4.5 约束种子







默认情况下，网格种子仅规定目标网格密度。当您使用自由网格划分技术生成三角形或四面体形状的单元时，Abaqus/CAE 通常会精确匹配网格种子。但在其他情况下，Abaqus/CAE 可能会更改单元分布，以便成功生成网格。如果您想防止 Abaqus/CAE 更改单元分布，可以通过约束沿边缘的种子来固定沿边缘的特定单元数量。您只能约束边缘种子，而不能约束零件或实例种子。

您可以为沿边缘的一组种子分配以下三种状态之一：

**未约束**

这是默认设置。沿边缘的单元数量可以增加或减少，以便网格可以变得比种子规定的更密或更稀疏。未约束种子显示为空心圆。

**部分约束**

沿边缘的单元数量可以在网格生成期间增加，但不能减少。此约束允许网格变得比种子规定的更密，但不能更稀疏。部分约束种子显示为向上指的三角形。

**完全约束**

沿边缘的约束种子规定的单元数量不能被网格生成过程更改。当种子完全约束时，网格生成将尝试使节点的位置与种子的位置完全对应。但是，不能保证种子和节点位置之间的完全匹配。完全约束种子显示为正方形。

Abaqus/CAE 总是在区域的每个几何顶点处创建完全约束种子，以指示将在每个顶点处放置有限元节点。

在许多情况下，网格生成器必须重新分配单元（并偏离种子的数量和位置）才能成功生成网格。为了最大限度地提高网格划分的成功率，请将种子保持为未约束
