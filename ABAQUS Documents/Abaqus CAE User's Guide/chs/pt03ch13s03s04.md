# 13.3.4 从从属更改为独立零件实例或反之







对从属零件实例的限制可能会限制您对装配进行分区或网格划分的能力，或者您可能希望将虚拟拓扑应用到实例。要在使零件实例从属和独立之间切换，您可以右键单击模型树中的实例，然后从出现的菜单中选择**设为从属**或**设为独立**。

如果您对零件进行了网格划分并创建了该零件的从属实例，Abaqus/CAE会将网格与实例关联。如果您随后将实例从从属更改为独立，Abaqus/CAE继续将网格与独立实例关联。但是，反过来则不然。如果您创建独立实例，对实例进行网格划分，然后随后将实例转换为从属，Abaqus/CAE会从从属实例中删除网格。分区和虚拟拓扑也是如此。当您将独立零件实例更改为从属时，Abaqus/CAE会删除应用于独立零件实例的任何分区或虚拟拓扑。

在某些情况下，您可以通过创建原始零件的副本并从副本创建独立实例来规避对从属零件实例的限制。然后您可以对新实例进行分区或网格划分，或将虚拟拓扑应用到新实例。类似地，虽然您不能创建同一零件的从属和独立实例，但您可以创建零件的副本，并从副本创建任一类型的实例。
