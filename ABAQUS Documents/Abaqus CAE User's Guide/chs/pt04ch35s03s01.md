# 35.3.1 为中面建模使用偏移面工具





您可以使用Geometry Edit工具集中的偏移面工具来创建中面。偏移面工具根据原始面组创建偏离一定距离的新面。偏移面使用与在Sketcher中偏移对象类似的过程（更多信息，请参见["偏移对象"，第20.8.6节](pt03ch20s08s06.md)）。偏移面工具的详细使用在["偏移面"，第69.7.7节](pt06ch69s07hlb07.md)中讨论。

除了创建新壳面的基本任务外，偏移面工具还提供两个对创建中面模型有用的功能：
- 偏移距离用于Abaqus/CAE为新壳面分配厚度。
- 自动修剪选项自动将新面修剪到参考表示。

选择要偏移的面后，您可以输入偏移距离值或选择一组目标面来表示所需壳厚度的距离，并让Abaqus/CAE计算偏移。Abaqus/CAE计算要偏移的面与目标面之间的距离。您可以偏移到平均距离的一半或目标面上最近或最远点的距离的某个分数。

[图35-1](pt04ch35s03s01.md#adv-mid-offsetdist)显示了使用三种计算偏移距离方法之间的差异。部件有三种不同的厚度。最薄的面在顶部，最厚的在中间，右侧垂直面的厚度介于两者之间。选择偏移的三个外表面，目标是三个内表面。左侧视图显示默认的**Half the average distance**方法，偏移大约是顶部面整个厚度。中间视图显示**Fraction distance to closest point on face**方法，分数距离设置为`0.5`。顶部目标面最接近顶部偏移面，因此Abaqus/CAE将新面偏移0.5倍的距离。右侧视图显示**Fraction distance to farthest point on face**方法，分数距离再次设置为`0.5`。中间（倾斜）目标面距中间偏移面最远，因此Abaqus/CAE将新面偏移0.5倍的距离——注意顶部偏移面被切断了，因为它会通过原始实体部件参考表示延伸，因为使用了**Auto trim to reference representation**选项。

**图35-1** 基于三种目标面计算方法的偏移距离差异。

![](../graphics/adv-mid-offsetdist.png)

Abaqus/CAE自动将基于偏移计算的新偏移壳面与厚度关联。如果您选择目标面来计算偏移距离，Abaqus/CAE还会自动重新计算顶部和底部面集几何体任何变化的偏移。使用**Assign Thickness and Offset**工具和**Render shell thickness**选项来编辑和查看分配给壳面的厚度。更多信息，请参见["分配厚度和偏移"，第35.5.2节](pt04ch35s05s02.md)和["可视化壳厚度"，第35.5.3节](pt04ch35s05s03.md)。

如果您在使用偏移面时使用自动修剪选项，Abaqus/CAE会偏移面，并将新面的外边缘延伸等于偏移距离的量。然后在新面与参考表示相交处修剪新面。[图35-2](pt04ch35s03s01.md#adv-mid-trimtorefrep)显示了使用自动修剪选项的可能效果。参考表示包含两个单元，顶部较薄单元的高亮面被选择进行偏移操作。使用自动修剪选项，新面延伸到参考表示的第二个单元中。

**图35-2** 修剪选项仅修剪到参考表示的外表面。

![](../graphics/adv-mid-trimtorefrep.png)

在这种情况下，可以不带修剪选项偏移所选面以创建不会改变几何体的偏移。或者，您可以使用延伸面工具（extend faces tool）来延伸偏移面以填充整个参考表示，为两个单元创建一个面。
