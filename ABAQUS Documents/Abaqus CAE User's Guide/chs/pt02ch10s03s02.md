# 10.3.2 零件属性是什么？





零件具有以下属性：

**建模空间（Modeling Space）**

当您导入零件时，Abaqus/CAE会扫描文件并尝试按如下方式确定所导入零件的建模空间：
- 如果Abaqus/CAE确定零件是三维的，则将建模空间设置为三维。
- 如果Abaqus/CAE确定零件是平面的，您可以选择建模空间是二维还是三维。
- 如果Abaqus/CAE确定零件是平面的且其几何形状不穿过Y轴，您可以选择建模空间是轴对称、二维还是三维。如果您选择轴对称，则假定Y轴为旋转轴，您可以添加扭曲自由度。

更多信息，请参阅["零件建模空间，"第11.4.1节"](pt03ch11s04s01.md)。

**类型（Type）**

Abaqus/CAE始终假定零件类型是可变形的。或者，您可以选择**离散刚体（Discrete rigid）**导入离散刚体零件，或选择**欧拉（Eulerian）**导入欧拉零件。更多信息，请参阅["零件类型，"第11.4.2节"](pt03ch11s04s02.md)。

尽管您无法将导入的零件定义为解析刚体零件，但您可以将解析刚体几何导入到草图中。然后您可以创建新的解析刚体零件并将导入的草图复制到草图器工具集中。
