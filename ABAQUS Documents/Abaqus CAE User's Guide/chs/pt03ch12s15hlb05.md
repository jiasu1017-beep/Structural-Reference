# 12.15.5 分配壳/膜法线方向







当您创建带有壳区域的零件或带有线框区域的轴对称零件时，Abaqus会为这些区域分配法线方向。您可以反转这些区域的法线方向。此外，您可以反转导入零件或所选孤立单元的法线方向。

您也可以反转皮肤增强的法线方向；但是，如果在几何零件的表面上定义了多层皮肤，您无法在不反转该表面所有皮肤法线的情况下反转其中一层皮肤的法线。如果具有多层皮肤的表面是网格零件的特征，您可以单独反转一层皮肤的法线，而无需反转为该表面定义的其他皮肤的法线。

当您反转区域的法线方向时，定义材料方向的局部法线会被反转。您可以使用查询工具集来验证材料方向对于选定区域是否正确。请参见"使用查询工具集获取分配信息"，第12.19节。

**为零件分配壳/膜法线：**

1. 如果您要分配法线的零件在当前视口中不可见，请单击上下文栏中**零件**列表中所需零件的名称。所选零件出现在当前视口中。
2. 从主菜单栏，选择**分配**![](../graphics/images/arrow.gif)**单元法线**。**提示：**您也可以单击属性模块工具箱中的![](../graphics/ico_assignShellNormal.png)工具，或在网格模块中选择**网格**![](../graphics/images/arrow.gif)**方向**![](../graphics/images/arrow.gif)**法线**。Abaqus/CAE使用阴影渲染样式显示零件。壳的法线与壳法线（顶面）重合的一面以棕色显示；另一面（底面）以紫色显示。
3. 如果所选零件是基于几何的，使用以下方法之一选择要反转其法线的区域：- 从视口中选择区域，然后单击鼠标按钮2。（有关更多信息，请参见第6章"在视口中选择对象"。）**提示：**您可以使用**选择**工具栏中的工具限制可以在视口中选择的对象类型。请参见"使用选择选项"，第6.3节。如果您选择**皮肤**作为对象类型，可以为整个皮肤或其一个或多个面分配截面。从提示区域，选择**（选择整个皮肤）**或**（选择部分皮肤）**，然后在视口中进行选择。如果所选零件有多个皮肤，Abaqus/CAE在您进行选择时会在提示区域显示模糊选择选项。- 要从现有集合列表中进行选择，请执行以下操作：1. 单击提示区域右侧的**集合**。Abaqus/CAE显示包含可用零件集合列表的**区域选择**对话框。2. 选择感兴趣的零件集合，然后单击**继续**。**注意：**默认选择方法基于您最近使用的选择方法。要切换回另一种方法，请单击提示区域右侧的**在视口中选择**或**集合**。Abaqus/CAE反转所选区域的壳/膜法线方向。
4. 如果所选零件是网格零件，使用以下方法之一选择要反转其法线的壳单元：**单独选择单元：**1. 单击提示区域中**选择方法**字段旁边的箭头，从出现的列表中选择**单独**。2. 选择要反转其法线的单元。3. **[Shift]**+单击**其他单元将它们添加到您的选择中。4. 如有必要，**[Ctrl]**+单击**所选单元以取消选择它们。5. 完成选择单元后，单击鼠标按钮2。**指定现有单元集合**1. 单击提示区域右侧的**集合**。Abaqus/CAE显示包含您创建的单元集合列表的**区域选择**对话框。2. 选择要编辑的单元集合，然后单击**继续**。**注意：**默认选择方法基于您最近使用的选择方法。要切换回另一种方法，请单击提示区域右侧的**在视口中选择**或**集合**。**使用角度方法选择单元：**1. 从提示区域中的字段，选择**按角度**。2. 输入一个角度（0到90之间），然后选择一个单元面。Abaqus/CAE从所选面开始选择每个相邻壳单元，直到相邻单元面之间的角度等于或超过您输入的角度。（有关更多信息，请参见"使用角度和特征边缘方法选择多个对象"，第6.2.3节。）3. 使用**按角度**方法后，您可以**[Shift]**+单击**其他单元将它们添加到您的选择中，或**[Ctrl]**+单击**所选单元将它们从您的选择中移除。（有关更多信息，请参见"组合选择技术"，第6.2.8节。）4. 完成选择单元后，单击鼠标按钮2。**提示：**您可以使用**选择**工具栏中的工具限制可以在视口中选择的对象类型。请参见第6章"在视口中选择对象"。
5. 从提示区域选择一种反转壳/膜法线的方法：- 单击**全部翻转**反转所有所选单元的法线。- 单击**选择法线**更改所选单元的法线，使它们指向您指定的参考单元的法线方向。
6. 如果您在前述步骤中选择了**选择法线**，请执行以下操作：1. 在视口中选择参考单元。2. 在提示区域中单击**确定**。Abaqus/CAE更改所选单元的法线，使它们指向与参考单元法线相同的方向。
7. 根据需要经常重复前述步骤以反转其他壳/膜法线。完成分配法线方向后，使用以下方法之一退出分配模式：- 如果您正在从视口中选择零件区域或单元，请单击鼠标按钮2或单击提示区域中的**完成**。- 如果您正在从**区域选择**对话框中选择预存集合，请单击**取消**关闭对话框，然后单击鼠标按钮2或单击提示区域中的**完成**。Abaqus/CAE恢复为在开始反转壳/膜法线程序之前选择的渲染样式。
