# F.1 常见问题





本节解答有关 Abaqus FLEXnet 许可的一些常见问题。

### F.1.1 FLEXnet 网络许可的网络规模是否有限制？

网络规模受到所有运行 Abaqus 的计算机必须能够直接和定期与 FLEXnet 许可证服务器连接这一要求的限制。如果与服务器通信花费的时间过长，客户端花费在与服务器通信上的时间将多于处理时间。未能定期通信将导致客户端丢失令牌并在重新获取时等待。最基本的网络通信测试是 `ping` 命令。如果客户端和服务器无法通过名称和 IP 地址相互 ping，则网络许可将无法在两台计算机之间工作。（但是，`ping` 命令正常运行并不一定能保证网络许可工作。）

### F.1.2 FLEXnet 网络许可是否需要运行名称服务器？

设置网络许可需要某种形式的主机名解析。如果您的网络上没有 NIS 或 DNS 名称服务器功能，则必须在每台机器上手动配置主机文件。有关详细信息，请参阅 Dassault Systmes Knowledge Base 中的"DNS 名称解析"，网址为 [www.3ds.com/support/knowledge-base](http://www.3ds.com/support/knowledge-base)。

### F.1.3 如何判断是否已安装 FLEXnet 许可证文件？

Abaqus 的许可证文件通常安装在名为 `*simulia_dir*/License` 的目录中，通常命名为 `simulialm.lic`。`License` 目录可以放在任何地方，但通常位于 SIMULIA 父目录内。

### F.1.4 FLEXnet 许可证服务器日志文件是什么？应该存储在哪里？

服务器日志文件记录 Abaqus 作业和会话（客户端）的所有许可证请求。它存储在 Abaqus `License` 目录中，您可以为其指定任何名称。通常该文件称为 `simulialm.log`。

### F.1.5 FLEXnet 许可证选项文件是什么？应该存储在哪里？

选项文件允许许可证管理员控制 FLEXnet Licensing 的各种操作参数，例如为网络上的某些用户、计算机或域保留一定数量的令牌，或限制对 Abaqus 许可证令牌的访问。虽然可以存储在任何地方并可以指定任何名称，但选项文件的推荐位置和名称是 `*simulia_dir*/License/SIMULIALM.opt`。如果您不想限制对 Abaqus 许可证令牌的访问，则不需要许可证选项文件。

创建或修改许可证选项文件后，您必须使用 `lmreread` 工具重新读取许可证文件并实施新选项（请参阅["lmreread，"第 3.6.6 节](ch03s06.md#sgb-chp-net-lic-lmreread)）。选项文件也在许可证服务器启动时读取。某些对选项文件的更改不会立即生效，除非重新启动许可证服务器。

选项文件示例在["使用 FLEXnet 选项文件，"第 3.7 节](ch03s07.md)中显示，创建选项文件的说明包含在 FLEXnet Licensing End User Guide Version 11.6 的第 5 章中，列在 **Support** 页面的 **Licensing** 下，网址为 [www.3ds.com/simulia](http://www.3ds.com/simulia)。
