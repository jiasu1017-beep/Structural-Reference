# 10.9.5 将视口数据导出为3D XML格式文件





从主菜单栏选择**文件**![](../graphics/images/arrow.gif)**导出**![](../graphics/images/arrow.gif)**3DXML**，可以将当前视口中的模型导出为3D XML格式文件。3D XML文件可用于将三维图像集成到常见的业务应用程序中，例如电子邮件、Web页面、文字处理文档和演示文稿。3D XML Player（可从Dassault Systmes免费获取）可用作独立的3D XML客户端，其他应用程序需要它来查看3D XML文件。您也可以在CATIA V5中不使用3D XML Player的情况下查看3D XML文件。此导出功能不能用于将几何或模型传输到3DEXPERIENCE平台应用程序。

您可以在Abaqus/CAE会话期间随时将模型导出为标准3D XML文件或压缩3D XML文件。压缩3D XML文件通常比相应的标准3D XML文件小得多，但压缩比取决于模型的内容。

3D XML文件以基于通用标准XML语言的XML格式存储。压缩3D XML文件必须解压缩才能转换为XML格式。

要在导出的3D XML文件中查看某些基于线条的特征（如元素边缘、定向前三轴和基准平面），，请在3D XML Player中将视图模式设置为**带边缘着色**。不会导出一维和二维特征（包括梁或桁架单元、线框图、参考点、草图和图层）以及坐标系三轴。此外，文本也不会导出；因此，注释、绘图图例和标签不会出现在3D XML文件中。

出于可视化目的，3D XML Player在*X-Y*平面中模拟一个"地板"，默认等轴测视图将*Z*轴定向为向上方向。在Abaqus/CAE中，默认等轴测视图将*Y*轴定向为向上方向。因此，在默认Abaqus/CAE视图中看起来直立的模型通常在默认3D XML Player视图中看起来侧卧在地板上。为避免此问题，Abaqus会自动在导出的3D XML文件中将模型绕*X*轴旋转90度，绕*Z*轴旋转90度。这些旋转使Abaqus/CAE和3D XML Player中的默认视图等效。

3D XML格式仅显示静止图像。有关导出动画的信息，请参见"保存动画文件"，第49.3节。

**将视口数据导出为3D XML格式文件：**

1. 从主菜单栏，选择**文件**![](../graphics/images/arrow.gif)**导出**![](../graphics/images/arrow.gif)**3DXML**。出现**导出至3DXML文件**对话框。
2. 从对话框底部的**文件过滤器**菜单中，选择以下之一：- **3DXML (*.3dxml)**- **压缩3DXML (*.3dxml)** Abaqus/CAE列出所选目录中所有具有`.3dxml`文件扩展名的文件。
3. 选择要导出图像的文件，或在**文件名**文本字段中输入新文件的名称。
4. 单击**确定**导出图像并关闭**导出至3DXML文件**对话框。
