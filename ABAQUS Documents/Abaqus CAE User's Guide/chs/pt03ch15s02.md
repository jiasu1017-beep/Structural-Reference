# 15.2 进入和退出相互作用模块





您可以在Abaqus/CAE会话期间的任何时候通过点击上下文栏中「Module」列表中的「Interaction」来进入「相互作用」模块。主菜单栏上会出现「Interaction」、「Constraint」、「Connector」、「Special」、「Feature」和「Tools」菜单；上下文栏下方会出现「Step」列表。

要退出「相互作用」模块，请点击「Module」列表中的另一个模块。在退出模块之前，您不必在「相互作用」模块中创建的对象上采取任何特定操作；在通过从主菜单栏选择 ****File**![](../graphics/images/arrow.gif)**Save**** 或 ****File**![](../graphics/images/arrow.gif)**Save As**** 保存整个模型时，它们会自动保存。
