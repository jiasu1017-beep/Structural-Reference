# 10.7.14 从装配文件导入装配





从主菜单栏选择**文件**![](../graphics/images/arrow.gif)**导入**![](../graphics/images/arrow.gif)**装配**，可以从装配文件导入装配。装配文件由所有Abaqus/CAE关联接口创建（参见"我可以使用关联接口做什么？"，第10.1.2节）。

装配中各个零件的几何文件必须位于与装配文件相同的目录中。装配文件始终具有文件扩展名`.eaf`；但是，零件几何文件的文件类型取决于用于写入模型的关联接口：

**CATIA V6关联接口**：CATIA V5零件文件（`.CATPart`）。（CATIA V6零件和装配在导出前转换为CATIA V5格式。）

**CATIA V5关联接口**：CATIA V5零件文件（`.CATPart`）

**SolidWorks关联接口**：ACIS文件（`.sat`）

**Pro/ENGINEER关联接口**：Pro/ENGINEER Elysium中性文件（`.enf_abq`）

**Abaqus/CAE NX关联接口**：NX Elysium中性文件（`.enf_abq`）

以下过程仅在执行模型装配的手动关联导入时才需要。有关创建装配文件和使用关联接口导入模型的各种方法的完整详细信息，请参阅每个关联接口的用户指南。用户指南可从Dassault Systmes知识库获取，网址为[www.3ds.com/support/knowledge-base](http://www.3ds.com/support/knowledge-base)。

**从装配文件导入装配：**

1. 从主菜单栏，选择**文件**![](../graphics/images/arrow.gif)**导入**![](../graphics/images/arrow.gif)**装配**。出现**导入装配**对话框。
2. 从**导入装配**对话框底部的**文件过滤器**菜单中，选择**装配文件 (*.eaf*)**。Abaqus/CAE列出所选目录中具有`.eaf`文件扩展名的文件。
3. 选择包含要导入的装配的文件，然后单击**确定**。Abaqus/CAE显示**从EAF文件导入装配**对话框。
4. 执行以下任一操作：-单击装配名称以导入装配中的所有零件实例。-单击装配名称左侧的箭头以显示装配中的所有零件实例。单击要导入的零件实例的名称。
5. 单击**确定**导入装配或选定的零件实例，并退出**从EAF文件导入装配**对话框。Abaqus/CAE显示一条消息，指示是否未找到任何零件文件或任何零件实例无效或不精确。如有必要，使用查询工具集定位无效和不精确几何的区域。更多信息，请参见"在零件模块中使用查询工具集"，第11.16.4节。
