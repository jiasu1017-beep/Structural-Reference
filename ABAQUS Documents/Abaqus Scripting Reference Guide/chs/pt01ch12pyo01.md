# 12.1 ConnectorBehaviorOption 对象

ConnectorBehaviorOption 对象是其他 ConnectorBehaviorOption 对象的抽象基类型。ConnectorBehaviorOption 对象没有显式构造函数。ConnectorBehaviorOption 对象的成员对所有派生自 ConnectorBehaviorOption 的对象是通用的。

**访问**

```
import section
mdb.models[*name*].sections[*name*].behaviorOptions[*i*]
import odbSection
session.odbs[*name*].sections[*name*].behaviorOptions[*i*]
```

### 12.1.1 成员

ConnectorBehaviorOption 对象没有成员。
