# 17.16.6 对已网格划分的零件、零件实例或区域进行种子布置





要重新对已经进行网格划分的零件、零件实例或区域进行种子布置，您必须首先通过以下方式之一删除网格：
- 在种子布置之前，使用网格删除工具（请参阅["删除网格，" 第17.17.2节](pt03ch17s17s02.md)）从零件、零件实例或感兴趣的区域删除网格。
- 不要自己删除网格； 而是尝试对感兴趣的零件、零件实例或区域进行种子布置。 当您选择要种子布置的零件或边时，Abaqus/CAE会显示类似于以下的窗口： ![](../graphics/mgn-delete-mesh-nls.png) 您可以通过单击**Delete Meshes**删除网格，或者通过按**Cancel**保持网格并退出种子布置过程。 您可以通过切换**Automatically delete meshes invalidated by seed changes**来避免在当前会话的剩余时间内显示此警告消息。 下次尝试对已包含网格的零件、零件实例或区域进行种子布置时，网格将被立即删除而不会显示任何警告。
