# *GEOSTATIC






### *GEOSTATIC获取地静应力场。

此选项用于验证地静应力场是否与模型上的施加载荷和边界条件平衡，并根据需要迭代以获得平衡。

**产品：** Abaqus/Standard  Abaqus/CAE  

**类型：** 历史数据  

**级别：** 步骤  

**Abaqus/CAE：** Step 模块

##### **参考文献：**

- ["地静应力状态，" Abaqus Analysis User's Guide 第 6.8.2 节](../usb/usb-link.md#usb-anl-ageostatstress)

### **可选参数：**

HEAT

如果模型中有使用耦合温度-孔隙压力单元的区域，此参数是相关的；它指定是否在这些区域中建模热传递效应。

设置 HEAT=YES（默认）以指定在这些区域中建模热传递效应。在这种情况下，Abaqus/Standard 将联立求解热传递方程、机械平衡方程和流体流动连续方程。

设置 HEAT=NO 以指定不在这些区域中建模热传递。

如果模型中仅使用耦合孔隙压力-位移单元，则此参数不相关。

UTOL

此参数将调用自动时间增量。

将此参数设置为位移最大变化的容差。Abaqus/Standard 将确保节点处位移的最大绝对值小于容差乘以模型中特征单元长度。如果使用此参数时未指定任何值，则使用默认值 105。如果省略此参数，则对位移值不加限制。
