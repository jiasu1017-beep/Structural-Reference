# *SUBSTRUCTURE DELETE







### *SUBSTRUCTURE DELETERemove a substructure from the substructure library.

This option is used to delete a substructure from a substructure library.

**Product: **Abaqus/Standard  

**Type: **Model data  

**Level: **This option is not supported in a model defined in terms of an assembly of part instances.

##### **Reference:**

- ["Using substructures," Section 10.1.1 of the Abaqus Analysis User's Guide](../usb/usb-link.md#usb-anl-asuperelements)

### **Required parameter: **

TYPE

Set this parameter equal to the TYPE identifier of the substructure being removed.

### **Optional parameter: **

LIBRARY

Set this parameter equal to the name of the substructure library from which the substructure is being deleted. The default library name is *jobname*.

**There are no data lines associated with this option.**
