# 82.8 生成后处理结果报告





**Report Generator** 插件使您能够使用 Abaqus 输出数据库（`.odb`）文件中的模型数据和结果数据创建 HTML 报告。您可以在插件中调整设置，以选择要包含在报告中的内容并自定义报告的外观。您还可以将常用的报告设置保存到报告选项文件中，该文件可以加载以供另一个会话或另一个输出数据库使用。

您可以通过在 Visualization 模块中从主菜单栏选择 ****Plug-ins**![](../graphics/images/arrow.gif)**Tools**![](../graphics/images/arrow.gif)**Report Generator**** 来启动 **Report Generator**。选择要包含的内容并点击 **Generate** 后，Abaqus/CAE 会生成报告并自动在默认浏览器中打开它。

默认情况下，Abaqus/CAE 在当前工作目录中生成报告输出，但您可以指定其他位置。生成报告时，Abaqus/CAE 会覆盖您选择的输出目录中的任何现有报告内容。如果您想保留报告，应将所有报告文件移动到安全的位置，以免覆盖其内容。HTML 报告中包含的所有文件都在报告的 **File Summary** 部分列出。

**Report Generator** 中的许多选项控制您添加到报告中的图形数据类型。您可以在报告中包含以下类型的图形：
- 3D XML 图形：切换 **Include 3DXML** 以包含 3D XML 格式的图形，这是一种使您能够在报告中直接操作三维装配的文件格式。如果您的报告包含 3D XML 格式的内容，必须使用 Internet Explorer 和 3D XML Player 才能正确显示报告。您可以从 Dassault Systèmes 网站 www.3ds.com 下载 3D XML Player。
- 二维自定义视图：切换一个或多个 **View Options** 设置以生成从 Abaqus/CAE 中一个或多个预定义视图或用户定义视图显示装配的图形。
- 其他图形：使用 **Save Current Viewport** 插件创建当前视口的自定义命名图形，然后通过从 **Report Generator** 插件中的 **Additional figures** 列表之一中选择它，将其包含在报告中。每个 **Additional figures** 列表仅显示在相应模块中创建的自定义图形；例如，报告的 **Material Information** 部分只能包含在 Property 模块中创建的其他图形。您可以通过选择 ****Plug-ins**![](../graphics/images/arrow.gif)**Tools**![](../graphics/images/arrow.gif)**Save Current Viewport**** 来启动 **Save Current Viewport** 插件。该插件在 Property 模块、Assembly 模块、Load 模块、Interaction 模块和 Visualization 模块中可用。

**Report Generator** 中的选项被组织成与 HTML 报告中的部分对应的选项卡页面。以下选项可用：

**General**

您可以选择生成 HTML 报告文件的目录、更改报告顶部显示的标题以及更改变报告中显示的用户名。您还可以在报告中以原始大小显示所有图像，或在报告中显示缩略图图像，这些图像链接到原始大小的版本。

其他选项允许您包含或排除报告的顶级组件，而此对话框中其他页面上的设置允许您选择要包含在每个部分中的特定内容。例如，在 **General** 页面上切换 **Include results** 会提示 Abaqus/CAE 在 HTML 报告中包含结果数据；**Results** 页面上的选项允许您控制报告中包含的特定结果数据，如来自单个输出变量的结果或选定的 *X–Y* 绘图。

**Materials**

您可以包含以突出显示材料数据的颜色编码显示装配的图形和 3D XML 格式文件。图形可以从 Abaqus/CAE 中的任何默认视图或用户定义视图中显示。您还可以包含使用 **Save Current Viewport** 插件在 Property 模块中保存的任何其他视口图像。

您还可以在 HTML 报告中将材料表附加到材料图形。默认情况下，材料表描述了装配中定义的材料的材料属性，并提供颜色键以指示图形中每种颜色对应的材料。您可以包含模型中每个超弹性和塑性材料的完整描述；这些描述是材料定义的一部分。

**Instances**

您可以包含以突出显示零件实例的颜色编码显示装配的图形和 3D XML 格式文件。您可以从列表中选择要显示的零件实例，并将零件实例一起显示在单个图像中或单独显示在单独的图像中。图形可以从 Abaqus/CAE 中的任何默认视图或用户定义视图中显示。您还可以包含使用 **Save Current Viewport** 插件在 Assembly 模块中保存的任何其他视口图像。

您可以将零件实例表附加到 HTML 报告中的零件实例图形。零件实例表描述了包含在装配中的实例的零件定义，并提供颜色键以指示图形中每个颜色对应的零件实例。

**Loads**

如果在 Load 模块中使用 **Save Current Viewport** 插件保存了任何视口图形，则可以在报告的 **Load Information** 部分包含这些图形。

**Interactions**

如果在 Interaction 模块中使用 **Save Current Viewport** 插件保存了任何视口图形，则可以在报告的 **Interaction Information** 部分包含这些图形。

**Results**

您可以包含以显示分析中单个步长和增量的装配等值线图的图形和 3D XML 格式文件，用于您选择的场输出变量。图形可以从 Abaqus/CAE 中的任何默认视图或用户定义视图中显示。您还可以包含使用 **Save Current Viewport** 插件在 Visualization 模块中保存的任何其他视口图像。要选择要为其创建等值线图的场输出变量，请点击 **Populate**，然后从出现的对话框中选择变量和图形选项。

您还可以在报告中包含来自当前输出数据库或当前会话的 *X–Y* 绘图。

**Format**

您可以通过使用自定义样式表生成报告并在报告中显示图像（如公司徽标）来自定义报告的外观。
