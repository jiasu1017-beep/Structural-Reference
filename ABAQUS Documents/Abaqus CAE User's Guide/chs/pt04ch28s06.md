# 28.6 欧拉网格运动





欧拉网格运动是一种允许您为某些模型减小欧拉网格尺寸的技术，从而提高分析性能。在某些情况下，分析开始时的欧拉域不足以捕获分析结束时的模型变形。一些典型例子包括：
- 安全气囊，其中充气气体被建模为欧拉材料：气体最初占据一个小区域，但随着安全气囊充气，该区域迅速扩大。
- 弹丸冲击分析，其中弹丸被建模为欧拉材料：弹丸最初占据的区域远离其最终目的地。

在这些情况下，可以在分析过程中调整欧拉域的大小和位置，使其始终捕获感兴趣的部分或材料。默认情况下，欧拉网格是刚性的且固定在适当位置；但启用欧拉网格运动允许欧拉单元在分析过程中进行缩放和平移。欧拉网格运动可以跟随欧拉材料实例的变形（如图28-10所示）或跟随拉格朗日表面（如图28-11所示）。欧拉网格运动的详细描述见["欧拉网格运动"，Abaqus Analysis User's Guide第14.1.3节](../usb/usb-link.md#usb-anl-aeulerianmeshmotion)。
