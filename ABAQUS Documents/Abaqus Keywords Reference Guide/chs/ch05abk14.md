# *EMBEDDED ELEMENT









### *EMBEDDED ELEMENT指定嵌入在模型中"主"单元组中的一个单元或一组单元。

此选项用于指定嵌入在模型中"主"单元组中的一个单元或一组单元。

**产品：**Abaqus/Standard  Abaqus/Explicit  Abaqus/CAE

**类型：**模型数据

**级别：**部件，部件实例，装配

**Abaqus/CAE：**相互作用模块

##### **参考：**

- ["嵌入单元，" Abaqus分析用户指南第35.4.1节](../usb/usb-link.md#usb-cni-pembeddedelement)

### **可选参数：**

ABSOLUTE EXTERIOR TOLERANCE

将此参数设置为嵌入单元上的节点可能位于模型中主单元区域之外的绝对值（以模型使用的单位给出）。如果此参数被省略或值为0.0，则适用EXTERIOR TOLERANCE。

EXTERIOR TOLERANCE

将此参数设置为模型中所有非嵌入单元平均尺寸的分数，嵌入单元的节点可能位于主单元区域之外。默认值为0.05。

如果用户指定了两个外部容差参数，Abaqus将使用两个容差中较小的那个。

HOST ELSET

将此参数设置为数据行上指定单元将被嵌入的主单元集的名称。如果省略此参数，Abaqus将搜索位于指定嵌入单元附近的所有非嵌入单元。

PARTIAL EMBED

如果主单元预计仅部分嵌入嵌入单元，则将此参数设置为YES。默认值为NO。

ROUNDOFF TOLERANCE

将此参数设置为一个小于该值的值，与嵌入节点相关的主单元节点上的权重因子将被清零。小的权重因子将按其初始权重的比例分配给主单元上的其他节点。嵌入节点的位置也将相应调整。默认值为106。
