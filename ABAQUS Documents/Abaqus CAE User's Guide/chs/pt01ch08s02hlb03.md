# 8.2.3 选择图像颜色





当你将画布上的图像直接打印到打印机或文件时，你可以使用**打印**对话框选择图像的颜色。可用的颜色选项如下：

****黑白****

使用此选项在白色背景上打印黑色图像。此选项适用于打印部件、装配体和网格的线框和消隐线图像，包括任何分割和基准几何体。你也可以打印未变形和变形形状绘图的黑白图像。当你选择**黑白**时，Abaqus/CAE 始终在白色背景上打印黑色图像，视口背景被打印为透明或白色。此选项不应用于打印严重依赖颜色的图像，如轮廓图。

****灰度****

使用此选项打印彩色图像的灰度版本，其中每种颜色被近似为灰色阴影。（Abaqus/CAE 将每种颜色转换为 256 种真实灰色阴影之一。）此选项适用于将彩色图像（如轮廓图）打印到黑白激光打印机上。为改善发送到打印机的图像外观，你可能希望关闭背景打印视口（使其显示为白色或透明）。

****颜色****

使用此选项打印你在屏幕上看到颜色的近似值。如果你尝试将彩色图像打印到黑白打印机，打印机会将颜色转换为灰色阴影。

默认情况下，Abaqus/CAE 对光栅打印图像（PNG、TIFF 和光栅格式 PostScript）使用最多 256 种不同颜色。但是，对于 [TIFF 或 PNG](pt01ch08s02hlb08.md) 打印图像，你可以选择允许更多颜色，这会增加文件大小但使图像更接近显示。更多信息，请参阅["自定义保存到 TIFF、PNG 或 SVG 文件中的图像，" 第 8.2.8 节](pt01ch08s02hlb08.md)。

**要选择图像颜色：**

1. 从主菜单栏中，选择****File**![](../graphics/images/arrow.gif)**Print****。**提示：**你也可以单击**文件**工具栏中的![](../graphics/ico_filePrint.png)。**打印**对话框出现。
2. 从**设置**字段中的**渲染**文本框中，选择以下颜色选项之一：
   - 选择**黑白**创建白色背景上的黑色图像。
   - 选择**灰度**打印彩色图像的灰度近似值。
   - 选择**颜色**打印屏幕上颜色的彩色近似值。
3. 当你完成**打印**对话框时，单击**确定**生成所需的输出。Abaqus/CAE 生成输出并关闭**打印**对话框。你的**打印**对话框设置会在会话期间保存。
