# *CONTOUR INTEGRAL






### *CONTOUR INTEGRAL提供轮廓积分估计。

**警告：**对于壳中的弯曲应力，轮廓积分计算不准确。如果需要弯曲应力显著的地方的轮廓积分值，请在评估积分的裂纹尖端区域使用二阶实体单元（C3D20 或 C3D27）而不是壳单元。不应在线性扰动步骤中请求轮廓积分。初始应力不包含在 *J* 积分、应力强度因子和 *T* 应力的评估中（请参阅 ["轮廓积分评估，" Abaqus 分析用户指南第 11.4.2 节](../usb/usb-link.md#usb-anl-acontintegral)，了解更多详细信息）。

[*CONTOUR INTEGRAL](ch03abk74.md) 选项提供基于常规有限元法或扩展有限元法（XFEM）的断裂力学研究的 *J* 积分、![](../graphics/key_eqn00378.gif) 积分、应力强度因子和 *T* 应力的评估。当评估应力强度因子时，该选项还计算裂纹起始时的裂纹扩展方向。可以通过在步骤定义中根据需要重复此选项来评估沿着多条不同裂纹前缘的轮廓积分。

### **必需参数：**

CONTOURS

将此参数设置为一个轮廓数。每个轮廓提供轮廓积分的一次评估。

### **可选参数：**

CRACK NAME

将此参数设置为一个标签，用于引用裂纹。当使用扩展有限元方法时，将此参数设置为在 [*ENRICHMENT](ch05abk25.md) 选项上分配给富集特征的名称。

CRACK TIP NODES

包含此参数以指示裂纹尖端节点被指定为形成裂纹前缘线。如果省略此参数，裂纹前缘线将沿着裂纹前缘节点集的第一个节点形成。（对于每个裂纹前缘节点集，第一个节点将是节点号最小的节点，除非节点集生成为未排序的。）

当指定了 XFEM 参数时，此参数不相关。

DIRECTION

此参数只能与 TYPE=K FACTORS 参数结合使用。

设置 DIRECTION=MTS（默认）以选择最大切向应力准则。

设置 DIRECTION=MERR 以选择最大能量释放率准则。

设置 DIRECTION=KII0 以选择 ![](../graphics/key_eqn00379.gif) 准则。

FREQUENCY

将此参数设置为输出频率（以增量为单位）。除非 FREQUENCY=0，否则输出将始终在每个步骤的最后增量打印。默认值为 FREQUENCY=1。设置 FREQUENCY=0 以禁止输出。

NORMAL

包含此参数以指示裂纹平面法向 ![](../graphics/key_eqn00380.gif) 被指定。省略此参数以指示虚拟裂纹扩展方向 ![](../graphics/key_eqn00381.gif) 被指定。

当指定了 XFEM 参数时，此参数不相关。

OUTPUT

如果省略此参数，则轮廓积分值将被打印在数据（`.dat`）文件中，但不会存储在结果（`.fil`）文件中。

设置 OUTPUT=FILE 以将轮廓积分值存储在结果文件中。

设置 OUTPUT=BOTH 以在数据文件中打印轮廓积分值并将其存储在结果文件中。

RESIDUAL STRESS STEP

使用此参数来考虑残余应力梯度对轮廓积分评估的影响。将此参数设置为一个步骤号，该步骤中最后可用增量的应力数据将被视为残余应力。默认值为 0，在这种情况下，残余应力由指定的初始条件定义。

此参数只能在与 XFEM 参数一起指定时设置为零。

SYMM

包含此参数以指示裂纹前缘在对称平面上定义，仅对一半结构建模。然后将从虚拟裂纹前缘推进计算的能量变化加倍，以计算正确的轮廓积分值。

当指定了 XFEM 参数时，此参数不相关。

TYPE

设置 TYPE=J（默认）以指定 *J* 积分计算。

设置 TYPE=C 以指定 ![](../graphics/key_eqn00378.gif) 积分计算。

设置 TYPE=K FACTORS 以指定应力强度因子的计算。

设置 TYPE=T-STRESS 以指定 *T* 应力计算。

XFEM

包含此参数以指示裂纹使用扩展有限元方法建模为富集特征。
