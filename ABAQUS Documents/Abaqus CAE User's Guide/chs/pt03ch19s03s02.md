# 19.3.2 我的网格自适应何时停止迭代？





当您创建自适应过程时，您定义了一系列要连续运行的作业。您可以通过指定允许的最大分析迭代次数 ![](../graphics/usi_eqn00692.gif) 来指定序列中的作业数量。最大重新网格操作次数将为 ![](../graphics/usi_eqn00692.gif)1，因为在运行最后一个分析作业后将不执行重新网格划分。当您提交重新网格过程时，Abaqus/CAE 运行序列中的每个作业，直到满足以下条件之一：
- 所有活动重新网格划分规则都已满足。有关更多信息，请参见[「什么是重新网格划分规则？，」第 17.13.1 节](pt03ch17s13s01.md)。
- 所有作业都已完成，表明 Abaqus/CAE 已达到您指定的最大分析迭代次数。
- 某个作业未能完成。此失败可能是由于收敛困难或机器资源问题。
