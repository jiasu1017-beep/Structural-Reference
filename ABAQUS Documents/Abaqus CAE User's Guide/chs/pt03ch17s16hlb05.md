# 17.16.5 对边种子施加约束





您可以在定义种子的同时使用**Local Seeds**对话框的**Constraints**标签页对种子施加约束。

**注意：**在约束网格种子时要小心；如果过度约束，当您尝试生成包含四边形或六面体形状单元的网格时，自动网格生成可能无法进行。

**施加约束：**

1. 从主菜单栏，选择 ****Seed**![](../graphics/images/arrow.gif)**Edges****。 然后选择要种子布置的边，并单击鼠标按钮2。 有关执行这些任务的更多信息，请参阅以下章节： - ["通过指定单元数量对边进行种子布置，" 第17.16.2节](pt03ch17s16hlb02.md) - ["通过指定单元尺寸对边进行种子布置，" 第17.16.3节](pt03ch17s16hlb03.md) - ["沿边施加有偏种子布置，" 第17.16.4节](pt03ch17s16hlb04.md) **注意：**如果您选择的边之前使用多种约束的组合进行了种子布置，Abaqus/CAE会提供**As Is**选项，允许您保留所选边上的约束。 Abaqus/CAE在提示区域显示提示以指导您完成该过程。
2. 单击**Local Seeds**对话框的**Constraints**标签页。
3. 选择所需的约束。 选项如下： ****允许单元数量增加或减少**** 此选项使种子完全不受约束。 因此，Abaqus/CAE沿边创建的单元数量可以大于或小于由种子决定的单元数量； 但是，网格生成器会尽可能遵守请求的种子模式。 此选项为自动网格生成器提供了最大的灵活性，因此在生成网格时提供了最大的成功机会。 ****仅允许单元数量增加**** 此选项（默认）对种子进行部分约束，以便沿边的单元数量只能大于或保持与种子决定的数量相同。 位于部分约束种子附近的未约束种子往往表现得好像它们是部分约束的。 ****不允许单元数量改变**** 此选项对种子进行完全约束，以便严格保持您用种子指定的单元数量。 请谨慎使用此选项，因为通常需要调整区域边界上的单元数量来生成网格； 阻止此类调整可能导致网格生成不可能。 **注意：**上述约束选项仅允许您控制沿边的单元数量，而不是沿边节点的位置。 Abaqus/CAE创建的网格会尽可能遵守种子。
4. 单击**OK**。 网格种子会改变形状以指示您选择的约束： - 圆形：未约束种子。 - 三角形：部分约束种子。 三角形指向上方，指示沿边的单元数量只能增加。 - 方形：完全约束种子。
