### 概述

JOINTC单元：
- 用于模拟关节相互作用；以及
- 由局部共旋坐标系中的平移和旋转弹簧以及并联阻尼器组成。

单元formulation的详细信息可在["柔性连接单元，" Abaqus理论指南3.9.6节](../stm/stm-link.md#stm-elm-jointc)中找到。

### 典型应用

提供JOINTC单元是为了模拟两个在几何上（几乎）重合的节点之间的相互作用，这些节点表示具有内部刚度和/或阻尼的关节（如汽车悬架系统中的橡胶衬套），使得关节的第二个节点可以相对于第一个节点轻微位移和旋转。

仅有一个或两个旋转轴且没有相对位移的关节更适合使用REVOLUTE-或UNIVERSAL型MPC进行建模（参见["通用多点约束，" 35.2.2节"](pt08ch35s02aus130.md)）。

使用连接器可以获得类似功能；参见["连接器概述，" 31.1.1节"](pt06ch31s01abo28.md)。

### 定义关节行为

关节行为由并联的线性或非线性弹簧和阻尼器组成，将关节中相对位移和相对旋转的相应分量耦合。如["弹簧，" 32.1.1节"](pt06ch32s01alm37.md)和["阻尼器，" 32.2.1节"](pt06ch32s02alm38.md)中所述定义弹簧和阻尼器行为。

每个弹簧或阻尼器定义定义六个局部方向之一的行为；最多可包含六个弹簧和六个阻尼器定义。如果对于关节中的特定局部相对运动没有给出规范，则假定关节对该分量没有刚度。

关节行为可以在随单元第一个节点运动旋转的局部坐标系中定义（["方向，" 2.2.5节"](pt01ch02s02aus15.md)）。如果未定义局部坐标系，则使用全局坐标系。

您必须将关节行为与一组JOINTC单元相关联。

JOINTC单元的运动学行为在["柔性连接单元，" Abaqus理论指南3.9.6节](../stm/stm-link.md#stm-elm-jointc)中有详细描述。

| **输入文件用法：** | 使用以下选项定义关节行为： |
| --- | --- |
|  | ``` [*JOINT](../key/key-link.md#usb-kws-mjoint), ELSET=*name*, ORIENTATION=*name* [*DASHPOT](../key/key-link.md#usb-kws-mdashpot) [*SPRING](../key/key-link.md#usb-kws-mspring) 最多可出现六个[*SPRING](../key/key-link.md#usb-kws-mspring)和[*DASHPOT](../key/key-link.md#usb-kws-mdashpot)选项。 ``` |

### 在大位移分析中使用JOINTC单元

在大位移分析中，力矩与旋转之间关系的formulation将这些单元的实用性限制为小相对旋转。JOINTC单元的相对旋转幅度应符合小旋转的条件。
