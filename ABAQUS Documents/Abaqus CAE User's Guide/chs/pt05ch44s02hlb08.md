# 44.5.8 标注最小和最大云图值





默认情况下，Abaqus/CAE通过云图颜色变化指示最小和最大云图值的大致位置。实际位置可能在使用所选颜色光谱极值的模型任何部分。您可以切换**Show location**以标注最小和/或最大云图值的节点位置。标注形式为`Min: *value*`和`Max: *value*`，分别带有指向最小值和最大值所在节点、面（对于拼布图）或单元质心的线条。

显示最小值或最大值标注会激活云图图例和**Show min/max values**选项（有关更多信息，请参见["自定义图例，" 第56.1节](pt05ch56s01.md)）。标注是自动创建的——您不能使用**Annotation Manager**编辑它们。标注字体和颜色与图例文本相匹配，节点符号由**Common Plot Options**对话框中的节点符号设置决定。

**注意：**在叠加图中不显示最小值和最大值位置。

**要在模型上标注最小值和最大值：**

1. 找到**Show location**复选框。从主菜单栏中选择****Options**![](../graphics/images/arrow.gif)**Contour****，或单击工具箱中的![](../graphics/ico_plotContourOptions.png)；然后在出现的对话框中单击[**Limits**选项卡](pt05ch44s02hlb07.md)。
2. 分别在**Min**和**Max**区域中切换**Show location**以在模型上标注每个位置。
3. 单击**Apply**在视口中实施您的更改。Abaqus/CAE激活云图图例，包括**Show min/max values**选项。图例显示最小值和最大值，无论您是否在模型上标注其中一个或两个。
4. 如果需要，使用**Viewport Annotation Options**自定义标注字体和颜色（由图例文本选项控制），在图例中隐藏最小值和最大值，或隐藏整个图例。使用**Common Plot Options**对话框的**Labels**选项卡页面上的符号和大小选项来编辑显示的节点符号。默认情况下，您的更改在会话期间保存，并将影响所有后续云图。如果您想在后续会话中保留更改，请将它们保存到文件。有关更多信息，请参见["保存自定义设置以供后续会话使用，" 第55.1.1节](pt05ch55s01s01.md)。
