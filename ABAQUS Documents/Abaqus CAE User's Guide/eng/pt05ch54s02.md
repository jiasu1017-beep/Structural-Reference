# 54.2 Overview of tabular report options







Select ****Report**![](../graphics/images/arrow.gif)**XY****, ****Report**![](../graphics/images/arrow.gif)**Field Output****, or ****Report**![](../graphics/images/arrow.gif)**Free Body Cut**** to customize and generate a tabular listing of *X–Y* data objects, field output variable values, or free body cuts, respectively. Select ****Tools**![](../graphics/images/arrow.gif)**Query**** to generate a tabular report of probe values. The following list identifies the tabular report characteristics that you can customize and the sections where you can find information. You can customize: 
- The *X--Y* data objects or field output variables to include in the report. See ["Selecting report data," Section 54.3](pt05ch54s03.md).
- The free body cuts to include in the report. Abaqus/CAE includes all active free body cuts in the current output database in a free body cut report. You can activate and suppress free body cuts from the **Free Body Cut Manager**. See ["Displaying, hiding, and highlighting free body cuts," Section 67.4](pt06ch67s04.md).
- The probe values to include in the report. See [Chapter 51, "Probing the model](pt05ch51.md)."
- The file name to which the report is written. See ["Specifying your report file name," Section 54.4](pt05ch54hla01.md).
- The layout and width of the report. See ["Controlling report layout, width, format, and coordinate system," Section 54.5](pt05ch54hla02.md).
- How field output or probe values are sorted. For field output, see ["Sorting field output data," Section 54.6](pt05ch54hla03.md); for probe values, see ["Entering tabular data," Section 3.2.7](pt01ch03s02s07.md).
- The format and significant digits of report values. See ["Formatting report values," Section 54.7](pt05ch54hla04.md).
- Whether or not to include minimum and maximum value summaries or column totals. See ["Reporting data values, minimums, maximums, and totals," Section 54.8](pt05ch54hla05.md).
- The form of complex results in the report. See ["Controlling the form of complex results," Section 42.6.9](pt05ch42s04hlb05.md).
