# *TORQUE PRINT





### *TORQUE PRINT打印轴对称滑动线上可传递的总扭矩摘要。

此选项用于获取模型中所有轴对称滑动线上可传递的总扭矩摘要。

**产品：**Abaqus/Standard

**类型：**历史数据

**级别：**步

##### **参考：**

- ["滑动线接触元素," Section 40.4.1 of the Abaqus Analysis User's Guide](../usb/usb-link.md#usb-cni-eslidelineelem)

### **可选参数：**

FREQUENCY

将此参数设置为输出频率，以增量为单位。当使用此选项时，除非FREQUENCY=0，否则扭矩输出将始终在每个步的最后一个增量处打印。默认值为FREQUENCY=1。

**此选项没有关联的数据行。**
