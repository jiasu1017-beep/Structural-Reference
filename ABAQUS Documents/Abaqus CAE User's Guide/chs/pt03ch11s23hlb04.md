# 11.23.4 添加从边缘创建线框特征







从主菜单栏选择**形状**![](../graphics/images/arrow.gif)**线框**![](../graphics/images/arrow.gif)**从边缘**，用线框特征替换视口中现有的壳或实体边缘。无论当前视口中零件的建模空间如何，从边缘创建线框工具始终可用。

您通过从当前零件中选择一个或多个边缘来添加从边缘创建的线框特征。Abaqus/CAE移除沿所选边缘的面，必要时将零件从实体转换为壳，并创建线框来替换移除的边缘。下图说明了从边缘创建线框特征：

![](../graphics/prt-hlp-wire-fromedge.png)

左侧图像显示原始实体零件上选择的边缘。右侧图像显示结果零件。每个选定的边缘都与两个面相关联；Abaqus/CAE移除这些面并将实体转换为壳。选定的边缘以及所有不再与面关联的其他边缘构成了新的线框特征。

从边缘创建的线框特征无法被修改。

**添加从边缘创建的线框特征：**

1. 从主菜单栏，选择**形状**![](../graphics/images/arrow.gif)**线框**![](../graphics/images/arrow.gif)**从边缘**。Abaqus/CAE在提示区域显示引导您完成该过程的提示。**提示：**您也可以使用零件模块工具箱中位于线框工具旁边的![](../graphics/ico_partWireFromEdge.png)工具添加从边缘创建的线框特征。有关零件模块工具箱中工具的图表，请参见"使用零件模块工具箱"，第11.17节。
2. 选择要被线框替换的边缘。
3. 单击**完成**创建线框特征。Abaqus/CAE显示一个对话框，指示将与选定边缘关联的面和单元将被删除。
4. 单击**是**继续或**否**取消该过程。如果您选择**是**，Abaqus/CAE会移除与选定边缘关联的面，并添加线框来替换所有移除的特征边缘。
