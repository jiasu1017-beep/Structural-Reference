# 13.11.8 用平行坐标系约束两个实例





从主菜单栏选择**约束**![](../graphics/images/arrow.gif)**平行坐标系**来施加约束，将可动实例定位，使所选基准坐标系平行于固定实例的所选基准坐标系。所有位置约束都是装配的特征，可以使用特征操作工具集来抑制或删除。

**要用平行坐标系约束两个零件或模型实例：**

1. 从主菜单，选择**约束**![](../graphics/images/arrow.gif)**平行坐标系**。**提示：**您也可以使用装配模块工具箱中的![](../graphics/ico_constraintParallelCsys.png)工具来施加平行坐标系约束。有关装配工具箱中工具的图表，请参见["使用装配模块工具箱，"第13.9节](pt03ch13s09.md)。Abaqus/CAE在提示区域显示提示来引导您完成过程。
2. 从可动实例中选择一个基准坐标系，并从固定实例中选择一个基准坐标系。当Abaqus/CAE提示您从固定实例中选择基准坐标系时，您可以选择在零件或装配模块中创建的基准坐标系。相比之下，当您从可动零件实例中选择基准坐标系时，您只能选择在零件模块中创建的基准坐标系。Abaqus/CAE旋转可动实例，使所选坐标系平行。如果平行坐标系约束与现有约束冲突，Abaqus/CAE显示错误消息并取消操作。要避免冲突，您可以尝试反转将移动的实例和将保持固定的实例的选择。或者，您可以先将现有约束转换为绝对位置，然后重新施加平行坐标系约束。
