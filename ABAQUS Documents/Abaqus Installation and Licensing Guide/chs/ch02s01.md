# 2.1 安装程序







本节总结了用于在本地挂载 DVD 的计算机上执行 Abaqus 安装的命令。如果您在使用远程挂载 DVD 的计算机上安装 Abaqus，请参阅["从远程挂载的 DVD 访问安装文件，"第 D.1 节](ap04s01.md)。由于 Windows 用户帐户控制 (UAC) 会更改共享网络映射驱动器，当 DVD 挂载在共享网络驱动器上时，您无法安装 Abaqus。要解决此问题，请禁用 UAC 或将安装 DVD 内容复制到本地硬盘，然后从本地运行安装程序。

如果您在已安装 Abaqus 以前版本的计算机上执行安装，Abaqus 6.14 版本将使用与以前安装相同的目录结构（详见[附录 B，"Abaqus 发布目录和文件"](ap02.md)）。如果您是首次安装 Abaqus，请为 Abaqus 创建父目录结构。建议此目录结构的路径为 `SIMULIA/Abaqus/`。在整个指南中，`SIMULIA` 目录在目录路径列表中称为 `*simulia_dir*`；Abaqus 文档和许可证安装在 `*simulia_dir*` 中。`SIMULIA/Abaqus/` 目录在目录路径列表中称为 `*abaqus_dir*`；Abaqus 产品安装在 `*abaqus_dir*` 中。

将相应的 DVD 插入驱动器。在 Windows 平台上，会提供启动屏幕，您可以从中启动用于执行 Abaqus 安装的命令、运行系统信息实用程序的命令以及其他安装相关任务。

如果 DVD 未自动挂载，请为您正在安装 Abaqus 的平台发出 DVD 挂载命令。

### 2.1.1 文档安装

所有 Abaqus 版本的在线文档（包括常规和扩展功能版本）提供两种格式：HTML 和 PDF。有多种选项可用于安装文档并向 Abaqus 用户提供文档：
- 在网络服务器上安装 HTML 和 PDF，并使用 Web 服务器软件向用户提供两种格式（这是最常见的选择）。
- 在各用户计算机上安装 PDF 文件的多个副本，用户可以使用 Adobe Acrobat Reader 等 PDF 查看器在本地查看。
- 跳过文档安装程序，直接将 PDF 文件从 DVD 复制到任何计算机或磁盘。

您必须在安装文档的目录中具有写权限。在 Windows 平台上，需要管理员权限才能运行安装程序。

用于安装文档的命令和选项如下所述。有关文档安装选项的更多详细信息，请参阅["Abaqus 文档安装详情，"第 2.2 节](ch02s02.md)。

**安装 Abaqus 文档：**

1. 从 SIMULIA 父目录，执行以下命令或使用以下选项：**Linux 平台** ``` /*doc_dir*/setup [-replay *full_path_to_replay_file*] ``` 其中 *doc_dir* 是 Documentation DVD 挂载时使用的目录或文档安装文件所在的目录。**Windows 平台** - 从启动屏幕，选择**安装 Abaqus 文档**。- 从命令行，键入 ``` *doc_drive*:\setup.exe [-replay *full_path_to_replay_file*] ``` 其中 *doc_drive* 是包含 Documentation DVD 的 DVD-ROM 驱动器。**命令选项** **`-replay *full_path_to_replay_file*`** 使用此选项可执行额外的相同文档安装，而无需为每次安装重新输入相同信息。有关更多信息，请参阅["在多台计算机上重复文档安装，"第 2.2.2 节](ch02s02.md#sgb-chp-install-onlinedocs-options)。
2. 按照出现的对话框中的说明完成安装。选择要安装的文档格式： - **HTML 和 PDF** - **仅 PDF**
3. 如果您安装了 **HTML 和 PDF** 格式并选择了"Abaqus Web 服务器"或"无 Web 服务器"选项，则在安装程序完成时会为您提供指向 HTML 文档的 URL。此 URL 也写入以下位置的信息日志文件： ``` *simulia_dir*/Documentation/installation_info/v6.14/info.log ``` 如果您选择了"现有非 Abaqus Web 服务器"选项，则会将示例 URL 写入 `info.log` 文件。要从 Abaqus 产品访问 HTML 文档，您必须在每次后续 Abaqus 产品安装期间提供有效的 URL。
4. 如果您仅安装了 PDF 文档，则会显示 `index.pdf` 文件的路径。

**从 DVD 复制 PDF 文档（无需运行安装程序）**

1. 在文档 DVD 上找到 `pdf_docs/` 目录。复制此目录的全部内容。
2. 要查看文档，请在 PDF 查看器或 Web 浏览器中打开 `index.pdf` 文件。

您可以按照以下条款复制 PDF 文档：
- 此材料的使用受您的 Abaqus 软件许可证协议的约束。
- 对文档保密，并尽最大努力防止未经授权披露或使用任何 Abaqus 程序，包括该程序的 Abaqus 文档。
- 对所有 Abaqus 程序（包括 Abaqus 文档）采取与您自己的同类信息相同的谨慎程度处理，您不希望公开披露或未经授权访问或使用的此类信息。
- 不得制作或允许制作任何超过您的 Abaqus 软件许可证协议授权数量的任何 Abaqus 程序（包括 Abaqus 文档）的额外副本。

### 2.1.2 Dassault Systèmes 许可证服务器安装

您可以根据所拥有的许可证文件的类型，使用 FLEXnet 或 Dassault Systèmes 许可证与 Abaqus 配合使用。Abaqus 产品安装程序允许您选择要安装的许可证服务器类型。对于 Dassault Systèmes 许可证，Abaqus 安装程序将在任何具有支持平台（操作系统）的计算机上安装 Dassault Systèmes 许可证服务器 (DSLS) 软件。

DSLS 软件包包括*Dassault Systèmes 许可证服务器安装和配置指南*（`DSLS.pdf`），其中包含配置许可证服务器和管理许可证的说明。请参阅此指南了解所有 DSLS 许可证操作。您可以在以下位置找到 Licensing & Products DVD 上的 `DSLS.pdf` 文件：

```
/*dvd_dir*/DSLS/DSLS.pdf
```

要安装 Dassault Systèmes 许可证服务器，您必须在 Windows 上是 Administrator或在 UNIX/Linux 上是 root。Dassault Systèmes 许可证服务器仅与 Abaqus 6.12 及更高版本兼容。

不支持 VMware 等虚拟机。无法在虚拟机上安装或运行 Dassault Systèmes 许可证服务器。

安装后，您可以使用 `dslsstat` 实用程序检查 DSLS 状态；请参阅["对 Dassault Systèmes 许可证服务器使用 `dslsstat` 实用程序，"第 3.9 节](ch03s09.md)。

新的许可证文件允许您运行当前版本的所有维护交付版本以及接下来的两个主要版本。

除非您使用的是冗余许可证服务器配置，否则 Dassault Systèmes 许可证服务器应仅安装在一台计算机上。您为文档和许可证选择的服务器计算机应能被您计划运行 Abaqus 的网络上的所有计算机访问，并且应该是稳定的机器，不会频繁重启或关闭。

建议大多数用户选择自动配置，特别是那些主要职责不是计算机系统或网络管理的用户。您可以允许安装程序自动安装、配置和启动 Dassault Systèmes 许可证服务器。

### 在 UNIX/Linux 上安装 Dassault Systèmes 许可证服务器

您可以从 Abaqus 产品安装程序内部或直接从 DVD 启动 DSLS 安装程序：
- 要使用 Abaqus 产品安装程序，请按照以下步骤操作： 1. 从 SIMULIA 父目录，执行以下命令： ``` /*dvd_dir*/setup ``` 其中 *dvd_dir* 是 Licensing & Products DVD 挂载时使用的目录或许可证安装文件所在的目录。 2. 在**安装类型**面板中，选择**Dassault Systèmes 许可证服务器 (DSLS)**。 3. 按照出现的对话框中的说明完成安装。
- 要直接从 DVD 启动 DSLS 安装程序，请按照以下步骤操作： 1. 为您的计算机设置 `DISPLAY` 环境变量。 2. 将目录更改 (`cd`) 到以下 OS/平台特定的目录之一： ``` /*dvd_dir*/DSLS/AIX/ /*dvd_dir*/DSLS/SUN/ /*dvd_dir*/DSLS/RedHat/ /*dvd_dir*/DSLS/Suse/ ``` 3. 执行文件 `startInstLicServ`。 4. 按照出现的对话框中的说明完成安装。

安装完成后，您必须使用 Dassault Systèmes 许可证管理工具在 Dassault Systèmes 许可证服务器上注册 Abaqus 许可证；请参阅*Dassault Systèmes 许可证服务器安装和配置指南*中的"注册产品许可证"。您必须先注册许可证，然后才能继续 Abaqus 产品安装。

### 在 Windows 上安装 Dassault Systèmes 许可证服务器

您可以通过以下任何方式在 Windows 上启动 DSLS 安装程序：
- 从 DVD 启动屏幕，选择**安装 Dassault Systèmes 许可证服务器 (DSLS)**。
- 在 Windows 资源管理器中，浏览到 DVD 并双击 DSLS 安装程序： ``` *dvd_drive*:\DSLS\SetupDSLSmsi.exe ```
- 要从 Abaqus 产品安装程序内部启动 DSLS 安装程序，请按照以下步骤操作： 1. 从启动屏幕，选择**安装 Abaqus 产品和许可证**，或从命令行执行 `setup.exe` 文件： ``` *dvd_drive*:\setup.exe ``` 2. 在**安装类型**面板中，选择**Dassault Systèmes 许可证服务器 (DSLS)**。 3. 按照出现的对话框中的说明完成安装。

安装完成后，您必须使用 Dassault Systèmes 许可证管理工具在 Dassault Systèmes 许可证服务器上注册 Abaqus 许可证；请参阅*Dassault Systèmes 许可证服务器安装和配置指南*中的"注册产品许可证"。您必须先注册许可证，然后才能继续 Abaqus 产品安装。

### 2.1.3 FLEXnet 许可证安装

您必须拥有 Abaqus 6.14 FLEXnet 许可证文件才能设置 Abaqus FLEXnet 许可证服务器。FLEXnet Abaqus 6.14 许可证使用 FLEXnet Licensing Version 11.6.1。许可证文件允许您运行后续的 Abaqus 6.14 维护交付版本、可追溯到 Abaqus 6.2-7 的以前版本，以及两个未来的主要版本。例如，Abaqus 6.14-1 的新许可证文件可用于运行所有 Abaqus 6.14 维护交付版本（Abaqus 6.14-2、Abaqus 6.14-3 等）以及网络上仍安装的任何以前版本的 Abaqus（Abaqus 6.13、Abaqus 6.12 等）。要申请 Abaqus 许可证文件，请联系您当地的销售办事处。

FLEXnet Abaqus 许可证应仅安装在一台计算机上，除非您使用的是冗余许可证服务器配置。您为文档和许可证选择的服务器计算机应能被您计划运行 Abaqus 的网络上的所有计算机访问，并且应该是稳定的机器，不会频繁重启或关闭。

在安装 Abaqus 产品之前，您必须拥有与 Abaqus 6.14 兼容的 FLEXnet 网络许可证服务器，并在您的网络上运行。

从以下方案中选择要执行的许可证安装：
- 自动配置推荐给大多数用户，特别是那些主要职责不是计算机系统或网络管理的用户。允许安装程序自动为您安装、配置和启动 Abaqus FLEXnet 许可证服务器。如果以前使用 Abaqus 许可证安装程序安装和配置了使用 FLEXnet Licensing 以前版本的现有许可证服务器，则在安装过程中会自动升级该服务器。对于冗余许可证服务器，安装程序将自动安装和配置 Abaqus 许可证服务器，但您需要手动启动每个服务器。使用此方法便于维护，以便可以自动配置未来升级。
- 手动配置是与其他供应商的 FLEXnet 许可证文件组合所必需的。在安装开始时，选择**仅安装许可证实用程序**。您需要手动配置和启动 Abaqus FLEXnet 许可证服务器。使用此方法可以让系统管理员更好地控制。Abaqus 6.14 需要 `lmgrd` Version 11.6.1 或更高版本。如果您使用 FLEXnet Licensing 组合许可证文件将 Abaqus 许可证功能与其他产品集成，请确保您的 `lmgrd` 版本满足此要求。有关更多信息，请参阅 FLEXnet Licensing End User Guide Version 11.6，可从 **Support** 页面上的 **Licensing** 部分下载，网址为 [www.3ds.com/simulia](http://www.3ds.com/simulia)。

用于安装 Abaqus FLEXnet 许可证服务器以及停止和删除许可证服务器的命令和选项如下所述。有关 FLEXnet 许可证安装选项的更完整讨论，请参阅["Abaqus FLEXnet 许可证安装详情，"第 2.3 节](ch02s03.md)。要替换现有或过期的 FLEXnet 许可证文件，请参阅["更新 Abaqus FLEXnet 许可证文件，"第 3.2 节](ch03s02.md)；有关安装冗余许可证服务器的详细信息，请参阅["冗余 FLEXnet 许可证服务器配置，"第 3.4 节](ch03s04.md)。

**安装 Abaqus FLEXnet 许可证：**

1. 从 SIMULIA 父目录，执行以下命令或使用以下选项：**Linux 平台** ``` /*dvd_dir*/setup ``` 其中 *dvd_dir* 是 Licensing & Products DVD 挂载时使用的目录或许可证安装文件所在的目录。**Windows 平台** - 从启动屏幕，选择**安装 Abaqus 产品和许可证**。- 从命令行，键入 ``` *dvd_drive*:\setup.exe ``` 其中 *dvd_drive* 是包含 Licensing & Products DVD 的 DVD-ROM 驱动器。
2. 在**安装类型**面板中，选择**SIMULIA FLEXnet 许可证服务器**。
3. 如果您在此系统上有现有的 Abaqus FLEXnet 许可证服务器，安装程序会尝试停止许可证服务器；如果无法停止，则提供仅安装许可证管理工具的选项。如果仅安装 FLEXnet 许可证管理工具，则必须手动配置许可证服务器。如果安装程序无法停止现有许可证服务器，而您想要完成完整的自动许可证服务器安装，则需要使用以下过程所述的步骤停止现有服务器。
4. 按照出现的对话框中的说明完成许可证安装。
5. 在 Linux 平台上，默认情况下，Abaqus FLEXnet 许可证服务器在系统重启后不会自动重启。如果要安装新的 FLEXnet 许可证服务器，并希望 FLEXnet 许可证服务器自动重启，则必须将 Abaqus 许可证命令添加到系统启动脚本中。用于启动许可证服务器的命令写入 ``` `*simulia_dir*/License/installation_info/license_startup.txt` ``` 在 Windows 平台上，自动配置的许可证服务器在重启后会自动启动。

**停止和删除现有 FLEXnet 许可证服务器**

如果您有在系统上运行的以前版本 Abaqus 的现有 FLEXnet 许可证服务器，则需要在安装新许可证服务器之前将其删除。许可证安装程序将尝试自动删除现有服务器。如果无法删除，您应使用以下过程之一删除现有服务器：

**Linux 平台**

1. 输入命令 ``` `*simulia_dir*/License/lmdown -c [*port*]@*license_server_host* -vendor ABAQUSLM` ``` 其中 *license_server_host* 是 FLEXnet 许可证服务器的主机名。如果许可证服务器使用的端口不在 27000 到 27009 之间，则还必须指定端口号。
2. 通过运行以下命令验证服务器已停止 ``` `*simulia_dir*/License/lmstat -c [*port*]@*license_server_host*` ``` 验证没有报告 `ABAQUSLM` 供应商守护程序。如果 Abaqus FLEXnet 许可证服务器已成功停止且没有其他供应商守护程序在指定主机上运行，则可能会出现错误状态消息。

**Windows 平台**

1. 从**开始**菜单，选择****程序**![](../graphics/images/arrow.gif)**Abaqus *版本***![](../graphics/images/arrow.gif)**许可证****，其中 *版本* 指 Abaqus 版本，以显示 **LMTOOLS** 对话框。
2. 在**服务/许可证文件**选项卡页上验证已启用**使用服务配置**。
3. 选择**启动/停止/重新读取**选项卡，选择相应的 Abaqus 6.*x* 许可证服务器（其中 6.*x* 指 Abaqus 版本），然后单击**停止服务器**。
4. 选择**配置服务**选项卡，选择相应的 Abaqus 6.*x* 许可证服务器，然后单击**删除服务**。
5. 选择****文件**![](../graphics/images/arrow.gif)**退出****以关闭 **LMTOOLS** 对话框。

### 2.1.4 Abaqus 产品安装

要安装 Abaqus 产品，您必须在网络上安装并运行与 Abaqus 6.14 兼容的网络许可证服务器。还建议您在安装 Abaqus 产品之前执行文档安装，以获取 HTML 文档的 URL（或 PDF 文件的基本安装目录）并启用从 Abaqus 访问文档。系统将提示您提供以下关于文档和许可证安装的信息：
- 许可证服务器的主机名（必需）。
- 以下之一（取决于安装了哪些格式的文档）： - Abaqus HTML 文档的 URL（可选，但建议使用）。如果您在产品之前未安装 Abaqus HTML 文档或未指定 URL，则必须执行其他步骤才能从 Abaqus 产品访问 HTML 文档（请参阅["安装后设置文档 URL，"第 C.2 节](ap03s02.md)）。 - 安装 PDF 文档文件的基本安装目录 (`*simulia_dir*`) 的路径（如果仅安装了 PDF 或您更喜欢 PDF 而非 HTML）。在安装程序提示时输入此路径将配置 `abaqus doc` 命令打开 PDF 而非 HTML 文档。请参阅["Abaqus 文档，"Abaqus Analysis User's Guide 第 3.2.11 节](../usb/usb-link.md#usb-int-ddocproc)，了解更多信息。

用于安装 Abaqus 产品的命令和选项如下所述。有关 Abaqus 产品安装选项的更完整讨论，请参阅["Abaqus 产品安装详情，"第 2.4 节](ch02s04.md)。

**安装 Abaqus 产品：**

1. 从 Abaqus 父目录，执行以下命令或使用以下选项：**Linux 平台** ``` /*prod_dir*/setup [-jre system|*path_to_jre*] [-replay *full_path_to_replay_file*] [-sysinfo] [-help] ``` 其中 *prod_dir* 是 Licensing & Products DVD 挂载时使用的目录或产品安装文件所在的目录。**Windows 平台** - 从启动屏幕，选择**安装 Abaqus 产品和许可证**。- 从命令行，键入 ``` *prod_drive*:\setup.exe [-replay *full_path_to_replay_file*] [-sysinfo] [-help] ``` 其中 *prod_drive* 是包含 Licensing & Products DVD 的 DVD-ROM 驱动器。您不能使用通用命名约定 (UNC) 路径来运行命令行安装程序。**命令选项** **`-jre system|*path_to_jre*`** 指定用于安装的 Java Runtime Environment (JRE)。此选项通常不需要，因为大多数支持平台的 JRE 都随安装盘提供并默认使用。指定值 `system` 以使用系统 JRE（如果已安装）；或指定用户定义 JRE 的完整路径。要使用系统 JRE，`java` 命令必须位于您的 PATH 环境变量中，且 JRE 必须为 Version 1.5 或更高版本。 **`-replay *full_path_to_replay_file*`** 使用此选项可执行额外的相同安装，而无需为每次安装重新输入相同信息。有关更多信息，请参阅["在多台计算机上重复产品安装，"第 2.4.3 节](ch02s04.md#sgb-chp-install-prod-options)。 **`-sysinfo`** 此选项列出有关系统软件和硬件资源的信息。如果使用此选项，则不会安装 Abaqus，且所有其他命令选项都会被忽略。 **`-help`** 显示 `setup` 选项。
2. 在**安装类型**面板中，选择 Abaqus **产品**。
3. 按照出现的对话框中的说明完成产品安装。
