# 12.7.7 显示超弹性材料行为的X–Y图





### 12.7.7 显示*X--Y*超弹性材料行为图

Abaqus/CAE允许您通过使用选定的应变能势自动创建响应曲线来评估超弹性材料行为。曲线拟合完成后，Abaqus/CAE打开Visualization模块并显示测试结果的*X–Y*图，以及包含每个应变能势稳定性限值的对话框。您可以查看结果并根据需要调整材料。有关更多信息，请参见["评估超弹性和粘弹性材料行为，"第12.4.7节"](pt03ch12s04s07.md)。

**显示超弹性材料行为的*X–Y*图：**

1. 从主菜单栏，选择****Material**![](../graphics/images/arrow.gif)**Evaluate**![](../graphics/images/arrow.gif)***material name*****。您选择的材料必须包含超弹性材料数据。
   **提示：**您也可以在**Material Manager**中选择材料名称，然后点击**Evaluate**。出现**Evaluate Material**对话框。
2. 如果您选择的超弹性材料也包含粘弹性材料数据，请切换**Perform hyperelastic evaluation**（如果尚未选择）。如有需要，您也可以评估材料的粘弹性行为。有关更多信息，请参见["显示粘弹性材料行为的*X--Y*图，"第12.7.8节"](pt03ch12s07hlb08.md)。
3. 在**Available Input Data**字段中，执行以下操作：
   1. 选择您选择的**Source**选项：
      - 如果您希望Abaqus从材料定义中指定的实验数据计算必要的应变能势系数，请选择**Test data**。
      - 如果您希望Abaqus使用材料定义中指定的系数，请选择**Coefficients**。
   2. 如果在上一步中选择了**Test data**，请指定您希望Abaqus用于计算应变能势系数的测试数据类型。（只有您在材料定义中为其指定了数据的数据类型会出现在列表中。）
   3. 如果您打算评估**Marlow**应变能势，请指定Abaqus将用于定义偏应体响应的测试数据类型。您还可以指定是否应使用压缩、拉伸或两种类型的测试数据，以及是否应使用体积测试数据来定义体积响应。（有关更多信息，请参见Abaqus Analysis User's Guide的第22.5.1节"橡胶类材料的超弹性行为"中的"Marlow形式"。）**注意：**如果您的超弹性材料模型包含横向名义应变、温度相关数据或场变量，则**Marlow**将是唯一可用于评估的应变能势。
4. 从**Standard Tests**列表中，选择您要使用材料定义中的数据进行响应计算的一个或多个测试。
5. 对于您选择的每个测试，输入最小和最大应变值，这些值将作为应力-应变响应曲线的上限和下限。
6. 点击**Strain Energy Potentials**标签，并执行以下操作：
   - 如果您选择了**Test data**作为数据源，则会出现所有可用应变能势的列表。从列表中，选择您希望Abaqus应用于实验数据的一个或多个。有关Abaqus中可用的应变能势的更多信息，请参见Abaqus Analysis User's Guide的第22.5.1节"橡胶类材料的超弹性行为"中的"应变能势"。
   - 如果您选择了**Coefficients**作为数据源，则会出现材料定义中指定的应变能势名称。您可以简单地查看信息并继续下一步。
7. 如果您评估的材料也包含粘弹性材料属性，请点击**Viscoelastic**标签；您可以关闭**Perform viscoelastic evaluation**，或选择粘弹性评估选项。有关更多信息，请参见["显示粘弹性材料行为的*X--Y*图，"第12.7.8节"](pt03ch12s07hlb08.md)。
8. 点击**OK**开始响应计算。如果在提取材料系数期间评估因非线性曲线拟合问题而失败，Abaqus/CAE会显示一个包含数据（`.dat`）文件名称的对话框；数据文件的路径打印在消息区域中。数据文件提供有关每个遇到的问题的详细信息。（有关数据文件的更多信息，请参见Abaqus Analysis User's Guide的第4.1.1节"输出"。）如果Abaqus成功完成测试，Abaqus/CAE进入Visualization模块并在新的视口中显示测试结果的*X--Y*图。（有关*X--Y*图的信息，请参见[第47章，"X--Y绘图"](pt05ch47.md)。"）数据对象出现在***X--Y* Data Manager**中；您可以将它们复制到输出数据库，或执行您在Visualization模块中对其他*X--Y*数据可以执行的任何任务。此外，Abaqus/CAE显示一个信息对话框，其中包含每个超弹性应变能势的稳定性限值和系数。如果执行了粘弹性评估，对话框还会显示粘弹性材料参数。Abaqus/CAE在消息区域显示包含所有材料评估信息的数据（`.dat`）文件的路径。
9. 如有需要，返回Property模块编辑材料数据或评估其他材料。例如，如果超弹性材料的**Strain energy potential**之前设置为**Unknown**，您可以使用评估结果使用最佳应变能势完成材料定义。
