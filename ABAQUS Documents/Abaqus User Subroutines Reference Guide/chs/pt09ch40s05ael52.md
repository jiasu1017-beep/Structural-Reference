# 40.5.2 轴对称刚表面接触单元库





**产品：** Abaqus/Standard  

##### **参考资料**

- ["解析刚表面定义，" 第2.3.4节](pt01ch02s03aus19.md)
- ["刚表面接触单元，" 第40.5.1节](pt09ch40s05alm67.md)
- [*RIGID SURFACE](../key/key-link.md#usb-kws-mrigidsurf)
- [*INTERFACE](../key/key-link.md#usb-kws-minterface)

### 概述

本节提供Abaqus/Standard中可用轴对称刚表面接触单元的参考。

### 单元类型

| IRS21A | 与一阶轴对称单元一起使用的轴对称刚表面接触单元 |
| --- | --- |

| IRS22A | 与二阶轴对称单元一起使用的轴对称刚表面接触单元 |
| --- | --- |

##### 活跃自由度

除最后一个节点外的每个节点处为1、2

最后一个节点处为1、2、6，刚体参考节点的运动

##### 其他解变量

每个节点处两个与接触应力相关的变量。

### 所需的节点坐标

*r*、*z*

### 单元属性定义

| **输入文件用法：** | 使用以下选项定义单元与之相互作用的表面： |
| --- | --- |
|  | ``` [*RIGID SURFACE](../key/key-link.md#usb-kws-mrigidsurf) ``` 使用以下选项定义刚表面单元的截面属性： ``` [*INTERFACE](../key/key-link.md#usb-kws-minterface) ``` |

### 基于单元的加载

无。

### 单元输出

| S11 | 单元与刚表面之间沿刚表面法线方向的压力。 |
| --- | --- |

| S12 | 单元与刚表面之间沿刚表面切线方向的剪切应力分量。 |
| --- | --- |

| E11 | 在最接近单元上积分点的位置处，沿刚表面法线方向的表面分离。 |
| --- | --- |

| E12 | 表面的累积相对切向位移。 |
| --- | --- |

### 单元上的节点排序

IRS21A的前两个节点和IRS22A的前三个节点在变形网格上。最后一个节点是定义刚体运动的刚体参考节点。

### 输出积分点编号

积分点位于位于变形模型表面上的节点处，并相应编号。
