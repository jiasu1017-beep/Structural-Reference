# 1.2.2 无约束单元分片的特征值提取







**产品：**Abaqus/Standard  

### I. 连续体单元

### 测试的单元

连续体单元（不包括耦合温度-位移和孔隙压力-位移单元）。

### 问题描述

模型由与["分片测试，" 第 1.5 节"](ch01s05.md)中定义的测试相同的单元分片组成。第一步对没有边界条件的模型进行特征值分析。第二步在所有四条边上施加均匀压力载荷并设置 NLGEOM 参数。第三步对没有边界条件的预应力模型进行特征值分析。仅打印第一步和第三步的结果。

### 结果与讨论

对于所有单元，步骤 1 和 3 的零能模态数量相同，与["单无约束单元的特征值提取，" 第 1.2.1 节"](ch01s02abv01.md)中给出的刚体模态数量匹配。
