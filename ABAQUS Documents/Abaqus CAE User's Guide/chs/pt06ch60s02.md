# 60.2 更新导入模型中的几何参数





参数更新功能允许双向关联导入，在其中您可以修改已导入 Abaqus/CAE 的模型的几何特征尺寸，然后将新尺寸传播回原始 CAD 系统的模型文件。此双向导入功能目前仅适用于 CATIA V5 关联接口、CATIA V6 关联接口和 Pro/ENGINEER 关联接口。

要使用参数更新功能，您必须首先使用 CAD 系统（CATIA V5、CATIA V6 或 Pro/ENGINEER）指示在 Abaqus/CAE 中可以修改哪些模型尺寸。可修改的尺寸必须根据特征级参数定义，这些参数会随模型一起导入 Abaqus/CAE。例如，参数可能与模型中孔的半径相关联；修改该参数的值也会修改孔的半径。每个参数名称必须以字符串 `ABQ_` 开头才能被导入 Abaqus/CAE。此外，为了避免用相同名称覆盖现有参数，每个参数名称必须唯一。

CATIA V5 关联接口、CATIA V6 关联接口和 Pro/ENGINEER 关联接口使用特殊的参数（`.par_abq`）文件将参数传递到 Abaqus/CAE。有关在导入的 CATIA V5、CATIA V6 或 Pro/ENGINEER 模型中定义可修改参数的更多信息，请参阅 CATIA V5 关联接口、CATIA V6 或 Pro/ENGINEER 关联接口用户指南。这些指南可从 Dassault Systèmes 知识库获取，网址为 [www.3ds.com/support/knowledge-base](http://www.3ds.com/support/knowledge-base)。

您可以使用 Abaqus/CAE 中的 **CAD Parameters** 对话框为导入的参数分配新值。当您更改参数值时，相关的几何特征会在 Abaqus/CAE 模型和保存的 CAD 模型中重新生成。此对话框还使您能够在更改 CAD 参数后保持 CAD 软件在后台运行。在后台保持 CAD 软件运行可以提高双向参数更新的性能，但此选项会在会话剩余时间内占用您的 CAD 软件许可证。

**更新导入模型中的几何参数的步骤：**

1. 执行以下操作之一：
   - 使用 CATIA V5 关联接口从 CATIA V5 导入模型。
   - 使用 CATIA V6 关联接口从 CATIA V6 导入模型。
   - 使用 Pro/ENGINEER 关联接口从 Pro/ENGINEER 导入模型。

2. 打开 **CAD Parameters** 对话框：
   - 在 Part 模块中，选择 ****Tools**![](../graphics/images/arrow.gif)**CAD Parameters****。
   - 在 Assembly 模块中，选择 ****Tools**![](../graphics/images/arrow.gif)**CAD Interfaces**![](../graphics/images/arrow.gif)**CAD Parameters****。
   **CAD Parameters** 对话框显示所有可修改的参数。

3. 如果从 CATIA V5 或 Pro/ENGINEER 导入参数，请单击参数名称以在视口中突出显示受该参数影响的模型部分。

4. 要修改参数值，请单击**值**列中的相应单元格，然后输入新值。

5. 如果从 Pro/ENGINEER 导入参数，您可以切换打开 **Keep CAD software running in the background after parameter update**。如果您执行双向导入或进行其他更新，此选项可以提高性能。

6. 当您更改了所有必要的参数值后，执行以下操作之一：
   - 单击 **Update** 以基于新参数值在 Abaqus/CAE 中重新生成模型。保存的 Pro/ENGINEER 零件（`.prt`）文件中的几何也会更新。当您执行更新时，原始模型不得在 Pro/ENGINEER 中打开。
   - 单击鼠标按钮 3，然后选择 **Write to File** 以创建更新的参数文件而不更新模型几何。此文件可用于使用 Abaqus Scripting Interface 手动更新参数。有关参数文件的更多信息，请参阅 Pro/ENGINEER 关联接口用户指南。
   - 单击 **Defaults** 以将 **CAD Parameters** 表中的所有参数重置为当前 Abaqus/CAE 模型中的值。
