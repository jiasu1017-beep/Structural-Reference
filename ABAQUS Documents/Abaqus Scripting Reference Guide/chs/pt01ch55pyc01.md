# 55.19 writeXYReport

### 55.19.1 writeXYReport(...)

此方法将 XYData 对象写入用户定义的 ASCII 文件。

**路径**

```
session.writeXYReport
```

**必需参数**

*fileName*

一个 String，指定要写入 *X–Y* 数据的文件名。

*xyData*

一个 XYData 对象序列，要写入输出文件。

**可选参数**

*appendMode*

一个 Boolean，指定是否将 *X–Y* 数据追加到现有文件。默认值为 ON。

**返回值**

None

**异常**

None。
