# 42.8.4 创建跟随圆上三个节点的坐标系





您可以选择位于圆弧上的三个节点来定义附加到模型的坐标系。当模型在空心圆柱轴线上或空心球中心没有任何节点，但您希望在这些位置定义坐标系时，此方法非常有用。由三个节点定义的圆的中心是坐标系的原点。从原点到节点1的线是*X*轴（对于直角坐标系）或*R*轴（对于圆柱坐标系或球坐标系）；因此，您选择节点的顺序决定了![](../graphics/usi_eqn00065.gif)增加的方向。三个共线节点是无效的。圆的法线平行于*Z*轴（对于直角或圆柱坐标系）或![](../graphics/usi_eqn00226.gif)轴（对于球坐标系）。坐标系的运动由三个节点的平移运动定义。

跟随模型上节点的坐标系的视觉显示会更新以反映所应用的任何变形比例因子；但是，基于这些坐标系的结果转换使用实际的、未缩放的变形。

**创建跟随圆上三个节点的坐标系：**

1. 从主菜单栏，选择****工具**![](../graphics/images/arrow.gif)**坐标系**![](../graphics/images/arrow.gif)**创建****。此时将出现**创建坐标系**对话框。**提示：**您也可以使用工具箱中的![](../graphics/ico_dtmCsys.png)工具或当前输出数据库结果树中的**会话坐标系**容器来创建坐标系。
2. 在**创建坐标系**对话框中：1. 接受坐标系的默认名称，或在文本字段中输入您选择的名称。2. 切换**跟随圆上3节点的坐标系**以指定坐标系的运动。3. 选择以下之一作为坐标系类型：- **直角**：*X*、*Y*和*Z*轴分别解释为1、2和3轴。- **圆柱**：*R*、![](../graphics/usi_eqn00065.gif)和*Z*轴分别解释为1、2和3轴。- **球面**：*R*、![](../graphics/usi_eqn00065.gif)和![](../graphics/usi_eqn00226.gif)轴分别解释为1、2和3轴。4. 点击**继续**。Abaqus/CAE在提示区域显示提示，以帮助您定义坐标系轴。
3. 在视口中选择在圆弧上的三个节点。坐标系出现在视口中、**坐标系管理器**的列表中，以及当前输出数据库结果树中的**会话坐标系**容器下。Abaqus仅将用户定义的坐标系保存到输出数据库关闭为止，除非您将它们保存到输出数据库文件。坐标系一旦创建就不能编辑；您可以重命名或删除尚未保存到输出数据库的坐标系。
