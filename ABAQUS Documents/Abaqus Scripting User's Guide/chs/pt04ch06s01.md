# 6.1 命令概述







在 Abaqus/CAE 中有两类命令：kernel 命令和 GUI 命令。

**Kernel 命令**

Kernel 命令用于构建、分析和后处理有限元模型。Kernel 命令记录在 [Abaqus Scripting Reference Guide](../ker/ker-link.md#ker) 中。

**GUI 命令**

GUI 命令由用户界面用于处理从用户收集的输入，并构造发送到 kernel 执行的内核命令字符串。GUI 命令记录在 [Abaqus GUI Toolkit Reference Guide](../gui/gui-link.md#gui) 中。
