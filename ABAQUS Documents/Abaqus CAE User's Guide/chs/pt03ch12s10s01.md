# 12.10.3 指定非弹性热分数







您可以指定非弹性热分数，以在绝热或完全耦合热应力分析中将非弹性能量耗散作为热源提供。

非弹性热分数通常用于模拟涉及大量非弹性应变的高速制造过程，其中材料因变形而产生的加热显著影响温度依赖性材料属性。产生的热量在热平衡方程中被视为体积热通量源项。

有关更多信息，请参见以下章节：
- "完全耦合热应力分析"中的"作为热源的非弹性能量耗散"，Abaqus分析用户指南第6.5.3节
- "绝热分析"，Abaqus分析用户指南第6.5.4节

**指定非弹性热分数：**

1. 从**编辑材料**对话框的菜单栏中，选择**热**![](../graphics/images/arrow.gif)**非弹性热分数**。（有关显示**编辑材料**对话框的信息，请参见"创建或编辑材料"，第12.7.1节。）
2. 在**分数**字段中，输入作为每单位体积热通量出现的非弹性耗散率分数。如有需要，该分数可以包含单位转换因子。默认值为0.9。
3. 单击**确定**关闭**编辑材料**对话框。或者，您可以从**编辑材料**对话框的菜单中选择其他材料行为进行定义（有关更多信息，请参见"浏览和修改材料行为"，第12.7.2节）。
