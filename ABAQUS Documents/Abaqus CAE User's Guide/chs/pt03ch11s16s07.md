# 11.16.7 在零件模块中使用集合工具集





通过从零件选择几何创建的集合称为零件集合，您使用集合工具集来创建和管理零件集合。您可以在零件模块中使用零件集合来选择应由几何编辑工具集修复的区域。此外，在属性模块中，您可以向由零件集合指定的区域分配截面。当Abaqus/CAE提示您选择区域时，您可以从当前视口中的零件中选择区域，也可以选择命名的零件集合。

只有在零件模块的**集合管理器**中才可见零件集合。当您在装配模块中实例化零件时，Abaqus会创建引用您之前创建的零件集合的零件实例集合。更多信息，请参见"理解集合和曲面"，第73.2节，以及"在装配模块中使用集合和曲面"，第13.8.6节。![](../graphics/images/black4rule.gif)有关相关信息，请单击以下任一项：- "理解零件模块中的工具集"，第11.16节- 第73章"集合和曲面工具集"
