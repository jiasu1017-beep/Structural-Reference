# 64.3.4 允许偏移实体网格中的分支





如果零件有"分支"，如[图64-15](pt06ch64s03s04.md#edt-offset-branch)所示，创建偏移实体网格并不简单，因为在分支区域中偏移方向是模糊的。

**图64-15** 在分支区域中偏移方向是模糊的。

![](../graphics/edt-offset-branch.png)

为了帮助您对此配置进行网格划分，Abaqus/CAE提供了将分支区域中的单元复制到集合的选项。

然后您可以执行以下操作：

1. 使用显示组从孤兒网格中移除包含分支区域单元的集合。
2. 从剩余的非分支区域创建偏移实体网格，如[图64-16](pt06ch64s03s04.md#edt-offset-branch-nonbranch)所示。 **图64-16** 从非分支区域创建偏移实体网格。 ![](../graphics/edt-offset-branch-nonbranch.png)
3. 使用显示组仅显示包含分支的集合，如[图64-17](pt06ch64s03s04.md#edt-offset-branch-branch)所示。 **图64-17** 使用显示组仅显示包含分支的集合。 ![](../graphics/edt-offset-branch-branch.png)
4. 从分支区域的一侧创建偏移实体网格。注意偏移方向是正确的（Abaqus/CAE通过为单元面上色来指示偏移方向）。此外，在完成分支偏移之前，不要删除基础壳单元。
5. 从分支区域的其他侧创建偏移实体网格，如[图64-18](pt06ch64s03s04.md#edt-offset-branch-both)所示。 **图64-18** 分支区域中的偏移网格。 ![](../graphics/edt-offset-branch-both.png)
6. 使用显示组显示两个区域。
7. 使用网格编辑工具集中的合并节点工具合并在步骤2、4和5中创建的偏移网格。
