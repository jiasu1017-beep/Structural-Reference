# 1.3.8 杯/槽成形





**产品：** Abaqus/Explicit  

本示例说明了在锻造问题中使用自适应网格划分，这些问题在工具-坯料界面包含大量剪切；成形了一个杯和一个槽。

### 问题描述

考虑了三种不同的几何模型，如图1.3.8-1所示。每个模型由一个刚性冲头、一个刚性模具和一个可变形坯料组成。坯料的外顶部和底部边缘呈弧形，这有利于材料相对于工具的流动。冲头和模具具有半圆形横截面；冲头半径为68.4 mm，模具半径为67.9 mm。坯料建模为von Mises弹性、完全塑性材料，杨氏模量为4000 MPa，屈服应力为5 MPa。泊松比为0.21；密度为1.E4 tonne/mm³。

在每种情况下，冲头移动61 mm，而模具完全约束。使用平滑幅值曲线将冲头速度斜坡上升到最大值，并保持恒定。速度的平滑促进了对加载的准静态响应。

#### 案例1：杯成形的轴对称模型

坯料用CAX4R单元进行网格划分，尺寸为50×64.77 mm。冲头和模具使用连接线段建模为解析刚性表面。在*r*=0处规定了对称边界条件。有限元模型如图1.3.8-2所示。

#### 案例2：槽成形的三维模型

坯料用C3D8R单元进行网格划分，尺寸为50×64.7×64.7 mm。冲头和模具建模为三维圆柱形解析刚性表面。在*x*=0和*z*=0平面上应用对称边界条件。坯料的有限元模型如图1.3.8-3所示。

#### 案例3：杯成形的三维模型

坯料用C3D8R单元进行网格划分。分析了半径为50 mm、高度为64.7 mm的坯料的90°扇形。冲头和模具建模为三维旋转解析刚性表面。在*x*=0和*y*=0平面上应用对称边界条件。坯料的有限元模型如图1.3.8-4所示。

### 自适应网格划分

每个模型使用包含整个坯料的单个自适应网格域。对称平面定义为拉格朗日边界区域（默认），接触表面定义为滑动边界区域（默认）。由于此问题是准静态的，每个增量的变形量相对较小，因此频率、网格扫描和其他自适应网格参数和控制的默认值就足够了。

### 结果和讨论

[图1.3.8-5](ch01s03aex39.md#exxalecupform-deform1)到[图1.3.8-7](ch01s03aex39.md#exxalecupform-deform3)显示了案例1-3锻造模拟结束时的网格配置。在每种情况下，网格质量在整个模拟过程中都保持良好。当坯料变平时，在分析开始时存在的几何边缘和角落被打破，自适应网格划分允许跨越它们。对于这类问题，几何边缘和角落的最终打破对于最小化单元扭曲和优化单元纵横比至关重要。

为了比较，[图1.3.8-8](ch01s03aex39.md#exxalecupform-deform1-lg)显示了案例1（轴对称模型）的纯拉格朗日模拟的变形网格。当使用连续自适应网格划分时，网格明显更好。在纯拉格朗日模拟中形成了几个具有极差纵横比的菱形单元。自适应网格划分显著改善了单元质量，特别是在杯的顶表面，解决方案梯度最高。[图1.3.8-9](ch01s03aex39.md#exxalecupform-cntr1)和[图1.3.8-10](ch01s03aex39.md#exxalecupform-cntr1-lg)分别显示了案例1自适应网格划分和纯拉格朗日分析在锻造完成时的等效塑性应变等值线。整体塑性应变比较非常接近。仅在顶表面存在细微差异，在模拟结束时纯拉格朗日网格在那里变得非常扭曲。对于锻造过程，自适应和纯拉格朗日分析的垂直冲头力时间历史密切一致，如图1.3.8-11所示。
