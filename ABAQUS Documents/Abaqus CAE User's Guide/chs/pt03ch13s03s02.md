# 13.3.2 从属和独立零件实例之间的区别是什么？







当您创建零件实例时，您可以选择创建从属零件实例或独立零件实例。您也可以编辑零件实例并将其从从属更改为独立或反之。当您创建模型实例时，它始终是从属的。

**从属零件实例**

默认情况下，Abaqus/CAE创建零件的从属实例。从属实例仅是指向原始零件的指针。实际上，从属实例共享原始零件的几何和网格。因此，您可以对原始零件进行网格划分，但不能对从属实例进行网格划分。当您对原始零件进行网格划分时，Abaqus/CAE会将相同的网格应用到该零件的所有从属实例。大多数修改不允许在从属零件实例上进行；例如，您不能添加分区或创建虚拟拓扑。但是，仍允许对从属零件实例不修改几何的操作；例如，您可以创建集合、施加载荷和边界条件，以及定义连接器截面分配。如果您已经对零件进行了网格划分或向零件添加了虚拟拓扑，则只能创建该零件的从属实例。

如果您在网格模块中对从属零件实例应用自适应重网格划分规则，Abaqus/CAE会重新网格划分原始零件，并将新网格应用到零件的每个从属实例。

您不能更改单个从属零件实例的网格属性；例如，网格种子、网格控制和单元类型。但是，您可以更改原始零件的网格属性，Abaqus/CAE会将更改传播到零件的所有从属实例。尽管您已经对原始零件进行了网格划分并将其应用到其从属实例，但网格仅在网格模块中可见。您继续在装配、交互和载荷模块中使用Abaqus/CAE的本几何。通常，您不能使用编辑网格工具集来编辑从属零件实例的网格；但是，您可以使用编辑网格工具集来编辑和投影从属零件实例的节点。Abaqus/CAE移动原始网格零件的节点，您的修改会出现在零件的所有从属实例上。

从属零件实例的优点是它们消耗更少的内存资源，您只需对零件进行一次网格划分。此外，Abaqus/CAE在输入文件中通过写入一组节点坐标和单元连接来实例化从属零件实例以定义零件，以及写入变换来定义每个零件实例。

**独立零件实例**

相比之下，独立零件实例是原始零件几何的副本。您不能对从中创建独立零件实例的零件进行网格划分；但是，您可以对独立实例进行网格划分。除了网格划分，您还可以在独立实例上执行大多数其他操作；例如，您可以添加分区和创建虚拟拓扑。独立实例的缺点是它们消耗更多内存资源，并且您必须分别对每个独立实例进行网格划分。此外，Abaqus/CAE在输入文件中不利用独立零件实例的实例化优势——会为每个独立零件实例写入节点坐标和单元连接集。

您不能创建同一零件的从属和独立实例。因此，如果您创建零件的从属实例，则所有后续实例都必须是从属的。独立实例也是如此。网格零件的实例始终是从属的。

您可以使用模型树来确定实例是从属还是独立。当您对独立零件实例进行网格划分时，网格会出现在模型树中零件实例容器下方，如图13-1所示。此外，如图13-1所示，当您将光标移到实例上时，模型树显示的信息指示实例是从属还是独立。
