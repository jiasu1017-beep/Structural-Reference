# 15.15.7 定义壳到实体耦合约束





您使用壳到实体耦合约束来耦合壳边缘的运动与相邻实体面的运动。您可以通过指定壳边缘表面和实体面区域来创建壳到实体耦合约束。被耦合的壳边缘表面和实体面区域必须属于不同的部件实例，除非壳边缘表面是实体模型内的中面区域。当壳截面和实体截面存在于同一部件中且几乎相互垂直时，Abaqus/CAE会自动创建壳到实体耦合约束。

有关壳到实体耦合约束的详细信息，请参见Abaqus Analysis User's Guide第35.3.3节的"壳到实体耦合"。

**创建壳到实体耦合约束的步骤：**

1. 从主菜单栏选择 ****Constraint**![](../graphics/images/arrow.gif)**Create****。**提示：**您也可以使用「相互作用」模块工具箱中的![](../graphics/ico_constraint.png)工具创建壳到实体耦合约束。
2. 在出现的「Create Constraint」对话框中，执行以下操作：1. 为约束命名。有关命名对象的更多信息，请参见"使用基本对话框组件"，第3.2.1节。2. 从「Type」列表中选择「Shell-to-solid coupling」，然后点击「Continue」。
3. 使用以下方法之一选择壳边缘表面：- 使用现有表面定义区域。点击提示区域右侧的「Surfaces」。从出现的「Region Selection」对话框中选择现有表面，然后点击「Continue」。**注意：**默认选择方法基于您最近使用的选择方法。要恢复为另一种方法，请点击提示区域右侧的「Select in Viewport」或「Surfaces」。- 使用鼠标在视口中选择一个区域。（有关更多信息，请参见"在当前视口中选择对象"，第6.2节。）点击鼠标按钮2表示您已完成选择。如果模型包含网格和几何的组合，请从提示区域点击以下之一：- 如果您想从几何区域选择表面，请点击「Geometry」。- 如果您想从原生网格或孤立网格选择中选择表面，请点击「Mesh」。您可以使用角度方法从几何或网格中选择一组面或边缘或一组元素面。有关更多信息，请参见"使用角度和特征边缘方法选择多个对象"，第6.2.3节。您选择的壳边缘表面在视口中以红色高亮显示。
4. 选择实体面区域。1. 在提示区域中，点击文本字段旁边的箭头并选择以下之一：- 如果您想选择表面，请选择「Surface」。- 如果您想选择要创建基于节点表面的区域，请选择「Node Region」。2. 使用与上一步所述相同的方法之一选择实体面区域。您选择的实体面区域在视口中以品红色高亮显示。约束编辑器会出现。
5. 从编辑器中，选择以下「Position Tolerance」方法之一：- **Use computed default**。Abaqus使用默认位置容差确定壳边缘表面上要与实体面区域耦合的节点。有关更多信息，请参见Abaqus Analysis User's Guide第35.3.3节的"壳到实体耦合"。- **Specify distance**。您可以指定从实体面区域的绝对距离，耦合中包含的所有壳节点必须位于此距离内。
6. 从编辑器中，选择以下「Influence Distance」方法之一：- **Use analysis default**。Abaqus使用默认影响距离确定实体面区域上要与壳边缘表面耦合的节点。有关更多信息，请参见Abaqus Analysis User's Guide第35.3.3节的"壳到实体耦合"。- **Specify value**。您可以指定从壳边缘表面的距离，耦合中包含的所有实体节点必须位于此距离内。
7. 点击「OK」保存约束定义并关闭编辑器。
