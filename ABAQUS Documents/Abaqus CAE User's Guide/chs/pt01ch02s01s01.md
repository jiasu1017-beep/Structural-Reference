# 2.1.1 启动 Abaqus/CAE（或 Abaqus/Viewer）







当您创建模型并进行分析时，Abaqus/CAE 会生成一组文件，包含模型定义、分析输入和分析结果。此外，Abaqus/CAE 和 Abaqus/Viewer 生成反映您与应用程序所有交互的回放文件。因此，在运行任一产品之前，您应该切换到有权创建文件的目录。

您可以通过运行 **abaqus** 执行过程并指定 **cae**（或 **viewer**）参数来执行 Abaqus/CAE（或 Abaqus/Viewer）：

| **abaqus cae** 或 **viewer** | **[****database**=*database-file***]** |
| --- | --- |
|  | **[****replay**=*replay-file***]** **[****recover**=*journal-file***]** **[****startup**=*startup-file***]** **[****script**=*script-file***]** **[****noGUI**=*noGUI-file***]** **[****noenvstartup****]** **[****noSavedOptions****]** **[****noSavedGuiPrefs****]** **[****noStartupDialog****]** **[****custom**=*script-file***]** **[****guiTester**=[*GUI-script*]**]** **[****guiRecord****]** **[****guiNoRecord****]** |

您可以在命令行上包含以下选项：

****database****

此选项指定要打开的模型数据库文件或输出数据库文件的名称。您可以在 Abaqus/CAE 中打开任一类型的文件；您只能在 Abaqus/Viewer 中打开输出数据库文件。要指定模型数据库文件，请在文件名中包含 `.cae` 文件扩展名或不包含文件扩展名。要在运行 Abaqus/CAE 时指定输出数据库文件，请在文件名中包含 `.odb` 文件扩展名。如果您正在运行 Abaqus/Viewer，则可以省略 `.odb` 文件扩展名。

****replay****

此选项指定要重放 Abaqus/CAE 命令的文件的名称。*replay-file* 中的命令将在 Abaqus/CAE 启动时立即执行。您不能使用 `replay` 选项执行带有控制流语句的脚本。有关更多信息，请参阅[「重放 Abaqus/CAE 会话，第 9.5.1 节」](pt02ch09s05s01.md)。

****recover****

此选项指定要重建模型数据库的文件的名称；如果您正在运行 Abaqus/Viewer，则此选项不可用。*journal-file* (*model_database_name*`.jnl`) 中的命令将在 Abaqus/CAE 启动时立即执行。有关更多信息，请参阅[「重新创建保存的模型数据库，第 9.5.2 节」](pt02ch09s05s02.md) 和[「重新创建未保存的模型数据库，第 9.5.3 节」](pt02ch09s05s03.md)。

****startup****

此选项指定包含要在应用程序启动时运行的 Python 配置命令的文件的名称。此文件中的命令在任何已在环境文件中设置的配置命令之后运行。Abaqus/CAE 在执行这些命令时不会将它们回显到回放文件。

****script****

此选项指定包含要在应用程序启动时运行的 Python 配置命令的文件的名称。此文件中的命令在任何已在环境文件中设置的配置命令之后运行。

可以通过在命令行上输入 `--` 后跟一个或多个空格分隔的参数来将参数传递到文件中。这些参数将被 Abaqus/CAE 执行过程忽略，但可以在脚本中访问。

****noGUI****

此选项指定要运行的文件包含 Python 脚本，但不带图形用户界面（GUI）。此选项对于自动执行预分析或后分析处理任务很有用，而无需运行显示的费用。由于没有提供界面，脚本不能包含任何用户交互。Abaqus/CAE 运行文件中的命令并在完成时退出。如果未给出文件扩展名，则默认扩展名为 `.py`。如果您使用 `noGUI` 选项，Abaqus/CAE 会忽略您提供的任何其他命令行选项。

可以通过在命令行上输入 `--` 后跟一个或多个空格分隔的参数来将参数传递到文件中。这些参数将被 Abaqus/CAE 执行过程忽略，但可以在 Python 脚本中访问。如果您使用 `noGUI` 选项，您可以使用参数传入一个否则会由命令行选项提供的变量。例如，您可以传入一个文件的名称，否则该文件将由 `script` 选项指定。

**noGUI** 选项的示例用法可在 [Abaqus Analysis User's Guide 第 3.2.6 节「Abaqus/CAE 执行」](../usb/usb-link.md#usb-int-dcaeproc) 中找到。

****noenvstartup****

此选项指定不应在应用程序启动时运行环境文件中的所有配置命令。此选项可与 **startup** 命令结合使用，以禁止除 **startup** 文件中之外的的所有配置命令。

****noSavedOptions****

此选项指定 Abaqus/CAE 不应应用存储在 `abaqus_v6.14.gpr` 文件中的显示选项设置（例如，渲染样式和基准平面的显示）。有关更多信息，请参阅[「使用 `abaqus_v6.14.gpr` 文件，第 2.1.3 节」](pt01ch02s01s03.md) 和[「保存您的显示选项设置，第 76.16 节」](pt07ch76hla15.md)。

****noSavedGuiPrefs****

此选项指定 Abaqus/CAE 不应应用存储在 `abaqus_v6.14.gpr` 文件中的 GUI 选项设置（例如，Abaqus/CAE 主窗口或其对话框的大小和位置）。

****noStartupDialog****

此选项指定不应显示 Abaqus/CAE 或 Abaqus/Viewer 的**开始会话**对话框。

****custom****

此选项指定包含 Abaqus GUI Toolkit 命令的文件的名称。此选项执行作为 Abaqus/CAE 或 Abaqus/Viewer 自定义版本的应用程序。有关更多信息，请参阅 [Abaqus GUI Toolkit User's Guide 第 1 章「简介」](../cus/cus-link.md#cus-int-int)。

****guiTester****

此选项启动一个包含 Abaqus Python 开发环境以及 Abaqus/CAE 或 Abaqus/Viewer 的独立用户界面。Abaqus Python 开发环境允许您创建、编辑、单步执行和调试 Python 脚本。有关更多信息，请参阅 [Abaqus Scripting User's Guide 第三部分「Abaqus Python 开发环境」](../cmd/cmd-link.md#cmd-pde)。

您可以将脚本指定为此选项的参数，这会提示 Abaqus/CAE 或 Abaqus/Viewer 运行 GUI 脚本。当到达脚本末尾时，Abaqus/CAE 或 Abaqus/Viewer 关闭。

****guiRecord****

此选项使您能够将 Abaqus/CAE 或 Abaqus/Viewer 用户界面中的操作记录到名为 `abaqus.guiLog` 的文件中。在 GUI 中创建操作记录可以帮助您捕获和重放 Abaqus/CAE 或 Abaqus/Viewer 中的常见活动，以用于演示或培训目的。您可以通过在 Abaqus Python 开发环境（PDE）中运行文件来重放 `.guiLog` 文件中的所有操作；有关更多信息，请参阅[「运行脚本，Abaqus Scripting User's Guide 第 7.3.2 节」](../cmd/cmd-link.md#cmd-pde-run)。

如果需要，您可以使用环境变量 `ABQ_CAE_GUIRECORD` 在启动时设置 **guiRecord**。**guiRecord** 选项不能与 **guiTester** 选项一起使用。

****guiNoRecord****

此选项使您能够在设置环境变量 `ABQ_CAE_GUIRECORD` 时禁用用户界面记录。

Abaqus/CAE 开始。如果您不包含 **database**、**replay**、**recover** 或 **noStartupDialog** 选项，则会出现**开始会话**对话框。选择以下会话启动选项之一：

**创建模型数据库：使用 Standard/Explicit 模型**

使用此选项（如果您正在运行 Abaqus/Viewer，则此选项不可用）开始新的 Abaqus/Standard 或 Abaqus/Explicit 分析（等同于从主菜单栏选择****文件**![](../graphics/images/arrow.gif)**新建模型数据库**![](../graphics/images/arrow.gif)**使用 Standard/Explicit 模型****）。

**创建模型数据库：使用 CFD 模型**

使用此选项（如果您正在运行 Abaqus/Viewer，则此选项不可用）开始新的 Abaqus/CFD 分析（等同于从主菜单栏选择****文件**![](../graphics/images/arrow.gif)**新建模型数据库**![](../graphics/images/arrow.gif)**使用 CFD 模型****）。

**创建模型数据库：使用电磁模型**

使用此选项（如果您正在运行 Abaqus/Viewer，则此选项不可用）开始电磁分析（等同于从主菜单栏选择****文件**![](../graphics/images/arrow.gif)**新建模型数据库**![](../graphics/images/arrow.gif)**使用电磁模型****）。

**打开数据库**

使用此选项打开先前保存的模型数据库或输出数据库文件（等同于从主菜单栏选择****文件**![](../graphics/images/arrow.gif)**打开****）。

**运行脚本**

使用此选项运行包含 Abaqus/CAE 命令的文件（等同于从主菜单栏选择****文件**![](../graphics/images/arrow.gif)**运行脚本****）。有关更多信息，请参阅[「创建和运行您自己的脚本，第 9.5.4 节」](pt02ch09s05s04.md)。

**开始教程**

使用此选项从在线文档开始入门教程（等同于从主菜单栏选择****帮助**![](../graphics/images/arrow.gif)**入门****）。

**最近文件**

使用此选项打开最近在 Abaqus/CAE 中打开的五个模型数据库文件或输出数据库文件之一（等同于选择**文件**菜单下列出的最近文件之一）。
