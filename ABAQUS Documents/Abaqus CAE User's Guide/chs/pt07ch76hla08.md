# 76.9 控制基准显示





您可以使用 ****View**![](../graphics/images/arrow.gif)**Part Display Options**** 和 ****View**![](../graphics/images/arrow.gif)**Assembly Display Options**** 菜单项来控制当前视口中零件和装配中基准几何的显示。您可以控制每种基准类型（点、轴、平面和坐标系）的显示，并且可以单独或同时切换它们的显示。您选择不显示的基准几何，虽然不可见，但仍然是零件或装配的特征。有关基准几何的更多信息，请参见 [第62章，"基准工具集](pt06ch62.md)。" 您也可以控制参考点的显示；有关更多信息，请参见 ["参考点，" 第11.8.1节](pt03ch11s08s01.md)。

在装配零件实例时，在零件上创建的基准几何可能会分散注意力；关闭基准几何的显示可以获得更清晰的装配显示。同样，关闭基准几何的显示对于生成干净的零件或装配打印图像很有用。

**控制基准显示的步骤：**

1. 找到 **Datum** 显示选项。从主菜单栏，选择 ****View**![](../graphics/images/arrow.gif)**Part Display Options**** 或 ****View**![](../graphics/images/arrow.gif)**Assembly Display Options****。在出现的对话框中，点击 **Datum** 选项卡。
2. 切换适当的按钮来控制： - 基准点 - 基准轴 - 基准平面 - 基准坐标系 或者，点击 **Show all datums** 以显示视口中的所有基准几何，或点击 **Show no datums** 以隐藏视口中的所有基准几何。
3. 点击 **OK** 实施您的更改并关闭对话框。您的更改仅适用于当前视口，并在会话期间保存。
