# 3.2.36 将Abaqus数据翻译为MSC.ADAMS模态中性文件





**产品：** Abaqus/Standard  

##### **参考**

- ["Abaqus执行程序：概述，" 第3.1.1节](pt01ch03s01abo02.md)
- ["将Abaqus数据翻译为用于MSC.ADAMS分析的模态中性文件格式，" Abaqus示例问题指南第15.1.7节](../exa/exa-link.md#exa-pst-adams)

### 概述

MSC.Software Corporation的ADAMS/Flex产品可用于在MSC.ADAMS中执行动态分析时考虑组件的柔性。ADAMS/Flex依赖有限元分析代码（如Abaqus）以ADAMS/Flex可用的格式提供组件的柔性信息。**abaqus adams**翻译器可用于创建MSC.ADAMS组件的Abaqus模型，并将Abaqus结果转换为MSC.ADAMS模态中性(`.mnf`)文件，这是ADAMS/Flex所需的格式。

### 使用翻译器

以下过程总结了**abaqus adams**翻译器的典型用法：

1. 为MSC.ADAMS模型的每个柔性组件创建Abaqus模型。每个组件建模为Abaqus子结构。
2. 运行Abaqus分析。有关更多信息，请参阅["准备子结构SIM数据库文件"](pt01ch03s02abx36.md#usb-int-dabaadmproc-input)。
3. 运行**abaqus adams**翻译器来读取分析生成的子结构SIM数据库，并创建MSC.ADAMS的模态中性(`.mnf`)文件。
4. 将模态中性文件读入MSC.ADAMS。必须为MSC.ADAMS中的每个柔性部件创建单独的模态中性文件。

### 准备子结构SIM数据库文件

本节介绍准备将生成ADAMS/Flex所需结果量的子结构SIM数据库。

#### Abaqus子结构模型

在MSC.ADAMS中考虑组件柔性的第一步是将该组件建模为Abaqus子结构。此过程涉及创建组件的Abaqus有限元模型。构建带子结构的Abaqus模型的一般指南在["使用子结构，" 第10.1.1节](pt04ch10s01aus58.md)中描述。

#### 设置Abaqus模型以创建模态中性文件

当创建要翻译到MSC.ADAMS的子结构时，子结构生成步骤必须包含以下选项：

```
[*SUBSTRUCTURE GENERATE*](../key/key-link.md#usb-kws-ssubgenerate), MASS MATRIX=YES, RECOVERY MATRIX=YES
[*FLEXIBLE BODY*](../key/key-link.md#usb-kws-hflexiblebody), TYPE=ADAMS
```

此外，您可以添加以下数据来将应力和/或应变翻译到MSC.ADAMS：
```
*ELEMENT RECOVERY MATRIX, POSITION=AVERAGED AT NODES
S,
E,
```

### 单位

MSC.ADAMS程序要求您定义组件模型中使用的单位，而Abaqus不要求。因此，在创建模态中性文件时，必须明确声明模型中使用的单位。在**abaqus adams**执行程序中执行此操作的方法与ADAMS/View **Units Settings**对话框中的方法非常相似。可以使用**abaqus adams**执行程序上的**units**选项指定预定义的单位系统。或者，可以使用**abaqus adams**执行程序上的**length**、**mass**、**force**和**time**选项指定单独的长度、质量、力和时间单位。指定的任何单独单位都会覆盖单位系统中的相应单位。默认单位系统是`mks`。**units**选项的有效单位系统在[表3.2.36-1](pt01ch03s02abx36.md#table-unitssystems)中列出。

**表3.2.36-1** 有效单位系统。
| 单位系统 | 长度单位 | 质量单位 | 力单位 | 时间单位 |
| --- | --- | --- | --- | --- |
| `mks` | 米 | 千克 | 牛顿 | 秒 |
| `mmks` | 毫米 | 千克 | 牛顿 | 秒 |
| `cgs` | 厘米 | 克 | 达因 | 秒 |
| `ips` | 英寸 | 磅 | 磅力 | 秒 |

**length**、**mass**、**force**和**time**每个选项的有效选项如下：

**长度单位**

长度单位的有效选项是：
- `meters`
- `millimeters`、`mm`
- `centimeters`、`cm`
- `kilometers`、`km`
- `inches`、`inch`、`in`
- `feet`、`foot`、`ft`
- `mile`

**质量单位**

质量单位的有效选项是：
- `kilograms`、`kg`
- `megagram`、`tonne`
- `gram`、`g`
- `pound_mass`、`lbm`、`pound`
- `slug`
- `kpound_mass`
- `ounce_mass`

**力单位**

力单位的有效选项是：
- `newtons`、`N`
- `knewton`、`kN`
- `kilogram_force`、`kgf`
- `dyne`
- `ounce_force`
- `pound_force`、`lbf`、`pound`
- `kpound_force`

**时间单位**

时间单位的有效选项是：
- `seconds`、`sec`
- `milliseconds`、`ms`
- `minutes`、`min`
- `hours`

默认的单位选项可以在Abaqus环境文件(`abaqus_v6.env`)中定义。**units**选项的默认值可以用**adams_units_family**参数定义。**length**、**mass**、**time**和**force**选项的默认值可以分别用**adams_length_units**、**adams_mass_units**、**adams_time_units**和**adams_force_units**参数定义。

### 翻译具有负特征值的模态

通常，对于三维中无预应力的非约束子结构，您希望找到六个具有相关零特征值的刚体模态。一般来说，预应力子结构的情况不同，其可能具有少于六个零特征值。预应力会将一些预期的零点更改为显著为正或为负的值（取决于预应力的符号）。

默认情况下，翻译器会删除具有负特征值的模态，并重新正交化简化基。如果您想保留具有负特征值的模态，请定义环境变量`MDI_MNFWRITE_OPTIONS`。
- 在UNIX平台上，键入以下命令：
  ``` setenv MDI_MNFWRITE_OPTIONS negative_roots_OK ```
- 在Windows平台上，键入以下命令：
  ``` set MDI_MNFWRITE_OPTIONS=negative_roots_OK ```

在这种情况下，翻译器将以与所有其他模态相同的方式处理具有负特征值的模态。

要确定模型在翻译器翻译时是否会有负特征值，您可以向输入文件添加无边界条件的特征频率提取步骤。

### 命令行选项

**job**

此选项指定结果翻译期间使用的输入和输出文件名。*job-name*值用于构造默认子结构SIM数据库文件名`*job-name*.sim`。输出模态中性文件被命名为`*job-name*.mnf`。

如果命令行上省略了此选项，系统将提示您输入此值。

**substructure_sim**

如果此选项与`*job-name*.sim`不同，此选项指定子结构SIM数据库(`.sim`)文件的名称。该文件通常命名为`*job-name*_Z*nn*.sim`。

**units**

此选项指定模型的单位系统。可能的值为`mmks`、`mks`、`cgs`或`ips`，对应ADAMS/View中相同名称的选项。默认值为`mks`。

此选项可在Abaqus环境文件中定义，如下所示：

```
adams_unit_family=*unit-family*
```

**length**

此选项指定模型的长度单位。如果指定了此选项，它将覆盖指定单位系统的长度单位。

此选项可在Abaqus环境文件中定义，如下所示：

```
adams_length_units=*length-unit*
```

**mass**

此选项指定模型的质量单位。如果指定了此选项，它将覆盖指定单位系统的质量单位。

此选项可在Abaqus环境文件中定义，如下所示：

```
adams_mass_units=*mass-unit*
```

**time**

此选项指定模型的时间单位。如果指定了此选项，它将覆盖指定单位系统的时间单位。

此选项可在Abaqus环境文件中定义，如下所示：

```
adams_time_units=*time-unit*
```

**force**

此选项指定模型的力单位。如果指定了此选项，它将覆盖指定单位系统的力单位。

此选项可在Abaqus环境文件中定义，如下所示：

```
adams_force_units=*force-unit*
```

**mnf_elset**

此选项定义其面将导出到模态中性文件并因此可在MSC.ADAMS中查看的元素集。此选项不影响求解的力学性能。
