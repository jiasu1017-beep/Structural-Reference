# 3.2.34 将LS-DYNA数据文件翻译为Abaqus输入文件





**产品：** Abaqus/Explicit  

##### **参考**

- ["Abaqus执行程序：概述，" 第3.1.1节](pt01ch03s01abo02.md)

### 概述

从LS-DYNA到Abaqus的翻译器将LS-DYNA输入文件中支持的关键字集转换为其在Abaqus中的对应物。

### 使用翻译器

翻译器支持翻译由LS-DYNA版本971 Rev 5或更早版本创建的输入文件。输入文件可以有任意名称和可选扩展名。

LS-DYNA支持的关键字列在下面的表格中。其他LS-DYNA关键字和数据被跳过并记录在日志文件中。

翻译器创建一个包含模型数据和历史数据的Abaqus输入文件。但是，翻译器不会为节点输出、元素输出和接触输出创建精确的Abaqus对应特定输出量；它使用预选的变量代替。您可以提供额外的输出实体来完成请求。

#### 元素编号和分组

生成Abaqus输入文件中的所有元素具有唯一的元素编号。自动为LS-DYNA输入中具有非唯一元素编号的元素分配新的元素编号；所有元素编号重新分配都在日志文件中注明。

分配了相同PART标识号的元素被分组在一个元素集中。具有不同材料或属性的元素必须分配不同的PART标识号；即，相同的材料和属性必须适用于组合在同一元素集中的所有元素。

当PART引用刚性材料时，该部件被视为刚体。与该部件对应的元素集用于刚体定义。

#### 材料模型

翻译器仅支持[表3.2.34-1](pt01ch03s02abx34.md#usb-int-ddynaabaproc-material)中所示的材料模型。所有不支持的材料模型在需要应力-应变律定义时被翻译为线性弹性。在这些情况下，翻译器会提供材料属性的标称值。

#### 映射以_ID或_TITLE结尾的LS-DYNA元素

许多LS-DYNA关键字包括选项_ID、_TITLE或两者。除非在本文档的映射表中指定了带有_ID或_TITLE的LS-DYNA关键字，否则翻译器会将这些选项的数据映射到为主LS-DYNA关键字指定的相同Abaqus关键字。

### 命令行选项

**job**

此选项用于指定要由翻译器输出的Abaqus输入文件的名称。必须不带`.inp`扩展名给出Abaqus输入文件的名称。翻译器生成的诊断信息被写入名为`*job-name*.log`的文件。

**input**

此选项用于指定包含LS-DYNA关键字数据的文件的名称。LS-DYNA输入文件可以有扩展名。

**splitFile**

此选项指定Abaqus输入文件是否要拆分为多个文件。如果**splitFile**=`ON`，则输出以下文件：
- `job-name_nodes.inc`：包含节点数据的include文件
- `job-name_elements.inc`：包含元素数据的include文件
- `job-name_model.inc`：包含剩余模型数据的include文件
- `job-name.inp`：包含所有上述include文件和历史数据的Abaqus输入文件
