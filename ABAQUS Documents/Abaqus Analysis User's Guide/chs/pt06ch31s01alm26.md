### 概述

连接器驱动：
- 用于模拟诸如部署机动中 attached to the body的电机加载连接器内部力或力矩历史，或液压系统施加已知运动等情况；
- 可用于固定可用的相对运动分量；以及
- 包括通过指定位移（旋转）或指定力（力矩）来驱动可用的相对运动分量。

指定的相对运动和载荷在连接器可用相对运动分量关联的局部方向上。

为可用相对运动分量指定位移/旋转（如果也包括连接器止挡或连接器锁行为）可能导致过约束。Abaqus将在发生过约束时发出警告消息。

### 固定可用的相对运动分量

一种常见做法是固定可用的运动分量。这种固定运动条件可用于为特定应用自定义连接类型。例如，REVOLUTE连接类型使用局部1方向作为共享旋转轴，因此也是可用的相对运动分量。如果需要局部3方向上的旋转连接，为了方便起见，您可以在CARDAN连接类型中固定局部1和2方向上的相对旋转。这样做将创建一个与REVOLUTE连接类型相同的连接类型；但是，共享轴将是局部3方向而非局部1方向。

本节后文提供了一个示例，其中销-槽连接的销部分用具有固定旋转的CARDAN连接类型建模。

| **输入文件用法：** | 在输入文件的模型部分使用以下选项来固定可用的连接器相对运动分量： |
| --- | --- |
|  | ``` [*CONNECTOR MOTION](../key/key-link.md#usb-kws-hconnectormotion) ``` |

| **Abaqus/CAE用法：** | 载荷模块：****Create Boundary Condition**：****Step: Initial**：****Mechanical**：****Connector displacement** |
| --- | --- |

### 位移控制的驱动

您可以类似于定义边界条件的方式指定连接器局部方向上两个部件之间的相对位移、速度或加速度（参见["Abaqus/Standard和Abaqus/Explicit中的边界条件，" 34.3.1节](pt07ch34s03aus118.md)）。您指定连接器单元集名称或连接器单元编号；标识被驱动的可用相对运动分量的分量号；以及相对位移、速度或加速度的值。

用于强制执行连接器运动的罚函数可能导致解噪声，特别是对于某些模型的单精度。因此，在这种情况下最好使用双精度。如果双精度性能是一个问题，您可以运行约束打包和约束求解器的双精度（参见["Abaqus/Standard、Abaqus/Explicit和Abaqus/CFD执行，" 3.2.2节](pt01ch03s02abx02.md)）。

您不能在子空间动态分析中指定连接器的运动。

| **输入文件用法：** | 在输入文件的历史部分使用以下选项来指定连接器的相对位移： |
| --- | --- |
|  | ``` [*CONNECTOR MOTION](../key/key-link.md#usb-kws-hconnectormotion), AMPLITUDE=*name*, OP=MOD *or* NEW, TYPE=DISPLACEMENT ``` 使用以下选项在输入文件的历史部分指定连接器的相对速度： ``` [*CONNECTOR MOTION](../key/key-link.md#usb-kws-hconnectormotion), AMPLITUDE=*name*, OP=MOD *or* NEW, TYPE=VELOCITY ``` 使用以下选项在输入文件的历史部分指定连接器的相对加速度： ``` [*CONNECTOR MOTION](../key/key-link.md#usb-kws-hconnectormotion), AMPLITUDE=*name*, OP=MOD *or* NEW, TYPE=ACCELERATION ``` |

| **Abaqus/CAE用法：** | 载荷模块：****Create Boundary Condition**：****Mechanical**：****Connector displacement**，****Connector velocity**** 或 ****Connector acceleration** |
| --- | --- |

#### 示例

[图31.1.3-1](pt06ch31s01alm26.md#econnector-pininslot)展示了一个销-槽连接，沿全局1轴45度方向，用CONN3D2单元类型建模。

**图31.1.3-1** 用SLOT和CARDAN连接类型建模的销-槽连接。

![](../graphics/econnector-pininslot-nls.png)

左侧是待建模连接的示意图，右侧是有限元网格。槽内的位移仅允许沿槽线，连接类型SLOT适用于强制执行这些运动。假设销和槽的构造使得销相对于槽的唯一旋转沿局部3方向。这是一个旋转约束；然而，基本旋转连接类型REVOLUTE使用局部1方向作为旋转轴。在这种情况下，可以将连接类型CARDAN与指定约束结合使用，以使用适当的旋转轴定义旋转型连接。

为了说明，假设连接由绕销轴的旋转速度![](../graphics/usb_eqn04486.gif)弧度每秒驱动。为方便起见，使用输入参数化，使用以下行：

```
*...*
[*ELEMENT](../key/key-link.md#usb-kws-melement), TYPE=CONN3D2, ELSET=pininslot

[*CONNECTOR SECTION](../key/key-link.md#usb-kws-mconnectorsection), ELSET=pininslot
cardan, slot
ori45,
[*CONNECTOR MOTION](../key/key-link.md#usb-kws-hconnectormotion)
pininslot, 4
pininslot, 5
[*ORIENTATION](../key/key-link.md#usb-kws-morientation), NAME=ori45
0.707, 0.707, 0.0, -0.707, 0.707, 0.0
*...*
[*STEP](../key/key-link.md#usb-kws-hstep)
*...*
[*CONNECTOR MOTION](../key/key-link.md#usb-kws-hconnectormotion), TYPE=VELOCITY
pininslot, 6, <rotangvel>
*...*
[*END STEP](../key/key-link.md#usb-kws-hendstep)
```

### 力控制的驱动

您可以类似于为Abaqus中其他单元定义集中载荷的方式，指定施加到可用相对运动分量上的集中载荷（参见["集中载荷，" 34.4.2节](pt07ch34s04aus121.md)）。但是，连接器载荷始终是跟随载荷，随着连接器单元运动时可用相对运动分量的旋转而旋转。您指定连接器单元集名称或连接器单元编号，标识被加载的可用相对运动分量，以及驱动载荷或力矩的值。

| **输入文件用法：** | 在输入文件的历史部分使用以下选项来指定连接器的集中载荷： |
| --- | --- |
|  | ``` [*CONNECTOR LOAD](../key/key-link.md#usb-kws-hconnectorload), AMPLITUDE=*name*, OP=MOD ``` |

| **Abaqus/CAE用法：** | 载荷模块：****Create Load**：****Mechanical**：****Connector force** 或 ****Connector moment** |
| --- | --- |

#### 示例

回到[图31.1.3-1](pt06ch31s01alm26.md#econnector-pininslot)中的示例，假设销沿槽被恒定力1000.0单位推动（例如，通过液压系统）。应将以下行添加到输入文件：

```
[*STEP](../key/key-link.md#usb-kws-hstep)
*...*
[*CONNECTOR LOAD](../key/key-link.md#usb-kws-hconnectorload)
pininslot, 1, 1000.0
*...*
[*END STEP](../key/key-link.md#usb-kws-hendstep)
```

### 线性摄动过程中的连接器驱动

仅在特征值屈曲、直接解稳态动态和线性静态摄动过程中允许非零幅值连接器运动。在特征频率提取过程中指定的任何非零幅值将被忽略，指定的可用相对运动分量将被固定。连接器运动不能用于任何基于模态的过程。

在直接解稳态动态分析中，任何可用连接器相对运动分量的实部和虚部同时被约束或不受约束；同时约束一个部分而另一个部分不受约束在物理上是不可能的。Abaqus/Standard将自动约束相对运动分量的实部和虚部，即使只明确指定了一个部分。未指定的部分将被假定具有零摄动幅值。

在特征值屈曲步骤中，非零指定连接器运动将对增量应力做出贡献，从而对差异初始应力刚度做出贡献。在指定非零连接器运动时，您必须仔细解释由此产生的特征问题。参见["特征值屈曲预测，" 6.2.3节](pt03ch06s02at02.md)中关于边界条件的讨论获取更多详细信息。

在稳态动态分析中，实部和虚部连接器载荷都可以以类似于集中载荷的方式施加（参见["基于模态的稳态动态分析，" 6.3.8节](pt03ch06s03at13.md)；["直接解稳态动态分析，" 6.3.4节](pt03ch06s03at09.md)；和["基于子空间的稳态动态分析，" 6.3.9节](pt03ch06s03at14.md)）。在随机响应分析中可定义多个连接器载荷情况（参见["随机响应分析，" 6.3.11节](pt03ch06s03at16.md)），其方式与集中载荷相同。连接器载荷在特征频率提取分析期间被忽略。
