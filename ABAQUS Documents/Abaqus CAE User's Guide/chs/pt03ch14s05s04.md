# 14.5.4 自由度监视器请求





您可以请求Abaqus在分析过程中在特定增量将一个选定点的自由度值写入状态（`.sta`）文件，对于Abaqus/Standard分析，写入消息（`.msg`）文件。此外，当您提交分析时，自由度值随时间变化的曲线会出现在自动生成的新视口中。（更多信息，请参见["监控分析作业的进度，"第19.2.6节](pt03ch19s02hlb01.md)。）您可以使用此信息监控求解的进度。

您必须通过选择现有几何或节点集，或通过在视口中选择一个点来指定您想要监视的顶点或节点。一旦您指定了点，您必须指出您想要在该顶点或节点上监视哪个自由度，多久在视口中显示一次信息，以及多久打印到状态和消息文件一次。

有关监视自由度的详细说明，请参见["配置监视器请求，"第14.13.4节](pt03ch14s13hlb04.md)。
