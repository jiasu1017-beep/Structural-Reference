# 12.5.3 从库向模型添加材料





要查看材料库，请选择Property模块的Model Tree区域中的**Material Library**标签页。如果有多个库可用，请从标签页顶部的列表中选择一个。在默认的**Tree**视图中，展开类别以查看每个类别中的材料。要隐藏类别并查看当前库中所有材料的字母列表，请选择**List**视图。

要从库向当前模型添加材料，请在树或列表视图中高亮显示材料名称，然后点击**Materials**列表右上角的**Add Material**图标![](../graphics/ico_materialAdd.png)。或者，您可以双击库中的材料名称来将其添加到模型。

**使用材料库向模型添加材料：**

1. 点击主窗口左侧Model Tree区域顶部的**Material Library**。
   **提示：**如果未显示Model Tree，请从主菜单栏选择****View**![](../graphics/images/arrow.gif)**Show Model Tree****。
2. 从**Name**列表中选择一个库名称。如果Abaqus/CAE在会话开始时未找到任何材料库，您可以创建新库。有关更多信息，请参见["管理材料库，"第12.5.2节](pt03ch12s05s02.md)。Abaqus/CAE显示库内容。默认情况下，库以树格式显示，材料可能按类别分开，类似于Model Tree。
3. 使用以下任一方法在库中定位所需的材料：
   - 在**Tree**视图中展开类别以查看其内容。
   - 切换**List**以隐藏类别并查看库中的所有材料。
   - 在**Filter**字段中输入字符串，以仅显示其名称包含该字符串的材料。
4. 点击材料名称以选择它。
5. 如有需要，点击**Specification Tags**（位于材料列表下方）右侧的箭头以查看有关所选材料的更多信息。
6. 点击**Materials**列表右上角的**Add Material**图标![](../graphics/ico_materialAdd.png)，将材料添加到当前模型。

将材料添加到模型后，使用**Edit Material**对话框查看或编辑材料属性。使用Property模块中的其他工具将材料与截面关联，并将截面分配给模型的相应部分。
