# 58.3 使用解析映射场







解析映射场允许您从外部数据源定义空间变化的参数值。本节涵盖以下主题：
- ["用于映射的点云数据文件格式，" 第 58.3.1 节](pt06ch58s03s01.md)
- ["用于点云映射场的坐标系，" 第 58.3.2 节](pt06ch58s03s02.md)
- ["从输出数据库文件进行网格到网格映射，" 第 58.3.3 节](pt06ch58s03s03.md)
- ["用于映射的支持元素，" 第 58.3.4 节](pt06ch58s03s04.md)

在 Abaqus/CAE 中，映射场用于在不同空间点提供选定的载荷、相互作用、属性等的值。例如，您可以通过在不同坐标处提供厚度或压力值来定义空间变化的壳厚度或压力载荷。参数值可以从第三方 CAE 应用程序生成的点云数据文件或从 Abaqus 输出数据库（`.odb`）文件读取。

映射场允许您导入离散和非连续的参数数据，并将其应用到您的 Abaqus/CAE 模型。Abaqus/CAE 将值应用到当前模型，将输入的 *X*、*Y* 和 *Z* 坐标映射到模型中的位置。Abaqus/CAE 将源数据映射到目标模型，Abaqus 计算要在分析期间使用的分布式参数值。参数值也称为场值或场数据；例如表面上不同点处的压力值。

当您从点云数据创建映射场时，您可以为源数据区域分配局部坐标系，以简化空间点的三维定义。局部坐标系可以是直角坐标、圆柱坐标或球坐标。例如，对于直角坐标系，*X*、*Y* 和 *Z* 坐标在该局部坐标系中解释。

映射场中只能使用标量数据值。每个场数据值作为标量值从源映射到目标区域。Abaqus/CAE 映射算法是纯几何的，没有物理考虑（如保守映射）。

您在属性、载荷、预定义字段或相互作用中指定的大小用作映射场数据值的乘数。您还可以缩放源数据坐标；例如，以考虑单位不匹配（即，米到毫米）。

当您使用映射场应用载荷、相互作用或预定义字段时，可以在视口中显示符号以可视化场值的位置和大小。但是，您必须首先对模型进行网格划分才能看到这些符号。

Abaqus/CAE 提供了一组映射容差控制，允许您调整源数据点可以距目标点多远的距离。根据每个源点距网格模型目标上最近节点的距离，Abaqus/CAE 必须决定是使用还是丢弃每个源点。

映射场不支持输出请求频率时间点。

**注意：**当使用映射场施加压力载荷时，您必须请求场输出变量 P 才能在 Visualization 模块中可视化映射的压力值。输出变量名称 P 在分析期间自动更改为 PDLOAD；请参阅 ["Abaqus/Standard 输出变量标识符，" Abaqus Analysis User's Guide](../usb/usb-link.md#usb-out-houtputvar) 第 4.2.1 节，或 ["Abaqus/Explicit 输出变量标识符，" Abaqus Analysis User's Guide](../usb/usb-link.md#usb-out-hfileoutputvar) 第 4.2.2 节。
