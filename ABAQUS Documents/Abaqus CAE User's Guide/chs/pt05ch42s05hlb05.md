# 42.7.5 通过对帧进行运算来创建场输出





您可以通过组合输出数据库中可用的多个帧的结果来创建场输出。例如，您可以创建新的场输出来显示多个载荷工况的组合响应。有关详细示例，请参见["组合多个帧的结果，" 第42.7.3节](pt05ch42s05hlb03.md)。

您可以对指定帧上的场输出进行求和，并且可以找到指定帧上的最小值或最大值。您可以通过以下方式显示您创建的场输出，就像显示输出数据库场输出变量一样：以变形、等值线或符号图的形式显示；通过探测任何模型或*X-Y*图；沿模型中的路径获取*X-Y*数据；或以表格报告的形式显示。您创建的场输出将保存在会话步骤中，直到结束会话或关闭场输出所在的输出数据库。

**注意：**使用复数结果创建的场输出将仅包含数据的实部。您无法创建包含复数结果虚部的场输出。

**通过对现有帧进行运算来创建场输出：**

1. 找到从帧创建场输出的选项。从主菜单栏，选择****工具**![](../graphics/images/arrow.gif)**创建场输出**![](../graphics/images/arrow.gif)**从帧****。此时将出现**从帧创建场输出**对话框。**提示：**您也可以使用工具箱中的![](../graphics/ico_fieldOutputFromFrames.png)工具从帧创建场输出。默认情况下，此对话框中的**帧**和**场**页面为空。
2. 从**操作**字段中选择您希望执行的操作。
3. 点击![](../graphics/ico_add.png)选择将包含在操作中的帧。此时将出现**添加帧**对话框。
4. 从**步骤**字段中选择分析步骤。该步骤的可用帧列表将出现。
5. 选择您希望包含在操作中的各个帧（有关详细信息，请参见["从列表和表中选择多个项目，" 第3.2.11节](pt01ch03s02s11.md)），或点击**全选**以包含所有可用帧。
6. 在**添加帧**对话框中点击**确定**或**应用**。您选择的帧将出现在**从帧创建场输出**对话框的**帧**字段中。您可以使用底部的![](../graphics/ico_delete.png)按钮（**移除所选**）和![](../graphics/ico_error.png)按钮（**移除全部**）来根据需要修改您的帧选择。您也可以继续从**添加帧**对话框添加帧。
7. 如果您选择了求和操作，**比例因子**列也会出现在**从帧创建场输出**对话框的**帧**字段中。每个帧的默认比例因子为1.0。您可以通过点击**比例因子**列并输入新的比例因子值来修改所选帧的比例因子。
8. 点击**从帧创建场输出**对话框中的**场**选项卡，以选择将包含在操作中的场输出。所选帧的可用场输出变量将显示，并附带有关输出变量类型（张量、向量或标量）的信息。如果您选择了最小值或最大值的操作，张量和向量变量的不变量/分量信息也将可用。默认情况下，所有可用输出变量在**选择**列中都有勾选标记。1. 点击**选择**列以取消选择您不希望包含在操作中的输出变量。2. 如果您选择了最小值或最大值操作，您可以点击**不变量/分量**列，并从右侧出现的列表中选择您希望用于确定最小值或最大场结果的不变量或分量。例如，如果您为应力变量选择了Mises不变量，Abaqus/CAE将创建一个新的应力场输出变量，其中每个点处的应力张量取自出现最大Mises值的帧（在第5步中选择的帧列表内）。3. 使用**全选**、**取消全选**和**默认**按钮根据需要修改您的场输出选择。
9. 您可以接受Abaqus/CAE提供的默认帧描述，也可以在**帧描述**字段中提供您自己的描述。
10. 默认情况下，Abaqus/CAE不会为新的场输出提供载荷工况名称；如果您的新场输出代表分析中的额外载荷工况，您可以在**载荷工况名称**字段中提供名称。此名称仅用于组织目的。
11. 点击**从帧创建场输出**对话框中的**确定**以创建您的新场输出并关闭对话框。Abaqus/CAE对所选帧上的选定场输出进行运算，以创建包含在会话步骤帧中的新场输出，并可从**场输出**对话框获取。新变量的描述指明了派生它们的操作。当前结果图（如果有）不受影响。当您选择最小值或最大值操作时，Abaqus/CAE还会为每个结果创建一个场输出变量，指示获得每个最小值或最大值的帧。（帧通过它们在所选帧列表中的索引来标识。）此变量名为*outputVariable*_Index，可从与最小值或最大值操作的实际结果相同的会话步骤帧获取，并且可以像其他场输出变量一样进行查询或绘图。
