# 3.3 执行 adams 命令以创建包含应力或应变的模态中性文件





### 3.3 执行 **adams** 命令以创建包含应力或应变的模态中性文件

要创建包含应力或应变的模态中性文件，Abaqus Interface for MSC.ADAMS 必须读取两个结果文件中的数据。您可以按照 ["设置 Abaqus 模型以创建包含应力或应变的模态中性文件" 第 2.3 节](ch02s03.md) 中描述的要求创建第一个结果文件。接下来，使用 **adams** 命令创建第二次 Abaqus 分析的输入文件，如 ["创建第二个输入文件" 第 3.3.1 节](ch03s03.md#adm-chp-mnf-command-input) 中所述。然后运行第二次 Abaqus 分析，它将写入第二个结果文件。最后，使用 **adams** 命令创建模态中性文件，如 ["从两个结果文件创建模态中性文件" 第 3.3.2 节](ch03s03.md#adm-chp-mnf-command-two) 中所述。

### 3.3.1 创建第二个输入文件

要创建第二次 Abaqus 分析的输入文件，请使用以下命令执行 Abaqus Interface for MSC.ADAMS：

| **abaqus adams** | **job**=*job-name* |
| --- | --- |
|  | **[****input**=*input-file***]** **[****make_se_recovery****]** **[****stress_modes**=**{**`ON`** | **`OFF`**}****]** **[****strain_modes**=**{**`ON`** | **`OFF`**}****]** **[****section_point**=*section_point_number***]** |

****job****

此选项控制将创建的新输入文件的名称。*job-name* 值用于构造新输入文件名 `*job-name*_se_recovery.inp`。

如果此选项从命令行中省略，系统将提示您输入此值。

****input****

此选项指定第一次 Abaqus 分析结果文件的名称（如果与 `*job-name*.fil` 不同）。

****make_se_recovery****

此选项指定翻译器将创建一个新的 Abaqus 输入文件。将不会创建模态中性文件。

****stress_modes****

此选项指定新的 Abaqus 输入文件将包含将应力写入结果文件的命令。可能的值为 `ON` 和 `OFF`。默认值为 `ON`。

****strain_modes****

此选项指定新的 Abaqus 输入文件将包含将应变写入结果文件的命令。可能的值为 `ON` 和 `OFF`。默认值为 `OFF`。

****section_point****

此选项指定将在其中写入壳应力和/或应变的截面点。默认值为 1。此选项对三维连续体单元将被忽略。

### 3.3.2 从两个结果文件创建模态中性文件

运行两个 Abaqus 分析后，如上所述，您可以通过使用以下命令执行 Abaqus Interface for MSC.ADAMS 来创建包含模态应力或应变的模态中性文件：

| **abaqus adams** | **job**=*job-name* |
| --- | --- |
|  | **[****input**=*input-file***]** **[****se_recovery_job**=*se_recovery_job-name***]** **[****units**=`mmks` | `mks` | `cgs` | `ips`**]** **[****length**=*length-units-name***]** **[****mass**=*mass-units-name***]** **[****time**=*time-units-name***]** **[****force**=*force-units-name***]** |

****job****

此选项控制将创建的模态中性文件的名称。*job-name* 值用于构造新模态中性文件名 `*job-name*.mnf`。

如果此选项从命令行中省略，系统将提示您输入此值。

****input****

此选项指定第一次 Abaqus 分析结果文件的名称（如果与 `*job-name*.fil` 不同）。

****se_recovery_job****

此选项指定第二次 Abaqus 分析结果文件的名称。

****units****

此选项指定模型的单位系统。可能的值为 `mmks`、`mks`、`cgs` 或 `ips`，对应于同名的 ADAMS/View 选项。默认值为 `mks`。

****length****

此选项指定模型的长度单位。如果指定此选项，它将覆盖指定单位系统的长度单位。

****mass****

此选项指定模型的质量单位。如果指定此选项，它将覆盖指定单位系统的质量单位。

****time****

此选项指定模型的时间单位。如果指定此选项，它将覆盖指定单位系统的时间单位。

****force****

此选项指定模型的力单位。如果指定此选项，它将覆盖指定单位系统的力单位。
