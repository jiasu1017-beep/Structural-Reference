# 70.4.1 分区边的方法概述







从主菜单栏中选择 ****Tools**![](../graphics/images/arrow.gif)**Partition****（工具→分区）以显示 **Create Partition**（创建分区）对话框。当您从 **Create Partition**（创建分区）对话框中选择 **Edge**（边）时，**Method**（方法）列表显示以下分区边的方法：

**![](../graphics/ico_ptnEdgeLocation.png) **通过位置指定参数****

沿边任意位置选择一点。详细说明，请参阅["使用通过位置指定参数方法分区边，" 第 70.5.1 节"](pt06ch70s05hlb01.md)。

**![](../graphics/ico_ptnEdgeParameter.png) **输入参数****

在提示区域中输入参数（如图 70-5 所示）。沿边的箭头指示从起始顶点（对应于边参数值零）到端顶点（对应于值一）的参数值增加方向。详细说明，请参阅["使用输入参数方法分区边，" 第 70.5.2 节"](pt06ch70s05hlb02.md)。

**图 70-5** 输入参数以分区边。

![](../graphics/par-edge-parameter-nls.png)

**![](../graphics/ico_ptnEdgeMidpoint.png) **选择中点/基准点****

选择边的中点或沿边的基准点（如图 70-6 所示）。详细说明，请参阅["使用选择中点/基准点方法分区边，" 第 70.5.3 节"](pt06ch70s05hlb03.md)。

**图 70-6** 选择中点或基准点以分区边。

![](../graphics/par-edge-pick-nls.png)

**![](../graphics/ico_ptnEdgeDatumPlane.png) **使用基准平面****

选择一个基准平面。Abaqus/CAE 在基准平面与边相交的位置创建分区（如图 70-7 所示）。使用基准平面分区一组选定的边是一种对齐一组分区的有用技术。详细说明，请参阅["使用基准平面方法分区边，" 第 70.5.4 节"](pt06ch70s05hlb04.md)。

**图 70-7** 选择基准平面以分区边。

![](../graphics/par-edge-datum-nls.png)
