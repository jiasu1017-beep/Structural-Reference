# 43 绘制未变形和变形形状





在打开输出数据库后立即，Abaqus/CAE会显示模型的未变形形状。对于通用分析步骤，未变形绘图显示没有任何变形的模型形状。对于线性摄动步骤，未变形绘图显示模型的基态。变形绘图根据节点变量（如位移）的值显示模型的形状。未变形和变形形状也用于显示Abaqus/CAE中可用的其他每种绘图状态。您可以在未变形或变形形状上查看云图、符号或材料方向，也可以叠加它们以查看形状之间的差异。本章解释未变形和变形形状绘图以及叠加绘图。涵盖以下主题：
