# 11.4.1 断裂力学：概述

Abaqus/Standard 提供以下方法进行断裂力学研究：

**开裂起始**：通过使用轮廓积分可以在准静态问题中研究开裂起始（["轮廓积分评估," 第 11.4.2 节"](pt04ch11s04aus68.md)）。*J* 积分、![](../graphics/usb_eqn01329.gif) 积分（用于蠕变）、均匀材料和界面裂纹的应力强度因子、裂纹扩展方向和 *T*-应力由 Abaqus/Standard 计算。轮廓积分可用于二维或三维问题。在这类问题中，通常需要集中网格，且不研究裂纹的扩展。

**裂纹扩展**：裂纹扩展能力允许研究准静态裂纹沿预定义路径的扩展，包括低周疲劳（["裂纹扩展分析," 第 11.4.3 节"](pt04ch11s04aus69.md)）。裂纹沿用户定义的表面脱粘。有几种裂纹扩展准则可用，分析中可包含多条裂纹。可在裂纹扩展问题中请求轮廓积分。

**线弹簧单元**：壳中的部分穿透裂纹可以通过在线弹簧单元中使用静态过程进行建模，如 ["用于建模壳中部分穿透裂纹的线弹簧单元," 第 32.9.1 节"](pt06ch32s09alm54.md) 中所述。

**扩展有限元方法 (XFEM)**：XFEM 通过在具有特殊位移函数的单元中添加自由度，将裂纹建模为富集特征（["使用扩展有限元方法将不连续性建模为富集特征," 第 10.7.1 节"](pt04ch10s07at36.md)）。XFEM 不需要网格与不连续性的几何形状匹配。它可用于模拟任意、依赖解的路径上离散裂纹的起始和扩展，无需重新网格划分。XFEM 还可用于执行轮廓积分评估，无需在裂纹尖端周围细化网格。
