# 14.9.3 编辑步骤





您可以使用步骤编辑器来编辑与现有步骤关联的分析过程设置。

**要编辑步骤：**

1. 从主菜单栏，选择**步骤**![](../graphics/images/arrow.gif)**编辑**![](../graphics/images/arrow.gif)***步骤名称***。步骤编辑器出现。**提示：**您也可以在**步骤管理器**中选择步骤名称并点击**编辑**。
2. 使用步骤编辑器中的选项卡来修改设置。（有关特定编辑器功能的详细帮助，请从主菜单栏选择**帮助**![](../graphics/images/arrow.gif)**上下文帮助**，然后点击感兴趣的功能。）
3. 点击**确定**关闭步骤编辑器并保存新设置。
