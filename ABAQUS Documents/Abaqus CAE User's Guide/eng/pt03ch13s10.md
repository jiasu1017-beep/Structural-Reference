# 13.10 Creating and manipulating part and model instances







This section describes how you use the Assembly module's **Instance** menu to create part and model instances and to position the instances relative to the global coordinate system. You can also use the **Instance** menu to replace one part instance with another and to convert the constraints applied to a selected part instance to an absolute position.
