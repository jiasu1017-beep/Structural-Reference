# 11.24.2 创建放样切割





从主菜单栏选择**形状**![](../graphics/images/arrow.gif)**切割**![](../graphics/images/arrow.gif)**放样**，向当前视口中的零件几何添加放样切割。您只能向三维零件添加放样切割。

您通过从选定边缘创建两个或多个截面并定义一个或多个放样路径来添加放样切割。

**添加放样切割：**

1. 从主菜单栏，选择**形状**![](../graphics/images/arrow.gif)**切割**![](../graphics/images/arrow.gif)**放样**。
2. 通过从视口中的零件选择边缘来创建放样截面。
3. 完成创建放样截面后，单击**编辑放样**对话框中的**过渡**选项卡。
4. 通过再次从视口中的零件选择边缘或使用放样切线来创建放样路径。
5. 单击**预览**。
6. 单击**完成**创建放样并关闭**编辑放样**对话框。
