# 6.7 小结





- 梁单元的行为可以通过截面的数值积分（[*BEAM SECTION](../key/key-link.md#usb-kws-mbeamsection) 或 [*BEAM GENERAL SECTION](../key/key-link.md#usb-kws-mbeamgensect)）来确定，也可以直接通过面积、惯性矩和扭转常数（[*BEAM GENERAL SECTION](../key/key-link.md#usb-kws-mbeamgensect)）来给定。
- 当使用数值积分的 [*BEAM GENERAL SECTION](../key/key-link.md#usb-kws-mbeamgensect) 时，计算在模拟开始时完成一次，并假定为弹性行为。
- Abaqus 包含许多标准横截面形状。其他形状，只要它们是"薄壁"的，可以使用 SECTION=ARBITRARY 进行建模。
- 横截面的方向必须通过指定第三个节点或作为单元属性定义的一部分定义一个向量来定义。方向可以在 Abaqus/Viewer 中绘制。
- 梁横截面可以从定义梁的节点偏移。此方法在建模壳上的加劲肋时很有用。
- 线和二次梁包含剪切变形的影响。Abaqus/Standard 中的三次梁不考虑剪切柔性。Abaqus/Standard 中的开口截面梁单元正确模拟薄壁开口截面中的扭矩和翘曲（包括翘曲约束）的影响。
- 多点约束、约束方程和连接器可用于连接节点处的自由度，以模拟铰接连接、刚性连杆等。
- "弯矩"型绘图允许轻松可视化一维单元（如梁）的结果。
- 显示选项允许您渲染梁轮廓，以增强未变形和变形绘图的图形表示。
- Abaqus/Viewer 绘图的硬拷贝可以 PostScript (PS)、封装 PostScript (EPS)、标记图像文件格式 (TIFF)、便携式网络图形 (PNG) 和可缩放矢量图形 (SVG) 格式获取。
