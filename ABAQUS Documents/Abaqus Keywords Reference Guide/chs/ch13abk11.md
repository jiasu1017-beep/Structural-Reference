# *MATRIX ASSEMBLE









### *MATRIX ASSEMBLE定义模型一部分的刚度矩阵、质量矩阵或阻尼矩阵。

此选项可用于识别将被组装到相应全局有限元矩阵中的刚度矩阵、质量矩阵或阻尼矩阵。此矩阵必须先前已使用[*MATRIX INPUT](ch13abk13.md)选项输入。该选项还将同一原始模型的所有矩阵合并，并允许您将其用户节点重新映射到新标签。必须通过重复[*MATRIX ASSEMBLE](ch13abk10.md)选项来指定来自不同原始模型的矩阵。

**产品：**Abaqus/Standard  

**类型：**模型数据  

**级别：**模型  

##### **参考：**

- ["定义矩阵，" Abaqus Analysis User's Guide第2.11.1节](../usb/usb-link.md#usb-int-idefiningmatrices)
- [*MATRIX INPUT](ch13abk13.md)

### **至少需要以下参数之一：**

MASS

将此参数设置为质量矩阵的名称。

STIFFNESS

将此参数设置为刚度矩阵的名称。

STRUCTURAL DAMPING

将此参数设置为结构阻尼矩阵的名称。

VISCOUS DAMPING

将此参数设置为粘性阻尼矩阵的名称。

### **可选参数：**

NSET

使用此参数重新映射组装矩阵的节点。将此参数设置为包含与矩阵所有用户定义节点对应的新节点标签的节点集名称。节点集的大小和集合中节点的顺序必须与此选项中指定的所有矩阵的节点组合集完全对应。如有必要，使用未排序的节点集以获得正确的映射。矩阵节点假定按生成时或矩阵数据中指定时的原始标签升序排序。

如果省略此参数，则所有用户定义节点保留其原始标签。

**此选项没有关联的数据行。**
