# 19.12.8 合并优化结果







要在可视化模块中查看优化结果，您必须将优化结果和分析结果合并为单个结果输出数据库文件，如["后处理优化，"第 19.5.3 节](pt03ch19s05s03.md)中所述。优化结果和分析结果将附加到您指定为基本结果的输出数据库文件。

**合并优化结果：**

1. 从主菜单栏中，选择****优化**![](../graphics/images/arrow.gif)**合并**![](../graphics/images/arrow.gif)***优化过程名称*****。您也可以通过在**优化过程管理器**中选择优化过程名称，然后点击**合并**来合并优化结果。出现**合并优化结果**对话框。
2. **优化结果目录**显示所选优化过程的输出文件所在目录的路径。点击![](../graphics/ico_fileOpen.png)更改路径。
3. 选择将用于创建基本结果的输出数据库文件。Abaqus/CAE 使用基本结果中定义的输出数据库文件作为合并输出数据库文件的源。在合并过程中，Abaqus/CAE 将附加数据追加到基本结果输出数据库文件。- 选择**初始**从初始优化设计循环期间生成的输出数据库文件创建基本结果。- 选择**最后**从最后一个优化设计循环期间生成的输出数据库文件创建基本结果。如果您执行了形状优化，则不应选择**最后**。- 选择**原始模型**从在优化之前运行模型单独分析生成的输出数据库文件创建基本结果。优化过程不运行原始模型的 Abaqus 分析。因此，您必须手动运行分析，然后才能选择原始模型生成的输出数据库文件作为基本结果。（原始模型的 Abaqus 输入文件存储在 `*优化过程名称*\SAVE.inp\*优化过程名称*_org.inp`。）
4. 选择将与基本结果输出数据库文件和优化结果合并的输出数据库文件。生成的输出数据库文件可能非常大，因此您应该只选择在可视化模块中查看所需的数据。- 选择**全部**将基本结果与优化过程生成的所有输出数据库文件中的分析结果合并。- 选择**最后**将基本结果与最后一个设计循环的输出数据库文件中的分析结果合并。- 选择**初始和最后**将基本结果与初始和最后设计循环的输出数据库文件中的分析结果合并。- 选择**每 n 个循环**并输入 *n* 的值，以将基本结果与从初始设计循环开始定期创建的输出数据库文件中的分析结果合并。- 选择**指定**将基本结果与指定输出数据库文件中的分析结果合并；例如，`1,5,7,9`。
5. 选择将合并到结果输出数据库文件中的**模型**和模型内的**步骤**。
6. 从出现的列表中，选择将包含在结果输出数据库文件中的**分析场变量**。Abaqus/CAE 始终将优化变量作为场变量包含在结果输出数据库文件中。优化数据取决于执行的优化类型：
7. 点击**提交**以创建结果输出数据库文件。Abaqus/CAE 显示**合并优化结果监视器**，其中包含合并操作的状态和正在合并的输出数据库文件的日志。
8. 点击**关闭**以关闭**合并优化结果监视器**。
9. 从**优化过程管理器**中，点击**结果**以在可视化模块中查看合并的结果输出数据库文件。
