# 15.15.5 定义调整点约束





您使用调整点约束将一个或多个点移动到指定表面上。有关调整点的更多信息，请参见Abaqus Analysis User's Guide第2.1.6节的"调整节点坐标"。此调整可能对组装紧固件模板模型和其他应用有用；请参见"关于组装紧固件"，第29.1.3节，和"创建组装紧固件"，第29.5节。

当主模型连接点位于螺栓孔中心点时，不应在组装紧固件模板模型中使用调整点约束。沿螺栓孔中心线的任何点都会被错误地移动到孔周长上的随机位置，而不是沿着表面法向投影到孔中心。

**创建调整点约束的步骤：**

1. 从主菜单栏选择 ****Constraint**![](../graphics/images/arrow.gif)**Create****。**提示：**您也可以使用「相互作用」模块工具箱中的![](../graphics/ico_constraint.png)工具创建调整点约束。
2. 在出现的「Create Constraint」对话框中，执行以下操作：1. 为约束命名。有关命名对象的更多信息，请参见"使用基本对话框组件"，第3.2.1节。2. 从「Type」列表中选择「Adjust points」，然后点击「Continue」。
3. 使用以下方法之一选择要移动的点或点集：- 在视口中选择一个或多个点。**提示：**「Selection」工具栏中的![](../graphics/ico_filterVisible.png)「Select the Entity Closest to the Screen」工具默认关闭。如果您进行模糊选择，Abaqus/CAE会高亮显示该点，并在视口左下角显示该点的描述。使用「Next」和「Previous」按钮循环遍历可能的选择，然后点击「OK」确认您的选择。点击鼠标按钮2表示您已完成选择。有关更多信息，请参见"在当前视口中选择对象"，第6.2节。如果模型包含网格和几何的组合，请从提示区域点击以下之一：- 如果您想从几何区域选择点或选择一个或多个参考点，请点击「Geometry」。- 如果您想从原生网格或孤立网格选择中选择点，请点击「Mesh」。- 使用现有集合定义点或点集。点击提示区域右侧的「Sets」。从出现的「Region Selection」对话框中选择现有点集合，然后点击「Continue」。**注意：**默认选择方法基于您最近使用的选择方法。要恢复为另一种方法，请点击提示区域右侧的「Select in Viewport」或「Sets」。您选择的点在视口中以红色高亮显示。
4. 在提示区域中，选择点将被移动到的区域类型。- 如果您想选择表面，请选择「Surface」。- 如果您想选择要创建基于节点表面的区域，请选择「Node Region」。
5. 使用以下方法之一选择点将被移动到的表面：- 在视口中选择一个表面。点击鼠标按钮2表示您已完成选择。如果模型包含网格和几何的组合，请从提示区域点击以下之一：- 如果您想从几何区域选择表面或选择一个或多个参考点，请点击「Geometry」。- 如果您想从原生网格或孤立网格选择中选择表面，请点击「Mesh」。- 使用现有表面或集合定义区域。点击提示区域右侧的「Surfaces」或「Sets」。从出现的「Region Selection」对话框中选择现有表面或集合，然后点击「Continue」。**注意：**默认选择方法基于您最近使用的选择方法。要恢复为另一种方法，请点击提示区域右侧的「Select in Viewport」或「Surfaces」（或「Sets」）。您选择的区域在视口中以品红色高亮显示，约束编辑器会出现。
6. 点击「OK」保存约束定义并关闭编辑器。
