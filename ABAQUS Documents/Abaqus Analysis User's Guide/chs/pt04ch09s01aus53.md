# 12.1.1 重启动分析概述

**产品：** Abaqus/Standard 和 Abaqus/Explicit

##### **参考资料**

- ["重启动分析，" 第 3.1 节](../stm/stm-link.md#stm-anl-restart)
- [*RESTART](../key/key-link.md#usb-kws-mrestart)

### 概述

重启动分析允许您：
- 从先前分析的任何时间点继续执行分析；
- 修改模型参数（如载荷、边界条件）；
- 添加新的分析步骤；
- 在不同的处理器上运行分析。

### 重启动文件

重启动分析使用以下文件：

| **文件扩展名** | **内容** |
| --- | --- |
| .res | 重启动数据文件 |
| .mdl | 模型数据文件 |
| .stt | 状态数据文件 |

### 基本重启动程序

1. **原始分析**：创建重启动文件
2. **重启动分析**：从重启动点继续

| **输入文件用法（原始分析）：** | 使用以下选项写入重启动数据： |
| --- | --- |
| | ``` [*RESTART](../key/key-link.md#usb-kws-mrestart), WRITE, FREQUENCY=*n* ``` |

| **输入文件用法（重启动分析）：** | 使用以下选项读取重启动数据： |
| --- | --- |
| | ``` [*RESTART](../key/key-link.md#usb-kws-mrestart), READ, STEP=*step*, INC=*increment* ``` |

### 重启动点选择

选择合适的重启动点非常重要：

1. **时间点**：在任意时间点重启动
2. **增量步**：在特定增量步重启动
3. **迭代**：在特定迭代重启动（仅 Abaqus/Standard）

### 修改模型

重启动分析中可以修改：

- 边界条件
- 载荷（幅值曲线可以改变）
- 分析步骤定义
- 输出请求

重启动分析中不能修改：

- 几何模型
- 材料定义
- 单元类型
- 网格

### 执行重启动

使用 oldjob 参数指定原始分析：

```
abaqus job=new_job oldjob=original_job
```

### 多步重启动

可以执行多次重启动，每次从前一次重启动点继续：

```
abaqus job=restart_step2 oldjob=restart_step1
```

### 应用场景

重启动分析常用于：

1. **长时分析**：将长时间分析分解为多个短分析
2. **参数研究**：修改载荷或边界条件
3. **恢复运行**：从意外中断中恢复
4. **后续分析**：添加额外的分析步骤

### 输出

重启动分析的输出与标准分析相同，但文件名包含重启动标识。

### 限制

- 重启动点必须是收敛的增量步
- 修改不能导致模型不稳定
- 某些单元类型不支持重启动

### 输入文件模板

#### 原始分析

```
[*HEADING](../key/key-link.md#usb-kws-mheading)
…
[*RESTART](../key/key-link.md#usb-kws-mrestart), WRITE, FREQUENCY=10
[*STEP](../key/key-link.md#usb-kws-hstep)
[*STATIC](../key/key-link.md#usb-kws-hstatic)
…
[*END STEP](../key/key-link.md#usb-kws-hendstep)
```

#### 重启动分析

```
[*HEADING](../key/key-link.md#usb-kws-mheading)
…
[*RESTART](../key/key-link.md#usb-kws-mrestart), READ, STEP=1, INC=50
[*STEP](../key/key-link.md#usb-kws-hstep)
[*STATIC](../key/key-link.md#usb-kws-hstatic)
*修改后的载荷或边界条件*
…
[*END STEP](../key/key-link.md#usb-kws-hendstep)
```
