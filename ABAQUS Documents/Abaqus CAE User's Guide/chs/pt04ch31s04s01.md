# 31.4.1 虚拟裂缝闭合技术概述





您可以使用虚拟裂缝闭合技术（VCCT）来研究准静态问题中的裂缝萌生和扩展。VCCT使用线弹性断裂力学（LEFM）原理，因此适用于沿预定义表面发生脆性裂缝扩展的问题。VCCT基于这样的假设：当裂缝扩展一定量时释放的应变能与闭合相同量裂缝所需的能量相同。

您可以在静态或准静态分析过程中包含VCCT裂缝。或者，您可以在隐式动态分析过程中包含VCCT裂缝，以模拟高速冲击载荷下结构中的断裂和失效。VCCT仅适用于Abaqus/Standard（三维实体和壳以及二维平面和轴对称模型）。您可以使用VCCT来研究包含几何体、孤立网格单元或两者组合的部件中的裂缝。您在Interaction模块中定义VCCT裂缝。您可以指定最初粘合的表面的位置。更多信息，请参见["使用扩展有限元方法将不连续性建模为富集特征"，Abaqus Analysis User's Guide第10.7.1节](../usb/usb-link.md#usb-anl-aenrichment)。

创建收敛到成功解的VCCT或增强型VCCT裂缝扩展模型需要了解VCCT背后的原理。请参见["Abaqus/Standard中使用VCCT或增强型VCCT准则的提示"中的"裂缝扩展分析"，Abaqus Analysis User's Guide第11.4.3节](../usb/usb-link.md#usb-anl-acrackpropagation-vcct-tipsas)，以获取可帮助您创建成功分析模型的信息。

详细说明请参见["为Abaqus/Standard创建VCCT裂缝"，第31.4.2节](pt04ch31s04hlb01.md)。
