# 17.7.7 哪些类型的网格无法自动生成？





有几类网格无法使用网格模块中的网格生成器创建：

**同一装配中零件实例之间的兼容网格**

兼容性意味着相邻零件实例的网格的单元面或单元边共享相同的节点，并在公共界面上具有相同的拓扑结构。您无法在实例之间指定网格兼容性。但是，您可以使用"装配"模块中的**合并/切割**工具将多个实例合并为单个零件实例。然后您可以从单个零件实例创建单个兼容网格。请参阅["零件实例之间的兼容网格，"第17.14.3节](pt03ch17s14s03.md)，了解更多相关信息。

**对称网格**

您无法确保Abaqus/CAE会为对称零件或零件实例生成对称网格。

**自底向上网格**

自底向上网格是一个手动过程。您必须为您分配了自底向上网格技术的每个区域设置并应用参数来创建网格。
