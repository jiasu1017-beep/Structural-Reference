### 全局标志

### **列表样式**

| **LIST_EXTENDEDSELECT** | 扩展选择模式，允许拖动选择范围内的项。 |
| --- | --- |
| **LIST_SINGLESELECT** | 单选模式，最多允许选择一项。 |
| **LIST_BROWSESELECT** | 浏览选择模式，强制始终选择一项。 |
| **LIST_MULTIPLESELECT** | 多选模式，用于选择单个项。 |
| **LIST_AUTOSELECT** | 自动选择光标下的项。 |
