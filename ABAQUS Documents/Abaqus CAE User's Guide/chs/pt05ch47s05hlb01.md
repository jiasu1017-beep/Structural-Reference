# 47.5.10 自定义X–Y图轴颜色和样式





### 47.5.10 自定义X–Y图轴颜色和样式

点击**Axes**选项卡来自定义*X–Y*图的图形轴的颜色和粗细。您的选择适用于*X*轴和*Y*轴；无法独立自定义轴。

**要自定义X–Y图中轴的颜色和样式：**

1. 找到**Style**选项。选择****Options**![](../graphics/images/arrow.gif)**XY Options**![](../graphics/images/arrow.gif)**Axis****；或点击![](../graphics/ico_xyAxisOptions.png)，它位于Visualization模块工具箱中的*X–Y*绘图工具中。在出现的对话框中点击**Axes**选项卡。轴的**Style**选项出现在**Axes**页面的底部。
2. 从**X Axis**或**Y Axis**字段中，高亮显示一个或多个轴。
3. 选择线条样式。从**Style**列表中，点击所需的线条样式（实线、虚线等）。所选线条样式出现在所选轴上，也出现在**Style**列表中。
4. 选择线条粗细。从**Thickness**列表中，点击所需的线条粗细。所选线条粗细出现在所选轴上，也出现在**Thickness**列表中。
5. 选择线条颜色：1. 点击颜色样本![](../graphics/usi-colorsample.png)。Abaqus/CAE显示**Select Color**对话框。2. 使用**Select Color**对话框中的方法之一选择新颜色。有关更多信息，请参见["自定义颜色，"第3.2.9节](pt01ch03s02s09.md)。3. 点击**OK**关闭**Select Color**对话框。轴线变为所选颜色。
6. 点击**Dismiss**关闭**Axis Options**对话框。
