# 72.3 创建参考点







从主菜单栏选择****Tools**![](../graphics/images/arrow.gif)**Reference Point****可在部件上或装配体上创建参考点。部件只能包含一个参考点，参考点会出现在装配体中该部件的每个实例上。或者，您可以使用Assembly、Interaction或Load模块在装配体上创建多个参考点。有关更多信息，请参见["什么是参考点？," 第72.1节](pt06ch72s01.md)。

**添加参考点的步骤：**

1. 从主菜单栏选择****Tools**![](../graphics/images/arrow.gif)**Reference Point****。**提示：**在Interaction模块中，您可以使用模块工具箱中的![](../graphics/ico_referencePoint.png)工具创建参考点。
2. 选择要作为参考点的点。您可以使用以下任一技术来定位参考点：
   - 从部件中选择任何现有的顶点，包括数据点。如果您想移动参考点，必须编辑顶点或数据点。或者，您可以删除参考点并选择新的顶点或数据点。
   - 输入表示参考点*X*、*Y*和*Z*坐标的向量分量。如果您想移动参考点，可以使用"特征操作工具集"编辑其坐标。
Abaqus/CAE在所需位置显示参考点及其标签。如果您在Part模块中的部件上创建参考点，Abaqus/CAE会将其标记为**RP**。如果您在装配体上创建参考点，Abaqus/CAE会将其标记为**RP-1**、**RP-2**、**RP-3**等。您可以通过在模型树中的特征上单击鼠标按钮3并从出现的菜单中选择**Rename**来更改参考点标签。如果需要，您可以关闭参考点符号和参考点标签的显示；有关更多信息，请参见["控制参考点显示," 第76.11节](pt07ch76hla10.md)。
