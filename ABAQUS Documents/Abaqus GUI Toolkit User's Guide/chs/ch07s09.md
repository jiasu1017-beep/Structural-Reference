# 7.9 其他动力学过程





现在我们简要回顾 Abaqus 中可用的其他动力学过程——即线性模态动力学和非线性动力学。

### 7.9.1 线性模态动力学

Abaqus/Standard 中还有几种使用模态叠加技术的线性动力学过程。与 [*MODAL DYNAMIC](../key/key-link.md#usb-kws-hmodaldyn)（在时域中计算响应）不同，这些过程在频域中提供结果，这可以提供对结构行为的额外洞察。

这些过程的完整描述在"动态应力/位移分析"《Abaqus 分析用户指南》第 6.3 节中给出。

**稳态动力学**

[*STEADY STATE DYNAMICS](../key/key-link.md#usb-kws-hsteadystdyn) 选项计算结构在用户指定的频率范围内由谐波激励引起的响应幅度和相位。典型示例包括：
- 汽车发动机支架在一系列发动机运行速度下的响应。
- 建筑物中的旋转机械。
- 飞机发动机上的组件。

**响应谱**

[*RESPONSE SPECTRUM](../key/key-link.md#usb-kws-hresponspec) 选项在结构受到其固定点的动力运动时提供峰值响应（位移、应力等）的估计。固定点的运动称为"基础运动"；一个例子是导致地面运动的地震事件。通常，当需要峰值响应的估计用于设计目的时使用该方法。

**随机响应**

[*RANDOM RESPONSE](../key/key-link.md#usb-kws-hrandomresp) 选项预测系统对随机连续激励的响应。激励使用功率谱密度函数以统计方式表达。随机响应分析的示例包括：
- 飞机对湍流的响应。
- 结构对噪声的响应，例如喷气发动机发出的噪声。

### 7.9.2 非线性动力学

如前所述，[*MODAL DYNAMIC](../key/key-link.md#usb-kws-hmodaldyn) 过程仅适用于线性问题。当对非线性动力学响应感兴趣时，必须直接积分运动方程。在 Abaqus/Standard 中，使用隐式动力学过程（[*DYNAMIC](../key/key-link.md#usb-kws-hdynamic)）执行运动方程的直接积分。当使用此过程时，质量、阻尼和刚度矩阵在每个时间点组装，并求解动态平衡方程。由于这些操作计算量大，直接积分动力学的计算成本比模态方法高。

由于 Abaqus/Standard 中的非线性动力学过程使用隐式时间积分，因此它适用于非线性结构动力学问题，例如，突发事件引发动力学响应的情况（如碰撞），或者当结构响应涉及通过塑性或粘性阻尼大量耗散能量时。在此类研究中，高频响应（在开始时很重要）很快被模型中的耗散机制阻尼掉。

非线性动力学分析的替代方案是 Abaqus/Explicit 中可用的显式动力学过程。正如第 2 章"Abaqus 基础"中讨论的，显式算法通过模型逐个单元地传播解作为应力波。因此，它最适用于应力波效应重要且模拟的事件时间很短（通常少于一秒）的应用。

与显式算法相关的另一个优势是它可以比 Abaqus/Standard 更轻松地建模不连续非线性（如接触和失效）。大型高度不连续的问题通常更容易用 Abaqus/Explicit 建模，即使响应是准静态的。显式动力学分析在第 9 章"非线性显式动力学"中进一步讨论。
