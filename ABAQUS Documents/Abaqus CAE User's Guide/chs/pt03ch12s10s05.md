# 12.10.5 指定潜热数据







材料的潜热模拟材料在相变期间内部能量的重大变化。潜热被假定为在从较低（固相线）温度到较高（液相线）温度的温度范围内释放。

如果相变发生在已知的温度范围内，您可以直接在**编辑材料**对话框中输入固相线和液相线温度。否则，您必须使用用户子程序对该效应进行建模。请参见"潜热"，Abaqus分析用户指南第26.2.4节。

**指定潜热数据：**

1. 从**编辑材料**对话框的菜单栏中，选择**热**![](../graphics/images/arrow.gif)**潜热**。（有关显示**编辑材料**对话框的信息，请参见"创建或编辑材料"，第12.7.1节。）
2. 在**数据**表中输入以下数据：**潜热**每单位质量的潜热。（单位为[JM1](../popups/usb-int-iconventions-unitsym.md)。）**固相线温度**相变发生的温度范围的下限。**液相线温度**相变发生的温度范围的上限。有关如何输入数据的详细信息，请参见"输入表格数据"，第3.2.7节。
3. 单击**确定**关闭**编辑材料**对话框。或者，您可以从**编辑材料**对话框的菜单中选择其他材料行为进行定义（有关更多信息，请参见"浏览和修改材料行为"，第12.7.2节）。
