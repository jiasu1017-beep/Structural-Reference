# 4.6.4 自定义背景图像和电影的外观





你可以通过在视口中重新定位图像或电影、沿 *X* 或 *Y* 方向拉伸或压缩图像或电影，以及调整其透明度级别来自定义背景图像或电影的外观。这些自定义选项对图像和电影都可用，但它们的设置是分开的；例如，如果你更改背景图像的 *X* 轴比例设置，可视化模块中背景电影的相同设置保持不变。

**要自定义背景图像或电影的比例、位置和透明度：**

1. 从主菜单栏中，选择****View**![](../graphics/images/arrow.gif)**Image/Movie Options****。**图像/电影选项**对话框出现，默认选中**图像**选项卡。要自定义背景电影，请单击**电影**选项卡。
2. 在**图像**页面上切换**显示图像**，或在**电影**页面上切换**动画期间显示电影**以激活自定义选项。
3. 为背景图像或电影选择**位置和比例**选项之一：
   **适应视口**
   此选项将背景图像或电影放置在视口中心，并将其缩放到当前视口的**宽度**或**高度**。此外，你可以选择**最佳**选项，该选项沿导致图像变形最小的方向缩放背景电影。
   **自动对齐**
   此选项使你能够将背景图像或电影对齐到视口中九个位置之一：其中心、四个角落中的任何一个，或四条边中任意一条的中心。展开**对齐**列表，然后从列表中的图形对齐图示中选择一个选项。
   **手动**
   此选项使你能够将背景图像放置在视口中心以外的位置。单击**手动**，然后在**原点**字段中输入你要放置背景图像左下角的位置。
4. 对于**自动对齐**或**手动**缩放，你可以指定沿任一轴的缩放量。为背景图像或电影指定 **X 比例**或 **Y 比例**。默认比例值为 1；增加比例会沿选定轴拉伸图像或电影，或减小值会沿选定轴压缩图像或电影。
5. 将**透明度**滑块拖动到你想要的透明度百分比。此设置更改背景图像或电影的透明度，并将其与显示的模型混合。值为 0.00 是透明的，值为 1.00 是不透明的。默认情况下，背景图像或电影和模型以不透明透明度显示。
6. 单击**确定**应用更改并关闭**图像/电影选项**对话框。
