# 9.2.1 Abaqus/CAE模型包含什么？





Abaqus/CAE模型包含以下类型的对象：
- 零件
- 材料和截面
- 装配
- 集合和曲面
- 步骤
- 载荷、边界条件和场
- 相互作用及其属性
- 网格

模型数据库可以包含任意数量的模型，这样您可以将与单个问题相关的所有模型保存在一个数据库中。（有关更多信息，请参阅["什么是Abaqus/CAE模型数据库？，"第9.1节"](pt02ch09s01.md)。）您可以同时从模型数据库中打开多个模型，并且可以在不同的视口中处理不同的模型。视口标题栏（如果可见）显示与视口关联的模型的名称。与当前视口关联的模型（由红色边框指示）称为当前模型，并且只有一个当前模型。[图9-1](pt02ch09s01.md#dbs-models)显示了两个视口显示同一模型数据库（`crankshaft.cae`）中的两个不同模型（`high-speed`和`low-speed`）；[图9-1](pt02ch09s01.md#dbs-models)中的当前视口显示的是`high-speed`模型。

您使用**模型管理器（Model Manager）**或主菜单栏上的**模型（Model）**菜单项来创建和管理模型。您使用上下文栏中的**模型（Model）**列表来切换到当前模型数据库中的不同模型。

您可以在模型数据库内创建模型的副本；此外，您还可以在模型之间复制以下对象：
- 草图
- 零件（零件集合也会被复制）
- 实例
- 材料
- 截面（包括连接器截面）
- 轮廓
- 幅值
- 相互作用属性

有关详细说明，请参阅["在模型数据库中操作模型，"第9.8.1节"](pt02ch09s07hlb01.md)和["在模型之间复制对象，"第9.8.3节"](pt02ch09s07hlb03.md)。

您还可以从另一个模型数据库文件导入模型，这会在当前模型数据库中创建模型的完整副本。有关更多信息，请参阅["从Abaqus/CAE模型数据库导入模型，"第10.5.1节"](pt02ch10s05s01.md)。

当您提交模型进行分析时，Abaqus/CAE会检查模型是否完整。例如，如果您请求动态分析，则必须指定材料的密度，以便可以计算模型的质量和惯性属性。如果您在"属性（Property）"模块中未提供材料密度，"作业（Job）"模块会报告错误；有关更多信息，请参阅["监控分析作业的进度，"第19.2.6节"](pt03ch19s02hlb01.md)。

在某些模块中，Abaqus/CAE不支持Abaqus/Standard、Abaqus/Explicit或Abaqus/CFD中您可能希望包含在分析中的功能。您可以通过使用**关键词编辑器（Keywords Editor）**编辑与模型关联的Abaqus关键词来添加此类功能。从主菜单栏选择****模型（Model）**![](../graphics/images/arrow.gif)**编辑关键词（Edit Keywords）**![](../graphics/images/arrow.gif)***模型名称*****来启动**关键词编辑器**。（您可以通过从主菜单栏选择****帮助（Help）**![](../graphics/images/arrow.gif)**关键词浏览器（Keyword Browser）****来查看Abaqus/CAE支持的关键词。）

您可以指定模型使用先前分析的信息。当您提交模型进行分析时，Abaqus/CAE会从选定的步骤继续分析。有关更多信息，请参阅["配置重启输出请求，"第14.13.2节"](pt03ch14s13hlb02.md)和["重启分析，"第19.6节"](pt03ch19s06.md)。
