# 69.7.9 融合面







您可以创建融合面来连接模型中现有的面。Abaqus/CAE 用新面封闭两组连接边之间的空间。您可以选择使新面与所选边相切，使用它们之间的最短路径连接边，或指定新面在所选边之间必须遵循的路径。[图 69-11](pt06ch69s07hlb09.md#rep-blend-face) 显示了具有所选边高亮显示的两个面。在此示例中，切向方法创建了一个与两个原始面相切的光滑半径面，最短路径方法创建了一个平面，选择路径方法创建了具有所选线边轮廓的面。

**图 69-11** 使用切向、最短路径和指定路径方法创建融合面。

![](../graphics/rep-blend-face.png)

您还可以创建融合面来连接边和面。Abaqus/CAE 使用最短路径方法在所选边和面之间创建融合面。当您在边和现有面之间创建融合面特征时，新面总是垂直于现有面。自由边根据需要扩展或缩短以与面的边相交。[图 69-12](pt06ch69s07hlb09.md#rep-blend-edgetoface) 显示了所选边以及通过将它们与平面面融合而创建的特征。中心边的两端是从平面面的垂直投影，左边和右边远端扩展以与平面面的边相遇。

**图 69-12** 在边和面之间创建融合面。

![](../graphics/rep-blend-edgetoface.png)

融合操作作为部件的特征存储；因此，您可以使用模型树删除您创建的面。

**融合面：**

1. 从主菜单栏中选择 ****Tools**![](../graphics/images/arrow.gif)**Geometry Edit****（工具→几何编辑）。Abaqus/CAE 显示 **Geometry Edit**（几何编辑）对话框。**提示：**您也可以使用位于 Part（部件）模块工具箱中编辑工具处的 ![](../graphics/ico_blendFaces.png) 工具来融合面。 有关工具箱中编辑工具的图表，请参阅["使用几何编辑工具集，" 第 69.1 节"](pt06ch69s01.md)。
2. 从对话框中选择 **Face**（面）类别和 **Blend**（融合）方法。
3. 选择融合面第一侧的边。您可以单独选择边或按边角度选择。 有关在视口中选择对象的更多信息，请参阅["在当前视口中选择对象，" 第 6.2 节"](pt01ch06s02.md)。您也可以选择现有的边集合。**注意：**默认选择方法基于您最近使用的选择方法。要恢复为其他方法，请点击提示区域右侧的 **Select in Viewport**（在视口中选择）或 **Sets**（集合）。从提示区域点击 **Done**（完成）。
4. 为融合面第二侧选择边或面。如果您为第二侧选择面，则第 3 步中选择的边必须在面上有垂直投影，否则 Abaqus/CAE 将无法为选择创建融合面。从提示区域点击 **Done**（完成）以完成您的选择并打开 **Blend Faces**（融合面）对话框。
5. 选择以下可用融合方法之一：**Tangent**（切向）Abaqus/CAE 将创建与包含第一侧和第二侧边的面相切的融合面。要使用此方法，第 3 步和第 4 步中选择的边必须是自由边——仅属于一个特征面的边——以便 Abaqus/CAE 计算切向性。**Shortest path**（最短路径）Abaqus/CAE 将使用它们之间的最短路径创建连接第一侧和第二侧边的融合面。如果您为融合的第二侧选择了面或多个面，这是唯一可用的融合方法。**Specify path**（指定路径）在边之间指定路径。点击 **Select**（选择）选择一个现有的边或线特征，该边或线特征连接第一侧边之一的一端到第二侧边之一的一端。Abaqus/CAE 将使用所选路径的轮廓创建融合面。
6. 点击 **Preview**（预览）预览将从当前选择 resulting 的融合面。
7. 点击 **OK**（确定）。Abaqus/CAE 创建融合面特征。融合过程在第 3 步使用最近使用的选择方法重新开始。
8. 要退出融合过程，请执行以下操作之一 - 点击提示区域中的取消按钮 ![](../graphics/fts-cancel.gif)，或 - 在 Abaqus/CAE 窗口中的任意位置点击鼠标按钮 2，或 - 从几何编辑工具集或 Part（部件）模块中的工具中选择其他操作。
