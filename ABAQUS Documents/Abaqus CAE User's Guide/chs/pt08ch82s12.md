# 82.12 绘制幅值数据





**Amplitude Plotter** 插件使您能够创建 *X–Y* 绘图，这些绘图以图形方式显示模型中一个或多个幅值定义的数据。您可以绘制以下类型的幅值对象的数据：
- Tabular
- Equally spaced
- Periodic
- Modulated
- Decay
- Smooth step

有关在 Abaqus/CAE 中定义和使用这些幅值类型的更多信息，请参见 [第57章 "The Amplitude toolset](pt06ch57.md)"。

您可以通过在 Interaction 模块或 Load 模块中从主菜单栏选择 ****Plug-ins**![](../graphics/images/arrow.gif)**Tools**![](../graphics/images/arrow.gif)**Amplitude Plotter**** 来启动 **Amplitude Plotter**。要绘制幅值数据，请在对话框上半部分选择一个或多个幅值行，然后点击 **Plot**。如果多个幅值对象属于相同的幅值类型，则可以将它们包含在同一 *X–Y* 绘图中。

默认情况下，**Amplitude Plotter** 创建的 *X–Y* 绘图的大小可以包含所选幅值或多个幅值中的所有数据。创建 *X–Y* 绘图时，您可以通过使用 **Min X Value** 或 **Max X Value** 选项指定自定义最小值或最大值来放大幅值数据中的特定区域。与 Abaqus/CAE 中的任何 *X–Y* 绘图一样，您也可以通过双击绘图的这些组件来自定义绘图、其轴或绘图图例；请参见 ["自定义 X--Y 绘图外观，" 第47.7节](pt05ch47s06.md)，了解有关这些自定义选项的更多信息。**Amplitude Plotter** 还包含一个 **Display plot legend** 选项，您可以切换它来显示或隐藏绘图图例。

切换 **Save XY Data** 选项，以为包含在 *X–Y* 绘图中的幅值对象创建命名的 *X–Y* 数据对象。您可以自定义创建的 *X–Y* 数据对象的名称；Abaqus/CAE 在绘图图例中显示这些名称，并且 *X–Y* 数据对象将保存为会话对象，可在 Visualization 模块中使用。**Amplitude Plotter** 生成的 *X–Y* 数据对象是特定于会话的，不能保存到模型数据库。
