# *POROUS BULK MODULI







### *POROUS BULK MODULI定义土壤和岩石的体积模量。

此选项用于定义固体颗粒和渗透流体的体积模量，以便在多孔介质分析中考虑其可压缩性。[*POROUS BULK MODULI](ch16abk19.md)选项不能与多孔金属塑性材料模型结合使用。

**产品：**Abaqus/Standard  Abaqus/CAE  

**类型：**模型数据  

**级别：**模型  

**Abaqus/CAE：**属性模块

##### **参考：**

- ["多孔体积模量，" Abaqus Analysis User's Guide第26.6.3节](../usb/usb-link.md#usb-mat-cporousbulkmod)

**此选项没有关联的参数。**

### **定义体积模量的数据行：**

**第一行：**

根据需要重复此数据行以将体积模量定义为温度的函数。
