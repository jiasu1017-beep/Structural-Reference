# 10.3 Abaqus/CAE 中接触检测工具的增强







**产品：**Abaqus/CAE  

**优点：**当搜索涉及同一实例上不同区域的接触对时，您现在可以包含模型实例。此增强提高了 Abaqus/CAE 中接触检测工具的可用性。

**说明：**接触检测算法已扩展，当您搜索涉及同一实例上不同区域的接触对时，可以包含模型实例。

**Abaqus/CAE 使用方法：**
```
Interaction 模块:
    ****Interaction**![](../graphics/images/arrow.gif)**Find contact pairs****: 选择 **Whole model** 或选择 **Instance** 并从视口中选择子部件实例，打开 **Include pairs with surfaces on the same instance**
```
