# 3.14.17 使用超泡沫传递结果





**产品：**Abaqus/Standard  Abaqus/Explicit

### 测试的单元

CPS4R    CPS3    CPE4R    CPE3    CAX4R    CAX3    C3D8R    C3D4    C3D6

M3D4R    M3D3    S3R    S4R    SAX1

C3D10M    CPE6M    CPS6M    CAX6M

### 问题描述

本节的验证测试由在顺序导入分析中承受单调递增压缩载荷的单元模型组成。测试序列包括从Abaqus/Standard到Abaqus/Explicit再到Abaqus/Standard的结果传递。

还包括了一些具有增强沙漏控制的一阶减缩积分单元的验证测试。

材料性能与["弹性泡沫测试数据的拟合，"Abaqus基准指南第3.1.5节](../bmk/bmk-link.md#bmk-mat-foamdatafitting)中使用的相同。

### 结果与讨论

结果表明，超泡沫材料模型在Abaqus/Explicit和Abaqus/Standard之间成功传递。
