# 62.5.1 An overview of the methods for creating a datum point







When you choose **Point** from the **Create Datum** dialog box, the **Method** list displays the following methods for creating a datum point:

**![](../graphics/ico_dtmPtCoordinates.png) Enter coordinates **

Enter the *X*-, *Y*-, and *Z*-coordinates of the datum point, as shown in [Figure 62--6](pt06ch62s05s01.md#dtm-hptc). For detailed instructions, see ["Creating a datum point by entering its coordinates," Section 62.6.1](pt06ch62s06hlb01.md).

**Figure 62–6** Creating a datum point by entering coordinates.

![](../graphics/dtm-hptc-nls.png)

**![](../graphics/ico_dtmPtOffsetPoint.png) Offset from point**

Enter the location of the datum point in the form of the *X*-, *Y*-, and *Z*-coordinates of an offset from a selected point, as shown in [Figure 62--7](pt06ch62s05s01.md#dtm-hptop). For detailed instructions, see ["Creating a datum point at an offset from a selected point," Section 62.6.2](pt06ch62s06hlb02.md).

**Figure 62–7** Creating a datum point by offsetting from a point.

![](../graphics/dtm-hptop-nls.png)

**![](../graphics/ico_dtmPtMidpoint.png) Midway between 2 points**

Select two points on the model; Abaqus/CAE creates the datum point midway between the two selected points, as shown in [Figure 62--8](pt06ch62s05s01.md#dtm-hpmid). For detailed instructions, see ["Creating a datum point midway between two points," Section 62.6.3](pt06ch62s06hlb03.md).

**Figure 62–8** Creating a datum point by selecting two end points.

![](../graphics/dtm-hpmid-nls.png)

**![](../graphics/ico_dtmPtOffsetEdges.png) Offset from 2 edges **

Select two edges on the model; and enter the distance from the datum point to each edge, as shown in [Figure 62--9](pt06ch62s05s01.md#dtm-hpted). For detailed instructions, see ["Creating a datum point at a specified distance from two edges," Section 62.6.4](pt06ch62s06hlb04.md).

**Figure 62–9** Positioning a datum point a specified distance from two edges.

![](../graphics/dtm-hpted-nls.png)

**![](../graphics/ico_dtmPtParameter.png) Enter parameter**

Select an edge on the model, and enter the location of the datum point in the form of a parameter value that represents a percentage of the edge length. An arrow along the edge indicates the direction of increasing parameter value from the start vertex (corresponding to an edge parameter value of zero) to the end vertex (corresponding to a value of one), as shown in [Figure 62--10](pt06ch62s05s01.md#dtm-hpted2). For detailed instructions, see ["Creating a datum point by entering an edge parameter," Section 62.6.5](pt06ch62s06hlb05.md).

**Figure 62–10** Positioning a datum point a specified distance along an edge.

![](../graphics/dtm-hpted2-nls.png)

**![](../graphics/ico_dtmPtProjectOnFace.png) Project point on face/plane**

Select a point and a face or plane on which to project the point. Abaqus/CAE creates the datum point where the face or plane intersects a line that is normal to it and passing through the selected point, as shown in [Figure 62--11](pt06ch62s05s01.md#dtm-hptpf). The datum point also marks the shortest distance between the selected point and the selected face or plane. For detailed instructions, see ["Creating a datum point by projecting a point on a face or plane," Section 62.6.6](pt06ch62s06hlb06.md).

**Figure 62–11** Creating a datum point by projecting a point onto a face.

![](../graphics/dtm-hptpf-nls.png)

**![](../graphics/ico_dtmPtProjectOnLine.png) Project point on edge/datum axis**

Select a point on the model and an edge or datum axis on which to project the point. Abaqus/CAE creates the datum point where the edge or datum axis intersects a line that is normal to it and passing through the selected point, as shown in [Figure 62--12](pt06ch62s05s01.md#dtm-hptpe). The datum point also marks the shortest distance between the selected point and the selected edge or datum axis. For detailed instructions, see ["Creating a datum point by projecting a point on an edge or datum axis," Section 62.6.7](pt06ch62s06hlb07.md).

**Figure 62–12** Creating a datum point by projecting a point onto an edge.

![](../graphics/dtm-hptpe-nls.png)
