# 64.7.2 从现有网格生成偏移的壳单元层





从主菜单栏选择****网格**![](../graphics/images/arrow.gif)**编辑****可以从现有网格的所选单元面法向偏移生成壳单元层，如下图所示：

![](../graphics/edt-shell-offset-nls.png)

**生成壳单元层的步骤：**

1. 进入"网格"模块。
2. 从上下文栏的**对象**字段中选择**零件**并从列表中选择一个零件。
3. 从主菜单栏选择****网格**![](../graphics/images/arrow.gif)**编辑****。Abaqus/CAE显示**编辑网格**对话框。 **提示：**您也可以使用位于"网格"模块工具箱底部的![](../graphics/ico_meshEdit.png)工具显示**编辑网格**对话框。
4. 在对话框中执行以下操作： 1. 在**类别**字段中选择**网格**。 2. 从**方法**列表中选择**偏移（创建壳层）**。
5. 使用以下方法之一选择Abaqus/CAE将从中生成壳单元层的单元面。您可以选择壳单元面或实体单元面，但不能同时选择两者。如果您选择壳单元面，相邻壳单元的表面法线方向必须一致。您可以使用**翻转法线**工具反转壳单元的表面法线方向。有关更多信息，请参见["反转壳单元的表面法线方向，" 第64.6.3节](pt06ch64s06s03.md)。 **逐个选择单元** 1. 从提示区域的菜单中选择**逐个**。 2. 选择单元面。 3. **按住[Shift]键并点击**其他单元面将其添加到您的选择中。 4. 如有需要，**按住[Ctrl]键并点击**所选单元面取消选择。 5. 完成单元面选择后，点击鼠标按钮2。 **指定现有曲面** 1. 点击提示区域右侧的**曲面**。Abaqus/CAE显示**区域选择**对话框，其中包含您已创建的曲面列表。 2. 选择曲面，然后点击**继续**。曲面必须仅包含四边形或三角形面。 **注意：**默认选择方法基于您最近使用的选择方法。要切换到另一种方法，请点击提示区域右侧的**在视口中选择**或**曲面**。 **使用角度方法选择单元** 1. 从提示区域的菜单中选择**按角度**。 2. 输入角度（0到90之间），然后选择一个单元面。Abaqus/CAE会选择从所选面到相邻单元面的每个相邻单元面，直到相邻单元面之间的角度等于或超过您输入的角度。（有关更多信息，请参见["使用角度和特征边缘方法选择多个对象，" 第6.2.3节](pt01ch06s02hlb03.md)。） 3. 使用**按角度**方法后，您可以**按住[Shift]键并点击**其他单元面将其添加到您的选择中，或**按住[Ctrl]键并点击**所选单元面取消选择。（有关更多信息，请参见["组合选择技术，" 第6.2.8节](pt01ch06s02hlb08.md)。） 4. 完成单元面选择后，点击鼠标按钮2。
6. 从出现的**偏移网格 - 壳层**对话框中执行以下操作： 1. 如果您选择了要偏移的壳单元面，您可以选择表示所需偏移方向的颜色。您可以选择的两种颜色对应于所选单元面的两侧。您也可以选择**两者**以指示Abaqus/CAE应在所选壳单元的两侧创建壳单元层。 2. 输入**层之间的距离**。您可以输入0；例如，如果您想创建共享节点的多个皮肤加强层。 3. 输入**层数**。 4. 如有需要，切换**指定初始偏移**并输入初始偏移的值。负初始偏移表示单元之间的过盈。 5. 如果要偏移的表面包含尖锐角落，请切换**恒定厚度环绕角落**。如果您预计生成的偏移单元可能会坍缩或倒置，也应该打开此选项。有关更多信息，请参见["减少网格偏移期间的单元畸变和坍缩，" 第64.3.3节](pt06ch64s03s03.md)。 6. 要将第一层壳单元的节点与所选单元面的节点共享，请切换**与基础壳/曲面共享节点**。仅当第一层壳单元未从所选单元面偏移时，此选项才可用。 7. 要在创建壳单元层后删除所选壳单元，请切换**删除基础壳单元**。此选项仅在您选择要偏移的孤兒壳单元面时可用。 8. 如果您想创建包含偏移壳网格中所有单元的单个集合，请切换**为新单元创建集合**。 9. 如果您想为偏移壳网格每层的单元创建单独的集合，请切换**每层使用单独的集合**。当Abaqus/CAE为每层创建单独集合时，它会在集合名称前缀后附加**Layer-*layer number***。 10. 如有需要，为单个集合或单独集合的前缀输入新名称。
7. Abaqus/CAE会沿所选单元面法向偏移创建壳单元层。完成壳单元层生成后，点击鼠标按钮2或提示区域中的取消按钮![](../graphics/afxI_cancel.png)退出过程。 **提示：**如果在生成单元层时出错，点击**编辑网格**对话框中的**撤销**可以撤消更改。Abaqus/CAE会将网格恢复到操作之前的状态。

编辑孤兒网格后，您应始终验证基于节点、单元和曲面的特征（如截面分配、载荷和相互作用），以确保它们正确应用于修改后的网格。
