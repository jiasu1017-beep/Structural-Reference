# 2 安装 Abaqus





本章描述 Abaqus 文档、许可和产品的安装过程，以及卸载这些组件的过程。完整的 Abaqus 安装需要两张 DVD：一张用于文档，一张用于许可和产品。

在启动 Abaqus 安装程序之前，您应确认您的计算机与此版本兼容。您可以使用许可和产品 DVD 上提供的系统信息实用程序来执行此操作。系统信息实用程序会检查您的计算机并报告它找到的软件组件。使用以下命令运行系统实用程序：

**Linux 平台**

```
/*dvd_dir*/setup -sysinfo
```

其中 *dvd_dir* 是 DVD 挂载时使用的目录或安装文件所在的目录。

**Windows 平台**

请执行以下操作之一：
- 从启动屏幕，选择**启动 Abaqus 系统信息实用程序**。
- 从命令行，键入

```
*dvd_drive*:\setup.exe -sysinfo
```

其中 *dvd_drive* 是 DVD-ROM 驱动器。

将您的系统配置与 Program Directory 中描述的 Abaqus 系统要求进行比较。Program Directory 是类似于发布说明的附加文档，可从 [http://media.3ds.com/support/progdir](http://media.3ds.com/support/progdir) 获取。选择 SIMULIA Abaqus 作为产品**系列**，选择 Abaqus 6.14 作为**级别**，然后在左侧窗格中选择**先决条件**。访问需要支持合同和私有登录账户（参见 [https://iam.3ds.com/self_service/login/service/websupport](https://iam.3ds.com/self_service/login/service/websupport/)/）。

安装使用的磁盘空间将计入 Abaqus 安装目录所有者的账户。在 Windows 平台上，安装活动需要管理员权限。

完整的 Abaqus 安装包括三个过程。您应按以下顺序执行安装过程：

1. [Abaqus 文档](ch02s02.md)。
2. [Abaqus 许可](ch02s03.md)。您可以使用 FLEXnet 许可或 Dassault Systmes 许可，取决于您拥有的许可证文件类型。
3. [Abaqus 产品](ch02s04.md)。

Abaqus 文档应安装在网络上至少一台计算机上。Abaqus 文档应在许可和产品之前安装，因为产品安装过程使用文档 URL 配置对在线文档的网络访问。

在安装过程期间，只要有可能就提供默认值；强烈建议您使用默认值，除非您有充分的理由更改它们。SIMULIA 目录结构旨在方便 Abaqus 的多个版本。为充分利用此目录结构，Abaqus 产品、Abaqus 许可和 Abaqus 文档应安装在同一个 SIMULIA 父目录中。目录结构的说明请参见[附录 B，"Abaqus 发布目录和文件](ap02.md)"。

您可以在已挂载的网络驱动器上安装 Abaqus，并配置用户的计算机将 Abaqus 作为客户端运行。有关更多信息，请参阅["Windows 客户端安装，"第 D.2 节](ap04s02.md)。
