# 20.9.5 重新对齐草图网格







在某些情况下，草图网格与草图的顶点和线或底层参考几何不对齐。如果使用 Sketcher 自定义选项重新对齐草图网格，您可能会发现创建所需的草图更容易。如果将现有草图添加到新的空白草图（例如用于新零件），Abaqus/CAE 会自动重新对齐草图网格。自定义选项允许您执行以下操作：
- 通过选择代表网格原点的顶点来相对于草图移动网格。为了帮助您找到草图的原点，草图的 *X* 和 *Y* 轴由在原点相交的虚线指示。
- 通过从草图中选择一条将与网格 *X* 轴平行的线来相对于草图旋转网格。
- 将网格重置为原始草图坐标。

重新对齐网格不会更改草图中的现有特征。但是，更改会应用于任何新的草图实体。例如，如果您旋转网格，现有的水平构造线不会改变；但是，新的水平构造线将与新的 *X* 轴平行。尺寸例外；水平和垂直尺寸——新的和现有的——与默认网格旋转对齐。

当您更改网格原点或旋转并创建新的草图几何时，Abaqus/CAE 会显示两组光标坐标。网格坐标相对于草图网格的当前对齐方式；它们显示在视口左上角。草图坐标相对于草图网格的原始对齐方式；它们显示在视口右上角，如果您重新对齐网格，它们不会改变。当两组坐标相同时，Abaqus/CAE 会在视口左上角显示单一坐标组。

**重新对齐草图网格：**

1. 从 Sketcher 工具箱底部，选择自定义工具![](../graphics/ico_sketchOptions-nls.png)。出现**Sketcher 选项**对话框。
2. 从**常规**选项卡页面中心的按钮，执行以下操作：- 点击**原点**相对于零件移动草图网格的原点。从草图中选择新原点或在提示区域输入原点的坐标。- 点击**角度**相对于零件旋转草图网格。选择一条将与草图 *X* 轴平行的线。- 点击**重置**恢复原始草图原点并恢复 *X* 轴的原始对齐方式。重置网格会使网格坐标与草图坐标对齐。您的网格对齐更改会立即应用到草图。
3. 点击**确定**应用您所做的任何其他自定义并关闭**Sketcher 选项**对话框。
