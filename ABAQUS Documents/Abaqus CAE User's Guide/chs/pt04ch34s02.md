# 34.2 管理载荷工况





**Load Case Manager**允许您组织和操作与给定模型关联的载荷工况。您通过从Load模块的主菜单栏中选择****Load Case**![](../graphics/images/arrow.gif)**Manager****来访问管理器。[图34-1](pt04ch34s02.md#usi-adv-lcmanager)显示了**Load Case Manager**。

**图34-1** **Load Case Manager**。

![](../graphics/adv-lcmanager-nls.png)

管理器左侧列中的图标允许您抑制和恢复现有载荷工况。管理器中的**Create**、**Edit**、**Copy**、**Rename**或**Delete**按钮允许您创建新的载荷工况或编辑、复制，重命名和删除现有载荷工况。您还可以使用主菜单栏中的**Load Case**菜单启动create、edit、copy、rename和delete过程。选择主菜单栏中的管理操作后，过程与在管理器对话框中单击相应按钮完全相同。

有关创建和编辑载荷工况的详细说明，请参见["定义载荷工况"，第34.5节](pt04ch34hla01.md)。
