# 50.1 可视化模块中查询工具集概述





查询工具集允许您获取有关模型的一般信息。可视化模块在消息区域显示请求的信息，相同的信息被写入重放文件。选择 ****工具（Tools）**![](../graphics/images/arrow.gif)**查询（Query）**** 从主菜单栏，或从 **查询（Query）** 工具栏中选择 ![](../graphics/ico_info.png) 工具来使用查询工具集。

**常规查询（General Queries）** 下的项目提供以下信息：

**节点**

您可以获取所选节点的标签、变形和未变形坐标以及位移的信息。

**距离**

您可以获取两个所选节点之间变形和未变形距离以及节点之间相对位移的信息。

**角度**

您可以获取在三个所选节点之间形成的变形和未变形角度的信息。您选择的第二个节点是角度的顶点。

**单元**

您可以获取所选单元的网格类型、材料、截面、连通性以及积分点位置的当前场输出变量的信息。此查询仅在选择输出数据库时可用。

**网格**

您可以获取有关当前输出数据库的名称、模型中的节点和单元数量以及单元类型的信息。

**质量属性**

您可以获取输出数据库中整个模型或模型一部分的基本质量属性。此查询仅在选择输出数据库时可用。Abaqus/CAE 返回以下信息：
- 体积
- 体积形心
- 质量
- 质心
- 关于质心或关于指定点的惯性矩

有关常规模型查询的更多信息，请参阅 ["在可视化模块中查询模型，" 第 50.2.1 节"](pt05ch50s02hlb01.md)。

**可视化模块查询（Visualization Module Queries）** 下的项目提供以下信息：

**探针值**

当您在整个当前视口中移动光标时，Abaqus/CAE 会在 **探针值（Probe Values）** 对话框中显示信息。探测模型绘图会显示模型数据和结果；探测 X–Y 绘图会显示 X–Y 曲线数据。有关探测的更多信息，请参阅 [第 51 章，"探测模型"](pt05ch51.md)。"

**应力线性化**

应力线性化是通过截面将应力分离为恒定薄膜应力和线性弯曲应力。Abaqus 执行应力线性化计算并以 X–Y 绘图的形式显示结果。应力线性化仅适用于输出数据库。有关应力线性化的更多信息，请参阅 [第 52 章，"计算线性化应力"](pt05ch52.md)。"

**活动单元和节点**

Abaqus/CAE 在当前视口中显示所有活动节点或活动单元的标签编号。有关更多信息，请参阅 ["查询活动节点或单元标签，" 第 50.2.2 节"](pt05ch50s02hlb02.md)。

**层压板堆叠绘图**

Abaqus/CAE 创建一个新视口并显示复合材料叠层的代表性图像。该图像显示叠层中的层以及每层的详细信息，例如其纤维方向、厚度、参考平面和积分点。层压板堆叠绘图仅适用于输出数据库。有关更多信息，请参阅 [第 53 章，"查看层压板堆叠绘图"](pt05ch53.md)。"

# 50.2.1 在可视化模块中查询模型





您可以使用查询工具集来获取当前视口中模型几何形状的一般信息。

**查询模型：**

1. 找到 **查询（Query）** 对话框。从主菜单栏中，选择 ****工具（Tools）**![](../graphics/images/arrow.gif)**查询（Query）**** 或点击 **查询（Query）** 工具栏中的 ![](../graphics/ico_info.png) 工具。出现 **查询（Query）** 对话框。
2. 要获取特定节点的信息，请执行以下操作：1. 从 **查询（Query）** 对话框的 **常规查询（General Queries）** 字段中选择 **节点（Node）**。Abaqus/CAE 在提示区域显示提示。2. 从视口中选择一个节点。该节点的未变形和变形 X、Y 和 Z 坐标以及节点的位移出现在消息区域中。消息区域中出现的相同信息也被写入重放文件。**注意：**要调整消息区域的大小，请拖动顶部边缘；要查看已滚出消息区域的信息，请使用右侧的滚动条。
3. 要获取两个节点之间距离的信息，请执行以下操作：1. 从 **查询（Query）** 对话框的 **常规查询（General Queries）** 字段中选择 **距离（Distance）**。Abaqus/CAE 在提示区域显示提示。2. 从视口中选择两个节点。以下信息出现在消息区域：- 每个节点的未变形和变形 X、Y 和 Z 坐标，以及节点的位移。- 节点之间的绝对未变形和变形距离。- 两个节点之间未变形和变形向量的 X、Y 和 Z 分量。- 节点之间的绝对相对位移。- 两个节点之间相对位移的 X、Y 和 Z 分量。
4. 要获取由三个节点形成的角度的信息，请执行以下操作：1. 从 **查询（Query）** 对话框的 **常规查询（General Queries）** 字段中选择 **角度（Angle）**。Abaqus/CAE 在提示区域显示提示。2. 从视口中选择三个节点。您选择的第二个节点是角度的顶点。以下信息出现在消息区域：- 每个节点的未变形和变形 X、Y 和 Z 坐标，以及节点的位移。- 节点之间的绝对未变形和变形角度。
5. 要获取输出数据库中特定单元的信息，请执行以下操作：1. 从 **查询（Query）** 对话框的 **常规查询（General Queries）** 字段中选择 **单元（Element）**。Abaqus/CAE 在提示区域显示提示。2. 从视口中选择一个单元。以下信息出现在消息区域：- 单元的标签、单元类型、材料和截面。- 连接单元的标签。- 积分点位置的当前场输出变量。
6. 要获取网格的一般信息，请从 **查询（Query）** 对话框的 **常规查询（General Queries）** 字段中选择 **网格（Mesh）**。以下信息出现在消息区域：- 当前输出数据库的名称。- 节点数量。- 单元数量。- 单元类型。
7. 要获取输出数据库中质量属性的一般信息，请执行以下操作：1. 从 **查询（Query）** 对话框的 **常规查询（General Queries）** 字段中选择 **质量属性（Mass properties）**。Abaqus/CAE 在提示区域显示提示，并在消息区域显示默认选择方法 **所有单元（All elements）** 的质量属性。**注意：**如果您已在当前会话中使用过 **质量属性（Mass properties）** 查询并选择了不同的选择方法，Abaqus/CAE 会在提示区域显示您上次使用的选择方法，但不会在消息区域生成新结果。2. 从提示区域列表中选择以下选择方法之一：- **所有单元（All elements）** - **从视口选择单元（Select elements from viewport）** - **部件实例（Part instances）** - **单元集（Element sets）** - **截面（Sections）** - **材料（Materials）** - **单元类型（Element types）** - **显示组（Display groups）** Abaqus/CAE 在提示区域显示附加选择字段，这取决于您选择的选择方法。3. 按如下方式完成您的选择：- 如果您在 previous step 中选择了 **从视口选择单元（Select elements from viewport）**，请从视口进行选择（有关更多信息，请参阅 [第 6 章，"在视口中选择对象"](pt01ch06.md)）。您可以在提示区域选择一个部件实例并输入单个单元编号以获取该单元的质量信息。- 对于其余选择方法，请从提示区域的列表中选择部件实例、单元集、截面、材料、单元类型或显示组名称，或从视口进行选择。当您从提示区域选择一个对象或点击 **完成（Done）** 完成视口选择时，Abaqus/CAE 会在消息区域显示您选择的质量属性。4. 如果您选择中的任何项目的密度或厚度定义不佳，请在提示区域点击 **选项（Options）** 以显示 **质量属性查询选项（Mass Properties Query Options）** 对话框，并指定一个值来替代 Abaqus/CAE 的未定义量。点击对话框中的 **确定（OK）** 保存您的条目并关闭对话框，或点击 **重置（Reset）** 返回到之前的设置。Abaqus/CAE 会在消息区域为您选择的项目显示质量属性。以下属性可用：- 表面积（仅适用于壳单元）- 面积形心- 体积- 体积形心- 质量- 质心- 关于质心或关于指定点的惯性矩
8. 完成信息请求后，关闭 **查询（Query）** 对话框。

# 50.2.2 查询活动节点或单元标签





您可以使用查询工具集来请求当前视口中所有当前活动节点或活动单元的标签编号。活动节点和单元是当前显示组中的那些实体，减去活动视图切割从中移除的任何实体。Abaqus/CAE 为每个适用的部件实例在提示区域显示标签编号，并将标签编号打印到重放文件。

当您请求活动节点或单元时，Abaqus/CAE 仅显示标签编号；它不提供任何其他节点或单元数据和结果。要显示有关单个节点或单元的更多信息，请分别参阅 ["探测节点，" 第 51.2.2 节"](pt05ch51s02hlb02.md) 或 ["探测单元，" 第 51.2.3 节"](pt05ch51s02hlb03.md)。

**显示活动节点或活动单元标签：**

1. 找到 **可视化模块查询（Visualization Module Queries）** 选项。从主菜单栏中，选择 ****工具（Tools）**![](../graphics/images/arrow.gif)**查询（Query）**** 或点击 **查询（Query）** 工具栏中的 ![](../graphics/ico_info.png) 工具。出现 **查询（Query）** 对话框，**可视化模块查询（Visualization Module Queries）** 显示在对话框的下部。
2. 点击 **活动节点（Active nodes）** 或 **活动单元（Active elements）**。在提示区域，Abaqus/CAE 为模型中的每个部件实例显示所有活动节点或活动单元的标签列表。

- ["探测单元，" 第 51.2.3 节"](pt05ch51s02hlb03.md)
