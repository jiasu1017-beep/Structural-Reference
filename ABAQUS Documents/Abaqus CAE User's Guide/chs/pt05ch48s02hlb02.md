# 48.2.2 创建或编辑点列表路径





路径是通过在模型中指定一系列点来定义的线。在点列表路径中，所有点都是模型坐标位置。这些位置可能与节点重合，也可能不重合。有关路径规格的替代形式的信息，请参阅["创建穿过模型的路径，"第48.2节](pt05ch48s02.md)。

您可以通过将点的坐标输入表格来创建点列表路径。对于与节点位置重合的点，您可以指定节点标签，Abaqus会将节点的未变形坐标输入表格。为了帮助确定感兴趣的节点标签，您可能需要在创建路径之前生成带有节点符号和节点编号可见的模型图。有关详细信息，请参阅["自定义模型标签，"第55.5节](pt05ch55s05.md)。

**创建或编辑点列表路径：**

1. 找到路径创建和编辑选项。**要创建新路径：**1. 从主菜单栏中，选择****工具**![](../graphics/images/arrow.gif)**路径**![](../graphics/images/arrow.gif)**创建****。"创建路径"对话框出现。2. Abaqus在"名称"文本字段中显示路径的默认名称。要提供更有意义的名称，请将此默认值替换为您选择的名称（包括空格如果您希望的话）。3. 在"创建路径"对话框中，单击"点列表"；然后单击"继续"。**要编辑现有路径：**- 从主菜单栏中，选择****工具**![](../graphics/images/arrow.gif)**路径**![](../graphics/images/arrow.gif)**编辑****。从出现的菜单中，选择要编辑的路径。**提示：**您也可以使用"路径管理器"来编辑路径。从主菜单栏中，选择****工具**![](../graphics/images/arrow.gif)**路径**![](../graphics/images/arrow.gif)**管理器****以显示管理器。选择要编辑的路径，然后单击管理器右侧的"编辑"按钮。"编辑点列表路径"对话框出现。
2. 在"点坐标"表中，使用标准键盘和鼠标编辑技术输入、修改或删除坐标。要输入坐标，请使用以下任一技术：**输入点的坐标值：**1. 指定为X、Y、Z坐标，用空格或逗号分隔；例如，1.0, 3.4, 2.0。2. 单击**[Enter]**。**包含节点位置的坐标：**1. 选择要确定其节点坐标的部件实例。单击"部件实例"字段旁边的箭头以查看可用部件实例的列表。要获取节点的坐标，请在"节点标签"字段中输入节点标签；然后单击**[Enter]**或"查询"。Abaqus显示节点的未变形X、Y和Z坐标。2. 要在当前高亮行之前、之后或位置将这些坐标输入路径定义表，请分别单击"添加之前"、"添加之后"或"替换"。有关特殊表编辑选项或从ASCII文件读取数据，请按鼠标按钮3。（有关详细信息，请参阅["输入表格数据，"第3.2.7节](pt01ch03s02s07.md)。）在当前视口中的模型图上，Abaqus高亮显示路径中包含的所有点，并显示路径连接。
3. 完成后，单击"确定"。如果创建了新路径，Abaqus/CAE会将路径名称添加到"路径管理器"列表中。默认情况下，Abaqus/CAE保存您在会话期间指定的路径。如果要保留路径以供后续会话使用，请将其保存到XML文件、模型数据库或输出数据库；有关详细信息，请参阅["管理会话对象和会话选项，"第9.9节](pt02ch09s08.md)。
