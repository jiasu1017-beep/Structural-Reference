# 47.2.5 从ASCII文件读取X–Y数据









### 47.2.5 从ASCII文件读取X--Y数据

您可以从包含由逗号、制表符或空格分隔的数值数据列的ASCII文本文件读取*X–Y*数据。文件可以包含多于两列数据；您可以指定哪些列（字段）对应于*X*和*Y*轴数据。您还可以指定要从文件读取哪些行数据。

**要从ASCII文件读取X–Y数据：**

1. 找到**来自ASCII文件的XY数据**选项：从主菜单栏选择****工具**![](../graphics/images/arrow.gif)**XY数据**![](../graphics/images/arrow.gif)**创建****。在出现的对话框中点击**ASCII文件**；然后点击**继续**。**提示：**您也可以通过点击**XY数据管理器**中的**创建**或使用工具箱中的![](../graphics/ico_xyData.png)工具来指定*X--Y*数据。出现**来自ASCII文件的XY数据**对话框。
2. 使用以下方式之一输入ASCII文件的名称：- 在对话框顶部的**文件**窗口中输入名称，或- 点击**选择**以过滤和浏览现有文件名。出现**ASCII文件选择**对话框。过滤和浏览现有文件。找到您要的文件后，点击名称以选择它；然后点击**确定**。
3. 指定要读取文件的哪些列。列可以由空格、制表符或逗号分隔。多个连续空格、制表符和/或逗号被解释为单个字段分隔符。1. 要指定文件中要读取为*X*值的列，请在**从字段读取X值**框中输入一个整数。默认是字段1，即第一列。2. 要指定文件中要读取为*Y*值的列，请在**从字段读取Y值**框中输入一个整数。默认是字段2，即第二列。
4. 要为*X*或*Y*列中的值分配量类型，展开**X**或**Y**列的**量类型**列表，然后从出现的列表中选择量类型。
5. 指定要读取文件的哪些行；默认是所有行。- 点击**全部读取**以读取文件的每一行，或- 点击**跳过**；然后在**读取间隔行数**框中输入要跳过的文件行数。值为0表示读取所有行；值为1表示每隔一行读取。读取总是从第一行开始。
6. 要评估和显示数据，点击**绘图**。当前视口中出现*X--Y*图。该图表示您在对话框中配置的数据，Abaqus认为这是临时数据，无论您是否点击**另存为**保存它。
7. 要保存您配置的数据，点击**另存为**。**注意：**要绘制您保存的*X--Y*数据对象，请从主菜单栏选择****工具**![](../graphics/images/arrow.gif)**XY数据**![](../graphics/images/arrow.gif)**绘图****，然后从弹出菜单中选择*X--Y*数据对象。
8. 完成后，点击**关闭**以关闭对话框。
