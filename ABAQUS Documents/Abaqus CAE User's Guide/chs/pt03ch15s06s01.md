# 15.6.1 接触检测对话框





要使用接触检测，请从主菜单栏选择 ****Interaction**![](../graphics/images/arrow.gif)**Find contact pairs**** 或 ****Constraint**![](../graphics/images/arrow.gif)**Find contact pairs****。接触检测对话框会出现（请参见[图15-1](pt03ch15s06s01.md#itn-contactdetector-initial)）。最初没有识别的接触对。

**图15-1** 接触检测工具的初始视图。

![](../graphics/itn-contactdetector-initial-nls.png)

使用接触检测工具是一个两步过程：首先，Abaqus/CAE搜索模型中很可能相互作用的表面；然后，您有机会在创建相互作用和约束之前审查已识别的表面并修改默认接触对参数。您提供一些基本标准来指导搜索。这些标准包括搜索域和可能接触的表面之间的距离。

输入必要的搜索标准后，点击「Find Contact Pairs」开始搜索。Abaqus/CAE会更新接触对候选表（如[图15-2](pt03ch15s06s01.md#itn-contactdetector-table)所示）。

**图15-2** 接触对候选表。

![](../graphics/itn-contactdetector-table-nls.png)

 您可以为表中的每个接触对候选创建一个接触相互作用或绑定约束。您也可以通过点击相应的表格单元格来修改相互作用或约束定义的参数（请参见[图15-3](pt03ch15s06s01.md#itn-contactdetector-modify)）。

**图15-3** 在接触对候选表中更改单元格值。

![](../graphics/itn-contactdetector-modify-nls.png)

 在表格上点击鼠标按钮3会显示扩展选项菜单，并允许您手动向表中添加接触对。当您切换「Show previously created interactions and ties」时，任何先前存在的相互作用和绑定约束都会被添加到接触对候选表中；您可以以与新检测到的接触对候选相同的方式修改现有接触对。

接触对候选表中显示的相互作用和约束在您点击「OK」之前不会成为模型的一部分。当您完成接触对的参数设置后，点击「OK」。Abaqus/CAE会根据指定的参数同时为表中的每个接触对创建接触相互作用和绑定约束。创建的相互作用和约束会被添加到模型树和「Interaction Manager」中；您可以使用任一界面来审查、修改、抑制和删除创建的相互作用。

有关使用自动接触检测工具的详细说明，请参见「使用接触和约束检测」，第15.16节。
