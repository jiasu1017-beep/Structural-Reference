# 45.4 生成自由体节点力符号图





您可以创建显示模型中各个节点上自由体节点力总和的符号图。符号场输出变量FREEBODY是节点处各个NFORC*n*分量的总和，但向量方向相反。FREEBODY输出变量还默认考虑显示的单元；相比之下，NFORC1的云图考虑整个模型。

FREEBODY符号图是当前显示组所代表的对象上不平衡力的表示。这种对当前显示组的敏感性在任何其他情况下都不会表现出来，例如云图。

**要生成将节点力显示为自由体切割的符号图：**

1. [打开包含已包含NFORC数据的分析结果的输出数据库](pt02ch09s06hlb02.md)。
2. 选择要显示的结果[步骤和帧](pt05ch42s01hlb01.md)。
3. 如有需要，使用[显示组](pt07ch78s02hlb01.md)将模型聚焦到其子集。
4. 选择**FREEBODY**作为要显示的[符号场输出变量](pt05ch42s03hlb05.md)。符号图状态被激活。
5. 选择您想要的绘图状态独立和符号图[自定义](pt05ch40s03s03.md)选项。
6. 从主菜单栏中，选择****Plot**![](../graphics/images/arrow.gif)**Symbols****来修改是在变形模型形状、未变形形状还是两者上绘制符号。**提示：**您还可以使用工具箱中的变形![](../graphics/ico_plotSymbolDeformed.png)、未变形![](../graphics/ico_plotSymbolUndeformed.png)或叠加![](../graphics/ico_plotSymbolDefUndef.png)符号工具生成符号图。当前视口更改为显示当前显示组在当前输出数据库指定步骤和帧的自由体节点力的自定义符号图。每次您在步骤和帧选择器对话框、绘图状态独立选项、叠加绘图选项（如果适用）或符号图选项对话框中单击**Apply**时，Abaqus会自动刷新您的符号图。
