### 7.5.1 AFXPickStep

`AFXPickStep`类允许用户在当前视口中选择实体。您必须按与使用关键字的选择步骤相同的顺序创建与选择步骤关联的关键字。例如，如果您有两个选择步骤，您必须在创建第二个关键字（传递给第二个选择步骤）之前创建传递给第一个选择步骤的关键字。按与使用关键字的选择步骤相同的顺序创建与选择步骤关联的关键字，可确保必要的设置命令按正确顺序发出，以使命令正常工作。

在从视口选择项目时，您可以指定许多参数。您可以在`AFXPickStep`构造函数中指定其中一些参数，并通过调用选择步骤的各种方法指定其他参数。

要构造选择步骤，您必须至少提供以下内容：

- 一个过程
- 一个对象关键字
- 提示行的提示
- 指定可选择实体类型的位标志或标志

以下示例显示如何编写选择步骤：

```
class MyProcedure(AFXProcedure):

    def __init__(self, owner):

        AFXProcedure.__init__(self, owner)

        self.cmd = AFXGuiCommand(self, 'myMethod', 'myObject')
        self.nodeKw = AFXObjectKeyword(self.cmd, 'node', True)

    def getFirstStep(self):

        return AFXPickStep(self, self.nodeKw,
            'Select a node', AFXPickStep.NODES)
```

构造函数中的可选参数允许您指定以下内容：

- 用户是否应选择一个实体或一个或多个实体（AFXPickStep.ONE，默认值，或AFXPickStep.MANY）
- 高亮级别（1--4）
- 序列样式（AFXPickStep.ARRAY，默认值，或AFXPickStep.TUPLE）

如果只允许用户选择一个实体，过程将在用户选择实体后自动前进到下一步；但是，用户可以返回上一步更改选择。如果允许用户选择一个或多个实体，用户必须通过单击鼠标按钮2或单击提示行上的"完成"按钮来提交选择。

高亮级别控制所选实体的颜色。在某些过程中，不同的步骤使用不同的颜色来区分选择。

序列样式控制拾取的对象序列如何在命令字符串中表示。如果序列样式为AFXPickStep.ARRAY，拾取的对象将表示为数组切片的拼接；例如，v[3:4] + v[5:8]，其中v是顶点数组。您不能使用AFXPickStep.ARRAY序列样式来选择多种类型实体的组合，因为只能拼接相同类型的对象。此外，您不能使用AFXPickStep.ARRAY序列样式来选择有意义的点，因为有意义的点是动态构造的，无法从数组切片访问。

如果序列样式为AFXPickStep.TUPLE，拾取的对象将表示为单个对象的元组；例如，(v[3], v[5], v[6], v[7])。您选择的样式取决于您打算发出的命令接受的格式。Abaqus/CAE中某些命令接受两种样式，但有些只接受其中一种或另一种。有关`AFXPickStep`构造函数的参数的更多详细信息，请参阅[Abaqus GUI工具包参考指南](../gui/gui-link.md#gui)。

### 7.5.2 细化用户可选择的内容

细化限定了`AFXPickStep`构造函数中指定的可选择实体类型。以下示例显示如何仅选择直线边：

```
step = AFXPickStep(self, self.edgeKw, 'Select a straight edge',
    AFXPickStep.EDGES)
step.setEdgeRefinements(AFXPickStep.STRAIGHT)
```

默认情况下，不设置细化。有关细化的完整列表，请参阅[Abaqus GUI工具包参考指南](../gui/gui-link.md#gui)。

### 7.5.3 不可选择的实体

默认情况下，过程模式会阻止先前选择的几何实体在同一过程中被再次选择。如果你不希望此行为，可以调用`allowRepeatedSelections`方法。以下示例显示如何允许重复选择：

```
step = AFXPickStep(self, self.edgeKw, 'Select a straight edge',
    AFXPickStep.EDGES)
step.allowRepeatedSelections(True)
```

禁止重复选择仅适用于几何项目（如顶点、边和面），不适用于节点和单元。

### 7.5.4 选择时的高亮显示

当用户取消过程时，过程模式会清除所有高亮显示。此外，过程模式在返回之前会清除当前步骤中的高亮显示。高亮实体的颜色由`AFXPickStep`构造函数中设置的高亮级别控制。

### 7.5.5 选择选项

"选择选项"对话框自动在任何选择步骤中可用。"选择选项"对话框中的可用选项基于用户选择的实体类型自动配置。例如，如果用户只选择面，"面"只出现在对话框的组合框中。同样，如果用户选择单个实体，拖动形状和拖动范围按钮不可用。因此，过程通常不需要显式设置可用选择选项。如果需要设置这些选项，可以使用过程的`setSelectionOptions`方法。你必须在创建第一个选择步骤之前设置过程选择选项。有关更多信息，请参阅[Abaqus GUI工具包参考指南](../gui/gui-link.md#gui)。

通常，过程只在过程开始时设置这些选项。但是，在过程中，用户可能会更改设置，并且在过程剩余步骤中，从步骤到步骤会保留修改后的设置。

### 7.5.6 允许用户输入点

如果你想允许用户输入点的坐标作为在视口中选择的替代方案，可以调用`addPointKeyIn`方法并向其传递一个元组关键字。`addPointKeyIn`方法在提示行上显示一个文本字段。传入`addPointKeyIn`方法的关键字类型决定从用户收集什么值；例如，两个或三个值，以及这些值是浮点类型还是整数类型。例如，在过程的构造函数中，你可以定义一个额外关键字，如以下代码所示：

```
    self.pointKw1 = AFXObjectKeyword(self.cmd, 'point', True)
    self.pointKw2 = AFXTupleKeyword(self.cmd, 'point', True,
        3, 3, AFXTUPLE_TYPE_FLOAT)
```

在过程的一个步骤中，你可以添加一个键入选项，如下所示：

```
    step = AFXPickStep(self, self.pointKw1, 'Select a point',
        AFXPickStep.POINTS)
    step.addPointKeyIn(self.pointKw2)
```

如果步骤有键入文本字段，用户在文本字段中输入一些值，然后用户通过按"Enter"提交值，这些值将用于命令。或者，如果步骤有键入文本字段并且用户在视口中选择一个实体，该实体将用于命令，无论文本字段中输入了什么内容。模式会自动处理根据这些规则停用需要停用的关键字。在上一个示例中，如果用户输入一个点，*self.pointKw1*将被停用，*self.pointKw2*将被激活。此外，*self.pointKw2*将包含用户输入的值。

### 7.5.7 按角度选择

当适用时，按面角度或边角度选择始终是启用的。例如，当用户选择面时，启用按面角度选择。你不能禁用按角度选择。

### 7.5.8 AFXOrderedPickStep

AFXOrderedPickStep是一种特殊的选择步骤，保留用户选择实体的顺序。例如，当选择四个节点来创建四边形单元时，用户选择节点的顺序很重要，必须在选择过程中保留。用户必须一次选择一个实体，不能拖动选择它们。此外，因为这是一个将所选实体视为单一选择的单个步骤，用户不能备份任何单独的选择。步骤继续循环，直到用户点击鼠标按钮二。

### 7.5.9 预填充选择步骤

在某些情况下，你可能希望使用一些选择预填充选择步骤。例如，假设你有一个创建对象的过程，包括选择模型的一个区域，并且你想允许用户编辑该对象。在编辑过程中，你需要用创建对象时用户最初所做的选择预填充区域选择步骤。你可以通过向选择步骤添加一组实体来实现，如下所示：

```
step = AFXPickStep(self, self.nodesKw, 'Select some nodes',
    AFXPickStep.NODES, AFXPickStep.MANY, 1, AFXPickStep.ARRAY)
step.addNodeSetSelection('NodeSet-1')

```

还有用于从几何集（`addGeometrySetSelection`）、单元集（`addElementSetSelection`）和曲面（`addSurfaceSelection`）添加实体的类似方法。当执行此步骤时，添加的实体将自动高亮显示，用户可以添加或删除该选择。

### 7.5.10 选择时的限制

以下限制适用于选择过程：

- 以下实体无法选择：集合、曲面
- 同时选择多种类型的实体不支持ARRAY序列样式；例如，用户不能在同一步骤中选择节点和单元。
- 选择特征或实例不能与选择其他类型的实体组合。此外，不支持ARRAY序列样式。
- 不支持取消选择属于已选实体的实体。例如，当用户选择一个面时，Abaqus也会选择属于所选面的所有边。用户不能取消选择其中任何一条边。
- 探针不受支持。

这些限制可能在未来版本的Abaqus GUI工具包中移除。
