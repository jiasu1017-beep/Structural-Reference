# 36.5 为表皮或桁条加固分配单元类型





您使用Property模块在模型中的部件上创建表皮或桁条加固。您使用Mesh模块为表皮或桁条加固分配单元类型。当提示您选择要分配单元类型的区域时，必须从**Selection**工具栏的对象类型列表中选择**Skins**或**Stringers**。

当您为几何部件创建表皮或桁条时，Abaqus/CAE会向几何实体添加属性。当您对部件进行网格划分时，此属性用于生成表皮或桁条单元，这些单元与部件的底层单元共享节点。

您可以在屈曲分析中使用桁条加固，并可以在耦合结构-声学分析中使用表皮加固。如果声介质与结构相邻，则会在界面处发生结构-声学耦合。建议您使用基于表面的绑定约束来强制耦合；但是，如果您进行结构-声学子模型分析，其中声域形成子模型，则必须用声学-结构界面（ASI）单元排列子模型边界的界面部分。您可以通过在界面上创建表皮并为表皮分配ASI单元来排列界面单元。有关在几何体和孤立网格上使用表皮创建ASI单元的详细信息，请参见Dassault Systmes Knowledge Base，网址为[www.3ds.com/support/knowledge-base](http://www.3ds.com/support/knowledge-base)。
