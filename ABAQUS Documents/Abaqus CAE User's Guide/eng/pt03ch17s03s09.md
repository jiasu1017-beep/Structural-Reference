# 17.3.9 Mesh verification







The Mesh module provides a set of tools that allow you to verify a mesh and to obtain mesh statistics and mesh information. The Mesh module also provides geometry diagnostic tools that will help you determine why Abaqus/CAE cannot mesh a region. For more information, see ["Verifying your mesh," Section 17.6.1](pt03ch17s06s01.md); ["Querying your mesh," Section 17.6.2](pt03ch17s06s02.md); and ["Using the geometry diagnostic tools," Section 71.2.4](pt06ch71s02hlb04.md).
