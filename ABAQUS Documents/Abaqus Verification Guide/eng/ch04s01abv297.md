**Product: **Abaqus/Standard  

### Element tested

DC2D4

### Feature tested

User subroutine to define a simple thermal constitutive behavior.

### Problem description

User subroutine [`UMATHT`](../sub/sub-link.md#sub-xsl-umatht) is used to define the thermal behavior of the material for a transient heat transfer analysis. Isotropic conductivity and a constant specific heat for the material are assumed. The heat conduction in the material is governed by Fourier's law, and the gradient of the heat flux with respect to temperature is zero. The material properties—namely, conductivity and specific heat—are defined and are passed into the user subroutine via the `PROPS` array.

Verification problem [ec24dfd2.inp](../eif/ec24dfd2.inp) solves the same problem using the standard data line options to define conductivity and specific heat.

### Results and discussion

The results of [umathtxx.inp](../eif/umathtxx.inp) match those obtained from [ec24dfd2.inp](../eif/ec24dfd2.inp).
