# 19.11.2 对共同执行执行数据检查







数据检查将模型的输入文件通过分析输入文件处理器运行，以确保模型的一致性并设置所有必需的模型选项。对共同执行执行数据检查将验证为共同执行创建的每个作业中的模型信息。

要对共同执行执行数据检查，请从主菜单栏中选择****共同执行**![](../graphics/images/arrow.gif)**数据检查**![](../graphics/images/arrow.gif)***共同执行名称*****。您也可以通过在**共同执行管理器**中选择共同执行名称，然后点击**数据检查**来执行数据检查。

共同执行中涉及的作业出现在模型树中**共同执行**容器内**作业**容器中的作业下。您可以通过在模型树中右键点击作业并选择**监视器**来显示作业监视器对话框，以查看数据检查的结果。有关更多信息，请参见["监视分析作业的进度，"第 19.2.6 节](
