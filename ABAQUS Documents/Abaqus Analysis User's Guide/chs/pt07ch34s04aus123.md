### 概述

热载荷用于模拟热量传递问题中的各种热边界条件。Abaqus 支持多种类型的热载荷，包括集中热流、分布热流、热通量和辐射。

### 热载荷类型

| 类型 | 描述 | 输入方式 |
| --- | --- | --- |
| CFLUX | 集中热流 | 节点 |
| DFLUX | 分布热通量 | 单元面 |
| FILM | 薄膜载荷 | 单元面 |
| RADiosity | 辐射载荷 | 单元面 |

### 集中热流（ CFLUX ）

| **输入文件用法：** | ```
*CFLUX
node_or_node_set, dof, magnitude
``` |
| --- | --- |

### 分布热通量（ DFLUX ）

均匀分布热通量：

| **输入文件用法：** | ```
*DFLUX
element_or_element_set, S, flux_value
``` |
| --- | --- |

其中 S 表示表面热通量。

非均匀热通量需要使用用户子程序：

| **输入文件用法：** | ```
*DFLUX, USER
element_or_element_set, S
``` |
| --- | --- |

### 薄膜载荷（ FILM ）

薄膜载荷用于模拟对流换热：

| **输入文件用法：** | ```
*FILM
element_or_element_set, S, film_coefficient, sink_temperature
``` |
| --- | --- |

### 辐射载荷（ RADiosity ）

辐射载荷：

| **输入文件用法：** | ```
*RADIATE
element_or_element_set, S, emissivity, ambient_temperature
``` |
| --- | --- |

### 使用幅值曲线

热载荷可以与幅值曲线结合使用：

| **输入文件用法：** | ```
*CFLUX, AMPLITUDE=name
node_or_node_set, dof, magnitude
``` |
| --- | --- |

### 热载荷与结构耦合

在耦合热-结构分析中，热载荷会导致结构变形：

| **输入文件用法：** | ```
*COUPLED TEMPERATURE-DISPLACEMENT
...
*DFLUX
element_set, S, heat_flux
*LOAD
...
``` |
| --- | --- |
