# 55.4.2 缩放坐标和缩小模型





您可以通过缩放来放大、缩小或以其他方式扭曲模型形状,您可以围绕每个单元的形心缩小每个单元。缩放修改X-、Y-和Z-方向中每个节点坐标。缩小围绕每个单元的形心均匀减小每个单元的尺寸。例如,[图55-6](pt05ch55s04hlb02.md#viw-shrink)中显示的单元已缩小系数0.40。

**图55-6** 显示缩小效果的单元。

![](../graphics/viw-shrink.png)

缩放和缩小选项是曲线图状态无关的;也就是说,您为所有未变形、变形、云图、符号和材料取向曲线图设置一次。您也可以设置Abaqus/CAE在变形形状叠加在未变形形状上时将应用于未变形形状的缩放和缩小选项。

**缩放或缩小模型的步骤:**

1. 找到公共或叠加"**Scaling**"选项。公共缩放选项始终应用于变形形状;它们适用于在任何曲线图状态(未变形、变形、云图、符号或材料取向)中单独绘制未变形形状时。当未变形形状与变形形状一起绘制时,应用叠加缩放选项。从主菜单栏,选择 ****Options**![](../graphics/images/arrow.gif)**Common**** 或 **Superimpose**;然后在出现的对话框中点击"**Other**"选项卡。点击"**Scaling**"选项卡;"**Scaling**"选项出现。
2. 切换"**Shrink elements**"以请求或抑制围绕每个单元形心缩小每个单元。 1. 当"**Shrink elements**"打开时,缩小"**Factor**"滑块变得可用。 2. 拖动"**Factor**"滑块到您想要的缩小因子。值为0.00表示不缩小。值为0.90将所有单元缩小到点。
3. 切换"**Scale coordinates**"以请求或抑制按节点坐标X-、Y-和Z-方向进行的模型缩放。 1. 当"**Scale coordinates**"打开时,缩放因子变得可用。点击"**X**"、"**Y**"和"**Z**"框分别输入X、Y和Z节点坐标的缩放因子。
4. 点击"**Apply**"实施您的更改。模型显示根据您的规格进行更改。默认情况下,您的更改会在会话期间保存,并会影响当前视口和从当前视口创建的任何新视口中的所有后续曲线图。如果要为后续会话保留您应用的更改,请将它们保存到文件。更多信息,请参阅 ["为后续会话保存自定义," 第55.1.1节](pt05ch55s01s01.md)。
