# 1.3.16 悬臂夹芯梁：剪切柔性壳







**产品：**Abaqus/Standard  

### 测试的单元

S4    S4R    S8R    S4T    S4RT    S8RT    

### 问题描述

![](../graphics/versandwichbeam-model.png)

**材料：**

对于面板，采用线弹性材料，弹性模量 = 1.0  107，泊松比 = 0。对于芯材，使用 LAMINA 定义给出横向剪切模量为 1.0  104，平面内所有其他属性设置为可以忽略的值。

**边界条件：**

一端的所有节点均为固定端。

**载荷：**

![](../graphics/ver_eqn00160.gif) 750.0 一致分布到自由端的节点。

对于 S4、S4R 和 S8R 单元，壳截面使用高斯积分。

对于 S4T、S4T、S4RT 和 S8RT 单元，壳截面使用 Simpson 积分。

### 参考解

自由端位移（Plantema，《Sandwich Construction》，John Wiley and Sons, Inc.，1966）：![](../graphics/ver_eqn00172.gif)= 5.5684。

固定端顶部的最大弯曲应力（防止翘曲的情况）：![](../graphics/ver_eqn00173.gif) = 3.7275  105。
