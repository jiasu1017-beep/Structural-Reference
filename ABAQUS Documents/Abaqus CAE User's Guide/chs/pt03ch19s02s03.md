# 19.2.3 作业管理器





**作业管理器**与 Abaqus/CAE 中的其他管理器类似，允许您执行以下操作：
- 创建分析作业并将新作业与所选模型或输入文件关联。
- 编辑所选分析作业。
- 复制、重命名或删除所选分析作业。**注意：**与模型关联的作业只能复制到与模型关联的作业。与输入文件关联的作业只能复制到与输入文件关联的作业。

此外，**作业管理器**允许您执行以下操作：- 在不提交分析的情况下为基于模型的作业[写入](pt03ch19s07hlb02.md)输入文件。
- 对模型执行[数据检查](pt03ch19s07hlb03.md)。
- [提交](pt03ch19s07hlb04.md)作业进行分析。
- 执行数据检查后[继续](pt03ch19s07hlb05.md)分析直至完成。
- [监控](pt03ch19s02hlb01.md)分析进度。
- 查看作业的[结果](pt03ch19s07hlb07.md)。
- [终止](pt03ch19s07hlb06.md)当前正在运行的作业。

您可以通过从主菜单栏选择 ****作业**![](../graphics/images/arrow.gif)**管理器**** 来显示**作业管理器**。[图 19-1](pt03ch19s02s03.md#ana-jobman) 显示了**作业管理器**的布局。

**图 19-1** **作业管理器**。

**作业管理器**的四列显示以下内容：

**名称**

**名称**列显示作业的名称。单击**重命名**可以重命名所选作业。

**模型**

**模型**列显示与作业关联的模型或输入文件的名称。

**类型**

**类型**列显示在您使用作业编辑器配置作业时选择的作业类型。作业类型可以是以下之一：
- **完整分析**
- **恢复**
- **重启**

（有关更多信息，请参见[「选择作业类型，」第 19.2.5 节](pt03ch19s02s05.md)。）只要作业不在运行，您就可以使用作业编辑器更改作业类型。

**状态**

**状态**列显示分析作业的当前状态，在您的作业运行时持续更新。状态可以是以下之一：

**无**

作业尚未提交进行分析。

**已提交检查**

输入文件已写入，模型正在提交进行数据检查。

**正在检查**

模型的数据检查正在运行。

**检查完成**

模型的数据检查已成功完成；您现在可以继续进行完整分析。

**已提交**

输入文件已写入，作业正在提交进行完整分析。

**正在运行**

作业已提交进行完整分析并且正在运行。

**已完成**

分析完成。您可以单击**结果**来查看输出数据库的内容并以图形方式验证您的结果。

**中止**

由于输入文件中的致命错误或磁盘空间不足等问题，作业已中止。

**已终止**

作业已被用户终止。
