# 23.1 复合铺层概述





[图 23-1](pt04ch23s01.md#usi-layups-threeplies) 展示了一个包含三个铺层的单一复合铺层。 每个铺层由具有均匀厚度的均匀材料组成，纤维沿单一方向排列。 但是，铺层也可以是各向同性材料，例如泡沫芯。

**图 23-1** 复合铺层。

![](../graphics/usi-layups-threeplies.png)

铺层代表在复合制造过程中放置在模具中的单片材料。 复合铺层在不同区域可以包含不同数量的铺层。 例如，[图 23-1](pt04ch23s01.md#usi-layups-threeplies) 中的复合铺层包括包含单个铺层的区域、两个铺层重叠的区域和三个铺层重叠的区域。

[图 23-2](pt04ch23s01.md#usi-layups-sections) 显示了[图 23-1](pt04ch23s01.md#usi-layups-threeplies) 中所示的相同模型，但使用复合壳截面定义。 铺层数量不能在复合壳截面内更改。 因此，需要四个复合壳截面，每个截面具有恒定的铺层数量，来定义这个简单模型。 截面1包含单个铺层，截面2和3各包含两个铺层，截面4包含三个铺层。

**图 23-2** 复合壳截面。

![](../graphics/usi-layups-sections-nls.png)

Abaqus/CAE中的复合铺层旨在帮助您管理典型复合模型中的大量铺层。 相比之下，复合截面是有限元分析的产品，可能难以应用于实际应用。 除非您的模型相对简单且所有铺层覆盖相同的区域，否则您会发现随着铺层数量的增加，使用复合截面对模型进行定义变得越来越困难。 添加新铺层或移除或重新定位现有铺层也可能很麻烦。

使用Abaqus/CAE创建复合铺层的过程反映了创建真实复合零件的过程——您从基本形状开始，然后向选定区域添加不同材料和厚度的铺层，并定向铺层以在特定方向上提供最大的强度。 Abaqus/CAE复合铺层编辑器允许您轻松添加铺层、选择应用它的区域、指定其材料属性并定义其方向。 您还可以从文本文件中的数据读取铺层中铺层的定义。 当数据存储在电子表格中或由第三方工具生成时，这很方便。 您还可以在铺层中抑制铺层，并尝试添加和移除不同方向铺层的效果。

指定铺层中纤维的正确方向非常重要。 Abaqus/CAE允许您为铺层定义参考方向，也可以为铺层中的每个铺层定义参考方向。 此外，您可以指定铺层中纤维相对于铺层参考方向的方向。 默认情况下，铺层的坐标系统与零件的坐标系统相同； 类似地，铺层的坐标系统与铺层的坐标系统相同。 方向定义在["理解复合铺层和方向，" 第23.3节](pt04ch23s03.md)中有更详细的描述。
