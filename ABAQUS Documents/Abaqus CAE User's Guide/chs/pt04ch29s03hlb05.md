# 29.3.5 调整紧固件定义





您可以使用**Adjust**标签页指定紧固件的方向。有关如何组合连接器方向和紧固件方向的描述，请参见["定义紧固件方向"中的"独立于网格的紧固件"，Abaqus Analysis User's Guide第35.3.4节](../usb/usb-link.md#usb-cni-afastener-orientation)。您还可以指定Abaqus将如何计算影响半径，影响半径决定将与每个紧固点关联的节点组。

**要调整紧固件定义：**

1. 从**Edit Fasteners**对话框中，显示**Adjust**标签页。
2. 默认情况下，Abaqus根据距离紧固点最近的表面的默认局部方向确定每个紧固件的方向。单击![](../graphics/ico_selectBlue.png)以选择指定紧固件方向的基准坐标系（**CSYS**）。单击![](../graphics/ico_dtmCsys.png)以创建尚不存在的基准坐标系。**CSYS**的*z*轴应大致垂直于正在连接的面，局部*x*和*y*轴应大致切于正在连接的面。
3. 默认情况下，Abaqus调整您指定的方向，使每个紧固件的局部*z*轴垂直于距离紧固点最近的表面。关闭**Adjust CSYS to make Z-axis normal to closest surface**以使用所选坐标系精确指定局部方向。
4. 每个紧固点与表面上紧固点附近区域中的一组节点相关联，称为影响区域。然后，紧固点的运动通过分布耦合约束以加权方式耦合到该区域中节点的运动。每个影响区域至少包含三个节点。选择以下任一方式来定义影响区域：
   - 选择**Facet-based default**以允许Abaqus根据紧固件的几何属性、连接面片的特征长度以及您选择的加权方法计算内部影响半径。默认影响半径始终选择为内部计算的影响半径、物理紧固件半径以及投影点到最近节点距离中的最大值。
   - 选择**Specify**，然后输入影响半径。如果您指定的影响半径小于计算的默认影响半径，Abaqus使用计算值。更多信息，请参见["定义影响半径"中的"独立于网格的紧固件"，Abaqus Analysis User's Guide第35.3.4节](../usb/usb-link.md#usb-cni-afastener-radius)。
