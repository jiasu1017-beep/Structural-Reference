# FXObject







所有 FOX 对象的基类。
![](../graphics/gui-fxobject.png)

### getClassName()

获取某个对象的类名。

### isMemberOf(metaclass)

检查对象是否为 metaclass 的成员。
| **参数** | **类型** | **默认值** | **描述** |
| --- | --- | --- | --- |
| metaclass | FXMetaClass |  |  |

### onDefault()

为未处理的消息调用。

在 FXDelegator、FXGLViewer、FXMDIChild 和 FXMDIClient 中重新实现。

### handle(sender, sel, ptr)

处理发送到此类别的消息。
| **参数** | **类型** | **默认值** | **描述** |
| --- | --- | --- | --- |
| sender | FXObject |  | 消息的发送者。 |
| sel | FXSelector |  | 消息的选择器。 |
| ptr | void* |  | 关联的数据。 |
