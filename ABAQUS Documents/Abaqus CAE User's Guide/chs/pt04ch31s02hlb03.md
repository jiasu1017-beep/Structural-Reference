# 31.2.10 修改轮廓积分数据





如果需要，您可以修改为定义裂缝前缘、裂缝尖端和裂缝扩展方向而选择的实体，这些将用于定义轮廓积分。您还可以指定裂缝前缘位于对称平面上，仅对结构的一半进行建模。更多信息，请参见["传统有限元方法"中的"对称"中的"轮廓积分评估"，Abaqus Analysis User's Guide第11.4.2节](../usb/usb-link.md#usb-anl-acontintegral-symmetry)。

**要定义用于轮廓积分分析的裂缝：**

1. 使用以下方法之一显示**Crack**编辑器：
   - 要配置新的轮廓积分，请按照["创建轮廓积分裂缝"，第31.2.9节](pt04ch31s02hlb02.md)中的说明进行操作。
   - 要编辑现有轮廓积分，请从Interaction模块的主菜单栏中选择****Special**![](../graphics/images/arrow.gif)**Crack**![](../graphics/images/arrow.gif)**Edit**![](../graphics/images/arrow.gif)***裂缝名称*****。
2. 切换**On symmetry plane (half-crack model)**以指定裂缝前缘位于仅对结构一半建模的对称平面上。Abaqus将使从虚拟裂缝前缘前进计算的电位能变化加倍，以计算正确的轮廓积分值。
3. 如果需要，单击![](../graphics/ico_selectBlue.png)以修改定义裂缝前缘、裂缝尖端或裂缝线的实体选择。如果Abaqus/CAE选择的裂缝尖端或裂缝线区域与裂缝前缘相同，则无法编辑选择。
4. 如果需要，更改定义裂缝扩展方向的方法。您也可以单击![](../graphics/ico_selectBlue.png)以修改定义裂缝扩展方向的实体选择。
5. 单击**OK**配置轮廓积分并关闭编辑器。Abaqus在区域上显示绿色十字以表示裂缝前缘。您必须使用Step模块中的**History Output Request**编辑器来将轮廓积分数据包含在分析生成的输出数据库中。更多信息，请参见["轮廓积分输出"，第31.2.6节](pt04ch31s02s06.md)和["修改历史输出请求"，第14.12.3节](pt03ch14s12hlb03.md)。
