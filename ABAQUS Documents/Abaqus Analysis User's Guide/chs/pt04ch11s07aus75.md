# 11.7 选择性子循环

**产品：** Abaqus/Explicit

##### **参考**

- ["Explicit dynamic analysis," Section 6.3.3](pt03ch06s03at06.md)
- [*SUB CYCLING](../key/key-link.md#usb-kws-msubcycling)

### 概述

选择性子循环（subcycling）是一种用于显式动力学分析的技术，允许对模型的不同区域使用不同的时间步长。子循环可以显著提高包含多种单元尺寸的模型的计算效率。

### 子循环原理

显式动力学分析的稳定性条件由最小单元尺寸决定：

```
dt_stable = 2 * L_min / c
```

其中L_min是模型中的最小单元尺寸。

对于包含细小单元和粗大单元的混合模型，使用统一的时间步长意味着大部分单元被迫使用远小于其稳定性要求的步长，造成计算效率低下。

子循环通过以下方式解决此问题：
- 不同区域使用不同的时间步长
- 细小单元使用小步长
- 粗大单元使用大步长
- 通过插值保证区域间的位移协调

### 子循环类型

#### 固定子循环

使用固定的步长比。细小单元区域的时间步长与粗大单元区域的时间步长保持固定比例。

#### 自适应子循环

步长比根据当前分析状态动态调整。当高频响应减弱时，可以增大步长比以提高效率。

### 定义子循环

使用**SUB CYCLING**关键字定义子循环：

```
[*SUB CYCLING](../key/key-link.md#usb-kws-msubcycling), OPTION=fixed
*单元集1*, *步长比1*
*单元集2*, *步长比2*
```

### 单元分组

子循环要求将模型中的单元分成多个"组"，每个组使用相同的时间步长。分组原则：

- 具有相似特征长度的单元分在同一组
- 组间通过界面单元或直接约束连接
- 通常2-4组即可满足大多数应用

### 步长比

步长比定义了小步长区域与大步长区域的时间步长比例：

- **步长比 = 2**：小步长区域每2个时间步，大步长区域执行1个时间步
- **步长比 = 4**：小步长区域每4个时间步，大步长区域执行1个时间步

步长比的选择受以下因素影响：
- 单元特征长度差异
- 区域间界面复杂性
- 求解精度要求

### 界面处理

#### 位移协调

子循环区域之间的位移通过以下方式协调：
- 界面节点位移通过插值匹配
- 粗网格区域的位移用于约束细网格区域
- 使用高阶插值保持精度

#### 应力同步

区域间的应力通过以下方式传递：
- 细网格区域的应力用于更新粗网格区域
- 通过界面力平衡保证协调

### 应用场景

#### 螺纹连接

螺栓头部和螺纹区域使用细小单元，杆件主体使用粗大单元。

#### 接触问题

接触区域使用细小单元以准确捕捉接触应力，远离接触区域使用粗大单元。

#### 焊接结构

焊缝区域使用细小单元，母材使用粗大单元。

### 限制与注意事项

1. **界面精度**：界面处理可能引入误差，需验证精度

2. **波传播问题**：涉及高频波传播的问题可能不适合子循环

3. **自适应与固定**：自适应子循环更稳健，但计算开销略高

4. **单元类型**：某些单元类型可能不支持子循环

### 验证与确认

使用子循环时应验证：

- 与统一小步长结果比较
- 能量平衡检查
- 界面应力连续性检查

### 输出

子循环分析的标准输出：
- 节点位移、速度、加速度
- 单元应力、应变
- 伪应变能（检查数值稳定性）
- 动能（检查整体响应）
