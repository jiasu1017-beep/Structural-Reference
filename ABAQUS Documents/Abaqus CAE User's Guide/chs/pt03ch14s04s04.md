# 14.4.4 输出请求管理器





Abaqus/CAE为场输出请求和历史输出请求提供单独的管理器。输出请求管理器是步骤相关管理器，这意味着它们包含有关分析中每个步骤中每个输出请求状态的信息，并允许你控制请求在步骤序列中的传播。更多信息，请参见["什么是步骤相关管理器？"第3.4.2节](pt01ch03s04s02.md)。

**场输出请求管理器**和**历史输出请求管理器**包含你创建的所有输出请求的列表。例如，[图14-2](pt03ch14s04s04.md#sim-odb-manager)显示了**场输出请求管理器**。

**图14-2****场输出请求管理器**。

![](../graphics/sim-odb-manager-nls.png)

选择步骤后，两个管理器中的**创建**按钮允许你在该步骤期间创建新的输出请求。类似地，**编辑**、**复制**、**重命名**和**删除**按钮允许你编辑、复制、重命名和删除选定的输出请求。你也可以使用主菜单栏中的****Output**![](../graphics/images/arrow.gif)**Field Output Requests****和****Output**![](../graphics/images/arrow.gif)**History Output Requests****子菜单来启动创建、编辑、复制、重命名和删除过程。

你可以使用**场输出请求管理器**和**历史输出请求管理器**中的**复制**按钮（或相应的菜单命令或模型树）来复制输出请求。你可以将输出请求从任何步骤复制到任何有效步骤，但有一些限制。更多详细信息，请参见["使用管理器对话框复制步骤相关对象，"第3.4.11节](pt01ch03s04hlb03.md)。

**向左移动**、**向右移动**、**激活**和**停用**按钮允许你控制输出请求在分析过程中的传播。更多信息，请参见["修改步骤相关对象的历史，"第3.4.6节](pt01ch03s04s06.md)。

你可以使用管理器左侧列中的图标来抑制输出请求或恢复先前被抑制的分析输出请求。抑制和恢复过程也可以从主菜单栏中的****Output**![](../graphics/images/arrow.gif)**Field Output Requests****和****Output**![](../graphics/images/arrow.gif)**History Output Requests****子菜单中获得。更多信息，请参见["抑制和恢复对象，"第3.4.3节](pt01ch03s04s03.md)。
