# 32.2 为垫片定义材料





您可以创建两种类型的材料以包含在垫片截面定义中：具有垫片特定行为的材料和通用材料。您创建的材料类型取决于您对垫片行为的要求。
- 如果希望厚度方向、横向剪切和膜行为解耦，请使用特殊垫片行为创建材料。垫片行为材料仅对垫片截面有效。有关定义垫片行为的此方法的详细信息，请参见["直接使用垫片行为模型定义垫片行为"，Abaqus Analysis User's Guide第32.6.6节](../usb/usb-link.md#usb-elm-egasketbehavior)。
- 如果只想考虑厚度方向行为，请创建通用材料。通用材料在垫片截面以及其他类型的截面中也有效。有关使用材料模型定义垫片行为的此方法的详细信息，请参见["使用材料模型定义垫片行为"，Abaqus Analysis User's Guide第32.6.5节](../usb/usb-link.md#usb-elm-egasketmatbehavior)。

您可以通过为****Other**![](../graphics/images/arrow.gif)**Gasket****子菜单中的 一种或多种行为输入数据来创建特定于垫片的材料。为任何其他行为输入的数据将被忽略，但以下情况除外：
- 您可以在垫片行为材料定义中包含**Expansion**行为（位于**Mechanical**菜单中）。
- 您可以在垫片行为材料定义中包含**Creep**行为（位于****Mechanical**![](../graphics/images/arrow.gif)**Plasticity****菜单中）。
- 您可以在垫片行为材料定义中包含**Depvar**和**User Output Variables**行为（位于**General**菜单中）。

您可以通过为除****Other**![](../graphics/images/arrow.gif)**Gasket****子菜单中发现的行为之外的任何对垫片截面有效的行为输入数据来创建通用材料。（如果您为****Other**![](../graphics/images/arrow.gif)**Gasket****子菜单中发现的行为输入数据，您将自动创建垫片行为材料。）有关哪些材料行为对包含在垫片截面定义中的通用材料有效的信息，请参见["垫片单元"，Abaqus Analysis User's Guide第32.6节](../usb/usb-link.md#usbgasket)。
