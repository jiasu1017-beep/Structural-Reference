# 32.4.1 分布耦合单元






**产品：** Abaqus/Standard  

##### **参考文献**

- ["分布耦合单元库，" 第32.4.2节](pt06ch32s04ael29.md)
- [*DISTRIBUTING COUPLING](../key/key-link.md#usb-kws-mdistcoupling)

### 概述

分布耦合单元：
- 可用于将力和力矩从参考节点分布到一组节点；
- 可用于为一组节点规定平均位移和旋转；
- 可用于将质量分布到一组节点；
- 可以通过为每个耦合节点指定的权重因子来控制力和质量的分布；
- 可用于在结构单元和实体单元之间创建柔性耦合；以及
- 可用于二维或三维应力/位移单元。

如果不需要质量分布，首选的定义分布约束的方法在["耦合约束，" 第35.3.2节](pt08ch35s03aus133.md)中描述。

### 典型应用

分布耦合单元约束耦合节点的运动以遵循单元节点的平移和旋转。这种约束以平均值的方式强制执行，并能够控制载荷的传递。这些特性使分布耦合单元在许多应用中很有用：
- 该单元可用于在边界上规定位移和旋转条件，而在边界上需要节点之间的相对运动。这种情况的一个例子是在预期翘曲和/或在端面内变形的结构末端规定扭转（请参见[图32.4.1-1](pt06ch32s04alm40.md#edistcoupling-rotate)）。
**图32.4.1-1** DCOUP3D单元用于在不约束面内运动的情况下对结构表面施加旋转。
![](../graphics/distributingcoupling-exa-nls.png)
- 该单元可用于通过参考节点的运动来提供耦合节点运动的加权平均值。
- 该单元可用于分布载荷，其中载荷分布可以用惯性矩表达式来描述。这种情况的例子包括经典的螺栓图案和焊接图案载荷分布表达式。
- 该单元可用作耦合两个部件（结构-实体）以传递力和力矩。在与MPC和运动耦合约束的比较中，分布耦合单元可以被认为是更"柔性"的连接。

### 选择适当的单元

提供了二维和三维分布耦合单元。DCOUP2D仅在全局*X*–*Y*平面中描述行为。DCOUP2D可用于轴对称分析；但是，其使用需要在选择载荷分布权重因子时小心。例如，对于结构均匀轴向载荷分布到结构，需要按耦合节点半径的比例指定载荷分布权重因子。由于这些节点的半径随变形而变化，因此在几何非线性分析中，这种DCOUP2D用法只能近似正确的载荷分布行为。

### 定义分布耦合

要定义分布耦合，您指定要分配载荷和质量的耦合节点，以及相应的分布权重。至少需要两个耦合节点。

| **输入文件用法：** | ``` [*DISTRIBUTING COUPLING](../key/key-link.md#usb-kws-mdistcoupling), ELSET=*name* *node number or node set, weight_factor_1* *node number or node set, weight_factor_2* ... ``` |
| --- | --- |

#### 示例

此示例（请参见[图32.4.1-1](pt06ch32s04alm40.md#edistcoupling-rotate)）举例说明使用DCOUP3D单元对预期以一般方式变形的结构表面施加旋转。在这种情况下，预期在端面平面内发生翘曲和运动。

```
[*ELEMENT](../key/key-link.md#usb-kws-melement), TYPE=DCOUP3D, ELSET=ROTATEELEMENT
1001, 1
[*DISTRIBUTING COUPLING](../key/key-link.md#usb-kws-mdistcoupling), ELSET=ROTATEELEMENT
COUPLESET, 1.0
…
[*STEP](../key/key-link.md#usb-kws-hstep), NLGEOM
…
[*BOUNDARY](../key/key-link.md#usb-kws-hboundary)
1, 6, 6, 1.0
…
[*END STEP](../key/key-link.md#usb-kws-hendstep)
```

### 定义载荷分布

单元分布载荷，使得耦合节点上力的合等于单元节点上的力和力矩。对于具有多个耦合节点的情况，仅通过平衡不能确定力的分布，使用用户指定的权重因子来定义分布。权重因子是无量纲的，在每个单元内归一化，使得所有权重因子之和为1。因此，归一化权重因子描述了通过特定耦合节点传递的总单元力和力矩的比例。在仅传递力的情况下，通过节点传递的力比例就是归一化权重因子。在力和力矩传递的一般情况下，力的分布遵循经典螺栓图案分析，其中权重因子可以被认为是特定螺栓横截面的面积。请参阅["分布耦合单元，" Abaqus理论指南第3.9.8节](../stm/stm-link.md#stm-elm-distcouplingelem)，了解载荷分布的具体细节。

在[图32.4.1-1](pt06ch32s04alm40.md#edistcoupling-rotate)所示的示例中，选择的权重因子分布是均匀的，值为1.0。对于所示的旋转，更准确的载荷分布应反映靠近槽边缘的节点上的剪切力会减小到零这一事实，这可以通过为靠近槽边缘的节点选择单个权重因子来描述。如果作用在单元上的载荷沿结构轴线，则所示的均匀分布是合适的。对于需要不同权重因子分布的不同载荷模式的情况，可以使用具有不同单元节点和不同权重因子的多个分布耦合单元。

#### 共线耦合节点排列

分布耦合单元在单元节点处传递力矩，即使这些节点具有旋转自由度。因此，当耦合节点排列共线时，单元无法传递单元节点处力矩的所有分量。具体来说，平行于共线耦合节点排列的力矩分量将不会被传递。当出现这种情况时，会发出警告消息，标识单元将不传递力矩的轴。

#### 与非均匀网格一起使用

当分布耦合单元与连接到不同大小单元的耦合节点一起使用时，在选择权重因子时应小心。为节点选择的权重因子通常应与连接到该节点的单元大小成比例。

### 定义质量分布

质量分布与力分布类似；指定的单元质量按权重因子比例分布到耦合节点。

| **输入文件用法：** | ``` [*DISTRIBUTING COUPLING](../key/key-link.md#usb-kws-mdistcoupling), ELSET=*name*, MASS=*total_element_mass* *node number or node set, weight_factor_1* *node number or node set, weight_factor_2* ... ``` |
| --- | --- |

### 输出

单元节点力（单元放置在单元和耦合节点上的力）可通过单元变量NFORC获得。在动态过程中，单元动能可通过整个单元变量ELKE获得。
