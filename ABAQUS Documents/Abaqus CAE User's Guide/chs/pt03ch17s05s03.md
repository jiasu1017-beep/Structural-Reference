# 17.5.3 单元类型分配







单元类型可以分配给以下对象：
- 从基于几何的零件或零件实例中选择区域。零件实例必须来自您在 Part 模块中创建的零件或导入的零件。
- 引用从基于几何的零件或零件实例中选择区域的集合。该集合也可以引用皮肤增强。
- 孤立单元或单元集。

所有基于几何的零件或零件实例的区域以及所有孤立单元都有默认的单元类型分配。这些分配取决于区域或单元所属的零件类型。您可以使用 **Element Type** 对话框查看和更改分配的 Abaqus 单元类型，该对话框可通过选择 ****Mesh**![](../graphics/images/arrow.gif)**Element Type**** 显示。例如，图 17-20 显示了二维结构区域的 **Element Type** 对话框。

**图 17-20** Abaqus/Standard 模型中二维结构区域的 **Element Type** 对话框。

![](../graphics/mgn-assign1-nls.png)

在对话框顶部，您可以输入单元库、几何阶次和族的首选项。然后，您可以通过点击对话框下半部分的选项卡并从出现的选项中进行选择来选择特定的单元类型。有关单元控制选项的更多信息，请参见《Abaqus Analysis User's Guide》第 27.1.4 节"Section controls"。

根据所选区域的维度，对话框可能包含一到三个选项卡：
- **Line** 选项卡允许您选择适用的单元类型并将其分配给区域中的一维网格单元。
- **Quad** 和 **Tri** 选项卡允许您选择适用的单元类型并将其分配给区域中的二维网格单元。
- **Hex**、**Wedge** 和 **Tet** 选项卡允许您为区域中的三维网格单元分配三维单元类型。

例如，在图 17-20 中选择了 Abaqus/Standard 单元库中线性壳单元的选项。点击 **Quad** 选项卡后，选择了减缩积分和有限膜应变。满足所有这些条件的四边形壳单元的名称和简要说明显示在选项卡页面的底部。

此对话框中的 **Tri** 选项卡如图 17-21 所示。满足对话框中指定条件的三角形壳单元的名称和简要说明显示在图 17-21 中 **Tri** 选项卡页面的底部。如果此示例中所选区域恰好包含三角形和四边形网格单元的组合：
- 四边形网格单元被分配了 S4R 单元类型。
- 三角形网格单元被分配了 S3 单元类型。

**图 17-21** **Tri** 选项卡。

![](../graphics/mgn-assign2-nls.png)

如果区域仅包含四边形单元，则所有单元都被分配了 S4R 单元类型。

有关将单元类型分配给网格区域的详细分步说明，请参见"将 Abaqus 单元与网格区域关联"第 17.18.10 节。有关可用单元类型的列表，请参阅《Abaqus Analysis User's Guide》中的"Section EI.1, Abaqus/Standard Element Index"、"Section EI.2, Abaqus/Explicit Element Index"和"Section EI.3, Abaqus/CFD Element Index"。您可以通过 **Element Type** 对话框选择大多数这些单元。"哪些类型的单元必须在网格模块外部生成？"第 17.5.2 节描述了无法选择的单元。
