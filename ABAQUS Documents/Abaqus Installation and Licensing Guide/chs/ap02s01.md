# B.2 Abaqus 父目录





与 Abaqus 产品安装相关的所有文件都存储在 Abaqus 父目录中。建议将 Abaqus 父目录放在 SIMULIA 父目录内，建议路径为 `SIMULIA/Abaqus`（请参见 [Figure B--1](ap02s01.md#sgb-directories-simulia-nls)）。在本指南中，Abaqus 父目录在路径列表中被称为 `*abaqus_dir*`。根据所选的安装选项，将在 `*abaqus_dir*` 下创建以下目录：

| `Commands` | 包含每个版本的驱动程序命令的链接（Linux）或批处理文件（Windows）的目录。 |
| --- | --- |

| *release* | 每个已安装版本的特定版本父目录。例如，Abaqus 6.14-1 的默认目录名为 `6.14-1`。 |
| --- | --- |
|  | `/api/cse` | SIMULIA Co-Simulation Engine Application Programming Interface 以及文档和示例。 |
|  | `/CAEresources` | 配置文件。 |
|  | `/code` | Abaqus 可执行文件和命令过程、第三方可执行文件和库。还包含使用 **abaqus make** 实用程序构建后处理程序的头文件。 |
|  | `/code/bin` | Abaqus 可执行文件和命令过程。 |
|  | `/Help` | Abaqus/CAE 中上下文相关帮助使用的文件。 |
|  | `/installation_info` | 产品安装和验证日志文件以及 Windows 卸载程序日志文件（如果使用）。 |
|  | `/samples` | 与已安装产品相关的输入文件和目录，包括安装验证问题、时间测试问题、Introduction to Abaqus 研讨会文件、Getting Started 教程指南文件、应用程序简介文件、与 **abaqus findkeyword** 实用程序一起使用的文件，以及 Example Problems、Benchmarks 和 Verification Guides 中的文件。 |
|  | `/SMA` | 杂项信息和特定于站点的文件。 |
|  | `/SMA/site` | 特定于站点的文件：Abaqus 环境文件（`abaqus_v6.env`）、示例环境文件（`abaqusinc.env`）、包含文件（*file_name*`.inc`）、包含验证过程中使用的信息的数据文件（`chksum.dat`）以及平台计算机表。 |
|  | `/tools` | Python 解释器。 |
|  | `/code/python/lib/` `abaqus_plugins` | Abaqus/CAE 插件的示例 Python 模块。 |
