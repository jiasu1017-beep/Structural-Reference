### 概述

如["预定义条件：概述，" 第34.1.1节](pt07ch34s01abo31.md)中所述，电磁载荷可施加于["压电分析，" 第6.7.2节](pt03ch06s07at21.md)；["耦合热电分析，" 第6.7.3节](pt03ch06s07at22.md)；["完全耦合热电结构分析，" 第6.7.4节](pt03ch06s07at23.md)；["涡流分析，" 第6.7.5节](pt03ch06s07at24.md)；和["静磁分析，" 第6.7.6节](pt03ch06s07at25.md)。

可用电磁载荷的类型取决于所执行的分析，如下所述。请参阅["施加载荷：概述，" 第34.4.1节](pt07ch34s04aus120.md)，了解适用于所有载荷类型的一般信息。

### 定义随时间变化的电磁载荷

集中或分布式电磁载荷的指定幅值可以在步骤期间根据幅值定义随时间变化，如["预定义条件：概述，" 第34.1.1节](pt07ch34s01abo31.md)中所述。如果不同的载荷需要不同的变化，每个载荷可以引用其自己的幅值定义。

在时谐涡流分析中，所有载荷假定为时谐的。

### 修改电磁载荷

可以如["施加载荷：概述，" 第34.4.1节](pt07ch34s04aus120.md)中所述添加、修改或删除集中或分布式电磁载荷。

### 为压电分析施加电磁载荷

在压电分析中，可以在节点处施加集中电荷，可以在单元面和表面上定义分布式电表面电荷，还可以在单元上定义分布式电体电荷。

#### 指定集中电荷

要指定集中电荷，请指定节点或节点集以及电荷幅值。

| **输入文件用法：** | ``` [*CECHARGE](../key/key-link.md#usb-kws-hcecharge) *node number or node set name, , charge magnitude* ``` |
| --- | --- |

| **Abaqus/CAE用法：** | Load模块：**Create Load**：选择**Electrical/Magnetic**作为**Category**，选择**Concentrated charge**作为**Types for Selected Step**；**Magnitude**：*charge magnitude* |
| --- | --- |

#### 指定基于单元的分布式电荷

您可以指定分布式表面电荷（在单元面上）或分布式体电荷（单位体积的电荷）。对于基于单元的表面电荷，您必须在电荷标签中标识指定电荷的单元面。可用的分布式电荷类型取决于单元类型。[第六部分，"单元"](pt06.md)列出了特定单元可用的分布式电荷。

| **输入文件用法：** | ``` [*DECHARGE](../key/key-link.md#usb-kws-hdecharge) *element number or element set name, charge label, charge magnitude* ``` |
| --- | --- |
|  | 其中*charge label*为ES*n*或EBF |

| **Abaqus/CAE用法：** | 使用以下输入定义单元面上的分布式表面电荷： |
| --- | --- |
|  | Load模块：**Create Load**：选择**Electrical/Magnetic**作为**Category**，选择**Surface charge**作为**Types for Selected Step**；**Distribution**：选择解析场，**Magnitude**：*charge magnitude* 使用以下输入定义体电荷： Load模块：**Create Load**：选择**Electrical/Magnetic**作为**Category**，选择**Body charge**作为**Types for Selected Step** |

#### 指定基于表面的分布式电荷

当在表面上指定分布式电荷时，基于单元的表面（请参见["基于单元的表面定义，" 第2.3.2节](pt01ch02s03aus17.md)）包含单元和面信息。您必须指定表面名称、电荷标签和电荷幅值。

| **输入文件用法：** | ``` [*DSECHARGE](../key/key-link.md#usb-kws-hdsecharge) *surface name*, ES, *charge magnitude* ``` |
| --- | --- |

| **Abaqus/CAE用法：** | Load模块：**Create Load**：选择**Electrical/Magnetic**作为**Category**，选择**Surface charge**作为**Types for Selected Step**；**Distribution**：**Uniform**，**Magnitude**：*charge magnitude* |
| --- | --- |

#### 在直接解稳态动力学分析中指定电荷

在直接解稳态动力学过程中，电荷以其实部和虚部给出。

| **输入文件用法：** | 使用以下选项在直接积分稳态动力学分析中定义电荷： |
| --- | --- |
|  | ``` [*CECHARGE](../key/key-link.md#usb-kws-hcecharge), REAL or IMAGINARY* (real or imaginary component)* [*DECHARGE](../key/key-link.md#usb-kws-hdecharge), REAL or IMAGINARY [*DSECHARGE](../key/key-link.md#usb-kws-hdsecharge), REAL or IMAGINARY ``` |

| **Abaqus/CAE用法：** | Load模块：**Create Load**：选择**Electrical/Magnetic**作为**Category**，选择**Concentrated charge**、**Surface charge**或**Body charge**作为**Types for Selected Step**；**Magnitude**：*real component* + *imaginary component* |
| --- | --- |

#### 基于模态和子空间步骤中的载荷

由于"无质量"模态效应，电荷载荷应仅与残差模态结合使用。由于电势自由度没有相关质量，这些自由度在特征值提取过程中基本上被消除了（类似于Guyan约简或质量凝聚）。残差模态表示与电荷载荷对应的静态响应，这将充分表示特征空间中的势自由度。

### 为耦合热电分析和完全耦合热电结构分析施加电磁载荷

在耦合热电分析和完全耦合热电结构分析中，可以在节点处施加集中电流，可以定义单元面上和表面上的分布式电流密度，还可以在单元上定义分布式体电流。

#### 指定集中电流密度

要定义集中电流，请指定节点或节点集以及电流幅值。

| **输入文件用法：** | ``` [*CECURRENT](../key/key-link.md#usb-kws-hcecurrent) *node number or node set name, , current magnitude* ``` |
| --- | --- |

| **Abaqus/CAE用法：** | Load模块：**Create Load**：选择**Electrical/Magnetic**作为**Category**，选择**Concentrated current**作为**Types for Selected Step**；**Magnitude**：*current magnitude* |
| --- | --- |

#### 指定基于单元的分布式电流密度

您可以指定分布式表面电流密度（在单元面上）或分布式体电流密度（单位体积的电流）。对于基于单元的表面电流密度，您必须在电流标签中标识指定电流的单元面。可用的分布式电流类型取决于单元类型。[第六部分，"单元"](pt06.md)列出了特定单元可用的分布式电流密度。

| **输入文件用法：** | ``` [*DECURRENT](../key/key-link.md#usb-kws-hdecurrent) *element number or element set name, current density label, current density magnitude* ``` |
| --- | --- |
|  | 其中*current density label*为CS*n*、CS1、CS2或CBF |

| **Abaqus/CAE用法：** | 使用以下输入定义单元面上的分布式表面电流密度： |
| --- | --- |
|  | Load模块：**Create Load**：选择**Electrical/Magnetic**作为**Category**，选择**Surface current**作为**Types for Selected Step**；**Distribution**：选择解析场，**Magnitude**：*current density magnitude* 使用以下输入定义体电流密度： Load模块：**Create Load**：选择**Electrical/Magnetic**作为**Category**，选择**Body current**作为**Types for Selected Step** |

#### 指定基于表面的分布式电流密度

当在表面上指定分布式电流密度时，基于单元的表面（请参见["基于单元的表面定义，" 第2.3.2节](pt01ch02s03aus17.md)）包含单元和面信息。您必须指定表面名称、电流密度标签和电流密度幅值。

| **输入文件用法：** | ``` [*DSECURRENT](../key/key-link.md#usb-kws-hdsecurrent) *surface name*, CS, *current density magnitude* ``` |
| --- | --- |

| **Abaqus/CAE用法：** | Load模块：**Create Load**：选择**Electrical/Magnetic**作为**Category**，选择**Surface current**作为**Types for Selected Step**：**Distribution**：**Uniform**，**Magnitude**：*current density magnitude* |
| --- | --- |

### 为涡流分析和/或静磁分析施加电磁载荷

在涡流分析中，可以在表面上定义分布式表面电流密度向量，还可以在单元上定义分布式体积电流密度向量。

#### 指定基于单元的分布式电流密度向量

当定义分布式体积电流密度向量时，您必须指定单元或单元集、电流密度向量标签、电流密度向量幅值、电流密度的向量分量，以及可选的定义向量分量所参考的局部坐标系的方向名称。默认情况下，电流密度的向量分量是相对于全局方向定义的。

Abaqus会归一化指定的电流密度向量方向分量，因此不会影响载荷的幅值。

| **输入文件用法：** | ``` [*DECURRENT](../key/key-link.md#usb-kws-hdecurrent) *element number or element set name*, CJ, *current density vector magnitude, current density vector direction components, orientation name* ``` |
| --- | --- |

| **Abaqus/CAE用法：** | Load模块：**Create Load**：选择**Electrical/Magnetic**作为**Category**，选择**Body current density**作为**Types for Selected Step**；**Distribution**：**Uniform** |
| --- | --- |

#### 指定基于表面的分布式电流密度向量

当在表面上指定分布式电流密度向量时，基于单元的表面（请参见["基于单元的表面定义，" 第2.3.2节](pt01ch02s03aus17.md)）包含单元和面信息。您必须指定表面名称、电流密度向量标签、电流密度向量幅值、电流密度的向量分量，以及可选的定义表面电流密度所参考的局部坐标系的方向名称。默认情况下，电流密度的向量分量是相对于全局方向定义的。

Abaqus会归一化指定的电流密度向量方向分量，因此不会影响载荷的幅值。

| **输入文件用法：** | ``` [*DSECURRENT](../key/key-link.md#usb-kws-hdsecurrent) *surface name*, CK, *current density vector magnitude, current density vector direction components, orientation name* ``` |
| --- | --- |

| **Abaqus/CAE用法：** | Load模块：**Create Load**：选择**Electrical/Magnetic**作为**Category**，选择**Surface current density**作为**Types for Selected Step**；**Distribution**：**Uniform** |
| --- | --- |

#### 在用户子程序中定义非均匀电流密度向量

非均匀体积电流密度向量可以在用户子程序[`UDECURRENT`](../sub/sub-link.md#sub-xsl-udecurrent)中定义，非均匀表面电流密度向量可以在用户子程序[`UDSECURRENT`](../sub/sub-link.md#sub-xsl-udsecurrent)中定义。如果给定了幅值和方向分量，这些值将被传递到用户子程序中。

| **输入文件用法：** | 使用以下选项定义非均匀基于单元的电流密度向量： |
| --- | --- |
|  | ``` [*DECURRENT](../key/key-link.md#usb-kws-hdecurrent) *element number or element set name*, CJNU, *current density vector magnitude, current density vector direction components, orientation name* ``` 使用以下选项定义非均匀基于表面的电流密度向量： ``` [*DSECURRENT](../key/key-link.md#usb-kws-hdsecurrent) *surface name*, CKNU, *current density vector magnitude, current density vector direction components, orientation name* ``` |

| **Abaqus/CAE用法：** | 使用以下选项定义非均匀体积电流密度： |
| --- | --- |
|  | Load模块：**Create Load**：选择**Electrical/Magnetic**作为**Category**，选择**Body current density**作为**Types for Selected Step**；**Distribution**：**User-defined** 使用以下选项定义非均匀表面电流密度： Load模块：**Create Load**：选择**Electrical/Magnetic**作为**Category**，选择**Surface current density**作为**Types for Selected Step**；**Distribution**：**User-defined** |

#### 在时谐涡流分析中指定电流密度向量的实部和虚部

在时谐涡流分析中，电流密度向量以其实部（同相）和虚部（异相）分量给出。

| **输入文件用法：** | 使用以下选项定义电流密度向量： |
| --- | --- |
|  | ``` [*DECURRENT](../key/key-link.md#usb-kws-hdecurrent), REAL or IMAGINARY [*DSECURRENT](../key/key-link.md#usb-kws-hdsecurrent), REAL or IMAGINARY ``` |

| **Abaqus/CAE用法：** | Load模块：**Create Load**：选择**Electrical/Magnetic**作为**Category**，选择**Body current density**或**Surface current density**作为**Types for Selected Step**；*real components + imaginary components* |
| --- | --- |
