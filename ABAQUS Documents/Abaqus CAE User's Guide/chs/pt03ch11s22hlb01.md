# 11.22.1 添加拉伸壳特征





从主菜单栏选择**形状**![](../graphics/images/arrow.gif)**壳**![](../graphics/images/arrow.gif)**拉伸**，向当前视口中的零件添加拉伸壳特征。您只能向三维零件添加拉伸壳特征。

您通过在选定面上绘制并沿垂直于面的方向将轮廓延伸指定距离来添加拉伸壳特征。

您还可以通过选择要拉伸到的单个面来定义拉伸距离。

**添加拉伸壳特征：**

1. 从主菜单栏，选择**形状**![](../graphics/images/arrow.gif)**壳**![](../graphics/images/arrow.gif)**拉伸**。
2. 如果需要，指定要用于为拉伸壳特征的草图选择原点的方法。
3. 选择壳将被拉伸的平面。
4. 选择一条边以及边在草图编辑器网格上的方向。
5. 使用草图编辑器绘制要拉伸的线的轮廓。
6. 在提示区域，单击**完成**退出草图编辑器并打开**编辑拉伸**对话框。
7. 在**编辑拉伸**对话框中单击以反转拉伸方向（如果需要）。
8. 选择以下终止条件之一：**盲孔**或**至面**。
9. 如果需要，执行以下操作之一：切换**包含扭曲**或**包含拔模**。
10. 切换**保持内部边界**。
11. 单击**确定**拉伸轮廓。
12. 如果选择了**至面**，Abaqus/CAE会提示您选择要拉伸轮廓到的面。
