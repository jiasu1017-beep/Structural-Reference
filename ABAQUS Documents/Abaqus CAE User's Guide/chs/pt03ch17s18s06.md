# 17.18.6 指定扫掠路径





如果您将扫掠网格划分技术应用于特定区域，**Mesh Controls**对话框中会显示该区域的**Redefine sweep path**按钮。 如果该区域有多个有效扫掠路径，您可以单击**Redefine sweep path**来选择您想要的路径。 （有关扫掠路径的更多信息，请参阅["扫掠网格划分，" 第17.9节](pt03ch17s09.md)。）

如果您正在为垫片、连续壳、圆柱形或粘聚区域分配网格控制，此选项特别重要。 除非您分配另一个方向（有关更多信息，请参阅["应用网格堆栈方向，" 第17.18.8节](pt03ch17s18hlb01.md)），否则这些区域类型的方向属性取决于扫掠路径。 当您对垫片区域进行网格划分时，每个垫片单元的轴将与扫掠路径方向重合。 （有关更多信息，请参阅[第32章，"垫片"](pt04ch32.md)，以及["垫片单元：概述，" Abaqus Analysis User's Guide第32.6.1节](../usb/usb-link.md#usb-elm-egasketoverview)。） 当您对连续壳或粘聚区域进行网格划分时，扫掠路径与堆栈方向对齐。 （有关更多信息，请参阅["使用连续壳单元进行网格划分的零件，" 第25.2节](pt04ch25s02.md)，以及["使用几何和网格工具创建具有粘聚单元的模型，" 第21.3节](pt04ch21s03.md)。） 当您使用圆柱单元对圆柱区域进行网格划分时，所选扫掠/旋转区域的扫掠路径沿圆柱几何形状的周向方向。 （有关更多信息，请参阅["圆柱实体的扫掠网格划分，" 第17.9.4节](pt03ch17s09s04.md)。）

**指定扫掠路径：**

1. 从主菜单栏，选择 ****Mesh**![](../graphics/images/arrow.gif)**Controls****。 Abaqus/CAE在提示区域显示提示以指导您完成该过程。 **提示：**您也可以单击Mesh模块工具箱中的 ![](../graphics/ico_meshControls.png) 工具。 （有关更多信息，请参阅["使用Mesh模块工具箱，" 第17.15节](pt03ch17s15.md)。）
2. 如果您的零件或装配包含多个区域，请选择感兴趣的可扫掠网格区域，然后按鼠标按钮2。 **Mesh Controls**对话框出现。
3. 如果尚未选择，请在**Mesh Controls**对话框中选择**Sweep**作为网格划分技术。 如果您选择的区域存在多个有效扫掠路径，**Redefine Sweep Path**按钮会出现在对话框底部附近。
4. 单击**Redefine Sweep Path**。 如果您选择了多个区域，此过程会依次考虑每个所选区域。 正在考虑的区域以品红色高亮显示，默认扫掠路径以红色高亮显示，箭头指示扫掠方向。
5. 在提示区域通过选择适当的选项指定您想要的扫掠路径和方向： - 单击**Accept Highlighted**接受视口中以红色高亮显示的扫掠路径。 - 单击**Flip**更改当前所选扫掠路径的方向。 然后单击**Yes**表示新的扫掠路径方向正确。 - 单击**Select New**（如果适用）选择不同的边作为扫掠路径。 然后执行以下步骤： 1. 在扫掠路径中选择一条边。 您可以通过在您想要与扫掠路径末端重合的边的末端附近单击鼠标按钮来指示所需的扫掠方向。 新的扫掠路径以红色高亮显示，箭头指示扫掠方向。 2. 如果路径和方向正确，请在提示区域单击**Yes**；如果您想更改扫掠路径的方向，请单击**Flip**。 然后单击**Yes**表示新的扫掠路径方向正确。 如果选择了具有多个有效扫掠路径的多个区域，过程会从下一个区域重新开始。 如果只选择了一个区域，过程完成，**Mesh Controls**对话框会重新出现。
