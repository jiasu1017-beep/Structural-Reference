# 6.2 公式与积分

所有Abaqus中的梁单元都是"梁-柱"单元——这意味着它们允许轴向、弯曲和扭转变形。Timoshenko梁单元还考虑了横向剪切变形的影响。

## 6.2.1 剪切变形

线性单元（B21和B31）以及二次单元（B22和B32）是可剪切变形的Timoshenko梁；因此，它们适用于模拟粗壮构件（其中剪切变形很重要）和细长梁（其中剪切变形不重要）。这些单元的横截面行为与厚壳单元的横截面行为相同，如图6-7(b)所示，并在5.2节"壳公式——厚或薄"中讨论。

**Figure 6–7** 细长梁和粗壮梁中横向梁截面的行为：(a) 细长梁；(b) 粗壮梁。

![细长梁和粗壮梁中横向梁截面的行为](../graphics/gss-transhellsect-nls.png)

Abaqus假设这些梁单元的横向剪切刚度是线弹性的且为常数。此外，这些梁的公式使得它们的横截面积可以随轴向变形而变化，这种效应仅在几何非线性模拟中考虑（见第8章，"非线性"，其中梁截面属性选项上的POISSON参数具有非零值）。只要横截面尺寸小于结构典型轴向尺寸的1/10，这些单元就可以提供有用的结果，这通常被认为是梁理论适用性的极限；如果梁横截面在弯曲变形下不保持平面，梁理论就不足以模拟变形。

Abaqus/Standard中可用的三次单元——所谓的Euler-Bernoulli梁单元（B23和B33）——不模拟剪切柔度。这些单元的横截面保持与梁轴垂直（见图6-7(a)）。因此，三次梁单元最适合模拟具有相对细长构件的结构。由于三次单元沿其长度模拟位移的三次变化，因此静态分析时通常可以用单个三次单元建模结构构件，动态分析时用少量单元即可。这些单元假设剪切变形可以忽略不计。通常，如果横截面尺寸小于结构典型轴向尺寸的1/15，则该假设有效。

## 6.2.2 扭转响应——翘曲

结构构件经常受到扭矩作用，扭矩几乎发生在任何三维框架结构中。引起一个构件弯曲的荷载可能在另一个构件中引起扭转，如图6-8所示。

**Figure 6–8** 框架结构中的扭转。

![框架结构中的扭转](../graphics/gss-torsion-nls.png)

梁对扭矩的响应取决于其横截面的形状。通常，梁中的扭矩会在横截面中产生翘曲或非均匀的面外位移。Abaqus仅在三维单元中考虑扭矩和翘曲的影响。翘曲计算假设翘曲位移很小。以下横截面在扭矩下表现出不同的行为：实心横截面；闭合薄壁横截面；和开口薄壁横截面。

**实心横截面**

实心非圆形横截面在扭矩下不会保持平面；相反，截面会翘曲。Abaqus使用St. Venant翘曲理论来计算横截面中每个截面点处由翘曲引起的单个剪切分量。这种实心截面中的翘曲被认为是无约束的，会产生可忽略的轴向应力。（翘曲约束仅在约束端附近的局部区域影响解。）具有实心截面的梁的扭转刚度取决于材料的剪切模量*G*和梁截面的扭转常数*J*。扭转常数取决于梁横截面的形状和翘曲特性。在横截面中产生大量非弹性变形的扭矩无法用这种方法准确建模。

**闭合薄壁横截面**

具有闭合薄壁非圆形横截面（BOX或HEX）的梁具有显著的扭转刚度，因此其行为方式与实心截面相似。Abaqus假设这些截面中的翘曲也是无约束的。横截面的薄壁特性使Abaqus能够考虑沿壁厚不变的剪切应变。薄壁假设通常在壁厚为典型梁横截面尺寸的1/10时有效。薄壁横截面的典型横截面尺寸示例包括：

- 管道截面的直径。
- 箱形截面一条边的长度。
- 任意截面典型的边长。

**开口薄壁横截面**

当翘曲无约束时，开口薄壁横截面在扭矩上非常灵活，这类结构扭转刚度的主要来源是轴向翘曲应变的约束。约束开口薄壁梁的翘曲会引入轴向应力，这些应力可能影响梁对其他荷载类型的响应。Abaqus/Standard具有剪切可变形的梁单元B31OS和B32OS，它们包含了开口薄壁截面中的翘曲效应。在模拟承受显著扭矩荷载的开口薄壁横截面结构时——如槽形截面（定义为ARBITRARY截面）或I形截面——必须使用这些单元。

翘曲引起的轴向变形沿梁横截面的变化由截面的翘曲函数定义。该函数的大小在开口截面梁单元中被视为一个额外的自由度——自由度7。约束这个自由度可以防止在施加约束的节点处发生翘曲。

为了让翘曲幅度在每个分支中都可以不同，框架结构中开口截面梁之间的连接通常应该用每个分支的独立节点来建模（见图6-9）。

**Figure 6–9** 连接开口截面梁。

![连接开口截面梁](../graphics/gss-opensectbeam-nls.png)

但是，如果连接设计为防止翘曲，所有分支应共享一个公共节点，并应使用*BOUNDARY选项约束翘曲自由度。
