# 29.5.3 修改主模型中的紧固件属性





您可以使用**Properties**标签页指定属性生成脚本，以修改从模板模型复制到主模型的属性。脚本仅在您单击**Edit Fasteners**对话框中的**OK**时由Abaqus/CAE执行，而不是在Abaqus/CAE创建输入文件时执行。完整的相关信息请参见["关于组装紧固件"中的"组装紧固件的属性生成脚本"，第29.1.3节](pt04ch29s01s03.md#usi-eng-fastener-overview-assm-props)。

您的属性生成脚本必须在启动Abaqus/CAE之前注册；请参见["关于组装紧固件"中的"注册属性生成脚本"，第29.1.3节](pt04ch29s01s03.md#usi-eng-fastener-overview-assm-props-reg)。任何已注册的脚本都可以从**Properties**标签页上选择。

**要修改模板模型中的属性：**

1. 从**Edit Fasteners**对话框中，显示**Properties**标签页。
2. 默认情况下，从模板模型复制到主模型的属性使用前缀*TM-1*命名（对于您创建的第一个组装紧固件）。例如，模板模型中名为*BoltSection*的连接器截面在主模型中将命名为*TM-1_BoltSection*。后续的组装紧固件将使用前缀*TM-2*、*TM-3*等。如有需要，您可以在**Property prefix string**字段中更改名称前缀。属性前缀字符串必须对每个组装紧固件唯一。
3. 切换**Specify a property generation script**，然后从列表中选择脚本文件名。
