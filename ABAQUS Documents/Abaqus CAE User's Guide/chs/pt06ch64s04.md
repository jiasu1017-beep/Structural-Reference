# 64.4 网格编辑工具概述





本节是网格编辑工具集中可用工具的概述。您可以通过选择****网格**![](../graphics/images/arrow.gif)**编辑****从主菜单栏访问这些工具。Abaqus/CAE显示**编辑网格**对话框。您也可以使用位于"网格"模块工具箱底部的![](../graphics/ico_meshEdit.png)工具显示**编辑网格**对话框。

- 拖动节点。有关更多信息，请参见["拖动节点，" 第64.5.3节](pt06ch64s05s03.md)。
- 投影节点。有关更多信息，请参见["投影节点，" 第64.5.4节](pt06ch64s05s04.md)。
- 删除节点。有关更多信息，请参见["删除节点，" 第64.5.5节](pt06ch64s05s05.md)。
- 合并节点。有关更多信息，请参见["合并节点，" 第64.5.6节](pt06ch64s05s06.md)。
- 调整副中节点。有关更多信息，请参见["调整副中节点的位置，" 第64.5.7节](pt06ch64s05s07.md)。
- 平滑外部节点。有关更多信息，请参见["平滑外部节点，" 第64.5.8节](pt06ch64s05s08.md)。
- 对节点重新编号。有关更多信息，请参见["对节点重新编号，" 第64.5.9节](pt06ch64s05s09.md)。
