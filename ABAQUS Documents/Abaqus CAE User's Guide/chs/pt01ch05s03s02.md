# 5.3.3 3D 罗盘的预定义视图





你可以使用 3D 罗盘快速将视点设置为六个预定义视图之一。你也可以通过罗盘访问**指定视图**对话框。

**预定义视图**

预定义视图对应于 3D 罗盘的三个平面。要应用预定义视图，请点击 3D 罗盘上任意轴的标签：

![](../graphics/usi-compass-predefined-nls.png)

视图会调整，使所选轴（上例中的 *Z* 轴）垂直于视口平面。再次点击相同的轴标签会将视图方向翻转到视口平面的相反侧。换句话说，反复点击相同的轴标签会使视图在显示模型的前面和背面之间振荡。

与 3D 罗盘关联的六个预定义视图与**视图**工具栏中的预定义视图相同。

**数值指定视图**

双击 3D 罗盘上的任意位置会打开**指定视图**对话框。你可以使用此方法数值指定视点或相机位置。更多信息，请参阅["数值指定视图，" 第 5.2.9 节](pt01ch05s02s09.md)。
