# 16.9.11 定义旋转体力





您可以创建旋转体力载荷以定义由模型旋转产生的载荷。您可以指定角速度或旋转加速度；Abaqus使用角速度的平方或直接使用加速度来定义载荷。在任何一种情况下，载荷定义都必须包括旋转轴，定义如下：
- 如果您在三维空间工作，您可以通过输入两个点的全局坐标来定义轴的位置和方向。
- 如果您在二维空间工作，您可以通过输入平面中一个点的坐标来指定轴的位置。轴的方向始终垂直于平面。
- 如果您在轴对称空间工作，轴始终位于正全局*Z*轴的位置和方向。

**注意：**您只能在二维或三维空间中定义旋转加速度力。

**创建或编辑旋转体力：**

1. 使用以下方法之一显示旋转体力载荷编辑器：- 若要创建新的旋转体力载荷，请遵循"创建载荷"，第16.8.1节中概述的过程（**类别**：`Mechanical`；**所选步骤的类型**：`Rotational body force`）。如果您在二维或三维空间工作，请在提示区域中输入关于旋转轴位置和（如果适用）方向所需的信息。- 若要使用菜单或管理器编辑现有的旋转体力载荷，请参见"编辑依赖于步骤的对象"，第3.4.13节。若要编辑应用载荷的区域，请参见"编辑应用规定条件的区域"，第16.8.4节。如果您在创建载荷的步骤中编辑载荷，则**编辑**（![](../graphics/ico_edit.png)）按钮会出现在您在载荷编辑器中指定的每个点旁边。如果您想更改确定旋转轴位置和（如果适用）方向的坐标，请单击![](../graphics/ico_edit.png)。（此选项仅适用于您在二维或三维空间工作的情况。）
2. 选择您选择的力类型：- 如果要定义离心力，请切换**离心**。- 如果要定义旋转加速度力，请切换**旋转加速度**。
3. 单击**分布**字段右侧的箭头，并从出现的列表中选择您的选项：- 选择**均匀**以定义在物体上均匀的载荷。- 选择分析场以定义空间变化载荷。只有对该载荷类型有效的分析场才会在选择列表中显示。或者，您可以单击![](../graphics/ico_analyticalField.png)创建新的分析场。（更多信息，请参见第58章"分析场工具集"。）
4. 在出现的文本字段中，输入适当的值：- 如果您要定义离心力，请输入角速度（弧度/时间）。- 如果您要定义旋转加速度力，请输入旋转加速度（弧度/时间<sup>2</sup>）。
5. 如果需要，单击**幅值**字段右侧的箭头，并从出现的列表中选择您选择的幅值。或者，您可以单击![](../graphics/ico_amplitudeSmall.png)创建新的幅值。（更多信息，请参见第57章"幅值工具集"。）对于离心加载，幅值应用于计算出的载荷，而不是角速度。
6. 单击**确定**保存您的数据并退出编辑器。
