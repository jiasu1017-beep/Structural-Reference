# 48 沿路径查看结果





路径是由您通过在模型中指定一系列点或边来定义的连接线系列。您可以沿路径以 X–Y 绘图的形式查看结果。本章说明如何沿路径查看结果。涵盖以下主题：
- ["理解沿路径的结果，" 第 48.1 节"](pt05ch48s01.md)
- ["在模型中创建路径，" 第 48.2 节"](pt05ch48s02.md)
- ["沿路径获取 X–Y 数据，" 第 48.3 节"](pt05ch48s03.md)





# 48.1.1 路径规范





路径是您通过在模型中指定一系列点或线段来定义的线。点可以是节点或坐标位置；线段可以是单元边或连接单元边端点的线。路径可以穿过多个部件实例。Abaqus/CAE 提供四种不同类型的路径：

**节点列表**

构成路径的点仅由节点位置组成。您可以使用节点标签和节点标签范围来指定点。节点标签是引用给定节点的持久方式；也就是说，节点标签不会随模型变形而改变。正因为如此，组成节点列表路径的节点标签同样适用于未变形或变形的模型形状。但是，节点标签是部件实例特定的。换句话说，多个部件实例可以具有相同的节点标签；因此，当您使用节点标签时，必须指定您引用的是哪个部件实例。有关更多信息，请参阅 ["创建或编辑节点列表路径，" 第 48.2.1 节"](pt05ch48s02hlb01.md)。

**点列表**

构成路径的点由模型内的坐标位置组成。这些位置可能与节点位置重合或不一致。点列表坐标在空间中是固定的，与模型无关。例如，与未变形形状上节点位置重合的坐标可能与变形形状上的任何位置都不重合。同样，点列表坐标与特定部件实例无关。有关更多信息，请参阅 ["创建或编辑点列表路径，" 第 48.2.2 节"](pt05ch48s02hlb02.md)。

**边列表**

构成路径的点由连接模型内节点的边组成。您可以通过从视口中选择单个单元边来指定边，或者通过选择起始边和方向并允许 Abaqus/CAE 自动完成路径到特征边的终点或您选择的终点的路径。边由模型中的单元定义。正因为如此，组成边列表路径的边同样适用于未变形或变形的模型形状。但是，单元标签是部件实例特定的。换句话说，多个部件实例可以具有相同的单元标签；因此，当您使用边时，必须指定您引用的是哪个部件实例。有关更多信息，请参阅  ["创建或编辑边列表路径，" 第 48.2.3 节"](pt05ch48s02hlb03.md)。

**圆形**

构成路径的点由模型内的坐标位置组成。您可以在 Abaqus/CAE 中创建两种类型的圆形路径：圆周和径向。圆周路径沿圆或圆弧的边缘；径向路径沿半径。您可以通过选择定义圆或圆弧的坐标，然后定义沿路径的起点、终点和点数来指定任一类型的路径。定义原始圆或圆弧后，您可以选择新的半径长度来创建与原始路径共享相同中心点的任何路径。您可以使用圆周路径来获取模型弯曲部分中的横截面结果，如 [图 48–1](pt05ch48s01s01.md#usv-path-circpath) 所示。

**图 48–1** 圆周路径。

![](../graphics/usv-path-circpath.png)

您可以使用径向路径来获取从曲线内面到外面延伸的结果，如 [图 48–2](pt05ch48s01s01.md#usv-path-radpath) 所示；径向路径非常适合用于应力线性化（有关应力线性化的更多信息，请参阅 [第 52 章，"计算线性化应力"](pt05ch52.md)"）。

**图 48–2** 径向路径。

![](../graphics/usv-path-radpath.png)

您可以通过从视口中拾取节点或通过在提示区域输入值来选择坐标；在这两种情况下，坐标在空间中是固定的，与模型无关。例如，与未变形形状上节点位置重合的坐标可能与变形形状上的任何位置都不重合。同样，圆形路径坐标定义与特定部件实例无关。有关更多信息，请参阅 ["创建或编辑圆形路径，" 第 48.2.4 节"](pt05ch48s02hlb04.md)。

Abaqus/CAE 通过按您指定的顺序连接节点、点或边来形成路径。

创建路径后，您可以选择 ****工具（Tools）**![](../graphics/images/arrow.gif)**路径（Path）**** 从主菜单栏进行编辑、复制、重命名、删除或绘制。绘制路径本身是一种可视化验证您已指定预期线条的手段；要沿路径查看结果，您必须形成 X–Y 数据对并生成 X–Y 绘图。有关管理路径的更多信息，请参阅 ["使用管理器对话框管理对象，" 第 3.4.9 节"](pt01ch03s04hlb01.md)；有关生成路径数据的 X–Y 绘图的更多信息，请参阅 ["沿路径获取 X–Y 数据，" 第 48.3 节"](pt05ch48s03.md)。

# 48.1.2 Abaqus/CAE 如何获取沿路径的结果





Abaqus/CAE 以 X–Y 数据对的形式获取沿路径的结果。Abaqus 根据构成路径的点确定数据对 X 值。这些点定义了获取数据的模型位置。Abaqus 根据这些模型位置的分析结果确定数据对 Y 值。Abaqus/CAE 在计算数据对时仅考虑当前显示组中的实体。

**路径数据 X 值**

您可以选择仅根据为路径指定的点形成 X 值，或者额外包括路径与模型相交的所有位置。（相交发生在路径穿过单元面、单元边、表面面或表面边的地方。如果相交位于实际数据点的用户指定距离内，您可以从路径定义中省略单个相交。您可以使用 Abaqus Scripting Interface 命令设置此相交容差值；有关更多信息，请参阅 ["会话对象，" Abaqus Scripting Reference Guide 第 55.18 节](../ker/ker-link.md#ker-session-xy-pyc)。）

您还可以选择 Abaqus/CAE 是否将构成路径的点解释为未变形或变形模型形状上的位置。当选择变形模型形状时，Abaqus/CAE 计算节点处的变形坐标作为基础坐标与该节点处变形乘以视口特定变形比例因子的和。节点列表标签和边列表指示模型位置，同样适用于未变形或变形的模型形状。点列表和圆形路径坐标是独立于模型几何形状的固定位置。Abaqus 不会为您选择的模型形状中不重合的点形成数据对。

构成路径的节点标签或点坐标通常不适合直接用作 X 值。Abaqus/CAE 提供了多个选项供您将此点系列转换为有用的 X 值和后续 X–Y 绘图轴标签。您可以选择以下选项之一将路径点转换为 X 值：
- **真实距离（True distance）**：X 值对应于每个点沿路径在模型空间坐标中的实际距离，从零开始。
- **归一化距离（Normalized distance）**：X 值对应于每个点沿路径的距离，作为路径总长度的一部分。
- **序列 ID（Sequence ID）**：X 值对应于每个点在路径结果列表中出现的顺序。
- **X、Y 或 Z 距离（X, Y, or Z distance）**：X 值对应于每个点沿路径在您指定的单个坐标方向上的实际距离，从零开始。此选项对于生成轴对称模型中结果与半径的关系图特别有用。

**路径数据 Y 值**

您可以通过选择 Abaqus/CAE 获取结果的结果步骤、帧和场输出变量来控制数据对 Y 值，并通过控制 Abaqus/CAE 计算某些类型结果的方式，如下所示：
- 对于要在节点位置获取的基于节点的场输出变量（如位移），Abaqus/CAE 直接从输出数据库读取结果，无需进一步计算。
- 对于要在节点位置获取的基于单元的场输出变量（如应力或应变），Abaqus/CAE 从输出数据库读取结果，将这些值外推到节点，然后根据您选择的选项有条件地对多个贡献进行平均。
- 对于要在与节点位置不重合的路径点上获取的基于节点和基于单元的变量，Abaqus/CAE 通过使用单元形状的几何近似从节点插值到请求的位置来计算值。您无法控制此计算。

基于单元变量的平均选项和复数值的复数形式选项位于 **结果选项（Result Options）** 对话框中。平均将多个贡献减少为单个值。当您选择结果选项部分或完全抑制平均时，接收多个贡献的路径点会产生多个数据对。这些数据对共享相同的 X 值，但每个贡献有单独的 Y 值。根据您的路径和模型的特性，可能需要以下技术来避免多个数据对共享相同的 X 值：
- 设置结果选项以完全启用平均。
- 设置结果选项以在计算值时忽略区域边界。
- 避免穿过区域不连续的路径。
- 避免沿分隔不连续区域的线的路径。
- 在获取沿路径的结果之前，使用显示组隔离各个区域。
- 避免穿过一个或多个在视觉上位于高度变形单元和大角度圆柱单元外部但由于等参映射在数学上位于单元内部的空间点的点列表路径。这些空间点往往由两个或多个单元共享，导致多个 Y 值。

有关值平均的更多信息，请参阅 ["理解结果值平均，" 第 42.6.2 节"](pt05ch42s04s02.md)。

如果您的选定场输出变量包含复数结果，则数据对的 Y 值会受到当前复数形式的影响。复数形式的缩写会附加到您绘制路径时 Y 轴标题。例如，如果 `S-Mises` 是选定的场输出变量，`Magnitude` 是复数形式，则 Y 轴标题为 `S-Mises CPX:Mg`。其他复数形式同样会缩写。有关复数形式的更多信息，请参阅 ["控制复数结果的形式，" 第 42.6.9 节"](pt05ch42s04hlb05.md)。

对于没有指定步骤、帧或场输出变量结果的点，Abaqus/CAE 不会形成数据对。
