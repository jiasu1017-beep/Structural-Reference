# 6.3 选择梁单元





- 在包含接触的任何模拟中，应使用一阶剪切可变形梁单元（B21、B31）。
- 如果横向剪切变形很重要，应使用 Timoshenko（二次）梁单元（B22、B32）。
- 如果结构非常刚性或非常柔软，在几何非线性模拟中应使用 Abaqus/Standard 中可用的混合梁单元（B21H、B32H 等）。
- Abaqus/Standard 中可用的 Euler-Bernoulli（三次）梁（B23、B33）对于包含分布载荷（如动力振动分析）的模拟非常准确。
- 具有开口薄壁横截面的结构应使用 Abaqus/Standard 中可用的开口截面翘曲理论（B31OS、B32OS）的单元进行建模。
