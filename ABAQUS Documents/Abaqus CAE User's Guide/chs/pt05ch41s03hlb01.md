# 41.3.1 诊断摘要





**作业诊断**对话框中的**摘要**页面始终可用。摘要包括**作业历史**树中当前高亮显示项目的属性，以及对话框其他页面上可用的诊断信息指示。当您从作业到迭代时，信息会变得越来越具体，如下所示。

**作业**

首次打开**作业诊断**对话框时，作业项目在**作业历史**树中高亮显示，**摘要**页面可见。显示作业名称和状态以及分析代码和版本。对于Abaqus/Explicit作业，还会显示Abaqus/Explicit精度（单精度或双精度）以及并行作业执行的域数。如果有警告或错误，也会显示每种的总数。

**步骤**

每个步骤的摘要显示步骤名称、步骤号、分析程序以及步骤中的警告数量（如果有）。根据程序，还可能显示其他信息作为步骤摘要的一部分。例如，通用非线性步骤的摘要还包括已完成的步骤时间、已完成的增量数、使用的时间增量方法（自动或固定）、步骤期间是否考虑了非线性几何，以及在每个增量开始时用于先前状态的外推类型。更多信息，请参见第14章"步骤模块"。

**增量**

每个增量的摘要显示增量号以及增量中的警告数量（如果有）。对于Abaqus/Standard分析，还会显示尝试次数。增量摘要还指示收敛状态；如果增量收敛，则显示增量大小和已完成的步骤时间。

**尝试**

Abaqus/Standard分析中每次尝试的摘要显示尝试号、尝试大小、警告数量（如果有）以及迭代次数。严重不连续迭代和平衡迭代分别列出，以及迭代总数。如果Abaqus无法找到解，它会减小增量大小并开始新的尝试；如果Abaqus进行了缩减，尝试摘要会指示缩减的原因。Abaqus/Explicit分析中没有尝试。

**迭代**

Abaqus/Standard分析中的迭代摘要显示收敛状态。如果迭代未收敛，摘要会指示提供未满足的收敛条件详细信息的其他页面（**警告**、**残差**、**接触**和**单元**）。Abaqus/Explicit分析中没有迭代。
