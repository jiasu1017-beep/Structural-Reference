# 7.1.1 压电换能器的特征值分析





**产品：** Abaqus/Standard

本问题对由带有黄铜端盖的压电材料制成的圆柱形换能器进行特征谱分析。分析中使用了各种单元。单元从轴对称单元到三维单元，包括低阶和高阶单元。Abaqus中压电能力的基础在["压电分析，" Abaqus Theory Guide的第2.10.1节](../stm/stm-link.md#stm-anl-piezoelectric)中描述。

1方向是径向的，2方向是轴向的，3方向是切向的。从这些矩阵可以看出，极化方向从对称轴向外的径向。（Abaqus中应力的顺序可能与电气应用中通常使用的不同。Abaqus使用标准机械约定，其中应力分量按顺序排列为。请参见["压电行为，" Abaqus Analysis User's Guide的第26.5.2节](../usb/usb-link.md#usb-mat-cpiezoelect)。）

黄铜是弹性和各向同性的，弹性模量为104 GPa，泊松比为0.37。

### 模型

换能器使用各种单元进行建模。它建模为轴对称结构，同时使用平面轴对称单元和三维单元。对于轴对称单元，使用4节点、6节点和8节点单元进行有限元离散化，使用五种网格。前两个网格使用4节点单元，具有两个细化级别，第三个网格使用6节点单元，最后两个网格使用8节点单元，具有两个细化级别。低阶单元使用集中质量矩阵。高阶单元使用一致质量矩阵。用于4节点和6节点轴对称单元的网格如图所示。用于8节点轴对称单元的网格如图所示。

三维模型使用结构的切片并施加轴对称边界条件；使用8节点和20节点砖单元。每个模型使用的离散化如图所示。这些模型使用局部坐标系来保持材料属性的正确定义。此外，为了规定轴对称边界条件，节点自由度被变换到圆柱坐标系中。

所有模型都被认为是开路的。内表面上的电位被限制为零。频率对应于反共振。

### 结果与讨论

各种Abaqus模型获得的结果如表所示。即使对于这些粗模型，结果也与实验结果非常接近。此外，Abaqus使用低阶轴对称单元与集中质量和具有一致质量矩阵的高阶轴对称单元计算的共振和反共振频率结果与Mercer等人报告的数值结果非常吻合。使用更精细的CAX8RE单元模型的最初四个模态形状如图所示。

还执行了考虑问题为闭路以获得共振频率的类似分析。对于这种情况，内表面和外表面上的电位都设置为零。结果也与Mercer等人给出的结果进行了良好比较。
