# 8.7 小结





- 结构问题中存在三个非线性来源：材料、几何和边界（接触）。Abaqus 分析中可能存在这些的任何组合。
- 当位移的大小影响结构的响应时，就会发生几何非线性。它包括大位移和大旋转、跳跃通过和载荷刚化的影响。
- 在 Abaqus/Standard 中，非线性问题使用 Newton-Raphson 方法迭代求解。非线性问题将需要线性问题所需计算机资源的多倍。
- Abaqus/Explicit 不需要迭代来获得解；但是，计算成本可能受到由于几何大变化导致的稳定时间增量减少的影响。
- 非线性分析步被分成多个增量。
  - Abaqus/Standard 迭代以找到每个新载荷增量结束时获得的近似静态平衡。Abaqus/Standard 通过在整个模拟过程中使用收敛控制来控制载荷增量。
  - Abaqus/Explicit 通过使用比隐式分析通常使用的更小的时间增量从前一个增量推进运动状态来确定解。增量的大小受稳定时间增量限制。默认情况下，Abaqus/Explicit 中的时间推进是完全自动的。
- 状态文件允许在分析运行时监控其进展。消息文件包含载荷增量和迭代的详细信息。
- 可以在每个增量结束时保存结果，以便在 Abaqus/Viewer 中可视化结构响应的演变。结果也可以 X-Y 图的形式绘制。
