# 42.4 自定义结果显示中的步骤和帧显示





您可以通过激活输出数据库中步骤和帧的子集来自定义分析结果的显示。Abaqus/CAE仅在您通过逐步切换帧、动画显示数据或从场数据创建X-Y历史数据来检查分析结果时显示活动的步骤和帧。活动步骤和帧的子集在会话期间保持不变，并适用于模型数据库结果的每次显示。

您可以从结果树或**Active Steps/Frames**对话框激活和停用步骤或帧。这两个工具还使您可以通过点击每个步骤名称左侧的"加"号和"减"号来展开和折叠步骤。展开步骤会显示其帧，使您可以激活部分帧而不是整个步骤。

结果树和**Active Steps/Frames**对话框还提供视觉提示以指示输出数据库中步骤或帧的活动状态。结果树在非活动帧的快捷方式左侧显示红色"X"；未以这种方式显示的帧当前处于活动状态。在**Active Steps/Frames**对话框中，步骤和帧状态通过出现在加/减号与步骤名称之间的区域中的复选标记表示。步骤的绿色复选标记表示其所有帧都处于活动状态；单个帧的绿色复选标记表示该帧处于活动状态；浅灰色背景上的深灰色复选标记表示步骤的部分帧处于活动状态；未出现复选标记时，步骤完全处于非活动状态。当步骤容器折叠时，此视觉提示可帮助您评估步骤的活动状态。[图42-1](pt05ch42s02s01.md#usv-anm-activesf)显示了对于两个步骤完全活动、一个步骤部分活动和两个步骤非活动的输出数据库的**Active Steps/Frames**对话框。

**Active Steps/Frames**对话框中的每一行还提供以下关于步骤的信息：
- **Step Name**和**Description**列反映创建模型时为该步骤定义的值。
- **Time**列中的值提供步骤的起始时间或描述步骤的性质。对于时间相关步骤，此列显示步骤的起始时间。如果步骤不是时间相关的，此列描述步骤是模态、频率还是弧长（Riks分析）步骤。
- **Period**列中的值也取决于步骤类型。对于时间相关步骤，周期是步骤的总持续时间。对于Riks分析步骤，周期是步骤的总弧长。模态和频率步骤在此列中显示破折号（`'-'`），表示此步骤不适用周期。只要周期不等于0，您可以修改时间相关步骤和Riks分析步骤的周期值。有关更多信息，请参见["更改步骤的周期，" 第42.4.2节](pt05ch42s02s02.md)。

**Active Steps/Frames**对话框提供两种激活步骤和帧的方法：您可以使用对话框**Selection Aids**部分的筛选工具激活一组步骤或帧，也可以通过点击对话框下半部分中的行来激活单个步骤和帧行。使用选择辅助工具激活步骤或帧会清除您当前活动的步骤和帧子集。如果您计划手动选择步骤和帧，请在使用选择辅助工具选择子集后再进行。

**选择辅助工具**

**Selection Aids**筛选选项使您可以根据多个选择条件选择步骤和帧的子集。
- **Select from**选项定义搜索的候选步骤和帧集。您可以搜索输出数据库中的所有步骤，也可以仅在**Active Steps/Frames**对话框下半部分当前选择的步骤或帧子集中搜索。
- **Select by**选项指定要搜索匹配步骤和帧的变量。您可以搜索在特定时间范围内发生的任何步骤或帧。对于频率提取步骤，您可以搜索模型在特定频率下表现出特定频率的任何步骤或帧。或者，您可以按帧号搜索帧。
- **Global range**字段使您可以指定搜索的上下界以及匹配步骤或帧之间的增量。例如，如果您想激活下限为帧0、上限为帧20之间的所有偶数帧，请输入2作为增量。
- 您还可以通过选择**Unselect base state frames**来停用输出数据库中的基状态帧。此选项仅在输出数据库包含线性扰动步骤时可用。
- 某些步骤可能具有重复前一步骤最终帧的初始帧。取消选择这些重复的首帧可以使数据分析更加平滑和真实。选择**Unselect duplicate first frames**可以取消选择在数据库所有步骤中的所有重复首帧。

**手动选择步骤和帧**

您可以通过点击**Active Steps/Frames**对话框下半部分中的行来激活和停用单个步骤和帧。点击单个步骤会激活或停用其所有帧，而点击帧则仅激活或停用该帧。如果您点击并拖动跨越多个步骤或帧，Abaqus/CAE会反转所选的所有步骤或帧的激活状态。

**提示：**您也可以从结果树激活或停用步骤和帧。高亮显示要切换的步骤或帧，点击鼠标按钮3，然后选择**Activate all**或**Deactivate all**来更改所选步骤内所有帧的活动状态，或选择**Activate**或**Deactivate**来更改所选帧的活动状态。

结果树在帧快捷方式左侧显示红色"X"来指示非活动帧。用户定义的会话步骤无法被停用，以红色感叹号指示；这些步骤不会显示在**Active Steps/Frames**对话框中。

在对话框中手动激活和停用步骤和帧可以补充和细化您使用选择辅助工具做出的选择。此外，在对话框下半部分点击鼠标按钮3可以访问以下手动选择快捷方式：
- **Expand All**展开所有步骤容器（已选择和未选择的），以显示输出数据库中每个步骤的帧。
- **Collapse All**折叠输出数据库中所有已选择和未选择的步骤容器。
- **Select All**选择输出数据库中每个步骤的每个帧。
- **Deselect All**取消选择输出数据库中每个步骤的每个帧。
- **Reset Selection**还原至上一次点击**Apply**时处于活动状态的步骤和帧子集。
- **Reset Periods**还原至上一次点击**Apply**时处于活动状态的周期设置。

**使用**Active Steps/Frames**对话框激活步骤和帧：**

1. 找到**Active Steps/Frames**对话框。从主菜单栏，选择****Result**![](../graphics/images/arrow.gif)**Active Steps/Frames****。**Active Steps/Frames**对话框出现。对话框的下半部分显示了输出数据库中的步骤，对于展开的步骤还显示了帧。在会话期间首次检查输出数据库的步骤和帧时，所有步骤和帧都处于活动状态。
2. 从对话框的**Selection Aids**部分，为您的搜索选择数据集、变量和全局范围值。
3. 点击**Update Selection**。Abaqus/CAE更新活动步骤和帧的子集。
4. 在对话框下半部分手动激活或停用步骤。您可以通过点击相应的行来切换单个步骤或帧的激活状态，或者通过点击并拖动多行来切换多行的激活状态。
5. 点击**Apply**。Abaqus/CAE使用新选择的活动步骤和帧子集更新所有显示此输出数据库的视口。
