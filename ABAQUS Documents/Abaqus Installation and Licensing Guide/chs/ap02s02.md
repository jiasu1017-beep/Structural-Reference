# C 配置 Abaqus 文档





## 附录 C：配置 Abaqus 文档

Abaqus HTML 文档与许可和产品安装分开安装，并通过 Web 浏览器查看。建议您安装并启动 Abaqus Web 服务器，以将 HTML（和 PDF）文件提供给用户的 Web 浏览器。您可以在没有 Web 服务器的情况下安装 HTML 文档；但是，在这种情况下，没有搜索功能可用。有关文档安装选项的信息，请参阅["Abaqus 文档安装详情，"第 2.2 节](ch02s02.md)。

本附录包含与配置 Web 服务器和 Abaqus HTML 文档相关的信息。如果您在 HTML 文档安装过程中使用现有的非 Abaqus Web 服务器，或者如果您在安装 Abaqus 产品之前安装 Abaqus HTML 文档，或者如果您在产品安装期间未指定 HTML 文档的 URL，则必须配置 Abaqus HTML 文档。

有关支持的 Web 服务器和配置现有非 Abaqus Web 服务器的最新信息，请参阅 Dassault Systmes Knowledge Base，网址为 [www.3ds.com/support/knowledge-base](http://www.3ds.com/support/knowledge-base)。

**注意：** 当您安装 Abaqus HTML 文档时，PDF 文件也会被安装。HTML 和 PDF 版本都可以通过 Web 浏览器获取，可通过 HTML 集合主页（`index.html`）到 PDF 集合索引文件（`index.pdf`）的链接访问。
