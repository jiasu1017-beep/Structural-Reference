# 12.14.4 创建实体复合铺层





在大多数情况下，您应该将复合实体建模为壳或连续壳复合铺层。但是，在以下情况下应使用实体复合铺层：
- 当横向剪切效应占主导时。
- 当您不能忽略正应力时。
- 当您需要精确的层间应力时，例如在复杂加载或几何形状的局部区域附近。

实体复合铺层预期在厚度方向上具有单个元素，该单个元素包含在铺层表格中定义的多个铺层。如果您分配实体复合铺层的区域包含多个元素，每个元素将包含铺层表格中定义的铺层，分析结果将不如预期。

以下章节描述如何创建实体复合铺层：
- ["创建实体复合铺层"](pt03ch12s14s04.md#usi-prp-composites-solid-create)
- ["指定实体复合铺层的铺层"](pt03ch12s14s04.md#usi-prp-composites-solid-plies)
- ["指定实体复合铺层所选铺层的显示"](pt03ch12s14s04.md#usi-prp-composites-solid-display)

### 创建实体复合铺层

在Abaqus/Standard中，实体元素可以包含用于层压复合实体分析的不同材料的几层；但是，在Abaqus/Explicit中，实体元素只能由单一均质材料组成。复合实体的使用仅限于仅具有位移自由度的三维砖元素。复合实体元素主要用于建模方便。它们通常不会比复合壳元素提供更准确的解决方案。请参见Abaqus Analysis User's Guide的第28.1.1节"实体（连续体）元素"中的"在Abaqus/Standard中定义复合实体元素"，以及"在Abaqus/Standard中使用实体元素建模厚复合材"。

**创建实体复合铺层：**

1. 从主菜单栏，选择****Composite**![](../graphics/images/arrow.gif)**Create****。出现**Create Composite Layup**对话框。
   **提示：**您也可以在**Composite Layup Manager**中点击**Create**，或在Property模块工具箱中选择创建复合铺层工具![](../graphics/ico_compositeLayup.png)。
2. 输入复合铺层名称。
3. 指定初始铺层数量。当复合铺层编辑器出现时，它将包含每一行的一个铺层；但是，您可以使用编辑器随后添加或删除铺层。
4. 选择**Solid**作为**Element Type**，然后点击**Continue**。出现复合铺层编辑器。
5. 输入铺层的描述。Abaqus/CAE此描述显示在复合铺层管理器中。
6. 执行以下操作之一来指定铺层取向：
   - 选择**Coordinate system**以选择现有坐标系（或创建新坐标系并选择它），然后执行以下操作：
     1. 选择代表**Rotation axis**的轴。
     2. 指定额外旋转。通过此角度绕选定轴旋转所选坐标系。您可以指定角度，或可以选择定义在铺层上空间变化的标量离散场。Abaqus/CAE仅允许您选择有效的离散场，对于额外旋转，这些是应用于元素的标量离散场。您还可以通过点击![](../graphics/ico_discreteField.png)创建新的离散场。请参见[第63章，"离散场工具集"](pt06ch63.md)。
   - 选择**Discrete**以定义离散取向，然后执行以下操作：
     1. 点击![](../graphics/ico_edit.png)。
     2. 在出现的**Edit Discrete Orientation**对话框中，使用["使用离散取向进行材料取向和复合铺层取向，"第12.16节"](pt03ch12hla01.md)中所述的过程定义法线轴和主轴。
     3. 选择代表**Rotation axis**的轴。
     4. 指定额外旋转。通过此角度绕所选法线轴旋转取向。您可以指定角度，或可以选择或创建标量离散场以定义在铺层上空间变化的角度。Abaqus/CAE仅允许您选择有效的离散场，对于额外旋转，这些是应用于元素的标量离散场。您还可以通过点击![](../graphics/ico_discreteField.png)创建新的离散场。请参见[第63章，"离散场工具集"](pt06ch63.md)。
   - 选择**User-defined**以在用户子程序[`ORIENT`](../sub/sub-link.md#sub-xsl-orient)中定义取向。此选项仅对Abaqus/Standard分析有效。请参阅以下章节获取更多信息：
     - ["指定常规作业设置，"第19.8.6节"](pt03ch19s08hlb06.md)
     - [Abaqus User Subroutines Reference Guide的第1.1.15节，"ORIENT"](../sub/sub-link.md#sub-rtn-uorient)
   - 选择取向离散场的名称以指定在铺层上空间变化的坐标系。您还可以通过点击![](../graphics/ico_discreteField.png)在**Definition**字段右侧创建新的离散场。请参见[第63章，"离散场工具集"](pt06ch63.md)。选择离散场后，您必须执行以下操作：
     1. 选择代表**Rotation axis**的轴。
     2. 指定额外旋转。通过此角度绕选定轴旋转所选坐标系。您可以指定角度，或可以选择或创建标量离散场以定义在铺层上空间变化的角度。
   铺层取向是任何使用默认取向系统的铺层的参考取向（由铺层表格**CSYS**列中的**<Layup>**指示）。此取向用于单个铺层中的材料计算和应力输出、截面力输出以及横向剪切刚度。您可以通过指定参考取向和/或旋转角度为实体复合铺层的单个铺层指定不同的取向。请参见["理解复合铺层和取向，"第23.3节"](pt04ch23s03.md)。
7. 选择以下之一来指定实体元素相对于一对元素面的堆叠方向：
   - **Element direction 1**
   - **Element direction 2**
   - **Element direction 3**
   您可以使用查询工具集确定网格堆叠方向。但是，显示的取向仅考虑扫掠路径；它们不考虑上述堆叠方向的变化。请参见["使用查询工具集查询模型，"第71.2.1节"](pt06ch71s02hlb01.md)获取有关查询工具集的更多信息。请参见Abaqus Analysis User's Guide的第29.6.1节"壳元素：概述"中的"定义堆叠和厚度方向"获取有关网格堆叠方向的更多信息。

### 指定实体复合铺层的铺层

复合铺层由一系列铺层组成。您选择分配铺层的区域，并指定每个铺层的名称、材料、厚度、取向和积分点数量。使用铺层表格上方的图标或点击铺层表格上的鼠标按钮3查看允许您编辑表格单元格内容和操作表格数据的菜单；例如，您可以添加和删除铺层、图案铺层和反转铺层。您还可以从文件读取数据到表格或从表格写入数据到文件。请参见["定义复合铺层时使用铺层表格，"第12.14.1节"](pt03ch12s14s01.md)。

**指定实体复合铺层的铺层：**

1. 从**Composite Layup**编辑器，点击**Plies**标签。
2. 如果复合铺层中的铺层关于中心芯对称，请切换**Make calculated sections symmetric**。在铺层表格中输入铺层，从第一行的底部铺层开始，到中心铺层结束。在分析过程中，Abaqus通过以相反顺序重复输入的铺层（包括中心铺层）将铺层附加到铺层定义到铺层的顶部。每个生成的铺层通过在重复铺层的原始名称前面添加`Sym_`来在铺层堆叠图和输出数据库中标记。如果任何铺层的旋转角度使用离散场定义，则此选项不能使用。
3. 对于每个铺层，在铺层表格中输入以下数据：
   **Ply Name** 铺层的名称。Abaqus/CAE在Visualization模块中查看复合铺层时显示此名称。
   **Region** 选择分配铺层的区域。您可以从视口中选择单元，或选择引用单元的集。如果您正在显示依赖网格的部件，可以从视口中选择实体单元或选择单元集。要从视口中选择单元，必须显示本地网格，并且必须使用**Selection**工具栏启用从视口选择**3D Elements**。您也可以从网格部件中选择实体单元。请参见["显示本地网格，"第17.3.11节"](pt03ch17s03s11.md)和["根据对象类型过滤您的选择，"第6.3.2节"](pt01ch06s03s01.md)。
   **Material** 此铺层的材料名称。点击鼠标按钮3，从出现的菜单中选择**Edit Material**，然后执行以下操作之一：
   - 从可用材料列表中选择所需的材料。
   - 点击![](../graphics/ico_material.png)创建新材料。
   **Element Relative Thickness** 相对铺层厚度。对于实体元素，Abaqus根据部件几何确定厚度，并且对于给定铺层，厚度可能在模型中变化。因此，您指定的厚度值仅是每层的相对厚度。铺层的实际厚度是元素厚度乘以每铺层占总厚度的分数。您不必使用物理单位来指定铺层的厚度比，并且铺层相对厚度的总和不必总和为一。
   **Coordinate system ** 要定义为铺层参考取向基础的坐标系，请执行以下操作：
   1. 点击鼠标按钮3，并从出现的菜单中选择**Edit CSYS**。
   **注意：**如果您从菜单中选择**Edit Orientation**，可以定义坐标系和旋转角度。
   2. 选择基础取向。您可以选择基础铺层取向，或选择坐标系。如果您选择坐标系，必须选择定义法线方向的轴。
   **Rotation Angle** 铺层参考取向的额外旋转（绕法线方向逆时针）。直接在表格中输入均匀旋转，或点击鼠标按钮3并选择**Edit Rotation Angle**以执行以下操作：
   - 选择预定义角度（0、45、90或45度）以定义均匀旋转。
   - 选择**Uniform**，并输入**Angle**以定义均匀旋转。
   - 选择标量离散场以定义在铺层上空间变化的旋转。您还可以通过点击![](../graphics/ico_discreteField.png)创建新的离散场。
   **Integration Points** 穿过铺层使用的积分点数量。此数字必须是奇数。

### 指定实体复合铺层所选铺层的显示

**指定所选铺层的显示：**

1. 从**Composite Layup**编辑器，点击**Display**标签。
2. 选择Abaqus/CAE高亮所选铺层的方式：
   - 选择**On**以高亮所选铺层。
   - 选择**Off**以停止高亮所选铺层。如果模型中有大量铺层，您可能希望关闭高亮以提高性能。
3. 选择Abaqus/CAE在所选铺层上显示的取向：
   - 选择**Ply**以在所选铺层上显示铺层取向。
   - 选择**Layup**以在所选铺层上显示铺层取向。
   - 选择**None**以停止显示取向。
4. 指定您要在所选铺层上显示的取向向量。您可以切换1方向、2方向和3（或法线）方向向量的显示，以及显示铺层材料方向旋转前1方向的参考方向。
