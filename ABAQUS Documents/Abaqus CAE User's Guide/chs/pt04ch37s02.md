# 37.2 管理弹簧和阻尼器





**Springs/Dashpots Manager**允许您创建和管理弹簧和阻阻器。管理器包含您已定义的弹簧和阻阻器的名称和类型列表。管理器中的**Create**、**Edit**、**Copy**、**Rename**和**Delete**按钮允许您创建新的弹簧和阻阻器或编辑、复制，重命名和删除现有弹簧和阻阻器。管理器左侧列中的图标允许您抑制和恢复现有弹簧和阻阻器。您还可以使用Property模块或Interaction模块主菜单栏中的****Special**![](../graphics/images/arrow.gif)**Springs/Dashpots****菜单启动这些过程。选择主菜单栏中的管理操作后，过程与在管理器对话框中单击相应按钮完全相同。
