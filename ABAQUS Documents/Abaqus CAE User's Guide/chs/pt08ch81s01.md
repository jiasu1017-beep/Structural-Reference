# 81.1 什么是插件？





插件是安装到另一个应用程序中以扩展该应用程序功能的软件。Abaqus 插件执行 Abaqus Scripting Interface 和 Abaqus GUI Toolkit 命令，它们提供了一种根据您的特定需求或偏好自定义 Abaqus/CAE 的方法。例如，一个简单的插件可以按照一些预定义的选项自动打印当前视口的内容。一个更复杂的插件可以为您的 specialized 后处理例程提供图形用户界面。

插件有两种类型：内核和 GUI。内核插件包含使用 Abaqus Scripting Interface 编写的函数。相比之下，GUI 插件是使用 Abaqus GUI Toolkit 编写的，包含创建图形用户界面的命令，这些命令反过来向内核发送命令。内核和 GUI 插件都可以从主菜单栏上的 **Plug-ins** 菜单或从插件工具箱中访问。

默认情况下，有几个内核和 GUI 示例插件可从主菜单栏上的 **Plug-ins** 菜单中获得。您可以使用这些示例来了解插件是如何创建的，以及插件如何与 Abaqus/CAE 交互。此外，您可以从插件工具箱中访问两个示例插件。当您从主菜单栏中选择 ****Plug-ins**![](../graphics/images/arrow.gif)**Toolboxes**![](../graphics/images/arrow.gif)**Examples**** 时，Abaqus/CAE 会显示 **Examples** 工具箱。您可以单击工具箱中的图标来启动插件。[图 81-1](pt08ch81s01.md#usi-plg-toolbox) 显示了 **Toolboxes** 菜单和 **Examples** 插件工具箱。
