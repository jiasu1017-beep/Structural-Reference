# 9.9.2 从文件加载会话对象和会话选项





您可以将会话对象或会话选项从文件加载到您的会话中。会话选项或对象可以从当前模型数据库、输出数据库或XML格式的设置文件加载。

当您加载会话对象或选项时，您可以加载在该文件中指定的所有会话特定设置，也可以从各个类别中选择设置。例如，您可以加载文件中所有的显示组和路径定义，同时排除所有视图切割定义。当您选择某个类别时，**加载会话对象和选项（Load Session Objects & Options）**对话框会加载该类别中的所有定义；您不能仅加载单个显示组同时排除其他显示组。如果您只想使用文件中包含的显示组的子集，则必须加载文件中包含的显示组，然后删除您不想使用的那些。

**要从文件加载会话对象或会话选项到您的会话中：**

1. 从主菜单栏选择****文件（File）**![](../graphics/images/arrow.gif)**加载会话对象（Load Session Objects）****。**提示：**您也可以从**文件**工具栏单击![](../graphics/icoR_persistenceLoad.png)。出现**加载会话对象和选项（Load Session Objects & Options）**对话框。
2. 从**源（Source）**选项中，执行以下操作：- 选择**文件（File）**从XML格式的设置文件加载，并指定文件名。- 选择**MDB（*.cae）**从当前模型数据库加载。- 选择**ODB**从输出数据库加载，并指定当前会话中打开的输出数据库之一。
3. 指定您要加载的会话对象或会话选项的类别：- 如果您想加载所有会话对象或所有会话选项，请分别切换**对象（Objects）**或**可视化选项（Visualization Options）**。- 如果您想单独选择会话对象或会话选项的类别，请展开**对象（Objects）**或**可视化选项（Visualization Options）**容器，然后切换您要加载的类别。
4. 单击**确定（OK）**。
