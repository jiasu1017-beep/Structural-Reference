# 11.9.1 实体特征





要创建实体特征，请从主菜单栏选择**形状**![](../graphics/images/arrow.gif)**实体**，或从零件模块工具箱中选择一个实体工具。绘制初始轮廓后，执行以下操作之一来创建特征：
- 要创建拉伸实体特征，请通过指定距离(*d*)拉伸轮廓。您可以向拉伸应用拔模角或扭曲。- 要创建实体放样特征，您可以从初始放样截面过渡到不同形状或方向的结束截面。Abaqus/CAE使用切线约束、中间截面和路径曲线来确定起始和结束截面之间的形状。- 要创建旋转实体特征，请通过指定角度旋转轮廓。构造线用作旋转轴。- 要创建扫掠实体特征，请沿指定路径扫掠轮廓。

您可以使用任何实体工具向在三维建模空间中创建的可变形或离散零件添加实体特征。您不能向二维或轴对称零件添加实体特征。
