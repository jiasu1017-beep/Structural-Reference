# 2.2.24 混凝土弥散裂纹

**产品：** Abaqus/Standard

### 测试单元

C3D8    C3D8H    C3D8R    CPS4    CPS4R    T3D2

### 问题描述

**材料：**

**弹性**

杨氏模量，*E* = 4.65E6

泊松比，![](../graphics/ver_eqn00181.gif) = 0.18

**塑性**

双轴/单轴压缩应力比 = 1.18

单轴拉伸/压缩应力比 = 0.1

双轴/单轴压缩塑性应变比 = 1.25

拉伸开裂应力/压缩应力比 = 0.2

硬化曲线：

| 屈服应力 | 塑性应变 |
| --- | --- |
| 1300.0 | 0.000000 |
| 2200.0 | 0.000027 |
| 3000.0 | 0.000100 |
| 3600.0 | 0.000225 |
| 4450.0 | 0.000550 |
| 4650.0 | 0.001000 |
| 4200.0 | 0.002000 |
| 2000.0 | 0.003500 |

开裂拉伸应力线性减小，在应变 = 5.0E4时达到零。

对于恒定断裂能开裂情况（mcou0o[1/2/3]hut.inp），开裂拉伸应力线性减小，在位移 = 6.0E4时达到零。

（单位不重要。）

### 结果与讨论

结果与精确解析或近似解非常一致。
