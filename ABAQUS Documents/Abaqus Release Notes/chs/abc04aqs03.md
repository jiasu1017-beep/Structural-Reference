# 4.3 在流体腔中指定附加体积





**产品：** Abaqus/Standard

**优势：** 现在，您可以在 Abaqus/Standard 分析中为流体腔指定附加体积。

**描述：** 此功能以前仅在 Abaqus/Explicit 中可用，现在也可在 Abaqus/Standard 中使用。您可以指定将添加到腔体实际体积的附加体积。
