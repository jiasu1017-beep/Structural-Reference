# 10.5 可以从模型导入什么？





您可以从输入文件、输出数据库或Abaqus/CAE模型数据库导入模型。您可以从输入文件或模型数据库导入完整模型；但是，您只能从输出数据库导入零件、截面定义、材料和梁轮廓。

以下章节描述了您可以如何导入模型。涵盖以下主题：
- ["从Abaqus/CAE模型数据库导入模型，"第10.5.1节"](pt02ch10s05s01.md)
- ["从Abaqus输入文件导入模型，"第10.5.2节"](pt02ch10s05s02.md)
- ["从输出数据库导入模型，"第10.5.3节"](pt02ch10s05s03.md)
- ["从Nastran输入文件导入模型，"第10.5.4节"](pt02ch10s05s04.md)
- ["从ANSYS输入文件导入模型，"第10.5.5节"](pt02ch10s05s05.md)

有关如何导入模型的详细说明，请参阅["导入模型，"第10.8节"](pt02ch10hla01.md)。
