# 35.5 创建和编辑中面模型





在Abaqus/CAE中创建中面模型的典型过程从实体模型开始。从实体模型中，选择要创建中面模型的单元。将这些实体单元分配为中面区域会在视口中创建一个参考表示，您可以从该参考表示创建中面模型的壳面。您可以使用Geometry Edit工具集中的工具来创建壳面，然后可以为新壳分配厚度。当您为壳几何体分配厚度和偏移时，默认设置将壳表示居中在厚度内。
