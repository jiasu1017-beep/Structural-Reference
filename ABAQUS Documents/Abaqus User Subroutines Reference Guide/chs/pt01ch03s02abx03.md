# 3.2.3 SIMULIA协同仿真引擎director执行





**产品：** Abaqus/Standard  Abaqus/Explicit  Abaqus/CFD

##### **参考文献**

- ["Abaqus执行过程：概述，" 第3.1.1节](pt01ch03s01abo02.md)

### 概述

Abaqus/Standard、Abaqus/Explicit和Abaqus/CFD之间的协同仿真由一个额外的过程SIMULIA协同仿真引擎（CSE）director控制。通常，您不需要调用CSE director过程；当您运行Abaqus协同仿真过程（["Abaqus/Standard、Abaqus/Explicit和Abaqus/CFD协同仿真执行，" 第3.2.4节](pt01ch03s02abx04.md)）或从Abaqus/CAE提交协同仿真时，它会自动调用。

如果您无法使用Abaqus协同仿真过程或Abaqus/CAE，并且需要使用Abaqus执行过程（["Abaqus/Standard、Abaqus/Explicit和Abaqus/CFD执行，" 第3.2.2节](pt01ch03s02abx02.md)）单独提交协同仿真分析，则必须按本节所述调用CSE director。

### 命令行选项

**job**

此选项的值指定运行期间生成的协同仿真摘要日志文件的名称。如果命令行中省略了此选项，系统将提示您输入其值。

**configure**

此选项指定控制协同仿真的SIMULIA协同仿真引擎配置文件的名称。更多信息，请参阅["准备Abaqus分析以进行协同仿真"中的"使用SIMULIA协同仿真引擎配置文件"第17.2.1节](pt04ch17s02aus98.md#usb-anl-acosimulationprep-config-file)。

**listenerport**

此选项用于指定协同仿真入站消息到director的TCP/UDP端口号。将**listenerport**设置为用于连接的端口号。

**datacheck**

此选项导致director仅检查配置文件的正确性。

**interactive**

此选项导致director以交互方式运行。

**timeout**

此选项用于指定协同仿真director连接的超时值（以秒为单位）。如果director在指定时间内未收到来自耦合分析程序的任何通信，则终止。默认值是3600秒。

### 示例

以下示例说明了当您需要单独提交协同仿真分析时协同仿真director执行过程的不同功能和能力。

#### 运行Abaqus/Standard到Abaqus/Explicit协同仿真

使用以下命令处理第一个Abaqus分析，在机器"earth"上运行，连接到在机器"mercury"上运行并监听端口44444的协同仿真director：

```
abaqus job=explicit csedirector=mercury:44444
```

使用以下命令处理第二个Abaqus分析，在机器"venus"上运行，连接到在机器"mercury"上运行并监听端口44444的协同仿真director：
```
abaqus job=standard csedirector=mercury:44444
```

使用以下命令处理协同仿真director，在机器"mercury"上运行，根据文件`explicit_standard_config.xml`中定义的协同仿真配置进行操作，并通过端口44444接收通信：
```
abaqus cse job=cosim listenerport=44444
   configure=explicit_standard_config.xml
```
