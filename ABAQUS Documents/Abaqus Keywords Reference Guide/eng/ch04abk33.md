# *DRAG CHAIN







### *DRAG CHAINSpecify parameters for drag chain elements.

This option is used to specify the maximum length of a drag chain, the frictional limit between the chain and the seabed, and the weight of the drag chain.

**Product: **Abaqus/Standard  

**Type: **Model data  

**Level: **Part,  Part instance  

##### **Reference:**

- ["Drag chains," Section 32.11.1 of the Abaqus Analysis User's Guide](../usb/usb-link.md#usb-elm-edragchain)

### **Required parameter: **

ELSET

Set this parameter equal to the name of the element set with which this behavior is associated.
