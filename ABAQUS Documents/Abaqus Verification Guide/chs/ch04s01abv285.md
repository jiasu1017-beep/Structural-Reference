# 4.1.10 RSURFU





### 4.1.10 [`RSURFU`](../sub/sub-link.md#sub-xsl-rsurfu)

**产品：**Abaqus/Standard  

### 测试单元

CAX4、S4R

### 测试功能

用于定义刚性表面几何形状的用户子程序。

### 问题描述

**ursurfux.inp**

此测试验证用户子程序[`RSURFU`](../sub/sub-link.md#sub-xsl-rsurfu)正确生成了三维刚性表面。该问题包括将弹性薄板成形在刚性圆柱体周围。此问题将与["变形体与网格化刚性表面之间的有限滑动接触，" 第1.6.7节](ch01s06abv85.md)中所示的测试进行比较，后者使用B样条表面而不是用户定义的刚性表面执行相同的分析。

圆柱体的半径为5英寸，其位移和旋转受到约束。薄板尺寸为5英寸×10英寸，用五十个4节点S4R壳单元建模。假设其为弹性材料，弹性模量为3×10^6 lb/in^2，泊松比为0.3。薄板最初定位为与圆柱体表面相切，一边缘固定在圆柱体表面上。施加700 lb/in^2的压力载荷到薄板表面以使其围绕圆柱体周向成形。除了沿固定边缘的节点外，所有壳节点都被放入接触节点集中。接触节点集被定义为从表面，用户定义的刚性表面被定义为主表面。不包括摩擦行为。

**ursurfu2.inp**

此测试比较两个模型，一个使用解析刚性表面，另一个使用用户子程序[`RSURFU`](../sub/sub-link.md#sub-xsl-rsurfu)。半径为10、厚度为1的圆形板使用二维轴对称CAX4单元建模。板假设为弹性材料，弹性模量为3×10^5，泊松比为0.3。板边界处的位移受到约束。轴对称刚性冲头位于板的一侧。施加1.5×10^5的载荷到冲头上以使板变形。

**ursurfu3.inp**

此测试比较两个模型，一个使用解析刚性表面，另一个使用用户子程序[`RSURFU`](../sub/sub-link.md#sub-xsl-rsurfu)。使用10×10网格的S4R壳单元建模方形板。板边界处的位移受到约束。刚性冲头位于板的一侧。施加3×10^4的载荷到冲头上以使板变形。

### 结果与讨论

第一个测试中变形薄板的位移与B样条刚性表面验证问题的结果在1%以内。

在第二个和第三个测试中，使用用户子程序的模型的结果与对应解析刚性表面模型的结果相同。
