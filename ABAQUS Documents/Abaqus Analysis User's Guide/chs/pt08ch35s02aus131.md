# 35.2.4 运动耦合约束

### 概述

运动耦合约束（Kinematic Coupling Constraints）用于将一个参考节点（控制节点）的运动与一组从节点的运动耦合起来。从节点保持与参考节点相同的刚体运动。

### 运动耦合原理

运动耦合约束强制从节点的自由度与参考节点保持线性关系：

$$

u_{i,j} = u_{ref,j} + \theta_{ref,k} \times r_{ij}

$$

其中 $u_{ref,j}$ 是参考节点位移，$\theta_{ref,k}$ 是参考节点转动，$r_{ij}$ 是从节点相对于参考节点的位置向量。

### 运动耦合定义

| **输入文件用法：** | ```
*KINEMATIC COUPLING
ref_node, TYPE=NAME
slave_node_1
slave_node_2
...
``` |
| --- | --- |

### 约束自由度

可以选择约束的自由度：

| **输入文件用法：** | ```
*KINEMATIC COUPLING, CONSTRAINT=NODAL
ref_node
slave_node_1
slave_node_2
...
``` |
| --- | --- |

### 全局运动耦合

约束所有6个自由度：

| **输入文件用法：** | ```
*KINEMATIC COUPLING, REF NODE=ref_node
slave_node_1
slave_node_2
...
``` |
| --- | --- |

### 部分运动耦合

仅约束平移自由度：

| **输入文件用法：** | ```
*KINEMATIC COUPLING, REF NODE=ref_node, DOF=1,2,3
slave_node_1
slave_node_2
...
``` |
| --- | --- |

### 应用场景

- 刚性连接模拟
- 连接件建模
- 加载条件简化
- 模型简化

### 与分布耦合的区别

| 特征 | 运动耦合 | 分布耦合 |
| --- | --- | --- |
| 从节点行为 | 保持刚体运动 | 可变形变形 |
| 力传递 | 集中力 | 分布力 |
| 计算效率 | 较高 | 较低 |
| 精度 | 较低 | 较高 |
