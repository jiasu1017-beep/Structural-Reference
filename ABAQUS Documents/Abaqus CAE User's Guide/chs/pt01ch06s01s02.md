# 6.1.2 什么是选择组？





你可以将视口中高亮显示的实体（顶点、边、面或单元体）复制到一个称为选择组的临时存储区域。Abaqus/CAE 在会话期间保存选择组。与其在后续选择过程中手动重新选择相同的实体，不如将选择组粘贴到你的选择中。例如，你可以将**几何诊断**查询工具高亮显示的所有小面复制到选择组。然后，在使用几何编辑工具集修复小面时，可以将相同的选择组粘贴到你的选择中。

当你将组粘贴到你的选择时，你可以从选择组和显示组中进行选择。选择组被设计为用户的临时便利，它们不会与显示组一起出现在显示组工具集中。你可以创建任意数量的显示组。相反，Abaqus/CAE 最多保存五个选择组。如果你创建了超过五个选择组，Abaqus/CAE 会覆盖现有的选择组。

选择所需的实体后，你可以通过在视口中单击鼠标按钮 3 并从出现的菜单中选择**复制**来创建选择组。Abaqus/CAE 将所有高亮显示的实体复制到一个选择组。你可以通过在视口中单击鼠标按钮 3 并从出现的菜单中选择**粘贴**将选择组粘贴到当前选择。Abaqus/CAE 显示**粘贴到选择**对话框，你可以选择一个或多个现有选择组粘贴到当前选择。当你粘贴组时，Abaqus/CAE 会将该组中的实体附加到你已经选择的其他实体。

**注意：**如果在选择过程之前选择对象，则无法创建或使用选择组。
