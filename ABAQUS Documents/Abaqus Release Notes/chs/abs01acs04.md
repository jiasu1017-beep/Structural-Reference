# 15.4 Abaqus输出变量标识符的变更





本节总结了在对Abaqus中使用的输出变量标识符所做的更改和添加。

| #### 单元积分点变量： |
| --- |
| **new** | **(S)(E)** | MMIXDME |
|  |  | 损伤演化过程中的模式混合比。 |
| **new** | **(S)(E)** | MMIXDMI |
|  |  | 损伤起始时的模式混合比。 |
| **new** | **(E)** | PEEQR |
|  |  | 等效塑性应变率。 |
| #### 整个单元变量： |
| **new** | **(S)** | ALEAKVRBXFEM |
|  |  | 富集单元底部裂纹表面的累积泄漏流量。 |
| **new** | **(S)** | ALEAKVRTXFEM |
|  |  | 富集单元顶部裂纹表面的累积泄漏流量。 |
| **new** | **(S)** | GFVRXFEM |
|  |  | 富集单元的间隙流体体积率。 |
| **new** | **(S)** | LEAKVRBXFEM |
|  |  | 富集单元底部裂纹表面的泄漏流率。 |
| **new** | **(S)** | LEAKVRTXFEM |
|  |  | 富集单元顶部裂纹表面的泄漏流率。 |
| **new** | **(S)** | PFOPENXFEM |
|  |  | 富集单元的断裂开口。 |
| **new** | **(S)** | PORPRES |
|  |  | 富集单元的流体压力。 |
| #### 节点变量： |
| **new** | **(E)** | AMAG |
|  |  | 平移加速度的大小。 |
| **new** | **(S)** | CLINELOAD |
|  |  | 由于边到表面和径向边到边约束中的线接触而产生的接触载荷，单位为力每长度。仅对于通用接触，法向（CLINELOADN）和摩擦剪切（CLINELOADT）分量可用。 |
| **new** | **(S)** | CPOINTLOAD |
|  |  | 由于使用交叉公式的边到边约束中的点接触而产生的接触载荷，单位为力。对于通用接触，法向（CPOINTLOADN）和摩擦剪切（CPOINTLOADT）分量可用。 |
| **new** | **(E)** | RFMAG |
|  |  | 反作用力的大小。 |
| **new** | **(E)** | UMAG |
|  |  | 平移位移的大小。 |
| **new** | **(E)** | VMAG |
|  |  | 平移速度的大小。 |
| #### 表面变量-节点量： |
| **new** | **(S)** | CRKDISP |
|  |  | 富集单元中裂纹表面的裂纹开口和相对切向运动。 |
| **mod** | **(S)** | CSDMG |
|  |  | 富集单元中裂纹表面的损伤变量。 |
| #### 总能量输出量： |
| **new** | **(S)** | ALLCCDW |
|  |  | 接触约束不连续功。 |
| **new** | **(S)** | ALLCCE |
|  |  | ALLCCEN和ALLCCET的总和。 |
| **new** | **(S)** | ALLCCEN |
|  |  | 由于惩罚约束施加而在法向方向的接触约束弹性能量。 |
| **new** | **(S)** | ALLCCET |
|  |  | 由于摩擦惩罚约束施加而在切向方向的接触约束弹性能量。 |
| **new** | **(S)** | ALLCCSD |
|  |  | ALLCCSDN和ALLCCSDT的总和。 |
| **new** | **(S)** | ALLCCSDN |
|  |  | 法向方向的接触约束稳定化耗散。 |
| **new** | **(S)** | ALLCCSDT |
|  |  | 切向方向的接触约束稳定化耗散。 |
| #### 状态和场变量： |
| **rem** | **(C)** | ENSTROPY |
|  |  | 单位质量的熵。 |
| **rem** | **(C)** | HELICITY |
|  |  | 涡量与速度的点积。 |
| **new** | **(C)** | QCRIT |
|  |  | 相干结构可视化器，即Q准则。 |
| #### 湍流变量： |
| **new** | **(C)** | TURBOMEGA |
|  |  | 比湍流能耗散率。 |
| **new** | **(C)** | TURBVISCOSITYRATIO |
|  |  | 涡流与分子粘度比。 |
