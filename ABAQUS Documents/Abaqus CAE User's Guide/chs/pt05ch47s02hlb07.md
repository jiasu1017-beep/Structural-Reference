# 47.2.7 保存X–Y数据对象









### 47.2.7 保存X--Y数据对象

Abaqus/CAE提供了多个对话框来帮助您[指定*X--Y*数据](pt05ch47s01s02.md)。每个这些对话框都包含一个**另存为**按钮，您可以使用它来保存创建的*X–Y*数据对象。您必须保存*X–Y*数据才能执行以下任一操作：
- 生成包含来自历史输出以外的任何源的多个*X--Y*数据对象的*X--Y*图。
- 编辑、复制、重命名、删除或运算您的*X--Y*数据。对于历史输出，您可以在保存过程中或保存后对数据执行运算。
- 为您的*X--Y*数据建立持久的曲线样式偏好。
- 生成您的*X--Y*数据的报告。
- 将您的*X--Y*数据复制到输出数据库文件。

Abaqus/CAE还保存为*X–Y*数据对象的每列指定的量类型，只要您选择的量类型属于**来自ASCII文件的XY数据**、**来自键盘的XY数据**和**编辑XY数据**对话框中可用的预设量类型。如果您通过对*X–Y*数据对象进行运算得出不同的量类型，Abaqus/CAE将不会在保存的*X–Y*数据对象中记录非标准量类型。

保存的数据仅在会话期间持续。要在会话结束后保存*X–Y*数据对象，必须将数据保存到文件。您可以将*X–Y*数据对象复制到输出数据库（二进制）文件，或将其写入报告（ASCII）文件。有关将*X–Y*数据复制到输出数据库文件的信息，请参见["将会话X--Y数据对象复制到输出数据库文件，" 第47.2.8节](pt05ch47s02hlb08.md)。有关将*X–Y*数据写入报告文件的信息，请参见[第54章，"生成表格数据报告"](pt05ch54.md)。

**要保存X–Y数据对象：**

1. 指定您的*X--Y*数据。1. 从主菜单栏选择****工具**![](../graphics/images/arrow.gif)**XY数据**![](../graphics/images/arrow.gif)**创建****。出现**创建XY数据**对话框。**提示：**您也可以通过点击**数据管理器**中的**创建**或使用工具箱中的![](../graphics/ico_xyData.png)工具来指定*X--Y*数据。2. [选择您的数据](pt05ch47s01s02.md)来源，然后点击**继续**。Abaqus显示相应的对话框。3. 在对话框中输入必要的数据或信息。
2. 为当前Abaqus会话保存数据。1. 点击**另存为**或**保存**（视可用性而定）。如果您从**来自ODB场输出的XY数据**对话框保存数据，则出现**保存XY数据**对话框。如果您从任何其他对话框保存*X--Y*数据，则出现**将XY数据另存为**对话框。2. 接受默认名称，或在文本字段中输入您选择的数据对象名称。**注意：**如果您从**来自ODB场输出的XY数据**对话框保存数据，则必须接受默认名称。如果您在**历史输出**对话框中选择了多个历史输出变量，*X--Y*数据集将使用默认名称*XYData*-1、*XYData*-2等保存。您可以输入新的名称前缀来替换*XYData*。3. 如果您要保存历史数据，请执行以下操作之一：- 要保存数据集而不更改它，请在**保存操作**列表中保留默认的**原样**选项。- 如果您选择了单个历史输出变量，您可以选择在**保存操作**列表中执行任何可用的[数学、三角、对数、指数或其他运算](pt05ch47s03hlb02.md)。选定的运算将在保存前对数据执行。- 如果您选择了两个历史输出变量，您可以选择执行任何可用的[双操作数运算](pt05ch47s03hlb02.md)：- [append((XY,XY))](pt05ch47s03hlb36.md)- [avg((XY,XY))](pt05ch47s03hlb08.md)- [combine((XY,XY))](pt05ch47s03hlb37.md)- [maxEnvelope((XY,XY))](pt05ch47s03hlb33.md)- [minEnvelope((XY,XY))](pt05ch47s03hlb34.md)- [power((XY,XY))](pt05ch47s03hlb23.md)- [rng((XY,XY))](pt05ch47s03hlb35.md)（范围）- [srss((XY,XY))](pt05ch47s03hlb17.md)（平方和的平方根）- [sum((XY,XY))](pt05ch47s03hlb18.md)- 如果您选择了三个或更多历史输出变量，您可以选择执行任何可用的[多操作数运算](pt05ch47s03hlb02.md)：- [append((XY,XY,...))](pt05ch47s03hlb36.md)- [avg((XY,XY,...))](pt05ch47s03hlb08.md)- [maxEnvelope((XY,XY,...))](pt05ch47s03hlb33.md)- [minEnvelope((XY,XY,...))](pt05ch47s03hlb34.md)- [rng((XY,XY,...))](pt05ch47s03hlb35.md)- [srss((XY,XY,...))](pt05ch47s03hlb17.md)- [sum((XY,XY,...))](pt05ch47s03hlb18.md)- [vectorMagnitude((XY,XY,XY))](pt05ch47s03hlb19.md)4. 如果可用，切换**确定时绘制曲线**以在点击**确定**时绘制*X--Y*数据集。5. 点击**确定**关闭对话框。
3. 点击**关闭**或**取消**（视可用性而定）以关闭输出对话框。
