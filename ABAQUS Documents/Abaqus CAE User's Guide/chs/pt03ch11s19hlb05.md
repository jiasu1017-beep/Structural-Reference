# 11.19.4 选择新零件的基础特征





使用**创建零件**对话框底部**基础特征**字段内的单选按钮和列表来选择您正在创建的零件的基础特征。选择取决于零件的建模空间和类型；例如，轴对称可变形体只能具有平面壳、平面线框或点基础特征。

基础特征类型的选择很重要，因为创建零件后不能更改类型。您可以修改基础特征，但应注意，随后添加到零件的任何特征都将链接到基础特征。因此，如果您修改基础特征，这些依赖的（或子）特征可能会移动或无法重新生成。

**选择基础特征：**

1. 从**创建零件**对话框底部，选择所需的基础特征形状（**实体**、**壳**、**线框**或**点**）。可用选择取决于您正在创建的零件的建模空间和类型。
2. 如果要创建三维可变形或离散刚体零件，还必须选择其**类型**（**拉伸**、**旋转**、**扫掠**、**平面**或**坐标**）。
3. 完成选项选择后，单击**继续**关闭**创建零件**对话框。草图编辑器启动，您绘制新零件基础特征的轮廓。
