# 19.6.1 控制重启分析





默认情况下，不会为 Abaqus/Standard 或 Abaqus/CFD 分析写入重启信息，而仅为 Abaqus/Explicit 分析在每个步骤的开始和结束时写入重启信息。您可以使用步骤模块更改写入重启信息的频率。更多信息，请参见[「重启输出请求，」第 14.5.2 节](pt03ch14s05s02.md)。

分析生成重启信息后，您可以控制后续重启分析的以下方面：

**模型属性**

要配置重启分析，您必须指定模型是否应重用同一模型先前分析的数据。您可以指定新分析应从中开始的先前分析的步骤和增量或时间间隔。您还可以选择以下选项之一：
- 允许所选步骤继续完成。
- 在指定增量处终止所选步骤并开始新步骤。

更多信息，请参见[「指定模型属性，」第 9.8.4 节](pt02ch09s07hlb04.md)。

**作业类型**

您使用作业编辑器指定作业类型。如果您编辑模型属性以指定模型应重用先前分析的数据，并且您创建引用此模型的作业，Abaqus/CAE 会将作业类型设置为**重启**。更多信息，请参见[「选择作业类型，」第 19.2.5 节](pt03ch19s02s05.md)。您不能创建引用与输入文件关联的作业的重启作业。
