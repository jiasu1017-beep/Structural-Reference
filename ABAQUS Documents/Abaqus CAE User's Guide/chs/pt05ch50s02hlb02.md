# 50.2.2 查询活动节点或单元标签





您可以使用查询工具集来请求当前视口中所有当前活动节点或活动单元的标签编号。活动节点和单元是当前显示组中的那些，减去活动视图切割从显示中移除的任何实体。Abaqus/CAE在提示区域显示每个适用部件实例的标签编号，并将标签编号打印到重放文件。

当您请求活动节点或单元时，Abaqus/CAE仅显示标签编号；它不提供任何其他节点或单元数据和结果。要显示有关单个节点或单元的更多信息，请分别参阅["探测节点，"第51.2.2节](pt05ch51s02hlb02.md)或["探测单元，"第51.2.3节](pt05ch51s02hlb03.md)。

**显示活动节点或活动单元标签：**

1. 找到"可视化模块查询"选项。从主菜单栏中，选择****工具**![](../graphics/images/arrow.gif)**查询****，或单击**查询**工具栏中的![](../graphics/ico_info.png)工具。"查询"对话框出现，"可视化模块查询"显示在对话框的下部。
2. 单击"活动节点"或"活动单元"。在提示区域中，Abaqus/CAE显示模型中每个部件实例的所有活动节点或活动单元的标签列表。

- ["探测单元，"第51.2.3节](pt05ch51s02hlb03.md)
