# 12.13.11 创建梁截面





梁截面行为根据梁截面对拉伸、弯曲、剪切和扭转的响应来定义。请参见[Abaqus Analysis User's Guide的第29.3.5节，"梁截面行为"](../usb/usb-link.md#usb-elm-ebeamsectionbehavior)。

创建梁截面时，必须选择截面积分方法。您可以选择在分析前提供截面属性数据（广义梁截面），或让Abaqus在分析期间从截面积分点计算（积分）横截面行为。以下章节描述如何为每种积分方法定义梁截面：
- ["为在分析期间积分的梁截面指定属性"](pt03ch12s13s04.md#usi-prp-section-beam-integrateduring)
- ["为广义梁截面指定属性"](pt03ch12s13s04.md#usi-prp-section-beam-integratebefore)

### 为在分析期间积分的梁截面指定属性

在分析期间积分的梁截面允许通过在横截面上应力的数值积分来计算横截面行为，以便在分析进行时定义梁的响应。材料行为在截面的每个点上独立评估。当截面非线性仅由非线性材料响应引起时，应使用这种类型的梁截面。请参见["使用在分析期间积分的梁截面来定义截面行为"，第29.3.6节Abaqus Analysis User's Guide](../usb/usb-link.md#usb-elm-eusingbeamsection)。

**为在分析期间积分的梁截面指定属性：**

1. 从主菜单栏，选择****Section**![](../graphics/images/arrow.gif)**Create****。出现**Create Section**对话框。
   **提示：**您也可以在**Section Manager**中点击**Create**，或在Property模块工具箱中选择创建截面工具![](../graphics/ico_section.png)。
2. 输入截面名称。有关命名对象的更多信息，请参见["使用基本对话框组件，"第3.2.1节"](pt01ch03s02s01.md)。
3. 选择**Beam**作为截面**Category**，选择**Beam**作为截面**Type**，然后点击**Continue**。出现梁截面编辑器。
4. 选择**During analysis**作为**Section integration**方法。
5. 为梁截面选择一个轮廓。如有需要，点击![](../graphics/ico_profile.png)创建轮廓；有关更多信息，请参见["创建轮廓，"第12.13.20节"](pt03ch12s13s05.md)。**Profile shape**字段更新以反映您的选择。
6. 在**Basic**标签页上：
   1. 选择要与此梁截面定义一起使用的**Material name**。如有需要，点击![](../graphics/ico_material.png)创建材料；有关更多信息，请参见["创建和编辑材料，"第12.7节"](pt03ch12s07.md)。
   2. 为**Section Poisson's ratio**输入值，以提供由于梁轴应变而在截面中产生的均匀应变（以便当梁被拉伸时横截面积发生变化）。此值必须介于1.0和0.5之间。值为0.5将强制不可压缩行为。
   3. 选择定义截面**Temperature variation**的方法：
      - 选择**Linear by gradients**以指示已指定横截面原点温度和穿过截面的温度梯度。您可以使用Load模块指定这些温度。
      - 选择**Interpolated from temperature points**以指示梁截面轮廓的形状决定了温度点的数量和位置。（有关温度点的更多信息，请参见[Abaqus Analysis User's Guide的第29.3.9节，"梁横截面库"](../usb/usb-link.md#usb-elm-ebeamcrosssectlib)。）您可以使用Load模块指定每个这些点的温度。
7. 在**Stiffness**标签页上，执行以下操作：
   1. 选择**Use consistent mass matrix formulation**以让Abaqus使用基于挠度的三次插值和旋转的二次插值的McCalley-Archer一致质量矩阵来计算梁截面的质量公式。如果关闭此选项，Abaqus使用集中质量公式进行此计算。
   2. 切换**Specify transverse shear**以在截面定义中包含非默认横向剪切刚度效应，并指定**Slenderness compensation**：
      - 选择**Use analysis product default**以让Abaqus从梁截面的弹性材料定义计算剪切刚度和细长比补偿因子。
      - 选择**Value**以直接指定横向剪切刚度效应。
      - 在提供的字段中输入细长比补偿因子。
      - 在提供的字段中输入![](../graphics/usi_eqn00383.gif)和![](../graphics/usi_eqn00384.gif)截面的剪切刚度值。
8. 在**Fluid Inertia**标签页上，切换**Specify fluid inertia effects**以模拟梁浸没在流体中的惯性效应。有关详细信息，请参见Abaqus Analysis User's Guide的第29.3.5节"梁截面行为"中的"浸没在流体中的额外惯性"。
   1. 指定梁是**Fully submerged**还是**Half submerged**在流体中。如果您选择**Half submerged**，每单位长度的附加惯性减少二分之一。
   2. 指定**Fluid density**。
   3. 在**Section radius**字段中输入湿润横截面的有效半径。
   4. 指定![](../graphics/usi_eqn00385.gif)，**Lateral motions of the beam**的附加质量系数。
   5. 指定![](../graphics/usi_eqn00386.gif)，**Motions along beam axis**的附加质量系数。
   6. 如果梁横截面原点与湿润横截面的质心不同，请在**X**和**Y**字段中指定质心相对于横截面原点的坐标。
9. 点击**OK**保存更改并关闭梁截面编辑器。

### 为广义梁截面指定属性

在广义梁截面中，横截面属性仅在预处理期间计算一次。分析期间的所有截面计算都以预计算的值进行。广义梁截面不需要材料定义。当梁的响应是线性的，或者当它是非线性的且非线性仅由材料非线性引起时（例如在截面坍塌的情况下），应使用这种类型的梁截面。请参见["使用广义梁截面定义截面行为"，第29.3.7节Abaqus Analysis User's Guide](../usb/usb-link.md#usb-elm-eusingbeamgensect)。

**为广义梁截面指定属性：**

1. 从主菜单栏，选择****Section**![](../graphics/images/arrow.gif)**Create****。出现**Create Section**对话框。
   **提示：**您也可以在**Section Manager**中点击**Create**，或在Property模块工具箱中选择创建截面工具![](../graphics/ico_section.png)。
2. 输入截面名称。有关命名对象的更多信息，请参见["使用基本对话框组件，"第3.2.1节"](pt01ch03s02s01.md)。
3. 选择**Beam**作为截面**Category**，选择**Beam**作为截面**Type**，然后点击**Continue**。出现梁截面编辑器。
4. 选择**Before analysis**作为**Section integration**方法。
5. 从**Beam shape along length**选项中，执行以下操作之一：
   - 要沿梁的整个长度保持相同的横截面轮廓，请选择**Constant**，并从**Beam Shape**选项中选择梁轮廓。如有需要，点击![](../graphics/ico_profile.png)创建轮廓；有关更多信息，请参见["创建轮廓，"第12.13.20节"](pt03ch12s13s05.md)。
   - 要在梁的每个端部定义不同的横截面轮廓，请选择**Tapered**，并分别从**Beam Start**和**Beam End**选项中选择起始和结束轮廓。如有需要，点击![](../graphics/ico_profile.png)创建轮廓；有关更多信息，请参见["创建轮廓，"第12.13.20节"](pt03ch12s13s05.md)。起始和结束轮廓必须具有相同的形状。锥形梁仅在Abaqus/Standard分析中受支持。Abaqus/CAE在起始和结束轮廓之间线性缩放梁轮廓。
6. 如果您选择了广义轮廓，您可以通过指定在横截面轴上移动截面质心和/或剪切中心的距离和方向来偏移梁截面。在**Centroid**和/或**Shear Center**的**local**![](../graphics/usi_eqn00387.gif)-和![](../graphics/usi_eqn00388.gif)-坐标字段中输入所需的值。
7. 在**Basic**标签页上：
   1. 要定义截面热膨胀系数，请切换**Use thermal expansion data**。数据表格中出现一个标为**Thermal Expansion**的列。
   2. 要定义依赖于温度的截面数据，请切换**Use temperature-dependent data**。数据表格中出现一个标为**Temperature**的列。
   3. 要定义依赖于场变量的截面数据，请点击**Number of field variables**字段右侧的箭头以增加或减少场变量数量。**Field**变量列出现在数据表格中。
   4. 在数据表格中为截面**Young's Modulus**和扭转**Shear Modulus**输入值。
   5. 为**Section Poisson's ratio**输入值，以提供由于梁轴应变而在截面中产生的均匀应变（以便当梁被拉伸时横截面积发生变化）。此值必须介于1.0和0.5之间。值为0.5将强制不可压缩行为。默认值为0。
   6. 要为梁截面定义密度，请切换**Specify section material density**，并在提供的字段中输入值。此值在Abaqus/Explicit分析中是必需的。在Abaqus/Standard分析中，仅在需要质量时（例如在动态分析或重力加载中）才需要此值。
   7. 如果热膨胀系数依赖于温度，请切换**Specify reference temperature**，并在提供的字段中输入热膨胀值。
8. 在**Damping**标签页上，指定阻尼属性以在截面的动态响应中包含质量和刚度比例阻尼：
   1. 在**Alpha**字段中输入值，以为直接积分动力学中的质量比例阻尼创建![](../graphics/usi_eqn00272.gif)因子。此值在模态动力学中被忽略。
   2. 在**Beta**字段中输入值，以为直接积分动力学中的刚度比例阻尼创建![](../graphics/usi_eqn00273.gif)因子。此值在模态动力学中被忽略。
   3. 在**Composite**字段中输入值，以指定在计算模态阻尼因子时使用的临界阻尼分数。此值仅适用于Abaqus/Standard分析，在直接积分动力学中被忽略。
9. 在**Stiffness**标签页上，执行以下操作：
   1. 选择**Use consistent mass matrix formulation**以让Abaqus使用基于挠度的三次插值和旋转的二次插值的McCalley-Archer一致质量矩阵来计算梁截面的质量公式。如果关闭此选项，Abaqus使用集中质量公式进行此计算。
   2. 切换**Specify transverse shear**以在截面定义中包含非默认横向剪切刚度效应，并指定**Slenderness compensation**：
      - 选择**Use analysis product default**以让Abaqus从梁截面的弹性材料定义计算剪切刚度和细长比补偿因子。
      - 选择**Value**以直接指定横向剪切刚度效应。
      - 在提供的字段中输入细长比补偿因子。
      - 在提供的字段中输入![](../graphics/usi_eqn00383.gif)和![](../graphics/usi_eqn00384.gif)截面的剪切刚度值。
10. 在**Fluid Inertia**标签页上，切换**Specify fluid inertia effects**以模拟梁浸没在流体中的惯性效应。有关详细信息，请参见Abaqus Analysis User's Guide的第29.3.5节"梁截面行为"中的"浸没在流体中的额外惯性"。
    1. 指定梁是**Fully submerged**还是**Half submerged**在流体中。如果您选择**Half submerged**，每单位长度的附加惯性减少二分之一。
    2. 指定**Fluid density**。
    3. 在**Section radius**字段中输入湿润横截面的有效半径。
    4. 指定![](../graphics/usi_eqn00385.gif)，**Lateral motions of the beam**的附加质量系数。
    5. 指定![](../graphics/usi_eqn00386.gif)，**Motions along beam axis**的附加质量系数。
    6. 如果梁横截面原点与湿润横截面的质心不同，请在**X**和**Y**字段中指定质心相对于横截面原点的坐标。
11. 在**Output Points**标签页上，您可以定位需要应力输出的梁截面中的点。在所需数量的截面点的局部![](../graphics/usi_eqn00389.gif)-和![](../graphics/usi_eqn00390.gif)-位置输入值。
12. 点击**OK**保存更改并关闭梁截面编辑器。
