# 4.4.9 在当前视口中隐藏标注





如果视口中的箭头和文本标注遮挡了模型视图，你可以使用**标注管理器**来隐藏它们。

Abaqus/CAE 生成的标注会根据**视口标注选项**对话框中的设置显示（有关详细信息，请参阅[第 56 章，"自定义视口标注"](pt05ch56.md)"）。

**要隐藏视口标注：**

1. 从主菜单栏中，选择****Viewport**![](../graphics/images/arrow.gif)**Annotation Manager****。**提示：**你也可以通过单击[**视口**工具栏](pt01ch04s02hlb02.md)中的![](../graphics/ico_manager.png)来显示**标注管理器**对话框。
2. 单击**MDB**或**ODB**单选按钮来选择要隐藏的标注来源。
   1. 如果你打开了多个输出数据库，或不确定涉及的数据库文件，请将光标悬停在**MDB**或**ODB**单选按钮上，以显示包含数据库路径和文件名的工具提示。
   2. 如有需要，通过在当前视口中显示所需数据库来更改活动输出数据库。或者，你可以打开新视口来显示数据库，或关闭所有其他输出数据库。Abaqus/CAE 会列出所选数据库中可用的标注。
3. 从对话框列表中选择标注（有关详细信息，请参阅["从列表和表中选择多个项目，" 第 3.2.11 节](pt01ch03s02s11.md)）。**提示：**切换**在视口中高亮显示选择**可以预览所选标注。
4. 单击**隐藏**。Abaqus/CAE 会在当前视口中隐藏所选标注。
