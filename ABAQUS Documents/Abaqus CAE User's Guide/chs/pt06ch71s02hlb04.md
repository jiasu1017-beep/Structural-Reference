# 71.2.4 使用几何诊断工具







本节描述如何使用几何诊断工具在当前视口中定位部件或部件实例中的以下内容：

**几何体**

- **无效实体**：Abaqus/CAE高亮显示部件或部件实例中具有无效几何体的区域。切换**Update validity**可更新所选部件或部件实例的状态。如果无效部件变为有效，它会在模型树中标记为有效。如果有效部件变为无效，Abaqus/CAE会显示一条警告消息，允许您忽略无效状态并继续使用该部件。
- **不精确实体**：Abaqus/CAE高亮显示具有不精确几何体的部件区域。

有关无效几何体的更多信息，请参见["什么是有效和精确的部件？," 第10.2.1节](pt02ch10s02s01.md)和["处理无效部件," 第10.2.3节](pt02ch10s02s03.md)。

**拓扑**

- **自由边**：Abaqus/CAE高亮显示部件或部件实例的自由边，包括壳边和线边。实体部件不应包含任何自由边。

- **实体单元**：Abaqus/CAE高亮显示部件或部件实例的实体单元。

- **壳面**：Abaqus/CAE高亮显示部件或部件实例的壳面。

- **线边**：Abaqus/CAE高亮显示部件或部件实例的线边。线边是线，而不是实体或壳特征的边。

**小几何体**

短边、小面和小面角小的面等小实体可能会影响网格质量。您可以使用以下工具定位这些区域：
- **Edges shorter than:** Abaqus/CAE高亮显示短于指定长度的边。指定的长度必须大于等于10^6。
- **Faces smaller than:** Abaqus/CAE高亮显示小于指定面积的面。指定的面积必须大于等于10^12。
- **Face corner angles less than:** Abaqus/CAE高亮显示两条边以小于指定角度相交的面。指定的角度必须小于90。Abaqus/CAE使用高长宽比的单元对小角面进行网格划分，这会影响网格质量。

**使用几何诊断工具的步骤：**

1. 从主菜单栏选择****Tools**![](../graphics/images/arrow.gif)**Query****，或单击**Query**工具栏中的![](../graphics/ico_info.png)工具启动查询工具集。
2. 从**Query**对话框中选择**Geometry diagnostics**。如果视口包含多个部件实例，请选择要查询的实例。Abaqus/CAE显示**Geometry Diagnostics**对话框。
3. 切换所需的，几何诊断选项并根据需要指定新值。
4. 单击**Highlight**。
   - Abaqus/CAE高亮显示所有具有相应无效、不精确或小几何体的区域。如果部件尺寸相对较大，即使高亮显示，短边和小面可能不可见。为了帮助您找到这些短边和小面，Abaqus/CAE会在高亮显示的区域周围显示标记。
   - 如果您选择了**Update validity**，Abaqus/CAE还会检查并更新部件或装配体的有效性状态。对于复杂几何体，有效性检查可能计算成本较高。
   - 如果没有找到特定诊断工具的任何区域，Abaqus/CAE会在消息区域显示一条消息。
5. 如果需要，在提示区域单击**Create Set**创建一个包含高亮显示区域的几何集。在出现的对话框中，输入集的名称并单击**Continue**。几何编辑工具集提供了修复无效或不精确区域的工具。几何编辑工具集允许您在选择要修复的区域时指定一个集。
   **注意：**您可以通过切换**Apply to reference representation**来查看参考表示的几何诊断。但是，参考表示中的项目将被排除在创建的几何集之外，因为参考表示无法修复。有关更多信息，请参见["理解参考表示," 第35.2节](pt04ch35s02.md)。
