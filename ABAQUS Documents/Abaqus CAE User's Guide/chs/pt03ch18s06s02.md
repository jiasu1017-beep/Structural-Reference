# 18.6.2 配置拓扑优化任务





优化模块提供了多种设置，允许您配置拓扑优化任务。配置设置取决于您是为通用拓扑优化还是为基于条件的拓扑优化配置优化任务。涵盖以下主题：
- ["配置通用拓扑优化任务](pt03ch18s06s01.md#usi-opz-topo-taskconfigure-general)"
- ["配置基于条件的拓扑优化任务](pt03ch18s06s01.md#usi-opz-topo-taskconfigure-stiffness)"

#### 配置通用拓扑优化任务

通用拓扑优化是一种灵活的、基于灵敏度的优化，允许您对模型应用一系列约束和目标函数。您使用优化任务编辑器来自定义通用拓扑优化的各个方面。要找到编辑器，请从主菜单栏中选择****任务**![](../graphics/images/arrow.gif)**编辑**![](../graphics/images/arrow.gif)***优化任务名称*****。要指定通用拓扑优化，请选择**高级**选项卡并选择**通用优化（基于灵敏度）**。

##### 配置基本设置

**配置基本设置：**

1. 在优化任务编辑器中，单击**基本**选项卡。
2. 选择是否冻结载荷或边界条件区域。建议您冻结应用了规定条件的区域，因为您不希望这些区域在优化期间被移除。冻结这些区域可以稳定优化过程，通常会导致显著更少的迭代次数。

##### 配置密度设置

**配置密度设置：**

1. 在优化任务编辑器中，单击**密度**选项卡。
2. 选择**密度更新策略**。此设置控制优化模块在优化期间更新设计单元相对材料密度的速率。在大多数情况下，您应该接受默认设置（**正常**）。但是，如果设计响应非常敏感且您在满足约束条件方面遇到问题，则可能需要更保守的速率，这需要更多的优化迭代。
3. 执行以下操作之一来指定每个单元在初始优化迭代期间的相对密度：
   - 选择**优化产品默认值**允许优化模块确定初始密度。如果材料体积被选为约束，优化模块计算初始密度使得体积约束恰好被满足。如果材料体积被选为目标函数，则每个单元的初始相对密度为50%。
   - 选择**指定**并输入一个值（0.0 < *初始密度* ≤ 1.0）。仅当体积被选为目标函数而非约束时，才应使用此选项，并且如果您在优化之前知道将初始密度设置为更大或更小的值将满足其他约束（例如位移约束），则可以使用此选项。您可以结合体积约束使用大于0.5的值来稳定非线性或接触问题并改善收敛行为。
4. 输入**最小密度**、**最大密度**和**每个设计周期的最大变化**。最小密度必须大于0.0，最大密度必须小于或等于1.0。不建议更改密度边界，特别是上限。如果默认值导致近乎奇异的刚度矩阵，您可能需要增加下限。数值实验表明，0.25（默认值）是密度最大变化的可接受值。对于复杂的设计响应和优化公式，建议使用0.1之类的密度变化下限。但是，较低的下限通常会导致更多的优化迭代。

##### 配置扰动设置

**配置扰动设置：**

1. 在优化任务编辑器中，单击**扰动**选项卡。
2. 输入要跟踪的特征模态数量。默认值为5，这意味着优化模块将尝试跟踪五个最低特征频率。在某些情况下，在优化迭代期间会出现许多局部低频特征模态，这导致要跟踪的模态数量众多，性能下降。您可以通过选择特征频率下限为第一个优化迭代中感兴趣特征频率的25%来避免跟踪大量模态。如果您的设计响应将使用Kreisselmaier-Steinhauser公式来评估特征频率，则不需要模态跟踪。您的Abaqus模型必须包含至少您正在跟踪的特征频率数量的输出请求。
3. 选择优化模块应该跟踪特征模态的区域。

##### 配置收敛选项

**配置收敛选项：**

1. 在优化任务编辑器中，单击**收敛**选项卡。
2. 指定**收敛标准**。以下选项允许您为通用拓扑优化指定收敛标准：
**指定何时开始检查收敛**
您可以指定优化模块将开始检查两个收敛标准的迭代。优化将始终至少继续直到达到此值。默认值为4。
**指定要检查哪个收敛标准**
您可以指定优化是在满足任一收敛标准还是两个收敛标准都满足时结束。默认值为必须满足两个标准。
**基于优化函数变化的收敛**
您可以指定优化将基于从一个迭代到下一个迭代的目标函数变化结束。默认值为0.001。
**基于单元密度变化的收敛**
单元密度是拓扑优化的设计变量。您可以指定优化将基于从一个迭代到下一个迭代的单元密度平均变化结束。默认值为0.005。

##### 配置高级选项

**配置高级选项：**

1. 在优化任务编辑器中，单击**高级**选项卡。
2. 选择**通用优化**算法。
3. 选择是否**删除区域中的软单元**。在拓扑优化过程中，优化模块在设计区域内分配给定的质量，同时尝试满足约束条件并优化目标函数。在优化结束时，结构包含硬（填充）单元和软（空）单元。软单元对结构的刚度影响可以忽略不计；但它们仍与结构的自由度数量相关，因此影响优化过程的速度。**删除软单元**选项允许您选择一个区域，从该区域中移除仅具有软邻单元的软单元。如有需要，删除的单元会被重新激活；例如，如果力流在优化期间发生变化。选择删除软单元可以帮助Abaqus收敛到解决方案，因为否则这些单元会退化或崩溃，建议您在优化非线性模型时使用。此外，选择**保守**密度更新策略和每个设计周期较小的密度变化将提高结果的准确性。更多信息，请参见["配置拓扑优化任务中的密度设置，"第18.6.2节](pt03ch18s06s01.md#usi-opz-topo-taskconfigure-general-density)。
4. 如果您选择删除软单元，您可以通过选择仅删除具有软邻单元的软单元来防止孤立的软单元被移除。您可以将邻单元定义为在**平均边长**（默认值）或您输入的值指定的半径内的单元。如果网格内的单元边长变化很大，则从平均边长计算的半径可能具有误导性。
5. 如果您选择删除软单元，您可以选择优化模块用于删除单元的方法：
   **优先连续性（标准）**
   选择**优先连续性（标准）**并输入**相对材料密度阈值**以在删除软单元之前检查连续性。如果优化模型包含通过软单元与模型其余部分分隔的硬单元"岛"，则优化模块不会移除软单元。此外，优化模块保留阻止硬单元相互移动的软单元；例如，共享公共边但不共享公共面的硬单元。如果单元的相对材料密度小于阈值，则认为该单元是"软"的，优化模块将其从分析中移除。
   **优先连续性（激进）**
   选择**优先连续性（激进）**并输入**相对材料密度阈值**以无论连续性如何都移除软单元。如果单元的相对材料密度小于阈值，则认为该单元是"软"的，优化模块将其从分析中移除。
   **最大剪切应变**
   选择**最大剪切应变**并输入**最大剪切应变阈值**。优化模块从分析中移除剪切应变大于阈值的单元。
   **最小主应变**
   选择**最小主应变**并输入**最小主应变阈值**。优化模块移除主应变低于阈值的单元。
   **最大弹塑性应变**
   选择**最大弹塑性应变**并输入**最大弹塑性应变阈值**。优化模块移除弹塑性应变大于阈值的单元。
   **体积压缩**
   选择**体积压缩**并输入**相对体积压缩**。优化模块移除正在压缩且相对体积低于阈值的单元。相对体积![](../graphics/usi_eqn00650.gif)定义为![](../graphics/usi_eqn00651.gif)，其中![](../graphics/usi_eqn00652.gif)是变形单元体积，![](../graphics/usi_eqn00653.gif)是原始单元体积。如果您的模型使用壳或膜单元，或者您的模型正在经历大变形，您应该选择**体积压缩**。
   **注意：**您选择的软删除方法取决于材料行为和单元类型，您可能需要实验以确定最佳方法及其阈值。文件`TOSCA.OUT`包含有关被移除单元的信息，并将帮助您确定最佳软删除方法和阈值。**优先连续性**方法提供默认**相对材料密度阈值**0.05。相比之下，应变和体积方法不提供默认阈值，因为适当值取决于您的模型；例如，取决于材料的属性。
6. 选择**材料插值技术**和**惩罚因子**。优化生成密度接近一的硬单元或密度接近零的空单元。拓扑优化引入密度介于1和0之间的单元，材料插值技术计算这些中间单元的密度与刚度之间的关系。**SIMP**（惩罚固体各向同性材料）插值方案定义单元密度与刚度之间的指数关系，适用于静态问题。惩罚因子应大于`1`，数值实验表明默认值3会产生良好结果。**RAMP**（材料属性的有理近似）插值方案适用于动态问题。惩罚因子应大于0，数值实验表明默认值3会产生良好结果。默认情况下，优化模块为静态问题选择**SIMP**插值方案，如果模型中至少出现一个动态载荷工况，则选择**RAMP**插值方案。

### 配置基于条件的拓扑优化任务

基于条件的拓扑优化使用应变能目标函数和体积约束。您使用优化任务编辑器来自定义基于条件的拓扑优化的各个方面。要找到编辑器，请从主菜单栏中选择****任务**![](../graphics/images/arrow.gif)**编辑**![](../graphics/images/arrow.gif)***优化任务名称*****。要指定基于条件的拓扑优化，请选择**高级**选项卡并选择**基于条件的优化**。

##### 配置高级选项

**配置高级选项：**

1. 在优化任务编辑器中，单击**高级**选项卡。
2. 选择**基于条件的优化**算法。
3. 选择是否**删除区域中的软单元**。在拓扑优化过程中，优化模块在设计区域内分配给定的质量，同时尝试满足约束条件并优化目标函数。在优化结束时，结构包含硬（填充）单元和软（空）单元。软单元对结构的刚度影响可以忽略不计；但它们仍与结构的自由度数量相关，因此影响优化过程的速度。**删除软单元**选项允许您选择一个区域，从该区域中移除仅具有软邻单元的软单元。如有需要，删除的单元会被重新激活；例如，如果力流在优化期间发生变化。
4. 如果您选择删除软单元，您可以通过选择仅删除具有软邻单元的软单元来防止孤立的软单元被移除。您可以将邻单元定义为在**平均边长**（默认值）或您输入的值指定的半径内的单元。如果网格内的单元边长变化很大，则从平均边长计算的半径可能具有误导性。
5. 如果您选择删除软单元，您可以选择优化模块用于删除单元的方法：
   - 选择**标准删除**在删除软单元之前检查连续性。如果优化模型包含通过软单元与模型其余部分分隔的硬单元"岛"，则优化模块不会移除软单元。此外，优化模块保留阻止硬单元相互移动的软单元；例如，共享公共边但不共享公共面的硬单元。
   - 选择**激进删除**无论连续性如何都删除软单元。
6. 如果需要，输入**相对材料密度阈值**的值。如果单元的相对材料密度小于此值，则认为该单元是"软"的，优化模块将其从分析中移除。
7. 选择优化模块在拓扑优化期间修改单元属性的速率。您可以选择速率（**非常小**、**小**、**中等**、**中**或**大**），并允许优化模块计算满足此速率所需的设计周期数。或者，您可以选择**动态**并输入最大设计周期数。最小设计周期数为10，默认值为15。减少设计周期数可能导致优化中的不希望的效果。虽然结果结构具有相同的刚度（应变能的总和几乎相等），但改变优化速度可能导致解决方案中出现不同的桁架配置。
8. 选择第一个周期后删除的体积。您可以输入百分比或绝对值。默认情况下，优化模块在第一个迭代中移除设计区域体积的5%。在某些情况下，增加此起始值将加速优化而不影响解决方案，特别是对于在较大区域中存在相对较低应力的模型。相反，如果起始值太高，优化模块可能在第一个迭代中移除太多单元，导致优化失败或结构粗糙。
