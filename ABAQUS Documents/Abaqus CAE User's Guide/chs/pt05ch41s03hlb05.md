# 41.3.5 接触





**作业诊断**对话框中的**接触**页面显示模型中接触状态的变化阻止Abaqus接受Abaqus/Standard步骤的解的区域的信息。**接触**页面也可以显示Abaqus/Explicit步骤的诊断，例如初始接触过盈和调整。

Abaqus/Standard接触信息可用于任何后面带有(SDI)（严重不连续迭代）的**作业历史**树中的迭代。接触信息也可用于接触影响分析的其他迭代，例如存在粘滑摩擦行为时。使用接触信息来定位模型中Abaqus无法建立正确条件的区域。您可能需要编辑接触控制来解决接触问题。更多信息，请参见"自定义接触控制"，第15.12.3节。

接触信息（包括每次迭代中的问题总数）以表格形式汇总，用于包含接触诊断的每次尝试。您在**描述**字段中的选择控制表中的数据，您可以通过选择**绘制所选列**来绘制表中的列。当您找到有问题的迭代时，在**作业历史**树中选择它。然后，您可以从该迭代的**接触**页面选择项目，并单击**在视口中高亮显示选择**来在当前视口中查看区域。单击列标题可以对迭代的**详细信息**字段中的信息进行排序。

对于Abaqus/Explicit步骤，接触摘要可能包括诊断，例如初始过盈数、未解决的初始过盈数和初始接触调整数。您在**描述**字段中的选择控制表中的数据。例如，如果您选择初始过盈或未解决的初始过盈，表会列出过盈的节点和单元、节点穿透的单元面以及过盈量。您可以从**详细信息**字段中选择项目，并单击**在视口中高亮显示选择**来在当前视口中查看接触节点和单元。
