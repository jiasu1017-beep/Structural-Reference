# 17.13.4 自动自适应重新网格划分和手动自适应重新网格划分有什么区别？





如果您选择允许Abaqus/CAE迭代地重新网格化您的模型，则"作业"模块中的自适应过程会为您控制自适应重新网格划分。您只需要在"网格"模块中定义重新网格划分规则，并将其应用于您想要重新网格化的模型区域。

相反，您可以手动应用修改后的重新网格划分规则，并查看修改对生成网格的影响。当您满意重新网格划分规则正在产生所需的网格时，您可以使用该规则来驱动由Abaqus/CAE控制的迭代重新网格划分和分析操作序列。从主菜单栏中选择****自适应**![](../graphics/images/arrow.gif)**手动自适应重新网格划分****来手动应用自适应重新网格划分规则。更多信息，请参见["手动调整大小和重新网格划分，"第17.21.6节](pt03ch17s21s06.md)。

当您应用手动自适应重新网格划分时，您必须输入由先前分析模型生成的输出数据库文件名。该输出数据库包含Abaqus/CAE用于计算网格尺寸函数的误差指标输出变量。存储在输出数据库中的误差指标输出变量限制了您可以对重新网格划分规则进行的更改。例如，如果您的原始规则在某个区域指定了能量密度，则您将无法将规则切换为使用等效塑性应变误差，而无需先重新运行分析。您可以修改重新网格划分规则中的尺寸方法和单元尺寸约束，但仍可使用先前分析的输出数据库。但是，如果您修改了步骤、区域或误差指标输出变量，则无法使用该输出数据库。
