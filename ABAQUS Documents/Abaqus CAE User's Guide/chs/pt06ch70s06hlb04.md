# 70.6.4 使用曲线路径方法分区面







您可以用垂直于界定面的两条边的曲线来分区面。这些边不需要是连续的，但它们必须界定该面且所夹角小于 180 度（如下图所示）：

![](../graphics/par-face-curve-nls.png)

您首先选择要分区的面，然后选择两种方法之一来定义分区沿这两条边的位置；Abaqus/CAE 绘制连接两点的 Bézier 曲线。您可以通过以下方式沿边定位分区：
- 输入介于零和一之间的参数值，其中零代表边的起点顶点，一代表边的终点。此方法允许您沿边精确地定位分区。此外，您随后可以使用特征操作工具集修改分区，以编辑您输入的两个参数来定位分区。
- 点击沿边的中点或基准点。如果您使用此方法定位曲线分区，您将无法稍后修改分区。

**使用曲线路径方法分区面：**

1. 从主菜单栏中选择 ****Tools**![](../graphics/images/arrow.gif)**Partition****（工具→分区）。出现 **Create Partition**（创建分区）对话框。Abaqus/CAE 在提示区域显示提示以指导您完成过程。**提示：**您也可以使用位于模块工具箱分区面工具处的 ![](../graphics/ico_ptnFaceCurvedPath.png) 工具来使用曲线路径方法分区面。 有关工具箱中分区工具的图表，请参阅["使用分区工具集，" 第 70.2 节"](pt06ch70s02.md)。
2. 从对话框顶部的 **Type**（类型）单选按钮中选择 **Face**（面）。**Method**（方法）列表显示可用于分区面的方法。
3. 从方法列表中选择 **Curved path normal to 2 edges**（垂直于两条边的曲线路径）。
4. 选择要分区的面。Abaqus/CAE 高亮显示所选面。
5. 从提示区域中的按钮中选择以下方法之一来定位曲线： - **Enter Parameter**（输入参数） - **Pick**（选择）
6. 选择界定所选面的任何边。**提示：**如果您无法选择所需的边，可以使用 **Selection**（选择）工具栏更改选择行为。 有关更多信息，请参阅["使用选择选项，" 第 6.3 节"](pt01ch06s03.md)。Abaqus/CAE 高亮显示所选边。
7. 执行以下操作之一： - 如果您选择了 **Enter Parameter**（输入参数）方法，输入介于零和一之间的值。所选边上的箭头指示参数值增加的方向。 - 如果您选择了 **Pick**（选择）方法，点击沿所选边的中点或基准点。
8. 选择界定面的第二条边并重复上一步。两 selected 边所夹的角必须小于 180 度。
9. 在提示区域中点击 **Create Partition**（创建分区）。Abaqus/CAE 创建分区。
