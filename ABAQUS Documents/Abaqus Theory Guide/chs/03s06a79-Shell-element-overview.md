# 3.6.1 壳单元概述

### 3.6.1 壳单元概述

Abaqus壳单元库提供了允许建模曲线、相交壳的单元，这些壳可以表现出非线性材料响应并承受大的整体运动（平移和旋转）。Abaqus壳单元也可以对复合材料的弯曲行为进行建模。

该库分为三类：通用壳、薄壳和厚壳单元。薄壳单元提供壳问题的解，这些问题被经典（Kirchhoff）壳理论充分描述；厚壳单元为最适合用剪切柔性（Mindlin）壳理论建模的结构提供解；通用壳单元可以为薄壳和厚壳问题提供解。所有壳单元使用近似于Koiter-Sanders壳理论（[Budiansky和Sanders, 1963](07s01a01-References.md)）的弯曲应变度量。虽然Abaqus/Standard提供所有三类的壳单元，Abaqus/Explicit仅提供通用壳单元。对于大多数应用，通用壳单元应该是用户从单元库中的首选。然而，对于特定应用，通过选择薄壳或厚壳单元之一可能获得增强的性能。还应注意，并非所有Abaqus壳单元都为有限应变分析而公式化。

通用壳单元是轴对称单元SAX1、SAX2和SAX2T以及三维单元S3、S4、S3R、S4R、S4RS、S3RS和S4RSW，其中S4RS、S3RS和S4RSW是小应变单元，仅在Abaqus/Explicit中可用。通用单元在薄壳和厚壳问题的所有加载条件下提供稳健和准确的解。在其公式中允许厚度变化作为面内变形的函数。它们不会受到横向剪切锁定的影响，也没有任何无约束的沙漏模式。除了小应变单元外，所有这些单元都考虑有限膜应变。轴对称通用壳不需要沙漏控制，完全积分单元S4的弯曲和膜响应也不需要。S4的膜运动学基于假设应变公式，为面内弯曲行为提供准确的解。Abaqus/Explicit单元S3RS、S4RS和S4RSW非常适合许多冲击动力学问题，包括经历大规模屈曲行为的结构，这些涉及小应变但大旋转和严重弯曲。这些单元使用简化的应变计算和沙漏控制方法，并提供计算速度方面的显著优势。

薄壳单元仅在Abaqus/Standard中可用。STRI3和STRI65是三角形小应变薄壳单元；S4R5、S8R5和S9R5是四边形小应变薄壳单元，而SAXA是有限应变薄壳单元，适合于建模承受任意加载的轴对称几何。薄壳单元可以在希望通过使用五自由度壳减少自由度数量的大型问题中提供增强的性能。但是，它们应该仅用于至多表现出弱非线性的薄结构建模，在不需要旋转自由度输出的问题中，以及壳表面和位移场平滑的情况，这样使用二阶壳可以实现更高的精度。SAXA单元在仅用少量周向Fourier模式就能准确描述变形周向变化时，非常有效地对承受非对称变形的轴对称结构进行建模。

离散Kirchhoff（DK）约束，指在壳表面上离散点满足Kirchhoff约束，在Abaqus的所有薄壳单元中被强制执行。对于单元类型STRI3，约束被解析地强制执行，不涉及横向剪切应变能计算。使用这些单元获得的解收敛于对应于经典壳理论的解。对于单元类型STRI65、S4R5、S8R5、S9R5和SAXA，离散Kirchhoff约束被数值强制执行，其中横向剪切刚度作为惩罚强制执行约束。

可以用剪切柔性壳理论正确定义且产生平滑位移场的壳行为可以用二阶Abaqus/Standard厚壳单元S8R准确分析。这个单元需要不可忽略的横向剪切柔性才能正常工作；因此，该单元适用于复合壳和夹层壳的分析。由于严重的横向剪切锁定，S8R单元的不规则网格收敛性很差；因此，建议在厚壳应用的规则网格几何形状中使用。
