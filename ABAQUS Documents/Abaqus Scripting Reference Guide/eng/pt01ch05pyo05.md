# 5.5 Viewport object







The following commands operate on Viewport objects. For more information about the Viewport object, see ["Viewport object," Section 11.4](pt01ch11pyo04.md).

**Access**

```
import annotationToolset
```

### 5.5.1 plotAnnotation(...)

This method plots an [Annotation](pt01ch05pyo01.md)                       object in aViewport.

**Required argument**

*annotation*

An [Annotation](pt01ch05pyo01.md)                                object to plot.

**Optional argument**

*index*

An Int specifying the index of the  [Annotation](pt01ch05pyo01.md)                                object in the sequence of annotations to plot. The default value is zero.

**Return value**

None

**Exceptions**

None.
