# 6.1 梁横截面几何







您可以使用 [*BEAM SECTION](../key/key-link.md#usb-kws-mbeamsection) 选项或 [*BEAM GENERAL SECTION](../key/key-link.md#usb-kws-mbeamgensect) 选项定义梁截面。使用任一选项，您可以通过指定截面的形状和尺寸来几何定义梁横截面。 [*BEAM GENERAL SECTION](../key/key-link.md#usb-kws-mbeamgensect) 选项也可以通过截面工程属性（如面积和惯性矩）定义梁截面。或者，梁截面可以基于特殊二维单元网格，对这些单元几何量数值计算。

如果您决定几何定义梁轮廓，Abaqus 提供了多种常见横截面形状，如[图 6-1](ch06s01.md#gss-beamsection) 所示。您也可以使用任意横截面定义定义几乎任何薄壁横截面。有关 Abaqus 中可用的梁横截面的详细讨论，请参阅 ["Beam cross-section library," Abaqus Analysis User's Guide 第 29.3.9 节](../usb/usb-link.md#usb-elm-ebeamcrosssectlib)。

**图 6-1** 梁横截面。

![](../graphics/gss-beamsection-nls.png)

[*BEAM SECTION](../key/key/link.md#usb-kws-mbeamsection) 选项的基本格式为

```
[*BEAM SECTION](../key/key-link.md#usb-kws-mbeamsection), ELSET=*<element set name>*, SECTION=*<section type>*, 
MATERIAL=*<material name>*
*<cross-section dimensions>* 
*<*![](../graphics/gsk_eqn00057.gif)*>*,*<*![](../graphics/gsk_eqn00058.gif)*>*,*<*![](../graphics/gsk_eqn00059.gif)*>*
```
将 SECTION 参数设置为 [图 6-1](ch06s01.md#gss-beamsection) 中所示的横截面之一。提供每种横截面类型所需的不同横截面尺寸，如 ["Beam cross-section library," Abaqus Analysis User's Guide 第 29.3.9 节](../usb/usb-link.md#usb-elm-ebeamcrosssectlib) 中所指定。第二数据行上的向量定义近似法线 ![](../graphics/gsk_eqn00060.gif)，稍后将在本节中解释。

[*BEAM GENERAL SECTION](../key/key-link.md#usb-kws-mbeamgensect) 选项的基本格式为

```
[*BEAM GENERAL SECTION](../key/key-link.md#usb-kws-mbeamgensect), ELSET=*<element set name>*, SECTION=*<section type>*
*<cross-section dimensions> or <section engineering properties>* 
*<*![](../graphics/gsk_eqn00057.gif)*>*,*<*![](../graphics/gsk_eqn00058.gif)*>*,*<*![](../graphics/gsk_eqn00059.gif)*>* 
*<Young's modulus (E)>*,*<torsional shear modulus (G)>*
```

要将截面属性几何定义，请将 SECTION 参数设置为 [图 6-1](ch06s01.md#gss-beamsection) 中所示的横截面之一。在这种情况下，您以与使用 [*BEAM SECTION](../key/key-link.md#usb-kws-mbeamsection) 相同的方式提供所需的横截面尺寸。第二数据行上的向量再次定义近似法线 ![](../graphics/gsk_eqn00060.gif)。在第三行上输入弹性材料常数，因为 [*BEAM GENERAL SECTION](../key/key-link.md#usb-kws-mbeamgensect) 不引用任何材料选项块。如果使用此选项几何定义截面属性，则材料行为必须是线性弹性的。

另一种方法是将 SECTION 参数设置为 GENERAL 或 NONLINEAR GENERAL，在这种情况下，您提供截面工程属性（面积、惯性矩和扭转常数）而不是横截面尺寸。这些参数允许您组合梁的几何和材料行为来定义其对载荷的响应。此响应可以是线性或非线性的。详见 ["Using a general beam section to define the section behavior," Abaqus Analysis User's Guide 第 29.3.7 节](../usb/usb-link.md#usb-elm-eusingbeamgensect)。

在 Abaqus/Standard 中，您还可以定义具有线性锥形截面的梁。支持线性响应的通用梁截面和标准库截面。

网格化梁截面允许描述包含多种材料和复杂几何的梁横截面。此类梁轮廓在 ["Meshed beam cross-sections," Abaqus Analysis User's Guide 第 10.6.1 节](../usb/usb-link.md#usb-anl-ameshedsection) 中进一步讨论。

### 6.1.1 截面点

当您指定 [*BEAM SECTION](../key/key-link.md#usb-kws-mbeamsection) 时，Abaqus 在梁截面上的整个截面点阵列计算梁单元的响应。截面点的数量以及截面点位置在 ["Beam cross-section library," Abaqus Analysis User's Guide 第 29.3.9 节](../usb/usb-link.md#usb-elm-ebeamcrosssectlib) 中显示。单元输出变量（如应力和应变）在任何截面点都可用；但是，默认情况下，仅在 ["Beam cross-section library," Abaqus Analysis User's Guide 第 29.3.9 节](../usb/usb-link.md#usb-elm-ebeamcrosssectlib) 中列出的选定截面点提供输出。矩形截面（SECTION=RECT）的所有截面点在[图 6-2](ch06s01.md#gss-rectbeamelem) 中显示。 

**图 6-2** B32 矩形梁单元中的积分和默认截面点。

![](../graphics/gss-rectbeamelem-nls.png)

对于此截面，默认情况下在点 1、5、21 和 25 提供输出。所示梁单元使用总共 50 个截面点，在每个积分点处 25 个，来计算其刚度。

当您指定 [*BEAM GENERAL SECTION](../key/key-link.md#usb-kws-mbeamgensect) 时，Abaqus 不在截面点计算梁的响应。相反，它使用截面工程属性来确定截面响应。因此，Abaqus 仅将截面点用作输出位置，您需要指定您需要输出的截面点。使用必须在 [*BEAM GENERAL SECTION](../key/key-link.md#usb-kws-mbeamgensect) 选项后跟随的 [*SECTION POINTS](../key/key-link.md#usb-kws-msectionpoints) 选项来指定截面点的位置： 

```
[*SECTION POINTS](../key/key-link.md#usb-kws-msectionpoints)
 *<*![](../graphics/gsk_eqn00025.gif)*>*,*<*![](../graphics/gsk_eqn00026.gif)*>*
```

![](../graphics/gsk_eqn00025.gif) 和 ![](../graphics/gsk_eqn00026.gif) 坐标在梁截面的局部 1-2 坐标系中给出。例如，如果我们需要如图 [图 6-3](ch06s01.md#gss-sectionpoints) 所示矩形梁截面的角点处应力，我们将使用以下选项块：

```
[*SECTION POINTS](../key/key-link.md#usb-kws-msectionpoints)
-0.01, -0.005
 0.01, -0.005 
 0.01,  0.005
-0.01,  0.005
```

![](../graphics/gss-sectionpoints.png)

您指定的点根据它们给出的顺序分配标识编号；即，第一个点是截面点 1，第二个是截面点 2，等等。

### 6.1.2 横截面方向

您必须在全局笛卡尔空间中定义梁截面的方向。沿梁单元的局部切线 ![](../graphics/gsk_eqn00061.gif) 定义为指向从单元第一个节点到下一个节点的单元轴向的向量。梁横截面垂直于此局部切线。局部（1-2）梁截面轴由向量 ![](../graphics/gsk_eqn00060.gif) 和 ![](../graphics/gsk_eqn00062.gif) 表示。三个向量 ![](../graphics/gsk_eqn00061.gif)、![](../graphics/gsk_eqn00060.gif) 和 ![](../graphics/gsk_eqn00062.gif) 形成局部右手坐标系（见[图 6-4](ch06s01.md#gss-crosssect-orient)）。

**图 6-4** 梁单元切线 ![](../graphics/gsk_eqn00061.gif) 和梁截面轴 ![](../graphics/gsk_eqn00060.gif) 和 ![](../graphics/gsk_eqn00062.gif) 的方向。

![](../graphics/gss-crosssect-orient-nls.png)

对于二维梁单元，![](../graphics/gsk_eqn00060.gif) 方向始终为 (0.0, 0.0, 1.0)。

对于三维梁单元，有几种方法可以定义局部梁截面轴的方向。最简单的是在 [*ELEMENT](../key/key-link.md#usb-kws-melement) 选项的数据行上定义单元时指定一个额外节点。从梁单元的第一个节点到此额外节点（见[图 6-4](ch06s01.md#gss-crosssect-orient)）的向量 ![](../graphics/gsk_eqn00063.gif) 最初用作近似的 ![](../graphics/gsk_eqn00060.gif) 方向。然后 Abaqus 将梁的 ![](../graphics/gsk_eqn00062.gif) 方向定义为 ![](../graphics/gsk_eqn00064.gif)。在确定 ![](../graphics/gsk_eqn00062.gif) 后，Abaqus 将实际 ![](../graphics/gsk_eqn00060.gif) 方向定义为 ![](../graphics/gsk_eqn00065.gif)。此过程确保局部切线和局部梁截面轴形成正交系统。

或者，您可以在元素截面选项（[*BEAM SECTION](../key/key-link.md#usb-kws-mbeamsection) 或 [*BEAM GENERAL SECTION](../key/key-link.md#usb-kws-mbeamgensect)）上给出近似的 ![](../graphics/gsk_eqn00060.gif) 方向。然后 Abaqus 使用上述过程计算实际梁截面轴。如果同时指定了额外节点和近似的 ![](../graphics/gsk_eqn00060.gif) 方向，则额外节点方法优先。如果未提供近似的 ![](../graphics/gsk_eqn00060.gif) 方向，Abaqus 使用从原点到点 (0.0, 0.0, 1.0) 的向量作为默认 ![](../graphics/gsk_eqn00060.gif) 方向。

有两种方法可以覆盖 Abaqus 定义的 ![](../graphics/gsk_eqn00062.gif) 方向。一种是在 [*NODE](../key/key-link.md#usb-kws-mnode) 选项的数据行上节点坐标后的第 4、第 5 和第 6 个数据值中给出 ![](../graphics/gsk_eqn00062.gif) 的分量。另一种是使用 [*NORMAL](../key/key-link.md#usb-kws-mnormal) 选项。如果同时使用两种方法，则 [*NORMAL](../key/key-link.md#usb-kws-mnormal) 选项优先。Abaqus 再次将 ![](../graphics/gsk_eqn00060.gif) 方向定义为 ![](../graphics/gsk_eqn00066.gif)。

您提供的 ![](../graphics/gsk_eqn00062.gif) 方向不必与梁单元切线 ![](../graphics/gsk_eqn00061.gif) 正交。当您提供 ![](../graphics/gsk_eqn00062.gif) 方向时，局部梁单元切线被重新定义为叉积 ![](../graphics/gsk_eqn00067.gif) 的值。在这种情况下，重新定义的局部梁切线 ![](../graphics/gsk_eqn00061.gif) 可能与从第一个到第二个节点定义的梁轴不一致。如果 ![](../graphics/gsk_eqn00062.gif) 方向与垂直于单元轴的平面成大于 20° 的角度，Abaqus 会在数据文件中发出警告消息。

["示例：货运起重机，" 6.4 节](ch06s04) 中给出的示例说明了如何在模型中分配梁横截面方向。

### 6.1.3 梁单元曲率

梁单元的曲率基于梁的 ![](../graphics/gsk_eqn00062.gif) 方向相对于梁轴的方向。如果 ![](../graphics/gsk_eqn00062.gif) 方向与梁轴不正交（即梁轴和切线 ![](../graphics/gsk_eqn00061.gif) 不重合），则梁单元最初被认为是弯曲的。由于弯曲梁的行为与直梁不同，您应始终检查模型以确保使用正确的法线，从而确保正确的曲率。对于梁和壳，Abaqus 使用相同算法来确定在多个单元共享的节点处的法线。描述在 ["Beam element cross-section orientation," Abaqus Analysis User's Guide 第 29.3.4 节](../usb/usb-link.md#usb-elm-ebeamcrosssection) 中给出。

如果您打算对弯曲梁结构建模，您应该使用前述两种方法之一直接定义 ![](../graphics/gsk_eqn00062.gif) 方向，从而允许您在建模曲率方面进行很大控制。即使您打算对由直梁组成的结构建模，当法线在共享节点处平均时，也可能引入曲率。您可以通过如前所述直接定义梁法线来纠正此问题。

### 6.1.4 梁截面中的节点偏移

当梁单元用作壳模型的加劲件时，使梁和壳单元共享相同节点是方便的。默认情况下，壳单元节点位于壳的中面，而梁单元节点位于梁截面的某个位置。因此，如果壳和梁单元共享相同节点，则壳和梁加劲件将重叠，除非梁截面从节点位置偏移（见[图 6-5](ch06s01.md#gss-nodaloffset)）。

**图 6-5** 使用梁作为壳模型的加劲件：(a) 无梁截面偏移；(b) 有梁截面偏移。

![](../graphics/gss-nodaloffset-nls.png)

对于横截面类型 I、TRAPEZOID 和 ARBITRARY，可以指定截面几何位于元素节点处截面局部坐标系的原点一定距离处。由于这种横截面的梁易于从其节点偏移，因此它们可以容易地用作加劲件（如图 [图 6-5](ch06s01.md#gss-nodaloffset)(b) 所示）。（如果加劲件的翼缘或腹板屈曲很重要，则应使用壳建模加劲件。）

如图 [图 6-6](ch06s01.md#gss-ibeam) 所示的 I 梁连接到 1.2 单位厚的壳。以下输入用于按图所示方向定向梁截面：

```
[*BEAM SECTION](../key/key-link.md#usb-kws-mbeamsection), SECTION=I, ELSET=*<element set name>*, MATERIAL=*<material>*
0.6, 2.4, 3.0, 2.0, 0.2, 0.2, 0.2
*<*![](../graphics/gsk_eqn00057.gif)*>*, *<*![](../graphics/gsk_eqn00058.gif)*>*,*<*![](../graphics/gsk_eqn00059.gif)*>*
```

**图 6-6** 用作壳单元加劲件的 I 梁。

![](../graphics/gss-ibeam-nls.png)

第一项在第一数据行上定义梁节点从 I 截面底部偏移的距离。偏移量为壳厚度的一半，即 0.6。其余数据项是梁深度、底部和顶部翼缘宽度、底部和顶部翼缘厚度以及腹板厚度。

如果使用 SECTION=GENERAL 参数指定 [*BEAM GENERAL SECTION](../key/key-link.md#usb-kws-mbeamgensect) 选项，您可以给出质心和剪切中心的位置。[*SHEAR CENTER](../key/key-link.md#usb-kws-mshearcenter) 和 [*CENTROID](../key/key-link.md#usb-kws-mcentroid) 选项允许这些位置从节点偏移，使您能够容易地建模加劲件。例如，对于如图 [图 6-6](ch06s01.md#gss-ibeam) 所示连接到壳的 I 梁的输入为：

![](../graphics/gss-c6-tors-rigid-input-nls.png)

也可以分别定义梁节点和壳节点，并使用刚性梁约束连接两个节点。详见 ["Linear constraint equations," Abaqus Analysis User's Guide 第 35.2.1 节](../usb/usb-link.md#usb-cni-pequation)。
