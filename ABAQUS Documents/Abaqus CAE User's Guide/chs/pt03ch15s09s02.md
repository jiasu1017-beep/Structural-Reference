# 15.9.2 交互编辑器





要创建交互，从主菜单栏选择****Interaction**![](../graphics/images/arrow.gif)**Create****。出现**创建交互**对话框，你可以在其中提供交互名称、选择将创建交互的步骤以及选择交互类型。

当你在**创建交互**对话框中点击**继续**时（选择除通用接触外的任何交互类型后），系统会提示你选择要应用交互的区域。选择区域或区域后，出现交互编辑器，你可以在其中指定有关交互的其他信息，例如你要与交互关联的交互属性。对于通用接触交互，点击**创建交互**对话框中的**继续**时会显示交互编辑器。例如，[图15-15](pt03ch15s09s02.md#itn-contacteditor)显示了Abaqus/Explicit分析的通用接触编辑器。

**图15-15** 通用接触编辑器。

![](../graphics/itn-contacteditor-nls.png)

每个交互编辑器在对话框顶部面板显示当前步骤以及你正在定义的交互的名称和类型。编辑器其余部分的格式因你定义的交互类型而异。

创建交互后，可以通过以下方式修改交互：
- 你可以修改在创建交互时在编辑器中输入的部分或全部数据。
- 你可以使用**交互管理器**修改交互的逐步历史。（更多信息，请参见["什么是步骤相关管理器？"第3.4.2节](pt01ch03s04s02.md)。）

你可以通过从主菜单栏选择****Help**![](../graphics/images/arrow.gif)**On Context****然后点击感兴趣的编辑器功能来显示有关特定编辑器功能的信息。
