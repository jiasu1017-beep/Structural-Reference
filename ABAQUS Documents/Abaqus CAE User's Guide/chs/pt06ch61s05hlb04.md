# 61.5.4 创建和修改自定义工具栏





从主菜单栏选择 ****Tools**![](../graphics/images/arrow.gif)**Customize**** 以在 Abaqus/CAE 中编辑自定义工具栏。您可以创建新工具栏、更改现有自定义工具栏的内容、重命名自定义工具栏或删除自定义工具栏。

**创建或修改自定义工具栏的步骤：**

1. 在任何模块中，从主菜单栏选择 ****Tools**![](../graphics/images/arrow.gif)**Customize****。会出现 **Customize** 对话框。

2. 要创建新的自定义工具栏：
   1. 在 **Toolbars** 选项卡页面上，单击 **Create**。
   2. 在出现的对话框中，为新工具栏输入**名称**，然后单击 **OK**。Abaqus/CAE 在主窗口中创建一个空的浮动工具栏。

3. 显示 **Functions** 选项卡页面。

4. 从 **Module/Toolset** 列表中，选择您的所需功能所属的 Abaqus/CAE 模块或工具集。在自定义工具栏中，当从 **Module/Toolset** 列表中选择 **View Manipulation** 时，您应该使用 **Functions** 可用的渲染样式，而不是 Visualization Display Options 可用的渲染样式。Abaqus/CAE 使用所选模块或工具集中的可用功能填充 **Functions** 列表。

5. 在 **Functions** 列表中，找到您要添加到工具栏的功能。将功能单击并拖动到自定义工具栏上；光标变为加号符号 ![](../graphics/usi-pointer-toolbar-add.png)。释放鼠标按钮以将功能添加到工具栏。您可以将功能拖动到新的或现有的自定义工具栏。

6. 如果需要，为所选功能添加或修改图标分配。请参见["更改图标分配，" 第61.5.5节](pt06ch61s05hlb05.md)，了解详情。

7. 要从自定义工具栏移除功能，请单击工具栏上的图标并将其拖离工具栏；光标变为减号符号 ![](../graphics/usi-pointer-toolbar-del.png)。释放鼠标按钮以从工具栏中删除图标。
   **注意：**您可以随时使用此过程从自定义工具栏移除图标。不需要打开 **Customize** 对话框。

8. 重复步骤 4 到 7，直到工具栏包含所有所需的功能。

9. 单击 **Dismiss** 关闭 **Customize** 对话框。

**重命名自定义工具栏的步骤：**

1. 在任何模块中，从主菜单栏选择 ****Tools**![](../graphics/images/arrow.gif)**Customize****。会出现 **Customize** 对话框。

2. 显示 **Toolbars** 选项卡页面。

3. 选择要重命名工具栏的名称，然后单击 **Rename**。

4. 在出现的对话框中，输入工具栏的新名称，然后单击 **OK**。

5. 单击 **Dismiss** 关闭 **Customize** 对话框。

**删除自定义工具栏的步骤：**

1. 在任何模块中，从主菜单栏选择 ****Tools**![](../graphics/images/arrow.gif)**Customize****。会出现 **Customize** 对话框。

2. 显示 **Toolbars** 选项卡页面。

3. 选择要删除的工具栏的名称，然后单击 **Delete**。

4. 单击 **Dismiss** 关闭 **Customize** 对话框。
