# 5.8 小结







- 壳单元的截面行为可以使用穿过壳厚度的数值积分（[*SHELL SECTION](../key/key-link.md#usb-kws-mshellsection)）或在分析开始时计算的截面刚度（[*SHELL GENERAL SECTION](../key/key-link.md#usb-kws-mshellgensect)）来确定。
- [*SHELL GENERAL SECTION](../key/key-link.md#usb-kws-mshellgensect) 高效，但只能用于线性材料。[*SHELL SECTION](../key/key-link.md#usb-kws-mshellsection) 可用于线性和非线性材料。
- 数值积分在穿过壳厚度的一定数量的截面点处执行。这些截面点是可输出单元变量的位置。默认最外层截面点位于壳的表面上。
- 壳单元法线的方向确定元素的正负表面。要正确定义接触和解释单元输出，您必须知道哪个是哪个。壳法线还定义施加到元素上的正压力载荷的方向，可以在 Abaqus/Viewer 中绘制。
- 壳单元使用对每个单元局部的材料方向。在大位移分析中，局部材料轴随单元旋转。[*ORIENTATION](../key/key-link.md#usb-kws-morientation) 可用于定义非默认局部坐标系。单元变量（如应力和应变）在局部方向输出。
- [*TRANSFORM](../key/key-link.md#usb-kws-mtransform) 定义节点的局部坐标系。集中载荷和边界条件在局部坐标系中施加。所有打印的节点输出（如位移）也默认参考局部系统。
- 符号图可以帮助您可视化模拟结果。它们对于可视化结构的运动和载荷路径特别有用。
