# 16.9.2 定义力矩





您可以创建力矩载荷以定义顶点或节点处的旋转。

**创建或编辑力矩：**

1. 使用以下方法之一显示力矩载荷编辑器：- 若要创建新的力矩载荷，请遵循"创建载荷"，第16.8.1节中概述的过程（**类别**：`Mechanical`；**所选步骤的类型**：`Moment`）。- 若要使用菜单或管理器编辑现有的力矩载荷，请参见"编辑依赖于步骤的对象"，第3.4.13节。若要编辑应用载荷的区域，请参见"编辑应用规定条件的区域"，第16.8.4节。
2. 单击**分布**字段右侧的箭头，并从出现的列表中选择您的选项：- 选择**均匀**以定义在区域上均匀的载荷。- 选择分析场以定义空间变化载荷。只有对该载荷类型有效的分析场才会在选择列表中显示。或者，您可以单击![](../graphics/ico_analyticalField.png)创建新的分析场。（更多信息，请参见第58章"分析场工具集"。）
3. 在**CM1**、**CM2**和**CM3**文本字段中，输入关于每个轴的力矩分量（单位[FL](../popups/usb-int-iconventions-unitsym.md)）。如果您留空文本字段，则自动为该方向分配零力矩。但是，您必须在编辑器中至少输入一个非零分量来定义载荷。
4. 如果需要，单击**幅值**字段右侧的箭头，并从出现的列表中选择您选择的幅值。或者，您可以单击![](../graphics/ico_amplitudeSmall.png)创建新的幅值。（更多信息，请参见第57章"幅值工具集"。）
5. 如果需要，切换打开**跟随节点旋转**，以使载荷方向随该节点的旋转而旋转。**跟随节点旋转**仅影响具有旋转自由度且**Nlgeom**设置被打开的步骤中的节点。
6. 如果您要更改力矩的坐标系（**CSYS**），请单击![](../graphics/ico_selectBlue.png)并使用以下方法之一：- 在视口中选择现有的基准坐标系。- 按名称选择现有的基准坐标系。1. 从提示区域单击**基准坐标系列表**以显示基准坐标系列表。2. 从列表中选择一个名称，然后单击**确定**。**注意：**您不应在圆柱坐标系的原点处施加力矩载荷；这样做会使径向和切向载荷变得不确定。- 从提示区域单击**使用全局CSYS**以恢复到全局坐标系。此坐标系编辑选项仅在创建力矩的步骤中可用。默认情况下，使用全局坐标系来定义力矩。
7. 单击**确定**保存您的数据并退出编辑器。
