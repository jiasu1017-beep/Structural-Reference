# 55.12.7 查看已移除的元素







您可以创建模型更改交互，以停用模型的选定组件（几何、元素、外壳和肋条）。对于模型几何上的模型更改交互，可以从视口中移除外壳、几何和元素。对于孤网格上的模型更改交互，可以从视口中移除外壳、肋条和元素。您可以从 **ODB 显示选项** 对话框控制这些已移除组件的可见性。

视口的行为取决于可见性切换的状态和绘图状态。在模型更改交互中停用的元素可以从所有绘图状态的视口中移除。当关闭从视口中移除已停用元素的选项时，已停用元素在轮廓图中显示为灰色，但在所有其他绘图状态中保持不变。如果关闭显示梁截面的选项，Abaqus/CAE 将不会从视口中移除具有梁截面的已停用元素，无论已移除元素切换的状态如何。

**控制已停用元素显示：**

1. 从主菜单栏，选择 ****视图**![](../graphics/images/arrow.gif)**ODB 显示选项****。在出现的对话框中点击 **常规** 选项卡。**模型更改** 显示选项位于 **常规** 页面底部。
2. 切换 **考虑从视口中移除已停用元素**。
3. 点击 **确定** 实现您的更改并关闭对话框。
