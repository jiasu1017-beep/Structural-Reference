# 1.1.1 简介：概述






### Abaqus 有限元系统概述

Abaqus 有限元系统包括：
- Abaqus/Standard，一个通用有限元程序；
- Abaqus/Explicit，一个显式动力学有限元程序；
- Abaqus/CFD，一个通用计算流体动力学程序；
- Abaqus/CAE，一个交互式环境，用于创建有限元模型、提交 Abaqus 分析、监控和诊断作业以及评估结果；以及
- Abaqus/Viewer，Abaqus/CAE 的子集，仅包含可视化模块的后处理功能。

多个附加选项可用于进一步扩展 Abaqus/Standard 和 Abaqus/Explicit 的功能。Abaqus/Aqua 选项可与 Abaqus/Standard 和 Abaqus/Explicit 配合使用。Abaqus/Design 和 Abaqus/AMS 选项可与 Abaqus/Standard 配合使用。Abaqus/Aqua 包含专门为水下梁式结构分析而设计的可选功能，这些结构受到水流和波浪作用载荷。Abaqus/Design 选项使您能够执行设计灵敏度分析（DSA）。Abaqus/AMS 是一个可选的特征求解器，在 Abaqus/Standard 内工作，可快速求解大型对称特征值问题。Abaqus 协同仿真技术提供了多种应用，作为单独的附加功能，可用于 Abaqus 与第三方分析程序之间的耦合。Abaqus/Foundation 是 Abaqus/Standard 的可选子集，可更经济高效地访问 Abaqus/Standard 中的线性静力和动力分析功能。这些选项仅在您的许可证包含它们时才可用。

有关 Abaqus 产品、实用程序和附加选项的完整列表，请参阅 ["Abaqus 产品，" Abaqus 发布说明的第 1.2 节](../rnb/rnb-link.md#rnb-chp-int-products)。

### 本指南概述

本指南是使用 Abaqus/Standard（包括 Abaqus/Aqua、Abaqus/Design 和 Abaqus/Foundation）、Abaqus/Explicit（包括 Abaqus/Aqua）和 Abaqus/CFD 的参考。Abaqus/Standard 在每个求解"增量"中隐式求解方程组。相比之下，Abaqus/Explicit 以小时间增量向前推进解，而无需在每个增量中求解耦合方程组（甚至形成全局刚度矩阵）。Abaqus/CFD 提供了计算流体动力学能力，在 Abaqus/CAE 中全面支持前处理、仿真和后处理。

在整个指南中，术语"Abaqus"最常用作指代 Abaqus/Standard 和 Abaqus/Explicit 以及 Abaqus/CFD（适用时）的统称；使用各个产品名称来表示信息仅适用于该产品。产品标识符出现在指南中每个章节的开头（概述章节除外），指示章节中的信息适用于哪些产品。

本指南分为几个部分：
- [第一部分，"简介、空间建模与执行"](pt01.md)，讨论 Abaqus 中的基本建模概念，如定义节点、单元和表面；使用 Abaqus时应遵循的约定和输入格式；以及 Abaqus/Standard、Abaqus/Explicit、Abaqus/CFD、Abaqus/CAE 和 Abaqus 系统提供的几个实用程序的执行过程。
- [第二部分，"输出"](pt02.md)，描述如何从 Abaqus 获取输出以及结果（`.fil`）文件的格式。它还描述了可用的输出变量标识符。
- [第三部分，"分析过程、求解与控制"](pt03.md)，描述可用的分析类型（静应力分析、动力学、特征值提取等）。本章提供了 Abaqus/Standard 和 Abaqus/Explicit 如何求解有限元分析的详细讨论。
- [第四部分，"分析技术"](pt04.md)，讨论 Abaqus 中可用的各种分析技术，如子模型、移除单元或表面以及从先前仿真导入结果以定义当前模型的初始条件。
- [第五部分，"材料"](pt05.md)，描述材料建模选项以及如何校准一些更高级的材料模型。
- [第六部分，"单元"](pt06.md)，描述 Abaqus 中可用的单元。
- [第七部分，"指定条件"](pt07.md)，描述指定条件（如分布载荷和节点速度）的使用。
- [第八部分，"约束"](pt08.md)，讨论约束（如多点约束）的使用。
- [第九部分，"相互作用"](pt09.md)，讨论 Abaqus 中可用的接触和相互作用模型。

本指南还包括 Abaqus/Standard、Abaqus/Explicit 和 Abaqus/CFD 中所有可用输出变量和单元的[索引](pt10.md)。

### 使用 Abaqus

Abaqus/Standard、Abaqus/Explicit 和 Abaqus/CFD 可以作为批处理应用程序运行（详情请参阅 ["Abaqus/Standard、Abaqus/Explicit 和 Abaqus/CFD 执行，" 第 3.2.2 节"](pt01ch03s02abx02.md)），或通过交互式 Abaqus/CAE 环境运行（详情请参阅 ["Abaqus/CAE 执行，" 第 3.2.6 节"](pt01ch03s02abx06.md)）。Abaqus/Standard、Abaqus/Explicit 和 Abaqus/CFD 分析产品的主要输入是一个文件，其中包含仿真所需的各种选项以及与这些选项相关的数据。也可能有补充文件，如先前分析的重新启动或结果文件，或辅助数据文件，如包含动态分析的加速度记录或地震记录的文件。输入文件通常由 Abaqus/CAE 或其他预处理器创建。本指南提供了输入文件使用和 Abaqus/CAE 使用的信息。

如 ["在 Abaqus 中定义模型，" 第 1.3.1 节"](pt01ch01s03aus03.md) 中所述，主输入文件由两部分组成：模型输入和历史输入。输入围绕一些自然概念和约定进行组织，这意味着即使复杂仿真的输入文件可能很大，也可以毫不困难地管理它们。控制 Abaqus 输入文件的基本语法规则在 ["输入语法规则，" 第 1.2.1 节"](pt01ch01s02aus01.md) 中讨论。[Abaqus 关键字参考指南](../key/key-link.md#key) 包含 Abaqus/Standard、Abaqus/Explicit 和 Abaqus/CFD 中所有可用输入选项的完整描述。有关使用 Abaqus 进行分析的详细入门，建议您遵循 [Abaqus 入门：交互版](../gsa/gsa-link.md#gsa) 中的自学教程。有关使用 Abaqus/CAE 的详细信息，请参阅 [Abaqus/CAE 用户指南](../usi/usi-link.md#usi)。

此外，[Abaqus 示例问题指南](../exa/exa-link.md#exa)、[Abaqus 基准指南](../bmk/bmk-link.md#bmk) 和 [Abaqus 验证指南](../ver/ver-link.md#ver) 中讨论了许多展示 Abaqus 众多功能的分析。作为 [Abaqus 分析用户指南](book01.md) 的补充，这些示例可以帮助您熟悉 Abaqus 提供的功能以及 Abaqus 输入文件的结构。例如，[Abaqus 验证指南中的"梁撞击圆柱体，" 第 1.6.12 节](../ver/ver-link.md#ver-elm-beamimpact) 讨论了可用于分析悬臂梁动态响应的各种建模技术。

### 审查 Abaqus 仿真结果

有关从 Abaqus 仿真请求输出的信息在 ["输出，" 第 4.1.1 节"](pt02ch04s01aus38.md) 中讨论。通过 Abaqus/CAE 中的可视化模块（也作为 Abaqus/Viewer 单独许可）查看 Abaqus 仿真的请求结果。输出数据库文件由 Abaqus/CAE 中的可视化模块读取，以创建 Abaqus 结果的云图、动画、*X–Y* 图表和表格输出。有关使用 Abaqus/CAE 中可视化模块的详细信息，请参阅 [第五部分，"查看结果，" Abaqus/CAE 用户指南](../usi/usi-link.md#usi-vismod)。
