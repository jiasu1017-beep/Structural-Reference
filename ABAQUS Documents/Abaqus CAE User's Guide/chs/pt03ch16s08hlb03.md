# 16.8.3 创建预定义场







创建预定义场时，必须指定场的名称、激活场的步骤、场的类型以及要应用场的装配件区域。

**注意：**创建温度场的过程单独说明；请参见"定义温度场"，第16.11.9节。

**创建预定义场：**

1. 从主菜单栏中，选择****预定义场**![](../graphics/images/arrow.gif)**创建****。将出现**创建预定义场**对话框，**名称**文本字段中显示默认名称。**提示：**您也可以使用载荷模块工具箱中的![](../graphics/ico_predefinedField.png)工具创建预定义场。
2. 为预定义场键入名称。有关命名对象的信息，请参见"使用基本对话框组件"，第3.2.1节。
3. 选择激活预定义场的步骤。单击**步骤**文本字段旁边的箭头，然后从出现的列表中选择。
4. 从对话框左侧的**类别**列表中，选择所需的类别。可用的**类别**选择取决于您正在执行的分析程序类型。对话框右侧的**所选步骤的类型**列表更改为所有可用预定义场类型的列表。
5. 从**所选步骤的类型**列表中，选择预定义场类型，然后单击**继续**。
6. 如果模型包含网格和几何的组合，您必须选择要应用预定义场的区域类型。从提示区域中选择以下之一：- 单击**几何**将预定义场应用于几何或参考点。- 单击**网格**将预定义场应用于本机或孤立网格选择。
7. 如果您要创建的预定义场不属于**流体**类别，或者您要将流体类型预定义场应用于区域而不是整个模型，请选择要应用预定义场的区域。如果您要创建材料分配场或初始状态场，请在视口中使用鼠标选择一个部件实例。对于所有其他预定义场，使用以下方法之一选择区域：- 使用鼠标在视口中选择一个区域。您可以使用角度方法从几何中选择一组面或边，或从网格中选择一组单元面或节点。更多信息，请参见"使用角度和特征边缘方法选择多个对象"，第6.2.3节。完成选择后，单击鼠标按钮2。**提示：**您可以通过在**选择**工具栏中指定过滤选项来限制可以在视口中选择的对象类型。更多信息，请参见"使用选择选项"，第6.3节。默认情况下，会创建包含所选对象的集合。您可以通过关闭提示区域中创建集合的选项来更改此行为。提示区域中提供了默认名称，但您可以输入新名称。- 要从现有集合列表中进行选择，请执行以下操作：1. 单击提示区域右侧的**集合**。Abaqus/CAE显示**区域选择**对话框，其中包含可用集合的列表。2. 选择感兴趣的集合，然后单击**继续**。**注意：**默认选择方法基于您最近使用的选择方法。要恢复为另一种方法，请单击提示区域右侧的**在视口中选择**或**集合**。将出现预定义场编辑器。您正在应用预定义场的区域在视口中高亮显示。
8. 输入定义预定义场所需的所有数据，然后单击**确定**。有关编辑器特定功能的详细信息，请从主菜单栏中选择****帮助**![](../graphics/images/arrow.gif)**关于上下文****，然后单击感兴趣的功能，或参见"使用预定义场编辑器"，第16.11节。符号出现在视口中，代表您刚创建的预定义场。更多信息，请参见"理解代表规定条件的符号"，第16.5节。
