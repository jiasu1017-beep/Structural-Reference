# 74.1 理解流显示







流线沿着与节点矢量场相切的方向延伸，流是一组流线，使您能够可视化从矢量场中特定位置发出的数据。[图74-1](pt06ch74s01.md#usi-str-intro-example)显示了一个包含12条流线的示例流，显示了流体流动分析中的速度数据。

**图74-1** 12条流线显示通过歧管的流动分析的速度数据。

![](../graphics/rnb610ef-out-stream.png)

您可以通过首先定义一个耙来创建流，耙是一条线段，沿其长度有一系列点，用于控制Abaqus/CAE显示流体流动分析的流线的位置。耙被放置在流体流动的关注区域中；当您在视口中创建和显示流时，Abaqus/CAE显示与耙上点数相对应数量的流线。除了使用非平凡路径定义定义的耙外，流线沿耙均匀分布；有关流耙定义的更多信息，请参见["创建流," 第74.2节](pt06ch74s02.md)。

流线形状由当前选择的流变量驱动，通常是速度或涡量。要更改当前流变量，请参见["选择流场输出变量," 第42.5.7节](pt05ch42s03hlb07.md)。流线颜色由当前选择的主变量驱动。自定义流线颜色选项的更多信息，请参见["自定义流颜色," 第74.4.1节](pt06ch74s04s01.md)。

您必须为每个要显示流动数据的部件实例定义单独的流耙。Abaqus/CAE仅计算单个部件实例的流数据，不会在不同部件实例之间显示流线。
