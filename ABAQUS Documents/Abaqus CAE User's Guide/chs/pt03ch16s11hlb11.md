# 16.11.11 定义初始状态场





您可以从模型中选择部件实例，并将初始状态场与这些实例关联。初始状态场使用从先前的Abaqus/Standard或Abaqus/Explicit分析导入的数据来定义变形网格及其相关的材料状态。您只能在初始步骤期间创建初始状态场。有关更多信息，请参见["在Abaqus分析之间传输结果"章节第16.6节](pt03ch16s06.md)和["在Abaqus分析之间传输结果：概述"，《Abaqus Analysis User's Guide》第9.2.1节](../usb/usb-link.md#usb-anl-atransferoverview)。

**创建或编辑初始状态场的步骤：**

1. 使用以下方法之一显示初始状态场编辑器： - 要创建新的初始场，请按照["创建预定义场"章节第16.8.3节](pt03ch16s08hlb03.md)中概述的程序进行操作（**类别**：`其他`；**所选步骤的类型**：`初始状态`）。您只能在初始步骤期间创建初始状态场。 - 要使用菜单或管理器编辑现有的初始状态场，请参见["编辑依赖于步骤的对象"章节第3.4.13节](pt01ch03s04hlb05.md)。您只能在初始步骤期间修改初始状态场。
2. 在编辑器的**作业名称**字段中，输入Abaqus/CAE将从中导入变形网格和材料状态的作业名称。Abaqus/CAE从原始分析创建的多个文件导入数据。因此，分析的文件必须位于您启动当前Abaqus/CAE会话的目录中。
3. 选择要从中导入的步骤。请执行以下操作之一： - 选择**最后步骤**从先前分析的最后步骤导入数据。 - 选择**指定**，并输入一个整数，指定要从先前分析导入数据的步骤。值为1指定第一个步骤。
4. 选择要从中导入的帧。请执行以下操作之一： - 选择**最后**从指定步骤的最后帧导入数据。 - 选择**指定**，并输入一个整数，指定指定步骤的帧。值为1指定第一帧。
5. 默认情况下，Abaqus/CAE不使用导入的数据来更新参考配置。因此，位移和应变是相对于分析开始时的参考配置计算的 total values，并将从分析开始时连续。开启**更新参考配置**可将参考配置重置为导入的配置。位移和应变现在相对于新的导入配置计算。
6. 单击**确定**创建初始状态场并关闭对话框。黄色圆圈出现在模型上，表示您刚创建的初始场。（有关更多信息，请参见["理解表示规定条件的符号"章节第16.5节](pt03ch16s05.md)。）虽然Abaqus/CAE在您创建初始状态场时会检查先前分析的文件是否存在，但它只在您在作业模块中分析模型时才从指定的步骤和增量导入变形网格和材料状态。因此，创建初始状态场后，您不会在当前模型中看到任何其他更改；例如，Abaqus/CAE不会更新当前模型的网格。
