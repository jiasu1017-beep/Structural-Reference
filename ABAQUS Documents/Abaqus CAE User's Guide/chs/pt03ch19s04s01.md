# 19.4.1 什么是协同执行？





协同执行是两个分析作业的协同仿真执行，这两个作业在 Abaqus/CAE 中相互同步执行，使用与[「理解分析作业，」第 19.2 节](pt03ch19s02.md)中所述相同的功能。有关此分析技术的更多信息，请参见[「结构到结构协同仿真，」Abaqus Analysis User's Guide 第 17.3.1 节](../usb/usb-link.md#usb-anl-acosimabqtoabq)和[「流体到结构和共轭热传递协同仿真，」Abaqus Analysis User's Guide 第 17.3.2 节](../usb/usb-link.md#usb-anl-acosimcfdtoabq)。
