# 3.2.12 使用键盘快捷方式





您可以使用键盘而不是鼠标来执行Abaqus/CAE主窗口和对话框中的大多数操作。以下操作具有键盘快捷方式：

**上下文相关帮助**

按**[F1]**显示有关Abaqus/CAE主窗口或对话框中当前选中对象的上下文相关帮助。有关使用**[F1]**获取上下文相关帮助的更多信息，请参阅["显示上下文相关帮助，" 第2.6.1节"](pt01ch02s06s01.md)。

**菜单**

您可以通过按**[Alt]**键并结合该菜单名称中带下划线的字符来显示特定菜单。例如，主菜单栏上**视图**菜单中的字母**V**带下划线：

![](../graphics/int-keyfile-nls.png)

因此，您可以输入**[Alt]****+V**显示**视图**菜单。

**菜单项**

显示菜单后，您可以继续按**[Alt]**键并按该菜单项名称中带下划线的字符来选择特定菜单项。例如，**视图**菜单中的**平移**字母**n**带下划线：

![](../graphics/int-keynew-nls.png)

因此，您可以输入**[Alt]****+V**显示**视图**菜单，然后不释放**[Alt]**键，输入**n**选择**平移**。

**模型树和结果树**

模型树和结果树包含允许您在树中导航以及打开和关闭其显示的键盘快捷方式。有关更多信息，请参阅["模型树概述，" 第3.5.1节"](pt01ch03s05s01.md)。
