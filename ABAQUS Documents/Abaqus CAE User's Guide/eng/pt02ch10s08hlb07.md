# 10.9.7 Exporting model data to Nastran bulk data file format







Select ****Job**![](../graphics/images/arrow.gif)**Export**![](../graphics/images/arrow.gif)**Nastran Input File**![](../graphics/images/arrow.gif)***jobname***** from the main menu bar in the Job module to export the model data and job data associated with an Abaqus job to Nastran bulk data file (`.bdf`) format. 

**Tip:**You can also initiate this export from the Model Tree. Expand the **Jobs** container, click mouse button 3 on the job, and select ****Export**![](../graphics/images/arrow.gif)**Nastran Input File****.

The Nastran bulk data file you create includes selected data from the Abaqus model and job; see ["Translating Abaqus files to Nastran bulk data files," Section 3.2.29 of the Abaqus Analysis User's Guide](../usb/usb-link.md#usb-int-dtonasabaproc), for more information about the supported Abaqus keywords and the Nastran entities to which they map. The bulk data file includes selected history data in the Abaqus model, but it does not include parts and assemblies, which are not supported in Nastran.![](../graphics/images/black4rule.gif)For information on related topics, click any of the following items:- ["Using file selection dialog boxes," Section 3.2.10](pt01ch03s02s10.md)
