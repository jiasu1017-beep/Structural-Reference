# 77.10 编辑自动颜色列表中的颜色





创建颜色映射时，Abaqus/CAE通过引用**自动颜色列表**中的颜色定义来为对象分配颜色。您可以通过添加、移除、重新排列和更改颜色来修改此列表的内容。

与颜色映射不同，自动颜色列表对所有视口通用。

**编辑自动颜色列表中的颜色的步骤：**

1. 点击**颜色编码**工具栏中的![](../graphics/ico_colorCode.png)工具。Abaqus/CAE显示**颜色编码**对话框。
2. 点击**高级**，然后从出现的列表中选择**编辑自动颜色**。Abaqus/CAE显示**编辑自动颜色**对话框。
3. 要在自动颜色列表中插入新颜色：
   1. 高亮显示您要添加颜色之前或之后的自动颜色列表中的颜色。
   2. 点击**插入之前**或**插入之后**可在自动颜色列表中指示的位置添加新颜色。**选择颜色**对话框打开。
   3. 选择一种颜色，然后点击**确定**关闭**选择颜色**对话框。Abaqus/CAE会在所选位置添加新颜色。
4. 要更改自动颜色列表中的一种颜色，请双击该颜色并从出现的**选择颜色**对话框中选择新颜色。
5. 要在自动颜色列表中移动颜色，请高亮显示该颜色，然后点击**上移**或**下移**可沿所选方向移动一个位置。
6. 要从自动颜色列表中删除颜色，请高亮显示该颜色并点击**删除**。
7. 继续添加、更改、移动和删除颜色，直到自动颜色列表包含您所需顺序的颜色。
8. 点击**确定**。Abaqus/CAE会将修订后的自动颜色列表用于任何后续颜色编码。
