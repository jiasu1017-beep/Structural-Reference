# 19.1 理解Job模块的作用





完成定义模型所涉及的所有任务（如定义模型几何形状、分配截面属性和定义接触）后，您可以使用Job模块分析模型。 Job模块允许您创建作业、提交作业进行分析，并监控其进度。 如果需要，您可以创建多个模型和作业，同时运行和监控作业。

此外，您可以选择仅为模型生成分析输入文件。 此选项允许您在提交分析之前查看和编辑输入文件。 对于Abaqus/Standard或Abaqus/Explicit分析，您还可以通过从主菜单栏选择 ****Model**![](../graphics/images/arrow.gif)**Edit Keywords**![](../graphics/images/arrow.gif)***model name***** 来查看和编辑模型的分析关键字。

您可以基于现有输入文件而不是Abaqus/CAE模型创建和提交作业；例如，如果您在Abaqus/CAE之外创建了输入文件，但希望在Abaqus/CAE中运行分析、监控其进度并查看结果。 只能为不包含对其他结果文件引用的输入文件创建作业；例如，您无法基于包含重启分析的输入文件创建作业。

如果您在Mesh模块中定义了自适应重网格规则，可以提交网格自适应过程。 Abaqus/CAE提交一系列作业，每个作业都尝试在前一个作业的基础上提高解精度并减少误差指标。

对于Abaqus协同仿真，您可以创建协同执行来同步执行两个分析作业。
