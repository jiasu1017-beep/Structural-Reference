# 54.3.1 选择 X–Y 数据对象





### 54.3.1 选择 X–Y 数据对象

您可以选择一个或多个 X–Y 数据对象包含在 X–Y 报告中。

**选择 X–Y 报告数据：**

1. 找到 X–Y 报告 **XY 数据（XY Data）** 选项。从主菜单栏中，选择 ****报告（Report）**![](../graphics/images/arrow.gif)**XY****；然后在出现的对话框中点击 **XY 数据（XY Data）** 选项卡。
2. 从可用数据对象列表中选择要包含在报告中的 X–Y 数据对象。默认情况下，列出了所有 [先前保存的](pt05ch47s02hlb07.md) X–Y 数据对象。选择以下选项之一来调整此列表的内容：- 点击 **所有 XY 数据（All XY data）** 浏览所有先前保存的 X–Y 数据对象。- 点击 **当前视口中的 XY 绘图（XY plot in current viewport）** 仅浏览当前视口中 X–Y 数据绘图中包含的那些 X–Y 数据对象。Abaqus/CAE 列出您请求的 X–Y 数据对象。
3. 要限制列表以便于选择，请在 **名称过滤器（Name filter）** 字段中输入过滤器模式，然后按 **[Enter]** 键应用过滤器。点击文本字段右侧的 ![](../graphics/ico_tipSmall.png) 显示有关如何构造过滤器的信息。Abaqus/CAE 仅列出满足您规范的 X–Y 数据对象。
4. 从 X–Y 数据对象名称和描述列表中，选择一个或多个要包含在报告中的数据对象。有关在对话框中选择多个项目的更多信息，请参阅 ["从列表和表格中选择多个项目，" 第 3.2.11 节"](pt01ch03s02s11.md)。

要生成报告，请在 **设置（Setup）** 页面上配置任何所需的自定义选项，**报告 XY 数据（Report XY Data）** 对话框中也有此页面；然后点击 **应用（Apply）**。完成后，点击 **取消（Cancel）** 关闭对话框。

# 54.1 生成表格报告





要生成 X–Y 数据对象、场输出结果或自由体切割的表格报告，您首先指定报告内容、格式和目的地。当您应用这些自定义选择时，Abaqus/CAE 会生成报告。

要生成探针值报告，您首先生成然后探测模型或 X–Y 绘图。在探测时，您可以选择感兴趣的值，并将选定的值写入文件。

根据您的选择，您的报告可能包括负单元标签。负标签表示由 Abaqus/CAE 创建的内部单元。这些单元被赋予负标签，以便您可以轻松地将它们与在定义模型时创建的单元区分开来。

**生成 X–Y 数据、场输出结果或自由体切割的报告：**

1. 从主菜单栏中，选择 ****报告（Report）**![](../graphics/images/arrow.gif)**XY****、****报告（Report）**![](../graphics/images/arrow.gif)**场输出（Field Output）**** 或 ****报告（Report）**![](../graphics/images/arrow.gif)**自由体切割（Free Body Cut）****。**报告 XY 数据（Report XY Data）**、**报告场输出（Report Field Output）** 或 **报告自由体切割（Report Free Body Cut）** 对话框变得可用。
2. 使用选项自定义报告内容、格式和目的地。
3. 完成后，点击 **应用（Apply）** 生成报告。

**生成探针值的报告：**

1. 从主菜单栏中，选择 ****工具（Tools）**![](../graphics/images/arrow.gif)**查询（Query）****，然后在出现的对话框中点击 **探针值（Probe values）**。出现 **探针值（Probe Values）** 对话框。
2. 移动光标以探测当前视口中的模型或 X–Y 绘图。
3. 要在 **探针值（Probe Values）** 对话框底部的数据表中选择和累积感兴趣的值，请在探测时点击鼠标按钮 1。
4. 完成后，点击 **写入文件（Write to File）** 访问报告文件选项并生成报告。





# 54.3.2 选择场输出变量





您可以选择一个或多个场输出变量包含在表格报告中。可用的变量包括您为当前步骤和帧保存到输出数据库的变量；此步骤和帧显示在 **报告场输出（Report Field Output）** 对话框顶部。

Abaqus/CAE 可以计算和报告在各种位置上给定变量的值。可能的报告位置包括：积分点、形心、单元节点和唯一节点。

单元节点和唯一节点位置都涉及在模型节点上报告结果；但是，唯一节点值的报告在每个节点处仅产生单个值，而单元节点值的报告在每个对该节点有贡献的单元处产生一个值。仅对于基于节点的变量（如位移）才提供唯一节点值。对于基于单元的变量（如应力），必须进行外推和平均以获得唯一节点值。平均阈值的使用以及跨区域和材料边界的平均控制将影响基于单元结果报告的单元节点值（有关更多信息，请参阅 ["选择结果选项，" 第 42.6 节"](pt05ch42s04.md)）。下表总结了在变量保存到输出数据库时使用的位置不同的情况下，哪些输出位置可以报告：

|  | 报告位置 |
| --- | --- |
| 输出数据库位置 | 积分点 | 形心 | 单元节点 | 唯一节点 |
| 积分点 | 是 | 是 | 是 | 是 |
| 形心 | 否 | 是 | 是 | 是 |
| 单元节点 | 否 | 否 | 是 | 是 |
| 唯一节点 | 否 | 否 | 否 | 是 |

接触表面变量（如 CNORMF、CSHEARF、CPRESS、COPEN、CSLIP、CSHEAR1 和 CSHEAR2）只能在单元节点位置报告。

有关选择报告输出变量的截面点位置的信息，请参阅 ["选择场输出截面点，" 第 54.3.3 节"](pt05ch54s03hlb03.md)。

**选择场输出报告数据：**

1. 找到场输出报告变量的选项。从主菜单栏中，选择 ****报告（Report）**![](../graphics/images/arrow.gif)**场输出（Field Output）****；然后在出现的对话框中点击 **变量（Variable）** 选项卡。
2. 点击 **位置（Position）** 箭头显示列出变量的可能位置；然后选择所需的位置。仅当在相同位置报告时，多个变量才能 [合并在单个表格中](pt05ch54hla02.md)。变量列表会刷新以仅显示可以在所选位置报告的变量。
3. 使用列表中每个变量旁边的复选框（此方法是默认方法）或页面底部的 **编辑（Edit）** 文本字段来选择要包含在报告中的场输出变量。**注意：**如果您想指定报告值的单个截面点，必须使用复选框方法。**要使用复选框方法：** 1. 要选择变量及其所有分量，请点击该变量的复选框。2. 要在变量的各个分量中进行选择，请点击该变量复选框旁边的箭头列出其分量；然后点击各个分量复选框来选择它们。**要使用编辑方法：** 在 **编辑（Edit）** 文本字段中，输入要包含在报告中的变量和分量的名称。要有效，变量必须在输出数据库中可用且可以在当前位置报告。

要生成报告，请在 **设置（Setup）** 页面上配置剩余选项，**报告场输出（Report Field Output）** 对话框中也有此页面；然后点击 **应用（Apply）**。完成后，点击 **取消（Cancel）** 关闭对话框。
