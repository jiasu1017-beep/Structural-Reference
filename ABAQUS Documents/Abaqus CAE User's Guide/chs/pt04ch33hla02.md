# 33.4 定义非结构质量





您可以为部件或装配上的区域定义非结构质量。更多信息，请参见["非结构质量定义"，Abaqus Analysis User's Guide第2.7.1节](../usb/usb-link.md#usb-int-anonstructuralmass)。

**要定义非结构质量：**

1. 从Property模块或Interaction模块的主菜单栏中，选择****Special**![](../graphics/images/arrow.gif)**Inertia**![](../graphics/images/arrow.gif)**Create****。
2. 在出现的**Create Inertia**对话框中，命名惯性，选择**Nonstructural mass**，然后单击**Continue**。
3. 使用以下方法之一选择要定义非结构质量的区域：
   - 在视口中选择区域。完成选择后，单击鼠标按钮2。如果模型包含几何体和网格组件的组合，请从提示区域选择以下之一：
     - 选择**Geometry**以为部件或装配的几何部分或参考点定义非结构质量。
     - 选择**Mesh**以为孤立网格组件定义非结构质量。您可以使用角度方法从孤立网格中选择一组节点。更多信息，请参见["使用角度和特征边缘方法选择多个对象"，第6.2.3节](pt01ch06s02hlb03.md)。
   - 要从现有集列表中选择，请执行以下操作：
     1. 在提示区域右侧单击**Sets**以显示包含可用集列表的**Region Selection**对话框。
     2. 选择感兴趣的集，然后单击**Continue**。
   **注意：**默认选择方法基于您最近使用的选择方法。要切换到另一种方法，请在提示区域右侧单击**Select in Viewport**或**Sets**。
   出现**Edit Inertia**对话框。视口中突出显示了您要应用非结构质量的区域。
4. 选择用于指定非结构质量大小的**Units**。可用类型取决于您选择的区域类型。
   - 选择**Total Mass**以总质量值的形式指定大小。
   - 选择**Mass per Volume**以单位体积质量的形式指定大小。
   - 选择**Mass per Area**以单位面积质量的形式指定大小。
   - 选择**Mass per Length**以单位长度质量的形式指定大小。
5. 在上一步中选择的单位对应的**Magnitude**字段中输入非结构质量大小。
6. 如果以总质量值的形式指定质量并且区域是集，则可以选择**Distribution**方法以在区域上分配质量。
   - 选择**Mass Proportional**以根据结构质量按比例将总质量分配到集的成员。底层结构密度在区域上均匀缩放；因此，质心不变。
   - 选择**Volume Proportional**以根据初始配置中的体积按比例将总质量分配到集的成员。底层结构密度在区域上添加均匀值；因此，如果区域具有非均匀结构密度，则质心可能会改变。
7. 单击**OK**保存数据并关闭对话框。符号出现在视口中，代表您刚创建的非结构质量。
