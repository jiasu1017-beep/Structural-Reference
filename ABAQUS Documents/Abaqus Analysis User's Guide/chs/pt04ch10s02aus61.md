# 12.4.2 节点-based 子模型

**产品：** Abaqus/Standard 和 Abaqus/Explicit

##### **参考资料**

- ["子模型技术概述，" 第 12.4.1 节](pt04ch10s02aus60.md)
- ["Surface-based 子模型，" 第 12.4.3 节](pt04ch10s02aus62.md)
- [*SUBMODEL](../key/key-link.md#usb-kws-hsubmodel)

### 概述

节点-based 子模型从全局分析结果中获取驱动点的位移，并将其作为边界条件应用到子模型上。这种方法适用于大多数静力和动力分析。

### 定义节点-based 子模型

| **输入文件用法：** | ``` [*SUBMODEL](../key/key-link.md#usb-kws-hsubmodel), STEP=*step*, INC=*increment* ``` |

### 驱动节点

驱动节点是子模型边界上从全局模型获取位移的节点：

| **输入文件用法：** | ``` [*BOUNDARY](../key/key-link.md#usb-kws-hboundary), SUBMODEL, OP=NEW *node_set*, *dof* ``` |

### 驱动方法

有两种方法从全局模型获取边界条件：

| **方法** | **描述** |
| --- | --- |
| 节点集 | 使用节点集定义驱动点 |
| 坐标 | 使用坐标定义驱动点 |

| **输入文件用法：** | ``` [*SUBMODEL](../key/key-link.md#usb-kks-hsubmodel), STEP=*step*, INC=*increment*, DRIVING REGION ``` |

### 数据传递

从全局分析传递到子模型的数据：

| **数据类型** | **传递内容** |
| --- | --- |
| 位移 | 三个平动自由度 |
| 旋转 | 三个转动自由度（如果适用） |
| 温度 | 温度自由度（如果适用） |

### 时间历史

对于瞬态分析，子模型可以获取完整的时间历史：

| **输入文件用法：** | ``` [*SUBMODEL](../key/key-link.md#usb-kws-hsubmodel), STEP=*step*, TIME=TOTAL TIME ``` |

### 幅值曲线

可以应用幅值曲线来缩放或修改全局结果：

| **输入文件用法：** | ``` [*SUBMODEL](../key/key-link.md#usb-kws-hsubmodel), STEP=*step*, AMPLITUDE=*amplitude* ``` |

### 动力分析

对于动力分析，子模型可以获取：

| **数据类型** | **描述** |
| --- | --- |
| 位移 | 静态或准静态分量 |
| 速度 | 速度场 |
| 加速度 | 加速度场 |

| **输入文件用法：** | ``` [*SUBMODEL](../key/key-link.md#usb-kws-hsubmodel), STEP=*step*, VELOCITY, ACCELERATION ``` |

### 重启动

子模型可以与重启动分析结合：

| **输入文件用法：** | ``` [*SUBMODEL](../key/key-link.md#usb-kws-hsubmodel), STEP=*step*, INC=*increment*, READ RESTART ``` |

### 输出

子模型分析的输出包括：

- 完整的节点位移和应力
- 单元应变和应力
- 界面反力
- 驱动点的位移（用于验证）

### 验证

验证子模型边界处与全局模型的一致性：

| **输出请求** | **目的** |
| --- | --- |
| 边界位移 | 确认与全局模型匹配 |
| 应力分布 | 检查应力平滑过渡 |
| 能量平衡 | 确认模型正确连接 |

### 局限性

- 驱动点必须在全局模型中有结果
- 单元类型必须兼容
- 接触定义必须在子模型中重新定义
- 动力分析需要正确的时间匹配

### 输入文件模板

```
*-heading
Global model analysis
**
*STEP
*STATIC
*BOUNDARY
*LOAD
*OUTPUT
*NODE FILE
U
*END STEP
**
*-heading
Node-based submodel
**
*SUBMODEL, STEP=1, INC=50, GLOBAL STEP=1, GLOBAL INC=50
*BOUNDARY, SUBMODEL
NDSET, 3, 3, 1.0
NDSET, 2, 2, 1.0
NDSET, 1, 1, 1.0
*STEP
*STATIC
*BOUNDARY, SUBMODEL
NDSET, 1, 3, 0.0
*CLOAD
NDSET, 3, 1000.0
*OUTPUT
*ELEMENT OUTPUT
*END STEP
```
