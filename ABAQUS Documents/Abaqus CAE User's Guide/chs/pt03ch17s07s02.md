# 17.7.2 保持节点坐标的精度





当您在"零件"模块中创建零件时，它存在于自己的坐标系中，独立于模型中的其他零件。相比之下，当您在"装配"模块中创建零件实例并将其相对于其他零件实例定位时，您是在装配的全局坐标系中工作的。

为保持精度，网格模块将零件实例的定位信息与实例的几何形状分开。因此，当您生成网格时，零件实例的节点坐标是相对于原始零件的坐标系计算的。（当"作业"模块生成输入文件时，Abaqus/CAE会将每个实例的节点坐标写入其自己的坐标系，并通过[*INSTANCE](../key/key-link.md#usb-kws-minstance)关键字将实例定位和方向信息传递给分析产品。）

网格模块以单精度存储这些节点坐标。如果零件几何形状远离其坐标系的原点，则会丢失某些节点坐标的精度。为防止这种精度损失，您应该尝试将零件放置在其坐标系原点附近。例如，Abaqus/CAE原生零件的坐标系位于定义基础特征的草图的原点。因此，如果可能，您应该将基础特征的草图放置在草图器网格的原点上。
