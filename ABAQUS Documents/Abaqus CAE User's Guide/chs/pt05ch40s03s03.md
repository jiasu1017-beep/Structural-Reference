# 40.3.3 自定义您的绘图





可视化模块提供了大量的自定义选项，可通过主菜单栏的**视口**、**选项**和**视图**菜单访问。这些选项分为三类：

**依赖于绘图状态的选项**

绘图类型选项仅影响特定的绘图状态。这些是影响云图、符号和材料方向绘图的独立选项；X-Y曲线；X-Y图形；时间历史动画；比例因子动画；以及谐波动画。

**不依赖于绘图状态的选项**

不依赖于绘图状态的选项是那些集体影响所有绘图的选项。这些是控制视点、图形、单个项目着色、显示体外观的通用选项，以及如图例、模型标签和给出模型标题和状态的文本块外观等一般特征。

**叠加选项**

叠加选项是一组特殊的、不依赖于绘图状态的选项，仅在您选择将未变形绘图叠加到变形、云图、符号或材料方向绘图上时影响未变形绘图状态。这些选项允许您控制许多常见的自定义选项，以便在两种形状都显示时将未变形绘图与变形绘图区分开来。

依赖于绘图状态的选项影响诸如云图间隔、限制和颜色等绘图属性（例如，云图的色谱或材料方向的轴颜色）。您可以使用与该状态关联的选项为每个绘图状态单独控制这些属性。例如，要选择云图类型，必须使用**云图**绘图选项。为此，您可以从主菜单栏中选择**选项**![](../graphics/images/arrow.gif)**云图**。或者，您可以使用工具箱中的**云图选项**工具![](../graphics/ico_plotContourOptions.png)。**选项**工具提供了对依赖于绘图状态和不依赖于绘图状态的自定义选项的快速访问。

依赖于绘图状态的选项仅影响关联状态中的绘图。如果您从材料方向绘图选项对话框框中选择轴颜色，这些颜色将仅影响材料方向绘图。如果当前视口中存在材料方向绘图，则当您在材料方向绘图选项对话框中单击**应用**或**确定**时会看到更改的效果。但是，如果当前视口不包含材料方向绘图，则在您生成一个之前不会看到更改的效果。

不依赖于绘图状态的选项会影响所有绘图状态下的绘图。例如，如果您从主菜单栏中选择**视口**![](../graphics/images/arrow.gif)**视口注释选项**来抑制视口三轴的出现，则视口三轴将在所有绘图中被抑制。**通用绘图选项**对话框中的设置也是不依赖于绘图状态的；当您在此对话框中更改渲染样式时，新渲染样式将用于所有绘图。

叠加选项会影响当您在某种绘图状态中将未变形绘图叠加到变形绘图上时的未变形绘图。例如，您可以独立设置渲染样式、边缘显示和填充颜色；当未变形和变形形状符号绘图都显示时，您可以在它们之间应用偏移。

选择**文件**![](../graphics/images/arrow.gif)**保存选项**可以从主菜单栏保存您的依赖于绘图状态的、不依赖于绘图状态的和叠加自定义选项。保存您的自定义选项允许您将它们应用到后续的Abaqus/CAE会话中。更多信息，请参见"保存您的显示选项设置"，第76.16节。有关可视化模块中可用的绘图自定义选项的更多信息，请参见第55章，"自定义绘图显示"。
