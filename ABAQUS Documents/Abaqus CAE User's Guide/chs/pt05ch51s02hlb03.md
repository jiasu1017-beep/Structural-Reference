# 51.2.3 探测单元





您可以使用查询工具集来获取单元数据以及所选模型中的属性值或从所选模型图获取分析结果。当您在当前视口中的模型或模型图上将光标置于单元上时，Abaqus/CAE会高亮该单元。您可以选择在数据表中显示以下关于该单元的信息：
- 单元的标签，
- 单元的连接性，以及
- 当前场输出变量（包括分量值，如果当前场输出变量是张量或向量）。

如果当前视口中选择了模型，当前场输出变量反映模型属性（如载荷）在所选单元处的值。

您还可以通过直接输入其单元标签来获取单元的数据，并且可以获取所选显示组中包含的所有单元的数据。这些方法使您无需将光标放在模型图中其位置即可探测单元数据。

仅外部单元可以被查询。如有必要，您可以创建显示组或使用视图切割来显示原本不可用的内部单元。更多信息，请参阅[第78章，"使用显示组显示模型的子集"](pt07ch78.md)。"

当您将单元添加到数据表时，您可以切换该行的复选框以在视口中显示该单元的探测注释，或者您可以切换标题行中的复选框以显示所有所选单元的探测注释。探测注释显示单元的标签和该单元处当前场输出变量的结果。您可以自定义探测注释文本的字体和颜色。

为了帮助在视口中定位单元，您可能希望在 使用查询工具集之前生成带有所有单元边缘可见的模型图。更多信息，请参阅["控制单元和表面边缘可见性，"第55.3.1节](pt05ch55s03hlb01.md)。

**探测单元：**

1. 找到"探测值"对话框。在可视化模块工具箱中，单击![](../graphics/ico_probe.png)。"探测值"对话框出现。
2. 选择Abaqus/CAE将获取值的步骤、帧和场输出变量。如果您从当前模型数据库探测数据，帧选择不可用，并且您的场输出变量选择是当前视口中显示模型的属性。在"探测值"对话框中，"场输出"字段标识当前步骤、帧和场输出变量。- 要更改步骤或帧，请单击![](../graphics/ico_stepSmall.png)。- 要更改场输出变量，请单击![](../graphics/ico_fieldOutput.png)。- 要更改结果选项，请从主菜单栏中选择****结果**![](../graphics/images/arrow.gif)**选项****。
3. 在对话框的"探测值"部分，执行以下操作：1. 单击"探测"箭头，然后选择"单元"。2. 如果当前主输出变量是张量或向量变量，请单击"分量"箭头，然后执行以下任一操作：- 对于张量变量，选择"所选"仅显示变量的所选分量，"所有直接"以在查询中包含张量变量的所有直接分量，或"所有主"以在查询中包含张量变量的所有主不变量。- 对于向量变量，选择"所选"仅显示变量的所选分量，或"全部"以在查询中包含向量张量变量的所有分量。3. 要选择显示场输出变量的输出位置，请单击"位置"箭头，然后选择"积分点"、"质心"、"单元节点"或"单元面"。数据表列中显示可用单元信息类型。
4. 单击鼠标按钮3在数据表任意位置，从出现的列表中选择"编辑可见列"，然后切换您想要作为数据表中的列显示的单元信息类型。当所需信息类型被切换打开时，Abaqus/CAE将该类型的数据显示为数据表中的列。但是，所有列（可见或不可见）都包含在探测值输出文件中。**注意：**仅当当前主变量是基于单元的变量（如应力或应变）时，场输出才可用。
5. 要选择您要探测的单元，请选择"从视口选择"、"键入标签"或"选择显示组"，然后使用以下技术之一：**直接从视口选择单元：**1. 移动光标以探测当前视口中的模型图。当您将光标放在单元上时，Abaqus/CAE在"探测值"对话框的数据表第一行显示您请求的项目值。2. 要选择和累积感兴趣的值，并允许随后将这些值写入文件，请在将光标置于单元上时单击鼠标按钮1。Abaqus/CAE将数据表第一行的数据存储到数据表的存储行中。**直接输入单元标签到表中：**1. 单击"部件实例"箭头，然后选择要从中选择单元标签的部件实例。2. 输入您要包含的单元标签或标签。您可以通过用逗号分隔来同时指定多个单元标签。**选择显示组中包含的单元：**1. 单击"显示组"箭头，然后选择包含您要包含在查询中的单元的显示组。**注意：**如果您更改显示组选择，Abaqus/CAE会用新选择的显示组中的单元替换数据表的内容。如果要将其他单元追加到您的列表，必须从视口选择或通过单元标签键入。
6. 重复单元选择步骤，直到"探测值"对话框中的数据表包含您要查询的所有单元。
7. 要从数据表删除数据或清除或排序表，请单击鼠标按钮3。更多信息，请参阅["输入表格数据，"第3.2.7节](pt01ch03s02s07.md)。
8. 要使用数据表中的单元创建显示组，请执行以下操作：1. 在数据表中单击鼠标按钮3，然后从出现的菜单中选择"创建显示组"。"创建显示组"对话框出现。2. 输入新显示组的名称，然后单击"确定"。
9. 要显示数据表中任何单元的探测注释，请切换对应于您要标注的单元的行的复选框，或切换标题行中的复选框以显示所有所选单元的探测注释。**注意：**您可以在"常规绘图选项"对话框中自定义单元探测注释的颜色和字体。更多信息，请参阅["常规绘图选项概述，"第43.3节](pt05ch43hla01.md)。
10. 要保存您选择的数据，请单击"写入文件"。更多信息，请参阅[第54章，"生成表格数据报告"](pt05ch54.md)。"
11. 要退出探测模式，请执行以下操作之一：- 单击提示区域中的取消![](../graphics/afxI_cancel.png)按钮。- 单击"探测值"对话框中的"取消"。- 单击鼠标按钮2。- 在其他（例如，未变形、变形）模式下生成图。
