# 54.2 表格报告选项概述





选择 ****Report**![](../graphics/images/arrow.gif)**XY****、 ****Report**![](../graphics/images/arrow.gif)**Field Output**** 或 ****Report**![](../graphics/images/arrow.gif)**Free Body Cut**** 可分别自定义和生成X-Y数据对象、场输出变量值或自由体切割的表格列表。选择 ****Tools**![](../graphics/images/arrow.gif)**Query**** 可生成探测值的表格报告。以下列表标识了您可以自定义的表格报告特征以及可以找到信息的部分。您可以自定义:

- 报告中包含的X-Y数据对象或场输出变量。请参阅 ["选择报告数据," 第54.3节](pt05ch54s03.md)。
- 报告中包含的自由体切割。Abaqus/CAE在当前输出数据库中将所有活动的自由体切割包含在自由体切割报告中。您可以从"**Free Body Cut Manager**"激活和抑制自由体切割。请参阅 ["显示、隐藏和高亮自由体切割," 第67.4节](pt06ch67s04.md)。
- 报告中包含的探测值。请参阅 [第51章,"探测模型](pt05ch51.md)。
- 写入报告的文件名。请参阅 ["指定报告文件名," 第54.4节](pt05ch54hla01.md)。
- 报告的布局和宽度。请参阅 ["控制报告布局、宽度、格式和坐标系," 第54.5节](pt05ch54hla02.md)。
- 场输出或探测值的排序方式。对于场输出,请参阅 ["对场输出数据进行排序," 第54.6节](pt05ch54hla03.md);对于探测值,请参阅 ["输入表格数据," 第3.2.7节](pt01ch03s02s07.md)。
- 报告值的格式和有效数字。请参阅 ["格式化报告值," 第54.7节](pt05ch54hla04.md)。
- 是否包含最小值和最大值摘要或列总计。请参阅 ["报告数据值、最小值、最大值和总计," 第54.8节](pt05ch54hla05.md)。
- 报告中复数结果的形式。请参阅 ["控制复数结果的形式," 第42.6.9节](pt05ch42s04hlb05.md)。
