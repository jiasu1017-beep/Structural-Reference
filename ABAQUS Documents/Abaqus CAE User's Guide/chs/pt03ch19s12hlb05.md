# 19.12.5 继续已终止的优化过程







您可以重新启动因优化外部原因而失败的优化过程。要继续优化过程，请选择****优化**![](../graphics/images/arrow.gif)**重新启动**![](../graphics/images/arrow.gif)***优化过程名称*****。您也可以通过在**优化过程管理器**中选择优化过程名称，然后点击**重新启动**来继续优化过程。

只有当优化过程因优化或 Abaqus 分析外部的问题而失败时，您才能重新启动优化过程，例如：
- 许可证不可用。
- Abaqus 分析因磁盘空间或内存不足而失败。
- 用户通过从作业监视器点击**终止**按钮终止了 Abaqus 分析作业。
