# 12.4.6 指定材料参数和数据





选择行为时，行为定义区域会更改以显示当前所选行为的所有关联参数和数据项。参数显示在行为描述区域的顶部，数据项显示在底部。

根据您的分析要求，您可以选择接受或更改默认参数值；例如，您可以使用弹性形式上的**Type**组合框来选择是否使用各向同性弹性，如图[12-10](pt03ch12s04s06.md#prp-type)所示。

**图12-10** **Type**组合框。

![](../graphics/prp-type-nls.png)

参数区域下方会出现一个包含剩余必需材料数据字段的表格；例如，[图12-11](pt03ch12s04s06.md#prp-tabdata)显示了选择各向同性弹性时出现的表格。

**图12-11** 各向同性弹性表格。

![](../graphics/prp-tabdata-nls.png)

根据您设置参数的方式，不同的字段会变为可用。例如，当您选择层压弹性而不是各向同性弹性时，会出现[图12-12](pt03ch12s04s06.md#prp-lamina)中的表格。

**图12-12** 层压弹性表格。

![](../graphics/prp-lamina-nls.png)

您可以使用键盘将数据输入表格。或者，您可以鼠标右键点击表格中的任意位置来查看用于指定表格数据的选项列表。例如，存在一个用于自动从文件输入数据的选项。另一个选项用于从表格中的数据创建*X–Y*数据对象；您可以在Visualization模块中绘制*X–Y*数据并直观地检查其有效性。有关每个选项的详细信息，请参见["输入表格数据，"第3.2.7节](pt01ch03s02s07.md)。
