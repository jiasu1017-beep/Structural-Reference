# 70.5.3 使用选择中点/基准点方法分区边







您可以通过选择边的中点或沿边的基准点来分区所选边（如下图所示）：

![](../graphics/par-edge-pick-nls.png)

您不能直接修改通过选择现有点创建的分区。但是，如果您随后移动该点，分区将随点一起移动。

**使用选择中点/基准点方法分区边：**

1. 从主菜单栏中选择 ****Tools**![](../graphics/images/arrow.gif)**Partition****（工具→分区）。出现 **Create Partition**（创建分区）对话框。Abaqus/CAE 在提示区域显示提示以指导您完成过程。**提示：**您也可以使用位于模块工具箱分区边工具处的 ![](../graphics/ico_ptnEdgeMidpoint.png) 工具来使用选择中点/基准点方法分区边。 有关工具箱中分区工具的图表，请参阅["使用分区工具集，" 第 70.2 节"](pt06ch70s02.md)。
2. 从对话框顶部的 **Type**（类型）单选按钮中选择 **Edge**（边）。**Method**（方法）列表显示可用于分区边的方法。
3. 从方法列表中选择 **Select midpoint/datum point**（选择中点/基准点）。
4. 选择要分区的边。Abaqus/CAE 高亮显示所选边。
5. 选择边的中点或沿边的基准点。Abaqus/CAE 高亮显示分区的位置。
6. 在提示区域中点击 **Create Partition**（创建分区）。Abaqus/CAE 创建分区。
