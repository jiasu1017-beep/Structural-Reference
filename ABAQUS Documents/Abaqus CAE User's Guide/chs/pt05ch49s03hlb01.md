# 49.3.1 保存动画





您可以通过首先在一个或多个视口中播放动画，然后从主菜单栏中选择 ****动画（Animate）**![](../graphics/images/arrow.gif)**另存为（Save As）**** 将动画保存到文件。

您可以选择保存全部或仅保存选定的视口。您保存的视口可以处于任何绘图状态（例如，变形、X–Y 或动画）。对于当前处于动画绘图状态的视口，Abaqus/CAE 会保存动画的所有帧，从第一帧开始；保存时动画是否正在播放并不重要。如果您保存包含动画的多个视口，Abaqus/CAE 会将它们全部保存为单个基于图像的动画。Abaqus/CAE 通过重复具有较少帧的任何动画的最后一帧来同步文件上的多个动画。

**保存动画：**

1. 生成一个或多个时间历史、比例因子或谐波动画。有关更多信息，请参阅 ["生成和自定义基于对象的动画，" 第 49.2 节"](pt05ch49s02.md)。
2. 从主菜单栏中，选择 ****动画（Animate）**![](../graphics/images/arrow.gif)**另存为（Save As）****。出现 **保存图像动画（Save Image Animation）** 对话框。
3. 对于 AVI 或 QuickTime 格式，使用 **选择（Selection）** 字段来选择要保存的视口：1. 点击 **捕获（Capture）** 字段旁边的箭头以保存 **所有视口（All Viewports）**（默认）或 **当前视口（Current Viewport）**。2. 切换 **捕获视口装饰（Capture viewport decorations）** 以控制 Abaqus/CAE 是否保存可见的视口装饰。对于 VRML 或压缩 VRML 格式，保存的动画将仅显示当前视口并省略视口装饰。
4. 使用 **设置（Settings）** 字段指定动画文件的名称和格式：1. 点击 ![](../graphics/ico_fileOpen.png) 过滤和浏览文件名，或在 **文件名（File name）** 字段中输入您选择的文件名。有关更多信息，请参阅 ["使用文件选择对话框，" 第 3.2.10 节"](pt01ch03s02s10.md)。2. 从 **格式（Format）** 选项中，为您的文件选择 **AVI**（默认）、**QuickTime**、**VRML** 或 **压缩 VRML（Compressed VRML）** 格式。要了解有关这些格式选项的更多信息，请参阅 ["选择动画文件格式，" 第 49.3.2 节"](pt05ch49s03hlb02.md)。
5. 对于 AVI 或 QuickTime 格式，使用 **帧速率（Frame Rate）** 字段来指定保存的动画中每秒显示的帧数。您可以通过在 **速率（Rate）** 字段中输入值或拖动滑块来指定 1 到 50 帧/秒之间的帧速率来指定任何正整数作为帧速率。默认情况下，帧速率与当前在 **播放器（Player）** 页面上选择的帧速率匹配。
6. 点击 **应用（Apply）** 保存您的动画。Abaqus/CAE 根据您的规范将画布视口捕获到文件。
