# 29.3.4 定义紧固件公式





您可以使用**Formulation**标签页选择Abaqus/CAE用于将紧固点运动耦合到所选曲面的方法。您还可以约束选定的旋转自由度并选择加权方法。

**要定义紧固件公式：**

1. 从**Edit Fasteners**对话框中，显示**Formulation**标签页。
2. 选择将紧固点运动耦合到影响半径内落在曲面上的节点平均运动的方法。
   - 选择**Continuum distributing**（默认）以将紧固点的位移和旋转耦合到节点的平均位移。
   - 选择**Structural distributing**以将紧固点的位移和旋转耦合到节点的平均位移和旋转。更多信息，请参见["定义表面耦合方法"中的"耦合约束"，Abaqus Analysis User's Guide第35.3.2节](../usb/usb-link.md#usb-cni-pcoupling-method)。
3. 如果需要，关闭旋转自由度以移除其约束。Abaqus/CAE始终约束紧固件中的所有平动自由度。
4. 选择Abaqus将用于修改影响半径内耦合节点处默认权重分布的加权方法。
   - 选择**Uniform**（默认）以选择等于1.0的均匀权重分布。
   - 选择**Linear**以选择随距离紧固点线性减小的权重分布。
   - 选择**Quadratic**以选择随距离紧固点按二次多项式函数减小的权重分布。
   - 选择**Cubic**以选择随距离紧固点按三次多项式单调函数减小的权重分布。更多信息，请参见["加权方法"中的"耦合约束"，Abaqus Analysis User's Guide第35.3.2节](../usb/usb-link.md#usb-cni-pcoupling-weighting)。
