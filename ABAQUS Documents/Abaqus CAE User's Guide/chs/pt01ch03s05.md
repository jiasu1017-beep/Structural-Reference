# 3.5 使用模型树和结果树







模型树和结果树是用于导航和管理模型及分析结果的方便工具。您可以使用模型树查看模型及其包含的项目，也可以使用结果树显示来自输出数据库的分析结果以及特定于会话的数据（如 X-Y 绘图）。两个树都提供了主菜单栏、模块工具箱和各种管理器大部分功能的快捷方式。本节描述模型树和结果树。涵盖以下主题：
