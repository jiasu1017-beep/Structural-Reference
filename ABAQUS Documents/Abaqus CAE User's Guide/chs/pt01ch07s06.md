# 7.6 控制拖动模式





拖动模式控制在旋转、平移或缩放视图操作期间模型的外观。拖动模式可设置为**快速（线框）**或**原样**。

****快速（线框）****

当你将拖动模式设置为**快速（线框）**时，会在视图操作期间绘制线框轮廓。**快速（线框）**是兼容系统上的默认拖动模式设置。

****原样****

当你将拖动模式设置为**原样**时，窗口中显示的所有内容将在视图操作期间继续显示。在较旧或较慢的系统上，在你操作视图时显示可能会落后于鼠标移动，特别是如果模型复杂的话。对于具有图形硬件加速的新系统，**原样**设置可以在不显著损失性能的情况下被接受。

将拖动模式设置为**原样**以在视图操作期间观察模型；例如，在旋转轮廓图时定位高应力集中区域。

此外，你可以选择是否在旋转视图后 Abaqus/CAE 会自动将图像拟合到当前视口。自动将图像拟合到视口等同于单击**视图操作**工具栏中的自动拟合工具 ![](../graphics/ico_viewFit.png)。自动拟合会调整你对模型的视图，使模型填充视口并在视口中居中。方向保持不变，如视图 triad 所示。

**要控制拖动模式：**

1. 找到拖动模式选项。从主菜单栏中，选择****View**![](../graphics/images/arrow.gif)**Graphics Options****。Abaqus/CAE 显示**图形选项**对话框。
2. 从**视图操作**字段中，选择**快速（线框）**或**原样**拖动模式。
3. 如有需要，切换**旋转后自动拟合**开启。
4. 单击**确定**实施更改并关闭对话框。你的更改会在会话期间保存。
