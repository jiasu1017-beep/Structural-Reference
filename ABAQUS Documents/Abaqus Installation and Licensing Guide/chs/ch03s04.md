# 3.4 冗余 FLEXnet 许可证服务器配置



Abaqus FLEXnet 许可证提供两种形式的服务器冗余来防止许可证中断：冗余许可证服务器 triad 和许可证文件列表。冗余许可证服务器 triad 由三台使用相同许可证密钥的许可证服务器组成。许可证文件列表由多个独立的许可证服务器组成，每个服务器都有自己的许可证密钥；令牌请求将检查列表中的每个许可证服务器，直到找到可用的令牌。有关两种冗余配置方案的详细比较，请参阅 Dassault Systèmes 知识库中的"比较多个许可证服务器与三服务器冗余"，网址为 [www.3ds.com/support/knowledge-base](http://www.3ds.com/support/knowledge-base)。本指南不详细介绍许可证文件列表的配置。

冗余许可证服务器 triad 需要三台计算机。至少必须有两台许可证服务器运行才能授予许可证。三台用于冗余服务器 triad 的计算机之间的网络通信必须一致和可靠。冗余许可证服务器 triad 被视为 FLEXnet Licensing 的高级使用。冗余许可证服务器的更详细要求出现在 FLEXnet Licensing End User Guide Version 11.6 中，可从 **Support** 页面上的 **Licensing** 部分下载，网址为 [www.3ds.com/simulia](http://www.3ds.com/simulia)。

您可以使用自动或手动过程安装和配置冗余 FLEXnet 许可证服务器 triad。如果您的服务器使用不同的端口号，则需要手动过程；否则，建议使用自动安装。在这两种情况下，您都需要使用 FLEXnet 许可证服务器管理器 `lmgrd` 手动启动服务器。在初始启动时，必须在一分钟内同时启动冗余许可证服务器。

### 3.4.1 自动安装冗余 FLEXnet 许可证服务器 triad

使用以下过程自动安装和配置冗余许可证服务器 triad。

1. 将您收到的许可证文件复制到三台许可证服务器计算机上。
2. 在其中一台服务器计算机上，使用 Abaqus 许可证安装过程自动安装、配置和启动 Abaqus 许可证服务器（请参阅["FLEXnet 许可证安装，"第 2.1.3 节](ch02s01.md#sgb-chp-install-glance-lic)）。一旦安装程序检测到冗余许可证文件，它将无法启动服务器。但是，它会自动安装和配置许可证服务器。
3. 按照对话框中出现的说明完成许可证安装。出现提示时，输入服务器将运行的端口号。另外，在相应的主机 ID 旁边输入您的服务器的主机名或 IP 地址。
4. 对剩余两台服务器计算机重复步骤 2 和 3。必须在每台安装过程中以相同顺序输入服务器信息。
5. 使用["启动冗余 FLEXnet 许可证服务器 triad，"第 3.4.3 节](ch03s04.md#sgb-chp-net-lic-redundantstart)中所述的过程启动服务器。

### 3.4.2 手动安装冗余 FLEXnet 许可证服务器 triad

使用以下过程手动安装和配置冗余许可证服务器 triad。如果配置中的三台服务器使用不同的端口，则需要此过程。

1. 在三台许可证服务器计算机中的每台上，使用 Abaqus 许可证安装过程安装 Abaqus FLEXnet Licensing 管理工具（请参阅["FLEXnet 许可证安装，"第 2.1.3 节](ch02s01.md#sgb-chp-install-glance-lic)）。许可证工具保存在 `*simulia_dir*/License` 目录中，其中 *simulia_dir* 是 SIMULIA 父目录。
2. 将您收到的许可证文件复制到一台服务器计算机上的许可证目录。将此许可证文件重命名为 `simulialm.lic`。
3. 编辑 `simulialm.lic` 文件，并将字符串 `this_host` 替换为您的计算机的实际主机名。验证主机名与提供的主机 ID 对应。如果您有不在服务器网络本地的客户端计算机，请使用 *hostname*.*domain_name*。
4. 将网络端口号添加到 `SERVER` 行。建议您不要使用 27000 到 27009 之间的端口号。
5. 对每个 `SERVER` 行重复此过程。
6. 将以下关键字追加到第一个 `SERVER` 行： ``` PRIMARY_IS_MASTER HEARTBEAT_INTERVAL=60 ``` 例如： ``` SERVER bifrost.simulia.com 69084992 2501 \ PRIMARY_IS_MASTER HEARTBEAT_INTERVAL=60 SERVER tank.simulia.com 69094954 2501 SERVER willow.simulia.com 69094344 2501 VENDOR ABAQUSLM ``` 其中每台服务器配置为使用端口号 2501，以 690 开头的数字是 `simulialm.lic` 文件中为每台服务器主机提供的 host id。必须为每台服务器分配端口号，但每台服务器上的数字不必相同。**注意：**许可证文件中的每行必须为 80 个字符或更少。您可以使用反斜杠字符 (`\`) 表示行继续；在反斜杠后出现在行上的关键字被视为前一行的一部分。
7. 保存 `simulialm.lic` 文件，并将文件复制到其他两台服务器计算机上的许可证目录。
8. 在 Windows 平台上，您必须使用["将 FLEXnet 许可证安装为 Windows 服务，"第 3.6.8 节](ch03s06.md#sgb-chp-net-lic-service)中所述的过程在每台许可证服务器上将许可证安装为服务。
9. 使用["启动冗余 FLEXnet 许可证服务器 triad，"第 3.4.3 节](ch03s04.md#sgb-chp-net-lic-redundantstart)中所述的过程启动许可证服务器。

### 3.4.3 启动冗余 FLEXnet 许可证服务器 triad

在所有三台计算机上安装和配置许可证服务器后，必须使用以下过程在一分钟内启动每台服务器。

**Linux 平台**

通过从许可证目录执行以下命令启动 Abaqus 许可证服务器：

```
 ./lmgrd -c simulialm.lic -l +simulialm.log 
```

**Windows 平台**

您必须以 Administrator 身份登录才能启动服务器。

1. 从**开始**菜单，选择****程序**![](../graphics/images/arrow.gif)**Abaqus 许可证**![](../graphics/images/arrow.gif)**许可证实用程序****。
2. 在**服务/许可证文件**选项卡页上验证已启用**使用服务配置**，然后选择**启动/停止/重新读取**选项卡。
3. 单击**启动服务器**。
