# 11.3.4 什么是零件实例？





零件实例可以看作是原始零件的表示。您在零件模块中创建零件，并在属性模块中定义其属性。但是，当您使用装配模块组装模型时，您只能使用零件的实例，而不是零件本身。相互作用模块和载荷模块也在装配体上操作，因此也在零件实例上操作。相比之下，网格模块允许您对装配体或其组件零件之一进行操作。

您在装配模块中创建零件实例。然后相对于全局坐标系定位这些实例，以形成装配体。您可以创建和定位单个零件的多个实例。此外，在求解接触问题时，您可以组装可变形、分析刚体和离散刚体零件的实例。有关可以在Abaqus/CAE中创建的零件类型的更多信息，请参见"零件类型"，第11.4.2节。

以下示例说明了零件与零件实例之间的关系。泵壳由三个零件组成：壳盖、垫圈和安装螺栓。在零件模块中，您创建如图所示的三个零件：
- 一个壳盖
- 一个垫圈
- 一个螺栓

**图11-7** 原始零件。

![](../graphics/prt-instances1-nls.png)

在装配模块中，您组装每个零件的实例：
- 一个壳体实例
- 一个垫圈实例
- 八个螺栓实例

然后相对于公共坐标系定位这些实例，从而创建泵壳模型，如图所示。

**图11-8** 该模型由零件实例组装而成。

![](../graphics/prt-instances2.png)

现在，假设您要更改螺栓的长度。您返回零件模块，通过编辑原始零件来修改螺栓的长度。当您返回装配模块时，Abaqus/CAE会识别出零件已被修改，并自动重新生成八个螺栓实例以反映长度的变化。

您不能直接修改零件实例的几何；您只能在零件模块中修改零件本身。当您修改零件时，Abaqus/CAE会自动重新生成装配体中修改后的零件的所有实例。零件实例在"使用零件实例"，第13.3节中的装配模块上下文中更详细地讨论。
