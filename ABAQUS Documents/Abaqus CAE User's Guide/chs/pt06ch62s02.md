# 62.2 使用Datum工具集







您可以通过从主菜单栏中选择****Tools**![](../graphics/images/arrow.gif)**Datum****来在任何模块中访问Datum工具集。此时将出现**Create Datum**对话框，您可以从对话框顶部的**Type**区域的按钮中选择要创建的datum类型——点、轴、平面或坐标系(CSYS)。**Method**列表会相应变化以反映您创建的datum。从**Method**列表中选择所需的datum工具，然后按照提示区域中的提示创建datum。["datum创建技术概述，" 第62.5节](pt06ch62s05.md)提供了每种datum类型可用方法的概述。关于在零件和装配上创建datum几何体的更多信息，请参见["在Part模块中使用Datum工具集，" 第11.16.1节](pt03ch11s16s01.md)和["在Assembly模块中使用datum几何体，" 第13.8.1节](pt03ch13s08s01.md)。关于显示datum几何体的更多信息，请参见["控制datum显示，" 第76.9节](pt07ch76hla08.md)。

您也可以从模块工具箱访问Datum工具集；[图62--3](pt06ch62s02.md#dtm-toolbox)显示了模块工具箱中所有datum工具的隐藏图标。

**图62–3** datum工具。

![](../graphics/dtm-toolbox-nls.png)

要查看包含每个datum工具定义的简要提示，请将鼠标在该工具上停留片刻。关于使用工具箱和选择隐藏图标的信息，请参见["使用包含隐藏图标的工具箱和工具栏，" 第3.3.2节](pt01ch03s03s02.md)。
