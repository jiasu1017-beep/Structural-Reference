# 4.9.3 R0031(3)：法向压力载荷下的三层夹芯壳





**产品：** Abaqus/Standard  Abaqus/Explicit  

### 测试单元

S4R    S8R    SC6R    SC8R    

### 问题描述

![](../graphics/exxr313-model-nls.png)

**网格：**

在Abaqus/Standard中使用4×4网格的复合S8R单元，在Abaqus/Explicit中使用8×8网格的复合S4R单元，在Abaqus/Standard中使用8×8网格的连续壳单元的四分之一模型。提供了两种类型的连续壳模型：（1）单个复合单元；（2）沿厚度方向叠加的三个单层单元。

**材料：**

面板：![](../graphics/bmk_eqn01833.gif) = 1.0×10⁷ psi，![](../graphics/bmk_eqn01834.gif) = 4.0×10⁶ psi，![](../graphics/bmk_eqn00005.gif) = 0.3，![](../graphics/bmk_eqn01838.gif) = 1.875×10⁶ psi，![](../graphics/bmk_eqn01839.gif) = 1.875×10⁶ psi，![](../graphics/bmk_eqn01840.gif) = 1.875×10⁶ psi

芯层：![](../graphics/bmk_eqn01833.gif) = 10.0 psi，![](../graphics/bmk_eqn01834.gif) = 10.0 psi，![](../graphics/bmk_eqn00005.gif) = 0，![](../graphics/bmk_eqn01838.gif) = 10.0 psi，![](../graphics/bmk_eqn01839.gif) = 3.0E4 psi，![](../graphics/bmk_eqn01840.gif) = 1.2E4 psi。连续壳模型中的厚度模量选择得足够高，以避免在解析解中忽略的夹持效应。

**边界条件：**

四条边缘全部简支。连续壳模型使用方程约束来提供等效的中面约束。

**载荷：**

100 psi的均匀法向压力。

### 参考解

这是英国国家有限元方法与标准机构（NAFEMS）推荐的测试：NAFEMS出版物R0031"Composites Benchmarks"（1995年2月）中的测试R0031/3。
