# *DEFORMATION PLASTICITY





### *DEFORMATION PLASTICITY指定变形塑性模型。

此选项用于将材料的机械行为定义为变形理论Ramberg-Osgood模型。

**产品：**Abaqus/Standard  Abaqus/CAE  

**类型：**模型数据  

**级别：**模型  

**Abaqus/CAE：**属性模块

##### **参考：**

- ["变形塑性，" Abaqus分析用户指南第23.2.13节](../usb/usb-link.md#usb-mat-cdeformationplast)

**此选项没有关联的参数。**

### **定义变形塑性的数据行：**

**第一行：**

根据需要重复此数据行，以将变形塑性参数对温度的依赖性定义为函数。
