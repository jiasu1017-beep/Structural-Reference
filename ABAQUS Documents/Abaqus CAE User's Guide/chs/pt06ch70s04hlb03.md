# 70.4.3 分区胞的方法概述







当您从 **Create Partition**（创建分区）对话框中选择 **Cell**（胞）时，**Method**（方法）列表显示以下分区胞的方法：

**![](../graphics/ico_ptnCellCuttingPlane.png) 定义切割平面**

通过用平面切割胞来分区胞；平面将完全穿过胞。使用以下三种方法之一来定义切割平面：
- 在切割平面上选择一个点；然后选择一个定义此平面法向的边或基准轴（如图 70-15 所示）。**图 70-15** 用点和法向定义切割平面。![](../graphics/par-cell-plane1-nls.png)
- 选择三个不同且不共线的点（如图 70-16 所示）。**图 70-16** 用三个点定义切割平面。![](../graphics/par-cell-plane2-nls.png)
- 选择一条边和沿该边的一个点；切割平面将在所选点处垂直于该边（如图 70-17 所示）。**图 70-17** 用边和点定义切割平面。![](../graphics/par-cell-plane3-nls.png)

详细说明，请参阅["使用切割平面方法分区胞，" 第 70.7.1 节"](pt06ch70s07hlb01.md)。

**![](../graphics/ico_ptnCellDatumPlane.png) 使用基准平面**

使用与基准平面扩展的交点来分区胞（如图 70-18 所示）。详细说明，请参阅["使用基准平面方法分区胞，" 第 70.7.2 节"](pt06ch70s07hlb02.md)。

**图 70-18** 使用基准平面分区胞。

![](../graphics/par-cell-datum-nls.png)

**![](../graphics/ico_ptnCellExtendFace.png) 扩展面**

通过用壳切割胞来分区胞，其中壳是面的扩展几何（如图 70-19 所示）。被扩展的面可以是平面、圆柱形、圆锥形或球形的。详细说明，请参阅["使用扩展面方法分区胞，" 第 70.7.3 节"](pt06ch70s07hlb03.md)。

**图 70-19** 使用面的扩展分区胞。

![](../graphics/par-cell-face-nls.png)

**![](../graphics/ico_ptnCellSweepEdges.png) 拉伸/扫掠边**

通过沿选定路径（称为扫掠路径）扫掠所选边（形成扫掠轮廓）来分区胞。您可以选择任意数量的边进行扫掠，尽管所有边必须连接、必须位于同一平面上且必须属于同一部件实例。

使用以下两种方法之一来定义扫掠路径：
- 通过沿平行于选定直边或作为扫掠路径的基准轴的方向无限扩展扫掠轮廓，在胞中创建直分区；分区在扫掠边穿过所选胞的位置创建（如图 70-20 所示）。扫掠路径必须直且垂直于被扫掠的边集合。**图 70-20** 沿方向扫掠轮廓。![](../graphics/par-cell-sweep1-nls.png)
- 通过沿选定边延伸或平行于选定边，在胞中创建直或曲线分区。分区仅延伸至选定边的范围；分区在扫掠边穿过所选胞的位置创建（如图 70-21 所示）。扫掠路径必须位于包含要扫掠的边的平面开始，其切线必须垂直于同一平面。**图 70-21** 沿边扫掠轮廓。![](../graphics/par-cell-sweep2-nls.png)

详细说明，请参阅["使用拉伸/扫掠方法分区胞，" 第 70.7.4 节"](pt06ch70s07hlb04.md)。

**![](../graphics/ico_ptnCellNSidedPatch.png) 使用 n 边面片**

通过用由连接边环形成的曲面面片划分胞来分区胞。这些边可以是弯曲的或直的，必须连接，并且必须属于与被分区胞相同的部件。此外，面片必须完全穿过胞。选择以下方法之一来定义面片：

****选择边****

您可以选择以下方法来选择形成 N 边面片的边：

****环****

选择一条边，并允许 Abaqus/CAE 搜索将划分胞的连续连接边环（如图 70-22 所示）。 resulting 面片可以具有任意数量的边。

**图 70-22** 选择一条边后允许 Abaqus/CAE 定义面片。

![](../graphics/par-cell-patch-edges-nls.png)

****边****

手动选择将划分胞的边。您可以选择任意数量的边，所选边必须形成闭合环。

****选择角点****

选择三个、四个或五个定义面片角的点。如果两个点由现有边连接， resulting 分区将遵循边的曲线（如图 70-23 所示）。这些点必须在被分区胞的边界边上。

**图 70-23** 用角点定义面片。

![](../graphics/par-cell-patch2-nls.png)

详细说明，请参阅["使用 N 边面片方法分区胞，" 第 70.7.5 节"](pt06ch70s07hlb05.md)。

**![](../graphics/ico_ptnCellSketch.png) 草绘平面分区**

使用草图绘制器草绘分区来分区所选胞（如图 70-24 所示）。在大多数情况下，您将在与所选胞相交的基准平面上草绘。您也可以选择一个现有面进行草绘，并在面边界外绘制草图。Abaqus/CAE 在草图与胞相交的任何位置创建分区。

**图 70-24** 使用草图绘制器分区胞。

![](../graphics/par-cell-sketch-nls.png)

详细说明，请参阅["使用草绘平面分区方法分区胞，" 第 70.7.6 节"](pt06ch70s07hlb06.md)。
