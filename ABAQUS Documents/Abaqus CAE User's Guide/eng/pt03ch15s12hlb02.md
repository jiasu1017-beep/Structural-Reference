# 15.12.2 Creating interaction properties







You can create an interaction property by entering data in an interaction property editor. The format of the editor varies according to the type of property you are defining; when you create a property, you must first specify the property type so that the appropriate editor appears. For more information, see ["Understanding interaction properties," Section 15.4](pt03ch15s04.md), and ["Using the interaction property editors," Section 15.14](pt03ch15s14.md).

**To create an interaction property:**

1. From the main menu bar, select ****Interaction**![](../graphics/images/arrow.gif)**Property**![](../graphics/images/arrow.gif)**Create****. **Tip:**You can also create an interaction property using the ![](../graphics/ico_interactionProperty.png) tool in the Interaction module toolbox. A **Create Interaction Property** dialog box appears.
2. Enter a property name. For more information about naming objects, see ["Using basic dialog box components," Section 3.2.1](pt01ch03s02s01.md).
3. Select the property type, and click **Continue**. The editor for the property type you have specified appears.
4. In the editor, enter all of the data necessary to define the interaction property. **Note:**You can display help on a particular editor feature by selecting ****Help**![](../graphics/images/arrow.gif)**On Context**** from the main menu bar and then clicking the editor feature of interest.
5. Click **OK** to save the data and to exit the editor.
