# 13.10.5 创建零件实例的线性阵列





要在线性阵列中创建所选零件实例的多个副本，请从主菜单栏选择****Instance**![](../graphics/images/arrow.gif)**Linear Pattern****。你可以创建沿一个方向延伸的阵列（例如，水平或垂直），或者创建沿两个方向延伸的阵列（例如，水平和垂直）。创建阵列后你不能编辑它。你可以指定以下内容：
- 每个方向上要创建的实例数量，包括选定实例。你可以创建任意数量的实例。
- 沿指定方向每个实例之间的间距。
- 定义Abaqus/CAE生成实例方向的线。

更多信息，请参见["创建零件实例阵列，"第13.6节](pt03ch13s06.md)。你不能将线性阵列工具与模型实例一起使用。

**要创建零件实例的线性阵列：**

1. 从主菜单栏，选择****Instance**![](../graphics/images/arrow.gif)**Linear Pattern****。
   **提示：**你也可以使用装配模块工具箱中的![](../graphics/ico_patternLinear.png)工具来创建零件实例的线性阵列。有关装配工具箱中工具的图表，请参见["使用装配模块工具箱，"第13.9节](pt03ch13s09.md)。
2. 选择要复制的零件实例。
   **提示：**要选择多个实例，请在点击每个实例时按住**[Shift]**键，或在实例周围拖动一个矩形。要取消选择实例，请使用**[Ctrl]****+Click**。更多信息，请参见[第6章"在视口内选择对象"](pt01ch06.md)。
3. 点击鼠标按钮2表示已完成选择实例。Abaqus/CAE显示**Linear Pattern**对话框。
4. 在**Linear Pattern**对话框中，配置**Direction-1**中的阵列（默认情况下，**Direction-1**是*X*方向）：
   1. 点击**Number**右侧的箭头增加或减少要创建的副本数量，包括选定实例。当你点击箭头时，装配中的副本数量会更新并提供设置的预览。或者，你可以输入数字并按**[Enter]**预览设置。你可以输入任何大于或等于1的数字。如果你输入1，Abaqus/CAE仅显示选定的实例，不创建选定实例的任何副本；实际上，你是在**Direction-1**中禁用副本。
   2. 输入沿指定方向每个副本之间的**Spacing**。
   3. 默认情况下，Abaqus/CAE沿*X*方向创建副本。如果你想更改Abaqus/CAE创建副本的方向，点击![](../graphics/ico_selectBlue.png)并从装配中选择一条线来定义新方向。你必须选择一条直边或基准轴。
   4. 默认情况下，Abaqus/CAE沿正方向创建副本。点击![](../graphics/ico_flipArrow.png)反转Abaqus/CAE创建副本的方向。
5. 要在第二个方向创建副本，请为**Direction-2**输入大于1的**Number**并指定**Spacing**、**Direction**和**Flip**方向（默认情况下，**Direction-2**是*Y*方向）。你必须至少在一个方向输入大于1的**Number**。
6. 在大多数情况下，你会在**Linear Pattern**对话框中输入值时预览Abaqus/CAE将创建的线性阵列。但是，如果你选择创建大量副本，预览功能可能会影响Abaqus/CAE的性能。在这种情况下，你应该关闭**Preview**按钮。
7. 要复制更多实例，从第1步开始重复上述步骤。
