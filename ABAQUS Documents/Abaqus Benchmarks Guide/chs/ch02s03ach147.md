# 2.3.1 筒拱屋顶问题





**产品：** Abaqus/Standard  Abaqus/Explicit  

过去几年中，一小组线性测试用例已成为壳单元的关键测试集（例如，参见Ashwell和Gallagher编辑的关于壳体数值建模的论文集，1976，以及Belytschko的综述论文，1986）。该测试集包含三个案例：筒拱屋顶（本例）、端部隔膜支撑的承受压痕荷载的圆柱体（["圆柱压痕问题，"第2.3.2节"](ch02s03ach148.md)），以及点荷载半球壳（["LE3：带点荷载的半球壳，"第4.2.3节"](ch04s02anf03.md)）。普遍认为，在所有三种案例中表现良好的单元应该能为大多数常规壳问题提供准确结果。这些测试用例包含在本指南中，以便评估Abaqus中提供的壳单元的性能。

包括Abaqus中的单元在内的大多数现代壳单元在这些问题上都表现良好。虽然这表明这些单元通常提供良好结果，但不应将其作为单元在所有情况下性能质量充分证明。例如，所有这三个问题都是完全规则的几何形状；候选单元在不规则几何形状中的实用性（大多数实际案例涉及高度几何不规则性）尚未测试。在本例中，我们通过不仅使用对问题而言自然选择的规则网格，还使用可能作为分析具有相同底层形状但具有某种局部不规则特征（如裂纹）的问题的基础的网格进行了一些尝试来解决这一问题。以下报告了两类网格的结果。正如所预期的，不规则网格结果不如具有相同变量数量的规则网格提供的结果好。

使用Abaqus中可用的各种壳单元和不同网格密度分析了该问题。因此，本例提供了这些单元相对效率的指示。

### 问题描述

问题示于[图2.3.1-1](ch02s03ach147.md#sxmbarrel-geom)。问题的物理基础是一个深拱屋顶，仅在其曲线边缘（飞机库）由隔膜支撑，在自身重量下变形。有趣的是，观察到几何形状使得屋顶的中心点在自重（向下方向）荷载下向上移动。也许这就是该问题在数值上不简单的原因之一。

研究了两种离散化：规则网格划分和在需要局部细化时可能使用的不规则网格划分（[图2.3.1-2](ch02s03ach147.md#sxmbarrel-mesh)）。不建议使用这种网格细化方法：引入不规则网格划分只是为了记录以这种方式使用的壳单元的一些结果。

实际屋顶跨度15.24 m（600 in） between supports，厚度为76.2 mm（3 in），因此应被视为"薄"壳。Abaqus/Standard中的一些壳单元（单元类型S4R5、S8R5、S9R5、STRI3和STRI65）旨在用作薄壳。在这些单元中，Kirchhoff假设——变形过程中最初垂直于壳参考面的线保持垂直于该面——以代数方式（在STRI3单元类型中）或数值方式（在S4R5、S8R5、S9R5和STRI65单元类型中）施加。壳单元S4R、S4和S3R以及连续体壳单元SC6R和SC8R对横向剪切使用应变假设处理，对薄壳数值施加Kirchhoff约束，并为厚壳提供准确的横向剪切预测。因此，单元类型S4R、S4、S3R、SC6R和SC8R适用于薄壳和厚壳应用。单元类型S8R主要用于厚壳建模，其中横向剪切柔性可能是变形的重要组成部分。当使用此单元对薄壳进行建模时，横向剪切刚度被视为惩罚，以离散方式施加Kirchhoff假设，惩罚基于Hughes等人（1977）描述的技术选择。

在Abaqus/Explicit中，使用S4单元、带增强沙漏控制的S4R单元、S3R单元和S3RS单元对问题进行建模。

### 结果与讨论

规则网格和不规则网格的结果描述如下。

#### 规则网格

这些结果总结于[表2.3.1-1](ch02s03ach147.md#table-barrel-disp-reg)，其中记录了自由边缘中心的垂直运动。基于深壳理论，该单一位移值普遍接受的解决方案为91.2 mm（3.59 in）（参见Ashwell在Ashwell和Gallagher（1976）中的章节，讨论了该示例的半解析和纯数值解）。[表2.3.1-1](ch02s03ach147.md#table-barrel-disp-reg)记录了该单一位移值相对于该"精确"解的误差。

从[表2.3.1-1](ch02s03ach147.md#table-barrel-disp-reg)可以明显看出，二阶薄壳单元（S8R5、S9R5）是本问题最有效的单元，S8R、STRI65和一阶四边形（S4、S4R5、S4R）除了最粗的网格外也提供了几乎同样好的解。

#### 不规则网格

不规则网格模型提供的结果总结于[表2.3.1-2](ch02s03ach147.md#table-barrel-disp-irreg)。这些结果不如具有大致相同数量单元和自由度的规则网格模型提供的结果准确。然而，当使用相对细的网格时，即使有不规则的网格模式，所有单元也能提供可接受的结果。单元类型S8R在粗不规则网格中提供的结果准确性较差，这一点特别明显，这提醒我们在高应变梯度的薄壳中使用此单元类型时使用规则网格是多么重要。

#### 总结

总之，基于本例：

1. 对于薄壳建模，Abaqus/Standard提供的最有效单元是S8R5和S9R5。STRI65与S8R5和S9R5完全兼容，推荐用于网格细化。
2. 一阶三角形单元不如相应的一阶四边形单元网格好。通常建议仅在不能容易地用四边形生成网格的地方使用三角形，并且仅在弯曲和薄膜应变梯度都不大的地方使用。
3. 尽可能确保网格遵循壳的主曲率线。
4. 对任何以前未使用过单元的问题进行收敛性研究。
5. 如果单元变形，预期其性能会下降。
