# 20.9.3 打开或关闭预选







[预选](pt03ch20s04s05.md)帮助您选择参考几何或 Sketcher 对象，如直线、顶点和尺寸。当启用预选并将光标移到这些对象之一附近时，Abaqus/CAE 会高亮显示该对象，以便您可以更轻松地选择它。预选还有助于在草绘时约束您的草图，方法是指示将创建水平、垂直、共线、垂直和重合约束的选择。当您在草图中移动光标时，关于不同 Sketcher 对象显示的预选符号列表，请参见["Sketcher 光标和预选，"第 20.4.5 节](pt03ch20s04s05.md)。

在**Sketcher 选项**对话框中切换**预选几何**以启用或禁用预选。（默认情况下，启动 Sketcher 时启用预选。）预选自定义在 Abaqus/CAE 会话期间存储并适用于所有草图。如果预选点和网格点都离光标位置很近，预选会覆盖网格捕捉。

**打开或关闭预选：**

1. 从 Sketcher 工具箱底部，选择 Sketcher 自定义工具![](../graphics/ico_sketchOptions-nls.png)。出现**Sketcher 选项**对话框。
2. 从**常规**选项卡页面，切换**预选几何**。当**预选几何**打开时，Abaqus/CAE 会在您在草图中移动时高亮显示有效选择，如顶点、中点和尺寸。当**预选几何**关闭时，Abaqus/CAE 不会高亮显示有效选择。
3. 选择完所需的自定义选项后，点击**确定**应用您的更改并关闭**Sketcher 选项**对话框。
