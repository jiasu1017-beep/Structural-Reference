# 32.9.1 用于模拟壳体中部分穿透裂纹的线弹簧单元






**产品：** Abaqus/Standard

##### **参考资料**

- ["线弹簧单元库，" 第32.9.2节](pt06ch32s09ael40.md)
- [*SHELL SECTION](../key/key-link.md#usb-kws-mshellsection)
- [*SURFACE FLAW](../key/key-link.md#usb-kws-msurfaceflaw)

### 概述

线弹簧单元：
- 用于低成本评估壳体中的部分穿透裂纹（缺陷）；
- 与壳单元一起使用；
- 可用于弹性或弹塑性（等向强化，Mises屈服）材料行为；
- 不包含热应变效应；
- 仅适用于小位移分析（不包含大旋转效应）；
- 不适用于线性扰动步骤；
- 使用相当显著的近似（特别是在弹塑性情况下），因此应谨慎使用；
- 对于裂纹深度小于壳体厚度2%或大于95%的情况不提供有用结果；并且
- 在裂纹末端或裂纹深度随位置快速变化的区域不会提供准确结果，因为这些区域解具有三维特性。

### 典型应用

线弹簧单元对壳体中的部分穿透裂纹进行低成本评估。其基本概念是，这些单元将受裂纹尖端奇异性主导的局部解引入未裂纹几何体的壳模型。这是通过允许沿裂纹线在模型中增加一个额外的自由度来实现的，该自由度由线弹簧单元提供，如图32.9.1-1所示。

**图32.9.1-1** 线弹簧模型。

![](../graphics/elinespring-models-nls.png)

线弹簧相对于这些额外自由度的柔度将局部解嵌入整体响应中。根据与该柔度共轭的相对位移和旋转，Abaqus/Standard计算并输出J积分，在线性情况下还输出裂纹尖端处的应力强度因子。由于单元简单，分析费用与未裂纹几何体的壳体分析相比不会显著增加。结果对许多常见应用具有可接受的精度。

有关这些单元背后理论的详细信息，请参见["线弹簧单元，" Abaqus理论指南第3.9.5节](../stm/stm-link.md#stm-elm-linesprings)。

### 选择适当的单元

提供了两个版本的单元——两者都旨在与二阶壳单元（S8R、S8R5、S9R5）一起使用。线弹簧单元LS6适用于一般情况，而线弹簧单元LS3S适用于裂纹位于对称面上且仅对对称面的一侧进行建模的情况。

### 定义单元的截面属性

您必须将壳截面属性与一组线弹簧单元关联。

| **输入文件用法：** | ``` [*SHELL SECTION](../key/key-link.md#usb-kws-mshellsection), ELSET=*name* ``` |
| --- | --- |

#### 定义恒定截面厚度

您可以在线弹簧单元的壳截面定义中定义恒定截面厚度。

| **输入文件用法：** | ``` [*SHELL SECTION](../key/key-link.md#usb-kws-mshellsection) *shell thickness* ``` |
| --- | --- |

#### 定义可变截面厚度

或者，您可以定义具有连续变化厚度的线弹簧单元，并指定线弹簧单元在节点处的厚度。在这种情况下，您指定的任何恒定截面厚度都将被忽略，线弹簧厚度将从节点插值（参见["节点厚度，" 第2.1.3节](pt01ch02s01aus07.md)）。必须在连接到单元的所有节点上定义厚度。

| **输入文件用法：** | 同时使用以下两个选项： |
| --- | --- |
|  | ``` [*SHELL SECTION](../key/key-link.md#usb-kws-mshellsection), NODAL THICKNESS [*NODAL THICKNESS](../key/key-link.md#usb-kws-mnodalthickness) ``` |

#### 为一组线弹簧单元指定材料定义

您必须将材料定义与每个壳截面定义关联。

线弹簧单元可用于等向弹性或弹塑性（等向强化，Mises屈服）材料行为（["线性弹性行为，" 第22.2.1节](pt05ch22s02abm02.md)，和["经典金属塑性，" 第23.2.1节](pt05ch23s02abm17.md)）；这些是仅与这些单元相关的材料行为定义。弹性行为必须是等向的。塑性仅包含I型（裂纹张开）响应；只有当I型行为主导时，弹塑性分析才是准确的。

整个截面必须使用相同的材料：不能用线弹簧定义分层截面。线弹簧单元不包含热应变效应；然而，大部分热应变发生在壳体中，因此在许多情况下这并不重要（在线弹簧所做的近似范围内）。

| **输入文件用法：** | ``` [*SHELL SECTION](../key/key-link.md#usb-kws-mshellsection), ELSET=*name*, MATERIAL=*name* ``` |
| --- | --- |

### 定义裂纹

裂纹通过沿裂纹前缘的每个节点指定其深度来定义。您必须确定裂纹是否来自壳的正表面或负表面（正表面位于沿表面法线从壳中面正距离的位置，如图32.9.1-1所示）。

在表面裂纹深度非常小或为零的位置，线弹簧单元的柔度也非常小。为了避免当小柔度被反演形成刚度时出现数值问题，Abaqus/Standard使用的最小表面裂纹深度为线弹簧单元指定厚度的2%，即使您指定了更小的表面裂纹深度。如果您想约束表面裂纹深度为零的两个节点具有相同的位移，应该用线性约束方程或多点约束将节点绑在一起（["运动约束：概述，" 第35.1.1节](pt08ch35s01abo32.md)）。这通常不需要。

| **输入文件用法：** | ``` [*SURFACE FLAW](../key/key-link.md#usb-kws-msurfaceflaw), SIDE=POSITIVE or NEGATIVE *node number or node set label*, *crack depth* ... ``` |
| --- | --- |

### 定义包含裂纹的壳体模型

您必须在截面定义中指定壳的未裂纹厚度。裂纹处壳的几何形状（坐标和表面法线）以通常的方式给出。

### 包含裂纹面上的压力载荷效应

裂纹通常出现在承受压力的表面上；要将此类载荷对裂纹面的影响纳入其中，提供了适当的分布式载荷类型。这些载荷类型不适用于弹塑性线弹簧，因为为压力计算的节点等效力基于仅在线弹性情况下有效的叠加方法。

### J积分输出

如果材料仅为线弹性，则输出J积分值和应力强度因子；对于弹塑性情况，还提供局部值![](../graphics/usb_eqn00331.gif)和![](../graphics/usb_eqn00332.gif)及其总和为单个J值。在这种情况下，只有当![](../graphics/usb_eqn00332.gif)远大于![](../graphics/usb_eqn00331.gif)时，J值才具有可接受的精度。有关详细信息，请参见["线弹簧单元，" Abaqus理论指南第3.9.5节](../stm/stm-link.md#stm-elm-linesprings)。
