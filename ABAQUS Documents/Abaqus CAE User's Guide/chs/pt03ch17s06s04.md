# 17.6.4 分割如何影响种子和其他属性？





在分割过程中，您布种的边上的种子分布可能会发生变化；Abaqus/CAE会重新分布种子以适应分割创建的任何新顶点。例如，图17-27中零件实例的左右边每条边布了7个单元。

**图17-27** 左右边各有7个单元。

![](../graphics/mgn-partition5.png)

如果您创建将零件实例分割为两个区域的分割，则两条边的中点都会创建新顶点。在图17-28中，您可以看到Abaqus/CAE如何在新的顶点处添加种子，以便在每个区域的角点处存在节点。

**图17-28** 种子重新分布。

![](../graphics/mgn-partition4.png)

Abaqus/CAE还重新分布了现有种子，以消除由新分割创建的任何过小的单元。但是，此重新分布可能导致种子未对齐。顶部区域的左侧比右侧多一个种子，底部区域则相反。在本例中，您可以将左右边的单元数量更改为偶数，以确保分割后种子对齐。

您应用的其他任何网格属性（如单元形状或单元类型）都会自动应用于使用分割创建的每个新区域。但是，创建不同区域后，您可以为每个区域分配不同的网格属性。
