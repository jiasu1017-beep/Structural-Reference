# 52.1 理解应力线性化





应力线性化是通过截面将应力分离为恒定薄膜应力和线性弯曲应力。应力线性化仅适用于输出数据库。要在 Abaqus/CAE 中进行应力线性化，使用以下过程：
- 您通过模型定义一个截面。您通过在模型中选择节点、空间中的坐标或保存的路径来指定截面的端点。
- Abaqus/CAE 通过在两点之间插值来定义应力线，以沿着直线获得用户指定的等间隔数量。
- 您可以将应力线保存为路径；保存的路径包括端点和沿应力线的每个间隔点。如果您将应力线保存为路径，Abaqus/CAE 始终创建点列表路径，无论您用于定义端点的方法是什么。
- 在由线定义的局部坐标系中，沿着线以等间隔获取指定的应力结果。
- Abaqus/CAE 执行应力线性化计算并以 X–Y 绘图的形式显示结果。您可以选择保存数据和/或将其写入文件。

如果您跨多个部件实例进行应力线性化，对于存在于多个部件实例中的任何点，结果将被平均。

要沿一条线获取线性化应力，请从主菜单栏中选择 ****工具（Tools）**![](../graphics/images/arrow.gif)**查询（Query）****，然后从 **可视化模块查询（Visualization Module Queries）** 选项中选择 **应力线性化（Stress linearization）**。





# 52.3 获取线性化应力结果





您可以沿穿过模型的线获取线性化应力。Abaqus/CAE 将沿该线的应力结果分离为恒定薄膜应力和线性弯曲应力，并以 X–Y 绘图的形式显示结果。

**获取线性化应力结果：**

1. 找到应力线性化选项。从主菜单栏中，选择 ****工具（Tools）**![](../graphics/images/arrow.gif)**查询（Query）**** 或点击 **查询（Query）** 工具栏中的 ![](../graphics/ico_info.png) 工具。出现 **查询（Query）** 对话框。
2. 选择 **应力线性化（Stress linearization）**。出现 **应力线性化（Stress Linearization）** 对话框。
3. 在 **应力线名称（Stress line name）** 字段中，为应力线提供一个名称。此名称将用作线性化结果的前缀。
4. 要保存将生成的 X–Y 数据，请切换 **保存 XY 数据（Save XY data）**。数据将在您的 Abaqus/CAE 会话期间可用。
5. 要将端点和间隔点保存为路径，请切换 **将应力线保存到路径（Save stress line to path）**。应力线的点将作为点列表路径保存，名称与应力线相同，将在您的 Abaqus/CAE 会话期间可用。
6. 通过选择节点或空间中的点或选择保存的路径来选择应力线的端点。**手动（Manual）** 这是默认方法。您可以直接从视口选择节点，或在空间中键入节点标签或点。**直接从视口选择节点：** 点击 **起点（Start）** 和 **终点（End）** 字段右侧的 ![](../graphics/ico_selectBlue.png)，然后在视口中点击所需的节点。节点标签（包括部件实例名称）将出现在应力线的 **起点（Start）** 和 **终点（End）** 点的文本字段中。**键入节点标签或空间中的点：** 1. 在 **起点（Start）** 文本字段中，输入部件实例名称和节点标签或空间中的点的坐标。部件实例名称和节点标签必须采用 `*实例.节点*` 的形式；指定坐标值作为用空格或逗号分隔的 X、Y 和 Z 坐标。如果不知道模型中的部件实例名称，请使用前述方法直接从视口选择节点。或者，您可以选择 ****工具（Tools）**![](../graphics/images/arrow.gif)**查询（Query）**** 或点击 **查询（Query）** 工具栏中的 ![](../graphics/ico_info.png) 工具，并选择 **探针值（Probe values）** 方法来确定实例名称和节点标签（有关更多信息，请参阅 ["理解探测，" 第 51.1 节"](pt05ch51s01.md)）。2. 重复前述步骤以完成 **终点（End）** 文本字段。**从路径（From a path）** 切换 **从路径（From a path）**，并从出现的列表中选择路径名称；您不能使用边列表路径来定义应力线的端点。（有关路径的更多信息，请参阅 [第 48 章，"沿路径查看结果"](pt05ch48.md)。"）Abaqus/CAE 使用保存路径的端点作为应力线的端点。点的定义方式与它们最初在路径中的定义方式相同——节点列表路径提供模型上的节点点，点列表路径或圆形路径提供空间中的点坐标。无论您使用哪种方法来选择端点，Abaqus 都会在视口中高亮显示应力线并标记起点和终点。如果您选择节点点或节点列表路径来定义应力线的端点，视口中的标签指示节点编号。**注意：**如果您在步骤 5 中选择将应力线保存为路径，Abaqus/CAE 始终保存点列表路径——即使您选择了节点、节点标签或节点列表路径来定义应力线。
7. 选择要获取应力结果的模型形状。默认是获取变形模型形状的结果。切换 **未变形（Undeformed）** 以获取未变形模型形状的结果。
8. 指定应力线应被划分的间隔数。在 **应力线上的间隔数（Number of intervals on stress line）** 文本字段中输入大于 0 的正整数。如果您不输入数字，Abaqus 将使用默认的间隔数。
9. 默认情况下，Abaqus/CAE 将线性化应力值（包括应力的所有可用分量和计算的线性化应力不变量）写入名为 `linearStress.rpt` 的文件。如果您不希望写入此报告，您可以切换关闭对话框 **报告（Report）** 区域中的 **写入文件（Write to file）**。您可以通过在 **文件名（File name）** 字段中输入名称或点击 ![](../graphics/ico_fileOpen.png) 并从出现的现有文件列表中进行选择来为报告文件指定新名称。如果您将报告写入现有文件，新数据将默认追加到文件；如果您希望覆盖文件，请切换关闭 **追加到文件（Append to file）**。
10. 点击 **计算（Computations）** 选项卡。
11. 通过切换每个薄膜和弯曲分量来选择要线性化的应力分量。
12. 选择用于计算不变量的弯曲分量。
13. 对于轴对称模型，请在应力线位置输入模型中面中平面曲率的近似值。默认值是 **无限（Infinite）**，表示缺乏曲率。要指定曲率半径，请点击 **指定（Specify）** 并在文本字段中输入数字。
14. 对于轴对称模型，请在应力线位置输入模型中面外曲率的近似值。默认值是 **计算（Compute）**，允许 Abaqus 根据轴对称形状和所选应力线的位置计算半径。要指定曲率半径，请点击 **指定（Specify）** 并在文本字段中输入数字。
15. 对于非轴对称模型，选择 Abaqus 是否应使用曲率校正。如果选择曲率校正，请指定局部坐标系或使用默认的（全局）坐标系；如果您使用局部坐标系进行曲率校正，您也可以使用该局部坐标系来评估应力线方向。
16. 点击 **确定（OK）**。类似于 [图 52–6](pt05ch52hla01.md#usv-stresslin-plot) 中所示的 X–Y 绘图出现在视口中。**图 52–6** 应力线性化结果。![](../graphics/usv-stresslin-plot.png)

# 52.1.1 定义应力线





您可以通过在模型中选择节点、空间中的点或保存的路径来定义要获取线性化应力的截面的端点。您可以通过直接从视口选择节点来选择模型中的节点，也可以通过键入节点标签来选择；您可以通过键入点坐标来定义空间中的点。如果您选择保存的路径，Abaqus/CAE 使用路径端点作为应力线的端点——路径上的其他点将被忽略。（有关路径的更多信息，请参阅 [第 48 章，"沿路径查看结果"](pt05ch48.md)。"）

节点标签是引用部件实例中给定节点的持久方式；也就是说，节点标签不会随模型变形而改变。因此，如果您使用节点标签或保存的节点列表路径指定截面的端点，这些点将同样适用于未变形或变形的模型形状。但是，空间中的点是固定的，与模型无关。例如，与未变形形状上节点位置重合的坐标可能与变形形状上的任何位置都不重合。如果您使用空间坐标、点列表路径或径向路径指定截面的端点，请确保相对于正确的形状指定坐标。边列表路径不能用于定义应力线。

为了获得轴对称模型的最佳结果，所选端点应垂直于模型的内表面和外表面以及中平面（参见 [图 52–1](pt05ch52s01s01.md#usv-stresslin-paths)）。

**图 52–1** 推荐的应力路径。

![](../graphics/stm-anl-stresslin-paths-nls.png)

截面可以从表面延伸到表面，也可以完全位于模型内部。截面也可以穿过多个部件实例。在选择截面的端点时，还应考虑以下准则：
- 避免穿过空间不连续的路径。
- 避免沿分隔不连续的线的路径。
- 在指定截面之前，使用显示组隔离各个区域。

如果截面跨越空间不连续（如部件实例中的孔或部件实例之间的空间），将给出错误消息。

选择截面的端点后，您可以指定线应被划分的段数，也可以接受默认数量。Abaqus/CAE 通过在两个端点之间插值来定义应力线，以沿着直线获得指定数量的等间隔。





# 52.1.2 应力结果的提取





沿应力线的每个点处的应力是通过使用 **结果选项（Result Options）** 对话框中定义的平均标准对唯一的节点应力值进行插值来获得的（有关指定平均标准的更多信息，请参阅 ["控制结果平均，" 第 42.6.6 节"](pt05ch42s04hlb02.md)）。Abaqus/CAE 使用由应力线定义的局部坐标系来获取应力结果。局部 x 轴沿着应力线；它起源于应力线上的第一个点，由连接应力线上第一点和第二点的向量定义。局部 y 轴和 z 轴的确定如下：

**对于二维模型：**

局部 y 轴是通过将全局 z 轴与局部 x 轴进行向量相乘获得的。局部 z 轴与全局 z 轴相同。

**对于三维模型：**

**如果局部 x 轴平行于全局 y 轴：**

如果正局部 x 轴沿着正全局 y 轴，则局部 y 轴将是全局 x 轴的负值。如果正局部 x 轴沿着负全局 y 轴，则局部 y 轴将是全局 x 轴。局部 z 轴是通过将局部 x 轴与局部 y 轴进行向量相乘获得的。

**如果局部 x 轴不平行于全局 y 轴：**

局部 z 轴是通过将局部 x 轴与全局 y 轴进行向量相乘获得的。局部 y 轴是通过将局部 z 轴与局部 x 轴进行向量相乘获得的。

然后对应力进行线性化。非轴对称和轴对称模型的应力线性化计算不同。有关这些计算的更多信息，请参阅 ["应力线性化，" Abaqus Theory Guide 第 2.17.1 节](../stm/stm-link.md#stm-anl-stresslinear)。

Abaqus 以 X–Y 绘图的形式显示 resulting 的线性化应力。为每个应力分量呈现三条独立的曲线，分别表示以下内容：

1. 沿应力线的实际应力分布。
2. 线性化薄膜应力。
3. 线性化薄膜应力与线性化弯曲应力之和。

此外，每个应力分量的线性化应力输出和线性化应力的应力不变量计算被写入报告文件。
