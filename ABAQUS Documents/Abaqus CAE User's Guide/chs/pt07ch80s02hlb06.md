# 80.2.6 自定义视图切割的封盖颜色





在可视化模块以外的模块中，您可以自定义在视图切割上显示切割平面部分时出现的"封盖"。**Cap Color** 选项提供两个自定义封盖颜色的选择：
- 选择 **Specify** 并单击 ![](../graphics/uss-cut-cap-fixed.png) 以固定颜色显示整个封盖。Abaqus/CAE 为切割平面上所有组件显示该颜色，并且不会随着您移动平面或更改颜色编码选择而更改颜色。
- 选择 **Use body color** 以使用模型上每个组件的当前颜色在切割平面上显示。Abaqus/CAE 会随着您重新定位平面或更改颜色编码选择而动态更改切割平面上模型组件的着色。

如果您为会话启用了透明度，Abaqus/CAE 会从视口中隐藏封盖；有关更多信息，请参见["更改透明度，" 第77.3节](pt07ch77s03.md)。此外，根据您选择的文件格式，Abaqus/CAE 在保存或导出视口内容时可能会隐藏封盖。
- 如果您将视口图像保存为文件并选择基于矢量的文件格式，Abaqus/CAE 会在结果文件中隐藏封盖。当您以 **SVG** 格式保存图像时，或当您以 **PS** 或 **EPS** 格式保存图像并从 **PS Options** 对话框或 **EPS Options** 对话框指定 **Vector** 作为 **Image format** 时，Abaqus/CAE 会创建基于矢量的文件。
- 如果您以 VRML 或 3DXML 格式导出视口数据，Abaqus/CAE 会在导出文件中隐藏封盖。对于所有其他导出格式，封盖会包含在导出文件中。

**自定义视图切割的封盖颜色的步骤：**

1. 从主菜单栏中选择****工具**![](../graphics/images/arrow.gif)**View Cut**![](../graphics/images/arrow.gif)**Manager****。出现 **View Cut Manager** 对话框。
2. 单击 **Options**。出现 **View Cut Options** 对话框。
3. 从 **Cap Color** 选项列表中，选择以下之一：
   - 选择 **Specify** 并单击 ![](../graphics/uss-cut-cap-fixed.png)，然后从出现的 **Select Color** 对话框中选择封盖颜色，以显示切割上模型显示部分的选定颜色。有关选择新颜色的更多信息，请参见["自定义颜色，" 第3.2.9节](pt01ch03s02s09.md)。
   - 选择 **Use body color** 以使用每个与视图切割相交的组件的当前颜色编码设置对封盖进行着色。
4. 单击 **OK**。
