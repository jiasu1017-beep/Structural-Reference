# 5.2.9 数值指定视图





你可以绕过视图操作工具并数值指定特定视图。如果你想要在 Abaqus/CAE 会话之间重现特定视图，或者数值指定视图比应用一系列视图操作更简单方便，那么指定视图会很有用。

从主菜单栏中选择****View**![](../graphics/images/arrow.gif)**Specify****来指定视图。

**提示：**你也可以通过双击 3D 罗盘来指定视图。

你可以使用以下方法来指定视图：

****旋转角度****

输入三个角度（![](../graphics/usi_eqn00001.gif)、![](../graphics/usi_eqn00002.gif)、![](../graphics/usi_eqn00003.gif)），分别表示模型视图围绕屏幕或模型 1、2 和 3 轴旋转的角度。旋转按（![](../graphics/usi_eqn00001.gif)、![](../graphics/usi_eqn00002.gif)、![](../graphics/usi_eqn00003.gif)）顺序解释，正角度表示绕轴的右手旋转。如果你之前在使用旋转视图工具时指定了非默认旋转中心（请参阅["旋转视图工具，" 第 5.2.3 节](pt01ch05s02s03.md)），指定的旋转也将围绕此点。你必须选择以下模式之一来应用旋转：
- **关于模型轴增量**。当你选择**关于模型轴增量**时，Abaqus/CAE 简单地将旋转应用到当前视图。[图 5-11](pt01ch05s02s09.md#viw-specify-rotrel) 显示了从等轴测视图应用 90、0、0 的增量模型轴旋转的结果。
**图 5-11** 指定增量模型轴旋转角度。
![](../graphics/viw-specify-rotrel.png)
- **关于屏幕轴增量**。屏幕 *X* 轴是水平的，*Y* 轴是垂直的，*Z* 轴是垂直于屏幕的。屏幕轴的原点是相机目标。在大多数情况下，相机目标与视口中心重合，但某些视图操作方法可以移动相机目标（有关更多信息，请参阅["相机模式和视图术语，" 第 5.1.1 节](pt01ch05s01s01.md)）。当你选择**关于屏幕轴增量**时，Abaqus/CAE 简单地将旋转应用到当前视图。[图 5-12](pt01ch05s02s09.md#viw-specify-rotscreen) 显示了从等轴测视图应用 90、0、0 的增量屏幕轴旋转的结果。
**图 5-12** 指定增量屏幕轴旋转角度。
![](../graphics/viw-specify-rotscreen.png)
- **从 (0,0,1) 的总旋转**。当你选择**从 (0,0,1) 的总旋转**时，Abaqus/CAE 首先将视图旋转到默认位置（沿 3 轴向下看，1 和 2 轴在屏幕平面中的视图），然后应用所需的旋转。[图 5-13](pt01ch05s02s09.md#viw-specify-rotabs) 显示了从等轴测视图应用 90、0、0 的总旋转的结果。
**图 5-13** 指定总旋转角度。
![](../graphics/viw-specify-rotabs.png)

**视点**

当你选择**视点**时，你输入三个值，分别代表观察者的 1、2 和 3 位置。Abaqus/CAE 从模型原点到你指定的位置构建一个向量，并旋转模型的视图，使该向量指向屏幕外。[图 5-14](pt01ch05s02s09.md#viw-specify-viewpoint) 显示了应用视点 1, 1, 1（等轴测视图）和视点 1, 0, 0 的结果。

**图 5-14** 指定视点。

![](../graphics/viw-specify-viewpoint.png)

当你使用**视点**方法指定视图时，你也可以指定**上向量**。Abaqus/CAE 定位模型的视图，使该向量指向上方。[图 5-15](pt01ch05s02s09.md#viw-specify-upvector) 显示了将上向量 0, 1, 0 和 0, 1, 0 应用到等轴测视图的结果。**上向量**不能等于**视点**向量。

**图 5-15** 指定**上向量**。

![](../graphics/viw-specify-upvector.png)

**缩放**

输入代表放大倍数的值。大于 1 的值会在视口中扩展模型的视图；例如，**缩放因子**为 2 会使模型的视图大小加倍。介于 0 和 1 之间的值会在视口中收缩模型的视图；例如，值为 0.25 会将模型的视图收缩到原始大小的四分之一。该值必须大于零。

你必须选择以下方法之一来应用缩放：
- **绝对**。当你选择**绝对**时，Abaqus/CAE 首先将视图拟合到视口，然后应用所需的**缩放因子**。
- **相对**。当你选择**相对**时，Abaqus/CAE 将**缩放因子**应用到当前视图。

**平移**

输入 Abaqus/CAE 用于按指定水平和垂直距离**平移**模型视图的值。Abaqus/CAE 相对于视图在视口中的当前位置移动视图。你输入的值对应于视口尺寸的分数；第一个值代表水平运动，第二个值代表垂直运动。正的第一个值将模型的视图向视口右边缘移动，正的第二个值将模型的视图向视口顶部移动。例如，如果视口宽 200 mm，高 100 mm，你在**平移视口分数 (X,Y)** 字段中输入 0.5, 0.1，Abaqus/CAE 会将模型的视图从当前位置向右移动 100 mm，向下移动 10 mm。
