# 55.1.1 为后续会话保存自定义





默认情况下,曲线图显示自定义仅在当前会话期间保持,但如果您想将它们应用于后续会话,可以保存到文件。Abaqus/CAE提供以下方法来保留曲线图显示自定义:

**保存到显示选项文件的自定义**

 从主菜单栏选择 ****File**![](../graphics/images/arrow.gif)**Save Display Options**** 可将所有曲线图状态相关和曲线图状态无关的自定义选项保存为显示选项。显示选项存储在名为`abaqus_v6.14.gpr`的文件中,Abaqus/CAE在启动时自动加载此文件中的设置作为会话期间的新默认设置。对于曲线图状态自定义,将显示选项保存到文件使您能够存储设置,例如渲染样式、规定条件和各种类型基准几何的可见性,以及网格、单元和节点标签的显示。

更多信息,请参阅 ["保存您的显示选项设置," 第76.16节](pt07ch76hla15.md)。

**保存到XML文件、模型数据库或输出数据库的自定义**

 从主菜单栏选择 ****File**![](../graphics/images/arrow.gif)**Save Session Objects**** 可将选定的曲线图状态相关和曲线图状态无关的自定义选项保存到XML文件、模型数据库或输出数据库。使用此方法时,您可以保存所有自定义选项,也可以仅保存与一个或多个曲线图状态相关的自定义选项。例如,您可以将您在公共曲线图选项和云图曲线图选项中所做的自定义保存到文件。

如果您将自定义保存到模型数据库或输出数据库,则在打开文件时将使用这些设置。从XML文件加载自定义时,这些自定义将成为会话的新设置。您可以通过选择 ****File**![](../graphics/images/arrow.gif)**Load Session Objects**** 从文件加载设置。

更多信息,请参阅 ["管理会话对象和会话选项," 第9.9节](pt02ch09s08.md)。
