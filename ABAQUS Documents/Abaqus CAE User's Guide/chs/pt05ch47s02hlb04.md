# 47.2.4 从所有活动的自由体切片读取X–Y数据









### 47.2.4 从所有活动的自由体切片读取X--Y数据

您可以从会话中定义的所有活动的自由体切片读取*X–Y*数据。可用于选择的数据包括：
- 力数据、弯矩数据或两者；以及
- 对于您选择的实体，合力大小以及1、2和3方向的各个分量。

然后，Abaqus/CAE通过在所有相关时间帧执行自由体计算来计算*X–Y*数据对，并创建显示力或弯矩随时间变化关系的*X–Y*数据对象。为您选择的每个实体和分量组合创建一个单独的*X–Y*数据对象；如果您同时打开两个实体选项和所有四个分量选项，Abaqus/CAE会为会话中每个活动的自由体切片创建八个*X–Y*数据对象。您可以在**自由体切片管理器**中激活和停用自由体切片；有关更多信息，请参见["显示、隐藏和高亮自由体切片，" 第67.4节](pt06ch67s04.md)。

**要从自由体读取X–Y数据：**

1. 找到**自由体**选项。从主菜单栏选择****工具**![](../graphics/images/arrow.gif)**XY数据**![](../graphics/images/arrow.gif)**创建****，然后选择**自由体**。出现**来自自由体的XY数据**对话框。
2. 点击**活动步/帧**以更改Abaqus/CAE从中读取*X--Y*数据的步或帧。有关更多信息，请参见["激活和停用步和帧，" 第42.4.1节](pt05ch42s02s01.md)。
3. 从**实体**选项中，切换**力**、**弯矩**或这两者以显示所选实体或实体的*X--Y*数据。
4. 从**分量**选项中，切换您要包含在*X--Y*数据集中的向量分量。选项包括合力大小以及每个方向的各个分量值。
5. 要评估和显示数据，点击**绘图**。当前视口中出现您已在对话框中配置的数据的*X--Y*图。
6. 要将您配置的数据保存为一个或多个*X--Y*数据对象，点击**保存**。*X--Y*数据对象仅在当前会话期间保存。**注意：**要绘制您保存的*X--Y*数据，请从主菜单栏选择****工具**![](../graphics/images/arrow.gif)**XY数据**![](../graphics/images/arrow.gif)**绘图****，然后从菜单中选择*X--Y*数据。
7. 完成后，点击**关闭**以关闭对话框。
