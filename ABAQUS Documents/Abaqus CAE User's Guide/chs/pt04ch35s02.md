# 35.2 理解参考表示





参考表示是不用于分析的部件或部件子集（一或多个单元）的替代表示。参考表示保留原始实体模型几何体，与Sketcher中的参考几何体一样，您可以使用来自参考表示的实体作为工具来构建更简单的部件版本。参考表示最常用于构建中面模型。当与偏移面工具结合使用来为中面模型创建面时，参考表示与新面之间的偏移计算还允许Abaqus/CAE自动计算中面模型中的壳厚度和单元偏移。

当您为实体部件的一个或多个单元分配中面属性时，Abaqus/CAE会创建参考表示。所选单元从部件的活动表示中移除并添加到参考表示中。参考表示默认显示在Part模块中，颜色为半透明棕色。您不能更改参考表示的颜色，但可以使用主工具栏中与可见对象工具一起的**Show Reference Representation**工具![](../graphics/ico_referenceRepToggle.png)在所有模块中切换其开启或关闭。您也可以使用显示选项切换参考表示的半透明度开启或关闭（更多信息，请参见["控制边缘可见性"，第76.3节](pt07ch76hla02.md)）。

您可以将参考表示与形状创建工具或Geometry Edit工具集中的工具结合使用，以帮助在活动表示中创建新的几何实体。例如，您可以在参考表示中偏移面，或者您可以从参考表示中选择一个平面作为草图平面，以使用拉伸方法创建新的壳面。您可以使用参考表示中的几何体来定义分区——从参考表示中选择边缘和顶点作为分割单元的点和平面——或创建基准实体。您还可以使用参考表示几何体在Assembly模块中定位实体和定义约束。

您不能编辑参考表示内的几何体，也不能向其分配分析属性（如载荷、边界条件和相互作用）。当您创建参考表示时，在创建参考表示之前应用于实体单元的任何分析属性现在将与空区域关联。您可以编辑空区域以将分析属性与活动表示一部分的几何体（实体几何体或您为中面模型创建的壳几何体）关联。或者，您可以抑制或删除分析属性。如果您不对受影响的分析属性进行任何更改，则在Job模块中提交作业时Abaqus/CAE将显示警告消息。
