# 15.15.9 定义方程约束





您可以通过在「Edit Constraint」对话框中输入数据来创建方程约束。方程的项由系数组成，该系数应用于集合中每个节点上的自由度。有关方程的详细信息，请参见Abaqus Analysis User's Guide第35.2.1节的"线性约束方程"。

**创建方程约束的步骤：**

1. 从主菜单栏选择 ****Constraint**![](../graphics/images/arrow.gif)**Create****。**提示：**您也可以使用「相互作用」模块工具箱中的![](../graphics/ico_constraint.png)工具创建方程约束。
2. 在出现的「Create Constraint」对话框中，执行以下操作：1. 为约束命名。有关命名对象的更多信息，请参见"使用基本对话框组件"，第3.2.1节。2. 从「Type」列表中选择「Equation」，然后点击「Continue」。约束编辑器会出现。
3. 在编辑器的表格中，为方程中的每一项输入一行数据。方程必须至少有两项。点击![](../graphics/ico_tipSmall.png)获取表格中数据与所需方程之间关系的描述。每一行应包含以下信息：- 系数。- 现有集合的名称。（有关创建集合的信息，请参见第73章，"集合和表面工具集"。）您可以在表格的第一行输入包含一个或多个点的集合。后续集合只能包含一个点。- 自由度。- 您将应用约束的坐标系的ID。您可以接受默认坐标系或选择现有基准坐标系。如果所需的基准坐标系不存在，您可以使用Datum工具集创建它。（有关更多信息，请参见"创建基准坐标系"，第62.9节。）要确定坐标系的ID，请从主菜单栏选择 ****Tools**![](../graphics/images/arrow.gif)**Query****。有关更多信息，请参见"获取模型的一般信息"，第71.2.2节。
4. 点击「OK」保存方程定义并关闭约束编辑器。
