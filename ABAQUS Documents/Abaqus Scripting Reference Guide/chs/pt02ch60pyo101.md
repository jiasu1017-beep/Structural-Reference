# 60.100 Trs 对象

Trs 对象用于定义时间历史粘弹性分析的温度-时间偏移。

**访问**

```
materialApi.materials()[*name*].viscoelastic().trs()
materialApi.materials()[*name*].viscosity().trs()
```

### 60.100.1 Trs(...)

此方法创建一个 Trs 对象。

**路径**

```
materialApi.materials()[*name*].viscoelastic().Trs
materialApi.materials()[*name*].viscosity().Trs
```

**原型**

```
odb_Trs&
Trs(const odb_String& definition,
    const odb_SequenceSequenceDouble& table);
```

**必需参数**

无。

**可选参数**

*definition*

一个 odb_String，指定偏移函数的定义。可能的值为"WLF"、"ARRHENIUS"和"USER"。默认值为"WLF"。

*table*

一个 odb_SequenceSequenceDouble，指定如下所述的项目。默认值为空序列。

此参数仅在 *definition*="WLF" 时有效。
