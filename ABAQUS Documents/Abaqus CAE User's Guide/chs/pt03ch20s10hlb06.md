# 20.10.8 草绘椭圆







使用 Sketcher 工具箱中的椭圆工具基于圆心、轴端点和一个任意点绘制椭圆，该点与第一个轴的距离决定了第二个轴的长度，如下所示：

![](../graphics/ske-ellipsebtn-nls.png)

在草绘时，您应该注意定位点，因为这会影响网格的质量。草图中的圆心点和第一个轴点会成为您正在创建的零件的顶点。反过来，当您在网格模块中对模型进行网格划分时，Abaqus/CAE 会将这些顶点转换为完全约束的种子，并将节点放置在其位置。有关如何随后移动顶点的信息，请参见["拖动 Sketcher 对象，"第 20.17.1 节](pt03ch20s14hlb01.md)。

您无法通过移动其顶点来编辑椭圆的大小。要编辑椭圆的大小，请创建半径尺寸，然后编辑该尺寸。椭圆的半径尺寸包括长半径和短半径。

**草绘椭圆：**

1. 从 Sketcher 工具箱中，选择椭圆工具![](../graphics/ico_sketchEllipse.png)。有关 Sketcher 工具箱中工具的图表，请参见["Sketcher 工具，"第 20.4.1 节](pt03ch20s04s01.md)。Abaqus/CAE 在提示区域显示提示以指导您完成该过程。
2. 点击椭圆中心的所需位置。
3. 将光标移动到椭圆第一个轴的端点。Abaqus/CAE 显示一个可以使用当前光标位置定义长轴和旋转来创建的椭圆。
4. 点击一个点以定义一个轴的端点。您选择的点可以是长轴或短轴端点。**提示：**如有必要，您可以使用提示区域中的文本框输入椭圆中心和第一个轴端点的精确坐标。有关精确定义几何的更多信息，请参见["指定精确几何，"第 20.6 节](pt03ch20s06.md)。Abaqus/CAE 设置椭圆的旋转和第一个轴的端点。
5. 点击一个点以定义第二个轴的长度。您选择的点定义了从第一个轴到第二个轴末端的距离。该点不需要位于第二个轴上，也不能位于第一个轴上，因为这会创建长度为 0 的第二个轴。Abaqus/CAE 创建椭圆。
6. 要创建更多椭圆，重复从第 2 步开始的上述步骤。
7. 完成创建椭圆后，执行以下操作之一：- 在 Abaqus/CAE 窗口中的任意位置点击鼠标按钮 2。- 在 Sketcher 工具箱中选择任何其他工具。- 点击提示区域中的取消按钮![](../graphics/afxI_cancel.png)。
