# 59.2 了解投影方法





附件工具集提供三种用于创建附件点的工具。这些工具提供了方便的点生成方法，例如沿边缘跟随、从文件读取每个点的坐标，或沿一条线间隔排列点。然而，在某些情况下，在一个位置定义附件点然后将点投影到所需面上可能会更容易。因此，Abaqus/CAE 提供投影选项作为在面上定位附件点的方法。

附件工具集提供以下技术通过将点投影到所需面来创建附件点：
- 将每个点投影到其最近的面。
- 选择投影向量的起点和终点。Abaqus/CAE 沿向量投影每个点，并在它首次与面相交处创建附件点。
