### 概述

橡树岭国家实验室（ORNL）本构模型：
- 允许使用核标准NEF 9-5T"一级高温核系统组件设计指南和程序"中规定的规则进行塑性和蠕变计算；
- 旨在用于在相对高温下建模304和316不锈钢；
- 只能与金属塑性模型（仅线性运动硬化）和/或金属蠕变定律的应变硬化形式结合使用；和
- 在["ORNL本构理论，" Abaqus理论指南第4.3.8节](../stm/stm-link.md#stm-mat-ornl)中详细描述。

### 与塑性结合使用

Abaqus/Standard中的ORNL本构模型基于1981年3月版核标准NEF 9-5T和1986年10月版（对本构模型进行了广泛修订）。该模型从原始材料状态到完全循环状态添加了塑性屈服面的各向同性硬化。最初，材料假定根据原始应力-应变曲线的双线性表示进行运动硬化。如果发生应变反转或蠕变应变达到0.2%，屈服面各向同性膨胀至用户定义的第十循环应力-应变曲线。随后根据第十循环应力-应变曲线的双线性表示进行运动硬化。

您必须通过塑性模型定义指定原始屈服应力和硬化，并通过线性弹性模型定义指定响应的弹性部分。您单独指定第十循环屈服应力和硬化值。每个温度下的屈服应力应通过给出在零塑性应变处和一个附加非零塑性应变点的值来定义，从而给出恒定的硬化率（线性加工硬化）。

### 与蠕变结合使用

ORNL本构模型假定蠕变使用应变硬化公式。它在应变反转发生时引入辅助硬化规则。提供细节的算法在["ORNL本构理论，" Abaqus理论指南第4.3.8节](../stm/stm-link.md#stm-mat-ornl)中呈现。它只能在蠕变行为由应变-硬化幂律定义时使用。

| **输入文件用法：** | 在同一材料数据块中使用以下两个选项： |
| --- | --- |
|  | ``` [*CREEP](../key/key-link.md#usb-kws-mcreep), LAW=STRAIN [*ORNL](../key/key-link.md#usb-kws-mornl) ``` |

#### 蠕变期间屈服面的平移

ORNL公式还可以导致屈服面中心在蠕变期间平移，以用于后续塑性增量；此行为通过两个可选用户定义参数定义。

##### 指定运动偏移的饱和率

您可以指定A，即核标准第4.3.3-3节方程式（15）定义的由蠕变应变引起的运动偏移的饱和率。默认值为0.3。设置A = 0.0以使用1986年修订版的标准。

| **输入文件用法：** | ``` [*ORNL](../key/key-link.md#usb-kws-mornl), A=*A* ``` |
| --- | --- |

| **Abaqus/CAE用法：** | 属性模块：材料编辑器：****子选项**![](../graphics/images/arrow.gif)**Ornl****：****运动偏移的饱和率：** *A* |
| --- | --- |

##### 指定运动偏移率

您可以指定H，即相对于蠕变应变的运动偏移率（核标准第4.3.2-1节方程式（7））。设置H = 0.0以使用1986年修订版的标准。如果不指定H的值，则根据1981年修订版标准的第4.3.3-3节确定。

| **输入文件用法：** | ``` [*ORNL](../key/key-link.md#usb-kws-mornl), H=*H* ``` |

### 初始条件

当我们需要研究已经经历了一些加工硬化的材料的行为时，可以提供初始等效塑性应变值来指定对应于加工硬化状态的屈服应力。更多详情见["非弹性行为，" 第23.1.1节"](pt05ch23s01abo20.md)。还可以为背应力张量α~0~提供初始值，以包含应变诱导的各向异性。更多信息见["Abaqus/Standard和Abaqus/Explicit中的初始条件，" 第34.2.1节"](pt07ch34s02aus116.md)。对于更复杂的情况，可以通过用户子程序[`HARDINI`](../sub/sub-link.md#sub-xsl-hardini)定义初始条件。

### 单元

ORNL本构模型可用于Abaqus/Standard中包含力学行为（具有位移自由度的单元）的任何单元。

### 输出

除了Abaqus/Standard中可用的标准输出标识符（["Abaqus/Standard输出变量标识符，" 第4.2.1节"](pt02ch04s02abv01.md)），与蠕变（["率相关塑性：蠕变和肿胀，" 第23.2.4节"](pt05ch23s02abm20.md)）和运动硬化塑性模型（["承受循环加载的金属模型，" 第23.2.2节"](pt05ch23s02abm18.md)）相关的变量可用于ORNL本构模型。
