# 4.3 Specifying additional volume in a fluid cavity







**Product: **Abaqus/Standard  

**Benefits: **You can now specify an additional volume for a fluid cavity in an Abaqus/Standard analysis.

**Description: **This feature, which was previously available only in Abaqus/Explicit, is now available in Abaqus/Standard as well. You can specify an additional volume that will be added to the actual volume of the cavity.
