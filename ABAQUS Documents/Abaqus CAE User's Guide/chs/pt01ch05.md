# 5 操作视图和控制透视





本章描述视图选项、视图操作工具、3D 罗盘和透视工具，所有这些都控制着在视口中创建视图的相机。视图选项允许你在两种相机模式之间切换并数值控制其某些属性。视图操作工具和 3D 罗盘控制相机以在视图中定位、定向和放大对象；你也可以选择自定义视图（如前视图和后视图），以及定义自己的视图。透视工具控制 Abaqus/CAE 是否使用透视显示模型；使用透视可以为三维模型提供更真实的外观。涵盖以下主题：
