# 13.7.2 合并和切割独立和从属零件实例





合并所选的零件实例会产生一个新的零件实例和一个新的零件。如果您合并独立零件实例，生成的零件实例也是独立的。类似地，如果您合并从属零件实例，生成的零件实例也是从属的。最后，如果您合并独立和从属零件实例的组合，生成的零件实例是从属的。

当您合并网格几何和/或孤儿网格元素时，生成的零件实例始终是孤儿网格零件，并且始终是从属的。当您合并包含几何和孤儿网格节点及元素的零件的几何和网格时，生成的零件实例是包含几何和孤儿节点及元素的混合体，并且始终是从属的。

切割所选零件实例的几何也会产生一个新的零件实例和一个新的零件。独立和从属零件实例几何合并的讨论适用于独立和从属零件实例几何的切割；但是，零件实例中的孤儿网格元素不能被切割或用于切割另一个实例的几何。
