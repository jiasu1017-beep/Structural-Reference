# 12.7.2 浏览和修改材料行为





材料编辑器窗口顶部的所选行为列表显示组成当前材料的行为和子选项；当您添加和删除行为时，列表会更新。下图显示了如果定义了包含基于应力的失效限值的弹塑性材料，列表的外观：

![](../graphics/prp-list-nls.png)

使用所选行为列表，您可以按如下方式添加、删除或更改材料：

**添加材料行为**

从所选行为列表下方的菜单中选择定义材料所需的行为。当您选择行为时，其名称会出现在列表中，与该行为关联的参数和数据会出现在编辑器窗口底部的数据区域中。子选项出现在相应主要行为的下方，并缩进以表明其从属地位。

**删除材料行为**

在所选行为列表中，点击您要删除的行为或子选项；然后点击行为列表右下角附近的![](../graphics/ico_delete.png)图标。此过程会从行为列表和材料定义中删除该行为。如果您删除列表中其下方显示子选项的行为，这些子选项也会被删除。

**更改材料参数或数据**

在所选行为列表中，点击您要更改其数据的行为。当与该行为关联的参数和数据出现在窗口底部的数据区域中时，进行所需的更改。
