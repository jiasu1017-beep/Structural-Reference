# 32.11.2 拖链单元库







**产品：** Abaqus/Standard

##### **参考资料**

- ["拖链，" 第32.11.1节](pt06ch32s11alm56.md)
- [*DRAG CHAIN](../key/key-link.md#usb-kws-mdragchain)
- [*RIGID SURFACE](../key/key-link.md#usb-kws-mrigidsurf)

### 概述

本节提供Abaqus/Standard中可用的拖链单元的参考。

### 单元类型

| DRAG2D | 二维拖链，用于仅研究水平运动的情况 |
| --- | --- |
|  |

| DRAG3D | 三维拖链 |
| --- | --- |
|  |

##### 激活的自由度

DRAG2D: 1, 2

DRAG3D: 在第一个节点：1, 2, 3。在第二个节点：1, 2, 3, 4, 5, 6。

##### 额外求解变量

无。

### 需要的节点坐标

DRAG2D: 管道连接节点在水平面上的(*X*, *Y*)坐标。

DRAG3D: 两个节点的(*X*, *Y*, *Z*)坐标。

### 单元属性定义

| **输入文件用法：** | 使用以下选项定义滑动时的水平长度和摩擦极限： |
| --- | --- |
|  | ``` [*DRAG CHAIN](../key/key-link.md#usb-kws-mdragchain) ``` 对于DRAG3D单元，使用以下选项定义海床： ``` [*RIGID SURFACE](../key/key-link.md#usb-kws-mrigidsurf) ``` 刚性表面必须平坦且平行于全局(*X*, *Y*)平面。 |

### 基于单元的载荷

无。

### 单元输出

| S11 | 拖链在平行于海床的平面中支持的力的水平分量。 |
| --- | --- |

| S12 | DRAG3D单元中拖链的垂直分量。 |
| --- | --- |

| E11 | DRAG2D单元的拖链水平长度。DRAG3D单元中在海底（未悬吊）的链条长度。 |
| --- | --- |

| E12 | 拖链的方向（与全局X轴的角度）。 |
| --- | --- |

### 单元关联的节点

DRAG2D: 一个节点在链条连接到管道的位置。

DRAG3D: 两个节点。第一个节点是链条连接到管道的节点；第二个节点是定义海床的刚性体的"参考节点"。
