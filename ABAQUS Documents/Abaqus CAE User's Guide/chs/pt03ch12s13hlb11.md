# 12.13.15 创建垫片截面





垫片截面用于定义位于结构组件之间的薄密封组件的截面属性。请参见[Abaqus Analysis User's Guide的第32.6.6节，"使用垫片行为模型直接定义垫片行为"](../usb/usb-link.md#usb-elm-egasketbehavior)。[第32章，"垫片"](pt04ch32.md)描述了涉及垫片的分析的整个建模过程。

**指定垫片截面属性：**

1. 从主菜单栏，选择****Section**![](../graphics/images/arrow.gif)**Create****。出现**Create Section**对话框。
   **提示：**您也可以在**Section Manager**中点击**Create**，或在Property模块工具箱中选择创建截面工具![](../graphics/ico_section.png)。
2. 输入截面名称。有关命名对象的更多信息，请参见["使用基本对话框组件，"第3.2.1节"](pt01ch03s02s01.md)。
3. 选择**Other**作为截面**Category**，选择**Gasket**作为截面**Type**，然后点击**Continue**。出现垫片截面编辑器。
4. 为垫片截面选择一种材料。如有需要，点击![](../graphics/ico_material.png)创建材料；有关更多信息，请参见["创建或编辑材料，"第12.7.1节"](pt03ch12s07hlb01.md)。
5. 指定**Stabilization stiffness**，用于稳定并非所有节点都有支撑的垫片元素，例如那些延伸到相邻组件之外的元素。
   - 切换**Use default**以使用厚度方向初始压缩刚度10^9倍的默认值。此默认值通常是合适的。
   - 切换**Specify**，并输入所需稳定刚度的值，单位为应力（[FL2](../popups/usb-int-iconventions-unitsym.md)）。
6. 指定垫片的**Initial thickness**。
   - 切换**Use nodal coordinates**以从节点坐标获取初始垫片厚度。
   - 切换**Specify**，并输入初始垫片厚度的值。
7. 为**Initial gap**输入值，这是生成压力所需的垫片闭合量。
8. 为**Initial void**输入值，这是垫片中的层间空间。
9. 根据垫片元素类型，为**Cross-sectional area, width, or out-of-plane thickness**输入值。对于二维和三维链接元素，应给出元素的横截面积。对于轴对称链接元素和三维线元素，应给出元素的宽度。对于一般二维元素，需要面外厚度。对于三维面积元素，不需要额外量来定义元素几何。
10. 点击**OK**保存更改并关闭垫片截面编辑器。
