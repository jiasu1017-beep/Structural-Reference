# 7.9 其他动力分析过程

> **摘要**
> 本节简要介绍Abaqus中可用的其他动力分析过程——线性模态动力学和非线性动力学。

## 7.9.1 线性模态动力学

Abaqus/Standard中有几种其他线性动力分析过程，它们采用模态叠加技术。与`*MODAL DYNAMIC`（在时域中计算响应）不同，这些过程在频域中提供结果，可以更深入地了解结构的行为。

这些过程的完整说明参见《Abaqus Analysis User's Guide》第6.3节"动态应力/位移分析"。

### 稳态动力学

`*STEADY STATE DYNAMICS`选项计算结构在用户指定的频率范围内受到谐激励时的响应幅值和相位。典型示例包括：

- 汽车发动机悬置在发动机工作转速范围内的响应。
- 建筑物中的旋转机械。
- 航空发动机上的部件。

### 响应谱

`*RESPONSE SPECTRUM`选项提供了当结构固定点受到动力运动时峰值响应（位移、应力等）的估计值。固定点的运动称为"基础运动"；典型示例是引起地面运动的地震事件。通常，在需要峰值响应估计值用于设计目的时使用该方法。

### 随机响应

`*RANDOM RESPONSE`选项预测系统受到随机连续激励时的响应。激励用功率谱密度函数以统计方式表示。随机响应分析的示例包括：

- 飞机对湍流的响应。
- 结构对噪声的响应，例如喷气发动机发出的噪声。

## 7.9.2 非线性动力学

如前所述，`*MODAL DYNAMIC`过程仅适用于线性问题。当需要非线性动力响应时，必须直接对运动方程进行积分。在Abaqus/Standard中，运动方程的直接积分使用隐式动力学过程（`*DYNAMIC`）。使用此过程时，质量矩阵、阻尼矩阵和刚度矩阵被组装，并在每个时间点求解动力平衡方程。由于这些操作计算量大，直接积分动力学的计算成本比模态方法更高。

由于Abaqus/Standard中的非线性动力学过程使用隐式时间积分，因此它适用于非线性结构动力学问题，例如，突然事件引发动力响应的情况（如碰撞），或结构响应涉及大量能量通过塑性或粘性阻尼耗散的情况。在此类研究中，最初重要的高频响应会被模型中的耗散机制快速阻尼掉。

非线性动力学分析的另一种方法是Abaqus/Explicit中的显式动力学过程。如第2章"Abaqus基础"中所讨论的，显式算法通过模型一次一个元素地传播解作为应力波。因此，它最适用于应力波效应重要且模拟的事件时间较短（通常小于一秒）的应用。

与显式算法相关的另一个优点是，它比Abaqus/Standard更容易模拟不连续非线性，如接触和失效。大型的、高度不连续的问题通常更容易用Abaqus/Explicit建模，即使响应是准静态的。显式动力学分析在第9章"非线性显式动力学"中进一步讨论。
