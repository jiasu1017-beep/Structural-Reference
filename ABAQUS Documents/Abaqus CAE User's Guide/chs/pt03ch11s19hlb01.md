# 11.19.2 选择新零件的建模空间





使用**创建零件**对话框顶部的**建模空间**单选按钮来选择您正在创建的零件的建模空间。Abaqus/CAE在建模过程中保留零件的建模空间；例如，建模空间决定零件模块中哪些工具可用以及网格模块中哪些单元可用。您可以通过在模型树中右键单击零件并从出现的菜单中选择**编辑**来更改零件的建模空间。

建模空间是指零件可以占据的空间，而不是零件本身。因此，您可以在三维建模空间中创建零件，但使用拓扑二维的壳或线框特征来构建它。在Abaqus/CFD模型中，只能创建三维零件。在Abaqus/Standard或Abaqus/Explicit模型中，新零件的建模空间可以设置为以下任一：

**三维**

Abaqus/CAE将零件置于三维空间中。

**二维平面**

Abaqus/CAE将零件置于平面二维空间中。

**轴对称**

Abaqus/CAE将零件置于轴对称二维空间中。如果您创建轴对称可变形零件，Abaqus/CAE允许您在模型中包含扭转自由度。

**选择新零件的建模空间：**

1. 从**创建零件**对话框顶部，选择所需的**建模空间**单选按钮。
2. 完成选项选择后，单击**继续**关闭**创建零件**对话框。草图编辑器启动，您开始绘制新零件基础特征的轮廓。
