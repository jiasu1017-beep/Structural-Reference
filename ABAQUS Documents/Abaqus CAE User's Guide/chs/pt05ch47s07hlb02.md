# 47.8.2 自定义X–Y曲线图标题的位置






### 47.8.2 自定义X–Y曲线图标题的位置

使用**Plot Title Options**中的**Position**选项将X–Y曲线图标题放置在视口中的特定位置。

**要自定义X–Y曲线图标题的位置：**

1. 找到**Position**选项。选择****Options**![](../graphics/images/arrow.gif)**XY Options**![](../graphics/images/arrow.gif)**Plot Title****；或点击![](../graphics/ico_xyPlotTitleOptions.png)，该图标位于可视化模块工具箱的*X--Y*绘图工具中。从出现的对话框中点击**Area**选项卡。
2. 切换**Inset**以在*X--Y*曲线图中显示曲线图标题图例。如果**Inset**关闭，Abaqus/CAE会将曲线图标题和*X--Y*曲线图保持在视口的独立区域中。
3. 选择以下两种方法之一来定位曲线图标题：**Auto-align**此选项使您可以将曲线图标题定位在视口中的九个位置之一：中心、四个角落中的任何一个，或四条边中任意一条的中心。展开**Alignment**列表，然后选择列表中以图形方式显示的选项之一。**Manual**此选项使您可以将曲线图标题定位在视口中的特定*X--Y*位置。拖动**X**和**Y**滑块可以在视口中水平或垂直移动标题。当这两个值都等于`0`时，曲线图标题位于视口的左下角；当两个值都等于`100`时，曲线图标题位于视口的右上角。
