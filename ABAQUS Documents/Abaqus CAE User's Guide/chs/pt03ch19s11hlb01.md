# 19.11.1 创建和编辑共同执行







共同执行是两个分析作业的协同仿真执行。您可以在共同执行中指定作业参数来创建两个分析作业。创建共同执行后，您只能单独编辑各个分析作业中的作业参数。

作业使用与["理解分析作业，"第 19.2 节](pt03ch19s02.md)中所述相同的功能相互同步执行。有关此分析技术的更多信息，请参见["结构对结构协同仿真，"《Abaqus 分析用户手册》第 17.3.1 节](../usb/usb-link.md#usb-anl-acosimabqtoabq)和["流体对结构和对流传热协同仿真，"《Abaqus 分析用户手册》第 17.3.2 节](../usb/usb-link.md#usb-anl-acosimcfdtoabq)。

**创建或编辑共同执行：**

1. 从主菜单栏中，选择****共同执行**![](../graphics/images/arrow.gif)**创建****或****共同执行**![](../graphics/images/arrow.gif)**编辑**![](../graphics/images/arrow.gif)***共同执行名称*****。出现**编辑共同执行**对话框。**提示：**您也可以通过点击模块工具箱中的![](../graphics/ico_Coexecution.png)图标或点击**共同执行管理器**中的**创建**来创建共同执行。
2. 在**名称**文本字段中输入新的共同执行的名称。您指定的名称必须遵守操作系统的文件命名规则。
3. 如有需要，在**描述**文本字段中输入共同执行的描述。描述存储在模型数据库和输出数据库中，您可以使用此描述来帮助管理您的共同执行。
4. 在**模型**表中，从**模型名称**列的列表中选择当前模型数据库中要与共同执行关联的模型。选定模型中使用的分析产品以及每个模型的默认作业名称显示在表中。
5. 如有需要，点击**作业名称**列并编辑作业名称。
6. 使用**提交**、**常规**、**内存**、**并行化**和**精度**选项卡页面来定义各个作业编辑器中的初始作业参数设置。创建共同执行后，共同执行编辑器中的作业参数将无法编辑；您必须编辑各个作业。按以下各节所述定义初始作业参数设置：- ["配置作业提交属性，"第 19.8.2 节](pt03ch19s08hlb02.md)- ["指定常规作业设置，"第 19.8.6 节](pt03ch19s08hlb06.md)- ["控制作业内存设置，"第 19.8.7 节](pt03ch19s08hlb07.md)- ["控制作业并行执行，"第 19.8.8 节](pt03ch19s08s01.md)- ["控制作业精度，"第 19.8.9 节](pt03ch19s08hlb08.md)
7. 如有需要在**通信超时**字段中输入不同的值，以指定 Abaqus/CAE 在未收到耦合分析的任何通信后将终止的时间。默认值为 10 分钟。
8. 点击**确定**。
