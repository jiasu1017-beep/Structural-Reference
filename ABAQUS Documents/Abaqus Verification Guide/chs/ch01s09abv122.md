# 1.9.4 单个连接器选项测试







**产品：**Abaqus/Standard  Abaqus/Explicit  

### I. 弹性和阻尼连接器行为

### 单元测试

CONN2D2    CONN3D2

### 问题描述

这些验证案例测试连接器行为选项的性能，这些选项在其他验证问题中不常使用。本节重点介绍通过连接器弹性和连接器阻尼选项的弹簧或阻尼器行为。这些验证案例中使用CARTESIAN和CARDAN连接。

通过施加带有连接器单元载荷的集中载荷并获得对应于分析解决方案的相对位移（对于连接器弹性）或速度（对于连接器阻尼）来验证行为选项。包含等效的非连接器单元进行比较。

对于CARTESIAN和CARDAN连接，测试以下连接器弹性案例：

1. 连接器弹性，其中指定弹性行为的相对运动连接器分量设置为1（CARTESIAN）或4（CARDAN），场变量依赖数量设置为2，具有以下依赖设置：1. 温度=10，场变量1=1.0，场变量2=0.5 2. 温度=90，场变量1=2.0，场变量2=1.0
2. 连接器弹性，其中指定弹性行为的相对运动连接器分量设置为1（CARTESIAN）或4（CARDAN），场变量依赖数量设置为1，具有以下依赖设置的非线性行为：1. 温度=10，场变量1=1.0 2. 温度=90，场变量1=2.0
3. 连接器弹性，其中指定弹性行为的相对运动连接器分量设置为1（CARTESIAN）或4（CARDAN），包含弹性定义中构成运动的分量依赖，非线性行为定义（无相关温度或场变量依赖）
4. 连接器弹性，其中指定弹性行为的相对运动连接器分量设置为1（CARTESIAN）或4（CARDAN），包含弹性定义中相对位置的分量依赖，非线性行为和周期性模型定义（无相关温度或场变量依赖）
5. 连接器弹性，其中连接器弹性数据定义中包含的场变量依赖数量设置为2（耦合）1. 场变量1=1.0，场变量2=0.5 2. 场变量1=2.0，场变量2=1.0

类似地，对于CARTESIAN和CARDAN连接，测试以下连接器阻尼案例：

1. 连接器阻尼，其中指定阻尼行为的相对运动连接器分量设置为1（CARTESIAN）或4（CARDAN），场变量依赖数量设置为2，具有以下依赖设置：1. 温度=10，场变量1=1.0，场变量2=0.5 2. 温度=90，场变量1=2.0，场变量2=1.0
2. 连接器阻尼，其中指定阻尼行为的相对运动连接器分量设置为1（CARTESIAN）或4（CARDAN），场变量依赖数量设置为1，具有以下依赖设置的非线性行为：1. 温度=10，场变量1=1.0 2. 温度=90，场变量1=2.0
3. 连接器阻尼，其中指定阻尼行为的相对运动连接器分量设置为1（CARTESIAN），包含阻尼定义中相对位置的分量依赖，非线性行为和周期性模型定义（无相关温度或场变量依赖）
4. 连接器阻尼，其中连接器阻尼数据定义中包含的场变量依赖数量设置为1（耦合），具有以下依赖设置：1. 场变量1=1.0 2. 场变量1=2.0

**模型：**

模型由一系列具有相关连接器行为的独立2节点连接器单元组成。

### 结果与讨论

Abaqus匹配所有相关依赖设置的分析解决方案。

### VI. 条件位置相关连接器行为

### 单元测试

CONN2D2    CONN3D2

### 问题描述

本节重点介绍使用连接器止动和连接器锁定定义停止和锁定行为。CARTESIAN和CARDAN连接都有使用。

通过两步载荷历史验证行为选项。在步骤1中，用连接器载荷施加集中载荷，使得结果连接器运动将超过连接器止动或锁定的规定运动限制。在步骤2中，载荷方向反转以确认停止或锁定行为。包含等效的非连接器单元进行比较。在Abaqus/Standard测试中，在第三步执行线性摄动静态步骤。

对于CARTESIAN连接，测试以下连接器锁定案例：

1. 连接器锁定，其中锁定准则基于的分量编号设置为1，当满足锁定准则时锁定所有相对运动分量。
2. 连接器锁定，其中锁定准则基于的分量编号设置为3，当满足锁定准则时锁定的分量编号设置为2。

对于CARDAN连接，测试以下连接器锁定案例：

1. 连接器锁定，其中锁定准则基于的分量编号设置为4，当满足锁定准则时锁定的分量编号设置为4（带旋转准则）或6（带力矩准则）
2. 连接器锁定，其中锁定准则基于的分量编号设置为4，当满足锁定准则时锁定的分量编号设置为4（带旋转准则）或6（带力矩准则）

对于CARTESIAN和CARDAN连接，测试以下连接器止动案例：

1. 连接器止动，其中定义连接器止动的分量编号设置为1（CARTESIAN）或4（CARDAN）
2. 连接器止动，其中定义连接器止动的分量编号设置为2（CARTESIAN）或6（CARDAN）

**模型：**

模型由一系列具有相关连接器行为的独立2节点连接器单元组成。

### 结果与讨论

Abaqus匹配所有相关设置的分析解决方案。

### IX. 连接器驱动选项

### 单元测试

CONN2D2    CONN3D2

### 问题描述

本节重点介绍使用可用的相对运动连接器分量进行驱动行为。在这些验证案例中使用CARTESIAN和CARDAN连接。

通过在载荷历史的第一步中在连接器节点之间引起相对位移，然后固定运动并施加集中载荷以验证没有运动发生，来验证变量值规定固定在其步骤开始时值的可用相对运动连接器分量。

通过向连接器单元施加相对速度或加速度并获得对应于规定条件的分析解决方案的相对位移和连接器载荷，来验证规定速度或加速度历史的可用相对运动连接器分量。

通过在连接器节点之间施加相对位移来验证与用户子程序[`DISP`](../sub/sub-link.md#sub-xsl-disp)中规定的变量相关的非零量级的可用相对运动连接器分量。

在Abaqus/Standard测试中，在最后一步执行线性摄动静态程序。

测试上面列出的每个案例。

**模型：**

模型由一系列具有相关连接器行为的独立2节点连接器单元组成。

### 结果与讨论

Abaqus匹配所有案例的预期解决方案。
