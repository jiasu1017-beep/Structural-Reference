# 41.3.2 增量





**作业诊断**对话框中的**增量**页面显示Abaqus在分析期间使用的增量控制设置和生成的增量。**增量**页面仅在您在**作业历史**树中高亮显示步骤时可用。

**状态**表的内容取决于您是在查看Abaqus/Standard还是Abaqus/Explicit分析步骤的诊断信息。对于Abaqus/Standard步骤，表显示每个增量以及尝试次数、严重不连续迭代和平衡迭代次数、迭代总数、步骤时间和增量大小。对于Abaqus/Explicit步骤，表显示所有摘要增量以及记录了诊断信息的任何其他增量。每个增量的关键单元、该单元的稳定时间增量、经过的步骤时间、总时间和动能也会列出。有关Abaqus/Explicit中增量诊断的更多信息，请参见《Abaqus分析用户指南》第6.3.3节"显式动态分析"。

如果选择一列数据并单击**绘制所选列**，Abaqus/CAE会使用增量号（第一列）作为X轴，所选列作为Y轴创建X-Y图。

**注意：**您无法绘制关键单元号。
