# 15.3 理解相互作用





您可以使用「相互作用」模块来定义以下类型的相互作用：

**通用接触**

通用接触相互作用允许您使用单个相互作用来定义模型中许多或所有区域之间的接触。通用接触也可用于在耦合Eulerian-Lagrangian分析中定义Lagrangian体与Eulerian材料之间的接触（请参见「在Eulerian-Lagrangian模型中定义接触」，第28.3节）。通常，通用接触相互作用是为包含所有外部面、特征边以及——在Abaqus/Explicit中——解析刚性表面、基于梁和桁架的边和Eulerian材料边界的所有包容表面定义的。为了细化接触域，您可以包括或排除特定的表面对。通用接触相互作用中使用的表面可以跨越模型的许多不连接区域。属性（如接触属性、表面属性和接触公式）作为接触相互作用定义的一部分进行分配，但独立于接触域定义，这允许您使用一组表面进行域定义，使用另一组表面进行属性分配。有关创建此类相互作用的详细说明，请参见「定义通用接触」，第15.13.1节。

通用接触相互作用和表面对表面或自接触相互作用可以在同一分析中一起使用。在分析过程中，一个步骤中只能有一个通用接触相互作用处于活动状态。

更多信息，请参见「接触相互作用分析概述」，Abaqus Analysis User's Guide第36.1.1节；「在Abaqus/Standard中定义通用接触相互作用」，Abaqus Analysis User's Guide第36.2.1节；「在Abaqus/Explicit中定义通用接触相互作用」，Abaqus Analysis User's Guide第36.4.1节；以及「Eulerian分析」，Abaqus Analysis User's Guide第14.1.1节。Abaqus/CAE不支持分配罚刚度比例因子。此外，在Abaqus/CAE中，基于节点的表面不能用于通用接触相互作用。

**表面对表面接触、自接触和压力渗透**

表面对表面接触相互作用描述两个可变形表面之间或可变形表面与刚性表面之间的接触。自接触相互作用描述单个表面上不同区域之间的接触。有关创建这些类型相互作用的详细说明，请参见「定义表面对表面接触」，第15.13.7节；「定义自接触」，第15.13.8节；以及「使用接触和约束检测」，第15.16节。更多信息，请参见「在Abaqus/Standard中定义接触对」，Abaqus Analysis User's Guide第36.3.1节，以及「在Abaqus/Explicit中定义接触对」，Abaqus Analysis User's Guide第36.5.1节。

如果您的模型包含复杂几何和大量接触相互作用，您可能需要自定义控制接触算法的变量以获得经济有效的解决方案。这些控制适用于高级用户，应非常小心地使用。更多信息，请参见「接触控制编辑器」，第15.9.4节。

压力渗透相互作用允许您模拟渗入参与表面对表面接触的两个表面之间的流体压力。流体压力垂直作用于表面。您必须创建一个表面对表面接触相互作用来指定压力渗透的主表面和从表面。形成接头的两个体都可以是可变形的（如螺纹连接器的情况），或者一个可以是刚性的（如使用软垫圈作为较硬结构之间的密封的情况）。压力渗透相互作用只能用于Abaqus/Standard分析。有关创建压力渗透相互作用的详细说明，请参见「定义压力渗透」，第15.13.16节。更多信息，请参见「压力渗透加载」，Abaqus Analysis User's Guide第37.1.7节。

**流体腔**

流体腔相互作用允许您为模型中选择并分配属性给液体或气体填充的流体腔。流体腔选择包括一个参考点和封闭该腔的表面。属性在流体腔相互作用属性中定义（更多信息，请参见「理解相互作用属性」，第15.4节）。您可以在Abaqus/Standard或Abaqus/Explicit分析的初始步骤中定义流体腔相互作用。流体腔相互作用在分析的所有步骤中保持不变；在初始步骤之后，您不能修改或停用它。有关创建流体腔相互作用的详细说明，请参见「定义流体腔相互作用」，第15.13.11节。

**流体交换**

流体交换相互作用允许您定义流体从腔到环境或从两个腔之间的流动。要创建流体交换相互作用，您必须首先为每个腔选择一个现有的流体腔相互作用（与环境交换为一个，两个腔之间交换为两个）。然后您可以选择或创建一个流体交换相互作用属性（更多信息，请参见「理解相互作用属性」，第15.4节）并设置有效的交换面积。有关创建流体交换相互作用的详细说明，请参见「定义流体交换相互作用」，第15.13.12节。

**XFEM裂纹扩展**

XFEM裂纹扩展相互作用允许您激活或停用使用扩展有限元方法创建的裂纹的扩展。有关创建此类相互作用的详细说明，请参见「停用和激活XFEM裂纹扩展」，第31.3.5节。

**模型更改**

模型更改相互作用允许您在分析过程中移除和重新激活单元。您可以在除静态、Riks过程和线性扰动过程之外的所有Abaqus/Standard分析过程中使用模型更改相互作用。有关创建此类相互作用的详细说明，请参见「定义模型更改相互作用」，第15.13.13节。有关移除和重新激活单元的更多信息，请参见「单元和接触对的移除和重新激活」，Abaqus Analysis User's Guide第11.2.1节。

**循环对称（仅限Abaqus/Standard）**

循环对称使您能够通过仅分析模型的单个重复扇区来以相当大的计算成本降低来建模整个360结构。您只能在初始步骤中创建循环对称相互作用。创建循环对称相互作用后，它适用于整个分析历史。如果您在频率步骤中停用循环对称相互作用，Abaqus/CAE会评估该步骤的所有可能节点直径。有关创建此类相互作用的详细说明，请参见「定义循环对称」，第15.13.19节。有关Abaqus中循环对称的更多信息，请参见「分析表现出循环对称的模型」，Abaqus Analysis User's Guide第10.4.3节。

**弹性基础（仅限Abaqus/Standard）**

弹性基础允许您对表面上的分布式支撑的刚度效应进行建模，而无需实际建模支撑的详细信息。您只能在初始步骤中创建弹性基础相互作用。一旦激活弹性基础，您就不能在后续分析步骤中将其停用。有关创建此类相互作用的详细说明，请参见「定义基础」，第15.13.20节。更多信息，请参见「单元基础」，Abaqus Analysis User's Guide第2.2.2节。

**空腔辐射（仅限Abaqus/Standard）**

空腔辐射相互作用描述封闭腔中的热辐射传递。Abaqus/CAE中提供了两种空腔辐射模型：完全隐式定义和近似。完整版本可用于二维、三维和轴对称模型中无变形的热传递。它可以包括开放或封闭空腔，并考虑对称性和表面遮挡，但不支持空腔内的表面运动。有关创建此类相互作用的详细说明，请参见「定义空腔辐射相互作用」，第15.13.21节。

空腔辐射近似使用表面辐射相互作用来定义。您可以在任何热传递分析中近似空腔辐射，无论是否有变形。但是，近似空腔辐射只能用于三维模型中的封闭空腔。该近似将空腔视为温度等于整个表面平均温度的黑体封闭腔。在这些有限条件下，近似空腔辐射可以节省相当大的计算成本。有关创建此类相互作用的详细说明，请参见「定义表面辐射相互作用」，第15.13.24节。

有关两种类型的空腔辐射的更多信息，请参见「空腔辐射」，Abaqus Analysis User's Guide第41.1.1节。

**热膜条件**

膜条件相互作用定义由周围流体的对流引起的加热或冷却。Abaqus/CAE中提供了两种类型的膜条件相互作用：表面膜条件定义模型表面的对流，集中膜条件定义来自节点或顶点的对流。您只能在热传递、完全耦合热应力或耦合热电步骤中定义膜条件相互作用。有关定义这些类型的相互作用的详细说明，请分别参见「定义表面膜条件相互作用」，第15.13.22节，和「定义集中膜条件相互作用」，第15.13.23节。更多信息，请参见「热载荷」，Abaqus Analysis User's Guide第34.4.4节。

**向周围环境的辐射和来自周围环境的辐射**

辐射相互作用描述由于辐射向非反射环境的热传递。Abaqus/CAE中提供了两种类型的辐射相互作用：表面辐射相互作用描述与非凹表面的热传递，集中辐射相互作用描述来自节点或顶点的辐射。您只能在热传递、完全耦合热应力或耦合热电步骤中定义辐射相互作用。有关创建这些类型的相互作用的详细说明，请分别参见「定义表面辐射相互作用」，第15.13.24节，和「定义集中辐射相互作用」，第15.13.25节。更多信息，请参见「热载荷」，Abaqus Analysis User's Guide第34.4.4节。

**Abaqus/Standard到Abaqus/Explicit共仿真**

对于Abaqus/Standard到Abaqus/Explicit共仿真，您必须指定共仿真的接口区域（数据交换区域）和耦合方案（时间增量过程和数据交换频率）。在每个模型中，您创建一个Standard-Explicit共仿真相互作用来定义共仿真行为；一个模型中只能有一个Standard-Explicit共仿真相互作用处于活动状态。每个共仿真相互作用中的设置必须在Abaqus/Standard模型和Abaqus/Explicit模型中相同。

Standard-Explicit共仿真相互作用只能在通用静态、隐式动态或显式动态步骤中创建。该相互作用仅在创建它的步骤中有效，不会传播到后续步骤。有关创建此类相互作用的详细说明，请参见「定义Standard-Explicit共仿真相互作用」，第15.13.14节。更多信息，请参见「结构对结构共仿真」，Abaqus Analysis User's Guide第17.3.1节。

**流固耦合仿真（介于Abaqus/CFD与Abaqus/Standard或Abaqus/Explicit之间）**

对于流固耦合（FSI），您必须指定共仿真的接口边界（数据交换区域）和耦合步骤周期。在每个模型中，您创建一个流固耦合仿真相互作用来定义行为；在特定模型中只能有一个共仿真相互作用处于活动状态。

在流体模型（Abaqus/CFD）中，FSI共仿真相互作用只能在流动步骤中创建。在结构模型（Abaqus/Standard或Abaqus/Explicit）中，FSI共仿真相互作用只能在隐式动态、显式动态或热传递步骤中创建。

有关创建此类相互作用的详细说明，请参见「定义流固耦合仿真相互作用」，第15.13.15节。更多信息，请参见「流体对结构和共轭热传递共仿真」，Abaqus Analysis User's Guide第17.3.2节。

**入射波**

入射波相互作用对外部声波源引起的入射波载荷进行建模。有关创建此类相互作用的详细说明，请参见「定义入射波」，第15.13.18节。更多信息，请参见「声学和冲击载荷」，Abaqus Analysis User's Guide第34.4.6节。

**声阻抗**

声阻抗指定声学介质压力与声学-结构界面处法向运动之间的关系。有关创建此类相互作用的详细说明，请参见「定义声阻抗」，第15.13.17节。更多信息，请参见「声学和冲击载荷」，Abaqus Analysis User's Guide第34.4.6节。

**致动器/传感器（仅限Abaqus/Standard）**

致动器/传感器相互作用对传感器和致动器的组合进行建模，因此允许对控制系统组件进行建模。目前，此类相互作用仅允许在一点进行传感和驱动。有关创建此类相互作用的详细说明，请参见「定义致动器/传感器相互作用」，第15.13.26节。

相互作用定义及其可选的关联属性用于定义相互作用的基本方面，但用户必须提供用户子程序[`UEL`](../sub/sub-link.md#sub-xsl-uel)来提供关于驱动如何依赖于传感器读数的特定公式。当您在Job模块中创建分析作业时，您需要指定包含用户子程序的文件名。

**警告：**此功能仅供高级用户使用。在所有但最简单测试示例中的使用将需要用户/开发人员进行大量编码。在继续之前，应阅读「用户定义单元」，Abaqus Analysis User's Guide第32.15.1节。

致动器/传感器相互作用仅适用于Abaqus/Standard分析。更多信息，请参见「用户子程序和工具」，Abaqus Analysis User's Guide第18.1节。
