# 70.7.4 使用拉伸/扫描方法分割单元





您可以使用扫描工具创建复杂的三维分割，将二维轮廓（称为扫描轮廓）扫描穿过三维零件或零件实例。使用扫描工具是一个两步操作。首先，通过选择要扫描的边来定义扫描轮廓，然后选择扫描所选边要遵循的路径（称为扫描路径）。您可以选择两种方法之一来定义扫描路径：
- **沿方向拉伸**
- **沿边扫描**

这两种方法如下图所示。

**使用拉伸/扫描方法分割单元：**

1. 从主菜单栏中，选择****工具**![](../graphics/images/arrow.gif)**分割****。将出现**创建分割**对话框。Abaqus/CAE在提示区域显示提示以指导您完成过程。**提示：**您也可以使用 ![](../graphics/ico_ptnCellSweepEdges.png) 工具通过拉伸/扫描方法分割单元，该工具位于模块工具箱的分割单元工具中。有关工具箱中分割工具的图示，请参见["使用分割工具集," 第70.2节](pt06ch70s02.md)。
2. 从对话框顶部的**类型**单选按钮中，选择**单元**。**方法**列表显示您可以用来分割单元的方法。
3. 从方法列表中，选择**拉伸/扫描边**。
4. 如果零件或装配体包含多个单元，请选择要分割的单元。您可以使用拖动选择、**[Shift]**+点击和**[Ctrl]**+点击的组合来选择多个单元进行分割。**提示：**如果您无法选择所需的单元，可以使用**选择**工具栏更改选择行为。有关更多信息，请参见["使用选择选项," 第6.3节](pt01ch06s03.md)。Abaqus/CAE高亮显示选定的单元。
5. 要创建扫描轮廓，请选择要扫描穿过单元的边。这些边必须都属于同一个零件、位于同一个平面上并且彼此相连。Abaqus/CAE高亮显示选定的边。
6. 从提示区域中的按钮，单击以下方法之一来定义扫描路径：
   - 使用**沿方向拉伸**方法通过沿直边或基准轴（扫描路径）沿选定方向拉伸所选边（扫描轮廓）来分割单元。扫描路径必须垂直于包含扫描轮廓的平面。分割沿选定方向无限延伸穿过选定的单元。通过沿方向拉伸扫描轮廓来分割单元如下图所示：![](../graphics/par-cell-sweep1-nls.png)Abaqus/CAE高亮显示扫描轮廓。
   - 使用**沿边扫描**方法通过沿直边或曲边（扫描路径）扫描所选边（扫描轮廓）来分割单元。扫描路径必须从包含扫描轮廓的平面开始，其切线必须垂直于该平面。结果分割将仅延伸至扫描路径的长度。通过沿边扫描扫描轮廓来分割单元如下图所示：![](../graphics/par-cell-sweep2-nls.png)Abaqus/CAE高亮显示扫描路径。
7. 在提示区域中，单击**创建分割**。Abaqus/CAE创建分割。
