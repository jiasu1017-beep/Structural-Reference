# 3.9 在 Abaqus/CAE 中切换模型实例的上下文





**产品：** Abaqus/CAE

**优势：** 现在，您可以使用 Model Tree 切换模型实例及其子零件实例的上下文，从而更轻松地访问原始模型或零件进行编辑。

**描述：** 您可以从 Model Tree 中选择一个模型实例或子零件实例，然后切换到从中创建所选实例的原始模型或零件的上下文。

**Abaqus/CAE 使用：**
```
装配模块或网格模块：
    从 Model Tree 中选择模型或子零件实例，然后单击鼠标按钮 3 并从菜单中选择 **Switch to part/model context**。
```
