# 40.4 理解可视化模块中的工具集





可视化模块中的工具集提供了对数据显示的额外控制。除非另有说明，本节描述的工具集仅用于后处理来自输出数据库的模型和结果数据；只有选定的工具集用于当前模型数据库中的模型数据。可视化模块中提供了以下工具集：
- 颜色编码工具集允许您自定义单个元素的边缘和填充颜色。更多信息，请参见"在可视化模块中为节点或元素着色"，第77.6节。
- 坐标系工具集允许您创建用于后处理的局部坐标系。更多信息，请参见"在后处理期间创建坐标系"，第42.8节。
- 创建场输出工具集允许您对输出数据库中可用的场输出执行操作。更多信息，请参见"创建新的场输出"，第42.7节。
- 显示组工具集允许您有选择地从模型或输出数据库中绘制一个或多个项目。更多信息，请参见第78章，"使用显示组显示模型的子集"。
- 自由体工具集允许您创建和删除自由体切割，在视口中显示或隐藏它们，并自定义其外观的多个方面。更多信息，请参见第67章，"自由体工具集"。
- 作业诊断工具集允许您访问在Abaqus/Standard分析作业期间写入输出数据库的诊断信息。更多信息，请参见第41章，"查看诊断输出"。
- 路径工具集允许您指定通过模型的路径，您可以在该路径上获取和查看X-Y数据。更多信息，请参见第48章，"沿路径查看结果"。
- 查询工具集允许您获取有关模型的信息，包括来自当前模型数据库的数据或来自输出数据库的数据。更多信息，请参见第50章，"在可视化模块中查询模型"。
- 流工具集允许您显示流线以研究流体流动分析中的速度或涡量。更多信息，请参见第74章，"流工具集"。
- 视图切割工具集允许您创建穿过模型的切割，以便您可以可视化模型内部或选定截面。此功能可用于来自当前模型数据库或输出数据库的模型数据。更多信息，请参见第80章，"切割模型"。
- X-Y数据工具集允许您创建和操作X-Y数据对象。更多信息，请参见第47章，"X-Y绘图"。
