# 80.2.3 允许多个自由体切割





在可视化模块中，您可以通过切换 **Allow for multiple cuts** 同时在模型上显示多个视图切割。当您选择此选项时，Abaqus/CAE 会显示会话中所有选择了 **Show** 复选框的视图切割的切割表面。您只能重新定位活动视图切割。实际上，您可能希望依次激活和重新定位每个视图切割，以将它们放置在您想要研究的位置。此外，您显示的切割不会相互作用；例如，您无法仅显示位于两个平行平面视图切割之间的模型部分。

**注意：**Abaqus/CAE 不支持同时显示多个基于 XFEM 的视图切割组件。

**显示多个视图切割的步骤：**

1. 从主菜单栏中选择****工具**![](../graphics/images/arrow.gif)**View Cut**![](../graphics/images/arrow.gif)**Manager****。出现 **View Cut Manager**，其中列出在当前会话期间创建的所有视图切割。活动视图切割左侧的 **Show** 列中出现一个勾选标记。
2. 切换 **Allow for multiple cuts**。Abaqus/CAE 显示每个选择了 **Show** 复选框的视图切割的切割处位置，并且 Abaqus/CAE 隐藏所有自由体切割并禁用 **Model** 列中 ![](../graphics/ico_freeBodyDiagram.png) 图标下的复选框。您可以通过切换其 **Show** 复选框来显示或隐藏当前显示中的各个视图切割。
