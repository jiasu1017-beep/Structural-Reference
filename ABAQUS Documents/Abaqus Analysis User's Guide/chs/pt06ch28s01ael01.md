### 基于单元的载荷

### 分布热通量

分布热通量适用于具有温度自由度的单元。如 ["热载荷，" 第 34.4.4 节](pt07ch34s04aus123.md) 中所述进行指定。

**载荷 ID (*DFLUX)：**  BF**Abaqus/CAE 载荷/相互作用：**  **体热通量****单位：**  [JL⁻³T⁻¹](../popups/usb-int-iconventions-unitsym.md)**描述：**  单位体积的 热体通量。

**载荷 ID (*DFLUX)：**  BFNU**Abaqus/CAE 载荷/相互作用：**  **体热通量****单位：**  [JL⁻³T⁻¹](../popups/usb-int-iconventions-unitsym.md)**描述：**  通过用户子程序 [`DFLUX`](../sub/sub-link.md#sub-xsl-dflux) 提供幅值的单位体积非均匀热体通量。

**载荷 ID (*DFLUX)：**  S1**Abaqus/CAE 载荷/相互作用：**  **表面热通量****单位：**  [JL⁻²T⁻¹](../popups/usb-int-iconventions-unitsym.md)**描述：**  进入连接第一端（节点 1）的单位面积表面热通量。

**载荷 ID (*DFLUX)：**  S2**Abaqus/CAE 载荷/相互作用：**  **表面热通量****单位：**  [JL⁻²T⁻¹](../popups/usb-int-iconventions-unitsym.md)**描述：**  进入连接第二端（节点 2 或节点 3）的单位面积表面热通量。

**载荷 ID (*DFLUX)：**  S1NU**Abaqus/CAE 载荷/相互作用：**  不支持**单位：**  [JL⁻²T⁻¹](../popups/usb-int-iconventions-unitsym.md)**描述：**  进入连接第一端（节点 1）的单位面积非均匀表面热通量，幅值通过用户子程序 [`DFLUX`](../sub/sub-link.md#sub-xsl-dflux) 提供。

**载荷 ID (*DFLUX)：**  S2NU**Abaqus/CAE 载荷/相互作用：**  不支持**单位：**  [JL⁻²T⁻¹](../popups/usb-int-iconventions-unitsym.md)**描述：**  进入连接第二端（节点 2 或节点 3）的单位面积非均匀表面热通量，幅值通过用户子程序 [`DFLUX`](../sub/sub-link.md#sub-xsl-dflux) 提供。

### 薄膜条件

薄膜条件适用于具有温度自由度的单元。如 ["热载荷，" 第 34.4.4 节](pt07ch34s04aus123.md) 中所述进行指定。

**载荷 ID (*FILM)：**  F1**Abaqus/CAE 载荷/相互作用：**  不支持**单位：**  [JL⁻²T⁻¹Θ⁻¹](../popups/usb-int-iconventions-unitsym.md)**描述：**  连接第一端（节点 1）的薄膜系数和热沉温度（单位为 ![](../graphics/usb_eqn00123.gif)）。

**载荷 ID (*FILM)：**  F2**Abaqus/CAE 载荷/相互作用：**  不支持**单位：**  [JL⁻²T⁻¹Θ⁻¹](../popups/usb-int-iconventions-unitsym.md)**描述：**  连接第二端（节点 2 或节点 3）的薄膜系数和热沉温度（单位为 ![](../graphics/usb_eqn00123.gif)）。

**载荷 ID (*FILM)：**  F1NU**Abaqus/CAE 载荷/相互作用：**  不支持**单位：**  [JL⁻²T⁻¹Θ⁻¹](../popups/usb-int-iconventions-unitsym.md)**描述：**  连接第一端（节点 1）的非均匀薄膜系数和热沉温度（单位为 ![](../graphics/usb_eqn00123.gif)），幅值通过用户子程序 [`FILM`](../sub/sub-link.md#sub-xsl-film) 提供。

**载荷 ID (*FILM)：**  F2NU**Abaqus/CAE 载荷/相互作用：**  不支持**单位：**  [JL⁻²T⁻¹Θ⁻¹](../popups/usb-int-iconventions-unitsym.md)**描述：**  连接第二端（节点 2 或节点 3）的非均匀薄膜系数和热沉温度（单位为 ![](../graphics/usb_eqn00123.gif)），幅值通过用户子程序 [`FILM`](../sub/sub-link.md#sub-xsl-film) 提供。

辐射条件适用于具有温度自由度的单元。如 ["热载荷，" 第 34.4.4 节](pt07ch34s04aus123.md) 中所述进行指定。

**载荷 ID (*RADIATE)：**  R2**Abaqus/CAE 载荷/相互作用：**  **表面辐射****单位：**  [无量纲](../popups/usb-int-iconventions-unitsym.md)**描述：**  连接第二端（节点 2 或节点 3）的发射率和热沉温度（单位为 ![](../graphics/usb_eqn00123.gif)）。

分布阻抗适用于具有声压自由度的单元。如 ["声学和冲击载荷，" 第 34.4.6 节](pt07ch34s04aus125.md) 中所述进行指定。

**载荷 ID (*IMPEDANCE)：**  I1**Abaqus/CAE 载荷/相互作用：**  不支持**单位：**  [无](../popups/usb-int-iconventions-unitsym.md)**描述：**  定义连接第一端（节点 1）阻抗的阻抗属性名称。

**载荷 ID (*IMPEDANCE)：**  I2**Abaqus/CAE 载荷/相互作用：**  不支持**单位：**  [无](../popups/usb-int-iconventions-unitsym.md)**描述：**  定义连接第二端（节点 2 或节点 3）阻抗的阻抗属性名称。

### 分布电流密度

分布电流密度适用于耦合热电单元。如 ["耦合热电分析，" 第 6.7.3 节](pt03ch06s07at22.md) 中所述进行指定。

**载荷 ID (*DECURRENT)：**  CS2**Abaqus/CAE 载荷/相互作用：**  **表面电流****单位：**  [CL⁻²T⁻¹](../popups/usb-int-iconventions-unitsym.md)**描述：**  连接第二端（节点 2 或节点 3）的电流密度。
