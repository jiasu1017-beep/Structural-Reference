# 54.3.4 为自由体切割报告选择选项和阈值





您可以创建会话中所有活动自由体切割的合力和力矩的表格报告。您可以通过切换 **自由体切割管理器（Free Body Cut Manager）** 中它们的 **显示（Show）** 复选框来激活或抑制单个自由体切割；有关更多信息，请参阅 ["显示、隐藏和突出显示自由体切割，" 第 67.4 节"](pt06ch67s04.md)。Abaqus/CAE 相对于全局坐标系报告合力矩数据，而不考虑为自由体切割或视图切割选择的 **分量分辨率（Component Resolution）** 选项设置。

您可以为当前帧或多个帧生成报告。您还可以为报告中的力和力矩数据指定自定义阈值；小于这些阈值的力和力矩在报告中报告为零。选择适当的阈值使您能够产生更清晰的报告数据；例如，将 `1.234E-17` 等小报告值替换为 `0` 使报告数据更容易阅读。力和力矩值的默认阈值是 `1E-006`。

**为自由体切割报告选择选项和阈值：**

1. 找到自由体切割报告的选项。从主菜单栏中，选择 ****报告（Report）**![](../graphics/images/arrow.gif)**自由体切割（Free Body Cut）****。
2. 选择 Abaqus/CAE 从中读取自由体切割数据的帧。- 选择 **指定（Specify）** 使用当前步骤/帧。要更改当前步骤或帧，请点击 ![](../graphics/ico_stepSmall.png)。有关更多信息，请参阅 ["选择结果步骤和帧，" 第 42.3 节"](pt05ch42s01.md)。- 选择 **所有活动步骤/帧（All active steps/frames）** 使用所有活动步骤/帧。点击 **活动步骤/帧（Active Steps/Frames）** 更改 Abaqus/CAE 从中读取自由体切割数据的步骤或帧。有关更多信息，请参阅 ["激活和停用步骤和帧，" 第 42.4.1 节"](pt05ch42s02s01.md)。
3. 从 **文件（File）** 选项中，指定报告文件名。具体说明请参阅 ["指定报告文件名，" 第 54.4 节"](pt05ch54hla01.md)。
4. 从 **输出格式（Output Format）** 选项中，指定报告数据的呈现格式。具体说明请参阅 ["控制报告布局、宽度、格式和坐标系，" 第 54.5 节"](pt05ch54hla02.md)。
5. 如果需要，从 **阈值（Thresholds）** 选项中指定力的阈值和力矩的阈值。Abaqus/CAE 将小于这些阈值的力和力矩值报告为零。

要生成报告，请点击 **应用（Apply）**。完成后，点击 **取消（Cancel）** 关闭对话框。

# 54.7 格式化报告值





您可以指定表格数据报告中呈现的有效数字位数（对于 X–Y 数据报告、场输出报告和探针值报告）或小数位数（对于自由体切割报告）。对于 X–Y 数据报告、场输出报告和探针值报告，默认有效数字位数是 6；对于自由体切割报告，默认小数位数是 3。更多位数提供更高精度但占用更多空间并增加表格宽度；所使用的有效数字或小数位数决定表格每列的宽度。此外，您可以为数据值选择以下数字格式之一：

**自动（Automatic）**

非常大的值和非常小的值以科学计数法表示。所有其他值均以无指数形式表示，并使用指定的有效数字位数。此格式适用于 X–Y 数据报告、场输出报告和探针值报告。

**工程（Engineering）**

大于或等于 1000（或小于或等于 0.001）的值表示为 1 到 999 之间的数字乘以 10 的 n 次方，其中 n 是 3 的倍数（例如，`20.5E+03` 或 `17.76E+06`）。所有其他值使用指定的有效数字位数表示。

**固定（Fixed）**

所有值均以不使用指数值的方式表示。当您选择此格式时，数字 501,000 在图例中显示为 `501000.00`。此数字格式仅适用于自由体切割报告，如果您更改 **小数位数（Decimal places）** 选项，您的图例值可能会以不同方式显示。

**科学（Scientific）**

所有值表示为 1 到 10 之间的数字乘以 10 的适当次方（例如，`2.05E+04` 或 `1.776E+07`）。

默认格式对于 X–Y 数据报告、场输出报告和探针值报告是 **工程（Engineering）**，对于自由体切割报告是 **科学（Scientific）**。

**格式化报告值：**

1. 找到 **输出格式（Output Format）** 选项。**对于 X–Y 数据或场输出的报告：** 从主菜单栏中，选择 ****报告（Report）**![](../graphics/images/arrow.gif)**XY**** 或 ****报告（Report）**![](../graphics/images/arrow.gif)**场输出（Field Output）****；然后在出现的对话框中点击 **设置（Setup）** 选项卡。**输出格式（Output Format）** 选项位于页面中央。**对于自由体切割的报告：** 从主菜单栏中，选择 ****报告（Report）**![](../graphics/images/arrow.gif)**自由体切割（Free Body Cut）****。**输出格式（Output Format）** 选项位于页面中央。**对于探针值的报告：** 从主菜单栏中，选择 ****工具（Tools）**![](../graphics/images/arrow.gif)**查询（Query）****，然后在出现的对话框中点击 **探针值（Probe values）**。出现 **探针值（Probe Values）** 对话框。将光标移到当前视口中的模型或 X–Y 绘图上探测，然后点击鼠标按钮 1 将感兴趣的值存储在对话框中。完成后，点击 **写入文件（Write to File）**。出现 **报告探针值（Report Probe Values）** 对话框；**输出格式（Output Format）** 选项位于对话框中央。
2. 指定报告中数值的精度级别：- 对于 X–Y、场输出或探针值报告，请在 **有效数字位数（Number of significant digits）** 文本字段中指定此值。- 对于自由体切割报告，请在 **小数位数（Decimal places）** 文本字段中输入此值。您也可以点击适用字段旁边的箭头，直到出现所需的数字。
3. 点击 **数字格式（Number Format）** 按钮，并从出现的列表中选择您选择的数字格式。

要生成报告，请配置对话框中的剩余选项；然后点击 **应用（Apply）**（对于 X–Y、场输出或自由体切割报告）或 **确定（OK）**（对于探针值报告）。完成后，点击 **取消（Cancel）** 关闭对话框。

# 54.5 控制报告布局、宽度、格式和坐标系





对于 X–Y 数据、场输出或探针值的表格报告，您可以限制表格的宽度；例如，限制为您打印机支持的字符数。

X–Y 数据报告可以包含一个或多个 X–Y 数据对象。同样，场输出报告可以包含一个或多个变量。对于这两种类型的报告，如果您包含多个项目，您可以选择将每个项目呈现在自己的表格中或将项目组合成单个表格。如果您将多个项目组合成单个表格，将形成额外的数据列，表格的宽度也会增加。

要将 X–Y 数据对象组合成单个表格，Abaqus/CAE 对齐数据对象的 X 值。如果任何数据对象没有匹配的 X 值，您可以选择让 Abaqus/CAE 通过插值和外推计算缺失的点。要了解更多关于插值和外推的信息，请参阅 ["理解 X–Y 数据插值和外推，" 第 47.4.2 节"](pt05ch47s03s02.md)。

对于场输出报告，Abaqus/CAE 为当前显示组中的每个区域创建单独的表格（根据您的规范可以是单个变量或组合变量的表格）。区域是模型中具有相容材料、截面属性和单元类型的部分。Abaqus/CAE 识别与报告中每个表格关联的区域。（有关显示组的更多信息，请参阅 [第 78 章，"使用显示组显示模型的子集"](pt07ch78.md)"；有关区域的更多信息，请参阅 ["理解结果值平均，" 第 42.6.2 节"](pt05ch42s04s02.md)。"

对于自由体切割，Abaqus/CAE 使您能够以正常注释格式或逗号分隔值（CSV）格式生成报告输出。在正常注释格式中，结果以易于阅读的方式呈现；在 CSV 格式中，结果旨在导出并在电子表格应用程序中阅读。您还可以相对于全局坐标系或相对于定义自由体切割的局部坐标系来报告来自自由体切割的数据。

**控制报告布局、宽度、格式和坐标系：**

1. 找到 **输出格式（Output Format）** 选项。**对于 X–Y 数据或场输出的报告：** 从主菜单栏中，选择 ****报告（Report）**![](../graphics/images/arrow.gif)**XY**** 或 ****报告（Report）**![](../graphics/images/arrow.gif)**场输出（Field Output）****；然后在出现的对话框中点击 **设置（Setup）** 选项卡。**输出格式（Output Format）** 选项位于页面中央。**对于自由体切割的报告：** 从主菜单栏中，选择 ****报告（Report）**![](../graphics/images/arrow.gif)**自由体切割（Free Body Cut）****。**输出格式（Output Format）** 选项位于页面中央。**对于探针值的报告：** 从主菜单栏中，选择 ****工具（Tools）**![](../graphics/images/arrow.gif)**查询（Query）****，然后在出现的对话框中点击 **探针值（Probe values）**。出现 **探针值（Probe Values）** 对话框。将光标移到当前视口中的模型或 X–Y 绘图上探测，然后点击鼠标按钮 1 将感兴趣的值存储在对话框中。完成后，点击 **写入文件（Write to File）**。出现 **报告探针值（Report Probe Values）** 对话框；**输出格式（Output Format）** 选项位于对话框中央。
2. 对于 X–Y 数据报告或场输出报告，选择您想要的 **布局（Layout）** 选项：- 选择 **所有 X–Y 数据的单个表格（Single table for all X–Y data）**（或 **所有场输出变量的单个表格（Single table for all field output variables）**）将所有选定的数据对象组合成单个表格。对于 X–Y 数据报告，如果您希望 Abaqus 计算缺失的点，请切换 **在 X 值之间插值（如有必要）（Interpolate between X values (if necessary)）**。当此选项关闭时，缺失的点（如果有）会在表格中显示为"No Value"。- 如果您希望每个选定的数据对象出现在自己的表格中，请选择 **每个 X–Y 数据的单独表格（Separate table for each X–Y data）**（或 **每个场输出变量的单独表格（Separate table for each field output variable）**）。
3. 对于 X–Y 数据报告或场输出报告，选择您想要的 **页面宽度（字符）（Page width (characters)）** 选项：- 选择 **无限制（No limit）** 允许无限表格宽度。- 选择 **指定（Specify）** 限制表格的宽度。然后，在 **指定（Specify）** 文本字段中，输入可以出现在表格宽度方向的最大字符数。
4. 对于自由体切割报告，执行以下操作：- 选择 **正常注释格式（Normal annotated format）** 或 **逗号分隔值（CSV）（Comma-separated values (CSV)）** 作为报告格式。- 选择 **全局 CSYS（Global CSYS）** 或 **局部 CSYS（Local CSYS）** 作为自由体切割的坐标系。

要生成报告，请配置对话框中的剩余选项；然后点击 **应用（Apply）**（对于 X–Y、场输出或自由体切割报告）或 **确定（OK）**（对于探针值报告）。完成后，点击 **取消（Cancel）** 关闭对话框。

# 54.6 对场输出数据进行排序





对于表格场输出报告，您可以选择根据表格任何单列中的值对表格中的所有数据进行排序，您可以选择升序或降序排序。

您使用 **报告场输出（Report Field Output）** 对话框中的 **变量（Variable）** 选项来控制 Abaqus/CAE 包含在表格中的列。（有关更多信息，请参阅 ["选择场输出变量，" 第 54.3.2 节"](pt05ch54s03hlb02.md)。）表格为您选择的每个场输出变量有一列，还有额外的列来识别场输出的来源；例如，如果您的表格包含唯一节点量，Abaqus/CAE 提供 **节点标签（Node Label）**（节点编号）列。

报告中使用负单元标签来区分 Abaqus/CAE 创建的内部单元与您在定义模型时创建的单元。如果您的报告包含内部元素且您选择按单元标签排序，这些元素将分组在报告的顶部或底部。

有关对探针值表格报告中的数据进行排序的信息，请参阅 ["输入表格数据，" 第 3.2.7 节"](pt01ch03s02s07.md)。

4. 点击 **升序（Ascending）** 或 **降序（Descending）** 来选择排序顺序。

要生成报告，请配置对话框中的剩余选项；然后点击 **应用（Apply）**。完成后，点击 **取消（Cancel）** 关闭对话框。
