# *FIXED MASS SCALING









### *FIXED MASS SCALING在步骤开始时指定质量缩放。

此选项用于在步骤开始时为模型的部分或全部指定质量缩放。

**产品：**Abaqus/Explicit

**类型：**历史数据

**级别：**步骤

**Abaqus/CAE：**步骤模块

##### **参考：**

- ["质量缩放，" Abaqus分析用户指南第11.6.1节](../usb/usb-link.md#usb-anl-amassscaling)
- ["输出，" Abaqus分析用户指南第4.1.1节](../usb/usb-link.md#usb-out-ooutput)

### **可选参数：**

DT

将此参数设置为所提供单元集的所需单元稳定时间增量。此参数必须与TYPE参数结合使用，以定义如何应用质量缩放。

如果省略FACTOR和DT参数，则使用默认质量缩放值1.0。如果两个参数都包含，则质量首先按FACTOR参数分配的值进行缩放，然后可能根据DT和TYPE参数分配的值再次缩放。

ELSET

将此参数设置为正在应用此质量缩放定义的单元集名称。如果省略此参数，则质量缩放定义将应用于模型中的所有单元。

可以使用不同的ELSET定义重复[*FIXED MASS SCALING](ch06abk13.md)选项，以为指定的单元集定义不同的质量缩放。

FACTOR

将此参数设置为质量缩放因子。单元的质量将在步骤开始时按FACTOR参数分配的值进行一次缩放。

如果省略FACTOR和DT参数，则使用默认质量缩放值1.0。如果两个参数都包含，则质量首先按FACTOR参数分配的值进行缩放，然后可能根据DT和TYPE参数分配的值再次缩放。

TYPE

设置 TYPE=UNIFORM 以对单元的质量进行均匀缩放，使缩放后单元的最小单元稳定时间增量等于分配给DT的值。

设置 TYPE=BELOW MIN（默认）以仅对单元稳定时间增量小于分配给DT值的单元进行质量缩放。这些单元的质量将被缩放，使单元稳定时间增量等于分配给DT的值。

设置 TYPE=SET EQUAL DT 以对所有单元进行质量缩放，使它们都具有等于分配给DT值的相同单元稳定时间增量。

**此选项没有关联的数据行。**
