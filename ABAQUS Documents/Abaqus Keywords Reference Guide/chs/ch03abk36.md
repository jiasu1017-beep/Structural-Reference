# *CONNECTOR CONSTITUTIVE REFERENCE





### *CONNECTOR CONSTITUTIVE REFERENCE定义用于指定连接器本构行为的参考长度和角度。

此选项用于定义连接器单元中本构响应的参考长度和角度。

**产品：**Abaqus/Standard  Abaqus/Explicit  Abaqus/CAE  

**类型：**模型数据

**级别：**模型

**Abaqus/CAE：**Interaction模块

##### **参考：**

- ["连接器行为"，Abaqus Analysis User's Guide第31.2.1节](../usb/usb-link.md#usb-elm-econnectorbehavior)
- [*CONNECTOR BEHAVIOR](ch03abk35.md)

**此选项没有关联的参数。**

### **定义参考长度和角度的数据行：**

**第一行（也是唯一一行）：**

输入空格以使用从初始几何形状计算的（默认）参考长度或角度。
