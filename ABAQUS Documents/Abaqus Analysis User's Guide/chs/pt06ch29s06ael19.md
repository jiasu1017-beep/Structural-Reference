### 概述

本节提供 Abaqus/Standard 和 Abaqus/Explicit 中可用轴对称壳单元的参考。对于预期非轴对称行为的轴对称壳几何形状，请使用 Abaqus/Standard 中可用的 SAXA 单元（见["具有非线性非对称变形的轴对称壳单元，" 第29.6.10节](pt06ch29s06ael20.md)）。

### 约定

坐标1是*r*，坐标2是*z*。*r*方向对应全局*X*方向，*z*方向对应全局*Y*方向。坐标1应大于或等于零。

自由度1是![](../graphics/usb_eqn00266.gif)，自由度2是![](../graphics/usb_eqn00267.gif)，自由度6是*r*–*z*平面内的旋转。

Abaqus 不会自动对位于对称轴上的节点施加任何边界条件。如果需要，您应该在这些节点上施加径向或对称边界条件。

点载荷和集中通量应作为沿圆周积分的值给出（即整个环上的载荷）。

子午线方向是*r*–*z*平面内与单元相切的方向；即子午线方向是绕对称轴旋转以生成完整三维体的线方向。

周向或环向方向是垂直于*r*–*z*平面的方向。

**描述：** 非均匀薄膜系数和吸声温度（![](../graphics/usb_eqn00123.gif)的单位）在单元表面上，幅度通过用户子程序[`FILM`](../sub/sub-link.md#sub-xsl-film)提供。

### 辐射类型

基于表面的辐射条件可用于具有温度自由度的单元。如["热载荷，" 第34.4.4节](pt07ch34s04aus123.md)中所述进行指定。

**载荷 ID (*SRADIATE*)：** R(S)
**Abaqus/CAE 载荷/相互作用：** **Surface radiation**
**单位：** [Dimensionless](../popups/usb-int-iconventions-unitsym.md)
**描述：** 单元表面的发射率和吸声温度（![](../graphics/usb_eqn00123.gif)的单位）。

### 入射波载荷

基于表面的入射波载荷可用。如["声学、冲击和耦合声-结构分析，" 第6.10.1节](pt03ch06s10at29.md)中所述进行指定。如果入射波场包括从网格边界外平面的反射，则可以包括此效应。

### 单元输出

#### 应力、应变和其他张量分量

对于具有位移自由度的单元，应力和其他张量（包括应变张量）可用。所有张量具有相同的分量。例如，应力分量如下：

| S11 | 子午线应力。 |
| --- | --- |

| S22 | 环向（周向）应力。 |
| --- | --- |

#### 截面力、弯矩和横向剪力

可用于具有位移自由度的单元。

| SF1 | 子午线方向单位宽度膜力。 |
| --- | --- |

| SF2 | 环向方向单位宽度膜力。 |
| --- | --- |

| SF3 | 子午线方向单位宽度横向剪力（仅从Abaqus/Standard可用）。 |
| --- | --- |

| SF4 | 厚度方向积分应力；始终为零（仅从Abaqus/Standard可用）。 |
| --- | --- |

| SM1 | 关于环向方向单位宽度弯矩。 |
| --- | --- |

| SM2 | 关于子午线方向单位宽度弯矩。 |
| --- | --- |

#### 截面应变、曲率变化和横向剪切应变

可用于具有位移自由度的单元。

| SE1 | 子午线方向膜应变。 |
| --- | --- |

| SE2 | 环向方向膜应变。 |
| --- | --- |

| SE3 | 子午线方向横向剪切应变（仅从Abaqus/Standard可用）。 |
| --- | --- |

| SE4 | 厚度方向应变（仅从Abaqus/Standard可用）。 |
| --- | --- |

| SK1 | 关于环向方向的曲率变化。 |
| --- | --- |

| SK2 | 关于子午线方向的曲率变化。 |
| --- | --- |

#### 壳厚度

| STH | 壳厚度，对于SAX1、SAX2和SAX2T单元是当前厚度。 |
| --- | --- |

#### 热通量分量

适用于具有温度自由度的单元。

| HFL1 | 子午线方向热通量。 |
| --- | --- |

| HFL2 | 厚度方向热通量。 |
| --- | --- |
