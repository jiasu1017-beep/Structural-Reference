# 19.8.2 配置作业提交属性







要找到作业编辑器，请从主菜单栏中选择****作业**![](../graphics/images/arrow.gif)**编辑**![](../graphics/images/arrow.gif)***作业名称*****。使用作业编辑器中的**提交**选项卡页面配置与作业提交相关的属性。您可以执行以下操作：
- 选择分析类型。（有关更多信息，请参见["选择作业类型，"第 19.8.3 节](pt03ch19s08hlb03.md)。）
- 选择运行模式。（有关更多信息，请参见["选择作业运行模式，"第 19.8.4 节](pt03ch19s08hlb04.md)。）
- 选择提交时间。（有关更多信息，请参见["设置作业提交时间，"第 19.8.5 节](pt03ch19s08hlb05.md)。）
