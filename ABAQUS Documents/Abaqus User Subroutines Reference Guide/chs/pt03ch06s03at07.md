### 概述

Abaqus/Standard中的直接积分动态分析：
- 必须在研究非线性动态响应时使用；
- 可以完全非线性（通用动态分析）或基于线性系统模态（子空间投影法）；以及
- 可用于研究各种应用，包括：需要瞬态保真度和最小能量耗散的动态响应；涉及非线性、接触和中等能量耗散的动态响应；以及在相当能量耗散提供稳定性并改善收敛行为以确定基本静态解的准静态响应。

### 通用动态分析

Abaqus/Standard中的通用非线性动态分析使用隐式时间积分来计算系统的瞬态动态或准静态响应。该过程可应用于各种应用，需要不同的数值求解策略，例如获得收敛所需的数值阻尼量以及自动时间增量算法通过求解的方式。该过程可应用于需要不同数值求解策略的各种应用，例如获得收敛所需的数值阻尼量以及自动时间增量算法通过求解的方式。典型动态应用分为三类：
- 瞬态保真度应用，如卫星系统分析，需要最小能量耗散。在这些应用中，采用小时间增量来准确解析结构的振动响应，数值能量耗散保持在最低限度。这些严格要求往往会降低涉及接触或非线性的模拟的收敛行为。
- 中等耗散应用涵盖更广泛的动态事件，其中通过塑性、粘性阻尼或其他效应耗散中等量的能量。典型应用包括各种插入、冲击和成形分析。这些结构的响应可以是单调的或非单调的。准确解析高频振动通常不是这些应用所关心的。在这些应用中，一些数值能量耗散往往可以减少求解噪声并改善收敛行为，而不会显著降低求解精度。
- 准静态应用主要关注最终静态响应。这些问题通常表现出单调行为，惯性效应主要用于正则化不稳定行为。例如，静态不稳定行为可能是由于暂时无约束的刚性体模式或"跳变"现象。尽可能采用大时间增量以最低计算成本获得最终解。在加载历史的某些阶段可能需要相当大的数值耗散来获得收敛。

瞬态保真度应用的示例可在["汽车悬架建模，" Abaqus实例问题指南第2.1.7节](../exa/exa-link.md#exa-dyn-jointcautosuspension)中获得。同时包含中等耗散步骤和准静态步骤的分析在["棘轮机构的冲击分析，" Abaqus实例问题指南第2.1.17节](../exa/exa-link.md#exa-dyn-pawlratchet)中描述。

#### 与质量特性相关的建模错误诊断

准确的惯性特性表示对于准确动态分析是必要的。在某些情况下，当Abaqus/Standard检测到与惯性特性规范相关的可能建模错误时，会提供诊断消息。指定惯性特性最常见的方式是使用材料密度。如果在动态分析中省略了材料密度，Abaqus/Standard会向数据（.dat）文件发出警告消息（如果密度仅对某些温度或场变量值为零，则不会发出此警告）。指定惯性特性的其他方法包括：
- 点质量和旋转惯性定义，以及
- 将自身没有惯性的约束节点连接到定义了惯性特性的节点。

在某些情况下，Abaqus/Standard尝试求解涉及有效反转全局质量矩阵的方程组，以在通用动态分析中直接调整速度和加速度，如下面的["初始条件"](pt03ch06s03at07.md#usb-anl-adynamic-ic)"和["间歇性接触/冲击"](pt03ch06s03at07.md#usb-anl-adynamic-intermit)"所述。这些额外的速度和加速度调整仅对上述定义的瞬态保真度应用类型默认活跃。如果发现全局质量矩阵是奇异的，Abaqus/Standard默认会发出错误消息，因为奇异质量表示由于建模错误，质量特性不现实。

对于准静态和中等耗散应用类型，通常不提供关于全局质量矩阵奇异性的特定诊断反馈，尽管通常会发出关于材料密度缺失的警告。奇异质量不一定对准静态分析有害。例如，在准静态分析中，仅在具有临时静态不稳定性的组件或区域中定义惯性特性（如密度）可能是合理的，例如暂时无约束的刚性体模式，一旦发生接触就会变得有约束。

用户可以控制Abaqus/Standard在检测到奇异全局质量矩阵时采取的行动方案。

| **输入文件用法：** | 使用以下默认选项，在检测到奇异全局质量矩阵时发出错误消息并停止执行（计算速度和加速度调整时）： |
| --- | --- |
|  | ``` [*DYNAMIC](../key/key-link.md#usb-kws-hdynamic), SINGULAR MASS=ERROR ``` 使用以下选项在检测到奇异全局质量矩阵时发出警告消息并避免速度和加速度调整（即，使用当前速度和加速度继续时间积分）： ``` [*DYNAMIC](../key/key-link.md#usb-kws-hdynamic), SINGULAR MASS=WARNING ``` 使用以下选项即使检测到奇异质量矩阵也调整速度和加速度。此设置可能导致较大、非物理的速度和/或加速度调整，这反过来可能导致较差的时间积分求解和人为收敛困难。*此方法通常不推荐；仅应在分析师对如何解释以这种方式获得的结果有透彻了解的特殊情况下使用。* ``` [*DYNAMIC](../key/key-link.md#usb-kws-hdynamic), SINGULAR MASS=MAKE ADJUSTMENTS ``` |

#### 数值细节

下面描述了应用类型分类对通用动态分析数值方面的影响。在大多数情况下，由应用类型确定的设置足以成功执行分析。但是，提供了详细的用户控制以逐项覆盖设置。

##### 时间积分方法

除非用户指定应用类型为准静态，否则Abaqus/Standard默认使用Hilber-Hughes-Taylor时间积分。Hilber-Hughes-Taylor算子是Newmark β-方法的扩展。与Hilber-Hughes-Taylor算子相关的数值参数针对中等耗散和瞬态保真度应用进行不同调整（将在本节后面讨论）。如果应用分类为准静态，则默认使用向后Euler算子。

这些时间积分算子是隐式的，这意味着必须对算子矩阵求逆，并且在每个时间增量必须求解一组非线性动态平衡方程。此求解使用Newton法迭代完成。这些算子的主要优点是它们对线性系统是无条件稳定的；对于可以用于积分线性系统的时间增量大小没有数学限制。当研究结构系统时，无条件稳定积分算子是非常重要的，因为有条件稳定积分算子（如显式方法中使用的算子）可能导致不切实际的小时间步长，从而导致计算昂贵的分析。

用有限时间增量大小穿越模拟通常会引入一定程度的数值阻尼。这种阻尼不同于["材料阻尼，" 第26.1.1节"](pt05ch26s01abm51.md)中讨论的材料阻尼（并且在许多情况下这两种形式的阻尼将很好地一起工作）。与时间积分相关的阻尼量因算子类型而异（例如，向后Euler算子往往比Hilber-Hughes-Taylor算子更具耗散性），在许多情况下（如Hilber-Hughes-Taylor算子）取决于与算子相关的数值参数设置。算子有效处理接触条件的能力通常与其有用性相关。例如，接触条件的某些变化可能导致许多时间积分器产生"负阻尼"（非物理能源），这可能是非常不可取的。

可以覆盖应用类型分类隐含的时间积分器；例如，用户可以使用向后Euler积分器执行中等耗散动态分析。更改默认积分器通常不推荐，但可能在特殊情况下有用。

| **输入文件用法：** | 使用以下选项使用与瞬态保真度应用对应的默认积分器参数设置来使用Hilber-Hughes-Taylor积分器： |
| --- | --- |
|  | ``` [*DYNAMIC](../key/key-link.md#usb-kws-hdynamic), TIME INTEGRATOR=HHT-TF ``` 使用以下选项使用与中等耗散应用对应的默认积分器参数设置来使用Hilber-Hughes-Taylor积分器： ``` [*DYNAMIC](../key/key-link.md#usb-kws-hdynamic), TIME INTEGRATOR=HHT-MD ``` 使用以下选项使用向后Euler积分器： ``` [*DYNAMIC](../key/key-link.md#usb-kws-hdynamic), TIME INTEGRATOR=BWE ``` |

##### 对积分器参数的额外控制

额外的用户控制允许修改与Hilber-Hughes-Taylor算子相关的数值参数设置（参见[Hilber、Hughes和Taylor（1977）](../stm/stm-link.md#stm-ref-hilber-hughes-taylor)对数值参数的描述）。默认参数设置取决于指定的应用类型，如表6.3.2-1所示（参见[Czekanski、El-Abbasi和Meguid（2001）](../stm/stm-link.md#stm-ref-czekanski)了解这些设置的基础）。

**表6.3.2-1** Hilber-Hughes-Taylor积分器的默认参数。
| 参数 | 应用 |
| --- | --- |
| 瞬态保真度 | 中等耗散 |
| α | --0.05 | --0.41421 |
| β | 0.275625 | 0.5 |
| γ | 0.55 | 0.91421 |

如果正在使用Hilber-Hughes-Taylor算子，可以单独调整或修改这些参数。如果这些参数的默认设置对应于表6.3.2-1中所示的瞬态保真度设置，并且用户单独修改α参数，则其他参数将自动调整为β和γ。此关系提供了对与时间积分器相关的数值阻尼的控制，同时保持积分器的理想特性。数值阻尼随时间增量与振动模态周期的比值增加而增加。α的负值提供阻尼；而α=--1/3导致无阻尼（能量保持）并且正是梯形规则（有时称为Newmark β-方法，其中β=1/4和γ=1/2）。设置α=--0.6提供最大数值阻尼。当时间增量是被研究模态振荡周期的40%时，它提供约6%的阻尼比。α、β和γ的允许值为：α∈[--0.6, 0]、β∈[0.25, 0.5]、γ∈[0.5, 1]。

| **输入文件用法：** | ``` [*DYNAMIC](../key/key-link.md#usb-kws-hdynamic), ALPHA=α, BETA=β, GAMMA=γ ``` |

| --- | --- |

##### 默认增量方案

自动时间增量默认用于非线性动态过程。用于控制隐式动态过程时间增量大小调整的主要因素是Newton迭代的收敛行为和时间积分的精度。时间增量大小可能在分析过程中变化很大。时间增量控制算法的细节取决于用户正在研究的动态应用类型。

如果用户指定准静态类型应用，则默认情况下时间增量控制算法考虑以下因素（相同因素控制纯静态分析的时间增量大小）：
- 如果增量似乎发散或收敛率缓慢，则减小时间增量大小。
- 如果先前增量中发生快速收敛，则相当激进地增加时间增量大小。

对于中等耗散类型应用的分析也使用这些相同因素，以及等于步骤持续时间十分之一的默认时间增量大小上限。

如果用户指定瞬态保真度类型应用，则默认情况下时间增量控制算法考虑以下因素：
- 如果增量似乎发散或收敛率缓慢，则减小时间增量大小。
- 如果在处理增量的第一次尝试期间检测到接触状态变化，则减小时间增量大小。新增量大小被设置为使得增量的结束对应于使用先前增量大小检测到的接触状态变化的平均时间。（在这种情况下，使用额外的非常小的时间增量来强制跨活跃接触界面的速度和加速度相容性。）
- 如果时间增量中点的半增量残差（不平衡力）超过半增量残差容差，则减小时间增量大小；对于接触分析，半增量残差容差是时间平均力的10,000倍；对于无接触分析，是时间平均力的1,000倍。
- 如果先前增量中发生快速收敛，则逐渐增加时间增量。
- 时间增量大小的上限等于步骤持续时间的1/100。

##### 间歇性接触/冲击

前面列表中描述的第二个和第三个因素通常会导致作为瞬态保真度应用执行的接触模拟的时间增量大小非常小（并且由于第四个因素，时间增量大小往往保持很小）。可以通过指定不同的应用类型或使用更详细的用户控制来避免此问题，如下所述。

##### 直接时间增量

用户可以直接指定要使用的时间增量大小。此方法通常不推荐，但可能在特殊情况下有用。如果在允许的最大迭代次数内不满足收敛容差，分析将终止。

可以忽略收敛容差：在达到允许的最大迭代次数后，即使不满足收敛容差，也接受增量的解。忽略收敛容差可能导致高度非物理结果，不推荐，除非分析师对如何解释以这种方式获得的结果有透彻了解。

| **输入文件用法：** | 使用以下选项直接指定时间增量： |
| --- | --- |
|  | ``` [*DYNAMIC](../key/key-link.md#usb-kws-hdynamic), DIRECT ``` 使用以下选项在达到最大迭代次数后忽略收敛容差： ``` [*DYNAMIC](../key/key-link.md#usb-kws-hdynamic), DIRECT=NO STOP ``` |

##### 载荷的默认振幅

如果用户选择了准静态应用分类，则默认情况下载荷（如施加的力或压力）会以斜坡方式施加；这种斜坡加载往往会增强稳健性，因为载荷增量大小与时间增量大小成正比。例如，如果Newton迭代无法为特定时间增量大小收敛，自动时间增量算法将减小时间增量大小并用较小的考虑载荷增量重新开始Newton迭代。

对于其他应用分类，动态过程默认使用阶跃函数施加载荷，使得在步骤的第一个增量中施加全部载荷（无论时间增量大小如何），并且载荷幅度在整个步骤中保持恒定。因此，如果第一个增量无法使用原始时间增量大小收敛，默认情况下减小时间增量不会减小载荷增量。在某些情况下，由于积分算子的正则化效应与时间增量大小的平方成反比，减小时间增量仍然会改善收敛行为。有关各种过程的默认振幅类型以及如何覆盖默认值的更多信息，请参见["定义分析，" 第6.1.2节](pt03ch06s01abo05.md)。

### "子空间投影"方法

Abaqus/Standard为非线性动态问题提供的替代方法是"子空间投影"方法。参见["子空间动力学，" Abaqus理论指南第2.4.3节](../stm/stm-link.md#stm-anl-subspacedyn)了解此方法的理论。在这种方法中，线性系统的模态在动态分析之前在特征频率提取步骤（["固有频率提取，" 第6.3.5节"](pt03ch06s03at10.md)）中提取，并用作一小组全局基向量来开发解。这些模态将包括特征模态，如果它们在特征频率提取步骤中被激活，还包括残余模态。当系统表现出轻度非线性行为时，例如小区域的塑性屈服或既不小但也不太大的旋转，此方法非常有效。

这种方法可能非常有效。与其他直接积分方法一样，就计算机时间而言，它比纯线性动态分析的模态方法更昂贵，但通常比直接积分模型所有运动方程的费用要低得多。然而，由于子空间投影方法基于系统模态，如果存在无法被形成求解基底的模态很好建模的极端非线性响应，则该方法将不准确。

#### 数值实现

子空间投影方法在Abaqus/Standard中使用显式（中心差分）算子实现，以积分以线性系统模态表示的运动方程。这种积分方法在这里特别有效，因为模态相对于质量矩阵是正交的，因此投影系统始终具有对角质量矩阵。

使用固定时间增量：此增量是用户指定的时间增量或稳定时间增量的80%中的较小者，其中是用作求解基底的模态的最高圆频率。80%因子旨在作为安全因子，使得非线性效应引起的该最高频率的任何增加不太可能导致积分变得不稳定。80%是相当任意的；在某些情况下它可能是不保守的。用户必须监测响应——例如能量平衡——以确保时间增量不会导致不稳定。如果非线性可能显著地使系统变硬，则不稳定是一个问题，尽管在许多实际情况下，这种变硬效应在增加系统较低频率方面比影响最可能保留以准确表示动态行为的高频方面更突出。

#### 子空间投影方法的准确性

子空间投影方法的有效性取决于线性系统模态作为问题的一组全局插值函数的值，这取决于用户的判断——与决定特定有限单元网格是否足够时所需的判断相同。该方法对轻度非线性系统和容易提取足够模态以确信它们能充分描述系统的情况很有价值。

如果在子空间动力学步骤中考虑了非线性几何效应，则可以在一段时间内执行动态模拟，通过使用另一次频率提取步骤重新提取当前受应力几何的模态，然后使用新模态作为子空间基系统继续分析。在某些情况下，此过程可以提高该方法的准确性。

### 初始条件

["Abaqus/Standard和Abaqus/Explicit中的初始条件，" 第34.2.1节"](pt07ch34s02aus116.md)描述了所有可用的初始条件。无论是否使用节点转换（参见["转换坐标系，" 第2.1.5节"](pt01ch02s01aus09.md)），必须在全局方向定义初始速度。

如果在也指定了位移边界条件的节点上指定了初始速度，则这些初始速度将在这些节点被忽略。但是，如果位移边界条件引用具有解析定义时间变化的振幅曲线（即，排除分段线性表格和等间距定义），Abaqus/Standard将计算边界条件中涉及节点的初始速度，作为分析变化（在时间零处求值）的时间导数。

当为动态分析指定初始速度时，它们应与模型上的所有约束一致，特别是与时间相关边界条件。Abaqus/Standard将确保初始速度与边界条件和多点约束一致，但不会检查与内部约束（如材料不可压缩性）的一致性。如果存在冲突，边界条件和多点约束优先于初始条件。

指定的初始速度仅在分析中的第一个动态步骤中使用。如果动态步骤不是第一个动态步骤，并且有一个紧邻前面的动态步骤，则使用前一步结束时的速度作为当前步骤的初始速度。如果动态步骤不是第一个动态步骤，并且紧邻前面的步骤不是动态步骤，则假定当前步骤的初始速度为零。

#### 控制动态步骤开始时的加速度计算

默认情况下，Abaqus/Standard将为瞬态保真度应用计算动态步骤开始时的加速度。用户可以选择绕过这些加速度计算，在这种情况下，Abaqus/Standard将假定当前步骤的初始加速度为零，除非有一个紧邻前面的动态步骤。如果紧邻前面的步骤也是动态步骤，则绕过加速度计算将导致Abaqus/Standard使用前一步结束时的加速度来继续新步骤。如果加载在动态步骤开始时没有突然变化，则绕过加速度计算是合适的；但如果第一个增量开始时的加载与前一步结束时的加载显著不同，则不正确。在突然施加大载荷的情况下，由于绕过加速度计算而产生的高频噪声可能大大增加半增量残差。

### 边界条件

边界条件可以应用于任何位移或旋转自由度（1-6）、开截面梁单元的翘曲自由度7、静水压力流体单元的流体压力自由度8或声学单元的声压自由度8（["Abaqus/Standard和Abaqus/Explicit中的边界条件，" 第34.3.1节"](pt07ch34s03aus118.md)）。

振幅参考可用于在直接积分动态步骤中指定随时间变化的边界条件。默认振幅变化在["定义分析，" 第6.1.2节"](pt03ch06s01abo05.md)中描述。

在直接时间积分动态分析中，当在方程约束或多点约束中使用具有指定运动的节点来控制另一个节点的运动时，方程或多点约束将正确地施加于从属节点的位移和速度。但是，加速度不会严格传递到从属节点，这可能导致一些高频噪声。

在子空间投影方法中，目前无法直接指定非零边界条件。相反，可以通过使用适当组合的大点质量和集中载荷来近似加速度边界条件。在需要这种边界条件的节点处，附加大约为原始模型质量10⁵–10⁶倍的大点质量。此外，必须在近似边界条件的方向上指定大小等于大点质量与所需加速度乘积的集中载荷。由于点质量显著大于模型的质量，大质量-集中载荷组合将准确地近似指定方向中的所需加速度。除加速度之外的边界条件必须转换为加速度历史，然后才能近似。

### 载荷

可以在动态分析中指定以下载荷：
- 集中节点力可以施加到位移自由度（1-6）；参见["集中载荷，" 第34.4.2节"](pt07ch34s04aus121.md)。
- 可以施加分布压力力或体力；参见["分布载荷，" 第34.4.3节"](pt07ch34s04aus122.md)。特定单元可用的分布载荷类型在[第六部分，"单元"](pt06.md)中描述。
- 可以施加分布压力或体积加速度（在声学单元上）；这些在["声学、冲击和耦合声学-结构分析，" 第6.10.1节"](pt03ch06s10at29.md)中描述。

### 预定义场

可以在动态分析中指定以下预定义场，如["预定义场，" 第34.6.1节"](pt07ch34s06aus128.md)中所述：
- 虽然温度不是应力/位移单元中的自由度，但可以将节点温度指定为预定义场。如果为材料提供了热膨胀系数（["热膨胀，" 第26.1.2节"](pt05ch26s01abm52.md)），则施加温度与初始温度之间的差异将导致热应变。指定温度也会影响温度相关材料特性（如果有）。
- 可以指定用户定义场变量的值。这些值仅影响场变量相关材料特性（如果有）。

### 材料选项

大多数描述机械行为的材料模型可用于动态分析。以下材料特性在动态分析期间不活跃：热特性（热膨胀除外）、质量扩散特性、电传导特性和孔隙流体流动特性。

### 单元
