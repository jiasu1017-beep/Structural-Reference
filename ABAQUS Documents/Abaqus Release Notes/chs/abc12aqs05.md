# 12.5 矢量节点变量的幅值输出







**产品：**Abaqus/Explicit  

**优点：**您现在可以输出位移、速度、加速度和反作用力的空间矢量节点量的幅值。

**说明：**引入了四个新的节点输出变量：空间位移幅值 UMAG、空间速度幅值 VMAG、空间加速度幅值 AMAG 和反作用力幅值 RFMAG。这些新的输出变量可用于历史输出请求。
