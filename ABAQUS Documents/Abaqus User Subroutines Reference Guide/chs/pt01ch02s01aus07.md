### 概述

节点厚度用于定义以下内容的连续变化厚度：
- 壳结构；
- 膜结构；或
- Abaqus/Explicit中的刚性单元。

### 定义节点厚度

您可以在特定节点或节点集上指定壳、膜或刚性单元的厚度。

| **输入文件用法：** | ``` [*NODAL THICKNESS*](../key/key-link.md#usb-kws-mnodalthickness) *node_number or node_set_name*, *thickness* ``` |
| --- | --- |

| **Abaqus/CAE用法：** | 对于常规壳复合铺层，使用以下选项： |
| --- | --- |
|  | 属性模块：复合铺层编辑器：**Shell Parameters**：**Nodal distribution**：选择解析场或基于节点的离散场 对于均质壳截面，使用以下选项：属性模块：壳截面编辑器：**Basic**：**Nodal distribution**：选择解析场或基于节点的离散场 对于复合壳截面，使用以下选项：属性模块：壳截面编辑器：**Advanced**：**Nodal distribution**：选择解析场或基于节点的离散场 |

#### 从备用文件读取节点厚度

节点厚度数据可以存储在单独的文件中，并在分析开始时从该文件读取。有关此类文件名的语法详细信息，请参见["输入语法规则"，第1.2.1节"](pt01ch01s02aus01.md)。

| **输入文件用法：** | ``` [*NODAL THICKNESS*](../key/key-link.md#usb-kws-mnodalthickness), INPUT=*file_name* ``` |
| --- | --- |

| **Abaqus/CAE用法：** | 在Abaqus/CAE中不支持从备用文件读取节点厚度。 |
| --- | --- |

#### 在两个节点或节点集之间生成连续变化的厚度

Abaqus可以在线性插值两个边界节点或节点集之间的厚度。必须首先定义边界节点处的厚度。

| **输入文件用法：** | 使用以下选项： |
| --- | --- |
|  | ``` [*NODAL THICKNESS*](../key/key-link.md#usb-kws-mnodalthickness) *first bounding node or node set*, *thickness* *second bounding node or node set*, *thickness* [*NODAL THICKNESS*](../key/key-link.md#usb-kws-mnodalthickness), GENERATE *first bounding node or node set, second bounding node or node set, * *number of intervals, increment in node numbers* ``` |

| **Abaqus/CAE用法：** | 在Abaqus/CAE中不支持在边界节点或节点集之间生成厚度。 |
| --- | --- |

### 为壳、膜和刚性单元指定连续变化的厚度

在定义单元截面时，必须指定壳或膜单元将具有连续变化的厚度而不是均匀厚度。详情请参见["膜单元"，第29.1.1节"](pt06ch29s01alm05.md)；["使用分析期间积分的壳截面定义截面行为"，第29.6.5节"](pt06ch29s06alm19.md)；和["使用通用壳截面定义截面行为"，第29.6.6节"](pt06ch29s06alm20.md)。

在Abaqus/Explicit中，在定义刚性单元所属的刚体时，必须指定刚性单元将具有连续变化的厚度；参见["刚性单元"，第30.3.1节"](pt06ch30s03alm23.md)。在Abaqus/Standard中，刚性单元不能具有连续变化的厚度。

使用连续变化厚度的壳、膜或刚性单元的每个节点都必须定义节点厚度。如果在使用连续变化厚度的单元中存在没有节点厚度的节点，Abaqus将发出错误消息。

#### 为复合壳指定连续变化的厚度

当复合壳结构具有连续变化的厚度时，壳在任何节点处的总厚度由节点厚度值定义。积分点处的总厚度从节点厚度插值。壳截面定义中给出的层厚度用作相对厚度，并按比例缩放，使得层厚度之和等于积分点处的总厚度。

##### 示例

例如，如果复合壳截面用以下输入定义：

```
[*SHELL SECTION*](../key/key-link.md#usb-kws-mshellsection), COMPOSITE, NODAL THICKNESS, ELSET=*name*
1.5, 3, STEEL
2.5, 3, FOAM
1.0, 3, STEEL
```

并且某点的总厚度仅为1.0，则该点各层的厚度为：第一钢层0.3、泡沫层0.5、第二钢层0.2。

#### 在壳、膜或刚性单元厚度中创建不连续性

您只能在每个节点指定一个厚度。因此，在壳、膜或刚性单元上厚度不连续的位置使用单独的节点，并可为每组节点分配适当的厚度。对于不属于刚体的单元，必须使用多点约束使相应节点处的位移（对于壳，还有旋转）相同。
