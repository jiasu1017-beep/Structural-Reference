# 44.3 生成云图





云图可以显示输出数据库在指定步骤和帧的分析变量值，或显示当前模型数据库中模型在指定步骤的所选属性值。Abaqus/CAE将这些值表示为模型上自定义的彩色线、彩色带、彩色面或彩色等值面，或表示为绘制在模型法线上的彩色刻度线。有关选择分析变量的更多信息，请参见["选择主场输出变量，" 第42.5.3节](pt05ch42s03hlb03.md)。有关选择特定步骤和帧的更多信息，请参见["选择特定结果步骤和帧，" 第42.3.1节](pt05ch42s01hlb01.md)。要了解如何显示与云图关联的最小值和最大值，请参见["自定义图例，" 第56.1节](pt05ch56s01.md)。

如果您的模型包含梁单元，可能需要一些额外步骤才能正确显示云图。请参见["生成线性梁截面应力的云图，" 第44.4节](pt05ch44hla03.md)。

如果您正在从模型数据库绘制数据，则只需指定主场变量。

**要生成云图：**

1. 使用**File**菜单[打开包含模型数据或分析结果的模型数据库或输出数据库](pt02ch09s06hlb02.md)。
2. 如果您正在从模型数据库绘制数据，请切换到可视化模块，展开Results Tree的**Model Database**容器，然后选择要使用的模型。
3. 使用**Result**菜单选择以下各项：1. 要显示的[步骤和帧](pt05ch42s01hlb01.md)（如果选择了模型数据库，则只需步骤）。2. 要显示的[主场](pt05ch42s03hlb03.md)输出变量。3. 要显示的[变形场](pt05ch42s03hlb04.md)输出变量（仅用于变形形状上的云图）。4. [要绘制的内容](pt05ch42s04hlb01.md)和[平均](pt05ch42s04s02.md)选项（仅适用于输出数据库）。
4. 选择您想要的绘图状态独立和云图[自定义选项](pt05ch40s03s03.md)。
5. 如果您选择的场输出变量包含复数，请在**Result Options**对话框中选择[**Complex Form**](pt05ch42s04hlb05.md)选项卡来控制要显示的数值形式。复数形式选项仅适用于输出数据库。
6. 从主菜单栏中，选择****Plot**![](../graphics/images/arrow.gif)**Contours****，然后选择是在未变形形状、变形形状还是两者上绘制云图。**提示：**您还可以使用工具箱中的变形![](../graphics/ico_plotContourDeformed.png)、未变形![](../graphics/ico_plotContourUndeformed.png)或叠加![](../graphics/ico_plotContourDefUndef.png)云图工具生成云图。当前视口显示当前输出数据库在指定步骤和帧的指定场输出变量的自定义云图，或当前模型数据库中所选模型在指定步骤的指定场输出变量的自定义云图。每次您在步骤和帧选择器、场输出选项、绘图状态独立选项、叠加绘图选项（如果适用）或云图选项对话框中单击**Apply**时，Abaqus会自动刷新您的云图。
