# 69.2.2 编辑面的方法概述







当您从 **Geometry Edit**（几何编辑）对话框中选择 **Face**（面）时，**Tool**（工具）列表显示以下修复面的方法：

**Remove（移除）![](../graphics/ico_repairRemoveFace.png)**

您可以从三维实体或壳或二维平面部件中移除所选的面（包括倒角、圆角和孔）。选择要移除的面后，Abaqus/CAE 查找定义特征的相邻面。您可以移除整个特征，也可以仅移除所选的面。当您从三维实体部件中移除一个或多个面时，Abaqus/CAE 将部件转换为壳。

**Cover edges（覆盖边）![](../graphics/ico_repairCreateFace.png)**

您可以通过选择新面的一个或多个边在三维部件上创建面。Abaqus/CAE 遍历相邻边并计算新面的位置。如果您选择了多个没有公共环连接的边，Abaqus/CAE 会创建多个面，每个边环一个面。Abaqus/CAE 将新面创建为壳。如果新壳形成闭合部件，您可以使用 solid-from-shell 工具将部件转换为实体。

**Replace（替换）![](../graphics/ico_repairReplaceFace.png)**

在某些情况下，当您导入部件时，Abaqus/CAE 可能无法准确重新创建某些面。例如，平面面可能看起来波浪状或扭曲，或者 Abaqus/CAE 可能创建了对网格密度有很大影响的小面。您可以选择相邻的面，Abaqus/CAE 用单个面替换它们。新面具有最小的面划分且比原始面更平滑。[图 69-4](pt06ch69s02s02.md#rep-replace-faces) 说明了替换所选面的效果。

**图 69-4** 替换所选面的效果。

![](../graphics/rep-replace-faces-nls.png)

或者，您可以用通过扩展相邻面形成的单个面替换所选的面。您可以使用此工具从导入的部件中移除凸台和小细节，如[图 69-5](pt06ch69s02s02.md#rep-replace-faces-extend)所示。

**图 69-5** 替换所选面并扩展相邻面的效果。

![](../graphics/rep-replace-faces-extend-nls.png)

**Repair small（修复小面）![](../graphics/ico_repairSmallFaces.png)**

您可以修复所选的小面。Abaqus/CAE 移除小面并编辑相邻面以创建闭合几何。[图 69-6](pt06ch69s02s02.md#rep-smallface) 说明了修复小面的效果。

**图 69-6** 修复小面。

![](../graphics/rep-smallface-nls.png)

**Repair sliver（修复细长面）![](../graphics/ico_repairRemoveSliver.png)**

细长面可以被认为是小块尖锐的多余材料。您可以从三维实体或壳的面或二维平面部件中移除不需要的细长面。您必须选择包含要移除的细长面的面和面上的两个点。Abaqus/CAE 在两点之间画一条线，将所选面分为两个区域。第一个区域是将被保留的面；第二个区域是要移除的细长面。[图 69-7](pt06ch69s02s02.md#rep-sliverface) 说明了移除细长面的效果。

**图 69-7** 移除细长面。

![](../graphics/rep-sliverface-nls.png)

**Repair normals（修复法向）![](../graphics/ico_repairFaceNormals.png)**

您可以修复壳和实体导入部件的面法向，您可以对这些修复执行在流形部件（每个边仅被一个或两个面共享的部件）和非流形部件上。这些工具对于实体或壳部件有不同的用途。

**实体**

在极少数情况下，查询工具集报告导入实体部件的体积为负值，因为面法向表明它在原始 CAD 系统中是内外翻转的。**Repair face normals**（修复面法向）工具将翻转法向并将实体翻正。

**壳**

导入的壳部件可能包含法向指向相反方向的面。**Repair face normals**（修复面法向）工具将对齐壳部件上的所有法向。如果面法向已经对齐，此工具将翻转所有法向，使它们保持对齐但指向相反方向。

**注意：**元素法向方向（使用 Property（属性）模块中的 **Element Normal**（元素法向）分配指定）指定相对于几何法向的相对元素法向。这些元素法向将在 **Repair face normals**（修复面法向）操作后使用新的几何法向进行更新。

当您对非流形壳部件执行修复时，您无法在单个操作中对部件中的所有面对齐法向；您必须单独选择面以翻转其法向方向。您可以在修复壳部件之前使用 **Query**（查询）对话框中的 **Shell element normals**（壳元素法向）选项执行查询以评估面法向的方向。

**Offset（偏移）![](../graphics/ico_offsetFaces.png)**

您可以通过选择要偏移的面然后指定偏移距离或选择目标面和距离计算方法在三维部件上创建面。偏移方向与源面的面法向方向相反，偏移可以是正的或负的。Abaqus/CAE 使用与草图绘制器中偏移相同的过程创建新面（有关更多信息，请参阅["偏移对象，" 第 20.8.6 节"](pt03ch20s08s06.md)）。

**注意：**无论偏移距离计算方法如何，Abaqus/CAE 应用恒定偏移来创建新面。您不能使用此方法创建与两个收敛或发散面等距的面。

**Extend（扩展）![](../graphics/ico_extendFaces.png)**

您可以通过指定扩展距离或选择目标面来控制扩展距离来扩展现有面。Abaqus/CAE 用扩展的面特征替换现有面。

**Blend（融合）![](../graphics/ico_blendFaces.png)**

您可以创建新面来融合模型中现有边的轮廓，以在这些边之间形成面。您可以选择通过在每条边上与现有面相切来创建新面，通过计算边之间的最短路径，或通过指定线形状作为路径来创建新面。

**Face from element faces（从单元面创建面）![](../graphics/icoR_elementFacesToFace.png)**

您可以从孤单元面创建新的几何面。Abaqus/CAE 根据所选单元面的节点位置创建新的几何面。顶点是在元素边方向有显著变化的边缘节点处创建的。
