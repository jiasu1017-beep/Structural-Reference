**To apply the **power** function:**

1. Locate the **Operate on XY Data** dialog box. From the main menu bar, select ****Tools**![](../graphics/images/arrow.gif)**XY Data**![](../graphics/images/arrow.gif)**Create****. Click **Operate on XY data** in the dialog box that appears; then click **Continue**. The **Operate on XY Data** dialog box appears.
2. From the **Operators** listed, click **power(A,A)**. The **power** function appears in the expression window.
3. From the **XY Data** choices, click the name of the *X--Y* data object on which to operate and click **Add to Expression**. You can choose from all *X--Y* data objects previously saved within this session (listed alphabetically in the **XY Data** field). The *X--Y* data object name appears within the **power** function parentheses in the expression window.
4. To specify the next argument, do one of the following: - Use your mouse and keyboard to enter a scalar as either the first or second argument of the **power** function in the expression window, depending on the operation you want to perform, and to separate the two arguments with a comma, or - From the **XY Data** choices, click the name of a data object argument for the **power** function in the expression window and click **Add to Expression**.
5. To continue to build your expression, position the cursor in the expression window and type in or select the functions, operators, and *X--Y* data you want to include.
6. To evaluate and display your expression, click **Plot Expression**.
7. To [save](pt05ch47s02hlb07.md) your new *X--Y* data object, click **Save As** and then provide a name in the dialog box that appears. Saving your data object makes it available for future operations within this session and for inclusion in *X--Y* plots containing multiple data objects.
8. When you are finished, click **Cancel** to close the dialog box.
