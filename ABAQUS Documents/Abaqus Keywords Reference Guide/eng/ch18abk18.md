# *SIMPLE SHEAR TEST DATA







### *SIMPLE SHEAR TEST DATAUsed to provide simple shear test data.

This option is used to provide simple shear test data. It can be used only in conjunction with the [*HYPERFOAM](ch08abk07.md) option.

**Products: **Abaqus/Standard  Abaqus/Explicit  Abaqus/CAE  

**Type: **Model data  

**Level: **Model  

**Abaqus/CAE: **Property module

##### **References:**

- ["Hyperelastic behavior in elastomeric foams," Section 22.5.2 of the Abaqus Analysis User's Guide](../usb/usb-link.md#usb-mat-chyperfoam)
- [*HYPERFOAM](ch08abk07.md)

**There are no parameters associated with this option.**
