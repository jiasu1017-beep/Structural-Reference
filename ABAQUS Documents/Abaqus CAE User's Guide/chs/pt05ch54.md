# 54 生成表格数据报告





您可以生成X-Y数据对象、场输出结果、探测值或自由体切割的表格数据报告。Abaqus/CAE将报告写入您选择的文件名。生成表格报告可以保存超出会话期限的数据值或打印这些值。此外,您可以使用表格报告作为后续Abaqus/CAE会话中的ASCII输入。
