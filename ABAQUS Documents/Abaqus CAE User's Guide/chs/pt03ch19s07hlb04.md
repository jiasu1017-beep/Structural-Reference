# 19.7.4 提交分析作业







要提交作业进行分析，请从主菜单栏中选择****作业**![](../graphics/images/arrow.gif)**提交**![](../graphics/images/arrow.gif)***您选择的作业*****。您也可以通过在**作业管理器**中选择作业名称，然后点击**提交**来提交作业进行分析。Abaqus/CAE 会使用作业编辑器中定义的作业设置提交作业进行分析。

有关监视已提交作业的信息，请参见["监视分析作业的进度，"第 19.2.6 节](pt03ch19s02hlb01.md)。

当您启动 Abaqus/CAE 会话并提交分析作业时，Abaqus 会向 Abaqus/CAE 发送消息以指示作业的当前状态，Abaqus/CAE 会相应地更新**作业管理器**中的**状态**列。

- ["创建、编辑和操作作业，"第 19.7 节](pt03ch19s07.md)
