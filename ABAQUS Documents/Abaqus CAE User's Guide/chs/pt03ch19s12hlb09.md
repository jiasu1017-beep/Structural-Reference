# 19.12.9 查看优化过程的结果







每个设计循环的结果存储在单独的输出数据库文件中。要查看优化的结果，您必须将单独的输出数据库文件合并为单个输出数据库文件，如["合并优化结果，"第 19.12.8 节](pt03ch19s12hlb08.md)中所述。

要查看优化过程的结果，请选择****优化**![](../graphics/images/arrow.gif)**结果**![](../graphics/images/arrow.gif)***优化过程名称*****。您也可以通过在**优化过程管理器**中选择优化过程名称，然后点击**结果**来查看结果。Abaqus/CAE 进入可视化模块，如果您执行了拓扑优化，会自动显示叠加在未变形视图上的视图切割。视图切割的等值面变量是优化模块用于"添加"或"移除"单元的归一化材料属性。如果您执行了形状优化，Abaqus/CAE 会显示未变形视图，其中显示表面节点的新位置。如果您执行了尺寸优化，Abaqus/CAE 会显示显示壳厚度的未变形视图。您可以在优化运行时查看优化结果。有关更多信息，请参见["监视您的优化过程，"第 19.12.6 节](pt03ch19s12hlb06.md)。
