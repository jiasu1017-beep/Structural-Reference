# 4.4.11 重新排列标注列表顺序





你可以更改**标注管理器**列表中视口标注的顺序。更改顺序可以让你将相关标注（如标注同一区域的文本和箭头）放在一起在列表中显示。如果你从**标注管理器**列表中将多个选择绘制到视口，显示顺序由列表顺序决定；也就是说，列表顶部的标注在视口中也位于所有其他标注的前面。但是，你可以独立更改每个视口中标注的显示顺序，可以通过从**标注管理器**绘制单个标注来更改，或者通过选择标注、单击鼠标右键 3，并使用弹出菜单中的选项来更改。（有关详细信息，请参阅["在当前视口中操作标注，" 第 4.4.5 节](pt01ch04s04hlb04.md)。）

**要重新排列视口标注：**

1. 从主菜单栏中，选择****Viewport**![](../graphics/images/arrow.gif)**Annotation Manager****。**提示：**你也可以通过单击[**视口**工具栏](pt01ch04s02hlb02.md)中的![](../graphics/ico_manager.png)来显示**标注管理器**。
2. 从对话框列表中选择标注（有关详细信息，请参阅["从列表和表中选择多个项目，" 第 3.2.11 节](pt01ch03s02s11.md)）。**提示：**切换**在视口中高亮显示选择**可以预览所选标注。
3. 使用**标注管理器**底部的按钮重新排列列表。**注意：**如果你从列表中选择了多个标注，**上移**和**下移**按钮将不可用。Abaqus/CAE 会在列表中移动所选标注。
