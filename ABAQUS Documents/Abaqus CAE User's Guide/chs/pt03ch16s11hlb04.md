# 16.11.4 定义地静力初始应力场





当为特定区域指定地静力应力场时，垂直方向（在三維和轴对称模型中假定为*z*方向，在二维模型中假定为*y*方向）的应力被假定为随该垂直坐标（分段）线性变化。地静力应力场仅适用于连续体单元。有关更多信息，请参见["Abaqus/Standard和Abaqus/Explicit中的初始条件"章节，《Abaqus Analysis User's Guide》第34.2.1节](../usb/usb-link.md#usb-prc-pinitialcond)。对于Abaqus/Explicit分析，不能为梁和壳指定依赖于高程的初始应力。

**创建或编辑地静力应力场的步骤：**

1. 使用以下方法之一显示地静力应力场编辑器： - 要创建新的地静力应力场，请按照["创建预定义场"章节第16.8.3节](pt03ch16s08hlb03.md)中概述的程序进行操作（**类别**：`机械`；**所选步骤的类型**：`地静力应力`）。 - 要使用菜单或管理器编辑现有的地静力应力场，请参见["编辑依赖于步骤的对象"章节第3.4.13节](pt01ch03s04hlb05.md)。**注意：**您只能在初始步骤中创建或编辑地静力应力场。
2. 在**应力幅值1**字段中，指定区域内第一位置的（有效）应力垂直分量。
3. 在**垂直坐标1**字段中，指定区域内第一位置对应的垂直坐标。
4. 在**应力幅值2**字段中，指定区域内第二位置的（有效）应力垂直分量。
5. 在**垂直坐标2**字段中，指定区域内第二位置对应的垂直坐标。
6. 在**侧向系数1**字段中，指定第一侧向应力系数。此系数定义*x*方向应力分量。
7. 如果需要，开启**侧向系数2**设置的**指定**并指定第二侧向应力系数。此系数在三维情况下定义*y*方向应力分量，在平面或轴对称情况下定义厚度或环向分量。如果省略此值，则假定它与第一侧向应力系数相同。
8. 单击**确定**保存数据并退出编辑器。浅蓝色圆圈出现在模型上，表示您刚创建的初始地静力应力场。
