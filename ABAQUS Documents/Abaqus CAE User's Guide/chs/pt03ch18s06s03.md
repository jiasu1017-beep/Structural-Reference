# 18.6.3 配置形状优化任务





形状优化确定每个表面节点的位移，以努力使表面应力均匀化并满足目标函数和任何约束。您可以使用优化任务编辑器来自定义形状优化的各个方面。要找到编辑器，请从主菜单栏中选择****任务**![](../graphics/images/arrow.gif)**编辑**![](../graphics/images/arrow.gif)***优化任务名称*****。形状任务的默认设置可在各种优化模型中提供合理的结果；并且在大多数情况下，您不需要修改默认设置。

- ["配置网格平滑质量](pt03ch18s06s02.md#usi-opz-shape-taskconfigure-stiffness-meshsmoothquality)"

### 配置基本设置

在形状优化期间，优化模块修改模型的外表面。如果优化模块仅修改表面节点而不调整内部节点，则表面单元层将变形。因此，Abaqus分析的结果将不再可靠，优化的质量也会受到影响。为了保持表面单元的质量，优化模块将网格平滑应用于选定区域，这会调整内部节点相对于表面节点的位置。更多信息，请参见["将网格平滑应用于形状优化"，"结构优化：概述"，Abaqus Analysis User's Guide第13.1.1节](../usb/usb-link.md#usb-anl-aoptover-shape-smoothing)。

优化模块只能将网格平滑应用于三角形、四边形和四面体单元。其他单元类型在网格平滑期间被忽略。

**注意：**在开始形状优化之前，拥有高质量的有限元网格非常重要，特别是在您期望形状发生变化的区域。

**配置基本设置：**

1. 在优化任务编辑器中，单击**基本**选项卡。
2. 选择是否冻结边界条件区域。您已在"载荷"模块中施加位移边界条件的区域在优化期间具有相同的位移边界条件。控制"载荷"模块中位移边界条件的坐标系用于控制优化中的边界条件。
3. 默认情况下，优化模块冻结整个模型的边界条件。如果需要，单击![](../graphics/ico_selectBlue.png)并选择应冻结边界条件的区域。
4. 选择将应用网格平滑的区域：
   - 选择**指定平滑区域**（默认），然后单击![](../graphics/ico_selectBlue.png)选择区域。您可以将网格平滑应用于单元或面。
   - 选择**指定第一层**，然后单击![](../graphics/ico_selectBlue.png)选择表示要平滑的第一层单元的面。输入要平滑的单元层数。
   - 选择**使用任务区域平滑六层**以平滑设计区域的六层单元。建议您接受默认选择并手动选择将应用网格平滑的区域。
5. 选择允许在网格平滑操作期间移动的设计区域相邻节点层数：
   - 选择**全部固定**（默认）以防止自由表面节点移动。
   - 选择**全部不固定**以允许所有自由表面节点移动。
   - 选择**指定**并输入允许移动的自由表面节点相邻层数。

### 配置网格平滑质量

网格平滑尝试改善网格质量，尽管形状优化期间设计节点位移会导致网格变形。您可以指定平滑网格的相对质量，并且可以指定定义被视为高质量单元的角度范围（四边形和三角形单元）或长宽比范围（四面体单元）。被视为差的单元会获得质量评级。单元评级越差，在改善单元质量时会得到更多考虑。

**配置网格平滑质量：**

1. 在形状优化的优化任务编辑器中，单击**网格平滑质量**选项卡。
2. 执行以下任一操作：
   - 切换**目标网格质量**，然后选择一个设置（**低**、**中**或**高**）。在大多数情况下，您应该接受默认设置**低**。只有在确定网格质量不满意后，才应选择更高的收敛级别。即使计算成本很高，如果您的网格包含大量四面体单元，您可能需要选择更高的收敛级别；否则，网格质量可能不可接受。如果即使使用**高**收敛级别也无法获得满意的网格质量，您应该考虑通过减小**增长比例因子**和**收缩比例因子**来减少形状优化期间的位移量，如["配置形状优化任务中的高级选项"第18.6.3节](pt03ch18s06s02.md#usi-opz-shape-taskconfigure-stiffness-advanced)中所述。
   - 切换**目标网格质量**关闭计算单元质量的算法。
3. 切换**报告差质量单元**以生成超出单元质量表中定义范围的单元列表。
4. 切换**报告求解器质量标准违规**以报告Abaqus认为质量差的单元。
5. 如果您切换**报告求解器质量标准违规**，您可以选择在Abaqus遇到质量差的单元时停止优化过程。优化模块可能会生成质量差的网格，这将不允许Abaqus分析成功完成，特别是随着设计周期数量的增加。如果Abaqus过早停止分析，则没有结果可供优化模块使用，优化也会过早结束。如果您允许优化模块由于违反Abaqus单元质量标准而停止优化，则您将更容易排除优化故障并确定其失败原因。
6. 如果您选择允许优化模块调整网格质量，您可以使用表格指定定义被视为高质量单元的角度范围（四边形和三角形单元）或长宽比范围（四面体单元）。您还可以输入四边形和四面体单元的最大偏斜角以及四边形单元的最大锥度。在大多数情况下，您不应修改默认值。修改角度或长宽比范围对网格质量的影响很小。您应该尝试将优化模块中的可接受网格质量与Abaqus中的可接受网格质量相匹配。最好让优化过程由于网格质量下降而结束，而不是允许Abaqus结束优化过程或产生无意义的结果。
7. 选择网格平滑操作将使用的策略或算法。默认情况下，优化模块使用**约束拉普拉斯**网格平滑算法。如果您的模型相对较小（平滑区域中少于1000个节点），您可以选择**局部梯度**网格平滑算法。
8. 如果您选择了**约束拉普拉斯**策略，请执行以下操作：
   1. 选择**收敛级别**，这是优化模块尝试改善网格质量所花费时间的度量。在大多数情况下，您应该接受默认值**低**，这会导致优化模块应用少量大增量的迭代。选择**中**或**高**将导致更多小增量的迭代；但是，计算时间将显著增加。在修改收敛级别之前，您应该使用**网格平滑质量**选项卡式页面调整目标网格质量。
   2. 选择**评估几何约束的频率**，这决定网格平滑算法执行期间优化模块应用几何约束的频率。在大多数情况下，您应该接受默认值**低**。选择**中**或**高**将导致优化模块更频繁地应用几何约束，计算时间将显著增加。
9. 如果您选择了**局部梯度**策略，请输入**特征识别角度**，这是优化模块在网格平滑操作期间通过检测边和角来识别特征使用的角度。默认值为30，在大多数情况下可提供良好结果。

### 配置高级选项

**配置高级选项：**

1. 在优化任务编辑器中，单击**高级**选项卡。
2. 输入指定**增长比例因子**和**收缩比例因子**的值。增长比例因子应用于因形状优化而增长（增加模型体积）的节点的位移。收缩比例因子应用于因形状优化而收缩（减小模型体积）的节点的位移。建议您使用默认比例因子1.0执行优化并检查结果，然后再尝试使用修改的比例因子进行优化。大于1.0的值会增加节点的增量位移并加速优化。相反，小于1.0的值会减少节点的增量位移并减慢优化。如果优化的前几次迭代在表面节点位置产生很少变化，您应该考虑增加比例因子；例如，如果您有具有小单元边长的密集网格。相反，如果比例因子太大，网格质量会受到影响，单个单元可能会崩溃，并且优化可能无法收敛到最优解。如果原始模型接近最优，您应该考虑减小比例因子。当优化包含许多几何约束且初始网格质量较差时，减小比例因子和减慢优化也是有益的。要优化处于接触中的区域，您可能需要输入负值来反转优化方向。结果，高应力区域将收缩，低应力区域将增长。
3. 选择是在每个优化周期后（默认）还是仅在第一个周期后更新优化形状向量。优化模块为设计区域中的每个节点确定优化位移向量。该向量沿节点处外表面的法线方向，并指示优化期间的位移方向。如果您选择每个优化周期后更新优化形状向量，优化模块会调整向量以考虑不断变化的条件，例如结构形状、网格质量和设计变量限制的变化。如果您选择仅在第一个优化周期后更新优化形状向量，则向量在后续周期中保持固定。在大多数情况下，默认值每个优化周期后更新优化形状向量会提供更好的结果，因为网格平滑算法受到的限制较少，从而产生改善的网格质量。
4. 选择步长是由优化期间设计区域中节点的最小位移还是平均位移决定。优化模块检查您的网格并限制设计区域中节点在每个优化周期期间的位移量。此限制防止一个节点的大位移导致相邻单元崩溃。此外，基于条件的优化算法在每个设计周期后提供对设计区域节点位移的控制——步长。步长取决于优化模块应用于节点的限制。例如，如果优化模块减小允许的位移，基于条件的优化算法会减小增量大小。此选项允许您选择基于条件的优化算法用于确定步长的位移。您可以选择优化期间设计区域中节点允许位移的平均值或最小值（默认）。选择平均值会导致更大的步长和更快的最优解计算。但是，选择平均值可能导致仅允许小位移的节点位移受限，从而在设计区域中产生不希望的角。
5. 选择优化模块用于插值中边节点的方法。如果您选择**按位置线性**（默认），优化从连接角节点的优化位置线性插值中边节点的位置。如果您选择**按角节点的优化位移**，优化从连接角节点的优化位移插值中边节点的位置。如果节点在其原始位置，则中边节点位于角节点之间的线上，两种插值方法没有区别。但是，为防止单元弯曲，您必须选择**按角节点的优化位移**。
6. 如果需要，切换**移动向量的边长**并输入一个值。优化模块修改高曲率区域节点的位移，以防止网格因大体积变化而崩溃。实际上，尖角被平滑。触发平滑的最小单元边长默认值为5.0。更大的值会导致平滑区域更大的半径。
7. 优化模块可以使用滤波器来平滑局部应力峰值。您可以通过切换**等效应力的最大影响半径**并输入以下内容来定义滤波器函数：
   - 受滤波器影响的节点之间最大距离的值。
   - 一个值，决定在调整受滤波器影响的节点之间的最大距离时使用多少局部表面曲率。默认值为0.2；较小的值会增加表面曲率的影响。
   - 一个加权值，控制滤波器根据距节点的距离产生的影响。
8. 体积是您可以应用于形状优化的唯一约束，您可以指定将体积减少到指定值或初始值的分数。**等约束容差**指定指定的体积约束与计算体积之间的最小差异，导致优化模块假定解已收敛。优化模块将差异的绝对值与您输入的容差值进行比较。默认值为0.001。
