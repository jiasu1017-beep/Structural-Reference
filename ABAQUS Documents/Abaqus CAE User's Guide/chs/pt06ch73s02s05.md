# 73.2.5 指定区域的特定侧或端







当您从壳、三维部件的线或内部面创建曲面定义时，必须指定要将哪一侧包含在曲面定义中。当您从壳或面选择侧时，Abaqus/CAE使用不同颜色的面来指示两侧，并提示您选择要选择的侧对应的颜色。在此过程中，模型显示变为半透明，以使受阻或内部侧可见。当您从线选择侧时，Abaqus/CAE使用不同颜色的箭头头来指示两侧。

**从壳中选择侧**

为了区分壳的两侧，Abaqus/CAE将壳的一侧染成棕色，另一侧染成紫色。您通过在提示区域单击**Brown**或**Purple**按钮来选择特定侧（参见[图73-1](pt06ch73s02s05.md#set-surf2)）。

**图73-1** 选择曲面的侧。

![](../graphics/set-surf2-nls.png)

您还可以选择**Both sides**选项，该选项定义一个双侧曲面，包含壳的棕色侧和紫色侧。双侧曲面只能在以下情况下使用：
- 在Abaqus/Explicit分析中使用三维壳、膜和刚体的接触相互作用
- 在Abaqus/Standard分析中使用小滑动、面-面接触公式的接触相互作用

考虑[图73-2](pt06ch73s02s05.md#set-surf)中的锥形壳。

**图73-2** 颜色指示壳的侧。

![](../graphics/set-surf.png)

Abaqus/CAE将圆锥的外侧显示为紫色，内侧显示为棕色。因此，要选择圆锥的外侧，您可以在提示区域单击**Purple**；要选择内侧，单击**Brown**。

如果您选择多个壳面作为曲面定义的一部分，您可以分别控制每个面的颜色编码。Abaqus/CAE包含一个**Flip a surface**选项，允许您在创建曲面定义之前反转任何单个面的方向（参见[图73-3](pt06ch73s02s05.md#set-surf4)）。

**图73-3** **Flip a surface**选项对于多面壳曲面可用。

![](../graphics/set-surf4-nls.png)

使用此选项使所有所需的侧面具有相同的颜色，然后从提示区域选择该颜色。如果您选择**Both Sides**，Abaqus/CAE将选择所有选定面的两侧，而不考虑您之前翻转的任何曲面。

在[图73-4](pt06ch73s02s05.md#set-surf3)的模型中，两个圆形端都被选择为曲面的一部分。

**图73-4** 两个面包含在曲面选择中。

![](../graphics/set-surf3.png)

最初，Abaqus/CAE将两个面的凸侧都染成棕色。但是，所需的曲面实际上包括一个凸面和一个凹面。要获得此曲面，请单击**Flip a surface**，然后选择模型右侧的面。Abaqus/CAE仅反转此面的颜色编码，如[图73-5](pt06ch73s02s05.md#set-surf5)所示。

**图73-5** 翻转曲面。

![](../graphics/set-surf5.png)

在提示区域单击**Brown**；Abaqus/CAE在模型中所有棕色面侧上定义曲面。

**在内部曲面上选择侧**

当您选择内部面创建曲面时，曲面侧是模糊的。例如，[图73-6](pt06ch73s02s05.md#set-interior-surf)中高亮显示的面可以是单元体A的右侧或单元体B的左侧。

**图73-6** 在内部面上定义曲面。

![](../graphics/set-interior-surf-nls.png)

Abaqus/CAE使用上面讨论的相同棕色/紫色颜色编码界面来确定您打算选择曲面的哪一侧。单击**Purple**可在单元体A的右侧定义曲面；单击**Brown**可在单元体B的左侧定义曲面。

**从线上选择侧**

如果您正在选择三维线部件实例的曲面，Abaqus/CAE会将线显示为红色以及[图73-7](pt06ch73s02s05.md#set-beam)中所示的箭头。

**图73-7** 箭头指示三维线的端。

![](../graphics/set-beam-nls.png)

您可以选择带有洋红色箭头头的端、带有黄色箭头头的端，或部件的圆周表面（由红线表示）（参见[图73-8](pt06ch73s02s05.md#set-beamprompt)）。

**图73-8** 选择曲面的端或圆周。

![](../graphics/set-beamprompt-nls.png)

如果线部件是二维的，Abaqus/CAE使用[图73-9](pt06ch73s02s05.md#set-beam2d)中所示的箭头头来区分曲面的两侧。

**图73-9** 箭头指示二维线的顶面和底面。

![](../graphics/set-beam2d-nls.png)

您选择一个箭头头颜色来确定曲面侧，如[图73-10](pt06ch73s02s05.md#set-2dprompt)所示。

**图73-10** 选择曲面的侧。

![](../graphics/set-2dprompt-nls.png)

如果部件是轴对称壳，部件的内表面和外表面用箭头指示（参见[图73-11](pt06ch73s02s05.md#set-axisym)）。选择选项与[图73-10](pt06ch73s02s05.md#set-2dprompt)中的相同。

**图73-11** 箭头指示轴对称壳的内侧和外侧。

![](../graphics/set-axisym-nls.png)
