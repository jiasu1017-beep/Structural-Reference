# 16.5.3 理解符号位置和方向







符号在模型上的放置可以取决于符号所代表的规定条件类型以及规定条件所施加的区域类型。表16-1指示符号在几何模型上出现的位置，表16-2指示符号在网格化模型上出现的位置。

**表16-1** 几何上的符号位置。
| 施加规定条件的区域类型 | 符号在模型上的位置 |
| --- | --- |
| 顶点 | 在顶点处 |
| 边 | 沿边均匀分布 |
| 装配件级线 | 在线的中点处 |
| 面 | 对于有方向的规定条件（例如，压力载荷），在面内部均匀分布 |
| | 对于无方向的规定条件（例如，表面电荷和边界条件），沿面的边缘均匀分布 |
| 体 | 沿体的每条边均匀分布 |
| 整个模型 | 在需要定义刚体运动的位置（仅用于惯性relief载荷）；否则，在表示全局坐标系原点和方向的三角箭头上 |

**表16-2** 网格上的符号位置。
| 施加规定条件的区域类型 | 符号在模型上的位置 |
| --- | --- |
| 节点 | 在节点处 |
| 单元边（对于二维网格） | 在单元边的中点处 |
| 单元面（对于三维网格） | 在单元面的形心处 |
| 装配件级线 | 在线的中点处 |

例如，图16-12显示施加到两个顶点的集中力和施加到几何模型表面的边界条件。

**图16-12** 集中力和边界条件。

![](../graphics/lbi-glocation.png)

图16-13显示施加到四个节点的边界条件和施加到网格若干单元面的压力载荷。

**图16-13** 压力载荷和边界条件。

![](../graphics/lbi-gmesh.png)

**注意：**如果您将压力载荷施加到平面几何面，其中表面积相对于封闭面积很小（例如，由两个同心圆形成的环），则载荷符号可能分布不均匀，无论**装配件显示选项**对话框中的符号密度设置如何。

当边界条件固定自由度到位时，代表该分量的箭头指向区域内部并缺少杆。例如，图16-14中的边界条件将自由度1、2和3固定到位。

**图16-14** 固定自由度的边界条件。

![](../graphics/lbi-gdisplace.png)

同样，如果正压力载荷或欧拉流入边界条件施加到区域，则代表该压力载荷或边界条件的箭头指向区域内部，如图16-15所示。

**图16-15** 正压力载荷。

![](../graphics/lbi-gpospressure.png)

如果载荷被定义为具有复大小且实部和虚部具有不同的符号（例如，![](../graphics/usi_eqn00638.gif)），则载荷将显示为两端都有箭头的箭头。类似地，包含流入和流出分量的欧拉边界条件将显示为两端都有箭头的箭头。

在其他所有情况下，代表规定条件分量的箭头从区域指向外。

**注意：**当集中力的分量为零时，该分量不显示箭头。同样，当边界条件使自由度不受约束时，该分量不显示箭头。
