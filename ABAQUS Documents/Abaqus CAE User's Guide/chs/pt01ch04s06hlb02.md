# 4.6.1 使用背景图像和电影





你可以通过在视口背景中显示图像或电影来自定义会话中的视口。背景图像和电影都是视口特定的，因此你可以在会话中的每个视口中显示不同的图像或电影。背景图像在切换模块时会保留在视口中，而背景电影仅出现在可视化模块中。Abaqus/CAE 在现有视口背景之上显示图像和电影，因此如果你自定义了视口背景的颜色，图像或电影可能会遮挡部分或全部自定义背景颜色。

你可以使用背景图像来帮助创建模型；例如，已完成原型的图像可以帮助你在装配中对齐部件实例。或者，背景图像可以用作水印，或在生成模型图像时显示徽标。

**注意：**在草图模块中，你可以显示第二个背景图像来更有效地帮助你绘制部件。当选择草图模块时，Abaqus/CAE 会在模块范围的背景图像之上显示草图器背景图像，并在所有其他模块中隐藏草图器背景图像。请参阅["管理草图器背景中的图像，" 第 20.9.11 节](pt03ch20s09hlb11.md)。

背景电影可以帮助你将 Abaqus 分析结果与实验结果进行比较。例如，如果你显示一个显示原型中变形的背景电影，你可以为类似的 Abaqus 分析结果设置动画，并在单个视口中比较动画。

在视口背景中显示图像或电影之前，必须先将文件添加到 Abaqus/CAE 会话中。要从**图像/电影选项**对话框添加图像，请单击![](../graphics/ico_imageSmall.png)；然后输入图像或电影的名称并提供其位置。会话中的图像在所有模块中都可用，而电影仅在可视化模块中可用。图像和电影都仅在会话期间保留；它们不会保存到模型数据库或输出数据库。

Abaqus/CAE 支持以下格式的背景图像：Bitmap（`.bmp`）、PNG（`.png`）、GIF（`.gif`）、JPEG（`.jpg`、`.jpeg`）、TIFF（`.tif`）、XPM（`.xpm`）、PCX（`.pcx`）、ICO（`.ico`）、TGA（`.tga`）和 RGB（`.rgb`）。

Abaqus/CAE 支持满足以下两个条件的背景电影：
- 电影文件的格式必须在你的 Abaqus/CAE 平台上受支持。对于 Linux 系统，Abaqus/CAE 支持 AVI 格式（`.avi`）和 Quicktime 格式（`.mov`）。对于 Windows 系统，Abaqus/CAE 支持 AVI 和 Quicktime 格式，以及 MPEG 电影格式（`.mpeg`、`.mpg`、`.mlv`、`.wm`）和 Windows Media Format（`.asf`、`.wmv`、`.wm`）。
- 用于创建背景电影文件的编解码器必须是 Abaqus/CAE 中可用于创建电影文件的编解码器之一。例如，你只能在电影文件使用 Abaqus/CAE 中可用于 Quicktime 电影创建的三种编解码器之一创建时，才能在视口背景中显示 Quicktime 电影文件：Raw 8、Raw 24 或 RLE 24。在 Abaqus/CAE 中创建每种电影文件格式的可用编解码器在["选择动画文件格式，" 第 49.3.2 节](pt05ch49s03hlb02.md)中有描述。
