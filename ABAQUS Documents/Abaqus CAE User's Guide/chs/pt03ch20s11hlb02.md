# 20.11.2 创建垂直构造线







使用 Sketcher 工具箱中的垂直构造线工具沿垂直线帮助定位和对齐对象。垂直构造线与 Sketcher 网格的 *Y* 轴平行创建。旋转 Sketcher 网格不会影响现有的垂直构造线，但新的垂直构造线将与旋转后的网格对齐。下图说明了水平构造线和一组垂直构造线如何用于对齐圆心。（虚线表示构造几何。）

![](../graphics/ske-congeom-horiz.png)

您还可以使用垂直构造线来定义旋转实体和曲面的旋转轴。

**草绘垂直构造线：**

1. 从 Sketcher 工具箱中的构造工具中，选择垂直构造工具![](../graphics/ico_sketchConstLineVertical.png)。有关 Sketcher 工具箱中工具的图表，请参见["Sketcher 工具，"第 20.4.1 节](pt03ch20s04s01.md)。当您在 Sketcher 工作表上移动光标时，垂直构造线会水平移动。Abaqus/CAE 在提示区域显示提示以指导您完成该过程。
2. 点击将位于垂直构造线上的点。或者，您可以在提示区域出现的文本字段中输入其 *X-Y* 坐标。（*Y* 坐标是任意的，因为会被忽略。）
3. 要创建其他垂直构造线，请重复上一步。
4. 完成创建垂直构造线后，执行以下操作之一：- 在 Abaqus/CAE 窗口中的任意位置点击鼠标按钮 2。- 在 Sketcher 工具箱中选择任何其他工具。- 点击提示区域中的取消按钮![](../graphics/afxI_cancel.png)。
