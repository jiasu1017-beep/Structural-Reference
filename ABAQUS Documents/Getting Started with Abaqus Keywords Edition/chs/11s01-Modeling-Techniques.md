# 11.1 常规分析流程

每个常规步骤的起点是上一个常规步骤结束时的变形状态。因此，模型的状态在一系列常规步骤中演变，同时响应每个步骤中定义的载荷。初始条件定义了模拟中第一个常规步骤的起点。

所有常规分析流程都遵循相同的载荷施加和"时间"定义概念。

## 11.1.1 常规分析步骤中的时间

Abaqus 在模拟中有两种时间度量。**总时间**在所有常规步骤中持续增长，它是每个常规步骤的步长时间累积。每个步骤也有自己的时间尺度（称为**步时间**），每个步骤从零开始。随时间变化的载荷和边界条件可以按照任一时间尺度指定。图 11-1 显示了一个分析的时间尺度，该分析被分为三个步骤，每个步骤持续 100 秒。

## 11.1.2 常规步骤中载荷的指定

在常规步骤中，载荷必须指定为总值，而非增量值。例如，如果集中载荷在第一个步骤中的值为 1000 N，在第二个常规步骤中增加到 3000 N，则两个步骤中载荷的给定值应为 1000 N 和 3000 N，而不是 1000 N 和 2000 N。

默认情况下，之前定义的所有载荷都会传播到当前步骤。在当前步骤中，您可以定义额外的载荷，也可以修改之前定义的任何载荷（例如，更改其大小或将其禁用）。在当前步骤中未明确修改的任何之前定义的载荷将继续遵循其相关的幅值定义，条件是幅值曲线以总时间定义；否则，载荷将保持其在最后一个常规步骤结束时的值。
