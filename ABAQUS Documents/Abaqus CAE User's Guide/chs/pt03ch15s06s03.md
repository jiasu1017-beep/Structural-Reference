# 15.6.3 默认相互作用和约束参数





完成对潜在接触对的搜索后，Abaqus/CAE会用创建相互作用所需的所有参数填充接触对候选表。接触对和任何创建的表面都会提供名称。[表15-1](pt03ch15s06s03.md#usi-itn-auto-detection-names)概述了命名算法。

**表15-1** 接触检测工具中用于创建名称的算法。
| 接触对 | *前缀*`-`*接触对编号*`-`*主实例*`-`*从实例* |
| --- | --- |
| 主表面 | *前缀*`-`*接触对编号*`-`*主实例* |
| 从表面 | *前缀*`-`*接触对编号*`-`*从实例* |
| 合并的主表面 | *前缀*`-All-m` |
| 合并的从表面 | *前缀*`-All-s` |
| 合并的"全部"表面 | *前缀*`-All` |

在执行搜索之前，使用「Names」选项卡页面可以修改命名前缀并控制表面的创建。详细说明，请参见「指定接触检测的搜索标准」，第15.16.1节中的「指定接触检测的命名选项」。

接触检测工具为接触对提供的默认参数与传统相互作用或约束编辑器中使用的默认值略有不同。最值得注意的是，接触检测工具最初为每个接触对分配表面对表面离散化，而不是节点对表面离散化。请参见「网格绑定约束」，Abaqus Analysis User's Guide第35.3.1节，以及「Abaqus/Standard中的接触公式」，Abaqus Analysis User's Guide第38.1.1节，以了解有关表面离散化及相关约束实施方法的讨论。

默认表面调整选项取决于接触对中表面之间的距离。您可以在执行搜索之前使用「Rules」选项卡页面来控制分配给检测到的接触对的默认调整选项。您也可以使用此页面来指定分离容差，在此范围内所有接触对默认为绑定约束。有关「Rules」页面的更多信息，请参见「指定接触检测的搜索标准」，第15.16.1节中的「定义默认接触对参数」。

[表15-2](pt03ch15s06s03.md#usi-itn-auto-detection-params)列出了Abaqus/CAE提供的默认接触对参数。在创建相互作用和约束之前，您可以单独编辑每个参数。有关编辑参数和默认值的详细说明，请参见「审查和修改检测到的接触对」，第15.16.3节。

**表15-2** 接触检测工具的默认接触对参数。
| 参数 | 默认值 |
| --- | --- |
| 激活/抑制 | 激活 |
| 类型1 | 相互作用 |
| 滑动 | 有限滑动 |
| 离散化 | 表面对表面 |
| 相互作用属性 | 「相互作用属性管理器」中列出的第一个接触相互作用属性2；如果尚未创建任何相互作用属性，此参数为空白 |
| 调整1 | 非相交表面之间的接触相互作用为「Off」；相交表面之间的接触相互作用为「0」；绑定约束为「On」 |
| 创建步骤 | 初始 |
| 表面平滑 | 自动 |
| 1类型和调整参数的默认值由「Rules」选项控制。 |
| 2相互作用属性管理器按名称字母顺序列出所有创建的相互作用属性。 |

**注意：**上述某些参数默认情况下在接触对候选表中不可见。在表格中任意位置点击鼠标按钮3，并选择「Edit Visible Columns」来控制哪些参数出现在表中。

有关相互作用和约束参数的更多信息，请参见「网格绑定约束」，Abaqus Analysis User's Guide第35.3.1节；「在Abaqus/Standard中定义接触对」，Abaqus Analysis User's Guide第36.3.1节；以及「在Abaqus/Explicit中定义接触对」，Abaqus Analysis User's Guide第36.5.1节。
