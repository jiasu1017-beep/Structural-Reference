# 15.17.7 定义损伤

您可以为可用的相对运动分量定义损伤行为。有关更多信息，请参见["连接器损伤行为，"Abaqus分析用户指南第31.2.7节](../usb/usb-link.md#usb-elm-econndamagebehav)。如果您指定损伤行为选项，则还必须指定弹性行为选项。此外，如果您要定义基于塑性运动的损伤起始行为，则还必须指定塑性行为选项。

**要创建或编辑损伤行为：**

1. 使用以下方法之一显示连接器截面编辑器：
   - 要创建新的连接器截面，请按照["创建连接器截面，"第15.12.11节"](pt03ch15s12s01.md)中概述的过程进行操作。
   - 要编辑现有的连接器截面，请从主菜单栏选择**连接器**![](../graphics/images/arrow.gif)**截面**![](../graphics/images/arrow.gif)**管理器**，从出现的列表中选择连接器截面，然后点击**编辑**。
2. 在**编辑连接器截面**对话框中，执行以下操作之一：
   - 要定义新的损伤行为，请点击**添加**并从出现的菜单中选择**损伤**。
   - 要编辑现有的损伤行为，请从**行为选项**列表中选择该行为以显示该行为的关联数据字段。
3. 在**耦合**字段中，选择以下之一：
   - 选择**非耦合**以独立地为每个可用相对运动分量指定损伤准则。
   - 选择**耦合**以指定耦合所有或部分可用相对运动分量的损伤准则。
4. 如果您要定义非耦合损伤行为，请在**力/力矩**字段中打开与您正在定义损伤行为的可用相对运动分量一致的力或力矩的开关。如果多个分量的行为相同，您可以定义将使用此函数的单个损伤行为。如果多个分量的行为不同，则必须定义单独的损伤行为。
5. 选择**起始准则**。
   - 选择**力**以使用连接器中的力/力矩指定损伤起始准则。您提供力/力矩损伤起始值的下限（压缩）、上限（拉伸）或两者。
   - 选择**运动**以使用连接器中的相对本构位移/旋转指定损伤起始准则。您提供本构位移/旋转损伤起始值的下限（压缩）、上限（拉伸）或两者。
   - 选择**塑性运动**以使用连接器中的等效相对塑性运动指定损伤起始准则。您提供损伤起始的相对等效塑性位移/旋转作为相对等效塑性率的函数。您还必须指定塑性行为选项；有关更多信息，请参见["定义塑性，"第15.17.6节"](pt03ch15s17hlb06.md)。
6. 如果您要定义耦合损伤行为，则必须按如下方式指定连接器势：
   - 如果您选择**力**或**运动**作为**起始准则**，则必须指定连接器势以定义等效力大小或等效运动大小。选择**起始势**标签页，并定义至少一个力势项。有关更多信息，请参见["指定势项，"第15.17.16节"](pt03ch15s17hlb16.md)。
   - 如果您选择**塑性运动**作为**起始准则**，则必须在耦合连接器塑性行为选项中指定连接器势以定义等效相对塑性运动。要使用力势定义耦合连接器塑性行为，请参见["定义塑性，"第15.17.6节"](pt03ch15s17hlb06.md)。如果耦合塑性定义在力势中包含至少两项，则您可以在损伤行为选项的**起始**标签页的数据表中提供**模式混合比**，以反映前两项对势贡献的相对权重。有关此量如何定义的更多信息，请参见["模式混合比"中的"连接器塑性行为，"Abaqus分析用户指南第31.2.6节](../usb/usb-link.md#usb-elm-econnplastbehav-modemixratio)。
7. 要定义损伤起始准则，请在**起始**标签页上执行以下操作：
   1. 要定义依赖于温度或场变量的损伤起始准则：
      1. 打开**使用与温度相关的数据**以定义随温度变化的行为数据。数据表中会出现一个标有**温度**的列。
      2. 要定义依赖于场变量的行为数据，请点击**场变量数量**字段右侧的箭头以增加或减少场变量的数量。数据表中会出现场变量列。
   2. 在表中输入适当的损伤起始准则数据。您可以使用键盘在表中输入数据。或者，您可以在表中的任意位置点击鼠标按钮3以查看用于指定表格数据的选项列表。有关每个选项的详细信息，请参见["输入表格数据，"第3.2.7节"](pt01ch03s02s07.md)。
   3. 要修改正则化（Abaqus/Explicit分析专用）或数据外推的行为设置，请使用["为表格数据指定行为设置，"第15.17.14节"](pt03ch15s17hlb14.md)中所述的过程。对于包含基于塑性运动的损伤起始准则的Abaqus/Explicit分析，您还可以指定与率相关数据评估的设置。
8. 如果需要，定义指定损伤变量如何演化的损伤演化定律，如["定义损伤演化，"第15.17.8节"](pt03ch15s17hlb08.md)中所述。
9. 选择以下之一：
   - 如果您想继续定义行为，请点击**添加**，选择所需的行为，并继续连接器截面定义。有关定义其他行为的说明，请参见["使用连接器截面编辑器，"第15.17节"](pt03ch15s17.md)。
   - 如果您想查看或编辑现有行为，请从**行为选项**列表中选择它。有关编辑行为的说明，请参见["使用连接器截面编辑器，"第15.17节"](pt03ch15s17.md)。
   - 如果您想保存连接器截面定义并退出编辑器，请点击**确定**。
