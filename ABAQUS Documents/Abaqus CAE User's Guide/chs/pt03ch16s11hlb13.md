# 16.11.13 定义初始孔隙比场







对于 Abaqus/Standard 分析，您可以指定多孔介质节点处初始孔隙比 *e* 的值（请参见《Abaqus Analysis User's Guide》中的"耦合孔隙流体扩散与应力分析"第 6.8.1 节）。初始孔隙比可以直接定义为与高程相关的函数，也可以从先前 Abaqus 分析的输出数据库文件中进行插值得到，或通过用户子程序 [`VOIDRI`](../sub/sub-link.md#sub-xsl-voidri) 定义。有关详细信息，请参见《Abaqus Analysis User's Guide》中的"Initial conditions in Abaqus/Standard and Abaqus/Explicit"第 34.2.1 节。您只能在初始步骤中创建或修改初始孔隙比场。

**创建或编辑初始孔隙比场：**

1. 使用以下方法之一显示初始孔隙比场编辑器： - 要创建新的初始孔隙比场，请按照"创建预定义场"第 16.8.3 节中概述的程序进行操作（**类别**：`Other`；**所选步骤的类型**：`Void ratio`）。 - 要使用菜单或管理器编辑现有的初始孔隙比场，请参见"编辑依赖于步骤的对象"第 3.4.13 节。
2. 点击 **Point 1 distribution** 字段右侧的箭头，从出现的列表中选择您需要的选项： - 选择 **Uniform** 为所选区域定义均匀的孔隙比。 - 选择 **From output database file** 从先前具有孔隙比分量的 Abaqus 分析的输出数据库文件中读取孔隙比值。 - 选择 **User-defined** 使用用户子程序 [`VOIDRI`](../sub/sub-link.md#sub-xsl-voidri) 中的选项指定孔隙比。请参阅以下章节获取更多信息： - "指定常规作业设置"第 19.8.6 节 - 《Abaqus User Subroutines Reference Guide》中的"VOIDRI"第 1.1.58 节 - 选择解析场来定义空间变化的初始孔隙比。或者，您可以点击 ![](../graphics/ico_analyticalField.png) 创建新的解析场。（有关更多信息，请参见第 58 章"解析场工具集"。）

3. 对于均匀分布，请执行以下操作： 1. 从 **Elevation distribution** 选项中，选择 **Linear** 或 **Constant**。 2. 在 **Void ratio 1** 字段中，指定模型第一个区域的初始孔隙比。 3. 如果您选择了线性变化的高程分布，请指定以下选项： - 在 **Vertical coordinate 1** 字段中，指定模型中第一个指定初始孔隙比位置处的垂直位置。 - 在 **Point 2 distribution** 字段中，指定第二个高程处孔隙比的 **Uniform** 分布，或选择解析场来定义第二个高程处空间变化的初始孔隙比。如果您选择 **Uniform**，请指定模型中第二个位置的孔隙比和垂直位置。

4. 如果要从输出数据库文件指定孔隙比输出，请执行以下操作： 1. 指定要从中导入孔隙比值的文件名。 2. 从 **Step** 选项中，选择 **Last** 从输出数据库的最后一步读取孔隙比输出，或选择 **Specify** 并输入要从中读取孔隙比数据的步骤编号。 3. 从 **Increment** 选项中，选择 **Last** 从所选步骤的最后增量读取孔隙比输出，或选择 **Specify** 并输入要从中读取孔隙比数据的增量编号。 4. 开启 **Interpolate midside nodes** 以在不同的网格之间插值孔隙比值。

5. 对于用户定义的孔隙比分布，请执行以下操作： 1. 进入 Job 模块，并显示感兴趣的分析作业的作业编辑器。（有关更多信息，请参见"创建、编辑和操作作业"第 19.7 节。） 2. 在作业编辑器中，点击 **General** 选项卡，并指定包含定义预定义场的用户子程序的文件。有关更多信息，请参见"指定常规作业设置"第 19.8.6 节。**注意：**您只能在作业编辑器中指定一个用户子程序文件；如果您的分析涉及多个用户子程序，则必须将用户子程序组合到一个文件中，然后指定该文件。

6. 点击 **OK** 创建初始孔隙比场并关闭对话框。模型上出现沙棕色方块，表示您刚刚创建的初始孔隙比场。
