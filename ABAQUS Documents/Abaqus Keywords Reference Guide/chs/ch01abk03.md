# *ACOUSTIC WAVE FORMULATION





### *ACOUSTIC WAVE FORMULATION在具有入射波载荷的声学问题中指定公式类型。

此选项用于识别声学问题中入射波载荷公式的类型。

**产品：**Abaqus/Standard  Abaqus/Explicit  Abaqus/CAE  

**类型：**模型数据

**级别：**模型

**Abaqus/CAE：**模型属性

##### **参考：**

- ["声学和冲击载荷，" Abaqus Analysis User's Guide第34.4.6节](../usb/usb-link.md#usb-prc-pacoustic)

### **可选参数：**

TYPE

设置TYPE=SCATTERED WAVE（默认值）以获得将由入射波载荷产生的散射波场解。

设置TYPE=TOTAL WAVE以获得总声压波解。

**此选项没有关联的数据行。**
