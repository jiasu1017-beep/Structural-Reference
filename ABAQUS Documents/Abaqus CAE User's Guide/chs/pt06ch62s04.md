# 62.4 理解datum作为特征







datum是特征建模过程中有用的辅助构造工具，本身也是一个特征。因此，您可以使用Feature Manipulation工具集来删除、抑制和恢复datum。Abaqus/CAE会随零件或装配一起重新生成datum，并考虑底层几何体的任何更改。您可以编辑定义datum的任何数值参数；例如，用于定义datum点的*X*、*Y*和*Z*坐标。但是，datum始终由创建时选择的相同底层几何体定义；如果需要使用不同的几何体来定义datum，则必须删除该datum并创建一个新的。

您可以在"Part"模块或"Property"模块中的零件上创建datum，也可以在其他模块中的装配上创建datum。如果在零件上创建datum，则datum会随装配中该零件的实例一起移动。如果通过从零件实例中选择几何体在装配上创建datum，则datum会随装配中的实例一起移动。但是，如果通过在提示区域中输入坐标在装配上创建datum，则当实例移动时datum保持固定。

修改特征时，您应注意datum与修改特征之间的任何父子关系。例如，考虑[图62--4](pt06ch62s04.md#dtm-pca)，它显示了一个穿过两个圆弧中点的datum轴。定义datum轴位置的圆弧中点是datum轴的父特征。
