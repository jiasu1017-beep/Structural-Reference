# 19.5.4 优化过程管理器





**优化过程管理器**与 Abaqus/CAE 中的其他管理器类似，允许您执行以下操作：
- 创建和配置优化过程。
- 编辑所选优化过程。
- 复制、重命名或删除所选优化过程。

此外，**优化过程管理器**允许您执行以下操作：- 将 TOSCA 参数（`.par`）文件和 Abaqus 输入（`.inp`）文件的副本写入您的工作目录。
- 在提交优化过程之前验证优化，以确保优化任务已正确配置且 Abaqus 模型存在。
- 提交优化过程。
- 重新启动因优化或 Abaqus 分析外部问题（例如未能获得 Abaqus 许可证）而失败的优化过程。
- 监控优化过程的进度。
- 以可以传输到 CAD 系统或返回 Abaqus/CAE 的文件形式提取优化模型表面的平滑等值面网格表示。
- 将优化过程创建的优化结果和分析结果组合成可由可视化模块显示的单个结果输出数据库文件。
- 查看优化过程的结果。

您可以通过从主菜单栏选择 ****优化过程管理器**![](../graphics/images/arrow.gif)**管理器**** 来显示**优化过程管理器**。[图 19-7](pt03ch19s05s04.md#ana-optman) 显示了**优化过程管理器**的布局。

**图 19-7** **优化过程管理器**。

![](../graphics/ana-optman-nls.png)

**优化过程管理器**的四列显示以下内容：

**名称**

**名称**列显示优化过程的名称。单击**重命名**可以重命名所选优化过程。

**模型**

**模型**列显示与优化过程关联的 Abaqus/CAE 模型。

**任务**

**任务**列显示与优化过程关联的优化任务。

**状态**

**状态**列显示优化过程中作业的状态，并在优化过程运行时持续更新。状态可以是以下之一：

**无**

优化过程尚未提交进行分析。

**已提交**

优化过程正在提交。

**正在运行**

优化过程已提交并正在运行。

**已完成**

优化过程已运行完成。

**中止**

由于某个模型中的致命错误或磁盘空间不足等问题，优化过程已中止。

**已终止**

优化过程已被用户终止。
