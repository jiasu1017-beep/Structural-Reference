# 5.4 自定义视图 triad





如下所示的视图 triad 是一组三个相互垂直的轴，指示你当前显示的模型视图的方向。*X*、*Y* 和 *Z* 标签分别对应于 1、2 和 3 方向。当你旋转模型视图时，triad 会变化以指示新方向。有关使用旋转工具的更多信息，请参阅["旋转视图，" 第 5.6.3 节](pt01ch05s06hlb03.md)。

![](../graphics/dsp-triad.png)

视图 triad 和 3D 罗盘都指示视图方向，并且它们始终相互对齐。你可以直接操作视口中 3D 罗盘的方向，从而更改模型的视图（请参阅["3D 罗盘，" 第 5.3 节](pt01ch05s03.md)）。视图 triad 仅作为参考；它随 3D 罗盘旋转，但不能直接操作。

你可以使用****Viewport**![](../graphics/images/arrow.gif)**Viewport Annotation Options****菜单项来请求或抑制 triad 的显示，并控制 triad 的大小、位置和外观。你也可以控制 triad 的标签，包括它们的颜色和字体。

**要控制 triad 显示选项：**

1. 从主菜单栏中，选择****Viewport**![](../graphics/images/arrow.gif)**Viewport Annotation Options****。**视口标注选项**对话框出现。
2. 切换**显示 triad** 以在当前视口中显示或抑制 triad。当**显示 triad**被切换开启时，triad 选项变得可用。
3. 单击 **Triad** 选项卡。Abaqus/CAE 显示 triad 视图选项。
4. 输入 **Triad 大小**作为视口大小的百分比。当你调整视口大小时，triad 的大小会相应变化。最小允许 triad 大小是视口的 1%，最大允许 triad 大小是视口的 50%。
5. 分别在 **% 视口 X** 和 **% 视口 Y** 框中输入 triad X 和 Y 位置的百分比值。**% 视口 X** 的值为 0 会将 triad 原点移动到视口最左侧，值为 100 会将其移动到最右侧。**% 视口 Y** 的值为 0 会将 triad 原点移动到视口最底部，值为 100 会将其移动到最顶部。
6. 选择 triad 标签的颜色：
   1. 单击颜色样本![](../graphics/usi-colorsample.png)。Abaqus/CAE 显示**选择颜色**对话框。
   2. 使用**选择颜色**对话框中的方法选择新颜色。有关详细信息，请参阅["自定义颜色，" 第 3.2.9 节](pt01ch03s02s09.md)。
   3. 单击**确定**关闭**选择颜色**对话框。颜色样本会变为所选颜色。
7. 单击**标签**字段旁边的箭头，然后为 triad 选择数字或字母标签。指定样式会显示在**标签**字段中。
8. 单击**设置标签字体**以使用出现的对话框设置字体类型、大小和样式。
9. 单击**应用**以实施更改。你的更改会在会话期间保存。
