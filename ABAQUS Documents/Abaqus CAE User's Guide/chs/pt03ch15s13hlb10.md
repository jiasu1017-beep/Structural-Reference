# 15.13.12 定义流体交换相互作用





流体交换相互作用允许您定义流体在空腔与周围环境之间或在两个空腔之间的移动。流体交换相互作用无法在后续分析步骤中修改或停用。详细讨论请参见"Abaqus Analysis User's Guide"第11.5.3节"Fluid exchange definition"。

**定义流体交换相互作用：**

1. 从主菜单栏中，选择****相互作用**![](../graphics/images/arrow.gif)**创建**。**提示：**您也可以使用相互作用模块工具箱中的![](../graphics/ico_interaction.png)工具创建流体腔体相互作用。
2. 在出现的**创建相互作用**对话框中，执行以下操作：- 为相互作用命名。有关命名对象的更多信息，请参见"使用基本对话框组件"，第3.2.1节。- 选择**初始**作为将创建相互作用的步骤。- 选择**流体交换**类型的相互作用。
3. 单击**继续**关闭**创建相互作用**对话框。Abaqus/CAE打开**编辑相互作用**对话框。
4. 切换打开所需的交换定义，**至环境**或**空腔之间**。
5. 选择主流体腔体相互作用，**流体腔体相互作用1**。
6. 如果您正在定义空腔之间的交换，请选择**流体腔体相互作用2**。
7. 选择**流体交换属性**。如果需要，单击![](../graphics/ico_interactionProperty.png)创建相互作用属性；有关更多信息，请参见"定义流体交换相互作用属性"，第15.14.5节。
8. 输入**有效交换面积**的值。有效交换面积表示流体交换所通过的管道的横截面积。
9. 单击**确定**创建相互作用并关闭编辑器。代表您刚创建的流体交换相互作用的符号出现在视口中。有关更多信息，请参见"理解代表相互作用、约束和连接器的符号"，第15.10节。
