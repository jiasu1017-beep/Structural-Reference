# 16.9.9 Defining a gravity load







You can create a gravity load to define a uniform acceleration in a fixed direction. Abaqus calculates the loading using the acceleration magnitude that you enter in the gravity load definition and the density specified in the material definition.

**To create or edit a gravity load:**

1. Display the gravity load editor using one of the following methods: - To create a new gravity load, follow the procedure outlined in ["Creating loads," Section 16.8.1](pt03ch16s08hlb01.md) (**Category**: `Mechanical`; **Types for Selected Step**: `Gravity`). - To edit an existing gravity load using menus or managers, see ["Editing step-dependent objects," Section 3.4.13](pt01ch03s04hlb05.md).
2. By default, a gravity load is applied to the whole model. If desired, you can apply a gravity load to particular regions of a model: 1. Click ![](../graphics/ico_selectBlue.png). 2. Select the region to which you want to apply the load, as described in ["Creating loads," Section 16.8.1](pt03ch16s08hlb01.md). 3. Click **Done** in the prompt area. **Note:**You cannot apply gravity loads to individual point masses. To include point masses in a gravity load, apply the load to the whole model.
3. Click the arrow to the right of the **Distribution** field, if available; and select the option of your choice from the list that appears: - Select **Uniform** to define a load that is uniform over the region. - Select an analytical field to define a spatially varying load. Only analytical fields that are valid for this load type are displayed in the selection list. Alternatively, you can click ![](../graphics/ico_analyticalField.png) to create a new analytical field. (See [Chapter 58, "The Analytical Field toolset](pt06ch58.md)," for more information.)
4. In the **Component 1**, **Component 2**, and (if you are working with a model in three-dimensional space) **Component 3** text fields, enter the components of the acceleration in each direction: - If you are working in three-dimensional or two-dimensional space, the **Component 1**, **Component 2**, and **Component 3** fields correspond to the 1-, 2-, and (if applicable) 3-directions. - If you are working in axisymmetric space, only the **Component 2** text field is available. **Component 2** corresponds to the axial direction. If you leave a text field blank, a value of zero is assigned to that direction automatically. However, you must enter at least one nonzero component in the editor to define the load.
5. If desired, click the arrow to the right of the **Amplitude** field, and select the amplitude of your choice from the list that appears. Alternatively, you can click ![](../graphics/ico_amplitudeSmall.png) to create a new amplitude. (See [Chapter 57, "The Amplitude toolset](pt06ch57.md)," for more information.)
6. Click **OK** to save your data and to exit the editor.
