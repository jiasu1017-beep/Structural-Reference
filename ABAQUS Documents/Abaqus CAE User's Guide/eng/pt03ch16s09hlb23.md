# 16.9.23 Defining a fluid reference pressure







You can create a fluid reference pressure load in an Abaqus/CFD analysis to specify the hydrostatic pressure level at a point in the model.

**To create or edit a fluid reference pressure:**

1. Display the fluid reference pressure load editor using one of the following methods: - To create a new fluid reference pressure, follow the procedure outlined in ["Creating loads," Section 16.8.1](pt03ch16s08hlb01.md) (**Category**: `Fluid`; **Types for Selected Step**: `Fluid reference pressure`). - To edit an existing fluid reference pressure load using menus or managers, see ["Editing step-dependent objects," Section 3.4.13](pt01ch03s04hlb05.md). To edit the region to which the load is applied, see ["Editing the region to which a prescribed condition is applied," Section 16.8.4](pt03ch16s08hlb04.md).
2. In the **Magnitude** field, specify the magnitude of the fluid reference pressure.
3. Click **OK** to save your data and to exit the editor.
