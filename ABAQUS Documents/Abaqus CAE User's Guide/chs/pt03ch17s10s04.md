# 17.10.4 什么是四面体边界网格？





当Abaqus/CAE使用四面体单元在实体上生成自由网格时，网格划分过程包括两个阶段：

1. 在实体区域的面上生成三角形边界网格。
2. Abaqus/CAE使用三角形作为四面体单元的面来生成四面体网格。

相同的过程用于为自底向上网格区域的集合面创建四边形边界网格和六面体实体网格（更多信息，请参见["为自底向上区域创建边界网格，"第17.11.5节](pt03ch17s11s05.md)）。

如果您的零件复杂，生成自由四面体网格可能非常耗时。预览边界面上的网格可以帮助您在为整个零件生成网格之前识别问题。您可以通过在提示区域中切换**预览边界网格**来在网格划分过程的第一阶段后预览边界面上的三角形。如果网格可接受，您可以继续对区域内部进行网格划分。如果网格不可接受，或者某些区域网格划分失败，Abaqus/CAE提供了多种工具来纠正问题。更多信息，请参见["我可以使用边界网格做什么？"第17.10.5节](pt03ch17s10s05.md)。

如果您决定预览区域的三角形边界网格，Abaqus/CAE允许您选择要网格划分的面，即使您的最终目的是对实体进行网格划分。当您切换**预览边界网格**时，Abaqus/CAE会自动更改提示区域中的选择过滤器以选择**面**。更多信息，请参见["根据对象类型过滤选择，"第6.3.2节](pt01ch06s03s01.md)。

当选择**网格默认值**颜色映射时，Abaqus/CAE使用白色显示边界网格以表示边界单元，这与Abaqus/CAE用于表示最终网格的青色形成对比。当您查询四面体边界网格时，Abaqus/CAE将单元称为**三角形边界单元**。相比之下，当您查询最终网格时，Abaqus/CAE将单元称为**线性四面体单元**或**二次四面体单元**。边界面上的三角形没有几何阶次的概念。
