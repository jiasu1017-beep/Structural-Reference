# 17.11.5 为自底向上区域创建边界网格





当您为几何区域创建自底向上网格时，网格划分过程包括两个阶段：

1. 在源侧生成四边形单元的边界网格——如果包含的话，也在扫掠网格的连接边上生成。
2. Abaqus/CAE使用四边形单元作为六面体单元的面来生成六面体网格。

创建边界网格允许您预览生成自底向上网格的第一阶段。查看边界单元可以帮助您识别可能阻止生成自底向上网格的问题。自底向上网格区域的集合面的边界网格功能与Abaqus/CAE中的四面体边界网格功能相同，只是自底向上边界网格由四边形单元而不是四面体单元组成（有关四面体边界网格的更多信息，请参见["什么是四面体边界网格？"第17.10.4节](pt03ch17s10s04.md)和["我可以使用边界网格做什么？"第17.10.5节](pt03ch17s10s05.md))。

要为自底向上区域的面创建边界网格，您必须使用自顶向下区域网格划分过程。

**创建边界网格：**

1. 从主菜单栏，选择****网格**![](../graphics/images/arrow.gif)**区域****。
2. 在提示区域中切换**预览边界网格**。Abaqus/CAE自动将默认选择选项从**区域**切换为**面**。
3. 选择要为其创建边界网格的自底向上区域的几何面。
4. 在提示区域中单击**完成**。Abaqus/CAE在选定面上创建二维网格。

当选择**网格默认值**颜色映射时，Abaqus/CAE使用白色显示边界网格以表示四边形单元，这与Abaqus/CAE用于表示最终实体网格的青色形成对比。当您查询自底向上边界网格时，Abaqus/CAE将二维边界单元称为**四边形边界单元**。相比之下，当您查询最终网格时，Abaqus/CAE将三维实体单元称为**线性六面体单元**。
