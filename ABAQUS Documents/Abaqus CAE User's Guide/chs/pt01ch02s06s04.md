# 2.6.4 查找有关关键字的信息





关键字浏览器是一个可滚动的表格，包含以下信息：
- 每个关键字的用途。
- 包含与每个关键字相关功能的Abaqus/CAE模块或工具集。

要查看关键字浏览器，请从主菜单栏选择****帮助**![](../graphics/images/arrow.gif)**关键字浏览器****。例如，您可以使用关键字浏览器验证[*ELASTIC](../key/key-link.md#usb-kws-melastic)选项允许您指定弹性材料属性，并且属性模块是与此关键字关联的Abaqus/CAE模块。

关键字浏览器还包含指向在线文档相关章节的超链接。您可以单击表格中的特定关键字来显示有关该关键字功能的详细信息。您也可以单击表格中模块或工具集的名称来查看[Abaqus/CAE用户指南](book01.md)中的相关文档。

**显示关键字浏览器：**

1. 从主菜单栏，选择****帮助**![](../graphics/images/arrow.gif)**关键字浏览器****。[Abaqus/CAE用户指南](book01.md)会打开到Abaqus关键字及其关联模块的表格。
2. 在**关键字**列中，单击感兴趣的关键字可查看描述该关键字的在线文档。
3. 在**模块**列中，单击感兴趣模块或工具集的名称可查看有关该模块或工具集的在线文档。
