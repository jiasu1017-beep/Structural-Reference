# 66 过滤器工具集







过滤器允许您在分析模型时移除无关的场输出数据或历史输出数据——噪声——而不会在所需数据范围内失去分辨率。您还可以在数据保存到输出数据库（`.odb`）文件之前使用过滤器对场输出或历史输出进行过滤；因此，过滤器还可以减小输出数据库的大小。过滤器工具集允许您在 Step（步骤）模块中创建和管理过滤器。本章涵盖以下主题：
- ["过滤场和历史数据，" 第 66.1 节"](pt06ch66s01.md)
- ["对场和历史数据应用边界值，" 第 66.2 节"](pt06ch66s02.md)
- ["创建过滤器，" 第 66.3 节"](pt06ch66s03.md)
