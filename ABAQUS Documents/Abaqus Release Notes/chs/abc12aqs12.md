# 12.10 选择面积质心作为自由体切割的求和点





**产品：** Abaqus/CAE

**优点：** 提供了一个新选项来指定自由体切割的求和点。

**描述：** 对于自由体计算中的求和点，您现在可以指定使用面积质心或节点平均值。

在之前的版本中，**切割质心**选项在**编辑自由体切割**对话框中将求和点放置在所选自由体横截面元素的节点坐标平均值处。现在通过选择**节点平均值**选项来实现该行为。

现在，当您为求和点选择**切割质心**选项时，自由体计算中使用由所选面外部边界边形成的Polygon的面积质心。[图12–2](abc12aqs10.md#rnb614-freebody)显示了自由体切割中求和点的可用选项。
