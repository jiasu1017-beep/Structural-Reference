# 14.7.1 通用求解器控制





您可以自定义控制Abaqus中收敛和时间积分精度算法的众多变量。默认的求解器控制通常工作良好，但自定义这些控制可能会得到更经济的求解结果，或帮助您为特别困难的分析获得求解。

**注意：**这些选项仅适用于通用Abaqus/Standard分析步骤。

您可以通过从主菜单栏选择**其他**![](../graphics/images/arrow.gif)**通用求解器控制**来访问求解器控制。更多信息，请参见["分析收敛控制，"第7.2节，Abaqus分析用户指南](../usb/usb-link.md#usbaconvcontrols)。

**警告：**求解器控制面向有经验的分析师，应非常小心地使用。这些控制的默认设置适用于大多数非线性分析。不恰当地更改这些值可能会大大增加分析的计算时间，或产生不准确的结果。

有关设置通用求解器控制的详细说明，请参见["自定义通用求解器控制，"第14.15.1节](pt03ch14s15hlb01.md)。
