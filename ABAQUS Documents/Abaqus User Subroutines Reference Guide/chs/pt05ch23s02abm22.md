### 概述

各向异性屈服和/或蠕变：
- 可用于在不同方向表现出不同屈服和/或蠕变行为的材料；
- 通过应用于Hill势函数的用户定义应力比引入；
- 只能与金属塑性结合使用，在Abaqus/Standard中，还必须与金属蠕变材料模型结合使用；
- 在Abaqus/Explicit中可用于非线性各向同性/运动硬化模型（["承受循环加载的金属模型，" 第23.2.2节"](pt05ch23s02abm18.md)）；和
- 可与Abaqus/Explicit中的逐步损伤和失效模型结合使用（["延性金属的损伤和失效：概述，" 第24.2.1节"](pt05ch24s02abm41.md)）以指定不同的损伤起始准则和损伤演化定律，允许材料刚度的逐步降解以及从网格中删除单元。

### 屈服和蠕变应力比

各向异性屈服或蠕变行为通过使用屈服或蠕变应力比R~ij~建模。在各向异性屈服的情况下，屈服比是相对于参考屈服应力σ^0~（为金属塑性定义给出）定义的，因此如果在仅施加σ~11~作为唯一非零应力时，相应的屈服应力为σ~22~ = R~22~ σ^0~。塑性流动法则在下面定义。

在各向异性蠕变的情况下，R~ij~是用于在计算蠕变应变率时缩放应力值的蠕变比。因此，如果σ~11~是唯一非零应力，则用户定义蠕变定律中使用的等效应力σ¯为σ¯ = R~11~ σ~11~。

屈服和蠕变应力比可以定义为常数或温度和预定义场变量的表格函数。必须使用局部方向来定义各向异性方向（见["方向，" 第2.2.5节"](pt01ch02s02aus15.md)）。

### 各向异性屈服

Hill势函数是Mises函数的简单扩展，可以用直角笛卡尔应力分量表示为：

f = F σ~22~^2 + G σ~11~^2 + H (σ~11~ - σ~22~)^2 + 2N σ~12~^2 = 1

其中F、G、H和N是通过材料在不同方向的测试获得的常数。它们定义为：

F = 1/2 (1/R~22~^2 + 1/R~11~^2 - 1/R~11~^2)

G = 1/2 (1/R~22~^2 + 1/R~11~^2 - 1/R~22~^2)

H = 1/2 (1/R~12~^2 + 1/R~21~^2 - 1/R~11~^2)

其中每个R~ij~是当σ~ij~被施加为唯一非零应力分量时测量的屈服应力值；σ^0~是为金属塑性定义指定的用户定义参考屈服应力；R~11~、R~22~、R~33~、R~12~、R~13~和R~23~是各向异性屈服应力比；且R~11~ = R~22~ = R~33~ = 1。因此，六个屈服应力比必须按以下顺序提供：

R~11~ = 1

R~22~ = σ~22~^y/σ^0~

R~33~ = σ~33~^y/σ^0~

R~12~ = σ~12~^y/σ^0~

R~13~ = σ~13~^y/σ^0~

R~23~ = σ~23~^y/σ^0~

由于屈服函数的形式，所有这些比值必须为正。如果常数F、G和H为正，屈服函数始终是良好定义的。然而，如果这些常数中的一个或多个为负，屈服函数可能在某些应力状态下未定义，因为平方根下的量为负。

流动法则是：

dε~ij~^p^ = dλ ∂f/∂σ~ij~

其中，根据上面f的定义：

∂f/∂σ~11~ = 2 [G σ~11~ + H (σ~11~ - σ~22~)]

### 各向异性蠕变

对于Abaqus/Standard中的各向异性蠕变，Hill函数可以表示为：

f = F σ~22~^2 + G σ~11~^2 + H (σ~11~ - σ~22~)^2 + 2L σ~23~^2 + 2M σ~13~^2 + 2N σ~12~^2 = σ¯^2

其中σ¯是等效应力，F、G、H、L、M和N是通过材料在不同方向的测试获得的常数。常数用与上述各向异性屈服相同的广义关系定义；然而，参考屈服应力σ^0~被单轴等效偏量应力σ¯（在蠕变定律中找到）代替，且R~11~ = R~22~ = R~33~ = 1，R~12~、R~13~、R~23~、R~21~、R~31~和R~32~被称为"各向异性蠕变应力比"。因此，六个蠕变应力比必须按以下顺序提供：

R~11~ = 1

R~22~ = σ~22~^c/σ¯

R~33~ = σ~33~^c/σ¯

R~12~ = σ~12~^c/σ¯

R~13~ = σ~13~^c/σ¯

R~23~ = σ~23~^c/σ¯

您必须定义每个方向上的比值R~ij~，这些比值将在计算蠕变应变率时用于缩放应力值。如果所有六个R~ij~值都设为1，则获得各向同性蠕变。

### 基于应变比（Lankford的r值）定义各向异性屈服行为

如上所述，Abaqus中的Hill各向异性塑性势从用户输入定义，该输入由相对于参考应力的不同方向屈服应力比组成。然而，在某些情况下，如板材成形应用，通常会发现各向异性材料数据以宽度应变与厚度应变比的形式给出。然后需要数学关系将应变比转换为可以输入Abaqus的应力比。

在板材成形应用中，我们通常关注平面应力条件。考虑x和y为板材平面中的"轧制"和"横向"方向；z是厚度方向。从设计角度来看，通常需要的各向异性类型是板材在平面内是各向同性的，在厚度方向具有增加的强度，这通常称为横向各向异性。另一种各向异性的特征是板材平面内不同方向具有不同的强度，称为平面各向异性。

在板材平面中沿x方向进行的简单拉伸测试中，此势的流动法则（上面给出）定义了增量应变比（假设小弹性应变）为：

dε~22~/dε~11~ = (R~11~ (σ~11~ - σ~22~) - R~22~ σ~22~) / (R~22~ (σ~11~ - σ~22~) - R~11~ σ~11~)

因此，宽度与厚度应变之比（通常称为Lankford的r值）为：

r = dε~22~/dε~33~ = (R~11~ (σ~11~ - σ~22~) - R~22~ σ~22~) / (R~22~ (σ~11~ - σ~22~) - R~11~ σ~11~)

类似地，对于在板材平面中沿y方向进行的简单拉伸测试，增量应变比为：

dε~11~/dε~22~ = (R~22~ (σ~22~ - σ~11~) - R~11~ σ~11~) / (R~11~ (σ~22~ - σ~11~) - R~22~ σ~22~)

且：

r~y~ = dε~11~/dε~33~ = (R~22~ (σ~22~ - σ~11~) - R~11~ σ~11~) / (R~11~ (σ~22~ - σ~11~) - R~22~ σ~22~)

#### 横向各向异性

横向各向异性材料是其中r~x~ = r~y~的材料。如果我们定义金属塑性模型中的σ^0~等于σ~11~^y，

R~11~ = 1

R~22~ = (1 + r~x~)/r~x~

且使用上述关系：

F = G = r~x~/[2 (1 + r~x~)]

H = 1/[2 (1 + r~x~)]

如果r~x~ = r~y~ = 1（各向同性材料），F = G = H = 1/3，恢复Mises各向同性塑性模型。

#### 平面各向异性

在平面各向异性的情况下，r~x~和r~y~不同，且R~11~和R~22~也不同。如果我们定义金属塑性模型中的σ^0~等于σ~11~^y，

R~11~ = 1

R~22~ = (1 + r~x~)/r~x~

且使用上述关系，我们获得：

R~12~^2 = (r~y~ (1 + r~x~))/[r~x~ (1 + r~y~)]

如果r~x~ = r~y~ = 1，F = G = H = 1/3，恢复Mises各向同性塑性模型。

#### 一般各向异性

到目前为止，我们只考虑了沿各向异性轴施加的加载。为了在平面应力中推导更一般的各向异性模型，板材必须在其平面内沿另一个方向加载。假设我们在与x方向成角度α的方向进行简单拉伸测试；然后，从平衡考虑，我们可以写出非零应力分量：

σ~11~ = σ~22~ = σ cos^2^α

σ~12~ = σ sinα cosα

将这些值代入流动方程并假设小弹性应变，得到：

σ~11~ = σ cos^2^α

σ~22~ = σ sin^2^α

σ~12~ = σ sinα cosα

假设小的几何变化，宽度应变增量（加载方向α垂直方向上的应变增量）写为：

dε~22~ = dε~11~ cos^2^α + dε~33~ sin^2^α + ...

且加载方向α的Lankford r值为：

r~α~ = dε~22~/dε~33~ = (dε~22~/dε~11~) cos^2^α + sin^2^α / [(dε~33~/dε~11~) cos^2^α + sin^2^α]

更常见的测试之一是加载方向为45°的情况。在这种情况下：

r~45°~ = (r~x~ + 2r~xy~ + r~y~) / [2 (1 + r~xy~)]

如果金属塑性模型中的σ^0~等于σ~11~^y，r~xy~ = r~x~。r~x~和r~y~如上述为横向或平面各向异性定义，且使用上述关系：

R~12~^2 = (r~45°~ (1 + r~x~))/[r~x~] - (1 + r~x~)/2

### 逐步损伤和失效

在Abaqus/Explicit中，各向异性屈服可与["延性金属的损伤和失效：概述，" 第24.2.1节"](pt05ch24s02abm41.md)中讨论的逐步损伤和失效模型结合使用。该功能允许指定一个或多个损伤起始准则，包括延性、剪切、成形极限图（FLD）、成形极限应力图（FLSD）和Mschenborn-Sonne成形极限图（MSFLD）准则。损伤起始后，材料刚度根据指定的损伤演化响应逐步降解。该模型提供两种失效选择，包括由于结构撕裂或扯裂而从网格中删除单元。逐步损伤模型允许材料刚度的平滑降解，使它们适用于准静态和动态情况。

### 单元

可以为Abaqus中可与经典金属塑性模型（["经典金属塑性，" 第23.2.1节"](pt05ch23s02abm17.md)）一起使用的任何单元定义各向异性屈服，Abaqus/Explicit中的一维单元（梁和桁架）除外。在Abaqus/Standard中，它也可以与线性运动硬化塑性模型（["承受循环加载的金属模型，" 第23.2.2节"](pt05ch23s02abm18.md)）一起使用的任何单元定义，但不能与非线性各向同性/运动硬化模型一起使用。同样，可以为Abaqus/Standard中可与经典金属蠕变模型（["率相关塑性：蠕变和肿胀，" 第23.2.4节"](pt05ch23s02abm20.md)）一起使用的任何单元定义各向异性蠕变。
