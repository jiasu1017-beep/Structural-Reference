# 64.1.2 操作单元





网格编辑工具集提供以下工具，允许您操作网格中的单元：
- 创建单元。您必须指定要创建的单元的形状，并且必须按适合该单元形状的顺序选择节点。
- 删除单元。您可以选择删除在所选单元被删除后将变得不与任何单元关联的任何节点。
- 翻转壳单元的表面法线方向。
- 折叠四边形或三角形单元的所选边缘。折叠边缘对于从四边形和三角形单元中移除细长单元非常有用，如[图64-1](pt06ch64s01s02.md#edt-meshedit-collapse-edge)所示。 **图64-1** 折叠边缘以从四边形和三角形单元中移除细长单元。 ![](../graphics/edt-meshedit-collapse-edge-nls.png) 当您预览四面体边界网格时，可以折叠三角形单元的所选边缘以移除细长单元。在某些情况下，当您使用前进front算法创建全四边形网格时，Abaqus/CAE可能会生成具有短边缘的四边形单元。您可以折叠短边缘并创建一个形状良好的三角形单元，如[图64-2](pt06ch64s01s02.md#edt-meshedit-collapse-quadedge)所示。 **图64-2** 折叠四边形单元的边缘。 ![](../graphics/edt-meshedit-collapse-quadedge-nls.png)
- 将四边形或三角形单元的所选边缘分割成两部分。您可以在其中点分割边缘，也可以在分割位置点击。您可以使用工具组合来清理网格。例如，[图64-3](pt06ch64s01s02.md#edt-meshedit-splitedge)说明瞭如何分割边缘然后折叠 resulting edge以移除一个狭长的三角形单元。 **图64-3** 分割和折叠三角形单元的边缘。 ![](../graphics/edt-meshedit-splitedge-nls.png) 如果您在"网格"模块中编辑原生网格，Abaqus/CAE会将新节点投影到几何体上。因此，您可以使用此工具来改进弯曲边缘周围的粗糙网格，如[图64-4](pt06ch64s01s02.md#edt-edge-cleanhole)所示。 **图64-4** 通过分割边缘改进孔周围的网格。 ![](../graphics/edt-edge-cleanhole-nls.png)
- 交换一对相邻三角形单元的对角线，如[图64-5](pt06ch64s01s02.md#edt-meshedit-flipdiagonal)所示。如果它们阶次相同，您可以交换一阶或二阶相邻三角形单元的对角线。此外，相邻三角形单元必须具有一致的法线。如有需要，您可以翻转法线以使其一致。 **图64-5** 交换一对相邻三角形单元的对角线。 ![](../graphics/edt-meshedit-flipdiagonal.png)
- 将四边形单元分割成两个三角形单元，如[图64-6](pt06ch64s01s02.md#edt-meshedit-splitquad)所示。您不能分割5节点四边形单元，也不能分割垫片单元。 **图64-6** 将四边形单元分割成两个三角形单元。 ![](../graphics/edt-meshedit-splitquad.png)
- 将两个相邻三角形单元合并为一个四边形单元，如[图64-7](pt06ch64s01s02.md#edt-meshedit-combinetris)所示。您可以合并一阶或二阶单元，但不能合并两者的组合。此外，只有当相邻三角形单元具有一致的法线时，您才能合并它们。如有需要，您可以翻转法线以使其一致。 **图64-7** 将两个相邻三角形单元合并为一个四边形单元。 ![](../graphics/edt-meshedit-combinetris.png)
- 定向连续壳、粘聚、圆柱形或垫片网格的堆叠方向。您只能堆叠六面体、楔形和四边形单元以形成连续壳、粘聚、圆柱形或垫片网格。因此，您可以使用此工具定向六面体、楔形和四边形单元的堆叠方向。如果您已为区域分配了圆柱形单元或二阶垫片单元，则必须先将分配更改为常规单元，然后才能重新定向堆叠方向。然后，您可以通过重新分配圆柱形单元或二阶垫片单元将区域转换回圆柱形或垫片网格。
- 在孤兒网格中对所选单元重新编号。您可以通过指定起始标签和增量来对所选单元重新编号，或者通过指定值偏移现有标签。

**提示：**网格编辑撤销功能可以回滚您对网格中单元所做的任何更改。有关更多信息，请参见["撤销或重做网格编辑工具集中的更改，" 第64.9节](pt06ch64s09.md)。

有关操作单元的详细说明，请参见["编辑单元，" 第64.6节](pt06ch64s06.md)。
