# 20.2.3 aStudy.define()





### 20.2.3 **_aStudy_.define()**: 定义参数化研究的参数。

**产品:** Abaqus/Standard  Abaqus/Explicit

此命令用于定义为参数化研究指定的参数。

##### **参考**

- ["参数化研究脚本," 第20.1.1节](pt04ch20s01aus108.md)

### **命令:**

**_aStudy_.define** (*token*, *additional data*)

### **标记:**

CONTINUOUS

使用此标记指示参数是连续值的。

DISCRETE

使用此标记指示参数是离散值的。

PRINT

使用此标记打印参数定义。

##### **CONTINUOUS的附加数据:**

### **必需数据:**

**par**

设置 **par** 等于正在定义的参数或参数序列。如果指定单个参数，必须用匹配引号括起来；例如，`'par1'`。如果指定参数列表，必须将其放在括号或方括号内，并且必须包含用匹配引号括起来并用逗号分隔的参数名称；例如，`('par1', 'par2', 'par3')` 或 `['par1', 'par2', 'par3']`。

### **可选数据:**

**domain**

设置 **domain** 等于参数的最小值和最大值，用逗号分隔并用括号或方括号括起来；例如，`(10., 20.)` 或 `[10., 20.]`。

如果从此命令中省略 **domain**，并且稍后使用需要域定义的方法对参数进行采样，则必须在 **sample** 命令中指定域。

**reference**

设置 **reference** 等于参数的参考值。

如果从此命令中省略 **reference**，并且稍后使用需要参考定义的方法对参数进行采样，则必须在 **sample** 命令中指定参考。

##### **DISCRETE的附加数据:**

### **必需数据:**

**par**

设置 **par** 等于正在定义的参数或参数列表。如果指定单个参数，必须用匹配引号括起来；例如，`'par1'`。如果指定参数列表，必须将其放在括号或方括号内，并且必须包含用匹配引号括起来并用逗号分隔的参数名称；例如，`('par1', 'par2', 'par3')` 或 `['par1', 'par2', 'par3']`。

### **可选数据:**

**domain**

设置 **domain** 等于参数可能具有的值序列。值必须用逗号分隔并用括号或方括号括起来；例如，`(1., 2., 5., 3.)` 或 `[1., 2., 5., 3.]`。

如果从此命令中省略 **domain**，并且稍后使用需要域定义的方法对参数进行采样，则必须在 **sample** 命令中指定域。

**reference**

设置 **reference** 等于参数值序列中的索引。索引从零开始，因此序列的第一个值对应索引零，序列的最后一个值对应等于序列中值数量减一的索引。

如果从此命令中省略 **reference**，并且稍后使用需要参考定义的方法对参数进行采样，则必须在 **sample** 命令中指定参考。

##### **PRINT的附加数据:**

### **可选数据:**

**par**

设置 **par** 等于要打印其定义的参数或参数序列的名称。如果指定单个参数，必须用匹配引号括起来；例如，`'par1'`。如果指定参数序列，必须将其放在括号或方括号内，并且必须包含用匹配引号括起来并用逗号分隔的参数名称；例如，`('par1', 'par2', 'par3')` 或 `['par1', 'par2', 'par3']`。

如果省略 **par**，则打印参数化研究中所有参数的参数定义。
