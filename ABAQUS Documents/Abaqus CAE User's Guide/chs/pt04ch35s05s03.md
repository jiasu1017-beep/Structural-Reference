# 35.5.3 可视化壳厚度





您可以在Abaqus/CAE中查看中面模型的厚度。厚度表示使用在部件上定义的偏移和厚度或通过截面分配属性来创建新的模型视图。可视化部件厚度以及参考表示可以帮助您确定中面模型是否准确表示了您正在替换的实体模型。

轴对称壳截面（平滑曲线面）和沿着直边缘连接的面挤压壳如图35-26所示。如图所示，平滑曲线壳的壳厚度显示是准确的；但是，在面沿边缘连接的地方，显示的厚度在面的角落之间包含间隙和重叠。

**图35-26** 在平滑曲线面和沿边缘连接的面上显示壳厚度。
