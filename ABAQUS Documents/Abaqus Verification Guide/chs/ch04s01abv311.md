# 4.1.36 VUINTER





### 4.1.36 [`VUINTER`](../sub/sub-link.md#sub-xsl-vuinter)

**产品：**Abaqus/Explicit  

### 测试的功能

用户子程序用于定义接触界面行为。

### 问题描述

此示例中使用用户子程序 [`VUINTER`](../sub/sub-link.md#sub-xsl-vuinter) 对具有均匀厚度的机械柔顺、热传导接触界面材料进行建模。界面材料假定粘合到两个接触表面中的每一个。界面材料表现出弹塑性行为，在法向接触方向具有线性硬化，对相对滑动具有纯弹性阻力。界面的膜应变不影响传递到表面的应力。界面材料是热传导的；电导率保持恒定，与界面处的间隙或压力无关。

使用此界面模型，相对于主表面的指定初始间隙距离内的所有从节点在整个分析过程中保持粘合状态。其他从节点未粘合（从不施加接触力或热通量）。初始间隙（参见示例用户子程序中的 `gapInit` 变量）考虑了表面厚度（在这些示例中等于零）以及指定的界面厚度。假定界面的初始应变为零。Abaqus/Explicit 不会进行无应变调整来解析使用用户子程序 [`VUINTER`](../sub/sub-link.md#sub-xsl-vuinter) 的接触对的初始过盈或间隙。

界面厚度的指定是可选的；这里为了方便使用它，以便在计算每个节点的穿透或间隙时使用界面厚度（变量 `rdisp(1,...)`）。或者，您可以通过构建一个包含每个节点偏移值的状态变量在用户子程序中对界面厚度进行建模。该偏移可以是初始穿透和节点处界面厚度的函数（例如，将偏移设置为初始穿透的负值）。实际穿透将是 `rdisp(1,...)` 中给定的值与存储的节点偏移值之和。由于不对节点进行无应变调整，此过程提供了一种方便的方法来消除因节点坐标不准确而产生的虚假初始接触力，从而无需在构建模型时精确放置表面节点。

法向的应变增量在用户子程序内计算为接触穿透变化除以指定的界面厚度。该厚度是界面模型的属性。为了一致性，应将此厚度设置为与这些示例中的界面厚度相同的值。界面横向剪切对应的应变增量分量同样计算为适当的滑动增量分量除以指定的界面厚度。

热通量通过将界面材料的热导率乘以节点面积和从节点与主接触点之间的温度差，再除以初始界面厚度来计算。不考虑摩擦生热等效应。

为该界面模型指定的属性完整列表，按接触属性定义第二数据行中指定值的顺序如下：

1. 间隙截止距离。只有初始间隙小于此距离的从节点才会被粘合。
2. 界面材料的杨氏模量。
3. 界面材料的泊松比。
4. 界面材料法向的初始屈服应力。
5. 界面材料法向的硬化模量。
6. 应变计算中使用的界面厚度。
7. 用于接触热通量计算的界面材料的热导率（单位为 [![](../graphics/ver_eqn01453.gif)](../popups/usb-int-iconventions-unitsym.md)）。

此示例中使用了三个用户定义的状态变量。第一个简单地指示初始化以确定哪些节点被粘合已完成。第二个用于标记哪些节点被粘合。第三个跟踪每个从节点处的当前屈服应力。

使用两种简单配置来在二维和三维中测试此用户子程序。在第一种配置中，两个相同的弹性体中的每一个都用一排四个单元进行建模：在纯机械分析中使用 CPS4R 或 C3D8R 单元；在热机械分析中使用 CPS4RT 或 C3D8RT 单元。第二种配置与第一种配置相同，但用固定的解析刚性表面替换了一排单元。物体最初是平行的，并由界面的厚度分开（即，考虑厚度后为零间隙）。一半节点位于接触界面上并被粘合。

在两个体都使用单元建模的纯机械分析中，边界条件施加到一个物体上的非粘合节点。向另一个物体施加三种独立的载荷条件，以在界面中产生以下应力状态：无非应的均匀法向应力、均匀剪切应力，以及在一个界面端引起显著屈服的纯属非均匀法向应力。

在两个体都使用单元建模的热机械分析中，两个物体上的非粘合节点都保持固定。给一个物体初始温度 100 度；给另一个物体初始温度 0 度。温度差异导致热量在物体之间流动，在两个物体中产生 50 度的均匀温度。

在包含解析刚性表面的热机械分析中，边界条件施加到可变形体的非粘合节点上以产生纯属非均匀法向应力。刚性体的参考节点保持固定。给刚性体初始温度 100 度；给可变形体初始温度 0 度。刚性体的热容被定义为与可变形体的热容相匹配，以便物体之间的温度差异将在分析结束时在两个物体中产生 50 度的均匀温度。

### 结果与讨论

将位移结果与使用 Abaqus 中可用的线性软化行为模型获得的解进行比较。将节点温度结果与使用界面热导率获得的解进行比较。所有情况的结果一致。
