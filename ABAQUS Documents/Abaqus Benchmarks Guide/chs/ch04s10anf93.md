# 4.10.10 3DNLG-10：压缩端部载荷作用下加筋圆柱面板的弹塑性行为





**产品：** Abaqus/Standard   

### 测试单元

S3    S3R    S4    S4R    S4R5    S8R    S8R5    S9R5    

STRI3    STRI65    

### 问题描述

![](../graphics/bmk3dnlg10-nls.png)

**模型：**

圆柱面板和筋都存在初始缺陷。

圆柱面板：![](../graphics/bmk_eqn01879.gif)，其中9×π/8 ≤ ![](../graphics/bmk_eqn01880.gif) ≤ 9×π/8且0 ≤ ![](../graphics/bmk_eqn01881.gif) ≤ 200。

筋：![](../graphics/bmk_eqn01882.gif)，其中0 ≤ ![](../graphics/bmk_eqn01883.gif) ≤ 200且0 ≤ ![](../graphics/bmk_eqn01884.gif) ≤ 10。

**材料：**

弹性模量 = 2.1×10⁵，泊松比 = 0.3，屈服应力 = 350。

**边界条件：**

在边缘1和2处当![](../graphics/bmk_eqn00191.gif) = π/9时切向对称（![](../graphics/bmk_eqn01885.gif)）。在A端对面板简支（![](../graphics/bmk_eqn01886.gif)），对筋简支（![](../graphics/bmk_eqn01887.gif)）。在![](../graphics/bmk_eqn01888.gif)处对称，对面板（![](../graphics/bmk_eqn01889.gif)），对筋（![](../graphics/bmk_eqn01846.gif)）。

**载荷：**

压缩均匀分布的边缘载荷。使用RIKS算法增加载荷直到D点的全局z方向位移超过0.2。

### 参考解

这是英国国家有限元方法与标准机构（NAFEMS）推荐的测试：NAFEMS出版物R0024"A Review of Benchmark Problems for Geometric Non-linear Behaviour of 3D Beams and Shells (SUMMARY)"中的测试3DNLG-10。

此问题的已发布结果由Abaqus获得。因此，Abaqus与NAFEMS结果的比较不是对Abaqus的独立验证。NAFEMS研究包括来自其他来源的比较结果，这些结果可能为此问题提供验证依据。

### 结果与讨论

在下表中，极限点1和2分别对应载荷-位移曲线的峰值和局部最小值。为了产生与S8R5/S9R5相当的结果，一阶单元需要在面板曲率方向上使用一半节点间距的更细网格。厚壳单元S8R在此薄壳应用中的极限载荷明显高于其他单元，即使使用与一阶单元相同的节点间距生成的细网格。
