# 44.5.1 选择线型、带型、拼布型或等值面型云图





您可以选择**Line**-、**Banded**-或**Isosurface**型云图来显示节点（如位移）或单元（如应力）场输出变量的值。线型云图在模型表面将值表示为自定义彩色线。带型云图将值表示为彩色填充带。等值面云图将线型云图延伸到模型中。线型、带型和等值面型云图基于单元变量值通过将结果外推到节点并有条件地[平均](pt05ch42s04s02.md)来计算。平均取决于模型的特征和您选择的选项。不建议对线形单元（如梁）使用线型云图。

您只能为单元场输出变量选择**Quilt**型云图。变量值被外推到模型表面的单元面，单元之间不进行平均。拼布云图对于按单元评估结果很有用。对于具有非对称变形的轴对称单元（CAXA或SAXA），每个单元只绘制一种颜色。

带型云图是默认设置。您必须选择线型、带型或等值面型云图来生成以下内容的图：
- 基于节点的场输出值。
- 跨单元平均的基于单元的场输出值。
- 基于单元的结果间断而不是场输出值。

**要选择线型、带型、拼布型或等值面型云图：**

1. 找到**Contour Type**选项。从主菜单栏中选择****Options**![](../graphics/images/arrow.gif)**Contour****，或单击工具箱中的![](../graphics/ico_plotContourOptions.png)；然后在出现的对话框中单击**Basic**选项卡。**Contour Type**选项位于页面左上角。
2. 单击**Line**、**Banded**、**Quilt**或**Isosurface**以选择您想要的云图类型。
3. 单击**Apply**实施您的更改。当前视口中的云图更改为显示您指定的云图类型。默认情况下，您的更改在会话期间保存，并将影响所有后续云图。如果您想在后续会话中保留更改，请将它们保存到文件。有关更多信息，请参见["保存自定义设置以供后续会话使用，" 第55.1.1节](pt05ch55s01s01.md)。
