# 3.3.2 使用包含隐藏图标的工具箱和工具栏





在某些工具箱（如作业模块工具箱）中，所有工具图标都立即可见；但是，大多数工具箱包含隐藏的图标以节省空间。由于画布上方有更多空间，并且您可以移动或隐藏工具栏以满足您的需求，因此大多数工具栏不包含隐藏的图标。

任何在其右下角包含小三角形的图标都隐藏着一组与可见图标功能密切相关的图标。

**选择最初隐藏的工具的图标：**

1. 单击并按住其右下角有三角形的任何图标。所有与原始图标密切相关的工具图标都会出现。例如，[图3-12](pt01ch03s03s02.md#uss-int-prt-flyout)显示了零件模块工具箱的上部，其中显示了用于创建圆角或倒角的所有图标。
   **图3-12** 显示圆角和倒角图标的零件模块工具箱。
   ![](../graphics/gst-prt-flyout-nls.png)
2. 将光标拖动到所需的图标，然后释放鼠标按钮。选定的图标会替换最初可见的图标，您可以立即开始使用相应的工具。
