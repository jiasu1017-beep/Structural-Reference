# 47.5.1 X–Y图轴选项概述





### 47.5.1 X–Y图轴选项概述

您可以使用轴选项来自定义*X–Y*图中轴的外观。[图47-38](pt05ch47s04hlb01.md#viw-xynavigate)标识了您可以自定义的*X–Y*图特征。

**图47–38** X–Y图的特征。

![](../graphics/viw-xynavigate-nls.png)

Abaqus/CAE从**Axis Options**对话框提供对这些选项的访问，您可以从Visualization模块中的以下位置打开此对话框：
- 从主菜单栏，选择****Options**![](../graphics/images/arrow.gif)**XY Options**![](../graphics/images/arrow.gif)**Axis****。
- 从Visualization模块工具箱中，点击![](../graphics/ico_xyPlotOptions.png)。
- 从Results Tree中，展开**X Axes**或**Y Axes**容器，高亮显示要修改的轴或多个轴，点击鼠标按钮3，并从出现的列表中选择**Axis Options**。
- 当视口中显示*X–Y*图时，双击图中的轴之一。*X–Y*图中的项目仅在当前没有活动过程时方可选择；您可能需要取消当前过程以激活*X–Y*图中的组件。

您可以点击以下选项卡来自定义所选轴的外观：
- **Scale**：选择轴刻度和限制并自定义刻度线的出现。
- **Tick Marks**：控制刻度线的外观和放置。
- **Title**：指定轴标题的内容和外观。
- **Axes**：选择轴的放置、颜色、样式和厚度，并控制数字轴标签。
