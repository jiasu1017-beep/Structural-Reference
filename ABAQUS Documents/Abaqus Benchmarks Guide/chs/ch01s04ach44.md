# 1.4.8 简支梁响应谱分析

**产品：** Abaqus/Standard

本问题通过将 Abaqus 结果与简单情况的精确解进行比较来验证 Abaqus 响应谱分析能力。

### 问题描述

该问题是 Biggs（1964）分析的简支梁，如图 1.4.8-1（[图 1.4.8-1](ch01s04ach44.md#sxmrespspect-testmodel)）所示。梁具有矩形横截面，宽度 37 mm（1.458 in），高度 355.6 mm（14 in）。梁的质量密度为 1.0473×10⁻⁵ kg/m³（0.0098 lb·s²/in⁴）。

有限元模型也如图 1.4.8-1（[图 1.4.8-1](ch01s04ach44.md#sxmrespspect-testmodel)）所示。响应谱在两个支撑处沿垂直方向施加，响应基于模型的第一振型确定。使用 B21 和 B23 单元类型进行分析，响应谱在下节定义。为问题指定零阻尼。梁截面定义为梁截面和通用梁截面，以测试两种规格。

### 响应谱定义

响应谱定义为单自由度弹簧-质量系统在施加到其基座的给定加速度历史激励下的峰值响应。Biggs（1964）将问题定义为两个支撑都根据加速度历史垂直移动，该加速度在 0.1 秒的时间段内从 +*g* 到 -*g*（其中 *g* 是重力加速度）线性斜升，之后为零。有了这个基座加速度历史，单自由度弹簧-质量系统中质量的加速度为

其中 ![](../graphics/bmk_eqn00283.gif) 是固有频率，![](../graphics/bmk_eqn00510.gif) 是加速度从 +*g* 到 -*g* 斜升的时间。

求解这两个方程以获得作为频率函数的最大加速度定义了响应谱。针对频率 5.、6.、6.098、7. 和 8. Hz 进行了计算。下表显示了 resulting 的响应谱：

| --- | --- |
| 5. | 2.0000 |
| 6. | 1.6667 |
| 6.098 | 1.6399 |
| 7. | 1.4286 |
| 8. | 1.4530 |

### 结果与讨论

Biggs（1964）计算第一模式的精确固有频率为 6.1 Hz，模态参与因子为 1.27324。Abaqus 对于使用 B23 单元类型的 10 单元模型给出第一模式频率为 6.098 Hz，对于使用 B21 单元类型的模型给出 6.0808 Hz。相应的模态参与因子为 1.2733 和 1.2628。两个 Abaqus 结果都非常接近 Biggs 的值，三次梁（B23）结果给出更好的一致性——可能是因为线性梁 B21 允许横向剪切变形，这增加了模型的柔性，因此降低了刚度。

Biggs 还使用 SRSS 求和给出了梁跨中最大位移、弯矩、曲率和弯曲应力的值。这些值用于表 1.4.8-2（[表 1.4.8-2](ch01s04ach44.md#table-respspect-results)）来检查 Abaqus 的计算（Abaqus 运行中报告的应力、弯矩和曲率值是通过将积分点值外推到跨中节点获得的）。对于所有四种测试情况，Abaqus 结果比较良好。
