# 32.2 为垫片定义材料





您可以创建两种类型的材料以包含在垫片截面定义中：具有垫片特定行为的材料和通用材料。您创建的材质类型取决于您对垫片行为的要求。
- 如果希望厚度方向、横剪和膜行为解耦，请使用特殊的垫片行为创建材料。垫片行为材料仅对垫片截面有效。有关定义垫片行为的这种方法详细信息，请参阅《Abaqus分析用户指南》第32.6.6节["直接使用垫片行为模型定义垫片行为"](../usb/usb-link.md#usb-elm-egasketbehavior)。
- 如果仅需考虑厚度方向行为，请创建通用材料。通用材料在垫片截面和其他类型的截面中都有效。有关定义垫片行为的这种方法详细信息，请参阅《Abaqus分析用户指南》第32.6.5节["使用材料模型定义垫片行为"](../usb/usb-link.md#usb-elm-egasketmatbehavior)。

您可以通过为****其他**![](../graphics/images/arrow.gif)**垫片****子菜单中的某种行为输入数据来创建垫片特定材料。在材料编辑器中为任何其他行为输入的数据将被忽略，但以下情况除外：
- 您可以在垫片行为材料定义中包含**膨胀**行为（位于**机械**菜单中）。
- 您可以在垫片行为材料定义中包含**蠕变**行为（位于****机械**![](../graphics/images/arrow.gif)**塑性****菜单中）。
- 您可以在垫片行为材料定义中包含**Depvar**和**用户输出变量**行为（位于**常规**菜单中）。

您可以通过为垫片截面有效但不属于****其他**![](../graphics/images/arrow.gif)**垫片****子菜单的行为输入数据来创建通用材料。（如果您为****其他**![](../graphics/images/arrow.gif)**垫片****子菜单中的行为输入数据，系统会自动创建垫片行为材料。）有关哪些材料行为对包含在垫片截面定义中的通用材料有效，请参阅《Abaqus分析用户指南》第32.6节["垫片单元"](../usb/usb-link.md#usbgasket)。
