# 20.12.4 添加单个尺寸







尺寸是一种特殊形式的约束；约束在 Sketcher 实体之间创建逻辑关系，而尺寸创建数值关系。尺寸为您的草图增加清晰度，并允许您精确地定位对象。使用 Sketcher 工具箱中的尺寸工具向草图添加单个尺寸。您也可以使用**自动尺寸**工具自动标注草图尺寸。有关更多信息，请参见["自动标注草图尺寸，"第 20.12.2 节](pt03ch20s12hlb02.md)。

尺寸工具是交互式的；Abaqus/CAE 使用预选和您选择的 Sketcher 实体来显示将要创建的尺寸的预览。如果显示的尺寸类型不是您要创建的类型，请将光标移动到其他位置。一旦您接受了尺寸类型和位置，您可以输入新的尺寸值。

**注意：**尺寸工具的交互功能仅在预选处于活动状态时可用。预选在**Sketcher 选项**对话框中默认处于打开状态。（有关更多信息，请参见["Sketcher 光标和预选，"第 20.4.5 节](pt03ch20s04s05.md)。）

**添加尺寸：**

1. 从 Sketcher 工具箱中的工具中，选择尺寸工具![](../graphics/ico_sketchDimOblique.png)。有关 Sketcher 工具箱中工具的图表，请参见["Sketcher 工具，"第 20.4.1 节](pt03ch20s04s01.md)。Abaqus/CAE 在提示区域显示提示以指导您完成该过程。
2. 选择第一个要标注尺寸的实体。Abaqus/CAE 根据您的选择提示您：**顶点** Abaqus/CAE 提示您为尺寸选择另一个顶点或直线。选择一条直线来标注该顶点与直线之间的距离。选择一个顶点来标注所选点之间的水平、垂直或斜向距离——Abaqus/CAE 创建的尺寸类型基于第二个顶点的位置和光标位置。Abaqus/CAE 根据光标位置创建将要创建的尺寸及其位置的预览。两个顶点之间的水平或垂直尺寸值前面分别标有 H 或 V。**直线** Abaqus/CAE 提示您为默认长度尺寸选择位置。要创建不同类型的尺寸，请将光标移到草图中的其他实体上；Abaqus/CAE 为所选直线与预选实体之间的关系创建适当尺寸类型的预览。点击所需的实体以接受显示的尺寸类型。**圆或圆弧** Abaqus/CAE 提示您为半径尺寸选择位置。没有其他实体选择可用。
3. 将光标移动到您希望尺寸出现的位置；当您对尺寸的外观满意时点击。
4. 如有需要在提示区域编辑尺寸值；按**Enter**或点击鼠标按钮 2接受尺寸值。Abaqus/CAE 在草图上创建尺寸。如有必要，顶点会被移动以符合编辑的尺寸值和草图中的任何其他约束。
5. 要创建其他尺寸，重复从第 2 步开始的上述步骤。
6. 完成创建尺寸后，执行以下操作之一：- 在 Abaqus/CAE 窗口中的任意位置点击鼠标按钮 2。- 在 Sketcher 工具箱中选择任何工具。- 点击提示区域中的取消按钮![](../graphics/afxI_cancel.png)。
