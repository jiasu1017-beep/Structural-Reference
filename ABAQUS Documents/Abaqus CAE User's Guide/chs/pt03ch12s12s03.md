### 定义膨胀凝胶

您可以对在部分饱和多孔介质中膨胀并捕获润湿液体的凝胶颗粒生长进行建模。有关更多信息，请参见[Abaqus Analysis User's Guide的第26.6.5节，"膨胀凝胶"](../usb/usb-link.md#usb-mat-cgelswelling)。

**定义膨胀凝胶：**

1. 从**Edit Material**对话框的菜单栏中，选择****Other**![](../graphics/images/arrow.gif)**Pore Fluid**![](../graphics/images/arrow.gif)**Gel****。（有关显示**Edit Material**对话框的信息，请参见["创建或编辑材料，"第12.7.1节"](pt03ch12s07hlb01.md)。）
2. 在**Data**表格中输入以下数据：
   **r_a(dry)** 完全干燥时的凝胶颗粒半径，![](../graphics/usi_eqn00342.gif)。（[L](../popups/usb-int-iconventions-unitsym.md)单位。）
   **r_a(f)** 凝胶颗粒完全膨胀时的半径，![](../graphics/usi_eqn00343.gif)。（L单位。）
   **k_a** 单位体积内的凝胶颗粒数，![](../graphics/usi_eqn00344.gif)。（[L3](../popups/usb-int-iconventions-unitsym.md)单位。）
   **tau_1** 凝胶颗粒长期膨胀的松弛时间常数，![](../graphics/usi_eqn00345.gif)。（[T](../popups/usb-int-iconventions-unitsym.md)单位。）
3. 点击**OK**关闭**Edit Material**对话框。或者，您可以从**Edit Material**对话框的菜单中选择另一种材料行为进行定义（有关更多信息，请参见["浏览和修改材料行为，"第12.7.2节"](pt03ch12s07hlb02.md)）。

### 定义湿气膨胀

**Edit material**对话框允许您在部分饱和流动条件下定义多孔介质固体骨架的饱和驱动体积膨胀。您可以在耦合润湿液体流动和多孔介质应力的分析中使用这种材料定义。请参阅以下章节获取更多信息：
- ["湿气膨胀，"第26.6.6节Abaqus Analysis User's Guide](../usb/usb-link.md#usb-mat-cmoistureswell)
- ["耦合孔隙流体扩散和应力分析，"第6.8.1节Abaqus Analysis User's Guide](../usb/usb-link.md#usb-anl-acoupdiffstress)

**定义湿气膨胀：**

1. 从**Edit Material**对话框的菜单栏中，选择****Other**![](../graphics/images/arrow.gif)**Pore Fluid**![](../graphics/images/arrow.gif)**Moisture Swelling****。（有关显示**Edit Material**对话框的信息，请参见["创建或编辑材料，"第12.7.1节"](pt03ch12s07hlb01.md)。）
2. 在**Data**表格中输入以下数据：
   **Strain** 体积湿气膨胀应变，![](../graphics/usi_eqn00346.gif)。
   **Saturation** 饱和度，*s*。此值必须在范围内![](../graphics/usi_eqn00347.gif)。
   有关如何输入数据的详细信息，请参见["输入表格数据，"第3.2.7节"](pt01ch03s02s07.md)。
3. 如果要定义各向异性膨胀，请从**Suboptions**菜单中选择**Ratios**。有关详细说明，请参见["定义各向异性膨胀"](pt03ch12s12s03.md#usi-prp-other-porefluid-anisotropicswelling)。
4. 点击**OK**关闭**Edit Material**对话框。或者，您可以从**Edit Material**对话框的菜单中选择另一种材料行为进行定义（有关更多信息，请参见["浏览和修改材料行为，"第12.7.2节"](pt03ch12s07hlb02.md)）。

### 定义各向异性膨胀

您可以通过定义比值来包含湿气膨胀行为中的各向异性，![](../graphics/usi_eqn00348.gif)、![](../graphics/usi_eqn00349.gif)和![](../graphics/usi_eqn00350.gif)，使得三个比值中有两个或更多不同。湿气膨胀应变方向的方向取决于用户指定的局部取向（请参见Abaqus Analysis User's Guide的第2.2.5节"取向"）。

**定义各向异性膨胀：**

1. 如["定义湿气膨胀"](pt03ch12s12s03.md#usi-prp-other-porefluid-moistureswelling)中所述定义湿气膨胀行为。
2. 从**Edit Material**对话框的**Suboptions**菜单中，选择**Ratios**。出现**Suboption Editor**。
3. 切换**Use temperature-dependent data**将各向异性比值定义为温度的函数。**Data**表格中出现一个标为**Temp**的列。
4. 点击**Number of field variables**字段右侧的箭头，以增加或减少各向异性比值定义中包含的场变量数量。
5. 在**Data**表格中，输入各向异性比值![](../graphics/usi_eqn00348.gif)、![](../graphics/usi_eqn00349.gif)和![](../graphics/usi_eqn00350.gif)。如适用，输入温度和场变量的值。有关如何输入数据的详细信息，请参见["输入表格数据，"第3.2.7节"](pt01ch03s02s07.md)。
6. 点击**OK**返回**Edit Material**对话框。

#### 在Abaqus/Standard分析中定义渗透率

对于有效应力/润湿液体扩散分析，您必须在Abaqus/Standard中为润湿液体指定渗透率。请参见["渗透率，"第26.6.2节Abaqus Analysis User's Guide](../usb/usb-link.md#usb-mat-cpermeabil)。

**在Abaqus/Standard分析中定义渗透率：**

1. 从**Edit Material**对话框的菜单栏中，选择****Other**![](../graphics/images/arrow.gif)**Pore Fluid**![](../graphics/images/arrow.gif)**Permeability****。（有关显示**Edit Material**对话框的信息，请参见["创建或编辑材料，"第12.7.1节"](pt03ch12s07hlb01.md)。）
2. 点击**Type**字段右侧的箭头，并指定渗透率的方向相关性。
3. 为**Specific weight of wetting liquid**输入一个值。
4. 切换**Use temperature-dependent data**将渗透率定义为温度的函数。**Data**表格中出现一个标为**Temp**的列。
5. 在**Data**表格中输入适用的数据：
   **k** 完全饱和介质的各向同性渗透率，*k*。（[LT1](../popups/usb-int-iconventions-unitsym.md)单位。）
   **k11, k22, 和 k33** 正交各向异性渗透率的三个值，![](../graphics/usi_eqn00311.gif)、![](../graphics/usi_eqn00312.gif)和![](../graphics/usi_eqn00313.gif)。（[LT1](../popups/usb-int-iconventions-unitsym.md)单位。）
   **k11, k12, k22, k13, k23, 和 k33** 各向异性渗透率的六个值，![](../graphics/usi_eqn00311.gif)、![](../graphics/usi_eqn00314.gif)、![](../graphics/usi_eqn00312.gif)、![](../graphics/usi_eqn00315.gif)、![](../graphics/usi_eqn00316.gif)和![](../graphics/usi_eqn00313.gif)。（[LT1](../popups/usb-int-iconventions-unitsym.md)单位。）
   **Void Ratio** 孔隙比，*e*。
   **Temp** 温度。
   有关如何输入数据的详细信息，请参见["输入表格数据，"第3.2.7节"](pt01ch03s02s07.md)。
6. 如果要定义渗透率对饱和度的依赖性，请从**Suboptions**菜单中选择**Saturation Dependence**。有关详细说明，请参见["定义渗透率的饱和度相关性"](pt03ch12s12s03.md#usi-prp-other-porefluid-saturationdependence)。
7. 如果要定义渗透率对速度的依赖性，请从**Suboptions**菜单中选择**Velocity Dependence**。有关详细说明，请参见["定义渗透率的速度相关性"](pt03ch12s12s03.md#usi-prp-other-porefluid-velocitydependence)。
8. 点击**OK**关闭**Edit Material**对话框。或者，您可以从**Edit Material**对话框的菜单中选择另一种材料行为进行定义（有关更多信息，请参见["浏览和修改材料行为，"第12.7.2节"](pt03ch12s07hlb02.md)）。

#### 定义渗透率的速度相关性

渗透率通常由Forchheimer定律定义，该定律考虑了渗透率作为流体流速函数的变化。**Suboption Editor**允许您将速度系数定义为材料孔隙比的函数。请参见["渗透率，"第26.6.2节Abaqus Analysis User's Guide](../usb/usb-link.md#usb-mat-cpermeabil)。

**定义饱和度相关性：**

1. 如["在Abaqus/Standard分析中定义渗透率"](pt03ch12s12s03.md#usi-prp-other-porefluid-permeability)中所述定义渗透率。
2. 从**Edit Material**对话框的**Suboptions**菜单中，选择**Velocity Dependence**。出现**Suboption Editor**。
3. 在**Data**表格中输入以下数据：
   **Beta** 速度系数，![](../graphics/usi_eqn00360.gif)。
   **Void ratio** 孔隙比，*e*。
   有关如何输入数据的详细信息，请参见["输入表格数据，"第3.2.7节"](pt01ch03s02s07.md)。
4. 点击**OK**返回**Edit Material**对话框。

#### 在Abaqus/CFD分析中定义各向同性渗透率

对于多孔介质流动，必须在Abaqus/CFD中指定渗透率，并且可以与孔隙率有依赖关系的各向同性渗透率。请参见["渗透率，"第26.6.2节Abaqus Analysis User's Guide](../usb/usb-link.md#usb-mat-cpermeabil)。

**在Abaqus/CFD分析中定义渗透率：**

1. 从**Edit Material**对话框的菜单栏中，选择****Other**![](../graphics/images/arrow.gif)**Pore Fluid**![](../graphics/images/arrow.gif)**Permeability****。（有关显示**Edit Material**对话框的信息，请参见["创建或编辑材料，"第12.7.1节"](pt03ch12s07hlb01.md)。）
2. 点击**Type**字段右侧的箭头，并选择**Isotropic (CFD)**。
3. 为**Inertial drag coefficient**指定一个值。将此参数设置为与多孔介质中惯性的（二次或形式）阻力成正比。默认值为0.142887。
4. 在**Data**表格中输入适用的数据：
   **K** 完全饱和介质的各向同性渗透率，*K*。（[L2](../popups/usb-int-iconventions-unitsym.md)单位。）
   **Porosity** 孔隙率，![](../graphics/usi_eqn00171.gif)。
   有关如何输入数据的详细信息，请参见["输入表格数据，"第3.2.7节"](pt01ch03s02s07.md)。
5. 点击**OK**关闭**Edit Material**对话框。或者，您可以从**Edit Material**对话框的菜单中选择另一种材料行为进行定义（有关更多信息，请参见["浏览和修改材料行为，"第12.7.2节"](pt03ch12s07hlb02.md)）。

### 定义多孔体积模量

您可以定义固体颗粒和渗透流体的体积模量，以便在多孔介质分析中考虑它们的可压缩性。请参见["多孔体积模量，"第26.6.3节Abaqus Analysis User's Guide](../usb/usb-link.md#usb-mat-cporousbulkmod)。

**定义多孔体积模量：**

1. 从**Edit Material**对话框的菜单栏中，选择****Other**![](../graphics/images/arrow.gif)**Pore Fluid**![](../graphics/images/arrow.gif)**Porous Bulk Moduli****。（有关显示**Edit Material**对话框的信息，请参见["创建或编辑材料，"第12.7.1节"](pt03ch12s07hlb01.md)。）
2. 切换**Use temperature-dependent data**将体积模量定义为温度的函数。**Data**表格中出现一个标为**Temp**的列。
3. 在**Data**表格中输入以下数据：
   **Bulk mod of grains** 固体颗粒的体积模量。（[FL2](../popups/usb-int-iconventions-unitsym.md)单位。）
   **Bulk mod of fluids** 渗透流体的体积模量。（[FL2](../popups/usb-int-iconventions-unitsym.md)单位。）

### 定义间隙表面上的法向流动

您可以通过为孔隙流体材料定义流体泄漏系数来对间隙表面上的法向流动进行建模。此系数定义了内聚元素中间节点与其相邻表面节点之间的压力-流动关系。流体泄漏系数可以解释为内聚元素表面上有限材料层的渗透率。请参阅以下章节获取更多信息：
- ["定义内聚元素间隙内流体的本构响应，"第32.5.7节Abaqus Analysis User's Guide](../usb/usb-link.md#usb-elm-ecohesivefluidbehavior)
- ["UFLUIDLEAKOFF，"第1.1.34节Abaqus User Subroutines Reference Guide](../sub/sub-link.md#sub-rtn-uufluidleakoff)

**定义流体泄漏系数**

1. 从**Edit Material**对话框的菜单栏中，选择****Other**![](../graphics/images/arrow.gif)**Pore Fluid**![](../graphics/images/arrow.gif)**Fluid Leakoff****。（有关显示**Edit Material**对话框的信息，请参见["创建或编辑材料，"第12.7.1节"](pt03ch12s07hlb01.md)。）
2. 点击**Type**字段右侧的箭头，并指定您希望如何定义流体泄漏系数：
   - 选择**Coefficients**以直接在**Edit Material**对话框中输入系数。
   - 选择**User**以在用户子程序[`UFLUIDLEAKOFF`](../sub/sub-link.md#sub-xsl-ufluidleakoff)中定义系数。如果选择此选项，请跳至第6步。
3. 切换**Use temperature-dependent data**将流体泄漏系数定义为温度的函数。**Data**表格中出现一个标为**Temp**的列。
4. 点击**Number of field variables**字段右侧的箭头，以增加或减少流体泄漏系数所依赖的场变量数量。
5. 在**Data**表格中输入以下数据：
   **Top Coefficient** 顶部元素表面上的流体泄漏系数。
   **Bottom Coefficient** 底部元素表面上的流体泄漏系数。
   **Temp** 温度。
   **Field *n*** 预定义的场变量。
   有关如何输入数据的详细信息，请参见["输入表格数据，"第3.2.7节"](pt01ch03s02s07.md)。
6. 点击**OK**关闭**Edit Material**对话框。或者，您可以从**Edit Material**对话框的菜单中选择另一种材料行为进行定义（有关更多信息，请参见["浏览和修改材料行为，"第12.7.2节"](pt03ch12s07hlb02.md)）。

您可以使用**Edit Material**对话框为孔隙压力内聚元素定义切向流动本构参数。请参见["定义内聚元素间隙内流体的本构响应，"第32.5.7节Abaqus Analysis User's Guide](../usb/usb-link.md#usb-elm-ecohesivefluidbehavior)。

**定义切向流动参数：**

1. 从**Edit Material**对话框的菜单栏中，选择****Other**![](../graphics/images/arrow.gif)**Pore Fluid**![](../graphics/images/arrow.gif)**Gap Flow****。（有关显示**Edit Material**对话框的信息，请参见["创建或编辑材料，"第12.7.1节"](pt03ch12s07hlb01.md)。）
2. 点击**Type**字段右侧的箭头，并指定您希望如何定义流动参数：

6. 如果在第2步中选择了**Newtonian**，请在**Data**表格中输入以下数据：

   **Field *n*** 预定义的场变量。
   有关如何输入数据的详细信息，请参见["输入表格数据，"第3.2.7节"](pt01ch03s02s07.md)。
7. 如果在第2步中选择了**Power law**，请在**Data**表格中输入以下数据：
   **Consistency** 流体稠度，*K*。
   **Exponent** 幂律指数，![](../graphics/usi_eqn00007.gif)。
   **Temp** 温度。
   **Field *n*** 预定义的场变量。
   有关如何输入数据的详细信息，请参见["输入表格数据，"第3.2.7节"](pt01ch03s02s07.md)。
8. 点击**OK**关闭**Edit Material**对话框。或者，您可以从**Edit Material**对话框的菜单中选择另一种材料行为进行定义（有关更多信息，请参见["浏览和修改材料行为，"第12.7.2节"](pt03ch12s07hlb02.md)）。
