# 11.6 质量缩放

**产品：** Abaqus/Explicit

##### **参考**

- [*Mass scaling](../key/key-link.md#usb-kws-mmmscaling)

### 概述

质量缩放是一种用于显式动力学分析的技术，通过调整模型的质量分布来增大临界时间步长，从而提高计算效率。质量缩放主要用于处理模型中存在的细小单元或高频响应区域。

### 质量缩放原理

显式动力学分析的稳定性条件由最小单元尺寸决定：

```
dt < 2 * L_e / c
```

其中：
- dt：时间步长
- L_e：单元特征长度
- c：材料中的声速

质量缩放通过增加单元的"有效质量"来降低声速，从而允许使用更大的时间步长。

### 质量缩放类型

#### 全局质量缩放

对整个模型的所有单元应用相同的质量缩放因子。全局缩放适用于均匀网格或问题不关心局部响应的情况。

#### 选择性质量缩放

仅对特定单元或单元组应用质量缩放。选择性缩放可以避免对关键区域（如裂纹尖端、接触区域）引入人为惯性效应。

#### 自适应质量缩放

在分析过程中自动调整质量缩放因子，以保持目标时间步长。自适应缩放可以在整个分析中维持稳定的计算效率。

### 何时使用质量缩放

质量缩放适用于以下情况：

- 模型包含细小单元导致极小时间步长
- 关心的响应频率低于最高固有频率
- 静力或准静态问题使用显式求解器求解
- 需要缩短分析时间

### 何时避免使用质量缩放

质量缩放可能不适合以下情况：

- 高频动力学响应很重要
- 涉及波的传播问题
- 冲击和爆炸问题
- 质量惯性对结果有显著影响

### 定义质量缩放

#### 全局质量缩放

使用**Mass scaling**关键字定义全局质量缩放：

```
[*Mass scaling](../key/key-link.md#usb-kws-mmmscaling), DEFINITION=global
*缩放因子
```

#### 选择性质量缩放

```
[*Mass scaling](../key/key-link.md#usb-kws-mmmscaling), DEFINITION=element set, ELSET=*单元集名称*
*缩放因子
```

#### 自适应质量缩放

```
[*Mass scaling](../key/key-link.md#usb-kws-mmmscaling), DEFINITION=adaptive, FREQUENCY=*调整频率*
*目标时间步长
```

### 质量缩放因子

质量缩放因子定义了单元质量的放大倍数：

- **质量缩放因子 = 2**：单元质量增加为原来的2倍
- **质量缩放因子 = 4**：单元质量增加为原来的4倍

缩放因子的选择取决于：
- 目标时间步长
- 可接受的质量惯性误差
- 材料属性

### 能量平衡

应用质量缩放时，应检查能量平衡：

#### 伪应变能

质量缩放引入的伪应变能应该远小于内能。如果伪应变能显著增长，表明质量缩放过大。

#### 动能

检查模型动能是否在合理范围内。异常高的动能可能表明质量缩放导致数值不稳定。

### 输出

质量缩放分析应监控以下输出：
- ALLAE：伪应变能
- ALLKE：动能
- ALLIE：内能
- 伪应变能与内能比值

### 质量缩放的最佳实践

1. **从最小缩放开始**：使用满足稳定性要求的最小质量缩放因子

2. **选择性缩放优于全局缩放**：仅对不影响关键响应的区域应用缩放

3. **验证结果**：通过比较缩放前后的结果验证质量缩放的影响

4. **监控能量**：确保伪应变能在可接受范围内

5. **避免过度缩放**：质量缩放因子不宜超过10-100倍

### 替代方法

如果质量缩放不适合，可以考虑：

- **网格重划分**：细化粗网格或粗化细网格
- **隐式-显式顺序分析**：静力部分使用隐式，动力部分使用显式
- **子cycling**：对不同区域使用不同时间步长
