# 33.2 Managing inertia







The **Inertia Manager** allows you to create and manage inertia definitions. The manager includes a list of the names and types of inertia that you have defined. The **Create**, **Edit**, **Copy**, **Rename**, and **Delete** buttons in the manager allow you to create new inertia definitions or to edit, copy, rename, and delete existing ones. The icons in the column along the left side of the manager allow you to suppress and resume existing inertia definitions. You can also initiate these procedures using the ****Special**![](../graphics/images/arrow.gif)**Inertia**** menu from the main menu bar in the Property module or the Interaction module. After you select a management operation from the main menu bar, the procedure is exactly the same as if you had clicked the corresponding button inside the manager dialog box.
