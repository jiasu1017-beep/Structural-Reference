# 48.1.1 路径规格





路径是通过在模型中指定一系列点或线段来定义的线。这些点可以是节点或坐标位置；线段可以是单元边缘或连接单元边缘端点的线。路径可以穿过多个部件实例。Abaqus/CAE提供四种不同类型的路径：

**节点列表**

构成路径的点仅由节点位置组成。您可以使用节点标签和节点标签范围来指定点。节点标签是引用给定节点的持久方式；也就是说，节点标签不会随模型变形而改变。正因为如此，构成节点列表路径的节点标签同样适用于未变形或变形的模型形状。但是，节点标签是部件实例特定的。换句话说，多个部件实例可以具有相同的节点标签；因此，在使用节点标签时必须指定您引用的是哪个部件实例。有关详细信息，请参阅["创建或编辑节点列表路径，"第48.2.1节](pt05ch48s02hlb01.md)。

**点列表**

构成路径的点由模型内的坐标位置组成。这些位置可能与节点位置重合，也可能不重合。点列表坐标在空间中保持固定，与模型无关。例如，与未变形形状上节点位置重合的坐标可能与变形形状上的任何位置都不重合。同样，点列表坐标与特定部件实例无关。有关详细信息，请参阅["创建或编辑点列表路径，"第48.2.2节](pt05ch48s02hlb02.md)。

**边缘列表**

构成路径的点由连接模型内节点的边缘组成。您可以通过从视口中选择单个单元边缘，或通过选择起始边缘和方向并允许Abaqus/CAE自动完成路径到特征边缘的末端或您选择的端点，来指定边缘。边缘由模型中的元素定义。正因为如此，构成边缘列表路径的边缘同样适用于未变形或变形的模型形状。但是，单元标签是部件实例特定的。换句话说，多个部件实例可以具有相同的单元标签；因此，在使用边缘时必须指定您引用的是哪个部件实例。有关详细信息，请参阅["创建或编辑边缘列表路径，"第48.2.3节](pt05ch48s02hlb03.md)。

**圆形**

构成路径的点由模型内的坐标位置组成。您可以在Abaqus/CAE中创建两种类型的圆形路径：周向和径向。周向路径沿着圆或圆弧的边缘；径向路径沿着半径。您可以通过选择定义圆或圆弧的坐标，然后定义路径上的起点、终点和点数，来指定任一类型的路径。定义原始圆或圆弧后，您可以选择新的半径长度来创建与原始圆共享相同中心点的任何路径。您可以使用周向路径来获取模型曲线部分中的截面结果，如图48-1所示（[图48-1](pt05ch48s01s01.md#usv-path-circpath)）。

您可以通过从视口中拾取节点或通过在提示区域中输入值来选择坐标；无论哪种情况，坐标都保持在空间中固定，与模型无关。例如，与未变形形状上节点位置重合的坐标可能与变形形状上的任何位置都不重合。同样，圆形路径坐标定义与特定部件实例无关。有关详细信息，请参阅["创建或编辑圆形路径，"第48.2.4节](pt05ch48s02hlb04.md)。

Abaqus/CAE通过按您指定的顺序连接您指定的节点、点或边缘来形成路径。

创建路径后，您可以选择****工具**![](../graphics/images/arrow.gif)**路径****从主菜单栏编辑、复制、重命名、删除或绘制它。绘制路径本身是一种可视化验证您已指定了预期线条的方法；要查看沿路径的结果，您必须形成X–Y数据对并生成X–Y曲线图。有关管理路径的详细信息，请参阅["使用管理器对话框管理对象，"第3.4.9节](pt01ch03s04hlb01.md)；有关生成路径数据的X–Y曲线图的详细信息，请参阅["沿路径获取X–Y数据，"第48.3节](pt05ch48s03.md)。
