# 47.9.3 向X–Y曲线图图例添加最小值和最大值








### 47.9.3 向X--Y曲线图图例添加最小值和最大值

您可以在曲线图图例上包含曲线图中最小值和最大值*X–Y*值的报告，并且可以自定义这些值的格式和精度。

**要在X–Y曲线图图例中显示最小值和最大值：**

1. 找到**Chart Legend Options**。选择****Options**![](../graphics/images/arrow.gif)**XY Options**![](../graphics/images/arrow.gif)**Chart Legend****；或点击![](../graphics/ico_xyChartLegendOptions.png)，该图标位于可视化模块工具箱的*X--Y*绘图工具中。**提示：**当显示*X--Y*曲线图时，您也可以通过在视口中双击图例直接打开此对话框。当活动过程生效时，您不能直接从视口中选择*X--Y*曲线图中的组件；在从*X--Y*曲线图中选择项目之前，请取消当前过程。
2. 从**Charts**字段中，选择要自定义其图例的一个或多个*X--Y*曲线图。
3. 点击**Contents**选项卡（如果尚未选择）。
4. 切换**Show min/max**以在图例中报告最小值和最大值*X--Y*值。
5. 选择以下数字格式之一用于图例中显示的最小值和最大值：**Automatic**非常大的值和非常小的值以科学记数法表示。所有剩余值以无指数形式表示，并使用指定的有效数字位数。**Decimal**所有值以无指数形式表示，并使用指定的有效数字位数。**Engineering**大于或等于1000（或小于或等于0.001）的值表示为1到999之间的数字乘以10的*n*次方，其中*n*是3的倍数（例如，`20.5E+03`或`17.76E+06`）。所有剩余值使用指定的有效数字位数表示。**Scientific**所有值表示为1到10之间的数字乘以10的适当幂（例如，`2.05E+04`或`1.776E+07`）。
6. 点击**Precision**箭头指定每个标签中的有效数字位数（非十进制格式）或小数位数（十进制格式）。
7. 点击**Dismiss**关闭**Chart Legend Options**对话框。
