# 4.1.7 传动轴机构

**产品：** Abaqus/Standard  Abaqus/Explicit   

本示例说明了使用连接器对传动轴机构进行建模。

### 几何与模型

传动轴机构是机动车辆的重要组成部分。这些机构通常包含万向节，用于将运动从一个轴传递到另一个轴。万向节不是等速万向节。如果输入和输出轴未对齐，输入轴的均匀旋转将导致输出轴的非均匀旋转。

本示例中的传动轴机构由三根轴组成：SHAFT AB（输入轴）、SHAFT CD（中心轴）和 SHAFT EF（输出轴）。SHAFT AB 通过万向节连接到 SHAFT CD。SHAFT CD 通过万向节连接到 SHAFT EF。使用 B31 梁单元对轴进行建模。

### 模型相互作用

除绕梁轴的旋转自由度外，A 和 F 处的所有自由度都被固定。在 A 处指定了 360 度的规定旋转。SHAFT AB 使用 UJOINT 连接器元件连接到 SHAFT CD。SHAFT CD 使用 UJOINT 连接器元件连接到 SHAFT EF。

### 结果与讨论

SHAFT AB 绕其轴的旋转导致 SHAFT EF 绕其轴旋转。这些轴的运动是完全同步的。

### 输入文件

- driveshaft_model.py - Python 回放文件
- driveshaft.inp - Abaqus/Standard 传动轴机构模型
- driveshaft_fric.inp - 带摩擦的 Abaqus/Standard 模型
- driveshaft_exp_fric.inp - 带摩擦的 Abaqus/Explicit 模型
