# 36.1 定义表皮加固





表皮加固定义粘合到现有部件表面的表皮并指定其工程属性。表面可以是三维实体部件的面、轴对称部件的任何边缘或二维部件的任何面。部件可以包括几何体和孤立网格单元；但是，必须分别为几何体和孤立网格单元定义表皮加固。您应该将表皮视为部件或区域的属性，就像截面是部件或区域的属性一样。

[图36-1](pt04ch36s01.md#prp-prop-skin)所示的复合梁是如何在模型中使用表皮加固的示例。

**图36-1** 由实体蜂窝芯和铝表皮建模的复合梁。

![](../graphics/prp-prop-skin-nls.png)

梁具有实体蜂窝芯和上下表面的铝表皮。您可以创建表示蜂窝的实体部件，并添加表示铝层的表皮加固。在Mesh模块中，您为蜂窝分配实体单元，为表皮分配壳单元。实体和壳单元共享相同的节点。

从Property模块的主菜单栏中选择****Special**![](../graphics/images/arrow.gif)**Skin**![](../graphics/images/arrow.gif)**Create****以定义一个或多个表皮。从同一菜单中选择**Edit**以对现有定义进行更改。您创建的所有表皮也出现在Model Tree中部件下的**Skins**容器中。默认情况下，表皮不在视口中显示，但您可以通过在视口中对它们进行颜色编码来使其可见。更多信息，请参见["对几何体和网格单元进行颜色编码"，第77.4节](pt07ch77s04.md)。

如果在几何体区域上创建表皮，Abaqus/CAE会在对底层几何体进行微小修改时更新表皮。如果使用表皮编辑孤立节点或单元，Abaqus/CAE会在您编辑或删除节点或单元时更新表皮；但是，如果在创建新节点或单元时不会更新表皮。

在后续建模操作中可能需要选择表皮；例如：
- 为表皮分配均匀壳截面、复合壳截面、膜截面、表面截面或垫片截面。截面分配在Property模块中进行。
- 为表皮分配材料方向或钢筋参考方向。这两种方向分配都在Property模块中进行。
- 在Property模块中为表皮分配法线方向。虽然不能直接为表皮分配法线方向，但可以为面分配法线方向，这会更新该面上定义的所有表皮的法线方向。
- 在Load模块中为表皮施加体力。
- 在Load模块中在表皮上施加热通量。实际上，您可以通过向底层面或面施加热通量载荷来执行此建模操作。Abaqus/CAE会在分析期间将该载荷应用于这些面上的所有表皮。
- 在Mesh模块中为表皮分配单元类型。
- 在Step模块中请求表皮的场数据输出或历史数据输出。
- 创建显示组以在Visualization模块中查看表皮单元上的应力值。虽然不能在**Create Display Group**对话框中按名称专门选择表皮，但可以通过搜索与模型中表皮单元具有相同单元类型、截面分配或其他属性的单元来在对话框中找到它们。此过程可以帮助您将单元列表缩小到所需的表皮。

当提示您为此类建模操作选择区域时，从**Selection**工具栏的对象类型列表中选择**Skins**，然后从视口中选择表皮区域。更多信息，请参见["根据对象类型过滤您的选择"，第6.3.2节](pt01ch06s03s01.md)。

执行接触计算时，Abaqus/CAE仅在某些情况下考虑几何表面的表皮加固；例如，当您为所有外表面指定一般接触时，会在Abaqus中自动定义一个将考虑表皮加固的默认全包容表面。表皮加固可能会显著影响接触计算，这由于表皮厚度以及对接触惩罚刚度等数值量的潜在影响。要在接触定义中明确包含表皮加固，您应该在表皮加固上创建一个集合，然后在接触定义中使用该集合。

有关创建表皮的详细信息，请参见["创建和编辑表皮加固"，第36.8节](pt04ch36s08.md)。
