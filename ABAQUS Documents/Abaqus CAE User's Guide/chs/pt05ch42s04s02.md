# 42.6.2 了解结果值平均

结果值平均适用于使用以下任何方法显示基于单元的场输出变量（如应力或应变）：
- 线条或带状等值线，
- 节点位置探测，
- 基于结果值形成显示组或颜色编码，
- 从场提取*X--Y*数据，
- 沿路径提取*X--Y*数据，或
- 生成场数据报告。

基于节点（如位移）的场输出变量、quilt等值线图和间断图的显示不涉及值平均。

共同属于两个或多个单元的节点处外推的输出数据库值进行有条件的平均。您可以使用以下选项来控制Abaqus/CAE平均节点值的程度：
- 选择[结果区域](pt05ch42s04hlb03.md)，并选择是否跨区域边界进行平均。
- 选择壳和膜特征边缘是否被视为平均的区域边界。
- 包含或排除来自未显示单元的贡献。
- 选择不变量是在平均之前还是之后计算和提取分量。
- 选择阈值值以根据贡献单元值之间的差异控制平均（仅当您选择在平均之前计算不变量时，此选项才可用）。

当贡献单元位于同一结果区域时，Abaqus/CAE在共同属于两个或多个单元的节点处平均值。默认结果区域与分配截面时为模型定义的区域相同；您也可以使用保存的单元集或显示组定义自定义结果区域。您可以选择Abaqus/CAE是否在共同属于两个或多个结果区域的节点处平均值。您可以抑制跨区域平均（使用区域边界）以强调区域边界的任何间断，或者您可以请求跨区域平均（忽略区域边界）以产生更连续的效果。例如，[图42-6](pt05ch42s04s02.md#viw-average)显示了在左侧使用区域边界和在右侧跨区域平均的等值线图。

![](../graphics/viw-average.png)

如果您选择跨区域平均，则值的平均程度由您指定的阈值控制，如下所示：

![](../graphics/usi_eqn00717.gif)

如果绘图中包含的每个节点的相对节点变化小于平均阈值，则在节点处平均贡献单元的值。如果相对节点变化超过您的设置，则不平均这些值。当您决定跨区域平均时，平均阈值与整个模型的值的变化相关，除非您将平均限制为显示的单元。设置较高的平均阈值允许您平滑除模型或所有显示结果中最极端的间断之外的所有间断。

如果您选择不跨区域平均，则每个区域内节点处值的平均程度由您指定的平均阈值控制，如下所示：

![](../graphics/usi_eqn00718.gif)

如果绘图中包含的每个节点的相对节点变化小于平均阈值，则在节点处平均贡献单元的值。如果相对节点变化超过您的设置，则不平均这些值。当您抑制跨区域平均时，平均阈值与每个区域内值的变化相关，而不是与整个模型或所有显示结果相关。

您应用的平均阈值越低，显示的值越能描述单个单元结果。相反，较高的平均阈值会产生更平滑的效果，单元间的间断不那么明显。

当将基于阈值的平均与从基于单元的场提取*X–Y*数据结合使用时，您可能需要调整阈值。提取*X–Y*数据时使用的单元区域缩小为仅包含连接到指定节点的单元。

如果您选择在平均之后计算不变量和提取分量，
- Abaqus/CAE不应用平均阈值；
- 张量在平均之前被变换到每个节点的平均方向；并且
- 结果区域（如果使用）必须包含兼容单元。

有关的详细信息，请参阅["了解结果如何计算，" 第42.6.1节](pt05ch42s04s01.md)。
