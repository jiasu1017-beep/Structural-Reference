# 12.15.6 分配梁/桁架切线方向







当您创建带有线框区域的零件时，Abaqus会为这些区域分配默认的切线方向。梁截面方向取决于切线方向。您可以反转梁或桁架切线方向以获得所需的梁截面方向。您可以使用查询工具集来验证梁截面方向对于选定区域是否正确。请参见"使用查询工具集获取分配信息"，第12.19节。

您也可以反转筋条增强的切线方向；但是，如果在几何零件的边缘上定义了多条筋条，您无法在不反转该边缘所有筋条切线的情况下反转其中一条筋条的切线。如果具有多条筋条的边缘是网格零件的特征，您可以单独反转一条筋条的切线，而无需反转为该边缘定义的其他筋条的切线。

**为零件分配梁/桁架切线方向：**

1. 如果您要分配切线的零件在当前视口中不可见，请单击上下文栏中**零件**列表中所需零件的名称。所选零件出现在当前视口中。
2. 从主菜单栏，选择**分配**![](../graphics/images/arrow.gif)**单元切线**。**提示：**您也可以单击属性模块工具箱中的![](../graphics/ico_assignBeamTangent.png)工具，或在网格模块中选择**网格**![](../graphics/images/arrow.gif)**方向**![](../graphics/images/arrow.gif)**切线**。Abaqus/CAE在每个区域中心显示指向切线方向的箭头。
3. 使用以下方法之一选择要反转其切线的区域：- 从视口中选择区域，然后单击鼠标按钮2。（有关更多信息，请参见第6章"在视口中选择对象"。）**提示：**您可以使用**选择**工具栏中的工具限制可以在视口中选择的对象类型。请参见"使用选择选项"，第6.3节。如果您选择**筋条**作为对象类型，可以为整个筋条或其一条或多条边缘分配梁/桁架切线方向。从提示区域，选择**（选择整个筋条）**或**（选择部分筋条）**，然后在视口中进行选择。如果所选零件有多条筋条，Abaqus/CAE在您进行选择时会在提示区域显示模糊选择选项。- 要从现有集合列表中进行选择，请执行以下操作：1. 单击提示区域右侧的**集合**。Abaqus/CAE显示包含可用零件集合列表的**区域选择**对话框。2. 选择感兴趣的零件集合，然后单击**继续**。Abaqus/CAE反转所选区域的切线方向。**注意：**默认选择方法基于您最近使用的选择方法。要切换回另一种方法，请单击提示区域右侧的**在视口中选择**或**集合**。
4. 根据需要经常重复前述步骤以反转其他切线方向。完成分配切线方向后，使用以下方法之一退出分配模式：- 如果您正在从视口中选择零件区域，请单击鼠标按钮2或单击提示区域中的**完成**。- 如果您正在从**区域选择**对话框中选择预存集合，请单击**取消**关闭对话框，然后单击鼠标按钮2或单击提示区域中的**完成**。
