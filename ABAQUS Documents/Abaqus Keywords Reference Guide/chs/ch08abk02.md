# *HEAT GENERATION






### *HEAT GENERATION在热传递分析中包含体积热生成。

此选项用于在材料数据块中包含热传递、耦合热-电、耦合温度-位移或耦合热-电-结构分析中的热生成。它必须与用户子程序 [`HETVAL`](../sub/sub-link.md#sub-xsl-hetval) 一起使用。

**产品：** Abaqus/Standard  Abaqus/CAE  

**类型：** 模型数据  

**级别：** 模型  

**Abaqus/CAE：** Property 模块
