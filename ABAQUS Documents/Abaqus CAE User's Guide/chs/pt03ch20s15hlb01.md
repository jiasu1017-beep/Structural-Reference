# 20.17.5 通过合并边缘来修改 Sketcher 对象


当您导入草图几何或对不在与草图平面平行的平面上的边缘使用投影边缘工具时,Sketcher 对象中可能会出现小间隙。Sketcher 工具箱中的**合并**工具通过将间隙附近的顶点移动到另一侧上的点来闭合这些小间隙。

**注意:**当您合并边缘时,Abaqus/CAE 可以通过将一个顶点移动到与另一个顶点重合,或通过将两个顶点移动到它们之间的位置来闭合间隙。如果您想控制对草图的更改,可以在执行合并操作之前约束您希望保留其当前位置的线条或点。

Abaqus/CAE 使您能够选择要保持静止的顶点,合并操作会将指定公差半径内的任何顶点移动到与您选择的顶点相同的位置。[图 20–26](pt03ch20s14hlb05.md#ske-merge-edges) 显示了一个简单的合并操作,闭合了两个 sketcher 对象之间的上部间隙。

**图 20–26** 合并边缘

![](../graphics/ske-merge-edges-nls.png)

**注意:**合并工具仅用于闭合小间隙。合并边缘以闭合较大的间隙可能会显著改变草图几何。如果您想将草图中的元素移动较长的距离,请使用拖动工具将草图中的顶点移动到新位置。更多信息,请参见["拖动 Sketcher 对象," 第 20.17.1 节](pt03ch20s14hlb01.md)。

**合并边缘的步骤:**

1. 从 Sketcher 工具箱的修改工具中,选择合并工具 ![](../graphics/ico_sketchMergeCurve.png)。有关 Sketcher 工具箱中工具的图示,请参见["Sketcher 工具," 第 20.4.1 节](pt03ch20s04s01.md)。Abaqus/CAE 会在提示区域显示提示,引导您完成操作。
2. 选择小间隙附近的至少两个顶点。
3. 在提示区域单击**完成**。Abaqus/CAE 会提示您输入此合并操作的**公差**值。
4. 输入公差值,或接受默认值 0.001。然后按 Enter,或单击鼠标按钮 2。Abaqus/CAE 允许您增加公差,但您应该避免会显著改变草图几何的合并。如果任何顶点在所选公差内,Abaqus/CAE 会将它们合并到您在第 2 步中选择的顶点位置。
5. 要合并更多边缘,请从第 2 步开始重复上述步骤。
6. 完成合并边缘后,执行以下操作之一: - 在 Abaqus/CAE 窗口任意位置单击鼠标按钮 2。 - 选择 Sketcher 工具箱中的任何工具。 - 单击提示区域中的取消按钮 ![](../graphics/afxI_cancel.png)。
