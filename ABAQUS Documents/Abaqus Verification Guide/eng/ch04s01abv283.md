### Element tested

### Feature tested

### Problem description

User subroutine [`HARDINI`](../sub/sub-link.md#sub-xsl-hardini) is used to define initial conditions for the hardening variables ![](../graphics/ver_eqn00724.gif) and ![](../graphics/ver_eqn01438.gif). These initial conditions are used to solve for the mechanical response of an M3D4 element with rebar in uniaxial tension. The problem definition is the same as in the verification file [mplchi3nt1.inp](../eif/mplchi3nt1.inp) (["Rate-independent plasticity," Section 2.2.10](ch02s02abv148.md)), where the initial conditions are specified. This input file verifies that initial conditions are assigned correctly using user subroutine [`HARDINI`](../sub/sub-link.md#sub-xsl-hardini).

### Results and discussion

The results match the verified results of [mplchi3nt1.inp](../eif/mplchi3nt1.inp).
