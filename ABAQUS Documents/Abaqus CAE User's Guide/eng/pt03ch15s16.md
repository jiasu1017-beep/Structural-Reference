# 15.16 Using contact and constraint detection







The contact detection tool greatly simplifies the process for defining contact interactions and tie constraints in a model. This section explains in detail how to use the contact detection dialog box to locate and create interactions or constraints automatically. The following topics are covered:
- ["Specifying search criteria for contact detection," Section 15.16.1](pt03ch15s16s01.md)
- ["Performing a search for contact pairs," Section 15.16.2](pt03ch15s16hlb01.md)
- ["Reviewing and modifying detected contact pairs," Section 15.16.3](pt03ch15s16s02.md)
- ["Creating interactions for automatically detected contact pairs," Section 15.16.4](pt03ch15s16hlb02.md)

For an overview of contact detection and a discussion of the search methods used by Abaqus/CAE, see ["Understanding contact and constraint detection," Section 15.6](pt03ch15s06.md).
