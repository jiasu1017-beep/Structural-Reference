# 12.7.4 输入温度相关数据





如果您的材料包含温度相关性，您可以输入数据来定义材料属性如何随温度升高而变化。

**输入温度相关数据：**

1. 在材料编辑器中切换**Use temperature-dependent data**。表格数据区域中出现一个标为**Temp**的列。
2. 用适当的值填充每一行。对于特殊的表格编辑选项或从ASCII文件读取数据，请按鼠标按钮3。（有关更多信息，请参见["输入表格数据，"第3.2.7节"](pt01ch03s02s07.md)。）
