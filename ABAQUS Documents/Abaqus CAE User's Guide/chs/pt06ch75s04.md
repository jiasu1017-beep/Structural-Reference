# 75.5 基于几何参数创建虚拟拓扑





您可以使用"创建虚拟拓扑"对话框中的控件，根据一组几何参数自动忽略不必要的模型细节。您可以控制以下参数以确定 Abaqus/CAE 考虑将哪些特征与周围几何合并：
- 边长度
- 面大小
- 面长宽比
- 面转角角度
- 代表小阶梯面的面的厚度
- 用于合并相邻边或面的转角角度
- 用于将圆角或倒角与周围面合并的角度和半径控件
- 冗余实体

当选定的参数超过时，Abaqus/CAE 尝试通过合并相邻的边和面来简化模型。

冗余实体参数只能切换开或关——该参数没有其他设置。冗余实体不会增加模型的几何。冗余实体包括：
- 位于现有面内部的边
- 将原本连续的面分开的边
- 位于面内部的顶点
- 分离两条直线边或平滑曲线但不连接新边的顶点

例如，[图75-7](pt06ch75s05.md#vto-redundant) 包含四条冗余边和一个冗余顶点。虚拟拓扑会将这些边与周围面合并，并将顶点与周围边合并，只留下底层的长方体。

**图75-7** 冗余边和顶点。

![](../graphics/vto-redundant.png)

请记住，冗余实体可能会定义 Abaqus/CAE 中其他工具使用的区域。例如，[图75-7](pt06ch75s05.md#vto-redundant) 中的圆形边可能定义了用于施加压力载荷的面。"创建虚拟拓扑"对话框中的控件完全基于几何参数；您应该仔细审查使用虚拟拓扑所做的更改，以确保更改的实体不需要用于 Abaqus/CAE 模型定义的其他部分。

默认情况下，对话框中的所有虚拟拓扑参数都是激活的，如[图75-8](pt06ch75s05.md#vto-createdb-nls) 所示。边长度、面大小、阶梯面和圆角半径的默认值是根据所选零件或零件实例的大小计算的；其他标准是无量纲的，并提供了合适的默认值。您可以切换关闭您希望被虚拟拓扑忽略的实体的控件。例如，如果您有小面是模型的关键组成部分，请切换关闭**小于面的面**。

**图75-8** "创建虚拟拓扑"对话框。

![](../graphics/vto-createdb-nls.png)

为了帮助您审查潜在的更改，对话框中的每个参数都包含工具，您可以使用这些工具来测量和高亮显示视口中满足相应几何条件的实体。高亮显示的项目如果与其他条件或模型要求冲突，可能不会被更改。要使用这些工具，请单击"创建虚拟拓扑"对话框中每个参数右侧的图标。测量实体 ![](../graphics/ico_measure.png) 工具允许您测量视口中的实体。Abaqus/CAE 在消息区域显示测量值；您可以使用测量值来修订模型的参数限制。高亮工具（如**高亮短边** ![](../graphics/ico_highlightEdge.png)）指示满足当前参数设置的实体。对于"创建虚拟拓扑"对话框的"待合并候选实体"区域，高亮工具指示可以与周围几何合并的实体。

高亮工具不考虑参数组合或可能阻止高亮显示的特征与周围几何合并的任何其他因素。例如，小阶梯面必须满足两个参数才能使用自动虚拟拓扑进行替换：
- 阶梯高度或厚度必须小于指定值。
- 将阶梯面与周围面分开的对向角度必须偏离180度的角度大于用于 sharp 边/顶点指定的角度容差（如下面的曲率控件所述）。

[图75-9](pt06ch75s05.md#vto-stair) 显示了构成阶梯面的角度和面尺寸。如果您使用高亮工具查看潜在面，Abaqus/CAE 在选择要高亮的面时不考虑角度测量值。

**图75-9** 形成阶梯面的面厚度（阶梯高度）*x* 以及角度 ![](../graphics/usi_eqn00007.gif) 和 ![](../graphics/usi_eqn00056.gif)。

![](../graphics/vto-stair.png)

"创建虚拟拓扑"对话框的"合并实体的曲率控件"区域用于合并转角角度和圆角实体。曲率控制参数和高亮工具指示哪些边、顶点或圆角将被保留——不会被虚拟拓扑替换——使用当前设置。以下示例说明了 Abaqus/CAE 如何解释转角角度和圆角的曲率控制参数。

**锐边/顶点的角度容差（偏离180度）**

转角角度是任意两个相邻面或边之间的角度，分别在模型中形成边或顶点。如果转角角度足够接近180度以被认为是平的，Abaqus/CAE 会考虑使用虚拟拓扑来忽略相关的边或顶点。默认参数设置认为偏离180度小于30度的角度为平的。[图75-10](pt06ch75s05.md#vto-cornerangle) 显示了合并面对结果网格的影响。两张上图使用了较小的角度，因此结果网格是几何的近似。随着合并面之间的角度增加，如下图所示，近似程度越来越差。转角角度参数也适用于二维零件边缘之间的面转角角度以及创建小阶梯面的角度。

**提示：**要保留所有转角角度，请指定偏离值为0。

**图75-10** 网格与几何的对应关系随着被虚拟拓扑替换的转角角度（峰和谷）偏离180度越来越大而降低。

![](../graphics/vto-cornerangle.png)

**保留具有大径向角和小半径的圆角**

圆角——圆角或倒角——必须满足两个参数才能避免被虚拟拓扑考虑。这是因为径向角或圆角半径单独都可能创建一个可以被网格良好近似的特征，但大角度与小半径的组合会导致较差的近似。[图75-11](pt06ch75s05.md#vto-blendangle) 说明了两条不同的圆角角度如果与周围几何合并，将如何被网格近似。两种情况的圆角半径相同，但上图中较大的径向角会在拐角处产生不平滑的网格。

**图75-11** 大圆角角度（上图）在被虚拟拓扑替换时可能会产生不平滑的网格。

![](../graphics/vto-blendangle.png)

在[图75-12](pt06ch75s05.md#vto-blendradius) 中，两种情况的径向角相同，但上图中较小的圆角半径会产生质量较差的网格。

**图75-12** 小圆角半径（上图）在被虚拟拓扑替换时可能会产生质量较差的网格。

![](../graphics/vto-blendradius.png)
