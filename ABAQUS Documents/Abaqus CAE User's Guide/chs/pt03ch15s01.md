# 15.1 理解相互作用模块的作用





您可以使用「相互作用」模块来定义以下内容：
- 接触相互作用。
- 弹性基础。
- 空腔辐射。
- 热膜条件。
- 向周围环境的辐射和来自周围环境的辐射。
- Abaqus/Standard到Abaqus/Explicit共仿真相互作用。
- 流固耦合仿真相互作用（介于Abaqus/CFD与Abaqus/Standard或Abaqus/Explicit之间）。
- 压力渗透。
- 入射波。
- 声阻抗。
- 循环对称。
- 用户定义的致动器/传感器相互作用。
- 模型更改相互作用。
- 绑定约束。
- 刚体约束。
- 显示体约束。
- 耦合约束。
- 调整点约束。
- MPC约束。
- 壳到实体耦合约束。
- 嵌入区域约束。
- 方程约束。
- 连接器截面分配。
- 惯性。
- 裂纹。
- 弹簧和阻尼器。

相互作用是步骤相关的对象，这意味着当您定义它们时，必须指明它们在分析的哪些步骤中处于活动状态。（有关步骤相关对象的更多信息，请参见「理解对象在步骤中的状态」，第3.4.4节。）例如，您只能在热传递、耦合温度-位移或耦合热电步骤中定义表面上的膜和辐射条件。同样，您只能在初始步骤中定义具有用户定义致动器/传感器的相互作用。

「相互作用」模块中的「集合和曲面」工具集允许您定义并命名您希望应用相互作用和约束的模型区域。您可以使用「幅值」工具集来定义某些相互作用属性在分析过程中的变化。「解析场」工具集允许您创建可用于为选定相互作用定义空间变化参数的解析场。「参考点」工具集允许您定义用于约束和创建装配级线特征的参考点。

除非在「相互作用」模块中指定，否则Abaqus/CAE不会识别部件实例或装配区域之间的机械接触；装配中两个表面的单纯物理接近不足以表示表面之间任何类型的相互作用。

有关定义裂纹以研究其萌生和扩展的信息，请参见第31章，「断裂力学」。有关为流固耦合定义裂纹的信息，请参见「定义流固耦合仿真相互作用」，第15.13.15节。有关定义惯性的信息，请参见第33章，「惯性」。有关定义弹簧和阻尼器的信息，请参见第37章，「弹簧和阻尼器」。
