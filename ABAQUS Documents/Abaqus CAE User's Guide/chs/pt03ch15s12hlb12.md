# 15.12.13 编辑应用相互作用或约束的区域





您只能在创建相互作用的步骤中编辑应用相互作用的区域。

**编辑应用相互作用或约束的区域的步骤：**

1. 从主菜单栏的「Interaction」或「Constraint」菜单中选择「Manager」以显示「Interaction Manager」或「Constraint Manager」。
2. 选择以下方法之一：- 要编辑相互作用，点击位于要修改的相互作用所在行与其创建步骤所在列交叉处的单元格，然后点击「Edit」。或者，您可以双击该单元格。**提示：**您也可以通过从上下文栏中的「Step」列表中选择相互作用创建所在的步骤来启动此过程。从主菜单栏的「Interaction」菜单中选择 ****Edit**![](../graphics/images/arrow.gif)***相互作用名称*****。- 要编辑约束，选择约束的名称并点击「Edit」。**提示：**您也可以通过从主菜单栏选择 ****Edit**![](../graphics/images/arrow.gif)***约束名称***** 来启动此过程。如果您要编辑基础相互作用，系统会提示您通过在视口中选择或取消选择对象来编辑区域，或者会出现「Region Selection」对话框，您可以在其中选择使用「Surface」工具集已创建的表面。如果您要编辑任何其他类型的相互作用或约束，则会出现相应的编辑器。除通用接触外的所有相互作用的编辑器都包含一个「Edit Region」选项，用于相互作用或约束定义中涉及的每个区域。如果您的相互作用或约束定义同时包含主表面和从表面或区域，则主表面会在编辑过程中以红色高亮显示，从表面或区域会以品红色高亮显示。
3. 如果您要编辑通用接触相互作用，请使用「定义通用接触」，第15.13.1节中描述的方法修改接触域。
4. 如果您要编辑约束或除通用接触或基础相互作用之外的相互作用，请点击您要修改的区域旁边的![](../graphics/ico_selectBlue.png)。例如，如果您要编辑表面对表面接触相互作用并想修改主表面，请点击编辑器中「Master surface」标签旁边的![](../graphics/ico_selectBlue.png)。
5. 如果您要编辑约束或除通用接触之外的相互作用，请通过在视口中选择和取消选择对象来编辑区域。如果您要编辑应用嵌入区域约束的主区域，请首先选择选择方法。完成区域编辑后，点击鼠标按钮2。（更多信息，请参见第6章，「在视口中选择对象」。）**提示：**您可以通过在「Selection」工具栏中指定过滤选项来限制可以在视口中选择的对象类型。请参见「使用选择选项」，第6.3节，以获取更多信息。如果您希望从现有集合或表面列表中进行选择，请执行以下操作：1. 点击提示区域右侧的「Sets」或「Surfaces」。（按钮的名称取决于您正在编辑的对象类型。例如，如果您正在编辑相互作用，则会出现「Surfaces」按钮。）Abaqus/CAE会显示包含可用集合或表面列表的「Region Selection」对话框。2. 选择您感兴趣的集合或表面，然后点击「Continue」。**注意：**默认选择方法基于您最近使用的选择方法。如需恢复为另一种方法，请点击提示区域右侧的「Select in Viewport」或「Sets」或「Surfaces」。
6. 根据需要完成编辑相互作用或约束定义，然后按鼠标按钮2（如果您正在编辑基础相互作用）或在编辑器中点击「OK」（如果您正在编辑任何其他类型的相互作用或如果您正在编辑约束）。
