# 19.2.2 进入和退出Job模块





您可以在会话期间的任何时候通过单击上下文栏中**Module**列表中的**Job**来进入Job模块。 **Job**菜单出现在主菜单栏上。

要退出Job模块，请从主菜单中选择任何其他模块。 如果您的作业成功完成，您也可以通过从主菜单栏选择 ****Job**![](../graphics/images/arrow.gif)**Results**** 来退出Job模块； 您将进入Visualization模块，分析作业的输出数据库将自动打开。

退出模块前无需保存作业； 当您通过从主菜单栏选择 ****File**![](../graphics/images/arrow.gif)**Save**** 或 ****File**![](../graphics/images/arrow.gif)**Save As**** 保存整个模型时，它会自动保存。
