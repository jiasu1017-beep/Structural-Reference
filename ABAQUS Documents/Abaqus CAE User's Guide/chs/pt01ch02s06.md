# 2.6 获取帮助





Abaqus/CAE HTML文档可通过主菜单栏上的**帮助**菜单访问。本节简要描述HTML在线文档，并解释如何使用**帮助**菜单查找信息。其他信息请参阅[使用Abaqus在线文档](../hhp/hhp-link.md#hhp)。

本节描述的功能仅适用于HTML文档，不适用于PDF格式的指南。

**注意：**
- 在Windows平台上，帮助系统使用您的默认Web浏览器显示在线文档。
- 在UNIX和Linux平台上，帮助系统搜索Firefox的系统路径。如果帮助系统找不到Firefox，则会显示错误。可以在环境文件中设置**browser_type**和**browser_path**变量来修改此行为。有关更多信息，请参阅["系统自定义参数，" Abaqus安装和许可指南第4.1.4节](../sgb/sgb-link.md#sgb-chp-env-sys)。
