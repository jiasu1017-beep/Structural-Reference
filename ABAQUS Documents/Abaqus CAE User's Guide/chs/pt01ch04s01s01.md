# 4.1.1 什么是视口？







虽然画布可以被认为是无限屏幕或公告板，但视口只是发布到该屏幕上的显示区域，您可以在其中显示模型或分析结果。您可以在画布上放置多个视口。视口类似于工作站上的其他窗口，因为它可以移动、调整大小、最小化和最大化；并且它可以与画布上的其他视口重叠。有关更多信息，请参阅[「使用视口，第 4.3 节」](pt01ch04s03.md)。

您可以轻松创建和删除视口并控制其大小、位置和外观。[图 4-1](pt01ch04s01s01.md#cnv-viewdemo) 说明瞭如何使用多个视口查看分析结果。

**图 4-1** 使用多个视口工作。

![](../graphics/cnv-viewdemo-nls.png)

[视图操作工具](pt01ch05.md)（如缩放和旋转）在包含光标的视口中操作。其他操作与当前视口或画布上的所有视口交互。

**当前视口**

要更改视口的内容，必须先将所需视口指定为当前视口。当前视口由深灰色标题栏指示。所有工作都在当前视口内进行。要选择另一个视口作为当前视口，请点击边框或标题栏。所选视口移动到画布上其他视口的前面，标题栏颜色变为蓝色。当您选择 Abaqus/CAE 工具或菜单时，标题栏恢复为深灰色。

**注意：**在 Windows 平台上，您可以自定义 Abaqus/CAE 使用的颜色。有关更多信息，请参阅[「Windows 平台上的常见自定义，Abaqus Installation and Licensing Guide 第 5.1.2 节」](../sgb/sgb-link.md#sgb-chp-view-cust-common)。

所有视口都与某个模型和模块关联。当您创建新模型或打开现有模型或输出数据库时，该模型会与当前视口关联。您可以创建不同的视口并将每个视口与不同的模型关联，从而将每个视口指定为当前视口会在关联模型之间切换。类似地，您可以通过在进入不同模块之前将新视口指定为当前视口来同时在多个模块中工作。
