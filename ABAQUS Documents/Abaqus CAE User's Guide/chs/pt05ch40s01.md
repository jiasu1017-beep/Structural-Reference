# 40.1 理解 Visualization 模块的作用





Visualization 模块提供有限元模型和结果的图形显示。它从当前模型数据库获取模型信息，或从输出数据库获取模型和结果信息。您可以通过在 Step 模块中修改输出请求来控制放置在输出数据库中的信息。（有关更多信息，请参见["什么是输出请求？，"第14.4.1节](pt03ch14s04s01.md)。）您可以使用等值线图或符号图查看模型数据库中的数据；您可以通过生成本节描述的任何绘图来查看输出数据库中的模型和结果数据。

**![](../graphics/ico_plotUndeformed.png) [未变形形状](pt05ch43s01.md)**

未变形形状图显示模型的初始形状或基本状态。

**![](../graphics/ico_plotDeformed.png) [变形形状](pt05ch43s02.md)**

变形形状图根据节点变量（如位移）的值显示模型的形状。

**![](../graphics/ico_plotContourUndeformed.png) [等值线](pt05ch44s01.md)**

对于输出数据库，等值线图显示分析变量（如应力或应变）在分析指定步骤和帧的值。对于当前模型数据库中的模型，等值线图显示负载、预定义场或交互在模型所选步骤的值。Visualization 模块将值表示为模型上自定义的彩色线、彩色带或彩色面。

**![](../graphics/ico_plotSymbolUndeformed.png) [符号](pt05ch45s01.md)**

对于输出数据库，符号图在分析的指定步骤和帧显示特定向量或张量变量的大小和方向。对于当前模型数据库中的模型，符号图显示负载、预定义场或交互在模型指定步骤的大小和方向。Visualization 模块将值表示为模型上特定位置的符号（例如箭头）。

**![](../graphics/ico_plotMatOrientUndeformed.png) [材料取向](pt05ch46s01.md)**

材料取向图在分析的指定步骤和帧显示模型中单元的材料方向。Visualization 模块将材料方向表示为单元积分点处的材料取向三轴。

**![](../graphics/ico_xyData.png) [X-Y 数据](pt05ch47s01.md)**

*X-Y* 图是一个变量相对于另一个变量的二维图形。

**![](../graphics/ico_animTimeHistory.png) [时间历史动画](pt05ch49s01s01.md)**

时间历史动画以快速连续的方式显示一系列绘图，产生电影般的效果。各个绘图根据实际结果值随时间变化。

**![](../graphics/ico_animScaleFactor.png) [比例因子动画](pt05ch49s01s02.md)**

比例因子动画以快速连续的方式显示一系列绘图，产生电影般的效果。各个绘图根据应用于特定变形的比例因子变化。

**![](../graphics/ico_animHarmonic.png) [谐波动画](pt05ch49s01s03.md)**

谐波动画以快速连续的方式显示一系列绘图，产生电影般的效果。各个绘图根据应用于所显示复数结果的角度变化。

其他功能包括：

**[可视化诊断信息](pt05ch41s01.md)**

诊断信息帮助您确定模型中不收敛的原因。您可以查看分析每个阶段的信息，并使用 Abaqus/CAE 在视口中高亮显示模型上的问题区域。

**[探测模型、模型绘图和 X-Y 绘图](pt05ch51s01.md)**

探测在您围绕模型或模型绘图移动光标时显示模型数据和剖析结果；探测 *X-Y* 绘图显示图形点的坐标。您可以将此信息写入文件。

**[沿路径绘制结果](pt05ch48s01.md)**

路径是通过在模型中指定一系列点来定义的线。您可以沿路径以 *X-Y* 图的形式查看结果。

**[应力线性化](pt05ch52s01.md)**

应力线性化是通过截面将应力分离为恒定薄膜应力和线性弯曲应力。您将截面指定为穿过模型的路径，Visualization 模块以 *X-Y* 图的形式显示线性化应力。

**[切割模型](pt07ch80s01.md)**

视图切割允许您切穿模型，以便可视化模型的内部或选定的截面。您可以定义平面、圆柱形或球形视图切割。此外，您可以沿恒定等值线变量值定义视图切割。

**[X-Y 和场输出报告](pt05ch54s01.md)**

*X-Y* 报告是 *X* 和 *Y* 数据值的表格列表；场输出报告是场输出值的表格列表。

**[可视化复合层压中的层](pt05ch53.md)**

层压堆叠图是复合层压中层压的图形表示。该图像显示层压中的层以及每层的详细信息，如纤维方向、厚度和参考平面。您也可以在创建复合层压时在 Property 模块中创建层压堆叠图。

**[绘图自定义](pt05ch55s01.md)**

Visualization 模块提供了许多可用于自定义绘图的选项。
