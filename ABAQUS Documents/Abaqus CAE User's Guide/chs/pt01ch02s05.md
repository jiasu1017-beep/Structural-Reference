# 2.5 在Abaqus/CAE中使用鼠标





Abaqus/CAE文档中的许多过程涉及使用三个鼠标按钮中的一个或多个。以下列表说明了在与Abaqus/CAE交互时每个鼠标按钮的重要性：

**鼠标按钮1**

您使用鼠标按钮1在视口中选择对象，展开下拉菜单，以及从菜单中选择项目。文档中的"单击"、"选择"和"拖动"指令指的是鼠标按钮1。

**鼠标按钮2**

在视口中单击鼠标按钮2表示您已完成当前任务。例如：
- 从模型中选择实体：当您创建节点集合时，您选择要包含在集合中的节点。单击鼠标按钮2表示您的选择已完成，您已准备好创建集合。
- 使用工具：单击鼠标按钮2表示您已完成视图操作工具。

此外，在视口中单击鼠标按钮2等同于单击提示区域中突出显示的按钮。例如，如果您尝试从模型中选择节点，而Abaqus/CAE显示了以下提示，则单击鼠标按钮2与单击**确定**具有相同的效果：

![](../graphics/gst-mb2-nls.png)

如果您的鼠标有滚轮作为鼠标按钮2，您可以垂直滚动滚轮来操作视口中模型或绘图您的视图。向下滚动可放大视口内容的视图，向上滚动可缩小视口内容的视图。

**鼠标按钮3**

您按住鼠标按钮3访问弹出菜单，该菜单包含与当前过程相关的功能的快捷方式。例如，当您在创建几何集合时在视口中按下鼠标按钮3时，Abaqus/CAE会显示以下菜单：

![](../graphics/gst-mb3menu-nls.png)

如果您在视口中使用鼠标按钮3，弹出菜单中的大多数项目会复制提示区域中的按钮。鼠标按钮3快捷方式也可用于从模型树和结果树中进行选择，如["在模型树和结果树中使用弹出菜单，" 第3.5.3节"](pt01ch03s05s03.md)中所述。
