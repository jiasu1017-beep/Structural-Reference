# 28.2.2 流体单元库







**产品：** Abaqus/CFD  Abaqus/CAE  

##### **参考资料**

- ["流体（连续体）单元，" 第28.2.1节](pt06ch28s02alm02.md)

### 概述

本节提供Abaqus/CFD中可用的流体单元的参考。

### 单元类型

#### 流体单元

| FC3D4 | 4节点四面体 |
| --- | --- |

| --- | --- |

| --- | --- |
|  |

##### 活动自由度

活动自由度取决于分析过程和选项，例如能量方程和湍流模型。更多信息，参见["Abaqus/CFD中的边界条件"中的"活动自由度"，第34.3.2节"](pt07ch34s03aus119.md#usb-prc-pboundarycfd-dofs)。

##### 额外解变量

无。

### 所需节点坐标

*X*、*Y*、*Z*

### 单元属性定义

| **输入文件用法：** | 使用以下选项定义流的单元属性： |
| --- | --- |
|  | ``` [*FLUID SECTION](../key/key-link.md#usb-kws-mfluidsection) ``` 使用以下选项定义无流动热传递的单元属性： ``` [*SOLID SECTION](../key/key-link.md#usb-kws-msolidsection) ``` |

| **Abaqus/CAE用法：** | 在Abaqus/CAE中，您只能定义流的单元属性。 |
| --- | --- |
|  | 属性模块：**创建截面**：选择**流体**作为截面 |

### 基于单元的载荷

### 分布载荷

分布载荷可用于所有流体单元类型。如["分布载荷，" 第34.4.3节"](pt07ch34s04aus122.md)中所述进行指定。

**载荷ID（*DLOAD)：**  BX**Abaqus/CAE载荷/相互作用：**  **体积力****单位：**  [FL3](../popups/usb-int-iconventions-unitsym.md)**描述：**  全局*X*方向的体积力。

**载荷ID（*DLOAD)：**  BY**Abaqus/CAE载荷/相互作用：**  **体积力****单位：**  [FL3](../popups/usb-int-iconventions-unitsym.md)**描述：**  全局*Y*方向的体积力。

**载荷ID（*DLOAD)：**  BZ**Abaqus/CAE载荷/相互作用：**  **体积力****单位：**  [FL3](../popups/usb-int-iconventions-unitsym.md)**描述：**  全局*Z*方向的体积力。

**载荷ID（*DLOAD)：**  GRAV**Abaqus/CAE载荷/相互作用：**  **重力****单位：**  [LT2](../popups/usb-int-iconventions-unitsym.md)**描述：**  指定方向的重力加载（幅值输入为加速度）。

**载荷ID（*DLOAD)：**  PDBF**Abaqus/CAE载荷/相互作用：**  **多孔阻力体积力****单位：**  [无](../popups/usb-int-iconventions-unitsym.md)**描述：**  多孔阻力体积力载荷（指定孔隙率作为输入）。

### 分布热通量

当分析过程中激活温度方程时，分布热通量可用。如["热载荷，" 第34.4.4节"](pt07ch34s04aus123.md)中所述进行指定。

**载荷ID（*DFLUX)：**  BF**Abaqus/CAE载荷/相互作用：**  **体积热通量****单位：**  [JL3T1](../popups/usb-int-iconventions-unitsym.md)**描述：**  单位体积热体积通量。

### 基于表面的载荷

### 分布热通量

当分析过程中激活温度方程时，基于表面的热通量可用于所有单元。如["热载荷，" 第34.4.4节"](pt07ch34s04aus123.md)中所述进行指定。

**载荷ID（*DSFLUX)：**  S**Abaqus/CAE载荷/相互作用：**  **表面热通量****单位：**  [JL2T1](../popups/usb-int-iconventions-unitsym.md)**描述：**  单位面积热表面通量，流入单元表面。

### 薄膜条件

当分析过程中激活温度方程时，基于表面的薄膜条件可用于所有单元。如["热载荷，" 第34.4.4节"](pt07ch34s04aus123.md)中所述进行指定。

**载荷ID（*SFILM)：**  F**Abaqus/CAE载荷/相互作用：**  **表面薄膜条件****单位：**  [JL2T1![](../graphics/usb_eqn00123.gif)1](../popups/usb-int-iconventions-unitsym.md)**描述：**  单元表面上提供的膜系数和热沉温度（![](../graphics/usb_eqn00123.gif)的单位）。

### 辐射类型

当分析过程中激活温度方程时，基于表面的辐射条件可用于所有单元。如["热载荷，" 第34.4.4节"](pt07ch34s04aus123.md)中所述进行指定。

**载荷ID（*SRADIATE)：**  R**Abaqus/CAE载荷/相互作用：**  **表面辐射****单位：**  [无量纲](../popups/usb-int-iconventions-unitsym.md)**描述：**  单元表面上提供的发射率和热沉温度（![](../graphics/usb_eqn00123.gif)的单位）。
