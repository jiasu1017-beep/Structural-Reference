# 4.1.9 后缘襟翼机构

**产品：** Abaqus/Standard   

本示例说明了使用连接器对飞机后缘襟翼机构展开进行建模。

### 几何与模型

后缘襟翼机构是飞机的重要组成部分。后缘襟翼的形状和定位是飞机升力和空气动力学行为的重要决定因素。

后缘襟翼结构由三个襟翼组成：BASE、MIDFLAP 和 ENDFLAP；这些襟翼通过臂连接。BASE、MIDFLAP 和 ENDFLAP 被建模为显示体；臂被建模为刚性桁架单元。

### 模型相互作用

使用 JOIN 连接器元件将臂在其端点连接到 BASE、MIDFLAP 和 ENDFLAP。使用 LINK 连接器元件来刚性地固定襟翼上枢轴点的位置。

### 结果与讨论

ARM AB 的旋转导致 MIDFLAP 和 ENDFLAP 的展开。

### 输入文件

- edgeflap_model.py - Python 回放文件
- edgeflap.inp - 后缘襟翼机构模型
- edgeflap_plas.inp - 带连接器塑性的模型
