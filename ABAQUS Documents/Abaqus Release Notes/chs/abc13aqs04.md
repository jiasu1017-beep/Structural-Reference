# 13.4 可分配数组





**产品：** Abaqus/Standard  Abaqus/Explicit  Abaqus/CFD

**优点：** 为促进用户子程序之间的数据积累和传输，Abaqus提供了供用户创建和管理动态存储的实用程序函数，形式为可分配数组。

**描述：** 支持两种数组：全局和线程局部。

全局数组在可执行文件的所有线程中可见和可访问。为防止竞争条件，必须使用互斥锁保护来自多个线程的写访问。每次访问都使用互斥锁会产生性能损失。在某些情况下，可以通过限制所有线程在全局数组的非交集范围内操作来避免这种情况。另一种选择是使用线程局部数组。

线程局部数组对线程是本地的。每个线程保存一个数组的私有副本。它们故意不共享，因此不需要锁定或保护免受竞争线程的影响。实际上，一个线程不能交叉引用另一个线程的本地数组。但是，在一个线程中运行的所有用户子程序可以访问该线程的所有数组。由于不需要锁定，线程局部数组可以并发访问，因此比全局数组更快。

这些数组旨在作为公共块的线程安全替代品。您可以创建任意数量的全局或线程局部数组，使用Fortran或C++。在创建时，为每个数组分配一个整数ID。之后，在其他用户子程序中，您可以简单地通过ID引用这些数组，并将它们用作原生C++或Fortran数组。
