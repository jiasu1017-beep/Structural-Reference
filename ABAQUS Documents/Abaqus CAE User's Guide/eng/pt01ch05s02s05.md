# 5.2.5 The box zoom tool







When you select the box zoom tool ![](../graphics/ico_viewZoomRectangle.png) and the viewport in which to work, Abaqus/CAE enters box zoom mode, as indicated by a crosshair-shaped cursor. You use this tool to select a rectangular area of your model or plot; Abaqus/CAE enlarges your view of the selected portion of your model or plot to fill the viewport. For *X–Y* plots, Abaqus/CAE enlarges your view of the selected *X–Y* curves and updates the axis values to match the data that you select.
