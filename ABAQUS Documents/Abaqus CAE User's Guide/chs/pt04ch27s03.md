# 27.3 在可视化模块中显示显示体






您可以使用"可视化"模块中的**显示体选项**来自定义显示体的外观。显示体可用的选项与 Abaqus/CAE 中的常见和叠加绘图选项类似：渲染样式；边缘可见性、颜色和样式；填充颜色；缩放；以及透明度都可以自定义。类似地，显示体选项与绘图状态无关；即应用于显示体的选项会反映在所有绘图状态中。有关更多信息，请参见["自定义显示体的外观，" 第 55.8 节"](pt05ch55s08.md)。

显示体选项适用于模型中的所有显示体；要自定义各个显示体的外观，您可以基于零件实例创建显示组（请参见["创建或编辑显示组，" 第 78.2.1 节"](pt07ch78s02hlb01.md)）。也可以将单个项目着色应用于各个显示体零件实例，以覆盖指定为通用显示体选项的边缘和/或填充颜色；有关更多信息，请参见["自定义单个对象的显示颜色，" 第 77.8 节"](pt07ch77s08.md)。
