# 17.9.3 三维实体的扫掠网格





Abaqus/CAE可以将扫掠网格技术应用于可以沿边从源侧扫掠到目标侧来复制的实体区域。对于三维实体，扫掠路径是一条边，但源侧和目标侧是面。图17-68展示了挤出扫掠网格——Abaqus/CAE对源侧进行网格划分，并沿边将网格挤出到目标侧。图17-69展示了旋转扫掠网格——Abaqus/CAE对源侧进行网格划分，并围绕圆边的轴线旋转该网格到目标侧。

**图17-68** 挤出扫掠网格技术沿边扫掠源侧上的网格。

![](../graphics/mgn-swept3d-nls.png)

**图17-69** 旋转扫掠网格技术沿圆边扫掠源侧上的网格。

![](../graphics/mgn-revolved3d-nls.png)

如果区域可以进行扫掠网格划分，Abaqus/CAE可以在已分配**六面体**、**六面体主导**或**楔形**单元形状选项的区域上生成扫掠网格。为了在源侧上生成初步二维网格，Abaqus/CAE分别使用具有**四边形**、**四边形主导**或**三角形**单元形状选项的自由网格技术。

当您使用扫掠网格技术用六面体或六面体主导单元对实体区域进行网格划分时，您可以在中轴和前进波网格算法之间进行选择。（Abaqus/CAE通过使用两种算法从源侧扫掠到目标侧生成的四边形和四边形主导单元来生成六面体和六面体主导网格。）但是，如果待网格化的区域包含虚拟拓扑，则只能使用前进波算法生成扫掠网格。有关更多信息，请参见["中轴算法和前进波算法有什么区别？"第17.7.6节](pt03ch17s07s06.md)和["使用四边形和四边形主导单元的自由网格，"第17.10.2节](pt03ch17s10s02.md)。

如果您选择前进波算法，Abaqus/CAE将在适当的地方使用映射网格来改进某些区域的网格。（映射网格与结构化网格相同，但仅适用于四边形区域。）Abaqus/CAE确定将前进波算法替换为属于源侧的面的映射网格是否适当。有关更多信息，请参见["什么是映射网格？"第17.8.2节](pt03ch17s08s02.md)和["Abaqus/CAE何时可以应用映射网格？"第17.8.6节](pt03ch17s08s06.md)。Abaqus/CAE使用映射网格在这些适当的边面上创建四边形和四边形主导单元，然后将单元从源侧扫掠到目标侧以创建六面体和六面体主导单元。

如果三维区域具有以下特征，则可以使用扫掠网格技术进行网格划分：
- 将源侧连接到目标侧的每条边必须只有一个面，或由形成规则网格图案的四边形组合面组成。图17-70提供了两个可接受的连接面图案示例。
**图17-70** 扫掠网格的可接受连接面图案。
![](../graphics/mgn-goodsweeppattern-nls.png)
图17-71中分割的连接边没有扫掠网格的可接受面图案；左侧模型有一个带有两个三边形面的扫掠面，而右侧模型没有规则的网格图案。
**图17-71** 扫掠网格的不可接受连接面图案。
![](../graphics/mgn-badsweeppath-nls.png)
- 目标侧必须只包含一个面，没有孤立的边或孤立的顶点。例如，图17-72左侧的区域可以使用扫掠网格技术进行网格划分，因为所有孤立边都在源侧上；但是，右侧的区域不能使用此技术进行网格划分，因为目标侧包含两个面。
**图17-72** 只有左侧的区域可以使用扫掠网格技术进行网格划分。
![](../graphics/mgn-sourcetarget-nls.png)
图17-73展示了一个沿变化横截面进行扫掠网格划分的零件。该零件看起来相对复杂；例如，源侧是非平面的，并且零件的横截面沿扫掠路径变化。但是，生成扫掠网格的规则仍然适用。
- 将源侧连接到目标侧的每条边只包含一个面。
- 虽然源侧包含两个面，但目标侧只包含一个面。
**图17-73** 沿变化横截面扫掠网格。
![](../graphics/mgn-swept-curvedpath-nls.png)
您可以使用虚拟拓扑来合并目标侧上的面，使零件可以进行扫掠网格划分。图17-74展示了一个零件在使用虚拟拓扑将目标侧上的五个面合并为单个面后进行了扫掠网格划分。但是，因为该零件现在包含虚拟拓扑，所以只能使用前进波算法进行扫掠网格划分。
**图17-74** 合并面使零件可以进行扫掠网格划分。
![](../graphics/mgn-swept-virtual-nls.png)
- 对于旋转区域，用于创建区域的轮廓不得在一个或多个孤立的点接触旋转轴，如图17-75所示。
**图17-75** 如果孤立的点接触旋转轴，则扫掠网格技术无法对零件进行网格划分。
![](../graphics/mgn-revolved-point-nls.png)
- 类似地，如果一条或多条边位于旋转轴上，则Abaqus/CAE无法用六面体或楔形单元对区域进行网格划分，如图17-76所示。
**图17-76** 如果一条或多条边位于旋转轴上，则Abaqus/CAE无法用六面体单元对区域进行网格划分。
![](../graphics/mgn-revolved-axis-nls.png)
但是，Abaqus/CAE可以通过沿轴生成楔形单元层来用六面体主导单元对区域进行网格划分，如图17-77所示。
**图17-77** Abaqus/CAE可以用六面体主导单元对区域进行网格划分。
![](../graphics/mgn-revolved-axis-mesh-nls.png)
因此，在对区域进行网格划分之前，您必须选择**六面体主导**单元形状选项。或者，您可以将区域分割成简单的结构化网格区域，并选择**六面体**单元形状选项来使用所有六面体单元创建网格。更多信息，请参见["对轮廓接触旋转轴的旋转实体进行扫掠网格划分，"第17.18.7节](pt03ch17s18s07.md)。
- 完全旋转且不接触旋转轴的区域只有在旋转轮廓关联的所有边都存在时才可进行网格划分。但是，轮廓的边界边不得创建面。图17-78展示了一个可进行网格划分的零件实例，其中旋转轮廓的所有边都存在。
**图17-78** 旋转轮廓的所有边都存在；因此，零件实例可进行网格划分。
![](../graphics/mgn-revolve-profile-all-nls.png)
在此示例中，用户绘制了轮廓，Abaqus/CAE旋转轮廓创建了零件；但是，轮廓的边界边不形成面。相比之下，图17-79展示了一个不可进行网格划分的零件实例，因为旋转轮廓的某些边缺失了。
**图17-79** 旋转轮廓的某些边缺失；因此，零件实例不可进行网格划分。
![](../graphics/mgn-revolve-profile-missing-nls.png)
- 完全旋转且接触旋转轴的区域只有在除了沿旋转轴的边之外，旋转轮廓关联的所有边都存在时才可进行网格划分。图17-80展示了一个可进行网格划分的零件实例，因为除了沿旋转轴的边外，旋转轮廓的所有边都存在。如果轮廓包含沿旋转轴的边，则零件实例将不可进行网格划分。
**图17-80** 除了沿旋转轴的边外，旋转轮廓的所有边都存在；因此，零件实例可进行网格划分。
![](../graphics/mgn-revolve-edge-missing-nls.png)
- 如果旋转区域是通过旋转包含样条的草图创建的，则只有当样条两端的顶点不在旋转轴上时，该区域才可进行网格划分。
- 如果零件是通过沿由闭合样条组成的扫掠路径扫掠横截面创建的，则只有在分割成两个或更多区域后，生成的零件才可进行网格划分。
