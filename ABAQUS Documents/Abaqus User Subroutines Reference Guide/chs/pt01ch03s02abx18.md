# 3.2.18 输出数据库升级实用工具





**产品：** Abaqus/Standard  Abaqus/Explicit  

##### **参考**

- ["Abaqus执行程序：概述，" 第3.1.1节](pt01ch03s01abo02.md)
- ["固定格式转换实用工具，" 第3.2.27节](pt01ch03s02abx27.md)

### 概述

**abaqus upgrade**实用工具会将输出数据库文件从Abaqus早期版本转换到当前版本。可以升级Abaqus 6.1或更高版本的输出数据库文件。**abaqus upgrade**实用工具将生成包含错误、警告、诊断和信息消息的日志文件(`*job-name*.log`)。您应该仔细查看转换日志文件，以确保对较旧版本输出数据库文件所做的更改是适当的。如果不需要转换，日志文件以及屏幕也会显示一条消息。

| --- | --- |
|  | **[**[**odb**](pt01ch03s02abx18.md#odb2-usb-int-dupgradeproc)=*old-odb-file-name***]** |

### 命令行选项

#### 必需选项

**job**

此选项用于指定要由实用工具输出的升级后输出数据库文件的名称。

#### 互斥选项

**odb**

此选项用于指定要升级的输出数据库文件的名称。
