# 11.14.3 定义放样切线





如果您接受放样的默认**指定切线**方法，则可以选择多种放样切线选项。放样切线会影响放样面离开第一个放样截面以及接近最后一个截面的角度。切线设置的效果是临时的，与起始或结束截面的距离成比例地减弱。中间截面之间的放样特征形状不受放样切线的影响。

除了**无**之外，您可以独立设置起始截面和结束截面的所有放样切线选项。例如，您可以将起始切线设置为**法向**，将结束切线设置为**径向**。您可以选择以下选项来定义放样切线：

**无**

**无**是默认设置，是唯一可以与非平面截面一起使用的切线设置。如果您选择**无**，则必须同时将其用于起始和结束切线。**无**不会对放样形状或方向施加任何条件。放样特征的边缘将从起始截面到第二个截面以及从倒数第二个截面到最后截面做出线性接近，如图11-53所示。

**图11-53** 无切线的放样特征。

![](../graphics/prt-loft-auto.png)

**法向**

**法向**设置会迫使由放样边缘创建的面在最初放样到第二个截面时与第一个放样截面成90度。类似地，此设置会迫使这些面在接近放样特征的最后截面时与最后截面成90度。如果您将**起始切线**设置为**法向**，则放样特征的初始部分将类似于直挤压，如图11-54所示。

**图11-54** 两端具有法向切线的放样特征。

![](../graphics/prt-loft-normtan.png)

**径向**

**径向**设置会迫使由放样边缘创建的面在最初放样到第二个截面时与第一个放样截面成0度。类似地，此设置会迫使这些面在接近放样特征的最后截面时与最后截面成0度。因此，面最初从起始放样截面径向向外辐射，或者向结束放样截面内部收缩。如果您将**起始切线**设置为**径向**，则放样特征的初始部分将类似于具有接近90度拔模角的挤压，如图11-55所示。

**图11-55** 左端具有径向切线、右端具有法向切线的放样特征。

![](../graphics/prt-loft-radtan.png)

**警告：**如果您尝试仅使用两个放样截面和不同数量的顶点创建放样特征，将**起始切线**和**结束切线**都设置为**径向**可能会导致放样特征失败。

**指定**

**指定**设置允许您控制应用于放样边缘的**角度**以及表示角度将影响放样的相对距离的**幅度%**。如果您将**起始切线**设置为**指定**，则放样特征的初始部分将类似于具有**角度**度拔模角的挤压，如图11-56所示。

**图11-56** 两端具有指定切线的放样特征。

![](../graphics/prt-loft-spectan.png)

[图11-56](pt03ch11s14s03.md#prt-loft-spectan)显示了45度的**起始切线**角度（左）和135度的**结束切线**角度（右），两者均以25%的幅度施加。作为参考，**法向**切线设置对应于指定90度角度和25%幅度，而**径向**切线设置对应于指定0度角度和25%幅度。

任何点处放样面的角度取决于**角度**和**幅度%**设置、连续放样截面之间的距离以及放样截面之间变化的剧烈程度。根据这些条件，从一个放样截面到下一个截面的平滑过渡要求可能会覆盖某些放样切线效果。如果您需要对放样的形状进行更精确的控制，请使用**选择路径**方法来定义放样特征将精确遵循的路径。
