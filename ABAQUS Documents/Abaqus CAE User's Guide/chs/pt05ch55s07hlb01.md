# 55.7.1 自定义法线样式







法线样式选项控制模型中单元和表面法线箭头的外观。

**自定义法线样式：**

1. 找到通用或叠加的法线样式选项。通用法线样式选项适用于在单独绘制时未变形和变形形状。当未变形形状与变形形状一起绘制时，叠加法线样式选项适用。从主菜单栏选择****选项**![](../graphics/images/arrow.gif)**通用****或**叠加**；然后单击出现的对话框中的"法线"选项卡。
2. 在"样式"区域中：
   1. 单击"长度"字段旁边的箭头，然后选择您想要的箭头长度。默认选择是"中等"。
   2. 单击"线条粗细"字段旁边的箭头，然后选择您想要的箭头粗细。
   3. 单击"箭头"字段旁边的箭头，然后选择您想要的箭头头样式。
3. 在"颜色"区域中，选择您希望不同类型法线显示的颜色。
4. 单击"应用"以实现您的更改。箭头出现在视口中，指示模型中单元或表面的法线方向。通常，箭头出现在外部单元面的形心处。默认情况下，您的更改会在会话期间保存，并会影响当前视口中所有后续绘图以及从当前视口创建的任何新视口中的绘图。如果您希望保留所应用更改以供后续会话使用，请将它们保存到文件。有关更多信息，请参见["保存自定义以供后续会话使用，" 第55.1.1节](pt05ch55s01s01.md)。
