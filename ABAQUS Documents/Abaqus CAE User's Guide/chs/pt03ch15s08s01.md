**基本类型**

基本连接类型包括平移类型和旋转类型。平移类型影响分配了连接器截面的线两端点的平移自由度，可能影响线第一点的旋转自由度。旋转类型仅影响线两端点的旋转自由度。您可以使用单个基本连接类型（平移或旋转）或一个平移和一个旋转类型。

**组装类型**

组装连接类型是基本连接类型的预定义组合。

**复杂类型**

复杂连接类型影响连接中自由度的组合，不能与其他连接类型组合。它们通常对高度耦合的物理连接进行建模。

**MPC类型**

MPC连接类型用于定义两点之间的多点约束。

有关每种连接类型的描述以及定义组装类型连接运动约束的等效基本连接类型，请参见「连接类型库」，Abaqus Analysis User's Guide第31.1.5节，以及「通用多点约束」，Abaqus Analysis User's Guide第35.2.2节。
