### 概述

本节描述的功能用于使用连续体方法对内聚单元进行建模，该方法假定内聚区包含有限厚度的材料，可以使用 Abaqus 中的常规材料模型进行建模。如果内聚区非常薄，从实际角度来看可以视为零厚度，则本构响应通常用牵引-分离定律来描述；此替代方法在 ["使用牵引-分离描述定义内聚单元的本构响应，" 第32.5.6节"](pt06ch32s05alm45.md) 中讨论。

建模为连续体的内聚单元的本构响应：
- 可以使用常规材料模型以 stiffness 和强度等宏观材料特性来定义；
- 可以以内置材料模型或用户定义材料模型的形式指定；
- 可以包括 Abaqus/Explicit 中材料损伤和失效的影响；以及
- 还可以包括 Abaqus/Standard 中低循环疲劳分析中材料损伤和失效的影响。

### 具有常规材料模型的内聚单元行为

Abaqus 中内聚单元的常规材料模型（包括用户定义模型）的实现基于关于内聚层变形状态的某些假设。考虑两类不同的问题：有限厚度粘合剂层的建模和垫片建模。

这些类问题的内聚单元损伤建模可以在 Abaqus/Standard 和 Abaqus/Explicit 中进行（有关损伤模型的详细信息，请参见 ["渐进损伤和失效，" 第24.1.1节"](pt05ch24s01abo21.md)）。您可能需要更改粘合剂材料的损伤模型，以考虑粘合接头失效可能发生在粘合剂与被粘物之间的界面而非粘合剂材料内部这一事实。

当与 Abaqus 中的常规材料模型一起使用时，内聚单元使用真实应力和应变度量。当与基于牵引-分离描述的材料模型一起使用时（参见 ["使用牵引-分离描述定义内聚单元的本构响应，" 第32.5.6节"](pt06ch32s05alm45.md) 了解有关此方法的详细信息），内聚单元使用名义应力和应变度量。

内聚单元的频率特性通过算法自动考虑，这些算法在 Abaqus/Explicit 中选择时间增量（["显式动力学分析，" 第6.3.3节"](pt03ch06s03at08.md)）。在许多涉及粘合剂或垫片的应用中，内聚单元可能相对于其他单元相当薄，这往往会降低稳定时间增量。有关此主题的进一步讨论，包括关于在使用内聚单元时如何避免稳定时间增量显著降低的建议，请参见 ["使用内聚单元建模" 第32.5.3节中的"Abaqus/Explicit 中的稳定时间增量"](pt06ch32s05alm42.md#usb-elm-ecohesiveusage-stabletime)。

#### 有限厚度粘合剂层的建模

对于有限厚度的粘合剂层，假定内聚层仅承受一个直接应变分量，即贯穿厚度应变，以及两个横向剪切应变分量（二维问题为一个横向剪切应变分量）。应变的其他两个直接分量（直接膜应变）和面内（膜）剪切应变假定对本构计算为零。更具体地说，贯穿厚度和横向剪切应变根据单元运动学计算。然而，膜应变不是根据单元运动学计算的；它们只是假定对本构计算为零。这些假设在相对薄且柔软的粘合剂层粘合两个相对刚性（与粘合剂相比）部件的情况下是合适的。上述运动学假设在内聚层内部除其外边缘附近外几乎到处都成立。

可以定义额外的线性弹性横向剪切行为，以向内聚单元提供更多稳定性，特别是在损伤发生后。横向剪切行为假定与常规材料响应无关，不会发生任何损伤。

| **输入文件用法：** | 使用以下选项（仅在需要定义解耦横向剪切响应时才需要第二个选项）： |
| --- | --- |
|  | ``` [*COHESIVE SECTION](../key/key-link.md#usb-kws-mcohesivesection), RESPONSE=CONTINUUM [*TRANSVERSE SHEAR STIFFNESS](../key/key-link.md#usb-kws-mtransshearstiff) ``` |

| **Abaqus/CAE 用法：** | 属性模块：**创建截面**：选择**其他**作为截面**类别**和**内聚**作为截面**类型**：**响应**：**连续体** |
| --- | --- |
|  | Abaqus/CAE 不支持内聚截面的横向剪切行为。 |

#### 垫片和/或小型粘合剂补丁的建模

垫片和/或小型粘合剂补丁的建模涉及内聚层没有侧向约束的情况。因此，这些层可以以无应力方式自由地向侧向膨胀。应用领域包括单个点焊和垫片。与 Abaqus/Standard 中的垫片单元系列相比，此选项提供的垫片建模功能有一些优点。内聚单元是完全非线性的（单元运动学正确地考虑了有限应变以及有限旋转），可以在动态分析中贡献质量和阻尼，并且在 Abaqus/Explicit 中可用。以上述方式建模的垫片响应类似于使用 Abaqus/Standard 中专用垫片单元进行建模，但仅具有厚度方向行为（参见 ["在模型中包含垫片单元，" 第32.6.3节"](pt06ch32s06alm48.md)）。

如果需要，可以定义解耦的线性弹性横向剪切行为。横向剪切行为可以定义垫片和/或粘合剂补丁的响应，或在厚度方向响应中发生损伤后提供稳定性。横向剪切响应没有相关的损伤。

| **输入文件用法：** | 使用以下选项（仅在需要定义解耦横向剪切响应时才需要第二个选项）： |
| --- | --- |
|  | ``` [*COHESIVE SECTION](../key/key-link.md#usb-kws-mcohesivesection), RESPONSE=GASKET [*TRANSVERSE SHEAR STIFFNESS](../key/key-link.md#usb-kws-mtransshearstiff) ``` |

| **Abaqus/CAE 用法：** | 属性模块：**创建截面**：选择**其他**作为截面**类别**和**内聚**作为截面**类型**：**响应**：**垫片** |
| --- | --- |
|  | Abaqus/CAE 不支持内聚截面的横向剪切行为。 |

### 输出

Abaqus 中的所有标准输出变量（["Abaqus/Standard 输出变量标识符，" 第4.2.1节"](pt02ch04s02abv01.md) 和 ["Abaqus/Explicit 输出变量标识符，" 第4.2.2节"](pt02ch04s02xbv01.md)）可用于与常规材料模型一起使用的内聚单元。额外横向剪切响应引起的应力使用输出变量 TSHR13 和（三维中）TSHR23 单独报告。这些应力不会添加到使用输出变量 S 报告的常规材料点应力中。
