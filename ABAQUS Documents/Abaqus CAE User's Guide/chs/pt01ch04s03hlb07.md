# 4.3.7 平铺视口





您可以将画布上的视口 arranged 成平铺图案。这可能有助于显示隐藏的视口，或者仅仅是为了减少绘图区域的混乱。Abaqus/CAE排列视口，使内容可见，但部分标题栏可能被遮挡。

**要平铺视口：**

1. 使用视口右上角的**最小化**按钮来最小化您不想平铺的任何视口。
2. 执行以下操作之一：- 从主菜单栏，选择****视口（Viewport）**![](../graphics/images/arrow.gif)**水平平铺（Tile Horizontally）****以在保持最大可能水平尺寸的同时平铺视口。- 从主菜单栏，选择****视口（Viewport）**![](../graphics/images/arrow.gif)**垂直平铺（Tile Vertically）****以在保持最大可能垂直尺寸的同时平铺视口。Abaqus/CAE将所有未最小化的视口设置为相同大小，并排列它们以填充绘图区域。所有平铺的视口按数字顺序排列。最小化的视口位于任何平铺视口的前面，但当前视口除外。**提示：**您也可以通过点击**视口**工具栏中的![](../graphics/ico_viewportTileHorizontal.png)或![](../graphics/ico_viewportTileVertical.png)来平铺未最小化的视口。
