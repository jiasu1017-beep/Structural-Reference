# 10.7.2 导入零件





从主菜单栏选择****文件（File）**![](../graphics/images/arrow.gif)**导入（Import）**![](../graphics/images/arrow.gif)**零件（Part）****从以下来源导入零件：

**外部格式文件**

您可以从以下格式导入几何信息：
- ACIS
- CATIA V4
- 由Pro/ENGINEER、NX、CATIA V5或CATIA V6生成的Elysium中性文件
- CATIA V5
- IGES
- Parasolid
- STEP
- VDA-FS

有关Abaqus/CAE支持的CAD软件版本的信息，请参阅Dassault Systmes知识库，网址为[www.3ds.com/support/knowledge-base](http://www.3ds.com/support/knowledge-base)。

您可以导入装配中的所有零件，也可以仅导入指定的零件。Abaqus/CAE为导入的每个零件创建一个单独的零件。因此，如果零件在装配中被实例化多次，Abaqus/CAE会创建对应于每个实例的单独零件。Abaqus/CAE不保留零件的名称或任何颜色编码。

**输出数据库**

您可以以网格零件形式导入存储在输出数据库中的零件实例。网格零件不包含特征信息，是作为节点、单元、曲面和集合的集合从输出数据库中提取的。您可以导入未变形或变形形状。如果您导入变形形状，可以指定要导入的步骤和增量。更多信息，请参阅["从输出数据库导入零件，"第10.7.12节"](pt02ch10s07hlb12.md)。

**子结构**

您可以将子结构作为新零件定义导入模型数据库。更多信息，请参阅["将子结构作为零件导入模型数据库，"第10.7.13节"](pt02ch10s07hlb13.md)。

**要导入零件：**

1. 从主菜单栏，选择****文件（File）**![](../graphics/images/arrow.gif)**导入（Import）**![](../graphics/images/arrow.gif)**零件（Part）****。出现**导入零件（Import Part）**对话框。
2. 从**导入零件**对话框底部的**文件过滤器（File Filter）**菜单中，选择以下之一：- **ACIS SAT (*.sat)**；更多信息，请参阅["从ACIS格式文件导入零件，"第10.7.4节"](pt02ch10s07hlb04.md)。- **IGES (*.igs, *.iges)**；更多信息，请参阅["从IGES格式文件导入零件，"第10.7.7节"](pt02ch10s07hlb07.md)。- **VDA (*.vda)**；更多信息，请参阅["从VDA-FS格式文件导入零件，"第10.7.11节"](pt02ch10s07hlb11.md)。- **STEP (*.stp, *.step)**；更多信息，请参阅["从STEP格式文件导入零件，"第10.7.10节"](pt02ch10s07hlb10.md)。- **CATIA V4 (*.model, *.catdata, *.exp)**；更多信息，请参阅["从CATIA V4或V5格式文件导入零件，"第10.7.5节"](pt02ch10s07hlb05.md)。- **CATIA V5 (*.CATPart, *.CATProduct)**；更多信息，请参阅["从CATIA V4或V5格式文件导入零件，"第10.7.5节"](pt02ch10s07hlb05.md)。- **Parasolid (*.x_t,*.x_b, *.xmt)**；更多信息，请参阅["从Parasolid格式文件导入零件，"第10.7.9节"](pt02ch10s07hlb09.md)。- **ProE/NX/CATIA V5 Elysium中性 (*.enf)**；更多信息，请参阅["从Elysium中性文件导入零件，"第10.7.6节"](pt02ch10s07hlb06.md)。- **输出数据库 (*.odb)**；更多信息，请参阅["从输出数据库导入零件，"第10.7.12节"](pt02ch10s07hlb12.md)。- **子结构 (*.sim)**；更多信息，请参阅["将子结构作为零件导入模型数据库，"第10.7.13节"](pt02ch10s07hlb13.md)。
3. Abaqus/CAE会列出所选目录中具有适当文件扩展名的所有文件。选择包含要导入的零件的文件，然后单击**确定（OK）**。Abaqus/CAE进入"零件"模块，并显示**从*格式*文件创建零件（Create Part from *format* File）**对话框。
4. 选择所需的修复选项和零件属性，然后单击**确定（OK）**。更多信息，请参阅["控制导入过程，"第10.3节"](pt02ch10s03.md)和["零件属性是什么？，"第10.3.2节"](pt02ch10s03s02.md)。
