# 4.10.8 3DNLG-8：纯弯曲作用下直管段的塌陷





**产品：** Abaqus/Standard   

### 测试单元

S3    S3R    S4    S4R    S4R5    S8R    S8R5    S9R5    

STRI3    STRI65    

### 问题描述

![](../graphics/bmk3dnlg8-nls.png)

测试了纯弯曲作用下梁的弹性和弹塑性情况。还测试了非线性多点约束和截面椭圆化。

**模型：**

由于对称性，使用四分之一模型。

**材料：**

弹性模量 = 1.93×10⁵，泊松比 = 0.26，塑性（等向硬化）。

| 等效应力 | 等效塑性应变 |
| --- | --- |
| 272 | 0.0 |
| 346 | 0.00473 |
| 379 | 0.01264 |
| 404 | 0.02836 |
| 424 | 0.0491 |

**边界条件：**

在z = 0平面上对称（![](../graphics/bmk_eqn01872.gif)）；在x = 0平面上对称（![](../graphics/bmk_eqn01873.gif)）；在旋转平面上对称：i）壳节点必须保持在旋转平面内（非线性MPC）。ii）旋转平面内壳节点的z方向旋转必须与主节点相同（线性MPC）。iii）关于局部![](../graphics/bmk_eqn01874.gif)轴的旋转分量必须为零（非线性MPC）。主节点被约束为仅在x轴上移动。

**载荷：**

对管道端部施加Z方向旋转（![](../graphics/bmk_eqn01875.gif) = 0.004）。

### 参考解

这是英国国家有限元方法与标准机构（NAFEMS）推荐的测试：NAFEMS出版物R0024"A Review of Benchmark Problems for Geometric Non-linear Behaviour of 3D Beams and Shells (SUMMARY)"中的测试3DNLG-8。

此问题的已发布结果由Abaqus获得。因此，Abaqus与NAFEMS结果的比较不是对Abaqus的独立验证。NAFEMS研究包括来自其他来源的比较结果，这些结果可能为此问题提供验证依据。
