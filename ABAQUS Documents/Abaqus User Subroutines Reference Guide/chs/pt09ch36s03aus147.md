### 概述

接触属性：
- 定义控制表面接触时行为的机械和热表面相互作用模型；和
- 分配给单个接触对。

### 为接触对分配表面相互作用定义

表面相互作用定义指定接触对使用的本构接触属性和约束施加方法。模型中的每个接触对必须引用表面相互作用定义，即使接触对使用默认接触属性模型。关于定义接触属性的信息，请参阅["机械接触属性概述，" 第37.1.1节"](pt09ch37s01aus165.md)。可以在表面相互作用定义中指定非默认约束施加方法，如["Abaqus/Standard中的接触约束施加方法，" 第38.1.2节"](pt09ch38s01aus178.md)中所描述。

多个接触对可以引用相同的表面相互作用定义。

表面相互作用定义还用于在以下情况下提供表面面外厚度的非默认值：
- 二维模型，或
- 涉及基于节点的表面的接触对，以在基于节点表面上每个节点处提供非默认横截面积。

Abaqus/Standard使用表面相互作用定义中提供的面外表面厚度和横截面积来计算接触应力。

| **输入文件用法：** | 同时使用以下两个选项： |
| --- | --- |
| | ``` [*CONTACT PAIR*](../key/key-link.md#usb-kws-hcontactpair), INTERACTION=*interaction_property_name* [*SURFACE INTERACTION*](../key/key-link.md#usb-kws-hsurfaceinteraction), NAME=*interaction_property_name* ``` |

| **Abaqus/CAE用法：** | 相互作用模块： |
| --- | --- |
| | **创建相互作用属性**：**名称**：*interaction_property_name*，**接触** 相互作用编辑器：**接触相互作用属性**：*interaction_property_name* |
