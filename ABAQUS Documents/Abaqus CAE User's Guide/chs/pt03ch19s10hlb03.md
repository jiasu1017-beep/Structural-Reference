# 19.10.3 指定常规自适应过程设置







要找到自适应过程编辑器，请从主菜单栏中选择****自适应**![](../graphics/images/arrow.gif)**编辑**![](../graphics/images/arrow.gif)***自适应过程名称*****。使用**常规**选项卡页面配置杂项自适应过程设置。您可以配置以下内容：

****预处理器打印输出****

**预处理器打印输出**选项允许您控制 Abaqus 是否将输入数据、接触约束、模型定义数据和历史数据的回显打印到数据（`.dat`）文件。默认情况下，这些选项都处于关闭状态。

****临时目录****

**临时目录**选项允许您指定用于临时文件的目录名称。在 UNIX 系统上，默认临时目录是 `$TMPDIR` 环境变量的值，如果未定义该变量，则为 `/tmp`。在 Windows 系统上，默认临时目录是 `TEMP` 环境变量的值，如果未定义该变量，则为 `\TEMP`。要指定临时目录，您可以执行以下操作之一：
- 点击**临时目录**文本字段，然后输入目录路径。
- 点击![](../graphics/ico_fileOpen.png)显示**选择临时目录**对话框，然后选择您选择的目录。

****用户子程序文件****

提供包含模型引用的所有用户子程序的文件名称。要指定用户子程序文件，您可以执行以下操作之一：
- 点击**用户子程序文件**文本字段，然后输入文件路径。
- 点击![](../graphics/ico_fileOpen.png)显示**选择用户子程序文件**对话框，然后选择您选择的文件。

如果您的模型引用了用户子程序，但您未在**常规**选项卡页面中指定子程序文件的名称，Abaqus 会生成作业监视器对话框报告的错误。（您可以通过从主菜单栏中选择****作业**![](../graphics/images/arrow.gif)**监视器**![](../graphics/images/arrow.gif)***您选择的作业*****来显示作业监视器对话框。）有关用户子程序的更多信息，请参见["用户子程序和实用程序，"《Abaqus 分析用户手册》第 18.1 节](../usb/usb-link.md#usbusubutil)。

**结果格式**

**结果格式**选项允许您以 ODB 格式、SIM 格式或两种格式写入 Abaqus 分析的结果。有关更多信息，请参见["输出数据库"中的"输出"，《Abaqus 分析用户手册》第 4.1.1 节](../usb/usb-link.md#usb-out-ooutput-formats)。
