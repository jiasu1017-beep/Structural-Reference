# 78.2 管理显示组





显示组管理器允许您创建显示组，以及绘制、编辑、复制、重命名或删除先前保存的显示组。当前模块的显示组管理器仅列出您在当前会话期间在同一类型模块中保存的显示组。零件相关模块中的**零件显示组管理器**列出每个显示组所属的模型和零件；装配相关模块中的**装配显示组管理器**列出每个显示组所属的模型。

在可视化模块中，您可以在同一视口中绘制多个显示组。可视化模块中的**ODB显示组管理器**允许您锁定和解锁绘制的显示组以有选择地自定义绘图选项。此外，您可以同步所有绘制的显示组以使用相同的绘图选项。
