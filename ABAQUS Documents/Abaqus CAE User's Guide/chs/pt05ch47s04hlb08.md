# 47.5.7 自定义X–Y图轴标题





### 47.5.7 自定义X–Y图轴标题

点击**Title**选项卡来自定义出现在*X–Y*图的*X*轴和*Y*轴上的标题。您可以自定义以下内容：
- 作为每个标题出现的文本。
- 每个标题的颜色。
- 每个标题的字体。

**要自定义X–Y图的轴标题：**

1. 找到轴**Title**选项。选择****Options**![](../graphics/images/arrow.gif)**XY Options**![](../graphics/images/arrow.gif)**Axis****；或点击![](../graphics/ico_xyAxisOptions.png)，它位于Visualization模块工具箱中的*X–Y*绘图工具中。在出现的对话框中点击**Title**选项卡。
2. 从**X Axis**或**Y Axis**字段中，高亮显示一个或多个轴。
3. 要为所选轴指定默认标题以外的标题文本，请在字段中输入标题。您可以通过切换**Use default**恢复默认标题。
4. 选择轴标题的字体。1. 点击![](../graphics/ico_text16.png)。Abaqus/CAE显示**Select Font**对话框。2. 使用**Select Font**对话框中的方法之一选择新字体、大小和样式。有关更多信息，请参见["自定义字体，"第3.2.8节](pt01ch03s02s08.md)。3. 点击**Apply**查看字体选择的效果。4. 点击**OK**关闭**Select Font**对话框。X–Y图标题变为所选字体、大小和样式。
5. 选择轴标题的颜色。1. 点击颜色样本![](../graphics/usi-colorsample.png)。Abaqus/CAE显示**Select Color**对话框。2. 使用**Select Color**对话框中的方法之一选择新颜色。有关更多信息，请参见["自定义颜色，"第3.2.9节](pt01ch03s02s09.md)。3. 点击**OK**关闭**Select Color**对话框。所选轴的轴标题变为所选颜色。
6. 点击**Dismiss**关闭**Axis Options**对话框。
