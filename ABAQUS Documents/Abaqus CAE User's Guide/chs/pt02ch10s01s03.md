# 10.1.3 Elysium插件可以做什么？





您可以使用Elysium，Inc.基于中性文件的翻译器将从以下CAD软件创建的零件导入Abaqus/CAE：
- CATIA V4
- 创建Parasolid格式文件的CAD软件，如SolidWorks、NX、Solid Edge、FEMAP和MSC.Patran

您也可以使用Elysium翻译器导入整个装配或仅从装配中选择零件实例。

此外，您可以使用Elysium中性文件格式将零件、装配或从装配中选择的零件实例导入Abaqus/CAE。Abaqus提供了来自Elysium的翻译器插件，可使用Elysium中性文件格式生成几何文件。该插件可用于以下产品：
- CATIA V5
- NX
- Pro/ENGINEER

有关Elysium翻译器支持的CAD软件版本的信息，请参阅Dassault Systmes知识库，网址为[www.3ds.com/support/knowledge-base](http://www.3ds.com/support/knowledge-base)。
