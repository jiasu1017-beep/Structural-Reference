# 4.7.6 Test 5: Axisymmetric crack in a bar







**Product: **Abaqus/Standard

![](../graphics/bmknlf5.png)

![](../graphics/bmk_eqn01282.gif) along edge AB, ![](../graphics/bmk_eqn01281.gif) along edge AE.
