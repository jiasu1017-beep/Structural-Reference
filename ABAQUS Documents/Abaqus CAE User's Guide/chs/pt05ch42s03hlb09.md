# 42.5.10 选择接触输出





非运动标量输出变量（如CPRESS、COPEN、CSLIP、CSHEAR1和CSHEAR2）的接触输出数据从所有连续单元为基础的表面合并为一个数据集。重叠表面的数据被合并。如果为膜或壳单元请求接触输出，则SPOS和SNEG表面的数据包含在单独的数据集中。当接触表面为双侧或基于节点时，会为每个接触表面对创建单独的接触输出数据集。

Abaqus/CAE用于在合并数据集中组合重叠小面数据的方法取决于输出变量：
- 对于变量CPRESS、CSHEAR和CSLIP，Abaqus/CAE使用模型中选定变量的最大值（正或负）。
- 对于变量COPEN，Abaqus/CAE使用接触开口的最小值，这可以反映模型中的最小开口，或者如果存在过盈，则为最大的过盈值。
- 对于向量变量CNORMF和CSHEARF，Abaqus/CAE评估所有单个表面对的所有向量值的总和，并将合成向量存储在合并数据集中。
- 对于变量CSTATUS，Abaqus/CAE为主表面和从表面上的各个节点分配三种接触状态之一。Abaqus/CAE在合并数据集中使用以下设置：如果任何重叠节点处于粘着状态，则为**Closed (Sticking)**；如果没有任何重叠节点处于粘着状态且一个或多个重叠节点正在滑动，则为**Closed (Slipping)**；如果所有重叠节点都处于开放状态，则为**Open**。

Abaqus/CAE中每个数据集出现的变量名指示接触输出适用的区域。例如，对于单个接触表面对（用于双侧或基于节点的接触表面）的CPRESS数据集被命名为`CPRESS *SLAVE NAME/MASTER NAME*`；对于膜或壳表面SPOS或SNEG侧的接触数据分别命名为`CPRESS SPOS`和`CPRESS SNEG`；所有其他表面变量输出命名为`CPRESS`。

任何单个接触表面对的接触输出数据（不包括来自其他接触对的求和贡献）可以使用**Create Field Output**对话框或Abaqus Scripting Interface访问输出数据库；这些数据集的名称格式为`CPRESS *SLAVE NAME/MASTER NAME*`。

由于CPRESS、CSHEAR1和CSHEAR2与模型中定义的特定表面相关，因此在探测这些输出变量结果时，这些表面必须包含在当前显示组中。这些变量只能在节点处探测。
