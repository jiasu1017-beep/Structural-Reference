# 15.13.18 定义入射波







您可以模拟由于外部声源引起的入射波载荷。从主菜单栏中选择****相互作用**![](../graphics/images/arrow.gif)**创建****，然后选择源点、 standoff点（对于除使用CONWEP模型之外的所有入射波载荷定义）以及表面。在指定源点和standoff点时，您只能选择未与任何其他模型组件关联的参考点。您可以在以下类型的分析中创建入射波相互作用：隐式动态、显式动态、直接稳态动态和基于子空间的稳态动态。入射波相互作用仅在其创建的步骤中活动，不会传播到任何后续步骤。如果您的模型包含入射波相互作用，您必须编辑模型属性（见下文）以为模型指定声波 formulations。

有关入射波相互作用的简要概述，请参见"理解相互作用"，第15.3节。有关更详细的讨论，请参见Abaqus Analysis User's Guide的第6.10.1节"声学、冲击和耦合声固分析"以及Abaqus Analysis User's Guide的第34.4.6节"声学和冲击载荷"中的"外部源引起的入射波载荷"。

**为模型指定声波 formulations：**
- 如果您的模型包含入射波相互作用，您必须编辑模型属性以指定声波formulation。使用"指定模型属性"，第9.8.4节中描述的过程选择散射波formulation或总波formulation。您选择的formulation适用于模型中的所有入射波相互作用。更多信息，请参见Abaqus Analysis User's Guide的第34.4.6节"声学和冲击载荷"中的"散射和总波formulation"。

**定义入射波相互作用：**

1. 从主菜单栏中，选择****相互作用**![](../graphics/images/arrow.gif)**创建****。**提示：**您也可以使用相互作用模块工具箱中的![](../graphics/ico_interaction.png)工具创建入射波相互作用。
2. 在出现的**创建相互作用**对话框中，执行以下操作：- 命名该相互作用。有关命名对象的信息，请参见第3.2.1节"使用基本对话框组件"。- 选择将创建该相互作用的步骤。您只能在隐式、显式、直接稳态或基于子空间的稳态动态分析期间定义入射波相互作用。入射波相互作用仅在其创建的步骤中活动。- 选择**入射波**类型的相互作用。
3. 单击**继续**关闭**创建相互作用**对话框。
4. 通过选择未与任何其他模型组件关联的参考点来选择入射波源点。更多信息，请参见第72.3节"创建参考点"。您选择的源点在视口中以红色高亮显示。
5. 对于除使用CONWEP模型之外的所有入射波定义，通过选择未与任何其他模型组件关联的参考点来选择入射波standoff点。更多信息，请参见第72.3节"创建参考点"。您选择的standoff点在视口中以绿色高亮显示。
6. 对于使用CONW数据的空气或表面爆炸数据的入射波定义，请在提示区域中单击**CONWEP（空气/表面爆炸）**。Abaqus/CAE为此入射波定义指定**定义**为**CONWEP**。
7. 使用以下方法之一选择表面：- 使用现有表面定义区域。在提示区域右侧单击**表面**。从出现的**区域选择**对话框中选择现有表面，然后单击**继续**。**注意：**默认选择方法基于您最近使用的选择方法。要恢复为另一种方法，请单击提示区域右侧的**在视口中选择**或**表面**。- 使用鼠标在视口中选择区域。您选择的表面或区域在视口中以洋红色高亮显示，并且出现**编辑相互作用**对话框。
8. 对于除使用CONWEP模型之外的所有入射波定义，单击**定义**字段的箭头，然后从出现的列表中选择一个选项。对于直接和基于子空间的稳态动态分析，您只能指定standoff点的流体压力时间历史。- 选择**压力**以指定standoff点的流体压力时间历史。- 选择**加速度**以指定standoff点的流体粒子加速度时间历史。- 选择**UNDEX**以指定UNDEX气泡数据。
9. 在编辑器的**波数据**部分，执行以下步骤：1. 选择入射波相互作用属性。如果需要，请单击![](../graphics/ico_interactionProperty.png)创建相互作用属性；有关更多信息，请参见第15.14.7节"定义入射波相互作用属性"。存在以下要求：- 如果您选择了**压力**定义，则必须选择具有平面定义或球形定义的相互作用属性，并具有声学或广义衰减传播模型。- 如果您选择了**加速度**定义，则必须选择具有平面定义的相互作用属性。- 如果您选择了**UNDEX**定义，则必须选择具有球形定义和UNDEX电荷传播模型的相互作用属性。- 如果您选择了**CONWEP**定义，则必须选择具有空气爆炸或表面爆炸定义的相互作用属性。2. 在**参考大小**字段中，输入用于缩放幅值定义中给出的值的参考大小。
10. 如果您选择了**压力**或**加速度**定义，请从编辑器的**Standoff点**部分执行以下步骤：- 对于隐式或显式动态分析，根据时间历史定义选择**压力幅值**或**加速度幅值**。- 对于直接或基于子空间的稳态动态分析，切换打开**实部幅值**和/或**虚部幅值**，然后选择幅值。如果需要，请单击![](../graphics/ico_amplitudeSmall.png)创建新幅值；请参见第57.3节"选择要定义的幅值类型"。
11. 如果您选择了**UNDEX**定义，请在编辑器的**UNDEX数据**部分执行以下步骤：1. 在**流体表面法线的方向余弦**字段中，输入流体表面法线方向余弦的*X*分量、*Y*分量和*Z*分量。2. 在**初始深度**字段中，输入UNDEX电荷的初始深度。
12. 如果您选择了**CONWEP**定义，请在编辑器的**CONWEP数据**部分执行以下步骤：1. 在**引爆时间**字段中，根据整个分析的总时间指定爆炸发生的时间。2. 在**大小缩放因子**字段中，指定要用来将CONWEP数据单位缩放到分析数据单位的因子。
13. 单击**确定**创建相互作用并关闭编辑器。
