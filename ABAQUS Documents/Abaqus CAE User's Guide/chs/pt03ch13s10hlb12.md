# 13.10.12 合并或切割零件实例





要合并所选的零件实例组，请从主菜单栏选择****Instance**![](../graphics/images/arrow.gif)**Merge/Cut****。你可以合并在零件模块中创建的零件几何，也可以合并基于几何和孤立网格节点和单元，或者可以同时合并几何和孤立网格特征。你也可以使用网格模块中的编辑网格工具集合并选定的节点；更多信息，请参见["操作节点，"第64.1.1节](pt06ch64s01s01.md)。你可以使用一个或多个其他零件实例的几何来切割所选零件实例的几何；被切割实例中的任何孤立网格节点或单元都会保留在新的切割零件实例中。

合并或切割操作会在装配中创建新的零件实例和新零件。你可以选择抑制或删除你选择的原始零件实例。更多信息，请参见["对零件实例执行布尔操作，"第13.7节](pt03ch13s07.md)。

你不能在模型实例上使用**Merge/Cut**命令。

**要合并或切割零件实例：**

1. 从主菜单栏，选择****Instance**![](../graphics/images/arrow.gif)**Merge/Cut****。
   **提示：**你也可以使用装配模块工具箱中的![](../graphics/ico_instanceMergeCut.png)工具来合并或切割零件实例。有关装配工具箱中工具的图表，请参见["使用装配模块工具箱，"第13.9节](pt03ch13s09.md)。Abaqus/CAE显示**Merge/Cut Instances**对话框。
2. 输入将由该操作创建的新零件名称。
3. 选择操作类型：
   - 要合并零件实例的几何，选择**Merge**和**Geometry**。任何原生或孤立网格节点和单元都不会包含在新零件中。
   - 要合并零件实例的网格，选择**Merge**和**Mesh**。任何几何都不会包含在新零件中，任何原生网格都会变成孤立网格。
   - 要同时合并零件实例的几何和网格特征，选择**Merge**和**Both**。
   - 要切割零件实例，选择**Cut geometry**。
4. 选择你希望Abaqus/CAE如何处理正在被合并或切割的原始实例：
   - 选择**Suppress**来抑制原始零件实例，但将它们保留在模型数据库中。完成**Merge/Cut**操作后，必要时可以恢复原始零件实例（请参见["抑制和恢复对象，"第3.4.3节](pt01ch03s04s03.md)）。
   - 选择**Delete**从模型数据库中删除原始零件实例。你不能恢复已删除的零件实例。
5. 如果你选择了**Merge**操作，请执行以下操作：
   1. 选择所需的选项：
      **Geometry** 默认情况下，Abaqus/CAE移除相交零件实例之间的边界。如果你想保留相交零件实例之间的边界，请从**Merge/Cut Instances**对话框底部选择**Retain**。移除和保留边界的效果如图13-25所示。
      **图13-25** 移除和保留相交边界的效果。
      ![](../graphics/asm-position-intersections.png)
      **Mesh** 选择Abaqus/CAE将用于合并节点的方法：
      - **仅边界**。默认情况下，Abaqus/CAE仅沿边界合并网格。因此，Abaqus/CAE不检查零件内部的重复节点，这加快了合并过程。如果零件实例仅在公共面处相交，你应该保持此默认设置。
      - **全部**。合并所选零件实例中的所有节点。默认情况下，Abaqus/CAE删除具有与现有单元相同连接性的单元。关闭**Remove duplicate elements**以保留重复单元。
      - **无**。将零件实例合并为单个零件实例，但保留原始节点。如果适用，输入**Node merging tolerance**。Abaqus/CAE删除距离小于指定容差的节点，并用单个新节点替换它们。新节点的位置是合并到新节点的节点组的平均位置。
   2. 点击**Continue**。
   3. 选择要合并的零件实例。你可以使用**[Ctrl]****+Click**和**[Shift]****+Click**的组合来选择多个零件实例。你也可以点击提示区域右侧的**Instance List**按钮并从出现的**Instance List**对话框中选择实例。
      **提示：**如果你无法选择所需的零件实例，可以使用**Selection**工具栏更改选择行为。更多信息，请参见["使用选择选项，"第6.3节](pt01ch06s03.md)。零件实例不必接触或重叠。
   4. 点击鼠标按钮2表示已完成选择零件实例。
   5. 如果你正在合并网格并且你输入的**Node merging tolerance**值太大，Abaqus/CAE可能会检测到来自同一单元的重复节点。Abaqus/CAE不会合并来自同一单元的节点，但太大的容差可能导致网格变形。如果**Node merging tolerance**太大，Abaqus/CAE会询问你是否要继续合并零件实例。
      - 点击**是**继续。
      - 点击**否**取消合并过程。
   6. Abaqus/CAE以品红色高亮显示将被合并的节点，并询问你是否继续。
      - 点击**是**继续。
      - 点击**否**取消合并过程。Abaqus/CAE合并距离小于指定容差的节点，并用单个新节点替换它们。新节点的位置是合并到新节点的节点组的平均位置。如果没有节点距离小于指定容差，Abaqus/CAE会询问你是取消过程还是将选定的实例合并为单个零件实例。Abaqus/CAE合并选定的实例，创建新的零件实例和新的零件，并修改集合和曲面以包含新的零件实例。
6. 如果你选择了**Cut**操作，请执行以下操作：
   1. 点击**Continue**。
   2. 选择要被切割的零件实例的几何。你只能选择一个零件实例，并且只选择几何，即使零件包含孤立网格节点和单元。
   3. 选择将进行切割的零件实例。切割几何必须接触或与被切割零件实例的几何重叠。
   4. 点击鼠标按钮2表示已完成选择零件实例。Abaqus/CAE切割选定的实例并创建新的零件实例和新的零件。被切割零件上的任何孤立网格都会复制到新实例和新零件中。
