# 16.11.6 定义流体热能场





您可以创建流体热能场来定义Abaqus/CFD分析中流体温度的初始值。有关更多信息，请参见[第30章，"流体动力学分析"](pt04ch30.md)。"

**创建或编辑流体热能场的步骤：**

1. 使用以下方法之一显示流体热能场编辑器： - 要创建新的流体热能场，请按照["创建预定义场"章节第16.8.3节](pt03ch16s08hlb03.md)中概述的程序进行操作（**类别**：`流体`；**所选步骤的类型**：`流体热能`）。 - 要使用菜单或管理器编辑现有的流体热能场，请参见["编辑依赖于步骤的对象"章节第3.4.13节](pt01ch03s04hlb05.md)。**注意：**您只能在初始步骤中创建或编辑流体热能场。有关更多信息，请参见[第30章，"流体动力学分析"](pt04ch30.md)。
2. 默认情况下，流体热能场应用于整个模型。如果要更改流体热能场适用的区域，请执行以下操作： 1. 单击![](../graphics/ico_selectBlue.png)。 2. 从视口中选择一个或多个区域或集合。 3. 在提示区域中单击**完成**。Abaqus/CAE更新流体热能场编辑器以反映您的选择。
3. 单击**分布**字段右侧的箭头，然后从出现的列表中选择所需的选项： - 选择**均匀**来定义在该区域上均匀的初始流体温度。 - 选择标记为(A)的解析场或标记为(D)的离散场来定义空间变化的初始流体温度。选择列表包含所有解析场和仅对温度场有效的离散场。或者，您可以单击![](../graphics/ico_analyticalField.png)来创建新的解析场。（有关更多信息，请参见[第58章，"解析场工具集"](pt06ch58.md)。）
4. 在**温度**字段中，输入流体温度的初始值。
5. 单击**确定**保存数据并退出编辑器。
