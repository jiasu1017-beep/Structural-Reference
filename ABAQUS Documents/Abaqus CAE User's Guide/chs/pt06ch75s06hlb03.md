# 75.6.3 合并边





您可以选择 ****Tools**![](../graphics/images/arrow.gif)**Virtual Topology**![](../graphics/images/arrow.gif)**Combine Edges**** 从主菜单栏来合并选定的边。在网格生成过程中，Abaqus/CAE 将合并的边视为单条边，并忽略被移除的所有顶点。

**合并边的步骤：**

1. 从上下文栏的 **Object** 字段中，选择一个零件或装配。
2. 从主菜单栏，选择 ****Tools**![](../graphics/images/arrow.gif)**Virtual Topology**![](../graphics/images/arrow.gif)**Combine Edges****。**提示：**您也可以使用位于 Mesh 模块工具箱中的 ![](../graphics/ico_virtTopoCombineEdges.png) 工具来合并边。
3. 选择要合并的边。您可以使用拖选、**[Shift]****+点击**、**[Ctrl]****+点击** 和角度方法的组合来选择多条要合并的边。有关更多信息，请参见 ["在当前视口中选择对象，" 第6.2节](pt01ch06s02.md)。**提示：**如果您无法选择所需的边，可以使用 **Selection** 工具栏更改选择行为。有关更多信息，请参见 ["使用选择选项，" 第6.3节](pt01ch06s03.md)。Abaqus/CAE 合并选定的边。您可以使用模型树或 **Restore Entities** 工具来恢复选定的边。有关恢复单个实体的更多信息，请参见 ["恢复被虚拟拓扑替换的实体，" 第75.6.5节](pt06ch75s06hlb05.md)。
