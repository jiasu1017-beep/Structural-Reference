# *DIELECTRIC





### *DIELECTRIC指定介电材料属性。

此选项用于为完全约束材料定义介电属性，以用于耦合压电分析。

**产品：**Abaqus/Standard  Abaqus/CAE  

**类型：**模型数据  

**级别：**模型  

**Abaqus/CAE：**属性模块

##### **参考：**

- ["压电行为，" Abaqus分析用户指南第26.5.2节](../usb/usb-link.md#usb-mat-cpiezoelect)

### **可选参数：**

DEPENDENCIES

将此参数设置为介电属性定义中包含的场变量数量。如果省略此参数，则假定介电属性不依赖于任何场变量，但可能仍取决于温度。请参见Abaqus分析用户指南第21.1.2节中的["指定场变量依赖性"](../usb/usb-link.md#usb-mat-cmaterialdata-fvdepen)，了解更多详细信息。

TYPE

设置 TYPE=ISO（默认）以获得各向同性行为。设置 TYPE=ORTHO 以获得正交各向异性行为。设置 TYPE=ANISO 以获得完全各向异性行为。
