# 62.6.5 通过输入边参数创建datum点







您可以通过在模型上选择一条边并输入表示边长度百分比的参数值来定位datum点。沿边的箭头表示参数值从起点（对应于边参数值为零）到终点（对应于值为一）增加的方向。

![](../graphics/dtm-hpted2-nls.png)

**通过输入边参数创建datum点：**

1. 从主菜单栏中，选择****Tools**![](../graphics/images/arrow.gif)**Datum****。此时将出现**Create Datum**对话框。该对话框概述了您可以创建的datum几何体类型。Abaqus/CAE在提示区域中显示提示以指导您完成此过程。**提示：**您也可以使用模块工具箱中的![](../graphics/ico_dtmPtParameter.png)工具创建datum点。有关工具箱中datum工具的图示，请参见["使用Datum工具集，" 第62.2节](pt06ch62s02.md)。
2. 从对话框顶部的类型列表中，选择**Point**。**Method**列表指示您可以使用的方法来创建datum点。
3. 从**Method**列表中，选择**Enter parameter**。
4. 从当前视口中的零件或装配中，选择将定位datum点的边。选定的边上会出现一个箭头，指示参数值增加的方向。
5. 在提示区域的文本框中，输入边参数的值。datum点会出现。您可以通过从主菜单栏中选择****Feature**![](../graphics/images/arrow.gif)**Edit****并修改边参数的值来修改datum点的位置。
