# 6.3.6 选择拖动选择区域选择哪些对象





选择工具允许你选择拖动选择区域选择哪些对象。从**选择**工具栏中，选择以下之一：

**内部**

![](../graphics/ico_selectInside.png) 仅选择落在拖动选择区域内部的对象。

**内部和交叉**

![](../graphics/ico_selectInsideCrossing.png) 仅选择落在拖动选择区域内部或与其交叉的对象。此工具默认是选中的。

**交叉**

![](../graphics/ico_selectCrossing.png) 仅选择与拖动选择区域交叉的对象。

**外部和交叉**

![](../graphics/ico_selectOutsideCrossing.png) 仅选择落在拖动选择区域外部或与其交叉的对象。

**外部**

![](../graphics/ico_selectOutside.png) 仅选择落在拖动选择区域外部的对象。
