# 16.9.29 定义体积电流密度





您可以创建体积电流密度载荷，以在涡流分析中在体积上定义电流密度。体积电流密度载荷仅在电磁模型中可用。

**创建或编辑体积电流密度：**

1. 使用以下方法之一显示体积电流密度载荷编辑器：- 若要创建新的体积电流密度载荷，请遵循"创建载荷"，第16.8.1节中概述的过程（**类别**：`Electrical/Magnetic`；**所选步骤的类型**：`Body current density`）。- 若要使用菜单或管理器编辑现有的体积电流密度载荷，请参见"编辑依赖于步骤的对象"，第3.4.13节。若要编辑应用载荷的区域，请参见"编辑应用规定条件的区域"，第16.8.4节。
2. 单击**分布**字段右侧的箭头，并从出现的列表中选择您的选项：- 选择**均匀**以定义在体积上均匀的载荷。- 选择**用户定义**以在用户子程序[`UDECURRENT`](../sub/sub-link.md#sub-xsl-udecurrent)中定义载荷的幅值和方向。请参见以下章节获取更多信息：- "指定常规作业设置"，第19.8.6节- "Abaqus User Subroutines Reference Guide"第1.1.24节"UDECURRENT"
3. 如果您选择了**均匀**分布选项，请执行以下步骤：1. 在**分量1**、**分量2**和（如果适用）**分量3**字段中，输入体积电流密度向量的实部（同相）和虚部（异相）。Abaqus/CAE计算体积电流密度向量的大小和方向。2. 如果需要，单击**幅值**字段右侧的箭头，并从出现的列表中选择您选择的幅值。或者，您可以单击![](../graphics/ico_amplitudeSmall.png)创建新的幅值。（更多信息，请参见第57章"幅值工具集"。）3. 单击**确定**保存您的数据并退出编辑器。
4. 如果您选择了**用户定义**分布选项，请执行以下步骤：1. 如果需要，在**分量1**、**分量2**和（如果适用）**分量3**字段中输入体积电流密度向量的实部（同相）和虚部（异相）。Abaqus/CAE计算体积电流密度向量的大小和方向，并将该信息传递到用户子程序中。2. 单击**确定**保存您的数据并退出编辑器。3. 进入作业模块并显示感兴趣的分析作业的作业编辑器。（更多信息，请参见"创建、编辑和操作作业"，第19.7节。）4. 在作业编辑器中，单击**常规**选项卡，并指定包含定义载荷幅值和方向的用户子程序的文件。更多信息，请参见"指定常规作业设置"，第19.8.6节。**注意：**您只能在作业编辑器中指定一个用户子程序文件；如果您的分析涉及多个用户子程序，您必须将用户子程序合并到一个文件中，然后指定该文件。
