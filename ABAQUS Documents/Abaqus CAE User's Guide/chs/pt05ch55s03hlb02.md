# 55.3.2 定义模型特征边缘





公共和叠加曲线图选项决定所有曲线图中出现哪些模型边缘。当您指定仅可见特征边缘时,Abaqus/CAE确定哪些边缘符合此标准。特征边缘位于法线相差超过"特征角度"的单元之间。您可以自定义特征角度。较大的角度会减少特征边缘的数量;相反,较小的角度会导致更多边缘可见。默认是20。特征角度的设置适用于所有曲线图状态。但是,对于分析刚性表面,无论特征角度设置如何,都会绘制所有分段边缘。

在[图55-3](pt05ch55s03hlb02.md#usv-featangle)中,左侧图在特征角度设置为0时显示特征边缘,中间图在同一模型上特征角度设置为15时显示特征边缘,右侧图特征角度设置为30。

**图55-3** 特征角度为0、15和30的曲线图。

![](../graphics/usv-featangle.png)

**自定义特征角度的步骤:**

1. 找到"**Feature Angle**"选项。从主菜单栏,选择 ****View**![](../graphics/images/arrow.gif)**ODB Display Options****。在出现的对话框中点击"**General**"选项卡。"**Feature Angle**"选项在"**General**"页面底部。
2. 拖动"**Feature Angle**"滑块到您想要的特征角度。
3. 点击"**Apply**"实施您的更改。所有曲线图状态的特征边缘都会根据您的特征角度规格进行更改。默认情况下,您的特征角度规格会在会话期间保存。如果要为后续会话保留此更改,请将它们保存到文件。更多信息,请参阅 ["为后续会话保存自定义," 第55.1.1节](pt05ch55s01s01.md)。
