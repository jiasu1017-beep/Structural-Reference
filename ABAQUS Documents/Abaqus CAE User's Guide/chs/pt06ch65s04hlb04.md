# 65.4.4 恢复被抑制的特征







选择 ****Feature**![](../graphics/images/arrow.gif)**Resume****（特征→恢复）从主菜单栏恢复被抑制的特征。您还可以通过在模型树中选择特征，点击鼠标按钮 3，然后从出现的菜单中选择 **Resume**（恢复）来恢复一组特征。

恢复特征会将其完全恢复到部件或装配中。您可以恢复上次抑制的特征、所有被抑制的特征或仅选定的特征。当您恢复子特征时，Abaqus/CAE 也会自动恢复父特征。但是，当您恢复父特征时，其子特征不会被恢复，除非您选择它们。为了帮助您选择特征的子特征，您可以在模型树中的特征上点击鼠标按钮 3，并从出现的菜单中选择 **Show children**（显示子特征）。

**恢复被抑制的特征：**

1. 从主菜单栏中选择 ****Feature**![](../graphics/images/arrow.gif)**Resume****（特征→恢复）。**提示：**您也可以使用模块工具箱中位于的 ![](../graphics/ico_featureResume.png) 工具来恢复特征。
2. 执行以下操作之一： - 点击 **Last Set**（上一组）恢复最近抑制的特征及其所有父特征。 - 点击 **All**（全部）恢复所有被抑制的特征。 - 从模型树中选择要恢复的特征。您可以选择多个要恢复的特征。恢复的特征重新出现在视口中。
