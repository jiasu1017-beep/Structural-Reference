# 48.3.3 控制数据对Y值





### 48.3.3 控制数据对Y值

Abaqus以X–Y数据对的形式获取沿路径的结果。数据对X值定义要获取数据的模型位置；Y值基于那些位置的分析结果。您可以通过选择Abaqus获取结果的结果步骤、帧和场输出变量来控制这些Y值；通过控制Abaqus平均某些类型结果的方式；以及通过选择复数结果（如适用）的显示形式。

**控制数据对Y值：**

1. 找到"XY数据来自路径"选项。从主菜单栏中，选择****工具**![](../graphics/images/arrow.gif)**XY数据**![](../graphics/images/arrow.gif)**创建****。从出现的对话框中，选择"路径"；然后单击"继续"。"XY数据来自路径"对话框出现。
2. 在对话框顶部，单击"路径"箭头选择要获取数据的路径。当前视口中的模型图会变为高亮显示您选择的路径。
3. "Y值"字段标识Abaqus将获取结果的步骤、帧和场输出变量。- 要更改步骤或帧，请单击"步骤/帧"。有关详细信息，请参阅["选择结果步骤和帧，"第42.3节](pt05ch42s01.md)。- 要更改场输出变量，请单击"场输出"。有关详细信息，请参阅["选择主场输出变量，"第42.5.3节](pt05ch42s03hlb03.md)。- 基于单元的场输出值的平均由当前视口的结果选项设置控制。要查看或更改这些设置，请从主菜单栏中选择****结果**![](../graphics/images/arrow.gif)**选项****。有关详细信息，请参阅["控制结果平均，"第42.6.6节](pt05ch42s04hlb02.md)。- 复数的值的格式由当前视口的结果选项设置控制。要查看或更改这些设置，请从主菜单栏中选择****结果**![](../graphics/images/arrow.gif)**选项****。有关详细信息，请参阅["控制复数结果的形式，"第42.6.9节](pt05ch42s04hlb05.md)。Abaqus根据您对步骤、帧、场输出变量或结果选项的任何更改来刷新"Y值"字段和当前视口中的变形、等值线或符号图。
4. 要评估和显示数据，请单击"绘图"。X–Y曲线图出现在当前视口中。该图表示您在对话框中配置的数据，Abaqus将其视为临时数据，无论您是否单击"另存为"保存它。
5. 要将您配置的X–Y数据对保存为X–Y数据对象，请单击"另存为"。**注意：**要绘制您保存的X–Y数据对象，请从主菜单栏中选择****工具**![](../graphics/images/arrow.gif)**XY数据**![](../graphics/images/arrow.gif)**绘图****，然后从下拉菜单中选择X–Y数据对象。
6. 完成后，单击"取消"关闭对话框。
