# 58.3.3 从输出数据库文件进行网格到网格映射





当数据源是 Abaqus 输出数据库（`.odb`）文件时，场输出变量值必须显示在视口中。这种技术称为网格到网格映射。场输出变量值可以位于节点、单元形心或积分点。在这种映射中，源数据的几何区域是网格连接数据，例如单元面或单元集。

要选择您想要使用的精确源数据，请打开所需的输出数据库文件，并在 Visualization 模块中显示未变形的等值线图。您也可以使用变形图，但数据仍从未变形网格映射。选择您要映射的场输出变量，以及分析的值对应的步骤和帧。当您在模型数据库（`.cae`）文件中的目标模型中创建新的映射场时，您可以将显示的视口指定为数据源。视口的当前状态被视为要映射到目标模型上的一组值的快照；即，在分析的特定步骤和帧显示的特定场输出变量。

例如，您可以在以下工作流程中使用网格到网格映射：

1. 在 Abaqus 中设置并运行热分析，以生成包含节点温度（输出变量 NT）的输出数据库。
2. 在 Visualization 模块中打开输出数据库（`.odb`）文件，并在分析的 Step-3、Frame 0 处显示输出变量 NT。
3. 打开包含主目标模型的模型数据库（`.cae`）文件。在 Load 模块的主菜单栏中选择 ****Tools**![](../graphics/images/arrow.gif)**Analytical Field**![](../graphics/images/arrow.gif)**Create****，以从节点温度值创建映射场。在**创建映射场**对话框中，选择包含输出数据库的视口作为源数据。
4. 对主模型进行网格划分（或重新划分网格）。
5. 创建温度预定义场，并选择映射分析场来定义温度分布。Abaqus/CAE 将源数据点及其相关温度映射到目标模型中的点。
6. 设置并运行后续分析。

输出数据库网格上的源数据可以位于节点、积分点或整个单元上。支持不同网格。

所选视口中的当前设置用于映射。这些设置如下：
- 主场输出变量
- 步骤/增量
- **结果选项**对话框中选择的平均选项
- 截面点（顶部或底部）
- 等等。

以下视口设置不支持网格到网格映射：
- 显示组
- 变换
- **结果选项**对话框中按单元集或显示组进行平均
- 单元面数据
- 当前会话中的用户数据（但保存到输出数据库文件中的数据受支持）

当您从输出数据库数据创建映射场并关闭**创建映射场**对话框时，源数据将保存到映射场对象，但视口本身不会被保存。在创建映射场后在视口中进行的任何更改都不会反映在映射数据中。使用网格到网格映射创建映射场后，您无法编辑源数据网格规范；您只能编辑现有映射场以更改默认场值或映射器控制选项。

体积网格到网格映射只能与节点温度和材料密度一起执行。当映射目标是基于网格的节点区域时，您必须指定目标是表面区域还是体积区域。Abaqus/CAE 对表面与体积区域使用不同的映射算法。
