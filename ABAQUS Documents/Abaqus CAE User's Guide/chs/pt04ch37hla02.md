# 37.4 编辑连接两点的弹簧和阻尼器





您可以编辑连接两点的弹簧和阻阻器，以添加更多点对、删除点对或编辑现有点对的点。

**要编辑连接两点的弹簧和阻阻器：**

1. 从Property模块或Interaction模块的主菜单栏中，选择****Special**![](../graphics/images/arrow.gif)**Springs/Dashpots**![](../graphics/images/arrow.gif)**Edit**![](../graphics/images/arrow.gif)***名称*****以显示**Edit Springs/Dashpots**对话框。视口中高亮显示了表示弹簧/阻阻器的符号。
2. 从**Spring/Dashpot Point Pairs**表中，您可以执行以下操作：
   - 要使用相同行为定义其他弹簧/阻阻器，单击![](../graphics/ico_add.png)并重复前述步骤中描述的点选择过程，以定义每个弹簧/阻阻器的第一个和第二个点。
   - 要编辑表中的点，选择表中的点，双击它或单击![](../graphics/ico_edit.png)，然后重新选择一个点。视口中的选择高亮显示会更新以显示新编辑的点。
   - 要在视口中识别特定的弹簧/阻阻器，选择所需行号。连接所选点对的突出虚线在视口中显得更粗。
   - 要从表中移除弹簧/阻阻器，选择所需行号（**Shift**+单击选择多行）并单击![](../graphics/ico_delete.png)。**Ctrl**+单击行号以取消选择。
   - 要从表中移除所有弹簧/阻阻器，单击![](../graphics/ico_error.png)。
3. 根据需要完成弹簧/阻阻器定义的编辑，然后单击**OK**。视口中表示弹簧/阻阻器的符号出现在新编辑的点上。
