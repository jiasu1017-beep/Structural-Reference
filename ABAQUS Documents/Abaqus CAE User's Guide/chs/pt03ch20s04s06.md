# 20.4.6 在 Sketcher 中使用链方法选择边







在草图中偏移或复制边时，您可以选择"链"连接的边。链是一组连接的边，其中每个边在每个端点最多连接另一个边——就像链条中的链环。链方法仅在 Sketcher 中可用，它允许您将链边作为单个实体进行选择。

当您执行允许从草图中选择多条边的任务时，Abaqus/CAE 会在提示区域显示一个字段。该字段允许您在两种选择方法之间进行选择——**单独**和**按链**，如图 20-5](pt03ch20s04s06.md#ske-edgechain)所示。

**图 20-5** 从提示区域的字段中选择选择方法。

![](../graphics/ske-edgechain-nls.png)

使用链方法后，您可以在提示区域点击**单独**方法，然后**[Shift]+点击**各个边将其追加到您的选择中。您也可以**[Ctrl]+点击**边来取消选择它们。此外，您可以继续使用链方法并使用**[Shift]+点击**将更多链追加到您的选择中。
