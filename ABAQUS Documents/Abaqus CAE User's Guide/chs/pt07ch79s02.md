# 79.2 生成和修改叠加绘图





本节说明如何通过创建层并在同一视口中绘制它们来叠加多个绘图，以及如何在创建后修改叠加绘图。叠加绘图中的每个层都是完全独立的。您可以更改输出数据库、绘图状态（或多个绘图状态）、相对于彼此重新排序层、操作每个单独层的视图，以及更改变应用于内容的各种显示选项。涵盖以下主题：
- ["生成叠加绘图，"第79.2.1节](pt07ch79s02hlb01.md)
- ["重新排序叠加绘图中的层，"第79.2.2节](pt07ch79s02hlb02.md)
- ["操作叠加绘图的视图，"第79.2.3节](pt07ch79s02hlb03.md)
- ["编辑叠加绘图中的层，"第79.2.4节](pt07ch79s02hlb04.md)
