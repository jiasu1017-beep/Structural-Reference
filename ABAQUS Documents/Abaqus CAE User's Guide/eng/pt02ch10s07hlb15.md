# 10.7.15 Importing an assembly from a CATIA V4-format file







Select ****File**![](../graphics/images/arrow.gif)**Import**![](../graphics/images/arrow.gif)**Assembly**** from the main menu bar to import an assembly from a CATIA V4-format file directly into Abaqus/CAE. For information about the versions of CATIA V4 supported by Abaqus/CAE, see the Dassault Systmes Knowledge Base at [www.3ds.com/support/knowledge-base](http://www.3ds.com/support/knowledge-base).

**To import an assembly from a CATIA-format file:**

1. From the main menu bar, select ****File**![](../graphics/images/arrow.gif)**Import**![](../graphics/images/arrow.gif)**Assembly****. The **Import Assembly** dialog box appears.
2. From the **File Filter** menu at the bottom of the **Import Assembly** dialog box, select **CATIA V4 (*.model*, *.catdata*, *.exp*)**. Abaqus/CAE lists all the files in the selected directory with a `.model*`, `.catdata*`, or `.exp*` file extension.
3. Select the CATIA file containing the assembly to import, and click **OK**. Abaqus/CAE displays the **Import Assembly from CATIA File** dialog box.
4. Do either of the following: - Click on the assembly name to import all of the part instances in the assembly. - Click on the arrow to the left of the assembly name to display all of the part instances in the assembly. Click on the names of the part instances that you want to import.
5. Click **OK** to import the assembly or selected part instances and to exit the **Import Assembly from CATIA File** dialog box. Abaqus/CAE displays a message indicating if any of the part instances are invalid or imprecise. If necessary, use the Query toolset to locate areas of invalid and imprecise geometry. For more information, see ["Using the Query toolset in the Part module," Section 11.16.4](pt03ch11s16s04.md).
