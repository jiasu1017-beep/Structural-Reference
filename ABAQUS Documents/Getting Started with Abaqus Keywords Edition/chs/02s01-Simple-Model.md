# 2.1 Abaqus分析模型的组成部分

An Abaqus模型由多个不同的组件组成，这些组件共同描述了要分析的物理问题以及要获得的结果。分析模型至少包含以下信息：离散几何体、单元截面属性、材料数据、载荷和边界条件、分析类型以及输出请求。本章的讨论主要针对结构应用。流体动力学应用也适用类似的概念。

**离散几何体**

有限元和节点定义了Abaqus中正在建模的物理结构的基本几何形状。模型中的每个单元代表物理结构的一个离散部分，而物理结构又由许多相互连接的单元表示。单元通过共享的节点相互连接。节点的坐标以及单元的连通性——即哪些节点属于哪些单元——构成了模型几何体。模型中所有单元和节点的集合称为*网格*。通常，网格只是对结构实际几何形状的近似。

单元类型、形状和位置，以及网格中使用的单元总数，都会影响仿真获得的结果。网格密度越大（即网格中的单元数量越多），结果就越准确。随着网格密度的增加，分析结果会收敛于一个唯一解，同时分析所需的计算机时间也会增加。从数值模型获得的结果通常是对被模拟的物理问题解的近似。模型几何形状、材料行为、边界条件和载荷中所做的近似程度决定了数值模拟与物理问题的匹配程度。

**单元截面属性**

Abaqus拥有广泛的单元库，其中许多单元的几何形状不能完全由节点坐标定义。例如，复合壳的层或I型梁截面的尺寸不是由单元节点定义的。这类额外的几何数据被定义为单元的物理属性，对于完全定义模型几何形状是必要的（参见[第3章，"有限元和刚体"](ch03.html)）。

**材料数据**

必须为所有单元指定材料属性。虽然高质量的材料数据往往难以获得，特别是对于更复杂的材料模型，但Abaqus结果的有效性受材料数据的准确性和完整性的限制。

**载荷和边界条件**

载荷使物理结构变形，从而在其中产生应力。最常见的载荷形式包括：

- 点载荷；
- 表面上的压力载荷；
- 表面上的分布牵引力；
- 壳边缘上的分布边缘载荷和弯矩；
- 体积力，如重力；以及
- 热载荷。

边界条件用于约束模型的某些部分保持固定（零位移）或按指定量移动（非零位移）。

在静力分析中，必须使用足够的边界条件来防止模型沿任何方向作为刚体移动；否则，未约束的刚体运动会导致刚度矩阵奇异。在求解阶段会出现求解器问题，并可能导致模拟过早停止。Abaqus/Standard如果在模拟过程中检测到求解器问题，会发出警告消息。学习解释此类错误消息非常重要。如果您在静力应力分析期间看到"数值奇异"或"零主元"警告消息，您应该检查您的模型整体或部分是否缺乏对刚体平移或旋转的约束。刚体运动可以由部件的平移和旋转组成。潜在的刚体运动取决于模型的维度。

| 维度 | 可能的刚体运动 |
|------|---------------|
| 三维 | 沿1、2和3方向的平移。 |
|      | 绕1、2和3轴的旋转。 |
| 轴对称 | 沿2方向的平移。 |
|        | 绕3轴的旋转（仅限轴对称刚体）。 |
| 平面应力 | 沿1和2方向的平移。 |
| 平面应变 | 绕3轴的旋转。 |

默认情况下，1、2和3方向与全局笛卡尔坐标系的轴对齐（稍后讨论）。

在动力分析中，惯性力可以防止模型瞬时经历无限运动，只要模型中的所有独立部件都有一定质量；因此，动力分析中的求解器问题警告通常表示其他建模问题，例如过大的塑性变形。

**分析类型**

Abaqus可以执行许多不同类型的仿真，但本指南仅涵盖两种最常见的类型：静力和动力应力分析。

在静力分析中获得结构对施加载荷的长期响应。在其他情况下，可能需要关注结构对载荷的动力响应：例如，突加载荷对部件的影响，如碰撞过程中发生的情况，或建筑物在地震中的响应。

**输出请求**

Abaqus仿真可以生成大量输出。为了避免使用过多磁盘空间，您可以将输出限制为解释结果所需的内容。
