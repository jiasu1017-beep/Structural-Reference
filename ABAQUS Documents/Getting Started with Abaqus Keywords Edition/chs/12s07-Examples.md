# 12.7 Abaqus/Standard中的通用接触

![](../images/blu4rule.gif)

在通道成型示例中（见"12.6 Abaqus/Standard二维示例：成型通道"第12.6节），接触相互作用使用接触对算法定义，这需要您明确指定可能接触的表面。作为替代方案，您可以在Abaqus/Standard分析中使用通用接触算法来指定接触。通用接触的接触相互作用域、接触属性和表面属性是独立指定的，这提供了一种更灵活的方式来逐步向模型添加细节。用于指定通用接触的简单界面允许高度自动化的接触定义；但是，也可以使用通用接触界面定义接触来模拟传统接触对。相反，如果使用表面到表面公式，使用接触对用户界面指定跨多个体的表面自接触会模拟通常用于通用接触的高度自动化方法。

在Abaqus/Standard中，传统的成对接触相互作用规范通常会比全包含自接触方法定义接触产生更高效的分析。因此，在接触定义的便捷性和分析性能之间通常存在权衡。Abaqus/CAE提供了接触检测工具，极大地简化了为Abaqus/Standard创建传统接触对的过程（见《Abaqus/CAE用户手册》第15.6节"理解接触和约束检测"）。
