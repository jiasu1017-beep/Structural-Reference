# 19.5.5 优化过程编辑器





您可以使用优化过程编辑器选择将包含在优化过程中的 Abaqus/CAE 模型和优化任务。您还可以配置优化设置，例如最大优化迭代次数。

您可以通过从主菜单栏选择 ****优化过程**![](../graphics/images/arrow.gif)**创建**** 或 ****优化过程**![](../graphics/images/arrow.gif)**编辑**![](../graphics/images/arrow.gif)***优化过程名称***** 来显示优化过程编辑器。（您也可以在**优化过程管理器**中单击**创建**或**编辑**。）

优化过程编辑器允许您配置以下内容：

**优化**

使用**优化**选项卡页面指定在过程终止前应运行的最大优化迭代次数以及数据保存的频率。更多信息，请参见[「创建和编辑优化过程，」第 19.12.1 节](pt03ch19s12hlb01.md)。

**提交**

使用**提交**选项卡页面配置优化过程的提交属性，例如提交时间以及是否将过程提交到由本地 Abaqus 环境文件或系统管理员配置的远程队列。更多信息，请参见[「配置作业提交属性，」第 19.8.2 节](pt03ch19s08hlb02.md)。

****内存****

使用**内存**选项卡页面配置优化过程期间分配给 Abaqus 作业的内存量。更多信息，请参见[「控制作业内存设置，」第 19.8.7 节](pt03ch19s08hlb07.md)。

****并行化****

使用**并行化**选项卡页面配置优化过程期间 Abaqus 分析作业的并行执行，例如要使用的处理器数量和并行化方法。更多信息，请参见[「控制作业并行执行，」第 19.8.8 节](pt03ch19s08s01.md)。
