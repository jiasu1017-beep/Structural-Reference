# 49.4.4 显示动画帧计数器





对于基于对象的动画，您可以显示或抑制动画帧计数信息，该信息显示在视口右上角，由以下部分组成：
- 比例因子动画中每个图像的比例因子，
- 谐波动画中每个图像的相位角，或
- 时间历史动画中每个图像包含的步骤和帧（步骤内的绘图编号）。

除了完全关闭它之外，您无法自定义计数器的内容或外观。

与每个动画步骤和帧关联的步骤和增量由状态块给出。比例因子动画中的所有图像都基于单个步骤和帧。有关自定义状态块的更多信息，请参阅["自定义状态块，"第56.3节](pt05ch56s03.md)。

**显示动画帧计数器：**

1. 找到"显示帧计数器"选项。在提示区域中，单击"动画选项"；然后在出现的对话框中单击"播放器"选项卡。"显示帧计数器"选项位于页面底部附近。
2. 切换"显示帧计数器"以显示或抑制动画帧计数信息。
3. 单击"应用"以实现您的更改。所有动画化视口中的动画根据您的规格进行更改。当"显示帧计数器"打开时，Abaqus/CAE在所有动画化视口的右上角显示状态信息。此信息为动画序列中的每个图像更新。您可能需要[停止](pt05ch49s04hlb01.md)播放才能读取它。您的更改在会话期间保存，并将影响此状态中的所有后续绘图。
