# 3.5.3 在模型树和结果树中使用弹出菜单







模型树和结果树的强大功能很大程度上来自在项目上点击鼠标按钮 3 时出现的弹出菜单。例如，[图 3-27](pt01ch03s05s03.md#uss-int-tree-shortcut-material) 显示了点击模型树中**零件**容器上的鼠标按钮 3 的效果。

**图 3-27** 在模型树的**零件**容器上点击鼠标按钮 3。

![](../graphics/uss-int-tree-shortcut-material-nls.png)

在[图 3-27](pt01ch03s05s03.md#uss-int-tree-shortcut-material)中，**创建**菜单项以粗体显示，因为它是默认操作。双击项目或选择项目并按 **[Enter]** 会调用默认操作。在大多数情况下，如果项目是容器，则默认操作是在容器中创建一个新项目。类似地，如果项目不是容器，则默认操作是编辑项目。例如，如果双击**零件**容器，Abaqus/CAE 会显示**创建零件**对话框并允许您创建新零件；如果双击容器内的某个零件，Abaqus/CAE 会显示**编辑零件**对话框并允许您编辑所选零件。

弹出菜单中的一些命令出现在模型树和结果树的所有项目中；其他命令仅出现在特定项目或两个树之一的项目中。例如，**切换上下文**命令出现在模型树和结果树的所有项目中，包括容器。以下命令出现在模型树和结果树的所有容器中：

**切换上下文**

如果选择**切换上下文**，Abaqus/CAE 会将项目设为当前项目。在模型树中，在适当时，Abaqus/CAE 也会切换到可以编辑项目的模块。例如，如果在**材料**容器上点击鼠标按钮 3 并选择**切换上下文**，Abaqus/CAE 会切换到 Property 模块。有关更多信息，请参阅[「什么是模块？，第 2.3 节」](pt01ch02s03.md)。在结果树中，您也可以将上下文切换到另一个输出数据库中的项目。

选择容器并按 **[Ctrl]****+[Space]** 与从任一树中选择**切换上下文**具有相同的效果。

**过滤器**

当您选择**过滤器**时，Abaqus/CAE 会在容器名称旁边提示您输入一串字符。按 **[Enter]** 后，Abaqus/CAE 会过滤容器的内容，仅显示与指定字符串匹配的项目。过滤器区分大小写。有关有效过滤语法的详细信息，请点击模型树或结果树顶部的提示按钮![](../graphics/uss-int-modeltree-tip.png)。[图 3-28](pt01ch03s05s03.md#uss-int-tree-filter) 显示了过滤容器的效果。

**图 3-28** 在模型树中过滤容器。

![](../graphics/uss-int-tree-filter-nls.png)

被过滤器隐藏的项目不能从模型树或结果树中进行操作；但是，这些项目不会从模型或输出数据库中移除。当过滤器生效时，容器名称旁边的括号中的数字表示容器中可见项目的数量，后跟容器中的项目总数（可见和隐藏）（请参阅[图 3-28](pt01ch03s05s03.md#uss-int-tree-filter)）。过滤器字符串出现在这些数字的右侧。要移除过滤器，请为相应容器选择**过滤器**，删除过滤器字符串，然后按 **[Enter]**。

过滤器只能应用于单个容器，并且仅在会话期间存在。选择容器并按 **[F2]** 与从任一树中选择**过滤器**具有相同的效果。

**设为主项**

如果选择**设为主项**，Abaqus/CAE 会将容器移动到模型树或结果树上方的下拉菜单中，并显示所选容器下的所有内容。有关更多信息，请参阅[「更改模型的视图，第 3.5.4 节」](pt01ch03s05s04.md)。

**全部展开**

如果选择**全部展开**，Abaqus/CAE 会展开所选容器内的所有容器和项目。

**全部折叠**

如果选择**全部折叠**，Abaqus/CAE 会折叠所选容器内的所有容器和项目。

**分组子项**

当容器包含超过 30 个项目时，Abaqus/CAE 会自动将项目分组为每组 30 个。如果关闭**分组子项**选项，Abaqus/CAE 会移除分组，并在容器中的同一级别列出所有项目。

选择容器并按 **[Ctrl]****+[G]** 与从任一树中选择**分组子项**具有相同的效果。

许多弹出菜单命令仅出现在特定项目中。在模型树中，这些命令反映您可以使用该项目的管理器执行的操作；例如，创建、编辑、删除、重命名、抑制和恢复。在结果树中，一些弹出菜单提供布尔运算符，使您能够控制当前视口中项目的显示。这些布尔运算符与控制显示组的五个命令相同：替换、添加、移除、交集和任意。有关更多信息，请参阅[「理解显示组布尔运算，第 78.1.2 节」](pt07ch78s01s02.md)。

某些菜单命令特定于模型树或结果树中的一个容器；例如，在步骤上点击鼠标按钮 3 允许您切换 **Nlgeom** 设置，在作业上点击鼠标按钮 3 允许您提交作业进行分析，在历史输出变量上点击鼠标按钮 3 允许您将另一个变量添加到现有绘图中。当您熟悉模型树和结果树后，您会发现可以快速执行主菜单栏、模块工具箱和各种管理器中的大多数操作。
