# 78.2.2 用于创建或编辑显示组的选择方法





您可以使用以下选择方法来指定将包含在显示组中的项目：
- 要通过直接从视口中选择来指定几何图形（单元、面或边）、装配线或参考点，请从**项目**列表中选择它们。（对于基准、单元和节点，您还必须从**方法**列表中选择**从视口选择**。）Abaqus/CAE自动进入选择模式，提示区域显示**选择显示组的*项目***。有关在视口中选择项目的更多信息，请参见[第6章，"在视口中选择对象](pt01ch06.md)。"点击提示区域中的**完成**结束在视口中选择项目。点击对话框中的**编辑选择**、**添加选择**或**删除选择**可进一步修改您的视口选择。
- 要按编号指定单元或节点，请从**方法**列表中选择**单元标签**或**节点标签**。从**创建显示组**对话框右侧**零件实例**字段的列表中选择节点或单元所属的零件实例名称。在**标签**字段中输入用逗号分隔的单元或节点编号列表，或输入可选地后跟增量编号范围的编号；例如，1:10或1:10:2。
- 要在可视化模块中按类型指定单元，请从**方法**列表中选择**单元类型**。所选方法可用单元类型的列表出现在**创建显示组**对话框的右侧。从列表中选择一个或多个单元类型（有关更多信息，请参见["从列表和表中选择多个项目，"第3.2.11节](pt01ch03s02s11.md)）。
- 要在可视化模块中按材料或截面指定单元，请从**方法**列表中选择**材料分配**或**截面分配**。您的模型中可用材料或截面的列表出现在**创建显示组**对话框的右侧。从列表中选择一个或多个材料或截面（有关更多信息，请参见["从列表和表中选择多个项目，"第3.2.11节](pt01ch03s02s11.md)）。
- 要在可视化模块中按层合结构或层指定单元，请从**方法**列表中选择**复合层合结构**或**复合层**。您的模型中可用复合层合结构或层的列表出现在**创建显示组**对话框的右侧。从列表中选择一个或多个层合结构或层（有关更多信息，请参见["从列表和表中选择多个项目，"第3.2.11节](pt01ch03s02s11.md)）。
- 要在零件和装配相关模块中指定几何、单元或节点集，或在装配相关模块中指定曲面，请从**方法**列表中选择**集合名称**或**曲面名称**。您的模型中可用集合或曲面名称的列表出现在**创建显示组**对话框的右侧。从列表中选择一个或多个集合名称（有关更多信息，请参见["从列表和表中选择多个项目，"第3.2.11节](pt01ch03s02s11.md)）。
- 要在零件和装配相关模块中指定Abaqus/CAE创建的内部集合，或在装配相关模块中指定内部曲面，请从**项目**列表中选择**内部集合**或**内部曲面**。您的模型中可用集合或曲面名称的列表出现在**创建显示组**对话框的右侧。从列表中选择一个或多个集合名称（有关更多信息，请参见["从列表和表中选择多个项目，"第3.2.11节](pt01ch03s02s11.md)）。
- 要在可视化模块中指定单元、节点、曲面或Abaqus/CAE创建的内部集合，请从**方法**列表中选择**单元集合**、**节点集合**、**曲面集合**或**内部集合**。您的模型中可用集合名称的列表出现在**创建显示组**对话框的右侧。从列表中选择一个或多个集合名称（有关更多信息，请参见["从列表和表中选择多个项目，"第3.2.11节](pt01ch03s02s11.md)）。
- 要选择模型中指定类型的所有项目，请从**方法**列表中选择**所有单元**、**所有节点**或**所有曲面**。不需要进一步指定项目。
- 如果您从**项目**列表中选择了**零件实例**或**显示组**，则可用零件实例或显示组的列表出现在**创建显示组**对话框的右侧。从列表中选择一个或多个零件实例或显示组名称（有关更多信息，请参见["从列表和表中选择多个项目，"第3.2.11节](pt01ch03s02s11.md)）。**注意：**在装配相关模块中，您还可以使用**装配显示选项**对话框**实例**标签页中的可见性选项来显示模型的零件实例子集（请参见["控制实例可见性，"第76.14节](pt07ch76hla13.md)）。在**装配显示选项**对话框中关闭的零件实例无法使用显示组变得可见。
- 要在可视化模块中指定包含在给定值范围内的结果的单元、节点或曲面，请选择**结果值**。要考虑的输出变量显示在对话框右上角。要选择新的结果变量，请点击**场输出**；有关**场输出**对话框的更多信息，请参见["选择要显示的场输出，"第42.5节](pt05ch42s03.md)。从**类型**字段的筛选方法中进行选择。
   - 如果选择![](../graphics/dgtG_ResRangeInside.png)（内部）或![](../graphics/dgtG_ResRangeOutside.png)（外部），分别在**最小值**和**最大值**字段中输入结果范围的下限和上限值。
   - 如果选择![](../graphics/dgtG_ResRangeAbove.png)，请在**最小值**字段中输入结果应高于的值。
   - 如果选择![](../graphics/dgtG_ResRangeBelow.png)，请在**最大值**字段中输入结果应低于的值。
无论当前活动显示组如何，模型中的每个单元（或节点或曲面）都用于筛选过程。如果您为不存在的单元（或节点或曲面）的值范围选择值，您将创建一个空的显示组。使用[状态场输出变量](pt05ch42s03hlb06.md)（而不是基于结果的显示组功能）可以在时间历史动画中获得更好的性能。状态场输出变量允许您从模型绘图中移除满足指定基于结果的失败标准的单元；请参见["选择状态场输出变量，"第42.5.6节](pt05ch42s03hlb06.md)了解更多信息。**注意：**基于单元、节点或曲面输出变量进行筛选的边界始终基于节点处变量的值。因此，在将用户定义的边界与单元和曲面输出量进行比较之前，单元和曲面输出量会在节点处进行外推和平均。**结果选项**对话框中的平均设置决定了单元和曲面变量在节点处的计算方式。例如，考虑一个使用75%的默认平均阈值基于Mises应力筛选单元的案例。外推至节点后，根据此阈值对值进行平均。这种条件平均可能导致节点处基于节点所属各个单元的贡献而产生几个不同的Mises应力值。任何Mises应力贡献落在用户定义边界内的单元都包含在显示组中。
- 要在可视化模块中指定坐标系，请从**项目**列表中选择**坐标系**，然后选择以下之一：
   - 要指定作为输出数据库一部分的坐标系，请从**方法**列表中选择**Odb系统**。输出数据库坐标系的列表出现在**创建显示组**对话框的右侧。
   - 要指定在会话期间创建的坐标系，请从**方法**列表中选择**用户系统**。用户定义坐标系的列表出现在**创建显示组**对话框的右侧。
   - 要指定输出数据库和会话中的坐标系，请从**方法**列表中选择**全部**。输出数据库坐标系和用户定义坐标系的列表出现在**创建显示组**对话框的右侧。从出现的列表中选择一个或多个坐标系（有关更多信息，请参见["从列表和表中选择多个项目，"第3.2.11节](pt01ch03s02s11.md)）。
- 要在可视化模块中指定分析约束，请从**项目**列表中选择以下约束类型之一：**绑定**、**壳到实体耦合**、**分布耦合**、**运动耦合**或**刚体**。
