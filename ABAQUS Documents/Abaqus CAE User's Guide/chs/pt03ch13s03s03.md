# 13.3.3 如何决定创建从属还是独立零件实例？







如果您的装配包含少量无关的零件实例，从属实例相对于独立实例几乎没有优势。每个零件都是不同的，您必须创建每个零件的实例。相比之下，如果您的装配包含相同的零件实例，您可以通过组装零件的从属实例来节省时间。当您随后对原始零件进行网格划分时，Abaqus/CAE会将该网格应用到装配中零件的每个从属实例。此外，从属实例消耗更少的内存资源，并生成更小的输入文件。

例如，图13-2说明独立和从属零件实例的装配。泵壳是独立零件实例，八个螺栓是从属零件实例。左图显示装配模块中的装配。右图显示网格模块中的装配。用户已经对表示螺栓的零件进行了网格划分，Abaqus/CAE将网格与螺栓的每个从属实例关联。

**图13-2** 装配和网格模块中的从属零件实例。

![](../graphics/asm-dependent-nls.png)

当您使用线性或径向阵列工具创建相同实例的阵列时，您会发现使用从属零件实例更方便。当您对原始零件进行网格划分时，Abaqus/CAE会将相同的网格应用到阵列中的每个从属实例。相比之下，如果您创建独立实例的阵列，则必须分别对每个实例进行网格划分。

Abaqus/CAE默认创建从属实例。除非您的装配只包含少量零件，否则建议您使用从属实例，因为可以节省内存并提高性能。
