# 9.1 适合使用 Abaqus/Explicit 分析的问题类型

在讨论显式动力学分析方法如何工作之前，了解哪些类型的问题非常适合使用 Abaqus/Explicit 是很有帮助的。在本指南中，我们整合了 Abaqus/Explicit 常用的一些问题的相关示例：

**高速动力事件**

显式动力学方法最初是为了分析高速动力事件而开发的，这类问题使用隐式程序（如 Abaqus/Standard）进行分析可能非常昂贵。例如，模拟短时爆炸载荷对钢板的影响见第10章"材料"。由于载荷施加得非常迅速且非常剧烈，结构的响应变化很快。准确追踪应力波在板中的传播对于捕获动力响应非常重要。由于应力波与系统的最高频率相关，获得精确的解需要很多小的时间增量。

**复杂接触问题**

使用显式动力学方法比使用隐式方法更容易建立接触条件。因此，Abaqus/Explicit 可以轻松分析涉及多个独立物体之间复杂接触相互作用的问题。Abaqus/Explicit 特别适合分析承受冲击载荷然后在结构内部发生复杂接触相互作用的结构的瞬态动力响应。这类问题的一个例子是第12章"接触"中介绍的电路板跌落测试。在该例子中，位于泡沫包装中的电路板从1米高处跌落到地面。该问题涉及包装与地面之间的冲击，以及电路板与包装之间快速变化的接触条件。

**复杂后屈曲问题**

Abaqus/Explicit 可以很容易地求解不稳定的后屈曲问题。在这类问题中，随着载荷的施加，结构的刚度会发生剧烈变化。后屈曲响应通常包括接触相互作用的影响。

**高度非线性准静态问题**

由于各种原因，Abaqus/Explicit 在求解某些本质上是静态的问题时通常非常高效。涉及复杂接触的准静态过程模拟问题（如锻造、轧制和板材成型）通常属于这类问题。板材成型问题通常包括非常大的薄膜变形、起皱和复杂的摩擦接触条件。体积成型问题的特征是大变形、飞边形成以及与模具的接触相互作用。第13章"使用 Abaqus/Explicit 进行准静态分析"中介绍了一个准静态成型模拟的例子。

**材料退化和失效**

材料退化和失效在隐式分析程序中经常导致严重的收敛困难，但 Abaqus/Explicit 可以很好地模拟这类材料。材料降解的一个例子是混凝土开裂模型，其中拉伸开裂导致材料刚度变为负值。材料失效的一个例子是金属的延性失效模型，其中材料刚度可以退化直至降至零。此时，失效的单元将从模型中完全移除。

每种类型的分析都可以包含温度和热传递效应。
