# 3.8.3 三维连续体应力/位移子模型





**产品：**Abaqus/Standard  Abaqus/Explicit  

### 测试的单元

C3D4    C3D6    C3D8    C3D8R    C3D10    C3D10I    C3D10M    C3D15    C3D20    C3D27    SC6R    SC8R    

### 测试的功能

子模型功能应用于三维连续体应力/位移单元和连续体壳单元。在Abaqus/Standard中，全局分析和子模型分析均使用各种组合的一般静态、动态和线性摄动程序。在Abaqus/Explicit中，全局模型和子模型都是准静态分析，但使用GLOBAL ELSET参数的测试除外，它是动态过程。在准静态测试中，每个子模型可以从任何全局模型驱动。

子模型功能也在Abaqus/Standard中测试了代表静态程序（一般和线性摄动）中单元刚度的直接输入矩阵。在Abaqus/Explicit子模型分析中测试了子模型边界条件中的缩放参数。

### 问题描述

**模型：**

所有全局模型的尺寸为8.0×1.5×1.0。

**材料：**

| 杨氏模量 | 3×10^6 |
| --- | --- |
| 泊松比 | 0.3 |
| 密度 | 10.0 |
| Rayleigh阻尼（![](../graphics/ver_eqn01224.gif)） | 0.2 |
| Rayleigh阻尼（![](../graphics/ver_eqn01225.gif)） | 0.4 |

**载荷和边界条件：**

Abaqus/Standard中涉及静态程序的所有全局模型和Abaqus/Explicit中的所有分析承受与["二维连续体应力/位移子模型，"第3.8.2节](ch03s08abv205.md)相同的载荷和边界条件。载荷的时间历史、执行相应子模型分析的时间以及从全局模型请求的文件输出对于每个单独分析都是唯一的。在Abaqus/Explicit中使用平滑步幅值来加载准静态模型。

Abaqus/Standard中涉及动态程序的所有全局模型承受与["二维连续体应力/位移子模型，"第3.8.2节](ch03s08abv205.md)相同的载荷和边界条件。对于使用直接积分隐式动态程序的瞬态模拟，可以通过更改输入文件中定义的参数来测试载荷的不同激励频率。与静态分析一样，载荷的时间历史、执行相应子模型分析的时间以及从全局模型请求的文件输出对于每个单独分析都是唯一的。

### 结果与讨论

子模型分析中所有驱动变量的幅值在全局分析文件输出中被正确识别，并应用于子模型分析中的驱动节点。
