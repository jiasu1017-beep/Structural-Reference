# 13 使用 Abaqus/Explicit 进行准静态分析





显式求解方法是一种真正的动态方法，最初为惯性在解中起主导作用的高速撞击事件建模而开发。不平衡力作为应力波在相邻单元之间传播，同时求解动态平衡状态。由于最小稳定时间增量通常相当小，大多数问题需要大量增量。

显式求解方法已被证明在求解准静态问题方面也有价值——Abaqus/Explicit 比 Abaqus/Standard 更易于求解某些类型的静态问题。显式过程相对于隐式过程的一个优势是更容易解决复杂的接触问题。此外，随着模型变得非常大，显式过程比隐式过程需要更少的系统资源。参阅 ["隐式和显式程序的比较，" 第 2.4 节](ch02s04.md)，获取隐式和显式程序的详细比较。

将显式动态过程应用于准静态问题需要一些特殊考虑。由于静态解，顾名思义，是一个长时间解，因此以自然时间尺度模拟事件通常在计算上不切实际，因为这需要过多的微小时间增量。为了获得经济的解，必须以某种方式加速事件。问题是，随着事件加速，静态平衡状态演变为动态平衡状态，惯性力变得更加主导。我们的目标是在惯性力保持不显著的最短时间周期内建模过程。

准静态分析也可以在 Abaqus/Standard 中进行。Abaqus/Standard 中的准静态应力分析用于分析具有时间相关材料响应（蠕变、膨胀、粘弹性和双层粘塑性）且可以忽略惯性效应的线性或非线性问题。有关 Abaqus/Standard 中准静态分析的更多信息，请参阅 ["准静态分析，" Abaqus 分析用户指南第 6.2.5 节](../usb/usb-link.md#usb-anl-avisco)。
