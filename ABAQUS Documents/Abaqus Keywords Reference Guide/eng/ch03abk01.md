# *C ADDED MASS







### *C ADDED MASSSpecify concentrated added mass in a *FREQUENCY step.

This option is used to include the “added mass” contributions due to concentrated fluid inertia loads in a [*FREQUENCY](ch06abk35.md) step.

**Product: **Abaqus/Aqua  

**Type: **History data 

**Level: **Step

##### **Reference:**

- ["Abaqus/Aqua analysis," Section 6.11.1 of the Abaqus Analysis User's Guide](../usb/usb-link.md#usb-anl-aaqua)

**There are no parameters associated with this option.**
