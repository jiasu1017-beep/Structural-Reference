# 11.7.1 选择性子循环

**产品：** Abaqus/Explicit

##### **参考文献**

- ["显式动态分析," 第 6.3.3 节"](pt03ch06s03at08.md)
- ["完全耦合热应力分析," 第 6.5.3 节"](pt03ch06s05at19.md)
- [*SUBCYCLING](../key/key-link.md#usb-kws-msubcycling)

### 概述

选择性子循环：
- 允许对不同的单元组使用不同的时间增量；
- 当模型中控制稳定时间增量的单元小区域存在时，减少分析的运行时间；以及
- 通过定义子循环区域来调用。

### 简介

Abaqus/Explicit 中的选择性子循环方法基于域分解。在这种方法中，定义了在整个分析过程中保持不变子循环区域。当定义子循环区域时，自动调用域级并行化方法（["Abaqus/Explicit 中的并行执行," 第 3.5.3 节"](pt01ch03s05aus34.md)）。每个子循环区域以及非子循环区域独立地分解为用户指定的并行域数量。"主"域被定义为从非子循环区域派生并使用最大稳定时间增量积分的并行域。从子循环区域派生的剩余并行域使用较小的时间增量或"子循环"进行积分。

子循环时间增量大小选择为主并行域中使用的时间增量的整数因子。因此，所有并行域精确地达到与主并行域相同的时间点。在子循环期间，位于与非子循环区域界面上的节点需要特殊处理。界面节点处的速度取自非子循环区域，并在子循环期间保持恒定。这产生在子循环期间线性变化的界面节点位移场。

### 定义子循环区域

子循环区域由单元集定义。您可以在这些集合中包含除欧拉单元类型 EC3D8R 和 EC3D8RT 外的所有单元类型。但是，所有并行域必须至少有一个可变形单元来提供稳定时间增量。如果并行域中没有可变形单元，Abaqus/Explicit 会发出错误消息。您可以定义任意数量的子循环区域。但是，某些建模特征不能在子循环区域之间分割。当包含无法分割的特征时，Abaqus/Explicit 会自动合并子循环区域。当以下情况时，子循环区域会合并：
- 区域重叠；
- 区域共享相同的节点；
- 一个节点位于一个子循环区域，但其相邻节点位于不同的子循环区域；
- 子循环区域参与相同的约束方程、连接器或刚体；或
- 分析中指定了通用接触。

当子循环区域合并时，使用合并区域中最小的稳定时间增量。如果约束、连接器或刚体的任何一个节点参与该子循环区域，则始终将其分配给子循环区域。由于使用域级并行化方法，所有关于并行域分解的限制都适用于子循环区域。这些限制阻止某些特征在主并行域之间以及包含子循环区域的并行域之间分割（参见 ["Abaqus/Explicit 中的并行执行," 第 3.5.3 节"](pt01ch03s05aus34.md)）。当定义子循环区域时，分析刚体表面不能包含在通用接触域中。

有效的选择性子循环需要适当选择子循环区域。对于每个子循环区域，时间增量大小应与非子循环区域相比较小，从而产生大量子循环。子循环数是非子循环区域中的稳定时间增量大小与子循环区域中的稳定时间增量大小之比。除了大量子循环外，为了获得最佳性能提升，子循环区域中的单元数量通常应相对于模型中的总单元数量较小。如果模型中的大多数单元位于子循环区域中，则不会有太多性能提升。

| **输入文件用法：** | 使用以下选项定义子循环区域： |
| --- | --- |
|  | ``` [*SUBCYCLING](../key/key-link.md#usb-kws-msubcycling), ELSET=*element_set_name* ``` |

### 结果的准确性

Abaqus/Explicit 中使用的子循环算法为大多数复杂动态模型提供了足够的准确性。然而，由于非子循环区域中使用的时间增量相对较大，以及区域界面节点上使用的插值，子循环解决方案可能引入截断误差，与传统解决方案相比可能略微改变结果。此误差不应影响模型的整体动态行为。当涉及通用接触时（参见 ["在 Abaqus/Explicit 中定义通用接触相互作用," 第 36.4.1 节"](pt09ch36s04aus155.md)），应特别注意子循环区域和非子循环区域之间的界面。不必定义有可能在同一区域内相互接触的一对表面。但是，为了最小化截断误差，强烈建议不要将有可能与其他表面接触的单个表面分割到不同区域。

### 输出和质量缩放

输出（参见 ["输出," 第 4.1.1 节"](pt02ch04s01aus38.md)）和质量缩放（参见 ["质量缩放," 第 11.6.1 节"](pt04ch11s06aus74.md)）始终在所有并行域达到的相同时间点执行。
