# 17.18.10 将Abaqus单元与网格区域关联





要将特定Abaqus单元与网格区域或孤儿单元关联，请从主菜单栏选择 ****Mesh**![](../graphics/images/arrow.gif)**Element Type****。 然后选择要为其分配单元类型的区域，并使用出现的**Element Type**对话框进行分配。

您可以使用对话框为您选择的区域可能出现的所有单元形状指定Abaqus单元设置，即使该区域目前只包含少数不同的单元形状。 例如，即使所选区域可能只包含四边形单元，您也可以关联其他形状的单元类型，例如三角形。

单元类型设置的功能类似于特征。 例如，如果您为区域分配了单元类型，然后将该区域划分为多个更小的区域，则新区域将继承原始父区域的单元类型设置。

**将Abaqus单元与网格区域关联：**

1. 从主菜单栏，选择 ****Mesh**![](../graphics/images/arrow.gif)**Element Type****。 Abaqus/CAE在提示区域显示提示以指导您完成该过程。 **提示：**您也可以使用Mesh模块工具箱中的 ![](../graphics/ico_meshElementType.png) 工具选择单元类型。 （有关更多信息，请参阅["使用Mesh模块工具箱，" 第17.15节](pt03ch17s15.md)。）
2. 如果您的装配或零件同时包含孤儿单元和几何区域，请在提示区域选择**geometry**来为几何分配单元类型。 装配中的孤儿单元从属于依赖零件实例；您不能为依赖零件实例分配单元类型。 要为孤儿单元分配单元类型，必须从上下文栏**Object**字段的零件列表中选择网格零件，然后为零件的所需单元分配单元类型。
3. 如果您要从多个几何区域中进行选择，或者要从从输出数据库导入的零件中选择单元，请使用以下选择技术： **Geometry** 使用鼠标在视口中选择所需的区域，然后在选择完成后单击鼠标按钮2。 您只能从相同类型的零件中选择区域；例如，您不能同时选择刚性表面和可变形体。 同样，您选择的区域必须具有相同的维度。 **Orphan mesh** 使用鼠标选择所需的孤儿单元，然后在选择完成后单击鼠标按钮2。 或者，您可以单击提示区域右侧的**Sets**。 会出现一个对话框，其中包含与网格化零件关联的所有单元集的列表。 选择您想要的单元集，然后单击**Continue**。 有关创建集的信息，请参阅[第73章，"The Set and Surface toolsets"](pt06ch73.md)。 无论选择方法如何，您选择的所有单元必须具有相同的阶次。 此外，单元必须属于相同类型的零件。 **Element Type**对话框出现。
4. 在对话框的左上角，选择您想要的**Element Library**选项。 选择**Standard**可以从Abaqus/Standard单元列表中选择，或选择**Explicit**可以从Abaqus/Explicit单元列表中选择。 在Abaqus/CFD模型中，只有流体单元类型可用。
5. 选择您想要的**Geometric Order**：**Linear**（一阶）或**Quadratic**（二阶）。
6. 从对话框右侧的**Family**列表中，为您将在模型上执行的分析类型选择适当的单元族。 例如，如果您计划进行热传递分析，请选择**Heat Transfer**族。 在Abaqus/CFD模型中，只有**Fluid**族可用。 您指定的单元库、几何阶次和族的默认单元名称出现在对话框的下半部分。 **注意：**您可以设置仅对应于单个族的单元类型。 例如，您无法同时为平面应变和热传递族设置线性三角形的单元类型；您必须选择热传递或平面应变。 如果在为族设置单元类型后切换到另一个族，则第一个族的设置将丢失。
7. 为每种单元形状选择Abaqus单元类型。 1. 单击与感兴趣单元形状对应的标签。 2. 选择您想要的单元特性。 有关单元控制选项的更多信息，请参阅["截面控制，" Abaqus Analysis User's Guide第27.1.4节](../usb/usb-link.md#usb-elm-esectioncontrol)。 满足您所有条件的Abaqus单元名称和简要描述出现在标签页底部。
8. 单击**OK**提交您的单元类型分配，或单击**Defaults**然后**OK**将所有单元设置返回到其默认值。 单元类型会根据您的规格进行更改。
9. 要为其他区域设置单元类型，从步骤2开始重复此过程。
