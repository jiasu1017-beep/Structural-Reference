# 13.10.10 替换零件实例





从主菜单栏选择****Instance**![](../graphics/images/arrow.gif)**Replace****可以用模型中另一个零件的实例替换所选零件实例。Abaqus/CAE定位新零件实例，使其原点位于原始零件实例的原点，并且它们的轴对齐。此外，你可以选择新零件实例是否继承它所替换的实例的所有约束。

替换操作不会更改实例的属性。例如，如果原始实例是依赖的，替换它的实例也将是依赖的。因此，如果存在零件的独立实例，则不能使用替换过程创建同一零件的依赖实例。

当你用具有相似几何形状的零件实例替换时，替换零件实例最有用。例如，新零件实例可能具有原始零件实例所没有的额外细节。

你不能在模型实例上使用**Replace**命令。

**要替换零件实例：**

1. 从主菜单栏，选择****Instance**![](../graphics/images/arrow.gif)**Replace****替换所选零件实例。
2. 从装配中选择要替换的零件实例。你也可以点击提示区域右侧的**Instance List**按钮并从出现的**Instance List**对话框中选择实例。
   **提示：**如果你无法选择所需的零件实例，可以使用**Selection**工具栏更改选择行为。更多信息，请参见["使用选择选项，"第6.3节](pt01ch06s03.md)。Abaqus/CAE显示**Replace Instance**对话框，其中包含模型中所有零件的列表。
3. 从**Replace Instance**对话框中，选择将替换装配中所选零件实例的零件。Abaqus/CAE在装配中显示新零件实例的临时图像，并将其定位使其原点位于原始零件实例的原点，并且它们的轴对齐。
4. 如果选择了正确的零件实例，点击**Replace Instance**对话框中的**确定**。如果你没有对原始零件实例应用任何位置约束，Abaqus/CAE会用新零件实例替换它。
5. 如果你对原始零件实例应用了位置约束，你必须选择提示区域中的以下按钮之一：
   - 点击**确定**将新零件实例定位在与被替换实例相同的位置。Abaqus/CAE删除应用于原始实例的任何约束，同时保持其位置。
   - 如果你希望新零件实例继承被替换零件实例的位置约束，点击**应用先前约束**。Abaqus/CAE应用所有可以被新零件实例满足的先前约束；任何无法满足的约束将被忽略。Abaqus/CAE用新零件实例替换原始零件实例。
