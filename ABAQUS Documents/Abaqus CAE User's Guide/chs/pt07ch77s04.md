# 77.4 对几何和网格单元进行着色





本节介绍如何从除可视化模块之外的任何模块对几何和网格单元应用颜色编码。如果您在可视化模块中应用颜色编码，请参见["在可视化模块中对所有几何进行着色，" 第77.5节](pt07ch77s05.md)，或["在可视化模块中对节点或单元进行着色，" 第77.6节](pt07ch77s06.md)。

Abaqus/CAE 根据特定类型对象（如零件实例、材料、截面或显示组）的颜色映射对当前视口应用颜色编码。本节介绍如何检查特定颜色映射中的分配，以及如何将颜色映射中的分配应用到当前视口。

Abaqus/CAE 提供了两种方法，您可以使用它们将颜色编码应用到当前视口中的预定义对象类型。您可以通过从 **颜色编码** 工具栏中 ![](../graphics/ico_colorCode.png) 工具右侧的列表中选择颜色映射名称来快速选择颜色映射。Abaqus/CAE 使用该颜色映射中指定的颜色编码刷新当前视口。或者，您可以从 **颜色编码** 对话框中选择颜色映射。如果要自定义颜色映射，必须使用对话框。请参见["自定义单个对象的显示颜色，" 第77.8节](pt07ch77s08.md)，了解有关更改分配给单个对象的颜色的选项说明。

**使用工具栏列表应用颜色映射：**

1. 找到颜色映射列表。该列表位于 **颜色编码** 工具栏中 ![](../graphics/ico_colorCode.png) 工具右侧。
2. 从列表中选择一个颜色映射。Abaqus/CAE 根据 **颜色编码** 对话框中显示的颜色对几何和网格设置进行颜色编码。

**使用颜色编码对话框应用颜色映射：**

1. 点击 **颜色编码** 工具栏中的 ![](../graphics/ico_colorCode.png) 工具。Abaqus/CAE 显示 **颜色编码** 对话框，其中显示当前模块的默认颜色映射。
2. 如果您正在对几何或网格单元进行颜色编码，请从 **颜色编码** 列表中选择一种颜色映射。（如果您正在执行不同的颜色编码操作，请参阅["在可视化模块中对所有几何进行着色，" 第77.5节](pt07ch77s05.md)，或["在可视化模块中对节点或单元进行着色，" 第77.6节](pt07ch77s06.md)。）Abaqus/CAE 在对话框的 **颜色映射** 部分显示选定的颜色映射。
3. 点击 **应用**。Abaqus/CAE 根据 **颜色编码** 对话框中显示的颜色对当前视口进行颜色编码。
