# 42.8.5 创建跟随单个节点的坐标系





您可以定义附加到模型上单个节点的坐标系。指定节点是坐标系的原点。节点处的自由度（平移和转动）决定了分析每个步骤和帧处坐标系轴的方向。如果节点仅有平移自由度，则坐标系将始终平行于全局坐标系。

跟随模型上节点的坐标系的视觉显示会更新以反映所应用的任何变形比例因子；但是，基于这些坐标系的结果转换使用实际的、未缩放的变形。

**创建跟随单个节点的坐标系：**

1. 从主菜单栏，选择****工具**![](../graphics/images/arrow.gif)**坐标系**![](../graphics/images/arrow.gif)**创建****。此时将出现**创建坐标系**对话框。**提示：**您也可以使用工具箱中的![](../graphics/ico_dtmCsys.png)工具或当前输出数据库结果树中的**会话坐标系**容器来创建坐标系。
2. 在**创建坐标系**对话框中：1. 接受坐标系的默认名称，或在文本字段中输入您选择的名称。2. 切换**跟随单节点的坐标系**以指定坐标系的运动。3. 选择以下之一作为坐标系类型：- **直角**：*X*、*Y*和*Z*轴分别解释为1、2和3轴。- **圆柱**：*R*、![](../graphics/usi_eqn00065.gif)和*Z*轴分别解释为1、2和3轴。- **球面**：*R*、![](../graphics/usi_eqn00065.gif)和![](../graphics/usi_eqn00226.gif)轴分别解释为1、2和3轴。4. 点击**继续**。Abaqus/CAE在提示区域显示提示，以帮助您定义坐标系轴。
3. 在视口中选择该节点。坐标系出现在视口中、**坐标系管理器**的列表中，以及当前输出数据库结果树中的**会话坐标系**容器下。Abaqus仅将用户定义的坐标系保存到输出数据库关闭为止，除非您将它们保存到输出数据库文件。坐标系一旦创建就不能编辑；您可以重命名或删除尚未保存到输出数据库的坐标系。
