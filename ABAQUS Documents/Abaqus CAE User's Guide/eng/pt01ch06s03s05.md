# 6.3.6 Choosing which objects are selected by the drag-select region







The selection tools allow you to choose which objects are selected by the drag-select region. From the **Selection** toolbar, choose one of the following:

**Inside**

![](../graphics/ico_selectInside.png) Select only the objects that fall inside the drag-select region.

**Inside and crossing**

![](../graphics/ico_selectInsideCrossing.png) Select only the objects that fall inside or cross the drag-select region. This tool is selected by default.

**Crossing**

![](../graphics/ico_selectCrossing.png) Select only the objects that cross the drag-select region.

**Outside and crossing**

![](../graphics/ico_selectOutsideCrossing.png) Select only the objects that fall outside or cross the drag-select region.

**Outside**

![](../graphics/ico_selectOutside.png) Select only the objects that fall outside the drag-select region.
