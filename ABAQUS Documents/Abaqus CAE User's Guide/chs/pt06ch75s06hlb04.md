# 75.6.4 忽略边和顶点





您可以选择 ****Tools**![](../graphics/images/arrow.gif)**Virtual Topology**![](../graphics/images/arrow.gif)**Ignore Entities**** 从主菜单栏来忽略选定的边和顶点。Abaqus/CAE 从装配中移除选定的实体，并在生成网格时忽略这些实体。

**忽略边和顶点的步骤：**

1. 从上下文栏的 **Object** 字段中，选择一个零件或装配。
2. 从主菜单栏，选择 ****Tools**![](../graphics/images/arrow.gif)**Virtual Topology**![](../graphics/images/arrow.gif)**Ignore Entities****。**提示：**您也可以使用位于 Mesh 模块工具箱中的 ![](../graphics/ico_virtTopoIgnoreEntities.png) 工具来忽略边和顶点。
3. 选择要忽略的边和顶点的组合。您可以使用拖选、**[Shift]****+点击**、**[Ctrl]****+点击** 和角度方法的组合来选择多条要移除的边或顶点。有关更多信息，请参见 ["在当前视口中选择对象，" 第6.2节](pt01ch06s02.md)。**提示：**如果您无法选择所需的边或顶点，可以使用 **Selection** 工具栏更改选择行为。有关更多信息，请参见 ["使用选择选项，" 第6.3节](pt01ch06s03.md)。
4. 如果所选边端点处的顶点也可以被忽略，Abaqus/CAE 会高亮显示这些顶点，并询问您是否要将它们包含在您的选择中。点击 **Yes** 忽略高亮的顶点。点击 **No** 保留高亮的顶点。Abaqus/CAE 忽略选定的边和顶点。您可以使用模型树或 **Restore Entities** 工具来恢复选定的实体。有关恢复单个实体的更多信息，请参见 ["恢复被虚拟拓扑替换的实体，" 第75.6.5节](pt06ch75s06hlb05.md)。
