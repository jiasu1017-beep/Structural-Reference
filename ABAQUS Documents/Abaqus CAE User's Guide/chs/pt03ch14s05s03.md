# 14.5.3 诊断打印





如果模型的分析失败或产生意外结果，你可以通过查看写入以下文件的选定诊断信息来检查其逐次迭代的进展：

**对于Abaqus/Standard分析：**

诊断信息写入消息（`.msg`）文件，信息子集写入输出数据库（`.odb`）文件。你可以在可视化模块中查看输出数据库中的诊断信息（更多信息，请参见[第41章"查看诊断输出"](pt05ch41.md)"）。默认情况下，信息在每次迭代期间写入；你可以通过指定零输出频率来请求Abaqus停止写入消息文件中的诊断信息。

**对于Abaqus/Explicit分析：**

诊断信息写入状态（`.sta`）文件。有关此信息写入的频率的信息，请参见["输出，"Abaqus分析用户指南第4.1.1节](../usb/usb-link.md#usb-out-ooutput)。

你通过从主菜单栏选择****Output**![](../graphics/images/arrow.gif)**Diagnostic Print****来显示**编辑诊断打印**对话框。

有关请求诊断打印的详细说明，请参见["配置诊断打印，"第14.13.3节](pt03ch14s13hlb03.md)。

**注意：**诊断打印请求的更改不影响Abaqus/Standard分析期间写入输出数据库的诊断信息。
