# 82.7.2 使用自适应绘图器插件





通过从主菜单栏选择 ****Plug-ins**![](../graphics/images/arrow.gif)**Tools**![](../graphics/images/arrow.gif)**Adaptivity Plotter**** 来启动插件。当您启动自适应绘图器插件时，它会读取当前模型数据库并显示 **Adaptivity Plotter** 对话框。在 **Adaptivity Plotter** 对话框中输入以下内容：
- 从模型数据库中选择所需的自适应过程。
- 切换您要绘制的重网格规则。在适用的情况下，您可以选择以下内容： - 由重网格规则生成的误差指示器。 - 这些误差指示器的全局范数。
- 切换 **Plot in separate viewport** 以在新视口中创建 *X--Y* 绘图。如果关闭此选项，插件将在当前视口中创建 *X--Y* 绘图。
- 切换 **Show element count** 以绘制单元计数，汇总所有活动重网格规则的单元计数。
- 切换 **Show remeshing rule targets** 以绘制每个规则的目标误差指示器范数。
- 点击 **Plot** 创建 *X--Y* 绘图，或点击 **Export** 以 HTML 格式创建误差指示器范数表。[图82-8](pt08ch82s07s02.md#usi-plg-adaptive-xyplot) 显示了由插件创建的 *X--Y* 绘图的示例。
**图82--8** 由 **Adaptivity Plotter** 插件创建的 *X--Y* 绘图。
![](../graphics/usi-plg-adaptive-xyplot.png)
[图82-8](pt08ch82s07s02.md#usi-plg-adaptive-xyplot) 中绘制了以下四个值：
- **RemeshingRule-1** 的误差指示器 ENDENERI 的 **MaxBase** 范数。该范数表示最大能量密度位置处的误差指示器值，按该密度归一化。
- **RemeshingRule-1** 的误差指示器 ENDENERI 的 **global Norm**。该范数表示整个重网格规则域上误差指示器的归一化平均值。
- 误差目标，这是创建重网格规则时设置的目标。在此示例中，目标固定为5%。
- 单元计数，表示每个活动重网格规则区域中单元的总和。*Y* 轴刻度为百分比，用于误差指示器和误差目标的绘图。要确定误差计数，必须将图例中指示的缩放因子（本例中为800）乘以 *Y* 轴上的百分比值。例如，迭代2处的单元计数约为20 800，即16000个单元。
