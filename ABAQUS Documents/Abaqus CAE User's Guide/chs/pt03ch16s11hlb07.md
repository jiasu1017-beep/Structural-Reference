# 16.11.7 定义流体湍流场





您可以创建流体湍流场来为Abaqus/CFD分析中的流体定义以下初始条件：
- Spalart-Allmaras湍流模型：运动涡粘度
- *k*--![](../graphics/usi_eqn00417.gif) RNG湍流模型：湍流动能和耗散率

有关更多信息，请参见[第30章，"流体动力学分析"](pt04ch30.md)。"

**创建或编辑流体湍流场的步骤：**

1. 使用以下方法之一显示流体湍流场编辑器： - 要创建新的流体湍流场，请按照["创建预定义场"章节第16.8.3节](pt03ch16s08hlb03.md)中概述的程序进行操作（**类别**：`流体`；**所选步骤的类型**：`流体湍流`）。 - 要使用菜单或管理器编辑现有的流体湍流场，请参见["编辑依赖于步骤的对象"章节第3.4.13节](pt01ch03s04hlb05.md)。**注意：**您只能在初始步骤中创建或编辑流体湍流场。有关更多信息，请参见[第30章，"流体动力学分析"](pt04ch30.md)。
2. 默认情况下，流体湍流场应用于整个模型。如果要更改流体湍流场适用的区域，请执行以下操作： 1. 单击![](../graphics/ico_selectBlue.png)。 2. 从视口中选择一个或多个区域或集合。 3. 在提示区域中单击**完成**。Abaqus/CAE更新流体湍流场编辑器以反映您的选择。
3. 为所选的湍流模型指定初始条件。流体湍流编辑器显示适用于当前流动步骤中活跃的湍流模型的参数。 - 对于Spalart-Allmaras湍流模型，请在**运动涡粘度**字段中输入初始粘度值。 - 对于*k*--![](../graphics/usi_eqn00417.gif) RNG湍流模型，请输入以下值： 1. 在**湍流动能**字段中，输入流体的初始湍流动能*k*。 2. 在**耗散率**字段中，输入流体的初始耗散率![](../graphics/usi_eqn00417.gif)。
4. 单击**确定**保存数据并退出编辑器。
