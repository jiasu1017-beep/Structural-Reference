# 3.2.11 从列表和表中选择多个项目





在某些Abaqus/CAE对话框中，执行某些功能之前必须从列表或表中选择一个项目。例如，如果要绘制X-Y数据，必须首先从[图3-7](pt01ch03s02s11.md#int-mult1)中所示的**XY数据管理器**列表中选择您需要的数据对象，然后单击**绘制**。

**图3-7** 已选择单个项目。

![](../graphics/int-mult1-nls.png)

某些功能允许您对多个项目进行操作。例如，如果您想删除[图3-7](pt01ch03s02s11.md#int-mult1)所示管理器中的前两个数据对象，可以同时选择它们，然后单击**删除**。

要从列表中选择单个项目，只需在对话框中单击该项目。要从表中选择单个项目，请单击表行标题。要选择多个项目，可以使用以下技术：

**从列表或表中选择连续项目**

从列表中单击第一个感兴趣的项目或从表中单击行标题，然后在继续按住鼠标按钮1的同时拖动光标掠过其余项目。当所有感兴趣的项目都被选中时释放鼠标按钮。例如，在[图3-8](pt01ch03s02s11.md#int-mult2)中选择了连续项目。

**图3-8** 已选择连续项目。

![](../graphics/int-mult2-nls.png)

选择连续项目的另一种方法是从列表中单击第一个感兴趣的项目或从表中单击行标题，然后**[Shift]****+单击**最后一个感兴趣的项目。第一个和最后一个之间的所有项目会自动选中。

**从列表或表中选择非连续项目**

从列表中单击第一个感兴趣的项目或从表中单击行标题，然后**[Ctrl]****+单击**任何其他要选择的项目。例如，在[图3-9](pt01ch03s02s11.md#int-mult3)中选择了非连续项目。

**图3-9** 已选择非连续项目。

![](../graphics/int-mult3-nls.png)

**取消选择**

您可以**[Ctrl]****+单击**先前选中的项目将其从选择中移除。例如，如果**[Ctrl]****+单击**[图3-9](pt01ch03s02s11.md#int-mult3)列表中的`Displacement`，该数据对象将不再被选中，如[图3-10](pt01ch03s02s11.md#int-mult4)所示。

**图3-10** 已从选择中移除单个项目。

![](../graphics/int-mult4-nls.png)

当您选择多个项目时，对话框中的某些功能可能会变得不可用。例如，[图3-10](pt01ch03s02s11.md#int-mult4)所示的**数据管理器**中的**编辑**、**复制**和**重命名**功能仅对单个数据对象有效。当您选择多个数据对象时，这三个功能会变得不可用。
