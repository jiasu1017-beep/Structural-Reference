# 14.10.1 步骤编辑器





当您创建、编辑或替换步骤时，步骤编辑器显示一组选项卡页面，允许您配置所选过程的设置。每个过程都有独特的页面；例如，当您配置**静态，通用**过程时，步骤编辑器显示**基本**、**增量**和**其他**选项卡。您可以使用这些选项卡页面配置的设置包括：步骤的时间周期、最大增量数、增量大小、随时间的默认载荷变化，以及是否考虑几何非线性。

Abaqus将您在**基本**选项卡页面上的**描述**字段中输入的文本存储在输出数据库中，并由可视化模块显示在状态块中。

如果您想将过程设置重置为默认值，可以用相同过程类型的步骤替换现有步骤。更多信息，请参见["重置步骤编辑器中的默认值，"第14.9.5节](pt03ch14s09hlb05.md)。
