# 4.4.1 为视口添加标注





Abaqus/CAE 提供三种类型的视口标注，可用于为模型或结果添加标注：文本字符串、箭头，以及箭头和文本组合标注。你使用**视口**菜单创建这些标注，并且可以将它们放置在视口中的任何位置。[图 4-6](pt01ch04s04s01.md#vp-annotate) 显示了用于标注模型的文本字符串和箭头。箭头和文本标注锚定到视口中的点。锚点定义了标注相对于视口大小和形状或模型坐标中某一点的位置。当你操作模型或调整视口大小时，标注会跟随其锚点的位置。

**图 4-6** 箭头和文本标注。

![](../graphics/vp-annotation.png)

视口箭头和文本标注会在退出 Abaqus/CAE 会话时与模型数据库或输出数据库一起保存。你也可以将标注从模型复制到输出数据库，反之亦然。

**警告：**你始终可以创建或复制标注到输出数据库；但是，只有在以写权限打开数据库时才能保存它们。

**文本**

文本标注可以包含可以使用工作站上可用字体显示的任何字符。Abaqus/CAE 不限制标注中文本的长度或行数。但是，在创建文本标注时必须考虑视口的大小；文本应该适合所需的位置。你可以将文本放置在视口中的任何位置，并且可以在创建后移动文本标注或更改其锚点。每个文本标注可以使用不同的字体、颜色和背景显示，但你不能在同一文本标注内更改这些属性。

**箭头**

你可以在视口中的任何位置创建箭头；通常，箭头会将文本标注连接到模型坐标中的某一点。箭头可以具有多种不同的线粗细、线型和端点样式，并且可以使用工作站上可用的任何颜色显示。你可以使用一个或两个锚点来控制箭头标注的位置。例如，你可以将箭头头部锚定到模型上的一个点，将尾部锚定到视口位置，这样当操作模型时箭头仍然指向所需特征，但尾部保持固定。你可以在创建后修改和移动箭头。

**箭头和文本组合**

你可以创建组合的箭头和文本标注。此方法使用简化的选项集来创建箭头标注（锚定在箭头头部），并在箭头尾部放置文本标注。提供此创建方法是为了简化向模型或结果的特定部分添加信息。你不能在创建期间编辑标注的位置，只能使用提示区域中的**上一步**按钮(![](../graphics/afxI_backup.png))。但是，一旦创建它们，箭头和文本会独立保存；因此你可以使用每种标注类型的所有可用选项来编辑它们。

除了你可以创建的视口标注外，Abaqus/CAE 还会生成一些标注来提供视口中视图的上下文。Abaqus/CAE 生成 3D 罗盘和视图 triad 来显示模型在视口中的方向；在可视化模块中，会生成图例、标题块和状态块来指示视口中分析结果的各种方面。**视口标注选项**对话框允许你显示或隐藏所有视口标注——你自己的和 Abaqus/CAE 生成的——你也可以使用它来编辑生成标注的属性。有关详细信息，请参阅[第 56 章，"自定义视口标注"](pt05ch56.md)。
