# 12.8.5 在材料点定义场变量







在Abaqus/Standard中，您可以通过用户子程序[`USDFLD`](../sub/sub-link.md#sub-xsl-usdfld)引入对解变量的依赖性。此子程序允许您将材料点处的场变量定义为时间、Abaqus分析用户指南第4.2.1节列出的任何可用材料点量以及材料方向的函数。因此，定义为这些场变量函数的材料属性可能依赖于解。

用户子程序[`USDFLD`](../sub/sub-link.md#sub-xsl-usdfld)在每个点上被调用，这些点的材料定义包含对该用户子程序的引用。

**在材料定义中包含对用户子程序[`USDFLD`](../sub/sub-link.md#sub-xsl-usdfld)的引用：**

1. 从**编辑材料**对话框的菜单栏中，选择**常规**![](../graphics/images/arrow.gif)**用户定义场**。（有关显示**编辑材料**对话框的信息，请参见"创建或编辑材料"，第12.7.1节。）
2. 单击**确定**关闭**编辑材料**对话框。或者，您可以从**编辑材料**对话框的菜单中选择其他材料行为进行定义（有关更多信息，请参见"浏览和修改材料行为"，第12.7.2节）。
