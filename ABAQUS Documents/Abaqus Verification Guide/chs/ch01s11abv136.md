# 1.11.8 无界声学区域的分析









**产品：**Abaqus/Standard  Abaqus/Explicit  

### I. 声学管道分析

### 测试单元

ACIN2D2    ACIN2D3    ACIN3D3    ACIN3D4    ACIN3D6    ACIN3D8    ACINAX2    ACINAX3    

### 测试功能

使用声学无限单元的稳态和瞬态动态分析。

### 问题描述

使用管道中平面波传播问题来验证声学无限单元的行为。管道长10个单位，一端被激励。管道本身用适当维度和插值阶数的声学有限单元建模。在另一端使用声学无限单元来模拟管道的无限延续。在每个输入文件中，提供了使用精确平面波吸收阻抗边界条件的另一个管道模型用于比较。虽然无限单元对于管道情况不是精确的，但它们应该给出与平面波阻抗情况相当的结果。

轴对称单元使用终止于轴对称声学无限单元的环形管道进行研究。比较管道相同但方向相反并以平面波阻抗条件终止。

**材料：**

声学流体：

| 声学体积模量，κ=1.28E5 |
| --- |
| 声学质量密度，ρ=1.25 |
| 声速，c=320.0 |

**载荷：**

在Abaqus/Standard验证文件中，执行两步分析。第一步在两个频率1和10下执行稳态动态分析。第二步中，管道中的流体最初处于静止状态，并在一端使用频率为ω的均匀正弦激励强制。对于ACIN3D6和ACIN3D8验证文件，除外的每种情况，激励都作为集中载荷提供；对于ACIN3D6和ACIN3D8，激励作为边界条件提供。参考解使用相同的声学有限单元网格找到，施加平面波阻抗条件。

在Abaqus/Explicit验证文件中，执行单步瞬态动态分析。管道中的流体最初处于静止状态，并在一端使用频率为ω的均匀正弦激励强制。对于每种情况，激励都使用集中载荷过程提供。参考解使用相同的声学有限单元网格找到，施加平面波阻抗条件。

### 结果与讨论

在每个步骤中，使用无限单元的解产生与伴随平面波阻抗情况中获得的结果相当的结果。

### II. 与实体单元的耦合

### 测试单元

ACIN2D2    ACIN2D3    ACIN3D3    ACIN3D4    ACIN3D6    ACIN3D8    ACINAX2    ACINAX3    

### 问题描述

研究了一个简单的瞬态问题来验证声学无限单元直接与结构单元的耦合。声学无限单元使用基于表面的绑定约束耦合到实体单元。使用边界条件对实体单元施加加速度。为了检查这些结果，在相同几何形状的声学无限单元上施加类似的加速度曲线。加速度时间历史使用幅值过程描述。

### 结果与讨论

对于测试的单元，两种情况的声压时间历史一致。Abaqus中施加加速度和载荷的方法存在小的数值差异，这解释了观察到的小差异。
