# 27.3 在可视化模块中显示显示体





您可以使用**显示体选项**来自定义显示体的外观。显示体可用的选项与Abaqus/CAE中的常用绘图选项和叠加绘图选项类似：渲染样式、边缘可见性、颜色和样式、填充颜色、缩放和透明度都可以自定义。同样，显示体选项与绘图状态无关；即应用于显示体的选项会在所有绘图状态中反映。更多信息，请参见["自定义显示体的外观，" 第55.8节](pt05ch55s08.md)。

显示体选项适用于模型中的所有显示体；要自定义单个显示体的外观，您可以基于部件实例创建显示组（请参见["创建或编辑显示组，" 第78.2.1节](pt07ch78s02hlb01.md)）。也可以将单个项目颜色应用于单个显示体部件实例，以覆盖在常规显示体选项中指定的边缘和/或填充颜色；请参见["自定义单个对象的显示颜色，" 第77.8节](pt07ch77s08.md)，了解更多信息。
