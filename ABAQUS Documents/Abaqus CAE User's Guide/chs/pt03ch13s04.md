# 13.4 使用模型实例







您可以在主模型中创建其他模型的实例，从而能够添加完整的子装配以及单个零件。模型实例的创建方式与零件实例完全相同，并且可以以类似的方式进行定位和操作。

当您创建新模型实例时，被引用模型的主装配会在当前工作模型的装配中被实例化。该实例从其他模型的内容生成一个子装配。由于被引用模型装配可能依次包含其他模型实例作为子实例，因此可能存在多级复杂子装配。

要被实例化的外部模型必须包含在当前模型数据库（`.cae`）文件中才可用。如果要实例化的模型包含在不同的模型数据库中，请使用**文件**![](../graphics/images/arrow.gif)**导入**![](../graphics/images/arrow.gif)**模型**将其导入当前模型数据库。模型数据库文件始终可以包含多个模型。

模型实例具有以下特征：
- 特定模型可以被实例化多次，并且您可以实例化任意数量的不同模型。
- 模型实例始终是从属的，而不是独立的。
- 您可以自由地混合模型实例和零件实例。
- 模型实例子装配可以包含几何零件或孤立网格零件。
- 可以通过变换（**平移**、**平移至**、**旋转**）和位置约束在主装配中定位和定向模型实例。变换和约束必须应用于完整的模型实例子装配，而不是其任何子实例。如果您选择模型实例内的子实例，变换或约束将被应用于整个父模型实例。
- 不支持线性阵列和径向阵列，也不能与模型实例一起使用。
- 可以对模型实例使用零件实例命令，如**抑制**/**恢复**、**隐藏**/**显示**、**删除**、**显示父级/子级**和**切换上下文**。**抑制**和**删除**命令不能用于模型实例的子实例，只能用于模型实例本身。如果您在原始（被引用）模型装配中抑制子实例（零件或模型），它也将在主模型中被抑制。要查看被抑制的实例在主模型中正确地被抑制，您必须使用上下文栏中的**模型**列表从原始（被引用）模型切换到主模型。在模型树中移动到主模型不会一致地重新生成模型实例子级。（有关上下文栏的信息，请参见"主窗口的组件"，第2.2.1节。）然后必须在原始模型中恢复子实例。
- 不支持**替换**、**从模拟中排除**、**合并/切割**和**链接实例**，也不能与模型实例一起使用。
- 不支持分区工具集，也不能与模型实例一起使用。
- 支持查询工具集，可用于确定模型实例的位置和属性。
- 被引用模型中定义的所有集合和表面都被带入模型实例，维护模型树的特征层次结构。这些集合和表面将在主模型中可用。
- 被引用模型中定义的历史级特征（如步骤、载荷、边界条件、交互和幅值）不会被带入模型实例。被引用模型中定义的一些模型级特征（如连接器分配、紧固件和其他工程特征）不会被带入模型实例。
- 模型实例在**显示组**和**装配显示选项**的**实例**选项卡中得到支持并可选择。
- 不支持模型实例的虚拟拓扑工具集。

子装配（被引用）模型中需要的任何零件级属性必须在该原始模型中创建和分配，不能在主模型装配中创建。例如，材料、截面、方向和皮肤/筋条分配必须在原始模型中创建。可以在原始独立零件实例上进行网格划分，网格将出现在模型实例中。

当您创建模型实例时，被引用模型装配的所有零件实例都会被添加为主模型装配中的子零件实例。任何被抑制的零件实例或被排除在模拟之外的实例将在子装配中保持相同的状态。

如果您在主模型装配中修改或删除现有零件实例或模型实例子装配，Abaqus/CAE会在您退出并重新进入主模型的装配模块时自动从所有父实例（零件和模型）重新生成子实例。

如果您尝试从另一模型创建新模型实例，而该模型又包含子模型实例，Abaqus/CAE将防止任何模型引用循环性的问题。Abaqus/CAE将阻止您创建这种有问题的实例。

Abaqus/CAE确保模型实例建模空间的一致性——如果主模型中的所有实例都是三维的，则要实例化的其他模型也必须是三维的。
