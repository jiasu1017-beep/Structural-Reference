# *ORNL









### *ORNL指定橡岭国家实验室开发的本构模型。

此选项用于根据核标准NEF 9-5 T"一级高温核系统元件设计指南和程序"的规定为304和316型不锈钢提供塑性和蠕变计算。它只能与[*PLASTIC](ch16abk14.md)选项和/或[*CREEP](ch03abk85.md)、LAW=STRAIN选项结合使用。

**产品：**Abaqus/Standard  Abaqus/CAE  

**类型：**模型数据  

**级别：**模型  

**Abaqus/CAE：**属性模块

##### **参考：**

- ["ORNL -- 橡岭国家实验室本构模型，" Abaqus Analysis User's Guide第23.2.12节](../usb/usb-link.md#usb-mat-cornl)

### **可选参数：**

A

将此参数设置为由蠕变应变引起的运动偏移的饱和率。默认值为0.3。设置A=0.0以使用1986年修订版的标准。

H

将此参数设置为相对于蠕变应变的运动偏移率。如果省略此参数，则根据1981年修订版标准的相应部分确定H的值。

MATERIAL

设置MATERIAL=SS以使用适用于304或316型不锈钢的硬化规律。这是目前唯一可用的选项，因此也是默认值。

RESET

包含此参数以调用核标准第4.3.5节中描述的可选重置程序。如果省略此参数，则不使用重置程序。

**此选项没有关联的数据行。**
