# 11.13.2 控制拉伸特征的方向





向三维零件添加拉伸特征时，Abaqus/CAE会根据您创建的特征类型从草图轮廓中选择默认的拉伸方向。默认情况下，实体或壳特征向外拉伸，材料被添加到现有基础特征。相反，切割特征向内拉伸，从现有基础特征中移除材料。

您可以按如下方式控制拉伸特征的方向：

**添加拉伸特征时选择方向**

完成草图以向现有零件添加拉伸特征时，Abaqus/CAE会在原始零件上显示新的草图轮廓。草图轮廓包含指示拉伸方向的箭头。Abaqus/CAE还会显示**编辑拉伸**对话框。

您可以通过在**编辑拉伸**对话框中单击来控制拉伸方向。视口中的箭头会更改方向以显示新的拉伸方向。

**编辑现有拉伸特征的方向**

当您选择要编辑的拉伸特征时，Abaqus/CAE会在视口中高亮显示选定的特征，并出现**编辑特征**对话框。

您可以通过在**编辑特征**对话框中切换**翻转拉伸方向**来反转拉伸方向。Abaqus/CAE不显示指示拉伸方向的箭头；但是，您可以单击**应用**来查看更改。当方向可接受时，单击**确定**结束编辑过程。

创建新零件时不能更改拉伸方向，因为无论方向如何，零件都是相同的。![](../graphics/images/black4rule.gif)有关相关信息，请单击以下任一项：- "添加拉伸实体特征"，第11.21.1节- "添加拉伸壳特征"，第11.22.1节- "创建拉伸切割"，第11.24.1节
