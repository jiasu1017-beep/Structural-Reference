# 44.4 生成线性梁截面应力的云图





如果您的模型包含梁几何体并且会话中已启用梁渲染，则可以将场输出结果沿梁横截面显示为云图。对于节点量和单元积分点量，Abaqus/CAE显示通过梁截面厚度恒定的云图。对于基于单元的数据，云图反映当前选择的截面点。

您还可以创建更真实地显示基于截面力和截面矩数据的梁应力分布的云图，Abaqus/CAE使用线性弹性固体力学理论进行计算。图44-3显示了一个工字梁截面梁应力的示例。

**图44-3** 显示在工字梁截面上的线性梁截面应力。

![](../graphics/rnb611-out-beamstress.png)

梁应力云图仅在所选步骤和帧包含集成输出量SF（截面力）和SM（截面矩）的数据时可用。此外，这些图仅适用于Abaqus/CAE中的部分截面型材：
- 薄壁箱型截面
- 薄壁管型截面
- 圆形截面
- 矩形截面
- 工字形截面
- L形截面
- T形截面

对于使用任何其他截面型材的梁几何体或锥形梁几何体，无法显示云图。

Abaqus/CAE在计算梁截面渲染时不包括用于特征频率提取的梁单元法线，因此您无法在这些图中可视化梁单元的扭转和面外模式。但是，由于梁单元具有扭转刚度，因此会计算梁单元的扭转模式；此信息可在数据（`.dat`）文件中获取。

**要生成线性梁截面应力的云图：**

1. 使用**File**菜单[打开包含分析结果的输出数据库](pt02ch09s06hlb02.md)。
2. 使用**Result**菜单选择以下各项：1. 要显示的[步骤和帧](pt05ch42s01hlb01.md)。2. 要显示的[主场](pt05ch42s03hlb03.md)输出变量。3. 要显示的[变形场](pt05ch42s03hlb04.md)输出变量（仅用于变形形状上的云图）。4. [要绘制的内容](pt05ch42s04hlb01.md)和[平均](pt05ch42s04s02.md)选项。
3. 选择您想要的绘图状态独立和云图[自定义选项](pt05ch40s03s03.md)。
4. 从**Output Variable**选项中，选择**BEAM_STRESS**。
5. 选择要显示云图数据的不变量或分量：- 从**Invariant**选项中选择**Mises**以显示von Mises应力。- 从**Component**选项中选择**S11**以显示沿梁的轴向应力。- 从**Component**选项中选择**S12**以显示由剪力和扭矩引起的沿局部梁截面2轴的剪应力。- 从**Component**选项中选择**S13**以显示由剪力和扭矩引起的沿局部梁截面1轴的剪应力。
6. 从主菜单栏中，选择****Plot**![](../graphics/images/arrow.gif)**Contours****，然后选择是在未变形形状、变形形状还是两者上绘制云图。**提示：**您还可以使用工具箱中的变形![](../graphics/ico_plotContourDeformed.png)、未变形![](../graphics/ico_plotContourUndeformed.png)或叠加![](../graphics/ico_plotContourDefUndef.png)云图工具生成云图。当前视口显示当前输出数据库在指定步骤和帧的指定场输出变量的自定义云图。每次您在步骤和帧选择器、场输出选项、绘图状态独立选项、叠加绘图选项（如果适用）或云图选项对话框中单击**Apply**时，Abaqus会自动刷新您的云图。
