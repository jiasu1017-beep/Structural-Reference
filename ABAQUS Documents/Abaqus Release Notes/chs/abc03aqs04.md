# 3.4 监控优化过程





**产品：** Abaqus/CAE

**优势：** 现在您在监控优化过程进程时具有更大的灵活性。

**描述：** Abaqus/CAE 现在可以显示表格中的输出数据，允许您监控优化过程的进程以确保它正在收敛到可接受的解决方案。您可以选择要监控的变量，并可以在新视口中将选定的变量显示为 *X–Y* 图。

**Abaqus/CAE 使用：**
```
Job 模块：
    ****Optimization**![](../graphics/images/arrow.gif)**Monitor****，**Name：** *任务名称*
```
