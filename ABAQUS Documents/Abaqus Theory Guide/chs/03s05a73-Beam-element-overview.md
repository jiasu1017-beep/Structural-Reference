# 3.5.1 梁单元概述

### 3.5.1 梁单元概述

Abaqus中的单元库包含几种类型的梁单元。此上下文中的"梁"是进行了假设以将问题数学上简化为一维的单元：主解变量仅是沿梁轴线位置位置的函数。为了使这些假设合理，直觉上很清楚，梁必须是一个连续体，我们可以在其中定义一条轴线，使得从该轴线到连续体中任何点的最短距离与沿轴线的典型长度相比较小。这个概念在"梁单元公式，" 第3.5.2节中的详细推导中变得更加精确。将问题简化为一维问题的假设有几种复杂程度，Abaqus中不同的梁单元使用不同的假设。

梁理论最简单的方法是经典的Euler-Bernoulli假设，即最初垂直于梁轴线的平面横截面保持平面、垂直于梁轴线且不变形。Abaqus中使用三次插值的梁单元（单元类型B23、B33等）都使用这个假设，在任意大旋转但小应变的背景下实现。Euler-Bernoulli梁单元在"Euler-Bernoulli梁单元，" 第3.5.3节中描述。这种近似也可用于为大轴向应变以及大旋转构建梁公式。Abaqus中使用线性和二次插值的梁单元（B21、B22、B31、B32、PIPE21、PIPE22、PIPE31、PIPE32等）基于这样的公式，并附加这些单元还允许"横向剪切应变"；也就是说，横截面不一定保持垂直于梁轴线。这种扩展导致Timoshenko梁理论（[Timoshenko, 1956](07s01a01-References.md)），通常被认为对较厚梁有用，其中剪切柔性可能很重要。（Abaqus中的这些单元被公式化，以便它们对薄梁——Euler-Bernoulli理论准确的地方——以及厚梁都高效：由于这个原因，它们是Abaqus中最有效的梁单元。）这些单元中的大应变公式允许任意大小的轴向应变；但相对于unity，忽略标称扭转应变中的二次项，并且在计算扭转剪切应变时假定轴向应变很小。因此，虽然轴向应变可能任意大，但只有"适度大"的扭转应变被正确建模，并且仅在轴向应变不大时才能正确建模。我们假定，在整个运动过程中，梁的曲率半径相对于横截面中的距离较大：梁不能折叠成紧密的铰链。另一个假设是，梁横截面中的应变在横截面的任何方向上以及整个截面上都是相同的。在推导这些单元时做出了一些额外的假设：这些在"梁单元公式，" 第3.5.2节中的详细推导中引入。

对于某些重要的设计，梁由薄段构造而成，形成开口截面。当材料颗粒沿平行于梁轴线的线移出截面平面，以最小化沿截面壁的线与沿梁轴线之间的剪切时，这种开口截面的响应受翘曲的强烈影响。梁单元公式（"梁单元公式，" 第3.5.2节）包括此类影响的规定。还推导了允许开口截面翘曲的梁单元（B31OS、B32OS等）。Abaqus中用于模拟开口截面翘曲的特定方法基于这样的假设：沿梁轴线任何地方的翘曲幅度都从不太大，因为翘曲将在沿梁的某些点被约束——可能是因为梁的一端或两端嵌入刚性结构，或者因为添加了某种形式的横向加劲件。

常规梁单元可用于细长和中等厚度的梁。对于极细长的梁，其长度与厚度比在![](../graphics/stm_eqn01509.gif)或以上，需要几何非线性分析（如管道），收敛可能变得非常差。对于这种情况，使用混合单元，其中轴向（和横向）力被视为独立自由度，可能是有益的。混合梁公式在"混合梁单元，" 第3.5.4节中描述。施加到梁上的分布压力载荷（如由于风或水流）将随梁旋转，导致跟随力效应。"梁单元的载荷刚度，" 第6.5.2节中给出了考虑这种效应的载荷刚度推导。

在一些管道应用中，薄壁圆形相对直的管道承受相对较大的内压。这具有在管道截面周围产生高环向应力的效果，因此，如果截面发生塑性屈服，由于与该环向应力的相互作用，轴向屈服应力在拉伸和压缩中将不同。PIPE单元通过提供由内压引起的横截面均匀径向膨胀来允许这种效应。

在其他管道情况下，薄壁直管可能承受大量弯曲，导致截面塌陷（"Brazier塌陷"）；或者管道的一段可能在其初始配置中已经是弯曲的——它可能是一个"弯头"。在这种情况下，截面 的椭圆化以及可能的翘曲可能很重要：这些效应可以将构件的弯曲刚度降低五倍或更多。对于材料线性分析，这些效应可以通过对截面弯曲刚度进行适当调整来纳入（将根据梁理论计算的弯曲刚度乘以适当的柔性系数）；但是当非线性材料响应是问题的一部分时，有必要明确地对这种椭圆化和翘曲进行建模。提供了弯头单元用于该目的；它们在"弯头单元，" 第3.9.1节中描述。弯头单元对用户看起来像梁单元，但它们包含允许椭圆化和翘曲的位移变量，因此在公式中更加复杂。特别是，截面的椭圆化意味着通过管道壁的应变相对于位置有强梯度：这需要在通过管道壁的数值积分之上，再加上围绕管道截面的积分，来捕获材料响应。这使得弯头单元在计算上比梁单元更昂贵。

由于仅考虑平面变形在公式化梁单元方面提供了相当大的简化，对于Abaqus中的每种梁单元类型，提供了仅在（![](../graphics/stm_eqn03495.gif)）平面中运动的相应梁单元。然而，出于显而易见的原因，开口截面梁仅在三维中提供。
