# 2.1 Abaqus 子结构模型





在 MSC.ADAMS 中考虑组件柔性的第一步是将该组件建模为 Abaqus 子结构。此过程涉及创建组件的 Abaqus 有限元模型。构建包含子结构的 Abaqus 模型的一般指南在 ["使用子结构" Abaqus Analysis User's Guide 第 10.1.1 节](../usb/usb-link.md#usb-anl-asuperelements) 中描述。构建可导出到 MSC.ADAMS 的子结构模型的具体要求在以下章节中描述。

Abaqus Interface for MSC.ADAMS 从由 Abaqus 分析写入的单个结果文件创建不包含应力或应变的模态中性文件，如 ["设置 Abaqus 模型以创建不包含应力或应变的模态中性文件" 第 2.2 节](ch02s02.md) 所述。如果您希望 Abaqus Interface for MSC.ADAMS 将应力或应变转换到模态中性文件，则需要第二次 Abaqus 分析，如 ["设置 Abaqus 模型以创建包含应力或应变的模态中性文件" 第 2.3 节](ch02s03.md) 中所述。
