# 9.5 动态振荡的阻尼







向模型添加阻尼有两个原因：限制数值振荡或向系统添加物理阻尼。Abaqus/Explicit 提供了多种将阻尼引入分析的方法。

### 9.5.1 体积粘度

体积粘度引入与体积应变相关的阻尼。其目的是改善高速动力学事件的建模。Abaqus/Explicit 包含线性和二次形式的体积粘度。您可以使用 [*BULK VISCOSITY*](../key/key-link.md#usb-kws-hbulkvisco) 选项从步骤到步骤将体积粘度设置为非默认值，尽管很少有必要这样做。体积粘度压力不包含在材料点应力中，因为它仅作为数值效应。因此，它不被视为材料本构响应的一部分。

### 9.5.3 材料阻尼

材料模型本身可能以塑性耗散或粘弹性形式提供阻尼。对于许多应用，这种阻尼可能就足够了。另一个选项是使用使用 [*DAMPING*](../key/key-link.md#usb-kws-mdamping) 选项定义的 Rayleigh 阻尼，它是 [*MATERIAL*](../key/key-link.md#usb-kws-mmaterial) 选项块的一部分。Rayleigh 阻尼有两个阻尼因子：![](../graphics/gsk_eqn00211.gif) 用于质量比例阻尼，![](../graphics/gsk_eqn00212.gif) 用于刚度比例阻尼。

**质量比例阻尼**

![](../graphics/gsk_eqn00211.gif) 因子定义了与单元质量矩阵成比例的阻尼贡献。引入的阻尼力是由模型中节点的绝对速度引起的。结果效果可以比作模型穿过粘性流体移动，因此模型中任何点的任何运动都会触发阻尼力。合理的质量比例阻尼不会显著降低稳定性极限。

### 9.5.4 离散阻尼器

另一个选项是定义单独的阻尼器单元。每个阻尼器单元提供与其两个节点相对速度成比例的阻尼力。这种方法的优点是它使您能够仅在您认为必要的地方施加阻尼。阻尼器应始终与其他单元（如弹簧或桁架）并联使用，这样它们不会导致稳定性极限显著降低。
