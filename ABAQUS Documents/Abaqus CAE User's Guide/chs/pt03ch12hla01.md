# 12.16 使用离散取向进行材料取向和复合铺层取向





离散取向在每个网格单元的质心处定义了一个空间变化的取向。取向可以基于部件的拓扑结构，允许您定义一个连续变化的取向。您定义法线轴和主轴，Abaqus/CAE使用这些轴来构建一个右手笛卡尔坐标系。对于二维和轴对称部件，法线轴始终是面外方向，代表坐标系的有效3轴。离散取向可用于材料取向和复合铺层取向。

有多种选择方法可用于定义所需的轴。基准轴和矢量值方法定义恒定的轴方向，而区域选择方法允许取向变化。例如，您可能希望选择一个曲面并允许法线跟随曲线的形状。

Abaqus/CAE使用您指定的法线轴和主轴以及以下算法来计算离散取向：

1. Abaqus/CAE保留法线轴方向，并使用指定的主轴方向作为构造方向。如果指定的主轴方向和法线轴方向形成90度角，最终主轴方向将与指定的方向相同。否则，将调整主轴方向以形成笛卡尔系统。
2. 如果您通过选择表面或面来定义法线轴，Abaqus/CAE会找到表面上或面上最近的点，并使用该点的表面法线作为法线轴方向（![](../graphics/usi_eqn00397.gif)）。
3. 如果您通过选择边来定义主轴，Abaqus/CAE会找到边上最近的点，并使用该点的边切线作为主轴构造方向（![](../graphics/usi_eqn00398.gif)）。
4. Abaqus/CAE通过取主轴方向和法线轴方向的叉乘来计算次轴方向（![](../graphics/usi_eqn00399.gif)）（![](../graphics/usi_eqn00400.gif)）。
5. Abaqus/CAE通过取次轴方向和法线轴方向的叉乘来计算最终主轴方向（![](../graphics/usi_eqn00401.gif)）（![](../graphics/usi_eqn00402.gif)）。

**定义离散取向：**

1. 显示**Edit Discrete Orientation**对话框，执行以下操作：
   1. 分配材料取向或创建复合铺层。
      - 使用["分配材料取向或钢筋参考取向，"第12.15.4节](pt03ch12s15s01.md)中描述的过程显示**Edit Material Orientation**对话框。
      - 使用["创建和编辑复合铺层，"第12.14节](pt03ch12s14.md)中描述的任一过程显示复合铺层编辑器。
   2. 点击**Definition**字段右侧的箭头，从出现的列表中选择**Discrete**。
   3. 点击![](../graphics/ico_edit.png)。
2. 对于三维部件，选择您希望**Normal axis direction**在最终取向中代表的坐标系轴。
3. 对于三维部件，使用以下方法之一定义取向的法线轴：
   - 选择**Surface/Faces**以选择一个区域，然后执行以下操作：
     1. 点击![](../graphics/ico_selectBlue.png)。
     2. 从提示区域，选择**individually**或选择**by angle**并输入角度。有关详细信息，请参见["使用角度和特征边方法选择多个对象，"第6.2.3节](pt01ch06s02hlb03.md)。
     3. 通过在视口中选择区域或点击提示区域中的**Surfaces**或**Sets**并选择现有表面或集，来识别定义材料取向法线轴的区域。
   - 选择**Datum axis**以选择基准几何体，然后执行以下操作：
     1. 要创建基准轴，点击![](../graphics/ico_createDtmAxis.png)并在视口中选择定义轴的两个点。
     2. 点击![](../graphics/ico_selectBlue.png)。
     3. 选择定义材料取向法线轴的基准轴。
   - 选择**Vector (i,j,k)**，并输入矢量分量的值。Abaqus/CAE在视口中显示箭头以指示您的选择和轴方向；例如，标记为`N-3`的箭头。如有必要，您可以点击**Flip Direction**来获得所需的方向。
4. 选择您希望**Primary axis direction**在最终取向中代表的坐标系轴。
5. 使用以下方法之一定义取向的主轴：
   - 选择**Edges**以选择一个区域，然后执行以下操作：
     1. 点击![](../graphics/ico_selectBlue.png)。
     2. 从提示区域，选择**individually**或选择**by edge angle**并输入角度。
     3. 通过在视口中选择区域或点击提示区域中的**Sets**并选择现有集，来识别定义材料取向主轴的区域。
   - 选择**Datum axis**以选择基准几何体，然后执行以下操作：
     1. 要创建基准轴，点击![](../graphics/ico_createDtmAxis.png)并在视口中选择定义轴的两个点。
     2. 点击![](../graphics/ico_selectBlue.png)。
     3. 选择定义材料取向主轴的基准轴。
   - 选择**Vector (i,j,k)**，并输入矢量分量的值。Abaqus/CAE在视口中显示箭头以指示您的选择和轴方向；例如，标记为`P-1`的箭头。如有必要，您可以点击![](../graphics/ico_flipArrow.png)来获得所需的方向。
6. 点击**Continue**保存离散取向定义并关闭**Edit Discrete Orientation**对话框。
