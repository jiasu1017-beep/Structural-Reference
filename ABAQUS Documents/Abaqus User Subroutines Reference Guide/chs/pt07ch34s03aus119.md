### 概述

边界条件：
- 用于规定流体动力学计算涉及的所有原始变量的值（例如，速度、温度、湍流变量、壁面法向距离等）；
- 可以作为"历史"输入数据给出（在分析步中）来添加、修改或移除零值或非零边界条件；并且
- 可以通过协同仿真区域为多物理场问题规定。

计算流体动力学问题通常需要规定多个变量如压力、温度和速度的边界条件。实际上，边界条件往往组合在一起共同定义一种物理行为；例如，壁面处的无滑移/不可穿透条件。相比之下，Neumann条件（如规定热通量）被规定为载荷（见["规定表面分布式热通量" in "热载荷，" 第34.4.4节"](pt07ch34s04aus123.md#usb-prc-pthermal-sbdistheatflux)）。在没有规定边界条件或载荷的情况下，Abaqus/CFD的默认行为是强制执行齐次（零）Neumann条件。例如，如果在壁面处未指定温度，则默认行为是自动指定完全绝热边界；即，零法向热通量。类似地，如果未规定速度，则速度的法向导数设置为零。

表示物理类型（例如，流入、流出或壁面行为）的边界条件组合被分组以便使用。关于Abaqus/CAE分组的更多信息，请参阅["使用边界条件编辑器，" Abaqus/CAE用户指南第16.10节"](../usi/usi-link.md#usi-lbi-bceditors)。或者，您可以在表面、元素面和节点上规定边界条件，包括空间变化条件。

### 基于物理类型规定流入和流出边界条件

您可以指定边界条件来描述流体进入分析域（速度入口）和流体离开分析域（压力出口）处的流动行为。

#### 流入边界条件

流入边界条件用于描述流体进入分析域的表面处的流动行为。对于不可压缩流动，可以为速度或压力、温度和湍流变量规定流入条件。如果未明确指定变量的边界条件，则自动假定齐次Neumann条件。这对应于允许变量（例如温度）在流入处变化，传入的流体对应于该局部变量。类似地，如果未指定压力，则其法向导数在流入表面处自动设置为零。可以独立规定速度分量。

您可以指定幅值曲线（见["幅值曲线，" 第34.1.2节"](pt07ch34s01aus115.md)）来定义流入行为变化的时间变化。

| **输入文件用法：** | 使用以下选项在表面上定义流入边界条件： |
| --- | --- |
|  | ``` [*FLUID BOUNDARY*](../key/key-link.md#usb-kws-hfluidboundary), TYPE=PHYSICAL, VELOCITYINLET, SURFACE=*surface name* *boundary condition type label*, *magnitude*, *amplitude name* ``` 其中*boundary condition type label*是VELX、VELY、VELZ、VELN、VELXNU、VELYNU、VELZNU、TEMP、TURBKE、TURBEPS、TURBOMEGA、TURBNU、TURBINTENSITY、TURBLENGTHSCALE或TURBVISCOSITYRATIO。 |

| **Abaqus/CAE用法：** | 使用以下选项在表面上定义流入边界条件： |
| --- | --- |
|  | 载荷模块：**创建边界条件**：**步**：***flow_step***：**类别**：**流体**：**流体入口/出口**：选择入口区域；在入口或出口处指定动量（压力或速度）、热能（温度）和湍流条件 |
|  | Abaqus/CAE不支持指定比能耗散率、湍流强度、湍流长度尺度和涡与分子粘度比。 |
| --- | --- |

#### 流出边界条件

流出边界对应于流体流动离开分析域的表面。在Abaqus/CFD中，流出条件与指定压力相关联。但是，也可以在流出边界规定许多其他流动变量。当变量未指定时，其法向导数假定为零。因此，对流流出将其量带出域，固定在某个水平上，从而产生本质上非反射的边界。

您可以指定幅值曲线（见["幅值曲线，" 第34.1.2节"](pt07ch34s01aus115.md)）来定义流出行为变化的时间变化。

| **输入文件用法：** | 使用以下选项在表面上定义流出压力边界条件： |
| --- | --- |
|  | ``` [*FLUID BOUNDARY*](../key/key-link.md#usb-kws-hfluidboundary), TYPE=PHYSICAL, PRESSUREOUTLET, SURFACE=*surface name* *boundary condition type label*, *magnitude*, *amplitude name* ``` 其中*boundary condition type label*是P、PNU、PASSIVEOUTFLOW、TEMP、TURBKE、TURBEPS、TURBOMEGA、TURBNU、TURBINTENSITY、TURBLENGTHSCALE或TURBVISCOSITYRATIO。对于PASSIVEOUTFLOW，*magnitude*和*amplitude name*的值被忽略。 |

| **Abaqus/CAE用法：** | 使用以下选项在表面上定义流出边界条件： |
| --- | --- |
|  | 载荷模块：**创建边界条件**：**步**：***flow_step***：**类别**：**流体**：**流体入口/出口**：选择出口区域；在入口或出口处指定动量（压力或速度）、热能（温度）和湍流条件 |
|  | Abaqus/CAE不支持指定比能耗散率、湍流强度、湍流长度尺度和涡与分子粘度比。 |
| --- | --- |

### 基于物理类型规定壁面边界条件

壁面边界条件通常与固体表面处的无滑移/不可穿透行为相关联。但是，固体壁面处的行为也可能需要规定温度和（取决于流动条件）湍流变量。如果需要壁面热通量，除了壁面边界条件外，还必须规定热通量载荷。

根据壁面的物理特性，可以修改壁面边界条件以实现各种物理行为，包括滑动、无滑移、渗透、对称等。

#### 无滑移/不可穿透壁面边界条件

无滑移（且不可穿透）壁面是流体粘附于壁面而不穿透它的表面。壁面处的速度分量全部设置为零，除非壁面在移动。不同湍流变量的边界条件由求解器自动实现。

您可以指定幅值曲线（见["幅值曲线，" 第34.1.2节"](pt07ch34s01aus115.md)）来定义壁面边界条件变化的时间变化。

| **输入文件用法：** | 使用以下选项在表面上定义无滑移/不可穿透壁面边界条件： |
| --- | --- |
|  | ``` [*FLUID BOUNDARY*](../key/key-link.md#usb-kws-hfluidboundary), TYPE=PHYSICAL, WALL, SURFACE=*surface name*, SLIP=NO (default) *boundary condition type label*, *magnitude*, *amplitude name* ``` 其中*boundary condition type label*是TEMP。 |

| **Abaqus/CAE用法：** | 使用以下选项在表面上定义无滑移/不可穿透壁面边界条件： |
| --- | --- |
|  | 载荷模块：**创建边界条件**：**步**：***flow_step***：**类别**：**流体**：**流体壁面条件**：选择区域；选择**条件**：**无滑移**；在壁面处指定热能（温度）和湍流条件 |
| --- | --- |

#### 滑动壁面边界条件

滑动壁面是流体不粘附于壁面且不能穿透它的表面。该壁面条件通过指定壁面法向流体速度等于壁面速度（如果壁面不移动则为零）来建模。这种情况也代表流体流动的对称条件，因为面内速度可以变化，但面外速度为零。当考虑移动边界时，必须结合表面流体速度规定相应的网格位移边界条件，以获得适当的行为。如果指定了湍流模型，则必须在壁面处将壁面法向距离边界条件设置为零。

| **输入文件用法：** | 使用以下选项在表面上定义滑动壁面边界条件： |
| --- | --- |
|  | ``` [*FLUID BOUNDARY*](../key/key-link.md#usb-kws-hfluidboundary), TYPE=PHYSICAL, WALL, SURFACE=*surface name*, SLIP=YES *boundary condition type label*, *magnitude*, *amplitude name* ``` 其中*boundary condition type label*是VELX、VELY、VELZ、VELXNU、VELYNU、VELZNU、TEMP、TURBKE、TURBEPS或TURBOMEGA。例如，对于使用Spalart-Allmaras湍流模型（壁面处湍流涡粘度和壁面法向距离边界条件设置为零）的非移动滑动壁，使用以下设置： ``` [*FLUID BOUNDARY*](../key/key-link.md#usb-kws-hfluidboundary), TYPE=PHYSICAL, WALL, SURFACE=`Surface-1`, SLIP=YES VELX, 1.0 VELY, 0.0 VELZ, 0.0 ``` |

| **Abaqus/CAE用法：** | 使用以下选项在表面上定义滑动壁面边界条件： |
| --- | --- |
|  | 载荷模块：**创建边界条件**：**步**：***flow_step***：**类别**：**流体**：**流体壁面条件**：选择区域；选择**条件**：**剪切**；在壁面处指定速度、热能（温度）和湍流条件 |
|  | Abaqus/CAE不支持指定比能耗散率。 |
| --- | --- |

##### 渗透壁

表面处的渗透允许流体穿透表面同时保持无滑移条件。该壁面条件通过指定壁面法向速度等于表示渗透速度的速度来建模，而壁面切向流体速度等于壁面速度（如果壁面不移动则为零）。在实现湍流模型的特殊情况下，必须在壁面处将壁面法向距离边界条件设置为零。如果启用了Spalart-Allmaras湍流模型，您可以指定允许在壁面处由于渗透而存在的Spalart-Allmaras湍流涡粘度 ![](../graphics/usb_eqn00857.gif) 的值。如果实现了*k*–![](../graphics/usb_eqn00856.gif) RNG、*k*–![](../graphics/usb_eqn00856.gif) 可实现或*k*–![](../graphics/usb_eqn00348.gif) SST模型，则可以在壁面处规定湍动能*k*、能耗散率 ![](../graphics/usb_eqn00856.gif) 和比能耗散率 ![](../graphics/usb_eqn00348.gif) 的值。

##### 规定温度

可以在壁面处规定温度。默认情况下，如果未在壁面处规定温度，则自动指定完全绝热边界。对于多物理场应用如共轭热传递，使用协同仿真区域自动施加可变温度条件（更多信息，请参阅["准备协同仿真的Abaqus分析，" 第17.2.1节"](pt04ch17s02aus98.md)）。

边界条件的规定幅值可以随步中时间按照幅值定义变化（["幅值曲线，" 第34.1.2节"](pt07ch34s01aus115.md)）。

| **输入文件用法：** | 使用以下两个选项定义移动边界处的规定位移： |
| --- | --- |
|  | ``` [*AMPLITUDE*](../key/key-link.md#usb-kws-mamplitude), NAME=*name* [*BOUNDARY*](../key/key-link.md#usb-kws-hboundary), AMPLITUDE=*name* ``` 使用以下两个选项定义随时间变化的流入、流出和壁面边界条件： ``` [*AMPLITUDE*](../key/key-link.md#usb-kws-mamplitude), NAME=*name* [*FLUID BOUNDARY*](../key/key-link.md#usb-kws-hfluidboundary), AMPLITUDE=*name* ``` |

| **Abaqus/CAE用法：** | 载荷或交互模块：**创建幅值**：**名称**：*amplitude_name* |
|  | 载荷模块：**创建边界条件**：**步：*flow_step***：***boundary condition***；**幅值**：*amplitude_name* |

### 规定位移

Abaqus/CFD提供了使用任意拉格朗日-欧拉（ALE）方法进行流体流动的变形网格和流体-结构相互作用（FSI）模拟的能力。对于FSI和变形网格问题，通常部分流体域会与边界运动一致的变形。为了管理网格运动，您必须在网格上规定位移边界条件。对于FSI问题，不允许在协同仿真区域规定位移边界条件，因为这些条件是自动规定的。

| **输入文件用法：** | ``` [*BOUNDARY*](../key/key-link.md#usb-kws-hboundary) *node or node set*, *first degree of freedom*, *last degree of freedom*, *magnitude* ``` |
| --- | --- |
|  | 其中*first degree of freedom*为1表示*x*位移，2表示*y*位移，3表示*z*位移。 |

| **Abaqus/CAE用法：** | 载荷模块：**创建边界条件**：**步**：***flow_step***：**类别**：**机械**：**位移/转角**：选择区域并切换自由度 |
| --- | --- |

### 边界条件传播

默认情况下，在先前通用分析步中定义的所有边界条件在后续通用步中保持不变。您定义的是相对于先前存在的边界条件在给定步中有效的边界条件。在每个新步中，可以修改现有边界条件并指定其他边界条件。或者，您可以释放步中所有先前应用的边界条件并指定新的边界条件。在这种情况下，必须重新指定要保留的任何边界条件。

#### 修改边界条件

当您修改现有边界条件时，必须以与先前完全相同的方式指定节点或节点集。例如，如果在一个步中为节点集指定了边界条件，而在另一个步中为包含在该集中的单个节点指定了边界条件，Abaqus会发出错误。您必须移除边界条件并重新指定才能更改节点或节点集的指定方式。

| **输入文件用法：** | 使用以下选项之一修改现有边界条件或指定附加边界条件： |
| --- | --- |
|  | ``` [*BOUNDARY*](../key/key-link.md#usb-kws-hboundary) [*BOUNDARY*](../key/key-link.md#usb-kws-hboundary), OP=MOD [*FLUID BOUNDARY*](../key/key-link.md#usb-kws-hfluidboundary) [*FLUID BOUNDARY*](../key/key-link.md#usb-kws-hfluidboundary), OP=MOD ``` |

| **Abaqus/CAE用法：** | 载荷模块：**创建边界条件**或**边界条件管理器**：**编辑** |
| --- | --- |

#### 移除边界条件

如果您选择在某个步中移除任何边界条件，则不会从先前的通用步传播边界条件。在这种情况下，必须在此步中重新指定有效的所有边界条件。

将边界条件设置为零与移除它不一样。

| **输入文件用法：** | 使用以下选项之一释放所有先前应用的边界条件并指定新的边界条件： |
| --- | --- |
|  | ``` [*BOUNDARY*](../key/key-link.md#usb-kws-hboundary), OP=NEW ``` 如果在任何[*BOUNDARY*](../key/key-link.md#usb-kws-hboundary)选项上使用了OP=NEW参数，则必须在该步中的所有[*BOUNDARY*](../key/key-link.md#usb-kws-hboundary)选项上使用它。 ``` [*FLUID BOUNDARY*](../key/key-link.md#usb-kws-hfluidboundary), OP=NEW ``` 如果在任何[*FLUID BOUNDARY*](../key/key-link.md#usb-kws-hfluidboundary)选项上使用了OP=NEW参数，则必须在该步中的所有[*FLUID BOUNDARY*](../key/key-link.md#usb-kws-hfluidboundary)选项上使用它。 |

| **Abaqus/CAE用法：** | 使用以下选项在步中移除边界条件： |
| --- | --- |
|  | 载荷模块：**边界条件管理器**：**停用** Abaqus/CAE自动重新指定在此步中应保持有效的任何边界条件。 |
