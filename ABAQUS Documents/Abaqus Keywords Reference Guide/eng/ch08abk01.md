# *HEADING







### *HEADINGPrint a heading on the output.

This option is used to define a title for the analysis.

**Products: **Abaqus/Standard  Abaqus/Explicit  Abaqus/CFD  Abaqus/CAE  

**Type: **Model data  

**Level: **Model  

**Abaqus/CAE: **Job module

##### **Reference:**

- ["Defining a model in Abaqus," Section 1.3.1 of the Abaqus Analysis User's Guide](../usb/usb-link.md#usb-int-imodel)

**There are no parameters associated with this option.**

### **Data lines to print a heading: **

**First line:**

The heading can be several lines long, but only the first 80 characters of the first line will be saved and printed as a heading.
