# 44.2 云纹图绘图选项概述





您可以使用云纹图绘图选项来自定义云纹图的外观。从主菜单栏选择****选项**![](../graphics/images/arrow.gif)**云纹****，或点击工具箱中的![](../graphics/ico_plotContourOptions.png)来访问**云纹图绘图选项**对话框。点击以下选项卡来自定义当前视口中云纹图的外观：
- **基本**：选择云纹类型（包括是否为线单元显示刻度线）、云纹间隔和云纹方法。
- **颜色和样式**：**颜色和样式**页面包含以下选项卡：- **模型边缘**：控制模型边缘的颜色。- **光谱**：选择云纹颜色。- **线条**：对于线型云纹，控制每条线的样式和厚度。- **带状/等值面**：对于带状或等值面型云纹，控制云纹边缘的颜色、样式和厚度。
- **限制**：控制云纹限制的计算以及最大和最小云纹值注释的显示。
- **其他**：控制刻度线图显示选项、节点平均向量或张量方向的显示，以及截面点或包络图显示。

渲染样式、边缘可见性、变形比例因子和半透明度等其他选项位于[**常用绘图选项**](pt05ch43hla01.md)对话框中。如果您选择在未变形和变形形状上绘制云纹图，可以使用[**叠加绘图选项**](pt05ch43hla02.md)来自定义未变形形状的外观。有关叠加绘图的更多信息，请参见["叠加变形和未变形模型绘图，" 第43.6节](pt05ch43hla02.md)。要了解如何自定义云纹图的渲染样式和底层模型，请参见[第55章，"自定义绘图显示"](pt05ch55.md)。有关结果值的计算信息，请参见["理解结果如何计算，" 第42.6.1节](pt05ch42s04s01.md)。
