# 10.1.1 可以将哪些类型的文件导入和从Abaqus/CAE导出？





Abaqus/CAE可以读取和写入以Abaqus文件格式以及非Abaqus文件格式存储的几何数据。

### Abaqus文件格式

Abaqus/CAE可以读取和写入以下Abaqus文件格式存储的几何数据：

**Abaqus输出数据库（`*output_database_name*.odb`）**

输出数据库包含Abaqus/Standard或Abaqus/Explicit分析期间生成的数据。您可以以网格形式从输出数据库导入零件。网格零件不包含特征信息，是作为节点、单元、曲面和集合的集合从输出数据库中提取的。如果输出数据库包含多个零件实例，您可以选择要导入的零件实例。Abaqus/CAE会将每个零件实例作为单独零件导入。您可以导入未变形形状或变形形状。如果导入变形形状，您可以指定要导入的步骤和帧。

要验证网格质量，您可以显示"网格（Mesh）"模块中的零件，并从主菜单栏选择****网格（Mesh）**![](../graphics/images/arrow.gif)**验证（Verify）****。此外，您可以使用"网格（Mesh）"模块更改分配给网格的单元类型，并编辑原始网格定义。有关更多信息，请参阅["从输出数据库导入零件，"第10.7.12节"](pt02ch10s07hlb12.md)；["编辑网格工具集可以做什么？，"第64.1节"](pt06ch64s01.md)；和["分配Abaqus单元类型，"第17.5节"](pt03ch17s05.md)。

您也可以从输出数据库导入模型。导入的模型将包含表示输出数据库中每个未变形零件实例的零件，以及未变形装配的网格表示。该模型还将包含在输出数据库中定义的任何集合、曲面、材料、截面定义和梁轮廓。有关更多信息，请参阅["从输出数据库导入模型，"第10.5.3节"](pt02ch10s05s03.md)。

**Abaqus/CAE模型数据库（`*model_database_name*.cae`）**

Abaqus/CAE可以将模型从不同的Abaqus/CAE模型数据库导入到当前模型数据库中。有关从另一个模型数据库导入模型数据的更多信息，请参阅["从Abaqus/CAE模型数据库导入模型，"第10.5.1节"](pt02ch10s05s01.md)。

**Abaqus/Standard和Abaqus/Explicit输入文件**

当您提交作业进行分析时，Abaqus/CAE会生成输入文件。您可以将输入文件导入Abaqus/CAE。Abaqus/CAE会将导入输入文件中的关键词和数据行转换为新模型；但是，仅支持有限的Abaqus/Standard和Abaqus/Explicit关键词集，如["从Abaqus输入文件导入模型，"第10.5.2节"](pt02ch10s05s02.md)中所述。有关创建和提交作业的更多信息，请参阅["分析模型的基本步骤，"第19.2.1节"](pt03ch19s02s01.md)。

**Abaqus子结构文件（`*substructure_name*.sim`）**

Abaqus/CAE可以将SIM数据库中的子结构定义作为新零件定义导入。您导入的`.sim`文件必须与SIM数据库所指向的支持Abaqus文件位于同一目录中；这些支持文件可能包括`.prt`、`.mdl`、`.stt`或`.sup`格式的数据。有关分步说明，请参阅["将子结构作为零件导入模型数据库，"第10.7.13节"](pt02ch10s07hlb13.md)。

### 支持的非Abaqus文件格式

Abaqus/CAE可以读取和写入以下非Abaqus文件格式存储的几何数据：

**3D XML（`*file_name*.3dxml`）**

3D XML是Dassault Systmes开发的基于XML的格式，用于编码三维图像和数据。该格式是开放和可扩展的，允许三维图形轻松共享和集成到现有应用程序和流程中。3D XML文件可以比典型模型数据库文件小很多倍。查看3D XML文件或将其集成到业务应用程序中需要Dassault Systmes的3D XML Player。您也可以在CATIA V5中查看3D XML文件。此导出功能不能用于将几何体或模型传输到3DEXPERIENCE平台应用程序。

您可以将视口数据从Abaqus/CAE导出为3D XML或压缩3D XML格式。有关更多信息，请参阅["将视口数据导出为3D XML格式文件，"第10.9.5节"](pt02ch10s08hlb05.md)。您无法将3D XML导入Abaqus/CAE。

**ACIS（`*file_name*.sat`）**

ACIS是Spatial开发的实体建模函数库，大多数CAD产品都可以生成ACIS格式的零件。您可以导入ACIS格式的零件，也可以以ACIS格式导出零件或装配。此外，您可以导入和导出ACIS格式的草图。有关更多信息，请参阅["从ACIS格式文件导入零件，"第10.7.4节"](pt02ch10s07hlb04.md)；["导入草图，"第10.7.1节"](pt02ch10s07hlb01.md)；和["导出几何、模型和网格数据，"第10.9节"](pt02ch10s08.md)。

**ANSYS输入文件（`*file_name*.cdb`）**

ANSYS Mechanical和ANSYS Multiphysics软件是计算机辅助工程产品，允许您执行有限元分析和计算流体动力学。您可以将ANSYS输入文件格式的模型导入Abaqus/CAE。有关更多信息，请参阅["从ANSYS输入文件导入模型，"第10.5.5节"](pt02ch10s05s05.md)和["导入模型，"第10.8节"](pt02ch10hla01.md)。

**装配文件（`*file_name*.eaf`）**

装配文件由关联接口应用程序创建，这些应用程序是第三方CAD系统的插件，允许您使用称为关联导入的技术将模型从CAD系统传输到Abaqus/CAE（请参阅["关联接口可以做什么？，"第10.1.2节"](pt02ch10s01s02.md))。关联接口插件将模型信息保存在装配文件中，您可以使用装配文件将模型从第三方CAD系统关联导入到Abaqus/CAE。有关更多信息，请参阅["从装配文件导入装配，"第10.7.14节"](pt02ch10s07hlb14.md)。您无法以装配文件格式从Abaqus/CAE导出装配。

**AutoCAD（`*file_name*.dxf`）**

存储在AutoCAD（`.dxf`）文件中的二维轮廓可以作为独立草图导入。但是，Abaqus/CAE仅支持有限的AutoCAD实体，并且您应该仅在没有其他格式可用时才使用此格式。有关更多信息和Abaqus/CAE支持的AutoCAD实体的详细信息，请参阅["导入草图，"第10.7.1节"](pt02ch10s07hlb01.md)。

**CATIA V4（`*file_name*.model`、`*file_name*.catdata`或`*file_name*.exp`）**

您可以导入CATIA格式的零件。您也可以将整个CATIA V4装配导入到Abaqus/CAE装配中，或者您可以选择仅导入选定的零件实例。有关更多信息，请参阅["从CATIA V4或V5格式文件导入零件，"第10.7.5节"](pt02ch10s07hlb05.md)；和["从CATIA V4格式文件导入装配，"第10.7.15节"](pt02ch10s07hlb15.md)。您无法以CATIA格式从Abaqus/CAE导出零件。

**CATIA V5 Elysium中性文件（`*file_name*.enf_abq`）**

Abaqus为CATIA V5提供了转换器插件，该插件将使用Elysium中性文件（`.enf`）格式生成几何文件。您可以使用Elysium中性文件导入CATIA V5零件。此外，您可以使用Elysium中性文件将整个CATIA V5装配导入到Abaqus/CAE装配中，或者您可以选择仅导入选定的零件实例。有关更多信息，请参阅["从Elysium中性文件导入零件，"第10.7.6节"](pt02ch10s07hlb06.md)；和["从Elysium中性文件导入装配，"第10.7.16节"](pt02ch10s07hlb16.md)。您无法以Elysium中性文件格式从Abaqus/CAE导出零件或装配。

**CATIA V5零件和装配（`*file_name*.CATPart`或`.CATProduct`）**

通过Abaqus/CAE的可选CATIA V5关联接口插件功能，您可以导入CATIA V5格式的零件和装配。有关更多信息，请参阅["从CATIA V4或V5格式文件导入零件，"第10.7.5节"](pt02ch10s07hlb05.md)。您无法以CATIA V5格式从Abaqus/CAE导出零件。

**CATIA V6零件和装配（`*file_name*.CATPart`或`.CATProduct`）**

您使用CATIA V6关联接口导入CATIA V6格式的零件和装配。CATIA V6零件和装配首先被转换为CATIA V5格式，然后Abaqus/CAE导入生成的CATIA V5 `CATPart`或`CATProduct`文件。有关更多信息，请参阅["从CATIA V4或V5格式文件导入零件，"第10.7.5节"](pt02ch10s07hlb05.md)。您无法导出到CATIA V6。

**IGES（`*file_name*.igs`或`.iges`）**

初始图形交换规范（IGES）是为计算机辅助设计（CAD）系统之间的图形交换而设计的中立数据格式。

您可以导入IGES格式的零件，也可以以IGES格式导出零件。此外，您可以导入和导出IGES格式的草图有关更多信息，请参阅["从IGES格式文件导入零件，"第10.7.7节"](pt02ch10s07hlb07.md)；["导入草图，"第10.7.1节"](pt02ch10s07hlb01.md)；和["导出几何、模型和网格数据，"第10.9节"](pt02ch10s08.md)。

IGES格式允许许多解释，您使用IGES格式导入Abaqus/CAE的大多数零件都需要在使用前进行修复。因此，建议您尽可能尝试使用其他格式。

**Nastran输入文件（`*file_name*.bdf`、`*file_name*.dat`、`*file_name*.nas`、`*file_name*.nastran`、`*file_name*.blk`或`*file_name*.bulk`）**

您可以将Nastran模型数据从Nastran输入文件导入Abaqus/CAE，也可以将数据从Abaqus/CAE模型和作业导出为Nastran批量数据格式。导入和导出的模型包括Nastran批量数据中的许多常见实体。有关将Nastran输入文件导入Abaqus/CAE所支持的实体的更多信息，请参阅["将Nastran批量数据文件转换为Abaqus输入文件，"Abaqus Analysis User's Guide第3.2.28节"](../usb/usb-link.md#usb-int-dnasabaproc)。有关将Abaqus/CAE作业和模型导出为Nastran所支持的实体的更多信息，请参阅["将Abaqus文件转换为Nastran批量数据文件，"Abaqus Analysis User's Guide第3.2.29节"](../usb/usb-link.md#usb-int-dtonasabaproc)。

**NX Elysium中性文件（`*file_name*.enf_abq`）**

Abaqus为NX提供了转换器插件，该插件将使用Elysium中性文件（`.enf`）格式生成几何文件。您可以使用Elysium中性文件导入NX零件。此外，您可以使用Elysium中性文件将整个NX装配导入到Abaqus/CAE装配中，或者您可以选择仅从装配中导入选定的零件实例有关更多信息，请参阅["从Elysium中性文件导入零件，"第10.7.6节"](pt02ch10s07hlb06.md)；和["从Elysium中性文件导入装配，"第10.7.16节"](pt02ch10s07hlb16.md)。您无法以Elysium中性文件格式从Abaqus/CAE导出零件或装配。

**OBJ（`*file_name*.obj`）**

OBJ是一种开放的文件格式，用于描述几何体，涉及其顶点位置、顶点之间的边以及构成几何体中每个多边形的面。OBJ格式的数据保存为文本文件。

您可以将几何数据或网格数据从Abaqus/CAE导出为OBJ格式有关更多信息，请参阅["将视口数据导出为OBJ格式文件，"第10.9.6节"](pt02ch10s08hlb06.md)。您无法将OBJ格式的数据导入Abaqus/CAE。

**Parasolid（`*file_name*.x_t`、`*file_name*.x_b`、`*file_name*.xmt_txt`或`*file_name*.xmt_bin`）**

Parasolid是UGS开发的实体建模函数库。各种CAD产品可以生成Parasolid格式的零件，如NX、SolidWorks、Solid Edge、FEMAP和MSC.Patran。您可以导入Parasolid格式的零件。您也可以将整个Parasolid装配导入到Abaqus/CAE装配中，或者您可以选择仅导入选定的零件实例有关更多信息，请参阅["从Parasolid格式文件导入零件，"第10.7.9节"](pt02ch10s07hlb09.md)；和["从Parasolid格式文件导入装配，"第10.7.17节"](pt02ch10s07hlb17.md)。您无法以Parasolid格式从Abaqus/CAE导出零件或装配。

**Pro/ENGINEER Elysium中性文件（`*file_name*.enf_abq`）**

Abaqus为Pro/ENGINEER提供了转换器插件，该插件将使用Elysium中性文件（`.enf`）格式生成几何文件。您可以使用Elysium中性文件导入Pro/ENGINEER零件。此外，您可以使用Elysium中性文件将整个Pro/ENGINEER装配导入到Abaqus/CAE装配中，或者您可以选择仅从装配中导入选定的零件实例有关更多信息，请参阅["从Elysium中性文件导入零件，"第10.7.6节"](pt02ch10s07hlb06.md)；和["从Elysium中性文件导入装配，"第10.7.16节"](pt02ch10s07hlb16.md)。您无法以Elysium中性文件格式从Abaqus/CAE导出零件或装配。

**STEP（`*file_name*.stp`或`.step`）**

产品模型数据交换标准（STEP ISO 10303–1）被设计为IGES的高级替代品，旨在克服IGES的一些缺点。STEP AP203标准旨在为整个生命周期中的机械产品提供计算机可解释的表示，独立于任何特定系统。

您可以导入STEP格式的零件，也可以以STEP格式导出零件。此外，您可以导入和导出STEP格式的草图有关更多信息，请参阅["从STEP格式文件导入零件，"第10.7.10节"](pt02ch10s07hlb10.md)；和["导出几何、模型和网格数据，"第10.9节"](pt02ch10s08.md)。

STEP格式的零件与IGES格式的零件类似，您使用STEP格式导入Abaqus/CAE的大多数零件都需要在使用前进行修复。因此，建议您尽可能尝试使用其他格式。

**VDA-FS（`*file_name*.vda`）**

德国汽车行业开发的几何标准VDA-FS（Verband der Automobilindustrie Flchen Schnittstelle）表面数据格式。VDA-FS和IGES文件都包含零件的数学表示为ASCII格式；然而，VDA-FS标准专注于几何信息。IGES标准涵盖的其他信息（如尺寸、文本和颜色）不存储在VDA-FS文件中。

您可以导入VDA-FS格式的零件，也可以以VDA-FS格式导出零件有关更多信息，请参阅["从VDA-FS格式文件导入零件，"第10.7.11节"](pt02ch10s07hlb11.md)；和["导出几何、模型和网格数据，"第10.9节"](pt02ch10s08.md)。

VDA-FS格式的零件与IGES格式的零件类似，您使用VDA-FS格式导入Abaqus/CAE的大多数零件都需要在 사용前进行修复。因此，建议您尽可能尝试使用其他格式。

**VRML（`*file_name*.wrl`）**

虚拟现实建模语言（VRML）是用于在Web浏览器或独立VRML客户端中显示三维图像的ISO标准。它是一种开放的、平台无关的、基于矢量的三维建模语言，对计算机生成的图形进行编码，以便在网络上轻松共享。VRML文件对所有长度和距离使用米为单位。VRML格式文件可以比典型模型数据库文件小很多倍。查看VRML文件需要特殊的插件浏览器，如Cortona或Cosmo。

您可以将视口数据从Abaqus/CAE导出为VRML格式或压缩VRML格式有关更多信息，请参阅["将视口数据导出为VRML格式文件，"第10.9.4节"](pt02ch10s08hlb04.md)。
