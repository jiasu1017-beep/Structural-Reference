# 18.1.2 可用的用户子程序





**产品：** Abaqus/Standard  Abaqus/Explicit  Abaqus/CFD  Abaqus/Aqua  

##### **参考文献**

- ["用户子程序：概述，" 第 18.1.1 节](pt04ch18s01aus104.md)
- [Abaqus 用户子程序参考指南](../sub/sub-link.md#sub)

### 概述

用户子程序允许高级用户自定义 Abaqus 的各种功能。关于编写用户子程序的信息以及每个子程序的详细说明，请参阅 [Abaqus 用户子程序参考指南](../sub/sub-link.md#sub)。该指南中还提供了相关实用程序的列表和说明。

### Abaqus/Standard 可用的用户子程序

 Abaqus/Standard 可用的用户子程序如下：
- [`CREEP`](../sub/sub-link.md#sub-xsl-creep)：定义时间相关的粘塑性行为（蠕变和膨胀）。
- [`DFLOW`](../sub/sub-link.md#sub-xsl-dflow)：在固结分析中定义非均匀孔隙流体速度。
- [`DFLUX`](../sub/sub-link.md#sub-xsl-dflux)：在热传递或质量扩散分析中定义非均匀分布通量。
- [`DISP`](../sub/sub-link.md#sub-xsl-disp)：指定边界条件。
- [`DLOAD`](../sub/sub-link.md#sub-xsl-dload)：指定非均匀分布载荷。
- [`FILM`](../sub/sub-link.md#sub-xsl-film)：定义热传递分析中的非均匀膜系数和相关热沉温度。
- [`FLOW`](../sub/sub-link.md#sub-xsl-flow)：定义固结分析中的非均匀渗流系数和相关热沉孔隙压力。
- [`FRIC`](../sub/sub-link.md#sub-xsl-fric)：定义接触面的摩擦行为。
- [`FRIC_COEF`](../sub/sub-link.md#sub-xsl-fric_coef)：定义接触面的摩擦系数。
- [`GAPCON`](../sub/sub-link.md#sub-xsl-gapcon)：在全耦合温度-位移分析、全耦合热-电-结构分析或纯热传递分析中，定义接触面或节点之间的电导。
- [`GAPELECTR`](../sub/sub-link.md#sub-xsl-gapelectr)：在全耦合热-电分析或全耦合热-电-结构分析中，定义表面之间的电导。
- [`HARDINI`](../sub/sub-link.md#sub-xsl-hardini)：定义初始等效塑性应变和初始背应力张量。
- [`HETVAL`](../sub/sub-link.md#sub-xsl-hetval)：在热传递分析中提供内部热生成。
- [`MPC`](../sub/sub-link.md#sub-xsl-mpc)：定义多点约束。
- [`ORIENT`](../sub/sub-link.md#sub-xsl-orient)：为定义局部材料方向、局部运动耦合约束方向或局部刚体方向提供方向定义。
- [`RSURFU`](../sub/sub-link.md#sub-xsl-rsurfu)：定义一个刚性表面。
- [`SDVINI`](../sub/sub-link.md#sub-xsl-sdvini)：定义初始解相关状态变量场。
- [`SIGINI`](../sub/sub-link.md#sub-xsl-sigini)：定义初始应力场。
- [`UAMP`](../sub/sub-link.md#sub-xsl-uamp)：指定振幅。
- [`UANISOHYPER_INV`](../sub/sub-link.md#sub-xsl-uanisohyper_inv)：使用不变量公式定义各向异性超弹性材料行为。
- [`UANISOHYPER_STRAIN`](../sub/sub-link.md#sub-xsl-uanisohyper_strain)：基于 Green 应变定义各向异性超弹性材料行为。
- [`UCORR`](../sub/sub-link.md#sub-xsl-ucorr)：定义随机响应载荷的互相关属性。
- [`UCREEPNETWORK`](../sub/sub-link.md#sub-xsl-ucreepnetwork)：为在平行流变框架中定义的模型定义时间相关行为（蠕变）。
- [`UDECURRENT`](../sub/sub-link.md#sub-xsl-udecurrent)：在涡流或静磁分析中定义非均匀体积电流密度。
- [`UDEMPOTENTIAL`](../sub/sub-link.md#sub-xsl-udempotential)：在涡流或静磁分析中，在表面上定义非均匀磁矢势。
- [`UDMGINI`](../sub/sub-link.md#sub-xsl-udmgini)：定义损伤起始准则。
- [`UDSECURRENT`](../sub/sub-link.md#sub-xsl-udsecurrent)：在涡流或静磁分析中定义非均匀表面电流密度。
- [`UEL`](../sub/sub-link.md#sub-xsl-uel)：定义一个单元。
- [`UELMAT`](../sub/sub-link.md#sub-xsl-uelmat)：定义可访问 Abaqus 材料的单元。
- [`UEXPAN`](../sub/sub-link.md#sub-xsl-uexpan)：定义增量热应变。
- [`UEXTERNALDB`](../sub/sub-link.md#sub-xsl-uexternaldb)：管理用户定义的外部数据库并计算与模型无关的历史信息。
- [`UFIELD`](../sub/sub-link.md#sub-xsl-ufield)：指定预定义场变量。
- [`UFLUID`](../sub/sub-link.md#sub-xsl-ufluid)：为静水力流体单元定义流体密度和流体压缩性。
- [`UFLUIDLEAKOFF`](../sub/sub-link.md#sub-xsl-ufluidleakoff)：为孔隙压力内聚单元定义流体泄漏系数。
- [`UGENS`](../sub/sub-link.md#sub-xsl-ugens)：定义壳截面的力学行为。
- [`UHARD`](../sub/sub-link.md#sub-xsl-uhard)：为各向同性塑性或组合硬化模型定义屈服面大小和硬化参数。
- [`UHYPEL`](../sub/sub-link.md#sub-xsl-uhypel)：定义一个次弹性应力-应变关系。
- [`UHYPER`](../sub/sub-link.md#sub-xsl-uhyper)：定义一个超弹性材料。
- [`UINTER`](../sub/sub-link.md#sub-xsl-uinter)：定义接触面的表面相互作用行为。
- [`UMASFL`](../sub/sub-link.md#sub-xsl-umasfl)：为对流/扩散热传递分析指定质量流率条件。
- [`UMAT`](../sub/sub-link.md#sub-xsl-umat)：定义材料的力学行为。
- [`UMATHT`](../sub/sub-link.md#sub-xsl-umatht)：定义材料的热行为。
- [`UMESHMOTION`](../sub/sub-link.md#sub-xsl-umeshmotion)：在自适应网格划分期间指定网格运动约束。
- [`UMOTION`](../sub/sub-link.md#sub-xsl-umotion)：在腔体辐射热传递分析或稳态传输分析中指定运动。
- [`UMULLINS`](../sub/sub-link.md#sub-xsl-umullins)：为 Mullins 效应材料模型定义损伤变量。
- [`UPOREP`](../sub/sub-link.md#sub-xsl-uporep)：定义初始流体孔隙压力。
- [`UPRESS`](../sub/sub-link.md#sub-xsl-upress)：指定等效压力应力条件。
- [`UPSD`](../sub/sub-link.md#sub-xsl-upsd)：定义随机响应载荷的频率依赖性。
- [`URDFIL`](../sub/sub-link.md#sub-xsl-urdfil)：读取结果文件。
- [`USDFLD`](../sub/sub-link.md#sub-xsl-usdfld)：在材料点重新定义场变量。
- [`UTEMP`](../sub/sub-link.md#sub-xsl-utemp)：指定温度条件。
- [`UTRACLOAD`](../sub/sub-link.md#sub-xsl-utracload)：指定非均匀牵引载荷。
- [`UTRS`](../sub/sub-link.md#sub-xsl-utrs)：为粘弹性材料定义缩减时间偏移函数。
- [`UTRSNETWORK`](../sub/sub-link.md#sub-xsl-utrsnetwork)：为在平行流变框架中定义的模型定义缩减时间偏移函数。
- [`UVARM`](../sub/sub-link.md#sub-xsl-uvarm)：生成单元输出。
- [`UWAVE`](../sub/sub-link.md#sub-xsl-uwave)：为 Abaqus/Aqua 分析定义波浪运动学。
- [`UXFEMNONLOCALWEIGHT`](../sub/sub-link.md#sub-xsl-uxfemnonlocalweight)：定义用于计算平均应力/应变以确定裂纹扩展方向的权函数。
- [`VOIDRI`](../sub/sub-link.md#sub-xsl-voidri)：定义初始孔隙比。

### Abaqus/Explicit 可用的用户子程序

Abaqus/Explicit 可用的用户子程序如下：
- [`VDISP`](../sub/sub-link.md#sub-xsl-vdisp)：指定边界条件。
- [`VDLOAD`](../sub/sub-link.md#sub-xsl-vdload)：指定非均匀分布载荷。
- [`VEXTERNALDB`](../sub/sub-link.md#sub-xsl-vexternaldb)：控制 Abaqus 用户子程序与外部程序或文件之间的数据交换分析。
- [`VFABRIC`](../sub/sub-link.md#sub-xsl-vfabric)：定义织物材料行为。
- [`VFRIC`](../sub/sub-link.md#sub-xsl-vfric)：定义使用接触对算法定义的表面之间的接触摩擦行为。
- [`VFRIC_COEF`](../sub/sub-link.md#sub-xsl-vfric_coef)：定义使用通用接触算法定义的表面之间的接触摩擦系数。
- [`VFRICTION`](../sub/sub-link.md#sub-xsl-vfriction)：定义使用通用接触算法定义的表面之间的接触摩擦行为。
- [`VUAMP`](../sub/sub-link.md#sub-xsl-vuamp)：指定振幅。
- [`VUANISOHYPER_INV`](../sub/sub-link.md#sub-xsl-vuanisohyper_inv)：使用不变量公式定义各向异性超弹性材料行为。
- [`VUANISOHYPER_STRAIN`](../sub/sub-link.md#sub-xsl-vuanisohyper_strain)：基于 Green 应变定义各向异性超弹性材料行为。
- [`VUCHARLENGTH`](../sub/sub-link.md#sub-xsl-vucharlength)：在材料点定义特征单元长度。
- [`VUEL`](../sub/sub-link.md#sub-xsl-vuel)：定义一个单元。
- [`VUEOS`](../sub/sub-link.md#sub-xsl-vueos)：定义状态方程材料模型。
- [`VUFIELD`](../sub/sub-link.md#sub-xsl-vufield)：指定预定义场变量。
- [`VUFLUIDEXCH`](../sub/sub-link.md#sub-xsl-vufluidexch)：定义流体交换的质量/热能流率。
- [`VUFLUIDEXCHEFFAREA`](../sub/sub-link.md#sub-xsl-vufluidexcheffarea)：定义流体交换的有效面积。
- [`VUHARD`](../sub/sub-link.md#sub-xsl-vuhard)：为各向同性塑性或组合硬化模型定义屈服面大小和硬化参数。
- [`VUINTER`](../sub/sub-link.md#sub-xsl-vuinter)：定义使用接触对算法定义的表面之间的接触相互作用。
- [`VUINTERACTION`](../sub/sub-link.md#sub-xsl-vuinteraction)：定义使用通用接触算法定义的表面之间的接触相互作用。
- [`VUMAT`](../sub/sub-link.md#sub-xsl-vumat)：定义材料行为。
- [`VUMULLINS`](../sub/sub-link.md#sub-xsl-vumullins)：为 Mullins 效应材料模型定义损伤变量。
- [`VUSDFLD`](../sub/sub-link.md#sub-xsl-vusdfld)：在材料点重新定义场变量。
- [`VUTRS`](../sub/sub-link.md#sub-xsl-vutrs)：为粘弹性材料定义缩减时间偏移函数。
- [`VUVISCOSITY`](../sub/sub-link.md#sub-xsl-vuviscosity)：为状态方程模型定义剪切粘度。
- [`VWAVE`](../sub/sub-link.md#sub-xsl-vwave)：为 Abaqus/Aqua 分析定义波浪运动学。

### Abaqus/CFD 可用的用户子程序

Abaqus/CFD 可用的用户子程序如下：
- [`SMACfdUserPressureBC`](../sub/sub-link.md#sub-xsl-smacfduserpressurebc)：指定压力边界条件。
- [`SMACfdUserVelocityBC`](../sub/sub-link.md#sub-xsl-smacfduservelocitybc)：指定速度边界条件。
