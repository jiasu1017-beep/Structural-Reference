# 4.1 Understanding viewports







Viewports are areas on the canvas where you can display models or analysis results. You can add arrow and text annotations to draw attention to or explain features within a viewport. You can create and manipulate viewports, text, and arrows using the **Viewport** menu. The following topics are covered:
- ["What is a viewport?," Section 4.1.1](pt01ch04s01s01.md)
- ["What are arrow and text annotations?," Section 4.1.2](pt01ch04s01s02.md)
