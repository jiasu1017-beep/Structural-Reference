# 3.1.3 与过程一起使用鼠标快捷方式





鼠标快捷方式可用于提示区域中发生的许多操作。要使用快捷方式，首先确保光标在当前视口中。
- 要提交提示区域中出现的任何文本字段的内容，请单击鼠标按钮2。
- 要接受提示区域中突出显示按钮描绘的任何默认选项，请单击鼠标按钮2。
- 要显示包含与提示区域中相同选项的菜单，请单击鼠标按钮3。例如，给定以下提示：
![](../graphics/int-procedure-fastprompt-nls.png)
单击鼠标按钮3会显示以下菜单：
![](../graphics/int-procedure-fastmenu-nls.png)
水平线上方的项目对应于提示区域右侧的选项按钮，而线下方的项目对应于上一步和取消按钮。
