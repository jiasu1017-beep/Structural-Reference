# 56.3 自定义状态块







状态块包含用于标识与当前绘图关联的分析结果的文本。此信息包括步骤、帧、结果变量、变形放大因子、振型和特征值（如适用）。

您可以为所有绘图状态显示或隐藏状态块，也可以显示或隐藏勾勒状态块的边框。此外，您可以控制状态块的位置、文本字体和文本颜色以及状态块背景的外观。您不能直接指定状态块的大小。但是，通过更改文本字体的大小，您可以增加或减小状态块的大小。状态块的内容是固定的，无法自定义。 

**自定义状态块：**

1. 找到 **状态块** 选项。从主菜单栏，选择 ****视口**![](../graphics/images/arrow.gif)**视口注释选项****。
2. 要为所有绘图状态显示或隐藏状态块，请在 **常规** 选项卡中切换 **显示状态块**。当 **显示状态块** 打开时，状态块选项变得可用。
3. 点击 **状态块** 选项卡。基本状态块选项位于 **状态块** 页面顶部。
4. 要显示或隐藏勾勒状态块的边框，请切换 **显示边界框**。这个"边界框"将状态块与底层绘图视觉上分开。
5. 确定状态块左上角的位置。要控制 *X* 或 ![](../graphics/usi_eqn00776.gif) 位置，请分别点击 **视口 X %** 或 **视口 Y %** 框。输入您想要的位置作为当前视口总宽度的百分比。
6. 自定义状态块文本字体。1. 点击 **设置字体**。出现 **选择字体** 对话框。2. 使用 **选择字体** 对话框选择您想要的字体特征。有关更多信息，请参见 ["自定义字体，" 第 3.2.8 节](pt01ch03s02s08.md)。3. 完成后，点击 **确定** 实现您的更改并关闭 **选择字体** 对话框。
7. 选择状态块文本的颜色。1. 点击颜色样本 ![](../graphics/usi-colorsample.png)。Abaqus/CAE 显示 **选择颜色** 对话框。2. 使用对话框中的方法之一选择新颜色。有关更多信息，请参见 ["自定义颜色，" 第 3.2.9 节](pt01ch03s02s09.md)。3. 点击 **确定** 关闭 **选择颜色** 对话框。颜色样本变为所选颜色。
8. 要自定义背景，请点击以下之一：- **匹配视口** 以将背景匹配到视口颜色。- **透明** 以消除背景，仅显示文本。- **其他颜色** 以显示其他背景颜色选项。如果您选择了 **其他颜色**：1. 点击颜色样本 ![](../graphics/usi-colorsample.png)。Abaqus/CAE 显示 **选择颜色** 对话框。2. 使用 **选择颜色** 对话框中的方法之一选择新颜色。有关更多信息，请参见 ["自定义颜色，" 第 3.2.9 节](pt01ch03s02s09.md)。3. 点击 **确定** 关闭 **选择颜色** 对话框。颜色样本变为所选颜色。
9. 点击 **应用** 实现您的更改。在当前视口中，所有绘图的状态块会根据您的规格进行更改，这会在会话期间保存。
