# 17.4.7 顶点和节点之间有什么关系？







当您对模型进行布种子时，Abaqus/CAE 会自动在模型边缘出现顶点的位置放置完全约束的种子。出现在顶点处的完全约束种子始终表示将在这些顶点处出现节点。（出现在区域边缘其他位置的完全约束种子不表示节点的确切位置；它们仅表示沿该边缘的节点数量。）因此，当您绘制零件时，应该记住零件中顶点的位置会影响 Abaqus/CAE 可以生成的网格质量。（有关更改顶点位置的信息，请参见"拖动 Sketcher 对象"第 20.17.1 节。）

例如，图 17-11 显示了二维零件的草图。

**图 17-11** 二维零件上的顶点。

![](../graphics/mgn-node1.png)

请注意九个顶点的位置。这些顶点是通过沿顶部和底部边缘绘制几条线段而不是沿每个边缘绘制一条连续线段来创建的。

当对该零件或零件实例进行布种子时，每个顶点处都会出现方形完全约束种子，如图 17-12 所示。

**图 17-12** 每个顶点处出现完全约束种子。

![](../graphics/mgn-node2.png)

当对模型进行网格划分时，Abaqus/CAE 始终会在位于顶点的完全约束种子位置放置节点，如图 17-13 所示。

**图 17-13** 节点出现在顶点处。

![](../graphics/mgn-node3.png)

同样，图 17-14 显示了将拉伸形成空心圆柱体的两个同心圆的草图。

**图 17-14** 对齐顶点的同心圆。

![](../graphics/mgn-node4.png)

请注意顶点的位置，Sketcher 在您点击定义圆周的位置创建顶点。

当对圆柱体进行布种子时，每个顶点处都会出现方形完全约束种子，如图 17-15 所示。

**图 17-15** 每个顶点处出现完全约束种子。

当对模型进行网格划分时，节点始终出现在位于顶点的完全约束种子位置，如图 17-16 所示。

**图 17-16** 节点出现在顶点处。

![](../graphics/mgn-node6.png)
