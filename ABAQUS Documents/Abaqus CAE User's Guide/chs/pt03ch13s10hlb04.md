# 13.10.4 创建零件或模型实例





要创建零件或模型实例，请从主菜单栏选择****Instance**![](../graphics/images/arrow.gif)**Create****并从出现的**Create Instance**对话框中选择所需的零件或模型。您可以从当前模型中的任何现有零件或当前模型数据库中的任何现有模型中进行选择。你可以创建同一零件或模型的多个实例，但不能装配在不同建模空间（三维、二维或轴对称）中创建的零件或模型的实例。

当你创建第一个实例时，Abaqus/CAE会显示一个图形符号，指示装配全局坐标系的原点和平移方向。此符号是一个基准坐标系。如果需要，可以使用装配显示选项隐藏它；更多信息，请参见["控制基准显示，"第76.9节](pt07ch76hla08.md)。

默认情况下，Abaqus/CAE创建依赖零件实例。依赖实例只是原始零件几何的指针。因此，许多操作不允许在依赖零件实例上执行；例如，你不能添加划分、创建虚拟拓扑或对实例进行网格划分。相比之下，独立零件实例是原始零件几何的副本。你可以在独立实例上执行大多数操作；例如，你可以添加划分、创建虚拟拓扑并对实例进行网格划分。你不能创建同一零件的独立实例和依赖实例。你可以从模型树中选择零件实例并将其从独立更改为依赖，反之亦然。更多信息，请参见["依赖和独立零件实例有什么区别？"第13.3.2节](pt03ch13s03s02.md)。

当你创建实例时，默认情况下Abaqus/CAE定位实例，使原始几何的原点与装配坐标系的原点对齐。当你创建多个实例时，新实例可以定位在现有实例之上。但是，如果你在**Create Instance**对话框中启用**Auto-offset from other instances**，Abaqus/CAE会沿*X*轴平移每个新实例，直到它不与任何现有实例重叠。如果装配是轴对称的，Abaqus/CAE会沿旋转轴而不是沿*X*轴平移新实例。

**要创建零件或模型实例：**

1. 从主菜单栏，选择****Instance**![](../graphics/images/arrow.gif)**Create****。Abaqus/CAE显示**Create Instance**对话框。
   **提示：**你也可以使用装配模块工具箱中的![](../graphics/ico_partInstance.png)工具来创建实例。有关装配工具箱中工具的图表，请参见["使用装配模块工具箱，"第13.9节](pt03ch13s09.md)。
2. 选择**Parts**或**Models**。根据你的选择，**Create Instance**对话框显示当前模型中所有现有零件的列表或模型数据库中所有其他模型的列表。
3. 从列表中选择所需的零件或模型。你可以使用**[Ctrl]****+Click**和**[Shift]****+Click**的组合来选择多个项目。选定实例的临时图像出现在当前视口中。Abaqus/CAE定位临时图像，使其原点与全局坐标系的原点重合。
4. 默认情况下，Abaqus/CAE创建**依赖**零件实例。如果需要，启用**独立**来创建独立零件实例。
5. 如果需要，启用**Auto-offset from other instances**来偏移新实例。
6. 如果你确认已选择正确的实例，点击**Apply**。Abaqus/CAE创建实例并在选择时应用自动偏移。
7. 要创建其他实例，从第2步开始重复此过程。完成创建实例后，点击**确定**关闭**Create Instance**对话框。
