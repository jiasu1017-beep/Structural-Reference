# 81.6.1 插件文件存储在哪里？





您可以随 Abaqus 一起安装插件，并使其对您站点的所有用户可用。或者，您可以将插件安装在自己的主目录中，其他用户将无法看到它。当您启动 Abaqus/CAE 时，它会在以下目录及其所有子目录中搜索插件文件：
- `*abaqus_dir*\abaqus_plugins`，其中 *abaqus_dir* 是 Abaqus 父目录。
- `*abaqus_dir*\code\python\lib\abaqus_plugins`，其中 *abaqus_dir* 是 Abaqus 父目录。
- `*home_dir*\abaqus_plugins`，其中 *home_dir* 是您的主目录。
- `*current_dir*\abaqus_plugins`，其中 *current_dir* 是当前目录。
- `*plugin_dir*`，其中 *plugin_dir* 是 `abaqus_v6.env` 文件中由环境变量 **plugin_central_dir** 指定的目录。如果 **plugin_central_dir** 引用的目录挂载在所有用户都可以访问的文件系统上，则可以将插件存储在中心位置供所有用户访问。例如：``` plugin_central_dir = r'\\fileServer\sharedDirectory' ```有关更多信息，请参见 [Abaqus Installation and Licensing Guide 第4.1节 "Using the Abaqus environment file"](../sgb/sgb-link.md#sgb-chp-env-using)。

Abaqus/CAE 会导入这些目录中与命名约定 `*_plugin.py` 匹配的所有文件。这些文件必须包含注册命令。文件名必须遵循 [Abaqus Scripting User's Guide 第5.5.2节 "Standard Abaqus Scripting Interface exceptions"](../cmd/cmd-link.md#cmd-int-acl-except-standardacl) 中描述的规则。
