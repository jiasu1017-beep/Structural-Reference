# 4.4.7 编辑文本标注属性





你可以更改文本标注的以下属性：
- 名称（使用**标注管理器**重命名现有标注）。
- 边界框显示。
- 背景颜色。
- 字体样式和颜色。
- 对齐方式。
- 旋转角度。
- 参考点位置和偏移量。
- 锚点位置。

如果你正在创建新标注，Abaqus/CAE 会将你的许多自定义设置应用到当前标注以及随后创建的任何新标注。如果你正在编辑现有标注，Abaqus/CAE 则只将更改应用到所选标注。

你也可以复制文本标注及其属性，然后编辑它们以创建新的文本标注。有关详细信息，请参阅["标注管理器，" 第 4.2.3 节](pt01ch04s02hlb03.md)。

**要编辑文本标注属性：**

1. 使用以下方法之一访问**文本编辑器**对话框：
   - 从**标注管理器**中选择一个文本标注，然后单击**编辑**。
   - 从主菜单栏中选择****Viewport**![](../graphics/images/arrow.gif)**Edit Annotations****，或从[**视口**工具栏](pt01ch04s02hlb02.md)中单击![](../graphics/ico_select.png)，然后在视口中双击一个文本标注。
   - 在视口中创建新的文本标注；Abaqus/CAE 在你选择文本位置后会打开编辑器。
2. 如果你正在创建新标注，可以编辑文本标注的名称。
3. 要显示或隐藏文本周围的轮廓框，请切换**显示边界框**。边界框可以直观地将文本与底层模型或结果分开。
4. 输入所需的文本。在输入文本标注时，你可以使用标准的鼠标和键盘编辑技术，如退格、复制和粘贴；按 **[Enter]** 开始新行。
5. 要自定义背景，请单击以下之一：
   - **匹配视口**以将背景匹配到视口颜色。
   - **透明**以消除背景，只显示文本。
   - **其他颜色**以显示其他背景颜色选项。
   如果你选择了**其他颜色**：
   1. 单击颜色样本![](../graphics/usi-colorsample.png)。Abaqus/CAE 显示**选择颜色**对话框。
   2. 使用**选择颜色**对话框中的方法选择新颜色。有关详细信息，请参阅["自定义颜色，" 第 3.2.9 节](pt01ch03s02s09.md)。
   3. 单击**确定**关闭**选择颜色**对话框。颜色样本会变为所选颜色。
6. 自定义文本字体。
   1. 单击**设置字体**。**选择字体**对话框出现。
   2. 使用**选择字体**对话框选择你想要的字体特性。有关详细信息，请参阅["自定义字体，" 第 3.2.8 节](pt01ch03s02s08.md)。
   3. 完成后，单击**确定**实施更改并关闭**选择字体**对话框。
7. 选择文本颜色。
   1. 单击颜色样本![](../graphics/usi-colorsample.png)。Abaqus/CAE 显示**选择颜色**对话框。
   2. 使用对话框中的方法选择新颜色。有关详细信息，请参阅["自定义颜色，" 第 3.2.9 节](pt01ch03s02s09.md)。
   3. 单击**确定**关闭**选择颜色**对话框。颜色样本会变为所选颜色。
8. 选择**左**、**居中**或**右**对齐方式来排列边界框区域内的文本。
9. 输入旋转角度（度）来定向文本；0 为水平。Abaqus/CAE 会围绕**参考点**逆时针旋转文本。
10. 单击**位置**选项卡。
11. 选择新的**参考点**。参考点是文本标注附加到其锚点的点；如果你旋转文本，Abaqus/CAE 也会使用参考点作为旋转中心。可使用以下方法：
    ****预定义**** 选择此方法使用文本边界框几何形状的预定义点作为参考点。可用点包括边界框角落、中心和各边中点。如果你修改标注文本，参考点定义会随之改变。
    ****% 文本 X、Y**** 选择此方法基于左下边界框角位置和边界框大小来定义参考点。在文本字段中输入作为文本边界框总宽度和高度百分比的参考点。与**预定义**方法一样，如果你修改文本，参考点定义会随之改变。
    **注意：**Abaqus/CAE 基于 0 度旋转角度确定左下角位置。如果你旋转文本，角位置也会相应旋转。
    **提示：**你可以通过输入小于 0% 或大于 100% 的值来选择边界框外部的参考点。
12. 输入新的 *X* 和 *Y* 值来更改参考点与锚点之间的**偏移量**（毫米）。**注意：**关闭**文本编辑器**后，你也可以通过在视口中拖动文本来更改偏移量。有关详细信息，请参阅["在当前视口中操作标注，" 第 4.4.5 节](pt01ch04s04hlb04.md)。
13. 选择新的锚点；可使用以下方法：
    **提示：**为避免更改标注的位置，请使用**拾取锚点**按钮更改锚点，并重新计算锚点与标注之间的**偏移量**。
    ****预定义**** 选择此方法将标注锚定到视口几何形状的预定义点。可用点包括视口角落、各边中点和视口中心。如果你修改视口形状，锚点定义会随之改变。
    ****% 视口 X、Y**** 选择此方法将标注锚定到基于左下视口角位置和视口大小的点。在文本字段中输入作为当前视口总宽度和高度百分比的锚点。与**预定义**方法一样，如果你修改视口形状，锚点定义会随之改变。
    ****模型点 X、Y、Z**** 选择此方法将标注锚定到模型上的点。在文本字段中输入新锚点的模型坐标。锚点定义及链接到它的任何标注，会在你操作模型时发生变化。例如，如果你在视口中旋转模型，锚点和箭头端点会随之旋转。
    ****拾取锚点**按钮** 选择此方法可以更改锚点而不改变标注的当前位置。Abaqus/CAE 会隐藏**箭头编辑器**对话框，并提示你从视口中选择一个点。或者，你可以从提示区域的列表中选择预定义的视口点，或单击**从模型拾取**从视口中拾取模型节点或顶点，或在提示区域中输入模型点的坐标。每种选择类型都对应于前述锚点选择方法之一。完成选择后，**文本编辑器**对话框会重新出现，你的选择会显示在相应的锚点选择方法中，偏移值也会为新锚点重新计算。
14. 单击**应用**以在视口中查看更改。**注意：**一旦你在视口中应用了更改，就无法恢复原始设置，只能通过重新创建标注或再次编辑选项来恢复。
15. 单击**确定**关闭**文本编辑器**。
