# 14.6 理解ALE自适应网格划分





任意拉格朗日-欧拉（ALE）自适应网格划分允许你即使在大变形或材料丢失的情况下也能在整个分析过程中保持高质量网格，方法是将网格与材料分离运动。自适应网格划分仅移动节点；网格拓扑保持不变。自适应网格划分仅适用于**耦合温度-位移**；**动态，显式**；**动态，温度-位移，显式**；**土壤**；和**静态，通用**步骤。

你可以通过从主菜单栏选择****Other**![](../graphics/images/arrow.gif)**ALE Adaptive Mesh Domain****来定义要自适应网格划分的模型区域。如有必要，你可以选择****Other**![](../graphics/images/arrow.gif)**ALE Adaptive Mesh Controls****或****Other**![](../graphics/images/arrow.gif)**ALE Adaptive Mesh Constraint****分别来自定义适应网格控制或添加区域自适应网格约束。目前，你只能为任何特定步骤定义一个ALE自适应网格域。

有关自适应网格划分的详细信息，请参见["ALE自适应网格划分，"Abaqus分析用户指南第12.2节](../usb/usb-link.md#usbaremesh)。
