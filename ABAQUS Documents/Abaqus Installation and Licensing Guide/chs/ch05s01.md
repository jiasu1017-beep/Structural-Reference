# 5.1 自定义用户界面



要自定义 Abaqus/CAE 和 Abaqus/Viewer 用户界面，您可以在 Windows 平台上指定常规显示属性。其他平台上的设置（如 Linux）也可能影响用户界面及其某些功能的外观。您还可以在名为 `abaqus.guiLog` 的文件中记录在 Abaqus/CAE 或 Abaqus/Viewer 用户界面中执行的所有操作。

### 5.1.1 硬件加速（所有平台）

对于某些图形设备，当打开硬件加速时，Abaqus/CAE 和 Abaqus/Viewer 可能会失败。如果完全必要，可以关闭硬件加速，但这不是推荐的。禁用硬件加速会严重降低 Abaqus/CAE 和 Abaqus/Viewer 中的图形性能。您可以使用以下方法之一禁用硬件加速：

**Linux 平台**

使用 `-mesa` 选项启动 Abaqus/CAE 或 Abaqus/Viewer：

```
abaqus cae -mesa
abaqus viewer -mesa
```

**Windows 平台**

有两种方法可以在 Windows 平台上禁用硬件加速：
- 将参数 **abaqus_no_hardware_acceleration**=ON 添加到 Abaqus 环境文件；或
- 使用以下命令创建系统环境变量： ``` set ABAQUS_NO_HARDWARE_ACCELERATION=1 ```

### 5.1.2 Windows 平台上的常见自定义

以下过程说明如何在 Windows 平台上指定一些常见设置：

**更改任何 Abaqus 快捷方式的"起始位置"：**

1. 使用**Windows 资源管理器**转到保存 Abaqus 快捷方式的目录。此目录中的快捷方式会影响计算机上的所有用户，可能需要特殊权限才能更改。
2. 右键单击要更改起始位置的快捷方式（**Abaqus CAE**、**Abaqus Command** 或 **Abaqus Viewer**），然后选择**属性**；接着单击**快捷方式**选项卡。
3. 在标记为**起始位置：**的文本框中，设置您要用于保存该 Abaqus 产品创建的文件 的目录的完整路径作为默认位置。

**阻止 Abaqus/CAE 和 Abaqus/Viewer 窗口在移动对话框时被擦除：**

1. 选择****开始**![](../graphics/images/arrow.gif)**设置**![](../graphics/images/arrow.gif)**控制面板****，然后双击**显示**。
2. 从出现的**显示属性**对话框中，选择**外观**并单击**效果**。
3. 在**效果**对话框中，关闭**拖动时显示窗口内容**。
4. 单击**确定**保存设置并关闭**显示属性**对话框。

**更改 Abaqus/CAE 和 Abaqus/Viewer 中使用的颜色和字体：**

1. 选择****开始**![](../graphics/images/arrow.gif)**设置**![](../graphics/images/arrow.gif)**控制面板****，然后双击**显示**。
2. 从出现的**显示属性**对话框中，选择**外观**。
3. 从列表中选择预设**颜色方案**。
4. 如果需要，从列表中选择新的**字体大小**。**注意：**某些颜色方案默认只允许一种字体大小。要增加特定项目的字体大小，请在**高级外观**对话框中编辑颜色方案。
5. 如果需要，更改所选颜色方案中各个项目的颜色或字体设置： 1. 单击**高级**。出现**高级外观**对话框。 2. 选择要更改颜色和字体显示的**项目**。对话框显示当前颜色，并且在适用的情况下显示所选项目的字体设置。Windows 排除不显示文本的项目的字体设置，如活动窗口边框。 3. 从**颜色 1** 列表中选择项目的新颜色。对于允许两种颜色之间渐变的项目（如活动标题栏），您也可以从**颜色 2** 列表中选择新颜色来调整渐变中的第二种颜色。 4. 从对话框底部的选项中调整所选项目的文本设置。您可以从列表中选择新**字体**，单击**大小**箭头增加或减小文本大小，从列表中选择新文本**颜色**，以及切换所选项目文本的粗体或斜体格式。 5. 重复前两步以更改颜色方案中其他单个项目的颜色和文本显示。 6. 单击**确定**关闭**高级外观**对话框。
6. 单击**确定**保存设置并关闭**显示属性**对话框。

**更改 Abaqus/CAE 和 Abaqus/Viewer 中使用的默认字体：**

- 默认情况下，Windows 通过引用系统字体目录中可用的字体来呈现 Abaqus/CAE 和 Abaqus/Viewer 视口窗口中的文本。您可以通过将 **hks_font_path** 参数添加到 Abaqus 环境文件来覆盖此默认行为并为会话使用其他字体。如果要为会话设置多个字体目录，请将此参数设置为多个逗号分隔的值。

**在 Abaqus/CAE 和 Abaqus/Viewer 中显示中文字符：**

1. 设置 Windows 的区域语言设置以启用中文字符： 1. 选择****开始**![](../graphics/images/arrow.gif)**设置**![](../graphics/images/arrow.gif)**控制面板****，然后双击**区域和语言选项**。出现**区域和语言选项**对话框。 2. 在**语言**选项卡页上，确保**为东亚语言安装文件**选项已打开。如果未安装东亚语言，则必须具有管理员权限才能更改此选项。 3. 在**区域选项**选项卡页上，在**标准和格式**部分中选择**中文(PRC)**。 4. 单击**确定**。
2. 设置系统环境变量 `ABAQUS_USE_LOCALIZATION=1`。 - 要为单个会话设置变量，请在命令提示符中输入以下命令： ``` set ABAQUS_USE_LOCALIZATION=1 ``` 您必须从同一命令提示符运行 Abaqus/CAE 或 Abaqus/Viewer。 - 要永久设置变量，请将其保存在 Windows 系统属性中。可以使用**系统属性**对话框的**高级**选项卡页保存环境变量（选择****开始**![](../graphics/images/arrow.gif)**设置**![](../graphics/images/arrow.gif)**控制面板****，然后双击**系统**）。

### 5.1.3 影响 Abaqus/CAE 和 Abaqus/Viewer 的 Linux 设置

Linux 操作系统为您提供了许多自定义选项。因为您可以更改其他平台上保持不变的操作系统环境的部分，所以您的 Linux 设置可能会改变 Abaqus/CAE 和 Abaqus/Viewer 中的一些基本交互。可用设置的确切值和更改方法因您安装的 Linux 版本而异。三种可能的设置及其影响如下：

**移除窗口标题栏**

如果您自定义了系统使其不显示窗口标题栏，则对某些对话框和工具箱功能的访问可能会受到限制。没有标题栏，您可能无法移动对话框。使用 **[Esc]** 键是关闭没有标题栏或**取消**按钮的对话框或工具箱的唯一方法。

**移除窗口边框**

调整对话框大小需要您单击并拖动边框。如果您的 Linux 自定义包括移除应用程序窗口边框，则可能无法在 Abaqus/CAE 和 Abaqus/Viewer 中调整对话框大小。相反，使用滚动条访问超出对话框边缘的数据。

**显示日文字符**

如果您的语言环境设置是日语，则 Abaqus/CAE 和 Abaqus/Viewer 可以在视口中显示日语文本。例如，文本注释以及状态和标题块以日语显示。要正确显示日文字符，必须在目录 `/usr/lib/X11/fonts` 中安装日文 TrueType 字体。

### 5.1.4 在日志文件中记录所有用户界面操作

您可以在名为 `abaqus.guiLog` 的文件中记录在 Abaqus/CAE 或 Abaqus/Viewer 用户界面中执行的所有操作。可以为单个会话打开此设置，也可以将其选择为默认行为。

**在运行时启用用户界面录制**

使用 `-guiRecord` 选项启动 Abaqus/CAE 或 Abaqus/Viewer：

```
abaqus cae -guiRecord
abaqus viewer -guiRecord
```

**使用 `ABQ_CAE_GUIRECORD` 启用用户界面录制**

您可以通过从命令行设置、在 Abaqus 环境文件中或在 Windows 平台上作为系统环境变量来指定用户界面录制。
- 要为单个会话设置变量，请在命令提示符中输入以下命令： ``` set ABQ_CAE_GUIRECORD=1 ``` 您必须从同一命令提示符运行 Abaqus/CAE 或 Abaqus/Viewer。
- 要将用户界面录制设为默认行为，请将参数 `ABQ_CAE_GUIRECORD``=ON` 添加到 Abaqus 环境文件；或在 Windows 平台上，将其保存在 Windows 系统属性中。可以使用**系统属性**对话框的**高级**选项卡页保存环境变量（选择****开始**![](../graphics/images/arrow.gif)**设置**![](../graphics/images/arrow.gif)**控制面板****，然后双击**系统**）。

当用户界面录制是默认行为时，您仍然可以通过使用 `-guiNoRecord` 选项启动 Abaqus/CAE 或 Abaqus/Viewer 来禁用它：

```
abaqus cae -guiNoRecord
abaqus viewer -guiNoRecord
```
