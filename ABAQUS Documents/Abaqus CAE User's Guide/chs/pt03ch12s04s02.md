# 12.4.2 创建轮廓





要创建轮廓，请从主菜单栏选择****Profile**![](../graphics/images/arrow.gif)**Create****。出现**Create Profile**对话框，您可以在其中输入轮廓的名称并选择轮廓类型。完成此信息的输入后，点击**Create Profile**对话框中的**Continue**以显示轮廓编辑器，该编辑器允许您创建和编辑轮廓。

所有轮廓编辑器都显示轮廓形状的图示和文本字段，您可以在其中输入定义轮廓所需的所有数据。例如，I形轮廓编辑器如图[12-3](pt03ch12s04s02.md#prp-profileeditor)所示。编辑器包含I形轮廓的图示和数据字段，您可以在其中输入每个尺寸。

**图12-3** I形轮廓编辑器。

![](../graphics/prp-profileeditor-nls.png)

创建轮廓后，您可以在梁截面定义中引用该轮廓。例如，在如图[12-4](pt03ch12s04s02.md#prp-profilename)所示的梁截面编辑器中，选择了一个名为`SupportBeam`的箱形轮廓。

**图12-4** 在梁截面编辑器中指定轮廓名称。

![](../graphics/prp-profilename-nls.png)

有关轮廓的更多信息，请参见["定义轮廓，"第12.2.2节](pt03ch12s02s02.md)。
