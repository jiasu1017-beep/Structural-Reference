# 70.1 理解分割的作用





在建模过程中，您可能会发现需要选择模型中不存在的特定区域。这些区域可用于定义材料边界、指示载荷和约束的位置，并帮助细化网格。您使用分割工具集将模型分割成区域。

分割可以在边、面和单元上创建。沿边的分割会创建新顶点，而穿过面和单元的分割分别创建新边和新面。在所有情况下，分割都用于细分被分割的几何。[图70-1](pt06ch70s01.md#par-simple-example)展示了边、面和单元的分割。

**图70-1** 分割边、面和单元。

![](../graphics/par-simple-example-nls.png)

您通过定义引用现有几何的分割来分割边、面和单元。您可以在零件模块或属性模块中分割零件，也可以在对装配体进行操作的模块中分割装配体。例如，您可以在网格模块中分割装配体的面，并可以在产生的内部边上设置种子以细化网格。分割是一个特征；与所有特征一样，它可以编辑、删除、抑制、恢复和查询。类似地，当装配体或零件再生时，分割也会再生。
