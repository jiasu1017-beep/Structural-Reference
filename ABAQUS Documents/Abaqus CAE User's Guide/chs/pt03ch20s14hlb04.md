# 20.17.4 通过分割边缘来修改 Sketcher 对象


使用 Sketcher 工具箱中的分割工具将 Sketcher 对象的各个边缘分割成多个边缘。您可以在边缘与其他草图、参考或构造几何的任何交点处分割边缘。如果您使用其他草图几何来分割边缘,Abaqus/CAE 会在交点处分割两条边缘。您也可以在扩展交点处分割边缘——即只有当草图中的另一个边缘被延伸时才会存在的交点。

**分割边缘的步骤:**

1. 从 Sketcher 工具箱的修改工具中,选择分割工具 ![](../graphics/ico_sketchSplitCurve.png)。有关 Sketcher 工具箱中工具的图示,请参见["Sketcher 工具," 第 20.4.1 节](pt03ch20s04s01.md)。Abaqus/CAE 会在提示区域显示提示,引导您完成操作。
2. 选择要分割的边缘;您可以选择 Abaqus/CAE 中任何绘制或导入的边缘。**注意:**如果您分割样条曲线,Abaqus/CAE 会删除控制点以保留原始曲线的精确形状;因此,您将无法再编辑曲线的形状。
3. 将光标移动到草图中的其他边缘上。当您移动光标时,Abaqus/CAE 会高亮显示当前光标位置将被选中的边缘。Abaqus/CAE 还会在分割点周围画圈——即要分割的边缘与光标位置边缘之间的交点。如果所选边缘之间有多个可能的分割点,Abaqus/CAE 会圈出距离光标位置最近的点。
4. 单击以接受指示的分割点。Abaqus/CAE 会分割第一边缘,创建两个共享该分割点作为端点的边缘。如果第二边缘是草图几何且分割点位于其端点之间,Abaqus/CAE 也会分割第二边缘。**注意:**如果您分割圆或椭圆,Abaqus/CAE 会使用边缘上的顶点作为第二个分割点来创建两个圆弧。如果上一步中指示的分割点与边缘点相同,则会在消息区域显示警告,并且曲线不会被分割。
5. 要分割更多边缘,请从第 2 步开始重复上述步骤。
6. 完成分割边缘后,执行以下操作之一 - 在 Abaqus/CAE 窗口任意位置单击鼠标按钮 2。 - 选择 Sketcher 工具箱中的任何工具。 - 单击提示区域中的取消按钮 ![](../graphics/afxI_cancel.png)。
