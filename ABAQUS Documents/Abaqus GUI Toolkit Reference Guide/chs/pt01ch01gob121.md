# 消息映射

消息映射由从 FXObject 派生的类使用（包括大多数 GUI 类），用于将发送到对象的消息路由到消息处理方法。典型的用法是将对话框中的按钮的消息路由到对话框的某个方法，以便在按下按钮时执行某些操作。

消息由两部分组成：类型和 ID。类型表示生成消息的动作，而 ID 标识发送消息的人。这两个数字使用 MKUINT 函数组合成单个值。

当调用消息处理方法时，它接收三个参数：发送者、选择器和一些可选数据。您的消息处理程序原型必须包含这三个参数，通常指定为 *sender*、*sel*、*ptr*。可选数据有时由发送者用来包含可能对消息处理程序有用的额外数据（例如单击鼠标按钮时光标的坐标）。但是，由于 C++ 和 Python 之间的语言差异，此可选数据在 Python 脚本中不可用。

有关更多详细信息，请参阅《Abaqus GUI Toolkit User's Guide》。

### FXMAPFUNC(object, messageType, messageId, method)

在对象的消息映射中创建一个条目，该条目将消息路由到方法。

| **参数** | **类型** | **默认值** | **描述** |
| --- | --- | --- | --- |
| object | FXObject |  | 要在其中进行消息映射条目的类的实例。通常是"self"。 |
| messageType | Int |  | 指定消息类型的整数标志（例如 SEL_COMMAND）。 |
| messageId | Int |  | 指定消息 ID 的整数。 |
| method | Function |  | 消息要路由到的方法。必须通过包含类名来指定此方法（例如 MyDB.myMethod）。 |

### FXMAPFUNCS(object, messageType, startMessageId, endMessageId, method)

在对象的消息映射中创建多个条目，这些条目将消息路由到方法。

| **参数** | **类型** | **默认值** | **描述** |
| --- | --- | --- | --- |
| object | FXObject |  | 要在其中进行消息映射条目的类的实例。通常是"self"。 |
| messageType | Int |  | 指定消息类型的整数标志（例如 SEL_COMMAND）。 |
| startMessageId | Int |  | 指定起始消息 ID 的整数。 |
| endMessageId | Int |  | 指定结束消息 ID 的整数。 |
| method | Function |  | 消息要路由到的方法。必须通过包含类名来指定此方法（例如 MyDB.myMethod）。 |

### MKUINT(messageId, messageType)

通过组合消息 ID 和消息类型来创建消息选择器。

| **参数** | **类型** | **默认值** | **描述** |
| --- | --- | --- | --- |
| messageId | Int |  | 指定消息 ID 的整数。 |
| messageType | Int |  | 指定消息类型的整数标志（例如 SEL_COMMAND）。 |

### SELID(selector)

从消息选择器返回消息 ID。

| **参数** | **类型** | **默认值** | **描述** |
| --- | --- | --- | --- |
| selector | Int |  | 消息选择器。 |

### SELTYPE(selector)

从消息选择器返回消息类型。

| **参数** | **类型** | **默认值** | **描述** |
| --- | --- | --- | --- |
| selector | Int |  | 消息选择器。 |
