# *HEADING






### *HEADING在输出中打印标题。

此选项用于为分析定义标题。

**产品：** Abaqus/Standard  Abaqus/Explicit  Abaqus/CFD  Abaqus/CAE  

**类型：** 模型数据  

**级别：** 模型  

**Abaqus/CAE：** Job 模块

##### **参考文献：**

- ["在 Abaqus 中定义模型，" Abaqus Analysis User's Guide 第 1.3.1 节](../usb/usb-link.md#usb-int-imodel)

**此选项没有关联的参数。**

### **打印标题的数据行：**

**第一行：**

标题可以有多行，但只有第一行的前 80 个字符会被保存并打印为标题。
