# 11 多步骤分析







Abaqus 模拟的总体目标是预测结构对施加载荷的响应。回想一下，在一般意义上，Abaqus 中的*载荷*指的是任何诱导结构响应从其初始状态发生变化的东西；例如，非零边界条件或施加的位移、点力、压力、场等。在某些情况下，载荷相对简单，例如作用在结构上的一组点载荷。在其他问题中，作用在结构上的载荷可能非常复杂。例如，可能在不同时间将不同的载荷施加到模型的不同部分，或者载荷大小可能随时间变化。术语*载荷历史*用于指此类复杂的模型加载。

在 Abaqus 中，用户将模拟的完整载荷历史划分为多个*步骤*。每个步骤是用户指定的一段时间，Abaqus 计算模型对特定载荷集和边界条件的响应。用户必须在每个步骤中指定响应类型（称为分析过程），并且可以从步骤到步骤更改分析过程。例如，可以在一个步骤中向结构施加静态死载荷（可能是重力载荷）；在下一步中计算负载结构对地震加速度的动力响应。隐式和显式分析都可以包含多个步骤；但是，隐式和显式步骤不能在同一个分析作业中组合。要组合一系列隐式和显式步骤，可以使用结果传递（或导入）功能。此功能在 ["在 Abaqus/Explicit 和 Abaqus/Standard 之间传递结果，" Abaqus Analysis User's Guide 第 9.2.2 节](../usb/usb-link.md#usb-anl-aexptostd) 中讨论，这里不再进一步讨论。

Abaqus 将其所有分析过程分为两个主要组：线性摄动和一般。通用分析步骤可以包含在 Abaqus/Standard 或 Abaqus/Explicit 分析中；线性摄动步骤仅在 Abaqus/Standard 中可用。加载条件和"时间"对两种情况定义不同。此外，每种类型过程的结果应不同地解释。

在一般分析过程中（称为*一般步骤*）的模型响应可以是非线性的或线性的。在使用摄动过程（称为*摄动步骤*）的步骤中，响应只能是线性的。Abaqus/Standard 将此类步骤视为在任何先前一般步骤创建的预加载、预变形状态（称为基态）周围的线性摄动；因此，其进行线性模拟的能力比纯线性分析程序更通用。
