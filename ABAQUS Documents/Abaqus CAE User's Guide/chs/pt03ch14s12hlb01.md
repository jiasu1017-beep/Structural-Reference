# 14.12.1 创建输出请求





当您在步骤序列中创建第一个步骤时，Abaqus/CAE会根据您为该步骤选择的分析过程生成默认的场和历史输出请求。输出请求被传播到分析中的后续步骤。更多信息，请参见["输出请求的传播，"第14.4.3节](pt03ch14s04s03.md)。您可以使用输出请求编辑器来编辑在创建它们的步骤中的默认输出请求。您也可以使用输出请求编辑器来创建新的输出请求。新的输出请求会覆盖任何传播的请求。

**要创建输出请求：**

1. 从主菜单栏，选择**输出**![](../graphics/images/arrow.gif)**场输出请求**![](../graphics/images/arrow.gif)**创建**或**输出**![](../graphics/images/arrow.gif)**历史输出请求**![](../graphics/images/arrow.gif)**创建**。**提示：**您也可以使用位于模块工具箱中的![](../graphics/ico_fieldOutput.png)和![](../graphics/ico_historyOutput.png)工具来创建场和历史输出请求。有关Step模块工具箱中工具的图表，请参见["使用Step模块工具箱，"第14.8节](pt03ch14s08.md)。Abaqus/CAE显示**创建场输出**或**创建历史输出**对话框。
2. 在对话框中，执行以下操作：
   1. 输入输出请求的名称或接受默认名称。
   2. 点击**步骤**文本字段旁边的箭头，并从出现的列表中选择步骤。Abaqus/CAE在所选步骤期间创建输出请求。创建输出请求后，您可以使用输出请求管理器将其移动到更早或更晚的步骤。
   3. 点击**继续**。Abaqus/CAE显示**编辑场输出请求**或**编辑历史输出请求**对话框。
3. 在编辑器中，输入定义输出请求所需的数据。更多信息，请参见["修改场输出请求，"第14.12.2节](pt03ch14s12hlb02.md)，或["修改历史输出请求，"第14.12.3节](pt03ch14s12hlb03.md)。
4. 完成输出请求的配置后，点击**确定**保存数据并退出编辑器。
