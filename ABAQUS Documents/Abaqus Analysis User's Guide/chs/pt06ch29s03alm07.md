### 概述

横截面的选择由横截面的几何形状及其行为决定。梁的横截面：
- 可以是实心的或薄壁的；
- 如果是薄壁的，可以是开口的或闭合的；以及
- 可以通过从 Abaqus 截面库中选择来定义；通过指定几何量（如面积、惯性矩和扭转常数）；或通过使用特殊二维单元网格来定义，Abaqus 会数值计算几何量。

您必须考虑截面应被视为实心截面还是薄壁截面。这个选择决定了 Abaqus 计算截面每一点轴向和剪切应变的基础。

### 实心截面

对于弯曲下的实心截面，平面（梁）截面保持平面。在扭转载荷下，任何非圆形梁截面都会翘曲：梁截面不会保持平面。然而，对于实心截面，截面的翘曲足够小，可以忽略翘曲产生的轴向应变，并且可以使用圣维南翘曲理论在截面的每个积分点构建单一的剪切应变分量。这对于梁截面库中的矩形和梯形截面是自动完成的。圣维南翘曲函数用于定义剪切应变，即使截面的响应不再完全是弹性的。这限制了涉及承受大量非弹性变形的非圆形实心梁截面的扭转载荷建模的准确性。使用网格化梁轮廓时，在用户指定的材料系统中可以输出两个剪切应变分量。厚壁管截面被视为实心截面。

### 非实心（"薄壁"）截面

在 Abaqus 中，非实心截面被视为"薄壁"截面处理；也就是说，在截面的平面中，截面分支的厚度被认为远小于其长度。薄壁梁理论根据截面是闭合还是开口来决定截面壁中的剪切。

#### 闭合截面

闭合截面是分支形成闭合回路的非实心截面。闭合截面提供显著的抗扭能力，不会显著翘曲。Abaqus 忽略闭合截面的翘曲效应。

在 Abaqus 中，预定义梁截面只能模拟一个闭合回路。具有多个回路的截面必须使用网格化梁截面（请参阅 ["网格化截面](pt06ch29s03alm07.md#usb-elm-ebeamsections-meshed)"）或壳来建模。

对于足够小的截面壁厚度，剪切应力在厚度方向上的变化可以忽略不计；Abaqus 中可用闭合截面的公式基于此假设。

#### 开口截面

开口截面是分支不形成闭合回路的非实心截面，例如 I 截面或 U 截面。在这些截面中，剪切应力假定在壁厚上线性变化，并在壁中心处消失。开口截面可能显著翘曲，通常需要在支撑或接头处使用开放截面翘曲理论（与 Abaqus/Standard 中的梁单元类型 B*xx*OS 一起使用）以及适当的翘曲约束（应用于自由度 7）。这种翘曲约束可能会显著增加梁的扭转刚度。分支为直线的开口薄壁截面，在单点相交（如 Abaqus 梁单元截面库中的 L 截面、T 截面或 X 截面）不会翘曲；因此，翘曲约束无效。这些截面的扭转刚度总是非常小。

如果开口截面与常规梁单元类型（不是 B*xx*OS）一起使用，则假定开口截面可以自由翘曲，并忽略翘曲产生的轴向应变。因此，截面将具有非常小的扭转刚度。

#### 截面属性计算

计算非实心截面属性时使用薄壁假设。包含相交直线段（任意、箱形、六角形、I 形和 L 形截面）的截面的属性还包括相交几何形状的近似值。

### 可用的梁截面

您可以指定以下任何类型的梁截面：Abaqus 库截面、直接指定几何量的广义截面或网格化截面。

#### Abaqus 梁截面库

Abaqus 梁截面库包含实心截面（圆形、矩形和梯形）、闭合薄壁截面（箱形、六角形和管形）、开口薄壁截面（I 形、T 形或 L 形）以及厚壁管截面。Abaqus 还提供任意薄壁截面定义；Abaqus 将根据截面的定义方式将此截面类型视为闭合或开口截面。

梯形、I 形和任意库截面允许您定义局部坐标系的原点位置。其他截面类型（如矩形、圆形、L 形或管形）具有预设的原点。

| **输入文件用法：** | 使用以下选项定义在分析期间集成的梁截面： |
| --- | --- |
|  | ``` [*BEAM SECTION](../key/key-link.md#usb-kws-mbeamsection), SECTION=*name* ``` 其中 *name* 可以是 ARBITRARY、BOX、CIRC、HEX、I、L、PIPE、RECT、THICK PIPE 或 TRAPEZOID。T 截面通过仅为 I 形截面的一个翼缘指定几何数据来定义。使用以下选项定义通用梁截面： ``` [*BEAM GENERAL SECTION](../key/key-link.md#usb-kws-mbeamgensect), SECTION=*name* ``` 其中 *name* 可以是 ARBITRARY、BOX、CIRC、HEX、I、L、PIPE、RECT 或 TRAPEZOID。T 截面通过仅为 I 截面类型的一个翼缘指定几何数据来定义。 |

| **Abaqus/CAE 用法：** | Property 模块：**Create Profile**：选择 **Box**、**Pipe**、**Circular**、**Rectangular**、**Hexagonal**、**Trapezoidal**、**I**、**L**、**T** 或 **Arbitrary** |
| --- | --- |

#### 广义截面

Abaqus 还允许您通过指定定义截面所需的几何量来指定"广义"截面。这种广义截面只能与线性材料行为一起使用，尽管截面响应可以是线性或非线性的。

| **输入文件用法：** | 使用以下选项定义线性广义截面： |
| --- | --- |
|  | ``` [*BEAM GENERAL SECTION](../key/key-link.md#usb-kws-mbeamgensect), SECTION=GENERAL ``` 使用以下选项定义非线性广义截面： ``` [*BEAM GENERAL SECTION](../key/key-link.md#usb-kws-mbeamgensect), SECTION=NONLINEAR GENERAL ``` |

| **Abaqus/CAE 用法：** | Property 模块：**Create Profile**：选择 **Generalized** |
| --- | --- |
|  | Abaqus/CAE 不支持非线性广义截面。 |

#### 网格化截面

Abaqus 允许您通过使用翘曲单元（请参阅 ["翘曲单元，" 第 28.4.1 节](pt06ch28s04alm04.md)）在二维分析中生成梁截面属性来网格化任意形状的实心截面，这些属性可在后续的二维或三维梁分析中使用。这种截面只允许线性弹性材料行为。因此，网格化截面只能与通用梁截面定义一起使用；有关详细信息，请参阅 ["网格化梁截面，" 第 10.6.1 节](pt04ch10s06at35.md)。

| **输入文件用法：** | ``` [*BEAM GENERAL SECTION](../key/key-link.md#usb-kws-mbeamgensect), SECTION=MESHED ``` |
| --- | --- |

| **Abaqus/CAE 用法：** | Abaqus/CAE 不支持网格化截面。 |
