### 3.2.21 连接重启分析的输出数据库（`.odb`）文件

**产品：**Abaqus/Standard  Abaqus/Explicit  

##### **参考**

- ["Abaqus执行程序：概述，" 3.1.1节](pt01ch03s01abo02.md)
- ["重启分析中的输出延续" 9.1.1节"重启分析"](pt04ch09s01aus53.md#usb-anl-arestart-continue)

### 概述

**abaqus restartjoin**工具将模型重启分析生成的输出数据库（`.odb`）文件追加到该模型原始分析生成的输出数据库。将原始和重启输出数据库文件组合成单个文件使您能够在Abaqus/CAE中检查分析的所有输出数据。

类似工具**abaqus append**用于连接结果（`.fil`）文件。请参阅["连接结果（`.fil`）文件，" 3.2.14节](pt01ch03s02abx14.md)，了解详细信息。

### 在步骤之间重启和中间步骤重启时追加数据

您可以追加从步骤之间重启的分析和从步骤中间重启的分析的输出数据库文件。虽然这两种类型的分析语法要求相同，但Abaqus以下列方式不同地追加数据：
- 对于在步骤之间停止并重启的分析，Abaqus简单地将新步骤的输出追加到原始分析现有步骤的输出。
- 对于在步骤中间停止并重启的分析，原始分析和重启分析会重叠，因为重启分析从中断步骤的开始恢复。在这种情况下，**abaqus restartjoin**工具保留原始分析中任何已完成步骤的结果，但用重启分析生成的输出数据替换中断步骤的结果。

### 定制组合输出数据库文件

默认情况下，Abaqus将重启分析生成的输出数据直接追加到原始输出数据库文件。如果您希望保留原始输出数据库文件，可以创建它的副本，然后将重启分析输出数据追加到副本。Abaqus使用格式`Restart_*original-odb-filename*`命名此副本；例如，原始输出数据库文件`job-1.odb`的副本将被命名为`Restart_job-1.odb`。

当您组合原始和重启输出数据库时，Abaqus会忽略历史数据；但是，您可以覆盖此默认设置。您还可以控制Abaqus是否压缩组合输出数据库文件。

### 命令行选项

**originalodb**

此选项指定原始分析生成的输出数据库文件。如果省略**copyoriginal**选项，Abaqus将重启输出数据库文件中的输出数据直接追加到原始输出数据库文件。

如果命令行中省略此选项，Abaqus将提示您输入其值。

**restartodb**

此选项指定重启分析生成的输出数据库文件。您一次只能指定一个重启分析输出数据库文件。

如果命令行中省略此选项，Abaqus将提示您输入其值。

**copyoriginal**

如果指定了此选项，Abaqus创建由**originalodb**选项指定的输出数据库文件的副本，并将**restartodb**输出数据库文件的内容追加到该副本，而不是原始文件。省略此选项时，Abaqus将重启分析的输出数据直接追加到原始输出数据库文件。

Abaqus通过在原始输出数据库文件的名称前添加前缀`Restart_`来命名复制的输出数据库文件；例如，原始输出数据库文件`original.odb`的副本将被命名为`Restart_original.odb`。

**history**

如果指定此选项，Abaqus会将历史数据从重启输出数据库复制到原始输出数据库或其副本。除非指定此选项，否则Abaqus会在连接的输出数据库文件中省略历史数据。

**compressresult**

如果指定此选项，Abaqus会压缩生成的输出数据库文件。

### 示例

如果您的模型生成了名为`Job-1.odb`的初始输出数据库文件和名为`Job-1_res.odb`的重启输出数据库文件，请发出以下命令将重启数据库的内容追加到初始输出数据库文件：

```
abaqus restartjoin originalodb=Job-1.odb restartodb=Job-1_res.odb
```
如果您希望保留原始输出数据库文件，可以创建此原始文件的副本，然后将重启输出数据库文件的内容追加到副本。Abaqus通过在原始文件名称前添加前缀`Restart_`来命名复制的输出数据库文件；在上一个示例中，原始文件`Job-1.odb`的副本将被命名为`Restart_Job-1.odb`。要使用原始文件的副本执行重启连接操作，请发出以下命令：
```
abaqus restartjoin originalodb=Job-1.odb restartodb=Job-1_res.odb
   copyoriginal
```
默认情况下，AbaUS不会将历史数据复制到组合输出数据库。要包含历史数据，请发出以下命令：
```
abaqus restartjoin originalodb=Job-1.odb restartodb=Job-1_res.odb
   history
```
