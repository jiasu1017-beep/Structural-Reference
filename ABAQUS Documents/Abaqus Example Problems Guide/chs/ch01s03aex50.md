# 1.3.19 Moldflow 翻译示例





**产品：**Abaqus/Standard   

### 目标

本示例演示 **abaqus moldflow** 翻译器的以下功能：
- 将 Moldflow 分析的有限元模型信息转换为部分 Abaqus 输入文件，
- 添加适当的分析数据，
- 执行固有频率分析，以及
- 计算由初始应力引起的变形。

### 应用描述

Moldflow 仿真（模拟塑料注射成型过程）的结果包括塑料零件的材料属性和残余应力的计算。本示例研究了两种类型的零件——填充支架和未填充支架——以说明如何将 Moldflow 仿真的有限元模型信息转换为部分 Abaqus 输入文件，添加边界条件、步骤数据并请求输出。

### Abaqus 建模方法与仿真技术

本示例说明了 **abaqus moldflow** 翻译器的使用方法。

### 运行步骤

三个案例采用相同的一般方法。以下步骤总结了 **abaqus moldflow** 翻译器的典型用法：

1. 从 Moldflow 仿真导出数据，如下所示：- 对于中平面 Moldflow 仿真，将有限元网格数据导出到名为 `*job-name*.pat` 的文件，将结果数据（材料属性和残余应力）导出到名为 `*job-name*.osp` 的文件。- 对于使用 Moldflow Version MPI 6 的三维实体 Moldflow 仿真，运行 Visual Basic 脚本 `mpi2abq.vbs` 将有限元网格数据导出到名为 `*job-name*_mesh.inp` 的文件，将结果数据导出到 `.xml` 文件。
2. 运行 **abaqus moldflow** 翻译器，从 Moldflow 接口文件创建部分 Abaqus 输入（`.inp`）文件。
3. 编辑 Abaqus 输入文件，添加适当的分析数据（例如，添加边界条件和步骤数据）。
4. 提交 Abaqus 输入文件进行分析。

#### 提取文件

所有与这些分析相关的文件都随 Abaqus 版本一起提供；您可以使用 Abaqus **fetch** 实用程序从压缩档案文件中提取示例问题文件。

要提取特定示例问题的所有相关文件，请输入以下命令：

| 案例 1 | `abaqus fetch job=moldflow_ex1*` |
| --- | --- |
| 案例 2 | `abaqus fetch job=moldflow_ex2*` |
| 案例 3 | `abaqus fetch job=bracket3d_mpi6*` |

有关将通配符表达式与 Abaqus **fetch** 实用程序一起使用的信息，请参阅["获取样本输入文件，" Abaqus Analysis User's Guide 第 3.2.16 节](../usb/usb-link.md#usb-int-dfetchproc)。

### 案例 1：纤维填充支架的固有频率分析

案例 1 中的支架由 926 个节点和 1719 个 S3R 单元组成。该模型包含七个不同的单元集。每个单元集具有不同的厚度，建模为具有 20 层的层合复合材料。

计算了十个无约束振型。前六个频率近似为零。前四个柔性模式的频率列于[表 1.3.19--1](ch01s03aex50.md#mfl-filledtable)中。

Abaqus 有限元模型如图 [1.3.19--1](ch01s03aex50.md#mfl-bracket) 所示。

### 案例 2：未填充支架的固有频率分析

案例 2 使用与案例 1 相同的 Abaqus 有限元模型，但材料属性为横观各向同性。壳截面定义为均匀而非复合。壳厚度方向使用 21 个等间距的 Simpson 积分点。

未填充支架的前四个柔性振动模式的频率列于[表 1.3.19--2](ch01s03aex50.md#mfl-unfilledtable)中。本示例中的未填充材料比案例 1 中的填充材料更软；因此，频率更低。

### 案例 3：三维填充支架由初始应力引起的变形

案例 3 使用与案例 1 中使用的模型相似的实体 Abaqus 有限元模型。

要执行 **abaqus moldflow** 翻译器，请输入以下命令：

```
abaqus moldflow job=bracket3d_mpi6 3D initial_stress=on
```

初始应力等值线图如图 [1.3.19--2](ch01s03aex50.md#mfl-bracket3d-stress2) 所示。
