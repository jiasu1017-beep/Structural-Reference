### 概述

Abaqus/Standard中用户定义的热材料行为：
- 通过一个接口提供，任何热本构模型都可以添加到库中；
- 需要在本构模型（或模型库）中编程用户子程序 [`UMATHT`](../sub/sub-link.md#sub-xsl-umatht)；以及
- 需要大量的努力和专业知识：该功能非常通用和强大，但其使用不是常规练习。

### 材料常数

用户子程序 [`UMATHT`](../sub/sub-link.md#sub-xsl-umatht) 所需的任何材料常数都必须作为用户定义热材料行为定义的一部分指定。同一材料定义中包含的任何其他热材料行为都将被忽略：用户定义的热材料行为要求所有热行为计算都在用户子程序 [`UMATHT`](../sub/sub-link.md#sub-xsl-umatht) 中编程。

| **输入文件用法：** | ``` [*USER MATERIAL](../key/key-link.md#usb-kws-musermaterial), TYPE=THERMAL, CONSTANTS=*number_of_constants* ``` |

### 非对称方程求解器

当电导率在用户子程序 [`UMATHT`](../sub/sub-link.md#sub-xsl-umatht) 中定义为温度的强函数时，热传递平衡方程变得非对称，您可以选择调用非对称方程求解功能；否则，收敛可能很差。

| **输入文件用法：** | ``` [*USER MATERIAL](../key/key-link.md#usb-kws-musermaterial), TYPE=THERMAL, CONSTANTS=*number_of_constants*, UNSYMM ``` |
| --- | --- |

| **Abaqus/CAE用法：** | 属性模块：材料编辑器：****通用**![](../graphics/images/arrow.gif)**用户材料****：**用户材料类型：热**，切换打开 **使用非对称材料刚度矩阵** |
| --- | --- |

### 材料状态

许多热本构模型需要存储解依赖状态变量。当材料发生相变时，这些状态变量可能包括微结构或相含量信息。您应在关联的材料定义中为这些变量分配存储（参见["分配空间" in "用户子程序：概述，" 第18.1.1节](pt04ch18s01aus104.md#usb-anl-usubrout-allocatespace)）。与用户定义材料关联的状态变量数量没有限制。

用户子程序 [`UMATHT`](../sub/sub-link.md#sub-xsl-umatht) 在每个增量的每次迭代中为每个材料点调用。它提供增量开始时材料的热状态（解依赖状态变量、温度和任何预定义场变量），以及温度、预定义状态变量和时间的增量。

### 所需计算

子程序 [`UMATHT`](../sub/sub-link.md#sub-xsl-umatht) 必须执行以下功能：它必须定义单位质量的内部能及其对温度和温度空间梯度的变化；它必须定义热通量向量及其对温度和温度空间梯度的变化；并且它必须将解依赖状态变量更新到增量结束时的值。用户子程序 [`UMATHT`](../sub/sub-link.md#sub-xsl-umatht) 中的热通量和空间梯度分量方向取决于局部方向的使用（参见["方向，" 第2.2.5节](pt01ch02s02aus15.md)）。

### 与其他用户子程序一起使用

用户子程序 [`UMAT`](../sub/sub-link.md#sub-xsl-umat) 可以与 [`UMATHT`](../sub/sub-link.md#sub-xsl-umatht) 结合使用来定义材料的机械本构行为。在材料定义中分配的可解依赖变量在 [`UMATHT`](../sub/sub-link.md#sub-xsl-umatht) 和 [`UMAT`](../sub/sub-link.md#sub-xsl-umat) 中都可以访问。此外，用户子程序 [`FRIC`](../sub/sub-link.md#sub-xsl-fric)、[`GAPCON`](../sub/sub-link.md#sub-xsl-gapcon) 和 [`GAPELECTR`](../sub/sub-link.md#sub-xsl-gapelectr) 可用于定义表面之间的机械、热和电气相互作用。

### 材料选项

密度、机械性能和电气性能可以包含在由用户子程序 [`UMATHT`](../sub/sub-link.md#sub-xsl-umatht) 定义其本构热行为的材料定义中。

### 单元

用户子程序 [`UMATHT`](../sub/sub-link.md#sub-xsl-umatht) 可用于Abaqus/Standard中所有包含热行为的单元（具有温度自由度的单元，如纯热传递、耦合热应力以及耦合热电单元）。
