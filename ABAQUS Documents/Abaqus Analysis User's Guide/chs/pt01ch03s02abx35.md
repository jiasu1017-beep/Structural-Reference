# 3.2.35 与 ZAERO 交换 Abaqus 数据





**产品：**Abaqus/Standard

##### **参考**

- [『Abaqus 执行程序：概述』，第 3.1.1 节](pt01ch03s01abo02.md)

### 概述

**abaqus tozaero** 接口使您能够在 Abaqus 和 ZAERO 分析产品之间交换气动弹性数据。通过在应用程序之间使用此接口，您可以在 Abaqus 中对模型进行结构模态分析，将其传输到 ZAERO 进行气动弹性分析，然后再传回 Abaqus 进行应力分析。

### 通用文件

通用文件是 Abaqus 和 ZAERO 之间数据交换的媒介。它由四个数据集组成：2411（描述节点和坐标数据）、2414（描述质量归一化振型）、2420（描述全局坐标系）和 2453（以文本格式描述质量矩阵），或 2453b（以二进制格式描述质量矩阵）。

您可以使用 **mode** 参数指定通用文件的输出格式。选择文本格式使您可以在文本编辑器中修改通用文件，但会使文件大小增加到二进制格式相同文件的兩倍以上。文本是默认格式，也是 ZAERO 支持的唯一格式。表 3.2.35-1（第 3.2.35-1 表）和表 3.2.35-2（第 3.2.35-2 表）分别描述了质量矩阵数据集的文本格式和二进制格式。

### 准备 Abaqus 分析输入文件

在接口可以创建通用文件之前，您必须对 Abaqus 输入（`.inp`）文件进行以下添加，然后运行 Abaqus：
- 在特征频率提取分析中对特征向量进行质量归一化（相对于结构的质量矩阵）。这种归一化是必要的，因为转换器假定振型是质量归一化的；如果您在 Abaqus 运行之前跳过此步骤，则转换的模态将不正确，并且会产生不正确的结果而不会发出警告或错误。有关更多信息，请参阅『自然频率提取』，第 6.3.5 节（第 6.3.5 节）。
- 在分析步骤中包含以下行： ``` [*ELEMENT MATRIX OUTPUT](../key/key-link.md#usb-kws-helemmatout), ELSET=allelements, MASS=YES, OUTPUT FILE=USER DEFINED, FILE NAME=*mtx-file-name* ``` 其中 `allelements` 是一个定义的单元集，包含应包含在全局质量矩阵中的所有单元。矩阵输出将放置到文件 `*mtx-file-name*.mtx` 中；您不应指定 `.mtx` 扩展名，因为 Abaqus 会自动添加。

### 工作流程

本节描述 Abaqus 和 ZAERO 之间工作流程中三个主要步骤的输入和输出。

#### 在 Abaqus 中进行模态分析

Abaqus 模态分析使用 Abaqus 输入文件，并将以下数据输出到输出数据库（`.odb`）文件和矩阵（`.mtx`）文件：结构模型节点、坐标系、模态频率、广义质量、振型和质量矩阵。

#### 在 ZAERO 中进行气动弹性分析

气动弹性分析需要 ZAERO 输入文件和由 toZAERO 创建的通用文件。ZAERO 将由于气动弹力而在结构节点上的力和力矩数据输出到通用文件。

#### 在 Abaqus 中进行应力分析

然后，ZAERO 输出的力和力矩可用于静态（线性或非线性）Abaqus 分析，以计算位移、应力和载荷。

### 命令行选项

**job**

此选项用于指定 Abaqus 输入文件的名称。它也是通用输出数据库和质量矩阵文件的默认名称。

**unvfile**

此选项用于在名称与 `*job-name*` 不同时指定通用文件的名称。如果未提供 `.unv` 扩展名，Abaqus 会自动添加。

**odbname**

此选项用于在名称与 `*job-name*` 不同时指定 Abaqus 输出数据库文件的名称。如果未提供 `.odb` 扩展名，Abaqus 会自动添加。

**mtxfile**

此选项用于指定包含 Abaqus 生成的单元质量矩阵的文件。如果未提供 `.mtx` 扩展名，Abaqus 会自动添加。

**step**

此选项指定包含 Abaqus 特征频率提取结果的步骤编号。默认值为 1。

**注意：**您必须在特征频率提取分析中对特征向量进行质量归一化（相对于结构的质量矩阵）。有关更多信息，请参阅『自然频率提取』，第 6.3.5 节（第 6.3.5 节）。

**mode**

此选项指定通用文件的输出格式。如果此选项设置为 `binary`，Abaqus 会以二进制格式写入部分通用文件以节省空间。如果此选项设置为 `text`，Abaqus 会以全文本格式写入整个文件。默认值为 `text`，这也是 ZAERO 当前支持的唯一模式。
