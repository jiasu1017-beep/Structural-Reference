# 15.13.15 定义流固协同仿真相互作用







您可以使用流固协同仿真相互作用来定义协同仿真的接口边界。协同仿真是在Abaqus/CFD与Abaqus/Standard或Abaqus/Explicit之间进行的，具体取决于结构模型中使用的分析步骤类型。从主菜单栏中选择****相互作用**![](../graphics/images/arrow.gif)**创建****，然后选择用于数据交换的边界区域；协同仿真的耦合方案由分析自动确定。更多信息，请参见第26章"协同仿真"。

对于涉及壳/膜的流固相互作用的Abaqus/CFD分析，接缝在网格中定义了零厚度壳。您可以创建一个表示接缝的双侧表面，然后选择该表面作为边界区域。更多信息，请参见"使用相互作用模块的Special菜单"中的"建模裂缝和接缝"，第15.12.14节。

在流体模型（Abaqus/CFD）中，FSI协同仿真相互作用只能在流动步骤中创建。在结构模型（Abaqus/Standard或Abaqus/Explicit）中，FSI协同仿真相互作用只能在隐式动态、显式动态或热传递步骤中创建。

**定义流固协同仿真相互作用：**

1. 从主菜单栏中，选择****相互作用**![](../graphics/images/arrow.gif)**创建****。**提示：**您也可以使用相互作用模块工具箱中的![](../graphics/ico_interaction.png)工具创建流固协同仿真相互作用。
2. 在出现的**创建相互作用**对话框中，执行以下操作：- 命名该相互作用。有关命名对象的信息，请参见第3.2.1节"使用基本对话框组件"。- 选择将创建该相互作用的步骤。在Abaqus/CFD流体模型中，步骤必须是流动步骤。- 选择**流固协同仿真边界**类型的相互作用。
3. 单击**继续**关闭**创建相互作用**对话框。
4. 使用以下方法之一选择区域：- 使用现有表面定义区域。在提示区域右侧单击**表面**。从出现的**区域选择**对话框中选择现有表面，然后单击**继续**。**注意：**默认选择方法基于您最近使用的选择方法。要恢复为另一种方法，请单击提示区域右侧的**在视口中选择**或**表面**。- 使用鼠标在视口中选择区域。更多信息，请参见"在当前视口中选择对象"，第6.2节。如果模型包含网格和几何的组合，请从提示区域单击以下之一：- 如果要从几何区域选择表面，请单击**几何**。- 如果要从本机或孤立网格选择表面，请单击**网格**。您可以使用角度方法从几何或网格中选择一组面或边或一组单元面。更多信息，请参见"使用角度和特征边缘方法选择多个对象"，第6.2.3节。
5. 单击**确定**创建相互作用并关闭编辑器。
