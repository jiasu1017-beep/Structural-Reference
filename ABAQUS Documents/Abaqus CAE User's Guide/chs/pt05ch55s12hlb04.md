# 55.12.4 为没有结果的元素着色







要生成轮廓图，Abaqus/CAE 会从输出数据库中读取您指定的变量和帧的结果。特定变量和帧的结果可能不可用或不适用于绘图中所包含的一个或多个元素。您可以为没有结果的元素指定显示颜色。默认颜色为白色。

例如，[图 55–18](pt05ch55s12hlb04.md#viw-noresults) 中的轮廓图显示了使用模具和冲头使坯料变形时产生的 von Mises 应力（图中未显示冲头）。在这种情况下，为没有结果的元素选择了灰色。模具元素显示为灰色，因为模具是刚性表面，不存在应力结果。

**图 55–18** 坯料和模具的轮廓图。

![](../graphics/viw-noresults.gif)

**为没有结果的元素着色：**

1. 找到 **没有结果的元素** 选项。从主菜单栏，选择 ****视图**![](../graphics/images/arrow.gif)**ODB 显示选项****。在出现的对话框中点击 **常规** 选项卡。**没有结果的元素** 选项位于 **常规** 页面的中心。
2. 点击颜色样本 ![](../graphics/usi-colorsample.png)。Abaqus/CAE 显示 **选择颜色** 对话框。
3. 使用 **选择颜色** 对话框中的方法之一选择新颜色。有关更多信息，请参见 ["自定义颜色，" 第 3.2.9 节](pt01ch03s02s09.md)。
4. 点击 **确定** 关闭 **选择颜色** 对话框。颜色样本变为所选颜色。
5. 点击 **应用** 实现您的更改。轮廓图（包括显示体）中没有结果的元素会根据您的着色规格进行更改，这会在会话期间保存。如果您希望保留所做的更改以供后续会话使用，请将它们保存到文件。有关更多信息，请参见 ["保存自定义设置以供后续会话使用，" 第 55.1.1 节](pt05ch55s01s01.md)。
