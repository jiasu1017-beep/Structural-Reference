# 1.4.16 Abaqus/Explicit单元载荷验证






**产品：**Abaqus/Explicit  

### I. 重力载荷

### 测试的单元

MASS、T2D2、T3D2、B21、B31、PIPE21、PIPE31、SAX1、S3R、S4R、M3D3、M3D4R、

CPE3、CPE4R、CPS3、CPS4R、CAX3、CAX4R、C3D4、C3D6、C3D8R、

### 测试的功能

重力和非结构质量。

### 问题描述

在此验证测试中，所有可用的单元类型都通过重力载荷进行测试。所有单元节点的位置都是固定的，节点上产生的反作用力用于验证单元载荷计算。

材料模型为各向同性线弹性。使用的材料属性定义如下：杨氏模量 = 193.1×10^9，泊松比 = 0.3，密度 = 7850。

在定义单元质量的非结构质量贡献时，通过将材料密度降低至所增加非结构质量的程度来维持上述指定的有效密度。由于GRAV载荷同时施加在结构质量和非结构质量上，用于验证数值结果的解析解保持不变。

在第一个步骤中，重力载荷沿垂直方向（*y*方向）施加。该重力载荷的幅值函数定义如下：在步骤的前半部分载荷线性增加至10，并在后半部分保持恒定。在第二个步骤中，垂直方向的重力载荷被替换为水平方向（*x*方向）的重力载荷，其幅值函数与垂直载荷类似。

### 结果与讨论

所有单元的结果与分析值一致，分析值包含在输入文件的顶部。

### 输入文件

[element_grav.inp](../eif/element_grav.inp)

用于此测试的输入数据。

### II. 均匀体积力

### 测试的单元

T2D2、T3D2、SAX1、S3R、S4R、M3D3、M3D4R、

CPE3、CPE4R、CPS3、CPS4R、CAX3、CAX4R、C3D4、C3D6、C3D8R、

### 测试的功能

均匀体积力。

### 问题描述

在此验证测试中，所有可用的单元类型都通过均匀体积力进行测试。所有单元节点的位置都是固定的，节点上产生的反作用力用于验证单元载荷计算。

材料模型为各向同性线弹性。使用的材料属性定义如下：杨氏模量 = 193.1×10^9，泊松比 = 0.3，密度 = 785。

在第一个步骤中，对于除轴对称单元外的所有单元，均匀体积力1.0×10^5沿*x*方向施加；对于轴对称单元，则沿*r*方向施加。该体积力的幅值函数定义如下：在步骤的前半部分载荷线性增加，并在分析剩余部分保持恒定。在第二个步骤中，对于除轴对称单元外的所有单元，另一个均匀体积力1.0×10^5沿*y*方向施加；对于轴对称单元，则沿*z*方向施加。该载荷使用与第一个步骤相同的幅值函数。对于C3D4、C3D6、C3D8R、S3R、S4R、M3D3和M3D4R单元，在第三个步骤中沿*z*方向施加另一个均匀体积力1.0×10^5。该载荷同样使用与第一个步骤相同的幅值函数。

### 结果与讨论

所有单元的结果与分析值一致，分析值包含在输入文件的顶部。

### 输入文件

[element_body.inp](../eif/element_body.inp)

用于此测试的输入数据。

### III. 均匀压力载荷

### 测试的单元

**分布单元载荷**
RAX2、R2D2、R3D3、R3D4、B21、B31、PIPE21、PIPE31、SAX1、S3R、S4R、M3D3、M3D4R、CPE3、CPE4R、CPS3、CPS4R、CAX3、CAX4R、C3D4、C3D6、C3D8R、

**分布表面载荷**
RAX2、R2D2、R3D3、R3D4、SAX1、S3R、S4R、M3D3、M3D4R、CPE3、CPE4R、CPS3、CPS4R、CAX3、CAX4R、C3D4、C3D6、C3D8R、

### 测试的功能

通过分布单元载荷和表面载荷施加的均匀压力载荷。

### 问题描述

在这些验证测试中，所有可用的单元类型都通过使用分布单元载荷和分布表面载荷的均匀压力进行测试。所有单元节点的位置都是固定的，节点上产生的反作用力用于验证载荷应用。管道单元（PIPE21和PIPE31）仅通过分布单元载荷进行测试。多个步骤用于施加不同的载荷。在每个步骤开始时，移除先前步骤中施加的所有载荷。载荷在每个步骤的前半部分线性增加，并在后半部分保持恒定。

所有单元使用各向同性线弹性材料。使用的材料属性定义如下：杨氏模量 = 193.1×10^9，泊松比 = 0.3，密度 = 785。

对于单元载荷情况下的梁单元（B21、B31）和管道单元（PIPE21、PIPE31），在第一和第二个步骤中分别沿*x*和*y*方向施加1.0×10^5的单位长度均匀分布力。在第三个步骤中，对于三维梁单元（B31）和管道单元（PIPE31），沿*z*方向施加1.0×10^5的单位长度均匀分布力。

对于壳单元（S3R、S4R）和轴对称线单元（SAX1），在第一个步骤中施加1.0×10^5的单位面积均匀分布法向力。

对于三边平面单元（CPE3、CPE6M、CPS3、CPS6M）和轴对称单元（CAX3、CAX4R），在前三个步骤中在每个单元边上施加1.0×10^5的单位长度均匀分布法向力。

对于四边平面单元（CPE4R、CPS4R）和轴对称单元（CAX4R），在前四个步骤中在每个单元边上施加1.0×10^5的单位长度均匀分布法向力。

对于四面体三维连续体单元（C3D4、C3D10M），在前四个步骤中在每个面上施加1.0×10^5的单位面积均匀分布力。

对于棱柱体三维连续体单元（C3D6），在前五个步骤中在每个面上施加1.0×10^5的单位面积均匀分布力。

对于六面体三维连续体单元（C3D8），在前六个步骤中在每个面上施加1.0×10^5的单位面积均匀分布力。

对于表面载荷，在第一个步骤中在其中一个单元边表面（CPE3、CPE4R、CPS3、CPS4R、CAX3、CAX4R、SAX1、R2D2和RAX2单元）或单元面（C3D4、C3D6、C3D8R、S3R、S4R、M3D3、M3D4R、R3D3和R3D4单元）上施加1.0×10^5的均匀压力。在第二个步骤中，将相同的均匀压力施加在其他单元边表面或单元面上。

### 结果与讨论

所有单元的结果与分析值一致，分析值包含在输入文件的顶部。

### 输入文件

[element_pres.inp](../eif/element_pres.inp)

用于此测试的分布单元压力载荷的输入数据。

[surface_pres.inp](../eif/surface_pres.inp)

用于此测试的分布表面压力载荷的输入数据。

### IV. 粘性压力载荷

### 测试的单元

**分布单元载荷和表面载荷**
SAX1、S3R、S4R、M3D3、M3D4R、CPE3、CPE4R、CPS3、CPS4R、CAX3、CAX4R、C3D4、C3D6、C3D8R、

### 测试的功能

粘性压力载荷。

### 问题描述

在此验证测试中，所有可用的单元类型都通过粘性压力载荷进行测试。属于平面应变、平面应力和轴对称单元（CPE3、CPE4R、CPS3、CPS4R、CAX3和CAX4R）的节点在*x*方向上受到约束；在*y*方向上施加100的初始速度。属于三维单元（C3D4、C3D6和C3D8R）的节点在*x*和*z*方向上受到约束；在*y*方向上施加100的初始速度。属于壳单元和膜单元（S3R、S4R、M3D3和M3D4R）的节点在*x*和*y*方向上受到约束；在*z*方向上施加100的初始速度。属于轴对称壳单元（SAX1）的节点在*z*方向上受到约束；在*r*方向上施加100的初始速度。

材料模型为各向同性线弹性。使用的材料属性定义如下：杨氏模量 = 193.1×10^9，泊松比 = 0.3，密度 = 7850。粘性系数为1000。

粘性压力载荷在节点上产生反作用力，用于验证单元载荷计算。此测试只有一个步骤。

### 结果与讨论

所有单元的结果与分析值一致，分析值包含在输入文件的顶部。

### 输入文件

[element_vpres.inp](../eif/element_vpres.inp)

用于此测试的分布单元载荷的输入数据。

[surface_vpres.inp](../eif/surface_vpres.inp)

用于此测试的分布表面载荷的输入数据。

### V. 粘性体积力和滞止载荷

### 测试的单元

**分布单元载荷和表面载荷**
SAX1、S3R、S4R、M3D3、M3D4R、CPE3、CPE4R、CPS3、CPS4R、CAX3、CAX4R、C3D4、C3D6、C3D8R、

### 测试的功能

粘性体积力和滞止载荷。

### 问题描述

在此验证测试中，所有可用的单元类型都通过粘性体积力或滞止载荷进行测试。属于平面应变、平面应力和轴对称单元（CPE3、CPE4R、CPS3、CPS4R、CAX3和CAX4R）的节点在*x*方向上受到约束；在*y*方向上施加100的初始速度。属于三维单元（C3D4、C3D6和C3D8R）的节点在*x*和*z*方向上受到约束；在*y*方向上施加100的初始速度。属于壳单元和膜单元（S3R、S4R、M3D3和M3D4R）的节点在*x*和*y*方向上受到约束；在*z*方向上施加100的初始速度。属于轴对称壳单元（SAX1）的节点在*z*方向上受到约束；在*r*方向上施加100的初始速度。

材料模型为各向同性线弹性。使用的材料属性定义如下：杨氏模量 = 193.1×10^9，泊松比 = 0.3，密度 = 7850。

粘性体积力和滞止载荷在节点上产生反作用力，用于验证单元载荷计算。

### 结果与讨论

粘性体积力载荷提供了一种定义质量比例阻尼的替代方法，作为相对速度和步骤相关阻尼系数的函数。在粘性体积力载荷测试中，结果与使用阻尼系数为7.85的质量比例阻尼获得的结果一致。

[surface_sp.inp](../eif/surface_sp.inp)

分布表面滞止压力载荷的输入数据。

[element_sp.inp](../eif/element_sp.inp)

分布单元滞止压力载荷的输入数据。
