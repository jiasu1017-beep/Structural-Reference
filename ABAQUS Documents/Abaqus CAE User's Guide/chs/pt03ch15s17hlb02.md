# 15.17.2 定义阻尼

您可以为可用的相对运动分量定义类似阻尼器的阻尼行为。有关更多信息，请参见["连接器阻尼行为，"Abaqus分析用户指南第31.2.3节](../usb/usb-link.md#usb-elm-econndampingbehav)。

**要创建或编辑阻尼：**

1. 使用以下方法之一显示连接器截面编辑器：
   - 要创建新的连接器截面，请按照["创建连接器截面，"第15.12.11节"](pt03ch15s12s01.md)中概述的过程进行操作。
   - 要编辑现有的连接器截面，请从主菜单栏选择**连接器**![](../graphics/images/arrow.gif)**截面**![](../graphics/images/arrow.gif)**管理器**，从出现的列表中选择连接器截面，然后点击**编辑**。
2. 在**编辑连接器截面**对话框中，执行以下操作之一：
   - 要定义新的阻尼行为，请点击**添加**并从出现的菜单中选择**阻尼**。
   - 要编辑现有的阻尼行为，请从**行为选项**列表中选择该行为以显示该行为的关联数据字段。
3. 选择**定义**。
   - 选择**线性**以定义线性阻尼系数。
   - 选择**非线性**以将力/力矩定义为相对速度的表格函数。
4. 如果您要定义线性阻尼行为，
   1. 在**力/力矩**字段中，打开与您正在定义阻尼行为的相对速度一致的力或力矩的开关。
   2. 在**耦合**字段中，选择以下之一：
      - 选择**非耦合**以指定与相对速度关联的各个阻尼系数；例如，**C11**、**C22**和**C33**，由**力/力矩**选择确定。您可以使用单个阻尼选项来指定所有系数，即使每个相对速度的值不同。
      - 选择**耦合**以指定与相对速度关联的耦合阻尼系数；例如，**C11**、**C12**和**C13**，由**力/力矩**选择确定。
5. 如果您要定义非线性、非耦合行为，
   1. 在**力/力矩**字段中，打开与您正在定义阻尼行为的相对速度一致的力或力矩的开关。如果多个分量的行为相同，您可以定义将使用此函数的单个阻尼行为。如果多个分量的行为不同，则必须定义单独的阻尼行为。
   2. 在**耦合**字段中选择**非耦合**，以将力/力矩指定为其各自可用相对速度分量的表格函数。
6. 如果您要定义非线性、耦合行为，
   1. 在**力/力矩**字段中，打开与您正在定义阻尼行为的相对速度一致的力或力矩的开关。
   2. 在**耦合**字段中选择**基于位置耦合**或**基于运动耦合**，以将力/力矩指定为一个或多个相对位置或速度分量的函数。
   3. 在**独立分量**字段中，打开您正在为其定义耦合阻尼行为的可用相对位置或本构运动分量的开关。您可能需要在步骤编辑器中使用非对称方程求解器来改善收敛。
7. 要定义依赖于频率、温度或场变量的行为数据，请执行以下操作：
   1. 如果可用，请打开**使用与频率相关的数据**以定义随频率变化的行为数据。表格数据区域中会出现一个标有**频率**的列。
   2. 打开**使用与温度相关的数据**以定义随温度变化的行为数据。表格数据区域中会出现一个标有**温度**的列。
   3. 要定义依赖于场变量的行为数据，请点击**场变量数量**字段右侧的箭头以增加或减少场变量的数量。表格数据区域中会出现场变量列。
8. 在表中输入阻尼数据。您可以使用键盘在表中输入数据。或者，您可以在表中的任意位置点击鼠标按钮3以查看用于指定表格数据的选项列表。有关每个选项的详细信息，请参见["输入表格数据，"第3.2.7节"](pt01ch03s02s07.md)。
9. 要修改正则化（Abaqus/Explicit分析专用）或数据外推的行为设置，请使用["为表格数据指定行为设置，"第15.17.14节"](pt03ch15s17hlb14.md)中所述的过程。
10. 选择以下之一：
    - 如果您想继续定义行为，请点击**添加**，选择所需的行为，并继续连接器截面定义。有关定义其他行为的说明，请参见["使用连接器截面编辑器，"第15.17节"](pt03ch15s17.md)。
    - 如果您想查看或编辑现有行为，请从**行为选项**列表中选择它。有关编辑行为的说明，请参见["使用连接器截面编辑器，"第15.17节"](pt03ch15s17.md)。
    - 如果您想保存连接器截面定义并退出编辑器，请点击**确定**。
