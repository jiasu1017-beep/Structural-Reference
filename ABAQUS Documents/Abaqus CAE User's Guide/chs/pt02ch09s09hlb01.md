# 9.10.1 将不支持的关键词添加到您的Abaqus/CAE模型中





Abaqus/CAE使用您的模型定义来生成Abaqus/Standard、Abaqus/Explicit或Abaqus/CFD关键词和数据，这些内容在您提交分析作业时被放在输入文件中。目前，Abaqus/CAE可能不支持您可能希望包含在模型中的Abaqus/Standard、Abaqus/Explicit或Abaqus/CFD功能。如果是这样，您可以使用**关键词编辑器（Keywords Editor）**添加功能。从主菜单栏选择****模型（Model）**![](../graphics/images/arrow.gif)**编辑关键词（Edit Keywords）**![](../graphics/images/arrow.gif)***模型名称*****来启动**关键词编辑器**。

要使用**关键词编辑器**，您应该熟悉Abaqus关键词和数据的语法。例如，"步骤（Step）"模块不允许您为声学和耦合声学-结构分析提供边界阻抗或非反射边界。要提供边界阻抗或非反射边界，您可以使用**关键词编辑器**将[*IMPEDANCE*](../key/key-link.md#usb-kws-himpedance)关键词添加到模型中。

当您在"作业（Job）"模块中提交模型进行分析时，Abaqus/CAE会将您使用**关键词编辑器**所做的更改合并到提交进行分析的输入文件中。您使用**关键词编辑器**添加到模型的关键词即使在您使用Abaqus/CAE修改或重新生成模型后仍会保留，因为Abaqus/CAE将**关键词编辑器**的内容与模型定义一起存储在模型数据库中。因此，输入文件的内容可能无效（例如，如果您添加了引用后来删除的步骤的关键词），分析可能会失败。

**警告：**如果您使用**关键词编辑器**编辑了原始模型，Abaqus/CAE在重启分析时会忽略这些更改。有关更多信息，请参阅["重启分析规则，"第19.6.3节"](pt03ch19s06s03.md)。

Abaqus/CAE会在**关键词编辑器**中缩进一些关键词，使输入文件在屏幕上更易于阅读。被缩进的关键词永远不能单独出现在输入文件中，必须始终与其他关键词一起使用。您在**关键词编辑器**中看到的缩进不会包含在Abaqus/CAE生成的输入文件中。

**关键词编辑器**不允许您编辑模型的几何形状；您必须使用Abaqus/CAE进行几何更改。因此，**关键词编辑器**仅在您生成网格后才可用。如果Abaqus/CAE生成的输入文件非常长，**关键词编辑器**会一次显示一个缓冲区的输入文件。您可以使用滚动条滚动浏览每个缓冲区，以及使用编辑器底部的按钮显示下一个或上一个缓冲区。

如果您使用**关键词编辑器**修改输入文件，然后使用Abaqus/CAE修改模型，Abaqus/CAE会将更改合并到输入文件中。如果在合并过程中出现冲突，Abaqus/CAE会发出警告消息。

建议您不要编辑Abaqus/CAE支持的关键词；例如，您应该使用"属性（Property）"模块，而不是**关键词编辑器**来更改材料的属性。这种方法保持了模型直接支持的方面与**关键词编辑器**添加的方面之间的一致性。如果您确实使用**关键词编辑器**编辑了关键词，然后使用Abaqus/CAE对引用相同关键词的模型进行更改，则会发生冲突。

为了最小化**关键词编辑器**中显示的输入文件长度，Abaqus/CAE会隐藏需要大量数据的关键词的数据行，特别是[*NODE*](../key/key-link.md#usb-kws-mnode)、[*ELEMENT*](../key/key-link.md#usb-kws-melement)和[*DISTRIBUTION*](../key/key-link.md#usb-kws-mdistribution)。如果您在**关键词编辑器**中修改紧跟此类关键词的线条，Abaqus/CAE在合并和写入输入文件时会发生冲突。

有关解决**关键词编辑器**生成的输入文件冲突的提示，请参阅["解决输入文件中的冲突，"第9.10.3节"](pt02ch09s09hlb03.md)。

您可以通过从主菜单栏选择****帮助（Help）**![](../graphics/images/arrow.gif)**关键词浏览器（Keyword Browser）****来查看Abaqus/CAE支持的关键词。

**要编辑模型的关键词：**

1. 从主菜单栏，选择****模型（Model）**![](../graphics/images/arrow.gif)**编辑关键词（Edit Keywords）**![](../graphics/images/arrow.gif)***模型名称*****。**关键词编辑器**会出现，并显示与您选择的模型关联的关键词。**注意：**关键词仅在您生成网格后才可编辑。**注意：**如果您在对象名称或描述中包含星号（*），内容会被截断。
2. 输入文件中的每个关键词都显示在自己的块中。**关键词编辑器**左下角的按钮允许您执行以下操作：****在之后添加（Add After）**** 在所选块下方添加一个空文本块；您添加到新块的文本以蓝色显示。****删除（Remove）**** 删除使用**关键词编辑器**添加的所选文本块。您无法删除Abaqus/CAE生成的块。****放弃编辑（Discard Edits）**** 放弃您在最近使用**关键词编辑器**时对Abaqus/CAE生成的块所做的更改。****缓冲区 *x* / *y* : ![](../graphics/dbs-nextprevious.png)**** 显示大输入文件的下一个和上一个缓冲区。使用滚动条在缓冲区内滚动。此外，您可以单击任何块并编辑其中的文本。青绿色文本表示您编辑的Abaqus/CAE生成的块。
3. 从**关键词编辑器**底部的按钮中，单击**确定（OK）**包含您的更改并关闭编辑器。单击**取消（Cancel）**关闭编辑器并忽略您的更改。单击**放弃所有编辑（Discard All Edits）**保持编辑器打开并删除您对输入文件所做的所有更改。
