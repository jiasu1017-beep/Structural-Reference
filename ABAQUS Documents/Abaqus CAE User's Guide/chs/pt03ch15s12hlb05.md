# 15.12.5 创建接触稳定化定义





接触稳定化引入粘性阻尼来反对两个表面之间的增量相对运动。此阻尼可用于在接触闭合之前稳定未约束的刚体运动，而不会降低结果的准确性。接触稳定化基于「Abaqus/Standard中通用接触的稳定化」，Abaqus Analysis User's Guide第36.2.5节中描述的一组因子。

每个接触稳定化定义包含一组稳定化因子；通用接触定义指定应用每个稳定化定义的表面（请参见「为通用接触指定和修改接触稳定化分配」，第15.13.4节）。

**创建接触稳定化定义的步骤：**

1. 从主菜单栏选择 ****Interaction**![](../graphics/images/arrow.gif)**Contact Stabilization**![](../graphics/images/arrow.gif)**Create****。此时会出现「Edit Contact Stabilization」对话框。
2. 为接触稳定化定义输入「Name」。
3. 指定稳定化类型：- 选择「Define new stabilization behavior」来定义标准稳定化。- 选择「Reset values from previous steps」来定义一种特殊的稳定化类型，以取消在先前分析步骤中应用的稳定化效果；取消稳定化仍然需要分配给通用接触定义中的表面。此类稳定化不需要任何额外数据。
4. 指定「Zero stabilization distance」；对于分离距离大于此值的表面，不应用稳定化。间隙相关比例因子在分析过程中在1（当表面接触时）和0（当表面之间的间隙超过指定的零稳定化距离时）之间变化。- 选择「Analysis default」将间隙距离设置为特征表面尺寸。- 选择「Specify」直接输入间隙距离。
5. 指定「Reduction factor」以确定阻尼值在连续增量中如何变化。小于1的值会导致阻尼随每个增量减小；大于1的值（不推荐）会导致阻尼随每个增量增加。
6. 指定「Scale factor」，该因子应用于法向的稳定化阻尼效应。
7. 指定「Tangential factor」，该因子应用于切向的稳定化阻尼效应。
8. 如有需要，选择「Amplitude」包络以在步骤过程中改变稳定化。或者，您可以点击![](../graphics/ico_amplitudeSmall.png)创建一个新的幅值。（更多信息，请参见第57章，「幅值工具集」。）选择非默认斜坡幅值的幅值可能会导致稳定化效应跨越多个分析步骤；详情请参见「Abaqus/Standard中通用接触的稳定化」，Abaqus Analysis User's Guide第36.2.5节。
9. 如果您想将对话框中的值重置为默认值，请点击对话框底部的「Defaults」。
10. 点击「OK」保存您的接触稳定化定义并关闭「Edit Contact Stabilization」对话框。
