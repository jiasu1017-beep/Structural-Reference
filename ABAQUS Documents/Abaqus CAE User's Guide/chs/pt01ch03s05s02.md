# 3.5.2 结果树概述







结果树提供了会话中可用输出数据的可视化描述，包括所有打开的输出数据库和特定于会话的数据（如 X-Y 数据和 X-Y 绘图）。此外，结果树使您能够导航到当前模型数据库中的可查看内容，例如特定模型的一个步骤中指定的载荷。此工具与模型树共享 Abaqus/CAE 界面的左侧，并且仅在 Property 模块中还有材料库。您可以点击**模型**、**结果**或**材料库**选项卡来在模型树、结果树和材料库之间切换显示。（有关材料库的更多信息，请参阅[「使用材料库，第 12.5 节」](pt03ch12s05.md)。）结果树还使用与模型树相同的所有键盘和导航快捷键；有关更多信息，请参阅[表 3-2](pt01ch03s05s01.md#uss-int-tree-shortcuts)。

[图 3-23](pt01ch03s05s02.md#uss-int-resultstree) 显示了在 Getting Started with Abaqus: Interactive Edition 的[附录 C，「使用额外技术创建和分析 Abaqus/CAE 中的模型」](../gsa/gsa-link.md#gsa-chp-caehinge)中完成铰链模型教程的分析后结果树的外观。**输出数据库**容器显示当前会话中打开的所有输出数据库文件。在[图 3-23](pt01ch03s05s02.md#uss-int-resultstree)所示的示例中，**输出数据库**容器已展开，显示只有一个输出数据库打开——**PullHinge** 输出数据库。

**图 3-23** 完成铰链模型教程分析后的结果树。

![](../graphics/uss-int-resultstree-nls.png)

展开 **PullHinge** 后，如[图 3-24](pt01ch03s05s02.md#uss-int-resultstree-odbs-pullhinge)所示，显示此输出数据库具有以下容器：**历史输出**、**步骤**、**实例**、**材料**、**截面**、**元素集合**、**节点集合**和**表面集合**。此外，输出数据库包含两个空容器——**会话坐标系**和**ODB 坐标系**。您不能从结果树中删除空容器，尽管您可以隐藏空容器（请参阅[「更改模型的视图，第 3.5.4 节」](pt01ch03s05s04.md)）。

**图 3-24** **PullHinge** 输出数据库中的容器。

![](../graphics/uss-int-resultstree-odbs-pullhinge-nls.png)

展开**历史输出**容器后，如[图 3-25](pt01ch03s05s02.md#uss-int-resultstree-history)所示，显示此分析请求了 16 个历史输出变量。每个变量列表还描述了请求历史输出的区域；在此示例中，每个历史输出请求都是针对整个模型做出的。您可以点击结果树中的任何历史输出变量，以在当前视口中绘制所选变量。

**图 3-25** 结果树的**历史输出**容器。

![](../graphics/uss-int-resultstree-history-nls.png)

每个输出数据库还包括一个**步骤**容器，其中包含输出数据库中每个步骤的容器，在每个步骤内还有输出数据库中的每一帧。您可以使用结果树来显示分析任何帧处的模型、激活或停用分析中的步骤或帧，或在所选帧处显示场输出。

**模型数据库**容器显示当前模型数据库中的所有模型。您可以展开每个模型以选择要查看的数据的步骤，并显示或隐藏单个零件实例。[图 3-26](pt01ch03s05s02.md#uss-int-resultstree-mdbs) 显示了铰链模型及其展开的**步骤**和**实例**容器。

**图 3-26** 结果树的**模型数据库**容器。

![](../graphics/uss-int-resultstree-mdb-nls.png)

结果树中的其他容器提供仅在会话期间存在的数据的快捷方式。通过使用这些快捷方式，您可以创建和管理轮廓光谱；创建和编辑 X-Y 数据并显示 X-Y 绘图，创建和管理路径和显示组；以及上传和显示背景图像和电影。
