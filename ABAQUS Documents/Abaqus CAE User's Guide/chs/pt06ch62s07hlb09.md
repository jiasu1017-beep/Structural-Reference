# 62.7.9 通过将现有边旋转指定角度创建datum轴







您可以通过围绕选定的旋转轴将选定的边或datum轴旋转指定角度来定义datum轴。

![](../graphics/dtm-harot-nls.png)

**通过将边旋转指定角度来创建datum轴：**

1. 从主菜单栏中，选择****Tools**![](../graphics/images/arrow.gif)**Datum****。此时将出现**Create Datum**对话框。该对话框概述了您可以创建的datum几何体类型。Abaqus/CAE在提示区域中显示提示以指导您完成此过程。**提示：**您也可以使用模块工具箱中的![](../graphics/ico_dtmAxisRotateLine.png)工具创建datum轴。有关工具箱中datum工具的图示，请参见["使用Datum工具集，" 第62.2节](pt06ch62s02.md)。
2. 从对话框顶部的类型列表中，选择**Axis**。**Method**列表指示您可以使用的方法来创建datum轴。
3. 从**Method**列表中，选择**Rotate from line**。
4. 从当前视口中的零件或装配中，选择要旋转的边或datum轴。
5. 指定旋转的技术取决于零件或装配的建模空间：**对于二维或轴对称零件或装配：** 1. 在旋转中心选择一个顶点或datum点。您也可以使用提示区域中出现的文本框输入旋转中心的精确坐标。 2. 在提示区域的文本框中，输入旋转角度。正角度表示绕*Z*轴逆时针旋转。datum轴会出现。 **对于三维零件或装配：** 1. 选择一条边或datum轴来表示旋转轴。Abaqus/CAE沿选定的边显示一个箭头，指示使用右手规则的旋转正方向。 2. 在提示区域出现的文本框中，输入旋转角度。datum轴会出现。您可以通过从主菜单栏中选择****Feature**![](../graphics/images/arrow.gif)**Edit****并选择datum轴来修改旋转角度。如果更改角度的符号，Abaqus/CAE会反转旋转方向。
