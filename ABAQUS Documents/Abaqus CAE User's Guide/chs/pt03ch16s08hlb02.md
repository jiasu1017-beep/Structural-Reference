# 16.8.2 创建边界条件







创建边界条件时，必须指定边界条件的名称、激活边界条件的步骤、边界条件类型以及要应用边界条件的装配件区域。

**创建边界条件：**

1. 从主菜单栏中，选择****边界条件**![](../graphics/images/arrow.gif)**创建****。将出现**创建边界条件**对话框，**名称**文本字段中显示默认名称。**提示：**您也可以使用载荷模块工具箱中的![](../graphics/ico_boundaryCondition.png)工具创建边界条件。
2. 为边界条件键入名称。有关命名对象的信息，请参见"使用基本对话框组件"，第3.2.1节。
3. 选择激活边界条件的步骤。单击**步骤**文本字段旁边的箭头，然后从出现的列表中选择。
4. 从对话框左侧的**类别**列表中，选择所需的类别。可用的**类别**选择取决于您正在执行的分析程序类型。对话框右侧的**所选步骤的类型**列表更改为所有可用边界条件类型的列表。
5. 从**所选步骤的类型**列表中，选择边界条件类型，然后单击**继续**。
6. 如果您要使用组装紧固件创建连接器边界条件，您可以单击提示区域中的**完成**以从模板模型中选择线集合。将出现边界条件编辑器。1. 单击**组装紧固件**字段旁边的箭头，然后从出现的列表中进行选择。编辑器中显示与组装紧固件关联的模板模型名称。**模板集合**列表填充了与所引用的模板模型关联的线集合。2. 从**模板集合**列表中选择一个线集合。您必须确保该线集合具有截面分配，该截面分配具有您要定义速度的相对运动可用分量。将显示相对运动可用分量对应的字段。
7. 对于所有其他边界条件类型，选择要应用边界条件的区域。如果要创建连接器位移、连接器速度或连接器加速度边界条件，您必须选择与连接器截面分配关联的线。选择线的最佳方法是使用线特征的默认几何集合名称（更多信息，请参见"为多个连接器创建或修改线特征"，第15.12.9节）。如果您选择多条线，您必须确保分配给线的连接器截面在连接器截面分配中具有您要定义位移、速度或加速度的相对运动可用分量。如果连接器边界条件的相对运动可用分量不足，则会出现一条消息，要求您选择不同的线或更改连接类型。如果要创建连接器材料流边界条件，您必须选择与连接器截面分配关联的线的端点。如果要创建欧拉网格运动边界条件，请在视口中选择一个欧拉部件实例。否则，使用以下方法之一选择边界条件的区域：- 在视口中选择一个区域。您可以使用角度方法从几何中选择一组面或边，或从网格中选择一组单元面。更多信息，请参见"使用角度和特征边缘方法选择多个对象"，第6.2.3节。完成选择后，单击鼠标按钮2。**提示：**您可以通过在**选择**工具栏中指定过滤选项来限制可以在视口中选择的对象类型。更多信息，请参见"使用选择选项"，第6.3节。如果模型包含网格和几何的组合，您必须选择要应用边界条件的区域类型。从提示区域中选择以下之一：- 单击**几何**将边界条件应用于几何或参考点。- 单击**网格**将边界条件应用于本机或孤立网格选择。默认情况下，会创建包含所选对象的集合或表面。您可以通过关闭提示区域中创建集合或表面的选项来更改此行为。提示区域中提供了默认名称，但您可以输入新名称。- 要从现有集合或表面列表中进行选择，请执行以下操作：1. 单击提示区域右侧的**集合**或**表面**。（按钮的名称取决于您正在创建的对象类型。例如，如果您要创建压力载荷，则会出现**表面**按钮。）Abaqus/CAE显示**区域选择**对话框，其中包含可用集合或表面的列表。2. 选择感兴趣的集合或表面，然后单击**继续**。**注意：**默认选择方法基于您最近使用的选择方法。要恢复为另一种方法，请单击提示区域右侧的**在视口中选择**或**集合**或**表面**。将出现边界条件编辑器。您正在应用边界条件的区域在视口中高亮显示。
8. 输入定义边界条件所需的所有数据，然后单击**确定**。**注意：**如果您创建的连接器位移边界条件超过连接器的失效准则，连接器位移将被忽略。有关编辑器特定功能的详细信息，请从主菜单栏中选择****帮助**![](../graphics/images/arrow.gif)**关于上下文****，然后单击感兴趣的功能，或参见"使用边界条件编辑器"，第16.10节。符号出现在视口中，代表您刚创建的边界条件。更多信息，请参见"理解代表规定条件的符号"，第16.5节。
