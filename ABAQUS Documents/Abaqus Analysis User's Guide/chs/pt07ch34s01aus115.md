# 34.1.2 幅值曲线









**产品：** Abaqus/StandardAbaqus/ExplicitAbaqus/CFD

##### **参考资料**

- [*AMPLITUDE*](#)
- ["幅值," 第2.1.1节](pt01ch02s01aus08.md)

### 概述

幅值曲线用于定义载荷、边界条件、预定义场和其他分析参数随时间或弧长（如适用）的变化关系。

在 Abaqus 中，幅值曲线可以用于：

- 定义载荷随时间的变化
- 定义边界条件的变化
- 定义预定义场的变化
- 定义接触属性参数的变化

### 幅值曲线类型

Abaqus 提供以下类型的幅值曲线：

| 幅值类型 | 描述 | 用途 |
| --- | --- | --- |
| TABULAR | 表格型幅值 | 用户指定时间-值数据点 |
| SMOOTH STEP | 光滑阶梯型幅值 | 连续平滑的载荷施加 |
| PERIODIC | 周期型幅值 | 周期性载荷 |
| PSDF | 功率谱密度函数型 | 随机振动分析 |
| RANDOM | 随机型幅值 | 随机载荷 |

### 表格型幅值（ TABULAR ）

表格型幅值是最常用的幅值类型，用户直接指定时间-值数据点。

| **输入文件用法：** | ```
*AMPLITUDE, NAME=name, DEFINITION=TABULAR
t1, v1
t2, v2
...
``` |
| --- | --- |

### 光滑阶梯型幅值（ SMOOTH STEP ）

光滑阶梯型幅值提供从零到指定值的平滑过渡，常用于避免瞬时冲击载荷。

| **输入文件用法：** | ```
*AMPLITUDE, NAME=name, DEFINITION=SMOOTH STEP
t1, v1
t2, v2
...
``` |
| --- | --- |

### 周期型幅值（ PERIODIC ）

周期型幅值用于定义周期性变化的载荷。

| **输入文件用法：** | ```
*AMPLITUDE, NAME=name, DEFINITION=PERIODIC, PERIOD=period_value
t1, v1
t2, v2
...
``` |
| --- | --- |

### 使用幅值曲线

在定义载荷或边界条件时，通过参数引用幅值曲线：

| **输入文件用法：** | ```
*CLOAD, OP=NEW
node, dof, amplitude=name
value
``` |
| --- | --- |
