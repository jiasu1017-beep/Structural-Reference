# 11.20 向零件添加特征





使用**形状**菜单向当前零件添加特征。您可以执行以下操作：
- 使用**实体**工具向三维实体零件添加实体特征。更多信息，请参见"添加实体特征"，第11.21节。
- 使用**壳**工具向零件添加壳特征。更多信息，请参见"添加壳特征"，第11.22节。
- 使用**线框**工具向零件添加线框特征。更多信息，请参见"添加线框特征"，第11.23节。
- 使用**切割**工具向零件添加切割特征。更多信息，请参见"添加切割特征"，第11.24节。
- 使用**混合**工具向三维实体零件添加混合特征。更多信息，请参见"混合边缘"，第11.27节。
- 使用**变换**工具关于选定平面对零件进行镜像。更多信息，请参见"镜像零件"，第11.28节。
