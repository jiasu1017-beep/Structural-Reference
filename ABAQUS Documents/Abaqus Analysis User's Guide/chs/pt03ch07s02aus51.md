# 11.1.3 非线性问题的收敛准则

**产品：** Abaqus/Standard 和 Abaqus/Explicit

##### **参考资料**

- ["求解非线性问题，" 第 11.1.1 节](pt03ch07s01aus49.md)
- ["常用求解控制参数，" 第 11.1.2 节](pt03ch07s02aus50.md)
- [*CONTROLS](../key/key-link.md#usb-kws-hcontrols)

### 概述

收敛准则用于判断非线性迭代是否已经达到可接受的解。Abaqus 提供了多种收敛准则来检查不同的解特征。

### 残差力收敛准则

残差力准则检查外力与内力之间的平衡。迭代被视为收敛当：

‖R‖ ≤ ε_R · max(‖F_ext‖, ‖F_int‖)

其中 R 是残差力向量，F_ext 是外力向量，F_int 是内力向量，ε_R 是容差。

| **输入文件用法：** | 使用以下选项启用残差力收敛检查： |
| --- | --- |
| | ``` [*CONTROLS](../key/key-link.md#usb-kws-hcontrols), CONVERGENCE=RESIDUAL ``` |

### 位移收敛准则

位移准则检查当前位移增量相对于参考值的大小。迭代被视为收敛当：

‖Δu‖ ≤ ε_u · max(‖u_ref‖, ‖Δu_ref‖)

| **输入文件用法：** | 使用以下选项启用位移收敛检查： |
| --- | --- |
| | ``` [*CONTROLS](../key/key-link.md#usb-kws-hcontrols), CONVERGENCE=DISPLACEMENT ``` |

### 能量收敛准则

能量准则检查能量增量的收敛性。迭代被视为收敛当：

ΔE ≤ ε_E · E_ref

其中 ΔE 是能量变化，E_ref 是参考能量。

| **输入文件用法：** | 使用以下选项启用能量收敛检查： |
| --- | --- |
| | ``` [*CONTROLS](../key/key-link.md#usb-kws-hcontrols), CONVERGENCE=ENERGY ``` |

### 组合收敛准则

Abaqus 默认使用组合收敛准则，要求所有启用的准则都满足：

| **输入文件用法：** | 使用以下选项启用所有收敛检查： |
| --- | --- |
| | ``` [*CONTROLS](../key/key-link.md#usb-kws-hcontrols), CONVERGENCE=ALL ``` |

### 容差值

收敛容差控制收敛要求的严格程度：

| **参数** | **描述** | **默认值** |
| --- | --- | --- |
| TOLERANCE | 通用容差值 | 0.005 |
| RESIDUAL TOLERANCE | 残差力容差 | 0.005 |
| DISPLACEMENT TOLERANCE | 位移容差 | 0.005 |
| ENERGY TOLERANCE | 能量容差 | 0.005 |

### 收敛准则选择指南

| **问题类型** | **推荐准则** |
| --- | --- |
| 线性/弱非线性 | 残差力 |
| 接触问题 | 位移 + 残差力 |
| 材料非线性 | 能量 |
| 通用非线性 | 能量 + 残差力 |

### 收敛困难识别

以下迹象表明收敛准则可能需要调整：

1. 迭代次数接近但未达到最大值
2. 残差力在迭代过程中振荡
3. 位移增量在微小范围内波动
4. 能量变化不单调

### 调整收敛准则

如果模型出现收敛困难，可以考虑：

1. 增大容差值（降低收敛要求）
2. 减小初始时间增量
3. 启用线性搜索
4. 使用弧长法（对于软化问题）
