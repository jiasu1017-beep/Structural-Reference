# 1.12.5 冲击管中压实波的传播

**产品：** Abaqus/Explicit

此问题建模多孔沙中压实波的一维传播。结果与 Wardlaw、McKeown 和 Chen（1996）中给出的解进行了比较。

### 问题描述

考虑的问题与["冲击管中的波传播"第 1.12.4 节](ch01s12ach91.md)中讨论的类似。这里我们考虑充满沙的分隔壁管的情况，如图 1.12.5-1（[图 1.12.5-1](ch01s12ach92.md#exxalecompaction-schematic)）所示。隔膜将管分成两个相等的隔室。左隔室充满初始压力 ![](../graphics/bmk_eqn00248.gif) = 423 MPa 的压实（完全饱和）沙。右隔室充满初始压力 ![](../graphics/bmk_eqn00248.gif) = 0 的部分饱和沙。分隔隔室的隔膜瞬间移除，导致压实波进入右隔室，稀疏波传回左隔室。压实波导致右隔室中沙的永久压实。

沙的力学响应使用 ![](../graphics/bmk_eqn01013.gif) 状态方程材料模型建模（["状态方程"《Abaqus 分析用户指南》第 25.2.1 节](../usb/usb-link.md#usb-mat-ceos)）。沙由沙粒、水和空气或孔隙组成。假定空气不承担任何压力。![](../graphics/bmk_eqn01013.gif) 模型的固相代表沙粒-水混合物，而多孔部分解释空气-孔隙含量。在没有空气-孔隙含量的情况下，沙被称为完全饱和或完全压实，这在 ![](../graphics/bmk_eqn01013.gif) 模型框架中对应于 ![](../graphics/bmk_eqn01014.gif)。完全饱和沙的本构行为由 Mie-Grneisen 状态方程描述。在其原始状态、零压力下，沙通常包含初始孔隙体积分数，被称为部分饱和。随着压力增加，沙经历不可逆压实和永久（塑性）体积变化。当压力达到压实压力 ![](../graphics/bmk_eqn01015.gif) 时，沙变得完全压实。

使用以下材料特性：

| 固相（Mie-Grneisen） | 压实特性 |
| --- | --- |
| ![](../graphics/bmk_eqn01016.gif) | 2070 kg/m³ | ![](../graphics/bmk_eqn01017.gif) | 600 m/sec |
| ![](../graphics/bmk_eqn01018.gif) | 1480 m/sec | ![](../graphics/bmk_eqn01019.gif) | 0.049758 |
| *s* | 1.93 | ![](../graphics/bmk_eqn01020.gif) | 0.0 MPa |
| ![](../graphics/bmk_eqn00797.gif) | 0.880 | ![](../graphics/bmk_eqn01015.gif) | 6.5 MPa |

假定左隔室中的沙完全压实（![](../graphics/bmk_eqn00276.gif) = 1），初始压力 ![](../graphics/bmk_eqn00248.gif) = 423 MPa（![](../graphics/bmk_eqn01021.gif)），初始比能 ![](../graphics/bmk_eqn01022.gif) = 5000 Joule/Kg。另一方面，右隔室中的沙最初处于原始状态：![](../graphics/bmk_eqn01023.gif)，![](../graphics/bmk_eqn01024.gif) = 0，孔隙率 ![](../graphics/bmk_eqn01025.gif) = 0.049758（或 ![](../graphics/bmk_eqn01026.gif) = 1.052364）。

使用平面应变 CPE4R 单元对管中填充体积为 5×0.1×1 m³ 的沙进行网格划分。在整个分析过程中，在管的所有四个外边界壁上规定对称边界条件。继续分析直到 ![](../graphics/bmk_eqn00138.gif) = 0.001 sec。

使用以下三种技术来解决问题：

1. 纯拉格朗日：使用纯拉格朗日分析解决问题；不执行自适应网格划分。
2. 带两个域的自适应网格划分：为每个隔室定义一个自适应网格域，并在每个域内执行连续自适应网格划分。由于两个域之间的边界区域，两个隔室之间的界面保持拉格朗日。这种约束的净效应是每个隔室中包含的沙不混合。将自适应网格划分的频率从默认值 10 更改为 1，因为冲击波传播时发生通过网格的大量材料流动。
3. 带一个域的自适应网格划分：使用传统欧拉方法执行分析。定义一个包含两个隔室的单一自适应网格域。这允许当隔膜移除时两个隔室的沙自由混合。当冲击波通过管移动时，可以使用两种技术之一保持网格静止：（1）在每个节点上施加空间自适应网格约束，或（2）基于上一个自适应网格增量结束时的节点位置执行自适应网格划分，这具有对于没有边界变形的均匀网格保持网格静止的效果。这里采用后一种技术。将自适应网格划分的频率从默认值 10 更改为 1，因为冲击波传播时发生通过网格的大量材料流动。

### 结果与讨论

在分析结束时（![](../graphics/bmk_eqn00138.gif) = 0.001 sec）沿管长度的路径图在密度 ![](../graphics/bmk_eqn00406.gif) 和膨胀 ![](../graphics/bmk_eqn00207.gif) 的[图 1.12.5-2](ch01s12ach92.md#exxalecompaction-density1-3) 和[图 1.12.5-3](ch01s12ach92.md#exxalecompaction-alpha1-3) 中显示。所有分析（情况 1、2 和 3）的结果几乎相同，这表明一维问题可以用所有三种方法满意地解决。然而，二维和三维冲击波问题通常需要第三种方法。[图 1.12.5-3](ch01s12ach92.md#exxalecompaction-alpha1-3) 显示当压实波向右侧隔室推进时，它在身后留下完全饱和沙的痕迹（![](../graphics/bmk_eqn01014.gif)）。在[图 1.12.5-4](ch01s12ach92.md#exxalecompaction-density3-ref) 中，情况 3（使用一个自适应网格域的分析）的密度路径图与 Wardlaw 等人给出的解进行了比较。结果非常一致。
