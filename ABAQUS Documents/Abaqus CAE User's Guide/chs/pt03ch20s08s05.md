# 20.8.5 复制草图对象以创建图案







您可以复制草图中的对象并创建复制对象的图案；例如，[图 20-19](pt03ch20s08s05.md#ske-copy-gear)展示实体零件上齿轮齿和孔的径向图案。

**图 20-19** 齿轮齿和孔的径向图案。

![](../graphics/ske-copy-gear.png)

您可以选择以下任一方法来定义图案：

**线性图案**

线性图案沿某个方向定位所选对象的副本；例如，*X* 方向。您可以指定副本的数量和副本之间的间距；并且您可以选择将副本放置在正方向或负方向。此外，您可以更改线性图案的方向。[图 20-20](pt03ch20s08s05.md#ske-copy-onedirection)显示了如何沿单一方向创建不同的线性图案。

**图 20-20** 单一方向的线性图案。

![](../graphics/ske-copy-onedirection-nls.png)

您可以通过在第二个方向（例如 *Y* 方向）创建副本来创建复制对象矩阵。选项与第一个方向相同；您可以控制副本数量、方向和角度。[图 20-21](pt03ch20s08s05.md#ske-copy-twodirections)显示了如何沿两个方向创建不同的线性图案。

**图 20-21** 两个方向的线性图案。

![](../graphics/ske-copy-twodirections-nls.png)

默认情况下，第一个方向是 *X* 轴，第二个方向是 *Y* 轴。但是，您可以通过从草图中选择一条代表新方向的直线来更改第一个或第二个方向的方向。[图 20-22](pt03ch20s08s05.md#ske-copy-line)显示沿选定直线定向的线性图案。

**图 20-22** 沿直线定向的线性图案。

![](../graphics/ske-copy-line-nls.png)

默认情况下，Abaqus/CAE 沿正方向创建图案；但是，您可以反转方向。例如，[图 20-23](pt03ch20s08s05.md#ske-copy-lineflip)显示了与[图 20-22](pt03ch20s08s05.md#ske-copy-line)相同但方向反转的线性图案。

**图 20-23** 反转线性图案方向的效果。

![](../graphics/ske-copy-lineflip.png)

有关创建线性图案的详细说明，请参见["创建对象的线性图案，"第 20.19.1 节](pt03ch20s16hlb01.md)。

**径向图案**

径向图案在圆形图案中定位所选对象的副本。您可以指定副本的数量，并且可以指定第一个和最后一个副本之间的角度，其中正角度对应逆时针方向。此外，您可以从草图中选择一个点来定义圆形图案的中心点。[图 20-24](pt03ch20s08s05.md#ske-copy-radialpattern)显示了如何创建不同的径向图案。

**图 20-24** 径向图案。

![](../graphics/ske-copy-radialpattern-nls.png)

有关创建径向图案的详细说明，请参见["创建对象的径向图案，"第 20.19.2 节](pt03ch20s16hlb02.md)。

默认情况下，Abaqus/CAE 会在您输入设置时显示线性和径向图案的预览。在大多数情况下，预览将帮助您决定将创建所需图案的设置。但是，如果要创建大量副本，预览可能会减慢草图的生成速度。在这种情况下，您应该关闭**预览**按钮。

还有几种方法可用于创建草图对象的单独副本。这些方法在["通过选择边修改或复制对象，"第 20.8.3 节](pt03ch20s08s03.md)中讨论。
