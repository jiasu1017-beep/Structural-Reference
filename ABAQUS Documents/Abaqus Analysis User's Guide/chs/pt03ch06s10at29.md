# 10.3.1 在 Abaqus/Explicit 中使用退火程序

**产品：** Abaqus/Explicit

##### **参考资料**

- ["退火程序，" 第 2.9 节](../stm/stm-link.md#stm-prc-annealing)
- [*ANNEAL](../key/key-link.md#usb-kws-hanneal)

### 概述

退火程序：
- 用于模拟金属成形操作中的退火处理，其中材料在成形操作后被加热以消除内应力并允许进一步成形；
- 可用于重新设置模型中所有单元的应力、应变和其他历史变量状态；
- 仅在 Abaqus/Explicit 中可用；以及
- 可用于静力分析和动力分析。

### 概述

退火是一种热处理工艺，用于消除材料中的内应力并软化材料，以便进行进一步的成形操作。在有限元分析中，退火过程会导致材料状态的完全重置，使得在退火后可以继续进行进一步的成形分析，而不会受到先前变形历史的影响。

退火程序可用于模拟：
- 金属成形操作后的退火处理；
- 在多步骤成形过程中重新设置材料状态；以及
- 任何需要重置材料历史的应用。

### 定义退火步骤

退火步骤的定义方式与其他分析步骤类似，但使用退火程序代替其他分析程序。

| **输入文件用法：** | 使用以下选项定义退火步骤： |
| --- | --- |
| | ``` [*STEP](../key/key-link.md#usb-kws-hstep) [*ANNEAL](../key/key-link.md#usb-kws-hanneal) [*END STEP](../key/key-link.md#usb-kws-hendstep) ``` |

### 退火过程中的状态重置

在退火步骤中：
- 所有单元的应力状态被重置为零；
- 所有历史变量（如等效塑性应变、损伤等）被重置为其初始值；
- 温度场被重新分布或重置（取决于定义的边界条件）；
- 单元的变形历史被清除。

### 应用

退火程序通常用于金属板材成形分析，其中：
1. 初始冲压操作产生零件形状；
2. 退火步骤消除应变硬化和残余应力；
3. 后续冲压操作继续成形零件。

### 限制

- 退火步骤会重置所有历史变量，但不会改变几何形状；
- 接触状态在退火过程中不被保留；
- 退火后的材料属性保持不变。

### 输入文件模板

```
[*HEADING](../key/key-link.md#usb-kws-mheading)
…
[*STEP](../key/key-link.md#usb-kws-hstep)
*定义成形分析步骤*
…
[*END STEP](../key/key-link.md#usb-kws-hendstep)
**
[*STEP](../key/key-link.md#usb-kws-hstep)
[*ANNEAL](../key/key-link.md#usb-kws-hanneal)
[*END STEP](../key/key-link.md#usb-kws-hendstep)
**
[*STEP](../key/key-link.md#usb-kws-hstep)
*定义后续成形步骤*
…
[*END STEP](../key/key-link.md#usb-kws-hendstep)
```
