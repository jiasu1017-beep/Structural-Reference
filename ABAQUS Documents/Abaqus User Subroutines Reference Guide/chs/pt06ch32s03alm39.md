**产品：** Abaqus/Standard

### 概述

JOINTC 单元：
- 用于模拟关节相互作用；以及
- 由局部、共旋转坐标系中的平移和旋转弹簧以及并联阻尼器组成。

单元公式的详细信息可在 ["柔性关节单元，" Abaqus 理论指南第3.9.6节](../stm/stm-link.md#stm-elm-jointc) 中找到。

### 典型应用

提供 JOINTC 单元来模拟两个几何上（几乎）重合的节点之间的相互作用，这些节点表示具有内部刚度和/或阻尼的关节（例如汽车悬架系统中的橡胶衬套），使得关节的第二个节点可以相对于第一个节点轻微位移和旋转。

仅有一个或两个旋转轴且没有相对位移的关节最好使用 REVOLUTE- 或 UNIVERSAL 型 MPC 建模（参见 ["通用多点约束，" 第35.2.2节"](pt08ch35s02aus130.md)）。

使用连接器可以获得类似的功能；参见 ["连接器概述，" 第31.1.1节"](pt06ch31s01abo28.md)。

### 定义关节行为

关节行为由并联的线性或非线性弹簧和阻尼器组成，耦合关节中相对位移和相对旋转的相应分量。您如 ["弹簧，" 第32.1.1节"](pt06ch32s01alm37.md) 和 ["阻尼器，" 第32.2.1节"](pt06ch32s02alm38.md) 中所述定义弹簧和阻尼器行为。

每个弹簧或阻尼器定义为一个六个局部方向中某个方向的行为；最多可包含六个弹簧和六个阻尼器定义。如果对于关节中的特定局部相对运动没有给出规范，则假定关节对该分量没有刚度。

关节行为可以在随单元第一个节点运动而旋转的局部坐标系中定义（["方向，" 第2.2.5节"](pt01ch02s02aus15.md)）。如果未定义局部坐标系，则使用全局坐标系。

您必须将关节行为与一组 JOINTC 单元相关联。

JOINTC 单元的运动行为在 ["柔性关节单元，" Abaqus 理论指南第3.9.6节](../stm/stm-link.md#stm-elm-jointc) 中有详细描述。

| **输入文件用法：** | 使用以下选项定义关节行为： |
| --- | --- |
|  | ``` [*JOINT](../key/key-link.md#usb-kws-mjoint), ELSET=*name*, ORIENTATION=*name* [*DASHPOT](../key/key-link.md#usb-kws-mdashpot) [*SPRING](../key/key-link.md#usb-kws-mspring) 最多可出现六个 [*SPRING](../key/key-link.md#usb-kws-mspring) 和 [*DASHPOT](../key/key-link.md#usb-kws-mdashpot) 选项。 ``` |

### 在大位移分析中使用 JOINTC 单元

在大位移分析中，力矩和旋转之间关系的公式将这些单元的实用性限制为小相对旋转。JOINTC 单元两端的相对旋转幅度应该足够小以符合小旋转条件。
