# 15.1.4 从原始坐标数据和特征向量创建扰动网格：FPERT





**产品：** Abaqus/Standard

本示例说明了使用FORTRAN程序通过将若干屈曲模式的加权和形式的微小缺陷叠加到初始几何上来创建扰动网格。该程序从Abaqus结果文件中检索原始节点坐标和所需的特征向量，然后计算扰动网格的新节点坐标。

### 概述

结构的屈曲后负载-位移（Riks）行为崩溃研究通常用于验证特征值屈曲分析预测的临界屈曲载荷和模式是否准确。它们还用于研究初始几何缺陷对负载-位移响应的影响。一个典型假设是，由与最低特征值相关的特征模态组合而成的缺陷将是最关键的。将这种类型的缺陷引入模型的一种方法是将添加到原始网格坐标中。在这种情况下，是第*i*个特征模态，是第*i*个特征模态的缩放因子，*M*是屈曲分析中提取的特征模态总数。由于特征向量通常归一化为最大绝对值1，通常是几何参数（如壳厚度）的一部分。下面描述的后处理程序可用于将这种类型的缺陷引入模型。

### 编程细节

在运行或修改此程序之前，应审查["Abaqus结果文件用户后处理概述，"第15.1.1节](ch15s01abo02.md)中关于编程概念和Abaqus FORTRAN接口的一般讨论。还建议审查Abaqus Analysis User's Guide第5章"文件输出格式"中的结果文件格式。

`FPERT`程序（在Abaqus发布媒体上命名为`fpert.f`）对将要读取的结果文件类型做一些假设。变量`NRU`、`LRUNIT(1,NRU)`和`LRUNIT(2,NRU)`在程序内部分别初始化为`1`、`8`和`2`。这些值表示将读取一个文件，将使用的FORTRAN单元为`8`，文件类型为二进制。有关打开和初始化后处理文件的更多信息，请参见["访问结果文件信息，" Abaqus Analysis User's Guide的第5.1.3节](../usb/usb-link.md#usb-out-faccessinfo)。

一旦设置了文件规范参数，就调用`INITPF`和`DBNRU`子程序来打开和准备好要读取的文件（其名称存储在`FNAME`中）。可以直接使用FORTRAN `OPEN`语句打开要写入扰动坐标的文件。不需要Abaqus文件工具，因为该文件是纯文本文件。

使用`DBFILE`例程读取包含原始节点坐标的记录，并存储在本地数组`COORDS(3,8000)`中。`COORDS`数组的第一个索引表示节点的*x*、*y*和*z*坐标。第二个索引表示节点号。如果模型中的节点超过8000个，应增加第二个维度。

特征向量的分量存储在本地数组`DISP(6,8000)`中。该数组保存每个节点最多6个位移项。如果模型中的节点超过8000个，应增加第二个维度。然后调用本地到此后处理程序的子程序`NODEGEN`来计算新的节点坐标。一旦所有请求的模态形状计算完成，新节点坐标就会被写入之前打开的纯文本文件。

### 程序编译和链接

**abaqus make**工具旨在编译和链接此类后处理程序。它还会在编译期间提供`aba_param.inc`文件。编译和链接`FPERT`程序的命令如下：

```
abaqus make job=fpert
```

如果在编译或链接过程中发现FORTRAN错误，必须重复此命令。必要时可以更改**abaqus make**工具使用的命令。[Abaqus Installation and Licensing Guide](../sgb/sgb-link.md#sgb)列出了每种计算机类型的典型编译和链接命令。

### 程序执行

在执行程序之前，必须使用Abaqus运行特征值屈曲作业。本示例中使用输入文件`fpert001.inp`生成结果文件`fpert001.fil`。当使用命令`abaqus fpert`执行`FPERT`程序时，第一个提示将是

```
Enter the name of the results file (w/o .fil):
```

输入`fpert001`以定义`FNAME`。第二个提示将是
```
Enter the mode shape(s) to be used in calculating the perturbed
mesh (zero when finished):
```

输入`1`后跟`0`，因为这是此示例结果文件中唯一可用的特征向量。在第三个提示处，
```
Enter the imperfection factor to be introduced into the geometry
for this eigenmode:
```

输入`0.25`。这设置 = 0.25，即此模型的壳厚度。程序然后处理数据，并将新网格的节点坐标写入`fpert002.015`。

### 分析描述

有关分析的完整讨论，请参阅["均匀轴向压力下圆柱壳的屈曲，" Abaqus Benchmarks Guide的第1.2.3节](../bmk/bmk-link.md#bmk-anl-bucklecylshell)。输入文件`fpert001.inp`（与`bucklecylshell_s9r5_n3.inp`相同）包含S9R5单元的2×20网格和屈曲分析的数据行。输入文件`fpert002.inp`包含使用扰动网格的Riks分析的数据行。`FPERT`程序的源代码在`fpert.f`中。
