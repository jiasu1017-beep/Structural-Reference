# 42.7 Creating new field output







The field and history output initially available in the output database are limited to those variables you have saved during the analysis. You can augment these analysis results by operating on field output to create new results. The following two methods can be used to create new field output:
- Select ****Tools**![](../graphics/images/arrow.gif)**Create Field Output**![](../graphics/images/arrow.gif)**From Fields**** from the main menu bar or use the ![](../graphics/ico_fieldOutputFromFields.png) tool in the toolbox to create new field output variables by [operating on individual existing field output variables](pt05ch42s05hlb01.md).
- Select ****Tools**![](../graphics/images/arrow.gif)**Create Field Output**![](../graphics/images/arrow.gif)**From Frames**** from the main menu bar or use the ![](../graphics/ico_fieldOutputFromFrames.png) tool in the toolbox to create new field output by [combining results from several output database frames](pt05ch42s05hlb03.md).
