# 4.1 单元公式与积分

悬臂梁静力分析示例（图4-1）将演示单元阶次（线性或二次）、单元公式和积分阶次对结构仿真精度的影响。这是一个经典的有限元测试案例。由于梁相对细长，我们通常会用梁单元来建模。然而，这里用它来帮助评估各种实体单元的性能。

梁长150 mm，宽2.5 mm，深5 mm；一端固定；自由端承受5 N的集中载荷。材料的杨氏模量*E*为70 GPa，泊松比为0.0。根据梁理论，载荷*P*作用下梁端部的静位移为：

![delta = Pl^3 / 3EI](../graphics/gsk_eqn00034.gif)

其中 ![I = bd^3/12](../graphics/gsk_eqn00035.gif)，*l*为长度，*b*为宽度，*d*为梁的深度。

当 ![P = 5 N](../graphics/gsk_eqn00036.gif) 时，端部位移为3.09 mm。

## 4.1.1 完全积分

"完全积分"是指在单元具有规则形状时，精确积分单元刚度矩阵中多项式项所需的高斯点数目。对于六面体和四边形单元，"规则形状"意味着边是直的且成直角相交，任何边上的节点都在边的中点上。完全积分的线性单元在每个方向上使用两个积分点。因此，三维单元C3D8在单元内使用2×2×2排列的积分点。完全积分的二次单元（在Abaqus/Standard中可用）在每个方向上使用三个积分点。二维四边形完全积分单元中积分点的位置如图4-2所示。

**图4-2** 完全积分二维四边形单元中的积分点。

![Integration points in fully integrated, two-dimensional, quadrilateral elements](../graphics/gss-linear-quad-nls.png)

在Abaqus/Standard对悬臂梁问题的仿真中使用了几种不同的有限元网格，如图4-3所示。这些仿真使用线性或二次完全积分单元，并说明了单元阶次（一阶与二阶）和网格密度对结果精度的影响。

**图4-3** 用于悬臂梁仿真的网格。

线性单元CPS4和C3D8对位移的预测偏差过大，结果无法使用。粗网格时精度最低，但即使使用细网格（8×24），预测的端部位移也仅为理论值的56%。注意，对于线性完全积分单元，梁厚度方向的单元数量没有影响。端部位移的低估是由*剪切锁定*引起的，这是所有完全积分一阶实体单元的问题。

如前所述，剪切锁定导致单元在弯曲时过于刚硬。解释如下。考虑结构受纯弯曲时的一小块材料。材料的变形如图4-4所示。

**图4-4** 受弯矩*M*作用的材料变形。

![Deformation of material subjected to bending moment M](../graphics/gss-deform.png)

最初平行于水平轴的线变成恒定曲率，厚度方向的线保持直线。水平和垂直线之间的夹角保持为90°。

线性单元的边无法弯曲；因此，如果用单个单元模拟小块材料，其变形形状如图4-5所示。

**图4-5** 受弯矩*M*作用的完全积分线性单元的变形。

![Deformation of a fully integrated, linear element subjected to bending moment M](../graphics/gss-linear-elem.png)

为了可视化，绘制了穿过积分点的虚线。可以看到，上面的线长度增加了，表明1方向的正应力![sigma_11](../graphics/gsk_eqn00032.gif)是拉应力。同样，下面的虚线长度减小了，表明![sigma_11](../graphics/gsk_eqn00032.gif)是压应力。垂直虚线的长度没有变化（假设位移很小）；因此，所有积分点上![sigma_22](../graphics/gsk_eqn00037.gif)均为零。所有这些都与纯弯曲小块材料的预期应力状态一致。然而，在每个积分点上，垂直线和水平线之间的夹角（最初为90°）发生了变化。这表明这些点上的剪应力![sigma_12](../graphics/gsk_eqn00033.gif)不为零。这是不正确的：纯弯曲材料的剪应力为零。

这种虚假剪应力的产生是因为单元的边无法弯曲。它的存在意味着应变能产生了剪切变形而不是预期的弯曲变形，因此总体位移较小：单元过于刚硬。

剪切锁定仅影响完全积分线性单元在弯曲载荷下的性能。这些单元在直接载荷或剪切载荷下表现良好。二次单元不会出现剪切锁定问题，因为它们的边可以弯曲（见图4-6）。表4-1中二次单元预测的端部位移接近理论值。然而，如果二次单元发生变形或弯曲应力有梯度，它们也会表现出一定程度的锁定，这在实际问题中都可能发生。

**图4-6** 受弯矩*M*作用的完全积分二次单元的变形。

![Deformation of a fully integrated, quadratic element subjected to bending moment M](../graphics/gss-quad-elem.png)

完全积分线性单元只应在确定载荷将在模型中产生最小弯曲时使用。如果不确定载荷会产生哪种类型的变形，请使用其他单元类型。完全积分二次单元在复杂应力状态下也可能锁定；因此，如果模型完全使用这类单元，应仔细检查结果。然而，它们对于模拟局部应力集中区域非常有用。

*体积锁定*是另一种形式的过度约束，发生在材料行为（几乎）不可压缩时使用完全积分单元的情况下。它导致本应不引起体积变化的变形出现过刚硬行为。这在第10章《材料》中进一步讨论。

## 4.1.2 减缩积分

只有四边形和六面体单元可以使用减缩积分格式；所有楔形、四面体和三角形实体单元都使用完全积分，尽管它们可以与减缩积分六面体或四边形单元在同一网格中使用。

减缩积分单元在每个方向上比完全积分单元少一个积分点。减缩积分线性单元只有一个位于单元形心的积分点。（实际上，Abaus中这些一阶单元使用更精确的"均匀应变"公式，其中计算单元应变分量的平均值。这个区别对本次讨论不重要。）减缩积分四边形单元的积分点位置如图4-7所示。

**图4-7** 减缩积分二维单元中的积分点。

线性减缩积分单元往往过于柔软，因为它们存在称为*沙漏模式*的数值问题。同样，考虑一个模拟小块材料受纯弯曲作用的单个减缩积分单元（见图4-8）。

**图4-8** 受弯矩*M*作用的线性减缩积分单元的变形。

![Deformation of a linear element with reduced integration subjected to bending moment M](../graphics/gss-reduced-integration.png)

两条虚线可视化线的长度都没有改变，它们之间的夹角也没有改变，这意味着单元单积分点上的所有应力分量都为零。因此，这种弯曲变形模式是一种零能模式，因为单元变形不产生应变能。单元无法抵抗这种类型的变形，因为它在这种模式下没有刚度。在粗网格中，这种零能模式可以在网格中传播，产生无意义的结果。

在Abaqus中，一阶减缩积分单元引入少量人工"沙漏刚度"来限制沙漏模式的传播。当模型中使用更多单元时，这种刚度在限制沙漏模式方面更有效，这意味着线性减缩积分单元只要使用足够细的网格就能给出可接受的结果。线性减缩积分单元较细网格中看到的误差（见表4-2）在许多应用中在可接受范围内。结果表明，使用这类单元模拟任何承受弯曲载荷的结构时，厚度方向至少应使用四个单元。当在梁厚度方向使用单个线性减缩积分单元时，所有积分点都位于中性轴上，模型无法抵抗弯曲载荷。（这些情况在表4-2中用*标记。）

线性减缩积分单元对变形非常容忍；因此，在任何变形水平可能很高的仿真中使用这些单元的细网格。

Abaqus/Standard中可用的二次减缩积分单元也有沙漏模式。然而，这些模式在正常网格中几乎不可能传播，如果网格足够细，很少成为问题。C3D20R单元的1×6网格除非在宽度方向使用两个单元，否则会因沙漏模式而无法收敛，但更细的网格即使宽度方向只使用一个单元也不会失败。二次减缩积分单元即使在复杂应力状态下也不会发生锁定。因此，这些单元通常是大多数一般应力/位移仿真的最佳选择，但不包括涉及大变形的位移仿真和某些类型的接触分析。

## 4.1.3 不兼容模式单元

不兼容模式单元主要用于Abaqus/Standard，试图克服完全积分一阶单元中剪切锁定的问题。由于剪切锁定是由单元位移场无法模拟与弯曲相关的运动学引起的，因此向一阶单元引入额外的自由度来增强单元的变形梯度。这些变形梯度增强使得一阶单元能够在其域上具有线性变化的变形梯度（如图4-9(a)所示）。标准单元公式导致单元上变形梯度恒定（如图4-9(b)所示），从而产生与剪切锁定相关的非零剪应力。

**图4-9** (a) 不兼容模式（增强变形梯度）单元和 (b) 使用标准公式的一阶单元中变形梯度的变化。

![Variation of deformation gradient in (a) an incompatible mode (enhanced deformation gradient) element and (b) a first-order element using a standard formulation](../graphics/gss-deform-gradient.png)

这些变形梯度增强完全在单元内部，不与沿单元边缘定位的节点相关联。与直接增强位移场的不兼容模式公式不同，Abaqus使用的公式不会导致两个单元之间边界上的材料重叠或孔洞（如图4-10所示）。

**图4-10** 使用增强位移场而不是增强变形梯度的不兼容模式单元之间潜在的运动学不兼容性。Abaqus对其不兼容模式单元使用后一种公式。

![Potential kinematic incompatibility between incompatible mode elements that use enhanced displacement fields rather than enhanced deformation gradients. Abaqus uses the latter formulation for its incompatible mode elements.](../graphics/gss-kinematic-nls.png)

此外，Abaqus中使用的公式可以轻松扩展到非线性有限应变仿真，而增强位移场单元则不容易做到这一点。

不兼容模式单元可以在弯曲问题中产生与二次单元相当的结果，但计算成本显著降低。然而，它们对单元变形敏感。图4-11显示了用故意变形的不兼容模式单元建模的悬臂梁：一种情况是"平行"变形，另一种是"梯形"变形。

**图4-11** 不兼容模式单元的变形网格。

![Distorted meshes of incompatible mode elements](../graphics/gss-distort-mesh-nls.png)

图4-12显示了悬臂梁模型的端部位移。端部位移按解析解归一化，并绘制成与单元变形程度的关系图。

**图4-12** 不兼容模式单元的平行变形和梯形变形的影响。

![Effect of parallel and trapezoidal distortion of incompatible mode elements](../graphics/gss-mode-elem-v-nls.png)

比较了Abaqus/Standard中的三种平面应力单元：完全积分线性单元、减缩积分二次单元和线性不兼容模式单元。完全积分线性单元在所有情况下都产生较差的结果，这是预期的。另一方面，减缩积分二次单元给出非常好的结果，直到单元严重变形时才会恶化。

当不兼容模式单元为矩形时，即使网格在悬臂梁厚度方向只有一个单元，也能给出接近理论值的结果。然而，即使很小的梯形变形也会使单元过于刚硬。平行变形也会降低单元精度，但程度较小。

不兼容模式单元有用，因为如果使用得当，它们可以以低成本提供高精度。然而，必须注意确保单元变形很小，这在网格化复杂几何形状时可能很困难；因此，在有这种几何形状的模型中，应再次考虑使用减缩积分二次单元，因为它们对网格变形的敏感度要低得多。然而，在严重变形的网格中，仅仅改变单元类型通常不会产生准确的结果。应尽可能最小化网格变形以提高结果的精度。

## 4.1.4 混合单元

混合单元公式可用于Abaqus/Standard中几乎所有类型的连续体单元，包括所有减缩积分和不兼容模式单元。混合单元在Abaqus/Explicit中不可用。使用此公式的单元名称中有字母"H"。

混合单元用于材料行为不可压缩（泊松比=0.5）或接近不可压缩（泊松比>0.475）的情况。橡胶是材料行为不可压缩的一个例子。不可压缩材料响应无法用常规单元建模（平面应力情况除外），因为单元中的压力应力是不确定的。考虑均匀静水压力下的单元（图4-13）。

**图4-13** 受静水压力作用的单元。

![Element under hydrostatic pressure](../graphics/gss-hydrostatic-nls.png)

如果材料不可压缩，它在这种载荷下体积不能改变。因此，压力应力无法从节点位移计算得出；因此，对于任何具有不可压缩材料行为的单元，纯位移公式都是不充分的。

混合单元包含一个额外的自由度，直接确定单元中的压力应力。节点位移仅用于计算偏斜（剪切）应变和应力。

第10章《材料》中给出了橡胶材料分析的更详细描述。
