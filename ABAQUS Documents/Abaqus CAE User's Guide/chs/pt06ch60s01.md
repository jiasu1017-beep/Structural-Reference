# 60.1 创建 CAD 连接





您可以使用 CAD Connection 工具集创建从 Abaqus/CAE 到运行关联接口插件的另一个 CAD 系统的连接。您可以使用该连接将模型从 CAD 系统导出到 Abaqus/CAE 的 Assembly 模块。您也可以使用该连接将 Abaqus/CAE 中导入模型的某些几何修改导出回 CAD 系统（请参见["更新导入模型中的几何参数，" 第60.2节](pt06ch60s02.md)）。以下 CAD 系统的关联接口插件可用：
- CATIA V6
- CATIA V5
- SolidWorks
- Pro/ENGINEER
- NX (Unigraphics)

[图 60--1](pt06ch60s01.md#usi-cad-catiacae) 显示了并发的 CATIA V5 和 Abaqus/CAE 会话，并说明了应用程序之间的通信。

**图 60–1** 使用 CATIA V5 关联接口的关联导入。

![](../graphics/tra-catia-cae-nls.png)

使用 CAD 连接将从运行关联接口插件的 CAD 系统传输模型到 Abaqus/CAE 称为关联导入。有关各个关联接口插件的更多信息，请参阅 Dassault Systèmes 知识库，网址为 [www.3ds.com/support/knowledge-base](http://www.3ds.com/support/knowledge-base)。此信息包括：
- 支持关联导入的 CAD 包以及支持的版本号。
- 关联接口插件的下载和安装说明。
- 每个关联接口插件的完整文档。

**创建 CAD 连接的步骤：**

1. 在 Assembly 模块中，从主菜单栏选择 ****Tools**![](../graphics/images/arrow.gif)**CAD Interfaces**![](../graphics/images/arrow.gif)***CAD System*****。***CAD System*** 是您要连接的 CAD 系统的名称；您可以与 CATIA V6、CATIA V5、SolidWorks、NX 和 Pro/ENGINEER 建立连接。

2. 从出现的对话框中，执行以下操作之一：
   **Auto-assign port** 选择 **Auto-assign port**，然后单击 **Enable** 从 Abaqus/CAE 打开一个端口。Abaqus/CAE 在消息区域显示它分配的端口号。当您从 CAD 系统导出模型时，数据通过此连接端口号传输。
   **Specify port** 选择 **Specify port**，然后输入端口号以指定端口。该数字必须在 1025 到 65535 之间。如果重新打开连接，您可以使用 Abaqus/CAE 首次分配端口时显示的端口号。
   **Disable port** 如果已启用连接，您可以从对话框中单击 **Disable** 来禁用连接。

3. 在使用 CAD Connection 工具集创建从 Abaqus/CAE 到运行关联接口插件的另一个 CAD 系统的连接后，您可以执行以下操作将装配导出到 Abaqus/CAE 会话：
   **CATIA V6**
   **CATIA V6R2013x 或更早版本** 从主菜单栏选择 ****Tools**![](../graphics/images/arrow.gif)**Associative Export**![](../graphics/images/arrow.gif)**Export to Abaqus/CAE****。
   **CATIA V6R2014 或更高版本** 选择 ****Share**![](../graphics/images/arrow.gif)**Export**![](../graphics/images/arrow.gif)**Associative Export**![](../graphics/images/arrow.gif)**Export to Abaqus/CAE****。
   更详细的说明可在 SIMULIA Associative Interface for Abaqus/CAE User's Guide 中找到；该指南在您安装 SIMULIA Associative Interface for Abaqus/CAE 应用程序后可用。
   **CATIA V5** 从 CATIA V5 主菜单栏选择 ****Abaqus**![](../graphics/images/arrow.gif)**Export to Abaqus/CAE****。在出现的**Export to Abaqus/CAE** 对话框中，选择 **Open Abaqus/CAE**，然后单击 **OK**。更详细的说明可在 CATIA V5 Associative Interface User's Guide 中找到；您可以从 Dassault Systèmes 知识库下载该指南，网址为 [www.3ds.com/support/knowledge-base](http://www.3ds.com/support/knowledge-base)。
   **SolidWorks** 从 SolidWorks 主菜单栏选择 ****Abaqus**![](../graphics/images/arrow.gif)**Export to Abaqus/CAE****。在出现的**Export to Abaqus/CAE** 对话框中，切换打开 **Open in Abaqus/CAE**，然后单击绿色复选标记。更详细的说明可在 SolidWorks Associative Interface User's Guide 中找到；您可以从 Dassault Systèmes 知识库下载该指南，网址为 [www.3ds.com/support/knowledge-base](http://www.3ds.com/support/knowledge-base)。
   **Pro/ENGINEER** 从 Pro/ENGINEER 主菜单栏选择 ****Abaqus**![](../graphics/images/arrow.gif)**Open in CAE****。更详细的说明可在 Pro/ENGINEER Associative Interface User's Guide 中找到；您可以从 Dassault Systèmes 知识库下载该指南，网址为 [www.3ds.com/support/knowledge-base](http://www.3ds.com/support/knowledge-base)。
   **NX** 从 NX 主菜单栏选择 ****Abaqus**![](../graphics/images/arrow.gif)**Open in /CAE****。更详细的说明可在 Abaqus/CAE Associative Interface for NX User's Guide 中找到；您可以从 Elysium, Inc. 网站下载该指南。
   有关关联导入的更多信息，请参见["我可以使用关联接口做什么？" 第10.1.2节](pt02ch10s01s02.md)。
