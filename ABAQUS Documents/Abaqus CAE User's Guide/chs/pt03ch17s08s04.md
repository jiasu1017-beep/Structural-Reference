# 17.8.4 三维结构化网格





图17-49展示了一些可以使用结构化网格技术进行网格划分的简单三维区域示例。

**图17-49** 可以使用结构化网格技术进行网格划分的区域。

![](../graphics/mgn-3dmappable.png)

使用此技术对更复杂的区域进行网格划分可能需要手动分割。如果不分割复杂区域，您唯一的网格选择可能是使用四面体单元的自由网格技术。使用结构化网格技术构建的网格由六面体单元组成，比四面体单元更受欢迎。

成功使用结构化网格技术对三维区域进行网格划分需要以下特征：
- 该区域不能有任何孔、孤立面、孤立边或孤立顶点。例如，图17-50所示的区域无法使用结构化网格技术进行网格划分。
**图17-50** 无法使用结构化网格技术进行网格划分的区域。
![](../graphics/mgn-3dmaplimit-nls.png)
您可以通过将孔的圆周分割成两半、四分之一等来消除孔（无论它们贯穿零件实例还是仅部分贯穿）。例如，图17-51中的四个分割将具有孔的单区域转换为四个没有孔的区域。
**图17-51** 分割可以使零件进行结构化网格划分。
![](../graphics/mgn-maphole-nls.png)
- 您应该将圆弧限制在90度或更小，以避免在侧面和边缘处产生凹面。例如，图17-52中的零件实例已被分割，因此具有180度圆弧的单区域变为两个具有90度圆弧的区域。
**图17-52** 将圆弧限制在90度或更小。
![](../graphics/mgn-mapconcave-nls.png)
- 该区域的所有面必须具有可以使用二维结构化网格技术进行网格划分的几何形状。例如，在图17-53中，如果不进行分割，两端的半圆只有两边。（一个面必须至少有三条边才能使用结构化网格技术进行网格划分。）如果您将零件分割成两半，每个半圆被分割成两个各有三边的面。
**图17-53** 分割创建两个有三边的面。
![](../graphics/mgn-mapsemicircle-nls.png)
- 区域的每个顶点必须恰好有三条边相交。例如，未分割金字塔顶部的顶点连接到四条边。但是，如果您将金字塔分割成两个四面体区域，则每个单独区域的顶点只连接到三条边。
**图17-54** 分割后，每个单独区域的顶点只连接到三条边。
![](../graphics/mgn-mapvertex-nls.png)
- 该区域必须由至少四个面（四面体区域）限定。如果区域由少于四个面限定，您可以根据需要分割区域以创建额外的面。
- 如果区域包含虚拟拓扑，则该区域必须由六个面限定。
- 如果区域无法使用结构化网格技术进行网格划分，您可以使用虚拟拓扑合并面，直到区域由六个面限定。图17-55展示了如何使用虚拟拓扑创建一个可以使用结构化网格技术进行网格划分的六面区域。
**图17-55** 虚拟拓扑可以使零件进行结构化网格划分。
![](../graphics/mgn-mapvirtual-nls.png)
- 侧面之间的角度应尽可能接近90度；您应该分割以消除大于150度的角度。
- 该区域的每条边必须符合以下定义之一：
  - 如果区域不是立方体，则边必须对应于单个面；也就是说，边不能包含多个面。
  - 如果区域是立方体，则边可以是在同一几何表面上的一组连接面。但是，每个面必须有四条边。此外，当对立方体进行网格划分时，面图案必须允许沿整个边创建规则网格图案的六面体单元行和列。例如，图17-56展示了两个可接受的面图案以及使用结构化网格技术对立方体进行网格划分时创建的规则单元网格图案。
**图17-56** 可接受的面图案及生成的网格。
![](../graphics/mgn-goodpatternandmesh.png)
图17-57中的边没有可接受的面图案：
**图17-57** 不可接受的面图案。
![](../graphics/mgn-badsides.png)
左侧所示的面图案对于结构化网格是不可接受的，因为每个面只有三条边。右侧图案中的每个面有四条边，但图案不允许在立方体的分割边上创建规则的单元网格，如图17-58所示。
**图17-58** 无法创建规则的单元网格。
![](../graphics/mgn-badpattern.png)
