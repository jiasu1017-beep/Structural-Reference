# 54.3.1 选择X-Y数据对象





### 54.3.1 选择X-Y数据对象

您可以选择一个或多个X-Y数据对象包含在X-Y报告中。

**选择X-Y报告数据的步骤:**

1. 找到X-Y报告"**XY Data**"选项。从主菜单栏,选择 ****Report**![](../graphics/images/arrow.gif)**XY****;然后在出现的对话框中点击"**XY Data**"选项卡。
2. 从可用数据对象列表中选择要包含在报告中的X-Y数据对象。默认情况下,列出了所有[先前保存的](pt05ch47s02hlb07.md)X-Y数据对象。选择以下选项之一来调整此列表的内容: - 点击"**All XY data**"浏览所有先前保存的X-Y数据对象。 - 点击"**XY plot in current viewport**"仅浏览当前视口中X-Y数据图中包含的那些X-Y数据对象。Abaqus/CAE列出您请求的X-Y数据对象。
3. 要限制列表以便更轻松地选择,在"**Name filter**"字段中输入过滤模式,然后按"**[Enter]**"键应用过滤器。点击文本字段右侧的 ![](../graphics/ico_tipSmall.png) 可显示有关如何构造过滤器的信息。Abaqus/CAE仅列出符合您规格的那些X-Y数据对象。
4. 从X-Y数据对象名称和描述列表中,选择一个或多个要包含在报告中的数据对象。有关在对话框中选择多个项目的更多信息,请参阅 ["从列表和表格中选择多个项目," 第3.2.11节](pt01ch03s02s11.md)。

要生成报告,请在"**Report XY Data**"对话框中的"**Setup**"页面上配置任何所需的自定义选项;然后点击"**Apply**"。完成后,点击"**Cancel**"关闭对话框。
