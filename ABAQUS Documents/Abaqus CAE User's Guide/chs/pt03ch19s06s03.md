# 19.6.3 管辖重启分析的规则





使用 Abaqus/CAE 定义模型和作业的重启信息很简单；但是，在使用分析重启功能之前，您应该了解以下几点：
- 重启分析中使用的模型必须与原始分析中使用的模型相同，直到重启位置。具体来说，- 对于新模型，请勿修改或添加任何几何形状、网格、材料、截面、梁截面轮廓、材料方向、梁截面方向、相互作用属性或约束。- 同样，请勿修改或添加任何在或之前于重启位置的步骤或规定条件（载荷、边界条件、场、相互作用或输出请求）。
- 如果您使用关键词编辑器编辑了原始模型，Abaqus/CAE 在重启分析时会忽略这些更改。

**警告：**Abaqus/CAE 不执行任何检查以确保为原始作业存储的重启数据与重启分析中所用的模型一致。在某些情况下，分析将终止并显示错误消息。在其他情况下，分析将运行，但结果可能与您的意图不符。一个会导致错误消息的特定示例是您在原始模型中定义耦合约束及其伴随的参考节点，然后在原始分析提交之前抑制该约束。在此示例中，Abaqus/CAE 保留参考节点定义，但 Abaqus/Standard 或 Abaqus/Explicit 可能将其从模型定义中移除（如果它未附加到其他元素或约束）。在此示例中，您后续的 Abaqus/CAE 重启分析将被视为与原始分析不一致。在类似此示例的许多情况下，您可以手动修改重启分析输入文件以纠正差异。
