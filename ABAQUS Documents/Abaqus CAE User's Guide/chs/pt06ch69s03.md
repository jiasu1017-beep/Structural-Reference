# 69.3 什么是缝合？







当您选择此选项时，Abaqus/CAE 尝试沿其自由边连接相邻面。缝合的目的通常是创建实体部件。缝合边通常会产生有效的几何。Abaqus/CAE 通过调整底层曲面直到边相交来缝合修剪表面之间的间隙。[图 69-8](pt06ch69s03.md#dbs-conc-trim-stitch) 说明了缝合过程。

**图 69-8** 缝合边直到它们相交。

![](../graphics/dbs-conc-trim-stitch-nls.png)

Abaqus/CAE 尝试从一组修剪表面识别实体的边。如果每个修剪表面的边接近相邻表面的边（在用户指定的公差范围内），Abaqus/CAE 会缝合修剪表面。缝合的最终结果是一个 B-rep 实体，其中边定义在两个相交的表面之间共享。有关 B-rep 实体的更多信息，请参阅["实体模型如何表示实体？" 第 10.1.7 节"](pt02ch10s01s07.md)。缝合在存在线体或内部面的情况下会失败。

当您导入 IGES 或 VDA-FS 格式的部件时，Abaqus/CAE 默认缝合部件。但是，由于 IGES 和 VDA-FS 格式部件的内部公差，小特征的 resulting 表示可能与原始文件中预期的几何不匹配。

在某些情况下，缝合 IGES 或 VDA-FS 格式部件会导致 Abaqus/CAE 导入应该作为独立部件导入的单个部件。如果您将导入的部件复制到新部件，您可以选择将断开连接的区域分离为独立的部件。 有关更多信息，请参阅["复制部件，" 第 11.18.3 节"](pt03ch11s18hlb03.md)。
