# 35.2.2 线性约束方程









**产品：** Abaqus/StandardAbaqus/Explicit

##### **参考资料**

### 概述

线性约束方程（Linear Constraint Equations）用于建立模型中不同自由度之间的线性关系。这种约束方法特别适用于以下情况：

- 连接不同类型的单元
- 施加周期性边界条件
- 定义对称或反对称条件
- 模拟刚性连接

### 约束方程形式

线性约束方程的一般形式为：

$$

\sum_j a_{ij} u_j = b_i

$$

其中 $a_{ij}$ 是系数，$u_j$ 是自由度，$b_i$ 是常数项。

### 约束方程定义

| **输入文件用法：** | ```
*CONSTRAINT
slave_node, dof, coefficient, master_node_1, coefficient_1, ...
constant
``` |
| --- | --- |

### 参数说明

| 参数 | 描述 |
| --- | --- |
| slave_node | 从节点编号 |
| dof | 自由度编号 |
| coefficient | 从节点系数 |
| master_node | 主节点编号 |
| coefficient | 主节点系数 |
| constant | 常数项 |

### 示例

定义节点 10 的自由度 2 受节点 1、2、3 控制：

```
*CONSTRAINT
10, 2, 1.0, 1, -0.5, 2, -0.3, 3, -0.2, 0.0
```

上述方程表示：

$$

u_{10,2} = 0.5 u_{1,2} + 0.3 u_{2,2} + 0.2 u_{3,2}

$$

### 刚性连接

使用约束方程模拟刚性连接：

| **输入文件用法：** | ```
*CONSTRAINT
slave_node, 1, 1.0, master_node, -1.0, 0.0
slave_node, 2, 1.0, master_node, -1.0, 0.0
slave_node, 3, 1.0, master_node, -1.0, 0.0
``` |
| --- | --- |

### 对称边界条件

使用约束方程施加对称条件：

| **输入文件用法：** | ```
*CONSTRAINT
node_set, dof, 1.0, symmetric_node, -1.0, 0.0
``` |
| --- | --- |

### 使用限制

- 约束方程不能形成闭环
- 从节点不能是另一个约束方程的主节点
- 约束方程与多point constraints 不能同时使用于同一自由度
