# 70.3.1 为什么要分区？







决定何时使用分区工具集取决于您将如何使用 resulting 区域，如下所述：

**分区部件**

在 Part（部件）或 Property（属性）模块中使用分区工具集将部件划分为区域。您可能在 Property（属性）模块中使用 resulting 区域来应用截面定义，其中包括横截面几何信息以及材料属性信息。在 Part（部件）模块和 Property（属性）模块中创建的分區与部件关联，并且在装配中部件的每个实例中都可用；但是，您无法修改或删除部件实例中的分区。

**分区装配**

在装配相关模块中使用分区工具集将装配中的部件实例划分为区域。您可能使用 resulting 区域执行以下操作：
- 将载荷、边界条件和预定义字段应用到区域。
- 将输出请求与区域关联。
- 通过添加可以播种的内部边来获得对网格划分过程的更精细控制。
- 将复杂的三维区域细分为自动网格生成器可以划分的更简单区域。

在装配中创建的分区仅与装配关联，而不与原始部件关联，并且在所有对装配进行操作的模块中都可用。它们不会出现在同一部件的其他实例中，并且在 Part（部件）或 Property（属性）模块中不可用。
