# 18.7.2 选择设计响应的数据源





默认情况下，Optimization Module 使用 Abaqus 模型的最后一个载荷工况（如果有）和最后一个步骤的数据来定义设计响应。或者，如果您的模型包含多个步骤或载荷条件，您可以选择将使用哪些步骤或载荷条件来定义设计响应。最后，如果您在会话中打开了多个模型，您可以选择将使用哪些模型以及每个模型中的哪些步骤或载荷条件来定义设计响应。

此外，如果设计响应正在计算固有频率，您可以选择要检查的单个模式或模式范围。

**选择设计响应的数据源的步骤：**

1. 从"创建设计响应"对话框中，选择**步骤**选项卡页面。
2. 执行以下任一操作：- 选择**使用最后一步和最后一种载荷工况**来使用最后一步或最后一种载荷工况的数据定义设计响应。- 选择**指定**以选择将用于定义设计响应的步骤或载荷条件。
3. 如果您选择了**指定**，请执行以下操作来选择将用于定义设计响应的模型、步骤或载荷工况（在一个步骤内）：- 选择 ![](../graphics/ico_add.png) 将当前模型中的一个步骤添加到步骤列表中。- 选择 ![](../graphics/icoR_addAll.png) 将当前模型中的所有有效步骤添加到步骤列表中。- 选择 ![](../graphics/icoR_addAllAll.png) 将会话中所有打开模型中的所有有效步骤添加到步骤列表中。（Abaqus/CAE 在当前模型的名称旁边显示星号。）- 选择 ![](../graphics/ico_delete.png) 从步骤列表中删除一个步骤。
4. 如果需要，单击表格中的一个单元格，并从出现的菜单中选择替代模型、步骤或载荷工况。
5. 如果设计响应将计算固有频率，您可以选择将从中计算固有频率的单个模式或模式范围。
6. 如果设计区域是壳，您可以选择 Optimization Module 将从中计算壳应力的壳截面中的位置。您可以选择以下选项：- 壳的顶层、中间层或底层的壳应力值。（中间层不承受弯曲，表现为薄膜。）- 壳的顶层、中间层或底层的壳应力的最大值。- 壳的顶层、中间层或底层的壳应力的最小值。
7. 选择 Optimization Module 将从所选区域提取设计响应的方式。您可以选择以下选项：- 所选区域的最大值。- 所选区域的最小值。- 所选区域的值之和。
8. 选择 Optimization Module 将从步骤和载荷工况中提取设计响应的方式。您可以选择以下选项：- 所有所选步骤和载荷工况的最大值。- 所有所选步骤和载荷工况的最小值。- 所有所选步骤和载荷工况的值之和。
9. 单击**确定**保存您的数据并退出编辑器。
