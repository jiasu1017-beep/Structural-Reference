# 20.2.4 aStudy.execute()





### 20.2.4 **_aStudy_.execute()**: 执行参数化研究设计的分析。

**产品:** Abaqus/Standard  Abaqus/Explicit

此命令用于执行参数化研究生成的设计的分析。

##### **参考**

- ["参数化研究脚本," 第20.1.1节](pt04ch20s01aus108.md)

### **命令:**

**_aStudy_.execute** (*token*, **execOptions**= , *additional data*)

### **标记:**

ALL

使用此标记按顺序执行参数化研究所有设计的分析。此选项是默认选项。

DISTRIBUTED

使用此标记使用本地和/或远程计算机的指定队列接口执行所有设计的分析。类似数量的分析将被分配到每个指定的队列。

INTERACTIVE

使用此标记以交互模式按顺序执行参数化研究所有设计的分析。在这种情况下，进程暂停以提示您进一步的执行指令。该提示允许您指定要执行的分析数量、执行剩余分析、指定要跳过的分析数量，或跳过所有剩余分析。

### **可选数据:**

**execOptions**

设置 **execOptions** 等于Abaqus执行选项的字符串（在执行参数化研究设计分析时添加到Abaqus执行命令中；请参阅["Abaqus/Standard、Abaqus/Explicit和Abaqus/CFD执行," 第3.2.2节](pt01ch03s02abx02.md)）；此字符串必须用匹配引号括起来。

##### **DISTRIBUTED的附加数据:**

### **必需数据:**

**queues**

设置 **queues** 等于队列接口名称或队列接口名称序列。如果给出单个名称，必须用匹配引号括起来。如果给出名称序列，必须将其放在括号或方括号内，并包含用匹配引号括起来并用逗号分隔的队列接口名称。

### **可选数据:**

**files**

设置 **files** 等于标识必须在远程执行后返回到本地计算机的文件或文件的符号常量或序列。序列项必须用逗号分隔，序列必须用括号或方括号括起来。

允许的符号常量有：`DAT`、`LOG`、`FIL`、`SEL`、`MSG`、`STA`、`ODB`、`IPM`、`RES`、`ABQ` 和 `PAC`。默认值为 **files** ` = (DAT, FIL, LOG, ODB, SEL)`。

##### **定义队列和队列接口:**

在使用分布式参数化研究之前，必须在Abaqus环境文件的 **design_startup** 部分内定义队列接口。例如，要为远程计算机 `server` 上的现有队列 `short` 定义队列接口，需要在环境文件中输入以下条目：

```
def onDesignStartup():
    from par_Session import Queue 
    import os    

    # convenience assignment  
    SCRATCH = '/scratch/' + os.environ['USER']   

    # create remote queue interface  
    Queue(name='short_interface', hostName='server',
          driver='abaqus', queueName='short', directory=SCRATCH)
```
 如果此外还需要本地队列，则必须扩展条目为：
```
def onDesignStartup():
    from par_Session import Queue 
    import os  

    # convenience assignment 
    SCRATCH = '/scratch/' + os.environ['USER']   

    # create remote queue interface 
    Queue(name='short_interface', hostName='server',
          driver='abaqus', queueName='short', directory=SCRATCH) 

    # create local queue interface
    Queue(name='local_interface', driver='abaqus', 
          queueName='local')

queue_name="local" 
local="echo "./%S 1>%L 2>&1" | batch"
```
