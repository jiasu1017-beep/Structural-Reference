### 概述

Abaqus中的材料定义：
- 指定材料的行为并提供所有相关属性数据；
- 可以包含多种材料行为；
- 被分配一个名称，用于引用由该材料制成的模型的那些部分；
- 可以具有温度和/或场变量依赖性；
- 在Abaqus/Standard中可以有解变量依赖性；以及
- 可以在局部坐标系中指定（["方向," 第2.2.5节](pt01ch02s02aus15.md)），如果材料不是各向同性的，则需要这样做。

### 材料定义

可以在分析中定义任意数量的材料。每个材料定义可以包含任意数量的材料行为，根据需要指定完整的材料行为。例如，在线性静态应力分析中可能只需要弹性材料行为，而在更复杂的分析中可能需要多种材料行为。

必须为每个材料定义分配一个名称。此名称允许从用于将材料分配给模型中区域的截面定义中引用该材料。

| **输入文件用法:** | ``` [*MATERIAL](../key/key-link.md#usb-kws-mmaterial), NAME=*name* ``` |
| --- | --- |
|  | 每个材料定义在一个数据块中指定，该数据块由 [*MATERIAL](../key/key-link.md#usb-kws-mmaterial) 选项启动。材料定义继续，直到引入不定义材料行为的选项（例如另一个 [*MATERIAL](../key/key-link.md#usb-kws-mmaterial) 选项），此时材料定义被视为完成。材料行为选项的顺序不重要。数据块内的所有材料行为选项被视为定义同一材料。 |

| **Abaqus/CAE用法:** | Property模块：material编辑器：**Name** 使用材料选项列表下的菜单栏向材料添加行为。 |

### 将材料数据指定为温度和独立场变量的函数

材料数据通常指定为独立变量（如温度）的函数。通过在几个不同的温度下指定材料属性，可以使材料属性具有温度依赖性。

在某些情况下，材料属性可以定义为Abaqus计算的变量的函数；例如，要定义加工硬化曲线，必须将应力给定为等效塑性应变的函数。

材料属性也可以依赖于"场变量"（用户定义的变量，可以表示任何独立量，并在节点上定义，是时间的函数）。例如，材料模量可以是复合材料中的织物密度或合金中的相分数的函数。请参阅 ["指定场变量依赖性](pt05ch21s01aus109.md#usb-mat-cmaterialdata-fvdepen)”了解更多详情。场变量的初始值作为初始条件给出（请参阅 ["Abaqus/Standard和Abaqus/Explicit中的初始条件," 第34.2.1节](pt07ch34s02aus116.md)），并且可以在分析期间作为时间的函数进行修改（请参阅 ["预定义场," 第34.6.1节](pt07ch34s06aus128.md)）。如果例如材料属性随时间变化由于辐照或某种其他预先计算的环境影响，此功能非常有用。

在Abaqus/Standard中，使用分布定义的任何材料行为（质量密度、线性弹性行为和/或热膨胀）不能与温度和/或场依赖性一起定义。但是，使用分布定义的材料行为可以与具有温度和/或场依赖性的其他材料行为一起包含在材料定义中。请参阅 ["密度," 第21.2.1节](pt05ch21s02abm01.md)；["线性弹性行为," 第22.2.1节](pt05ch22s02abm02.md)；和 ["热膨胀," 第26.1.2节](pt05ch26s01abm52.md)。

#### 材料数据的插值

在常数属性最简单的 case 中，只输入常数值。当材料数据仅是一个变量的函数时，必须按独立变量递增值的顺序给出数据。然后Abaqus线性插值介于给定值之间的值。假定属性在给定独立变量范围之外保持恒定（对于织物材料，在指定范围之外使用最后一个指定数据点的斜率进行线性外推）。因此，您可以根据材料模型需要给出尽可能多或尽可能少的输入值。如果材料数据强非线性依赖于独立变量，必须指定足够的数据点，以便线性插值能够准确捕获非线性行为。

当材料属性依赖于几个变量时，首先必须在其他变量的固定值下给出属性随第一个变量变化的 variation，然后按第二个变量、第三个变量等的升序列出。此过程确保材料属性在属性所依赖的任何独立变量值下被完整且唯一地定义。请参阅 ["输入语法规则," 第1.2.1节](pt01ch01s02aus01.md)，了解进一步的说明和示例。

### 在Abaqus/Explicit中定义材料点的特征单元长度

特征单元长度用于Abaqus对表现出应变软化的模型进行正则化，或传递给在材料点处调用的用户子例程。默认情况下，Abaqus使用基于几何平均值的定义计算特征单元长度。

一阶单元的默认值是穿过单元的线的典型长度，二阶单元的默认值是同一典型长度的一半。对于桁架，默认值是沿单元轴的特征长度。对于膜和壳，默认值是参考表面中的特征长度。对于轴对称单元，默认值只是 *r*–*z* 平面中的特征长度。

在Abaqus/Explicit中，您可以根据用户子例程 [`VUCHARLENGTH`](../sub/sub-link.md#sub-xsl-vucharlength) 中的单元拓扑和几何重新定义特征单元长度的值。

### 在Abaqus/Explicit和Abaqus/CFD中正则化用户定义的数据

在分析期间对独立变量的函数进行插值需要查找材料数据值的表。表查找在Abaqus/Explicit和Abaqus/CFD中频繁发生，如果独立变量的插值是规则的间隔，则最经济。例如，[图21.1.2-1](pt05ch21s01aus109.md#cmaterialdata-exa) 中显示的数据不是规则的，因为相邻数据点之间温度（独立变量）的间隔不同。您不需要指定规则的材料数据。Abaqus/Explicit和Abaqus/CFD将自动正则化用户定义的数据。例如，[图21.1.2-1](pt05ch21s01aus109.md#cmaterialdata-exa) 中的温度值可以定义为10、20、25、28、30和35 C。在这种情况下，Abaqus/Explicit和Abaqus/CFD可以通过在25个1 C增量上定义数据来正则化数据，您的前缀线性数据将被精确重现。此正则化需要将数据从6个温度点的值扩展到26个温度点的值。此示例是一个简单正则化可以精确重现数据的示例。

如果存在多个独立变量，规则数据的概念还要求在指定其他独立变量时，每个独立变量的最小值和最大值（范围）保持不变。[图21.1.2-2](pt05ch21s01aus109.md#cmaterialdata-2var-exa) 中的材料定义说明了材料数据不规则的情况，因为 ![](../graphics/usb_eqn02188.gif)、![](../graphics/usb_eqn02189.gif) 和 ![](../graphics/usb_eqn02190.gif)。Abaqus/Explicit还将正则化涉及多个独立变量的数据，尽管提供的数据必须满足 ["输入语法规则," 第1.2.1节](pt01ch01s02aus01.md) 中指定的规则。

#### 正则化用户定义数据时使用的误差容差

将输入数据正则化以便以分段线性方式精确重现数据并不总是可取的。假设屈服应力在Abaqus/Explicit中定义为塑性应变的函数，如下所示：

| 屈服应力 | 塑性应变 |
| --- | --- |
| 50000 | .0 |
| 75000 | .001 |
| 80000 | .003 |
| 85000 | .010 |
| 86000 | 1.0 |

可以精确正则化数据，但并不非常经济，因为它需要将数据细分为1000个规则间隔。如果定义的最小间隔与独立变量的范围相比较小，则正则化更加困难。

Abaqus/Explicit和Abaqus/CFD使用误差容差来正则化输入数据。每个独立变量范围内间隔的数量被选择为，使得分段线性规则化数据与每个定义点之间的误差小于容差乘以因变量的范围。在某些情况下，间隔数量变得过多，Abaqus/Explicit或Abaqus/CFD无法使用合理数量的间隔来正则化数据。考虑的合理间隔数量取决于您定义的间隔数量。如果您定义了50个或更少的间隔，Abaqus/Explicit和Abaqus/CFD用于正则化的最大间隔数等于用户定义间隔数的100倍。如果您定义了超过50个间隔，则用于正则化的最大间隔数等于5000加上50以上用户定义间隔数的10倍。如果间隔数量变得过多，程序将在数据检查阶段停止并发出错误消息。您可以重新定义材料数据或更改容差值。默认容差为0.03。

上述示例中的屈服应力数据是可能发出此类错误消息的典型情况。在这种情况下，您可以简单地删除最后一个数据点，因为它对最终屈服值的影响很小。

#### 在Abaqus/Explicit中应变率依赖数据的正则化
