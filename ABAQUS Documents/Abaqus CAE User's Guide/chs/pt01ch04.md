# 4 管理画布上的视口







画布可以被认为是无限屏幕或公告板，您可以在上面发布视口；您可以想象画布延伸到主窗口和显示器之外。画布的可见部分称为绘图区域，您可以通过增大主窗口的大小来增加其大小。

您可以在画布上的任何位置放置视口，并且可以将它们拖动到绘图区域之外。当视口放置在绘图区域之外时，您可以层叠或平铺视口以将它们带回视野。视口不是模型的一部分，不会话之间保存。
