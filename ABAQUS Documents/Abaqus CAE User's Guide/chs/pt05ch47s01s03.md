# 47.1.3 理解"Temp"和其他X–Y数据对象名称









### 47.1.3 理解"Temp"和其他X--Y数据对象名称

默认情况下，当您保存或绘制*X–Y*数据对象时，Abaqus/CAE会为您命名。此名称是识别和引用*X–Y*数据对象的重要方式。*X–Y*数据对象名称出现在以下情况中：
- 当您绘制*X--Y*数据时，*X--Y*图图例按名称识别图中出现的所有*X--Y*数据对象。
- 当您从主菜单栏选择****工具**![](../graphics/images/arrow.gif)**XY数据****时，**管理器**、**编辑**、**复制**、**重命名**、**删除**和**绘图**工具按名称列出会话期间创建的所有*X--Y*数据对象。
- 当您[自定义表示*X--Y*数据对象](pt05ch47s05.md)的曲线外观时，首先从**XY曲线选项**对话框中列出的*X--Y*数据对象名称中选择*X--Y*数据对象。

在大多数情况下，Abaqus/CAE能够为您的*X–Y*数据对象提供有意义的默认名称。当您保存*X–Y*数据对象时，Abaqus/CAE会向您显示默认名称，并给您机会用您选择的名称覆盖此默认值。保存后，您可以随时通过重命名来为*X–Y*数据对象提供新名称。有关重命名的信息，请参见["使用管理器对话框管理对象，" 第3.4.9节](pt01ch03s04hlb01.md)。

*X–Y*数据对象名称有三种基本形式，如下所示：

**Temp-*n***

如果通过从以下任何对话框中点击**绘图**来生成*X–Y*图：
- **来自ASCII文件的XY数据**
- **来自键盘的XY数据**
- **来自路径的XY数据**
- **对XY数据运算**对话框中的**绘图表达式**

Abaqus/CAE会根据模式**Temp-1**、**Temp-2**等为您的*X–Y*数据命名。此名称表示该图表示在对话框中配置的数据，无论您是否点击**另存为**保存它，这些数据都被视为临时数据。

相反，如果**Temp-***n*出现在您的图例中，则您正在绘制临时数据。您无法编辑、复制、重命名、删除、运算或生成临时数据的报告。要完成任何这些任务，必须首先保存数据。您可以为临时数据选择曲线样式偏好，但如果在不保存的情况下重新绘制数据，曲线样式将恢复为默认值。要了解如何保存数据，请参见["保存X--Y数据对象，" 第47.2.7节](pt05ch47s02hlb07.md)。

**XYData-*n***

如果使用**来自ASCII文件**或**来自键盘**对话框中提供的**另存为**按钮创建新的*X–Y*数据对象，Abaqus/CAE会根据模式**XYData-1**、**XYData-2**等为您的*X–Y*数据建议默认名称。要建立更有意义的名称，您可以使用**将XY数据另存为**对话框用您选择的名称覆盖此默认值。

要找到**来自ASCII文件的XY数据**或**来自键盘的XY数据**对话框，请从主菜单栏选择****工具**![](../graphics/images/arrow.gif)**XY数据**![](../graphics/images/arrow.gif)**创建****；然后在出现的对话框中点击感兴趣的选项，并点击**继续**。

**有意义的名称**

您可以通过在保存数据时指定名称或随后重命名来为*X–Y*数据对象建立有意义的名称（您选择的名称）。当您保存数据时，Abaqus/CAE在**将XY数据另存为**对话框中显示默认名称；您可以覆盖此默认值。Abaqus/CAE为从**历史输出**或**来自ODB场输出的XY数据**对话框创建的*X–Y*数据生成有意义的默认名称。
