# 15.17.4 配置连接器摩擦的切向行为

您可以使用连接器截面编辑器中的**切向行为**标签页为预定义和用户定义的连接器摩擦模型选择摩擦公式。有关更多信息，请参见["库仑摩擦模型的使用"中的"摩擦行为，"Abaqus分析用户指南第37.1.5节](../usb/usb-link.md#usb-cni-afriction-coulomb)。您可以选择以下选项：
- **惩罚**。指定摩擦系数。
- **静摩擦-动摩擦指数衰减**。使用由指数曲线定义的平滑过渡区指定单独的静摩擦和动摩擦系数。

**要使用**惩罚**摩擦公式配置切向行为：**

1. 按照["定义摩擦，"第15.17.3节"](pt03ch15s17hlb03.md)中概述的过程显示连接器截面编辑器，并选择**切向行为**标签页。
2. 从**摩擦公式**字段中选择**惩罚**。
3. 在**摩擦系数**标签页上，输入以下数据：
   - **摩擦系数** 摩擦系数，![](../graphics/usi_eqn00045.gif)。
   - **滑动率** 如果需要，打开**使用与滑动率相关的数据**并输入滑动率，![](../graphics/usi_eqn00627.gif)。
   - **接触压力** 如果需要，打开**使用与接触压力相关的数据**并输入接触压力，*p*。
   - **温度** 如果需要，打开**使用与温度相关的数据**并输入接触点处两个接触表面之间的平均温度，![](../graphics/usi_eqn00628.gif)。
   - **场1、场2等** 如果需要，指定**场变量数量**并输入第一个场变量的平均值，![](../graphics/usi_eqn00629.gif)，第二个场变量的平均值，![](../graphics/usi_eqn00630.gif)，等等。
4. 在**剪切应力**标签页上，指定剪切应力极限。
   - **无极限** 指示对等效剪切应力没有限制。
   - **指定** 输入等效剪切应力极限，![](../graphics/usi_eqn00577.gif)；即，等效剪切应力的最大可达值。
5. 在**弹性滑移**标签页上，选择修改允许弹性滑移的方法（仅适用于Abaqus/Standard分析）。
   - **特征模型尺寸的分数** 输入允许最大弹性滑移与特征模型尺寸的比率。默认值为0.0001。
   - **绝对距离** 输入允许弹性滑移的绝对大小，![](../graphics/usi_eqn00578.gif)，用于粘附摩擦的刚度方法。

**要使用**静摩擦-动摩擦指数衰减**摩擦公式配置切向行为：**

1. 按照["定义摩擦，"第15.17.3节"](pt03ch15s17hlb03.md)中概述的过程显示连接器截面编辑器，并选择**切向行为**标签页。
2. 从**摩擦公式**字段中选择**静摩擦-动摩擦指数衰减**。
3. 在**摩擦系数**标签页上，选择指数衰减定义并输入系数或测试数据。
   - **系数**。直接提供静摩擦系数、动摩擦系数和衰减系数。
     - **静摩擦系数** 输入静摩擦系数，![](../graphics/usi_eqn00580.gif)。
     - **动摩擦系数** 输入动摩擦系数，![](../graphics/usi_eqn00581.gif)。
     - **衰减系数** 输入衰减系数，![](../graphics/usi_eqn00582.gif)。
   - **测试数据**。提供测试数据点以拟合指数模型。
     - **摩擦系数** - 输入在![](../graphics/usi_eqn00631.gif)处指定的静摩擦系数。
     - 输入在参考滑动率下测量的动摩擦系数，![](../graphics/usi_eqn00585.gif)。
     - 输入动摩擦系数，![](../graphics/usi_eqn00586.gif)。此值对应于无限滑动率下摩擦系数的渐近值，![](../graphics/usi_eqn00587.gif)。
     - **滑动率** 输入用于测量动摩擦系数的参考滑动率，![](../graphics/usi_eqn00585.gif)。
4. 在**弹性滑移**标签页上，选择修改允许弹性滑移的方法（仅适用于Abaqus/Standard分析）。
   - **特征模型尺寸的分数** 输入允许最大弹性滑移与特征模型尺寸的比率。默认值为0.0001。
   - **绝对距离** 输入允许弹性滑移的绝对大小，![](../graphics/usi_eqn00578.gif)，用于粘附摩擦的刚度方法。
