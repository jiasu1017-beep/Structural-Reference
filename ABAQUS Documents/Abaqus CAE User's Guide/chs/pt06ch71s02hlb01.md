# 71.2.1 使用查询工具集查询模型







您可以使用查询工具集获取当前视口中模型几何体以及定义模型的特征的一般信息。此外，您还可以获取特定于您正在使用的模块的信息；例如，如果您正在使用Mesh模块，查询工具集可以提供有关网格、节点和单元的信息。

**查询模型的步骤：**

1. 从主菜单栏，选择****Tools**![](../graphics/images/arrow.gif)**Query****。**提示：**您也可以通过单击**Query**工具栏中的![](../graphics/ico_info.png)工具来查询模型。Abaqus/CAE显示**Query**对话框。
2. 从**Query**对话框中，选择以下之一：
   - **General Queries**
     - **Point/Node**
     - **Distance**
     - **Angle**
     - **Feature**
     - **Shell element normals**
     - **Beam element tangents**
     - **Mesh stack orientation**
     - **Mesh**
     - **Element**
     - **Mesh gaps/intersections**
     - **Mass properties**
     - **Geometry diagnostics**
   有关更多信息，请参见["获取模型的一般信息，"第71.2.2节](pt06ch71s02hlb02.md)。
   - 特定于模块的查询。有关每个模块中可用信息的完整列表，请参见["理解查询工具集的作用，"第71.1节](pt06ch71s01.md)。
3. 对于壳法线、梁切线、网格间隙和网格堆栈方向，Abaqus/CAE使用箭头和高亮显示在视口中显示请求的信息。对于所有其他查询，Abaqus/CAE在提示区域显示提示以指导您完成剩余步骤。Abaqus/CAE在消息区域显示请求的信息。要调整消息区域的大小，请拖动顶部边缘；要查看已滚出消息区域的信息，请使用右侧的滚动条。消息区域中显示的相同信息也会写入重放文件。
4. 完成后，在提示区域中单击取消按钮![](../graphics/afxI_cancel.png)。
