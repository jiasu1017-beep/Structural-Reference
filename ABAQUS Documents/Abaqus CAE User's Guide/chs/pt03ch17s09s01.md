# 17.9.1 什么是扫掠网格？





Abaqus/CAE使用扫掠网格对复杂的实体和表面区域进行网格划分。扫掠网格技术涉及两个阶段：
- Abaqus/CAE在区域的一侧创建网格，称为源侧。
- Abaqus/CAE一次复制一层该网格的节点，直到到达称为目标侧的最终侧。Abaqus/CAE沿边复制节点，该边称为扫掠路径。扫掠路径可以是任何类型的边——直边、圆边或样条。如果扫掠路径是直边或样条，则生成的网格称为挤出扫掠网格。如果扫掠路径是圆边，则生成的网格称为旋转扫掠网格。

例如，图17-65展示了挤出扫掠网格。为了对该模型进行网格划分，Abaqus/CAE首先在模型的源侧创建二维网格。然后，二维网格中的每个节点沿直边复制到每一层，直到到达目标侧。

**图17-65** 挤出实体的扫掠网格技术。

![](../graphics/mgn-expsweep-nls.png)

为确定区域是否可以进行扫掠网格划分，Abaqus/CAE测试区域是否可以沿扫掠路径从源侧扫掠到目标侧来复制。通常，Abaqus/CAE选择最复杂的边（例如，有孤立边或顶点的边）作为源侧。在某些情况下，您可以使用网格控制选择扫掠路径。如果模型的某些区域太复杂而无法进行扫掠网格划分，Abaqus/CAE会在对剩余区域生成扫掠网格之前询问您是否要从中移除这些区域。您可以使用自由网格技术对复杂区域进行网格划分，或者将区域分割成可以采用结构化或扫掠网格划分的简化几何体。

当您为区域分配网格控制时，Abaqus/CAE会指示扫掠路径的方向并允许您控制方向。如果区域可以沿多个方向进行扫掠，Abaqus/CAE可能会在其选择作为源侧的面生成非常不同的二维网格。因此，扫掠路径的方向会影响生成的三维扫掠网格的均匀性，如图17-66所示。

**图17-66** 扫掠方向可以影响扫掠网格的均匀性。

![](../graphics/mgn-conc-sweep-direction-nls.png)

此外，扫掠路径控制用于模拟垫片、连续壳、使用圆柱单元的圆柱区域以及使用粘性单元的粘接接头的六面体和楔形单元的默认方向。有关更多信息，请参见["为区域分配垫片单元，"第32.3节](pt04ch32s03.md)；["使用连续壳单元的零件网格划分，"第25.2节](pt04ch25s02.md)；["圆柱实体的扫掠网格，"第17.9.4节](pt03ch17s09s04.md)；以及["使用几何和网格工具创建具有粘性单元的模型，"第21.3节](pt04ch21s03.md)。
