# 42.6.3 了解复数结果

如果当前步骤是稳态动力学分析，则像应力(S)或位移(U)这样的输出变量的值可以是具有实部和虚部的复数。复数结果在分析期间保存为实部和虚部数据对。当您使用Abaqus/CAE显示复数结果时，可以选择以下任何形式来查看分析数据：
- **幅值**显示结果值的实部和虚部两者的组合幅值。
- **相角**显示正水平轴与表示复数的绘图点（*实部，虚部*）之间的角度。此选择仅对标量或矢量和张量分量有效。
- **实部**仅显示实部分量。此选项是默认设置。
- **虚部**仅显示复数结果的虚部分量的系数。
- **指定角度的值**以您指定的角度显示结果在实部和虚部的组合值。当角度为0时，显示结果的实部；当角度为90时，显示结果的虚部。

**幅值**在任何不变量之后计算，使得幅值是实部和虚部不变量分量平方和的平方根。类似地，其他复数形式也在不变量之后计算。**指定角度的值**是例外；它在不变量之前计算以保持正确的物理意义。

您还可以选择使用谐波动画来为复数结果设置动画。此技术通过显示**指定角度的值**来为复数场输出设置动画穿过一系列角度。Abaqus/CAE根据您的规格生成从0到180或从-180到180的范围内的角度，并显示每个角度处复数结果的值。
