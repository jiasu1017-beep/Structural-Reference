# *GASKET BEHAVIOR






### *GASKET BEHAVIOR开始垫片行为的规范。

此选项用于指示垫片行为定义的开始。

**产品：** Abaqus/Standard  Abaqus/CAE  

**类型：** 模型数据  

**级别：** 模型  

**Abaqus/CAE：** Property 模块

##### **参考文献：**

- ["使用垫片行为模型直接定义垫片行为，" Abaqus Analysis User's Guide 第 32.6.6 节](../usb/usb-link.md#usb-elm-egasketbehavior)

### **必需参数：**

NAME

将此参数设置为一个标签，用于在 [*GASKET SECTION](ch07abk11.md) 选项中引用该行为时进行引用。同一输入文件中的垫片行为名称必须唯一。

**此选项没有关联的数据行。**
