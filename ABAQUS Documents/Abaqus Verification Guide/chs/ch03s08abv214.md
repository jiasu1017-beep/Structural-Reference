### 测试的功能

子模型功能应用于二维、三维和轴对称连续体耦合温度-位移单元。在Abaqus/Standard和Abaqus/Explicit中，全局分析和子模型分析分别使用调用稳态耦合温度-位移和动态温度-位移程序的一般步骤。

### 问题描述

**模型：**

所有全局模型在x-y或r-z平面中的尺寸为7.0×7.0。每个子模型在x-y或r-z平面中的尺寸为5.0×5.0，占据相应全局模型的右下角。除轴对称模型外，平面外尺寸为1.0。在轴对称模型中，分析的结构是外半径为8.0的空心圆柱体。

**材料：**

在Abaqus/Standard中：

| 杨氏模量 | 30×10^6 |
| --- | --- |
| 泊松比 | 0.3 |
| 热膨胀系数 | 1×10^-5 |
| 热传导率 | 3.77×10^5 |
| 比热 | 0.39 |
| 密度 | 82.9 |

在Abaqus/Explicit中：

| 杨氏模量 | 110×10^9 |
| --- | --- |
| 泊松比 | 0.3 |
| 热膨胀系数 | 1×10^-3 |
| 热传导率 | 390 |
| 比热 | 384 |
| 密度 | 8900 |

**载荷：**

在所有Abaqus/Standard模型中，大小为0.3的分布通量施加在右面；在Abaqus/Explicit中，通量大小为0.5×10^4。

**边界和初始条件：**

在全局模型中，固定边界条件![](../graphics/ver_eqn00528.gif)=0和![](../graphics/ver_eqn00196.gif)=0分别规定在左面和底面。在三维模型中，附加约束![](../graphics/ver_eqn00529.gif)=0施加在前后面上的节点。初始温度处处为零，固定温度边界条件施加在左面。在子模型中，![](../graphics/ver_eqn00196.gif)=0规定在底面所有位置，而顶面和左面节点上的自由度1、2和11由全局解决方案驱动。质量缩放技术用于Abaqus/Explicit模型以加速分析。

### 结果与讨论

在全局分析中，Abaqus预测的温度场在非轴对称模型中在x方向线性变化，在轴对称模型中在r方向对数变化。预测的位移场在所有模型中都是非均匀的。下图显示了在Abaqus/Standard中为温度和x或r位移等值线图预测的结果。为了比较，子模型预测的温度和位移解决方案也显示在相同的等值线图中，全局和子模型结果之间获得了极好的一致性。因此，子模型分析中所有驱动变量的幅值在全局分析文件输出中被正确识别，并应用于子模型分析中的驱动节点。

[图3.8.11-1](ch03s08abv214.md#vertempdispsubmodel-temp-plane4)和[图3.8.11-2](ch03s08abv214.md#vertempdispsubmodel-ux-plane4)中显示了在Abaqus/Standard中4节点平面应力单元的全局和子模型分析结果。

[图3.8.11-3](ch03s08abv214.md#vertempdispsubmodel-temp-plane8)和[图3.8.11-4](ch03s08abv214.md#vertempdispsubmodel-ux-plane8)中显示了在Abaqus/Standard中8节点平面应变单元的全局和子模型分析结果。

[图3.8.11-5](ch03s08abv214.md#vertempdispsubmodel-temp-axisym)和[图3.8.11-6](ch03s08abv214.md#vertempdispsubmodel-ur-axisym)中显示了在Abaqus/Standard中8节点轴对称单元的全局和子模型分析结果。

[图3.8.11-7](ch03s08abv214.md#vertempdispsubmodel-temp-brick)和[图3.8.11-8](ch03s08abv214.md#vertempdispsubmodel-ux-brick)中显示了在Abaqus/Standard中20节点砖单元（前表面）的全局和子模型分析结果。

在Abaqus/Explicit中，子模型中被驱动的温度和位移从全局分析文件输出正确插值。二维、三维或轴对称子模型中的每一个都可以从具有相同维数的任何全局模型驱动。全局模型和子模型之间的结果非常一致。
