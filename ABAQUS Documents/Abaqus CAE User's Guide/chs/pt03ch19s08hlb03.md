# 19.8.3 选择作业类型







要找到作业编辑器，请从主菜单栏中选择****作业**![](../graphics/images/arrow.gif)**编辑**![](../graphics/images/arrow.gif)***作业名称*****。使用**提交**选项卡页面中的**作业类型**选项来选择 Abaqus 将运行的作业类型。您可以选择以下选项之一：

**完整分析**

选择此选项可提交完整模型或输入文件进行分析。

**恢复（Explicit）**

选择此选项可重新提交之前提前终止的 Abaqus/Explicit 分析。此选项不适用于 Abaqus/Standard 和 Abaqus/CFD 分析。

**重启**

选择此选项可提交使用同一模型先前分析保存的数据进行分析的模型。模型属性必须指定保存数据的位置。

可用作业类型的详细信息请参见["选择作业类型，"第 19.2.5 节](pt03ch19s02s05.md)。

**选择作业类型：**

1. 在作业编辑器中，点击**提交**选项卡以显示**提交**选项卡页面。
2. 从选项卡页面顶部的**作业类型**选项中，选择您选择的作业类型。
