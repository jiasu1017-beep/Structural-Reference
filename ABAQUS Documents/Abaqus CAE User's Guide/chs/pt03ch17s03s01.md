# 17.3.1 网格划分过程







要创建可接受的网格，请使用以下过程：

**分配网格属性并设置网格控制**

网格模块提供了多种工具，允许您指定不同的网格特征，例如网格密度、单元形状和单元类型。

**生成网格**

网格模块使用多种技术来生成网格。不同的网格技术为您提供不同程度的网格控制。

**细化网格**

网格模块提供了多种用于细化网格的工具：
- 种子工具允许您在所选区域中调整网格密度。
- 分区工具集允许您将复杂模型分割为更简单的子区域。
- 虚拟拓扑工具集允许您通过将小面与小边缘合并到相邻的面和边缘来简化模型。
- 编辑网格工具集允许您对网格进行微调。

**优化网格**

您可以为模型的区域分配重网格划分规则。重网格划分规则支持基于分析结果对网格进行连续细化。

**验证网格**

验证工具为您提供有关网格中所使用单元质量的信息。
