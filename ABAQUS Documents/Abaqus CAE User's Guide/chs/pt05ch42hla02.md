# 42.2 从当前模型数据库选择结果概述





Abaqus/CAE可以从当前模型数据库中的任何模型读取选定的数据。当您在可视化模块中选择一个模型时，Abaqus/CAE会使该模型中指定的载荷、预定义场、边界条件和相互作用的子集可用于选择作为场输出变量。选择后，您可以在分析的特定步骤中为选定项绘制云图或符号图。Abaqus/CAE为模型数据库中的每个步骤创建一个虚拟帧。

在模型中可以定义的载荷、预定义场、边界条件和相互作用中，只有部分子集可用于可视化；本节中的属性在其传播状态为**在本步骤创建**、**从先前步骤传播**或**在本步骤修改**时可以显示。以下属性可以在可视化模块的云图或符号图中显示：

**支持的载荷**

除非另有说明，以下载荷支持除用户定义分布外的所有分布的显示。Abaqus/CAE在您在可视化模块中显示载荷时不考虑与载荷定义关联的任何幅值数据。Abaqus/CAE在当前步骤中可用于显示的每个载荷名称前显示**(L)**。
- 集中力
- 力矩
- 集中电荷
- 集中热通量
- 表面电荷
- 集中浓度通量
- 表面热通量（除总通量外的所有分布）
- 表面浓度通量
- 压力载荷（除总力外的所有分布）

如果您使用用户指定的坐标系定义了载荷，则修改后的方向也会反映在可视化模块中。Abaqus/CAE还自定义使用表达式场作为其自定义分布定义的任何载荷的显示；不支持具有映射场的载荷的显示。

**预定义场**

Abaqus/CAE在当前步骤中可用于显示的每个预定义场名称前显示**(P)**。
- 温度（除**从结果或输出数据库文件**外的所有分布，且仅支持**截面恒定**变化）
- 速度（仅支持平动速度分量）

如果您使用表达式场或映射场定义了预定义场，则自定义分布也会反映在可视化模块中。

**边界条件**

Abaqus/CAE在当前步骤中可用于显示的每个边界条件名称前显示**(B)**。对于包含自由度（如位移/旋转）的边界条件，您可以在选择边界条件后选择要显示的单个自由度。下方括号中注明了可用于显示的各个分量。

您可以显示任何支持分布的边界条件参数的数据：
- 位移/旋转（平动和转动位移）
- 速度/角速度（平动和角速度）
- 加速度/角加速度（平动和角加速度）
- 流体入口/出口
- 流体壁面
- 温度
- 孔隙压力
- 电势
- 质量浓度
- 声压
- 连接器材料流

**相互作用**

Abaqus/CAE在当前步骤中可用于显示的每个相互作用名称前显示**(I)**。表面膜条件（膜系数和汇温度）是唯一支持的相互作用。
