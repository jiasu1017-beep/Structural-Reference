# 52.1.1 定义应力线





您可以通过在模型中选择节点、空间中的点或保存的路径来定义要获取线性化应力的截面的端点。您可以通过直接从视口选择节点或输入节点标签来选择在模型中的节点,您可以通过输入点坐标来定义空间中的点。如果您选择了保存的路径,Abaqus/CAE使用路径端点作为应力线的端点——路径上的其他点将被忽略。(有关路径的更多信息,请参阅[第48章,"沿路径查看结果](pt05ch48.md)。")

节点标签是引用给定部件实例中某个节点的持久方式;也就是说,节点标签不会随着模型变形而改变。因此,如果您使用节点标签或保存的节点列表路径来指定截面的端点,这些点将同样适用于未变形或变形的模型形状。但是,空间中的点是固定的,与您的模型无关。例如,与未变形形状上节点位置重合的坐标可能不会与变形形状上的任何位置重合。如果您使用空间坐标、点列表路径或径向路径来指定截面的端点,请确保相对于正确的形状指定坐标。边列表路径不能用于定义应力线。

对于轴对称模型获得最佳结果,所选端点应垂直于模型的内表面和外表面以及中平面(请参阅[图52-1](pt05ch52s01s01.md#usv-stresslin-paths))。

**图52-1** 推荐的应力路径。

![](../graphics/stm-anl-stresslin-paths-nls.png)

截面可以从表面到表面,也可以完全在模型内部。截面也可以穿过多个部件实例。在选择截面的端点时还应考虑以下准则:- 避免穿过空间不连续的路径。
- 避免沿分隔不连续的线。
- 在指定截面之前使用显示组来隔离各个区域。

如果截面跨越空间不连续处,例如部件实例中的孔或部件实例之间的空间,则会给出错误消息。

选择截面的端点后,您可以指定要划分线条的段数,或者接受默认值。Abaqus/CAE通过在两点之间进行插值来定义应力线,以便沿着直线上获得指定数量的等间隔。
