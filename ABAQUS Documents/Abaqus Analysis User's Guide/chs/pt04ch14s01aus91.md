### 概述

在欧拉分析中，您可以在欧拉边界定义独立的流入和流出条件。欧拉边界条件：
- 可用于控制流入欧拉域的材料流动；
- 可用于定义欧拉域边界处的压力场；
- 可用于在截断人工边界处施加非反射边界条件以模拟无限域；以及
- 与在发生流入或流出的欧拉网格边界处定义的表面相关联。

### 定义欧拉边界

欧拉边界必须在欧拉网格边界处的表面上定义。您不能在同一表面定义多个欧拉边界。

| **输入文件用法：** | ``` [*EULERIAN BOUNDARY](../key/key-link.md#usb-kws-heulerianboundary) *surface name* ``` |
| --- | --- |

| **Abaqus/CAE 用法：** | 载荷模块：**Create Boundary Condition**: **Category:** **Other**, **Types for Selected Step:** **Eulerian boundary**: select region |
| --- | --- |

### 定义流入条件

您可以使用流入条件来控制流入欧拉域的材料流动。

#### 自由流入

如果未定义欧拉边界，材料可以自由流入欧拉域；流入材料的内容和每种流入材料的状态等于该单元内当前存在的状态。如果定义了欧拉边界，自由流入是默认流入条件。

| **输入文件用法：** | ``` [*EULERIAN BOUNDARY](../key/key-link.md#usb-kws-heulerianboundary), INFLOW=FREE ``` |
| --- | --- |

| **Abaqus/CAE 用法：** | 欧拉边界条件编辑器：**Flow type:** **Inflow**, **Inflow:** **Free** |
| --- | --- |

#### 无流入

您可以指定不能发生流入的欧拉边界——没有材料或空可以通过指定边界流入欧拉域。如果velocity在边界处指向内部，则速度的法向分量设置为零，而速度的切向分量保持不变。

| **输入文件用法：** | ``` [*EULERIAN BOUNDARY](../key/key-link.md#usb-kws-heulerianboundary), INFLOW=NONE ``` |
| --- | --- |

| **Abaqus/CAE 用法：** | 欧拉边界条件编辑器：**Flow type:** **Inflow**, **Inflow:** **None** |
| --- | --- |

#### 空流入

您还可以指定可以发生流入但流入体积仅包含空的边界。由于空流入，最初完全充满材料的欧拉域可能在分析过程中变得部分充满。

| **输入文件用法：** | ``` [*EULERIAN BOUNDARY](../key/key-link.md#usb-kws-heulerianboundary), INFLOW=VOID ``` |
| --- | --- |

| **Abaqus/CAE 用法：** | 欧拉边界条件编辑器：**Flow type:** **Inflow**, **Inflow:** **Void** |
| --- | --- |

### 定义流出条件

流出条件可用于通过减少流出边界的反射来模拟无限域，或用于在边界处指定压力场。

#### 自由流出

如果未指定欧拉边界条件，材料可以自由流出欧拉域；流出材料的内容和每种流出材料的状态等于该单元内当前存在的状态。如果定义了欧拉边界条件，且在同一表面指定了空流入条件，则自由流出是默认行为。

| **输入文件用法：** | ``` [*EULERIAN BOUNDARY](../key/key-link.md#usb-kws-heulerianboundary), OUTFLOW=FREE ``` |
| --- | --- |

| **Abaqus/CAE 用法：** | 欧拉边界条件编辑器：**Flow type:** **Outflow**, **Outflow:** **Free** |
| --- | --- |

#### 非反射流出

非反射流出条件可用于在无限域中定义的边界值问题，或与周围介质相比感兴趣区域很小的问题。像 ["在动态分析中使用实体介质无限单元" 中描述的无限单元公式，"无限单元," 第 28.3.1 节"](pt06ch28s03alm03.md#usb-elm-einfinite-solid) 中所述，非反射流出条件在域边界上引入额外的法向和剪切牵引力，它们与边界速度的法向和剪切分量成正比。这些边界阻尼常数被选择以最小化膨胀波和剪切波能量反射回有限元网格。这种条件不能提供完美的能量透射出网格，除非是平面体波在各向同性介质中正交入射到边界的情况。然而，它通常为大多数实际情况提供可接受的建模。一个例外是当 significant 材料传输通过边界发生时，这种条件不适合使用。

| **输入文件用法：** | ``` [*EULERIAN BOUNDARY](../key/key-link.md#usb-kws-heulerianboundary), OUTFLOW=NONREFLECTING ``` |
| --- | --- |

| **Abaqus/CAE 用法：** | 欧拉边界条件编辑器：**Flow type:** **Outflow**, **Outflow:** **Nonreflecting** |
| --- | --- |

#### 平衡流出

平衡流出是另一种流出条件，可以有效减少无限域中人工流出边界的虚假反射。假定应力在边界上元素面之间是零阶连续的。牵引力施加在这些元素面上以平衡由边界元素中应力创建的节点力。这种条件通常应用于压力分布未知的流出边界。

| **输入文件用法：** | ``` [*EULERIAN BOUNDARY](../key/key-link.md#usb-kws-heulerianboundary), OUTFLOW=NONUNIFORM PRESSURE ``` |
| --- | --- |

| **Abaqus/CAE 用法：** | 欧拉边界条件编辑器：**Flow type:** **Outflow**, **Outflow:** **Equilibrium** |
| --- | --- |

#### 零压力流出

在流动问题中，通常在流动出口指定零压力。由于边界上的法向牵引力包含压力和剪切应力的贡献，如果也考虑流动的剪切行为，自然边界条件（也称为"不做什么条件"）不足以提供这种条件。零压力流出条件施加抵消剪切贡献的牵引力，从而在边界上产生均匀分布的压力场。您可以在同一边界上施加分布式表面载荷（参见 ["分布式载荷," 第 34.4.3 节"](pt07ch34s04aus122.md#usb-prc-ploaddistributed-surfload) 中的"表面牵引力和压力载荷"）以指定非零压力。如果未指定流入条件，这是默认流出条件。

| **输入文件用法：** | ``` [*EULERIAN BOUNDARY](../key/key-link.md#usb-kws-heulerianboundary), OUTFLOW=ZERO PRESSURE ``` |
| --- | --- |

| **Abaqus/CAE 用法：** | 欧拉边界条件编辑器：**Flow type:** **Outflow**, **Outflow:** **Zero pressure** |
| --- | --- |

### 在重启分析中使用欧拉边界

您可以在重启分析中定义新的欧拉边界，但您不能在该边界指定空流入条件。此外，您不能在重启分析中将现有欧拉边界上的流入条件更改为空流入条件。
