# 4.2.5 LE5: Z-section cantilever







**Products: **Abaqus/Standard  Abaqus/Explicit  

### Elements tested

S3    S3R    S4R    S4R5    S4RS    S4RSW    S8R    S8R5    S9R5    STRI3    STRI65    

B31OS    B32OS    

### Problem description

![](../graphics/exxle5-model-nls.png)

**Model: **

Z-section cantilever under torsional loading.

**Material: **

Linear elastic, Young's modulus = 210 GPa, Poisson's ratio = 0.3, density = 7800 kg/m3.

**Boundary conditions: **

All displacements are zero along the edge at ![](../graphics/bmk_eqn00621.gif) 0.

**Loading: **

Torque of 1.2 MN-m applied at ![](../graphics/bmk_eqn00621.gif) 10. The torque is applied by two uniformly distributed edge shears of 0.6 MN at each flange when shell elements are used. In the explicit dynamic analysis the loading rate is applied such that a quasi-static solution is obtained.

### Reference solution

This is a test recommended by the National Agency for Finite Element Methods and Standards (U.K.): Test LE5 from NAFEMS publication TNSB, Rev. 3, “The Standard NAFEMS Benchmarks,” October 1990.

Target solution: Axial stress, ![](../graphics/bmk_eqn01702.gif) 108 MPa at midsurface, point A.
