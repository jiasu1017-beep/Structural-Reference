# *NO COMPRESSION







### *NO COMPRESSIONIntroduce a compressive failure theory (tension only materials).

This option is used to modify the elasticity definition so that no compressive stress is allowed. It can be used only in conjunction with the [*ELASTIC](ch05abk03.md) option.

**Products: **Abaqus/Standard  Abaqus/CAE  

**Type: **Model data  

**Level: **Model  

**Abaqus/CAE: **Property module
