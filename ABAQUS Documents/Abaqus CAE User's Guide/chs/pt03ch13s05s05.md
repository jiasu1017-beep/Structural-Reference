# 13.5.5 替换零件实例







您可以用第二个零件的实例替换零件实例。更准确地说，您是在替换创建零件实例的零件。Abaqus/CAE定位新零件实例，使其原点位于原始零件实例的原点，并且它们的轴对齐。此外，您可以选择新零件实例是否继承它所替换的实例的所有约束。

替换操作不改变实例的属性。例如，如果原始实例是从属的，替换它的实例也将是从属的。因此，如果零件存在独立实例，则不能使用替换过程创建同一零件的从属实例。

当您用具有相似几何的零件实例替换时，替换零件实例很有用。例如，新零件实例可能具有原始零件实例所没有的额外细节。您也可以用同一零件的网格表示替换几何零件。例如，您可以用从输出数据库导入的变形零件的网格表示替换零件。
