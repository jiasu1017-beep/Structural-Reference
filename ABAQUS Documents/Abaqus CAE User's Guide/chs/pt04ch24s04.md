# 24.4 什么是 CORM？






### 24.4 什么是 **CORM**？

CORM 是相对运动分量的缩写：连接器局部的相对位移和旋转。可用的相对运动分量是非运动学约束的相对位移和旋转。根据连接类型，某些分量可能是可用的，而某些可能是被约束的。当您创建或修改连接器截面时，Abaqus/CAE 会在指定连接类型的编辑器中显示可用的和被约束的相对运动分量。除了对相对运动分量施加行为外，您还可以对连接器的可用相对运动分量施加连接器载荷和连接器边界条件（请参见["施加连接器载荷和连接器边界条件，" 第 24.10 节"](pt04ch24s10.md)）。有关创建连接器截面的更多信息，请参见["连接器截面编辑器，" 第 15.9.7 节"](pt03ch15s09s07.md)。
