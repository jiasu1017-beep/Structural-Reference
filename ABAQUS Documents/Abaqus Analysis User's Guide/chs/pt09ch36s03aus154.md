### 概述

Abaqus/Standard 中的接触控制：
- 对于大多数问题，不应从默认设置进行修改；
- 可用于标准接触控制未提供经济有效的解决方案的问题；
- 可用于标准控制未有效建立所需接触条件的问题；以及
- 可在某些情况下用于控制是否创建补充接触约束。

从 Abaqus/Standard 中的接触控制调整中受益的问题通常是具有复杂几何形状和众多接触界面的大型模型。

### 应用接触控制

您可以按步骤将接触控制应用于步骤中活跃的所有接触对和接触单元，或应用于各个接触对。这使得可以将接触控制应用于特定接触对以使模拟通过困难阶段。接触控制保持有效，直到被更改或重置为其默认值。如果在任何给定步骤中为整个模型和特定接触对都声明了接触控制，则特定接触对的控制将覆盖整个模型对该接触对的控制。

此外，您可以按以下所述为各个接触对指定补充接触约束（见 ["补充接触约束](pt09ch36s03aus150.md#usb-cni-acontacttrouble-supplementary-constraints)"）。

### 接触问题中刚体运动的自动稳定

Abaqus/Standard 提供接触稳定化，以帮助在接触闭合和摩擦约束这种运动之前自动控制静力学问题中的刚体运动。

建议您首先尝试通过建模技术（修改几何、施加边界条件等）来稳定刚体运动。自动稳定化功能旨在用于明确将建立接触但建模过程中多个体的精确定位困难的情况。它不旨在模拟一般刚体动力学；也不适用于接触抖动情况或解决配合表面之间初始紧密间隙的问题。

使用自动接触稳定化时，Abaqus/Standard 为所有从节点处的接触对的相对运动激活粘性阻尼，方式与接触阻尼相同（见 ["接触阻尼，" 第 37.1.3 节](pt09ch37s01aus167.md)）。与大多数接触控制不同（这些控制在后续步骤中持续有效直到被修改或重置），自动稳定化阻尼仅在指定该阻尼的步骤期间适用。在后续步骤中，稳定化被移除，即使未建立接触或接触对完全分离后出现刚体运动。如果需要，您还应为后续步骤指定稳定化。

默认情况下，阻尼系数：
- 根据底层单元的刚度和步骤时间自动为每个接触约束计算，
- 在法向和切向方向上平等地应用于所有接触对，
- 在步骤中线性斜降，
- 仅当接触表面之间的距离小于特征表面尺寸时激活，以及
- 对于使用接触单元（如间隙接触单元、管-管接触单元等）建模的接触为零。

尽管自动计算的阻尼系数通常提供足够的阻尼来消除刚体模式而不会对解决方案产生重大影响，但无法保证该值是最优的甚至合适的。对于薄壳模型尤其如此，其中阻尼可能太高。因此，如果收敛行为有问题，您可能需要增加阻尼；如果阻尼扭曲了解决方案，则可能需要减少阻尼。第一种情况很明显，但后者需要事后检查。有几种方法可以执行此类检查。最简单的方法是考虑粘性阻尼耗散的能量与模型更一般的能量度量（如弹性应变能）之间的比率。这些量可以作为输出变量 ALLSD 和 ALLSE 分别获得。更详细的信息可以通过将接触阻尼应力 CDSTRESS（具有各个分量 CDPRESS、CDSHEAR1 和 CDSHEAR2）与真实接触应力 CSTRESS（具有各个分量 CPRESS、CSHEAR1 和 CSHEAR2）进行比较来获得。如果接触阻尼应力太高，您应该减少阻尼。应在接触牢固建立后进行比较；当接触尚未或仅部分建立时，接触阻尼应力总是相对较高。

增加或减少阻尼量的最简单方法是指定一个乘数，自动计算的阻尼系数将乘以该乘数。通常，您应该首先考虑更改默认阻尼（至少）一个数量级；如果这足以解决问题，您可以执行一些后续微调。在某些情况下，可能需要更大或更小的因子；只要获得收敛解决方案且耗散的能量和接触阻尼应力足够小，这就不是问题。

也可以直接指定阻尼系数。直接指定阻尼值并不容易，可能需要一些试错。为效率起见，这最好在类似规模的缩减模型上完成。如果直接指定阻尼系数，则忽略为默认阻尼系数指定的任何乘数因子。

#### 在增量内更改稳定化

为了减少或消除接触稳定化显著影响报告解决方案的可能性，可以引入在增量迭代中变化的比例因子。在增量的早期迭代中让更多稳定化生效可能有助于在建立一些接触之前避免数值问题。在增量的后期迭代中让较少或没有稳定化生效可能有助于提高增量的最终收敛迭代的准确性。

您可以指定这些比例因子。例如，指定"1,0"会导致比例因子在初始迭代期间（直到各种收敛度量满足或几乎满足）为统一，然后比例因子被重置为零（有效关闭稳定化）直到收敛检查再次满足。

| **输入文件用法：** | 要在增量内更改阻尼系数： |
| --- | --- |
|  | ``` [*CONTACT CONTROLS](../key/key-link.md#usb-kws-hcontactcontrols), STABILIZE=USER ADAPTIVE ``` |

#### 指定稳定化斜降因子

您可以指定步骤结束时的斜降因子。默认值等于零，因此阻尼在步骤结束时完全消失。在某些情况下输入此因子的非零值可能很有用，例如在步骤结束时刚体模式未完全约束；例如，如果问题无摩擦且可能发生滑动但滑动方向上没有净力。在这种情况下，通常希望在下一步中保持小阻尼，方法是将用于斜降的值作为阻尼系数的乘数。如果需要，您可以通过将斜降因子设置为一来维持此阻尼水平。

| **输入文件用法：** | ``` [*CONTACT CONTROLS](../key/key-link.md#usb-kws-hcontactcontrols), STABILIZE , *ramp-down factor* ``` |
| --- | --- |

| **Abaqus/CAE 用法：** | Interaction 模块：Abaqus/Standard 接触控制编辑器： **Stabilization**: **Automatic stabilization** 或 **Stabilization coefficient**, **Fraction of damping at end of step:** *ramp-down factor* |
| --- | --- |

#### 指定阻尼范围

默认情况下，应用阻尼的开口距离（阻尼范围）等于特征从表面面片尺寸；如果此尺寸不可用（例如，对于基于节点的表面），则使用为整个模型获得的特征单元长度。阻尼在开口小于阻尼范围一半时为参考值的 100%，从那里斜降到开口等于阻尼范围时为零。或者，您可以覆盖计算的阻尼范围直接指定阻尼范围。如果阻尼应该仅适用于窄间隙，或者如果阻尼应该无论开口距离如何都有效，则这可能很有用。在后一种情况下，应输入一个大的值。

| **输入文件用法：** | ``` [*CONTACT CONTROLS](../key/key-link.md#usb-kws-hcontactcontrols), STABILIZE , , *damping range* ``` |
| --- | --- |

| **Abaqus/CAE 用法：** | Interaction 模块：Abaqus/Standard 接触控制编辑器： **Stabilization**: **Automatic stabilization** 或 **Stabilization coefficient**, **Clearance at which damping becomes zero: Specify:** *damping range* |
| --- | --- |

#### 指定切向阻尼

默认情况下，切向方向的阻尼与法向方向的阻尼相同。但是，如果需要更低或更高的值，您可以减少或增加切向阻尼或将其设置为零。

| **输入文件用法：** | ``` [*CONTACT CONTROLS](../key/key-link.md#usb-kws-hcontactcontrols), STABILIZE, TANGENT FRACTION=*value* ``` |
| --- | --- |

| **Abaqus/CAE 用法：** | Interaction 模块：Abaqus/Standard 接触控制编辑器： **Stabilization**: **Automatic stabilization** 或 **Stabilization coefficient**, **Tangent fraction:** *value* |

### 与法向接触约束关联的接触控制

这些控制允许您指定接触界面上的节点可以违反"硬"接触条件。此外，这些控制可用于修改"软化"压力-过盈量关系以及增强拉格朗日或惩罚接触约束强制方法。"不分离"压力-过盈量关系不能被接触控制修改。

节点可以以两种方式之一违反接触条件。首先，Abaqus/Standard 可能认为该节点没有接触，即使该节点已穿透主表面一小段距离。其次，Abaqus/Standard 可能认为节点处于接触状态，即使在该节点处传输的接触表面之间的法向压力为负（即，正在传输拉应力）。

#### 修改增强拉格朗日或惩罚接触约束强制的行为

对于增强拉格朗日接触，您可以指定允许穿透（直接指定或作为特征接触表面尺寸的分数），该穿透允许违反不可穿透条件。此外，对于增强拉格朗日或惩罚接触，您可以缩放 Abaqus/Standard 计算的默认惩罚刚度。增强拉格朗日和惩罚约束强制方法的控制在 ["Abaqus/Standard 中的接触约束强制方法，" 第 38.1.2 节](pt09ch38s01aus178.md) 中讨论。

### 在线 性扰动步骤中修改切向惩罚刚度

在线性扰动步骤中用于强制执行切向约束的惩罚刚度通常与在一般步骤中用于强制执行粘附的惩罚刚度不同。在扰动步骤中，当相应的法向约束在基础状态下活跃且接触属性（表面相互作用）定义包含摩擦模型时，Abaqus/Standard 激活切向接触约束。默认情况下，切向惩罚刚度等于默认法向惩罚刚度。

您可以按步骤缩放切向惩罚刚度以模拟粘附/滑动条件。此缩放仅影响指定该缩放的扰动步骤；它不会延续到后续步骤。如果您希望在一系列扰动步骤中应用相同的比例因子，则必须在每个步骤中明确指定比例因子。

某些依赖频率分析的程序（如复频率分析和基于子空间的稳态动态分析）会受到先前频率分析有效的切向刚度缩放以及这些步骤有效的切向刚度缩放的影响。在这种情况下，建议这些步骤使用一致的缩放。对于其他基于频率分析的模式方法，切向刚度的缩放被忽略，仅考虑先前频率分析的影响。

| **输入文件用法：** | 要在线性扰动步骤中修改所有接触对的切向惩罚刚度： |
| --- | --- |
|  | ``` [*CONTACT CONTROLS](../key/key-link.md#usb-kws-hcontactcontrols), PERTURBATION TANGENT SCALE FACTOR=*factor* ``` 要在线性扰动步骤中修改特定接触对的切向惩罚刚度： ``` [*CONTACT CONTROLS](../key/key-link.md#usb-kws-hcontactcontrols), PERTURBATION TANGENT SCALE FACTOR=*factor*, SLAVE=*slave surface*, MASTER=*master surface* ``` |

### 线性扰动步骤中与接触压力相关的约束强制

在线性扰动步骤中，接触约束通常对所有闭合接触界面完全强制执行，与基础状态中的局部法向压力无关。您可以使用两个控制压力系数 ![](../graphics/usb_eqn02314.gif) 和 ![](../graphics/usb_eqn06124.gif) 来放松在基础状态中压力较低的约束，甚至完全移除它们。法向和切向约束都受到影响。对于基础状态中压力小于 ![](../graphics/usb_eqn02314.gif) 的情况，法向和切向约束通过将约束刚度设置为零有效地被移除。对于压力大于 ![](../graphics/usb_eqn06124.gif) 的情况，约束被完全强制执行。对于介于 ![](../graphics/usb_eqn02314.gif) 和 ![](../graphics/usb_eqn06124.gif) 之间的压力，约束刚度降低，并在 ![](../graphics/usb_eqn02314.gif) 和 ![](../graphics/usb_eqn06124.gif) 之间线性斜升。在这个压力范围内，即使对于原本使用严格拉格朗日乘子强制的接触约束，也会生效有限的接触刚度。初始应力刚度项也被缩放。

所有其他法向和切向接触惩罚的控制都适用。基础状态中开放的约束不受影响。压力相关约束强制不能在一般步骤中使用。

您可以按步骤指定与接触压力相关的约束强制。此规范影响指定该规范的扰动步骤；它不会延续到后续步骤。如果您希望在一系列扰动步骤中应用相同的规范，则必须在每个步骤中明确指定它。

某些依赖频率分析的程序（如复频率分析和基于子空间的稳态动态分析）会受到先前频率分析有效的规范以及这些步骤有效的规范的影响。在这种情况下，建议这些步骤使用一致的规范。对于其他基于频率分析的模式方法，规范被忽略，仅考虑先前频率分析的影响。

| --- | --- |
|  | ``` [*CONTACT CONTROLS](../key/key-link.md#usb-kws-hcontactcontrols), PRESSURE DEPENDENT PERTURBATION=![](../graphics/usb_eqn06124.gif) ![](../graphics/usb_eqn02314.gif) ``` 要在线性扰动步骤中为特定接触对指定基础状态压力相关约束强制： ``` [*CONTACT CONTROLS](../key/key-link.md#usb-kws-hcontactcontrols), PRESSURE DEPENDENT PERTURBATION=![](../graphics/usb_eqn06124.gif), SLAVE=*slave surface*, MASTER=*master surface* ![](../graphics/usb_eqn02314.gif) ``` |

### 与二阶面关联的接触控制

二阶单元不仅提供更高的精度，而且还能更有效地捕获应力集中，比一阶单元更好地建模几何特征。基于二阶单元类型的表面与面-面接触 formulation 配合良好，但在某些情况下与节点-面 formulation 配合不佳（有关这些接触 formulations 的讨论，请参见 ["Abaqus/Standard 中的接触 formulations，" 第 38.1.1 节](pt09ch38s01aus177.md)）。

某些二阶单元类型不适合与节点-面接触 formulation 和"硬"接触条件的严格强制结合使用，因为当压力作用在单元面上时，等效节点力的分布。如 [图 36.3.6-1](pt09ch36s03aus150.md#eq-nodal-loads-contactcontrolsstd) 所示，作用在没有面中节点的二阶单元面上的恒定压力会在角节点处产生与压力方向相反的力。

**图 36.3.6-1**"硬"接触模拟中作用在二阶单元面上的恒定压力产生的等效节点载荷。

![](../graphics/adefsurf-3d-2nd-order-cont.png)

二阶单元中节点力的这种模糊性可能导致 Abaqus/Standard 不适当地改变其内部接触逻辑。基于二阶四面体单元的从表面对于节点-面接触 formulation 也可能有问题，因为对这些单元面上的压力作用等效节点力的分布使得角节点具有零力。

下面讨论 Abaqus/Standard 中使涉及二阶从面的节点-面接触对更容易使用的选项。您也可以通过使用面-面接触 formulation 来避免潜在的困难，这通常是优选的。

#### 手动或自动调整单元类型

修改的 10 节点四面体单元（C3D10M 等）不会对节点-面接触 formulation 造成根本困难，通常为具有节点-面接触对的模型中的 10 节点二阶四面体单元（C3D10、C3D10I 等）提供了可行的替代方案。修改的 10 节点四面体单元与二阶四面体单元特性之间的权衡在 ["实体（连续体）单元"中的"修改的三角形和四面体单元" 第 28.1.1 节](pt06ch28s01alm01.md#usb-elm-esolidcont-modifiedelems) 中讨论。如果需要，您必须对单元类型进行此调整，因为它不会自动发生。

对于大多数与非绑定节点-面接触对关联的 8 节点从面片的 serendipity 单元，Abaqus/Standard 自动添加面中节点。对于三维 18 节点垫片单元，如果用户未在单元连通性中给出面中节点，也会自动生成它们。面中节点的存在导致对接触算法不模糊的节点力分布。单元族 C3D20(RH)、C3D15(H)、S8R5 和 M3D8 分别转换为族 C3D27(RH)、C3D15V(H)、S9R5 和 M3D9。由于 Abaqus/Standard 不转换二阶耦合温度-位移、耦合热-电-结构和耦合孔隙压力-位移单元，因此您应该使用替代方法来避免节点-面接触 formulation 中 serendipity 单元的问题。当从表面用于粘接接触对时，Abaqus/Standard 不会转换二阶 serendipity 单元。

默认情况下，Abaqus/Standard 不会自动为面-面接触对的底层 serendipity 单元添加面中节点；但是，有一个选项可以启用与节点-面接触对使用的自动添加面中节点的相同算法。

| **输入文件用法：** |  |
| --- | --- |
|  | ``` [*CONTACT PAIR](../key/key-link.md#usb-kws-hcontactpair), TYPE=SURFACE TO SURFACE, MIDFACE NODES=YES ``` |

| **Abaqus/CAE 用法：** | 您不能在 Abaqus/CAE 中启用面-面接触对从表面的 serendipity 单元的自动转换。 |
| --- | --- |

#### 补充接触约束

避免某些单元类型给节点-面接触 formulation 带来困难的另一种方法是在不更改底层单元 formulation 的情况下添加补充接触约束。此方法仅适用于使用惩罚或增强拉格朗日约束强制或软化压力-过盈量关系的节点-面接触对，因为如果严格的"硬"接触条件生效，它将导致过度约束情况。补充接触约束有时有助于改善收敛行为或改善接触压力和底层单元应力的平滑度和准确性；然而，额外的约束存在降低收敛行为的风险。默认情况下，补充约束对于具有非修改单元 6 节点从面和 8 节点从面的节点-面接触对选择性地使用，除非严格的"硬"接触条件生效。您可以停用补充约束或为底层从表面的其他二阶单元类型添加激活补充约束。

| **输入文件用法：** | ``` [*CONTACT PAIR](../key/key-link.md#usb-kws-hcontactpair), INTERACTION=*interaction_property_name*, SUPPLEMENTARY CONSTRAINTS=SELECTIVE *slave_surface_name*, *master_surface_name* ``` |
| --- | --- |
|  | 使用以下选项为其他二阶单元类型添加补充接触约束： ``` [*CONTACT PAIR](../key/key-link.md#usb-kws-hcontactpair), INTERACTION=*interaction_property_name*, SUPPLEMENTARY CONSTRAINTS=YES *slave_surface_name*, *master_surface_name* ``` 使用以下选项放弃补充接触约束： ``` [*CONTACT PAIR](../key/key-link.md#usb-kws-hcontactpair), INTERACTION=*interaction_property_name*, SUPPLEMENTARY CONSTRAINTS=NO *slave_surface_name*, *master_surface_name* ``` |

| **Abaqus/CAE 用法：** | 对于节点-面接触 formulation： |
| --- | --- |
|  | Interaction 模块： **Create Interaction**: **Surface-to-surface contact (Standard)**: 选择主表面；点击 **Surface**；选择从表面；Interaction 编辑器； **Use supplementary contact points**: **Selectively**, **Always**, 或 **Never**； **Contact interaction property**: *interaction_property_name* |

### 面-面接触对滑动时接触力重新分配的平滑度

您可以控制面-面接触对滑动时节点接触力重新分配的平滑度。默认设置（通常适当）导致接触力重新分配的平滑度与从表面底层单元的阶次相同；即线性单元的线性重新分配平滑度，以及二阶单元的二次重新分配平滑度。二次重新分配平滑度通常倾向于改善收敛行为，并改善快速变化的接触应力区域内的接触应力分辨率。然而，二次重新分配平滑度倾向于增加每个约束涉及的节点数量，这可能增加方程求解器的计算成本。线性重新分配平滑度倾向于更好地解析主动接触区域边缘附近的接触应力，因此偶尔会产生更好的收敛行为。

| **输入文件用法：** | 使用以下选项指示面-面接触对滑动时接触力重新分配的平滑度应与从表面底层单元的阶次相同： |
| --- | --- |
|  | ``` [*CONTACT PAIR](../key/key-link.md#usb-kws-hcontactpair), TYPE=SURFACE TO SURFACE, SLIDING TRANSITION=ELEMENT ORDER SMOOTHING *slave_surface_name*, *master_surface_name* ``` 使用以下选项指示面-面接触对滑动时接触力重新分配的线性平滑度： ``` [*CONTACT PAIR](../key/key-link.md#usb-kws-hcontactpair), TYPE=SURFACE TO SURFACE, SLIDING TRANSITION=LINEAR *slave_surface_name*, *master_surface_name* ``` 使用以下选项指示面-面接触对滑动时接触力重新分配的二次平滑度： ``` [*CONTACT PAIR](../key/key-link.md#usb-kws-hcontactpair), TYPE=SURFACE TO SURFACE, SLIDING TRANSITION=QUADRATIC *slave_surface_name*, *master_surface_name* ``` |

| **Abaqus/CAE 用法：** | 您不能在 Abaqus/CAE 中更改默认接触力重新分配。 |
| --- | --- |
