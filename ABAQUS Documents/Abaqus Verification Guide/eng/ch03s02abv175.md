# 3.2.3 Steady-state dynamic analysis for infinite elements







**Product: **Abaqus/Standard  

### Elements tested

CIN3D8    CIN3D12R    CIN3D18R    CINAX4    CINAX5R    CINPE4    CINPE5R    CINPS4    CINPS5R    

### Feature tested

Direct-solution steady-state dynamic analysis of infinite elements.

### Problem description

![](../graphics/verssdynamicinf-model-nls.png)

The model consists of a single infinite element connected to one regular continuum finite element. The model is subjected to a plane wave and a shear wave. The results from this analysis are compared with a reference solution obtained from a model in which the infinite element is replaced by dashpots attached to the regular continuum element at points A and B. The damping coefficient corresponding to a plane wave, ![](../graphics/ver_eqn00906.gif), is computed as ![](../graphics/ver_eqn00907.gif), where ![](../graphics/ver_eqn00908.gif) is the plane wave speed. Similarly, the damping coefficient corresponding to a shear wave, ![](../graphics/ver_eqn00909.gif), is computed as ![](../graphics/ver_eqn00910.gif), where ![](../graphics/ver_eqn00911.gif) is the shear wave speed.

**Material: **

Young's modulus = 1.0, Poisson's ratio = 0.1, density = 0.01.

**Boundary conditions: **

Plane wave: ![](../graphics/ver_eqn00272.gif) = 1.0  104 along edge CD, ![](../graphics/ver_eqn00107.gif) = 0 throughout the model.

Shear wave: ![](../graphics/ver_eqn00107.gif) = 1.0  104 along edge CD, ![](../graphics/ver_eqn00272.gif) = 0 throughout the model.

### Results and discussion

The results are confirmed by comparing them to a direct-solution steady-state dynamic analysis of the model in which the infinite elements are replaced by dashpots. The displacements and phase angles match the reference solution in all cases.
