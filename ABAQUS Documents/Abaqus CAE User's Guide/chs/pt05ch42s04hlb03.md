# 42.6.7 控制区域边界处的计算

输出数据库中的默认结果区域复制了分析前用于为模型分配截面属性的区域。或者，您可以选择使用单元集或显示组作为结果区域。您可以控制在两个或多个结果区域共用的节点处的计算。选择忽略区域边界会以计算方式组合结果，从而获得更平滑、更连续的效果。选择不忽略区域边界会以视觉方式强调模型间断。请参见[图42-6](pt05ch42s04s02.md#viw-average)，了解跨区域平均和不跨区域平均的等值线图示例；跨区域平均发生在您选择忽略区域边界时。

**控制区域边界处的计算：**

1. 找到"Region Boundaries"选项。从主菜单栏中，选择****Result**![](../graphics/images/arrow.gif)**Options****；然后在出现的对话框中单击**Computation**标签。"Region Boundaries"选项出现在"平均"控件内。
2. 要控制在两个或多个结果区域共用的节点处的计算，请切换**使用区域边界**。使用区域边界意味着在计算节点平均值或计算间断时*不*组合来自相邻结果区域的值。
3. 选择以下选项之一来定义结果区域：
   **提示：**要查看当前结果区域，请使用**平均区域**属性选择方法对未变形或变形图进行颜色编码。（有关详细信息，请参阅["在可视化模块中对所有几何进行颜色编码，" 第77.5节](pt07ch77s05.md)。）
   **ODB区域** 默认情况下，结果区域与分析前保存到输出数据库(`.odb`)文件的截面分配区域相同。有关的详细信息，请参阅["为部件分配截面、方向、法向和切线，" 第12.15节](pt03ch12s15.md)。
   **单元集** 通过选择单元集指定结果区域。
   **显示组** 通过选择显示组指定结果区域。
   **注意：**如果所需的显示组不存在，请选择****Tools**![](../graphics/images/arrow.gif)**Display Group**![](../graphics/images/arrow.gif)**Create****来创建它们（有关详细信息，请参阅["创建或编辑显示组，" 第78.2.1节](pt07ch78s02hlb01.md)）。区域边界使用选择时定义的显示组来定义。如果您随后修改或删除显示组，区域边界不会更改，除非您重新定义它们。
4. 如果您选择了"单元集"或"显示组"方法来设置区域边界，请完成以下步骤：
   1. 单击![](../graphics/ico_selectBlue.png)。Abaqus/CAE打开"指定平均区域"对话框。
   2. 从对话框左侧的列表中，单击所需的区域以在列表中突出显示它们。使用**[Shift]****+Click**和**[Ctrl]****+Click**选择多个项目。
   3. 单击对话框中心的右箭头按钮，将突出显示的项目复制到**Selection**列。要移动整个列表，请单击双右箭头按钮。
   4. 要删除区域选择，请在**Selection**列中突出显示它们，然后使用左箭头按钮。未包含在区域边界选择中的任何单元将使用"无结果"颜色进行着色（有关详细信息，请参阅["对没有结果的单元进行颜色编码，" 第55.12.4节](pt05ch55s12hlb04.md)）。重叠的选择将被合并。
5. 默认情况下，Abaqus/CAE使用特征边缘作为额外的区域边界。要更改用于创建特征边缘的最小角度，请单击![](../graphics/ico_edit.png)并在"ODB Display Options"对话框中更改**Feature Angle**设置。特征边缘对应于未变形模型，不会因模型变形而重新计算。（有关详细信息，请参阅["定义模型特征边缘，" 第55.3.2节](pt05ch55s03hlb02.md)。）要跨特征边缘平均结果，请切换关闭**包括壳/膜特征边缘边界**。
6. 单击**Apply**以实现您的更改。Abaus根据您的规格在当前视口中计算用于显示的结果值。默认情况下，您的更改会在会话期间保存，并会影响后续的所有结果显示。如果您希望将更改保留到后续会话中，请将其保存到文件。有关的详细信息，请参阅["保存自定义设置以供后续会话使用，" 第55.1.1节](pt05ch55s01s01.md)。
