# 17.11.8 创建自底向上网格







自底向上网格划分是一个手动过程，可能耗时较长，需要一些反复试验才能生成可接受的网格。自底向上网格划分主要用于需要六面体单元进行分析且无法使用自顶向下网格划分技术生成六面体单元的情况。您也可以在需要大量分区才能在几何体上创建高质量自顶向下网格的情况下，或在您想扩展网格零件的情况下使用自底向上网格划分。以下过程包含创建自底向上网格的基本步骤。要创建自底向上网格，您必须首先拥有网格区域或分配了自底向上网格划分技术的实体区域（有关更多信息，请参见"分配网格控制"第 17.18.1 节）。您也可以在包含本地零件独立实例或包含孤立网格和几何任意组合的零件的从属实例的装配体上创建自底向上网格。

**创建自底向上网格：**

1. 从主菜单栏中，选择 ****Mesh**![](../graphics/images/arrow.gif)**Create Bottom-Up Mesh****。**提示：**您也可以点击网格模块工具箱中的 ![](../graphics/ico_meshBottomUp.png) 工具。（有关更多信息，请参见"使用网格模块工具箱"第 17.15 节。）Abaqus/CAE 打开 **Create Bottom-Up Mesh** 对话框，并在提示区域显示提示以指导您完成过程。

2. 选择要为其创建自底向上网格的 **Domain**。如果当前零件或装配体只有一个适当的域，Abaqus/CAE 会自动进行选择。如果有多个分配了自底向上技术的几何区域或单元，请点击 **Select** 来选择将与您正在创建的自底向上网格关联的单元。您可以选择的区域显示为浅棕色。如果您使用孤立单元工作，则单元与零件中的任何几何区域之间没有关联。**注意：**为自底向上网格划分选择几何区域后，您可以使用 **Edit** 按钮更改您的选择。

3. 选择以下网格划分方法之一： - **Sweep** - **Extrude** - **Revolve** - **Offset**（仅适用于单元面选择） 有关每种方法的说明，请参见"自底向上网格划分方法"第 17.11.3 节。

4. 如果您之前为当前区域选择了自底向上网格划分参数，可以点击 **Fetch Last Selections** 重用您之前的 selections。Abaqus/CAE 会保存您的域、源侧、连接侧、目标侧、挤出向量和旋转轴的选择，直到被新的选择覆盖。**注意：**如果您向模型添加分区，则无法重用之前选择的几何；但是，只要您没有修改网格，就可以重用网格选择，例如单元面和二维单元。

5. 点击 **Source side** 旁边的 **Select**；然后选择要开始网格划分的区域面、单元面或二维单元。**提示：**使用 **[Shift]****+Click** 选择多个面。如果您正在创建偏移网格并选择了二维单元，Abaqus/CAE 会提示您指示将偏移的选定的哪一侧。您可以选择 **Brown** 侧、**Purple** 侧或 **Both sides**。Abaqus/CAE 将您的选择显示为品红色。

6. 完成源侧选择后，点击提示区域中的 **Done**。

7. 如果您选择了 **Sweep** 方法，必须选择 **Connecting sides** 或 **Target side** 或两者来完成网格定义。完成以下步骤进行选择： 1. 开启 **Connecting sides**，然后点击 **Select** 选择连接源侧到网格末端的一个或多个侧面。连接几何侧面必须是四边的。您也可以选择形成规则网格模式的四边单元面或二维单元。 有关更多信息，请参见"为自底向上扫掠网格定义连接边"第 17.11.7 节。 2. 如果您使用几何体工作，可以开启 **Target side** 并点击 **Select** 选择 Abaqus/CAE 用于结束扫掠网格的单个面。

8. 如果您选择了 **Extrude** 方法，必须指定挤出向量并选择挤出深度。 1. 点击 **Vector** 旁边的 **Select** 从视口选择两个点或在提示区域输入坐标，以定义挤出向量的起点和终点。Abaqus/CAE 在视口中显示黄色箭头以指示挤出向量。 2. 选择定义挤出深度的方法： - 开启 **Use vector length** 按上一步创建的向量长度进行挤出。 - 开启 **Specify** 并输入挤出深度。 - 开启 **Project to target** 并点击 **Select** 指定目标侧。目标侧必须沿挤出向量规定的方向。Abaqus/CAE 使用目标侧来设置挤出距离。挤出的网格将密切匹配目标侧的形状，但不会符合目标侧上的网格。（有关更多信息，请参见"自底向上网格划分方法"第 17.11.3 节。）

9. 如果您选择了 **Revolve** 方法，必须指定轴和旋转角度。 1. 点击 **Axis** 旁边的 **Select** 从视口选择两个点或在提示区域输入坐标，以定义旋转轴的起点和终点。Abaqus/CAE 在视口中显示黄色箭头以指示轴。 2. 在 **Angle** 字段中，输入旋转网格所需的角度（以度为单位）。

10. 如果您选择了 **Offset** 方法，必须指定总厚度，并且可以更改二维网格角落的偏移方式。 1. 在 **Total thickness** 字段中，输入偏移网格的厚度。 2. 对于形成尖锐角落的壳单元的源侧，开启 **Constant thickness around corners** 以在角落处元素相遇的地方保持与选择其余部分相同的总厚度。使用此选项可以减少单元扭曲并防止单元坍塌，特别是如果元素偏移到角落的内侧（有关更多信息，请参见"在网格偏移期间减少单元扭曲和坍塌"第 64.3.3 节）。

11. 在适用的情况下，输入 **Number of layers** 来定义源侧与自底向上网格末端之间的单元层数。**注意：**仅当您指定没有连接侧的目标侧时，**Number of layers** 参数对于 **Sweep** 方法是必需的。对于 **Extrude**、**Revolve** 和 **Offset** 方法，它始终是必需的。

12. 如果需要，为挤出自底向上网格输入 **Bias ratio**，以使单元层厚度向源侧倾斜或远离源侧。**Bias ratio** 与 **Number of layers** 结合使用。偏差比是源侧处单元层厚度与距源侧最远处单元层厚度之比。

13. 如果需要，使用 **Options** 区域将新的自底向上单元添加到单元集：**Extend existing sets** 每个新的自底向上单元将被添加到包含其源侧父单元的任何网格集中。**Create a set for new elements** 为自底向上单元创建新的单元集。您可以使用默认集名称或为该集创建一个名称。如果您使用孤立单元工作，也可以为每一层单元创建单独的集合。如果您选择此选项，Abaqus/CAE 会用 `-Layer-*n*` 递增基本集名称，其中 *n* 是层号。

14. **Create Bottom-Up Mesh** 对话框底部的 **Undo** 字段允许您撤消和重做自底向上网格更改、网格-几何关联更改以及网格编辑。您可以在编辑网格工具集中访问相同的功能。 有关更多信息，请参见"在编辑网格工具集中撤消或重做更改"第 64.9 节。

15. 点击 **Create Bottom-Up Mesh** 对话框中的 **Mesh** 以生成自底向上网格。

16. 重复步骤 3 到 15 以在同一区域中创建另一个自底向上网格，或点击 **Cancel** 结束过程。
