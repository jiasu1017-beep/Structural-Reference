# 10.9.6 将视口数据导出为OBJ格式文件





从主菜单栏选择**文件**![](../graphics/images/arrow.gif)**导出**![](../graphics/images/arrow.gif)**OBJ**，可以将当前视口中的模型导出为OBJ格式文件。当从可视化模块导出数据时，Abaqus/CAE仅将网格数据导出到OBJ格式文件。如果从任何其他模块导出数据，Abaqus/CAE会在当前视口中显示网格时导出网格数据；否则，Abaqus/CAE导出几何数据。

从可视化模块导出时，会考虑当前步骤和帧。如果在视口中显示未变形图，Abaqus/CAE会将未变形数据导出到OBJ格式文件；如果显示变形图，Abaqus/CAE会将当前步骤和帧的变形数据导出到OBJ格式文件。

在导出过程中，Abaqus/CAE对几何和网格数据执行以下转换：
- 模型数据库中的实体单元在OBJ格式文件中转换为壳单元。
- 模型数据库中的四边形单元在OBJ格式文件中转换为三角形单元。
- 模型数据库中的二阶单元在OBJ格式文件中转换为一阶单元。如果您的模型使用大量二阶单元进行网格划分，导出过程会创建大型OBJ文件，因为Abaqus/CAE会将二阶单元拆分为最终的一阶单元。

您导出的几何或网格数据的质量以及生成的文件大小都取决于曲线细化设置。有关更改曲线细化质量的更多信息，请参见"控制曲线细化"，第76.4节。

**将视口数据导出为OBJ格式文件：**

1. 如果从可视化模块导出数据，请选择要导出数据的绘图状态。对于从变形绘图状态导出数据，请选择要导出数据的步骤和帧。有关选择当前步骤和帧的更多信息，请参见"选择结果步骤和帧"，第42.3节。
2. 从主菜单栏，选择**文件**![](../graphics/images/arrow.gif)**导出**![](../graphics/images/arrow.gif)**OBJ**。出现**导出至OBJ文件**对话框。
3. 从对话框底部的**文件过滤器**菜单中，选择**OBJ (*.obj)**。Abaqus/CAE列出所选目录中所有具有`.obj`文件扩展名的文件。
4. 选择要导出几何和网格数据的文件，或在**文件名**文本字段中输入新文件的名称。
5. 单击**确定**导出数据并关闭**导出至OBJ文件**对话框。
