# 34.4.4 施加载荷：概述









**产品：** Abaqus/StandardAbaqus/ExplicitAbaqus/CFD

##### **参考资料**

- [*LOAD*](#)

### 概述

载荷定义是有限元分析的重要组成部分，用于模拟作用于结构上的外部作用。Abaqus 提供了多种类型的载荷，可以施加于节点、单元或几何实体。

### 载荷类型分类

Abaqus 中的载荷可以分为以下几类：

#### 集中载荷

集中载荷作用于单个节点或点：

- **集中力**（CLOAD）：作用于节点的外力
- **集中弯矩**：作用于节点的力矩
- **集中热流**：作用于节点的热通量

#### 分布载荷

分布载荷作用于单元面或单元边：

- **压力**（DLOAD）：均匀或非均匀压力
- **线载荷**：作用于梁或桁架单元
- **面载荷**：作用于壳或实体单元

#### 体积载荷

体积载荷作用于整个单元：

- **重力**：体积力
- **离心力**：旋转惯性力
- **热载荷**：由温度梯度引起的体积力

### 载荷施加方式

载荷可以通过以下方式施加：

| 方法 | 描述 | 输入用法 |
| --- | --- | --- |
| 直接定义 | 在分析步中直接定义 | `*CLOAD` 或 `*DLOAD` |
| 幅值曲线 | 使用幅值曲线定义时间变化 | `AMPLITUDE=name` |
| 表格定义 | 通过表格方式定义空间变化 | 空间插值 |

### 分析步中的载荷

载荷在分析步（Step）中定义，可以是：

- **静力载荷**：静力学分析
- **动力载荷**：动力学分析
- **耦合载荷**：多物理场耦合

| **输入文件用法：** | ```
*STEP
*STATIC
*LOAD
...
*END STEP
``` |
| --- | --- |

### 载荷组合

多个载荷可以同时施加：

| **输入文件用法：** | ```
*LOAD
node_set_1, dof1, value1
node_set_2, dof2, value2
``` |
| --- | --- |
