# 13.8.4 Querying the assembly







You can use the Query toolset to request either general information or module-specific information. For a discussion of the information displayed by general queries, see ["Obtaining general information about the model," Section 71.2.2](pt06ch71s02hlb02.md).

In addition, you can use the Assembly module-specific queries to determine the following attributes of a part instance or a model instance:
- Name, type, and modeling space
- Origin
- The sum of the translations and rotations applied to the instance

For more information, see ["Using the Query toolset to query the assembly," Section 13.12](pt03ch13hla01.md).
