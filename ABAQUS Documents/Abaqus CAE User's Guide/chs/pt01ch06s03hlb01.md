# 6.3.3 基于对象位置过滤你的选择





选择工具允许你基于对象在视口中的位置来选择从哪些对象中选择。**选择**工具栏包含所有选择工具。以下基于位置的选择工具仅在你开始需要对象选择的过程后才可用（基于位置的选择在过程之外不可用）：

**屏幕上最接近的对象**

![](../graphics/ico_filterVisible.png) 切换开启此工具以仅选择屏幕上最接近的对象。此工具默认是开启的。

如果你关闭此工具，Abaqus/CAE 允许你循环浏览所有可能的选择。使用提示区域中的**下一个**和**上一个**按钮在视口中所有可能的选择对象之间向前和向后循环；每个对象会依次高亮显示。更多信息，请参阅["循环浏览有效选择，" 第 6.2.10 节](pt01ch06s02hlb10.md)。

此过滤器适用于本机顶点、边、面和单元体，以及孤立节点和孤立单元。

**内部和外部对象**

选择以下过滤器之一：
- ![](../graphics/ico_filterBothIntExt.png) 选择位于部件外部和内部的对象。
- ![](../graphics/ico_filterExterior.png) 仅选择位于部件外部的对象。在大多数情况下，此工具默认是选中的。
- ![](../graphics/ico_filterInterior.png) 仅选择位于部件内部的对象。
