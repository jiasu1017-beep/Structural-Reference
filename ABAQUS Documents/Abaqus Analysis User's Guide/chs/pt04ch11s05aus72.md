### 概述

流体交换用于定义流体腔体之间或腔体与外部环境之间的流体流动。流体交换可以是无源的（如通过孔口自然流动）或主动控制的（如通过阀门）。

### 流体交换类型

#### 自然流动

流体通过孔口或缝隙在腔体之间或与外部环境之间自然流动。流动由压差驱动。

####  orifice流动

通过定义orifice（节流孔）的特性来控制流动：
- **orifice直径**：控制流动截面
- **流动系数**：定义流动效率
- **临界压力比**：定义何时达到声速流动

#### 主动控制流动

通过阀门或其他控制装置主动调节流体流量。流量可以是时间的函数，也可以是系统状态的函数。

### 定义流体交换

使用**FLUID EXCHANGE**关键字定义流体交换：

```
[*FLUID EXCHANGE](../key/key-link.md#usb-kws-mfluid-exchange), TYPE=ORIFICE
*腔体1*, *腔体2*, 
*orifice参数
```

### orifice方程

orifice流动使用以下方程描述：

#### 亚声速流动

当下游压力与上游压力比大于临界压力比时：

```
m = Cd * A * P_up * sqrt(2 * gamma / (R * T) * (gamma-1)) * sqrt((P_down/P_up)^(2/gamma) - (P_down/P_up)^((gamma+1)/gamma))
```

#### 声速流动

当下游压力与上游压力比小于或等于临界压力比时，流动达到声速，流量达到最大值。

### 应用场景

#### 安全气囊系统

多个气囊腔体之间通过orifice连接，模拟气体在气囊间的流动，影响展开行为。

#### 液压系统

液压缸与蓄能器之间通过orifice连接，模拟压力波动和能量存储。

#### 呼吸系统

气袋与外部环境通过可调节阀门连接，模拟呼吸循环。

### 耦合与解耦

#### 耦合分析

流体交换与结构变形完全耦合，同时求解。

#### 顺序耦合

先求解结构问题，然后更新流体交换状态。

### 初始条件

流体交换的初始条件包括：
- 初始流量（如果非零）
- 初始压差

### 数值稳定性

处理流体交换时的数值考虑：
- 使用适当的时间步长
- 对于快速变化的过程，使用显式动力学分析
- 考虑使用质量缩放来改善收敛性

### 输出

流体交换分析会产生以下输出：
- 通过orifice的质量流率
- 压差
- 上下游腔体压力
- 能量耗散

### 诊断

常见问题及解决方案：
- 流量不连续：检查时间步长和orifice参数
- 收敛困难：考虑使用更小的初始时间步长
- 压力振荡：可能需要添加阻尼或调整orifice参数
