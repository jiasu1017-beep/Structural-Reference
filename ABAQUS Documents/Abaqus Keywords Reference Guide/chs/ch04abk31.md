# *DOMAIN DECOMPOSITION





### *DOMAIN DECOMPOSITION定义域分解的区域和/或定义域分解的约束。

此选项用于定义域分解区域和/或定义域分解的约束。

**产品：**Abaqus/Explicit  

**类型：**模型数据  

**级别：**装配  

##### **参考：**

- ["Abaqus/Explicit中的并行执行，" Abaqus分析用户指南第3.5.3节](../usb/usb-link.md#usb-int-aparallelexecution)

### **可选参数：**

ELSET

将此参数设置为您要定义的域分解区域的单元集合名称。此单元集合将被拆分为用户指定的并行域数量。如果省略此参数，则此域分解区域将包含所有未包含在其他域分解区域中的单元。

### **定义域分解约束的数据行：**

**第一行：**

根据需要重复此数据行，以定义域分解的约束。
