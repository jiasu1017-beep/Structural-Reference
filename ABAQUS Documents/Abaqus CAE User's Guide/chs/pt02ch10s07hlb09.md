# 10.7.6 从Elysium中性文件导入零件





从主菜单栏选择**文件**![](../graphics/images/arrow.gif)**导入**![](../graphics/images/arrow.gif)**零件**，可以从Elysium中性文件导入零件。您可以导入由以下软件生成的Elysium中性文件：
- 使用Elysium翻译器插件的CATIA V5。
- 使用Abaqus/CAE NX关联接口的NX。
- 使用Pro/ENGINEER关联接口的Pro/ENGINEER。

有关Abaqus/CAE支持的CATIA V5、NX和Pro/ENGINEER版本的信息，请参见Dassault Systmes知识库，网址为[www.3ds.com/support/knowledge-base](http://www.3ds.com/support/knowledge-base)。

在零件导入过程中，Abaqus/CAE自动确定所选文件是由CATIA V5、NX还是Pro/ENGINEER生成，并相应地定制导入过程。来自Elysium中性文件的导入零件在Abaqus/CAE中形成新零件的基础特征；您不能直接修改此基础特征，但可以向其添加其他特征，例如实体拉伸或盲孔切割。

**从Elysium中性文件导入零件：**

1. 从主菜单栏，选择**文件**![](../graphics/images/arrow.gif)**导入**![](../graphics/images/arrow.gif)**零件**。出现**导入零件**对话框。
2. 从**导入零件**对话框底部的**文件过滤器**菜单中，选择**ProE/NX/CATIA V5 Elysium中性 (*.enf*)**。Abaqus/CAE列出当前目录中所有具有`.enf*`文件扩展名的文件。
3. 选择包含要导入的零件的Elysium中性文件，然后单击**确定**。Abaqus/CAE显示**从ENF文件创建零件**对话框。
4. 单击**名称-修复**选项卡设置以下内容：**名称** 新零件的名称。更多信息，请参见"使用基本对话框组件"，第3.2.1节。**修复选项** Abaqus/CAE在导入过程中自动修复Elysium中性文件零件。**零件过滤器** Elysium中性文件可以包含多个零件，Abaqus/CAE默认导入文件中的所有零件。如果导入Elysium中性文件中的所有零件，您可以为每个零件创建单独的Abaqus零件，也可以将它们组合成一个Abaqus零件；此外，如果组合这些零件，Abaqus/CAE允许您使用用户指定的缝合容差值将这些导入的零件缝合在一起。或者，您可以切换**导入零件编号**并输入要从文件导入的单个零件的编号。
5. 单击**零件属性**选项卡设置以下内容：-正在导入的零件的建模空间。导入零件时，Abaqus/CAE会扫描文件并尝试确定建模空间。-正在导入的零件的类型。Abaqus/CAE始终假定零件类型为可变形。或者，您可以选择**离散刚体**导入离散刚体零件，或选择**欧拉**导入欧拉零件。更多信息，请参见"零件类型"，第11.4.2节。在大多数情况下您应该接受默认设置；更多信息，请参见"什么是零件属性？"，第10.3.2节。
6. 单击**缩放**选项卡以在导入过程中缩放零件。-选择**不更改单位**以保持文件中存储的单位和尺寸。-选择**将单位转换为mm**以将Elysium中性文件中存储的单位转换为毫米。-选择**将所有长度乘以**，然后输入缩放因子。Abaqus/CAE将缩放因子应用于文件中的所有坐标。因此，距原点的任何偏移都将按比例缩放。
7. 单击**确定**导入零件并退出**从ENF文件创建零件**对话框。Abaqus/CAE扫描Elysium中性文件并根据所选选项启动修复过程。如果希望取消导入过程，请单击提示区域中的**停止**。
8. 导入单个零件时，Abaqus/CAE会在消息区域显示一条消息，指示零件是否存在任何有效性或精度问题。如果从单个Elysium中性文件导入多个零件，可以使用**零件管理器**检查任何零件是否无效或不精确。如有必要，使用查询工具集定位无效和不精确几何的区域。更多信息，请参见"在零件模块中使用查询工具集"，第11.16.4节。
