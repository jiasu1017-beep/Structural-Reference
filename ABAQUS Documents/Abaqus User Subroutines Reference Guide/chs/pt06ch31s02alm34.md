### 概述

连接止动器和锁可以：
- 在任何具有相对运动可用分量的连接器中指定；
- 用于指定单个相对运动分量中的接触强制止动；以及
- 用于在满足特定准则时将相对运动的可用分量锁定在位置。

### 定义连接止动器

在大多数连接器的物理构造中，一个物体相对于另一个物体的可接受位置受一定范围限制。在 Abaqus 中，这些限制建模为内置不等式约束。您指定要定义连接止动器的相对运动可用分量，以及连接在相对运动分量方向上可接受范围的下限和上限值。

| **输入文件用法：** | 使用以下选项定义连接止动器： |
| --- | --- |
|  | ``` [*CONNECTOR BEHAVIOR](../key/key-link.md#usb-kws-mconnectorbehavior), NAME=*name* [*CONNECTOR STOP](../key/key-link.md#usb-kws-mconnectorstops), COMPONENT=*component number* *lower limit*, *upper limit* ``` |

| **Abaqus/CAE 用法：** | 相互作用模块：连接截面编辑器：****添加**![](../graphics/images/arrow.gif)**止动器****：**分量：***component or components*，**下限：***lower limit*，**上限：***upper limit* |
| --- | --- |

### 定义连接锁

连接机构可能具有一旦达到所需配置就将连接器锁定的装置。例如，旋转连接可能具有落针机制，在达到所需角度后锁定旋转运动。可以为包含相对运动可用分量的连接单元定义用户定义的连接锁定准则。您可以选择为其定义锁定准则的相对运动分量。

连接锁可用于为约束以及相对运动的可用分量指定连接行为。可以为连接中涉及的所有相对运动分量指定力或力矩的极限值。用于评估准则的力/力矩如输出变量 CTF 中所计算。此外，可以为相对运动的可用分量对应的相对位置指定极限值。如果没有为相对运动的可用分量指定其他行为，力锁定准则将无用，因为 CTF 为零。

在 Abaqus/Explicit 中，您还可以指定可用分量中速度的极限值作为锁定的准则。速度相关的锁定准则在建模汽车安全带系统时很有用（参见 ["简化碰撞假人的安全带分析，" Abaqus 例题指南第3.3.1节](../exa/exa-link.md#exa-veh-seatbelt)）。此外，极限值可以依赖于温度和场变量。场变量依赖性可用于建模时间相关锁。

如果满足为所选相对运动分量指定的锁定准则，则所有分量锁定或单个可用分量锁定。默认情况下，满足锁定准则时，所有相对运动分量都被锁定。在这种情况下，连接单元将从该点开始完全运动锁定。在动态分析中，这种锁定可能引入高加速度。您可以指定是否只锁定所选的相对运动分量。

| **输入文件用法：** | 使用以下选项定义连接锁： |
| --- | --- |
|  | ``` [*CONNECTOR BEHAVIOR](../key/key-link.md#usb-kws-mconnectorbehavior), NAME=*name* [*CONNECTOR LOCK](../key/key-link.md#usb-kws-mconnectorlock), COMPONENT=*component number*, LOCK=ALL or *component number* ``` |

| **Abaqus/CAE 用法：** | 相互作用模块：连接截面编辑器：****添加**![](../graphics/images/arrow.gif)**锁****：**分量：***component or components*，**锁定**：**全部**或**指定***component* |
| --- | --- |

#### 示例

在 [图31.2.8-1](pt06ch31s02alm34.md#econnectorbehavior-shock-stoplock) 的示例中，假设如果局部3方向上的力超过 500.0 单位，则绕减震器的相对旋转将锁定。

```
[*CONNECTOR BEHAVIOR](../key/key-link.md#usb-kws-mconnectorbehavior), NAME=sbehavior
[*CONNECTOR LOCK](../key/key-link.md#usb-kws-mconnectorlock), COMPONENT=3, LOCK=4
, , -500.0, 500.0
```

### 在线姓扰动过程中定义连接止动器和锁

连接锁或止动器的状态不能在线性扰动分析期间改变；所有连接止动器和连接锁定义保持在与基础状态相同的状态。

### 输出

连接的可用 Abaqus 输出变量列在 ["Abaqus/Standard 输出变量标识符，" 第4.2.1节](pt02ch04s02abv01.md) 和 ["Abaqus/Explicit 输出变量标识符，" 第4.2.2节](pt02ch04s02xbv01.md) 中。在连接中定义止动器和锁时，以下输出变量特别令人关注：

| CSLST | 连接止动器和锁的标志。 |
| --- | --- |

| CRF | 连接反作用力/力矩。 |
| --- | --- |

在给定时间和特定相对运动分量 *i*，如果连接器在该分量中实际停止或锁定（满足止动器或锁定准则），则输出变量 CSLST*i* 为 1。在这种情况下，相应的 CRF 输出变量很可能为非零，等于强制执行止动器或锁定约束所需的实际力/力矩。由于 CRF 包含在 CTF 的计算中，因此当锁或止动器处于活动状态时，后者也会改变。

如果在给定时间特定分量 *i* 未满足止动器或锁定准则，则输出变量 CSLST*i* 为 0，并且在大多数情况下相应的反作用力 CRF 为零（唯一的可能例外是当在该分量中也施加了连接运动时）。
