# 17.13.1 什么是重新网格划分规则？





重新网格划分规则使Abaqus/CAE能够迭代地调整您的网格以满足您指定的误差指标目标。您可以允许Abaqus/CAE执行迭代重新网格划分和分析操作，或者您可以手动重新网格划分并研究重新网格划分规则对网格和结果分析的影响。Abaqus/CAE重新网格化您为其分配了自适应规则的面和单元以及任何相邻的面或单元；其他区域上的网格不会改变。更多信息，请参见["自适应重新网格划分：概述，"Abaqus Analysis User's Guide第12.3.1节](../usb/usb-link.md#usb-anl-aadpover)。

重新网格划分规则描述了自适应网格规范的所有方面：
- 应用重新网格划分规则的区域。您可以将重新网格划分规则应用于整个模型或选定的区域。
- Abaqus/CAE将应用该规则的特定步骤。重新网格划分规则仅在此步骤期间应用；但是，您可以将具有相同设置的不同重新网格划分规则应用于模型中的另一个步骤。
- 误差指标输出变量——将用于计算误差估计的输出变量。更多信息，请参见["影响自适应重新网格划分的误差指标选择，"Abaqus Analysis User's Guide第12.3.2节](../usb/usb-link.md#usb-anl-aadperrorindicators)。
- 尺寸方法——Abaqus/CAE将用于计算网格中单元尺寸的方法。更多信息，请参见["基于求解的网格尺寸，"Abaqus Analysis User's Guide第12.3.3节](../usb/usb-link.md#usb-anl-aadpsizing)。
- 重新网格划分计算的任何约束。

重新网格划分规则与边缘布种、单元类型和网格方法结合使用，以确定特定自适应迭代中的网格。重新网格划分规则存储在模型数据库中，并在会话之间保持。要创建重新网格划分规则，请从主菜单栏中选择****自适应**![](../graphics/images/arrow.gif)**重新网格划分规则**![](../graphics/images/arrow.gif)**创建****。更多信息，请参见["创建重新网格划分规则，"第17.21.1节](pt03ch17s21s01.md)。

您可以在模型的多个区域上定义多个重新网格划分规则。如果您将多个重新网格划分规则应用于模型的同一区域，Abaqus/CAE应用保守的单元尺寸规范，在特定点上定义更细网格的规则优先。如果您将重新网格划分规则分配给从属实例，Abaqus/CAE会重新网格化原始零件，零件的每个从属实例继承相同的网格。

当重新网格划分规则处于活动状态时，Abaqus/CAE会在您创建的每个作业中请求误差指标输出变量。重新网格划分规则在第一个作业期间对网格没有影响。然而，在第一个作业期间，Abaqus使用重新网格划分规则计算误差指标输出变量。在后续的自适应重新网格划分迭代中，重新网格划分规则会增加您的网格尺寸规范，以生成尝试优化单元尺寸和放置从而实现规则中描述的误差指标目标的网格。
