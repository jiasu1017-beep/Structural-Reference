# 19.7.1 创建新分析作业






要创建新分析作业，请从主菜单栏选择****作业**![](../graphics/images/arrow.gif)**创建****。分析作业存储在模型数据库中，并在会话之间保持。

**创建新分析作业：**

1. 从主菜单栏，选择****作业**![](../graphics/images/arrow.gif)**创建****。此时出现**创建作业**对话框。**提示：**您也可以通过在模块工具箱中单击 ![](../graphics/ico_job.png) 图标，或通过在**作业管理器**中单击**创建**来创建新分析作业。
2. 在**名称**文本字段中输入新作业的名称。您指定的名称必须遵守操作系统的文件名规则。
3. 单击**源**字段右侧的箭头以选择作业的来源。- 选择**模型**以创建基于在 Abaqus/CAE 中创建的模型的作业。**模型**列表显示模型数据库中定义的所有模型。从列表中，选择要与新作业关联的模型。- 选择**输入文件**以创建基于可能由 Abaqus/CAE 创建或未创建的输入文件的作业。单击 ![](../graphics/ico_fileOpen.png)；Abaqus/CAE 列出所选目录中带有 `.inp` 文件扩展名的所有文件。选择要与新作业关联的输入文件，然后单击**确定**。**注意：**您不能创建基于引用其他结果文件（如重启分析、导入分析或子模型分析）的输入文件的作业。
4. 单击**继续**。此时出现作业编辑器。
5. 如果需要，输入作业描述。当您提交作业进行分析时，Abaqus/CAE 会在输入文件标题之后立即写入作业描述。作业描述不会写入输出数据库，导入到 Abaqus/CAE 时也不会保留。有关更多信息，请参阅[「导入描述」中的「从 Abaqus 输入文件导入模型，第 10.5.2 节」](pt02ch10s05s02.md#usi-imp-inp-desc)。
6. 在编辑器中，输入定义作业所需的所有数据，然后单击**确定**。（有关更多信息，请参阅[「使用作业编辑器，第 19.8 节」](pt03ch19s08.md)。）
