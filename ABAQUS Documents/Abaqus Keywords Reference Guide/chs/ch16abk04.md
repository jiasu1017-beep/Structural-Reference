# *PART









### *PART开始部件定义。

此选项用于开始部件定义。它必须与[*ASSEMBLY](ch01abk14.md)、[*END PART](ch05abk19.md)和[*INSTANCE](ch09abk19.md)选项结合使用。

**产品：**Abaqus/Standard  Abaqus/Explicit  Abaqus/CFD  Abaqus/CAE  

**类型：**模型数据  

**级别：**模型  

**Abaqus/CAE：**部件模块

##### **参考：**

- ["定义装配，" Abaqus Analysis User's Guide第2.10.1节](../usb/usb-link.md#usb-int-ipartassy)
- [*END PART](ch05abk19.md)

### **必需参数：**

NAME

将此参数设置为一个标签，该标签将用于引用部件。

**此选项没有关联的数据行。**
