# 17.4.6 最小化种子重定位







在网格生成过程中，Abaqus/CAE 使用您创建的种子作为网格边缘沿线的节点目标位置。但是，如果您使用四边形或六面体形状的单元，种子与节点之间的紧密匹配在很大程度上取决于以下因素：

**您在过渡区域允许的单元形状**

如果您在过渡区域允许三角形单元，则种子与网格节点的匹配会更好。如果您限制网格仅包含四边形单元，则种子和节点不太可能匹配。

**网格过渡设置**

如果您允许网格过渡，种子与网格节点的匹配会更好。

**网格划分技术**

使用前进前缘网格划分算法生成的网格比使用中轴算法生成的网格更符合您的布种子。

**种子约束**

完全约束的种子在数量和位置上与生成的节点紧密匹配。但是，您必须仅对零件或零件实例的少数边缘进行完全约束；否则，Abaqus/CAE 将无法生成网格。

**相邻区域的布种子方式**

当对多个区域进行网格划分时，Abaqus/CAE 通常会重新分配单元，以使网格在区域之间兼容。即使单个区域的种子安排足以生成该区域上的网格，种子安排可能需要更改，因为沿共享边缘的单元数量必须与相邻区域兼容。

**注意：**零件实例之间的网格兼容性无法保证。在某些简单情况下，布种子可以帮助实现零件间的网格兼容性。获得兼容网格的技术在"零件实例之间的兼容网格"第 17.14.3 节中有描述。

Abaqus/CAE 在为整个模型平衡单元重新分配时，会尽可能遵守您规定的种子数量和位置。如果在单个布种子边缘进行大更改和在多个边缘进行小更改之间进行选择，Abaqus/CAE 会进行多个小更改。
