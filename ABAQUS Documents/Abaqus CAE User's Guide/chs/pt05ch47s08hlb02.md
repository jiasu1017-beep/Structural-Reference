# 47.9.2 自定义 X–Y 绘图图例的标题





### 47.9.2 自定义 X–Y 绘图图例的标题

使用 **标题（Title）** 选项来显示或隐藏 X–Y 绘图图例的标题，并自定义其内容。

**要自定义 X–Y 绘图的图例标题：**

1. 找到 **文本（Text）** 选项。选择 ****选项（Options）**![](../graphics/images/arrow.gif)**XY 选项（XY Options）**![](../graphics/images/arrow.gif)**图表图例（Chart Legend）****；或点击 ![](../graphics/ico_xyChartLegendOptions.png)，该图标位于可视化模块工具箱的 X–Y 绘图工具中。**提示：**当显示 X–Y 绘图时，您还可以通过双击视口中的图例直接打开此对话框。当活动过程生效时，您无法直接从视口中选择 X–Y 绘图中的组件；在从 X–Y 绘图中选择项目之前，请取消当前过程。图例的 **文本（Text）** 选项出现在 **内容（Contents）** 页面的下半部分。
2. 从 **图表（Charts）** 字段中，选择一个或多个要自定义其图例的 X–Y 绘图。**注意：**要使 X–Y 曲线可供选择，您必须首先绘制它。
3. 如果 **内容（Contents）** 选项卡尚未选择，请点击它。
4. 如果需要，在 **标题（Title）** 字段中输入 X–Y 绘图图例的标题。
5. 选择 X–Y 绘图图例标题的字体。1. 点击 ![](../graphics/ico_text16.png)。Abaqus/CAE 显示 **选择字体（Select Font）** 对话框。2. 使用 **选择字体（Select Font）** 对话框中的方法选择新字体、大小和样式。有关更多信息，请参阅 ["自定义字体，" 第 3.2.8 节"](pt01ch03s02s08.md)。3. 点击 **应用（Apply）** 查看字体选择的效果。4. 点击 **确定（OK）** 关闭 **选择字体（Select Font）** 对话框。X–Y 绘图图例标题变为所选的字体、大小和样式。
6. 选择 X–Y 绘图图例标题的颜色。1. 点击颜色样本 ![](../graphics/usi-colorsample.png)。Abaqus/CAE 显示 **选择颜色（Select Color）** 对话框。2. 使用 **选择颜色（Select Color）** 对话框中的方法选择新颜色。有关更多信息，请参阅 ["自定义颜色，" 第 3.2.9 节"](pt01ch03s02s09.md)。3. 点击 **确定（OK）** 关闭 **选择颜色（Select Color）** 对话框。X–Y 绘图图例标题文本变为所选颜色。
7. 点击 **关闭（Dismiss）** 关闭 **图表图例选项（Chart Legend Options）** 对话框。

![](../graphics/images/black4rule.gif)有关相关信息，请点击以下项目：- ["选择要自定义的一条或多条 X–Y 曲线，" 第 47.6.2 节"](pt05ch47s05s01.md)





# 47.9.3 向 X–Y 绘图图例添加最小值和最大值





### 47.9.3 向 X–Y 绘图图例添加最小值和最大值

您可以在绘图图例中包含绘图中 X–Y 最小值和最大值的报告，并且可以自定义这些值的格式和精度。

**要在 X–Y 绘图图例中显示最小值和最大值：**

1. 找到 **图表图例选项（Chart Legend Options）**。选择 ****选项（Options）**![](../graphics/images/arrow.gif)**XY 选项（XY Options）**![](../graphics/images/arrow.gif)**图表图例（Chart Legend）****；或点击 ![](../graphics/ico_xyChartLegendOptions.png)，该图标位于可视化模块工具箱的 X–Y 绘图工具中。**提示：**当显示 X–Y 绘图时，您还可以通过双击视口中的图例直接打开此对话框。当活动过程生效时，您无法直接从视口中选择 X–Y 绘图中的组件；在从 X–Y 绘图中选择项目之前，请取消当前过程。
2. 从 **图表（Charts）** 字段中，选择一个或多个要自定义其图例的 X–Y 绘图。
3. 如果 **内容（Contents）** 选项卡尚未选择，请点击它。
4. 切换 **显示最小/最大（Show min/max）** 以在图例中报告 X–Y 最小值和最大值。
5. 为图例中显示的最小值和最大值选择以下数字格式之一：**自动（Automatic）** 非常大的值和非常小的值以科学计数法表示。所有其他值均以无指数形式表示，并使用指定的有效数字位数。**十进制（Decimal）** 所有值均以无指数形式表示，并使用指定的有效数字位数。**工程（Engineering）** 大于或等于 1000（或小于或等于 0.001）的值表示为 1 到 999 之间的数字乘以 10 的 n 次方，其中 n 是 3 的倍数（例如，`20.5E+03` 或 `17.76E+06`）。所有其他值使用指定的有效数字位数表示。**科学（Scientific）** 所有值表示为 1 到 10 之间的数字乘以 10 的适当次方（例如，`2.05E+04` 或 `1.776E+07`）。
6. 点击 **精度（Precision）** 箭头指定每个标签中有效数字（非十进制格式）或小数位（十进制格式）的所需数量。
7. 点击 **关闭（Dismiss）** 关闭 **图表图例选项（Chart Legend Options）** 对话框。

# 47.9.4 自定义图例的位置





您可以通过在视口中拖动来移动 X–Y 绘图图例，或者通过 **图表图例选项（Chart Legend Options）** 对话框中的 **位置（Position）** 选项指定视口位置。只有当 Abaqus/CAE 没有活动过程时，您才能拖动图例；如果您无法拖动图例或从视口中选择任何其他 X–Y 绘图组件，请取消当前过程。

**要使用图表图例选项对话框自定义 X–Y 绘图图例的位置：**

1. 找到 **图表图例选项（Chart Legend Options）**。选择 ****选项（Options）**![](../graphics/images/arrow.gif)**XY 选项（XY Options）**![](../graphics/images/arrow.gif)**图表图例（Chart Legend）****；或点击 ![](../graphics/ico_xyChartLegendOptions.png)，该图标位于可视化模块工具箱的 X–Y 绘图工具中。**提示：**当显示 X–Y 绘图时，您还可以通过双击视口中的图例直接打开此对话框。当活动过程生效时，您无法直接从视口中选择 X–Y 绘图中的组件；在从 X–Y 绘图中选择项目之前，请取消当前过程。
2. 从 **图表（Charts）** 字段中，选择一个或多个要移动其图例的 X–Y 绘图。
3. 如果 **区域（Area）** 选项卡尚未选择，请点击它。
4. 切换 **嵌入（Inset）** 以在 X–Y 绘图网格内显示 X–Y 绘图图例。如果 **嵌入（Inset）** 关闭，Abaqus/CAE 会将图例和 X–Y 绘图保持在视口的独立区域中。
5. 选择以下定位选项之一：**自动对齐（Auto-align）** 此选项使您可以将图例对齐到视口中九个位置之一：其中心、四个角中的任意一个，或沿四个边缘中的任意一个居中。展开 **对齐（Alignment）** 列表，然后选择列表中以图形方式描绘的选项之一。**手动（Manual）** 此选项使您可以将图例定位在视口中的特定 X–Y 位置。拖动 **X** 和 **Y** 滑块可以在视口中水平或垂直移动图例。当这些滑块的值设置为 `0` 时，图例放置在视口的左下角。
6. 点击 **关闭（Dismiss）** 关闭 **图表图例选项（Chart Legend Options）** 对话框。

### 47.9.5 为 X–Y 绘图图例自定义边框和填充

使用 **图表图例选项（Chart Legend Options）** 对话框中的 **边框（Bounding Box）** 选项通过用边框包裹图例或用指定颜色填充来自定义图例的外观。

2. 从 **图表（Charts）** 字段中，选择一个或多个要自定义其图例的 X–Y 绘图。**注意：**要使 X–Y 曲线可供选择，您必须首先绘制它。

4. 执行以下步骤来显示和修改边框：1. 切换 **显示边框（Show border）** 以在图例周围显示边框。2. 为边框线选择颜色。1. 点击颜色样本 ![](../graphics/usi-colorsample.png)。Abaqus/CAE 显示 **选择颜色（Select Color）** 对话框。2. 使用 **选择颜色（Select Color）** 对话框中的方法选择新颜色。有关更多信息，请参阅 ["自定义颜色，" 第 3.2.9 节"](pt01ch03s02s09.md)。3. 点击 **确定（OK）** 关闭 **选择颜色（Select Color）** 对话框。边框线变为所选颜色。
5. 控制边框填充颜色：1. 切换 **填充（Fill）** 以在边框中显示背景颜色。2. 选择要在边框背景中显示的填充颜色。1. 点击颜色样本 ![](../graphics/usi-colorsample.png)。Abaqus/CAE 显示 **选择颜色（Select Color）** 对话框。2. 使用 **选择颜色（Select Color）** 对话框中的方法选择新颜色。有关更多信息，请参阅 ["自定义颜色，" 第 3.2.9 节"](pt01ch03s02s09.md)。3. 点击 **确定（OK）** 关闭 **选择颜色（Select Color）** 对话框。边框背景变为所选颜色。

# 47.10 为 X–Y 绘图自定义边框和填充颜色





### 47.10 为 X–Y 绘图自定义边框和填充颜色

使用 **绘图选项（Plot Options）** 对话框中的选项为您会话中的一个或多个 X–Y 绘图创建自定义边框颜色和自定义背景填充颜色。默认情况下，X–Y 绘图没有边框，它们用浅灰色背景颜色填充。如果绘图中使用的 X–Y 曲线在默认背景下不够突出，您可能需要调整边框颜色以区分叠加绘图中使用的不同 X–Y 绘图。

**要为 X–Y 绘图自定义边框和填充颜色：**

1. 选择 ****选项（Options）**![](../graphics/images/arrow.gif)**XY 选项（XY Options）**![](../graphics/images/arrow.gif)**绘图选项（Plot Options）****；或点击 ![](../graphics/ico_xyPlotOptions.png)，该图标位于可视化模块工具箱的 X–Y 绘图工具中。**绘图选项（Plot Options）** 对话框打开。
2. 从 **绘图（Plots）** 框架中，高亮显示要修改的一个或多个 X–Y 绘图。此框架列出绘图名称及其显示的视口。
3. 切换 **显示边框（Show border）** 以在 X–Y 绘图周围显示边框。
4. 如果需要，通过执行以下步骤更改 X–Y 绘图边框的颜色：1. 点击 **显示边框（Show border）** 标签右侧的颜色样本 ![](../graphics/usi-colorsample.png)。Abaqus/CAE 显示 **选择颜色（Select Color）** 对话框。2. 使用 **选择颜色（Select Color）** 对话框中的方法选择新颜色。有关更多信息，请参阅 ["自定义颜色，" 第 3.2.9 节"](pt01ch03s02s09.md)。3. 点击 **确定（OK）** 关闭 **选择颜色（Select Color）** 对话框。颜色样本和 X–Y 绘图边框变为所选颜色。
5. 切换 **填充（Fill）** 以用 **填充（Fill）** 标签右侧显示的颜色填充 X–Y 绘图。
6. 如果需要，通过执行以下步骤更改 X–Y 绘图填充颜色：1. 点击 **填充（Fill）** 标签右侧的颜色样本 ![](../graphics/usi-colorsample.png)。Abaqus/CAE 显示 **选择颜色（Select Color）** 对话框。2. 使用 **选择颜色（Select Color）** 对话框中的方法选择新颜色。有关更多信息，请参阅 ["自定义颜色，" 第 3.2.9 节"](pt01ch03s02s09.md)。3. 点击 **确定（OK）** 关闭 **选择颜色（Select Color）** 对话框。X–Y 绘图填充颜色变为所选颜色。
