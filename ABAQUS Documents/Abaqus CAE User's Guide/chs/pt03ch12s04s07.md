# 12.4.7 评估超弹性和粘弹性材料行为





Abaqus/CAE提供了一个方便的**Evaluate**选项，允许您查看超弹性或粘弹性材料预测的行为，并允许您选择合适的材料公式。您可以评估任何超弹性材料，但只有当粘弹性材料在时域中定义并包含超弹性和/或弹性材料数据时，才能对其进行评估和查看。如果您的材料定义包含在频域中定义的粘弹性数据，则无法在Abaqus/CAE中评估其粘弹性材料行为，但其材料评估数据会被写入数据（`.dat`）文件。**Evaluate**选项提示Abaqus/CAE使用现有材料执行一项或多项标准测试。（有关超弹性和粘弹性材料的标准测试的信息，请分别参见[Abaqus Analysis User's Guide的第22.5节，"超弹性"](../usb/usb-link.md#usbhyperelast)和[Abaqus Analysis User's Guide的第22.7节，"线性粘弹性"](../usb/usb-link.md#usbviscoelast)。）标准测试完成后，Abaqus/CAE进入Visualization模块，并将测试结果显示为新的视口中的*X–Y*图。（有关*X–Y*图的更多信息，请参见[第47章，"X--Y绘图"](pt05ch47.md)。"）Abaqus/CAE还会显示一个信息对话框，其中包含每个超弹性应变能势的稳定性限值和系数，以及粘弹性响应的粘弹性材料参数。评估的信息被保存在`*material_name_i*.dat`文件中，其中*i*从`1`开始，并对同一材料的后续评估递增。您可以查看评估结果并根据需要调整材料定义。

要启动评估过程，请从主菜单栏选择****Material**![](../graphics/images/arrow.gif)**Evaluate**![](../graphics/images/arrow.gif)***material name*****。或者，您可以在**Material Manager**中选择感兴趣的材料，然后点击**Evaluate**。出现**Evaluate Material**对话框，您可以在其中指定希望Abaqus/CAE如何执行标准测试。有关评估超弹性材料行为的详细说明，请参见["显示超弹性材料行为的*X--Y*图，"第12.7.7节](pt03ch12s07hlb07.md)。有关评估粘弹性材料行为的详细说明，请参见["显示粘弹性材料行为的*X--Y*图，"第12.7.8节](pt03ch12s07hlb08.md)。

**注意：**材料评估过程生成的作业名称与材料名称相同；因此，这些材料名称必须遵守与作业名称相同的规则（有关命名对象的更多信息，请参见["使用基本对话框组件，"第3.2.1节](pt01ch03s02s01.md)）。

**Evaluate**选项在以下场景中特别有用：

**将测试数据与特定应变能势预测的行为进行比较**

当您使用实验数据定义超弹性材料时，您还需要指定要应用于数据的应变能势。Abaqus使用实验数据来计算指定应变能势所需的系数。但是，验证材料定义预测的行为与实验数据之间是否存在可接受的相关性非常重要。

您可以使用**Evaluate**选项基于您在材料定义中指定的应变能势，使用实验数据计算材料响应。测试完成后，Abaqus/CAE进入Visualization模块并显示测试结果的*X–Y*图。每个图都包含实验数据和每个评估的应变能势的曲线。Abaqus/CAE还会打开一个对话框，其中包含每个应变能势的稳定性限值和系数。

例如，[图12-13](pt03ch12s04s07.md#prp-evaluate)中的*X–Y*图显示了使用Ogden N=3应变能势的平面测试结果。

此外，以下信息被报告给数据（`.dat`）文件：
- 为应变能势计算的系数。
- 测试期间检测到的任何材料不稳定性。

一旦分析成功完成，数据（`.dat`）文件的路径会出现在Abaqus/CAE主窗口的消息区域中。

**评估多个应变能势**

如果您正在使用实验数据定义超弹性材料，而不确定要指定哪个应变能势，则可以从材料编辑器中的**Strain energy potential**列表中选择**Unknown**。然后，您可以使用**Evaluate**选项使用多个应变能势对实验数据执行标准测试。

测试完成后，Abaqus/CAE进入Visualization模块，并为每个测试显示一个*X–Y*图，以及一个包含每个应变能势稳定性限值和系数的对话框。每个图都包含实验数据和每个评估的应变能势的曲线。您可以直观地比较应变能势曲线和实验数据曲线，并选择提供最佳拟合的应变能势。

一旦确定了哪个应变能势提供与实验数据的最佳拟合，您必须返回Property模块中的材料编辑器，并将**Strain energy potential**选择从**Unknown**更改为您选择的应变能势。

**查看系数预测的行为**

如果您已经获得了特定应变能势的系数（无论是通过评估一个或多个超弹性应变能势（如上所述），还是从其他来源获得），您可能希望验证应变能势预测的行为是否可接受地与您的实验数据匹配或满足其他标准。

您可以使用**Evaluate**选项使用您在材料定义中提供的系数绘制应变能势曲线图。如果材料定义也包含实验数据，则该数据的曲线也会出现在图中。

**查看粘弹性材料的响应曲线**

如果您有剪切或体积测试结果，您可能希望验证Abaqus预测的蠕变和松弛行为是否可以接受地与您的实验数据匹配或满足其他标准。同样，如果您有频率数据，您可能希望验证预测的剪切和体积模量的存储和损耗分量是否与您的数据匹配。

您可以使用**Evaluate**选项使用您在材料定义中提供的系数绘制曲线图。如果材料定义包含实验数据，则这些数据的曲线也会出现在图中。生成的曲线类型取决于材料定义。对于使用Prony级数定义的粘弹性材料，用于时间的蠕变数据或松弛测试数据，您可以产生蠕变和松弛图与时间的关系。对于使用频率数据定义的粘弹性材料，您可以产生剪切和体积模量的存储和损耗分量与频率对数刻度的图。

**调整材料数据**

如果您对测试数据与材料预测的行为之间的拟合不满意，可以返回Property模块并调整测试数据，然后重新评估材料。您可以重复此过程，直到对材料行为满意为止。在某些情况下，可以使用这种方法来优化超弹性材料定义中包含的系数值。有关更多信息，请参见Abaqus Analysis User's Guide的第22.5.1节"橡胶类材料的超弹性行为"中的"提高测试数据拟合的准确性和稳定性"。
