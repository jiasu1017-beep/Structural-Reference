# 19.12.7 提取平滑网格







当优化过程完成后，您可以选择以可以传输到 CAD 系统或返回 Abaqus/CAE 的文件形式提取优化模型表面的平滑等值面网格表示。您可以从拓扑优化或形状优化中提取平滑表面网格。

优化模型通常是四面体和六面体单元的组合，可能包含大量不包括在将生成的等值面表示中的内部元素。此外，模型表面可能粗糙且不规则，这取决于单元密度。平滑操作是一个迭代过程，置换表面上的节点并在模型表面上生成平滑的三角形网格。

平滑操作之后可以是数据缩减操作，该操作合并具有几乎共面得相邻三角形，并减少表面三角形的数量。数据缩减继续迭代，直到减少的三角形数量达到原始三角形数量的指定百分比。您可以指定确定相邻平面是否共面的角度。

最后，您可以选择应用可选的滤波操作以去除局部不规则性。滤波在平滑操作之前应用。

**提取平滑表面网格：**

1. 从主菜单栏中，选择****优化**![](../graphics/images/arrow.gif)**提取**![](../graphics/images/arrow.gif)***优化过程名称*****。您也可以通过在**优化过程管理器**中选择优化过程名称，然后点击**提取**来平滑结果。出现**提取表面网格**对话框。
2. 输入将创建的输出文件的名称。
3. 选择将生成的文件的格式。您可以选择以下之一：- 可导入 Abaqus/CAE 的 Abaqus 输入文件（`.inp`）。- 可被 3D CAD 系统（如 CATIA V5 和 SolidWorks）读取的 STL 文件（`.stl`）。
4. 在**设计循环**字段中，输入应提取等值面的设计循环。默认情况下，**编辑平滑选项**对话框显示最后一个设计循环。
5. 输入介于零和一之间的值指定**等值**。平滑过程确定哪些单元在模型表面上，并使用等值来计算在单元内部边缘上创建新节点的位置。增加等值会导致等值面向模型内部移动，从而导致模型体积减少。对于具有薄组件的结构，您不应输入大于 `0.7` 的等值，以防止结构断开。
6. 输入**缩减百分比**，定义应在数据缩减操作期间移除的面百分比。值为 0 表示不应移除任何面。值为 100 表示给定缩减角度值，数据缩减在无法再移除面时停止。
7. 输入介于 0 和 90 之间的值指定**缩减角度**。缩减角度定义节点处相邻面之间的最大角度，以便在数据缩减期间可以移除该节点。
8. 选择迭代平滑循环的次数。较大的值会导致更平滑的模型，但可能导致计算时间增加。实际上，通常 5 到 10 个平滑循环就足够了。进一步的平滑可能导致薄组件变窄或收缩。您可以通过选择值 `0` 来关闭平滑。
9. 输入介于零和一之间的值指定**目标体积**。实体模型的相对目标体积是等值面定义的体积与原始体积的比率（根据优化生成的实体单元的体积计算）。同样，壳模型的目标面积是等值面定义的面积与原始面积的比率（根据优化生成的壳单元的面积计算）。如果模型同时包含实体和壳单元，则仅考虑实体单元。计算体积时不考虑实体模型内部的空隙。如果您指定目标体积，数据缩减过程会计算将产生指定体积的等值值（您输入的等值被忽略）。如果您为目标体积输入的值过大（大于等值为零时计算的体积）或过小（小于等值为一计算的体积），数据缩减过程会产生错误。如果在 20 次迭代之前收敛失败，数据缩减过程也会产生错误。
10. 在某些情况下，优化模型将包含您可以选择移除的小不规则性。滤波是可选的，在平滑操作之前应用，结果是更均匀的材料分布。选择**中等**以应用单个滤波循环，选择**完全**以应用五个滤波循环。
11. 点击**确定**以提取平滑网格。指定的输出文件（`.inp` 或 `.stl`）在 TOSCA_POST 目录中创建。对于后续运行，文件名称会附加唯一编号；例如，`*name*_002.inp`。**注意：**如果您以 Abaqus 输入文件的形式提取了平滑网格，您可以从主菜单栏中选择****文件**![](../graphics/images/arrow.gif)**导入**![](../graphics/images/arrow.gif)**模型****将网格导入 Abaqus/CAE。有关更多信息，请参见["导入模型，"第 10.
