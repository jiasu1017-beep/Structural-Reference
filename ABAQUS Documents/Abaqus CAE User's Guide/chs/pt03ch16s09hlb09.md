# 16.9.9 定义重力载荷





您可以创建重力载荷以在固定方向上定义均匀加速度。Abaqus使用您在重力载荷定义中输入的加速度幅值以及材料定义中指定的密度来计算载荷。

**创建或编辑重力载荷：**

1. 使用以下方法之一显示重力载荷编辑器：- 若要创建新的重力载荷，请遵循"创建载荷"，第16.8.1节中概述的过程（**类别**：`Mechanical`；**所选步骤的类型**：`Gravity`）。- 若要使用菜单或管理器编辑现有的重力载荷，请参见"编辑依赖于步骤的对象"，第3.4.13节。
2. 默认情况下，重力载荷应用于整个模型。如果需要，您可以将重力载荷应用于模型的特定区域：1. 单击![](../graphics/ico_selectBlue.png)。2. 如"创建载荷"，第16.8.1节中所述，选择要应用载荷的区域。3. 在提示区域中单击**完成**。**注意：**您不能将重力载荷应用于单个点质量。要将点质量包含在重力载荷中，请将载荷应用于整个模型。
3. 如果可用，单击**分布**字段右侧的箭头，并从出现的列表中选择您的选项：- 选择**均匀**以定义在区域上均匀的载荷。- 选择分析场以定义空间变化载荷。只有对该载荷类型有效的分析场才会在选择列表中显示。或者，您可以单击![](../graphics/ico_analyticalField.png)创建新的分析场。（更多信息，请参见第58章"分析场工具集"。）
4. 在**分量1**、**分量2**和（如果您在三维空间模型中工作）**分量3**文本字段中，输入每个方向的加速度分量：- 如果您在三维或二维空间工作，**分量1**、**分量2**和**分量3**字段对应于1、2和（如果适用）3方向。- 如果您在轴对称空间工作，只有**分量2**文本字段可用。**分量2**对应于轴向方向。如果您留空文本字段，则自动为该方向分配零值。但是，您必须在编辑器中至少输入一个非零分量来定义载荷。
5. 如果需要，单击**幅值**字段右侧的箭头，并从出现的列表中选择您选择的幅值。或者，您可以单击![](../graphics/ico_amplitudeSmall.png)创建新的幅值。（更多信息，请参见第57章"幅值工具集"。）
6. 单击**确定**保存您的数据并退出编辑器。
