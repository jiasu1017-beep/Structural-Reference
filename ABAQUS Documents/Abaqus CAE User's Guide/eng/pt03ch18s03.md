# 18.3 Understanding optimization







An optimization is an iterative process that modifies the structure of your model while it searches for an optimized solution given a set of objectives and constraints that must be satisfied. This section briefly describes the components of an optimization that you can create with the Optimization Module. For more detailed information, see ["Structural optimization: overview," Section 13.1.1 of the Abaqus Analysis User's Guide](../usb/usb-link.md#usb-anl-aoptover).
