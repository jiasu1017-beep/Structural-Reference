# AFXToolboxGroup





此类创建一个用作工具箱中组的容器。它将使用实用方法，以便正确管理模块和工具集的组。
![](../graphics/gui-afxtoolboxgroup.png)

### AFXToolboxGroup(owner, parent=None)

构造函数。
| **参数** | **类型** | **默认值** | **描述** |
| --- | --- | --- | --- |
| owner | AFXGuiObjectManager |  | 工具箱组的创建者。 |
| parent | FXComposite | None | 父 widget。 |

### getOwner()

返回工具箱组的所有者。

从 FXWindow 重新实现。
