# 16.1 理解载荷模块的作用







Abaqus/CAE中的规定条件是依赖于步骤的对象，这意味着您必须指定它们处于活动状态的分析步骤。您可以使用载荷、边界条件和预定义场管理器来查看和操作规定条件的逐步历史。您还可以使用上下文栏中的**步骤**列表来指定新载荷、边界条件和预定义场默认处于活动状态的步骤。

您可以使用载荷模块中的幅值工具集来指定可以应用于规定条件的复杂时间或频率依赖关系。载荷模块中的集合和表面工具集允许您定义和命名要应用规定条件的模型区域。解析场工具集和离散场工具集允许您创建可用于定义选定规定条件空间变化参数的场。

载荷工况是用于定义特定载荷条件的载荷和边界条件集。您可以在静态扰动和稳态动态、直接步骤中创建载荷工况。有关载荷工况的信息，请参见第34章"载荷工况"。
