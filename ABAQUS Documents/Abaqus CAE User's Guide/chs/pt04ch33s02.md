# 33.2 管理惯性





**惯性管理器**允许您创建和管理惯性定义。该管理器包含您已定义的惯性的名称和类型列表。管理器中的**创建**、**编辑**、**复制**、**重命名**和**删除**按钮允许您创建新的惯性定义或编辑、复制、重命名和删除现有惯性定义。管理器左侧列中的图标允许您抑制和恢复现有惯性定义。您还可以使用属性模块或相互作用模块主菜单栏中的****特殊**![](../graphics/images/arrow.gif)**惯性****菜单启动这些过程。从主菜单栏选择管理操作后，过程与单击管理器对话框内相应按钮完全相同。
