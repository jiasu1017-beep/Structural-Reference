# 1.4 Abaqus GUI Toolkit用户指南的组织





本指南按功能组织，旨在指导开发人员完成编写应用程序的过程，解释如何使用工具包的组件并提供示例代码片段。提供了单独的[Abaqus GUI Toolkit参考指南](../gui/gui-link.md#gui)，其中包含所有工具包调用的字母顺序列表。

Abaqus GUI Toolkit基于FOX GUI工具包。虽然本指南解释了FOX工具包的一些基本概念，但并没有为FOX工具包的许多其他方面提供详细信息。有关FOX GUI工具包的更多详细信息，请参阅FOX[网站](http://www.fox-toolkit.org)。

本指南包括以下部分：

**Widget**

本节描述了Abaqus GUI Toolkit中最常用的widget。

**布局管理器**

本节描述了如何使用Abaqus GUI Toolkit中的各种布局管理器在对话框中排列widget。

**对话框**

本节描述了您可以使用Abaqus GUI Toolkit创建的对框。

**命令**

在使用图形用户界面的应用程序中，界面必须收集用户的输入，然后将输入传递给应用程序。此外，图形用户界面必须根据应用程序的状态保持其最新状态。本节描述了如何使用Abaqus GUI Toolkit完成这些任务，以及Abaqus/CAE中的两种命令类型——kernel命令和GUI命令。

**模式**

模式是一种收集用户输入、处理输入然后向kernel发出命令的机制。本节描述了Abaqus GUI Toolkit中可用的模式。

**创建GUI模块**

本节描述了如何创建GUI模块。

**创建GUI工具集**

本节描述了如何创建GUI工具集。

**自定义现有模块或工具集**

前面的章节描述了如何创建新模块或工具集。或者，Abaqus GUI Toolkit允许您从现有模块或工具集派生新模块或工具集，并从中添加或删除功能。

**创建应用程序**

本节解释如何创建应用程序，例如Abaqus/CAE。它还描述了负责运行应用程序的高级基础设施。

**应用程序对象**

本节描述了Abaqus应用程序对象。应用程序对象管理消息队列、计时器、琐事、GUI更新和其他系统设施。

**主窗口**

本节描述了Abaqus主窗口的布局、组件和行为。

**自定义主窗口**

主窗口基类提供GUI基础设施，允许用户交互、模块操作和在视口中显示对象。本节描述了如何通过从主窗口基类派生然后注册模块和工具集来向应用程序添加功能。
