# 19.7.3 对模型执行数据检查







数据检查将输入文件通过分析输入文件处理器运行，以确保模型的一致性并设置所有必需的模型选项。如果作业源是 Abaqus/CAE 模型，则此选项会生成（或重新生成）该作业的输入文件。

要对模型执行数据检查，请从主菜单栏中选择****作业**![](../graphics/images/arrow.gif)**数据检查**![](../graphics/images/arrow.gif)***您选择的作业*****。您也可以通过选择您选择的作业，然后在**作业管理器**中点击**数据检查**来执行数据检查。

要查看数据检查分析的结果，请在您的工作目录中查看数据（`.dat`）文件。您还可以通过选择****作业**![](../graphics/images/arrow.gif)**监视器**![](../graphics/images/arrow.gif)***您选择的作业*****来查看数据检查的结果，以显示作业监视器对话框。有关作业监视器的更多信息，请参见["监视分析作业的进度，"第 19.2.6 节](pt03ch19s02hlb01.md)。

数据检查还会创建输出数据库。您可以在可视化模块中查看输出数据库，以验证模型信息，如面标签、表面法向和材料方向。有关查看输出数据库的更多信息，请参见[第五部分，"查看结果](pt05.md)。"
