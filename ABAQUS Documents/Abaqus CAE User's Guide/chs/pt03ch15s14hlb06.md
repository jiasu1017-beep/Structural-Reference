# 15.14.8 定义执行器/传感器相互作用属性







执行器/传感器相互作用属性提供传递到与执行器/传感器相互作用一起使用的[`UEL`](../sub/sub-link.md#sub-xsl-uel)用户子程序中的`PROPS`、`JPROPS`、`NPROPS`和`NJPROPS`变量。更多信息，请参见第15.13.26节"定义执行器/传感器相互作用"。

**定义执行器/传感器相互作用属性：**

1. 从主菜单栏中，选择****相互作用**![](../graphics/images/arrow.gif)**属性**![](../graphics/images/arrow.gif)**创建****。
2. 在出现的**创建相互作用属性**对话框中，执行以下操作：- 命名该相互作用属性。有关命名对象的信息，请参见第3.2.1节"使用基本对话框组件"。- 选择**执行器/传感器**类型的相互作用属性。
3. 单击**继续**关闭**创建相互作用属性**对话框。
4. 单击**实属性**选项卡，然后输入要传递到[`UEL`](../sub/sub-link.md#sub-xsl-uel)用户子程序的实属性值以提供`PROPS`和`NPROPS`变量。
5. 单击**整数属性**选项卡，然后输入要传递到[`UEL`](../sub/sub-link.md#sub-xsl-uel)用户子程序的整数属性值以提供`JPROPS`和`NJPROPS`变量。
6. 单击**确定**创建执行器/传感器相互作用属性并退出编辑器。
