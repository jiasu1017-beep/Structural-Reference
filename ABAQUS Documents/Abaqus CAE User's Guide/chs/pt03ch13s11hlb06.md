# 13.11.6 用共轴面约束两个实例





从主菜单栏选择**约束**![](../graphics/images/arrow.gif)**共轴**来施加约束，将可动实例定位，使所选面的旋转轴与固定实例所选面的旋转轴重合。所有位置约束都是装配的特征，可以使用特征操作工具集来抑制或删除。

可动和固定实例所选的面必须是圆柱形或圆锥形。此外，共轴约束只能应用于三维实例。更多信息，请参见["位置约束方法有何不同，"第13.5.2节](pt03ch13s05s02.md)。

**要用共轴面约束两个零件或模型实例：**

1. 从主菜单，选择**约束**![](../graphics/images/arrow.gif)**共轴**。**提示：**您也可以使用装配模块工具箱中的![](../graphics/ico_constraintCoaxial.png)工具来施加共轴约束。有关装配工具箱中工具的图表，请参见["使用装配模块工具箱，"第13.9节](pt03ch13s09.md)。Abaqus/CAE在提示区域显示提示来引导您完成过程。
2. 从将移动的实例和将保持固定的实例中选择圆柱形或圆锥形面，如下图所示：![](../graphics/asm-help-coaxial-select-nls.png)Abaqus/CAE沿所选面的旋转轴显示箭头。
3. 从提示区域中的按钮，执行以下操作之一：
   - 点击**确定**接受可动实例面旋转轴上箭头的方向。
   - 点击**翻转**反转可动实例面旋转轴上箭头的方向，然后点击**确定**。Abaqus/CAE将可动实例定位，使两个轴重合且箭头指向相同方向。固定实例的方向保持不变。当箭头选择如上所示时，共轴约束的效果如下图所示：![](../graphics/asm-help-coaxial.png)
   如果共轴约束与现有约束冲突，Abaqus/CAE显示错误消息并取消操作。要避免冲突，您可以尝试反转将移动的实例和将保持固定的实例的选择。或者，您可以先将现有约束转换为绝对位置，然后重新施加共轴约束。
