# 1.9.2 阻尼弹簧-质量系统的正弦激励







**产品：**Abaqus/Standard  Abaqus/Explicit  

### 单元测试

CONN2D2    CONN3D2

### 问题描述

平移和旋转基本连接器组件以及平移-旋转组合连接在阻尼弹簧-质量连接器系统的正弦激励下进行测试。一系列模拟阻尼弹簧-质量系统的简单2节点连接器单元验证了表中描述的连接。连接器单元中的动力行为使用连接器弹性、连接器阻尼和连接器摩擦行为进行建模。

在Abaqus/Standard中，使用小或大位移公式进行直接积分隐式动态步骤。在Abaqus/Explicit中，使用默认的大位移公式进行通常的显式动态过程。在每种情况下，通过分配连接器单元载荷来驱动弹簧-质量系统。
