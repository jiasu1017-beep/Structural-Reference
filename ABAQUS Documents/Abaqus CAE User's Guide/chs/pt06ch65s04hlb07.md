# 65.4.7 调整再生性能







**Feature Options**（特征选项）对话框提供了使您能够控制 Abaqus/CAE 是否执行自相交检查的设置，并在 Assembly（装配）模块中控制 Abaqus/CAE 相对于其他特征再生位置约束的顺序。调整这些选项可以帮助您加快部件或装配的特征再生。从主菜单栏中选择 ****Feature**![](../graphics/images/arrow.gif)**Options****（特征→选项）以显示 **Feature Options**（特征选项）对话框。您还可以通过在模型树中的特征上点击鼠标按钮 3 并从出现的菜单中选择 **Options**（选项）来调整再生。

在大多数情况下，默认设置将产生可接受的再生性能。只有在 Abaqus/CAE 再生部件或装配花费很长时间时，您才应该修改特征选项。有关更多信息，请参阅["调整特征再生，" 第 65.3 节"](pt06ch65s03.md)。

**注意：**Abaqus/CAE 在 **Options**（选项）对话框的 **Memory**（内存）选项卡页面上提供对其他性能调整选项的访问。此页面上的选项使您能够指定内核内存限制，设置 Abaqus/CAE 以减少内存模式运行的阈值，并在内存缓存中控制几何状态。有关更多信息，请参阅["自定义内存限制和再生选项，" 第 68.1 节"](pt06ch68s01.md)。

**调整再生性能：**

1. 从主菜单栏中选择 ****Feature**![](../graphics/images/arrow.gif)**Options****（特征→选项）。Abaqus/CAE 显示 **Feature Options**（特征选项）对话框。
2. 如果您知道您的特征不会自相交，或者您计划在后续操作中移除自相交，您可以关闭 **Perform self-intersection checks**（执行自相交检查）以加快特征再生。有关更多信息，请参阅["什么是自相交检查？" 第 65.3.4 节"](pt06ch65s03s04.md)。
3. 如果需要，修改决定 Abaqus/CAE 如何再生位置约束的行为。有关更多信息，请参阅["位置约束如何再生？" 第 65.3.5 节"](pt06ch65s03s05.md)。
4. 点击 **OK**（确定）更改当前模型的特征选项。特征选项仅适用于当前模型，并保存在模型数据库中。如果需要，您还可以更改 **Options**（选项）对话框的 **Memory**（内存）选项卡页面中的缓存设置，以更改 Abaqus/CAE 在内存中自动缓存的几何状态数量；清除与部件和装配缓存关联的内存；或缓存部件的当前状态。有关更多信息，请参阅["自定义内存限制和再生选项，" 第 68.1 节"](pt06ch68s01.md)。
