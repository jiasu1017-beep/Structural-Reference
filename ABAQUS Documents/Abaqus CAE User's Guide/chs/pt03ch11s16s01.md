# 11.16.1 在零件模块中使用基准工具集





基准可以被认为是参考几何或构造辅助工具，当零件不包含必要的几何时，可以帮助您创建特征；您使用基准工具集创建基准几何。基准是零件的特征，与零件的其余部分一起重新生成。此外，基准几何是可见的，除非通过从主菜单栏选择**视图**![](../graphics/images/arrow.gif)**零件显示选项**![](../graphics/images/arrow.gif)**基准**将其关闭。在零件模块中创建的基准会出现在装配模块或任何其他基于装配的模块中该零件的每个实例上。

基准点被投影到草图编辑器中的草图平面上，投影点可以选择。但是，您不能在草图编辑器中引用基准轴或基准平面。以下是在零件模块中使用基准平面和基准轴的示例。

**基准平面**

您可以直接在基准平面上绘制草图，在基准平面上绘制的任何特征都将投影到零件上。如果零件已经包含便利的草图平面，则从基准平面向下投影草图很有用。

**基准轴**

您可以使用基准工具集创建基准轴。然后可以选择基准轴来控制在向三维实体添加或修改特征时零件在草图编辑器网格上的方向。当零件不包含必要的轴时，创建基准轴很有用。
