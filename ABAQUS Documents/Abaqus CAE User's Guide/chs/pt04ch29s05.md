# 29.5 创建组装紧固件





您可以在Interaction模块中创建组装紧固件，前提是您已经开发了模板模型。更多信息，请参见["关于紧固件"，第29.1节](pt04ch29s01.md)。

**要创建组装紧固件：**

1. 从Interaction模块的主菜单栏中，选择****Special**![](../graphics/images/arrow.gif)**Fasteners**![](../graphics/images/arrow.gif)**Create****。Abaqus/CAE显示**Create Fasteners**对话框。**提示：**您也可以使用Interaction模块工具箱中的![](../graphics/ico_fastener.png)工具创建组装紧固件。
2. 在出现的**Create Fasteners**对话框中，选择**Assembled**紧固件，然后单击**Continue**。
3. 选择主模型中连接点（或参考点）的集合，模板模型的控制点将被映射到这些点上。使用以下方法之一：
   - 使用现有集合来定义点或多个点。此方法允许您使用预定义的连接点集轻松选择大量点。在提示区域右侧单击**Sets**。从出现的**Region Selection**对话框中选择现有的点集，然后单击**Continue**。**注意：**默认选择方法基于您最近使用的选择方法。要切换到另一种方法，请在提示区域右侧单击**Select in Viewport**或**Sets**。
   - 在视口中选择一个或多个点。**提示：**默认情况下，**Selection**工具栏中的![](../graphics/ico_filterVisible.png)**选择屏幕最近处的实体**工具处于关闭状态。如果进行模糊选择，Abaqus/CAE会高亮显示该点并在视口左下角显示该点的描述。使用**Next**和**Previous**按钮循环浏览可能的选择，然后单击**OK**确认您的选择。单击鼠标按钮2表示您已完成选择。更多信息，请参见["在当前视口中选择对象"，第6.2节](pt01ch06s02.md)。如果模型包含孤立网格实例和原生几何部件实例的组合，请从提示区域单击以下之一：
     - 如果要从原生几何部件实例或一个或多个参考点中选择点，请单击**Geometry**。
     - 如果要从孤立网格实例中选择点，请单击**Mesh**。
4. 完成选择连接点后单击**Done**。Abaqus/CAE显示**Edit Fasteners**对话框。
5. 选择要使用的模板模型。模板模型应包含您要在主模型中复制的类似紧固件的结构。模板模型必须包含在当前模型数据库（`.cae`文件）中，且模板模型必须包含一个或多个引用命名曲面的绑定、耦合或调整点约束。
6. 从模板模型中选择控制点集。此集必须包含紧固件结构的控制点。
7. 在**Surfaces**、**Orientations**和**Properties**标签页上，配置紧固件设置，如下述过程所述：
   - ["将模板模型表面映射到主模型表面"，第29.5.1节](pt04ch29s05hlb01.md)
   - ["定向从模板模型读取的几何体"，第29.5.2节](pt04ch29s05hlb02.md)
   - ["修改主模型中的紧固件属性"，第29.5.3节](pt04ch29s05hlb03.md)
8. 单击**OK**创建组装紧固件。小的浅绿色方块出现在视口中，显示组装紧固件的位置。

创建组装紧固件后，您可以使用**Re-run**标签页重新运行属性生成脚本；请参见["重新运行属性生成脚本"，第29.5.4节](pt04ch29s05hlb04.md)。
