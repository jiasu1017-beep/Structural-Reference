# 5.3 显示和隐藏对话框







对话框具有 `show` 和 `hide` 方法，用于从屏幕上显示或隐藏对话框。在大多数情况下，您不需要调用这些方法，因为模式基础设施会为您调用它们。但是，您可能希望编写自己的 `show` 和 `hide` 方法来执行一些特殊处理，这些处理将在应用程序显示或隐藏对话框之前执行。例如，您可以在 `show` 和 `hide` 方法中注册和注销查询。您必须调用 `show` 和 `hide` 方法的基类版本，否则这些方法的行为将不符合预期。例如，在对话框类代码中，您可以添加以下行：

```
def show(self):

    # Do some special processing here.
    ...

    # Call base class method.
    AFXDataDialog.show(self)

def hide(self):

    # Do some special processing here.
    ...

    # Call base class method.
    AFXDataDialog.hide(self)
```
