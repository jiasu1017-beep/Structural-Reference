# 31.2.9 创建轮廓积分裂缝





您可以使用Interaction模块中的**Special**菜单创建轮廓积分裂缝。更多信息，请参见["断裂力学"，Abaqus Analysis User's Guide第11.4节](../usb/usb-link.md#usbafrac)。

您选择的用于定义轮廓积分的实体取决于部件是二维还是三维，以及您是使用几何体还是使用孤立网格单元和节点定义部件。

**要创建轮廓积分裂缝：**

1. 从Interaction模块的主菜单栏中，选择****Special**![](../graphics/images/arrow.gif)**Crack**![](../graphics/images/arrow.gif)**Create****。
2. 从出现的**Create Crack**对话框中，选择**Contour integral**。
3. 输入裂缝名称，然后单击**Continue**关闭对话框。
4. 从视口中的模型中选择表示裂缝前缘区域的实体。有关选择的实体的描述，请参见["定义裂缝前缘"，第31.2.2节](pt04ch31s02s02.md)。
5. 单击鼠标按钮2表示您已完成裂缝前缘区域的选择。
6. 从视口中的模型中选择表示裂缝尖端区域的实体。在某些情况下，根据模型的建模空间和您为定义裂缝前缘而选择的实体，Abaqus/CAE会为您选择裂缝尖端并跳至步骤7。更多信息，请参见["定义裂缝尖端或裂缝线"，第31.2.3节](pt04ch31s02s03.md)。从提示行切换**Select mesh entities**以从孤立网格中选择实体。
7. 单击鼠标按钮2表示您已完成裂缝尖端区域的选择。
8. 从提示区域，选择定义裂缝扩展方向的方法。
**Normal to crack plane**
选择**Normal to crack plane**以指定裂缝平面的法线，然后选择表示法线起点和终点的点。
**q vectors**
选择**q vectors**以直接指定裂缝扩展方向，然后选择表示向量起点和终点的点。
更多信息，请参见["定义裂缝扩展方向"，第31.2.4节](pt04ch31s02s04.md)。Abaqus/CAE显示红色箭头指示裂缝扩展方向，并显示**Edit Crack**对话框。
9. 使用**Edit Crack**对话框配置控制轮廓积分分析的参数。
   - 单击对话框的**General**标签页以执行以下操作：
     - 指定裂缝前缘位于仅对结构一半建模的对称平面上。
     - 更改定义裂缝前缘或裂缝尖端/线的实体。
     - 更改定义裂缝扩展方向的方法。您也可以更改定义裂缝扩展方向的实体。更多信息，请参见["修改轮廓积分数据"，第31.2.10节](pt04ch31s02hlb03.md)。
   - 单击对话框的**Singularity**标签页以为裂缝尖端的应变场建模奇异性。更多信息，请参见["控制裂缝尖端的奇异性"，第31.2.8节](pt04ch31s02hlb01.md)。
10. 单击**OK**配置轮廓积分并关闭编辑器。Abaqus在区域上显示绿色十字以表示裂缝前缘。您必须使用Step模块中的**History Output Request**编辑器来请求Abaqus在分析过程中将签名距离函数PHILSM写入输出数据库。更多信息，请参见["查看XFEM裂缝"，第31.3.3节](pt04ch31s03s03.md)和["修改场输出请求"，第14.12.2节](pt03ch14s12hlb02.md)。
