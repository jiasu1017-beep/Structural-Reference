# 20.22.4 添加独立草图


在绘制时,您可以添加独立草图,使其几何覆盖当前草图。Abaqus/CAE 定位添加的独立草图,使其原点与当前草图的原点重合。

**添加独立草图的步骤:**

1. 从 Sketcher 工具箱中,选择添加草图工具 ![](../graphics/ico_sketchAdd.png)。有关 Sketcher 工具箱中工具的图示,请参见["Sketcher 工具," 第 20.4.1 节](pt03ch20s04s01.md)。Abaqus/CAE 会显示**添加草图**对话框。
2. 从独立草图列表中选择要添加的草图,然后单击**确定**关闭**添加草图**对话框。Abaqus/CAE 会: - 定位添加的草图,使其原点与当前草图的原点重合。 - 调整视图,使当前草图和添加的草图都可见。
3. 使用提示区域中出现的按钮来指示您将如何定位添加的草图。 - 选择**平移**可以沿指定向量平移添加的草图。您可以通过从草图中选择或通过输入每端点的 *X*- 和 *Y*- 坐标来定义平移向量的起点和终点。 - 选择**旋转**可以围绕指定点通过指定角度旋转添加的草图。您可以通过从草图中选择或输入 *X*- 和 *Y*- 坐标来指定旋转中心点。您指定的旋转角度必须介于 -360 和 360 度之间。 - 选择**镜像**可以围绕选定的线镜像添加的草图。您必须通过选择一条现有直线(不是添加草图中的线)来指定镜像线。 - 选择**缩放**可以通过指定量围绕指定点缩放添加的草图顶点之间的间距。您可以通过从草图中选择或输入 *X*- 和 *Y*- 坐标来指定该点。根据您的规范修改和定位添加的草图预览。
4. 如有需要,重复第 3 步继续修改或重新定位添加的草图。
5. 在提示区域选择**完成**。Abaqus/CAE 会将独立草图添加到现有几何中。
