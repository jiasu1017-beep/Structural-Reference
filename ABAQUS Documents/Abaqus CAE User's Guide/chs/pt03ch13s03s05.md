# 13.3.5 在模型之间链接零件实例







您可以在模型之间链接零件实例。链接零件实例允许在您修改原始模型中的实例或零件时自动更新实例和零件。

在模型树中，选择要链接到另一模型中零件实例的零件实例（子实例）。右键单击，选择**链接实例**，并指定您要将每个子实例链接到的父模型和零件实例。同样，您可以取消链接先前链接的零件实例。有关详细说明，请参见"使用模型树操作零件实例"，第13.10.2节。

如果您选择要链接的零件的所有实例，零件也会自动链接。零件及其特征、集合和表面使用父零件更新。装配级特征、集合和表面不被复制。实例使用父实例更新，并保留在实例上定义的集合和表面。

如果您只选择要链接的零件的部分实例，则在链接实例之前会创建一个新零件（零件名称后附加`—LinkedCopy`），并将新零件链接到父模型。

链接的零件实例和零件不可编辑。链接子实例的位置完全由父实例的位置决定，不能更新。

默认情况下，链接的零件实例和零件在视口中以灰色显示。模型树中显示图标以指示零件和零件实例的链接状态，以及如果零件实例也被排除在分析之外，则还指示零件实例的链接和排除状态，如图13-3所示。`Beam-1`是链接的零件实例，`Beam-2`是链接且被排除的零件实例。有关更多信息，请参见"从分析中排除零件实例"，第13.3.6节。
