# 64.8.1 细化平面三角形网格







选择 ****Mesh**![](../graphics/images/arrow.gif)**Edit****（网格→编辑）从主菜单栏细化平面三角形网格。您可以使用以下方法重新网格化部件：

**使用指定的全局面元大小**

在重新网格化部件之前，您可以选择为整个部件分配目标单元大小。然后您可以重新网格化部件，新网格的密度反映新的目标单元大小。例如，当[图 64-22](pt06ch64s08s01.md#prt-distorted) 中的部件用 15.0 的全局面元大小重新网格化时，[图 64-20](pt06ch64s08s01.md#prt-global15) 显示了生成的网格。

**图 64-20** 全局面元大小为 15.0。

![](../graphics/prt-global15.png)

[图 64-21](pt06ch64s08s01.md#prt-global8) 显示了用 8.0 的全局面元大小重新网格化的部件。

**图 64-21** 全局面元大小为 8.0。

![](../graphics/prt-global8.png)

**未指定全局面元大小**

如果未指定全局面元大小，Abaqus/CAE 会在改善部件内部网格质量的同时保持沿部件边界的单元边缘。生成的网格拓扑结构与原始网格拓扑结构不同。例如，[图 64-22](pt06ch64s08s01.md#prt-distorted) 显示了扭曲的网格。

**图 64-22** 扭曲的网格。

![](../graphics/prt-distorted.png)

当部件重新网格化时，网格质量显著改善，如[图 64-23](pt06ch64s08s01.md#prt-remeshed) 所示。

**图 64-23** 在未指定全局面元大小的情况下重新网格化的部件。

![](../graphics/prt-remeshed.png)

**使用指定的全局面元大小细化平面三角形网格：**

1. 进入 Mesh（网格）模块。
2. 从上下文栏的 **Object**（对象）字段中选择 **Part**（部件），然后从列表中选择仅包含孤网格单元的部件。**提示：**要细化混合部件中的孤单元，请抑制几何特征并在完成细化操作后恢复它们。
3. 从主菜单栏中选择 ****Mesh**![](../graphics/images/arrow.gif)**Edit****（网格→编辑）。Abaqus/CAE 显示 **Edit Mesh**（编辑网格）对话框。**提示：**您也可以使用位于 Mesh 模块工具箱底部的 ![](../graphics/ico_meshEdit.png) 工具来显示 **Edit Mesh**（编辑网格）对话框。
4. 在对话框中，从 **Category**（类别）字段中选择 **Refinement**（细化）。
5. 从 **Method**（方法）列表中选择 **Set size**（设置大小）。
6. 在提示区域中输入您选择的全局面元大小，然后按 **[Enter]**。Abaqus/CAE 显示一个圆圈，指示重新网格化部件后单元的大小。
7. 从 **Method**（方法）列表中选择 **Remesh**（重新网格化）。
8. 点击提示区域中出现的 **Yes**（是）按钮。Abaqus/CAE 尝试细化网格。如果在细化网格时出错，点击 **Edit Mesh**（编辑网格）对话框中的 **Undo**（撤销）可以撤销细化。

**不使用指定的全局面元大小细化平面三角形网格：**

1. 进入 Mesh（网格）模块。
2. 从上下文栏的 **Object**（对象）字段中选择 **Part**（部件），然后从列表中选择包含孤网格单元的部件。**提示：**要细化混合部件中的孤单元，请抑制几何特征并在完成细化操作后恢复它们。
3. 从主菜单栏中选择 ****Mesh**![](../graphics/images/arrow.gif)**Edit****（网格→编辑）。Abaqus/CAE 显示 **Edit Mesh**（编辑网格）对话框。**提示：**您也可以使用位于 Mesh 模块工具箱底部的 ![](../graphics/ico_meshEdit.png) 工具来显示 **Edit Mesh**（编辑网格）对话框。
4. 在对话框中，从 **Category**（类别）字段中选择 **Refinement**（细化）。
5. 从 **Method**（方法）列表中选择 **Remove size**（删除大小）。
6. 从 **Method**（方法）列表中选择 **Remesh**（重新网格化）。
7. 点击提示区域中出现的 **Yes**（是）按钮。Abaqus/CAE 尝试细化网格。如果在细化网格时出错，点击 **Edit Mesh**（编辑网格）对话框中的 **Undo**（撤销）可以撤销细化。
