# 35.5.1 单元端点释放





**产品：** Abaqus/Standard   

##### **参考资料**

- ["运动约束：概述，" 第35.1.1节](pt08ch35s01abo32.md)
- [*RELEASE](../key/key-link.md#usb-kws-mrelease)

### 概述

单元端点释放：
- 允许在一个或两个单元或单元集的一端或两端的旋转自由度或旋转自由度的组合；
- 可用于几何线性或非线性分析；以及
- 仅适用于Abaqus/Standard中的梁单元和管道单元。

### 引言

单元端点释放用于模拟单元一端或两端的铰接连接（沿一个、两个或三个正交方向铰接）。通过释放旋转自由度，单元端点允许相对于节点沿选定的自由度自由旋转。未释放的任何旋转自由度与节点共享。您必须注意不要在共享该节点的所有单元的某个节点上释放给定的自由度；否则，该节点对该自由度没有刚度，Abaqus/Standard会发出零枢轴警告消息。

单元端点释放作用于单元局部自由度。有关梁类型单元的局部轴（![][usb_eqn06151.gif]，![][usb_eqn06152.gif]，*t*）的定义，请参见["梁单元横截面方向，" 第29.3.4节](pt06ch29s03alm09.md)。受释放影响的旋转自由度是：绕局部![][usb_eqn06151.gif]轴的旋转、绕局部![][usb_eqn06152.gif]轴的旋转，以及空间梁绕局部*t*轴的旋转。对于平面梁，只有绕局部![][usb_eqn06151.gif]轴的旋转是活跃的（与绕负全局*z*轴的旋转重合）。

#### 等效MPC

如果只释放一个旋转自由度，则运动约束等效于MPC类型REVOLUTE加上MPC类型PIN（两个节点之间）。如果释放两个旋转自由度，则运动约束等效于MPC类型UNIVERSAL加上MPC类型PIN。如果释放所有旋转自由度，则运动约束等效于MPC类型PIN。有关详细信息，请参见["广义多点约束，" 第35.2.2节](pt08ch35s02aus130.md)。

### 识别释放所涉及的单元端点

可以为释放定义指定单元集或单个单元。可以在单元的第一端、第二端或两端释放自由度。单元的第一端S1是单元定义中定义的节点1；第二端S2是最后一个节点（节点2或3，视情况而定）。有关梁单元节点排序的定义，请参见["梁单元库，" 第29.3.8节](pt06ch29s03ael14.md)。

### 识别释放所涉及的局部旋转自由度

指定旋转组合代码而不是自由度来识别释放所涉及的旋转自由度。

| M1 | 指绕![][usb_eqn06151.gif]轴的旋转， |
| --- | --- |
| M2 | 指绕![][usb_eqn06152.gif]轴的旋转， |
| M1-M2 | 指绕![][usb_eqn06151.gif]轴和![][usb_eqn06152.gif]轴的旋转自由度的组合， |
| T | 指绕*t*轴的旋转， |
| M1-T | 指绕![][usb_eqn06151.gif]轴和*t*轴的旋转自由度的组合， |
| M2-T | 指绕![][usb_eqn06152.gif]轴和*t*轴的旋转自由度的组合，以及 |
| ALLM | 表示所有旋转自由度（即M1、M2和T）的组合。 |

| **输入文件用法：** | ``` [*RELEASE](../key/key-link.md#usb-kws-mrelease) *单元编号或单元集，单元端点ID，释放组合代码* ``` |
| --- | --- |
|  | 例如，要在单元10的第一端释放绕![][usb_eqn06151.gif]轴的旋转自由度，并在单元的第二端释放所有旋转自由度，请使用以下输入：``` [*RELEASE](../key/key-link.md#usb-kws-mrelease) 10, S1, M1 10, S2, ALLM ``` |

### 与变换坐标系的使用

应用于释放节点的变换（["变换坐标系，" 第2.1.5节](pt01ch02s01aus09.md)）对释放没有影响。释放作用于单元的局部自由度。

### 从备用输入文件读取数据

释放定义的数据可以包含在单独的文件中。

| **输入文件用法：** | ``` [*RELEASE](../key/key-link.md#usb-kws-mrelease), INPUT=*file_name* ``` |
| --- | --- |
|  | 如果省略INPUT参数，则假定数据行跟在关键字行之后。
