# 61.5.2 创建、修改和移除键盘快捷键





从主菜单栏选择 ****Tools**![](../graphics/images/arrow.gif)**Customize**** 为 Abaqus/CAE 中的特定功能分配键盘快捷键。有关可接受的按键组合的信息，请参见["配置键盘快捷键，" 第61.2节](pt06ch61s02.md)。

**创建、修改或删除键盘快捷键的步骤：**

1. 在任何模块中，从主菜单栏选择 ****Tools**![](../graphics/images/arrow.gif)**Customize****。

2. 从打开的 **Customize** 对话框中，单击 **Functions** 选项卡。

3. 从 **Module/Toolset** 列表中，选择您的功能所属的 Abaqus/CAE 模块或工具集。Abaqus/CAE 使用所选模块或工具集中的可用功能填充 **Functions** 列表。

4. 选择您要为其创建、修改或移除键盘快捷键的功能。

5. 执行以下步骤之一：
   **创建或修改键盘快捷键：** 输入您想要使用的功能键或按键组合，然后单击 **Assign**。新的键盘快捷键出现在 **Current shortcut** 字段中。
   **移除键盘快捷键：** 单击 **Remove**。Abaqus/CAE 删除键盘快捷键分配并清除 **Current shortcut** 字段。
