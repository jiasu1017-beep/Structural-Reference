# 40.5.1 刚性表面接触单元





**产品：** Abaqus/Standard

##### **参考资料**

- [*RIGID SURFACE](../key/key-link.md#usb-kws-mrigidsurf)

### 概述

刚性表面接触单元：
- 可用于模拟刚性表面与变形体之间的接触；
- 仅在几个特殊应用中使用，例如当子结构接触刚性表面时，或者当CAXA或SAXA单元类型参与接触时；
- 可用于几何线性和非线性模拟；以及
- 使用与Abaqus/Standard中基于表面的接触功能相同的"主-从"概念来强制执行接触约束。

对于大多数问题，[第36章"定义接触相互作用"](pt09ch36.md)中所述的基于表面的接触功能提供了更直接和通用的方法来模拟刚性表面与变形体之间的接触。

### 使用刚性表面和刚性表面接触单元建模接触

确定接触区域的位置以及接触结构之间的表面牵引力是Abaqus模拟的常见目标。当其中一个结构假定为刚性时，刚性表面接触单元可用于建模接触。这些单元仅需用于以下所述的特定应用，因为Abaqus中的基于表面的接触定义可用于大多数模拟。

#### 使用轴对称刚性表面接触单元建模接触

轴对称刚性表面接触单元仅应在以下特定应用中使用：
- 当变形表面在子结构上时（请参阅["如果存在子结构时的接触建模，" 第36.3.9节](pt09ch36s03aus153.md)），或者
- 当CAXA或SAXA单元参与接触时（请参阅["如果存在非对称-轴对称单元时的接触建模，" 第36.3.10节](pt09ch36s03aus154.md)）。

其他平面、轴对称或三维问题应使用基于表面的接触功能。

### 接触应力和表面相对运动的局部坐标系

Abaqus/Standard在与刚性表面关联的局部坐标系中报告结构之间的接触应力和结构的相对运动。刚性表面的法向（即接触方向）是在创建刚性表面时定义的。详细信息，请参阅["解析刚性表面定义，" 第2.3.4节](pt01ch02s03aus19.md)。在轴对称问题中，Abaqus/Standard定义第一局部切向位于模型平面中，第二切向与此平面正交。

### 刚性表面接触单元的主-从概念

刚性表面接触单元使用"主-从"概念来强制执行接触约束。刚性表面接触单元形成"从"表面，这些单元的节点被约束不能穿透刚性（"主"）表面。

### 定义刚性表面

使用["当使用拖链或刚性表面单元时定义解析刚性表面"中的方法定义解析刚性表面，请参阅"解析刚性表面定义，" 第2.3.4节](pt01ch02s03aus19.md#usb-int-arigidsurf-dragchain-etc)。

#### 为刚性表面分配刚体参考节点

刚性表面的运动由与其关联的单个节点（即刚体参考节点）的运动控制。当在模型中使用刚性表面接触单元时，在定义IRS单元时会识别刚体参考节点（请参阅下面的详细信息）。

### 定义刚性表面接触单元

刚性表面接触单元定义从表面。它们还定义了与其相互作用的刚性表面的刚体参考节点。所有IRS单元通过将其节点编号作为连接性中的最后一个节点来识别刚体参考节点。变形体上形成IRS单元的节点始终首先给出。

在使用部件实例装配定义的模型中，刚性表面定义和参考节点必须出现在与刚性表面接触单元相同的部件定义内。

#### 示例

例如，以下输入将用于定义由变形体上的两个节点组成的IRS单元1和2，并指定节点1000作为刚体参考节点：

```
[*ELEMENT](../key/key-link.md#usb-kws-melement), TYPE=[IRS21A], ELSET=*element_set_name*
1, 10, 11, 1000
2, 11, 12, 1000
[*RIGID SURFACE](../key/key-link.md#usb-kws-mrigidsurf), ELSET=*element_set_name*
```

IRS22A单元使用类似的输入结构。

### 将解析刚性表面与一组刚性表面接触单元关联

您必须识别与特定刚性表面相互作用的一组刚性表面接触单元。

| **输入文件用法：** | ``` [*RIGID SURFACE](../key/key-link.md#usb-kws-mrigidsurf), ELSET=*element_set_name* ``` |
| --- | --- |

### 定义刚性表面单元的截面属性

必须将截面属性与一组刚性表面接触单元相关联。

轴对称刚性表面接触单元没有截面数据。

| **输入文件用法：** | ``` [*INTERFACE](../key/key-link.md#usb-kws-minterface), ELSET=*element_set_name* ``` |
| --- | --- |

### 使用刚性表面接触单元定义非默认机械表面相互作用

默认情况下，Abaqus/Standard对刚性表面接触单元使用"硬"无摩擦机械表面相互作用模型。您可以分配可选的机械表面相互作用模型。以下机械表面相互作用模型可用：
- 摩擦。详细信息请参阅["摩擦行为，" 第37.1.5节](pt09ch37s01aus169.md)。
- 修正的"硬"接触、软化接触和粘性阻尼。详细信息请参阅["接触压力-闭合关系，" 第37.1.2节](pt09ch37s01aus166.md)和["接触阻尼，" 第37.1.3节](pt09ch37s01aus167.md)。
