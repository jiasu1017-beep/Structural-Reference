# 5.7 通用对话框







Abaqus GUI Toolkit 为处理常见操作提供了一些预构建的对话框。本节提供有关如何使用这些对话框的详细信息。以下主题将被涵盖：
- "文件/目录选择器"，第 5.7.1 节
- "打印对话框"，第 5.7.2 节
- "颜色选择器对话框"，第 5.7.3 节

### 5.7.1 文件/目录选择器

**文件选择**对话框用于从用户收集文件或目录名称。它具有以下特性：
- 可以设置标题栏。
- 可以设置文件过滤器。
- 提供以下错误检查：- 检查文件是否存在。- 检查适当的权限。- 检查选择是否为文件。
- 允许只读访问。
- 接受关键字和目标。

文件选择对话框具有以下原型：

```
AFXFileSelectorDialog(form, title, fileNameKw,
    readOnlyKw, opts, patterns, patternIndexTgt) 

AFXFileSelectorDialog(parent, title, fileNameKw,
    readOnlyKw, opts, patterns, patternIndexTgt)
```

当您有一个与发出命令的对话框关联的表单时，使用第一个构造函数；例如，当您点击****文件**![](../graphics/images/arrow.gif)**打开数据库****时出现的对话框。当对话框收集用户输入用于另一个对话框时，使用第二个构造函数。例如，当从**打印**对话框打印到文件时，会向用户提供一个文本字段来输入文件名和一个**选择**按钮。**选择**按钮显示文件选择对话框，将选定的文件返回给**打印**对话框，但不发出任何命令。

您必须使用 `AFXStringKeyword` 方法创建 *fileNameKw* 参数。同样，您必须使用 `AFXBoolKeyword` 方法创建 *readOnlyKw* 参数。如果用户点击**确定**，文件选择对话框会自动更新 *fileNameKw* 和 *readOnlyKw* 参数。此外，当显示对话框时，它将根据 *fileNameKw* 参数的路径设置当前目录。这意味着对话框会记住用户上次访问的目录，当应用程序再次显示对话框时。

以下标志可用于 *opts* 参数：

**AFXSELECTFILE_EXISTING**

仅允许选择现有文件。

**AFXSELECTFILE_MULTIPLE**

仅允许选择多个现有文件。

**AFXSELECTFILE_DIRECTORY**

仅允许选择现有目录。

**AFXSELECTFILE_REMOTE_HOST**

允许打开远程主机上的文件。

您将 *patterns* 参数指定为用 `\n` 分隔的一系列模式。*patternIndexTgt* 参数指定的目标的值决定显示对话框时最初显示哪个模式。

以下是如何从表单显示文件选择对话框的示例：

```
def getFirstDialog(self):

    patterns = 'Output Database (*.odb)\nAll Files (*.*)'
    db = AFXFileSelectorDialog(self, 'Open ODB',
        self.nameKw, self.readOnlyKw, AFXSELECTFILE_EXISTING,
        patterns, self.patternIndexTgt)
    db.setReadOnlyPatterns('*.odb')
    self.setModal(True)
    return db 
```

以下是如何从另一个对话框显示目录选择对话框的示例：

```
def onCmdDirectory(self, sender, sel, ptr):

    if not self.dirDb:
        self.dirDb = AFXFileSelectorDialog(self,
            'Select a Directory',self.form.dirNameKw,
             None, AFXSELECTFILE_DIRECTORY)
        self.dirDb.create()

    self.dirDb.showModal()
    return 1
```

### 5.7.2 打印对话框

**打印**对话框提供标准打印功能。要从对话框中的按钮显示**打印**对话框，首先使用 FileToolsetGui 类的 `getPrintForm` 方法访问打印表单模式。可以通过如下例所示存储指向表单的指针来完成：

```
from sessionGui
import FileToolsetGui

class MyMainWindow(AFXMainWindow):

    #~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
    def __init__(self, app, windowTitle=''):

        ...

        fileToolset = FileToolsetGui()
        self.printForm = fileToolset.getPrintForm()
        self.registerToolset(fileToolset,
            GUI_IN_MENUBAR|GUI_IN_TOOLBAR)

        ...
```

然后您可以在对话框类中使用打印表单，如下所示：

```
printForm = getAFXApp().getAFXMainWindow().printForm 
FXButton(parent, 'Print...', None, printForm,
    AFXMode.ID_ACTIVATE)
```

要访问打印表单，必须构造并注册文件工具集。但是，您不能从插件内部访问打印表单。因此，您只能在自定义应用程序中使用此处描述的方法。

### 5.7.3 颜色选择器对话框

`AFXColorSelector` 组件提供从预定义调色板中选择颜色的功能。此对话框由 `AFXColorButton` 显示。如需更多信息，请参阅"颜色按钮"，第 3.1.10 节。
