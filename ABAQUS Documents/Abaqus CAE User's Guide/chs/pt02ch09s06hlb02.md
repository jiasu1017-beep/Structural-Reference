# 9.7.2 打开模型数据库或输出数据库





从主菜单栏选择****文件（File）**![](../graphics/images/arrow.gif)**打开（Open）****来打开以下任一文件：
- 模型数据库（文件扩展名`.cae`）
- 输出数据库（文件扩展名`.odb`）

从出现的**打开数据库（Open Database）**对话框中，选择**文件过滤器（File Filter）**和要打开的文件，然后单击**确定（OK）**。

您可以通过使用**追加到图层（Append to layers）**选项来打开多个输出数据库，并在单个视口中以叠加图的形式显示输出数据库的组合内容。有关使用叠加图的详细信息，请参阅[第79章，"叠加多个图"](pt07ch79.md)。

默认情况下，输出数据库文件以只读方式打开。您可以选择以写权限打开输出数据库文件；如果您想将任何X-Y数据对象复制到输出数据库，则必须这样做（有关更多信息，请参阅["将会话X-Y数据对象复制到输出数据库文件，"第47.2.8节"](pt05ch47s02hlb08.md)）。位于远程机器上的输出数据库文件只能以只读方式打开；您无法写入远程输出数据库文件。

来自先前版本的Abaqus的输出和模型数据库文件必须在打开时升级到当前版本（有关更多信息，请参阅["升级模型数据库或输出数据库，"第9.7.3节"](pt02ch09s06hlb03.md))。当您打开多个输出数据库文件时，所有文件必须已升级到当前版本；否则，Abaqus/CAE会在消息区域打印警告，并且不会打开需要升级的文件。

如果您使用的是旧版本的Abaqus/CAE，则无法打开从较新版本创建的模型或输出数据库文件。例如，Abaqus 6.7输出数据库文件无法在Abaqus 6.5中打开。

**要打开模型数据库或输出数据库：**

1. 从主菜单栏，选择****文件（File）**![](../graphics/images/arrow.gif)**打开（Open）****。**提示：**您也可以单击**文件**工具栏中的![](../graphics/ico_fileOpen.png)来打开模型数据库或输出数据库。Abaqus/CAE会显示**打开数据库（Open Database）**对话框。
2. 从**打开数据库**对话框底部的**文件过滤器（File Filter）**菜单中，选择以下之一：****模型数据库（*.cae）（Model Database (*.cae)）**** Abaqus/CAE列出所选目录中所有文件扩展名为`.cae`的文件。****输出数据库（*.odb）（Output Database (*.odb*)）**** Abaqus/CAE列出所选目录中所有文件扩展名为`.odb`的文件。****模型和输出数据库（*.cae，*.odb）（Model & Output Databases (*.cae, *.odb*)）**** Abaqus/CAE列出所选目录中所有文件扩展名为`.cae`或`.odb`的文件。
3. 如果在步骤2中选择了**输出数据库（*.odb）**，请使用以下选项进一步过滤文件列表或更改文件打开行为：****网络连接器（Network connectors）**** 如果您之前创建并启动了网络ODB连接器，**目录（Directory）**字段会包含一个**网络连接器（Network connectors）**项目，允许您访问远程目录并打开远程输出数据库。有关更多信息，请参阅["创建网络ODB连接器，"第9.7.4节"](pt02ch09s06hlb04.md)。****只读（Read-only）**** 默认情况下，输出数据库文件以只读方式打开。要以写权限打开输出数据库文件，请在单击**确定（OK）**之前关闭**打开数据库**对话框底部的**只读（Read-only）**。如果您想将X-Y数据对象从会话复制到文件或永久将旧文件升级到当前版本的Abaqus/CAE，则必须以写权限打开输出数据库文件。您只能以只读方式打开远程输出数据库文件。****追加到图层（Append to layers）**** 如果您想打开多个输出数据库并在单个视口中以叠加图形式显示组合内容，请切换**追加到图层（Append to layers）**并选择多个文件。您可以使用以下任一方法选择多个输出数据库文件：- 使用**[Shift]****+单击**或**[Ctrl]****+单击**选择文件。- 在**文件名（File Name）**字段中输入以逗号分隔的文件名列表，例如`lug.odb,hinge.odb`- 在**文件名（File Name）**字段中输入以双引号括起来的文件名列表，例如`"lug.odb" "hinge.odb"`**注意：**如果在创建输出数据库文件的分析运行但结果写入之前打开该文件，则可能需要在结果可用后关闭并重新打开该文件。
4. 单击**确定（OK）**打开所选的文件或文件。Abaqus/CAE会保存选定的过滤器类型，以便下次打开文件时作为默认值，并关闭**打开数据库**对话框。如果您打开了模型数据库，Abaqus/CAE会在主窗口的标题栏中显示其名称。所有操作现在都指向新的模型数据库。如果您修改了当前模型数据库，Abaqus/CAE会在打开所选模型数据库之前询问您是否要保存它。如果您打开了一个或多个输出数据库，Abaqus/CAE会在当前视口中启动可视化模块，并显示按字母顺序排列在未变形绘图状态中的最后一个模型。任何其他选定的输出数据库都会打开但不会显示，除非您切换了**追加到图层（Append to layers）**，在这种情况下，所选输出数据库都会绘制在同一视口中。
