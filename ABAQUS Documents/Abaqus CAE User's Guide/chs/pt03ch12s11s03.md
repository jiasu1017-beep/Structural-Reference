# 12.11.3 定义压电属性







您可以通过提供应力系数![](../graphics/usi_eqn00330.gif)或应变系数![](../graphics/usi_eqn00331.gif)来定义压电材料属性。请参见"压电行为"，Abaqus分析用户指南第26.5.2节获取更多信息。

**定义压电属性：**

1. 从**编辑材料**对话框的菜单栏中，选择**电气/磁性**![](../graphics/images/arrow.gif)**压电**。（有关显示**编辑材料**对话框的信息，请参见"创建或编辑材料"，第12.7.1节。）
2. 切换**使用温度相关数据**将压电属性定义为温度的函数。**数据**表中会出现一列标为**温度**。
3. 单击**场变量数量**字段右侧的箭头增加或减少压电属性所依赖的场变量数量。
4. 在**数据**表中输入压电应力或应变系数矩阵。请参见"压电行为"中的"指定压电材料属性"，Abaqus分析用户指南第26.5.2节获取更多信息。如适用，输入温度和场变量的值。有关如何输入数据的详细信息，请参见"输入表格数据"，第3.2.7节。
5. 单击**确定**关闭**编辑材料**对话框。或者，您可以从**编辑材料**对话框的菜单中选择其他材料行为进行定义（有关更多信息，请参见"浏览和修改材料行为"，第12.7.2节）。
