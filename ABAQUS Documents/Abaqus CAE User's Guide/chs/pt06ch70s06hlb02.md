# 70.6.2 使用最短路径方法分区面







您可以用在两个所选点之间形成最短可能路径的线或曲线来分区所选面。这些点可以是顶点、中点、弧中心或基准点。如果被分区的面不是平面的，则路径将是弯曲的（如下图所示）：

![](../graphics/par-face-shortest-nls.png)

**使用最短路径方法分区面：**

1. 从主菜单栏中选择 ****Tools**![](../graphics/images/arrow.gif)**Partition****（工具→分区）。出现 **Create Partition**（创建分区）对话框。Abaqus/CAE 在提示区域显示提示以指导您完成过程。**提示：**您也可以使用位于模块工具箱分区面工具处的 ![](../graphics/ico_ptnFaceTwoPoints.png) 工具来使用最短路径方法分区面。 有关工具箱中分区工具的图表，请参阅["使用分区工具集，" 第 70.2 节"](pt06ch70s02.md)。
2. 从对话框顶部的 **Type**（类型）单选按钮中选择 **Face**（面）。**Method**（方法）列表显示可用于分区面的方法。
3. 从方法列表中选择 **Shortest path between 2 points**（两点之间的最短路径）。
4. 如果部件或装配包含多个面，选择要分区的面。您可以使用拖动选择、**[Shift]****+点击**、**[Ctrl]****+点击** 和角度方法的组合来选择多个要分区的面。 有关更多信息，请参阅["在当前视口中选择对象，" 第 6.2 节"](pt01ch06s02.md)。**提示：**如果您无法选择所需的面，可以使用 **Selection**（选择）工具栏更改选择行为。 有关更多信息，请参阅["使用选择选项，" 第 6.3 节"](pt01ch06s03.md)。Abaqus/CAE 高亮显示所选面。
5. 选择定义路径起点和终点的顶点、中点、弧中心或基准点。Abaqus/CAE 高亮显示这些点。
6. 在提示区域中点击 **Create Partition**（创建分区）。Abaqus/CAE 创建分区。
