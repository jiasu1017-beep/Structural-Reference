# 1.6.1 应力/位移单元间的小滑动接触





**产品：**Abaqus/Standard  

### 单元测试

C3D4    C3D8    C3D8H    C3D10    C3D10I    C3D10M    C3D10MH    C3D20    C3D20H    C3D27

### 功能测试

自动"收缩"配合

小滑动接触对

### 问题描述

模型由接触表面初始过闭合的单元组成。初始过闭合在第一个分析步（非线性）中消除。第二个分析步是线性扰动步，在两个表面之间进行相对滑动。第三个非线性分析步中摩擦值发生变化。第四个分析步是线性扰动步，在两个表面之间进行相对摩擦滑动。第五个分析步是直接求解的稳态动态分析，分析两个接触表面。第六个分析步提取固有频率，然后在第七个分析步中使用这些频率进行基于子空间的稳态动态分析。

对于广义轴对称单元进行五步测试。整个测试使用各向异性摩擦。第一和第二步与前面提到的相同。第三步是线性扰动步，在两个表面之间进行相对扭转。第4步和第5步类似，只是第4步是线性扰动步。在这些分析步中，两个表面之间同时进行相对滑动和扭转。

对于圆柱型单元，只进行四步测试。

**模型：**

| 间隙 clearance | 0.01 |
| --- | --- |
| 圆柱型单元的间隙 clearance | 0.2 |
| 桁架长度 | 5.0 |
| 桁架面积 | 0.5 |
| 二维实体单元尺寸 | 1.0 5.0 1.0 |
| 轴对称固体和圆柱型单元的内半径 | 1.0 |
| 三维实体单元尺寸 | 每个方向1.0单位 |

**材料：**

| 杨氏模量 | 30 106 |
| --- | --- |
| 泊松比 | 0.3 |
| 间隙摩擦 | 0.0 |
| 密度 | 7700.0 |

### 结果与讨论

接触压力和牵引力与分析结果一致。
