# 12.4.1 网格到网格解映射






**产品：** Abaqus/Standard

##### **参考文献**

- [*MAP SOLUTION](../key/key-link.md#usb-kws-mmapsolution)

### 概述

将解从一种网格映射到另一种网格是重网格划分分析技术中的一个步骤，其中已从原始配置发生显著变形的网格被更优质的新网格替代，分析继续进行。解映射技术：
- 用于分析过程中单元变得严重变形以至于不再能良好地离散问题的情况；
- 将解从旧的变形网格映射到新网格，以便分析能够继续；以及
- 只能与连续体单元一起使用。

有关将此方法与其他 Abaqus 自适应方法进行比较的高级讨论，请参见 ["自适应技术," 第 12.1.1 节](pt04ch12s01aus77.md)。

### 何时重网格划分

Abaqus/Standard 使用拉格朗日 formulation：网格附着在材料上，因此随材料一起变形。当几何非线性分析中应变变得很大时，单元可能变得严重变形，以至于不再能良好地离散问题。严重变形可能发生在橡胶弹性问题或塑性/黏塑性计算中，特别是在模拟制造过程时。当发生严重变形时，有必要进行重网格划分：创建更适合继续分析的新网格，并将旧模型的解映射到此网格上。

您必须决定何时需要进行重网格划分。这个决定可以通过查看使用特定网格的分析阶段中发生的应变幅度来辅助，如后文所述。当需要重网格划分时，必须使用 Abaqus 中的网格生成功能或外部网格生成器为变形后的对象生成新网格。然后使用新网格作为新问题继续分析。在大多数情况下，需要将解从旧网格传输到新网格。

#### 解中的不连续性

每当解从另一个网格映射时，由于网格的变化和解映射算法的结果，预期会出现一些解的不连续性。如果不连续性显著，则表明网格过于粗糙，或者重网格划分应在太变形之前更早阶段进行。

只要网格足够精细以解决该问题，并且重网格划分在单元变得过于变形之前进行，重网格划分技术就能良好运行。

### 重网格划分准则

重网格划分的第一个要求是某种指示，表明网格在可能导致解不准确的区域中正在变形。重网格划分的一个可能准则是需要在高精度解析应变梯度的区域中出现极端单元变形。如果变形单元已进入应变场变化均匀的区域，则不准确性不太令人关注；无论变形多么严重，单元都能准确表示常应变状态。然而，最终，重网格划分的决定是判断的问题。

### 生成新网格

一旦您确定当前网格不合适，必须使用 Abaqus 中的网格生成功能或外部网格生成器生成更适合当前问题状态的新网格。变形构型图可能有助于提供关于被建模对象当前形状的数据。通常可以从形成物体表面的节点集输出的结果文件中定义用于网格生成器的外表面。参见 ["油井筒中的材料侵蚀（出砂）," Abaqus 例题指南第 1.1.22 节](../exa/exa-link.md#exa-sta-boreholeerosion) 和 ["圆柱坯料的镦粗：使用网格到网格解映射（Abaqus/Standard）和自适应网格划分（Abaqus/Explicit）的准静态分析," Abaqus 例题指南第 1.3.1 节](../exa/exa-link.md#exa-sta-rezonebillet)。

#### 接触问题的重网格划分

在接触区域，新网格必须紧密贴合旧分析中表面的形状。这个要求在涉及两个可变形体之间接触的问题中尤为重要；如果新网格定义的表面与旧分析中的表面即使略有不同，接触算法可能无法收敛。

### 指定要插值到新网格的解

通过从使用旧网格生成的输出数据库将解插值到新网格来继续模拟。

#### 指定必须读取解的时间

默认情况下，解传输将从最新步骤和增量进行，只要有解变量可用。或者，您可以指定将读取旧解的步骤和增量。

#### 平移和旋转旧作业网格

在执行映射之前，可以通过相对于全局原点给出平移和/或旋转来重新定位旧作业的网格。通过给出平移向量来指定平移。通过给出两个点来定义旋转轴，加上围绕该轴的右手角旋转来指定旋转。

| **输入文件用法：** | ``` [*MAP SOLUTION](../key/key-link.md#usb-kws-mmapsolution), STEP=*step*, INC=*increment* *translation vector data* *rotation axis and angular rotation data* ``` |
| --- | --- |

#### 识别旧作业

通过在运行 Abaqus 的命令中使用 **oldjob** 参数或通过回答命令过程的请求来指定将从中获取重启和结果数据的旧作业的名称（参见 ["Abaqus/Standard、Abaqus/Explicit 和 Abaqus/CFD 执行," 第 3.2.2 节](pt01ch03s02abx02.md)）。旧作业所需的文件包括：重启文件（`.res`）、输出数据库（`.odb`）、模型数据库（`.mdl`）、状态数据库（`.stt`）和零件（`.prt`）文件。

### 解映射算法

解映射通过从旧网格中的节点将结果插值到新网格中的点（节点或积分点）来操作。因此，第一步涉及将解变量与旧网格中的节点关联。对于节点解变量（如节点温度或孔隙压力），关联已经完成。对于积分点变量，Abaqus 通过从积分点外推到每个单元的节点，然后对连接每个节点的所有相似单元求平均值来获得旧网格节点处的解变量。

接下来，获得新网格中每个点相对于旧网格的位置。新网格点包括所有情况下的积分点，以及除了位移外还记录节点状态的程序中的节点（例如耦合温度-位移程序中的节点温度）。

1. 找到该点所在的（旧网格中的）元素，并获得该点在该元素中的位置。（此过程假设新网格中的所有点都位于旧网格的边界内：如果不是这样，将发出警告消息，并将变量值设置为零。）
2. 然后将变量从旧单元的节点插值到新模型中的点。

所有必要的变量以这种方式自动插值，以便可以使用新网格继续解。

#### 解扩散

此算法在映射解中引入了一些扩散。扩散的影响随旧网格中的解梯度而缩放；因此，即使在模型中网格不从前一个模型变为新模型的区域，当旧网格解梯度较高时，映射引起的扩散也可能导致显著不同的映射量。您可以通过在解梯度较高的区域细化旧网格或更早进行重网格划分来缓和此效果。

### 初始条件

从初始分析映射的解构成重网格划分分析的初始条件。可以指定纯应力/位移分析的初始条件（如温度）。任何其他指定的初始条件将被忽略。

### 边界条件

边界条件不会从旧网格携带到新网格。在重网格划分分析开始时应用的边界条件通常应该与在从初始分析选择的步骤和增量时生效的边界条件相同。虽然可以更改边界条件，但如果结构远离平衡状态，问题可能无法收敛。

在映射解分析中应用边界条件没有限制。边界条件可以以与分析无映射解相同的方式应用于所有可用自由度（参见 ["Abaqus/Standard 和 Abaqus/Explicit 中的边界条件," 第 34.3.1 节](pt07ch34s03aus118.md)）。

### 载荷

在映射解分析中应用载荷没有限制。载荷可以与分析无映射解相同的方式应用。

在重网格划分分析开始时应用的载荷通常应该与在初始分析结束时生效的载荷相同。虽然可以更改载荷，但如果结构远离平衡状态，问题可能无法收敛。

### 预定义场

温度和场变量从旧网格映射到新网格。如果重网格划分分析中场变量的数量发生变化，则将传输两个分析共有的数量。预定义场可以与分析无解映射时修改的方式相同的方式进行修改（参见 ["预定义场," 第 34.6.1 节](pt07ch34s06aus128.md)）。

### 材料选项

Abaqus 中可用的任何机械本构模型都可以在映射解分析中使用（参见 [第五部分，"材料"](pt05.md)"）。新旧分析之间的材料模型没有一致性限制。解映射算法将传输两个模型共有的那些变量。您必须确保材料模型是兼容的。

### 单元

解映射功能只能与连续体单元一起使用（参见 ["实体（连续体）单元," 第 28.1.1 节](pt06ch28s01alm01.md)）。

### 输出

映射解分析没有特定的输出。输出可以与分析无映射解相同的方式请求。Abaqus 中可用的输出变量列在 ["Abaqus/Standard 输出变量标识符," 第 4.2.1 节](pt02ch04s02abv01.md)。
