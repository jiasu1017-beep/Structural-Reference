# 46.4.1 自定义材料方向图三轴









您可以自定义材料方向图三轴的颜色、长度、粗细和箭头外观。您还可以抑制一个或多个三轴的显示；对于具有复合截面的结果，您可以使用整个层压的材料方向或其各个层来显示结果。

**要自定义三轴外观：**

1. 从主菜单栏选择****选项**![](../graphics/images/arrow.gif)**材料方向****，或点击工具箱中的![](../graphics/ico_plotMatOrientOptions.png)。三轴选项出现。
2. 切换左侧每个轴的按钮以显示或抑制它们。
3. 点击颜色样本以为图中可见的三轴选择您想要的颜色。
4. 拖动**大小**滑块以增加或减少图中材料方向三轴的长度。轴长度范围为0到30；默认长度为6。**注意：**您可以使用Abaqus脚本界面指定大于30的轴长度值。请参见["使用Python解释器，" Abaqus脚本用户指南第4.3节](../cmd/cmd-link.md#cmd-int-pyt-intro-interpreter)。
5. 从**基准**选项中，选择**屏幕大小**或**模型大小**作为材料方向三轴轴长度计算的基础。如果您选择**屏幕大小**，当您更改视口大小时，Abaqus/CAE会调整三轴的大小；如果您选择**模型大小**，当您缩放时，Abaqus/CAE会调整三轴的大小。
6. 点击**粗细**字段旁边的箭头，然后选择您想要的轴粗细。
7. 点击**箭头**字段旁边的箭头，然后选择您想要的箭头设计。
8. 如果需要，您可以显示材料方向三轴的较小子集以澄清具有许多三轴的图的显示。将**符号密度**滑块拖动到**高**和**低**之间的值。
9. 如果您的结果包含SORIENT输出变量的场输出以及具有复合截面的元素的输出，请指定您要使用的材料方向。选择**层**以使用各自的材料方向显示每个层，或选择**层压**以使用复合层压定义中指定的方向显示整个复合层压。
10. 点击**应用**以实现您的更改。当前视口中的材料方向图三轴会更改以反映您的设置。默认情况下，您的更改在会话期间保存，并会影响所有后续材料方向图。如果要为后续会话保留您的更改，请将它们保存到文件。有关更多信息，请参见["保存自定义设置以供后续会话使用，" 第55.1.1节](pt05ch55s01s01.md)。
