# 64.7.5 扩展线性孤立网格的短边







从主菜单栏中选择 ****Mesh**![](../graphics/images/arrow.gif)**Edit**** 可扩展线性孤立网格的选定边。您可以指定应被扩展的单元边的最小长度，Abaqus/CAE 将移动短于指定长度的单元边的端节点。您可以指定整个网格以外的单元域，并且可以指定方向和/或参考边以仅扩展与这些条件匹配的边。只有当单元域仅包含线性单元时，才能在网格中扩展边。

**扩展线性网格的短边：**

1. 进入 Mesh 模块。从上下文栏的 **Object** 字段中选择 **Part**，然后从列表中选择一个零件。
2. 从主菜单栏中选择 ****Mesh**![](../graphics/images/arrow.gif)**Edit****。Abaqus/CAE 显示 **Edit Mesh** 对话框。**提示：**您也可以使用位于 Mesh 模块工具箱底部的 ![](../graphics/ico_meshEdit.png) 工具来显示 **Edit Mesh** 对话框。
3. 从 **Category** 字段中选择 **Mesh**。
4. 从 **Method** 列表中选择 **Grow short edges**。Abaqus/CAE 显示 **Grow short edges** 对话框。
5. 输入 **Edge tolerance** 以指定单元边的最小长度，表示您要扩展短于该容差的单元边。默认值约为零件中平均单元大小的 10%。
6. 如需要，切换 **Specify** 并点击 **Select** 以选择整个网格以外的单元域。**注意：**Abaqus/CAE 完全不会检查指定域之外的边。如果与所选域相邻的单元边较短，而域内的边扩展，则重新定位的节点可能会使域外的单元太小或可能倒置一个单元。
7. 如需要，切换 **Thickness direction** 并输入方向向量或点击箭头图标通过在视口中拾取点来创建向量。Abaqus/CAE 使用方向向量来测量单元，而不是仅测量节点之间的距离。当您使用方向向量时，扩展操作必须限于所选单元的特定边，以避免扩展垂直于预期方向的边。如果单元的定向使得方向在拓扑上不明确，Abaqus/CAE 会高亮显示这些单元，以便您可以选择排除它们的域或添加参考边以进一步定义所需的边。
8. 如需要，切换 **Reference edge** 并点击 **Select** 以选择参考边。当您使用参考边时，Abaqus/CAE 仅在拓扑上与所选边平行的那些单元边中检查短边。**注意：**如果您的单元域包含未连接的单元部分，您只能在该域的单个连续部分内选择参考边。您可能需要多次在小单元域上使用 **Grow short edges** 工具以获得预期结果。
9. 点击 **OK**。Abaqus/CAE 高亮显示将要扩展边的单元。在提示区域点击 **Yes** 以扩展边，或点击 **No** 以返回 **Grow short edges** 对话框。如果没有任何单元边短于指定的最小长度，Abaqus/CAE 会显示一条错误消息，指示最短的单元长度。**提示：**如果在折叠网格中的短边时出错，点击 **Edit Mesh** 对话框中的 **Undo** 可撤消更改。Abaqus/CAE 会恢复原始网格。

编辑孤立网格后，您应始终验证基于节点、单元和表面的特征（如截面分配、载荷和相互作用），以确保它们正确应用于修改后的网格。
