# 1.16.2 轮廓积分评估：三维情况

**产品：** Abaqus/Standard

此示例是完全三维裂纹配置中轮廓积分评估的说明。此示例提供了该方法（对于线弹性响应）的验证，因为对于此几何有可用的比较结果。

Abaqus 提供 *J* 积分；应力强度因子，![](../graphics/bmk_eqn01130.gif)；和 *T* 应力，作为三维几何中裂纹前缘位置函数的参数。可以使用多个轮廓；并且，由于积分应该是路径无关的，使用不同轮廓获得值的离散可作为结果质量的指标。Abaqus 中用于计算轮廓积分的域积分方法通常给出准确的结果，即使模型相当粗，如图所示。Abaqus 提供基于传统有限元方法或扩展有限元方法（XFEM）的断裂力学研究参数评估。

### 问题描述

在此示例中分析了几何形状。此外，同时使用了传统有限元方法和扩展有限元方法。

#### 半空间中的半椭圆裂纹

分析的第一个几何是半空间中的半椭圆裂纹，如图 1.16.2-1（[图 1.16.2-1](ch01s16ach122.md#sxm3djint-surfcrack)）所示。裂纹通过远场拉力以 I 型加载。当使用传统有限元方法时，由于对称性，只需分析身体的四分之一。网格如图 1.16.2-2（[图 1.16.2-2](ch01s16ach122.md#bmk-elliptic-crack-mesh)）所示。使用减缩积分单元（C3D20R），中间节点被移动到聚焦到裂纹尖端节点的单元边缘的四分之一点位置。这种四分之一点方法提供应变奇异性，从而改善了裂纹尖端附近应变场的建模（见"轮廓积分评估：二维情况"第 1.16.1 节对该技术的讨论）。裂纹前缘的法向用于指定裂纹扩展方向，如图 1.16.2-3（[图 1.16.2-3](ch01s16ach122.md#bmk-ellipticcrsurf-normal)）所示。网格向外延伸得足够远，使得模型远表面上的边界条件对解的影响可以忽略不计。使用围绕裂纹尖端的三个单元环来评估轮廓积分。

当使用扩展有限元方法时，网格不需要与裂纹几何匹配。裂纹的存在通过特殊的富集函数以及额外的自由度来确保。这种方法还取消了在评估轮廓积分时明确定义裂纹前缘或指定虚拟裂纹扩展方向的要求。轮廓积分所需的数据将基于单元中节点的符号距离函数自动确定。第一个几何的一阶砖块单元（C3D8）网格如图 1.16.2-4（[图 1.16.2-4](ch01s16ach122.md#bmk-elliptic-crack-xfem-mesh)）所示；裂纹前缘由输出变量 PSILSM 的符号距离等值线图表示，如图 1.16.2-5（[图 1.16.2-5](ch01s16ach122.md#bmk-elliptic-crackfront-xfem)）所示。

#### 矩形板中的半椭圆裂纹

分析的第二个几何是矩形板中的半椭圆裂纹，如图 1.16.2-6（[图 1.16.2-6](ch01s16ach122.md#sxm3dcint-surfcrack)）所示。板受到均匀拉力。由于对称性，当使用传统有限元方法时，只需分析身体的四分之一。板相对于板厚度 *t* 的尺寸如下：半高度 ![](../graphics/bmk_eqn01141.gif) = 16；半宽度 ![](../graphics/bmk_eqn01142.gif) = 8；中间平面裂纹深度 ![](../graphics/bmk_eqn01143.gif) = 0.6；表面裂纹半长度 ![](../graphics/bmk_eqn01144.gif) = 2.5，这导致表面裂纹深宽比 ![](../graphics/bmk_eqn01145.gif) = 0.24。使用传统有限元方法的板的网格如图 1.16.2-7（[图 1.16.2-7](ch01s16ach122.md#bmk-ellipticcrack-meshrectplate)）所示，其在裂纹平面上的轮廓如图 1.16.2-8（[图 1.16.2-8](ch01s16ach122.md#bmk-ellipticcrrect-meshprofile)）所示。该模型对传统有限元方法使用 C3D8 单元，对扩展有限元方法使用 C3D8 和 C3D4 单元。
