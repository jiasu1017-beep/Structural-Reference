# 69.7.4 修复小面







您可以修复部件中所选的小面。Abaqus/CAE 移除小面并修复相邻边和面以创建闭合几何。修复小面的操作作为部件的特征存储；因此，您可以使用模型树删除或抑制操作。

**修复小面：**

1. 从主菜单栏中选择 ****Tools**![](../graphics/images/arrow.gif)**Geometry Edit****（工具→几何编辑）。Abaqus/CAE 显示 **Geometry Edit**（几何编辑）对话框。**提示：**您也可以使用位于 Part（部件）模块工具箱中编辑工具处的 ![](../graphics/ico_repairSmallFaces.png) 工具来修复小面。 有关工具箱中编辑工具的图表，请参阅["使用几何编辑工具集，" 第 69.1 节"](pt06ch69s01.md)。
2. 从对话框中选择 **Face**（面）类别和 **Repair small**（修复小面）方法。
3. 选择要修复的面。您可以从视口选择面或选择包含面的现有集合。**注意：**默认选择方法基于您最近使用的选择方法。要恢复为其他方法，请点击提示区域右侧的 **Select in Viewport**（在视口中选择）或 **Sets**（集合）。您可以使用拖动选择、**[Shift]****+点击**、**[Ctrl]****+点击** 和角度方法的组合来选择多个要修复的小面。 有关更多信息，请参阅["在当前视口中选择对象，" 第 6.2 节"](pt01ch06s02.md)。**提示：**如果您无法选择所需的面，可以使用 **Selection**（选择）工具栏更改选择行为。 有关更多信息，请参阅["使用选择选项，" 第 6.3 节"](pt01ch06s03.md)。
4. 点击鼠标按钮 2 表明您已完成选择小面。Abaqus/CAE 修复小面并提示您更新部件的有效性。
5. 执行以下操作之一： - 点击 **Yes**（是）更新部件的有效性。（每当您修复小面时都应该更新部件的有效性。） - 点击 **No**（否）跳过更新并继续您的修复。 - 点击 **Cancel**（取消）取消当前修复操作中的更改。您也可以随时通过在模型树中的部件上点击鼠标按钮 3 并从出现的菜单中选择 **Update validity**（更新有效性）来更新部件的有效性。
