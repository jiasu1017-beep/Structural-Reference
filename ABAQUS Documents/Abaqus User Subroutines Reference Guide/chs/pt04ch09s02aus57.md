### 概述

Abaqus 提供了将所需结果和模型信息从 Abaqus/Explicit 分析传输到新的 Abaqus/Explicit 分析的功能，在继续分析之前可以在其中指定其他模型定义。例如，在装配过程中，分析人员可能首先对特定组件的局部行为感兴趣，但随后关心的是装配产品的行为。在这种情况下，可以首先在 Abaqus/Explicit 分析中分析局部行为。随后，可以将此分析中的模型信息和结果传输到第二个 Abaqus/Explicit 分析，在其中可以为其他组件指定额外的模型定义，然后可以分析整个产品的行为。

为此功能正常工作，必须在同一台二进制兼容的计算机上运行相同版本的 Abaqus/Explicit。

有关如何在 Abaqus 分析之间传输结果的信息，请参阅["在 Abaqus 分析之间传输结果：概述，" 第 9.2.1 节"](pt04ch09s02aus54.md)。

### 与重新启动功能的比较

Abaqus/Explicit 中的导入和重新启动功能都允许将结果和模型信息从一个 Abaqus/Explicit 分析传输到另一个 Abaqus/Explicit 分析。但是，这两个功能是为不同应用设计的。

重新启动功能允许重新启动并继续已完成的 Abaqus/Explicit 分析。原始分析的全部模型和结果被传输到重新启动运行，在其中可以定义额外的分析步骤。在重新启动的分析中不能指定太多新的模型数据；只允许新幅值定义、新节点集和新单元集等模型信息。有关重新启动功能的详细信息，请参阅["重新启动分析，" 第 9.1.1 节"](pt04ch09s01aus53.md)。

导入功能还允许继续已完成的 Abaqus/Explicit 分析。此外，此功能允许仅使用原始分析中所需的组件继续分析；无需传输整个模型。可以在导入分析期间指定新的模型数据，如元素、节点、表面、接触对等。在导入分析期间，可以选择是仅传输先前分析的模型信息，还是也传输与该模型关联的结果。

对于目标是继续原始分析而不更改模型信息的情况，建议使用重新启动功能。对于模型信息需要更改的情况，或者对于需要控制结果传输的情况，应使用导入功能。

### 在导入分析中指定新数据

可以在导入分析期间定义额外的模型定义，如新元素、节点、表面等。也可以在导入分析期间指定初始条件。

#### 新模型定义

一旦指定了导入，就可以在导入分析中向模型添加新节点、元素和材料属性。无论在导入时是否更新了参考配置（请参阅["更新参考配置"中的"在 Abaqus 分析之间传输结果：概述，" 第 9.2.1 节"](pt04ch09s02aus54.md#usb-anl-atransferoverview-update)），必须在更新的配置中定义节点坐标。可以使用常规 Abaqus/Explicit 输入。导入的材料定义可以与新元素一起使用（将需要新的截面属性定义）。

#### 节点变换

节点变换（["变换坐标系统，" 第 2.1.5 节"](pt01ch02s01aus09.md)）不会被导入；变换可以在导入分析中独立定义。只有在导入分析中的节点变换与原始分析中的节点变换相同时，才能获得连续的位移、速度等。对于在局部系统中定义边界条件或点载荷的节点，也建议使用相同的变换。

#### 在导入分析中指定几何非线性

默认情况下，Abaqus/Explicit 使用大应变公式。对于分析的每个步骤，您可以指定是否应包括几何非线性；请参阅["几何非线性"中的"通用和线性扰动过程，" 第 6.1.3 节"](pt03ch06s01aus44.md#usb-anl-alinearnonlinear-nlgeom)，了解更多详情。

导入分析的公式默认值与导入时的值相同。一旦在导入分析中的任何步骤中使用了大连移公式，它将在所有后续步骤中保持活动状态，无论分析是否被导入。

如果在导入时使用了小位移公式，则不能更新参考配置。

#### 为导入的元素和节点指定初始条件

只能在某些条件下对导入的元素或节点指定初始条件。[表 9.2.4-1](pt04ch09s02aus57.md#table-aexptoexp-type-ics) 列出了根据是否导入材料状态（请参阅["导入材料状态"中的"在 Abaqus 分析之间传输结果：概述，" 第 9.2.1 节"](pt04ch09s02aus54.md#usb-anl-atransferoverview-state)）允许的初始条件。参考配置可以根据需要更新或不更新，但有一个例外：对于初始温度或场变量条件，必须更新参考配置。

**表 9.2.4-1** 有效的初始条件。
| 初始条件 | 导入了材料状态 |
| --- | --- |
| 场变量 | 否 |
| 硬化 | 否 |
| 相对密度 | 否 |
| 旋转速度 | 是或否 |
| 解决方案相关的状态变量 | 否 |
| 应力 | 否 |
| 温度 | 否 |
| 速度 | 是或否 |
| 孔隙比 | 否 |

### 边界条件

原始分析中指定的边界条件（包括连接器运动）不会被导入。必须在导入分析中重新定义。

在某些情况下，需要在原始分析中施加的非零边界条件在导入分析中保持在相同的值（当导入的配置未更新时）。在这种情况下，您可以为分析步骤规定常数（阶跃函数）幅值变化（请参阅["规定非默认幅值变化"中的"定义分析，" 第 6.1.2 节"](pt03ch06s01abo05.md#usb-anl-aover-nondefaultamp)），以便新施加的边界条件立即应用并在步骤持续时间内保持该值。或者，您可以在边界条件定义中引用幅值曲线（请参阅["幅值曲线，" 第 34.1.2 节"](pt07ch34s01aus115.md)）。如果在原始分析中在变换坐标系统中施加了边界条件（请参阅["变换坐标系统，" 第 2.1.5 节"](pt01ch02s01aus09.md)），应在导入分析中定义并使用相同的坐标系统。

有关应用边界条件和多点约束的讨论，请参阅["Abaqus/Standard 和 Abaqus/Explicit 中的边界条件，" 第 34.3.1 节"](pt07ch34s03aus118.md)和["运动学约束：概述，" 第 35.1.1 节"](pt08ch35s01abo32.md)。

### 载荷

原始分析中定义的载荷（包括用于连接器驱动的载荷）不会被导入。因此，可能需要在导入分析中重新定义载荷。在将结果从一个分析导入到另一个分析时，可以应用的载荷没有限制。当载荷需要保持在与原始分析相同的值时，您可以为分析步骤规定常数（阶跃函数）幅值变化（请参阅["规定非默认幅值变化"中的"定义分析，" 第 6.1.2 节"](pt03ch06s01abo05.md#usb-anl-aover-nondefaultamp)），以便在步骤开始时立即应用载荷并在步骤持续时间内保持。或者，您可以在载荷定义中引用幅值曲线（请参阅["幅值曲线，" 第 34.1.2 节"](pt07ch34s01aus115.md)）。如果在原始分析中在变换坐标系统中施加了点载荷（请参阅["变换坐标系统，" 第 2.1.5 节"](pt01ch02s01aus09.md)）并且载荷必须在导入分析中保持，则如果使用相同的坐标系统并在导入分析中使用，载荷应用会简化。

请参阅["应用载荷：概述，" 第 34.4.1 节"](pt07ch34s04aus120.md)，了解 Abaqus/Explicit 中可用载荷类型的概述。

### 预定义场

温度（无论是规定的还是作为自由度，如在耦合热应力分析中）以及节点处的场变量如果导入了材料状态，则会被导入。

如果更新了参考配置并且导入了材料状态，则导入节点处的温度和场变量的初始条件将重置为导入的值；例如，现在将相对于导入的温度测量热应变。如果更新了参考配置但未导入材料状态，则初始条件重置为零。在这种情况下，您可以重新指定导入节点上的初始条件。

如果温度是状态变量（如在温度是积分点量的绝热分析中），则在导入材料状态时会导入。

### 材料选项

与导入元素关联的所有材料属性定义和取向默认会被导入。可以通过使用相同材料名称重新指定材料属性定义来更改材料属性。在这种情况下，所有相关材料属性必须重新定义，因为默认导入的旧定义将被覆盖。只有在更新了参考配置且未导入材料状态时，才能更改与导入元素关联的材料取向；对于参考配置和材料状态的其他组合，不能重新定义与导入元素关联的材料取向。

#### 超弹性材料

当导入超弹性材料时，如果未更新配置，则必须导入状态；如果未导入状态，则必须更新配置。

#### 连接器元素

当导入连接器元素时，关联的连接器行为定义默认会被导入。只有在未导入状态时，才能修改导入的连接器行为定义。

#### 材料阻尼

如果在导入分析中需要更改材料阻尼，则必须重新定义材料模型。

#### 材料定义的更改

更改材料定义时，应小心确保保持一致的材料状态。有时可以简化材料定义。例如，如果在第一个 Abaqus/Explicit 分析中使用了 Mises 塑性模型，并且预期在后续 Abaqus/Explicit 分析中不会有进一步的塑性屈服，则可以为后续 Abaqus/Explicit 分析使用线弹性材料。但是，如果预期会有进一步的非线性材料行为，则不应更改现有材料定义。如果两个分析中的材料模型不同，状态变量的历史将不会被保留。

此外，以下限制适用于导入功能：
- 使用钢筋层定义的钢筋（["定义钢筋，" 第 2.2.3 节"](pt01ch02s02aus13.md)）会随底层元素一起导入。使用嵌入单元技术（["嵌入单元，" 第 35.4.1 节"](pt08ch35s04aus136.md)）定义的钢筋增强会随此定义中使用的宿主和嵌入元素一起导入。定义为单元属性（["将钢筋定义为单元属性，" 第 2.2.4 节"](pt01ch02s02aus14.md)）的钢筋不能导入。
- 如果导入连接器元素，只要未导入状态，就可以更新配置，只要未更新配置，就可以导入状态。
- 包含变形和刚性单元的刚体不能导入。包含刚性单元的刚体在指定用于定义刚体的单元集时被导入。包含变形单元的刚体在指定用于定义刚体的单元集时被导入。导入的刚体定义如果使用相同的单元集重新指定，则会被覆盖。当模型定义为部件实例装配时，导入刚体的参考节点必须属于导入的实例。
- 当导入刚体时，任何关联的数据（如引脚节点集和绑定节点集）都是导入定义的一部分。但是，导入时这些集合仅包含连接到导入元素的那些节点。

### 约束

原始分析中指定的运动学约束（包括多点点约束和基于表面的绑定约束）不会被导入，必须在导入分析中重新定义。请参阅["运动学约束：概述，" 第 35.1.1 节"](pt08ch35s01abo32.md)，了解各种运动学约束的讨论。

### 相互作用

对于一般接触，如果在两个分析中都定义了一般接触，则会导入接触状态。

对于由接触对定义的接触，原始分析中指定的接触定义和接触状态不会被导入。可以通过指定表面和接触对在导入分析中重新定义接触。

可以在导入分析中通过指定新的表面、接触对和相互作用来定义额外的接触信息。

有关 Abaqus/Explicit 中接触能力的详细描述，请参阅["接触相互作用分析：概述，" 第 36.1.1 节"](pt09ch36s01abo33.md)。

### 输出

可以像在未导入结果的分析中一样为导入分析请求输出。原始分析中的输出请求不会传输到导入分析；必须在导入分析中重新指定输出请求。Abaqus/Explicit 中可用的输出变量列在["Abaqus/Explicit 输出变量标识符，" 第 4.2.2 节"](pt02ch04s02xbv01.md)中。

当导入材料状态时，以下材料点输出变量的值在导入分析中将是连续的：应力、等效塑性应变（PEEQ）和 [`VUMAT`](../sub/sub-link.md#sub-xsl-vumat) 的解决方案相关状态变量（SDV）。同样，对于连接器行为，塑性相对位移（CUP）、运动硬化移位力（CALPHAF）、整体损伤（CDMG）、损伤起始标准（CDIF、CDIM、CDIP）、摩擦累积滑移（CASU）和连接器状态（CSLST、CFAILST）将是连续的。

如果未更新参考配置，位移、应变、整个单元变量、截面变量和能量量将相对于原始配置报告。

如果更新了参考配置，位移、应变、整个单元变量、截面变量和能量量在导入分析中不会连续，将相对于更新的参考配置报告。

如果更新了参考配置，时间和步骤号在原始分析和导入分析之间不会连续。只有在未更新参考配置时，时间和步骤号才会连续。

### 限制

导入功能具有以下已知限制。在适用的情况下，详细说明在相关部分中给出。
- 必须在二进制兼容的计算机上运行相同版本的 Abaqus/Explicit。
- 此功能不适用于弹簧和阻尼单元。请参阅前面关于["单元"](pt04ch09s02aus57.md#usb-anl-aexptoexp-elements)的讨论以了解更多详情。
- 如果导入连接器元素，只要未导入状态，就可以更新配置，只要未更新配置，就可以导入状态。
- 表面不会被导入。
- 当导入部件实例时，原始分析中的所有元素和节点必须至少属于一个集。
- 接触对的接触状态不会被导入。
- 如果导入了材料状态，对于除线弹性、超弹性、超泡沫、粘弹性、Mises 塑性和内聚单元损伤之外的材料模型，只会导入应力。请参阅["导入材料状态"中的"在 Abaqus 分析之间传输结果：概述，" 第 9.2.1 节"](pt04ch09s02aus54.md#usb-anl-atransferoverview-state)，了解更多详情。对于连接器行为，塑性位移、摩擦滑移和损伤状态会被导入，连接器力会被重新计算。请参阅["导入材料状态"中的"在 Abaqus 分析之间传输结果：概述，" 第 9.2.1 节"](pt04ch09s02aus54.md#usb-anl-atransferoverview-state)，了解更多详情。
- 载荷、边界条件、多点点约束、方程和基于表面的绑定约束不会被导入。
- 运动学和分布耦合约束不会被导入。此外，耦合约束的参考节点不会被导入，除非该参考节点是属于要导入的另一单元定义的一部分。
- 在导入分析期间生成的结果文件、重新启动文件或输出数据库文件不会追加到原始分析的结果文件、重新启动文件或输出数据库文件。
- 网格独立点焊（请参阅["网格独立紧固件，" 第 35.3.4 节"](pt08ch35s03aus135.md)）和绑定约束（请参阅["网格绑定约束，" 第 35.3.1 节"](pt08ch35s03aus132.md)）不会被导入。这些约束可以在导入分析中重新定义，并使用导入模型的参考配置形成。如果更新了参考配置，重新定义的约束可能与旧约束不完全匹配。如果定义了新约束且未更新导入模型的参考配置，则如果约束中的节点有非零位移，它们最初可能不兼容。这可能导致数值困难和导入分析的可能中止。在这种情况下，建议您在导入时更新参考配置。
