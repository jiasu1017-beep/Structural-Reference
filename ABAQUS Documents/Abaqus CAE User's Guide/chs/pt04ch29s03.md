# 29.3 创建基于点的紧固件





您可以通过从视口中拾取每个定位点或从文件读取定位点坐标来在相互作用模块中创建基于点的紧固件。您可以允许Abaqus/CAE将定位点沿表面法线投影到最近的面，或者可以指定点投影到最近面时所沿的方向向量。更多信息，请参见["关于紧固件，" 第29.1节](pt04ch29s01.md)。

**创建基于点的紧固件：**

1. 从相互作用模块的主菜单栏中，选择****特殊**![](../graphics/images/arrow.gif)**紧固件**![](../graphics/images/arrow.gif)**创建****。Abaqus/CAE显示**创建紧固件**对话框。**提示：**您也可以使用相互作用模块工具箱中的![](../graphics/ico_fastener.png)工具创建基于点的紧固件。
2. 从出现的**创建紧固件**对话框中，选择**基于点**的紧固件，然后单击**继续**。
3. 选择定位点，然后单击**完成**。您可以选择连接点、参考点或来自孤立网格的节点。更多信息，请参见["通过拾取或从文件读取创建连接点，" 第59.3节](pt06ch59hla01.md)；["通过选择方向和间距创建连接点，" 第59.4节](pt06ch59hla02.md)；以及["基于边创建连接点模式，" 第59.5节](pt06ch59hla03.md)。Abaqus/CAE显示**编辑紧固件**对话框。
4. 在**域**、**准则**、**属性**、**公式**和**调整**选项卡页面上，配置设置，例如将创建点的区域、连接器截面、耦合类型和影响半径，如以下过程中所述：
   - ["定义紧固件将连接到的表面，" 第29.3.1节](pt04ch29s03hlb01.md)
   - ["定义紧固件附着区域，" 第29.3.2节](pt04ch29s03hlb02.md)
   - ["定义紧固件属性，" 第29.3.3节](pt04ch29s03hlb03.md)
   - ["定义紧固件公式，" 第29.3.4节](pt04ch29s03hlb04.md)
   - ["调整紧固件定义，" 第29.3.5节](pt04ch29s03hlb05.md)
5. 单击**确定**创建基于点的紧固件。符号出现在视口中，表示刚创建的基于点的紧固件的定位点。
