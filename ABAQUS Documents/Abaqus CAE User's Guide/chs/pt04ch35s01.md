# 35.1 理解中面建模





中面建模是一种创建实体模型简化壳表示的技术。当可以使用具有定义厚度和较少细节的壳模型进行所需分析时，中面建模可以降低分析完整实体模型的计算成本。

中面建模过程依赖于准确实体模型作为起点。中面建模最适用于壁厚恒定或可以容易地近似每个点壁厚的薄实体或薄壁实体。您可以将中面建模应用于模型中的任何实体单元；您不需要将其应用于整个模型。如果您仅将中面建模应用于实体模型的一部分，Abaqus/CAE会自动创建壳-实体耦合约束，以将中面壳边缘的运动耦合到剩余实体模型面的运动。如果壳表面与实体面的角度明显偏离90度，则不会创建壳-实体耦合约束。

除了降低成本之外，您可能使用中面模型代替实体模型，以更好地考虑模型薄截面中的弯曲响应。壳单元被设计为在单个单元的厚度内管理弯曲载荷，而单个实体单元将几乎没有或没有弯曲阻力。

在Abaqus/CAE中创建中面模型是一个手动过程。以下过程概述了所涉及的一般步骤。每个步骤中的任务可能因原始实体模型的复杂性和完整性以及分析要求而异。某些步骤（如以下步骤4）可能不需要或可能以与此处呈现不同的顺序完成。

**要创建中面模型：**

1. 在实体部件上分配中面区域。使用**Assign Midsurface Region**工具从实体部件中选择一个或多个单元，以开始创建中面模型。更多信息，请参见["分配中面区域"，第35.5.1节](pt04ch35s05s01.md)。Abaqus/CAE从活动表示中移除所选单元，并创建包含所选单元的参考表示。
2. 创建新的壳面以替换您移动到参考表示的实体几何体。您可以使用Geometry Edit工具集中的工具或壳特征创建工具来向中面模型添加壳特征。更多信息，请分别参见[第69章，"Geometry Edit工具集"](pt06ch69.md)和["添加壳特征"，第11.22节](pt03ch11s22.md)。Offset Faces工具是用于中面模型的最常见面创建方法。偏移过程基于所选几何体创建新面，并基于偏移参数自动分配壳厚度。
3. 为新壳特征分配厚度。**Assign Thickness and Offset**工具允许您为新的中面几何体分配壳厚度（更多信息，请参见["分配厚度和偏移"，第35.5.2节](pt04ch35s05s02.md)）。该工具允许您为单个面分配厚度，您也可以使用它来编辑Abaqus/CAE为使用偏移过程创建的面自动分配的厚度。您仍然必须在Property模块中为新面分配截面。在截面分配过程中，您可以最后决定是使用截面属性定义的厚度还是使用几何体的定义。您可以在所有基于部件和装配的模块中切换壳厚度显示，以在视口中查看厚度（更多信息，请参见["可视化壳厚度"，第35.5.3节](pt04ch35s05s03.md)）。
4. 修订与用于创建参考表示的实体几何体关联的任何分析属性。分析属性（如载荷、边界条件或相互作用）不能应用于参考表示中的几何体。在几何体移动到参考表示之前与实体几何体关联的任何属性，如果几何体不再是活动表示的一部分，将与空区域关联。更多信息，请参见["理解参考表示"，第35.2节](pt04ch35s02.md)。
