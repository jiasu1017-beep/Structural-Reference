# 12.5.2 管理材料库





**Material Library Manager**允许您创建、编辑和重命名材料库、库类别和库材料。默认的**Specification Tags**表示材料数据的来源、描述、测量单位和供应商材料名称。您可以为库中的每种材料创建、编辑、重命名和删除规范标签。材料属性和类别按字母顺序呈现，与Abaqus/CAE主窗口中库的默认视图相同；材料类别首先出现，然后是不在类别中的材料。您可以使用管理器从模型复制材料到库，或从库复制材料到模型。**Material Library Manager**如图[12-15](pt03ch12s05s02.md#prp-matlibrary-manager-nls)所示。

**图12-15** **Material Library Manager**。

![](../graphics/prp-matlibrary-manager-nls.png)

使用材料库时，建议您仔细考虑材料和类别名称。您可以创建具有完全相同名称的多个类别和/或材料。例如，您可以具有相同等级钢材的相同条目，每个条目包含不同单位集的属性。使用合适的类别名称，您可以轻松识别每种材料。但是，如果以列表格式显示库，相同条目都会出现在列表中。修改材料名称以包含单位信息可能更容易识别所需的材料。例如，您可以将一种材料命名为`Steel 1020 US`，将另一种命名为`Steel 1020 SI`。或者，您可以在材料库管理器中点击**Tags**来指示显示在主窗口材料列表下方的**Specification Tags**中的单位。在材料库中重命名材料只会更改库中显示的名称，而不会更改从Abaqus/CAE模型复制或复制到模型的底层材料名称。从库添加到模型的材料仍会保留旧名称。

您无法在材料库管理器中查看或编辑材料属性。要查看或编辑材料的属性，必须将材料添加到模型并使用**Edit Material**对话框（有关更多信息，请参见["创建或编辑材料，"第12.7.1节](pt03ch12s07hlb01.md)）。

**管理材料库：**

1. 从Model Tree区域中的**Material Library**标签页，点击**Name**字段右侧的**Material Library Manager**图标![](../graphics/ico_manager.png)。Abaqus/CAE显示**Material Library Manager**。
2. 从管理器顶部，点击**Create**创建新材料库，或选择库名称来编辑现有库。
3. 如果在上一步中选择了**Create**，则会出现**Create Material Library**对话框。
   1. 输入新库的名称。
   2. 选择**Home**或**Current**来选择库将保存的`abaqus_plugins`目录位置。
   3. 点击**OK**。Abaqus/CAE创建一个空的材料库文件并在管理器中打开它进行编辑。如有必要，Abaqus/CAE还会创建指定的`abaqus_plugins`目录。有关更多信息，请参见["材料库概述，"第12.5.1节](pt03ch12s05s01.md)。
4. 要编辑当前材料库，请从管理器左侧的**Library Materials**列表中选择一个项目，然后选择以下操作之一：
   - 如果要创建类别来组织库中的材料，请点击**Add Category**。出现**Create Category**对话框。
     1. 输入类别名称。
     2. 在**Create Category**对话框中点击**OK**。Abaqus/CAE创建新类别。该类别出现在与您所选项相同的库级别。例如，如果您选择了库名称，新类别会出现在任何其他类别的库名称下方；如果您选择了现有类别，新类别会出现在该类别中。
   - 如果要重命名所选项，请点击**Rename**。在出现的对话框中输入新名称，然后点击**OK**。
     **注意：**重命名库会更改在Abaqus/CAE中显示的名称；它不会更改库（`.lib`）文件的名称。
   - 如果要编辑展开材料库底部**Specification Tags**字段时显示的信息，请点击**Tags**。标签包含材料的来源、描述、单位供应商名称。默认情况下，供应商名称标签包含材料名称，其他标签每个都包含"Imported from CAE"。您可以使用标签来澄清具有相同名称或相似属性的材料的预期用途。
   - 如果要从中删除材料或空类别，请选择它并点击**Delete**。
     **注意：**您无法从Abaqus/CAE中删除材料库；您必须从系统上的保存位置删除库文件。
5. 使用位于**Library Materials**列表和**Model Materials**列表之间的箭头，从库向模型复制材料数据，反之亦然。当您复制材料时，Abaqus/CAE会将新材料放在所选类别（如果有）的末尾，或库的末尾。
   1. 从**Material Library Manager**中的**Models**列表中，选择要向其（或从其）复制材料的模型。
   2. 从**Library Materials**列表或**Model Materials**列表中选择所需的材料。
   3. 如果要将材料复制到库中，请选择**Library Materials**列表中您希望Abaqus/CAE放置材料的类别。
   4. 点击相应的箭头以复制材料。Abaqus/CAE将新材料添加到库的所选类别末尾或模型的材料列表末尾。

您在**Material Library Manager**中所做的更改会立即在管理器对话框中可见。但是，只有点击**Save Changes**后才会将更改提交到库文件。在您关闭材料库管理器之前，Abaqus/CAE不会更新主窗口中的库视图。
