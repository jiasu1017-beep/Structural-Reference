# 2.1 Accessing the Learning Community from Abaqus/CAE







**Product: **Abaqus/CAE  

**Benefits: **Abaqus/CAE provides easy access to the Learning Community.

**Description: **You can now access the Learning Community ([http://www.3ds.com/products-services/simulia/academics/simulia-learning-community](http://www.3ds.com/products-services/simulia/academics/simulia-learning-community)) from the main menu bar in Abaqus/CAE. The Learning Community contains online tutorials and technical content. The community also hosts a question-and-answer area that enables the global community of users to share their expertise and learn how to leverage the latest features and enhancements available in the SIMULIA portfolio.

**Abaqus/CAE Usage: **
```
All modules:
   ****Help**![](../graphics/images/arrow.gif)**Learning Community****   
```
