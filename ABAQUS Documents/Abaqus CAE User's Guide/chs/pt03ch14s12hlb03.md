# 14.12.3 修改历史输出请求





你可以使用历史输出编辑器来修改现有的历史输出请求。如果你在请求被传播到的步骤中修改历史输出请求，则只能修改输出变量和输出频率。

**要修改历史输出请求：**

1. 从主菜单栏，选择****Output**![](../graphics/images/arrow.gif)**History Output Request**![](../graphics/images/arrow.gif)**Manager****。Abaqus/CAE显示**历史输出请求管理器**。管理器指示在哪个步骤中创建了输出请求以及被传播到哪些步骤。
2. 从管理器中的历史输出请求列表中，选择你要修改请求的步骤。
3. 从**历史输出请求管理器**右侧的按钮中，点击**编辑**。Abaqus/CAE显示**编辑历史输出请求**编辑器。编辑器顶部的信息指示以下内容：
   - 输出请求的名称。
   - 你正在编辑输出请求的步骤名称。
   - 与步骤关联的分析过程。
4. 如果你在创建请求的步骤中编辑输出请求，则可以更改将从中输出变量的区域。从编辑器顶部，点击**域**文本字段旁边的箭头并选择以下之一：
   - 选择**整个模型**请求Abaqus将历史数据写入整个模型的输出数据库。
   - 选择**集合**请求Abaqus仅为你指定的命名区域将历史数据写入输出数据库。点击箭头并从出现的集合列表中选择名称。
   - 选择**螺栓载荷**请求Abaqus仅为指定的螺栓载荷将历史数据写入输出数据库。点击箭头并从出现的螺栓载荷列表中选择名称。
   - 选择**复合铺层**请求Abaqus仅为指定的复合铺层中的铺层将历史数据写入输出数据库。点击箭头并从出现的复合铺层列表中选择名称。
   - 选择**轮廓积分**请求Abaqus仅为指定的轮廓积分将历史数据写入输出数据库。点击箭头并从出现的轮廓积分列表中选择名称。当从轮廓积分请求输出时，**历史输出请求**编辑器仅允许你选择输出频率、轮廓积分数量和轮廓积分计算类型。更多信息请参见["请求轮廓积分输出，"第31.2.11节](pt04ch31s02hlb04.md)。
   - 选择**紧固件**请求Abaqus仅为指定的紧固件将历史数据写入输出数据库。点击箭头并从出现的紧固件列表中选择名称。
   - 选择**通用接触曲面**请求Abaqus仅为指定的通用接触曲面将历史数据写入输出数据库（此选项仅适用于Abaqus/Explicit分析）。点击箭头并从出现的曲面列表中选择曲面。历史输出将仅写入通用接触域内的曲面。
   - 选择**集成输出截面**请求Abaqus仅为指定的集成输出截面将历史数据写入输出数据库。点击箭头并从出现的截面列表中选择名称。有关创建集成输出截面的信息，请参见["定义集成输出截面，"第14.13.1节](pt03ch14s13hlb01.md)。
   - 选择**相互作用**请求Abaqus仅为指定的作用将历史数据写入输出数据库。点击箭头并从出现的表面到表面接触和自接触相互作用列表中选择名称。
   - 选择**蒙皮**请求Abaqus仅为指定的蒙皮增强将历史数据写入输出数据库。点击箭头并从出现的蒙皮列表中选择名称。
   - 选择**弹簧/阻尼器**请求Abaqus仅为指定的弹簧/阻尼器将历史数据写入输出数据库。点击箭头并从出现的弹簧/阻尼器列表中选择名称。
   - 选择**纵梁**请求Abaqus仅为指定的纵梁增强将历史数据写入输出数据库。点击箭头并从出现的纵梁列表中选择名称。
5. 指定所需的输出频率：
   - 选择**最后增量**仅请求最后增量的历史输出。此输出频率仅在你选择Abaqus/Standard分析过程时可用。
   - 选择**每n个增量**请求Abaqus以增量将历史数据写入输出数据库。然后你可以在出现的**n**字段中指定增量数。如果你以增量指定频率，Abaqus也会在步骤的最后增量后写入输出。此输出频率在你选择Abaqus/Standard或Abaqus/CFD分析过程时可用。
   - 选择**每n个时间增量**请求Abaqus以时间增量将历史数据写入输出数据库。然后你可以在出现的**n**字段中指定时间增量数。此输出频率仅在你选择Abaqus/Explicit分析过程时可用。
   - 选择**等间距时间间隔**请求Abaqus在多个等间距时间间隔将历史数据写入输出数据库。然后你可以在出现的**间隔**字段中指定间隔数。
   - 选择**每x单位时间**请求Abaqus在经过特定时间段后将历史数据写入输出数据库。然后你可以在出现的**x**字段中指定时间段。
   - 选择**从时间点**请求Abaqus根据一组时间点将历史数据写入输出数据库。然后你可以从出现的**时间点**列表中选择一组时间点，或点击![](../graphics/ico_timePointSmall.png)创建一组新的时间点。请参见["定义时间点，"第14.13.5节](pt03ch14s13hlb05.md)，了解有关创建时间点集的更多信息。此输出频率仅在你选择Abaqus/Standard分析过程时可用。
6. 如果你选择了Abaqus/Standard分析过程并请求**等间距时间间隔**、**每x单位时间**或**从时间点**的输出，你也可以从**时间**字段中选择**在精确时间输出**，以更改时间增量大小以精确匹配时间间隔。
7. 从**输出变量**字段中，选择要输出的变量和变量分量：
   **从列表下方选择** 选择此方法从变量类别列表中选择感兴趣的输出变量。使用以下技术选择特定变量：
   - 在编辑器顶部，点击所需变量类别旁边的箭头。从出现的变量列表中，选择你选择的变量。如果感兴趣的变量有分量，点击变量旁边的箭头并选择感兴趣的分量。要选择或取消选择变量的所有分量，请切换变量本身。
   - 切换所需的变量类别以选择或取消选择该类别中的所有变量和变量分量。当该类别中的所有变量及其所有分量都被选中时，变量类别名称旁边的复选框将完全填充。如果仅选择了该类别中的某些变量或变量分量，则该框变为半填充。同样，当选择了变量的所有分量时，变量名称旁边的复选框将完全填充。如果仅选择了该变量的某些分量，则该框变为半填充。
   **预选默认值** 选择此方法允许Abaqus/CAE为步骤的分析过程选择一组预选（默认）输出变量和分量。
   **全部** 选择此方法自动选择列表中每个变量类别中的所有允许输出变量和变量分量。
   **编辑变量** 选择此方法在变量类别列表上方的文本字段中输入或删除输出变量和分量。
   **注意：**除了当前分析过程外，模型的其他方面可能会影响预选默认输出变量。例如，如果输出变量对分析过程有效但对网格中使用的单元类型无效，Abaqus将在分析期间删除该变量。
8. 如果你想包含传感器输出，请启用**在可用时包含传感器**。此选项在选择**集合**域时可用。
9. 切换**对向量值输出使用全局方向**。此选项在选择**集合**域时可用。当**对向量值输出使用全局方向**开启时，以全局方向请求节点历史输出。当**对向量值输出使用全局方向**关闭时，以由节点变换定义的局部方向请求节点历史输出。
10. 如果你的域设置为**整个模型**、**集合**、**蒙皮**或**纵梁**，请执行以下操作：
    1. 如果你的模型包含钢筋，并且你在创建请求的步骤中编辑输出请求，则可以包含钢筋的输出在Abaqus写入输出数据库的历史数据中。从编辑器底部，启用**钢筋输出**并选择出现的以下选项之一：
       ****包含**** 选择**包含**请求Abaqus除了底层材料的输出外还将钢筋输出写入输出数据库。
       ****仅**** 选择**仅**请求Abaqus仅将钢筋输出写入输出数据库。如果你想在可视化模块中查看钢筋方向，必须启用**钢筋输出**。
    2. 如果你在创建请求的步骤中编辑输出请求，则可以更改将从中输出变量的截面点。从编辑器底部，选择以下之一：
       ****使用默认值**** 选择**使用默认值**请求Abaqus从默认截面点将场数据写入输出数据库。Abaqus根据在属性模块中选择的截面选择默认截面点。（默认截面点通常是截面的外层。）更多信息请参见[第12章"属性模块"](pt03ch12.md)。
       ****指定**** 选择**指定**输入Abaqus将写入输出数据库的场数据的截面点。指定的截面点仅在选定的输出请求期间使用；Abaqus在后续输出请求中恢复为默认截面点。
11. 如果你的域设置为**复合铺层**，指定将从中输出变量的截面点。
    **注意：**默认情况下，Abaqus/CAE仅从复合铺层的顶部和底部写入场输出数据，不从铺层生成数据。因此，如果你的模型包含复合铺层并且你需要来自各个铺层的数据，则必须创建新的输出请求或编辑默认输出请求并指定将从中输出变量的截面点。从编辑器底部，选择以下之一：
    **选定** 选择**为每个铺层选择点**请求Abaqus从每个铺层的顶部、中部和/或底部截面点将场数据写入输出数据库。
    **全部** 选择**所有铺层中的所有截面点**请求Abaqus从所有铺层中的所有截面点将场数据写入输出数据库。
    ****指定**** 选择**指定**输入Abaqus将写入输出数据库的截面点。截面点从第一个铺层的顶部到最后一个铺层的底部按顺序编号。例如，如果你有三个铺层，每个有三个截面点，并且你想从每个铺层的中间截面点获取输出，你应该输入`2,5,8`。指定的截面点仅在选定的输出请求期间使用；Abaqus在后续输出请求中恢复为默认截面点。
12. 如果你在创建请求的步骤中为Abaqus/Explicit分析过程编辑输出请求，你可以应用滤波器从历史输出中去除高频数据。从编辑器底部，启用**应用滤波器**并选择默认的**抗混叠**滤波器或选择使用滤波器工具集创建的命名滤波器。更多信息请参见[第66章"滤波器工具集"](pt06ch66.md)。
13. 完成定义输出请求后，点击**确定**保存你的更改。
