# 14.12.2 修改场输出请求





你可以使用场输出编辑器来修改现有的场输出请求。如果你在请求被传播到的步骤中修改场输出请求，则只能修改输出变量和输出频率。

**要修改场输出请求：**

1. 从主菜单栏，选择****Output**![](../graphics/images/arrow.gif)**Field Output Request**![](../graphics/images/arrow.gif)**Manager****。Abaqus/CAE显示**场输出请求管理器**。管理器指示在哪个步骤中创建了输出请求以及被传播到哪些步骤。
2. 从管理器中的场输出请求列表中，选择你要修改请求的步骤。
3. 从**场输出请求管理器**右侧的按钮中，点击**编辑**。Abaqus/CAE显示**编辑场输出请求**编辑器。编辑器顶部的信息指示以下内容：
   - 输出请求的名称。
   - 你正在编辑输出请求的步骤名称。
   - 与步骤关联的分析过程。
4. 如果你在创建请求的步骤中编辑输出请求，则可以更改将从中输出变量的区域。从编辑器顶部，点击**域**文本字段旁边的箭头并选择以下之一：
   - 选择**整个模型**请求Abaqus将场数据写入整个模型的输出数据库。启用**仅外部**仅请求外部节点和单元的输出；此选项仅适用于Abaqus/Standard或Abaqus/Explicit分析中的三维模型。
   - 选择**集合**请求Abaqus仅为你指定的命名区域将场数据写入输出数据库。点击箭头并从出现的集合列表中选择名称。
   - 选择**螺栓载荷**请求Abaqus仅为指定的螺栓载荷将场数据写入输出数据库。点击箭头并从出现的螺栓载荷列表中选择名称。
   - 选择**复合铺层**请求Abaqus仅为指定的复合铺层中的铺层将场数据写入输出数据库。点击箭头并从出现的复合铺层列表中选择名称。
   - 选择**紧固件**请求Abaqus仅为指定的紧固件将场数据写入输出数据库。点击箭头并从出现的紧固件列表中选择名称。
   - 选择**子结构**请求Abaqus仅为指定的子结构集合将场数据写入输出数据库。点击![](../graphics/ico_edit.png)打开**选择子结构集合**对话框，然后为你需要写入场数据的每个子结构零件实例切换子结构集合。如果你想为特定子结构零件实例中的所有集合写入场数据，请切换其**整个子结构**选项。
   - 选择**相互作用**请求Abaqus仅为指定的作用将场数据写入输出数据库。点击箭头并从出现的表面到表面接触和自接触相互作用列表中选择名称。
   - 选择**蒙皮**请求Abaqus仅为指定的蒙皮增强将场数据写入输出数据库。点击箭头并从出现的蒙皮列表中选择名称。
   - 选择**纵梁**请求Abaqus仅为指定的纵梁增强将场数据写入输出数据库。点击箭头并从出现的纵梁列表中选择名称。
5. 指定所需的输出频率：
   - 选择**最后增量**仅请求最后增量的场输出。此输出频率仅在你选择Abaqus/Standard分析过程时可用。
   - 选择**每n个增量**请求Abaqus以增量将场数据写入输出数据库。然后你可以在出现的**n**字段中指定增量数。如果你以增量指定频率，Abaqus也会在步骤的最后增量后写入输出。此输出频率在你选择Abaqus/Standard或Abaqus/CFD分析过程时可用。
   - 选择**每个时间增量**请求Abaqus在每个时间增量将场数据写入输出数据库。此输出频率仅在你选择Abaqus/Explicit分析过程时可用。
   - 选择**等间距时间间隔**请求Abaqus在多个等间距时间间隔将场数据写入输出数据库。然后你可以在出现的**间隔**字段中指定间隔数。
   - 选择**每x单位时间**请求Abaqus在经过特定时间段后将场数据写入输出数据库。然后你可以在出现的**x**字段中指定时间段。
   - 选择**从时间点**请求Abaqus根据一组时间点将场数据写入输出数据库。然后你可以从出现的**时间点**列表中选择一组时间点，或点击![](../graphics/ico_timePointSmall.png)创建一组新的时间点。请参见["定义时间点，"第14.13.5节](pt03ch14s13hlb05.md)，了解有关创建时间点集的更多信息。此输出频率在你选择Abaqus/Standard或Abaqus/Explicit分析过程时可用。
6. 如果你请求**等间距时间间隔**、**每x单位时间**或**从时间点**的输出，你也可以从**时间**字段中选择**在精确时间输出**，以更改时间增量大小以精确匹配时间间隔。
7. 在编辑器的**输出变量**部分，选择以下变量选择方法之一：
   **从列表下方选择** 选择此方法从下方变量类别列表中选择感兴趣的输出变量。使用以下技术选择特定变量：
   - 点击所需变量类别旁边的箭头。从出现的变量列表中，选择你选择的变量。
   - 切换所需的变量类别。此操作选择或取消选择该类别中的所有变量。当该类别中的所有变量都被选中时，变量类别旁边的复选框将完全填充。如果仅选择了该类别中的某些变量，则该框变为半填充。
   **预选默认值** 选择此方法允许Abaqus/CAE为步骤的分析过程选择一组预选（默认）输出变量。
   **全部** 选择此方法自动选择列表中每个变量类别中的所有允许输出变量。
   **编辑变量** 选择此方法在变量类别列表上方的文本字段中输入或删除输出变量。
   **注意：**除了当前分析过程外，模型的其他方面可能会影响预选默认输出变量。例如，如果所选输出变量对分析过程有效但对网格中使用的单元类型无效，Abaqus将在分析期间删除该变量。
8. 如果你的域设置为**整个模型**、**集合**、**蒙皮**或**纵梁**，请执行以下操作：
   1. 如果你的模型包含钢筋，并且你在创建请求的步骤中编辑输出请求，则可以包含钢筋的输出在Abaqus写入输出数据库的场数据中。从编辑器底部，启用**钢筋输出**并选择出现的以下选项之一：
      ****包含**** 选择**包含**请求Abaqus除了底层材料的输出外还将钢筋输出写入输出数据库。
      ****仅**** 选择**仅**请求Abaqus仅将钢筋输出写入输出数据库。如果你想在可视化模块中查看钢筋方向，必须启用**钢筋输出**。
   2. 如果你在创建请求的步骤中编辑输出请求，则可以更改将从中输出变量的截面点。从编辑器底部，选择以下之一：
      ****使用默认值**** 选择**使用默认值**请求Abaqus从默认截面点将场数据写入输出数据库。Abaqus根据在属性模块中选择的截面选择默认截面点。（默认截面点通常是截面的外层。）更多信息请参见[第12章"属性模块"](pt03ch12.md)。
      ****指定**** 选择**指定**输入Abaqus将写入输出数据库的场数据的截面点。指定的截面点仅在选定的输出请求期间使用；Abaqus在后续输出请求中恢复为默认截面点。
   3. 关闭**在可用时包含局部坐标方向**可以通过从保存的数据中排除材料方向来减小输出数据库的大小。
9. 如果你的域设置为**螺栓载荷**或**相互作用**，关闭**在可用时包含局部坐标方向**可以通过从保存的数据中排除材料方向来减小输出数据库的大小。
   **注意：**如果你从输出数据库中排除局部坐标方向，Abaqus/CAE会在默认坐标系中显示所有分析结果。
10. 如果你的域设置为**复合铺层**，指定将从中输出变量的截面点。更多信息请参见["从复合铺层请求输出，"第23.5节](pt04ch23s05.md)。
    **注意：**默认情况下，Abaqus/CAE仅从复合铺层的顶部和底部写入场输出数据，不从铺层生成数据。因此，如果你的模型包含复合铺层并且你需要来自各个铺层的数据，则必须创建新的输出请求或编辑默认输出请求并指定将从中输出变量的截面点。从编辑器底部，选择以下之一：
    **选定** 选择**为每个铺层选择点**请求Abaqus从所选复合铺层中每个铺层的顶部、中部和/或底部截面点将场数据写入输出数据库。
    **全部** 选择**所有铺层中的所有截面点**请求Abaqus从所选复合铺层中所有铺层的所有截面点将场数据写入输出数据库。
    ****指定**** 选择**指定**输入Abaqus将写入输出数据库的截面点。截面点从第一个铺层的顶部到最后一个铺层的底部按顺序编号。指定的截面点仅在选定的输出请求期间使用；Abaqus在后续输出请求中恢复为默认截面点。
11. 如果你在创建请求的步骤中为Abaqus/Explicit分析过程编辑输出请求，你可以应用滤波器从场输出中去除高频数据。从编辑器底部，启用**应用滤波器**并选择默认的**抗混叠**滤波器或选择使用滤波器工具集创建的命名滤波器。更多信息请参见[第66章"滤波器工具集"](pt06ch66.md)。
12. 完成定义输出请求后，点击**确定**保存你的更改。
