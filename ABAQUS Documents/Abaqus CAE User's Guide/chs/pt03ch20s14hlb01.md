# 20.17.2 通过修剪或延伸边缘来修改 Sketcher 对象


使用 Sketcher 工具箱中的修剪和延伸工具来修剪或延伸 Sketcher 对象的各个边缘。您可以在边缘与其他草图、参考或构造几何的任何交点处修剪或延伸边缘。您也可以在扩展交点处修剪或延伸边缘——即只有当草图中的另一个边缘被延伸时才会存在的交点。您无法修剪或延伸圆角;如果您修剪或延伸定义圆角端点的边缘,Abaqus/CAE 会删除该圆角。

**修剪或延伸边缘的步骤:**

1. 从 Sketcher 工具箱的修改工具中,选择修剪和延伸工具 ![](../graphics/ico_sketchTrimExtend.png)。有关 Sketcher 工具箱中工具的图示,请参见["Sketcher 工具," 第 20.4.1 节](pt03ch20s04s01.md)。Abaqus/CAE 会在提示区域显示提示,引导您完成操作。
2. 选择要保留的部分附近的边缘。您可以选择任何绘制或导入的边缘,除了圆和椭圆。圆和椭圆只有一个边缘,除非您先将它们转换为圆弧,否则无法延伸或修剪。**提示:**使用自动修剪工具删除圆或椭圆的一部分,或使用分割工具将其分割成圆弧。更多信息,请分别参见["通过自动修剪边缘来修改 Sketcher 对象," 第 20.17.3 节](pt03ch20s14hlb03.md)和["通过分割边缘来修改 Sketcher 对象," 第 20.17.4 节](pt03ch20s14hlb04.md)。您无法延伸样条曲线。如果您修剪或分割样条曲线,Abaqus/CAE 会删除控制点以保留原始曲线的精确形状;因此,您将无法再编辑曲线的形状。
3. 将光标移动到草图中的其他边缘上。当您移动光标时,Abaqus/CAE 会高亮显示当前光标位置将被选中的边缘。Abaqus/CAE 还会通过以不同颜色高亮显示延伸或修剪部分来预览第一边缘的延伸或修剪部分。
4. 单击以选择高亮的边缘。Abaqus/CAE 会将第一边缘修剪或延伸至与第二边缘的交点(或潜在交点)。如果交点位于第二边缘的端点之间,Abaqus/CAE 也会分割第二边缘。
5. 要修剪或延伸更多边缘,请从第 2 步开始重复上述步骤。
6. 完成修剪和延伸边缘后,执行以下操作之一: - 在 Abaqus/CAE 窗口任意位置单击鼠标按钮 2。 - 选择 Sketcher 工具箱中的任何工具。 - 单击提示区域中的取消按钮 ![](../graphics/afxI_cancel.png)。
