# 42.5.6 选择状态场输出变量





您可以为选定场输出变量指定基于结果的标准用于单元失败；这个变量称为状态场输出变量。当您启用状态场输出变量时，满足指定标准的单元被视为失败并从模型图中移除。使用**Field Output**对话框中的**Status Variable**选项来启用状态变量的使用；选择变量以及如果适用的话选择不变量或分量；并指定基于结果的标准。如果需要，您可以将状态场输出变量应用于未变形形状图。默认情况下，在未变形状态下显示模型时不会移除失败的单元。

默认情况下，状态场输出变量未启用，除非输出数据库中出现STATUS输出变量。在这种情况下，默认启用状态场输出变量的使用，选择输出变量STATUS，并将失败标准设置为移除值小于或等于0.5的单元（单元的状态为1.0表示单元处于活动状态，0.0表示单元处于非活动状态）。值为0.0的单元不会显示在当前视口的模型图中。

选择状态场输出变量时，Abaqus/CAE列出当前[步骤和帧](pt05ch42s01hlb01.md)的[输出数据库](pt02ch09s06hlb02.md)中所有可用的变量（表面场输出变量除外）供您选择。描述左侧的星号表示该变量包含复数结果。有关单个输出变量标识符的信息，请参见["输出变量，" Abaqus Analysis User's Guide第4.2节](../usb/usb-link.md#usbovsect)。

**选择状态场输出变量：**

1. 找到控制状态场输出变量的选项。
   1. 从主菜单栏，选择****Result**![](../graphics/images/arrow.gif)**Field Output****。在出现的对话框中点击**Status Variable**选项卡。
   2. 切换**Use status variable**以禁用或启用状态变量的使用。当**Use status variable**开启时，状态变量被启用，**Status Variable**选项变为可用。**提示：**您也可以通过点击任何出现此按钮的对话框中的**Field Output**按钮来访问这些选项。要查看列出变量的完整描述，请通过拖动角落来增加对话框的宽度。
2. 切换开启**Apply to undeformed state**以将状态场输出变量应用于未变形形状图。如果您的模型不包含变形，您必须开启此设置以应用状态场输出变量。
3. 要控制**Name**和**Description**列表中显示哪些变量：
   1. 切换**List only variables with results**以按变量的存储位置限制显示列表。限制列表有助于您通过例如仅显示积分点量来选择变量。当**List only variables with results**开启时，下拉菜单中会出现筛选选项。
   2. 点击**List only variables with results**箭头显示筛选选项。
   3. 点击**Name**和**Description**列表中要包含的变量位置的文本。文本出现在**List only variables with results**框中，**Name**和**Description**列表会刷新以仅包含具有该位置的变量。
4. 从**Name**和**Description**列表中，点击您想要的分析变量名。列表中描述左侧的星号表示该变量包含复数结果。选中的变量高亮显示。如果适用，对话框底部的**Invariant**和**Component**列表会刷新以分别显示可用的不变量或分量。
5. 如果**Invariant**或**Component**列表中有项目，点击您想要的不变量或分量。选中的不变量或分量高亮显示。
6. 从**Remove elements**字段的筛选方法中选择，以指定失败标准。如果元素的任何结果值满足指定标准，则该元素被视为失败并从模型图中移除。单元输出量与指定标准进行比较；不会对单元数据执行外推或平均。如果选择了节点量且给定节点的值满足指定标准，则连接到该失败节点的所有单元都被移除。
   - 选择![](../graphics/dgtG_ResRangeBelow.png)并在**Max**字段中输入值，以移除值小于或等于指定值的所有单元。
   - 选择![](../graphics/dgtG_ResRangeAbove.png)并在**Min**字段中输入值，以移除值大于或等于指定值的所有单元。
   - 选择![](../graphics/dgtG_ResRangeInside.png)（内部）并在**Min**和**Max**字段中输入范围的上限和下限，以移除值在指定范围内（包括边界）的所有单元。
   - 选择![](../graphics/dgtG_ResRangeOutside.png)（外部）并在**Min**和**Max**字段中输入范围的上限和下限，以移除值在指定范围外的所有单元。
7. 点击**Apply**实现您的更改。当前视口中的模型图会移除所选状态场输出变量值满足指定标准的单元。如果处于活动状态，图例和状态块中的文本会更改以标识与图关联的变量。有关图例和状态块的更多信息，请参见["自定义图例，" 第56.1节](pt05ch56s01.md)和["自定义状态块，" 第56.3节](pt05ch56s03.md)。您的更改将保存到会话结束。
