# 10.4.2 从对称网格或部分三维网格传递结果到完整三维网格

**产品：** Abaqus/Standard  

##### **参考文献**

- ["对称模型生成，" 第 10.4.1 节"](pt04ch10s04aus63.md)
- [*SYMMETRIC RESULTS TRANSFER](../key/key-link.md#usb-kws-maxiresulttransfer)

### 概述

对称结果传递：
- 降低了可能在加载历史中先经历对称变形然后稍后经历非对称变形的结构的分析成本；
- 可用于将轴对称模型的解传递到三维模型；
- 可用于将三维模型对称部分的解传递到完整的三维模型；
- 必须与对称模型生成功能结合使用（请参阅["对称模型生成，" 第 10.4.1 节"](pt04ch10s04aus63.md)）；以及
- 只能用于将应力/位移、热传递、耦合温度-位移或耦合声学-结构分析的解传递到新模型。

### 将解从对称网格或部分三维网格传递到完整三维网格

对称结果传递功能可用于将轴对称模型的解传递到三维模型，或将三维模型对称部分的解传递到完整的三维模型。["对称模型生成，" 第 10.4.1 节"](pt04ch10s04aus63.md) 中描述的对称模型生成功能必须用于生成三维模型。

对称结果传递功能不适用于定义为部件实例装配的模型。

传递到新模型的解由变形构型和相应材料状态组成，包括应变和所有状态变量。节点以其原始坐标导入。此解成为新分析的初始或基础状态。

#### 指定必须读取原始模型中获得解的时间

您指定必须读取原始模型中获得解的时间。所需的步骤和增量或迭代必须在原始分析期间写入重启文件。

| **输入文件用法：** | 如果解是从直接循环过程以外的任何分析传递的，请使用以下选项： |
| --- | --- |
|  | ``` [*SYMMETRIC RESULTS TRANSFER](../key/key-link.md#usb-kws-maxiresulttransfer), STEP=*step*, INC=*increment* ``` 如果解是从先前的直接循环分析传递的，请使用以下选项： ``` [*SYMMETRIC RESULTS TRANSFER](../key/key-link.md#usb-kws-maxiresulttransfer), STEP=*step*, ITERATION=*iteration* ``` |

#### 获得平衡

您必须确保模型在分析开始时处于平衡状态。建议包含初始步骤定义，使用与结果传递来源的模型状态匹配的边界条件和载荷。对于此步骤，应使用等于总时间的初始时间增量，以允许 Abaqus/Standard 尝试在一个增量中实现平衡。如果需要，Abaqus/Standard 可以在线性上解决整个步骤中的应力不平衡，从而使用多个增量。您可以选择改为在步骤的第一个增量中解决应力不平衡。

| **输入文件用法：** | 使用以下选项使 Abaqus/Standard 在线性的整个步骤上解决应力不平衡： |
| --- | --- |
|  | ``` [*SYMMETRIC RESULTS TRANSFER](../key/key-link.md#usb-kws-maxiresulttransfer), UNBALANCED STRESS=RAMP ``` 使用以下选项使 Abaqus/Standard 在步骤的第一个增量中解决应力不平衡： ``` [*SYMMETRIC RESULTS TRANSFER](../key/key-link.md#usb-kws-maxiresulttransfer), UNBALANCED STRESS=STEP ``` |

### 识别重启文件

对称结果传递功能使用旧分析的重启（`.res`）、分析数据库（`.stt` 和 `.mdl`）、部件（`.prt`）和输出数据库（`.odb`）文件将解数据传递到新网格。必须在运行 Abaqus 的命令中使用 **oldjob** 参数或在命令过程发出的请求中回答时指定旧分析的重启文件名称（请参阅["Abaqus/Standard、Abaqus/Explicit 和 Abaqus/CFD 执行，" 第 3.2.2 节"](pt01ch03s02abx02.md)）。

### 验证新模型

建议在传递结果或执行任何分析之前验证新模型是否正确生成。模型生成功能仅需要数据检查运行期间存储在重启文件中的信息来生成新模型，这允许您在执行原始模型的分析之前验证新模型。数据检查分析通过在运行 Abaqus 的命令中使用 **datacheck** 参数来执行（请参阅["Abaqus/Standard、Abaqus/Explicit 和 Abaqus/CFD 执行，" 第 3.2.2 节"](pt01ch03s02abx02.md)）。

一旦模型被验证，原始模型的分析就可以执行，结果可以传递到新模型。

传递的解可以通过在步骤开始时请求输出（零增量；请参阅["输出，" 第 4.1.1 节"](pt02ch04s01aus38.md)）写入结果文件。此解也可以在 Abaqus/CAE 中查看。

### 方向系统

当结果从轴对称模型传递到三维模型时，局部圆柱坐标系用于应力、应变等的单元输出。如果原始轴对称模型中的材料不包含方向定义，则提供默认局部方向定义（["方向，" 第 2.2.5 节"](pt01ch02s02aus15.md)）。此默认方向定义为系统的极轴沿旋转轴，并额外绕局部 1 方向旋转 90，因此局部轴为 1=径向、2=轴向、3=周向。如果使用壳或膜，则将局部 2 和 3 轴投影到壳或膜表面上作为表面上的局部方向。假定材料特性在此系统中指定。另一方面，如果在原始模型中的材料上关联了方向定义，则新三维模型中的方向将是绕对称轴旋转的方向定义。

当通过反射部分三维模型将结果从部分三维模型传递到完整三维模型时，会在完整三维模型中创建基于部分三维模型中相应方向定义的局部材料方向。但是，如果在部分三维模型中材料不包含方向定义，并且部分三维模型不是通过旋转轴对称模型创建的，则不会在完整三维模型中创建局部方向定义。完整三维模型使用全局坐标系。

当通过绕其对称轴旋转三维扇区将结果从三维扇区传递到周期三维模型时，局部圆柱坐标系始终用于应力、应变等的单元输出。如果在原始三维扇区中指定了方向，则新模型中的方向系统通过绕对称轴旋转原始方向系统来定义。如果使用壳或膜，则将局部 2 和 3 轴投影到壳或膜表面上作为表面上的局部方向。如果原始三维扇区中的材料不包含方向定义，则提供默认局部方向定义。此默认方向通过绕新模型中的对称轴旋转原始模型中的全局坐标系来定义。

### 节点处的坐标系

从原始模型获得的位移和旋转分量在传递结果之前首先转换为全局直角笛卡尔轴系统。如果新模型中需要局部坐标方向，则必须在 new 模型中指定节点变换（["变换坐标系，" 第 2.1.5 节"](pt01ch02s01aus09.md)）来定义此坐标系。

### 限制

从轴对称模型到三维模型的结果传递存在以下限制：
- 当单元是接触对应中从表面下的底层时，不支持从 8 节点减缩积分轴对称单元（CAX8R 和 CAX8RH）到相应 20 节点六面体单元（C3D20R 和 C3D20RH）的结果传递。
- SAX2 是有限应变壳，而 S8R 是小应变壳。当原始分析中变形很大时，不要使用此组合。

从对称三维模型到完整三维模型的结果传递存在以下限制：
- 不支持具有每节点五个自由度的壳的结果传递（STRI65、S8R5 和 S9R5）。

### 初始条件

初始条件（请参阅["Abaqus/Standard 和 Abaqus/Explicit 中的初始条件，" 第 34.2.1 节"](pt07ch34s02aus116.md)）可以在所有节点和单元上指定，包括使用对称模型生成生成的模型部分（请参阅["对称模型生成，" 第 10.4.1 节"](pt04ch10s04aus63.md)）。但是，在大多数情况下，对称结果传递功能将用从原始模型获得的解覆盖初始条件值。初始温度和场变量是例外。初始温度和场变量仅在原始模型中指定了温度和场变量时被覆盖。如果只有部分原始模型包含温度或场变量的规范，则模型的其余部分假定具有零大小的初始条件。此场分布将被传递到新模型。如果在原始模型中的任何地方都未定义温度和/或场变量，则应用新模型中指定的初始条件。

### 边界条件

所有边界条件必须重新定义；对称结果传递功能忽略原始模型中指定的边界条件。您必须确保模型在分析开始时处于平衡状态；因此，应包含初始步骤定义，使用与结果传递来源的模型状态匹配的边界条件和载荷。

### 载荷

所有载荷必须重新定义；对称结果传递功能忽略原始模型中指定的载荷。您必须确保模型在分析开始时处于平衡状态；因此，应包含初始步骤定义，使用与结果传递来源的模型状态匹配的边界条件和载荷。

### 材料选项

原始模型中定义的所有材料定义将被传递到新模型。

### 单元

任何在原始模型中处于活动状态的单元或接触对移除/重新激活定义（请参阅["单元和接触对移除和重新激活，" 第 11.2.1 节"](pt04ch11s02aus66.md)）都应重新指定。

### 输出

应力/位移单元可用的所有标准输出变量可以与对称结果传递功能一起使用。

传递到新模型的解可以通过在步骤开始时请求输出（零增量；请参阅["输出，" 第 4.1.1 节"](pt02ch04s01aus38.md)）写入结果（`.fil`）文件。它也可以在 Abaqus/CAE 中显示。
