### 概述

变形理论Ramberg-Osgood塑性模型：
- 主要用于为延性金属断裂力学应用开发完全塑性解；和
- 不能与其他机械响应材料模型一起出现，因为它完全描述了材料的机械响应。

### 一维模型

在一维中，模型为：

ε/E = σ/E + α (σ/σ^0^)^n

其中σ是应力；ε是应变；E是杨氏模量（定义为应力-应变曲线在零应力处的斜率）；α是"屈服"偏移；σ^0^是屈服应力，意义在于当σ/σ^0^ = 1时，ε/E = 1 + α；且n是"塑性"（非线性）项的硬化指数：ε^p^/ε^0^ = (σ/σ^0^)^n。

该模型描述的材料行为在所有应力水平下都是非线性的，但对于常用的硬化指数值（n ≥ 10或更多），非线性仅在应力大小接近或超过σ^0^时变得显著。

### 多轴应力状态的推广

一维模型使用胡克定律用于线性项和Mises应力势及相关流动定律用于非线性项推广到多轴应力状态：

ε~ij~ = (1+ν)/E σ~ij~ - ν/E σ~kk~ δ~ij~ + 3/2 (1-2ν)/E (σ~ij~ - 1/3 σ~kk~ δ~ij~) [1 + α (σ¯/σ^0^)^(n-1)] - 1/3 (σ~ij~ - 1/3 σ~kk~ δ~ij~)/σ^0^ (ε^p)^n-1 δ~ij~

其中ε~ij~是应变张量；σ~ij~是应力张量；σ~m~ = 1/3 σ~kk~是等效静水应力；σ¯ = √(3/2 S~ij~ S~ij~)是Mises等效应力；S~ij~ = σ~ij~ - σ~m~ δ~ij~是应力偏量；且ν是泊松比。

行为的线性部分可以是可压缩或不可压缩的，取决于泊松比的值，但行为的非线性部分是不可压缩的（因为流动垂直于Mises应力势）。该模型在["变形塑性，" Abaqus理论指南第4.3.9节](../stm/stm-link.md#stm-mat-deformplast)中详细描述。

您直接指定参数E、ν、σ^0^、n和α。它们可以定义为温度的表格函数。

### 典型应用

变形塑性模型最常用于小位移分析的静态加载，其中必须在模型的一部分中开发完全塑性解。通常，加载斜坡上升，直到被监测区域中的所有点满足"塑性应变"占主导的条件，因此表现出完全塑性行为，定义为：

|σ¯ - σ^0^| / σ^0^ >> 1

或等效地：

ε^p^ / ε >> 1

您可以指定要在静态分析步骤中监测完全塑性行为的特定单元集的名称。当该单元集中所有本构计算点的解都是完全塑性时、当达到步骤指定的最大增量数时、或当超过为静态步骤指定的时间段时（以先到者为准），步骤结束。

### 单元

变形塑性可用于Abaqus/Standard中的任何应力/位移单元。由于它通常用于变形以塑性流动为主的案例，建议将此材料模型与"混合"（混合公式）或减缩积分单元一起使用。
