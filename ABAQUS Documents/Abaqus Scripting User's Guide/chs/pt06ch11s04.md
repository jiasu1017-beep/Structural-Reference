# 11.4 安装





你可以使用更简单的语法来启动已安装在站点的应用程序。安装应用程序涉及以下步骤：
- 在`*abaqus_dir*/SMA/site/abaqus.app`文件中添加一个条目，其中*abaqus_dir*是安装Abaqus的目录名称。要确定你站点的*abaqus_dir*位置，请在操作系统提示符下输入`abaqus whereami`。`abaqus.app`文件中条目的格式为：``` *applicationName* cae | viewer -custom *startupScript* ```其中*applicationName*是用户必须在命令行上指定以启动应用程序的名称。第二个参数决定应用程序将检出的令牌类型——`cae`或`viewer`。*startupScript*是启动脚本的名称，不带任何文件扩展名。脚本必须位于`PYTHONPATH`环境变量指定的目录中。*applicationName*和*startupScript*可以相同。
- 编辑`*abaqus_dir*/SMA/site/abaqus.aev`文件。你必须将包含自定义脚本的目录添加到`PYTHONPATH`环境变量的定义中。按照惯例，自定义脚本位于`*abaqus_dir*/customApps`下的目录中。你应该将你的目录添加到`PYTHONPATH`定义靠近末尾的位置，就在当前目录`(.)`之前。这将确保你不会覆盖`PYTHONPATH`定义中的任何现有设置。为了保持应用程序路径的可移植性和通用性，你应该使用环境变量来指定路径的根目录。对于标准Abaqus安装，`$ABA_HOME`环境变量指向与*abaqus_dir*相同的目录。因此，你可以使用`$ABA_HOME`环境变量来指定包含自定义脚本的目录；例如：``` $ABA_HOME/customApps/myApp ```例如，要将上面显示的`myApp`目录包含在`PYTHONPATH`中，你应该将`abaqus.aev`中的`PYTHONPATH`定义从：``` PYTHONPATH $ABA_HOME/cae/Python/Lib:$ABA_HOME/cae/Python/Obj: $ABA_HOME/cae/exec/lbr:.:$PYTHONPATH ```更改为：``` PYTHONPATH $ABA_HOME/cae/Python/Lib:$ABA_HOME/cae/Python/Obj: $ABA_HOME/cae/exec/lbr:$ABA_HOME/customApps/myApp: .:$PYTHONPATH ```
- UNIX系统和Windows系统之间存在语法差异。默认情况下，Abaqus使用UNIX语法，如果应用程序在Windows平台上运行，会自动将UNIX语法转换为Windows语法。但是，如果需要在Windows系统上指定路径的驱动器字母，必须使用Windows语法。要使用Windows语法，必须对整个`PYTHONPATH`行进行以下更改：| UNIX | Windows | | --- | --- | | `:` | `;` | | `/` | `\` | | `$*NAME*` | `%*NAME*%` | 以下示例显示了一个`abaqus.aev`文件，该文件引用了驱动器字母并已修改为在Windows系统上运行：``` ABA_PATH $ABA_HOME:$ABA_HOME/cae PYTHONPATH %ABA_HOME%\cae\Python\Lib;%ABA_HOME%\cae\Python\Obj; %ABA_HOME%\cae\exec\lbr;d:\boltApp1;.;%PYTHONPATH% ABA_LIBRARY_PATH $ABA_HOME/cae/ABA_SELECT: $ABA_HOME/cae/exec/lbr:$ABA_HOME/cae/Python/Obj/lbr: $ABA_HOME/cae/External/Acis:$ABA_HOME/cae/External: $ABA_HOME/cae/External/ebt:$ABA_HOME/exec ```使用以下语法启动你的应用程序：``` abaqus applicationName ```
