# 64.6.8 分割四边形单元





从主菜单栏选择 ****网格**![](../graphics/images/arrow.gif)**编辑**** 可以将四边形单元分割成两个三角形单元，如下图所示：

![](../graphics/edt-meshedit-splitquad.png)

您不能分割5节点四边形单元，也不能分割四边形垫片单元。只要四边形单元来自同一零件，您就可以选择多个四边形单元进行分割。

**分割四边形单元的步骤：**

1. 进入"网格"模块，并执行以下操作之一： - 从上下文栏的 **对象** 字段中选择 **装配**。 - 从上下文栏的 **对象** 字段中选择 **零件** 并从列表中选择一个零件。
2. 从主菜单栏选择 ****网格**![](../graphics/images/arrow.gif)**编辑****。Abaqus/CAE 显示 **编辑网格** 对话框。 **提示：** 您也可以使用位于"网格"模块工具箱底部的 ![](../graphics/ico_meshEdit.png) 工具显示 **编辑网格** 对话框。
3. 在对话框中执行以下操作： 1. 在 **类别** 字段中选择 **单元**。 2. 从 **方法** 列表中选择 **分割（四边转三角）**。
4. 选择要分割的四边形单元。您可以使用 ["在当前视口中选择对象，" 第6.2节](pt01ch06s02.md) 中描述的任何选择技术。如果您的选择包含非四边形的单元，Abaqus/CAE 会自动将其从选择中移除。Abaqus/CAE 会高亮显示它将用于将每个四边形单元分割成两个三角形单元的对角线，并提示您继续。Abaqus/CAE 会选择将产生基于角度测量的最佳形状单元的对角线。您不能选择 Abaqus/CAE 用于分割单元的四边形的对角线。但是，在 Abaqus/CAE 分割单元之后，您可以使用 **交换对角线（三角）** 工具更改对角线；有关更多信息，请参见 ["交换一对相邻三角形单元的对角线，" 第64.6.7节](pt06ch64s06s07.md)。
5. 点击 **是** 以分割所选单元。Abaqus/CAE 会提示您选择更多要分割的单元。 **提示：** 如果在分割四边形单元时出错，点击 **编辑网格** 对话框中的 **撤销** 可以撤消更改。
6. 完成单元分割后，点击鼠标按钮2或提示区域中的取消按钮 ![](../graphics/afxI_cancel.png) 退出过程。
