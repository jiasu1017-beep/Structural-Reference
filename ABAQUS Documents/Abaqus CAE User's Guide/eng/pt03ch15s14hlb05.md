# 15.14.7 Defining an incident wave interaction property







You can define the speed of the incident wave and other characteristics of the wave loading. For spherical incident wave loading, you can optionally specify the loading effects due to an UNDEX bubble or spatial decay of the incident wave field. An incident wave interaction property can be referred to only by an incident wave interaction. Incident wave interaction properties that specify UNDEX data can be referred to only by incident wave interactions with an UNDEX definition. For more information, see ["Defining incident waves," Section 15.13.18](pt03ch15s13hlb16.md).

**To define an incident wave interaction property:**

1. From the main menu bar, select ****Interaction**![](../graphics/images/arrow.gif)**Property**![](../graphics/images/arrow.gif)**Create****.
2. In the **Create Interaction Property** dialog box that appears, do the following: - Name the interaction property. For more information about naming objects, see ["Using basic dialog box components," Section 3.2.1](pt01ch03s02s01.md). - Select the **Incident wave** type of interaction property.
3. Click **Continue** to close the **Create Interaction Property** dialog box.
4. In the **Speed of sound in fluid** field, enter the sound speed in fluid, ![](../graphics/usi_eqn00610.gif).
5. In the **Fluid density** field, enter fluid mass density, ![](../graphics/usi_eqn00611.gif).
6. From the **Definition** field, choose the type of wave to define the incident wave property. - Select **Planar** to specify a planar incident wave. - Select **Spherical** to specify a spherical incident wave. - Select **Diffuse** to specify a field of planar waves incident from multiple angles. - Select **Air blast** to specify air blast loading on structures using the CONWEP model. - Select **Surface blast** to specify surface blast loading on structures using the CONWEP model.
7. If you are specifying a spherical incident wave, select the propagation model. For more information, see ["Describing incident wave loading" in "Acoustic and shock loads," Section 34.4.6 of the Abaqus Analysis User's Guide](../usb/usb-link.md#usb-prc-pacoustic-incidentwave-describe). - Select **Acoustic** to specify an incident wave in which the amplitude is inversely proportional to the distance from the source. - Select **UNDEX charge** to specify UNDEX bubble data. - Select **Generalized decay** to specify spatial decay of the incident wave field.
8. If you selected the **UNDEX charge** propagation model, click the **Physical**, **Material**, and **Bubble Model** tabs and specify the following values: ****Physical Data**** - Ratio of specific heats for gas, ![](../graphics/usi_eqn00102.gif). - Acceleration due to gravity, *g*. - Atmospheric pressure, ![](../graphics/usi_eqn00612.gif). - Flow drag coefficient, ![](../graphics/usi_eqn00613.gif). - Flow drag exponent, ![](../graphics/usi_eqn00614.gif). - Toggle on **Neglect wave effects in fluid and gas** to neglect the wave effects. ****Material Data**** - Charge material constant, *K*. - Charge material constant, *k*. - Charge material constant, *A*. - Charge material constant, *B*. - Adiabatic charge constant, ![](../graphics/usi_eqn00175.gif). - Density of charge material, ![](../graphics/usi_eqn00615.gif). - Mass of charge material, ![](../graphics/usi_eqn00616.gif). ****Bubble Model Step Data**** - Time duration, ![](../graphics/usi_eqn00617.gif). - Maximum number of time steps for the bubble simulation, ![](../graphics/usi_eqn00618.gif). The bubble amplitude simulation ceases when the number of steps reaches ![](../graphics/usi_eqn00618.gif) or the time duration, ![](../graphics/usi_eqn00617.gif), is reached. - Relative step size control parameter, ![](../graphics/usi_eqn00619.gif). - Absolute step size control parameter, ![](../graphics/usi_eqn00620.gif). - Step size control exponent, ![](../graphics/usi_eqn00621.gif). The step size, ![](../graphics/usi_eqn00622.gif), is decreased or increased according to the error estimate: ![](../graphics/usi_eqn00623.gif).
9. If you selected the **Generalized decay** propagation model, enter values for the dimensionless constants **A**, **B**, and **C**, where ![](../graphics/usi_eqn00624.gif), ![](../graphics/usi_eqn00625.gif), and ![](../graphics/usi_eqn00626.gif).
10. If you are defining a diffuse incident wave, specify the seed number in the **Seed number** field for the diffuse source calculation.
11. If you selected the **Air blast** or **Surface blast** definition, specify the following settings in the **CONWEP Charge** portion of the dialog box: - In the **Equivalent mass of TNT** field, specify the equivalent mass of TNT in any preferred mass unit. - In the **Conversion for mass to kilograms** field, specify a multiplication factor to convert the preferred mass unit to kilograms. - In the **Conversion for length to meters** field, specify a multiplication factor to convert the analysis length unit to meters. - In the **Conversion for time to seconds** field, specify a multiplication factor to convert the analysis time unit to seconds. - In the **Conversion for pressure to pascals** field, specify a multiplication factor to convert the analysis pressure unit to pascals.
12. Click **OK** to create the incident wave interaction property and to exit the editor.
