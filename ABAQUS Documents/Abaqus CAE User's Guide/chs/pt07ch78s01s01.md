# 78.1.1 理解如何创建显示组





显示组可以包含模型组件的组合：零件实例、几何图形（单元、面或边）、基准几何（点、轴、平面或坐标系）、单元、节点、曲面、约束、输出数据库坐标系或整个模型。此外，您可以通过对先前保存的显示组进行操作来创建显示组。但是，在创建显示组时，您一次只能对一种类型的模型组件执行操作。创建包含多种类型所选组件的显示组是一个增量过程。可用于创建显示组的模型组件取决于您正在工作的模块，如[表78-1](pt07ch78s01s01.md#uss-dgp-table-by-module-part-assembly)和[表78-2](pt07ch78s01s01.md#uss-dgp-table-by-module-vis)所示。

要创建显示组，首先选择您感兴趣的特定项目。然后，您可以对选择和当前视口内容执行布尔运算。可以根据需要重复此顺序以创建所需的组。此外，您可以通过编辑（例如，对先前保存的显示组的内容执行额外的布尔运算）来创建新的显示组。

您可以保存对话框中的选择或当前视口的内容作为显示组。默认情况下，显示组仅在会话期间持续存在。如果您想在后续会话中保留显示组，请将其保存到XML文件、模型数据库或输出数据库；有关更多信息，请参见["管理会话对象和会话选项，"第9.9节](pt02ch09s08.md)。您只能在与创建它的模块相同类型的模块中访问显示组（请参见[表78-1](pt07ch78s01s01.md#uss-dgp-table-by-module-part-assembly)和[表78-2](pt07ch78s01s01.md#uss-dgp-table-by-module-vis)）。例如，如果您在零件模块中创建并保存了一个显示组，您将只能在该零件模块和属性模块中访问此显示组。

Abaqus/CAE会自动创建一个名为`All`的显示组，其中包含当前零件、装配或输出数据库中的所有对象。此显示组出现在当前模块的显示组管理器中，无法编辑、复制、重命名或删除。它不与特定的零件、装配或输出数据库关联。您可以使用此显示组快速返回到执行布尔运算后整个零件或模型的绘图。
