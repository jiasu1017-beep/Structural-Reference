### **必需参数：**

LOCATION

设置 LOCATION=ELEMENT 以在单元上定义分布。

设置 LOCATION=NODE 以在节点上定义分布。

设置 LOCATION=NONE 以定义与流体边界条件一起使用的分布。

NAME

将此参数设置为一个标签，用于引用分布。

TABLE

将此参数设置为定义数据行上数据格式的分布表。

### **可选参数：**

INPUT

将此参数设置为包含此选项数据行的备用输入文件名称。请参见Abaqus分析用户指南第1.2.1节中的["输入语法规则"](../usb/usb-link.md#usb-int-iinputsyntax)，了解此类文件名的语法。如果省略此参数，则假定数据跟在关键字行之后。
