# 49.2.3 生成谐波动画





谐波动画从输出数据库的单个步骤和帧生成图像序列。这些图像可以由变形形状、等值线或符号图组成。对于变形形状图以及变形形状上的等值线和符号图，图像随施加到变形值的角度变化。对于等值线和符号图，图像随施加到场输出变量的角度变化；结果的实际和虚部都被使用。

默认情况下，Abaqus/CAE在谐波动画期间显示帧计数器。帧计数器指示查看复数结果时的当前角度。

**生成谐波动画：**

1. 使用"文件"菜单[打开包含分析结果的输出数据库](pt02ch09s06hlb02.md)。
2. 使用"结果"菜单选择要绘制的分析结果，包括步骤和帧、变形场输出变量，以及对于等值线或符号图动画，要绘制的[主场输出变量](pt05ch42s03hlb03.md)或[符号场输出变量](pt05ch42s03hlb05.md)。有关场输出变量选择的更多信息，请参阅["场输出变量选择概述，"第42.5.1节](pt05ch42s03hlb01.md)。
3. 使用"选项"和"视图"菜单选择您想要的[自定义选项](pt05ch40s03s03.md)。
4. 要启动动画，请从主菜单栏中选择****动画**![](../graphics/images/arrow.gif)**谐波****。**提示：**您也可以使用工具箱中的![](../graphics/ico_animHarmonic.png)工具来生成谐波动画。动画在所有动画化视口中开始，动画控件出现在提示区域。Abaqus现在处于谐波动画绘图状态。在此状态下，每次您单击"结果"、"选项"或"视图"菜单对话框中的"应用"或"确定"时，Abaqus会自动刷新您的图。所有动画化视口将保持谐波动画状态，直到您在其他（例如，未变形、变形）状态中生成图。有关自定义基础模型形状的更多信息，请参阅["自定义模型形状，"第55.4节](pt05ch55s04.md)。
