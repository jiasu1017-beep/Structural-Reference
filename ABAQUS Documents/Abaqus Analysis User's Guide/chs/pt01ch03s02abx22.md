# 3.2.22 组合子结构输出





**产品：**Abaqus/Standard  Abaqus/Explicit  

##### **参考**

- ["Abaqus执行程序：概述，" 3.1.1节](pt01ch03s01abo02.md)
- ["获取子结构内的结果输出" 10.1.1节"使用子结构"](pt04ch10s01aus58.md#usb-anl-asuperelements-output)

### 概述

**abaqus substructurecombine**工具将模型的两个子结构生成的模型和结果数据组合到单个输出数据库（`.odb`）文件中。通过组合模型的所有子结构分析输出数据库文件，您可以在Abaqus/CAE中显示子结构分析生成的所有数据。

Abaqus通过将从第二个指定的文件（副本输出数据库）的内容直接添加到第一个指定的文件（基础输出数据库）来组合输出数据。因为此过程会更改基础输出数据库，所以在使用此工具之前请考虑备份您的数据。

### 组合具有两个以上子结构的模型的数据

因为**abaqus substructurecombine**工具一次仅组合两个输出数据库，所以对于具有两个以上子结构的分析，您必须多次运行该工具才能从所有子结构创建单个输出数据库。首先组合两个子结构的数据，然后重复操作将生成的输出数据库文件与每个剩余子结构的数据组合。

### 定制组合输出数据库

您可以通过仅将副本输出数据库中的数据子集添加到基础输出数据库来定制子结构组合操作。Abaqus使您能够从副本输出数据库的单个步骤或帧添加输出数据到基础输出数据库。您还可以仅包含与特定变量相关的副本输出数据库中的输出数据；例如，您可以复制与Mises应力相关的输出数据。

### 命令行选项

**baseodb**

此选项指定Abaqus向其中添加副本输出数据库内容的基础输出数据库的名称。

如果命令行中省略此选项，Abaqus将提示您输入其值。

**copyodb**

此选项指定Abaqus将其内容添加到基础输出数据库的副本输出数据库的名称。您一次只能为此选项指定一个文件。

如果命令行中省略此选项，Abaqus将提示您输入其值。

**all**

此选项指示所有步骤和帧的输出数据应复制到组合输出数据库。当指定此选项时，Abaqus会忽略**step**、**frame**和**variable**选项。

**step**

此选项指示Abaqus将从中复制结果数据的步骤名称。您只能指定一个步骤；如果省略此选项，Abaqus会从输出数据库的最后一步复制数据。

如果指定了**all**选项，Abaqus会忽略此选项。

**frame**

此选项指示Abaqus将从中复制结果数据的帧号。您只能指定一帧；如果省略此选项，Abaqus会使用**step**选项指定的步骤中的最后一帧。

如果指定了**all**选项，Abaqus会忽略此选项。

**variable**

此选项指示Abaqus将从中复制结果数据的变量变量键。如果省略此选项，Abaqus会复制输出数据库中所有变量的数据。如果指定了**all**选项，Abaqus会忽略此选项。

只有对输出数据库文件输出有效的输出变量键才可用于**abaqus substructurecombine**。通常，如果一个键对应于集合输出变量而不是单个分量，则可以与此执行程序一起使用。集合输出变量键与其单个分量的区别在于，它们在["Abaqus/Standard输出变量标识符，" 4.2.1节](pt02ch04s02abv01.md)表格中的某个**`.odb`**类别后有"yes"一词。

### 示例

以下示例说明使用**abaqus substructurecombine**执行程序组合子结构的不同方法。

#### 组合两个子结构

如果您的模型包含生成名为`subst1.odb`和`subst2.odb`的输出数据库文件的两个子结构，请发出以下命令，用两个文件的组合内容覆盖`subst1.odb`：

```
abaqus substructureCombine baseodb=subst1.odb copyodb=subst2.odb
```

#### 组合两个以上子结构

如果您的模型包含两个以上的子结构，您必须首先组合两个子结构的输出数据库文件，然后将组合输出数据库与每个其他子结构的输出数据库依次组合。在这个示例中，子结构分析生成四个名为`subst1.odb`、`subst2.odb`、`subst3.odb`和`subst4.odb`的输出数据库文件，因此您必须发出**abaqus substructure**命令总共三次才能将所有四个文件组合成单个输出数据库，如下例所示：

```
abaqus substructureCombine baseodb=subst1.odb copyodb=subst2.odb
abaqus substructureCombine baseodb=subst1.odb copyodb=subst3.odb
abaqus substructureCombine baseodb=subst1.odb copyodb=subst4.odb
```

#### 组合子结构的特定元素

如果您只想在组合输出数据库中包含**Step-1**步骤的输出数据，请发出以下命令：

```
abaqus substructureCombine baseodb=subst1.odb copyodb=subst2.odb
   step="Step-1"
```
如果您只想在组合输出数据库中包含**Mises**变量的输出数据，请发出以下命令：
```
abaqus substructureCombine baseodb=subst1.odb copyodb=subst2.odb
   variable="Mises"
```
