# 64.6.6 分割四边形或三角形单元的边缘





从主菜单栏选择 ****网格**![](../graphics/images/arrow.gif)**编辑**** 可以将四边形或三角形单元的边缘分割成两部分。以下图形显示了如何分割边缘然后折叠生成的边缘以移除狭长的三角形单元：

![](../graphics/edt-meshedit-splitedge-nls.png)

当您分割四边形单元时，Abaqus/CAE 会创建基于角度测量将产生最佳形状单元的边缘。您不能选择 Abaqus/CAE 将创建的边缘。但是，在 Abaqus/CAE 分割边缘之后，您可以使用 **交换对角线（三角）** 工具更改对角线；有关更多信息，请参见 ["交换一对相邻三角形单元的对角线，" 第64.6.7节](pt06ch64s06s07.md)。

**分割四边形或三角形单元边缘的步骤：**

1. 进入"网格"模块，并执行以下操作之一： - 从上下文栏的 **对象** 字段中选择 **装配**。 - 从上下文栏的 **对象** 字段中选择 **零件** 并从列表中选择一个零件。
2. 从主菜单栏选择 ****网格**![](../graphics/images/arrow.gif)**编辑****。Abaqus/CAE 显示 **编辑网格** 对话框。 **提示：** 您也可以使用位于"网格"模块工具箱底部的 ![](../graphics/ico_meshEdit.png) 工具显示 **编辑网格** 对话框。
3. 在对话框中执行以下操作： 1. 在 **类别** 字段中选择 **单元**。 2. 从 **方法** 列表中选择 **分割边缘（三角/四边）**。
4. 从提示区域的菜单中，选择用于确定沿边缘分割位置的方法。您可以选择 **中点** 或 **选择位置**。
5. 如果您选择 **中点** 来确定位置，请选择要分割的边缘。您只能选择一个边缘。Abaqus/CAE 会高亮显示它将插入以在中点分割所选边缘的边缘。
6. 如果您选择 **选择位置** 来确定位置，请在所需分割位置点击边缘。Abaqus/CAE 会高亮显示它将插入以在选择位置分割所选边缘的边缘。
7. 从提示区域点击 **分割边缘**。Abaqus/CAE 会分割边缘并提示您选择下一个要分割的边缘。 **提示：** 如果在分割单元边缘时出错，点击 **编辑网格** 对话框中的 **撤销** 可以撤消分割。
8. 根据需要重复上述步骤以分割其他边缘。
9. 完成边缘分割后，点击鼠标按钮2或提示区域中的取消按钮 ![](../graphics/afxI_cancel.png) 退出过程。
