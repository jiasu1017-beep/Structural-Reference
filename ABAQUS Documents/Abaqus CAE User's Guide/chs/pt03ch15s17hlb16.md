# 15.17.16 指定势项

连接器势是用户定义的数学函数，表示连接器中相对运动分量空间中的屈服面、极限面或大小量度。您使用势来定义耦合摩擦、塑性或损伤行为选项。有关更多信息，请参见["耦合行为的连接器函数，"Abaqus分析用户指南第31.2.4节](../usb/usb-link.md#usb-elm-econnderivedandpotential)。

要显示势贡献编辑器中选项的上下文相关帮助，您必须选择感兴趣的选项，然后按**[F1]**。（编辑器显示时，主菜单栏中的**帮助**菜单不可用。）

**要创建或编辑势项：**

1. 显示势标签页。
   - 对于摩擦行为，请使用["定义摩擦，"第15.17.3节"](pt03ch15s17hlb03.md)中概述的过程显示**力势**标签页。
   - 对于塑性行为，请使用["定义塑性，"第15.17.6节"](pt03ch15s17hlb06.md)中概述的过程显示**力势**标签页。
   - 对于损伤行为，请使用["定义损伤，"第15.17.7节"](pt03ch15s17hlb07.md)和["定义损伤演化，"第15.17.8节"](pt03ch15s17hlb08.md)中概述的过程显示**起始势**和**演化势**标签页。
2. 选择**运算符**。
   - **和** 将势定义为贡献之和。
   - **最大值** 将势定义为产生最大值的贡献。
3. 如果您在**运算符**字段中选择了**和**，请在**指数**字段中输入正值，以指定势定义中整体指数的倒数，![](../graphics/usi_eqn00056.gif)，该指数定义了势的一般椭圆形式。默认值`2`定义了势的二次形式。
4. 在标签页的**势贡献**部分，选择以下选项之一：
   - 要定义势的贡献，![](../graphics/usi_eqn00636.gif)，请点击![](../graphics/ico_add.png)。势贡献编辑器出现。
   - 要编辑现有贡献，请从列表中选择一个贡献，然后点击![](../graphics/ico_edit.png)。势贡献编辑器出现。
   - 要删除现有贡献，请从列表中选择一个贡献，然后点击![](../graphics/ico_delete.png)。
5. 在势贡献编辑器中，执行以下操作：
   1. 指定用于势贡献的分量。
      - **指定分量** 选择可用相对运动分量以定义势贡献。
      - **指定导出分量** 点击![](../graphics/ico_edit.png)以指定连接器导出分量来定义势贡献。有关更多信息，请参见["指定连接器导出分量，"第15.17.15节"](pt03ch15s17hlb15.md)。
   2. 选择**符号**。
      - **正** 为势贡献选择正号。
      - **负** 为势贡献选择负号。
   3. 输入以下数据：
      - **缩放因子** 输入非零缩放因子*R*。默认值为1。
      - **正指数** 输入正指数的值![](../graphics/usi_eqn00007.gif)。如果您在**运算符**字段中选择了**最大值**，则正指数值将被忽略。
      - **偏移因子** 输入偏移因子*a*。默认值为0。
      - **H函数** 选择用于生成贡献的函数*H*。默认值为**ABS**。
        - **ABS**。使用绝对值函数来生成贡献。
        - **MACAULEY**。使用Macauley括号（![](../graphics/usi_eqn00637.gif)）来生成贡献。
        - **IDENTITY**。仅在以下情况为真时选择恒等函数：
          - **正指数**（![](../graphics/usi_eqn00007.gif)）= **指数**（![](../graphics/usi_eqn00056.gif)）= 1
