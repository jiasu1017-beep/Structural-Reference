# 44.5.6 选择云图方法





云图值使用纹理映射或镶嵌方法绘制在单元面上。纹理映射是默认和首选方法，可以最大化Abaqus/CAE的性能。尽管某些图形适配器不能正确支持纹理映射，但Abaqus/CAE可以在软件中模拟纹理映射。但是，某些显示选项仅受镶嵌方法支持；在这种情况下，Abaqus/CAE将自动覆盖纹理映射云图的用户设置。

**要选择云图方法：**

1. 找到**Contour Method**选项。从主菜单栏中选择****Options**![](../graphics/images/arrow.gif)**Contour****，或单击工具箱中的![](../graphics/ico_plotContourOptions.png)；然后在出现的对话框中单击**Basic**选项卡。**Contour Method**选项位于页面底部。
2. 选择以下渲染方法之一：- **Texture-mapped**。此选项使用高性能图形渲染功能在每个单元面上绘制云图值。如果系统的图形参数已配置为禁用纹理映射（请参见["调整图形卡，" Abaqus安装和许可指南第5.3节](../sgb/sgb-link.md#sgb-chp-view-tune)），Abaqus/CAE将在软件中模拟纹理映射。- **Tessellated**。此选项使用镶嵌过程在每个单元面上绘制云图值。
3. 单击**Apply**实施您的更改。当前视口中的云图更改为使用您指定的云图方法。默认情况下，您的更改在会话期间保存，并将影响所有后续云图。如果您想在后续会话中保留更改，请将它们保存到文件。有关更多信息，请参见["保存自定义设置以供后续会话使用，" 第55.1.1节](pt05ch55s01s01.md)。
