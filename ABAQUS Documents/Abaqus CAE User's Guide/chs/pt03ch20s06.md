# 20.6 指定精确几何





当您使用Sketcher绘制特征时，可以使用底层网格和预选择来定位光标并创建或修改草图几何。 此外，您可以使用以下技术来精确定位几何：

**输入坐标**

当您使用Sketcher创建特征且Abaqus/CAE要求您定位点时（例如，定义线的端点或圆心），您可以执行以下操作之一：
- 使用Sketcher网格、现有顶点或预选择来选择草图上的点。
- 在提示区域的文本框中输入点的所需X和Y坐标。 为帮助您确定要提供的坐标，草图的X和Y轴由虚线指示，虚线在草图原点处相交。 您指定的坐标是相对于此原点的。 此外，当您选择Sketcher工具时，当前光标位置的坐标会显示在视口左上角，您可以使用这些坐标作为指导。

有关详细说明，请参阅["绘制简单对象，" 第20.10节](pt03ch20s10.md)。

**创建和修改尺寸**

创建几何后，Sketcher允许您在线和顶点以及草图中的其他几何之间添加尺寸。 然后您可以通过将这些尺寸替换为精确尺寸来细化草图，草图会反映新的尺寸。 此外，尺寸允许您为将来参考标注草图。

您可以通过在Sketcher工具箱中选择尺寸工具 ![](../graphics/ico_sketchDimOblique.png) 或通过从主菜单栏选择 ****Add**![](../graphics/images/arrow.gif)**Dimension**** 来添加尺寸——编辑尺寸值是创建新尺寸的最后一步。 要修改现有尺寸，请从Sketcher工具箱中选择修改尺寸工具 ![](../graphics/ico_sketchDimensionEdit.png) 或从主菜单栏选择 ****Edit**![](../graphics/images/arrow.gif)**Dimension****。 有关详细说明，请参阅["编辑尺寸，" 第20.13节](pt03ch20hla01.md)。
