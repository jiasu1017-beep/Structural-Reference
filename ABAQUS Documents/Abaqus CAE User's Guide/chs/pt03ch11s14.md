# 11.14 什么是放样？





放样是一种允许您创建无法通过拉伸、旋转或扫掠创建的复杂三维特征的方法。例如，由于横截面不同，您可以使用放样来建模难以用其他方法创建的排气歧管。您可以在Abaqus/CAE中创建实体、壳或切割放样特征。放样特征从起始截面形状和方向转换到结束形状和方向。首先创建定义放样穿过空间区域时的形状的截面。然后Abaqus/CAE可以自动创建截面之间的路径，或者您可以定义一条或多条连接每个放样截面上的一个点到下一截面对应点的连续路径。您还可以从多个切线选项中进行选择，以控制放样在离开起始截面或接近结束截面时的形状。本节介绍在创建放样特征之前用于定义放样截面、放样路径和放样切线的选项，并解释自相交。涵盖以下主题：
- "定义放样截面"，第11.14.1节
- "定义放样路径"，第11.14.2节
- "定义放样切线"，第11.14.3节
- "自相交检查"，第11.14.4节
