# 11.1.2 常用求解控制参数

**产品：** Abaqus/Standard 和 Abaqus/Explicit

##### **参考资料**

- ["求解非线性问题，" 第 11.1.1 节](pt03ch07s01aus49.md)
- [*CONTROLS](../key/key-link.md#usb-kws-hcontrols)
- [*LINE SEARCH](../key/key-link.md#usb-kws-hlinesearch)

### 概述

Abaqus 提供了多个控制参数来调整非线性求解器的行为。这些参数影响迭代收敛速度、稳定性以及对困难非线性问题的鲁棒性。

### 迭代控制

| **参数** | **描述** | **有效值** |
| --- | --- | --- |
| ITERATION | 每个增量步的最大迭代次数 | 正整数 |
| CUTBACK | 迭代失败后时间增量减小的倍数 | 正数 |
| TIME INTERVAL | 最小和最大时间增量 | 正数 |

| **输入文件用法：** | ``` [*CONTROLS](../key/key-link.md#usb-kws-hcontrols), PARAMETER=ITERATION *最大迭代次数* ``` |

### 收敛准则控制

| **参数** | **描述** | **默认值** |
| --- | --- | --- |
| TOLERANCE | 收敛容差值 | 0.005 |
| CHECK | 收敛检查类型 | 能量和残差 |
| AVERAGE | 平均类型 | 仅用于某些选项 |

### 线性搜索

线性搜索是一种加速收敛的技术。当牛顿迭代在解附近振荡时，线性搜索可以稳定收敛过程。

| **输入文件用法：** | ``` [*LINE SEARCH](../key/link.md#usb-kws-hlinesearch) ``` |

### 时间步长控制

| **参数** | **描述** | **默认值** |
| --- | --- | --- |
| TIME INTERVAL | 初始时间增量 | 0.1 |
| MINIMUM | 最小时间增量 | 1e-8 |
| MAXIMUM | 最大时间增量 | 无限制 |

### 求解器容差

| **参数** | **描述** | **默认值** |
| --- | --- | --- |
| TOLERANCE | 线性方程组求解容差 | 1e-7 |
| MAX ITERATION | 线性求解器最大迭代次数 | 取决于求解器 |

### 矩阵存储

| **参数** | **描述** |
| --- | --- |
| FULL | 完整矩阵存储 |
| SYMMETRIC | 对称矩阵存储 |
| UNSYMMETRIC | 非对称矩阵存储 |

### 接触控制

接触算法的收敛行为可以通过以下参数控制：

| **参数** | **描述** |
| --- | --- |
| MAX ITERATION | 接触搜索最大迭代次数 |
| TOLERANCE | 接触容差 |

### 稳定性控制

对于可能出现不稳定响应的问题（如屈曲或材料软化），可以使用稳定性控制：

| **输入文件用法：** | ``` [*STABILIZE](../key/key-link.md#usb-kws-hstabilize) ``` |

### 诊断输出

诊断信息可以帮助识别收敛困难的原因：

| **参数** | **描述** |
| --- | --- |
| DIAGNOSTICS | 打印诊断信息 |
| PRINT | 打印每次迭代的信息 |

### 参数调优建议

1. **迭代次数过多**：减小时间增量或增大收敛容差
2. **收敛困难**：启用线性搜索或稳定性控制
3. **振荡行为**：使用自适应时间步长
4. **矩阵存储问题**：检查模型是否需要非对称存储

### 输入文件模板

```
[*HEADING](../key/key-link.md#usb-kws-mheading)
…
[*STEP](../key/key-link.md#usb-kws-hstep)
[*STATIC](../key/key-link.md#usb-kws-hstatic)
1.0, 1.0
[*CONTROLS](../key/key-link.md#usb-kws-hcontrols)
PARAMETER=ITERATION, 20
PARAMETER=TOLERANCE, 0.01
[*LINE SEARCH](../key/link.md#usb-kws-hlinesearch)
[*END STEP](../key/key-link.md#usb-kws-hendstep)
```
