# 10.4.3 IGES日志文件包含什么？





当您从IGES文件导入零件时，Abaqus/CAE会在您启动会话的目录中创建一个名为`abaqus_read_iges.log`的文件。IGES日志文件包含有关已翻译的实体以及遇到的任何问题的信息。

Abaqus/CAE会在每次IGES导入后覆盖`abaqus_read_iges.log`文件。以下信息会被写入IGES日志文件：
- 全局标题信息。
- 找到的IGES实体摘要。
- IGES读取选项。显示的选项是导入期间Abaqus/CAE中ACIS几何引擎使用的选项。您只能控制以下选项：- MSBO- 读取修剪曲线
- 转换日志。
- 错误。
