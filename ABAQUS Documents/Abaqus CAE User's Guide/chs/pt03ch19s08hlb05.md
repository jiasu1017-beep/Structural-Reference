# 19.8.5 设置作业提交时间







要找到作业编辑器，请从主菜单栏中选择****作业**![](../graphics/images/arrow.gif)**编辑**![](../graphics/images/arrow.gif)***作业名称*****。使用**提交**选项卡页面中的**提交时间**选项来选择分析作业的执行时间。

**设置提交时间：**

1. 在作业编辑器中，点击**提交**选项卡以显示**提交**选项卡页面。
2. 从页面底部的**提交时间**选项中，选择以下选项之一：****立即**** 作业立即在本地计算机后台执行或立即提交到批处理队列。****等待**** 作业在等待一段时期后在本地计算机后台执行。在小时和分钟字段中输入等待时间。（此选项仅在 UNIX 平台上可用。此外，如果作业正在提交到批处理队列，则此选项不可用。）****在**** 作业在您指定的时间在本地计算机后台或批处理队列中执行。点击**在**字段右侧的![](../graphics/ico_tipSmall.png)查看有关指定时间所用语法的信息。如果您希望在指定时间在批处理队列中运行作业，则必须使用 Abaqus 环境文件中的 **after_prefix** 队列参数将队列配置为接受时间变量。有关更多信息，请参见["定义分析批处理队列，"《Abaqus 安装和许可指南》第 4.2 节](../sgb/sgb-link.md#sgb-chp-env-queue)。（**在**选项可用于在 UNIX 平台上以后台运行或以批处理队列运行的作业。但是，在 Windows 平台上，**在**选项仅适用于在批处理队列中执行的作业。）
