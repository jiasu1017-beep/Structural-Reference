# 3.10 使用 reporttool 实用程序



### 3.10 使用 `reporttool` 实用程序

您可以使用 **reporttool** 实用程序生成 Abaqus 许可证使用历史报告。报告工具从 Dassault Systèmes 许可证服务器日志文件或 FLEXnet 调试日志文件读取数据，并根据您的选择自定义报告。

如果您在与许可证服务器所在的网络上的安装了 Abaqus 产品，则可以使用 `abaqus` 命令运行 **reporttool** 实用程序，如下所示：

```
abaqus licensing reporttool -log *logfile* 
-logtype {dsls | flexnet} 
[-start *start_date*] [-end *end_date*] 
[-type {export | query}] -accessor *accessors* [-list_accessors] 
[-filter *filter*] [-list_filters] [-sort *sorter*] [-list_sorters] 
[-aggregator *aggregators*] [-list_aggregators] [-duration *bucket_size*] 
[-output *report_file*] [-format *output_format*] [-list_formats] 
[-custom *customization_script*] [-help]
```
有关 Abaqus 许可证执行过程的更多信息，请参阅["许可证实用程序，"Abaqus Analysis User's Guide 第 3.2.12 节](../usb/usb-link.md#usb-int-dlicproc)。

在此报告工具的上下文中，许可证会话定义为在 Abaqus 产品功能上执行的有许可证的作业，具有检出时间和检入时间。

### 3.10.1 通用选项

`reporttool` 实用程序有以下命令行选项可用。

**`-log`**

指定要读取的日志文件的路径和文件名。DSLS 日志文件的默认位置在 Windows 上为 `C:\ProgramData\DassaultSystemes\LicenseServer\LogFiles\`，在 Linux 上为 `/var/DassaultSystemes/LicenseServer/LogFiles/`。对于 FLEXnet，您必须指定调试日志文件。如果日志文件位于远程服务器上，则必须将其复制到本地计算机可访问的目录。此选项是必需的。

可以通过两种方式指定多个日志文件：
- 多次使用 `-log` 选项：`-log first.log -log second.log`
- 给出文件名以逗号分隔的列表：`-log first.log,second.log`

**`-logtype`**

指定与 Abaqus 一起使用的许可证服务器类型：`dsls` 或 `flexnet`。此选项是必需的，区分大小写（必须小写）。

**`-start`**

指定报告期的开始日期和时间。如果省略 `-start` 选项，则报告期从日志文件中记录的最旧项目开始。日期和时间必须按以下格式之一指定：
- `*dd*-*mmm*-*yyyy*_*hh*:*mm*:*ss*` 例如，`-start 01-jan-2012_09:00:00` 表示 2012 年 1 月 1 日上午 9:00 的开始时间。时间指定是必需的，包括小时、分钟和秒。小时字段 (*hh*) 必须以 24 小时格式输入。
- `-*x*hours` 或 `-*x*days` 使用此格式选择过去的时间或日期。例如，`-start -6hours` 指定六小时前的时间。

**`-end`**

指定报告期的结束日期和时间。如果未指定 `-end` 选项，则报告期从日志文件中记录的最新项目结束。 日期和时间必须按以下格式之一指定：
- `*dd*-*mmm*-*yyyy*_*hh*:*mm*:*ss*` 例如，`-end 31-mar-2012_18:00:00` 表示 2012 年 3 月 31 日下午 6:00 的结束时间。时间指定是必需的，包括小时、分钟和秒。小时字段 (*hh*) 必须以 24 小时格式输入。
- `-*x*hours` 或 `-*x*days` 使用此格式选择过去的时间或日期。例如，`-end -3hours` 指定三小时前的时间。

**`-type`**

指定所需的报告类型：`export` 或 `query`。如果省略此选项，默认值为 `export`。导出报告以表格格式或逗号分隔值 (CSV) 提供基本许可证会话信息。数据可以可选地被过滤或排序。查询报告允许您汇总查看会话；您可以对原始数据进行操作，以计算最大值、峰值和平均值等内容。查询报告将总时间段划分为相等大小的时间桶。您使用 `-duration` 选项选择每个时间桶的大小。您还可以使用导出报告将原始数据保存到 CSV 文件，导入到 Excel 电子表格，并使用您自己的工具执行自定义计算。

**`-accessor`**

要从日志数据中读取的访问器以逗号分隔的列表。访问器是输出报告的字段或列。例如：`-accessor username,checkout,duration`。此选项是必需的。

**`-list_accessors`**

使用此选项获取可用访问器的列表。基本访问器如下：
- `duration`---许可证会话的持续时间，以秒为单位
- `username`---用户名
- `checkout`---检出日期
- `feature`---Dassault Systèmes 许可证功能（三字母代码）；例如 QSD
- `quantity`---请求的许可证数量
- `project`---从 `lmproject` 环境文件参数记录的自定义项目名称或编号（请参阅["许可证管理参数，"第 4.1.6 节](ch04s01.md#sgb-chp-env-net)）
- `session`---表示许可证作业的内部会话对象；这仅在使用 `peak` 聚合器或设计您自己的自定义聚合器时有用

**`-filter`**

过滤报告数据。此选项的必需格式为 `-filter *accessor*:*value*`。报告输出被过滤以仅包括 *accessor*=*value* 的数据记录。您可以在逗号分隔的列表中包含多个 `*accessor*:*value*` 对；例如，

```
-filter username:tsmith,hostname:zulu
```
此示例将生成仅显示用户 `tsmith` 在计算机 `zulu` 上的许可证检出的报告。要创建其他过滤器，请将 `-custom` 选项与 Python 程序一起使用。

**`-list_filters`**

使用此选项获取可用过滤器的列表。一个内置过滤器采用 `-filter *accessor*:*value*` 形式。如果您使用 `-custom` 选项创建任何自定义过滤器，它们将出现在此列表中。

### 3.10.2 导出选项

以下命令行选项可用于对 `reporttool` 实用程序的输出进行排序。

**`-sort`**

按时间顺序（从最旧到最新）对报告数据进行排序。此选项的必需格式为 `-sort date`。

**`-list_sorters`**

使用此选项获取可用排序器的列表。内置排序器是 `date`。如果您使用 `-custom` 选项创建任何自定义排序器，它们将出现在此列表中。

### 3.10.3 查询选项

`reporttool` 实用程序的查询报告有以下命令行选项可用。

**`-aggregator`**

要应用于每个时间桶中访问器值的聚合器函数以逗号分隔的列表。例如：`-aggregator max,average`。如果使用 `-type query`，则此选项是必需的。

**`-list_aggregators`**

使用此选项获取可用聚合器函数的列表。基本聚合器如下：
- `max`---每个时间桶中访问器的最大值
- `min`---每个时间桶中访问器的最小值
- `sum`---每个时间桶中所有访问器值的总和
- `average`---每个时间桶中访问器的平均值
- `len`---每个时间桶中的项目数
- `peak`---使用的最大许可证数量；必须对 `session` 访问器进行操作

并非所有聚合器都能与所有访问器一起使用。例如，您可以成功计算会话的最大持续时间，但尝试查找用户名的平均值毫无意义。

**`-duration`**

查询报告的时间桶大小，以分钟、小时、天、周或月为单位。如果省略此选项，默认值为 24 小时。示例：`-duration 1hour`、`-duration 2days`

### 3.10.4 输出选项

`reporttool` 实用程序的格式化输出有以下命令行选项可用。

**`-output`**

指定要将报告写入的文件的路径和文件名。如果省略此选项或使用 `-output -`，则默认写入标准输出（将报告显示在命令提示符或 shell 窗口中）。

**`-format`**

指定要使用的输出格式。

**`-list_formats`**

使用此选项获取可用输出格式的列表。基本格式如下：
- `table`---每个选定的访问器和/或聚合器的列的普通表格
- `csv`---逗号分隔值
- `json`---JavaScript 对象表示法（请参阅 [http://json.org](http://json.org)）

### 3.10.5 高级选项

**`-custom`**

指定包含您的自定义定义的 Python 程序文件。

### 3.10.6 示例

下面显示了几个示例，以演示使用 `reporttool` 实用程序的不同方式。

以下示例生成一个简单类型的导出报告，显示检出日期、用户名、产品功能和数量。

```
abaqus licensing reporttool -log today.log -logtype dsls -type export 
  -accessor checkout,username,feature,quantity

       Checkout Date | Username | Feature | Quantity |
-------------------- | -------- | ------- | -------- |
2012-Dec-10 03:05:16 | bbaggins |     QEX |       50 |
2012-Dec-10 03:33:50 | gandalf  |     QAE |        1 |
2012-Dec-10 04:08:34 | fbaggins |     QAE |        1 |
2012-Dec-10 06:27:18 | gollum   |     QEX |       50 | 
```

以下示例生成一个查询报告，计算每个 1 小时时间桶中使用的许可证令牌的平均数量。

```
abaqus licensing reporttool -log today.log -logtype dsls -type query 
  -accessor quantity -aggregator average -duration 1hour

              Bucket | Average
-------------------- | -------
2012-Sep-19 12:37:53 |      25
2012-Sep-19 13:37:53 |      19
2012-Sep-19 14:37:53 |       8
2012-Sep-19 15:37:53 |      18
2012-Sep-19 16:37:53 |      13
2012-Sep-19 17:37:53 |      34
```
