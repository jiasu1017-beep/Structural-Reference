# 19.5.2 理解优化过程生成的文件





当优化过程执行时，它会创建两种类型的数据，这些数据保存在单独的文件中——优化结果和分析结果。

**优化结果**

优化结果由优化变量和优化值组成。优化变量取决于您正在执行的优化类型。

**拓扑优化**

优化变量是归一化材料分布变量（MAT_PROP_NORMALIZED）。

**形状优化**

优化变量是位移变量：
- DISP_OPT：表示在形状优化期间节点移动方向的向量。由于网格平滑和过滤，向量可能与节点法向向量不一致。
- DISP_OPT_VAL：DISP_OPT 向量的大小，符号表示位移方向——正表示增长，负表示收缩。
- CTRL_INPUT：每个节点上目标函数的值（例如应力）。

**尺寸优化**

优化变量是壳厚度和壳厚度变化（THICKNESS 和 DELTA_THICKNESS）。

优化变量在每个设计周期后保存在单独的优化（`.onf`）文件中。此外，优化值（如目标和约束的值）在每个设计周期后写入逗号分隔的文本文件（`.csv`）。（当您监控优化过程的进度时，您正在查看优化值，如[「监控您的优化过程，」第 19.12.6 节](pt03ch19s12hlb06.md)中所述。）

**分析结果**

分析结果是 Abaqus 在分析期间生成的场数据和历史数据。在每个设计周期中会创建一个新的输出数据库文件。在初始设计周期中，Abaqus 将场数据和历史数据都写入新的输出数据库文件；但在后续设计周期中，Abaqus 仅写入场数据。Abaqus 还会在每个设计周期期间生成数据（`.dat`）、消息（`.msg`）和状态（`.sta`）文件。为了节省磁盘空间并加快后处理，输出数据库文件以及数据、消息和状态文件仅在指定间隔保存；默认情况下，在初始、第一个和最后一个设计周期之后保存。（当您创建优化过程时，您可以指定文件保存的频率，如[「创建和编辑优化过程，」第 19.12.1 节](pt03ch19s12hlb01.md)中所述。）此外，Abaqus 保存在每个设计周期期间生成的新输入文件。
