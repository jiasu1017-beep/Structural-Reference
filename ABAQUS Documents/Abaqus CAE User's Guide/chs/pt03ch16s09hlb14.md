# 16.9.14 定义连接器力矩





您可以创建连接器力矩以将集中力矩应用于连接器的可用相对运动分量。

**创建或编辑连接器力矩：**

1. 使用以下方法之一显示连接器力矩载荷编辑器：- 若要创建新的连接器力矩，请遵循"创建载荷"，第16.8.1节中概述的过程（**类别**：`Mechanical`；**所选步骤的类型**：`Connector moment`）。- 若要使用菜单或管理器编辑现有的连接器力矩，请参见"编辑依赖于步骤的对象"，第3.4.13节。若要编辑应用连接器力矩的区域，请参见"编辑应用规定条件的区域"，第16.8.4节。
2. 在可用的**M1**、**M2**和**M3**文本字段中，输入每个可用相对运动旋转分量的力矩分量（单位[FL](../popups/usb-int-iconventions-unitsym.md)）。只有所有选定线共用的分量才在编辑器中可用。如果您留空文本字段，则自动为该相对运动分量分配零力矩。但是，您必须在编辑器中至少输入一个非零分量来定义载荷。**警告：**无论连接器力矩是否超过连接器的失效准则，连接器力矩都将被应用。
3. 如果需要，单击**幅值**字段右侧的箭头，并从出现的列表中选择您选择的幅值。或者，您可以单击![](../graphics/ico_amplitudeSmall.png)创建新的幅值。（更多信息，请参见第57章"幅值工具集"。）
4. 单击**确定**保存您的数据并退出编辑器。
