### 概述

Abaqus/Explicit中的接触诊断允许您获取有关接触表面和接触相互作用进程的详细信息。诊断可用于：
- 审查模型中两个表面之间的自动调整，
- 揭示模型中潜在有问题的初始表面配置，
- 跟踪两个接触表面之间的过度穿透，和
- 审查与扭曲表面接触相关的警告。

### 审查初始过闭表面的调整

在模型的初始配置中过闭的接触表面由Abaqus/Explicit自动调整以移除过闭（参见["在Abaqus/Explicit中控制通用接触的初始接触状态，" 第36.4.4节](pt09ch36s04aus158.md)和["调整Abaqus/Explicit接触对的初始表面位置和指定初始间隙，" 第36.5.4节](pt09ch36s05aus163.md)）。关于过闭表面调整的信息有三个来源：状态（`.sta`）文件、消息（`.msg`）文件和输出数据库（`.odb`）文件。

#### 在状态和消息文件中获取过闭表面的调整

默认情况下，Abaqus/Explicit将所有节点调整 and—for general contact surfaces—contact offsets 写入消息（`.msg`）文件，同时将第一次模拟步骤中定义的接触对的最大初始过闭和最大节点调整的摘要列表写入状态（`.sta`）文件。您可以选择禁止写入消息文件的信息，仅将摘要信息写入状态文件。写入消息和状态文件的信息也写入输出数据库（`.odb`）以在Abaqus/CAE中使用。

| **输入文件用法：** | 使用以下选项获取详细诊断输出到消息文件和摘要诊断输出到状态文件： |
| --- | --- |
|  | ``` [*DIAGNOSTICS](../key/key-link.md#usb-kws-hdiagnostics), CONTACT INITIAL OVERCLOSURE=DETAIL (default) ``` 使用以下选项仅获取摘要诊断输出到状态文件（不会将接触诊断写入消息文件）： ``` [*DIAGNOSTICS](../key/key-link.md#usb-kws-hdiagnostics), CONTACT INITIAL OVERCLOSURE=SUMMARY ``` |

| **Abaqus/CAE用法：** | 您无法从Abaqus/CAE内部控制接触初始过闭的诊断信息。使用以下选项查看保存的诊断信息： |
| --- | --- |
|  | Visualization模块： ****Tools**![](../graphics/images/arrow.gif)**Job Diagnostics**** |

#### 查看表面的调整

在第一步中，可以"Abaqus/CAE中查看初始过闭表面的调整。可以在零时刻的原始场输出帧上绘制显示第一步中定义的接触对调整的位移形状图。此外，输出变量STRAINFREE（参见["Abaqus/Explicit输出变量标识符，" 第4.2.2节](pt02ch04s02xbv01.md)）包含表示初始无应变调整的节点向量。默认情况下，如果Abaqus/Explicit进行了任何无应变调整，STRAINFREE会写入输出数据库（`.odb`）文件中零时刻的原始场输出帧。Abaqus/CAE可视化模块中此变量的符号图显示表示各个节点如何调整的向量，其等值线图显示调整大小的分布（您必须在选择STRAINFREE输出变量之前在Abaqus/CAE可视化模块中选择零时刻的原始输出帧）。对于第一步以外步骤中的过闭，节点位移和加速度的向量图在可视化调整时可能特别有帮助。可以在数据检查分析后"Abaqus/CAE中查看此类图（参见["Abaqus/Standard、Abaqus/Explicit和Abaqus/CFD执行，" 第3.2.2节](pt01ch03s02abx02.md)）。

#### 可视化小滑动接触对的精确初始间隙

当为小滑动接触对指定精确初始间隙时，Abaqus/Explicit不会调整从属表面的坐标（参见["调整Abaqus/Explicit接触对的初始表面位置和指定初始间隙，" 第36.5.4节](pt09ch36s05aus163.md)）。因此，指定的间隙无法在Abaqus/CAE等后处理器中看到。因此，根据表面的初始几何和间隙或过闭的大小，表面在模拟中实际刚刚接触时，它们可能在后处理器中显示为打开或闭合。

### 检测通用接触域中的交叉表面

如果从属表面最初穿透双面主表面的距离大于主表面的厚度，则严重过闭的从属节点将把主表面的背面视为适当的接触力方向。这些交叉表面中的从属节点 effectively become trapped behind the master surface。此问题在["在Abaqus/Explicit中控制通用接触的初始接触状态，" 第36.4.4节](pt09ch36s04aus158.md)和["调整Abaqus/Explicit接触对的初始表面位置和指定初始间隙，" 第36.5.4节](pt09ch36s05aus163.md)中有更详细的讨论。

对于通用接触定义，默认情况下会激活识别初始配置中表面交叉区域的诊断测试。当诊断测试被激活时，如果检测到两个相邻从属节点（通过面元边缘连接）位于主表面的相对两侧，则会向消息（`.msg`）文件发出警告消息。对于位于主表面相对两侧的基于节点的表面节点，不会发出此类警告，因为在基于节点的表面节点之间无法确定邻接关系。在某些涉及主表面角落的情况下，即使相邻从属节点实际上在主表面的同一侧，也可能发出此警告消息。在大型模型上执行诊断测试的CPU成本 potentially significant。在这种情况下，您可以选择 deactivating the diagnostic testing and avoiding the extra CPU cost。

| **输入文件用法：** | 使用以下选项停用初始交叉表面的诊断测试： |
| --- | --- |
|  | ``` [*DIAGNOSTICS](../key/key-link.md#usb-kws-hdiagnostics), DETECT CROSSED SURFACES=OFF ``` |

| **Abaqus/CAE用法：** | 您无法从Abaqus/CAE内部排除初始交叉表面的诊断测试。使用以下选项查看保存的诊断信息： |
| --- | --- |
|  | Visualization模块： ****Tools**![](../graphics/images/arrow.gif)**Job Diagnostics**** |

### 通用接触表面之间的过度穿透

如["Abaqus/Explicit中的接触约束施加方法，" 第38.2.3节](pt09ch38s02aus182.md)所述，Abaqus/Explicit中通用接触算法使用的罚约束施加方法允许一个表面略微穿透另一个表面。"弹簧"刚度自动施加到表面以抵抗这些穿透。如果涉及通用接触的节点没有足够的质量，则Abaqus/Explicit自动选择的默认"弹簧"刚度可能不足以防止大穿透。例如，这种情况可能发生在被运动学耦合定义完全约束的无质量节点云与没有质量的完全约束刚性面接触时。

默认情况下，如果在节点-面接触中，节点穿透其跟踪面的穿透超过通用接触域中典型面尺寸的50%，则认为穿透过度，Abaqus/Explicit向状态（`.sta`）文件发出诊断消息。包含深度穿透节点的节点集也会写入输出数据库（`.odb`）文件以在Abaqus/CAE中使用。您可以控制用于触发诊断消息的典型元素面尺寸的分数。

| **输入文件用法：** | 使用以下选项控制用于触发深度穿透诊断消息的典型元素面尺寸的分数： |
| --- | --- |
|  | ``` [*DIAGNOSTICS](../key/key-link.md#usb-kws-hdiagnostics), DEEP PENETRATION FACTOR=*value* ``` |

| **Abaqus/CAE用法：** | 您无法从Abaqus/CAE内部控制深度穿透的诊断信息。使用以下选项查看保存的诊断信息： |

### 高度扭曲表面的警告消息

沿着高度扭曲的表面计算正确的接触条件非常困难，Abaqus/Explicit采用专门的算法在扭曲表面之间施加接触；此专门算法比默认接触算法更昂贵（参见["Abaqus/Explicit中接触对的接触控制，" 第36.5.5节](pt09ch36s05aus164.md)）。默认情况下，Abaqus/Explicit每20个增量检查一次高度扭曲的表面。

Abaqus/Explicit第一次检测到表面高度扭曲时，在状态（`.sta`）文件中写入警告消息。消息很简短；它仅说明哪个表面具有高度扭曲的面元。如果稍后在分析中此表面上的其他面元变得高度扭曲，则不会发出其他警告消息。

如果您请求更详细的诊断警告消息，消息文件将在每次发现特定表面上的扭曲面元时包含警告。警告将给出与扭曲面元关联的父元素（父元素是形成面元的元素）以及面元的扭曲角度。

请求详细警告可能会显著增加计算时间和消息文件的大小。您可以在后续步骤中切换回摘要警告或完全禁止扭曲表面警告。

如果分析因 fatal error 终止，则 preselected output variables 将自动作为最后增量的场数据添加到输出数据库。

| **输入文件用法：** | 使用以下选项请求扭曲表面的详细诊断警告输出： |
| --- | --- |
|  | ``` [*DIAGNOSTICS](../key/key-link.md#usb-kws-hdiagnostics), WARPED SURFACE=DETAIL ``` 使用以下选项请求扭曲表面的默认摘要诊断警告输出： ``` [*DIAGNOSTICS](../key/key-link.md#usb-kws-hdiagnostics), WARPED SURFACE=SUMMARY ``` 使用以下选项完全禁止扭曲表面的诊断警告输出： ``` [*DIAGNOSTICS](../key/key-link.md#usb-kws-hdiagnostics), WARPED SURFACE=OFF ``` |

| **Abaqus/CAE用法：** | 不支持在Abaqus/CAE中请求扭曲表面的诊断输出。 |
