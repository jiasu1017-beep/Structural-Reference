# 15.13.24 定义表面辐射相互作用







您可以通过创建表面辐射相互作用来模拟由于辐射引起的非凹表面与非反射环境之间的热传递。表面辐射也可用于在三维模型中近似封闭空腔的空腔辐射。从主菜单栏中选择****相互作用**![](../graphics/images/arrow.gif)**创建****，然后选择表面。有关辐射相互作用的简要概述，请参见"理解相互作用"，第15.3节。有关更详细的讨论，请参见Abaqus Analysis User's Guide的第34.4.4节"热载荷"。有关空腔辐射的更多信息，请参见第15.13.21节"定义空腔辐射相互作用"和Abaqus Analysis User's Guide的第41.1.1节"空腔辐射"。

**定义表面辐射相互作用：**

1. 从主菜单栏中，选择****相互作用**![](../graphics/images/arrow.gif)**创建****。**提示：**您也可以使用相互作用模块工具箱中的![](../graphics/ico_interaction.png)工具创建表面辐射相互作用。
2. 在出现的**创建相互作用**对话框中，执行以下操作：- 命名该相互作用。有关命名对象的信息，请参见第3.2.1节"使用基本对话框组件"。- 选择步骤。您只能在热传递、耦合温度-位移或耦合热-电步骤期间定义表面的辐射。- 选择**表面辐射**类型的相互作用。
3. 单击**继续**关闭**创建相互作用**对话框。
4. 使用以下方法之一选择表面：- 使用现有表面定义区域。在提示区域右侧单击**表面**。从出现的**区域选择**对话框中选择现有表面，然后单击**继续**。**注意：**默认选择方法基于您最近使用的选择方法。要恢复为另一种方法，请单击提示区域右侧的**在视口中选择**或**表面**。- 使用鼠标在视口中选择区域。（更多信息，请参见"在当前视口中选择对象"，第6.2节。）如果模型包含网格和几何的组合，请从提示区域单击以下之一：- 如果要从几何区域选择表面或顶点，请单击**几何**。- 如果要从本机或孤立网格选择表面或节点，请单击**网格**。您可以使用角度方法从几何或网格中选择一组面或边或一组单元面。更多信息，请参见"使用角度和特征边缘方法选择多个对象"，第6.2.3节。
5. 在出现的**编辑相互作用**对话框中，选择**辐射类型**。- 选择**到环境**以指定向周围环境的热传递。- 选择**空腔近似（仅3D）**以使用均匀发射率、封闭空腔和平均空腔温度在三维模型中近似空腔辐射。
6. 如果在上一步中选择了**到环境**，请按如下方式完成辐射定义：1. 单击**发射率分布**字段右侧的箭头，然后从出现的列表中选择您想要的选项：- 选择**均匀**以定义在表面上均匀的发射率。- 选择解析场以定义空间变化的发射率。只有对该相互作用类型有效的解析场才显示在选择列表中。或者，您可以单击![](../graphics/ico_analyticalField.png)创建新解析场。（请参见第58章"解析场工具集"获取更多信息。）2. 在**发射率**字段中，输入表面的发射率，![](../graphics/usi_eqn00171.gif]。3. 在**环境温度**字段中，输入环境温度，![](../graphics/usi_eqn00275.gif]。4. 如果要使环境温度随时间变化，请单击**环境温度幅值**字段右侧的箭头，然后从出现的列表中选择幅值。如果需要，请单击![](../graphics/ico_amplitudeSmall.png)创建新幅值；请参见第57.3节"选择要定义的幅值类型"获取更多信息。
7. 如果在步骤5中选择了**空腔近似（仅3D）**，请在**发射率**字段中输入表面的发射率，![](../graphics/usi_eqn00171.gif]。
8. 在**编辑模型属性**对话框中，指定正在使用的温度标尺上的绝对零温度，![](../graphics/usi_eqn00574.gif)，以及Stefan-Boltzmann常数，![](../graphics/usi_eqn00416.gif]，如第9.8.4节"指定模型属性"中所述。
9. 单击**确定**创建相互作用并关闭编辑器。
