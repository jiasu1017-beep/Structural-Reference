# 附录 D：查看分析结果

本后处理教程面向有经验的 Abaqus 用户，介绍如何使用可视化模块（也作为独立许可产品 Abaqus/Viewer 提供）以图形形式显示分析结果。

## D.1 概述

在本教程中，您将显示例题案例 2 的输出结果，即"用半球形冲头压入弹性泡沫试样"，详见 Abaqus 例题指南第 1.1.4 节。该问题研究一个重型冲头撞击软质弹性泡沫块的行为；产生的变形和应变如图 D-1 所示。

**图 D-1** 显示变形和应变的云图。

![../graphics/gst-tut3-overviewfig.png](../graphics/gst-tut3-overviewfig.png)

该问题在二维条件下建模，分为三个步骤：

1. 冲头最初放置在泡沫块的表面上，在其自身重量作用下压缩泡沫块。重力载荷在两秒内逐渐增加；但分析持续进行总共五秒，使泡沫充分松弛。该分析使用准静态方法来模拟在此步骤中泡沫块的响应。

2. 冲头被迫向下移动，施加按半正弦波变化的一个脉冲载荷，持续一秒。泡沫块的响应使用隐式动态方法建模。

3. 脉冲载荷被移除，冲头自由移动，泡沫膨胀和收缩。粘弹性泡沫阻尼掉振动，步骤运行 10 秒，模型返回稳态。与第二步一样，泡沫块的响应使用隐式动态方法建模。

## D.2 输出数据库中有哪些变量？

在弹性泡沫示例的第一步中，包含一组选项来控制分析每个步骤中的数据输出。Abaqus/Standard 根据输出类型将此输出写入输出数据库的场输出或历史输出部分。

**场输出**

输出数据库的场输出部分包含在分析过程中应相对不频繁输出的变量；本例中为每 10 个增量输出一次以及在步骤的最后增量输出一次。通常，您可以为整个模型或模型的大部分区域选择输出，Abaqus 会以选定频率写入每个分量。只将选定的变量写入输出数据库。

以下输入文件片段显示了控制弹性泡沫块示例中场输出变量的选项：

```
*OUTPUT, FIELD, FREQUENCY=10
*CONTACT OUTPUT, SLAVE=ASURF, MASTER=BSURF, VARIABLE=PRESELECT
*NODE OUTPUT
U,
*ELEMENT OUTPUT, ELSET=FOAM
S, E
```

Abaqus/Standard 在每 10 个增量后和每个步骤结束时向输出数据库的场输出部分写入以下变量：

- 泡沫块中每个积分点处的应力分量；
- 泡沫块中每个积分点处的对数应变分量（默认情况下，当用户请求几何非线性分析的应变时，对数应变会被写入输出数据库）；
- 模型中每个节点的位移；以及
- 冲头与泡沫块之间接触产生的默认接触输出变量（间隙、压力、剪应力和切向运动）。

**历史输出**

输出数据库的历史输出部分包含在分析过程中可以相对频繁输出的变量，频率可达每个增量一次。为避免生成大量数据，您通常只从模型的较小区域（如单个单元或小区域）选择输出。此外，您必须选择写入输出数据库的变量的各个分量。历史输出通常用于生成 X-Y 数据图。

以下输入文件片段显示了控制弹性泡沫块示例中历史输出变量的选项：

```
*OUTPUT, HISTORY, FREQUENCY=1
*NODE OUTPUT, NSET=N9999
U2, V2, A2
*ELEMENT OUTPUT, ELSET=CORNER
MISES, E22, S22
```

Abaqus/Standard 在每个增量后从冲头的刚体参考节点（包含在节点集 N9999 中）向输出数据库的历史部分写入以下变量：

- 垂直位移，
- 垂直速度，以及
- 垂直加速度。

此外，在每个增量后，Abaqus/Standard 从块角点处的单元向输出数据库的历史部分写入以下变量：

- von Mises 应力，
- 2 平面上 2 方向的对数应变，以及
- 2 平面上 2 方向的应力。

应力应变变量会为该单元中所有积分点写入。

## D.3 读取输出数据库

要开始本教程，请打开 Abaqus/Standard 在分析例题期间生成的输出数据库。

**读取输出数据库的步骤：**

1. 您可以使用 Abaqus **fetch** 实用程序将输出数据库复制到本地目录。在操作系统提示符下输入以下命令：

```
abaqus fetch job=viewer_tutorial
```

2. 如果尚未执行此操作，请通过在操作系统提示符下输入 `abaqus cae` 或 `abaqus viewer` 来启动 Abaqus/CAE 或 Abaqus/Viewer。

3. 在出现的**开始会话**对话框中，选择**打开数据库**。如果您已在 Abaqus/CAE 或 Abaqus/Viewer 会话中，请从主菜单栏选择**文件 -> 打开**。

   出现**打开数据库**对话框。

4. 在**打开数据库**对话框底部的**文件过滤器**列表中，选择**输出数据库 (*.odb)**。

   对话框的其余部分会发生变化，仅显示扩展名为 `.odb` 的文件。

   **注意：**如果您正在运行 Abaqus/Viewer，则不必选择文件过滤器；在**打开数据库**对话框中仅显示输出数据库文件。

5. 调整窗口大小，以便您可以按照本教程操作并看到 Abaqus/CAE 主窗口。有关更多信息，请参阅 Abaqus/CAE 用户指南第 2.1.1 节"启动 Abaqus/CAE（或 Abaqus/Viewer）"。

6. 在**打开数据库**对话框顶部的**目录**列表中，选择您的本地目录。

7. 从出现的输出数据库文件列表中，选择 `viewer_tutorial.odb`。

8. 点击**确定**。

   Abaqus/CAE 启动可视化模块并显示模型的未变形形状，如图 D-2 所示。

   **图 D-2** 未变形的模型形状。

   ![../graphics/gst-tut3-fastplot-nls.png](../graphics/gst-tut3-fastplot-nls.png)

   视口底部的标题块指示以下信息：

   - 模型的描述（来自输入文件中 `*HEADING` 选项的第一行）。
   - 输出数据库的名称（来自分析作业的名称）。
   - 用于生成输出数据库的产品名称（Abaqus/Standard 或 Abaqus/Explicit）。
   - 输出数据库上次修改的日期。

   视口底部的状态块指示以下信息：

   - 当前显示的是哪个步骤。
   - 该步骤中的增量。
   - 步骤时间。

   视图方向三角轴指示模型在全球坐标系中的方向。位于视口右上角的 3D 指南针允许您直接操作视图。

   **注意：**为清晰起见，本教程中的大多数图例都不包含标题块、状态块、方向三角轴和 3D 指南针。但是，当显示三角轴时，使用非默认的 1-2-3 三角轴标签。一般来说，这些图例说明了更改和自定义绘图对模型的影响。您可以通过从主菜单栏选择**视口 -> 视口注释选项**来关闭和自定义标题块、状态块和方向三角轴。

## D.4 自定义模型图

现在您将使用绘图选项来请求显示单元编号。

### D.4.1 自定义模型图

一组适用于所有绘图状态（未变形、变形、云图等）的通用绘图选项允许您自定义图形的外观。无论绘图状态如何，自定义选项仅适用于当前视口，不会保存在会话之间。

**自定义模型图的步骤：**

1. 从主菜单栏中，选择**选项 -> 通用**。

   Abaqus 显示**通用绘图选项**对话框。

2. 默认情况下，Abaqus 以绿色填充模型并隐藏单元标签。您将把单元标签的颜色从青色改为红色并显示它们。从**通用绘图选项**对话框中，点击**标签**选项卡，然后执行以下操作：

   a. 开启**显示单元标签**。

   b. 点击**应用**。

      Abaqus 使用青色文本显示单元编号。

   c. 点击单元标签的颜色样本。

      Abaqus 显示**选择颜色**对话框。

   d. 点击 **RGB** 选项卡，将红色、绿色和蓝色值分别设置为 255、0 和 0。

      **提示：**您也可以从对话框底部附近的颜色中选择红色，或使用任何其他可用的选择方法。

   e. 点击**确定**接受您的选择。

      Abaqus 关闭**选择颜色**对话框，并将颜色样本更新为红色。

   f. 点击**确定**。

      单元标签的颜色从青色变为红色，**通用绘图选项**对话框关闭。

## D.5 显示变形后的模型形状

您可以显示模型在分析每个帧中显示变形形状的图。当您请求力-位移分析数据的变形形状图时，Abaqus 默认绘制节点位移；但您可以显示输出数据库中可用的任何节点矢量场输出变量。您也可以使用绘图选项来自定义变形图的外观。

### D.5.1 显示变形形状图

Abaqus/Standard 或 Abaqus/Explicit 中的大多数程序默认会将位移写入输出数据库，也默认选择位移作为要使用的默认变形变量节点矢量量。当 Abaqus 读取输出数据库时，它使用默认变形变量来确定变形模型形状。在弹性泡沫块示例中，用户请求在每 10 个增量后对模型中每个节点输出位移（`U`），并且位移被选为默认变形变量。

（某些程序（例如热传递）默认不会将节点矢量量写入输出数据库，也不会选择变量作为默认变形变量。因此，Abaqus 无法显示变形图，因为在这种情况下输出数据库不包含任何可用于计算变形形状的变量。）

**显示变形形状图的步骤：**

1. 从主菜单栏中，选择**绘图 -> 变形形状**。

   **提示：**您也可以使用可视化模块工具箱中的工具来绘制变形模型。

   Abaqus 在上一个未变形模型显示的相同增量和步骤中显示变形模型。单元标签也会出现，因为使用了通用绘图选项来显示它们。

2. 打开**通用绘图选项**对话框。点击**默认值**然后点击**应用**来恢复和应用默认选项。

   状态块指示正在绘制的默认变形变量（`U`）和变形比例因子（`1.000e+00`）。对于大位移分析，Abaqus 选择默认变形比例因子 1.0。（对于小位移分析，Abaqus 选择默认比例因子以最佳方式适应视口。）

3. 上下文栏中的按钮允许您在分析的各个帧之间移动。特别地，上下文栏最右侧的按钮是帧选择器工具；它允许您拖动滑块来选择感兴趣的帧。

   您也可以使用以下技术直接跳转到选定的步骤和增量：

   a. 从主菜单栏中，选择**结果 -> 步骤/帧**。

      Abaqus 显示**步骤/帧**对话框。

   b. 选择 `Step 1`、`Increment 0`，然后点击**应用**。

   c. **步骤/帧**对话框也显示与增量关联的步骤时间。使用**步骤/帧**对话框来显示大约在第二步中间位置的变形模型。

4. 结合使用上下文栏中的按钮和**步骤/帧**对话框来查看不同帧和不同步骤中的变形图。

5. 显示第三步最后增量后的变形模型（`Step 3` 和 `Step Time = 10.00`），如图 D-3 所示。

   **图 D-3** 第三步最后增量后模型的变形图。

   ![../graphics/gst-tut3-figure5a.png](../graphics/gst-tut3-figure5a.png)

6. 点击**取消**关闭**步骤/帧**对话框。

### D.5.2 将未变形形状叠加到变形形状上

您可以同时绘制未变形和变形的模型形状。

**绘制未变形和变形模型形状的步骤：**

1. 点击工具箱中的工具以允许视口中显示多个绘图状态。然后点击该工具或选择**绘图 -> 未变形形状**将未变形形状图添加到视口中现有的变形图中。

   Abaqus 显示叠加了未变形图的变形图。

2. 要自定义叠加的（即未变形的）图，请从主菜单栏中选择**选项 -> 叠加**。

   Abaqus 显示**叠加绘图选项**对话框。

3. 在**叠加绘图选项**对话框中，选择**其他**选项卡。

4. 在**其他**选项卡页面中，选择**半透明度**选项卡。关闭**应用半透明度**。

5. 接下来选择**偏移**选项卡。选择**均匀**，然后输入 `0.001` 作为偏移值。

6. 点击**确定**应用更改并关闭**叠加绘图选项**对话框。

   Abaqus 显示自定义后的变形图，如图 D-4 所示。未变形模型形状显示为白色，并与变形模型形状略微偏移，以防止颜色重叠。

   **图 D-4** 自定义后的变形图。

## D.6 显示和自定义云图

您可以显示模型中某个变量（如应力、应变或温度）的云图。在所有绘图中（包括云图），Abaqus 默认选择要显示的变量。默认选择的变量取决于输出数据库中可用的变量，而可用的变量又取决于分析过程和请求的输出。您可以选择显示输出数据库场输出部分中可用的任何变量。如果您在无法显示该变量的绘图状态下选择了某个变量，则会出现一个对话框提示您切换到有效的绘图状态。

您可以使用绘图选项来自定义云图的外观。Abaqus 将您的自定义设置应用到当前视口中显示的每个云图。如果您在新的视口中显示云图，Abaqus 将恢复为默认绘图选项。

### D.6.1 显示云图

您将首先显示默认变量的云图。继续之前，请关闭多个绘图状态选项。

**显示云图的步骤：**

1. 从主菜单栏中，选择**绘图 -> 云图 -> 在变形形状上**。

   **提示：**您也可以使用可视化模块工具箱中的工具来显示云图。

   状态块指示绘制的变量是 `S, MISES`，即 Abaqus 默认选择的变量。Abaqus 在您用于显示变形形状图的相同步骤和帧中显示结果。

2. 使用上下文栏中的按钮或**步骤/帧**对话框来查看不同帧和不同步骤中的云图。

   **注意：**当您在帧之间移动时，图例会发生变化。Abaqus 会在每个帧中更新最大值和最小值，并计算云图间隔。

### D.6.2 选择要绘制的变量

Abaqus 在云图中默认选择一个要显示的变量，但您可以显示输出数据库场输出部分中可用的任何变量。

**选择要绘制的变量的步骤：**

1. 从主菜单栏中，选择**结果 -> 场输出**。

   Abaqus 显示**场输出**对话框。您使用**场输出**对话框来选择要显示的变量。

   **提示：**您也可以使用**场输出**工具栏中的列表来选择大多数场输出变量及其分量或不变量。有关更多信息，请参阅 Abaqus/CAE 用户指南第 42.5.2 节"使用场输出工具栏"。

2. 如果**主变量**选项卡尚未选中，请点击它。

3. 要选择应变的 22 分量作为主变量，请执行以下操作：

   a. 从**输出变量**字段中，选择 `LE`（积分点处的对数应变分量）。

   b. 从**分量**字段中，选择分量 `LE22`。

4. 点击**确定**选择 `LE22` 作为主变量并关闭**场输出**对话框。

   当前视口中的云图会变为 `LE22` 的图，如图 D-5 所示。

   **图 D-5** 第三步最后增量后模型的云图。

   ![../graphics/gst-tut3-contour-nls.png](../graphics/gst-tut3-contour-nls.png)

   **提示：**您可以点击**场输出**工具栏的左侧来从**场输出**对话框（而非工具栏）中进行选择。如果您使用对话框，则必须点击**应用**或**确定**，Abaqus/CAE 才能在视口中显示您的选择。

### D.6.3 自定义云图

默认情况下，Abaqus 使用所选变量最大值和最小值之间的 12 个均匀间隔来显示云图。Abaqus 会在每个帧中更新最大值和最小值，并计算新的云图间隔。图例指示计算的间隔和每个间隔对应的颜色。您可以更改间隔数量，也可以设置最大和最小云图限制对应的值。当您设置云图限制时，Abaqus 会在此后显示的每个云图中使用您提供的值，无论帧和正在绘制轮廓的变量如何。

**自定义云图的步骤：**

1. 在第二步最后增量的末尾显示云图（`Step 2` 和 `Step Time = 1.000`）。

2. 从主菜单栏中，选择**选项 -> 云图**。

   Abaqus 显示**云图选项**对话框。

3. 如果**基本**选项卡尚未选中，请点击它，然后将均匀云图间隔滑块拖动到 `16`。

4. 点击**限制**选项卡以访问云图限制选项。

   a. 在 **Max** 字段中，开启**指定**按钮并输入 `0.1` 作为最大云图限制。

   b. 在 **Min** 字段中，开启**指定**按钮并输入 `-0.75` 作为最小值。

5. 点击**应用**查看自定义后的云图。

   绘图会发生变化，如图 D-6 所示。

   **图 D-6** 自定义后的云图。

   ![../graphics/gst-tut3-custcontour-nls.png](../graphics/gst-tut3-custcontour-nls.png)

   尽管您选择了 16 个云图间隔，但图例显示 17 个间隔。Abaqus 添加间隔以指示大于最大云图限制或小于最小云图限制的任何值，并以浅灰色和深灰色分别显示这些值。在本例中，压应变大于 0.75 的区域以深灰色显示。最小应变显示在云图图例的底部。您可以使用这两种颜色中的任何一种来指示落在所选变量设计范围之外的单元。

6. 在**限制**选项卡下，检查 **Min** 和 **Max** **自动计算**选项。

   云图应变最小值和最大值显示在这两个**自动计算**选项旁边。

## D.7 动画显示云图

您可以使用以下方式之一来动画显示变形图、云图、符号图或材料取向图（仅限时间历史动画）：

- 是否显示动画状态

   Abaqus 显示**动画选项**对话框。

5. 如果**播放器**选项卡尚未选中，请点击它，然后执行以下操作：

   a. 选择 **Swing**。

   b. 点击**确定**。

## D.11 对 X-Y 数据进行运算

X-Y 数据对象是 Abaqus 以两列（X 列和 Y 列）形式存储的有序对集合。**对 XY 数据进行运算**对话框允许您通过对先前保存的 X-Y 数据对象执行运算来创建新的 X-Y 数据对象。在本教程中，您将通过将应力相对于时间数据对象与应变相对于时间数据对象相结合来创建应力-应变数据对象。然后，您将绘制应力-应变曲线。

### D.11.1 创建应力相对于时间和应变相对于时间数据对象

创建应力-应变曲线的第一步是从历史输出中创建应力相对于时间和应变相对于时间数据对象。数据对象仅包含分析第一步中的数据，其中冲头放置在泡沫块表面上并在其自身重量作用下压缩泡沫块。

**创建 X-Y 数据对象的步骤：**

1. 在结果树中，双击 **XYData** 容器。

   出现**创建 XY 数据**对话框。

2. 在此对话框中，如果尚未选中**ODB 历史输出**，请选中它，然后点击**继续**。

   出现**历史输出**对话框。

3. 在**历史输出**对话框中，执行以下操作：

   a. 点击**变量**选项卡。

   b. 按 `*LE22*` 筛选数据。

   c. 在**输出变量**字段中，选择 `Logarithmic strain components: LE22 at Element 1 Int Point 1 in ELSET CENT`。

   d. 点击**步骤/帧**选项卡。

   e. 选择 `Step 1`。

   f. 点击**另存为**。

      出现**将 XY 数据另存为**对话框。

   g. 将 X-Y 数据命名为 `Strain`，然后点击**确定**。

      名为 `Strain` 的数据对象出现在 **XYData** 容器中，其中包含分析第一步中单元 1 积分点 1 处的对数应变数据（`LE22`）。

4. 使用类似的技术创建包含分析第一步中单元 1 积分点 1 处应力数据（`S22`）的数据对象。将此数据对象命名为 `Stress`。

   **提示：**按 `*S22*` 筛选变量列表。

   现在您可以组合这两个数据对象来创建应力-应变数据对象了。

5. 关闭**历史输出**对话框。

### D.11.2 组合数据对象

在本节中，您将通过组合应力相对于时间和应变相对于时间数据对象来创建应力-应变数据对象。

**组合数据对象的步骤：**

1. 在结果树中，双击 **XYData** 容器。

2. 从出现的**创建 XY 数据**对话框中，选择**对 XY 数据进行运算**然后点击**继续**。

   出现**对 XY 数据进行运算**对话框。该对话框包含以下列表：

   - 左侧的 **XY 数据**字段包含现有 X-Y 数据对象的列表。
   - 右侧的**运算符**字段包含您可以对数据对象执行的所有可能运算的列表。

3. 从**运算符**字段中，点击 `combine(X,X)`。

   `combine( )` 出现在对话框顶部的表达式文本字段中。

4. 在 **XY 数据**字段中，拖动光标穿过 `Strain` 和 `Stress` 数据对象以选择两者，然后点击对话框底部的**添加到表达式**。

   表达式 `combine("Strain","Stress")` 出现在表达式文本字段中。在此表达式中，`"Strain"` 将决定组合图中 X 值，而 `"Stress"` 将决定 Y 值。

5. 从对话框底部的按钮中，点击**另存为**。

6. 在出现的**将 XY 数据另存为**对话框中，输入名称 `Stress-Strain` 然后点击**确定**。

   新的数据对象 `Stress-Strain` 出现在 **XYData** 容器中。

7. 取消**对 XY 数据进行运算**对话框。

### D.11.3 绘制和自定义应力-应变曲线

现在您将使用刚创建的 `Stress-Strain` 数据对象来绘制应力-应变曲线。

**绘制应力-应变曲线的步骤：**

1. 在 **XYData** 容器中，双击 `Stress-Strain`。

   应力-应变曲线出现在视口中。

## D.14 总结

- 当您打开输出数据库时，Abaqus/CAE 会自动加载可视化模块。
- 要执行许多可视化模块功能，您可以使用菜单项或工具箱中的工具。
- 您可以使用上下文栏右侧的按钮来显示模型在分析每个帧中的状态。
- 可视化模块有不同的绘图类型。通用绘图选项可用于控制所有绘图类型中模型的外观。有些绘图类型也有专门的绘图选项。自定义选项仅适用于当前视口，不会保存在会话之间。您可以使用**默认值**按钮恢复默认绘图选项。
- 您使用视口注释选项来自定义所有绘图中出现的项目的外观，例如标题块、状态块和方向三角轴。标题块显示关于生成输出数据库的分析的信息。状态块包含关于所显示的步骤和增量的信息。
- 在所有绘图中，Abaqus 从输出数据库的场输出部分选择要显示的默认变量。您可以使用**场输出**工具栏或对话框来选择要显示的变量。
- 您可以显示输出数据库中数据的时间历史动画，也可以基于单个增量结果生成比例因子动画。您可以动画显示变形图、云图、符号图或材料取向（仅限时间历史动画）图；动画使用各自的绘图选项来控制模型的外观。您可以在动画运行时自定义这些绘图。
- 符号图在指定步骤和帧处显示特定矢量或张量变量的大小和方向。默认情况下，符号图显示矢量变量的大小和张量变量的所有主分量。
- 材料取向图显示模型中元素在分析指定步骤和帧处的材料方向。材料取向以逐元素为基础显示在材料积分点处，不跨元素平均。
- 您可以显示输出数据库中存储的任何变量的 X-Y 图。在大多数情况下，假设 X 轴是时间。
- 您可以使用**对 XY 数据进行运算**对话框基于对现有数据对象的运算来创建新的 X-Y 数据对象。
- 您可以使用查询工具集来探测模型或 X-Y 图。您可以将获得的值写入文件。
