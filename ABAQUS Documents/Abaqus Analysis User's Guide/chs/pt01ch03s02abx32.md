# 3.2.32 将 RADIOSS 输入文件转换为部分 Abaqus 输入文件





**产品：**Abaqus/Explicit

##### **参考**

- [『Abaqus 执行程序：概述』，第 3.1.1 节](pt01ch03s01abo02.md)

### 概述

RADIOSS 到 Abaqus 的转换器将 RADIOSS 输入文件中的某些关键字转换为 Abaqus/Explicit 中的等效项。

### 使用转换器

转换器需要由 RADIOSS Version 4.4 或 5.1 创建的块格式输入文件。输入文件可以是任何名称和可选扩展名。

转换的 RADIOSS 数据条目列于下表中。其他 RADIOSS 关键字和数据将被跳过并在日志文件中注明。

转换器创建一个仅包含模型数据和时间历程输出数据的部分 Abaqus 输入文件。您可以提供其他输出数据来完成输入。

#### 单元编号和分组

生成的 Abaqus 输入文件中的所有单元将具有唯一的单元编号。RADIOSS 输入中具有非唯一单元编号的单元将自动分配新的单元编号。分配了相同 PART 标识号的单元被分组到一个单元集中。

具有不同材料或属性的单元应分配不同的 PART 标识号；即，相同的材料和属性必须适用于分组在同一单元集中的所有单元。

如果由 SPRING 转换而产生的单元具有不同的单元属性（如用于定义局部方向的斜交系统）并且被分配了相同的 PART 标识号，转换器会自动将它们分到不同的单元集中。

#### 材料模型

转换器仅支持下表中所示的材料模型（第 3.2.32-1 表）。所有不支持的材料模型在需要应力-应变定律定义时都作为线弹性材料进行转换。在这些情况下，转换器会提供材料属性的标称值。

### 命令行选项

**job**

此选项用于指定转换器输出的 Abaqus 输入文件的名称。Abaqus 输入文件的名称必须不带 `.inp` 扩展名。转换器创建的诊断信息写入名为 `*job-name*_fromradioss.log` 的文件。

**input**

此选项用于指定包含 RADIOSS 数据的文件的名称。文件扩展名是可选的。

**splitAirbagElements**

此选项用于指定是否将 4 节点气囊薄膜单元拆分为两个 3 节点气囊薄膜单元。默认值为 `ON`。气囊薄膜单元由 SHELL 或 SH3N（厚度方向有一个积分点）的转换产生。此选项仅在 RADIOSS 输入文件中指定了关键字 MONVOL/AIRBAG 时有效。

**readAbaqusDat**

此选项支持使用先前 Abaqus 分析的 Abaqus 数据（`.dat`）文件重新制定点焊定义。数据文件应标识无法形成的点焊。使用此选项，使用分布式耦合约束翻译已识别点焊的连接点。

**userDefaultMass**

此选项用于指定在转换过程中生成的额外节点的节点质量，这些节点需要非零质量。该值应该很小（通常为整个模型质量的 10^6 倍）。如果省略此选项，默认质量设置为 10^4。

**userDefaultInertia**

此选项用于指定在转换过程中生成的额外节点的旋转惯性，这些节点需要非零旋转惯性。该值应该很小（通常为整个模型惯性的 10^6 倍）。如果省略此选项，默认旋转惯性设置为 10^3。

**userHistoryTime**

此选项用于指定用于时间历程输出的时间间隔。如果省略此选项，时间历程间隔设置为 10^5。
