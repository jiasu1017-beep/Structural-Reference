# 4.3.3 移动视口





您可以将选定的视口移动到画布上的任何位置。这可能需要来显示隐藏的视口，或者仅仅是为了减少绘图区域的混乱。要移动视口，请单击视口标题栏上的任意位置，然后将其拖动到所需位置。

**要移动视口：**

1. 将光标放在视口标题栏上的任意位置。
2. 单击鼠标按钮1，并将光标拖动到新位置。光标变为四向箭头![](../graphics/crs-4headarrow.gif)。拖动时，视口的轮廓指示其新位置。
3. 释放鼠标按钮1。视口移动到新位置，如果之前不是当前视口，它会成为当前视口。
