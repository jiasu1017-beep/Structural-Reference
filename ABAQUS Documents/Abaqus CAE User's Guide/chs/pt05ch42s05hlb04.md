# 42.7.4 通过对场进行运算创建场输出

您可以通过使用输出数据库中可用的场输出计算新结果来创建场输出。例如，您可能创建新的场输出变量来显示两个增量之间的应力场差异。有关详细示例，请参阅["构建有效的场输出表达式，" 第42.7.1节](pt05ch42s05hlb01.md)。

您可以通过对输出数据库中找到的分析结果进行运算来从场创建新的场输出变量。您可以采用与输出数据库场输出变量相同的方式显示您创建的场输出变量：以变形图、云图或符号图的形式；通过探测任何模型或*X–Y*图；作为沿模型路径获取的*X–Y*数据；或在表格报告中。您创建的场输出变量保存在会话步骤中，直到结束会话或关闭场输出所源自的输出数据库。

**注意：**使用复数结果创建的场输出将仅包含数据的实部。您无法创建包含复数结果虚部的场输出。

如果创建从张量变量提取不变标量分量（如张量变量的Mises应力或其主应力之一）的场输出，则结果场输出的云图可能与原始不变量的云图不同。差异产生是因为这些等值线的计算方式不同。Abaqus/CAE通过首先将张量外推到节点、计算不变量值，然后在节点处平均值来计算原始不变量。对于场输出，Abaqus/CAE将不变量值外推到单元节点并对其进行平均，这可能导致不同的云图。

**注意：**某些场输出变量（如PEEQ和PEMAG）在积分点处始终为正值。对于这些变量，从积分点到单元节点的结果外推可能导致某些节点处出现负值。为避免这些负值结果，Abaqus/CAE在内部将此类场的单元节点值限制为0.0，并将负值重置为0.0。如果您对这些场输出变量进行运算以创建新的场输出，Abaqus/CAE不会将新结果场输出的值限制为0.0，云图可能显示一些负值。

**通过对现有场进行运算来创建场输出：**

1. 找到从场创建场输出的选项。从主菜单栏中，选择****Tools**![](../graphics/images/arrow.gif)**Create Field Output**![](../graphics/images/arrow.gif)**From Fields****。随即出现**Create Field Output**对话框。
   **提示：**您也可以使用工具箱中的![](../graphics/ico_fieldOutputFromFields.png)工具来指定场输出。
2. 新场输出变量的默认名称出现在**Name**文本字段中。要提供更有意义的名称，请用您选择的名称替换此默认名称（包括空格，如果您愿意）。
3. 在表达式文本字段中使用以下程序组合来构建您的表达式：
   1. 从**Create Field Output**对话框右侧的菜单中选择一个函数。
      - 选择**运算符**将内置算术运算符应用于所选场输出。
      - 选择**变换**将坐标系变换应用于所选场输出。
      - 选择**标量**从所选场输出中提取标量分量。菜单下方的列表会反映您的选择。
   2. 从**Create Field Output**对话框左侧的列表中选择场输出变量。您可以为分析中的任何步骤和帧选择场输出。
      **提示：**将光标放在输出变量表中分隔每列的垂直线上，然后放大列宽以查看完整文本。场输出变量出现在表达式文本字段中。
   3. 从**Create FieldOutput**对话框右侧的列表中选择运算符、变换或标量。所选函数出现在表达式文本字段中。
      **注意：**如果您从变换列表中选择动态坐标系（跟随模型中节点的坐标系），则`deformationField=`出现在变换函数内的表达式文本字段中。您必须选择Abaqus/CAE将用于计算坐标系位置的变形场输出变量。
   4. 使用表达式文本字段中的标准鼠标和键盘编辑技术来定位光标并配置您的表达式。
   5. 如有必要，请调整[表达式的语法](pt05ch42s05hlb01.md)：可能需要括号。
   6. 如果您犯错并希望重新开始，请单击**Clear Expression**。有关详细信息，请参阅["构建有效的场输出表达式，" 第42.7.1节](pt05ch42s05hlb01.md)。
4. 单击**OK**创建新的场输出变量并关闭对话框。Abaqus创建包含在会话步骤帧中的新场输出变量，可从**Field Output**对话框中获得。当前结果图（如果有）不受影响。
