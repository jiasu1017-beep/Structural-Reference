# 16.9.23 定义流体参考压力





您可以在Abaqus/CFD分析中创建流体参考压力载荷，以指定模型中某一点的静水压力水平。

**创建或编辑流体参考压力：**

1. 使用以下方法之一显示流体参考压力载荷编辑器：- 若要创建新的流体参考压力，请遵循"创建载荷"，第16.8.1节中概述的过程（**类别**：`Fluid`；**所选步骤的类型**：`Fluid reference pressure`）。- 若要使用菜单或管理器编辑现有的流体参考压力载荷，请参见"编辑依赖于步骤的对象"，第3.4.13节。若要编辑应用载荷的区域，请参见"编辑应用规定条件的区域"，第16.8.4节。
2. 在**幅值**字段中，指定流体参考压力的大小。
3. 单击**确定**保存您的数据并退出编辑器。
