# 13.8.3 分区装配





在装配模块中，您可以使用分区工具集将装配分区为其他区域。您可以使用一个零件实例的顶点、边和面来创建分割第二个零件实例的分区；例如，您可以使用**延伸面**方法通过将一个零件实例的面延伸进第二个零件实例来分割一个单元。分区不能跨越零件实例。

装配中的分区出现在对装配进行操作的所有模块中。当您在装配模块中创建零件实例时，在零件模块中创建的分区会与零件几何的其他部分一起传递。分区是装配的特征，它们会与装配的其他部分一起重新生成。您无法关闭分区的显示。分区修改零件实例的几何；因此，您无法分区从属零件实例。

分区工具集不受支持，不能与模型实例一起使用。
