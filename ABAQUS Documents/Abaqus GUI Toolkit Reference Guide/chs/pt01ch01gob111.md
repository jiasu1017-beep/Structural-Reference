# FXSplitter







分割器窗口用于交互式重新分配两个或多个子面板。空间可以水平或垂直细分。当分割器本身调整大小时，最右侧（最底部）的子窗口将被调整大小，除非分割器窗口是反向的；如果分割器是反向的，则最左侧（最顶部）的子窗口将被调整大小。分割器窗口在调整面板大小期间向其目标发送 SEL_CHANGED；在调整大小交互结束时，它发送 SEL_COMMAND 以表示调整大小操作完成。通常，子窗口可以从 0 开始调整大小；但是，如果在水平方向的分割器中，子窗口同时具有 LAYOUT_FILL_X 和 LAYOUT_FIX_WIDTH，则它不会被做得小于其默认宽度，除非该子窗口是最后一个可见窗口（或者当 SPLITTER_REVERSED 选项传递给分割器时是第一个可见窗口）。在垂直方向的分割器中，具有 LAYOUT_FILL_Y 和 LAYOUT_FIX_HEIGHT 的子窗口行为类似。这些选项仅影响交互式调整大小。
