# 20.8.4 通过修剪、延伸、分割或合并修改边







修改草图时，您可以通过修改尺寸、移动选定的顶点或修剪、延伸、分割或合并边来实现更改。Abaqus/CAE 提供以下方法来修剪、延伸、分割和合并边：

**修剪/延伸**

您可以修剪或延伸直线或圆弧的一端；您也可以修剪样条曲线，但样条曲线不能延伸。要修剪或延伸边，首先选择要修改的端点附近的边，然后选择第二条边来定义交点。第二条边可以是草图中的任何对象，包括构造几何。交点可能位于任一所选边的当前端点之外。Abaqus/CAE 在交点处修剪或延伸第一条边。如果您想将第二条边修剪或延伸到同一点，请重复此过程，反转选择顺序（请参见[图 20-16](pt03ch20s08s04.md#ske-tech-trimext)）。

**图 20-16** 延伸两条边以创建角。

![](../graphics/ske-tech-trimext.png)

有关修剪和延伸边的详细说明，请参见["通过修剪或延伸边修改 Sketcher 对象，"第 20.17.2 节](pt03ch20s14hlb02.md)。

**自动修剪**

您可以从直线、圆弧、圆或样条曲线中修剪中间或端部线段。**自动修剪**是用于从草图中移除不需要的边段的最快方法。Abaqus/CAE 使用预选来高亮显示您可以移除的边。预选基于光标接近度和两个最近的"修剪点"。修剪点包括交点、端点、顶点和定义圆边的选择点。与使用**修剪/延伸**不同，Abaqus/CAE 不会分割相交的边。[图 20-17](pt03ch20s08s04.md#ske-tech-autotrim)显示了基于左上角显示的圆和直线的几种可能的修剪组合。

**图 20-17** **自动修剪**选择。

![](../graphics/ske-tech-autotrim.png)

有关自动修剪边的详细说明，请参见["通过自动修剪边修改 Sketcher 对象，"第 20.17.3 节](pt03ch20s14hlb03.md)。

**分割**

您可以分割边，在草图中它们相交的位置创建具有公共顶点的独立片段。选择要打断的第一条边，然后选择第二条边，该边在所需分割点与第一条边相交。当您将光标靠近第二条边时，会在您将创建的分割点周围出现一个圆。如果两条边之间有多个交点，Abaqus/CAE 会高亮显示离光标最近的交点。Abaqus/CAE 在指示点分割第一条边。[图 20-18](pt03ch20s08s04.md#ske-tech-break)左侧显示了一条相交的直线和样条曲线，中间图像指示了分割点（未显示光标），右侧显示了三条结果边。在这种情况下，样条曲线首先被选中，因此直线保持不断开。

**注意：**分割或修剪样条曲线时，它们会保留其精确形状，但控制点被移除，因此您不再能编辑曲线的形状。

**图 20-18** 分割直线和样条曲线。

![](../graphics/ske-tech-break.png)

有关分割边的详细说明，请参见["通过分割边修改 Sketcher 对象，"第 20.17.4 节](pt03ch20s14hlb04.md)。

**合并**

合并工具使您能够闭合草图中的小间隙。这些间隙通常出现在以下情况：
- 当您对不在平行于草图平面的平面上的边使用**投影边**工具时，端点可能与其他所选边略有不同位置创建。

- 当您将草图几何导入现有草图时，边可能无法完全对齐。

合并工具仅用于闭合小间隙。合并边以闭合较大间隙可能会显著改变草图几何。如果您想在较大距离上移动草图中的元素，请使用 Sketcher 工具箱中的拖动工具将草图中的顶点移动到新位置。有关更多信息，请参见["拖动 Sketcher 对象，"第 20.17.1 节](pt03ch20s14hlb01.md)。

有关合并边的详细说明，请参见["通过合并边修改 Sketcher 对象，"第 20.17.5 节](pt03ch20s14hlb05.md)。
