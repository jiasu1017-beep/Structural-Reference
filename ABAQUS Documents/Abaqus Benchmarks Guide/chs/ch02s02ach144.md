# 2.2.2 无限单元：布西内斯克问题和弗拉芒问题





**产品：** Abaqus/Standard  

在本例中，我们求解两个问题来验证无限单元在建模远场域时的性能。半空间上点荷载问题和半空间上线荷载问题的结果分别与布西内斯克和弗拉芒（Timoshenko和Goodier，1970）给出的解析解进行比较。为比较起见，还给出了仅使用有限单元获得的结果。

### 问题描述

对于半空间上点荷载的布西内斯克问题，使用了两种网格配置。无限单元网格（[图2.2.2-1](ch02s02ach144.md#sxmbousflam-fin-infmesh)）由12个有限单元组成，延伸至半径4.0，用4个无限单元模拟远场域。有限单元网格（[图2.2.2-2](ch02s02ach144.md#sxmbousflam-finmesh)）由16个有限单元组成，在半径5.0处截断，在该处施加完全固定边界条件。

无限单元中无限方向上第二个节点的位置使得第一个节点位于荷载源和第二个节点之间等距处。

对于半空间上线荷载的弗拉芒问题，使用了相同的网格配置。在这种情况下，它们处于平面应变，并使用了垂直对称平面。

材料为线弹性，杨氏模量![](../graphics/bmk_eqn00429.gif) = 1.0，泊松比![](../graphics/bmk_eqn00430.gif) = 0.1。两个问题都施加单位荷载。
