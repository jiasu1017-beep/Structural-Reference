# 17.9.5 几何特征可能阻止零件进行扫掠网格划分





["三维实体的扫掠网格，"第17.9.3节](pt03ch17s09s03.md)中所述的区域特征被称为"拓扑特征"。在某些情况下，所有拓扑特征都适用于一个区域；但是，由于某些"几何特征"不满足，Abaqus/CAE不允许该区域进行扫掠网格划分。在确定区域是否可以进行扫掠网格划分时，Abaqus/CAE查找的几何特征难以量化。一般来说，如果三维区域满足以下几何特征，则可以使用扫掠网格技术进行网格划分：
- 如果源侧包含多个面，则面之间的角度必须相对平坦（接近180度）。
- 将源侧连接到目标侧的每个面（连接面）必须有四个角。每个角的角度必须接近90度。
- 源侧与每个连接面之间的角度应接近90度。同样，目标侧与每个连接面之间的角度也应接近90度。

例如，图17-85展示了一个在源侧有三个面的零件。当形成源侧的面之间的角度减小时，零件不再满足可扫掠网格区域的几何特征。

**图17-85** 扫掠网格取决于几何特征。

![](../graphics/mgn-sweep-topo-nls.png)

您可以尝试应用虚拟拓扑来满足几何特征，并使零件可以进行扫掠网格划分。例如，图17-86说明了当使用虚拟拓扑合并源侧上的三个面时，零件变得可以进行扫掠网格划分。然而，生成的网格质量较差。

**图17-86** 应用虚拟拓扑可能导致较差的网格质量。

![](../graphics/mgn-sweep-poormesh-nls.png)

在某些情况下，即使不满足几何特征，Abaqus/CAE仍会允许您创建扫掠网格。其目的是允许您尽可能创建扫掠网格。然而，生成的网格质量可能较差，或者可能无效。为确保您的扫掠网格可接受，您应该使用网格验证工具来验证其质量。更多信息，请参见["验证单元质量，"第17.19.1节](pt03ch17s19hlb01.md)。
