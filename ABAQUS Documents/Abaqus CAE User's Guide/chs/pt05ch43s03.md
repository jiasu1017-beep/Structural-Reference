# 43.4 生成未变形形状图

对于常规分析步骤，未变形图显示模型没有任何变形的形状。对于线性扰动步骤，未变形图显示模型的基础状态。由于不需要分析结果，来自 **datacheck** 运行的输出数据库足以生成未变形形状图。模型和分析特性（如表面定义）可能导致未变形模型的外观随步骤而变化；您可以选择显示未变形图的步骤和帧。有关更多信息，请参阅["选择结果步骤和帧，"第 42.3 节](pt05ch42s01.md)。

默认情况下，当您[打开输出数据库](pt02ch09s06hlb02.md)时，Abaqus/CAE 会显示分析最后一步和帧的未变形图状态。该图使用当前会话的默认显示选项生成（有关保存显示选项的信息，请参阅["保存显示选项设置，"第 76.16 节](pt07ch76hla15.md)）。要为已打开的输出数据库生成未变形形状图，请从主菜单栏选择****绘图**![](../graphics/images/arrow.gif)**未变形形状****，或使用工具箱中的 ![](../graphics/ico_plotUndeformed.png) 工具。

**修改未变形形状图：**

1. 使用 **File** 菜单[打开](pt02ch09s06hlb02.md)包含模型数据的输出数据库。
2. 使用 **Result** 菜单选择要显示的[结果步骤](pt05ch42s01hlb01.md)。
3. 选择您想要的绘图状态独立[自定义选项](pt05ch40s03s03.md)。

每次在结果或绘图状态独立对话框中点击 **Apply** 时，Abaqus 会自动刷新您的未变形图。
