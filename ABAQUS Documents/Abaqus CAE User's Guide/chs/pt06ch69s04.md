# 69.4 修复几何的策略







几何编辑工具集中的工具通常用于修复导入到 Abaqus/CAE 中的几何。修复操作的目标是创建没有无效实体的部件，并创建具有最少边和面的部件，以避免影响网格生成。如果您无法创建有效的部件，可以忽略无效性。 有关更多信息，请参阅["使用无效部件，" 第 10.2.3 节"](pt02ch10s02s03.md)。理想情况下，部件将没有小边或小面。您不必创建精确的部件，如果可能，您应该继续使用不精确的部件。具有不精确几何的部件可以进行网格划分。然而，在极少数情况下，其他操作可能在具有不精确几何的部件上失败；例如，其他网格划分功能、添加几何特征和分区。如果您无法使用不精确的部件并且无法使部件变得精确，您应该返回生成原始文件的 CAD 应用程序并提高精度。 有关更多信息，请参阅["什么是有效和精确的部件？" 第 10.2.1 节"](pt02ch10s02s01.md)。

Abaqus/CAE 提供了许多不同的编辑工具，通常很难决定使用哪些工具以及按什么顺序使用它们。通常，修复导入部件的几何是一门需要实验和经验来掌握的艺术。最佳方法取决于部件的复杂性以及您在网格划分和分析部件时希望保留多少细节。

以下指南将帮助您开发修复部件的有效方法：

**使用几何诊断工具**

使用几何诊断工具查询部件中的无效和不精确实体、自由边、短边、小面和尖锐角。

**使用选择组**

您可以使用鼠标按钮 3 创建临时选择组，以使几何诊断工具和修复工具的组合更易于使用。您可以使用几何诊断工具高亮显示无效和不精确几何的区域，您可以将该区域复制到选择组。然后，当指定要修复的区域时，您可以将选择组粘贴到几何编辑工具集中。

**使用显示组**

使用显示组有选择地移除面和胞，以帮助您查看复杂部件内部并确定几何诊断工具高亮显示的实体。

**采取增量方法**

对所选面和边的小组使用修复工具，并在继续之前检查修复过程的结果是否正确。

**保持实体完整**

尽量不要通过从实体移除面来将实体转换为壳。为了避免移除面，请为面使用 **Replace**（替换）工具，为边和面使用 **Repair small**（修复小边/小面）工具。

**避免创建线体**

如果您的部件是实体或壳，尽量不要执行导致线体的操作。Abaqus/CAE 很难从线体重建面。

**移除圆角和倒角**

使用 **Replace**（替换）工具和 **Extend neighboring faces**（扩展相邻面）选项从部件中移除圆角和倒角，并扩展相邻面以封闭 resulting gap。

**移除相切的面**

使用 **Replace**（替换）工具或 **Repair small**（修复小面）工具来修复近似相切的面。您应该使用 **Repair small**（修复小面）工具来移除相对较小的面。**Repair small**（修复小面）工具只是扩展剩余面直到它们相遇。在此过程中小面被删除，尽管结果可能不精确。为相对较大且相切的面使用 **Replace**（替换）工具。**Replace**（替换）工具使用底层点用近似曲面替换原始面。

**检查部件的有效性**

如果您的部件无效，您可以通过在模型树中的部件上点击鼠标按钮 3 并从出现的菜单中选择 **Update validity**（更新有效性）来检查修复操作是否使部件有效。检查有效性可能是一个耗时的操作，您应该等到完成编辑部件后再检查其有效性。如果部件已经有效，任何修复操作都不会使其无效。

**修复后创建分区**

如果您的部件包含分区，几何编辑工具集可能会在修复操作期间删除它们。为了避免此问题，您应该等到完成编辑后再对部件进行分区。
