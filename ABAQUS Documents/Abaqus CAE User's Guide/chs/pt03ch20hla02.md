# 20.14 添加参考几何







绘制特征的轮廓时，Abaqus/CAE 会将与草图平面相同的平面上的零件边和顶点投影到草图平面上。此外，Abaqus/CAE 会将所有基准轴和基准点投影到草图平面上。这些投影的线和点称为参考几何，您可以在草绘时使用它们作为参考。如果您想使用其他边和顶点（包括孤立单元边和孤立节点），则必须通过从主菜单栏中选择****添加**![](../graphics/images/arrow.gif)**参考****将它们作为参考几何投影到草图平面上。或者，如果您想从现有模型边创建新的草图边，可以从主菜单栏中选择****添加**![](../graphics/images/arrow.gif)**边****（有关更多信息，请参见["将边投影到草图上，"第 20.15 节](pt03ch20hla03.md)）。

**添加参考几何：**

1. 从 Sketcher 工具箱中，选择**投影参考**工具![](../graphics/ico_sketchProjectReferences.png)。有关 Sketcher 工具箱中工具的图表，请参见["Sketcher 工具，"第 20.4.1 节](pt03ch20s04s01.md)。Abaqus/CAE 在提示区域显示提示以指导您完成该过程。
2. 选择要作为参考几何投影到草图平面上的边、顶点、孤立单元边和孤立节点。
3. 完成选择后点击鼠标按钮 2。Abaqus/CAE 为投影的参考顶点和节点创建孤立点，并为几何边和孤立单元边创建参考边。
