# 71.1.1 常规查询







无论您使用哪个模块，都可以使用查询工具集获取模型的一般信息，但查询工具集在Job模块中不可用。Visualization模块中的**General Queries**与其他模块中的类似，但由于某些选择方法的差异以及结果中包含未变形和变形模型数据，它们在["查询模型，"第50章](pt05ch50.md)中单独讨论。

对于所有其他模块，查询工具集中**General Queries**下的项目提供以下一般信息：

**Point/Node**

所选点或节点的坐标

**Distance**

两个选定点或节点之间的距离；或在Part模块、Property模块或Mesh模块中，两个点、节点、边、面或这些对象的任意组合之间的距离

**Angle**

两条边或两个面之间的角度，或一条边与一个面之间的角度

**Feature**

对于选定的特征：
- 特征名称
- 描述
- 状态（如果特征被抑制或无法重新生成）
- 父特征名称
- 子特征名称
- 参数

**Shell element normals**

显示壳/膜法线方向

**Beam element tangents**

显示梁/桁架切线方向

此外，如果当前视口包含网格，查询工具集提供以下信息：

**Mesh stack orientation**

对于可在连续壳、内聚、圆柱形或垫片网格中使用的六面体、楔形和四边形单元，Abaqus/CAE会指示网格堆栈方向。对于六面体和楔形单元，Abaqus/CAE将顶面染成棕色，底面染成紫色。对于四边形单元，箭头指示单元的方向。此外，Abaqus/CAE会高亮显示方向不一致的任何单元面和边。

**注意：**查询结果不考虑在Property模块中定义固体复合铺层或复合壳铺层时对网格堆栈方向所做的更改。

****Mesh****

对于装配体、部件或部件实例、几何区域或单元：
- 所选区域中的节点和单元总数
- 每种单元形状的单元数量

默认情况下，Abaqus/CAE在消息区域显示网格信息，但您可以通过在提示区域切换**Display detailed report**在**Mesh statistics**对话框中以表格格式显示此信息。**Mesh statistics**对话框还允许您按部件实例或按单元类型显示网格信息。

**Element**

对于选定的单元：
- 单元标签
- 单元拓扑；例如，线性六面体
- Abaqus单元名称；例如，C3D8I
- 节点连通性

**Mesh gaps/intersections**

对于选定的部件或部件实例：
- 显示具有不兼容界面的边界面的单元边
- 显示具有裂纹或间隙的边界面的单元边
- 显示与其他面相交的边界面的单元边

**Mass properties**

对于装配体、选定的部件或部件实例、几何区域、实体单元、壳、实体面、梁或桁架，Abaqus/CAE返回以下部分或全部信息：
- 表面积（仅对壳单元和实体面显示）
- 面积质心
- 体积
- 体积质心
- 质量
- 质心
- 关于质心或关于指定点的惯性矩

有关此查询的更多信息，请参见["查询质量属性，"第71.2.3节](pt06ch71s02hlb03.md)。

**Geometry diagnostics**

- 无效、不精确或小几何体
- 拓扑

有关更多信息，请参见["获取模型的一般信息，"第71.2.2节](pt06ch71s02hlb02.md)。
