# 2.1.5 网球拍和球









**产品：** Abaqus/Explicit  

本示例模拟了网球以6.706 m/sec（264 in/sec）的速度斜向撞击球拍。本示例说明了变形表面与节点集之间的接触、初始应力的定义，以及使用基于表面的流体腔能力建模充满可压缩气体的流体腔。

### 问题描述

网球拍上的弦线使用T3D2桁架单元建模。它们被假定为线弹性，杨氏模量为6.895 GPa（1.0×10⁶ psi），泊松比为0.3，密度为1143 kg/m³（1.07×10⁴ lb·sec²/in⁴）。弦线处于44.48 N（10 lb）的初始张力下，使用初始应力条件指定。

球拍框被假定为刚性，使用R3D4单元建模。弦线（桁架单元）周围的节点与用于R3D4单元的节点相同。刚性球拍框的参考节点施加了边界条件以约束刚体上的所有六个自由度，使球拍框不移动。

网球建模为球体，使用150个S4R壳单元。它被假定由橡胶制成，用一般超弹性材料性能建模为Mooney-Rivlin材料，常数为![](../graphics/exa_eqn00117.gif) 0.690 MPa（100 lb/in²）和![](../graphics/exa_eqn00118.gif) 0.173 MPa（25 lb/in²）。Abaqus/Explicit对超弹性材料需要一些可压缩性。在此处显示的结果中，![](../graphics/exa_eqn00119.gif) 0.0145 MPa⁻¹（10⁴ psi⁻¹）。这给出了初始体积模量（![](../graphics/exa_eqn00417.gif)），它是初始剪切模量（![](../graphics/exa_eqn00418.gif)）的80倍。此比率低于典型橡胶的比率，但在本例中结果对该值不特别敏感，因为橡胶是无约束的。如果橡胶被较硬的相邻组件或增强材料约束，则需要更准确地表示材料的可压缩性。将![](../graphics/exa_eqn00419.gif)降低一个数量级（从而将初始体积模量增加10倍）对整体结果影响很小，但由于体积模量增加，会导致稳定时间增量减少一个![](../graphics/exa_eqn00420.gif)因子 网球的密度为1068 kg/m³（1.07×10⁴ lb·sec²/in⁴）。

网球除了100 kPa（14.7 psi）的大气压力外，还处于41 kPa（6 psi）的初始内部压力下。在网球内部定义了一个基于单元的表面。此表面用于定义充满气体的流体腔。网球内气体的分子量和摩尔热容被定义为流体腔流体行为的一部分。气体的分子量和摩尔热容被任意选择为0.062 kg（0.1367 lb）和28.110 J/kg·K（112.847 lb·in/lbm·K）。由于球对气体是不可渗透的，当球的体积减小时气体压力将增加，反之亦然。静力平衡给出球壳单元中初始双轴膜应力为![](../graphics/exa_eqn00421.gif) 155 kPa（22.5 psi）以平衡内部压力（这里*p*是内部气体压力，*r*是球半径，*t*是网球厚度）。球中的这种初始应力状态使用初始应力条件定义。

指定球和弦线之间的摩擦系数为0.1。球以6.706 m/sec（264 in/sec）的速度在15°角撞击弦线。

没有尝试生成球和弦线的准确模型：模型参数的选择只是为了提供相对于弦线的"软"球来说明接触效应。

完整模型如图2.1.5-1所示。模型中有2241个自由度。

在网球上定义了基于单元的表面。由于桁架单元是线单元，它们不形成平面表面。定义了包含弦线所有节点的基于节点的表面。然后使用接触对来定义球的基于单元表面与基于节点表面中定义的任何节点之间的接触。还提供了一个使用通用接触算法的输入文件。

### 结果和讨论

图2.1.5-2显示了未变形构型中球相对于弦线的位置。分析不同时刻的变形形状如图2.1.5-3到图2.1.5-7所示。图2.1.5-8显示了模型能量的时间历史。这些包括总内能（ALLIE）、动能（ALLKE）、粘性耗散（ALLVD）、摩擦耗散的能量（ALLFD）、外功（ALLWK）和模型的总能平衡（ETOTAL）。可以看到总能量在分析过程中保持几乎恒定，这是应有的情况。图2.1.5-9和图2.1.5-10给出了球内部压力历史和球实际体积历史。可以看到球内气体压力和球体积在10毫秒后都稳定了。
