# 14.9.2 创建步骤





您可以创建Abaqus分析产品允许的任何过程序列；**创建步骤**对话框中的过程列表会更新以仅显示新步骤的可用过程。例如，如果您的第一个步骤包含静态应力/位移过程，则不能用包含热传递过程的新步骤跟随它。

**要创建步骤：**

1. 从主菜单栏，选择**步骤**![](../graphics/images/arrow.gif)**创建**。**创建步骤**对话框出现。**提示：**您可以通过另外两种方式启动**创建**过程：
   - 在**步骤管理器**中点击**创建**。（您可以通过从主菜单栏选择**步骤**![](../graphics/images/arrow.gif)**管理器**来显示**步骤管理器**。）
   - 点击Step模块工具箱中的![](../graphics/ico_step.png)工具。
2. 如果需要，使用**名称**文本字段更改新步骤的名称。所有步骤必须具有唯一名称，且不能将步骤命名为"Initial"。
3. 从现有步骤列表中，选择新步骤将被插入其后的步骤。
4. 点击**过程类型**字段旁边的箭头，并从出现的列表中选择**通用**或**线性扰动**。对话框的下半部分显示可用过程列表。
5. 选择所需的过程并点击**继续**。**编辑步骤**对话框出现。
6. 使用**编辑步骤**对话框从默认值修改设置，并为可选设置提供值。（有关特定编辑器功能的详细帮助，请从主菜单栏选择**帮助**![](../graphics/images/arrow.gif)**上下文帮助**，然后点击感兴趣的功能。）
7. 点击**确定**。Abaqus/CAE关闭**编辑步骤**对话框，新步骤出现在**步骤管理器**中。
