# 9.7 总结







- Abaqus/Explicit 使用中心差分规则通过时间显式地积分运动学。
- 显式方法需要许多小的时间增量。由于不需要求解联立方程，每个增量都很便宜。
- 随着模型规模的增加，显式方法比隐式方法节省大量成本。
- *稳定性极限*是可用于推进运动状态并保持准确性的最大时间增量。
- Abaqus/Explicit 在整个分析过程中自动控制时间增量大小以保持稳定性。
- 随着材料刚度增加，稳定性极限减小；随着材料密度增加，稳定性极限增加。
- 对于单一材料的网格，稳定性极限大致与最小单元维度成正比。
- 通常，在 Abaqus/Explicit 中使用质量比例阻尼来阻尼低频振荡，使用刚度比例阻尼来阻尼高频振荡。
- 在某些情况下，Abaqus/Explicit 分析可能变得不稳定。本章中的示例问题描述了如何识别和纠正不稳定问题。
