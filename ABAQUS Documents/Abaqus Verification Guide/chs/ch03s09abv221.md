# 3.9.1 体积拖曳







**产品：**Abaqus/Standard  Abaqus/Explicit  

### 测试的单元

AC1D2    AC1D3    

AC2D3    AC2D4    AC2D4R    AC2D6    AC2D8    

AC3D4    AC3D5    AC3D6    AC3D8    AC3D8R    AC3D10    AC3D15    AC3D20    

ACAX3    ACAX4    ACAX4R    ACAX6    ACAX8    

### 测试的功能

在稳态（直接和基于子空间）以及瞬态分析中，对具有高不连续性的体积拖曳进行声学分析。

### 问题描述

模型由一根长度为4 m、截面积恒定的流体管组成。管水平放置（沿*x*轴），在*x* = 0 m处有一个声源，以向内体积加速度的形式给出。从*x* = 0 m到*x* = 3 m，管内的声学材料为空气，体积模量为1.424×10^5 N/m^2，密度为1.21 kg/m^3。从*x* = 3 m到*x* = 4 m的区域填充了耗散材料，其体积模量和密度与空气相同，但体积拖曳为10,000 Ns/m^4。在*x* = 4 m处的条件为封闭端。管使用400个一阶或200个二阶声学单元建模。

对于这些空气常数，声速为*c* = 343 m/s。在最高频率1100 Hz时，波长为0.312 m。网格的节点间距（节点之间的距离）始终为0.01 m；因此，在这个频率下，每波长有30个一阶单元或15个二阶单元。

在Abaqus/Standard中，对从100到1100 Hz的3个频率执行直接解和基于子空间的稳态动态分析。瞬态模拟在Abaqus/Explicit中使用100 Hz的激励频率执行。可以通过更改输入文件中定义的参数来测试不同的激励频率。瞬态分析也在Abaqus/Standard中使用AC2D4单元执行，以作为Abaqus/Explicit的参考解决方案。

### 结果与讨论

在最高频率下，Abaqus/Standard中二阶网格的结果与空气中压力和相位的解析解误差在0.1%以内。对于一阶网格，结果误差在7%以内。正如所预期的，对于相同数量的自由度，二阶单元的性能比一阶单元好得多。在较低频率下（每波长的单元数更多），两种网格类型的结果都有所改善。

Abaqus/Explicit中瞬态分析的结果与从Abaqus/Standard获得的结果非常一致。
