# 15 变更摘要





本节总结了在对定义Abaqus模型的元素、关键词、用户子程序和输出变量所做的更改和添加。有关这些修改的更多信息，请参阅前面的章节。

使用以下标识符：

| new | 6.14中的新功能。 |
| --- | --- |
| mod | 在6.13中存在，但在6.14中已修改或增强。 |
| rem | 在6.13中存在，但在6.14中已移除。 |
| (S) | Abaqus/Standard中的新增、修改或移除。 |
| (E) | Abaqus/Explicit中的新增、修改或移除。 |
| (C) | Abaqus/CFD中的新增、修改或移除。 |
