# 16.11.2 定义硬化场





您可以创建硬化场来定义塑性硬化的状态变量初始值。有关更多信息，请参见["Abaqus/Standard和Abaqus/Explicit中的初始条件"章节，《Abaqus Analysis User's Guide》第34.2.1节](../usb/usb-link.md#usb-prc-pinitialcond)。

**创建或编辑硬化场的步骤：**

1. 使用以下方法之一显示硬化场编辑器： - 要创建新的硬化场，请按照["创建预定义场"章节第16.8.3节](pt03ch16s08hlb03.md)中概述的程序进行操作（**类别**：`机械`；**所选步骤的类型**：`硬化`）。 - 要使用菜单或管理器编辑现有的硬化场，请参见["编辑依赖于步骤的对象"章节第3.4.13节](pt01ch03s04hlb05.md)。**注意：**您只能在初始步骤中创建或编辑硬化场。
2. 单击**分布**字段右侧的箭头，然后从出现的列表中选择所需的选项： - 选择**均匀**来定义在该区域上均匀的初始值。 - 选择解析场来定义空间变化的初始值。或者，您可以单击![](../graphics/ico_analyticalField.png)来创建新的解析场。（有关更多信息，请参见[第58章，"解析场工具集"](pt06ch58.md)。）
3. 单击**定义**字段右侧的箭头，然后从出现的列表中选择所需的选项： - 选择**运动硬化**来为运动硬化材料模型指定初始等效塑性应变和初始背应力张量。 - 选择**可压碎泡沫**来为可压碎泡沫材料模型指定初始体积压缩塑性应变。 - 选择**钢筋**来为钢筋层指定初始等效塑性应变和初始背应力张量。（此选项仅对Abaqus/Standard分析有效。） - 选择**截面点**来指定壳单元厚度方向上截面点的初始等效塑性应变和初始背应力张量。 - 选择**用户自定义**来在用户子程序[`HARDINI`](../sub/sub-link.md#sub-xsl-hardini)中指定初始等效塑性应变，以及（如果相关）初始背应力张量。（此选项仅在Abaqus/Standard分析中有效。）有关更多信息，请参见以下章节： - ["指定常规作业设置"章节第19.8.6节](pt03ch19s08hlb06.md) - ["HARDINI"，《Abaqus User Subroutines Reference Guide》第1.1.12节](../sub/sub-link.md#sub-rtn-uhardini)
4. 如果适用，单击**背应力数量**字段右侧的箭头来指定要包含在模型中的背应力数量。默认的背应力数量为1。允许的最大背应力数量为10。**数据**表中会出现额外的列来指定多个背应力张量。
5. 在**数据**表中，输入与您的**定义**选择相关的数据（并非以下所有参数都适用）：**等效塑性应变**初始等效塑性应变，![](../graphics/usi_eqn00639.gif)。**体积塑性应变**初始体积压缩塑性应变，![](../graphics/usi_eqn00640.gif)。**钢筋层名称**钢筋层名称。**截面点编号**截面点编号。**Alpha 1_11、Alpha 1_22等**初始第一背应力张量每个分量的值，![](../graphics/usi_eqn00641.gif)。**Alpha *k*_11、Alpha *k*_22等**初始第*k*个背应力张量每个分量的值，![](../graphics/usi_eqn00642.gif)。
6. 单击**确定**保存数据并退出编辑器。
