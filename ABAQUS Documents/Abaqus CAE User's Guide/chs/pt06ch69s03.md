# 69.3 什么是缝合？





选择此选项时，Abaqus/CAE会尝试沿其自由边连接相邻的面。缝合的意图通常是创建实体零件。缝合边通常会产生有效的几何形状。Abaqus/CAE通过调整底层曲面直到边缘相交来缝合修剪曲面之间的间隙。[图69-8](pt06ch69s03.md#dbs-conc-trim-stitch)说明了缝合过程。

**图69-8** 缝合边直到它们相交。

![](../graphics/dbs-conc-trim-stitch-nls.png)

Abaqus/CAE尝试从一组修剪曲面中识别实体的边缘。如果每个修剪曲面的边缘接近相邻曲面的边缘（在用户指定的容差范围内），Abaqus/CAE会缝合这些修剪曲面。缝合的最终结果是一个B-rep实体，其中两个相交曲面共享边缘定义。有关B-rep实体的更多信息，请参见["实体模型如何表示实体？," 第10.1.7节](pt02ch10s01s07.md)。缝合在存在线或内部面的情况下会失败。

当您导入IGES-或VDA-FS格式的零件时，Abaqus/CAE默认会缝合该零件。但是，由于IGES-和VDA-FS格式零件的内部容差，生成的小特征的表示可能与原始文件中预期的几何形状不匹配。

在某些情况下，缝合IGES-或VDA-FS格式的零件会导致Abaqus/CAE导入一个应该作为单独零件导入的单一零件。如果您将导入的零件复制到新零件，可以选择将断开的区域分离为单独的零件。有关更多信息，请参见["复制零件," 第11.18.3节](pt03ch11s18hlb03.md)。
