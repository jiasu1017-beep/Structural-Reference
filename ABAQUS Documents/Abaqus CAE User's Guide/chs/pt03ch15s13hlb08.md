# 15.13.10 在Abaqus/Explicit分析中指定接触控制





您可以为Abaqus/Explicit分析中的表面对表面接触和自接触相互作用指定接触控制。更多信息，请参见「Abaqus/Explicit中接触对的接触控制」，Abaqus Analysis User's Guide第36.5.5节。

**警告：**接触控制适用于高级用户。这些控制的默认设置适合大多数分析。使用这些控制的非默认值可能会大大增加分析的计算时间或产生不准确的结果。

**在Abaqus/Explicit分析中指定接触控制的步骤：**

1. 从主菜单栏选择 ****Interaction**![](../graphics/images/arrow.gif)**Contact Controls**![](../graphics/images/arrow.gif)**Create****。
2. 在出现的「Create Contact Controls」对话框中，执行以下操作：- 为接触控制命名。- 选择「Abaqus/Explicit contact controls」作为接触控制的类型。
3. 点击「Continue」关闭「Create Contact Controls」对话框。此时会出现「Edit Contact Controls」对话框。
4. 默认情况下，自接触相互作用的全局接触搜索之间的最大增量数为4，表面对表面接触相互作用为100。如果您想指定全局搜索之间的最大增量数，请切换编辑器的「Global Search Frequency」部分中的「Specify max number of increments」并输入值。
5. 默认情况下，Abaqus/Explicit使用局部接触搜索技术（局部跟踪），以最小的计算时间。如果您在实施适当接触条件方面有困难，可以关闭「Fast local tracking」以使用更保守的局部接触搜索。此设置仅适用于表面对表面接触相互作用。
6. 在「Penalty stiffness scaling factor」字段中，输入Abaqus/Explicit将用于缩放默认罚刚度的因子，以获得用于罚接触对的刚度。默认值为1。
7. 在「Warp check increment」字段中，输入主表面高度翘曲单元面检查之间的增量数。默认值为20。更频繁的检查会导致计算时间略有增加。
8. 在「Angle criteria for highly warped facet (degrees)」字段中，输入Abaqus/Explicit将认为单元面高度翘曲的平面外翘曲角（以度为单位）。平面外翘曲角定义为单元面上表面法线的变化量。默认值为20。
9. 点击「OK」创建接触控制并关闭编辑器。
