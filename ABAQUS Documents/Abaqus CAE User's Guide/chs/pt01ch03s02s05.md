# 3.2.5 理解确定、应用、默认值、继续、取消和关闭按钮





### 3.2.5 理解**确定**、**应用**、**默认值**、**继续**、**取消**和**关闭**按钮

当您完成对话框中的工作时，可以使用不同的操作按钮指定如何继续。例如，如果在对话框中输入数据，可以单击**确定**保存并应用数据。如果对话框是过程中间步骤的一部分，可以单击**继续**进入下一步。

对话框中可能出现以下操作按钮：

**确定**

单击**确定**可提交对话框的当前内容并关闭对话框。

**应用**

当您单击**应用**时，对话框中所做的任何更改都会生效，但对话框仍会显示。如果您想在关闭对话框之前查看这些更改的效果，此按钮很有用。

**默认值**

如果想在对话框中输入数据或指定首选项后恢复预定义的默认值，可以单击**默认值**。此按钮仅影响在对话框中输入的信息。它不会应用您的更改或关闭对话框；因此，要查看恢复默认值的効果，必须单击**应用**或**确定**。

**取消**

单击**取消**可关闭对话框而不应用您所做的任何更改。如果对话框出现在过程中间，单击**取消**通常也会取消该过程。在某些情况下，单击**取消**会使您返回到过程的上一步。

**继续**

出现在过程中间的对话框包含**继续**按钮。当您单击**继续**时，表示您已完成当前对话框中的数据输入，并希望进入过程的下一步。**继续**会导致对话框关闭，其中的所有数据都会被保存，除非您在过程的后续步骤中单击**取消**。

**关闭**

**关闭**按钮出现在包含您无法修改的数据的对话框中。例如，某些管理器包含存在对象的列表，但没有可以输入数据或指定首选项的字段。**关闭**按钮也会出现在消息对话框中。当您单击**关闭**时，对话框会关闭。

要关闭没有**取消**或**关闭**按钮的工具箱或对话框，请单击工具箱或对话框右上角的关闭按钮。或者，您可以通过按**[Esc]**关闭活动的工具箱或对话框。

**注意：**在Linux平台上，根据您的设置，**[Esc]**可能是关闭工具箱或对话框的唯一方法。有关更多信息，请参阅["影响Abaqus/CAE和Abaqus/Viewer的Linux设置，" Abaqus安装和许可指南第5.1.3节](../sgb/sgb-link.md#sgb-chp-view-cust-linux)。
