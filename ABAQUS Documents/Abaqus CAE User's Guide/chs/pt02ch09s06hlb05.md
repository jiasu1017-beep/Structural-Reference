# 9.7.5 自定义网络ODB连接器





如果您在与远程系统建立通信时遇到困难，或者您的站点需要特定配置，则可能需要自定义网络ODB连接器。使用**编辑网络ODB连接器（Edit Network ODB Connector）**对话框的**高级（Advanced）**选项卡页面来自定义网络ODB连接器。

**要自定义网络ODB连接器：**

1. 从主菜单栏，选择****文件（File）**![](../graphics/images/arrow.gif)**网络ODB连接器（Network ODB Connector）**![](../graphics/images/arrow.gif)**编辑（Edit）**![](../graphics/images/arrow.gif)***连接器名称*****。
2. 从出现的**编辑网络ODB连接器（Edit Network ODB Connector）**对话框中，单击**高级（Advanced）**选项卡。
3. 从**高级（Advanced）**选项卡页面中，选择服务器的启动方式。- 选择**自动启动服务器（Automatically start server）**来表示当您启动网络ODB连接器时，Abaqus/CAE将启动网络ODB服务器。- 选择**使用手动启动的服务器（Use manually started server）**来表示您已经使用**abaqus networkDBConnector**执行过程启动了网络ODB服务器。
4. 选择本地系统将用于在网络ODB服务器上执行命令的shell。- 选择**rsh**以使用远程shell命令。`rsh`命令随Windows和UNIX操作系统提供。rsh守护进程服务必须在远程机器上运行。- 选择**ssh**以使用安全shell命令。安全shell命令在与服务器通信时使用身份验证和加密，比远程shell命令提供更高的安全性。标准Windows安装不提供安全shell命令；您必须在工作站上安装和配置该命令。相比之下，一些Linux系统只提供安全shell命令，而远程shell命令不可用。如果安全shell命令不在PATH环境变量中可用，您必须在**ssh路径（ssh path）**字段中输入命令的完整路径。ssh守护进程服务必须在远程机器上运行。**注意：**安全shell命令和远程shell命令必须配置为不提示用户输入密码。有关更多信息，请参阅Dassault Systmes知识库，网址为[www.3ds.com/support/knowledge-base](http://www.3ds.com/support/knowledge-base)。
5. 如果您选择**自动启动服务器（Automatically start server）**来表示Abaqus/CAE将启动网络ODB服务器，请执行以下步骤：1. 从以下选项中选择以指定端口号：- 选择**自动分配端口（Auto-assign port）**以允许主机和远程系统建立自己的网络通信端口号。- 选择**指定端口（Specify port）**以强制主机和远程系统使用指定的端口号。在出现的**端口（Port）**字段中，输入所需的端口号。端口号必须是有效的端口号。您不能使用系统保留的端口号或已使用的端口号。2. 在**远程ABAQUS执行过程（Remote ABAQUS execution procedure）**字段中，输入在远程系统上运行Abaqus的命令。默认命令是您用于启动当前Abaqus/CAE会话的命令；但是，您的站点可能有用于执行Abaqus的自定义命令。3. 在**服务器超时（Server timeout）**字段中，输入网络ODB服务器超时时间（以分钟为单位）。默认值为一天（1440分钟）。如果在指定时间内服务器未收到来自客户端的任何通信，服务器将退出。无论此设置如何，如果您使用Abaqus/CAE启动了服务器，当您结束Abaqus/CAE会话时服务器将退出。您也可以通过从主菜单栏选择****文件（File）**![](../graphics/images/arrow.gif)**网络ODB连接器（Network ODB Connector）**![](../graphics/images/arrow.gif)**停止（Stop）**![](../graphics/images/arrow.gif)***服务器名称*****来停止服务器。4. 单击**确定（OK）**创建网络ODB连接器并关闭编辑器。您仍然需要启动网络ODB连接器以使其处于活动状态并打开远程输出数据库；有关更多信息，请参阅["管理网络ODB连接器，"第9.7.6节"](pt02ch09s06hlb06.md)。
6. 如果您选择**使用手动启动的服务器（Use manually started server）**来表示您已经从命令行启动了网络ODB服务器，请执行以下步骤：1. 输入**abaqus networkDBConnector**执行过程返回的**端口（Port）**号。2. 单击**确定（OK）**创建网络ODB连接器并关闭编辑器。您仍然需要启动网络ODB连接器以使其处于活动状态并打开远程输出数据库；有关更多信息，请参阅["管理网络ODB连接器，"第9.7.6节"](pt02ch09s06hlb06.md)。如果您从命令行手动启动了服务器，您可以使用**abaqus networkDBConnector**执行过程的**stop**参数关闭它，或者等待服务器超时。**abaqus networkDBConnector**执行过程在["网络输出数据库文件连接器，"Abaqus Analysis User's Guide第3.2.24节"](../usb/usb-link.md#usb-int-dremoteodbproc)中有描述。
