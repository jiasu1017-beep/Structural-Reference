# 20.10.10 草绘样条曲线







使用 Sketcher 工具箱中的样条曲线工具绘制连接一系列点的样条曲线。Abaqus/CAE 使用沿样条曲线所有点之间的三次样条拟合来计算曲线的形状；此外，样条曲线的一阶和二阶导数是连续的。在创建样条曲线时，您可以通过将顶点放得更近或更远来影响曲线的形状。但是，绘制样条曲线后，您无法向其添加或从中移除顶点；您必须删除样条曲线并使用所需数量的顶点创建新的样条曲线。您可以使用**修改**工具移动原始顶点。样条曲线的端点才会成为您正在创建或修改的零件的顶点；样条曲线的中间点不会出现在 Sketcher 之外。

如果希望样条曲线与现有直线相切起始，请在样条曲线起始处创建两个与直线共线的相邻顶点。这种方法对于创建平滑刚性表面很有用，如下图所示：

![](../graphics/ske-conc-spline2-nls.png)

当您创建分析[刚性表面](pt03ch11s07s01.md)时，您需要草绘一系列直线、圆弧和抛物线来定义其轮廓。要创建抛物线，请草绘仅由三个顶点定义的样条曲线。

**草绘样条曲线：**

1. 从 Sketcher 工具箱中，选择样条曲线工具![](../graphics/ico_sketchSpline.png)。有关 Sketcher 工具箱中工具的图表，请参见["Sketcher 工具，"第 20.4.1 节](pt03ch20s04s01.md)。Abaqus/CAE 在提示区域显示提示以指导您完成该过程。
2. 要构造样条曲线，请点击每个顶点。**提示：**如有必要，您可以使用提示区域中的文本框输入样条曲线每个顶点的精确坐标。有关创建所需样条曲线的更多信息，请参见["指定精确几何，"第 20.6 节](pt03ch20s06.md)。当您点击每个顶点或输入每个坐标时，样条曲线会出现，Abaqus/CAE 会调整曲线以在所有点之间保持三次样条。**提示：**如果在构造样条曲线时出错，您可以点击**上一步**按钮![](../graphics/afxI_backup.png)返回到上一个顶点。或者，您可以点击 Sketcher 工具箱中的**撤销**按钮删除整个样条曲线。
3. 要完成样条曲线，请点击鼠标按钮 2。
4. 要创建更多样条曲线，重复从第 2 步开始的上述步骤。
5. 完成创建样条曲线后，执行以下操作之一：- 在 Abaqus/CAE 窗口中的任意位置点击鼠标按钮 2。- 在 Sketcher 工具箱中选择任何其他工具。- 点击提示区域中的取消按钮![](../graphics/afxI_cancel.png)。
