# 2.6 卸载 Abaqus 文档



本节介绍如何从计算机中删除 Abaqus 文档。

### 2.6.1 Linux 平台

以下过程说明如何从 Linux 平台删除 Abaqus 文档。

**删除 HTML 和 PDF 文档的安装：**

可以通过使用以下过程之一停止 Web 服务器并删除所有已安装的文档版本或特定版本的目录及其所有从属文件来安全地删除 Abaqus HTML 和 PDF 文档。您必须具有管理员权限才能卸载 Abaqus 组件。
- 删除所有已安装的 Abaqus 文档版本： 1. 将当前目录设置为 Abaqus 文档的父目录 (*simulia_dir*)。 2. 如果您使用监控进程来提供 HTML 文档（即 Abaqus Web 服务器）或使用监控进程来执行搜索（即现有的非 Abaqus Web 服务器），请使用命令 ``` Documentation/bin/monitor -k ``` 停止监控进程 3. 通过键入命令删除文档目录 ``` rm -rf Documentation ``` 4. 从计算机的系统启动文件中删除 Web 服务器重启命令。请联系您的系统管理员或平台供应商，获取有关计算机系统启动文件的信息。 5. 如果您使用的是现有的非 Abaqus Web 服务器，可能需要执行其他步骤。有关更多信息，请参阅 Dassault Systèmes 知识库，网址为 [www.3ds.com/support/knowledge-base](http://www.3ds.com/support/knowledge-base)。
- 删除特定版本的 Abaqus 文档： 1. 将当前目录设置为 `*simulia_dir*/Documentation` 目录。 2. 如果您要删除的版本使用监控进程来提供 HTML 文档（即 Abaqus Web 服务器）或使用监控进程来执行搜索（即现有的非 Abaqus Web 服务器），请使用命令 ``` bin/monitor -k ``` 停止监控进程 3. 键入以下命令 ``` rm -rf installation_info/*release* rm -rf texis/*release* rm -rf docs/*release* ``` 其中 *release* 是您希望删除的 Abaqus 文档版本。删除文档版本后，您可以使用以下过程配置 Web 服务器来提供剩余的 HTML 集合： 1. 将当前目录设置为 `*simulia_dir*/Documentation` 目录。 2. 验证您希望提供的版本的版本号在 `texis.cnf` 文件中引用，如下所示： ``` Default Database = %INSTALLDIR%/texis/*release* ``` 其中 *release* 是您希望提供的文档版本。 3. 运行命令 ``` installation_info/*release*/initializeExecutable ``` 4. 要启动 Web 服务器来提供剩余的文档集合，请运行命令 ``` installation_info/*release*/startServer ```

**删除仅 PDF 文档的安装：**

1. 将当前目录设置为 Abaqus 文档的父目录 (*simulia_dir*)。
2. 通过键入命令删除文档目录 ``` rm -rf Documentation ```

### 2.6.2 Windows 平台

您可以使用 Windows 提供的卸载快捷方式、**添加或删除程序**工具或 Abaqus 无提示卸载程序批处理文件来删除 Abaqus 文档。无提示卸载程序允许您从另一个批处理/脚本文件自动执行这些任务。无提示卸载程序仅在 Windows 平台上可用。您必须具有 Windows 管理员权限才能卸载 Abaqus 组件。

Windows 卸载快捷方式和无提示卸载程序批处理文件都会生成一个可查看的日志文件。日志文件指示卸载是否成功，以及是否应该重启/重新启动计算机。此信息显示在日志文件的最后两行。例如，

```
Uninstall Status: SUCCESS
Restart Needed: YES_RECOMMENDED
```
`Uninstall Status` 行将指示 `SUCCESS` 或 `INCOMPLETE`。`Restart Needed` 行将指示 `YES_RECOMMENDED`、`YES_REQUIRED` 或 `NO`。

卸载程序可能无法删除某些文件；例如，在其他应用程序中打开的文件。此外，卸载程序仅删除 Abaqus 安装程序写入的目录和文件。您可能希望手动删除任何剩余的目录和文件。

**删除 HTML 和 PDF 文档的安装：**

可以通过以下任何方式删除 Abaqus HTML 和 PDF 文档：
- 从**开始**菜单，选择****程序**![](../graphics/images/arrow.gif)**Abaqus *release* 文档**![](../graphics/images/arrow.gif)**卸载文档****，其中 *release* 是您希望删除的 Abaqus 文档版本。
- 从**开始**菜单，选择****设置**![](../graphics/images/arrow.gif)**控制面板**![](../graphics/images/arrow.gif)**添加或删除程序****。选择 Abaqus 文档，然后单击**更改或删除程序**。
- 要使用 Abaqus 文档无提示卸载程序，请从命令行或从另一个批处理/脚本文件执行以下批处理文件： ``` *simulia_dir*\Documentation\installation_info\*release*\ html_uninstaller\silent_doc_uninstall.bat ```

文档卸载程序生成以下日志文件：
```
*simulia_dir*\Documentation\installation_info\*release*\
   uninstall_log.log
```
系统上其他版本的 Abaqus 文档不受此过程影响。

**删除仅 PDF 文档的安装：**

可以通过以下任何方式删除 Abaqus PDF 文档：
- 从**开始**菜单，选择****程序**![](../graphics/images/arrow.gif)**Abaqus *release* PDF 文档**![](../graphics/images/arrow.gif)**卸载文档****，其中 *release* 是您希望删除的文档版本。
- 从**开始**菜单，选择****设置**![](../graphics/images/arrow.gif)**控制面板**![](../graphics/images/arrow.gif)**添加或删除程序****。选择 Abaqus 文档，然后单击**更改或删除程序**。
- 要使用 Abaqus 文档无提示卸载程序，请从命令行或从另一个批处理/脚本文件执行以下批处理文件： ``` *simulia_dir*\Documentation\installation_info\*release*\ html_uninstaller\silent_doc_uninstall.bat ```

文档卸载程序生成以下日志文件：
```
*simulia_dir*\Documentation\installation_info\*release*\
   uninstall_log.log
```
系统上其他版本的 Abaqus 文档不受此过程影响。
