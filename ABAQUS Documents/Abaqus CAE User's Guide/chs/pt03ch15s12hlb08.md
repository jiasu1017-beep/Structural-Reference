# 15.12.8 创建单个连接器





**连接器构建器**使您能够执行建模单个连接器所涉及的所有步骤。这些步骤包括：
- 在两点之间或点与地面之间创建线特征。
- 在任一端点创建参考点，并沿线的长度创建基准坐标系。
- 为新线特征创建连接器截面分配。
- 指定线特征所连接的两个端点的方向。

由于**连接器构建器**一次创建一个连接器，因此您应该使用它向装配中添加少量连接器。如果您计划定义大量连接器，请首先在「Create Wire Feature」对话框中向装配添加线特征。请参见「为多个连接器创建或修改线特征」，第15.12.9节，获取更多信息。

**创建单个连接器的步骤：**

1. 从主菜单栏选择 ****Connector**![](../graphics/images/arrow.gif)**Connector Builder****。**提示：**您也可以使用「相互作用」模块工具箱中的![](../graphics/ico_connBuilder.png)工具启动**连接器构建器**。
2. 选择要连接的点。1. 在视口中选择连接器的第一个点和第二个点。**注意：**您也可以点击提示区域中的「Connect to Ground」将连接器的一侧接地。您不能将两个点都接地。「连接器构建器」打开，所选点显示在对话框的「Endpoints」部分。第一个点在视口中以红色高亮显示，第二个点以品红色高亮显示。2. 如有需要，点击![](../graphics/ico_swap.png)切换您选择的两个端点。3. 如有需要，切换「Create a reference point」下的任一点描述，以便在保存线特征时在该端点创建参考点。如果所选点是基准点或有趣点，Abaqus/CAE会自动在该位置创建参考点。
3. 选择连接器截面。如有需要，点击![](../graphics/ico_connectorSection.png)创建新的连接器截面。连接类型会显示在编辑器中；对于基本、组装和复杂连接类型，您可以点击![](../graphics/ico_tipSmall.png)查看连接类型的示意图以及Abaqus对连接的理想化。对于MPC连接类型，第二个点的运动被约束到第一个点的运动。您选择的连接类型也决定了为连接器方向选择的初始值。如果您更改了连接器截面分配，Abaqus/CAE会将这些方向设置重置为该连接器类型的默认值。
4. 如有需要，修改以下任一方向设置：1. 沿端点之间的轴投影基准坐标系。在对话框的「CSYS 1」部分，选择「Create CSYS on axis between points」选项，然后将「Axis」选择为新基准坐标系的「1」、「2」或「3」。对于某些连接器类型，Abaqus/CAE可能默认选择此选项。2. 在对话框的「CSYS 1」部分为连接器的第一个点指定全局坐标系以外的方向。如果连接器未连接到地面且所选连接器截面分配允许您调整连接器中的第二个点，您也可以在对话框的「CSYS 2」部分为第二个点指定方向。在对话框的任一部分中，选择「Specify CSYS」，点击![](../graphics/ico_selectBlue.png)，然后使用以下方法之一选择基准坐标系：- 按名称选择预定义的基准坐标系。从提示区域点击「Datum CSYS List」，从列表中选择一个名称，然后点击「OK」。- 在视口中选择预定义的坐标系。- 对于连接器中的第二个点，选择「Use CSYS 1」以在连接器的第二个点也使用为第一个点定义的坐标系。所选方向在视口中以红色高亮显示。「连接器构建器」重新出现并显示用于所选连接器点方向坐标系的描述。**提示：**您也可以通过点击对话框「CSYS 1」或「CSYS 2」部分的![](../graphics/ico_dtmCsys.png)来创建新的基准坐标系。3. 如有需要，您可以为任一端点指定关于基准坐标系轴的附加旋转。为点选择「Specify additional rotation」选项，输入旋转角度值（以度为单位），并为「About axis」选项选择「1」、「2」或「3」。仅当您指定全局坐标系以外的基准坐标系时，「Specify additional rotation」选项才可用。
5. 点击「OK」保存连接器并关闭「连接器构建器」，或点击「OK/Repeat」保存装配级线，保持对话框打开以创建新线。
