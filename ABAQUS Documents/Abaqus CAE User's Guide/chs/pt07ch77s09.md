# 77.9 显示多种颜色映射





您可以同时对视口应用两个或三个不同颜色映射的颜色编码。同时显示多种颜色映射可以揭示模型各方面之间的相互作用，这些作用在单独显示颜色映射时可能不明显。例如，您可能希望在同一个视口中同时显示**边界条件**和**载荷**的颜色映射。

要向**颜色映射**部分的对话框添加另一个颜色映射，请点击**高级**并从出现的列表中选择**添加映射**。Abaqus/CAE会向对话框添加另一个带标签的颜色映射，您可通过从**颜色编码依据**列表中选择映射来显示任何颜色映射。[图77-2](pt07ch77s09.md#uss-cco-colorcode-multiple)显示了显示**边界条件**和**载荷**颜色映射的**颜色编码**对话框。

**图77-2** 显示多种颜色映射的**颜色编码**对话框。

![](../graphics/uss-cco-colorcode-multiple-nls.png)

您还可以通过点击其标签并从**高级**列表中选择**移除当前映射**来从对话框中移除颜色映射。

当多个颜色映射打开时，Abaqus/CAE首先应用最左侧颜色映射的颜色映射，然后再处理其右侧的映射或多个映射。您最多可以同时显示三个颜色映射，但Abaqus/CAE不允许您重新排序表。如果要以不同顺序对属性进行颜色编码，必须关闭每个颜色映射，然后按您希望Abaqus/CAE在视口中显示它们的顺序添加颜色映射。
