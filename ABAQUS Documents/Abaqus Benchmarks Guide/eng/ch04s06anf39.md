# 4.6.2 NL2: Axisymmetric thick cylinder







**Product: **Abaqus/Standard  

### Elements tested

CAX8R    CCL24R    

### Problem description

![](../graphics/bmknl2.png)

**Material: **

Linear elastic, Young's modulus = 207 GPa, Poisson's ratio = 0.3, yield stress = 207.9 MPa.

**Boundary conditions: **

![](../graphics/bmk_eqn00175.gif) = 0 for all nodes.

**Loading: **

An initial internal pressure of 80 MPa is increased in steps of 20 MPa to 160 MPa.

### Results and discussion

All results agree exactly with the reference solution, except for the result of ![](../graphics/bmk_eqn00174.gif) at a load of 80 MPa and a distance of 124.2 mm. The value obtained here for both element types is 95.8 MPa, a difference of 2.74%.
