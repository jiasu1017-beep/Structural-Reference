# D 单元和输出变量支持







## 附录 D：单元和输出变量支持

大多数 Abaqus 单元在 Visualization 模块中受支持，但有以下限制：
- 垫片元素不支持用户定义的法线。
- 应用于非对称-轴对称模型的滑移线元素始终显示在![](../graphics/usi_eqn00065.gif)=0 平面中，无论为元素指定的![](../graphics/usi_eqn00065.gif)如何。
- CAXA 元素的扫描轮廓图可能包含不代表节点当前状态的颜色。使用两个傅里叶模式可以改善外观，但为获得最佳效果，请使用****报告**![](../graphics/images/arrow.gif)**场输出****打印积分点值。

有关 Abaqus/CAE 的 Mesh 模块中单元支持的信息，请参阅[「必须在 Mesh 模块外部生成哪些类型的单元？，」第 17.5.2 节](pt03ch17s05s02.md)。
