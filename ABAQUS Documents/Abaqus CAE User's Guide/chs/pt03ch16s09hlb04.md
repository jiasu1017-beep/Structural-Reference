# 16.9.4 定义壳边缘载荷





您可以创建壳边缘载荷以沿壳边缘定义一般载荷、剪切载荷、法向载荷、横 向载荷或力矩。

**创建或编辑壳边缘载荷：**

1. 使用以下方法之一显示壳边缘载荷编辑器：- 若要创建新的壳边缘载荷，请遵循"创建载荷"，第16.8.1节中概述的过程（**类别**：`Mechanical`；**所选步骤的类型**：`Shell edge load`）。- 若要使用菜单或管理器编辑现有的壳边缘载荷，请参见"编辑依赖于步骤的对象"，第3.4.13节。若要编辑应用载荷的区域，请参见"编辑应用规定条件的区域"，第16.8.4节。
2. 单击**分布**字段右侧的箭头，并从出现的列表中选择您的选项：- 选择**均匀**以定义在壳边缘上均匀的载荷。- 选择**用户定义**以在用户子程序[`UTRACLOAD`](../sub/sub-link.md#sub-xsl-utracload)中定义载荷幅值（对于Abaqus/Standard）。有关更多信息，请参见以下章节：- "指定常规作业设置"，第19.8.6节- "Abaqus User Subroutines Reference Guide"第1.1.52节"UTRACLOAD"- 选择分析场以定义空间变化载荷。只有对该载荷类型有效的分析场才会在选择列表中显示。或者，您可以单击![](../graphics/ico_analyticalField.png)创建新的分析场。（更多信息，请参见第58章"分析场工具集"。）
3. 单击**牵引**字段右侧的箭头，并从出现的列表中选择您的选项：- 选择**法向**以定义法向壳边缘牵引。- 选择**横向**以定义横向壳边缘牵引。- 选择**剪切**以定义剪切壳边缘牵引。- 选择**力矩**以定义壳边缘力矩。- 选择**一般**以定义一般壳边缘牵引。
4. 如果您选择了**一般**牵引类型，请指定载荷方向。1. 单击**矢量**旁边的![](../graphics/ico_selectBlue.png)以指定方向向量的坐标。2. 默认情况下，牵引分量是根据全局轴指定的。要引用牵引方向分量的局部坐标系：- 选择**CSYS：拾取的**并单击![](../graphics/ico_selectBlue.png)拾取先前定义的局部坐标系。- 选择**CSYS：用户定义**并输入定义局部坐标系的用户子程序的名称。3. 如果您选择了**CSYS：拾取的**，您可以定义绕一个轴的额外旋转。单击**绕轴额外旋转**字段右侧的箭头，选择其他两个轴将绕其旋转的轴，并输入额外旋转角度的值。
5. 在**幅值**文本字段中，输入壳边缘载荷幅值（单位[FL<sup>-1</sup>](../popups/usb-int-iconventions-unitsym.md)）。
6. 如果需要，单击**幅值**字段右侧的箭头，并从出现的列表中选择您选择的幅值。或者，您可以单击![](../graphics/ico_amplitudeSmall.png)创建新的幅值。（更多信息，请参见第57章"幅值工具集"。）
7. 如果需要，单击**牵引按单位定义**字段右侧的箭头，并选择**变形面积**以根据当前（变形）面积定义壳边缘载荷，或选择**未变形面积**以根据参考（原始）面积定义壳边缘载荷。
8. 如果您选择了**一般**牵引类型，您可以切换关闭**跟随旋转**以定义非跟随载荷（即，载荷始终沿固定全局方向作用，而不是在大变形分析中随壳边缘旋转）。
9. 单击**确定**保存您的数据并退出编辑器。
