# C.4 Visualization 模块中使用的符号









[表 C-8](ap03s04.md#vismodsymbols) 给出了 Visualization 模块使用的特殊图形符号。当您对包含任何这些组件的模型生成未变形、变形、轮廓或符号图时，会显示这些特殊符号。关于自定义这些符号的颜色和大小的信息，请分别参阅[「为 Visualization 模块中的所有几何体着色，」第 77.5 节](pt07ch77s05.md) 和[「控制模型实体的显示，」第 55.10 节](pt05ch55s10.md)。

**表 C-8** Visualization 模块使用的特殊图形符号。
| ![](../graphics/dashpota.png) | **两节点阻尼元件** |
| --- | --- |
| Visualization 模块使用所示符号显示两节点阻尼元件。您可以选择显示一条直线来代替此符号，方法是为您模型中的所有曲线边缘和面选择**extra coarse**细化。有关详细信息，请参阅[「细化曲线边缘和面，」第 55.12.3 节](pt05ch55s12hlb03.md)。 |
| ![](../graphics/spring2.png) | **两节点弹簧元件** |
| Visualization 模块使用所示符号显示两节点弹簧元件。您可以使用细化曲线边缘和面的选项来自定义此符号中曲线的外观。当您为您模型中的所有曲线边缘和面选择**Extra coarse**细化时，Visualization 模块将两节点弹簧元件显示为一条直线。有关详细信息，请参阅[「细化曲线边缘和面，」第 55.12.3 节](pt05ch55s12hlb03.md)。 |
| ![](../graphics/refnode2.png) | **参考节点** |
| Visualization 模块将刚体表面的参考节点显示为交叉符号。您无法自定义此符号的形状。 |
| ![](../graphics/tracer2.png) | **示踪粒子** |
| Visualization 模块将示踪粒子显示为方形内的交叉符号。您无法自定义此符号的形状。 |

除了 Visualization 模块用于显示上面列出的模型组件的特殊符号外，您还可以选择显示表示结果、模型实体（边界条件、连接器、坐标系和点元件）以及节点位置的符号。有关更多信息，请参阅以下章节：
