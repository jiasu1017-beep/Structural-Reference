# 48.1.2 Abaqus/CAE如何沿路径获取结果





Abaqus/CAE以X–Y数据对的形式获取沿路径的结果。Abaqus根据构成路径的点确定数据对X值。这些点定义了要获取数据的模型位置。Abaqus根据这些模型位置的分析结果确定数据对Y值。Abaqus/CAE在计算数据对时仅考虑当前显示组中的实体。

**路径数据X值**

您可以选择仅基于为路径指定的点形成X值，或者可选地包括路径与模型相交的所有位置。（相交发生在路径穿过单元面、单元边缘、表面面或表面边缘的地方。如果相交位于实际数据点的用户指定距离内，您可以从路径定义中忽略单个相交。您可以使用Abaqus Scripting Interface命令设置此相交容差值；有关详细信息，请参阅["Session对象，"Abaqus Scripting Reference Guide第55.18节](../ker/ker-link.md#ker-session-xy-pyc)。）

您还可以选择将构成路径的点解释为未变形模型形状或变形模型形状上的位置。当选择变形模型形状时，Abaqus/CAE将节点处的变形坐标计算为基坐标与该节点变形值（乘以视口特定的变形比例因子）之和。节点标签和边缘列表指示模型位置，同样适用于未变形或变形的模型形状。点列表和圆形路径坐标是独立于模型几何形状的固定位置。Abaqus不会为您选择的模型形状中不重合的点形成数据对。

构成路径的节点标签或点坐标通常不适合直接用作X值。Abaqus/CAE为您提供了多种选项将这些点系列转换为有用的X值和后续的X–Y曲线图轴标签。您可以选择以下选项之一将路径点转换为X值：
- **真实距离**：X值对应于每个点在模型空间坐标中沿路径的实际距离，从零开始。
- **归一化距离**：X值对应于每个点沿路径的距离，作为路径总长度的分数。
- **序列ID**：X值对应于每个点在路径结果列表中出现的顺序。
- **X、Y或Z距离**：X值对应于每个点在指定单一坐标方向上沿路径的实际距离，从零开始。此选项对于生成轴对称模型中结果与半径的关系图特别有用。

**路径数据Y值**

您可以通过选择Abaqus/CAE获取结果的分析步骤、帧和场输出变量来控制数据对Y值，并通过控制Abaqus/CAE计算某些类型结果的方式，如下所示：
- 对于要在节点位置获取的基于节点的场输出变量（如位移），Abaqus/CAE直接从输出数据库读取结果，无需进一步计算。
- 对于要在节点位置获取的基于单元的场输出变量（如应力或应变），Abaqus/CAE从输出数据库读取结果，将这些值外推到节点，然后根据您选择的方式有条件地对多个贡献进行平均。
- 对于在非节点位置的路径点上获取的基于节点和基于单元的变量，Abaqus/CAE使用单元形状的几何近似通过从节点插值到请求位置来计算值。您无法控制此计算。

对于基于单元变量的平均选项和复数值的复数形式选项位于"结果选项"对话框中。平均将多个贡献减少为单个值。当您选择部分或完全抑制平均的结果选项时，接收多个贡献的路径点会产生多个数据对。这些数据对共享相同的X值，但每个贡献都有单独的Y值。根据您的路径和模型的特性，可能需要以下技术来避免多个数据对共享相同的X值：
- 设置结果选项以完全启用平均。
- 设置结果选项以在计算值时忽略区域边界。
- 避免穿过区域不连续的路径。
- 避免沿分离不连续区域的线。
- 在获取沿路径的结果之前，使用显示组隔离单个区域。
- 避免穿过一个或多个空间点的点列表路径，这些空间点在视觉上位于高度变形单元和大角度所对的圆柱单元外部，但由于等参数映射，在数学上位 于单元内部。这些空间点往往被两个或多个单元共享，导致多个Y值。

有关值平均的详细信息，请参阅["理解结果值平均，"第42.6.2节](pt05ch42s04s02.md)。

如果所选场输出变量包含复数结果，则数据对的Y值会受到当前复数形式的影响。复数形式的缩写会附加到绘制路径时Y轴标题上。例如，如果`S-Mises`是所选的场输出变量，`Magnitude`是复数形式，则Y轴标题为`S-Mises CPX:Mg`。其他复数形式类似地缩写。有关复数形式的详细信息，请参阅["控制复数结果的形式，"第42.6.9节](pt05ch42s04hlb05.md)。

对于没有指定步骤、帧或场输出变量的结果的点，Abaqus/CAE不会形成数据对。
