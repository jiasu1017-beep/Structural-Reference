# 2.1.7 汽车悬架建模





**产品：** Abaqus/Standard   

本示例说明了JOINTC单元的使用。重复此示例以说明连接器单元的使用。JOINTC单元（["柔性连接单元，"Abaqus分析用户指南第32.3.1节](../usb/usb-link.md#usb-elm-ejoint)）可用于建模两个在几何上几乎重合且代表具有内部刚度和/或阻尼的连接的节点之间的相互作用。连接的行为在局部坐标系中定义（["方向，"Abaqus分析用户指南第2.2.5节](../usb/usb-link.md#usb-int-corientation)）。该系统随单元第一个节点的运动而旋转，可能由线性或非线性弹簧和阻尼器并联组成，在连接中耦合相对位移和相对旋转的相应分量。此功能可用于建模例如汽车悬架中的橡胶轴承。

在连接器单元模型中，JOINTC单元被连接器单元取代（见["连接器单元，"Abaqus分析用户指南第31.1.2节](../usb/usb-link.md#usb-elm-econnectorelem)），使用CARTESIAN连接类型定义平移行为，使用ROTATION定义旋转行为。这些连接类型允许在随单元上第一个节点旋转的局部坐标系中定义线性或非线性弹簧和阻尼器行为。可使用多种不同的连接类型来建模有限旋转响应。见["连接类型库，"Abaqus分析用户指南第31.1.5节](../usb/usb-link.md#usb-elm-econnectortypelibrary)，了解使用不同有限旋转参数化的连接类型。在此模型中，旋转幅度被假定为小。因此，使用ROTATION的连接的旋转向量参数化是合适的。

本示例的主要目的是验证在经历刚体旋转运动的结构中JOINTC和连接器单元的准确性。本示例的次要目的是演示在Abaqus/Standard中使用等效刚体运动输出变量。

### 几何与模型

分析的结构是汽车的左前悬架子总成（见图2.1.7-1）。集合中包含的物理组件是轮胎、车轮、车轴（毂）、A臂（叉形杆）、螺旋弹簧和车架。轮胎用JOINTC或连接器单元建模；连接了弯曲杆单元用于可视化。（使用JOINTC或连接器单元是因为这是在局部坐标系中定义轮胎非线性刚度的方便方法。）车轮的垂直刚度由梁单元表示。车轴和A臂都用梁单元建模。在JOINTC模型中车轴通过销型MPC连接到A臂，或在连接器模型中使用连接类型JOIN。螺旋弹簧在JOINTC模型中用SPRINGA单元建模，或在连接器模型中使用连接类型AXIAL。汽车车架由MASS单元表示。螺旋弹簧顶部直接连接到车架，而A臂通过两个JOINTC单元或两个具有连接类型CARTESIAN和ROTATION的连接器单元连接到车架（代表A臂衬套）。初始位置代表满载车辆，轮胎和螺旋弹簧具有相应的初始预载荷。

分析的第一步允许悬架系统达到平衡。第二步对轮胎越过道路凸起进行建模。凸起被理想化为高100 mm、长400 mm的三角形，假设车辆以5 km/hr行驶。第二个输入文件用于显示大旋转对悬架响应的影响。此文件包括一个初始旋转步骤，将模型绕垂直轴刚性旋转90°，但其他方面与第一个输入文件相同。我们预期两个分析的响应相同。

在此示例中，我们主要感兴趣的是A臂的等效刚体运动，特别是平均位移和旋转。

### 结果与讨论

连接器单元模型的结果在性质和数量上与JOINTC模型相当。因此，下面仅进一步讨论JOINTC结果。

第二步中垂直位移历史如图2.1.7-2所示，显示了轮胎与地面接触点、车轮中心和车架的位移。图2.1.7-3显示了一系列叠加的位移图，因为轮胎滚上凸起。

为了验证JOINTC单元在大旋转下的行为，第二个模型在施加凸起激励之前将整个结构刚性旋转90°。图2.1.7-4显示了两个模型的位移时间历史叠加在同一图上。它们几乎相同。

在经历大运动的可变形体分析中，获取有关等效刚体运动的信息是方便的：平均位移和旋转，以及关于质心的线和角动量。为此，Abaqus提供了一组等效刚体输出变量。如上所述，本示例的次要目的是演示这些输出变量的使用，它们代表指定单元集的平均运动。包括了对结果文件和数据文件的单元输出请求。如果未指定单元集，则给出整个模型的平均运动。这种类型的输出只能在直接积分隐式动态分析中请求，并且只有具有质量的单元才会对等效刚体运动做出贡献。关于可变形体等效刚体运动的精确定义，请参阅["等效刚体动态运动，"Abaqus理论指南第2.4.4节](../stm/stm-link.md#stm-anl-equivrbm)。

图2.1.7-5和图2.1.7-6使用等效刚体输出变量生成。图2.1.7-5显示了节点5001（A臂的支点"A"）、节点5080（A臂最接近轮胎的点）和A臂的平均垂直运动（输出变量UC3）的垂直运动。正如所预期的，组件质心的位移介于其两端的位移之间。图2.1.7-6显示了A臂组件关于其质心的平均刚体旋转。刚体旋转可用于关于三个全局轴。在此我们感兴趣的是关于全局*X*轴的旋转（输出变量URC1）。旋转的标准输出是弧度，但结果已被缩放以度为单位绘制旋转。

### 输入文件

[jointcautosuspension.inp](../eif/jointcautosuspension.inp)

使用JOINTC单元的悬架分析。

[jointcautosuspension_rotated.inp](../eif/jointcautosuspension_rotated.inp)

使用JOINTC单元的旋转悬架分析。此输入文件包括一个额外的（旋转）步骤，但其他方面与jointcautosuspension.inp相同。

[jointcautosuspension_depend.inp](../eif/jointcautosuspension_depend.inp)

与jointcautosuspension.inp相同，只是在JOINTC单元中使用了场变量相关的线性和非线性弹簧属性。

[connautosuspension.inp](../eif/connautosuspension.inp)

使用连接器单元的悬架分析。

[connautosuspension_rotated.inp](../eif/connautosuspension_rotated.inp)

使用连接器单元的旋转悬架分析。此输入文件包括一个额外的（旋转）步骤，但其他方面与connautosuspension.inp相同。

[connautosuspension_depend.inp](../eif/connautosuspension_depend.inp)

与connautosuspension.inp相同，只是在具有连接类型CARTESIAN、ROTATION和AXIAL的连接器单元中使用了场变量相关的线性和非线性弹簧属性。
