# 68.1 自定义内存限制和再生选项





几何状态是零件或独立零件实例内部表示的快照。将几何状态保存在内存缓存中可以加快再生性能；但是，保存的状态会消耗大量内存并降低整体性能。

**选项**对话框的**内存**选项卡上的选项允许您控制分配给Abaqus内核的内存量、指定触发Abaqus/CAE以减少内存模式运行的内存使用百分比，以及调整保存状态的便利性与增加内存消耗导致的性能下降之间的平衡。从任何模块的主菜单栏中选择****工具**![](../graphics/images/arrow.gif)**选项****以显示**选项**对话框，然后单击**内存**选项卡。

在大多数情况下，默认设置将产生可接受的再生性能。只有在Abaqus/CAE花费很长时间再生零件或装配体时才应修改特征选项。有关更多信息，请参见["调整特征再生," 第65.3节](pt06ch65s03.md)。

**要自定义会话的内存限制和内存缓存设置：**

1. 从任何模块的主菜单栏中，选择****工具**![](../graphics/images/arrow.gif)**选项****。将出现**选项**对话框。
2. 显示**内存**选项卡页面。
3. 如果需要，以兆字节为单位指定新的**内核内存限制**。如果此设置为零，则不对内核内存施加限制。
4. 切换**当内存达到时以减少内存模式运行**，并指定内核内存限制的百分比，超过该百分比时Abaqus/CAE以减少内存模式运行。Abaqus/CAE通过将几何数据分页到磁盘或删除可自动再生（需要时）的不活动零件中的数据来减少内存。
5. 如果需要，更改Abaqus/CAE自动缓存在内存中的几何状态数量。理想情况下，为了获得最大的再生性能，您可以在每次特征修改后保存几何状态。但是，存储大量几何状态会消耗大量内存并阻碍整体性能。决定保存多少个状态是再生速度与内存消耗之间的权衡。有关更多信息，请参见["什么是几何状态？," 第65.3.2节](pt06ch65s03s02.md)。
6. 如果发现Abaqus/CAE正在消耗工作站上的大量内存并影响系统性能，您可以使用**选项**对话框中的**清除**按钮来清除与零件和装配缓存关联的内存。清除此内存将提高性能；但是，几何状态的快照将被擦除，再生会更慢。有关更多信息，请参见["什么是缓存？," 第65.3.3节](pt06ch65s03s03.md)。
7. 如果要将当前状态保存在内存中，请单击**选项**对话框中的**缓存当前状态**。Abaqus/CAE可以返回此状态并再生任何后续的添加或修改。**选项**对话框指示缓存中存储了多少个几何状态。单击**查询**以确定几何状态相对于当前零件缓存或装配缓存中特征序列的定位。当Abaqus/CAE以减少内存模式运行时，如果内存消耗高于指定阈值，几何状态可能会被删除。
8. 单击**确定**更改当前模型的内存限制和再生性能选项。这些选项仅适用于当前会话，不会保存在模型数据库中。
