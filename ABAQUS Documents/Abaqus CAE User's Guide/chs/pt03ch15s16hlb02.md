# 15.16.4 为自动检测到的接触对创建相互作用

Abaqus/CAE根据您指定的参数为接触对候选者表中的每个项目创建接触相互作用和绑定约束。

**要创建接触相互作用和绑定约束：**

1. 从主菜单，选择**交互**![](../graphics/images/arrow.gif)**查找接触对**或**约束**![](../graphics/images/arrow.gif)**查找接触对**。**提示：**您也可以使用交互模块工具箱中的![](../graphics/ico_contactPairFind.png)工具来查找接触对。Abaqus/CAE显示接触检测对话框。
2. 按照["指定接触检测的搜索条件，"第15.16.1节](pt03ch15s16s01.md)中所述输入搜索参数。
3. 按照["执行接触对搜索，"第15.16.2节](pt03ch15s16hlb01.md)中所述执行接触对搜索。
4. 在接触对候选者表中，输入定义相互作用和绑定所需的所有参数，如["审查和修改检测到的接触对，"第15.16.3节](pt03ch15s16s02.md)中所述。
5. 点击**确定**。Abaqus/CAE创建指定的接触相互作用和绑定约束。
