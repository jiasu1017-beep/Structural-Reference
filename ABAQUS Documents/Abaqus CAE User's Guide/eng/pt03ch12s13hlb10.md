# 12.13.14 Creating fluid sections for porous media







Fluid sections are used to define the section properties of porous media in an Abaqus/CFD model. For more information, see [Chapter 30, "Fluid dynamic analyses](pt04ch30.md).”

**To specify fluid section properties for porous media:**

1. From the main menu bar, select ****Section**![](../graphics/images/arrow.gif)**Create****. A **Create Section** dialog box appears. **Tip:**You can also click **Create** in the **Section Manager** or select the create section tool ![](../graphics/ico_section.png) in the Property module toolbox.
2. Enter a section name. For more information on naming objects, see ["Using basic dialog box components," Section 3.2.1](pt01ch03s02s01.md).
3. Select **Fluid** as the section **Category** and **Porous** as the section **Type**, and click **Continue**. The fluid section editor appears.
4. Select a solid (matrix) material to be used for the porous media in the fluid section. If desired, click ![](../graphics/ico_material.png) to create a material; see ["Creating or editing a material," Section 12.7.1](pt03ch12s07hlb01.md), for more information.
5. Select a fluid material to be used for the porous media in the fluid section. If desired, click ![](../graphics/ico_material.png) to create a material; see ["Creating or editing a material," Section 12.7.1](pt03ch12s07hlb01.md), for more information.
6. Click **OK** to save your changes and to close the fluid section editor.
