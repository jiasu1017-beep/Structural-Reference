# 64.5.7 调整副中节点的位置





从主菜单栏选择****网格**![](../graphics/images/arrow.gif)**编辑****可以调整二次单元副中节点的位置。

您可以使用此工具来适应断裂力学分析中裂纹尖端的奇异性。但是，此工具允许您编辑任何二次单元；它不会检查您是否在编辑断裂力学分析所需的退化四边形或楔形单元。有关更多信息，请参见["控制小应变分析中裂纹尖端的奇异性，" 第31.2.5节](pt04ch31s02s05.md)，以及Abaqus Analysis User's Guide ["轮廓积分评估，" 第11.4.2节中的"使用常规有限元法进行小应变分析的断裂力学网格构建"](../usb/usb-link.md#usb-anl-acontintegral-singularity)。

您可以选择节点并输入介于0和1之间的偏置参数。Abaqus/CAE会沿着连接的单元边缘将副中节点移动到基于您输入的参数的位置。例如，如果您输入0.25的参数，Abaqus/CAE会将副中节点的位置偏置到远离所选节点的单元边缘长度的四分之一处。

**调整副中节点位置的步骤：**

1. 进入"网格"模块。
2. 从上下文栏的**对象**字段中选择**零件**，然后从列表中选择一个零件。此工具仅适用于包含孤兒网格节点的零件。
3. 从主菜单栏选择****网格**![](../graphics/images/arrow.gif)**编辑****。Abaqus/CAE显示**编辑网格**对话框。 **提示：**您也可以使用位于"网格"模块工具箱底部的![](../graphics/ico_meshEdit.png)工具显示**编辑网格**对话框。
4. 在对话框中执行以下操作： 1. 在**类别**字段中选择**节点**。 2. 从**方法**列表中选择**调整副中节点**。Abaqus/CAE显示现有节点。在隐藏和着色模式下，仅显示可见节点。
5. 选择副中节点沿连接单元边缘的位置将向其调整的节点。您可以选择节点，也可以选择单元边缘。如果您选择单元边缘，Abaqus/CAE会选择沿单元边缘的节点。您可以**按住[Shift]键并点击**各个节点将其添加到您的选择中，或**按住[Ctrl]键并点击**所选节点将其从选择中移除。 **提示：**您可以使用**选择**工具栏更改选择行为。有关更多信息，请参见[第6章，"在视口中选择对象"](pt01ch06.md)。如果您希望从现有节点集列表中进行选择，请执行以下操作： 1. 点击提示区域右侧的**集合**。Abaqus/CAE显示**区域选择**对话框，其中包含您已创建的节点集列表。 2. 选择节点集，然后点击**继续**。 **注意：**默认选择方法基于您最近使用的选择方法。要切换到另一种方法，请点击提示区域右侧的**在视口中选择**或**集合**。
6. 在提示区域输入偏置参数的值。您必须输入介于0和1之间的值。Abaqus/CAE会将副中节点沿连接的单元边缘移动到指定位置。 **提示：**如果在调整节点时出错，点击**编辑网格**对话框中的**撤销**可以撤消调整。Abaqus/CAE会将节点恢复到之前的位置。
7. 根据需要重复步骤4和5以调整其他副中节点。
8. 完成节点调整后，点击提示区域中的取消按钮![](../graphics/afxI_cancel.png)退出过程。
