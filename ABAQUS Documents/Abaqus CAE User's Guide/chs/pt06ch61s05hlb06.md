# 61.5.6 将工具栏恢复为默认设置





**Customize** 对话框中的 **Reset** 按钮将所有工具栏恢复为其原始设置。重置工具栏：
- 将所有默认 Abaqus/CAE 工具栏返回到其原始可见性设置；
- 将所有默认 Abaqus/CAE 工具栏重新定位到主菜单栏下方的原始位置；并
- 删除所有自定义工具栏。

这些设置不会立即生效；您必须重新启动 Abaqus/CAE 才能看到更改。在单击 **Reset** 按钮和重新启动 Abaqus/CAE 之间对工具栏设置进行的任何更改都不会被保存。

**将工具栏恢复为默认设置的步骤**

1. 在任何模块中，从主菜单栏选择 ****Tools**![](../graphics/images/arrow.gif)**Customize****。会出现 **Customize** 对话框。

2. 在 **Toolbars** 选项卡页面上，单击 **Reset**。

3. 重新启动 Abaqus/CAE。
