# 17.3.4 自顶向下网格划分







自顶向下网格划分依赖于零件的几何形状来定义网格的外部边界。自顶向下网格与几何形状匹配；您可能需要简化和/或分割复杂几何形状，以便 Abaqus/CAE 识别可用于生成高质量网格的基本形状。在某些情况下，自顶向下方法可能不允许您使用所需类型的单元对复杂零件的部分进行网格划分。自顶向下技术——结构化、扫掠和自由网格划分——及其几何要求定义明确，施加到零件上的载荷和边界条件会自动与生成的网格关联。

**结构化网格划分**

结构化网格划分是为您提供最多网格控制权的自顶向下技术，因为它将预先建立的网格模式应用于特定的模型拓扑。大多数未分割的实体模型过于复杂，无法使用预先建立的网格模式进行网格划分。但是，您通常可以将复杂模型分割为具有结构化网格划分模式存在的简单拓扑的区域。图 17-3 显示了结构化网格的示例。有关更多信息，请参见"结构化网格划分和映射网格划分"第 17.8 节。

**扫掠网格划分**

Abaqus/CAE 通过在边缘或面上内部生成网格，然后沿扫掠路径扫掠该网格来创建扫掠网格。结果可以是边缘生成的二维网格，也可以是面生成的三维网格。与结构化网格划分一样，扫掠网格划分是一种受特定拓扑和几何形状限制的自顶向下技术。图 17-4 显示了扫掠网格的示例。有关更多信息，请参见"扫掠网格划分"第 17.9 节。

**图 17-4** 扫掠网格。

![](../graphics/mgn-swept3d-nls.png)

**自由网格划分**

自由网格划分技术是最灵活的自顶向下网格划分技术。它不使用预先建立的网格模式，可应用于几乎任何模型形状。但是，自由网格划分为您提供最少的网格控制，因为无法预测网格模式。图 17-5 显示了自由网格的示例。有关更多信息，请参见"自由网格划分"第 17.10 节。
