# 11.5.4 充气器定义

**产品：** Abaqus/Standard Abaqus/Explicit

##### **参考**

- ["Fluid cavity overview," Section 11.5.1](pt04ch11s05aus70.md)
- [*INFLATOR](../key/key-link.md#usb-kws-minflator)

### 概述

充气器（inflator）是安全气囊系统中的关键组件，用于描述气体发生器向气囊腔体充气的过程。充气器定义包括充气器特性、充气速率曲线和气体属性。

### 充气器类型

#### 化学推进剂充气器

使用固体推进剂燃烧产生气体。充气过程通常持续20-100毫秒。

特性：
- 高充气速率
- 短充气时间
- 气体温度高（需考虑热效应）

#### 压缩气体充气器

使用预压缩气体（如氮气）作为气源。充气过程持续时间取决于储气罐压力和体积。

特性：
- 相对平稳的充气速率
- 可预测的充气过程
- 气体温度接近环境温度

#### 混合型充气器

结合化学推进剂和压缩气体的优点，先使用压缩气体快速充气，再用推进剂补充。

### 充气器参数

定义充气器时需要指定以下参数：

#### 充气器类型

- **TYPE=CHEMICAL**：化学推进剂充气器
- **TYPE=GAS**：压缩气体充气器
- **TYPE=HYBRID**：混合型充气器

#### 充气器特性

```
[*INFLATOR](../key/key-link.md#usb-kws-minflator), TYPE=CHEMICAL, CAVITY=*腔体名称*
*充气器直径
*充气速率曲线
```

### 充气速率曲线

充气速率曲线定义充气器产生的气体质量流率随时间的变化。曲线可以通过以下方式定义：

#### 用户定义曲线

直接指定时间-质量流率数据点：

```
*INFLATOR, TYPE=GAS, CAVITY=airbag
0.01
0.0, 0.0
0.001, 0.5
0.01, 1.0
0.05, 0.8
0.1, 0.0
```

#### 预定义曲线类型

也可以使用Abaqus提供的标准充气曲线类型：
- **SCALED**：缩放型曲线
- **PEAK**：峰值型曲线
- **TRIANGULAR**：三角型曲线

### 气体属性

#### 化学推进剂气体

化学推进剂产生的气体需要定义：
- 气体常数（对于燃气混合物）
- 比热容比（gamma）
- 气体成分

#### 压缩气体

对于压缩气体充气器，需要定义：
- 气体类型（如氮气、空气）
- 储气罐初始压力
- 储气罐体积

### 充气器位置

充气器位置定义了气体进入腔体的位置。可以指定：
- 充气器出口节点位置
- 充气器出口面积

### 与流体腔体耦合

充气器向流体腔体充气，需要使用**FLUID CAVITY**定义接收气体的腔体。

充气器与腔体的连接通过：
- 充气器出口面积
- 充气器直径

### 温度效应

#### 化学推进剂

化学推进剂产生的气体温度可能高达2000-3000K，需要考虑：
- 高温对气体属性的影响
- 对腔体壁面的热传递
- 热应力效应

#### 压缩气体

压缩气体充气时可能产生冷却效应（焦耳-汤姆逊效应），尤其对于实际气体。

### 诊断

充气器仿真常见问题：

1. **充气速率过高**
   - 检查充气速率曲线
   - 考虑使用更长的充气时间

2. **气囊折叠展开异常**
   - 检查初始体积和压力
   - 验证表面接触定义

3. **收敛困难**
   - 减小初始时间步长
   - 检查气体属性定义

### 示例输入

```
*FLUID CAVITY, NAME=airbag
airbag_surface, INITIAL VOLUME=0.001, INITIAL PRESSURE=0.0
*FLUID PROPERTY, TYPE=GAS
1.4
*INFLATOR, TYPE=CHEMICAL, CAVITY=airbag
0.02
0.0, 0.0
0.002, 0.5
0.01, 1.0
0.05, 0.3
0.1, 0.0
*STEP
*DYNAMIC, EXPLICIT
0.1
*END STEP
```
