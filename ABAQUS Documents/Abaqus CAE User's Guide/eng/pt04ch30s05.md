# 30.5 Meshing a fluid model







The fields active in a fluid analysis are not determined by the element type but by the analysis procedure and its options. The sole purpose of the element type is to define the shape of the element used to discretize the continuum. Three fluid type elements are available. Fluid elements are assigned to the mesh by default. For more information, see ["Fluid continuum elements," Section 28.2 of the Abaqus Analysis User's Guide](../usb/usb-link.md#usbfluidcont).
