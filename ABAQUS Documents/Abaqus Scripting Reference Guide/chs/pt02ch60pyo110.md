# 60.110 VolumetricTestData 对象





VolumetricTestData 对象提供体积测试数据。

**访问**

```
materialApi.materials()[*name*].hyperelastic().volumetricTestData()
materialApi.materials()[*name*].hyperfoam().volumetricTestData()
materialApi.materials()[*name*].viscoelastic().volumetricTestData()
```

### 60.110.1 VolumetricTestData(...)

此方法创建一个 VolumetricTestData 对象。

**路径**

```
materialApi.materials()[*name*].hyperelastic().VolumetricTestData
materialApi.materials()[*name*].hyperfoam().VolumetricTestData
materialApi.materials()[*name*].viscoelastic().VolumetricTestData
```

**原型**

```
odb_VolumetricTestData&
VolumetricTestData(const odb_SequenceSequenceDouble& table,
                   odb_Union volinf,
                   odb_Union smoothing,
                   bool temperatureDependency,
                   int dependencies);
```

**必需参数**

*table*

一个 odb_SequenceSequenceDouble，指定下述项目。

**可选参数**

*volinf*

字符串 "NONE" 或一个 Double，指定归一化体积值，该值取决于 [Viscoelastic](pt02ch60pyo106.md) 对象的 *time* 成员的值。默认值为 "NONE"。

如果 *time*="RELAXATION_TEST_DATA"，*volinf* 指定长期归一化体积模量 ![](../graphics/ker_eqn00113.gif) 的值。

如果 *time*="CREEP_TEST_DATA"，*volinf* 指定长期归一化体积柔量 ![](../graphics/ker_eqn00114.gif) 的值。

此参数仅对粘弹性材料模型有效。

*smoothing*

字符串 "NONE" 或一个 Int，指定平滑值。如果 *smoothing*="NONE"，则不使用平滑。默认值为 "NONE"。

*temperatureDependency*

一个布尔值，指定数据是否依赖于温度。默认值为 false。

*dependencies*

一个 Int，指定场变量依赖项的数量。默认值为 0。

**表数据**

对于超弹性或超泡沫材料模型，表数据指定以下内容：
- 压力，![](../graphics/ker_eqn00092.gif)。
- 体积比，![](../graphics/ker_eqn00022.gif)（当前体积/原始体积）。

对于粘弹性材料模型，值取决于 [Viscoelastic](pt02ch60pyo106.md) 对象的 *time* 成员的值。

如果 *time*=RELAXATION_TEST_DATA，表数据指定以下内容：
- 归一化体积（体积）模量 ![](../graphics/ker_eqn00401.gif)。
- 时间 ![](../graphics/ker_eqn00093.gif) ![](../graphics/ker_eqn00121.gif)。

如果 *time*=CREEP_TEST_DATA，表数据指定以下内容：
- 归一化体积（体积）柔量 ![](../graphics/ker_eqn00402.gif)。
- 时间 ![](../graphics/ker_eqn00093.gif) ![](../graphics/ker_eqn00121.gif)。

**返回值**

一个 VolumetricTestData 对象。

**异常**

无。
