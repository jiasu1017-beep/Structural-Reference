# 14.11.2 配置线性扰动分析过程

### 配置静态、线性扰动过程

在线性分析步骤中，响应是关于*基础状态*的线性扰动响应。基础状态是线性扰动步骤之前最后一个常规分析步骤结束时模型的当前状态。如果分析的第一个步骤是扰动步骤，则基础状态由初始条件确定。更多信息，请参见Abaqus Analysis User's Guide的第6.1.3节"常规和线性扰动过程"中的"线性扰动分析步骤"。

**创建或编辑静态、线性扰动过程：**

1. 按照"创建步骤"，第14.9.2节（**过程类型：**`线性扰动；静态，线性扰动`）或"编辑步骤"，第14.9.3节中概述的过程显示**编辑步骤**对话框。
2. 在**基本**和**其他**选项卡页上，按照以下过程中概述的方式描述步骤并输入方程求解器偏好设置。

**配置静态、线性扰动过程：**

1. 在**编辑步骤**对话框中，显示**基本**选项卡页。
2. 在**描述**字段中，输入分析步骤的简短描述。Abaqus将您输入的文本存储在输出数据库中，该文本由可视化模块显示在状态块中。
3. 显示**其他**选项卡页。
4. 选择**方程求解器方法**选项：- 选择**直接**以使用默认的直接稀疏求解器。- 选择**迭代**以使用迭代线性方程求解器。迭代求解器通常对于具有数百万自由度的大块结构最有用。更多信息，请参见Abaqus Analysis User's Guide的第6.1.6节"迭代线性方程求解器"。
5. 选择**矩阵存储**选项：- 选择**使用求解器默认值**以允许Abaqus/Standard决定是否需要对称或非对称矩阵存储和求解方案。- 选择**非对称**以将Abaqus/Standard限制为非对称存储和求解方案。- 选择**对称**以将Abaqus/Standard限制为对称存储和求解方案。有关矩阵存储的更多信息，请参见Abaqus Analysis User's Guide的第6.1.2节"定义分析"中的"Abaqus/Standard中的矩阵存储和求解方案"。
6. 完成静态、线性扰动步骤的设置配置后，单击**确定**关闭**编辑步骤**对话框。

### 配置频率过程

本节提供配置频率提取过程的详细说明。

#### 频率提取过程概述

当您为频率过程配置步骤时，Abaqus/Standard会在该步骤期间执行特征值提取过程，以计算系统的固有频率和相应的振型。

配置频率步骤时，您必须选择以下特征值提取方法之一：

**Lanczos**

Lanczos方法是默认方法，因为它具有通用能力。但是，此方法通常比AMS方法慢。更多信息，请参见Abaqus Analysis User's Guide的第6.3.5节"固有频率提取"中的"Lanczos特征求解器"和Abaqus Theory Guide的第2.5.1节"特征值提取"。

有关配置Lanczos特征值提取方法设置的详细说明，请参见"使用Lanczos特征求解器进行频率提取过程"。

**自动多级子结构化（AMS）**

AMS方法是Abaqus/Standard的附加分析能力。AMS方法比Lanczos方法快，特别是在您需要大量特征模态且系统具有许多自由度时。但是，AMS方法有几个限制。更多信息，请参见Abaqus Analysis User's Guide的第6.3.5节"固有频率提取"中的"自动多级子结构化（AMS）特征求解器"。

有关配置AMS提取方法设置的详细说明，请参见"使用AMS特征求解器进行频率提取过程"。

**子空间迭代**

对于子空间迭代过程，您只需指定所需的特征值数量；Abaqus/Standard选择合适的向量数进行迭代。您还可以指定感兴趣的最小频率；Abaqus/Standard提取特征值，直到提取了所需数量的特征值或最后提取的频率超过最大感兴趣频率。更多信息，请参见Abaqus Theory Guide的第2.5.1节"特征值提取"。

有关配置子空间迭代提取方法设置的详细说明，请参见"使用子空间迭代特征求解器进行频率提取过程"。

有关频率提取过程的背景信息，请参见Abaqus Analysis User's Guide的第6.3.5节"固有频率提取"。

#### 使用Lanczos特征求解器进行频率提取过程

**编辑步骤**对话框提供**基本**和**其他**选项卡页，您可以在其中指定Lanczos特征求解器的设置。

您可以按照"创建步骤"，第14.9.2节（**过程类型：**`线性扰动；频率`）或"编辑步骤"，第14.9.3节中概述的过程显示**编辑步骤**对话框。

**在**基本**选项卡页上配置设置：**

1. 在**编辑步骤**对话框中，显示**基本**选项卡页。
2. 在**描述**字段中，输入分析步骤的简短描述。Abaqus将您输入的文本存储在输出数据库中，该文本由可视化模块显示在状态块中。
3. 从**特征求解器**选项列表中，选择**Lanczos**。
4. 选择**请求的特征值数量**选项之一以指示您要计算的特征值数量：- 选择**频率范围内的全部**如果您希望Abaqus/Standard计算由最大频率值（以及，如果需要，最小频率值）确定的范围内的所有特征值。- 选择**值**如果您希望计算特定数量的特征值。在提供的字段中输入该值。
5. 切换打开**频率偏移（周期/时间）**以指定正或负的偏移平方频率*S*。如果您切换打开此选项，请在提供的字段中输入频率偏移的值。更多信息，请参见Abaqus Analysis User's Guide的第6.3.5节"固有频率提取"中的"频率偏移"。
6. 切换打开**感兴趣的最小频率（周期/时间）**以指定Abaqus/Standard将计算特征值的频率范围的下限。如果您切换打开此选项，请在提供的字段中输入最小频率的值。如果要以并行模式使用Lanczos求解器，则需要此选项。
7. 切换打开**感兴趣的最大频率（周期/时间）**以指定Abaqus/Standard将计算特征值的频率范围的上限。如果您切换打开此选项，请在提供的字段中输入最大频率的值。如果要以并行模式使用Lanczos求解器，则需要此选项。
8. 如果模型包含耦合在一起的声学和结构单元或ASI型单元，您可以切换打开**在适用的情况下包括声-结构耦合**以在频率提取期间包括声-结构耦合效应。（此选项默认处于打开状态。）更多信息，请参见Abaqus Analysis User's Guide的第6.3.5节"固有频率提取"中的"结构-声耦合"。
9. 指定您的块大小偏好设置：- 选择**默认值**以使用默认块大小7，这通常是合适的。- 选择**值**以输入特定的块大小。在提供的字段中输入该值。通常，Lanczos方法的块大小应与最大预期特征值多重性一样大。更多信息，请参见Abaqus Analysis User's Guide的第6.3.5节"固有频率提取"中的"Lanczos特征求解器"。
10. 指定**最大块Lanczos步数**的偏好设置：- 选择**默认值**以允许Abaqus/Standard在每次Lanczos运行中确定块Lanczos步数。默认值通常是合适的。- 选择**值**以输入每次Lanczos运行中Lanczos步数的限制。在提供的字段中输入该值。通常，如果特定类型的特征问题收敛缓慢，提供更多块Lanczos步将降低分析成本。另一方面，如果您知道特定类型的问题收敛快速，提供更少的块Lanczos步将减少使用的核心内存量。
11. 切换打开**使用基于SIM的线性动力学过程**以激活高性能、基于模态的线性动力学分析能力。如果在此步骤中提取的特征模态将用于后续的基于模态或基于子空间的线性动态过程，基于SIM的能力可为大型模型提供更好的效率，而输出请求最少。更多信息，请参见Abaqus Analysis User's Guide的第6.3.1节"动态分析过程：概述"中的"使用SIM架构进行模态叠加动态分析"。
12. 如果您在**基本**选项卡页上切换打开了**使用基于SIM的线性动力学过程**，Abaqus会投影在模型中定义的结构和黏性阻尼算子，以供后续的稳态动态步骤使用。要停用阻尼算子的投影，请切换关闭**投影阻尼算子**。停用阻尼算子可以提高频率提取步骤的性能；但是，结构和黏性材料阻尼将在后续稳态动态步骤中被忽略。更多信息，请参见Abaqus Analysis User's Guide的第6.3.1节"动态分析过程：概述"中的"使用SIM架构的基于模态的稳态和瞬态线性动态分析中的阻尼"。
13. 切换打开**包括残余模态**以请求Abaqus/Standard根据紧邻前面的静态、线性扰动步骤中指定的载荷计算残余模态。更多信息，请参见Abaqus Analysis User's Guide的第6.3.5节"固有频率提取"中的"获取用于基于模态过程的残余模态"。

**在**其他**选项卡页上配置设置：**

1. 在**编辑步骤**对话框中，显示**其他**选项卡页。
2. 接受默认**矩阵存储**设置。由于Abaqus/Standard仅对对称矩阵提供特征值提取，因此具有特征频率提取或特征值屈曲预测过程的步骤始终使用对称矩阵存储和求解方案。您无法更改此设置。在此类步骤中，Abaqus/Standard将对刚度矩阵的所有贡献进行对称化。
3. 选择用于归一化特征向量的选项：- 选择**位移**以归一化特征向量，使每个向量中的最大位移、旋转或声压项为单位1。- 选择**质量**以相对于结构的质量矩阵归一化特征向量（特征向量被缩放，使每个向量的广义质量为单位1）。如果您在**基本**选项卡页上切换打开了**使用基于SIM的线性动力学过程**，则只有质量归一化可用。更多信息，请参见Abaqus Analysis User's Guide的第6.3.5节"固有频率提取"中的"归一化"。
4. 切换打开**在频率处评估相关属性**以在特征值提取期间评估黏弹性、弹簧和阻尼器的频率相关属性。如果您切换打开此选项，请在提供的字段中输入所需的评估频率。更多信息，请参见Abaqus Analysis User's Guide的第6.3.5节"固有频率提取"中的"评估频率相关材料属性"。

完成频率步骤的设置配置后，单击**确定**关闭**编辑步骤**对话框。

#### 使用AMS特征求解器进行频率提取过程

**编辑步骤**对话框提供**基本**和**其他**选项卡页，您可以在其中指定AMS特征求解器的设置。

您可以按照"创建步骤"，第14.9.2节（**过程类型：**`线性扰动；频率`）或"编辑步骤"，第14.9.3节中概述的过程显示**编辑步骤**对话框。

**在**基本**选项卡页上配置设置：**

1. 在**编辑步骤**对话框中，显示**基本**选项卡页。
2. 在**描述**字段中，输入分析步骤的简短描述。Abaqus将您输入的文本存储在输出数据库中，该文本由可视化模块显示在状态块中。
3. 从**特征求解器**选项列表中，选择**AMS**。
4. 结构和声学区域的耦合不影响使用AMS特征求解器的频率提取过程：结构和声学模态被计算为好像未耦合。默认情况下，Abaqus会投影在模型中定义的声-结构耦合算子，以供后续的稳态动态步骤使用。要停用投影，请切换关闭**在适用的情况下投影声-结构耦合**。停用耦合算子可以提高分析速度，但声-结构耦合将在后续稳态动态步骤中被忽略。
5. 对于同时包含结构和声学元素的模型，Abaqus默认在计算声学特征值时使用与计算结构特征值时相同的频率范围。要为两个区域指定不同的频率范围，请输入**声学范围因子**。Abaqus将此因子乘以**感兴趣的最大频率**来确定声学区域的不同最大频率。两个区域的最小频率保持相同。
6. 切换打开**感兴趣的最小频率（周期/时间）**以指定Abaqus/Standard将计算特征值的频率范围的下限。如果您切换打开此选项，请在提供的字段中输入最小频率的值。
7. 在**感兴趣的最大频率（周期/时间）**字段中，输入Abaqus/Standard将提取所有模态的频率范围的上限。
8. 切换打开**限制保存的特征向量的区域**如果您希望将特征向量计算限制为仅特定区域中的节点。如果您切换打开此选项，请单击提供的字段右侧的箭头以选择感兴趣的区域。
9. 默认情况下，Abaqus会投影在模型中定义的任何结构或黏性阻尼算子，以供后续的稳态动态步骤使用。要停用这些算子的投影，请切换关闭**投影阻尼算子**。停用阻尼算子可以提高频率提取步骤的性能；但是，结构和黏性材料阻尼将在后续稳态动态步骤中被忽略。更多信息，请参见Abaqus Analysis User's Guide的第6.3.1节"动态分析过程：概述"中的"使用SIM架构的基于模态的稳态和瞬态线性动态分析中的阻尼"。
10. 切换打开**包括残余模态**以请求Abaqus/Standard根据模型对标称（或单位）载荷的静态响应计算残余模态。如果您切换打开此选项，请指定残余模态区域和您要计算残余模态的自由度（DOF）。Abaqus/Standard为每个请求的自由度计算一个残余模态。更多信息，请参见Abaqus Analysis User's Guide的第6.3.5节"固有频率提取"中的"获取用于基于模态过程的残余模态"。

**在**其他**选项卡页上配置设置：**

1. 在**编辑步骤**对话框中，显示**其他**选项卡页。
2. 接受默认**矩阵存储**设置。由于Abaqus/Standard仅对对称矩阵提供特征值提取，因此具有特征频率提取或特征值屈曲预测过程的步骤始终使用对称矩阵存储和求解方案。您无法更改此设置。在此类步骤中，Abaqus/Standard将对刚度矩阵的所有贡献进行对称化。
3. 在**子结构特征问题的截止乘数**字段中，输入![](../graphics/usi_eqn00424.gif)，即子结构特征问题的截止频率，定义为感兴趣最大频率的乘数。默认值为5。
4. 在**简化特征问题的第一个截止乘数**字段中，输入![](../graphics/usi_eqn00425.gif)，即简化特征问题的第一个截止频率，定义为感兴趣最大频率的乘数。默认值为1.6。![](../graphics/usi_eqn00426.gif)。
5. 在**简化特征问题的第二个截止乘数**字段中，输入![](../graphics/usi_eqn00427.gif)，即简化特征问题的第二个截止频率，定义为感兴趣最大频率的乘数。默认值为1.3。![](../graphics/usi_eqn00428.gif)。更多信息，请参见Abaqus Analysis User's Guide的第6.3.5节"固有频率提取"中的"自动多级子结构化（AMS）特征求解器"。

完成频率步骤的设置配置后，单击**确定**关闭**编辑步骤**对话框。

#### 使用子空间迭代特征求解器进行频率提取过程

**编辑步骤**对话框提供**基本**和**其他**选项卡页，您可以在其中指定子空间迭代特征求解器的设置。

您可以按照"创建步骤"，第14.9.2节（**过程类型：**`线性扰动；频率`）或"编辑步骤"，第14.9.3节中概述的过程显示**编辑步骤**对话框。

**在**基本**选项卡页上配置设置：**

1. 在**编辑步骤**对话框中，显示**基本**选项卡页。
2. 在**描述**字段中，输入分析步骤的简短描述。Abaqus将您输入的文本存储在输出数据库中，该文本由可视化模块显示在状态块中。
3. 从**特征求解器**选项列表中，选择**子空间**。
4. 在**请求的特征值数量**字段中，输入您希望Abaqus/Standard计算的特征值数量。
5. 切换打开**频率偏移（周期/时间）**以指定正或负的偏移平方频率*S*。如果您切换打开此选项，请在提供的字段中输入频率偏移的值。更多信息，请参见Abaqus Analysis User's Guide的第6.3.5节"固有频率提取"中的"频率偏移"。
6. 切换打开**感兴趣的最大频率（周期/时间）**以指定Abaqus/Standard将计算特征值的频率范围的上限。如果您切换打开此选项，请在提供的字段中输入最大频率的值。Abaqus/Standard将提取特征值，直到提取了所需数量的特征值或最后提取的频率超过最大感兴趣频率。
7. 输入**每次迭代使用的向量数**的值。一般来说，使用更多向量收敛更快，但内存需求也越大。因此，如果您知道特定类型的特征问题收敛缓慢，通过此选项提供更多向量可能会降低分析成本。
8. 输入**最大迭代次数**的值。默认值为30。

**在**其他**选项卡页上配置设置：**

1. 在**编辑步骤**对话框中，显示**其他**选项卡页。
2. 接受默认**矩阵存储**设置。由于Abaqus/Standard仅对对称矩阵提供特征值提取，因此具有特征频率提取或特征值屈曲预测过程的步骤始终使用对称矩阵存储和求解方案。您无法更改此设置。在此类步骤中，Abaqus/Standard将对刚度矩阵的所有贡献进行对称化。
3. 选择用于归一化特征向量的选项：- 选择**位移**以归一化特征向量，使每个向量中的最大位移、旋转或声压项为单位1。- 选择**质量**以相对于结构的质量矩阵归一化特征向量（特征向量被缩放，使每个向量的广义质量为单位1）。更多信息，请参见Abaqus Analysis User's Guide的第6.3.5节"固有频率提取"中的"归一化"。
4. 切换打开**在频率处评估相关属性**以在特征值提取期间评估黏弹性、弹簧和阻尼器的频率相关属性。如果您切换打开此选项，请在提供的字段中输入所需的评估频率。更多信息，请参见Abaqus Analysis User's Guide的第6.3.5节"固有频率提取"中的"评估频率相关材料属性"。

### 配置屈曲过程

特征值屈曲分析通常用于估计刚性结构（经典特征值屈曲）的临界屈曲载荷。这种分析是线性扰动过程，屈曲载荷是相对于结构的基础状态计算的。更多信息，请参见Abaqus Analysis User's Guide的第6.2.3节"特征值屈曲预测"。

**创建或编辑屈曲过程：**

1. 按照"创建步骤"，第14.9.2节（**过程类型：**`线性扰动；屈曲`）或"编辑步骤"，第14.9.3节中概述的过程显示**编辑步骤**对话框。
2. 在**基本**选项卡页上，按照以下过程中概述的方式配置特征求解器提取方法和最大迭代次数等设置。**其他**选项卡页显示默认矩阵存储选项。您无法更改此设置，因为Abaqus/Standard仅对对称矩阵提供特征值提取。在特征值屈曲预测过程中，Abaqus/Standard对刚度矩阵的所有贡献进行对称化。

**在**基本**选项卡页上配置设置：**

1. 在**编辑步骤**对话框中，显示**基本**选项卡页。
2. 在**描述**字段中，输入分析步骤的简短描述。Abaqus将您输入的文本存储在输出数据库中，该文本由可视化模块显示在状态块中。
3. 选择**Lanczos**特征求解器或**子空间**迭代特征求解器。当需要大量特征模态且系统具有许多自由度时，Lanczos方法通常更快。但是，此方法确实有一些限制（请参见**基本**选项卡页底部的警告）。当只需要少量（少于20个）特征模态时，子空间迭代方法可能更快。更多信息，请参见Abaqus Analysis User's Guide的第6.2.3节"特征值屈曲预测"中的"选择特征值提取方法"。
4. 在**请求的特征值数量**字段中，输入您希望估计的特征值数量。严重高估实际特征值数量可能会创建非常大的文件。如果您低估了实际特征值数量，Abaqus/Standard将发出相应的警告消息。
5. 如果您选择了**Lanczos**特征求解器，请执行以下操作：1. 切换打开**感兴趣的最小特征值**以输入Abaqus/Standard将提取的特征值范围的下限。如果您切换打开此选项，请在提供的字段中输入该值。2. 切换打开**感兴趣的最大特征值**以输入Abaqus/Standard将提取的特征值范围的上限。如果您切换打开此选项，请在提供的字段中输入该值。如果您指定了特征值范围，Abaqus/Standard将提取特征值，直到提取了所需数量的特征值或给定范围内的所有特征值都已提取。3. 选择**块大小**选项：- 选择**默认值**以使用默认块大小7，这通常是合适的。- 选择**值**以在提供的字段中输入特定的块大小。一般来说，Lanczos方法的块大小应与最大预期特征值多重性一样大。更多信息，请参见Abaqus Analysis User's Guide的第6.2.3节"特征值屈曲预测"中的"选择特征值提取方法"。4. 指定**最大块Lanczos步数**的偏好设置：- 选择**默认值**以允许Abaqus/Standard在每次Lanczos运行中确定块Lanczos步数。默认值通常是合适的。- 选择**值**以输入每次Lanczos运行中Lanczos步数的限制。一般来说，如果特定类型的特征问题收敛缓慢，提供更多块Lanczos步将降低分析成本。另一方面，如果您知道特定类型的问题收敛快速，提供更少的块Lanczos步将减少使用的核心内存量。
6. 如果您选择了**子空间**特征求解器，请执行以下操作：1. 切换打开**感兴趣的最大特征值**以输入Abaqus/Standard将提取的特征值范围的上限。如果您切换打开此选项，请在提供的字段中输入该值。Abaqus/Standard将提取特征值，直到提取了所需数量的特征值或最后提取的特征值超过最大感兴趣特征值。2. 输入**每次迭代使用的向量数**的值。一般来说，使用更多向量收敛更快，但内存需求也越大。因此，如果您知道特定类型的特征问题收敛缓慢，通过此选项提供更多向量可能会降低分析成本。3. 输入**最大迭代次数**的值。默认值为30。
7. 单击**确定**保存步骤并关闭**编辑步骤**对话框。

### 配置响应谱过程

您可以使用响应谱分析来估计结构对特定基础运动的峰值响应（位移、应力等）。该方法只是近似的，但通常是初步设计研究的有用且廉价的方法。响应谱过程基于系统模态的一个子集，这些模态必须首先使用特征频率提取过程提取（如"配置频率过程"中所述）。更多信息，请参见Abaqus Analysis User's Guide的第6.3.10节"响应谱分析"。

**创建或编辑响应谱过程：**

1. 按照"创建步骤"，第14.9.2节（**过程类型：**`线性扰动；响应谱`）或"编辑步骤"，第14.9.3节中概述的过程显示**编辑步骤**对话框。
2. 在**基本**和**阻尼**选项卡页上，按照以下过程中概述的方式配置阻尼系数和多方向激励组合方法等设置。

**在**基本**选项卡页上配置设置：**

1. 在**编辑步骤**对话框中，显示**基本**选项卡页。
2. 在**描述**字段中，输入分析步骤的简短描述。Abaqus将您输入的文本存储在输出数据库中，该文本由可视化模块显示在状态块中。
3. 单击**激励**字段右侧的箭头，然后选择方向求和方法。- 对于以下选项，Abaqus/Standard首先对方向激励分量求和，然后执行模态求和：- 选择**单一方向**以代数方式对单个方向的方向激励分量求和。- 选择**多方向绝对值求和**以代数方式对多个方向的方向激励分量求和。- 对于以下选项，Abaqus/Standard首先执行模态求和，然后对方向激励分量求和：- 选择**多方向平方和根**以使用平方和的平方根对多个方向的方向激励分量求和。- 选择**多方向30%规则**以使用30%规则对多个方向的方向激励分量求和。- 选择**多方向40%规则**以使用40%规则对多个方向的方向激励分量求和。更多信息，请参见Abaqus Analysis User's Guide的第6.3.10节"响应谱分析"中的"方向求和方法"。
4. 单击**求和**字段右侧的箭头，然后选择模态求和方法。有关每种方法的信息，请参见Abaqus Analysis User's Guide的第6.3.10节"响应谱分析"中的"模态求和方法"。
5. 在**第一方向**选项卡页上（如果适用，还在**第二方向**和**第三方向**选项卡页上），执行以下操作：1. 在**使用响应谱**字段中，选择用于计算响应的谱振幅。或者，您可以单击![](../graphics/ico_amplitudeSmall.png)创建新的幅值。（有关更多信息，请参见第57.11节"定义谱"。）。2. 输入**方向余弦X**、**Y**和**Z**。3. 在**缩放因子**字段中，输入乘以响应谱中幅值的因子。4. 在**持续时间**字段中，输入动态事件的持续时间，该事件创建了响应谱。仅当指定了**双重和组合**模态求和方法时，此设置才适用。对于多方向激励，您可以选择在**第三方向**选项卡页上切换打开**应用第三方向**以包含第三方向的数据。

**在**阻尼**选项卡页上配置设置：**

3. 如果您在步骤2中选择了**模态**，请选择以下一个或多个选项用于定义阻尼：- 显示**直接模态**选项卡页以指定特定模态范围的临界阻尼分数，![](../graphics/usi_eqn00431.gif)。执行以下操作：1. 切换打开**使用直接阻尼数据**。2. 在数据表中输入以下内容：- **起始模态**：范围起始模态号。- **结束模态**：范围结束模态号。- **临界阻尼分数**：临界阻尼分数，![](../graphics/usi_eqn00431.gif)。- 显示**复合模态**选项卡页以使用在前述频率步骤中计算的阻尼系数选择复合模态阻尼。（频率步骤中执行的阻尼计算使用材料定义中提供的阻尼数据。）执行以下操作：1. 切换打开**使用复合阻尼数据**。2. 在数据表中输入以下内容：- **起始模态**：范围起始模态号。- **结束模态**：范围结束模态号。- 显示**Rayleigh**选项卡页以定义Rayleigh阻尼，并执行以下操作：1. 切换打开**使用Rayleigh阻尼数据**。2. 在数据表中输入以下内容：- **起始模态**：范围起始模态号。- **结束模态**：范围结束模态号。- **Alpha**：质量比例阻尼，![](../graphics/usi_eqn00432.gif)。- **Beta**：刚度比例阻尼，![](../graphics/usi_eqn00433.gif)。有关输入数据的详细信息，请参见"输入表格数据"，第3.2.7节。

### 配置直接解稳态动态过程

在直接解稳态动态过程中，Abaqus/Standard根据模型的质量、阻尼和刚度矩阵直接计算物理自由度方面的稳态谐响应。更多信息，请参见Abaqus Analysis User's Guide的第6.3.4节"直接解稳态动态分析"。

5. 在**数据**表中输入以下数据：**下限频率** 频率范围的下限或单一频率（周期/时间）。**上限频率** 频率范围的上限（周期/时间）。如果您输入零，Abaqus/Standard假定仅需要在**下限频率**列中指定的频率处的结果。**点数** 要给出结果的频率范围内的点数。如果您切换打开了**使用特征频率细分每个频率范围**，这是要给出结果的点数，包括端点，从频率范围的下限到该范围内的第一个特征频率；从特征频率到特征频率的每个区间；以及从范围内的最高特征频率到频率范围的上限。如果您切换关闭了**使用特征频率细分每个频率范围**，这是频率范围内的总点数，包括端点。**偏置** 偏置参数。此参数仅在您请求4个或更多频率点的结果时有用。它用于将结果点偏置到区间的端点，以便在那里获得更好的分辨率。如果您切换打开了**使用特征频率细分每个频率范围**，建议使用此选项，因为每个区间的端点是响应幅度变化最快的特征频率。更多信息，请参见Abaqus Analysis User's Guide的第6.3.8节"基于模态的稳态动态分析"中的"偏置参数"。有关输入数据的详细信息，请参见"输入表格数据"，第3.2.7节。

**在**阻尼**选项卡页上配置设置：**

1. 在**编辑步骤**对话框中，显示**阻尼**选项卡页。（有关显示**编辑步骤**对话框的信息，请参见"创建步骤"，第14.9.2节，或"编辑步骤"，第14.9.3节。）
2. 指示您要如何提供阻尼值：- 选择**指定模态范围的阻尼**以为特定模态范围提供阻尼值。- 选择**指定频率范围的阻尼**以为特定频率提供阻尼值。Abaqus/Standard在指定频率之间线性插值模态的阻尼系数。如果您在**阻尼**选项卡页上省略阻尼数据，Abaqus/Standard假定阻尼值为零。更多信息，请参见Abaqus Analysis User's Guide的第6.3.8节"基于模态的稳态动态分析"中的"指定模态阻尼"。
3. 如果您在步骤2中选择了**模态**，请选择以下一个或多个选项用于定义阻尼：- 显示**直接模态**选项卡页以指定特定模态范围的临界阻尼分数，![](../graphics/usi_eqn00431.gif)。执行以下操作：1. 切换打开**使用直接阻尼数据**。2. 在数据表中输入以下内容：- **起始模态**：范围起始模态号。- **结束模态**：范围结束模态号。- **临界阻尼分数**：临界阻尼分数，![](../graphics/usi_eqn00431.gif)。- 显示**复合模态**选项卡页以使用在前述频率步骤中计算的阻尼系数选择复合模态阻尼。（频率步骤中执行的阻尼计算使用材料定义中提供的阻尼数据。）执行以下操作：1. 切换打开**使用复合阻尼数据**。2. 在数据表中输入以下内容：- **起始模态**：范围起始模态号。- **结束模态**：范围结束模态号。- 显示**Rayleigh**选项卡页以定义Rayleigh阻尼，并执行以下操作：1. 切换打开**使用Rayleigh阻尼数据**。2. 在数据表中输入以下内容：- **起始模态**：范围起始模态号。- **结束模态**：范围结束模态号。- **Alpha**：质量比例阻尼，![](../graphics/usi_eqn00432.gif)。- **Beta**：刚度比例阻尼，![](../graphics/usi_eqn00433.gif)。- 显示**结构**选项卡页以定义与内力成正比但与速度方向相反的阻尼，并执行以下操作：1. 切换打开**使用结构阻尼数据**。2. 在数据表中输入以下内容：- **起始模态**：范围起始模态号。- **结束模态**：范围结束模态号。- **阻尼常数**：阻尼因子，*s*。有关输入数据的详细信息，请参见"输入表格数据"，第3.2.7节。
4. 如果您在步骤2中选择了**频率**，请选择以下一个或多个选项用于定义阻尼：- 显示**直接模态**选项卡页以指定特定频率范围的临界阻尼分数，![](../graphics/usi_eqn00431.gif)。执行以下操作：1. 切换打开**使用直接阻尼数据**。2. 在数据表中输入以下内容：- **频率**：频率值（周期/时间）。- **临界阻尼分数**：临界阻尼分数，![](../graphics/usi_eqn00431.gif)。- 显示**Rayleigh**选项卡页以定义Rayleigh阻尼，并执行以下操作：1. 切换打开**使用Rayleigh阻尼数据**。2. 在数据表中输入以下内容：- **频率**：频率值（周期/时间）。- **Alpha**：质量比例阻尼，![](../graphics/usi_eqn00432.gif)。- **Beta**：刚度比例阻尼，![](../graphics/usi_eqn00433.gif)。- 显示**结构**选项卡页以定义与内力成正比但与速度方向相反的阻尼，并执行以下操作：1. 切换打开**使用结构阻尼数据**。2. 在数据表中输入以下内容：- **频率**：频率值（周期/时间）。- **阻尼常数**：阻尼因子，*s*。有关输入数据的详细信息，请参见"输入表格数据"，第3.2.7节。
5. 根据需要重复步骤2-4以创建多个阻尼定义。

完成步骤的设置配置后，单击**确定**关闭**编辑步骤**对话框。

### 配置基于子空间的稳态动态分析

您可以配置基于子空间的稳态动态分析，以计算系统对谐激励的稳态动态线性化响应。这种类型的过程基于投影到模态子空间上的稳态动态方程的直接解。您必须首先使用特征频率提取过程提取模态（如"配置频率过程"中所述）。更多信息，请参见Abaqus Analysis User's Guide的第6.3.9节"基于子空间的稳态动态分析"。

**创建或编辑基于子空间的稳态动态过程：**

1. 按照"创建步骤"，第14.9.2节（**过程类型：**`线性扰动；稳态动力学，子空间`）或"编辑步骤"，第14.9.3节中概述的过程显示**编辑步骤**对话框。
2. 在**基本**和**其他**选项卡页上，按照以下过程中概述的方式配置频率范围和矩阵求解器偏好等设置。

**在**基本**选项卡页上配置设置：**

1. 在**编辑步骤**对话框中，显示**基本**选项卡页。（有关显示**编辑步骤**对话框的信息，请参见"创建步骤"，第14.9.2节，或"编辑步骤"，第14.9.3节。）
2. 在**描述**字段中，输入分析步骤的简短描述。Abaqus将您输入的文本存储在输出数据库中，该文本由可视化模块显示在状态块中。
3. 选择以下选项之一：- 选择**仅计算实响应**如果您希望Abaqus/Standard忽略阻尼项。此选项可以减少计算时间。- 选择**计算复响应**如果您希望包括阻尼项并允许复系统矩阵被分解。
4. 选择**刻度**选项以指示您是要使用**对数**还是**线性**刻度来划分感兴趣的频率范围。
5. 切换打开**包括摩擦引起的阻尼效应**以包括摩擦引起的阻尼矩阵贡献。更多信息，请参见Abaqus Analysis User's Guide的第6.3.9节"基于子空间的稳态动态分析"中的"具有滑动摩擦的接触条件"。
6. 切换打开**使用特征频率细分每个频率范围**如果您希望使用系统的特征频率细分感兴趣的频率范围。更多信息，请参见Abaqus Analysis User's Guide的第6.3.9节"基于子空间的稳态动态分析"中的"选择请求输出的频率区间类型"。
7. 单击**投影**字段右侧的箭头，然后选择控制子空间投影频率的选项：- 选择**在每个频率处评估**以在您指定的每个频率上将动态方程投影到子空间。此方法计算成本最高。- 选择**常量**以仅使用所有范围和您指定的单个频率点中心频率处评估的模型属性执行一次投影。此方法成本最低。但是，仅当材料属性不强烈依赖于频率时，才应选择此方法。- 选择**在特征频率处插值**以在请求频率范围内每个提取的特征频率以及紧邻该范围外部的特征频率处执行投影。然后在每个请求的频率点处对投影的质量、刚度和阻尼矩阵进行插值。- 选择**作为属性变化的函数**以根据作为频率函数材料属性的变化来选择子空间投影到模态子空间的频率。如果您选择此选项，请执行以下操作：1. 在**最大阻尼变化**字段中，输入在执行新投影之前阻尼材料属性的最大相对变化。2. 在**最大刚度变化**字段中，输入在执行新投影之前刚度材料属性的最大相对变化。- 选择**在上下频率限制处插值**以在最后频率范围的上下限制处执行投影。此方法只能与SIM架构一起使用。
8. 在**数据**表中输入以下数据：**下限频率** 频率范围的下限或单一频率（周期/时间）。**上限频率** 频率范围的上限（周期/时间）。如果您输入零，Abaqus/Standard假定仅需要在**下限频率**列中指定的频率处的结果。**点数** 要给出结果的频率范围内的点数。如果您切换打开了**使用特征频率细分每个频率范围**，这是要给出结果的点数，包括端点，从频率范围的下限到该范围内的第一个特征频率；从特征频率到特征频率的每个区间；以及从范围内的最高特征频率到频率范围的上限。如果您切换关闭了**使用特征频率细分每个频率范围**，这是频率范围内的总点数，包括端点。**偏置** 偏置参数。此参数仅在您请求4个或更多频率点的结果时有用。它用于将结果点偏置到区间的端点，以便在那里获得更好的分辨率。如果您切换打开了**使用特征频率细分每个频率范围**，建议使用此选项，因为每个区间的端点是响应幅度变化最快的特征频率。更多信息，请参见Abaqus Analysis User's Guide的第6.3.9节"基于子空间的稳态动态分析"中的"偏置参数"。有关输入数据的详细信息，请参见"输入表格数据"，第3.2.7节。

**在**其他**选项卡页上配置设置：**

1. 在**编辑步骤**对话框中，显示**基本**选项卡页。（有关显示**编辑步骤**对话框的信息，请参见"创建步骤"，第14.9.2节，或"编辑步骤"，第14.9.3节。）
2. 选择**矩阵求解器**选项：- 选择**使用求解器默认值**以允许Abaqus/Standard决定是否需要对称或非对称矩阵存储和求解方案。- 选择**非对称**以将Abaqus/Standard限制为非对称存储和求解方案。- 选择**对称**以将Abaqus/Standard限制为对称存储和求解方案。有关矩阵存储的更多信息，请参见Abaqus Analysis User's Guide的第6.1.2节"定义分析"中的"Abaqus/Standard中的矩阵存储和求解方案"。

### 配置子结构生成过程

您可以配置子结构生成过程以控制为子结构生成的数据。

**创建或编辑子结构生成过程：**

1. 按照"创建步骤"，第14.9.2节（**过程类型：**`线性扰动；子结构生成`）或"编辑步骤"，第14.9.3节中概述的过程显示**编辑步骤**对话框。
2. 在**基本**、**选项**和**阻尼**选项卡页上，配置子结构生成选项和阻尼规范等设置。

**在**基本**选项卡页上配置设置：**

1. 在**编辑步骤**对话框中，显示**基本**选项卡页。（有关显示**编辑步骤**对话框的信息，请参见"创建步骤"，第14.9.2节，或"编辑步骤"，第14.9.3节。）
2. 在**描述**字段中，输入分析步骤的简短描述。Abaqus将您输入的文本存储在输出数据库中，该文本由可视化模块显示在状态块中。
3. 在**子结构标识符**字段中，输入1到999之间的整数，该整数在此模型中唯一标识此子结构。Abaqus在您指定数字前面添加"Z"，当它在模型数据库中存储子结构标识符时。
4. 如果需要，切换打开**评估恢复矩阵**以启用分析中的选择性恢复，并执行以下操作之一：- 选择**整个模型**以为整个模型启用选择性恢复。- 选择**区域**，然后从列表中选择节点集或单元集，以在模型中的单个集合上启用选择性恢复。

**在**选项**选项卡页上配置设置：**

1. 在**编辑步骤**对话框中，显示**选项**选项卡页。（有关显示**编辑步骤**对话框的信息，请参见"创建步骤"，第14.9.2节，或"编辑步骤"，第14.9.3节。）
2. 从**生成选项**中，执行以下任何操作：- 切换打开**计算重力载荷向量**以在子结构生成期间计算子结构的重力载荷向量。- 切换打开**计算缩减质量矩阵**以为子结构生成缩减质量矩阵。- 切换打开**计算缩减结构阻尼矩阵**以生成子结构的缩减结构阻尼矩阵。- 切换打开**计算缩减黏性阻尼矩阵**以生成子结构的缩减黏性阻尼矩阵。- 切换打开**在频率处评估频率相关属性**以在子结构生成期间评估频率相关材料属性。如果您切换打开此选项，您还可以指定频率相关属性的自定义频率。
3. 如果要为耦合声-结构子结构的生成指定保留的特征模态，请切换打开**按以下方式指定保留的特征模态**，然后选择**模态范围**或**频率范围**。
4. 如果您按模态范围指定保留的特征模态，请在**数据**表中输入以下数据以生成特征模态列表：**起始模态** 第一个模态号。**结束模态** 结束模态号。**增量** 起始和结束模态号之间的模态增量。有关输入数据的详细信息，请参见"输入表格数据"，第3.2.7节。
5. 如果您按频率范围指定保留的特征模态，请在**数据**表中输入以下数据：**下限频率** 频率范围的下限或单一频率（周期/时间）。**上限频率** 频率范围的上限（周期/时间）。有关输入数据的详细信息，请参见"输入表格数据"，第3.2.7节。

**在**阻尼**选项卡页上配置设置：**

1. 在**编辑步骤**对话框中，显示**阻尼**选项卡页。（有关显示**编辑步骤**对话框的信息，请参见"创建步骤"，第14.9.2节，或"编辑步骤"，第14.9.3节。）
2. 从**全局阻尼比**选项中，指定此步骤中全局阻尼将应用于的**场**：- 选择**无**以排除全局阻尼的影响。- 选择**全部**以将全局阻尼应用于模型中的位移、旋转和声学场。- 选择**声学**以仅将全局阻尼应用于模型中的声学场。- 选择**机械**以仅将全局阻尼应用于模型中的位移和旋转场。
3. 如果您选择了**场**设置**无**以外的任何设置，请执行以下任何操作：- 对于**Alpha**选项，指定第一个Rayleigh阻尼比的值。- 对于**Beta**选项，指定第二个Rayleigh阻尼比的值。- 对于**结构**选项，指定结构阻尼比的值。
4. 从**黏性阻尼**选项中，选择以下之一：- 选择**无**以在使用级别完全排除黏性阻尼的影响。- 选择**单元**以仅激活子结构的生成缩合黏性阻尼矩阵。- 选择**因数**以仅激活Rayleigh黏性阻尼。- 选择**组合**以激活组合生成和Rayleigh黏性阻尼矩阵。
5. 从**结构阻尼**选项中，选择以下之一：- 选择**无**以排除结构阻尼矩阵。- 选择**单元**以仅激活子结构的生成缩合结构阻尼矩阵。- 选择**因数**以仅激活刚度比例结构阻尼矩阵。- 选择**组合**以激活组合生成和刚度比例。
6. 完成子结构生成步骤的设置配置后，单击**确定**关闭**编辑步骤**对话框。

1. 在**编辑步骤**对话框中，显示**基本**选项卡页。
2. 在**描述**字段中，输入分析步骤的简短描述。Abaqus将您输入的文本存储在输出数据库中，该文本由可视化模块显示在状态块中。
3. 在**数据**表中输入以下数据：**下限频率** 频率范围的下限或单一频率（周期/时间）。**上限频率** 频率范围的上限（周期/时间）。如果您输入零，Abaqus/Standard假定仅需要在**下限频率**列中指定的频率处的结果。**点数** 要给出结果的频率范围内的点数。最小值为2。如果该值被省略，则假定默认值为20点。有关输入数据的详细信息，请参见"输入表格数据"，第3.2.7节。

**在**其他**选项卡页上配置设置：**

1. 在**编辑步骤**对话框中，显示**其他**选项卡页。（有关显示**编辑步骤**对话框的信息，请参见"创建步骤"，第14.9.2节，或"编辑步骤"，第14.9.3节。）
2. 选择**矩阵存储**选项：- 选择**使用求解器默认值**以允许Abaqus/Standard决定是否需要对称或非对称矩阵存储和求解方案。- 选择**非对称**以将Abaqus/Standard限制为非对称存储和求解方案。- 选择**对称**以将Abaqus/Standard限制为对称存储和求解方案。有关矩阵存储的更多信息，请参见Abaqus Analysis User's Guide的第6.1.2节"定义分析"中的"Abaqus/Standard中的矩阵存储和求解方案"。

完成步骤的设置配置后，单击**确定**关闭**编辑步骤**对话框。
