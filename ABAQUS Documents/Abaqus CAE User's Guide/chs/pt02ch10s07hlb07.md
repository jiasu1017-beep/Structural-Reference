# 10.7.8 导入零件或草图时Abaqus/CAE识别的IGES实体





在导入过程中，Abaqus/CAE将IGES文件中存储的实体转换为Abaqus/CAE识别的内部表示。IGES文件可能包含Abaqus/CAE不识别的实体；但是，这些实体在转换过程中会被忽略。

[表10-2](pt02ch10s07hlb08.md#usi-dbs-import-iges)列出了Abaqus/CAE在导入零件或草图时识别的IGES实体。
