### 分布载荷

分布载荷如["分布载荷，" 第 34.4.3 节](pt07ch34s04aus122.md)中所述进行指定。

**载荷 ID (*DLOAD)：**  CENTRIF(S)**Abaqus/CAE 载荷/相互作用：**  不支持**单位：**  [T2](../popups/usb-int-iconventions-unitsym.md)**描述：**  离心载荷（大小输入为 ![](../graphics/usb_eqn00349.gif)，其中 ![](../graphics/usb_eqn00348.gif) 是角速度）。

**载荷 ID (*DLOAD)：**  GRAV**Abaqus/CAE 载荷/相互作用：**  不支持**单位：**  [LT2](../popups/usb-int-iconventions-unitsym.md)**描述：**  指定方向的重力加载。

**载荷 ID (*DLOAD)：**  ROTA(S)**Abaqus/CAE 载荷/相互作用：**  不支持**单位：**  [T2](../popups/usb-int-iconventions-unitsym.md)**描述：**  旋转加速度载荷（大小输入为 ![](../graphics/usb_eqn00156.gif)，其中 ![](../graphics/usb_eqn00156.gif) 是旋转加速度）。
