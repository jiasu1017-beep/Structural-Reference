# 64.5.3 拖动节点





您可以在网格化零件、装配中的网格化零件实例或孤兒网格上拖动节点。拖动是重新定位节点和提高单元质量的快速方法。当您激活节点拖动功能时，Abaqus/CAE会自动激活求解器单元质量检查，以黄色高亮显示有警告的单元，以洋红色高亮显示有错误的单元。您还可以在Abaqus/CAE自动执行的测试中包含形状指标或尺寸指标测试。拖动节点到新位置时会更新高亮显示；因此，您可以立即看到是否正在提高单元质量。通常，您不能使用网格编辑工具集修改从属零件实例的网格；但是，您可以拖动从属零件实例的节点。Abaqus/CAE修改原始网格的节点，您的修改会显示在该零件的所有从属实例上。

拖动节点不会改变它们的关联。拖动前与几何特征关联的节点在拖动后将保持与该特征的关联；未关联的节点在您将它们拖动到几何体上后仍将保持未关联状态。

**拖动节点的步骤：**

1. 进入"网格"模块，并执行以下操作之一： - 从上下文栏的**对象**字段中选择**装配**。 - 从上下文栏的**对象**字段中选择**零件**，然后从列表中选择一个零件。
2. 从主菜单栏选择****网格**![](../graphics/images/arrow.gif)**编辑****。Abaqus/CAE显示**编辑网格**对话框。 **提示：**您也可以使用位于"网格"模块工具箱底部的![](../graphics/ico_meshEdit.png)工具显示**编辑网格**对话框。
3. 在对话框中执行以下操作： 1. 在**类别**字段中选择**节点**。 2. 从**方法**列表中选择**拖动**。Abaqus/CAE显示现有节点。
4. 如果需要，从提示区域点击**单元失败标准**以在Abaqus/CAE拖动节点时自动执行的测试中包含形状指标或尺寸指标。
5. 从提示区域切换**投影到几何体**以控制外部节点的拖动行为。 - 如果打开**投影到几何体**，Abaqus/CAE会将拖动的外部节点约束到其关联的几何体。只要节点保持在底层几何面内，您就可以在网格面上拖动节点。类似地，您只能沿创建边的折线拖动边缘节点，并且不能拖动与顶点关联的节点。在孤兒网格区域中，您可以拖动任何节点；但您不能将节点拖动到网格的外部边缘之外。 - 如果关闭**投影到几何体**，Abaqus/CAE会将拖动的节点约束到平行于计算机屏幕并穿过节点原始位置的平面。因此，您可以在拖动节点之前使用视图操作工具更改可用的拖动位置。如果拖动内部节点，这也是行为，因为它们不受几何特征的约束。
6. 将光标放在要移动的节点上，然后按住鼠标按钮1并将节点拖动到新位置。当您释放鼠标按钮1时，Abaqus/CAE会在消息区域显示节点编号和新坐标。
7. 根据需要重复上述步骤以拖动其他节点。
