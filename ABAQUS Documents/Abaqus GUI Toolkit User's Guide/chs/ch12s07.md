# 12.7 Abaqus/Standard 三维实例：搭接接头剪切





该搭接接头剪切模拟说明了在 Abaqus/Standard 中使用通用接触。

模型由通过钛铆钉连接的两个重叠铝板组成。底板左端固定，力施加到顶板右端以剪切接头。图 12-27 显示了组件的基本排列。由于对称性，只对接头的一半进行建模以降低计算成本。假定存在摩擦接触。

**图 12-27** 搭接接头分析。

![](../graphics/gsa-lap-assy.png)

### 12.7.1 网格设计

在设计网格之前选择单元类型。用于板的网格应由 C3D8I 单元组成；铆钉应使用 C3D8R 和 C3D6 单元进行网格划分（如图 12-28 所示的代表性网格）。

**图 12-28** 网格。

### 12.7.2 预处理——创建模型

以下步骤假定您可以访问此实例的完整输入文件。该输入文件 `lap_joint.inp` 在 ["搭接接头剪切，" 第 A.14 节](ap01s14.md) 中提供。获取和运行脚本的说明在 [附录 A，"示例文件"](ap01.md) 中给出。如果您希望使用 Abaqus/CAE 创建整个模型，请参阅 ["Abaqus/Standard 三维实例：搭接接头剪切，" Getting Started with Abaqus: Interactive Edition 第 12.8 节](../gsa/gsa-link.md#gsa-cnt-exalapjoint)。

### 12.7.3 检查输入文件——模型数据

我们首先回顾模型定义，包括节点和单元定义，以及截面和材料属性。

**模型描述**

输入文件以 [*HEADING](../key/key-link.md#usb-kws-mheading) 选项中模拟和模型的相关描述开始。

```
*HEADING
Shearing of a lap joint
SI units (N, kg, mm, s)
```

**节点坐标和单元连接**

检查预处理器是否为板和铆钉使用了正确的单元类型。为单元提供有意义的单元集名称，如 `PLATE` 和 `RIVET`。该模型中的 [*ELEMENT](../key/key-link.md#usb-kws-melement) 选项如下：

```
*ELEMENT, TYPE=C3D8I, ELSET=PLATE
...
*ELEMENT, TYPE=C3D8R, ELSET=RIVET
...
*ELEMENT, TYPE=C3D6,  ELSET=RIVET
...

```

模型定义还指定了节点集的创建，以便可以轻松约束和移动模型的各个部分。这些集合位于以下位置：底板左下角（集合 `CORNER`）、底板左面（集合 `FIX`）、顶板右面（集合 `PULL`）和对称平面（集合 `SYMM`）。

```
*NSET, NSET=CORNER
...
*NSET, NSET=FIX
...
*NSET, NSET=PULL
...
*NSET, NSET=SYMM
...

```
第一个集合将用于防止 3 方向的刚体运动；接下来两个将用于固定一板的端部并拉动另一板的端部；最后一个将用于施加对称条件。

### 12.7.4 接触定义**

这里讨论模型的接触定义。

**定义接触**

接触将用于强制板和铆钉之间的相互作用。所有部件之间的摩擦系数假定为 0.05。

此问题可以使用接触对或通用接触算法。我们将在此问题中使用通用接触，以展示接触定义的简单性。

使用 [*SURFACE INTERACTION](../key/key-link.md#usb-kws-hsurfaceinteraction) 选项定义接触属性；指定摩擦系数 0.05。

```
*SURFACE INTERACTION, NAME=FRIC
*FRICTION
 0.05,
```

使用 [*CONTACT](../key/key-link.md#usb-kws-hcontact) 选项定义通用接触相互作用。使用 [*CONTACT INCLUSIONS](../key/key-link.md#usb-kws-hcontactinclusions) 选项上的 ALL EXTERIOR 参数指定对 Abaqus/Standard 自动定义的无名全包容表面的自接触。[*CONTACT PROPERTY ASSIGNMENT](../key/key-link.md#usb-kws-hcontpropassign) 选项用于将名为 `FRIC` 的接触属性分配给通用接触相互作用。

```
*CONTACT
*CONTACT INCLUSIONS, ALL EXTERIOR
*CONTACT PROPERTY ASSIGNMENT
 , , FRIC
```

### 12.7.5 检查输入文件——历史数据

**步骤定义和边界条件**

创建一个静态一般步骤，并包括几何非线性的影响。将初始时间增量设置为 `0.05`，总时间设置为 `1.0`。接受默认的输出请求。

组件的一端固定，另一端沿板长（1 方向）拉动。此外，固定单个节点在垂直（3-）方向以防止刚体运动，并对称平面上的节点在垂直于平面的方向（2 方向）固定。表 12-1 总结了边界条件。

**表 12-1** 边界条件汇总。
| 几何集 | 边界条件 |
| --- | --- |
| fix | U1 = 0.0 |
| pull | U1 = 2.5 |
| symm | U2 = 0.0 |
| corner | U3 = 0.0 |

模型所需的完整步骤定义如下：

```
*STEP, NLGEOM=YES
*STATIC
 0.05, 1.
*BOUNDARY
 FIX,    1, 1
 PULL,   1, 1, 2.5
 SYMM,   2, 2
 CORNER, 3, 3
*OUTPUT, FIELD, VARIABLE=PRESELECT
*OUTPUT, HISTORY, VARIABLE=PRESELECT
*END STEP
```

将输入保存到文件 `lap_joint.inp`（参见 ["搭接接头剪切，" 第 A.14 节](ap01s14.md)）。使用以下命令运行模拟：

**接触压力**

现在我们将绘制搭接接头中的接触压力。

由于在整个模型显示时很难看到接触压力，请使用**显示组**工具栏仅在视口中显示顶板。

创建路径图以检查顶板螺栓孔周围接触压力的变化。

**创建路径图：**

1. 在结果树中，双击**路径**。在**创建路径**对话框中，选择**边缘列表**作为类型，然后单击**继续**。
2. 在**编辑边缘列表路径**对话框中，选择对应于顶板的实例，然后单击**添加之后**。
3. 在提示区域中，选择**按最短距离**作为选择方法。
4. 在视口中，选择螺栓孔左端的边缘作为路径的起始边缘，将螺栓孔右端的节点选择为路径的终点节点，如图 12-33 所示。**图 12-33** 路径定义。![](../graphics/gsa-lap-path1-nls.png)
5. 在提示区域中单击**完成**以指示您已完成路径的选择。单击**确定**保存路径定义并关闭**编辑边缘列表路径**对话框。
6. 在结果树中，双击 **XYData**。在**创建 XY 数据**对话框中选择**路径**，然后单击**继续**。
7. 在**XY 数据来源路径**对话框的 **Y 值**框架中，单击**步骤/帧**。在**步骤/帧**对话框中，选择步骤的最后一帧。单击**确定**关闭**步骤/帧**对话框。
8. 确保场输出变量设置为 **CPRESS**，然后单击**绘图**查看路径图。单击**另存为**保存绘图。路径图如图 12-34 所示。**图 12-34** 顶板螺栓孔周围的 CPRESS 分布。![](../graphics/gsa-lap-path2-nls.png)
