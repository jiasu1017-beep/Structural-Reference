# *EXTREME ELEMENT VALUE









### *EXTREME ELEMENT VALUE定义要监控的单元变量。

此选项用于定义要监控并与用户指定值进行比较的单元变量。它必须与[*EXTREME VALUE](ch05abk37.md)选项结合使用。

**产品：**Abaqus/Explicit

**类型：**历史数据

**级别：**步骤

##### **参考：**

- ["显式动态分析，" Abaqus分析用户指南第6.3.3节](../usb/usb-link.md#usb-anl-aexpdynamic)
- [*EXTREME VALUE](ch05abk37.md)

### **必需参数：**

ELSET

将此参数设置为要监控变量的单元集名称。

### **必需的互斥参数：**

ABS

如果用户指定的值是变量绝对值的上界，则包含此参数。在每个增量中，Abaqus/Explicit将检查变量的绝对值是否已超过指定值。

MAX

如果用户指定的值是变量的上界，则包含此参数。在每个增量中，Abaqus/Explicit将检查变量是否已超过指定值。

MIN

如果用户指定的值是变量的下界，则包含此参数。在每个增量中，Abaqus/Explicit将检查变量是否已降至指定值以下。

### **可选参数：**

OUTPUT

如果首次超过用户指定边界时，要将请求的场类型输出写入输出数据库和额外的重新启动状态，则设置 OUTPUT=YES（默认）。输出将在发生这种情况的增量之后的增量中写入。

设置 OUTPUT=NO 以阻止任何输出被写入。

**第一行（可选，且仅在监控壳或梁单元的变量时相关）：**

**第二行：**

根据需要重复第二数据行，以定义要监控的其他变量及其最大值或最小值。
