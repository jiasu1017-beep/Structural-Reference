# 1.5.6 板弯曲片测试





**产品：**Abaqus/Standard  

### 测试的单元

S3    S3R    S4    S4R    S4R5    S8R    S8R5    S9R5    STRI3    STRI65  

### 问题描述

![](../graphics/verbendpatch-model.png)

**模型：**

厚度，*t* = 0.001。

**材料：**

线弹性，杨氏模量 = 1.0×10^6，泊松比 = 0.25。

**边界条件：**

（施加于所有外节点）

![](../graphics/ver_eqn00426.gif) = 0，![](../graphics/ver_eqn00145.gif) ×10^3(![](../graphics/ver_eqn00427.gif))/2，![](../graphics/ver_eqn00428.gif) ×10^3(![](../graphics/ver_eqn00429.gif)/2)，![](../graphics/ver_eqn00430.gif) ×10^3(![](../graphics/ver_eqn00431.gif)/2)。

### 参考解

应力：![](../graphics/ver_eqn00432.gif) = 0.6667，![](../graphics/ver_eqn00433.gif) = 0.20。

### 结果与讨论

除S8R外的所有单元都产生精确解。如果单元形状为菱形，S8R将通过片测试，但对于一般四边形，它们不能通过测试。
