# 1.3.2 矩形盒的超塑成形






**产品：** Abaqus/Standard   

在这个示例中，我们考虑矩形盒的超塑成形。该示例说明了使用刚性单元创建平滑三维刚性表面的方法。

超塑性金属表现出高延展性和非常低的变形抗力，因此适用于需要非常大变形的成形工艺。与传统成形方法相比，超塑成形具有许多优点。成形通常一步完成而不是分几步，中间退火步骤通常不需要。该工艺能够生产相对复杂、深腔且厚度相当均匀的零件。此外，模具成本更低，因为通常只需要一个模具。与该方法相关的缺点包括需要对温度和变形率进行严格控制。非常长的成形时间使该方法对于大批量零件生产不切实际。

超塑成形工艺通常包括将板料夹紧在模具上，模具表面形成所需形状的空腔。然后将气体压力施加到板料的相对表面，迫使其形成模具形状。

### 刚性表面

刚性面表面可以从任意网格的三维刚性单元（三角形R3D3或四边形R3D4单元）创建。参见["解析刚性表面定义，" Abaqus分析用户指南第2.3.4节](../usb/usb-link.md#usb-int-arigidsurf)，了解主表面平滑的讨论。Abaqus自动平滑表面面之间任何不连续的法向过渡。

### 解相关幅值

超塑成形零件的主要困难之一是加工参数的控制。材料经历的温度和应变率必须保持在一定范围内以维持超塑性。前者相对容易实现。后者则更困难，因为零件中应变率的分布是未知的。制造工艺必须设计得尽可能快，但不能超过任何材料点的最大允许应变率。为此，Abaqus有一个功能，允许通过解相关幅值和目标最大蠕变应变率来控制加载（通常是气体压力）。在加载选项中，用户指定一个参考值。幅值定义需要一个初始值、最小值和最大负载乘数。在准静态过程期间，Abaqus将监控最大蠕变应变率并将其与目标值进行比较。基于此比较调整负载幅值。该控制算法简单且相对粗糙。目的不是精确跟随目标值，而是获得实际的加载时间表。

### 几何和模型

这里处理的示例对应于矩形盒的超塑成形，其最终尺寸为长1524 mm（60 in）×宽1016 mm（40 in）×深508 mm（20 in），边缘有50.8 mm（2 in）的凸缘。所有倒角半径为101.6 mm（4 in）。盒子通过均匀流体压力成形。

使用704个M3D4型膜单元对坯料的四分之一进行建模。这些是全积分的双线性膜单元。坯料的初始尺寸为1625.6 mm（64 in）×1117.6 mm（44 in），厚度为3.175 mm（0.125 in）。坯料在其所有边缘被夹紧。膜单元模型的平坦初始构型在法向方向上完全是奇异的，除非在双轴拉伸中受到应力。通过初始应力条件施加6.89 kPa（1 lb/in²）的小双轴初始应力来防止这种困难。

凹模具建模为刚性体，并用刚性R3D3单元进行网格划分。刚性表面可以通过将用于建模模具的231个R3D3单元中面向接触方向的面分组在一起来定义。有关刚性单元网格的说明，请参见[图1.3.2-1](ch01s03aex33.md#sxmsuperbox-diesurf)。

为避免在分析过程中点"脱落"刚性表面，建模了超过四分之一的模具，如图1.3.2-2所示。始终建议将刚性表面延伸得足够远，以便接触节点不会从主表面上滑落。

默认情况下，Abaqus在每个节点点生成指向刚性表面的唯一法向，基于共享每个节点的单元的法向平均值。然而，有时应该直接指定表面的法向。这在["节点定义，" Abaqus分析用户指南第2.1.1节](../usb/usb-link.md#usb-int-inode)中讨论。在本例中，盒子周围的凸缘必须是平的，以确保原始平坯料与模具之间的相容性。因此，在构成凸缘内边缘的10个节点处指定了外法向（0, 1, 0）。这是通过在节点坐标后输入方向余弦来完成的。这10个顶点的标签为9043、9046、9049、9052、9089、9090、9091、9121、9124和9127；它们的定义可以在[superplasticbox_node.inp](../eif/superplasticbox_node.inp)中找到。

### 加载和控制

我们进行两次分析以比较恒压加载和自动调整压力以达到0.02/sec最大应变率的压力时间表。在恒定载荷情况下，预应力坯料受到快速施加的68.8 kPa（10 lb/in²）外部压力，然后在3000秒内保持恒定，直到盒子成形。在第二种情况下，预应力坯料受到快速施加的1.38 kPa（0.2 lb/in²）外部压力。然后由Abaqus选择压力时间表。

压力的初始施加假定发生得非常快，以至于它涉及纯弹性响应。这是通过使用静态过程实现的。蠕变响应在第二步中使用准静态过程发展。

在准静态步骤中，精度容差控制时间增量，从而控制瞬态蠕变解的准确性。Abaqus比较增量开始和结束时的等效蠕变应变率。差值应小于此容差除以时间增量。否则，将使用较小的时间增量重新尝试该增量。设置此精度容差的通常准则是确定可接受的应力误差，然后通过除以弹性模量将其转换为应变误差。对于这个问题，我们假定需要中等精度，并选择此容差为0.5%。一般来说，较大的容差值允许Abaqus使用较大的时间增量，导致分析精度较低但成本较低。

在自动调度分析中，压力指允许最大压力为1.38 MPa（200 lb/in²）和最小压力为0.138 kPa（0.02 lb/in²）的幅值。目标蠕变应变率是使用蠕变应变率控制输入的常量。对于使用蠕变应变率控制压力幅值的情况，考虑了节点-表面（默认）和表面-表面接触公式。在节点-表面接触公式中，坯料厚度被忽略，坯料被定位为使用其中面进行接触计算作为近似。然而，表面-表面接触公式明确考虑了坯料厚度，就像实际情况一样。

### 结果和讨论

[图1.3.2-3](ch01s03aex33.md#sxmsuperbox-def-34s)到[图1.3.2-5](ch01s03aex33.md#sxmsuperbox-def-77)显示了自动控制成形过程中变形的序列。恒定载荷过程中的变形阶段非常相似。然而，自动加载所需的时间要短得多——在83.3秒后达到最大允许压力。变形的初始阶段对应于坯料的膨胀，因为除了盒子边缘外没有接触。然后接触发生在盒子的底部，最终底部角落被填充。尽管底部角落有一些局部变薄，应变约为100%，但这些应变比中侧壁上看到的80%应变大不了多少。

[图1.3.2-6](ch01s03aex33.md#sxmsuperbox-strain-surf)和[图1.3.2-7](ch01s03aex33.md#sxmsuperbox-strain)分别显示了使用表面-表面和节点-表面接触公式在过程结束时等效塑性应变。结果的差异主要是由于处理坯料厚度的方式不同。因此，来自表面-表面接触公式的结果更可靠，因为考虑了坯料厚度。

[图1.3.2-8](ch01s03aex33.md#sxmsuperbox-rathist)显示了两个接触公式的最大蠕变应变率与目标蠕变应变率之比随时间的演变。最初施加的载荷在分析开始时产生较低的最大蠕变应变率。最后，最大蠕变应变率随着模具空腔的填充而大幅下降。虽然曲线看起来非常锯齿，但它表明最大峰值应变率始终相对接近目标值。这在实践中是可以接受的。[图1.3.2-9](ch01s03aex33.md#sxmsuperbox-presshist)显示了Abaqus为此问题计算的压力时间表。在大部分时间里，当板料不接触模具底部时，压力较低。一旦模具开始约束变形，压力就可以大大增加而不会产生高应变率。同样，模拟结束时压力时间表的差异主要是由于处理坯料厚度的差异。
