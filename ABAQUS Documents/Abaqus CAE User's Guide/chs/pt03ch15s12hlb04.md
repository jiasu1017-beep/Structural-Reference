# 15.12.4 创建接触初始化





默认情况下，Abaqus会调整通用接触域中略微过度闭合的表面位置，如「Abaqus/Standard中控制初始接触状态」，Abaqus Analysis User's Guide第36.2.4节中所讨论。接触初始化用于修改接触表面调整的默认行为。每个接触初始化定义包含一组调整规则；通用接触定义指定应用每个初始化定义的表面（请参见「为通用接触指定和修改接触初始化分配」，第15.13.3节）。

接触初始化旨在纠正表面之间的小间隙或过度闭合。指定大的初始化调整可能导致网格变形并增加分析的计算成本。

**创建接触初始化的步骤：**

1. 从主菜单栏选择 ****Interaction**![](../graphics/images/arrow.gif)**Contact Initialization**![](../graphics/images/arrow.gif)**Create****。此时会出现「Edit Contact Intialization」对话框。
2. 为接触初始化定义输入「Name」。
3. 您可以指定以下技术来处理两个表面之间的「Initial Overclosures」：- 选择「Resolve with strain-free adjustments」来调整某些表面在分析开始时正好接触，而不在线模型中产生应变。只有位于指定距离范围内的表面部分会被调整。- 选择「Treat as interference fits」在分析第一步过程中逐渐解决表面过度闭合；此技术会在表面位移时在线模型中产生应变。只有位于指定过度闭合距离范围内的表面部分才会使用干涉配合进行调整。- 要在解决干涉配合之前建立均匀过度闭合，请切换「Specify interference distance」，并输入过度闭合距离值。位于指定距离范围内的表面部分（过度闭合和开口）会被调整到指定的过度闭合量。调整发生在分析开始时，不会在模型中产生应变；分析第一步中随后的干涉配合解决将在线模型中产生应变。- 选择「Specify clearance distance」来调整某些表面在分析开始时以指定值分隔，而不在线模型中产生应变。只有位于指定距离范围内的表面部分会被调整。
4. 指定过度闭合距离范围；通过小于指定距离的值过度闭合的节点将使用无应变调整或逐渐干涉配合进行调整。- 选择「Analysis default」让Abaqus根据每个表面上底层单元面的尺寸计算最大过度闭合调整距离。- 选择「Specify value」直接输入最大过度闭合调整距离。如果您输入的值小于表面的计算分析默认值，Abaqus将对该表面使用分析默认值。**警告：**如果两个表面（或两个表面的部分）初始过度闭合距离大于指定的调整值，则在这些严重过度闭合表面区域之间的接触将不会在分析期间被强制执行；更多信息，请参见「Abaqus/Standard中控制初始接触状态」，Abaqus Analysis User's Guide第36.2.4节。
5. 指定开口距离范围；从对立表面分离且小于指定距离的节点将使用无应变调整进行调整。- 选择「Analysis default」在初始化调整期间忽略所有开放节点。- 选择「Specify value」直接输入最大开口调整距离。
6. 如果您想将对话框中的值重置为默认值，请点击对话框底部的「Defaults」。
7. 点击「OK」保存您的接触初始化定义并关闭「Edit Contact Initialization」对话框。
