### 概述

湿胀：
- 定义多孔介质固体骨架在部分饱和流动条件下的饱和度驱动体积膨胀；
- 可用于耦合润湿液体流动和多孔介质应力的分析（参见["耦合孔隙流体扩散和应力分析，" 第6.8.1节](pt03ch06s08at26.md)）；以及
- 可以是各向同性或各向异性的。

### 湿胀模型

湿胀模型假定多孔介质固体骨架的体积膨胀是部分饱和流动条件下润湿液体饱和度的函数。当孔隙液体压力 ![](../graphics/usb_eqn01163.gif) 为负时，多孔介质处于部分饱和状态（参见["多孔介质的有效应力原理，" 第2.8.1节Abaqus理论指南](../stm/stm-link.md#stm-anl-poreffstress)）。

### 定义初始饱和度值

可以将初始饱和度值定义为初始条件。如果未给出初始饱和度值，默认为完全饱和条件（饱和度为1.0）。对于部分饱和，初始饱和度和孔隙流体压力必须一致，即孔隙流体压力必须位于初始饱和度值的吸收和解吸值范围内（参见["渗透率，" 第26.6.2节](pt05ch26s06abm64.md)）。如果不是这种情况，Abaqus/Standard将调整饱和度值以满足此要求。

| **输入文件用法：** | ``` [*INITIAL CONDITIONS](../key/key-link.md#usb-kws-minitialcond), TYPE=SATURATION ``` |
