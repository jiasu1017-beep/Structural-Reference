# 9.2 GUI工具集示例

`AFXToolsetGui`基类提供各种工具集基础设施支持功能。例如，`AFXToolsetGui`基类跟踪工具集中的菜单以及工具栏和工具箱按钮，以便当用户切换模块时可以自动换入和换出。要创建你自己的工具集GUI，首先从`AFXToolsetGui`类派生。或者，如果另一个模块GUI类提供了你所需的大部分功能，你可以从该类派生然后进行修改。有关更多信息，请参阅[第10章，"自定义现有模块或工具集"](pt05ch10.md)。

以下示例显示如何通过从`AFXToolsetGui`派生来创建新工具集类：

```
from abaqusGui import *

class MyToolsetGui(AFXToolsetGui):

    #~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
    def __init__(self):

        # 构造基类
        #
        AFXToolsetGui.__init__(self, toolsetName)
        ...
```

在新类的构造函数中调用基类的构造函数。`AFXToolsetGui`类接受以下参数：

**toolsetName**

一个指定工具集名称的String。工具集名称提供工具集的标识符。
