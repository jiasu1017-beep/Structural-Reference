# A.5 MPI





必须安装 Message Passing Interface (MPI) 组件才能使用 Abaqus/Standard 中基于 MPI 的并行执行、使用 Abaqus/Explicit 中的域级并行化，或运行 Abaqus/CFD 中的任何作业（无论 CPU 数量如何）。如果您的 Abaqus 用户将运行这些类型的模拟，您必须预先安装所需的 MPI 组件，或允许 Abaqus 安装程序为您安装它们。MPI 是一种用于促进处理器间通信的中间件方案。基于 MPI 的并行化可用于计算机集群和共享内存架构平台。必要的 MPI 软件随 Abaqus 一起提供，并在大多数平台的 Abaqus 产品安装过程中安装；请参阅["Visual C++ 和 MPI 库，"第 2.4.1 节](ch02s04.md#sgb-chp-install-prod-libraries)。
