# *FLUID INFLATOR





### *FLUID INFLATOR定义流体充气机。

此选项用于定义流体充气机以建模气囊展开。

**产品：**Abaqus/Explicit   

**类型：**模型数据  

**级别：**部件，部件实例，装配  

##### **参考：**

- ["充气机定义，" Abaqus Analysis User's Guide第11.5.4节](../usb/usb-link.md#usb-anl-afluidinflator)
- [*FLUID INFLATOR PROPERTY](ch06abk29.md)

### **必需参数：**

NAME

将此参数设置为用于引用流体充气机的标签。

PROPERTY

将此参数设置为定义流体充气机属性的[*FLUID INFLATOR PROPERTY](ch06abk29.md)选项的名称。
