# 12.8.6 指定用户变量数量







您可以为在用户子程序[`UVARM`](../sub/sub-link.md#sub-xsl-uvarm)中定义的用户定义输出变量在每个材料计算点分配空间。

**指定用户变量数量：**

1. 从**编辑材料**对话框的菜单栏中，选择**常规**![](../graphics/images/arrow.gif)**用户输出变量**。（有关显示**编辑材料**对话框的信息，请参见"创建或编辑材料"，第12.7.1节。）
2. 单击**每个材料点的用户定义变量数量**字段右侧的箭头，指定您要为每个材料计算点分配空间的用户定义变量数量。可以使用任意数量的用户定义输出变量。
3. 单击**确定**关闭**编辑材料**对话框。或者，您可以从**编辑材料**对话框的菜单中选择其他材料行为进行定义（有关更多信息，请参见"浏览和修改材料行为"，第12.7.2节）。
