# 18.1 理解Optimization模块的作用





您可以使用Optimization模块执行以下任务：

**创建优化任务**

优化任务包含您的优化定义。 您可以在Job模块中使用优化过程运行优化。 优化过程引用优化任务。

**构建设计响应**

设计响应是从优化中提取的单个标量值。 设计响应可以直接从输出数据库中提取，例如模型的体积。 或者，Optimization模块可以从输出数据库提取数据并计算设计响应，例如模型的总应变能（其柔度的度量）。

**创建目标函数**

目标函数定义优化的目标，并引用设计响应或设计响应组合的值。 例如，优化的目标函数可以是最小化模型中的总应变能（最大化其刚度）。

**创建约束**

约束定义Optimization模块在优化期间可以对模型的拓扑或形状应用的更改。 例如，优化模型的体积可以约束为原始体积的50%。 如果约束无法满足，则优化不可行。 约束也引用设计响应的值，但不能引用设计响应的组合。

**创建几何限制**

几何限制对Optimization模块对模型拓扑所做的更改施加限制。 几何限制包括材料不能被移除的冻结区域，以及制造限制（例如对空腔和倒扣的限制），这些限制会阻止优化模型从模具中移除。

**创建停止条件**

停止条件是优化已收敛到解的指示器。 例如，在指定次数的迭代之后或当迭代之间优化函数的变化小于指定值时，可以认为优化完成。
