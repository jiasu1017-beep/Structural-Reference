# A.4 编译器





FORTRAN 或 C++ 编译器是编译和链接 Abaqus 用户子程序所必需的。SIMULIA 无法保证 Abaqus 与使用 FORTRAN 或 C++ 以外的语言编写的用户子程序的兼容性。有关详细信息，请参阅["用户子程序和实用程序，"Abaqus Analysis User's Guide 第 18.1 节](../usb/usb-link.md#usbusubutil)。

使用 Abaqus 6.14 中的 **abaqus make** 实用程序创建的用户后处理程序应使用程序语言的编译器进行编译，并使用 C++ 编译器进行链接。因此，用户编写的 FORTRAN 程序需要同时使用 FORTRAN 和 C++ 编译器。使用先前版本的 Abaqus（在 Abaqus 6.14 之前）中的 **abaqus make** 实用程序编译和链接的、仅处理结果（`.fil`）文件的用户编写程序可以继续使用 Abaqus 6.14 结果文件，无需任何更改。在这些情况下不需要 C++ 编译器。
