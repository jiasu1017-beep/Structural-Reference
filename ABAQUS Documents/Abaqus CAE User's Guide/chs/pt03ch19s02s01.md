# 19.2.1 分析模型的基本步骤





定义模型后，您就可以进行分析了。 分析模型涉及以下步骤，每个步骤都可以使用主菜单栏上的**Job**菜单或**Job Manager**执行：

**创建和配置分析作业**

您通过从主菜单栏选择 ****Job**![](../graphics/images/arrow.gif)**Create**** 来创建分析作业。 Abaqus/CAE要求您为新作业命名，并将其与从模型数据库选择的模型或现有输入文件关联。 您可以选择数据库中存在的任何模型； 不限于当前模型。 作业编辑器允许您配置作业设置。

**写入输入文件**

当您提交与模型关联的作业进行分析时，Abaqus/CAE首先生成表示模型的输入文件，然后Abaqus/Standard、Abaqus/Explicit或Abaqus/CFD使用此文件的内容执行分析。 或者，您可以要求Abaqus/CAE仅生成输入文件； Abaqus/CAE以ASCII格式写入输入文件，您可以在工作目录中查看和编辑它。

**警告：**如果您在Abaqus/CAE外部使用文本编辑器编辑模型的输入文件，然后在该模型的Job模块中提交作业，您对输入文件的更改将丢失。 相反，您必须通过创建新作业并选择**Input file**作为作业**Source**来直接提交修改后的输入文件进行分析。

但是，如果您使用**Keywords Editor**修改为模型生成的关键字，这些修改会保留在模型中并应用于与该模型关联的任何作业。

**提交作业进行分析**

您通过从主菜单栏选择 ****Job**![](../graphics/images/arrow.gif)**Submit**** 来提交作业进行分析。 随着分析的进行，Abaqus/CAE在作业监控对话框中显示来自状态、数据、日志和消息文件的信息。 作业完成后，您可以通过从主菜单栏选择 ****Job**![](../graphics/images/arrow.gif)**Results**** 在Visualization模块中显示输出数据库中的结果。
