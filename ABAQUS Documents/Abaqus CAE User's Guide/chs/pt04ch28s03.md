# 28.3 在欧拉-拉格朗日模型中定义接触






耦合欧拉-拉格朗日分析中的接触必须使用通用接触来定义。通用接触定义支持拉格朗日零件表面与欧拉材料实例表面之间的相互作用；模型中不同拉格朗日零件之间的接触也被启用。

在 Abaqus/CAE 中，您可以使用全局**所有*与自身**接触域来强制模型中所有拉格朗日零件与所有欧拉材料实例之间的接触。或者，您可以包括或排除特定拉格朗日表面与特定欧拉材料实例之间的接触。欧拉材料实例出现在**包含对**和**排除对**对话框的表面列表中，如图 [图 28-3](pt04ch28s03.md#usi-eulerian-surf-editor-nls) 所示。您还可以在**单独接触属性分配**对话框中为特定拉格朗日表面和特定欧拉材料实例分配唯一接触属性。

**图 28-3****编辑排除对**对话框中的欧拉材料实例。

![](../graphics/usi-eulerian-surf-editor-nls.png)

不同欧拉材料实例之间不能定义真实接触。强制执行基本的接触条件，因为材料实例不能相互穿透。但是，材料实例一旦接触就不会分离，这阻止了滑动或回弹行为的建模。欧拉材料实例不能使用接触输出。有关欧拉分析接触公式的详细讨论，请参阅["相互作用"中的"欧拉分析，" Abaqus 分析用户指南第 14.1.1 节](../usb/usb-link.md#usb-anl-aeuleriananal-int)。如果两种材料之间的接触条件对您的分析很重要，至少有一种材料应建模为拉格朗日零件。
