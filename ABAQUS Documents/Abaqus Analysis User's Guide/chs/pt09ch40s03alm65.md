# 40.3.1 管对管接触单元





**产品：** Abaqus/Standard  

##### **参考文献**

- ["管对管接触单元库，" 第40.3.2节](pt09ch40s03ael50.md)
- [*INTERFACE](../key/key-link.md#usb-kws-minterface)
- [*SLIDE LINE](../key/key-link.md#usb-kws-mslideline)

### 概述

管对管单元：
- 在其中一个管位于另一个内部或两个管或棒彼此相邻的两个管道或管之间建模有限滑动相互作用；
- 是滑动线接触单元，因为它们假设两个管或管的相对运动主要沿着由其中一个管的轴定义的线（假设管或管轴的相对旋转很小）；
- 可与管、梁或桁架单元一起使用；和
- 不考虑管或管横截面的变形。

[第36章，"定义接触相互作用"](pt09ch36.md) contains a general discussion of contact modeling。

### 典型应用

管对管接触单元可用于建模两类特定的管对管接触问题：内部（管内管）接触和外部接触，其中两个管大致平行并沿其外表面相互接触。

### 选择合适的单元

将ITT21单元与二维梁、管或桁架单元一起使用。将ITT31单元与三维梁、管或桁架单元一起使用。这些单元中的每一个由单个节点定义。

### 将管对管接触单元与滑动线关联

您必须指示哪组管对管接触单元将与特定滑动线相互作用。下面讨论定义滑动线的详细信息。

| **输入文件用法：** | ``` [*SLIDE LINE](../key/key-link.md#usb-kws-mslideline), ELSET=*element_set_name* ``` |
| --- | --- |

### 定义单元的截面属性

您必须将几何截面属性与一组管对管接触单元关联。

| **输入文件用法：** | ``` [*INTERFACE](../key/key-link.md#usb-kws-minterface), ELSET=*element_set_name* ``` |
| --- | --- |

#### 当在一个管内建模另一个管的接触时定义径向间隙

您定义管之间的径向间隙。当一个管（带有管对管接触单元的那个管）位于另一个管内部时，给出正值来建模两个管之间的接触。给出的值是外部管的内半径与内部管的外半径之差。

| **输入文件用法：** | ``` [*INTERFACE](../key/key-link.md#usb-kws-minterface) *radial clearance* ``` |
| --- | --- |

#### 当建模两个管外表面之间的接触时定义径向间隙

您可以通过为径向间隙指定负值来建模外部管对管接触。值的幅度必须是两个管或棒的外半径之和。

### 接触输出变量的局部基础

ITT单元的单元输出变量在关联于滑动线的局部基础系统中给出。第一个切向量![](../graphics/usb_eqn00238.gif)由形成滑动线的节点序列定义。接触方向![](../graphics/usb_eqn00842.gif)是垂直于滑动线并指向ITT单元节点的法线。对于ITT31单元，Abaqus/Standard形成第二个切向量![](../graphics/usb_eqn00239.gif)，它既垂直于![](../graphics/usb_eqn00238.gif)又垂直于![](../graphics/usb_eqn00842.gif)。当单元移动时，局部基础系统将随滑动线的轴旋转。

### 选择哪个管（梁或桁架）将具有滑动线

对于内部管对管接触，滑动线可以放在内部管或外部管上。通常，滑动线应与外部管关联（参见[图40.3.1-1](pt09ch40s03alm65.md#eitt-slide-line)）；但是，如果内部管比外部管更 stiff，滑动线应 attached to the inner tube。

**图40.3.1-1** 内部管对管接触示例。

![](../graphics/eitt-slide-line-nls.png)

如果发生在管的外表面之间的接触，滑动线应关联到较 stiffer 的管（如果材料或管半径不同）或与网格较粗的管关联（如果相同）。

### 定义滑动线

您可以指定组成滑动线的节点，或者可以按如下所述生成它们。如果您选择直接指定节点，必须按定义连续滑动线的顺序指定它们。节点序列定义滑动线的切向量![](../graphics/usb_eqn00238.gif)。滑动线必须由线性 segments 组成。

| **输入文件用法：** | ``` [*SLIDE LINE](../key/key-link.md#usb-kws-mslideline), ELSET=*element_set_name*, TYPE=LINEAR *first node number, second node number, etc.* ``` |
| --- | --- |

#### 生成滑动线节点

或者，您可以指示应生成滑动线节点，仅指定第一个节点号、最后一个节点号和节点号之间的增量。

| **输入文件用法：** | ``` [*SLIDE LINE](../key/key-link.md#usb-kws-mslideline), GENERATE *first node number, last node number, increment between node numbers* ``` |
| --- | --- |

#### 平滑滑动线

通过平滑滑动线段之间表面切线的 discontinuities，通常可以改善收敛，从而沿滑动线提供平滑变化的切线。关于平滑滑动线的详细信息，请参见["Abaqus/Standard中的接触公式，" 第38.1.1节](pt09ch38s01aus177.md)。

### 为管对管接触单元定义非默认机械表面相互作用

默认情况下，Abaqus/Standard对管对管接触单元使用"硬"、无摩擦接触。您可以分配可选的机械表面相互作用模型。以下机械表面相互作用模型可用：
- 摩擦。详见["摩擦行为，" 第37.1.5节](pt09ch37s01aus169.md)。
- 改进的"硬"接触、软化接触和粘性阻尼。详见["接触压力-过盈关系，" 第37.1.2节](pt09ch37s01aus166.md)和["接触阻尼，" 第37.1.3节](pt09ch37s01aus167.md)。
