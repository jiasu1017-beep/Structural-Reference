# 3.2.15 查询关键字/问题数据库





**产品：**Abaqus/Standard  Abaqus/Explicit  Abaqus/CFD  

##### **参考**

- ["Abaqus执行程序：概述，" 3.1.1节](pt01ch03s01abo02.md)

### 概述

**abaqus findkeyword**工具查询包含Abaqus/Standard、Abaqus/Explicit和Abaqus/CFD示例问题、验证问题、培训研讨会使用的问题、Abaqus技术简介中的问题、基准计时问题以及教程书籍[Getting Started with Abaqus: Keywords Edition](../gsk/gsk-link.md#gsk)中的问题的关键字/问题数据库。您指定感兴趣的关键字、参数和值；此工具将列出包含这些关键字、参数和值的输入文件。您可以指定多个关键字，这会导致**findkeyword**工具列出包含所有指定关键字的输入文件。然后，您可以使用**abaqus fetch**工具获取输入文件（请参阅["获取示例输入文件，" 3.2.16节](pt01ch03s02abx16.md)）。输出按问题集分组；例如，Abaqus示例问题或Abaqus/Standard技术简介问题。

### 命令行选项

**job**

此选项用于指定输出列表的输出文件名。如果命令行中省略此选项，输出将打印到标准输出设备。

**maximum**

此选项用于限制每个集合列出的示例问题的数量。如果省略此选项，每个集合默认列出100个示例问题。

*关键字数据行*

*关键字数据行*指定用户感兴趣的关键字、参数和值。包含指定关键字、参数和值的示例问题名称将打印到标准输出设备或由**job**命令行参数指定的文件。关键字是必需的，但参数和值是可选的。如果指定关键字而没有参数或值，将列出使用该关键字（无论是否有参数和值）的所有示例问题。如果指定参数而没有值，将列出使用该参数但具有任何值的示例问题。用户指定的数据（例如数值、集合名、方向名等）的参数值将被忽略。关键字数据行的结束由空行或文件结束指示。

### 示例

以下示例说明**findkeyword**执行程序使用的不同类型的搜索条件。

#### 查询关键字和参数

要列出使用[*RESTART](../key/key-link.md#usb-kws-mrestart)选项和WRITE参数的所有示例问题，请输入以下命令和数据行：

```
abaqus findkeyword
[*RESTART](../key/key-link.md#usb-kws-mrestart),WRITE
```
要生成在同一文件中包含两个关键字行的所有示例问题列表，请将两个关键字都包含为数据行。例如，
```
abaqus findkeyword
[*RESTART](../key/key-link.md#usb-kws-mrestart),WRITE
[*NGEN](../key/key-link.md#usb-kws-mngen)
```
要列出使用带有值的关键字和参数的所有示例问题，必须在数据行上包含该值。例如，
```
abaqus findkeyword job=beam
[*BEAM SECTION](../key/key-link.md#usb-kws-mbeamsection),SECTION=ARBITRARY
```
输出被写入文件`beam.dat`。

#### 查询用户指定的参数值

用户指定的参数值（例如数值、集合名、方向名等）将被忽略。以下两个示例是等效的，因为值`MYSET`是一个元素集名称。

```
abaqus findkeyword
[*ELSET](../key/key-link.md#usb-kws-melset),ELSET=MYSET
```

```
abaqus findkeyword
[*ELSET](../key/key-link.md#usb-kws-melset),ELSET
```
