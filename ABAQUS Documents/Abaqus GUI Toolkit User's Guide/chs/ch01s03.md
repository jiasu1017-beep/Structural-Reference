# 1.3 Abaqus文档

Abaqus的文档内容丰富且完整。以下文档和出版物可通过Abaqus HTML文档和PDF格式从SIMULIA获得。有关访问HTML指南的更多信息，请参阅《Abaqus分析用户指南》中关于执行过程的讨论（《Abaqus分析用户指南》](../usb/usb-link.md#usb)）。有关打印指南的更多信息，请参阅《使用Abaqus在线文档》第5.3节["从PDF书籍打印"](../hhp/hhp-link.md#hhp-pdf-print)。

**《Abaqus分析用户指南》**

本指南包含对单元、材料模型、过程、输入规范等的完整描述。它是Abaqus/Standard、Abaqus/Explicit和Abaqus/CFD的基本指南；并提供输入文件使用和Abaqus/CAE使用信息。本指南经常引用《Abaqus分析用户指南》，因此您在完成示例时应参阅该指南。

**《Abaqus/CAE用户指南》**

本指南包括有关使用Abaqus/CAE进行模型生成、分析和结果评估与可视化的详细说明。Abaqus/Viewer用户应参阅本指南中关于可视化模块的信息。

**《使用Abaqus在线文档》**

本指南包含导航、查看和搜索Abaqus HTML和PDF文档的说明。此外，本指南还说明如何使用PDF文档制作高质量打印副本，以及如何使用所有PDF书籍中的![](../graphics/hhp-pdf-print-exa-button.gif)图标（《Abaqus脚本参考指南》和《Abaqus GUI工具包参考指南》除外）打印书籍的选定部分。

**其他Abaqus文档：**

**《Abaqus示例问题指南》**

本指南包含详细示例，旨在说明执行有意义的线性和非线性分析所需的方法和决策。许多示例使用几种不同的单元类型、网格密度和其他变体进行了演算。典型案例包括弹性塑性管道撞击刚性墙的大运动；薄壁弯头的非弹性弯曲塌陷；弹性粘塑性薄环的爆炸载荷；基础下的固结；带孔复合壳的屈曲；以及金属板的深拉。本指南通常有助于查找相关示例，并在开始处理新类问题时查看它们。

当您想使用以前未使用过的功能时，应查找使用该功能的一个或多个示例。然后，使用该示例来熟悉该功能的正确用法。要查找使用某个功能的示例，请搜索在线文档或使用**abaqus findkeyword**实用程序（有关更多信息，请参阅["查询关键词/问题数据库"，《Abaqus分析用户指南》第3.2.15节](../usb/usb-link.md#usb-int-dkeywordproc)）。

随Abaqus安装一起提供了与示例相关的所有输入文件。使用**abaqus fetch**实用程序从随版本提供的压缩档案文件中提取样本Abaqus输入文件（有关更多信息，请参阅["获取样本输入文件"，《Abaqus分析用户指南》第3.2.16节](../usb/usb-link.md#usb-int-dfetchproc)）。您可以获取任何示例文件，以便自己运行模拟并查看结果。您也可以通过《Abaqus示例问题指南》中的超链接访问输入文件。

**《Abaqus基准指南》**

本指南包含基准问题和用于评估Abaqus性能的分析；这些测试是简单几何或实际问题简化版本的多单元测试。本指南中包含了NAFEMS基准问题。

**《Abaqus验证指南》**

本指南包含基本测试用例，提供对各个程序功能（过程、输出选项、MPC等）与精确计算和其他已发布结果的验证。在学习使用新功能时，运行这些问题可能很有用。此外，提供的输入数据文件为检查单元、材料等的行为提供了良好的起点。

**《Abaqus理论指南》**

本指南包含Abaqus所有理论方面的详细、准确的讨论。它面向具有工程背景的用户编写。

**《Abaqus关键词参考指南》**

本指南包含Abaqus/Standard、Abaqus/Explicit和Abaqus/CFD中所有可用输入选项的完整描述。

**《Abaqus用户子程序参考指南》**

本指南包含可用于Abaqus分析的所有用户子程序的完整描述。它还讨论了编写用户子程序时可以使用的实用程序。

**《Abaqus术语表》**

本指南定义了Abaqus统一FEA产品套件中技术术语的含义。

**《Abaqus发布说明》**

本指南包含Abaqus产品线最新版本中可用新功能的简要描述。

**《Abaqus安装和许可指南》**

本指南描述如何安装Abaqus以及如何为特定情况配置安装。其中与用户最相关的一些信息也在《Abaqus分析用户指南》中提供。

除了上面列出的文档外，还有以下适用于Abaqus界面和自定义编程技术的指南，这些在本指南中未讨论：
- 《Abaqus脚本用户指南》
- 《Abaqus脚本参考指南》
- 《Abaqus GUI工具包用户指南》
- 《Abaqus GUI工具包参考指南》

SIMULIA还为["Abaqus产品"第1.1节》中所述的所有几何翻译器提供文档。
