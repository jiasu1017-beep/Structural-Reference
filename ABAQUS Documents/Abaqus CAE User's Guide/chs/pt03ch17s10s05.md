# 17.10.5 我可以使用边界网格做什么？





当Abaqus/CAE使用四面体单元在实体上生成自由网格时，它首先在实体区域的外部面上生成三角形边界网格，如["什么是四面体边界网格？"第17.10.4节](pt03ch17s10s04.md)中所述。

当您创建自由四面体网格时，Abaqus/CAE可以在边界面创建自由网格或映射网格；两种机制控制此三角形表面网格的网格图案：

**与三维区域关联的全局网格控制**：全局控制始终存在。当您将自由四面体网格技术分配给区域时设置这些控制。您可以使用它们在Abaqus/CAE确定映射网格适当的区域创建映射网格，或在所有边界面上创建自由网格。

**在区域选定面上指定的局部网格控制**：要分配局部控制，您选择一个或多个区域面，并指定Abaqus/CAE是否应在选择上创建自由网格或映射/结构化网格。如果您定义了局部控制，它们会覆盖全局控制。
如果您将结构化技术分配给将进行四面体网格划分的实体区域的面，这些面会着色为绿色以显示表面网格的技术分配。（有关分配网格控制的详细说明，请参见["分配网格控制，"第17.18.1节](pt03ch17s18s01.md)。）

**注意：**绿色着色仅在您为面分配网格控制时应用。过程结束时，显示全局网格控制颜色。

此外，Abaqus/CAE会高亮显示网格划分失败的边界上的任何面。这些失败通常是由于布种太粗或由于极小的边或面造成的。您可以将高亮显示的面保存在集合中，并且可以仅对该集合中的面应用更细的种子。有关在集合中对面进行布种的更多信息，请参见["我可以对面或单元进行布种吗？"第17.4.2节](pt03ch17s04s02.md)。您可以使用显示组仅显示集合中的面。更多信息，请参见["绘制显示组，"第78.2.4节](pt07ch78s02hlb04.md)。

您可以使用查询工具集查询边界网格。此外，您可以使用网格验证工具检查边界网格的质量。网格验证工具允许您检查所有边界三角形的质量，或者您可以使用选择过滤器仅检查选定面的边界三角形的质量。

如果极小的边或面阻止Abaqus/CAE生成可接受的四面体网格，您可以尝试以下操作：
- 使用几何诊断工具查找可能影响网格质量的小实体，例如短边、小面和具有小面角的面。您可以创建一个包含这些小实体的集合。更多信息，请参见["使用几何诊断工具，"第71.2.4节](pt06ch71s02hlb04.md)。
- 使用"几何编辑"工具集移除多余的边和顶点。您也可以移除一个面并缝合产生的间隙。更多信息，请参见["编辑技术概述，"第69.2节](pt06ch69s02.md)。
- 使用"虚拟拓扑"工具集忽略极小的边或面。更多信息，请参见["我可以使用虚拟拓扑工具集做什么？"第75.2节](pt06ch75s02.md)。
- 添加分割以减少狭长面或单元的长宽比。更多信息，请参见[第70章，"分割"工具集](pt06ch70.md)。
- 使用"编辑网格"工具集修改预览网格。您可以在网格模块中执行以下操作：
  - 编辑节点
  - 折叠单元边
  - 交换一对相邻三角形单元的对角线
  - 分割单元边
更多信息，请参见["我可以使用编辑网格工具集做什么？"第64.1节](pt06ch64s01.md)。

在某些情况下，由于表面网格中非常薄的三角形单元或某些薄片面无法用三角形进行网格划分，您将无法使用四面体单元对导入的实体零件进行网格划分。["使用工具组合使用四面体单元对导入的实体零件进行网格划分，"第64.3.6节](pt06ch64s03s06.md)描述了您如何可以使用"编辑网格"工具集和"网格"模块中的其他工具成功对零件进行网格划分。
