# 15.9.1 管理交互模块中的对象





交互模块提供以下管理器，可用于组织和操作与给定模型关联的对象：
- **交互管理器**允许你创建和管理交互。
- **交互属性管理器**允许你创建和管理交互属性。
- **接触控制管理器**允许你为表面到表面接触和自接触交互创建和管理接触控制。
- **接触初始化管理器**允许你为Abaqus/Standard中的通用接触交互创建和管理接触初始化规则。
- **约束管理器**允许你创建和管理约束。
- **连接器截面管理器**允许你创建和管理连接器截面。
- **连接器截面分配管理器**允许你创建和管理连接器截面分配。

例如，[图15-13](pt03ch15s09s01.md#itn-propertyman)中显示的**交互属性管理器**中会出现交互属性列表。

**图15-13****交互属性管理器**。

![](../graphics/itn-propertyman-nls.png)

管理器中的**创建**、**编辑**、**复制**、**重命名**和**删除**按钮允许你创建新对象或编辑、复制、重命名和删除现有对象。在**连接器截面分配管理器**中，你只能创建、编辑或删除连接器截面分配。你也可以使用主菜单栏中的**Interaction**、****Interaction**![](../graphics/images/arrow.gif)**Property****、****Interaction**![](../graphics/images/arrow.gif)**Contact Controls****、****Interaction**![](../graphics/images/arrow.gif)**Contact Initialization****、**Constraint**、****Connector**![](../graphics/images/arrow.gif)**Section****和****Connector**![](../graphics/images/arrow.gif)**Assignment****菜单来启动这些过程。从主菜单栏选择管理操作后，过程与点击管理器对话框内的相应按钮完全相同。

你可以使用**交互管理器**中的**复制**按钮、相应的菜单命令或模型树来复制交互。你可以将交互从任何步骤复制到任何有效步骤，但有一些限制。更多详细信息，请参见["使用管理器对话框复制步骤相关对象，"第3.4.11节](pt01ch03s04hlb03.md)。

**交互管理器**是一个步骤相关管理器，这意味着它包含有关每个交互在分析过程中历史的其他信息。[图15-14](pt03ch15s09s01.md#itn-interactionman)显示了**交互管理器**。

**图15-14****交互管理器**。

![](../graphics/itn-interactionman-nls.png)

**向左移动**、**向右移动**、**激活**和**停用**按钮允许你操作交互的逐步历史。更多信息，请参见["修改步骤相关对象的历史，"第3.4.6节](pt01ch03s04s06.md)。

你可以从管理器中抑制和恢复先前定义的交互、约束和连接器截面分配。你可以使用管理器左侧列中的图标来抑制这些属性或恢复先前被抑制的分析属性。抑制和恢复过程也可以从主菜单栏中的**Interaction**、**Constraint**和**Connector**菜单中获得。更多信息，请参见["抑制和恢复对象，"第3.4.3节](pt01ch03s04s03.md)。

有关创建交互、交互属性、约束、连接器截面和连接器截面分配的详细说明，请参见["使用交互模块，"第15.12节](pt03ch15s12.md)。
