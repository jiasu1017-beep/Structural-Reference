# 43.3 常用绘图选项概述





您可以使用常用绘图选项来自定义未变形和变形绘图的外观。常用选项还与**视图**菜单中的其他绘图状态无关选项以及等值线、符号和材料方向绘图的从属和独立自定义选项结合使用。从主菜单栏选择****选项**![](../graphics/images/arrow.gif)**常用****或使用工具箱中的![](../graphics/ico_plotCommonOptions.png)工具来访问**常用绘图选项**对话框。点击以下选项卡来自定义当前视口中绘图的外观：
- **基本**：选择渲染样式、边缘可见性和变形比例因子（仅适用于变形绘图）。
- **颜色和样式**：控制模型边缘颜色和样式、模型面颜色、边缘样式和边缘厚度。
- **标签**：控制单元、面和节点标签以及节点符号；并控制探测注释标签。
- **法线**：控制单元和表面法线。
- **其他**：**其他**页面包含以下选项卡：- **缩放**：控制模型缩放和收缩。- **半透明度**：控制着色和填充渲染样式的半透明度。

如果选择叠加未变形和变形模型形状，常用选项控制变形形状的显示，叠加绘图选项控制未变形形状的显示（有关详细信息，请参见["叠加变形和未变形模型绘图，" 第43.6节](pt05ch43hla02.md)）。绘图状态相关选项可能会覆盖某些常用选项；例如，等值线绘图的颜色选项将始终覆盖常用颜色选项。

要了解如何自定义绘图的渲染样式和其他显示特征，请参见[第55章，"自定义绘图显示"](pt05ch55.md)。
