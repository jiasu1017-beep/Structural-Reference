# 4 使用连续体单元







连续体（实体）应力/位移单元家族是 Abaqus 中最全面的单元库。Abaqus/Standard 和 Abaqus/Explicit 中可用的实体单元库有一些差异。

**Abaqus/Standard 实体单元库**

Abaqus/Standard 实体单元库包括一阶（线性）插值单元和二阶（二次）插值单元，维度为二维或三维，使用完全积分或减缩积分。二维中提供三角形和四边形；三维中提供四面体、三角楔形和六面体（"砖块"）。还提供改进的二阶三角形和四面体单元。

此外，在 Abaqus/Standard 中还可使用混合和不兼容模式单元。

**Abaqus/Explicit 实体单元库**

Abaqus/Explicit 实体单元库包括二维或三维的减缩积分一阶（线性）插值单元。还提供改进的二阶插值三角形和四面体单元。Abaqus/Explicit 中没有完全积分或常规二阶单元，但完全积分一阶六面体单元除外（此单元的不兼容模式版本也可用）。

有关连续体单元可用选项的详细信息，请参阅 ["Solid (continuum) elements," Abaqus Analysis User's Guide 第 28.1.1 节](../usb/usb-link.md#usb-elm-esolidcont)。

当考虑所有这些不同单元选项的排列时，可供您使用的实体单元总数很大——仅三维模型就有超过 20 种。模拟的准确性很大程度上取决于您在模型中使用的单元类型。选择这些单元中最好的一个对于您的模型可能看起来令人生畏，特别是起初。然而，您会将此选择视为一套 20+ 件工具，使您能够为特定工作选择正确的工具或单元。

本章讨论不同单元公式和积分级别对特定分析准确性的影响。还给出了选择连续体单元的一些一般指南。这些提供了您可以建立知识的基础，因为您在使用 Abaqus 方面获得更多经验。本节最后的示例将使您能够将这些知识付诸实践，因为您构建和分析一个连接耳片。
