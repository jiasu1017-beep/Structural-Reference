# 67.5.2 自定义矢量分量的颜色







您可以调整 Abaqus/CAE 用于显示合向量和在 1、2 或 3 方向上显示每个分量向量的颜色。这些设置针对力向量和力矩向量独立提供；**Free Body Plot Options**（自由体绘图选项）对话框为 **Force**（力）和 **Moment**（力矩）向量颜色提供相同的（但独立的）选项。

**自定义合向量或分量向量的颜色：**

1. 找到 **Colors**（颜色）选项。在可视化模块中从主菜单栏中选择 ****Options**![](../graphics/images/arrow.gif)**Free Body****（选项→自由体），然后在出现的对话框中点击 **Color & Style**（颜色和样式）选项卡。**Colors**（颜色）选项出现在页面顶部。
2. 分别点击 **Force**（力）选项卡或 **Moment**（力矩）选项卡来设置力向量或力矩向量的向量颜色。
3. 更改其中一个向量的向量颜色： 1. 点击合向量或其中一个分量向量的颜色样本（![](../graphics/usi-colorsample.png)）。Abaqus/CAE 显示 **Select Color**（选择颜色）对话框。 2. 使用 **Select Color**（选择颜色）对话框中的方法之一来选择新颜色。 有关更多信息，请参阅["自定义颜色，" 第 3.2.9 节"](pt01ch03s02s09.md)。 3. 点击 **OK**（确定）关闭 **Select Color**（选择颜色）对话框。颜色样本和所选向量更改为所选颜色。重复这些步骤以更改其他向量之一的向量颜色。
4. 点击 **OK**（确定）以实施您的更改并关闭 **Free Body Plot Options**（自由体绘图选项）对话框。默认情况下，您的更改会在会话期间保存，并将影响该视口中所有后续的自由体切割图。如果您想在后续会话中保留您的更改，请将它们保存到文件。有关更多信息，请参阅["保存自定义设置以供后续会话使用，" 第 55.1.1 节"](pt05ch55s01s01.md)。
