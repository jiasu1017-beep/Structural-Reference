# 28.5 体积分数工具





体积分数工具通过在欧拉部件实例和与之相交的第二个部件实例（参考部件实例）之间执行布尔比较来创建标量离散场。该比较确定两个部件实例重叠的位置，然后根据欧拉实例中也被参考实例占据的单元百分比，为欧拉实例中的每个单元分配体积分数。体积分数指定为零到一之间的小数。

体积分数工具创建的离散场可用于为欧拉部件实例分配材料实例（请参见["为欧拉部件实例分配材料，" 第28.4节](pt04ch28s04.md)）。所分配的欧拉材料实例的拓扑对应于欧拉部件实例内参考部件实例的形状。

[图28-7](pt04ch28s05.md#usi-adv-volfrac-proc-nls)和以下步骤总结了使用体积分数工具的过程：

**图28–7** 使用体积分数工具的过程。

![](../graphics/usi-adv-volfrac-proc-nls.png)

1. 使用Abaqus/CAE中的任何建模工具和技术，创建一个对应于所需欧拉材料区域几何形状的参考部件。2. 在欧拉部件实例中实例化参考部件。参考部件实例应在空间上对应于所需的欧拉材料区域。3. 使用体积分数工具基于参考部件实例和欧拉部件实例的比较创建离散场。4. 使用体积分数工具创建的离散场为欧拉部件实例定义材料分配预定义场。体积分数工具有一个选项，用于控制计算的离散场是表示参考实例内部的空间（在与参考实例重叠的单元中体积分数为非零）还是表示参考实例外部的空间（在与参考实例不重叠或部分重叠的单元中体积分数为非零），如图[28-8](pt04ch28s05.md#usi-adv-eulerian-volfrac-exa)所示。

**图28–8** 表示阴影参考区域内部（左侧）和外部（右侧）体积分数的离散场。

![](../graphics/usi-adv-eulerian-volfrac-exa.png)

通常，计算参考实例外部的体积分数用于在耦合欧拉-拉格朗日分析中在拉格朗日部件实例周围创建欧拉材料分配。计算参考实例内部的体积分数通常用于在纯欧拉部件实例内建模复杂的材料分配场；在这种情况下，分配欧拉材料后会抑制参考实例。您还可以计算参考实例内部的体积分数，以在耦合欧拉-拉格朗日分析中在封闭拉格朗日壳的内部创建欧拉材料分配。

要使用体积分数工具，请从主菜单栏中选择****工具**![](../graphics/images/arrow.gif)**离散场**![](../graphics/images/arrow.gif)**体积分数工具****。有关使用该工具的分步说明，请参阅["为材料体积分数创建离散场，" 第63.4节](pt06ch63hla03.md)。
