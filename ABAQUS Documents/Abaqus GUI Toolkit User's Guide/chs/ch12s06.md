# 12.6 Abaqus/Standard 中的通用接触





在 ["Abaqus/Standard 二维实例：通道成形，" 第 12.5 节](ch12s05.md) 的通道成形示例中，接触相互作用使用接触对算法定义，这需要您明确可能接触的表面。作为替代方案，您可以使用通用接触算法在 Abaqus/Standard 分析中指定接触。接触相互作用域、接触属性和表面属性独立地为通用接触指定，提供了更灵活的方式来增量添加模型细节。指定通用接触的简单界面允许高度自动化的接触定义；但是，也可以使用通用接触界面定义接触以模拟传统接触对。相反，使用接触对用户界面（如果使用表面-表面公式）指定跨越多个体的表面的自接触，模拟了通常用于通用接触的高度自动化方法。

在 Abaqus/Standard 中，接触相互作用的传统配对规范通常比为定义接触而采用全包容自接触方法更高效。因此，在定义接触的便捷性和分析性能之间通常存在权衡。Abaqus/CAE 提供了一种接触检测工具，极大地简化了为 Abaqus/Standard 创建传统接触对的过程（参见 ["理解接触和约束检测，" Abaqus/CAE 用户指南第 15.6 节](../usi/usi-link.md#usi-itn-auto-detection))。
