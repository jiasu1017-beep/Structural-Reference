# 5.6 Using the view manipulation tools







This section provides details of using the tools in the **View Manipulation** toolbar that allow you to manipulate the position, orientation, and scaling of the model or *X–Y* plot within a viewport. When the viewport you manipulate is linked to other viewports, Abaqus/CAE also changes the view within every linked viewport in your session. The following topics are covered:

- ["Rotating the view," Section 5.6.3](pt01ch05s06hlb03.md)
- ["Magnifying or reducing the view," Section 5.6.4](pt01ch05s06hlb04.md)
- ["Zooming in to a selected area of the view," Section 5.6.5](pt01ch05s06hlb05.md)
- ["Rescaling the view to fit the viewport," Section 5.6.6](pt01ch05s06hlb06.md)
- ["Cycling through views," Section 5.6.7](pt01ch05s06hlb07.md)
- ["Applying custom views," Section 5.6.8](pt01ch05s06hlb08.md)
