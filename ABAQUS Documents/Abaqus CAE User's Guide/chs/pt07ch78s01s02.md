# 78.1.2 理解显示组布尔运算





要创建或编辑显示组，您可以对所选模型组件和当前视口的内容执行布尔运算。Abaqus/CAE在显示组工具集中提供以下布尔运算：**替换**、**添加**、**移除**、**交集**和**异或**。

作为简单布尔运算的示例，假设当前视口显示整个模型。如果您选择一个单元集然后应用**移除**操作，则该单元集将从当前视口中的显示中消除。

对于您为创建或编辑显示组而执行的每个布尔运算，您只能选择一种类型的模型组件：零件实例、几何图形（单元、面或边）、基准几何（点、轴、平面或坐标系）、单元、节点、曲面、先前保存的显示组或整个模型。对于给定的显示组，Abaqus/CAE最初假定您想要包含组中所有单元连接的所有节点。但是，如果您选择了特定节点，则该显示组的所有后续操作仅包括您所选的节点。

以下是对每个布尔运算的说明。在下面的图标中，左侧的圆圈表示当前视口中的项目；右侧的圆圈表示您的选择。阴影部分表示生成的显示组。除最后一个操作**全部替换**外，所有这些布尔运算都在**创建显示组**对话框中可用；**显示组**工具栏提供对**替换**、**移除**、**异或**和**全部替换**布尔运算以及**撤销**和**重做**操作的访问。

**![](../graphics/ico_displayGroupReplace.png) 替换**

使用**替换**运算符将当前视口内容替换为您的选择。

**![](../graphics/ico_displayGroupAdd.png) 添加**

使用**添加**运算符将您的选择添加到当前视口内容。

**![](../graphics/ico_displayGroupRemove.png) 移除**

使用**移除**运算符从当前视口内容中移除您的选择。如果您的选择包含一个或多个单元或曲面，且您没有专门操作将这些节点包含在显示组中，则连接到这些项目的节点也会被移除。

**![](../graphics/ico_displayGroupIntersect.png) 交集**

使用**交集**运算符仅显示您的选择和当前视口中显示的公共项目。

**![](../graphics/ico_displayGroupEither.png) 异或**

使用**异或**运算符仅显示模型组件中位于您的选择或当前视口中、但不同时位于两者的组件。

**注意：****显示组**工具栏中的![](../graphics/ico_displayGroupInvertDisplay.png)按钮会隐藏当前显示组中显示的所有模型组件，并显示所有隐藏的模型组件。此操作等同于在**创建显示组**或**编辑显示组**对话框中选择所有对象并点击**异或**按钮。

**![](../graphics/ico_displayGroupShowAll.png) 全部替换**

使用**全部替换**运算符替换您的显示组并显示当前视口中的所有实体。此运算符仅在工具栏中可用。

当您从工具栏中选择**![](../graphics/ico_displayGroupReplace.png)（替换所选）**或**![](../graphics/ico_displayGroupRemove.png)（移除所选）**运算符时，Abaqus/CAE会在提示区域显示模型组件列表。选择您要从所选显示组中替换或移除的模型组件类型，然后点击视口中的一个或多个模型组件以更改显示组的内容。

**注意：****替换所选**和**移除所选**运算符执行与显示组对话框中**替换**和**移除**运算符相同的操作。工具栏运算符具有不同的名称，以指示您必须在使用这些运算符更改当前显示之前从视口中选择模型组件。
