# 19.10.5 Controlling adaptivity process parallel execution







To locate the adaptivity process editor, select ****Adaptivity**![](../graphics/images/arrow.gif)**Edit**![](../graphics/images/arrow.gif)***adaptivity process name***** from the main menu bar. Use the **Parallelization** tabbed page to control the parallel execution of an Abaqus process job as follows:
- Select the number of processors to use for the analysis if parallel processing is available.
- Select the **Threads** multiprocessing mode. Only the linear equation solver in Abaqus/Standard will execute in parallel. **Note:**You use the Step module to choose between the Abaqus/Standard iterative and direct sparse solvers. For more information, see ["Using the step editor," Section 14.10](pt03ch14s10.md).
