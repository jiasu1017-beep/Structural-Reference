# 42.3.2 逐步切换帧





您可以使用上下文栏中可用的**ODB Frame**控件按钮逐步切换结果帧。任何已保存分析结果且被设置为活动的帧都可用。有关选择特定结果帧的信息，请参见["选择特定的结果步骤和帧，" 第42.3.1节](pt05ch42s01hlb01.md)。有关激活结果步骤和帧的信息，请参见["激活和停用步骤和帧，" 第42.4.1节](pt05ch42s02s01.md)。

逐步切换帧数据仅适用于输出数据库。

**逐步切换结果帧：**

1. 找到**ODB Frame**按钮。当您显示未变形图、变形图、云图或符号图时，这些按钮会出现在上下文栏的右侧。![](../graphics/usv-anm-moviecontrols-nls.png)
2. 点击以下帧按钮之一：
   - **First frame**显示当前步骤的第一个活动帧的结果。如果已经显示步骤第一个帧的结果，此按钮无效。
   - **Previous frame**显示上一个活动帧的结果。如果当前显示的是步骤的第一个帧，Abaqus/CAE会显示前一个步骤的最后一个活动帧的结果。如果已经显示第一个步骤的第一个帧的结果，此按钮无效。
   - **Next frame**显示下一个帧的结果。如果当前显示的是步骤的最后一个活动帧，Abaqus/CAE会显示下一个步骤的第一个帧的结果。如果已经显示最后一个步骤的最后一个帧的结果，此按钮无效。
   - **Last frame**显示当前步骤最后一个帧的结果。如果已经显示步骤最后一个帧的结果，此按钮无效。
   - **Frame Selector**启动帧选择器对话框，使您可以直接导航到特定帧，方法是输入其帧号或拖动滑块到要显示的帧。有关更多信息，请参见["导航到动画中的特定帧，" 第49.4.2节](pt05ch49s04hlb02.md)。

   当前视口中的模型图会更改以显示你在所选步骤和帧下的模型。如果处于活动状态，[状态块](pt05ch56s03.md)中的文本会更改以标识所选的步骤和帧。Abaqus会刷新**Step/Frame**对话框，高亮显示所选步骤和帧，以及**Field Output**对话框，列出所选帧可用的变量。Abaqus还会刷新所有标识当前步骤和帧的对话框。
3. 继续点击帧按钮逐步切换可用帧。
