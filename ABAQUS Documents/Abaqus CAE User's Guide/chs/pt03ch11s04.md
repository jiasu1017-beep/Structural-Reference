# 11.3.3 简化零件的特征列表





复制零件到新零件时，可以将所有特征和参数信息简化为简单定义。如果简化特征列表，Abaqus/CAE在您随后修改零件时会更快地重新生成；但是，您将不再能够修改零件的任何参数。您可以通过从主菜单栏选择**零件**![](../graphics/images/arrow.gif)**复制**![](../graphics/images/arrow.gif)***零件名称***来复制零件。

如果您的零件花费了大量时间创建且设计迭代了很多次，简化零件的特征列表尤其有用。例如，如果您创建了一个槽并在确定最终设计之前重新调整了槽的尺寸，原始零件将包含定义槽的每个变体的特征。如果您复制零件并简化特征列表，新零件将只包含一个定义槽最终版本的功能。
