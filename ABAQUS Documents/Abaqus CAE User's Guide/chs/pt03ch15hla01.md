# 15.18 使用查询工具集获取连接器分配信息





您可以使用查询工具集来显示连接器截面分配的详细信息。

**显示线或附件线信息的步骤：**

1. 从主菜单栏选择 ****Tools**![](../graphics/images/arrow.gif)**Query****。**提示：**您也可以通过点击「Query」工具栏中的![](../graphics/ico_info.png)工具来查询模型。Abaqus/CAE会显示「Query」对话框。您可以请求通用查询或特定于模块的查询。有关通用查询显示的信息的讨论，请参见「获取模型的一般信息」，第71.2.2节。「Connector assignments」查询特定于相互作用模块。
2. 从「Interaction Module Queries」列表中，选择「Connector assignments」。此时会显示连接器截面分配的摘要。
3. 在视口中选择您要查询的线或附件线。（更多信息，请参见第6章，「在视口中选择对象」。）
4. 点击「Done」或点击鼠标按钮2以显示所选线或附件线的连接器截面分配的详细信息。
5. 要退出查询过程，请点击提示区域中的取消按钮![](../graphics/afxI_cancel.png)。
