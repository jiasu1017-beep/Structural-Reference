### 概述

ALE自适应网格划分的输出：
- 可用于验证用户定义域的自动分割、拉格朗日边和角的形成、几何边和角的形成以及非自适应节点的确定
- 必须仔细解释，因为网格特定位置的输出变量值不再与特定材料点上的值相关联
- 可包括示踪粒子的定义，示踪粒子跟随材料点并允许您检查这些点的轨迹并绘制这些点上所有单元和节点变量的材料时间历史
- 可包括自适应网格划分效率和平流精度的诊断信息

### 验证模型

可用于验证自适应网格划分模型的输出在数据（`.dat`）文件和输出数据库（`.odb`）中可用（见["Output," Section 4.1.1](pt02ch04s01aus38.md)，了解这些文件的详细信息）。

#### 单元集

当用户定义的自适应网格域被Abaqus/Explicit分割时，组成新细分域的单元被打印到数据（`.dat`）文件。

为所有自适应网格域创建新的单元集并写入输出数据库（`.odb`）。为每个域创建的单元集的名称是用户定义的名称，加上细分数（如果没有创建细分则为1），加上步号。例如，如果为单元集*domain_name*指定的用户定义自适应网格域跨越三个不相交的部分，Abaqus/Explicit将把用户定义的域分割成三个域，并在第一步的输出数据库（`.odb`）中为域创建三个单元集：*domain_name*`-1-1`、`*domain_name*`-2-1`和`*domain_name*`-3-1`。

Abaqus/CAE可用于验证细分域的创建。

#### 边和非自适应节点

Abaqus/Explicit根据自适应网格域的拓扑、与非自适应域的连接以及用户指定的边界区域自动形成拉格朗日边和角并识别非自适应节点。此外，几何边和角根据初始几何形状和初始特征角度值自动形成。见["在Abaqus/Explicit中定义ALE自适应网格域，" Section 12.2.2](pt04ch12s02aus78.md)。拉格朗日边、几何边和角以及非自适应节点（包括拉格朗日角点）被输出到每个自适应网格域的数据（`.dat`）文件。可以通过请求到数据文件的模型和历史定义摘要打印输出来获取此信息（见["Model and history definition summaries" in "Output," Section 4.1.1](pt02ch04s01aus38.md#usb-out-ooutput-modelhist-sum)），或者通过监控自适应网格划分的进度（见下面["监控ALE自适应网格划分的进度"](pt04ch12s02aus81.md#usb-anl-aaleoutput-adaptive-prog)"）。

此外，最多为每个自适应网格域中的每步创建三个节点集在输出数据库（`.odb`）中。节点集的名称通过连接以下信息创建：
- 域单元集名称
- 细分编号（如果没有创建细分则为1）
- 字母LE表示拉格朗日边，GE表示几何边或角，NA表示非自适应节点（包括拉格朗日角点）
- 步号

例如，如果为单元集*domain_name*指定的用户定义三维自适应网格域自动分割成两个自适应网格域，Abaqus/Explicit将在第一步的输出数据库中为域生成最多六个节点集：*domain_name*`-1-LE-1`、`*domain_name*`-1-GE-1`、`*domain_name*`-1-NA-1`、`*domain_name*`-2-LE-1`、`*domain_name*`-2-GE-1`和`*domain_name*`-2-NA-1`。

由于在二维中边界区域由角点分隔（不是边），因此不会为二维自适应网格域中的拉格朗日边创建节点集。拉格朗日角点包含在非自适应（NA）节点集中，与三维域相同。

Abaqus/CAE可用于验证拉格朗日边和角、几何边和角以及非自适应节点的创建。

### 解释结果

当不执行自适应网格划分时，有限元网格跟随材料，这使得分析结果的解释很简单。您可以通过研究网格的运动来可视化变形和材料运动。由于网格在整个时间固定到同一材料点，每个节点和单元输出变量对应于特定材料位置。

一旦执行自适应网格划分，网格和材料点的位置会偏离，必须相应地解释分析结果。自适应网格域内部的网格运动代表材料运动和自适应网格划分的综合效应。在拉格朗日和滑移边界区域上，网格和材料的运动在垂直于边界的方向上相同，但在切向方向上不同。

#### 节点变量

当执行自适应网格划分时，在步开始时与节点重合的材料点可能不会在该步剩余时间内保持与该节点重合。位移值和当前坐标代表节点的运动，不一定是材料的运动。所有其他节点变量——包括速度、加速度和反作用力——代表材料点在节点当前位置处的变量值。这些变量的等值线或矢量图将显示其正确的空间分布，因此是有意义的。然而，对于经历自适应网格划分的节点，节点变量的时间历史通常没有意义。然而，在稳态问题中，基于固定空间位置（而不是特定材料点）的速度或加速度时间历史可能有用。

#### 单元变量

类似地，当执行自适应网格划分时，在步开始时与单元积分点重合的材料粒子可能不会在该步剩余时间内保持重合。因此，单元积分点变量不一定代表整个步中同一材料点上的值。单元积分点变量的等值线或矢量图由于与节点变量相同的原因是有意义的。然而，时间历史基于单元积分点的空间位置，而不是特定材料点。

整体单元变量具有类似的解释。

### 在材料点上跟踪节点或单元变量

可以定义示踪粒子来跟踪自适应网格域中的材料点。这些粒子也可用于获取节点或单元积分点变量的时间历史，这些变量对应于特定材料点上变量随时间的变化。示踪粒子定义如下所述（见["输出到输出数据库，" Section 4.1.3](pt02ch04s01aus40.md)，了解更多相关信息）。可以为示踪粒子集请求节点和单元变量输出，以检查材料粒子的轨迹或获取材料时间历史。示踪粒子的输出只能写入输出数据库（`.odb`）。

#### 在拉格朗日域中使用示踪粒子

在使用拉格朗日域的大多数自适应网格划分模拟中，域中的节点和单元既不对应特定的的空间位置也不对应特定的材料点或体积。因此，节点和单元积分点变量的时间历史在拉格朗日自适应网格域中通常没有物理意义。应定义示踪粒子来查看时间历史信息。示踪粒子也可用于可视化材料的运动。

示踪粒子的初始位置定义为与称为父节点的节点重合。示踪粒子通过定义多个父节点或节点集来定义为集合。您指出其当前位置对应于示踪粒子初始位置的节点，并为示踪粒子集分配一个名称以用于输出请求。示踪粒子在其定义的步中重复地从父节点释放，间隔指定。在该步剩余时间和所有后续步中，粒子跟随材料点。

示踪粒子通常仅在自适应网格域上定义，尽管它们可以定义在模型中任何低阶实体单元连接的节点上。对于在后续步之前不执行自适应网格划分的分析，可以在分析开始时在非自适应域上定义示踪粒子，并将在域变得自适应时持续跟踪。类似地，如果自适应网格域拓扑从步到步变化，示踪粒子将从域跟踪到域。

| **输入文件用法：** | 使用以下选项定义示踪粒子集： |
| --- | --- |
|  | ``` [*TRACER PARTICLE](../key/key-link.md#usb-kws-htracerparticle), TRACER SET=*tracer_set_name* *list of tracer particle parent nodes* ``` |

| **Abaqus/CAE用法：** | Abaqus/CAE不支持示踪粒子。 |
| --- | --- |

#### 在欧拉域中使用示踪粒子

欧拉域中节点和单元积分点的时间历史在施加空间自适应网格约束的位置可能具有物理意义。例如，沿流出欧拉边界单元中等效塑性应变的 시간变化充当该变量的空间时间历史，可用于评估过程是否达到稳态解。

可以定义示踪粒子来评估材料点流过欧拉域时变量在材料点处的材料时间历史。示踪粒子也可用于评估材料点流过域时的轨迹和路径。

示踪粒子可以被分配到欧拉自适应网格域中的任何父节点。如果示踪粒子到达流出边界且材料继续流出，则不再跟踪示踪粒子，且与示踪粒子关联的所有输出历史变量在停用后将为零。

当材料通过网格域流动显著时，可以在步中多次从父节点当前位置释放示踪粒子集。每次示踪粒子释放称为粒子出生。粒子出生后，示踪粒子无论网格如何运动都跟随材料的运动。您可以指示步中的粒子出生阶段数。这些阶段将在步的时间段内均匀分布。

例如，可以定义示踪粒子集，使得沿流入欧拉边界的所有节点都是父节点。可以指定多个出生阶段，使得在步期间示踪粒子集周期性地从流入边界处的网格释放。如果定义了足够多的出生阶段，当材料从流入边界流到流出边界时，域最终将被示踪粒子跨越。

| **输入文件用法：** | 使用以下选项定义具有多个出生阶段的示踪粒子集： |
| --- | --- |
|  | ``` [*TRACER PARTICLE](../key/key-link.md#usb-kws-htracerparticle), TRACER SET=*tracer_set_name*, PARTICLE BIRTH STAGES=*n* *list of tracer particle parent nodes* ``` |

| **Abaqus/CAE用法：** | Abaqus/CAE不支持示踪粒子。 |
| --- | --- |

### 监控ALE自适应网格划分的进度

诊断信息可以写入消息（`.msg`）文件以跟踪自适应网格划分的效率和准确性。您可以选择写入的诊断输出级别。

#### 获取步结束时的摘要

默认情况下，每个自适应网格域的自适应网格划分信息摘要将在每个步结束时写入消息（`.msg`）文件。此摘要信息包括：
- 移动节点的平均百分比
- 移动节点的最大百分比
- 移动节点的最小百分比
- 平均平流扫描次数

每个值针对单个自适应网格域在所有自适应网格划分增量上计算。平流成本与移动节点的百分比大致成比例，因为对于在自适应网格划分期间未被重新定位的单元，不平流变量。

| **输入文件用法：** | 使用以下选项请求每个自适应网格域在每个步结束时的摘要： |
| --- | --- |
|  | ``` [*DIAGNOSTICS](../key/key-link.md#usb-kws-hdiagnostics), ADAPTIVE MESH=STEP SUMMARY ``` |

| **Abaqus/CAE用法：** | Abaqus/CAE不支持自适应网格诊断。 |
| --- | --- |

#### 获取每个ALE自适应网格划分增量的摘要

除了步摘要信息外，以下诊断信息可为每个自适应网格域在每个自适应网格划分增量获取：
- 移动节点的百分比
- 平流扫描次数

| **输入文件用法：** | 使用以下选项在步结束时和每个自适应网格划分增量结束时获取摘要信息： |
| --- | --- |
|  | ``` [*DIAGNOSTICS](../key/key-link.md#usb-kws-hdiagnostics), ADAPTIVE MESH=SUMMARY ``` |

| **Abaqus/CAE用法：** | Abaqus/CAE不支持自适应网格诊断。 |
| --- | --- |

#### 获取每个ALE自适应网格划分增量的平流精度详细信息

以下详细诊断信息也可写入消息（`.msg`）文件以跟踪平流的准确性：
- 平流前后的质量和动量
- 体积变化百分比

| **输入文件用法：** | 使用以下选项请求最详细的诊断信息，包括平流精度测量和每个自适应网格域的摘要信息，在每个自适应网格划分增量报告： |
| --- | --- |
|  | ``` [*DIAGNOSTICS](../key/key-link.md#usb-kws-hdiagnostics), ADAPTIVE MESH=DETAIL ``` |

| **Abaqus/CAE用法：** | Abaqus/CAE不支持自适应网格诊断。 |
| --- | --- |

#### 抑制ALE自适应网格诊断

您可以抑制所有自适应网格诊断信息的输出。

| **输入文件用法：** | ``` [*DIAGNOSTICS](../key/key-link.md#usb-kws-hdiagnostics), ADAPTIVE MESH=OFF ``` |
| --- | --- |

| **Abaqus/CAE用法：** | Abaqus/CAE不支持自适应网格诊断。 |
| --- | --- |
