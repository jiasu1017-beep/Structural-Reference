# 71.1.2 特定于模块的查询







除了常规查询外，您还可以查询模型以获取特定于您正在使用的模块的信息。Abaqus/CAE在查询工具集底部的"***模块名* Module Queries**"下显示这些特定于模块的查询。查询工具集可以提供以下特定于模块的信息：

**Part模块**

**Part Module Queries**下的项目提供有关当前部件的以下特定于模块的信息：

**Part attributes**

- 名称
- 建模空间
- 类型（可变形或刚体）

**Regeneration warnings**

如果选定点中的某些集或曲面由于其底层几何体已被修改或删除而无法重新生成，Abaqus/CAE会显示集或曲面名称、原始面数以及查询期间找到的面数。

**Substructure statistics**

Abaqus/CAE显示有关所选子结构部件的以下信息：部件中保留的节点数、特征模态数和子结构载荷数；子结构中恢复矩阵、重力载荷向量、缩减质量矩阵、缩减结构阻尼矩阵和缩减粘性阻尼矩阵的可用性；以及子结构的质量属性。

有关更多信息，请参见["在Part模块中使用查询工具集，"第11.16.4节](pt03ch11s16s04.md)。

**Property模块**

**Property Module Queries**下的项目提供有关当前部件的以下特定于模块的信息：

**Section assignments**

分配给选定区域的截面

**Regions missing sections**

需要分配截面的区域

**Beam orientations**

分配给选定线区域的梁方向（Abaqus/CAE在选定的线区域上显示(![](../graphics/usi_eqn00394.gif), ![](../graphics/usi_eqn00395.gif), ![](../graphics/usi_eqn00396.gif))轴系）

**Material orientations**

分配给选定区域的材料方向

**Rebar orientations**

分配给选定区域的钢筋参考方向

**Ply stack plot**

Abaqus/CAE创建一个新视口，并显示复合铺层或复合截面区域中芯样的图形表示。图像显示铺层中的铺层以及每个铺层的详细信息，例如其纤维方向、厚度、参考平面和积分点。

**Disjoint ply regions**

Abaqus/CAE在消息区域显示包含不连续区域的复合铺层及其内部的铺层名称。

有关更多信息，请参见["使用查询工具集获取分配信息，"第12.19节](pt03ch12hla03.md)和["查看铺层图，"第53章](pt05ch53.md)。

**Assembly模块**

**Assembly Module Queries**下的项目提供有关选定部件实例的以下特定于模块的信息：

**Instance attributes**

名称、类型和建模空间

**Instance position**

- 相对于全局坐标系原点的位置
- 应用于实例的平移和旋转的总和
- 应用的平移和旋转约束的数量

有关更多信息，请参见["使用查询工具集查询装配体，"第13.12节](pt03ch13hla01.md)。

**Step模块**

查询工具集在Step模块中仅提供一般信息。

**Interaction模块**

**Interaction Module Queries**下的项目提供有关选定线的以下特定于模块的信息：
- 连接器分配信息

有关更多信息，请参见["使用查询工具集获取连接器分配信息，"第15.18节](pt03ch15hla01.md)。

**Load模块**

查询工具集在Load模块中仅提供一般信息。

**Mesh模块**

**Mesh Module Queries**下的项目提供有关部件或部件实例的以下特定于模块的信息：
- 自由/非流形边
- 未网格化的区域
- 未关联的几何体

有关更多信息，请参见["获取网格信息，"第17.19.2节](pt03ch17s19hlb02.md)。

**Job模块**

Abaqus/CAE工具集在Job模块中均不可用。

**Visualization模块**

**Visualization Module Queries**下的项目提供以下特定于模块的信息：
- 探针值。当您在当前视口中移动光标时，Abaqus/CAE会在**Probe Values**对话框中显示信息。对模型图进行探测会显示模型数据和分析了结果；对*X--Y*图进行探测会显示*X--Y*曲线数据。有关更多信息，请参见["探测模型，"第51章](pt05ch51.md)。
- 应力线性化。应力线性化是通过截面将应力分离为恒定膜应力和线性弯曲应力。Abaqus/CAE执行应力线性化计算，并以*X--Y*图的形式显示结果。有关更多信息，请参见["计算线性化应力，"第52章](pt05ch52.md)。
- 活动单元或节点。Abaqus/CAE显示当前视口中所有活动节点或活动单元的标签编号。有关更多信息，请参见["查询活动节点或单元标签，"第50.2.2节](pt05ch50s02hlb02.md)。
- 铺层图。Abaqus/CAE创建一个新视口，并显示复合铺层或复合截面区域中芯样的图形表示。图像显示铺层中的铺层以及每个铺层的详细信息，例如其纤维方向、厚度、参考平面和积分点。有关更多信息，请参见["查看铺层图，"第53章](pt05ch53.md)。

**Sketch模块**

**Sketch Module Queries**下的项目提供有关选定约束或草图的以下信息：

**Constraint**

- 约束类型
- 受约束的实体名称

此外，受约束的实体在草图中高亮显示。

**Detail**

- 几何体数量
- 顶点数量
- 约束数量
- 维度数量
- 未约束自由度数量
