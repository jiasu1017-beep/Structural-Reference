# 64.9 撤销或重做网格编辑工具集中的更改





本节介绍如何撤消或重做您使用网格编辑工具集中的工具对网格进行的更改。本节还解释如何配置存储网格编辑操作回滚信息的缓存。涵盖以下主题：
- ["撤消或重做网格编辑操作，" 第64.9.1节](pt06ch64s09s01.md)
- ["启用撤销和管理缓存，" 第64.9.2节](pt06ch64s09s02.md)

这些主题也适用于撤消和重做自底向上网格划分操作（有关自底向上网格划分的更多信息，请参见["自底向上网格划分，" 第17.11节"](pt03ch17s11.md)）。

Abaqus/CAE支持对网格编辑工具集中的任何工具以及您使用自底向上网格划分技术生成的网格进行多级撤销和重做。这意味着单个撤销会移除您对网格的最新更改，第二个撤销会移除下一个最近的更改，依此类推。类似地，单个重做会恢复最近撤消的网格更改。下图显示**编辑网格**对话框的**撤销**部分包括**撤销**和**重做**按钮旁边的简短描述，高亮显示如果您点击按钮将撤消或重做的网格更改。

![](../graphics/edt-undo-edit-mesh-win-nls.png)

当撤销被禁用时，描述空间中会出现一条注释，指示撤销在**设置**下被禁用。要重新启用撤销，请参见["启用撤销和管理缓存，" 第64.9.2节"](pt06ch64s09s02.md)。

Abaqus/CAE通过在内存缓存中记录网格编辑和自底向上网格划分操作的历史来支持撤销操作。此缓存的允许大小、您的网格大小以及您执行的网格编辑类型都决定了允许的撤销级别数。例如，如果您的网格有100,000个单元，并且您定义了0.5百万个单元的最大缓存大小，Abaqus/CAE至少可以提供五级撤销。创建或移动节点等操作不需要大缓存即可撤消。

**提示：**您可以通过运行**零件网格**或**实例网格**查询来确定网格中的单元数。有关更多信息，请参见["查询模型，" 第71.2节"](pt06ch71s02.md)。

大缓存会消耗大量内存，因此请考虑零件或零件实例所需的撤销级别数。如果您想确保任何网格编辑操作至少有一级撤销，请将缓存大小指定为至少与零件或零件实例中的单元数相同。
