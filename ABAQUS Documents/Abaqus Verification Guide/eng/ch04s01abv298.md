### Feature tested

User subroutine to allow reading of the results file at the end of any increment in an analysis.

### Problem description

This set of verification problems ensures that the subroutine [`URDFIL`](../sub/sub-link.md#sub-xsl-urdfil) is called properly for the various procedure types. In each problem the utility routine `POSFIL` is called from within [`URDFIL`](../sub/sub-link.md#sub-xsl-urdfil). Use is made of the `LSTOP` and `LOVRWRT` parameters such that the results file differs from that which would be produced by the same analysis without [`URDFIL`](../sub/sub-link.md#sub-xsl-urdfil).

### Results and discussion

The results file can be checked to ensure that [`URDFIL`](../sub/sub-link.md#sub-xsl-urdfil) and `POSFIL` are functioning correctly for the various procedures.
