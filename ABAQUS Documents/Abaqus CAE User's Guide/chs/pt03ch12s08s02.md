# 12.8.2 指定与解相关的状态变量







与解相关的状态变量是用户子程序中的值，您可以定义这些值以随着分析的解而演变。如果您在用户子程序中引用材料定义，可以使用**编辑材料**对话框指定在应用该材料的点或节点处所需的状态变量数量。请参见"用户子程序：概述"，Abaqus分析用户指南第18.1.1节。

**指定与解相关的状态变量数量：**

1. 从**编辑材料**对话框的菜单栏中，选择**常规**![](../graphics/images/arrow.gif)**状态变量**。（有关显示**编辑材料**对话框的信息，请参见"创建或编辑材料"，第12.7.1节。）
2. 单击**与解相关的状态变量数量**字段右侧的箭头，指定您要为每个适用的积分点或接触从属节点分配空间的状态变量数量。
3. 如果适用，请在标为**控制元素删除的变量编号（仅限Abaqus/Explicit）**的字段中输入控制元素删除的状态变量编号。有关更多信息，请参见"使用状态变量从网格中删除元素"，Abaqus分析用户指南第26.7.1节"用户定义的机械材料行为"。
4. 单击**确定**关闭**编辑材料**对话框。或者，您可以从**编辑材料**对话框的菜单中选择其他材料行为进行定义（有关更多信息，请参见"浏览和修改材料行为"，第12.7.2节）。
