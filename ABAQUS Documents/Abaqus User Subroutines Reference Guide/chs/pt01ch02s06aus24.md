# 2.6.1 调整和/或重新分配单元集的质量





**产品：** Abaqus/Explicit

##### **参考文献**

- ["密度，" 第21.2.1节](pt05ch21s02abm01.md)
- ["点质量，" 第30.1.1节](pt06ch30s01alm21.md)

- [*MASS ADJUST](../key/key-link.md#usb-kws-mmassadjust)

### 概述

质量调整：
- 可用于将模型中一个或多个部件的净质量设置为已知值；
- 可用于修正由于建模近似造成的质量误差；
- 可用于考虑来自模型中省略的非结构特征（如油漆）的质量；
- 可应用于所有具有质量的单元类型；
- 按比例调整单元集中各个单元的质量（包括任何非结构质量），以达到为该集指定的目标值；
- 可用于在集中重新分配质量，以将最小稳定时间增量提高到目标值；
- 在模型定义期间只能在Abaqus/Explicit分析中指定一次；以及
- 可以按层次方式应用，先调整各个部件的质量，然后调整这些部件装配的质量。

### 将单元集的总质量调整到已知值

数值模型中部件的质量可能由于多种原因（包括建模近似和从模型中省略次要特征）而与其实际值不同。您可以通过识别定义这些部件的单元集及其各自的总体质量值来指定此类部件的数值模型中的质量调整。对于给定的单元集，质量在分析开始时调整，使得该集中每个单元的调整量与该单元调整前的质量成比例，从而保持质心和转动惯量的主方向。单元的调整前质量包括由于任何相关材料密度产生的质量；直接在截面定义上指定的质量（如梁、管、壳、膜、刚性和表面单元的情况）；以及直接施加到该单元的任何非结构质量。["膝部冲击与通用接触，" Abaqus示例问题指南第2.1.9节](../exa/exa-link.md#exa-dyn-kneebolster)是使用质量调整设置单元集总质量的示例。

当对具有活动转动自由度的单元调整质量时，该单元的转动惯量贡献也会按与质量调整中单元质量缩放成比例的方式修改，从而保持转动惯量的主方向。计算单元的稳定时间增量时会考虑调整后的质量值。诸如质量比例阻尼（请参阅["材料阻尼，" 第26.1.1节](pt05ch26s01abm51.md)）和重力等载荷会考虑调整后的质量。

质量调整可以按层次方式应用，先调整各个部件的质量，然后调整这些部件装配的质量。在这种情况下，定义在装配上的质量调整可以进一步修改各个部件的调整后质量。您必须以所需顺序将所有质量调整的单元集与单个质量调整定义关联。

如果请求模型定义数据，Abaqus/Explicit会自动计算每个单元集的质量、质心和转动惯量，并将结果打印到数据（`.dat`）文件中（请参阅["控制在数据文件中写入的分析输入文件处理器信息量"中的"输出，" 第4.1.1节](pt02ch04s01aus38.md#usb-out-ooutput-data-control)）。质量调整的贡献也会列在这些表中。可以请求单元输出变量MASSADJUST作为输出到输出数据库（`.odb`）文件的输出，它将指示如何对包含在该集中的每个单元调整或重新分配质量（请参阅["Abaqus/Explicit输出变量标识符，" 第4.2.2节](pt02ch04s02xbv01.md)）。此输出变量可作为场输出（等值线图）在第一个分析步骤的第一个输出帧中获得。

### 重新分配质量以将最小稳定时间增量提高到目标值

您可以通过在单元集中重新分配质量，将初始配置中单元集的最小稳定时间增量提高到指定的目标值。影响稳定时间增量的质量重新分配和达到目标总质量的质量调整可以相互独立地请求。如果为给定单元集请求了两个选项，则首先调整质量以达到集的目标总质量，然后在单元之间重新分配以达到目标时间增量。

您可以设置适用于所有质量调整单元集的默认目标时间增量，以及为任何单个单元集设置特定目标。在每个集中，质量从低于目标值的单元转移到具有高于目标值的单元。如果请求模型定义数据，Abaqus/Explicit会将可用于重新分配的质量量以及重新分配到数据（`.dat`）文件的此数量的百分比打印出来（请参阅["控制在数据文件中写入的分析输入文件处理器信息量"中的"输出，" 第4.1.1节](pt02ch04s01aus38.md#usb-out-ooutput-data-control)）。如果没有足够的质量来满足指定的目标时间增量，分析将终止并显示错误消息。["盛水瓶子的冲击，" Abaqus示例问题指南第2.3.2节](../exa/exa-link.md#exa-dyn-bottledrop)是使用质量调整保持单元集目标稳定时间增量的示例。

与固定质量缩放功能相比，上述重新分配功能不会改变集的总质量。但是，这两个功能都会影响质心和转动惯量的主方向。重新分配功能仅在分析开始时的初始配置中执行；而固定质量缩放在请求该质量缩放的步骤开始时的配置中执行。当您指定质量调整和质量缩放时，质量缩放会在调整后的质量之上根据需要添加质量。

### 定义质量调整

要调整模型中一个或多个部件的总质量，首先需要识别相应的单元集。如果指定多个单元集，质量将按单元集指定的顺序调整。对于共享单元的单元集，您必须确定指定单元集的顺序以获得所需结果。

#### 为单元集定义总质量而不改变其质心

您必须为每个质量调整的单元集指定总质量。

| **输入文件用法：** | ``` [*MASS ADJUST](../key/key-link.md#usb-kws-mmassadjust) *element_set_name, element_set_mass* ``` |
| --- | --- |

#### 定义质量重新分配以提高时间增量

您可以重新分配单元集的质量以达到目标时间增量，并可为每个质量调整的单元集指定总质量，或者可以在不改变单元集现有总质量的情况下重新分配质量。您可以设置适用于所有质量调整集的默认目标时间增量，以及为任何单个集设置特定目标。当同时指定了默认目标和特定目标时，该集使用特定目标。

| **输入文件用法：** | 使用以下选项提高时间增量并指定总质量： |
| --- | --- |
|  | ``` [*MASS ADJUST](../key/key-link.md#usb-kws-mmassadjust), TARGET DT=*min_stable_time_increment* *element_set_name, element_set_mass, element_set_min_stable_time_increment* ``` 使用以下选项在不改变总质量的情况下提高时间增量： ``` [*MASS ADJUST](../key/key-link.md#usb-kws-mmassadjust), TARGET DT=*min_stable_time_increment* *element_set_name*, CURRENT, *element_set_min_stable_time_increment* ``` |
