# 7.2.1 收敛与时间积分准则：概述







Abaqus/Standard中有许多与收敛和积分精度算法相关的控制参数。这些参数被赋予默认值，这些值经过选择以优化各种非线性问题的求解精度和效率。用户可以更改求解控制参数，如以下各节所述：
- ["常用控制参数，" 第7.2.2节](pt03ch07s02aus50.md)提供了更重要的求解控制参数的简要概述，以及可以有效使用它们的情况描述。本节可能对一般用户最有帮助，应该首先阅读。
- Abaqus/Standard包含一种经验算法，旨在准确且经济地求解非线性系统的平衡方程。用于建立非线性增量收敛的准则以及基于收敛率自动调整增量大小的方法在["非线性问题的收敛准则，" 第7.2.3节](pt03ch07s02aus51.md)中描述。
- Abaqus/Standard允许用户在具有物理时间尺度的问题中选择"时间积分精度参数"。使用这些参数自动控制时间增量大小的算法在["瞬态问题的时间积分精度，" 第7.2.4节](pt03ch07s02aus52.md)中描述。
- Abaqus/CFD允许用户选择Abaqus/CFD与Abaqus/Standard或Abaqus/Explicit协同仿真中使用的控制参数，以缓解分析过程中的不稳定性和平面网格畸变。

### 修改默认求解控制

在大多数情况下，无需调整求解控制参数的默认值。但是，用户可以在步骤定义中重置它们。

求解控制参数的值将在分析的剩余部分保持有效，直到被重置。
