# 4.3.4 T4：带对流的二维热传递







**产品：** Abaqus/Standard  Abaqus/Explicit  

### 测试单元

DC2D3    DC2D4    DC2D6    DC2D8  

DC3D4    DC3D6    DC3D8    DC3D10    DC3D15    DC3D20  

CPE3T    CPE4RHT    CPE4RT    CPE6MHT    CPE6MT    CPS3T    CPS4RT    CPS6MT    C3D4T    C3D6T    C3D8RHT    C3D8RT    C3D8T    C3D10MHT    C3D10MT  

EC3D8RT  

SC6RT    SC8RT  

### 问题描述

![](../graphics/bmkt4.png)

**模型：**

二维几何形状如上图所示。三维单元在z方向上测试厚度为1.0。在Abaqus/Standard中进行稳态模拟，而在Abaqus/Explicit中进行瞬态模拟。后者的总模拟时间为20000秒，这为瞬态解达到该问题的稳态条件提供了足够的时间。

**材料：**

导热系数 = 52 W/m·℃，表面对流系数 = 750 W/m²/℃。

对于耦合温度-位移单元，使用虚拟机械特性来完成材料定义。

**边界条件：**

沿边缘AB温度 = 100℃。沿边缘DA热通量为零。沿边缘BC和CD对流到环境温度0℃。

**载荷：**

零内部热生成。

### 参考解

这是英国国家有限元方法与标准机构（NAFEMS）推荐的测试：NAFEMS出版物TNSB第3版"The Standard NAFEMS Benchmarks"（1990年10月）中的测试T4。

目标解：E点处的温度为18.3℃。
