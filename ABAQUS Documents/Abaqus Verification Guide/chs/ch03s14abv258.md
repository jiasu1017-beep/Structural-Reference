# 3.14.18 通过方向进行结果传递







**产品：** Abaqus/Standard  Abaqus/Explicit  

### I. 在Abaqus/Explicit和Abaqus/Standard之间传递结果

### 测试的元素

C3D8R  CPE4R  CPS4R  C3D10M  CPE6M  CPS6M  M3D4R  S4R  S4  

### 问题描述

本节的验证测试包括在Abaqus/Standard和Abaqus/Explicit之间传递方向定义以及反向传递的测试。测试涉及在单调递增载荷作用下的单元素简单剪切。

验证测试还包括一些具有增强沙漏控制的一阶减缩积分单元。

所有测试使用的材料模型为正交弹性，通过在主方向上指定广义杨氏模量、泊松比和剪切模量来定义。使用以下材料属性（单位不重要）：

| --- |
| ![](../graphics/ver_eqn01343.gif) 100 109 |
| ![](../graphics/ver_eqn01344.gif) 100 109 |
| ![](../graphics/ver_eqn01345.gif) 0.3 |
| ![](../graphics/ver_eqn01346.gif) 0.23 |
| ![](../graphics/ver_eqn01347.gif) 0.34 |
| ![](../graphics/ver_eqn01348.gif) 76.9 109 |
| ![](../graphics/ver_eqn01349.gif) 76.9 109 |
| ![](../graphics/ver_eqn01350.gif) 9.0 109 |
| Density = 7850. |

由于定义了非各向同性材料行为，因此需要局部坐标系来将各向异性行为与材料方向相关联。在原始分析中指定非默认方向，使局部材料方向相对于单元局部方向倾斜45度。采用大位移分析，导致非默认局部坐标系随着材料点处的平均刚体运动而旋转。默认情况下，方向定义会传递到导入分析。

默认情况下，应力、应变、截面力和截面应变（如果适用）都以局部坐标系报告。

还对一个具有三层的复合壳进行了验证测试。使用S4R单元。相对于局部方向定义45度的截面方向。针对材料计算，为各层定义了相对于新定义的截面方向15度、30度和45度的附加方向。用于测试的材料模型为上述正交弹性。

### 结果与讨论

结果表明，方向定义在Abaqus/Explicit和Abaqus/Standard之间成功传递。

### II. 从一个Abaqus/Standard分析向另一个Abaqus/Standard分析传递结果

### 测试的元素

C3D8  C3D8R  CPE4  CPE4R  CPS4  CPS4R  C3D10M  CPE6M  CPS6M  M3D4R  S4R  

### 问题描述

本节的验证测试用于测试方向定义从一个Abaqus/Standard分析到另一个的传递。测试涉及在单调递增载荷作用下的单元素简单剪切。第一分析包括两个步骤，其中单元承受简单剪切载荷。第二分析从第一分析第一步结束处导入结果，并使单元承受与第一分析第二步相同的载荷。使用当前材料状态导入选项以及导入分析中参考构型更新选项来验证方向的传递。

所有测试使用的材料模型与上一节相同。

由于定义了非各向同性材料行为，因此需要局部坐标系来将各向异性行为与材料方向相关联。在原始分析中指定非默认方向，使局部材料方向相对于单元局部方向倾斜45度。采用大位移分析，导致非默认局部坐标系随着材料点处的平均刚体运动而旋转。默认情况下，方向定义会传递到导入分析。

默认情况下，应力、应变、截面力和截面应变（如果适用）都以局部坐标系报告。

还对一个具有三层的复合壳进行了验证测试。使用S4R单元。相对于局部方向定义45度的截面方向。针对材料计算，为各层定义了相对于新定义的截面方向15度、30度和45度的附加方向。用于测试的材料模型为正交弹性，如上一节所定义。

验证测试还包括一些具有增强沙漏控制的一阶减缩积分单元。

### 结果与讨论

当材料状态被导入且参考构型未更新时，两个Abaqus/Standard分析的结果相同。当材料状态被导入且参考构型被更新时，应力和材料方向相同；应变不同，因为参考构型被更新了。
