# 13.5.3 位置约束、平移和旋转之间如何产生冲突







在某些情况下，尝试应用位置约束会与现有位置约束产生冲突。如果是这种情况，Abaqus/CAE显示错误消息，您可以应用不同的位置约束或使用特征操作工具集修改现有位置约束。

类似地，尝试平移或旋转零件或模型实例可能会与现有位置约束产生冲突。如果发生冲突，Abaqus/CAE会执行以下操作：

**平移**

Abaqus/CAE仅沿未约束自由度应用平移分量。如果所有自由度都被约束，Abaqus/CAE显示错误消息且平移失败。

**旋转**

Abaqus/CAE显示错误消息且旋转失败。

如果您遇到与现有位置约束的冲突，您可以移除所有现有位置约束而不改变实例的位置，方法是使用**实例**![](../graphics/images/arrow.gif)**转换约束**。然后您可以应用新的位置约束、平移或旋转。您无法恢复已移除的位置约束。或者，您可以删除位置约束，Abaqus/CAE会将实例移回其原始位置。
