# 69.1 使用几何编辑工具集







您可以使用 Part（部件）模块中的几何编辑工具集来编辑或修复部件的几何。几何编辑工具集使您能够编辑导入的部件以提高其精度和有效性；但是，Abaqus/CAE 不要求部件完全精确。此外，您可以使用几何编辑工具集从部件中移除小特征。您可以使用几何编辑工具集编辑边、面或整个部件。["编辑技术概述，" 第 69.2 节"](pt06ch69s02.md)提供了每种类型分区可用方法的概述。Abaqus/CAE 将大多数编辑操作存储为特征。因此，您可以使用特征操作工具集通过删除或抑制相应特征来撤销编辑。

您可以通过从主菜单中选择 ****Tools**![](../graphics/images/arrow.gif)**Geometry Edit****（工具→几何编辑）并从出现的 **Geometry Edit**（几何编辑）对话框中选择所需的工具来访问编辑工具。您也可以通过 Part（部件）模块工具箱访问几何编辑工具。[图 69-1](pt06ch69s01.md#rep-toolbox) 显示了 Part（部件）模块工具箱中所有几何编辑工具的隐藏图标。

**图 69-1** 几何编辑工具集工具箱。

![](../graphics/rep-toolbox-nls.png)

要查看包含每个几何编辑工具简短描述的工具提示，请将鼠标在该工具上停留片刻。有关更多信息，请参阅["使用包含隐藏图标的工具箱和工具栏，" 第 3.3.2 节"](pt01ch03s03s02.md)。
