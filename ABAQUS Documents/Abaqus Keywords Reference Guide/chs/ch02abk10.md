# *BLOCKAGE





### *BLOCKAGE控制接触表面的阻塞。

此选项用于控制可能导致基于表面的流体腔流出的表面组合。它必须与[*SURFACE INTERACTION](ch18abk50.md)选项配合使用。

**产品：**Abaqus/Explicit

**类型：**模型或历史数据

**级别：**模型、步骤
