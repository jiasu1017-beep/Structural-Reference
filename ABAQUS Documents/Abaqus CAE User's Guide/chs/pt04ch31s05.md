# 31.5 管理裂缝





**Crack Manager**允许您创建和管理裂缝。管理器包含您已定义的裂缝名称和类型的列表。管理器中的**Create**、**Edit**、**Copy**、**Rename**和**Delete**按钮允许您创建新裂缝或编辑、复制、重命名和删除现有裂缝。管理器左侧列中的图标允许您抑制和恢复现有裂缝。您还可以使用Interaction模块主菜单栏中的****Special**![](../graphics/images/arrow.gif)**Crack****菜单启动这些过程。选择主菜单栏中的管理操作后，过程与在管理器对话框中单击相应按钮完全相同。
