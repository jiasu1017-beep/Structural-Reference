### 概述

接触属性：
- 定义了当表面处于接触状态时控制其行为的机械和热表面相互作用模型；以及
- 分配给各个接触对。

### 为接触对分配表面相互作用定义

表面相互作用定义指定接触对使用的本构接触属性和约束 enforcement 方法。模型中的每个接触对必须引用一个表面相互作用定义，即使接触对使用默认接触属性模型。见 ["机械接触属性概述，" 第 37.1.1 节](pt09ch37s01aus165.md)，了解定义接触属性的信息。可以在表面相互作用定义中指定非默认约束 enforcement 方法，如 ["Abaqus/Standard 中的接触约束 enforcement 方法，" 第 38.1.2 节](pt09ch38s01aus178.md) 中所述。

多个接触对可以引用相同的表面相互作用定义。

表面相互作用定义还用于在以下情况下提供表面垂直向外厚度的非默认值：
- 二维模型，或
- 涉及基于节点的表面的接触对，以在基于节点的表面中的每个节点提供非默认横截面积。

Abaqus/Standard 使用表面相互作用定义中提供的垂直向外表面厚度和横截面积来计算接触应力。

| **输入文件用法：** | 使用以下两个选项： |
| --- | --- |
|  | ``` [*CONTACT PAIR](../key/key-link.md#usb-kws-hcontactpair), INTERACTION=*interaction_property_name* [*SURFACE INTERACTION](../key/key-link.md#usb-kws-hsurfaceinteraction), NAME=*interaction_property_name* ``` |

| **Abaqus/CAE 用法：** | Interaction 模块： |
| --- | --- |
|  | **Create Interaction Property**: **Name:** *interaction_property_name*, **Contact** Interaction 编辑器： **Contact interaction property**: *interaction_property_name* |
