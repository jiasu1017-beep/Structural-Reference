# 47.2 AutoColors 对象





AutoColors 对象定义颜色调色板。

**访问**

```
session.autoColors
```

### 47.2.1 setValues(...)

此方法更改颜色调色板。

**必要参数**

*colors*

String 序列，指定调色板的颜色。字符串必须是命名颜色或代表红、绿、蓝成分的十六进制形式。例如，红色为 '#FF0000'，绿色为 '#00FF00'，蓝色为 '#0000FF'

**可选参数**

无。

**返回值**

无

**异常**

无。

### 47.2.2 成员

AutoColors 对象的成员与 [setValues](pt01ch47pyo02.md#ker-autocolors-setvalues-pyc) 方法的参数具有相同的名称和描述。
