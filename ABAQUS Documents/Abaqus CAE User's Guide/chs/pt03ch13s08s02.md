# 13.8.2 在装配模块中操作特征





除了基准几何和划分外，以下内容被视为装配的特征，并出现在模型树中的特征列表中：

**零件实例**

您可以抑制、恢复和删除零件实例。您可以划分零件实例，但不能编辑其形状或其特征。要修改零件实例，您必须在零件模块中编辑原始零件；当你返回装配模块时，Abaqus/CAE会自动重新生成修改后零件的实例。

你可以通过从主菜单栏选择****View**![](../graphics/images/arrow.gif)**Assembly Display Options**![](../graphics/images/arrow.gif)**Instance****使零件实例不可见，同时仍将其保留在装配中。更多信息，请参见["控制实例可见性，"第76.14节](pt07ch76hla13.md)。此技术与抑制零件实例不同；被抑制的零件实例会从装配中移除，直到你恢复它。你也可以使用显示组使零件实例不可见；更多信息，请参见[第78章"使用显示组显示模型的子集"](pt07ch78.md)。

您可以链接零件实例，并且可以排除零件实例进行分析；更多信息，请参见["在模型之间链接零件实例，"第13.3.5节](pt03ch13s03s05.md)和["从分析中排除零件实例，"第13.3.6节](pt03ch13s03s06.md)。

**模型实例**

您可以抑制、恢复和删除模型实例。要修改模型实例，你必须编辑原始模型的装配。

你可以通过从主菜单栏选择****View**![](../graphics/images/arrow.gif)**Assembly Display Options**![](../graphics/images/arrow.gif)**Instance****使模型实例不可见，同时仍将其保留在装配中。你也可以使用显示组使模型实例不可见。

**位置约束**

您可以编辑、抑制、恢复和删除位置约束。您可以修改位置约束的以下参数：
- 箭头法向方向（垂直于所选曲面）或沿可动零件实例的所选边缘方向。
- 可动零件实例的所选曲面或边缘与固定零件实例的所选曲面或边缘之间的间隙。间隙参数仅适用于面对面、边缘对边缘和接触约束。

平移和旋转不存储为特征，不能编辑、抑制、恢复或删除。

您可以使用特征操作工具集来修改装配的特征。当系统提示你选择要修改的特征时，你可以从视口选择一个可见的特征，如零件实例、基准或划分。但是，要选择位置约束，必须从模型树中选择它。

以下特征操作工具可从特征操作工具集中使用：

**编辑**

当你编辑特征时，Abaqus/CAE会显示**编辑特征**对话框，你可以修改特征的参数或定义该特征的草图。你不能编辑零件实例；你必须返回零件模块来修改原始零件。

**重新生成**

当你在复杂装配中修改特征时，延迟重新生成直到你完成所有更改可能会很方便，因为重新生成可能非常耗时。当你准备好重新生成装配时，选择****Feature**![](../graphics/images/arrow.gif)**Regenerate****。

**重命名**

重命名特征。

**抑制**

抑制特征会将其从装配定义中临时移除。被抑制的特征不可见，不能被网格划分，也不包含在模型分析中。抑制父特征将抑制其所有子特征。

**恢复**

恢复特征会将抑制的特征恢复到装配中。你可以选择恢复所有特征、最近抑制的特征集，或仅恢复选定特征。

**删除**

删除特征会将其从装配中移除；你不能恢复已删除的特征。

**查询**

当你查询特征时，Abaqus/CAE会在消息区域显示信息，并将相同信息以注释形式写入回放文件（`*abaqus*.rpy`）。

**选项**

**特征选项**对话框允许你控制Abaqus/CAE是否执行自相交检查，并允许你优先重新生成约束特征而不是其他装配特征。

有关特征操作工具集的更详细说明，请参见[第65章"特征操作工具集"](pt06ch65.md)。
