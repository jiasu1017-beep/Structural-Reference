# 39.1.1 在模型数据库中创建子结构





您可以通过以下一般步骤在 Abaqus/CAE 中创建子结构：

****

1. 创建或打开要在 Abaqus/CAE 中指定子结构的模型数据库。
2. 在 Step 模块中，创建**子结构生成**步骤。Abaqus/CAE 会将整个模型转换为单个子结构。有关更多信息，请参见["生成子结构，"第39.2节](pt04ch39s02.md)。
3. 在 Load 模块中，创建**保留节点自由度**边界条件，以确定将作为子结构外部自由度保留哪些自由度。如果您想在子结构生成步骤中定义一个负载情况（如果您想将负载应用于子结构上保留自由度之外的位置），您也可以定义负载情况。有关更多信息，请参见["为子结构指定保留节点自由度和负载情况，"第39.3节](pt04ch39s03.md)。
4. 在 Job 模块中，创建新作业并提交分析。当您对包含子结构数据的装配执行分析时，Abaqus/CAE 会为每个子结构零件实例创建单独的输出数据库，并且不会将子结构零件实例的结果包含在装配的输出数据库中。Visualization 模块提供了使您能够将子结构组件的结果整合回装配结果中的工具；有关更多信息，请参见["可视化子结构输出，"第39.8节](pt04ch39s08.md)。
