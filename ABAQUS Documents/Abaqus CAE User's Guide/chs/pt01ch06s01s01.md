# 6.1.1 你能从视口中选择哪些对象？





在建模过程中，在当前视口中选择对象是你需要执行的最常见任务之一。在各种过程中，你可能需要通过直接从视口中拾取来选择几何对象（顶点、边、面、单元体和基准）或离散对象（节点和单元）。[图 6-1](pt01ch06s01s01.md#pic-entities) 显示了这些不同的对象类型。

**图 6-1** 你可以选择的对象类型。

![](../graphics/pic-entities-nls.png)

你可以在某些过程中在视口中选择对象，如下所列：
- 创建集和面
- 分割部件实例
- 编辑特征
- 为网格划分对部件实例播种
- 创建或编辑由单元或节点组成的显示组
- 对模型中的单元进行颜色编码
- 通过模型创建节点列表路径
- 创建载荷

你也可以在选择过程之前在视口中预先选择对象。如果你在做选择过程之前进行选择，Abaqus/CAE 不会限制你的选择。当你选择过程时，Abaqus/CAE 会过滤你做的任何选择，只保留适合该过程的选择。有关更多信息，请参阅["在选择过程之前选择对象，" 第 6.3.7 节](pt01ch06s03s06.md)。

如果你作为过程的一部分选择对象，在大多数情况下，Abaqus/CAE 只允许你选择适合当前过程的对象。例如，分割边的第一步是选择感兴趣的边。因此，在过程的这一步，你只能选择一条边；你不能选择单元体、面或顶点。提示区域中的消息会引导你完成过程的步骤，并指示哪些类型的对象可用于选择。你只能选择属于当前显示组的对象。

在某些情况下，Abaqus/CAE 无法确定哪些对象适合选择，因此不会限制你的选择。例如，当你创建集时，你可以从单元体、面、边和顶点中选择要包含在集中的对象，Abaqus/CAE 允许你选择任何这些对象。如果你在过程中从视口进行了模糊选择，Abaqus/CAE 允许你循环浏览可用对象，直到选择所需对象。这种模糊性在["循环浏览有效选择，" 第 6.2.10 节](pt01ch06s02hlb10.md)中有描述。你可能会发现使用选择过滤器来限制你可以选择的对象类型更容易。有关更多信息，请参阅["使用选择选项，" 第 6.3 节](pt01ch06s03.md)。

许多定义属性的过程（相互作用、约束、载荷、边界条件、预定义场和工程特征）允许你从视口中选择对象以标识要应用属性的区域。这些过程的默认行为是创建包含所选对象的集或面。你可以通过在提示区域中切换关闭创建集或面的选项来更改此行为。提示区域中提供了默认名称，但你可以输入新名称。
