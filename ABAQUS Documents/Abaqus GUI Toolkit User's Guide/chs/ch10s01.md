# 10.1 在 Abaqus 中定义材料







您可以在模拟中使用任意数量的不同材料。每个材料定义以 [*MATERIAL*](../key/key-link.md#usb-kws-mmaterial) 选项开始。NAME 参数标识与正在定义的材料关联的名称。此名称用于将材料定义分配给模型中的特定单元。

材料定义是 Abaqus 输入文件中选项块位置重要的少数情况之一。所有定义材料特定方面的选项块（如其弹性模量或密度）都必须直接跟在 [*MATERIAL*](../key/key-link.md#usb-kws-mmaterial) 选项之后。此外，定义特定材料行为的材料选项块不能被其他非材料选项块中断。如果 Abaqus 无法将材料行为选项块（如 [*ELASTIC*](../key/key-link.md#usb-kws-melastic)）与之前的 [*MATERIAL*](../key/key-link.md#usb-kws-mmaterial) 选项关联，它会发出错误消息。
