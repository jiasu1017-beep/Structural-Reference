# *MASS ADJUST









### *MASS ADJUST调整和/或重新分配单元集的质量。

此选项用于均匀地增加或减少单元集中单元的质量，以使该集合的总质量与指定值匹配。它也可以用于重新分配单元集中单元之间的质量，以将该集合的最小稳定时间增量提高到指定的目标值。可以为多个单元集调整质量；质量按指定单元集的顺序进行调整。此行为会影响共享单元的单元集的最终结果。此选项在分析中只能出现一次。

**产品：**Abaqus/Explicit  

**类型：**模型数据  

**级别：**装配  

##### **参考：**

- ["调整和/或重新分配单元集的质量，" Abaqus Analysis User's Guide第2.6.1节](../usb/usb-link.md#usb-int-amassadjust)

### **可选参数：**

TARGET DT

将此参数设置为所有质量调整单元集的默认最小单元稳定时间增量。如有需要，可以为选定的集合指定个体化的目标时间增量。非零目标值会导致单元集内的质量重新分配。如果省略此参数，则仅对在数据行上指定了最小单元稳定时间增量值的单元集进行质量重新分配。

### **调整和/或重新分配单元集质量的数据行：**

**第一行：**

根据需要重复此数据行以为不同的单元集调整质量。不能多次输入相同的单元集标签。质量按数据行上指定单元集的顺序进行调整。
