# 16.5.2 单箭头和双箭头代表什么？







在许多情况下，Abaqus/CAE使用箭头在视口中代表规定条件。这些箭头代表规定条件的每个分量（除了流体边界条件，在这种情况下，箭头代表合成方向）。例如，图16-8中出现的箭头代表施加到两个顶点的集中力的三个分量。

**图16-8** 具有三个分量的集中力。

![](../graphics/lbi-gforce.png)

具有单个箭头头的箭头代表施加到平移自由度的规定条件的分量。例如，图16-8中集中力的三个分量施加到自由度1到3；因此，图中的每个箭头都有一个箭头头。

当规定条件的分量施加到旋转自由度时，该分量显示为双箭头头。图16-9中的箭头表示**速度/角速度**边界条件施加到顶点的自由度4和6。

**图16-9** 施加到旋转自由度的边界条件。

![](../graphics/lbi-gbc.png)

双箭头头的放大视图如图16-10所示。

**图16-10** 放大的双箭头头。

![](../graphics/lbi-gbc2.png)

如果您将规定条件同时施加到平移和旋转自由度，则会出现单箭头头和双箭头头。例如，**速度/角速度**边界条件施加到图16-11中顶点自由度1、3、4和6。

**图16-11** 施加到平移和旋转自由度的边界条件的放大视图。

![](../graphics/lbi-gbc3.png)

在此图中，单箭头头是沙棕色的，表示顶点自由度1和3被固定。双箭头头是洋红色的，直接出现在单箭头头后面；双箭头头表示顶点自由度4和6被固定。

有关箭头颜色的信息，请参见"理解规定条件符号类型、颜色和大小"，第16.5.1节。有关何时期望箭头指向或远离区域的信息，请参见"理解符号位置和方向"，第16.5.3节。
