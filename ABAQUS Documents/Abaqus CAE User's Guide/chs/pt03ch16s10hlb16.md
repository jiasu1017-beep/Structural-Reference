# 16.10.15 定义流体腔压力边界条件





您可以创建流体腔压力边界条件来控制整个分析过程中腔内的压力。流体腔压力与流体腔相互作用相关。它用于确定流体交换相互作用的流量。您可以通过为步骤分配新值和指定一个幅值来定义从前一个值开始的变化率，从而在分析过程中改变流体腔压力。（有关更多信息，请参见《Abaqus Analysis User's Guide》第34.2.1节"Abaqus/Standard和Abaqus/Explicit中的初始条件"。）

**要指定流体腔压力：**

1. 使用以下方法之一显示流体腔压力边界条件编辑器：- 要创建新的流体腔压力边界条件，请按照"创建边界条件"中的步骤操作（第16.8.2节）（**类别**：`Other`；**所选步骤的类型**：`Fluid cavity pressure`）。- 要使用菜单或管理器编辑现有流体腔压力边界条件，请参见"编辑依赖于步骤的对象"（第3.4.13节）。
2. 如果编辑器顶部出现**Method**字段，请单击字段右侧的箭头，然后选择以下之一：- 如果要指定流体腔压力值，请选择**Specify pressure**。- 如果要将流体腔压力大小固定为前一个通用步骤的最终值，请选择**Fixed at current pressure**。仅当所选步骤两种方法都有效时，编辑器中才会显示**Method**选项。
3. 如果出现**Fluid cavity interaction**字段，请单击字段右侧的箭头，然后选择压力适用的相互作用。**Fluid cavity interaction**选项仅在您创建新边界条件或在创建边界条件的步骤中编辑流体压力边界条件时才会出现。在传播步骤中编辑边界条件时，无法更改相互作用。
4. 如果出现**Fluid cavity pressure**字段，请为当前步骤输入值。
5. 如果适用，单击**Amplitude**字段右侧的箭头，然后从出现的列表中选择所需的幅值。或者，您可以单击![](../graphics/ico_amplitudeSmall.png)创建新的幅值。（有关更多信息，请参见第57章"幅值工具集"。）**注意：**如果您在初始步骤中创建边界条件（此时大小始终为零），或者将大小固定为前一个通用步骤的最终值（例如在编辑频率步骤时），则**Amplitude**字段不可用。
6. 单击**OK**保存数据并退出编辑器。
