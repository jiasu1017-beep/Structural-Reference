# 17.11.3 自底向上网格方法





每种自底向上方法的控制参数与Abaqus/CAE在内部用于创建类似的自顶向下扫掠网格的三维实体的参数相似。自底向上方法及相关参数定义如下：

**扫掠**

扫掠方法通过沿扫掠路径移动二维网格来创建三维网格。扫掠方法如图17-95所示。当区域的横截面在起始侧和终止侧之间发生变化时，您应该使用自底向上扫掠网格方法。要使用扫掠方法，您必须首先选择定义Abaqus/CAE将创建二维网格的面或面的组合的**源侧**。源侧可以是几何面、单元面和二维单元的任意组合。您可以通过选择定义所需扫掠区域边的**连接边**来定义扫掠路径。如果您定义连接边，则网格会沿所选边紧密符合几何体或网格。或者，对于几何体，您可以选择**目标侧**并指定**层数**，并允许Abaqus/CAE通过在源侧和目标侧之间插值来创建扫掠路径。**目标侧**是Abaqus/CAE用于结束网格的单个面。层数是指将放置在源侧和目标侧之间的单元层数——如果您使用连接边，则连接边的二维网格定义单元层数。Abaqus/CAE将二维网格从源侧扫掠到实体区域的体积中以创建网格。

**挤出**

挤出方法是扫掠方法的一种特殊情况，具有由方向和距离定义的线性路径。挤出方法如图17-96所示。您应该对具有恒定横截面和线性扫掠路径的区域使用挤出方法。自底向上挤出网格有三个必需参数。与扫掠方法一样，您选择定义Abaqus/CAE将创建二维网格的区域的**源侧**。然后您选择定义挤出方向并可用于定义挤出距离的**向量**的起点和终点。最后，您指示**层数**以定义源侧与挤出网格末端之间的单元数量。如果您使用向量定义挤出距离，则定义完成。但是，您可以**指定**距离或使用**投影到目标**并选择目标侧来定义挤出距离。目标侧可以从视口中的任何几何体、网格或基准平面中选择；它不必与源侧属于同一零件实例。Abaqus/CAE沿挤出向量的方向从源侧挤出二维网格。如果您选择目标侧来定义挤出距离，Abaqus/CAE会在目标侧结束挤出网格。图17-97左侧显示了源侧和目标侧；挤出向量（未显示）从矩形源侧的中心延伸到圆柱体的中心。生成的挤出网格是源侧网格的延伸。它紧密匹配目标侧形状，但不会尝试匹配目标侧上网格节点的位置。

**图17-97** 可选的目标侧（白色）用于定义挤出距离。

![](../graphics/mgn-bu-extrudetarget.png)

**偏差比**参数定义了在创建多层挤出自底向上网格中，源侧与挤出网格末端之间单元厚度的变化。偏差比是挤出网格中第一层单元厚度与最后一层单元厚度的比值。默认偏差比`1.0`表示整个挤出距离内单元厚度相等。

**旋转**

旋转方法是扫掠方法的另一种特殊情况。在这种情况下，扫掠路径是由轴线和旋转角度定义的圆形路径。旋转方法如图17-98所示。您应该对具有恒定横截面和圆形扫掠路径的区域使用此方法。与扫掠和挤出方法一样，您选择定义Abaqus/CAE将创建二维网格的区域的**源侧**。源侧不能与旋转轴相交，但可以包含与轴重合的边。如果是这样，接触轴的单元会在旋转网格中创建一层楔形单元。源侧不应在旋转轴上包含任何三角形单元。然后您选择定义旋转轴的**轴线**的起点和终点。最后，您指示**角度**和**层数**以定义源侧与旋转网格末端之间的单元数量。Abaqus/CAE将二维网格从源侧按指定角度旋转，并均匀地将结果区域分割成所需数量的单元层。如果您从起点到终点沿旋转轴看，旋转方向是顺时针的。

**偏移**

偏移方法的工作方式与"编辑网格"工具集中的**偏移（创建实体层）**相同；它通过偏移选定单元来创建一层或多层实体单元。偏移仅在您使用孤立单元工作时可用。要创建偏移自底向上网格，您输入偏移单元的总厚度和所需的单元层数。您可以创建单元集或扩展现有集，但不能像使用"编辑网格"工具集那样创建顶面和底面。如果您选择壳单元作为源，则必须在提示区域中指示所需的偏移方向；您可以双向偏移壳单元。如果您的壳单元选择包含尖角，请切换**在角处保持恒定厚度**以在角处元素相遇的地方与选择其余部分保持相同的总厚度。使用此选项可以减少单元畸变并防止单元崩溃，特别是如果单元偏移到角的内侧（更多信息，请参见["在网格偏移期间减少单元畸变和崩溃，"第64.3.3节](pt06ch64s03s03.md)）。
