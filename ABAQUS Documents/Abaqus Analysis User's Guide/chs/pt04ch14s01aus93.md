**产品：** Abaqus/Explicit

##### **参考文献**

### 概述

自适应网格细化功能：
- 可以在欧拉网格内部局部细化单元；
- 允许用户定义各种细化准则；
- 可以自动移除细化一旦细化准则不再满足；以及
- 仅适用于欧拉单元类型 EC3D8R。

### 自适应网格细化

在传统欧拉分析中，欧拉网格的拓扑在分析过程中不会改变。尽管欧拉网格运动功能允许欧拉网格在空间中移动以覆盖感兴趣区域，但其创建随时间变化的非均匀细化网格的能力有限。自适应网格细化功能可以通过细分由用户定义准则标识的单元来局部细化网格。一旦准则不再满足，可以自动移除细化。与均匀细化网格相比，此功能可以大大节省计算成本。请参阅 ["铜棒的冲击," Abaqus 基准指南第 1.3.10 节](../bmk/bmk-link.md#bmk-anl-rodimpact)，了解使用自适应网格细化功能的示例。

### 激活自适应网格细化

您可以独立地为模型中的每个欧拉截面激活自适应网格细化。该功能适用于元素集中指定的所有单元；元素集中的所有单元必须位于同一欧拉截面中。

| **输入文件用法：** | ``` [*ADAPTIVE MESH REFINEMENT](../key/key-link.md#usb-kws-hadaptivemeshrefinement), ELSET=*name* ``` |
| --- | --- |

### 设置细化限制

当发生自适应网格细化时，会向欧拉网格添加单元。您可以通过指定添加单元与原始单元的比率上限来限制可以创建的单元数量。此上限比率的默认值为 8.0。

| **输入文件用法：** | ``` [*ADAPTIVE MESH REFINEMENT](../key/key-link.md#usb-kws-hadaptivemeshrefinement), RATIO=*maximum increase in number of elements/original number of elements* ``` |
| --- | --- |

### 设置细化级别

一级细化时，每次细化用户定义的欧拉单元时，它被等分为八个子单元。如果允许两级细化，这些子单元随后可以再次细分。您可以设置最大细化级别数的限制。默认最大级别为一。

| **输入文件用法：** | ``` [*ADAPTIVE MESH REFINEMENT](../key/key-link.md#usb-kws-hadaptivemeshrefinement), LEVEL=*maximum level of refinement* ``` |
| --- | --- |

### 关闭粗化

您可以指定一旦细化准则不再满足是否移除细化。

| **输入文件用法：** | ``` 使用以下选项指定一旦细化准则不再满足，可以移除细化： [*ADAPTIVE MESH REFINEMENT](../key/key-link.md#usb-kws-hadaptivemeshrefinement), COARSENING=YES (default) 使用以下选项指定即使细化准则不再满足，也不能移除细化： [*ADAPTIVE MESH REFINEMENT](../key/key-link.md#usb-kws-hadaptivemeshrefinement), COARSENING=NO ``` |
| --- | --- |

### 定义细化准则

您必须指定至少一个细化准则。如果满足任何准则，将选择元素进行细化。为了减少网格过渡边界（细网格与粗网格相遇的地方）处的数值伪影，也会细化与选定元素相邻的元素。一旦细化准则不再满足，元素就会被粗化。每个选定的元素在每个增量中只能细化或粗化一级。[表 14.1.4--1](pt04ch14s01aus93.md#refinementcriterialabels) 列出了 Abaqus/Explicit 中所有可用的细化准则。

**表 14.1.4–1** 细化准则。
| 细化准则描述 | 细化准则标签 | 用户指定的值 |
| --- | --- | --- |
| 细化包含材料界面的单元 | VF | 不适用 |
| 细化与拉格朗日体接触的单元 | CONT | 您可以指定值 ALL 来细化所有与拉格朗日表面相交的元素，即使接触尚未发生；使用此选项可避免与跳动接触相关的频繁细化和粗化。您还可以指定值 MAT 来仅细化包含与拉格朗日表面接触材料的单元。如果未指定值，则除了具有 Mie-Grüneisen 状态方程的材料外，将使用 MAT。 |
| 细化发生显著塑性变形的单元。不支持临界状态（粘土）塑性模型。 | PEEQ | 等效塑性应变的临界值 |
| 细化在锐利密度梯度附近的单元 | DENSITY | 您可以为该准则指定两个值。第一个值是密度梯度的临界值，计算为跨元素面的密度变化与元素内材料密度的比率；第二个值是临界密度。要选择元素，密度和密度梯度都必须超过临界值。 |
| 细化在锐利压力梯度附近的单元 | PRESS | 您可以为该准则指定两个值。第一个值是压力梯度的临界值，计算为跨元素面的压力变化与元素内材料压力的比率；第二个值是临界压力。要选择元素，压力和压力梯度都必须超过临界值。 |

| **输入文件用法：** | ``` [*ADAPTIVE MESH REFINEMENT](../key/key-link.md#usb-kws-hadaptivemeshrefinement), *refinement criteria label*,* value of the criteria* ``` |
| --- | --- |

### 接触

当在涉及通用接触的欧拉截面中指定自适应网格细化时，默认激活动态播种。动态播种允许在靠近拉格朗日面的欧拉单元细化后创建更多种子；一旦这些欧拉单元被粗化，这些种子将被删除。

对于没有自适应网格细化的欧拉分析，动态播种也可能有用。

| **输入文件用法：** | ``` [*CONTACT CONTROLS ASSIGNMENT](../key/key-link.md#usb-kws-hcontcntrlassign), SEEDING=DYNAMIC ``` |
| --- | --- |
