# 42.4.2 更改步骤的周期





步骤周期值的含义取决于该步骤是Riks分析步骤还是时间相关步骤。对于Riks分析步骤，周期指定步骤的总弧长。对于时间相关步骤，周期指定步骤的总持续时间，因此更改此值会影响此步骤的显示时间以及所有后续步骤的起始时间。更改单个步骤的时间周期也会更改整个动画或逐帧显示的持续时间，影响时间相关动画的自动计算起始时间和结束时间，并影响此模型动画与其他模型的同步方式。如果您想同步不同模型中发生在公共时间线上不同时间的事件，可能需要调整时间周期。

周期值的更改会持续到会话结束。Abaqus/CAE不会将这些更改保存到输出数据库。

如果Riks分析或时间相关步骤的周期值等于0，则无法编辑该值。此外，模态和频率提取步骤没有周期值；Abaqus/CAE在这些类型步骤的**Period**列中显示不可编辑的破折号（`'-'`）。

**提示：**如果要将周期值还原为输出数据库的设置，请点击窗口下半部分的鼠标按钮3，然后选择**Reset Periods**。

**更改步骤的周期：**

1. 找到**Active Steps/Frames**对话框。从主菜单栏，选择****Result**![](../graphics/images/arrow.gif)**Active Steps/Frames****。**Active Steps/Frames**对话框出现。对话框的下半部分显示了输出数据库中的步骤，对于展开的步骤还显示了帧。在会话期间首次检查输出数据库的步骤和帧时，所有步骤和帧都处于活动状态。
2. 从对话框下半部分，点击步骤的**Period**列中的值，然后在字段中输入新值。为每个要自定义的步骤周期重复此更改。
3. 点击**Apply**。Abaqus/CAE使用修订后的周期和时间值更新所有显示此输出数据库的视口。
