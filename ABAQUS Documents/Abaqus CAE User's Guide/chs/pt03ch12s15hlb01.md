# 12.15.1 分配截面





您可以通过首先创建截面然后选择****Assign**![](../graphics/images/arrow.gif)**Section****将截面属性分配给部件；部件的区域，包括其蒙皮或桁条；或元素集。您分配给部件的截面属性自动分配给该部件在装配中的所有实例。Abaqus/CAE在Property模块中将区域着色为绿色以表示该区域具有截面分配；如果存在重叠的截面分配，则将区域着色为黄色。如果您删除截面分配中使用的截面，Abaqus/CAE将区域着色为红色以表示未找到截面。

**注意：**分配的顺序可能是相关的；当截面分配重叠时，最后的分配将具有优先级。

您也可以使用**Section Assignment Manager**来查看、创建、编辑、抑制、恢复和删除截面分配。请参见["管理截面分配，"第12.15.2节"](pt03ch12s15hlb02.md)。您可以使用查询工具集来验证您已为所选区域分配了正确的截面。请参见["使用查询工具集获取分配信息，"第12.19节"](pt03ch12hla03.md)。

您可以在模型中任何地方创建重叠的截面分配，但包含欧拉部件的区域除外。欧拉部件只能分配单个截面分配，即使其原始截面分配被抑制，您也不能对欧拉部件应用第二个截面分配。

从输入文件导入网格时，与该网格关联的某些截面属性也可能被导入；在这些情况下，可能不需要为部件分配截面属性。请参见["从Abaqus输入文件导入模型，"第10.5.2节"](pt02ch10s05s02.md)。

**为部件分配截面：**

1. 如果您要分配截面的部件在当前视口中不可见，请在上下文栏中的**Part**列表中点击所需部件的名称。您选择的部件出现在当前视口中。
2. 从主菜单栏，选择****Assign**![](../graphics/images/arrow.gif)**Section****。
   **提示：**您也可以在**Section Assignment Manager**中点击**Create**，或在Property模块工具箱中选择![](../graphics/ico_sectionAssign.png)工具。
3. 从视口中选择部件的区域，然后点击鼠标按钮2以表示您已完成选择。（有关更多信息，请参见[第6章，"在视口中选择对象"](pt01ch06.md)。"）如果欧拉部件在当前视口中，Abaqus/CAE自动选择整个部件。
   **提示：**您可以通过使用**Selection**工具栏中的工具来限制可以在视口中选择的 对象类型。请参见["使用选择选项，"第6.3节"](pt01ch06s03.md)，了解更多。
   如果您选择**Skins**作为对象类型，可以将截面分配给整个蒙皮或其一个或多个面。从提示区域，选择**（pick entire skin）**或**（pick partial skin）**，然后在视口中进行选择。如果所选部件有多个蒙皮，Abaqus/CAE在您进行选择时会在提示区域显示模糊选择选项。
   如果您选择**Stringers**作为对象类型，可以将截面分配给整个桁条或其一个或多个边。从提示区域，选择**（pick entire stringer）**或**（pick partial stringer）**，然后在视口中进行选择。如果所选部件有多个桁条，Abaqus/CAE在您进行选择时会在提示区域显示模糊选择选项。
   如果您希望从现有集的列表中进行选择，请执行以下操作：
   1. 点击提示区域右侧的**Sets**。Abaqus/CAE显示**Region Selection**对话框，其中包含可用部件集和元素集的列表。
   2. 选择所需的集，然后点击**Continue**。
   **注意：**默认选择方法基于您最近使用的选择方法。要切换到另一种方法，请点击提示区域右侧的按钮——**Select in Viewport**或**Sets**。
   出现**Edit Section Assignment**对话框。此对话框包含可分配给所选区域或集的现有截面列表。例如，如果您选择了实体区域，任何现有实体截面会出现在**Edit Section Assignment**对话框中。此外，对话框包含**Create Section**按钮以及显示的截面的截面类型、材料和区域。
4. 在**Edit Section Assignment**对话框中，选择感兴趣的截面，然后点击**OK**。Abaqus/CAE将所选截面分配给部件或集，并将所选区域着色为绿色以表示该区域具有截面分配。如果区域上存在重叠的截面分配，Abaqus/CAE将区域着色为黄色。
   **注意：**分配的顺序可能是相关的；当截面分配重叠时，最后的分配将具有优先级。
5. 分配截面厚度：
   **From section** 使用截面定义中定义的厚度。
   **From geometry** 使用几何截面的厚度。
6. 如果您将均匀或复合截面分配给壳，您可以定义用于**Shell Offset**的方法。点击**Definition**字段右侧的箭头，并从出现的列表中选择您选择的选项：
   - 选择**Middle surface**、**Top surface**或**Bottom surface**来表示参考表面。
   - 选择**Specify value**，并在**Offset ratio**字段中输入从中间表面到壳参考表面的正或负距离（作为壳厚度的分数）。正偏移将生成更靠近壳顶面的节点和元素，负偏移将生成更靠近底面的节点和元素。大于`0.5`的偏移将导致节点超出壳的表面。
   - 选择定义在截面上空间变化的偏移的标量离散场。Abaqus/CAE仅允许您选择有效的离散场，对于偏移，这些是应用于元素的标量离散场。此选项仅对Abaqus/Standard分析有效。或者，您可以点击![](../graphics/ico_discreteField.png)创建新的离散场。（请参见[第63章，"离散场工具集"](pt06ch63.md)，了解更多。）
   - 选择**From geometry**以让Abaqus/CAE根据几何上定义的厚度计算偏移。如果在**Thickness**字段中选择了**From geometry**，则此选项是默认的。您不能使用此选项在Abaqus/Explicit分析中计算偏移。
7. 如果您要将截面分配给其他区域，请重复步骤3和4。完成截面分配后，使用以下方法之一退出截面分配模式：
   - 如果您正在从视口中选择部件的区域，请点击鼠标按钮2或点击提示区域中的**Done**。
   - 如果您正在从**Region Selection**对话框中选择预定义集，请点击**Cancel**关闭对话框，然后点击鼠标按钮2或点击提示区域中的**Done**。
