# 55.2.3 选择整体填充颜色





填充颜色是Abaqus/CAE显示模型面的颜色。此颜色是曲线图状态无关的;也就是说,您为所有未变形、变形、符号和材料取向曲线图设置一次。您也可以设置Abaqus/CAE在变形形状叠加在未变形形状上时将应用于未变形形状的填充颜色。填充颜色仅适用于填充和光源着色[渲染样式](pt05ch55s02hlb01.md)曲线图;它不适用于云图曲线图。

您可以为整个模型选择一种显示颜色,或者可以选择为特定项目(如特定单元)覆盖此显示颜色。例如,您可以用绿色显示模型,而用红色显示一组单元。有关单个项目着色的更多信息,请参阅 ["自定义单个对象的显示颜色," 第77.8节](pt07ch77s08.md)。

**选择模型填充颜色的步骤:**

1. 找到公共或叠加填充颜色选项。公共填充颜色始终应用于变形形状;当在任何曲线图状态(未变形、变形、符号或材料取向)中单独绘制未变形形状时,它也适用于未变形形状。当未变形形状与变形形状一起绘制时,应用叠加填充颜色。从主菜单栏,选择 ****Options**![](../graphics/images/arrow.gif)**Common**** 或 **Superimpose**;然后在出现的对话框中点击"**Color & Style**"选项卡。"**Fill color in filled/shaded plots**"选项在"**Color & Style**"页面中间。
2. 选择填充颜色: 1. 点击"**Fill color in filled/shaded plots**"颜色样本 ![](../graphics/usi-colorsample.png)。Abaqus/CAE显示"选择颜色"对话框。 2. 使用"选择颜色"对话框中的方法之一来选择新颜色。更多信息,请参阅 ["自定义颜色," 第3.2.9节](pt01ch03s02s09.md)。 3. 点击"**OK**"关闭"选择颜色"对话框。颜色样本变为所选颜色。在您点击"**Plot Options**"对话框中的"**OK**"或"**Apply**"之前,新颜色不会出现在视口中。
3. 切换"**Allow color code selections to override options in this dialog**"关闭以使用上面指定的填充颜色显示所有单元。将其打开以使用单个项目颜色选择覆盖此颜色。默认是打开。更多信息,请参阅 ["自定义单个对象的显示颜色," 第77.8节](pt07ch77s08.md)。
4. 点击"**Apply**"实施您的更改。填充颜色会根据您的规格进行更改。默认情况下,您的更改会在会话期间保存,并会影响当前视口和从当前视口创建的任何新视口中的所有后续曲线图。如果要为后续会话保留您应用的更改,请将它们保存到文件。更多信息,请参阅 ["为后续会话保存自定义," 第55.1.1节](pt05ch55s01s01.md)。
