# 55.8 自定义显示体的外观





显示体是不参与分析但在后处理过程中可见的部件实例。您可以使用可视化模块中的**显示体选项**来自定义模型中此类部件实例的外观。显示体的可用选项与Abaqus/CAE中所有绘图状态的可用选项类似：渲染样式、边缘可见性、颜色和样式、填充颜色、缩放和透明度都可以自定义。与通用绘图选项类似，显示体选项与绘图状态无关；即应用于显示体的选项会在所有绘图状态中反映。显示体选项适用于模型中的所有显示体；要自定义单个显示体的外观，您可以基于部件实例创建显示组（请参阅["创建或编辑显示组，" 第78.2.1节](pt07ch78s02hlb01.md)）。也可以将单个项目颜色应用于单个显示体部件实例，以覆盖作为通用显示体选项指定的边缘和/或填充颜色；有关更多信息，请参阅[第77章，"几何和网格元素的颜色编码"](pt07ch77.md)。

要找到显示体选项，请从主菜单栏选择****Options**![](../graphics/images/arrow.gif)**Display Body****。单击以下选项卡来自定义当前视口中所有显示体的外观：
- **Basic**：选择渲染样式（["选择渲染样式，" 第55.2.1节](pt05ch55s02hlb01.md)）和边缘可见性（["控制元素和表面边缘可见性，" 第55.3.1节](pt05ch55s03hlb01.md)）。**注意：**位于**渲染样式**工具栏中的渲染样式图标不适用于任何绘图状态下的显示体。此外，默认情况下，显示体只显示自由边缘，而其他模型组件则显示所有外部边缘。
- **Color & Style**：控制模型边缘颜色（["选择整体元素和表面边缘颜色，" 第55.3.3节](pt05ch55s03hlb03.md)）、模型边缘样式（["自定义元素和表面边缘样式，" 第55.3.4节](pt05ch55s03hlb04.md)）和模型填充颜色（["选择整体填充颜色，" 第55.2.3节](pt05ch55s02s01.md)）。
- **Other**：**Other**页面包含以下选项卡：
  - **Scaling**：控制模型缩放和收缩（["缩放坐标和收缩模型，" 第55.4.2节](pt05ch55s04hlb02.md)）。
  - **Translucency**：控制着色渲染样式透明度（["自定义透明度，" 第55.2.2节](pt05ch55s02hlb02.md)）。
