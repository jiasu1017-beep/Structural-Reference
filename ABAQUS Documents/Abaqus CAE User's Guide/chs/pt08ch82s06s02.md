# 82.6.2 准备结构和/或声学数据





典型的 Abaqus/Standard 模态频率响应分析使用频率步长，然后是每个基态的多个稳态动态步长（用于多个载荷情况）。插件从分析生成的输出数据库中读取以下变量：
- 每个频率步长的场输出必须包括节点位移（U）。此外，如果执行耦合结构声学分析，每个频率步长的场输出必须包括节点孔隙或声压（POR）。
- 在每个频率步长之后，必须至少有一个模态稳态动态步长（不是稳态动态、直接稳态动态或子空间稳态动态）。
- 在每个模态稳态动态步长中，历史输出必须包括至少一个模态的广义位移（GU）和广义位移的相位角（GPU）。

您可以在以下位置找到模态稳态动态分析的示例输入文件：["使用子结构和循环对称性分析旋转风扇，" Abaqus Example Problems Guide 第2.2.1节](../exa/exa-link.md#exa-dyn-fansuperelem) 和 ["皮卡车耦合声固分析，" Abaqus Example Problems Guide 第9.1.7节](../exa/exa-link.md#exa-aco-acouststructpickup)。
