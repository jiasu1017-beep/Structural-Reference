# 17.21.5 重新网格划分规则管理器





**重新网格划分规则管理器**与 Abaqus/CAE 中的其他管理器类似，允许您执行以下操作：
- 创建重新网格划分规则。有关更多信息，请参见["创建重新网格划分规则，" 第 17.21.1 节](pt03ch17s21s01.md)。
- 编辑所选的重新网格划分规则。
- 复制、重命名、抑制、恢复或删除所选的重新网格划分规则。

您可以通过从主菜单栏选择 ****自适应**![](../graphics/images/arrow.gif)**重新网格划分规则**![](../graphics/images/arrow.gif)**管理器**** 来显示**重新网格划分规则管理器**。

**重新网格划分规则管理器**中的两列显示以下内容：

**名称**

**名称**列显示重新网格划分规则的名称。单击**重命名**可以重命名所选的重新网格划分规则。

**描述**

**描述**列显示重新网格划分规则的描述。您可以使用描述来帮助您跟踪重新网格划分规则的范围和目的。
