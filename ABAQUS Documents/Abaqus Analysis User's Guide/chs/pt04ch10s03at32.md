# 12.5.1 生成矩阵

**产品：** Abaqus/Standard

##### **参考资料**

- ["矩阵生成，" 第 3.6 节](../stm/stm-link.md#stm-anl-matrix-generation)
- [*ELEMENT OPERATOR OUTPUT](../key/key-link.md#usb-kws-helementoperatoroutput)

### 概述

矩阵生成功能：
- 允许通过生成代表模型惯性、刚度和阻尼特性的全局或单元矩阵来对模型数据进行数学抽象
- 将矩阵数据写入二进制 SIM 文档，可由 Abaqus 进一步处理
- 可用于模型排序减少、与其他软件包交换数据等目的

### 矩阵类型

可以生成以下类型的矩阵：

| **矩阵类型** | **描述** |
| --- | --- |
| 质量矩阵 | 单元质量分布 |
| 刚度矩阵 | 单元刚度特性 |
| 阻尼矩阵 | 单元阻尼特性 |
| 载荷向量 | 外力分布 |

### 生成矩阵

使用 `*ELEMENT OPERATOR OUTPUT` 生成矩阵：

| **输入文件用法：** | 生成质量矩阵： |
| --- | --- |
| | ``` [*ELEMENT OPERATOR OUTPUT](../key/key-link.md#usb-kws-helementoperatoroutput), DAMPING ``` |
| | 生成刚度矩阵： |
| | ``` [*ELEMENT OPERATOR OUTPUT](../key/key-link.md#usb-kws-helementoperatoroutput), STIFFNESS ``` |
| | 生成阻尼矩阵： |
| | ``` [*ELEMENT OPERATOR OUTPUT](../key/key-link.md#usb-kws-helementoperatoroutput), DAMPING ``` |

### 矩阵输出格式

矩阵可以以以下格式输出：

| **格式** | **描述** |
| --- | --- |
| 单元-by-单元 | 默认格式，逐单元存储 |
| 组装 | 组装后的全局矩阵 |

| **输入文件用法：** | 组装矩阵输出： |
| --- | --- |
| | ``` [*ELEMENT OPERATOR OUTPUT](../key/key-link.md#usb-kws-helementoperatoroutput), ASSEMBLE ``` |

### 生成频率

控制矩阵生成的频率：

| **输入文件用法：** | ``` [*ELEMENT OPERATOR OUTPUT](../key/key-link.md#usb-kws-helementoperatoroutput), FREQUENCY=*n* ``` |

### 部分模型矩阵生成

可以为模型的特定部分生成矩阵：

| **输入文件用法：** | ``` [*ELEMENT OPERATOR OUTPUT](../key/key-link.md#usb-kws-helementoperatoroutput), ELSET=*element_set* ``` |

### 输出文件

矩阵写入 SIM 文档：

| **文件** | **描述** |
| --- | --- |
| jobnameTHERM.sim | 热矩阵文件 |
| jobnameSIM.sim | 通用矩阵文件 |

### 矩阵格式

SIM 文档中的矩阵格式：

| **特性** | **描述** |
| --- | --- |
| 稀疏存储 | 仅存储非零元素 |
| 对称优化 | 对称矩阵仅存储上三角 |
| 精度 | 双精度存储 |

### 矩阵组装实用工具

可以使用矩阵组装实用工具来：

- 组装单元矩阵为全局矩阵
- 将矩阵写出到文本文件
- 验证矩阵内容

### 应用

矩阵生成常用于：

1. **模型交换**：与外部软件包共享数据
2. **模型排序减少**：生成用于模型排序减少的矩阵
3. **结果验证**：验证 Abaqus 内部计算
4. **教学**：理解有限元矩阵结构

### 局限性

| **限制** | **描述** |
| --- | --- |
| 线性单元 | 仅线性单元支持 |
| 静力分析 | 必须在静力分析步骤中生成 |
| 接触 | 接触矩阵不生成 |
| 约束 | 某些约束不被处理 |

### 输入文件模板

```
*-heading
Matrix generation
**
*STEP
*STATIC
*BOUNDARY
*LOAD
*ELEMENT OPERATOR OUTPUT, ASSEMBLE, STIFFNESS, DAMPING,
LOAD, FREQUENCY=1
*END STEP
```
