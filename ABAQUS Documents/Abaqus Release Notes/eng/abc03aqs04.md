# 3.4 Monitoring an optimization process







**Product: **Abaqus/CAE  

**Benefits: **You now have more flexibility when monitoring the progression of an optimization process.

**Description: **Abaqus/CAE can now display output data in a table that allows you to monitor the progression of an optimization process to ensure it is converging on an acceptable solution. You can select which variables to monitor, and you can display selected variables in an *X–Y* plot in a new viewport.

**Abaqus/CAE Usage: **
```
Job module:
    ****Optimization**![](../graphics/images/arrow.gif)**Monitor****, **Name:** *task name*
```
