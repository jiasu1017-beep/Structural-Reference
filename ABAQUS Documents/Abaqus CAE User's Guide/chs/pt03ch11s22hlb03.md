# 11.22.3 添加扫掠壳特征





从主菜单栏选择**形状**![](../graphics/images/arrow.gif)**壳**![](../graphics/images/arrow.gif)**扫掠**，向当前视口中的零件添加扫掠壳特征。您只能向三维零件添加扫掠壳特征。

您通过首先定义扫掠路径，然后定义扫掠轮廓来添加扫掠壳特征。

**添加扫掠壳特征：**

1. 从主菜单栏，选择**形状**![](../graphics/images/arrow.gif)**壳**![](../graphics/images/arrow.gif)**扫掠**。
2. 如果要绘制扫掠路径，请执行以下操作。
3. 如果要将扫掠路径指定为一系列边缘或线框，请执行以下操作。
4. 如果要绘制扫掠轮廓，请执行以下操作。
5. 如果要选择一个或多个边缘作为扫掠轮廓，请执行以下操作。
6. 如果需要，执行以下任何操作：切换**包含扭曲**、**包含拔模**、**保持轮廓法线恒定**或**保持内部边界**。
7. 单击**确定**创建新的扫掠壳。
