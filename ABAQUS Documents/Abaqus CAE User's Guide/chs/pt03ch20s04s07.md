# 20.4.7 Sketcher 自定义选项如何初始化和保存







Sketcher 自定义选项分为以下设置：

**Sketcher 选项**

- 捕捉
- 预选行为
- 是否显示构造几何、尺寸、参数名称和约束
- Abaqus/CAE 可以自动添加的约束和尺寸类型

**草图选项**

- 工作表大小、网格间距和网格显示
- 网格原点和轴对齐
- 尺寸的文本高度和小数位数以及参数名称的文本高度

Sketcher 选项控制 Sketcher 的交互行为，而草图选项控制单个草图的外观。Abaqus/CAE 仅在会话期间存储和使用您的 Sketcher 自定义设置。相比之下，Abaqus/CAE 将您的草图自定义设置与每个草图一起存储在模型数据库中。因此，如果您退出 Abaqus/CAE 会话并稍后返回草图，草图自定义选项会被保留。

当您创建新零件或独立草图时，Abaqus/CAE 始终使用默认的草图选项。工作表大小和网格间距根据零件或所选草图平面的大致大小重新计算，草图原点位于 `0,0`，显示网格并与 1 和 2 轴对齐，文本高度和小数位数处于默认设置。
