# 12.11.4 定义磁导率







对于时间谐波涡流分析，您必须定义材料的磁导率。您可以指定各向同性、正交异性或完全各向异性的磁导率。请参见以下章节获取更多信息：
- "磁导率"，Abaqus分析用户指南第26.5.3节
- "涡流分析"，Abaqus分析用户指南第6.7.5节

**定义磁导率：**

1. 从**编辑材料**对话框的菜单栏中，选择**电气/磁性**![](../graphics/images/arrow.gif)**磁导率**。（有关显示**编辑材料**对话框的信息，请参见"创建或编辑材料"，第12.7.1节。）
2. 切换**使用非线性B-H曲线指定**来定义非线性磁导率。**数据**表中标记为**B**和**H**的列将取代用于线性数据的**磁导率**列。
3. 单击**类型**字段右侧的箭头，指定磁导率的方向依赖性。
4. 切换**使用频率相关数据**定义随频率变化的磁导率。**数据**表中会出现一列标为**频率**。**注意：**频率依赖性不适用于非线性磁导率。
5. 切换**使用温度相关数据**将磁导率定义为温度的函数。**数据**表中会出现一列标为**温度**。
6. 单击**场变量数量**字段右侧的箭头增加或减少磁导率所依赖的场变量数量。
7. 在**数据**表中输入适用数据：**B**磁通密度向量的强度。您必须为各向同性非线性磁导率提供一组值，为正交异性非线性磁导率的三个主方向各提供一组值。**H**磁场向量的强度。您必须为各向同性非线性磁导率提供一组值，为正交异性非线性磁导率的三个主方向各提供一组值。**磁导率**各向同性线性磁导率。（单位为[FA2](../popups/usb-int-iconventions-unitsym.md)。）**mu11(E)、mu22(E)和mu33(E)**正交异性线性磁导率的三个值，![](../graphics/usi_eqn00332.gif)、![](../graphics/usi_eqn00333.gif)和![](../graphics/usi_eqn00334.gif)。（单位为[FA2](../popups/usb-int-iconventions-unitsym.md)。）**mu11(E)、mu12(E)、mu22(E)、mu13(E)、mu23(E)和mu33(E)**各向异性线性磁导率的六个值，![](../graphics/usi_eqn00332.gif)、![](../graphics/usi_eqn00335.gif)、![](../graphics/usi_eqn00333.gif)、![](../graphics/usi_eqn00336.gif)、![](../graphics/usi_eqn00337.gif)和![](../graphics/usi_eqn00334.gif)。（单位为[FA2](../popups/usb-int-iconventions-unitsym.md)。）**频率**频率，单位为循环/时间。**温度**温度。**场变量* n ***预定义的场变量。您可能需要展开对话框以查看**数据**表中的所有列。有关如何输入数据的详细信息，请参见"输入表格数据"，第3.2.7节。
8. 单击**确定**关闭**编辑材料**对话框。或者，您可以从**编辑材料**对话框的菜单中选择其他材料行为进行定义（有关更多信息，请参见"浏览和修改材料行为"，第12.7.2节）。
