# 3.2.1 使用基本对话框组件





Abaqus/CAE中的对话框包含以下类型的组件：

**文本字段**

文本字段是对话框中可输入信息的区域。例如，当保存显示组时，您必须在以下显示的文本字段中输入其名称：

![](../graphics/int-textfield-nls.png)

如果输入浮点数，大多数文本字段允许您输入表达式；例如，`cos(2.5/(4.9*pi))`。表达式可以是任何有效的Python表达式。有关更多信息，请参阅["输入表达式，" 第3.2.2节"](pt01ch03s02s02.md)。

只要需要为对象命名（如零件、材料、集合、路径或X-Y数据）或为对象提供描述（如材料或步骤），就可以使用文本字段。通常，您应该避免在对象名称或描述中使用星号（*）。

对象名称必须遵守以下规则：
- 名称最多可包含38个字符。
- 名称可以包含空格和大多数标点符号及特殊字符；但是，仅支持7位ASCII字符。
- 名称不能以数字开头。
- 名称不能以下划线或空格开头或结尾。
- 名称不能包含句点或双引号。
- 名称不能包含反斜杠。

对模型名称和作业名称适用其他限制。
- 当您命名模型或作业时，名称可以数字开头。
- 当您命名模型时，不能使用以下字符：``` $&*~!()[]{}|;'`",.?/\>< ```
- 当您命名作业时，不能使用以下字符：``` <space>$&*~!()[]{}|:;'`",.?/\>< ```此外，作业名称不能以破折号`-`开头。

材料评估过程（["评估超弹性和粘弹性材料行为，" 第12.4.7节"](pt03ch12s04s07.md)）会生成与材料同名的作业；因此，这些材料名称必须遵守与作业名称相同的规则。通常，当您指定将在Abaqus/CAE外部使用的名称（如文件名）时，应避免使用可能在您的平台上有保留含义的任何字符。

**注意：**Abaqus/CAE会保留您在文本字段中输入的任何文本的大小写。例如，如果您在属性模块的**编辑材料**对话框中创建名为`Steel Alloy`的材料，该材料将在图形用户界面（材料管理器、截面编辑器、模型树等）中显示为`Steel Alloy`。在图形用户界面中，对象名称不区分大小写。例如，您不能创建另一个名为`steel alloy`的材料。相反，Python（在命令行界面中使用）区分大小写，但您不应依赖此行为来区分对象。

**数字字段**

数字字段是用于整数输入值的专用文本字段。它们在文本区域右侧直接有两个相对的箭头。您可以在文本字段中输入数字值，也可以使用箭头在固定值列表中上下循环。

![](../graphics/int-spinbox.png)

与其他文本字段不同，数字字段不接受文本或特殊字符。

数字字段通常有上限和下限。如果您输入的值超过限制，当您移动到另一个字段或尝试应用该值时，Abaqus/CAE会将条目更改为您最接近的可接受值。

**组合框**

组合框是字段右侧直接有箭头的字段。如果单击此箭头，会出现可输入该字段的可能选择列表。例如，如果单击上下文栏中**模块**字段右侧的箭头，会出现所有Abaqus/CAE模块的列表，您可以从列表中选择所需的模块。

![](../graphics/gst-module-list-nls.png)

**单选按钮**

单选按钮呈现互斥选择。当选项由单选按钮控制时，您一次只能选择其中一个按钮。

![](../graphics/int-radio-nls.png)

**复选框**

您可以切换复选框以关闭或打开特定选项。

例如，当前视口中三轴的可见性取决于**显示三轴**复选框的状态。如果复选框被切换为开启，如下所示，三轴会出现在视口中。

![](../graphics/int-checkboxon-nls.png)

如果复选框被切换为关闭，如下所示，三轴不会出现在视口中。

![](../graphics/int-checkboxoff-nls.png)

在某些情况下，由复选框控制的选项可以应用于多个对象。例如，**XY曲线选项**对话框中的单个**显示线条**复选框单独控制X-Y绘图中所有X-Y曲线线条的显示。如果您对某些曲线切换**显示线条**为开启而对其他曲线切换为关闭，该复选框会显示为灰色并带有较深的灰色勾选标记，如下所示。

![](../graphics/int-tristate-nls.png)

**滚动条**

滚动条出现在内容太多无法显示的列表中；它们允许您滚动浏览列表的可见内容以及任何隐藏的内容。当需要列出大量项目时，通常需要滚动，如下所示。

![](../graphics/int-scrollbar-nls.png)
