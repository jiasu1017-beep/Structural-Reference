# 30.8 查看流体分析结果





您可以在可视化模块中显示流体分析的结果。动画、线型等值线图、等值面型等值线图和沿路径显示的结果对于查看流体分析结果特别有用。此外，您可以查看共仿真结果，例如使用Abaqus/CFD和Abaqus/Standard的共轭热传递分析，如图[30-1](pt04ch30s08.md#adv-fluid-circuit-board)所示。
