# 17.10.3 使用三角形和四面体单元的自由网格





使用三角形单元的自由网格可应用于任何平面或曲面，零件可以是精确的或不精确的。更多信息，请参见["什么是有效且精确的零件？"第10.2.1节](pt02ch10s02s01.md)。三角形单元的自由网格会精确匹配网格种子。此网格技术可以处理单元尺寸的较大变化，这在您只想细化网格的一部分时非常有用。Abaqus/CAE计算自由三角形网格的时间大约与单元和节点的数量成线性关系。图17-91展示了使用此技术生成的网格示例。

**图17-91** 使用三角形单元生成的自由网格。

![](../graphics/mgn-freemesh.png)

使用四面体单元的自由网格可应用于几乎任何三维区域；事实上，非常复杂的模型可以在不需要分割帮助的情况下使用此技术进行网格划分。使用四面体单元的自由网格与使用三角形单元的自由网格类似，因为零件可以是精确的或不精确的。一般来说，Abaqus/CAE会精确匹配网格种子，尽管如果网格种子未完全约束，网格在小孔周围可能会更细。图17-92展示了自由四面体网格的示例。

**图17-92** 使用四面体单元生成的自由网格。

![](../graphics/mgn-tetmesh.png)

不支持使用六面体单元对三维实体进行自由网格划分。

在尝试使用四面体单元生成自由网格之前，您应该使用"零件"模块或"网格"模块中的查询工具集来检查零件或装配的几何形状。您应该检查以下内容：
- 实体中没有自由边。
- 没有短边、小面或小的面角。

更多信息，请参见["使用几何诊断工具，"第71.2.4节](pt06ch71s02hlb04.md)。

当您使用四面体单元创建自由网格时，您可以使用默认网格算法，或关闭默认算法以使用Abaqus/CAE 6.4及更早版本中包含的算法。默认算法更加稳健，特别是在对复杂形状和薄实体进行网格划分时。此外，默认算法允许您增加内部单元的尺寸。如果正在分析的模型的网格密度足够，且感兴趣的区域在网格边界上，增加内部单元的尺寸将提高计算效率。

当您使用自由网格在曲面上创建三角形单元时，Abaqus/CAE会在适当的地方对某些区域使用映射网格。（映射网格与结构化网格相同，但仅适用于四边形区域。）有关更多信息，请参见["什么是映射网格？"第17.8.2节](pt03ch17s08s02.md)和["Abaqus/CAE何时可以应用映射网格？"第17.8.6节](pt03ch17s08s06.md)。只有当可能导致网格质量提高时，Abaqus/CAE才会用映射三角形网格替换自由三角形网格。如果需要，您可以使用网格控制来防止Abaqus/CAE使用映射网格。更多信息，请参见["分配网格控制，"第17.18.1节](pt03ch17s18s01.md)。

类似地，当您使用自由网格创建四面体单元时，您可以允许Abaqus/CAE决定映射网格是否适当。在Abaqus/CAE确定网格质量将提高之后，它会用映射网格替换简单边界面上的自由网格，并将生成的三角形转换为四面体单元。

图17-93展示了在适当的地方允许映射网格时，使用四面体单元的实体零件自由网格的效果。可以通过映射网格划分的四边形区域的网格质量得到改善。

**图17-93** 允许映射网格的效果。

![](../graphics/mgn-mapped-caliper-nls.png)

要查看Abaqus/CAE生成的内部四面体单元，您可以从当前网格创建新的网格零件，并使用显示组移除选定的单元。您也可以在分析完成后使用"可视化"模块中的视图切割来查看内部单元。更多信息，请参见[第78章，"使用显示组显示模型的子集"](pt07ch78.md)和[第80章，"切割模型"](pt07ch80.md)。
