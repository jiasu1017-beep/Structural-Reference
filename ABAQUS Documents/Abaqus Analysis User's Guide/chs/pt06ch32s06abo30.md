### 单元属性定义

您必须定义单元的横截面积（对于链接单元）或面外宽度（对于通用单元）、初始间隙和初始空隙。

您可以在垫片截面定义中指定厚度方向，也可以在节点处指定法向方向；您可以在垫片截面定义中指定单元厚度。否则，Abaqus/Standard将计算厚度方向。对于链接单元，厚度方向是从第一个节点到第二个节点的方向，厚度是节点之间的距离。对于通用单元，厚度方向基于单元的中面，积分点处的厚度基于节点位置。详见["定义垫片单元的初始几何，" 第32.6.4节](pt06ch32s06alm49.md)。

### 单元输出

#### GK2D2单元

| S11 | 垫片单元中的压力或厚度方向力。 |
| --- | --- |

| CS11 | 垫片单元中的接触压力（仅当S11是垫片单元中的力且垫片响应未使用材料模型定义时才可用）。 |
| --- | --- |

| S12 | 剪切应力或剪切力。 |
| --- | --- |

| E11 | 如果垫片响应直接使用垫片行为模型定义，则为垫片闭合；如果垫片响应使用材料模型定义，则为应变。 |
| --- | --- |

| E12 | 如果垫片响应直接使用垫片行为模型定义，则为剪切运动；如果垫片响应使用材料模型定义，则为应变。 |
| --- | --- |

| NE11 | 垫片单元中有效的厚度方向应变。 |
| --- | --- |

| NE12 | 垫片单元中有效的剪切应变。 |
| --- | --- |
