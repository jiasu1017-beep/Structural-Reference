# 19.12.3 验证优化过程







验证检查优化过程以确保优化任务已正确配置且 Abaqus 模型存在。验证不检查您的 Abaqus 模型。在尝试运行优化过程之前，您应该运行模型的完整分析并确保它运行完成。

要验证优化过程，请从主菜单栏中选择****优化**![](../graphics/images/arrow.gif)**验证**![](../graphics/images/arrow.gif)***优化过程名称*****。您也可以通过在**优化过程管理器**中选择过程名称，然后点击**验证**来验证优化过程。
