# 73.3.3 创建曲面







"曲面工具集"允许您创建几何或网格曲面。几何曲面可以由几何面和边组成；网格曲面可以由单元面和边组成。

**创建曲面的步骤：**

1. 从主菜单栏，选择****Tools**![](../graphics/images/arrow.gif)**Surface**![](../graphics/images/arrow.gif)**Create****。**提示：**您也可以在**Surface Manager**中单击**Create**。将出现"创建曲面"对话框。该对话框列出您正在创建的曲面类型。
2. 在对话框中：
   1. 输入曲面的名称。不同部件上创建的曲面可以使用相同的名称；但是，装配体曲面名称必须唯一。有关命名对象的信息，请参见["使用基本对话框组件," 第3.2.1节](pt01ch03s02s01.md)。
   2. 如果需要，选择曲面类型。
   3. 单击**Continue**。如果您在原生部件上创建网格曲面，Abaqus/CAE会自动显示网格（如果尚未可见）。
3. 在视口中选择要包含在曲面中的实体。如果您正在创建几何曲面，Abaqus/CAE只允许您选择面和边。同样，如果您正在创建网格曲面，Abaqus/CAE只允许您选择单元面和单元边。您可以选择内部实体包含在曲面中。更多信息，请参见["选择内部曲面," 第6.2.12节](pt01ch06s02hlb11.md)。要创建基于边的曲面，您必须选择壳的边或单元的边。更多信息，请参见["什么是曲面？," 第73.2.3节](pt06ch73s02s03.md)。
4. 选择完实体后，单击鼠标按钮2。
5. 如果您从原生部件创建几何曲面，必须指定所选面或所选线的侧或端。更多信息，请参见["指定区域的特定侧或端," 第73.2.5节](pt06ch73s02s05.md)和["理解几何和物理对象之间的对应关系," 第6.1.3节](pt01ch06s01s03.md)。
