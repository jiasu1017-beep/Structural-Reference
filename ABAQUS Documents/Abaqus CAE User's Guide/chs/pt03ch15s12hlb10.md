# 15.12.10 创建重合点连接器





在「相互作用」模块中，从主菜单栏选择 ****Connector**![](../graphics/images/arrow.gif)**Coincident Builder**** 可以为一组重合点创建连接器线特征和截面分配。有关创建装配级线特征的详细说明，请参见「添加点对点线特征」，第11.23.2节。

**创建重合点连接器线特征的步骤：**

1. 在「相互作用」模块中，从主菜单栏选择 ****Connector**![](../graphics/images/arrow.gif)**Coincident Builder****。
2. 在视口中选择要包含在连接器线特征中的重合点，然后在提示区域点击「Done」。有关从视口中选择多个点的方法概述，请参见「拖动选择多个对象」，第6.2.2节。「Coincident Point Builder」对话框打开，所选重合点显示在对话框的「Endpoints」部分。
3. 如有需要，通过执行以下操作自定义要使用的端点的内容和顺序：- 高亮任何重合点的行，并点击![](../graphics/ico_delete.png)删除该重合点对。- 高亮任何重合点的行，并点击![](../graphics/ico_swap.png)交换重合点对中的第一个和第二个点。
4. 选择连接器截面。如有需要，点击![](../graphics/ico_connectorSection.png)创建新的连接器截面。连接类型会显示在编辑器中。对于基本、组装和复杂连接类型，您可以点击![](../graphics/ico_tipSmall.png)查看连接类型的示意图以及Abaqus对连接的理想化。对于MPC连接类型，第二个点的运动被约束到第一个点的运动。
5. 如有需要，修改以下任一方向设置：1. 在对话框的「CSYS 1」部分为重合点连接器线特征指定全局坐标系以外的方向。如果所选连接器截面分配允许您调整连接器中的第二个点，您也可以在对话框的「CSYS 2」部分为第二个点指定方向。在对话框的任一部分中，点击![](../graphics/ico_selectBlue.png)，然后使用以下方法之一选择基准坐标系：- 按名称选择预定义的基准坐标系。从提示区域点击「Datum CSYS List」，从列表中选择一个名称，然后点击「OK」。- 在视口中选择预定义的坐标系。- 对于连接器中的第二个点，选择「Use CSYS 1」以在连接器的第二个点也使用为第一个点定义的坐标系。所选方向在视口中以红色高亮显示。「Coincident Point Builder」重新出现并显示用于所选连接器点方向坐标系的描述。**提示：**您也可以通过点击对话框「CSYS 1」或「CSYS 2」部分的![](../graphics/ico_dtmCsys.png)来创建新的基准坐标系。2. 如有需要，您可以为任一端点指定关于基准坐标系轴的附加旋转。为点选择「Additional rotation angle」选项，输入旋转角度值（以度为单位），并为「About axis」选项选择「1」、「2」或「3」。仅当您指定全局坐标系以外的基准坐标系时，「Additional rotation angle」选项才可用。
6. 点击「OK」创建线特征和连接器截面分配。Abaqus/CAE还会创建一个包含装配级线的集合。原始线特征被重命名并抑制。修改后的装配级线特征出现在模型树中「Features」容器下的装配中，使用与原始线特征相同的名称。
