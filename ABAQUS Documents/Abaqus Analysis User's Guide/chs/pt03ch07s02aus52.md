# 11.1.4 瞬态问题的时间积分精度

**产品：** Abaqus/Standard 和 Abaqus/Explicit

### 概述

瞬态分析涉及随时间变化的载荷和响应。时间积分精度直接影响计算结果的可靠性。Abaqus 提供了两种主要的时间积分方法：隐式和显式，每种方法都有其精度和效率特点。

### 隐式时间积分

隐式方法（如 Newmark-β 法）在每个时间步求解平衡方程时包含未知的时间导数。这种方法是无条件稳定的，允许使用较大的时间步长。

#### Newmark 参数

| **参数** | **描述** | **默认值** |
| --- | --- | --- |
| BETA | Newmark-β 参数 | 0.25 |
| GAMMA | Newmark-γ 参数 | 0.5 |

| **输入文件用法：** | 使用以下选项设置 Newmark 参数： |
| --- | --- |
| | ``` [*DYNAMIC](../key/key-link.md#usb-kws-hdynamic), SOLVER=SPARSE BETA=*value*, GAMMA=*value* ``` |

### 显式时间积分

显式方法（如中心差分法）使用前一时间步的信息计算当前时间步的响应。这种方法条件稳定，需要满足 Courant-Friedrichs-Lewy (CFL) 条件：

Δt ≤ Δt_critical

### 临界时间步长

临界时间步长由模型的最高固有频率决定：

Δt_critical = 2 / ω_max

其中 ω_max 是最高固有频率。

| **参数** | **描述** | **默认值** |
| --- | --- | --- |
| INITIAL | 初始时间增量 | 自动计算 |
| MINIMUM | 最小时间增量 | 1e-8 |
| MAXIMUM | 最大时间增量 | 无限制 |

### 时间积分精度控制

#### 隐式方法精度

对于隐式方法，精度主要由时间步长和 Newmark 参数决定：

| **参数组合** | **精度类型** | **稳定性** |
| --- | --- | --- |
| β=0.25, γ=0.5 | 二阶精度 | 无条件稳定 |
| β=0.0, γ=0.5 | 一阶精度 | 无条件稳定 |

#### 显式方法精度

对于显式方法，精度主要由时间步长决定：

| **时间步长选择** | **精度** |
| --- | --- |
| Δt = Δt_critical | 最高精度（可能有数值阻尼） |
| Δt < Δt_critical | 较高精度 |
| Δt > Δt_critical | 不稳定 |

### 自动时间步长调整

Abaqus 可以自动调整时间步长以平衡精度和效率：

| **输入文件用法：** | 使用以下选项启用自动时间步长： |
| --- | --- |
| | ``` [*DYNAMIC](../key/key-link.md#usb-kws-hdynamic), DIRECT=NO ``` |

### 数值阻尼

在某些情况下，数值阻尼可以帮助稳定高频响应：

| **参数** | **描述** |
| --- | --- |
| HUBER | Huber 数值阻尼 |
| HALPERIN | Halperin 数值阻尼 |

### 高频响应过滤

高频响应可能会干扰低频响应解析。可以使用以下选项过滤：

| **输入文件用法：** | 使用以下选项过滤高频响应： |
| --- | --- |
| | ``` [*CONTROLS](../key/key-link.md#usb-kws-hcontrols), HIGH FREQUENCY=DAMPING ``` |

### 精度验证建议

1. **网格收敛性研究**：使用不同网格密度进行分析
2. **时间步长敏感性研究**：减小时间步长直到结果不变
3. **能量平衡检查**：检查动能和内能的比值
4. **结果验证**：与解析解或实验数据比较
