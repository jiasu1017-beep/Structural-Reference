# 3.6.1 涡流分析







**产品：**Abaqus/Standard  

### 测试的单元

EMC2D3    EMC2D4    EMC3D4    EMC3D8

### 测试的功能

由体积或体电流密度![](../graphics/ver_eqn01163.gif)或表面电流密度![](../graphics/ver_eqn01164.gif)激励的涡流边界值问题的时间谐波和瞬态响应。

### 问题描述

求解对应于两种激励类型的问题。两个问题都在域中产生恒定的磁通密度![](../graphics/ver_eqn01165.gif)。具有体电流激励![](../graphics/ver_eqn01163.gif)的输入文件归类为CCBL（恒定旋度体载荷）问题，具有表面电流激励![](../graphics/ver_eqn01164.gif)的输入文件归类为CCSC（恒定旋度表面电流）问题。

**CCBL问题：**

在此类别中仅测试了时间谐波涡流问题。二维问题的域是位于平面第一象限的正方形；三维问题的域是位于空间第一卦限的长方体。对于微分方程![](../graphics/ver_eqn01166.gif)，寻求的解为![](../graphics/ver_eqn01167.gif)，其中![](../graphics/ver_eqn01168.gif)。对于此解，微分方程中的第一项消失。因此，在域中处处施加非均匀体载荷（CJNU）![](../graphics/ver_eqn01169.gif)。还在外边界上规定非零边界条件（分布式表面磁向量势），并在对称平面上规定对称边界条件。

**CCSC问题：**

在此类别中测试了时间谐波和瞬态涡流问题。部分二维问题的域是位于平面第一象限的四分之一圆；三维问题的域是位于空间第一卦限的四分之一圆柱，圆柱轴线与全局![](../graphics/ver_eqn01170.gif)方向对齐。表面电流载荷![](../graphics/ver_eqn01171.gif)作为Neumann型边界条件规定在外边界上。在对称平面上规定对称边界条件。对于时间谐波问题，此情况的解析解为![](../graphics/ver_eqn01172.gif)，与CCBL问题相同。瞬态问题的解（仅实部）相同。

测试瞬态涡流过程的问题具有相似的域（除了输入文件名以ccsc_2d_开头的二维问题，它由承受边界条件/载荷的独立电磁单元组成）。在部分问题中，磁特性在模型的不同区域定义为不同；特别是，在一个区域使用线性特性，而在另一区域使用非线性特性。表面电流载荷在域内产生恒定磁场，但磁通密度根据材料行为而变化。已经测试了各向同性和正交异性磁行为。

部分问题还测试了运动对解的影响。在所有情况下施加均匀平移速度。磁场与无运动问题相同，但由于运动影响，电场被修改。

**材料特性：**

在整个所有时间谐波问题和瞬态问题中具有线性磁行为的区域中，使用自由空间的磁导率![](../graphics/ver_eqn01173.gif) H/m或N/A^2。对于具有非线性磁行为的区域，响应以B-H曲线的形式定义，描述磁通密度强度作为磁场强度的函数。[表3.6.1-1](ch03s06abv200.md#table-bh-curve)提供了这些测试中使用的B-H曲线。

**时间谐波问题的激励频率：**

![](../graphics/ver_eqn01175.gif) rad/s，![](../graphics/ver_eqn01176.gif) 50或60 Hz。

**瞬态问题的载荷：**

瞬态问题加载随时间变化的表面电流幅度。部分问题使用正弦时间变化。
