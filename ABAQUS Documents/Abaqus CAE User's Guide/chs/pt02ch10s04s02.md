# 10.4.2 什么是IGES实体？





当您导入IGES格式文件时，Abaqus/CAE会在显示**从IGES文件创建零件（Create Part from IGES File）**对话框之前扫描文件内容。然后，您可以使用**IGES选项（IGES Options）**选项卡页面上的按钮查看以下内容：

**IGES标题（IGES Header）**

IGES标题信息包括有关写入IGES文件的应用程序的详细信息。它还包括文件作者的信息以及写入文件的日期，以及比例、分辨率和单位。

**实体列表（Entity List）**

实体可以是几何实体，如点、弧或线。或者，实体可以与几何图形分离，如注释。IGES为每个实体分配一个编号；例如，圆弧是实体编号100。IGES实体列表显示在文件中找到的每种类型实体的列表。该列表包括IGES实体编号、实体描述和在文件中找到的数量。

有关可导入Abaqus/CAE的IGES实体的完整列表，请参阅["导入零件或草图时Abaqus/CAE识别的IGES实体，"第10.7.8节"](pt02ch10s07hlb08.md)。
