# 49.1.2 比例因子动画





在比例因子动画中，Abaqus/CAE从输出数据库的单个步骤和帧创建变形形状、等值线或符号图序列中的所有图像。Abaqus/CAE生成根据您的规范从0到1或从1到1的动画比例因子。各个图像是通过将动画比例因子范围应用于图中显示的场输出值形成的。这给出了场输出随时间演变的 appearance，尽管所有图像都是从您的结果的单个步骤和帧生成的。（要查看结果随时间的实际演变，您必须生成时间历史动画。）

如果您正在动画化等值线图，场输出值的显示取决于您选择的选项。默认行为是图例中的值基于所选结果步骤和帧的缩放最小值和最大值固定。要了解如何设置等值线限制，请参阅["设置等值线限制，"第44.5.7节](pt05ch44s02hlb07.md)。要了解如何自定义图例，请参阅["自定义图例，"第56.1节](pt05ch56s01.md)。
