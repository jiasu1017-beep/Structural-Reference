# 30.4 为流体模型指定预定条件





您可以在Load模块中指定流体模型的预定条件。您可以在流体分析的初始步骤中指定流体预定义场，也可以在流体分析步骤中为流体模型指定载荷和边界条件。根据分析的不同，您可能需要指定以下预定条件：

**预定义场**

流体分析可用的预定义场类型包括流体密度、流体热能、流体湍流和流体速度。更多信息，请参见["Abaqus/CFD中的初始条件"，Abaqus Analysis User's Guide第34.2.2节](../usb/usb-link.md#usb-prc-pinitialcondcfd)。
- 需要密度值；但是，如果您未指定初始流体密度，则假定使用材料密度定义。
- 对于包含能量方程的不可压缩流体动力学分析，您必须定义初始流体温度。
- 对于指定湍流模型的不可压缩流体动力学分析，您必须定义初始流体湍流值，如湍流涡粘度。
- 如果未指定值，则假定初始速度为零。

**载荷**

流体分析可用的载荷类型包括体积力、重力、体热通量和多孔拖曳体积力。如果未指定压力边界条件，流体参考压力可用于指定不可压缩流动的静水压力水平。更多信息，请参见以下各节：
- ["集中载荷"，Abaqus Analysis User's Guide第34.4.2节](../usb/usb-link.md#usb-prc-ploadgeneral)
- ["分布载荷"，Abaqus Analysis User's Guide第34.4.3节](../usb/usb-link.md#usb-prc-ploaddistributed)
- ["热载荷"，Abaqus Analysis User's Guide第34.4.4节](../usb/usb-link.md#usb-prc-pthermal)

**边界条件**

流体分析可用的边界条件类型包括位移/旋转、流体入口/出口和流体壁条件。更多信息，请参见["Abaqus/CFD中的边界条件"，Abaqus Analysis User's Guide第34.3.2节](../usb/usb-link.md#usb-prc-pboundarycfd)。
- 您指定边界条件以定义流体域边界上的流动和热条件。
- 您可以指定压力或速度边界条件。
- 对于包含能量方程的不可压缩流体动力学分析，您可以定义温度或热通量。
