# 2.4 什么是工具集？





当您进入大多数模块时，**工具**菜单会出现在主菜单栏中，其中包含与该模块相关的所有工具集。工具集是一个功能单元，允许您执行特定的建模任务。

在大多数情况下，您在一个模块中使用工具集创建的对象在其他模块中也很有用。例如，您可以使用集合工具集在装配模块中创建集合，然后可以在载荷模块中对这些集合施加边界条件。大多数工具集包括管理器菜单和管理器对话框，允许您编辑、复制、重命名和删除使用工具集创建的对象。

Abaqus/CAE中提供以下工具集：
- 振幅工具集允许您定义载荷、位移和其他规定变量的任意时间或频率变化。有关更多信息，请参阅[第57章，"振幅工具集"](pt06ch57.md)。
- 分析场工具集允许您创建可用于定义选定相互作用和规定条件的空间变化参数的分析场。有关更多信息，请参阅[第58章，"分析场工具集"](pt06ch58.md)。
- 附件工具集允许您创建附件点和线，您可以使用它们来定义点-based和离散紧固件、连接器的连接点以及耦合定义的区域、点质量、载荷或边界条件。有关更多信息，请参阅[第59章，"附件工具集"](pt06ch59.md)。
- CAD连接工具集允许您创建连接，您可以使用该连接从CATIA和第三方CAD系统将零件关联导入到Abaqus/CAE中。有关更多信息，请参阅[第60章，"CAD连接工具集"](pt06ch60.md)。
- 颜色编码工具集允许您自定义单个元素的边缘和填充颜色。有关更多信息，请参阅[第77章，"几何体和网格元素的颜色编码"](pt07ch77.md)。
- 坐标系工具集允许您创建用于后处理的局部坐标系。有关更多信息，请参阅["在后处理期间创建坐标系，" 第42.8节"](pt05ch42s06.md)。
- 创建场输出工具集允许您对输出数据库中可用的场输出执行操作。有关更多信息，请参阅["创建新场输出，" 第42.7节"](pt05ch42s05.md)。
- 自定义工具集允许您控制Abaqus/CAE工具栏的外观，创建自定义工具栏，并为许多Abaqus/CAE功能指定键盘快捷键。有关更多信息，请参阅[第61章，"自定义工具集"](pt06ch61.md)。
- 基准工具集允许您为各种建模任务创建基准点、基准轴、基准平面和坐标系。有关更多信息，请参阅[第62章，"基准工具集"](pt06ch62.md)。
- 离散场工具集允许您创建空间变化的场，其中值与节点或单元相关联。有关更多信息，请参阅[第63章，"离散场工具集"](pt06ch63.md)。
- 显示组工具集允许您有选择地绘制一个或多个模型或输出数据库项目。有关更多信息，请参阅[第78章，"使用显示组显示模型的子集"](pt07ch78.md)。
- 编辑网格工具集允许您修改网格以提高网格质量。有关更多信息，请参阅[第64章，"编辑网格工具集"](pt06ch64.md)。
- 特征操作工具集允许您修改和管理模型中现有的特征。有关更多信息，请参阅[第65章，"特征操作工具集"](pt06ch65.md)。
- 过滤器工具集允许您在分析模型时去除多余的输出数据（噪声），而不会失去所需数据范围的分辨率。有关更多信息，请参阅[第66章，"过滤器工具集"](pt06ch66.md)。
- 自由体工具集允许您在Abaqus/CAE的可视化模块中创建和自定义自由体切割。有关更多信息，请参阅[第67章，"自由体工具集"](pt06ch67.md)。
- 几何编辑工具集允许您修复无效和不精确的导入零件。有关更多信息，请参阅[第69章，"几何编辑工具集"](pt06ch69.md)。
- 分区工具集允许您将零件或装配体划分为区域。有关更多信息，请参阅[第70章，"分区工具集"](pt06ch70.md)。
- 路径工具集允许您指定通过模型的路径，您可以沿该路径获取和查看X-Y数据。有关更多信息，请参阅[第48章，"沿路径查看结果"](pt05ch48.md)。
- 查询工具集允许您获取有关模型的一般信息，并探测模型和X-Y曲线的输出数据。有关更多信息，请参阅[第71章，"查询工具集"](pt06ch71.md)。
- 参考点工具集允许您创建与零件或装配体关联的参考点。有关更多信息，请参阅[第72章，"参考点工具集"](pt06ch72.md)。
- 集合工具集和曲面工具集允许您从模型的区域定义集合和曲面。有关更多信息，请参阅[第73章，"集合和曲面工具集"](pt06ch73.md)。
- 流线工具集允许您显示流线，以研究流体流动分析中的速度或涡量。有关更多信息，请参阅[第74章，"流线工具集"](pt06ch74.md)。
- 虚拟拓扑工具集允许您在为零件或零件实例划分网格时忽略细节，例如非常小的面和边。有关更多信息，请参阅[第75章，"虚拟拓扑工具集"](pt06ch75.md)。
- XY数据工具集允许您创建X-Y数据对象并对其进行操作。有关更多信息，请参阅[第47章，"X-Y绘图"](pt05ch47.md)。
