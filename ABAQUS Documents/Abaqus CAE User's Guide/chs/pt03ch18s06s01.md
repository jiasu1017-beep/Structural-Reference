# 18.6.2 配置拓扑优化任务






优化模块提供了多种设置，允许您配置拓扑优化任务。配置设置取决于您是为一般拓扑优化还是基于条件的拓扑优化配置优化任务。涵盖以下主题：
- [「配置一般拓扑优化任务」](pt03ch18s06s01.md#usi-opz-topo-taskconfigure-general)
- [「配置基于条件的拓扑优化任务」](pt03ch18s06s01.md#usi-opz-topo-taskconfigure-stiffness)

### 配置一般拓扑优化任务

一般拓扑优化是一种灵活的、基于灵敏度的优化，允许您对模型应用一系列约束和目标函数。您可以使用优化任务编辑器来自定义一般拓扑优化的各个方面。要找到编辑器，请从主菜单栏选择****任务**![](../graphics/images/arrow.gif)**编辑**![](../graphics/images/arrow.gif)***优化任务名称*****。要指定一般拓扑优化，请选择**高级**选项卡，然后选择**一般优化（基于灵敏度）**。

#### 配置基本设置

**配置基本设置：**

1. 在优化任务编辑器中，单击**基本**选项卡。
2. 选择是否冻结载荷或边界条件区域。建议您冻结应用规定条件的区域，因为在优化过程中不希望移除这些区域。冻结这些区域可以稳定优化，通常会大大减少迭代次数。

#### 配置密度设置

**配置密度设置：**

1. 在优化任务编辑器中，单击**密度**选项卡。
2. 选择**密度更新策略**。此设置控制优化模块在优化过程中更新设计单元相对材料密度的速率。在大多数情况下，您应该接受默认设置（**正常**）。但是，如果设计响应非常敏感，并且您在满足约束方面遇到问题，您可能需要更保守的速率，这需要更多的优化迭代。
3. 执行以下任一操作来指定每个单元在初始优化迭代中的相对密度：- 选择**优化产品默认值**以允许优化模块确定初始密度。如果选择材料体积作为约束，优化模块会计算初始密度，以精确满足体积约束。如果选择材料体积作为目标函数，每个单元的初始相对密度为 50%。- 选择**指定**并输入值（0.0 < *初始密度* ≤ 1.0）。仅当体积被选为目标函数而非约束时，并且如果您在优化之前知道将初始密度设置为更大或更小的值将满足其他约束（例如位移约束），才应使用此选项。您可以结合体积约束使用大于 0.5 的值来稳定非线性或接触问题，并改善收敛行为。
4. 输入**最小密度**、**最大密度**和**每个设计循环的最大变化**。最小密度必须大于 0.0，最大密度必须小于或等于 1.0。不建议更改密度边界，特别是上限。如果默认值导致近乎奇异的刚度矩阵，您可能需要增加下限。数值实验表明，0.25（默认值）是最大密度变化的可接受值。对于复杂的设计响应和优化公式，建议使用 0.1 的密度变化下限。但是，较低的下限通常会导致更多的优化迭代。

#### 配置扰动设置

**配置扰动设置：**

1. 在优化任务编辑器中，单击**扰动**选项卡。
2. 输入要跟踪的特征模态数量。默认值为 5，这意味着优化模块将尝试跟踪五个最低特征频率。在某些情况下，在优化迭代过程中会出现许多局部低频特征模态，这导致要跟踪的模态数量很多，并降低性能。您可以通过选择特征频率下限为第一次优化迭代中感兴趣特征频率的 25% 来避免跟踪大量模态。如果您的设计响应将使用 Kreisselmaier-Steinhauser 公式来评估特征频率，则不需要模态跟踪。您的 Abaqus 模型必须包含至少您正在跟踪的特征频率数量的输出请求。
3. 选择优化模块应跟踪特征模态的区域。

#### 配置收敛选项

**配置收敛选项：**

1. 在优化任务编辑器中，单击**收敛**选项卡。
2. 指定**收敛标准**。以下选项允许您指定一般拓扑优化的收敛标准：**指定何时开始检查收敛**您可以指定优化模块将开始检查两个收敛标准的迭代。优化将始终至少继续到达到此值。默认值为 4。**指定要检查哪个收敛标准**您可以指定优化是在满足任一收敛标准还是两个收敛标准都满足时结束。默认值为必须满足两个标准。**基于优化函数变化的收敛**您可以指定优化将基于从一个迭代到下一个迭代的目标函数变化结束。默认值为 0.001。**基于单元密度变化的收敛**单元密度是拓扑优化的设计变量。您可以指定优化将基于从一个迭代到下一个迭代的单元平均密度变化结束。默认值为 0.005。

#### 配置高级选项

**配置高级选项：**

1. 在优化任务编辑器中，单击**高级**选项卡。
2. 选择**一般优化**算法。
3. 选择是否**删除区域中的软单元**。在拓扑优化过程中，优化模块在设计区域内分配给定的质量，同时尝试满足约束并优化目标。在优化结束时，结构包含硬（填充）单元和软（空）单元。软单元对结构刚度的影响可以忽略不计；但它们仍然与结构的自由度数量相关，因此影响优化过程的速度。**删除软单元**选项允许您选择将仅具有软邻居单元的软单元移除的区域。如果需要，删除的单元会被重新激活；例如，如果力流在优化过程中发生变化。选择删除软单元可以帮助 Abaqus 收敛到解决方案，因为否则这些单元会退化或坍塌，建议在优化非线性模型时使用。此外，选择**保守**密度更新策略和每个设计循环的较小密度变化将提高结果的准确性。有关更多信息，请参阅[「配置拓扑优化任务」中的「配置密度设置」，第 18.6.2 节](pt03ch18s06s01.md#usi-opz-topo-taskconfigure-general-density)。
4. 如果您选择删除软单元，您可以通过选择仅删除具有相邻软单元的软单元来防止孤立软单元被移除。您可以将相邻单元定义为在**平均边长**（默认值）指定的半径内，或在您输入的值指定的半径内。如果网格内单元边长变化很大，则从平均边长计算的半径可能具有误导性。
5. 如果您选择删除软单元，您可以选择优化模块将使用的方法来删除单元：**保持连续性（标准）**选择**保持连续性（标准）**并输入**相对材料密度阈值**以在删除软单元之前检查连续性。如果优化模型包含通过软单元与模型其余部分分隔的硬单元「孤岛」，优化模块不会移除软单元。此外，优化模块保留阻止硬单元相对移动的软单元；例如，共享公共边但不共享公共面的硬单元。如果单元的相对材料密度小于阈值值，则该单元被视为「软」，优化模块会将其从分析中移除。**保持连续性（激进）**选择**保持连续性（激进）**并输入**相对材料密度阈值**以无论连续性如何都移除软单元。如果单元的相对材料密度小于阈值值，则该单元被视为「软」，优化模块会将其从分析中移除。**最大剪切应变**选择**最大剪切应变**并输入**最大剪切应变阈值**。优化模块从分析中移除剪切应变大于阈值的单元。**最小主应变**选择**最小主应变**并输入**最小主应变阈值**。优化模块移除主应变低于阈值的单元。**最大弹塑性应变**选择**最大弹塑性应变**并输入**最大弹塑性应变阈值**。优化模块移除弹塑性应变大于阈值的单元。**体积压缩**选择**体积压缩**并输入**相对体积压缩**。优化模块移除正在压缩且相对体积低于阈值的单元。相对体积 ![](../graphics/usi_eqn00650.gif) 定义为 ![](../graphics/usi_eqn00651.gif)，其中 ![](../graphics/usi_eqn00652.gif) 是变形后的单元体积，![](../graphics/usi_eqn00653.gif) 是原始单元体积。如果您的模型使用壳或膜单元，或者您的模型正在经历大变形，您应该选择**体积压缩**。**注意：**您选择的软删除方法取决于材料行为和单元类型，您可能需要实验来确定最佳方法及其阈值值。文件 `TOSCA.OUT` 包含有关正在被移除的单元的信息，并将帮助您确定最佳软删除方法及其阈值值。**保持连续性**方法提供默认**相对材料密度阈值** 0.05。相比之下，应变和体积方法不提供默认阈值，因为适当值取决于您的模型；例如，取决于材料的属性。
6. 选择**材料插值技术**和**惩罚因子**。优化生成密度接近 1 的硬单元或密度接近 0 的空单元。拓扑优化引入密度介于 1 和 0 之间的单元，材料插值技术计算这些中间单元的密度与刚度之间的关系。**SIMP**（带惩罚的固体各向同性材料）插值方案定义了单元密度与刚度之间的指数关系，适用于静态问题。惩罚因子应大于 `1`，数值实验表明默认值 3 会产生良好结果。**RAMP**（材料属性的有理近似）插值方案适用于动态问题。惩罚因子应大于 0，数值实验表明默认值 3 会产生良好结果。默认情况下，优化模块为静态问题选择 **SIMP** 插值方案，如果您的模型中出现至少一个动态载荷情况，则选择 **RAMP** 插值方案。

### 配置基于条件的拓扑优化任务

基于条件的拓扑优化使用应变能目标函数和体积约束。您可以使用优化任务编辑器来自定义基于条件的拓扑优化的各个方面。要找到编辑器，请从主菜单栏选择****任务**![](../graphics/images/arrow.gif)**编辑**![](../graphics/images/arrow.gif)***优化任务名称*****。要指定基于条件的拓扑优化，请选择**高级**选项卡，然后选择**基于条件的优化**。

#### 配置高级选项

**配置高级选项：**

1. 在优化任务编辑器中，单击**高级**选项卡。
2. 选择**基于条件的优化**算法。
3. 选择是否**删除区域中的软单元**。在拓扑优化过程中，优化模块在设计区域内分配给定的质量，同时尝试满足约束并优化目标。在优化结束时，结构包含硬（填充）单元和软（空）单元。软单元对结构刚度的影响可以忽略不计；但它们仍然与结构的自由度数量相关，因此影响优化过程的速度。**删除软单元**选项允许您选择将仅具有软邻居单元的软单元移除的区域。如果需要，删除的单元会被重新激活；例如，如果力流在优化过程中发生变化。
4. 如果您选择删除软单元，您可以通过选择仅删除具有相邻软单元的软单元来防止孤立软单元被移除。您可以将相邻单元定义为在**平均边长**（默认值）指定的半径内，或在您输入的值指定的半径内。如果网格内单元边长变化很大，则从平均边长计算的半径可能具有误导性。
5. 如果您选择删除软单元，您可以选择优化模块将使用的方法来删除单元：- 选择**标准删除**以在删除软单元之前检查连续性。如果优化模型包含通过软单元与模型其余部分分隔的硬单元「孤岛」，优化模块不会移除软单元。此外，优化模块保留阻止硬单元相对移动的软单元；例如，共享公共边但不共享公共面的硬单元。- 选择**激进删除**以无论连续性如何都删除软单元。
6. 如果需要，输入**相对材料密度阈值**的值。如果单元的相对材料密度小于此值，则该单元被视为「软」，优化模块会将其从分析中移除。
7. 选择优化模块在拓扑优化期间修改单元属性的速率。您可以选择速率（**非常小**、**小**、**中等**、**中**或**大**），并允许优化模块计算满足此速率所需的设计循环数。或者，您可以选择**动态**并输入最大设计循环数。最小设计循环数为 10，默认值为 15。减少设计循环数可能导致优化中出现不期望的效果。尽管生成的结构具有相同的刚度（应变能之和几乎相等），但更改优化速度可能导致解决方案中出现不同的桁架配置。
8. 选择第一个循环后删除的体积。您可以输入百分比或绝对值。默认情况下，优化模块在第一次迭代中移除优化区域体积的 5%。在某些情况下，增加此起始值将加速优化，而不会影响解决方案，特别是在模型的大部分区域存在相对较低应力的情况下。相反，如果起始值太高，优化模块可能在第一次迭代中移除太多单元，导致优化失败或结构粗糙。
