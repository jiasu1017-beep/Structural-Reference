# 20.7 控制草图几何





控制草图几何通过在草图实体之间创建关系来帮助您防止意外更改。 如果草图中的所有几何都受到控制，则草图是完全约束的——草图中的实体在不违反约束的情况下无法修改，并且您无法添加另一个约束而不与已存在的约束产生冲突。

本节描述如何使用约束、尺寸和参数来控制草图几何。 所有三种类型的控制都可以被视为约束，因为它们都用于限制草图中的自由度，但每种控制都在所选实体之间创建不同类型的关系。
