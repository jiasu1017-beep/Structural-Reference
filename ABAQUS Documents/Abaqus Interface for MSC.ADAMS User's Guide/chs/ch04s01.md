# 4.1 示例 1：使用实体单元建模的连杆





本示例使用三维连续体单元对简单的柔性连杆组件进行建模。

**对使用实体单元建模的连杆执行分析：**

1. 输入以下命令从 Abaqus 版本提供的压缩档案文件中提取输入文件：``` abaqus fetch job=adams_ex1 abaqus fetch job=adams_ex1_nodes abaqus fetch job=adams_ex1_elements ```
2. 输入以下命令执行 Abaqus 分析：``` abaqus job=adams_ex1 ```
3. 输入以下命令执行 Abaqus Interface for MSC.ADAMS，并将 Abaqus 分析生成的结果文件转换为用于 ADAMS/Flex 的模态中性文件：``` abaqus adams job=adams_ex1 ```

MSC.ADAMS 四杆机构模型中使用的实体单元连杆模型如图 4-1 所示（参见 [图 4-1](ch04s01.md#adm-link-example)）。

**图 4-1** 实体连杆模型。

![](../graphics/adm-link-example-nls.png)

连杆使用 642 个 C3D10 四面体实体单元（1368 个节点）进行建模。

由于实体单元在其节点处只有位移自由度，因此使用多点约束来提供与 MSC.ADAMS 模型中其他组件的连接。沿梁轴线在铰链孔中心添加了两个节点。位于铰链孔面上的 C3D10 节点通过 BEAM 型多点约束连接到额外节点，允许节点在连杆和其他 MSC.ADAMS 组件之间传递力和力矩。

用于定义单个子结构的选项如 ["Abaqus 子结构模型" 第 2.1 节](ch02s01.md) 中所述。计算了 20 个固定界面振动模态来表示连杆的动力学行为。

MSC.ADAMS 使用固定界面振动模态和约束模态来表征连杆的柔性。Abaqus 计算的 8 个最低固定界面振动频率如 [表 4-1](ch04s01.md#table-admsolidlink1) 所示。这些频率在 `adams_ex1.dat` 文件中报告。
