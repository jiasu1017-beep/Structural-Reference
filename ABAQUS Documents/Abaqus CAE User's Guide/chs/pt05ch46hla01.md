# 46.2 材料取向图选项概述

您可以使用材料取向图选项来自定义材料取向三轴的外观。从主菜单栏中选择****Options**![](../graphics/images/arrow.gif)**Material Orientation****，或单击工具箱中的![](../graphics/ico_plotMatOrientOptions.png)访问**Material Orientation Plot Options**对话框。使用对话框中的选项来控制材料取向三轴的可见性、颜色、长度、厚度和箭头外观。

渲染样式、边缘可见性、变形比例因子和透明度等其他选项位于[**Common Plot Options**](pt05ch43hla01.md)对话框中。如果您选择在未变形和变形形状上绘制材料取向，可以使用[**Superimpose Plot Options**](pt05ch43hla02.md)来自定义未变形形状的外观。有关自定义叠加图的更多信息，请参见["叠加变形和未变形模型图，" 第43.6节](pt05ch43hla02.md)。要了解如何自定义材料取向图的渲染样式和其他显示特征，请参见[第55章，"自定义绘图显示"](pt05ch55.md)。有关选择结果步骤的更多信息，请参见["选择特定结果步骤和帧，" 第42.3.1节](pt05ch42s01hlb01.md)。有关选择截面点位置的信息，请参见["选择截面点数据"中的"按类别选择截面点数据，" 第42.5.9节](pt05ch42s03s01.md#usv-res-visvariableshelldb)。
