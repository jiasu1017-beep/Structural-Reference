# 58 解析场工具集







Abaqus/CAE 提供两种类型的解析场：表达式场和映射场。您可以使用解析场为选定的属性、载荷、相互作用和预定义字段定义空间变化值，例如压力载荷在某个区域上的压力变化。解析场工具集允许您在属性模块、相互作用模块或载荷模块中创建和管理解析场。本章涵盖以下主题：

- ["使用解析表达式场，" 第 58.2 节](pt06ch58s02.md)
- ["使用解析映射场，" 第 58.3 节](pt06ch58s03.md)
- ["显示使用解析场的相互作用和规定条件的符号，" 第 58.4 节](pt06ch58s04.md)
- ["显示符号以可视化映射源数据，" 第 58.5 节](pt06ch58s05.md)

- ["创建映射场，" 第 58.7 节](pt06ch58s06.md)
