# 63.2 创建离散场






离散场工具集可在"属性"模块、"相互作用"模块和"载荷"模块中使用。从主菜单栏选择****工具**![](../graphics/images/arrow.gif)**离散场**![](../graphics/images/arrow.gif)**创建****可创建离散场。

**创建离散场的步骤：**

1. 从主菜单栏选择****工具**![](../graphics/images/arrow.gif)**离散场**![](../graphics/images/arrow.gif)**创建****。 **提示：**您也可以在**离散场管理器**中点击**创建**。
2. 在**名称**文本框中，输入离散场的名称。有关对象命名的信息，请参见["使用基本对话框组件，" 第3.2.1节](pt01ch03s02s01.md)。
3. 在**描述**文本框中，输入离散场的描述。
4. 选择**单元**或**节点**以指定场将应用到的位置类型。
5. 选择数据类型。 - 选择**标量**以输入标量场数据。 - 对于与单元关联的场，**方向**选项可用。选择**方向**，然后执行以下操作： 1. 选择坐标系类型---**笛卡尔**、**圆柱**或**球形**。 2. 切换**提供的方向定义在零件空间中**以指定方向定义在零件或零件实例级别。当您在零件空间中定义方向时，Abaqus/CAE使用零件实例变换将方向值从零件坐标系转换到装配坐标系。关闭此选项可指定方向在装配级别定义。 - 选择**边界条件**以在场数据中包含自由度值。
6. 输入默认分量值。有关Abaqus如何使用默认值的信息，请参见["评估离散场，" 第63.3节](pt06ch63hla02.md)。标量离散场需要一个默认分量值。方向离散场需要六个默认分量值。边界条件离散场需要六个默认分量值和多达六个自由度值。
7. 在**场数据**表中，输入以下值： - 单元或节点编号及其关联的分量值。在创建将与装配级或历史对象（如载荷或相互作用）一起使用的离散场时，必须指定节点或单元编号的完整名称，如["定义装配中的命名约定，" Abaqus Analysis User's Guide第2.10.1节](../usb/usb-link.md#usb-int-ipartassy-naming)中所述。例如，您需要使用*零件实例名称.EID*表示法指定零件实例名称和单元ID，如`PART-2--1.20`。 - 对于边界条件离散场，输入自由度值。在表格上点击鼠标按钮3可以看到一个菜单，允许您执行以下操作：- 剪切、复制和粘贴值 - 插入和删除行 - 清除表格单元格或整个表格 - 读取或写入逗号分隔的文件 有关更多信息，请参见["输入表格数据，" 第3.2.7节](pt01ch03s02s07.md)。
8. 点击**确定**创建离散场并退出编辑器。
