# 15.13.13 定义模型更改相互作用





模型更改相互作用允许您停用和重新激活单元，以模拟模型部分的移除，可以是临时移除或分析剩余部分的移除。您可以在除静态、Riks步骤和线性扰动步骤之外的所有Abaqus/Standard步骤中创建模型更改相互作用。详细讨论请参见"Abaqus Analysis User's Guide"第11.2.1节"Element and contact pair removal and reactivation"。

**定义模型更改相互作用：**

1. 从主菜单栏中，选择****相互作用**![](../graphics/images/arrow.gif)**创建**。**提示：**您也可以使用相互作用模块工具箱中的![](../graphics/ico_interaction.png)工具创建模型更改相互作用。
2. 在出现的**创建相互作用**对话框中，执行以下操作：- 为相互作用命名。有关命名对象的更多信息，请参见"使用基本对话框组件"，第3.2.1节。- 选择将创建相互作用的步骤。- 选择**模型更改**类型的相互作用。
3. 单击**继续**关闭**创建相互作用**对话框。将出现**编辑相互作用**对话框。
4. 指定模型更改定义。- 选择**区域**以为当前仿真定义模型更改相互作用的区域。- 选择**重启**以允许在后续重启分析中进行单元或接触模型更改。当不存在其他模型更改相互作用时，请使用此模型更改定义。如果您在第一个分析步骤中停用了接触对，或者在第一个分析步骤之后创建了接触对，请勿使用此定义。
5. 如果您选择了**区域**模型更改定义，请执行以下步骤：1. 选择区域类型和区域。- 选择**几何**以使用模型几何作为模型更改区域，单击![](../graphics/ico_selectBlue.png)选择区域，并在视口中进行选择。- 选择**外皮**以使用外皮作为模型更改区域，然后单击![](../graphics/ico_selectBlue.png)选择区域。从提示区域中选择**（拾取整个外皮）**或**（拾取部分外皮）**，然后在视口中进行选择。如果所选部件有多个外皮，Abaqus/CAE在您进行选择时会在提示区域显示歧义拾取选项。- 选择**纵梁**以使用纵梁作为模型更改区域，然后单击![](../graphics/ico_selectBlue.png)选择区域。从提示区域中选择**（拾取整个纵梁）**或**（拾取部分纵梁）**，然后在视口中进行选择。如果所选部件有多个纵梁，Abaqus/CAE在您进行选择时会在提示区域显示歧义拾取选项。- 选择**单元**以使用单元作为模型更改区域，然后单击![](../graphics/ico_selectBlue.png)选择区域。从提示区域中选择**单独**或**按角度**，然后在视口中进行选择。必须可见网格才能选择单元。有关此选择方法的更多信息，请参见"使用角度和特征边缘方法选择多个对象"，第6.2.3节。对于所有区域类型，您可以使用现有集合来定义区域。在提示区域右侧，单击**集合**。从出现的**区域选择**对话框中选择有效的集合，然后单击**继续**。默认选择方法基于您最近使用的选择方法。若要恢复为旧方法，请单击提示区域右侧的**在视口中选择**或**集合**。2. 选择区域单元的激活状态。- 选择**在此步骤中停用**以在当前步骤中停用所选区域。- 选择**在此步骤中重新激活**以在当前步骤中激活所选区域。在后续步骤中，您可以选择此选项以重新激活先前停用的所选区域。3. 如果您选择了**在此步骤中重新激活**，您可以切换打开**重新激活具有应变的单元（如果适用）**以包括重新激活的应力/位移单元的应变。关闭此选项可将单元重置为初始应变配置。
6. 单击**确定**创建相互作用并关闭编辑器。代表您刚创建的模型更改相互作用的符号出现在视口中。有关更多信息，请参见"理解代表相互作用、约束和连接器的符号"，第15.10节。
