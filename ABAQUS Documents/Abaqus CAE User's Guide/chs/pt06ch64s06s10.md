# 64.6.9 合并三角形单元







从主菜单栏中选择 ****Mesh**![](../graphics/images/arrow.gif)**Edit**** 可将两个相邻的三角形单元合并为一个四边形单元，如下图所示：

![](../graphics/edt-meshedit-combinetris.png)

您可以合并一阶或二阶单元，但不能合并两者的组合。此外，只有当相邻三角形单元的法线方向一致时才能进行合并。如果需要，您可以使用 Edit Mesh 工具集翻转法线；有关更多信息，请参见["反转壳单元的表面法线方向，" 第 64.6.3 节](pt06ch64s06s03.md)。

**合并三角形单元：**

1. 进入 Mesh 模块，并执行以下操作之一：- 从上下文栏的 **Object** 字段中选择 **Assembly**。- 从上下文栏的 **Object** 字段中选择 **Part**，然后从列表中选择一个零件。
2. 从主菜单栏中选择 ****Mesh**![](../graphics/images/arrow.gif)**Edit****。Abaqus/CAE 显示 **Edit Mesh** 对话框。**提示：**您也可以使用位于 Mesh 模块工具箱底部的 ![](../graphics/ico_meshEdit.png) 工具来显示 **Edit Mesh** 对话框。
3. 在对话框中执行以下操作：1. 在 **Category** 字段中选择 **Element**。2. 从 **Method** 列表中选择 **Combine (tri to quad)**。
4. 选择要合并的第一个三角形单元。
5. 选择要合并的第二个三角形单元。第二个单元必须与第一个相邻，且第二个单元的阶数必须与第一个相同。Abaqus/CAE 显示它如何将两个相邻三角形单元合并为一个四边形单元，并提示您继续。
6. 点击 **Yes** 以合并单元。Abaqus/CAE 提示您选择下一组要合并的单元。**提示：**如果在合并三角形单元时出错，点击 **Edit Mesh** 对话框中的 **Undo** 可撤消更改。Abaqus/CAE 会恢复两个三角形单元。
7. 完成合并单元后，在提示区域点击鼠标按钮 2 或取消按钮 ![](../graphics/afxI_cancel.png) 以退出该过程。
