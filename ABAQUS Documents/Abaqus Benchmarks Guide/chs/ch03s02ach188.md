# 3.2.15 内压下厚壁圆筒的蠕变





**产品：** Abaqus/Standard

本问题是高温蠕变分析的示例。响应的稳态部分有精确解可用；因此，本情况为这种类型的蠕变分析的Abaqus能力提供了一些验证。

### 问题描述

问题如图3.2.15-1所示。结构是一个圆筒，内半径为25.4 mm（1英寸），外半径为50.8 mm（2英寸）。假定圆筒处于平面应变条件（轴向应变为零），解是一维的（与轴向位置无关）且轴对称的。因此，单排轴对称单元就足够了。使用五个等大小的CAX8R单元。没有进行网格收敛研究，但与精确弹性和稳态解的比较表明，这种离散化提供了准确的应力预测。

材料假定为各向同性弹性，杨氏模量为138 GPa（20×10⁶ lb/in²），泊松比为0.3，具有Mises蠕变势，单轴蠕变行为定义为

### 加载和控制

圆筒承受快速施加的60 MPa（8700 lb/in²）内压，并长时间保持，以达到稳态蠕变条件。

压力的初始施加假定发生得很快，涉及纯弹性响应，这是通过使用静态过程获得的。然后在第二步中使用准静态过程开发蠕变响应。要求180000秒（50小时）的响应，这足以达到稳态条件。在准静态步骤中，需要容差来控制时间增量选择，从而控制瞬态蠕变解的准确性。在本例中，我们假定需要中等准确性。应力误差约0.7 MPa（100 lb/in²）将对增量内添加的蠕变应变产生很小的影响。通过将其除以弹性模量将此应力误差转换为应变误差，得到容差为5×10⁻⁶。通过减小此容差可以获得蠕变本构模型积分的更高准确性，但代价是使用更多时间增量。或者，使用大的容差值将允许Abaqus使用可能的最大时间增量，因此在瞬态期间准确性较低，但将以最小成本达到稳态解。因此，如果稳态解是唯一感兴趣的解部分，则将容差设置为一个大的数字是有效的。

对于准静态过程中规定的容差，Abaqus使用自动时间增量。该方案相当简单，旨在随着解向稳态推进而逐渐增加时间增量。在像这样的小位移情况下，显式积分蠕变本构模型通常是有效的，因为该方法每个时间增量成本不高（因为不需要形成和求解新的刚度矩阵），并且其稳定性极限通常与解中感兴趣的时间相比相当大。自动时间步进方案包括稳定性极限的内部计算，时间增量被控制保持在此极限内。如果这太严格——如果它导致一系列时间增量都远小于与准静态过程关联的数据行上要求的剩余时间段的2%（连续10个增量中时间增量小于剩余时间的2%）——Abaqus自动切换到无条件稳定的隐式时间积分方案。时间增量选择的唯一限制是容差规定的准确性。用户可以在准静态过程中抑制这种隐式积分的切换。在本例中，隐式积分的切换发生在第44增量时，在270108秒（7.53小时）的蠕变之后。这允许Abaqus在解的后期选择大的时间增量（最多39600秒，或11小时）。
