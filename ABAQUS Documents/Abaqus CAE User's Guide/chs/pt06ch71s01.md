# 71.1 理解查询工具集的作用







查询工具集允许您获取有关模型的信息。在大多数情况下,Abaqus/CAE在消息区域显示请求的信息,相同的信息也会写入回放文件。选择****工具**![](../graphics/images/arrow.gif)**查询****从主菜单栏使用查询工具集,或选择**查询**工具栏中的![](../graphics/ico_info.png)工具。

查询对话框分为两部分。对话框顶部包含在除Job模块外的每个模块中可用的常规查询,查询工具集在Job模块中完全不可用。对话框底部包含特定于模块的查询;当您在模块之间切换时,Abaqus/CAE显示适合当前模块内容的查询。
