# 41 Plug-in registration commands







Plug-in commands register kernel and GUI plug-ins in either the **Plug-ins** menu or in a toolbox. Plug-in registration commands for accessing both kernel and GUI plug-ins are located in [Chapter 2, "Plug-in registration commands"](../gui/gui-link.md#gui-obj-plg-pyc) of the [Abaqus GUI Toolkit Reference Guide](../gui/gui-link.md#gui).
