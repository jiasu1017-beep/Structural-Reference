# 20.15 将边投影到草图上







绘制特征的轮廓时，您可以通过将现有边从零件投影到草图平面上来创建新边。您可以使用投影边来复制现有特征从当前草图平面看起来的形状。要将零件边（包括孤立单元边）投影到您的草图上，请从主菜单栏中选择****添加**![](../graphics/images/arrow.gif)**边****。

**投影边：**

1. 从 Sketcher 工具箱中，选择**投影边**工具![](../graphics/ico_sketchProjectEdges.png)。有关 Sketcher 工具箱中工具的图表，请参见["Sketcher 工具，"第 20.4.1 节](pt03ch20s04s01.md)。Abaqus/CAE 在提示区域显示提示以指导您完成该过程。
2. 如有需要，在提示区域关闭**约束到背景**以将边作为独立对象投影。如果投影边被约束到背景，它们的形状和位置取决于原始特征边。如果投影边不受约束，对原始特征的更改不会影响它们——您可以独立于原始特征编辑投影边和顶点。
3. 选择要投影到草图平面上的边。
4. 点击鼠标按钮 2 表明您已完成选择边。Abaqus/CAE 在草图中创建新边。
