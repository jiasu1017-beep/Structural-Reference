# 16.11.3 定义初始应力





您可以为模型的某个区域创建初始应力场。您只能在初始步骤中创建或修改初始应力场。有关更多信息，请参见["Abaqus/Standard和Abaqus/Explicit中的初始条件"章节，《Abaqus Analysis User's Guide》第34.2.1节](../usb/usb-link.md#usb-prc-pinitialcond)。

**创建或编辑应力场的步骤：**

1. 使用以下方法之一显示应力场编辑器： - 要创建新的应力场，请按照["创建预定义场"章节第16.8.3节](pt03ch16s08hlb03.md)中概述的程序进行操作（**类别**：`机械`；**所选步骤的类型**：`应力`）。 - 要使用菜单或管理器编辑现有的应力场，请参见["编辑依赖于步骤的对象"章节第3.4.13节](pt01ch03s04hlb05.md)。**注意：**您只能在初始步骤中创建或编辑应力场。
2. 单击**规范**字段右侧的箭头，然后从出现的列表中选择所需的选项： - 选择**直接指定**，然后在数据表中指定应力分量。您最多可以指定六个分量值。 - 选择**从输出数据库文件**从用户指定的输出数据库文件指定应力值。
3. 如果要从输出数据库文件指定初始应力，请执行以下操作： 1. 指定要从中导入应力值的文件名。 2. 从**步骤**和**增量**字段中，指定要从中导入应力值的步骤名称和增量编号。
4. 单击**确定**创建初始应力场并关闭对话框。蓝色圆圈出现在模型上，表示您刚创建的初始应力场。
