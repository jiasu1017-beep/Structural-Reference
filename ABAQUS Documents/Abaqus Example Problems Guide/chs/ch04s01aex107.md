# 4.1.3 限位器臂机构

**产品：** Abaqus/Standard   

本示例说明了使用连接器元件对限位机构（即两个固体接触）进行建模。

### 几何与模型

限位器用于缓冲飞机上整流罩的关闭。整流罩通过在固定点绕 Z 轴施加扭矩来打开和关闭。限位器臂组件通过 C 点连接到整流罩。限位器臂在 A 点被弹簧约束，并允许绕 C 点相对于整流罩旋转。弹簧的另一端连接到整流罩上的 E 点。鞋状件通过 B 点连接到限位器臂，并在罩关闭时与滑块接触。鞋状件允许绕 B 点旋转。当罩打开时，弹簧以 50 lbs 的初始力将限位器臂保持靠在整流罩上的挡块上。当罩关闭时，鞋状件与滑块接触。

一个小旋转弹簧（10 lb/rad）使鞋状件在未与滑块接触时不会相对于限位器臂旋转。滑块固定在地面上。当鞋状件接触时，两个表面相对滑动。滑块和鞋状件之间的摩擦系数为 0.5。

整流罩的关闭速度足够慢，可以忽略惯性效应。

### 模型相互作用

模型中的所有刚体使用显示体建模，连接到相关连接器节点。SHOE（可变形单元）和 BLOCK（刚性表面）例外于此规则，以允许定义这些刚体之间的接触相互作用。有些模型包括连接器中的摩擦和塑性效应。

### 结果与讨论

此类分析不仅允许研究限位器臂机构的动力学，还允许研究滑块和鞋状件之间的接触相互作用。

### 输入文件

- snubber_model.py - Python 回放文件
- snubber.inp - 限位器臂机构模型
- snubber_surf.inp - 带表面到表面接触的模型
- snubber_fric.inp - 带摩擦的模型
- snubber_plas.inp - 带连接器塑性的模型
