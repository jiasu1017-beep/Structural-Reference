# 19.10.6 Controlling adaptivity process precision







To locate the adaptivity process editor, select ****Adaptivity**![](../graphics/images/arrow.gif)**Edit**![](../graphics/images/arrow.gif)***adaptivity process name***** from the main menu bar. Use the **Precision** tabbed page to control the precision of nodal output. You can select either single or full nodal output precision. This option is equivalent to the `output_precision` option in the Abaqus execution procedure. For more information, see ["Abaqus/Standard, Abaqus/Explicit, and Abaqus/CFD execution," Section 3.2.2 of the Abaqus Analysis User's Guide](../usb/usb-link.md#usb-int-danalysisproc).
