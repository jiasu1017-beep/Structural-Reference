# 10.9 减少体积锁定的技术







在橡胶材料模型中引入了少量可压缩性以缓解体积锁定。只要可压缩性很小，近不可压缩材料获得的结果将与完全不可压缩材料获得的结果非常相似。

通过将材料常数 ![](../graphics/gsk_eqn00292.gif) 设置为非零值来引入可压缩性。选择该值以使初始泊松比，![](../graphics/gsk_eqn00293.gif)，接近 0.5。["橡胶样材料的超弹性行为，" Abaqus Analysis User's Guide 第 22.5.1 节](../usb/usb-link.md#usb-mat-chyperelastic) 中给出的方程可用于将 ![](../graphics/gsk_eqn00292.gif) 和 ![](../graphics/gsk_eqn00293.gif) 与 ![](../graphics/gsk_eqn00294.gif) 和 ![](../graphics/gsk_eqn00295.gif)（分别为初始剪切和体积模量）联系起来，用于应变能势的多项式形式。例如，从测试数据（参见 ["示例：轴对称支架，" 第 10.7 节](ch10s07.md) 中的"超弹性材料参数"）获得的超弹性材料系数为 ![](../graphics/gsk_eqn00296.gif) = 176051 和 ![](../graphics/gsk_eqn00297.gif) = 4332.63；因此，设置 ![](../graphics/gsk_eqn00298.gif) = 5.E7 得到 ![](../graphics/gsk_eqn00299.gif) = 0.46。
