# 34.4.7 孔隙流体流速









**产品：** Abaqus/Standard

##### **参考资料**

- [*FLOW*](#)
- ["施加载荷：概述," 第34.4.1节](pt07ch34s04aus120.md)

### 概述

孔隙流体流速（FLOW）用于模拟多孔介质中的流体流动问题。这种载荷类型主要用于 Geostatic 分析或耦合孔隙介质-位移分析。

### 孔隙流体流速定义

孔隙流体流速可以施加于节点或单元：

| **输入文件用法：** | ```
*FLOW
node_or_element_set, dof, magnitude
``` |
| --- | --- |

### 参数说明

| 参数 | 描述 | 适用范围 |
| --- | --- | --- |
| node_or_element_set | 节点编号、节点集或单元集 | 节点或单元 |
| dof | 自由度编号 | 取决于分析类型 |
| magnitude | 流速大小 | 实数 |

### 使用限制

- FLOW 载荷仅适用于 Abaqus/Standard
- 需要定义孔隙介质材料属性
- 必须使用耦合孔隙介质-位移分析步

### 与孔隙压力耦合

在耦合孔隙介质分析中：

| **输入文件用法：** | ```
*COUPLED PORE FLUID-DISPLACEMENT
...
*FLOW
element_set, 1, flow_rate
``` |
| --- | --- |

### 时变流速

可以使用幅值曲线定义随时间变化的流速：

| **输入文件用法：** | ```
*FLOW, AMPLITUDE=name
element_set, dof, magnitude
``` |
| --- | --- |
