# 13 装配模块







您使用装配模块来创建和修改装配。一个模型包含一个主装配，主装配由来自模型中的零件实例以及其他模型的实例组成。入门教程"使用Abaqus/CAE创建和分析模型的附加技术"（Getting Started with Abaqus: Interactive Edition附录C）包含有关如何使用装配模块创建零件实例并在全局坐标系中相对于彼此定位的示例。本章解释如何使用装配模块内的工具来创建装配。涵盖以下主题：
- "理解装配模块的角色"，第13.1节
- "进入和退出装配模块"，第13.2节
- "使用零件实例"，第13.3节
- "使用模型实例"，第13.4节
- "创建装配"，第13.5节
- "创建零件实例阵列"，第13.6节
- "对零件实例执行布尔运算"，第13.7节
- "理解装配模块中的工具集"，第13.8节
- "使用装配模块工具箱"，第13.9节

- "对零件和模型实例应用约束"，第13.11节
