# 53.1 层压板堆叠绘图概述





层压板堆叠绘图是复合模型所选区域中层的图形表示。实际上，层压板堆叠绘图是所选区域中层的芯样本。区域可以来自复合叠层或复合截面。

您可以自定义层压板堆叠绘图的外观。例如，图像可以显示层的顺序、每层中纤维的方向、每层中的材料、层的相对厚度以及参考面的位置。[图 53–1](pt05ch53hla01.md#usv-stack-overview) 说明了层压板堆叠绘图。

**图 53–1** 层压板堆叠绘图。

![](../graphics/usv-stack-overview-nls.png)

阶梯状外观没有对应的物理意义；它只是一种允许您查看区域中所有层的机制。例如，层不会在每个楼梯处终止。

层压板堆叠绘图中的三轴表示单元取向系统，它显示壳法线或堆叠方向（3 方向）以及层的 1 和 2 方向。纤维始终绘制在 1–2 平面中，与 1 方向成一定角度。在实体复合叠层中，一层中的纤维并不总是平行于 1–2 平面运行（例如，如果层取向的 3 方向与单元堆叠方向未对齐）。在这种情况下，层压板堆叠绘图中的纤维并不是复合层中纤维的真实描绘，而是复合截面或叠层定义中旋转角度的图形表示：层压板堆叠绘图中绘制的角度是层表中指定的相对于单元堆叠方向轴测量的旋转角度。有关更多信息，请参阅 ["理解复合叠层和取向，" 第 23.3 节"](pt04ch23s03.md)。

对于基于用户定义坐标系或旋转角度分布的层取向，层压板堆叠绘图不会绘制纤维。Abaqus/CAE 在此类层上显示星号（对于坐标系）或脱字号（对于旋转角度分布），以表示它无法在 1–2 平面中绘制准确表示层纤维方向的线。同样，如果您使用离散场分布来定义层的厚度，Abaqus/CAE 会根据叠层中其他均匀层的平均厚度在层压板堆叠绘图中绘制该层，并在绘图旁边显示离散场名称（如果厚度标签已打开）。

当您使用传统壳单元建模复合层时，了解参考面的位置很重要。网格将定位在参考面上，接触是相对于参考面定义的。层压板堆叠绘图允许您查看参考面相对于复合层中其他层的位置，如图 [图 53–2](pt05ch53hla01.md#usv-stack-offset) 所示。

**图 53–2** 层压板堆叠绘图中的参考面。

![](../graphics/usv-stack-offset-nls.png)

您可以在创建复合叠层或将复合截面分配给区域后在属性模块中显示层压板堆叠绘图。您也可以在分析具有复合叠层或复合截面的模型后在可视化模块中显示层压板堆叠绘图。有关更多信息，请参阅本指南 HTML 版本中的 ["定义复合叠层，" 第 12.2.4 节"](pt03ch12s02s04.md) 和 ["创建复合壳截面，" 第 12.13.7 节"](pt03ch12s13s02.md)。

要生成层压板堆叠绘图，请从主菜单栏中选择 ****工具（Tools）**![](../graphics/images/arrow.gif)**查询（Query）****，并从出现的对话框中选择 **层压板堆叠绘图（Ply stack plot）**。显示层压板堆叠绘图后，您可以通过从提示区域点击 **层压板堆叠绘图选项（Ply Stack Plot Options）** 按钮或通过点击工具箱中的 ![](../graphics/ico_plotPlyStackOptions.png) 工具（在可视化模块中）来自定义其外观。有关自定义层压板堆叠绘图的详细说明，请参阅 ["自定义层压板堆叠绘图，" 第 53.2 节"](pt05ch53s01.md)。

# 53.2.2 自定义基本层压板堆叠绘图选项





点击 **基本（Basic）** 选项卡来选择要在层压板堆叠绘图中显示哪些层以及层的排列方式。如果您的复合叠层包含很多层，层压板堆叠绘图可能会变得混乱且难以解释。为了使层压板堆叠绘图更易读，基本层压板堆叠绘图选项允许您仅查看可管理的层数。

**自定义基本层压板堆叠绘图选项：**

1. 找到基本选项。1. 从提示区域点击 **层压板堆叠绘图选项（Ply Stack Plot Options）** 按钮，或执行以下操作之一：- 在属性模块中，选择 ****视图（View）**![](../graphics/images/arrow.gif)**层压板堆叠绘图选项（Ply Stack Plot Options）****。- 在可视化模块中，选择 ****选项（Options）**![](../graphics/images/arrow.gif)**层压板堆叠绘图（Ply Stack Plot）**** 或点击工具箱中的 ![](../graphics/ico_plotPlyStackOptions.png) 工具。2. 在出现的 **层压板堆叠绘图选项（Ply Stack Plot Options）** 对话框中点击 **基本（Basic）** 选项卡。
2. 从 **显示（Display）** 选项中，执行以下操作：- 选择渲染样式——着色、填充或线框。- 选择是否显示每层的边缘以及边缘的样式和粗细。- 选择是将层显示为从最后一层到第一层下降的"楼梯"图像，还是关于中心层或多个层对称的图像。
3. 从 **可见层（Visible Plies）** 选项中，选择要显示的层。当您修改显示参数时，Abaqus/CAE 会更新层压板堆叠绘图，这允许您快速方便地显示复合层中的层。您可以选择以下内容：- 显示从 **起始（Start）** 层开始到 **结束（End）** 层结束的指定层。- 显示从 **起始（Start）** 层开始的 **数量（Number）** 层。点击 **按数量递增（Increment by Number）** 按 **数量（Number）** 的层数递增 **起始（Start）** 值，而不是按 1 递增。这允许您的层压板堆叠绘图作为一个窗口增量地移动穿过叠层，每个增量显示下一组相邻的层。- 显示以 **中心（Center）** 层为中心的 **数量（Number）** 层。点击 **按数量递增（Increment by Number）** 按 **数量（Number）** 的层数递增 **中心（Center）** 值，而不是按 1 递增。这允许您的层压板堆叠绘图作为一个窗口增量地移动穿过叠层，每个增量显示下一组相邻的层。- 显示以 **中心（Center）** 层为中心的 **数量（Number）** 层。拖动 **中心（Center）** 滑块在叠层中移动。
4. 选择 **偶数（Even）** 和 **奇数（Odd）** 层的颜色：1. 点击颜色样本 ![](../graphics/usi-colorsample.png)。Abaqus/CAE 显示 **选择颜色（Select Color）** 对话框。2. 使用 **选择颜色（Select Color）** 对话框中的方法选择新颜色。有关更多信息，请参阅 ["自定义颜色，" 第 3.2.9 节"](pt01ch03s02s09.md)。3. 点击 **确定（OK）** 关闭 **选择颜色（Select Color）** 对话框。颜色样本变为所选颜色。Abaqus/CAE 相对于所显示的层应用您的颜色选择，而不是相对于实际层号。例如，如果您显示第 2、3 和 4 层，Abaqus/CAE 会将 **奇数（Odd）** 颜色应用于第 2 和 4 层，将 **偶数（Even）** 颜色应用于第 3 层。
5. 点击 **应用（Apply）** 实施您的更改。层压板堆叠绘图会更改以反映您的规范。默认情况下，您的更改将在会话期间保存，并将影响该视口中所有后续层压板堆叠绘图。如果您想在后续会话中保留您的更改，请将它们保存到文件。有关更多信息，请参阅 ["保存自定义以供后续会话使用，" 第 55.1.1 节"](pt05ch55s01s01.md)。
