# 82.10 从 stereolithography (STL) 格式文件导入模型





从主菜单栏选择 ****Plug-ins**![](../graphics/images/arrow.gif)**Tools**![](../graphics/images/arrow.gif)**STL Import**** 以从外部 stereolithography (STL) 格式文件（`*.stl`）将零件导入 Abaqus/CAE。**STL Import** 插件将所有零件定义和模型数据转换为 Abaqus 格式，创建一个新的 Abaqus/CAE 模型和一个新的 Abaqus 输入文件。

实际上，您可能希望将导入的 STL 模型数据添加到现有的 Abaqus/CAE 模型中。导入后将 STL 零件或零件实例添加到现有 Abaqus/CAE 模型的方法是，使用 **Copy Objects** 对话框将这些对象从新创建的模型复制到任何其他 Abaqus/CAE 模型。有关复制模型对象的更多信息，请参见 ["在模型之间复制对象，" 第9.8.3节](pt02ch09s07hlb03.md)。

您可以在 Windows 或 Linux 平台上导入以 ASCII 格式保存的 STL 文件。您只能在 Windows 平台上导入以二进制格式保存的 STL 文件。

**要将模型数据从 STL 文件导入 Abaqus/CAE：**

1. 从主菜单栏中，选择 ****Plug-ins**![](../graphics/images/arrow.gif)**Tools**![](../graphics/images/arrow.gif)**STL Import****。出现 **STL Import** 对话框。
2. 指定要从中导入模型数据的 STL 文件的名称。Abaqus/CAE 会用您选择的 STL 文件的名称填充 **Model name** 字段。
3. 如果需要，编辑 **Model name** 字段中为将创建的新 Abaqus/CAE 模型和新 Abaqus 输入文件指定的默认模型名称。
4. 如果需要，指定节点合并容差值。此容差必须为正。
5. 点击 **OK** 导入模型数据。插件转换 stereolithography 文件中的模型数据，并在当前模型数据库中创建一个 Abaqus/CAE 模型，在当前工作目录中创建一个 Abaqus 输入文件。如果任何 STL 数据无法转换，插件会在日志文件中记录有关未转换数据的错误或警告。
