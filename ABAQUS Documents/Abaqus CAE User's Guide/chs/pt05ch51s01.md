# 51.1 理解探测





当您在可视化模块工具箱中单击![](../graphics/ico_probe.png)工具时，Abaqus/CAE进入探测模式，当您将光标在当前视口中的模型上移动时，会在"探测值"对话框中显示信息。您也可以使用查询工具栏中的![](../graphics/ico_info.png)工具进入探测模式。探测模型图会显示模型数据和分 析结果；探测X–Y曲线图会显示X–Y曲线数据；探测当前模型数据库中的模型会显示模型数据以及属性（如载荷和预定义场）的值。您可以将此信息写入文件。

"探测值"对话框底部的数据表包含两个动态显示行（标题行正上方的行和首先出现在表中的粗体数据行）以及一系列额外的存储行。Abaqus/CAE不断更新动态行，以显示当您在当前视口中将光标移到模型上时的探测值，您可以通过单击列标题来更改顶部显示行中显示的值。仅当您通过在视口中选择、在提示区域指定节点或单元标签或在特定显示组中选择节点或单元来选择感兴趣的值时，Abaqus/CAE才会用额外行更新表格。值在对话框底部累积，直到您删除它们、清除表格或取消探测模式。您还可以将所选值添加到新显示组，并且可以为单个节点或单元标注场输出值。
