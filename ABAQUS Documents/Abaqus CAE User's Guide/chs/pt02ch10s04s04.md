# 10.4.4 导出到IGES文件





CAD应用程序使用自己对IGES标准的解释将数据存储在IGES格式文件中。Abaqus/CAE能够解释大多数应用程序生成的IGES格式文件。此外，当您将零件或装配导出为IGES格式文件时，Abaqus/CAE允许您指定将读取文件的应用程序，并以适当的定制格式或风格写出数据。您可以选择以下风格之一：
- 标准（Standard）
- MSBO
- AutoCAD
- SolidWorks
- JAMA（日本汽车制造商协会）

默认情况下，Abaqus/CAE使用标准风格将数据导出到IGES文件。Abaqus/CAE将几何数据写入IGES文件中的单个层。
