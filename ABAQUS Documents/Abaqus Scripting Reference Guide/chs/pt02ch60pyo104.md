# 60.104 UserOutputVariables 对象





UserOutputVariables 对象指定用户定义输出变量的数量。

**访问**

```
materialApi.materials()[*name*].userOutputVariables()
```

### 60.104.1 UserOutputVariables(...)

此方法创建一个 UserOutputVariables 对象。

**路径**

```
materialApi.materials()[*name*].UserOutputVariables
```

**原型**

```
odb_UserOutputVariables&
UserOutputVariables(int n);
```

**必需参数**

无。

**可选参数**

*n*

一个 Int，指定每个材料点所需的用户定义变量数量。默认值为 0。

**返回值**

一个 UserOutputVariables 对象。

**异常**

RangeError。

### 60.104.2 成员

UserOutputVariables 对象的成员与 [UserOutputVariables](pt02ch60pyo104.md#ker-useroutputvariables-useroutputvariables-cpp) 方法的参数具有相同的名称和描述。
