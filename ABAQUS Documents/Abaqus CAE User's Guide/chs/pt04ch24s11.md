# 24.11 在可视化模块中显示连接器






Abaqus/CAE 在"可视化"模块中显示使用连接器截面分配建模的连接器。您可以使用显示组来控制连接器的显示。每个连接器在输出数据库中列为一个单元集。有关显示组的更多信息，请参见["创建或编辑显示组，" 第 78.2.1 节"](pt07ch78s02hlb01.md)。

您还可以使用**ODB 显示选项**对话框中的**实体显示**选项来控制表示连接器的符号的显示。您可以控制以下内容：
- 线框端点的突出显示
- 连接器局部取向轴的显示
- 连接器类型标签的显示
- 显示符号的大小

有关表示在 Abaqus/CAE 中建模的连接器的符号的信息，请参见["理解表示相互作用、约束和连接器的符号，" 第 15.10 节"](pt03ch15s10.md)。有关控制符号显示的更多信息，请参见["控制模型实体的显示，" 第 55.10 节"](pt05ch55s10.md)。

例如，您可以使用显示组来显示曲柄机构的四个零件实例和三个连接器单元集，如图 [图 24-4](pt04ch24s11.md#adv-conn-viewer) 所示。

**图 24-4** 曲柄机构的"可视化"模块中选定的零件实例、连接器和连接器符号显示。

![](../graphics/adv-conn-viewer.png)

线框端点的突出显示和连接器类型标签的显示已被切换关闭，因此仅显示局部连接器取向。您可以生成时间历史动画，以查看机构运行过程中连接器取向的变化。有关更多信息，请参见["时间历史动画，" 第 49.1.1 节"](pt05ch49s01s01.md)。

有关绘制连接器输出结果的信息，请参见[第 47 章，"X-Y 绘图"](pt05ch47.md)。
