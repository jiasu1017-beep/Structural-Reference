# 69.1 Using the Geometry Edit toolset







You can use the Geometry Edit toolset in the Part module to edit or repair the geometry of a part. The Geometry Edit toolset enables you to edit an imported part to improve its precision and validity; however, Abaqus/CAE does not require parts to be completely precise. In addition, you can use the Geometry Edit toolset to remove small features from a part. You can use the Geometry Edit toolset to edit edges, faces, or the entire part. ["An overview of editing techniques," Section 69.2](pt06ch69s02.md), provides an overview of the methods available for each type of partition. Abaqus/CAE stores most of the edit operations as features. As a result, you can undo an edit by deleting or suppressing the corresponding feature using the Feature Manipulation toolset. 

You can access the edit tools by selecting ****Tools**![](../graphics/images/arrow.gif)**Geometry Edit**** from the main menu and choosing the desired tool from the **Geometry Edit** dialog box that appears. You can also access the Geometry Edit tools through the Part module toolbox. [Figure 69--1](pt06ch69s01.md#rep-toolbox) shows the hidden icons for all the Geometry Edit tools in the Part module toolbox.
