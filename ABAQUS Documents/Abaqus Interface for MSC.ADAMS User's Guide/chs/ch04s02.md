# 4.2 示例 2：使用梁单元建模的连杆





本示例使用三维梁单元对简单的柔性连杆组件进行建模。

**对使用梁单元建模的连杆执行分析：**

1. 输入以下命令从 Abaqus 版本提供的压缩档案文件中提取输入文件：``` abaqus fetch job=adams_ex2 ```
2. 输入以下命令执行 Abaqus 分析：``` abaqus job=adams_ex2 ```
3. 输入以下命令执行 Abaqus Interface for MSC.ADAMS，并将 Abaqus 分析生成的结果文件转换为用于 ADAMS/Flex 的模态中性文件：``` abaqus adams job=adams_ex2 ```

梁模型与实体模型的主要区别在于，梁模型仅使用 10 个 B31 单元（11 个节点）。由于梁单元在其节点处同时具有位移和旋转自由度，因此不需要多点约束来将连杆连接到其他 MSC.ADAMS 组件。模型的其余部分与连杆的实体模型基本相同。

无约束模型的前 8 个非零频率如 [表 4-3](ch04s02.md#table-admbeamlink) 所示。

这些频率与连杆实体模型的频率接近。尽管 Abaqus 中该模型的计算成本比实体模型低得多，但 MSC.ADAMS 中两个模型的计算成本非常相似，因为两个模型都有 32 个模态（12 个约束模态和 20 个固定界面振动模态）。
