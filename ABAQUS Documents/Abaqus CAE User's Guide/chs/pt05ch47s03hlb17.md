# 47.4.17 归一化X–Y数据对象







### 47.4.17 归一化X–Y数据对象

使用 **normalize** 函数对[之前保存的](pt05ch47s02hlb07.md) *X–Y* 数据对象（有序对的集合）进行运算，生成一个新的 *X–Y* 数据对象。此操作类似于归一化向量。新的 *X–Y* 数据对象具有与原始 *X–Y* 数据对象相同的 *X* 坐标值。Abaqus/CAE 通过将每个原始 *Y* 坐标除以所有原始 *Y* 坐标值的均方根（平方和的平方根）来计算新的 *Y* 坐标。

[图47-15](pt05ch47s03hlb15.md#usv-xyplot-normalize) 说明了可以使用 **normalize** 函数生成的 *X–Y* 曲线图类型。

**图47-15** 使用 **normalize** 函数生成的 *X–Y* 曲线图。

![](../graphics/usv-xyplot-normalize.png)

**归一化X–Y数据对象：**

1. 找到 **Operate on XY Data** 对话框。从主菜单栏，选择 ****Tools**![](../graphics/images/arrow.gif)**XY Data**![](../graphics/images/arrow.gif)**Create****. 在出现的对话框中点击 **Operate on XY data**；然后点击 **Continue**。**Operate on XY Data** 对话框出现。
2. 从列出的 **Operators** 中，点击 **normalize(X)**。**normalize** 函数出现在表达式窗口中。
3. 从 **XY Data** 选项中，点击要操作的 *X–Y* 数据对象的名称，然后点击 **Add to Expression**。您可以从本次会话中之前保存的所有 *X–Y* 数据对象中选择（在 **XY Data** 字段中按字母顺序列出）。*X–Y* 数据对象名称出现在表达式窗口中 **normalize** 函数的括号内。
4. 要继续构建表达式，将光标定位在表达式窗口中，然后输入或选择要包含的函数、运算符和 *X–Y* 数据。
5. 要评估并显示表达式，点击 **Plot Expression**。
6. 要[保存](pt05ch47s02hlb07.md)您的新 *X–Y* 数据对象，点击 **Save As**，然后在出现的对话框中提供名称。保存数据对象使其可用于本次会话中的未来操作以及包含多个数据对象的 *X–Y* 曲线图。
7. 完成后，点击 **Cancel** 关闭对话框。
