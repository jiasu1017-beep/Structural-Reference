# 43.5 生成变形形状图

变形图根据变形场变量的值，显示模型在分析结果的指定步骤和帧处的形状。有关选择步骤和帧的更多信息，请参阅["选择特定结果步骤和帧，"第 42.3.1 节](pt05ch42s01hlb01.md)。有关选择变形场变量的更多信息，请参阅["选择变形场输出变量，"第 42.5.4 节](pt05ch42s03hlb04.md)。了解如何显示与变形图关联的最小值和最大值，请参阅["自定义图例，"第 56.1 节](pt05ch56s01.md)。

**生成变形形状图：**

1. 使用 **File** 菜单[打开](pt02ch09s06hlb02.md)包含分析结果的输出数据库。
2. 使用 **Result** 菜单选择以下内容：
   1. Abaqus 获取值的步骤和帧。
   2. Abaqus 获取值的变形场变量。分析期间保存的变量可用。
   3. Abaqus 显示复数结果的数字形式。
3. 选择您想要的绘图状态独立[自定义选项](pt05ch40s03s03.md)。
4. 从主菜单栏选择 ****Plot**![](../graphics/images/arrow.gif)**Deformed Shape**** 以显示变形图。
   **提示：**您也可以使用工具箱中的 ![](../graphics/ico_plotDeformed.png) 工具生成变形形状图。当前视口显示输出数据库在指定步骤和帧处指定变形场变量的自定义变形图。如果处于活动状态，状态块会更改以显示所显示变量的名称。每次在结果或绘图状态独立对话框中点击 **Apply** 时，Abaqus 会自动刷新您的变形图。
