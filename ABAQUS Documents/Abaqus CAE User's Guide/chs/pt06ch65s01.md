# 65.1 使用特征操作工具集





您可以通过从主菜单栏选择**特征**，或通过在模型树中的特征上点击鼠标按钮3来访问特征操作工具集。您也可以从模块工具箱访问特征工具。[图65-1](pt06ch65s01.md#features-toolbox)显示了模块工具箱中所有特征工具的图标。

**图65-1** 特征工具。

![](../graphics/features-toolbox-nls.png)

以下列表描述瞭如何使用**编辑**、**重新生成**、**抑制**、**恢复**和**删除**工具来修改特征或控制零件或装配中特征的外观。

**![](../graphics/ico_featureEdit.png) [编辑特征](pt06ch65s04hlb01.md)**

您只能在"零件"模块中编辑零件的特征。您对零件所做的更改会自动应用于该零件在装配中的每个实例。同样，您只能在"装配"和"网格"模块中编辑装配的特征。有关编辑分区和基准几何体时适用的限制的信息，请参见["将基准理解为一个特征，" 第62.4节"](pt06ch62s04.md)和["理解分区，" 第70.3节"](pt06ch70s03.md)。

**![](../graphics/ico_featureRegenerate.png) [重新生成特征](pt06ch65s04hlb06.md)**

重新生成是在修改模型的特征后重新计算模型几何的过程。默认情况下，Abaqus/CAE会在您编辑其特征后自动重新生成零件或装配。但是，您可以通过在**特征编辑器**中关闭或打开**确定时重新生成**选项来控制特征是否自动重新生成。有关更多信息，请参见["编辑特征，" 第65.4.1节"](pt06ch65s04hlb01.md)。

**![](../graphics/ico_featureSuppress.png) [抑制特征](pt06ch65s04hlb03.md)**

当您在复杂模型中操作许多特征时（例如，当您探索设计替代方案时），Abaqus/CAE允许您临时禁用某些特征以简化显示或加速重新生成。被抑制的特征不可见，也不能进行分区或网格划分。此外，Abaqus/CAE在重新生成零件或装配时会忽略所有被抑制的特征。您可以编辑被抑制的特征，尽管您的更改在您恢复特征之前不会对零件或装配产生影响。抑制特征也会抑制该特征的任何子特征。（有关父特征和子特征的信息，请参见["零件和特征之间的关系，" 第11.3.1节"](pt03ch11s03s01.md)。）

Abaqus/CAE不会在提交作业进行分析时生成的输入文件中包含被抑制的特征。但是，Abaqus/CAE会在输入文件中包含引用被抑制特征或其区域的边界条件。为确保分析成功完成，您应该执行以下操作之一：
- 恢复被抑制的特征。
- 编辑边界条件，并将边界条件应用到不同区域。
- 删除边界条件。

如果您想临时使装配中的零件实例不可见，可以通过在**装配显示选项**对话框中选择选项来隐藏零件实例。有关更多信息，请参见["控制实例可见性，" 第76.14节"](pt07ch76hla13.md)。

**![](../graphics/ico_featureResume.png) [恢复特征](pt06ch65s04hlb04.md)**

当您恢复被抑制的特征时，它会重新出现在零件或装配的显示中，并在模型中重新建立。如果您在特征被抑制时对其进行了编辑，恢复特征会导致Abaqus/CAE自动重新生成模型以考虑您的更改。当您恢复特征时，您可以选择也恢复其所有子特征。（您不能在不恢复其父特征的情况下恢复子特征。）

**[调整重新生成性能](pt06ch65s04hlb07.md)**

调整特征重新生成性能是在保存状态的便利性与内存消耗对性能的影响之间取得平衡。在大多数情况下，默认设置将产生可接受的重新生成性能；但是，您可以通过从主菜单栏选择****特征**![](../graphics/images/arrow.gif)**选项****来调整重新生成速度。有关更多信息，请参见["调整特征重新生成，" 第65.3节"](pt06ch65s03.md)。

**![](../graphics/ico_featureDelete.png) [删除特征](pt06ch65s04hlb05.md)**

删除特征会从模型中永久移除该特征。如果您删除父特征，其所有子特征也将被删除且无法恢复。

此外，当您将零件复制到新零件时，可以将所有特征和参数信息 reduce 为零件的简单定义。如果在复制零件时减少特征列表，Abaqus/CAE会在您随后修改它时更快地重新生成新零件；但是，您将不再能够修改新零件的任何参数。要复制零件，请在"零件"模块中从主菜单栏选择****零件**![](../graphics/images/arrow.gif)**复制****。
