# 44.5.12 自定义云图区间





Abaqus/CAE将[最小和最大云图值](pt05ch44s02hlb07.md)之间的范围均匀划分为区间。您可以自定义区间数量以及区间是用离散颜色还是连续颜色表示。默认情况下，有12个离散颜色区间。[图44-8](pt05ch44s02hlb12.md#usv-cont-intervals)显示了一个带型云图，左侧有九个均匀云图区间，右侧有连续云图。

**图44-8** 显示均匀和连续云图区间的云图。

![](../graphics/usv-cont-intervals.png)

当您使用离散区间表示云图时，您还可以控制区间值之间的 progression 类型。如果您选择均匀区间类型，Abaqus/CAE会在区间值之间创建均匀的算术级数；如果您选择对数区间类型，Abaqus/CAE会在区间值之间创建对数级数。此外，您可以创建一组包含您希望包含的任何值的新云图区间。您还可以仅提供几个区间值，然后在您定义的区间之间插入其他区间。您提供的区间值必须按从下到上的顺序在**Edit Intervals**对话框中以降序显示。

如果您选择自定义区间，Abaqus/CAE会覆盖**Contour Plot Options**对话框**Limits**区域中指定的云图限制，并使用最大和最小区间值作为自定义云图限制。有关更多信息，请参见["设置云图限制，" 第44.5.7节](pt05ch44s02hlb07.md)。

云图图例（如果激活）将比您选择的数量多两个区间。Abaqus/CAE在图例顶部和底部添加区间，以指示超出云图限制的任何值。有关云图图例的更多信息，请参见["自定义图例，" 第56.1节](pt05ch56s01.md)。

本节介绍如何自定义云图区间以及如何编辑您使用的云图区间。

**要自定义云图区间：**

1. 找到**Contour Intervals**选项。从主菜单栏中，选择****Options**![](../graphics/images/arrow.gif)**Contour****；然后在出现的对话框中单击**Basic**选项卡。**Contour Intervals**选项位于页面左侧。
2. 选择**Continuous**或**Discrete**以分别请求连续或离散云图区间。**Continuous**区间仅适用于[带型云图](pt05ch44s02hlb01.md)。Abaqus将您的云图值表示为平滑、非勾勒的颜色光谱。当您选择**Discrete**区间时，**Contour Interval**滑块变得可用。
3. 对于**Discrete**区间，执行以下任务：1. 通过将**Contour Intervals**滑块拖动到您想要的区间数量（2到24之间）来选择区间数量。2. 从**Interval type**列表中选择以下选项之一：- 选择**Uniform**以在云图中使用值的算术级数。- 选择**Log**以在云图中使用值的对数级数。- 选择**User-defined**以在云图中使用自定义值集。3. 要自定义**User-defined**区间类型的区间值，请单击![](../graphics/ico_edit.png)。**Edit Intervals**对话框出现，其中包含一组默认云图区间，您可以以下列任何方式进行修改：- 通过单击其行并输入新值来编辑任何区间值。您必须按从对话框上到下的降序提供区间值。- 单击**Insert Before**或**Insert After**以在所选区间之前或之后插入另一个区间。新区间添加时没有值，因此您必须通过直接在相应行中输入值或插值来提供区间值。- 单击**Delete**从集合中删除所选区间。- 单击**Interpolate**以计算集合中所有空白区间的插值区间值。Abaqus/CAE使用算术级数插值。- 单击**Clear Values**清除所选区间的值。当您完成云图区间集后，单击**OK**关闭**Edit Intervals**对话框。
4. 单击**Apply**在视口中实施您的更改。当前视口中的云图和（如果激活的话）[云图图例](pt05ch56s01.md)更改为反映您选择的区间选项。默认情况下，您的更改在会话期间保存，并将影响所有后续云图。如果您想在后续会话中保留更改，请将它们保存到文件。有关更多信息，请参见["保存自定义设置以供后续会话使用，" 第55.1.1节](pt05ch55s01s01.md)。
