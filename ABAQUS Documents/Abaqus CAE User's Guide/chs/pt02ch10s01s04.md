# 10.1.4 导入装配





来自CAD系统（如CATIA）的文件可以包含单个零件或零件装配。Abaqus/CAE允许您从主菜单栏选择****文件（File）**![](../graphics/images/arrow.gif)**导入（Import）****并选择**零件（Part）**或**装配（Assembly）**。两个选项都允许您导入装配中的所有零件；但是，最终结果略有不同。

**导入零件**

如果您选择从包含零件装配的文件导入零件，您可以导入文件中的所有零件或仅导入指定的零件。如果您导入所有零件，Abaqus/CAE会创建一组对应于原始装配中每个零件实例的零件。要重新创建原始装配，您必须使用"装配（Assembly）"模块来实例化每个导入的零件。但是，零件与原始装配中零件实例之间的关系在导入过程中会丢失。例如，如果原始装配包含一个被实例化九次的螺栓，Abaqus/CAE会创建九个相同的零件。当您在"装配"模块中重新创建装配时，Abaqus/CAE会为九个螺栓中的每一个创建零件实例。尽管零件与零件实例之间的关系会丢失，但Abaqus/CAE确实保留了零件的位置。因此，当您实例化每个零件时，它会出现在装配中的正确位置。

**导入装配**

如果您选择导入装配，您可以导入整个装配或仅导入选定的零件实例。Abaqus/CAE会将您的选择追加到现有装配中，并保留实例的原始位置。此外，Abaqus/CAE会创建对应于导入零件实例的零件，并维护零件与其实例之间的关系。例如，如果螺栓在装配中被实例化九次，Abaqus/CAE会导入九个螺栓实例，但只创建一个零件。

导入装配还提供以下优点：
- 在大多数情况下，Abaqus/CAE会保留原始文件中零件和零件实例的名称。
- 如果原始文件中的零件和零件实例由第三方CAD系统着色，Abaqus/CAE会在导入过程中保留这些颜色。有关修改颜色编码的信息，请参阅[第77章，"几何和网格元素的颜色编码"](pt07ch77.md)。
