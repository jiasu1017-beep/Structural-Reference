# 31.3.5 停用和激活XFEM裂缝扩展





默认情况下，XFEM裂缝扩展在所有步骤中都处于活动状态。但是，如果包含XFEM裂缝扩展的分析未能收敛，您可以在所选步骤中停用裂缝，并允许分析在后续步骤中重新激活裂缝之前达到平衡。

**要在步骤中停用和激活XFEM裂缝扩展：**

1. 在初始步骤中创建相互作用。更多信息，请参见["创建相互作用"，第15.12.1节](pt03ch15s12hlb01.md)。
2. 选择**XFEM crack growth**相互作用类型，然后单击**Continue**。
3. 从**Edit Interaction**对话框中，选择您要激活或停用的XFEM裂缝。您只能选择允许裂缝扩展的裂缝。
4. 单击**OK**创建相互作用。
5. 打开**Interaction Manager**。更多信息，请参见["在Interaction模块中管理对象"，第15.9.1节](pt03ch15s09s01.md)。
6. 在所需步骤中选择XFEM裂缝扩展相互作用，然后单击**Edit**。
7. 从**Edit Interaction**对话框中，切换关闭（或打开）**Allow crack growth **以在所选步骤中停用（或激活）XFEM裂缝扩展，然后单击**OK**。**Interaction Manager**显示**Modified**以指示该步骤中XFEM裂缝扩展的状态。Abaqus将XFEM裂缝扩展的状态（激活或停用）传播到后续步骤。
