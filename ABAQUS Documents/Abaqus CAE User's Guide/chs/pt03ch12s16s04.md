# 12.17.4 定义校准行为







本节描述如何从校准数据集提取材料行为常数。您可以为以下材料模型定义校准行为：
- 各向同性弹性
- 各向同性弹塑性
- 带永久变形的超弹性

您还可以添加对自定义校准行为的支持，这些行为将作为新选项出现在**校准行为**对话框中。请参见Dassault Systmes知识库中的"在Abaqus/CAE中创建自定义材料校准插件"获取更多信息，网址为www.3ds.com/support/knowledge-base。

Abaqus/CAE在计算材料属性并从除体积测试外的测试数据绘制材料响应曲线时，假定材料是完全不可压缩的。

### 校准各向同性弹性材料行为数据

各向同性弹性校准行为使您能够从校准数据集中推导各向同性弹性数据（杨氏模量和泊松比），并将这些材料常数应用到模型中材料定义的弹性材料属性。

**校准各向同性弹性材料行为数据：**

1. 从模型树中，展开**校准**容器并双击**行为**。**创建校准行为**对话框出现。
2. 输入材料校准行为的名称，选择**弹性各向同性**，然后单击**继续**。**编辑行为**对话框出现。
3. 从**参数集1**选项中，执行以下操作来计算杨氏模量的值：1. 从**数据集**列表中，选择要从中计算杨氏模量的数据。2. 单击![](../graphics/ico_calculator.png)。Abaqus/CAE计算杨氏模量并将其值显示在**杨氏模量**标签的右侧。
4. 从**参数集2**选项中，执行以下操作来计算泊松比的值：1. 从**数据集**列表中，选择要从中计算泊松比的数据。2. 单击![](../graphics/ico_calculator.png)。Abaqus/CAE计算泊松比并将其值显示在**泊松比**标签的右侧。
5. 从**材料**列表中，选择要应用此校准行为的材料定义；或单击![](../graphics/ico_material.png)为此校准行为创建新的材料定义。有关定义新材料模型的更多信息，请参见"创建或编辑材料"，第12.7.1节。
6. 单击**确定**将各向同性弹性行为保存到所选材料模型。Abaqus/CAE将新行为添加到模型树，并将指定的杨氏模量和泊松比添加到所选材料的**弹性**材料属性中。

![](../graphics/images/black4rule.gif)有关相关信息，请单击以下任一项：- "定义弹性"中的"创建线性弹性材料模型"，第12.9.1节

### 校准各向同性弹塑性材料行为数据

各向同性弹塑性校准行为使您能够推导各向同性弹性和塑性材料行为。

**校准各向同性弹塑性材料行为数据：**

1. 从模型树中，展开**校准**容器并双击**行为**。**创建校准行为**对话框出现。
2. 输入材料校准行为的名称，选择**弹塑性各向同性**，然后单击**继续**。**编辑行为**对话框出现。
3. 从**弹塑性数据**选项中，执行以下操作：1. 展开**数据集**列表并选择要从中计算第一组校准值的数据。2. 从**极限点**选项中，单击![](../graphics/ico_calculator.png)自动计算极限点，或单击![](../graphics/ico_select.png)从视口中选择极限点。Abaqus/CAE在视口中绘制极限点并在对话框中显示其坐标。3. 从**屈服点**选项中，单击![](../graphics/ico_select.png)并从视口中选择屈服点。Abaqus/CAE在视口中在原点和屈服点之间绘制一条线，在对话框中显示屈服点的坐标，并计算杨氏模量并将其值显示在**杨氏模量**标签的右侧。4. 通过执行以下操作之一选择此材料校准的塑性点：- 拖动**塑性点**滑块向右以计算更多塑性点，或向左拖动滑块以计算更少的塑性点。- 单击![](../graphics/ico_select.png)从视口中选择塑性点。Abaqus/CAE将塑性数据点添加到对话框的表中。如果要进一步自定义塑性数据，您可以编辑这些数据中的任何一个。
4. 从**泊松比数据**选项中，执行以下操作：1. 从**数据集**列表中，选择要从中计算泊松比的数据。2. 单击![](../graphics/ico_calculator.png)。Abaqus/CAE计算泊松比，在**泊松比**字段中显示其值，并在视口中绘制它。如有需要，您可以通过更改字段中的值来调整计算的泊松比值。
5. 从**材料**列表中，选择要应用此校准行为的材料定义；或单击![](../graphics/ico_material.png)为此校准行为创建新的材料定义。有关定义新材料模型的更多信息，请参见"创建或编辑材料"，第12.7.1节。
6. 单击**确定**。Abaqus/CAE更新新的校准行为。如果您指定了材料定义，Abaqus/CAE会将各向同性弹塑性校准行为参数映射到该材料定义的**弹性**和**塑性**材料行为。**注意：**当您将校准行为中的数据映射到材料定义时，所选材料中的任何弹性或塑性材料行为都会被覆盖。

![](../graphics/images/black4rule.gif)有关相关信息，请单击以下任一项：- "定义弹性"中的"创建线性弹性材料模型"，第12.9.1节- "定义塑性"中的"定义经典金属塑性"，第12.9.2节

### 校准带永久变形的超弹性数据

带永久变形的超弹性校准行为使您能够从弹性体和热塑材料的加载、卸载和再加载的单轴和双轴数据集中提取塑性和超弹性材料行为以及Mullins效应。您可以从单轴测试、双轴测试或两种类型的测试中提取数据。校准过程包括以下步骤：

1. 将单轴和/或双轴测试数据文件作为新数据集上传到Abaqus/CAE。
2. 从您提供的数据文件中提取加载、卸载和再加载循环以及永久变形数据，并为每个循环的加载、卸载和再加载阶段创建单独的数据集。
3. 如有需要，选择您想从材料行为计算中排除的任何数据循环。
4. 从视口中选择屈服点，如需要，编辑主加载数据集上的单个点以创建更平滑的曲线。永久变形曲线基于当前屈服点，因此当您选择新屈服点时，这些曲线也会改变。
5. 一旦确定了您想用于推导材料行为的数据集并指定了主曲线选项，您可以从所选数据中推导材料行为。Abaqus/CAE将塑性、超弹性和Mullins效应材料行为映射到您选择的材料。

**校准带永久变形的超弹性数据：**

1. 从模型树中，展开**校准**容器并双击**行为**。**创建校准行为**对话框出现。
2. 输入材料校准行为的名称，选择**带永久变形的超弹性**，然后单击**继续**。**编辑行为**对话框出现。
3. 从**单轴**或**双轴**选项卡页面执行以下步骤：1. 展开**数据集**列表，并选择要从中计算单轴或双轴数据测试校准值的数据。2. 单击![](../graphics/ico_calculator.png)。Abaqus/CAE提取主加载曲线、每个循环应变水平的卸载和再加载曲线以及永久变形曲线，然后为每个循环应变水平的每个分量创建新的校准数据集。每个新数据集都可在**单轴测试数据集**或**双轴测试数据集**选项中使用，并在视口中绘制。3. 切换您想包含在材料校准计算中的各个加载、卸载或再加载数据集。当您切换数据集时，Abaqus/CAE会在视口中显示其相应的*X-Y*曲线。您可以选择以下任何一项：- 选择**全部**以包含所选测试数据文件中找到的所有原始数据。- 选择**主**以包含来自主加载曲线的数据。- 选择**卸载**以包含来自每个循环应变水平卸载曲线的数据，或展开此容器以选择单个卸载曲线。- 选择**再加载**以包含来自每个循环应变水平再加载曲线的数据，或展开此容器以选择单个再加载曲线。- 选择**永久变形**以包含来自两个永久变形曲线的数据，或展开此容器以选择永久变形的应力或应变相关分量。4. 从**主曲线**选项中，执行以下操作：- 单击![](../graphics/ico_select.png)，然后从视口中的主曲线上选择屈服点。如有需要，单击![](../graphics/ico_edit.png)并从出现的对话框中编辑主曲线。
4. 如有需要，从**单轴**或**双轴**选项卡页面提取第二个数据集。
5. 如果您同时提取了单轴和双轴测试数据，默认情况下Abaqus/CAE在材料行为计算中同等应用数据。如果您希望一个数据集在这些计算中具有更大的权重，请从**选项**选项卡页面执行以下步骤：1. 在**材料属性**选项中，将**权重**滑块向您想赋予更大权重的数据类型（单轴或双轴）拖动。2. 指定相对权重选择是基于线性插值还是对数插值。
6. 从**材料**列表中，选择要应用此校准行为的材料定义；或单击![](../graphics/ico_material.png)为此校准行为创建新的材料定义。有关定义新材料模型的更多信息，请参见"创建或编辑材料"，第12.7.1节。
7. 单击**确定**。Abaqus/CAE更新新的校准行为，并将带永久变形的超弹性校准行为参数映射到该材料定义的**超弹性**、**塑性**和**Mullins效应**材料行为。**注意：**当您将校准行为中的数据映射到材料定义时，所选材料中的任何超弹性、塑性或Mullins效应材料行为都会被覆盖。
