# 20.10.6 通过三个点草绘圆弧







使用 Sketcher 工具箱中的通过三点圆弧工具绘制通过三个非共线点的圆弧。下图显示了生成的圆弧：

![](../graphics/ske-arc3ptsbtn-nls.png)

使用通过三点方法绘制圆弧时，您应该考虑与["使用圆心和两个端点草绘圆弧，"第 20.10.5 节](pt03ch20s10hlb05.md)中所述相同的角度大小和点选择前提条件。

**通过三个点草绘圆弧：**

1. 从 Sketcher 工具箱中的圆弧工具中，选择通过三点圆弧工具![](../graphics/ico_sketchArcThreePts.png)。有关 Sketcher 工具箱中工具的图表，请参见["Sketcher 工具，"第 20.4.1 节](pt03ch20s04s01.md)。Abaqus/CAE 在提示区域显示提示以指导您完成该过程。
2. 点击您想要用作圆弧端点的两个点。**提示：**如有必要，您可以使用提示区域中的文本框输入端点的精确坐标。有关精确定义圆弧的更多信息，请参见["指定精确几何，"第 20.6 节](pt03ch20s06.md)。当您选择圆弧的第二个端点时，Abaqus/CAE 会绘制一个具有预定半径的示例圆弧。当您选择第三个点时，您可以更改此示例半径。
3. 将光标移动到圆弧上的第三个点，然后点击鼠标按钮 1。Abaqus/CAE 绘制一条连接端点并通过所选点的圆弧。**提示：**如有必要，点击**上一步**按钮![](../graphics/afxI_backup.png)可以反转任一端点的选择。点击一次可以使您反转第二个端点的选择；点击两次也可以反转第一个端点的选择。
4. 要创建更多圆弧，重复从第 2 步开始的上述步骤。
5. 完成创建圆弧后，执行以下操作之一：- 在 Abaqus/CAE 窗口中的任意位置点击鼠标按钮 2。- 在 Sketcher 工具箱中选择任何其他工具。- 点击提示区域中的取消按钮![](../graphics/afxI_cancel.png)。
