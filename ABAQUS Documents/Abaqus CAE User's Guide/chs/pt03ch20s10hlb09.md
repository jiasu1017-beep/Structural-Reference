# 20.11.6 将草图组件设置为构造几何


使用 Sketcher 工具箱中的**设置为构造**工具将草图的组件转换为构造几何。**取消设置构造**工具可以反转此过程,用于之前已转换为构造几何的项目。只有使用**设置为构造**工具转换为构造几何的项目才能被取消设置构造;作为构造几何添加到草图的项目无法被取消设置。

**将草图中的项目设置为构造几何的步骤:**

1. 从 Sketcher 工具箱的构造工具中,选择**设置为构造**工具 ![](../graphics/ico_sketchSetConst.png)。有关 Sketcher 工具箱中工具的图示,请参见["Sketcher 工具," 第 20.4.1 节](pt03ch20s04s01.md)。Abaqus/CAE 会在提示区域显示提示,引导您完成操作。
2. 选择要设置为构造几何的草图项目。您可以拖动选择、**[Shift]****+单击**或**[Ctrl]****+单击**来选择多个项目。Abaqus/CAE 会将您选择的项目以红色显示。
3. 要完成转换,请在提示区域单击**完成**,或在 Abaqus/CAE 窗口任意位置单击鼠标按钮 2。Abaqus/CAE 会用虚线显示您选择的项目,表示它们现在是构造几何。
4. 完成选择要转换为构造几何的几何时,执行以下操作之一: - 在 Abaqus/CAE 窗口任意位置单击鼠标按钮 2。 - 选择 Sketcher 工具箱中的任何工具。 - 单击提示区域中的取消按钮 ![](../graphics/afxI_cancel.png)。

**取消设置构造几何项目的步骤:**

1. 从 Sketcher 工具箱的构造工具中,选择**取消设置构造**工具 ![](../graphics/ico_sketchUnsetConst.png)。有关 Sketcher 工具箱中工具的图示,请参见["Sketcher 工具," 第 20.4.1 节](pt03ch20s04s01.md)。Abaqus/CAE 会在提示区域显示提示,引导您完成操作。
2. 选择一个或多个构造几何项目。您可以拖动选择、**[Shift]****+单击**或**[Ctrl]****+单击**来选择多个项目。所有构造几何都在草图中以虚线显示。Abaqus/CAE 会将您选择的项目以红色显示。
3. 要完成转换,请在提示区域单击**完成**,或在 Abaqus/CAE 窗口任意位置单击鼠标按钮 2。Abaqus/CAE 会用实线显示您选择的项目,表示它们不再是构造几何。
4. 完成选择要还原的构造几何项目后,执行以下操作之一: - 在 Abaqus/CAE 窗口任意位置单击鼠标按钮 2。 - 选择 Sketcher 工具箱中的任何工具。 - 单击提示区域中的取消按钮 ![](../graphics/afxI_cancel.png)。
