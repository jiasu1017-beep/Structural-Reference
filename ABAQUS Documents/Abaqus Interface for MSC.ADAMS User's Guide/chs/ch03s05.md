# 3.5 诊断错误消息和问题





在执行 **adams** 命令期间，可能会输出以下警告和错误消息：

```
WARNING: There are 'N' elements in substructure Z1, but only 'M'
   -=> mass matrices have been processed from the results file.
   -=> Carefully review this discrepancy before proceeding.
```
说明：如果读取的质量矩阵数量为零，请验证在 [*SUBSTRUCTURE GENERATE](../key/key-link.md#usb-kws-ssubgenerate) 步骤之前的 [*FREQUENCY](../key/key-link.md#usb-kws-hfrequency) 步骤中是否存在 [*ELEMENT MATRIX OUTPUT](../key/key-link.md#usb-kws-helemmatout), MASS=YES 选项。

如果 *m* 非零但小于 *n*，则模型可能是正确的。某些 Abaqus 单元（如阻尼器）没有质量矩阵。其他单元的质量可能在表示子结构的质量时不显著而被忽略。不需要在元素集中具有可忽略质量的单元。

```
WARNING: No fixed-interface normal modes.
   -=> The results file did not contain any modes from
   -=> a *FREQUENCY step. Typically, this step contains:
   -=> *NODE FILE
   -=> U,
```
说明：结果文件中缺少振型。[*SUBSTRUCTURE GENERATE](../key/key-link.md#usb-kws-ssubgenerate) 步骤之前必须有 [*FREQUENCY](../key/key-link.md#usb-kws-hfrequency) 步骤。[*FREQUENCY](../key/key-link.md#usb-kws-hfrequency) 步骤使用以下命令将振型写入结果文件：
```
*NODE FILE
U,
```

通常，所有振型都会被写入。但是，如果在 [*SUBSTRUCTURE GENERATE](../key/key-link.md#usb-kws-ssubgenerate) 步骤中保留了计算模态的子集（使用 [*SELECT EIGENMODES](../key/key-link.md#usb-kws-hselecteigenmodes) 选项），则 [*FREQUENCY](../key/key-link.md#usb-kws-hfrequency) 步骤必须仅写入那些模态。

```
ERROR: Missing element mass matrices.
   -=> No element mass matrices were found in the results file.
   -=> The input file must contain an entry similar to:
   -=> *ELEMENT MATRIX OUTPUT, MASS=YES, ELSET=elset_name
   -=> This entry must be in the *FREQUENCY step preceding
   -=> the *SUBSTRUCTURE GENERATE step.
```
说明：[*FREQUENCY](../key/key-link.md#usb-kws-hfrequency) 步骤必须使用错误消息中给出的语法将单元质量矩阵写入结果文件。

```
ERROR: Missing generalized stiffness and mass matrices.
   -=> Verify that the input file defines this substructure
   -=> and contains the following option:
   -=> *SUBSTRUCTURE MATRIX OUTPUT, STIFFNESS=YES, MASS=YES,
   -=> RECOVERY MATRIX=YES [, SLOAD=YES]
```
说明：[*SUBSTRUCTURE GENERATE](../key/key-link.md#usb-kws-ssubgenerate) 步骤必须将广义（简化）质量和刚度矩阵以及恢复矩阵写入结果文件。也可以使用 SLOAD=YES 写入子结构载荷向量。
