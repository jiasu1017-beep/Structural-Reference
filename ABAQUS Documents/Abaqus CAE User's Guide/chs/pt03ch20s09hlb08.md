# 20.9.8 设置记录的草绘操作的最大数量







Abaqus/CAE 在缓存中记录您对当前草图的更改，使您能够撤销和重做多个连续的草绘更改。默认情况下，撤销缓存支持最多 10 级撤销，但您可以增加或减少此数量。如果您的大规模更改需要大量草绘操作，您可能希望增加级数；为了提高大型复杂草图的系统性能，您可能希望减少此数量。

最大撤销级别适用于 Abaqus/CAE 会话中的所有草图。每次您更改当前草图或切换到不同模块时，Abaqus/CAE 都会清除撤销缓存。

**设置记录的草绘操作的最大数量：**

1. 从 Sketcher 工具箱底部，选择自定义工具![](../graphics/ico_sketchOptions-nls.png)。出现**Sketcher 选项**对话框。
2. 从**常规**选项卡页面，在**草图撤销最大级别**文本字段中输入 0 到 100 之间的值。
3. 选择完所需的自定义选项后，点击**确定**应用您的更改并关闭**Sketcher 选项**对话框。
