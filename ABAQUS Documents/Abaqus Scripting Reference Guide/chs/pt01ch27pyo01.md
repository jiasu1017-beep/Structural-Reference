# 27.1 Load 对象

Load 对象是其他 Load 对象的抽象基类型。Load 对象没有显式构造函数。Load 对象的方法和成员是从 Load 派生的所有对象共有的。

**访问**

```
import load
mdb.models[*name*].loads[*name*]
```

### 27.1.1 deactivate(...)

此方法在指定步骤及其所有后续步骤中停用载荷。

**必需参数**

*stepName*

String，指定要停用载荷的步骤名称。

**可选参数**

无。

**返回值**

无。

**异常**

TextError。

### 27.1.2 move(...)

此方法将载荷状态对象从一个步骤移动到不同的步骤。

**必需参数**

*fromStepName*

String，指定要移出载荷状态的步骤名称。

*toStepName*

String，指定要移入载荷状态的步骤名称。

**可选参数**

无。

**返回值**

无。

**异常**

TextError。

### 27.1.3 reset(...)

此方法将指定步骤的载荷状态重置为前一个通用分析步骤的状态。

**必需参数**

*stepName*

String，指定要重置载荷状态的步骤名称。

**可选参数**

无。

**返回值**

无。

**异常**

TextError。

### 27.1.4 resume()

此方法恢复之前被抑制的载荷。

**参数**

无。

**返回值**

无。

**异常**

无。

### 27.1.5 suppress()

此方法抑制载荷。

**参数**

无。

**返回值**

无。

**异常**

无。

### 27.1.6 delete(...)

此方法允许您删除现有的载荷。

**必需参数**

*indices*

Int 序列，指定要删除的每个载荷的索引。

**可选参数**

无。

**返回值**

无。

**异常**

无。

### 27.1.7 成员

Load 对象可以具有以下成员：

*name*

String，指定载荷存储库密钥。

*region*

[Region](pt01ch45pyo03.md) 对象，指定要施加载荷的区域。
