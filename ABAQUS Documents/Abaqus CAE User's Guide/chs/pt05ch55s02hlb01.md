# 55.2.1 选择渲染样式





渲染样式是Abaqus/CAE显示模型的样式。渲染样式是曲线图状态无关的;也就是说,您为所有未变形、变形、云图、符号和材料取向曲线图设置一次。您也可以设置Abaqus/CAE在变形形状叠加在未变形形状上时将应用于未变形形状的第二个渲染样式。可能的渲染样式是"**Wireframe**"、"**Hidden**"、"**Filled**"和"**Shaded**";这些样式如[图55-1](pt05ch55s02hlb01.md#viw-render)所示。以下是对这些选择的说明。

**图55-1** 显示渲染样式选项的模型。从左到右:线框、填充、消隐和光源着色渲染样式。

![](../graphics/viw-render.png)

**Wireframe**

显示模型边缘;内部和外部边缘都可能可见。线框曲线图产生框架般的视觉效果,其中不显示模型面。

**Filled**

显示以均匀颜色"绘制"的模型面。填充曲线图产生实体而非框架般的外观,其中仅可见外部面。

**Hidden**

显示其中隐藏了被模型遮挡的边缘的线框曲线图。消隐曲线图产生实体而非框架般的外观。

**Shaded**

显示其中光源似乎照射在模型上的填充曲线图。着色曲线图产生高度三维的视觉效果。

**选择曲线图渲染样式的步骤:**

1. 找到公共或叠加"**Render Style**"选项。公共渲染样式始终应用于变形形状;当在任何曲线图状态(未变形、云图、符号或材料取向)中单独绘制未变形形状时,它也适用于未变形形状。当未变形形状与变形形状一起绘制时,应用叠加渲染样式。从主菜单栏,选择 ****Options**![](../graphics/images/arrow.gif)**Common**** 或 **Superimpose**;然后在出现的对话框中点击"**Basic**"选项卡。"**Render Style**"选项变得可用。
2. 从"**Render Style**"列表中,选择您想要的渲染样式。 **提示:**您也可以通过点击位于"**Render Style**"工具栏中的线框 ![](../graphics/ico_displayWire.png)、消隐 ![](../graphics/ico_displayHidden.png)、填充 ![](../graphics/ico_displayFilled.png) 或着色 ![](../graphics/ico_displayShaded.png) 图标来为当前曲线图状态(不包括显示体和叠加时的未变形形状)选择渲染样式。 **注意:**如果您激活[透明度](pt05ch55s02hlb02.md),则在着色、填充和消隐渲染样式中的符号和材料取向曲线图更容易查看。
3. 点击"**Apply**"实施您的更改。渲染样式会根据您的规格进行更改。默认情况下,您的更改会在会话期间保存,并会影响当前视口和从当前视口创建的任何新视口中的所有后续曲线图。如果要为后续会话保留您应用的更改,请将它们保存到文件。更多信息,请参阅 ["为后续会话保存自定义," 第55.1.1节](pt05ch55s01s01.md)。
