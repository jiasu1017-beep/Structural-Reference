# 31.2.11 请求轮廓积分输出





您可以使用Step模块中的**History Output Request**编辑器来请求来自轮廓积分分析的输出。您必须从**Domain**字段中选择**Crack**。然后编辑器允许您选择输出频率和要执行的轮廓积分分析类型。更多信息，请参见["修改历史输出请求"，第14.12.3节](pt03ch14s12hlb03.md)和["轮廓积分评估"，Abaqus Analysis User's Guide第11.4.2节](../usb/usb-link.md#usb-anl-acontintegral)。

您也可以为XFEM断裂分析请求轮廓积分输出。更多信息，请参见["为XFEM请求轮廓积分输出"，第31.3.7节](pt04ch31s03hlb04.md)。

**要请求轮廓积分输出：**

1. 进入Step模块。
2. 从主菜单栏中，选择****Output**![](../graphics/images/arrow.gif)**History Output Request**![](../graphics/images/arrow.gif)**Create****。Abaqus/CAE显示**Create History**对话框。
3. 从**Create History**对话框中，输入输出请求的名称以及将创建该请求的步骤。单击**Continue**关闭对话框。Abaqus/CAE显示**History Output Requests**编辑器。
4. 从**Domain**字段中，选择**Crack**，然后选择所需的轮廓积分。
5. 从**History Output Requests**编辑器中出现的字段中，输入输出频率。
6. 输入要计算的轮廓数。Abaqus/CAE通过向先前域定义的区域添加一层单元来计算下一个轮廓积分，如图31-4所示。每个轮廓为轮廓积分生成一个值或一组值。
7. 如果需要，切换**Step for residual stress initialization values**并选择Abaqus将从最后可用增量中读取应力数据并使用该数据计算残余应力的步骤。如果您选择初始步骤，则残余应力由指定的初始条件定义。您不能选择在创建历史输出请求的步骤之后发生的步骤。更多信息，请参见["轮廓积分评估"中的"在*J*积分评估中包含残余应力场的影响"，Abaqus Analysis User's Guide第11.4.2节](../usb/usb-link.md#usb-anl-acontintegral-jintegral-stressfield)。
8. 选择要执行的轮廓积分计算类型。您可以选择以下类型：
***J*积分**
您使用*J*积分来表征与裂缝扩展相关的能量释放的准静态断裂分析。*J*积分可以与应力强度因子相关联（如果材料响应是线性的）。更多信息，请参见["轮廓积分评估"中的"*J*积分"，Abaqus Analysis User's Guide第11.4.2节](../usb/usb-link.md#usb-anl-acontintegral-jintegral)。
**<img src="../graphics/usi_eqn00701.gif">积分**
您使用<img src="../graphics/usi_eqn00701.gif>积分来描述时间相关的蠕变行为，其中它表征某些蠕变条件下的蠕变裂缝变形，包括瞬态裂缝扩展。更多信息，请参见["轮廓积分评估"中的"*Ct*积分"，Abaqus Analysis User's Guide第11.4.2节](../usb/usb-link.md#usb-anl-acontintegral-cintegral)。
***T*应力**
您使用*T*应力分量来表示平行于裂缝前缘的应力。更多信息，请参见["轮廓积分评估"中的"*T*应力"，Abaqus Analysis User's Guide第11.4.2节](../usb/usb-link.md#usb-anl-acontintegral-tstress)。
**应力强度因子**
您在线弹性断裂力学中使用应力强度因子<img src="../graphics/usi_eqn00708.gif>、<img src="../graphics/usi_eqn00709.gif>和<img src="../graphics/usi_eqn00710.gif>来表征局部裂缝尖端应力和位移场。更多信息，请参见["轮廓积分评估"中的"应力强度因子"，Abaqus Analysis User's Guide第11.4.2节](../usb/usb-link.md#usb-anl-acontintegral-stressintensity)。如果您使用应力强度因子请求轮廓积分计算，Abaqus还会计算起始处的裂缝扩展方向。您必须选择以下之一来表示裂缝起始标准：
- **Maximum tangential stress**
- **Maximum energy release rate**
- <img src="../graphics/usi_eqn00711.gif>
更多信息，请参见["轮廓积分评估"中的"裂缝扩展方向"，Abaqus Analysis User's Guide第11.4.2节](../usb/usb-link.md#usb-anl-acontintegral-crackprop)。
9. 单击**OK**配置轮廓积分分析并关闭编辑器。
