# 47.7.4 Customizing the position of an X–Y plot







### 47.7.4 Customizing the position of an X--Y plot

Use the **Position** options in the **Chart Options** dialog box to customize the position of an *X–Y* plot in the viewport. 

**To customize the position of an X–Y plot:**

1. Locate the **Position** options. Select ****Options**![](../graphics/images/arrow.gif)**XY Options**![](../graphics/images/arrow.gif)**Chart****; or click ![](../graphics/ico_xyChartOptions.png), which is located with the *X--Y* plotting tools in the Visualization module toolbox. Click the **Grid Position** tab in the dialog box that appears.
2. From the **Charts** field, select one or more *X--Y* plots whose positions or shapes you wish to customize.
3. Select one of the following *X--Y* plot positioning options: **Auto-align** This option enables you to align the selected *X--Y* plot in one of nine positions in the viewport: its center, any of the four corners, or centered along any of the four edges. Expand the **Alignment** list, then select one of the options depicted graphically in the list. **Manual** This option enables you to position the *X--Y* plot origin at a specific *X--Y* location in the viewport. Drag the **X** and **Y** sliders to move the lower left corner of the *X--Y* plot horizontally or vertically in the viewport. When these sliders' values are set to `0`, the *X--Y* plot origin is placed in the lower left corner of the viewport.
4. Click **Dismiss** to close the **Chart Options** dialog box.
