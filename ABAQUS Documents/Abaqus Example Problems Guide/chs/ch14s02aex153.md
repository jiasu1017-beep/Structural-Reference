# 14.2.3 挡风玻璃雨刮的设计敏感性分析







**产品：** Abaqus/Standard  Abaqus/Design

在本例中，我们考虑静态加载的挡风玻璃雨刮片的设计敏感性分析。传统雨刮系统由三个主要子系统组成：刮片-臂组件、连杆机构和电动机。我们将设计和分析限制在刮片-臂子系统中。擦拭性能由雨刮系统的动态性能决定，而安全法规要求刮片擦拭挡风玻璃表面上的特定区域。实现优化擦拭性能并满足面积条件的系统是重大挑战。刮片（橡胶）元件的力/挠度响应与刮片-玻璃界面摩擦之间的相互作用是设计中的关键要素。雨刮-玻璃相互作用的敏感性分析提供了可在进行更复杂的动摩擦分析之前用于改进设计的洞察。敏感性分析也识别了对高度应力区域应力减少最有效的设计参数。本例不考虑橡胶元件凸缘之间的自接触。

### 几何形状、模型特性和设计参数

橡胶刮片的几何形状如图14.2.3-1所示。臂相对于橡胶刮片假设为刚硬，并通过规定固定边界条件建模。橡胶刮片用231个一阶混合单元建模为平面应变模型。使用具有多项式应变能函数的不可压缩超弹性材料来模拟橡胶材料行为。挡风玻璃假设为刚性，并建模为解析刚性表面。刮片与挡风玻璃之间的表面相互作用使用有限滑动接触和各向同性库仑摩擦系数0.2建模。选择三个形状参数和摩擦系数![](../graphics/exa_eqn00274.gif)作为设计研究的设计参数。我们选择![](../graphics/exa_eqn00274.gif)作为设计参数来研究摩擦系数对刮片-玻璃界面力的影响。选择可以修改而不需要更改组件其他部分几何的形状参数作为设计参数。这些形状参数将帮助我们研究雨刮几何方面对雨刮刚度的影响。形状参数是两个凸缘之间颈部厚度![](../graphics/exa_eqn00864.gif)；雨刮尖端厚度![](../graphics/exa_eqn00865.gif)；以及下凸缘宽度![](../graphics/exa_eqn00866.gif)。敏感性分析所需输入的相应形状梯度（节点坐标关于形状设计参数的导数）显示为符号图，分别如图14.2.3-2、图14.2.3-3和图14.2.3-4所示。这些梯度使用参数形状变化输入，并使用从Abaqus/CAE重放文件派生的自定义Python脚本计算。此方法需要熟悉Abaqus脚本接口。可以使用Abaqus/CAE更轻松地生成形状变化数据；详情见["设计敏感性分析：概述," 第14.1.1节](ch14s01abo01.md)。

### 加载、边界条件和设计响应

刮片和挡风玻璃之间2.0单位的干涉配合用于模拟它们之间的静态垂直荷载。挡风玻璃在垂直方向保持固定，上凸缘的表面节点在两个方向上被约束。擦拭运动通过向挡风玻璃参考节点规定9.0单位的水平位移来模拟。

期望减少橡胶元件中的应力集中，因为高应力将降低刮片的使用寿命并影响性能。因此，von Mises应力的敏感性被选为本模型中的主要设计响应。任何设计更改应考虑到雨刮与挡风玻璃之间接触压力的行为。为此，也选择CSTRESS的敏感性作为设计响应。包括摩擦使刚度矩阵非对称；因此，使用非对称求解器运行分析。由于我们关心尖端接触压力的历史，因此选择增量DSA公式。为了量化忽略非对称项和使用总DSA的影响，我们也比较各种DSA公式和刚度矩阵对称化组合的最大敏感性。

### 结果与讨论

橡胶刮片变形构型上von Mises应力分布的等值线图如图14.2.3-5所示。显然，两个凸缘之间的颈部是感兴趣的区域。在颈部von Mises应力分布的放大等值线图中（图14.2.3-6），我们可以看到下颈部的应力集中高于上部颈部。下颈部最大Mises应力的归一化敏感性列于表14.2.3-1中。敏感性通过除以刮片最大Mises值（刮片元件131中的67857.7单位）然后乘以设计参数初始值来归一化。我们注意到形状参数有直接影响（增加参数增加应力，反之亦然），而摩擦系数对Mises应力有反向影响。减少颈部厚度使雨刮更柔顺，减少应力集中。增加摩擦系数增加尖端的剪切力。这个增加的剪切力增加上颈部的弯曲并减少下颈部的弯曲，从而减少下颈部的应力集中。我们从表中推断出，Mises应力对![](../graphics/exa_eqn00864.gif)和![](../graphics/exa_eqn00865.gif)比凸缘宽度和摩擦系数更敏感。为了减少应力集中，考虑对![](../graphics/exa_eqn00864.gif)和![](../graphics/exa_eqn00865.gif)进行更改的设计。选择![](../graphics/exa_eqn00864.gif)减少20%，![](../graphics/exa_eqn00865.gif)减少10%。预测的Mises应力减少百分比为6.76（0.2832×0.2 + 0.1094×0.1）。

提出的设计更改将影响雨刮-挡风玻璃界面的接触压力，并使用敏感性结果来量化这种影响。尖端一点接触压力归一化敏感性的时间历史如图14.2.3-7所示。该点接触压力对![](../graphics/exa_eqn00864.gif)最敏感。这个结果有些意外，因为此设计参数不明确影响界面。![](../graphics/exa_eqn00864.gif)对接触压力有直接比例影响，而![](../graphics/exa_eqn00865.gif)有反比例影响。因此，减少![](../graphics/exa_eqn00864.gif)将减少接触压力，而减少![](../graphics/exa_eqn00865.gif)将增加接触压力。提出的设计和当前设计的接触压力历史分别如图14.2.3-8所示。图表明，提出的设计中减少的应力集中是以降低雨刮性能为代价的。（提出的设计具有较低的接触压力，这可能是次优的。）如果需要，可以通过考虑![](../graphics/exa_eqn00865.gif)的正变化来积极影响接触压力。

上述所有结果都是在使用非对称求解器的增量DSA分析中获得的。如果用户只对特定增量（通常是最后增量）处的结果感兴趣，总DSA在计算上是有优势的。同样，忽略非对称项将提高计算效率。为了量化这种近似的误差，我们将各种组合的结果与总体有限差分法（OFD）的结果进行比较。表14.2.3-2列出了与使用OFD方法获得的结果相比，最大Mises和接触压力敏感性的相对误差。对于主导形状参数![](../graphics/exa_eqn00864.gif)，最大Mises应力的敏感性结果与所有组合的OFD方法良好一致。然而，在总DSA中忽略非对称项对接触压力敏感性给出不准确的结果。相对误差对于次主导形状参数较大，总DSA对Mises和接触压力敏感性都给出较差的结果。总DSA对关于摩擦系数的敏感性给出不准确的结果。虽然敏感性结果是问题依赖的，但我们可以推断，如果忽略非对称项，总DSA可能给出较差的结果。此外，相互作用压力的敏感性比结构中最大应力的敏感性对近似更敏感。不太重要的敏感性比较主导敏感性受近似影响更大。

进行了提出的设计更改并重新运行分析。观察到应力集中减少9%。预测和实际接触压力历史如图14.2.3-9所示。预测和实际结果之间的差异是由于设计响应对设计参数的非线性依赖。然而，重新运行证实了提出的设计更 robust的预测，可能具有次优的擦拭压力。
