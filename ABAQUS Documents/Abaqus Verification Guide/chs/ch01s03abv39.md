# 1.3.36 承受非对称孔隙压力场的圆柱：CAXA 单元







**产品：**Abaqus/Standard  

### 测试的单元

CAXA8P*n*    CAXA8RP*n*    

（*n* = 1、2、3、4）

### 问题描述

![](../graphics/verporecaxa-model.png)

内半径为 ![](../graphics/ver_eqn00176.gif)、外半径为 ![](../graphics/ver_eqn00211.gif)、长度为 ![](../graphics/ver_eqn00212.gif) 的圆截面空心圆柱土柱，承受以下形式的非对称孔隙压力分布

![](../graphics/ver_eqn00251.gif)

其中 ![](../graphics/ver_eqn00252.gif) 是圆柱外表面在 ![](../graphics/ver_eqn00216.gif) = 0 处的恒定孔隙压力，*r* 和 ![](../graphics/ver_eqn00001.gif) 是圆柱坐标。径向和周向孔隙压力梯度的存在导致土壤中的孔隙流体在这些方向上流动，从而产生圆柱的弯曲。这是一个耦合问题，必须同时求解应力平衡和流体连续性方程以及孔隙压力 CAXA 单元。为说明目的，我们仅考虑稳态耦合问题，并假设材料是线性且具有恒定渗透率的，由不可压缩颗粒和流体组成。孔隙压力 CAXA 模型预测的结果将与相应三维模型的预测结果进行比较。

仅考虑结构的一半，对称平面在 ![](../graphics/ver_eqn00057.gif) = 0。网格收敛性研究表明，单个二阶 CAXA 单元可以充分建模结构并产生准确结果。但是，在径向方向上使用两个单元，以便与三维模型获得的结果进行直接比较。在三维模型中，C3D20P 单元用于有限元网格，径向 2 个单元，轴向 1 个单元，周向 12 个单元。为了便于与 CAXA 模型的结果进行比较，三维模型中的所有节点都转换到局部圆柱坐标系，并且对材料施加圆柱方向，以便在同一圆柱坐标系中输出位移、应力和应变分量。

**材料：**

线弹性，弹性模量 = 1 × 10⁸，泊松比 = 0.3，渗透率 = 1 × 10⁻⁵，各处初始孔隙比 = 1.0。

**边界条件：**

在 ![](../graphics/ver_eqn00057.gif) = 0 平面上 ![](../graphics/ver_eqn00145.gif) = 0；在 ![](../graphics/ver_eqn00215.gif) 和 ![](../graphics/ver_eqn00057.gif) = 0 处施加 ![](../graphics/ver_eqn00232.gif) = 0，以消除全球 *x* 方向的刚体运动。

**载荷：**

施加以下形式的孔隙压力场 ![](../graphics/ver_eqn00253.gif)。计算圆柱内壁和外壁上每个角节点的孔隙压力，并通过自由度 8 将孔隙压力值指定为边界条件。

### 结果与讨论

对于参数为 ![](../graphics/ver_eqn00220.gif) = 6、![](../graphics/ver_eqn00221.gif) = 2、![](../graphics/ver_eqn00222.gif) = 6 和 ![](../graphics/ver_eqn00254.gif) = 3 × 10⁶ 的结构，获得的 CAXA8P*n* 和 CAXA8RP*n*（*n* = 1、2、3 或 4）单元以及 C3D20P 单元的结果如下表所示。输出位置在 ![](../graphics/ver_eqn00216.gif) = 0 平面上的点 ![](../graphics/ver_eqn00223.gif)、![](../graphics/ver_eqn00224.gif)、![](../graphics/ver_eqn00255.gif)、![](../graphics/ver_eqn00256.gif)、![](../graphics/ver_eqn00257.gif) 和 ![](../graphics/ver_eqn00258.gif) 处（如图所示）。在 ![](../graphics/ver_eqn00216.gif) = 180° 平面上相同位置处获得与下面所示完全相等且相反的结果。显然，CAXA 模型的结果与三维模型的结果非常接近。如下表所示的应力解表明，有效应力分量在各处都与孔隙压力相等，因此圆柱各处的总应力为零。CAXA 模型获得的结果与 *n*（傅里叶模式数）无关，并且由于施加的非对称孔隙压力场可以在 CAXA 模型中精确指定，因此看起来比三维模型更准确。在三维模型中，需要在 ![](../graphics/ver_eqn00001.gif) 方向上使用更多单元才能获得更高精度的结果。请注意此问题的解与["承受非对称温度场的圆柱：CAXA 单元"，第 1.3.34 节](ch01s03abv37.md)所述的非对称温度分析的相似性。
