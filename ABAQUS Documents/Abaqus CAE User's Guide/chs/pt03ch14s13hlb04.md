# 14.13.4 配置自由度监视器请求





使用「DOF Monitor」对话框可以在分析过程中监视某一点处的自由度。Abaqus会将该点的自由度值报告给Job模块、状态文件（`*jobname*.sta`），对于Abaqus/Standard分析，还会报告给消息文件（`*jobname*.msg`）。更多信息，请参见「自由度监视器请求」，第14.5.4节。

**监视自由度的步骤：**

1. 从主菜单栏选择 ****Output**![](../graphics/images/arrow.gif)**DOF Monitor****。此时会出现「DOF Monitor」对话框。
2. 在对话框中切换「Monitor a degree of freedom throughout the analysis」。如果「Monitor a degree of freedom throughout the analysis」处于开启状态，则对话框中的监视选项将变为可用。
3. 点击「Region」字段旁边的![](../graphics/ico_selectBlue.png)，并使用以下方法之一选择要监视的点：- 使用现有集合来定义点。在提示区域右侧点击「Points」。从出现的「Region Selection」对话框中选择一个现有集合，然后点击「Continue」。（当您在「DOF Monitor」对话框中点击「OK」时，如果选择的集合包含多个顶点或节点，Abaqus/CAE会显示错误消息。）**注意：**默认选择方法基于您最近使用的选择方法。如需恢复为另一种方法，请点击提示区域右侧的「Select in Viewport」或「Points」。- 使用鼠标在视口中选择一个点。（更多信息，请参见「在当前视口中选择对象」，第6.2节。）您选择的点会在视口中以红色高亮显示。
4. 在「Degree of freedom」字段中，输入您要监视的自由度。
5. 如果您正在执行Abaqus/Standard分析，请在「Print to the message file every *n* increments」字段中，输入您希望Abaqus写入消息文件的频率。（没有控制写入状态文件频率的选项，因为该频率由Abaqus决定。更多信息，请参见「输出」，Abaqus Analysis User's Guide第4.1.1节。）
6. 点击「OK」保存您的请求并退出对话框。
