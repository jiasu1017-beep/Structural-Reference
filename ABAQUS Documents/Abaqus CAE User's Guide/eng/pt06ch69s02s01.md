# 69.2.1 An overview of the methods for editing edges







Select ****Tools**![](../graphics/images/arrow.gif)**Geometry Edit**** from the main menu to display the **Geometry Edit** dialog box. When you choose **Edge** from the dialog box, the **Tool** list displays the following methods for editing edges:

**Stitch ![](../graphics/ico_repairStitch.png) **

If a part is imported as a group of disconnected faces, you can stitch the resulting small edge gaps. Similarly, you can stitch the resulting gaps after you remove small faces or small slivers from a part. You can perform stitching as a global operation during which Abaqus/CAE stitches all gaps in the part, or you can pick the edges that you want to stitch, stitch edges with gaps smaller than a user-specified tolerance, or use both of these options. You should perform a global stitching operation for your entire part for small gaps only, and this process can be lengthy. For more information, see ["What is stitching?," Section 69.3](pt06ch69s03.md). You can use the Query toolset to highlight any free edges. For more information, see ["Using the geometry diagnostic tools," Section 71.2.4](pt06ch71s02hlb04.md).

**Repair small![](../graphics/ico_repairSmallEdges.png) **

You can repair selected small edges. Abaqus/CAE removes the small edges and edits the adjoining edges to create a closed geometry. 

**Merge ![](../graphics/ico_repairMergeEdges.png)**

You can select a series of connected edges, and Abaqus/CAE merges them into a single edge and deletes redundant vertices along the edges. [Figure 69--2](pt06ch69s02s01.md#rep-mergeedges) illustrates the effect of merging edges.

**Figure 69–2** Merging edges.

![](../graphics/rep-mergeedges-nls.png)

**Remove redundant entities ![](../graphics/ico_repairRemoveRedundant.png) **

An imported part can contain redundant vertices that are positioned along a continuous edge. Similarly, an imported part can include redundant edges that are internal edges. Redundant vertices and edges do not change the shape or the area of a part and are not required for a complete definition, as shown in [Figure 69--3](pt06ch69s02s01.md#prt-repair-redundant).

**Figure 69–3** Redundant edges and vertices.

![](../graphics/prt-repair-redundant-nls.png)

**Repair invalid![](../graphics/ico_repairInvalidEdges.png)**

In rare cases after you import a part, Abaqus will report that some of its edges are invalid. The **Repair invalid** tool will try to repair the invalid edges by recomputing the data that define them. You should also use this tool if the Query toolset indicates that the part contains only invalid edges. 

**Remove wire ![](../graphics/ico_repairRemoveWire.png) **

You can remove selected wire edges. Abaqus/CAE removes the wire edges.
