# 3.5 树widget





本节描述Abaqus GUI Toolkit中的树widget。树widget以分层方式排列其子项，允许展开或折叠各个分支。诸如Microsoft Windows Explorer之类的文件浏览器是使用树widget的应用程序的常见示例。涵盖以下主题：
- ["树列表"，第3.5.1节](pt03ch03s05.md#cus-wgt-widget-tree-treelist)
- ["选项树列表"，第3.5.2节](pt03ch03s05.md#cus-wgt-widget-tree-optiontree)

### 3.5.1 树列表

`FXTreeList`widget提供可展开和折叠的子项树结构。`FXTreeList`构造函数由以下原型定义：

```
FXTreeList(p, nvis, tgt=None, sel=0,
    opts=TREELIST_NORMAL, x=0, y=0, w=0, h=0)
```

`FXTreeList`构造函数的参数如下所述：

**parent**

构造函数中的第一个参数是父参数。`FXTreeList`不在其周围绘制框架；因此，您可能希望创建一个`FXVerticalFrame`作为树的父项。您应该将框架中的填充设为零，以便框架紧密包裹树。

**可见项目数**

树首次显示时将可见的项目数。

**目标和选择器**

您可以在树构造函数参数中指定目标和选择器。

**opts**

您可以在树构造函数中指定的选项标志如下表所示：

| 选项标志 | 效果 |
| --- | --- |
| TREELIST_NORMAL（默认） | TREELIST_EXTENDEDSELECT |
| TREELIST_EXTENDEDSELECT | 扩展选择模式允许用户拖动选择项目范围。 |
| TREELIST_SINGLESELECT | 单选模式允许用户最多选择一个项目。 |
| TREELIST_BROWSESELECT | 浏览选择模式强制始终只选择一个项目。 |
| TREELIST_MULTIPLESELECT | 多选模式用于选择各个项目。 |
| TREELIST_AUTOSELECT | 自动选择光标下的项目。 |
| TREELIST_SHOWS_LINES | 显示项目之间的线条。 |
| TREELIST_SHOWS_BOXES | 当项目可以展开时显示框。 |
| TREELIST_ROOT_BOXES | 也显示根项目框。 |

以下语句显示了创建树的示例：

```
vf = FXVerticalFrame(gb, FRAME_SUNKEN|FRAME_THICK,
    0,0,0,0, 0,0,0,0)
tree = FXTreeList(vf, 5, None, 0,
    LAYOUT_FILL_X|LAYOUT_FILL_Y|
    TREELIST_SHOWS_BOXES|TREELIST_ROOT_BOXES|
    TREELIST_SHOWS_LINES|TREELIST_BROWSESELECT)
```

您通过提供父项和文本标签向树中添加项目。首先向树中添加根项目。根项目的父项为`None`。Abaqus GUI Toolkit提供了多种向树添加项目的方法；但是，最方便的方法是使用`addItemLast`方法，如下例所示：

```
vf = FXVerticalFrame(gb, FRAME_SUNKEN|FRAME_THICK,
    0,0,0,0, 0,0,0,0)
self.tree = FXTreeList(vf, 5, None, 0,
    LAYOUT_FILL_X|LAYOUT_FILL_Y|
    TREELIST_SHOWS_BOXES|TREELIST_ROOT_BOXES|
    TREELIST_SHOWS_LINES|TREELIST_BROWSESELECT)
option1 = self.tree.addItemLast(None, 'Option 1')
self.tree.addItemLast(option1, 'Option 1a')
self.tree.addItemLast(option1, 'Option 1b')
option2 = self.tree.addItemLast(None, 'Option 2')
self.tree.addItemLast(option2, 'Option 2a')
option2b = self.tree.addItemLast(option2, 'Option 2b')
self.tree.addItemLast(option2b, 'Option 2bi')
option3 = self.tree.addItemLast(None, 'Option 3')
```

您还可以指定用于每个树项目的图标。"打开"图标在项目被选中时显示在项目旁边；"关闭"图标在项目未选中时显示。这些图标与分支的展开/折叠状态无关。例如，Microsoft的Windows Explorer使用打开和关闭文件夹图标来显示选中状态。

您可以使用树的`isItemSelected`方法检查项目是否被选中。树widget会在用户点击项目时向其目标发送SEL_COMMAND消息。您可以处理此消息，然后遍历树中的所有项目以找到选中的项目。以下示例使用消息处理程序，假定树是浏览选择模式，允许用户一次只选择一个项目：

```
def onCmdTree(self, sender, sel, ptr):

    w = self.tree.getFirstItem()
    while(w):
        if self.tree.isItemSelected(w):
            # Do something here based on
            # the selected item, w.
            break
        if w.getFirst():
            w=w.getFirst()
            continue
        while not w.getNext() and w.getParent():
             w=w.getParent()
        w=w.getNext()
```

### 3.5.2 选项树列表

`AFXOptionTreeList`widget提供可切换的子项树结构。树结构包括分支以及分支末端的叶。用户可以切换树的叶的开或关。用户也可以切换整个分支的开或关。切换控制分支的所有子项的设置——如果分支关闭，所有子项都会关闭，反之亦然。例如：

```
tree = AFXOptionTreeList(parent, 6)
tree.addItemLast('Item 1')
item = tree.addItemLast('Item 2')
item.addItemLast('Item 3')
item.addItemLast('Item 4')
item.addItemLast('Item 5')
```
