# 69.5 从孤单元创建部件







您可以使用几何编辑工具集中的工具从孤网格单元的面创建几何。当您有一个需要修改的网格部件，但您没有部分或全部几何可用时，此技术很有用。将几何特征添加到孤网格、从头开始重新创建部件几何或从孤单元反向设计几何的决定取决于多种因素，包括：
- 现有部件的复杂性
- 要添加的特征的复杂性
- 修改现有孤单元的难度
- 为整个部件创建新网格的愿望

项目的时间限制以及导出模型几何的能力等因素也可能影响您的决定。

从孤网格创建部件的目标是创建与原始设计意图相匹配的几何，同时为当前和未来的修改提供平台。几何还必须适合通过与当前网格完全关联或创建新网格来创建原生网格。

以下指南将帮助您为孤网格创建几何：

**创建新的几何面**

使用 **Face from element faces**（从单元面创建面）工具 ![](../graphics/icoR_elementFacesToFace.png) 为模型创建外表面几何。每次使用该工具都会从所选外部孤单元面创建单个壳面。该工具包含多种用于选择多个孤单元边界面的独特方法，它会自动将新创建的面缝合到任何相邻几何。 有关更多信息，请参阅["从单元面创建面，" 第 69.7.10 节"](pt06ch69s07hlb10.md)。当您完成创建面后，所有外部孤单元面都应该被几何面覆盖。

**移除多余的边和面**

使用几何编辑工具集中的其他工具移除多余的边、小面以及会限制良好质量网格生成的任何其他特征。如果需要，将部件转换为精确几何。 有关更多信息，请参阅["修复几何的策略，" 第 69.4 节"](pt06ch69s04.md)。

**转换为实体**

如果您正在处理实体部件，请使用 **Create solid from shell**（从壳创建实体）工具 ![](../graphics/ico_repairSolidFromShell.png) 来填充创建的壳面内部的空间。

**分区和网格划分部件**

抑制或删除孤网格，并为部件创建新网格。

**修复后创建分区**

如果您的部件包含分区，几何编辑工具集可能会在修复操作期间删除它们。为了避免此问题，您应该等到完成编辑后再对部件进行分区。
