# 20.9.10 在 Sketcher 中自定义约束的使用







[约束](pt03ch20s12hlb03.md)创建控制草图几何的数值关系。在您编辑草图时，Abaqus/CAE 应用求解方法以避免违反约束。有多种方法可在[拖动](pt03ch20s14hlb01.md)实体或使用其他 sketcher 工具编辑它们时使用。

**注意：**约束求解方法不会阻止草图修改工具覆盖已应用的约束。例如，您可以使用**平移**工具移动具有**固定**约束的直线。

此外，您可以使用**显示约束**选项隐藏约束并减少草图中的混乱，以及**在实体创建期间添加约束**选项添加草绘时隐含的约束。您可以选择线性和角度容差以及可以使用**自动约束**工具自动创建的约束类型。

约束自定义选项适用于整个 Abaqus/CAE 会话。它们不会随单个草图或当前模型保存。

**在 Sketcher 中自定义约束的使用：**

1. 从 Sketcher 工具箱底部，选择 Sketcher 自定义工具![](../graphics/ico_sketchOptions-nls.png)。出现**Sketcher 选项**对话框。
2. 点击**约束**选项卡。
3. 选择用于拖动草图实体或在其他类型的编辑期间所需的约束求解方法。以下选项可用：**标准** 这是编辑的默认求解方法。Abaqus/CAE 移动或修改几何的最小量以满足约束和尺寸。在移动过程中，Abaqus/CAE 还会尝试保持无关几何的刚体外观。**加权** Abaqus/CAE 移动或修改几何的最小量以满足约束和尺寸。在移动期间不考虑无关几何的刚体外观。**最小移动** 这是拖动的默认求解方法。Abaqus/CAE 移动最少数量的实体以满足约束。**松弛** Abaqus/CAE 使用数值求解技术移动几何，以最小化移动平方和。与其他方法相比，这种求解方法会导致更多几何被移动。
4. 如有需要，切换**显示约束**。当**显示约束**打开时，Abaqus/CAE 在草图中显示约束符号。
5. 如有需要，切换**在实体创建期间添加约束**。当**在实体创建期间添加约束**打开时，Abaqus/CAE 会添加草绘时隐含的约束。例如，如果您正在草绘一条线，其预选符号表明 Abaqus/CAE 将应用水平约束且**在实体创建期间添加约束**处于打开状态，则水平约束会添加到该线。
6. 在页面下半部分，切换可用于**自动约束**工具的约束类型。（有关更多信息，请参见["自动约束草图，"第 20.12.1 节](pt03ch20s12hlb01.md)。）
7. 选择完所需的自定义选项后，点击**确定**应用您的更改并关闭**Sketcher 选项**对话框。
