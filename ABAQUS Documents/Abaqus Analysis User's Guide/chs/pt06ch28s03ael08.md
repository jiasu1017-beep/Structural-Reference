### 所需节点坐标

平面应力和平面应变实体连续体单元：*X*、*Y* 

2D声学单元：*X*、*Y* 

3D实体连续体和声学单元：*X*、*Y*、*Z*

轴对称实体连续体和声学单元：*r*、*z*

不在声学无限单元的节点处指定法向；它们将自动计算。详见["无限单元，" 第28.3.1节](pt06ch28s03alm03.md)。

### 单元属性定义

对于二维、平面应变和平面应力单元，必须提供单元的厚度；默认为单位厚度。

对于三维和轴对称实体单元，不需要指定厚度。

对于声学单元，除了厚度外还必须指定参考点。

| **输入文件用法：** | ``` [*SOLID SECTION](../key/key-link.md#usb-kws-msolidsection) ``` |
| --- | --- |

| **Abaqus/CAE用法：** | 仅在Abaqus/CAE中支持声学无限截面。 |
| --- | --- |
|  | 属性模块：**创建截面**：选择**其他**作为截面**类别**，选择**声学无限**作为截面**类型** |

### 基于单元的载荷

无。

### 单元输出

#### 应力、应变和其他张量分量

从Abaqus/Explicit的无限单元没有输出可用。从Abaqus/Standard，对于具有位移自由度的无限单元，应力和其他张量（包括应变张量）均可用。所有张量具有相同的分量。例如，应力分量如下：

| S22 | ![](../graphics/usb_eqn04472.gif) 直接应力或轴对称单元的轴向应力。 |
| --- | --- |

| S33 | ![](../graphics/usb_eqn04473.gif) 直接应力（不适用于平面应力单元）或轴对称单元的周向应力。 |
| --- | --- |

| S12 | ![](../graphics/usb_eqn04474.gif) 剪切应力或轴对称单元的剪切应力。 |
