# 16.9.20 定义向内体积加速度





您可以创建向内体积加速度载荷，以在声学介质边界上的顶点或节点处指定体积加速度。（更多信息，请参见"Abaqus Analysis User's Guide"第6.10.1节"Acoustic, shock, and coupled acoustic-structural analysis"中的"Loads"。）

**创建或编辑向内体积加速度：**

1. 使用以下方法之一显示向内体积加速度载荷编辑器：- 若要创建新的向内体积加速度载荷，请遵循"创建载荷"，第16.8.1节中概述的过程（**类别**：`Acoustic`；**所选步骤的类型**：`Inward volume acceleration`）。- 若要使用菜单或管理器编辑现有的向内体积加速度载荷，请参见"编辑依赖于步骤的对象"，第3.4.13节。若要编辑应用载荷的区域，请参见"编辑应用规定条件的区域"，第16.8.4节。
2. 单击**分布**字段右侧的箭头，并从出现的列表中选择您的选项：- 选择**均匀**以定义在区域上均匀的载荷。- 选择分析场以定义空间变化载荷。只有对该载荷类型有效的分析场才会在选择列表中显示。或者，您可以单击![](../graphics/ico_analyticalField.png)创建新的分析场。（更多信息，请参见第58章"分析场工具集"。）
3. 在**幅值**字段中，输入体积加速度（单位[LT<sup>-2</sup>](../popups/usb-int-iconventions-unitsym.md)）。
4. 如果需要，单击**幅值**字段右侧的箭头，并从出现的列表中选择您选择的幅值。或者，您可以单击![](../graphics/ico_amplitudeSmall.png)创建新的幅值。（更多信息，请参见第57章"幅值工具集"。）
5. 单击**确定**保存您的数据并退出编辑器。
