# 11.14.1 定义放样截面







放样截面表示放样特征在沿放样路径的特定点处具有的形状。创建放样特征至少需要两个截面。您可以创建额外的截面来控制起始截面和终止截面之间放样的形状。在实体或切割放样中，每个放样截面必须是没有分支的闭合环。在壳放样中，放样截面可以全部为开放的或全部为闭合的。您可以定义平面或非平面放样截面来创建放样特征。放样创建后，截面数量及其在放样中的顺序无法更改。

您通过从当前视口中的零件上现有边缘进行选择来创建放样截面。可以选择任何边缘，例如：- 定义拉伸、旋转或扫掠特征的边缘。- 定义平面线框或壳特征的边缘。- 样条线框特征。

您可以使用来自多个特征的单个边缘来定义单个截面。然而，在基准面上绘制的平面线框是定义放样截面最简单的方法之一。使用最简单的方法定义放样截面将使您获得更多控制权，并产生更稳定的放样特征。

您不能直接修改放样截面。创建放样特征后，您可以使用特征操作工具集来编辑创建放样截面中使用的边缘的特征。移动用于放样截面中的顶点或边缘将改变该截面以及相应放样特征的形状。
