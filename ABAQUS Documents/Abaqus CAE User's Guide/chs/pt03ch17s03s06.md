# 17.3.6 网格技术颜色编码







当选择 **Mesh defaults** 颜色映射时，Abaqus/CAE 使用不同的颜色来指示当前分配给区域的网格划分技术（如果有的话）。例如，如果实体区域可以使用结构化网格划分技术进行网格划分，则当您进入网格模块时，该区域会变为绿色；绿色表示结构化网格划分技术已分配给该区域。黄色表示扫掠网格划分技术已应用于区域。如果区域使用当前分配的单元形状无法网格划分，则当您进入网格模块时，该区域会变为橙色。粉色或浅棕色的区域分别分配了自由和自底向上网格划分技术。

**注意：**您必须使用 **Mesh Controls** 对话框将自底向上网格划分技术分配给区域。Abaqus/CAE 不会自动分配自底向上网格划分技术，也不会指示分配了自底向上技术的区域是否也可以使用自顶向下技术进行网格划分。（有关更多信息，请参见"分配网格控制"第 17.18.1 节。）

您可以通过将区域分割为具有更简单拓扑的较小区域、更改分配给区域的单元形状或使用虚拟拓扑工具集来更改适用的网格划分技术。
