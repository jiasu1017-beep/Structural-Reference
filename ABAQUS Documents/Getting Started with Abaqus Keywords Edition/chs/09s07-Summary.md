# 9.7 总结

- Abaqus/Explicit 采用中心差分法则对运动学量进行显式时间积分。
- 显式方法需要许多小的时间增量。由于不需要求解联立方程组，每个增量计算成本较低。
- 随着模型规模的增大，显式方法相对于隐式方法可节省大量成本。
- **稳定性极限**是指能够用于推进运动状态并保持精度的最大时间增量。
- Abaqus/Explicit 在整个分析过程中自动控制时间增量大小以维持稳定性。
- 随着材料刚度的增加，稳定性极限减小；随着材料密度的增加，稳定性极限增大。
- 对于包含单一材料的网格，稳定性极限大致与最小单元尺寸成正比。
- 通常，在 Abaqus/Explicit 中使用质量比例阻尼来抑制低频振荡，使用刚度比例阻尼来抑制高频振荡。
- 在某些情况下，Abaqus/Explicit 分析可能会变得不稳定。本章中的示例问题描述了如何识别和纠正不稳定现象。
