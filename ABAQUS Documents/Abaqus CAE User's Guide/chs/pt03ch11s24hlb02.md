# 11.24.1 创建拉伸切割





从主菜单栏选择**形状**![](../graphics/images/arrow.gif)**切割**![](../graphics/images/arrow.gif)**拉伸**，通过当前视口中的零件几何创建拉伸切割。无论当前视口中零件的建模空间如何，拉伸切割工具始终可用。

您通过在选定面上绘制切割的二维横截面并定义Abaqus/CAE拉伸切割的距离来创建三维零件中的拉伸切割。您可以选择以下方法之一来定义拉伸切割的距离：
- **盲孔**从草图平面沿选定方向延伸切割，但仅达到指定深度。
- **至面**从草图平面拉伸切割到选定的面。
- **贯穿所有**从草图平面沿选定方向穿过几何延伸切割。

**创建拉伸切割：**

1. 从主菜单栏，选择**形状**![](../graphics/images/arrow.gif)**切割**![](../graphics/images/arrow.gif)**拉伸**。
2. 如果需要，指定要用于为拉伸切割特征的草图选择原点的方法。
3. 如果当前视口包含二维或轴对称平面零件，Abaqus/CAE会进入草图编辑器，您在零件平面上绘制拉伸切割的闭合轮廓。
4. 在提示区域，单击**完成**表示完成绘制轮廓。
5. 如果当前视口包含三维零件，Abaqus/CAE显示零件的原始方向。
6. 如果需要，选择**扭曲**或**拔模**。
7. 单击**确定**拉伸轮廓。
8. 如果选择了**至面**，Abaqus/CAE会提示您选择要拉伸轮廓到的面。
