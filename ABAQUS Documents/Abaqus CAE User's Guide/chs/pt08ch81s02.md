# 81.2 在哪里可以获取插件？





Abaqus 安装盘上提供了多个插件。您可以通过从主菜单栏选择 **Plug-ins**![](../graphics/images/arrow.gif)**Abaqus** 或 **Tools** 来查看这些插件。

有关如何查看描述插件及其用法的文档，请参见 ["如何获取插件的相关信息？，" 第81.3节](pt08ch81s03.md)。

[SIMULIA Learning Community](http://swym.3ds.com/#community:73) 中提供了其他插件，该社区提供插件示例，并让您访问促进 Abaqus 脚本接口和 Abaqus GUI Toolkit 发展的用户社区。点击 **Blog** 并按 **Process Automation** 筛选可以浏览该社区中的插件示例。您也可以编写自己的插件，如本章其余部分所述。
