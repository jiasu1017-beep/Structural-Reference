# 20.4.2 Sketcher 工作表和网格







当您进入 Sketcher 并创建新草图或编辑现有草图时，Abaqus/CAE 会在当前视口中显示一个用于草绘的工作表。此外，视口左下角的三轴指示器表示零件或装配相对于 Sketcher 工作表的方向。工作表始终为正方形，其高度和宽度由工作表大小决定。Abaqus/CAE 根据您正在草绘的内容确定工作表大小和原点位置：
- 如果您正在草绘新零件的基础特征，工作表大小与您创建零件时提供的零件近似大小相同。工作表的原点位于零件坐标系的原点。类似地，如果您正在创建独立草图，工作表大小与您创建草图时提供的草图近似大小相同。
- 如果您正在向零件或装配添加特征，默认工作表大小取决于您正在草绘的面的大上。工作表的原点位于所选面的质心。如果您选择基准平面作为草绘平面，工作表的原点位于零件或装配的中心；工作表大小取决于零件或装配的总大小。

如果工作表大小与您尝试草绘的几何大小不符，您可以使用 Sketcher 自定义选项增加或减小工作表大小。要访问 Sketcher 自定义选项，请从 Sketcher 工具箱中选择自定义工具![](../graphics/ico_sketchOptions-nls.png)。您可能需要使用放大工具![](../graphics/ico_viewZoom.png)在视口中查看整个 Sketcher 工作表。

Abaqus/CAE 用不可见网格点网格覆盖工作表，以帮助您在绘制、移动、调整大小或重塑对象时定位光标。默认情况下，当您将光标移到网格点附近时，光标会自动移动或捕捉到该点。这种行为允许您在轻松地将光标精确定位在网格点上的同时，在光标远离任何网格点时为您提供完全控制。如果更方便，您可以[禁用](pt03ch20s09hlb02.md)捕捉行为，以便完全控制光标。为帮助您可视化 Sketcher 网格下的网格点，Abaqus/CAE 以选定间隔显示穿过网格点的可见网格线；例如，每隔一个网格点。

例如，[图 20-2](pt03ch20s04s02.md#ske-customgrid)显示间距设置为 2 个单位的工作表的放大视图。

**图 20-2** Sketcher 网格的放大视图。

![](../graphics/ske-customgrid-nls.png)

在此示例中，Sketcher 同时显示主网格线（实线样式）和次网格线（细虚线样式）。仅当放大倍数较高时，次网格线才会出现在 Sketcher 中。此外，工作表中的粗虚线表示草图的 *X* 和 *Y* 轴；这些线在草图原点处相交。

您可以[自定义](pt03ch20s09hlb04.md)网格的外观和行为，包括选择网格点间距、覆盖网格点的可见网格线间距、次间隔数和 Sheet size。您还可以通过移动网格原点和旋转网格来重新对齐网格相对于草图。您的草图可以超出 Sketcher 网格；然而，如果您发现需要在网格外草绘，建议您增加其大小以包含整个草图。如果更改网格原点或旋转，Sketcher 会在您继续修改草图时以两种形式显示光标坐标。网格坐标使用新网格原点或旋转表示光标位置；草图坐标基于原始网格位置表示位置。

如果零件坐标远离零件原点，模型的解析可能会受到影响精度。例如，将草图导入以创建零件的基础特征后，草图可能相对远离 Sketcher 原点；因此，零件将远离其原点。为了提高精度，您可能需要使用平移工具![](../graphics/ico_sketchTranslate.png)将草图几何移近 Sketcher 网格的原点，从而将零件移近其原点。您不应将草图原点移近几何。将 Sketcher 网格原点移近草图只是一种图形便利，对底层坐标精度没有影响。
