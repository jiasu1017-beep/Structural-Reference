# 20.2.2 aStudy.constrain()





### 20.2.2 **_aStudy_.constrain()**: 约束参数化研究中的参数值组合。

**产品:** Abaqus/Standard  Abaqus/Explicit

此命令用于定义参数值组合的约束；违反任何约束的组合将从参数化研究中消除。

##### **参考**

- ["参数化研究脚本," 第20.1.1节](pt04ch20s01aus108.md)

### **命令:**

**_aStudy_.constrain** (*constraint expression*)

### **必需数据:**

***constraint expression***

提供用匹配引号括起来的约束表达式。此表达式可能涉及参数、数字和先前定义的Python变量之间的运算；例如，`'height*width < maxArea-2.0'`。约束可以是等式或不等式。
