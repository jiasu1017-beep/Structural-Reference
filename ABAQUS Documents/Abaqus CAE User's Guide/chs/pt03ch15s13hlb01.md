# 15.13.1 定义通用接触





在Abaqus/Standard中，通用接触只能在初始步骤中定义，此通用接触定义在所有后续步骤中处于活动状态。在Abaqus/Explicit中，您可以在任何分析步骤或初始步骤中定义通用接触；一个步骤中只能有一个通用接触相互作用处于活动状态。有关通用接触的简要概述，请参见「理解相互作用」，第15.3节。更详细的讨论，请参见「在Abaqus/Standard中定义通用接触相互作用」，Abaqus Analysis User's Guide第36.2.1节，以及「在Abaqus/Explicit中定义通用接触相互作用」，Abaqus Analysis User's Guide第36.4.1节。

在Abaqus/Standard中，通用接触定义可以创建模型中所有外部面和壳周缘边缘的相互作用。在Abaqus/Explicit中，通用接触定义可以创建模型中所有外部面、解析刚性表面、壳周缘边缘、基于梁和桁架的边缘以及Eulerian材料边界的相互作用；但是，您不能指定两个解析刚性表面之间的通用接触（包括自接触）。

在Abaqus/Explicit中，您可以通过在「Step」模块中使用历史输出请求编辑器来获取通用接触域中特定表面的接触数据。在编辑器的「Domain」部分，选择「General contact surface」并从出现的菜单中选择表面。更多信息，请参见「创建输出请求」，第14.12.1节。

**定义通用接触的步骤：**

1. 从主菜单栏选择 ****Interaction**![](../graphics/images/arrow.gif)**Create****。**提示：**您也可以使用「相互作用」模块工具箱中的![](../graphics/ico_interaction.png)工具创建通用接触相互作用。
2. 在出现的「Create Interaction」对话框中，执行以下操作：- 为相互作用命名。有关命名对象的更多信息，请参见「使用基本对话框组件」，第3.2.1节。- 选择创建相互作用的步骤。在Abaqus/Standard中，通用接触只能在初始步骤中创建。- 根据模型中定义的分析步骤，选择「General contact (Standard)」或「General contact (Explicit)」类型的相互作用。
3. 点击「Continue」关闭「Create Interaction」对话框。此时会出现「Edit Interaction」对话框。
4. 使用以下方法之一指定接触域：- 选择「All* with self」为所有允许的单元面和模型实体指定接触（包括自接触）。这是定义接触域的最简单方法。- 要指定单个接触表面对：1. 选择「Selected surface pairs」，然后点击![](../graphics/ico_edit.png)。此时会出现「Edit Included Pairs」对话框。默认情况下，当您从列表或表中选择一个表面时，Abaqus/CAE会在视口中高亮显示该表面；但是，高亮不适用于「(All*)」、「(Self)」和Eulerian材料表面。您可以关闭对话框底部的「Highlight selected regions」以关闭选择高亮。2. 从左侧第一列的现有表面列表中选择一个或多个表面。选择「(All*)」指定包含所有允许的单元面和模型实体的表面。**提示：**您可以点击![](../graphics/ico_surface.png)定义新表面并将其添加到列表中。请参见「创建表面」，第73.3.3节，获取定义表面的说明。3. 从第二列的现有表面列表中选择第二个或多个表面以定义表面对。- 当在任一列中选择多个表面时，所有可能的组合都会在表中生成。- 要指定自接触，请在第二列中选择相同的表面名称或「(Self)」。- 表面的指定顺序对分析无关紧要。4. 点击对话框中间的箭头![](../graphics/usi-itn-contactdomainarrow.png)将表面对转移到将包含在接触域中的对列表中。对话框右侧的表会更新以反映您的选择（表面对的顺序无关紧要）。5. 根据需要重复上述步骤以完全定义接触域包含。如果您想删除包含的对，请选择行并点击![](../graphics/ico_delete.png)。6. 点击「OK」保存您的选择并关闭「Edit Included Pairs」对话框。相互作用编辑器重新出现，并更新了所选包含在接触域中的表面对数量的信息。
5. 如有必要，选择要从接触域中排除的表面对。1. 点击「Excluded surface pairs」旁边的「Edit」。此时会出现「Edit Excluded Pairs」对话框。默认情况下，当您从列表或表中选择一个表面时，Abaqus/CAE会在视口中高亮显示该表面；但是，高亮不适用于「(All*)」、「(Self)」和Eulerian材料表面。您可以关闭对话框底部的「Highlight selected regions」以关闭选择高亮。2. 从左侧第一列的现有表面列表中选择一个或多个表面。选择「(All*)」指定包含所有允许的单元面和模型实体的表面。**提示：**您可以点击![](../graphics/ico_surface.png)定义新表面并将其添加到列表中。请参见「创建表面」，第73.3.3节，获取定义表面的说明。3. 从第二列的现有表面列表中选择第二个或多个表面以定义表面对。- 当在任一列中选择多个表面时，所有可能的组合都会在表中生成。- 要指定要排除的自接触，请在第二列中选择相同的表面名称或「(Self)」。- 表面的指定顺序对分析无关紧要。- 如果排除的区域与包含的区域重叠，则接触排除将优先于接触包含。4. 点击对话框中间的箭头![](../graphics/usi-itn-contactdomainarrow.png)将表面对转移到将从接触域中排除的对列表中。对话框右侧的表会更新以反映您的选择（表面对的顺序无关紧要）。5. 根据需要重复上述步骤以完全定义接触域排除。如果您想删除排除的对，请选择行并点击![](../graphics/ico_delete.png)。6. 点击「OK」保存您的选择并关闭「Edit Excluded Pairs」对话框。相互作用编辑器重新出现，并更新了所选排除在接触域之外的表面对数量的信息。
6. 在相互作用编辑器底部指定「Attribute Assignments」。在Abaqus/Standard中，您可以修改通用接触相互作用处于活动状态的任何步骤中的接触属性或稳定化，但所有其他属性为整个分析分配。在Abaqus/Explicit中，您可以指定或修改通用接触相互作用处于活动状态的任何步骤中的属性。您可以指定以下分配：- **Contact Properties**。详细说明，请参见「为通用接触指定和修改接触属性分配」，第15.13.2节。- **Contact Initializations**（仅限Abaqus/Standard）。详细说明，请参见「为通用接触指定和修改接触初始化分配」，第15.13.3节。- **Contact Stabilizations**（仅限Abaqus/Standard）。详细说明，请参见「为通用接触指定和修改接触稳定化分配」，第15.13.4节。- **Surface Properties**。详细说明，请参见「为通用接触指定表面属性分配」，第15.13.5节。- **Contact Formulation**。详细说明，请参见「为通用接触指定主从分配」，第15.13.6节。
7. 点击「OK」创建相互作用并关闭编辑器。
