### 概述

桁架单元：
- 是细长的结构构件，只能传递轴向力（非结构连接单元见["一维实体（连接）单元库，" 第 28.1.2 节](pt06ch28s01ael01.md)）；以及
- 不传递弯矩。

### 典型应用

桁架单元在二维和三维中用于模拟细长的、线状结构，这些结构仅沿单元轴线或中心线承受荷载。不支持垂直于中心线的弯矩或力。

二维桁架单元可用于轴对称模型，以表示螺栓或连接器等组件，其中应变仅根据 *r*–*z* 平面中的长度变化计算。二维桁架也可用于在 Abaqus/Standard 中定义接触应用的主面（见["接触相互作用分析：概述，" 第 36.1.1 节](pt09ch36s01abo33.md)）。在这种情况下，主面外法线方向对于正确检测接触至关重要。

Abaqus/Standard 中提供的 3 节点桁架单元通常用于模拟结构中的曲线加强索，例如钢筋混凝土中的预应力钢筋或海上工业中使用的细长管道。

### 选择合适的单元

2 节点直线桁架单元（用于位置和位移的线性插值，应力恒定）在 Abaqus/Standard 和 Abaqus/Explicit 中均可用。此外，在 Abaqus/Standard 中提供 3 节点曲线桁架单元（用于位置和位移的二次插值，因此应变沿单元线性变化）。

在 Abaqus/Standard 中，应力/位移桁架、耦合温度-位移桁架和压电桁架的混合版本均可用。

#### 混合应力/位移桁架单元

在 Abaqus/Standard 中，二维和三维的应力/位移桁架的混合版本（其中轴力作为额外未知量处理）均可用。当桁架代表一个刚度远大于整体结构模型刚度的刚性连接时，这些单元非常有用（用于抵消数值病态对方程的影响）。在这种情况下，混合桁架提供了一个真正刚性连接的替代方案，通过多点约束（见["通用多点约束，" 第 35.2.2 节](pt08ch35s02aus130.md)）或刚性单元（见["刚性单元，" 第 30.3.1 节](pt06ch30s03alm23.md)）建模。

#### 耦合温度-位移桁架单元

耦合温度-位移桁架单元在 Abaqus/Standard 中的二维和三维均可用。这些单元将温度作为额外的自由度（11）。有关 Abaqus/Standard 中完全耦合温度-位移分析的信息，请参见["完全耦合热应力分析，" 第 6.5.3 节](pt03ch06s05at19.md)。

#### 压电桁架单元

压电桁架单元在 Abaqus/Standard 中的二维和三维均可用。这些单元将电势作为额外的自由度（9）。有关压电分析的信息，请参见["压电分析，" 第 6.7.2 节](pt03ch06s07at21.md)。

|  | 其中 ELSET 参数指一组桁架单元。 |

| **Abaqus/CAE 用法：** | Property 模块：**Create Section**：选择 **Beam** 作为 section **Category**，选择 **Truss** 作为 section **Type****Assign**![](../graphics/images/arrow.gif)**Section****：选择区域 |
| --- | --- |

#### 定义桁架单元的横截面积

可以将与桁架单元关联的横截面积作为截面定义的一部分进行定义。如果未指定横截面积的值，则假定为单位面积。

当桁架单元用于大位移分析时，更新后的横截面积通过假设桁架由不可压缩材料制成来计算，无论实际材料定义如何。此假设仅在应变较大的情况下产生影响。采用此假设是因为大应变时桁架最常见的应用涉及屈服金属行为或橡胶弹性，在这些情况下材料实际上是不可压缩的。因此，当轴向应变不是无限小小时，线性弹性桁架单元不会提供与线性 SPRINGA 弹簧单元相同的力-位移响应。

| **输入文件用法：** | ``` [*SOLID SECTION](../key/key-link.md#usb-kws-msolidsection), ELSET=*name* *横截面积* ``` |
| --- | --- |

| **Abaqus/CAE 用法：** | Property 模块：**Create Section**：选择 **Beam** 作为 section **Category**，选择 **Truss** 作为 section **Type**：**Cross-sectional area:** *横截面积* |
| --- | --- |

#### 为一组桁架单元分配材料定义

必须将材料定义与每个实体截面定义关联。桁架单元不允许材料方向。

| **输入文件用法：** | ``` [*SOLID SECTION](../key/key-link.md#usb-kws-msolidsection), MATERIAL=*name* ``` |
| --- | --- |
|  | [*SOLID SECTION](../key/key-link.md#usb-kws-msolidsection) 选项上 ORIENTATION 参数的任何值都将被桁架单元忽略。 |

| **Abaqus/CAE 用法：** | Property 模块：**Create Section**：选择 **Beam** 作为 section **Category**，选择 **Truss** 作为 section **Type**：**Material:** *name* |
| --- | --- |

### 在大位移隐式分析中使用桁架单元

桁架单元没有初始刚度来抵抗垂直于其轴线的荷载。如果在 Abaqus/Standard 中沿垂直于无应力桁架线的方向施加荷载，可能会导致数值奇异性和收敛失败。在大位移隐式分析的第一次迭代后，垂直于单元初始线的刚度会发展出来，有时可以使分析克服数值问题。

在某些情况下，先沿桁架轴线施加荷载或包含初始拉应力可以克服这些数值奇异性。但是，必须选择荷载或初始应力的幅值，使最终解决方案不受影响。
