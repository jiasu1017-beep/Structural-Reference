# 4.1.11 三点连杆机构分析中衬套连接器的应用

**产品：** Abaqus/Standard  Abaqus/Explicit   

本示例说明了使用详细有限元衬套模型来校准衬套的本构行为，以便在后续分析中使用非常便宜的 2 节点连接器元件表示衬套。这种方法在减少计算成本的同时准确捕获衬套的宏观响应方面是有效的。

### 几何与材料

三点连杆（TPL）的几何形状包括前衬套和后衬套。后衬套是一个空心圆柱体，橡胶部分封闭在两个薄钢管之间。橡胶部分有两个对称放置的不同大小的切口。前衬套由三根非共轴钢管和它们之间的两根橡胶部分组成。

### 模型

为了表征衬套的非线性本构行为，在 Abaqus/Standard 中对前衬套执行一系列静态分析，在 Abaqus/Explicit 中对后衬套执行一系列准静态分析。

### 结果与讨论

可以看到，当前衬套行为在局部 3 方向和局部 2 方向上受到局部 3 方向预载量的显著影响。TPL 中三种建模方法计算的 Mises 应力非常相似。

### 文件

- tpl_fb_uncoupled.inp - 前衬套模型模板
- tpl_rb_uncoupled.inp - 后衬套模型模板
- tpl_full_loadcase1.inp - 完全网格化模型分析
- tpl_connectors_loadcase1.inp - 衬套连接器模型分析
- tpl_substructure_gen.inp - 子结构生成分析
