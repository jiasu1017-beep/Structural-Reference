# 10.3.1 在导入期间修复零件





您可以导入零件，然后使用"零件"模块中的几何编辑工具集来应用可能需要使零件被Abaqus/CAE使用的任何修复操作；有关更多信息，请参阅[第69章，"几何编辑工具集"](pt06ch69.md)。或者，您可以导入零件并在导入过程中修复零件，如本节所述。

当您导入零件时，Abaqus/CAE会扫描文件内容并显示一个带有**名称-修复（Name-Repair）**选项卡页面的对话框，允许您控制以下内容：

**名称（Name）**

零件的名称。

**修复选项（Repair Options）**

对于大多数支持的文件格式，Abaqus/CAE会在导入过程中自动修复零件。但是，在导入IGES或VDA-FS格式文件时，Abaqus/CAE提供以下额外选项：
- **转换为解析表示（Convert to analytical representation）**
- **缝合间隙（Stitch gaps）**

在大多数情况下，这些选项的默认设置会提供最佳结果。更多信息，请参阅["编辑技术概述，"第69.2节"](pt06ch69s02.md)。

**零件过滤器（Part Filter）**

以下文件格式可以在单个文件中包含多个零件：
- ACIS
- CATIA V4
- Elysium中性文件（CATIA V5或Pro/ENGINEER）
- Parasolid
- STEP

默认情况下，Abaqus/CAE会导入文件中的所有零件。如果您导入文件中的所有零件，可以为每个零件创建单独的Abaqus零件，或者可以将它们组合成单个Abaqus零件；此外，如果组合这些零件，Abaqus/CAE使您能够使用用户指定的缝合容差值将这些导入的零件缝合在一起。或者，您可以切换**导入零件号（Import part number）**并输入要从文件导入的单个零件的编号。Abaqus/CAE会指示是否有任何零件存在有效性或精度问题。

在某些情况下，当您将零件导入Abaqus/CAE时，零件的几何形状包含没有用途的额外边缘和顶点。额外的几何形状会将面分割成额外的面，将边缘分割成额外的边缘，从而导致不必要的复杂性。额外的几何形状会过度影响您的网格，您应该使用几何编辑工具集来删除冗余边缘和顶点。您也可以在"网格"模块中使用虚拟拓扑来合并小面和边缘，并忽略不必要的顶点和边缘；更多信息，请参阅[第75章，"虚拟拓扑工具集"](pt06ch75.md)。
