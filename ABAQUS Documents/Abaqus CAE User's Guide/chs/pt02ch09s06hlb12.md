# 9.7.12 压缩当前模型数据库的文件大小





从主菜单栏选择****文件（File）**![](../graphics/images/arrow.gif)**压缩MDB（Compress MDB）****来压缩当前模型数据库（MDB）。压缩MDB会尝试减小文件大小。如果您从模型中删除了多个项目，这种更改会最明显。

如果您选择****文件（File）**![](../graphics/images/arrow.gif)**另存为（Save As）****将文件保存为新文件名，Abaqus/CAE会使用压缩功能。
