# 15.14.5 定义流体交换相互作用属性





您使用流体交换相互作用属性来定义空腔与环境之间或两个空腔之间的流体传递方法。流体交换相互作用属性只能由流体交换相互作用引用。有关更多信息，请参见"定义流体交换相互作用"，第15.13.12节。

**定义流体交换相互作用属性的步骤：**

1. 从主菜单栏选择 ****Interaction**![](../graphics/images/arrow.gif)**Property**![](../graphics/images/arrow.gif)**Create****。
2. 在出现的「Create Interaction Property」对话框中，执行以下操作：- 为相互作用属性命名。有关命名对象的更多信息，请参见"使用基本对话框组件"，第3.2.1节。- 选择「Fluid exchange」类型的相互作用属性。
3. 点击「Continue」关闭「Create Interaction Property」对话框。Abaqus/CAE会打开「Edit interaction property」对话框。
4. 选择以下流体交换定义之一：**Bulk viscosity** 流体交换速率基于粘性阻力系数和流体动力阻力系数。两个系数都可能依赖于平均绝对压力、平均温度以及任何用户定义场变量的平均值。**Mass flux** 流体交换速率基于流体交换相互作用中定义的有效面积每单位面积的质量流量。**Mass rate leakage** 流体交换速率基于由主空腔与环境之间或主空腔与第二个空腔之间压力差的绝对值驱动的质量流量。质量流量和压力差的绝对值都从零开始且必须为正。质量泄漏率可能依赖于平均绝对压力、平均温度以及任何用户定义场变量的平均值。**Volume flux** 流体交换速率基于流体交换相互作用中定义的有效面积每单位面积的体积流量。**Volume rate leakage** 流体交换速率基于由主空腔与环境之间或主空腔与第二个空腔之间压力差的绝对值驱动的体积流量。体积流量和压力差的绝对值都从零开始且必须为正。体积泄漏率可能依赖于平均绝对压力、平均温度以及任何用户定义场变量的平均值。
5. 如果您选择了「Bulk viscosity」、「Mass rate leakage」或「Volume rate leakage」流体交换定义，请执行以下操作来完成流体交换属性定义：1. 切换所需的选项以向表格添加数据列。您可以添加压力或温度依赖性数据，也可以添加场变量。2. 在数据表中，输入作为温度、压力和场变量函数的相应数据。您可以使用键盘输入数据。或者，您可以在表格中任意位置点击鼠标按钮3，以查看用于指定表格数据的选项列表。有关每个选项的详细信息，请参见"输入表格数据"，第3.2.7节。
6. 如果您选择了「Mass flux」或「Volume flux」流体交换定义，请分别输入每单位面积的质量流量或每单位面积的体积流量。
7. 点击「OK」创建流体交换相互作用属性并退出编辑器。
