# 17.11.6 提高自底向上区域的边界网格质量





源侧和连接边上的边界网格质量是影响扫掠网格质量的主要因素。创建边界网格允许您在用它们创建自底向上网格之前提高其质量。例如，要使用非关联几何面和非关联单元面的组合来创建可接受的源侧，您可能需要创建边界网格并合并二维网格的节点。同样，您不能使用几何面和单元面的组合来创建连接边，但您可以创建单个边界网格并使用其二维单元和相邻三维网格的单元面来创建连接边。

您可以使用查询工具集查询边界网格。此外，您可以使用网格验证工具检查边界网格的质量。网格验证工具允许您检查所有边界单元的质量，或者您可以使用选择过滤器仅检查选定面的边界单元的质量。

要提高自底向上区域的边界网格质量，您可以尝试以下操作：
- 修改自底向上区域面的网格控制。更多信息，请参见["分配网格控制，"第17.18.1节](pt03ch17s18s01.md)。
- 使用几何诊断工具查找可能影响网格质量的小实体，例如短边、小面和具有小面角的面。您可以创建一个包含这些小实体的集合。更多信息，请参见["使用几何诊断工具，"第71.2.4节](pt06ch71s02hlb04.md)。
- 使用"几何编辑"工具集移除多余的边和顶点。您也可以移除一个面并缝合产生的间隙。更多信息，请参见["编辑技术概述，"第69.2节](pt06ch69s02.md)。
- 使用"虚拟拓扑"工具集忽略极小的边或面。更多信息，请参见["我可以使用虚拟拓扑工具集做什么？"第75.2节](pt06ch75s02.md)。
- 添加分割以减少狭长面或单元的长宽比。更多信息，请参见[第70章，"分割"工具集](pt06ch70.md)。
- 使用"编辑网格"工具集修改边界网格。您可以在网格模块中执行以下操作：
  - 编辑节点
  - 折叠单元边
  - 交换一对相邻三角形单元的对角线
  - 分割单元边
更多信息，请参见["我可以使用编辑网格工具集做什么？"第64.1节](pt06ch64s01.md)。
