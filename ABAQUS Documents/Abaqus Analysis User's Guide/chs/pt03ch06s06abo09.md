# 6.6.1 流体动力学分析过程：概述





### 概述

Abaqus/CFD 提供了先进的计算流体动力学能力，在 Abaqus/CAE 中提供广泛的预处理和后处理支持。这些可扩展的并行 CFD 模拟能力解决了广泛的非线性耦合流体-热和流体-结构问题。

Abaqus/CFD 可以解决以下类型的不可压缩流动问题：

**层流和湍流**：稳态或瞬态的内部或外部流动，跨越广泛的雷诺数范围，并涉及复杂几何形状，可以使用 Abaqus/CFD 进行模拟。这包括由空间变化的分布体积力引起的流动问题。

**热对流**：涉及热传递并需要能量方程的问题，以及可能涉及浮力驱动流动（即自然对流）的问题，也可以使用 Abaqus/CFD 求解。这种问题类型包括适用于广泛普朗特数范围的热湍流传递。

**变形网格 ALE**：Abaqus/CFD 包括使用运动方程、热传递和湍流传输的任意拉格朗日-欧拉（ALE）描述执行变形网格分析的能力。变形网格问题可能包括引起流体流动的预定边界运动，或者是边界运动与流体流动相对独立的 FSI 问题。
有关更多详细信息，请参阅『[不可压缩流体动力学分析，第 6.6.2 节](pt03ch06s06aus48.md)'。

### Abaqus/CFD 中场的激活

在 Abaqus/CFD 中，活动场（自由度）由分析过程和指定的选项决定，例如湍流模型和辅助传输方程。例如，将能量方程与不可压缩流动过程结合使用会激活速度、压力和温度自由度。有关可用自由度的完整列表，请参阅『[Abaqus/CFD 中的边界条件，第 34.3.2 节中的"活动自由度"](pt07ch34s03aus119.md#usb-prc-pboundarycfd-dofs)'。
