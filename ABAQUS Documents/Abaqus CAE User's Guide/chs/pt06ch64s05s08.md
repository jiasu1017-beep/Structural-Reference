# 64.5.8 平滑外部节点





从主菜单栏选择****网格**![](../graphics/images/arrow.gif)**编辑****可以平滑零件或装配表面上节点的位置。节点平滑操作可以在不重新网格划分区域或不手动重新定位节点的情况下，改善网格密度过渡或以其他方式改善局部网格质量。您可以在将表面用作自底向上网格或扫掠网格的源之前，使用节点平滑来改善表面上的网格。

Abaqus/CAE对您选择的每个节点应用拉普拉斯平滑算法。所选节点会被重新定位，以平衡它们与相邻节点的距离，创建更平滑的表面网格。

您只能选择零件或装配的外部节点进行平滑。边界节点不会通过平滑操作重新定位。

**平滑外部节点的步骤：**

1. 进入"网格"模块，并执行以下操作之一： - 从上下文栏的**对象**字段中选择**装配**。 - 从上下文栏的**对象**字段中选择**零件**，然后从列表中选择一个零件。
2. 从主菜单栏选择****网格**![](../graphics/images/arrow.gif)**编辑****。Abaqus/CAE显示**编辑网格**对话框。 **提示：**您也可以使用位于"网格"模块工具箱底部的![](../graphics/ico_meshEdit.png)工具显示**编辑网格**对话框。
3. 在对话框中执行以下操作： 1. 在**类别**字段中选择**节点**。 2. 从**方法**列表中选择**平滑**。Abaqus/CAE显示现有节点。在隐藏和着色模式下，仅显示可见节点。
4. 选择要平滑的节点。您可以**按住[Shift]键并点击**各个节点将其添加到您的选择中，或**按住[Ctrl]键并点击**所选节点将其从选择中移除。 **提示：**您可以使用**选择**工具栏更改选择行为。有关更多信息，请参见[第6章，"在视口中选择对象"](pt01ch06.md)。如果您希望从现有节点集列表中进行选择，请执行以下操作： 1. 点击提示区域右侧的**集合**。Abaqus/CAE显示**区域选择**对话框，其中包含您已创建的节点集列表。 2. 选择节点集，然后点击**继续**。 **注意：**默认选择方法基于您最近使用的选择方法。要切换到另一种方法，请点击提示区域右侧的**在视口中选择**或**集合**。
5. 在提示区域点击**完成**。Abaqus/CAE会平滑所选节点的位置。 **提示：**如果要将节点恢复到之前的位置，点击**编辑网格**对话框中的**撤销**可以撤消平滑。
6. 根据需要重复步骤4和5以平滑其他节点。您可以对同一节点多次应用平滑。经过几次迭代后，除非移动其他周围节点，否则节点将不再被平滑算法移动。
7. 完成节点调整后，点击提示区域中的取消按钮![](../graphics/afxI_cancel.png)退出过程。
