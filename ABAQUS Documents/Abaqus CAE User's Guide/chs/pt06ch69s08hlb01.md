# 69.8.1 转换为解析







当您将部件转换为其解析表示时，Abaqus/CAE 尝试将边、面和胞的内部定义更改为可以用解析法表示的更简单形式。例如，几乎平面的平面将转换为表示该平面的方程。转换为解析表示通常提供以下优点：
- 部件处理更快。
- 转换后的实体在特征操作期间可用。
- 几何得到改善。
- 如果您随后需要缝合边，缝合操作更有可能成功。

操作完成后，您可以使用查询工具集中的几何诊断工具来定位任何无效和不精确几何的区域。 有关更多信息，请参阅["在 Part 模块中使用查询工具集，" 第 11.16.4 节"](pt03ch11s16s04.md)。

**转换为解析：**

1. 从主菜单栏中选择 ****Tools**![](../graphics/images/arrow.gif)**Geometry Edit****（工具→几何编辑）。Abaqus/CAE 显示 **Geometry Edit**（几何编辑）对话框。**提示：**您也可以使用位于 Part（部件）模块工具箱中编辑工具处的 ![](../graphics/ico_repairConvertToAnalytical.png) 工具来转换为解析。 有关工具箱中编辑工具的图表，请参阅["使用几何编辑工具集，" 第 69.1 节"](pt06ch69s01.md)。
2. 从对话框中选择 **Part**（部件）类别和 **Convert to analytical**（转换为解析）方法。
3. 从出现的警告对话框中点击 **OK**（确定）。Abaqus/CAE 尝试将部件转换为其解析表示。
4. 如果您已完成编辑部件，您应该通过在模型树中的部件上点击鼠标按钮 3 并从出现的菜单中选择 **Update validity**（更新有效性）来检查其有效性。
