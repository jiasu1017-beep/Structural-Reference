# 61.5.1 隐藏工具栏





从主菜单栏选择 ****Tools**![](../graphics/images/arrow.gif)**Customize**** 以设置 Abaqus/CAE 中工具栏的可见性。有关隐藏工具栏的其他方法，请参见["配置工具栏的可见性，" 第61.1节](pt06ch61s01.md)。

**设置工具栏可见性的步骤：**

1. 在任何模块中，从主菜单栏选择 ****Tools**![](../graphics/images/arrow.gif)**Customize****。会出现 **Customize** 对话框。

2. 在 **Toolbars** 选项卡页面上，切换打开您想要显示的任何工具栏的名称；切换关闭您想要隐藏的任何工具栏的名称。

3. 当您按所需设置了所有工具栏的可见性后，单击 **Dismiss** 关闭 **Customize** 对话框。
