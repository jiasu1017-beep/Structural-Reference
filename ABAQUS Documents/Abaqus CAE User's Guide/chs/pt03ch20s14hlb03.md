# 20.17.3 通过自动修剪边缘来修改 Sketcher 对象


使用 Sketcher 工具箱中的自动修剪工具删除不需要的绘制边缘部分。边缘在距离光标最近的两个"修剪点"之间被删除。修剪点包括:
- 与其他边缘、构造几何或参考几何的交点
- 端点
- 用于定义圆周长的点

如果端点是边缘上唯一的修剪点,Abaqus/CAE 会删除整个边缘。

您无法修剪圆角;如果您修剪定义圆角端点的边缘,Abaqus/CAE 会删除该圆角。

**自动修剪边缘的步骤:**

1. 从 Sketcher 工具箱的修改工具中,选择自动修剪工具 ![](../graphics/ico_sketchTrimAuto.png)。有关 Sketcher 工具箱中工具的图示,请参见["Sketcher 工具," 第 20.4.1 节](pt03ch20s04s01.md)。Abaqus/CAE 会在提示区域显示提示,引导您完成操作。
2. 将光标移动到草图中的边缘上。当您移动光标时,Abaqus/CAE 会高亮显示将被删除的部分。
3. 单击以删除高亮的边缘段。与使用**修剪/延伸**不同,Abaqus/CAE 不会分割相交的边缘。**注意:**如果您修剪样条曲线,Abaqus/CAE 会删除控制点以保留原始曲线的精确形状;因此,您将无法再编辑曲线的形状。
4. 完成修剪边缘后,执行以下操作之一 - 在 Abaqus/CAE 窗口任意位置单击鼠标按钮 2。 - 选择 Sketcher 工具箱中的任何工具。 - 单击提示区域中的取消按钮 ![](../graphics/afxI_cancel.png)。
