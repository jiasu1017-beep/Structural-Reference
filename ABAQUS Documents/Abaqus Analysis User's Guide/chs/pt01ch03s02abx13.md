# 3.2.13 结果(.fil)文件的ASCII转换





### 3.2.13 结果（`.fil`）文件的ASCII转换

**产品：**Abaqus/Standard  Abaqus/Explicit  

##### **参考**

- ["Abaqus执行程序：概述，" 3.1.1节](pt01ch03s01abo02.md)

### 概述

**abaqus ascfil**转换工具：
- 用于将结果（`.fil`）文件（Abaqus分析生成的）转换为ASCII格式，以便在不同操作系统之间移植；
- 允许将结果数据移动到不同系统进行后处理；并且
- 还可用于将ASCII格式的结果文件转换为二进制格式以节省磁盘空间。

### 命令摘要

| **abaqus ascfil** | [**job**](pt01ch03s02abx13.md#job1-usb-int-dasciitrans)=*job-name* |
| --- | --- |
|  | **[**[**input**](pt01ch03s02abx13.md#input2-usb-int-dasciitrans)=*input-file***]** |

### 命令行选项

**job**

此选项指定在结果文件转换期间使用的输入和输出文件名。*job-name*值用作默认输入文件名。转换后的输出文件将名为`*job-name*.fin`。

如果输入文件是二进制格式（默认），此工具将创建ASCII格式的`*job-name*.fin`文件。要在移植到不同操作系统后将结果文件转回二进制格式，请将`*job-name*.fin`文件重命名为`*job-name*.fil`，然后再次使用此工具；生成的`*job-name*.fin`文件将是二进制格式。

如果命令行中省略此选项，系统将提示您输入此值。

**input**

如果此选项指定的输入文件名与*job-name*不同。

### 示例

要将结果文件`c4.fil`从二进制转换为ASCII格式，请使用以下命令：

```
abaqus ascfil job=c4
```
转换后的文件将名为`c4.fin`。
