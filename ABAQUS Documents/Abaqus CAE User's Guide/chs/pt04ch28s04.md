# 28.4 为欧拉部件实例分配材料





在纯拉格朗日分析中，截面定义包括对单一材料的引用。当您将截面分配给拉格朗日部件中的区域或单元时，该区域或单元完全充满所引用的材料。因此，区域或单元的几何形状决定了材料的几何形状。

在纯欧拉分析中，截面定义与材料之间的关系有着根本的不同。欧拉截面定义可以引用材料列表。当您将欧拉截面分配给欧拉部件时，您是在定义分析过程中哪些材料可能存在于该部件中。然而，该部件最初是空的。要在欧拉部件的初始状态中引入材料，您必须使用材料分配预定义场。

材料分配预定义场依赖于材料体积分数的概念。在欧拉分析过程中，Abaqus根据分配给每个材料实例的体积分数来跟踪每个单元中存在的材料；体积分数表示给定材料实例占据的单元体积百分比。对于部分填充或被多种材料填充的单元，单元内材料的确切几何组成是未知的；Abaqus从相邻单元插值材料体积分数来估计单元内的材料边界。这些计算在《Abaqus分析用户指南》第14.1.1节"欧拉分析"中的"材料界面"中有更详细的讨论。

在Abaqus/CAE中，欧拉部件中的初始材料体积分数通过在载荷模块中创建材料分配预定义场来指定。预定义场将欧拉部件实例中的每个区域与每个材料实例的体积分数相关联。分配体积分数的区域可以是单元（在几何中）、网格单元或单元组。如果您选择单元或单元组，体积分数值将传播到单元或组中的每个底层欧拉单元。

材料分配预定义场中的体积分数表示为零到一之间的数字；体积分数为1表示该区域完全充满指定材料。体积分数小于1表示该区域仅部分充满指定材料；例如，体积分数为0.25意味着指定材料实例占据该区域的25%。如前所述，Abaqus根据相邻单元中的体积分数确定部分填充单元的材料边界；要对区域内的材料边界进行更精确的控制，您必须细化部件网格或重新定义区域边界。

如果未为欧拉部件实例的区域定义材料体积分数，则该区域被分配为空隙。同样，如果区域中所有材料的体积分数之和不为1，则该区域体积分数的剩余部分被分配为空隙。空隙区域没有材料属性，但其他材料可以在分析过程中流入和穿过空隙区域。

材料分配预定义场有效地定义了模型初始配置中材料的拓扑结构。欧拉部件的形状通常是任意的；材料分配预定义场向部件添加了在分析过程中将相互作用的欧拉材料。例如，考虑[图28-4](pt04ch28s04.md#usi-eulerian-materials-nls)中耦合欧拉-拉格朗日模型的横截面。欧拉部件只是一个空立方体。部件上定义的四个区域决定了地面坡度和水箱中的水量，材料被相应地分配给这些区域。

**图28–4** 欧拉-拉格朗日模型中的材料分配。

![](../graphics/usi-eulerian-materials-nls.png)

材料分配预定义场只能在欧拉分析的初始步骤中创建。在后续步骤中，材料从其初始配置变形，并根据模型中存在的力在欧拉网格上流动。

Abaqus/CAE提供了两种定义材料分配预定义场的技术：

**均匀定义**

均匀材料分配场定义通过从欧拉部件实例中选择区域并直接指定这些区域内每个材料实例的体积分数来创建。几何必须被分割成代表材料区域的独立单元。在包含孤立单元的部件实例中，您可以选择单个单元作为区域。

[图28-5](pt04ch28s04.md#usi-eulerian-volfrac-uniform-nls)展示了一个使用均匀定义创建的材料分配场。欧拉部件被分割成三个区域，每个区域中定义了材料体积分数；每个区域完全充满单个材料实例。未为空隙区域定义体积分数，因为空隙是默认的材料分配。

**图28–5** 均匀材料分配场。

![](../graphics/usi-eulerian-volfrac-uniform-nls.png)

均匀材料分配定义应仅用于相对简单的、均匀填充材料的区域。创建复杂区域所需的分割可能会对欧拉网格的质量产生负面影响，部分填充区域难以定义和解释，特别是在使用几何时。

有关在Abaqus/CAE中创建均匀材料分配场的详细信息，请参阅["定义材料分配场，" 第16.11.10节](pt03ch16s11hlb10.md)。《Abaqus基准指南》第1.7.2节"水压下弹性坝的挠度"中提供了均匀材料分配定义的示例（请参阅["弹性坝在水压作用下的挠度，" 第1.7.2节](../bmk/bmk-link.md#bmk-anl-dambreak)）。

**离散场定义**

网格几何和孤立网格的材料分配可以使用标量离散场来定义。对于部件中的每个材料实例，您可以创建一个离散场，将单个单元与该材料实例的体积分数相关联。有关创建离散场的更多信息，请参阅[第63章，"离散场工具集"](pt06ch63.md)。

当您使用离散场分配材料时，您仍然必须选择部件实例中离散场适用的区域。如果离散场包含所选区域外单元的数据，这些数据将被忽略。与离散场关联的默认值被分配给所选区域内未在离散场中明确列出的任何单元。

[图28-6](pt04ch28s04.md#usi-eulerian-volfrac-discfield-nls)显示了一个使用离散场定义的材料分配的非常简单的示例。欧拉部件由四个单元和两个材料实例组成。如[表28-1](pt04ch28s04.md#disc-field-volfrac)中所定义的，两个离散场用于指定单元内的材料组成。水和沙子之间的边界是基于相邻单元中材料体积分数的插值估计的。

**图28–6** 离散场材料分配。

![](../graphics/usi-eulerian-volfrac-discfield-nls.png)

**表28–1** 离散场中定义的体积分数。
| 离散场 | 单元3 | 默认值 |
| --- | --- | --- |
| Water_Field | 0.5 | 1 |
| Sand_Field | 0.5 | 0 |

**注意：**在任何给定单元中，所有材料体积分数的总和不应大于1。Abaqus/CAE通过从右到左读取**体积分数**表中的离散场来递增分配体积分数；一旦单元的体积分数达到1，分配给该单元的其他体积分数将被忽略。

由于离散场可以为每个单独的单元分配唯一的体积分数，因此它允许比均匀定义方法更复杂的材料边界，而无需过多的分割。Abaqus/CAE中的体积分数工具专门创建用于材料分配预定义场的离散场。通过此工具，您可以使用Abaqus/CAE中可用的部件建模技术定义复杂的欧拉材料区域。更多信息，请参见["体积分数工具，" 第28.5节](pt04ch28s05.md)。

有关在Abaqus/CAE中创建离散场材料分配的详细信息，请参阅["定义材料分配场，" 第16.11.10节](pt03ch16s11hlb10.md)。《Abaqus例题指南》第2.3.1节"铆钉成形"中提供了离散场材料分配定义的示例（包括体积分数工具的使用）（请参阅["铆钉成形，" 第2.3.1节](../exa/exa-link.md#exa-dyn-rivet)）。
