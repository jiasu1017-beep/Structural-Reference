# 19.8.4 选择作业运行模式







要找到作业编辑器，请从主菜单栏中选择****作业**![](../graphics/images/arrow.gif)**编辑**![](../graphics/images/arrow.gif)***作业名称*****。使用**提交**选项卡页面中的**运行模式**选项来选择 Abaqus 运行作业的模式。您可以选择以下选项之一：
- **后台**可在本地后台运行作业。
- **队列**可将作业提交到指定的本地或远程批处理队列；从下拉列表中选择批处理队列。（您必须在 Abaqus 环境文件中定义队列。有关更多信息，请参见["定义分析批处理队列，"《Abaqus 安装和许可指南》第 4.2 节](../sgb/sgb-link.md#sgb-chp-env-queue)。您还必须使队列对 Abaqus/CAE 可用。有关更多信息，请参见["远程提交作业，"第 19.2.7 节](pt03ch19s02s06.md)。）

**运行模式**设置类似于 Abaqus 执行过程的参数。

**选择运行模式：**

1. 找到作业编辑器。从主菜单栏中，选择****作业**![](../graphics/images/arrow.gif)**编辑**![](../graphics/images/arrow.gif)***作业名称*****。Abaqus/CAE 显示**编辑作业**对话框。
2. 点击**提交**选项卡以显示**提交**选项卡页面。
3. 从页面中间的**运行模式**选项中，选择**后台**或**队列**。
