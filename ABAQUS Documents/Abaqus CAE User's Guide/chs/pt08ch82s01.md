# 82.1 运行 Abaqus 入门示例





此插件允许您运行 [Getting Started with Abaqus: Interactive Edition](../gsa/gsa-link.md#gsa) 中描述的示例。该插件会创建一个重现每个示例的模型。该插件还会创建与模型相关联的分析作业；但是，该插件不会提交作业进行分析。如果需要，您可以转到 Job 模块并提交作业进行分析。然后可以转到 Visualization 模块查看分析结果。

该插件获取与示例关联的文件，并将这些文件放在当前目录中。您可以从插件运行以下示例：
- ["示例：创建电动环起重机的模型，" Getting Started with Abaqus: Interactive Edition 第2.3节](../gsa/gsa-link.md#gsa-abs-model)
- ["示例：连接吊环，" Getting Started with Abaqus: Interactive Edition 第4.3节](../gsa/gsa-link.md#gsa-ctm-connectlug)
- ["示例：斜板，" Getting Started with Abaqus: Interactive Edition 第5.5节](../gsa/gsa-link.md#gsa-shl-example-skewplate)
- ["示例：货物起重机，" Getting Started with Abaqus: Interactive Edition 第6.4节](../gsa/gsa-link.md#gsa-bms-exacargocrane)
- ["示例：动态载荷下的货物起重机，" Getting Started with Abaqus: Interactive Edition 第7.5节](../gsa/gsa-link.md#gsa-dyn-exacargodyn)
- ["示例：非线性斜板，" Getting Started with Abaqus: Interactive Edition 第8.4节](../gsa/gsa-link.md#gsa-nln-skewplate)
- ["示例：杆中的应力波传播，" Getting Started with Abaqus: Interactive Edition 第9.4节](../gsa/gsa-link.md#gsa-ovw-exastresswave)
- ["示例：带塑性的连接吊环，" Getting Started with Abaqus: Interactive Edition 第10.4节](../gsa/gsa-link.md#gsa-mat-connectlug)
