# 29.1.1 关于基于点的紧固件





基于点的紧固件利用定位点在Abaus/Standard和Abaus/Explicit中创建独立于网格的紧固件，如["独立于网格的紧固件"，Abaqus Analysis User's Guide第35.3.4节](../usb/usb-link.md#usb-cni-afastener)中所述。定位点可以是连接点、参考点或来自孤立网格的节点。[图29-1](pt04ch29s01s01.md#eng-fasteners-pointbracket)显示了围绕支架边缘建模点焊的基于点的紧固件。用户首先在支架边缘等间距的位置创建了连接点。连接点用于定义紧固件定位点的位置。

**图29-1** 基于点的紧固件建模点焊。

![](../graphics/eng-fasteners-pointbracket.png)

基于点的紧固件可以使用连接器或刚性（梁）多点约束连接所选面。如果要建模刚性连接，可以使用刚性连接器或刚性多点约束。

**连接器**

如果使用连接器连接面，您可以使用连接器行为定义的通用性对刚性、弹性或非弹性连接以及失效进行建模。您可以使用刚性连接器对刚性连接进行建模。但是，如果您使用的是刚性连接器，并且Abaqus检测到两个相邻的基于点的紧固件共享节点，您必须通过在连接器行为中定义一些弹性来避免过度约束模型以降低刚度。

您可以请求来自连接器的输出。

**刚性（梁）多点约束**

刚性多点约束比连接器计算成本更低，并且在两个相邻紧固件共享节点时不太可能导致过度约束。当Abaqus检测到两个相邻紧固件共享节点并使用刚性多点约束时，它使用惩罚分布耦合公式，在一定程度上放松紧固点与其耦合节点之间的约束，以避免过度约束。

您无法从多点约束请求输出。

基于点的紧固件使用分布耦合约束连接面，无论您如何对面进行网格划分。当您提交作业进行分析时，Abaqus使用您的紧固件定义通过耦合和连接器连接面。当您在Visualization模块中打开输出数据库文件时，可以显示耦合和连接器；但是，在Visualization模块外部，仅显示基于点的紧固件定位点的符号。

如果您的模型包含许多紧固件（超过一千个），基于点的紧固件比离散紧固件提供更好的性能。此外，如果您有来自CAD系统的文件定义了每个定位点的坐标，您可能需要使用基于点的紧固件。基于点的紧固件仅适用于三维模型。基于点的紧固件用于建模独立于网格的紧固件，如["独立于网格的紧固件"，Abaqus Analysis User's Guide第35.3.4节](../usb/usb-link.md#usb-cni-afastener)中所述。

您可以使用以下两种方法创建基于点的紧固件，如图29-2所示：
- 选择一个定位点，允许Abaqus/CAE沿着垂直于表面的方向将点投影到最近的面。第一个紧固点创建在法线与最近面相交的位置。
- 选择一个定位点并指定方向向量，点沿该向量投影。第一个紧固点创建在向量与最近面相交的位置。

**图29-2** 创建基于点的紧固件。

![](../graphics/eng-fasteners-point-nls.png)

当您提交作业进行分析时，Abaqus通过沿着最近面的法线将第一个紧固点投影到要连接的其他曲面上来创建第二个（以及后续）紧固点。
