# 31.3.4 创建XFEM裂缝





您可以使用Interaction模块中的**Special**菜单创建扩展有限元方法（XFEM）裂缝。更多信息，请参见["使用扩展有限元方法将不连续性建模为富集特征"，Abaqus Analysis User's Guide第10.7.1节](../usb/usb-link.md#usb-anl-aenrichment)。

**要创建XFEM裂缝：**

1. 从Interaction模块的主菜单栏中，选择****Special**![](../graphics/images/arrow.gif)**Crack**![](../graphics/images/arrow.gif)**Create****。
2. 从出现的**Create Crack**对话框中，选择**XFEM**。
3. 输入裂缝名称，然后单击**Continue**关闭对话框。
4. 从视口中的模型中选择表示裂缝域的实体。您可以从三维部件实例中选择单元或从二维部件实例中选择面。如果您有孤立网格或包含孤立网格和原生网格单元的实例，可以选择单元来表示裂缝域。您应该选择包含现有裂缝的实体以及裂缝可能扩展到的任何实体。
5. 单击鼠标按钮2表示您已完成裂缝域的选择。**Edit Crack**对话框出现。
6. 执行以下操作之一：
   - 切换**Allow crack growth**以定义随 solution 进行而沿任意路径扩展的裂缝。
   - 关闭**Allow crack growth**以定义不能扩展的静止裂缝。
7. 如果您选择允许裂缝扩展，可以执行以下操作之一来指定裂缝位置：
   - 关闭**Crack location**，表示您将允许Abaqus根据您指定的损伤萌生标准确定裂缝位置。
   - 打开**Crack location**并单击![](../graphics/ico_selectBlue.png)以通过从三维模型中选择内部面或从二维平面部件中选择边缘来指定裂缝位置。您不应选择裂缝裂缝。
8. 如果您选择防止裂缝扩展，请执行以下操作：
   1. 单击![](../graphics/ico_selectBlue.png)以通过从三维模型中选择内部面或从二维平面部件中选择边缘来指定裂缝位置。您不应选择裂缝裂缝。
   2. 要指定富集半径，请执行以下操作之一：
      - 选择**Analysis default**，以允许Abaqus确定富集半径。默认半径是富集区域中典型单元特征长度的三倍。
      - 选择**Specify**并输入值。该值应该是裂缝尖端周围的半径，包含在该半径内的单元用于计算裂缝奇异性。
9. 选择XFEM分析的类型：
   - 默认情况下，Abaqus将使用牵引-分离内聚行为方法。您可以选择或创建指定基于小滑动接触公式的裂缝面压缩和摩擦行为的接触相互作用属性。更多信息，请参见["定义面-面接触"，第15.13.7节](pt03ch15s13s01.md)。
   - 如果您创建裂缝准则接触相互作用属性，Abaqus将使用线弹性断裂力学（LEFM）方法。更多信息，请参见["定义接触相互作用属性"中的"为裂缝扩展指定裂缝准则属性"，第15.14.1节](pt03ch15s14s01.md#usi-itn-help-prop-contact-mech-fracture)。
10. 单击**OK**配置XFEM裂缝并关闭编辑器。Abaqus显示绿色十字以表示裂缝域和裂缝位置。要在Visualization模块中查看裂缝扩展，您必须在Step模块中使用**Field Output Request**编辑器并请求Abaqus在分析期间将签名距离函数PHILSM写入输出数据库。更多信息，请参见["查看XFEM裂缝"，第31.3.3节](pt04ch31s03s03.md)和["修改场输出请求"，第14.12.2节](pt03ch14s12hlb02.md)。
