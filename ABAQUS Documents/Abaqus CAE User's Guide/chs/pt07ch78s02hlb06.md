# 78.2.6 将显示组应用于多个视口和模型





您可以使用单个显示组来查看不同模型的相同子集，也可以在多个视口中查看显示组。只要显示组对每个模型有效，单个显示组就可以应用于不同的模型。例如，[图78-2](pt07ch78s02hlb06.md#dgp-vpnode)显示了在两个不同视口中显示的两个不同模型应用的单个显示组。

**图78-2** 应用于两个不同模型的显示组。

![](../graphics/dgp-vpnode.png)

**在任何视口中绘制先前保存的显示组的步骤：**

1. 点击视口边框使视口成为当前视口。
2. 从主菜单栏中选择****工具**![](../graphics/images/arrow.gif)**显示组**![](../graphics/images/arrow.gif)**绘制****。从出现的菜单中选择要绘制的显示组。Abaqus会在当前视口中绘制显示组。Abaqus仅绘制显示组中对当前模型有效的组件。

**在任何显示显示组的视口中编辑先前保存的显示组的步骤：**

1. 点击视口边框使视口成为当前视口。
2. 从主菜单栏中选择****工具**![](../graphics/images/arrow.gif)**显示组**![](../graphics/images/arrow.gif)**编辑****。从出现的菜单中选择要编辑的显示组。**编辑显示组**对话框出现。
3. 选择所需的模型组件，并应用所需的布尔运算。在编辑过程中，任何布尔运算的结果仅显示在当前视口中。
4. 点击**确定**关闭**编辑显示组**对话框。您的编辑更改会应用于引用所选显示组的所有视口。如果修改后的显示组对某个模型无效（例如，通过包含该模型中不存在的节点），Abaqus会警告部分显示组现在无效。
