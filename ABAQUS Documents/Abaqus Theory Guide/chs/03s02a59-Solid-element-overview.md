# 3.2.1 实体单元概述

### 3.2.1 实体单元概述

Abaqus包含用于二维和三维应用的实体单元库。二维单元允许对平面和轴对称问题进行建模，并包括广义平面应变的扩展（当模型存在于两个可能相对移动的平面之间时，提供厚度方向应变，该应变可能随模型平面内的位置变化但相对于厚度位置保持恒定）。三维实体单元的材料描述可以包括不同材料的不同方向的多个层，用于层合复合固体的分析。还提供了一组用于轴对称模型非对称加载的非线性单元，以及可在二维和三维中使用的线性无限单元来模拟无界域。

实体单元库包括等参单元：二维中的四边形和三维中的"砖块"（六面体）。这些等参单元通常是首选，因为它们是Abaqus中更具成本效益的单元。它们提供一阶和二阶插值，并在"实体等参四边形和六面体，" 第3.2.4节中有详细描述。出于实际原因，有时不可能在整个模型中使用等参单元；例如，一些商业网格生成器使用自动网格划分技术，依靠三角剖分来填充任意形状的区域。由于这些需求，Abaqus包含了三角形、四面体和楔形单元。对于大多数情况，建议仅使用这些单元来填充网格中难以处理的部分，特别是，在任何关键区域（如必须准确预测应变的区域）应使用形状良好的等参单元。等参单元也可以退化为更简单的形状。通常，对于特定几何形状编写的单元优于这种方法。此规则的一个例外是当要模拟奇异性时（如在断裂力学应用中），因为退化的二阶等参单元可以通过"四分之一节点"技术提供![](../graphics/stm_eqn02858.gif)奇异性（将奇异点处节点的边中点放置在距离边中点1/4的位置，而不是在边的中点处）。

实体单元提供一阶（线性）和二阶（二次）插值，用户必须决定哪种方法更适合应用。一些指导原则如下。标准一阶单元本质上是常应变单元：等参形式可以提供超出常应变的响应，但它们给出的解的高阶内容通常不准确，因此价值不大。"不兼容模式"单元（在"带不兼容模式的连续体单元，" 第3.2.5节中描述）从用户角度来看是低阶单元，但具有内部自由度，使单元能够表示几乎所有线性应变模式。这些单元可以精确表示某些重要的线性应变场：最重要的场是由弯曲引起的场。二阶单元能够表示所有可能的线性应变场。因此，对于椭圆问题——控制偏微分方程具有椭圆特征的问题，如弹性力学、热传导、声学，其中解的光滑性是有保证的——高阶单元通常每个自由度提供更高的解精度。因此，通常建议对此类情况使用可用的最高阶单元：在Abaqus中，这意味着二阶单元。这个观察顺理成章地引向了"层次化"有限元技术或"p"方法的使用——通过增加关键区域单元的插值阶数来细化模型：这种方法目前在Abaqus中尚不可用。

不兼容模式单元和二阶单元都能有效使用的一种情况是承受弯曲的相对较薄构件的应力分析：这种情况在实际应用中经常遇到。在这种情况下，厚度方向的应变变化至少必须是线性的，而常应变（一阶）单元在表示这种变化方面表现很差。完全积分的一阶等参单元在这些几何形状中也受到"剪切锁定"的影响：它们无法提供纯弯曲解，因为它们必须在数值积分点处剪切，以产生对应于弯曲的适当运动学行为。这种剪切然后锁定单元——响应过于刚硬。对于等参单元，缩减积分可以解决这些问题，但代价是允许伪奇异模式（"沙漏"）。使用二阶单元是一种更可靠的替代方案，因为二阶插值自然包含线性应变场——一个单元穿过厚度就足以相当准确地表示承受弯曲载荷的薄组件的行为。另一种选择是由不兼容模式单元形成的：如果这些单元呈矩形，则其中的线性应变场包含精确解决弯曲问题所需的模式。关于Abaqus连续体单元在弯曲问题中性能的详细讨论，请参阅"Abaqus基准指南"第2.3.5节"弯曲问题线性分析的连续体和壳单元性能"。（但是，应该记住，Abaqus提供了专门为薄几何形状编写的壳和梁单元：只有在梁或壳单元不实用的情况下才应考虑对这种情况使用实体单元。）

由于所有这些原因，二阶单元在椭圆应用中是首选。该论点可以很容易地扩展到更高阶插值（三次、四次等），但高阶形式每个单元的成本快速增长意味着——即使每个自由度的精度更高——每个计算成本的精度可能不会增加。实际经验表明——除特殊情况外——超越二阶单元几乎没有好处，因此Abaqus不提供任何更高阶的形式。

许多实际感兴趣的问题不是椭圆形的：各种形式的局部化都会出现。塑性应用就是一个例子——当解接近极限载荷时，大多数塑性模型趋于双曲线行为。这允许解中出现不连续性——经典理想塑性理论的滑移线解是双曲线方程中速度不连续性特征线的绘图。如果有限元解要表现出准确性，解的梯度场中的这些不连续性应该相当好地被模拟。使用固定网格且不使用在其公式中允许不连续性的特殊单元，这意味着最低阶单元——一阶单元——可能最成功，因为对于给定的节点数，它们在更多位置提供解的梯度某个分量可能是不连续的（单元边缘）。这个论点几乎不严格，但尽管如此，一阶单元在此类情况下往往是首选。不兼容模式单元特别擅长表示不连续性。它们还能够表示应变局部化，如在剪切带中发生的那样。但是，应该认识到，定义更好的剪切局部化会增加应变幅度，从而倾向于增加分析所需的增量数和迭代次数。

Abaqus中除无限单元外的所有实体单元都编写为包含有限应变效应。当这些单元与超弹性（弹性体）材料定义一起使用时，本构行为直接从变形梯度矩阵![](../graphics/stm_eqn00319.gif)计算。当单元用于任何其他材料定义的几何非线性分析时（在有限应变情况下，这意味着材料具有一些非弹性行为，因为Abaqus中除超弹性模型外的所有弹性定义都假定弹性应变很小），应变计算为变形率的积分，

同时考虑相对于坐标系的材料旋转效应。在所有情况下，实体单元将应力报告为"真实"（柯西）应力。除非为单元指定了局部方向，否则应力和应变分量作为相对于全局空间方向的物理分量给出。当为实体单元定义了局部方向时，应力和应变分量在用户定义的局部坐标系中给出：该系统随每个材料点处计算的平均材料旋转而旋转。
