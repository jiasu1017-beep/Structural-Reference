**产品：** Abaqus/Standard  Abaqus/Explicit  Abaqus/CFD  

##### **参考**

- ["输入语法规则，" 第 1.2.1 节"](pt01ch01s02aus01.md)

### 概述

Abaqus 中的分析通过*输入文件*定义，该文件：
- 包含*关键字行*和*数据行*；并且
- 分为*模型*数据和*历史*数据。

### 输入文件

Abaqus 输入文件是 ASCII 数据文件。它可以通过使用文本编辑器或使用图形预处理程序（如 Abaqus/CAE）创建。输入文件由一系列包含 Abaqus 选项（关键字行）和数据（数据行）的行组成。关键字行和数据行的输入语法在 ["输入语法规则，" 第 1.2.1 节"](pt01ch01s02aus01.md) 中描述。

大多数输入文件具有相同的基本结构。指定以下部分来定义有限元模型：

1. 输入文件通常以 [*HEADING](../key/key-link.md#usb-kws-mheading) 选项开头，用于定义分析的标题。可以使用任意数量的数据行来给出标题；它们将出现在输出文件的开头（["输出，" 第 4.1.1 节"](pt02ch04s01aus38.md)）。第一个标题行将出现在输出的每一页顶部。虽然包含标题有助于检查您的输入文件，但 [*HEADING](../key/key-link.md#usb-kws-mheading) 选项不是必需的。
2. 标题之后，输入文件通常包含*模型*数据部分，用于定义节点、单元、材料、初始条件等。模型数据部分在下面解释。
3. 如果模型被组织成部件实例的装配，则模型数据被进一步分类，必须落在适当的级别：部件、装配、实例或模型。以部件实例装配形式定义的模型在 ["定义装配，" 第 2.10.1 节"](pt01ch02s10aus28.md) 中讨论。
4. 最后，输入文件包含*历史*数据，用于定义分析类型、载荷、输出请求等。步骤定义将输入文件中的模型数据与历史数据分开：第一个步骤定义之前出现的所有内容都是模型数据，第一个步骤定义之内及其后出现的所有内容都是历史数据。历史数据部分在下面解释。

在执行适当的分析产品 Abaqus/Standard、Abaqus/Explicit 或 Abaqus/CFD 之前，输入文件由"分析输入文件处理器"处理。分析输入文件处理器的功能是解释 Abaqus 选项、执行必要的一致性检查并为分析产品准备数据。

大多数计算力学建模选项（单元类型、载荷类型等）在 Abaqus/Standard 和 Abaqus/Explicit 中都可用，尽管某些选项仅在一个分析产品或另一个分析产品中可用。输入文件中使用的所有步骤过程类型必须来自同一分析产品；但是，可以将解从 Abaqus/Standard 导入 Abaqus/Explicit，反之亦然（请参阅 ["导入和传输结果，" 第 9.2 节"](pt04ch09s02.md)），这允许在分析的不同阶段使用每个分析产品（例如，在 Abaqus/Explicit 中进行动态分析之前，先在 Abaqus/Standard 中进行静力预加载）。

### 模型数据

模型数据定义节点、单元、材料、初始条件等。

#### 必需的模型数据

输入文件中必须包含以下模型数据来定义有限元模型：

**几何**：模型的几何由单元及其节点描述。定义节点和单元的规则和方法在 ["节点定义，" 第 2.1.1 节"](pt01ch02s01aus05.md)；["单元定义，" 第 2.2.1 节"](pt01ch02s02aus11.md)；和 ["定义装配，" 第 2.10.1 节"](pt01ch02s10aus28.md) 中描述。结构单元（如梁）的截面必须定义。特殊特征可以用特殊单元定义，如弹簧、阻尼器、点质量等。建模可用的单元类型在 [第六部分，"单元"](pt06.md) 中描述，以及如何定义单元的解释。数据检查运行后，您可以在 Abaqus/CAE 的可视化模块中查看初始网格或调整初始过盈后的配置（请参阅 ["Abaqus/Standard、Abaqus/Explicit 和 Abaqus/CFD 执行，" 第 3.2.2 节"](pt01ch03s02abx02.md)）。

**材料定义**：材料类型必须与几何的大部分相关联。材料库在 [第五部分，"材料"](pt05.md) 中描述。特殊单元（如弹簧或阻尼器）没有相关联的材料，但必须定义它们的属性。

#### 可选的模型数据

可以根据需要包含以下模型数据：

**部件和装配**：模型的几何可以通过将其组织成部件来定义，部件在装配中相对于彼此定位（["定义装配，" 第 2.10.1 节"](pt01ch02s10aus28.md)）。

**初始条件**：可以指定非零初始条件，如初始应力、温度或速度（["初始条件，" 第 34.2 节"](pt07ch34s02.md)）。

**边界条件**：可以将零值边界条件（包括对称条件）施加到单个求解变量，如位移或旋转（["Abaqus/Standard 和 Abaqus/Explicit 中的边界条件，" 第 34.3.1 节"](pt07ch34s03aus118.md)）。

**运动约束**：可以定义涉及模型中多个基本求解变量的方程（["线性约束方程，" 第 35.2.1 节"](pt08ch35s02aus129.md)）或多点约束（["通用多点约束，" 第 35.2.2 节"](pt08ch35s02aus130.md)）。

**相互作用**：可以定义部件之间的接触和其他相互作用（["接触相互作用分析：概述，" 第 36.1.1 节"](pt09ch36s01abo33.md)）。

**幅值定义**：可以为以后使用定义幅值曲线，以指定随时间变化的载荷或边界条件（["幅值曲线，" 第 34.1.2 节"](pt07ch34s01aus115.md)）。

**输出控制**：您可以控制模型定义输出到数据文件（["输出，" 第 4.1.1 节"](pt02ch04s01aus38.md)）。

**环境属性**：可能需要定义环境属性，如围绕模型的流体属性。

**分析继续**：可以写入重新启动数据或使用先前分析的结果，并用新的模型或历史数据继续分析（["重新开始分析，" 第 9.1.1 节"](pt04ch09s01aus53.md)）、新网格（["子模型：概述，" 第 10.2.1 节"](pt04ch10s02aus60.md)；["网格到网格解映射，" 第 12.4.1 节"](pt04ch12s04aus86.md)；和 ["对称模型生成，" 第 10.4.1 节"](pt04ch10s04aus63.md)），或使用相同或不同的 Abaqus 程序（["在 Abaqus 分析之间传输结果：概述，" 第 9.2.1 节"](pt04ch09s02aus54.md)）。

### 历史数据

分析的目的是预测模型对某种外部载荷或某种非平衡初始条件的响应。Abaqus 分析基于*步骤*的概念，这些步骤在输入文件的历史数据部分描述。（有关步骤的更多信息，请参阅 ["定义分析，" 第 6.1.2 节"](pt03ch06s01abo05.md)。）历史输入数据在步骤中根据需要组合，以定义分析的历史。

可以在分析中定义多个步骤。引入步骤只是为了更改输出请求或更改载荷、边界条件、分析过程等。分析中的步骤数量没有限制。

Abaqus 中有两种步骤：一般响应分析步骤，可以是线性的或非线性的；以及 Abaqus/Standard 中的线性扰动步骤（请参阅 ["一般和线性扰动过程，" 第 6.1.3 节"](pt03ch06s01aus44.md)）。一般分析步骤对系统响应历史有贡献；线性扰动步骤允许在响应历史期间的任何阶段研究系统的线性化响应。

一般步骤结束时的状态为下一步提供初始条件，使得轻松模拟模型的连续加载，例如静态预载后的动态响应，或模拟制造过程后的产品使用期间加载。

以下描述的可选历史数据（规定载荷；边界条件；输出控制；辅助控制；以及在 Abaqus/Explicit 中的接触条件）从一个一般分析步骤继续到下一个一般分析步骤，除非被修改。例如，在 Abaqus/Standard 的一般分析步骤中规定的求解控制（请参阅 ["收敛和时间积分准则：概述，" 第 7.2.1 节"](pt03ch07s02abo11.md)）将在所有后续一般分析步骤中保持有效，直到它们被修改或重置。对于线性扰动步骤，仅当没有中间一般分析步骤且输出控制未被重新定义时，输出控制才从一个线性扰动步骤继续到下一个线性扰动步骤（请参阅 ["输出，" 第 4.1.1 节"](pt02ch04s01aus38.md)）。类似地，在 Abaqus/CFD 分析中指定的条件从一个步骤继续到下一个步骤，除非被修改。

| **输入文件用法：** | 使用以下选项开始步骤定义： |
| --- | --- |
|  | ``` [*STEP](../key/key-link.md#usb-kws-hstep) ``` 使用以下选项结束步骤定义： ``` [*END STEP](../key/key-link.md#usb-kws-hendstep) ``` |

#### 必需的历史数据

输入文件中必须包含以下历史数据来定义分析过程：

**响应类型**：定义分析过程类型的选项必须出现在步骤定义的开头之后。Abaqus 可以执行多种类型的分析——线性的或非线性的、静力的或动力的等。（请参阅 ["定义分析，" 第 6.1.2 节"](pt03ch06s01abo05.md)。）分析类型可以从步骤到步骤更改。例如，在 Abaqus/Standard 中可以首先分析静态预载，然后可以将响应类型更改为瞬态动力。这样，可以基于静力解结束时的条件执行线性或非线性动力学分析。

#### 可选的历史数据

可以根据需要包含以下历史数据：

**载荷**：通常定义某种形式的外部载荷。例如，可以施加集中载荷或分布载荷（["施加载荷：概述，" 第 34.4.1 节"](pt07ch34s04aus120.md)）、可以规定导致热膨胀的温度变化（["热膨胀，" 第 26.1.2 节"](pt05ch26s01abm52.md)），或者可以使用接触条件来施加载荷（["接触相互作用分析：概述，" 第 36.1.1 节"](pt09ch36s01abo33.md)）。载荷可以规定为时间的函数（["幅值曲线，" 第 34.1.2 节"](pt07ch34s01aus115.md)）。此功能可用于规定载荷，如地震事件期间的地运动、已知加速度或发动机瞬态期间的温度和压力历史。如果未定义幅值曲线，Abaqus 假设载荷随步骤线性变化，或者根据所选响应类型在步骤开始时立即施加载荷（请参阅 ["定义分析，" 第 6.1.2 节"](pt03ch06s01abo05.md)）。

**边界条件**：可以在分析期间添加、修改或删除边界条件（["边界条件，" 第 34.3 节"](pt07ch34s03.md)）。

**输出控制**：应力、应变、反作用力、温度和能量等量可以作为输出。输出选项在 ["输出到数据和结果文件，" 第 4.1.2 节"](pt02ch04s01aus39.md) 和 ["输出到输出数据库，" 第 4.1.3 节"](pt02ch04s01aus40.md) 中描述；所有输出变量在 ["输出变量，" 第 4.2 节"](pt02ch04s02.md) 中列出。可用的输出文件在 ["输出，" 第 4.1.1 节"](pt02ch04s01aus38.md) 中描述。

**接触**：在 Abaqus/Explicit 分析期间，接触表面和接触相互作用可以作为步骤相关的历史数据添加、修改或删除（请参阅 ["接触相互作用分析：概述，" 第 36.1.1 节"](pt09ch36s01abo33.md)）。

**辅助控制**：您可以覆盖内置在 Abaqus 中的求解控制。在某些过程中，这些值在过程定义中给出。更一般地说，在 Abaqus/Standard 中，它们通过定义求解控制来给出（["常用控制参数，" 第 7.2.2 节"](pt03ch07s02aus50.md)）。也可以定义接触问题的求解控制（["在 Abaqus/Standard 中调整接触控制，" 第 36.3.6 节"](pt09ch36s03aus150.md)；["在 Abaqus/Explicit 中使用接触对进行接触建模的常见困难，" 第 39.2.2 节"](pt09ch39s02aus186.md)；或 ["在 Abaqus/Explicit 中通用接触的接触控制，" 第 36.4.5 节"](pt09ch36s04aus159.md)）。

**单元和表面移除/重新激活**：在 Abaqus/Standard 中，可以从步骤到步骤移除或重新激活模型的某些部分。请参阅 ["单元和接触对移除和重新激活，" 第 11.2.1 节"](pt04ch11s02aus66.md)。

**协同仿真**：Abaqus 模型中的步骤必须定义，使得协同仿真完全位于单个 Abaqus 步骤内。此外，Abaqus 作业中只能有一个协同仿真。

### 从外部文件包含模型或历史数据

您可以指定一个包含部分 Abaqus 输入文件的外部文件。此文件可以包含模型和历史定义数据、注释行以及对其他外部文件的引用。当遇到对外部文件的引用时，Abaqus 会立即处理指定文件中的数据。到达文件末尾时，Abaqus 将返回处理原始文件。

最多可以使用五层嵌套的外部文件引用。UNIX 环境变量可用于指定文件名。

| **输入文件用法：** | ``` [*INCLUDE](../key/key-link.md#usb-kws-minclude), INPUT=*file_name* ``` |
| --- | --- |

#### 包含加密数据文件

您可以通过引用在 Abaqus 输入文件或其他数据文件中包含加密文件。当您引用加密文件时，还必须提供文件的密码。如果密码正确，Abaqus 会处理指定文件中的数据，就像处理未加密的外部文件一样。加密输入文件中的材料和连接器行为定义不会写入输出数据库。此外，如果加密文件用作模型任何部分的输入，则所有材料和连接器行为定义都不会输出到数据文件。有关加密实用程序的详细信息，请参阅 ["加密和解密 Abaqus 输入数据，" 第 3.2.38 节"](pt01ch03s02abx38.md)。

某些加密文件仅允许具有特定 Abaqus 功能许可证的用户（如 Abaqus/Explicit）或特定站点的用户包含。如果您尝试包含您没有适当权限的加密文件，Abaqus 会发出错误消息。

您不能包含包含参数化输入的加密输入文件。

| **输入文件用法：** | ``` [*INCLUDE](../key/key-link.md#usb-kws-minclude), INPUT=*file_name*, PASSWORD=*password* ``` |
