# 8.2.5 自定义发送到 Windows 打印机的图像





当你直接将画布上的视口打印到 Windows 打印机时，你可以使用**页面设置**对话框自定义结果打印图像。你可以配置以下内容：

**方向**

你可以选择**纵向**或**横向**方向。纵向和横向方向如下图所示：

![](../graphics/pri-orient-nls.png)

**单位**

你可以选择**英寸**或**毫米**单位。你的选择决定**图像大小**和**边距**设置使用的单位，这些设置也出现在此对话框中。

**质量**

你可以选择三个质量级别之一。每个设置限制发送到打印机的最大文件大小：
- **粗糙**限制文件为 2 兆字节，
- **中等**限制文件为 10 兆字节，
- **精细**限制文件为 50 兆字节。

质量级别与图像大小和打印机设置协同工作，以确定打印图像的分辨率。

**注意：**点击**打印**对话框中的**打印机属性**按钮访问打印机设置。

**日期和标志**

默认情况下，Abaqus/CAE 在直接发送到 Windows 打印机的图像顶部包含日期和时间以及 Abaqus/CAE 标志。你可以选择从输出中移除日期和时间或标志。

**图像大小**

你可以选择三种方法来确定打印图像的大小：
- **适应页面**使所选图像适应当前页面大小和边距。
- **使用屏幕大小**按屏幕上显示的方式打印图像。不适合当前页面大小和边距的图像部分将在打印输出中被裁剪。
- **使用下方设置**允许你输入宽度或高度；Abaqus/CAE 调整你不编辑的维度以保持当前纵横比。

**边距**

你可以提供**顶部**、**底部**、**左侧**和**右侧**边距。Abaqus/CAE 将最大图像大小计算为页面大小减去边距。你可以指定零宽度边距；但是，打印机无法打印到纸张边缘，通常至少需要 0.25 英寸（6 毫米）的边距。页面大小在打印机设置中设置。

**注意：**点击**打印**对话框中的**打印机属性**按钮访问打印机设置。

Abaqus/CAE 无论纸张方向如何都保持你指定的边距。例如，假设你选择了**纵向**图像并输入了**顶部**边距的值。如果你现在选择**横向**图像，Abaqus/CAE 使用你为**顶部**边距输入的值来计算**左侧**边距。类似地，你为**右侧**边距输入的值变为**顶部**边距。

你也可以使用**打印**对话框设置要打印的份数并访问所选打印机的特定设置。点击**打印机属性**按钮打开***打印机名称*文档属性**对话框。可用的打印机设置由安装的打印机驱动程序和配置决定，不是由 Abaqus/CAE 决定。

更多信息，请参阅["Windows 和 PostScript 图像布局，" 第 8.1.2 节](pt01ch08s01s02.md)和["硬拷贝图像质量，" 第 8.1.5 节](pt01ch08s01s05.md)。

**要自定义发送到 Windows 打印机的图像：**

1. 从主菜单栏中，选择****File**![](../graphics/images/arrow.gif)**Print****。**提示：**你也可以单击**文件**工具栏中的![](../graphics/ico_filePrint.png)。**打印**对话框出现。
2. 从**目标**单选按钮中，选择**打印机**。
3. 从**打印机**列表中，选择你要使用的打印机名称。
4. 点击**份数**文本字段右侧的箭头以增加或减少要打印的份数，或直接在文本字段中输入份数。你可以打印 1 到 100 份。
5. 从**打印**对话框的右下角，点击**页面设置**。**页面设置**对话框出现。
6. 从**方向**字段中，选择纸张方向。
7. 从**单位**字段中，选择用于图像大小和边距的单位。
8. 从**质量**字段中，点击箭头并选择**粗糙**、**中等**或**精细**图像质量。
9. 如有需要，关闭**打印日期**从输出中移除日期和时间。
10. 如有需要，关闭**打印 SIMULIA 标志**从输出中移除标志。
11. 从**图像大小**字段中，为你的打印图像选择大小。
12. 从**边距**字段中，输入**顶部**、**底部**、**左侧**和**右侧**边距（使用你先前选择的单位）。
13. 点击**确定**保存你的自定义设置并关闭**页面设置**对话框。
14. 如有需要，点击**打印**对话框中的**打印机属性**按钮打开**文档属性**对话框以访问特定于打印机的选项。**注意：****文档属性**对话框是 Windows 对话框，不是 Abaqus/CAE 的一部分。如果你对此对话框中的信息有疑问，应该咨询你的系统管理员或查阅打印机的文档或打印机驱动程序。
15. 当你完成**打印**对话框时，单击**确定**生成所需的输出。Abaqus/CAE 生成输出并关闭**打印**对话框。你的**打印**对话框设置会在会话期间保存。
