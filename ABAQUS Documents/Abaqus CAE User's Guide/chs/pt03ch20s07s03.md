# 20.7.3 使用参数控制草图几何







参数是已赋予变量名的尺寸或数值常数，用于在几何片段之间创建参数方程。尺寸始终具有数值，但参数可能具有数值或数学表达式的结果。例如，您可以将圆的半径尺寸参数化，使其始终是草图中另一个尺寸的一半大小，但您必须首先将两个尺寸与参数关联以创建表达式。您可以使用草图选项来控制 Abaqus/CAE 是在草图中显示参数的名称还是值（有关更多信息，请参见["在 Sketcher 中自定义尺寸的格式和使用，"第 20.9.9 节](pt03ch20s09hlb09.md)）。

参数必须在单个草图中定义——您无法在一个草图中定义参数并在另一个草图中使用它们创建表达式。定义参数会将其添加到草图的**参数管理器**中。管理器中参数的顺序表示它们可以在表达式中使用的顺序。[图 20-7](pt03ch20s07s03.md#ske-parametermgr)显示了两个参数；*length* 参数用于定义 *width* 参数。如果您想用宽度来定义长度，则宽度参数需要首先出现在管理器中。您可以使用对话框右侧的按钮通过从视口选择尺寸来创建参数，或在管理器中插入、移动或删除参数。有关详细说明，请参见["添加和编辑参数，"第 20.12.5 节](pt03ch20s12hlb05.md)。
