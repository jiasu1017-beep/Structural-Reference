# 2.2 设置 Abaqus 模型以创建不包含应力或应变的模态中性文件





如果您希望 Abaqus Interface for MSC.ADAMS 创建不包含应力或应变的模态中性文件，可以使用以下模板准备 Abaqus 分析的输入文件：

```
*HEADING
...
********************
*STEP
*FREQUENCY, EIGENSOLVER=...
...
*BOUNDARY
...
*ELEMENT MATRIX OUTPUT, MASS=YES, ELSET=...
*NODE FILE
U,
*END STEP
********************
*STEP, UNSYMM=NO
*SUBSTRUCTURE GENERATE, TYPE=Z...,
 RECOVERY MATRIX=YES, MASS MATRIX=YES
*RETAINED NODAL DOFS
...
*SELECT EIGENMODES
...
*SUBSTRUCTURE LOAD CASE, NAME=...
*CLOAD
...
*SUBSTRUCTURE MATRIX OUTPUT,
 RECOVERY MATRIX=YES, MASS=YES,
 STIFFNESS=YES, SLOAD=YES
*END STEP
********************
```

输入文件的历史部分必须包含一个 [*FREQUENCY](../key/key-link.md#usb-kws-hfrequency) 步骤来计算固定界面正则模态，随后是 [*SUBSTRUCTURE GENERATE](../key/key-link.md#usb-kws-ssubgenerate) 步骤。[*FREQUENCY](../key/key-link.md#usb-kws-hfrequency) 步骤之前可以有任意数量的步骤来施加所需的预载荷。

关于 [*FREQUENCY](../key/key-link.md#usb-kws-hfrequency) 步骤注意以下几点：
- [*FREQUENCY](../key/key-link.md#usb-kws-hfrequency) 步骤必须对将在 [*SUBSTRUCTURE GENERATE](../key/key-link.md#usb-kws-ssubgenerate) 步骤中保留的每个自由度施加零边界条件。其他自由度可以根据需要施加约束。
- 此步骤必须将单元质量矩阵和特征向量写入结果文件。对于每个单元，节点处的质量通过汇总该节点第一个自由度对应的单元质量矩阵行中的所有项来计算。对于 Abaqus 单元库中的所有单元，此计算是精确的，但对于用户单元可能不正确。特别是，对于表示子结构的用户单元，此计算不正确。

关于 [*SUBSTRUCTURE GENERATE](../key/key-link.md#usb-kws-ssubgenerate) 步骤注意以下几点：
- [*STEP](../key/key-link.md#usb-kws-hstep) 选项上的 UNSYMM=NO 参数是可选的，但建议使用。某些预加载历史（例如，具有高摩擦系数的接触）可能会产生非对称刚度矩阵。在此类预加载历史之后创建的子结构矩阵在所有情况下都是对称的。但是，默认情况下，如果刚度矩阵在此前的分析中是非对称的，Abaqus 将创建并将完整的子结构矩阵写入结果文件。使用 UNSYMM=NO 可节省计算时间和存储空间，且不影响精度。
- [*SUBSTRUCTURE GENERATE](../key/key-link.md#usb-kws-ssubgenerate) 选项必须包含参数 RECOVERY MATRIX=YES 和 MASS MATRIX=YES。
- [*RETAINED NODAL DOFS](../key/key-link.md#usb-kws-sretainnodaldofs) 列表必须等于或为前一个 [*FREQUENCY](../key/key-link.md#usb-kws-hfrequency) 步骤中设为零的自由度列表的子集。
- [*SELECT EIGENMODES](../key/key-link.md#usb-kws-hselecteigenmodes) 列表必须等于或为 [*FREQUENCY](../key/key-link.md#usb-kws-hfrequency) 步骤中计算的固有模态列表的子集。如果列表是子集，则不得将未使用的固有模态写入结果文件。
- [*SUBSTRUCTURE MATRIX OUTPUT](../key/key-link.md#usb-kws-ssubmatrixoutput) 选项必须将恢复矩阵、刚度矩阵和质量矩阵写入结果文件。
- 如果在 [*SUBSTRUCTURE MATRIX OUTPUT](../key/key-link.md#usb-kws-ssubmatrixoutput) 选项上使用 SLOAD=YES 参数，则对应于作用在子结构上的所有内力和外力的模态载荷分量将被转换为模态中性文件中的模态预载荷（块 27）。
- [*SUBSTRUCTURE LOAD CASE](../key/key-link.md#usb-kws-ssubloadcase) 选项是可选的。如果存在此选项，[*CLOAD](../key/key-link.md#usb-kws-hcload) 数据应复制先前通用步骤中定义的载荷，以表示模型其他部分对子结构的影响。如前一段所述，如果在 [*SUBSTRUCTURE MATRIX OUTPUT](../key/key-link.md#usb-kws-ssubmatrixoutput) 选项上使用 SLOAD=YES，这些外力（连同任何内力）将被写入模态中性文件的预载荷数据块。用于载荷情况的 NAME 值无关紧要。

Abaqus 输入文件的历史部分可以包含在所需的 [*FREQUENCY](../key/key-link.md#usb-kws-hfrequency) 和 [*SUBSTRUCTURE GENERATE](../key/key-link.md#usb-kws-ssubgenerate) 步骤之前的通用步骤。关于这些可选通用步骤注意以下几点：
- 在这些步骤中写入结果文件的位移将用于更新写入模态中性文件的节点坐标。较晚步骤的位移将更新较早步骤的位移。因此，如果为任何步骤写了部分节点的位移，则最后一步应写入子结构所有节点的位移；否则，某些节点将以原始坐标转换，其他节点将以变形坐标转换。
- 如果在 [*SUBSTRUCTURE MATRIX OUTPUT](../key/key-link.md#usb-kws-ssubmatrixoutput) 选项上使用 SLOAD=YES，则这些步骤中的载荷可以转换为模态中性文件中的模态预载荷（块 27）。内力和外力的处理方式不同：
  - 内力包括分布载荷（如自重）和边界条件。这些载荷被认为是子结构的固有部分。所有 [*DLOAD](../key/key-link.md#usb-kws-hdload) 数据和由边界条件引起的反作用力被视为内力。内力的合成结果可能不为零。例如，如果将重力载荷施加到限制刚体运动的结构上，则等于物体重量的净力将通过保留的自由度作用在 MSC.ADAMS 模型的其余部分上。
  - 外力代表模型其他部分对子结构的影响。所有 [*CLOAD](../key/key-link.md#usb-kws-hcload) 选项被视为外力。要将这些载荷转换到 MSC.ADAMS，通用步骤中的 [*CLOAD](../key/key-link.md#usb-kws-hcload) 必须作为 [*SUBSTRUCTURE GENERATE](../key/key-link.md#usb-kws-ssubgenerate) 步骤中的 [*SUBSTRUCTURE LOAD CASE](../key/key-link.md#usb-kws-ssubloadcase) 数据进行复制。
