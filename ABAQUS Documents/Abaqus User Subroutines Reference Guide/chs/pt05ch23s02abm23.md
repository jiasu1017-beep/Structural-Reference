### 概述

Johnson-Cook塑性模型：
- 是一种特定类型的Mises塑性模型，包含硬化定律和率相关性的分析形式；
- 适用于包括大多数金属在内的高应变率变形；
- 通常用于绝热瞬态动力学模拟；
- 可与Abaqus/Explicit中的Johnson-Cook动态失效模型结合使用；
- 可与拉伸失效模型结合使用，在Abaqus/Explicit中模拟拉伸层裂或压力截止；
- 可与逐步损伤和失效模型（[第24章"逐步损伤和失效"](pt05ch24.md)"）结合使用，以指定不同的损伤起始准则和损伤演化定律，允许材料刚度的逐步降解以及从网格中删除单元；和
- 必须与线性弹性材料模型（["线性弹性行为，" 第22.2.1节"](pt05ch22s02abm02.md)）或状态方程材料模型（["状态方程，" 第25.2.1节"](pt05ch25s02abm50.md)）结合使用。

### 屈服面和流动法则

Johnson-Cook塑性模型使用具有相关流动的Mises屈服面。

### Johnson-Cook硬化

Johnson-Cook硬化是一种特殊类型的各向同性硬化，其中静态屈服应力σ^0^假定为以下形式：

σ^0^ = (A + B ε¯^p^n) (1 + C ln ε¯˙^p^/ε¯˙^p^0) (1 - θ^m^)

其中ε¯^p^是等效塑性应变，A、B、n和m是在转变温度θ~t~或以下测量的材料参数。θ^m^是无量纲温度，定义为：

θ^m^ = (θ - θ~t~)/(θ~m~ - θ~t~)

其中θ是当前温度，θ~m~是熔化温度，θ~t~是转变温度，定义为屈服应力对温度没有依赖性的温度或温度以下。材料参数必须在转变温度或以下测量。

当θ^m^ = 1时，材料将熔化并将像流体一样行为；由于σ^0^ = 0，将没有剪切阻力。硬化记忆将通过将等效塑性应变设置为零来消除。如果为模型指定了背应力，这些也将设置为零。

如果您在材料定义中包含退火行为，且退火温度定义为低于为金属塑性模型指定的熔化温度，则硬化记忆将在退火温度消除，熔化温度严格用于定义硬化函数。否则，硬化记忆将在熔化温度自动消除。如果材料点的温度在后续时间点降至退火温度以下，材料点可以再次加工硬化。更多详情，请参见["退火或熔化，" 第23.2.5节"](pt05ch23s02abm21.md)。

您在金属塑性材料定义中提供A、B、n、m、θ~m~和θ~t~的值。

### Johnson-Cook应变率相关性

Johnson-Cook应变率相关性假定：

σ¯^y^ = σ¯^0^ R(ε¯˙^p^)

且：

R(ε¯˙^p^) = 1 + C ln(ε¯˙^p^/ε¯˙^p^0)

其中σ¯^y^是非零应变率下的屈服应力；ε¯˙^p^是等效塑性应变率；D和C是在转变温度θ~t~或以下测量的材料参数；σ¯^0^是静态屈服应力；且R = σ¯^y^/σ¯^0^是非零应变率下屈服应力与静态屈服应力之比。

因此，屈服应力表示为：

σ¯^y^ = (A + B ε¯^p^n) (1 + C ln ε¯˙^p^/ε¯˙^p^0) (1 - θ^m^)

当您定义Johnson-Cook率相关性时，提供D和C的值。

使用Johnson-Cook硬化不一定需要使用Johnson-Cook应变率相关性。

### Johnson-Cook动态失效

Abaqus/Explicit为Johnson-Cook塑性模型提供了专门的动态失效模型，仅适用于金属的高应变率变形。该模型被称为"Johnson-Cook动态失效模型"。Abaqus/Explicit还将Johnson-Cook失效模型作为损伤起始准则系列的一部分提供更一般的实现，这是建模材料逐步损伤和失效的推荐技术（见["延性金属的损伤和失效：概述，" 第24.2.1节"](pt05ch24s02abm41.md)）。Johnson-Cook动态失效模型基于单元积分点处的等效塑性应变值；当损伤参数超过1时，假定失效发生。损伤参数D通过以下方式定义：

D = Σ (Δε¯^p^/ε¯^p^f~)

其中Δε¯^p^是等效塑性应变增量，ε¯^p^f~是失效时的应变，且求和遍及分析中的所有增量。失效时的应变ε¯^p^f~假定依赖于无量纲塑性应变率ε¯˙^p^/ε¯˙^p^0~；无量纲压力-偏量应力比p/q~（其中p是压力应力，q是Mises应力）；以及Johnson-Cook硬化模型中先前定义的无量纲温度θ^m^。假设依赖性是可分离的，形式为：

ε¯^p^f~ = [d~1~ + d~2~ exp(d~3~ p/q~)] [1 + d~4~ ln ε¯˙^p^/ε¯˙^p^0~) (1 + d~5~ θ^m^)

其中d~1~ - d~5~是在转变温度θ~t~或以下测量的失效参数，ε¯˙^p^0~是参考应变率。当您定义Johnson-Cook动态失效模型时，提供d~1~ - d~5~的值。ε¯^p^f~的这个表达式与[Johnson和Cook（1985）](pt05ch23s02abm23.md#usb-mat-ref-jc)发表的原始公式在参数d~2~的符号上不同。这种差异是由于大多数材料随压力-偏量应力比增加而经历ε¯^p^f~增加的事实；因此，上述表达式中的d~2~通常为正值。

当满足此失效准则时，偏量应力分量设置为零，并在分析的剩余部分保持为零。根据您的选择，压力应力也可以在计算的剩余部分设置为零（如果是这种情况，您必须指定单元删除，单元将被删除），或者可能需要保持在压缩状态（如果是这种情况，您必须选择不使用单元删除）。默认情况下，满足失效准则的单元被删除。

Johnson-Cook动态失效模型适用于金属的高应变率变形；因此，它最适用于真正的动态情况。对于需要单元删除的准静态问题，建议使用逐步损伤和失效模型（[第24章"逐步损伤和失效"](pt05ch24.md)"）或多孔金属塑性模型（["多孔金属塑性，" 第23.2.9节"](pt05ch23s02abm25.md)）。

使用Johnson-Cook动态失效模型需要使用Johnson-Cook硬化，但不一定需要使用Johnson-Cook应变率相关性。但是，只有在定义了Johnson-Cook应变率相关性时，才会包含Johnson-Cook动态失效准则中的率相关项。["延性金属的损伤起始，" 第24.2.2节"](pt05ch24s02abm42.md)中描述的Johnson-Cook损伤起始准则没有这些限制。

### 逐步损伤和失效

Johnson-Cook塑性模型可与["延性金属的损伤和失效：概述，" 第24.2.1节"](pt05ch24s02abm41.md)中讨论的逐步损伤和失效模型结合使用。该功能允许指定一个或多个损伤起始准则，包括延性、剪切、成形极限图（FLD）、成形极限应力图（FLSD）、Mschenborn-Sonne成形极限图（MSFLD），以及在Abaqus/Explicit中的Marciniak-Kuczynski（M-K）准则。损伤起始后，材料刚度根据指定的损伤演化响应逐步降解。该模型提供两种失效选择，包括由于结构撕裂或扯裂而从网格中删除单元。逐步损伤模型允许材料刚度的平滑降解，使它们适用于准静态和动态情况。这与上面讨论的动态失效模型相比是一个很大的优势。

### 拉伸失效

在Abaqus/Explicit中，拉伸失效模型可与Johnson-Cook塑性模型结合使用来定义材料的拉伸失效。拉伸失效模型使用静水压力应力作为失效度量来模拟动态层裂或压力截止，并提供多种失效选择，包括单元删除。与Johnson-Cook动态失效模型类似，Abaqus/Explicit拉伸失效模型适用于金属的高应变率变形，最适用于真正的动态问题。更多详情，请参见["动态失效模型，" 第23.2.8节"](pt05ch23s02abm24.md)。

### 塑性功产生的热量

Abaqus允许执行绝热热应力分析（["绝热分析，" 第6.5.4节"](pt03ch06s05at20.md)）、完全耦合温度-位移分析（["完全耦合热应力分析，" 第6.5.3节"](pt03ch06s05at19.md)）或完全耦合热-电-结构分析（["完全耦合热-电-结构分析，" 第6.7.4节"](pt03ch06s07at23.md)），其中计算材料塑性应变产生的热量。此方法通常用于大量非弹性应变涉及的 bulk metal成形或高速制造过程的模拟，其中材料因变形而产生的加热由于材料属性的温度依赖性而成为重要效应。由于Johnson-Cook塑性模型是由高应变率瞬态动力学应用推动的，该模型中的温度变化通常通过假定绝热条件（单元之间没有热传递）来计算。热量由塑性功在单元中产生，产生的温升使用材料的比热计算。

此效应通过定义每体积热通量的非弹性耗散率分数来引入。

### 单元

Johnson-Cook塑性模型可用于Abaqus中包含力学行为（具有位移自由度的单元）的任何单元。
