# 73.2.4 几何集和曲面的重新生成







当您更改模型的基础几何体时，几何集和曲面会自动重新生成。但是，如果您对模型的几何体进行了重大更改，则先前包含在集或曲面中的对象可能变得不可识别。

例如，如果您删除或抑制部件的特征，则与该特征关联的集和曲面会发生更改：
- 如果该特征的组件是集或曲面中的唯一对象，则集或曲面仍显示在**Set Manager**中但为空。您可以编辑集以使其包含新的几何体。
- 如果该特征的组件与其他对象一起包含在集或曲面中，则集或曲面不再包含该特征的组件，但继续包含所有其他对象。
- 如果您抑制然后恢复特征，则与该特征关联的集和曲面以及分配给这些集和曲面的任何对象（如载荷或相互作用）也会恢复。
