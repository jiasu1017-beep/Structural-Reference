# 12.13.3 创建欧拉截面





欧拉截面用于指定可以存在于欧拉域中的材料。欧拉截面只能分配给欧拉类型部件，并且必须将单个欧拉截面分配给整个欧拉部件。

欧拉截面不在欧拉部件内创建材料；而是提供可能存在于部件内的材料列表。欧拉部件默认只包含空隙。创建欧拉截面后，您可以使用材料分配预定义场将材料添加到部件中（请参见["定义材料分配场，"第16.11.10节"](pt03ch16s11hlb10.md)）。有关更多信息，请参见[Abaqus Analysis User's Guide的第14.1.1节，"欧拉分析"](../usb/usb-link.md#usb-anl-aeuleriananalysis)。有关在Abaqus/CAE中建模欧拉分析的概述，请参见[第28章，"欧拉分析"](pt04ch28.md)。

**创建欧拉截面：**

1. 从主菜单栏，选择****Section**![](../graphics/images/arrow.gif)**Create****。出现**Create Section**对话框。
   **提示：**您也可以在**Section Manager**中点击**Create**，或在Property模块工具箱中选择创建截面工具![](../graphics/ico_section.png)。
2. 输入截面名称。有关命名对象的更多信息，请参见["使用基本对话框组件，"第3.2.1节"](pt01ch03s02s01.md)。
3. 选择**Solid**作为截面**Category**，选择**Eulerian**作为截面**Type**，然后点击**Continue**。出现欧拉截面编辑器。
4. **Material Instances**表格中的每一行代表欧拉截面中存在的每种材料。要向表格添加行，请在行上点击鼠标按钮3，并从出现的菜单中选择**Insert Row Before**或**Insert Row After**。
5. 对于**Material Instances**表格中的每一行，输入以下数据：
   **Base Material** 可能存在于欧拉部件中的材料。点击**Base Material**列，然后点击出现的箭头以显示可用材料列表，并选择适当的材料。
   **Instance Name** 用于在装配相关模块中引用基础材料的名称，例如当您使用材料分配预定义场定义欧拉域的初始组成时（请参见["定义材料分配场，"第16.11.10节"](pt03ch16s11hlb10.md)）。将为每个材料实例名称创建单独输出（请参见["查看欧拉分析的输出，"第28.7节"](pt04ch28s07.md)）。在某些情况下，可能需要创建多个引用相同基础材料的唯一材料实例名称；例如，如果您希望对包含相同基础材料的模型不同区域获取单独的输出数据。当您选择基础材料时，Abaqus/CAE自动创建材料实例名称；但是，您可以根据需要覆盖此默认实例名称。
6. 点击**OK**保存更改并关闭欧拉截面编辑器。
