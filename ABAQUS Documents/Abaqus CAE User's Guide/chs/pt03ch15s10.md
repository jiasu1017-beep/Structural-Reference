# 15.10 理解表示相互作用、约束和连接器的符号





当您将相互作用、约束和连接器应用到模型区域时，您可以选择在视口中显示符号，以指示您已应用相互作用、约束和连接器的位置。有关图形符号类型的信息，请参见「用于表示相互作用、约束和连接器的符号」，第C.2节。

您可以将相互作用、约束和连接器应用于几何或孤儿节点和单元。

**相互作用和约束**

如果您将相互作用或约束应用于几何，符号会大致均匀地出现在应用了相互作用或约束的表面或多个表面上。如果相互作用或约束定义涉及节点区域而不是表面，则符号会出现在节点区域的边缘和节点区域中的任何顶点上均匀分布。如果相互作用或约束应用于单个顶点，则会在该顶点处出现一个符号。

如果您将相互作用或约束应用于孤儿节点或单元，则符号会出现在基于表面的区域中每个单元面的中心，或基于节点的区域的节点处。

对于使用解析场分布的相互作用（和规定条件），符号会根据解析场值进行缩放。此外，在每个符号内部显示加号（+）或减号（-）以指示在该位置相互作用的大小是正还是负。当解析场对其区域的一部分评估为零时，Abaqus/CAE会显示缩小后的相互作用符号。这些缩小后的符号明显小于默认符号大小。有关这些符号的示例，请参见「理解规定条件符号类型、颜色和大小」，第16.5.1节。更多信息，请参见「使用解析表达式场」，第58.2节。

**连接器**

当您将连接器应用于线时，正方形会出现在线的第一点处，三角形会出现在线的第二点处。如果您为方向1指定了全局坐标系以外的 orientations，则方向 triad 会出现在线的第一点处。对于方向2，仅当您按名称指定坐标系时，方向 triad 才会出现在线的第二点处；如果您切换到**使用方向1**选项，则方向 triad 不会出现在第二点处。连接类型标签出现在所选线端点之间的线中点。您还可以显示与连接器截面分配关联的标签，尽管此显示默认情况下是关闭的。

有关控制这些符号可见性的信息，请参见「控制属性的显示」，第76.15节。
