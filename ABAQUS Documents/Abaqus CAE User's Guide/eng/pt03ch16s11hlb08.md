# 16.11.8 Defining a fluid velocity field







You can create a fluid velocity field to define initial values for fluid velocity in an Abaqus/CFD analysis. For more information, see [Chapter 30, "Fluid dynamic analyses](pt04ch30.md).”

**To create or edit a fluid velocity field:**

1. Display the fluid velocity field editor using one of the following methods: - To create a new fluid velocity field, follow the procedure outlined in ["Creating predefined fields," Section 16.8.3](pt03ch16s08hlb03.md) (**Category**: `Fluid`; **Types for Selected Step**: `Fluid velocity`). - To edit an existing fluid velocity field using menus or managers, see ["Editing step-dependent objects," Section 3.4.13](pt01ch03s04hlb05.md). **Note:**You can create or edit a fluid velocity field only in the initial step. For more information, see [Chapter 30, "Fluid dynamic analyses](pt04ch30.md)."
2. By default, the fluid velocity field is applied to the whole model. If you want to change the region for which the fluid velocity field applies, do the following: 1. Click ![](../graphics/ico_selectBlue.png). 2. Select one or more regions or sets from the viewport. 3. From the prompt area, click **Done**. Abaqus/CAE updates the fluid velocity field editor to reflect your selection.
3. Enter the initial values for the velocity components.
4. Click **OK** to save your data and to exit the editor.
