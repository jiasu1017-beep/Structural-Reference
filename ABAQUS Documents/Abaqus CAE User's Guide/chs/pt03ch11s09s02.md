# 11.9.2 壳特征





壳特征是实体的理想化，其中厚度被认为是相对于宽度和深度较小的。要创建壳特征，请从主菜单栏选择**形状**![](../graphics/images/arrow.gif)**壳**，或从零件模块工具箱中选择一个壳工具。您可以使用壳工具通过执行以下操作之一来创建壳特征：
- 要创建拉伸壳特征，请绘制轮廓并通过指定距离(*d*)拉伸。- 要创建壳放样特征，您可以从初始放样截面过渡到不同形状或方向的结束截面。- 要创建旋转壳特征，请绘制轮廓并通过指定角度(*α*)旋转。- 要创建扫掠壳特征，请绘制轮廓并沿指定路径扫掠。- 要创建平面壳特征，请在选定的平面或基准平面上绘制壳的轮廓。- 要创建实体抽壳特征，请将实体特征的表面转换为壳特征。

您可以使用任何壳工具向在三维建模空间中创建的零件添加壳特征；但是，在处理在二维或轴对称建模空间中创建的零件时，只能使用平面壳工具添加壳特征。您使用属性模块创建指定所需厚度的截面，并将截面分配给壳特征。
