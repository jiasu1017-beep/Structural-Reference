### 概述

连接器失效行为：
- 可在Abaqus/Standard中任何具有可用相对运动分量的连接器中定义；
- 可在Abaqus/Explicit中任何连接器中定义；
- 可在Abaqus/Standard中用于在满足失效准则时使所有或指定相对运动分量失效；
- 可在Abaqus/Explicit中用于在满足失效准则时使所有或指定分量失效；
- 如果指定分量中的连接器相对运动或连接器力超出指定范围，则可被触发；以及
- 在大多数情况下，可被更复杂的连接器损伤起始/演化行为取代（参见["连接器损伤行为，" 31.2.7节](pt06ch31s02alm33.md)）。

### 定义连接器失效行为

典型的连接器可能具有在相对运动分量、力或力矩变得过大时断裂的部件。Abaqus提供了一种方法来定义哪些相对运动分量将断裂以及用于释放这些分量的准则。您可以选择要基于其定义失效准则的相对运动分量。

在Abaqus/Standard中，连接器失效可用于基于可用相对运动分量指定连接器行为。在Abaqus/Explicit中，连接器失效可用于基于受约束以及可用的相对运动分量指定连接器行为。可以为连接中涉及的所有相对运动分量指定力或力矩的极限值。此外，对于具有可用相对运动分量的连接器，可以为对应于可用分量的相对位置指定极限值。

在Abaqus/Standard中，如果满足为所选相对运动分量指定的失效准则，则所有相对运动分量或单个可用分量失效。默认情况下，在满足失效准则时，所有相对运动分量都被释放。在满足失效准则的增量期间，将移除连接器单元所有已释放分量的节点力贡献。

在Abaqus/Explicit中，如果满足为所选分量指定的失效准则，则所有分量或单个可用分量失效。默认情况下，在满足失效准则时，所有分量都被释放。在满足失效准则的增量期间，将移除连接器单元所有已释放分量的节点力贡献。

| **输入文件用法：** | 使用以下选项定义连接器失效： |
| --- | --- |
|  | ``` [*CONNECTOR BEHAVIOR](../key/key-link.md#usb-kws-mconnectorbehavior), NAME=*name* [*CONNECTOR FAILURE](../key/key-link.md#usb-kws-mconnectorfailure), COMPONENT=*component number*, RELEASE=ALL or *component number* ``` |

#### Abaqus/Standard中的粘性阻尼

在Abaqus/Standard中，失效连接的突然释放可能导致收敛问题。为避免收敛问题，您可以向分量添加粘性阻尼。分量中的阻尼力计算为![](../graphics/usb_eqn05470.gif)，其中![](../graphics/usb_eqn00434.gif)是用户定义的阻尼系数，![](../graphics/usb_eqn05471.gif)是失效分量的速度。仅当释放所选可用相对运动分量时才应用粘性阻尼。

| **输入文件用法：** | 使用以下选项在Abaqus/Standard中向失效分量添加粘性阻尼： |
| --- | --- |
|  | ``` [*SECTION CONTROLS](../key/key-link.md#usb-kws-msectioncontrols), NAME=*name*, VISCOSITY=![](../graphics/usb_eqn00434.gif) [*CONNECTOR SECTION](../key/key-link.md#usb-kws-mconnectorsection), CONTROLS=*name* ``` |

| **Abaqus/CAE用法：** | Abaqus/CAE不支持粘性正则化。 |
| --- | --- |

### 输出

连接器可用的Abaqus输出变量在["Abaqus/Standard输出变量标识符，" 4.2.1节](pt02ch04s02abv01.md)和["Abaqus/Explicit输出变量标识符，" 4.2.2节](pt02ch04s02xbv01.md)中列出。在连接器中定义失效时，以下输出变量特别令人关注：

| CFAILST | 连接器失效状态标志。 |
| --- | --- |

| ALLVD | 添加到失效分量的粘性阻尼所耗散的能量。 |
| --- | --- |

在任何给定时刻，对于特定相对运动分量*i*，如果连接器在该特定相对运动分量中失效（满足失效准则），则输出变量CFAILST*i*为1。

如果在给定时刻对于特定分量*i*不满足失效准则，则输出变量CFAILST*i*为0。
