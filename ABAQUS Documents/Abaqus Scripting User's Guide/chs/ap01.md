# C 布局提示





## 附录C：布局提示

Abaqus GUI Toolkit中的布局管理器支持以下布局提示：

| 布局提示 | 用于 | 效果 |
| --- | --- | --- |
| LAYOUT_SIDE_TOP（默认）LAYOUT_SIDE_BOTTOM LAYOUT_SIDE_LEFT LAYOUT_SIDE_RIGHT | FXPacker FXGroupBox FXTopLevel | 如果指定这四个布局提示之一，子控件将分别粘贴在布局管理器空腔的顶部、底部、左侧或右侧。空腔的尺寸将根据被压缩控件切除的量而减小。LAYOUT_SIDE_TOP和LAYOUT_SIDE_BOTTOM将减小空腔的高度。LAYOUT_SIDE_LEFT和LAYOUT_SIDE_RIGHT将减小空腔的宽度。对于其他复合控件，这些提示可能没有任何效果。 |
| LAYOUT_LEFT（默认）LAYOUT_RIGHT | 所有 | 控件将被放置在容器中剩余空间的左侧或右侧。当用于FXPacker、FXGroupBox或FXTopLevel的子项时，除非指定了LAYOUT_SIDE_TOP或LAYOUT_SIDE_BOTTOM，否则该提示将被忽略。 |
| LAYOUT_TOP（默认）LAYOUT_BOTTOM | 所有 | 控件将被放置在容器中剩余空间的顶部或底部。对于FXPacker等的子项，这些选项仅在指定了LAYOUT_SIDE_RIGHT或LAYOUT_SIDE_LEFT时才会生效。 |
| LAYOUT_CENTER_X LAYOUT_CENTER_Y | 所有 | 控件将在父控件的X方向（或Y方向）上居中。将在控件周围添加额外间距以将其放置在可用空间的中心。控件的尺寸将为其默认尺寸，除非指定了LAYOUT_FIX_WIDTH或LAYOUT_FIX_HEIGHT。 |
| LAYOUT_FILL_X LAYOUT_FILL_Y | 所有 | 可以指定这些提示中的一个、两个或都不指定。LAYOUT_FILL_X将导致父布局管理器拉伸或收缩控件以适应可用空间。如果多个具有此选项的子控件并排放置，可用空间将按其默认尺寸的比例细分。LAYOUT_FILL_Y在垂直方向上具有相同的效果。 |
| LAYOUT_FIX_X LAYOUT_FIX_Y | 所有 | 可以指定这些提示中的一个、两个或都不指定。LAYOUT_FIX_X提示将导致父布局管理器将控件放置在指定的X位置，该位置作为控件构造函数参数列表中的可选参数传递。同样，LAYOUT_FIX_Y提示将导致放置在指定的Y位置。X和Y位置在父控件的坐标系统中指定。 |
| LAYOUT_FILL_ROW LAYOUT_FILL_COLUMN | FXMatrix | 如果为矩阵布局管理器中某一列的所有子控件指定了LAYOUT_FILL_COLUMN，则当矩阵水平拉伸时，整列可以拉伸。类似地，如果为某一行中的所有子控件指定了LAYOUT_FILL_ROW，则当矩阵布局管理器垂直拉伸时，整行将被拉伸。 |
| LAYOUT_FIX_WIDTH LAYOUT_FIX_HEIGHT | 所有 | 这些选项将把控件的宽度（或高度）固定为构造函数中指定的值。您可以使用setWidth()和setHeight()方法更改控件的尺寸；但是，布局管理器通常会遵守控件的指定尺寸而不尝试修改它（除非其他选项覆盖）。 |
| LAYOUT_MIN_WIDTH（默认）LAYOUT_MIN_HEIGHT（默认） | 所有 | 可以指定这些提示中的一个、两个或都不指定。您几乎不需要指定这些选项，除了可能是为了代码的可读性。如果未指定LAYOUT_FIX_WIDTH或LAYOUT_FIX_HEIGHT，这些选项将导致父布局控件分别使用默认（或最小）宽度和高度。 |
