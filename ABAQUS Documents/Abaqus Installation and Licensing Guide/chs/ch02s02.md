# 2.2 Abaqus 文档安装详情





Abaqus 在线文档提供两种格式：HTML 和 PDF。大多数安装从单个服务器共享 HTML 和 PDF 文档。这两种格式的文档也可以安装在任意数量的计算机上，以供个人用户本地访问。

用于安装 Abaqus 文档的命令和选项列在["文档安装，"第 2.1.1 节](ch02s01.md#sgb-chp-install-glance-doc)中。

Abaqus HTML 文档可以安装在支持 Abaqus 6.14 的平台上，通常应仅安装在网络的一台计算机上。您为 HTML 和 PDF 文档选择的服务器计算机应能被您计划运行 Abaqus 的所有计算机访问。

Abaqus HTML 加 PDF 文档的磁盘空间要求约为 2.0 GB。

Abaqus PDF 文档并不意味着替代 HTML 格式文档。但是，它具有以下优点：
- 需要更少的磁盘空间。
- 方便的便携式参考资料。
- 能够打印一定范围的页面，例如一个章节。
- 比 HTML 文档提供更高质量的打印输出。
- HTML 文档的替代搜索方式，特别是在 HTML 文档因未安装 Web 服务器而无法搜索的情况下。

仅安装 PDF 文档具有以下缺点：- 使用每个 Abaqus/CAE 功能的详细分步说明仅在 HTML 版本的 [Abaqus/CAE User's Guide](../usi/usi-link.md#usi) 和上下文相关帮助中提供。它们不会出现在指南的 PDF 版本中。
- 如果您未安装 Abaqus HTML 文档，上下文相关帮助中的超链接将无法运行。
- Abaqus/CAE 和 Abaqus/Viewer 中**帮助**菜单下的一些项目只会连接到 HTML 帮助，而不是 PDF 文档。

 Abaqus PDF 集合包含 HTML 集合中的所有书籍。所有指南的 PDF 版本（[Abaqus Scripting Reference Guide](../ker/ker-link.md#ker) 和 [Abaqus GUI Toolkit Reference Guide](../gui/gui-link.md#gui) 除外）的每个页面左上角都有一个 ![](../graphics/hhp-pdf-print-exa-button.gif) 图标，允许您打印选定的部分而无需指定页面范围。有关导航和打印 Abaqus PDF 文档的更多信息，请参阅 Using Abaqus Online Documentation 的[第 5 章，"Abaqus PDF 文档概述"](../hhp/hhp-link.md#hhp-chp-pdf-use)。

### 2.2.1 文档安装配置

当 Abaqus 文档安装程序运行时，它会确定任何现有 Abaqus 文档的配置：
- 如果您已安装以前版本的文档（例如 Abaqus 6.13），新的 Abaqus 6.14 文档将安装在與旧版本相同的位置。Abaqus 安装程序创建一个名为 `v6.14` 的新子目录来存放 Abaqus 6.14 文档；请参阅[附录 B，"Abaqus 发布目录和文件"](ap02.md)，了解目录结构信息。如果您使用的是 Abaqus Web 服务器软件，系统会提示您选择是否同时提供所有版本的文档。
- 如果未安装以前版本的 Abaqus 文档，系统会提示您输入安装目录。这应该与您的 SIMULIA 父目录相同（称为 `*simulia_dir*`）。默认位置在 Linux 平台上是当前工作目录，在 Windows 平台上是 `C:\SIMULIA`。您不应将 Abaqus 文档安装在指示特定 Abaqus 版本号的目录中。安装程序会在指定目录下创建一个名为 `Documentation` 的子目录。

在 HTML（加 PDF）安装期间，您必须选择以下选项之一：- 安装并启动 Abaqus Web 服务器软件（推荐选项，允许搜索功能）。
- 安装 HTML（加 PDF）文档并使用现有的非 Abaqus Web 服务器。在这种情况下，您必须手动配置 Web 服务器软件以访问 Abaqus HTML 文档（请参阅["配置现有的非 Abaqus Web 服务器，"第 C.1.1 节](ap03s01.md#sgb-chp-web-server-configure)）。
- 安装 HTML（加 PDF）文档但无 Web 服务器。在这种情况下，HTML 文档中没有搜索功能。如果您想从本地系统查看 HTML 文档而无需执行活动的 Web 服务器，请选择此选项。

除"无 Web 服务器"外，您选择的安装选项适用于 Abaqus 6.14 HTML 文档和（如果存在）以前版本的 Abaqus 文档。如果您为 Abaqus 6.14 安装选择"无 Web 服务器"选项，则 Abaqus 6.14 HTML 文档没有 Web 服务器，而以前版本的文档不受影响。

**注意：**在 Windows 平台上，Abaqus Web 服务器软件将在重启后自动启动。在 Linux 平台上，Abaqus Web 服务器由安装程序启动，但您应该将重启 Web 服务器的脚本（`*simulia_dir*/Documentation/installation_info/v6.14/startServer`）包含在系统启动文件中，以便 Abaqus Web 服务器在系统重启后自动重启。请联系您的系统管理员或平台供应商，获取有关计算机系统启动文件的信息。

安装完成后，日志文件将写入以下目录：

```
`*simulia_dir*/Documentation/installation_info/v6.14/`
```

日志文件有助于排除问题，不应删除。有关访问和使用 Abaqus HTML 文档的更多信息，请参阅["Abaqus 文档，"Abaqus Analysis User's Guide 第 3.2.11 节](../usb/usb-link.md#usb-int-ddocproc)，以及在线书籍 [Using Abaqus Online Documentation](../hhp/hhp-link.md#hhp)。

### 2.2.2 在多台计算机上重复文档安装

当您安装文档时，会在 `*simulia_dir*/Documentation/installation_info/v6.14/` 目录中创建一个名为 `installer.properties` 的文件。此文件称为重放文件，包含文档安装期间使用的信息，例如安装目录和 Web 服务器配置。`-replay` 命令选项可用于在相同平台上重复以前安装而无需为每次安装重新输入相同信息。您可以使用此选项执行无提示安装（不显示图形用户界面）。它使用重放文件中存储的信息执行与以前安装相同的安装（相同的安装目录、相同的配置等）。

为第一次安装输入的信息必须对使用 `-replay` 选项的后续安装有效。使用此选项时，会跳过所有安装错误检查。在执行安装之前，您应确保系统满足[附录 A，"系统要求说明"](ap01.md)中的要求，并且您在安装目录中具有写权限和足够的磁盘空间。要使用 `-replay` 选项，请执行以下操作：

1. 使用图形用户界面（不使用 `-replay` 选项）在计算机上安装文档。您可以安装任一选项：HTML 和 PDF，或仅 PDF。
2. 将 `installer.properties` 文件（重放文件）复制到要安装 Abaqus 文档的其他计算机上（必须是相同平台）或复制到已挂载的网络驱动器上。
3. 使用 `-replay` 选项在另一台计算机上安装文档，如下所示：**Linux 平台** ``` /*doc_dir*/setup -replay *full_path_to_replay_file* ``` **Windows 平台** ``` *doc_drive*:\setup.exe -replay *full_path_to_replay_file* ``` 文档将被无提示安装。
4. 查看安装日志文件中的错误： - `html_InstallLog.log` 和 `info.log`（用于 HTML 和 PDF 安装） - `pdf_InstallLog.log`（用于仅 PDF 安装） 这些文件位于以下目录：`*simulia_dir*/Documentation/installation_info/v6.14/` 此步骤是可选的，但建议执行。
5. 使用 `-replay` 选项在每台剩余计算机上重复文档安装。
