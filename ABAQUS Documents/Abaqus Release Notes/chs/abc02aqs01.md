# 2.1 从 Abaqus/CAE 访问 Learning Community





**产品：** Abaqus/CAE

**优势：** Abaqus/CAE 提供对 Learning Community 的便捷访问。

**描述：** 现在，您可以从 Abaqus/CAE 的主菜单栏访问 Learning Community（[http://www.3ds.com/products-services/simulia/academics/simulia-learning-community](http://www.3ds.com/products-services/simulia/academics/simulia-learning-community)）。Learning Community 包含在线教程和技术内容。该社区还设有问答区域，使全球用户社区能够分享他们的专业知识，并了解如何利用 SIMULIA 产品组合中可用的最新功能和增强功能。

**Abaqus/CAE 使用：**
```
所有模块：
   **Help**![](../graphics/images/arrow.gif)**Learning Community**
```
