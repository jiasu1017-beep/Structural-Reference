# 14.9.1 步骤管理器





### 14.9.1 **步骤管理器**

您使用**步骤管理器**来创建、编辑和操作与当前模型关联的分析步骤。要启动**步骤管理器**，请从主菜单栏选择**步骤**![](../graphics/images/arrow.gif)**管理器**。**步骤管理器**对话框中的列显示有关每个步骤的以下信息：

****名称****

步骤的名称。线性扰动步骤的名称相对于通用步骤的名称缩进。

**过程**

在创建步骤时为您选择此步骤的分析过程。您可以在创建步骤后更改分析过程。点击**替换**为所选步骤选择新的过程类型。**过程**列还指示热步骤和土步骤是否假设稳态或瞬态条件，或者两者都不适用。

****Nlgeom****

分析步骤是否考虑几何非线性。您使用**Nlgeom**按钮控制特定步骤的**Nlgeom**设置。一旦您为某个步骤设置了**Nlgeom**选项，您的设置将影响所有后续步骤。

****时间****

步骤的时间周期。时间周期的默认值是1.0时间单位。点击**编辑**显示步骤编辑器，以便您可以修改时间周期。

您使用**步骤管理器**对话框底部的按钮来创建跟随所选步骤的步骤，或操作所选步骤。您使用**关闭**按钮来关闭**步骤管理器**对话框。您可以使用位于主菜单栏的**步骤**菜单中的下拉菜单执行相同的任务。

您可以抑制分析步骤以从分析中排除该过程。被抑制的步骤从上下文栏、重启动请求对话框和诊断打印对话框中移除。在步骤中创建的任何步骤相关或传播属性会自动被抑制，并在分析期间被忽略。恢复步骤时，每个属性的状态将返回到原始状态。例如，抑制和恢复步骤不会恢复之前被抑制的关联载荷。只要步骤序列保持有效，您就可以抑制或恢复步骤。

**警告：**如果您使用**步骤管理器**或**步骤**菜单删除步骤，则与该步骤关联的对象（如指定条件或输出请求）也会被删除。如果您使用**步骤管理器**或**步骤**菜单替换步骤，则与新分析过程不兼容的对象会被替换为等效对象（如果可能），或被删除。
