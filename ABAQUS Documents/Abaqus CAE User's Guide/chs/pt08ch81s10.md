# 81.10 隐藏插件的源代码





如果您不想让其他用户看到插件的源代码，可以提供插件的编译版本而不是源代码版本。当 Abaqus/CAE 导入插件的源代码版本时，Abaqus/CAE 首先解析插件，然后将其编译成可执行版本（称为 `*filename*.pyc*），然后再执行该文件。如果自上次编译插件以来源代码没有更改，则 Abaqus/CAE 直接执行编译后的版本并跳过编译步骤。如果您删除了包含源代码的文件但保留了插件的编译版本，Abaqus/CAE 仍然会导入该插件。

如果您只有某些插件的编译版本，您可能希望将它们存储在 `abaqus_plugins` 目录下的一个目录中。这将通过把没有源代码的插件的编译版本与仍有源代码的插件的编译版本分开来避免混淆。
