# 19.2.5 选择作业类型





作业编辑器中的**提交**选项卡页面允许您在以下作业类型之间进行选择：

****完整分析****

选择此选项提交作业以生成（或重新生成）输入文件（如果作业与模型关联），对模型执行完整分析，并将结果写入输出数据库。此选项是默认选项。

****恢复（Explicit）****

仅当您运行 Abaqus/Explicit 时此选项可用。选择此选项提交作业以在 Abaqus/Explicit 意外停止后完成您的分析；例如，在磁盘填满后或网络问题后。

您不能使用此作业类型来恢复过早终止的 Abaqus/Standard 作业。相反，您应该使用 Abaqus/Standard 重启功能，如[「恢复 Abaqus/Standard 分析，」第 19.6.9 节](pt03ch19s06s09.md)中所述。

**重启**

选择此选项提交作业以使用指定模型先前分析的数据启动分析。您必须使用**编辑模型属性**对话框来指定要从中读取数据的作业，以及指定要重启分析的步骤。当您创建引用具有重启数据属性的模型的作业时，Abaqus/CAE 默认选择作业类型为**重启**。

重启分析在[「重启分析，」第 19.6 节](pt03ch19s06.md)中有更详细的描述。您不能为与输入文件关联的作业创建重启作业。

**作业类型**设置类似于 Abaqus 执行过程的参数；有关更多信息，请参见[「Abaqus/Standard、Abaqus/Explicit 和 Abaqus/CFD 执行，」Abaqus Analysis User's Guide 第 3.2.2 节](../usb/usb-link.md#usb-int-danalysisproc)。有关选择作业类型的详细说明，请参见[「选择作业类型，」第 19.8.3 节](pt03ch19s08hlb03.md)。
