# *BEAM SECTION OFFSET





### *BEAM SECTION OFFSET定义梁截面原点的偏移。

此选项仅能与[*BEAM GENERAL SECTION](ch02abk05.md)选项配合使用，用于梁横截面库中可用的标准横截面（详见["梁单元横截面方向，" Abaqus Analysis User's Guide第29.3.4节](../usb/usb-link.md#usb-elm-ebeamcrosssection)）。它用于定义横截面原点相对于梁轴关于局部（1，2）轴系的偏移。

**产品：**Abaqus/Standard

**类型：**模型数据

**级别：**部件、部件实例

##### **参考：**

- ["使用通用梁截面定义截面行为，" Abaqus Analysis User's Guide第29.3.7节](../usb/usb-link.md#usb-elm-eusingbeamgensect)
- [*BEAM GENERAL SECTION](ch02abk05.md)

**此选项没有参数。**
