# 29.4 创建离散紧固件





您可以在Interaction模块中通过选择连接所需表面的连接线来创建离散紧固件。对于每个连接线，Abaqus/CAE确定紧固点，并在紧固点及其对应表面之间应用分布耦合。

您必须为每个选择的连接线分配连接器截面。您可以使用任何Abaqus连接类型来建模紧固件的复杂行为，如包含弹性、损伤、塑性、摩擦影响的可变形连接器。此外，您可以为不同的连接线分配不同的连接器截面。

**要创建离散紧固件：**

1. 从Interaction模块的主菜单栏中，选择****Special**![](../graphics/images/arrow.gif)**Fasteners**![](../graphics/images/arrow.gif)**Create****。Abaqus/CAE显示**Create Fasteners**对话框。**提示：**您也可以使用Interaction模块工具箱中的![](../graphics/ico_fastener.png)工具创建离散紧固件。
2. 在出现的**Create Fasteners**对话框中，选择**Discrete**紧固件，然后单击**Continue**。
3. 选择表示离散紧固件位置的连接线，然后单击**Done**。更多信息，请参见["通过投影点创建连接线"，第59.6节](pt06ch59hla04.md)。Abaqus/CAE显示**Edit Fasteners**对话框。
4. 选择将紧固点运动耦合到影响半径内落在曲面上的节点平均运动的方法。
   - 选择**Continuum distributing**（默认）以将每个紧固点的位移和旋转耦合到节点的平均位移。
   - 选择**Structural distributing**以将每个紧固点的位移和旋转耦合到节点的平均位移和旋转。更多信息，请参见["定义表面耦合方法"中的"耦合约束"，Abaqus Analysis User's Guide第35.3.2节](../usb/usb-link.md#usb-cni-pcoupling-method)。
5. 如果需要，关闭旋转自由度以移除其约束。Abaqus/CAE始终约束分布耦合中的所有平动自由度。
6. 选择Abaqus将用于修改耦合节点处默认权重分布的加权方法：
   - 选择**Uniform**（默认）以选择等于1.0的均匀权重分布。
   - 选择**Linear**以选择随距离紧固点线性减小的权重分布。
   - 选择**Quadratic**以选择随距离紧固点按二次多项式函数减小的权重分布。
   - 选择**Cubic**以选择随距离紧固点按三次多项式单调函数减小的权重分布。更多信息，请参见["加权方法"中的"耦合约束"，Abaqus Analysis User's Guide第35.3.2节](../usb/usb-link.md#usb-cni-pcoupling-weighting)。
7. 如果需要，选择或创建指定约束自由度定义的初始方向的基准坐标系。默认情况下，约束自由度的方向由全局坐标系定义。
8. 指定以紧固点为中心的影響半径。您可以输入值或允许Abaqus使用整个曲面来定义耦合约束。
9. 单击**OK**关闭**Edit Fasteners**对话框。Abaqus/CAE显示离散紧固件在紧固点和连接面之间的分布耦合。圆的半径反映指定的影响半径。如果耦合约束定义使用整个曲面，则从紧固点出发的线将延伸超出圆。对于孤立网格模型，您可以通过更改装配显示选项中的面和边缘密度来修改分布耦合的外观；更多信息，请参见["控制属性的显示"，第76.15节](pt07ch76hla14.md)。
10. 如下所述为连接线分配连接器截面来完成离散紧固件定义，请参见["创建和修改连接器截面分配"，第15.12.12节](pt03ch15s12hlb11.md)。Abaqus/CAE显示离散紧固件的连接器符号。
