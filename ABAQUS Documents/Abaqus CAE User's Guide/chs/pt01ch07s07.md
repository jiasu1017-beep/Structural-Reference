# 7.7 选择背景颜色





模型的外观受模型颜色与视口背景颜色之间对比度差异的影响。你可以通过更改视口背景中显示的颜色来改善这种对比度。Abaqus/CAE 提供两种颜色选择：你可以显示填充整个视口背景的单一颜色，或者创建将两种不同颜色混合的渐变背景。

Abaqus/CAE 为会话中的每个视口实施你的背景颜色选择。

**要选择背景颜色：**

1. 找到背景颜色选项。从主菜单栏中，选择****View**![](../graphics/images/arrow.gif)**Graphics Options****。Abaqus/CAE 显示**图形选项**对话框。
2. 从**视口背景**字段中，选择以下之一：
   - **纯色**为视口背景选择一种颜色
   - **渐变**选择用于从视口顶部到底部逐渐混合的背景顶部和底部颜色
3. 选择视口背景颜色或颜色。
   1. 单击颜色样本![](../graphics/usi-colorsample.png)。Abaqus/CAE 显示**选择颜色**对话框。
   2. 使用**选择颜色**对话框中的方法选择新颜色。更多信息，请参阅["自定义颜色，" 第 3.2.9 节](pt01ch03s02s09.md)。
   3. 单击**确定**关闭**选择颜色**对话框。颜色样本会变为所选颜色。
   4. 对于渐变背景，你可以直接选择顶部和底部颜色；或者你可以选择顶部颜色，然后单击**自动选择**让 Abaqus/CAE 根据顶部颜色的变体选择底部颜色。
   5. 草图或 X–Y 绘图在渐变背景下可能显示不清楚。对于这些视图，Abaqus/CAE 会显示纯色背景，即使你指定了渐变背景。要禁用此行为，请关闭**允许覆盖**。
4. 单击**确定**实施更改并关闭对话框。Abaqus/CAE 为会话中的每个视口实施你的背景颜色选择。你的更改会在会话期间保存。
