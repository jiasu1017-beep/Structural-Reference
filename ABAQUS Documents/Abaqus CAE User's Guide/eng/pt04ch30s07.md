# 30.7 Using Abaqus/CFD for co-simulation







You can solve complex fluid-structure interactions and conjugate heat transfer problems using the co-simulation technique to couple Abaqus/CFD with Abaqus/Standard and Abaqus/Explicit. For more information, see ["Co-simulation: overview," Section 17.1.1 of the Abaqus Analysis User's Guide](../usb/usb-link.md#usb-anl-acosimulationover).
