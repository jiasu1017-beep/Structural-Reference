# *DISPLAY BODY





### *DISPLAY BODY 定义仅用于显示的部件实例。

此选项用于指定部件实例应仅用于显示目的，不应参与分析。此选项必须与 [*ASSEMBLY](ch01abk14.md) 和 [*INSTANCE](ch09abk19.md) 选项结合使用。

**产品：**Abaqus/Standard  Abaqus/Explicit  Abaqus/CAE  

**类型：**模型数据  

**级别：**装配  

**Abaqus/CAE：**相互作用模块

##### **参考：**

- ["显示体定义，" Abaqus分析用户指南第2.9.1节](../usb/usb-link.md#usb-int-adisplaybody)

### **必需参数：**

INSTANCE

将此参数设置为要用作显示体的部件实例的名称。
