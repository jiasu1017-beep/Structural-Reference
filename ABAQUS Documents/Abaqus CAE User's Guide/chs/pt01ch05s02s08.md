# 5.2.8 自定义视图





**视图**工具栏允许你将自定义视图应用到所选视口中的模型。（视图是视口中模型位置、方向和缩放因子的组合。）

**注意：**默认情况下，**视图**工具栏在 Abaqus/CAE 主窗口中不可见。要显示**视图**工具栏，请从主菜单中选择****View**![](../graphics/images/arrow.gif)**Toolbars**![](../graphics/images/arrow.gif)**Views****。

自定义视图包括七个预定义视图（如前和后）和最多四个用户定义视图。

**预定义视图**

预定义视图基于假想立方体的六个面和一个等轴测视图。视图 triad 指示这个假想立方体在视口中的方向。[图 5-10](pt01ch05s02s08.md#viw-cube) 说明了六个预定义立方体面视图。

**注意：**当 *X–Y* 绘图显示在当前视口中时，预定义视图无效。

**图 5-10** 预定义视图。

![](../graphics/viw-cube-nls.png)

**用户定义视图**

你可以使用视图操作工具在视口中定位模型的视图，然后单击**视图**工具栏中的 ![](../graphics/ico_viewSave.png) 将视图保存为四个用户定义视图之一。你可以使用此保存的视图将视口中的对象恢复到已知方向，你也可以将保存的视图应用到其他视口。默认情况下，保存的视图不会在会话之间存储。如果你想保留保存的视图以供后续会话使用，可以将其保存到 XML 文件、模型数据库或输出数据库。有关更多信息，请参阅["管理会话对象和会话选项，" 第 9.9 节](pt02ch09s08.md)。

视图由三个组件组成：方向、缩放因子和位置。你可以使用**缩放和位置**选项选择是否保存这三个组件，如下所示：

****自动拟合****

当你选择此选项保存视图时，只保存方向。当你应用以此选项保存的视图时，会应用保存的方向，但会调整缩放因子和位置以使视图适应视口。

****保存当前****

当你选择此选项保存视图时，方向、缩放因子和位置都会被保存。当你应用以此选项保存的视图时，会将保存的方向、缩放因子和位置全部应用到视口中的对象。要通过并排放置视口并将已知方向、缩放因子和位置应用到每个视口来比较不同视口中的不同对象，请选择**保存当前**选项。
