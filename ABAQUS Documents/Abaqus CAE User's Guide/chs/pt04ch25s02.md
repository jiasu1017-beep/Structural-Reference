# 25.2 使用连续壳单元为零件划分网格


您可以使用连续壳单元来以比传统壳单元更高的精度对类壳实体进行建模,如["壳单元:概述," Abaqus Analysis User's Guide 第 29.6.1 节](../usb/usb-link.md#usb-elm-eshelloverview)中所述。此外,虽然您使用六面体或楔形单元对连续壳进行建模,但单元公式在计算上仍比实体连续单元更高效。

当您生成将被分配给连续壳单元的单元时,网格中的单元方向必须一致。例如,[图 25–2](pt04ch25s02.md#adv-continuum-stack) 显示了沿扫掠路径方向生成的扫掠网格。生成的单元沿扫掠路径方向堆叠;但是,如果您计划使用连续壳单元,单元应该沿其厚度方向堆叠。

**图 25–2** 堆叠方向对于连续壳单元不正确。

![](../graphics/adv-continuum-stack-nls.png)

您可以使用查询工具集来确定哪些面被指定为底面和顶面,并检查单元之间是否存在不一致的方向。更多信息,请参见["获取模型的一般信息," 第 71.2.2 节](pt06ch71s02hlb02.md)。在某些情况下,您可以分割模型并更改扫掠路径方向以获得正确的方向。或者,您可以分配与扫掠路径无关的方向。更多信息,请参见["应用网格堆叠方向," 第 17.18.8 节](pt03ch17s18hlb01.md)。
