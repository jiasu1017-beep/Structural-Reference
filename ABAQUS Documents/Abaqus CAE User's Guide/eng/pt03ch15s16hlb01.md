# 15.16.2 Performing a search for contact pairs







Once you have indicated the appropriate contact criteria, Abaqus/CAE searches your model to locate the surfaces that meet those criteria. For a discussion of how Abaqus/CAE interprets search criteria and detects contact pair candidates, see ["The contact detection algorithm," Section 15.6.2](pt03ch15s06s02.md).

**To perform a search for contact pairs:**

1. From the main menu, select either ****Interaction**![](../graphics/images/arrow.gif)**Find contact pairs**** or ****Constraint**![](../graphics/images/arrow.gif)**Find contact pairs****. **Tip:**You can also find contact pairs using the ![](../graphics/ico_contactPairFind.png) tool in the Interaction module toolbox. Abaqus/CAE displays the contact detection dialog box.
2. Enter the search parameters as described in ["Specifying search criteria for contact detection," Section 15.16.1](pt03ch15s16s01.md).
3. Click **Find Contact Pairs**.
4. Abaqus/CAE searches for potential contact pairs. When the search is complete, Abaqus/CAE populates the contact pair candidates table. To abort a search before it is complete, click **Stop** in the prompt area.
