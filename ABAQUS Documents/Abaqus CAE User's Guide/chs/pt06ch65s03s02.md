# 65.3.2 什么是几何状态？







几何状态是指部件或装配的内部几何表示；例如，定义拓扑的曲线和曲面的方程以及这些曲线和曲面的连接。几何状态是该内部表示的快照。Abaqus/CAE 在您处理模型并再生特征时，会在内存中定期存储此快照并创建几何状态。

如果 Abaqus/CAE 没有保存几何状态，再生将从您创建的第一个特征开始，并继续通过所有需要再生的特征，使再生非常耗时。通过保存状态，Abaqus/CAE 可以确定最接近被修改特征的状态。再生从该点开始，并继续通过所有需要再生的特征。因此，再生速度可以显著提高。

理想情况下，为了获得最大的再生性能，您会在每次特征修改后保存几何状态。但是，存储每个状态会消耗大量内存并阻碍整体性能。决定保存多少个状态是在再生速度和内存消耗之间的权衡。

默认情况下，Abaqus/CAE 自动为每个部件和装配保存最多五个几何状态。如果 Abaqus/CAE 已经存储了指定数量的状态，它会在保存最新状态之前删除一个较旧的状态。您可以使用 **Options**（选项）对话框中的 **Geometry Caching for Fast Regeneration**（快速再生的几何缓存）选项来更改 Abaqus/CAE 自动保存的状态数。Abaqus/CAE 使用内部逻辑来决定何时创建几何状态；您无法控制何时创建状态。
