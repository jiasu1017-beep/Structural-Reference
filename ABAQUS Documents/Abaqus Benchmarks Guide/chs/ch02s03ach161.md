# 2.3.15 螺旋桨旋转模拟



**产品：** Abaqus/Explicit

该基准问题包含大量刚体旋转，旨在说明Abaqus/Explicit中壳单元和实体单元的性能。

### 问题描述

模拟初始转速为314.16 rad/s（3000 rpm或每秒50转）的螺旋桨，持续时间为2秒。螺旋桨由一个刚性中心轮毂和两个可变形叶片组成，如图2.3.15-1所示。空心圆柱形轮毂长5英寸（0.1270 m），内半径为4英寸（0.1016 m），外半径为5英寸（0.1270 m）。叶片厚0.5英寸（0.0127 m），跨度为50英寸（1.27 m）。叶片与螺旋桨中心轴成20°角。整个螺旋桨组件由钢制成，具有以下属性：

| 弹性模量 = 30.0×10⁶ psi（207 GPa） |
| --- |
| 泊松比 = 0.3 |
| 密度 = 7.3×10⁻⁴ lbf-sec²/in⁴（7800 kg/m³） |

假设螺旋桨在真空中旋转，初始应力为零。一旦建立了由于离心载荷引起的应力状态，解（位移、应力和应变）预计将随时间围绕该状态振荡。由于离心载荷下叶片的拉伸导致平均应力状态，螺旋桨关于其中心轴的质量转动惯量略有增加。由于角动量守恒，质量转动惯量的增加预计将导致平均转速略低于初始转速。

#### 叶片和轮毂的网格划分

在这项研究中，螺旋桨的轮毂区域被建模为使用C3D8R单元离散的刚体。螺旋桨叶片使用S4R、S4RS或C3D8R单元进行离散。每个螺旋桨叶片使用单一单元类型进行离散，尽管两个不同的单元类型可以用于两个叶片。这导致了六种不同的模型排列，所有这些都在本例中进行了研究。

#### 单元公式的精度阶数

由于有限元经历了大量（>5）刚体转动，在截面控制中指定了二阶精确单元公式。使用二阶精确单元分析了具有不同单元组合的螺旋桨问题的六个模型。此外，使用两个叶片都使用S4R单元离散的模型，也使用默认的一阶精确单元公式选项进行了分析。

### 结果与讨论

该问题中的所有分析都使用双精度浮点精度进行，因为完成2秒持续时间模拟的时间增量数相当大（使用任一叶片中的C3D8R单元约需230万个增量，使用两个叶片中的壳单元约需130万个增量）。

图2.3.15-2显示了在2秒时间段结束时一个叶片使用S4R单元离散、另一个叶片使用C3D8R单元的模型的代表性变形构型图。所有七种情况下的变形构型都没有单元扭曲，并且每种情况下螺旋桨在2秒时间段结束时已经完成了约99.5转。相应地，当达到平均应力状态时，平均旋转速度约为312 rad/s（见图2.3.15-3）。

图2.3.15-4显示了这里考虑的七种情况的总模型能（ETOTAL）和总模型动能（ALLKE）。当ETOTAL保持不变时，能量平衡在整个分析中明显保持。动能也相当恒定，仅比模型的初始动能低约0.4%。这0.4%的差异由单元的内部能解释。一般来说，发现单元的弹性能（ALLSE）、人工能（ALLAE）和粘性能（ALLVD）与模型动能（ALLKE）相比可以忽略不计。

#### 总结

总之，基于此例：

1. 对于涉及大量刚体转动的问题，二阶精确单元公式能够捕捉行为而没有任何单元扭曲，并且强烈建议用于经历超过5转的单元的问题。
2. 对于使用S4R单元对两个叶片进行离散的螺旋桨模型，一阶精确公式选项产生的结果与二阶精确公式选项的结果比较吻合。
