# 11.8.1 参考点





您可以使用参考点工具集通过从主菜单栏选择**工具**![](../graphics/images/arrow.gif)**参考点**来创建与零件关联的参考点。零件只能包含一个参考点，Abaqus/CAE将其标记为**RP**。如果您尝试创建第二个点，Abaqus/CAE会询问您是否要删除原始点。零件上的参考点出现在装配中该零件的所有实例上。装配可以包含多个参考点，Abaqus/CAE将它们标记为**RP-1**、**RP-2**、**RP-3**等。有关参考点的更多信息，请参见第72章"参考点工具集"。

Abaqus/CAE在所需位置显示参考点及其标签。您可以通过从模型树中选择**重命名**来更改参考点标签。如果需要，您可以关闭参考点符号和参考点标签的显示；更多信息，请参见"控制参考点显示"，第76.11节。

如果零件是离散或分析刚体零件，您可以使用参考点来指示刚体参考点。创建装配时，参考点出现在零件的每个实例上。您使用相互作用模块对参考点施加约束，或使用载荷模块通过载荷或边界条件定义参考点的运动。您应用于参考点的运动或约束将应用于整个刚体零件。
