# *COUPLED TEMPERATURE-DISPLACEMENT






### *COUPLED TEMPERATURE-DISPLACEMENT完全耦合、同时进行热传递和应力分析。

此选项用于分析需要同时求解温度和应力/位移场的问题。

**产品：**Abaqus/Standard  Abaqus/CAE  

**类型：**历史数据 

**级别：**步骤

**Abaqus/CAE: **步骤模块

##### **参考：**

- ["完全耦合热应力分析，" Abaqus 分析用户指南第 6.5.3 节](../usb/usb-link.md#usb-anl-acouptempdisp)
- ["率相关塑性：蠕变和膨胀，" Abaqus 分析用户指南第 23.2.4 节](../usb/usb-link.md#usb-mat-cratedepcreep)

### **可选参数：**

ALLSDTOL

包含此参数以指示将在此步骤中激活自适应自动阻尼算法。将此参数设置为稳定能量与总应变能量的最大允许比值。初始阻尼因子通过 STABILIZE 参数或 FACTOR 参数指定。然后，阻尼因子将根据收敛历史和 ALLSDTOL 值在步骤中调整。如果此参数设置为零，则不会激活自适应自动阻尼算法；将在整个步骤中使用恒定阻尼因子。如果包含此参数但未指定值，则默认值为 0.05。如果省略此参数但包含了具有默认耗散能量分数值的 STABILIZE 参数，则将自动激活自适应自动阻尼算法，ALLSDTOL=0.05。

此参数必须与 STABILIZE 参数结合使用（请参阅 ["求解非线性问题，" Abaqus 分析用户指南第 7.1.1 节](../usb/usb-link.md#usb-anl-anonlineareqns)）。

CONTINUE

设置 CONTINUE=NO（默认）以指定此步骤不会从先前通用步骤的结果中继承阻尼因子。在这种情况下，初始阻尼因子将根据步骤第一个增量的求解重新计算，或可以直接指定。

设置 CONTINUE=YES 以指定此步骤将从紧邻前一个通用步骤结束时继承阻尼因子。

此参数必须与 ALLSDTOL 和 STABILIZE 参数结合使用。

CREEP

设置 CREEP=EXPLICIT 以在整个步骤中使用显式积分进行蠕变和膨胀效应，这有时可能计算成本较低。当 CREEP=EXPLICIT 时，时间增量将受到精度容差（CETOL 和 DELTMX）以及前向差分算子的稳定性限制的限制。有关积分方案的详细信息，请参阅 ["率相关塑性：蠕变和膨胀，" Abaqus 分析用户指南第 23.2.4 节](../usb/usb-link.md#usb-mat-cratedepcreep)。

设置 CREEP=NONE 以指定即使已定义蠕变或粘弹性材料属性，在此步骤中也不会发生蠕变或粘弹性响应。

ELECTRICAL

包含此参数以指定将在使用热-电-结构元素的模型区域中对电传导进行建模。

在这种情况下，Abaqus/Standard 将结合机械平衡和电传导方程求解热传递方程。

省略此参数以指定不会在这些区域中对电传导进行建模。

FACTOR

如果由于局部不稳定预计问题不稳定，且 Abaqus/Standard 计算的阻尼因子不合适，则将此参数设置为自动阻尼算法中使用的阻尼因子（请参阅 ["求解非线性问题，" Abaqus 分析用户指南第 7.1.1 节](../usb/usb-link.md#usb-anl-anonlineareqns)）。此参数必须与 STABILIZE 参数结合使用，并覆盖基于耗散能量分数值计算的自动阻尼因子。

STABILIZE

如果由于局部不稳定预计问题不稳定，则包含此参数以使用自动稳定。将此参数设置为自动阻尼算法（请参阅 ["求解非线性问题，" Abaqus 分析用户指南第 7.1.1 节](../usb/usb-link.md#usb-anl-anonlineareqns)）的耗散能量分数。如果省略此参数，则不会激活稳定算法。如果包含此参数但未指定值，则耗散能量分数的默认值为 2×10^-4，自适应自动阻尼算法将默认在此步骤中激活，ALLSDTOL=0.05；设置 ALLSDTOL=0 以停用自适应自动阻尼算法。如果使用 FACTOR 参数，则任何耗散能量分数值都将被阻尼因子覆盖。

STEADY STATE

包含此参数以选择稳态分析。如果省略此参数，则假定步骤涉及瞬态响应。如果包含此参数，将使用自动时间增量。

### **用于在瞬态分析中控制时间增量的可选参数：**

CETOL

将此参数设置为从增量开始和结束时计算的蠕变应变率得出的蠕变应变增量之间的最大差异，从而控制蠕变积分的精度。容差有时通过选择可接受的应力误差容差并除以典型弹性模量来计算。此参数仅在材料响应与时间相关（蠕变和膨胀）时才有意义。如果在此瞬态分析中省略此参数和 DELTMX 参数，则将使用固定时间增量，初始时间增量值恒定。

DELTMX

将此参数设置为增量内允许的最大温度变化。Abaqus/Standard 将限制时间步长，以确保在步骤的任何增量期间任何节点都不会超过此值。如果在此瞬态分析中省略此参数和 CETOL 参数，则将使用固定时间增量，初始时间增量值恒定。
