# 42.1 从输出数据库选择结果概述





Abaqus/CAE从输出数据库读取分析结果。输出数据库结果包括您在分析期间保存的场输出和历程输出变量。例如，如果在每10个增量后将数据写入输出数据库的场输出部分，则现在您只能以10个增量间隔选择结果。这些增量间隔称为帧。

除了您保存的分析结果外，您还可以对输出数据库场输出进行运算以创建新结果。您可以以下列形式显示输出数据库场输出变量和您在Abaqus/CAE会话中创建的场输出变量的值：变形图、云图、符号图或X-Y图；沿模型路径获得的X-Y数据；通过探测任何模型或X-Y图；或以表格报告形式显示。此外，您可以基于场输出值显示模型的子集，并可以根据这些值对模型的某部分进行颜色编码。

您可以以X-Y图的形式显示输出数据库历程输出值。

使用主菜单栏中的**Result**菜单访问影响结果的选项；以下菜单项可用：
- **Step/Frame**：控制Abaqus/CAE获取结果的步骤和帧。
- **Active Steps/Frames**：控制Abaqus/CAE在逐帧逐步查看结果、动画和X-Y图中显示的步骤和帧子集。
- **Section Points**：控制哪些截面点为积分点变量提供结果。
- **Field Output**：**Field Output**对话框包含以下选项卡：
  - **Primary Variable**：控制Abaqus/CAE显示结果的变量，以及适用的不变量或分量。
  - **Deformed Variable**：控制Abaqus/CAE用于显示变形模型形状的变量。
  - **Symbol Variable**：控制符号图中Abaqus/CAE使用的向量或张量变量以及可选分量。
  - **Status Variable**：控制从模型图中移除满足指定结果相关失败标准的单元。
- **History Output**：为X-Y绘图选择历程输出。
- **Options**：**Result Options**对话框包含以下选项卡：
  - **Computation**：显示场输出或不连续性，控制基于单元的场输出结果的平均，以及控制在区域边界处结果的计算。
  - **Transformation**：对场输出结果应用坐标系变换。
  - **Complex Form**：控制Abaqus/CAE显示复数结果的数字形式。
  - **Caching**：控制在后处理期间分析结果是否缓存在内存中，以及检查输出数据库更新结果的频率。
