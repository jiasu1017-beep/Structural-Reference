# 23.3 理解复合铺层和方向





复合铺层每个铺层中纤维的方向在决定模型的物理质量方面起着重要作用； 然而，在基于实际应用模型中定义这个方向并不简单。 Abaqus/CAE中的复合铺层通过从三个相互关联的参数导出纤维方向来使过程更易于管理——铺层方向、铺层方向和额外的旋转，如[图 23-5](pt04ch23s03.md#usi-layups-orientation)所示。

**图 23-5** 确定铺层方向。

![](../graphics/usi-layups-orientation-nls.png)

**铺层方向**

铺层方向定义铺层中所有铺层的基本或参考方向。 在常规和连续壳铺层中，Abaqus将指定的方向投影到壳表面上，使您选择的铺层方向方向与壳法线对齐。 在实体复合铺层中，方向不会被投影。

Abaqus/CAE提供了几个用于定义铺层方向的选项：
- **Part global**：默认情况下，铺层方向与零件的方向相同。
- **Coordinate system**：您可以创建和选择定义方向的基准坐标系统。
- **Discrete orientation**：您可以创建离散方向，为每个网格单元提供方向值来定义方向。
- **Discrete field**：您可以创建和选择定义空间变化方向的方向离散场。
- **User-defined**：您可以在用户子程序[`ORIENT`](../sub/sub-link.md#sub-xsl-orient)中定义方向。 此选项仅对Abaqus/Standard分析有效。
- **Normal direction**：对于除**User-defined**之外的所有选项，您可以选择定义复合铺层近似法线方向的轴。
- **额外旋转**：如果您选择**Coordinate system**、**Discrete orientation**或**Discrete field**来定义铺层方向，您可以指定整个铺层的法线方向额外旋转的角度（以度为单位）。 您可以使用标量离散场来指定空间变化的额外旋转角度。

**铺层方向**

铺层方向定义每个铺层的相对方向。 在常规和连续壳铺层中，Abaqus将指定铺层方向投影到壳表面，使铺层法线方向与壳法线和铺层堆叠方向对齐。 在实体复合铺层中，铺层是相对于铺层堆叠方向创建的，未投影的铺层方向定义铺层内的材料方向（请参阅[图 23-6](pt04ch23s03.md#usi-layups-ply-orientation-nls)）。

**图 23-6** 壳和实体复合铺层的铺层和铺层方向。

![](../graphics/usi-layups-ply-orientation-nls.png)

Abaqus/CAE从您可以指定的两个变量的组合计算铺层方向——坐标系统（CSYS）和关于法线方向的旋转角度。

在Abaqus/CAE尝试为复合铺层中的铺层方向绘制坐标系统的系统奇异点（即用户选择的坐标系统和来自几何或单元的几何法线无法解析为Abaqus/CAE中用于显示目的的有效方向）的情况下，坐标系统将被绘制为塌陷的。

如果铺层方向使用离散场指定，铺层方向将相对于零件的基础坐标系统显示。 对于连续壳单元，Abaqus/CAE不会将显示的方向投影到中面表面上。 在这两种情况下，您都可以执行数据检查并在Visualization模块中查看输出数据库以验证方向。 有关更多信息，请参阅["对模型执行数据检查，" 第19.7.3节](pt03ch19s07hlb03.md)。

**坐标系统**

Abaqus/CAE提供了以下选项来定义铺层的坐标系统：
- **Layup**：默认情况下，铺层的坐标系统与铺层的坐标系统相同。
- **CSYS**：您可以创建和选择定义铺层坐标系统的基准坐标系统。 如果您为铺层选择使用坐标系统，它会覆盖该铺层的铺层方向。 您还可以选择定义铺层法线方向的坐标系统的轴。 您选择的轴会在铺层表格的**CSYS**列中显示为最后一位数字。 例如，`Datum csys1.3`表示您选择`Datum csys-1`来定义铺层的坐标系统，并且您选择`3-axis`来定义法线。

**旋转角度**

旋转角度定义每个铺层中纤维相对于铺层坐标系统的方向。 例如，在典型复合层中，纤维可能相对于坐标系统以45度或+90度定向。 您也可以使用标量离散场来指定在铺层中空间变化的纤维方向。 如果您指定旋转角度，铺层将关于坐标系统法线逆时针旋转，角度是相对于1轴测量的。

[图 23-7](pt04ch23s03.md#adv-layups-csys) 显示了复合铺层中四个铺层的方向以及铺层表格中的相应条目。

**图 23-7** 确定每个铺层的方向。

![](../graphics/adv-layups-csys-nls.png)

Abaqus/CAE按如下方式确定铺层方向：
- 您选择铺层方向来定义**VerticalTape-1**的坐标系统，并输入旋转角度0。 结果铺层方向沿铺层方向的1轴。
- 您选择铺层方向来定义**VerticalTape-2**的坐标系统，并输入旋转角度90。 结果铺层方向是关于铺层方向3轴（法线方向）逆时针旋转90度。 旋转角度是相对于1轴测量的。
- 您选择**Datum csys-1**来定义**DiagonalTape-1**的坐标系统，并输入旋转角度0。 结果铺层方向沿**Datum csys-1**的1轴。
- 您选择**Datum csys-1**来定义**DiagonalTape-2**的坐标系统，并输入旋转角度90。 结果铺层方向是关于基准坐标系统3轴（法线方向）逆时针旋转90度。 角度是相对于1轴测量的。
