# 12.8.5 Defining field variables at a material point







In Abaqus/Standard you can introduce dependence on solution variables with user subroutine [`USDFLD`](../sub/sub-link.md#sub-xsl-usdfld). This subroutine allows you to define field variables at a material point as functions of time, of any of the available material point quantities listed in ["Abaqus/Standard output variable identifiers," Section 4.2.1 of the Abaqus Analysis User's Guide](../usb/usb-link.md#usb-out-houtputvar), and of material directions. Material properties defined as functions of these field variables may, thus, be dependent on the solution.

User subroutine [`USDFLD`](../sub/sub-link.md#sub-xsl-usdfld) is called at each point for which the material definition includes a reference to the user subroutine.

**To include a reference to user subroutine [`USDFLD`](../sub/sub-link.md#sub-xsl-usdfld) in a material definition:**

1. From the menu bar in the **Edit Material** dialog box, select ****General**![](../graphics/images/arrow.gif)**User Defined Field****. (For information on displaying the **Edit Material** dialog box, see ["Creating or editing a material," Section 12.7.1](pt03ch12s07hlb01.md).)
2. Click **OK** to close the **Edit Material** dialog box. Alternatively, you can select another material behavior to define from the menus in the **Edit Material** dialog box (see ["Browsing and modifying material behaviors," Section 12.7.2](pt03ch12s07hlb02.md), for more information).
