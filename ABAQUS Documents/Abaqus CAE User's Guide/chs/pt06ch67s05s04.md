# 67.5.4 缩放自由体切割的向量长度







**Scaling**（缩放）选项使您能够调整在自由体切割中显示的向量长度。Abaqus/CAE 将向量长度计算为模型大小或视口大小的百分比。如果显示多个自由体切割，Abaqus/CAE 使用最大的力向量或力矩向量作为长度计算的基础。您可以更改百分比值以增加或减小向量的大小，并且您可以选择模型或视口作为向量长度计算的基础。您还可以为力向量和力矩向量的显示设置不同的向量缩放值；**Free Body Plot Options**（自由体绘图选项）对话框为 **Force**（力）和 **Moment**（力矩）向量缩放提供相同的（但独立的）选项。

**设置向量缩放选项：**

1. 找到 **Color & Style**（颜色和样式）选项。在可视化模块中从主菜单栏中选择 ****Options**![](../graphics/images/arrow.gif)**Free Body****（选项→自由体），然后在出现的对话框中点击 **Color & Style**（颜色和样式）选项卡。
2. 分别点击 **Force**（力）选项卡或 **Moment**（力矩）选项卡来更改力向量或力矩向量的向量缩放设置。**Scaling**（缩放）选项位于页面下方的三分之一处。
3. 从 **Basis**（基准）选项中，选择 **Screen size**（屏幕大小）或 **Model size**（模型大小）。
4. 拖动 **Percent**（百分比）滑块以将向量长度建立为屏幕或模型大小的百分比。
5. 点击 **OK**（确定）以实施您的更改并关闭 **Free Body Plot Options**（自由体绘图选项）对话框。默认情况下，您的更改会在会话期间保存，并将影响该视口中所有后续的自由体切割图。如果您想在后续会话中保留您的更改，请将它们保存到文件。有关更多信息，请参阅["保存自定义设置以供后续会话使用，" 第 55.1.1 节"](pt05ch55s01s01.md)。
