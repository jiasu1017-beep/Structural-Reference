# 69.7.10 从单元面创建面







您可以从部件中的孤单元面创建几何面。Abaqus/CAE 基于所选单元面的节点位置创建新的几何面。顶点是在元素边方向有显著变化的边缘节点处创建的。新面看起来像模型中的其他几何面，Abaqus/CAE 将新面缝合到部件中相邻的面。默认情况下，新面与网格关联；您可以选择更改此行为。在模型树中，新面显示为 **Face from mesh**（来自网格的面）特征。创建几何面后，您无法更改构成几何面的单元面组；但是，您可以删除面并创建新面，并且可以缝合在几何面原始创建时未缝合在一起的面。

新的几何面与模型中现有的孤单元面共享空间。这可能导致某些区域显示为灰色未网格化几何颜色，而其他区域显示为孤网格的深绿色和可见单元边。这类似于自底向上网格化区域的外观，其中棕色区域颜色和青色网格颜色同时出现。您可以在模型树中抑制孤网格以仅查看新几何。

从孤网格面创建几何时，请记住您对结果几何的预期用途。您希望创建能够成为修改和网格划分良好基础的面。例如，当您选择孤单元面时，您的选择应在任何尖锐角或其他特征处结束，以便几何面也在该处结束，创建逻辑特征边。如果您确实选择了跨越尖锐角的单元面，Abaqus/CAE 将创建包含该角在内的单个几何面——此几何可能难以或无法使用可用的自顶向下网格划分技术进行网格划分。

**注意：**当您使用实体孤单元工作时，包含角两侧单元面的选择（来自同一孤单元的多个选择）不可接受用于创建单个几何面。

当 Abaqus/CAE 创建几何面时，默认情况下它会缝合边之间的小间隙以及分析表面之间的小间隙。您可以自定义决定小间隙是否在任一情况下缝合的容差值，并且可以完全关闭边之间小间隙的缝合。将大部分或全部缝合推迟到建模过程后期可能是一种更有效的建模选项，因为每个缝合操作都是处理器密集型的。分析表面的拟合不能推迟；您必须在创建新几何面之前执行此步骤。

**从单元面创建几何面：**

1. 从主菜单栏中选择 ****Tools**![](../graphics/images/arrow.gif)**Geometry Edit****（工具→几何编辑）。Abaqus/CAE 显示 **Geometry Edit**（几何编辑）对话框。**提示：**您也可以使用位于 Part（部件）模块工具箱中编辑工具处的 ![](../graphics/icoR_elementFacesToFace.png) 工具从单元面创建几何面。 有关工具箱中编辑工具的图表，请参阅["使用几何编辑工具集，" 第 69.1 节"](pt06ch69s01.md)。
2. 从对话框中选择 **Face**（面）类别和 **From element faces**（从单元面）方法。
3. 为新几何面选择单元面。您可以单独、按角度、按限制角度、按层或按分析形状选择单元面。 有关在视口中选择对象的更多信息，请参阅["在当前视口中选择对象，" 第 6.2 节"](pt01ch06s02.md)。您也可以选择现有曲面。您不能从二维或轴对称部件中选择单元面。**注意：**默认选择方法基于您最近使用的选择方法。要恢复为其他方法，请点击提示区域右侧的 **Select in Viewport**（在视口中选择）或 **Surfaces**（曲面）。
4. 如果您想控制缝合、分析表面拟合和网格关联选项，请点击提示区域中的 **Options**（选项），然后从出现的对话框中执行以下任何操作： - 切换开启 **Stitch with tolerance**（带容差缝合）以应用您创建的几何面中节点的缝合，并指定缝合容差值。 - 在 **Fit analytic surfaces with tolerance**（用容差拟合分析表面）字段中，指定您希望用于拟合选择中分析表面的缝合容差值。 - 默认情况下，Abaqus/CAE 将几何面与网格关联。更改此行为，请关闭 **Associate face with mesh**（将面与网格关联）。点击 **OK**（确定）关闭对话框。
5. 从提示区域点击 **Done**（完成）。Abaqus/CAE 创建几何面特征。过程在第 3 步使用最近使用的选择方法重新开始。
6. 要退出过程，请执行以下操作之一 - 点击提示区域中的取消按钮 ![](../graphics/fts-cancel.gif)，或 - 在 Abaqus/CAE 窗口中的任意位置点击鼠标按钮 2，或 - 从几何编辑工具集或 Part（部件）模块中的工具中选择其他操作。
