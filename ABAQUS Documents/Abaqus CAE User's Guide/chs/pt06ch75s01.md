# 75.1 什么是虚拟拓扑？







在某些情况下，部件或部件实例包含非常小的面和边等细节。这些特征对于组件的详细加工和包装设计可能很重要；但如果它们对所研究问题的力学影响很小，它们在数值分析中可能是冗余的。实际上，这些小细节可能会过度约束网格的生成，导致网格质量差或网格密度更细。在某些情况下，如果小面和边出现在您需要细网格的关注区域，则它们可能很重要。但是，在大多数情况下，这些细节对分析没有贡献，您希望Abaqus/CAE在网格生成过程和分析期间忽略它们。

模型的拓扑是其由面、边和顶点组成的组成。"虚拟拓扑工具集"允许您操作此拓扑并为网格划分目的创建一个简化形式。这个简化形式不同于真实模型，称为"虚拟拓扑"。由此操作产生的面和边被称为"虚拟的"。类似地，包含虚拟面和边的部件和部件实例被称为"虚拟的"。

"虚拟拓扑工具集"允许您通过将小面与相邻面组合或将小边与相邻边组合来移除小细节。要组合的面或边可以直接指定，也可以选择要忽略的边和顶点。

例如，[图75-1](pt06ch75s01.md#vto-piston-model)显示了一个活塞，其中混合表面相交处有一些小细节。这些细节导致包含一些薄片状单元的密集网格。您可以使用虚拟拓扑来忽略小细节，从而产生[图75-2](pt06ch75s01.md#vto-piston-virtual)所示的更均匀、更高质量的网格。

**图75-1** 小细节导致薄片状单元的密集网格。

![](../graphics/vto-piston-model-nls.png)

**图75-2** 使用虚拟拓扑忽略小细节可产生更均匀、更高质量的网格。

![](../graphics/vto-piston-virtual-nls.png)

Abaqus/CAE以与标准几何体相同的方式处理虚拟拓扑。例如，您可以执行以下操作：
- 划分虚拟拓扑。
- 在虚拟拓扑上使用几何编辑工具集。
- 在虚拟拓扑上使用Part模块工具，如拉伸、扫掠和混合。

例如，[图75-3](pt06ch75s01.md#vto-piston-quarter)显示了在Part模块中使用拉伸切割工具切割后，包含来自[图75-2](pt06ch75s01.md#vto-piston-virtual)虚拟拓扑的活塞。

**图75-3** 包含虚拟拓扑的部件被切割。

![](../graphics/vto-piston-quarter.png)

**注意：**当您将包含虚拟拓扑的部件导出到ACIS文件时，Abaqus/CAE会从导出的部件中移除虚拟拓扑。
