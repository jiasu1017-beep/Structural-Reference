# 79.2.1 生成叠加绘图





叠加绘图是在单个视口中包含多个绘图的显示。叠加绘图由层组成；您可以通过打开多个输出数据库自动创建层，或者可以使用**叠加绘图层管理器**手动创建层。您还使用**叠加绘图层管理器**配置叠加绘图。

**提示：**您还可以使用**允许多个绘图状态**![](../graphics/ico_plotMultipleStates.png)工具为单个输出数据库绘制任意组合的变形和未变形模型形状（无论是否带有轮廓、符号或材料方向）。由于所有绘图状态都是在单个层上创建的，**允许多个绘图状态**工具仅限于显示来自单个输出数据库、步骤和帧的数据。有关更多信息，请参见["显示多个绘图状态，"第55.6节](pt05ch55s06.md)。

**生成叠加绘图的步骤：**

1. 确定用于生成叠加绘图的方法：
   - 通过打开多个输出数据库自动创建层。
   - 手动创建层。
2. 要自动创建层，请执行以下操作：
   1. 从主菜单栏中选择****文件**![](../graphics/images/arrow.gif)**打开****。
   2. 从出现的**打开数据库**对话框中，选择**输出数据库(*.odb*)**作为**文件筛选器**。
   3. 选择要打开的输出数据库，切换**追加到层**，然后点击**确定**。有关更多详细信息，请参见["打开模型数据库或输出数据库，"第9.7.2节](pt02ch09s06hlb02.md)。叠加绘图会自动在视口中创建，包含输出数据库的组合内容，每个输出数据库被分配到一个单独的层。您可以选择****视图**![](../graphics/images/arrow.gif)**叠加绘图****从主菜单栏显示**叠加绘图层管理器**并查看层。
3. 要手动创建层，请执行以下操作：
   1. 创建您要包含在显示中的第一个绘图。
   2. 通过从主菜单栏中选择****视图**![](../graphics/images/arrow.gif)**叠加绘图****打开**叠加绘图层管理器**。**提示：**您也可以通过点击工具箱中的**叠加绘图层管理器**![](../graphics/ico_manager.png)工具来打开管理器。
   3. 点击**创建**创建一个包含当前视口中显示的绘图的层。**创建视口层**对话框出现。
   4. 输入层的名称，然后点击**确定**。新层出现在**叠加绘图层管理器**中，**可见**和**当前**列中都有复选标记。
   5. 在当前视口中创建一个新绘图。
   6. 重复步骤c到步骤e，根据需要创建尽可能多的层。在所有后续层的**创建视口层**对话框中，您可以通过切换**复制视图自**并从列表中选择一个层来选择复制现有层的视图。新层会在您创建时出现在**叠加绘图层管理器**中；最近创建的层将在**当前**列中有一个复选标记。
   7. 确认您要包含在绘图中的层在**可见**列中有复选标记，然后点击**绘制叠加**。层按照它们在**叠加绘图层管理器**中列出的顺序堆叠在当前视口中。
4. 从**叠加绘图层管理器**中，您可以通过点击层的**名称**或**对象**列来选择层，然后点击**删除**来删除该层。
5. 当叠加绘图显示在视口中时，您可以点击**叠加绘图层管理器**中的**绘制单个**退出叠加绘图状态。叠加绘图从视口中消失，显示返回到之前的绘图状态；但是**叠加绘图层管理器**保持打开。点击**解散**关闭**叠加绘图层管理器**。**提示：**您也可以随时点击工具箱中的![](../graphics/ico_overlayState.png)工具在单个绘图和叠加绘图状态之间切换。
