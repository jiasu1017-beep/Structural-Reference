# 13.10.7 平移零件或模型实例





从主菜单栏选择****Instance**![](../graphics/images/arrow.gif)**Translate****可以沿选定矢量移动所选零件或模型实例。矢量的方向和大小是任意的，但你只能沿旋转轴平移轴对称零件实例。如果平移与先前的位置约束冲突；例如，约束对齐两个面，Abaqus/CAE仅沿未约束自由度分量应用平移分量。如果所有自由度都被约束，Abaqus/CAE显示错误消息，平移失败。

当你在装配中创建第一个实例时，Abaqus/CAE显示一个图形，指示装配默认坐标系的原点和平移方向。你可以使用此图形来帮助你决定如何平移零件实例。此外，你可以使用查询工具集查看先前应用于实例的平移和旋转总和，以及选定顶点之间的距离。平移和旋转不被视为装配的特征，不能被编辑或删除。

**要平移零件或模型实例：**

1. 从主菜单栏，选择****Instance**![](../graphics/images/arrow.gif)**Translate****。
   **提示：**你也可以使用装配模块工具箱中的![](../graphics/ico_instanceTranslate.png)工具来平移实例。有关装配工具箱中工具的图表，请参见["使用装配模块工具箱，"第13.9节](pt03ch13s09.md)。Abaqus/CAE在提示区域显示提示来指导你完成过程。
2. 选择要平移的零件或模型实例。你也可以点击提示区域右侧的**Instance List**按钮并从出现的**Instance List**对话框中选择要平移的实例。
   **提示：**如果你无法选择所需的实例，可以使用**Selection**工具栏更改选择行为。更多信息，请参见["使用选择选项，"第6.3节](pt01ch06s03.md)。你可以使用**[Ctrl]****+Click**和**[Shift]****+Click**的组合来选择多个实例。Abaqus/CAE高亮显示选定的实例。
3. 选择平移矢量的起点。你可以选择的任意现有顶点或基准点，或者可以在提示区域的文本框中输入坐标。
4. 选择平移矢量的终点。同样，你可以选择任意现有顶点或基准点，或者可以在提示区域的文本框中输入坐标。Abaqus/CAE显示临时图像，指示将应用于选定实例的平移。应用后你不能编辑或删除平移。尝试平移实例可能会与现有位置约束冲突。Abaqus/CAE仅沿未约束自由度分量应用平移分量。如果所有自由度都被约束，Abaqus/CAE显示错误消息，平移失败。
5. 执行以下操作之一：
   1. 如果你确认平移正确，点击提示区域中的**OK**按钮。Abaqus/CAE平移实例并将其定位在与实例临时图像相同的位置。
   2. 如果你对平移不满意，点击**Previous**按钮（![](../graphics/afxI_backup.png)）并指定新的平移矢量。
   3. 通过点击取消按钮（![](../graphics/afxI_cancel.png)）中止平移。
