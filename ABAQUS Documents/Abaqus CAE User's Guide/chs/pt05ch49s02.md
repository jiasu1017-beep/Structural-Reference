# 49.2 生成和自定义基于对象的动画





在时间历史动画中，您可以自定义Abaqus/CAE包含在动画中的分析步骤和帧。在比例因子或谐波动画中，您可以自定义模型的表观运动。对于所有三种动画类型，您可以自定义基础的变形形状、等值线、符号或材料方向图。

在任何基于对象的动画播放之前或期间，您可以自定义支撑动画的变形形状、等值线、符号或材料方向图。例如，您可以切换图例、调整变形比例因子或选择不同的场输出变量。使用"视图"、"结果"和"选项"菜单来自定义动画的基础图。
