# 17.5.1 网格单元如何对应于 Abaqus 单元？







网格模块可以生成包含图 17-19 所示单元形状的网格。

**图 17-19** 单元形状。

![](../graphics/mgn-attributes1-nls.png)

Abaqus/Standard、Abaqus/Explicit 和 Abaqus/CFD 中的大多数单元都对应于所示的一种形状；也就是说，它们在拓扑上等同于这些形状。例如，尽管 CPE4、CAX4R 和 S4R 用于应力分析，DC2D4 用于传热分析，AC2D4 用于声学分析，但这五种单元在拓扑上都等同于线性四边形。

每个网格区域默认分配有一个或多个 Abaqus 单元类型。每个单元类型对应于可在该区域中使用的单元形状。例如，壳网格区域默认分配有四边形和三角形单元类型。但是，您可以更改与分配给区域的单元形状拓扑等效的任何 Abaqus 单元的单元分配。因此，您可以选择仅用全三角形单元对壳区域进行网格划分，而 Abaqus/CAE 会忽略四边形单元分配。

要更改与分配给区域的单元形状拓扑等效的 Abaqus 单元的单元分配，请从主菜单栏中选择 ****Mesh**![](../graphics/images/arrow.gif)**Element Type****。类似地，您可以选择 ****Mesh**![](../graphics/images/arrow.gif)**Controls**** 来选择网格划分的单元形状。
