# 3.8.16 垫片子模型





**产品：**Abaqus/Standard  

### 测试的单元

GKPS4    GKPE4    GKAX4    GK3D8    GK3D6    

### 测试的功能

子模型功能应用于垫片单元。全局分析和子模型分析都使用一般静态程序。

### 问题描述

**模型：**

所有全局模型由矩形垫片组成，长度为3，厚度为0.125。三维模型的宽度为1。子模型输入文件建模全局模型的右半部分。子模型上的所有节点都被驱动。

**材料：**

| 膜弹性模量 | 68.7×10^3 |
| --- | --- |
| 横向剪切刚度 | 1.06×10^4 |
| 泊松比 | 0.3 |
| 厚度行为 | 闭合为1时压力为2.75×10^4 |

**载荷和边界条件：**

垫片通过与两个刚性表面的接触加载。下刚性表面保持固定。上刚性表面向下移动并旋转，以在垫片中产生空间变化的应力响应。额外的边界条件施加在垫片上以抑制刚体运动。

### 结果与讨论

确认子模型应力分布与全局模型一致。
