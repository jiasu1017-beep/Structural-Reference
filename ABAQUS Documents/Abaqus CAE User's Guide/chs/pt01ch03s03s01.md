# 3.3.1 什么是工具箱和工具栏？





工具箱和工具栏是提供对常用Abaqus/CAE功能快速访问的图标集合。例如，可视化模块工具箱包含用于生成不同类型绘图的工具图标。[图3-11](pt01ch03s03s01.md#int-toolboxes)显示了可视化模块工具箱。一旦您进入模块，所有模块工具箱都会在绘图区域左侧立即可用。

**图3-11** 可视化模块工具箱。

![](../graphics/int-toolboxes-nls.png)

工具栏也包含用于访问Abaqus/CAE功能的图标集合。工具栏提供帮助您保存、操作和从模型中进行选择的辅助功能的访问；而工具箱包含用于创建或更改模型的关键功能。除了工具图标外，工具栏还可以包含与工具相关的选项列表。例如，**颜色编码**工具栏中的颜色映射列表包含各种用于对当前视口中显示的对象进行着色的方法。您还可以自定义工具栏内容，将工具栏移动到新位置，或关闭它们（有关更多信息，请参阅[第61章，"自定义工具集"](pt06ch61.md)）。工具箱不能移动或隐藏。

要获取工具的简短描述，请将光标放在该工具上一会儿；会出现一个包含描述或"工具提示"的小框。工具提示不适用于显示为暗淡的图标；要获取有关这些图标的信息，请改用[上下文相关帮助](pt01ch02s06s01.md)。
