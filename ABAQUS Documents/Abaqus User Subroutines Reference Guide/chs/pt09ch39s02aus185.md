### 概述

Abaqus/Explicit中的接触诊断允许您获取有关接触相互作用的表面和进展的详细信息。诊断可用于：
- 审查两个表面之间的自动调整，
- 揭示模型中潜在有问题的初始表面配置，
- 跟踪两个接触表面之间的过度穿透，和
- 审查与翘曲表面接触相关的警告。

### 审查初始过闭合表面的调整

在模型的初始配置中处于过闭合状态的接触表面由Abaqus/Explicit自动调整以消除过闭合（参见["在Abaqus/Explicit中控制通用接触的初始接触状态，" 第36.4.4节](pt09ch36s04aus158.md)，和["调整初始表面位置和在Abaqus/Explicit接触对中指定初始间隙，" 第36.5.4节](pt09ch36s05aus163.md)）。关于过闭合表面调整的信息有三个来源：状态（`.sta`）文件、消息（`.msg`）文件和输出数据库（`.odb`）文件。

#### 在状态和消息文件中获取过闭合表面的调整

默认情况下，Abaqus/Explicit将所有节点调整写入消息（`.msg`）文件——对于通用接触表面，还有接触偏移摘要——并在状态（`.sta`）文件中写入最大初始过闭合和最大节点调整的摘要列表，用于模拟第一步中定义的接触对。您可以选择禁止写入消息文件中的信息，仅将摘要信息写入状态文件。写入消息和状态文件的信息也写入输出数据库（`.odb`）以在Abaqus/CAE中使用。

| **输入文件用法：** | 使用以下选项获取详细诊断输出到消息文件和状态文件的摘要诊断输出： |
| --- | --- |
|  | ``` [*DIAGNOSTICS](../key/key-link.md#usb-kws-hdiagnostics), CONTACT INITIAL OVERCLOSURE=DETAIL (default) ``` 使用以下选项仅获取状态文件的摘要诊断输出（不会将接触诊断写入消息文件）： ``` [*DIAGNOSTICS](../key/key-link.md#usb-kws-hdiagnostics), CONTACT INITIAL OVERCLOSURE=SUMMARY ``` |

| **Abaqus/CAE用法：** | 您无法从Abaqus/CAE内部控制接触初始过闭合的诊断信息。使用以下选项查看保存的诊断信息： |
| --- | --- |
|  | 可视化模块：**工具**![](../graphics/images/arrow.gif)**作业诊断** |

#### 查看表面的调整

在第一步中，可以在中查看初始过闭合表面的调整。可以为第一步中定义的接触对绘制显示调整的位移形状图，用于零时刻的原始场输出帧。此外，输出变量STRAINFREE（参见["Abaqus/Explicit输出变量标识符，" 第4.2.2节](pt02ch04s02xbv01.md)）包含表示初始无应变调整的节点向量。默认情况下，如果Abaqus/Explicit进行了任何无应变调整，STRAINFREE会被写入零时刻原始场输出帧的输出数据库（`.odb`）文件。在Abaqus/CAE的可视化模块中，此变量的符号图显示表示各个节点如何被调整的向量，而等值线图则显示调整量的大小分布（您必须在中选择零时刻的原始输出帧，然后选择STRAINFREE输出变量）。对于第一步以外步骤中的过闭合，节点位移和加速度的矢量图在可视化调整时特别有帮助。可以在数据检查分析后在中查看此类图（参见["Abaqus/Standard、Abaqus/Explicit和Abaqus/CFD执行，" 第3.2.2节](pt01ch03s02abx02.md)）。

#### 可视化小滑动接触对的精确初始间隙

当为小滑动接触对指定精确初始间隙时，Abaqus/Explicit不会调整从表面的坐标（参见["调整初始表面位置和在Abaqus/Explicit接触对中指定初始间隙，" 第36.5.4节](pt09ch36s05aus163.md)）。因此，指定的间隙无法在后处理器（如Abaqus/CAE的可视化模块）中看到。因此，根据表面的初始几何和间隙或过闭合的大小，表面可能在后处理器中显示为打开或闭合，而实际上在模拟中它们刚刚接触。

### 检测通用接触域中的交叉表面

如果从表面最初通过大于主表面厚度的距离穿透双侧主表面，则严重过闭合的从节点会将主表面的背面视为适当的接触力方向。这些交叉表面中的从节点实际上被困在主表面后面。此问题在["在Abaqus/Explicit中控制通用接触的初始接触状态，" 第36.4.4节](pt09ch36s04aus158.md)和["调整初始表面位置和在Abaqus/Explicit接触对中指定初始间隙，" 第36.5.4节](pt09ch36s05aus163.md)中有更详细的讨论。

对于通用接触定义，激活了识别初始配置中表面交叉区域的诊断测试。默认情况下，当检测到两个相邻从节点（通过小面边缘连接）在主表面的相对侧上时，会向消息（`.msg`）文件发出警告消息。不会对主表面相对侧上的基于节点的表面节点发出此类警告，因为无法确定基于节点的表面节点之间的邻接关系。在某些涉及主表面角落的情况下，即使相邻从节点实际上在主表面的同一侧上，也可能发出此警告消息。在大型模型上执行诊断测试的CPU成本可能很大。您可以选择停用诊断测试并在这种情况下避免额外的CPU成本。

| **输入文件用法：** | 使用以下选项停用初始交叉表面的诊断测试： |
| --- | --- |
|  | ``` [*DIAGNOSTICS](../key/key-link.md#usb-kws-hdiagnostics), DETECT CROSSED SURFACES=OFF ``` |

| **Abaqus/CAE用法：** | 您无法从Abaqus/CAE内部排除初始交叉表面的诊断测试。使用以下选项查看保存的诊断信息： |
| --- | --- |
|  | 可视化模块：**工具**![](../graphics/images/arrow.gif)**作业诊断** |

### 通用接触表面之间的过度穿透

如["Abaqus/Explicit中的接触约束强制方法，" 第38.2.3节](pt09ch38s02aus182.md)中所述，Abaqus/Explicit中通用接触算法使用的惩罚约束强制方法允许一个表面轻微穿透另一个表面。"弹簧"刚度自动应用于表面以抵抗这些穿透。如果通用接触中涉及的节点没有足够的质量，默认选择的"弹簧"刚度可能不足以防止大穿透。例如，当无质量节点云（由运动耦合定义完全约束）与没有质量的完全约束刚面接触时，就会出现这种情况。

默认情况下，如果在节点到面接触中，节点穿透其跟踪面的距离超过通用接触域中典型面尺寸的50%，则认为穿透过度，Abaqus/Explicit会向状态（`.sta`）文件发出诊断消息。包含深度穿透节点的节点集也写入输出数据库（`.odb`）文件以在Abaqus/CAE中使用。您可以控制用于触发诊断消息的典型面尺寸分数。

| **输入文件用法：** | 使用以下选项控制用于触发深度穿透诊断消息的典型单元面尺寸的分数： |
| --- | --- |
|  | ``` [*DIAGNOSTICS](../key/key-link.md#usb-kws-hdiagnostics), DEEP PENETRATION FACTOR=*value* ``` |

| **Abaqus/CAE用法：** | 您无法从Abaqus/CAE内部控制深度穿透的诊断信息。使用以下选项查看保存的诊断信息： |

|  | 可视化模块：**工具**![](../graphics/images/arrow.gif)**作业诊断** |

### 高度翘曲表面的警告消息

沿着高度翘曲的表面计算正确的接触条件是非常困难的，Abaqus/Explicit采用专门算法在翘曲表面之间强制接触；这种专门算法比默认接触算法更昂贵（参见["Abaqus/Explicit中接触对的接触控制，" 第36.5.5节](pt09ch36s05aus164.md)）。默认情况下，Abaqus/Explicit每20个增量检查一次高度翘曲的表面。

Abaqus/Explicit第一次检测到表面高度翘曲时，会在状态（`.sta`）文件中写入警告消息。消息很简短；它仅说明哪个表面具有高度翘曲的小面。如果此表面上的其他小面后来变得高度翘曲，则不会发出其他警告消息。

如果您请求更详细的诊断警告消息，则消息文件将在每次在特定表面上发现翘曲小面时包含警告。警告将给出与翘曲小面关联的父单元（父单元是形成该小面的单元）以及小面的翘曲角度。

如果请求详细警告，计算时间和消息文件的大小可能会显著增加。您可以在后续步骤中切换回摘要警告或完全禁止翘曲表面警告。

如果分析以致命错误终止，预选输出变量将自动作为最后一个增量的场数据添加到输出数据库。

| **输入文件用法：** | 使用以下选项请求翘曲表面的详细诊断警告输出： |
| --- | --- |
|  | ``` [*DIAGNOSTICS](../key/key-link.md#usb-kws-hdiagnostics), WARPED SURFACE=DETAIL ``` 使用以下选项请求翘曲表面的默认摘要诊断输出： ``` [*DIAGNOSTICS](../key/key-link.md#usb-kws-hdiagnostics), WARPED SURFACE=SUMMARY ``` 使用以下选项完全禁止翘曲表面的诊断警告输出： ``` [*DIAGNOSTICS](../key/key-link.md#usb-kws-hdiagnostics), WARPED SURFACE=OFF ``` |

| **Abaqus/CAE用法：** | Abaqus/CAE不支持翘曲表面的诊断输出请求。 |
| --- | --- |
