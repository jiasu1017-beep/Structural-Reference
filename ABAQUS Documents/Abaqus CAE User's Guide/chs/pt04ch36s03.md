# 36.3 管理表皮和桁条加固





**Skin Manager**和**Stringer Manager**分别允许您创建和管理表皮和桁条加固。这些管理器各自包含您定义的表皮或桁条名称的列表。管理器中的**Create**、**Edit**、**Rename**和**Delete**按钮允许您创建新的表皮和桁条，或编辑、重命名和删除现有表皮和桁条。您还可以使用Property模块主菜单栏中的****Special**![](../graphics/images/arrow.gif)**Skin****菜单和****Special**![](../graphics/images/arrow.gif)**Stringer****菜单启动这些过程。选择主菜单栏中的管理操作后，过程与在管理器对话框中单击相应按钮完全相同。

**注意：**如果在为表皮或桁条分配截面、方向、法线或其他属性后重命名它们，分配将变为无效。
