# 3.2.26 元素矩阵装配工具





**产品：**Abaqus/Standard  

##### **参考**

- ["生成矩阵，" 10.3节](pt04ch10s03.md)

### 概述

**abaqus mtxasm**工具装配SIM文档中包含的元素矩阵，并且可选地将装配矩阵写入文本文件。如果SIM文档中已存在装配矩阵，您可以使用此工具将它们写入文本文件。

### 命令摘要

| **abaqus mtxasm** | [**job**](pt01ch03s02abx26.md#job1-usb-int-dmtxassemblyutil)=*name-of-assembled-mtx-sim-doc* |
| --- | --- |
|  | [[**oldjob**](pt01ch03s02abx26.md#oldjob2-usb-int-dmtxassemblyutil)=*name-of-element-mtx-sim-doc*] [[**text**](pt01ch03s02abx26.md#text3-usb-int-dmtxassemblyutil)] |

### 命令行选项

**job**

此选项用于指定包含装配矩阵或将要写入装配矩阵的SIM文档的名称，具体取决于操作模式。

**oldjob**

此选项用于指定包含要装配的元素矩阵的SIM文档的名称。如果未使用此选项，则使用**job**选项指定的SIM文档必须已存在并包含装配矩阵。

**text**

此选项用于将装配矩阵以矩阵输入格式写入文本文件。每个矩阵写入一个遵循命名约定的文件*jobname*_*matrix**N*.`mtx`，其中*jobname*是使用**job**选项指定的名称，*matrix*是一个四字母标识符（`STIF`、`MASS`、`DMPV`、`DMPS`或`LOAD`），指示与结构或热矩阵相关的矩阵类型，*N*是增量号。
