# 8 非线性





本章讨论 Abaqus 中的非线性结构分析。以下总结了线性分析和非线性分析之间的差异。

**线性分析**

到目前为止讨论的所有分析都是线性的：施加载荷和系统响应之间存在线性关系。例如，如果线性弹簧在 10 N 载荷下静态延伸 1 m，则在施加 20 N 载荷时将延伸 2 m。这意味着在线性 Abaqus/Standard 分析中，结构的柔度只需计算一次（通过组装刚度矩阵并求逆）。结构对其他载荷情况的线性响应可以通过将新的载荷向量乘以逆刚度矩阵来找到。此外，可以通过对常数进行缩放和/或叠加各种载荷情况来确定其对全新载荷情况的响应，前提是新载荷情况是先前载荷情况的和（或倍数）。这种载荷情况叠加原理假定所有载荷情况使用相同的边界条件。

Abaqus/Standard 在线性动力学模拟中使用载荷情况叠加原理，这在第 7 章"线性动力学"中讨论。

**非线性分析**

非线性结构问题是结构的刚度随其变形而变化的问题。所有物理结构都表现出非线性行为。线性分析是一种方便的近似，通常足以用于设计目的。显然，对于许多结构模拟，包括制造过程（如锻造或冲压）、碰撞分析以及橡胶组件（如轮胎或发动机支架）的分析，线性分析是不足的。一个简单的例子是具有非线性硬化响应的弹簧（参见[图 8-1](ch08.md#gss-spring)）。

![](../graphics/gss-spring-nls.png)

由于刚度现在依赖于位移，因此不能再通过将初始柔度乘以施加的载荷来计算任何载荷下弹簧的位移。在非线性隐式分析中，结构的刚度矩阵必须在分析过程中多次组装和求逆，使其比线性隐式分析求解起来贵得多。在显式分析中，非线性分析成本增加是由于稳定时间增量的减少。稳定时间增量在第 9 章"非线性显式动力学"中进一步讨论。

由于非线性系统的响应不是施加载荷大小的线性函数，因此不能通过叠加为不同载荷情况创建解。每个载荷情况必须定义为单独的分析。
