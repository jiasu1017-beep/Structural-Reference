# 15.13.3 为通用接触指定和修改接触初始化分配





接触初始化确定Abaqus在分析之前如何调整通用接触域中的表面。表面的接触初始化独立于接触域在相互作用编辑器的「Attribute Assignments」部分指定。接触初始化只能分配给Abaqus/Standard分析的初始步骤。有关接触初始化的更多信息，请参见「创建接触初始化」，第15.12.4节，以及「Abaqus/Standard中控制初始接触状态」，Abaqus Analysis User's Guide第36.2.4节。

**指定接触初始化分配的步骤：**

1. 使用以下方法之一显示通用接触相互作用编辑器：- 要创建新的通用接触相互作用，请按照「定义通用接触」，第15.13.1节中的说明操作。- 要编辑现有的通用接触相互作用，请从主菜单栏选择 ****Interaction**![](../graphics/images/arrow.gif)**Edit**![](../graphics/images/arrow.gif)***相互作用名称*****。
2. 点击相互作用编辑器「Attribute Assignments」部分中的「Contact Properties」选项卡（如果尚未选择）。
3. 如有必要，点击相互作用编辑器「Contact Properties」部分中的![](../graphics/icoR_contactInitsSmall.png)创建接触初始化定义；请参见「创建接触初始化」，第15.12.4节，获取更多信息。
4. 点击「Initialization assignments」旁边的![](../graphics/ico_edit.png)。此时会出现「Edit Initialization Assignments」对话框。默认情况下，当您从列表或表中选择一个表面时，Abaqus/CAE会在视口中高亮显示该表面；但是，高亮不适用于「(Global)」和「(Self)」。您可以关闭对话框底部的「Highlight selected regions」以关闭选择高亮。
5. 从左侧第一列的现有表面列表中选择一个或多个表面。选择「(Global)」在整个接触域与单个表面之间分配初始化。**提示：**您可以点击![](../graphics/ico_surface.png)定义新表面并将其添加到列表中。请参见「创建表面」，第73.3.3节，获取定义表面的说明。
6. 从第二列的现有表面列表中选择第二个或多个表面以定义表面对。- 当在任一列中选择多个表面时，所有可能的组合都会生成。- 要为自接触分配初始化，请在第二列中选择相同的表面名称或「(Self)」。- 任何对于接触域之外区域的初始化分配都将被忽略。
7. 从第三列的现有初始化列表中，选择要分配的初始化。您可以在「Edit Initialization Assignments」对话框中创建新的接触初始化（请参见「创建接触初始化」，第15.12.4节）。
8. 点击对话框中间的箭头![](../graphics/usi-itn-contactattribarrow.png)将您的选择转移到属性分配列表中。对话框右侧的表会更新以反映您的选择。
9. 根据需要重复上述步骤以完成初始化分配。如果您想删除初始化分配，请选择行并点击![](../graphics/ico_delete.png)。**注意：**分配顺序可能是相关的；当初始化分配重叠时，最后的分配将优先。
10. 点击「OK」保存您的选择并关闭「Edit Initialization Assignments」对话框。相互作用编辑器重新出现，并更新了单个初始化分配数量的信息。
11. 点击「OK」创建相互作用并关闭编辑器。

**修改接触初始化分配的步骤：**

1. 从主菜单栏选择 ****Interaction**![](../graphics/images/arrow.gif)**Edit**![](../graphics/images/arrow.gif)***相互作用名称*****。此时会出现「Edit Interaction」对话框。
2. 点击相互作用编辑器「Attribute Assignments」部分中的「Contact Properties」选项卡（如果尚未选择）。
3. 点击「Initialization assignments」旁边的![](../graphics/ico_edit.png)。此时会出现「Edit Initialization Assignments」对话框。
4. 从对话框右侧的表中选择包含要修改的初始化分配的行。
5. 从对话框左侧第三列的列表中选择新的初始化。
6. 点击对话框中间的箭头![](../graphics/usi-itn-modcontactprop.png)用新选择替换初始化。
7. 点击「OK」保存您的选择并关闭对话框。相互作用编辑器重新出现。
8. 点击「OK」保存您的更改并关闭相互作用编辑器。
