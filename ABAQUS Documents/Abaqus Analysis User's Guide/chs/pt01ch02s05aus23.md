### 概述

积分输出截面：
- 可以是二维或三维的；
- 可用于跟踪表面的平均运动；
- 可与积分输出请求结合使用来研究模型中的"力流"；以及
- 不对表面运动施加任何约束。

### 简介

积分输出截面是将表面与坐标系和/或参考节点相关联的一种方式，用于以下一个或两个目的：
- 跟踪表面的平均运动；和/或
- 在局部坐标系中表达通过表面传递的力和力矩，力矩关于随表面移动的点。

表面的平均运动可以作为积分输出截面定义上参考节点处的位移和/或旋转历史获得。您必须定义一个未连接到有限元模型任何其他部分的参考节点，并选择参考节点是仅跟随表面的平均平移还是同时跟随平移和旋转。由于参考节点未连接到模型其余部分，用于跟踪平均表面运动的积分输出截面定义不会对模型中任何节点的运动形成约束。

可以使用在穿过模型各个部分的各种内部横截面状表面上定义的积分输出来研究复杂模型中的"力流"。对接触外表面上的力求和或对通过表面之间的绑定约束传递的力求和同样有用，这是通过将积分输出截面定义与积分输出请求相关联来完成的。通过在积分输出截面定义上指定方向，矢量输出量可以用所选坐标系表示。该坐标系可以按参考节点处旋转自由度的量旋转。此外，跨表面的积分力矩输出可以关于可以按参考节点处平移自由度的量平移的位置进行。通过在同一表面上使用多个积分输出截面定义，可以用不同的坐标系和参考节点请求给定表面上的积分输出。

### 创建积分输出截面

您必须为每个积分输出截面分配一个名称。此名称用于将截面与积分输出请求相关联。此外，您必须识别定义截面的表面（请参阅 ["基于单元的表面定义，" 第 2.3.2 节"](pt01ch02s03aus17.md)）。

| **输入文件用法：** | ``` [*INTEGRATED OUTPUT SECTION](../key/key-link.md#usb-kws-mintegratedoutputsect), NAME=*section_name*, SURFACE=*surface_name* ``` |

| --- | --- |

#### 创建内部横截面表面

要研究通过模型中各种路径的"力流"，您必须创建穿过一个或多个区域的内部表面（类似于横截面），以便可以请求通过这些表面传递的总力和力矩的积分输出。您可以通过用平面切割模型的一个或多个区域来简单地创建在单元面、边缘或端点上的此类内部表面；有关更多信息，请参阅 ["基于单元的表面定义中的创建内部横截面表面，" 第 2.3.2 节"](pt01ch02s03aus17.md#usb-int-adeformablesurf-intsect)。

#### 积分输出截面参考节点

可以将参考节点与积分输出截面相关联，用于以下一个或两个目的：
- 跟踪表面的平均运动；和/或
- 在随参考节点运动移动的坐标系中计算积分输出请求中的变量。

如果必须跟踪平均表面运动，您必须定义一个未连接到有限元模型任何其他部分的参考节点，并选择参考节点是仅跟随表面的平均平移还是同时跟随平移和旋转。如果选择跟随表面的平均旋转，则除了参考节点处的平移自由度外，还将激活旋转自由度。此外，参考节点的初始位置可以自动调整到位于表面中心。

当具有参考节点的积分输出截面与积分输出请求相关联时，通过截面的总力矩将根据参考节点的当前位置计算。如果参考节点具有活动的旋转自由度，则用于表示积分输出变量的坐标系随参考节点的旋转旋转。

##### 将参考节点定位在表面中心

当参考节点未连接到模型的其余部分时，可以自动将参考节点重新定位在初始配置中表面的中心。

默认是使参考节点保持在其指定位置。

| **输入文件用法：** | 使用以下选项将参考节点定位在表面中心： |
| --- | --- |
|  | ``` [*INTEGRATED OUTPUT SECTION](../key/key-link.md#usb-kws-mintegratedoutputsect), REF NODE=*n*, POSITION=CENTER ``` |

| **Abaqus/CAE 用法：** | 步骤模块：积分输出截面编辑器：**锚定在参考点：编辑**：选择参考点：**将点移动到表面中心** |
| --- | --- |

##### 设置参考节点以跟踪表面的平均运动

通常，使用随平均表面运动移动的坐标系和点来获取表面上的积分输出是有意义的。当参考节点未连接到模型的其余部分时，可以指定它随表面的平均平移而平移（无任何旋转），或同时随表面的平均运动而平移和旋转。平均运动基于表面上不属于任何刚体的各个节点的质量加权运动。

默认情况下，参考节点不跟踪表面的平均运动。

| **输入文件用法：** | 如果参考节点必须随表面的平均平移而平移，请使用以下选项： |
| --- | --- |
|  | ``` [*INTEGRATED OUTPUT SECTION](../key/key-link.md#usb-kws-mintegratedoutputsect), REF NODE=*n*, REF NODE MOTION=AVERAGE TRANSLATION ``` 如果参考节点必须同时随表面的平均平移而平移和旋转，请使用以下选项： ``` [*INTEGRATED OUTPUT SECTION](../key/key-link.md#usb-kws-mintegratedoutputsect), REF NODE=*n*, REF NODE MOTION=AVERAGE ``` |

| **Abaqus/CAE 用法：** | 步骤模块：积分输出截面编辑器：**锚定在参考点：编辑**：选择参考点：**点运动：平均平移和旋转**或**平均平移** |
| --- | --- |

#### 积分输出截面局部坐标系

您可以在积分输出截面上定义局部坐标系，并将截面与积分输出请求相关联，以在局部坐标系中表达积分输出变量。您可以指定方向作为局部坐标系，并可能进一步将其投影到表面上。或者，您可以通过 following the Abaqus 约定将全局坐标系投影到表面上来形成局部坐标系（请参阅 ["约定，" 第 1.2.2 节"](pt01ch01s02aus02.md)）。如果未明确局部坐标系，则局部坐标系将初始化为全局坐标系。

如果指定了参考节点且该参考节点具有活动的旋转自由度，则初始坐标系（无论是明确定义的还是初始化为全局坐标系的）将随变形旋转。如果参考节点未连接到模型的其余部分且其运动基于表面的平均平移和旋转，则旋转和平移自由度在参考节点处被激活。

| **输入文件用法：** | 使用以下选项定义截面的初始坐标系： |
| --- | --- |
|  | ``` [*INTEGRATED OUTPUT SECTION](../key/key-link.md#usb-kws-mintegratedoutputsect), ORIENTATION=*orientation_name* ``` |

| **Abaqus/CAE 用法：** | 步骤模块：积分输出截面编辑器：**CSYS：编辑**：选择方向 |
| --- | --- |

##### 将坐标系投影到截面表面上
