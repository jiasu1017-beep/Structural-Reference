# 17.10.1 什么是自由网格？





与结构化网格不同，自由网格不使用预建立的网格图案。当您使用结构化网格技术对区域进行网格划分时，您可以根据区域拓扑预测网格图案。相比之下，在创建网格之前不可能预测自由网格图案。

由于是非结构化的，自由网格比结构化网格更具灵活性。使用自由网格技术进行网格划分的区域拓扑可以非常复杂。

您可以使用此技术对二维区域使用**三角形**、**四边形**或**四边形主导**单元形状选项，或对三维区域使用**四面体**单元形状选项进行网格划分。有关将单元形状分配给区域的更多信息，请参见["选择单元形状，"第17.18.2节](pt03ch17s18s02.md)。
