# 1.5.2 圆盘与基础接触的稳态旋转

**产品：** Abaqus/Standard

本示例说明了稳态滚动问题中粘弹性材料效应的性质，并作为稳态传输过程中使用的材料对流算法的验证测试。由于稳态传输能力使用运动学描述，这意味着材料通过网格流动，因此必须考虑历史依赖性材料响应的对流效应。Abaqus 在稳态传输分析中为粘弹性材料提供材料对流。该能力的概述在["稳态传输分析"，《Abaqus 分析用户指南》第 6.4.1 节](../usb/usb-link.md#usb-anl-asteadystatetransport)中提供。

我们使用独立的瞬态拉格朗日分析来获得用于验证稳态传输材料对流算法的参考解。Oden 等人（1986）也发表了一个类似问题的有限元分析和数值结果。

### 问题描述

模型由内半径为 1 和外半径为 2 的圆盘组成。没有使用特定的单位制，但假定单位是一致的。圆盘与平坦刚性表面接触并以恒定角速度旋转。忽略摩擦，因此圆盘沿表面旋转而不平移。也忽略惯性效应。材料是具有瞬时弹性模量 ![](../graphics/bmk_eqn00614.gif) = 100 和 ![](../graphics/bmk_eqn00615.gif) = 25、剪切松弛系数 ![](../graphics/bmk_eqn00616.gif) = 0.2 和松弛时间 ![](../graphics/bmk_eqn00617.gif) = 0.1 s 的不可压缩超弹性材料。沿轴向施加平面应变边界条件。

稳态传输分析能力需要身体横截面的有限元网格作为起点。横截面用轴对称 CAX4RH 单元离散化。假定圆盘内侧与刚性轮辋接触。我们通过运动学耦合约束来建模，该约束将内侧表面上的所有节点耦合到放置在车轴中心的参考节点。该节点用于在后续三维拉格朗日参考分析中指定圆盘的运动。指定了运动学耦合约束。

执行 **datacheck** 分析，将轴对称模型信息写入重新启动文件。然后在后续运行中读取重新启动文件，Abaqus 通过使用对称模型生成围绕对称轴旋转横截面来生成三维模型。生成有限元模型的这种方法被 Abaqus 需要用于定义模型中的流线。轴对称 CAX4RH 单元在模型生成期间转换为 C3D8RH 单元。由于基础不是轴对称的，它在三维模型中被定义为刚性表面。三维有限元网格如图 1.5.2-1（[图 1.5.2-1](ch01s05ach52.md#rolling3dmodel)）所示。为了获得参考解，拉格朗日分析使用类似的网格，只是整个圆周被细离散化以适应旋转运动过程中变化的接触条件。

我们还包括一个使用圆柱（CCL12）单元的模型。

### 载荷

载荷分两个分析步骤施加。在第一步中，通过向基础上的刚体参考节点施加 0.3 个单位的指定位移来使圆盘与基础接触（[图 1.5.2-1](ch01s05ach52.md#rolling3dmodel)）。使用提供完全松弛长期粘弹性解的静态载荷步进行此分析。长期解确保静态和慢速滚动解之间的平滑过渡。

第二个分析步骤是稳态传输分析。通过指定传输速度获得各种角速度（从 ![](../graphics/bmk_eqn00618.gif) = 0.001 rad/s 到 ![](../graphics/bmk_eqn00618.gif) = 1000 rad/s）的稳态解。

参考拉格朗日解使用准静态过程获得。文件 [spinningdisk_visco.inp](../eif/spinningdisk_visco.inp) 包含此分析的输入数据。

### 结果与讨论

当旋转体材料中的应力受应变率影响时（如粘弹性材料），变形取决于物体的角速度。在旋转运动过程中，进入接触区域（前缘）的材料被突然增加的接触压力压缩，而离开基础的材料则松弛。对于完全弹性材料，变形是可逆的，因此接触区域（和应力状态）关于垂直于基础并包含车轴的平面对称。另一方面，粘弹性材料对突然增加的接触压力瞬时响应，但在离开接触区域后需要有限时间来松弛。在这样的加载/卸载应力-应变循环中，一些应变能被耗散。换句话说，与完全弹性材料不同，变形是不可逆的，加载和卸载应力-应变路径不重合。因此，材料离开基础的点比材料进入接触区域的点更靠近中心平面。此外，由于接触压力不对称，滚动受到围绕车轴的力矩阻碍。

这个问题中粘弹性材料效应的性质如图 1.5.2-2（[图 1.5.2-2](ch01s05ach52.md#normal-reaction)）到图 1.5.2-4（[图 1.5.2-4](ch01s05ach52.md#pressure)）所示。图 1.5.2-2（[图 1.5.2-2](ch01s05ach52.md#normal-reaction)）显示了垂直于基础的支反力；图 1.5.2-3（[图 1.5.2-3](ch01s05ach52.md#reaction-torque)）显示了作为角旋转速度函数的围绕车轴的力矩。两图中的圆点代表参考瞬态拉格朗日解。图 1.5.2-4（[图 1.5.2-4](ch01s05ach52.md#pressure)）显示了不同角速度下的接触压力。这些图表明，在低角速度下，当材料点在接触区域停留的时间与材料松弛时间相比很长时，圆盘的行为对应于完全松弛的长期弹性解。垂直支反力（[图 1.5.2-2](ch01s05ach52.md#normal-reaction)）最小，应力状态关于中平面对称（[图 1.5.2-4](ch01s05ach52.md#pressure)），因此围绕车轴的力矩为零（[图 1.5.2-3](ch01s05ach52.md#reaction-torque)）。在高角速度下，解对应于垂直支反力达到极限值的瞬时（或动态）弹性解。应力状态仍然关于中平面对称，因此围绕车轴的力矩为零。当材料点在接触区域停留的时间与材料松弛时间相当时，粘弹性效应变得重要。在这些条件下，每次加载/卸载循环都会耗散能量，因此接触区域变得不对称（[图 1.5.2-4](ch01s05ach52.md#pressure)），滚动受到围绕车轴的力矩阻碍（[图 1.5.2-3](ch01s05ach52.md#reaction-torque)）。

图 1.5.2-5（[图 1.5.2-5](ch01s05ach52.md#stress-variation1)）到图 1.5.2-7（[图 1.5.2-7](ch01s05ach52.md#stress-variation3)）比较了粘弹性效应最大时（![](../graphics/bmk_eqn00618.gif) = 2.5 rad/s）两种分析方法之间的径向、环向和剪切应力。实线代表稳态传输解；虚线代表参考瞬态拉格朗日解。这些图沿着流线绘制了外表面附近的应力——角度是沿材料流动方向关于 ![](../graphics/bmk_eqn00619.gif) 轴测量的，![](../graphics/bmk_eqn00620.gif) 轴定义 ![](../graphics/bmk_eqn00198.gif) = 0。参考解通过在分析历史期间监测流线上一个积分点的应力来获得。由于解是稳态的，应力随时间的变化可以转换为沿流线的变化。这些图显示两种求解方法之间非常好的吻合。

使用圆柱单元获得的稳态传输解也与参考解非常吻合。此模拟的结果此处未报告。
