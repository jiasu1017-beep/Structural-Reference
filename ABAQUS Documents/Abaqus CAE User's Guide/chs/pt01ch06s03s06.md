# 6.3.7 在选择过程之前选择对象





你可以在选择过程之前从当前视口中选择对象。你可以使用**选择**工具栏来切换[选择](pt01ch06s03s03.md)并基于对象类型限制视口选择。**选择**工具是工具栏中的第一个工具；它仅在视口中没有活动过程运行时才可用。默认情况下，选择是活动的，对象类型不受限制。你可以使用["组合选择技术，" 第 6.2.8 节](pt01ch06s02hlb08.md) 中描述的任何方法选择多个对象。关闭对象选择以防止在非过程期间选择对象。

**注意：**对于大型模型，预选高亮可能会有延迟；关闭选择可能会通过阻止预选和选择来提高显示速度，除非你先选择一个需要对象选择的过程。

当你在从视口中选择对象之后选择过程时，Abaqus/CAE 会应用该过程的选择过滤器。例如，如果你选择了顶点、面和边，然后启动了一个仅能接受顶点的过程，Abaqus/CAE 会接受选中的顶点，取消选择面和边，并从第二步开始过程。类似地，如果你选择一个需要单个对象选择的过程，而你已选择了多个有效对象，Abaqus/CAE 会接受第一个有效选择并取消其余选择。如果 Abaqus/CAE 无法确定你首先选择了哪个对象，它将取消所有选择并从第一步开始过程。

如果一个过程包含多个选择步骤，你在过程之前选择的对象只能用于完成第一个选择步骤。任何后续选择步骤都需要你从视口中交互式地选择新对象，或者，如果适用的话，使用保存的选择组。你无法保存过程开始之前所做的选择。
