# 69.6.3 合并边







您可以将部件中所选的边合并为单条边。合并边也会移除冗余顶点和边。当您选择要合并的边时，Abaqus/CAE 搜索所有连接的边，直到遇到分支点，并将连接的边追加到您的选择中。合并边的操作作为部件的特征存储；因此，您可以使用模型树删除或抑制操作。

**合并边：**

1. 从主菜单栏中选择 ****Tools**![](../graphics/images/arrow.gif)**Geometry Edit****（工具→几何编辑）。Abaqus/CAE 显示 **Geometry Edit**（几何编辑）对话框。**提示：**您也可以使用位于 Part（部件）模块工具箱中编辑工具处的 ![](../graphics/ico_repairMergeEdges.png) 工具来合并边。 有关工具箱中编辑工具的图表，请参阅["使用几何编辑工具集，" 第 69.1 节"](pt06ch69s01.md)。
2. 从对话框中选择 **Edge**（边）类别和 **Merge**（合并）方法。
3. 选择要合并的边。Abaqus/CAE 搜索所有连接的边，直到遇到分支并选择所有连接的边。您可以从视口选择边或选择包含边的现有集合。**注意：**默认选择方法基于您最近使用的选择方法。要恢复为其他方法，请点击提示区域右侧的 **Select in Viewport**（在视口中选择）或 **Sets**（集合）。您可以使用拖动选择、**[Shift]****+点击**、**[Ctrl]****+点击** 和角度方法的组合来选择多条要合并的边。 有关更多信息，请参阅["在当前视口中选择对象，" 第 6.2 节"](pt01ch06s02.md)。**提示：**如果您无法选择所需的边，可以使用 **Selection**（选择）工具栏更改选择行为。 有关更多信息，请参阅["使用选择选项，" 第 6.3 节"](pt01ch06s03.md)。
4. 点击鼠标按钮 2 表明您已完成选择边。在可能的情况下，Abaqus/CAE 将所选边合并为单条边。如果没有可合并的边，Abaqus/CAE 会显示错误消息；否则，Abaqus/CAE 会提示您更新部件的有效性。
5. 执行以下操作之一： - 点击 **Yes**（是）更新部件的有效性。（每当您合并边时都应该更新部件的有效性。） - 点击 **No**（否）跳过更新并继续您的修复。 - 点击 **Cancel**（取消）取消当前修复操作中的更改。您也可以随时通过在模型树中的部件上点击鼠标按钮 3 并从出现的菜单中选择 **Update validity**（更新有效性）来更新部件的有效性。
