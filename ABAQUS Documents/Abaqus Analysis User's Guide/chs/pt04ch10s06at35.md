**产品：** Abaqus/Standard  Abaqus/Explicit

### 概述

网格化横截面：
- 允许描述包括多种材料和复杂几何形状的梁横截面；
- 在Abaqus/Standard中使用二维翘曲元素进行网格化，翘曲位移作为唯一的自由度；
- 生成可在Abaqus/Standard或Abaqus/Explicit的后续梁单元分析中使用的梁横截面属性；
- 仅允许各向同性线性弹性材料行为（["定义各向同性弹性" "线性弹性行为，" 第22.2.1节](pt05ch22s02abm02.md#usb-mat-clinearelastic-isotropic)）或用于翘曲元素的正交线性弹性材料行为（["为翘曲元素定义正交弹性" "线性弹性行为，" 第22.2.1节](pt05ch22s02abm02.md#usb-mat-clinearelastic-orthowarp)）；并且
- 允许在梁单元模型或二维翘曲单元模型上进行应力和应变后处理。

### 引言

某些结构的响应类似于梁，但梁横截面的几何形状或横截面的多材料组成不允许使用预定义的库梁横截面。在这些情况下，可以使用网格化横截面来对梁横截面建模，并生成适合在后续Timoshenko梁分析中使用的梁横截面属性。梁属性生成假设为厚壁（实体）横截面，具有无约束的面外翘曲，因此开口截面梁单元不能使用从网格化截面生成的梁横截面属性（参见["梁建模：概述，" 第29.3.1节](pt06ch29s03abo26.md)）。生成的梁横截面属性包括轴向、弯曲、扭转和横向剪切刚度；质量、转动惯量和阻尼属性；以及横截面的质心和剪切中心。此外，等效梁横截面属性包括应力恢复信息，如翘曲函数及其导数。

需要网格化横截面的结构典型示例是用于鞭击分析的船体，船体具有多单元格和多材料构造。其他示例包括翼形转子叶片或机翼、分层复合材料I梁（纤维沿梁轴向或垂直于轴向）等。

### 建模方法

如图[图10.6.1-1](pt04ch10s06at35.md#ebeamlib-meshed)所示，网格化横截面允许对梁横截面进行复杂描述：可以包含任意形状、多种材料、多个单元格和非结构质量。基本思想是创建梁横截面的二维有限元模型。网格化横截面在Abaqus/Standard中用于数值计算表征横截面结构响应所需的属性，以进行后续梁单元分析。二维Abaqus/Standard分析将横截面属性写入输入文件就绪的文本文件（`jobname.bsp`）。在后续的Abaqus/Standard或Abaqus/Explicit梁单元分析中，需要网格化横截面属性的梁单元将文本文件`jobname.bsp`作为通用梁截面数据包含在内。一旦梁单元分析完成，Abaqus/CAE的Visualization模块用于可视化沿梁长度预选点的结果，或检查直接显示在二维网格化横截面上的详细应力和应变结果。

总之，使用网格化横截面分析和后处理梁分析的过程如下：

1. 对梁横截面的二维Abaqus/Standard模型进行网格化并分析。
2. 在Abaqus/Standard或Abaqus/Explicit梁分析中使用生成的横截面属性。
3. 使用梁分析结果，从梁模型或二维横截面模型进行后处理。

#### 对梁横截面进行网格化并分析二维模型

横截面使用专用的二维单元进行网格化：WARP2D3（3节点三角形）和WARP2D4（4节点四边形）。这些单元每个节点有一个自由度，代表面外翘曲函数的值（参见["翘曲单元库，" 第28.4.2节](pt06ch28s04ael09.md)），并使用实体截面定义；不需要截面数据。横截面网格中的相邻单元必须共享公共节点；不允许使用多点约束进行网格细化。

横截面网格中的每个单元可以引用不同的弹性材料，使用各向同性线性弹性材料行为（参见["定义各向同性弹性" "线性弹性行为，" 第22.2.1节](pt05ch22s02abm02.md#usb-mat-clinearelastic-isotropic)）或用于翘曲元素的正交线性弹性材料行为（参见["为翘曲元素定义正交弹性" "线性弹性行为，" 第22.2.1节](pt05ch22s02abm02.md#usb-mat-clinearelastic-orthowarp)）。或者，密度（["密度，" 第21.2.1节](pt05ch21s02abm01.md)）可以是指定的唯一材料属性，这对于模拟如油箱中的燃料等非结构质量很有用。

然后，通过在步定义中使用梁截面属性生成过程来分析模型。此横截面分析将数值计算截面的几何、刚度和惯性属性，包括翘曲函数和剪切中心（参见["网格化梁横截面，" Abaqus理论指南第3.5.6节](../stm/stm-link.md#stm-elm-meshedsections)），并将计算出的属性写入`jobname.bsp`文本文件。此文本文件的内容（可在后续Abaqus/Standard或Abaqus/Explicit梁分析中使用）在下面详细描述。

| **输入文件用法：** | 使用以下选项为网格化横截面生成梁截面属性： |
| --- | --- |
|  | ``` [*BEAM SECTION GENERATE](../key/key-link.md#usb-kws-hbeamsectiongen) ``` |

##### 定义横截面的原点

默认情况下，横截面的原点是用于定义网格的坐标系原点。您可以覆盖此默认值，直接输入原点的坐标，或指定原点与横截面的剪切中心或质心重合。当实际分析中使用的梁节点与二维坐标系原点不重合时，非默认原点特别有用。

| **输入文件用法：** | 使用以下两个选项直接输入原点的坐标： |
| --- | --- |
|  | ``` [*BEAM SECTION GENERATE](../key/key-link.md#usb-kws-hbeamsectiongen) [*SECTION ORIGIN](../key/key-link.md#usb-kws-hsectionorigin) ``` 使用以下两个选项将原点定位在质心或剪切中心： ``` [*BEAM SECTION GENERATE](../key/key-link.md#usb-kws-hbeamsectiongen) [*SECTION ORIGIN](../key/key-link.md#usb-kws-hsectionorigin), ORIGIN=CENTROID或SHEAR CENTER ``` |

##### 请求在特定积分点的输出

在分析过程中，可以在横截面的特定积分点上恢复对输出数据库的输出。请求大量横截面点上的输出可能会降低性能。

| **输入文件用法：** | 使用以下两个选项在特定积分点请求输出： |

|  | ``` [*BEAM SECTION GENERATE](../key/key-link.md#usb-kws-hbeamsectiongen) [*SECTION POINTS](../key/key-link.md#usb-kws-msectionpoints) ``` |

#### 从梁模型或二维横截面模型进行后处理

刻度轮廓图可用于可视化沿梁模型长度的应力和应变输出。二维横截面模型生成期间请求的所有应力和应变分量都将可用。也可以在二维横截面上使用应力和应变的轮廓图。截面几何形状从横截面分析生成的输出数据库中读取，而广义截面结果从梁分析生成的输出数据库中读取。

### 初始条件

生成梁截面属性时初始条件没有意义，将被忽略。

### 边界条件

生成梁截面属性时边界条件没有意义，将被忽略。

### 载荷

生成梁截面属性时载荷没有意义，将被忽略。

### 预定义场

温度和场变量不允许用于网格化截面。

### 材料选项

网格化截面仅允许以下材料行为：
- 各向同性线性弹性（["定义各向同性弹性" "线性弹性行为，" 第22.2.1节](pt05ch22s02abm02.md#usb-mat-clinearelastic-isotropic)）
- 用于翘曲元素的正交线性弹性（["为翘曲元素定义正交弹性" "线性弹性行为，" 第22.2.1节](pt05ch22s02abm02.md#usb-mat-clinearelastic-orthowarp)）
- 密度（["密度，" 第21.2.1节](pt05ch21s02abm01.md)）

### 单元

必须使用翘曲元素对二维横截面进行网格化。详细信息请参见["翘曲元素，" 第28.4.1节](pt06ch28s04alm04.md)。

### 输出

在实际梁分析期间，在属性生成分析中选择的网格化横截面上的积分点计算单元输出。属性生成分析的输出仅在输出数据库上可用。Abaqus/CAE的Visualization模块可用于在横截面上生成单元输出的轮廓图，这需要截面属性生成分析（横截面模型）和实际梁分析两者的输出数据库。更多信息，请参见["查看网格化梁横截面的分析" Abaqus脚本用户指南第9.10.10节](../cmd/cmd-link.md#cmd-odb-intro-exa-beamanalysis-pyc)中的示例Python脚本。
