# 20.11.5 创建构造圆







使用 Sketcher 工具箱中的构造圆工具帮助围绕圆定位和对齐对象。下图说明了构造圆和一对倾斜构造线如何用于定位两个圆的中心。（虚线表示构造几何。）

![](../graphics/ske-congeom-circle.png)

**草绘构造圆：**

1. 从 Sketcher 工具箱中的构造工具中，选择构造圆工具![](../graphics/ico_sketchConstCircle.png)。有关 Sketcher 工具箱中工具的图表，请参见["Sketcher 工具，"第 20.4.1 节](pt03ch20s04s01.md)。Abaqus/CAE 在提示区域显示提示以指导您完成该过程。
2. 点击构造圆中心的所需位置。或者，您可以在提示区域出现的文本字段中输入圆心的 *X-Y* 坐标。当您在 Sketcher 工作表上移动光标时，构造圆的半径会发生变化。
3. 点击将位于构造圆圆周上的点，或在提示区域中输入其 *X-Y* 坐标。Abaqus/CAE 使用您指定的点创建构造圆。
4. 要创建更多构造圆，重复从第 2 步开始的上述步骤。
5. 完成创建构造圆后，执行以下操作之一：- 在 Abaqus/CAE 窗口中的任意位置点击鼠标按钮 2。- 在 Sketcher 工具箱中选择任何其他工具。- 点击提示区域中的取消按钮![](../graphics/afxI_cancel.png)。
