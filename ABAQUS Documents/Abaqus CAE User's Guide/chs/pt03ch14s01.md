# 14.1 理解步骤模块的作用





你可以使用步骤模块执行以下任务：

**创建分析步骤**

在模型中，你可以定义一个或多个分析步骤的序列。步骤序列提供了一种方便的方式来捕获模型加载和边界条件的变化、模型各部分之间相互作用的变化、零件的移除或添加，以及在分析过程中可能发生的任何其他变化。此外，步骤允许你更改分析过程、数据输出和各种控制。你也可以使用步骤来定义关于非线性基础状态的线性扰动分析。你可以使用替换功能更改现有步骤的分析过程。

**指定输出请求**

Abaqus将分析输出写入输出数据库；你通过创建输出请求来指定输出，这些请求会被传播到后续分析步骤。输出请求定义了在分析步骤期间将输出哪些变量、从模型的哪个区域输出以及以什么速率输出。例如，你可能请求在步骤结束时输出整个模型的位移场，同时也请求约束点处的反作用力历史。

**指定自适应网格划分**

你可以定义自适应网格区域并为这些区域指定自适应网格划分控制。

**指定分析控制**

你可以自定义通用求解控制和解算器控制。
