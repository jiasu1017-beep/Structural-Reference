# 20.13 编辑尺寸







Abaqus/CAE 中的尺寸用于精确控制大小和距离、标注参考值，以及创建用于定义不同几何片段之间关系的参数。您可以按如下方式编辑草图中的任何尺寸：
- 更改尺寸值。更改尺寸值会修改关联的 Sketcher 几何。
- 将尺寸指定为参考尺寸。参考尺寸表示其值在同一草图中的其他位置或在其他草图中受控制的数量，或表示在另一个草图中受控制的参考几何的尺寸。
- 将尺寸与参数关联。参数通过定义不同几何片段（包括参考几何）之间的关系来控制 Sketcher 几何。

修改尺寸的效果取决于尺寸在草图中的使用方式。例如，如果半径具有相等半径约束，编辑半径尺寸会更改所有受约束的半径。同样，如果尺寸与参数关联，编辑其值会更改所有引用它的参数的值。有关约束、尺寸和参数的更多信息，请参见["控制草图几何，"第 20.7 节](pt03ch20s07.md)。

**编辑尺寸：**

1. 从 Sketcher 工具箱中的工具中，选择尺寸修改工具![](../graphics/ico_sketchDimensionEdit.png)。有关 Sketcher 工具箱中工具的图表，请参见["Sketcher 工具，"第 20.4.1 节](pt03ch20s04s01.md)。Abaqus/CAE 在提示区域显示提示以指导您完成该过程。
2. 选择要更改的尺寸。Abaqus/CAE 高亮显示所选尺寸并打开**编辑尺寸**对话框。
3. 输入尺寸的新值。**注意：**您无法编辑参考尺寸或使用方程定义其值的参数的值字段。
4. 切换**参考**可在控制草图几何的标准尺寸和从几何读取当前值的参考尺寸之间切换。
5. 点击 **f(x)** 将尺寸与新参数关联或编辑先前关联的参数。Abaqus/CAE 打开**参数管理器**进行编辑。有关更多信息，请参见["添加和编辑参数，"第 20.12.5 节](pt03ch20s12hlb05.md)。
6. 完成后，点击**编辑尺寸**对话框中的**应用**以更新草图。Abaqus/CAE 根据需要更新尺寸和几何。如果您的编辑导致过度约束，则冲突的约束、尺寸和参数会以洋红色显示（有关更多信息，请参见["完全约束几何，"第 20.7.4 节](pt03ch20s07s04.md)）。
7. 要编辑更多尺寸，请从第 2 步开始重复上述步骤。
8. 完成尺寸编辑后，执行以下操作之一：- 在 Abaqus/CAE 窗口中的任意位置点击鼠标按钮 2。- 在 Sketcher 工具箱中选择任何工具。- 点击提示区域中的取消按钮![](../graphics/afxI_cancel.png)。
