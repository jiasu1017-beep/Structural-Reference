# 15.13.20 定义基础







您可以通过定义选定表面每面积（或梁每长度）的弹性基础刚度来模拟弹性基础。从主菜单栏中选择****相互作用**![](../graphics/images/arrow.gif)**创建****，然后选择要建模为弹性基础的表面。有关弹性基础的简要概述，请参见"理解相互作用"，第15.3节。有关更详细的讨论，请参见Abaqus Analysis User's Guide的第2.2.2节"单元基础"。

弹性基础允许您模拟分布式支撑的刚度效应，而无需实际建模支撑的详细信息。您只能在初始步骤中创建弹性基础相互作用。一旦激活弹性基础，您无法在后续分析步骤中将其停用。

**定义基础：**

1. 从主菜单栏中，选择****相互作用**![](../graphics/images/arrow.gif)**创建****。**提示：**您也可以使用相互作用模块工具箱中的![](../graphics/ico_interaction.png)工具创建弹性基础相互作用。
2. 在出现的**创建相互作用**对话框中，执行以下操作：- 使用现有表面定义区域。在提示区域右侧单击**表面**。从出现的**区域选择**对话框中选择现有表面，然后单击**继续**。**注意：**默认选择方法基于您最近使用的选择方法。要恢复为另一种方法，请单击提示区域右侧的**在视口中选择**或**表面**。- 命名该相互作用。有关命名对象的信息，请参见第3.2.1节"使用基本对话框组件"。- 选择初始步骤。- 选择**弹性基础**类型的相互作用。
3. 单击**继续**关闭**创建相互作用**对话框。
4. 使用以下方法之一选择表面：- 使用鼠标在视口中选择区域。（更多信息，请参见"在当前视口中选择对象"，第6.2节。）如果模型包含网格和几何的组合，请从提示区域单击以下之一：- 如果要从几何区域选择表面，请单击**几何**。- 如果要从本机或孤立网格选择表面，请单击**网格**。您可以使用角度方法从几何或网格中选择一组面或边或一组单元面。更多信息，请参见"使用角度和特征边缘方法选择多个对象"，第6.2.3节。
5. 在提示区域出现的文本字段中，输入每面积的弹性基础刚度。Abaqus/CAE创建弹性基础相互作用。
