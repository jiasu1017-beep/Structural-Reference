# 34.3 载荷工况编辑器





进入Load模块并创建载荷和边界条件后，您可以从主菜单栏中选择****Load Case**![](../graphics/images/arrow.gif)**Create****来创建载荷工况。出现的**Create Load Case**对话框中您可以提供载荷工况的名称并选择将创建载荷工况的步骤。当您在**Create Load Case**对话框中单击**Continue**时，载荷工况编辑器出现。

载荷工况编辑器的顶部面板显示载荷工况的名称和当前分析步骤。载荷工况编辑器包含两个带表的标签页，允许您选择载荷和边界条件来定义载荷工况。默认情况下，Abaqus/CAE会在您创建的每个载荷工况中包含从基态传播或修改的所有边界条件。您可以在**Boundary Condition**标签页上关闭此行为。

单击每个标签页底部的![](../graphics/ico_add.png)可以从列表中选择载荷和边界条件。您可以输入缩放因子来缩放单个载荷和边界条件的大小。如果为载荷输入负缩放因子，载荷将沿相反方向施加。您也可以直接在表中输入载荷和边界条件的名称及其缩放因子。在载荷工况创建期间，Abaqus/CAE会按字母顺序对载荷和边界条件的名称进行排序，并在载荷工况编辑器中列出。您可以在视口中高亮显示所选载荷和边界条件。

例如，您可以定义一组载荷工况来分析承受端载荷和均匀压力载荷的悬臂梁的响应。悬臂梁模型如图34-2所示。

**图34-2** 具有端载荷、压力载荷和固定边界条件的悬臂梁模型。

![](../graphics/adv-lcmodel.png)

固定边界条件施加在初始步骤中，端载荷和压力载荷施加在静态、线性扰动步骤中。

您可以创建一个仅包含端载荷的载荷工况，另一个仅包含压力载荷的载荷工况。此外，您可以创建一个包含压力载荷和端载荷的载荷工况，端载荷沿相反方向施加（缩放因子值为1），如图34-3所示。

**图34-3** 包含悬臂梁模型载荷的载荷工况定义。

![](../graphics/adv-lceditor-load-nls.png)

默认情况下，每个载荷工况都包含从基态传播的固定边界条件，如图34-4所示。

**图34-4** 包含悬臂梁模型从基态传播的边界条件的载荷工况定义。
