# 29.107 Viscosity 对象

Viscosity 对象指定机械粘度。

**访问**

```
import material
mdb.models[*name*].materials[*name*].viscosity
import odbMaterial
session.odbs[*name*].materials[*name*].viscosity
```

### 29.107.1 Viscosity(...)

此方法创建 Viscosity 对象。

**路径**

```
mdb.models[*name*].materials[*name*].Viscosity
session.odbs[*name*].materials[*name*].Viscosity
```

**必需参数**

*table*

Float 元组序列，指定下述项目。

**可选参数**

*type*

SymbolicConstant，指定粘度类型。默认值为 NEWTONIAN。

*temperatureDependency*

Boolean，指定数据是否依赖于温度。默认值为 OFF。

*dependencies*

Int，指定场变量依赖项的数量。默认值为 0。

**表格数据**

如果 *type*=NEWTONIAN，表格数据指定以下内容：
- 粘度，![](../graphics/ker_eqn00143.gif)。
- 温度（如果数据依赖于温度）。
- 第一个场变量的值（如果数据依赖于场变量）。
- 第二个场变量的值。
- 以此类推。

**返回值**

Viscosity 对象。

**异常**

RangeError。

### 29.107.2 setValues(...)

此方法修改 Viscosity 对象。

**必需参数**

无。

**可选参数**

`setValues` 的可选参数与 [Viscosity](pt01ch29pyo107.md#ker-viscosity-viscosity-pyc) 方法的参数相同。

**返回值**

无

**异常**

RangeError。

### 29.107.3 成员

Viscosity 对象的成员与 [Viscosity](pt01ch29pyo107.md#ker-viscosity-viscosity-pyc) 方法的参数具有相同的名称和描述。

此外，Viscosity 对象可以具有以下成员：

*trs*

[Trs](pt01ch29pyo100.md) 对象。
