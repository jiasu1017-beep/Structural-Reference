# 38 子建模





您使用子建模来详细研究模型中您感兴趣的区域；例如，高应力区域。在大多数情况下，您将使用更细的网格为感兴趣的区域划分网格，子模型可以提供准确、详细的解决方案。您还可以将建模空间从壳全局模型更改为更具代表性的实体子模型——壳到实体子建模。

创建子模型是一个两步过程。首先创建并分析全局模型。然后创建子模型，并使用时变变量驱动子模型的边界，这些变量是在全局模型分析期间保存的。您可以使用边界条件或（在某些情况下）来自全局模型的应力来驱动子模型边界。子建模在["子建模：概述"，Abaqus Analysis User's Guide第10.2.1节](../usb/usb-link.md#usb-anl-asubmodeloverview)中详细描述。["壳到实体子建模和 pipe joint的壳到实体耦合"，Abaqus Example Problems Guide第1.1.10节](../exa/exa-link.md#exa-sta-shellsolidpipe)包括使用Abaqus/CAE创建的子模型示例。
