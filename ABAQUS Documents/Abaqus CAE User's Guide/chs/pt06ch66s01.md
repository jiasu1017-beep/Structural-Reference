# 66.1 过滤场和历史数据







您可以在 Abaqus/CAE 中创建过滤器，并将其应用于 Abaqus/Explicit 分析的场输出请求或历史输出请求。Abaqus 在分析运行时过滤数据；分析期间过滤（"实时"过滤）可以通过在保存之前排除高频数据来减小输出数据库的大小。实时过滤还可以避免结果数据中潜在的混叠问题。混叠是有效结果数据的丢失；如果采样频率（保存数据的频率）低于预期结果中最高频率的两倍，则会发生混叠。例如，如果正弦波仅在两个点采样，混叠结果将显示为直线，而至少在四个点采样将重现曲线的形状。

过滤器工具集允许您为 Abaqus/Explicit 场输出请求和历史输出请求创建以下过滤器：
- **Butterworth（巴特沃斯）**
- **Type I Chebyshev（切比雪夫 I 型）**
- **Type II Chebyshev（切比雪夫 II 型）**

有关这些过滤器的更具体信息，请参阅["输出到输出数据库"中的"Abaqus/Explicit 中的过滤输出和对输出进行操作"，Abaqus Analysis User's Guide 第 4.1.3 节](../usb/usb-link.md#usb-out-odboutput-filter)。不同过滤器类型的区别在于它们从接受低频数据到拒绝高于过滤器**截止频率**的数据的能力。理想过滤器会停止所有高于截止频率的数据，并具有平坦的响应幅度（不影响已接受的数据）；"真实"过滤器在截止频率周围包含一个过渡带，其中一些数据被接受，它们通常对已接受的数据有一定影响。Butterworth 过滤器提供最大平坦的响应幅度，过渡带比 Chebyshev 过滤器更宽（过渡更慢）。Chebyshev 过滤器在响应幅度中引入振荡——纹波，但它们的过渡带比 Butterworth 过滤器更窄。两种 Chebyshev 过滤器的区别在于其响应纹波发生的位置；**纹波因子**指示您允许多少振荡以换取改善的过滤器响应。您还可以指定过滤器的**阶数**，它决定过滤器过渡带的大小：阶数越高，过渡带越窄，尽管计算成本随阶数增加而增加。过滤器阶数必须是小于或等于 20 的正偶数。

除了可以使用过滤器工具集定义的过滤器外，Abaqus 还包含一个默认的**抗混叠**过滤器。Abaqus 根据分析期间保存场输出或历史输出的时间间隔自动设置**抗混叠**过滤器的截止频率。当您定义过滤器时，必须根据您对解决方案中预期频率的了解指定截止频率。无论您设置截止频率还是由 Abaqus 计算它，都不会执行任何检查以确保截止频率是合适的。如果截止频率设置过低，有效数据将从结果中被过滤掉；如果设置过高（高于采样频率的一半），将不执行任何过滤。

从主菜单中选择 ****Tools**![](../graphics/images/arrow.gif)**Filter**![](../graphics/images/arrow.gif)**Create****（工具→过滤器→创建）以创建新的过滤器定义；从同一菜单中选择 **Edit**（编辑）以更改现有定义。任一命令都会打开过滤器编辑器，允许您选择选项并提供定义过滤器所需的数据。

要将过滤器应用于分析，请将其包含在 Abaqus/Explicit 分析过程的场输出请求或历史输出请求中（有关更多信息，请参阅["定义输出请求，" 第 14.12 节"](pt03ch14s12.md)。
