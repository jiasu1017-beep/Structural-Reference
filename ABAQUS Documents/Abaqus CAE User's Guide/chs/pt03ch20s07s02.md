# 20.7.2 使用尺寸控制草图几何







尺寸是一种约束，使用数值来定义草图中的大小、角度或距离。尺寸通过防止尺寸量发生变化来控制草图几何。要更改尺寸量，必须修改关联的尺寸。

Sketcher 中提供以下尺寸类型：
- 水平
- 垂直
- 斜向
- 角度
- 半径

前三种尺寸类型对应于两个点（包括单条线的端点）、两条线或一个点与一条线之间的线性距离。角度尺寸始终表示两条线之间的角度（以度为单位）。半径尺寸表示圆、圆弧或圆角的半径，或椭圆的长半径和短半径。

您可以使用以下技术向草图添加尺寸：

**自动添加尺寸**

创建草图后，您可以使用自动尺寸工具![](../graphics/ico_sketchDimensionAuto.png)向整个草图或选定的实体组添加尺寸。使用草图选项控制 Abaqus/CAE 可以自动添加的尺寸类型。有关更多信息，请参见["在 Sketcher 中自定义尺寸的格式和使用，"第 20.9.9 节](pt03ch20s09hlb09.md)。

**添加单个尺寸**

创建草图几何后，您可以添加单个尺寸。这种方法给您最大的控制权，但也最耗时。有关更多信息，请参见["添加单个尺寸，"第 20.12.4 节](pt03ch20s12hlb04.md)。

您可以修改草图中的任何尺寸。编辑现有尺寸时，可以通过多种方式修改它。您可以
- 编辑数值来修改草图，
- 将尺寸转换为参考尺寸，或
- 将尺寸链接到参数。

参考尺寸标注已在草图其他位置控制的量，或投影参考几何等无法在当前草图中控制的量。参考尺寸值括在括号中，Abaqus/CAE 会自动更新它们以匹配关联几何的变化。链接到参数的尺寸被分配一个变量名，用于创建数学表达式（参数方程）来定义草图中的其他参数。有关编辑尺寸的详细说明，请参见["编辑尺寸，"第 20.13 节](pt03ch20hla01.md)。

添加或修改尺寸时，不得使草图几何过度约束。当多个控件应用于单个自由度时，几何是过度约束的（有关更多信息，请参见["完全约束几何，"第 20.7.4 节](pt03ch20s07s04.md)）。Abaqus/CAE 将冲突约束显示为洋红色以指示过度约束条件。在使用草图创建特征或将其与模型数据库一起保存之前，必须解决任何过度约束问题。
