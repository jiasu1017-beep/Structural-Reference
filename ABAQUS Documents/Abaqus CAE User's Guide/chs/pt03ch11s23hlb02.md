# 11.23.2 添加点对点线框特征





您可以向零件或装配添加点对点线框特征。

装配级线框特征不可网格划分。线框特征包含从当前视口中的零件或装配连接点的线框，或从零件或装配连接到地面的线框。

要添加点对点线框特征，请从零件模块的主菜单栏选择**形状**![](../graphics/images/arrow.gif)**线框**![](../graphics/images/arrow.gif)**点对点**。

您通过选择几何类型（多段线或样条线）来向零件添加点对点线框特征。您可以选择通过创建边缘来刻印线框、将线框与现有零件合并，或创建与现有零件分开的线框。

**添加点对点线框特征：**

1. 显示**创建线框特征**对话框。
2. 如果在零件模块中创建线框，选择**多段线**创建一条或多条直线，或选择**样条线**创建连续样条曲线。
3. 如果要向零件添加线框特征，请在对话框的**线框合并方案**部分指定合并选项。
4. 在对话框的**点对**部分，指定点选择方法。
5. 单击添加点。
6. 完成选择点后，单击**完成**。
7. 从**点对**表中，您可以执行以下操作。
8. 在对话框的**集合创建**部分，执行以下操作。
9. 单击**确定**创建点对点线框特征。
