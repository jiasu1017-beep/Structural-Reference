# 17.19 Obtaining mesh information and statistics







This section explains how you use the verification tool in the Mesh module to verify graphically the quality of the elements used in a mesh. This section also describes how you use the Query toolset in the Mesh module to obtain listings of mesh, element, and node statistics. The following topics are covered:

- ["Obtaining mesh information," Section 17.19.2](pt03ch17s19hlb02.md)
