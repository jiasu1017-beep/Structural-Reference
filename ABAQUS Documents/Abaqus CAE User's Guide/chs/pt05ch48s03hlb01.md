# 48.3.2 控制数据对 X 值





### 48.3.2 控制数据对 X 值

Abaqus 以 X–Y 数据对的形式获取沿路径的结果，根据您为路径指定的点以及路径与模型相交的其他附加点来计算数据对 X 值。有关更多信息，请参阅 ["选择在获取数据的位置的路径，" 第 48.3.1 节"](pt05ch48s03hlb01.md)。

构成路径的节点标签或点坐标通常不适合直接用作 X 值。Abaqus 提供了多个选项供您将此点系列转换为有用的 X 值和后续 X–Y 绘图轴标签。

**控制数据对 X 值：**

1. 找到 **沿路径的 XY 数据（XY Data from Path）** 选项。从主菜单栏中，选择 ****工具（Tools）**![](../graphics/images/arrow.gif)**XY 数据（XY Data）**![](../graphics/images/arrow.gif)**创建（Create）****。从出现的对话框中，选择 **路径（Path）**；然后点击 **继续（Continue）**。出现 **沿路径的 XY 数据（XY Data from Path）** 对话框。
2. 在对话框顶部，点击 **路径（Path）** 箭头选择要获取数据的路径。当前视口中的模型绘图会更改以高亮显示您选择的路径。
3. 在 **X 值（X Values）** 字段中，选择以下选项之一将路径点转换为 **X 值**：- **真实距离（True distance）**：X 值对应于每个点沿路径在模型空间坐标中的实际距离，从零开始。- **归一化距离（Normalized distance）**：X 值对应于每个点沿路径的距离，作为路径总长度的一部分。- **序列 ID（Sequence ID）**：X 值对应于每个点在路径结果列表中出现的顺序。- **X、Y 或 Z 距离（X, Y, or Z distance）**：X 值对应于每个点沿路径在您指定的单个坐标方向上的实际距离，从零开始。此选项对于生成轴对称模型中结果与半径的关系图特别有用。
4. 要评估和显示数据，请点击 **绘图（Plot）**。X–Y 绘图出现在当前视口中。该绘图表示您在对话框中配置的数据，无论您是否点击 **另存为（Save As）** 来保存它，Abaqus 都将其视为临时数据。
5. 要将您配置的 X–Y 数据对保存为 X–Y 数据对象，请点击 **另存为（Save As）**。**注意：**要绘制您保存的 X–Y 数据对象，请从主菜单栏中选择 ****工具（Tools）**![](../graphics/images/arrow.gif)**XY 数据（XY Data）**![](../graphics/images/arrow.gif)**绘图（Plot）****，并从弹出菜单中选择 X–Y 数据对象。
6. 完成后，点击 **取消（Cancel）** 关闭对话框。

# 48.3.1 选择在获取数据的位置的路径





Abaqus/CAE 以 X–Y 数据对的形式获取沿路径的结果。数据对 X 值定义获取数据的模型位置；Y 值是那些位置的分析结果。Abaqus 在计算数据对时仅考虑当前显示组中的实体。

Abaqus 根据您为路径指定的点来计算数据对 X 值。您可以选择仅根据您指定的点形成 X 值，或者额外包括路径与模型相交的所有位置。（相交发生在路径穿过单元面、单元边、表面面或表面边的地方。）

您还可以选择 Abaqus 是否将构成路径的点解释为未变形或变形模型形状上的位置。对于节点列表路径，构成路径的节点标签同样适用于未变形和变形的模型形状。但是，点列表路径坐标在空间中是固定的，与模型无关。例如，与未变形形状上节点位置重合的坐标可能与变形形状上的任何位置都不重合。使用点列表路径坐标和变形模型获取的值受变形比例因子的影响。对于与模型不重合的点，Abaqus 不会形成数据对。

**选择在获取数据的位置的路径：**

1. 找到 **沿路径的 XY 数据（XY Data from Path）** 选项。从主菜单栏中，选择 ****工具（Tools）**![](../graphics/images/arrow.gif)**XY 数据（XY Data）**![](../graphics/images/arrow.gif)**创建（Create）****。从出现的对话框中，选择 **路径（Path）**；然后点击 **继续（Continue）**。出现 **沿路径的 XY 数据（XY Data from Path）** 对话框。
2. 点击 **路径（Path）** 箭头选择要获取数据的路径。当前视口中的模型绘图会更改以高亮显示您选择的路径。
3. 点击 **未变形（Undeformed）** 或 **变形（Deformed）** 来选择 Abaqus 是将构成路径的点解释为未变形还是变形模型形状上的位置。
4. 从 **点位置（Point Locations）** 选项中，选择以下之一：- 选择 **路径点（Path points）** 仅在构成路径的点处获取数据。您可以通过切换 **包括相交点（Include intersections）** 来在路径与模型相交的位置以及构成路径的点处获取 X–Y 数据。- 选择 **均匀间距（Uniform spacing）** 在沿路径的规则插值间隔处获取数据，但不一定在路径定义包含的点处。您可以更改 **间隔数（Number of intervals）** 字段中指定的值来增加或减少沿路径获取插值数据的位置数。
5. 要评估和显示数据，请点击 **绘图（Plot）**。X–Y 绘图出现在当前视口中。该绘图表示您在对话框中配置的数据，无论您是否点击 **另存为（Save As）** 来保存它，Abaqus 都将其视为临时数据。
6. 要保存您配置的数据，请点击 **另存为（Save As）**。**注意：**要绘制您保存的 X–Y 数据对象，请从主菜单栏中选择 ****工具（Tools）**![](../graphics/images/arrow.gif)**XY 数据（XY Data）**![](../graphics/images/arrow.gif)**绘图（Plot）****，并从弹出菜单中选择 X–Y 数据对象。
7. 完成后，点击 **取消（Cancel）** 关闭对话框。

# 49.1 理解动画





Abaqus/CAE 提供两种动画方式：

**基于对象的动画**

基于对象的动画是变形形状绘图、云图、符号绘图或材料取向绘图的序列显示。Abaqus/CAE 可以生成三种不同类型的绘图序列。这三种序列类型称为"时间历史动画"、"比例因子动画"和"谐波动画"。

时间历史动画根据实际分析结果生成随时间变化的变形形状、云图、符号或材料取向绘图序列。比例因子动画生成仅在单个变形形状、云图或符号绘图的缩放方面变化的图像序列。谐波动画生成表示根据施加角度变化的复数结果值的图像序列。为了更好地理解这三种类型的动画，请分别参阅 ["时间历史动画，" 第 49.1.1 节"](pt05ch49s01s01.md)；["比例因子动画，" 第 49.1.2 节"](pt05ch49s01s02.md)；和 ["谐波动画，" 第 49.1.3 节"](pt05ch49s01s03.md)。

当基于对象的动画播放时，您可以动态更改显示特征，如视图、任何视口注释以及绘图状态相关的自定义选项。

**基于图像的动画**

基于图像的动画是动画文件的回放。您可以通过首先在一个或多个视口中播放基于对象的动画，然后从主菜单栏中选择 ****动画（Animate）**![](../graphics/images/arrow.gif)**另存为（Save As）**** 在 Abaqus/CAE 中创建动画文件。保存后，您可以使用行业标准动画软件在 Abaqus/CAE 外部播放您的动画。您可以选择以 QuickTime 或 Audio Video Interleave（AVI）格式保存基于图像的动画文件。

在 Abaqus/CAE 内部，您还可以通过将动画文件选择为背景电影来显示基于图像的动画。当激活时，背景电影会在可视化模块的视口中显示。有关更多信息，请参阅 ["显示和自定义背景电影，" 第 4.6.3 节"](pt01ch04s06hlb02.md)。

**注意：**Abaqus/CAE 还使您能够以 VRML 格式保存动画，这会创建动画的三维渲染。由于这些文件是三维的，它们不是严格的基于图像的；但是，您可以像播放和分发 QuickTime 或 AVI 格式的文件一样播放和分发 VRML 格式的动画文件。

通常，从文件回放动画比基于对象的动画提供更好的性能，尤其是对于大型模型。当基于图像的动画播放时，您无法更改其显示特征。
