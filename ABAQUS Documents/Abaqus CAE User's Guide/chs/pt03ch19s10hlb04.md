# 19.10.4 控制自适应过程内存设置







要找到自适应过程编辑器，请从主菜单栏中选择****自适应**![](../graphics/images/arrow.gif)**编辑**![](../graphics/images/arrow.gif)***自适应过程名称*****。使用**内存**选项卡页面控制分配给 Abaqus 分析的内存量。Abaqus/CAE 最初使用 Abaqus 环境文件（`abaqus_v6.env`）中指定的值填充**内存**选项卡页面中的设置，如果您想更改默认内存设置，可以编辑环境文件。有关自定义内存设置的更多信息，请参见["使用 Abaqus 环境设置，"《Abaqus 分析用户手册》第 3.3.1 节](../usb/usb-link.md#usb-int-denvfile)。

如果 Abaqus 环境文件中未指定内存值，Abaqus 会自动检测机器上的物理内存并分配此可用内存的百分比。默认百分比是平台特定的，但它们通常代表可用物理内存的很大一部分。有关默认内存分配设置的详细信息，请参阅位于 [www.3ds.com/support/knowledge-base](http://www.3ds.com/support/knowledge-base) 的达索系统知识库。

较小的作业可能使用的内存少于最大可用量。如果您通常同时运行多个作业，您可能希望减少作业的内存分配或调整默认分配。您可以控制以下内容：
- 内存大小规格的**内存分配单位**。您可以将内存分配为**物理内存的百分比**或以**兆字节**或**千兆字节**为单位。
- **最大预处理器和分析内存**值，用于指定将分配给分析的最大物理内存的百分比或兆字节或千兆字节数。

有关内存设置的更多信息，请参见["管理 Abaqus 中的内存和磁盘使用，"《Abaqus 分析用户手册》第 3.4.1 节](../usb/usb-link.md#usb-int-dmemory)。
