# 47.4.16 Linearizing an X–Y data object







### 47.4.16 Linearizing an X--Y data object

Use the **linearize** function to separate the *Y*-values of a [previously saved](pt05ch47s02hlb07.md) *X–Y* data object (a collection of ordered pairs) into constant or linear components. The new *X–Y* data object produced by the **linearize** function contains exactly two points.

[Figure 47--14](pt05ch47s03hlb14.md#usv-xyplot-linearize) shows an example of an *X–Y* plot produced using the **linearize** function.

**Figure 47–14** *X–Y* plot produced using both the constant and linear forms of the **linearize** function.

![](../graphics/usv-xyplot-linearize.png)

**Note:** The *X*-values of the *X–Y* data object that you input to the **linearize** function must be equally spaced.

**To apply linearization to an X–Y data object:**

1. Locate the **Operate on XY Data** dialog box. From the main menu bar, select ****Tools**![](../graphics/images/arrow.gif)**XY Data**![](../graphics/images/arrow.gif)**Create****. Click **Operate on XY data** in the dialog box that appears; then click **Continue**. The **Operate on XY Data** dialog box appears.
2. From the **Operators** listed, click either **linearize(X, CONSTANT)** or **linearize(X, LINEAR)** to apply linearization to create a constant Y term or to linearize the Y term, respectively. The **linearize** function appears in the expression window.
3. From the **XY Data** choices, click the name of the *X--Y* data object on which to operate and click **Add to Expression**. You can choose from all *X--Y* data objects previously saved within this session (listed alphabetically in the **XY Data** field). The *X--Y* data object name appears within the **linearize** function parentheses in the expression window.
4. To continue to build your expression, position the cursor in the expression window and type in or select the functions, operators, and *X--Y* data you want to include.
5. To evaluate and display your expression, click **Plot Expression**.
6. To [save](pt05ch47s02hlb07.md) your new *X--Y* data object, click **Save As** and then provide a name in the dialog box that appears. Saving your data object makes it available for future operations within this session and for inclusion in *X--Y* plots containing multiple data objects.
7. When you are finished, click **Cancel** to close the dialog box.
