# FXTabBook





选项卡簿布局管理器排列成对的子窗口；偶数编号的子窗口（0,2,4,...）通常是选项卡项，位于顶部。奇数编号的子窗口通常是布局管理器，位于下方；所有奇数编号的子窗口彼此堆叠在一起，类似于 switcher 组件。当用户按下其中一个选项卡项时，该选项卡项会升高到相邻选项卡的上方，相应的面板也会升到顶部。因此，选项卡簿可以通过将多个面板堆叠在一起并使用选项卡项选择所需面板来在有限的空间中呈现许多 GUI 控件。
