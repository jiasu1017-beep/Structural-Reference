# 44.1.2 理解云图的渲染方式





云图值使用两种方法之一绘制在单元面上：纹理映射或镶嵌。纹理映射是一种高性能渲染方法，本质上是将云图值（纹理）的图像覆盖在模型图像上，类似于包装礼品的过程。镶嵌是一种将任意云图值转换为重复图案（如三角形或简单多边形）的方法；形状值逐面计算，对于大型模型可能需要很长时间。纹理映射是默认和首选方法，可以最大化Abaqus/CAE的性能。尽管某些图形适配器不能正确支持纹理映射，但Abaqus/CAE可以在软件中模拟纹理映射。但是，线型和等值面云图、云图边缘的显示、关于质心收缩单元以及CAXA或SAXA单元上的云图仅受镶嵌方法支持。

使用纹理映射或镶嵌生成的云图之间的差异很小；您可能会在四边形单元面上的云图带外观上看到轻微差异。纹理映射有时会引入精度问题；例如，如果云图值恰好在限制值上，颜色可能会显示为刚好高于或低于该值。

**注意：**如果在强光下显示镶嵌云图，云图中的颜色可能与图例中显示的颜色不匹配，因为光线会使表面小平面上的某些颜色变淡。有关更多信息，请参见["控制模型照明，" 第76.13节](pt07ch76hla12.md)。

有关选择云图方法的详细信息，请参见["选择云图方法，" 第44.5.6节](pt05ch44s02hlb06.md)。
