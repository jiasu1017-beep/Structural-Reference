# 64.4 An overview of the mesh editing tools







This section is an overview of the tools that are available in the Edit Mesh toolset. You can access the tools from the main menu bar by selecting ****Mesh**![](../graphics/images/arrow.gif)**Edit****. Abaqus/CAE displays the **Edit Mesh** dialog box. You can also display the **Edit Mesh** dialog box using the ![](../graphics/ico_meshEdit.png) tool, located at the bottom of the Mesh module toolbox.

**Node editing tools**

The following tools are available for editing nodes:
- Create node. For more information, see ["Creating a node," Section 64.5.1](pt06ch64s05s01.md).
- Edit node. For more information, see ["Editing the position of selected nodes," Section 64.5.2](pt06ch64s05s02.md).
- Drag node. For more information, see ["Dragging nodes," Section 64.5.3](pt06ch64s05s03.md).
- Project nodes. For more information, see ["Projecting nodes," Section 64.5.4](pt06ch64s05s04.md).
- Delete node. For more information, see ["Deleting nodes," Section 64.5.5](pt06ch64s05s05.md).
- Merge nodes. For more information, see ["Merging nodes," Section 64.5.6](pt06ch64s05s06.md).
- Adjust midside nodes. For more information, see ["Adjusting the position of midside nodes," Section 64.5.7](pt06ch64s05s07.md).
- Smooth external nodes. For more information, see ["Smoothing external nodes," Section 64.5.8](pt06ch64s05s08.md).
- Renumber nodes. For more information, see ["Renumbering nodes," Section 64.5.9](pt06ch64s05s09.md).

**Element editing tools**

The following tools are available for editing elements:
- Create element. For more information, see ["Creating an element," Section 64.6.1](pt06ch64s06s01.md).
- Delete element. For more information, see ["Deleting elements," Section 64.6.2](pt06ch64s06s02.md).
- Flip normal. For more information, see ["Reversing the surface normal direction of shell elements," Section 64.6.3](pt06ch64s06s03.md).
- Orient stack direction. For more information, see ["Orienting the stack direction," Section 64.6.4](pt06ch64s06s04.md).
- Collapse edge. For more information, see ["Collapsing an edge of a quadrilateral or triangular element," Section 64.6.5](pt06ch64s06s05.md).
- Split edge. For more information, see ["Splitting an edge of a quadrilateral or triangular element," Section 64.6.6](pt06ch64s06s06.md).
- Swap diagonal. For more information, see ["Swapping the diagonal of a pair of adjacent triangular elements," Section 64.6.7](pt06ch64s06s07.md).
- Split. For more information, see ["Splitting quadrilateral elements," Section 64.6.8](pt06ch64s06s08.md).
- Combine. For more information, see ["Combining triangular elements," Section 64.6.9](pt06ch64s06s09.md).
- Renumber elements. For more information, see ["Renumbering elements," Section 64.6.10](pt06ch64s06s10.md).

**Mesh editing tools**

The following tools are available for editing the entire mesh:
- Offset solid. For more information, see ["Generating layers of solid elements offset from an existing mesh," Section 64.7.1](pt06ch64s07s01.md).
- Offset shell. For more information, see ["Generating layers of shell elements offset from an existing mesh," Section 64.7.2](pt06ch64s07s02.md).
- Wrap. For more information, see ["Wrapping a mesh about the Z-axis," Section 64.7.3](pt06ch64s07s03.md).
- Collapse. For more information, see ["Collapsing short edges of a linear orphan mesh," Section 64.7.4](pt06ch64s07s04.md).
- Grow. For more information, see ["Growing short edges of a linear orphan mesh," Section 64.7.5](pt06ch64s07s05.md).
- Convert. For more information, see ["Converting a triangular shell mesh to a solid tetrahedral mesh," Section 64.7.6](pt06ch64s07s06.md), and ["Converting a solid orphan mesh to a shell orphan mesh," Section 64.7.7](pt06ch64s07s07.md).
- Merge. For more information, see ["Merging elements," Section 64.7.8](pt06ch64s07s08.md).
- Subdivide. For more information, see ["Subdividing elements," Section 64.7.9](pt06ch64s07s09.md).
- Copy mesh pattern. For more information, see ["Copying a mesh pattern," Section 64.7.10](pt06ch64s07s10.md).
- View and edit mesh-geometry associativity. For more information, see ["Viewing and editing mesh-geometry associativity," Section 64.7.11](pt06ch64s07s11.md).
- Delete mesh-geometry associativity. For more information, see ["Deleting mesh-geometry associativity," Section 64.7.12](pt06ch64s07s12.md).

**Mesh refinement tools**

The following tools are available for refining the mesh:
- Set size.
- Remove size.
- Remesh.

For more information, see ["Refining a planar, triangular mesh," Section 64.8.1](pt06ch64s08s01.md).

**Undo and redo for mesh edits**

You can undo and redo multiple changes that you make using the tools in the Edit Mesh toolset, and control the number of available undo levels. For more information, see ["Undoing or redoing a change in the Edit Mesh toolset," Section 64.9](pt06ch64s09.md).
