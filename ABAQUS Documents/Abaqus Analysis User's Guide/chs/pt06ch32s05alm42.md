# 32.5.4 定义内聚单元的初始几何







**产品：** Abaqus/Standard  Abaqus/Explicit  Abaqus/CAE  

##### **参考资料**

- ["内聚单元：概述，" 第32.5.1节](pt06ch32s05abo29.md)
- [Abaqus/CAE用户指南第21章"粘接接头和粘合界面"](../usi/usi-link.md#usi-adv-cohesive)

### 概述

内聚单元的初始几何由以下方式定义：
- 由单元的节点连通性和这些节点的位置定义；
- 由堆叠方向定义，可用于独立于节点连通性指定内聚单元的顶面和底面；和
- 由初始本构厚度的幅度定义，该厚度可以与节点位置和堆叠方向所暗示的几何厚度相对应，也可以直接指定。

### 定义单元连通性

内聚单元的连通性与连续体单元的连通性类似；然而，将内聚单元视为由两个面（底面和顶面）组成会更实用，这两个面被内聚区厚度分开。单元在其底面上有节点，在其顶面上有对应的节点。孔隙压力内聚单元包括第三个中间面，用于模拟单元内部的流体流动。

有三种方法可用于定义单元连通性。

#### 通过直接定义单元的完整连通性

内聚单元的完整连通性可以直接给出（见["单元定义"中的"定义内聚单元"，第2.2.1节](pt01ch02s02aus11.md#usb-int-ielement-cohesive)）。

#### 通过定义底面单元连通性和整数偏移量

或者，您可以指定底面的连通性加上一个正整数偏移量（见["单元定义"中的"定义内聚单元"，第2.2.1节](pt01ch02s02aus11.md#usb-int-ielement-cohesive)），该偏移量将用于确定其余的内聚单元节点。

| **输入文件用法：** | ``` [*ELEMENT](../key/key-link.md#usb-kws-melement), OFFSET=*n* ``` |
| --- | --- |

| **Abaqus/CAE用法：** | Abaqus/CAE不支持单元偏移。 |
| --- | --- |

##### 与位移内聚单元一起使用

整数偏移量将用于定义内聚单元顶面的节点编号。Abaqus会自动将顶面节点定位为与底面节点重合，除非顶面节点已通过节点定义直接分配了坐标（["节点定义，" 第2.1.1节](pt01ch02s01aus05.md)）。

##### 与孔隙压力-位移内聚单元一起使用

当您仅定义底面节点时，整数偏移量将首先用于定义内聚单元顶面的节点编号，顶面节点编号相对于底面节点编号偏移。整数偏移量将再次用于定义中间面节点编号偏移，中间面节点编号相对于顶面节点编号偏移。Abaqus会自动将顶面和中间面的节点定位为与底面节点重合，除非顶面节点已通过节点定义直接分配了坐标（["节点定义，" 第2.1.1节](pt01ch02s01aus05.md)）。

#### 通过定义底面和顶面单元连通性以及整数偏移量

对于孔隙压力内聚单元，您还可以指定底面和顶面的连通性加上一个正整数偏移量（见["单元定义"中的"定义内聚单元"，第2.2.1节](pt01ch02s02aus11.md#usb-int-ielement-cohesive)），该偏移量将用于确定中间面的内聚单元节点。

当您定义底面和顶面节点时，整数偏移量将用于定义中间面的节点编号，中间面节点编号相对于底面节点编号偏移。Abaqus会自动将中间面的节点定位为位于底面和顶面节点之间的一半位置，除非中间面节点已通过节点定义直接分配了坐标（["节点定义，" 第2.1.1节](pt01ch02s01aus05.md)）。

| **输入文件用法：** | ``` [*ELEMENT](../key/key-link.md#usb-kws-melement), OFFSET=*n* ``` |
| --- | --- |

| **Abaqus/CAE用法：** | Abaqus/CAE不支持单元偏移。 |
| --- | --- |

### 为二维单元指定面外厚度

对于二维内聚单元，需要面外厚度。您在内聚截面定义中指定此附加信息；默认值为1.0。

| **输入文件用法：** | ``` [*COHESIVE SECTION](../key/key-link.md#usb-kws-mcohesivesection) *第一数据行* *面外厚度* ``` |
| --- | --- |

| **Abaqus/CAE用法：** | 属性模块：内聚截面编辑器：切换**Out-of-plane thickness:**并指定面外厚度 |
| --- | --- |

### 指定本构厚度

您可以直接指定内聚单元的本构厚度，或允许Abaqus根据节点坐标计算它，使本构厚度等于几何厚度。默认行为取决于应用程序的性质。

如果内聚单元的几何厚度相对于其表面尺寸非常小，则从节点坐标计算的厚度可能不准确。在这种情况下，您可以在定义这些单元的截面属性时直接指定恒定厚度。

内聚单元的特征单元长度等于其本构厚度。特征单元长度通常在定义材料的损伤演化时很有用（见["渐进损伤和失效，" 第24.1.1节中的"网格依赖性"](pt05ch24s01abo21.md#usb-mat-cdamageoverview-meshdep)）。

#### 当内聚单元响应基于连续体方法时

当内聚单元的响应基于连续体方法时，默认情况下，本构厚度由Abaqus根据节点坐标计算。您可以通过直接指定本构厚度来覆盖此默认值。

| **输入文件用法：** | 使用以下选项让Abaqus根据节点坐标计算厚度： |
| --- | --- |
|  | ``` [*COHESIVE SECTION](../key/key-link.md#usb-kws-mcohesivesection), RESPONSE=CONTINUUM, THICKNESS=GEOMETRY (默认) ``` 使用以下选项直接指定厚度： ``` [*COHESIVE SECTION](../key/key-link.md#usb-kws-mcohesivesection), RESPONSE=CONTINUUM, THICKNESS=SPECIFIED *厚度（默认为1.0）* ``` |

| **Abaqus/CAE用法：** | 属性模块：内聚截面编辑器：**Response**：**Continuum**：**Initial thickness**：**Use nodal coordinates**、**Specify**：*厚度*或**Use analysis default** |
| --- | --- |

#### 当内聚单元响应基于牵引-分离方法时

当内聚单元的响应基于牵引-分离方法时，Abaqus默认假定本构厚度等于1。此默认值的动机是，对于牵引-分离本构响应适用的应用类型，内聚单元的几何厚度通常等于（或非常接近）零。此默认选择确保名义应变等于相对分离位移（见["使用牵引-分离描述定义内聚单元的本构响应，" 第32.5.6节](pt06ch32s05alm45.md)，了解更多详细信息）。您可以通过指定另一个值或指定本构厚度应等于几何厚度来覆盖此默认值。

| **输入文件用法：** | 使用以下选项直接指定厚度： |
| --- | --- |
|  | ``` [*COHESIVE SECTION](../key/key-link.md#usb-kws-mcohesivesection), RESPONSE=TRACTION SEPARATION, THICKNESS=SPECIFIED (默认) *厚度（默认为1.0）* ``` 使用以下选项让Abaqus根据节点坐标计算厚度： ``` [*COHESIVE SECTION](../key/key-link.md#usb-kws-mcohesivesection), RESPONSE=TRACTION SEPARATION, THICKNESS=GEOMETRY ``` |

| **Abaqus/CAE用法：** | 属性模块：内聚截面编辑器：**Response**：**Traction Separation**：**Initial thickness**：**Specify**：*厚度*、**Use analysis default**或**Use nodal coordinates** |
| --- | --- |

#### 当内聚单元响应基于单轴应力状态时

当内聚单元的响应基于单轴应力状态时，没有计算本构厚度的默认方法。您必须指明确定本构厚度的方法。

| **输入文件用法：** | 使用以下选项指定厚度： |
| --- | --- |
|  | ``` [*COHESIVE SECTION](../key/key-link.md#usb-kws-mcohesivesection), RESPONSE=GASKET, THICKNESS=SPECIFIED *厚度（默认为1.0）* ``` 使用以下选项让Abaqus根据节点坐标计算厚度： ``` [*COHESIVE SECTION](../key/key-link.md#usb-kws-mcohesivesection), RESPONSE=GASKET, THICKNESS=GEOMETRY ``` |

| **Abaqus/CAE用法：** | 属性模块：内聚截面编辑器：**Response**：**Gasket**：**Initial thickness**：**Specify**：*厚度*或**Use nodal coordinates** |
| --- | --- |

#### 将堆叠方向设置为等参方向

"堆叠方向"指的是内聚单元的顶面和底面堆叠所沿的等参方向。默认情况下，对于三维内聚单元，顶面和底面沿第三个等参方向堆叠；对于二维和轴对称内聚单元，沿第二个等参方向堆叠。您可以选择沿替代等参方向堆叠顶面和底面（COH3D6单元只能以第三个等参方向作为堆叠方向）。等参方向的选择取决于单元的连通性。对于网格无关的规格，请使用下面描述的基于方向的方法。三维内聚单元的等参方向选择如图32.5.4--3所示。

**图32.5.4-3** COH3D8（左）和COH3D6（右）单元的堆叠方向。

![](../graphics/eshell-scon-stackdir-nls.png)

| **输入文件用法：** | 使用以下选项基于单元的等参方向定义单元顶面和底面： |
| --- | --- |
|  | ``` [*COHESIVE SECTION](../key/key-link.md#usb-kws-mcohesivesection), STACK DIRECTION=*n* ``` |

| **Abaqus/CAE用法：** | 您不能在Abaqus/CAE中基于等参方向定义堆叠方向。堆叠方向将对应于上面讨论的默认值。 |
| --- | --- |

#### 基于用户定义方向设置堆叠方向

您还可以通过用户定义的局部方向（["方向，" 第2.2.5节](pt01ch02s02aus15.md)）来控制堆叠方向。当您为内聚单元定义方向时，您还指定一个轴，局部1和2材料方向可绕该轴旋转。该轴还定义了一个近似法向方向。堆叠方向将是与该近似法向最接近的单元等参方向（见图32.5.4--4）。

**图32.5.4-4** 说明使用圆柱系统定义堆叠方向的示例。

![](../graphics/ecohesive-stackori-nls.png)

| **输入文件用法：** | 使用以下选项基于用户定义的方向定义单元厚度方向： |
| --- | --- |
|  | ``` [*COHESIVE SECTION](../key/key-link.md#usb-kws-mcohesivesection), STACK DIRECTION=ORIENTATION, ORIENTATION=*name* ``` |

| **Abaqus/CAE用法：** | 您不能在Abaqus/CAE中基于方向定义堆叠方向。堆叠方向将对应于上面讨论的默认值。 |
| --- | --- |

#### 验证堆叠方向

可以使用堆叠方向查询工具在Abaqus/CAE中直观验证堆叠方向（见["理解查询工具集的作用，" Abaqus/CAE用户指南第71.1节](../usi/usi-link.md#usi-qre-role)）。对于三维单元，Abaqus/CAE将顶面着色为紫色，底面着色为棕色。对于二维和轴对称单元，箭头指示单元的方向。此外，Abaqus/CAE突出显示任何具有不一致方向的单元面和边。

或者，可以在Abaqus/CAE的可视化模块中绘制材料轴，以验证3轴是否指向三维单元的期望法向方向；如果单元方向不当，面内轴（1轴或2轴）之一将指向法向方向。对于二维和轴对称单元，堆叠方向与2轴材料方向一致。

#### 二维和轴对称单元的厚度方向计算

为了计算二维和轴对称单元的厚度方向，Abaqus通过平均形成单元底面和顶面的节点对的坐标来形成中面。该中面穿过单元的积分点，对于默认的底面和顶面选择如图32.5.4--5所示。对于每个积分点，Abaqus计算一条切线，其方向由底面和顶面上给出的节点序列定义。厚度方向然后通过面外方向和切线方向的叉乘获得。

**图32.5.4-5** 二维或轴对称单元的厚度方向。

![](../graphics/ecohesive-2d-axi-normal-nls.png)

#### 三维单元的厚度方向计算

为了计算三维单元的厚度方向，Abaqus通过平均形成单元底面和顶面的节点对的坐标来形成中面。该中面穿过单元的积分点，对于默认的底面和顶面选择如图32.5.4--6所示。Abaqus将厚度方向计算为每个积分点处中面的法向；正方向通过使用右手定则绕单元底面或顶面上的节点获取。

**图32.5.4-6** 三维单元的厚度方向。

![](../graphics/ecohesive-3d-normal-nls.png)

### 积分点处的局部方向

Abaqus在每个积分点计算默认的局部方向。局部方向用于描述内聚单元当前变形状态的所有量的输出。下面分别讨论具有两个与三个局部方向的内聚单元的局部方向细节。

#### 二维和轴对称内聚单元的局部方向

二维和轴对称内聚单元的局部2方向对应于厚度方向，如上["单元厚度方向定义"](pt06ch32s05alm43.md#usb-elm-ecohesiveinit-thickdir)中所讨论计算。局部1方向定义使得局部1方向和2方向的叉乘给出面外方向（见图32.5.4--7）。对于给定的堆叠方向，您不能修改这些单元的任一局部方向。横向剪切行为在这些单元的1-2平面中定义。

**图32.5.4-7** 二维和轴对称内聚单元的局部方向。

![](../graphics/ecohesive-2d-axi-local-dir.png)

#### 三维内聚单元的局部方向

三维内聚单元的局部3方向对应于厚度方向，如上["单元厚度方向定义"](pt06ch32s05alm43.md#usb-elm-ecohesiveinit-thickdir)中所讨论计算，并且对于给定的堆叠方向不能修改。局部1和2方向垂直于厚度方向，默认情况下由表面上局部方向的标准Abaqus约定定义（["约定，" 第1.2.2节](pt01ch01s02aus02.md)）。三维内聚单元的默认局部方向如图32.5.4--8所示。

**图32.5.4-8** 三维内聚单元的局部方向。

![](../graphics/ecohesive-3d-local-dir-nls.png)

对于这些单元，横向剪切行为在局部1-3和2-3平面中定义。您可以使用局部方向定义（["方向，" 第2.2.5节](pt01ch02s02aus15.md)）修改垂直于厚度方向的平面中三维内聚单元的局部1和2方向。
