# 22.1 理解螺栓载荷





螺栓载荷对螺栓或紧固件中的紧固力或长度调整进行建模。 例如，[图 22-1](pt04ch22s01.md#usi-adv-boltexa) 显示了一个容器（A），该容器通过拧紧固定盖子的螺栓来密封，这会使垫片受到压力。

**图 22-1** 建模预紧螺栓。

![](../graphics/lbi-boltexample-nls.png)

您可以通过在分析的第一步中对每个螺栓施加螺栓载荷来对拧紧螺栓中的张力进行建模。 您可以以集中力或规定的长度变化的形式定义载荷，并可以将载荷施加到您指定的螺栓横截面表面上。 在后续步骤中，您可以修改载荷以防止进一步的长度变化，从而使螺栓作为标准变形组件响应装配上的其他载荷。

创建螺栓载荷时，必须指定以下内容：

**定义螺栓横截面的表面**

Abaqus/CAE将螺栓载荷施加到您指定的横截面表面上。 定义螺栓横截面的表面必须穿过螺栓几何。 Abaqus/CAE在该位置创建一个"内部"表面。

如果您使用的是原生或导入几何的螺栓零件实例，通常需要在您想要定义横截面的位置对螺栓进行划分。 例如，在[图 22-2](pt04ch22s01.md#usi-adv-geombolt-fig) 中，选择一个划分作为螺栓横截面。

**图 22-2** 使用划分指定螺栓横截面。

![](../graphics/lbi-geombolt.png)

如果您使用孤儿网格单元，必须通过选择单元面来指定横截面表面。 例如，[图 22-3](pt04ch22s01.md#usi-adv-meshbolt-fig) 显示了单元面定义网格零件横截面表面的方式。

**图 22-3** 使用单元面指定螺栓横截面。

![](../graphics/lbi-meshbolt.png)

单元面只需要从预拉伸截面一侧的单元中选择。 您可以使用显示组从视口中移除所选单元以显示横截面表面的单元面。 有关选择表面的更多信息，请参阅[第6章，"在视口中选择对象"](pt01ch06.md)。 有关在-wire零件实例上选择表面的详细信息，请参阅["指定区域的特定侧面或端部，" 第73.2.5节](pt06ch73s02s05.md)。

**注意：**螺栓载荷只能应用于三维实体、二维实体和三维线零件实例。 二维和轴对称线零件实例上的螺栓载荷不受支持。

**螺栓轴线**

如果您在实体区域上定义螺栓载荷，必须选择指示螺栓轴线方向的基准轴或基准坐标系统的一个轴。 如果您在线区域上定义螺栓载荷，螺栓轴线方向始终假定为线在螺栓横截面处的切线方向。

Abaqus/CAE使用您指定的横截面表面和螺栓轴线来定义Abaqus/Standard使用的预拉伸截面数据和预拉伸参考节点。 （有关更多信息，请参阅["预定义的装配载荷，" Abaqus Analysis User's Guide第34.5.1节](../usb/usb-link.md#usb-prc-ppretension)。）

**施加载荷的方法**

创建螺栓载荷时，必须选择以下加载方法之一：
- 向螺栓施加力。 此方法对拧紧螺栓进行建模，使其承受指定的载荷。
- 调整螺栓长度。 此方法对拧紧螺栓进行建模，直到其自由长度发生指定值的变化。
- 将螺栓固定在其当前长度。 此方法仅在您在第一步分析中创建载荷并在后续分析步骤中编辑它时才可用。 此方法允许螺栓长度保持不变，以便螺栓中的力可以根据模型的响应而变化。

**所选方法的幅度**

如果您正在向螺栓施加力，必须输入力的大小。 如果您正在调整螺栓长度，必须输入长度变化。

您只能在第一步分析中创建螺栓载荷，但可以在后续步骤中修改载荷方法或载荷大小。 例如，您可以在第一步中施加特定的张力，然后在第二步中更改方法以固定螺栓长度。

有关创建螺栓载荷的详细信息，请参阅["创建和编辑螺栓载荷，" 第22.2节](pt04ch22s02.md)。
