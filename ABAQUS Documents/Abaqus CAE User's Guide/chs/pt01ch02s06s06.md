# 2.6.6 获取有关版本和许可的信息





以下**帮助**菜单项允许您获取其他信息：

**关于Abaqus**

选择****帮助**![](../graphics/images/arrow.gif)**关于Abaqus****可确定您当前使用的Abaqus/CAE版本。Abaqus还提供Abaqus/CAE使用的开源软件（例如Python）版本信息的位置。

**关于许可**

选择****帮助**![](../graphics/images/arrow.gif)**关于许可****可确定产品许可信息。Abaqus显示您的站点标识和许可证服务器名称，以及您的许可证号和您的站点可用的许可证总数。
