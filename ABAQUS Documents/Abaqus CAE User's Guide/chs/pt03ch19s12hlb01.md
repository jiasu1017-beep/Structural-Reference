# 19.12.1 创建和编辑优化过程







优化过程读取您在优化模块中定义的优化任务，并根据您在优化任务中定义的目标函数和约束迭代搜索优化解决方案。

**创建或编辑优化过程：**

1. 从主菜单栏中，选择****优化**![](../graphics/images/arrow.gif)**创建****或****优化**![](../graphics/images/arrow.gif)**编辑**![](../graphics/images/arrow.gif)***优化过程名称*****。出现**编辑优化过程**对话框。**提示：**您也可以通过点击模块工具箱中的![](../graphics/icoR_optProcess.png)图标或点击**优化过程管理器**中的**创建**来创建优化过程。
2. 在**名称**文本字段中输入新的优化过程的名称。您指定的名称必须遵守操作系统的文件命名规则。将使用此名称创建一个目录，优化过程的输出存储在此目录中。
3. 从**模型**字段中选择要与优化过程关联的模型。
4. 从**优化任务**字段中选择要与优化过程关联的优化任务。如有需要，在**描述**文本字段中输入优化过程的描述。描述存储在模型数据库和输出数据库中，您可以使用此描述来帮助管理您的优化过程。
5. 输入在 Abaqus 结束优化过程之前允许的最大优化设计循环次数。
6. 选择将分析结果写入输出数据库文件的频率，该文件由优化过程每个设计循环运行的 Abaqus 作业创建。此选项还指定创建相关 Abaqus 分析文件的频率，如状态文件和消息文件。您可以选择以下内容：- **初始**：在初始循环（*n*=0）后保存数据。优化模块使用初始循环来准备模型以进行第一次设计循环；例如，通过创建组和输出请求。在初始循环期间不执行有限元分析。- **第一个**：保存第一次设计循环（*n*=1）的数据。- **最后一个**：保存最后一次设计循环的数据。- **每*n*个循环**：每*n*个设计循环保存一次数据，相对于初始（*n*=0）设计循环。例如，如果*n*=3，优化过程将保存对应于设计循环 2、5、8 等的数据。- **每个**：在每个设计循环后保存数据，包括初始（*n*=0）设计循环。有关更多信息，请参见["理解创建和分析模型生成的文件，"第 9.4 节](pt02ch09s04.md)。要后处理优化结果，您必须将每个设计循环生成的输出数据库合并为一个输出数据库。有关更多信息，请参见["合并优化结果，"第 19.12.8 节](pt03ch19s12hlb08.md)。
7. 使用**提交**、**内存**和**并行化**选项卡页面为优化过程运行时运行的每个 Abaqus 作业定义初始设置，如以下各节所述：- ["配置作业提交属性，"第 19.8.2 节](pt03ch19s08hlb02.md)- ["控制作业内存设置，"第 19.8.7 节](pt03ch19s08hlb07.md)- ["控制作业并行执行，"第 19.8.8 节](pt03ch19s08s01.md)
8. 点击**确定**。
