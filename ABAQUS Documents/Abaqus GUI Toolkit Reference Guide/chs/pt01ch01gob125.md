# 消息对话框

以下方法提供对标准消息对话框的访问。

### showAFXDismissableWarningDialog(owner, message, tgt=None, sel=0)

发布一个模态可忽略警告对话框，该对话框具有"下次显示此对话框"复选按钮，用户可以通过它请求将来不再看到特定类型的消息。对话框具有 getCheckButtonState 方法，可用于获取对话框中复选按钮的状态。

| **参数** | **类型** | **默认值** | **描述** |
| --- | --- | --- | --- |
| owner | FXWindow |  | 对话框将在其上居中的窗口。 |
| message | String |  | 要在对话框中显示的文本。 |
| tgt | FXObject | None | 消息目标。 |
| sel | Int | 0 | 消息 ID。 |

### showAFXErrorDialog(owner, message, tgt=None, sel=0)

| owner | FXWindow |  | 对话框将在其上居中的窗口。 |
| message | String |  | 要在对话框中显示的文本。 |
| tgt | FXObject | None | 消息目标。 |
| sel | Int | 0 | 消息 ID。 |

### showAFXItemsWarningDialog(owner, topMessage, items, bottomMessage, buttonIds=YES|NO|CANCEL, tgt=None, sel=0)

显示一个模态警告对话框，该对话框具有可滚动的项目列表和两条消息：一条放在列表上方，一条放在列表下方。

| **参数** | **类型** | **默认值** | **描述** |
| --- | --- | --- | --- |
| owner | FXWindow |  | 对话框将在其上居中的窗口。 |
| topMessage | String |  | 显示在项目列表上方的文本。 |
| items | String |  | 包含逗号分隔项目列表的字符串，这些项目将显示在 topMessage 和 bottomMessage 之间。 |
| bottomMessage | String |  | 显示在项目列表下方的文本。 |
| buttonIds | Int | YES|NO|CANCEL | 要创建的操作区域按钮的 ID。 |
| tgt | FXObject | None | 消息目标。 |
| sel | Int | 0 | 消息 ID。 |
