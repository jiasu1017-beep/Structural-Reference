# 5.1.1 访问结果文件：概述

### 将信息写入结果文件

Abaqus结果文件是可用于将分析结果转移到其他软件（如后处理程序）的媒介。可以将以下类型的输出写入结果文件：
- 元素输出、节点输出、能量输出、模态输出、接触表面输出和截面输出（参见["输出到数据和结果文件，" 第4.1.2节](pt02ch04s01aus39.md)）
- 元素矩阵输出（参见["Abaqus/Standard中的元素矩阵输出" in "输出，" 第4.1.1节](pt02ch04s01aus38.md#usb-out-ooutput-elemmatrix)）
- 子结构矩阵输出（参见["将恢复矩阵、减缩刚度矩阵、质量矩阵、载荷情况向量和重力向量写入文件" in "定义子结构，" 第10.1.2节](pt04ch10s01aus59.md#usb-anl-asuperelementdef-output)）
- 空腔辐射视角因子矩阵（参见["将视角因子矩阵写入结果文件" in "空腔辐射，" 第41.1.1节](pt09ch41s01aus187.md#usb-cni-acavityradiation-viewfactoroutput)）

["输出，" 第4.1.1节](pt02ch04s01aus38.md)描述了结果文件的一般格式。

Abaqus模型可以按照部件实例的装配来定义（参见["定义装配，" 第2.10.1节](pt01ch02s10aus28.md)）。但是，结果文件不是按部件组织的；它包含内部节点和单元编号（参见["输出，" 第4.1.1节](pt02ch04s01aus38.md)）。原始编号与部件实例名称和内部编号之间的映射被写入数据文件。
