# 47.7.2 自定义X–Y图的边框和填充颜色





### 47.7.2 自定义X–Y图的边框和填充颜色

点击**Grid Display**选项卡访问*X–Y*图的**Grid Area**选项。这些选项使您能够自定义*X–Y*图中的网格，即绘制曲线的矩形区域。您可以在网格周围包裹边框或用背景色填充网格。

**注意：**您还可以控制整个*X–Y*图的边框和背景颜色设置，这包括*X–Y*图的网格覆盖的视口部分及其所有轴。Abaqus/CAE在整*X–Y*图的背景上显示网格的背景颜色。有关更多信息，请参见["自定义X–Y图的边框和填充颜色，"第47.10节](pt05ch47s09.md)。

**要自定义X–Y图的边框和填充颜色：**

1. 找到**Grid Area**选项。选择****Options**![](../graphics/images/arrow.gif)**XY Options**![](../graphics/images/arrow.gif)**Chart****；或点击![](../graphics/ico_xyChartOptions.png)，它位于Visualization模块工具箱中的*X–Y*绘图工具中。在出现的对话框中点击**Grid Display**选项卡。**Grid Area**选项在**Grid Display**页面底部可用。
2. 从窗口顶部的**Charts**字段中选择一个或多个*X–Y*图。
3. 切换打开**Show border**以在整网格区域周围显示边框。
4. 如果需要，通过执行以下步骤更改边框颜色：1. 点击紧邻**Show border**标签右侧的颜色样本![](../graphics/usi-colorsample.png)。Abaqus/CAE显示**Select Color**对话框。2. 使用**Select Color**对话框中的方法之一选择新颜色。有关更多信息，请参见["自定义颜色，"第3.2.9节](pt01ch03s02s09.md)。3. 点击**OK**关闭**Select Color**对话框。颜色样本和网格边框变为所选颜色。
5. 切换打开**Fill**以用**Fill**标签右侧显示的颜色填充网格区域。
6. 如果需要，通过执行以下步骤更改网格区域填充颜色：1. 点击紧邻**Fill**标签右侧的颜色样本![](../graphics/usi-colorsample.png)。Abaqus/CAE显示**Select Color**对话框。2. 使用**Select Color**对话框中的方法之一选择新颜色。有关更多信息，请参见["自定义颜色，"第3.2.9节](pt01ch03s02s09.md)。3. 点击**OK**关闭**Select Color**对话框。网格区域变为所选颜色。
7. 点击**Dismiss**关闭**Chart Options**对话框。
