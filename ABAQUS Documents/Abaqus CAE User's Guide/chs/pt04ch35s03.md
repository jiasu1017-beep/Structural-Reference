# 35.3 为中面模型创建壳面





拥有参考表示后，您必须手动构建壳模型来替换已从模型中移除的实体单元。您可以使用壳特征工具和Geometry Edit工具集中的工具（更多信息，请分别参见["添加壳特征"，第11.22节](pt03ch11s22.md)和[第69章，"The Geometry Edit工具集"](pt06ch69.md)）来创建和编辑壳面。

本节讨论在使用这些工具为中面模型创建面时可能适用的某些特殊条件。涵盖以下主题：
- ["为中面建模使用偏移面工具"，第35.3.1节](pt04ch35s03s01.md)
- ["为中面建模使用延伸面工具"，第35.3.2节](pt04ch35s03s02.md)
