# 69.7.8 扩展面







您可以通过扩展现有面来编辑面。扩展操作作为部件的特征存储；因此，如果使用距离方法，您可以使用模型树编辑距离参数，或删除扩展的面。

[图 69-10](pt06ch69s07hlb08.md#rep-extendface) 显示了使用目标面扩展面。顶部图像显示了完整部件，其中包括与周围面被小间隙分开的中心平面面。周围面包围着平面面的大约两个半边。平面面被选择用于扩展，周围面用作目标面。所选面沿将与目标面相交的三个边扩展。两个底部图像详细说明了您如何在所选面左侧超出目标面末端的区域控制扩展。在左下图像中，使用了 **Trim to extended underlying target surfaces**（修剪到扩展的底层目标面）选项，使得所选面的整个左边缘均匀扩展——就好像目标面扩展到所选面的整个高度一样。在右下图像中，面仅扩展到目标面结束的位置。

**图 69-10** 扩展面时切换 **Trim to extended underlying target surfaces** 选项的效果：带有间隙的初始模型（左上和右上），开启（左下），关闭（右下）。

![](../graphics/rep-extendface.png)

**扩展面：**

1. 从主菜单栏中选择 ****Tools**![](../graphics/images/arrow.gif)**Geometry Edit****（工具→几何编辑）。Abaqus/CAE 显示 **Geometry Edit**（几何编辑）对话框。**提示：**您也可以使用位于 Part（部件）模块工具箱中编辑工具处的 ![](../graphics/ico_extendFaces.png) 工具来扩展面。 有关工具箱中编辑工具的图表，请参阅["使用几何编辑工具集，" 第 69.1 节"](pt06ch69s01.md)。
2. 从对话框中选择 **Face**（面）类别和 **Extend**（扩展）方法。Abaqus/CAE 显示 **Extend Faces**（扩展面）对话框。
3. 选择以下方法之一来扩展面： - 切换开启 **Specify edges of faces to extend**（指定要扩展的面的边），然后选择一个或多个边来仅沿所选边扩展由这些边限定的面。 - 切换开启 **Specify faces to extend**（指定要扩展的面），然后选择面以沿其所有自由边扩展它们。点击 **Select**（选择）指定要扩展的边或面。您可以单独选择边或按边角度选择；并且您可以单独、按面角度或按面曲率选择面。 有关在视口中选择对象的更多信息，请参阅["在当前视口中选择对象，" 第 6.2 节"](pt01ch06s02.md)。您也可以选择现有的边或面集合。**注意：**默认选择方法基于您最近使用的选择方法。要恢复为其他方法，请点击提示区域右侧的 **Select in Viewport**（在视口中选择）或 **Sets**（集合）。
4. 从提示区域点击 **Done**（完成）。
5. 选择以下可用方法之一来确定扩展距离：**Distance**（距离）输入扩展距离，或选择 ![](../graphics/ico_measure.png) 来测量并输入视口中对象之间的距离。**Up to target faces**（到目标面）选择目标面以限制扩展。Abaqus/CAE 将目标面显示为黄色高亮。如果需要，切换开启 **Trim to extended underlying target surfaces**（修剪到扩展的底层目标面）以让 Abaqus/CAE 临时扩展目标面以与您要扩展的面创建相交。**Up to reference representation**（到参考表示）Abaqus/CAE 使用参考表示的面来限制扩展。如果模型不包含参考表示，则此选项不可用。
6. 点击 **OK**（确定）。如果您在第 5 步中选择的目标面没有完全包围正在扩展的面，Abaqus/CAE 会指示将与目标面相交的边——您第 3 步选择的子集——并为您提供仅扩展这些边而不是原始选择的选项。Abaqus/CAE 创建扩展面特征。如果您使用目标面或参考表示来限制扩展，Abaqus/CAE 会在扩展面与限制面相交处修剪它们。扩展面特征替换视口中的原始面，过程在第 3 步使用最近使用的选择方法重新开始。
7. 要退出扩展过程，请执行以下操作之一 - 点击提示区域中的取消按钮 ![](../graphics/fts-cancel.gif)，或 - 在 Abaqus/CAE 窗口中的任意位置点击鼠标按钮 2，或 - 从几何编辑工具集或 Part（部件）模块中的工具中选择其他操作。
