### 概述

接触属性：
- 定义了表面相互作用模型，当表面处于接触状态时控制其行为；以及
- 可选择性地应用于通用接触域内的特定区域。

### 分配接触属性

Abaqus/Standard 中的默认接触属性模型假定法向为"硬"接触、无摩擦、无热相互作用等。您可以为通用接触域的指定区域分配非默认接触属性定义（表面相互作用）。

Abaqus/Standard 中通用接触的接触属性在分析开始时分配，不能跨步骤修改，但摩擦模型的更改除外，如下所述。

用于指定应分配非默认接触属性的区域的表面名称不必与用于指定通用接触域的表面名称相对应。在许多情况下，接触相互作用将针对大域定义，而非默认接触属性将分配给该域的子集。任何在通用接触域之外区域的接触属性分配都将被忽略。如果指定区域重叠，则最后分配的优先。

| **输入文件用法：** | ``` [*CONTACT PROPERTY ASSIGNMENT](../key/key-link.md#usb-kws-hcontpropassign) *surface_1*, *surface_2*, *interaction_property_name* ``` |
| --- | --- |
|  | 此选项必须与 [*CONTACT](../key/key-link.md#usb-kws-hcontact) 选项结合使用，至多出现一次；数据行可以根据需要重复多次，以将接触属性分配给不同的区域。如果省略第一个表面名称，则假定一个包含整个通用接触域的默认表面。如果省略第二个表面名称或与第一个表面名称相同，则假定第一个表面与自身之间的接触。表面可以定义为跨越多个分离体，因此自接触不限于单个物体与自身的接触。如果省略相互作用属性名称，则假定使用 Abaqus/Standard 中默认接触属性的未命名集合。如果指定了相互作用属性名称，它也必须出现在输入文件模型部分中 [*SURFACE INTERACTION](../key/key-link.md#usb-kws-hsurfaceinteraction) 选项的 NAME 参数的值中。 |

| **Abaqus/CAE 用法：** | 使用以下选项向整个通用接触域分配全局接触属性： |
| --- | --- |
|  | Interaction 模块： **Create Interaction**: **General contact (Standard)**: **Contact Properties**: **Global property assignment:** *interaction_property_name* 使用以下选项向各个表面配对分配接触属性： Interaction 模块： **Create Interaction**: **General contact (Standard)**: **Contact Properties**: **Individual property assignments: Edit**: 选择左列中的表面和接触属性，然后点击中间的箭头将它们转移到接触属性分配列表中 在 Abaqus/CAE 中，您必须分配全局接触属性；Abaqus/CAE 不会假定默认接触相互作用属性。分配给各个表面配对的接触属性会覆盖全局分配。 |

#### 在分析期间改变摩擦属性

与给定命名表面相互作用定义关联的摩擦属性可以在 Abaqus/Standard 分析的任何特定步骤中进行修改，如 ["在 Abaqus/Standard 分析期间改变摩擦属性" 中所述，在 "摩擦行为，" 第 37.1.5 节](pt09ch37s01aus169.md#usb-cni-afriction-change-std)。

#### 示例

以下接触属性分配在通用接触分析中指定为模型数据：
- 向整个通用接触域分配 `contProp1`；
- 向 `surf1` 的自接触局部分配 `contProp2`；
- 向 `surf2` 和 `surf3` 之间的接触局部分配默认 Abaqus 接触属性；以及
- 向整个接触域和 `surf4` 之间的接触局部分配 `contProp3`。`contProp3` 的摩擦系数在第二步中从初始值 0.20 重置为 0.05。

```
[*SURFACE INTERACTION](../key/key-link.md#usb-kws-hsurfaceinteraction), NAME=contProp1
[*FRICTION](../key/key-link.md#usb-kws-hfriction)
0.1
[*SURFACE INTERACTION](../key/key-link.md#usb-kws-hsurfaceinteraction), NAME=contProp2
[*FRICTION](../key/key-link.md#usb-kws-hfriction)
0.15
[*SURFACE INTERACTION](../key/key-link.md#usb-kws-hsurfaceinteraction), NAME=contProp3
[*FRICTION](../key/key-link.md#usb-kws-hfriction)
0.20
[*CONTACT](../key/key-link.md#usb-kws-hcontact)
[*CONTACT INCLUSIONS](../key/key-link.md#usb-kws-hcontactinclusions), ALL EXTERIOR
[*CONTACT PROPERTY ASSIGNMENT](../key/key-link.md#usb-kws-hcontpropassign)
 , , contProp1 
surf1, surf1, contProp2 
surf2, surf3,
 , surf4, contProp3
…
[*STEP](../key/key-link.md#usb-kws-hstep)
Step1
[*STATIC](../key/key-link.md#usb-kws-hstatic)
…
[*END STEP](../key/key-link.md#usb-kws-hendstep)
[*STEP](../key/key-link.md#usb-kws-hstep)
Step2
[*STATIC](../key/key-link.md#usb-kws-hstatic)
…
[*CHANGE FRICTION](../key/key-link.md#usb-kws-hchangefriction), INTERACTION NAME=contProp3
[*FRICTION](../key/key-link.md#usb-kws-hfriction)
0.05
[*END STEP](../key/key-link.md#usb-kws-hendstep)
```
