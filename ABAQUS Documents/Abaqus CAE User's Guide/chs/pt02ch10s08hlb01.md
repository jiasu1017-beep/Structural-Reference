# 10.9.2 将零件导出为ACIS、IGES、STEP或VDA格式文件





从主菜单栏选择**文件**![](../graphics/images/arrow.gif)**导出**![](../graphics/images/arrow.gif)**零件**，可以使用以下格式之一将当前零件导出为文件：
- **ACIS SAT (*.sat)**
- **IGES (*.igs)**
- **STEP (*.stp)**
- **VDA (*.vda)**

导出零件之前，必须在当前视口中显示要导出的零件。您只能导出零件的几何体；如果存在孤儿元素，则不会将其包含在导出的零件中。

**导出零件：**

1. 在当前视口中，显示要导出的零件。
2. 从主菜单栏，选择**文件**![](../graphics/images/arrow.gif)**导出**![](../graphics/images/arrow.gif)**零件**。出现**导出零件**对话框。
3. 从**导出零件**对话框底部的**文件过滤器**菜单中，选择文件格式。Abaqus/CAE列出所选目录中所有具有相应文件扩展名的文件。
4. 选择要导出零件的文件，或在**文件名**文本字段中输入新文件的名称，然后单击**确定**。
5. 如果要将零件导出为IGES文件，Abaqus/CAE会显示**IGES风格**对话框。执行以下操作：1.选择用于写入IGES文件的以下风格之一：-**标准**-**AutoCAD**-**Solid Works**-**JAMA**2.单击**确定**。Abaqus/CAE调整IGES文件的内部表示以匹配所选应用程序期望的格式。默认情况下，Abaqus/CAE以**标准**格式导出零件。Abaqus/CAE将所有几何数据写入IGES文件中的单个图层。
