# 30.3 定义流体材料属性





流体材料及其物理属性决定流体响应。您需要在Property模块中创建流体材料。根据分析的不同，您可能需要指定以下属性：

**密度**

必须为Abaqus/CFD分析中的流体定义密度；对于瞬态Navier-Stokes计算是必需的。流体密度在不可压缩流动中被认为是常数。

**粘度**

对于粘性流动，您必须指定粘度，仅支持恒定粘度。

**比热**

对于包含能量方程的不可压缩流体动力学分析，您必须定义比热和热导率。需要恒压比热；您可以定义恒压比热，或者定义恒容比热并指定理想气体常数。

**热导率**

仅支持各向同性恒定热导率。

**热膨胀系数**

热膨胀系数与Boussinesq近似一起用于自然对流流动。

**渗透率**

您可以指定各向同性渗透率（仅依赖于孔隙度），或使用Carman-Kozeny关系指定渗透率。
