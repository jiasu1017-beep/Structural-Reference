# 11.22.6 添加实体抽壳特征





从主菜单栏选择**形状**![](../graphics/images/arrow.gif)**壳**![](../graphics/images/arrow.gif)**从实体**，从实体特征的面创建壳特征。您只能向三维零件添加实体抽壳特征。

您通过选择要从零件中移除的单元来添加实体抽壳特征；Abaqus/CAE将任何与移除单元关联的剩余面转换为壳。

**添加实体抽壳特征：**

1. 从主菜单栏，选择**形状**![](../graphics/images/arrow.gif)**壳**![](../graphics/images/arrow.gif)**从实体**。
2. 选择一个或多个要转换为壳的单元。**添加更多单元，单击**[Shift]**+单击**，**取消选择单元，单击**[Ctrl]**+单击**。单击鼠标按钮2表示完成选择要转换的单元。
