# 62.6.1 通过输入坐标创建datum点







您可以通过输入其*X*、*Y*和*Z*坐标来定位datum点，如下图所示：

![](../graphics/dtm-hptc-nls.png)

**通过输入坐标创建datum点：**

1. 从主菜单栏中，选择****Tools**![](../graphics/images/arrow.gif)**Datum****。此时将出现**Create Datum**对话框。该对话框概述了您可以创建的datum几何体类型。Abaqus/CAE在提示区域中显示提示以指导您完成此过程。**提示：**您也可以使用模块工具箱中的![](../graphics/ico_dtmPtCoordinates.png)工具创建datum点。有关工具箱中datum工具的图示，请参见["使用Datum工具集，" 第62.2节](pt06ch62s02.md)。
2. 从对话框顶部的类型列表中，选择**Point**。**Method**列表指示您可以使用的方法来创建datum点。
3. 从**Method**列表中，选择**Enter coordinates**。
4. 在提示区域出现的文本框中，输入要创建的datum点的*X*、*Y*和*Z*坐标。如果datum与装配相关联，则您提供的坐标相对于装配的全局坐标系。如果datum与零件相关联，则坐标相对于零件的坐标系。如果您不确定零件坐标系的位置或方向，首先在零件原点创建一个默认坐标系。有关更多信息，请参见["由三点定义的datum坐标系，" 第62.9.1节](pt06ch62s09hlb01.md)。datum点会出现。您可以通过从主菜单栏中选择****Feature**![](../graphics/images/arrow.gif)**Edit****并选择该datum来修改datum点的坐标。
