# 15.14.1 定义接触相互作用属性





接触属性编辑器包含以下菜单，您可以从这些菜单中选择要包含在属性定义中的选项：
- **Mechanical**；请参见"定义机械接触属性选项"
- **Thermal**；请参见"定义热接触属性选项"
- **Electrical**；请参见"定义电接触属性选项"

接触相互作用属性可以由通用接触、表面到表面接触或自接触相互作用引用。有关更多信息，请参见Abaqus Analysis User's Guide第37.1.1节的"机械接触属性：概述"、Abaqus Analysis User's Guide第37.2.1节的"热接触属性"和Abaqus Analysis User's Guide第37.3.1节的"电接触属性"。

编辑器顶部的「Contact Property Options」列表显示当前包含在属性定义中的选项；该列表会在您添加和删除选项时更新。您可以按如下方式添加、删除或更改属性选项：

**添加属性选项**

从「Mechanical」、「Thermal」和「Electrical」菜单中选择定义属性所需的选项。当您选择某个选项时，其名称会出现在「Contact Property Options」列表中，与该选项关联的数据字段会出现在编辑器底部半部分的 data area 中。使用数据字段输入当前所选选项的信息。

**删除属性选项**

在「Contact Property Options」列表中，选择您要删除的选项，然后点击编辑器右侧的「Delete」。此过程会从选项列表和属性定义中删除该选项。

**更改选项数据**

在「Contact Property Options」列表中，选择要更改其数据的选项。当与该选项关联的数据字段出现在窗口底部半部分时，按需更改您为该选项输入的信息。

### 定义机械接触属性选项

您可以定义机械接触属性选项来指定切向行为（摩擦和弹性滑移）、法向行为（硬、软或阻尼接触及分离）以及摩擦阻尼。以下部分描述了如何指定机械接触属性模型：
- "为机械接触属性选项指定摩擦行为"
- "为机械接触属性选项指定压力-闭合关系"
- "为机械接触属性选项指定阻尼"
- "为机械接触属性选项指定内聚行为属性"
- "为机械接触属性选项指定内聚损伤属性"
- "为裂纹扩展指定断裂准则属性"
- "为机械接触属性选项指定几何属性"

#### 为机械接触属性选项指定摩擦行为

您可以指定一个摩擦模型，该模型定义了机械接触分析中表面相对切向运动所受到的阻力。有关更多信息，请参见Abaqus Analysis User's Guide第37.1.5节的"摩擦行为"。

**指定摩擦行为的步骤：**

1. 从主菜单栏选择 ****Interaction**![](../graphics/images/arrow.gif)**Property**![](../graphics/images/arrow.gif)**Create****。
2. 在出现的「Create Interaction Property」对话框中，执行以下操作：- 为相互作用属性命名。有关命名对象的更多信息，请参见"使用基本对话框组件"，第3.2.1节。- 选择「Contact」类型的相互作用属性。
3. 点击「Continue」关闭「Create Interaction Property」对话框。
4. 从接触属性编辑器的主菜单栏中选择 ****Mechanical**![](../graphics/images/arrow.gif)**Tangential Behavior****。
5. 在出现的编辑器中，点击「Friction formulation」字段右侧的箭头，选择您想要定义接触表面之间摩擦的方式：- 如果您希望Abaqus假设接触表面自由滑动且无摩擦，请选择「Frictionless」。- 选择「Penalty」以使用刚度（惩罚）方法，该方法允许表面在应该粘附时发生一些相对运动（"弹性滑移"）。当表面粘附时（即![](../graphics/usi_eqn00575.gif)），滑移量限于此弹性滑移。Abaqus将不断调整惩罚约束的幅度以强制执行此条件。有关更多信息，请参见Abaqus Analysis User's Guide第37.1.5节"摩擦行为"中的"在Abaqus/Standard中施加摩擦约束的刚度方法"和Abaqus Analysis User's Guide第37.1.5节"摩擦行为"中的"在Abaqus/Explicit中施加摩擦约束的刚度方法"。- 选择「Static-Kinetic Exponential Decay」以直接指定静摩擦系数和动摩擦系数。在该模型中，摩擦系数从静摩擦值指数衰减到动摩擦值。或者，您可以输入测试数据来拟合指数模型。（此「Friction formulation」选项还允许您指定弹性滑移。）有关更多信息，请参见Abaqus Analysis User's Guide第37.1.5节"摩擦行为"中的"指定静摩擦系数和动摩擦系数"。- 选择「Rough」以指定无穷大的摩擦系数。有关更多信息，请参见Abaqus Analysis User's Guide第37.1.5节"摩擦行为"中的"无论接触压力如何都防止滑动"。- 选择「Lagrange Multiplier (Standard only)」以使用Lagrange乘子实现来强制执行两个表面之间界面上的粘附约束。使用此方法，两个闭合表面之间在![](../graphics/usi_eqn00576.gif)之前没有相对运动。有关更多信息，请参见Abaqus Analysis User's Guide第37.1.5节"摩擦行为"中的"在Abaqus/Standard中施加摩擦约束的Lagrange乘子方法"。- 选择「User-defined」以使用用户子程序[`FRIC`](../sub/sub-link.md#sub-xsl-fric)或[`VFRIC`](../sub/sub-link.md#sub-xsl-vfric)定义接触表面之间的剪切相互作用。有关更多信息，请参见Abaqus Analysis User's Guide第37.1.5节"摩擦行为"中的"用户定义的摩擦模型"。
6. 如果您选择了「Penalty」或「Lagrange Multiplier (Standard only)」摩擦公式，请执行以下步骤：1. 显示「Friction」选项卡页面。2. 选择「Directionality」：- 选择「Isotropic」以输入均匀摩擦系数。- 选择「Anisotropic (Standard only)」以允许接触表面上两个正交方向具有不同的摩擦系数。有关更多信息，请参见Abaqus Analysis User's Guide第37.1.5节"摩擦行为"中的"在Abaqus/Standard中使用各向异性摩擦模型"。3. 如果摩擦系数依赖于滑移率，切换「Use slip-rate-dependent data」。4. 如果摩擦系数依赖于接触压力，切换「Use contact-pressure-dependent data」。5. 如果摩擦系数依赖于温度，切换「Use temperature-dependent data」。6. 点击「Number of field variables」字段右侧的箭头以指定摩擦系数所依赖的场变量数量。7. 在提供的数据表中输入所需的数据。8. 显示「Shear Stress」选项卡页面，并选择「Shear stress limit」选项：- 如果您不想限制界面在表面开始滑动之前可以承受的剪切应力，请选择「No limit」。- 选择「Specify」以输入等效剪切应力极限值![](../graphics/usi_eqn00577.gif)。如果您选择此选项，如果等效剪切应力的幅度达到此值，无论接触压力应力的大小如何，都会发生滑动。有关更多信息，请参见Abaqus Analysis User's Guide第37.1.5节"摩擦行为"中的"使用可选的剪切应力极限"。9. 如果您选择了「Penalty」摩擦公式，请显示「Elastic Slip」选项卡页面，并指定您想要如何定义弹性滑移：- 如果您正在执行Abaqus/Standard分析，请选择「Specify maximum elastic slip」的选项：- 选择「Fraction of characteristic surface dimension」以将允许的弹性滑移计算为特征接触表面长度的一小部分。- 选择「Absolute distance」以输入允许弹性滑移的绝对值![](../graphics/usi_eqn00578.gif)。（对于稳态传输分析，将此参数设置为允许弹性滑移速度的绝对值（![](../graphics/usi_eqn00579.gif)），以用于粘附摩擦的刚度方法。）- 如果您正在执行Abaqus/Explicit分析，请选择「Elastic slip stiffness」选项：- 选择「Infinite (no slip)」以停用剪切软化。- 选择「Specify」以激活软化切向行为。输入定义两个表面之间弹性滑移函数的剪切牵引力曲线斜率。有关更多信息，请参见Abaqus Analysis User's Guide第37.1.5节"摩擦行为"中的"粘附时的剪切应力与弹性滑移"。
7. 如果您选择了「Static-Kinetic Exponential Decay」摩擦公式，请执行以下步骤：1. 显示「Friction」选项卡页面。2. 选择定义指数衰减摩擦模型的选项：- 选择「Coefficients」以直接提供静摩擦系数、动摩擦系数和衰减系数。- 选择「Test data」以提供拟合指数模型的测试数据点。有关更多信息，请参见Abaqus Analysis User's Guide第37.1.5节"摩擦行为"中的"指定静摩擦系数和动摩擦系数"。3. 如果您选择了「Coefficients」定义选项，请在提供的数据表中输入以下内容：- 静摩擦系数，![](../graphics/usi_eqn00580.gif)。- 动摩擦系数，![](../graphics/usi_eqn00581.gif)。- 衰减系数，![](../graphics/usi_eqn00582.gif)。如果您选择了「Test data」定义选项，请在提供的数据表中输入以下内容：- 在第一行中，输入静摩擦系数，![](../graphics/usi_eqn00583.gif)。- 在第二行中，输入动摩擦系数，![](../graphics/usi_eqn00584.gif)，以及测量![](../graphics/usi_eqn00584.gif)的参考滑移率，![](../graphics/usi_eqn00585.gif)。- 在第三行中，输入动摩擦系数，![](../graphics/usi_eqn00586.gif)。此值对应于无限滑移率下摩擦系数的渐近值，![](../graphics/usi_eqn00587.gif)。如果省略此数据行，Abaqus/Standard会自动计算![](../graphics/usi_eqn00586.gif)，使得![](../graphics/usi_eqn00588.gif)。4. 显示「Elastic Slip」选项卡页面，并指定您想要如何定义弹性滑移：- 如果您正在执行Abaqus/Standard分析，请选择「Specify maximum elastic slip」的选项：- 选择「Fraction of characteristic surface dimension」以将允许的弹性滑移计算为特征接触表面长度的一小部分。- 选择「Absolute distance」以输入允许弹性滑移的绝对值![](../graphics/usi_eqn00578.gif)。（对于稳态传输分析，将此参数设置为允许弹性滑移速度的绝对值（![](../graphics/usi_eqn00579.gif)），以用于粘附摩擦的刚度方法。）- 如果您正在执行Abaqus/Explicit分析，请选择「Elastic slip stiffness」选项：- 选择「Infinite (no slip)」以停用剪切软化。- 选择「Specify」以激活剪切软化。输入定义两个表面之间弹性滑移函数的剪切牵引力曲线斜率。有关更多信息，请参见Abaqus Analysis User's Guide第37.1.5节"摩擦行为"中的"粘附时的剪切应力与弹性滑移"。
8. 如果您选择了「User-defined」摩擦公式，请执行以下步骤：1. 点击「Number of state-dependent variables」字段右侧的箭头，以指示将在用户子程序[`FRIC`](../sub/sub-link.md#sub-xsl-fric)或[`VFRIC`](../sub/sub-link.md#sub-xsl-vfric)中定义的状态变量数量。2. 在「Friction Properties」表中，输入用户子程序[`FRIC`](../sub/sub-link.md#sub-xsl-fric)或[`VFRIC`](../sub/sub-link.md#sub-xsl-vfric)所需的属性值。（有关输入数据的详细信息，请参见"输入表格数据"，第3.2.7节。）有关更多信息，请参见Abaqus Analysis User's Guide第37.1.5节"摩擦行为"中的"用户定义的摩擦模型"。
9. 点击「OK」创建接触属性并退出「Edit Contact Property」对话框。或者，您可以从「Edit Contact Property」对话框的菜单中选择另一个接触属性选项进行定义。

#### 为机械接触属性选项指定压力-闭合关系

您可以定义一个本构模型，用于控制机械接触分析中表面运动的接触压力-闭合关系。有关更多信息，请参见Abaqus Analysis User's Guide第37.1.2节的"接触压力-闭合关系"。

**指定接触压力-闭合关系的步骤：**

1. 从主菜单栏选择 ****Interaction**![](../graphics/images/arrow.gif)**Property**![](../graphics/images/arrow.gif)**Create****。
2. 在出现的「Create Interaction Property」对话框中，执行以下操作：- 为相互作用属性命名。有关命名对象的更多信息，请参见"使用基本对话框组件"，第3.2.1节。- 选择「Contact」类型的相互作用属性。
3. 点击「Continue」关闭「Create Interaction Property」对话框。
4. 从接触属性编辑器的主菜单栏中选择 ****Mechanical**![](../graphics/images/arrow.gif)**Normal Behavior****。
5. 从「Pressure-Overclosure」字段中，选择"Hard" Contact，在Abaqus/Standard分析中使用经典Lagrange乘子约束强制方法，在Abaqus/Explicit分析中使用惩罚接触强制。您也可以关闭「Allow separation after contact」，如果您希望防止表面在接触后分离。此方法仅适用于Abaqus/Standard分析。如果您选择了"Hard" Contact，您还可以自定义约束强制方法的设置。有关约束强制方法的更多信息，请参见Abaqus Analysis User's Guide第38.2.3节的"Abaqus/Explicit中的接触约束强制方法"。要指定这些设置，请从「Constraint enforcement method」列表中选择一个选项，并执行以下操作：1. 选择「Default」以使用接触压力-闭合关系强制执行约束。2. 选择「Augmented Lagrange (Standard)」以使用增强Lagrange方法强制执行接触约束。此方法仅适用于Abaqus/Standard分析。如果您选择此选项，请从「Contact Stiffness」选项中指定以下附加设置：- 从「Stiffness value」字段中，选择「Use default」以让Abaqus自动计算惩罚接触刚度，或选择「Specify」并输入惩罚接触刚度的正值。- 在「Stiffness scale factor」字段中指定乘以所选惩罚刚度的因子。- 指定「Clearance at which contact pressure is zero」。默认值为0。3. 选择「Penalty (Standard)」约束强制方法以使用惩罚方法强制执行接触约束。此方法仅适用于Abaqus/Standard分析。如果您选择此选项，请从「Contact Stiffness」选项中指定以下附加设置：- 从「Behavior」字段中，选择「Linear」以使用线性惩罚方法强制执行接触约束，或选择「Nonlinear」以使用非线性惩罚方法强制执行接触约束。有关更多信息，请参见Abaqus Analysis User's Guide第38.1.2节的"Abaqus/Standard中的接触约束强制方法"中的"惩罚方法"。- 指定接触刚度。- 对于线性惩罚方法，请在「Stiffness value」字段中指定接触刚度。您可以选择「Use default」以让Abaqus自动计算惩罚接触刚度，或者选择「Specify」并输入线性惩罚刚度的正值。- 对于非线性惩罚方法，请在「Maximum stiffness value」字段中指定接触刚度。您可以选择「Use default」以让Abaqus自动计算惩罚接触刚度，或者选择「Specify」并输入最终非线性惩罚刚度的正值。- 在「Stiffness scale factor」字段中指定乘以所选惩罚刚度的因子。- 对于非线性惩罚方法，您可以指定以下选项的值：- 在「Initial/Final stiffness ratio」字段中输入初始惩罚刚度与最终惩罚刚度的比值。- 在「Upper quadratic limit scale factor」字段中输入上二次限制![](../graphics/usi_eqn00589.gif)的比例因子，该值等于比例因子乘以特征接触面长度。- 在「Lower quadratic limit ratio」字段中输入比值（![](../graphics/usi_eqn00590.gif)![](../graphics/usi_eqn00591.gif)）/（![](../graphics/usi_eqn00589.gif)![](../graphics/usi_eqn00591.gif)），该比值定义了下二次限制![](../graphics/usi_eqn00590.gif)。- 指定「Clearance at which contact pressure is zero」。默认值为0。4. 选择「Direct (Standard)」以直接强制执行接触约束，无需近似或使用增强迭代。
6. 从「Pressure-Overclosure」字段中，选择「Exponential」以定义指数压力-闭合关系。如果您选择此选项，请指定以下内容：1. 在数据表中输入零间隙时的接触压力，![](../graphics/usi_eqn00592.gif)，以及接触压力为零时的间隙，![](../graphics/usi_eqn00591.gif)。2. 指定模型可以达到的接触刚度极限，![](../graphics/usi_eqn00593.gif)（仅适用于Abaqus/Explicit分析）。- 选择「Infinite (no slip)」以将![](../graphics/usi_eqn00593.gif)设置为等于运动接触的无穷大和惩罚接触的默认惩罚刚度。- 选择「Specify」，并输入最大刚度的值。
7. 从「Pressure-Overclosure」字段中，选择「Linear」以定义线性压力-闭合关系。如果您选择此选项，请指定以下内容：- 在「Contact stiffness」字段中输入压力-闭合曲线斜率*k*的正值。
8. 从「Pressure-Overclosure」字段中，选择「Tabular」以表格形式定义分段线性压力-闭合关系。如果您选择此选项，请指定以下内容：- 按闭合升序输入数据，以将闭合定义为压力的函数。数据表必须从零压力开始。压力-闭合关系通过延续最后闭合点之后的相同斜率来推断。
9. 从「Pressure-Overclosure」字段中，选择「Scale Factor (General Contact, Explicit)」以基于缩放默认接触刚度来定义分段线性压力-闭合关系。此选项仅适用于Abaqus/Explicit中的通用接触算法。如果您选择此选项，请指定以下内容：1. 要将闭合度量定义为最小元素尺寸的百分比，请在「Overclosure」字段中选择「factor」，并输入正值![](../graphics/usi_eqn00270.gif)。2. 要直接定义闭合度量，请在「Overclosure」字段中选择「measure」，并输入正值![](../graphics/usi_eqn00589.gif)。3. 在「Contact stiffness scale factor」字段中输入值![](../graphics/usi_eqn00274.gif)]，以定义"基础"刚度的几何缩放比例。4. 在「Initial stiffness scale factor」字段中输入正值![](../graphics/usi_eqn00361.gif)]，以定义"基础"默认接触刚度的附加比例因子。默认值为1。
10. 点击「OK」创建接触属性并退出「Edit Contact Property」对话框。或者，您可以从「Edit Contact Property」对话框的菜单中选择另一个接触属性选项进行定义。

#### 为机械接触属性选项指定阻尼

您可以定义一个阻尼模型，该模型定义了机械接触分析中抵抗接触表面相对运动的力。有关更多信息，请参见Abaqus Analysis User's Guide第37.1.3节的"接触阻尼"。

**指定接触阻尼的步骤：**

1. 从主菜单栏选择 ****Interaction**![](../graphics/images/arrow.gif)**Property**![](../graphics/images/arrow.gif)**Create****。
2. 在出现的「Create Interaction Property」对话框中，执行以下操作：- 为相互作用属性命名。有关命名对象的更多信息，请参见"使用基本对话框组件"，第3.2.1节。- 选择「Contact」类型的相互作用属性。
3. 点击「Continue」关闭「Create Interaction Property」对话框。
4. 从接触属性编辑器的主菜单栏中选择 ****Mechanical**![](../graphics/images/arrow.gif)**Damping****。
5. 在出现的编辑器中，点击「Definition」字段右侧的箭头，选择确定阻尼系数维度的选项：- 选择「Damping coefficient」以指定具有单位压力每相对速度的阻尼系数。有关更多信息，请参见Abaqus Analysis User's Guide第37.1.3节"接触阻尼"中的"指定阻尼系数，使得阻尼力与表面之间的相对运动速率成正比"。- 选择「Critical damping fraction (Explicit only)」以相对于与接触刚度相关的临界阻尼的分数来指定无量纲阻尼系数；此方法仅适用于Abaqus/Explicit。有关更多信息，请参见Abaqus Analysis User's Guide第37.1.3节"接触阻尼"中的"在Abaqus/Explicit中指定阻尼系数作为临界阻尼的分数"。
6. 选择指定「Tangent fraction」（切向阻尼系数与法向阻尼系数的比值）的选项：- 选择「Use default」以接受默认的切向分数值。对于Abaqus/Standard，默认值为0.0，因此切向方向的阻尼系数为零。对于Abaqus/Explicit，切向分数的默认值为1.0，因此切向方向的阻尼系数等于法向方向的阻尼系数。- 选择「Specify value」以输入切向分数的值。有关更多信息，请参见Abaqus Analysis User's Guide第37.1.3节"接触阻尼"中的"指定切向阻尼系数"。
7. 选择描述间隙与阻尼系数之间关系的曲线形状：- 如果您正在执行Abaqus/Explicit分析，请选择「Step (Explicit only)」。当表面接触时，阻尼系数将保持在指定的常数值，否则为零。- 选择「Linear (Standard only)」以定义在特定间隙值（![](../graphics/usi_eqn00591.gif)）时从零线性增加到表面接触时达到其全值的阻尼系数。- 选择「Bilinear (Standard only)」以定义在特定间隙值（![](../graphics/usi_eqn00591.gif)）时从零线性增加到间隙减小到另一值（![](../graphics/usi_eqn00094.gif)）时达到其全值的阻尼系数。当间隙继续从![](../graphics/usi_eqn00094.gif)减小到零时，阻尼系数保持在其全值不变。
8. 在提供的表格中输入适当的数据：- 如果您正在执行Abaqus/Explicit分析，请输入阻尼系数或临界阻尼分数的值（取决于您在第5步中的选择）。- 如果您正在执行Abaqus/Standard分析，并在上一步中选择了「Linear (Standard only)」，请输入以下内容：- 在第一行中，输入阻尼系数的值。- 在第二行中，输入![](../graphics/usi_eqn00591.gif]的值，即阻尼系数为零的间隙。- 如果您正在执行Abaqus/Standard分析，并在上一步中选择了「Bilinear (Standard only)」，请输入以下内容：- 在第一行中，输入阻尼系数的值。- 在第二行中，输入![](../graphics/usi_eqn00094.gif]的值，即阻尼系数达到其全值的间隙。- 在第三行中，输入![](../graphics/usi_eqn00591.gif]的值，即阻尼系数为零的间隙。
9. 点击「OK」创建接触属性并退出「Edit Contact Property」对话框。或者，您可以从「Edit Contact Property」对话框的菜单中选择另一个接触属性选项进行定义。

#### 为机械接触属性选项指定内聚行为属性

您可以定义将在表面接触相互作用中考虑的内聚行为属性。有关更多信息，请参见Abaqus Analysis User's Guide第37.1.10节的"基于表面的内聚行为"。

此外，您可以通过定义基于断裂的内聚行为表面相互作用来完成裂纹扩展功能的定义。您通过将其分配给最初部分粘合的表面对来激活裂纹扩展。如果满足断裂准则，则裂纹扩展发生在这两个表面之间。内聚行为也用于指定键的弹性行为。

**指定内聚行为接触属性的步骤：**

1. 从主菜单栏选择 ****Interaction**![](../graphics/images/arrow.gif)**Property**![](../graphics/images/arrow.gif)**Create****。
2. 在出现的「Create Interaction Property」对话框中，执行以下操作：- 为相互作用属性命名。有关命名对象的更多信息，请参见"使用基本对话框组件"，第3.2.1节。- 选择「Contact」类型的相互作用属性。
3. 点击「Continue」关闭「Create Interaction Property」对话框。
4. 从接触属性编辑器的主菜单栏中选择 ****Mechanical**![](../graphics/images/arrow.gif)**Cohesive Behavior****。
5. 切换「Allow cohesive behavior during repeated post-failure contacts」以修改渐进损伤定义后的默认失效后行为。默认情况下，一旦这些节点上发生最终失效，从属表面上的节点不会强制执行内聚行为。当此选项被启用时，Abaqus/CAE会在从属表面节点上最终失效后的重复接触处强制执行内聚行为。
6. 从「Eligible Slave Nodes」选项中，选择以下之一：- 选择「Any slave nodes experiencing contact」，不仅为在步骤开始时与主表面接触的从属表面的所有节点定义内聚行为，还为在步骤过程中可能接触的最初未接触的从属节点定义内聚行为。- 选择「Only slave nodes initially in contact」，将内聚行为限制为仅在步骤开始时与主表面接触的从属表面的节点。- 选择「Specify the bonding node set in the surface-to-surface Std interaction」，将内聚行为限制为在指定初始粘合接触条件时定义的要害节点子集。此选项仅适用于Abaqus/Standard分析。
7. 从「Traction-separation Behavior」选项中，接受默认的接触惩罚强制方法，或切换「Specify stiffness coefficients」，并执行以下附加步骤：1. 指定您是否要为「Uncoupled」或「Coupled」牵引行为指定刚度系数。2. 如果牵引-分离行为依赖于温度，切换「Use temperature-dependent data」。3. 点击「Number of field variables」字段右侧的箭头以指定牵引-分离行为所依赖的场变量数量。4. 在提供的数据表中输入所需的数据。
8. 点击「OK」创建接触属性并退出「Edit Contact Property」对话框。或者，您可以从「Edit Contact Property」对话框的菜单中选择另一个接触属性选项进行定义。

#### 为机械接触属性选项指定内聚损伤属性

您可以定义将在表面接触相互作用中考虑的损伤起始、演化和稳定化属性。有关更多信息，请参见Abaqus Analysis User's Guide第37.1.10节的"基于表面的内聚行为"。

**指定机械接触损伤属性的步骤：**

1. 从主菜单栏选择 ****Interaction**![](../graphics/images/arrow.gif)**Property**![](../graphics/images/arrow.gif)**Create****。
2. 在出现的「Create Interaction Property」对话框中，执行以下操作：- 为相互作用属性命名。有关命名对象的更多信息，请参见"使用基本对话框组件"，第3.2.1节。- 选择「Contact」类型的相互作用属性。
3. 点击「Continue」关闭「Create Interaction Property」对话框。
4. 从接触属性编辑器的主菜单栏中选择 ****Mechanical**![](../graphics/images/arrow.gif)**Damage****。
5. 从「Initiation」选项卡页面，执行以下步骤：1. 从「Criterion」列表中，选择以下之一：- 选择「Maximum nominal stress」以指定基于内聚元素最大名义应力准则的损伤起始准则。- 选择「Maximum separation」以指定基于最大分离值的损伤起始准则。- 选择「Quadratic traction」以指定基于内聚元素二次牵引-相互作用准则的损伤起始准则。- 选择「Quadratic separation」以指定基于二次分离-相互作用准则的损伤起始准则。2. 如果损伤起始行为依赖于温度，切换「Use temperature-dependent data」。3. 点击「Number of field variables」字段右侧的箭头以指定损伤起始行为所依赖的场变量数量。4. 在提供的数据表中输入所需的数据。
6. 如果您想指定损伤演化准则；切换「Specify damage evolution」，点击「Evolution」选项卡，并执行以下步骤：1. 从「Type」选项中，选择以下之一：- 选择「Displacement」以将损伤演化定义为失效后总位移（对于内聚元素中的弹性材料）或塑性位移（对于体积弹性-塑性材料）的函数。- 选择「Energy」以将损伤演化定义为损伤起始后失效所需的能量（断裂能）的函数。2. 从「Softening」选项中，选择以下之一：- 选择「Linear」以指定线性软化应力-应变响应（对于线性弹性材料在损伤起始后）或损伤变量随变形的线性演化（对于弹性-塑性材料在损伤起始后）。- 选择「Exponential」以指定指数软化应力-应变响应（对于线性弹性材料在损伤起始后）或损伤变量随变形的指数演化（对于弹性-塑性材料在损伤起始后）。- 选择「Tabular」以表格形式指定损伤变量随变形（损伤起始后）的演化。此选项仅适用于以位移定义的损伤演化。3. 如果您想指定模式相关行为，切换「Specify mixed mode behavior」，并选择以下选项之一：- 选择「Tabular」以直接指定作为内聚元素剪切-法向模式混合函数的断裂能或位移（总计或塑性）。当以位移定义损伤演化时，必须使用此方法为内聚元素指定混合模式行为。- 选择「Power law」以通过幂律混合模式断裂准则将断裂能定义为模式混合的函数。- 选择「Benzeggagh-Kenane」以通过Benzeggagh-Kenane混合模式断裂准则将断裂能定义为模式混合的函数。4. 如果您为混合模式行为选择了「Tabular」，请选择以下之一：- 选择「Energy」以将模式混合定义为不同模式中断裂能的比值。- 选择「Traction」以将模式混合定义为牵引分量的比值。5. 如果您启用了「Specify mixed mode behavior」并选择了「Power law」或「Benzeggagh-Kenane」作为断裂准则，您可以指定幂律或Benzeggagh-Kenane准则中的指数，该准则定义了内聚元素中断裂能随模式混合的变化。切换「Specify power-law/criterion」并在字段中输入指数值。6. 如果损伤演化行为依赖于温度，切换「Use temperature-dependent data」。7. 点击「Number of field variables」字段右侧的箭头以指定损伤演化行为所依赖的场变量数量。8. 在提供的数据表中输入所需的数据。
7. 如果您想指定定义基于表面的内聚行为的本构方程的粘性正则化；切换「Specify damage stabilization」，点击「Stabilization」选项卡，并指定粘度系数。
8. 点击「OK」创建接触属性并退出「Edit Contact Property」对话框。或者，您可以从「Edit Contact Property」对话框的菜单中选择另一个接触属性选项进行定义。

#### 为裂纹扩展指定断裂准则属性

您可以指定使用虚拟裂纹闭合技术（VCCT）在Abaqus/Standard模型中建模裂纹扩展的断裂准则。断裂准则指定临界能量释放率。有关更多信息，请参见Abaqus Analysis User's Guide第11.4.3节的"裂纹扩展分析"。

10. 如果需要，请指定用于粘性正则化的粘度系数。默认值为0.0。
11. 如果您为断裂准则类型选择了「VCCT」，请定义每种模式的能量释放率（对于裂纹起始和裂纹扩展）：![](../graphics/usi_eqn00594.gif)、![](../graphics/usi_eqn00595.gif)和![](../graphics/usi_eqn00596.gif)。

14. 如果您选择了「Power」作为断裂准则，请定义幂律模型中的三个指数，![](../graphics/usi_eqn00601.gif)、![](../graphics/usi_eqn00602.gif)和![](../graphics/usi_eqn00603.gif)。
15. 如果断裂准则依赖于温度，切换「Use temperature-dependent data」。
16. 点击「Number of field variables」字段右侧的箭头以指定断裂准则所依赖的场变量数量。
17. 在提供的数据表中输入所需的数据。
18. 点击「OK」创建接触属性并退出「Edit Contact Property」对话框。或者，您可以从「Edit Contact Property」对话框的菜单中选择另一个接触属性选项进行定义。

#### 为热接触属性选项指定热传导

您可以指定热传导，以定义紧密相邻（或接触）表面之间的传导热传递。有关更多信息，请参见Abaqus Analysis User's Guide第37.2.1节的"热接触属性"中的"建模表面之间的传导"。

**指定热传导的步骤：**

1. 从主菜单栏选择 ****Interaction**![](../graphics/images/arrow.gif)**Property**![](../graphics/images/arrow.gif)**Create****。
2. 在出现的「Create Interaction Property」对话框中，执行以下操作：- 为相互作用属性命名。有关命名对象的更多信息，请参见"使用基本对话框组件"，第3.2.1节。- 选择「Contact」类型的相互作用属性。
3. 点击「Continue」关闭「Create Interaction Property」对话框。
4. 从接触属性编辑器的主菜单栏中选择 ****Thermal**![](../graphics/images/arrow.gif)**Thermal Conductance****。会出现「Edit Contact Property」对话框。
5. 在出现的编辑器中，点击「Definition」字段右侧的箭头，选择定义热传导的方式：- 选择「Tabular」以输入将热传导与接触表面之间的间隙或压力相关联的数据。- 选择「User-defined」以在用户子程序[`GAPCON`](../sub/sub-link.md#sub-xsl-gapcon)中定义热传导。如果您选择此选项，请跳到第9步。
6. 指示您是想将热传导定义为表面之间间隙的函数、接触表面之间压力的函数，还是两者。
7. 如果您想将热传导定义为间隙的函数，请显示「Clearance Dependency」选项卡页面，并执行以下操作：1. 如果数据依赖于温度，切换「Use temperature-dependent data」。2. 如果数据依赖于平均质量流量每单位面积，![](../graphics/usi_eqn00604.gif)，切换「Use mass flow rate-dependent data (Standard only)」。3. 点击「Number of field variables」字段右侧的箭头以指定数据所依赖的场变量数量。4. 在数据表中，将热传导定义为间隙的函数。表格数据必须从零间隙（闭合间隙）开始，并随着间隙增加定义热传导。您必须提供至少两对点。热传导值在最后一个数据点之后立即降为零，因此当间隙大于与最后一个数据点对应的值时，没有热传导。如果热传导也被定义为压力的函数，它将对于所有压力保持在零间隙值不变。
8. 如果您想将热传导定义为接触压力的函数，请显示「Pressure Dependency」选项卡页面，并执行以下操作：1. 如果数据依赖于温度，切换「Use temperature-dependent data」。2. 如果数据依赖于平均质量流量每单位面积，![](../graphics/usi_eqn00604.gif)，切换「Use mass flow rate-dependent data (Standard only)」。3. 点击「Number of field variables」字段右侧的箭头以指定数据所依赖的场变量数量。4. 在数据表中，将热传导定义为界面处接触压力的函数。表格数据必须从零接触压力开始（或者，在可以承受拉力的接触情况下，从最负压力的数据点开始），并随着压力增加定义热传导。热传导值对于数据点定义区间之外的接触压力保持不变。如果热传导也没有定义为间隙的函数，则对于所有正间隙值，热传导为零，在零间隙处不连续。
9. 点击「OK」创建接触属性并退出「Edit Contact Property」对话框。或者，您可以从「Edit Contact Property」对话框的菜单中选择另一个接触属性选项进行定义。

#### 为热接触属性选项指定热量产生

您可以指定由于机械或电相互作用产生的能量耗散而引起的热量产生有关更多信息，请参见Abaqus Analysis User's Guide第37.2.1节的"热接触属性"中的"建模由非热表面相互作用产生的热量"。

**指定热量产生的步骤：**

1. 从主菜单栏选择 ****Interaction**![](../graphics/images/arrow.gif)**Property**![](../graphics/images/arrow.gif)**Create****。
2. 在出现的「Create Interaction Property」对话框中，执行以下操作：- 为相互作用属性命名。有关命名对象的更多信息，请参见"使用基本对话框组件"，第3.2.1节。- 选择「Contact」类型的相互作用属性。
3. 点击「Continue」关闭「Create Interaction Property」对话框。
4. 从接触属性编辑器的主菜单栏中选择 ****Thermal**![](../graphics/images/arrow.gif)**Heat Generation****。
5. 在出现的编辑器中，指定**由摩擦或电流产生的并转换为热量的耗散能量的分数**：- 选择「Use default (1.0)」以将所有耗散能量转换为热量。- 选择「Specify」以输入您选择的分数。
6. 指定**转换热量分配给从属表面的分数**：- 选择「Use default (0.5)」以将热量平均分配给主表面和从属表面。- 选择「Specify」以输入分配给从属表面的热量分数。剩余分数将分配给主表面。
7. 点击「OK」创建接触属性并退出「Edit Contact Property」对话框。或者，您可以从「Edit Contact Property」对话框的菜单中选择另一个接触属性选项进行定义。

#### 为热接触属性选项指定辐射

您可以指定紧密相邻表面之间的辐射热传递。有关更多信息，请参见Abaqus Analysis User's Guide第37.2.1节的"热接触属性"中的"当间隙小时建模表面之间的辐射"。

**指定辐射的步骤：**

1. 从主菜单栏选择 ****Interaction**![](../graphics/images/arrow.gif)**Property**![](../graphics/images/arrow.gif)**Create****。
2. 在出现的「Create Interaction Property」对话框中，执行以下操作：- 为相互作用属性命名。有关命名对象的更多信息，请参见"使用基本对话框组件"，第3.2.1节。- 选择「Contact」类型的相互作用属性。
3. 点击「Continue」关闭「Create Interaction Property」对话框。
4. 从接触属性编辑器的主菜单栏中选择 ****Thermal**![](../graphics/images/arrow.gif)**Radiation****。
5. 在出现的编辑器中，输入主表面和从属表面的发射率，![](../graphics/usi_eqn00171.gif)。
6. 在提供的表格中，将视角系数定义为间隙的函数。视角系数的值应在0.0到1.0之间。至少需要两对点。表格数据必须从零间隙（闭合间隙）开始，并随着间隙增加定义视角系数。视角系数值在最后一个数据点之后立即降为零，因此当间隙大于与最后一个数据点对应的值时，没有辐射热传递。
7. 点击「OK」创建接触属性并退出「Edit Contact Property」对话框。或者，您可以从「Edit Contact Property」对话框的菜单中选择另一个接触属性选项进行定义。

### 定义电接触属性选项

您可以定义电接触属性选项来指定间隙电导。

#### 为电接触属性选项指定间隙电导

您可以指定紧密相邻或接触表面之间的间隙电导。电导与界面两侧电势差的差值成正比。电导是表面之间间隙（分离）的函数，也可以是接触压力的函数。有关更多信息，请参见Abaqus Analysis User's Guide第37.3.1节的"电接触属性"。

**指定电间隙电导的步骤：**

1. 从主菜单栏选择 ****Interaction**![](../graphics/images/arrow.gif)**Property**![](../graphics/images/arrow.gif)**Create****。
2. 在出现的「Create Interaction Property」对话框中，执行以下操作：- 为相互作用属性命名。有关命名对象的更多信息，请参见"使用基本对话框组件"，第3.2.1节。- 选择「Contact」类型的相互作用属性。
3. 点击「Continue」关闭「Create Interaction Property」对话框。
4. 从接触属性编辑器的主菜单栏中选择 ****Electrical**![](../graphics/images/arrow.gif)**Electrical Conductance****。会出现「Edit Contact Property」对话框。
5. 在出现的编辑器中，点击「Definition」字段右侧的箭头，选择定义电导的方式：- 选择「Tabular」以输入将电导与接触表面之间的分离相关联的数据。- 选择「User-defined」以在用户子程序[`GAPELECTR`](../sub/sub-link.md#sub-xsl-gapelectr)中定义电导。如果您选择此选项，请跳到第9步。
6. 指示您是想将电导定义为表面之间间隙的函数、接触表面之间压力的函数，还是两者。
7. 如果您想将电导定义为间隙的函数，请显示「Clearance Dependency」选项卡页面，并执行以下操作：1. 如果数据依赖于温度，切换「Use temperature-dependent data」。2. 点击「Number of field variables」字段右侧的箭头以指定数据所依赖的场变量数量。3. 在数据表中，将电导定义为间隙的函数。表格数据必须从零间隙（闭合间隙）开始，并随着间隙增加定义电导。您必须提供至少两对点。电导值在最后一个数据点之后立即降为零，因此当间隙大于与最后一个数据点对应的值时，没有电导。如果电导也没有定义为压力的函数，它将对于所有压力保持在零间隙值不变。
8. 如果您想将电导定义为接触压力的函数，请显示「Pressure Dependency」选项卡页面，并执行以下操作：1. 如果数据依赖于温度，切换「Use temperature-dependent data」。2. 点击「Number of field variables」字段右侧的箭头以指定数据所依赖的场变量数量。3. 在数据表中，将电导定义为界面处接触压力的函数。表格数据必须从零接触压力开始（或者，在可以承受拉力的接触情况下，从最负压力的数据点开始），并随着压力增加定义电导。电导值对于数据点定义区间之外的接触压力保持不变。如果电导也没有定义为间隙的函数，则对于所有正间隙值，电导为零，在零间隙处不连续。
9. 点击「OK」创建接触属性并退出「Edit Contact Property」对话框。或者，您可以从「Edit Contact Property」对话框的菜单中选择另一个接触属性选项进行定义。
