# 17.14.3 零件实例之间的兼容网格





您无法在零件实例之间指定兼容的网格。如果您需要在两个或多个实例之间保持网格兼容性，您可以执行以下操作之一：
- 创建一个包含所有实体的单一零件，这样就不需要多个实例。
- 在"装配"模块中装配零件实例，并使用**合并/切割**工具将实例合并为单个实例。如果您需要保持独立零件实例的概念，您可以在合并实例的公共界面上创建分割。同样，您可以使用**合并/切割**工具从包含孤立单元和节点的多个实例创建单个零件实例，并合并重复的节点。更多信息，请参见["对零件实例执行布尔操作，"第13.7节](pt03ch13s07.md)。
- 如果您必须使用单独的零件，您可以使用绑定接触来避免网格兼容性问题。请记住，这不是真正的兼容性，解决方案的准确性可能会受到影响。有关绑定接触的更多信息，请参见["理解相互作用，"第15.3节](pt03ch15s03.md)。
