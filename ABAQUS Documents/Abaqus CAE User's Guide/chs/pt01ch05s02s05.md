# 5.2.5 框选缩放工具





当你选择框选缩放工具 ![](../graphics/ico_viewZoomRectangle.png) 和要工作的视口时，Abaqus/CAE 进入框选缩放模式，如十字光标所示。你使用此工具选择模型或绘图的矩形区域；Abaqus/CAE 将你选择的模型或绘图部分的视图放大以填充视口。对于 *X–Y* 绘图，Abaqus/CAE 会放大你选择的 *X–Y* 曲线的视图，并更新轴值以匹配你选择的数据。
