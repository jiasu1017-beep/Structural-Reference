# 13.12 使用查询工具集查询装配





从主菜单栏选择**工具**![](../graphics/images/arrow.gif)**查询**来启动查询工具集。您可以使用查询工具集来请求常规信息或特定模块信息。有关常规查询显示的信息的讨论，请参见["获取模型常规信息，"第71.2.2节](pt06ch71s02hlb02.md)。此外，您可以使用查询工具集中的装配模块特定工具来确定所选零件或模型实例的属性和位置。

**要查询装配：**

1. 从主菜单栏，选择**工具**![](../graphics/images/arrow.gif)**查询**。**提示：**您也可以在**查询**工具栏中选择![](../graphics/ico_info.png)工具。Abaqus/CAE显示**查询**对话框。
2. 从**查询**对话框中，点击以下之一：**实例属性**选择一个零件或模型实例。Abaqus/CAE在消息区域显示以下内容：
   - 实例的名称、建模空间和类型（可变形或刚性，从属或独立）
   **实例位置**选择一个零件或模型实例。Abaqus/CAE在消息区域显示以下内容：
   - 实例原点相对于全局坐标系的位置
   - 相对于装配全局坐标系应用到实例的旋转总和
   - 应用到实例的约束列表
3. 查询完装配后，关闭**查询**对话框。
