### 概述

["延性金属的损伤和失效：概述，" 第24.2.1节](pt05ch24s02abm41.md)中描述的渐进损伤和失效模型是Abaqus中建模材料损伤和失效的推荐方法；这些模型适用于准静态和动态情况。Abaqus/Explicit提供了两种额外的仅适用于高应变率动态问题的单元失效模型。剪切失效模型由塑性屈服驱动。拉伸失效模型由拉伸加载驱动。这些失效模型可用于限制单元在达到应力极限后（直到移除单元）的后续载荷承载能力。两种模型可以用于同一材料。

剪切失效模型：
- 专为许多材料（包括大多数金属）的高应变率变形而设计；
- 使用等效塑性应变作为失效度量；
- 提供两种失效后处理选择，包括从网格中移除单元；
- 可与Mises或Johnson-Cook塑性模型结合使用；并且
- 可与拉伸失效模型结合使用。

拉伸失效模型：
- 专为许多材料（包括大多数金属）的高应变率变形而设计；
- 使用静水压力应力作为失效度量来模拟动态层裂或压力截止；
- 提供多种失效后处理选择，包括从网格中移除单元；
- 可与Mises或Johnson-Cook塑性模型或状态方程材料模型结合使用；并且
- 可与剪切失效模型结合使用。

#### 单元移除

当在积分点满足剪切失效准则时，所有应力分量将被设置为零，该材料点失效。默认情况下，如果单元任何截面上的所有材料点都失效，则从网格中移除该单元；不需要单元中的所有材料点都失效。例如，在一阶减缩积分实体单元中，一旦其唯一积分点失效，就会移除单元。然而，在壳单元中，必须通过厚度的所有积分点都失效后才能从网格中移除单元。对于二阶减缩积分梁单元，默认情况下，梁轴上两个单元积分位置中任一位置的截面通过的所有积分点的失效都会导致单元移除。类似地，在改进的三角形和四面体实体单元中，任何一个积分点的失效都会默认导致单元移除。单元删除是默认的失效选择。

一种替代的失效选择是，当材料点满足剪切失效准则时，不删除单元，而是将偏应力分量设置为零，并且将在计算的剩余部分保持为零。然后要求压力应力保持为压缩；也就是说，如果在增量中在失效材料点计算出负压力应力，则将其重置为零。当使用平面应力、壳、膜、梁、管道和桁架单元时，不允许此失效选择，因为可能会违反结构约束。

| **输入文件用法：** | 使用以下选项在满足失效准则时允许单元删除（默认）： |
| --- | --- |
|  | ``` [*SHEAR FAILURE](../key/key-link.md#usb-kws-mshearfailure), ELEMENT DELETION=YES ``` 使用以下选项在满足失效准则时允许单元仅承受静水压缩应力： ``` [*SHEAR FAILURE](../key/key-link.md#usb-kws-mshearfailure), ELEMENT DELETION=NO ``` |

#### 确定何时使用剪切失效模型

Abaqus/Explicit中的剪切失效模型适用于惯性很重要的高应变率动态问题。剪切失效模型的不当使用可能导致模拟不正确。

对于可能需要单元移除的准静态问题，推荐使用渐进损伤和失效模型（[第24章，"渐进损伤和失效"](pt05ch24.md)"）或多孔金属塑性模型（["多孔金属塑性，" 第23.2.9节](pt05ch23s02abm25.md)）。

### 拉伸失效模型

拉伸失效模型可与Abaqus/Explicit中的Mises或Johnson-Cook塑性模型或状态方程材料模型结合使用来定义材料的拉伸失效。

#### 拉伸失效准则

Abaqus/Explicit拉伸失效模型使用静水压力应力作为失效度量来模拟动态层裂或压力截止。拉伸失效准则假定当压力应力*p*变得比用户指定的静水截止应力![](../graphics/usb_eqn03179.gif)更具拉伸性时发生失效。静水截止应力可以是温度和预定义场变量的函数。此应力没有默认值。

拉伸失效模型可与Mises或Johnson-Cook塑性模型或状态方程材料模型结合使用。

| **输入文件用法：** | 对于Mises或Johnson-Cook塑性模型使用以下两个选项： |
| --- | --- |
|  | ``` [*PLASTIC](../key/key-link.md#usb-kws-mplastic) [*TENSILE FAILURE](../key/key-link.md#usb-kws-mtensilefailure) ``` 对于状态方程材料模型使用以下两个选项： ``` [*EOS](../key/key-link.md#usb-kws-meos) [*TENSILE FAILURE](../key/key-link.md#usb-kws-mtensilefailure) ``` |

#### 失效选择

当在单元积分点满足拉伸失效准则时，材料点失效。为失效材料点提供了五种失效选择：包括单元移除的默认选择，以及四种不同的层裂模型。这些失效选择在下面描述。

##### 单元移除

当在积分点满足拉伸失效准则时，所有应力分量将被设置为零，该材料点失效。默认情况下，如果单元任何截面上的所有材料点都失效，则从网格中移除该单元；不需要单元中的所有材料点都失效。例如，在一阶减缩积分实体单元中，一旦其唯一积分点失效，就会移除单元。然而，在壳单元中，必须通过厚度的所有积分点都失效后才能从网格中移除单元。对于二阶减缩积分梁单元，默认情况下，梁轴上两个单元积分位置中任一位置的截面通过的所有积分点的失效都会导致单元移除。类似地，在改进的三角形和四面体实体单元中，任何一个积分点的失效都会默认导致单元移除。

| **输入文件用法：** | ``` [*TENSILE FAILURE](../key/key-link.md#usb-kws-mtensilefailure), ELEMENT DELETION=YES (default) ``` |
| --- | --- |

#### 确定何时使用拉伸失效模型

Abaqus/Explicit中的拉伸失效模型适用于惯性效应重要的高应变率动态问题。拉伸失效模型的不当使用可能导致模拟不正确。

### 将失效模型与钢筋一起使用

可以将剪切失效模型和/或拉伸失效模型与也定义了钢筋的单元一起使用。当这些单元根据失效准则失效时，根据所选失效类型，基础材料对单元应力承载能力的贡献被移除或调整，但对单元应力承载能力的钢筋贡献不会被移除。但是，如果您还在钢筋材料定义中包含失效，则如果满足为钢筋指定的失效准则，对单元应力承载能力的钢筋贡献也将被移除或调整。

### 单元

带单元删除的剪切和拉伸失效模型可与Abaqus/Explicit中包含力学行为（具有位移自由度）的任何单元一起使用。不带单元删除的剪切和拉伸失效模型只能与Abaqus/Explicit中的平面应变、轴对称和三维实体（连续体）单元一起使用。
