# 12.7.8 显示粘弹性材料行为的X–Y图





### 12.7.8 显示*X--Y*粘弹性材料行为图

Abaqus/CAE允许您通过基于材料定义创建松弛和蠕变曲线（对于Prony级数系数、松弛测试数据或蠕变测试数据）或剪切和体积模量曲线（对于频率数据）来评估粘弹性材料行为。曲线拟合完成后，Abaqus/CAE打开Visualization模块并显示测试结果的*X–Y*图以及包含材料参数的对话框。您可以查看结果并根据需要调整材料。有关更多信息，请参见["评估超弹性和粘弹性材料行为，"第12.4.7节"](pt03ch12s04s07.md)。

**显示粘弹性材料行为的*X–Y*图：**

1. 从主菜单栏，选择****Material**![](../graphics/images/arrow.gif)**Evaluate**![](../graphics/images/arrow.gif)***material name*****。您选择的材料必须包含与超弹性和/或弹性材料数据结合定义的时域粘弹性材料数据。
   **提示：**您也可以在**Material Manager**中选择材料名称，然后点击**Evaluate**。出现**Evaluate Material**对话框。
2. 如果您选择的粘弹性材料也包含超弹性材料数据，请点击**Viscoelastic**标签；如果尚未选择，请切换**Perform viscoelastic evaluation**。如有需要，您也可以评估材料的超弹性行为。有关更多信息，请参见["显示超弹性材料行为的*X--Y*图，"第12.7.7节"](pt03ch12s07hlb07.md)。
3. 在**Available Input Data**字段中，执行以下操作：
   1. 选择您选择的**Source**选项：
      - 如果您希望Abaqus使用材料定义中指定的实验数据计算粘弹性响应，请选择**Test data**。
      - 如果您希望Abaqus使用材料定义中指定的系数计算粘弹性响应，请选择**Coefficients**。如果材料是使用Prony级数、用于时间的松弛测试数据或蠕变测试数据定义的，Abaqus使用超弹性或弹性系数数据。如果材料是使用频率数据定义用于时间，Abaqus使用粘弹性材料定义中指定的频率系数。
   2. 如果在上一步中选择了**Test data**，请切换您希望Abaqus用于计算材料响应的测试数据类型。（只有您在材料定义中为其指定了数据的数据类型会出现在列表中。）
      **注意：****Combined**数据不能与**Shear**或**Volumetric**数据同时选择。
4. 在**Normalized Response Plots**字段中，切换**Stress Relaxation**和/或**Creep**来定义Abaqus将计算的响应模式；并输入归一化响应曲线的时间段。如果粘弹性是使用时域中的频率数据定义的，则**Normalized Response Plots**字段不可用。相反，Abaqus会产生对数频率刻度上的剪切和体积模量响应曲线。
   **注意：**当使用时域频率数据评估粘弹性材料时，Abaqus通过将Prony级数项从时域转换到频域来获得剪切和体积模量的表达式。建议您独立验证数据将用于的域中的材料模型。有关更多信息，请参见Abaqus Analysis User's Guide的第22.7.1节"时域粘弹性"中的"粘弹性材料参数的确定"。
5. 点击**OK**开始响应计算。如果在提取材料系数期间评估因非线性曲线拟合问题而失败，Abaqus/CAE会显示一个包含数据（`.dat`）文件名称的对话框；数据文件的路径打印在消息区域中。数据文件提供有关每个遇到的问题的详细信息。（有关数据文件的更多信息，请参见Abaqus Analysis User's Guide的第4.1.1节"输出"。）如果Abaqus成功完成测试，Abaqus/CAE进入Visualization模块并在新的视口中显示测试结果的*X--Y*图。（有关*X--Y*图的信息，请参见[第47章，"X--Y绘图"](pt05ch47.md)。"）数据对象出现在***X--Y* Data Manager**中；您可以将它们复制到输出数据库，或执行您在Visualization模块中对其他*X--Y*数据可以执行的任何任务。此外，Abaqus/CAE显示一个信息对话框，其中包含粘弹性材料参数，如果执行了超弹性评估，还包含每个超弹性应变能势的稳定性限值和系数。Abaqus/CAE在消息区域显示包含所有材料评估信息的数据（`.dat`）文件的路径。
6. 如有需要，返回Property模块编辑材料数据或评估其他材料。
