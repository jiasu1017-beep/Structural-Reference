# 10.4.1 对称模型生成



**产品：** Abaqus/Standard

**参考资料**

- [*SYMMETRIC MODEL GENERATION](../key/key-link.md#usb-kws-maximodelgen)

### 概述

在Abaqus/Standard中，可以通过以下方式创建三维模型：
- 绕其旋转轴线旋转轴对称模型；
- 绕其对称轴线旋转单个三维扇区；或
- 组合对称三维模型的两个部分，其中一个部分是原始模型，另一部分通过沿对称线或对称平面对原始模型进行反射获得。

Abaqus/Standard还提供将原始分析中获得的结果转换到新模型的功能（参见["将结果从对称网格或部分三维网格转换到完整三维网格，" 第10.4.2节](pt04ch10s04aus64.md)）。

只能使用应力/位移、热传递、耦合温度-位移和声学元素来生成新模型。

### 模型生成

对称模型生成功能可用于通过绕旋转轴线旋转轴对称二维模型来创建三维模型，通过绕对称轴线旋转单个三维扇区来创建三维模型，或通过组合对称模型的两个部分来创建三维模型，其中一个部分是原始模型，另一个部分是沿一条线或一个平面对原始模型进行反射得到的。原始模型必须保存到重启动文件。对称模型生成功能不适用于定义为部件实例组合的模型。因此，不支持包含引号的元素集名称或节点集名称。

整个三维模型（包括节点、单元、截面定义、材料和方向定义、钢筋以及接触对定义）是从原始模型生成的。不允许从具有一般接触的模型进行对称模型生成。您必须重新定义大多数类型的运动约束（["运动约束：概述，" 第35.1.1节](pt08ch35s01abo32.md)）。但是，原始模型中定义的基于表面的约束（["网格绑定约束，" 第35.3.1节](pt08ch35s03aus132.md)）和嵌入单元约束（["嵌入单元，" 第35.4.1节](pt08ch35s04aus136.md)）将在新三维模型中自动生成。历史数据中对模型所做的更改（如单元或接触对的移除/重新激活（["单元和接触对的移除与重新激活，" 第11.2.1节](pt04ch11s02aus66.md)）或摩擦属性的更改（["在Abaqus/Standard分析中更改摩擦属性" "摩擦行为，" 第37.1.5节](pt09ch37s01aus169.md#usb-cni-afriction-change-std)））不会转换到新模型。这些更改必须在新的历史数据中重新定义。原始模型中定义的所有单元集和节点集将用于新模型。这些集合将包含源自原始集合的所有新单元和节点。

还可以定义附加节点、单元、接触面等，以创建原始模型中未指定的模型部分。您必须确保这些节点和单元的编号与对称模型生成功能使用的编号不冲突。您可以控制新模型中的节点和单元编号（如下所述针对每种旋转模型类型），以便可以定义模型的附加部分而不会产生单元和节点标签冲突的风险。定义模型附加部分所用的小节点/单元编号应大于对称模型生成功能生成的大节点/单元编号。

#### 消除重复节点

在某些情况下会生成重复节点。可以消除这些重复节点以确保网格正确连接。重复节点可以在旋转模型的旋转轴线上、在周期模型扇区之间的连接平面上，以及在反射模型两部分之间的连接平面上生成。您可以指定用于搜索重复节点的容差距离*d*。默认距离为平均单元尺寸的1.0%。在某些情况下，如果原始周期模型某个连接平面上两个节点之间的距离小于默认容差距离，则需要指定小于默认值的容差距离。模型其他位置（如界面表面之间或使用任何其他模型定义选项生成的模型部分）中的紧密间距节点不会被消除。

| **输入文件用法：** | 使用以下选项之一指定用于搜索重复节点的容差： |
| --- | --- |
|  | ``` [*SYMMETRIC MODEL GENERATION](../key/key-link.md#usb-kws-maximodelgen), PERIODIC, TOLERANCE=*d* [*SYMMETRIC MODEL GENERATION](../key/key-link.md#usb-kws-maximodelgen), REVOLVE, TOLERANCE=*d* [*SYMMETRIC MODEL GENERATION](../key/key-link.md#usb-kws-maximodelgen), REFLECT, TOLERANCE=*d* ``` |

#### 将新模型定义写入外部文件

您可以指定外部文件的名称（无扩展名），新模型定义的数据将写入该文件。扩展名`.axi`将被添加到提供的文件名中。可以编辑该文件以修改或扩展Abaqus/Standard生成的模型。

| **输入文件用法：** | 使用以下选项之一： |
| --- | --- |
|  | ``` [*SYMMETRIC MODEL GENERATION](../key/key-link.md#usb-kws-maximodelgen), PERIODIC, FILE NAME=*name* [*SYMMETRIC MODEL GENERATION](../key/key-link.md#usb-kws-maximodelgen), REVOLVE, FILE NAME=*name* [*SYMMETRIC MODEL GENERATION](../key/key-link.md#usb-kws-maximodelgen), REFLECT, FILE NAME=*name* ``` |

#### 识别重启动文件

对称模型生成功能使用旧模型的重启动（`.res`）、分析数据库（`.stt`和`.mdl`）以及部件（`.prt`）文件来生成新模型。运行Abaqus时必须使用**oldjob**参数指定旧模型的重启动文件名，或者通过命令过程的请求来指定（参见["Abaqus/Standard、Abaqus/Explicit和Abaqus/CFD执行，" 第3.2.2节](pt01ch03s02abx02.md)）。

#### 验证新模型

建议在执行分析之前仔细验证新模型。对称模型生成功能只需要在数据检查运行期间存储在重启动文件中的信息来生成新模型，这使您可以在对原始模型执行分析之前验证新模型。通过在运行Abaqus的命令中使用**datacheck**参数来执行数据检查分析（参见["Abaqus/Standard、Abaqus/Explicit和Abaqus/CFD执行，" 第3.2.2节](pt01ch03s02abx02.md)）。

### 旋转轴对称横截面

您可以通过绕从指定参考平面![](../graphics/usb_eqn01597.gif)开始的对称轴旋转二维轴对称模型的横截面来创建三维模型。新三维模型的对称轴和参考平面可以相对于全局坐标系以任何方向定向（参见[图10.4.1-1](pt04ch10s04aus63.md#ksymm-model-gen-revolve)）。可以指定圆周方向的不均匀离散化。

**图10.4.1-1** 旋转轴对称横截面。

![](../graphics/ksymm-model-gen-revolve-nls.png)

指定[图10.4.1-1](pt04ch10s04aus63.md#ksymm-model-gen-revolve)中所示点a、b和c的坐标，然后定义圆周方向离散化的列表，包括段角度、每段单元数以及段的偏置比。可以使用多个段角度，每个段角度具有不同的单元细分数量和不同的偏置比，来定义旋转模型圆周方向的完整离散化。通过360.0度旋转的横截面的端点将始终连接到旋转原点![](../graphics/usb_eqn01598.gif)，无论为重复节点容差指定的值如何。

| **输入文件用法：** | ``` [*SYMMETRIC MODEL GENERATION](../key/key-link.md#usb-kws-maximodelgen), REVOLVE ``` |
| --- | --- |

#### 局部方向系统

对于应力、应变等的单元输出，始终使用局部圆柱方向系统。如果原始轴对称模型中的材料不包含方向定义，则提供默认局部方向定义。此默认方向定义为系统的极轴沿旋转轴线，并额外绕局部1方向旋转90.0度，使得局部轴为1=径向、2=轴向、3=周向。如果使用壳或膜，则将局部2轴和3轴投影到壳或膜的表面上作为表面上的局部方向。即使在原始轴对称模型中指定了方向，也会始终提供此方向系统。但是，如果将轴对称分析的结果映射到新三维模型（参见["将结果从对称网格或部分三维网格转换到完整三维网格，" 第10.4.2节](pt04ch10s04aus64.md)），并且原始模型中的材料关联了方向定义，则沿对称轴旋转原始方向定义来替换此默认方向定义。

#### 控制新节点和单元编号

您可以定义三维模型圆周方向上每个节点和单元之间的编号增量。编号从参考横截面![](../graphics/usb_eqn01597.gif)开始。参考横截面使用与原始轴对称模型相同的编号。默认值是原始轴对称模型中使用的最大节点和单元编号。对编号的控制使您可以定义模型的附加部分而不会产生单元和节点标签冲突的风险。每个偏移值应分别大于或等于原始模型中使用的最大节点或单元标签。在指定偏移值时，必须注意生成的节点或单元不超过允许的最大值，即999,999,999。

| **输入文件用法：** | ``` [*SYMMETRIC MODEL GENERATION](../key/key-link.md#usb-kws-maximodelgen), REVOLVE, NODE OFFSET=*offset*, ELEMENT OFFSET=*offset* ``` |
| --- | --- |

#### 轴对称与三维单元之间的对应关系

原始二维模型中使用的单元类型决定了新三维模型中的单元类型。您可以指定新单元是通用三维单元还是圆柱单元。通用单元和圆柱单元可以在同一模型中使用。

| **输入文件用法：** | ``` [*SYMMETRIC MODEL GENERATION](../key/key-link.md#usb-kws-maximodelgen), REVOLVE *a点和b点的坐标* *c点的坐标* *段角度、每段单元数、偏置比*, CYLINDRICAL或GENERAL ``` |
| --- | --- |
|  | 例如，以下输入在300度段中指定4个圆柱单元，在60度段中指定10个通用单元： ``` [*SYMMETRIC MODEL GENERATION](../key/key-link.md#usb-kws-maximodelgen), REVOLVE *ax, ay, az, bx, by, bz* *cx, cy, cz* 300.0, 4, 1.0, CYLINDRICAL 60.0, 10, 1.0, GENERAL ``` |

#### 限制条件

- 一阶和二阶单元不能在轴对称模型中一起使用。
- 弹簧、阻尼器、梁和桁架等非轴对称单元将在模型生成中被忽略。
- 只有基于表面的接触对可以旋转。使用一般接触的模型不能旋转。使用接触单元建模的接触条件将在模型生成中被忽略。
- 包含不兼容模式单元、一阶减缩积分连续单元、壳单元或刚性单元的二维模型不能用于生成圆柱单元。
- 径向方向钢筋间距不均匀的轴对称单元不能旋转。
- 大多数类型的运动约束不能旋转。但是，原始模型中定义的基于表面的约束（["网格绑定约束，" 第35.3.1节](pt08ch35s03aus132.md)）和嵌入单元约束（["嵌入单元，" 第35.4.1节](pt08ch35s04aus136.md)）将在新三维模型中自动生成。
- 只有应力/位移、热传递、耦合温度-位移和声学单元可以旋转。

### 旋转三维扇区以创建周期模型

您可以通过绕对称轴线旋转单个三维扇区来创建三维周期模型。周期模型中每个生成的扇区可以在圆周方向上跨越相同的角度（如通风盘），或者可以具有可变角度（如带胎面的轮胎）。在这两种情况下，每个扇区始终具有相同的几何形状和网格。对称轴和原始三维扇区都可以相对于全局坐标系以任何方向定向（参见[图10.4.1-2](pt04ch10s04aus63.md#ksymm-model-gen-per)）。扇区之间可以使用不匹配的表面网格。可以生成开放（结构有端边缘）或闭合循环周期结构。如果需要创建闭合循环周期结构，则所有扇区的段角度之和必须等于360。

**图10.4.1-2** 旋转三维扇区以形成周期模型。

![](../graphics/ksymm-model-gen-per2.png)

#### 定义具有恒定角度的周期模型

要定义具有恒定角度的周期模型，您必须指定[图10.4.1-2](pt04ch10s04aus63.md#ksymm-model-gen-per)中所示点a和b的坐标来定义对称轴。然后，您需要定义原始扇区的段角度![](../graphics/usb_eqn00123.gif)（以度为单位）以及生成周期模型中包括原始扇区在内的三维重复扇区数量*N*。

| **输入文件用法：** | ``` [*SYMMETRIC MODEL GENERATION](../key/key-link.md#usb-kws-maximodelgen), PERIODIC=CONSTANT *a点和b点的坐标* *θ, N* ``` |
| --- | --- |

#### 定义具有可变角度的周期模型

要定义具有可变角度的周期模型，原始扇区两侧的表面必须完全平面。您指定[图10.4.1-2](pt04ch10s04aus63.md#ksymm-model-gen-per)中所示点a和b的坐标来定义对称轴。然后定义原始扇区的段角度θ（以度为单位）以及生成周期模型中包括原始扇区在内的三维重复扇区数量*N*。接下来，您需要指定要生成的额外三维扇区数量*M*，以及相对于原始扇区应用于这些额外扇区的圆周方向角度缩放因子*f*。您可以根据需要定义额外的扇区和缩放因子对。

| **输入文件用法：** | ``` [*SYMMETRIC MODEL GENERATION](../key/key-link.md#usb-kws-maximodelgen), PERIODIC=VARIABLE *a点和b点的坐标* *θ, N* *M1, f1* *M2, f2* *等* ``` |
| --- | --- |
|  | 例如，以下输入创建一个210度三维模型，包含7个扇区，角度分别为20、20、30、30、30、40和40度： ``` [*SYMMETRIC MODEL GENERATION](../key/link.md#usb-kws-maximodelgen), PERIODIC=VARIABLE *ax, ay, az, bx, by, bz* 20.0,2 3,1.5 2,2.0 ``` |

#### 对具有不匹配网格的对称表面应用约束

如果原始扇区中的对称表面具有精确匹配的网格（参见[图10.4.1-3](pt04ch10s04aus63.md#ksymm-model-gen-per-map)），则在绕对称轴线旋转原始扇区以创建周期模型时，自动消除生成的重复节点以确保相邻扇区之间的网格正确连接。

**图10.4.1-3** 原始扇区上具有精确匹配网格的表面。

![](../graphics/ksymm-model-gen-per-map-nls.png)

在所有其他情况下，您必须在原始模型中定义原始扇区每侧的一对或多对相应表面（参见["表面：概述，" 第2.3.1节](pt01ch02s03aus16.md)），并在对称模型生成定义中指定这些相应表面对。

或者，您还可以指定一个容差距离，在此距离内一个扇区表面上的节点必须位于相邻扇区相应表面的范围内，以便受到约束。距相邻扇区相应表面超过此距离的扇区表面上的节点将不受约束。容差距离的默认值是原始扇区表面中典型单元尺寸的5%或10%，分别取决于使用的是节点-表面还是表面-表面类型的约束。

您还可以指定是使用表面-表面（默认）还是节点-表面约束。然后，当绕对称轴线旋转原始扇区以创建周期模型时，在自动生成的相邻表面对之间应用约束（参见["网格绑定约束，" 第35.3.1节](pt08ch35s03aus132.md)）。每对的第一个表面是从属表面，该表面中所有节点的自由度将通过内部生成的多点约束被消除。

| **输入文件用法：** | 在原始模型中使用以下选项： |
| --- | --- |
|  | ``` [*SURFACE](../key/key-link.md#usb-kws-msurface), NAME=*master* [*SURFACE](../key/key-link.md#usb-kws-msurface), NAME=*slave* ``` 在每个扇区具有恒定角度的新模型中使用以下选项： ``` [*SYMMETRIC MODEL GENERATION](../key/key-link.md#usb-kws-maximodelgen), PERIODIC=CONSTANT *ax, ay, az, bx, by, bz* *θ, N* *slave, master, 容差距离,* SURFACE或NODE ``` 在每个扇区具有可变角度的新模型中使用以下选项： ``` [*SYMMETRIC MODEL GENERATION](../key/key-link.md#usb-kws-maximodelgen), PERIODIC=VARIABLE *ax, ay, az, bx, by, bz* *θ, N* *M, f* *slave, master, 容差距离,* SURFACE或NODE ``` |

#### 局部方向系统

对于应力、应变等的单元输出，始终使用局部圆柱方向系统。如果在原始三维扇区中指定了方向（参见["方向，" 第2.2.5节](pt01ch02s02aus15.md)），则新模型中的方向系统通过绕对称轴线旋转原始方向系统来定义。如果使用壳或膜，则将局部2轴和3轴投影到壳或膜的表面上作为表面上的局部方向。如果原始三维扇区中的材料不包含方向定义，则提供默认局部方向定义。此默认方向通过在原始模型中围绕新模型中的对称轴线旋转全局坐标系来定义。

#### 控制新节点和单元编号

您可以定义三维模型圆周方向上每个节点和单元之间的编号增量。编号从原始三维重复扇区开始。原始三维重复扇区使用与原始模型相同的编号。默认值是原始模型中使用的最大节点和单元编号。对编号的控制使您可以定义模型的附加部分而不会产生单元和节点标签冲突的风险。每个偏移值应分别大于或等于原始模型中使用的最大节点或单元标签。在指定偏移值时，必须注意生成的节点或单元不超过允许的最大值，即999,999,999。

| **输入文件用法：** | ``` [*SYMMETRIC MODEL GENERATION](../key/key-link.md#usb-kws-maximodelgen), PERIODIC, NODE OFFSET=*offset*, ELEMENT OFFSET=*offset* ``` |
| --- | --- |

#### 限制条件

- 只有基于表面的接触对可以旋转。使用一般接触的模型不能旋转。使用接触单元建模的接触条件将在模型生成中被忽略。
- 大多数类型的运动约束不能旋转。但是，原始模型中定义的基于表面的约束（["网格绑定约束，" 第35.3.1节](pt08ch35s03aus132.md)）和嵌入单元约束（["嵌入单元，" 第35.4.1节](pt08ch35s04aus136.md)）将在新三维模型中自动生成。一个例外是，用于强制循环对称约束的基于表面的绑定不会旋转。
- 基于表面的分布式耦合约束（如耦合（["耦合约束，" 第35.3.2节](pt08ch35s03aus133.md)）、壳-实体耦合（["壳-实体耦合，" 第35.3.3节](pt08ch35s03aus134.md)）和紧固件（["独立于网格的紧固件，" 第35.3.4节](pt08ch35s03aus135.md)）不能旋转，必须重新定义。
- 只有应力/位移、热传递、耦合温度-位移和声学单元可以旋转。梁和框架单元不能旋转。

#### 限制条件

- 只有基于表面的接触对可以反射。使用一般接触的模型不能反射。使用接触单元建模的接触条件将在模型生成中被忽略。

- 刚性表面不能反射。原始模型的刚性表面定义在新模型中简单地重复。因此，您必须在原始模型中指定完整的刚性表面。
- 大多数类型的运动约束不能反射。但是，原始模型中定义的基于表面的约束（["网格绑定约束，" 第35.3.1节](pt08ch35s03aus132.md)）和嵌入单元约束（["嵌入单元，" 第35.4.1节](pt08ch35s04aus136.md)）将在新三维模型中自动生成。
- 只有应力/位移、热传递、耦合温度-位移和声学单元可以反射。
- 弹簧、阻尼器、梁和桁架等非轴对称单元不能反射。
