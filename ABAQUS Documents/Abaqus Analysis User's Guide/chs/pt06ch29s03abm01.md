### 概述

本节描述Abaqus/Standard和Abaqus/Explicit中可用于梁单元的标准梁截面。可在Abaqus/Standard中的框架单元使用的标准梁截面子集。一般（非标准）梁截面可以按照["选择梁截面," 第29.3.2节](pt06ch29s03alm07.md)中所述进行定义。

### 任意、薄壁、开和闭截面

![](../graphics/ebeamlib-arbitrary-nls.png)

提供任意截面类型是为了允许对简单的任意、薄壁、开和闭截面进行建模。您通过在梁的薄壁截面中定义一系列点来指定截面；然后这些点通过直线段连接，每个直线段沿截面轴线进行数值积分，以便截面可以与非线性材料行为一起使用。每个构成任意截面的线段关联一个独立的厚度。

当任意截面与开截面梁单元（仅在Abaqus/Standard中可用）一起使用时，包含翘曲效应。

| **输入文件用法：** | 使用以下任一选项： |
| --- | --- |
|  | ``` [*BEAM SECTION](../key/key-link.md#usb-kws-mbeamsection), SECTION=ARBITRARY [*BEAM GENERAL SECTION](../key/key-link.md#usb-kws-mbeamgensect), SECTION=ARBITRARY ``` |

| **Abaqus/CAE用法：** | 属性模块：**创建配置文件**：**任意** |
| --- | --- |

#### 限制

- 任意截面只能与空间梁（三维模型）一起使用。
- 任意截面不应用于定义带分支的闭截面、多连通闭截面或断开区域的开口截面。
- 对于任意截面的每个单独线段，没有关于连接线段端点的线的弯曲刚度。因此，任意截面不能仅由一个线段组成。

#### 几何输入数据

首先，给出线段数、点*A*和*B*的局部坐标，以及连接这两个顶点的线段的厚度。然后，继续给定点*C*的局部坐标以及点*B*和*C*之间线段的厚度，然后给定点*D*的局部坐标以及点*C*和*D*之间线段的厚度，依此类推。任意截面可以包含任意数量的线段。截面定义点的所有坐标都在截面的局部1-2轴系统中给出。

局部1-2轴系统的原点是梁节点，用于定义截面的此节点位置是任意的：它不必是质心。

##### 定义闭截面

通过使起点和终点重合来定义闭截面。只能精确建模单胞闭截面。带肋片的闭截面（连接到胞的单个分支）不能在Abaqus中用此功能建模。

##### 定义带不连续分支的任意截面

如果任意截面包含不连续截面（分支），则应使用零厚度截面从分支的结束点返回到后续截面的起始点。此零厚度截面应始终与非零厚度截面重合。关于使用此方法定义的I截面的示例，请参见["梁的屈曲分析," Abaqus基准指南第1.2.1节](../bmk/bmk-link.md#bmk-anl-beambuckle)。

#### 默认积分

对构成截面的每个线段使用三点Simpson积分方案。对于更精确的积分，请在截面的每个直线部分指定几个线段。

#### 如果使用分析过程中积分的梁截面，默认应力输出点

截面的顶点。

#### 在分析过程中积分的梁截面的特定点处的温度和场变量输入

给出截面每个顶点的值（图中点*A*、*B*、*C*、*D*）。
