# 82.9 在 Abaqus/CAE 和 Microsoft Excel 之间交换数据





**Excel Utilities** 插件使您能够从 Microsoft Excel 工作簿导入幅值数据或 *X–Y* 数据，以及将任一类型的数据从 Abaqus/CAE 导出到 Microsoft Excel 工作簿。

**注意：****Excel Utilities** 插件仅适用于 Windows。

通过从主菜单栏选择 ****Plug-ins**![](../graphics/images/arrow.gif)**Tools**![](../graphics/images/arrow.gif)**Excel Utilities**** 来启动插件。在 **Excel Utilities** 对话框中输入以下内容：

**Object**

选择 **XY Data** 以传输 *X–Y* 数据对象，或选择 **Amplitude** 以传输幅值定义。

**Transfer Direction**

选择 **From Abaqus/CAE to Excel** 以将数据从 Abaqus/CAE 导出到 Microsoft Excel，或选择 **From Excel to Abaqus/CAE** 以将数据从 Microsoft Excel 电子表格导入当前会话或打开的模型数据库之一。

**从 Excel 导入数据的选项**

要从 Microsoft Excel 导入幅值数据或 *X–Y* 数据，请指定要从中导入数据的 Excel 文件的名称，选择该文件中的一个工作表，并识别要导入数据的起始位置。默认情况下，插件只导入两列数据——所选列中的数据和紧接其右侧的列中的数据——但如果在导入 *X–Y* 数据时切换 **Import multiple curves**，插件会成对导入数据，为每对创建一个幅值定义或 *X–Y* 数据对象。如果要导入幅值数据，请指定创建的幅值数据对象是tabular型还是smooth step型。

幅值对象保存到当前模型，而 *X–Y* 数据对象可用于当前会话。

**导出数据到 Excel 的选项**

要将幅值数据导出到 Microsoft Excel，请选择数据所在的模型，然后选择一个或多个幅值定义并点击 **OK**。Abaqus/CAE 会创建一个新的 Excel 文件，并在工作表中显示导出的数据。如果要导出单个幅值对象，Abaqus/CAE 还会在图表中绘制数据；如果导出多个幅值对象，则不会创建图表。只有 tabular 或 smooth step 型幅值定义可用于导出。

要将 *X–Y* 数据导出到 Microsoft Excel，请选择一个或多个 *X–Y* 数据对象并点击 **OK**。Abaqus/CAE 会创建一个新的 Excel 文件，并在工作表中显示导出的数据。如果要导出单个 *X–Y* 数据对象，Abaqus/CAE 还会在图表中绘制数据；如果导出多个 *X–Y* 数据对象，则不会创建图表。您可以导出会话中的任何 *X–Y* 数据对象，包括任何临时对象或当前显示在视口中的数据。
