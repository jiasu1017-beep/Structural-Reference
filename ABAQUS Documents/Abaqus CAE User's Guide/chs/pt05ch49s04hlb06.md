# 49.4.6 控制多视口中的动画





默认情况下，Abaqus/CAE在会话中的每个符合动画条件的视口中动画化数据。所有动画都是同步的，因此当您逐帧检查数据、动画化模型图或动画化时间相关X–Y曲线图时，视口会一起播放、停止和递增。

您可以激活和停用单个视口的动画，这使您可以仅在会话中符合动画条件的视口子集中显示动画。"动画选项"对话框的"视口"页面（如图49-1所示（[图49-1](pt05ch49s04hlb06.md#usv-anm-viewport-tab)））显示会话中所有符合动画条件的视口列表。在此页面上切换视口的复选框可以包含或排除该视口参与动画。

**图49-1** "动画选项"对话框的"视口"页面。

![](../graphics/usv-anm-viewport-tab-nls.png)

**控制要动画化数据的视口：**

1. 找到"视口"选项。在提示区域中，单击"动画选项"；然后在出现的对话框中单击"视口"选项卡。"视口"页面出现。
2. 切换您要动画化的视口的复选框。
3. 如果需要，通过从"动画选项"对话框底部的字段中选择"动画类型"来切换动画类型。
4. 单击"应用"。
