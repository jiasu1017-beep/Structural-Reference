# 9.8.3 在模型之间复制对象





从主菜单栏选择****模型（Model）**![](../graphics/images/arrow.gif)**复制对象（Copy Objects）****来在当前模型数据库中的模型之间复制对象。您可以复制以下对象：
- 草图
- 零件（零件集合也会被复制）
- 实例
- 材料
- 截面（包括连接器截面）
- 轮廓
- 幅值
- 相互作用属性

当您选择要复制的实例时，相应的零件默认也会被选中；如果零件已存在，您可以取消选择零件。您无法复制其他单独的对象，如装配、载荷或步骤；但是，您可以通过将整个模型复制到新模型并在新模型中编辑对象来实现类似的效果。有关更多信息，请参阅["在模型数据库中操作模型，"第9.8.1节"](pt02ch09s07hlb01.md)。当您在模型之间复制对象时，依赖对象不会自动复制。例如，如果您复制截面，关联的材料不会随截面一起复制；您必须在单独的操作中复制材料。

**要在模型之间复制对象：**

1. 从主菜单栏，选择****模型（Model）**![](../graphics/images/arrow.gif)**复制对象（Copy Objects）****。出现**复制对象（Copy Objects）**对话框。
2. 从对话框中，选择要从中复制对象的模型。
3. 使用以下技术指定要从所选模型中复制的对象：- 单击所需对象类别旁边的箭头。从出现的对象列表中，切换您选择的对象的名称。如果对象类别不包含任何对象，则该类别不可用。- 切换所需的对象类别。此操作将选择或取消选择该类别内的所有对象。当选定类别内的所有对象时，对象类别旁边的复选框会显示白色背景上的黑色复选标记。如果仅选择了该类别中的某些对象，复选框会显示浅灰色背景上的深灰色复选标记。您必须至少选择一个对象或对象类别进行复制。
4. 从**复制对象（Copy Objects）**对话框底部，选择要将所选对象复制到哪个模型。
5. 单击**确定（OK）**复制所选对象并关闭**复制对象（Copy Objects）**对话框。Abaqus/CAE会复制所选的对象。如果要复制到的模型中已存在同名的对象，Abaqus/CAE会询问您是否要覆盖现有对象。单击**全部是（Yes to All）**用与您正在复制的对象同名的所有现有对象覆盖。
