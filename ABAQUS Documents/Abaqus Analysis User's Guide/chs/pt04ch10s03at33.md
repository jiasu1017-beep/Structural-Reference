# 12.5.2 生成热矩阵

**产品：** Abaqus/Standard

##### **参考资料**

- ["单元矩阵组装实用工具，" 第 3.2.26 节](pt01ch03s02abx26.md)
- [*ELEMENT OPERATOR OUTPUT](../key/key-link.md#usb-kws-helementoperatoroutput)

### 概述

热矩阵生成：
- 通过生成代表热传导、热容和特定时间载荷的全局或单元矩阵，允许对模型数据进行数学抽象
- 将矩阵数据写入二进制 SIM 文档，可由 Abaqus 进一步处理
- 只能作为非耦合传热分析的一部分使用

### 概述

线性化传热有限元模型可以基于热载荷向量和代表热容与热传导的矩阵来总结。这种数学抽象有多种用途。例如，您可以使用这些矩阵与其他用户、供应商或软件包交换模型数据，而无需交换网格或材料信息。您还可以在模型排序减少等技术中使用这些矩阵。

热矩阵生成在传热分析中进行，并考虑模型中所有当前边界条件、载荷和材料响应。生成的矩阵存储在名为 `*jobname*THERM*n*.sim` 的 SIM 文档中，其中 jobname 是输入文件或分析作业的名称，n 是生成矩阵的 Abaqus 传热步骤的编号。

### 指定矩阵类型

可以生成代表以下模型特性的热矩阵：
- 热容
- 热传导
- 载荷

如果热传导属性是温度依赖的，则热传导矩阵具有非对称贡献。只有在步骤定义中激活了非对称求解器（参见 ["定义分析，" 第 6.1.2 节](pt03ch06s01abo05.md)），才会考虑此项。

载荷矩阵包含与传热步骤中定义的载荷对应的节点外部通量向量或净通量向量。

| **输入文件用法：** | 使用以下选项生成热容矩阵： |
| --- | --- |
| | ``` [*ELEMENT OPERATOR OUTPUT](../key/key-link.md#usb-kws-helementoperatoroutput), DAMPING ``` |
| | 使用以下选项生成热传导矩阵： |
| | ``` [*ELEMENT OPERATOR OUTPUT](../key/key-link.md#usb-kws-helementoperatoroutput), STIFFNESS ``` |
| | 使用以下选项生成外部通量向量： |
| | ``` [*ELEMENT OPERATOR OUTPUT](../key/key-link.md#usb-kws-helementoperatoroutput), LOAD, LOADTYPE=EXTERNAL ``` |
| | 使用以下选项生成净通量向量： |
| | ``` [*ELEMENT OPERATOR OUTPUT](../key/key-link.md#usb-kws-helementoperatoroutput), LOAD, LOADTYPE=NET ``` |

### 为模型的一部分生成矩阵

默认情况下，会为模型中所有支持的单元类型生成热矩阵。您可以请求 Abaqus/Standard 为由单元集定义的模型部分生成矩阵。

| **输入文件用法：** | ``` [*ELEMENT OPERATOR OUTPUT](../key/key-link.md#usb-kws-helementoperatoroutput), ELSET=*element set name* ``` |

### 指定矩阵生成频率

默认情况下，会在请求热矩阵的步骤的每个增量中生成矩阵。您可以请求以指定频率生成矩阵。

| **输入文件用法：** | ``` [*ELEMENT OPERATOR OUTPUT](../key/key-link.md#usb-kws-helementoperatoroutput), FREQUENCY=*output frequency* ``` |

### 生成组装矩阵

默认情况下，热矩阵以单元-by-单元形式写入 SIM 文档。您可以将组装矩阵写入 SIM 文档，这在使用大型单元集或频繁请求热矩阵输出时推荐。

| **输入文件用法：** | ``` [*ELEMENT OPERATOR OUTPUT](../key/key-link.md#usb-kws-helementoperatoroutput), ASSEMBLE ``` |

### 初始条件

热矩阵生成发生在通用分析过程中。因此，生成的矩阵包括瞬态分析中初始条件的影响。

### 边界条件

规定的温度边界条件不会施加到生成的热矩阵和载荷向量上。

### 载荷

所有类型的非耦合传热分析支持的载荷都可以用于热矩阵生成。温度依赖的载荷（如薄膜和辐射）会对热传导矩阵贡献额外的"载荷刚度"项。

### 预定义场

可以为热矩阵生成指定所有类型的预定义场。

### 材料选项

可用于 Abaqus/Standard 非耦合传热的所有材料类型都可以用于热矩阵生成。

### 单元

只有连续扩散传热单元和热接触单元支持热矩阵生成。热矩阵仅针对支持的单元写入 SIM 文档。

### 输出

生成的矩阵以单元-by-单元或组装形式写入输出 SIM 文档。为了效率，仅在 SIM 文档中存储非零矩阵条目。如果矩阵是对称的，则仅存储上三角部分的非零条目。您可以使用矩阵组装实用工具来组装 SIM 文档中的单元矩阵和/或将组装矩阵写出到文本文件。

### 局限性

使用自由度消除技术实现的约束（如绑定约束）不会为热矩阵输出进行处理。此外，空腔辐射效应不会被考虑用于热矩阵输出。
