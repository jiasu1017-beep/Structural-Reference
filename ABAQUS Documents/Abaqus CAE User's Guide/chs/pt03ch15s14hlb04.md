# 15.14.6 定义声阻抗相互作用属性







您可以使用声阻抗相互作用属性来定义声学分析中压力与表面位移和速度法向分量之间的比例因子。声阻抗相互作用属性只能由声阻抗相互作用引用。更多信息，请参见第15.13.17节"定义声阻抗"。

**定义声阻抗相互作用属性：**

1. 从主菜单栏中，选择****相互作用**![](../graphics/images/arrow.gif)**属性**![](../graphics/images/arrow.gif)**创建****。
2. 在出现的**创建相互作用属性**对话框中，执行以下操作：- 命名该相互作用属性。有关命名对象的信息，请参见第3.2.1节"使用基本对话框组件"。- 选择**声阻抗**类型的相互作用属性。
3. 单击**继续**关闭**创建相互作用属性**对话框。
4. 从**数据类型**字段中，选择用于定义声阻抗的表格数据类型。- 选择**阻抗**以使用阻抗的实部和虚部指定阻抗。- 选择**导纳**以使用导纳值指定阻抗。
5. 切换打开**使用频率相关数据**以定义随频率变化的声阻抗。数据表中会出现一个标有**频率**的列。
6. 如果您使用阻抗数据，请在表中输入以下数据：- ![](../graphics/usi_eqn00606.gif)，表面阻抗的实部。（单位：[F L3T](../popups/usb-int-iconventions-unitsym.md)。）- ![](../graphics/usi_eqn00607.gif)，表面阻抗的虚部。（单位：[F L3T](../popups/usb-int-iconventions-unitsym.md)。）
7. 如果您使用导纳数据，请在表中输入以下数据：- ![](../graphics/usi_eqn00608.gif)，压力与表面法向速度之间比例因子。这是复导纳的实部。（单位：[F1L3T1](../popups/usb-int-iconventions-unitsym.md)。）- ![](../graphics/usi_eqn00609.gif)，压力与表面法向位移之间比例因子。这是复导纳虚部除以角频率。（单位：[F1L3](../popups/usb-int-iconventions-unitsym.md)。）
8. 单击**确定**创建声阻抗相互作用属性并退出编辑器。
