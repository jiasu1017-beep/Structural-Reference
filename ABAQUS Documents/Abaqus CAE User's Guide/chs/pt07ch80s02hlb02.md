# 80.2.2 显示切割截面及其合力与力矩向量





要显示模型的切割截面，您需要激活视图切割并选择是在切割处、切割下方和/或切割上方显示模型。切割本身不可见。在可视化模块中，您可以同时在切割下方和切割上方显示模型；在所有其他模块中，您可以显示一种视图或另一种。

在可视化模块中，您可以激活多个视图切割；在所有其他模块中，一次只能有一个视图切割处于活动状态。对于可视化模块中的每个视图切割，您还可以显示整个模型或当前显示组上的合力与力矩。

对于显示输出数据库数据的视图切割，您还可以显示所选视图切割的合力和力矩向量，这些向量在模型中以规则间隔显示。有关更多信息，请参见["自定义活动视图切割上合力和力矩的显示和计算，" 第80.2.8节](pt07ch80s02hlb08.md)。

每个视图切割的激活状态仅在会话期间保留。如果您想保留此视图切割激活状态以供后续会话使用，可以将这些数据保存到XML文件、模型数据库或输出数据库。有关更多信息，请参见["管理会话对象和会话选项，" 第9.9节](pt02ch09s08.md)。

**显示切割截面的步骤：**

1. 从主菜单栏中选择****工具**![](../graphics/images/arrow.gif)**View Cut**![](../graphics/images/arrow.gif)**Manager****。出现 **View Cut Manager**，其中列出在当前会话期间创建的所有视图切割。活动视图切割左侧的 **Show** 列中出现一个勾选标记。
2. 单击要激活或停用的视图切割左侧的 **Show** 列中的复选框。活动视图切割用于切割当前视口中的模型。
   **提示：**您也可以使用以下方法之一激活或停用视图切割：
   - 在可视化模块以外的任何模块中，单击视图切割工具栏中的 ![](../graphics/ico_viewCutMain.png) 工具。
   - 在可视化模块中，单击工具箱中的 ![](../graphics/ico_viewCut.png) 工具。
   如果没有视图切割处于活动状态，选择这些工具中的任何一个将激活 **View Cut Manager** 列表中的第一个视图切割（通常是默认的X平面切割）。如果视图切割处于活动状态，选择此工具将停用该切割。
3. 单击视图切割右侧 **Model** 列中的复选框，以在切割下方 ![](../graphics/visG_CutBelow.png)、切割处 ![](../graphics/visG_CutOn.png) 和/或切割上方 ![](../graphics/visG_CutAbove.png) 显示模型（这些位置不互相排斥）。模型的选定部分显示在视口中。
4. 如果您在可视化模块中工作，请单击活动视图切割 **Model** 列中 ![](../graphics/ico_freeBodyDiagram.png) 图标下的复选框，以沿该视图切割显示作用在可见部分上的合力和力矩。例如，当您从"显示切割下方"选项切换到"显示切割上方"选项时，您将看到大小相等方向相反的合力和力矩向量。
   **注意：****View Cut Options** 使您能够控制视图切割上合力和力矩显示的三个方面。您可以基于当前显示组或整个模型计算合力和力矩，并且可以调整求和点和分量分辨率。您可以为实体几何以及壳截面或梁截面启用视图切割上合力和力矩的显示。有关更多信息，请参见["自定义活动视图切割上合力和力矩的显示和计算，" 第80.2.8节](pt07ch80s02hlb08.md)。
5. 在可视化模块中，切换 **Allow for multiple cuts** 以启用多个视图切割的显示。
