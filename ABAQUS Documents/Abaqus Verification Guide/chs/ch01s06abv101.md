# 1.6.23 接触控制







**产品：**Abaqus/Standard  

### I. 接触稳定

### 单元测试

C3D8    CPE4

### 功能测试

接触稳定

### 问题描述

接触稳定可用于控制可能在接触完全发展之前存在于模型中的刚体运动。该选项在法向和切向方向上添加粘性阻尼。默认情况下，阻尼是自动计算的，但可以修改阻尼系数、阻尼系数在步骤中的变化、阻尼工作的范围以及法向与切向阻尼之间的比率。使用此选项指定的控制保持有效，直到它们被另一个接触控制过程改变或重置为默认值。接触稳定可以为特定接触对或整个模型定义。关于稳定控制的更多描述，请参阅《Abaqus分析用户手册》第36.3.6节"在Abaqus/Standard中调整接触控制"。在这些测试中，在具有多个接触对的多步骤分析中测试了各种稳定控制组合。

第一组分析由六对块组成，在步骤1中被推到一起，在步骤2中受到切向滑动，并在步骤3中被拉开。块是弹性的，块的运动由边界条件控制。为整个模型指定接触稳定参数，并由几个单独接触对的不同参数覆盖。稳定参数随步骤变化。写入重启文件，并进行一些重启以测试重启功能。

第二组分析由三个块组成，在步骤1中被推到一起，在步骤2中受到切向滑动，并在步骤3中被拉开。块是弹性的；上下块由边界条件控制，而中间块完全自由并由接触稳定固定。每个接触对使用不同的接触稳定参数。此外，为一个接触对指定了摩擦特性。该组包含二维和三维静态分析以及动态分析。

### 结果与讨论

结果显示接触阻尼压力CDPRESS以及与预期一致的接触阻尼剪应力CDSHEAR1和CDSHEAR2。此外，在第二组问题中，中间块的刚体运动被控制，没有观察到求解器消息。

### II. 切向接触控制

### 单元测试

C3D20R    C3D27R

### 功能测试

在线性扰动步骤中修改所有接触对的切向惩罚刚度。

### 问题描述

在线性扰动步骤中，所有接触点（即具有"闭合"状态的点）如果存在摩擦则假定为粘附。然而，对于由参考框架运动或传输速度施加速度差的接触节点，不强制执行粘附条件。默认情况下，粘附条件使用惩罚方法强制执行，扰动切向比例因子可用于缩放惩罚刚度。例如，将此参数设置为零将导致零惩罚刚度，使得在扰动步骤期间不强制执行粘附条件。将此参数设置为大于一的值将导致大于默认的惩罚刚度，从而在扰动步骤期间更严格地强制执行粘附条件。

模型由两个大小不同、接触的块组成，有效摩擦系数不为零。在第一和第二通用步骤中，我们建立接触并施加切向位移边界条件，使小块沿大块滑动。在后续扰动步骤中计算固有频率：

| 步骤名称 | 扰动切向比例因子 |
| --- | --- |
| Frequency1 | 未指定（默认设置为1.0） |
| Frequency2 | 设置为1.0（与默认相同） |
| Frequency3 | 设置为0.0（与无摩擦相同） |
| Frequency4 | 设置为103 |

**材料：**

| 杨氏模量 | 2 107 |
| --- | --- |
| 泊松比 | 0.3 |
| 摩擦系数 | 0.2 |

### 结果与讨论

步骤3和4（步骤名称"Frequency1"和"Frequency2"）提供相同的结果，如预期的那样。步骤5（步骤名称"Frequency3"）具有三个零频率特征模式，对应于两个块之间的相对滑动，与无摩擦行为一致。在步骤6（步骤名称"Frequency4"）的特征模式中明显可见粘附条件的严格强制执行。

### 输入文件

[pertbcntctrl.inp](../eif/pertbcntctrl.inp)

此分析的输入文件。

### III. 压力相关接触控制

### 单元测试

CPS4    C3D8

### 功能测试

压力相关接触约束强制

### 问题描述

默认情况下，在线性扰动步骤中，对于接触活跃的表面，接触约束通常被完全强制执行，与基础状态中的局部法向压力无关。这种接触约束处理可能会过度刚化结构，因为模型中某些接触压力接近零的区域可能在振动期间实际进入和脱离接触。线性扰动过程的性质不允许这种接触跳动。但是，您可以使用压力相关约束强制来放松甚至完全移除具有低压力的约束。

您可以通过两个用户指定的接触压力系数来控制行为：![](../graphics/ver_eqn00473.gif)和![](../graphics/ver_eqn00474.gif)；默认情况下，![](../graphics/ver_eqn00473.gif)等于![](../graphics/ver_eqn00474.gif)。对于大于![](../graphics/ver_eqn00474.gif)的压力，约束被完全强制执行，如默认接触约束处理。对于介于之间的压力（如![](../graphics/ver_eqn00473.gif)小于![](../graphics/ver_eqn00474.gif)），约束逐渐减弱。对于小于![](../graphics/ver_eqn00473.gif)的压力，约束被完全移除。

模型由放置在两个砧之间的平面应力圆盘组成。施加一系列线性扰动频率步骤和通用步骤。在通用步骤期间，圆盘在砧之间被压缩，线性扰动频率步骤计算具有不同参数设置的特征对，如表1.6.23-1所示。

### 结果与讨论

在`Frequency1`步骤期间，施加最小边界条件以计算圆盘在没有预应力的情况下的非刚体特征对。`General1`步骤在砧之间压缩圆盘，因此建立接触约束，最大接触压力值超过6×104。`Frequency2`步骤计算应力结构的特征对，与`Frequency1`相比`Frequency2`谱增加。在`Frequency3`步骤中，![](../graphics/ver_eqn00473.gif)=0且![](../graphics/ver_eqn00474.gif)=5×104，因此一些接触约束被放松。因此，`Frequency3`的特征值相对于`Frequency2`降低。在`Frequency4`中，![](../graphics/ver_eqn00473.gif)=![](../graphics/ver_eqn00474.gif)=5×104，因此一些约束被完全移除，谱进一步降低。`Frequency5`具有与`Frequency3`相同的压力相关约束强制规范，但使用局部接触对而不是全局规范，`Frequency5`和`Frequency3`的结果相同。`General2`步骤产生额外压缩，因此下一个`Frequency6`线性扰动步骤的特征值相对于`Frequency2`增加。`Frequency6` ![](../graphics/ver_eqn00473.gif)=![](../graphics/ver_eqn00474.gif)=0的结果与`Frequency7`默认约束强制结果一致。在`Frequency8`中，![](../graphics/ver_eqn00473.gif)和![](../graphics/ver_eqn00474.gif)都等于一个大数，因此所有接触约束都被移除。然而，由于加载结构的几何非线性效应，`Frequency8`结果与`Frequency1`结果不一致。

还使用上述相同步骤序列运行了两个简单的二维和三维两单元问题。然而，这些问题是几何线性的，因此`Frequency8`结果与`Frequency1`结果相同。
