# Q







### quantity type

A category that describes the data in one of the columns of an *X–Y* data object. Abaqus/CAE includes over 80 predefined quantity types that describe the output of every variable in Abaqus, including commonly used quantity types such as temperature, time, stress, and strain.
For more information:- ["Understanding quantity types," Section 47.1.4 of the Abaqus/CAE User's Guide](../usi/usi-link.md#usv-xyp-quantitytypes)

### Query toolset

An Abaqus/CAE toolset that allows you to obtain information about your model. Abaqus/CAE displays the requested information in the message area; and, in most cases, the same information is written to the replay file.
For more information:- [Chapter 71, "The Query toolset," of the Abaqus/CAE User's Guide](../usi/usi-link.md#usi-qre)

### queue

An option that executes jobs on a computer in a sequential manner.
For more information:- ["Submitting a job remotely," Section 19.2.7 of the Abaqus/CAE User's Guide](../usi/usi-link.md#usi-ana-conc-remote)
