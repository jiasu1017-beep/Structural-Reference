# 58.2.1 构建有效表达式







表达式场描述选定相互作用和规定条件（如压力大小）在空间中某点的参数变化。您可以通过在表达式场编辑器中构建数学表达式来创建表达式场。要找到表达式场编辑器，请从主菜单栏选择 ****工具**![](../graphics/images/arrow.gif)**解析场**![](../graphics/images/arrow.gif)**创建****。

表达式场使用全局坐标系或局部坐标系的坐标来指示空间中的点。在表达式场编辑器中，这些坐标列为 **参数名称**。默认情况下，表达式使用全局坐标系的坐标（**X**、**Y** 和 **Z** 参数名称）构建。可用的参数名称取决于您选择的坐标系类型，如 [表 58–1](pt06ch58s02s01.md#parameter-names) 所示。对于圆柱坐标系和球坐标系，**Th** 和 **P** 的值以弧度为单位进行评估，范围从 ![](../graphics/usi_eqn00792.gif) 到 ![](../graphics/usi_eqn00793.gif)。

**表 58–1** 表达式场编辑器中坐标系类型与参数名称的关系。
| 坐标系类型 | 参数 | 参数名称 |
| --- | --- | --- |
| 直角坐标或全局（默认） | *X*、*Y* 和 *Z* 轴 | **X**、**Y**、**Z** |
| 圆柱坐标 | *R*、![](../graphics/usi_eqn00065.gif) 和 *Z* 轴 | **R**、**Th**、**Z** |
| 球坐标 | *R*、![](../graphics/usi_eqn00065.gif) 和 ![](../graphics/usi_eqn00226.gif) 轴 | **R**、**Th**、**P** |

表达式由一个或多个参数名称和一个或多个运算符组成。您可以通过从列表中选择参数名称和运算符，或直接在表达式窗口中键入参数名称和运算符，在编辑器的表达式窗口中构建数学表达式。参数名称和运算符区分大小写。表达式使用 Python 要求的语法；包含错误的条目将生成标准 Python 错误。请参阅 ["表达式中的运算和函数概述，" 第 58.2.2 节](pt06ch58s02s02.md)，了解支持的操作符。 

以下示例演示了有效的表达式。 

**示例 1**

要定义沿面线性距离的空间变化，请在表达式窗口中键入：

```
4.5 * X + 2.75
```
`X` 是表示沿面距离的参数名称。

**示例 2**

要在圆柱坐标系中定义更复杂的空间变化，请在表达式窗口中键入：

```
R + sin(Th*pi/2)
```

有关详细信息，请参阅 ["创建表达式场，" 第 58.6 节](pt06ch58hla01.md)。
