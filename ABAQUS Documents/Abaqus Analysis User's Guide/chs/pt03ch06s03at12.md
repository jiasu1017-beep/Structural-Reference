### 概述

模态动态分析：
- 使用模态叠加技术分析瞬态线性动力学问题；
- 只能在对频率提取过程之后执行，因为它基于系统模态的结构响应；
- 可以使用高性能SIM软件架构（参见["动态分析过程概述，" 第6.3.1节中的"使用SIM架构进行模态叠加动态分析"](pt03ch06s03abo07.md#usb-anl-alineardynamics)）；
- 仅在使用SIM架构时可以包括非对角阻尼效应（即来自材料或单元阻尼）；以及
- 是线性扰动过程。

### 模态动态分析

瞬态模态动态分析根据给定的时间依赖载荷给出模型响应随时间的变化。结构响应基于系统模态的子集，这些模态必须首先使用特征频率提取过程提取（["固有频率提取，" 第6.3.5节"](pt03ch06s03at10.md)）。模态将包括特征模态，如果在特征频率提取步骤中激活了残余模态，则还包括残余模态。提取的模态数量必须足以充分模拟系统的动态响应，这需要您自行判断。

模态振幅通过时间进行积分，并从这些模态响应合成响应。对于线性系统，模态动态过程比动态过程中整个方程组直接积分（["使用直接积分的隐式动态分析，" 第6.3.2节"](pt03ch06s03at07.md)）在计算上要便宜得多。

只要系统是线性的，并且被使用的模态正确表示（这些模态通常只是有限元模型总模态的一小部分），该方法也非常准确，因为只要强迫函数随时间分段线性变化，使用的积分算子就是精确的。您应确保强迫函数定义和时间增量的选择为此目的一致。例如，如果强迫是一个地震记录，其中加速度值每毫秒给出一次，并且假定加速度在这些值之间线性变化，则模态动态过程中使用的时间增量应为毫秒。

用户指定的最大增量数在模态动态步骤中被忽略。增量数基于该步骤选择的时间增量和工作时间。

虽然此过程中的响应是线性振动，但先前的响应可以是非线性的，如果在对特征频率提取过程的基础状态进行步骤定义时包含了非线性几何效应，则响应将包括应力刚化（初始应力）效应，如["固有频率提取，" 第6.3.5节"](pt03ch06s03at10.md)中所解释。

##### 选择模态和指定阻尼系数的规则

选择模态和指定模态阻尼系数遵循以下规则：
- 默认情况下不包含模态阻尼。
- 模态选择和模态阻尼必须以相同方式指定，使用模态编号或频率范围。
- 如果不选择任何模态，则叠加中将使用先前频率分析中提取的所有模态，包括已激活的残余模态。
- 如果不为已选择的模态指定阻尼系数，则这些模态将使用零阻尼值。
- 阻尼仅应用于所选的模态。
- 所选模态中超出指定频率范围的阻尼系数为常数，等于指定第一个或最后一个频率的阻尼系数（取决于哪个更近）。这与Abaqus解释振幅定义的方式一致。

#### 指定全局阻尼

为方便起见，您可以为所有选定的特征模态指定恒定的全局阻尼因子，用于质量比例阻尼和刚度比例黏性阻尼，以及刚度比例结构阻尼。结构阻尼是一种常用的阻尼模型，将阻尼表示为复刚度。这种表示法不会给频域分析（如稳态动力学）带来困难，因为解已经是复数。然而，解必须在时域中保持实值。为了允许用户在时域中应用其结构阻尼模型，开发了一种将结构阻尼转换为等效黏性阻尼的方法。只要投影阻尼矩阵是对角的，该技术在频域中应用的黏性阻尼就与结构阻尼相同。更多详细信息，请参见["模态动态分析，" 第2.5.5节，Abaqus理论指南](../stm/stm-link.md#stm-anl-modaldynamic)。

| **输入文件用法：** | ``` [*GLOBAL DAMPING](../key/key-link.md#usb-kws-hglobaldamping), ALPHA=*factor*, BETA=*factor*, STRUCTURAL=*factor* ``` |

### 初始条件

默认情况下，模态动态步骤将从零初始位移开始。如果已定义初始速度（["Abaqus/Standard和Abaqus/Explicit中的初始条件，" 第34.2.1节"](pt07ch34s02aus116.md)），则将使用这些值；否则，初始速度为零。

或者，您可以强制模态动态步骤继承紧前步骤的初始条件，该步骤必须是另一个模态动态步骤或静态扰动步骤：
- 在大多数情况下，如果紧前步骤是模态动态步骤，则从该步骤结束处继承位移和速度，并用作当前步骤的初始条件。对于基于SIM的分析，您应该使用辅助基础运动而不是主基础运动（参见["模态叠加过程中的规定运动"](pt03ch06s03at12.md#usb-anl-amodaldynamic-motion)）来继承初始条件；如果使用主基础运动，Abaqus会发出警告消息。
- 如果紧前步骤是静态扰动步骤，则从该步骤继承位移。如果已定义初始速度（["Abaqus/Standard和Abaqus/Explicit中的初始条件，" 第34.2.1节"](pt07ch34s02aus116.md)），则将使用这些值；否则，初始速度为零。

### 边界条件

在基于模态的动态响应过程中，不可能直接规定非零位移和旋转（或声压）作为边界条件（["Abaqus/Standard和Abaqus/Explicit中的边界条件，" 第34.3.1节"](pt07ch34s03aus118.md)）。在这些过程中，只能将节点的运动规定为基础运动，如下所述。在模态叠加过程中，作为边界条件给出的非零位移或加速度历史定义被忽略，并且特征频率提取步骤中支撑条件的变化被标记为错误。

#### 模态叠加过程中的规定运动

在特征频率提取步骤中，必须将边界条件应用于将在模态动态过程中规定的自由度。这些自由度被分组为一个或多个"基"（参见["固有频率提取，" 第6.3.5节"](pt03ch06s03at10.md)）。未命名的基称为"主"基。命名的"辅助"基必须通过在频率提取步骤中指定边界条件来定义。可以为每个基规定不同的运动。

##### 指定自由度和运动的时间历史

与基相关联的位移和旋转在模态动态响应过程中被规定。基运动最多由三个全局平移和三个全局旋转完全定义。因此，每个平移和旋转分量最多可以定义一个基运动。基运动始终以全局方向指定，无论是否使用了节点变换。您指定正在定义基运动的全局方向（1-6）。如果相对于不是坐标原点的原点指定了旋转，则必须指定旋转中心。

运动的时间历史必须通过振幅曲线定义（["振幅曲线，" 第34.1.2节"](pt07ch34s01aus115.md)）。

##### 示例

为了说明主基和辅助基的概念，考虑一个单跨框架，其支撑在节点1和节点4。如果在特征频率提取步骤之前的输入包括以下边界条件：
- 自由度1至6在节点1处受到约束
- 自由度1在节点4处受到约束
- 自由度3至6在节点4处受到约束

并且不同的基础运动被分配给节点1和节点4处的自由度2，则可以使用以下步骤定义：
- 特征频率提取步骤，包括与`BASE2`相关联的边界条件，约束节点4处的自由度2；以及
- 模态动态步骤，包括两个基础运动定义：分配给自由度2的主基础运动（不引用基）和分配给自由度2的辅助基础运动（引用`BASE2`）。

如果未在特征频率提取步骤之前给出边界条件，则必须在特征频率提取步骤中定义它们。同样，辅助基由具有基名称的边界条件定义。

#### 计算结构响应

主基相关联的自由度在特征频率提取步骤中被设置为零，主基运动通过将基础加速度乘以模态参与因子引入。因此，Abaqus/Standard计算结构相对于主基的响应。如果在主基础运动定义中引用了旋转自由度，则除非您提供旋转中心，否则默认围绕坐标原点定义旋转。

辅助基相关联的自由度在特征频率提取步骤中未设置为零；相反，向每个自由度添加了"大"质量。辅助基中任何在先前一般步骤中通过常规边界条件约束的自由度将被释放，并向该自由度添加大质量。辅助基运动通过节点力引入，通过将基础加速度（或声压）乘以大质量（或声大质量）获得。尽管辅助基运动以绝对方式定义，但计算出的辅助基响应是相对于主基的运动（对于平移自由度）。旋转辅助基关于基名称定义中指定的节点集中包含的节点定义。因此，您无法更改辅助基的旋转中心。

有关基础运动过程的更详细说明，请参见["基于模态的过程中的基础运动，" 第2.5.9节Abaqus理论指南](../stm/stm-link.md#stm-anl-basemotions)。

### 载荷

在模态动态分析中可以规定以下载荷，如["集中载荷，" 第34.4.2节"](pt07ch34s04aus121.md)中所述：
- 可以将集中节点力应用于位移自由度（1-6）。
- 可以施加分布式压力力或体力；特定单元可用的分布式载荷类型在[第六部分，"单元"](pt06.md)中描述。

### 预定义场

在瞬态模态动态分析中不允许使用预定义温度场。其他预定义场被忽略。

### 材料选项

必须定义材料的密度（["密度，" 第21.2.1节"](pt05ch21s02abm01.md)）。以下材料属性在模态动态分析期间不活跃：塑性和其它非弹性效应、率相关材料属性、热属性、质量扩散属性、电属性（压电分析中的电位除外）以及孔隙流体流动属性。参见["一般和线性扰动过程，" 第6.1.3节"](pt03ch06s01aus44.md)。

### 单元

除了广义轴对称扭曲单元外，Abaqus/Standard中的任何应力/位移单元（包括具有温度和压力自由度的单元）都可用于模态动态分析。

### 输出

Abaqus/Standard中的所有输出变量列在["Abaqus/Standard输出变量标识符，" 第4.2.1节"](pt02ch04s02abv01.md)中。时域模态动力学中节点解变量U、V和A的值是相对于主基的运动。因此，相对运动和主基运动的和得出总运动；通过请求输出变量TU、TV和TA可以获得此总运动。在没有主基运动的情况下，相对运动和总运动是相同的。

以下模态变量可以输出到数据或结果文件（参见["输出到数据和结果文件，" 第4.1.2节"](pt02ch04s01aus39.md)）：

| GU | 所有模态的广义位移。 |
| --- | --- |

| GV | 所有模态的广义速度。 |
| --- | --- |

| GA | 所有模态的广义加速度。 |
| --- | --- |

| SNE | 每个模态整个模型的弹性应变能。 |
| --- | --- |

| KE | 每个模态整个模型的动能。 |
| --- | --- |

| T | 每个模态整个模型的外功。 |
| --- | --- |

| BM | 基础运动。 |
| --- | --- |

模态动态分析不提供单元能量密度（如弹性应变能密度SENER）或整个单元能量（如单元的总动能ELKE）的输出。但是，整个模型变量（如ALLIE（总应变能））可作为基于模态的过程输出到数据或结果文件（参见["输出到数据和结果文件，" 第4.1.2节"](pt02ch04s01aus39.md)）。

通过减少请求的输出量，可以显著降低模态动态分析的计算成本。
