# 42.6 选择结果选项

Abaqus/CAE提供了多种方法来显示存储在输出数据库中的场输出结果。其中一些方法要求对最初保存在单元形心或积分点位置的场输出数据进行节点位置计算。这些计算适用于线和带状等值线图、节点位置探测、基于结果值形成显示组或颜色编码，以及沿路径提取基于单元的*X–Y*数据。您可以控制与这些计算相关的一些选项，例如如何对基于单元的结果进行平均。

此外，您可以选择对场输出结果应用坐标系变换；对于复数结果，您可以从几种显示形式中进行选择：幅值、相角、实部、虚部或指定角度的值；以及您可以选择是否将结果缓存到内存中以提高性能。
