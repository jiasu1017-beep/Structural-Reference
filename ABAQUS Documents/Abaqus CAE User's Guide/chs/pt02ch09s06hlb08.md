# 9.7.8 设置工作目录





工作目录是Abaqus/CAE在您提交作业进行分析时写入其生成文件的目录，如输入文件和输出数据库文件。有关更多信息，请参阅["理解创建和分析模型时生成的文件，"第9.4节"](pt02ch09s04.md)。

从主菜单栏选择****文件（File）**![](../graphics/images/arrow.gif)**设置工作目录（Set Work Directory）****来更改工作目录。当您启动Abaqus/CAE会话时，工作目录是您启动Abaqus/CAE的目录。更改工作目录不会更改保存重放文件的位置，也不会更改打开或保存文件（如模型数据库文件）的默认目录。但是，Abaqus/CAE确实使用新的工作目录来保存您在保存时不显示路径的任何文件。例如，报告文件（`abaqus.rpt`）被写入工作目录。

当您使用文件选择对话框时，可以单击工作目录图标![](../graphics/dbs-fileopen-setwork.png)来访问工作目录。（文件选择对话框显示您正在访问的目录的完整路径。）有关更多信息，请参阅["使用文件选择对话框，"第3.2.10节"](pt01ch03s02s10.md)。
