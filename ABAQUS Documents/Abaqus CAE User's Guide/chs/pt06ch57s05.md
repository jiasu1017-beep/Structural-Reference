# 57.5 使用等间距数据定义振幅曲线







使用等间距定义方法在从指定时间值开始的固定时间间隔处提供振幅值列表。Abaqus 在每个时间间隔之间进行线性插值。有关更多信息，请参见 Abaqus Analysis User's Guide](../usb/usb-link.md#usb-prc-pamplitude-data-eqspaced) 的"振幅曲线"第 34.1.2 节中的"定义等间距数据"。

**使用等间距数据定义振幅曲线：**

1. 如 ["选择要定义的振幅类型，" 第 57.3 节](pt06ch57s03.md) 中所述显示 **编辑振幅** 对话框。
2. 点击 **时间跨度** 字段旁边的箭头，并指定您希望如何将振幅定义为时间的函数：- 选择 **步骤时间** 表示从每个步骤开始测量的时间。- 选择 **总时间** 表示在所有非扰动分析步骤中累积的总时间。
3. 指示您希望如何定义 **平滑**：- 选择 **使用求解器默认值** 以接受 Abaqus/Standard 中的默认值 0.25 和 Abaqus/Explicit 中的默认值 0.0。- 选择 **指定** 以在相邻字段中输入平滑参数的值。建议为包含大时间间隔的振幅定义使用 0.05 的值，以避免与指定定义的严重偏差。**平滑** 参数是在每个时间点之前和之后的时间间隔的分数，在该时间间隔内分段线性时间变化被平滑二次时间变化替换。此参数仅在需要时间导数时（用于直接积分动态分析中的位移或速度边界条件）适用，对于此选项的所有其他用途都会被忽略。
4. 显示 **振幅数据** 选项卡页面。
5. 在 **固定间隔** 字段中，输入您将提供振幅数据的固定时间或频率间隔。
6. 在数据表的 **时间/频率** 列的第一行中，输入您将输入第一个振幅的时间（或最低频率）。
7. 输入每个时间或频率的 **振幅** 值。有关如何输入数据的详细信息，请参见 ["输入表格数据，" 第 3.2.7 节](pt01ch03s02s07.md)。
8. 如果需要，显示 **基线校正** 选项卡页面。当您使用振幅定义来定义时域中的加速度历史记录（例如地震记录）时，加速度记录通过时间的积分可能导致事件结束时出现相对较大的位移。这种行为通常是由于仪器误差或采样频率不足以捕获实际加速度历史记录造成的。在 Abaqus/Standard 中，可以使用"基线校正"来补偿它。有关更多信息，请参见 Abaqus Analysis User's Guide](../usb/usb-link.md#usb-prc-pamplitude-baseline) 的"振幅曲线"第 34.1.2 节中的"Abaqus/Standard 中的基线校正"。
9. 点击 **校正** 字段旁边的箭头，并选择以下选项之一：- 选择 **无** 表示无基线校正。- 选择 **单间隔** 将整个振幅定义时间视为单个校正间隔。- 选择 **多间隔** 将整个振幅定义时间视为多个校正间隔。如果您选择此选项，请输入定义不同校正间隔的时间点（即，在第一行中输入定义第一个校正间隔结束和第二个校正间隔开始的时间点；在第二行中输入定义第二个校正间隔结束和第三个校正间隔开始的时间点，等等）。
10. 点击 **确定** 保存您的振幅定义并关闭 **编辑振幅** 对话框。
