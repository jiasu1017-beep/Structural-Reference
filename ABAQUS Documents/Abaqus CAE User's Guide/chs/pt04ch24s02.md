# 24.2 什么是连接器？






连接器允许您对装配中两点之间的机械关系进行建模。连接可以是简单的，例如链条，也可以施加更复杂的约束，例如恒速接头。连接器几何使用包含一根或多根线框的装配级线框特征进行建模。线框可以连接装配中的两点，也可以将一点连接到地面。您创建的连接器截面指定连接类型和连接器行为，例如弹簧状弹性行为。为了完成连接器建模，您创建连接器截面分配，将连接器截面与您选择的线框关联，并指定您选择的线框的第一点和第二点的取向。有关连接器的更多信息，请参见["连接器：概述，" Abaqus 分析用户指南第 31.1.1 节](../usb/usb-link.md#usb-elm-econnectoroverview)。

例如，[图 24-1](pt04ch24s02.md#adv-conn-multimech) 展示了一个曲柄机构，该机构用于例题["曲柄机构，" Abaqus 例题指南第 4.1.2 节](../exa/exa-link.md#exa-mec-crank)。
