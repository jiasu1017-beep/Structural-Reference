# 20.12.5 添加和编辑参数


参数是具有变量名的约束。它们可以具有数值,也可以具有涉及其他参数的数学表达式(参数方程)的值。参数可以与当前草图中的尺寸关联,也可以是独立于草图的数值常量和表达式。与尺寸关联的参数的值直接影响草图,而独立参数只有在您使用它们为草图中的参数创建表达式时才会影响草图。

您可以使用**参数管理器**添加和编辑参数。您可以使用**尺寸**按钮通过选择尺寸来创建新参数,也可以通过在管理器的空白行中输入来添加独立参数。

**添加或编辑参数的步骤:**

1. 从 Sketcher 工具箱中的工具中,选择**参数管理器**工具 ![](../graphics/ico_parameter.png)。有关 Sketcher 工具箱中工具的图示,请参见["Sketcher 工具," 第 20.4.1 节](pt03ch20s04s01.md)。**参数管理器**会出现。**提示:**您也可以从主菜单栏选择****编辑**![](../graphics/images/arrow.gif)**参数管理器****来打开**参数管理器**,或者在**编辑尺寸**对话框中使用 **f(x)** 按钮。
2. 单击**尺寸**,然后从草图中选择尺寸以将它们与新参数关联。Abaqus/CAE 使用 `dimension_*x*` 形式的默认名称将新参数添加到**参数管理器**。
3. 单击**名称**列中的单元格可以编辑现有名称,或者通过编辑空白单元格来添加新参数名称。如果所选参数与尺寸关联,Abaqus/CAE 会在草图中高亮显示该参数。
4. 单击**表达式**列中的单元格可以直接在单元格中编辑,或选择它以使用**表达式构建器**进行编辑。您必须先定义要在表达式中使用的所有参数,然后才能定义表达式。**提示:**使用管理器右侧的按钮可以插入、重新排列或删除行。
5. 如果需要,使用**表达式构建器**编辑表达式。有关详细说明,请参见["创建参数方程," 第 20.12.6 节](pt03ch20s12hlb06.md)。
6. 在**参数管理器**中单击**应用**,以在草图中实现您的更改。Abaqus/CAE 会使用任何新的表达式值更新草图,并且如果**显示参数名称**在**Sketcher 选项**中被切换打开,则显示修改后的参数名称。
