# 76.10 控制各个坐标系的显示





您可以在视口中高亮显示、显示和隐藏各个坐标系。Abaqus/CAE 在建模和后处理期间提供以下显示选项：

**在建模期间控制基准坐标系的显示**

您创建的所有基准几何（包括基准坐标系）都被视为适用于它们的零件或装配的特征。Abaqus/CAE 在模型树中零件或装配的 **Features** 容器下为基准坐标系和其他特征提供快捷方式。您可以通过点击模型树中的快捷方式来高亮显示各个基准坐标系；高亮显示时，坐标系在视口中以红色渲染，并显示其标题。要在视口中隐藏或显示坐标系，请右键点击快捷方式并选择 **Suppress** 或 **Resume**。

您还可以通过将各个基准坐标系添加到显示组来控制它们的显示。有关更多信息，请参见 ["管理显示组，" 第78.2节](pt07ch78s02.md)。

**在后处理期间控制任何坐标系的显示**

在可视化模块中，可用的坐标系分为两组：ODB 坐标系，它们是输出数据库文件的一部分；以及会话坐标系，它们是在后处理期间创建的。会话坐标系仅适用于一个输出数据库，并且仅在您的 Abaqus/CAE 会话期间持续存在，除非您将会话坐标系移到输出数据库。请参见 ["将坐标系保存到输出数据库文件，" 第42.8.6节](pt05ch42s06hlb05.md)。

您可以使用以下任一技术来高亮显示、显示或隐藏各个坐标系：
- 所有可用的坐标系在结果树中的 **ODB Coordinate Systems** 和 **Session Coordinate Systems** 容器下都有快捷方式。您可以点击任意快捷方式在视口中高亮显示该坐标系；高亮显示时，坐标系在视口中以红色渲染，并显示其标题。您还可以通过右键点击快捷方式并从出现的列表中选择布尔运算符来显示或隐藏任意坐标系。
- 您可以通过将 ODB 坐标系和会话坐标系添加到显示组来控制它们的显示，显示组可以使用布尔显示选项显示或隐藏。请参见 ["管理显示组，" 第78.2节](pt07ch78s02.md)，了解更多信息。

**坐标系管理器** 还提供有关所选输出数据库的 ODB 坐标系和会话坐标系的信息。您不能从此管理器更改坐标系的显示，但可以重命名或删除它们。请参见 ["在后处理期间创建坐标系，" 第42.8节](pt05ch42s06.md)。
