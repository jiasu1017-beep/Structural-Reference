# 3.2.6 三角形、四面体和楔形单元

### 3.2.6 三角形、四面体和楔形单元

**产品：** Abaqus/Standard  Abaqus/Explicit

Abaqus 中的实体单元库包括用于平面、轴对称和三维分析的一阶和二阶三角形、四面体和楔形单元。

这些单元的混合版本用于不可压缩和近似不可压缩本构模型（参见"混合不可压缩实体单元公式"，第3.2.3节，了解所用公式的详细讨论）。但是，这些混合形式应仅用于填充由六面体单元组成的网格中的区域；否则，可能会引入过多的约束变量。

二阶四面体不适用于接触问题分析：单元面上的恒定压力在角节点处产生零等效载荷。在接触问题中，这使得角点处的接触条件变得不确定，可能由于过大的间隙跳动而导致求解失败。相同的论证适用于楔形单元三角形面上的接触。

### 积分

一阶三角形和四面体是常应力单元，在应力/位移应用中用于刚度计算时使用单个积分点。这两个单元都使用集中质量矩阵，总质量平均分配到各个节点。对于传热应用，一阶三角形的传导率和热容矩阵使用三点积分格式，积分点位于顶点与单元重心之间的中点；一阶四面体使用四点积分格式。分布载荷分别使用两个和三个积分点对一阶三角形和四面体进行积分。

三点格式也用于二阶三角形在应力/位移应用中的刚度。质量矩阵使用六点格式积分，可以精确积分四次多项式（[Cowper, 1973](07s01a01-References.md)）。分布载荷使用三个点积分。该单元的传热版本使用六点格式用于传导率和热容矩阵。

对于应力/位移应用，二阶四面体使用4个积分点用于其刚度矩阵，15个积分点用于其一致质量矩阵。对于传热应用，传导率和热容矩阵使用15个积分点积分。一阶楔形使用2个积分点用于其刚度矩阵，但使用6个积分点用于其集中质量矩阵。二阶楔形使用9个积分点用于其刚度矩阵，但使用18个积分点用于其一致质量矩阵。二阶四面体和楔形单元使用的积分格式可在[Stroud（1971）](07s01a01-References.md)中找到。
