# 49.2.1 生成时间历史动画





通过时间历史进行动画化会根据实际分析结果生成随时间变化的图序列。您可以选择对变形形状、等值线、符号或材料方向图进行动画化。

**生成时间历史动画：**

1. 使用"文件"菜单打开包含分析结果的[输出数据库](pt02ch09s06hlb02.md)。
2. 要绘制Abaqus默认选择的以外的变量，请使用"结果"菜单。- 对于变形或材料方向图的动画，选择Abaqus随时间获取值的[变形场输出变量](pt05ch42s03hlb04.md)。- 对于等值线或符号图的动画，选择要显示的[主场输出变量](pt05ch42s03hlb03.md)或[符号场输出变量](pt05ch42s03hlb05.md)。如果您想在变形形状上显示动画化等值线或符号图，请选择[变形场变量](pt05ch42s03hlb04.md)。有关场输出变量选择的更多信息，请参阅["场输出变量选择概述，"第42.5.1节](pt05ch42s03hlb01.md)。
3. 要自定义Abaqus/CAE包含在动画中的步骤和帧，请找到"活动步骤/帧"对话框。从主菜单栏中，选择****结果**![](../graphics/images/arrow.gif)**活动步骤/帧****。有关激活步骤和帧的更多信息，请参阅["自定义结果中步骤和帧的显示，"第42.4节](pt05ch42s02.md)。
4. 使用"选项"和"视图"菜单选择您想要的[自定义选项](pt05ch40s03s03.md)。这些可能包括变形、等值线、符号、方向和视图选项。
5. 要启动动画，请从主菜单栏中选择****动画**![](../graphics/images/arrow.gif)**时间历史****。**提示：**您也可以使用工具箱中的![](../graphics/ico_animTimeHistory.png)工具来生成时间历史动画。动画在所有动画化视口中开始，动画控件出现在上下文栏的右侧。Abaqus现在处于时间历史动画绘图状态。在此状态下，每次您单击"结果"、"选项"或"视图"菜单对话框中的"应用"或"确定"时，Abaqus会自动刷新您的图。所有动画化视口保持时间历史动画状态，直到您在其他状态（如未变形或变形状态）中生成图。要了解如何控制动画，请参阅["控制动画播放，"第49.4节](pt05ch49s04.md)。
