# 17.4.3 控制种子密度





您可以使用以下方法控制所选边上的种子密度：
- 为整个零件或零件实例的每条边指定平均单元尺寸。
- 指定沿边的所需单元数量。
- 指定沿边的平均单元尺寸。（如果边长度不是单元长度的整数倍，Abaqus/CAE会稍微更改单元长度以获得沿边的整数个单元。）
- 指定沿边非均匀分布的单元。单元密度可以从边的一端增加到另一端（单侧偏差），或者单元密度可以从边的中心变化到两端（双侧偏差）。对于非均匀分布，您可以指定以下任一选项：
  - 沿边所需的单元数量和偏差比。偏差比是最大单元与最小单元的比值。
  - 最小单元的尺寸和最大单元的尺寸。

如果您选择之前使用这些方法的组合进行布种的边，Abaqus/CAE会提供**保持原样**选项，允许您保留布种方法。如果选择具有混合曲率控制或种子约束的边，Abaqus/CAE会提供类似的选项。

通过为整个零件或零件实例指定平均单元尺寸创建的种子分别称为零件种子或实例种子，以白色显示；使用其他方法创建的种子称为边缘种子，以品红色显示。边缘种子始终覆盖零件或实例种子；因此，当您为整个零件或零件实例指定平均单元尺寸时，零件或实例种子仅出现在尚无边缘种子的区域的边上。分割创建的新边默认获得零件或实例种子。

当您对分配了扫掠或旋转网格技术的区域的边进行布种时，边缘布种工具会自动将种子从所选边传播到区域中的匹配边。换句话说，扫掠路径起始处的面或边的种子会自动传播到扫掠路径末端的面或边。同样，沿扫掠路径创建的一条边上的种子会自动传播到沿扫掠路径的其他边。因此，即使您只选择面的单条边进行布种，Abaqus/CAE也会将种子传播到其他边和面。有关更多信息，请参见["什么是扫掠网格？"第17.9.1节](pt03ch17s09s01.md)。
