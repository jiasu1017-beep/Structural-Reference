# 44.1.3 理解云图限制





云图中显示的最小值和最大值可以由Abaqus/CAE计算，也可以由您指定。您可能需要指定一个或两个限制；例如，要消除极值或在固定范围内检查变化。

云图仅显示在模型可见表面上；但是，根据您选择的云图类型和结果选项，云图值可以根据当前显示组中的所有单元或模型中的所有单元计算。（有关更多信息，请参见["选择结果选项，" 第42.6节](pt05ch42s04.md)。）因此，报告的最小值和最大值可能出现在内部单元上。您可以使用显示组仅显示绘图中的内部单元来显示模型内部的云图，或者收缩所有单元关于其质心，使内部单元在外部单元主体之间可见。

Abaqus/CAE根据显示的场输出变量类型（节点或单元）、云图类型（拼布型与线型或带型）以及您选择的选项以不同方式计算最小值和最大值。对于节点场输出变量的云图，限制是最大和最小节点值。对于拼布型云图，限制是最小和最大单元面值。对于基于单元场输出变量的线型、带型或等值面型云图，最小值和最大值由外推并根据当前平均标准平均的值确定。

当Abaqus/CAE在节点处计算平均值上的最小值和最大值时，限制基于用于创建图表的值。因此，图例边界可能随您改变平均标准而变化。要了解如何控制平均标准，请参见["控制结果平均，" 第42.6.6节](pt05ch42s04hlb02.md)。

您还可以控制用于云图动画每一帧的云图限制：计算第一帧和最后一帧的限制、当前帧的计算限制、每个单独帧的重新计算限制，或所有帧的计算限制。基于动画中所有帧计算限制需要一次完整遍历动画序列，以建立用于后续遍历的云图限制。

有关控制云图限制的详细信息，请参见["设置云图限制，" 第44.5.7节](pt05ch44s02hlb07.md)。要了解如何显示与云图关联的最小值和最大值，请参见["自定义图例，" 第56.1节](pt05ch56s01.md)。
