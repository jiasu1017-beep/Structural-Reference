# 34.3 Load case editors







After you enter the Load module and create loads and boundary conditions, you can select ****Load Case**![](../graphics/images/arrow.gif)**Create**** from the main menu bar to create a load case. A **Create Load Case** dialog box appears in which you can provide a name for the load case and select the step in which the load case will be created. When you click **Continue** in the **Create Load Case** dialog box, the load case editor appears.

The top panel of the load case editor displays the name of the load case and the current analysis step. The load case editor contains two tabbed pages with tables that allow you to select the loads and boundary conditions to define the load case. By default, Abaqus/CAE includes all boundary conditions propagated or modified from the base state in each load case that you create. You can toggle off this behavior on the **Boundary Condition** tabbed page.

Click ![](../graphics/ico_add.png) at the bottom of each tabbed page to select loads and boundary conditions from a list. You can enter a scale factor to scale the magnitude of individual loads and boundary conditions. If you enter a negative scale factor for a load, the load will be applied in the opposite direction. You can also enter the names of the loads and boundary conditions and the scale factors directly in the tables. During load case creation, Abaqus/CAE sorts the names of the loads and boundary conditions alphabetically and lists the names in the load case editor. You can highlight selected loads and boundary conditions in the viewport.

For example, you can define a set of load cases to analyze the response of a cantilever beam subjected to an end load and a uniform pressure load. The cantilever beam model is shown in [Figure 34--2](pt04ch34s03.md#usi-adv-lcmodel). 

**Figure 34–2** Cantilever beam model with end load, pressure load, and encastre boundary condition.

![](../graphics/adv-lcmodel.png)

An encastre boundary condition is applied in the initial step, and the end load and pressure load are applied in a static, linear perturbation step.

You can create a load case that contains only the end load and another that contains only the pressure load. In addition, you can create a load case that contains both the pressure load and the end load with the end load applied in the opposite direction (scale factor value of 1), as shown in [Figure 34--3](pt04ch34s03.md#usi-adv-lceditor-load). 

**Figure 34–3** Load case definition that includes loads for the cantilever beam model.

![](../graphics/adv-lceditor-load-nls.png)

By default, each load case contains the encastre boundary condition propagated from the base state, as shown in [Figure 34--4](pt04ch34s03.md#usi-adv-lceditor-bc). 

**Figure 34–4** Load case definition that includes boundary conditions propagated from the base state for the cantilever beam model.

![](../graphics/adv-lceditor-bc-nls.png)
