# 4.1.17 UFLUID







### 4.1.17 [`UFLUID`](../sub/sub-link.md#sub-xsl-ufluid)

**Product: **Abaqus/Standard  

### Features tested

This section provides basic verification tests for the fluid behavior associated with fluid elements that are generated in Abaqus/Standard when the fluid cavity capability is used.

### Elements tested

F2D2    SPRING1    

### Feature tested

User subroutine to define fluid density and fluid compliance for an ideal gas.

### Problem description

The fluid is modeled using a two-dimensional fluid block that measures 1  1 with unit thickness. The user-defined fluid is modeled as an ideal gas with the following properties:

| Ambient pressure, ![](../graphics/ver_eqn00561.gif) = 14.7 |
| --- |
| Absolute zero temperature, ![](../graphics/ver_eqn00451.gif) = 460. |
| Reference density, ![](../graphics/ver_eqn00562.gif) = 10.0 |
| Reference pressure for density, ![](../graphics/ver_eqn00563.gif) = 0. |
| Reference temperature for density, ![](../graphics/ver_eqn00564.gif) = 200. |

**Loading: **

The following five steps are executed:

1. Load the fluid to induce a pressure of 100.0 units.
2. Increase fluid temperature to 300.0.
3. Add prescribed amount of fluid.
4. Remove prescribed amount of fluid.
5. Decrease fluid temperature to 200.0.

### Results and discussion

The results match the exact solution.
