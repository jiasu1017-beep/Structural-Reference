# 72.1 什么是参考点？







参考点是您在部件上创建的一个点。您也可以在装配体上创建参考点。您可以将参考点定位在空间中的任何位置，当模型中需要创建一个顶点不可用的点时，参考点非常有用；例如，在孔的中心。与顶点不同，参考点在生成网格时会被Mesh模块忽略。

您可以使用Part模块中的"参考点工具集"通过从主菜单栏选择****Tools**![](../graphics/images/arrow.gif)**Reference Point****来创建与部件关联的参考点。创建装配体时，参考点会出现在装配体中该部件的每个实例上。部件只能包含一个参考点，Abaqus/CAE会为参考点添加标签**RP**。如果您尝试分配第二个点，Abaqus/CAE会询问您是否要删除原始点。

或者，您可以使用Assembly、Interaction或Load模块中的"参考点工具集"通过从主菜单栏选择****Tools**![](../graphics/images/arrow.gif)**Reference Point****在装配体上创建参考点。在Interaction模块中，您可以使用模块工具箱中的![](../graphics/ico_referencePoint.png)工具创建参考点。装配体可以包含多个参考点，Abaqus/CAE会为它们添加标签**RP-1**、**RP-2**、**RP-3**等。有关更多信息，请参见["创建参考点," 第72.3节](pt06ch72hla01.md)。如果需要，您可以关闭参考点符号和参考点标签的显示；有关更多信息，请参见["控制参考点显示," 第76.11节](pt07ch76hla10.md)。
