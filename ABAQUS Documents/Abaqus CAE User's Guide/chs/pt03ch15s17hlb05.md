# 15.17.5 指定预定义的摩擦参数或接触力

您可以使用连接器截面编辑器中的**预定义摩擦参数**标签页为预定义的连接器摩擦模型指定几何常数和内部接触力。您可以使用连接器截面编辑器中的**接触力**标签页为用户定义的连接器摩擦模型指定摩擦产生的接触力和内部接触力。有关更多信息，请参见["连接器摩擦行为，"Abaqus分析用户指南第31.2.5节](../usb/usb-link.md#usb-elm-econnfrictionbehav)。

**要指定预定义的摩擦参数：**

1. 按照["定义摩擦，"第15.17.3节"](pt03ch15s17hlb03.md)中概述的过程显示连接器截面编辑器，并输入必要的数据。
2. 选择**预定义**作为**摩擦模型**。
3. 选择**预定义摩擦参数**标签页。
4. 根据您选择的连接类型指定几何常数和内部接触力。要显示定义所需数据输入的连接类型描述，请参见["连接类型，"第15.8.1节"](pt03ch15s08s01.md)。

**要指定摩擦产生的接触力和内部接触力：**

1. 按照["定义摩擦，"第15.17.3节"](pt03ch15s17hlb03.md)中概述的过程显示连接器截面编辑器，并输入必要的数据。
2. 选择**用户定义**作为**摩擦模型**。
3. 选择**接触力**标签页。
4. 如果需要，打开**指定分量**以使用固有连接器分量或导出连接器分量定义摩擦产生的接触力/力矩。
   - 选择**固有分量**，然后选择与您正在定义摩擦产生接触力/力矩的可用相对运动分量一致的力/力矩。
   - 选择**导出分量**，然后点击![](../graphics/ico_edit.png)以定义连接器导出分量。有关更多信息，请参见["指定连接器导出分量，"第15.17.15节"](pt03ch15s17hlb15.md)。
5. 如果**指定分量**处于关闭状态，则必须在**接触力**标签页的**内部接触力**部分的数据表中输入内部接触力的值。如果需要，您可以执行以下操作来指定内部接触力：
   - **使用独立分量** 打开**使用独立分量**，选择**位置**或**运动**，然后选择要用作自变量的可用相对位置或本构运动分量。在表的相应字段中输入由独立分量指定方向的连接器相对位置或本构运动。
   - **累积滑移** 打开**使用累积滑移依赖**，然后输入滑动方向上的累积滑移。
   - **温度** 打开**使用与温度相关的数据**，然后输入温度。
   - **场1、场2等** 指定**场变量数量**，然后输入第一个场变量的值、第二个场变量的值，等等。
   - **内部接触力** 指定内部接触力数据。
6. 要修改正则化（Abaqus/Explicit分析专用）或数据外推的行为设置，请使用["为表格数据指定行为设置，"第15.17.14节"](pt03ch15s17hlb14.md)中所述的过程。
