### 识别协同仿真接口区域

当耦合 Abaqus/CFD 到 Abaqus/Standard 或 Abaqus/Explicit 时，您使用表面指定接口区域。您必须定义基于单元的表面，并且只能指定一个表面用作分析中的接口区域。您可能在模型定义中共享的区域中具有不同的网格。

| **输入文件用法：** | 使用以下选项将基于单元的表面定义为协同仿真区域： |
| --- | --- |
|  | ``` [*CO-SIMULATION REGION](../key/key-link.md#usb-kws-hcosimulationregion), TYPE=SURFACE *surface_A* ``` |

| **Abaqus/CAE 用法：** | 相互作用模块：**创建相互作用**：**流固耦合协同仿真边界**：选择表面区域 |

### 定义会合方案

SIMULIA Co-Simulation Engine 配置文件用于定义时间增量过程和两个 Abaqus 分析之间交换的频率。Abaqus/CAE 自动创建和使用此配置文件。如果您不使用 Abaqus/CAE 执行协同仿真，则必须手动创建配置文件。

预定义模板可用于常用耦合方案。您可以在创建配置文件时引用这些模板。本节描述会合方案设置和预定义配置文件模板。

#### 定义耦合方案

顺序显式耦合方案（也称为 Gauss-Seidel 耦合算法）是 Abaqus/CFD 到 Abaqus/Standard 或到 Abaqus/Explicit 协同仿真可用的唯一耦合方案。在所有预定义模板中，Abaqus/CFD 分析滞后于协同仿真，Abaqus/Standard 或 Abaqus/Explicit 分析领先于协同仿真。对于共轭热传递，Abaqus/CFD 分析可以滞后或领先于协同仿真。对于流固耦合，Abaqus/CFD 分析必须滞后于协同仿真，Abaqus/Standard 或 Abaqus/Explicit 分析必须领先于协同仿真。

#### 耦合步长

耦合步长是连续两次协同仿真数据交换之间的时间段。耦合步长根据分析类型自动确定，用于获得耦合物理问题的时精确解。对于耦合 Abaqus/CFD 和 Abaqus/Standard 的流固耦合（FSI）和共轭热传递（CHT）分析，耦合步长是个体分析自动时间增量方案确定的最小时间步长。对于耦合 Abaqus/CFD 和 Abaqus/Explicit 的 FSI 问题，Abaqus/Explicit 从 Abaqus/CFD 导入耦合步长；因此，Abaqus/CFD 将耦合步长导出到 Abaqus/Explicit。

#### 时间增量方案

根据分析类型，Abaqus 可能执行每个耦合步一个增量（称为"锁定步"）或几个增量（称为"子循环"）。对于耦合 Abaqus/CFD 和 Abaqus/Explicit 的 FSI 分析，Abaqus/Explicit 通常使用子循环，而 Abaqus/CFD 使用锁定步行为。

### 执行耦合分析

您可以在 Abaqus/CAE 中交互执行协同仿真或从命令行执行，如 ["执行协同仿真，" 第 17.3.4 节"](pt04ch17s03aus102.md) 中所述。默认情况下，当耦合 Abaqus/CFD 到 Abaqus/Explicit 时，Abaqus/Explicit 打包程序和分析都以单精度运行。

### 限制

除 ["准备 Abaqus 分析进行协同仿真，" 第 17.2.1 节"](pt04ch17s02aus98.md) 中讨论的限制外，以下限制适用于 Abaqus/CFD 到 Abaqus/Standard 或到 Abaqus/Explicit 协同仿真。

#### 一般限制

只能将单个区域定义为接口区域；不支持多个接口区域。

#### Abaqus/Standard 和 Abaqus/Explicit 分析限制

以下 Abaqus/Standard 和 Abaqus/Explicit 单元不能用于与 Abaqus/CFD 的协同仿真：
- 具有扭转自由度的轴对称单元（CGAX 单元族）
- 具有不对称变形的轴对称实体单元（CAXA 单元族）
- 广义平面应变单元（CPEG 单元族）
- 耦合孔隙压力-位移单元
- 声学单元
- 压电单元
