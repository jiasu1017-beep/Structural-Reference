# 4.2.7 LE7：压力作用下的轴对称圆柱体/球体







**产品：** Abaqus/Standard  Abaqus/Explicit  

### 测试单元

SAX1    SAX2  

### 问题描述

![](../graphics/exxle7-model-nls.png)

**模型：**

薄壁压力容器。

**网格：**

对每个单元分别测试粗网格和细网格。

**材料：**

线弹性，弹性模量 = 210 GPa，泊松比 = 0.3，密度 = 7800 kg/m3。

**边界条件：**

在A点处![](../graphics/bmk_eqn01704.gif) = 0。在F点处![](../graphics/bmk_eqn01705.gif) = 0。

**载荷：**

均匀内压1.0 MPa。在显式动态分析中，载荷的施加方式使得可以获得准静态解。

### 参考解

这是英国国家有限元方法与标准机构（NAFEMS）推荐的测试：NAFEMS出版物TNSB第3版"The Standard NAFEMS Benchmarks"（1990年10月）中的测试LE7。

目标解：D点处外表面上的轴向应力![](../graphics/bmk_eqn01706.gif) = 25.9 MPa。

### 结果与讨论

结果如下表所示。括号中的值是相对于目标解的百分比差异。

| 单元 | ![](../graphics/bmk_eqn01707.gif)，粗网格 | ![](../graphics/bmk_eqn01707.gif)，细网格 |
| --- | --- | --- |
| SAX1 (Abaqus/Explicit) | 25.6 MPa (1%) | 25.7 MPa (0.5%) |
| SAX2 (Abaqus/Standard) | 26.034 MPa (+0.67%) | 25.878 MPa (+0.07%) |
