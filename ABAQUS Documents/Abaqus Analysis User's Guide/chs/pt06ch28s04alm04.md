# 28.4.1 翘曲单元





**产品：** Abaqus/Standard  

##### **参考资料**

- ["网格化梁截面," 第10.6.1节](pt04ch10s06at35.md)
- [*SOLID SECTION](../key/key-link.md#usb-kws-msolidsection)

### 概述

翘曲单元：
- 用于对任意形状的梁截面轮廓进行建模，以便与Timoshenko梁一起使用；
- 与["网格化梁截面," 第10.6.1节](pt04ch10s06at35.md)中所述的梁截面生成程序结合使用；以及
- 仅模拟线性弹性行为。

### 典型应用

翘曲单元是专用单元，用于将梁截面的二维模型离散化。这个二维截面模型在Abaqus/Standard中用于计算翘曲函数的平面外分量，以及后续梁分析（无论是Abaqus/Standard还是Abaqus/Explicit）中所需的截面刚度和质量特性。适用于任何整体行为类似梁但截面为非标准或包含多种材料的结构。例如，用于执行鞭击分析的船舶截面、翼形转子叶片或机翼的梁模型、层合工字梁等。

### 选择合适的单元

为了网格化任意形状的实心梁截面，Abaqus/Standard提供了两种单元：3节点线性三角形单元WARP2D3和4节点双线性四边形单元WARP2D4。截面网格中相邻单元必须共享公共节点；不允许使用多点约束进行网格细化。

### 定义单元的截面特性

使用实心截面定义来定义截面特性。必须将这些特性与模型的某个区域关联。不需要其他数据。

| **输入文件用法：** | ``` [*SOLID SECTION](../key/key-link.md#usb-kws-msolidsection), ELSET=*name* ``` |
| --- | --- |
|  | 其中ELSET参数引用一组翘曲单元。 |

#### 为一组翘曲单元分配材料定义

必须将线性弹性材料定义与每个翘曲单元截面定义关联。可选择将材料方向定义与截面关联（参见["方向," 第2.2.5节](pt01ch02s02aus15.md)）。

仅各向同性线性弹性（["定义各向同性弹性" in "线性弹性行为," 第22.2.1节](pt05ch22s02abm02.md#usb-mat-clinearelastic-isotropic)）或翘曲单元的正交各向异性线性弹性（["为翘曲单元定义正交各向异性弹性" in "线性弹性行为," 第22.2.1节](pt05ch22s02abm02.md#usb-mat-clinearelastic-orthowarp)）是翘曲单元有效的材料模型。

| **输入文件用法：** | ``` [*SOLID SECTION](../key/key-link.md#usb-kws-msolidsection), ELSET=*name*, MATERIAL=*name*, ORIENTATION=*name* ``` |
