# 18.8 创建目标函数





目标函数定义您的优化的目标。 要配置目标函数，请选择设计响应或设计响应组合并指定目标；例如，最小化模型中的总应变能。

**创建或编辑目标函数：**

1. 从主菜单栏，选择 ****Objective Function**![](../graphics/images/arrow.gif)**Create****。 出现**Create Objective Function**对话框。 **提示：**您可以通过另外两种方式启动**Create**过程： - 单击**Objective Function Manager**中的**Create**。 （您可以通过从主菜单栏选择 ****Objective Function**![](../graphics/images/arrow.gif)**Manager**** 来显示**Objective Function Manager**。） - 单击Optimization模块工具箱中的 ![](../graphics/icoR_optObjFunc.png) 工具。
2. 从出现的**Create Objective Function**对话框中，输入目标函数的名称。 出现**Edit Objective Function**对话框。
3. 选择目标函数的**Target**。 您可以选择以下之一： - **Minimize the sum of the weighted differences from the reference**：创建尝试最小化设计响应与参考值之间加权差异之和的优化模型。 - **Maximize the sum of the weighted differences from the reference**：创建尝试最大化设计响应与参考值之间加权差异之和的优化模型。 - **Minimize the maximum weighted difference from the reference**：找到设计响应与参考值之间的最大加权差异，并创建最小化该最大差异的优化模型。
4. 从**Name**列的菜单中，选择将用于定义此目标函数的设计响应。 目标函数编辑器显示设计响应的类型及其默认权重和参考值。
5. 要创建作为多个设计响应加权和的目标函数，请从以下选项中选择： - 选择 ![](../graphics/ico_add.png) 将设计响应添加到设计响应列表。 - 选择 ![](../graphics/icoR_addAll.png) 将所有有效设计响应添加到设计响应列表。 - 选择 ![](../graphics/ico_delete.png) 从设计响应列表中删除设计响应。 - 选择 ![](../graphics/icoR_optDesResp.png) 创建新的设计响应并将其添加到设计响应列表。
6. 如果需要，单击目标函数的权重或参考值并输入新值。
7. 单击**OK**保存数据并退出编辑器。
