# 64.5.10 选择要编辑的节点





当您选择要编辑的节点时，Abaqus/CAE 会在提示区域显示一个菜单，允许您选择用于选择节点的技术。这些技术允许您选择单个节点或一组相邻节点。

**选择要编辑的节点的步骤：**

1. 使用以下方法之一来选择要编辑的节点：**单独选择节点：** 1. 从提示区域的菜单中选择 **单独**。 2. 选择要重新编号的节点，或拖动选择一组节点。 3. 按 **[Shift]**+**点击** 可将其他节点添加到您的选择中。 4. 如有必要，按 **[Ctrl]**+**点击** 可从选择中移除已选的节点。**使用角度方法选择节点：** 1. 从提示区域的菜单中选择 **按角度**。 2. 输入角度（0 到 90），然后选择一个节点。Abaqus/CAE 会选择与所选节点相邻的所有单元面上 的每个节点，直到相邻单元面之间的角度等于或超过您输入的角度。（有关更多信息，请参见 ["使用角度和特征边方法选择多个对象，" 第6.2.3节](pt01ch06s02hlb03.md)。） 3. 使用 **按角度** 方法后，您可以按 **[Shift]**+**点击** 可将其他节点添加到选择中，或按 **[Ctrl]**+**点击** 从选择中移除已选的节点。（有关更多信息，请参见 ["组合选择技术，" 第6.2.8节](pt01ch06s02hlb08.md)。）**使用特征边方法选择节点：** 1. 从提示区域的菜单中选择 **按特征边**。 2. 输入角度（0 到 90）。Abaqus/CAE 通过查找所有单元边（其中两个相邻单元面之间的角度大于指定角度）来识别模型中的所有特征边。 3. 选择一个单元边或节点。Abaqus/CAE 会沿着穿过所选单元边或节点的特征边进行追踪。如果另一个特征边以大于上一步中指定的角度与之相交，则特征边会被截断。 4. Abaqus/CAE 会选择沿特征边的所有节点。（有关更多信息，请参见 ["使用角度和特征边方法选择多个对象，" 第6.2.3节](pt01ch06s02hlb03.md)。）或者，您可以选择多条特征边交点处的节点，Abaqus/CAE 会选择沿所有这些特征边的节点。 5. 使用 **按特征边** 方法后，您可以按 **[Shift]**+**点击** 可将其他节点添加到选择中，或按 **[Ctrl]**+**点击** 从选择中移除已选的节点。（有关更多信息，请参见 ["组合选择技术，" 第6.2.8节](pt01ch06s02hlb08.md)。）**通过指定现有节点集来选择节点：** 1. 点击提示区域右侧的 **集合**。Abaqus/CAE 显示 **区域选择** 对话框，其中包含您已创建的节点集列表。 2. 选择要重新编号的节点集，然后点击 **继续**。**注意：** 默认选择方法基于您最近使用的选择方法。如需恢复为其他方法，请点击提示区域右侧的 **在视口中选择** 或 **集合**。
2. 完成选择节点后，从提示区域选择 **完成**。
