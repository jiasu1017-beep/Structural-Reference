# 62.8.7 通过将现有面旋转指定角度创建datum平面







您可以通过围绕选定的旋转轴将选定的面或datum平面旋转指定角度来定义datum平面。

![](../graphics/dtm-hprot-nls.png)

**通过将面旋转指定角度来创建datum平面：**

1. 从主菜单栏中，选择****Tools**![](../graphics/images/arrow.gif)**Datum****。此时将出现**Create Datum**对话框。该对话框概述了您可以创建的datum几何体类型。Abaqus/CAE在提示区域中显示提示以指导您完成此过程。**提示：**您也可以使用模块工具箱中的![](../graphics/ico_dtmPlaneRotatePlane.png)工具创建datum平面。有关工具箱中datum工具的图示，请参见["使用Datum工具集，" 第62.2节](pt06ch62s02.md)。
2. 从对话框顶部的类型列表中，选择**Plane**。**Method**列表指示您可以使用的方法来创建datum平面。
3. 从**Method**列表中，选择**Rotate from plane**。
4. 从当前视口中的零件或装配中，选择要旋转的面或datum平面。
5. 选择任何边或datum轴来表示旋转轴。您为旋转轴选择的边或datum轴不必与为旋转选择的面或datum平面共面。Abaqus/CAE沿选定的边显示一个箭头，指示使用右手规则的旋转正方向。
6. 在提示区域出现的文本框中，输入旋转角度。datum平面会出现。您可以通过从主菜单栏中选择****Feature**![](../graphics/images/arrow.gif)**Edit****并选择datum平面来修改旋转角度。如果更改角度的符号，Abaqus/CAE会反转旋转方向。
