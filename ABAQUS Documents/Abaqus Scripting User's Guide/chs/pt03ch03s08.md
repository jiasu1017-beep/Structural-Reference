# 3.8 create方法





### 3.8 `create`方法

Abaqus GUI Toolkit中的大多数widget采用两阶段创建过程。第一阶段，widget构造函数构建widget的数据结构。第二阶段，工具包调用widget的`create`方法。`create`方法构造widget所需的所有窗口，以便widget可以显示在屏幕上。

在大多数情况下，应用程序启动脚本首先调用构建应用程序初始结构所需的所有widget的构造函数——通过构造主窗口。脚本然后调用应用程序对象的`create`方法。此调用遍历整个widget层次结构，调用每个widget的`create`方法。更多信息和示例脚本，请参见["启动脚本"，第11.2节](pt06ch11s02.md)。

如果在启动脚本调用应用程序的`create`方法之后创建widget，则必须在这些新widget上调用`create`方法；否则，它们不会在屏幕上可见。

如果您的对话框是由表单或过程发布的，基础设施会在对话框上调用`create()`方法。但是，如果您自己发布对话框，则必须在调用其`show()`方法之前在对话框上调用`create()`方法。

类似地，如果在widget创建之后构造图标，则必须在widget中使用图标之前在这些图标上调用`create()`方法。例如，如果要在对话框中已显示的标签上更改标签的图标，必须执行以下操作：

1. 构造新图标。
2. 调用新图标的`create()`方法。
3. 使用标签的`setIcon()`方法将图标传递给标签。
