# *SYMMETRIC MODEL GENERATION





### *SYMMETRIC MODEL GENERATION从轴对称或部分三维模型创建三维模型。

此选项用于通过以下方式创建三维模型：将轴对称模型的横截面绕对称轴旋转；将单个三维扇区绕对称轴旋转；或组合对称三维模型的两个部分，其中一部分是原始模型，另一部分通过一条线或一个平面反射原始部分获得。

**产品：**Abaqus/Standard

**类型：**模型数据

**级别：**此选项在以部件实例装配定义的模型中不受支持。

##### **参考：**

- ["对称模型生成," Section 10.4.1 of the Abaqus Analysis User's Guide](../usb/usb-link.md#usb-anl-aaximodelgen)

### **必需的互斥参数：**

PERIODIC

使用此参数指示必须通过绕对称轴旋转模型的单个三维扇区来生成周期性三维模型（参见[图18.57-1](ch18abk57.md#ksymm-model-gen-per)）。

设置PERIODIC=CONSTANT（默认）以指示周期性模型中的每个生成扇区具有恒定角度。

设置PERIODIC=VARIABLE以指示周期性模型中的每个生成扇区可以在周向上具有可变角度。

在两种情况下，每个扇区始终具有相同的几何形状和网格。当PERIODIC=VARIABLE时，原始扇区两侧的表面必须完全平面。如果原始扇区两侧的表面网格不完全匹配，则在生成周期性三维模型时，将使用自动生成的[*TIE](ch19abk06.md)选项在数据行上指定的相邻对应表面对之间施加约束。

REFLECT

设置REFLECT=LINE以指示必须通过对称线反射部分三维模型来生成三维模型（[图18.57-3](ch18abk57.md#ksymm-model-gen-refl-line)）。

设置REFLECT=PLANE以指示必须通过对称平面反射部分三维模型来生成三维模型（[图18.57-4](ch18abk57.md#ksymm-model-gen-refl-plane)）。

REVOLVE

包含此参数以指示必须将轴对称网格的横截面绕对称轴旋转来生成三维模型。参见[图18.57-2](ch18abk57.md#ksymm-model-gen-revolve)。

### **可选参数：**

ELEMENT OFFSET

将此参数设置为一个整数，以定义元素编号的偏移量。当使用REVOLVE参数时，偏移量被添加到上一个横截面上的每个元素编号，以获得下一个横截面上的元素编号，从参考横截面开始，![](../graphics/key_eqn01094.gif)。参考横截面使用与原始轴对称模型相同的编号。当使用REFLECT参数时，偏移量被添加到原始元素编号，以定义反射部分上的编号。默认值和最小值是原始模型中使用的最大元素编号。

FILE NAME

将此参数设置为一个外部文件的名称（不带扩展名），模型定义的关键词和数据行将被写入该文件。扩展名`.axi`将被添加到用户提供的文件名。请参见["输入语法规则," Section 1.2.1 of the Abaqus Analysis User's Guide](../usb/usb-link.md#usb-int-iinputsyntax)，了解此类文件名的语法。

NODE OFFSET

将此参数设置为一个整数，以定义用于节点编号的偏移量。当使用REVOLVE参数时，偏移量被添加到上一个横截面上的每个节点编号，以获得下一个横截面上的节点编号，从参考横截面开始，![](../graphics/key_eqn01094.gif)。参考横截面使用与原始轴对称模型相同的编号。当使用REFLECT参数时，偏移量被添加到原始节点编号，以定义反射部分上的编号。默认值和最小值是原始模型中使用的最大节点编号。

TOLERANCE

将此参数设置为用于搜索重复节点的距离。将消除旋转模型的旋转轴上、周期性模型中扇区之间的连接平面上以及反射模型两部分之间的连接平面上的重复节点。默认距离是平均元素尺寸的1.0%。
