# 15.6.4 使用接触检测工具的提示





接触检测工具可用于需要创建接触相互作用和绑定约束的任何三维模型。它根据最小规格快速而彻底地识别和创建相互作用和绑定。该工具大大简化了不适用通用接触定义的模型中的接触定义过程。

以下章节概述的一些基本准则可确保最有效和高效地使用该工具：
- 「选择分离容差和扩展角」
- 「审查接触对候选」
- 「保存搜索参数」
- 「可能导致接触检测工具困难的特征」

### 选择分离容差和扩展角

指定的分离容差是接触对搜索算法的主要驱动因素。Abaqus/CAE根据模型中面的相对大小提供默认分离容差。您可能需要根据分析期间模型的预期响应修改此值。为了有效捕获所有重要接触对，指定的分离容差应该与模型中的预期位移或挠度处于同一数量级或更大。

指定非常大的分离容差通常会捕获比分析中必要的更多的接触对。虽然额外的接触对不一定会降低模型质量，但冗余定义难以管理并可能降低性能。

在选择控制表面扩展的角度时，应考虑可能接触区域的拓扑和表面特征。表面应略微扩展到潜在接触区域之外，因此设置扩展角以捕获面边缘上的任何倒角或软角。凹槽、凹槽或凸起有时会破坏表面的定义；这些特征与主面形成的角度应决定扩展角。

对于网格模型，您可以通过仅显示模型上的特征边来预览搜索接触对之前的表面扩展（请参见「定义网格特征边」，第76.5节）。如果扩展角等于特征角，则特定区域中的表面定义会扩展到最近的可见特征边。调整特征角直到可见边缘包围您要捕获的区域，然后相应地设置扩展角。

### 审查接触对候选

在创建相互作用和约束之前，您应始终审查接触对候选。查找表面定义中的任何不连续性。不连续性通常是由小的连接面引起的，这些面在接触对中与逻辑接触表面不是直观上相对的（请参见[图15-10](pt03ch15s06s04.md#itn-detection-excluded-faces)）。

**图15-10** 自动接触检测工具不会识别高亮的垂直面。

![](../graphics/itn-detection-excluded-faces.png)

您可能希望使用修订的扩展和合并选项重新运行搜索，以将不连续性合并到更大的表面中。如有必要，使用「Add」选项手动添加接触对。您也可以使用「Merge」选项合并不连续的表面。

您应该调查任何相交表面以验证它们符合您的建模意图。只有单个过度闭合节点的接触对将被报告为相交，因此轻微的差异可能导致过度闭合。没有适当调整或干扰配合选项的过度闭合接触对可能导致分析中的收敛困难。您还应检查与过度闭合接触对相邻的任何面或表面，以确保它们不是封闭面。更多信息，请参见「接触检测算法」，第15.6.2节中的「检测过度闭合表面」。

### 保存搜索参数

默认情况下，您在「Find Contact Pairs」对话框中指定的搜索参数仅在对话框打开时保持有效；如果您关闭对话框，则下次访问接触检测工具时会提供默认搜索参数。

如果您点击「Advanced」选项卡页面上的![](../graphics/ico_fileSave.png)，Abaqus/CAE会将当前指定的搜索参数设置为默认搜索参数。这些参数在所有未来Abaqus/CAE会话中作为默认值提供。唯一不保存的参数是搜索域，它始终使用「Whole model」的默认值。

当您保存当前搜索参数时，Abaqus/CAE会询问您是否要将当前分离容差保存为默认值。通常Abaqus/CAE会根据当前模型重新计算默认分离容差；如果您选择保存分离容差，则跳过此计算，相同的值始终作为默认分离容差提供。

接触检测工具的默认搜索参数保存在`abaqus_v6.14.gpr`文件中；更多信息，请参见「理解Abaqus/CAE GUI设置」，第3.6节。要将默认搜索参数返回到其原始设置，请点击「Advanced」选项卡页面上的![](../graphics/ico_fileReload.png)。

### 可能导致接触检测工具困难的特征

在使用某些模型特征和设计时，您可能会遇到接触检测工具的困难。这些情况不会导致性能或稳定性问题，但搜索结果通常不会符合您的建模意图。

**堆叠的壳和薄层**

紧密平行堆叠的壳或薄板层可能导致定义额外的接触对。只要这些表面直观上相对且在分离容差范围内，自动接触检测工具可以找到涉及被中间层分隔的表面的接触对。此外，如果启用在同一实例内搜索并禁用重叠表面检查，接触检测工具可能会检测到薄连续板顶侧和底侧之间的潜在接触。Abaqus/CAE为所有这些表面创建接触对候选，即使它们永远不会接触。当层或板是模型的局部特征时，此问题最常见，因为需要更大的分离容差来捕获模型其他区域的表面。要克服此问题，请将搜索域限制在模型的特定区域，并使用适合该区域的分离容差。您也可以使用接触检测对话框的「Entities」选项卡页面从搜索域中消除某些几何或单元类型（例如壳）。否则，您应该在创建相互作用之前删除额外的接触对候选。

**凹面**

虽然接触搜索算法有效地考虑了大多数适当的表面，但它可能会误解凹面和平面之间的关系。凹面造成困难是因为它们的表面法线方向可能在单个表面的跨度上有很大差异，而且表面之间的最近点有时是一个不好的参考。例如，考虑[图15-11](pt03ch15s06s04.md#itn-detection-rounds-bad)中的情况。

**图15-11** 阴影表面的法线在最近点处不是直观上相对的。

![](../graphics/itn-detection-rounds-bad.png)

即使这些模型中最近点在分离容差范围内，这些点处的表面法线也不通过方向测试。接触检测工具不会将这些表面报告为接触对候选，调整分离容差对此行为没有影响。您有时可以修改扩展角以将凹面包含在另一个表面定义中。否则，您必须使用「Add」选项手动定义接触对。

**涉及大旋转的机构**

在建模经历大旋转的机构时，接触检测工具通常无法有效地捕获您的建模意图。在这些机构中 intended contact surfaces最初可能彼此相距很远，而附近表面实际上永远不会接触。[图15-12](pt03ch15s06s04.md#usi-itn-auto-detection-geneva)中描述的日内瓦机构是一个典型示例。

**图15-12** 日内瓦机构的运动。

![](../graphics/usi-itn-auto-detection-geneva.png)

此模型中的重要接触表面是右侧物体上的销和左侧物体上的槽。在初始配置中，销相对远离任何槽。另一方面，相邻表面对接触条件无关紧要。对于此类模型，最好使用相互作用编辑器手动定义接触（请参见「定义表面对表面接触」，第15.13.7节）。

### 接触检测工具的局限性

接触检测无法创建涉及以下特征的接触对：
- 二维模型
- 轴对称模型
- 梁和桁架
- 面-边接触
- 边-边接触
- 孤儿网格单元与解析刚性表面之间的接触
- 同时包含孤儿网格和未网格划分的几何的混合模型

最小允许分离容差为1×10⁻⁵。最大允许分离容差为1×10⁵。Abaqus/CAE无法准确计算此范围之外的分离。如果您的模型需要使用不满足这些要求的分离容差，您应该缩放整个模型的尺寸，使其落在功能范围内。
