# 18.11 创建局部停止条件





局部停止条件是一个可选设置，向Optimization模块指示您的优化何时收敛到解。 例如，您可以指定当优化函数在迭代之间的变化低于指定值时优化完成。 您可以选择将比较的变量，如位移或等效应力。 您可以选择比较运算符——大于、等于、小于等。 优化函数的变化可以从上一次迭代（默认）或第一次迭代计算。

此外，当您在Job模块中创建优化过程时，您可以输入全局停止条件——优化过程结束前应完成的最大优化周期数。 有关更多信息，请参阅["创建和编辑优化过程，" 第19.12.1节](pt03ch19s12hlb01.md)。 在大多数优化问题中，您将仅使用全局停止条件。 局部停止条件仅由形状优化支持，很少需要。

**创建或编辑停止条件：**

1. 从主菜单栏，选择 ****Stop Condition**![](../graphics/images/arrow.gif)**Create****。 出现**Create Stop Condition**对话框。 **提示：**您可以通过另外两种方式启动**Create**过程： - 单击**Stop Condition Manager**中的**Create**。 （您可以通过从主菜单栏选择 ****Stop Condition**![](../graphics/images/arrow.gif)**Manager**** 来显示**Stop Condition Manager**。） - 单击Optimization模块工具箱中的 ![](../graphics/icoR_optStopCond.png) 工具。
2. 从出现的**Create Stop Condition**对话框中，输入停止条件的名称，然后单击**Continue**。
3. 从视口中，选择将应用停止条件的区域，或单击**Done**将停止条件应用到整个模型。 默认情况下，Abaqus/CAE允许您选择模型的所有部分。 要选择面或单元，请使用**Selection**工具栏将可选择的对象类型更改为**Face**或**Cells**。 有关更多信息，请参阅["根据对象类型过滤您的选择，" 第6.3.2节](pt01ch06s03s01.md)。 如果您宁愿从现有集列表中选择，请执行以下操作： 1. 单击提示区域右侧的**Sets**。 Abaqus/CAE显示包含可用集列表的**Region Selection**对话框。 2. 选择感兴趣的集，然后单击**Continue**。 **注意：**默认选择方法基于您最近使用的选择方法。 要恢复为另一种方法，请单击提示区域右侧的按钮——**Select in Viewport**或**Sets**。
4. 完成选择停止条件区域后，在提示区域单击**Done**。 有关选择对象的更多信息，请参阅[第6章，"在视口中选择对象"](pt01ch06.md)。 出现**Edit Stop Condition**对话框。
5. 选择用于计算目标函数的以下运算符之一： - **最大值** - **最小值** - **值之和** - **值的数量**
6. 选择停止条件将使用的变量。 对于形状优化，您可以选择位移或等效应力。 - 每个节点相对于原始形状的位移。 您可以选择总位移，仅由于添加材料而导致的位移，或仅由于减少材料而导致的位移。 此外，您可以选择位移的相对值或绝对值。 - 形状优化中的等效应力。 您可以选择整个模型的等效应力，或从以下内容计算等效应力： - 仅任务所选区域中的节点。 - 任务所选区域中仅受约束的节点。 受约束节点受约束或Abaqus边界条件约束。 - 任务所选区域中仅自由或无约束的节点。 自由或无约束节点不受约束。 - 仅表面节点。
7. 选择比较运算符。
8. 选择用于定义参考值的迭代。 Optimization模块将优化函数的当前值与参考值进行比较。
9. 如果需要，输入用于修改参考值的值和运算。 实际上，您正在归一化定义停止条件的变量。 有关参考值的讨论，请参阅["目标和约束，" Abaqus Analysis User's Guide第13.2.2节](../usb/usb-link.md#usb-anl-aoptobjectives)。
10. 单击**OK**保存数据并退出编辑器。
