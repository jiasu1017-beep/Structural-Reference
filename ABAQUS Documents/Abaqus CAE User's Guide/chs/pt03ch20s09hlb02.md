# 20.9.2 打开或关闭捕捉







捕捉到网格可帮助您在要选择 Sketcher 网格点或将对象拖动到网格上的特定点时定位光标。当启用捕捉并将光标移到网格点附近时，辅助光标会跳转或捕捉到网格点。在**Sketcher 选项**对话框中切换**捕捉到网格**以启用或禁用捕捉。（默认情况下，启动 Sketcher 时启用捕捉。）您为捕捉选择的行为适用于 Abaqus/CAE 会话中的所有草图。如果预选点和网格点都离光标位置很近，Sketcher 预选会覆盖捕捉。

**打开或关闭捕捉：**

1. 从 Sketcher 工具箱底部，选择 Sketcher 自定义工具![](../graphics/ico_sketchOptions-nls.png)。出现**Sketcher 选项**对话框。
2. 从**常规**选项卡页面，切换**捕捉到网格**。当**捕捉到网格**打开时，当您在草图中移动时，辅助光标会捕捉到附近的网格点。根据网格大小、视口放大倍数以及预选是否处于活动状态，您可能还可以选择网格点以及现有草图几何之外的其他点。当**捕捉到网格**关闭时，当您在草图中移动时，辅助光标会与主光标对齐。您可以在工作表上的任何位置进行选择。
3. 选择完所需的自定义选项后，点击**确定**应用您的更改并关闭**Sketcher 选项**对话框。
