# 4.2.1 Managing viewports and viewport annotations from the main menu bar







Use the **Viewport** menu, located on the main menu bar, to create, delete, modify, or rearrange viewports and to create or edit viewport annotations—both those that you create and those generated by Abaqus/CAE. If you prefer, you can select ****View**![](../graphics/images/arrow.gif)**Toolbars**![](../graphics/images/arrow.gif)**Viewport**** from the main menu bar to display a toolbar containing most of the functionality of the items in the **Viewport** menu.

The **Viewport** menu and toolbar allow you to do the following:
- [Create](pt01ch04s03hlb01.md) a viewport.
- [Edit](pt05ch56.md) options for generated viewport annotations (triad, legend, title block, and state block).
- Create an [arrow](pt01ch04s04hlb01.md) annotation.
- Create a [text](pt01ch04s04hlb02.md) annotation.
- Create a [combined arrow and text](pt01ch04s04hlb03.md) annotation.
- Edit [arrow](pt01ch04s04hlb05.md) and [text](pt01ch04s04hlb06.md) annotations.
- Open the **Annotation Manager** to manipulate arrow and text annotations. **Note:** The **Annotation Manager** provides several unique management functions; for more information, see ["The Annotation Manager," Section 4.2.3](pt01ch04s02hlb03.md).
- Open the **Viewport Annotation Options** to show or hide all annotations and to manipulate the viewport annotations generated by Abaqus/CAE.

In addition, the **Viewport** menu lists all the viewports in the session and allows you to delete the current viewport.![](../graphics/images/black4rule.gif)For information on related topics, click the following item: - [Chapter 4, "Managing viewports on the canvas](pt01ch04.md)"
