# 38.6 修改子模型





您可以对子模型进行以下修改：
- 您可以使用Step模块更改分析过程。子模型可以使用通用过程或线性扰动过程。更多信息，请参见["子建模：概述"，Abaqus Analysis User's Guide第10.2.1节](../usb/usb-link.md#usb-anl-asubmodeloverview)。
- 在Load模块中，您必须移除应用于已移除的全局区域上的任何载荷、边界条件或初始条件。
- 如果边界条件应用在您应用子模型边界条件的区域之外的区域，必须确保其与全局模型的加载对应。
- 类似地，如果载荷应用到子模型，必须确保其与全局模型的加载对应。
- 在大多数情况下，您将在Mesh模块中为子模型应用更细化的网格。您可以更改分配给子模型的单元类型；但是，您不能更改维度。全局模型和子模型都必须为二维或三维。
