### 概述

ORNL constitutive model:
- is available in Abaqus/Explicit only;
- is a user-defined material model that can be customized via user subroutine;
- can be combined with linear elastic models (["Linear elastic behavior," Section 22.2.1](pt05ch22s02abm02.md));
- is designed for simulation of metal forming processes; and
- supports rate-dependent plasticity.

### ORNL 本构模型的本构行为

ORNL constitutive model 是为金属成形过程仿真设计的。它结合了塑性理论和粘性理论来描述材料在高速变形过程中的行为。

该模型的主要特征包括：
- 考虑应变率效应
- 考虑温度软化效应
- 支持各向同性硬化

### 与其他材料模型的结合

ORNL 本构模型可与线性弹性模型结合使用。

### 元素

ORNL 本构模型可与 Abaqus/Explicit 中的连续体元素一起使用。
