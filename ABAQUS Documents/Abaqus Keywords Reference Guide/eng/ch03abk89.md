# *CYCLED PLASTIC







### *CYCLED PLASTICSpecify cycled yield stress data for the *ORNL model.

This option is used to specify the tenth-cycle yield stress and hardening values for the ORNL constitutive model. It is relevant only if the [*ORNL](ch15abk02.md) option is used.

**Products: **Abaqus/Standard  Abaqus/CAE  

**Type: **Model data  

**Level: **Model  

**Abaqus/CAE: **Property module

##### **References:**

- [*ORNL](ch15abk02.md)
- ["ORNL -- Oak Ridge National Laboratory constitutive model," Section 23.2.12 of the Abaqus Analysis User's Guide](../usb/usb-link.md#usb-mat-cornl)

**There are no parameters associated with this option.**

### **Data lines to specify cycled yield stress data: **

**First line:**

Repeat this data line as often as necessary to define the dependence of yield stress on plastic strain and, if needed, on temperature.
