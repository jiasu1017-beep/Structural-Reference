# 1.2 使用Abaqus GUI Toolkit的前提条件





要编写使用Abaqus GUI Toolkit的应用程序，您需要在以下领域有一些经验：

**Python编程**

在编写Abaqus/CAE kernel脚本之前，您应该对Python语言有一些经验。在编写GUI应用程序时，您应该有类似的经验。

**Abaqus kernel命令**

GUI的最终目标是向kernel发送命令以供执行；因此，您应该了解kernel命令的工作方式。

**面向对象编程**

Python是一种面向对象的语言，编写应用程序通常包括派生您自己的新类、为它们编写方法以及操作它们的数据。您应该理解面向对象编程的概念。

**GUI设计**

根据应用程序的复杂性，界面设计和可用性测试方面的培训可能会有所帮助。这将帮助您创建一个既直观又易于使用的应用程序。

Abaqus提供涵盖Python、kernel脚本和GUI设计的培训课程。更多信息请联系您当地的销售办公室。GUI设计培训也可从许多独立的培训组织获得。
