### 概述

您对在特定分析的自适应重新网格划分规则中使用哪些误差指标变量的选择应考虑：
- 误差指标变量的特征；
- 存在哪些场以及哪些是感兴趣的；和
- 载荷的性质。

### 误差指标特征

误差指标输出变量提供解准确度的估计（见 ["误差指标输出，" 第 4.1.4 节"](pt02ch04s01aus41.md)）。在自适应重新网格划分的背景下，误差指标有助于确定应在何处细化或粗化网格以达到指定的精度目标（见 ["自适应重新网格划分：概述，" 第 12.3.1 节"](pt04ch12s03abo15.md) 和 ["基于解的网格尺寸，" 第 12.3.3 节"](pt04ch12s03aus85.md)）。本节讨论误差指标在各种分析类型中影响自适应重新网格划分的适用性方面的附加特征。

### 存在哪些场以及哪些是感兴趣的

某些变量自然适用于某些类型的分析。例如，热通量指标（HFLERI）用于具有温度自由度的分析。当在 Abaqus/CAE 的**重新网格划分规则**编辑器中选择误差指标变量时（见 ["什么是重新网格划分规则？，" Abaqus/CAE 用户指南第 17.13.1 节"](../usi/usi-link.md#usi-mgn-conc-adaptivity-rulerole)），您的选择将限于所选过程类型可用的变量。

### 载荷的性质

某些误差指标变量仅指示当前分析时间的离散误差——步骤中的特定增量。其他误差指标变量提供到当前分析时间为止的解历史记录。例如，如果您的模拟涉及非比例加载或显著非线性响应，则在使用记录解历史过程的误差指标变量时，通常会看到更好的自适应重新网格划分结果。[表 12.3.2-1](pt04ch12s03aus84.md#usb-anl-aadperrorindicators-history) 列出了适用于自适应重新网格划分的误差指标变量，并指示它们是否记录解历史过程。

**表 12.3.2-1** 适用于自适应重新网格划分并记录解历史过程的误差指标变量。
| 解的量 | 误差指标变量 (![](../graphics/usb_eqn00290.gif)) | 记录解历史过程？ |
| --- | --- | --- |
| 单元能量密度 | ENDENERI | 是 |
| Mises 应力 | MISESERI | 否 |
| 等效塑性应变 | PEEQERI | 是 |
| 塑性应变 | PEERI | 否 |
| 蠕变应变 | CEERI | 否 |
| 热通量 | HFLERI | 否 |
| 电通量 | EFLERI | 否 |
| 电势梯度 | EPGERI | 否 |

默认情况下，当您创建重新网格划分规则时，会为分析最后一步的最后增量指定误差指标，自适应重新网格划分基于此最后增量中的误差指标。当您选择记录解历史过程的误差指标时，此默认误差指标指定适用于几乎所有分析。但是，对于其他不记录解历史过程的误差指标变量，您可能会发现定义多个针对相同区域但应用于不同步骤的重新网格划分规则是适当的（例如，对于具有非比例加载的多步情况）。

以下示例提供了典型案例的简单说明，并显示了误差指标输出变量的适当选择。

#### 线性响应示例

[图 12.3.2-1](pt04ch12s03aus84.md#aadaptivity-proportional-example) 说明了最简单载荷情况，其中载荷与步骤时间成比例，模型的响应是线性的。在这种情况下，最后增量处的解将与任何其他增量成比例。因此，基于任何误差指标变量选择在最后增量中的误差指标值进行重新网格划分是适当的。

**图 12.3.2-1** 比例加载、线性响应示例：悬臂的小挠度。

![](../graphics/aadaptivity-proportional-example.png)

#### 单调响应示例

[图 12.3.2-2](pt04ch12s03aus84.md#aadaptivity-nlgeom-example) 说明了一个更一般的情况，其中模型具有非线性响应——在这种情况下由几何非线性引起——并且载荷是单调的，但通常不与步骤时间成比例。

**图 12.3.2-2** 单调响应示例：悬臂的大挠度。

![](../graphics/aadaptivity-nlgeom-example.png)

模型的响应稍微更一般，因为特定增量处的解与最后增量处的解不成比例。但是，最后增量中误差指标输出的值仍然反映了模型对载荷历史响应的极端情况。因此，基于任何误差指标变量选择在最后增量中的误差指标值进行重新网格划分是适当的。

#### 一般响应示例

[图 12.3.2-3](pt04ch12s03aus84.md#aadaptivity-general-example) 说明了一种在分析期间载荷特性发生显著变化的情况。

**图 12.3.2-3** 一般响应示例：承受刚性压头的块体。

![](../graphics/aadaptivity-general-example.png)

在这种情况下，您的误差指标选择将取决于材料模型。单元能量密度误差指标 ENDENERI 将考虑载荷历史的复杂性（并导致生成提供整个分析准确解的自适应网格），无论材料类型如何。如果发生塑性变形，您还可以选择使用等效塑性应变 PEEQERI 或塑性应变 PEERI 误差指标。塑性应变和塑性应变误差指标通常不能捕获历史效果；例如，它们不考虑对称应变反向模型中的峰值应变。然而，此示例不涉及应变反向；因此，PEERI 将是有效的误差指标选择。

#### 一般多步响应示例：模具成形和回弹

[图 12.3.2-4](pt04ch12s03aus84.md#aadaptivity-multistep-example) 说明了进一步推广的一般响应。这里，模拟了成形操作，并对操作的不同阶段使用不同的步骤。

**图 12.3.2-4** 一般多步响应示例。

![](../graphics/aadaptivity-multistep-example-nls.png)

在这种情况下，模型的响应随步骤变化。您通常会希望误差指标充分捕获模型对载荷历史响应的极端情况。但是，您不知道是否有任何特定增量捕获了此极端情况。因此，您应该选择记录解历史过程的误差指标变量。
