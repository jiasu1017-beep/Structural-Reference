# 64.1.1 操作节点





网格编辑工具集提供以下工具，允许您操作网格中的节点：
- 创建节点。您可以在全局坐标系或您指定的基准坐标系中指定新节点的坐标。
- 编辑节点。您可以在全局坐标系或您指定的基准坐标系中指定节点的新坐标。或者，您可以指定相对于当前位置的偏移量。您可以编辑单个节点，也可以同时编辑多个节点。
- 拖动节点。您可以在模型中点击并拖动节点。一次只能拖动一个节点，您可以使用**投影到几何体**选项将外部节点投影到其关联的几何体上。拖动节点时会自动激活单元质量检查，以显示单元错误和警告。有关单元质量检查的更多信息，请参见["验证单元质量，" 第17.19.1节](pt03ch17s19hlb01.md)。
- 投影节点。您可以指定顶点、边或面，Abaqus/CAE会将节点从其当前位置投影到所选实体上。您可以投影单个节点，也可以同时投影多个节点。您还可以将节点投影到其他节点、单元边缘、单元面、基准点、基准轴或基准平面上。
- 删除节点。与已删除节点关联的任何单元也会被删除。此外，您可以选择删除在所选删除节点及其关联单元被删除后将变得不与任何单元关联的任何剩余节点。
- 合并所选节点。如果您只选择两个节点进行合并，Abaqus/CAE会在所选节点的中点创建一个新节点。如果您选择两个以上节点，则可以指定**节点合并容差**，这是将被合并的节点之间的最大距离。Abaqus/CAE会删除距离小于指定距离的节点，并用单个新节点替换它们。新节点的位置是被合并到新节点的节点组的平均位置。当Abaqus/CAE移除重复节点时，您可以选择移除具有相同连接性的重复单元。您还可以在"装配"模块中合并零件实例，并且可以选择在此过程中合并节点；有关更多信息，请参见["合并和切割零件实例，" 第13.7.1节](pt03ch13s07s01.md)。
- 平滑所选节点。您可以选择外部节点，Abaqus/CAE会根据周围节点平滑节点位置。您可以平滑零件或装配上的原生网格的任何外部节点，只要它们不是区域边界的一部分；您还可以平滑位于平面网格面内的外部孤兒网格节点。
- 调整二次单元中副中节点的位置，以适应断裂力学分析中裂纹尖端的奇异性。您可以选择节点并输入介于0和1之间的偏置参数。Abaqus/CAE会沿着连接的单元边缘将副中节点移动到基于您输入的参数的位置。例如，如果您输入0.25的参数，Abaqus/CAE会将副中节点的位置偏置到远离所选节点的单元边缘长度的四分之一处。有关更多信息，请参见["控制小应变分析中裂纹尖端的奇异性，" 第31.2.5节](pt04ch31s02s05.md)，以及Abaqus Analysis User's Guide ["轮廓积分评估，" 第11.4.2节中的"使用常规有限元法进行小应变分析的断裂力学网格构建"](../usb/usb-link.md#usb-anl-acontintegral-singularity)。
- 在孤兒网格中对所选节点重新编号。您可以通过指定起始标签和增量来对所选节点重新编号，或者通过指定值偏移现有标签。

**提示：**网格编辑撤销功能可以回滚您对网格节点所做的任何更改。有关更多信息，请参见["撤销或重做网格编辑工具集中的更改，" 第64.9节](pt06ch64s09.md)。

有关这些节点操作技术的详细说明，请参见["编辑节点，" 第64.5节](pt06ch64s05.md)。
