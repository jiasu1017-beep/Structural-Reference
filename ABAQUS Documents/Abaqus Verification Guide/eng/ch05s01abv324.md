# 5.1.8 Define a part instance that will be used for display only







**Products: **Abaqus/Standard  Abaqus/Explicit  

### Features tested

Definition of an instance used by Abaqus for display only without affecting the results of the rest of the model.

### Problem description

The tests contain two instances, one of which is defined to be used for display only. This test verifies that the instance is not included in the analysis. In addition, it verifies the cases where the definition of the instance references zero, one, or three nodes from the other instance.

### Results and discussion

These tests verify that the defined instance does not take part in the analysis.
