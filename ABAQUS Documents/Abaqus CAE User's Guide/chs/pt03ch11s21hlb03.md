# 11.21.3 添加扫掠实体特征





从主菜单栏选择**形状**![](../graphics/images/arrow.gif)**实体**![](../graphics/images/arrow.gif)**扫掠**，向当前视口中的零件添加扫掠实体特征。您只能向三维零件添加扫掠实体特征。

您通过首先定义扫掠路径，然后定义扫掠轮廓来添加扫掠实体特征。有不同的方法可用于定义每个组件：
- 您可以通过在选定面上绘制路径或通过选择一系列您希望扫掠路径遵循的边缘或线框来定义扫掠路径。
- 您可以通过使用草图编辑器绘制扫掠轮廓或通过在模型中选择一个面作为轮廓来定义扫掠轮廓。

**添加扫掠实体特征：**

1. 从主菜单栏，选择**形状**![](../graphics/images/arrow.gif)**实体**![](../graphics/images/arrow.gif)**扫掠**。
2. 如果要绘制扫掠路径，请执行以下操作。
3. 如果要将扫掠路径指定为一系列边缘或线框，请执行以下操作。
4. 如果要绘制扫掠轮廓，请执行以下操作。
5. 如果要选择一个面作为扫掠轮廓，请执行以下操作。
6. 如果需要，执行以下任何操作：切换**包含扭曲**、**包含拔模**、**保持轮廓法线恒定**或**保持内部边界**。
7. 单击**确定**创建新的扫掠实体。
