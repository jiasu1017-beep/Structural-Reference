# 32.1 垫片建模概述





垫片是位于结构组件之间的薄密封组件。（有关垫片理论的详细信息，请参见["垫片单元"，Abaqus Analysis User's Guide第32.6节](../usb/usb-link.md#usbgasket)。）在三维空间中建模垫片的一般过程涉及以下步骤：

1. 在Part模块中，定义实体几何体。垫片部件通常是非常薄的扁平实体。
2. 在Property模块中，定义垫片材料。该材料可以是常规材料或包含特殊垫片行为的材料。有关更多信息，请参见["为垫片定义材料"，第32.2节](pt04ch32s02.md)。
3. 在Property模块中，定义引用垫片材料的垫片截面。然后将垫片截面分配给垫片区域。
4. 在Interaction模块中，在垫片表面和相邻区域的表面之间建立适当的绑定约束或接触相互作用。
5. 在Mesh模块中，将自上而下扫掠网格技术或自下而上网格划分技术分配给垫片区域。无论您选择哪种网格划分技术，都必须沿垂直于垫片平面的方向扫掠、拉伸或旋转网格，以生成具有正确方向的垫片单元。更多信息，请参见["指定扫掠路径"，第17.18.6节](pt03ch17s18s06.md)或["自下而上网格划分"，第17.11节](pt03ch17s11.md)。
6. 在Mesh模块中，为垫片区域分配垫片单元类型并划分网格。

当使用实体对垫片进行建模时，您可以使用以下一种或多种技术来定义垫片与周围区域之间的相互作用：
- 您可以创建单独的垫片部件，然后使用绑定约束或接触相互作用将垫片部件实例耦合到其他部件实例。
- 您可以在部件内创建一个薄区域，然后为该区域分配垫片截面和单元类型。如果垫片网格与相邻区域的网格兼容性很重要，建议使用此方法。

如果您使用由多个层和嵌件组成的垫片，则必须执行以下附加任务：

1. 使用Partition工具集分割实体垫片区域，以便在嵌件位置创建内部表面。
2. 在Property模块中，在表示嵌件的内部表面上定义表皮增强。（有关更多信息，请参见["定义表皮增强"，第36.1节](pt04ch36s01.md)。）分配给实体和嵌件的垫片截面通常不同，其材料也不同。
3. 嵌件表皮不需要（也不允许）网格划分，但您必须在Mesh模块中为表皮分配三维"线"垫片单元类型。更多信息，请参见["为表皮或桁条增强分配单元类型"，第36.5节](pt04ch36s05.md)。
