# 30.6 运行流体分析





当您提交与流体模型相关联的作业进行分析时，Abaqus/CAE首先生成表示您模型的输入文件，然后Abaqus/CFD使用该文件的内容执行分析。随着分析的进行，Abaqus/CFD创建数据（`.dat`）文件和状态（`.sta`）文件。Abaqus/CFD使用基于域的并行性，通过显式消息传递在共享内存和分布式内存计算机上实现。更多信息，请参阅《Abaqus分析用户指南》第3.2.2节["Abaqus/Standard、Abaqus/Explicit和Abaqus/CFD执行"](../usb/usb-link.md#usb-int-danalysisproc)和《Abaqus分析用户指南》第3.5.1节["并行执行：概述"](../usb/usb-link.md#usb-int-aparallelmodes)。
