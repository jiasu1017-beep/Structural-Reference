# 4.4.5 在当前视口中操作标注





要操作视口箭头和文本标注，你必须先选择它们。使用主菜单栏中的****Viewport**![](../graphics/images/arrow.gif)**Edit Annotations****从当前视口中选择箭头或文本标注。

**注意：**要操作 Abaqus/CAE 生成的视口标注，请使用**视口标注选项**对话框（有关详细信息，请参阅[第 56 章，"自定义视口标注"](pt05ch56.md)"）。

你可以移动、复制、编辑、隐藏或删除所选标注。你也可以重新排列 Abaqus/CAE 显示标注的顺序；该顺序决定了当视口或模型操作导致多个标注出现在同一位置时，哪个标注会"位于前面"。

**要操作箭头或文本标注：**

1. 从主菜单栏中，选择****Viewport**![](../graphics/images/arrow.gif)**Edit Annotations****。**提示：**你也可以通过单击[**视口**工具栏](pt01ch04s02hlb02.md)中的![](../graphics/ico_select.png)来编辑标注。
2. 单击箭头或文本标注以选择它。要选择其他标注，请按住 **[Shift]**+单击而不是直接单击。有关详细信息，请参阅["在当前视口中选择对象，" 第 6.2 节](pt01ch06s02.md)。Abaqus/CAE 会高亮显示所选标注及其锚点。圆形手柄是锚点连接点；它们出现在所选箭头的每个端点附近，以及所选文本标注的参考点处。虚线表示锚点与手柄之间的偏移量。
3. 执行以下操作之一：
   **要移动或复制标注：**
   如果你选择了文本或多个标注，将选区拖动到视口中移动。按住 **[Ctrl]** 然后释放鼠标按钮 1 可以复制标注而不是移动。如果你选择了单个箭头标注，拖动箭头杆来移动箭头。你可以将标注拖动到视口中的任何位置，甚至可以拖动到可视区域之外。在视口中移动或复制标注不会改变其锚点。
   **要移动或复制箭头标注的端点：**
   如果你选择了单个箭头标注，请注意圆形手柄比选择多个标注时要大；这表明你可以独立移动箭头的每个端点。拖动其中一个圆形手柄来加长、缩短或重新调整箭头方向。按住 **[Ctrl]** 然后释放鼠标按钮 1 可以创建箭头的径向副本而不是移动。你可以拖动端点到视口中的任何位置，甚至可以拖动到可视区域之外。通过拖动端点移动或复制箭头不会改变箭头的锚点。
   **要编辑标注：**
   双击箭头或文本标注，分别打开**箭头编辑器**或**文本编辑器**对话框。如果你愿意，也可以单击鼠标右键 3 并从弹出菜单中选择**编辑**。
   - 对于箭头标注，你可以更改线型、线宽或颜色；并且可以更改箭头每个端点的偏移量、箭头样式和锚点，以及在手柄位置和箭头端点之间添加间隙。有关详细信息，请参阅["编辑箭头标注属性，" 第 4.4.6 节](pt01ch04s04hlb05.md)。
   - 对于文本标注，你可以编辑文本；显示或隐藏文本周围的框；更改背景颜色、字体、字体颜色、对齐方式和旋转；或更改参考点、锚点和偏移量。有关详细信息，请参阅["编辑文本标注属性，" 第 4.4.7 节](pt01ch04s04hlb06.md)。
   **要隐藏所选标注：**
   单击鼠标右键 3，并从出现的菜单中选择**隐藏**。Abaqus/CAE 会从当前视口中移除所选标注。要再次显示标注，你可以从[**标注管理器**](pt01ch04s02hlb03.md)中绘制它们。你也可以在模型树中选择标注，单击鼠标右键 3，并从出现的菜单中选择所需选项来隐藏或绘制标注。
   **要删除所选标注：**
   按 **Backspace** 或 **Delete**。Abaqus/CAE 会显示警告对话框；单击**是**删除标注。Abaqus/CAE 会从当前视口和关联数据库中删除所选标注。
   **要重新排列标注的显示顺序：**
   默认情况下，Abaqus/CAE 按照在视口中创建或绘制的顺序显示视口标注；最后创建或绘制的标注将出现在任何现有标注的前面。显示顺序的更改不会影响当前视图，除非两个或多个标注出现在同一位置。所有视口标注都显示在模型前面。单击鼠标右键 3，并从弹出菜单中选择**置于顶层**、**置于底层**、**上移一层**或**下移一层**来更改显示顺序。或者，要将单个标注移到前面，可以从[**标注管理器**](pt01ch04s02hlb03.md)中重新绘制它。
