# 7.4 动力学网格设计





在为动力学模拟设计网格时，您需要考虑响应中被激发的振型，并使用能够充分表示这些振型的网格。这意味着对于静态模拟适当的网格可能不适合于计算激发高频模态的载荷的动力响应。

例如，考虑如图 7-3 所示的板。一阶壳单元的网格对于均匀载荷下的静态分析是适当的，对于预测第一振型也是合适的。但是，网格显然太粗，无法准确模拟第六振型。

**图 7-3** 基于粗网格的板的振动频率和相应振型。

![](../graphics/gss-coarse-mesh-nls.png)

[图 7-4](ch07s04.md#gss-fine-mesh) 显示了使用细化一阶单元网格建模的同一板。第六振型的变形形状现在看起来好多了，该振型预测的频率也更准确。如果板上动力载荷使得该振型被显著激发，则必须使用细化网格；粗网格的结果将不准确。

**图 7-4** 基于细网格的板的振动频率和相应振型。
