# 12.13.5 创建电磁实体截面





电磁实体截面用于定义二维和三维实体区域的截面属性，必须仅分配给电磁模型中的电磁元素。请参见[Abaqus Analysis User's Guide的第28.1.1节，"实体（连续体）元素"](../usb/usb-link.md#usb-elm-esolidcont)。

**指定电磁实体截面属性：**

1. 从主菜单栏，选择****Section**![](../graphics/images/arrow.gif)**Create****。出现**Create Section**对话框。
   **提示：**您也可以在**Section Manager**中点击**Create**，或在Property模块工具箱中选择创建截面工具![](../graphics/ico_section.png)。
2. 输入截面名称。有关命名对象的更多信息，请参见["使用基本对话框组件，"第3.2.1节"](pt01ch03s02s01.md)。
3. 选择**Solid**作为截面**Category**，选择**Electromagnetic, Solid**作为截面**Type**，然后点击**Continue**。出现电磁实体截面编辑器。
4. 为电磁实体截面选择一种材料。如有需要，点击![](../graphics/ico_material.png)创建材料；有关更多信息，请参见["创建或编辑材料，"第12.7.1节"](pt03ch12s07hlb01.md)。
5. 为截面**Plane stress/strain thickness**输入值。如果截面将用于二维区域，则必须指定截面厚度。如果区域类型不需要厚度信息，Abaqus/CAE会忽略厚度信息。
6. 点击**OK**保存更改并关闭电磁实体截面编辑器。
