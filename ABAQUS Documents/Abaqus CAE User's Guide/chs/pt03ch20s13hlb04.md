# 20.16.4 围绕镜像线移动或复制 Sketcher 对象


使用 Sketcher 工具箱中的镜像工具,围绕指定线镜像 Sketcher 对象——线、圆弧、圆、椭圆、圆角或样条曲线。您可以选择移动原始对象或在 新位置创建镜像副本。如果您选择移动对象,则更改受草图中任何约束和草图选项中的约束求解方法的控制(更多信息,请参见["自定义 Sketcher 中约束的使用," 第 20.9.10 节](pt03ch20s09hlb10.md))。如果您选择创建副本,Abaqus/CAE 会在原始对象和副本之间应用镜像约束,以便在后续任何草图编辑中保持镜像关系。

**镜像绘制对象的步骤:**

1. 从 Sketcher 工具箱中的移动和复制工具中,选择镜像工具 ![](../graphics/ico_sketchMirror.png)。有关 Sketcher 工具箱中工具的图示,请参见["Sketcher 工具," 第 20.4.1 节](pt03ch20s04s01.md)。
2. 在提示区域选择**复制**或**移动**。Abaqus/CAE 会在提示区域显示提示,引导您完成操作。
3. 从草图中选择一条现有的直线作为镜像线。您可以从草图、构造或参考几何中选择任何直边。
4. 选择要镜像的所有对象。您可以选择草图和构造几何进行移动,或者可以选择草图、构造和参考几何进行复制;尺寸不能被镜像。**提示:**要选择多个对象,请在单击每个对象时按住 **[Shift]** 键,或围绕对象拖动一个矩形。要取消选择对象,请使用 **[Ctrl]****+单击**。更多信息,请参见[第 6 章,"在视口中选择对象"](pt01ch06.md)。
5. 单击鼠标按钮 2 表示已完成选择对象。Abaqus/CAE 按指定移动或复制对象。
6. 如有需要,重复第 4 步和第 5 步围绕同一镜像线移动或复制更多对象。
7. 完成移动和复制对象后,执行以下操作之一: - 在 Abaqus/CAE 窗口任意位置单击鼠标按钮 2。 - 选择 Sketcher 工具箱中的任何工具。 - 单击提示区域中的取消按钮 ![](../graphics/afxI_cancel.png)。
