# 44.5.5 自定义刻度线云图





当您选择显示模型中线形单元的刻度线时，Abaqus/CAE将云图级别表示为绘制在单元法线上的线条（轴）上的刻度线，并在轴之间显示云图曲线。您可以指定刻度线轴的长度、云图曲线的颜色、将与单元相交的云图值（刻度线）以及轴在梁单元上绘制的法线方向。此外，对于线型云图，您应用于单个云图线的样式选择将显示在刻度线轴上。有关更多信息，请参见["美化线型云图，" 第44.5.2节](pt05ch44s02hlb02.md)。

**要自定义刻度线云图：**

1. 找到**Tick Marks**选项。从主菜单栏中选择****Options**![](../graphics/images/arrow.gif)**Contour****，或单击工具箱中的![](../graphics/ico_plotContourOptions.png)。在出现的对话框中单击**Other**选项卡；然后单击**Tick Marks**选项卡。**Tick Marks**选项出现。
2. 要选择刻度线轴在梁单元上绘制的法线方向，请在**Orientation**字段中切换**N1**或**N2**按钮。**注意：**当在梁以外的线单元上显示刻度线云图时，此选项将被忽略。在这种情况下，刻度线轴沿单元法线方向绘制。
3. 选择刻度线轴的长度：1. 单击**Axis length**按钮显示长度选择：**Short**、**Medium**或**Long**。2. 单击您想要的长度。指定长度出现在**Axis length**按钮上。
4. 选择将与单元相交的云图值：1. 在**Base Value**字段中输入所需的基准云图值。将检查指定值以确定它是否在云图范围内。如果它高于最大云图值，则使用最大云图值作为基准值；如果它低于最小云图值，则使用最小云图值作为基准值。默认情况下，使用零作为基准值。
5. 选择刻度线曲线的颜色：1. 单击颜色样本![](../graphics/usi-colorsample.png)。Abaqus/CAE显示**Select Color**对话框。2. 使用**Select Color**对话框中的方法选择新颜色。有关更多信息，请参见["自定义颜色，" 第3.2.9节](pt01ch03s02s09.md)。3. 单击**OK**关闭**Select Color**对话框。颜色样本更改为所选颜色。在您单击**Contour Plot Options**对话框中的**OK**或**Apply**之前，新颜色不会出现在视口中。
6. 单击**Apply**实施您的更改。当前视口中刻度线云图的云图线更改为反映您的规格。默认情况下，您的更改在会话期间保存，并将影响所有后续云图。如果您想在后续会话中保留更改，请将它们保存到文件。有关更多信息，请参见["保存自定义设置以供后续会话使用，" 第55.1.1节](pt05ch55s01s01.md)。
