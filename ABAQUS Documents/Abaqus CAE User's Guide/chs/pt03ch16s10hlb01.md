# 16.10.1 定义对称/反对称/固定边界条件





您可以通过在对称/反对称/固定边界条件编辑器中选择列出的常见类型之一来定义边界条件。

**创建或编辑对称/反对称/固定边界条件：**

1. 使用以下方法之一显示对称/反对称/固定边界条件编辑器：- 若要创建新的对称/反对称/固定边界条件，请遵循"创建边界条件"，第16.8.2节中概述的过程（**类别**：`Mechanical`；**所选步骤的类型**：`Symmetry/Antisymmetry/Encastre`）。- 若要使用菜单或管理器编辑现有的对称/反对称/固定边界条件，请参见"编辑依赖于步骤的对象"，第3.4.13节。您只能在创建边界条件的步骤中编辑对称/反对称/固定边界条件。
2. 如果您在屈曲步骤中创建边界条件，请选择**用于BC的选项**，以指定您希望边界条件用于其中计算的选项。更多信息，请参见"Abaqus Analysis User's Guide"第6.2.3节"Eigenvalue buckling prediction"中的"边界条件"。
3. 默认情况下，使用全局坐标系来定义边界条件。要更改应用边界条件的坐标系，请单击**CSYS**选项的![](../graphics/ico_selectBlue.png)，并执行以下操作之一：- 在视口中选择现有的基准坐标系。- 按名称选择现有的基准坐标系。1. 从提示区域单击**基准坐标系列表**以显示基准坐标系列表。2. 从列表中选择一个名称，然后单击**确定**。- 从提示区域单击**使用全局CSYS**以恢复到全局坐标系。
4. 选择以下选项之一：| **XSYMM** | 关于X = 常数平面的对称（U1 = UR2 = UR3 = 0）。 | | --- | --- | | **YSYMM** | 关于Y = 常数平面的对称（U2 = UR1 = UR3 = 0）。 | | **ZSYMM** | 关于Z = 常数平面的对称（U3 = UR1 = UR2 = 0）。 | | **XASYMM** | 关于X = 常数平面的反对称（U2 = U3 = UR1 = 0；仅限Abaqus/Standard）。 | | **YASYMM** | 关于Y = 常数平面的反对称（U1 = U3 = UR2 = 0；仅限Abaqus/Standard）。 | | **ZASYMM** | 关于Z = 常数平面的反对称（U1 = U2 = UR3 = 0；仅限Abaqus/Standard）。 | | **PINNED** | 铰接（U1 = U2 = U3 = 0）。 | | **ENCASTRE** | 完全固定（U1 = U2 = U3 = UR1 = UR2 = UR3 = 0）。 |
5. 单击**确定**保存您的数据并退出编辑器。
