# 50.1 可视化模块中查询工具集概述





查询工具集允许您获取有关模型的一般信息。可视化模块在消息区域显示请求的信息，相同的信息被写入重放文件。选择****工具**![](../graphics/images/arrow.gif)**查询****从主菜单栏，或从**查询**工具栏中选择![](../graphics/ico_info.png)工具来使用查询工具集。

**常规查询**下的项目提供以下信息：

**节点**

您可以获取所选节点的标签、变形和未变形坐标以及位移的信息。

**距离**

您可以获取两个所选节点之间的变形和未变形距离以及节点之间相对位移的信息。

**角度**

您可以获取三个所选节点之间形成的变形和未变形角度的信息。您选择的第二个节点是角度的顶点。

**单元**

您可以获取所选单元的网格类型、材料、截面、连接性以及积分点位置的当前场输出变量的信息。此查询仅在选择输出数据库时可用。

**网格**

您可以获取当前输出数据库的名称、模型中的节点和单元数量以及单元类型的信息。

**质量属性**

您可以获取输出数据库中整个模型或模型部分的基本质量属性。此查询仅在选择输出数据库时可用。Abaqus/CAE返回以下信息：
- 体积
- 体积形心
- 质量
- 质心
- 关于质心或关于指定点的惯性矩

有关常规模型查询的更多信息，请参阅["在可视化模块中查询模型，"第50.2.1节](pt05ch50s02hlb01.md)。

**可视化模块查询**下的项目提供以下信息：

**探测值**

当您在当前视口中移动光标时，Abaqus/CAE会在"探测值"对话框中显示信息。探测模型图会显示模型数据和分 析结果；探测X–Y曲线图会显示X–Y曲线数据。有关探测的更多信息，请参阅[第51章，"探测模型"](pt05ch51.md)。"

**应力线性化**

应力线性化是通过截面将应力分离为恒定薄膜应力和线性弯曲应力。Abaqus执行应力线性化计算，并以X–Y曲线图形式显示结果。应力线性化仅适用于输出数据库。有关应力线性化的更多信息，请参阅[第52章，"计算线性化应力"](pt05ch52.md)。"

**活动单元和节点**

Abaqus/CAE显示当前视口中所有活动节点或活动单元的标签编号。有关更多信息，请参阅["查询活动节点或单元标签，"第50.2.2节](pt05ch50s02hlb02.md)。

**铺层堆栈图**

Abaqus/CAE创建新视口并显示复合铺层的代表性图像。该图像显示铺层中的铺层以及每个铺层的详细信息，例如其纤维方向、厚度、参考平面和积分点。铺层堆栈图仅适用于输出数据库。有关更多信息，请参阅[第53章，"查看铺层堆栈图"](pt05ch53.md)。"
