# 1.3.24 梁流体惯性







**产品：**Abaqus/Standard  Abaqus/Explicit  

### 测试的功能

本节提供使用所有 Timoshenko 梁的流体惯性的基本验证测试。

### 测试的单元

B21    B22    B31    B32    

### 问题描述

本节介绍两组问题。第一组包括输入文件 [b21_circ_bfi.inp](../eif/b21_circ_bfi.inp) 和 [b21_circ_bai.inp](../eif/b21_circ_bai.inp)，并验证了在直接积分隐式动态过程中使用带流体惯性的梁。第二组由其余输入文件组成，测试在带梁截面或通用梁截面的瞬态动态分析中使用带流体惯性的梁。瞬态动态分析也在 Abaqus/Explicit 中执行，以验证 Abaqus/Explicit 中的带流体惯性的梁。

### 结果与讨论

使用带流体惯性的梁从 [b21_circ_bfi.inp](../eif/b21_circ_bfi.inp) 获得的结果与使用带附加惯性的梁添加等效质量的 [b21_circ_bai.inp](../eif/b21_circ_bai.inp) 的结果非常吻合。比较仅在施加外部激励的方向上有意义。

Abaqus/Explicit 瞬态分析结果与使用 Abaqus/Standard 获得的瞬态分析结果非常一致。
