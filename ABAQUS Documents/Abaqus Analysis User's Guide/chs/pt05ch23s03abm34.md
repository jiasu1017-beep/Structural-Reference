### 概述

可压碎泡沫塑性模型：
- 可与线弹性模型（["Linear elastic behavior," Section 22.2.1](pt05ch22s02abm02.md)）结合使用；
- 基于各向同性强化屈服面；
- 考虑体积压缩响应；
- 适用于泡沫材料和蜂窝结构；以及
- 支持率相关（粘性）响应。

| **输入文件用法：** | ``` [*CRUSHABLE FOAM](../key/key-link.md#usb-kws-mcrushablefoam) [*HARDENING](../key/key-link.md#usb-kws-mhardening) ``` |
| --- | --- |

### 率相关响应

可压碎泡沫塑性模型支持率相关响应。

| **输入文件用法：** | ``` [*CRUSHABLE FOAM](../key/key-link.md#usb-kws-mcrushablefoam), RATE=参数 ``` |
| --- | --- |

### 与其他材料模型的结合

可压碎泡沫塑性模型可与线弹性模型结合使用。

### 元素

可压碎泡沫塑性模型可与 Abaqus/Standard 和 Abaqus/Explicit 中的连续体元素一起使用。
