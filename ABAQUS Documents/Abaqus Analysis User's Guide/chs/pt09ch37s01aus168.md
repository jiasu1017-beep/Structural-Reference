### 概述

本节提供Abaqus/Standard中可用的轴对称滑动线单元的参考。

### 单元类型

| ISL21A | 用于一阶轴对称单元的2节点单元 |
| --- | --- |

| ISL22A | 用于二阶轴对称单元的3节点单元 |
| --- | --- |

**活动自由度**

节点处的1、2

**额外求解变量**

每个节点有两个与接触应力相关的额外变量。

**所需节点坐标**

*r*、*z*

**单元属性定义**

| **输入文件用法：** | 使用以下选项识别滑动线单元所交互的滑动线（主曲面）： |
| --- | --- |
| | ``` [*SLIDE LINE](../key/key-link.md#usb-kws-mslideline) ``` 使用以下选项定义滑动线单元的截面属性： ``` [*INTERFACE](../key/key-link.md#usb-kws-minterface) ``` |

**基于单元的加载**

无。

**单元输出**

**应力分量**

| S11 | 物体节点与相互作用的滑动线之间的压力。 |
| --- | --- |

| S12 | 物体节点与相互作用的滑动线之间的剪切应力。 |
| --- | --- |

**应变分量**

| E11 | 物体节点与滑动线之间的分离量。 |
| --- | --- |

| E12 | 物体节点与滑动线之间累积的相对切向位移。 |
| --- | --- |
