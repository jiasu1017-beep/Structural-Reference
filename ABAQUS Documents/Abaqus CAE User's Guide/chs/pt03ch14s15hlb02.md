# 14.15.2 自定义求解器控制





求解器控制编辑器允许您修改控制迭代线性方程求解器的参数。更多信息，请参见「迭代线性方程求解器」，Abaqus Analysis User's Guide第6.1.6节。

**注意：**求解器控制仅适用于**Static, General**；**Static, Linear perturbation**；**Visco**；**Heat transfer**；**Geostatic**；和**Soils**分析步骤。

**自定义通用求解器控制的步骤：**

1. 从主菜单栏选择 ****Other**![](../graphics/images/arrow.gif)**Solver Controls**![](../graphics/images/arrow.gif)**Manager****。Abaqus/CAE会显示「Solver Controls Manager」，其中显示模型中定义的步骤以及与每个步骤关联的求解器控制。
2. 在「Solver Controls Manager」中，选择您感兴趣的分析步骤，然后点击「Edit」。此时会出现「Edit Solver Controls」对话框。
3. 选择您要如何指定求解器控制：- 选择「Propagate from previous step」如果您希望前一步骤的求解器控制设置在当前步骤中保持有效。- 选择「Reset all parameters to their system-defined defaults」如果您希望对话框中的所有参数恢复为默认值。- 选择「Specify」如果您想为该步骤输入特定参数的新值。
4. 输入指定自定义求解器控制所需的数据：- 在「Relative tolerance」字段中，输入收敛的相对容差值。- 在「Max. number of iterations」字段中，输入线性求解器迭代的最大次数。- 在「ILU factorization fill-in level」字段中，输入不完全LU分解的填充水平（0 ≤ *level* ≤ 3）。此设置仅适用于**Geostatic**和**Soils**分析步骤。有关这些参数的更多信息，请参见「Abaqus Analysis User's Guide」的「迭代线性方程求解器」，第6.1.6节中的「设置迭代线性求解器的控制」。
5. 点击「OK」保存您的自定义设置并关闭编辑器。
