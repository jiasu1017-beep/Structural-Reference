# 3.5.1 模型树概述







模型树提供了模型中项目层次结构的可视化描述。例如，[图 3-19](pt01ch03s05s01.md#uss-int-tree) 显示了在 Getting Started with Abaqus: Interactive Edition 的[附录 B，「在 Abaqus/CAE 中创建和分析简单模型」](../gsa/gsa-link.md#gsa-chp-caebeam)中完成悬臂梁教程后模型树的外观。

**图 3-19** 完成悬臂梁教程后的模型树。

![](../graphics/uss-int-tree-nls.png)

模型树与结果树共享 Abaqus/CAE 界面的左侧，并且仅在 Property 模块中还有材料库。您可以点击**模型**、**结果**或**材料库**选项卡来在模型树、结果树和材料库之间切换显示。有关结果树和材料库的更多信息，请分别参阅[「结果树概述，第 3.5.2 节」](pt01ch03s05s02.md)和[「使用材料库，第 12.5 节」](pt03ch12s05.md)。此外，模型树顶部的提示按钮![](../graphics/uss-int-modeltree-tip.png)提供了模型树和结果树功能的快速摘要以及本节末尾描述的键盘快捷键的摘要。

完整的 Abaqus/CAE 模型包含执行分析所需的所有信息；例如，所有零件、材料、步骤、载荷以及装配的网格表示。模型还包含提交给 Abaqus 分析产品的作业。有关更多信息，请参阅[「Abaqus/CAE 模型包含什么？，第 9.2.1 节」](pt02ch09s02s01.md)。所有这些项目都在模型树中表示。

模型树中的项目由小图标表示；例如，**步骤**图标，![](../graphics/uss-int-tree-steps-nls.png)。此外，项目旁边的括号表示该项目是一个容器，括号中的数字表示容器中的项目数。您可以点击模型树中的「加」和「减」符号来展开和折叠容器。右箭头键和左箭头键执行相同的操作。

例如，**步骤**容器包含模型中的所有步骤。在[图 3-19](pt01ch03s05s01.md#uss-int-tree)所示的示例中，展开**步骤**容器显示模型包含两个步骤——**初始**步骤和**梁载荷**步骤。展开**梁载荷**步骤后，如[图 3-20](pt01ch03s05s01.md#uss-int-tree-steps-expanded)所示，显示该步骤有四个容器，每个容器包含一个项目——**场输出请求**、**历史输出请求**、**载荷**和**边界条件**。

**图 3-20** **梁载荷**步骤中的容器。

![](../graphics/uss-int-tree-steps-expanded-nls.png)

此外，该步骤包含四个空容器——**ALE 自适应网格约束**、**相互作用**、**预定义场**和**载荷工况**。您不能从模型树中删除空容器，尽管您可以隐藏空容器（请参阅[「更改模型的视图，第 3.5.4 节」](pt01ch03s05s04.md)）。最后，展开**载荷**容器后，如[图 3-21](pt01ch03s05s01.md#uss-int-tree-loads-pressure)所示，显示在此步骤中创建的一个名为**压力**的载荷。

**图 3-21** **载荷**容器中的载荷。

![](../graphics/uss-int-tree-loads-pressure-nls.png)

模型树中容器和项目的排列反映了您创建模型的顺序。模块菜单中模块的顺序遵循类似的逻辑——您先创建零件，然后创建装配；您先创建步骤，然后创建载荷。此排列是固定的——您不能在模型树中移动项目。有关更多信息，请参阅[「什么是模块？，第 2.3 节」](pt01ch02s03.md)。

Abaqus/CAE 在模型树中为当前对象加下划线，并在上下文栏中显示它们。您正在处理的模型是当前对象。当前零件或当前步骤也是当前对象。当您在模型树中选择一个项目时，如果所选项目属于当前对象，Abaqus/CAE 会在当前视口中高亮显示该项目。例如，如果选择载荷，如果载荷应用于当前模型的当前步骤，则 Abaqus/CAE 会在当前视口中高亮显示载荷。容器不会被高亮显示。

您可以在模型树中选择多个项目，如果它们属于当前对象，Abaqus/CAE 会高亮显示每个项目。例如，您可以在当前模型的当前步骤中选择一个相互作用和一个载荷，Abaqus/CAE 会在装配中高亮显示这两个项目。当您将光标移动到某个项目上时，模型树会显示有关该项目的一些信息，如[图 3-22](pt01ch03s05s01.md#uss-int-tree-tooltip)所示。在大多数情况下，相同的信息可以从项目的管理器中获得。

**图 3-22** 模型树显示光标下方项目的信息。

![](../graphics/uss-int-tree-tooltip-nls.png)

当光标在模型树中时，按字母键（a-z）会选择树中名称以该字符开头的第一个项目。继续按其他键可以匹配项目名称中的更多字符。[表 3-2](pt01ch03s05s01.md#uss-int-tree-shortcuts) 描述了模型树中可用于导航的所有键盘快捷键；您也可以使用这些快捷键在结果树中导航。

**表 3-2** 模型树和结果树中的键盘快捷键。
| 键盘快捷键 | 操作 |
| --- | --- |
| **[Home]** | 转至模型树或结果树的顶部 |
| **[End]** | 转至模型树或结果树的底部 |
| 上箭头 | 向上移动一个项目 |
| 下箭头 | 向下移动一个项目 |
| 右箭头 | 展开分支或向下移动一个项目 |
| 左箭头 | 折叠分支或向上移动一个项目 |
| **[Del]** | 删除项目 |
| **[F2]** | 对容器应用[过滤器](pt01ch03s05s03.md) |

模型树提供了主菜单栏和模块管理器的大部分功能。例如，如果双击**零件**容器，您可以创建新零件（相当于从主菜单栏选择****零件**![](../graphics/images/arrow.gif)**创建****）。如果双击零件的特征，您可以编辑该特征（相当于从主菜单栏选择****特征**![](../graphics/images/arrow.gif)**编辑****）。

您可以拖动模型树和画布之间的分隔线来更改模型树的宽度。此外，您可以通过从主菜单栏选择****视图**![](../graphics/images/arrow.gif)**显示模型树****来关闭模型树的显示。按 **[Ctrl]****+T** 具有相同的效果。要切换到结果树，请点击**结果**选项卡。

步骤相关对象是可以在步骤之间传播的对象；例如，载荷和相互作用。有关更多信息，请参阅[「什么是步骤相关管理器？，第 3.4.2 节」](pt01ch03s04s02.md)。模型树中步骤相关对象旁边的文本（如 **（Created）** 和 **（Propagated）**）表示对象的状态。您可以通过在对象上点击鼠标按钮 3 并从出现的菜单中选择一个操作来更改步骤相关对象的状态。这些操作与步骤相关管理器中可用的操作相对应。有关更多信息，请参阅[「理解修改后的步骤相关对象，第 3.4.7 节」](pt01ch03s04s07.md)。

您可以通过在项目上点击鼠标按钮 3 并从出现的菜单中选择**抑制**来抑制特征、约束（在 Interaction 模块中）、截面分配（在 Property 模块中）或步骤相关对象。红色「X」出现在模型树中该项旁边，以指示该项被抑制。您可以通过在项目上点击鼠标按钮 3 并选择**恢复**来恢复该项目。Abaqus/CAE 从模型树中移除红色「X」以指示该项不再被抑制。相同的信息显示在管理器中。有关更多信息，请参阅[「抑制和恢复对象，第 3.4.3 节」](pt01ch03s04s03.md)。
