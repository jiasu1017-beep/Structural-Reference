# 15.12.7 选择定义连接器几何的过程





您必须创建装配级线特征来定义用于建模连接器的底层几何。线特征包含从当前视口中装配连接点的线，或从装配连接到地面的点。

Abaqus/CAE提供了两种可用于为装配创建连接器几何的方法：
- **连接器构建器**使您能够执行建模连接器所涉及的所有步骤：创建单个装配级线特征、可选地在其端点创建参考点、为其分配连接器截面，以及为其任一端点指定方向。您应该使用此对话框添加少量线特征。请参见「创建单个连接器」，第15.12.8节，获取使用**连接器构建器**的说明。
- **Create Wire Feature**对话框使您能够在单个线特征中创建多条线。您应该使用此对话框来定义大量相似的连接器。此对话框仅创建线特征；您必须使用「相互作用」模块中的其他对话框来执行后续建模步骤，如创建参考点和基准坐标系、将连接器截面分配给线，以及指定线端点的方向。请参见「为多个连接器创建或修改线特征」，第15.12.9节，获取使用**Create Wire Feature**对话框的说明。
