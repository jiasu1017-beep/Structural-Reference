# 20.11.3 创建斜构造线


使用 Sketcher 工具箱中的斜构造线工具,帮助沿两条点定义的任意斜线来定位和对齐对象。下图展示了如何使用斜构造线和一组垂直构造线来对齐圆心:

![](../graphics/ske-congeom-example-nls.png)

您也可以使用斜构造线来定义旋转实体和曲面的旋转轴。

**绘制斜构造线的步骤:**

1. 从 Sketcher 工具箱的构造工具中,选择斜构造线工具 ![](../graphics/ico_sketchConstLineOblique.png)。有关 Sketcher 工具箱中工具的图示,请参见["Sketcher 工具," 第 20.4.1 节](pt03ch20s04s01.md)。Abaqus/CAE 会在提示区域显示提示,引导您完成操作。
2. 单击将位于斜构造线上的两个点。或者,您可以在提示区域出现的文本框中输入它们的 *X-Y* 坐标。选择第一个点时会出现斜构造线,该点会固定线条直到您选择第二个点。
3. 要创建更多斜构造线,请重复上一步。
4. 完成创建斜构造线后,执行以下操作之一: - 在 Abaqus/CAE 窗口任意位置单击鼠标按钮 2。 - 选择 Sketcher 工具箱中的任何其他工具。 - 单击提示区域中的取消按钮 ![](../graphics/afxI_cancel.png)。
