# 12.13.19 定义钢筋层





创建均匀壳截面、复合壳截面、膜截面或表面截面时，您可以使用**Rebar Layers**选项定义一层或多层增强（钢筋）。您可以从截面编辑器访问**Rebar Layers**对话框。请参见[Abaqus Analysis User's Guide的第2.2.3节，"定义增强"](../usb/usb-link.md#usb-int-erebarlayer)。

**定义钢筋层：**

1. 从壳、膜或表面截面编辑器的**Options**字段中，点击![](../graphics/ico_rebarLayers.png)。出现**Rebar Layers**对话框。
2. 指定钢筋几何类型。
   - 选择**Constant**以获得恒定钢筋间距。
   - 如果钢筋间距随圆柱坐标系中的径向位置而变化，请选择**Angular**。
   - 如果钢筋间距和取向由轮胎升起方程决定，请选择**Lift equation--based**。
3. 在表格中，为每个钢筋层输入一行数据。每行应包含以下信息：
   - 钢筋层的名称。
   - 形成钢筋层的材料名称。点击**Material**列，然后点击出现的箭头以显示可用材料列表，并选择形成钢筋层的材料。
   - 每个钢筋的横截面积。
   - 截面平面内的钢筋间距。对于角度钢筋间距，请以度为单位指定间距角。对于基于升起方程的间距，请在未固化几何中指定间距。
   - 钢筋相对于钢筋参考取向1方向的角度取向（度）。对于基于升起方程的间距，请在未固化几何中指定角度。或者，您可以指定取向名称，在这种情况下，Abaqus/CAE假定为用户定义的取向。您必须提供用户子程序[`ORIENT`](../sub/sub-link.md#sub-xsl-orient)，其中包含指定取向名称的用户定义取向的定义。
   - 钢筋在壳厚度方向的位置，相对于壳的中面测量，正向为壳正法线方向（不适用于膜截面或表面截面）。
   - 帘线延伸比*e*，（仅适用于基于升起方程的间距）。
   - 半径，![](../graphics/usi_eqn00391.gif)，描述钢筋在未固化几何中的位置，相对于圆柱坐标系中的旋转轴测量（仅适用于基于升起方程的间距）。
4. 点击**OK**返回截面编辑器。
