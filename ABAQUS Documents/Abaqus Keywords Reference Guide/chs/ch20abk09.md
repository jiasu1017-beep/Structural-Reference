# *USER OUTPUT VARIABLES





### *USER OUTPUT VARIABLES指定用户变量数量。

此选项用于允许Abaqus在每个材料计算点为在用户子程序[`UVARM`](../sub/sub-link.md#sub-xsl-uvarm)中定义的用户定义输出变量分配空间。如果使用[*USER OUTPUT VARIABLES](ch20abk09.md)选项，它必须出现在每个相关材料或垫圈行为定义中。

**产品：**Abaqus/Standard  Abaqus/CAE

**类型：**模型数据

**级别：**模型

**Abaqus/CAE：**Property模块

##### **参考：**

- ["UVARM," Section 1.1.55 of the Abaqus User Subroutines Reference Guide](../sub/sub-link.md#sub-rtn-uuvarm)

**此选项没有关联的参数。**
