# 65.4.4 Resuming suppressed features







Select ****Feature**![](../graphics/images/arrow.gif)**Resume**** from the main menu bar to resume suppressed features. You can also resume a group of features by selecting the features in the Model Tree, clicking mouse button 3, and selecting **Resume** from the menu that appears.

Resuming a feature fully restores it to the part or assembly. You can resume the last feature you suppressed, all suppressed features, or just selected features. When you resume a child feature, Abaqus/CAE also resumes the parent features automatically.  However, when you resume a parent feature, its children are not resumed unless you select them. To help you select a feature's children, you can click mouse button 3 on the feature in the Model Tree and select **Show children** from the menu that appears.

**To resume a suppressed feature:**

1. From the main menu bar, select ****Feature**![](../graphics/images/arrow.gif)**Resume****. **Tip:**You can also resume a feature using the ![](../graphics/ico_featureResume.png) tool located in the module toolbox.
2. Do one of the following: - Click **Last Set** to resume the most recently suppressed feature and all its parents. - Click **All** to resume all suppressed features. - Select the feature to resume from the Model Tree. You can select multiple features to resume. The resumed features reappear in the viewport.
