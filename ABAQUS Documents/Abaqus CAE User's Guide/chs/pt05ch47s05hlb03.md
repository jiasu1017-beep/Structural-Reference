# 47.6.4 自定义 X–Y 曲线外观





### 47.6.4 自定义 X–Y 曲线外观

使用 **曲线选项（Curve Options）** 对话框来自定义在 X–Y 绘图中表示 X–Y 数据对象的线条的颜色、样式和粗细。[图 47–42](pt05ch47s05hlb03.md#usv-xyplot-curveopts) 说明了对 X–Y 数据曲线进行自定义的几种方式。

**图 47–42** 带自定义曲线的 X–Y 绘图。

![](../graphics/usv-xyplot-curveopts.png)

您选择的颜色、样式和粗细会沿曲线显示，并在 [图例](pt05ch56s01.md) 中显示。只有当 **显示（Show）** 处于开启状态时，自定义选项才可用。

默认情况下，Abaqus/CAE 通过引用 X–Y 自动颜色列表来为绘图中 X–Y 曲线着色。您可以从 **编辑 XY 自动颜色（Edit XY-Auto-Colors）** 对话框修改此列表；请参阅 ["编辑 X–Y 绘图的自动颜色列表，" 第 47.6.6 节"](pt05ch47s05hlb05.md)。

**要自定义 X–Y 绘图中曲线的外观：**

1. 找到 **显示（Show）** 选项。**显示（Show）** 选项出现在 **曲线选项（Curve Options）** 对话框的右下角。
2. 从 **曲线（Curves）** 字段中，[选择一条或多条 X–Y 曲线](pt05ch47s05s01.md) 来自定义其属性。**注意：**要使 X–Y 曲线可供选择，您必须首先绘制它。您选择的自定义选项将应用于您选择的所有曲线；如果要更改多条曲线的外观，必须一次自定义一条。
3. 切换 **显示（Show）** 以显示或隐藏表示每条所选 X–Y 曲线的线条。当 **显示（Show）** 开启时，曲线会显示，线条属性选项会启用。
4. 选择线条颜色：1. 点击颜色样本 ![](../graphics/usi-colorsample.png)。Abaqus/CAE 显示 **选择颜色（Select Color）** 对话框。2. 使用 **选择颜色（Select Color）** 对话框中的方法选择新颜色。有关更多信息，请参阅 ["自定义颜色，" 第 3.2.9 节"](pt01ch03s02s09.md)。3. 点击 **确定（OK）** 关闭 **选择颜色（Select Color）** 对话框。所选的 X–Y 曲线变为所选颜色。
5. 选择线条样式：1. 点击 **样式（Style）** 按钮显示线条样式选项（实线、虚线等）。2. 从样式列表中，点击所需的线条样式。所选的 X–Y 曲线变为所选样式。
6. 选择线条粗细：1. 点击 **粗细（Thickness）** 按钮显示线条粗细选项。2. 从粗细列表中，点击所需的线条粗细。所选的 X–Y 曲线变为指定的粗细。
7. 点击 **关闭（Dismiss）** 关闭 **坐标轴选项（Axis Options）** 对话框。

# 47.6.5 自定义 X–Y 曲线上使用的符号





### 47.6.5 自定义 X–Y 曲线上使用的符号

使用 **曲线选项（Curve Options）** 对话框来自定义用于表示 X–Y 曲线上数据点的符号外观。例如，在 [图 47–43](pt05ch47s05hlb04.md#viw-symbolopt) 中，左侧的绘图使用默认符号，而右侧的绘图使用自定义符号。

**图 47–43** 带不同数据点符号的 X–Y 绘图。

![](../graphics/viw-symbolopt.png)

您选择的符号会沿曲线显示，并在 [图例](pt05ch56s01.md) 中显示。只有当 **显示符号（Show symbol）** 处于开启状态时，自定义选项才可用。

**要自定义 X–Y 曲线上使用的符号：**

1. 找到 **属性（Attributes）** 选项。**属性（Attributes）** 选项出现在 **曲线选项（Curve Options）** 对话框的下半部分。
2. 从 **曲线（Curves）** 字段中，[选择一条或多条 X–Y 曲线](pt05ch47s05s01.md) 来自定义其符号。**注意：**要使 X–Y 曲线可供选择，您必须首先绘制它。您选择的自定义选项将应用于您选择的所有曲线；如果要更改多条曲线上的符号，必须一次自定义一条。
3. 切换 **显示符号（Show symbol）** 以显示或隐藏表示每条所选 X–Y 曲线数据点的符号。当 **显示符号（Show symbol）** 开启时，符号会显示，符号属性会启用。
4. 选择符号颜色：1. 点击颜色样本 ![](../graphics/usi-colorsample.png)。Abaqus/CAE 显示 **选择颜色（Select Color）** 对话框。2. 使用 **选择颜色（Select Color）** 对话框中的方法选择新颜色。有关更多信息，请参阅 ["自定义颜色，" 第 3.2.9 节"](pt01ch03s02s09.md)。3. 点击 **确定（OK）** 关闭 **选择颜色（Select Color）** 对话框。颜色样本变为所选颜色。在您点击 **XY 绘图选项（XY Plot Options）** 对话框中的 **确定（OK）** 或 **应用（Apply）** 之前，新颜色不会出现在视口中。
5. 选择符号：1. 点击 **符号（Symbol）** 按钮显示符号选项。2. 从符号列表中，点击所需的符号。指定符号出现在所选的 X–Y 曲线上。
6. 选择符号大小：1. 点击 **大小（Size）** 按钮显示符号大小选项。2. 从符号大小选项中，点击所需的符号大小。指定符号大小出现在所选的 X–Y 曲线上。
7. 选择符号的频率。在 **频率（Frequency）** 文本字段中输入一个整数，以选择符号沿曲线出现的频率。例如，如果输入频率为 2，Abaqus 会在每隔一个数据点显示符号。不允许值为零；要禁止显示符号，请关闭 **显示符号（Show symbol）**。
8. 点击 **关闭（Dismiss）** 关闭 **曲线选项（Curve Options）** 对话框。

# 47.6.2 选择要自定义的一条或多条 X–Y 曲线





### 47.6.2 选择要自定义的一条或多条 X–Y 曲线

要自定义曲线，您首先选择它，然后选择自定义选项。使用 **曲线选项（Curve Options）** 对话框中的 **曲线（Curves）** 字段来选择一条或多条要自定义的 X–Y 曲线。**曲线（Curves）** 字段列出显示的 X–Y 曲线，并通过名称标识每条曲线。有关曲线命名约定的信息，请参阅 ["理解"临时"和其他 X–Y 数据对象名称，" 第 47.1.3 节"](pt05ch47s01s03.md)。

选择了一条或多条要自定义的 X–Y 曲线后，您可以使用 **属性（Attributes）** 框架中的选项来控制表示数据的线条和符号，并输入将出现在图例中的文本。

**要选择一条或多条 X–Y 曲线进行自定义：**

1. 找到 **曲线（Curves）** 字段。**曲线（Curves）** 字段位于对话框顶部。
2. 选择一条或多条要自定义的 X–Y 曲线。有关在对话框中选择多个项目的更多信息，请参阅 ["从列表和表格中选择多个项目，" 第 3.2.11 节"](pt01ch03s02s11.md)。**注意：**要使 X–Y 曲线可供选择，您必须首先绘制它。如果您选择了一条曲线，**属性（Attributes）** 框架会显示该曲线的当前属性。如果您选择了多条曲线，Abaqus/CAE 会将 **图例文本（Legend text）** 字段显示为空白。**属性（Attributes）** 框架中的显示根据所选曲线是否具有相同属性而有所不同。如果属性相同，Abaqus/CAE 会显示它们；如果属性不同，Abaqus/CAE 会显示以下内容：- **显示（Show）** 和 **显示符号（Show symbol）** 复选框为灰色。- 线条 **颜色（Color）** 和符号 **颜色（Color）** 文本字段显示术语 **保持原样（As is）**。- **样式（Style）**、**粗细（Thickness）**、**符号（Symbol）** 和 **大小（Size）** 按钮为空白。- **频率（Frequency）** 文本字段为空白。
3. 从 **属性（Attributes）** 字段中，您可以控制以下内容：- 图例中描述数据的文本。有关更多信息，请参阅 ["自定义 X–Y 绘图图例，" 第 47.6.3 节"](pt05ch47s05hlb02.md)。- 表示数据（曲线）的线条外观。有关更多信息，请参阅 ["自定义 X–Y 曲线外观，" 第 47.6.4 节"](pt05ch47s05hlb03.md)。- 曲线沿线符号的外观。有关更多信息，请参阅 ["自定义 X–Y 曲线上使用的符号，" 第 47.6.5 节"](pt05ch47s05hlb04.md)。

# 47.6.6 编辑 X–Y 绘图的自动颜色列表





### 47.6.6 编辑 X–Y 绘图的自动颜色列表

Abaqus/CAE 通过引用 X–Y 自动颜色列表中的颜色定义来自动为 X–Y 曲线分配颜色。此列表为 X–Y 曲线提供颜色，方式与常规 **自动颜色（Auto Colors）** 列表用于为几何和网格元素着色的方式相同，但这两个颜色列表是两个独立的实体。

本节介绍如何编辑 X–Y 自动颜色列表以修改 X–Y 曲线的默认颜色分配集。您也可以从 **曲线选项（Curve Options）** 对话框更改单个 X–Y 曲线的颜色分配。请参阅 ["自定义 X–Y 曲线外观，" 第 47.6.4 节"](pt05ch47s05hlb03.md)。

**要编辑 X–Y 自动颜色列表中的颜色：**

1. 在结果树中，在 **XY 绘图（XY Plots）** 容器上点击鼠标按钮 3，然后从出现的列表中选择 **编辑 XY 自动颜色（Edit XY-Auto-Colors）**。Abaqus/CAE 显示 **编辑 XY 自动颜色（Edit XY-Auto-Colors）** 对话框。
2. 要在 X–Y 自动颜色列表中插入新颜色：1. 在 X–Y 自动颜色列表中，高亮显示您要添加的颜色之前或之后的颜色。2. 点击 **插入之前（Insert Before）** 或 **插入之后（Insert After）** 以在 X–Y 自动颜色列表中的指示位置添加新颜色。**选择颜色（Select Color）** 对话框打开。3. 按照 ["自定义颜色，" 第 3.2.9 节"](pt01ch03s02s09.md) 中描述的步骤选择一种颜色；然后点击 **确定（OK）** 关闭 **选择颜色（Select Color）** 对话框。Abaqus/CAE 在所选位置添加新颜色。
3. 要更改 X–Y 自动颜色列表中的某种颜色，双击该颜色并从出现的 **选择颜色（Select Color）** 对话框中选择新颜色。
4. 要在 X–Y 自动颜色列表中移动颜色，高亮显示该颜色，然后点击 **上移（Move Up）** 或 **下移（Move Down）** 以沿所选方向移动一个位置。
5. 要从 X–Y 自动颜色列表中删除颜色，高亮显示该颜色，然后点击 **删除（Delete）**。
6. 继续添加、更改、移动和删除颜色，直到 X–Y 自动颜色列表包含您所需顺序的颜色。
7. 点击 **确定（OK）**。Abaqus/CAE 将修订后的 X–Y 自动颜色列表用于此后任何 X–Y 曲线的颜色编码。
