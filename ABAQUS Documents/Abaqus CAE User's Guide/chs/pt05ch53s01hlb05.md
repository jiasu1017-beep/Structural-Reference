# 53.2.5 自定义铺层堆叠图中的标签





点击"**Labels**"选项卡来自定义铺层堆叠图中显示的文本和符号;例如,材料和铺层名称、状态块以及积分点。

**自定义铺层堆叠图中标签的步骤:**

1. 找到标签选项。 1. 点击提示区域中的"**铺层堆叠图选项**"按钮,或执行以下操作之一: - 在"属性"模块中,选择 ****View**![](../graphics/images/arrow.gif)**Ply Stack Plot Options****。 - 在"可视化"模块中,选择 ****Options**![](../graphics/images/arrow.gif)**Ply Stack Plot**** 或点击工具箱中的 ![](../graphics/ico_plotPlyStackOptions.png) 工具。 2. 在出现的"铺层堆叠图选项"对话框中点击"**Labels**"选项卡。
2. 选择铺层堆叠图中所有文本的字体。 1. 点击"**Set Font for All Labels**"显示"选择字体"对话框。 2. 使用字体选择器对话框设置您想要的字体属性。 3. 点击"**OK**"关闭"选择字体"对话框并返回"铺层堆叠图选项"对话框。
3. 切换打开一个选项以显示它。
4. 选择文本的颜色: 1. 点击颜色样本 ![](../graphics/usi-colorsample.png)。Abaqus/CAE显示"选择颜色"对话框。 2. 使用"选择颜色"对话框中的方法之一来选择新颜色。有关更多信息,请参阅 ["自定义颜色," 第3.2.9节](pt01ch03s02s09.md)。 3. 点击"**OK**"关闭"选择颜色"对话框。颜色样本变为所选颜色。
5. 点击"**Apply**"实施您的更改。铺层堆叠图会根据您的规格进行更改。默认情况下,您的更改会在会话期间保存,并会影响该视口中所有后续的铺层堆叠图。如果要为后续会话保留更改,请将它们保存到文件。有关更多信息,请参阅 ["为后续会话保存自定义," 第55.1.1节](pt05ch55s01s01.md)。
