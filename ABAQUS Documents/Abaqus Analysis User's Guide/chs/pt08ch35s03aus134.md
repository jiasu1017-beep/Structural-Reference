### 概述

壳-实体耦合（Shell-to-Solid Coupling）是一种基于表面的技术，用于将壳单元与实体单元耦合在一起。这种方法特别适用于以下情况：

- 从壳单元区域过渡到实体单元区域
- 局部详细分析与全局简化分析结合
- 保留几何特征同时简化计算

### 壳-实体耦合特性

- 允许壳网格与实体网格之间不匹配
- 自动创建内部分布耦合约束
- 保持位移和转动协调
- 适用于线性和非线性分析

### 壳-实体耦合定义

| **输入文件用法：** | ```
*SHELL TO SOLID COUPLING, NAME=name
shell_edge_surface, solid_surface
``` |
| --- | --- |

### 参数说明

| 参数 | 描述 |
| --- | --- |
| NAME | 耦合约束名称 |
| shell_edge_surface | 壳边缘表面 |
| solid_surface | 实体表面 |

### 位置容差

位置容差控制壳节点与实体表面之间的最大允许距离：

| **输入文件用法：** | ```
*SHELL TO SOLID COUPLING, NAME=name, POSITION TOLERANCE=value
shell_edge_surface, solid_surface
``` |
| --- | --- |

### 影响距离

影响距离定义了壳边缘到实体节点的有效范围：

| **输入文件用法：** | ```
*SHELL TO SOLID COUPLING, NAME=name, INFLUENCE DISTANCE=value
shell_edge_surface, solid_surface
``` |
| --- | --- |

### 约束生成

壳-实体耦合通过以下方式强制位移协调：

1. 对于每个壳边缘节点，创建一个内部分布耦合约束
2. 壳节点作为参考节点
3. 实体表面节点作为从节点
4. 力以自平衡方式分布

### 使用建议

- 壳边缘应位于实体表面上
- 实体网格应足够细以捕捉耦合区域的变形
- 避免在应力梯度大的区域使用
- 建议至少两层实体单元穿过厚度方向
