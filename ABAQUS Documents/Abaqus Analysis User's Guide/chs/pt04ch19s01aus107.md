### 概述

设计灵敏度分析（DSA）：
- 使用 Abaqus/Design 执行，这是 Abaqus/Standard 的附加选项；
- 提供响应相对于指定设计参数的灵敏度；
- 可用于仅包含应力/位移单元的模型的静态应力和频率分析；
- 可包括影响以下内容的 design parameters：材料属性（弹性、超弹性和超泡沫模型）；截面属性；集中力和力矩；以及节点坐标（以及适用的梁和壳的法线）。

### 设计灵敏度分析

设计灵敏度分析（DSA）功能提供某些输出变量相对于指定设计参数的导数。这些导数通常被称为*灵敏度*，因为它们提供了一阶度量，表明输出变量对设计参数变化的敏感程度。计算灵敏度的输出变量被称为*设计响应*或简称为*响应*。设计参数从现有分析参数集中选择。例如，您可以获取应力相对于杨氏模量的导数；应力是响应，杨氏模量是设计参数。灵敏度基于直接微分法并结合半解析计算技术进行计算。在半解析技术中，某些导数使用数值（有限）差分计算，因此需要设计参数的摄动。对于这些导数，默认情况下 Abaqus/Design 将使用中心差分方案，并基于启发式算法自动确定适当的摄动大小。您可以直接指定数值差分方法和摄动大小来覆盖这些默认值。关于 DSA 理论的完整讨论，请参阅 ["设计灵敏度分析，" Abaqus 理论指南第 2.18.1 节](../stm/stm-link.md#stm-anl-dsa)。

### 指定响应

响应请求使用与向输出数据库指定输出请求类似的语法进行指定。除了特征值和特征频率外，没有默认响应——如果没有请求响应，则不会输出响应灵敏度。如果 DSA 在频率步骤中处于活动状态，特征值和特征频率灵敏度将自动输出。指定响应将导致输出响应和响应灵敏度。

| **输入文件用法：** | 使用以下选项请求设计响应： |
| --- | --- |
|  | ``` [*DESIGN RESPONSE](../key/key-link.md#usb-kws-hdesignresponse), FREQUENCY=*interval*, MODE LIST [*CONTACT RESPONSE](../key/key-link.md#usb-kws-hcontactresponse), MASTER=*master name*, NSET=*nset name*, SLAVE=*slave name* [*ELEMENT RESPONSE](../key/key-link.md#usb-kws-helementresponse), ELSET=*elset name* [*NODE RESPONSE](../key/key-link.md#usb-kws-hnoderesponse), NSET=*nset name* ``` |

#### 在多个步骤中请求响应

除非重新指定，否则在步骤中定义的响应请求将根据以下规则传播到后续步骤：

1. General 步骤中的请求传播到后续 general 步骤。
2. 线性扰动步骤中的请求传播到后续连续线性扰动步骤。
3. 当非 DSA 步骤出现在 DSA 步骤之间时，必须在非 DSA 步骤之后的 DSA 步骤中重新指定响应。

#### 关于响应的限制

可用响应是现有输出变量的子集。以下描述了基于过程类型的有效响应。
- 对于静态步骤，有效响应为： - 节点响应：U 和 RF - 单元响应：S、SF、SINV、SP、E、SE、EP、EE、EEP、LE、LEP、NE、NEP、ENER、ELEN、EVOL 和 MASS - 接触响应：CSTRESS 和 CDISP
- 对于频率步骤，有效响应为： - 节点响应：无 - 单元响应：MASS - 接触响应：无 - 特征值（EIGVAL）和特征频率（EIGFREQ）灵敏度会自动输出。

### 多增量分析中的历史依赖性和公式类型

DSA 同时实现了 total 和 incremental 公式。公式的选择取决于分析是否具有历史依赖性。以下是对这些公式类型的简要描述。更详细的讨论可以在 ["设计灵敏度分析，" Abaqus 理论指南第 2.18.1 节](../stm/stm-link.md#stm-anl-dsa) 中找到。默认情况下，使用增量 DSA 公式。您只能为整个模型指定 DSA 公式；如果作为步骤定义的一部分给出，此规范将被忽略。

#### 增量 DSA 公式

在增量公式中，假设问题具有历史依赖性。Abaqus/Design 求解增量位移灵敏度，并在增量结束时更新总位移灵敏度。由于历史依赖性，当前增量的增量位移灵敏度取决于增量开始时状态变量的灵敏度，就像在平衡分析中增量位移取决于增量开始时的状态变量一样。因此，Abaqus/Design 还必须在每个增量中计算和更新状态变量灵敏度。因此，必须在最后一个活跃 DSA 步骤之前的所有步骤中激活 DSA，并且将在这些步骤的所有增量中执行 DSA 计算，无论是否为给定步骤请求了设计响应。如果为某个步骤请求了响应，则指定的响应频率将被忽略用于 DSA 计算目的（输出将被写入的频率仍由指定的响应频率控制）。

增量 DSA 公式的缺点是其成本，因为需要在最后一个 DSA 增量之前的所有增量中计算状态变量和增量位移灵敏度。如果问题具有历史依赖性，这种增加的成本是不可避免的；但如果问题不具有历史依赖性，则是不必要的。因此，对于不具有历史依赖性的问题，应选择总 DSA 公式。

| **输入文件用法：** | ``` [*DSA CONTROLS](../key/key-link.md#usb-kws-mdsacontrols), FORMULATION=INCREMENTAL ``` |
| --- | --- |

#### 总 DSA 公式

在总位移公式中，总位移灵敏度是基于问题不具有历史依赖性的假设直接计算的。换句话说，位移灵敏度不依赖于先前增量中计算的灵敏度结果。因此，总公式的优点是只需要在感兴趣的增量中执行灵敏度计算。您可以通过仅为所需步骤激活 DSA 并为每个设计响应请求指定所需的频率来控制何时执行 DSA 计算。

您可以选择在与已知具有历史依赖性的问题中使用总 DSA 公式。但是，在这种情况下，DSA 解决方案是近似的，并且近似程度随着问题历史依赖性的增强而增加。为了评估使用总 DSA 公式的有效性，建议您对典型问题同时运行增量和总灵敏度分析并比较结果。

| **输入文件用法：** | ``` [*DSA CONTROLS](../key/key-link.md#usb-kws-mdsacontrols), FORMULATION=TOTAL ``` |
| --- | --- |

### 线性扰动步骤中的 DSA

可以在线性扰动步骤中计算扰动响应的灵敏度（请参阅 ["一般和线性扰动过程，" 第 6.1.3 节](pt03ch06s01aus44.md)）。如果考虑几何非线性，则扰动响应将包括基础状态中应力和载荷刚化的影响。由于我们需要计算增量（扰动）响应的灵敏度，因此必须在基础步骤结束时已知应力和载荷刚化效果的灵敏度。因此，如果在基础步骤中考虑了几何非线性，则无论使用何种公式（total 或 incremental），DSA 在基础步骤中也必须是活跃的。

### 设计参数摄动大小的确定

半解析技术的基础是使用数值差分来获得某些单元向量和矩阵的导数（请参阅 ["设计灵敏度分析，" Abaqus 理论指南第 2.18.1 节](../stm/stm-link.md#stm-anl-dsa)）。除非您直接指定，否则 Abaqus/Design 将自动确定用于半解析技术的适当摄动大小。Abaqus/Design 使用基于与单元关联的标量 *s* 的行为的启发式摄动大小调整算法来确定摄动大小。默认情况下，摄动大小调整算法仅应用于每个 DSA 活跃步骤的第一个增量（静态过程）或第一个模式（频率过程）。然后在 DSA 计算所在的步骤中剩余的增量或模式中重复使用这些摄动大小。

该算法的目标是找到适合数值差分的最佳摄动大小。差分公式基于泰勒级数展开，并且要计算的导数的近似阶数反映在级数中被忽略的项中。近似导数的准确性通常强烈依赖于差分公式中使用的摄动大小。选择过大的摄动大小将导致截断误差，当近似阶数不再有效时会发生这种误差（即，由于截断了泰勒级数中的高阶项）。太小的摄动大小将导致差分运算中的不准确，通常称为消除误差。

该算法试图通过观察 *s* 的行为来找到在消除误差和截断误差之间取得最佳折衷的摄动大小。对于每个设计参数，计算跨越多个数量级的摄动大小的 *s*。计算连续摄动大小之间 *s* 的误差为 ![](../graphics/usb_eqn02142.gif)。选择产生可接受误差 ![](../graphics/usb_eqn02143.gif) 的摄动大小作为最佳摄动大小。

该标量 *s* 的选择如下：
- 静态过程。对于静态步骤，*s* 被选择为单元伪载荷的范数（单元残差相对于设计参数的偏导数）。
- 频率过程。对于频率步骤，*s* 由矩阵的元素贡献计算得出，该矩阵涉及质量和刚度矩阵的导数（即 ![](../graphics/usb_eqn02144.gif)，其中 ![](../graphics/usb_eqn02145.gif) 是刚度，![](../graphics/usb_eqn00668.gif) 是质量，*h* 是设计参数，而 ![](../graphics/usb_eqn00445.gif) 是特征值）。标量 *s* 取为该矩阵到特征向量 ![](../graphics/usb_eqn02146.gif) 的投影。如果将摄动大小调整算法应用于具有不同特征值的模式，则 ![](../graphics/usb_eqn02146.gif) 取为与该模式关联的特征向量。但是，如果一个模式恰好与重复特征值关联，则 ![](../graphics/usb_eqn02146.gif) 取为与重复特征值关联的所有特征向量的和。因此，与重复特征值关联的整个模式集将由摄动大小调整算法同时处理（重复特征值的特征值灵敏度从相同的缩减特征值系统同时获得）。

有关 *s* 的选择的其他详细信息，请参阅 ["设计灵敏度分析，" Abaqus 理论指南第 2.18.1 节](../stm/stm-link.md#stm-anl-dsa)。

#### 控制数值差分行为

您可以控制数值差分操作的各个方面。这些方面将在以下章节中详细描述。您可以为整个模型和/或各个步骤指定 DSA 控制。为整个模型指定这些控制的效果是创建各种设置的新默认值。当为各个步骤指定这些控制时，将执行以下传播规则：
- 一旦在非扰动步骤中指定了 DSA 控制，它们将在所有后续非扰动步骤中保持有效，除非重新指定或重置。
- 一旦在扰动步骤中指定了 DSA 控制，它们将在所有后续连续扰动步骤中保持有效，除非重新指定或重置。

#### 重置 DSA 控制

您只能为各个步骤重置 DSA 控制。如果为整个模型指定了 DSA 控制，则在特定步骤中重置它们会将数值差分行为重置为为整个模型指定的行为；否则，该行为将被重置为原始默认值。将在行为重置后应用任何其他更改。

| **输入文件用法：** | 使用以下选项重置特定步骤的 DSA 控制： |
| --- | --- |
|  | ``` [*DSA CONTROLS](../key/key-link.md#usb-kws-mdsacontrols), RESET ``` |

#### 更改启发式摄动大小调整算法的默认值

以下两节描述如何更改与扰动大小调整算法相关联的某些参数的默认值，以实现计算效率和准确性。

##### 更改默认容差

默认情况下，容差 ![](../graphics/usb_eqn02143.gif) 设置为 1.0×10⁴。对于未达到此容差的单元，警告消息将被写入消息文件。这些单元被收集在单元集中，可以在 Abaqus/CAE 的可视化模块中查看。重要的是要理解，此容差控制着获得最佳摄动大小所投入的精力；它不是数值微分准确性的直接度量。

| **输入文件用法：** | 使用以下选项覆盖默认容差： |
| --- | --- |
|  | ``` [*DSA CONTROLS](../key/key-link.md#usb-kws-mdsacontrols), TOLERANCE=*tolerance* ``` |

##### 更改使用摄动大小调整算法的频率

使用启发式算法确定摄动大小的计算成本很高。您可以指定重新计算摄动大小的频率。例如，指定大小频率 *n* 将导致 Abaqus/Design 每隔 *n* 个增量或特征模式确定新的摄动大小。摄动大小将始终在每个 DSA 活跃步骤的第一个增量或特征模式重新计算，这相当于指定大小频率为 0。由于摄动大小调整算法的计算成本很高，因此应谨慎确保大小频率尽可能大（或为零）。

如上所述，摄动大小调整算法同时应用于与重复特征值关联的所有模式。因此，基于大小频率与重复特征值关联的实际模式数量是无关紧要的，只要至少有一个即可。

| **输入文件用法：** | 使用以下选项指定重新计算摄动大小的频率： |
| --- | --- |
|  | ``` [*DSA CONTROLS](../key/key-link.md#usb-kws-mdsacontrols), SIZING FREQUENCY=*frequency* ``` |

#### 覆盖默认启发式摄动大小调整算法

### DSA 解决方案的准确性

从 ["设计灵敏度分析，" Abaqus 理论指南第 2.18.1 节](../stm/stm-link.md#stm-anl-dsa) 中可以看出，DSA 解决方案的准确性取决于数值计算的导数的准确性，以及对于非线性静态分析，取决于切线刚度矩阵的准确性。数值计算的导数的准确性受半解析 DSA 算法控制；您可以通过指定 DSA 控制来控制它。在非线性静态分析中，DSA 使用在最后一次平衡迭代期间形成的切线刚度矩阵。获得准确平衡解决方案所需的切线刚度矩阵的准确性可能不足以获得准确的 DSA 解决方案，这种情况是可能的。在这种情况下，您可以在平衡分析期间收紧收敛容差，以获得更准确的切线刚度矩阵（请参阅 ["常用控制参数，" 第 7.2.2 节](pt03ch07s02aus50.md)）。此外，当忽略切线刚度中的不对称项时（即，不使用不对称矩阵存储和求解方案时；请参阅 ["定义分析，" 第 6.1.2 节](pt03ch06s01abo05.md)），通常可以获得准确的平衡解决方案。然而，即使忽略了轻微的不对称刚度项，DSA 解决方案也可能不准确。因此，建议在切线刚度矩阵已知为不对称时对 DSA 使用不对称求解方案。

在某些情况下，特定时刻的响应可能相对于设计参数不连续。例如，在不连续点处，设计参数的变化可能导致节点进入接触，摩擦行为从粘附变为滑动，或材料点从弹性变为非弹性行为。由于 DSA 计算使用数值差分，差分方案中使用的设计参数的摄动可能导致被差分的响应值位于不连续面的两侧。如果发生这种情况，则无法保证计算导数的准确性。从数学上讲，响应相对于设计参数的导数（灵敏度）在不连续点处不存在。实际上，在任何给定时刻，响应恰好位于不连续面上的可能性很小。在响应接近不连续面的情况下，如果您选择使用默认摄动大小调整算法，该算法将尝试选择设计参数摄动大小，使扰动响应的值保持在不连续面的同一侧。此外，对于接触单元，DSA 计算不会在与关联接触节点打开的增量中执行。通常，任何增量中的全局结果不会受到模型中少数不连续点的影响。

### 接触相互作用

在设计灵敏度分析中，支持变形体和刚体表面之间基于表面的接触，包括小或有限滑动相对表面运动以及摩擦。在所有摩擦模型中，只有摩擦系数（无测试数据输入）可以成为设计相关的。形状设计参数不适用于刚体表面。不支持变形体表面之间的接触。

### 重启设计灵敏度分析

可以重启设计灵敏度分析（请参阅 ["重启分析，" 第 9.1.1 节](pt04ch09s01aus53.md)）。但是，必须在基础分析中激活了 DSA，并且不能在重启运行中修改设计参数或梯度数据。重启的分析将遵循适用于常规分析的所有 DSA 传播规则。对于总公式 DSA，您可以选择在添加到重启运行的任何新步骤中激活或停用 DSA。但是，对于增量公式 DSA，必须在尝试重启的步骤中激活 DSA，才能在重启的分析中继续执行 DSA。

### 过程

DSA 在以下分析过程中可用：
- 频率分析
- 静态应力分析（包括几何非线性和接触）

以下分析过程和技术不受支持：
- 使用 Riks 方法的静态应力分析
- 子结构
- 网格修改或替换
- 导入和传输结果
- 对称模型生成和结果传输
- 轮廓积分
- 频率过程中的循环对称

#### 子模型限制

设计灵敏度分析可以在全局模型和子模型中执行，但限制是 DSA 解决方案不会从全局模型插值到子模型。这意味着，只有当从全局模型插值到子模型边界的结果可以被视为独立于为子模型灵敏度分析选择的设计参数时，DSA 在子模型中才有效。

### 材料选项

以下材料模型受支持：
- 各向同性、各向异性和正交各向异性弹性
- 超弹性
- 超泡沫

在这些模型中，只有直接输入的材料系数（而非测试数据）可以成为设计相关的。如果指定了测试数据，则可以通过由 Abaqus/Design 直接计算的村料系数替换该材料定义。支持和不支持的材料模型可以在同一分析中混合使用。

### 单元

固体、桁架、壳、梁、垫片和膜应力/位移单元受支持。在总 DSA 公式中，不能使用每个节点具有五个自由度的壳单元。支持和不支持的单元可以在同一分析中混合使用。
