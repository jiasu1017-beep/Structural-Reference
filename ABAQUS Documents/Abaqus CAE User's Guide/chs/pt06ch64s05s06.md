# 64.5.6 合并节点





从主菜单栏选择****网格**![](../graphics/images/arrow.gif)**编辑****可以合并节点。如果您只选择两个节点进行合并，Abaqus/CAE会在所选节点的中点创建一个新节点。如果您选择两个以上节点，则可以指定节点之间的最大距离，Abaqus/CAE将合并距离小于指定距离的节点。您还可以选择移除具有相同连接性的重复单元。

**合并节点的步骤：**

1. 进入"网格"模块。
2. 从上下文栏的**对象**字段中选择**零件**，然后从列表中选择一个零件。此工具仅适用于包含孤兒网格节点或使用自底向上网格划分程序创建的节点的零件。
3. 从主菜单栏选择****网格**![](../graphics/images/arrow.gif)**编辑****。Abaqus/CAE显示**编辑网格**对话框。 **提示：**您也可以使用位于"网格"模块工具箱底部的![](../graphics/ico_meshEdit.png)工具显示**编辑网格**对话框。
4. 在对话框中执行以下操作： 1. 在**类别**字段中选择**节点**。 2. 从**方法**列表中选择**合并**。
5. 从提示区域选择选择方法并选择要合并的节点。有关更多信息，请参见["选择要编辑的节点，" 第64.5.10节](pt06ch64s05s10.md)。 **提示：**您可以使用**选择**工具栏更改选择行为。有关更多信息，请参见[第6章，"在视口中选择对象"](pt01ch06.md)。
6. 如果您只选择两个节点进行合并，Abaqus/CAE会在所选节点的中点创建一个新节点。
7. 如果您选择两个以上节点进行合并，请执行以下操作： 1. 在提示区域输入**节点合并容差**以指定要合并的节点之间的最大距离。 2. 如果您输入的**节点合并容差**值太大，Abaqus/CAE可能会检测到来自同一单元的重复节点。Abaqus/CAE不会合并来自同一单元的节点，但过大的容差可能导致网格变形。如果**节点合并容差**太大，Abaqus/CAE会询问您是否要继续合并所选节点。 - 点击**是**继续。 - 点击**否**取消合并过程。 3. Abaqus/CAE会以洋红色高亮显示将被合并的任何节点，并询问您是否要继续。 - 点击**是**继续。 - 点击**否**取消合并过程。Abaqus/CAE会合并距离小于指定距离的节点，并用单个新节点替换它们。新节点的位置是被合并到新节点的节点组的平均位置。您还可以在"装配"模块中合并零件实例。有关更多信息，请参见["对零件实例执行布尔操作，" 第13.7节](pt03ch13s07.md)。如果您之前创建了包含合并操作删除的节点或单元的节点集或单元集，Abaqus/CAE会相应更新您的集合。 **提示：**如果在合并节点时出错，点击**编辑网格**对话框中的**撤销**可以撤消合并。Abaqus/CAE会将节点恢复到合并前的状态。
8. 根据需要重复上述步骤以合并其他节点。
9. 完成节点合并后，点击鼠标按钮2或提示区域中的取消按钮![](../graphics/afxI_cancel.png)退出过程。
