# 7.4 选择高亮方法





高亮方法控制在你与模型交互时 Abaqus/CAE 如何在视口中显示高亮。更改高亮方法的效果在你使用视图操作工具旋转模型或在与草图模块中草绘轮廓时最为明显。**硬件覆盖**提供最佳图形性能；但是，此选项并非所有图形适配器都支持。

在大多数情况下，你不应该需要更改默认高亮方法设置。当你启动会话时，Abaqus/CAE 会检测系统上安装的图形硬件并选择适当的高亮方法。但是，如果你使用的是不受支持的图形适配器，Abaqus/CAE 默认选择的设置可能不是最佳选择。

**要控制高亮方法：**

1. 找到高亮方法选项。从主菜单栏中，选择****View**![](../graphics/images/arrow.gif)**Graphics Options****。Abaqus/CAE 显示**图形选项**对话框。
2. 从**高亮方法**菜单中，选择以下之一：
   - **硬件覆盖**。此选项仅在你的系统上的图形适配器支持时才会出现。它使用图形硬件显示高亮层。如果你的工作站支持此选项，选择**硬件覆盖**可提供最佳质量和性能。
   - **XOR**。此选项使用软件来模拟高亮层。选择 **XOR** 选项使用布尔像素操作来模拟绘图操作，但可以根据底层像素的颜色产生不同的颜色。
   - **软件覆盖**。此选项也使用软件来模拟高亮层，并在合理的性能下提供良好的质量。但是，在某些系统上选择此选项会导致性能下降。
   - **混合**。混合方法将底层像素的颜色与期望的颜色组合，产生瞬态图形的近似效果。只有在其他选项都不令人满意时才应选择此选项。选择**混合**选项可能会影响性能和质量问题。
3. 单击**确定**实施更改并关闭对话框。你的更改会在会话期间保存。
