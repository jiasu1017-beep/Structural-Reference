# 16.9.27 定义体积电流





您可以创建体积电流载荷，以在耦合热电分析中在体积上定义电流密度。

**创建或编辑体积电流：**

1. 使用以下方法之一显示体积电流载荷编辑器：- 若要创建新的体积电流载荷，请遵循"创建载荷"，第16.8.1节中概述的过程（**类别**：`Electrical/Magnetic`；**所选步骤的类型**：`Body current`）。- 若要使用菜单或管理器编辑现有的体积电流载荷，请参见"编辑依赖于步骤的对象"，第3.4.13节。若要编辑应用载荷的区域，请参见"编辑应用规定条件的区域"，第16.8.4节。
2. 单击**分布**字段右侧的箭头，并从出现的列表中选择您的选项：- 选择**均匀**以定义在体积上均匀的载荷。- 选择分析场以定义空间变化载荷。只有对该载荷类型有效的分析场才会在选择列表中显示。或者，您可以单击![](../graphics/ico_analyticalField.png)创建新的分析场。（更多信息，请参见第58章"分析场工具集"。）
3. 在**幅值**文本字段中，输入电流密度（单位[C L<sup>-3</sup>T<sup>-1</sup>](../popups/usb-int-iconventions-unitsym.md)）。正幅值表示电流流入体积。
4. 如果需要，单击**幅值**字段右侧的箭头，并从出现的列表中选择您选择的幅值。或者，您可以单击![](../graphics/ico_amplitudeSmall.png)创建新的幅值。（更多信息，请参见第57章"幅值工具集"。）
5. 单击**确定**保存您的数据并退出编辑器。
