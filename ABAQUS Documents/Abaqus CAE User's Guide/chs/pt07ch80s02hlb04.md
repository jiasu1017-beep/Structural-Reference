# 80.2.4 重新定位视图切割





默认情况下，当首次激活视图切割时，Abaqus/CAE 将其放置在模型中间的位置。您可以使用视图切割管理器重新定位模型上的切割。您可以沿法线轴平移平面切割，或绕其他两个轴旋转它们。您可以重新定义圆柱或球形切割的半径，以及用于等值面切割的轮廓值，以重新定位它们穿过模型的切割位置。视图切割管理器中的定位选项仅适用于选定的切割，这可能与当前活动的切割不同。

在可视化模块以外的所有模块中，如果您修改模型，Abaqus/CAE 会将切割平面重新定位回中间位置。

**重新定位视图切割的步骤：**

1. 从主菜单栏中选择****工具**![](../graphics/images/arrow.gif)**View Cut**![](../graphics/images/arrow.gif)**Manager****。出现 **View Cut Manager**，其中列出在当前会话期间创建的所有视图切割。
2. 通过单击其名称在管理器中选择视图切割。选定的视图切割在管理器中突出显示。
3. 要平移 **Plane** 视图切割：
   1. 从视图切割管理器底部的菜单中选择 **Translate**。
   2. 使用以下方法之一重新定位视图切割：
      - 输入法线轴上 **Position** 的值。
      - 拖动 **Position** 滑块选择一个值。该滑块仅在视图切割当前处于活动状态时可用。
      - 增加滑块的 **Sensitivity** 以精细控制视图切割的重新定位。每个灵敏度增量将滑块的动作范围聚焦10倍，因此灵敏度设置为100使您能够在模型的1/100范围内重新定位视图切割。或者，您可以双击 **Sensitivity** 栏中的某个位置将视图切割重新定位到大致位置，然后拖动 **Position** 滑块选择一个值。选定的切割按指定值平移。
4. 要旋转 **Plane** 视图切割：
   1. 从视图切割管理器底部的菜单中选择 **Rotate**。
   2. 选择旋转轴（对于 **Key-in** 方法定义的切割为2轴或3轴，或对于 **From CSYS** 方法定义的切割为两个非法线轴中的任意一个；请参见["创建或编辑视图切割，" 第80.2.1节](pt07ch80s02hlb01.md)）；并输入 **Angle** 的旋转值，或使用滑块选择值。选定的切割绕旋转轴的原点按指定角度旋转。应用于视图切割的任何平移都会被忽略。
5. 要重新定位 **Cylinder**、**Sphere** 或 **Isosurface** 视图切割，请在可用文本字段中输入值，或使用滑块选择值。该滑块仅在视图切割当前处于活动状态时可用；您可以增加滑块的灵敏度以获得更高的精度。
   - 对于 **Cylinder** 切割，请输入圆柱 **Radius** 的值。
   - 对于 **Sphere** 切割，请输入球体 **Radius** 的值。
   - 对于 **Isosurface** 切割，请输入切割应对应的轮廓 **Value**。
   选定的切割按指定方式重新定位。
