# 73.2.2 部件集和装配体集有何不同？







您从部件创建的集或从装配体中的部件实例创建的集的使用方式不同：

**部件集**

部件集是您在部件相关模块（Part或Property）中创建的集。Mesh模块在从上下文栏中选择**Part**对象时也充当部件相关模块。当您从输出数据库导入孤立网格时，您还会自动将任何集作为部件集导入。部件集显示在模型树中该部件下的**Set**容器中。只有部件集在Part和Property模块中的**Set Manager**中可见。您不能直接在Part模块中使用部件集；但是，在Property模块中，您可以向部件集指定的区域分配截面。如果您向由部件集定义的区域分配截面，Abaqus会将截面分配应用于装配体中该部件的所有实例。

当您在Assembly模块中实例化部件时，您可以引用先前创建的任何部件集；但是，装配体相关模块仅提供对这些集的只读访问，您不能从装配体相关模块中的**Set Manager**访问部件集。来自实例化部件的集显示在模型树中装配体下方。您可以在操作过程中（例如，在施加载荷或边界条件时）通过单击提示区域右侧的**Set**按钮并从出现的**Region Selection**对话框中选择集来选择合格的部件集。Abaqus将集命名为*部件实例名称.集名称*。您不能从装配体相关模块中的**Set Manager**访问部件集。

**装配体集**

装配体集是您在装配体相关模块（Assembly、Step、Interaction或Load）中创建的集。Mesh模块在从上下文栏中选择**Assembly**对象时也充当装配体相关模块。装配体集显示在模型树中装配体下的**Set**容器中，以及来自实例化部件的任何集。只有装配体集在装配体相关模块中的**Set Manager**中可见。您可以使用装配体集来指示例如要施加载荷或边界条件或获取输出的装配体区域。装配体集可以包含来自多个部件实例的区域。

装配体集引用装配体本身，而不是单个部件实例。因此，如果您删除包含在集中的部件实例，Abaqus/CAE不会删除装配体集。您必须手动删除装配体集。

单个部件不能包含与现有集同名的几何集、节点集或单元集；但是，不同部件可以包含同名的集。所有装配体集名称必须唯一。
