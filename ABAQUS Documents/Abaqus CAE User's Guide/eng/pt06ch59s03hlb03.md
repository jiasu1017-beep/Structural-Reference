# 59.7.3 Editing attachment lines created by projecting points







You can edit attachment lines created by projecting points.

**To edit attachment lines created by projecting points:**

1. From the main menu bar, select ****Feature**![](../graphics/images/arrow.gif)**Edit****. **Tip:**You can also edit a feature by clicking the ![](../graphics/ico_featureEdit.png) tool located in the module toolbox.
2. Select the attachment line feature to modify. You can select the feature either directly from the current viewport or from the Model Tree. The **Edit Attachment Lines Feature** dialog box appears.
3. Select the method for determining the length of the projection vector. - Choose **Maximum projections along direction**, and enter the maximum number of target faces to be intersected by the attachment lines. - Choose **Maximum length of projection vector**, and enter the maximum length of the attachment lines.
4. If desired, toggle **Regenerate on OK** to control the model regeneration when the edit procedure is finished.
