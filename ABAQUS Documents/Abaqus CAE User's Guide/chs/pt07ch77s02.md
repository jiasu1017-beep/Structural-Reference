# 77.2 更改初始颜色





Abaqus/CAE 通过对当前视口中的可视图几何和网格单元应用 **初始颜色** 来开始颜色编码过程。默认情况下，此初始颜色为灰色，但您可以通过从 **初始颜色** 字段中选择以下选项之一来自定义初始颜色：
- 您可以选择当前颜色，这会提示 Abaqus/CAE 保留当前在视口中显示的颜色。选择此选项不会更改任何颜色；因此，当您不想更改已在视口中应用的任何几何或网格单元颜色时，此选项最有用。
- 您可以将初始颜色设置为显示所选模块的默认颜色映射。此选项对于将各个选择重置为默认颜色很有用，而不从视口中移除所有颜色更改。例如，当您在装配模块中选择此选项时，Abaqus/CAE 根据装配模块默认设置对组件进行颜色编码：**显示体**、**几何**、**原生网格** 和 **孤儿网格**。
- 您可以选择自定义颜色作为初始颜色。

如果您在任何非可视化模块之外的模块中工作，您可以使用 **颜色编码** 工具栏中的半透明度工具使初始颜色半透明，并选择半透明度级别。有关更多信息，请参见["更改透明度，" 第77.3节](pt07ch77s03.md)。

**注意：**在可视化模块中，您从 **通用绘图选项** 或 **叠加绘图选项** 对话框的 **其他** 页面控制透明度。请参见["自定义渲染样式、透明度和填充颜色，" 第55.2节](pt05ch55s02.md)，了解更多信息。

**更改初始颜色：**

1. 点击 **颜色编码** 工具栏中的 ![](../graphics/ico_colorCode.png) 工具。Abaqus/CAE 显示 **颜色编码** 对话框。
2. 点击并按住 **初始颜色** 样本，然后从出现的列表中选择以下选项之一： - 选择等号 (`=`) 以选择当前颜色。 - 选择星号 (`*`) 以选择模块的默认颜色映射。 - 选择颜色样本 (![](../graphics/usi-colorsample.png)) 以选择自定义颜色。选择颜色样本符号会打开 **选择颜色** 对话框，供您选择新颜色。（有关更多信息，请参见["自定义颜色，" 第3.2.9节](pt01ch03s02s09.md)。）
3. 点击 **应用**。Abaqus/CAE 在当前视口中显示新的初始颜色选择。
