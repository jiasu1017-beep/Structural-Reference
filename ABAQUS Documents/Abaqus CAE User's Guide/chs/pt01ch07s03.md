# 7.2 使用显示列表





显示列表帮助你更快地显示重复图像。当启用显示列表时，每个绘图操作都被记录在一个列表中，必要时可以快速重放。这在大多数系统上会导致更快的图像刷新，但需要图形内存来记录每个绘图操作。显示列表在可视化模块（Abaqus/Viewer）中根本不使用。在 Abaqus/CAE 中，显示列表通常默认是启用的；如果你在显示非常大的模型时遇到内存问题或图形性能下降，应该禁用它们。

**要控制显示列表：**

1. 找到显示列表选项。从主菜单栏中，选择****View**![](../graphics/images/arrow.gif)**Graphics Options****。Abaqus/CAE 显示**图形选项**对话框。
2. 从**硬件**选项中，切换**使用显示列表**开启或关闭，以控制图像显示列表的使用。此选项在可视化模块中没有影响。当**使用显示列表**开启时，你可能会在首次绘制图像时注意到短暂延迟；这是因为 Abaqus/CAE 必须构建显示列表。后续图像绘制会更快。
3. 单击**确定**实施更改并关闭对话框。你的更改会在会话期间保存。
