# 12.15.4 分配材料方向或钢筋参考方向







您可以分配材料方向或钢筋参考方向。以下章节描述如何分配这些方向：
- "分配材料方向"
- "分配钢筋参考方向"

### 分配材料方向

全局坐标系决定默认的材料方向。但是，您可以通过选择基准坐标系或离散场或通过定义离散方向来为壳、实体零件、区域、皮肤或筋条分配特定的材料方向。对于Abaqus/Standard分析，您可以在用户子程序[`ORIENT`](../sub/sub-link.md#sub-xsl-orient)中定义局部坐标系。基准坐标系可以是矩形、圆柱形或球形的；如需要，您可以应用额外的旋转到坐标系。请参见"方向"，Abaqus分析用户指南第2.2.5节。

在Abaqus/CAE尝试为系统奇点处的材料方向绘制坐标系的情况下（即，对于用户选择的坐标系和几何或单元的法线无法解析为Abaqus/CAE显示目的的有效方向的情况），坐标系将被绘制为折叠的。

对于连续壳单元，Abaqus/CAE不会将显示的材料方向投影到中面表面上。您可以执行数据检查并在可视化模块中查看输出数据库以验证材料方向。请参见"对模型执行数据检查"，第19.7.3节。

您分配给零件或区域的材料方向会自动分配给该零件在装配中的所有实例。您可以使用查询工具集来验证您已为选定区域分配了正确的材料方向。请参见"使用查询工具集获取分配信息"，第12.19节。

您可以通过在模型树中右键单击材料方向并从出现的列表中选择抑制或恢复选项来编辑、抑制和恢复先前定义的材料方向。请参见"抑制和恢复对象"，第3.4.3节。

**为零件分配材料方向：**

1. 如果您要分配材料方向的零件在当前视口中不可见，请单击上下文栏中**零件**列表中所需零件的名称。所选零件出现在当前视口中。
2. 从主菜单栏，选择**分配**![](../graphics/images/arrow.gif)**材料方向**。**提示：**您也可以单击属性模块工具箱中的![](../graphics/ico_assignMaterialOrientation.png)工具。
3. 从视口中选择壳或实体区域，然后单击鼠标按钮2表示已完成选择。（有关更多信息，请参见第6章"在视口中选择对象"。）**提示：**您可以使用**选择**工具栏中的工具限制可以在视口中选择的对象类型。请参见"使用选择选项"，第6.3节。如果您选择**皮肤**作为对象类型，可以为整个皮肤或其一个或多个面分配材料方向。从提示区域，选择**（选择整个皮肤）**或**（选择部分皮肤）**，然后在视口中进行选择。如果所选零件有多个皮肤，Abaqus/CAE在您进行选择时会在提示区域显示模糊选择选项。如果您选择**筋条**作为对象类型，可以为整个筋条或其一条或多条边缘分配材料方向。从提示区域，选择**（选择整个筋条）**或**（选择部分筋条）**，然后在视口中进行选择。如果所选零件有多条筋条，Abaqus/CAE在您进行选择时会在提示区域显示模糊选择选项。如果您希望从现有集合列表中进行选择，请执行以下操作：1. 单击提示区域右侧的**集合**。Abaqus/CAE显示包含可用零件集合列表的**区域选择**对话框。仅包含壳或实体的零件集合出现在列表中。2. 选择感兴趣的零件集合，然后单击**继续**。**注意：**默认选择方法基于您最近使用的选择方法。要切换回另一种方法，请单击提示区域右侧的**在视口中选择**或**集合**。
4. 如果您想使用现有基准坐标系指定材料方向，请执行以下操作：1. 选择基准坐标系。- 在视口中选择一个坐标系。- 按名称选择基准坐标系。单击提示区域中的**基准坐标系列表**，从列表中选择一个名称，然后单击**确定**。Abaqus/CAE显示为该区域指定的材料方向，**编辑材料方向**对话框出现。2. 选择**轴1**、**轴2**或**轴3**以指定坐标系轴。- 对于壳区域，在对话框的**法线方向**部分指定近似壳法线的坐标系轴。- 对于实体区域，在对话框的**额外旋转方向**部分指定发生可选额外旋转的坐标系轴。3. 如有需要，您可以使用以下方法之一提供关于壳法线（对于壳区域）或基准坐标系的局部轴（对于实体区域）的材料方向的额外旋转：- 选择**角度**，然后输入额外旋转的度数。- 选择**分布**，然后选择定义在整个区域上空间变化角度的现有标量离散场。或者，您可以单击![](../graphics/ico_discreteField.png)创建新的离散场。（请参见第63章"离散场工具集"获取更多信息。）Abaqus/CAE应用额外旋转（如果有），并显示应用于该区域的结果材料方向。
5. 如果您想使用全局坐标系（默认）或在离散方向、用户子程序或离散场中指定材料方向，请从提示区域单击**使用默认方向或其他方法**。您也可以使用此过程选择现有基准坐标系或创建新的基准坐标系。1. 在出现的**编辑材料方向**对话框中，单击**定义**字段右侧的箭头，然后选择以下选项之一：- 选择**坐标系**，然后单击![](../graphics/ico_selectBlue.png)从视口或按名称选择坐标系。或者，您可以单击![](../graphics/ico_dtmCsys.png)创建新的基准坐标系。- 选择**全局**以使用全局坐标系或将先前分配的材料方向重置为全局坐标系。- 选择**离散**，然后单击![](../graphics/ico_edit.png)定义离散方向。在出现的**编辑离散方向**对话框中，使用"使用离散方向进行材料方向和复合铺层方向"，第12.16节中描述的过程定义法线轴和主轴。- 选择**用户定义**以在用户子程序[`ORIENT`](../sub/sub-link.md#sub-xsl-orient)中定义方向。此选项仅对Abaqus/Standard分析有效。请参见以下章节获取更多信息：- "指定常规作业设置"，第19.8.6节- "ORIENT"，Abaqus用户子程序参考指南第1.1.15节- 选择离散场以定义空间变化的方向。或者，您可以单击**定义**字段右侧的![](../graphics/ico_discreteField.png)创建新的离散场。（请参见第63章"离散场工具集"获取更多信息。）2. 选择**轴1**、**轴2**或**轴3**以指定坐标系轴。- 对于壳区域，在对话框的**法线方向**部分指定近似壳法线的坐标系轴。- 对于实体区域，在对话框的**额外旋转方向**部分指定发生可选额外旋转的坐标系轴。3. 如有需要，您可以使用以下方法之一提供关于壳法线（对于壳区域）或基准坐标系的局部轴（对于实体区域）的材料方向的额外旋转：- 选择**角度**，然后输入额外旋转的度数。- 选择**分布**，然后选择定义在整个区域上空间变化角度的现有标量离散场。或者，您可以单击![](../graphics/ico_discreteField.png)创建新的离散场。（请参见第63章"离散场工具集"获取更多信息。）Abaqus/CAE应用额外旋转（如果有），并显示应用于该区域的结果材料方向。4. 选择以下选项之一以指定连续壳单元或实体单元的堆叠方向：- **单元等参方向1**- **单元等参方向2**- **单元等参方向3（从底到顶）**- **材料方向法线（仅限连续壳）**。堆叠方向是材料方向的3方向。如果所选区域已网格化，Abaqus/CAE会显示每个单元的材料堆叠方向。
6. 在对话框中单击**确定**。

### 分配钢筋参考方向

钢筋层的角度方向是相对于钢筋参考方向定义的。全局坐标系决定默认的钢筋参考方向。但是，您可以通过从视口中选择现有基准坐标系然后选择近似壳法线方向的基准坐标系轴来为壳零件、区域、皮肤或筋条分配钢筋参考方向。基准坐标系可以是矩形、圆柱形或球形的；如需要，您可以应用额外的旋转到坐标系。请参见"方向"，Abaqus分析用户指南第2.2.5节。

您分配给零件或区域的钢筋参考方向会自动分配给该零件在装配中的所有实例。您可以使用查询工具集来验证您已为选定区域分配了正确的钢筋参考方向。请参见"使用查询工具集获取分配信息"，第12.19节。

**为零件分配钢筋参考方向：**

1. 如果您要分配钢筋参考方向的零件在当前视口中不可见，请单击上下文栏中**零件**列表中所需零件的名称。所选零件出现在当前视口中。
2. 从主菜单栏，选择**分配**![](../graphics/images/arrow.gif)**钢筋参考方向**。**提示：**您也可以单击属性模块工具箱中的![](../graphics/ico_assignRebarOrientation.png)工具。
3. 从视口中选择区域，然后单击鼠标按钮2表示已完成选择。（有关更多信息，请参见第6章"在视口中选择对象"。）**提示：**您可以使用**选择**工具栏中的工具限制可以在视口中选择的对象类型。请参见"使用选择选项"，第6.3节。如果您选择**皮肤**作为对象类型，可以为整个皮肤或其一个或多个面分配钢筋参考方向。从提示区域，选择**（选择整个皮肤）**或**（选择部分皮肤）**，然后在视口中进行选择。如果所选零件有多个皮肤，Abaqus/CAE在您进行选择时会在提示区域显示模糊选择选项。如果您选择**筋条**作为对象类型，可以为整个筋条或其一条或多条边缘分配钢筋参考方向。从提示区域，选择**（选择整个筋条）**或**（选择部分筋条）**，然后在视口中进行选择。如果所选零件有多条筋条，Abaqus/CAE在您进行选择时会在提示区域显示模糊选择选项。如果您希望从现有集合列表中进行选择，请执行以下操作：1. 单击提示区域右侧的**集合**。Abaqus/CAE显示包含可用集合列表的**区域选择**对话框。2. 选择感兴趣的零件集合，然后单击**继续**。**注意：**默认选择方法基于您最近使用的选择方法。要切换回另一种方法，请单击提示区域右侧的**在视口中选择**或**集合**。
4. 使用以下方法之一在所选区域中指定局部坐标系：- 在视口中选择一个坐标系。- 按名称选择基准坐标系。单击提示区域中的**基准坐标系列表**，从列表中选择一个名称，然后单击**确定**。- 单击**使用默认方向**以使用全局坐标系。Abaqus/CAE显示为该区域指定的钢筋方向，**编辑钢筋方向**对话框出现。
5. 选择**轴1**、**轴2**或**轴3**以指定坐标系轴。- 对于壳区域，在对话框的**法线方向**部分指定近似壳法线的坐标系轴。- 对于实体区域，在对话框的**额外旋转方向**部分指定发生可选额外旋转的坐标系轴。
6. 如有需要，您可以提供关于壳法线（对于壳区域）或基准坐标系的局部轴（对于实体区域）的钢筋参考方向的额外旋转。选择**角度**，然后输入额外旋转的度数。Abaqus/CAE应用额外旋转（如果有），并显示应用于该区域的结果钢筋参考方向。
7. 在对话框中单击**确定**。
