# 4.1.1 NAFEMS基准测试：概述







本章定义了英国国家有限元方法与标准机构（National Agency for Finite Element Methods and Standards，简称NAFEMS）推荐的一些有限元基准测试。本文档中记录的前三类基准测试是NAFEMS出版物TNSB第3版"The Standard NAFEMS Benchmarks"（1990年10月）中描述的线弹性、温度和自由振动测试。草案文件"Selected Benchmarks for Forced Vibration"（R0016，1993年3月）和"NAFEMS Non-linear Benchmarks"（1989年10月）中的建议基准测试也包含在此。本章中的基准测试作为Abaqus验证的一部分进行测试。
