# 13.8.1 在装配模块中使用基准几何





在装配模块中，您使用基准工具集来提供装配本身未提供的额外参考几何（顶点、边缘和曲面）。您使用参考几何来帮助您定义位置约束并定位零件或模型实例。例如，在创建平行面或面对面约束时，如果所需的曲面不存在，您可以使用基准平面。同样，在创建平行边缘或边缘对边缘约束时，如果所需的边缘不存在，您可以使用基准轴。基准是其中选择它的任何约束的父特征。基准不会修改零件或模型实例的几何形状；因此，您可以创建引用独立和依赖零件实例的基准。

当您在零件模块中创建基准几何时，它会在装配模块中创建零件实例时随零件几何的其余部分一起传递。此外，当您在装配模块中平移和旋转零件实例时，在零件模块中创建的基准会随实例一起平移和旋转。相比之下，在装配模块中创建的基准仅遵循用于创建基准的参考点。因此，如果您平移和旋转零件实例，基准的行为可能不会反映您的设计意图。如果您知道基准应该与零件关联，则应在零件模块中创建基准。

[图13-22](pt03ch13s08s01.md#asm-tools-datum-part)展示了一个模型，其中可变形的弯曲壳将在两个刚性表面之间被压缩。

**图13-22** 应用于基准轴和选定边缘之间的边缘对边缘约束。

![](../graphics/asm-tools-datum-part-nls.png)

通过在下刚性表面（固定零件实例）的选定边缘与壳体关联的基准轴（可动零件实例）之间应用边缘对边缘位置约束，可以轻松定位壳体。基准轴是在零件模块中与可变形零件一起创建的，在应用位置约束时会随可动零件实例一起移动。相比之下，[图13-23](pt03ch13s08s01.md#asm-tools-datum-assembly)展示了一个边缘对边缘位置约束，它应用于三个可动零件实例和一个提供参考几何的固定基准轴之间。在此示例中，基准轴是沿装配的*X*轴创建的，不与任何零件实例关联。对显示的三个零件实例中的每一个应用三个边缘对边缘位置约束，将导致三个实例沿基准轴对齐。

**图13-23** 应用于多个零件和固定基准轴之间的边缘对边缘约束。

![](../graphics/asm-tools-datum-assembly-nls.png)

基准是装配的特征，会随装配的其余部分一起重新生成。您可以通过从主菜单栏选择****View**![](../graphics/images/arrow.gif)**Assembly Display Options****使基准几何不可见，同时仍将其保留在装配中。更多信息，请参见["控制基准显示，"第76.9节](pt07ch76hla08.md)。

指示全局坐标系原点方向的三角轴是装配模块创建的基准坐标系。您可以抑制或删除它，但不能修改它。
