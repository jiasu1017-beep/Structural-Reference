# 64.9.2 启用撤销和管理缓存







启用撤消是全局性的和持久性的；启用此设置后，您可以在会话期间撤销任何部件或装配网格的网格编辑更改。该设置在下一个 Abaqus/CAE 会话中也会保持启用。禁用撤销会释放与此设置关联的任何内存，并导致任何部件或装配上的现有撤销或重做缓存丢失。

**启用撤销并设置缓存大小：**

1. 进入 Mesh（网格）模块。
2. 从主菜单栏中选择 ****Mesh**![](../graphics/images/arrow.gif)**Edit****（网格→编辑）。Abaqus/CAE 显示 **Edit Mesh**（编辑网格）对话框。**提示：**您也可以使用位于 Mesh 模块工具箱底部的 ![](../graphics/ico_meshEdit.png) 工具来显示 **Edit Mesh**（编辑网格）对话框。
3. 点击 **Settings**（设置）。出现 **Mesh Edit Undo Settings**（网格编辑撤销设置）对话框。
4. 切换开启 **Enable Undo**（启用撤销）以启用网格编辑操作的撤销。
5. 在 **Max total cache size**（最大总缓存大小）字段中输入以百万单元为单位的值，然后点击 **OK**（确定）。
