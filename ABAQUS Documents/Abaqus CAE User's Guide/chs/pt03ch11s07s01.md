# 11.7.1 刚体零件





当您的模型包含彼此接触的零件时，您可以指定其中一个或多个零件为刚体。刚体零件表示比模型其余部分 stiffness 大得多的零件，其变形可以忽略不计。

与定义为刚性的零件相反，您定义为可变形的零件可以在与刚性零件或另一个可变形零件接触时变形。例如，金属冲压过程模型可能使用可变形零件来建模坯料，使用刚性零件来建模模具和冲头，如图所示。

**图11-9** 刚体和可变形零件。

![](../graphics/prt-conc-rigid-nls.png)

在此示例中，模具被约束为没有运动，冲头在冲压过程中通过预定路径移动。您通过选择刚体参考点并约束或指定其运动来控制刚性零件的运动。更多信息，请参见"参考点"，第11.8.1节。

计算效率是刚性零件相对于可变形零件的主要优势。在分析过程中，不对刚性零件执行单元级计算。虽然需要一些计算工作来更新刚体运动并组装集中载荷和分布载荷，但刚体运动完全由参考点决定。要将零件类型从可变形更改为刚性，反之亦然，您可以右键单击模型树中的零件并从出现的菜单中选择**编辑**。更多信息，请参见"刚体零件和刚体约束有什么区别？"，第11.7.3节，以及第27章"显示体"。

您可以在两种刚性零件之间进行选择：

**离散刚体零件**

您声明为离散刚体零件的零件可以是任意三维、二维或轴对称形状。因此，您可以使用所有零件模块特征工具——实体、壳、线框、切割和混合——来创建离散刚体零件。但是，只有包含壳和线框的离散刚体零件才能在网格模块中用刚性单元网格划分。如果尝试在装配模块中创建实体离散刚体零件的实例，Abaqus/CAE会显示错误消息；您必须返回零件模块并将实体的面转换为壳。

**分析刚体零件**

分析刚体零件与离散刚体零件类似，因为它用于在接触分析中表示刚性零件。如果可能，在描述刚性零件时应使用分析刚体零件，因为它比离散刚体零件的计算成本更低。分析刚体零件的形状不是任意的，轮廓必须平滑。您只能使用以下方法来创建分析刚体零件：
- 您可以绘制零件的二维轮廓，并绕对称轴旋转轮廓以形成三维旋转分析刚体零件。- 您可以绘制零件的二维轮廓，并无限拉伸轮廓以形成三维拉伸分析刚体零件。虽然Abaqus/CAE认为拉伸是无限延伸的，但零件模块显示具有您指定的深度的三维拉伸分析刚体零件。- 您可以绘制平面二维分析刚体零件的轮廓。- 您可以绘制轴对称二维分析刚体零件的轮廓。

您可以从包含第三方格式几何的文件导入零件并将其定义为可变形或离散刚体零件；但是，您不能将导入的零件定义为分析刚体零件。作为替代，您可以将分析刚体零件的几何导入草图。然后您可以创建新的分析刚体零件并将导入的草图复制到草图工具集。

Abaqus/CAE中的刚性零件等效于Abaqus/Standard或Abaqus/Explicit分析中的刚性表面。更多信息，请参见以下内容：
