# 19.7.6 终止分析作业







您可以使用以下方法之一来终止分析作业：
- 从主菜单栏中，选择****作业**![](../graphics/images/arrow.gif)**终止**![](../graphics/images/arrow.gif)***您选择的作业*****。
- 在**作业管理器**中，选择作业名称，然后点击**终止**。
- 在该特定作业的作业监视器对话框中，点击**终止**。（有关更多信息，请参见["监视分析作业的进度，"第 19.2.6 节](pt03ch19s02hlb01.md)。）

Abaqus/CAE 会要求您确认，终止作业，并将**作业管理器**中的作业状态更新为`已终止`。
