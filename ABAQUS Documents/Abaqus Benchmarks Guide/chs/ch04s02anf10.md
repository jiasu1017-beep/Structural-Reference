# 4.2.10 LE10：压力作用下的厚板







**产品：** Abaqus/Standard  Abaqus/Explicit  

### 测试单元

C3D20    C3D20R    C3D10    C3D10I    C3D10M  

### 问题描述

![](../graphics/exxle10-model.png)

**模型：**

均匀压力作用下的厚板。

**网格：**

分别测试粗网格和细网格。

**材料：**

线弹性，弹性模量 = 210 GPa，泊松比 = 0.3，密度 = 7800 kg/m3。

**边界条件：**

在面DCD′C′上![](../graphics/bmk_eqn01052.gif) = 0。在面ABA′B′上![](../graphics/bmk_eqn01053.gif) = 0。在面BCB′C′上![](../graphics/bmk_eqn01703.gif) = 0。在线EE′上![](../graphics/bmk_eqn01705.gif) = 0（E是边缘CC′的中点；E′是边缘BB′的中点）。

**载荷：**

板上表面施加1.0 MPa的均匀法向压力。

### 参考解

这是英国国家有限元方法与标准机构（NAFEMS）推荐的测试：NAFEMS出版物TNSB第3版"The Standard NAFEMS Benchmarks"（1990年10月）中的测试LE10。

目标解：D点处的直接应力![](../graphics/bmk_eqn01710.gif) = 5.38 MPa。

### 结果与讨论

Abaqus/Standard结果如表4.2.10-1所示。括号中的值是相对于参考解的百分比差异。

**表4.2.10-1** Abaqus/Standard分析。

| 单元 | ![](../graphics/bmk_eqn01434.gif)，粗网格 | ![](../graphics/bmk_eqn01434.gif)，细网格 |
| --- | --- | --- |
| C3D20 | 6.72 MPa (+25.00%) | 5.64 MPa (+4.83%) |
| C3D20R | 7.93 MPa (+47.39%) | 5.53 MPa (+2.78%) |
| C3D10 | 5.44 MPa (+1.15%) | 5.77 MPa (+7.24%) |
| C3D10I | --5.08 MPa (3.72%) | --5.51 MPa (+2.42%) |
| C3D10M | 5.57 MPa (+3.53%) | 5.89 MPa (+9.48%) |

C3D10和C3D10M单元在粗网格下比细网格下更准确：在粗网格中，四个单元在感兴趣的点处汇合，在平均到节点后得到更准确的结果。在更细化的网格中，只有一个单元包含感兴趣的点；因此，外推到节点的结果不太准确。

与Abaqus/Standard不同，Abaqus/Explicit没有将积分点输出（如应力）外推到节点的选项。因此，除非对颜色等值线图进行粗略解释，否则无法提取D点处的所需应力分量。作为替代方案，将Abaqus/Standard模拟和Abaqus/Explicit模拟之间D点附近积分点处的![](../graphics/bmk_eqn01434.gif)值进行比较。

在Abaqus/Explicit分析中，压力在0.4秒的时间内从零平稳地增加到最终值1.0 MPa，这足够慢，可以视为准静态（惯性效应起的作用很小）。

| 分析类型 | ![](../graphics/bmk_eqn01434.gif)，粗网格 | ![](../graphics/bmk_eqn01434.gif)，细网格 |
| --- | --- | --- |
| Abaqus/Standard | 3.70 MPa | 4.61 MPa |
| Abaqus/Explicit | 3.79 MPa | 4.55 MPa |

对于粗网格，比较点位于单元18、积分点3。对于细网格，比较点位于单元199、积分点1。两者都是实际角点D的近邻。
