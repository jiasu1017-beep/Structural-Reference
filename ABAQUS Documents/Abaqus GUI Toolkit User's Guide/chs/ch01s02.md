# 1.4 获取帮助

您可能希望在教程的不同阶段阅读有关Abaqus/Viewer功能的更多信息。上下文相关帮助系统允许您快速轻松地定位相关信息。主窗口和所有对话框中的每个项目都有上下文相关帮助。

**注意：**
- 在Windows平台上，帮助系统使用您的默认Web浏览器显示在线文档。
- 在UNIX和Linux平台上，帮助系统搜索系统路径以查找Firefox。如果帮助系统找不到Firefox，则会显示错误。可以在环境文件中设置**browser_type**和**browser_path**变量来修改此行为。有关更多信息，请参见["系统自定义参数"，《Abaqus安装和许可指南》第4.1.4节](../sgb/sgb-link.md#sgb-chp-env-sys)。

**获取上下文相关帮助：**

1. 从主菜单栏中，选择****帮助**![](../graphics/images/arrow.gif)**关于上下文****。 **提示：**您也可以单击帮助工具![](../graphics/ico_help.png)访问上下文相关帮助。光标会变为问号。
2. 单击主窗口中除其框架之外的任何部分。帮助窗口会出现在您的浏览器窗口中。帮助窗口显示您所选项目的信息。
3. 滚动到帮助窗口底部。在窗口底部，会出现蓝色带下划线的项目列表。这些项目是指向《Abaqus/CAE用户指南》的链接，其中包含所有Abaqus/Viewer帮助主题。
4. 单击其中任何一个项目。您的默认Web浏览器中会出现一个书籍窗口。该窗口分为四个框架，如下所示：- 右侧的文本框架中显示《Abaqus/CAE用户指南》](../usi/usi-link.md#usi)。指南会翻到您所选的项目。- 左下方有一个可展开的目录表，便于在整个书籍中导航。- 左上方的目录控制工具允许您更改目录表中显示的详细程度，或更改框架的大小。单击![](../graphics/hhp-toc-expand.png)可展开在线书籍目录中的多个级别。单击![](../graphics/hhp-toc-collapse.png)可折叠所有展开的部分。分别单击![](../graphics/hhp-toc-wide.gif)和![](../graphics/hhp-toc-narrow.gif)加宽或收窄目录表框架。- 书籍窗口顶部的导航框架允许您从整个Abaqus文档集合中选择另一本书。导航框架还允许您搜索整个指南。
5. 单击目录表中的任何项目。右侧的文本框架会变为您所选的内容。
6. 单击主题标题左侧的![](../graphics/collapsd.gif)图标以展开它。副标题会出现在主题标题下方，符号变为![](../graphics/expanded.gif)，表示该部分已展开。如果副标题旁边出现![](../graphics/leaf.gif)，则该小节没有更多级别可展开。要折叠目录表的展开部分，请单击主题标题旁边的![](../graphics/expanded.gif)。
7. 在导航框架的搜索面板中，键入文本框架右侧出现的任何单词，然后单击**搜索**。搜索完成后，目录表框架会在每个主题标题旁边显示命中次数，所有命中会在文本框架中突出显示。单击导航框架中的**下一个匹配**或**上一个匹配**可在文档中从一个命中移动到下一个。您可以在搜索面板中输入单个单词或短语，并且可以使用[*]字符作为通配符。有关使用在线文档搜索功能的详细说明，请参见[使用Abaqus在线文档](../hhp/hhp-link.md#hhp)。
8. 关闭Web浏览器窗口。
