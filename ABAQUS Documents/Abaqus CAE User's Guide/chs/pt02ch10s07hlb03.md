# 10.7.3 导入装配





从主菜单栏选择****文件（File）**![](../graphics/images/arrow.gif)**导入（Import）**![](../graphics/images/arrow.gif)**装配（Assembly）****从以下格式导入装配：
- 装配文件
- CATIA V4
- 由Pro/ENGINEER、CATIA V5或CATIA V6生成的Elysium中性文件
- Parasolid

有关Abaqus/CAE支持的CAD软件版本的信息，请参阅Dassault Systmes知识库，网址为[www.3ds.com/support/knowledge-base](http://www.3ds.com/support/knowledge-base)。Abaqus/CAE不支持子装配。因此，导入文件中装配和子装配在Abaqus/CAE中显示为单个装配。

您可以导入整个装配，也可以仅导入选定的零件实例。Abaqus/CAE会将您的选择追加到现有装配中，并保留实例的原始位置。

**要导入装配：**

1. 从主菜单栏，选择****文件（File）**![](../graphics/images/arrow.gif)**导入（Import）**![](../graphics/images/arrow.gif)**装配（Assembly）****。出现**导入装配（Import Assembly）**对话框。
2. 从**导入装配**对话框底部的**文件过滤器（File Filter）**菜单中，选择以下之一：- **装配文件 (*.eaf)**；更多信息，请参阅["从装配文件导入装配，"第10.7.14节"](pt02ch10s07hlb14.md)。- **CATIA V4 (*.model, *.catdata, *.exp)**；更多信息，请参阅["从CATIA V4格式文件导入装配，"第10.7.15节"](pt02ch10s07hlb15.md)。- **Parasolid (*.x_t,*.x_b, *.xmt)**；更多信息，请参阅["从Parasolid格式文件导入装配，"第10.7.17节"](pt02ch10s07hlb17.md)。- **ProE/NX/CATIA V5 Elysium中性 (*.enf)**；更多信息，请参阅["从Elysium中性文件导入装配，"第10.7.16节"](pt02ch10s07hlb16.md)。
3. Abaqus/CAE会列出所选目录中具有适当文件扩展名的所有文件。选择包含要导入的装配的文件，然后单击**确定（OK）**。Abaqus/CAE进入"装配"模块，并显示**从*格式*文件导入装配（Import Assembly from *format* File）**对话框。
4. 执行以下任一操作：- 单击装配名称导入装配中的所有零件实例。- 单击装配名称左侧的箭头显示装配中的所有零件实例。单击要导入的零件实例的名称。
5. 单击**确定（OK）**导入装配或选定的零件实例，并退出**从*格式*文件导入装配**对话框。Abaqus/CAE会创建零件实例和对应的零件。Abaqus/CAE会在消息区域显示消息，指示任何零件实例是否无效或不精确。如果需要，使用查询工具集定位无效和不精确几何的区域。更多信息，请参阅["在零件模块中使用查询工具集，"第11.16.4节"](pt03ch11s16s04.md)。
