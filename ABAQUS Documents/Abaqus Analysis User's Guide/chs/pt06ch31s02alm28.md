### 概述

弹簧类弹性连接器行为：
- 可在任何具有可用相对运动分量的连接器中定义；
- 可独立为每个可用相对运动分量指定，在这种情况下行为可以是线性或非线性的；
- 可指定为依赖于多个局部方向上的相对位置或本构运动；以及
- 可指定为所有可用相对运动分量的耦合线性弹性行为。

或者，可在任何可用相对运动分量中使用自动选择的刚性弹簧来指定刚性类行为。

力和力矩作用的方向以及位移和旋转的测量方向由每个连接类型的局部方向决定，如["连接类型库，" 31.1.5节](pt06ch31s01aus114.md)中所述。

### 示例

[图31.2.2-1](pt06ch31s02alm28.md#econnectorbehavior-shock-elast)中的组合连接器有两个可用的相对运动分量：沿1方向的相对位移（来自SLOT连接）和绕1方向的旋转（来自REVOLUTE连接）——见["连接类型库，" 31.1.5节](pt06ch31s01aus114.md)。因此，连接器相对运动分量1和4可用于指定连接器行为。

**图31.2.2-1** 减震器的简化连接器模型。

![](../graphics/econnector-shock-nls.png)

要定义非线性扭簧以抵抗顶部和底部连接点之间绕局部1方向的相对旋转，使用以下输入：
```
[*CONNECTOR SECTION](../key/key-link.md#usb-kws-mconnectorsection), ELSET=shock, BEHAVIOR=sbehavior

[*CONNECTOR BEHAVIOR](../key/key-link.md#usb-kws-mconnectorbehavior), NAME=sbehavior

-900., -0.7
   0.,  0.0
1250.,  0.7
```

虽然假定两个可用相对运动分量之间不会发生弹性耦合，但您可以用耦合线性弹性行为替换非线性力矩-旋转数据，以定义绕减震轴线的旋转刚度与轴向位移的耦合。

### 定义刚性连接器行为

刚性类弹性连接器行为可用于使原本可用的相对运动分量变为刚性。考虑一个没有固有运动约束的CARTESIAN连接器。如果在局部2和3方向指定刚性行为，连接器将类似于SLOT连接器。

使用具有可用相对运动分量的连接器（其中指定了刚性行为）而不是具有固有运动约束的连接器的技术特别有用，当您需要：
- 在具有可用相对运动分量的连接器中自定义受约束分量；例如，您可以在CARTESIAN连接器中约束局部1和2方向，以在3方向定义类似SLOT的连接器；
- 定义刚性塑性行为（参见["连接器塑性行为，" 31.2.6节](pt06ch31s02alm32.md)）；或
- 定义刚性损伤行为（参见["连接器损伤行为，" 31.2.7节](pt06ch31s02alm33.md)）。

例如，如果您使用SLOT连接器，则无法在固有受约束的2和3方向指定塑性和损伤行为。为解决此问题，您可以使用具有分量2和3刚性行为的CARTESIAN连接器（如上所述），然后在这些分量中定义刚性塑性（和/或损伤）。参见["连接器塑性行为，" 31.2.6节](pt06ch31s02alm32.md)中的示例进行说明。

在Abaqus/Standard中，如果在活动连接器止挡、连接器锁或指定的连接器运动相同的局部方向上定义刚性分量，可能会出现过约束。

#### 强制执行刚性类弹性行为

特定分量中的刚性类弹性行为通过在该分量中使用刚性线性弹性弹簧来强制执行。弹簧的刚度自动选择，取决于连接器使用的环境。在Abaqus/Standard中，刚度取为连接器单元所附着的周围单元平均刚度的10倍。如果无法计算平均刚度（如连接器单元未连接到其他单元或连接到刚体的情况），则使用刚度![](../graphics/usb_eqn05172.gif)。在Abaqus/Explicit中，首先通过考虑连接器单元节点的平均质量和分析中的稳定时间增量来计算Courant刚度。在大多数情况下，Courant刚度然后用于根据取决于建模环境和分析精度（单精度或双精度）的启发式方法计算刚性类弹性行为的值。例如，如果在连接器中定义了塑性，则在塑性定义涉及的分量中，刚性类弹性刚度不超过初始屈服值的千分之一。如果没有定义塑性，则刚性刚度计算为Courant刚度的倍数。

在大多数情况下，计算刚性刚度中使用的启发式方法产生的刚度值是足够的。如果此刚度不能满足您的应用需求，您可以始终通过直接指定线性刚度值来自定义弹性刚度。

由于Abaqus/Standard和Abaqus/Explicit中用于刚性类弹性行为的刚度值不同，当此类模型从一个求解器导入另一个求解器时，您可能会注意到行为的不连续性。

### 在线形摄动过程中定义弹性连接器行为

具有连接器弹性的可用相对运动分量使用来自基态的线性化弹性刚度。在直接解稳态动态和基于子空间的稳态动态分析中，由非耦合连接器弹性行为定义的线性弹性刚度可能是频率相关的。
