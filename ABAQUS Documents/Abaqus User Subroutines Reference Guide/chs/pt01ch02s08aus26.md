### 概述

分布：
- 是在Abaqus模型中单元或节点上定义的空间变化场；
- 可用于在单元基础上定义壳厚度；
- 可用于在单元基础上定义壳刚度；
- 可用于在单元基础上在实体连续体和壳单元上定义局部坐标系；
- 可用于在复合壳单元的层上定义方向角；
- 可用于为连接器单元定义方向角；
- 可用于在传统复合壳单元的层上定义厚度；
- 可用于指定初始接触间隙；
- 可用于在Abaqus/CFD分析中指定随穿过表面的流体总体积而变化的压力；以及
- 在Abaqus/Standard分析中，可用于为实体连续体单元定义质量密度、线性弹性材料行为和热膨胀；壳偏移；复合实体连续体单元层上的方向角；膜单元上的局部坐标系；以及在单元基础上定义膜厚度。

### 分布

分布是幅值定义的空间类比（请参阅["幅值曲线，" 第34.1.2节](pt07ch34s01aus115.md)）。幅值定义用于提供载荷、位移和其他规定变量的任意时间变化。分布用于指定选定单元属性、材料属性、局部坐标系和初始接触间隙的空间变化。

分布的两个主要组成部分是其*位置*和*场数据*。位置标识分布在哪里定义，是在单元上还是在节点上。场数据是为分布中的每个单元或节点定义的指定数量的浮点值。

要定义分布，必须为其分配一个唯一名称。还必须通过引用分布表来指定分布中每个数据值的数量和物理维度。

### 指定分布的位置

您可以在单元或节点上定义分布。当前仅支持在节点上的分布用于定义初始接触间隙，如["控制Abaqus/Explicit中通用接触的初始接触状态，" 第36.4.4节](pt09ch36s04aus158.md)中所述。对于与Abaqus/CFD中流体边界定义一起使用的分布，您指定不需要位置。所有其他分布应用都需要在单元上定义的分布。

给定单元或节点可以属于的分布数量没有限制。单元和节点不能组合在同一个分布定义中。

#### 在单元上定义分布

在单元上定义分布需要为分布定义中包含的每个单元或单元集指定场数据。所有单元上的分布都需要定义默认数据。默认数据用于未在分布中被明确赋值的所有单元。

| **输入文件用法：** | ``` [*DISTRIBUTION](../key/key-link.md#usb-kws-mdistribution), LOCATION=ELEMENT *blank space*, *field data* *element set or element number*, *field data* ``` |
| --- | --- |
|  | 默认数据通过在分布定义的第一数据行的第一个数据项的位置使用空格而不是单元编号或单元集来定义。一个分布只能定义一组默认数据。如果您仅指定默认数据，则引用该分布的所有单元都使用默认值。如果在给定分布定义中多次指定了某个单元，则使用最后给出的规范。 |

| **Abaqus/CAE用法：** | 属性、相互作用或载荷模块：****工具**![](../graphics/images/arrow.gif)**离散场**![](../graphics/images/arrow.gif)**创建****：**定义**：**单元** |
| --- | --- |

### 应用分布

除非分布被支持分布的功能按名称引用，否则分布中定义的数据在Abaqus分析中不使用，且分布仅应用于与引用功能关联的单元或节点。此外，分布定义可以在给定模型中被引用多次。下面的示例说明了这些点。

如果在Abaqus/Standard或Abaqus/Explicit分析中某个单元被声明为刚性的（请参阅["刚体定义，" 第2.4.1节](pt01ch02s04aus22.md)），则用于定义单元属性、材料属性（密度除外）或局部坐标系的任何分布都将被忽略。

### 示例

下面的简单示例说明了如何定义分布。在["空间变化的单元属性，" Abaqus验证指南第5.1.4节](../ver/ver-link.md#ver-msc-elementproperties)中可以看到使用分布的大量说明性示例问题。

#### 示例1

为壳厚度定义了一个分布，并通过SHELL THICKNESS参数应用于两个不同的壳截面定义——如上所述，如果分布`dist0`未被支持分布的功能引用，则不会使用它。有关更多详细信息，请参阅["使用在分析期间集成的壳截面定义截面行为，" 第29.6.5节](pt06ch29s06alm19.md)。分布表定义了厚度数据的数据值数量（一个）和物理维度（LENGTH）。分布`dist0`中定义的厚度仅分配给属于单元集`elset1`或`elset2`的壳单元。`dist0`第一数据行中定义的默认厚度(t0)将分配给`elset1`和`elset2`中未在`dist0`中明确分配厚度的所有单元。

```
[*DISTRIBUTION TABLE](../key/key-link.md#usb-kws-mdistributiontable), NAME=tab0
	LENGTH
[*DISTRIBUTION](../key/key-link.md#usb-kws-mdistribution), NAME=dist0, LOCATION=element, TABLE=tab0
	                     , t0
	element set or number, t1
	element set or number, t2
 …
[*SHELL SECTION](../key/key-link.md#usb-kws-mshellsection), ELSET=elset1, SHELL THICKNESS=dist0
[*SHELL SECTION](../key/key-link.md#usb-kws-mshellsection), ELSET=elset2, SHELL THICKNESS=dist0
```

#### 示例2

为空间变化的各向同性弹性材料行为定义了一个分布，并应用于材料定义（["线性弹性行为，" 第22.2.1节](pt05ch22s02abm02.md)）。然后此材料被实体截面定义引用。这很重要，因为与任何材料定义一样，除非被截面定义引用，否则由分布定义的材料不会被使用，然后它仅应用于与截面定义关联的单元。分布表定义了各向同性弹性数据的数据值数量（两个）和物理维度（MODULUS和RATIO）。其他材料行为（在本例中为塑性）也可以包含在材料定义中。分布`dist1`中的默认弹性常数(E0, ![](../graphics/usb_eqn00270.gif)0)将分配给`elset3`中未在`dist1`中明确分配弹性常数的所有单元。

```
[*DISTRIBUTION TABLE](../key/key-link.md#usb-kws-mdistributiontable), NAME=tab1
	MODULUS, RATIO
[*DISTRIBUTION](../key/key-link.md#usb-kws-mdistribution), NAME=dist1, LOCATION=element, TABLE=tab1
	                     , E0, ![](../graphics/usb_eqn00270.gif)0
	element set or number, E1, ![](../graphics/usb_eqn00270.gif)1
	element set or number, E2, ![](../graphics/usb_eqn00270.gif)2
 …
[*MATERIAL](../key/key-link.md#usb-kws-mmaterial), NAME=MAT
[*ELASTIC](../key/key-link.md#usb-kws-melastic)
	dist1
[*PLASTIC](../key/key-link.md#usb-kws-mplastic)
 …
[*SOLID SECTION](../key/key-link.md#usb-kws-msolidsection), ELSET=elset3, MATERIAL=MAT
```

#### 示例3

通过同时指定点*a*和*b*的空间变化坐标以及空间变化附加旋转角来定义空间变化局部坐标系（["方向定义，" 第2.2.5节](pt01ch02s02aus15.md)）。然后此方向被通用壳截面定义引用。这很重要，因为与任何方向定义一样，除非被截面定义引用，否则由分布定义的方向不会被使用，然后它仅应用于与截面定义关联的单元。坐标的分布表指定COORD3D两次，以指示必须为分布中的每个单元指定两个三维坐标点的数据。

```
	element set or number, aX1,aY1,aZ1,bX1,bY1,bZ1
	element set or number, aX2,aY2,aZ2,bX2,bY2,bZ2
 …
[*DISTRIBUTION TABLE](../key/key-link.md#usb-kws-mdistributiontable), NAME=tab3
	ANGLE
[*DISTRIBUTION](../key/key-link.md#usb-kws-mdistribution), NAME=dist3, LOCATION=element, TABLE=tab3
	                      , ![](../graphics/usb_eqn00123.gif)0 
 element set or number, ![](../graphics/usb_eqn00123.gif)1
 element set or number, ![](../graphics/usb_eqn00123.gif)2
 …
[*ORIENTATION](../key/key-link.md#usb-kws-morientation), NAME=ORI, DEFINITION=COORDINATES
	dist2
 3, dist3
[*SHELL GENERAL SECTION](../key/key-link.md#usb-kws-mshellgensect), ELSET=elset4, ORIENTATION=ORI
```
