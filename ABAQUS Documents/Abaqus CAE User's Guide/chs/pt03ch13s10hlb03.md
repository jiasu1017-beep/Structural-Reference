# 13.10.11 转换约束





要从所选零件或模型实例中移除所有面、边、共轴和接触约束，同时将实例留在其当前位置，请从主菜单栏选择**实例**![](../graphics/images/arrow.gif)**转换约束**。转换相当于应用单个平移和旋转到实例，使其从原始位置移动到当前位置。之前任何约束不再出现在特征列表中，无法恢复。

**要为零件或模型实例转换约束：**

1. 从主菜单栏，选择**实例**![](../graphics/images/arrow.gif)**转换约束**来将任何现有约束转换到当前位置。
2. 从装配中，选择要转换其约束的零件或模型实例。您也可以点击提示区域右侧的**实例列表**按钮，并从出现的**实例列表**对话框中选择实例。**提示：**如果您无法选择所需的实例，可以使用**选择**工具栏更改选择行为。更多信息，请参见["使用选择选项，"第6.3节](pt01ch06s03.md)。实例不会移动，但Abaqus/CAE会将任何现有约束转换到当前位置。您无法恢复原始的面、边、共轴和接触约束。
