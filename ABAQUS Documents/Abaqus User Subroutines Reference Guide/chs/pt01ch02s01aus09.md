# 2.1.4 节点处法向定义





**产品：** Abaqus/Standard  Abaqus/Explicit  

##### **参考文献**

- [*NORMAL](../key/key-link.md#usb-kws-mnormal)
- [*NODE](../key/key-link.md#usb-kws-mnode)

### 概述

法向可以在节点处定义：
- 使用用户指定的法向定义；
- 作为梁和壳单元节点定义的一部分跟随节点坐标；
- 在Abaqus/Standard中接触对使用的刚性主曲面；
- 在梁和壳单元中；
- 用于给出结构中缺陷方向的线弹簧单元；
- 用于给出单元厚度方向的垫片单元；以及
- 用于轮廓积分评估。

节点处定义的法向不影响单元面法向，单元面法向由单元连通性定义。法向不必是单位长度。

### Abaqus/Standard中的接触曲面

Abaqus/Standard中接触曲面的用户指定曲面法向仅在小滑动接触方法使用或在使用刚性元素组成主曲面的有限滑动接触方法中使用时相关。在有限滑动中使用时，接触对中变形主曲面上定义的用户指定曲面法向将被忽略。

小滑动接触公式使用主曲面上每个节点处的曲面法向来定义一个在曲面上从点到点平滑变化的法向矢量。有关如何使用每个从节点处的曲面法向构建"主平面"的详细讨论，请参阅["Abaqus/Standard中的接触公式，" 第38.1.1节](pt09ch38s01aus177.md)。

对于由刚性元素组成的主曲面，Abaqus/Standard平滑刚性元素之间任何不连续的曲面法向过渡。节点处的曲面法向用于控制曲面法向插值。有关此类主曲面平滑的详细讨论，请参阅["解析刚性曲面定义，" 第2.3.4节](pt01ch02s03aus19.md)。

要定义法向，请在全局坐标系中指定法向分量。

| **输入文件用法：** | ``` [*NORMAL](../key/key-link.md#usb-kws-mnormal), TYPE=CONTACT SURFACE ``` |
| --- | --- |

### 单元

对于梁和壳单元、线弹簧单元、垫片单元或涉及轮廓积分评估的单元，可能需要用户指定的法向。在这种情况下，请在全局坐标系中指定法向分量。

| **输入文件用法：** | ``` [*NORMAL](../key/key-link.md#usb-kws-mnormal), TYPE=ELEMENT ``` |
| --- | --- |

#### 梁和壳单元

可能需要用户指定的法向来定义壳表面交叉处或梁交叉处所需的法向方向，此时自动确定的法向可能不适合模型（参见["梁单元横截面方向，" 第29.3.4节](pt06ch29s03alm09.md)，或["定义常规壳单元的初始几何，" 第29.6.3节](pt06ch29s06alm17.md)）。

节点法向也可以作为节点定义的一部分来定义。虽然您可以为连接到节点的所有单元定义单个法向作为节点定义的一部分，但用户指定的法向定义为节点处的特定单元定义法向，从而允许您为连接到节点的每个单元定义不同的法向。用户指定的法向定义优先于作为节点定义一部分定义的法向。

| **输入文件用法：** | ``` [*NODE](../key/key-link.md#usb-kws-mnode) *在数据行的第五、第六和第七位置指定法向。* ``` |
| --- | --- |
|  | 例如，以下行定义了一些作为节点定义一部分的法向；然后使用用户指定的法向定义重新定义单元2中节点7处使用的法向： ``` [*NODE](../key/key-link.md#usb-kws-mnode) 6, 5., 5., , -0.5, .8 7, 10., 8., , -0.5, .8 9, 14., 4., , .6, .6 [*NORMAL](../key/key-link.md#usb-kws-mnormal) 2, 7, .6, .6 ``` |

#### 线弹簧单元

对于线弹簧单元，用户指定的法向可用于给出结构中缺陷的法向方向。有关这些单元的说明，请参阅["用于建模壳中部分穿透裂纹的线弹簧单元，" 第32.9.1节](pt06ch32s09alm54.md)。

#### 垫片单元

对于垫片单元，用户指定的法向可用于指定单元的厚度方向。节点厚度方向也可以作为垫片截面定义的一部分来定义。用户指定的法向定义的厚度方向优先于作为垫片截面定义一部分定义的厚度方向。有关这些单元厚度方向定义的说明，请参阅["定义垫片单元的初始几何，" 第32.6.4节](pt06ch32s06alm49.md)。

#### 轮廓积分评估

对于轮廓积分评估（["轮廓积分评估，" 第11.4.2节](pt04ch11s04aus68.md)），应在位于请求轮廓范围内的所有表面节点处指定表面法向。这些节点在数据（.dat）文件中的"轮廓积分"信息下打印输出。对于准确的轮廓积分评估，虚拟裂纹扩展方向位于以下情况下的曲面平面中非常重要：当裂纹前缘与三维实体的外表面相交时，当裂纹前缘与材料不连续表面相交时，或者当裂纹位于弯曲壳中时。如果未指定法向，Abaqus将自动计算法向。

作为节点定义一部分指定的节点法向数据除非在模型中使用用户指定的法向定义，否则不会被激活用于实体单元；仅在一个节点上包含用户指定的法向定义就足以激活作为节点定义一部分指定的节点法向数据的使用。

### 定义法向的坐标系

Abaqus模型可以以部件实例装配的形式定义（参见["定义装配，" 第2.10.1节](pt01ch02s10aus28.md)）。在部件（或部件实例）内定义的节点法向是相对于部件坐标系定义的。这些法向根据为部件的每个实例给出的定位数据进行旋转。如有需要，可以在装配级别的参考节点处定义法向。在装配级别定义的法向在全局坐标系中定义。

对于不是以部件实例装配形式定义的模型，法向在全局坐标系中定义。
