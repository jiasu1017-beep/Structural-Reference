# 41.3 解释诊断信息





在典型的Abaqus/Standard分析中，载荷以增量形式施加到模型上，Abaqus尝试计算模型对每个增量载荷的响应。Abaqus进一步通过执行迭代来减少响应计算，以接近增量的结果。如果迭代没有接近解（收敛），Abaqus会停止并尝试用较小的载荷增量重新求解。如果Abaqus在无解的情况下尝试太多次，分析将结束。有关载荷增量的更多信息，请参见《Abaqus分析用户指南》第7章"分析求解和控制"。

在Abaqus/Explicit分析中，增量基于大量小时间增量。时间增量根据基于模型中单元大小的估计稳定时间增量大小进行。Abaqus/Explicit使用中心差分时间积分规则来积分运动方程，因此每个增量中不需要多次尝试和迭代。

查看诊断信息可以帮助您确定收敛问题的原因，以便您可以对模型进行必要的更正。即使达到收敛解，诊断信息也会指示潜在问题和改进区域。通过正确解释可用的诊断信息，您可以改进模型以获得与您的分析意图相匹配的结果。以下部分描述了**作业诊断**对话框中的各个页面（可用页面取决于分析类型和结果）：
