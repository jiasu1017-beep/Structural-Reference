# 3.2.10 使用文件选择对话框





文件选择对话框允许您从根据文件类型或位置过滤的列表中选择文件。要使用文件选择对话框，首先选择要打开的文件类型，然后指定要列出的目录。Abaqus/CAE会刷新对话框以仅列出符合您条件的文件。从该列表中，选择要打开的文件。

[图3-6](pt01ch03s02s10.md#int-selectbox-cae)显示了用于选择模型数据库或输出数据库的对话框。

**图3-6** 选择模型数据库或输出数据库。

![](../graphics/int-selectbox-nls.png)

**注意：**在Abaqus/Viewer中，您只能打开输出数据库文件；因此，**输出数据库(*.odb)**是**文件过滤器**字段中唯一的可用类型。

当您执行其他**文件**菜单功能（如导入零件或打印到文件）时，会出现类似的文件选择对话框。

使用以下技术选择您需要的文件：

**根据文件类型过滤文件列表**

文件选择对话框包含**文件过滤器**字段，允许您选择感兴趣的文件扩展名。例如，[图3-6](pt01ch03s02s10.md#int-selectbox-cae)中的**文件过滤器**选择是**输出数据库(*.odb)**。因此，只有扩展名为`.odb`的文件会出现在对话框中心的列表中。

**使用通配符搜索文件名**

您可以使用通配符过滤器搜索文件的部分名称。当您有大量文件存储在同一目录中时，通配符搜索非常有用。通配符搜索也会覆盖文件扩展名（如上所述的**文件过滤器**字段），允许您打开以非标准文件扩展名保存的文件。

要使用通配符搜索，请在**文件名**字段中使用以下形式之一输入部分文件名：

**?**：

匹配单个字符

*******：

匹配零个或多个字符

****[abc]****：

匹配单个字符，但必须是列出的字符之一

****[^abc]**或**[!abc]****：

匹配单个字符，但必须不是列出的字符之一

****[a-zA-Z]****：

匹配单个字符，但必须在提供的范围内

****[^a-zA-Z]**或**[!a-zA-Z]****：

匹配单个字符，但必须不在提供的范围内

**pat1|pat2或pat1,pat2**：

匹配pat1或pat2

**(pat1|pat2)或(pat1,pat2)**：

匹配pat1或pat2，模式可以嵌套

您可以组合多个通配符过滤器以进一步缩小搜索范围。例如，输入`[abc]*.(cae,odb)`会列出所有以`a`、`b`或`c`开头且具有`.cae`或`.odb`文件扩展名的文件。

**指定要从中选择文件的目录**

默认情况下，**目录**字段显示您启动Abaqus/CAE的目录。如果您想查看不同目录中的文件列表，可以单击列表中的目录名称以查看当前路径内的目录，或者单击**目录**字段旁边的箭头访问系统中可用的其他路径。此外，对话框顶部的图标允许您执行以下操作（如果有快捷键，会显示在括号中）：
- 向上移动一级目录（**[Backspace]**）。
- 访问系统默认或**主目录**（**[Ctrl]****+H**）。
- 访问**工作**目录（**[Ctrl]****+W**）。工作目录是您启动Abaqus/CAE的目录，除非您使用****文件**![](../graphics/images/arrow.gif)**设置工作目录****指定了目录。
- 对系统上的任何目录设置或使用**书签**。
- 创建新目录（**[Ctrl]****+N**）。

**目录**字段包含**网络连接器**项目。如果您已创建并启动了网络ODB连接器，您可以使用此项目访问远程目录并打开远程输出数据库。有关更多信息，请参阅["创建网络ODB连接器，" 第9.7.4节"](pt02ch09s06hlb04.md)。

**选择文件**

要选择并打开文件，请双击列表中感兴趣的文件名。您也可以开始键入文件名；光标会重新定位到文件列表中的匹配位置，并会选择以您键入的字母开头的第一个文件。或者，您可以直接在**文件名**字段中输入完整的目录路径和感兴趣的文件名，然后单击**确定**。对话框顶部的图标允许您将显示的文件格式更改为以下之一（快捷键显示在括号中）：
- 列表（**[Ctrl]****+S**）。
- 图标（**[Ctrl]****+B**）。
- 详细列表（**[Ctrl]****+L**）。

最右边的图标允许您显示或隐藏"隐藏"文件。
