# 15.17.15 指定连接器导出分量

您可以创建连接器导出分量，以将连接器中的摩擦产生法向力指定为连接器力和力矩的复杂组合，或将其用作连接器势函数中的中间结果。有关连接器导出分量的更多信息，请参见["耦合行为的连接器函数，"Abaqus分析用户指南第31.2.4节](../usb/usb-link.md#usb-elm-econnderivedandpotential)。

要显示导出分量编辑器或**编辑导出分量项**对话框中选项的上下文相关帮助，您必须选择感兴趣的选项，然后按**[F1]**。（编辑器和对话框显示时，主菜单栏中的**帮助**菜单不可用。）

**要创建或编辑连接器导出分量：**

1. 要显示连接器导出分量编辑器，请在定义用户定义的摩擦模型时或从势贡献编辑器中点击连接器截面编辑器中的![](../graphics/ico_edit.png)。有关更多信息，请参见["指定预定义的摩擦参数或接触力，"第15.17.5节"](pt03ch15s17hlb05.md)和["指定势项，"第15.17.16节"](pt03ch15s17hlb16.md)。
2. 在**编辑导出分量**对话框中，您可以按如下方式添加、编辑和删除导出分量项：
   - **添加项** 点击![](../graphics/ico_add.png)以显示**编辑导出分量项**对话框。
   - **修改项** 在**导出分量项**列表中，选择要更改其数据的项，然后点击![](../graphics/ico_edit.png)以显示**编辑导出分量项**对话框。
   - **删除项** 在**导出分量项**列表中，选择要删除的项，然后点击![](../graphics/ico_delete.png)。
3. 在**编辑导出分量项**对话框中，执行以下操作：
   1. 选择**项运算符**以指定用于计算导出分量项的方法。
      - **平方和的平方根** Abaqus将导出分量项计算为每个固有分量贡献的平方和的平方根。
      - **直接和** Abaqus将导出分量项计算为固有分量贡献的直接和。
      - **Macauley和** Abaqus将导出分量项计算为对每个贡献应用Macauley括号函数的固有分量贡献之和。
   2. 选择**整体项符号**。
      - **正** 为导出分量项选择正号。
      - **负** 为导出分量项选择负号。
   3. 选择用于定义导出分量的固有分量，并在数据表中为每个分量输入缩放因子（![](../graphics/usi_eqn00635.gif)）。
   4. 如果需要，打开**使用局部方向**，选择**独立位置分量**或**独立本构运动分量**，然后选择要用作连接器导出分量定义中自变量的可用相对位置或本构运动分量。在表的相应字段中输入由独立分量指定方向的连接器相对位置或本构运动。有关更多信息，请参见["定义依赖于相对位置或本构位移/旋转的非线性连接器行为属性"中的"连接器行为，"Abaqus分析用户指南第31.2.1节](../usb/usb-link.md#usb-elm-econnectbehav-indcomps)。
   5. 如果需要，打开**使用与温度相关的数据**以定义随温度变化的数据。数据表中会出现一个标有**温度**的列。
   6. 如果需要，点击**场变量数量**字段右侧的箭头以增加或减少场变量的数量，以定义依赖于场变量的数据。数据表中会出现场变量列。
4. 要修改正则化（Abaqus/Explicit分析专用）或数据外推的行为设置，请使用["为表格数据指定行为设置，"第15.17.14节"](pt03ch15s17hlb14.md)中所述的过程。
5. 点击**确定**保存数据并退出编辑器。
