### 创建冻结区域约束

您可以选择在拓扑优化或尺寸优化期间不会被修改的模型区域。这些区域继续对模型的重量和惯性等变量做出贡献。您可以冻结与其他零件接触的区域，例如用于支撑模型的区域和形成轴承表面的区域。建议您冻结应用了规定条件的区域。您可以使用几何约束冻结这些区域；或者，您可以在创建拓扑优化任务时请求 Optimization Module 自动冻结它们，如[「创建优化任务，」第 18.6.1 节](pt03ch18s06hlb01.md)中所述。冻结这些区域可以稳定优化，通常会大大减少迭代次数。

**创建冻结区域几何约束的步骤：**

1. 从主菜单栏，选择 ****几何约束**![](../graphics/images/arrow.gif)**创建****。此时将出现"创建几何约束"对话框。**提示：**您可以通过另外两种方式启动**创建**过程：- 在**几何约束管理器**中单击**创建**。（您可以通过从主菜单栏选择 ****几何约束**![](../graphics/images/arrow.gif)**管理器**** 来显示**几何约束管理器**。）- 在优化模块工具箱中单击 ![](../graphics/icoR_optGeomRest.png) 工具。
2. 在出现的"创建几何约束"对话框中，输入几何约束的名称。
3. 从几何约束列表中选择**冻结区域**或**冻结区域（尺寸）**，然后单击**继续**。
4. 从视图区，选择将被冻结的区域。默认情况下，Abaqus/CAE 允许您选择整个模型。要选择面或单元，请使用**选择**工具栏将可选择的对象类型更改为**面**或**单元**。有关更多信息，请参见[「根据对象类型过滤您的选择，」第 6.3.2 节](pt01ch06s03s01.md)。如果您希望从现有集合列表中进行选择，请执行以下操作：1. 单击提示区域右侧的**集合**。Abaqus/CAE 显示包含可用集合列表的**区域选择**对话框。2. 选择感兴趣的集合，然后单击**继续**。**注意：**默认选择方法基于您最近使用的选择方法。要恢复为另一种方法，请单击提示区域右侧的按钮——**在视图中选择**或**集合**。
5. 完成选择要冻结的区域后，在提示区域单击**完成**。有关选择对象的更多信息，请参见[第 6 章，"在视图中选择对象"](pt01ch06.md)。此时将出现"编辑几何约束"对话框。
6. 单击**确定**冻结所选区域并退出编辑器。

### 创建构件尺寸约束

对于拓扑优化，您可以指定模型所选区域的最小和/或最大尺寸；例如，优化创建桁架的区域的最小尺寸。您还可以指定桁架之间的最小距离。指定最大构件尺寸会强制优化将厚区域分割成几个较小的区域，并防止创建可能难以铸造的大型连续区域。指定最小构件尺寸可以避免可能难以制造的薄桁架。最小构件尺寸必须大于平均单元尺寸的两倍，以防止结果依赖于网格尺寸。

在大多数情况下，您可以为最小和最大构件尺寸指定相同的值，在这种情况下，Optimization Module 创建的桁架大约等于指定值。为了防止结构倒塌，Optimization Module 尝试避免在应用了规定条件的区域创建薄桁架。

指定最小和/或最大构件尺寸的计算成本非常高，应仅在必要时使用。您应该执行没有构件尺寸约束的优化，以确定应应用约束的任何区域。

**注意：**对于通用拓扑优化，您只能将构件尺寸约束和脱模约束组合使用。

**创建构件尺寸几何约束的步骤：**

1. 从主菜单栏，选择 ****几何约束**![](../graphics/images/arrow.gif)**创建****。此时将出现"创建几何约束"对话框。**提示：**您可以通过另外两种方式启动**创建**过程：- 在**几何约束管理器**中单击**创建**。（您可以通过从主菜单栏选择 ****几何约束**![](../graphics/images/arrow.gif)**管理器**** 来显示**几何约束管理器**。）- 在优化模块工具箱中单击 ![](../graphics/icoR_optGeomRest.png) 工具。
2. 在出现的"创建几何约束"对话框中，输入几何约束的名称。
3. 从几何约束列表中选择**构件尺寸（拓扑）**，然后单击**继续**。
4. 从视图区，选择将应用构件尺寸约束的区域，或单击**完成**将构件尺寸约束应用于整个模型。默认情况下，Abaqus/CAE 允许您选择整个模型。要选择面或单元，请使用**选择**工具栏将可选择的对象类型更改为**面**或**单元**。有关更多信息，请参见[「根据对象类型过滤您的选择，」第 6.3.2 节](pt01ch06s03s01.md)。如果您希望从现有集合列表中进行选择，请执行以下操作：1. 单击提示区域右侧的**集合**。Abaqus/CAE 显示包含可用集合列表的**区域选择**对话框。2. 选择感兴趣的集合，然后单击**继续**。**注意：**默认选择方法基于您最近使用的选择方法。要恢复为另一种方法，请单击提示区域右侧的按钮——**在视图中选择**或**集合**。
5. 完成选择几何约束区域后，在提示区域单击**完成**。有关选择对象的更多信息，请参见[第 6 章，"在视图中选择对象"](pt01ch06.md)。此时将出现"编辑几何约束"对话框。
6. 执行以下操作之一：- 选择**最小厚度**，然后输入最小构件厚度。- 选择**最大厚度**，然后输入最大构件厚度。- 选择**包络**，然后输入以下内容：- **最小**构件厚度。- **最大**构件厚度。- 构件之间的**最小间隙**。
7. 单击**确定**创建冻结区域几何约束并退出编辑器。

**图 18-3** 拓扑优化中的循环对称。

![](../graphics/opz-cyclicsymmetry.png)

所选区域沿坐标系的轴复制。您可以使用全局坐标系或创建基准坐标系，详情请参见[「创建基准坐标系方法的概述，」第 62.5.4 节](pt06ch62s05s04.md)。您可以选择从对称约束中移除冻结区域。

6. 选择坐标系，并选择将沿其复制区域的轴。

8. 如果需要，切换**忽略冻结区域**以从将被平移的区域中移除任何冻结区域。
9. 单击**确定**创建循环对称几何约束并退出编辑器。

### 创建最小宽度约束

尺寸优化在使用壳单元对金属板结构建模时确定最佳单元厚度。指定包含相同厚度单元的区域的最小宽度可以防止在尺寸优化后在解中出现具有相同厚度的窄条单元。指定最小宽度还可以防止壳厚度振荡或单元厚度的「棋盘」模式。最小宽度必须大于单元边缘的平均长度。

**创建最小宽度几何约束的步骤：**

1. 从主菜单栏，选择 ****几何约束**![](../graphics/images/arrow.gif)**创建****。此时将出现"创建几何约束"对话框。**提示：**您可以通过另外两种方式启动**创建**过程：- 在**几何约束管理器**中单击**创建**。（您可以通过从主菜单栏选择 ****几何约束**![](../graphics/images/arrow.gif)**管理器**** 来显示**几何约束管理器**。）- 在优化模块工具箱中单击 ![](../graphics/icoR_optGeomRest.png) 工具。
2. 在出现的"创建几何约束"对话框中，输入几何约束的名称。
3. 从几何约束列表中选择**构件尺寸（尺寸）**，然后单击**继续**。
4. 从视图区，选择将应用最小宽度约束的区域，或单击**完成**将最小宽度约束应用于整个模型。默认情况下，Abaqus/CAE 允许您选择整个模型。要选择面或单元，请使用**选择**工具栏将可选择的对象类型更改为**面**或**单元**。有关更多信息，请参见[「根据对象类型过滤您的选择，」第 6.3.2 节](pt01ch06s03s01.md)。如果您希望从现有集合列表中进行选择，请执行以下操作：1. 单击提示区域右侧的**集合**。Abaqus/CAE 显示包含可用集合列表的**区域选择**对话框。2. 选择感兴趣的集合，然后单击**继续**。**注意：**默认选择方法基于您最近使用的选择方法。要恢复为另一种方法，请单击提示区域右侧的按钮——**在视图中选择**或**集合**。
5. 完成选择几何约束区域后，在提示区域单击**完成**。有关选择对象的更多信息，请参见[第 6 章，"在视图中选择对象"](pt01ch06.md)。此时将出现"编辑几何约束"对话框。
6. 选择**最小宽度**并输入最小宽度。
7. 单击**确定**创建最小宽度几何约束并退出编辑器。

### 创建厚度聚类约束

您可以指定所选区域在尺寸优化后包含相同厚度的壳单元聚类。您可以使用聚类在正在优化的金属板结构中生成加强肋或环，或定义相同厚度区域之间的边界。聚类区域可以通过使用恒定厚度的板材进行制造再现；例如，通过焊接和冲压单个金属板结构形成的汽车「白车身」。为了获得最大的设计灵活性，您应该首先在不指定聚类的情况下优化您的结构，并使用初始设计来决定在最终优化中要聚类的区域。

**创建厚度聚类几何约束的步骤：**

1. 从主菜单栏，选择 ****几何约束**![](../graphics/images/arrow.gif)**创建****。此时将出现"创建几何约束"对话框。**提示：**您可以通过另外两种方式启动**创建**过程：- 在**几何约束管理器**中单击**创建**。（您可以通过从主菜单栏选择 ****几何约束**![](../graphics/images/arrow.gif)**管理器**** 来显示**几何约束管理器**。）- 在优化模块工具箱中单击 ![](../graphics/icoR_optGeomRest.png) 工具。
2. 在出现的"创建几何约束"对话框中，输入几何约束的名称。
3. 从几何约束列表中选择**聚类区域（尺寸）**，然后单击**继续**。此时将出现"编辑几何约束"对话框。
4. 从**合格集合**列表中，选择指定在优化后将包含相同厚度壳单元聚类的区域的集合。**注意：**如果所需的集合不存在，请单击 ![](../graphics/ico_CreateSet.png) 创建它。
5. 单击箭头将所选集合移动到**所选集合**列表中。
6. 单击**确定**创建厚度聚类几何约束并退出编辑器。

### 创建脱模约束

您可以为拓扑优化指定脱模几何约束。脱模几何约束强制优化后的模型满足指定的制造要求；例如，它可以防止必须从模具中提取的零件出现倒扣和空腔区域。

**创建脱模约束的步骤：**

1. 从主菜单栏，选择 ****几何约束**![](../graphics/images/arrow.gif)**创建****。此时将出现"创建几何约束"对话框。**提示：**您可以通过另外两种方式启动**创建**过程：- 在**几何约束管理器**中单击**创建**。（您可以通过从主菜单栏选择 ****几何约束**![](../graphics/images/arrow.gif)**管理器**** 来显示**几何约束管理器**。）- 在优化模块工具箱中单击 ![](../graphics/icoR_optGeomRest.png) 工具。
2. 在出现的"创建几何约束"对话框中，输入几何约束的名称。
3. 从几何约束列表中选择**脱模**，然后单击**继续**。
4. 从视图区，选择将强制执行脱模约束的区域，或单击**完成**将脱模对称约束应用于整个模型。默认情况下，Abaqus/CAE 允许您选择整个模型。要选择面或单元，请使用**选择**工具栏将可选择的对象类型更改为**面**或**单元**。有关更多信息，请参见[「根据对象类型过滤您的选择，」第 6.3.2 节](pt01ch06s03s01.md)。如果您希望从现有集合列表中进行选择，请执行以下操作：1. 单击提示区域右侧的**集合**。Abaqus/CAE 显示包含可用集合列表的**区域选择**对话框。2. 选择感兴趣的集合，然后单击**继续**。**注意：**默认选择方法基于您最近使用的选择方法。要恢复为另一种方法，请单击提示区域右侧的按钮——**在视图中选择**或**集合**。
5. 完成选择几何约束区域后，在提示区域单击**完成**。有关选择对象的更多信息，请参见[第 6 章，"在视图中选择对象"](pt01ch06.md)。此时将出现"编辑几何约束"对话框。
6. 默认情况下，Optimization Module 检查碰撞的区域与强制执行脱模约束的区域相同。如果需要，从"编辑几何约束"对话框顶部选择 ![](../graphics/ico_selectBlue.png)，然后选择碰撞检查区域。为了避免空腔，碰撞检查区域应包括脱模区域和与脱模区域相邻的任何单元。
7. 选择以下脱模技术之一：- 选择**带中心平面的脱模**，然后选择 Optimization Module 将如何确定中心平面：- 选择**自动确定**。Optimization Module 确定中心平面的最佳位置，使得优化仅在拉拔方向或相反方向添加或移除材料。切换**防止孔洞形成**以防止优化从中心平面移除材料。- 选择**指定**，然后单击 ![](../graphics/ico_selectBlue.png) 选择中心平面上的一个点。Optimization Module 创建一个可以从模具中沿中心平面两侧的拉拔方向释放的结构。- 选择**锻造（仅在拉拔方向变形）**。假定模具两半相遇的平面（即中心平面）位于模型背面并垂直于拉拔方向。优化仅在拉拔方向添加或移除材料。- 选择**冲压**。如果 Optimization Module 决定从结构中移除一个单元，它还会移除该单元后面或前面的所有单元（相对于指示拉拔方向的向量）。- 选择**在区域表面脱模**，强制优化仅从约束区域的表面添加或移除材料。
8. 单击 ![](../graphics/ico_selectBlue.png)，然后选择两个点，指定沿拉拔方向的向量。
9. 输入**拔模角度**。拔模角度是相对于拉拔方向的小角度，将被引入优化后的模型中以确保零件可以从其模具中移除。通常该值在 0 到 10 之间。
10. 单击**确定**创建脱模几何约束并退出编辑器。
