# *SUBSTRUCTURE DIRECTORY





### *SUBSTRUCTURE DIRECTORY列出关于子结构库中子结构的信息。

此选项用于提供关于存储在子结构库中的子结构的信息摘要。

**产品：**Abaqus/Standard

**类型：**模型数据

**级别：**此选项在以部件实例装配定义的模型中不受支持。

##### **参考：**

- ["Using substructures," Section 10.1.1 of the Abaqus Analysis User's Guide](../usb/usb-link.md#usb-anl-asuperelements)

### **必需参数：**

LIBRARY

将此参数设置为子结构库的名称。

**此选项没有关联的数据行。**
