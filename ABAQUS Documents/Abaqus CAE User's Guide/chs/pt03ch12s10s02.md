# 12.10.2 在热传递分析中包括体积热生成







用户子程序[`HETVAL`](../sub/sub-link.md#sub-xsl-hetval)允许您为热传递、耦合热-电或耦合温度-位移分析定义由内部热生成引起的热通量。

Abaqus/Standard在每个材料定义包含对该用户子程序引用的点上调用用户子程序[`HETVAL`](../sub/sub-link.md#sub-xsl-hetval)。有关更多信息，请参见"HETVAL"，Abaqus用户子程序参考指南第1.1.13节。

**在材料定义中包含对用户子程序[`HETVAL`](../sub/sub-link.md#sub-xsl-hetval)的引用：**

1. 从**编辑材料**对话框的菜单栏中，选择**热**![](../graphics/images/arrow.gif)**热生成**。（有关显示**编辑材料**对话框的信息，请参见"创建或编辑材料"，第12.7.1节。）
2. 单击**确定**关闭**编辑材料**对话框。或者，您可以从**编辑材料**对话框的菜单中选择其他材料行为进行定义（有关更多信息，请参见"浏览和修改材料行为"，第12.7.2节）。
3. 进入作业模块并显示感兴趣的分析作业的作业编辑器。（有关更多信息，请参见"创建、编辑和操作作业"，第19.7节。）
4. 在作业编辑器中，单击**常规**选项卡，并指定包含用户子程序[`HETVAL`](../sub/sub-link.md#sub-xsl-hetval)的文件。有关更多信息，请参见"指定常规作业设置"，第19.8.6节。**注意：**您只能在作业编辑器中指定一个用户子程序文件；如果您的分析涉及多个用户子程序，您必须将用户子程序组合成一个文件，然后指定该文件。
