# 10.3 树工具集

树工具集提供一个包含Abaqus/CAE中模型树和结果树的标签区域。有关更多信息，请参阅["使用模型树和结果树，" Abaqus/CAE User's Guide的第3.5节](../usi/usi-link.md#uss-top-modeltree)。树工具集包含以下可用于自定义标签区域外观的方法：

- `makeModelTab`方法创建包含模型树的标签页。标签页的名称为"Model"。
- `makeMaterialLibraryTab`方法创建包含材料库的标签页。标签页的名称为"Material Library"。
- `makeResultsTab`方法创建包含结果树的标签页。标签页的名称为"Results"。
- `makeTabs`方法调用上面列出的所有方法。

此外，主窗口有一个`appendTreeTab`方法，它在标签区域创建一个新标签项目并返回一个垂直框架，你可以向其中添加窗口部件。如果你想在模型和结果标签之后简单地添加一个标签页，可以从自定义代码中使用`appendTreeTab。但是，如果你想更改标签页的顺序或删除标准标签页之一，必须从树工具集派生你自己的工具集。例如：

```
class MyTreeToolsetGui(TreeToolsetGui):

    def makeTabs(self):

        self.makeModelTab()
        self.makeMyTab()
        self.makeMaterialLibraryTab()
        self.makeResultsTab()

    def makeMyTab(self):

        vf = getAFXApp().getAFXMainWindow().appendTreeTab(
            'My Tab', 'My Tab')
        FXLabel(vf, 'This is my tab item')
```

`appendTreeTab`方法的第一个参数是你想在标签按钮中显示的文本。第二个参数是标签的名称，用于在各种应用程序编程接口（如`setCurrentTreeTab`(*name*)）中进行标识。

默认情况下，当你创建标签页时，它在所有模块中都可见，并且适用于所有模块。如果你不希望标签页对所有模块都可见或适用，可以使用`setApplicabilityForTreeTab`和`setVisibilityForTreeTab`方法。当用户切换到新模块时，应用程序将检查当前标签页在新模块中是否可见和适用。如果标签页不可见，它将被隐藏。如果不适用，应用程序将搜索适用于新模块的第一个标签页并使其成为当前标签页。例如：

```
    def makeMyTab(self):

        vf = getAFXApp().getAFXMainWindow().appendTreeTab(
            'My Tab', 'My Tab')
        getAFXApp().getAFXMainWindow().setApplicabilityForTreeTab(
            'My Tab', 'Part, Property')
        getAFXApp().getAFXMainWindow().setVisibilityForTreeTab(
            'My Tab', 'Part, Property')

        FXLabel(vf, 'This is my tab')

```

在这种情况下，当用户在Part模块中时，将显示"我的标签页"。如果用户点击"我的标签页"使其成为当前标签页，然后切换到Property模块，"我的标签页"将保持可见和当前。如果用户切换到Step模块，"我的标签页"将被隐藏，"模型"标签页将成为当前标签页（因为它已被定义为适用于除可视化模块外的所有模块）。
