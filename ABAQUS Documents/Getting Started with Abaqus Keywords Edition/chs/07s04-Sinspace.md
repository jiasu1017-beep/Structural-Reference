# 7.4 动力学网格设计

在设计动态模拟的网格时，需要考虑响应中将被激发的振型，并使用能够充分表达这些振型的网格。这意味着，对于静态模拟足够的网格可能不适合计算激发高频模态的载荷的动态响应。

例如，考虑图7-3所示的板。一阶壳单元的网格对于板在均匀载荷下的静态分析是足够的，也适合预测第一振型。然而，该网格显然太粗，无法准确模拟第六振型。

**图7-3** 基于粗网格的板振动频率及相应振型。

![../graphics/gss-coarse-mesh-nls.png](../graphics/gss-coarse-mesh-nls.png)

图7-4展示了使用一阶单元细化网格建模的同一板。第六振型的位移形状现在看起来好得多，该模态预测的频率也更加准确。如果作用于板的动态载荷使得该振型被显著激发，则必须使用细化网格；粗网格的结果将不准确。

**图7-4** 基于细网格的板振动频率及相应振型。
