# 13.3.6 从分析中排除零件实例







您可以从分析中排除零件实例，以便在提交分析作业时不将它们写入输入文件。被排除的零件实例参与除分析之外的所有操作。

在模型树中，选择要从分析中排除的零件实例。右键单击，然后选择**从模拟中排除**。同样，您可以通过选择**包含在模拟中**来包含先前排除的零件实例。如果您从分析中排除实例然后又包含它们，则会保留对零件实例的约束。

默认情况下，从分析中排除的零件实例在视口中以深灰色显示。模型树中显示图标以指示零件实例的排除状态，以及如果零件实例也在模型之间链接，则还指示零件实例的链接和排除状态，如图13-3所示。`Contact-1`和`Contact-2`是从分析中排除的零件实例，`Beam-2`是链接且被排除的零件实例。请参见"在模型之间链接零件实例"，第13.3.5节。
