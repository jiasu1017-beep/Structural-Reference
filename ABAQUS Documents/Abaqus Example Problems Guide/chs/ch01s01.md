# 1.1 静力与准静力应力分析

**产品：** Abaqus/Standard

## 概述

本节包含例题，用于说明 Abaqus/Standard 在静力和准静力应力分析中的应用。这些例题展示了以下各种能力：

- 线性与非线性静力分析
- 接触建模
- 塑性与蠕变行为
- 热耦合
- 子模型技术
- 单元移除与重新激活
