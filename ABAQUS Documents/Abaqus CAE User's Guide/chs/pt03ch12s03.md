# 12.3 我可以为部件分配哪些属性？





创建截面后，您可以为部件分配以下属性：

**截面**

您可以将截面分配给部件的区域。**Section Assignment Manager**允许您查看、创建、编辑、抑制、恢复和删除截面分配。在Property模块中，Abaqus/CAE将区域着色为绿色，以表示该区域具有截面分配。如果存在重叠的截面分配，Abaqus/CAE将区域着色为黄色。

**梁截面取向**

您可以为线区域分配梁截面取向。您通过定义横截面的大致局部1方向来为梁截面分配取向。

**材料取向**

您可以为壳和实体区域分配材料取向。整体坐标系决定默认的材料取向。您可以通过选择现有基准坐标系或离散场，或通过定义离散取向来定义材料取向。对于Abaqus/Standard分析，您可以在用户子程序中定义材料取向。

**钢筋参考取向**

钢筋层的角度取向是相对于钢筋参考取向定义的。您可以为壳区域分配钢筋参考取向。整体坐标系决定默认的钢筋参考取向。您可以通过从视口中选择现有基准坐标系，然后选择该基准坐标系上近似于壳法线方向的轴来定义钢筋参考取向。

**元素法线**

您可以为孤立元素、壳和膜区域以及具有线区域的轴对称部件分配壳/膜法线方向。壳/膜法线影响分配给该区域的材料取向。如果您反转壳区域的法线，材料2方向将被反转。材料2方向的反转对分析结果没有影响。但是，在解释壳的截面点输出时应小心。

**元素切线**

您可以为孤立元素和线区域分配梁/桁架切线方向。梁截面取向取决于梁切线方向。如果您反转切线方向，局部2方向将被反转，在解释结果时应小心，特别是在识别梁截面点位置时。

您可以使用Property模块主菜单栏中的**Assign**菜单为部件分配属性。您可以通过以下方式选择要分配属性的区域：
- 直接在视口中选择区域。
- 单独选择元素或使用角度方法（为孤立元素分配壳法线）。
- 使用Set工具集创建由部件区域或孤立元素组成的集。（Set工具集可从主菜单栏的**Tools**菜单访问。）然后，您可以将为集定义的区域或元素分配属性。

如果您为某个区域分配了截面，然后重命名或删除了该截面，则该截面不再应用于该区域。如果模型的某个区域缺少截面属性，您的分析作业将失败，问题将由Job模块报告。

但是，重命名或删除的截面的原始名称继续与已分配它们的区域相关联，直到您采取以下操作之一：
- 为该区域分配不同的截面。
- 创建一个具有原始截面名称且对该区域类型适当的截面（例如，壳区域的壳截面）；新截面中定义的属性将自动应用于该区域。
- 如果您重命名了截面，请将截面名称改回原始名称。

（您可以使用查询工具集来确定分配给该区域的截面的名称；有关更多信息，请参见["理解查询工具集的作用，"第71.1节](pt06ch71s01.md)。）

同样，如果您在截面定义中引用了某材料，然后重命名或删除了该材料，则该截面变得无效；该截面中定义的属性不再应用于分配该截面的区域。但是，重命名或删除的材料的原始名称继续与引用这些材料的截面相关联；因此，您可以使用类似于上面列出的技术来恢复截面。
