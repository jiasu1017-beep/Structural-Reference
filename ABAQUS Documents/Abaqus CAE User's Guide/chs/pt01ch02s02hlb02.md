# 2.2.2 主菜单栏的组成





启动会话时，以下列出的菜单会出现在主菜单栏上。Abaqus/CAE会根据当前使用的模块显示额外的菜单选项并提供对工具集的访问。

**文件**

**文件**菜单中的项目允许您创建、打开和保存模型数据库；打开和关闭输出数据库；导入和导出文件；保存和加载会话对象和选项；运行脚本；管理宏；打印视口；以及退出Abaqus/CAE。有关更多信息，请参阅["使用文件菜单，" 第9.6节](pt02ch09hla01.md)。

**模型**

**模型**菜单中的项目允许您打开、复制、重命名和删除当前模型数据库中的模型。有关更多信息，请参阅["管理模型，" 第9.8节](pt02ch09s07.md)。

**视口**

**视口**菜单中的项目允许您创建或操作视口以及视口注释。有关更多信息，请参阅[第4章，"管理画布上的视口"](pt01ch04.md)。

**视图**

**视图**菜单中的项目允许您操作视图，自定义模型或绘图外观的某些方面，控制显示性能，以及关闭模型树、结果树和各个工具栏的显示。视图操作菜单中的一些操作也可以在**视图操作**工具栏中找到。有关更多信息，请参阅：
- ["使用模型树和结果树，" 第3.5节](pt01ch03s05.md)
- [第5章，"操作视图和控制透视"](pt01ch05.md)
- [第7章，"配置图形显示选项"](pt01ch07.md)
- [第55章，"自定义绘图显示"](pt05ch55.md)
- [第61章，"自定义工具集"](pt06ch61.md)
- [第76章，"自定义几何体和网格显示"](pt07ch76.md)

**插件**

**插件**菜单中的项目允许您访问随Abaqus/CAE分发的插件或您已下载或创建的插件。有关更多信息，请参阅[第81章，"插件工具集"](pt08ch81.md)。

**帮助**

**帮助**菜单中的项目允许您请求上下文相关帮助，搜索或浏览文档，访问学习社区，以及获取有关版本和许可的信息。有关更多信息，请参阅["获取帮助，" 第2.6节](pt01ch02s06.md)。
