### 问题描述

![](../graphics/verthermstress-model-nls.png)

上图所示为圆柱壳。在 Abaqus/Standard 分析和使用耦合热壳单元的 Abaqus/Explicit 分析中使用单个单元。在使用实体单元的 Abaqus/Explicit 分析中，径向使用两个单元。对于非轴对称单元，单元在中心张开的角度为 11.25，相当于圆周上的 32 个单元。

在 Abaqus/Standard 模拟中假设稳态条件。在 Abaqus/Explicit 中执行瞬态模拟。使用实体单元的分析的总模拟时间为 0.4 秒，使用壳单元的分析为 0.06 秒。这为瞬态解达到该问题的稳态条件提供了足够的时间。质量缩放用于实体单元分析，以降低 Abaqus/Explicit 分析的计算成本。

**材料：**

| 密度 | 7800 kg/m3 |
| --- | --- |
| 导热系数 | 52 J/ms C |
| 比热容 | 586 J/kg C |
| 热膨胀系数 | 1.2 105 |
| 弹性模量 | 200 103 MPa |
| 泊松比 | 0.3 |

**边界条件：**

对于热分析，内表面和外表面温度分别规定为 200C 和 100C。对于应力分析，环向的旋转矢量受到约束，但圆筒可以自由轴向膨胀。对于连续体单元网格，使用方程提供旋转约束。对于非轴对称情况，在环向施加对称约束以模拟完整圆筒。

在 Abaqus/Explicit 模拟中，温度逐渐施加以确保准静态响应。

对于所有分析，除了使用耦合温度-位移单元的分析（ Abaqus/Standard 中的 SAX2T、S8RT、CAX4RT、CAX4RHT、CGAX4RT、CGAX4RHT、CAX8RT、CGAX8RT 和 C3D20RT，以及 Abaqus/Explicit 中的 S4RT、CAX3T、CAX4RT、C3D4T、C3D6T、C3D8RT 和 C3D8T），分析成对运行：热分析 followed by its corresponding stress analysis。

对于输入文件 [es54sxsj.inp](../eif/es54sxsj.inp)，壳截面使用高斯积分。

### 结果与讨论

轴对称和二阶壳单元与理论完全一致。一阶三维壳（S4R5）显示 5.1% 的误差。连续体单元与参考解存在微小差异（< 1%）。

Abaqus/Explicit 获得的结果与解析解和 Abaqus/Standard 获得的结果非常一致。
