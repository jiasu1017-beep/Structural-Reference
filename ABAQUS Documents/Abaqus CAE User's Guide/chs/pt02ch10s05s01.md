# 10.5.1 从Abaqus/CAE模型数据库导入模型





您可以通过从主菜单栏选择****文件（File）**![](../graphics/images/arrow.gif)**导入（Import）**![](../graphics/images/arrow.gif)**模型（Model）****将模型导入到当前Abaqus/CAE模型数据库中。从出现的对话框中选择模型数据库（`.cae`）文件，然后从**从模型数据库导入模型（Import Model from Model Database）**对话框中选择要导入的模型。必须先将先前版本的Abaqus模型数据库升级到当前版本，然后才能导入其模型数据。

当您从模型数据库导入模型时，Abaqus/CAE会创建该模型的副本，其中包括原始版本中的所有模型数据。但是，此导入排除原始模型数据库文件中存在但与模型没有直接关系的以下数据：
- 作业数据
- 使用Abaqus Scripting Interface创建的自定义用户数据

您还可以提示Abaqus/CAE在模型导入后立即显示**复制对象（Copy Objects）**对话框。此对话框使您能够在当前模型数据库中的模型之间复制单个对象，如零件、草图或材料。有关复制对象的更多信息，请参阅["在模型之间复制对象，"第9.8.3节"](pt02ch09s07hlb03.md)。

从不同模型数据库导入的模型与其原始模型没有链接；导入后对原始模型所做的任何更改都不会传播到副本。
