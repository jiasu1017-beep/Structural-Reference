# 62.6.3 Creating a datum point midway between two points







You can position a datum point by selecting two points on the model; Abaqus/CAE creates the datum point midway between the two selected points.

![](../graphics/dtm-hpmid-nls.png)

**To create a datum point midway between two points:**

1. From the main menu bar, select ****Tools**![](../graphics/images/arrow.gif)**Datum****. The **Create Datum** dialog box appears. The dialog box outlines the types of datum geometry you can create. Abaqus/CAE displays prompts in the prompt area to guide you through the procedure. **Tip:**You can also create a datum point using the ![](../graphics/ico_dtmPtMidpoint.png) tool, located in the module toolbox. For a diagram of the datum tools in the toolbox, see ["Using the Datum toolset," Section 62.2](pt06ch62s02.md).
2. From the list of types at the top of the dialog box, choose **Point**. The **Method** list indicates the methods you can use to create a datum point.
3. From the **Method** list, select **Midway between 2 points**.
4. From the part or assembly in the current viewport, select two points. The datum point appears. You cannot modify a datum point created with this method; you must delete the old point and create a new one.
