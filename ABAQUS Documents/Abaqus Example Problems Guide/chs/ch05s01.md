# 5.1.1 盘式制动器的热应力分析

**产品：** Abaqus/Standard  Abaqus/Explicit   

盘式制动器的工作原理是将一组复合材料的制动片压靠在旋转的钢盘上：摩擦力导致减速。摩擦热耗散对于有效的制动性能至关重要。制动器的温度变化导致轴向和径向变形；这种形状变化反过来又影响制动片和制动盘之间的接触。因此，系统应该作为全耦合热力系统进行分析。

本节讨论两个热耦合盘式制动器分析示例。第一个示例是一个轴对称模型，其中制动片和由制动产生的摩擦热被抹平到模型的整个360度。该问题仅使用Abaqus/Standard求解。热量通过用户子程序 FRIC 提供，分析模拟了由于制动导致的速度线性下降。

第二个示例是整个制动盘的完整三维模型，制动片仅接触部分圆周。制动盘旋转，使热量由摩擦产生。该问题使用Abaqus/Standard和Abaqus/Explicit求解。

也可以对制动系统进行非耦合分析。可以计算热通量并施加到热模型；然后可以将得到的温度施加到应力分析。但是，由于热分析和应力分析是非耦合的，因此这种方法不考虑热变形对接触的影响，而接触又影响热量产生。

另一种几何模型用于盘式制动器。详细参见内容。
