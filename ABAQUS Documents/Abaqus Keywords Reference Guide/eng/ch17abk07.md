# *RANDOM RESPONSE







### *RANDOM RESPONSECalculate response to random loading.

This option is used to give the linearized response of a model to random excitation.

**Products: **Abaqus/Standard  Abaqus/CAE  

**Type: **History data 

**Level: **Step

**Abaqus/CAE: **Step module
