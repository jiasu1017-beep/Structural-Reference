# 4.6 行和列布局管理器







`FXMatrix` 组件以行和列的方式排列其子组件。您可以使用 *opts* 参数的默认值（MATRIX_BY_ROWS）按行进行布局，或者通过设置 *opts*=MATRIX_BY_COLUMNS 按列进行布局。如果您指定 *opts*=MATRIX_BY_ROWS，矩阵将创建指定数量的行，并根据需要创建尽可能多的列来容纳其所有子组件。相反，如果您指定 *opts*=MATRIX_BY_COLUMNS，矩阵将创建指定数量的列，并根据需要创建尽可能多的行来容纳其所有子组件。

例如，使用默认的 *opts*=MATRIX_BY_ROWS 设置：

```
m = FXMatrix(parent, 2)
FXButton(m, 'Button 1')
FXButton(m, 'Button 2')
FXButton(m, 'Button 3')
FXButton(m, 'Button 4')
FXButton(m, 'Button 5')
FXButton(m, 'Button 6') 
```
