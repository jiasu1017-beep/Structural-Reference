# 1.5.4 轴对称单元片测试





**产品：**Abaqus/Standard  Abaqus/Explicit  

### 测试的单元

CAX3    CAX3H    CAX3T    CAX4    CAX4H    CAX4I    CAX4IH    CAX4R    CAX4RH    CAX4RHT    CAX4RT  

CAX6    CAX6H    CAX6M    CAX6MH    CAX8    CAX8H    CAX8R    CAX8RH  

### 问题描述

![](../graphics/veraxipatch-model.png)

**材料：**

线弹性，杨氏模量 = 1.0×10^6，泊松比 = 0.25。

对于耦合温度-位移单元，指定虚拟热特性以完成材料定义。

**步骤1的载荷：**

施加于所有外节点处的位移边界条件：![](../graphics/ver_eqn00409.gif) ×10^3×*r*，![](../graphics/ver_eqn00393.gif) ×10^3×![](../graphics/ver_eqn00410.gif)。

非均匀体积力：为了保持恒定剪应力![](../graphics/ver_eqn00051.gif) = 400并维持平衡，在用户子程序[`DLOAD`](../sub/sub-link.md#sub-xsl-dload)中定义平衡体积力BZNU为BZNU = ![](../graphics/ver_eqn00411.gif) ×400×![](../graphics/ver_eqn00412.gif)，其中*r*是积分点半径。

在Abaqus/Explicit模拟中，此步骤之后是一个中间步骤，将模型返回到无载荷状态。

**步骤2的载荷：**

施加于所有外节点处的位移边界条件：![](../graphics/ver_eqn00409.gif) ×10^2×*r*，![](../graphics/ver_eqn00393.gif) ×10^2×*z*。

**步骤3的载荷：**

施加于步骤2变形状上所有外节点处的位移边界条件：![](../graphics/ver_eqn00409.gif) ×10^3×*r*，![](../graphics/ver_eqn00393.gif) ×10^3×![](../graphics/ver_eqn00410.gif)。

非均匀体积力（与步骤1描述相同）：BZNU = ![](../graphics/ver_eqn00416.gif) ×400×![](../graphics/ver_eqn00412.gif)。

在Abaqus/Standard模拟中，此步骤定义为扰动步骤；在Abaqus/Explicit模拟中，则指定产生扰动的速度边界条件。

### 参考解

每个步骤的分析结果如下所示。

#### 步骤1：PERTURBATION（扰动）

- ![](../graphics/ver_eqn00413.gif) = 2000。
- ![](../graphics/ver_eqn00051.gif) = 400。
- ![](../graphics/ver_eqn00414.gif) = 10^3。
- ![](../graphics/ver_eqn00415.gif) = 10^3。

#### 步骤2：NLGEOM（非线性几何）

- ![](../graphics/ver_eqn00413.gif) = 19900。
- ![](../graphics/ver_eqn00051.gif) = 0。
- ![](../graphics/ver_eqn00414.gif) = 9.95×10^3。
- ![](../graphics/ver_eqn00415.gif) = 0。

在Abaqus/Explicit模拟中，这是第三个步骤。（Abaqus/Explicit模拟中的第二个步骤将模型返回到无载荷状态。）

#### 步骤3：PERTURBATION（扰动）

- ![](../graphics/ver_eqn00413.gif) = 2000。
- ![](../graphics/ver_eqn00051.gif) = 400。
- ![](../graphics/ver_eqn00414.gif) = 1×10^3。
- ![](../graphics/ver_eqn00415.gif) = 1×10^3。

在Abaqus/Explicit模拟中，这是第四个步骤。必须从第四个步骤的结果中减去第三个步骤的结果，以获得关于加载状态的扰动。

### 结果与讨论

所有单元都产生精确解。
