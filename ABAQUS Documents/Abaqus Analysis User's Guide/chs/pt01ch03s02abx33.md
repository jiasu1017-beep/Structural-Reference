# 3.2.33 将 Abaqus 输出数据库文件转换为 Nastran Output2 结果文件





**产品：**Abaqus/Standard

##### **参考**

- [『Abaqus 执行程序：概述』，第 3.1.1 节](pt01ch03s01abo02.md)

### 概述

该转换器将 Abaqus 输出数据库（`.odb`）文件中的某些结果转换为 Nastran Output2 文件格式。

### 使用转换器

toOutput2 转换器只能用于转换 [*STATIC](../key/key-link.md#usb-kws-hstatic) 或 [*FREQUENCY](../key/key-link.md#usb-kws-hfrequency) 过程的 Abaqus 输出数据库。通过使用 [*OUTPUT](../key/key-link.md#usb-kws-houtput) 选项将 Abaqus 分析的结果写入 Abaqus 输出数据库。应在 Abaqus 输入文件中包含以下选项以确保要转换的结果在 Abaqus 输出数据库中可用：

```
[*OUTPUT](../key/key-link.md#usb-kws-houtput), FIELD
[*NODE OUTPUT](../key/key-link.md#usb-kws-hnodeoutput)
U,
RF,
CF,
[*ELEMENT OUTPUT](../key/key-link.md#usb-kws-helementoutput)
S,
E,
SF,
NFORC,
```
Abaqus 输出数据库中除上述指定结果之外的其他结果将在转换过程中被跳过。仅转换弹簧单元、三维连续体、壳、薄膜、梁和桁架单元的结果。

对于壳单元，转换器将最低编号截面点的应力和应变视为底面处的应力和应变，将最高编号截面点的应力和应变视为顶面处的应力和应变。转换到 Output2 文件的中面应力和应变计算为底面和顶面应力和应变的平均值。

节点结果始终位于全局坐标系中。单元张量结果位于 Abaqus 单元坐标系中。

输出数据库中的模型数据（节点坐标、单元拓扑、材料属性和单元属性）在适用的记录存在时写入 Output2 文件。

### 命令行选项

**job**

此选项指定转换器将创建的 Nastran Output2 文件的名称。它也是 Abaqus 输出数据库的默认名称。

**odb**

此选项在名称与 *job-name* 不同时指定 Abaqus 输出数据库的名称。

**step**

此选项指定转换器要转换的 Abaqus 输出数据库的步骤编号。如果指定步骤包含多个载荷情况，则所有载荷情况都将被转换。默认值为分析的最后一个步骤。

**increment**

此选项仅在与 **step** 选项结合使用时有效。它用于指定转换器要转换的 Abaqus 输出数据库中该步骤的增量编号。默认值为指定步骤的最后一个增量。

**slim**

此选项用于在 Output2 文件中包含 SLIM/VISION 软件（可从 Third Millennium Productions, Inc. 获得）后处理所需的数据块。

**quad4corner**

此选项用于请求在角节点而非形心处进行壳输出。此选项与应力、应变和截面力输出相关。
