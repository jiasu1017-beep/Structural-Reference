# 62.1 理解datum几何体的作用







在建模过程中，您可能需要模型本身不存在的特定几何体（如顶点或边）。您可以使用Datum工具集创建所需的几何体，称为datum。您可以创建datum点、datum轴、datum平面或datum坐标系。
- 当系统提示您选择点时，您可以从零件、datum点或datum坐标系的原点中进行选择。
- 当系统提示您选择边时，您可以从零件、datum轴或datum坐标系的某个轴中进行选择。
- 当系统提示您选择面时，您可以从零件或datum平面中进行选择。
- 当系统提示您选择坐标系时，您必须选择datum坐标系。

使用datum平面作为草绘平面的一个示例如[图62--1](pt06ch62s01.md#dtm-dpl-1)所示的零件。

**图62–1** 创建草绘平面。

为了在弯曲表面上进行盲切，您需要在与弯曲表面相切的平面上草绘轮廓。所需的平面不存在，但您可以使用Datum工具集创建一个，并在结果平面上进行草绘。[图62--2](pt06ch62s01.md#dtm-dpl-2)显示了datum平面和由此产生的切割特征。

**图62–2** 结果特征。

![](../graphics/dtm-dpl-2-nls.png)

Abaqus/CAE不会在datum几何体上生成网格，datum几何体对模型的分析没有影响；它只是为了帮助您构建复杂几何体而提供的便利。如果您想向模型本身添加几何体（如沿边的顶点或穿过体的平面），应该使用Partition工具集。

您可以通过相对于现有几何体（如顶点、平面和边）或其他datum几何体来定义datum来创建datum。您可以在"Part"模块或"Property"模块中的零件上创建datum，也可以在其他模块中的装配上创建datum。例如，您可以在"Part"模块中定义一个穿过零件两个选定点的datum轴，并使用该轴在"Assembly"模块中对零件实例进行对齐。datum是一个特征；并且，与所有特征一样，它可以进行编辑、删除、抑制和恢复。同样，当装配或零件重新生成时，datum也会重新生成。您还可以使用显示组来显示或隐藏视口中的datum几何体。
