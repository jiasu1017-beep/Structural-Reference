# 15.4 理解相互作用属性





您可以定义一组称为相互作用属性的数据，这组数据被相互作用引用但独立于相互作用；例如，定义接触期间摩擦系数的系数。一相互作用属性可以被许多不同的相互作用引用。

您可以创建以下类型的相互作用属性：

**接触**

接触相互作用属性可以定义切向行为（摩擦和弹性滑移）和法向行为（硬接触、软接触或阻尼接触和分离）。此外，接触属性可以包含阻尼、热传导、热辐射和摩擦热生成的信息。接触相互作用属性可以被通用接触、表面对表面接触或自接触相互作用引用。有关定义此类相互作用属性的详细说明，请参见「定义接触相互作用属性」，第15.14.1节。

**膜条件**

膜条件相互作用属性定义膜系数与温度和场变量的函数关系。膜条件相互作用属性只能被膜条件相互作用引用。有关定义此类相互作用属性的详细说明，请参见「定义膜条件相互作用属性」，第15.14.2节。

**空腔辐射**

空腔辐射相互作用属性定义空腔的发射率与温度和场变量的函数关系。空腔辐射相互作用属性只能被空腔辐射相互作用引用。有关定义此类相互作用属性的详细说明，请参见「定义空腔辐射相互作用属性」，第15.14.3节。

**流体腔**

流体腔相互作用属性定义占据腔的流体类型和流体属性。您可以选择液压流体或气动流体。液压流体必须包括流体密度；它们可能包括流体体积模量、热膨胀系数和其他温度相关数据。气动流体必须包括理想气体分子量，它们可能包括摩尔热容（仅限Abaqus/Explicit）。有关定义此类相互作用属性的详细说明，请参见「定义流体腔相互作用属性」，第15.14.4节。

**流体交换**

流体交换相互作用属性定义流体从腔到环境或从一个腔到另一个腔的流动。您可以基于体积粘度、质量通量、质量流率泄漏、体积通量或体积流率泄漏来定义流体交换。有关定义此类相互作用属性的详细说明，请参见「定义流体交换相互作用属性」，第15.14.5节。

**声阻抗**

声阻抗相互作用属性定义表面阻抗，或声学分析中压力与表面位移和速度法向分量之间的比例系数。声阻抗相互作用属性只能被声阻抗相互作用引用。有关定义此类相互作用属性的详细说明，请参见「定义声阻抗相互作用属性」，第15.14.6节。

**入射波**

入射波相互作用属性定义入射波的速度和波载荷的其他特征。入射波相互作用属性只能被人射波相互作用引用。有关定义此类相互作用属性的详细说明，请参见「定义入射波相互作用属性」，第15.14.7节。

**致动器/传感器**

致动器/传感器相互作用属性提供传递给与致动器/传感器相互作用一起使用的[`UEL`](../sub/sub-link.md#sub-xsl-uel)用户子程序的`PROPS`、`JPROPS`、`NPROPS`和`NJPROPS`变量。有关定义此类相互作用属性的详细说明，请参见「定义致动器/传感器相互作用属性」，第15.14.8节。
