# 69.6.4 移除冗余实体







导入的部件可以包含未连接到边的顶点或沿直边定位的顶点。类似地，导入的部件可以包括未连接到面或内部边的冗余边。冗余顶点和边不会改变部件的形状或面积，也不需要完整定义，如下图所示：

![](../graphics/prt-repair-redundant-nls.png)

您可以使用 **Remove redundant entities**（移除冗余实体）工具从所选区域移除冗余顶点和边。查询工具集提供了一组几何诊断工具，使您能够定位无效和不精确几何的区域。 有关更多信息，请参阅["在 Part 模块中使用查询工具集，" 第 11.16.4 节"](pt03ch11s16s04.md)。移除冗余实体的操作作为部件的特征存储；因此，您可以使用模型树恢复您删除的顶点和边。

**移除冗余实体：**

1. 从主菜单栏中选择 ****Tools**![](../graphics/images/arrow.gif)**Geometry Edit****（工具→几何编辑）。Abaqus/CAE 显示 **Geometry Edit**（几何编辑）对话框。**提示：**如果您使用的是 Part（部件）模块，您也可以使用位于模块工具箱中编辑工具处的 ![](../graphics/ico_repairRemoveRedundant.png) 工具来移除冗余实体。 有关 Part（部件）模块工具箱中编辑工具的图表，请参阅["使用几何编辑工具集，" 第 69.1 节"](pt06ch69s01.md)。
2. 从对话框中选择 **Edge**（边）类别和 **Remove redundant entities**（移除冗余实体）方法。
3. 选择 Abaqus/CAE 应从中移除冗余边和顶点的区域。您可以从视口选择区域或选择包含区域的现有集合。**注意：**默认选择方法基于您最近使用的选择方法。要恢复为其他方法，请点击提示区域右侧的 **Select in Viewport**（在视口中选择）或 **Sets**（集合）。
4. 如果您不想移除顶点，请在提示区域中关闭 **Remove redundant edge vertices**（移除冗余边顶点）。
5. 从提示区域点击 **Done**（完成）。Abaqus/CAE 从所选区域移除冗余实体，并提示您更新部件的有效性。
6. 执行以下操作之一： - 点击 **Yes**（是）更新部件的有效性。（每当您移除冗余实体时都应该更新部件的有效性。） - 点击 **No**（否）跳过更新并继续您的编辑。 - 点击 **Cancel**（取消）取消当前编辑操作中的更改。您也可以随时通过在模型树中的部件上点击鼠标按钮 3 并从出现的菜单中选择 **Update validity**（更新有效性）来更新部件的有效性。
