# 14.3.2 线性和非线性过程





**步骤管理器**通过缩进线性扰动步骤的名称和过程描述来区分常规非线性步骤和线性扰动步骤。常规非线性分析步骤定义顺序事件：一个常规步骤结束时模型的状态为下一个常规步骤的开始提供初始状态。线性扰动分析步骤提供在最后一个常规非线性步骤结束时达到的状态附近模型的线性响应。你在**创建步骤**对话框中选择过程时，使用**过程类型**字段在**通用**和**线性扰动**步骤之间进行选择。

对于分析中的每个步骤，**步骤管理器**还指示Abaqus是否会考虑大位移和大变形的非线性效应。如果在步骤期间由于加载导致模型中的位移相对较小，则效应可能小到可以忽略。但是，在模型上的载荷导致大位移的情况下，非线性几何效应可能变得重要。步骤的**Nlgeom**设置决定Abaqus是否会在该步骤中考虑几何非线性。

**Nlgeom**设置在Abaqus/Explicit步骤中默认开启，在Abaqus/Standard步骤中默认关闭。步骤序列和当前的**Nlgeom**设置决定你是否可以更改特定步骤中的**Nlgeom**设置。例如，如果Abaqus已经在考虑几何非线性，则**Nlgeom**设置在所有后续步骤中都为开启状态，你不能将其关闭。在允许的情况下，以下方法允许你更改步骤的**Nlgeom**设置：
- 在**步骤编辑器**中点击**基本**选项卡，并切换**Nlgeom**设置。
- 从主菜单栏选择****Step**![](../graphics/images/arrow.gif)**Nlgeom****。
- 在**步骤管理器**中点击**Nlgeom**。
