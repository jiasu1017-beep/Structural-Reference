### 概述

除状态文件中的信息外，只能通过后处理从Abaqus/Explicit获取结果。

本节中的表格列出了Abaqus/Explicit中所有可用的输出变量。这些输出变量可以请求输出到结果（`.fil`）文件（请参见["输出到数据和结果文件，" 4.1.2节](pt02ch04s01aus39.md)），或作为场类型或历史类型输出到输出数据库（`.odb`）文件（请参见["输出到输出数据库，" 4.1.3节](pt02ch04s01aus40.md)）。通常，可以作为场类型或历史类型输出请求到ODB格式输出数据库的输出变量也可以SIM格式请求输出（请参见["输出中的输出数据库" 4.1.1节](pt02ch04s01aus38.md#usb-out-ooutput-formats)）。当请求将输出变量输出到结果文件时，Abaqus/Explicit首先将这些变量输出到选定结果（`.sel`）文件，然后在分析完成后将选定结果文件转换为结果文件。

### 输出变量描述中使用的符号

变量描述后面的`.fil`、`.odb` Field和`.odb` History表示输出变量的可用性。`.fil`指输出到结果文件。如果类别名称后面显示"yes"，则可以将输出变量写入相应的文件；"no"表示该变量对该文件不可用。

### 方向定义

方向定义取决于变量类型。

#### 元素变量的方向定义

对于应力、应变和类似材料变量的分量，1、2和3指向正交坐标系中的方向。对于实体元素，这些是全局方向；对于壳单元和膜单元，这些是表面方向；对于梁单元和管道单元，这些是轴向和横向方向。但是，如果与请求输出的元素关联了局部方位（["方位，" 2.2.5节](pt01ch02s02aus15.md)），则1、2和3是局部方向。

#### 节点变量的方向定义

对于节点变量，1、2和3指向全局方向（1=*X*，2=*Y*，3=*Z*，轴对称元素除外，在轴对称情况下1=*R*，2=*Z*）。即使在节点处定义了局部坐标系（["变换坐标系，" 2.1.5节](pt01ch02s01aus09.md)），结果文件和选定结果文件中的数据仍以全局方向输出。

如果为定义了局部坐标系的节点请求节点场输出，则表示从全局方向旋转的四元数将写入输出数据库。Abaqus/CAE自动使用此四元数将节点结果变换到局部方向。写入输出数据库的节点历史数据始终以全局方向存储。

#### 积分变量的方向定义

对于通过在表面上积分获得的总力、总矩和类似变量的分量，1、2和3指向正交坐标系中的方向。如果表面直接为积分输出请求指定，则使用固定的全局坐标系。如果表面由与积分输出请求关联的积分输出截面定义标识（请参见["积分输出截面定义，" 2.5.1节](pt01ch02s05aus23.md)），则可以指定初始配置中的局部坐标系，并且可以随变形平移或旋转。

### 分布载荷输出和用户子程序

可以为["载荷，" 34.4节](pt07ch34s04.md)中讨论的许多分布载荷请求输出。但是，不显示通过用户子程序（请参见["Abaqus/Explicit子程序，" Abaqus User Subroutines Reference Guide的1.2节](../sub/sub-link.md#sub-rtn-explicit)）定义的这些载荷的贡献。

### 主值输出

可以请求应力、对数应变和名义应变的主值输出。可以获取所有主值或最小值、中间值或最大值。张量*ABC*的所有主值通过请求*ABC*P获得，最小、中间和最大主值分别通过请求*ABC*P1、*ABC*P2和*ABC*P3获得。对于三维、平面应变和轴对称元素，获取所有三个主值。对于平面应力、膜和壳单元，仅获取面内主值用于历史类型输出，且无法请求面外主值。对于场类型输出，通过Abaqus/CAE获取所有三个主值。无法获得梁单元、管道单元和桁架单元的主值，且无法请求塑性应变的主值。

如果为场类型输出请求了主值或不变量，则输出请求将替换为相应张量分量的输出请求。Abaqus/CAE从这些分量计算所有主值和不变量。如果需要历史类型输出的主值，则必须明确请求，因为Abaqus/CAE不对历史数据进行计算。

### 张量输出

作为场类型输出写入输出数据库的张量变量，作为["方位，" 2.2.5节](pt01ch02s02aus15.md)中给出的约定定义的默认方向（实体元素为全局方向，壳单元和膜单元为表面方向，梁单元和管道单元为轴向和横向方向）或用户定义的局部系统中的分量写入。Abaqus/CAE从这些分量计算所有主值和不变量。有关不同类型张量变量的说明，请参见["写入场输出数据，" Abaqus Scripting User's Guide的9.6.4节](../cmd/cmd-link.md#cmd-odb-intro-write-field-pyc)。

张量变量的分量以单精度写入输出数据库。因此，在计算变量主值时可能会发生少量精度舍入误差。例如，当解析为零的值被计算为相对较小但非零的值时，可能会观察到这种舍入误差。

### 请求分量输出

可以在输出数据库中请求变量的各个分量作为历史类型输出，用于Abaqus/CAE中的*X–Y*绘图。场类型输出不提供单个分量请求。如果需要特定分量用于Abaqus/CAE中的等值线绘制，请请求通用变量的场输出（例如，S表示应力）。然后可以在Abaqus/CAE的Visualization模块中请求此场输出的各个分量的输出。

### 元素积分点变量

您可以请求将元素积分点变量输出到结果或输出数据库文件（请参见["输出到数据和结果文件中的元素输出，" 4.1.2节](pt02ch04s01aus39.md#usb-out-oprintfile-elementoutput)和["输出到输出数据库中的元素输出，" 4.1.3节](pt02ch04s01aus40.md#usb-out-odboutput-elementoutput)）。
