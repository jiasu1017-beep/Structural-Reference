# 4.1.10 单活塞发动机模型的子结构分析

**产品：** Abaqus/Standard   

本示例说明了在 Abaqus 中使用子结构功能来有效地建模经历大运动但仅表现出小线性变形的刚柔耦合多体系统。这些模型还说明了如何根据建模需求在部件的完整网格表示、子结构表示和刚体表示之间切换。执行了静态和动态分析，并比较了各种模型的结果。

### 几何与材料

这里讨论的多体系统是简化单活塞发动机模型。该模型由多个使用连接器元件约束在一起的部件组成。每个部件建模为完全可变形的常规网格、刚体或子结构。

曲轴的两个部件使用 BEAM 连接器元件刚性连接。曲轴和活塞头之间的连杆在两端具有 HINGE 连接。曲轴和凸轮之间的齿轮传动使用两个旋转自由度之间的方程约束建模。

### 模型

考虑了以下三个模型：
1. 所有部件都使用常规网格建模。这是最大的模型。
2. 所有部件都建模为刚体。
3. 某些部件建模为子结构，其余部件建模为刚体。

### 结果与讨论

进行了静态和直接积分隐式动态分析。监测气门的垂直位移和凸轮从动件上的接触力。

### 输入文件

- crank_substr_sta.inp - STATIC 子结构全局分析
- crank_substr_dyn.inp - DYNAMIC 子结构全局分析
- crank_substr_sta_fric.inp - 带摩擦效应的 STATIC 子结构全局分析
- crank_substr_dyn_fric.inp - 带摩擦效应的 DYNAMIC 子结构全局分析
- crank_rb_sta.inp - 仅带刚体的 STATIC 分析
- crank_all_def_sta.inp - STATIC 常规网格分析
- crank_all_def_dyn.inp - DYNAMIC 常规网格分析
