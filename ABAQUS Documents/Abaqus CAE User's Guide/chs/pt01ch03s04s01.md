# 3.4.1 什么是基本管理器？





基本管理器由对象列表和一系列按钮组成；您使用按钮对从列表中选择的对象执行任务，或向列表添加新对象。

[图3-13](pt01ch03s04s01.md#int-manager)显示了**材料管理器**，这是Abaqus/CAE中使用的基本管理器的示例。

**图3-13** **材料管理器**。

![](../graphics/int-manager-nls.png)

左侧的列表框显示了您在当前模型上下文中定义的所有材料。您使用右侧的按钮创建新的材料定义，并编辑、复制、重命名和删除现有材料定义。**关闭**按钮用于关闭管理器对话框。

通常，管理器提供有关对象的更多信息，而不仅仅是其名称；例如，在作业模块中，**作业管理器**提供有关当前正在执行的作业的信息，并提供允许您写入输入文件、提交作业、监控分析或查看给定作业的输出文件的按钮。[图3-14](pt01ch03s04s01.md#ana-jobman2)显示了**作业管理器**。

**图3-14** **作业管理器**。

![](../graphics/ana-jobman-nls.png)

您可以使用主菜单栏上的下拉菜单执行与管理器相同的大多数任务；例如，[图3-15](pt01ch03s04s01.md#int-menu)显示了与**作业管理器**对应的菜单项。

**图3-15** 与**作业管理器**对应的菜单项。

![](../graphics/int-menu-nls.png)

从主菜单栏选择管理操作后，过程与您单击管理器对话框内的相应按钮完全相同。此外，您可以使用管理器执行的大多数任务也可以通过在模型树中的对象上单击鼠标按钮3来执行。有关更多信息，请参阅["使用模型树和结果树，" 第3.5节"](pt01ch03s05.md)。

选择使用菜单、对话框还是模型树由您决定。通常，如果您正在执行孤立操作，菜单更方便；当您按顺序执行多个操作时、需要浏览长对象列表时或需要快速访问某些管理器显示的附加信息时，管理器对话框的优势变得明显。模型树为您提供模型的图形概览，并允许您执行操作而无需切换模块。此外，模型树允许您使用拖动选择来选择多个项目；例如，您可以选择多个集合进行合并或选择多个零件进行删除。
