# 12.15.5 分配壳/膜法线方向







当您创建具有壳区域的部件或具有线区域的对称部件时，Abaqus会为这些区域的法线分配方向。您可以反转这些区域的法线方向。此外，您还可以反转导入部件或选定孤立元素的法线方向。

您还可以反转皮肤增强的法线；但是，如果在几何部件的表面上定义了多个皮肤，则不能仅反转其中一个皮肤的法线，而必须反转为该表面定义的所有皮肤的法线。如果具有多个皮肤的表面是网格部件的特征，则可以反转单个皮肤的法线，而无需反转为该表面定义的其他皮肤的法线。

当您反转区域的法线方向时，定义材料方向的局部法线会被反转。您可以使用查询工具集来验证选定区域的材料方向是否正确。更多信息，请参见"使用查询工具集获取分配信息"，第12.19节。

**为部件分配壳/膜法线：**

1. 如果要将法线分配到的部件在当前视口中不可见，请在上下文栏中的**部件**列表中单击所需部件的名称。选定部件将出现在当前视口中。
2. 从主菜单栏中，选择****分配**![](../graphics/images/arrow.gif)**单元法线****。**提示：**您也可以单击属性模块工具箱中的![](../graphics/ico_assignShellNormal.png)工具，或在网格模块中选择****网格**![](../graphics/images/arrow.gif)**方向**![](../graphics/images/arrow.gif)**法线****。Abaqus/CAE使用着色渲染样式显示部件。表面法线与壳法线重合的壳侧（顶面）以棕色显示；相反侧（底面）以紫色显示。
3. 如果选定部件是基于几何的，请使用以下方法之一选择要反转其法线的区域：- 从视口中选择区域，然后单击鼠标按钮2。（更多信息，请参见第6章，"在视口中选择对象"。）**提示：**您可以使用**选择**工具栏中的工具来限制可以在视口中选择的对象类型。有关更多信息，请参见"使用选择选项"，第6.3节。如果选择**皮肤**作为对象类型，可以将截面分配给整个皮肤或其一个或多个面。从提示区域中，选择**（拾取整个皮肤）**或**（拾取部分皮肤）**，然后在视口中进行选择。如果选定部件有多个皮肤，Abaqus/CAE会在您进行选择时在提示区域中显示模糊拾取选项。- 要从现有集合列表中进行选择，请执行以下操作：1. 单击提示区域右侧的**集合**。Abaqus/CAE显示**区域选择**对话框，其中包含可用部件集的列表。2. 选择感兴趣的部件集，然后单击**继续**。**注意：**默认选择方法基于您最近使用的选择方法。要恢复为另一种方法，请单击提示区域右侧的**在视口中选择**或**集合**。Abaqus/CAE会反转选定区域的壳/膜法线方向。
4. 如果选定部件是网格部件，请使用以下方法之一选择要反转其法线的壳单元：**选择单个单元：**1. 单击提示区域中**选择方法**字段旁边的箭头，然后从出现的列表中选择**单独**。2. 选择要反转其法线的单元。3. 按**[Shift]+单击**其他单元以将其添加到您的选择中。4. 如有必要，按**[Ctrl]+单击**已选单元以取消选择它们。5. 完成选择单元后，单击鼠标按钮2。**指定现有单元集**1. 单击提示区域右侧的**集合**。Abaqus/CAE显示**区域选择**对话框，其中包含您创建的单元集列表。2. 选择要编辑的单元集，然后单击**继续**。**注意：**默认选择方法基于您最近使用的选择方法。要恢复为另一种方法，请单击提示区域右侧的**在视口中选择**或**集合**。**使用角度方法选择单元：**1. 从提示区域的字段中，选择**按角度**。2. 输入一个角度（0到90之间），然后选择一个单元面。Abaqus/CAE会从选定面开始选择每个相邻壳单元，直到单元面之间的角度等于或超过您输入的角度。（更多信息，请参见"使用角度和特征边缘方法选择多个对象"，第6.2.3节。）3. 使用**按角度**方法后，您可以按**[Shift]+单击**其他单元以将其添加到您的选择中，或按**[Ctrl]+单击**已选单元以将其从选择中移除。（更多信息，请参见"组合选择技术"，第6.2.8节。）4. 完成选择单元后，单击鼠标按钮2。**提示：**您可以使用**选择**工具栏中的工具来限制可以在视口中选择的对象类型。更多信息，请参见第6章，"在视口中选择对象"。
5. 从提示区域中，选择反转壳/膜法线的方法：- 单击**全部翻转**以反转所有选定单元的法线。- 单击**选择法线**以更改选定单元的法线，使其指向您指定的参考单元法线的相同方向。
6. 如果在上一步中选择了**选择法线**，请执行以下操作：1. 在视口中选择参考单元。2. 在提示区域中单击**确定**。Abaqus/CAE会更改选定单元的法线，使其指向与参考单元法线相同的方向。
7. 根据需要重复上述步骤以反转其他壳/膜法线。完成法线方向分配后，使用以下方法之一退出分配模式：- 如果您正在从视口中选择部件区域或单元，请单击鼠标按钮2或单击提示区域中的**完成**。- 如果您正在从**区域选择**对话框中选择预定义集合，请单击**取消**关闭对话框，然后单击鼠标按钮2或单击提示区域中的**完成**。Abaqus/CAE会恢复为在开始反转壳/膜法线过程之前选择的渲染样式。
