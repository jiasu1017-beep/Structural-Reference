# 6.1.3 理解几何对象和物理对象之间的对应关系





当你在视口中选择几何对象时，理解每个对象代表的物理结构是很重要的。构成模型的几何对象——单元体、面、边和顶点——可以根据它们所处的空间代表不同的物理结构。

例如，梁和其他线框部件由几何模型中的边表示（请参阅[图 6-2](pt01ch06s01s03.md#pic-beams)）。

**图 6-2** 选择线框部件。

![](../graphics/pic-beams-nls.png)

这些部件的端表面由边两侧的顶点表示，圆周表面由连接顶点的线表示。要选择线框部件，你可以点击边，如果需要，Abaqus/CAE 会提示你指定感兴趣的表面。

同样，轴对称壳体也由几何模型中的边表示（请参阅[图 6-3](pt01ch06s01s03.md#pic-shellaxis)）。

**图 6-3** 选择轴对称壳体。

![](../graphics/pic-shellaxis-nls.png)

你可以通过点击视口中的边来选择轴对称壳体，如果需要，Abaqus/CAE 会提示你指定壳体的内表面或外表面。如果你要将预定条件或接触定义应用到表面，必须选择内表面或外表面。例如，如果要对壳体施加压力载荷，必须指定壳体的哪一侧应该接收载荷。

有关选择表面的更多信息，请参阅["指定区域的特定侧面或端部，" 第 73.2.5 节](pt06ch73s02s05.md)。有关建模空间的更多信息，请参阅["部件和特征之间的关系，" 第 11.3.1 节](pt03ch11s03s01.md)和["部件建模空间，" 第 11.4.1 节](pt03ch11s04s01.md)。
