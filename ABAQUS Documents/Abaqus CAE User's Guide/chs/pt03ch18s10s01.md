# 18.10.1 创建和编辑几何限制





您可以创建对Optimization模块对模型拓扑所做更改施加限制的几何限制。 几何限制包括不能移除材料的冻结区域，以及制造限制，例如会阻止优化模型从模具中移除的倒扣限制。

**创建或编辑几何限制：**

1. 从主菜单栏，选择 ****Geometric Restriction**![](../graphics/images/arrow.gif)**Create****。 出现**Create Geometric Restriction**对话框。 **提示：**您可以通过另外两种方式启动**Create**过程： - 单击**Geometric Restriction Manager**中的**Create**。 （您可以通过从主菜单栏选择 ****Geometric Restriction**![](../graphics/images/arrow.gif)**Manager**** 来显示**Geometric Restriction Manager**。） - 单击Optimization模块工具箱中的 ![](../graphics/icoR_optGeomRest.png) 工具。
2. 从出现的**Create Geometric Restriction**对话框中，输入几何限制的名称。
3. 选择要应用的几何限制类型，然后单击**Continue**。
4. 从视口中，选择将应用几何限制的区域，或单击**Done**将几何限制应用到整个模型。 默认情况下，Abaqus/CAE允许您选择模型的所有部分。 要选择面或单元，请使用**Selection**工具栏将可选择的对象类型更改为**Face**或**Cells**。 有关更多信息，请参阅["根据对象类型过滤您的选择，" 第6.3.2节](pt01ch06s03s01.md)。 如果您宁愿从现有集列表中选择，请执行以下操作： 1. 单击提示区域右侧的**Sets**。 Abaqus/CAE显示包含可用集列表的**Region Selection**对话框。 2. 选择感兴趣的集，然后单击**Continue**。 **注意：**默认选择方法基于您最近使用的选择方法。 要恢复为另一种方法，请单击提示区域右侧的按钮——**Select in Viewport**或**Sets**。
5. 完成选择几何限制区域后，在提示区域单击**Done**。 有关选择对象的更多信息，请参阅[第6章，"在视口中选择对象"](pt01ch06.md)。 出现**Edit Geometric Restriction**对话框。 对话框的内容取决于您正在创建的几何限制类型。
6. 单击**OK**保存数据并退出编辑器。
