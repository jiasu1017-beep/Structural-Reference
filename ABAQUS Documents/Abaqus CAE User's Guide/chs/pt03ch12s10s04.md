# 12.10.1 指定热导率







您可以使用**编辑材料**对话框指定各向同性、正交异性或完全各向异性的热导率。有关更多信息，请参见"热导率"，Abaqus分析用户指南第26.2.2节。

**指定热导率：**

1. 从**编辑材料**对话框的菜单栏中，选择**热**![](../graphics/images/arrow.gif)**热导率**。（有关显示**编辑材料**对话框的信息，请参见"创建或编辑材料"，第12.7.1节。）
2. 单击**类型**字段右侧的箭头，指定热导率的方向依赖性。
3. 切换**使用温度相关数据**将热导率定义为温度的函数。**数据**表中会出现一列标为**温度**。
4. 单击**场变量数量**字段右侧的箭头增加或减少热导率所依赖的场变量数量。
5. 在**数据**表中输入适用数据：**热导率**各向同性热导率，*k*。（单位为[JT1L1![](../graphics/usi_eqn00065.gif)1](../popups/usb-int-iconventions-unitsym.md)。）**k11、k22和k33**正交异性热导率的三个值，![](../graphics/usi_eqn00311.gif)、![](../graphics/usi_eqn00312.gif)和![](../graphics/usi_eqn00313.gif)。（单位为[JT1L1![](../graphics/usi_eqn00065.gif)1](../popups/usb-int-iconventions-unitsym.md)。）**k11、k12、k22、k13、k23和k33**各向异性热导率的六个值，![](../graphics/usi_eqn00311.gif)、![](../graphics/usi_eqn00314.gif)、![](../graphics/usi_eqn00312.gif)、![](../graphics/usi_eqn00315.gif)、![](../graphics/usi_eqn00316.gif)和![](../graphics/usi_eqn00313.gif)。（单位为[JT1L1![](../graphics/usi_eqn00065.gif)1](../popups/usb-int-iconventions-unitsym.md)。）**温度**温度。**场变量* n ***预定义的场变量。您可能需要展开对话框以查看**数据**表中的所有列。有关如何输入数据的详细信息，请参见"输入表格数据"，第3.2.7节。
6. 单击**确定**关闭**编辑材料**对话框。或者，您可以从**编辑材料**对话框的菜单中选择其他材料行为进行定义（有关更多信息，请参见"浏览和修改材料行为"，第12.7.2节）。
