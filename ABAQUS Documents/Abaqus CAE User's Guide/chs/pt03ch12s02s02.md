# 12.2.2 定义轮廓





轮廓指定与梁截面横截面形状和尺寸相关的属性（例如，横截面积和惯性矩）。当您定义梁截面时，必须在截面定义中包含对轮廓的引用。

您可以创建以下类型的轮廓：

**基于形状的轮廓**

基于形状的轮廓定义梁横截面的特定形状和尺寸。Abaqus使用基于形状的轮廓提供的信息来计算截面的工程特性。

您可以通过首先从形状选项列表中选择，然后指定该特定形状的尺寸来创建这种类型的轮廓。例如，如果您选择箱形，则必须随后指定箱形的高度和宽度以及四个壁的厚度。您可以从以下形状选项中选择：
- 任意
- 箱形
- 圆形
- 六角形
- I、L、T
- 管形
- 矩形
- 梯形

形状选项如图[12-1](pt03ch12s02s02.md#usi-prp-prop-shapes)所示。

**广义轮廓**

广义轮廓直接指定截面的工程特性。您可以通过指定面积、惯性矩、扭转常数，以及适用时的扇形矩和翘曲常数的值来创建广义轮廓。有关更多信息，请参见[Abaqus Analysis User's Guide的第29.3.7节，"使用广义梁截面定义截面行为"](../usb/usb-link.md#usb-elm-eusingbeamgensect)。

您创建的每个轮廓都有自己的名称，独立于任何特定梁截面；您可以在任意多个梁截面中引用单个轮廓。在您将梁截面和梁取向分配给部件后，您可以使用部件显示选项来查看基于形状或广义的梁轮廓的理想化表示。显示梁轮廓有助于检查正确的轮廓已被分配给特定区域，以及分配的梁取向是否导致预期的轮廓方向。有关更多信息，请参见["控制梁轮廓显示，"第76.7节](pt07ch76hla06.md)。
