# 2.2.4 上下文栏





上下文栏位于画布和绘图区域的上方；您可以使用它执行以下操作：

**选择当前模块**

上下文栏上的**模块**列表允许您在模块之间移动。（有关更多信息，请参阅["什么是模块？," 第2.3节"](pt01ch02s03.md)。）[图2-4](pt01ch02s02s01.md#usi-contextbar)显示了上下文栏。要移动到不同的模块，您可以从列表中选择（右侧的箭头），或单击上下箭头（左侧）移动到上一个或下一个模块。

**图2-4** 零件模块中的上下文栏。

![](../graphics/usi-contextbar-nls.png)

**注意：**Abaqus/Viewer仅包含可视化模块。

**选择特定于模块的项目**

当您在模块之间移动时，Abaqus/CAE会在上下文栏上显示额外的项目，帮助您选择当前操作的上下文。例如，当您在零件模块或网格模块中时，Abaqus/CAE会在上下文栏中显示**零件**列表。**零件**列表包含模型中的每个零件；您可以使用它检索特定零件。这些列表还包括上下导航箭头，允许您移动到列表中的上一个或下一个项目。

上下文栏还允许您在模型数据库中的模型之间移动，或更改与当前视口关联的输出数据库。上下文栏中的额外项目是您正在工作的模块的功能。

上下文栏中显示的项目始终引用当前视口，当前视口由深灰色标题栏指示。例如，如果您在不同的视口中显示不同的零件，上下文栏会指示当前视口中显示的零件名称。
