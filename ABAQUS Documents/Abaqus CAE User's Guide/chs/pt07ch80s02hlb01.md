# 80.2.1 创建或编辑视图切割





您可以基于平面创建视图切割。此外，在可视化模块中，您可以基于圆柱体、球体或对应于任意轮廓变量常量的等值面创建视图切割。默认情况下，会沿X-、Y-和Z平面创建切割。

编辑等值面视图切割时，Abaqus/CAE会更新其定义以反映当前的结果选项和变量选择。

**创建或编辑视图切割的步骤：**

1. 找到用于创建或编辑视图切割的选项。
   - 要创建视图切割，请从主菜单栏中选择****工具**![](../graphics/images/arrow.gif)**View Cut**![](../graphics/images/arrow.gif)**Create****。出现 **Create Cut** 对话框。
   - 要编辑先前创建的视图切割，请从主菜单栏中选择****工具**![](../graphics/images/arrow.gif)**View Cut**![](../graphics/images/arrow.gif)**Edit****。从出现的菜单中选择要编辑的视图切割。出现 **Edit Cut** 对话框。
   **提示：**您也可以使用视图切割管理器来创建或编辑视图切割。从主菜单栏中选择****工具**![](../graphics/images/arrow.gif)**View Cut**![](../graphics/images/arrow.gif)**Manager****。要创建视图切割，请单击 **Create**。要编辑视图切割，请从列表中选择它，然后双击它或单击 **Edit**。
2. 输入视图切割的名称。有关命名对象的更多信息，请参见["使用基本对话框组件，" 第3.2.1节](pt01ch03s02s01.md)。您无法在 **Edit Cut** 对话框中更改名称，但可以重命名视图切割（请参见["复制、重命名和删除视图切割，" 第80.2.5节](pt07ch80s02hlb05.md)）。
3. 如果编辑器中出现 **Shape** 文本字段，请选择切割形状。单击 **Shape** 文本字段旁边的箭头，从出现的列表中选择形状：**Plane**、**Cylinder**、**Sphere** 或 **Isosurface**。一旦创建了切割，就无法更改形状。**Shape** 文本字段仅对在可视化模块中创建的视图切割可编辑。
4. 如果您选择 **Isosurface** 作为切割形状，则切割将在当前主场输出变量的常量值上创建。切换 **Override primary variable averaging** 可为选定的视图切割应用100%的变量平均阈值，或关闭此选项以使用 **Result Options** 对话框中 **Averaging** 选项指定的变量平均阈值；有关更多信息，请参见["控制结果平均，" 第42.6.6节](pt05ch42s04hlb02.md)。无法为存储在表面上的变量创建等值面切割，因为没有可用于模型内部的信息。请参见["选择主场输出变量，" 第42.5.3节](pt05ch42s03hlb03.md)，了解更改主场输出变量的信息。
5. 默认情况下，切割在显示的帧上定义，结构在变形时穿过切割。切换 **Follow model deformation** 可定义拉格朗日切割；即切割在参考框架上定义，当您显示不同帧时与模型一起变形。选择 **Reference frame** 为分析的第一个、最后一个或当前帧。**Follow model deformation** 选项仅对可视化模块中的视图切割可用。
6. 对于可视化模块中的平面、圆柱或球形视图切割，切换到您希望用来定义视图切割值的方法：**Key-in** 或 **From CSYS**（后者仅在输出数据库包含坐标系或您在当前会话中创建了坐标系时才可用）。
   1. 如果您选择 **Key-in** 方法，请根据切割形状输入以下点的坐标：
      - 对于 **Plane** 切割，请输入原点的坐标；法线轴；以及可选的轴2。轴2是位于平面内的轴；视图切割可以绕此轴或绕轴3（定义为垂直于法线轴和轴2的轴）旋转。请参见["重新定位视图切割，" 第80.2.4节](pt07ch80s02hlb04.md)。
      - 对于 **Cylinder** 切割，请输入原点和圆柱轴的坐标。
      - 对于 **Sphere** 切割，请输入原点的坐标。
   2. 如果您选择 **From CSYS** 方法，请从出现的列表中选择坐标系（带星号的系统已保存到当前输出数据库），然后根据切割形状选择以下内容：
      - 对于 **Plane** 切割，请选择坐标系的 **Axis 1**、**Axis 2** 或 **Axis 3** 作为法线轴。切割可以绕其他两个坐标系轴中的任意一个旋转；请参见["重新定位视图切割，" 第80.2.4节](pt07ch80s02hlb04.md)。
      - 对于 **Cylinder** 切割，请选择坐标系的 **Axis 1**、**Axis 2** 或 **Axis 3** 作为圆柱轴。
      - 对于 **Sphere** 切割，坐标系的原点将成为切割的原点；您无需选择其他内容。
   **From CSYS** 选项仅对可视化模块中的视图切割可用。
7. 单击 **OK** 关闭 **Create Cut** 或 **Edit Cut** 对话框。创建新的视图切割时，它会自动激活（即用于在当前视口中显示模型）。默认情况下，切割位于模型中间的位置。默认情况下，视图切割仅在会话期间保留。如果您想保留已定义的视图切割以供后续会话使用，可以将其保存到XML文件、模型数据库或输出数据库。有关更多信息，请参见["管理会话对象和会话选项，" 第9.9节](pt02ch09s08.md)。
