### Element property definition

You must define the element's initial gap and initial void. In addition, for link elements you must define the element's width.

You can specify the thickness direction as part of the gasket section definition or by specifying a normal direction at the nodes; you can specify the element thickness as part of the gasket section definition. Otherwise, Abaqus/Standard will calculate the thickness direction and the thickness. For link elements the thickness direction is the direction from the first to the second node and the thickness is the distance between the nodes. For general elements the thickness direction is based on the midsurface of the element and the thicknesses at the integration points are based on the nodal positions. See ["Defining the gasket element's initial geometry," Section 32.6.4](pt06ch32s06alm49.md), for more details.
