# 74.2 创建流







流定义有两个组成部分：您要用作"耙"的位置的线段或路径定义，用于调查速度或涡量数据，以及您要显示流线流体流动数据的该耙上的点数。您可以通过从视口中选择节点、通过在对话框中输入坐标，或通过从会话中的路径定义中进行选择来指定耙位置。边列表路径不能用于流定义。

您不能将流保存到输出数据库文件。如果您想保留已定义的流，请在Abaqus/CAE中将该过程的定义记录为宏。有关宏的更多信息，请参见["管理宏," 第9.11节](pt02ch09hla02.md)。

**创建流的步骤：**

1. 在Visualization模块中，从主菜单栏选择****Tools**![](../graphics/images/arrow.gif)**Stream**![](../graphics/images/arrow.gif)**Create****。**提示：**您也可以通过使用Visualization模块工具箱中的![](../graphics/ico_streamline.png)工具来创建流。将出现"创建流"对话框。
2. 如果需要，在**Name**字段中为流定义指定自定义名称。
3. 如果需要，增加或减少**Number of points**字段中指定的点数。默认情况下，Abaqus/CAE创建一个具有五个点的耙。
4. 从**Specification**选项中，选择**By nodes**、**By coordinates**或**By path**。
5. 要指定流耙的位置，请执行以下操作之一：
   **直接从视口选择节点**
   1. 单击![](../graphics/ico_selectBlue.png)。Abaqus/CAE提示您选择第一个点。
   2. 选择耙的起始节点。
   3. 选择耙的结束节点。Abaqus/CAE用您的选择填充**Start**和**End**字段。
   **在对话框中输入坐标**
   1. 在**Start (X, Y, Z)**字段中，输入耙起始点的坐标。
   2. 在**End (X, Y, Z)**字段中，输入耙结束点的坐标。
   **指定路径作为流耙**
   从**Path name**列表中，选择要用作流耙的路径。
6. 单击**OK**。Abaqus/CAE创建流定义并将其显示在视口中。
