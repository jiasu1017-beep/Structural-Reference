# 14.1 Enhancements to the addData method







### 14.1 Enhancements to the *addData* method

**Product: **Abaqus/CAE  

**Benefits: **The *addData* method is enhanced for the C++ ODB API.

**Description: **For the C++ ODB API, the *addData* method is enhanced to support adding double precision nodal data. In addition, the performance of the *addData* method is improved for adding large amounts of data.
