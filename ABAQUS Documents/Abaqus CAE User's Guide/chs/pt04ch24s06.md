# 24.6 创建连接器几何、连接器截面和连接器截面分配






您可以在"相互作用"模块中创建装配级线框特征、连接器截面和连接器截面分配来对连接器进行建模。Abaqus/CAE 提供了两种连接器建模方法：您可以使用**连接器构建器**执行创建单个连接器所涉及的所有步骤，包括创建线框特征、连接器截面、连接器截面分配以及任何所需的参考点和基准坐标系；或者您可以通过在"相互作用"模块的不同对话框中创建线框、连接器截面和连接器截面分配来创建多个连接器。如果您选择后一种方法，则应在开始对连接器进行建模之前创建所需的参考点和基准坐标系。

[图 24-2](pt04ch24s06.md#adv-conn-modeltree) 中所示的模型树有助于理解您在装配相关模块中创建的参考点、基准坐标系和线框特征的组织，以及您在"相互作用"模块中创建的连接器截面和连接器截面分配。您可以使用"相互作用"模块中的查询工具集来获取选定线框的连接器分配信息。
