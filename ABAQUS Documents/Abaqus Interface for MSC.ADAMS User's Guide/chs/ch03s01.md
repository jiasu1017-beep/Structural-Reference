# 3.1 单位





MSC.ADAMS 程序要求用户定义组件模型中使用的单位，而 Abaqus 则不需要。因此，在创建模态中性文件时，用户必须明确声明模型中使用的单位。Abaqus Interface for MSC.ADAMS 中的方法与 ADAMS/View **Units Settings** 对话框中的方法非常相似。可以使用 Abaqus Interface for MSC.ADAMS 执行过程中的 **units** 选项指定预定义的单位系统。或者，可以使用 Abaqus Interface for MSC.ADAMS 执行过程中的 **length**、**mass**、**force** 和 **time** 选项指定单个长度、质量、力和时间单位。指定的任何单个单位都会覆盖单位系统中对应的单位。默认单位系统是 `mks`。[**units** 选项的有效单位系统见 [表 3-1](ch03s01.md#table-unitssystems)。

**表 3-1** 有效的单位系统。
| 单位系统 | 长度单位 | 质量单位 | 力单位 | 时间单位 |
| --- | --- | --- | --- | --- |
| `mks` | 米 | 千克 | 牛顿 | 秒 |
| `mmks` | 毫米 | 千克 | 牛顿 | 秒 |
| `cgs` | 厘米 | 克 | 达因 | 秒 |
| `ips` | 英寸 | 磅质量 | 磅力 | 秒 |

**length**、**mass**、**force** 和 **time** 选项的有效选项如下：

**长度单位**

长度单位的有效选项包括
- `meters`
- `millimeters`, `mm`
- `centimeters`, `cm`
- `kilometers`, `km`
- `inches`, `inch`, `in`
- `feet`, `foot`, `ft`
- `mile`

**质量单位**

质量单位的有效选项包括
- `kilograms`, `kg`
- `megagram`, `tonne`
- `gram`, `g`
- `pound_mass`, `lbm`, `pound`
- `slug`
- `kpound_mass`
- `ounce_mass`

**力单位**

力单位的有效选项包括
- `newtons`, `N`
- `knewton`, `kN`
- `kilogram_force`, `kgf`
- `dyne`
- `ounce_force`
- `pound_force`, `lbf`, `pound`
- `kpound_force`

**时间单位**

时间单位的有效选项包括
- `seconds`, `sec`
- `milliseconds`, `ms`
- `minutes`, `min`
- `hours`
