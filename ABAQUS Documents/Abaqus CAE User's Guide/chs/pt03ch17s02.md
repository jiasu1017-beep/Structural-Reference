# 17.2 进入和退出网格模块







您可以在任何时候通过点击上下文栏中 **Module** 列表中的 **Mesh** 来进入网格模块。进入 **Mesh** 模块后，Abaqus/CAE 界面会以下列方式发生变化：
- **Seed**、**Mesh**、**Feature** 和 **Tools** 菜单出现在主菜单栏上。
- 上下文栏中出现的 **Object** 字段允许您显示零件或装配体。
- Abaqus/CAE 可能会更改视口中显示的装配体中零件实例的颜色。这些颜色提示描述了每个实例的可网格划分性和依赖性。独立实例以其可网格划分性所描述的颜色显示，而从属实例在 Assembly 上下文中显示为蓝色，在 Part 上下文中显示为白色。请参见"独立和从属零件实例的网格划分"第 17.3.10 节。**注意：**只有在选择 **Mesh defaults** 颜色映射时，零件实例才会根据其可网格划分性和依赖性进行颜色编码。如果您在其他模块中显示了默认颜色映射，Abaqus/CAE 会在您进入网格模块时自动应用 **Mesh defaults** 颜色映射。如果您在其他模块中选择了非默认颜色映射（如 **Materials**），则在您进入网格模块时，Abaqus/CAE 会继续根据该颜色映射（在本例中，按材料类型）对进行颜色编码。

要退出网格模块，请从 **Module** 列表中选择任何其他模块。退出模块前无需保存网格；当您通过从主菜单栏中选择 ****File**![](../graphics/images/arrow.gif)**Save**** 或 ****File**![](../graphics/images/arrow.gif)**Save As**** 保存整个模型时，网格会自动保存。
