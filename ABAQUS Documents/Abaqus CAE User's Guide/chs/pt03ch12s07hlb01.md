# 12.7.1 创建或编辑材料





您使用**Edit Material**对话框来创建新材料或编辑现有材料。当您从主菜单栏选择****Material**![](../graphics/images/arrow.gif)**Create****时，您可以输入您选择的材料名称或接受默认名称，您可以提供材料的描述，并可以定义材料属性。当您选择****Material**![](../graphics/images/arrow.gif)**Edit****时，您可以重新定义材料描述或属性，但必须使用****Material**![](../graphics/images/arrow.gif)**Rename****来更改现有材料的名称。

使用**Material Behaviors**列表下方的菜单栏为材料添加属性。某些菜单项包含子菜单；例如，下图显示了****Mechanical**![](../graphics/images/arrow.gif)**Elasticity****菜单项下可用的行为：

![](../graphics/prp-buttons-nls.png)

**注意：**要显示特定材料行为的信息，请点击并按住该行为，然后按**F1**。会出现一个帮助窗口，其中包含与该行为关联的参数和数据的信息。

使用**Material Behaviors**列表选择现有材料行为进行编辑。

**警告：**Abaqus/CAE不会检查缺失或无效的材料行为，直到您提交作业进行分析。（任何警告和错误都由Job模块报告。）因此，您必须小心为分析所需的所有材料行为提供有效数据。

**创建或编辑材料：**

1. 使用以下方法之一显示材料编辑器：
   - 从主菜单栏，选择****Material**![](../graphics/images/arrow.gif)**Create****。
     **提示：**您也可以在**Material Manager**中点击**Create**，或在Property模块工具箱中选择创建材料工具![](../graphics/ico_material.png)。
   - 从主菜单栏，选择****Material**![](../graphics/images/arrow.gif)**Edit**![](../graphics/images/arrow.gif)***material name*****。
     **提示：**您也可以在**Material Manager**中选择材料并点击**Edit**。
   出现**Edit Material**对话框。
2. 如果要创建新材料，请输入您选择的材料名称。有关命名对象的更多信息，请参见["使用基本对话框组件，"第3.2.1节](pt01ch03s02s01.md)。
3. 如有需要，输入材料的描述。
   1. 在**Edit Material**对话框中点击![](../graphics/ico_edit.png)。出现材料描述编辑器。
   2. 在材料描述编辑器中，输入您要记录的有关材料的信息。
   3. 点击**OK**存储描述并关闭材料描述编辑器。当您提交作业时，Abaqus/CAE使用注释行将材料描述写入输入文件；材料描述不会写入输出数据库。有关更多信息，请参见["向Abaqus/CAE模型添加描述，"第9.10.2节](pt02ch09s09hlb02.md)。
4. 使用菜单栏或**Material Behaviors**列表分别选择新的或现有的材料行为。对话框中的行为定义区域会更改以显示与所选材料行为关联的所有参数和数据。
5. 编辑参数和数据以完成材料定义。
6. 如果您希望移除材料行为，请选择它并点击菜单栏右侧的![](../graphics/ico_delete.png)。
7. 编辑完材料定义后，点击**OK**保存材料并关闭对话框。
