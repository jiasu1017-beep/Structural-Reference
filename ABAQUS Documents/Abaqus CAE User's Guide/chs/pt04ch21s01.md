# 21.1 粘接接头和粘接界面建模概述





您可以使用粘聚单元创建模型来模拟以下内容：
- 粘接接头——通过具有有限厚度的类似胶水材料连接的两个组件。
- 粘接界面处的断裂——非常薄的胶水材料中的裂纹扩展，实际上可以认为是零厚度。
- 垫片（有限能力）——两个组件之间的密封。 没有专门的垫片行为（通常以压力与闭合的关系定义）。 如果您想对专门的垫片行为进行建模，请使用第32章"垫片"中所述的专用垫片单元。

有关更多信息，请参阅["粘聚单元：概述，" Abaqus Analysis User's Guide第32.5.1节](../usb/usb-link.md#usb-elm-ecohesiveoverview)。

您可以在模型中包含粘聚单元的两种主要方法是：
- 在现有模型的网格中嵌入一层或多层粘聚单元；或
- 使用几何和网格工具创建分析模型。

您可以通过共享节点或定义绑定约束来对粘聚层与周围散体材料之间界面处的连接进行建模。 绑定约束方法允许您使用比散体材料更细的离散化来对粘聚层进行建模，在某些建模情况下可能更可取。 有关更多信息，请参阅["定义粘聚层与周围散体材料之间的绑定约束，" 第21.4节](pt04ch21s04.md)。

与垫片单元一样，粘聚单元具有与其关联的方向。 此方向定义单元的厚度方向，它应该在整个粘聚层中保持一致。 应使用扫掠或偏移网格划分技术来生成粘聚层中的网格，因为这些工具产生的网格方向一致。 您也可以使用自底向上扫掠网格划分方法，但必须沿单元厚度方向扫掠以保持正确的方向。 您应该创建单层实体单元来对粘聚区域进行建模。 使用多于一层穿过厚度可能会产生不可靠的结果，不建议这样做。 您可以使用任何网格工具生成周围散体材料中的网格，因为这些单元不需要定向。

对粘接接头和粘接界面进行建模的一般过程涉及以下步骤：

1. 使用以下方法之一创建模型，并在Mesh模块中将粘聚单元类型分配给粘聚区域： - ["在现有三维网格中嵌入粘聚单元，" 第21.2节](pt04ch21s02.md) - ["使用几何和网格工具创建具有粘聚单元的模型，" 第21.3节](pt04ch21s03.md)
2. 在Property模块中，定义材料和引用该材料的粘聚截面，并将粘聚截面分配给粘聚区域。 有关更多信息，请参阅["分配粘聚建模数据，" 第21.5节](pt04ch21s05.md)。
3. 要对粘聚层的渐进损伤和失效进行建模，如["使用牵引-分离描述定义粘聚单元的 Constitutive Response，" Abaqus Analysis User's Guide第32.5.6节](../usb/usb-link.md#usb-elm-ecohesivebehavior)中所讨论的，请在材料定义中包含所需的损伤起始和损伤演化信息（选择 ****Mechanical**![](../graphics/images/arrow.gif)**Damage for Traction Separation Laws**![](../graphics/images/arrow.gif)***damage type*****）。 有关更多信息，请参阅["定义损伤，" 第12.9.3节](pt03ch12s09s03.md)。
