# 12.11.2 定义介电材料属性







**编辑材料**对话框允许您为用于耦合压电分析的完全约束材料定义介电属性（介电常数）。请参见"压电行为"，Abaqus分析用户指南第26.5.2节获取更多信息。

**定义介电属性：**

1. 从**编辑材料**对话框的菜单栏中，选择**电气/磁性**![](../graphics/images/arrow.gif)**介电（介电常数）**。（有关显示**编辑材料**对话框的信息，请参见"创建或编辑材料"，第12.7.1节。）
2. 单击**类型**字段右侧的箭头，指定介电常数的方向依赖性。
3. 切换**使用温度相关数据**将介电常数定义为温度的函数。**数据**表中会出现一列标为**温度**。
4. 单击**场变量数量**字段右侧的箭头增加或减少介电常数所依赖的场变量数量。
5. 在**数据**表中输入适用数据：**介电常数**各向同性行为的介电常数。（单位为[C![](../graphics/usi_eqn00317.gif)1L1](../popups/usb-int-iconventions-unitsym.md)。）**D11、D22和D33**定义正交异性行为的三个值，![](../graphics/usi_eqn00324.gif)、![](../graphics/usi_eqn00325.gif)和![](../graphics/usi_eqn00326.gif)。（单位为[C![](../graphics/usi_eqn00317.gif)1L1](../popups/usb-int-iconventions-unitsym.md)。）**D11、D12、D22、D13、D23、D33**定义各向异性行为的六个值，![](../graphics/usi_eqn00324.gif)、![](../graphics/usi_eqn00327.gif)、![](../graphics/usi_eqn00325.gif)、![](../graphics/usi_eqn00328.gif)、![](../graphics/usi_eqn00329.gif)和![](../graphics/usi_eqn00326.gif)。（单位为[C![](../graphics/usi_eqn00317.gif)1L1](../popups/usb-int-iconventions-unitsym.md)。）**温度**温度。**场变量* n ***预定义的场变量。您可能需要展开对话框以查看**数据**表中的所有列。有关如何输入数据的详细信息，请参见"输入表格数据"，第3.2.7节。
6. 单击**确定**关闭**编辑材料**对话框。或者，您可以从**编辑材料**对话框的菜单中选择其他材料行为进行定义（有关更多信息，请参见"浏览和修改材料行为"，第12.7.2节）。
