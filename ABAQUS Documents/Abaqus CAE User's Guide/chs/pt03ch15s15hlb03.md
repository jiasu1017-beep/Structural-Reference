# 15.15.3 定义显示体约束





您可以通过选择一个将被显示但不会包含在分析中的部件实例来创建显示体约束。您可以将显示体约束为固定在空间中，或者约束它跟随装配中选定的节点。您可以将显示体约束应用于模型中的任何部件实例。有关更多信息，请参见Abaqus Analysis User's Guide第2.9.1节的"显示体定义"。此外，您可以在可视化模块中控制显示体的外观；有关更多信息，请参见"自定义显示体的外观"，第55.8节。对于包含与连接器组合的显示体约束的模型示例，请参见第27章，"显示体"。

**创建显示体约束的步骤：**

1. 从主菜单栏选择 ****Constraint**![](../graphics/images/arrow.gif)**Create****。**提示：**您也可以使用「相互作用」模块工具箱中的![](../graphics/ico_constraint.png)工具创建显示体约束。
2. 在出现的「Create Constraint」对话框中，执行以下操作：1. 为约束命名。有关命名对象的更多信息，请参见"使用基本对话框组件"，第3.2.1节。2. 从「Type」列表中选择「Display body」；然后点击「Continue」。
3. 选择将作为显示体的部件实例。约束编辑器会出现。
4. 默认情况下，部件实例是固定的；或者，您可以约束它跟随装配中选定的点。从约束编辑器的「Motion control」字段中，选择以下之一：**No motion** 选择「No motion」以在分析期间将所选部件实例固定在空间中。这是默认选项。**Follow single point** 选择「Follow single point」，点击![](../graphics/ico_selectBlue.png)，并选择部件实例将被约束到的点。您必须从不同的部件实例中选择该点，并且该部件实例不能是显示体。在分析期间，显示体将跟随所选点的平移和旋转。**Follow three points** 选择「Follow three points」，点击![](../graphics/ico_selectBlue.png)，并选择部件实例将被约束到的三个点。您必须从不同的部件实例中选择这些点，并且该部件实例不能是显示体。在分析期间，显示体将跟随由所选点定义的坐标系的平移和旋转。第一个点表示坐标系的原点，第二个点表示*X*方向，第三个点表示*X*-*Y*平面。这些点应该是不共线的，并且在分析期间应保持不共线。
5. 点击「OK」保存约束定义并关闭编辑器。
