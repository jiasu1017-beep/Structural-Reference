# 14.3.3 步骤序列限制





当您从主菜单栏选择**步骤**![](../graphics/images/arrow.gif)**创建**时，会出现一个**创建步骤**对话框，您可以在其中指定您正在创建的步骤的过程类型。类似地，当您从主菜单栏选择**步骤**![](../graphics/images/arrow.gif)**替换**时，会出现一个**替换步骤**对话框，您可以在其中为现有步骤指定新的过程类型。**创建步骤**和**替换步骤**对话框中的过程类型选择取决于以下因素：
- 模型类型。
- 您已经与现有步骤关联的过程。
- 新步骤或替换步骤在分析步骤序列中的位置。

例如，当您在分析中创建第一个步骤时，您可以从有效过程类型列表中选择；Abaqus/Standard和Abaqus/Explicit过程类型都会出现在列表中。但是，一旦您创建了第一个步骤，**创建步骤**对话框中有效过程类型的列表将更改，仅包含与第一个步骤兼容的过程。例如，如果第一个步骤是Abaqus/Standard步骤，则Abaqus/Explicit过程不再出现在列表中。
