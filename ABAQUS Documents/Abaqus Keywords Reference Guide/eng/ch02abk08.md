# *BEAM SECTION OFFSET







### *BEAM SECTION OFFSETDefine an offset for the beam cross-section origin.

This option can be used only in conjunction with the [*BEAM GENERAL SECTION](ch02abk05.md) option for the standard cross-sections available in the beam cross-section library (See ["Beam element cross-section orientation," Section 29.3.4 of the Abaqus Analysis User's Guide](../usb/usb-link.md#usb-elm-ebeamcrosssection), for details). It is used to define an offset for the cross-section origin from the beam axis with respect to the local (1, 2) axis system.

**Product: **Abaqus/Standard  

**Type: **Model data  

**Level: **Part,  Part instance  

##### **References:**

- ["Using a general beam section to define the section behavior," Section 29.3.7 of the Abaqus Analysis User's Guide](../usb/usb-link.md#usb-elm-eusingbeamgensect)
- [*BEAM GENERAL SECTION](ch02abk05.md)

**There are no parameters associated with this option.**
