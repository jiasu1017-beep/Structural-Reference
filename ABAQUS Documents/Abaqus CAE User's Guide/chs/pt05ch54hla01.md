# 54.4 指定报告文件名





Abaqus/CAE将您的表格数据报告写入您选择的文件名。如果您选择将报告写入现有文件,您可以进一步指定是新信息应该附加到当前文件内容还是覆盖文件。默认是附加新信息。

**指定报告文件名的步骤:**

1. 找到"**File**"选项。 **对于X-Y数据或场输出的报告:** 从主菜单栏,选择 ****Report**![](../graphics/images/arrow.gif)**XY**** 或 ****Report**![](../graphics/images/arrow.gif)**Field Output****;然后在出现的对话框中点击"**Setup**"选项卡。"**File**"选项在页面顶部。 **对于自由体切割的报告:** 从主菜单栏,选择 ****Report**![](../graphics/images/arrow.gif)**Free Body Cut****。"**File**"选项在页面顶部。 **对于探测值的报告:** 从主菜单栏,选择 ****Tools**![](../graphics/images/arrow.gif)**Query****,然后在出现的对话框中点击"**Probe values**"。出现"**Probe Values**"对话框。将光标移动到当前视口中探测模型或X-Y曲线图,然后点击鼠标按钮1将感兴趣的值存储到对话框中。完成后,点击"**Write to File**"。出现"**Report Probe Values**"对话框;"**File**"选项在对话框顶部。
2. 要使用标准文件浏览器选择文件名,点击"**Select**"。出现"**File Selection**"对话框。使用该对话框过滤和浏览现有文件。有关使用此对话框的更多信息,请参阅 ["使用文件选择对话框," 第3.2.10节](pt01ch03s02s10.md)。完成后,点击"**OK**"关闭对话框。
3. 在"**Name**"字段中输入要写入报告的文件名。
4. 切换"**Append to file**"以将此报告附加到您指定的文件的当前内容(如果有)。当"**Append to file**"关闭时,现有文件内容将被覆盖。

要生成报告,请在对话框中配置剩余选项;然后点击"**Apply**"(对于X-Y、场输出或自由体切割)或"**OK**"(对于探测值报告)。完成后,点击"**Cancel**"关闭对话框。
