# 12.12.2 定义质量扩散





您可以使用**Edit Material**对话框来定义质量扩散的某些方面。有关详细信息，请参阅以下章节：
- ["定义扩散率"](pt03ch12s12s02.md#usi-prp-other-massdiffusion-diffusivity)
- ["定义一般温度驱动的质量扩散"](pt03ch12s12s02.md#usi-prp-other-massdiffusion-soret)
- ["定义压力驱动的质量扩散"](pt03ch12s12s02.md#usi-prp-other-massdiffusion-pressure)
- ["定义溶解度"](pt03ch12s12s02.md#usi-prp-other-massdiffusion-solubility)

### 定义扩散率

扩散率定义了 一种材料通过另一种材料的扩散或移动。质量扩散的控制方程是菲克方程的扩展：它们允许扩散物质在基体材料中的非均匀溶解性，以及由温度和压力梯度驱动的质量扩散。有关更多信息，请参阅以下章节：
- [Abaqus Analysis User's Guide的第26.4.1节，"扩散率"](../usb/usb-link.md#usb-mat-cdiffusivity)
- [Abaqus Analysis User's Guide的第6.9.1节，"质量扩散分析"](../usb/usb-link.md#usb-anl-amassdiffusion)

**定义扩散率：**

1. 从**Edit Material**对话框的菜单栏中，选择****Other**![](../graphics/images/arrow.gif)**Mass Diffusion**![](../graphics/images/arrow.gif)**Diffusivity****。（有关显示**Edit Material**对话框的信息，请参见["创建或编辑材料，"第12.7.1节"](pt03ch12s07hlb01.md)。）
2. 点击**Type**字段右侧的箭头，并指定扩散率的方向相关性。
3. 选择**Law**选项以指定您希望如何定义扩散率行为：
   - 选择**General**以选择一般质量扩散行为。
   - 选择**Fick**以选择菲克扩散定律。有关更多信息，请参见Abaqus Analysis User's Guide的第26.4.1节"扩散率"中的"定义扩散率"。
4. 切换**Use temperature-dependent data**将扩散率数据定义为温度的函数。**Data**表格中出现一个标为**Temp**的列。
5. 点击**Number of field variables**字段右侧的箭头以增加或减少扩散率数据所依赖的场变量数量。
6. 在**Data**表格中，输入适用的数据：
   **D** 各向同性扩散率。（[L2T1](../popups/usb-int-iconventions-unitsym.md)单位。）
   **D11, D22, 和 D33** 正交各向异性扩散率项。（[L2T1](../popups/usb-int-iconventions-unitsym.md)单位。）
   **D11, D12, D22, D13, D23, D33** 各向异性扩散率项。（[L2T1](../popups/usb-int-iconventions-unitsym.md)单位。）
   **Concentration** 扩散材料的质量浓度。
   **Temp** 温度。
   **Field *n*** 预定义的场变量。
   您可能需要扩展对话框以查看**Data**表格中的所有列。有关如何输入数据的详细信息，请参见["输入表格数据，"第3.2.7节"](pt01ch03s02s07.md)。
7. 要描述温度驱动的扩散，请从**Suboptions**菜单中选择**Soret Effect**。（此选项仅在您在第3步中选择**General**时有效。）有关详细说明，请参见["定义一般温度驱动的质量扩散"](pt03ch12s12s02.md#usi-prp-other-massdiffusion-soret)。
8. 要描述压力驱动的质量扩散，请从**Suboptions**菜单中选择**Pressure Effect**。有关详细说明，请参见["定义压力驱动的质量扩散"](pt03ch12s12s02.md#usi-prp-other-massdiffusion-pressure)。
9. 点击**OK**关闭**Edit Material**对话框。或者，您可以从**Edit Material**对话框的菜单中选择另一种材料行为进行定义（有关更多信息，请参见["浏览和修改材料行为，"第12.7.2节"](pt03ch12s07hlb02.md)）。

### 定义一般温度驱动的质量扩散

Soret效应因子，![](../graphics/usi_eqn00339.gif)，控制温度驱动的质量扩散。您可以将Soret效应因子定义为浓度、温度和/或场变量的函数。请参见["扩散率，"第26.4.1节Abaqus Analysis User's Guide](../usb/usb-link.md#usb-mat-cdiffusivity)，了解更多。

**注意：**仅当您在扩散率定义中选择了一般质量扩散行为时，才能指定Soret效应因子。（如果您选择了菲克扩散定律，Soret效应因子会自动计算。）有关更多信息，请参见Abaqus Analysis User's Guide的第6.9.1节"质量扩散分析"中的"菲克定律"。

**定义Soret效应因子：**

1. 如["定义扩散率"](pt03ch12s12s02.md#usi-prp-other-massdiffusion-diffusivity)中所述定义扩散率。
2. 从**Edit Material**对话框的**Suboptions**菜单中，选择**Soret Effect**。出现**Suboption Editor**。
3. 切换**Use temperature-dependent data**将Soret效应因子定义为温度的函数。**Data**表格中出现一个标为**Temp**的列。
4. 点击**Number of field variables**字段右侧的箭头，以增加或减少Soret效应因子定义中包含的场变量数量。
5. 在**Data**表格中输入以下数据：
   **kappa_s** Soret效应因子，![](../graphics/usi_eqn00339.gif)。（[F1/2L1](../popups/usb-int-iconventions-unitsym.md)单位。）
   **Concentration** 扩散材料的质量浓度。
   **Temp** 温度。
   **Field *n*** 预定义的场变量。
6. 点击**OK**返回**Edit Material**对话框。

### 定义压力驱动的质量扩散

压力应力因子，![](../graphics/usi_eqn00340.gif)，控制由等效压力应力梯度驱动的质量扩散。您可以将压力应力因子定义为浓度、温度和/或场变量的函数。请参见["扩散率，"第26.4.1节Abaqus Analysis User's Guide](../usb/usb-link.md#usb-mat-cdiffusivity)，了解更多。

**定义压力应力因子：**

1. 如["定义扩散率"](pt03ch12s12s02.md#usi-prp-other-massdiffusion-diffusivity)中所述定义扩散率。
2. 从**Edit Material**对话框的**Suboptions**菜单中，选择**Pressure Effect**。出现**Suboption Editor**。
3. 切换**Use temperature-dependent data**将压力应力因子定义为温度的函数。**Data**表格中出现一个标为**Temp**的列。
4. 点击**Number of field variables**字段右侧的箭头，以增加或减少压力应力因子定义中包含的场变量数量。
5. 在**Data**表格中输入以下数据：
   **kappa_p** 压力应力因子，![](../graphics/usi_eqn00340.gif)。（[LF1/2](../popups/usb-int-iconventions-unitsym.md)单位。）
   **Concentration** 扩散材料的质量浓度。
   **Temp** 温度。
   **Field *n*** 预定义的场变量。
6. 点击**OK**返回**Edit Material**对话框。

   **Temp** 温度。
   **Field *n*** 预定义的场变量。
   有关如何输入数据的详细信息，请参见["输入表格数据，"第3.2.7节"](pt01ch03s02s07.md)。
5. 点击**OK**关闭**Edit Material**对话框。或者，您可以从**Edit Material**对话框的菜单中选择另一种材料行为进行定义（有关更多信息，请参见["浏览和修改材料行为，"第12.7.2节"](pt03ch12s07hlb02.md)）。
