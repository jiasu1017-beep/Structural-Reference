# 1.3.32 三杆桁架







**产品：**Abaqus/Standard  

### 测试的单元

T2D2    T2D2H    T2D3    T2D3H    T3D2    T3D2H    T3D3    T3D3H    

FRAME2D    FRAME3D    

### 问题描述

![](../graphics/ver3bartruss-model.png)

**材料：**

线弹性，弹性模量 = 30.0 × 10⁶。

**边界条件：**

节点 *A*、![](../graphics/ver_eqn00142.gif) 和 *C* 为铰接。

**载荷：**

![](../graphics/ver_eqn00208.gif) = 10000.0。

### 参考解

![](../graphics/ver_eqn00209.gif) = 1.3711 × 10²，在单元 1 和 3 中 ![](../graphics/ver_eqn00210.gif) = 32907，在单元 2 中 ![](../graphics/ver_eqn00210.gif) = 41134。

### 结果与讨论

所有单元都产生精确解。三节点单元测试使用桁架单元时需要多点约束来消除奇异性；例如 T3D3。

测试的框架单元具有与测试的桁架单元相同截面积的矩形截面。在框架单元的截面中，通过声明框架截面的相关参数来指定框架单元末端的铰接连接。因此接头以铰接连接为特征。由于框架单元是根据截面属性公式化的，因此无法获得应力输出；而是可以获得截面力。从轴向力和截面积计算的应力与从桁架单元测试获得的应力一致。
