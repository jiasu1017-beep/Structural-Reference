# 14.14.1 定义ALE自适应网格区域





您可以定义希望进行ALE自适应网格划分的模型区域，并且可以指定该域的自适应网格划分的频率和强度。更多信息，请参见以下章节：
- 「ALE自适应网格概述」，Abaqus Analysis User's Guide第12.2.1节
- 「在Abaqus/Explicit中定义ALE自适应网格域」，Abaqus Analysis User's Guide第12.2.2节
- 「在Abaqus/Standard中定义ALE自适应网格域」，Abaqus Analysis User's Guide第12.2.6节

**注意：**对于任何特定步骤，您只能定义一个自适应网格域。

**将ALE自适应网格应用于区域的步骤：**

1. 从主菜单栏选择 ****Other**![](../graphics/images/arrow.gif)**ALE Adaptive Mesh Domain**![](../graphics/images/arrow.gif)**Manager****。Abaqus/CAE会显示「ALE Adaptive Mesh Domain Manager」，其中显示模型中定义的步骤以及与每个步骤关联的自适应网格控制。
2. 在「ALE Adaptive Mesh Domain Manager」中，选择要配置自适应网格划分的步骤，然后点击「Edit」。Abaqus/CAE会显示ALE自适应网格域编辑器。
3. 在编辑器顶部，选择「Use the ALE adaptive mesh domain below」。
4. 点击「Region」字段旁边的![](../graphics/ico_selectBlue.png)，并使用以下方法之一选择要网格划分的区域：- 使用现有集合来定义区域。在提示区域右侧点击「Sets」。从出现的「Region Selection」对话框中选择一个现有集合，然后点击「Continue」。- 使用鼠标在视口中选择一个区域。（更多信息，请参见「在当前视口中选择对象」，第6.2节。）点击鼠标按钮2表示您已完成选择。Abaqus/CAE会在视口中将所选区域以红色高亮显示。**注意：**默认选择方法基于您最近使用的选择方法。如需恢复为另一种方法，请点击提示区域右侧的「Select in Viewport」或「Sets」。
5. 如果要指定非默认的自适应网格控制，请切换「ALE Adaptive mesh controls」，并选择要与该区域关联的自适应网格控制名称。如果您尚未创建网格控制，您可以点击「ALE Adaptive mesh controls」字段右侧的![](../graphics/ico_add.png)来定义要分配给自适应网格域的控制。请参见「指定ALE自适应网格的控制」，第14.14.3节。
6. 输入Abaqus网格化该区域的频率，以增量为单位。
7. 输入每个增量的「Remeshing sweeps per increment」数量。在这种情况下，增量是每个自适应网格增量，如上所述的「Frequency」设置所定义。
8. 对于Abaqus/Explicit步骤中的自适应网格划分，请输入Abaqus/Explicit将在步骤开始时应用的「Initial remeshing sweeps」数量。
9. 点击「OK」保存设置并关闭ALE自适应网格域编辑器。「ALE Adaptive Mesh Domain Manager」会显示与该步骤关联的自适应网格控制。
10. 点击「Dismiss」关闭「ALE Adaptive Mesh Domain Manager」。
