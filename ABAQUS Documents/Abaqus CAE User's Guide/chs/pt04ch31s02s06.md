# 31.2.6 轮廓积分输出





定义裂缝后，您必须使用Step模块中的**History Output Request**编辑器将轮廓积分数据包含在分析生成的输出数据库中。编辑器允许您配置以下内容：
- 输出频率。
- 要执行的轮廓积分计算的类型（*J*积分、<img src="../graphics/usi_eqn00701.gif>积分、应力强度因子或*T*应力）。
- 要计算的轮廓数。

Abaqus为它计算的每个轮廓积分将历史输出变量写入输出数据库。[图31-13](pt04ch31s02s06.md#eng-crack-historydb)显示了表示轮廓积分的输出变量名称的格式。
