# 53.1 铺层堆叠图概述





铺层堆叠图是复合模型所选区域中各铺层的图形表示。实际上,铺层堆叠图是所选区域中各铺层的岩心样本。该区域可以来自复合铺层或复合截面。

您可以自定义铺层堆叠图的外观。例如,图像可以显示铺层的顺序、每个铺层中纤维的方向、每个铺层中的材料、铺层的相对厚度以及参考面的位置。[图53-1](pt05ch53hla01.md#usv-stack-overview) 展示了一个铺层堆叠图。

**图53-1** 铺层堆叠图。

![](../graphics/usv-stack-overview-nls.png)

阶梯状外观没有相应的物理意义;它只是一种允许您查看区域中所有铺层的机制。例如,铺层并不是在每个阶梯处终止。

铺层堆叠图中的三轴表示单元取向系统,它显示壳法线或堆叠方向(3方向)以及铺层的1和2方向。纤维总是绘制在1-2平面中,相对于1方向成一定角度。在固体复合铺层中,铺层中的纤维并不总是平行于1-2平面运行(例如,如果铺层取向的3方向与单元堆叠方向不对齐)。在这种情况下,铺层堆叠图中的纤维并不是复合层中纤维的真实描绘,而是复合截面或铺层定义中旋转角的图形表示:铺层堆叠图中绘制的角度是相对于单元堆叠方向轴测量的铺层表中指定的旋转角。有关更多信息,请参阅["理解复合铺层和取向," 第23.3节](pt04ch23s03.md)。

对于基于用户定义坐标系或旋转角分布的铺层取向,铺层堆叠图不绘制纤维。Abaqus/CAE在此类铺层上显示星号(对于坐标系)或脱字号(对于旋转角分布),以表示它无法在1-2平面中绘制能准确代表该铺层纤维方向的线。类似地,如果您使用离散场分布来定义铺层的厚度,Abaqus/CAE会根据铺层中其他均匀铺层的平均厚度在铺层堆叠图中绘制该铺层,并在图旁边显示离散场名称(如果厚度标签已打开)。

当您使用传统壳单元对复合层进行建模时,了解参考面的位置非常重要。网格将定位在参考面上,接触是相对于参考面定义的。铺层堆叠图允许您查看参考面相对于复合层中其他铺层的位置,如[图53-2](pt05ch53hla01.md#usv-stack-offset)所示。

**图53-2** 铺层堆叠图中的参考面。

![](../graphics/usv-stack-offset-nls.png)

您可以在"属性"模块中在创建复合铺层或将复合截面分配给区域后显示铺层堆叠图。您也可以在分析具有复合铺层或复合截面的模型后在"可视化"模块中显示铺层堆叠图。有关更多信息,请参阅本指南HTML版本中的["定义复合铺层," 第12.2.4节](pt03ch12s02s04.md)和["创建复合壳截面," 第12.13.7节](pt03ch12s13s02.md)。

要生成铺层堆叠图,请从主菜单栏选择 ****Tools**![](../graphics/images/arrow.gif)**Query**** 并从出现的对话框中选择"**铺层堆叠图**"。显示铺层堆叠图后,您可以通过点击提示区域中的"**铺层堆叠图选项**"按钮或点击工具箱中的 ![](../graphics/ico_plotPlyStackOptions.png) 工具来自定义其外观。有关自定义铺层堆叠图的详细说明,请参阅["自定义铺层堆叠图," 第53.2节](pt05ch53s01.md)。
