### 概述

铸铁塑性模型：
- 旨在用于灰口铸铁的本构建模；
- 提供在拉伸和压缩中具有不同屈服强度、流动和硬化的弹塑性行为；
- 基于屈服函数，该屈服函数在拉伸加载条件下依赖于最大主应力，在压缩加载条件下依赖于与压力无关的（Mises型）行为；
- 在拉伸加载条件下允许同时发生非弹性膨胀和非弹性剪切；
- 在压缩加载条件下仅允许非弹性剪切；
- 旨在仅在基本单调加载条件下模拟材料响应；和
- 不能用于建模率相关性。

### 弹性和塑性行为

铸铁塑性模型描述灰口铸铁的力学行为，这是一种微观结构由钢基体中分布的石墨片组成的材料。在拉伸时，石墨片起到应力集中的作用，导致屈服是最大主应力的函数，随后是脆性行为。在压缩时，石墨片对宏观响应没有显著影响，导致类似于许多钢材的延性行为。

您单独指定响应的弹性部分；只能使用线性各向同性弹性（见["线性弹性行为，" 第22.2.1节"](pt05ch22s02abm02.md)）。假定弹性刚度在拉伸和压缩中相同。

铸铁塑性模型用于提供塑性"泊松比"的值，即单轴拉伸下横向与纵向塑性应变之比的绝对值。塑性泊松比可能随塑性变形而变化。但是，Abaqus中的模型假定其相对于塑性变形是恒定的。它可以依赖于温度和场变量。如果未指定塑性泊松比的值，则假定默认值为0.04。此默认值基于单轴拉伸超过屈服时永久体积应变的实验结果（详情见["铸铁塑性，" Abaqus理论指南第4.3.7节](../stm/stm-link.md#stm-mat-castironplasticity)）。

可以指定独立的拉伸和压缩硬化（见图23.2.10-1）。拉伸硬化数据提供单轴拉伸下作为塑性应变、温度和场变量函数的单轴拉伸屈服应力。压缩硬化数据提供单轴压缩下作为塑性应变、温度和场变量函数的单轴压缩屈服应力。

### 屈服条件

Abaqus利用复合屈服面来描述拉伸和压缩中的不同行为。在拉伸中，屈服假定由最大主应力控制，而在压缩中，屈服假定与压力无关，仅由偏量应力控制（Mises屈服条件）。

该模型在["铸铁塑性，" Abaqus理论指南第4.3.7节](../stm/stm-link.md#stm-mat-castironplasticity)中详细描述。

### 流动法则

为了讨论流动和硬化行为，将子午面分为图23.2.10-2中所示的两个区域是很用的。

**图23.2.10-2** p-q平面中流动势的示意图。

![](../graphics/ccastiron-potential-nls.png)

单轴压缩线左侧的区域（标记为UC）被称为"拉伸区域"，而单轴压缩线右侧的区域被称为"压缩区域"。流动势由压缩区域中的Mises圆柱和拉伸区域中的椭圆形"帽盖"组成。两个表面之间的过渡是光滑的。流动势在子午面上的投影（见图23.2.10-2）在压缩区域中是一条直线，在拉伸区域中是一个椭圆。相应的偏平面上的投影是一个圆。上述选择的结果是拉伸区域中的塑性流动导致非弹性体积膨胀，压缩区域中没有非弹性体积变化（详情见["铸铁塑性，" Abaqus理论指南第4.3.7节](../stm/stm-link.md#stm-mat-castironplasticity)）。

#### 非关联流动

由于流动势与屈服面不同（"非关联"流动），材料雅可比矩阵是非对称的。因此，为了改善收敛，请使用非对称矩阵存储和求解方案（见["定义分析，" 第6.1.2节"](pt03ch06s01abo05.md)）。

### 硬化

由于灰口铸铁在单轴拉伸和单轴压缩中的硬化不同，您需要以表格形式提供两组硬化数据：一组基于定义 σ~t~ = σ~t~(ε¯^p^~t~) 的单轴拉伸实验，另一组基于定义 σ~c~ = σ~c~(ε¯^p^~c~) 的单轴压缩实验。这里，ε¯^p^~t~和ε¯^p^~c~分别是单轴拉伸和单轴压缩中的等效塑性应变。

### 材料数据的限制

塑性泊松比 ν~p~ 预期小于0.5，因为实验结果表明灰口铸铁在单轴拉伸中超过屈服时体积永久增加。为了使势定义良好，ν~p~ 必须大于1.0。因此，塑性泊松比必须满足 1.0 < ν~p~ < 0.5。

铸铁塑性材料模型旨在建模铸铁和其他类似于铸铁的材料，其单轴拉伸和单轴压缩中的行为与图23.2.10-1所示的行为一致。特别是，模型期望单轴拉伸中的初始屈服应力小于单轴压缩中的初始屈服应力。即使某些非铸铁材料的整体应力-应变响应和硬化行为与铸铁一致，您也必须确保（专门为建模铸铁构建的）模型的流动势对其他材料是有意义的。仅当单轴拉伸中的初始屈服应力等于或大于单轴压缩中的初始屈服应力时，Abaqus才会发出警告消息。在这方面不执行任何其他检查。

如果单轴拉伸中的屈服应力高于单轴压缩中的屈服应力，单轴拉伸中的材料点实际上可能在单轴压缩指定的初始屈服应力处屈服。这种明显的异常行为是由于（由于不切实际的用户指定材料属性）单轴拉伸应力路径在应力空间中首先遇到屈服面的压缩（Mises）部分。

### 单元

铸铁塑性模型可用于Abaqus中除假定应力状态为平面应力的单元（平面应力连续体、壳和膜单元）外的任何应力/位移单元。它可以与一维单元（平面中的桁架和梁）一起使用，在Abaqus/Standard中，还可以与空间梁一起使用。
