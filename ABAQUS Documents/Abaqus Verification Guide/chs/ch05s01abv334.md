# 5.1.18 用户定义的坐标系





**产品：**Abaqus/Standard  

### 测试单元

S4R

### 测试功能

通过用户定义的坐标系定义局部材料轴。

### 问题描述

在每个测试中，用户定义的坐标系用于定义材料点方向，如[图5.1.18-1](ch05s01abv334.md#verorient-model)所示。

**图5.1.18-1** 材料点方向。

![](../graphics/verorient-model-nls.png)

**材料：**

线弹性，杨氏模量= 3.0×10^6，泊松比= 0.3。

**边界条件：**

在节点11和14处=0，=0，=0，在所有节点处=0。

**载荷：**

集中力1000以与*x*轴成45°的角度施加到节点12和13。

### 备注

在输入文件[xorisrdc.inp](../eif/xorisrdc.inp)中使用局部坐标方向。

### 结果与讨论

所有测试应产生以下结果：

| 应力分量 | 应变分量 |
| --- | --- |
| = 1499.9 | = 4.4997E5 |
| = 499.97 | = 1.6665E6 |
| = 865.97 | = 7.5050E5 |

| 主应力 | 主应变 |
| --- | --- |
| = 2000.0 | = 6.6662E5 |
| = 0.0 | = 1.9998E5 |

[xorisrdc.inp](../eif/xorisrdc.inp)

```

SYSTEM=RECTANGULAR,
DEFINITION=COORDINATES
2., 1.7071, 0., 1.,
1.4142, 0., 1., 0.7071, 0.
3, 30.
```
