# 81.6 如何使插件在 Abaqus/CAE 中可用？





要使插件在 Abaqus/CAE 中可用，必须将包含注册命令的专门命名的文件放置在 Abaqus/CAE 搜索插件的目录之一中。注册命令可以使插件从主菜单栏的 **Plug-ins** 菜单、单独的插件工具箱或两者同时可用。本节介绍如何使插件在 Abaqus/CAE 中可用。涵盖以下主题：
