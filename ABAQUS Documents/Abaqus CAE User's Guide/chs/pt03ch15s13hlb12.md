# 15.13.14 定义Standard-Explicit协同仿真相互作用







您可以使用Standard-Explicit协同仿真相互作用来定义Abaqus/Standard到Abaqus/Explicit协同仿真的接口区域和耦合方案。从主菜单栏中选择****相互作用**![](../graphics/images/arrow.gif)**创建****，然后选择用于数据交换的区域。更多信息，请参见第26章"协同仿真"。

Standard-Explicit协同仿真相互作用只能在常规静态、隐式动态或显式动态步骤中创建。该相互作用仅在其创建的步骤中有效，不会传播到后续步骤。一个模型中只能有一个Standard-Explicit协同仿真相互作用处于活动状态。

**定义Standard-Explicit协同仿真相互作用：**

1. 从主菜单栏中，选择****相互作用**![](../graphics/images/arrow.gif)**创建****。**提示：**您也可以使用相互作用模块工具箱中的![](../graphics/ico_interaction.png)工具创建Standard-Explicit协同仿真相互作用。
2. 在出现的**创建相互作用**对话框中，执行以下操作：- 命名该相互作用。有关命名对象的信息，请参见第3.2.1节"使用基本对话框组件"。- 选择将创建该相互作用的步骤。- 选择**Standard-Explicit协同仿真**类型的相互作用。
3. 单击**继续**关闭**创建相互作用**对话框。
4. 在提示区域中，为协同仿真区域类型选择以下之一：- 如果要选择表面，请选择**表面**。- 如果要选择节点，请选择**节点区域**。
5. 使用以下方法之一选择区域：- 使用现有集合或表面定义区域。在提示区域右侧单击**集合**或**表面**。从出现的**区域选择**对话框中选择现有集合或表面，然后单击**继续**。**注意：**默认选择方法基于您最近使用的选择方法。要恢复为另一种方法，请单击提示区域右侧的**在视口中选择**或**集合**或**表面**。- 使用鼠标在视口中选择区域。（更多信息，请参见"在当前视口中选择对象"，第6.2节。）如果模型包含网格和几何的组合，请从提示区域单击以下之一：- 如果要从几何区域选择表面，请单击**几何**。- 如果要从本机或孤立网格选择表面，请单击**网格**。您可以使用角度方法从几何或网格中选择一组面或边或一组单元面。更多信息，请参见"使用角度和特征边缘方法选择多个对象"，第6.2.3节。
6. 选择**增量控制**方法。必须在Abaqus/Standard和Abaqus/Explicit分析中使用相同的增量方法。此选项在静态常规步骤中不可用。- 选择**允许子循环**以允许Abaqus/Standard增量大小与Abaqus/Explicit中的增量大小不同；将根据需要交换场。- 选择**锁定时间步**以强制Abaqus/Standard匹配Abaqus/Explicit的增量大小；将在每个共享增量处交换场。
7. 选择**耦合步骤周期**。在Abaqus/Standard分析中，Abaqus始终使用下一个增量大小作为建议的耦合步骤大小。- 选择**由分析确定**以让Abaqus使用下一个增量大小作为建议的耦合步骤大小。- 选择**指定**，然后输入耦合步骤大小的值（仅在Abaqus/Explicit分析中可用）。
8. 单击**确定**创建相互作用并关闭编辑器。
