# 11.8.2 点零件





创建零件时，您可以选择其基础特征的形状为实体、壳、线框或点。如果您选择点，必须指定点的坐标，Abaqus/CAE会在该位置创建一个基础特征为点的零件。此外，该点也是零件的参考点。点零件的建模空间可以是三维、二维或轴对称。点零件的类型可以是可变形或刚性的。

您可以继续向点零件添加特征，例如基准和线框。更典型的是，您将使用点零件通过用具有质量和惯量的点零件替换刚性零件来简化模型。您可以向刚性点零件添加质量；参见第33章"惯性"。此外，您可以将显示体附加到点并使用显示体来表示原始刚性零件；参见第27章"显示体"。最后，您可以通过建模连接器（如JOIN或REVOLUTE）将点零件约束到模型；参见第24章"连接器"。
