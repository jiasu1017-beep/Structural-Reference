# *FOUNDATION





### *FOUNDATION规定单元基础。

此选项用于对单元上的基础进行建模。

**产品：**Abaqus/Standard   Abaqus/CAE   

**类型：**模型数据  

**级别：**部件，部件实例，装配  

**Abaqus/CAE：**相互作用模块

##### **参考：**

- ["单元基础，" Abaqus Analysis User's Guide第2.2.2节](../usb/usb-link.md#usb-int-afoundation)

**此选项没有关联的参数。**

### **定义单元基础的数据行：**

**第一行：**

根据需要重复此数据行，以定义各种单元或单元集的基础。
