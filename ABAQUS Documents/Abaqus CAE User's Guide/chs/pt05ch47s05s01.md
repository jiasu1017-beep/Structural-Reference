# 47.6.2 选择一个或多个X-Y曲线进行自定义





### 47.6.2 选择一个或多个X-Y曲线进行自定义

要自定义曲线，您首先选择它，然后选择自定义选项。使用**曲线选项**对话框中的**曲线**字段来选择要自定义的一个或多个*X-Y*曲线。**曲线**字段列出了显示的*X-Y*曲线，每个曲线由名称标识。有关曲线命名约定的信息，请参见["理解"Temp"和其他X-Y数据对象名称，" 第47.1.3节](pt05ch47s01s03.md)。

选择了一个或多个要自定义的*X-Y*曲线后，您使用**属性**框架中的选项来控制表示数据的线条和符号，并输入将出现在图例中的文本。

**选择一个或多个X-Y曲线进行自定义：**

1. 找到**曲线**字段。**曲线**字段位于对话框顶部。
2. 选择一个或多个要自定义的*X-Y*曲线。有关在对话框中选择多个项目的更多信息，请参见["从列表和表中选择多个项目，" 第3.2.11节](pt01ch03s02s11.md)。**注意：**要使*X-Y*曲线可供选择，您必须首先绘制它。如果您选择了一条曲线，**属性**框架将显示该曲线的当前属性。如果您选择了多条曲线，Abaqus/CAE会将**图例文本**字段显示为空白。**属性**框架中的显示取决于所选曲线是否具有相同的属性。如果属性相同，Abaqus/CAE会显示它们；如果属性不同，Abaqus/CAE会显示以下内容：- **显示**和**显示符号**复选框为灰色。- 线条**颜色**和符号**颜色**文本字段显示术语**原样**。- **样式**、**厚度**、**符号**和**大小**按钮为空白。- **频率**文本字段为空白。
3. 从**属性**字段，您可以控制以下内容：- 图例中描述数据的文本。有关更多信息，请参见["自定义X-Y绘图图例，" 第47.6.3节](pt05ch47s05hlb02.md)。- 表示数据的线条（曲线）的外观。有关更多信息，请参见["自定义X-Y曲线的外观，" 第47.6.4节](pt05ch47s05hlb03.md)。- 曲线沿线符号的外观。有关更多信息，请参见["自定义X-Y曲线上使用的符号，" 第47.6.5节](pt05ch47s05hlb04.md)。
