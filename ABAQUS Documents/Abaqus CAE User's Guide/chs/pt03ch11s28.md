# 12.18 在Property模块中使用Special菜单





您可以使用Property模块中的**Special**菜单来定义以下工程特性：
- **Skin（蒙皮）**。蒙皮增强定义了一个与现有部件表面粘合的蒙皮，并指定其工程特性。有关详细信息，请参见[第36章，"蒙皮和桁条增强"](pt04ch36.md)。
- **Inertia（惯性）**。您可以在部件上的某点定义集中质量、旋转惯性和热容。还可以定义质量和惯性比例阻尼。在Abaqus/Standard分析中，可以定义复合阻尼。有关详细信息，请参见[第33章，"惯性"](pt04ch33.md)。
- **Springs/Dashpots（弹簧/阻尼器）**。您可以定义弹簧和阻尼器，它们表现出与场变量无关的相同线性行为。您也可以在同一组点上同时定义弹簧和阻尼器行为。在Abaqus/Explicit或Abaqus/Standard分析中，您可以建立连接两点的弹簧和阻尼器，沿两点间的连线作用。在Abaqus/Standard分析中，您还可以建立沿固定方向连接两点的弹簧和阻尼器，或连接点到地面的弹簧和阻尼器。有关详细信息，请参见[第37章，"弹簧和阻尼器"](pt04ch37.md)。
