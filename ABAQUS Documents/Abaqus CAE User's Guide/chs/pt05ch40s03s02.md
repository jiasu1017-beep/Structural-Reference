# 40.3.2 激活绘图状态





绘图状态结合所有活动的自定义选项来在视口中生成绘图。有些选项适用于所有绘图类型，有些仅在您选择叠加变形和未变形模型形状时应用，有些则特定于当前绘图类型。您通过生成相应类型的绘图来进入特定的绘图状态。例如，如果您生成未变形绘图，则当前视口将处于未变形绘图状态。视口的绘图状态会保持不变，Abaqus/CAE会使用您对自定义选项的任何更改来更新它，直到您在那个视口中生成其他状态的绘图。如果您创建多个视口，每个视口可以包含不同的绘图状态。此外，您可以通过为单一绘图类型叠加未变形和变形形状，或在视口中显示多种绘图类型，来选择在单个视口中允许多个绘图状态。
