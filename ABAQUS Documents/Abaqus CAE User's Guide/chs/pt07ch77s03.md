# 77.3 更改透明度





默认情况下，Abaqus/CAE 以不透明颜色显示着色渲染样式中的几何和网格单元。内部特征和视口中其他对象"后面"的特征不可见。在某些情况下，Abaqus/CAE 会向模型应用半透明度，以帮助您在过程中选择内部或隐藏的实体。当过程不需要时，您也可以使用 **颜色编码** 工具栏中的 ![](../graphics/ico_displayTranslucency.png) 工具来切换半透明度的开关。

要设置 Abaqus/CAE 使用的半透明度百分比，请点击 ![](../graphics/ico_displayTranslucency.png) 工具右侧的箭头。Abaqus/CAE 显示一个垂直滑块。向上拖动滑块使显示颜色更不透明，向下拖动使其更透明。

**注意：**在可视化模块中，您从 **通用绘图选项** 或 **叠加绘图选项** 对话框的 **其他** 页面控制透明度。请参见["自定义渲染样式、透明度和填充颜色，" 第55.2节](pt05ch55s02.md)，了解更多信息。
