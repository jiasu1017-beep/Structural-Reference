# 3.6 理解 Abaqus/CAE GUI 设置







GUI 设置在退出 Abaqus/CAE 时总是自动保存到主目录中名为 `abaqus_v6.14.gpr` 的二进制文件中。有关更多信息，请参阅[「使用 `abaqus_v6.14.gpr` 文件，第 2.1.3 节」](pt01ch02s01s03.md)。

这些 GUI 设置包括以下内容：
- 主窗口的大小和位置。
- 特定对话框的大小和位置；例如，**打开数据库**和**创建零件**对话框。
- 单个工具栏的位置、方向和可见性。
- 自定义工具栏。
- 自定义键盘快捷键。
- 消息区域和命令行界面的大小。
- 是否显示模型树和结果树。树区域的宽度也会被存储。
- 打开文件时创建的目录书签。

您不能编辑 `abaqus_v6.14.gpr` 文件；但是，您可以删除它以恢复默认的 GUI 和显示选项设置。

**警告：**删除 `abaqus_v6.14.gpr` 文件会重置上面列出的所有 GUI 设置。您不能从已删除的 `abaqus_v6.14.gpr` 文件中恢复设置，除非在 Abaqus/CAE 中手动重新创建它们。
