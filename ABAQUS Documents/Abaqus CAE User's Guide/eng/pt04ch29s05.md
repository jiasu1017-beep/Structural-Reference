# 29.5 Creating assembled fasteners







You can create assembled fasteners in the Interaction module after you have developed your template model. For more information, see ["About fasteners," Section 29.1](pt04ch29s01.md).

**To create assembled fasteners:**

1. From the main menu bar in the Interaction module, select ****Special**![](../graphics/images/arrow.gif)**Fasteners**![](../graphics/images/arrow.gif)**Create****. Abaqus/CAE displays the **Create Fasteners** dialog box. **Tip:**You can also create assembled fasteners using the ![](../graphics/ico_fastener.png) tool in the Interaction module toolbox.
2. From the **Create Fasteners** dialog box that appears, select **Assembled** fasteners, and click **Continue**.
3. Select the set of attachment points (or reference points) in your main model that will have the template model's control point mapped onto them. Use one of the following methods: - Use an existing set to define the point or points. This method lets you easily select a large number of points using a predefined set of attachment points. On the right side of the prompt area, click **Sets**. Select an existing set of points from the **Region Selection** dialog box that appears, and click **Continue**. **Note:**The default selection method is based on the selection method you most recently employed. To revert to the other method, click **Select in Viewport** or **Sets** on the right side of the prompt area. - Select one or more points in the viewport. **Tip:**The ![](../graphics/ico_filterVisible.png) **Select the Entity Closest to the Screen** tool in the **Selection** toolbar is toggled off by default. If you make an ambiguous selection, Abaqus/CAE highlights the point and displays a description of the point in the lower left corner of the viewport. Use the **Next** and **Previous** buttons to cycle through the possible selections, and click **OK** to confirm your selection. Click mouse button 2 to indicate that you have finished selecting. For more information, see ["Selecting objects within the current viewport," Section 6.2](pt01ch06s02.md). If the model contains a combination of orphan mesh instances and native geometric part instances, click one of the following from the prompt area: - Click **Geometry** if you want to select the points from a native geometric part instance or from one or more reference points. - Click **Mesh** if you want to select the points from an orphan mesh instance.
4. Click **Done** when you have finished selecting the attachment points. Abaqus/CAE displays the **Edit Fasteners** dialog box.
5. Choose the template model to be used. The template model should contain the fastener-like construct you want to replicate in the main model. The template model must be contained in the current model database (`.cae` file), and the template model must contain one or more tie, coupling, or adjust points constraints that reference a named surface.
6. Choose the control point set from the template model. This set must include the control point of the fastener construct.
7. On the **Surfaces**, **Orientations**, and **Properties** tabbed pages, configure settings for the fasteners as described in the following procedures: - ["Mapping the template model surfaces to main model surfaces," Section 29.5.1](pt04ch29s05hlb01.md) - ["Orienting the geometry read in from the template model," Section 29.5.2](pt04ch29s05hlb02.md) - ["Modifying fastener properties in the main model," Section 29.5.3](pt04ch29s05hlb03.md)
8. Click **OK** to create the assembled fasteners. Small light green squares appear in the viewport to show you the locations of the assembled fasteners.

After your assembled fasteners have been created, you can use the **Re-run** tabbed page to re-run the property generation script; see ["Re-running a property generation script," Section 29.5.4](pt04ch29s05hlb04.md).
