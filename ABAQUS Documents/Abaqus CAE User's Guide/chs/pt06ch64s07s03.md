# 64.7.3 沿 Z 轴包裹网格







使用 Abaqus Scripting Interface 方法 `wrapMesh` 将平面孤立网格包裹到由全局 *Z* 轴和指定半径定义的圆柱面上。平面网格必须位于 *X–Y* 平面中，零件的建模空间应为三维。如有必要，您可以通过在 Model Tree 中对零件点击鼠标按钮 3 并从出现的菜单中选择 **Edit** 来更改零件的建模空间。

包裹过程会将平面网格上位于点 (![](../graphics/usi_eqn00797.gif), ![](../graphics/usi_eqn00798.gif)) 处的节点重新定位到 (![](../graphics/usi_eqn00799.gif))，其中 ![](../graphics/usi_eqn00270.gif) 是指定半径，![](../graphics/usi_eqn00065.gif) = ![](../graphics/usi_eqn00800.gif)，而 ![](../graphics/usi_eqn00801.gif)=![](../graphics/usi_eqn00798.gif)，如图 [64–19](pt06ch64s07s03.md#edt-mesh-wrap) 所示。位于 (0, 0) 的平面孤立网格上的点被映射到圆柱面上的位置 (![](../graphics/usi_eqn00270.gif), 0, 0)。

**图 64–19** 包裹平面孤立网格。

![](../graphics/edt-mesh-wrap.png)

因为只有零件实例可以被定位，只有零件可以被包裹，所以如果您想创建一个在装配中正确包裹和定向的孤立网格实例，则需要一些规划。以下步骤将帮助您实现预期目标：

1. 使用 Assembly 模块中的工具在装配中平移和/或旋转平面孤立网格的实例。
2. 使用 Mesh 模块中的 **Create Mesh Part** 工具从重新定向的实例创建平面网格零件。
3. 包裹重新定向的网格零件。
4. 创建包裹零件的实例，并使用 Assembly 模块中的工具在装配中平移和/或旋转该实例。

如果您尝试创建完整的 360 度包裹，由于离散化近似，沿着接缝边缘的节点之间可能会出现小间隙。您必须使用合并节点工具通过将节点对合并为单个节点来缝合接缝；有关更多信息，请参见["合并节点，" 第 64.5.6 节](pt06ch64s05s06.md)。

以下展示了使用 Abaqus Scripting Interface 包裹网格的示例：

```
model = mdb.models['Model-1']
part = model.parts['Part-1']
part.wrapMesh(radius=5.0)
```
