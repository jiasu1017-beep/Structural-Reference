# 2.5 小结







- Abaqus 输入文件包含分析模型的完整描述。它是预处理器（例如 Abaqus/CAE）和分析产品（Abaqus/Standard 或 Abaqus/Explicit）之间的沟通桥梁。
- 输入文件包含两个部分：定义被分析结构的部分数据，以及定义结构发生什么的历史数据。
- 输入文件的每个部分由多个选项块组成，每个选项块由一个关键字行组成，后跟数据行。
- 创建输入文件后，您可以执行 **datacheck** 分析。错误和警告消息会打印到数据文件。**datacheck** 分析成功后，会将对模拟所需计算机资源的估计打印到数据文件。
- 使用 Abaqus/Viewer 验证模型几何和边界条件，使用在 **datacheck** 阶段生成的输出数据库文件以图形方式显示。
- 在数据（`.dat`）文件中检查材料属性错误通常是最简单的；几何、载荷和边界条件使用 Abaqus/Viewer 等图形后处理器更容易检查。
- 始终检查结果是否满足基本工程原理，如平衡。
- Abaqus/Viewer 允许您以多种方式图形化可视化分析结果，还可以编写表格数据报告。
- 在隐式或显式方法之间的选择很大程度上取决于问题的性质。
