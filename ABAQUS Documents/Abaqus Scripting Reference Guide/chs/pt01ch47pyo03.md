# 47.3 Color 对象





Color 对象包含系统颜色的 RGB 定义。

**访问**

```
session.colors[*name*]
```

### 47.3.1 setByRGB(...)

此方法更改用户定义颜色的 RGB 值。但是，用户无法定义颜色，此方法不会修改系统定义的颜色。

**必要参数**

*rgb*

三个 Float 的序列，指定颜色的 RGB 值。Float 值必须在 0.0 和 1.0 之间。

**可选参数**

无。

**返回值**

无

**异常**

无。

### 47.3.2 成员

Color 对象具有以下成员：

*name*

String，指定颜色的名称。

*rgb*

三个 Float 的元组，指定颜色的 RGB 值。Float 值必须在 0.0 和 1.0 之间。
