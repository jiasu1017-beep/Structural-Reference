# 3.3 小结







- Abaqus 拥有广泛的单元库，可用于各种结构应用。单元类型的选择对模拟的准确性和效率有重要影响。Abaqus/Explicit 中可用的单元（通常）是 Abaqus/Standard 中可用单元的子集。
- 节点处活跃的自由度取决于连接到节点的单元类型。
- 单元名称完全标识单元的家族、公式、节点数和积分类型。
- 所有单元必须引用截面属性定义。截面属性提供定义单元几何所需的任何额外数据，并标识关联的材料属性定义。
- 对于连续体单元，Abaqus 根据全局笛卡尔坐标系定义单元输出变量（如应力和应变）。您可以使用 [*ORIENTATION](../key/key-link.md#usb-kws-morientation) 选项更改为局部坐标系。
- 对于三维壳单元，Abaqus 根据基于壳表面的坐标系定义单元输出变量。您可以使用 [*ORIENTATION](../key/key-link.md#usb-kws-morientation) 选项更改为局部坐标系。
- 为了计算效率，模型的任何部分都可以定义为刚体，刚体仅在其参考节点处具有自由度。
- 作为 Abaqus/Explicit 分析中的约束方法，刚体在计算上比多点约束更高效。
