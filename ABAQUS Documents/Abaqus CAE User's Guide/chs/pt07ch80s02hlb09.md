# 80.2.9 自定义切片选项





您可以使用可视化模块中 **Slicing** 选项卡页面上的设置来沿活动视图切割的范围显示一系列视图切割切片。视图切割切片类似于视图切割的切割处部分，但它们沿活动视图切割的范围以一系列间隔显示，而不是仅在切割平面上显示。您可以沿活动视图切割的范围以规则间隔显示视图切割切片，或者可以沿会话中预定义的路径显示切片。启用切片时，您还可以在模型的那个位置显示合力和力矩。

切片仅在单个视图切割处于活动状态时才能启用，且仅在可视化模块中。您可以为平移和旋转视图切割显示切片。切片沿活动视图切割的整个范围显示，无论在 **View Cut Manager** 中 **Motion of Selected Cut** 选项下如何重新定位视图切割。对于旋转视图切割，您还可以旋转切片并更改其旋转轴。

如果您在具有大量折叠或曲率的模型中沿路径启用视图切割切片，切片的切割表面有时会同时穿过模型的不同部分。您可以通过聚焦模型的子集来限制或消除此行为：要么使用显示组隐藏模型的某个部分，要么更改路径定义，以便切片仅沿部分曲率显示。

**自定义活动视图切割的切片的步骤：**

1. 在可视化模块中，从主菜单栏中选择****Options**![](../graphics/images/arrow.gif)**View Cut****。出现 **View Cut Options** 对话框。
   **提示：**您也可以使用视图切割管理器自定义切割选项。从主菜单栏中选择****Tools**![](../graphics/images/arrow.gif)**View Cut**![](../graphics/images/arrow.gif)**Manager****，然后在视图切割管理器中单击 **Options**。
2. 单击 **Slicing** 选项卡。
3. 切换 **Display slicing** 以启用会话中视图切割的切片。
4. 如果您想沿视图切割的整个范围显示视图切割切片，请选择 **Step through the active view cut's range**。如果您在多个位置显示视图切割切片或自由体切割，您还可以自定义 Abaqus/CAE 将显示合力和力矩的线段内的最小值和最大值。默认情况下，最小值和最大值设置为允许切片分布在整个模型中。
5. 如果您想沿预定义的路径显示视图切割切片，请选择 **Step along a predefined path**，然后执行以下操作：
   1. 选择路径。
   2. 使用 **Display slices at path nodes** 选项指定要显示视图切割切片的位置：
      - 切换此选项可在构成路径定义的节点处放置视图切割切片。
      - 关闭此选项可在路径上的一系列规则间隔处放置视图切割切片。从对话框底部的字段和滑块中指定切片的数量。
   3. 使用 **Set free body summation point on the path** 选项指定合力和力矩的求和点位置：
      - 切换此选项可将合力和力矩的求和点放置在选定路径上。
      - 关闭此选项可将合力和力矩的求和点放置在获取合力矩的位置。
6. 单击 **Apply** 以实现您的更改。视图切割会更改以反映您的规范。默认情况下，您的更改会在会话期间保留，并会影响该视口中所有后续的视图切割。如果您想保留更改以供后续会话使用，请将它们保存到文件。有关更多信息，请参见["保存自定义设置以供后续会话使用，" 第55.1.1节](pt05ch55s01s01.md)。
