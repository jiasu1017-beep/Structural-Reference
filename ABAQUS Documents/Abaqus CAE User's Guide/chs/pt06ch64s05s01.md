# 64.5.1 创建节点





从主菜单栏选择****网格**![](../graphics/images/arrow.gif)**编辑****，可以通过在全局坐标系或局部坐标系中输入坐标来创建节点。

**创建节点的步骤：**

1. 进入"网格"模块。
2. 从上下文栏的**对象**字段中选择**零件**，然后从列表中选择一个零件。
3. 从主菜单栏选择****网格**![](../graphics/images/arrow.gif)**编辑****。Abaqus/CAE显示**编辑网格**对话框。 **提示：**您也可以使用位于"网格"模块工具箱底部的![](../graphics/ico_meshEdit.png)工具显示**编辑网格**对话框。
4. 在对话框中执行以下操作： 1. 在**类别**字段中选择**节点**。 2. 从**方法**列表中选择**创建**。Abaqus/CAE显示现有节点。在隐藏和着色模式下，仅显示可见节点。
5. 如果需要，您可以在已创建的基准坐标系中而不是在全局坐标系中输入新节点的坐标： 1. 在提示区域点击**选择**。 2. 在视口中，选择要与新节点坐标关联的基准坐标系。（有关更多信息，请参见["创建基准坐标系，" 第62.9节](pt06ch62s09.md)。）
6. 在提示区域的**坐标**字段中，输入新节点的坐标。Abaqus/CAE会创建新节点。 **提示：**如果在创建节点时出错，点击**编辑网格**对话框中的**撤销**可以移除最近创建的节点。
7. 根据需要重复步骤3和4以创建其他节点。
8. 完成节点创建后，点击提示区域中的取消按钮![](../graphics/afxI_cancel.png)退出过程。
