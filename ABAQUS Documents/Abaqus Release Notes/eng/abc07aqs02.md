# 7.2 Linear pyramid acoustic element







**Product: **Abaqus/Standard  

**Benefits: **The pyramid element is useful to transition between brick elements and tetrahedra elements during mesh generation.

**Description: **Acoustic elements are provided in Abaqus/Standard for modeling acoustic domain in a purely acoustic analysis or in a coupled acoustic-structural analysis. The new 5-node pyramid element AC3D5 supports all of the loadings, material behaviors, and boundary conditions supported by the existing acoustic elements.
