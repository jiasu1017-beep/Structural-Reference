# 49.1 理解动画





Abaqus/CAE提供两种动画方式：

**基于对象的动画**

基于对象的动画是变形形状图、等值线图、符号图或材料方向图序列的显示。Abaqus/CAE可以生成三种不同类型的这些图序列。这三种序列类型称为"时间历史动画"、"比例因子动画"和"谐波动画"。

时间历史动画生成根据实际分析结果随时间变化的变形形状、等值线、符号或材料方向图的序列。比例因子动画生成仅缩放单个变形形状、等值线或符号图的图像序列。谐波动画生成表示复数结果值根据施加角度变化的图像序列。要更好地理解这三种动画类型，请分别参阅["时间历史动画，"第49.1.1节](pt05ch49s01s01.md)、["比例因子动画，"第49.1.2节](pt05ch49s01s02.md)和["谐波动画，"第49.1.3节](pt05ch49s01s03.md)。

当基于对象的动画播放时，您可以动态更改显示特性，例如视图、任何视口注释以及绘图状态相关的自定义选项。

**基于图像的动画**

基于图像的动画是动画文件的播放。您可以通过首先在一个或多个视口中播放基于对象的动画，然后从主菜单栏中选择****动画**![](../graphics/images/arrow.gif)**另存为****来在Abaqus/CAE中创建动画文件。保存后，您可以使用行业标准动画软件在Abaqus/CAE外部播放您的动画。您可以选择以QuickTime或Audio Video Interleave (AVI)格式保存基于图像的动画文件。

在Abaqus/CAE内部，您还可以通过将动画文件选择为背景电影来显示基于图像的动画。活动时，背景电影显示在可视化模块的视口中。有关详细信息，请参阅["显示和自定义背景电影，"第4.6.3节](pt01ch04s06hlb02.md)。

**注意：**Abaqus/CAE还使您能够以VRML格式保存动画，这会创建动画的三维渲染。由于这些文件是三维的，它们严格来说不是基于图像的；但是，您可以像播放和分发QuickTime或AVI格式的文件一样播放和分发VRML格式的动画文件。

通常，从文件播放动画比基于对象的动画提供更好的性能，特别是对于大型模型。当基于图像的动画播放时，您无法更改其显示特性。
