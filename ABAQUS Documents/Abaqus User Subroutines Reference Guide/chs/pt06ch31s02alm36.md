### 概述

连接单轴行为：
- 可通过指定加载和卸载行为在任何具有相对运动可用分量的连接器中定义；
- 可独立为每个相对运动的可用分量指定；
- 可定义拉伸和压缩方向上的单独响应；
- 可表现出非线性弹性行为、损伤弹性行为，或具有完全卸载后永久变形的弹塑性类型行为；
- 可指定卸载响应；以及
- 可指定为依赖于几个局部方向中的本构运动。

每种连接类型的局部方向（如 ["连接类型库，" 第31.1.5节](pt06ch31s01aus114.md) 中所述）决定了力/力矩作用的方向以及位移/旋转的测量方向。

### 为相对运动的可用分量指定单轴行为

通过定义该分量的加载/卸载响应，可为相对运动的可用分量指定单轴行为。对于每个分量，可以为拉伸和压缩方向的响应分别定义加载/卸载响应数据。加载和卸载响应可根据三种可用的行为类型进行分类：
- 非线性弹性行为；
- 损伤弹性行为；和
- 具有永久变形的弹塑性类型行为。

要定义加载响应，您可以将力或力矩指定为相对运动分量的非线性函数。这些函数也可以依赖于温度、场变量以及其他分量方向中的本构位移/旋转。有关将数据定义为温度和场变量函数的信息，请参见 ["输入语法规则，" 第1.2.1节](pt01ch01s02aus01.md)。

卸载响应可以通过以下方式定义：
- 您可以指定几条卸载曲线，这些曲线将力或力矩表示为相对运动分量的非线性函数；Abaqus 对这些曲线进行插值，创建一个经过分析中卸载点的卸载曲线。
- 您可以指定能量耗散因子（以及对于具有永久变形的模型，指定永久变形因子），Abaqus 从中计算指数/二次卸载函数。
- 您可以将力或力矩指定为相对运动分量的非线性函数，并指定过渡斜率；连接器沿指定的过渡斜率卸载，直到与指定的卸载函数相交，此时它根据该函数进行卸载。（这种卸载定义称为组合卸载。）
- 您可以将力或力矩指定为相对运动分量的非线性函数；Abaqus 沿应变轴平移指定的卸载函数，使其经过分析中的卸载点。

#### 定义变形方向

可以通过指定变形方向分别为拉伸和压缩定义加载/卸载数据。如果定义了变形方向（拉伸或压缩），则应使用相对运动指定分量中力/力矩和位移/旋转的正值来指定定义拉伸或压缩行为的数据表，加载数据必须从原点开始。如果未在加载方向上定义行为，则该方向的力响应为零（连接器在该方向上没有阻力）。

如果未定义变形方向，数据适用于拉伸和压缩。但是，此时行为被视为非线性弹性，不能指定损伤或永久变形。如果省略了拉伸或压缩数据，响应数据将被视为关于原点对称。

### 定义带损伤的模型

损伤模型在卸载时耗散能量，并且在完全卸载时没有永久变形。卸载行为控制损伤机制耗散的能量量，可以通过以下方式之一指定：
- 解析卸载曲线（指数/二次）；
- 从多个用户指定的卸载曲线插值的卸载曲线；或
- 沿过渡卸载曲线（由用户指定的恒定斜率）卸载到用户指定的卸载曲线（组合卸载）。

有关不同可用行为的概述，请参见上文 ["为相对运动的可用分量指定单轴行为"](pt06ch31s02alm36.md#usb-elm-econnuniaxialbehav-behavior)。各种卸载类型将在下文中讨论。

#### 定义损伤起始

您可以通过定义位移来指定损伤起始，低于该位移时沿加载曲线发生卸载。

| **输入文件用法：** | ``` [*LOADING DATA](../key/key-link.md#usb-kws-mloadingdata), TYPE=DAMAGE, DAMAGE ONSET=*value* ``` |

### 定义带永久变形的模型

这些模型在卸载时耗散能量，并在完全卸载时表现出永久变形。卸载行为控制耗散的能量量以及永久变形量。卸载行为可以通过以下方式之一指定：
- 解析卸载曲线（指数/二次）；
- 从多个用户指定的卸载曲线插值的卸载曲线；或
- 通过将用户指定的卸载曲线平移到卸载点获得的卸载曲线。

有关不同可用行为的概述，请参见上文 ["为相对运动的可用分量指定单轴行为"](pt06ch31s02alm36.md#usb-elm-econnuniaxialbehav-behavior)。各种卸载类型将在下文中讨论。

#### 定义永久变形起始

默认情况下，屈服的起始将在沿加载曲线行进时，加载曲线斜率从此时记录的最大斜率下降 10% 时获得。要覆盖确定屈服起始的默认方法，您可以指定加载曲线斜率下降值（而不是默认值 10%，即斜率下降 = 0.1），或者通过定义位移来指定低于该位移时沿加载曲线发生卸载。如果指定了斜率下降，则当加载曲线斜率从此时记录的最大斜率下降指定因子时，将获得屈服起始。

| **输入文件用法：** | 使用以下选项通过定义位移来指定屈服起始，低于该位移时沿加载曲线发生卸载： |
| --- | --- |
|  | ``` [*LOADING DATA](../key/key-link.md#usb-kws-mloadingdata), TYPE=PERMANENT DEFORMATION, YIELD ONSET=*value* ``` 使用以下选项通过定义加载曲线的斜率下降来指定屈服起始： ``` [*LOADING DATA](../key/key-link.md#usb-kws-mloadingdata), TYPE=PERMANENT DEFORMATION, SLOPE DROP=*value* ``` |
