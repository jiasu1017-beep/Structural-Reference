# 15.9.7 连接器截面编辑器





要创建连接器截面，从主菜单栏选择****Connector**![](../graphics/images/arrow.gif)**Section**![](../graphics/images/arrow.gif)**Create****。出现**创建连接器截面**对话框，你可以在其中指定要创建的连接器截面的名称、类别和类型。当你从**Assembled/Complex**或**Basic**类别中选择连接类型时，该连接类型的可用和约束相对运动分量（CORM）会显示在对话框中。此外，你可以点击![](../graphics/ico_tipSmall.png)查看连接类型的示意图以及Abaqus对连接的理想化。指定名称、类别和类型后，点击**创建连接器截面**对话框中的**继续**以显示连接器截面编辑器。对于**MPC**类别中的连接类型，选择类型并在需要时输入数据。点击**确定**完成创建**MPC**截面并关闭**创建连接器截面**对话框。

连接器截面编辑器允许你添加Abaqus/Standard和Abaqus/Explicit中可用的连接器行为。当你在**行为选项**选项卡页面上点击**添加**时，会出现行为列表。选择行为后，行为名称出现在编辑器顶部的**行为选项**列表中，该行为成为连接器截面定义的一部分。编辑器下半部分的选项定义区域会更改以提供字段，你可以在其中为当前选定的行为指定信息。如果你想从连接器截面定义中移除行为，可以从**行为选项**列表中选择它，然后点击**删除**。

**注意：**Abaqus/CAE不执行任何关于其他行为依赖性的检查；你应确保定义了所有所需的行为。例如，如果你定义了塑性行为，则还必须定义弹性行为。

你可以定义相同类型的多个行为，例如弹性。只有与所选连接类型的可用相对运动分量一致的作用力或力矩才能被选择来定义行为。例如，[图15-20](pt03ch15s09s07.md#itn-connpropeditor)中的**行为选项**列表反映了两个**弹性**行为和一个**参考长度**行为已包含在连接器截面定义中。高亮的**弹性**行为在与所选力矩相同的方向上定义弹性行为。

**图15-20** 连接器截面编辑器。

![](../graphics/itn-connpropeditor-nls.png)

在**表格选项**选项卡页面上，你可以为连接器截面中的所有行为选项指定正则化（Abaqus/Explicit分析仅适用）和表格数据外推的行为设置。或者，你可以通过点击编辑器下半部分选项定义区域中的**表格选项**按钮为单个行为选项指定行为设置。单个行为选项的行为设置优先于连接器截面行为设置。

当截面数据适用于指定连接类型时，你可以在**截面数据**选项卡页面上输入数据。

你可以通过从主菜单栏选择****Help**![](../graphics/images/arrow.gif)**On Context****然后点击感兴趣的功能来显示有关编辑器特定功能的信息。有关创建连接器截面和定义行为和截面数据的详细说明，请参见["创建连接器截面，"第15.12.11节](pt03ch15s12s01.md)和["使用连接器截面编辑器，"第15.17节](pt03ch15s17.md)。
