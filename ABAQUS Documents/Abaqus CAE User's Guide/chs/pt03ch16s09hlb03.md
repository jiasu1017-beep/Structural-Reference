# 16.9.3 定义压力载荷





您可以创建压力载荷以在表面上定义压力。

**创建或编辑压力载荷：**

1. 使用以下方法之一显示压力载荷编辑器：- 若要创建新的压力载荷，请遵循"创建载荷"，第16.8.1节中概述的过程（**类别**：`Mechanical`；**所选步骤的类型**：`Pressure`）。- 若要使用菜单或管理器编辑现有的压力载荷，请参见"编辑依赖于步骤的对象"，第3.4.13节。若要编辑应用载荷的区域，请参见"编辑应用规定条件的区域"，第16.8.4节。
2. 单击**分布**字段右侧的箭头，并从出现的列表中选择您的选项：- 选择**均匀**以定义在表面上均匀分布的压力。对于此选项，您提供的数值必须是每单位面积的力。- 选择**总力**以定义在表面上均匀分布的压力。对于此选项，您提供的数值必须是将应用于表面的力的总幅值（而不是每单位面积的力）。- 选择**静水压**以定义应用于表面的静水压力。（此选项仅对Abaqus/Standard分析有效。）- 选择**滞止**以定义应用于表面的滞止压力。（此选项仅对Abaqus/Explicit分析有效。）- 选择**粘性**以定义应用于表面的粘性压力。（此选项仅对Abaqus/Explicit分析有效。）- 选择**用户定义**以在用户子程序[`DLOAD`](../sub/sub-link.md#sub-xsl-dload)（对于Abaqus/Standard）或[`VDLOAD`](../sub/sub-link.md#sub-xsl-vdload)（对于Abaqus/Explicit）中定义载荷幅值。有关更多信息，请参见以下章节：- "指定常规作业设置"，第19.8.6节- "Abaqus User Subroutines Reference Guide"第1.1.5节"DLOAD"- "Abaqus User Subroutines Reference Guide"第1.2.2节"VDLOAD"- 选择分析场（标记为(A)）或离散场（标记为(D)）以定义空间变化压力。只有对这些载荷类型有效的分析场和离散场才会在选择列表中显示。或者，您可以单击![](../graphics/ico_analyticalField.png)创建新的分析场。（更多信息，请参见第58章"分析场工具集"。）
3. 如果您选择了**均匀**、**总力**、分析场或离散场分布选项，请执行以下步骤：1. 在**幅值**文本字段中，输入压力幅值。对于**均匀**分布，输入所施加力的总幅值除以施加力的表面面积（单位[FL<sup>2</sup>](../popups/usb-int-iconventions-unitsym.md)）。对于**总力**分布，输入力的总幅值（单位[F](../popups/usb-int-iconventions-unitsym.md)）。基于未变形模型几何，Abaqus/CAE根据输入的力幅值计算恒定的均匀表面压力。然而，在大位移分析中，由于加载表面的变形，实际总力可能在分析过程中发生变化。2. 如果需要，单击**幅值**字段右侧的箭头，并从出现的列表中选择您选择的幅值。或者，您可以单击![](../graphics/ico_amplitudeSmall.png)创建新的幅值。（更多信息，请参见第57章"幅值工具集"。）3. 单击**确定**保存您的数据并退出编辑器。
4. 如果您选择了**静水压**分布选项，请执行以下步骤：1. 在**幅值**文本字段中，输入压力幅值（单位[FL<sup>2</sup>](../popups/usb-int-iconventions-unitsym.md)）。2. 在**零压力高度**字段中，输入压力为零的高度处的*Z*坐标（如果您正在三维或轴对称空间工作）或*Y*坐标（如果您正在二维空间工作）。3. 在**参考压力高度**字段中，输入压力为**幅值**字段中指定数值处的高度处的*Z*坐标（如果您正在三维或轴对称空间工作）或*Y*坐标（如果您正在二维空间工作）。（更多信息，请参见"Abaqus Analysis User's Guide"第34.4.3节"Distributed loads"中的"Abaqus/Standard中的二维、三维和轴对称单元上的静水压力载荷"。）
5. 如果您选择了**滞止**或**粘性**分布选项，请执行以下步骤：1. 在**幅值**文本字段中，输入压力幅值（单位[FL<sup>2</sup>](../popups/usb-int-iconventions-unitsym.md)）。2. 如果需要，单击**幅值**字段右侧的箭头，并从出现的列表中选择您选择的幅值。或者，您可以单击![](../graphics/ico_amplitudeSmall.png)创建新的幅值。（更多信息，请参见第57章"幅值工具集"。）3. 如果需要，切换打开**从参考点确定速度**以从施加压力的表面的速度中减去参考节点的速度。4. 单击![](../graphics/ico_selectBlue.png)使用以下方法之一选择参考点：- 从视口中选择一个点。- 在提示区域中单击**点**，然后选择一个命名集合。**注意：**您选择的集合必须包含单个节点或顶点。5. 单击**确定**保存您的数据并退出编辑器。
6. 如果您选择了**用户定义**分布选项，请执行以下步骤：1. 如果需要，在**幅值**字段中输入压力幅值（单位[FL<sup>2</sup>](../popups/usb-int-iconventions-unitsym.md)）。您在编辑器中输入的幅值数据在Abaqus/Standard分析中被传递到用户子程序中，但在Abaqus/Explicit分析中被忽略。2. 单击**确定**保存您的数据并退出编辑器。3. 进入作业模块并显示感兴趣的分析作业的作业编辑器。（更多信息，请参见"创建、编辑和操作作业"，第19.7节。）4. 在作业编辑器中，单击**常规**选项卡，并指定包含定义载荷幅值的用户子程序的文件。更多信息，请参见"指定常规作业设置"，第19.8.6节。**注意：**您只能在作业编辑器中指定一个用户子程序文件；如果您的分析涉及多个用户子程序，您必须将用户子程序合并到一个文件中，然后指定该文件。
