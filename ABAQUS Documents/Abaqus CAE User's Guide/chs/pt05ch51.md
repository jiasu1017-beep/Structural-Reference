# 51.1 理解探测





当您点击可视化模块工具箱中的 ![](../graphics/ico_probe.png) 工具时，Abaqus/CAE 进入探测模式，当您将光标移到当前视口中的模型上时，会在 **探针值（Probe Values）** 对话框中显示信息。您也可以使用查询工具栏中的 ![](../graphics/ico_info.png) 工具进入探测模式。探测模型绘图会显示模型数据和结果；探测 X–Y 绘图会显示 X–Y 曲线数据；从当前模型数据库探测模型会显示模型数据以及属性值（如载荷和预定义场）的信息。您可以将此信息写入文件。

**探针值（Probe Values）** 对话框底部的数据表包含两个动态显示行（紧邻行标题上方的行以及在表中首先出现的粗体数据行）和一系列额外的存储行。Abaqus/CAE 不断更新动态行，以在您将光标移到当前视口中的模型上时显示探针值，您可以通过点击列标题来更改顶部显示行中显示的值。Abaqus/CAE 仅在您通过在视口中点击、指定节点或单元标签或在特定显示组中选择节点或单元来选择感兴趣的值时，才使用额外行更新表格。值会累积在对话框底部，直到您删除它们、清除表格或取消探测模式。您还可以将选定的值添加到新的显示组，并且您可以使用该位置的场输出值注释各个节点或单元。

# 51.1.2 X–Y 绘图探测





### 51.1.2 X–Y 绘图探测

如果当前视口包含 X–Y 绘图，Abaqus/CAE 会将 X–Y 曲线图例文本、曲线内每个点的序列标识以及曲线点的 X 和 Y 坐标显示为探针值。您可以选择仅显示构成 X–Y 数据对象的曲线点，或通过插值显示曲线上的任意点。





# 51.1.1 模型或模型绘图的探测





如果当前视口包含模型绘图（未变形、变形、云图、符号或材料取向绘图）或来自当前模型数据库的模型，Abaqus/CAE 允许您探测来自所选模型或输出数据库的结果。**探针值（Probe Values）** 对话框标识 Abaqus/CAE 将获取其值的步骤、帧和场输出变量。要了解如何更改步骤、帧或场输出变量，请参阅 ["选择结果步骤和帧，" 第 42.3 节"](pt05ch42s01.md) 和 ["选择主场输出变量，" 第 42.5.3 节"](pt05ch42s03hlb03.md)。

当您探测三维壳或复合实体的模型绘图时，Abaqus/CAE 返回的结果取决于您在 **截面点（Section Points）** 对话框中选择的活动位置。当 **顶部（Top）** 是活动位置时，Abaqus/CAE 从 **顶部位置（Top Location）** 截面点返回结果；当活动位置设置为 **底部（Bottom）** 或 **顶部和底部（Top and bottom）** 时，从 **底部位置（Bottom Location）** 截面点返回结果。有关截面点的更多信息，请参阅 ["选择截面点数据，" 第 42.5.9 节"](pt05ch42s03s01.md)。

对于模型绘图，您可以选择获取基于节点或基于单元的数据和结果，如下所示：

**探测节点**

当您选择探测节点并将光标定位在当前视口中的节点位置时，Abaqus/CAE 高亮显示该节点，并在数据表中显示以下有关该节点的信息：
- 节点的标签，
- 节点的原始坐标，
- 节点的变形坐标（使用当前变形场输出变量），
- 共享该节点的单元，以及
- 当前场输出变量（包括分量值，如果当前场输出变量是张量或矢量）。如果在当前视口中选择了模型，当前场输出变量反映模型属性（如所选节点处的载荷）的值。

无论当前主变量最初是保存在节点、形心还是积分点位置的输出数据库中，您都可以显示该节点的场输出。Abaqus/CAE 通过外推到节点并根据您选择的选项进行平均来计算形心和积分点数据的节点值。如果节点引用未平均的单元数据，将显示与探测节点关联的所有单元数据。有关平均的更多信息，请参阅 ["理解结果值平均，" 第 42.6.2 节"](pt05ch42s04s02.md)。

您还可以显示或隐藏为每个所选节点出现在视口中的注释。探测注释显示节点的标签以及该节点处当前场输出变量的结果。您可以自定义探针注释文本的字体和颜色。

**探测单元**

当您选择探测单元并将光标定位在当前视口中的单元上时，Abaqus/CAE 高亮显示该单元，并在数据表中显示以下有关该单元的信息：
- 单元的标签，
- 单元的连通性，以及
- 当前场输出变量（包括分量值，如果当前场输出变量是张量或矢量）。如果在当前视口中选择了模型，当前场输出变量反映模型属性（如所选单元处的载荷）的值。

仅当当前主变量是基于单元的变量（如应力或应变）时，才能显示该单元的场输出。您可以选择计算场输出结果的输出位置：积分点、形心、单元节点或单元面。对于这些输出位置中的每一个，Abaqus/CAE 以逐单元为基础计算探测结果，不进行平均。

您还可以显示或隐藏为每个所选单元出现在视口中的注释。探测注释显示单元的标签以及该单元处当前场输出变量的结果。您可以自定义探针注释文本的字体和颜色。

无论模型以何种渲染样式显示，只能探测可见单元。如有必要，您可以创建显示组或使用视图切割来显示原本不可用的内部单元。

当单元被裂纹切割时，裂纹单元会分成两部分，每部分由实域和虚域组成。当您探测裂纹单元时，仅报告来自单元一部分的贡献。有关更多信息，请参阅 ["使用扩展有限元方法将不连续性建模为富集特征，" Abaqus Analysis User's Guide 第 10.7.1 节中的"可视化"](../usb/usb-link.md#usb-anl-aenrichment-vis)。
