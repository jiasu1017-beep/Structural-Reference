# 12.4.3 创建截面





您可以使用Property模块创建以下类型的截面：
- 均匀实体截面
- 广义平面应变截面
- 欧拉截面
- 复合实体截面
- 电磁实体截面
- 均匀壳截面
- 复合壳截面
- 膜截面
- 表面截面
- 通用壳刚度截面
- 梁截面
- 桁架截面
- 流体截面
- 垫片截面
- 内聚截面
- 声学无限截面
- 声学界面截面

要创建截面，请从主菜单栏选择****Section**![](../graphics/images/arrow.gif)**Create****。出现**Create Section**对话框，您可以在其中命名截面并指定要创建的截面类型。指定截面名称和类型后，点击**Create Section**对话框中的**Continue**以显示截面编辑器，该编辑器允许您创建和编辑截面。

截面编辑器的格式因您定义的截面类型而异。例如，均匀壳截面编辑器如图[12-5](pt03ch12s04s03.md#prp-homshelleditor-nls)所示。

**图12-5** 均匀壳截面编辑器。

![](../graphics/prp-homshelleditor-nls.png)

**注意：**您可以通过从主菜单栏选择****Help**![](../graphics/images/arrow.gif)**On Context****，然后点击感兴趣的编辑器功能，来显示有关此处未讨论的编辑器特定方面的帮助。将出现一个包含本指南相关部分的帮助窗口。

某些编辑器包含**Rebar Layers**选项（![](../graphics/ico_rebarLayers.png)图标），如图[12-5](pt03ch12s04s03.md#prp-homshelleditor-nls)所示。如果您点击此图标，会出现另一个对话框，您可以在其中输入有关钢筋层的数据，如图[12-6](pt03ch12s04s03.md#prp-rebarlayeroptions)所示。

**图12-6** **Rebar Layers**对话框。

![](../graphics/prp-rebarlayeroptions-nls.png)

**注意：**要显示**Rebar Layers**对话框中项目的上下文相关帮助，您必须选择感兴趣的项目，然后按**[F1]**。（显示选项对话框时，主菜单栏中的**Help**菜单不可用。）

输入定义截面所需的所有数据后，您可以点击**OK**关闭截面编辑器并保存截面。
