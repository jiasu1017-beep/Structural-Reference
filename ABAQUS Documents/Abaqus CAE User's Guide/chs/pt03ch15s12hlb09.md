# 15.12.9 为多个连接器创建或修改线特征





在「相互作用」模块中，从主菜单栏选择 ****Connector**![](../graphics/images/arrow.gif)**Geometry**![](../graphics/images/arrow.gif)**Create Wire Feature**** 以添加一个或多个线特征。您可以添加不相交的线、链接的线或连接到地面的线。创建线特征时，您可以创建一个几何集合，包含线特征中的所有线，以在后续选择过程中使用；例如，当您为连接器截面分配定义或应用连接器载荷时选择线。有关创建装配级线特征的详细说明，请参见「添加点对点线特征」，第11.23.2节。

您可以通过从主菜单栏选择 ****Connector**![](../graphics/images/arrow.gif)**Geometry**![](../graphics/images/arrow.gif)**Modify Wire Feature**** 来修改装配级线特征。您选择要修改的特征中的任何线，并在下面的过程中在「Modify Wire Feature」对话框中进行更改。当您在对话框中点击「OK」时，Abaqus/CAE会执行以下操作：
- 重命名并抑制原始线特征；
- 使用与原始线特征相同的名称创建新线特征；
- 重命名包含原始线特征的几何集合（如果存在）；以及
- 如果适用，创建一个包含修改后线特征的新几何集合。

例如，[图15-23](pt03ch15s12hlb09.md#itn-modify-wire)显示`Wire-1`被重命名为`OldWire-1-1`并被抑制。类似地，`Wire-1-Set–1`被重命名为`OldWire-1-Set–1`。

**图15-23** 模型树显示原始线特征（左）和修改后的线特征（右）的「Features」和「Sets」容器。

![](../graphics/itn-modify-wire-nls.png)

通过使用为线特征创建默认几何集名称来选择线以定义连接器截面分配、连接器载荷和连接器边界条件，您可以确保在修改线特征时这些对象保持有效。否则，如果修改线特征，引用原始线特征任何部分的对象将失效；例如，如果您定义了连接器力并从视口选择线或使用不同名称的几何集。

您可以通过从主菜单栏选择 ****Connector**![](../graphics/images/arrow.gif)**Geometry**![](../graphics/images/arrow.gif)**Remove Wires From Feature**** 从线特征中移除线。从特征中移除线边的操作存储为特征；因此，您可以使用模型树删除或抑制该操作。

**修改装配级线特征的步骤：**

1. 在「相互作用」模块中，从主菜单栏选择 ****Connector**![](../graphics/images/arrow.gif)**Geometry**![](../graphics/images/arrow.gif)**Modify Wire Feature****。**提示：**您也可以使用「相互作用」模块工具箱中的![](../graphics/ico_wireFeatureModify.png)工具修改装配级线特征。
2. 在视口中，选择要修改的特征的任何线。「Modify Wire Feature」对话框会出现，并显示定义原始线特征中装配级线的点对。
3. 在对话框的「Point Pairs」部分，您可以执行以下操作：- 要添加更多点对，指定方法，点击![](../graphics/ico_add.png)，并从视口中选择点。所添加线的表示以品红色高亮显示。- 要编辑点，在表格中选择点，点击![](../graphics/ico_edit.png)，并重新选择点。视口中的选择高亮会更新以显示新编辑的点。- 要在视口中识别特定的点对，选择所需的行。连接所选点对的线会以红色高亮显示。- 要移除点对，选择所需的行并点击![](../graphics/ico_delete.png)。- 要交换点对中「Point 1」和「Point 2」的条目，选择所需的行并点击![](../graphics/ico_swap.png)。
4. 在对话框的「Set Creation」部分，如果您希望Abaqus/CAE为修改后的线特征创建新的线几何集，请切换「Create set of wires」。
5. 点击「OK」修改线特征。原始线特征被重命名并抑制。修改后的装配级线特征出现在模型树中「Features」容器下的装配中，使用与原始线特征相同的名称。
