# 55.5.1 设置标签字体





Abaqus/CAE使用单一字体显示所有单元、面和节点标签。您可以为所有曲线图状态设置此字体的属性(字体族、大小、粗体、斜体),并且可以为叠加曲线图中的未变形形状单独设置。

**设置标签字体的步骤:**

1. 找到公共或叠加标签字体选项。公共字体选项始终应用于变形形状;它们适用于在任何曲线图状态(未变形、变形、云图、符号或材料取向)中单独绘制未变形形状时。当未变形形状与变形形状一起绘制时,应用叠加标签字体选项。从主菜单栏,选择 ****Options**![](../graphics/images/arrow.gif)**Common**** 或 **Superimpose**;然后在出现的对话框中点击"**Labels**"选项卡。字体选项在"**Labels**"页面顶部。
2. 点击"**Set Font for All Model Labels**"。出现"选择字体"对话框。
3. 使用"选择字体"对话框设置您想要的字体属性。
4. 当您设置了所需的字体属性后,点击"**OK**"。关闭"选择字体"对话框。
5. 点击"**Apply**"实施您的更改。单元、面和节点标签根据您的规格进行更改。默认情况下,您的更改会在会话期间保存,并会影响当前视口和从当前视口创建的任何新视口中的所有后续曲线图。如果要为后续会话保留您应用的更改,请将它们保存到文件。更多信息,请参阅 ["为后续会话保存自定义," 第55.1.1节](pt05ch55s01s01.md)。
