### 概述

连续体粒子单元：
- 仅能用于显式动态分析；
- 必须只有一个节点；
- 有一个积分点；
- 可以类似于连续体单元的方式进行初始化；并且
- 完全充满材料。

### 典型应用

连续体粒子单元（PC3D）适用于涉及材料经历极端变形的模拟，如开表面流体流动或固体结构的消灭/碎裂。它们仅使用一个节点定义；但是，以给定节点（粒子）为中心的单元接收影响球范围内所有粒子的贡献，其半径通常称为平滑长度。光顺粒子流体力学（SPH）公式在分析的每个增量中确定与给定粒子关联的连通性。由于节点连通性不固定，避免了严重的单元畸变，因此该公式允许非常高的应变梯度。

1节点 PC3D 单元用于定义要建模物体表面和内部的点。您可以像质量单元一样定义这些节点，并且可以将节点放置在与常规砖块网格节点相同的空间中。光顺粒子流体力学网格通常是均匀分布的单元网格，与被建模物体的形状一致。

更多信息，请参阅["光滑粒子流体力学，" 第15.2.1节](pt04ch15s02aus95.md)。

### 定义单元的截面属性

您必须将固体截面定义与一组连续体粒子单元相关联。截面定义提供了与 PC3D 单元关联的材料。

作为固体截面定义的一部分，您可以定义特征长度。此特征长度，不要与平滑长度混淆，用于计算粒子体积。体积假定为一个立方体，其边长等于指定特征长度的两倍。

| **输入文件用法：** | ``` [*SOLID SECTION](../key/key-link.md#usb-kws-msolidsection), ELSET=*element_set_name* *characteristic length associated with the particle volume* ``` |
| --- | --- |
|  | 其中 ELSET 参数引用一组粒子单元。 |
