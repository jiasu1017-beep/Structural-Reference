# 20.19.3 Offsetting the edges of Sketcher objects







Use the offset tool from the Sketcher toolbox to offset edges. You select the edges, enter the offset distance, and choose the offset direction. Abaqus/CAE applies the offset distance perpendicular to each point along the edges to create the offset. 

**Note:**Edges containing features with dimensions of similar size to the offset distance will lose their details in the offset copy.

**To offset Sketcher edges:**

1. From the Sketcher toolbox, select the offset tool ![](../graphics/ico_sketchOffset.png). For a diagram of the tools in the Sketcher toolbox, see ["The Sketcher tools," Section 20.4.1](pt03ch20s04s01.md). Abaqus/CAE displays prompts in the prompt area to guide you through the procedure.
2. Select a single group of connected edges to offset. Only sketch geometry can be offset; [construction geometry](pt03ch20s05s02.md), [reference geometry](pt03ch20s05s01.md), and dimensions cannot be offset. **Tip:**To select a group of edges that are connected end-to-end, choose **by chain** from the prompt area (for more information, see ["Using the chain method to select edges in the Sketcher," Section 20.4.6](pt03ch20s04s06.md)). If necessary, you can then use the other selection methods in Abaqus/CAE to enhance your selection. For more information, see [Chapter 6, "Selecting objects within the viewport](pt01ch06.md)."
3. Click mouse button 2 to indicate that you have finished selecting objects.
4. Specify an offset distance or accept the default value displayed in the prompt area. Abaqus/CAE displays a preview of the offset edges in the viewport.
5. Use the buttons that appear in the prompt area to accept or change the offset direction.
6. To offset more edges, repeat the above steps beginning with Step 2.
7. When you have finished offsetting edges, do one of the following: - Click mouse button 2 anywhere in the Abaqus/CAE window. - Select any tool in the Sketcher toolbox. - Click the cancel button ![](../graphics/afxI_cancel.png) in the prompt area.
