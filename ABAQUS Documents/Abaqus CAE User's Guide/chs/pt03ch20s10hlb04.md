# 20.10.4 草绘圆







使用 Sketcher 工具箱中的圆工具基于圆心和对圆周上任意一点的中心点和任意点绘制圆，如下所示：

![](../graphics/ske-circlebtn-nls.png)

在草绘时，您应该注意定位点，因为这会影响网格的质量。草图中的点会成为您正在创建或修改的零件的顶点。反过来，当您在网格模块中对模型进行网格划分时，Abaqus/CAE 会将这些顶点转换为完全约束的种子，并将节点放置在其位置。有关如何随后移动顶点的信息，请参见["拖动 Sketcher 对象，"第 20.17.1 节](pt03ch20s14hlb01.md)。

**草绘圆：**

1. 从 Sketcher 工具箱中，选择圆工具![](../graphics/ico_sketchCircle.png)。有关 Sketcher 工具箱中工具的图表，请参见["Sketcher 工具，"第 20.4.1 节](pt03ch20s04s01.md)。Abaqus/CAE 在提示区域显示提示以指导您完成该过程。
2. 点击圆中心的所需位置。
3. 将光标移动到圆周上的一点。Abaqus/CAE 显示使用当前光标位置将创建的圆的预览。
4. 点击所需圆周上的任意点。**提示：**如有必要，您可以使用提示区域中的文本框输入圆心和圆周上一点的精确坐标。有关精确定义圆的更多信息，请参见["指定精确几何，"第 20.6 节](pt03ch20s06.md)。Abaqus/CAE 绘制圆。
5. 要创建更多圆，重复从第 2 步开始的上述步骤。
6. 完成创建圆后，执行以下操作之一：- 在 Abaqus/CAE 窗口中的任意位置点击鼠标按钮 2。- 在 Sketcher 工具箱中选择任何其他工具。- 点击提示区域中的取消按钮![](../graphics/afxI_cancel.png)。
