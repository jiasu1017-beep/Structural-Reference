# 2.2.13 退火温度







**产品：**Abaqus/Standard  Abaqus/Explicit  

### 测试单元

C3D8    C3D8R    CPS4    T3D2    

### 问题描述

**材料：**

**弹性**

杨氏模量，*E* = 200.0E3

泊松比，![](../graphics/ver_eqn00181.gif) = 0.3

**塑性**

硬化：

| 屈服应力 | 塑性应变 | 温度 |
| --- | --- | --- |
| 200. | 0.0000 | 0.0000 |
| 220. | 0.0009 | 0.0000 |
| 220. | 0.0029 | 0.0000 |
| 2. | 0.0000 | 100.00 |

**退火温度**

100.00

（单位不重要。）

### 结果与讨论

在所有测试中，单个单元在塑性范围内加载然后卸载。通过将温度升高到退火温度来退火产生的等效塑性应变（以及运动模型的反向应力张量）。随后，温度降低到退火温度以下，单元再次加载到塑性范围。
