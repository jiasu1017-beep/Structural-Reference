# 5.6.4 放大或缩小视图





使用**视图操作**工具栏中的放大工具 ![](../graphics/ico_viewZoom.png) 更改视口中视图的缩放比例。在使用放大工具时按住 **[Shift]** 可以访问替代放大模式。替代模式将相机移近或远离视图中的对象，而不是更改放大倍数。替代模式在你使用电影相机模式时最有用（有关更多信息，请参阅["放大工具，" 第 5.2.4 节](pt01ch05s02s04.md)）。

你的视图操作选项配置设置会影响放大工具的行为。如果你使用默认的 Abaqus/CAE 视图操作配置，放大工具通过水平拖动光标来操作：向右拖动放大，向左拖动缩小。在所有其他视图操作配置中，放大工具通过垂直拖动光标来操作：向上拖动放大，向下拖动缩小。请参阅["使用视图操作快捷方式，" 第 68.2 节](pt06ch68s02.md)获取更多信息。

如果你的鼠标有滚轮作为鼠标按钮 2，当光标在视口中时，你也可以通过向上或向下滚动来更改视图的缩放比例。对于 *X–Y* 绘图，此滚动的效果取决于光标的位置：
- 当光标在 *X–Y* 绘图网格内时，*X–Y* 绘图会在你放大或缩小绘图中 *X–Y* 曲线视图时保持其纵横比。
- 当光标放置在其中一个轴上时，Abaqus/CAE 会沿你选择的轴拉伸或压缩 *X–Y* 绘图。此功能使你能够调整沿一个轴比另一个轴变化更多的 *X–Y* 绘图的视图。

如果当前视口链接到其他视口，Abaqus/CAE 也会更改会话中所有链接视口中的视图缩放比例。有关更多信息，请参阅["链接视口以进行视图操作，" 第 4.5 节](pt01ch04s05.md)。

**要放大或缩小视图：**

1. 从**视图操作**工具栏中，单击放大工具 ![](../graphics/ico_viewZoom.png) 进入放大模式。**提示：**你也可以从主菜单中选择****View**![](../graphics/images/arrow.gif)**Magnify****，或按 **[F4]**。
2. 将光标定位在你要更改视图的视口中。光标变为放大镜：![](../graphics/crs-mag.png)
3. 拖动光标以更改视图。拖动方向取决于你的视图操作配置设置。你可以通过从主菜单栏中选择****Tools**![](../graphics/images/arrow.gif)**Options****并从**视图操作选项**选项卡页面中选择设置来更改这些设置。
   - 如果你使用默认视图操作配置，将光标拖动到起始点右侧以放大视图（放大）。将光标拖动到起始点左侧以缩小视图（缩小）。
   - 如果你使用非默认视图操作配置，将光标拖动到起始点上方以放大视图（放大）。将光标拖动到起始点下方以缩小视图（缩小）。
   当你在屏幕上拖动光标时，Abaqus/CAE 会从起始点绘制橡皮筋线条。橡皮筋线条表示已应用的缩放量，与你在有效方向上拖动动作的分量成正比。
4. 重复步骤 2 和 3，直到达到所需的视图。
5. 要退出放大模式，请执行以下操作之一：
   - 单击鼠标按钮 2。
   - 单击提示区域中的取消按钮 ![](../graphics/afxI_cancel.png)。
   - 单击放大工具。
   - 单击任何其他视图操作工具。
   **提示：**使用[循环](pt01ch05s06hlb07.md)视图操作工具返回上一个视图。
