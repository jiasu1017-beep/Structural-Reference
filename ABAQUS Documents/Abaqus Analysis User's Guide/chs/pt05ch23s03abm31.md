### 概述

修正德鲁克-普拉格/盖帽模型：
- 可与线弹性模型（["Linear elastic behavior," Section 22.2.1](pt05ch22s02abm02.md)）和各向同性硬化塑性模型（见["Classical metal plasticity," Section 22.2.1](pt05ch22s02abm02.md)）结合使用；
- 包含两个屈服面：一个用于剪切失败（扩展德鲁克-普拉格），另一个用于压缩盖帽；
- 考虑静水压力对屈服的影响；
- 支持率相关（粘性）响应；以及
- 适用于土壤、岩石和混凝土等材料。

### 流动规则

修正德鲁克-普拉格/盖帽模型使用非关联流动规则。剪切部分使用德鲁克-普拉格势，盖帽部分使用圆盖帽势。

### 硬化定律

修正德鲁克-普拉格/盖帽模型支持硬化定律，其中盖帽位置随塑性压缩应变而变化。

| **输入文件用法：** | ``` [*DRUCKER-PRAGER/CAP](../key/key-link.md#usb-kws-mdruckerpragercap) [*HARDENING](../key/key-link.md#usb-kws-mhardening) ``` |
| --- | --- |

### 率相关响应

修正德鲁克-普拉格/盖帽模型支持率相关响应。

| **输入文件用法：** | ``` [*DRUCKER-PRAGER/CAP](../key/key-link.md#usb-kws-mdruckerpragercap), RATE=参数 ``` |
| --- | --- |

### 与其他材料模型的结合

修正德鲁克-普拉格/盖帽模型可与线弹性模型和各向同性硬化塑性模型结合使用。

### 元素

修正德鲁克-普拉格/盖帽模型可与 Abaqus/Standard 和 Abaqus/Explicit 中的连续体元素一起使用。
