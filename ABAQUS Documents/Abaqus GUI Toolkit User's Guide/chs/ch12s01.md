# 12.1 Abaqus 接触功能概述





Abaqus/Standard 中的接触模拟可以是基于表面或基于接触单元的。Abaqus/Explicit 中的接触模拟仅基于表面。在本指南中，讨论基于表面的接触。

基于表面的接触可以利用通用（"自动"）接触算法或接触对算法。通用接触算法允许高度自动化的接触定义，其中接触基于自动生成的全包容表面定义。相反，接触对算法要求您明确配对可能接触的表面。两种算法都需要指定表面之间的接触属性（例如，摩擦）。

本指南中关于接触的讨论涉及 Abaqus/Standard 中的接触对方法和通用接触，以及 Abaqus/Explicit 中的通用接触。
