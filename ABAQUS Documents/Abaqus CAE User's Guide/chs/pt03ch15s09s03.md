# 15.9.3 交互属性编辑器





要创建交互属性，从主菜单栏选择****Interaction**![](../graphics/images/arrow.gif)**Property**![](../graphics/images/arrow.gif)**Create****。出现**创建交互属性**对话框，你可以在其中指定交互属性名称和要创建的交互属性类型。指定此信息后，点击**创建交互属性**对话框中的**继续**以显示交互属性编辑器。

交互属性编辑器的格式取决于你定义的交互属性类型。例如，薄膜条件和执行器/传感器属性编辑器显示数据字段，你可以在其中输入定义属性所需的全部信息。[图15-16](pt03ch15s09s03.md#itn-propeditor)显示了薄膜条件属性编辑器。

**图15-16** 薄膜条件属性编辑器。

![](../graphics/itn-propeditor-nls.png)

另一方面，接触属性编辑器的格式与属性模块中的材料编辑器完全相同（更多信息请参见["创建材料，"第12.4.1节](pt03ch12s04s01.md)）。与材料编辑器一样，接触属性编辑器包含菜单，你可以从中选择选项以包含在属性定义中，如[图15-17](pt03ch15s09s03.md#prp-mechpropeditor)所示。

**图15-17** 接触属性编辑器包含**Mechanical**和**Thermal**选项菜单。

![](../graphics/itn-mechpropeditor-nls.png)

当你从菜单中选择选项时，选项名称出现在编辑器顶部的**接触属性选项**列表中，选项成为交互属性定义的一部分。此外，编辑器下半部分的选项定义区域会更改以提供字段，你可以在其中为当前选定的选项指定信息。

例如，[图15-18](pt03ch15s09s03.md#prp-propoptions)中的**接触属性选项**列表反映了**切向行为**和**法向行为**选项（位于**Mechanical**菜单中）已包含在属性定义中。**切向行为**当前被选中，相关参数出现在编辑器下半部分。如果你想从接触属性定义中移除选项，可以从**接触属性选项**列表中选择该选项，然后点击![](../graphics/ico_delete.png)。

**图15-18** 包含**切向行为**和**法向行为**选项的机械接触属性定义。

![](../graphics/prp-propoptions-nls.png)

你可以通过从主菜单栏选择****Help**![](../graphics/images/arrow.gif)**On Context****然后点击感兴趣的功能来显示有关编辑器特定功能的帮助。有关创建属性的详细说明，请参见["创建交互属性，"第15.12.2节](pt03ch15s12hlb02.md)和["使用交互属性编辑器，"第15.14节](pt03ch15s14.md)。
