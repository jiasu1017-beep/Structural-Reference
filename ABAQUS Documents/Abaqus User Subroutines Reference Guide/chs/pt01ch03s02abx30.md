# 3.2.30 将ANSYS输入文件翻译为部分Abaqus输入文件





**产品：** Abaqus/Standard  Abaqus/Explicit  

##### **参考**

- ["Abaqus执行程序：概述，" 第3.1.1节](pt01ch03s01abo02.md)

### 概述

从ANSYS到Abaqus的翻译器将ANSYS阻止编码数据库文件中的某些实体转换为其在Abaqus输入文件中的对应物。

### 使用翻译器

**abaqus fromansys**翻译器可以将ANSYS阻止编码数据库文件(`.cdb`)转换为"平面"Abaqus输入文件；即，不是用部件和装配术语编写的Abaqus输入文件。`.cdb`文件必须使用以下命令在ANSYS中创建：

`CDWRITE , , <jobname>, cdb`

`CDWRITE`命令的第二个字段可能包含`ALL`或`DB`。第八个字段可能包含`BLOCKED`。`CDWRITE`命令的任何其他使用都会为翻译器带来问题。

### 命令行选项

**job**

此选项用于指定要由翻译器输出的Abaqus输入文件的名称。它也是包含ANSYS数据的输入文件的默认名称。翻译器生成的诊断信息将被写入名为`*job-name*.log`的文件。

**input**

此选项用于指定包含ANSYS数据的文件的名称（如果与*job-name*不同）。
