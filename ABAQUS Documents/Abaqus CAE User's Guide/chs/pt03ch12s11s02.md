# 12.10.6 定义比热容







您可以将材料每单位质量的比热容定义为温度和场变量的函数。在可能的情况下，您应该使用潜热定义而不是比热容定义来模拟相变期间内部能量的重大变化。请参见以下章节获取更多信息：
- "比热容"，Abaqus分析用户指南第26.2.3节
- "指定潜热数据"，第12.10.5节

**指定比热容：**

1. 从**编辑材料**对话框的菜单栏中，选择**热**![](../graphics/images/arrow.gif)**比热容**。（有关显示**编辑材料**对话框的信息，请参见"创建或编辑材料"，第12.7.1节。）
2. 选择比热容**类型**：- 对Abaqus/Standard或Abaqus/Explicit中的结构模型使用**恒定体积**。- 对Abaqus/CFD中的流体模型（不可压缩流动）使用**恒定压力**。
3. 切换**使用温度相关数据**将比热容定义为温度的函数。**数据**表中会出现一列标为**温度**。
4. 单击**场变量数量**字段右侧的箭头增加或减少比热容所依赖的场变量数量。
5. 在**数据**表中输入以下数据：**比热容**每单位质量的比热容。（单位为[JM1![](../graphics/usi_eqn00065.gif)1](../popups/usb-int-iconventions-unitsym.md)。）**温度**温度**场变量* n ***预定义的场变量。有关如何输入数据的详细信息，请参见"输入表格数据"，第3.2.7节。
6. 单击**确定**关闭**编辑材料**对话框。或者，您可以从**编辑材料**对话框的菜单中选择其他材料行为进行定义（有关更多信息，请参见"浏览和修改材料行为"，第12.7.2节）。
