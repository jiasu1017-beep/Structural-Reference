# 20.18.1 移除间隙和重叠


使用**移除间隙和重叠**工具修复包含间隙或重叠的导入草图。当您从 CAD 系统导入草图时,可能会出现间隙或重叠,因为 CAD 系统使用的草图几何公差值与 Abaqus/CAE 不同,所以会产生这些问题。[图 20–27](pt03ch20s15hlb01.md#usi-ske-repair-example1) 中的草图在左上角有一个间隙,在顶线的中点有一个小的重叠。

**图 20–27** 包含间隙和重叠的简单导入草图。

![](../graphics/usi-ske-repair-example1.png)

您可以通过选择边缘并指定公差值来修复此例中的间隙和重叠。当间隙的大小或重叠的长度小于指定的公差值时,Abaqus/CAE 会合并顶点以闭合间隙。

**注意:**在移除间隙或重叠期间,Abaqus/CAE 可以通过将一个顶点移动到与另一个顶点重合,或通过将两个顶点移动到它们之间的位置来合并顶点。如果您想控制对草图的更改,可以在执行间隙/重叠移除之前约束您希望保留其当前位置的线条或点。

实际上,您应该使用迭代过程从草图中移除间隙或重叠。首先选择要移除的间隙或重叠相邻的边缘,并在第一次尝试时使用 0.001 的默认长度公差值。如果所有间隙或重叠都没有闭合,请重新选择边缘并增加公差。重复此过程,直到 Abaqus/CAE 合并所选顶点以闭合间隙或重叠。

**移除间隙和重叠的步骤:**

1. 从 Sketcher 工具箱的修改工具中,选择**移除间隙和重叠**工具 ![](../graphics/ico_sketchRemoveGaps.png)。有关 Sketcher 工具箱中工具的图示,请参见["Sketcher 工具," 第 20.4.1 节](pt03ch20s04s01.md)。Abaqus/CAE 会在提示区域显示提示,引导您完成操作。
2. 选择要移除其间间隙和重叠的边缘。您可以 **[Shift]****+单击**或 **[Ctrl]****+单击**单独指定边缘,或者您可以拖动选择来指定草图中某个区域内的所有边缘。Abaqus/CAE 会将所选边缘着色为红色。
3. 在 Abaqus/CAE 窗口任意位置单击鼠标按钮 2。Abaqus/CAE 会提示您选择公差值。
4. 对于第一次尝试,接受默认公差值并在 Abaqus/CAE 窗口任意位置单击鼠标按钮 2。Abaqus/CAE 会移除小于默认公差的间隙和重叠。
5. 如果您仍然有剩余的间隙和重叠要移除,请执行以下步骤: 1. 重新选择要移除其间间隙和重叠的边缘。 2. 在提示区域增加公差值。 3. 在 Abaqus/CAE 窗口任意位置单击鼠标按钮 2。Abaqus/CAE 会移除小于默认公差的所选间隙或重叠。 4. 继续选择边缘并增加公差,直到 Abaqus/CAE 移除了您想要从模型中移除的所有间隙和重叠。
6. 完成移除边缘和重叠后,执行以下操作之一: - 在 Abaqus/CAE 窗口任意位置单击鼠标按钮 2。 - 选择 Sketcher 工具箱中的任何其他工具。 - 单击提示区域中的取消按钮 ![](../graphics/afxI_cancel.png)。
