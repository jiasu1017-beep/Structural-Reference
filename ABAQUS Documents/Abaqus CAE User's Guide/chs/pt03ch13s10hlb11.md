# 13.10.11 转换约束





要从所选零件或模型实例中移除所有面、边缘、同轴和接触约束，同时将实例留在当前位置，请从主菜单栏选择****Instance**![](../graphics/images/arrow.gif)**Convert Constraints****。转换等同于向实例应用单个平移和旋转，使其从原始位置移动到当前位置。先前的任何约束不再出现在特征列表中，不能恢复。

**要转换零件或模型实例的约束：**

1. 从主菜单栏，选择****Instance**![](../graphics/images/arrow.gif)**Convert Constraints****将任何现有约束转换到当前位置。
2. 从装配中选择要转换其约束的零件或模型实例。你也可以点击提示区域右侧的**Instance List**按钮并从出现的**Instance List**对话框中选择实例。
   **提示：**如果你无法选择所需的实例，可以使用**Selection**工具栏更改选择行为。更多信息，请参见["使用选择选项，"第6.3节](pt01ch06s03.md)。实例不移动，但Abaqus/CAE将任何现有约束转换到当前位置。你不能恢复原始的面、边缘、同轴和接触约束。
