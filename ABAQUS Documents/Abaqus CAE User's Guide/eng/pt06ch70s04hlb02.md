# 70.4.2 An overview of the methods for partitioning faces







When you choose **Face** from the **Create Partition** dialog box, the **Method** list displays the following methods for partitioning faces:

**![](../graphics/ico_ptnCellSketch.png) Sketch planar partition**

Partition a selected face by sketching a partition with the Sketcher, as shown in [Figure 70--8](pt06ch70s04hlb02.md#par-face-sketch). For detailed instructions, see ["Using the sketch method to partition faces," Section 70.6.1](pt06ch70s06hlb01.md).

You can sketch directly on the face to be partitioned, or you can sketch on a second face or datum plane and then project the sketch onto the face that you want to partition. For an example of projecting a sketch from a datum plane, see ["Using the Datum toolset in the Part module," Section 11.16.1](pt03ch11s16s01.md).

**Figure 70–8** Partitioning a face using the Sketcher.

![](../graphics/par-face-sketch-nls.png)

**![](../graphics/ico_ptnFaceTwoPoints.png) Shortest path between 2 points**

Partition the face along the shortest path connecting two selected points; the resulting partition will be curved if the face being partitioned is curved, as shown in [Figure 70--9](pt06ch70s04hlb02.md#par-face-shortest). You can select points that are not associated with the face being partitioned; for example, the points can be located on a different face or even a different part instance. For detailed instructions, see ["Using the shortest path method to partition faces," Section 70.6.2](pt06ch70s06hlb02.md).

**Figure 70–9** Partitioning a face using the shortest path between two points.

![](../graphics/par-face-shortest-nls.png)

**![](../graphics/ico_ptnFaceDatumPlane.png) Use datum plane**

Partition a face using the intersection with the extension of a datum plane, as shown in [Figure 70--10](pt06ch70s04hlb02.md#par-face-datum). For detailed instructions, see ["Using the datum plane method to partition faces," Section 70.6.3](pt06ch70s06hlb03.md).

**Figure 70–10** Partitioning a face using a datum plane.

![](../graphics/par-face-datum-nls.png)

**![](../graphics/ico_ptnFaceCurvedPath.png) Curved path normal to 2 edges**

Partition the face along a Bzier curve that is normal to two of the face's edges, as shown in [Figure 70--11](pt06ch70s04hlb02.md#par-face-curve). Position the curve by selecting two points anywhere along the two edges. The arc subtended by the two edges must be less than 180. For detailed instructions, see ["Using the curved path method to partition a face," Section 70.6.4](pt06ch70s06hlb04.md).

**Figure 70–11** Partitioning a face using a Bzier curve.

![](../graphics/par-face-curve-nls.png)

**![](../graphics/ico_ptnFaceExtendFace.png) Extend another face**

Partition the face using the intersection with the extension of another face, as shown in [Figure 70--12](pt06ch70s04hlb02.md#par-face-extend). The face being extended can be either planar, cylindrical, conical, or spherical; and it need not belong to the part containing the face to be partitioned. For detailed instructions, see ["Using the extended face method to partition faces," Section 70.6.5](pt06ch70s06hlb05.md).

**Figure 70–12** Partitioning a face using the extension of another face.

![](../graphics/par-face-extend-nls.png)

**![](../graphics/ico_ptnFaceIntersectFace.png) Intersect by other faces**

Partition the face using the intersection of the target face with one or more other faces, as shown in [Figure 70--13](pt06ch70s04hlb02.md#par-face-intersect). The faces can be intersecting or tangential. For detailed instructions, see ["Using the intersection method to partition faces," Section 70.6.6](pt06ch70s06hlb06.md).

**Figure 70–13** Partitioning a face using an intersection of faces.

![](../graphics/par-face-intersect-nls.png)

**![](../graphics/icoR_ptnFaceProjectEdges.png) Project edges**

Partition the face by projecting edges in the model, as shown in [Figure 70--14](pt06ch70s04hlb02.md#par-face-projedge). The partition is created using a perpendicular projection from the face being partitioned to the partitioning edge. You can choose to use only the projection or, if necessary, to extend the ends of the projected edge to complete the face partition. For detailed instructions, see ["Using the project edges method to partition faces," Section 70.6.7](pt06ch70s06hlb07.md).

**Figure 70–14** Partitioning a face by projecting an edge.

![](../graphics/prt-face-projedge-nls.png)

**![](../graphics/ico_ptnFaceAuto.png) Auto-partition**

When you mesh a face with quadrilateral elements using the free meshing technique, the Mesh module internally partitions the face into regions with three to five logical sides before meshing the face. For more information, see ["Free meshing with quadrilateral and quadrilateral-dominated elements," Section 17.10.2](pt03ch17s10s02.md). However, if you want to view and perhaps modify the automatically generated regions before generating the mesh, you can use the auto-partitioning tool to partition the face without meshing it. This tool is available only in the Mesh module. For detailed instructions, see ["Using the automatic generation method to partition faces," Section 70.6.8](pt06ch70s06hlb08.md).
