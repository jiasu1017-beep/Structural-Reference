# 80.2.8 自定义活动视图切割上合力和力矩的显示和计算





您可以使用可视化模块中视图切割的自由体选项来自定义活动视图切割上显示的合力和力矩的计算方式，以及自定义求和点和坐标系转换。Abaqus/CAE 可以通过切割当前显示组或切割整个模型来计算合力和力矩。通过切割当前显示组计算这些值可以提供更大的灵活性——例如，您可以通过将四个特定位置都包含在显示组中来计算模型中四个特定位置的总合力和力矩——但 Abaqus/CAE 仅在视图切割和显示组相交时显示合力和力矩数据。

您还可以自定义当前视口中所有自由体切割的内容和外观；请参见["自定义自由体切割的常规显示选项，" 第67.5.1节](pt06ch67s05s01.md)。

合力和力矩数据的显示仅适用于输出数据库的视图切割。当模型数据库中的模型显示在可视化模块中时，您无法为此类视图切割显示此数据。

默认情况下，Abaqus/CAE 仅为每个切换显示合力和力矩的视图切割显示一次合力和力矩。如果需要，您还可以显示一系列向量，这些向量在模型中以规则间隔显示合力和力矩数据，或者通过模型的一部分显示。

**自定义活动视图切割上合力和力矩的显示和计算的步骤：**

1. 在可视化模块中，从主菜单栏中选择****Options**![](../graphics/images/arrow.gif)**View Cut****。出现 **View Cut Options** 对话框。
   **提示：**您也可以使用视图切割管理器自定义切割选项。从主菜单栏中选择****Tools**![](../graphics/images/arrow.gif)**View Cut**![](../graphics/images/arrow.gif)**Manager****，然后在视图切割管理器中单击 **Options**。
2. 单击 **Free Body** 选项卡。
3. 从 **Computation based on** 选项中：
   - 选择 **Cutting through the current display group** 以基于当前显示组的视图切割计算合力和力矩。
   - 选择 **Cutting through the whole model** 以基于整个模型的视图切割计算合力和力矩。
4. 如果需要，自定义合力和力矩向量的求和点或坐标系转换选项。
   1. 从 **Summation point** 选项中，选择合力和力矩向量起源的三维位置。
      - 选择 **Centroid of cut** 将求和点自动放置在视图切割表面的质心处。
      - 选择 **User-defined**，并指定空间中自定义的三维位置。
   2. 从 **Component resolution** 选项中，您可以指定向量以分量形式显示时发生的坐标系转换。（有关以分量形式显示力和力矩向量的更多信息，请参见["自定义自由体切割的常规显示选项，" 第67.5.1节](pt06ch67s05s01.md)。）
      - 选择 **Normal and tangential** 以使用您选择的表面的法线和切线方向分量向量。如果需要，请指定分量向量的切线Y轴值。当您在视图切割切片上显示一系列自由体切割时，此选项使您能够强制切线向量使用统一方向。
      - 选择 **CSYS** 和坐标系以将分量向量转换为自定义坐标系。**Component resolution** 选项仅在 **Free Body Plot Options** 对话框中选择分量向量显示时影响合力和力矩的显示。
5. 切换 **Show heat flow rate if available** 以显示跨视图切割的能量/时间形式的热流率。
6. 如果您想自定义视口中自由体切割的内容和外观——包括活动视图切割上显示的自由体切割和使用自由体工具集创建的自由体切割——请单击 ![](../graphics/ico_selectionOptions.png) 访问 **Free Body Plot Options**。请参见["自定义自由体切割的常规显示选项，" 第67.5.1节](pt06ch67s05s01.md)。
7. 单击 **OK**。默认情况下，视图切割仅在会话期间保留。如果您想保留已定义的视图切割以供后续会话使用，可以将其保存到XML文件、模型数据库或输出数据库。有关更多信息，请参见["管理会话对象和会话选项，" 第9.9节](pt02ch09s08.md)。
8. 单击 **Apply** 以实现您的更改。视图切割会更改以反映您的规范。默认情况下，您的更改会在会话期间保留，并会影响该视口中所有后续的视图切割。如果您想保留更改以供后续会话使用，请将它们保存到文件。有关更多信息，请参见["保存自定义设置以供后续会话使用，" 第55.1.1节](pt05ch55s01s01.md)。
