# 29.5.2 定向从模板模型读取的几何体





您可以使用**Orientations**标签页指定模板紧固件结构在每个连接点处的定向方式。模板模型被旋转进入主模型并定向，使得模板模型的全局坐标系与您指定的坐标系对齐。默认是根据主模型中第一个表面在每个连接点处的法线向量来定向模板模型副本。此来自第一表面法线的默认方向在每个连接点创建*Z*轴对齐。然后通过将全局*X*轴投影到曲面上来计算*X*轴。

**要在主模型中定向组装紧固件：**

1. 从**Edit Fasteners**对话框中，显示**Orientations**标签页。
2. 选择Abaqus/CAE将用于在每个连接点处定向紧固件模板的方向选项。
   - 选择**Compute orientation based on first surface normal**（默认）以定向紧固件几何体，使模板模型的全局坐标系与主模型中第一个分配表面的法线向量对齐。如有需要，您可以切换**Flip normal direction**来反转方向。
   - 选择**Specify uniform CSYS**以使用您选择的坐标系来定向紧固件几何体。单击![](../graphics/ico_selectBlue.png)（**Edit**）以从主模型中的现有基准坐标系中进行选择。默认是使用主模型的全局坐标系。单击![](../graphics/ico_dtmCsys.png)（**Create Datum CSYS**）以创建新坐标系。
3. 如有需要，切换**Display orientation direction at attachment points**以在每个组装紧固件的连接点处显示方向三轴。这些可视化选项在您选择控制点集并完成第一个表面分配之前不可用。选择以下任一子选项：
   - 切换**Display only the normal direction**以从三轴中移除1轴和2轴，仅保留表面法线向量。
   - 切换**Include template model surfaces every *x* points**以显示模板模型约束表面的草图轮廓。调整数字***x***以在每个连接点、每隔一个点、每隔两个点等显示表面。表面将根据此标签页上显示的颜色和**Constraint Surface Substitutions**表上的颜色进行颜色编码。此选项有助于正确定位和定向模板模型。但是，如果您的模板模型表面太大（相对于主模型尺寸），结果可能会太拥挤。您可能希望返回并调整模板模型中的表面大小（如果出现这种情况）。
