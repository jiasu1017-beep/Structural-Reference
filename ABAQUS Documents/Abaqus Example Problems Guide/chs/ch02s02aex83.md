# 2.2.4 制动啸叫分析





**产品：** Abaqus/Standard  

本示例说明了复特征值提取程序的使用（"Abaqus Analysis User's Guide"第6.3.6节"Complex eigenvalue extraction"）（../usb/usb-link.md#usb-anl-acomplexfreqextract）。盘式制动器通过将一组制动片压靠在旋转的盘上来工作。制动片和盘之间的摩擦导致减速，但也可能引起系统的动态不稳定性，称为制动啸叫。制动啸叫现象的一种可能解释是两个相邻模式的耦合。两个在频率范围内彼此接近且具有相似特征的模态可能随着摩擦贡献的增加而合并。当这些模态在同一频率（变得耦合）时，其中一个变得不稳定。可以在复特征值提取期间识别不稳定模式，因为对应于不稳定模式的特征值的实部为正。可以通过改变制动部件的几何形状或材料属性来使制动系统设计稳定化，以解耦这些模态。

本分析的目的是识别特定盘式制动器系统中存在的（如果有的话）不稳定模式。

### 问题描述和模型定义

本示例中使用的制动器模型是用于国内客车的盘式制动系统的简化版本。简化模型由一个转子和位于转子两侧的两个制动片组成。制动片由有机摩擦材料制成，建模为各向异性弹性材料。转子直径为288 mm，厚度为20 mm，由铸铁制成。背板和绝缘体位于制动片后面，由钢制成。在本问题中，材料阻尼被忽略。mesh（如图2.2.4-1所示）（ch02s02aex83.md#brakesqueal-mesh）使用C3D6和C3D8I单元生成。转子两侧与制动片之间的接触使用小滑动接触公式定义。初始摩擦系数设置为零。

在第一步中，通过向绝缘体外表面施加压力来建立转子与制动片之间的接触。在下一步中，使用预定义的旋转运动对转子施加旋转速度![](../graphics/exa_eqn00422.gif)=5 rad/s。施加的速度对应于低速制动。通过更改摩擦特性，摩擦系数![](../graphics/exa_eqn00274.gif)也增加到0.3。通常，摩擦系数可以取决于滑移率、接触压力和温度。如果摩擦系数取决于滑移率，则使用运动预定义场施加的速度来确定![](../graphics/exa_eqn00274.gif)的相应值。摩擦系数从零斜坡上升到所需值，以避免可能由于接触体相对运动时通常发生的摩擦系数变化引起的不连续性和收敛问题。此问题在"Abaqus Analysis User's Guide"第6.2.2节"Static stress analysis"（../usb/usb-link.md#usb-anl-astatic）中有详细描述。在这一步中使用非对称求解器。在步骤结束时获得稳态制动条件。

在下一步中，在此稳态条件下执行特征值提取程序。因为复特征求解器使用子空间投影技术，所以首先提取实特征向量以定义投影子空间。在特征值提取程序中，在定义速度差的接触节点上，切向自由度不受约束。提取100个实特征模式，默认情况下，所有这些都用于定义投影子空间。可以通过选择要用于复特征值提取步骤的特征模式来减少子空间。复特征分析执行高达10 kHz（前55个模式）。

### 结果与讨论

图2.2.4-2（ch02s02aex83.md#brakesqueal-freqdamp1）显示了阻尼比与频率的关系。阻尼比定义为![](../graphics/exa_eqn00637.gif)，其中![](../graphics/exa_eqn00200.gif)是复特征值。阻尼比的负值表示不稳定模式。在感兴趣范围内，复特征求解器在2.0 kHz频率处发现了一个阻尼比为0.0138的不稳定模式。图2.2.4-3（ch02s02aex83.md#brakesqueal-mode1）以复特征向量实部和虚部组合幅值的形式展示了不稳定模式。

在第二个分析中，转子和制动片之间的摩擦系数增加到0.5。因为实特征向量在不同摩擦系数的问题之间没有显著差异，所以在第一次分析中确定的特征空间可以重用为第二次复特征值提取程序的投影子空间。分析在前一步的特征值提取步骤之后重新启动，并定义了两个新的过程步骤。在第一个非线性静态步骤中，通过更改摩擦特性来增加摩擦系数，然后在第二步中进行复特征值提取。结果以阻尼比图的形式呈现，如图2.2.4-4所示（ch02s02aex83.md#brakesqueal-freqdamp2）。在这种情况下，在2.0、3.0、8.0和9.0 kHz处发现四个不稳定模式。随着摩擦系数的增加，更多的相邻模式耦合并变得不稳定。
