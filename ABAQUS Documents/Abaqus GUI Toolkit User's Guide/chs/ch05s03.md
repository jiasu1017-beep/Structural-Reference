# 5.3 壳材料方向







壳单元，不像连续体单元，使用对每个单元局部的材料方向。各向异性材料数据（如纤维增强复合材料）和单元输出变量（如应力和应变）根据这些局部材料方向定义。在大位移分析中，壳表面上的局部材料轴随每个积分点处材料的平均运动旋转。

### 5.3.1 默认局部材料方向

局部材料 1 和 2 方向位于壳平面内。默认局部 1 方向是全局 1 轴到壳表面的投影。如果全局 1 轴垂直于壳表面，则局部 1 方向是全局 3 轴到壳的投影。局部 2 方向垂直于局部 1 方向在壳表面内，因此局部 1 方向、局部 2 方向和表面正法线形成右手系（见[图 5-6](ch05s03.md#gss-localshell)）。

**图 5-6** 默认局部壳材料方向。

![](../graphics/gss-localshell-nls.png)

默认局部材料方向集有时会导致问题；一个例子是如图 [图 5-7](ch05s03.md#gss-one-direct) 所示的圆柱体。 

**图 5-7** 圆柱体中的默认局部材料 1 方向。

![](../graphics/gss-one-direct.png)

对于图中的大多数单元，局部 1 方向是周向的。然而，有一行单元垂直于全局 1 轴。对于这些单元，局部 1 方向是全局 3 轴到壳的投影，使局部 1 方向成为轴向而不是周向。局部 1 方向直接应力的云图看起来非常奇怪，因为对于大多数单元 ![](../graphics/gsk_eqn00032.gif) 是周向应力，而对于某些单元它是轴向应力。在这种情况下，有必要为模型定义更合适的局部方向，如下一节所述。

### 5.3.2 创建替代材料方向

[*ORIENTATION](../key/key-link.md#usb-kws-morientation) 选项允许您直接控制局部材料方向。通过它，您可以用局部直角、圆柱或球坐标系替换全局笛卡尔坐标系。您通过指定两个点 *a* 和 *b* 的位置来定义局部（![](../graphics/gsk_eqn00042.gif)、![](../graphics/gsk_eqn00043.gif)、![](../graphics/gsk_eqn00044.gif)）坐标系统的方向，如[图 5-8](ch05s03.md#gss-coord-system) 所示。例如，局部直角系统用以下选项定义：

```
[*ORIENTATION](../key/key-link.md#usb-kws-morientation), SYSTEM=RECTANGULAR, NAME=LOCALR
<![](../graphics/gsk_eqn00045.gif)>, <![](../graphics/gsk_eqn00046.gif)>, <![](../graphics/gsk_eqn00047.gif)>, <![](../graphics/gsk_eqn00048.gif)>, <![](../graphics/gsk_eqn00049.gif)>, <![](../graphics/gsk_eqn00050.gif)>
```

**图 5-8** 局部坐标系统的定义。

![](../graphics/gss-coord-system-nls.png)

参数 NAME 指定此方向的标签，点 *a*（![](../graphics/gsk_eqn00045.gif)、![](../graphics/gsk_eqn00046.gif)、![](../graphics/gsk_eqn00047.gif)）和点 *b*（![](../graphics/gsk_eqn00048.gif)、![](../graphics/gsk_eqn00049.gif)、![](../graphics/gsk_eqn00050.gif)）的坐标以全局笛卡尔系统给出。然后通过 [*SHELL SECTION](../key/key-link.md#usb-kws-mshellsection) 或 [*SHELL GENERAL SECTION](../key/key-link.md#usb-kws-mshellgensect) 选项上的 ORIENTATION 参数引用局部坐标系统。

您仍必须指定另一条信息。还必须告诉 Abaqus 局部轴中的哪个对应哪个材料方向。在 [*ORIENTATION](../key/key-link.md#usb-kws-morientation) 后的第二数据行上，指定局部轴（1、2 或 3）最接近垂直于壳表面。Abaqus 遵循轴的循环排列（1、2、3），并将您选择后的轴投影到壳区域以形成材料 1 方向。例如，如果您选择 ![](../graphics/gsk_eqn00042.gif) 轴，Abaqus 将 ![](../graphics/gsk_eqn00043.gif) 轴投影到壳上以形成材料 1 方向。材料 2 方向由壳法线和材料 1 方向的叉积定义。通常，最终材料 2 方向和投影的其它局部轴（本例中的 ![](../graphics/gsk_eqn00044.gif) 轴）对于曲壳不会重合。

如果这些局部轴未创建所需的材料方向，您可以指定关于所选轴的旋转。在投影到壳表面以给出最终材料方向之前，其他两个局部轴旋转此量。以下选项块将创建如图 [图 5-9](ch05s03.md#gss-rotation) 所示的局部系统：

```
[*ORIENTATION](../key/key-link.md#usb-kws-morientation), SYSTEM=RECTANGULAR, NAME=LOCALR 
<![](../graphics/gsk_eqn00045.gif)>, <![](../graphics/gsk_eqn00046.gif)>, <![](../graphics/gsk_eqn00047.gif)>, <![](../graphics/gsk_eqn00048.gif)>, <![](../graphics/gsk_eqn00049.gif)>, <![](../graphics/gsk_eqn00050.gif)>
1, ![](../graphics/gsk_eqn00051.gif)
```
同样，是旋转后的 ![](../graphics/gsk_eqn00043.gif) 和 ![](../graphics/gsk_eqn00044.gif) 轴被 Abaqus 投影到壳单元表面。为了易于解释投影，所选轴应尽可能接近壳法线。

如果如图 [图 5-7](ch05s03.md#gss-one-direct) 所示的圆柱体中心线与全局 3 轴重合，则可以使用以下选项块定义一致的材料方向：

```
[*ORIENTATION](../key/key-link.md#usb-kws-morientation), SYSTEM=CYLINDRICAL, NAME=CYLIND1
0., 0., 0., 0., 0., 1.
1, 0.
```

点 *a* 和 *b* 位于圆柱体中心线上。由于圆柱体的方向与新定义的圆柱坐标系统的方向匹配，![](../graphics/gsk_eqn00042.gif) 轴是径向的，![](../graphics/gsk_eqn00043.gif) 轴是周向的，![](../graphics/gsk_eqn00044.gif) 轴是轴向的。![](../graphics/gsk_eqn00042.gif) 轴对应于壳法线方向的近似值，并指定了零旋转；因此，![](../graphics/gsk_eqn00043.gif) 轴到壳表面的投影是材料 1 方向。因此，材料 1 方向始终是周向的，相应的材料 2 方向始终是轴向的。
