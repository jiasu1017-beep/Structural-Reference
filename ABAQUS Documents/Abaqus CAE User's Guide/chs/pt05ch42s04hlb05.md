# 42.6.9 控制复数结果的形式

如果当前步骤是稳态动力学分析，则像应力(S)或位移(U)这样的输出变量的值可以是具有实部和虚部的复数。在Abaqus/CAE中显示模型图或记录模型探测值时，您可以控制复数显示的形式。您可以选择查看分析结果的幅值、相角、实部、虚部或指定角度的值。所选形式将应用于当前Abaqus/CAE会话中的所有复数结果，谐波动画除外。谐波动画根据定义始终使用"指定角度的值"形式，而不管当前数值形式如何。

**控制数值形式：**

1. 找到"Numeric Form"选项。从主菜单栏中，选择****Result**![](../graphics/images/arrow.gif)**Options****；然后在出现的对话框中单击**Complex Form**标签。将出现"Numeric Form"选项。
2. 选择要用于复数的所需形式：
   - **幅值**显示结果值的实部和虚部两者的组合幅值。
   - **相角**显示正水平轴与表示复数的绘图点（*实部，虚部*）之间的角度。此选择仅对标量或矢量和张量分量有效。
   - **实部**仅显示复数结果值的实部分量。此选项是默认设置。
   - **虚部**仅显示复数结果值的虚部分量的系数。
   - **指定角度的值**以您指定的角度显示结果在实部和虚部组合值。当角度为0时，显示结果的实部；当角度为90时，显示结果的虚部。
3. Abaqus根据您的规格更新当前视口中的显示。默认情况下，您的更改会在会话期间保存，并会影响后续的所有结果显示。如果您希望将更改保留到后续会话中，请将其保存到文件。有关的详细信息，请参阅["保存自定义设置以供后续会话使用，" 第55.1.1节](pt05ch55s01s01.md)。
