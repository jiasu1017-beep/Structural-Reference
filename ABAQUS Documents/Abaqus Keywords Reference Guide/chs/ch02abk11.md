# *BOND





### *BOND定义粘接和粘接属性。

此选项用于定义最初将两个接触边界绑定在一起的可断裂粘接。此选项必须与[*SURFACE INTERACTION](ch18abk50.md)选项配合使用。

**产品：**Abaqus/Explicit

**类型：**历史数据

**级别：**步骤

##### **参考：**

- ["可断裂粘接，" Abaqus Analysis User's Guide第37.1.9节](../usb/usb-link.md#usb-cni-aspotweld)
- [*SURFACE INTERACTION](ch18abk50.md)

**此选项没有参数。**
