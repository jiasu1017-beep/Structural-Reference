# 42.3 选择结果步骤和帧





您可以控制Abaqus/CAE从输出数据库获取模型数据和结果的步骤和帧。您可以使用**Step/Frame**对话框选择特定的步骤和帧，也可以使用上下文栏中的控件逐步切换帧。

**注意：**如果您在可视化模块中显示来自当前模型数据库的模型数据，可以使用**Step/Frame**对话框选择特定的步骤，但帧控件被禁用。模型数据库数据不包括单独的帧。
