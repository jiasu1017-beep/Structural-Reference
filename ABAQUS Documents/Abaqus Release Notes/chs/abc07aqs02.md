# 7.2 线性棱锥声学单元





**产品：** Abaqus/Standard

**优势：** 棱锥单元在网格生成过程中用于在砖块单元和四面体单元之间进行过渡。

**说明：** Abaqus/Standard 中提供了声学单元，用于在纯声学分析或耦合声学-结构分析中建模声学域。新的 5 节点棱锥单元 AC3D5 支持现有声学单元支持的所有加载、材料行为和边界条件。
