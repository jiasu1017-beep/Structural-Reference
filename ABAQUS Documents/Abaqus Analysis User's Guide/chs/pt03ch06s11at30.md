# 10.3.2 Abaqus/Aqua 分析

**产品：** Abaqus/Standard

##### **参考资料**

- ["Abaqus/Aqua 分析概述，" 第 2.10 节](../stm/stm-link.md#stm-anl-aqua)
- [*AQUA](../key/key-link.md#usb-kws-haqua)

### 概述

Abaqus/Aqua 分析：
- 用于模拟 offshore 结构承受稳态流、波浪、风荷载的作用；
- 可用于模拟海底管道、平台立管和系泊系统的静力和动力分析；
- 提供 Morison 方程作用在细长构件上的流体力；
- 支持线性波理论和高阶波理论；以及
- 可用于静力分析和动力分析。

### 概述

Abaqus/Aqua 分析功能专门用于模拟海洋结构物与水的相互作用。它能够模拟：
- 稳态流作用；
- 规则波和不规则波作用；
- 风荷载作用；
- 流体力对细长构件的作用（使用 Morison 方程）。

### 定义 Aqua 分析

Abaqus/Aqua 分析通过在步骤定义中激活 Aqua 功能来启用。

| **输入文件用法：** | 使用以下选项在步骤中激活 Abaqus/Aqua 功能： |
| --- | --- |
| | ``` [*AQUA](../key/key-link.md#usb-kws-haqua) ``` |

### 定义流荷载

流荷载（稳态流和波浪）可以作用在细长构件上。荷载基于 Morison 方程计算。

| **输入文件用法：** | 使用以下选项定义流荷载： |
| --- | --- |
| | ``` [*DMU](../key/key-link.md#usb-kws-hdmu) *节点或单元集* *直径* *拖曳系数* *惯性系数* *附加质量系数* ``` |

### 定义波浪理论

Abaqus/Aqua 支持多种波浪理论：

| **波浪理论** | **描述** |
| --- | --- |
| 线性波 | 适用于小振幅波 |
| Stokes 五阶波 | 适用于中等振幅波 |
| 流函数波 | 适用于大振幅波 |

| **输入文件用法：** | 使用以下选项指定波浪理论： |
| --- | --- |
| | ``` [*WAVE GEN](../key/key-link.md#usb-kws-hwavegen), THEORY=*theory_type* ``` |

### 定义风荷载

风荷载可以作用在暴露于空气中的结构构件上。

| **输入文件用法：** | 使用以下选项定义风荷载： |
| --- | --- |
| | ``` [*WIND LOAD](../key/key-link.md#usb-kws-hwindload) *构件集* *风速* *空气密度* *拖曳系数* ``` |

### 动力分析考虑

在动力分析中：
- 流体力被添加到质量矩阵中（附加质量效应）；
- 可以包含波浪的时间变化效应；
- 惯性力和阻尼力被正确计算。

### 输出一

Abaqus/Aqua 分析的输出包括：
- 结构位移、速度和加速度；
- 单元应力和应变；
- 流体力（阻力、惯性力、附加质量力）；
- 波浪表面高程。

### 局限性

- Abaqus/Aqua 不支持有限振幅波与结构的完全耦合分析；
- 流体力仅适用于细长构件（管状、圆形或近似圆形的截面）；
- 不支持空化效应。
