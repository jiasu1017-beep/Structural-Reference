# 15.17.14 为表格数据指定行为设置

您可以为正则化（Abaqus/Explicit分析专用）和用于定义弹性、阻尼、用户定义摩擦、塑性和损伤行为选项以及连接器导出分量项的表格数据的外推指定行为设置。您还可以为选定行为选项的Abaqus/Explicit分析中的率相关数据评估指定设置。有关更多信息，请参见["使用表格数据定义连接器行为"中的"连接器行为，"Abaqus分析用户指南第31.2.1节](../usb/usb-link.md#usb-elm-econnectorbehav-table)。

您可以修改连接器截面中所有行为选项的行为设置，也可以为单个行为选项指定行为设置。单个行为选项的行为设置优先于连接器截面行为设置。您可以在连接器截面编辑器的**表格选项**标签页上为连接器截面中的所有行为选项指定行为设置。有关更多信息，请参见["创建连接器截面，"第15.12.11节"](pt03ch15s12s01.md)。

要显示**编辑表格选项**对话框中选项的上下文相关帮助，您必须选择感兴趣的选项，然后按**[F1]**。（对话框显示时，主菜单栏中的**帮助**菜单不可用。）

**要为单个连接器行为指定表格数据的行为设置：**

1. 显示**编辑表格选项**对话框。
   - 对于使用表格数据的弹性、阻尼、用户定义摩擦、塑性或损伤行为，请在连接器截面编辑器的**行为选项**标签页上点击**表格选项**。有关更多信息，请参见以下章节：
     - ["定义弹性，"第15.17.1节](pt03ch15s17hlb01.md)
     - ["定义阻尼，"第15.17.2节](pt03ch15s17hlb02.md)
     - ["指定预定义的摩擦参数或接触力，"第15.17.5节](pt03ch15s17hlb05.md)
     - ["定义塑性，"第15.17.6节](pt03ch15s17hlb06.md)
     - ["定义损伤，"第15.17.7节](pt03ch15s17hlb07.md)
     - ["定义损伤演化，"第15.17.8节](pt03ch15s17hlb08.md)
   - 对于连接器导出分量项，请从**编辑导出分量项**对话框中点击**表格选项**。有关更多信息，请参见["指定连接器导出分量，"第15.17.15节"](pt03ch15s17hlb15.md)。
2. 在对话框的**正则化**部分，指定Abaqus/Explicit分析中数据正则化的设置。
   1. 指定用于此行为选项的正则化行为设置。
      - 打开**使用行为设置**以应用适用于连接器截面中所有行为选项的正则化行为设置。
      - 关闭**使用行为设置**以使用您在此对话框中指定的正则化行为设置。
   2. 默认情况下，Abaqus/Explicit将数据正则化为以自变量均匀间隔定义的表格。
      - 打开**正则化数据**以正则化表格数据。
      - 关闭**正则化数据**以关闭表格公认的正则化并直接使用您定义的数据。
   3. 如果您想正则化表格数据，请指定误差容限。
      - 选择**使用默认值**以使用默认值0.03。
      - 选择**指定**，然后输入误差容限值。
3. 如果您要定义包含等向硬化的塑性行为或损伤起始行为，则对话框的**率选项**部分可用。您可以为Abaqus/Explicit分析中的率相关数据评估指定设置。这些设置仅适用于包含使用**表格**定义的等向硬化的塑性行为（力-本构运动数据直接以表格形式指定）或具有基于塑性运动起始准则的损伤行为。
   1. 指定率过滤器因子的值，![](../graphics/usi_eqn00122.gif)。
      - 选择**使用默认值**以使用默认值0.9。
      - 选择**指定**，然后输入一个值（![](../graphics/usi_eqn00634.gif)）。值为![](../graphics/usi_eqn00264.gif)表示没有过滤，应谨慎使用。
   2. 指定插值方法。
      - 选择**线性**以在线性间隔的等效相对塑性运动率下对率相关损伤起始数据进行插值。
      - 选择**对数**以在对数间隔的等效相对塑性运动率下对率相关损伤起始数据进行插值。
4. 在对话框的**外推**部分，指定数据外推的设置。
   1. 指定用于此行为选项的外推行为设置。
      - 打开**使用行为设置**以应用适用于连接器截面中所有行为选项的外推行为设置。
      - 关闭**使用行为设置**以使用您在此对话框中指定的外推行为设置。
   2. 指定表格数据外推的方法。您输入的数据点构成本构空间中的一条非线性曲线。默认情况下，Abaqus/CAE在外推自变量指定范围之外的因变量为与曲线端点对应的常数值。
      - 选择**常数**以在外推自变量指定范围之外的因变量时使用常数外推。
      - 选择**线性**以在外推自变量指定范围之外的因变量时使用线性外推。
5. 点击**确定**保存您的行为设置并返回连接器截面编辑器。
