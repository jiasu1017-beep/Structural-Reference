# 42.7 创建新的场输出

输出数据库中最初可用的场和历史输出仅限于您在分析期间保存的那些变量。您可以通过对场输出进行运算来创建新结果，从而增强这些分析结果。可以使用以下两种方法创建新的场输出：
- 从主菜单栏中选择****Tools**![](../graphics/images/arrow.gif)**Create Field Output**![](../graphics/images/arrow.gif)**From Fields****，或使用工具箱中的![](../graphics/ico_fieldOutputFromFields.png)工具，通过[对单个现有场输出变量进行运算](pt05ch42s05hlb01.md)来创建新的场输出变量。
- 从主菜单栏中选择****Tools**![](../graphics/images/arrow.gif)**Create Field Output**![](../graphics/images/arrow.gif)**From Frames****，或使用工具箱中的![](../graphics/ico_fieldOutputFromFrames.png)工具，通过[组合多个输出数据库帧的结果](pt05ch42s05hlb03.md)来创建新的场输出。

- ["场输出运算概述，" 第42.7.2节](pt05ch42s05hlb02.md)

- ["通过对帧进行运算创建场输出，" 第42.7.5节](pt05ch42s05hlb05.md)
