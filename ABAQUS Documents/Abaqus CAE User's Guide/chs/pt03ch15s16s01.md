# 15.16.1 指定接触检测的搜索条件





接触检测工具使用一系列用户定义的参数来识别模型中潜在的接触对。从主菜单选择**交互**![](../graphics/images/arrow.gif)**查找接触对**或**约束**![](../graphics/images/arrow.gif)**查找接触对**，然后输入接触检测参数。您可以在以下区域指定参数：
- [指定接触检测的常规搜索选项](pt03ch15s16s01.md#usi-itn-detection-search)
- [指定接触检测的命名选项](pt03ch15s16s01.md#usi-itn-detection-naming)
- [指定接触检测的实体选项](pt03ch15s16s01.md#usi-itn-detection-entities)
- [定义默认接触对参数](pt03ch15s16s01.md#usi-itn-detection-rules)
- [指定接触检测的高级搜索选项](pt03ch15s16s01.md#usi-itn-detection-advanced)

### 指定接触检测的常规搜索选项

常规搜索选项指示您希望Abaqus/CAE在其中查找接触对候选者的范围。您可以指定模型中包含接触对的区域，以及估计的需要相互作用或约束定义的两个表面之间的距离。

**要指定常规搜索选项：**

1. 从主菜单，选择**交互**![](../graphics/images/arrow.gif)**查找接触对**或**约束**![](../graphics/images/arrow.gif)**查找接触对**。**提示：**您也可以使用交互模块工具箱中的![](../graphics/ico_contactPairFind.png)工具来查找接触对。Abaqus/CAE显示接触检测对话框。
2. 如果**搜索选项**标签页尚不可见，则显示它。
3. 指定**搜索域**：
   - 选择**整个模型**以包括当前模型中的所有实例。
   - 选择**实例**以仅包括指定的零件实例和子零件实例。使用以下过程指定实例：
     1. 点击![](../graphics/ico_selectBlue.png)。
     2. 从视口中选择实例（有关选择实例的信息，请参见[第6章，"在视口中选择对象"](pt01ch06.md)"）。
     3. 选择完所有所需的实例后，在提示区域点击**完成**。
   - 选择**显示的实体**以仅包括当前显示在视口中的模型部分。您可以使用装配显示选项和显示组来控制模型的显示。有关详细信息，请参见["控制实例可见性，"第76.14节](pt07ch76hla13.md)和[第78章，"使用显示组显示模型的子集"](pt07ch78.md)。
4. 在**包括分离公差内的接触对**字段中输入潜在接触对表面之间的最大距离。有关选择分离公差的提示，请参见["选择分离公差和扩展角度"中的"使用接触检测工具的提示，"第15.6.4节](pt03ch15s06s04.md#usi-itn-auto-detection-tolerance)。
5. 默认情况下，Abaqus/CAE将表面定义扩展到搜索检测到的表面20度角范围内的任何面（对于几何体）或小平面（对于网格化模型）。您可以通过更改**按角度扩展每个发现的表面**字段中的值来调整此角度。要阻止表面定义的扩展，请关闭**按角度扩展每个发现的表面**。有关Abaqus/CAE如何扩展表面的更多详细信息，请参见["定义接触对的额外标准"中的"接触检测算法，"第15.6.2节](pt03ch15s06s02.md#usi-itn-auto-detection-filter)。
6. 默认情况下，Abaqus/CAE从位于不同零件实例或模型实例上的表面创建接触对。要搜索涉及同一零件实例或模型实例上的不同区域的接触对，或搜索潜在的自接触相互作用，请打开**包括同一实例上表面的接触对**。有关详细信息，请参见["定义同一实例内和自接触"中的"接触检测算法，"第15.6.2节](pt03ch15s06s02.md#usi-itn-auto-detection-self)。

### 指定接触检测的命名选项

除了创建接触对交互外，自动接触检测工具还为参与接触对的每个表面创建命名表面（请参见["默认交互和约束参数，"第15.6.3节"](pt03ch15s06s03.md)）。**命名**标签页提供了一些用于调整此表面创建行为的选项。

**要指定命名选项：**

1. 从主菜单，选择**交互**![](../graphics/images/arrow.gif)**查找接触对**或**约束**![](../graphics/images/arrow.gif)**查找接触对**。**提示：**您也可以使用交互模块工具箱中的![](../graphics/ico_contactPairFind.png)工具来查找接触对。Abaqus/CAE显示接触检测对话框。
2. 显示**命名**标签页。
3. 在**名称前缀**字段中，输入用于命名每个接触对和表面的前缀（如["默认交互和约束参数，"第15.6.3节"](pt03ch15s06s03.md)中所讨论的）。
4. 要阻止命名表面的创建，请关闭**命名每个发现的表面**。此选项不会影响接触对的创建。
5. **创建包含以下内容的附加命名表面**字段允许您创建额外的复合表面：
   - 要创建由所有充当主表面的接触对表面组成的单个命名表面，请打开**所有主表面**。先前存在的接触对中的任何主表面也将包含在创建的复合表面中。
   - 要创建由所有充当从表面的接触对表面组成的单个命名表面，请打开**所有从表面**。先前存在的接触对中的任何从表面也将包含在创建的复合表面中。
   - 要创建由参与接触对定义的所有表面组成的单个命名表面，请打开**所有**。先前存在的接触对中的任何表面也将包含在创建的复合表面中。

### 指定接触检测的实体选项

**实体**标签页允许您控制包含在搜索中的几何体和元素的类型。您可以使用这些选项来忽略模型或零件实例的某些对接触相互作用或约束不重要的特征；例如，固体外表面的膜元素层。从搜索中移除特征也可以提高性能；例如，如果复杂模型中的所有接触面都是平面的，则从接触对搜索中移除圆柱形和样条特征可能会缩短搜索时间。通常，默认的实体选项提供可接受的性能，只应在搜索花费过长时间才能完成时进行修改。

**要指定实体选项：**

1. 从主菜单，选择**交互**![](../graphics/images/arrow.gif)**查找接触对**或**约束**![](../graphics/images/arrow.gif)**查找接触对**。**提示：**您也可以使用交互模块工具箱中的![](../graphics/ico_contactPairFind.png)工具来查找接触对。Abaqus/CAE显示接触检测对话框。
2. 显示**实体**标签页。
3. 默认情况下，Abaqus/CAE在计算壳实体之间的分离时考虑壳厚度和偏移属性。要忽略壳截面属性并仅基于模型中壳实体的几何表示计算分离，请关闭**搜索期间考虑壳厚度和偏移**。
4. 执行几何搜索时，切换**搜索以下几何实体**字段中的以下特征以将其包括在搜索域中或从中排除：
   - **平面**指模型中的所有平面。这些特征默认处于开启状态。
   - **圆柱/球面/环形**指由 extrusion 或 revolving 直线、圆或椭圆而生成的模型中的曲面。这些特征默认处于开启状态。
   - **基于样条**指由样条线或路径生成的模型中的不规则曲面（有关更多信息，请参见["绘制样条，"第20.10.10节"](pt03ch20s10hlb10.md)）。这些特征默认处于开启状态。
5. 执行网格化几何体搜索时，切换**搜索以下网格实体**字段中的以下元素类型以将其包括在搜索域中或从中排除：
   - **实体**指实体连续体元素上的面。这些元素默认处于开启状态。
   - **壳**指壳元素上的面。这些元素默认处于开启状态。
   - **膜**指膜或表面元素上的面。这些元素默认处于关闭状态。

### 定义默认接触对参数

接触检测规则允许您指定一组条件选项，这些选项确定分配给检测到的接触对的默认设置。您始终可以在检测到接触对后更改其设置，但在搜索前指定规则可以根据您特定的建模意图快速定义适当的接触对。当Abaqus/CAE执行搜索以查找接触对候选者时，将应用这些规则。有关接触对默认设置的更多信息，请参见["默认交互和约束参数，"第15.6.3节"](pt03ch15s06s03.md)。

**要为默认接触对设置指定规则：**

1. 从主菜单，选择**交互**![](../graphics/images/arrow.gif)**查找接触对**或**约束**![](../graphics/images/arrow.gif)**查找接触对**。**提示：**您也可以使用交互模块工具箱中的![](../graphics/ico_contactPairFind.png)工具来查找接触对。Abaqus/CAE显示接触检测对话框。
2. 显示**规则**标签页。
3. 点击![](../graphics/ico_edit.png)。Abaqus/CAE显示**编辑规则**对话框。
4. 要为表面位于特定公差内的所有接触对创建绑定约束，请打开**当分离值不超过时使用绑定约束**，并在提供的字段中输入公差。此规则默认处于禁用状态。
5. 要将绑定约束上的位置公差设置为等于接触检测工具报告的表面分离，请打开**当分离值非零时将绑定位置公差设置为分离值**。此规则默认处于禁用状态。有关绑定约束位置公差的更多信息，请参见["网格绑定约束，"Abaqus分析用户指南第35.3.1节](../usb/usb-link.md#usb-cni-ptiedconstraint)。
6. 要将接触相互作用上的表面调整设置设置为等于接触检测工具报告的表面分离，请打开**通过分离值调整分离的相互作用**。此规则默认处于禁用状态。有关表面调整选项的更多信息，请参见["调整初始表面位置并在Abaqus/Standard接触对中指定初始间隙，"Abaqus分析用户指南第36.3.5节](../usb/usb-link.md#usb-cni-aadjustsurfaces)。
7. 要将接触相互作用中任何过闭合的从节点直接移动到主表面上，请打开**调整相互作用以移除过闭合**。调整将在分析期间执行。此规则默认处于启用状态。有关表面调整选项的更多信息，请参见["调整初始表面位置并在Abaqus/Standard接触对中指定初始间隙，"Abaqus分析用户指南第36.3.5节](../usb/usb-link.md#usb-cni-aadjustsurfaces)。
8. 点击**确定**。

### 指定接触检测的高级搜索选项

高级搜索选项对搜索范围和接触对表面的创建提供了额外的控制级别。常规搜索选项在大多数情况下已足够；但是，您可能需要使用高级选项来考虑独特的建模条件。请参见["使用接触检测工具的提示，"第15.6.4节"](pt03ch15s06s04.md)，了解各种建模条件和建议的搜索技术的讨论。

**要指定高级搜索选项：**

1. 从主菜单，选择**交互**![](../graphics/images/arrow.gif)**查找接触对**或**约束**![](../graphics/images/arrow.gif)**查找接触对**。**提示：**您也可以使用交互模块工具箱中的![](../graphics/ico_contactPairFind.png)工具来查找接触对。Abaqus/CAE显示接触检测对话框。
2. 显示**高级**标签页。
3. 默认情况下，Abaqus/CAE合并其相邻表面彼此之间在20度角内的任何接触对（有关此计算的详细信息，请参见["定义接触对的额外标准"中的"接触检测算法，"第15.6.2节"](pt03ch15s06s02.md#usi-itn-auto-detection-filter)）。您可以通过更改**当表面在角度内时合并接触对**字段中的值来调整此角度。要阻止相邻接触对的合并，请关闭**当表面在角度内时合并接触对**。
4. 默认情况下，Abaqus/CAE在搜索接触对时包括过闭合和相交的表面。要忽略过闭合的表面，请关闭**包括过闭合的表面**。有关自动接触检测工具如何解释过闭合表面的详细信息，请参见["接触检测算法"中的"过闭合表面的检测，"第15.6.2节"](pt03ch15s06s02.md#usi-itn-auto-detection-overclosed)。
5. 默认情况下，如果两个表面横向偏移（即，来自任一表面的表面法线从不与相反表面相交），Abaqus/CAE不会创建接触对候选者。要将这些表面包括在接触对候选者表中，请打开**包括不重叠的相对表面**。此类表面仍必须满足搜索的分离和方向要求。请参见["定义接触对的额外标准"中的"接触检测算法，"第15.6.2节"](pt03ch15s06s02.md#usi-itn-auto-detection-filter)，以进一步讨论此选项。
6. 如果您的搜索包括已从几何体网格化的零件实例，请指示Abaqus/CAE在搜索期间应如何解释这些实例：
   - 要将实例视为几何体，请选择**几何体**。Abaqus/CAE默认使用**几何体**选项。
   - 要将实例视为元素网格，请选择**网格**。**提示：**要检测几何体和网格实例之间的接触，请先对几何体进行网格划分，然后执行基于网格的搜索。有关几何体和网格之间差异的讨论，请参见"接触检测算法"中的"网格化模型的接触检测，"第15.6.2节"](pt03ch15s06s02.md#usi-itn-auto-detection-mesh)。
7. 要将当前搜索参数保存为Abaqus/CAE未来会话的默认值，请点击![](../graphics/ico_fileSave.png)（有关更多信息，请参见"使用接触检测工具的提示"中的"保存搜索参数，"第15.6.4节"](pt03ch15s06s04.md#usi-itn-auto-detection-savecurrent)）。要重置保存的搜索参数为原始默认值，请点击![](../graphics/ico_fileReload.png)。
