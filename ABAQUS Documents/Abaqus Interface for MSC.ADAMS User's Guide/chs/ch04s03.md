# 4.3 示例 3：轮胎





本示例对轮胎进行建模。子结构是在解决高度非线性预应力问题后创建的，以考虑轮胎充气和与道路接触产生的接触印痕。

**对轮胎执行分析：**

1. 输入以下命令从 Abaqus 版本提供的压缩档案文件中提取输入文件：``` abaqus fetch job=adams_ex3A abaqus fetch job=adams_ex3A_nodes abaqus fetch job=adams_ex3B abaqus fetch job=adams_ex3C ```
2. 您必须执行三次 Abaqus 分析。
   1. 输入以下命令求解轮胎轴对称模型以进行充气：``` abaqus job=adams_ex3A ```
   2. 输入以下命令从轴对称模型及其结果创建轮胎的三维模型，并计算轮胎与道路接触的接触印痕：``` abaqus job=adams_ex3B oldjob=adams_ex3A ```
   3. 输入以下命令创建子结构模型：``` abaqus job=adams_ex3C oldjob=adams_ex3B ```
3. 输入命令执行 Abaqus Interface for MSC.ADAMS 并创建用于 ADAMS/Flex 的模态中性文件。
   - 在 UNIX 平台上，输入以下命令：``` setenv MDI_MNFWRITE_OPTIONS negative_roots_OK abaqus adams job=adams_ex3C unsetenv MDI_MNFWRITE_OPTIONS ```
   - 在 Windows 平台上，输入以下命令：``` set MDI_MNFWRITE_OPTIONS=negative_roots_OK abaqus adams job=adams_ex3C set MDI_MNFWRITE_OPTIONS= ```

本示例扩展了 ["静态轮胎分析的对称结果传递" Abaqus Example Problems Guide 第 3.1.1 节](../exa/exa-link.md#exa-veh-resultstrans) 中描述的模型讨论。`adams_ex3A` 和 `adams_ex3B` 的 Abaqus 分析实质上复制了该节中描述的轮胎充气和接触印痕分析。但是，对 `adams_ex3B` 进行了一些修改，为接下来的子结构分析做准备：
- 轮辋和轮毂建模为刚体，其参考节点位于轴上。参考节点的六个自由度将成为子结构保留自由度的一部分。
- 通过向该参考节点施加载荷和边界条件来控制接触印痕分析。
- 分析的第一步使用了 [*MODEL CHANGE](../key/key-link.md#usb-kws-hmodelchange), ACTIVATE 选项。此选项不影响该步骤的结果，但需要这样做，以便在 `adams_ex3C` 的 Abaqus 重启动分析中创建子结构之前可以移除道路-轮胎接触对。
- 第三步有 CDISP 和 CSTRESS 的输出请求，以确定在接触印痕分析结束时与道路接触的轮胎节点。这些节点的子集将成为子结构的保留节点。

`adams_ex3C` 的 Abaqus 分析从 `adams_ex3B` 的充气和接触印痕分析重启动，包含以下三个步骤：
- 轮胎与道路分离。使用 [*MODEL CHANGE](../key/key-link.md#usb-kws-hmodelchange), REMOVE, TYPE=CONTACT PAIR 选项移除代表道路的刚体表面。求解的力学保持不变，因为使用 [*BOUNDARY](../key/key-link.md#usb-kws-hboundary), FIXED 选项指定节点集 `FOOTPR` 中的节点具有与上一步结束时计算值相同的位移。此步骤的一个作用是在创建子结构时重新公式化轮胎的刚度矩阵，去除用于施加接触约束的拉格朗日乘子；这会产生更现实的子结构矩阵。此步骤将所有节点的位移写入结果文件，以便将变形的节点坐标写入结果文件。
- 计算轮胎的二十个正则模态。此步骤有边界条件来约束将在子结构中保留的所有自由度，外加额外的约束来保持接触印痕形状。此步骤将所有单元的单元质量矩阵和所有模态的特征向量写入结果文件。此步骤计算的 8 个最低振动频率如 [表 4-4](ch04s03.md#table-admtire-restrained) 所示。

- 创建子结构。保留的节点自由度列表包括轮毂处的六个自由度和接触印痕处 35 个节点的三个自由度。这些贡献了 111 个自由度到子结构。此外，保留了 20 个固定界面正则模态，因此子结构的质心和刚度矩阵具有 131 个自由度。根据子结构的工程用途，您可以选择其他要保留的自由度。您可以尝试保留不同数量的节点，或者可能仅保留某些节点上的法向位移分量。此外，可以改变保留的固定界面正则模态数量。[*SUBSTRUCTURE MATRIX OUTPUT](../key/key-link.md#usb-kws-ssubmatrixoutput) 选项使用可选参数 SLOAD=YES 将模态载荷分量写入结果文件。因此，转换后，对应于预应力轮胎的车辆重量部分的载荷将位于 ADAMS/Flex 使用的模态中性文件中。
