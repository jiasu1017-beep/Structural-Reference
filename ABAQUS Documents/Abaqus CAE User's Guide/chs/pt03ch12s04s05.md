# 12.4.5 选择材料行为





材料编辑器包含多个菜单，允许您将Abaqus/Standard、Abaqus/Explicit或Abaqus/CFD中的大多数材料行为添加到材料定义中。（有关Abaqus/CAE中可用的材料行为的信息，请参见[附录A，"关键字支持"](ap01.md)。）

材料编辑器菜单反映了所有材料行为分为五类：**General**、**Mechanical**、**Thermal**、**Electrical/Magnetic**和**Other**。[图12-8](pt03ch12s04s05.md#prp-buttons)显示了**Mechanical**菜单下可用的弹性行为。

**图12-8** **Mechanical**菜单下的弹性行为。

![](../graphics/prp-buttons-nls.png)

行为列表不会更改以排除对您正在运行的分析类型无效的行为。此外，Abaqus/CAE不会检查您在编辑器中输入的数据是否有效，或者您的材料是否适合您的分析类型。例如，如果您请求动态分析，Abaqus/Standard或Abaqus/Explicit要求您指定模型中使用的材料的密度，以便计算模型的质量和惯性属性。如果您在材料定义中未提供材料密度，Abaqus/CAE允许您创建材料；但是，当您提交分析作业时，Abaqus/CAE将报告错误。

选择行为时，行为的名称会出现在编辑器顶部的**Material Behaviors**列表中，并且该行为成为材料定义的一部分。例如，[图12-9](pt03ch12s04s05.md#prp-list)中的列表反映了已选择**Elastic**和**Plastic**行为，以及**Elastic**行为的**Fail Stress**子选项。

**图12-9** **Material Behaviors**列表。

![](../graphics/prp-list-nls.png)

**Elastic**和**Plastic**等行为是主要行为。测试数据和子选项（如**Fail Stress**）出现在相应主要行为的下方，并缩进以表明其从属地位。

如果您想从材料定义中移除行为或子选项，可以从**Material Behaviors**列表中选择该行为或子选项，然后点击![](../graphics/ico_delete.png)。

如果您正在创建新材料，所选行为列表最初为空白。当您选择行为时，行为名称会出现在列表中；如果行为太多而无法一次看到，列表右侧会出现滚动条。
