# 11.13.4 在拉伸中包含拔模





您可以选择创建带拔模的拉伸。拔模可用于准确表示通常应用于便于从模具中移除铸件或成型零件的小角度。拉伸中的拔模也可用于创建锥形零件。

在直拉伸中，拔模角度为0，因此所有拉伸表面都垂直于原始轮廓。拔模通过调整拉伸表面与原始草图平面之间的角度来修改拉伸。Abaqus/CAE会反转拔模角度的应用，从内部特征应用到外部特征。如果草图轮廓中的外部环正在扩展，则内部环正在收缩；这是拔模的预期行为，也是从模具中移除零件所需要的（所有表面沿同一方向锥化）。

您可以使用特征操作工具集修改拔模以及拉伸轮廓和方向。您可以在创建拉伸实体、壳和切割特征期间添加拔模。

**注意：**拔模切割的完整草图轮廓是一个三角形。如果轮廓是一个梯形，其顶边与块的边缘重合，切割看起来会非常不同。当轮廓被拉伸时，拔模的应用使其变小。梯形轮廓的顶面会立即落到块表面下方，而不是延伸穿过顶面。

Abaqus/CAE无法使用六面体单元对包含拔模的拉伸实体进行网格划分，除非将实体分区为结构化区域。
