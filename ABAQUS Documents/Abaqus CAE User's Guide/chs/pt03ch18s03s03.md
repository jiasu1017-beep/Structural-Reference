# 18.3.3 关于目标函数





目标函数定义您的优化的目标。 目标函数从设计响应中提取，例如最低固有频率或最小应力。 目标函数可以由多个设计响应构成。 如果您指定目标函数最小化或最大化设计响应，Optimization模块通过添加从每个设计响应确定的值来计算目标函数。 此外，您可以指定加权因子（默认加权因子为1.0）。 对于最常见的优化公式，您不需要更改加权因子的默认值。 但是，在某些情况下，您可能需要更改加权因子以平衡主导优化的目标函数的影响。 您应该知道，更改加权因子可能会对最终设计产生重大影响。 此外，在优化开始时占主导地位的设计响应可能会随着Optimization模块修改模型而影响减小。 有关更多信息，请参阅["创建目标函数，" 第18.8节](pt03ch18s08.md)和["目标和约束，" Abaqus Analysis User's Guide第13.2.2节](../usb/usb-link.md#usb-anl-aoptobjectives)。
