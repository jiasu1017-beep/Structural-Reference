# 15.13.16 定义压力渗透







压力渗透相互作用允许您模拟流体在涉及面-面接触的两个表面之间渗透的压力。流体压力垂直于表面施加。形成接头的物体可以都是变形的，如螺纹连接器的情况；或者其中一个可以是刚性的，如使用软垫片作为较硬结构之间的密封时。

压力渗透相互作用可以应用于三维、平面（二维）或轴对称模型。压力渗透相互作用只能用于Abaqus/Standard分析。

在定义压力渗透相互作用之前，您必须创建面-面接触相互作用以指定压力渗透的主表面和从表面；请参见"定义面-面接触"中的"在Abaqus/Standard分析中定义面-面接触"，第15.13.7节。创建面-面接触相互作用时，可以使用**滑动 formulations**和**离散化方法**的任何组合（与压力渗透兼容）。

在压力渗透定义中，您可以识别接触表面、暴露于流体压力的表面区域、流体压力的大小以及作用在区域上的临界接触压力。在三维模型中，可以选择点、边和面作为暴露于流体压力的区域。在二维模型中，只能选择点。在二维模型中，您必须在主表面和从表面上都识别渗透点（除非主表面是解析刚性表面）。

流体可以从表面上的一个或多个区域渗透。这些区域始终承受压力渗透，无论其接触状态如何。流体会渗透到接触物体之间的区域，直到达到接触压力大于指定临界值的点，从而切断流体的进一步渗透。

有关压力渗透的更多信息，请参见Abaqus Analysis User's Guide的第37.1.7节"压力渗透载荷"。

**定义压力渗透：**

1. 从主菜单栏中，选择****相互作用**![](../graphics/images/arrow.gif)**创建****。**提示：**您也可以使用相互作用模块工具箱中的![](../graphics/ico_interaction.png)工具创建压力渗透相互作用。
2. 在出现的**创建相互作用**对话框中，执行以下操作：- 命名该相互作用。有关命名对象的信息，请参见第3.2.1节"使用基本对话框组件"。- 选择将创建该相互作用的步骤。- 选择**压力渗透**类型的相互作用。
3. 单击**继续**关闭**创建相互作用**对话框。
4. 从**接触相互作用**列表中，选择将应用压力渗透的面-面接触相互作用。接触相互作用的主表面和从表面显示在对话框中，表面在视口中高亮显示。
5. 对于三维模型，请在**渗透区域**表中执行以下操作：1. 识别从表面上暴露于流体压力的第一个区域。双击**从表面上的区域**列中的空单元格，或选择该单元格并单击![](../graphics/ico_edit.png)按钮；然后使用以下方法之一选择区域：- 使用鼠标在视口中的模型上选择一个面、边或点。如果您在网格中工作，可以使用鼠标选择节点。- 选择现有集合以指定面、边或点。在提示区域右侧单击**集合**。从出现的**区域选择**对话框中选择现有的面、边或顶点，然后单击**继续**。**注意：**默认选择方法基于您最近使用的选择方法。要恢复为另一种方法，请单击提示区域右侧的**在视口中选择**或**集合**。对于三维模型，不必识别**主表面上的区域**。但是，这样做可以帮助您解决分析中遇到的任何问题。您可以以与**从表面上的区域**完全相同的方式选择**主表面上的区域**。**注意：**如果选定的接触相互作用具有解析刚性主表面，则**主表面上的区域**列显示为暗淡，表示无法添加或编辑主表面点。已指定的任何主表面区域都将被忽略。2. 输入**临界接触压力**，低于该值时流体将开始渗透。该值越高，流体越容易渗透。默认值为零，在这种情况下，流体仅在失去接触时渗透。更多信息，请参见Abaqus Analysis User's Guide的第37.1.7节"压力渗透载荷"中的"指定临界机械接触压力"。3. 输入参考**流体压力**的大小。更多信息，请参见Abaqus Analysis User's Guide的第37.1.7节"压力渗透载荷"中的"指定施加的流体压力"。如果分析步骤是稳态动态（线性扰动）步骤，您可以在表的**流体压力（实部）**和**流体压力（虚部）**列中同时指定压力的实部（同相）和虚部（异相）。有关更多详细信息，请参见Abaqus Analysis User's Guide的第37.1.7节"压力渗透载荷"中的"在线性扰动分析中的使用"。4. 要向表中添加行并继续选择其他渗透区域，请单击![](../graphics/ico_add.png)按钮。此操作直接将您带到视口中的区域拾取步骤。根据需要重复以指定所有区域。5. 要编辑渗透区域，请在表中选择该区域，单击![](../graphics/ico_edit.png)按钮，然后重新选择该区域。6. 要删除区域，请在表中选择该行并单击![](../graphics/ico_delete.png)按钮。
6. 对于平面（二维）或轴对称模型，请在**渗透区域**表中执行以下操作：1. 识别暴露于流体压力的第一对点。双击**主表面上的区域**列中的空单元格，或选择该单元格并单击![](../graphics/ico_edit.png)按钮；然后使用以下方法之一选择点：- 使用鼠标在视口中的模型上选择点。- 选择现有集合以指定点。在提示区域右侧单击**集合**。从出现的**区域选择**对话框中选择现有的节点或顶点，然后单击**继续**。**注意：**默认选择方法基于您最近使用的选择方法。要恢复为另一种方法，请单击提示区域右侧的**在视口中选择**或**集合**。重复此过程以使用表的**从表面上的区域**列选择从表面上对应的渗透点。**注意：**如果选定的接触相互作用具有解析刚性主表面，则**主表面上的区域**列显示为暗淡，表示无法添加或编辑主表面点。已指定的任何主表面点都将被忽略。2. 输入**临界接触压力**，低于该值时流体将开始渗透。该值越高，流体越容易渗透。默认值为零，在这种情况下，流体仅在失去接触时渗透。更多信息，请参见Abaqus Analysis User's Guide的第37.1.7节"压力渗透载荷"中的"指定临界机械接触压力"。3. 输入参考**流体压力**的大小。更多信息，请参见Abaqus Analysis User's Guide的第37.1.7节"压力渗透载荷"中的"指定施加的流体压力"。如果分析步骤是稳态动态（线性扰动）步骤，您可以在表的**流体压力（实部）**和**流体压力（虚部）**列中同时指定压力的实部（同相）和虚部（异相）。有关更多详细信息，请参见Abaqus Analysis User's Guide的第37.1.7节"压力渗透载荷"中的"在线性扰动分析中的使用"。4. 要向表中添加行并继续选择渗透点，请单击![](../graphics/ico_add.png)按钮。此操作直接将您带到视口中的点拾取步骤。根据需要重复以指定所有点。5. 要编辑渗透点，请在表中选择该点，单击![](../graphics/ico_edit.png)按钮，然后重新选择该点。6. 要删除一对渗透点，请在表中选择该行并单击![](../graphics/ico_delete.png)按钮。
7. 在**渗透时间**字段中，输入流体压力达到新渗透表面段上当前全部大小的所需时间周期。默认渗透时间为当前步骤时间的0.001。渗透时间在线性扰动分析中不可用。更多信息，请参见Abaqus Analysis User's Guide的第37.1.7节"压力渗透载荷"中的"指定流体压力的渗透时间"。
8. （可选）您可以通过在**幅值**列表中选择幅值曲线来定义步骤中流体压力的变化。默认情况下，参考大小在步骤开始时立即施加或在整个步骤上线性斜升，具体取决于分配给步骤的幅值变化。对于某些步骤，流体压力幅值曲线不可用。更多信息，请参见Abaqus Analysis User's Guide的第37.1.7节"压力渗透载荷"中的"指定施加的流体压力"。
9. 单击**确定**创建相互作用并关闭编辑器。
