# 76.14 控制实例可见性




默认情况下，Abaqus/CAE 显示装配中包含的所有零件实例。您可以打开或关闭所有实例的显示，也可以切换单个实例的显示。您已抑制或不属于当前显示组的零件实例无法使用此对话框设置为可见；您必须使用特征操作工具集或显示组工具集。有关更多信息，请参见[第13章，"装配模块"](pt03ch13.md)，"或[第78章，"使用显示组显示模型的子集"](pt07ch78.md)。

本节介绍如何从 **装配显示选项** 对话框控制实例可见性。您也可以从模型树或视口更改实例可见性：从模型树中，高亮显示您要显示或隐藏的零件实例，点击鼠标按钮3，并选择 **隐藏** 或 **显示**；从视口中，高亮显示您要隐藏的实例，点击鼠标按钮3，并从出现的菜单中选择 **隐藏实例**。

**控制实例可见性：**

1. 找到 **实例** 显示选项。从主菜单栏，选择 ****视图**![](../graphics/images/arrow.gif)**装配显示选项****。在出现的对话框中点击 **实例** 选项卡。**实例** 选项变得可用；装配中的每个零件实例都会列出。
2. 要控制实例可见性，请执行以下任一操作： - 点击 **全部打开** 使所有（除抑制的）实例可见。 - 点击 **全部关闭** 关闭所有实例的显示。 - 点击各个实例名称以切换其显示状态。
3. 点击 **确定** 实施更改并关闭对话框。更改仅适用于当前视口，并在会话期间保存。
