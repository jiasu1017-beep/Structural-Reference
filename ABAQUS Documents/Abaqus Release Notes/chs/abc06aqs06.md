# 6.6 牵引-分离弹性的拉压不同刚度





**产品：** Abaqus/Standard

**优势：** 现在可以定义具有不同拉压刚度值的非耦合牵引-分离弹性行为。

**说明：** 在某些使用内聚单元同时在拉伸和压缩下模拟粘附行为的应用中，指定牵引-分离响应中拉伸与压缩的不同刚度值非常有用。此功能仅影响法向牵引分量。
