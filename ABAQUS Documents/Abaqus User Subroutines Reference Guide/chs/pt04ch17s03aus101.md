### 概述

本节讨论特定于使用 Abaqus/Standard 程序的电磁-结构和电磁-热协同仿真的分析设置和执行详细信息。支持时谐或瞬态电磁分析与静态、瞬态隐式动态、耦合温度-位移或瞬态热传递分析之间的协同仿真。

电磁到热传递协同仿真分析对诸如感应加热之类的应用有用，它涉及双向耦合：来自电磁分析的 Joule 热产生驱动热传递分析并确定温度分布，而温度分布又通过温度依赖的材料属性（如电导率和磁导率）影响电磁场。对于电磁-热耦合，支持时谐或瞬态电磁分析与瞬态热传递分析之间的协同仿真。

电磁到瞬态隐式动态协同仿真对诸如电磁成形之类的应用有用，其中来自电磁分析的 Lorentz 体力驱动瞬态动态分析。支持瞬态电磁分析与静态或瞬态隐式动态分析之间的协同仿真。然而，耦合只有一个方向；也就是说，域部分变形（如金属工件）对电磁场的影响不考虑。因此，此类分析仅应在变形对电磁场的影响相对较小时使用。

### 识别协同仿真接口区域

电磁模型和结构模型之间的相互作用通过公共体积接口区域进行。

您必须使用单元集在两个 Abaqus/Standard 分析之间指定体积接口区域。您必须在两个 Abaqus/Standard 模拟中的区域定义中保持一致；换句话说，您必须在两个分析中定义相同的接口区域。

您可能在两个模型定义中共享的区域中具有不同的网格。

| **输入文件用法：** | 使用以下选项在 Abaqus/Standard 模型中定义基于单元的协同仿真区域： |
| --- | --- |
|  | ``` [*CO-SIMULATION REGION](../key/key-link.md#usb-kws-hcosimulationregion), TYPE=VOLUME *elset_A* ``` 每个 Abaqus/Standard 分析只能定义一个 [*CO-SIMULATION REGION](../key/key-link.md#usb-kws-hcosimulationregion) 选项。此外，只能定义一个单元集。 |

### 定义会合方案

SIMULIA Co-Simulation Engine 配置文件用于定义两个 Abaqus/Standard 分析之间时间增量过程和交换的频率。

预定义模板可用于常用耦合方案。您可以在创建配置文件时引用这些模板。本节描述会合方案设置和预定义配置文件模板。

#### 时间增量方案

您可以强制两个瞬态 Abaqus/Standard 分析使用相同的增量大小，或者可以允许增量大小不同（子循环）。您为耦合选择的时间增量方案影响求解的计算成本和准确性，但不影响求解稳定性。当使用子循环方法时，此数据交换不表示对 Abaqus/Standard 增量的约束；Abaqus/Standard 分析使用其正常时间增量逻辑推进时间，但根据需要在耦合步长间隔执行数据交换。

时谐电磁程序在频域中定义，没有与瞬态分析相关联的求解时间标度。为时谐电磁程序引入一个伪求解时间标度是方便的，该标度与协同仿真分析中涉及的瞬态分析的某个求解时间间隔相关联。时谐电磁分析的伪时间标度跟随瞬态热传递分析中的求解时间标度，并在每个耦合步中以如下所述的方式重置。

#### 定义耦合方案

顺序显式耦合方案（也称为 Gauss-Seidel 耦合算法）和迭代耦合方案可用于电磁-结构和电磁-热协同仿真。电磁分析必须始终领先于协同仿真，而热传递或应力分析必须始终滞后于协同仿真。所有预定义模板都使用上述领先-滞后序列设置。

#### 耦合步长

耦合步长是两个 Abaqus/Standard 分析之间连续两次协同仿真数据交换之间的时间段。对于瞬态电磁到瞬态热传递或瞬态隐式动态协同仿真，耦合步长可以指定为等于个体分析自动时间增量方案确定的最小时间步长，或者可以指定为恒定用户定义值。

当领先电磁分析是时谐的时，耦合步长可以指定为等于滞后瞬态热传递或隐式动态分析的时间步长，或者可以指定为恒定用户定义值。在后一种情况下，时谐电磁分析将在每个连续恒定用户定义耦合步长结束时求解场，而滞后热传递或应力分析通常将子循环直到达到目标耦合步长时间。

对于迭代耦合，两个分析必须在每个增量结束时耦合，不应使用子循环。如果在此情况下使用子循环，则在迭代期间交换的更新求解仅在最后一个增量时使用，并且在先前增量（在上一次耦合和当前耦合之间）上的更新累积效果将丢失。

```

```

下面显示了示例文件 `exa_em_std_export.xml`。
```

<CoupledMultiphysicsSimulation>
   <template_em_std_export>
      <EM_Job>*em_job_name*</EM_Job>

```

在某些情况下，您可能需要使用预定义模板中未描述的协同仿真配置功能。例如，您可能希望更改不同网格映射搜索容差；这些容差在配置文件中通常可用，但未在预定义模板中描述。对于这些情况，您必须创建详细的配置文件；更多信息，请参见 ["准备 Abaqus 分析进行协同仿真，" 第 17.2.1 节中的"使用详细的配置文件"](pt04ch17s02aus98.md#usb-anl-acosimulationprep-elab)。
