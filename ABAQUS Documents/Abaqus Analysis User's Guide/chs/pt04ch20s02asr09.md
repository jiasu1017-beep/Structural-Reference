# 20.2.9 aStudy.report()





### 20.2.9 **_aStudy_.report()**: 报告参数化研究结果。

**产品:** Abaqus/Standard  Abaqus/Explicit

此命令用于报告跨参数化研究设计收集的结果。

##### **参考**

- ["参数化研究脚本," 第20.1.1节](pt04ch20s01aus108.md)

### **命令:**

**_aStudy_.report** (*token*, **results**= , **par**= , **designSet**= , **variations**=, **truncation**= , *additional data*)

### **标记:**

FILE

使用此标记指定结果要作为带有相关标题的表写入ASCII文件。

PRINT

使用此标记指定结果要作为带有相关标题的表打印。由于结果被打印到默认输出设备（屏幕），您可能希望限制表中的列数，以使表可读。

XYPLOT

使用此标记指定结果要作为不带标题的表写入ASCII文件。随后可以由Abaqus/CAE的Visualization模块读取此表，以显示结果和参数值的X-Y图。

### **必需数据:**

**results**

设置 **results** 等于要报告的结果名称序列；此序列必须用括号或方括号括起来。例如，`results=('e33_sinv.1', 'e52_strain', 'n25_u.3')`，其中 `'e33_sinv.1'` 是单元33的Mises应力（SINV记录的第一个分量），`'e52_strain'` 是单元52的所有应变分量，`'n25_u.3'` 是节点25的位移的第三个分量。此示例假设上述三个结果是通过分别请求SINV、E和U变量标识符键在先前的 **gather** 命令中收集的。

### **可选数据:**

**par**

设置 **par** 等于要包含在报告表中的参数或参数序列的名称。如果指定单个参数，必须用匹配引号括起来；例如，`'par1'`。如果指定参数序列，必须将其放在括号或方括号内，并且必须包含用匹配引号括起来并用逗号分隔的参数名称；例如，`('par1', 'par2', 'par3')` 或 `['par1', 'par2', 'par3']`。

如果省略 **par**，则参数化研究中的所有参数都包含在报告表中。

**designSet**

设置 **designSet** 等于要包含其结果的报告表的设计集名称；此名称必须用匹配引号括起来。

如果省略 **designSet**，则参数化研究中的所有设计集的结果都包含在报告表中。

**variations**

设置 **variations** 等于ON，以指示报告表的第一列必须显示被报告的设计名称。设置 **variations** 等于OFF，以指示不被报告的设计名称不应出现在报告表的第一列中。

如果省略 **variations**，则设计名称列不包含在报告表中。

**truncation**

设置 **truncation** 等于ON，以指示必须以有限精度报告报告表的数据。设置 **truncation** 等于OFF，以指示必须以完整精度报告报告表的数据。

如果省略 **truncation**，则以有限精度报告报告表的数据。

##### **FILE和XYPLOT的附加数据:**

### **必需数据:**

**file**

设置 **file** 等于要写入报告表的文件名。该文件名必须用匹配引号括起来。
