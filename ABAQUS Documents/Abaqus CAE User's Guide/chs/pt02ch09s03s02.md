# 9.3.2 访问网络ODB连接器的安全性如何？





Abaqus/CAE通过生成在服务器和客户端之间来回传递的密钥来维护与网络ODB连接器的安全连接。如果远程服务器上的主目录中存在名为`.abaqus_net_passwd`的文件，Abaqus/CAE会使用该文件中的密码进行身份验证，而不是使用Abaqus/CAE生成的密钥。Abaqus/CAE会检查您是否是唯一有权读取和写入密码文件的用户。此外，您必须在30天后更新密码，密码必须至少八个字符长。如果您手动启动网络ODB服务器，Abaqus会使用密码文件来验证客户端和服务器之间的连接。这些文件在["网络输出数据库文件连接器，"Abaqus Analysis User's Guide第3.2.24节"](../usb/usb-link.md#usb-int-dremoteodbproc)中有描述。
