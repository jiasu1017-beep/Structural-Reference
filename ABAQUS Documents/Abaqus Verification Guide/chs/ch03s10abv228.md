# 3.10.3 应力/位移模型变化：动态







**产品：**Abaqus/Standard  

### 测试的单元

CPS4

### 测试的功能

我们研究当两个不同系统的质量和几何形状随单元移除而变化时，它们的固有频率如何变化。移除的单元也被重新添加，以便恢复原始系统的响应。主导模态通过频率提取运行计算，也使用直接积分动力学计算。

### 问题描述

下面讨论频率提取和动态分析。

#### 频率分析

对悬臂梁进行固有频率提取。移除单元以缩短梁的长度，从而改变频率含量。

**材料特性：**

超弹性材料，多项式，N=1

![](../graphics/ver_eqn00068.gif) = 56.0×10^4，![](../graphics/ver_eqn00069.gif) = 14.0×10^4，![](../graphics/ver_eqn00073.gif) = 1.43×10^7

**尺寸：**

在*x*–*y*平面上为10.0×1.0，平面外为1.0。

**载荷和边界条件：**

梁在一端为悬臂端。步骤1是建立基态的空步骤。步骤2中提取前五个特征值。步骤3中移除单元4–6、14–16、24–26和34–36。步骤4中梁的端部缩短0.5个单位（![](../graphics/ver_eqn00107.gif) = 0.5）。在步骤5中，将移除的单元重新添加到模型中。在步骤6中获得缩短梁的前五个特征值。[图3.10.3-1](ch03s10abv228.md#vermodelchangedyn-mesh-freq)显示了该测试中使用的网格。

#### 动态分析

一个由八个单元组成的块连接到一个接地弹簧，被给予超出静态平衡的初始位移并被允许振动。响应与具有原始质量四分之一的相同系统的振动进行比较。

**材料特性：**

弹性模量 = 207.0×10^12

泊松比 = 0.3

密度 = 7800.0

弹簧刚度 = 9.8538×10^6

**尺寸：**

模型在*x*–*y*平面上具有8.0×4.0的尺寸，平面外为1.0。

**载荷和边界条件：**

在步骤1中，模型右侧的节点沿1方向位移1.0个单位。在步骤2中它们被释放。所有节点被约束为只能沿1方向滑动。在步骤3中，所有节点被固定，单元2–4和12–14被移除。剩余单元在步骤4中被允许移动。在此步骤中，移除单元上的自由节点（即没有任何单元贡献质量）被保持固定。在步骤5中，整个模型再次被固定，单元被重新添加到模型中。在步骤6中所有节点都被释放。[图3.10.3-2](ch03s10abv228.md#vermodelchangedyn-mesh-dyn)显示了该测试的网格。

**图3.10.3-2** 动态程序测试中使用的网格。

悬臂梁的第一固有频率与分析解的误差在2%以内。瞬态动力学中弹簧-质量系统的周期与上述所有动态步骤（步骤2、4和6）的预期分析解一致。输出垂直左边缘上的总力。
