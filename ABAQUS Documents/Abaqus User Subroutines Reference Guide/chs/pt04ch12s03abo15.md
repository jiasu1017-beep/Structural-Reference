# 12.3.1 自适应重新网格划分：概述







Abaqus/CAE 提供了一个自动化流程来自适应地重新网格划分您的模型。自适应重新网格划分过程的目的是为指定模型及其伴随的载荷历史接近或达到所选误差指标的目标。见 ["自适应技术，" 第 12.1.1 节"](pt04ch12s01aus77.md)，了解此过程与其他 Abaqus 自适应方法的比较。

### 概述

将自适应重新网格划分整合到 Abaqus/CAE 模型中需要以下步骤：
- 您确定希望应用一个或多个自适应重新网格划分规则的模型区域。重新网格划分规则定义了将应用它的步骤、该步骤的误差指标输出变量及其目标、尺寸方法以及任何单元尺寸约束。见 ["什么是重新网格划分规则？，" Abaqus/CAE 用户指南第 17.13.1 节"](../usi/usi-link.md#usi-mgn-conc-adaptivity-rulerole)。
- 您定义了一系列分析作业，即"自适应过程"，它将作为 Abaqus/CAE 尝试满足您的重新网格划分规则目标而运行。见 ["什么是自适应过程？，" Abaqus/CAE 用户指南第 19.3.1 节"](../usi/usi-link.md#usi-ana-adaptivity-steps)。

基于这些重新网格划分规则和您的自适应过程定义，Abaqus/CAE 迭代地：
- 执行 Abaqus/Standard 分析，它将根据您的重新网格划分规则设置写入所选的误差指标输出变量（见 ["影响自适应重新网格划分的误差指标选择，" 第 12.3.2 节"](pt04ch12s03aus84.md)），
- 使用误差指标变量在尺寸函数中计算新网格的单元尺寸，遵守您可能指定的任何尺寸约束（见 ["基于解的网格尺寸，" 第 12.3.3 节"](pt04ch12s03aus85.md)），和
- 根据计算的单元尺寸在指定区域生成新网格。相邻区域也将被重新网格划分。

这些迭代继续进行，直到：
- 所有重新网格划分规则目标都满足，或
- 达到最大重新网格迭代次数。

有关更多详细信息，请参阅 ["我的网格自适应何时停止迭代？，" Abaqus/CAE 用户指南第 19.3.2 节"](../usi/usi-link.md#usi-ana-adaptivity-automatic)。[图 12.3.1-1](pt04ch12s03abo15.md#aadaptivity-bigloop) 显示了在此过程中 Abaqus 产品和文件的交互。

### 典型应用

自适应重新网格划分可以提高模拟结果的质量。在以下情况下，自适应重新网格划分可能会有所帮助：
- 您不确定需要多么精细的网格才能达到特定的精度水平，或者网格可以多么粗糙而不会不可接受地影响求解精度；
- 在感兴趣区域（如应力集中区域）附近难以设计足够精细的网格；或
- 您不知道感兴趣的位置，如塑性区域的形成，先验地。

在 ["反应堆压力容器螺栓封闭的热-应力分析，" Abaqus 例题指南第 5.1.6 节"](../exa/exa-link.md#exa-htr-reactor) 中提供了一个使用自适应重新网格划分研究螺栓容器热行为和应力行为的示例。该示例包括一个 Python 脚本，您可以将其从 Abaqus/CAE 运行以创建模型和重新网格划分规则。第二个脚本允许您提交自适应过程并查看网格的变化，因为 Abaqus/CAE 计算新的单元尺寸。

### 为自适应重新网格划分准备模型

在使用自适应重新网格划分时，您使用 Abaqus/CAE 执行以下操作：
- 创建模型并指定边界条件和载荷历史，
- 创建重新网格划分规则，
- 创建自适应过程，和
- 启动并监控自适应过程的进度。

#### 创建模型

在创建模型并指定边界条件和载荷历史时，您不必考虑自适应重新网格划分；但是，在使用自适应重新网格划分之前，您必须执行以下操作：
- 创建模型的几何形状——您不能使用孤立网格部件——和
- 提供初始标称网格。此网格可以相当粗糙。但是，提供极其粗糙的网格可能会因早期重新网格迭代误差指标计算的质量不佳而导致更多自适应重新网格迭代。在典型情况下，您可以通过使用 Abaqus/CAE 中的默认部件实例网格播种来定义合理的初始网格。

#### 创建重新网格划分规则

您使用 Abaqus/CAE 中的网格模块创建和配置重新网格划分规则。有关定义重新网格规则的详细信息，请参阅 ["创建重新网格划分规则，" Abaqus/CAE 用户指南第 17.21.1 节"](../usi/usi-link.md#usi-mgn-adaptivity-rule)。有关用于确定修订网格尺寸分布的方法的详细信息，请参阅 ["影响自适应重新网格划分的误差指标选择，" 第 12.3.2 节"](pt04ch12s03aus84.md) 和 ["基于解的网格尺寸，" 第 12.3.3 节"](pt04ch12s03aus85.md)。

### 使用临时分析执行自适应重新网格划分

在某些情况下，您可能希望在执行完全详细分析之前确定模型的适当网格，这可能包括许多步骤和复杂行为。"临时"分析通常可以与自适应重新网格划分结合使用，以有效地确定模型的良好网格。临时分析可能包括完全详细分析的各种简化，例如：
- 用单一线性扰动步骤替换您的步骤，该步骤的载荷充分反映您更一般的载荷情况，
- 移除塑性和其他材料非线性，和
- 禁用几何非线性。

临时分析方法可能会导致网格不太适合您的最终载荷选择。但是，从临时模型获取网格的成本可能显著低于您的自适应过程考虑完全详细分析的所有复杂性的情况，并且您可能会发现细化的网格适用于各种分析情况。

### 特殊注意事项

一般来说，Abaqus 自适应重新网格划分过程自动迭代以获得更好的网格质量；但是，您应该注意某些考虑事项。

#### 奇异性

应力奇异性通常由几何抽象引起，如弹性材料中的凹角和尖锐边缘的接触，以及点载荷或突然结束的分布载荷区域。在这些情况下，奇异性附近的应力场是无限的，并且没有任何网格细化能够解析正确的解决方案。如果您将自适应重新网格划分过程应用到包含奇异性的模型区域，该过程将驱动靠近奇异性的单元变得非常小。最终结果可能是代价过于昂贵的分析。

您可以使用以下技术防止具有奇异性的模型进行过度昂贵的分析：
- 从重新网格划分过程中排除奇异区域。您通过对模型进行分区并仅对远离奇异性的区域分配重新网格划分规则来排除区域。
- 在重新网格划分规则中应用最小单元尺寸约束。Abaqus/CAE 默认分配最小单元尺寸，它是默认部件实例网格播种的一部分。您可以修改此约束以在奇异性附近获得良好的解决方案，同时避免过度细化的网格。您也可以使用重新网格划分规则来控制 Abaqus/CAE 细化单元尺寸的速率。单元尺寸约束可能会阻止自适应过程实现指定的误差指标目标。
- 为重新网格划分规则区域指定最大单元数量。Abaqus/CAE 调整网格尺寸，使得生成的单元总数大致满足此约束。

[图 12.3.1-6](pt04ch12s03abo15.md#aadaptivity-costanderror) 中的示例显示了理想的收敛曲线。求解误差指标单调地快速下降到所需的 25% 误差指标目标。随着误差指标的下降，计算成本适度增加，以模型自由度或 Abaqus/Standard 中的时间衡量。某些情况可能会干扰这种理想的收敛曲线，如下所示：
- 如果您的初始网格太粗糙，误差指标变量的质量可能不足以在下次迭代中获得充分改进的网格。自适应重新网格划分过程通常最终会创建高质量的网格，即使初始网格相当粗糙。但是，通过合理细化的初始网格可以避免一些网格迭代。
- 您在创建重新网格划分规则时指定的最小单元尺寸约束和最大单元数量约束可能会阻止网格实现足够的细化（在奇异性的极端情况下总是如此）以满足您的误差指标目标。您可能可以通过放宽这些约束来满足您的目标；例如，通过减小最小单元尺寸。有关更多信息，请参阅 ["什么是重新网格划分规则？，" Abaqus/CAE 用户指南第 17.13.1 节"](../usi/usi-link.md#usi-mgn-conc-adaptivity-rulerole)。
- 除了产生导致大量单元的小网格外，奇异性还可能导致自适应过程无法实现误差目标或需要更多重新网格迭代。如上所述 ["奇异性"](pt04ch12s03abo15.md#usb-anl-aadpover-singularities)，您可以通过指定最小单元尺寸约束或最大单元数量来控制计算成本。在任何奇异性存在于重新网格划分规则区域内的情况下，您可能会在误差指标结果中看到较差的收敛。
- 与二次单元（C3D10、CPS6 等）相比，线性单元（C3D4、CPS4 等）和修改单元（C3D10M、CPS6M 等）收敛缓慢，需要相对大量的单元才能达到给定的误差目标。因此，只要可能，您就应该使用二次单元。

#### 继续停止的自适应重新网格划分过程

自适应重新网格划分过程设计为自动的——Abaqus/CAE 执行一系列分析，因为它继续细化您的网格。但是，在某些情况下该过程会停止，您可能希望从最新的网格继续自适应重新网格划分：
- 当您想要更改后续重新网格迭代的重新网格划分规则时，或
- 当自适应重新网格过程由于机器资源问题而无法完成时。

您可以通过重新提交现有自适应过程、创建和提交新的自适应过程或执行手动重新网格划分来继续自适应重新网格划分过程。见 ["手动调整大小和重新网格划分，" Abaqus/CAE 用户指南第 17.21.6 节"](../usi/usi-link.md#usi-mgn-adaptivity-remeshing)。

### 限制

自适应重新网格划分需要使用 Abaqus/CAE，并且仅支持 Abaqus/Standard 过程。其他特定限制也适用。

#### 单元类型

Abaqus/CAE 只能对以下形状的单元执行自适应重新网格划分（见 ["我可以将哪些网格控制与自适应重新网格划分一起使用？，" Abaqus/CAE 用户指南第 17.13.2 节"](../usi/usi-link.md#usi-mgn-conc-adaptivity-elements)）：
- 平面连续体三角形和四边形
- 壳三角形和四边形
- 四面体

- ["准静态分析，" 第 6.2.5 节"](pt03ch06s02at04.md)。
