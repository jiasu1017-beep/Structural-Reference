# C.2 安装后设置文档 URL





如果您在安装 Abaqus HTML 文档之前安装 Abaqus 产品，或者如果您在产品安装期间未指定 HTML 文档的 URL，则必须将 URL 添加到 `abaqus_v6.env` 和 `url.js` 文件。

### C.2.1 在 `abaqus_v6.env` 文件中设置 **doc_root** 参数

文档 URL 在 `abaqus_v6.env` 文件中是必需的，以提供对 Abaqus 中 HTML 文档的访问。`abaqus_v6.env` 文件位于 `*abaqus_dir*/6.14-*n*/SMA/site/` 目录中，其中 *abaqus_dir* 是 Abaqus 父目录，*n* 是 Abaqus 更新版本。所需参数定义如下：
- 带 Web 服务器的 HTML 和 PDF 文档安装 ``` doc_root="http://*computername*:*port_number*/v6.14" ``` 其中 *computername* 是 Web 服务器的完全限定域名，*port_number* 是 Web 服务器使用的端口号（如果需要）。例如，有效的参数定义是 ``` doc_root="http://willow.xyz.com:2080/v6.14" ```
- 无 Web 服务器的 HTML 和 PDF 文档安装 ``` doc_root="file:///*simulia_dir*/Documentation/docs/ v6.14/index.html" ``` 其中 *simulia_dir* 是 HTML 文档安装父目录。例如，有效的参数定义是 **Linux 平台** ``` doc_root="file:///usr/local/SIMULIA/Documentation/docs/ v6.14/index.html" ``` **Windows 平台** ``` doc_root="file:///d:/SIMULIA/Documentation/docs/ v6.14/index.html" ```
- 仅 PDF 文档安装（无 Web 服务器）。在这种情况下，您必须设置两个参数的值，如下所示： ``` doc_root_type="pdf" doc_root="*simulia_dir*" ``` 其中 *simulia_dir* 是安装基础目录的路径。

### C.2.2 在 `url.js` 文件中设置文档 URL

文档 URL 在 `url.js` 文件中是必需的，以启用 Abaqus/CAE 中帮助窗口到 Abaqus HTML 文档中相关主题的链接。`url.js` 文件位于 `*abaqus_dir*/6.14-*n*/Help/redirects` 目录中，其中 *abaqus_dir* 是 Abaqus 父目录，*n* 是 Abaqus 更新版本。`url.js` 文件中的行应采用以下形式：
- 带 Web 服务器的 HTML 文档安装 ``` var baseURL="http://*computername*:*port_number*/v6.14/books/"; ``` 其中 *computername* 是 Web 服务器的完全限定域名，*port_number* 是 Web 服务器使用的端口号（如果需要）。例如，有效的定义是 ``` var baseURL="http://willow.xyz.com:2080/v6.14/books/"; ```
- 无 Web 服务器的 HTML 文档安装 ``` var baseURL="file:///*simulia_dir*/Documentation/docs/ v6.14/books/"; ``` 其中 *simulia_dir* 是 HTML 文档安装父目录。例如，有效的定义是 **Linux 平台** ``` var baseURL="file:///usr/local/SIMULIA/Documentation/docs/ v6.14/books/"; ``` **Windows 平台** ``` var baseURL="file:///d:/SIMULIA/Documentation/docs/ v6.14/books/"; ```
