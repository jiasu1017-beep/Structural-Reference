# 17.14.1 对多个三维实体区域进行网格划分





Abaqus/CAE根据每个区域的几何形状和拓扑分配默认的网格技术。然而，有时三维零件或零件实例相邻区域应用的默认网格技术不兼容，Abaqus/CAE无法生成兼容的网格。

例如，Abaqus/CAE无法使用默认网格技术在图17-123的整个零件实例上生成兼容网格，因为左侧结构化网格的节点无法与右侧扫掠网格的节点合并。（零件实例右侧的立方体是一个扫掠区域，因为它与也是扫掠区域的圆柱体相连。）

**图17-123** 使用这些默认网格技术无法生成兼容网格。

![](../graphics/mgn-mapsweep-nls.png)

如果您分别对两个区域进行网格划分，结构化区域节点和扫掠区域节点之间将发生的不匹配是明显的，如图17-124所示。

**图17-124** 结构化区域和扫掠区域分别进行网格划分。

![](../graphics/mgn-incompat.png)

如果您启动网格划分过程且Abaqus/CAE无法使用默认网格技术生成兼容网格，则Abaqus/CAE会尝试用新的网格技术替换默认网格技术。这些新技术不仅由区域的几何形状和拓扑决定，还由零件或零件实例中相邻区域的特征决定。Abaqus/CAE评估区域之间的界面，并尝试最小化不兼容界面的数量。

例如，图17-123中零件实例左侧立方体的默认网格技术是结构化的，结果导致的不兼容网格如图17-124所示。然而，这个立方体也可以使用扫掠网格技术进行网格划分。因此，Abaqus/CAE将此区域分配的网格技术从结构化更改为扫掠，并在整个零件实例上生成兼容的网格。（当Abaqus/CAE更改分配给区域的网格技术时，分配给区域的单元形状保持不变。）

当您启动三维零件或零件实例的网格划分过程时，Abaqus/CAE会确定是否可以使用分配给每个区域的默认技术生成兼容网格。如果可以生成兼容网格，则继续进行网格划分。如果无法使用默认技术生成兼容网格，则Abaqus/CAE会检查是否可以替换默认网格技术为允许生成兼容网格的不同技术。
- 如果不同的技术允许生成兼容网格，Abaqus/CAE会高亮显示不兼容的界面，并提示您选择以下选项之一：
  - 取消网格划分过程。
  - 允许Abaqus/CAE根据需要替换默认技术并生成兼容网格。
  - 允许Abaqus/CAE使用默认网格技术并在不兼容界面上自动生成绑定约束。Abaqus/CAE自动选择界面的一侧作为从属面，另一侧作为自动生成的绑定约束的主面，但在不兼容界面的周边创建公共（合并）节点。从属面上的节点被约束为具有与它们绑定到的主表面上的点相同的位移、温度、孔隙压力或电势值。Abaqus/CAE通常选择具有较细网格的面作为从属面。绑定约束的从属节点调整区深度的计算基于界面区域的边界尺寸。（有关绑定约束的更多信息，请参见["网格绑定约束，"Abaqus Analysis User's Guide第35.3.1节](../usb/usb-link.md#usb-cni-ptiedconstraint)。）
- 如果不同的技术仍然不允许生成兼容网格，Abaqus/CAE会高亮显示不兼容的界面，并提示您选择以下选项之一：
  - 取消网格划分过程。
  - 如上所述，在不兼容界面上自动生成绑定面相互作用。

如果无法生成兼容网格，您可以尝试以下方法之一：
- 根据需要分割以生成兼容网格。
- 使用自由网格技术对整个零件或零件实例进行网格划分。

一般来说，对三维实体零件或零件实例生成兼容网格时适用以下限制：
- 扫掠区域不能与其结构化区域共享目标侧。但是，它可以与结构化区域共享源侧或连接边，如图17-125所示。
**图17-125** 扫掠区域的连接边与结构化区域共享。
![](../graphics/mgn-mapsweep2-nls.png)
- 在某些情况下，Abaqus/CAE无法对包含多个都分配了扫掠网格技术的区域的零件或零件实例进行网格划分。例如，Abaqus/CAE无法沿图17-126所示的零件实例扫掠网格，因为在共享目标面上无法生成兼容网格。
**图17-126** 沿此零件实例无法生成兼容的扫掠网格。
![](../graphics/mgn-multisweep-nls.png)
然而，图17-127展示了如何使用分割来生成包含四个扫掠区域的网格。
**图17-127** 使用分割生成兼容的扫掠网格。
![](../graphics/mgn-extendfaces-nls.png)

同一零件或零件实例的不同区域可以使用六面体和四面体单元进行网格划分，如图17-128所示。

**图17-128** 同一零件或零件实例的不同区域可以使用六面体和四面体单元进行网格划分。

![](../graphics/mgn-hexandtet-nls.png)

您可以在精度重要的区域使用六面体单元，例如接触面附近或需要细网格的特殊感兴趣区域。您可以在其他区域使用四面体单元，Abaqus/CAE在区域连接处创建绑定面。当您对一个区域进行网格划分时，Abaqus/CAE不会调整相邻区域上的现有网格。
