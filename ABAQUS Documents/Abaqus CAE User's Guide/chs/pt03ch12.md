# 12 Property模块





您可以使用Property模块执行以下任务：
- 定义材料。
- 定义梁截面轮廓。
- 定义截面。
- 为部件分配截面、取向、法线和切线。
- 定义复合铺层。
- 定义蒙皮增强。
- 定义部件上的惯性（点质量、旋转惯性和热容）。
- 定义两点之间或点到地面之间的弹簧和阻尼器。
- 定义材料校准。
