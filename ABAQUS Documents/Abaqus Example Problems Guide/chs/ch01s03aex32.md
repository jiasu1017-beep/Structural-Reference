# 1.3.1 圆柱坯料的镦粗：网格到网格解映射（Abaqus/Standard）和自适应网格划分（Abaqus/Explicit）的准静态分析





**产品：** Abaqus/Standard  Abaqus/Explicit  Abaqus/CAE  

本示例说明了在金属成形应用中Abaqus/Standard的解映射功能和Abaqus/Explicit的自适应网格划分功能的使用；分析结果与Taylor（1981）的结果进行了比较。同一问题也使用耦合温度-位移单元进行了分析，参见["圆柱坯料的镦粗：耦合温度-位移和绝热分析，"第1.3.16节](ch01s03aex47.md)。在本示例中包含耦合温度-位移单元仅用于解映射验证目的；这些单元不产生热量。

当几何非线性分析中的应变变得很大时，单元往往会发生严重扭曲，不再能够很好地离散问题。当发生这种情况时，有必要将解映射到一个新的网格上，这个网格更适合继续分析。在Abaqus/Standard中，程序是监控网格的扭曲程度——例如，通过观察变形构型图——并决定何时需要映射网格。当网格扭曲非常严重以至于必须创建新网格时，可以使用Abaqus/CAE中的网格生成选项来生成新网格。输出数据库在此上下文中很有用，因为可以从输出数据库中的数据提取模型的当前几何形状。一旦定义了新网格，分析通过从旧网格在映射点的解开始一个新的问题来继续，方法是在上一个分析中指定应从中读取解的步骤数和增量数。Abaqus/Standard将解从旧网格插值到新网格以开始新问题。这种技术提供了相当大的通用性。例如，新网格可能在高应变梯度区域更密集，而在刚性移动区域元素更少——对新旧网格的元素数量没有限制，也不要求单元类型一致。在典型的制造工艺实际分析中，由于此类工艺涉及的大形状变化，可能需要多次进行网格到网格解映射。

Abaqus/Explicit具有使用自适应网格划分进行自动解映射的功能。因此，映射过程更容易，因为它包含在分析内部，用户只需要决定重网格化的频率以及使用什么方法将解从旧网格映射到新网格。Abaqus/Explicit为自适应网格划分提供了默认选择，已被证明适用于各种问题。最后，使用解相关的网格划分来将网格细化集中在演化边界曲率的区域。这抵消了基本平滑方法在凹边界附近减少网格细化的趋势，而解精度在凹边界处很重要。

### 几何和模型

几何是Lippmann（1979）的标准测试用例，定义在["圆柱坯料的镦粗：耦合温度-位移和绝热分析，"第1.3.16节](ch01s03aex47.md)中。它是一个圆形坯料，长30 mm，半径10 mm，在两个平的刚性模具之间被压缩，模具被定义为完全粗糙。

用于开始分析的网格如图1.3.1-1所示。有限元模型是轴对称的，仅包括坯料的上半部分，因为坯料的中间表面是对称平面。在Abaqus/Standard和Abaqus/Explicit模拟中都使用CAX4R单元类型：这是一个4节点四边形，具有单个积分点和"沙漏控制"来控制由完全减缩积分引起的伪机制。选择此单元是因为对于涉及非线性本构行为的问题，它相对便宜，因为材料计算只在每个单元的一个点进行。此外，在Abaqus/Standard模拟中还使用CGAX4R、CGAX4T、CAX4P和CAX4I单元类型来建模坯料；在Abaqus/Explicit模拟中还使用CAX6M单元类型来建模坯料。

坯料顶部和侧面外表面的接触以及刚性模具用接触对建模。坯料表面在模型中指定为表面定义。刚性模具有多种不同的建模方式，如[表1.3.1-1](ch01s03aex32.md#case-table)所述。接触表面之间的机械相互作用假定为非间歇性、粗糙的摩擦接触。因此，接触属性包括两个附加规格：粗糙摩擦以在两个表面之间施加无滑移约束，以及确保一旦建立接触就不会发生分离的no-separation接触压力-过盈关系。Abaqus/Standard中的所有接触模拟都使用节点-表面公式，但有一个案例引入了表面-表面公式。

[表1.3.1-1](ch01s03aex32.md#case-table)总结了所研究的不同分析案例。列标题表示问题是否使用Abaqus/Standard和/或Abaqus/Explicit进行了分析。

对于案例1，执行了多个不同的分析来比较Abaqus/Explicit中可用的不同截面控制选项，并评估使用CAX4R单元建模的坯料的网格细化效果。使用纯刚度形式的沙漏控制的粗网格（分析COARSE_SS）和细网格（分析FINE_SS）进行了分析。使用组合沙漏控制的粗网格（分析COARSE_CS）进行了分析。使用基于增强应变方法的沙漏控制的粗网格（分析COARSE_ENHS）和细网格（分析FINE_ENHS）进行了分析。使用积分粘弹性形式的沙漏控制的默认截面控制，在粗网格（分析COARSE）和细网格（分析FINE）上进行了测试。由于这是准静态分析，不应使用粘性沙漏控制选项。所有其他情况使用默认截面控制。

案例1的Abaqus/Standard分析比较了两种沙漏控制选项，并评估了使用CAX4R单元建模的坯料的网格细化效果。使用纯刚度形式的沙漏控制的粗网格（分析COARSE_S）和细网格（分析FINE_S）进行了分析。使用基于增强应变方法的沙漏控制的粗网格（分析COARSE_EH）和细网格（分析FINE_EH）进行了分析。使用CGAX4R单元的粗网格（分析COARSE_EHG）也使用基于增强应变方法的沙漏控制进行了分析，用于比较目的。

没有进行网格收敛研究，但与Lippmann（1979）给出的结果的一致性表明，这里使用的网格足以对模具上的总体力提供合理的预测。

### 材料

坯料假定的材料模型如Lippmann（1979）所述。杨氏模量为200 GPa，泊松比为0.3，密度为7833 kg/m³。使用率无关的von Mises弹塑性材料模型，屈服应力为700 MPa，硬化斜率为0.3 GPa。

### 边界条件和加载

运动学边界条件是轴线上的对称（节点在![](../graphics/exa_eqn00260.gif) 0处，在节点集`AXIS`中，具有规定的![](../graphics/exa_eqn00261.gif) 0）和关于![](../graphics/exa_eqn00047.gif) 0的对称（所有节点在![](../graphics/exa_eqn00047.gif) 0处，在节点集`MIDDLE`中，具有规定的![](../graphics/exa_eqn00040.gif) 0）。坯料顶部表面上位于对称轴上的节点不属于节点集`AXIS`，以避免过度约束：该节点的径向运动已经由无滑移摩擦约束约束（参见["Abaqus/Standard中接触建模的常见困难，"Abaqus分析用户指南第39.1.2节](../usb/usb-link.md#usb-cni-acontacttrouble)，以及["使用接触对的Abaqus/Explicit中接触建模的常见困难，"Abaqus分析用户指南第39.2.2节](../usb/usb-link.md#usb-cni-aexpcontacttrouble)）。

在Abaqus/Standard中，刚性模具在轴向方向上被位移9 mm，使用位移边界条件。在Abaqus/Explicit中，刚性模具的![](../graphics/exa_eqn00262.gif)位移使用速度边界条件规定，其值斜坡上升至20 m/s的速度，然后保持恒定，直到模具总共移动9 mm。Abaqus/Explicit分析的总模拟时间为0.55毫秒，加载速率足够慢，可以视为准静态。在Abaqus/Standard和Abaqus/Explicit中，刚性模具的径向和旋转自由度都受到约束。

对于所有情况，分析分两步进行，以便第一步可以在对应于44%镦粗的模具位移处停止；第二步将分析推进至60%镦粗。在Abaqus/Standard模拟中，解映射分析从第一步结束处使用新网格重新开始，一直进行到达到60%镦粗。

### Abaqus/Standard中的网格到网格解映射

解映射中使用的插值技术是一个两步过程。首先，通过从积分点外推到每个单元的节点，并平均关于每个节点的相邻所有单元的值，获得旧网格节点上所有解变量的值。第二步是定位新网格中相对于旧网格的每个积分点（这假设新网格中的所有积分点都在旧网格的范围内：如果不是这样，会发出警告消息，新模型积分点处的解变量被设置为零）。然后将变量从旧网格中单元的节点插值到新网格中的位置。所有解变量都以这种方式自动插值，以便可以在新网格上继续分析。每当模型被映射时，由于网格的变化，解决方案中会出现一些不连续性是可以预期的。如果不连续性很显著，表明网格太粗，或者映射应该在较早阶段进行，在发生太多扭曲之前。

### 使用Abaqus/CAE提取二维轮廓和重网格化

模型使用Abaqus/CAE构建和网格划分。Abaqus/Standard分析的解映射通过从输出中提取变形坯料的二维轮廓来完成；用户必须在Abaqus/CAE主窗口底部的命令行界面中输入命令。要将变形几何从输出数据库提取为孤立网格部件，请使用命令`PartFromOdb`，它接受以下参数：

| *name* | 要创建的孤立网格部件的名称。 |
| --- | --- |
| *odb* | 从命令`openOdb`返回的输出数据库对象。 |
| *instance* | 初始模型中部件实例的名称（大写）。 |
| *shape* | 确定以其未变形还是变形形状导入部件。 |

命令`PartFromOdb`返回一个Part对象，传递给命令`Part2DGeomFrom2DMesh`。此命令从先前导入的孤立网格创建一个几何部件对象。它接受以下参数：

| *name* | 要创建的部件的名称。 |
| --- | --- |
| *part* | 从命令`PartFromOdb`返回的部件对象。 |
| *featureAngle* | 一个浮点数，指定触发几何中断的线段之间的角度（度）。 |

一旦创建了变形部件的轮廓，用户可以切换到网格模块，重网格化部件，并写出用于映射分析的新节点和单元定义。Python脚本文件[billet_rezone.py](../eif/billet_rezone.py)包含在内，以演示上述过程。

### Abaqus/Explicit中的自适应网格划分

自适应网格划分包括两个基本任务：创建新网格，以及使用称为对流的过程将解变量从旧网格重映射到新网格。为每个自适应网格域以指定频率创建新网格。通过在自适应网格域上迭代扫描并移动节点来平滑网格，从而找到网格。将解变量从旧网格映射到新网格的过程称为对流扫描。至少在一个自适应网格增量中执行一次对流扫描。用于将解变量对流到新网格的方法是一致的；单调的；（默认）二阶精确的；并保持质量、动量和能量守恒。本示例问题使用自适应网格域的默认设置。

### 结果和讨论

以下讨论主要关注案例1的结果，其中坯料使用CAX4R单元建模，刚性模具使用解析刚性表面建模，在Abaqus/Explicit中使用纯刚度沙漏控制。44%坯料镦粗（总模具位移的73.3%）时的变形网格如图1.3.1-2、图1.3.1-3和图1.3.1-4所示。坯料顶部外表面的折叠到模具上清晰可见。在Abaqus/Standard中（图1.3.1-2）可以看到试样中心的严重应变和单元扭曲。此时，Abaqus/Standard网格被映射。映射模型的新网格如图1.3.1-3所示。图1.3.1-4清楚地表明了自适应网格划分的好处，因为Abaqus/Explicit中使用的网格几乎没有扭曲。

60%坯料镦粗时的最终构型如图1.3.1-5和图1.3.1-6所示。Abaqus/Standard和Abaqus/Explicit结果比较良好，网格仅略微扭曲。同样，等效塑性应变大小也比较良好（图1.3.1-7和图1.3.1-8）。

图1.3.1-9是刚性表面参考节点处镦粗力与垂直位移的关系图。Abaqus/Standard和Abaqus/Explicit分析的结果都与Taylor（1981）获得的率无关结果表现出良好的一致性。值得注意的是，Abaqus/Standard中的映射似乎对总镦粗力没有显著影响。

图1.3.1-10是刚性表面参考节点处镦粗力与垂直位移的关系图，其中包含[表1.3.1-2](ch01s03aex32.md#table-upset-analopts)中识别的截面控制选项。使用CAX4R和CAX6M单元获得的曲线非常接近，并与Taylor（1981）获得的率无关结果表现出良好的一致性。COARSE_SS分析的结果与FINE分析的结果几乎相同，但成本大大降低；因此，建议对此问题使用此类分析选项。所有其他情况（使用默认截面控制但不同的刚性表面模型）的结果与使用默认截面控制的案例1的结果相同。

### 表格

**表1.3.1-1** 描述刚性模具建模的案例。

| 案例 | 描述 | STD | XPL |
| --- | --- | --- | --- |
| 1 | 模具使用平面解析表面和刚性体约束建模为解析刚性表面。刚性表面通过其指定的参考节点与刚性体关联。 | Yes | Yes |
| 2 | 使用轴对称刚性单元RAX2对刚性模具进行建模。 | Yes | Yes |
| 3 | 模具使用RAX2单元建模，如案例2。但是，通过在RAX2单元的节点上指定点质量来为模具分配质量。 | Yes | Yes |
| 4 | 刚性模具使用RAX2单元建模，如案例2。赋予刚性单元的厚度和密度值使得模具的质量与案例3中的相同。 | No | Yes |
| 5 | 模具使用RAX2单元建模，如案例2。在这种情况下，刚性单元的厚度从节点处指定的厚度插值。规定的厚度值与案例4相同。 | No | Yes |
| 6 | 使用轴对称壳单元SAX1对模具进行建模，并将其包含在刚性体定义中。 | Yes | Yes |
| 7 | 模具使用轴对称壳单元SAX1和轴对称刚性单元RAX2建模。可变形单元包含在刚性体定义中。两种单元类型的厚度和密度与案例4相同。 | No | Yes |
