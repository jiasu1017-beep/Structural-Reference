# 40.3.1 什么是绘图状态？





可视化模块提供了几种不同类型的绘图来查看您的模型和结果。这些绘图类型包括：
- [未变形形状](pt05ch43.md)
- [变形形状](pt05ch43.md)
- [云图](pt05ch44.md)
- [符号](pt05ch45.md)
- [材料方向](pt05ch46.md)
- [历史或X-Y数据](pt05ch47.md)
- [时间历史动画](pt05ch49s02hlb01.md)
- [比例因子动画](pt05ch49s02hlb02.md)
- [谐波动画](pt05ch49s02hlb03.md)

每种绘图都对应一个*绘图状态*。绘图状态很重要，因为可视化模块提供的某些自定义选项仅适用于特定的绘图状态。

- ["自定义您的绘图"，第40.3.3节](pt05ch40s03s03.md)
