# 58.2.3 评估表达式场







如果您为相互作用或规定条件指定了表达式场，Abaqus/CAE 必须首先评估表达式场，以确定为分析提交的值。表达式作为 Python 输入进行评估；包含错误的条目将生成标准 Python 错误。对于圆柱坐标系和球坐标系，**Th** 和 **P** 的值以弧度为单位进行评估，范围从 ![](../graphics/usi_eqn00792.gif) 到 ![](../graphics/usi_eqn00793.gif)。Abaqus/CAE 在写入输入文件时评估表达式场。

Abaqus/CAE 在模型上评估表达式场的位置取决于您在创建相互作用或规定条件时选择的区域类型。[表 58–2](pt06ch58s02s03.md#usi-fld-evaluate) 指示了 Abaqus/CAE 用于表达式场评估的模型上的位置。对于流体边界条件，Abaqus/CAE 对每个元素的节点上评估的表达式进行平均，并将该值应用于元素面的质心。

**表 58–2** 表达式场评估位置。
| 区域类型 | 表达式场评估位置 |
| --- | --- |
| 节点或顶点 | 在节点或顶点处 |
| 边缘 | 在每个元素边缘的中点 |
| 表面或面 | 区域中每个元素面的质心 |
| 在元素节点上（流体边界条件） |
| 单元 | 元素的质心 |

然后，Abaqus/CAE 将您为空间变化参数（如压力大小）指定的大小乘以在每个元素或节点处评估的表达式，以确定提交给分析的最终值。表达式场应用于相互作用或规定条件中的大小，包括复数的实部和虚部。不受表达式场影响的梁和壳梯度值（如温度梯度）。在分析过程中，Abaqus/CAE 应用您为相互作用或规定条件指定的任何振幅。
