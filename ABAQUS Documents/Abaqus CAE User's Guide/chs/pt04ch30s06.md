# 30.6 运行流体分析





当您提交与流体模型关联的作业进行分析时，Abaqus/CAE首先生成表示您模型的输入文件，然后Abaqus/CFD使用该文件的内容执行分析。随着分析的进行，Abaqus/CFD创建数据（`.dat`）文件和状态（`.sta`）文件。Abaqus/CFD使用在共享内存和分布式内存计算机上通过显式消息传递实现的基于域的并行性。更多信息，请参见["Abaqus/Standard、Abaqus/Explicit和Abaqus/CFD执行"，Abaqus Analysis User's Guide第3.2.2节](../usb/usb-link.md#usb-int-danalysisproc)和["并行执行：概述"，Abaqus Analysis User's Guide第3.5.1节](../usb/usb-link.md#usb-int-aparallelmodes)。
