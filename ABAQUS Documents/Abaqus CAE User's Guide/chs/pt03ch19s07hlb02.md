# 19.7.2 仅写入输入文件







默认情况下，当您提交与模型关联的作业进行分析时，Abaqus/CAE 会生成一个表示模型的输入文件，然后 Abaqus 分析该输入文件。但是，有时您可能希望生成输入文件，然后在执行分析之前查看或编辑它。

要在不立即执行分析的情况下写入输入文件，请从主菜单栏中选择****作业**![](../graphics/images/arrow.gif)**写入输入**![](../graphics/images/arrow.gif)***您选择的作业*****。输入文件 `*作业名称*.inp` 被写入您启动 Abaqus/CAE 的目录。您也可以通过选择您选择的作业，然后在**作业管理器**中点击**写入输入**来写入输入文件。

输入文件以 ASCII 格式编写，可以使用文本编辑器查看和编辑。如果您熟悉 Abaqus 关键字，您可以检查输入文件中的错误，并检查关键字、参数和数据是否按预期生成。您还可以修改输入文件的内容。例如，您可以更改材料属性或载荷的大小。

**警告：**如果您在 Abaqus/CAE 外部使用文本编辑器编辑模型的输入文件，然后在该模型的作业模块中提交作业，您对输入文件的更改将丢失。相反，您必须通过创建新作业并选择**输入文件**作为作业**源**来直接提交修改后的输入文件进行分析。

但是，如果您使用**关键字编辑器**修改为模型生成的关键字，这些修改将保留在模型中并应用于与该模型关联的任何作业。（您可以通过从主菜单栏中选择****模型**![](../graphics/images/arrow.gif)**编辑关键字****来显示**关键字编辑器**。）
