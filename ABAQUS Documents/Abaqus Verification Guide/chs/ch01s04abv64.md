# 1.4.14 刚性单元






**产品：**Abaqus/Standard  

### I. 二维刚性单元

### 问题描述

**模型：**

| 长度 | 1.0 |
| --- | --- |
| 厚度 | 0.1 |
| 旋转离心轴 | (1, 0, 0)，通过 (1, 1, 0) |

### 结果与讨论

计算的反作用力与施加的载荷一致。

### 输入文件

[erp2sxd1.inp](../eif/erp2sxd1.inp)

R2D2：CENT、BX、BY。

### II. 轴对称刚性单元

### 问题描述

**模型：**

| 连杆长度 | 10 |
| --- | --- |
| 连杆厚度 | 0.5 |
| 半径 | 5 |
| 旋转离心轴 | (0, 1, 0)，通过原点 |

**初始条件：**

| 静水压力基准 | 10.0 |
| --- | --- |
| 静水压力高度 | 0.0 |

### 结果与讨论

计算的反作用力与施加的载荷一致。

### 输入文件

[era2sxd1.inp](../eif/era2sxd1.inp)

RAX2：CENT、BR、BZ、P、HP。

[era2sxdi.inp](../eif/era2sxdi.inp)

RAX2：P、HP。
