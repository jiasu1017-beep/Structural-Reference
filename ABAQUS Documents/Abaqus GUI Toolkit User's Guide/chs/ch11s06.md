# 11.7 小结





- Abaqus 模拟可以包含任意数量的步骤。
- 隐式和显式步骤不允许出现在同一个分析作业中。
- 分析步骤是计算模型对指定荷载和边界条件响应的一段"时间"期间。该响应的特性由步骤期间使用的特定分析过程决定。
- 一般分析步骤中结构的响应可以是线性的或非线性的。
- 每个一般步骤的起始条件是前一个一般步骤的结束条件。因此，模型的响应在模拟的一系列一般步骤中逐步发展。
- 线性扰动步骤（仅在 Abaqus/Standard 中可用）计算结构对扰动荷载的线性响应。响应相对于由最后一个一般步骤结束时的模型条件定义的基态进行报告。
- 在一般步骤中，任何荷载选项（如 [*BOUNDARY](../key/key-link.md#usb-kws-hboundary)、[*CLOAD](../key/key-link.md#usb-kws-hcload) 和 [*DLOAD](../key/key-link.md#usb-kws-hdload)）上的 OP 参数控制这些选项指定的值与先前步骤中定义的值如何相互作用。
- 只要保存了重启文件，就可以重新启动分析。重启文件可用于继续被中断的分析，或向模拟添加额外的荷载历史。
