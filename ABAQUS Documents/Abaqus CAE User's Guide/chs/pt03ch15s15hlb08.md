# 15.15.8 定义嵌入区域约束





您使用嵌入区域约束将模型的区域嵌入到模型的"宿主"区域或整个模型中。您可以通过指定嵌入区域、宿主区域、权重因子舍入容差以及绝对外部容差或分数外部容差来创建嵌入区域约束。有关更多信息，请参见Abaqus Analysis User's Guide第35.4.1节的"嵌入元素"。

**创建嵌入区域约束的步骤：**

1. 从主菜单栏选择 ****Constraint**![](../graphics/images/arrow.gif)**Create****。**提示：**您也可以使用「相互作用」模块工具箱中的![](../graphics/ico_constraint.png)工具创建嵌入区域约束。
2. 在出现的「Create Constraint」对话框中，执行以下操作：1. 为约束命名。有关命名对象的更多信息，请参见"使用基本对话框组件"，第3.2.1节。2. 从「Type」列表中选择「Embedded region」，然后点击「Continue」。
3. 使用以下方法之一选择嵌入区域：- 在视口中选择一个区域。（有关更多信息，请参见"在当前视口中选择对象"，第6.2节。）点击鼠标按钮2表示您已完成选择。如果模型包含网格和几何的组合，请从提示区域点击以下之一：- 如果您想选择几何区域，请点击「Geometry」。- 如果您想从原生网格或孤立网格选择中选择区域，请点击「Mesh」。- 使用现有集合定义区域。点击提示区域右侧的「Sets」。从出现的「Region Selection」对话框中选择现有集合，然后点击「Continue」。**注意：**默认选择方法基于您最近使用的选择方法。要恢复为另一种方法，请点击提示区域右侧的「Select in Viewport」或「Sets」。您选择的区域在视口中以红色高亮显示。
4. 选择宿主区域。在提示区域中，选择以下之一：- **Select Region**如果您想在视口中选择区域或使用现有集合定义区域。使用与上一步所述相同的方法选择宿主区域。您选择的区域在视口中以品红色高亮显示。- **Whole Model**如果您想将嵌入区域嵌入到整个模型中。约束编辑器会出现。
5. 在编辑器中，指定容差参数的值。如果您为两个外部容差参数都指定了值，Abaqus将使用两个容差中较小的值。- **Weight factor roundoff tolerance**。您可以指定一个小于该值的权重因子将被清零。默认值为10^(-6)。- **Absolute exterior tolerance**。您可以指定嵌入区域上的节点可能位于宿主区域外部的绝对值。如果此参数被省略或值为0.0，则适用「Fractional exterior tolerance」。- **Fractional exterior tolerance**。您可以指定嵌入区域上的节点可能位于宿主区域外部的分数值。分数值基于宿主区域内的平均元素尺寸。默认值为0.05。
6. 点击「OK」保存约束定义并关闭编辑器。
