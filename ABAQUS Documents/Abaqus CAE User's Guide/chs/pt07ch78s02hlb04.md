# 78.2.4 绘制显示组





您可以为当前视口中的模型绘制显示组。在可视化模块中，您可以在同一视口中绘制多个显示组。Abaqus/CAE仅绘制每个显示组中对当前模型有效的组件。例如，如果显示组引用名为`Fixture`的单元集，则当前视口中的模型也应包含名为`Fixture`的单元集。Abaqus/CAE忽略无效的模型组件。

**在任何模块中绘制单个显示组的步骤：**

1. 从主菜单栏中选择****工具**![](../graphics/images/arrow.gif)**显示组**![](../graphics/images/arrow.gif)**绘制****。
2. 从出现的菜单中选择要绘制的显示组。**提示：**您也可以使用当前模块的显示组管理器来绘制显示组。从主菜单栏中选择****工具**![](../graphics/images/arrow.gif)**显示组**![](../graphics/images/arrow.gif)**管理器****。选择要绘制的显示组，然后从管理器右侧的按钮中点击**绘制**。当前视口中的模型绘图会更改以仅显示所选显示组。

**在可视化模块中绘制一个或多个显示组的步骤：**

1. 从主菜单栏中选择****工具**![](../graphics/images/arrow.gif)**显示组**![](../graphics/images/arrow.gif)**管理器****。**ODB显示组管理器**出现，在其顶部列出当前会话中所有保存的显示组。
2. 从列表中选择一个或多个显示组（有关更多信息，请参见["从列表和表中选择多个项目，"第3.2.11节](pt01ch03s02s11.md)），然后从管理器右侧的按钮中点击**绘制**。当前视口中的模型绘图会更改以仅显示所选显示组，并且所绘制的显示组出现在**ODB显示组管理器**底部显示组实例列表中。
3. 要将一个或多个显示组添加到当前视口的绘图中，请从**ODB显示组管理器**顶部保存的显示组列表中选择它们，然后从管理器底部的按钮中点击**添加**。所选显示组会添加到当前视口中的模型绘图中，并出现在**ODB显示组管理器**底部的显示组实例列表中。
4. 要从当前视口的绘图中移除显示组，请从**ODB显示组管理器**底部的列表中选择实例，然后从管理器底部的按钮中点击**移除**。所选显示组会从当前视口中的模型绘图中移除，并从显示组实例列表中移除。
5. 您可以分别自定义每个显示组的与绘图状态相关和无关的选项。有关更多信息，请参见["在可视化模块中自定义显示组，"第78.2.5节](pt07ch78s02hlb05.md)。
