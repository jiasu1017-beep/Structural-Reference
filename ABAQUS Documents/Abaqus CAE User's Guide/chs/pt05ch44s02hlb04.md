# 44.5.4 使用刻度线云图绘制线形单元





在线形单元上绘制的云图线难以清晰可视化。为了更好地查看线形单元（梁、一维单元、垫片链接单元、三维线垫片单元和二维接触表面）的云图，您可以选择使用刻度线云图显示云图线。

刻度线云图显示绘制在单元法线两侧线条之间的云图曲线。云图级别由这些法线上的"刻度线"表示。Abaqus仅在云图曲线落在您指定的限制范围内时显示它。有关限制绘图范围的更多信息，请参见["设置云图限制，" 第44.5.7节](pt05ch44s02hlb07.md)。

[图44-7](pt05ch44s02hlb04.md#usv-cont-tickmark)显示了刻度线云图的外观。

**图44-7** 刻度线云图。

![](../graphics/usv-cont-tickmark.png)

法线或刻度线轴绘制在节点处。在零件边界处，绘制两条刻度线轴。在零件内部，如果两个相邻线单元之间的角度小于指定特征角，则在与共享节点的单元的平均法线方向绘制单条刻度线轴。否则，为共享节点的每个单元绘制一条刻度线轴。当平均法线时，计算中仅考虑相同类型的显示单元。

**要显示刻度线云图：**

1. 找到**Contour Type**选项。从主菜单栏中选择****Options**![](../graphics/images/arrow.gif)**Contour****，或单击工具箱中的![](../graphics/ico_plotContourOptions.png)；然后在出现的对话框中单击**Basic**选项卡。**Contour Type**选项位于页面左上角。
2. 切换**Show tick marks for line elements**。
3. 单击**Apply**实施您的更改。当前视口中的云图更改为显示模型中线单元的刻度线图。默认情况下，您的更改在会话期间保存，并将影响所有后续云图。如果您想在后续会话中保留更改，请将它们保存到文件。有关更多信息，请参见["保存自定义设置以供后续会话使用，" 第55.1.1节](pt05ch55s01s01.md)。
