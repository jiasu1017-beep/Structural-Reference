# 19.6.8 可视化重启分析结果






每次重启分析都会创建一个新的输出数据库。考虑[「在向模型添加更多分析步骤后重启，第 19.6.5 节」](pt03ch19s06s05.md) 中描述的示例，其中您执行了以下操作：
- 运行了涉及包含 `Step-1` 和 `Step-2` 的 `Model-A` 的 `Job-A` 的分析。因此，分析生成的输出数据库 `Job-A.odb` 包含来自 `Step-1` 和 `Step-2` 的结果。
- 运行了涉及 `Model-A-restart` 的 `Job-A-restart` 的重启分析。虽然 `Model-A-restart` 包含 `Step-1`、`Step-2` 和 `Step-3`，但重启分析生成的输出数据库 `Job-A-restart.odb` 仅包含来自 `Step-3` 的结果。

您可以使用可视化模块创建场数据的变形图、轮廓图和符号图；但是，您一次只能绘制一个输出数据库的结果。如果您正在查看 `Step-1` 或 `Step-2` 的结果并想查看 `Step-3` 的结果，则必须更改输出数据库。

如果您想创建所有三个步骤结果的动画，必须合并两个输出数据库。Abaqus 为此目的提供了执行过程。有关更多信息，请参阅 [Abaqus Analysis User's Guide 第 3.2.21 节「连接重启分析的输出数据库（`.odb`）文件」](../usb/usb-link.md#usb-int-drestartjoinproc)。

或者，您可以使用以下技术创建跨所有三个步骤的变量历史图：
- 从原始输出数据库 `Job-A.odb` 为变量创建一个 *X-Y* 数据对象。
- 从重启分析生成的输出数据库 `Job-A-restart.odb` 创建第二个 *X-Y* 数据对象。
- 创建两个 *X-Y* 数据对象的 *X-Y* 图。或者，您可以使用**对 XY 数据进行运算**选项创建新的 *X-Y* 数据对象，附加前两个 *X-Y* 数据对象，并创建结果的 *X-Y* 图。

有关更多信息，请参阅[第 47 章，「X-Y 绘图」](pt05ch47.md)。

您应该注意确保您的 *X-Y* 数据对象不引用相同的步骤。例如，考虑[「在修改现有分析步骤后重启，第 19.6.6 节」](pt03ch19s06s06.md) 中描述的重启分析，其中您修改了现有步骤并在继续分析之前添加了新步骤。第一个输出数据库包含来自 `Step-1` 和 `Step-2` 的数据。第二个输出数据库包含来自修改后的 `Step-2` 和新 `Step-3` 的数据。您创建的第一个 *X-Y* 数据对象应该仅使用来自 `Step-1` 的数据，而不包括来自原始 `Step-2` 的数据。第二个 *X-Y* 数据对象应该使用来自修改后的 `Step-2` 和新 `Step-3` 的数据。
