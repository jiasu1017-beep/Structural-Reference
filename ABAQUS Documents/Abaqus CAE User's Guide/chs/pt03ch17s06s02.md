# 17.6.2 查询网格





网格模块中的查询工具集允许您获取有关网格中节点和单元的信息。此外，您可以从主菜单栏选择****工具**![](../graphics/images/arrow.gif)**查询****来请求有关网格的以下信息：
- 所选零件、零件实例或区域中的节点和单元总数，以及每种单元形状的单元数量。
- 所选单元的类型和连通性。
- 壳和膜面的正侧和负侧。
- 梁和桁架切线的方向。
- 网格栈方向。
- 边界面是否有任何边存在不兼容的界面、裂纹或间隙，以及是否有任何边与其他面相交。
- 自由边或非流形边的位置——未恰好被两个外部单元共享的外部壳或实体单元边。
- 任何未网格化区域的位置。

有关使用查询工具集的详细信息，请参见["获取网格信息，"第17.19.2节](pt03ch17s19hlb02.md)。
