# 76.1 几何和网格显示选项概述






根据您正在使用的模块，Abaqus/CAE 会为您提供以下菜单选项来自定义几何和网格显示：

******视图**![](../graphics/images/arrow.gif)**零件显示选项******

在几何相关模块中，您可以使用此菜单项来控制几何渲染样式、边缘可见性、曲线细化、单元和节点标签以及各种类型的基准几何的可见性。

******视图**![](../graphics/images/arrow.gif)**装配显示选项******

在装配相关模块中，您可以使用此菜单项来控制装配渲染样式、边缘可见性以及各种类型的基准几何的可见性。您还可以使用它来控制在这些相同的装配相关模块中的网格、单元和节点标签、零件实例、载荷、边界条件和预定义字段的显示。

******视图**![](../graphics/images/arrow.gif)**ODB显示选项******

此菜单项仅适用于可视化模块。有关更多信息，请参见[第55章，"自定义绘图显示"](pt05ch55.md)。

您可以通过从主菜单栏选择 ****文件**![](../graphics/images/arrow.gif)**保存选项**** 来保存当前显示选项并将其应用到未来的 Abaqus/CAE 会话中。有关更多信息，请参见["保存显示选项设置，" 第76.16节](pt07ch76hla15.md)。
