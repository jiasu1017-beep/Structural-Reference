# 17.11.1 什么是自底向上网格？





自底向上网格是一种手动、增量网格划分过程，允许您在任何实体区域中构建六面体网格。结构化、扫掠和自由网格技术都以自顶向下的方式工作——它们直接绑定到几何体，使得生成的网格填充几何体。自底向上网格放宽了将网格绑定到几何体的约束，这样您可以构建忽略某些几何特征的网格。您也可以使用自底向上网格技术修改网格零件，在这种情况下，不需要考虑几何特征。如果您使用几何体工作，您在自底向上网格中创建的单元始终与您网格化的实体区域关联，但网格边界可能不会与区域的几何边界关联。这允许您创建仅使用六面体单元的网格，而自顶向下网格技术可能需要大量分割或使用四面体单元来完成网格。然而，放宽的几何约束也意味着您必须仔细选择用于创建网格的参数，因为它可能与几何体有很大差异。一旦Abaqus生成了自底向上网格，您需要评估它是否适合分析，如果存在几何体，还要验证网格是否正确关联。

您可以将自底向上网格应用于任何实体区域，包括可以使用自顶向下技术进行网格划分的区域，以及没有任何关联几何体的网格。由于自底向上网格是一个可能非常耗时的手动过程，建议您仅在自顶向下方法无法产生满意网格时才使用此方法。

生成所需的网格可能需要多次应用自底向上网格技术。如果需要多次应用，每个自底向上网格都会成为下一个网格的增量构建块，直到您完成该区域的网格。每次应用自底向上网格技术都涉及四个阶段：

1. 您选择Abaqus/CAE将创建网格的域。您可以选择三维几何区域或孤立单元。
2. 您选择Abaqus/CAE将用于创建网格的方法。您可以从以下方法中选择：
   - 扫掠
   - 挤出
   - 旋转
   - 偏移（仅适用于孤立单元选择）
3. 您选择Abaqus/CAE将用于创建二维网格的一侧，称为源侧，该网格将被扫掠、挤出或旋转以填充三维区域。
4. 您选择剩余的参数以完成自底向上网格的定义。例如，如果您选择了扫掠方法，您可以选择连接边和目标侧。

此外，您可能需要编辑自底向上网格或将它们与几何体关联，然后才能使用它们继续生成区域网格。更多信息，请参见[第64章，"编辑网格"工具集](pt06ch64.md)。

您可以从扫掠、挤出或旋转方法中选择来对选定的几何区域进行网格划分，如果您使用孤立单元工作，也可以选择偏移方法。根据区域的形状和分析意图，多种方法可能产生可接受的结果。您选择的方法及相应参数将决定网格与几何体的符合程度（如果适用）。例如，图17-96展示了与图17-95相同的零件；这次使用挤出方法对零件进行网格划分。挤出方法的参数是源侧、挤出向量和层数。如图所示，挤出的网格没有捕获模型后部的锥形边和圆角。

**图17-96** 使用挤出方法的自底向上网格。

![](../graphics/mgn-bottomup-extrude-nls.png)

如果这些细节不重要，此简化网格可能是可接受的。或者，您可以通过修改或移除超出原始几何体的节点和单元来编辑生成的网格。有关编辑网格的更多信息，请参见[第64章，"编辑网格"工具集](pt06ch64.md)。

如果生成的自底向上网格不可接受，您可以删除它并尝试相同区域的不同自底向上方法，或分割区域并对生成的更简单的区域进行网格划分。您也可以使用网格**撤销**和**重做**功能来撤销自底向上网格划分过程中的步骤，而不是删除该区域的整个自底向上网格并重新开始。

**注意：**某些操作后无法使用撤销和重做，例如删除区域网格。

撤销和重做功能与"编辑网格"工具集使用的功能相同（更多信息，请参见["在编辑网格工具集中撤销或重做更改，"第64.9节](pt06ch64s09.md))。
