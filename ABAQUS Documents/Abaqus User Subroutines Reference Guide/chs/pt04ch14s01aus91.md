### 概述

在欧拉分析中，您可以在欧拉边界上定义独立的流入和流出条件。欧拉边界条件：
- 可用于控制流入欧拉域的材料流动；
- 可用于定义欧拉域边界处的压力场；
- 可用于在截断的人工边界处施加非反射边界条件以模拟无限域；和
- 与在欧拉网格边界上发生流入或流出的表面相关联。

### 定义欧拉边界

欧拉边界必须在欧拉网格边界处的表面上定义。您不能在相同表面处定义多个欧拉边界。

| **输入文件用法：** | ``` [*EULERIAN BOUNDARY](../key/key-link.md#usb-kws-heulerianboundary) *surface name* ``` |
| --- | --- |

| **Abaqus/CAE 用法：** | 载荷模块：**创建边界条件**：**类别：** **其他**，**所选步骤的类型：** **欧拉边界**：选择区域 |
| --- | --- |

### 定义流入条件

您可以使用流入条件来控制流入欧拉域的材料流动。

#### 自由流入

如果没有定义欧拉边界，材料可以自由流入欧拉域；每个流入材料的内容和状态与当前存在于单元内的相同。如果定义了欧拉边界，自由流入是默认的流入条件。

| **输入文件用法：** | ``` [*EULERIAN BOUNDARY](../key/key-link.md#usb-kws-heulerianboundary), INFLOW=FREE ``` |
| --- | --- |

| **Abaqus/CAE 用法：** | 欧拉边界条件编辑器：**流动类型：** **流入**，**流入：** **自由** |
| --- | --- |

#### 无流入

您可以指定不能发生流入的欧拉边界——没有材料或空隙可以通过指定边界流入欧拉域。如果速度在边界处向内指向，则速度的法向分量设置为零，而速度的切向分量保持不变。

| **输入文件用法：** | ``` [*EULERIAN BOUNDARY](../key/key-link.md#usb-kws-heulerianboundary), INFLOW=NONE ``` |
| --- | --- |

| **Abaqus/CAE 用法：** | 欧拉边界条件编辑器：**流动类型：** **流入**，**流入：** **无** |
| --- | --- |

#### 空隙流入

您还可以指定可以发生流入但流入体积仅包含空隙的边界。由于空隙的流入，最初完全充满材料的欧拉域可能在分析期间变得部分充满。

| **输入文件用法：** | ``` [*EULERIAN BOUNDARY](../key/key-link.md#usb-kws-heulerianboundary), INFLOW=VOID ``` |
| --- | --- |

| **Abaqus/CAE 用法：** | 欧拉边界条件编辑器：**流动类型：** **流入**，**流入：** **空隙** |
| --- | --- |

### 定义流出条件

流出条件可用于通过减少流出边界的反射来模拟无限域，或在边界处规定压力场。

#### 自由流出

如果没有指定欧拉边界条件，材料可以自由流出欧拉域；每个流出材料的内容和状态与当前存在于单元内的相同。如果定义了欧拉边界条件，并且如果在相同表面处指定了空隙流入条件，则自由流出是默认行为。

| **输入文件用法：** | ``` [*EULERIAN BOUNDARY](../key/key-link.md#usb-kws-heulerianboundary), OUTFLOW=FREE ``` |
| --- | --- |

| **Abaqus/CAE 用法：** | 欧拉边界条件编辑器：**流动类型：** **流出**，**流出：** **自由** |
| --- | --- |

#### 非反射流出

非反射流出条件可用于在无限域中定义的边界值问题，或与周围介质相比感兴趣区域很小的问题。像在 ["在动态分析中使用固体介质无限单元" 在 "无限单元，" 第 28.3.1 节"](pt06ch28s03alm03.md#usb-elm-einfinite-solid) 中描述的无限单元公式一样，非反射流出条件在域边界上引入附加的法向和切向牵引力，这些力与边界速度的法向和切向分量成正比。选择这些边界阻尼常数以最小化膨胀波和剪切波能量反射回有限元网格。这种条件不能提供完美的能量透射出网格，除非在平面体波在各项同性介质中正交入射到边界的情况下。然而，它通常为大多数实际情况提供可接受的建模。例外情况是当显著的材料传输通过边界发生时，这种条件不适合使用。

| **输入文件用法：** | ``` [*EULERIAN BOUNDARY](../key/key-link.md#usb-kws-heulerianboundary), OUTFLOW=NONREFLECTING ``` |
| --- | --- |

| **Abaqus/CAE 用法：** | 欧拉边界条件编辑器：**流动类型：** **流出**，**流出：** **非反射** |
| --- | --- |

#### 平衡流出

平衡流出是另一种流出条件，可以有效地减少无限域中人工流出边界的虚假反射。假定应力在边界上单元面上是零阶连续的。牵引力施加到这些单元面上以平衡由边界单元中应力产生的节点力。这种条件通常施加在压力分布未知的流出边界处。

| **输入文件用法：** | ``` [*EULERIAN BOUNDARY](../key/key-link.md#usb-kws-heulerianboundary), OUTFLOW=NONUNIFORM PRESSURE ``` |
| --- | --- |

| **Abaqus/CAE 用法：** | 欧拉边界条件编辑器：**流动类型：** **流出**，**流出：** **平衡** |
| --- | --- |

#### 零压力流出

在流动问题中，通常在流动出口处规定零压力。由于边界上的法向牵引力包含压力和剪切应力的贡献，如果还考虑流动的剪切行为，则自然边界条件（也称为"无作用条件"）不足以提供这种条件。零压力流出条件施加抵消剪切贡献的牵引力，从而在边界上产生均匀分布的压力场。您可以在相同边界上施加分布表面载荷（见 ["分布式载荷中的表面牵引力和压力载荷，" 第 34.4.3 节"](pt07ch34s04aus122.md#usb-prc-ploaddistributed-surfload)）来规定非零压力。如果未指定流入条件，这是默认的流出条件。

| **输入文件用法：** | ``` [*EULERIAN BOUNDARY](../key/key-link.md#usb-kws-heulerianboundary), OUTFLOW=ZERO PRESSURE ``` |
| --- | --- |

| **Abaqus/CAE 用法：** | 欧拉边界条件编辑器：**流动类型：** **流出**，**流出：** **零压力** |
| --- | --- |

### 在重启分析中使用欧拉边界

您可以在重启分析中定义新的欧拉边界，但不能在此边界处指定空隙流入条件。此外，在重启分析中，您不能将现有欧拉边界上的流入条件更改为空隙流入条件。
