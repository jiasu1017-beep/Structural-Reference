# 42.6.6 控制结果平均

您可以控制基于单元的场输出结果（如应力或应变）的平均。平均适用于创建线和带状等值线、节点位置探测、基于结果值形成显示组和颜色编码，或沿路径提取*X–Y*数据。

首先将单元值外推到节点。共同属于两个或多个单元的节点将获得多个贡献。要控制多次贡献的平均，您可以：
- 启用或抑制平均。
- 定义结果区域，并选择是否跨区域边界进行平均。
- 将壳和膜特征边缘作为额外的区域边界进行平均或排除。
- 抑制未显示单元的平均。
- 选择Abaqus/CAE是在平均之前还是之后计算不变量。
- 设置平均阈值。

平均阈值控制平均的范围；当Abaqus/CAE在平均之前计算不变量或提取分量时，此选项可用。如果节点处贡献之间的相对差异大于您设置的阈值百分比，Abaqus/CAE将不平均贡献值，并且结果在该节点处将显示为间断。使用较高的百分比可以产生更平滑、更连续的效果。

有关跨区域平均的信息，请参阅["控制区域边界处的计算，" 第42.6.7节](pt05ch42s04hlb03.md)。

**控制结果平均：**

1. 找到"平均"选项。从主菜单栏中，选择****Result**![](../graphics/images/arrow.gif)**Options****；然后在出现的对话框中单击**Computation**标签。将出现"平均"选项。
2. 要控制Abaqus/CAE是否在共享节点处平均来自多个单元的输出，请切换**在节点处平均单元输出**。抑制平均可以识别场输出中的间断，或基于外推的未平均结果建立等值线限制。
3. 切换关闭**使用区域边界**以忽略区域边界并在整个模型中平均结果。
4. 如果使用区域边界，请指定结果区域以设置边界。有关详细信息，请参阅["控制区域边界处的计算，" 第42.6.7节](pt05ch42s04hlb03.md)。**提示：**要查看当前结果区域，请使用**平均区域**属性选择方法对未变形或变形图进行颜色编码。（有关详细信息，请参阅["在可视化模块中对所有几何进行颜色编码，" 第77.5节](pt07ch77s05.md)。）
5. 如果使用区域边界，默认情况下Abaqus/CAE会将壳和膜特征边缘作为额外的区域边界。要更改用于创建特征边缘的最小角度，请单击![](../graphics/ico_edit.png)并在"ODB Display Options"对话框中更改**Feature Angle**设置。特征边缘对应于未变形模型，不会因模型变形而重新计算。（有关详细信息，请参阅["定义模型特征边缘，" 第55.3.2节](pt05ch55s03hlb02.md)。）要跨特征边缘平均结果，请切换关闭**包括壳/膜特征边缘边界**。
6. 要控制Abaqus/CAE是基于所有贡献单元还是仅基于当前显示组中的单元对节点结果进行平均，请切换**仅平均显示的单元**。
7. 选择Abaqus是计算不变量（标量）并在平均之前还是之后提取分量。默认情况下，Abaqus在平均之前计算标量；如果您选择在平均之后计算结果，Abaqus会平均所有结果及其方向以保持用于计算不变量的有效基础。**提示：**要在云图中查看节点平均方向，您可以，使用"Contour Plot Options"对话框显示其标签。有关的详细信息，请参阅["显示节点平均方向，" 第55.5.6节](pt05ch55s05hlb06.md)。
8. 如果在平均之前计算标量，您可以设置**平均阈值(%)**来控制相邻单元之间的平均。值为0会抑制所有平均；值为100会平均所有结果。
9. 单击**Apply**以根据您的规格实现更改。Abaqus会根据您的规格在当前视口中计算用于显示的结果值。如果图例处于活动状态，等值线图例会更改以说明您指定的平均阈值。有关等值线图例的详细信息，请参阅["自定义图例，" 第56.1节](pt05ch56s01.md)。默认情况下，您的更改会在会话期间保存，并会影响后续的所有结果显示。如果您希望将更改保留到后续会话中，请将其保存到文件。有关详细信息，请参阅["保存自定义设置以供后续会话使用，" 第55.1.1节](pt05ch55s01s01.md)。
