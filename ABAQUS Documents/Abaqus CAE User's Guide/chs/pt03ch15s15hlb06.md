# 15.15.6 定义MPC约束





您使用MPC约束来约束区域从节点的运动，使其受到一个点的运动的约束。您可以通过指定一个控制点和一个由节点、边缘和表面组成的区域来创建MPC约束。有关多点约束的详细信息，请参见Abaqus Analysis User's Guide第35.2.2节的"通用多点约束"。

**创建MPC约束的步骤：**

1. 从主菜单栏选择 ****Constraint**![](../graphics/images/arrow.gif)**Create****。**提示：**您也可以使用「相互作用」模块工具箱中的![](../graphics/ico_constraint.png)工具创建多点约束。
2. 在出现的「Create Constraint」对话框中，执行以下操作：1. 为约束命名。有关命名对象的更多信息，请参见"使用基本对话框组件"，第3.2.1节。2. 从「Type」列表中选择「MPC Constraint」，然后点击「Continue」。
3. 使用以下方法之一选择一个点来定义约束控制点：- 使用鼠标在视口中选择一个点。**提示：**「Selection」工具栏中的![](../graphics/ico_filterVisible.png)「Select the Entity Closest to the Screen」工具默认关闭。如果您进行模糊选择，Abaqus/CAE会高亮显示该点，并在视口左下角显示该点的描述。使用「Next」和「Previous」按钮循环遍历可能的选择，然后点击「OK」确认您的选择。（有关更多信息，请参见"在当前视口中选择对象"，第6.2节。）点击鼠标按钮2表示您已完成选择。如果模型包含网格和几何的组合，请从提示区域点击以下之一：- 如果您想从几何区域选择控制点，请点击「Geometry」。- 如果您想从原生网格或孤立网格选择中选择控制点，请点击「Mesh」。- 使用现有集合定义区域。点击提示区域右侧的「Sets」。从出现的「Region Selection」对话框中选择现有集合，然后点击「Continue」。**注意：**默认选择方法基于您最近使用的选择方法。要恢复为另一种方法，请点击提示区域右侧的「Select in Viewport」或「Sets」。您选择的点在视口中以红色高亮显示。
4. 选择从节点所在的区域。使用以下方法之一选择区域：- 使用鼠标在视口中选择一个区域。（有关更多信息，请参见"在当前视口中选择对象"，第6.2节。）您选择的区域可以跨越多个部件实例。点击鼠标按钮2表示您已完成选择。如果模型包含网格和几何的组合，请从提示区域点击以下之一：- 如果您想选择几何区域，请点击「Geometry」。- 如果您想从原生网格或孤立网格选择中选择区域，请点击「Mesh」。您可以使用角度方法从网格中选择一组节点。有关更多信息，请参见"使用角度和特征边缘方法选择多个对象"，第6.2.3节。- 使用现有集合定义区域。点击提示区域右侧的「Sets」。从出现的「Region Selection」对话框中选择现有集合，然后点击「Continue」。**注意：**默认选择方法基于您最近使用的选择方法。要恢复为另一种方法，请点击提示区域右侧的「Select in Viewport」或「Sets」。您选择的区域在视口中以品红色高亮显示。约束编辑器会出现。
5. 从编辑器中，选择「MPC type」。- 选择「Beam」以定义刚性梁连接，约束每个从节点的位移和旋转到控制点的位移和旋转。- 选择「Tie」以使每个从节点和控制点处的所有活动自由度相等。- 选择「Link」以在每个从节点和控制点之间定义一个铰接刚性链接。- 选择「Pin」以在每个从节点和控制点之间定义一个铰接 joint。- 选择「Elbow」以将ELBOW31或ELBOW32元素的节点约束在一起（请参见Abaqus Analysis User's Guide第29.5.1节的"具有变形横截面的管道和弯管：弯管元素"）。- 选择「User-defined」以在用户子程序[`MPC`](../sub/sub-link.md#sub-xsl-mpc)中定义多点约束（用于Abaqus/Standard）。请参阅以下章节获取更多信息：- "指定常规作业设置"，第19.8.6节- "MPC"，Abaqus User Subroutines Reference Guide第1.1.14节
6. 如果您选择了用户定义的MPC类型，请执行以下操作：1. 选择用户子程序[`MPC`](../sub/sub-link.md#sub-xsl-mpc)的编码模式。- 如果您希望每次调用用户子程序约束一个单独的自由度，请选择「DOF-by-DOF」。- 如果您希望每次调用用户子程序立即施加一组约束，请选择「Node-by-Node」。2. 在「Constraint type」字段中，输入一个整数值，用于用户子程序中以区分不同的约束类型。默认值为0。
7. 如果您想更改耦合约束的坐标系（**CSYS**），请点击![](../graphics/ico_selectBlue.png)，并使用以下方法之一：- 按名称选择预定义的基准坐标系。1. 从提示区域点击「Datum CSYS List」以显示基准坐标系列表。2. 从列表中选择一个名称，然后点击「OK」。- 在视口中选择预定义的坐标系。**提示：**「Selection」工具栏中的![](../graphics/ico_filterVisible.png)工具默认关闭。对于原点重合的坐标系，当您循环遍历所有可能的选择时，Abaqus/CAE会高亮显示坐标系并在视口中显示坐标系的描述。- 从提示区域点击「Use Global CSYS」以使用全局坐标系。
8. 点击「OK」保存约束定义并关闭编辑器。
