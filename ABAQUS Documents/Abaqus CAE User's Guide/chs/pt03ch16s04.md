# 16.4 创建和修改规定条件







要创建载荷、边界条件或预定义场，请从主菜单栏上的相应菜单中选择**创建**。将出现**创建**对话框，您可以在其中提供规定条件的名称并选择要创建的规定条件的类型。

当您在**创建**对话框中单击**继续**时，系统会提示您选择要应用规定条件的区域，除非规定条件应用于整个模型。您只能将连接器载荷和连接器边界条件（位移、速度和加速度）应用于与连接器截面分配关联的线。如果选择多条线，则分配给这些线的连接器截面必须具有您要定义载荷和边界条件的相对运动可用分量。您只能将连接器材料流边界条件应用于与连接器截面分配关联的线的端点。选择区域后，将出现一个编辑器，您可以在其中指定关于规定条件的其他信息，如其大小。

每个规定条件编辑器的顶部面板显示规定条件的名称和类型、您当前所在的分析步骤以及将应用规定条件的模型区域。如果您在首次创庺规定条件的步骤中编辑规定条件，则**区域**字段旁边会出现**编辑区域**（![](../graphics/ico_selectBlue.png)）按钮；此按钮允许您编辑应用规定条件的区域。如果编辑区域需要完全重新定义规定条件（例如，如果规定条件应用于整个模型或引用原始选定区域内的子区域），则**编辑区域**按钮不会出现。更多信息，请参见"编辑应用规定条件的区域"，第16.8.4节。

编辑器其余部分的格式取决于您正在定义的规定条件类型以及编辑器顶部指定的步骤。例如，集中力的编辑器如图16-2所示。

**图16-2** 集中力编辑器。

![](../graphics/lbi-forceeditor-nls.png)

此编辑器包含特殊文本字段，您可以在其中指定1、2和3方向上的力分量。编辑器还包含一个**幅值**文本字段，允许您将规定条件的大小作为时间函数进行变化。您可以接受默认幅值，选择使用幅值工具集定义的幅值，或单击![](../graphics/ico_amplitudeSmall.png)定义新幅值。（更多信息，请参见第57章"幅值工具集"。）

您可以指定将应用以下载荷或边界条件的坐标系：

**载荷**

- 集中力
- 力矩
- 一般和剪切表面牵引
- 一般壳边缘载荷
- 惯性 relief
- 电流密度

**边界条件**

- 对称/反对称/固定
- 位移/旋转
- 速度/角速度
- 加速度/旋转加速度
- 欧拉网格运动
- 磁矢量势

所有其他规定条件使用全局坐标系，压力除外，压力施加于选定表面的法向。

如果载荷或边界条件允许您指定坐标系，您可以选择现有基准坐标系或接受全局坐标系。如果所需的基准坐标系不存在，您可以使用基准工具集创建它。（更多信息，请参见"创建基准坐标系"，第62.9节。）或者，您可以引用定义坐标系的Abaqus/Standard用户子程序（请参见Abaqus User Subroutines Reference Guide的第1.1.15节"ORIENT"）。

**注意：**如果您删除或抑制基准坐标系，则载荷或边界条件的方向将恢复为全局坐标系。

创建和修改预定义场的规则因预定义场类型而异：
- 某些预定义场要求您仅指定初始条件。您只能在初始步骤中创建和编辑此类型的预定义场。Abaqus计算预定义场在分析进行时的后续值。此类型的预定义场是初始速度规格、硬化规格和材料分配（用于欧拉分析）。更多信息，请参见Abaqus Analysis User's Guide的第34.2.1节"Abaqus/Standard和Abaqus/Explicit中的初始条件"。
- 您可以为分析中的任何步骤创建预定义温度场。您可以通过为所需步骤输入值或读取具有热组件的先前分析中Abaqus计算的温度值来定义当前模型的温度。更多信息，请参见Abaqus Analysis User's Guide的第34.6.1节"预定义场"中的"温度"。

**注意：**如果您不为预定义场定义初始值，则该场在分析开始时被假定为零值。

创建规定条件后，您可以通过以下方式修改规定条件：
- 您可以修改在创建规定条件时在编辑器中输入的部分或全部数据。
- 您可以使用管理器修改规定条件的逐步历史。（更多信息，请参见"什么是依赖于步骤的管理器？"，第3.4.2节。）
