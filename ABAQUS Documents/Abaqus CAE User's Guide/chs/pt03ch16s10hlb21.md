# 16.10.21 定义欧拉边界条件





您可以创建欧拉边界条件来控制材料流入和流出欧拉区域。欧拉边界条件只能应用于具有欧拉截面分配的部件，并且不能应用于分析的初始步骤。

**指定欧拉边界条件的步骤：**

1. 使用以下方法之一显示欧拉边界条件编辑器： - 要创建新的欧拉边界条件，请按照["创建边界条件"章节第16.8.2节](pt03ch16s08hlb02.md)中概述的程序进行操作（**类别**：`其他`；**所选步骤的类型**：`欧拉边界`）。 - 要使用菜单或管理器编辑现有的欧拉边界条件，请参见["编辑依赖于步骤的对象"章节第3.4.13节](pt01ch03s04hlb05.md)。要编辑应用边界条件的区域，请参见["编辑应用规定条件的区域"章节第16.8.4节](pt03ch16s08hlb04.md)。
2. 指定边界条件将控制的**流动类型**： - **流入**控制欧拉材料通过所选表面流入该区域。 - **流出**控制欧拉材料通过所选表面流出该区域。 - **两者**允许您在同一边界条件定义中指定流入和流出条件的控制。
3. 如果您选择了**流入**（或**两者**）作为**流动类型**，请选择**流入**条件： - 选择**自由**允许任何新材料自由流入欧拉区域。流入区域的材料基于当前占据该区域的材料：当区域内部的欧拉材料远离边界表面时，相同的材料会流过边界来替换它。 - 选择**无**阻止任何新的欧拉材料（包括空隙）通过边界表面流入。区域内部的欧拉材料可以沿着边界表面切向移动，但不能与边界表面分离。 - 选择**空隙**仅允许空隙流入欧拉区域。当区域内部的欧拉材料远离边界表面时，空隙会流过边界来替换它。
4. 如果您选择了**流出**（或**两者**）作为**流动类型**，请选择**流出**条件： - 选择**零压力**允许材料流出欧拉区域，同时使用人工牵引力在边界表面强制执行零净压力条件。 - 选择**自由**允许材料自由流出欧拉区域，不施加任何人工牵引力。 - 选择**无反射**允许材料流出欧拉区域，同时使用人工牵引力最小化膨胀波和剪切波能量反射回区域。 - 选择**平衡**允许材料流出欧拉区域，同时使用人工牵引力在边界表面强制执行零阶连续应力条件。
