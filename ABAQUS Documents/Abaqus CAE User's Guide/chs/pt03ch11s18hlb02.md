# 11.18.1 管理零件





要创建、复制、重命名和删除零件，请使用以下方法之一：
- 主菜单栏上**零件**菜单下的**创建**、**复制**、**重命名**和**删除**项。**复制**、**重命名**和**删除**项包含列出当前模型中所有零件的子菜单。
- **零件管理器**对话框。**零件管理器**对话框包含与主菜单栏上**零件**菜单下列出的类似的功能，但有一个方便的浏览器，列出当前模型中所有可用零件的名称及其建模空间（三维、二维或轴对称）、类型（可变形、离散刚体、分析刚体或欧拉）和状态（锁定、无效或无效（已忽略））。要显示**零件管理器**对话框，请从主菜单栏选择**零件**![](../graphics/images/arrow.gif)**管理器**。

要锁定或解锁当前模型数据库中用于编辑的零件，请使用**零件管理器**对话框中的**锁定**和**解锁**按钮。**状态**字段指示**已锁定**的零件。

要更新零件的有效性或忽略无效性，请使用**零件管理器**对话框中的**更新有效性**和**忽略无效性**按钮。**状态**字段指示**无效**或**无效（已忽略）**的零件。

要从模型数据库中检索零件并将其显示在当前视口中，请从上下文栏中的**零件**列表中选择零件。**零件**列表包含当前模型中的所有零件。
