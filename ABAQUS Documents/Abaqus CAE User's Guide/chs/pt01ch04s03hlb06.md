# 4.3.6 层叠视口





您可以将画布上的视口 arranged 成层叠图案。这可能有助于显示隐藏的视口，或者仅仅是为了减少绘图区域的混乱。Abaqus/CAE排列视口，使标题栏可见，但内容可能被另一个视口遮挡。

**要层叠视口：**

1. 使用视口右上角的**最小化**按钮来最小化您不想层叠的任何视口。
2. 从主菜单栏，选择****视口（Viewport）**![](../graphics/images/arrow.gif)**层叠（Cascade）****。Abaqus/CAE将所有未最小化的视口设置为相同大小，并从绘图区域的左上角向右下角排列它们。当前视口始终位于画布的前面；所有其他视口按数字顺序排列。最小化的视口位于任何层叠视口的前面，但当前视口除外。**提示：**您也可以通过点击**视口**工具栏中的![](../graphics/ico_viewportCascade.png)来层叠未最小化的视口。
