# 47.3 生成X–Y图









### 47.3 生成X--Y图

要生成*X–Y*图，首先指定一个*X–Y*数据对象。然后您可以选择简单地绘制指定的*X–Y*数据对象，或保存数据对象以便稍后绘制。

历史数据和场数据对象可以包含复数；您可以使用**结果选项**对话框控制复数的显示形式。当您绘制变量时，复数形式的缩写会附加到*Y*轴标题，并成为保存变量名的一部分。例如，如果`S-Mises`是所选的场输出变量，`Magnitude`是复数形式，则*Y*轴标题为`S-Mises CPX: Mg`。有关Abaqus/CAE中复数形式的更多信息，请参见["控制复数结果的形式，" 第42.6.9节](pt05ch42s04hlb05.md)。

使用以下方法之一生成*X–Y*图：

**从输出数据库生成X–Y历史数据图：**

从主菜单栏选择****工具**![](../graphics/images/arrow.gif)**XY数据**![](../graphics/images/arrow.gif)**创建****以启动**创建XY数据**对话框；然后选择**ODB历史输出**。点击**继续**。从出现的对话框中，选择要绘制的一个或多个变量以及感兴趣的步；然后点击**绘图**。

**提示：**要从结果树启动**创建XY数据**对话框，请在**XYData**容器上点击鼠标按钮3，然后从出现的列表中选择**创建**。

您也可以通过从主菜单栏选择****结果**![](../graphics/images/arrow.gif)**历史输出****来指定历史输出。

**从输出数据库生成X–Y场数据图：**

从主菜单栏选择****工具**![](../graphics/images/arrow.gif)**XY数据**![](../graphics/images/arrow.gif)**创建****；然后选择**ODB场输出**。点击**继续**。从出现的对话框中，选择要绘制的一个或多个变量、要读取数据的位置或位置，以及感兴趣的步；然后点击**绘图**。

**从模型路径生成输出数据库结果的X–Y图：**

要指定通过模型的路径，请从主菜单栏选择****工具**![](../graphics/images/arrow.gif)**路径**![](../graphics/images/arrow.gif)**创建****；然后配置出现的对话框。有关更多信息，请参见["在模型中创建路径，" 第48.2节](pt05ch48s02.md)。

创建路径后，从主菜单栏选择****工具**![](../graphics/images/arrow.gif)**XY数据**![](../graphics/images/arrow.gif)**创建****。选择**路径**作为*X–Y*数据对象的来源，然后点击**继续**。配置出现的对话框，然后点击**绘图**。有关更多信息，请参见["沿路径获取X--Y数据，" 第48.3节](pt05ch48s03.md)。

**从非输出数据库的新数据生成X–Y图：**

从主菜单栏选择****工具**![](../graphics/images/arrow.gif)**XY数据**![](../graphics/images/arrow.gif)**创建****。选择**对XY数据运算**、**ASCII文件**或**键盘**作为*X–Y*数据对象的来源；然后点击**继续**。在出现的对话框中，输入您的*X–Y*数据对象；然后点击**绘图**（或在**对XY数据运算**对话框中点击**绘图表达式**）。

**从已保存的X–Y数据对象生成X–Y图：**

- 要显示单个*X--Y*数据对象，请从主菜单栏选择****工具**![](../graphics/images/arrow.gif)**XY数据**![](../graphics/images/arrow.gif)**绘图****；然后从弹出菜单中选择*X--Y*数据对象。
- 要在单个*X--Y*图上显示多个*X--Y*数据对象，请从主菜单栏选择****工具**![](../graphics/images/arrow.gif)**XY数据**![](../graphics/images/arrow.gif)**管理器****。从出现的对话框中选择*X--Y*数据对象；然后点击**绘图**。

您也可以从结果树执行这些操作。要使用结果树显示单个*X–Y*数据对象，展开**XYData**容器，点击要绘制的*X–Y*数据对象上的鼠标按钮3，然后从出现的列表中选择**绘图**。要使用结果树显示多个*X–Y*数据对象，请在**XYData**容器下高亮显示多个*X–Y*对象，在其中一个高亮对象上点击鼠标按钮3，然后从出现的列表中选择**绘图**。
