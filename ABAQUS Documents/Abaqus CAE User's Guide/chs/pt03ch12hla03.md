# 12.19 使用查询工具集获取分配信息





您可以使用查询工具集显示以下内容：
- 已分配截面的所有区域列表。
- 分配给所选区域的截面名称。
- 关于需要截面分配的区域的信息。
- 分配给所选线区域的梁取向。
- 分配给所选壳和实体区域的材料取向。
- 复合铺层或复合截面中铺层的图形表示。
- 分配给所选壳区域的钢筋参考取向。
- 分配给所有壳和轴对称线区域的壳/膜法线方向。
- 分配给所有线区域的梁/桁架切线方向。
- 包含不连续区域的复合铺层和铺层。

**显示区域信息：**

1. 从主菜单栏，选择****Tools**![](../graphics/images/arrow.gif)**Query****。
   **提示：**您也可以通过点击**Query**工具栏中的![](../graphics/ico_info.png)工具来查询模型。Abaqus/CAE显示**Query**对话框。您可以请求通用查询或特定于模块的查询。**Shell element normals**和**Beam element tangents**查询是通用查询。有关通用查询显示的信息的讨论，请参见["获取模型的一般信息，"第71.2.2节](pt06ch71s02hlb02.md)。**Section assignments**、**Regions missing sections**、**Beam orientations**、**Material orientations**、**Rebar orientations**、**Ply stack plot**和**Disjoint ply regions**查询特定于Property模块。
2. 从**Property Module Queries**列表中，选择感兴趣的属性。
3. 在视口中选择要查询的区域。
   **提示：**您可以通过点击提示区域中的选择过滤工具![](../graphics/ico_sketchOptions-nls.png)，然后在出现的对话框中点击您选择的选择过滤器来限制可以在视口中选择的对象类型。有关详细信息，请参见["使用选择选项，"第6.3节](pt01ch06s03.md)。
4. 选择要查询的区域后，会出现以下信息：
   **截面分配查询**
   Abaqus/CAE在消息区域显示分配给所选区域的截面名称。如果查询的区域的截面分配已被抑制，Abaqus/CAE报告该区域没有截面分配。
   **缺少截面的区域**
   如果部件的任何区域需要截面分配但没有，Abaqus/CAE会在视口中高亮显示这些区域，并提示您将区域保存为集。如果要将这些区域保存为命名集，请切换出现对话框中的**Save regions in a set**；如果需要，还可以自定义默认集名称。
   **梁截面取向查询**
   Abaqus/CAE在消息区域显示分配给所选梁区域的梁取向名称。此外，部件中每个梁区域的![](../graphics/usi_eqn00394.gif)方向会出现在消息区域。
   **材料取向查询**
   Abaqus/CAE在消息区域显示分配给所选区域的材料取向类型。对于`GLOBAL`、`SYSTEM`和`DISCRETE`类型，Abaqus/CAE会在视口中显示材料取向 triad。此外，关于部件中每个区域的材料取向的信息会出现在消息区域。
   **钢筋取向查询**
   Abaqus/CAE在消息区域显示分配给所选区域的钢筋取向名称。此外，关于部件中每个区域的钢筋参考取向的信息会出现在消息区域。
   **铺层堆叠图**
   Abaqus/CAE创建一个新的视口并显示复合铺层或复合截面的芯样图形表示。图像显示铺层中的铺层以及每个铺层的详细信息，如其纤维取向、厚度、参考平面和积分点。有关更多信息，请参见[第53章，"查看铺层堆叠图"](pt05ch53.md)。
   **不连续的铺层区域**
   Abaqus/CAE在消息区域显示包含不连续区域的复合铺层及其内部铺层的名称。
5. 要退出查询过程，请点击提示区域中的取消按钮![](../graphics/afxI_cancel.png)。
