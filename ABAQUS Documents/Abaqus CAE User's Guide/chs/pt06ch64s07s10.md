# 64.7.10 复制网格图案







选择 ****Mesh**![](../graphics/images/arrow.gif)**Edit****（网格→编辑）从主菜单栏将二维网格图案复制到目标面上。图案的单元必须形成可以覆盖几何面的连接集。目标面必须是同一装配或部件的一部分，且目标面必须至少包含与图案相同数量的环。

**复制网格图案：**

1. 进入 Mesh（网格）模块，并执行以下操作之一： - 从上下文栏的 **Object**（对象）字段中选择 **Assembly**（装配）。 - 从上下文栏的 **Object**（对象）字段中选择 **Part**（部件），然后从列表中选择一个部件。
2. 从主菜单栏中选择 ****Mesh**![](../graphics/images/arrow.gif)**Edit****（网格→编辑）。Abaqus/CAE 显示 **Edit Mesh**（编辑网格）对话框。**提示：**您也可以使用位于 Mesh 模块工具箱底部的 ![](../graphics/ico_meshEdit.png) 工具来显示 **Edit Mesh**（编辑网格）对话框。
3. 从 **Category**（类别）字段中选择 **Mesh**（网格）。
4. 从 **Method**（方法）列表中选择 **Copy mesh pattern**（复制网格图案）。
5. 选择源网格。您可以使用以下任何方法： - 从视口中选择单元。您可以单独、按面角度或按面曲率选择单元（有关更多信息，请参阅["在当前视口中选择对象，" 第 6.2 节](pt01ch06s02.md)）。完成选择后点击鼠标按钮 2。 - 选择一个现有曲面。点击提示区域右侧的 **Surfaces**（曲面）以显示包含可用曲面列表的 **Region Selection**（区域选择）对话框。从列表中选择一个曲面，然后点击 **Continue**（继续）。所选单元必须形成单个连接的二维面。**注意：**默认选择方法基于您最近使用的选择方法。要恢复为其他方法，请点击提示区域右侧的 **Select in Viewport**（在视口中选择）或 **Surfaces**（曲面）。
6. 选择目标面。目标面必须与源位于同一装配或部件中，且面必须至少包含与源相同数量的环。
7. 在 **Copy mesh pattern**（复制网格图案）对话框中，将源网格上的节点位置与目标面上的点关联。您应该至少将源网格外边缘（主环）的三个节点与目标几何上的点关联起来。 1. 点击 **Copy mesh pattern**（复制网格图案）对话框中的 ![](../graphics/ico_add.png) 以向表中添加节点和点。 2. 从源网格中选择所需的节点，然后在提示区域中点击 **Done**（完成）。 3. 在目标面上选择对应的点。Abaqus/CAE 在选择时将当前节点显示为粉红色高亮。当您为每个节点选择一个点后，**Copy mesh pattern**（复制网格图案）对话框会显示表中列出的选择。 4. 在表中高亮一个位置，然后点击 ![](../graphics/ico_edit.png) 来编辑位置。**提示：**您不能编辑节点。要更改节点选择，请使用 ![](../graphics/ico_delete.png) 删除高亮行，然后添加新的节点和位置来替换。 5. 点击对话框中的 **OK**（确定）Abaqus/CAE 将剩余节点映射到目标面并显示复制的网格图案。如果将剩余节点映射到目标时出现问题，Abaqus/CAE 会显示错误消息以帮助您解决它们。**提示：**如果复制的网格图案不符合您的预期，点击 **Edit Mesh**（编辑网格）对话框中的 **Undo**（撤销）可以将其移除。
