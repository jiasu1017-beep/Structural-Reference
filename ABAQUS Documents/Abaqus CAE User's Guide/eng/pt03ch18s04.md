# 18.4 Using the Optimization module toolbox







You can access all the Optimization module tools through either the main menu bar or through the Optimization module toolbox. [Figure 18--1](pt03ch18s04.md#opz-toolbox) shows the icons for all the Optimization module tools in the toolbox. 

**Figure 18–1** The Optimization module tools.

![](../graphics/opz-toolbox-nls.png)

To see a tooltip containing a brief definition of an Optimization module tool, hold the mouse over the tool for a moment. For information on using toolboxes and selecting hidden icons, see ["Using toolboxes and toolbars that contain hidden icons," Section 3.3.2](pt01ch03s03s02.md).
