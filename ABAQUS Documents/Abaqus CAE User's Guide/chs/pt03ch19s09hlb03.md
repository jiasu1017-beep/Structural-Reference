# 19.9.3 提交自适应过程







要提交自适应过程进行分析，请从主菜单栏中选择****自适应**![](../graphics/images/arrow.gif)**提交**![](../graphics/images/arrow.gif)***自适应过程名称*****。（您也可以通过在**自适应过程管理器**中选择过程名称，然后点击**提交**来提交自适应过程进行分析。）Abaqus/CAE 使用自适应过程中定义的设置提交过程进行分析和重新网格划分。

当您启动 Abaqus/CAE 会话并提交自适应过程时，Abaqus/Standard 会向 Abaqus/CAE 发送消息，指示过程的当前状态。然后 Abaqus/CAE 相应地更新**自适应过程管理器**中的**状态**列。您必须保持当前的 Abaqus/CAE 会话活动，以便在分析作业之间自动进行重新网格划分。如果您退出 Abaqus/CAE 并启动新会话，则必须使用手动重新网格划分来继续自适应过程。有关更多信息，请参见["何时需要使用手动自适应重新网格划分？"，第 17.13.5 节](pt03ch17s13s05.md)。

此外，Abaqus/CAE 在自适应过程的每次迭代后在消息区域显示状态信息。状态信息列出每个重新网格划分规则中的错误指标变量，以及该变量是否满足重新网格划分规则。
