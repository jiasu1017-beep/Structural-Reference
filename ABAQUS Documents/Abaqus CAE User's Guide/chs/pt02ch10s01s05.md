# 10.1.5 了解您想要的最终产品





如果您认识到自己想要作为最终产品的是什么——使用Abaqus/Standard或Abaqus/Explicit分析的零件有限元网格——那么导入复杂实体零件到Abaqus/CAE时遇到的许多问题都可以减轻。

导入零件中的小特征将导致该细节区域产生细网格。细网格会影响相邻区域的网格，并可能主导执行分析所需的时间。如果您对分析该特征不感兴趣，则应在将零件导入Abaqus/CAE之前使用CAD系统从零件中删除该细节。删除小特征可以解决导入零件中的精度错误。小特征的示例包括：
- 倒角
- 斜角
- 孔

简化实体零件将增加成功导入Abaqus/CAE的机会。您必须决定将产生有意义的分析结果的详细程度。

最后，您应该考虑分析所需的网格类型。如果您计划使用推进波前算法生成三角形或四面体单元或四边形单元对零件进行网格划分，则可以在生成网格之前使用"网格"模块中的"虚拟拓扑"工具集从零件中删除小细节有关更多信息，请参阅[第75章，"虚拟拓扑工具集"](pt06ch75.md)。
