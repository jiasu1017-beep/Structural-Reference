# 53.2.4 自定义铺层堆叠图中的参考面





点击"**Reference Plane**"选项卡来选择是否在铺层堆叠图中显示参考面,并控制参考面的外观。传统壳单元使用参考面来定义零件的几何形状。因此,您只能在外壳复合层上显示参考面;您不能在连续壳复合铺层或固体复合铺层上显示参考面。此外,如果铺层的偏移量由分布定义,Abaqus/CAE无法显示参考面。

**自定义铺层堆叠图中参考面外观的步骤:**

1. 找到参考面选项。 1. 点击提示区域中的"**铺层堆叠图选项**"按钮,或执行以下操作之一: - 在"属性"模块中,选择 ****View**![](../graphics/images/arrow.gif)**Ply Stack Plot Options****。 - 在"可视化"模块中,选择 ****Options**![](../graphics/images/arrow.gif)**Ply Stack Plot**** 或点击工具箱中的 ![](../graphics/ico_plotPlyStackOptions.png) 工具。 2. 在出现的"铺层堆叠图选项"对话框中点击"**Reference Plane**"选项卡。
2. 执行以下操作: - 切换打开"**Show reference surface**"以在铺层堆叠图中显示参考平面。 - 选择参考面的颜色和半透明度。 - 切换打开"**Show reference outline**"以显示参考面边缘周围的线条。选择线条的颜色、样式和粗细。
3. 点击"**Apply**"实施您的更改。铺层堆叠图会根据您的规格进行更改。默认情况下,您的更改会在会话期间保存,并会影响该视口中所有后续的铺层堆叠图。如果要为后续会话保留更改,请将它们保存到文件。有关更多信息,请参阅 ["为后续会话保存自定义," 第55.1.1节](pt05ch55s01s01.md)。
