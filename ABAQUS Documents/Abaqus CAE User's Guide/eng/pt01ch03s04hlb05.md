# 3.4.13 Editing step-dependent objects







You can use either menus, managers, or the Model Tree to edit step-dependent objects in a particular step. You cannot edit suppressed objects; you must resume the object before editing. For more information, see ["Suppressing and resuming objects," Section 3.4.3](pt01ch03s04s03.md). (For information about the status of modified objects, see ["Understanding modified step-dependent objects," Section 3.4.7](pt01ch03s04s07.md).)

**To edit step-dependent objects using menus: **

1. From the main menu bar, select ****Edit**![](../graphics/images/arrow.gif)***object of your choice***** from the appropriate menu. For example, if you want to edit a load in the Load module, select ****Load**![](../graphics/images/arrow.gif)**Edit**![](../graphics/images/arrow.gif)***load of your choice*****. The appropriate editor appears. The region to which the object is applied becomes highlighted in the current viewport.
2. In the editor, modify the object definition as desired, and click **OK**.

**To edit step-dependent objects using managers:**

1. In the manager, locate the cell of interest. The cell is located in the row of the object that you want to modify and in the column of the step of interest.
2. In the manager, double-click the cell. **Note:**Alternatively, you can click the cell of interest and then click **Edit**. The appropriate editor appears. Abaqus/CAE highlights the region to which the object is applied.
3. In the editor, modify the object definition as desired, and click **OK**.

**To edit step-dependent objects using the Model Tree:**

1. In the Model Tree select the object of interest.
2. Click mouse button 3 on the object and select **Edit** from the menu that appears The appropriate editor appears. Abaqus/CAE highlights the region to which the object is applied.
3. In the editor, modify the object definition as desired, and click **OK**.
