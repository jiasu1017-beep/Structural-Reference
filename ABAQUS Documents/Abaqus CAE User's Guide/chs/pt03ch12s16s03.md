# 12.17.3 处理校准数据







校准数据处理选项使您能够在使用数据定义材料行为之前对其进行清理。可用选项如下：
- 转换
- 缩放
- 平滑
- 截断和偏移

### 在名义形式和真实形式之间转换校准数据

**转换**选项使您能够将校准数据集中的数据从名义形式转换为真实形式，反之亦然。Abaqus/CAE使用"非弹性行为"中"应力和应变测量"第23.1.1节中描述的算法执行转换。

**在名义形式和真实形式之间转换校准数据：**

1. 在模型树中，右键单击要转换的数据集，然后从出现的菜单中选择**处理**。**数据集处理**对话框出现。
2. 选择**转换**，然后单击**继续**。**更改数据集形式**对话框出现。
3. 选择要将所选数据集转换到的形式。选项为**名义（工程）形式**和**真实形式**。
4. 如有需要，您可以在执行数据转换时保留原始数据集并创建新数据集。您也可以自定义新创建的数据集的名称。
5. 单击**确定**。Abaqus/CAE对数据执行所选形式转换。

### 缩放校准数据

**缩放**选项使您能够对校准数据集中的任一列应用不同的缩放因子。

**缩放校准数据：**

1. 在模型树中，右键单击要转换的数据集，然后从出现的菜单中选择**处理**。**数据集处理**对话框出现。
2. 选择**缩放**，然后单击**继续**。**缩放数据集**对话框出现。
3. 通过在**第1列**或**第2列**字段中编辑值来指定数据集任一列的缩放因子。
4. 如有需要，单击**预览**查看由缩放因子值创建的新数据值。Abaqus/CAE反映数据集中每列的新缩放因子。
5. 如有需要，您可以在执行数据转换时保留原始数据集并创建新数据集。您也可以自定义新创建的数据集的名称。
6. 单击**确定**。Abaqus/CAE对数据执行所选缩放。

### 平滑校准数据

**平滑**选项使您能够对校准数据集进行操作以产生具有更平滑曲线的数据集。结果数据集具有与原始数据集相同的*X*坐标值。Abaqus/CAE使用指数移动平均算法来平滑数据；此算法也被**smooth2** *X-Y*运算符使用。请参见"平滑X-Y数据对象"，第47.4.31节。

**平滑校准数据：**

1. 在模型树中，右键单击要转换的数据集，然后从出现的菜单中选择**处理**。**数据集处理**对话框出现。
2. 选择**平滑**，然后单击**继续**。**平滑数据集**对话框出现。
3. 在**权重**字段中，指定平滑参数的值。您可以为曲线指定0到1之间的平滑因子；较小的值会产生更平滑的曲线。默认平滑因子为0.75。
4. 如有需要，您可以在执行数据转换时保留原始数据集并创建新数据集。您也可以自定义新创建的数据集的名称。
5. 单击**确定**。Abaqus/CAE平滑数据集。

### 截断和偏移校准数据

**截断和偏移**选项使您能够从校准数据集中排除X坐标值小于或大于用户指定值的数据点，或仅包括位于两个指定X坐标值之间的数据点。

**截断和偏移校准数据：**

1. 在模型树中，右键单击要转换的数据集，然后从出现的菜单中选择**处理**。**数据集处理**对话框出现。
2. 选择**截断和偏移**，然后单击**继续**。**截断数据集**对话框出现。
3. 指定数据集左侧或右侧的截断点，或指定数据集两侧的点。您可以通过在视口中拖动滑块或在相应的**X轴**字段中输入值来设置截断点。
4. 使用以下技术之一偏移所选数据点子集：- 切换**将第一点偏移到(0,0)**以偏移所选数据点子集，使其最左边的点位于(0,0)。- 通过在**偏移X轴**或**偏移Y轴**字段中指定值来沿任一轴偏移所选数据点子集。
5. 如有需要，您可以在执行数据转换时保留原始数据集并创建新数据集。您也可以自定义新创建的数据集的名称。
6. 单击**确定**。Abaqus/CAE执行指定的截断或偏移。
