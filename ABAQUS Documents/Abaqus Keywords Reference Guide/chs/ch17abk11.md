# *REBAR







### *REBAR将钢筋定义为单元属性。

此选项用作在壳单元、膜单元和实体（连续）单元中将钢筋定义为单元属性的替代方法。在Abaqus/Standard分析中，它必须用于在梁单元中定义钢筋。在壳单元、膜单元和表面单元中定义钢筋的首选选项是[*REBAR LAYER](ch17abk12.md)选项，该选项必须与[*SHELL SECTION](ch18abk15.md)、[*MEMBRANE SECTION](ch13abk16.md)或[*SURFACE SECTION](ch18abk54.md)选项结合使用。在实体中定义钢筋的首选方法是使用[*EMBEDDED ELEMENT](ch05abk14.md)选项将具有钢筋层的加强表面或膜单元嵌入"宿主"实体单元中。

**产品：**Abaqus/Standard  Abaqus/Explicit  

**类型：**模型数据  

**级别：**零件，零件实例

- ["将钢筋定义为单元属性，" Abaqus Analysis User's Guide第2.2.4节](../usb/usb-link.md#usb-int-erebar)

### **必需参数：**

ELEMENT

设置ELEMENT=BEAM以在Abaqus/Standard分析的梁单元中定义钢筋。

设置ELEMENT=SHELL以在三维壳单元中定义钢筋。钢筋不能用于三角形壳单元。

设置ELEMENT=AXISHELL以在轴对称壳单元中定义钢筋。

设置ELEMENT=MEMBRANE以在三维膜单元中定义钢筋。钢筋不能用于三角形膜单元。

设置ELEMENT=AXIMEMBRANE以在Abaqus/Standard分析的轴对称膜单元中定义钢筋。

设置ELEMENT=CONTINUUM以在连续（实体）单元中定义钢筋。钢筋不能用于任何平面三角形、三角棱柱、四面体或无限单元。

MATERIAL

将此参数设置为这些钢筋所用材料的名称。

NAME

将此参数设置为一个标签，用于引用此钢筋集。此标签可用于定义钢筋预应力和输出请求。在特定单元或单元集中，每层钢筋必须分配一个单独的名称。

### **可选参数：**

GEOMETRY

此参数对于梁中的钢筋、轴对称壳或轴对称膜中的钢筋，或实体单元中的单个钢筋没有意义。

设置GEOMETRY=ISOPARAMETRIC（默认）以指示钢筋层平行于单元局部（等参）坐标系的一个方向。

设置GEOMETRY=SKEW以指示钢筋层相对于单元面处于倾斜方向。

ISODIRECTION

将此参数设置为等参方向，钢筋角度输出将从该方向测量。默认为1。

ORIENTATION

此参数仅对壳单元和膜单元中的倾斜钢筋有意义。将此参数设置为定义钢筋角度方向的方向定义名称。此参数不允许与轴对称壳和轴对称膜单元一起使用。

SINGLE

此参数仅对连续单元有意义。如果每个数据行定义单个钢筋，则包含此参数。如果省略此参数，则每一行定义单元等参空间中均匀分布的钢筋层。
