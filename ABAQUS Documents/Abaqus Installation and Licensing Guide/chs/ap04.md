# C.1 使用现有的非 Abaqus Web 服务器





当您安装 Abaqus 文档并选择使用现有的非 Abaqus Web 服务器时，您必须手动配置服务器以访问 HTML（和 PDF）文档。如果您想从**开始**菜单（仅限 Windows 平台）访问 HTML 文档，则在配置服务器后必须修改 `documentation.htm` 文件（请参阅["在 `documentation.htm` 文件中设置文档 URL，"第 C.1.2 节](ap03s01.md#sgb-chp-web-server-start)）。

### C.1.1 配置现有的非 Abaqus Web 服务器

您可以使用以下过程配置现有的非 Abaqus Web 服务器来访问 Abaqus HTML 文档。有关非 Abaqus Web 服务器配置的最新信息，请参阅"Dassault Systmes Knowledge Base"中的"如何使用现有 Web 服务器安装 Abaqus HTML 文档"，网址为 [www.3ds.com/support/knowledge-base](http://www.3ds.com/support/knowledge-base)。

**Linux 平台**

1. 在 SIMULIA 父目录中安装 Abaqus HTML 文档（请参阅["安装过程，"第 2.1 节](ch02s01.md)）。
2. 在 Web 服务器的配置文件中为 HTML 文档的根目录创建一个别名。例如，在 Apache Web 服务器上，您必须添加 ``` Alias /v6.14 "*simulia_dir*/Documentation/docs/v6.14" ``` 其中 *simulia_dir* 是 HTML 文档安装父目录。
3. 在 Web 服务器的配置文件中为 Texis 可执行文件创建一个别名。例如，在 Apache Web 服务器上，您必须添加 ``` ScriptAlias /texis/ "*simulia_dir*/Documentation/scripts/texis/" ```
4. 重新启动 Web 服务器以重新读取服务器的配置文件。
5. 使用以下 URL 访问 Abaqus HTML 文档： ``` http://*computername*:*port_number*/v6.14 ``` 其中 *computername* 是 Web 服务器的完全限定域名，*port_number* 是 Web 服务器运行所在的端口号（仅当 Web 服务器不在默认端口 80 上运行时才需要指定 `:*port_number*`）。

**Windows 平台**

1. 在 SIMULIA 父目录中安装 Abaqus HTML 文档。SIMULIA 父目录的推荐位置是 `C:\SIMULIA`。
2. 在 Web 服务器的配置文件中为 HTML 文档的根目录创建一个别名。例如，在 Apache Web 服务器上，您必须添加 ``` Alias /v6.14 "*simulia_dir*/Documentation/docs/v6.14" ``` 其中 *simulia_dir* 是 HTML 文档安装父目录。
3. 在 Web 服务器的配置文件中为 Texis 可执行文件创建一个别名。例如，在 Apache Web 服务器上，您必须添加 ``` ScriptAlias /texis/ "*simulia_dir*/Documentation/scripts/ texis.exe/" ```
4. 重新启动 Web 服务器以重新读取服务器的配置文件。
5. 使用以下 URL 访问 Abaqus HTML 文档： ``` http://*computername*:*port_number*/v6.14 ``` 其中 *computername* 是 Web 服务器的完全限定域名，*port_number* 是 Web 服务器运行所在的端口号（仅当 Web 服务器不在默认端口 80 上运行时才需要指定 `:*port_number*`）。
6. 要从**开始**菜单访问 Abaqus HTML 文档，请参阅["在 `documentation.htm` 文件中设置文档 URL，"第 C.1.2 节](ap03s01.md#sgb-chp-web-server-start)中的说明。

### C.1.2 在 `documentation.htm` 文件中设置文档 URL

在 Windows 平台上配置现有的非 Abaqus Web 服务器后，您可以修改以下文件以从**开始**菜单访问 Abaqus HTML 文档：

```
`*simulia_dir*/Documentation/installation_info/v6.14/`documentation.htm``
```
其中 *simulia_dir* 是 HTML 文档安装父目录。`documentation.htm` 文件中的行应采用以下形式：
```
<META HTTP-EQUIV="Refresh" CONTENT="0; 
   URL=http://*computername*:*port_number*/v6.14">
```
其中 *computername* 是 Web 服务器的完全限定域名，*port_number* 是 Web 服务器运行所在的端口号（仅当 Web 服务器不在默认端口 80 上运行时才需要指定 `:*port_number*`）。例如，有效的参数定义是
```
<META HTTP-EQUIV="Refresh" CONTENT="0; 
   URL=http://velouria.xyz.com:2080/v6.14">
```
