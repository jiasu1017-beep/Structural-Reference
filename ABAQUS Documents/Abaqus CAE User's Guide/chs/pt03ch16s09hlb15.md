# 16.9.15 定义子结构载荷定义以激活子结构载荷工况





您可以在子结构使用模型中创建子结构载荷定义，以激活分析中一个或多个子结构的子结构载荷工况。激活过程使您可以使用两个选项更改子结构载荷工况载荷和边界条件的幅值：
- 幅值乘数选项将缩放值应用于指定的子结构载荷工况载荷和边界条件。若要精确重现子结构生成期间定义的加载条件，请接受默认值1.0。
- 幅值选择使您可以将可变乘数因子应用于子结构载荷工况载荷和边界条件。

有关Abaqus/CAE中载荷工况的更多信息，请参见第34章"载荷工况"。

**创建或编辑子结构载荷定义以激活子结构载荷工况：**

1. 使用以下方法之一显示子结构载荷编辑器：- 若要创建新的子结构载荷，请遵循"创建载荷"，第16.8.1节中概述的过程（**类别**：`Mechanical`；**所选步骤的类型**：`Substructure load`）。- 若要使用菜单或管理器编辑现有的子结构载荷，请参见"编辑依赖于步骤的对象"，第3.4.13节。
2. 执行以下操作：1. 单击![](../graphics/ico_edit.png)打开**选择子结构载荷工况**对话框。2. 切换打开您要激活的每个子结构载荷工况的复选框。3. 单击**确定**关闭**选择子结构载荷工况**对话框。
3. 在**幅值乘数**文本字段中，输入您要缩放指定的子结构载荷工况载荷和边界条件的值。默认值为1.0，表示不进行缩放。
4. 如果需要，单击**幅值**字段右侧的箭头，并从出现的列表中选择您选择的幅值。或者，您可以单击![](../graphics/ico_amplitudeSmall.png)创建新的幅值。（更多信息，请参见第57章"幅值工具集"。）
5. 单击**确定**保存您的数据并退出编辑器。
