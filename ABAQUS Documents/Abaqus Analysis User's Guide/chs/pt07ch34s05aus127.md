### 概述

预紧力载荷：
- 可用于模拟紧固件在结构上的载荷；
- 施加在用户定义的预紧力截面上；
- 施加在与预紧力截面关联的预紧力节点上；以及
- 需要指定预紧力载荷或拧紧调整量。

### 预紧力载荷的概念

[图34.5.1-1](pt07ch34s05aus127.md#ppretension-assembly-exa)是一个简单示例，说明了预紧力载荷的概念。

**图34.5.1-1** 预紧力载荷示例。

![](../graphics/ppretension-assembly-exa-nls.png)

容器*A*通过预紧螺栓来密封盖子，使垫圈处于压力下。在Abaqus/Standard中，通过在螺栓中添加一个"切割面"或预紧力截面来模拟这种预紧力（如图所示），并对其施加拉伸载荷。通过修改截面一侧的单元，Abaqus/Standard可以自动调整预紧力截面处螺栓的长度，以达到指定的预紧量。在后续步骤中，可以阻止进一步的长度变化，使螺栓成为响应组件上其他载荷的标准变形组件。

### 对预紧力载荷进行建模

Abaqus/Standard允许您通过连续体单元、桁架单元或梁单元建模的紧固件来施加预紧力载荷。根据用于建模紧固件的单元类型，对预紧力载荷进行建模所需的步骤略有不同。

#### 使用连续体单元对紧固件进行建模

在连续体单元中，预紧力截面定义为紧固件内部的一个表面，该表面将其"切割"成两部分（见[图34.5.1-2](pt07ch34s05aus127.md#ppretension-solid)）。对于由多个段组成的紧固件，预紧力截面可以是一组表面。

**图34.5.1-2** 使用连续体单元定义的预紧力截面。

![](../graphics/ppretension-solid-nls.png)

基于单元的表面包含单元和面信息（请参见["基于单元的表面定义，" 第2.3.2节](pt01ch02s03aus17.md)）。您必须将表面转换为可以施加预紧力载荷的预紧力截面，并为其分配一个控制节点。

##### 为预紧力截面分配控制节点

预紧力载荷通过预紧力节点在预紧力截面上传递。预紧力节点不应连接到模型中的任何单元。它只有一个自由度（自由度1），该自由度表示切割两侧沿法线方向的相对位移（见[图34.5.1-3](pt07ch34s05aus127.md#ppretension-normal)）。此节点的坐标不重要。

**图34.5.1-3** 预紧力截面的法线；该法线应背离底层单元。

![](../graphics/ppretension-normal-nls.png)

##### 定义预紧力截面的法线

Abaqus/Standard计算截面的平均法线——沿正表面方向，背离用于生成表面的连续体单元——以确定施加预紧力的方向。您也可以直接指定法线（当所需的载荷方向与预紧力截面的平均法线不同时）。在进行大位移分析时，法线不会更新。

##### 识别预紧力截面两侧的单元

对于所有通过至少一个节点连接到预紧力截面的单元，Abaqus/Standard必须确定每个单元位于预紧力截面的哪一侧。此过程对于预紧力载荷的正确工作至关重要。

用于定义截面的单元在此讨论中称为"基础单元"。与基础单元位于截面同一侧的所有单元称为"下层单元"。所有与截面连接且与基础单元共享面（在二维问题中为边）的单元被添加到下层单元列表中。这是一个重复的过程，使Abaqus/Standard能够找到几乎所有网格中的下层单元——三角形；楔形；四面体；以及嵌入的梁、桁架、壳和膜单元——这些单元未用于表面定义（见[图34.5.1-4](pt07ch34s05aus127.md#ppretension-underlying)）。

**图34.5.1-4** 基础单元用于查找下层单元。

![](../graphics/ppretension-underlying-nls.png)

在大多数情况下，此过程会将所有连接到截面的单元分组为两个区域，如图所示。在极少数情况下，此过程可能会将单元分组为两个以上的区域，特别是当线单元跨越单元边界时。示例如[图34.5.1-5](pt07ch34s05aus127.md#ppretension-add-under)所示，它有三个区域，其中区域1是下层区域。

**图34.5.1-5** 发现额外的下层单元。

![](../graphics/ppretension-add-under-nls.png)

对于区域1以外的每个区域，需要额外的步骤来确定该区域位于截面的哪一侧。Abaqus/Standard为属于截面的该区域的所有节点计算平均法线，![](../graphics/usb_eqn01806.gif)，并计算所有这些节点的平均位置（![](../graphics/usb_eqn06078.gif)）。此外，它还计算该区域剩余节点的平均位置（![](../graphics/usb_eqn06079.gif)）。如果法线![](../graphics/usb_eqn01806.gif)与向量![](../graphics/usb_eqn04408.gif)的点积为负，则假定该区域是一个下层区域，并被添加到区域1。此额外步骤在[图34.5.1-5](pt07ch34s05aus127.md#ppretension-add-under)中针对区域2和3进行了说明。

对于[图34.5.1-6](pt07ch34s05aus127.md#ppretension-no-add-under)所示的梁单元，此额外步骤会产生不正确的分离，因为该梁单元未被确定为下层单元。

**图34.5.1-6** 未找到额外的下层单元。

![](../graphics/ppretension-no-add-under-nls.png)

如果预紧力截面形状特殊，并且一个或多个跨越单元边界的线单元连接到它，请查阅数据（`.dat`）文件中给出的下层单元列表，以确保下层单元被正确列出。

仅连接到预紧力截面上节点的单元（包括单节点单元，如SPRING1、DASHPOT1和MASS单元）不包括为下层单元：它们被认为连接到截面的另一侧。

#### 使用桁架或梁单元对紧固件进行建模

当使用桁架或梁单元对预紧组件进行建模时，预紧力截面会缩小为一个点。截面假定位于单元定义中最后一个节点处（请参见["梁单元库，" 第29.3.8节](pt06ch29s03ael14.md)和["桁架单元库，" 第29.2.2节](pt06ch29s02ael13.md)中关于梁和桁架单元节点排序的定义），其法线沿元素从第一个节点到最后一个节点的方向。因此，只需指定要施加预紧力载荷的单元并将其与预紧力节点关联，即可完全定义截面。

| **Abaqus/CAE用法：** | Load模块：**Create Load**：选择**Mechanical**作为**Category**，选择**Bolt load**作为**Types for Selected Step** |
| --- | --- |

与基于表面的预紧力截面一样，该节点只有一个自由度（自由度1），该自由度表示切割两侧沿法线方向的相对位移（见[图34.5.1-7](pt07ch34s05aus127.md#ppretension-truss-beam)）。此节点的坐标不重要。

![](../graphics/ppretension-truss-beam-nls.png)

##### 定义预紧力截面的法线

Abaqus/Standard将法线计算为底层单元连接中从第一个节点到最后一个节点的向量。或者，您可以直接指定截面的法线。此法线在大位移分析期间不会更新。

### 定义多个预紧力截面

您可以通过重复预紧力截面定义来定义多个预紧力截面。每个预紧力截面应有自己的预紧力节点。

### 与节点转换一起使用

预紧力节点不能使用局部坐标系（请参见["转换坐标系，" 第2.1.5节](pt01ch02s01aus09.md)）。可以在预紧力截面上的节点处使用局部坐标系。

### 施加指定的预紧力载荷

预紧力载荷通过预紧力节点在预紧力截面上传递。

#### 指定预紧力

您可以将集中载荷施加到预紧力节点。该载荷是在预紧力截面上传递的自平衡力，沿法线方向作用于包含预紧力截面下层的紧固件部分（包含用于定义预紧力截面的单元的部分；见[图34.5.1-8](pt07ch34s05aus127.md#ppretension-load)）。

| **输入文件用法：** | ``` [*CLOAD](../key/key-link.md#usb-kws-hcload) ``` |
| --- | --- |

| **Abaqus/CAE用法：** | Load模块：**Create Load**：选择**Mechanical**作为**Category**，选择**Bolt load**作为**Types for Selected Step**：选择表面，必要时选择基准轴：**Method: Apply force** |
| --- | --- |

您可以通过在预紧力节点上使用非零边界条件来指定预紧力截面的拧紧调整量（对应于预紧力截面所切割组件长度沿法线方向的变化）。

#### 在分析过程中控制预紧力节点

通过在初始预紧力施加到紧固件后，使用边界条件将自由度固定在步骤开始时的当前值，您可以保持预紧力截面的初始调整量；该技术使预紧力截面上的载荷能够根据外部施加的载荷变化以维持平衡。如果未保持截面的初始调整量，紧固件中的力将保持恒定。

当预紧力节点不受边界条件控制时，请确保结构组件受到运动约束；否则，由于刚体模式的存在，结构可能会散开。Abaqus/Standard如果在分析的第一步中未找到预紧力节点上的任何边界条件或载荷，将发出警告消息。

### 结果显示

Abaqus/Standard自动调整组件在预紧力截面处的长度，以达到指定的预紧量。此调整通过将位于预紧力截面上下层单元的节点相对于这些节点出现在连接到预紧力截面的其他单元中时的相同节点进行移动来实现。因此，下层单元将显示为收缩，尽管在施加预紧力时它们承受拉应力。

### 使用预紧力载荷时的限制

预紧力载荷受以下限制：
- 预紧力载荷不能在子结构中指定。
- 如果执行子模型分析（["子模型：概述，" 第10.2.1节](pt04ch10s02aus60.md)），任何预紧力截面都不应跨越指定驱动节点的区域。换句话说，预紧力截面应完全出现在全局模型中不属于子模型的区域，或完全出现在全局模型中属于子模型的区域。在后一种情况下，执行子模型分析时，预紧力截面也必须出现在子模型中。
- 预紧力截面的节点不应通过多点约束（["通用多点约束，" 第35.2.2节](pt08ch35s02aus130.md)）连接到身体的其他部分。这些节点可以通过方程（["线性约束方程，" 第35.2.1节](pt08ch35s02aus129.md)）连接到身体的其他部分。但是，将预紧力截面上的节点连接到位于截面下层侧的节点会引入跨越预紧力切割的约束，因此直接与预紧力载荷的施加相互作用。另一方面，将预紧力截面上的节点连接到截面另一侧的节点不会影响预紧力载荷的施加。

### 步骤

可以使用任何使用具有位移自由度的单元类型的Abaqus/Standard步骤。在指定初始预紧力时，静力分析是最可能的步骤类型（["静力应力分析，" 第6.2.2节](pt03ch06s02at01.md)）。也可以使用其他分析类型，如耦合温度-位移（["顺序耦合热应力分析，" 第16.1.2节](pt04ch16s01at39.md)）或耦合热电结构（["完全耦合热电结构分析，" 第6.7.4节](pt03ch06s07at23.md)）。一旦施加了初始预紧力，可以例如使用静力或动态分析（["动态分析步骤：概述，" 第6.3.1节](pt03ch06s03abo07.md)）来在保持拧紧调整量的同时施加其他载荷。

### 输出

预紧力截面上的总力是预紧力节点的反作用力加上在该节点上指定的集中载荷的总和。总力可通过输出变量标识符TF获得（请参见["Abaqus/Standard输出变量标识符，" 第4.2.1节](pt02ch04s02abv01.md)）。力沿法线方向。预紧力截面上的剪力不可用于输出。

预紧力截面的拧紧调整量可作为预紧力节点的位移输出。使用输出标识符U请求位移输出。由于在任何其他方向上没有调整，因此仅输出垂直于预紧力截面的调整量。

预紧力截面上的应力分布不能直接获得；但是，可以很容易地显示下层单元中的应力。或者，可以在预紧力截面位置插入绑定接触对，以通过输出标识符CPRESS和CSHEAR启用应力分布输出。有关定义绑定接触的详细信息，请参见["在Abaqus/Standard中定义绑定接触，" 第36.3.7节](pt09ch36s03aus151.md)。

### 概述

表面属性分配：
- 可用于为表面区域指定几何修正；
- 可用于更改基于结构元素的表面区域使用的接触厚度，或为基于实体元素的表面区域添加接触厚度；
- 可用于为基于壳、膜、刚体和表面元素的表面区域指定表面偏移；
- 可有选择性地应用于一般接触域内的特定区域；以及
- 不能应用于解析刚体表面。

### 分配表面属性

您可以为涉及一般接触交互的表面分配非默认表面属性。这些属性仅在表面涉及一般接触交互时才被考虑；当表面涉及其他交互（如接触对）时，不考虑这些属性。一般接触算法不考虑作为表面定义一部分指定的表面属性。

Abaqus/Standard中的一般接触表面属性在分析开始时分配，在步骤之间不能修改。

用于指定具有非默认表面属性的区域的表面名称不必与用于指定一般接触域的表面名称对应。在许多情况下，接触交互将为一个大域定义，而非默认表面属性将分配给该域的子集。位于一般接触域外的区域的任何表面属性分配都将被忽略。如果指定区域重叠，最后的分配优先。

### 表面几何修正

默认情况下，接触计算基于一般接触域中有限元表面的未平滑化 faceted 表示。可选的接触平滑技术模拟接触计算中曲面的更真实表示，从而提高接触应力和压力精度。此接触平滑技术在["Abaqus/Standard中的接触表面平滑，" 第38.1.3节](pt09ch38s01aus179.md)中讨论。

### 表面厚度

默认表面厚度等于原始父单元厚度。或者，您可以指定表面厚度的值或厚度缩放因子。可以为实体单元表面分配非零厚度；例如，用于模拟有限厚度表面涂层的效果。

#### 使用原始父单元厚度

默认表面厚度等于原始父单元厚度。

#### 对表面厚度应用缩放因子

您可以对任何表面厚度值应用缩放因子。例如，如果指定对`surf1`使用原始父单元厚度，并应用0.5的缩放因子，则当`surf1`涉及一般接触交互时，将使用原始父单元厚度一半的值（一般接触域中所有其他表面将使用默认的原始父单元厚度）。以这种方式缩放表面厚度可在某些情况下避免初始过盈接触。Abaqus/Standard将自动调整表面位置以解决与一般接触相关的初始过盈接触（请参见["控制Abaqus/Standard中的初始接触状态，" 第36.2.4节](pt09ch36s02aus142.md)）。但是，如果不需要节点位置调整（例如，如果它们会在原本平坦的零件中引入缺陷，导致不现实的屈曲模式），您可能希望减少表面厚度并完全避免过盈接触。

### 表面偏移

表面偏移是薄体中面与其参考平面（由节点坐标和单元连接性定义）之间的距离。它是通过将偏移分数（指定为表面厚度的分数）乘以表面厚度和单元面法线来计算的。这定义了中面的位置，从而定义了身体相对于参考表面的位置；参考表面上节点的坐标不会被修改。表面偏移只能为在壳和类似元素上定义的表面指定（即膜、刚体和表面元素）。为其他元素（如实体或梁元素）指定的表面偏移将被忽略。默认情况下，元素截面定义中指定的表面偏移将在一般接触算法中使用。

您可以将表面偏移指定为表面厚度的分数。表面偏移分数可以设置为与表面父元素使用的偏移分数相等，或设置为指定值。为一般接触指定的表面偏移不会改变单元积分。

| **输入文件用法：** | 使用以下选项使用来自表面父元素的表面偏移分数（默认）： |
| --- | --- |
|  | ``` [*SURFACE PROPERTY ASSIGNMENT](../key/key-link.md#usb-kws-hsurfpropassign), PROPERTY=OFFSET FRACTION *surface*, ORIGINAL ``` 使用以下选项指定表面偏移分数的值： ``` [*SURFACE PROPERTY ASSIGNMENT](../key/key-link.md#usb-kws-hsurfpropassign), PROPERTY=OFFSET FRACTION *surface*, *offset* ``` 偏移可以指定为值或标签（SPOS或SNEG）。指定SPOS等效于指定0.5的值；指定SNEG等效于指定-0.5的值。 |

| **Abaqus/CAE用法：** | Interaction模块：**Create Interaction**：**General contact (Standard)**：**Surface Properties**：**Shell/Membrane offset assignments: Edit**：选择表面，单击箭头将表面转移到偏移分配列表。在**Offset Fraction**列中，输入ORIGINAL以使用来自表面父元素的表面偏移分数，输入SPOS以使用0.5的表面偏移分数，输入SNEG以使用-0.5的表面偏移分数，或输入表面偏移分数的值。 |

### 特征边

模型的特征边在梁和桁架元素上以及实体和结构元素的面的边缘（周长和其他）上定义。Abaqus/Standard中的一般接触包括边对表面接触公式和边对边接触公式（作为表面对表面公式的补充），如["在Abaqus/Standard中定义一般接触交互，" 第36.2.1节](pt09ch36s02aus139.md)中所讨论。默认情况下，边对表面接触公式考虑梁和桁架元素的"边缘"、周长边缘以及初始几何特征角为45及更高的边缘。您可以全局或局部控制边对表面和边对边接触的特征边准则。特征边准则对梁和桁架元素的"边缘"没有影响——它们通过包含在接触域中被激活。

某些接触属性分配选项仅适用于表面对表面公式（有关一般接触的接触属性的进一步讨论，请参见["Abaqus/Standard中一般接触的接触属性，" 第36.2.3节](pt09ch36s02aus141.md)）。边对表面和边对边公式始终使用惩罚 enforcement 方法，仅涉及位移自由度。例如，边对表面公式或交叉边对边公式不会对接触界面的热间隙电导做出贡献。

#### 指定截止特征角

特征角是两个连接边缘的 facet 法线之间形成的角度。facet之间的角度基于初始构型。在facet的凹形连接处产生负角度；因此，这些边缘不会被包含在接触域中。[图36.2.2-1](pt09ch36s04aus156.md#feature-angles)显示了一些如何计算不同边缘特征角的示例。

周长边缘（例如，[图36.2.2-1](pt09ch36s04aus156.md#feature-angles)中的边缘D）可以被认为是特征边的一种特殊类型，其特征角为180。

如果特征角准则生效（默认或您指定），则特征角大于或等于指定角度的实体和壳体的几何边缘被包含在一般接触域中。接触包含和排除选项（在["在Abaqus/Standard中定义一般接触交互，" 第36.2.1节](pt09ch36s02aus139.md)中讨论）适用于表面对表面接触公式、边对表面接触公式和边对边接触公式（并进一步控制哪些表面部分可能与任一公式交互）。在确定是否应将几何特征边包含在一般接触域中时，会考虑特征角的符号。例如，如果指定了20的截止特征角，则边缘A将被激活为接触模型中的特征边（因为90的特征角大于20的截止值），但边缘B和C不会被激活（因为边缘B处的特征角为25，边缘C处的最大特征角为0，都小于20的截止值）。截止特征角不能设置为小于0或大于180。指定小的截止特征角（例如小于20）可能会大大增加运行时间，而对结果的影响与较大的截止角（>20）相比不大。边对表面接触的默认特征角截止值为45，而默认不将特征边包含在边对边接触中。用于边对表面接触的包含边缘的准则可以与用于边对边接触的包含边缘的准则不同；两者完全独立。

#### 指定应仅激活周长边缘
