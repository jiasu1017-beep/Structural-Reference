# 11.26.1 创建放样截面







放样截面表示放样特征在沿放样路径的特定点处具有的形状。创建放样特征至少需要两个截面。您通过从当前视口中零件的现有边缘进行选择来创建放样截面。

**创建放样截面：**

1. 从**编辑放样**对话框中，选择**向前插入**或**向后插入**来创建第一个放样截面。
2. 选择定义放样截面的边缘。**[Shift]**+单击**添加更多边缘到您的选择，**[Ctrl]**+单击**已选择的边缘以取消选择。选定的边缘以红色高亮显示，并将保持高亮直到放样特征完成。放样截面可以由当前零件的任何边缘组成，但必须满足以下条件：- 对于实体放样或切割放样，截面必须是闭合的，且第一个和最后一个截面必须是平面的。- 对于壳放样，如果一个截面是开放的，则所有截面都必须是开放的；如果一个截面是闭合的，则所有截面都必须是闭合的。- 截面必须是没有分支的单一连续环。- 在一个放样截面中使用的边缘不能在同一放样的另一个截面中使用。所有壳放样截面以及实体放样和切割放样的中间截面都可以是非平面的。要创建非平面截面，您可以选择使用样条线框创建的边缘或选择现有非平面面的边缘。
3. 在提示区域，单击**完成**表示您已完成选择边缘来定义放样截面。Abaqus/CAE返回到**编辑放样**对话框，截面出现在**截面**列表中。
4. 重复步骤2和3来定义所有放样截面。使用**向前插入**和**向后插入**按钮来指示下一个放样截面在列表中相对于当前高亮显示的放样截面的位置。列表中高亮显示的截面也在视口中以品红色高亮显示，以区别于其他放样截面。放样截面按您创建它们的顺序编号，但Abaqus/CAE按**截面**列表中显示的顺序创建放样特征。您只能通过添加或删除放样截面来更改列表的顺序。无法编辑已完成的放样截面。
