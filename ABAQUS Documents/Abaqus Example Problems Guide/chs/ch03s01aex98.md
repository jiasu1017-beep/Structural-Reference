# 3.1.10 带流动的圆形管道中的声学







**产品：**Abaqus/Standard

本示例说明了使用对称模型生成和对称结果传递功能来创建声学模型，以及在Abaqus中使用稳态传递功能来指定声学介质中的旋转速度并分析受旋转流动影响的声场。分析了轮胎形状的声腔。

### 问题描述与模型定义

该模型是车辆轮胎声腔的简化版本。腔的横截面使用轴对称单元建立，这些单元被分配对应于空气的材料特性。虚拟步用于建立后续三维分析步骤中使用的各种文件。

完整的环形腔使用旋转对称模型定义创建，形成围绕圆周的60个均匀网格段。稳态动态、特征频率提取和复频率提取过程使用声流速度来包括旋转流动的影响。

空气腔在静止时、28.0弧度/秒（对应于100公里/小时）和320弧度/秒（对应于腔外边缘处的马赫0.3）进行分析。

稳态动态结果在150至370 Hz之间的频率下计算。对于复分析，要求在45至1000 Hz之间的模式。

### 载荷

在时谐动态分析中，在torus底部的节点对上施加单位虚数集中体积加速度声载荷。未施加边界条件。

在实频率和复频率分析中，未施加声载荷或边界条件。

### 结果与讨论

稳态动态分析的结果显示了旋转流动对声压场的影响。随着速度增加，场中出现了强烈的影响。

实频率分析的结果显示了旋转流动对声压模式和频率的一些影响。然而，实频率分析忽略了公式中的重要对流项，该项是复值的。因此，旋转声介质的实频率分析应被视为复频率计算的初步步骤，实频率分析的结果不应被赋予太多物理意义。复频率分析使用完整的非对称复声学单元公式。然而，复贡献是反对称的；因此，复频率过程计算的共振频率是纯虚数的；即，没有与能量增益或损失相关的特征值的实部。这与旋转介质中的声学物理一致。流动对特征值的影响在速度增加时共振的明显分裂中是显而易见的。

完整三维模型，实和复频率分析。
