# 3.3 小结

- Abaqus 拥有丰富的单元库，可用于各种结构分析应用。单元类型的选择对仿真结果的准确性和效率有着重要影响。Abaqus/Explicit 中的单元类型通常是 Abaqus/Standard 中单元类型的子集。
- 节点上激活的自由度取决于连接到该节点的单元类型。
- 单元名称完全确定了单元的族、公式、节点数和积分类型。
- 所有单元都必须引用截面属性定义。截面属性提供定义单元几何形状所需的任何附加数据，同时标识关联的材料属性定义。
- 对于连续体单元，Abaqus 使用全局笛卡尔坐标系定义单元输出变量（如应力和应变）。您可以通过使用 `*ORIENTATION` 选项更改为局部坐标系。
- 对于三维壳单元，Abaqus 使用基于壳表面的坐标系定义单元输出变量。您可以通过使用 `*ORIENTATION` 选项更改坐标系。
- 为了提高计算效率，模型的任何部分都可以定义为刚体，刚体仅在其参考节点处具有自由度。
- 作为 Abaqus/Explicit 分析中的约束方法，刚体比多点约束具有更高的计算效率。
