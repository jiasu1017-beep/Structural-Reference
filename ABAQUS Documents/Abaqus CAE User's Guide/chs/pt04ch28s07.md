# 28.7 查看欧拉分析的结果





必须以不同于拉格朗日分析的方式解释欧拉分析的结果。特别是，基于节点位移的任何结果在欧拉模型中都没有意义，因为欧拉部件是固定且刚性的。有关欧拉结果如何写入输出数据库的详细信息，请参见["输出"中的"欧拉分析"，Abaqus Analysis User's Guide第14.1.1节](../usb/usb-link.md#usb-anl-aeuleriananal-output)。

在Abaqus/CAE的Visualization模块中查看欧拉部件内的材料实例需要采取特殊步骤。默认情况下，Abaqus/CAE在未变形和变形绘图状态下显示完整的欧拉部件网格，但不显示网格内材料实例边界的指示。

材料实例的可视化基于输出变量EVF，即欧拉材料体积分数。输出变量EVF测量元素内特定材料实例的数量作为相对分数。EVF值为1表示该元素完全充满指定的材料实例；EVF值为0表示该元素完全不含指定的材料实例。

对于部分填充或填充有多种材料的元素，Abaqus通过插值相邻元素中的EVF值来估计材料之间的简单边界。这些简单边界可能在元素之间略微不连续。为了改善欧拉材料的显示，您应指示Abaqus/CAE使用100%的结果平均阈值或在平均结果后计算标量；Abaqus/CAE会重新映射材料边界，使它们在元素之间看起来平滑且连续。有关结果平均的更多信息，请参见["控制结果平均"，第42.6.6节](pt05ch42s04hlb02.md)；有关Abaqus如何计算欧拉材料边界的更详细讨论，请参见["材料界面"中的"欧拉分析"，Abaqus Analysis User's Guide第14.1.1节](../usb/usb-link.md#usb-anl-aeuleriananal-mat)。

如果您在场输出请求编辑器中请求**Preselected defaults**，则输出变量EVF会被写入输出数据库（参见["修改场输出请求"，第14.12.2节](pt03ch14s12hlb02.md)）。当您请求EVF输出时，Abaqus会为模型中的每个材料实例创建一个单独的材料体积分数输出变量；例如，`EVF_WATER`是名为`Water`的材料实例的体积分数。创建一个名为`EVF_VOID`的输出变量来测量欧拉部件中空区域的体积分数。

以下技术可用于Visualization模块查看欧拉部件中材料的初始状态和变形状态：

**等值线图**

特定材料实例的输出变量EVF等值线图允许您可视化分析过程中模型的哪些区域被该材料占据。被材料占据的区域（EVF等于1）以等值线谱顶部的统一颜色显示，而未被材料占据的区域以等值线谱底部的不同颜色显示；根据您的等值线图设置，当EVF从1过渡到0时，材料实例边界以一定范围内的颜色显示（参见[图28-12](pt04ch28s07.md#usi-eulerian-slosh-contour)）。

**图28-12** 欧拉材料实例的等值线图。

![](../graphics/usi-eulerian-slosh-contour.png)

在可视化欧拉材料时，等值线图的用途有限，因为等值线出现在欧拉部件的表面上。您无法有效地可视化欧拉部件内部的材料体积分数等值线。

有关在Visualization模块中使用等值线图的更多详细信息，请参见[第44章，"分析结果的等值线绘制"](pt05ch44.md)。

**视图切割**

要可视化欧拉部件内部材料的行为，请沿该材料实例关联的EVF变量的等值面激活视图切割。Abaqus/CAE自动为模型中的每个材料实例创建这些等值面视图切割，但您必须在**View Cut Manager**中激活它们。使用视图切割选项，您可以消除不包含所选材料的部件部分，方法是将其渲染为未填充、半透明或从显示中移除（参见[图28-13](pt04ch28s07.md#usi-eulerian-slosh-cut)）。

**图28-13** 沿欧拉材料实例等值面的视图切割。

![](../graphics/usi-eulerian-slosh-cut.png)

如果您的欧拉部件包含未分配材料的区域，激活基于`EVF_VOID`输出变量的等值面视图切割可能会有帮助。通过切除所有`EVF_VOID`大于0.5的区域，您能够看到部件内材料的形状。

激活基于EVF变量的等值面视图切割后，您可以更改主场输出变量而不影响视图切割。这使您能够沿材料实例边界而不是在欧拉部件面上可视化结果等值线（参见[图28-14](pt04ch28s07.md#usi-eulerian-slosh-cutresults)）。

**图28-14** 切割欧拉部件中应力的等值线图。

![](../graphics/usi-eulerian-slosh-cutresults.png)

基于输出变量EVF的等值面视图切割不影响耦合欧拉-拉格朗日模型中的拉格朗日部件实例。切割处于活动状态时，拉格朗日部件保持可见。因此，此技术对于可视化拉格朗日部件与欧拉材料实例之间的相互作用很有用。

有关在Visualization模块中使用视图切割的更多详细信息，请参见[第80章，"切割模型"](pt07ch80.md)。

**组合视图切割和等值线图**

在包含三个材料实例的欧拉部件中，您可以使用视图切割和等值线图的组合来区分未变形和变形模型状态中的材料实例。首先，使用等值面视图切割从显示中移除其中一个材料实例，如上所述。然后，为剩余材料实例之一创建输出变量EVF的等值线图。结果图中的颜色区分了一种材料与另一种材料。为了在材料之间产生更明确的边界，您可以将等值线间隔减少到两个。

例如，[图28-15](pt04ch28s07.md#usi-eulerian-projectile)描述了一个铅弹丸撞击黄铜板的欧拉模型。欧拉部件的空区域被切除。应用了两间隔等值线图，以一种颜色渲染黄铜，另一种颜色渲染铅（即非黄铜）。结果图提供了弹丸和板变形形状的有用概括。

**图28-15** 在弹丸冲击分析中使用等值线图区分两种欧拉材料。

![](../graphics/usi-eulerian-projectile.png)

目前没有办法使用Abaqus/CAE同时视觉上区分三个以上的欧拉材料实例。

**颜色编码**

颜色编码不能用于可视化欧拉部件中的材料行为。Abaqus/CAE中的颜色编码工具无法识别欧拉截面或材料分配。基于单元集的颜色编码在变形形状图中也无效，因为欧拉单元不会随材料变形。

然而，在耦合欧拉-拉格朗日模型中，颜色编码可以区分欧拉和拉格朗日部件实例或欧拉和拉格朗日单元类型。当与上述可视化技术结合使用时，颜色编码有助于区分模型中的拉格朗日主体和欧拉材料。

有关将颜色编码应用于模型的更多详细信息，请参见[第77章，"几何和网格元素的颜色编码"](pt07ch77.md)。

**显示组**

某些类型的耦合欧拉-拉格朗日模型在整个欧拉部件中涉及单一欧拉材料实例；例如，穿透器穿过均匀欧拉材料的拉格朗日主体。在这些分析中，欧拉材料的变形不如欧拉材料与拉格朗日主体之间的相互作用重要。您可以使用显示组从显示中移除欧拉单元，仅可视化拉格朗日主体上的结果（如接触压力或应力）。

有关使用显示组的更多详细信息，请参见[第78章，"使用显示组显示模型的子集"](pt07ch78.md)。
