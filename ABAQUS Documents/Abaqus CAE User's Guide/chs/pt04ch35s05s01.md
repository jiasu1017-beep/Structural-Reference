# 35.5.1 分配中面区域





要开始创建中面模型，请选择要表示为壳的实体模型区域。当您将一个或多个单元的实体模型分配为中面区域时，Abaqus/CAE会创建**Assign Midsurface**特征，并将所选单元从部件的活动表示移动到参考表示。参考表示的工作方式与草图中的参考几何体类似——使用Geometry Edit工具集，您可以从参考表示中选择点和边缘，以在中面模型中创建点、边缘和面。您创建的新中面几何体将成为将进行网格划分和分析的活动表示的一部分。

**创建参考表示：**

1. 从主菜单栏中，选择****Tools**![](../graphics/images/arrow.gif)**Midsurface**![](../graphics/images/arrow.gif)**Assign****。
   **提示：**您也可以使用Part模块工具箱中与中面工具一起的![](../graphics/ico_assignMidSurface.png)工具来分配中面区域。
2. 选择要从中面模型创建其实体单元。您可以从视口中选择单个单元或选择现有单元集。有关使用Abaqus/CAE中选择方法的更多信息，请参见[第6章，"在视口中选择对象"](pt01ch06.md)。Abaqus/CAE创建一个包含所选单元的**Assign midsurface**特征，并在视口中以半透明方式显示它们。
3. 重复步骤2以分配更多中面区域，或单击**Done**结束过程。
