# 8.1.3 Windows 打印机图像大小





当你直接将所选视口的快照打印到 Windows 打印机时，图像大小由三种方法之一决定：

**适应页面**

使用此方法将打印图像缩放以适应可用页面大小。此方法为默认方法。

**使用屏幕大小**

使用此方法将打印图像大小与画布上的当前大小匹配。如果图像大小超出可用页面大小，你必须调整图像大小、更改页面大小或边距，或选择其他方法打印图像。

**使用设置**

使用此方法直接指定打印图像的大小。你指定宽度或高度，Abaqus/CAE 调整另一个维度以保持图像纵横比。如果图像超出可用页面大小，你必须调整图像大小、更改页面大小或边距，或选择其他方法打印图像。
