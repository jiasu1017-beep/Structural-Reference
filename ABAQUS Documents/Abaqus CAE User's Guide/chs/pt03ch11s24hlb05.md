# 11.24.5 切割圆孔





从主菜单栏选择**形状**![](../graphics/images/arrow.gif)**切割**![](../graphics/images/arrow.gif)**圆孔**，通过当前视口中的零件切割圆孔。无论当前视口中零件的建模空间如何，圆孔工具始终可用。

您通过指定距两条选定边缘的距离并指定孔的直径来切割圆孔。

零件必须至少包含两条直边；例如，您不能使用此工具切割穿过圆形零件的孔。

**切割圆孔：**

1. 从主菜单栏，选择**形状**![](../graphics/images/arrow.gif)**切割**![](../graphics/images/arrow.gif)**圆孔**。
2. 如果当前视口包含二维零件，选择第一条边来定位孔的中心。
3. 在提示区域的文本字段中，输入从选定边缘到孔中心的距离。
4. 选择第二条边来定位孔的中心。
5. 在提示区域的文本字段中，输入孔的直径。
