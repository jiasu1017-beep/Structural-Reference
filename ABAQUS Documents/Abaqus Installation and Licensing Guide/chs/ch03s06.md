# 3.6 FLEXnet Licensing 管理工具



FLEXnet Licensing 提供管理工具来帮助管理网络许可证活动。Abaqus 许可证安装过程安装许可证文件和 Abaqus 许可证服务器。FLEXnet Licensing 管理工具安装在 `*simulia_dir*/License` 中。如果您在许可证服务器上安装了 Abaqus 产品，则可以使用 `abaqus` 命令访问 `License` 目录。运行命令

```
abaqus licensing
```
 不带额外参数会显示所有可用 FLEXnet Licensing 管理工具的命令用法摘要。FLEXnet Licensing End User Guide Version 11.6 包含有关 FLEXnet Licensing 管理工具的语法和选项的详细信息。您可以从 **Support** 页面上的 **Licensing** 部分下载此指南，网址为 [www.3ds.com/simulia](http://www.3ds.com/simulia)。

在 Windows 平台上，许可证工具也可以使用 FLEXnet Licensing 工具箱 **LMTOOLS** 访问。可以从**开始**菜单或通过执行以下命令之一来访问 **LMTOOLS** 工具箱：

```
abaqus licensing lmtools

*or*

*simulia_dir*/License/lmtools
```

### 3.6.1 lmdiag

`lmdiag` 工具允许您在无法检出许可证时诊断问题。

**语法和选项**

`lmdiag` [`-c` *license_file_list*] [`-n`] [*feature*[:*keyword=value*]]

**`-c *license_file_list*`**

诊断指定文件。

**`-n`**

以非交互模式运行；在此模式下，`lmdiag` 不会提示任何输入。在此模式下，扩展连接诊断（见下文）不可用。

**`*feature*`**

仅诊断此功能。

**`*keyword*=*value*`**

如果许可证文件包含某功能的多个行，您可以选择 `lmdiag` 报告的特定行。例如，

```
lmdiag f1:HOSTID=12345678
```
 尝试在具有 hostid "12345678" 的行上检出。*keyword* 可以是以下之一：
- VERSION
- HOSTID
- EXPDATE
- KEY
- VENDOR_STRING
- ISSUER

如果未指定 *feature*，`lmdiag` 会对列表中许可证文件中的所有功能进行操作。`lmdiag` 首先打印有关许可证的信息，然后尝试检出每个许可证。如果检出成功，`lmdiag` 会指示这一点。如果检出失败，`lmdiag` 会给出失败原因。如果检出失败是因为 `lmdiag` 无法连接到许可证服务器，您可以选择运行"扩展连接诊断"。

这些扩展诊断尝试连接到许可证服务器计算机上的每个 TCP/IP 端口，并检测许可证文件中的端口号是否不正确。`lmdiag` 指示每个正在监听的 TCP/IP 端口号。如果是 `lmgrd` 进程，`lmdiag` 也会指示。如果 `lmdiag` 找到被测功能的供应商守护程序，它会指示许可证文件的正确端口号以纠正问题。

请参阅 FLEXnet Licensing End User Guide Version 11.6 获取其他信息。此指南可从 **Support** 页面上的 **Licensing** 部分下载，网址为 [www.3ds.com/simulia](http://www.3ds.com/simulia)。

**使用 **LMTOOLS**（Windows 平台）运行 `lmdiag`：**

1. 从**开始**菜单，选择****程序**![](../graphics/images/arrow.gif)**Abaqus 许可证**![](../graphics/images/arrow.gif)**许可证实用程序****。
2. 在**服务/许可证文件**选项卡页上验证已启用**使用服务配置**，然后选择**服务器诊断**选项卡。
3. 单击**执行诊断**。

### 3.6.2 lmdown

`lmdown` 工具允许在所有计算机上正常关闭选定的许可证守护程序（`lmgrd` 和选定的供应商守护程序）。

**语法和选项**

`lmdown` `-c` *license_file_list* [`-vendor` *vendor*] [`-q`] [`-all`]

**`-c *license_file_list*`**

使用指定的许可证文件。始终建议使用 `-c` *license_file_list* 指定 `lmdown`。

**`-vendor *vendor*`**

仅关闭此供应商守护程序。`lmgrd` 继续运行。

**`-q`**

不提示或打印标题。否则，`lmdown` 会询问"您确定吗？[y/n]："。

**`-all`**

如果指定了多个服务器，自动关闭所有服务器。使用 `-all` 时会暗示 `-q`。

您可以在启动许可证管理器守护程序 `lmgrd` 时保护未经授权的 `lmdown` 执行。关闭服务器会导致用户失去其许可证。请参阅["FLEXnet 许可证服务器管理器 lmgrd，"第 3.5 节](ch03s05.md)中的 `-local`、`-2`、`-p` 和 `-x` 选项，获取有关保护 `lmdown` 访问的详细信息。

如果 `lmdown` 遇到多个服务器（例如，如果 `-c` 指定包含许多 `*.lic` 文件的目录），则会显示要关闭的许可证服务器选择。

**注意：**在 UNIX 平台上，不要使用 `kill -9` 关闭许可证服务器。在 Windows 平台上，如果您必须使用 **任务管理器**终止 FLEXnet Licensing 服务，请先结束 `lmgrd` 进程，然后结束所有供应商守护程序进程。

要停止和重启单个供应商守护程序，请使用 `lmdown` `-vendor` *vendor*，然后使用 `lmreread` `-vendor` *vendor* 重启供应商守护程序（请参阅["lmreread，"第 3.6.6 节](ch03s06.md#sgb-chp-net-lic-lmreread)）。

关闭三服务器冗余许可证服务器时，服务器关闭前一分钟会有延迟。`lmdown` 关闭一组冗余许可证服务器的所有三个许可证服务器。如果需要关闭一组冗余许可证服务器中的一台（不推荐，因为如果任一剩余机器变得不可用，许可证服务器将停止提供许可证），则必须在该许可证服务器计算机上终止 `lmgrd` 和供应商守护程序进程。

**使用 **LMTOOLS**（Windows 平台）运行 `lmdown`：**

1. 从**开始**菜单，选择****程序**![](../graphics/images/arrow.gif)**Abaqus 许可证**![](../graphics/images/arrow.gif)**许可证实用程序****。
2. 在**服务/许可证文件**选项卡页上验证已启用**使用服务配置**，然后选择**启动/停止/重新读取**选项卡。
3. 单击**停止服务器**。

### 3.6.3 lmhostid

`lmhostid` 工具返回当前平台的 FLEXnet Licensing host id。

**语法和选项**

`lmhostid` [`-n`] 

**`-n`**

仅返回作为字符串的 host id，适合与许可证文件中的 `HOSTID=` 一起使用。标题文本被抑制。

**示例**

以下是 `lmhostid` 输出的示例：


2. 在**服务/许可证文件**选项卡页上验证已启用**使用服务配置**，然后选择**系统设置**选项卡。host id 列在**以太网地址**下。

### 3.6.4 lmpath

`lmpath` 工具允许直接控制 FLEXnet 许可证路径设置。它最有用 于检查诊断目的的当前许可证路径设置。

**语法和选项**

`lmpath` {`-add` | `-override`} {*vendor* | `all`} *license_file_list*

**`-add`**

将 *license_file_list* 前置到当前许可证文件列表，或创建许可证文件列表（如果不存在），并将其初始化为 *license_file_list*。重复项被丢弃。

**`-override`**

用 *license_file_list* 覆盖现有许可证文件列表。如果 *license_file_list* 是空字符串 (`""`)，则删除指定列表。
- `lmpath -override all ""` 删除 `LM_LICENSE_FILE` 的值。
- `lmpath -override *vendor* ""` 删除 `*VENDOR*_LICENSE_FILE` 的值

**`*vendor*`**

供应商守护程序名称。影响 `*VENDOR*_LICENSE_FILE` 的值。

**`all`**

指所有供应商守护程序。影响 `LM_LICENSE_FILE` 的值。

**`*license_file_list*`**

在 Linux 平台上是用冒号分隔的列表，在 Windows 平台上是用分号分隔的列表。如果 *license_file_list* 是空字符串 (`""`)，则删除指定条目。

**注意：**`lmpath` 通过在 Linux 平台上设置 `$HOME/.flexlmrc` 以及在 Windows 平台上设置 FLEXnet Licensing 注册表项来工作。

要显示当前许可证路径设置，请使用命令

```
lmpath -status
```
显示以下信息：
```
lmutil - Copyright (C) 1989-2008 Acresso Software
Known Vendors:
_____________
demo: ./counted.lic:./uncounted.lic
_____________
Other Vendors:
_____________
/usr/local/flexlm/licenses/license.lic
```
如果路径设置为目录，则所有 `*.lic` 文件会单独列出。

**使用 **LMTOOLS**（Windows 平台）运行 `lmpath`：**

1. 从**开始**菜单，选择****程序**![](../graphics/images/arrow.gif)**Abaqus 许可证**![](../graphics/images/arrow.gif)**许可证实用程序****。
2. 在**服务/许可证文件**选项卡页上验证已启用**使用服务配置**，然后选择**实用程序**选项卡。
3. 单击**列出所有供应商路径**。

### 3.6.5 lmremove

`lmremove` 工具允许您删除单个用户对指定功能的许可证。如果应用程序处于活动状态，它会在 `lmremove` 释放许可证后不久重新检出许可证。如果 Abaqus 进程异常终止，即使不再需要令牌，它也可能不会将许可证令牌返回到许可证池。在这种情况下，`lmremove` 可用于将 Abaqus/CAE 和 Abaqus/Viewer 令牌返回到许可证池。*user*、*user_host*、*display*、*server_host*、*port* 和 *handle* 信息必须从 `lmstat -a` 的输出中获取。`lmremove` 工具不应用于将 Abaqus 分析作业令牌返回到许可证池；请参阅[附录 F，"排除 Abaqus FLEXnet 许可证故障"](ap06.md)，了解更多信息。

**注意：**`lmremove` 工具不会释放许可证供其他作业使用。要临时释放许可证供其他作业使用，可以暂停正在运行的作业（使用 Abaqus **suspend** 实用程序；请参阅["作业执行控制，"Abaqus Analysis User's Guide 第 3.2.39 节](../usb/usb-link.md#usb-int-dsuspend)，了解详细信息）。可以使用 Abaqus **terminate** 实用程序或适当的操作系统实用程序终止正在运行的作业来停止分析作业的可执行文件。有关使用 `lmremove` 工具的示例，请参阅["lmstat，"第 3.6.7 节](ch03s06.md#sgb-chp-net-lic-lmstat)。

**语法和选项**

`lmremove` [`-c` *license_file_list*] *feature* *user* *user_host* *display* 或 `lmremove` [`-c` *license_file_list*] `-h` *feature* *server_host* *port* *handle*

**`-c *license_file_list*`**

指定许可证文件。

**`*feature*`**

用户检出的功能名称。

 **`*user*`**

要删除其许可证的用户名，如 `lmstat -a` 所报告。

**`*user_host*`**

用户登录的主机名，如 `lmstat -a` 所报告。

**`*display*`**

用户正在工作的显示名称，如 `lmstat -a` 所报告。

**`*server_host*`**

许可证服务器运行所在的主机名，如 `lmstat -a` 所报告。

**`*port*`**

许可证服务器运行的 TCP/IP 端口号，如 `lmstat -a` 所报告。

**`*handle*`**

许可证句柄，如 `lmstat -a` 所报告。

`lmremove` 工具从 *user* on *user_host* 和 *display* 对 *feature* 的使用中移除所有实例。如果指定了 `-c *license_file_list*` 选项，则使用指示的文件作为许可证文件。`-h` 变体使用 `lmstat -a` 所报告的 *server_host*、*port* 和许可证 *handle*。

### 3.6.6 lmreread

`lmreread` 工具使许可证管理器重新读取许可证文件并启动任何已添加的新供应商守护程序。此外，所有当前运行的供应商守护程序都会收到信号以重新读取许可证文件及其最终用户选项文件，以获取功能许可证信息或选项设置的更改。如果启用报告日志记录，则会刷新供应商守护程序内部数据缓冲区中仍存在的任何报告日志数据。`lmreread` 识别服务器计算机主机名的更改，但不能用于更改服务器 TCP/IP 端口号。

如果指定了可选的供应商守护程序名称，则只有命名的守护程序重新读取许可证文件及其最终用户选项文件（在这种情况下 `lmgrd` 不会重新读取许可证文件）。

**语法和选项**

`lmreread` [`-c` *license_file_list*] [`-vendor` *vendor*] [`-all`]

**`-c *license_file_list*`**

使用指定的许可证文件。

**`-vendor *vendor*`**

仅此供应商守护程序重新读取许可证文件。`lmgrd` 在必要时重启供应商守护程序。

**`-all`**

如果指定了多个 `lmgrd`，则指示所有 `lmgrd` 重新读取。

您可能需要保护 `lmreread` 的执行。请参阅["FLEXnet 许可证服务器管理器 lmgrd，"第 3.5 节](ch03s05.md)中的 `-2`、`-p` 和 `-x` 选项，获取有关保护 `lmreread` 访问的详细信息。

要停止和重启单个供应商守护程序，请使用 `lmdown` `-vendor` *vendor*，然后使用 `lmreread` `-vendor` *vendor`，这会重启供应商守护程序。

**注意：**如果使用 `-c` *license_file_list* 选项，则 `lmreread` 会读取指定的许可证文件，而不是 `lmgrd`；`lmgrd` 会重新读取它最初读取的文件。

**使用 **LMTOOLS**（Windows 平台）运行 `lmreread`：**

1. 从**开始**菜单，选择****程序**![](../graphics/images/arrow.gif)**Abaqus 许可证**![](../graphics/images/arrow.gif)**许可证实用程序****。
2. 在**服务/许可证文件**选项卡页上验证已启用**使用服务配置**，然后选择**启动/停止/重新读取**选项卡。
3. 选择 **Abaqus 6.14 FLEXnet License Manager**，然后单击**重新读取许可证文件**。

### 3.6.7 lmstat

`lmstat` 工具帮助您监控所有 FLEXnet 网络许可证活动的状态，包括：
- 正在运行的守护程序
- 许可证文件
- 各功能的用户
- 由特定供应商守护程序提供功能的用户

`lmstat` 工具打印从许可证服务器接收的信息；因此，它不报告未提供的许可证（如未计数许可证）。要报告未计数许可证，必须将许可证添加到提供的许可证文件中，并且应用程序必须被定向为该许可证文件使用许可证服务器（通过 *@host, port@host,* 或 `USE_SERVER`）。排队的用户和因重复分组而共享的许可证也不会由 `lmstat` 返回。

**语法和选项**

`lmstat` [`-a`] [`-c` *license_file_list*] [`-f` [*feature*]] [`-i` [*feature*]] [`-s` [*server*]][`-S` [*vendor*]] [`-t` *timeout_value*]

**`-a`**

显示所有信息。

**`-c *license_file_list*`**

指定要使用的许可证文件。

**`-f [*feature*]`**

显示 *feature* 的用户。如果未指定 *feature*，则显示所有功能的用法信息。

**`-i [*feature*]`**

显示指定 *feature* 或所有功能（如果未指定 *feature*）的 `FEATURE/INCREMENT` 行中的信息。

**`-s [*server*]`**

显示 `$VENDOR_LICENSE_FILE` 或 `$LM_LICENSE_FILE` 中所有许可证文件在 *server* 上或所有服务器上（如果未指定 *server*）的状态。

**`-S [*vendor*]`**

列出指定供应商功能的所有用户。

**`-t *timeout_value*`**

设置连接超时为 *timeout_value*。这限制了 `lmstat` 尝试连接到 *server* 所花费的时间。

**注意：**`lmstat -a` 命令是可能代价高昂的命令。对于许多活跃用户，此命令会产生大量网络活动。

`lmremove` 工具需要 `lmstat -a` 命令的输出，如下例所示。

**示例**

命令

```
*simulia_dir*/License/lmstat -a
```
的输出（其中 *simulia_dir* 是 SIMULIA 父目录）类似于以下内容：
```
License server status: 27000@firestar
    License file(s) on firestar: *simulia_dir*/License/simulialm.lic:
   firestar: license server UP (MASTER) v11.6
Vendor daemon status (on firestar):
 ABAQUSLM: UP v11.6
Feature usage info:
Users of cae: (Total of 4 licenses issued; Total of 1 license
in use)
  "cae" v61.2, vendor: ABAQUSLM
  floating license
    smith watt watt (v61.2) (firestar/27000 101), 
start Tue 3/1 9:29
```
其中

| `smith` | *user* | 用户名 |
| --- | --- | --- |
| `watt` | *user_host* | 用户运行的主机 |
| `watt` | *display* | 用户运行的显示 |
| `v61.2` | *release* | 功能版本 |
| `firestar` | *server_host* | 许可证服务器运行的主机 |
| `27000` | *port* | 许可证服务器运行的 *server_host* 上的端口 |
| `101` | *handle* | 许可证句柄 |
| `start Tue 3/1 9:29` | *checkout_time* | 检出此许可证的时间 |

**注意：**`lmremove` 工具不会释放许可证供其他作业使用（请参阅["lmremove，"第 3.6.5 节](ch03s06.md#sgb-chp-net-lic-lmremove)，了解更多信息）。

要使用 `lmremove` 工具释放用户 `smith` 运行的作业的许可证，您可以使用命令
```
abaqus licensing lmremove cae smith watt watt
```
其中

| `cae` | *feature* | 用户检出的功能名称 |
| --- | --- | --- |
| `smith` | *user* | 要删除其许可证的用户名，如 `lmstat -a` 所报告 |
| `watt` | *user_host* | 用户登录的主机名，如 `lmstat -a` 所报告 |
| `watt` | *display* | 用户工作的显示名称，如 `lmstat -a` 所报告 |

**使用 **LMTOOLS**（Windows 平台）运行 `lmstat -a`：**

1. 从**开始**菜单，选择****程序**![](../graphics/images/arrow.gif)**Abaqus 许可证**![](../graphics/images/arrow.gif)**许可证实用程序****。
2. 在**服务/许可证文件**选项卡页上验证已启用**使用服务配置**，然后选择**服务器状态**选项卡。
3. 验证已启用**显示所有**，然后单击**执行状态查询**。

### 3.6.8 将 FLEXnet 许可证安装为 Windows 服务

您必须以 Administrator 身份登录才能将许可证安装为 Windows 服务。您使用安装过程安装许可证管理工具。您必须使用以下过程来安装和启动服务：

1. 要从**开始**菜单访问 **LMTOOLS**，请选择****程序**![](../graphics/images/arrow.gif)**Abaqus 许可证**![](../graphics/images/arrow.gif)**许可证实用程序****。
2. 在**服务/许可证文件**选项卡页上验证已启用**使用服务配置**，然后选择**配置服务**选项卡。
3. 在**服务名称**文本字段中，键入 `Abaqus 6.14 FLEXnet License Manager`。
4. 指定 `lmgrd.exe` 文件、许可证文件和调试日志文件的路径，如[图 3-1](ch03s06.md#sgb-lmtools-service)所示。**图 3-1****LMTOOLS** 的**配置服务**选项卡页面。 ![](../graphics/sgb-lmtools-service-nls.png)
5. 切换**使用服务**。**启动服务器 at Power Up** 变得可用。
6. 切换**启动服务器 at Power Up**，然后单击**保存服务**。
7. 选择**启动/停止/重新读取**选项卡，然后单击**启动服务器**。
8. 从主菜单栏中，选择****文件**![](../graphics/images/arrow.gif)**退出****以关闭对话框。

### 3.6.9 使用 **LMTOOLS** 启动 FLEXnet 服务器

您必须以 Administrator 身份登录才能启动服务器。

1. 要从**开始**菜单访问 **LMTOOLS**，请选择****程序**![](../graphics/images/arrow.gif)**Abaqus 许可证**![](../graphics/images/arrow.gif)**许可证实用程序****。
2. 在**服务/许可证文件**选项卡页上验证已启用**使用服务配置**，然后选择**启动/停止/重新读取**选项卡。
3. 单击**启动服务器**。
4. 从主菜单栏中，选择****文件**![](../graphics/images/arrow.gif)**退出****以关闭对话框。
