# 4.3.4 调整视口大小





您可以通过拖动视口的边框来更改视口的大小和形状。

**要调整视口大小：**

1. 将光标放在要移动的视口边框上的任意位置。光标变为一对相反方向的箭头。箭头的方向取决于光标与视口的相对位置。
2. 单击鼠标按钮1，并拖动光标以更改边框的位置。
3. 释放鼠标按钮1。视口以新大小显示，如果之前不是当前视口，它会成为当前视口。
