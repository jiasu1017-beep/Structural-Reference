# 15.16.2 执行接触对搜索





一旦您指定了适当的接触标准，Abaqus/CAE就会搜索模型以定位满足这些标准的表面。有关Abaqus/CAE如何解释搜索标准并检测接触对候选者的讨论，请参见["接触检测算法，"第15.6.2节"](pt03ch15s06s02.md)。

**要执行接触对搜索：**

1. 从主菜单，选择**交互**![](../graphics/images/arrow.gif)**查找接触对**或**约束**![](../graphics/images/arrow.gif)**查找接触对**。**提示：**您也可以使用交互模块工具箱中的![](../graphics/ico_contactPairFind.png)工具来查找接触对。Abaqus/CAE显示接触检测对话框。
2. 按照["指定接触检测的搜索条件，"第15.16.1节"](pt03ch15s16s01.md)中所述输入搜索参数。
3. 点击**查找接触对**。
4. Abaqus/CAE搜索潜在的接触对。当搜索完成时，Abaqus/CAE填充接触对候选者表。要在搜索完成前中止搜索，请在提示区域点击**停止**。

- ["使用接触和约束检测，"第15.16节](pt03ch15s16.md)
