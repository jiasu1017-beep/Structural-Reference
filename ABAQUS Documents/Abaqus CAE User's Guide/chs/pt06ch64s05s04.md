# 64.5.4 投影节点





您可以将来自网格化零件、装配中的网格化零件实例或孤兒网格的一个或多个所选节点投影到选定的顶点、边缘、面、节点、单元边缘、单元面、基准点、基准轴或基准平面上。如果您想将自底向上网格中的节点移动到附近的几何体上，这特别有用。通常，您不能使用网格编辑工具集修改从属零件实例的网格；但是，您可以投影从属零件实例的节点。Abaqus/CAE投影原始网格的节点，您的修改会显示在该零件的所有从属实例上。

投影节点不会改变它们的关联。投影前与几何特征关联的节点在投影后将保持与该特征的关联；未关联的节点在您将它们投影到几何体上后仍将保持未关联状态。

**投影节点的步骤：**

1. 进入"网格"模块，并执行以下操作之一： - 从上下文栏的**对象**字段中选择**装配**。 - 从上下文栏的**对象**字段中选择**零件**，然后从列表中选择一个零件。
2. 从主菜单栏选择****网格**![](../graphics/images/arrow.gif)**编辑****。Abaqus/CAE显示**编辑网格**对话框。 **提示：**您也可以使用位于"网格"模块工具箱底部的![](../graphics/ico_meshEdit.png)工具显示**编辑网格**对话框。
3. 在对话框中执行以下操作： 1. 在**类别**字段中选择**节点**。 2. 从**方法**列表中选择**投影**。Abaqus/CAE显示现有节点。在隐藏和着色模式下，仅显示可见节点。
4. 从提示区域选择选择方法并选择要投影的节点。有关更多信息，请参见["选择要编辑的节点，" 第64.5.10节](pt06ch64s05s10.md)。 **提示：**您可以使用**选择**工具栏更改选择行为。有关更多信息，请参见[第6章，"在视口中选择对象"](pt01ch06.md)。
5. 选择顶点、边缘、面、节点、单元边缘、单元面、基准点、基准轴或基准平面，Abaqus/CAE会将所选节点移动到该位置。Abaqus/CAE将所选实体着为洋红色。
6. 在提示区域选择**是**以确认您的选择。Abaqus/CAE使用最短距离算法将所选节点投影到它们的新位置。如果您选择了边缘、面、单元边缘或单元面，Abaqus/CAE会将节点投影到通过扩展所选边缘或面创建的轴或平面上。 **提示：**如果在编辑节点时出错，点击**编辑网格**对话框中的**撤销**可以撤消编辑。Abaqus/CAE会将节点恢复到其原始位置。如果选择**否**，则会取消您的节点和实体选择，过程将从步骤4继续。
7. 根据需要重复上述步骤以投影其他节点。
