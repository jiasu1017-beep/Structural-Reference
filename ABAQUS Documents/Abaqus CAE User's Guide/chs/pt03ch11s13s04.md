# 11.13.5 为轴对称零件和旋转特征定义旋转轴





创建轴对称零件以及向零件添加旋转特征时，轮廓的草图必须包含定义旋转轴的构造线。以下规则适用于草图和构造线：

**创建轴对称零件**

您可以通过从主菜单栏选择**零件**![](../graphics/images/arrow.gif)**创建**来创建由壳或线框以及对称轴定义的轴对称零件。创建轴对称零件时，Abaqus/CAE允许您在模型中包含扭转自由度。

绘制零件基础特征的草图时，Abaqus/CAE会在草图的*Y*轴上显示一条垂直构造线，表示对称轴。您必须仅在该线的右侧绘制草图。您的草图可以接触此线但不能穿过它。

您只能向轴对称基础特征添加壳和线框特征。添加特征时，Abaqus/CAE会显示原始草图和构造线，相同的规则适用——您不能删除此构造线，必须仅在其右侧绘制草图。

**创建旋转特征**

您可以通过从主菜单栏选择**零件**![](../graphics/images/arrow.gif)**创建**来创建具有旋转实体或旋转壳基础特征的三维零件。类似地，您可以通过从主菜单栏选择**形状**![](../graphics/images/arrow.gif)**实体**![](../graphics/images/arrow.gif)**旋转**、**形状**![](../graphics/images/arrow.gif)**壳**![](../graphics/images/arrow.gif)**旋转**或**形状**![](../graphics/images/arrow.gif)**切割**![](../graphics/images/arrow.gif)**旋转**来向三维实体和壳添加旋转实体、壳和切割。

旋转特征的草图必须包含表示旋转轴的构造线。创建新的旋转零件时，Abaqus/CAE会通过草图编辑器网格的原点创建一条垂直构造线。如果需要，您可以删除此构造线并以新的角度和位置重新绘制。相比之下，当向现有零件添加旋转特征时，必须绘制表示旋转轴的构造线。您可以在构造线的右侧或左侧绘制草图。您的草图可以接触此线但不能穿过它。如果完成的草图包含多条构造线，Abaqus/CAE会提示您选择将代表旋转轴的线。

绘制表示旋转轴的构造线时，如果底层零件存在基准轴，您可以通过选择基准轴来定位构造线。您不能直接选择基准轴；必须从基准轴的任一端选择点。您可以使用基准轴创建同心特征。

退出草图编辑器时，Abaqus/CAE会打开一个对话框以完成旋转特征的定义。您输入轮廓将旋转的角度和旋转方向，并且可以选择是否通过包含pitch来沿旋转轴平移轮廓。您还可以指定平移方向。pitch值是轮廓在360旋转期间将平移的距离。Pitch允许创建诸如螺旋弹簧和诸如螺纹的零件细节。

如果要创建复杂形状，其中草图轮廓被拉伸而不是旋转（例如扭曲电缆或螺旋齿轮），可以在拉伸实体、壳或切割特征中包含扭转。有关所有可用特征类型的基本信息，请参见"可以创建哪些类型的特征？"，第11.9节；有关拉伸特征的更多信息，请参见"定义拉伸距离"，第11.13.1节。![](../graphics/images/black4rule.gif)有关相关信息，请单击以下任一项：- "添加拉伸实体特征"，第11.21.1节- "添加旋转实体特征"，第11.21.2节- "添加拉伸壳特征"，第11.22.1节- "添加旋转壳特征"，第11.22.2节- "创建拉伸切割"，第11.24.1节- "创建旋转切割"，第11.24.3节
