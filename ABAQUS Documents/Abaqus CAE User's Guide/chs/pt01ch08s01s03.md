# 8.1.4 EPS、TIFF、PNG 和 SVG 图像大小





当你将所选视口的快照打印到封装的 PostScript (EPS)、TIFF、PNG 或 SVG 格式文件时，Abaqus/CAE 根据你指定的大小和视口的整体纵横比来确定图像大小。你可以通过在画布上操作视口来控制纵横比。

在选项对话框（**EPS 选项**、**TIFF 选项**、**PNG 选项**或 **SVG 选项**）中，你可以选择以下方法之一来指定打印图像的大小：
- 使用屏幕上的图像大小。（Abaqus/CAE 在选项对话框中指示当前图像大小。）此方法为默认方法。
- 设置宽度或高度。你只指定一个维度；Abaqus/CAE 计算另一个维度以保持视口的纵横比。当你创建 EPS 格式文件时，你可以用英寸或毫米指定宽度或高度。当你创建 TIFF、PNG 或 SVG 格式文件时，你用屏幕像素指定宽度或高度；增加像素数会增加图像大小。允许的最大图像大小为 1280 1024 像素。
