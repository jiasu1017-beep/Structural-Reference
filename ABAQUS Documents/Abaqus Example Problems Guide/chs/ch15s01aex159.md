# 15.1.6 创建便于后处理弯头单元结果的数据文件：FELBOW





**产品：** Abaqus/Standard

本示例说明了使用FORTRAN程序从Abaqus结果文件读取选定的单元积分点记录，以便于后处理弯头单元结果。创建适合在Abaqus/CAE中使用*X–Y*绘图功能的*X–Y*数据。

### 概述

本程序展示了如何从Abaqus结果文件检索弯头单元的积分点数据，以可视化以下之一：

1. 沿弯头单元直线的变量变化，
2. 给定弯头单元圆周上的变量变化，或
3. 给定弯头单元的椭圆化。

创建一个包含*X–Y*数据的ASCII文件，可以读入Abaqus/CAE用于可视化。

要执行选项1，弯头单元必须按单调递增的顺序编号；在所考虑范围内的所有单元必须是弯头单元。*X–Y*数据将使用弯头单元直线的距离作为*X*数据（沿弯头中心线测量），变量值作为*Y*数据创建。如果需要选项2或3，用户必须确保积分点坐标（COORD）被写入结果文件。对于选项2，*X*数据是沿弯头单元圆周的距离（沿中面测量），*Y*数据是变量值。对于选项3，*X–Y*数据是弯头单元圆周上中面积分点的当前坐标，投影到变形截面平面中的局部坐标系。局部系统的原点与截面中心重合；变形截面平面定义为包含截面中心和积分点1和2的平面。

### 编程细节

系统会提示用户输入结果文件的名称（假定为二进制）和后处理选项（1、2或3）。然后根据所选选项提示用户输入其他信息，包括
- 单元编号范围（选项2和3仅需要一个单元编号），
- 截面点编号（仅选项1和2），
- 积分点编号（仅选项1），
- 单元变量（仅选项1和2），
- 变量分量（如["结果文件输出格式，" Abaqus Analysis User's Guide的第5.1.2节](../usb/usb-link.md#usb-out-fformat)所定义，仅选项1和2），
- 步编号，以及
- 增量编号。

数据通过调用`DBFILE`在双重`DO`循环中处理所有记录。所需数据存储在变量`VAR`中；积分点坐标存储在`COORDS`中。程序检查以确保结果文件中有所需的数据。如果用户尝试处理结果文件中未找到的数据，则会发出错误。

### 程序编译和链接

在程序执行之前，使用**abaqus make**工具编译和链接FORTRAN程序：

```
abaqus make job=felbow
```

重复此命令，直到所有FORTRAN错误被纠正。成功编译后，程序的目标代码会自动与存储在共享程序库和接口库中的Abaqus目标代码链接，以构建可执行程序。请参阅[Abaqus Installation and Licensing Guide](../sgb/sgb-link.md#sgb-chp-environment)的第4章"自定义Abaqus环境"，以查看特定计算机使用的编译和链接命令。

### 程序执行

在执行程序之前，运行一个创建包含适当输出的结果文件的分析。此分析包括，例如，指定范围内单元的输出以及单元的积分点坐标。当使用命令`abaqus felbow`执行程序时，会出现提示

```
Enter the name of the input file (w/o .fil):
```

输入结果文件的名称以定义`FNAME`。然后系统会提示用户输入其他信息，例如所需的后处理选项、单元编号等。程序处理数据并生成名为`output.dat`的文件，其中包含可视化弯头单元结果所需的信息。
