# 12.12 在链接视口上执行显示组布尔运算





**产品：** Abaqus/CAE

**优点：** 在可视化模块中，您现在可以在所有链接视口中同时执行选定的显示组操作。创建显示组时，您还可以显示链接视口之间的共同项目列表。

**描述：** 您现在可以在所有链接视口中同时执行选定的显示组操作。如果您通过名称或标签为显示组选择特定项目，则在所有共享指定项目的链接视口中执行相同的显示组操作。此选项仅适用于可视化模块。

创建显示组时，您现在可以选择在**方法**列表中显示所有链接视口的共同项目。对于**零件实例**，考虑包含模型数据库或输出数据库的视口；对于其他模型组件，仅考虑包含输出数据库的视口。

**Abaqus/CAE使用方法：**
```
可视化模块：
    ****视口**![](../graphics/images/arrow.gif)**链接视口****：切换 **链接视口**，切换 **显示组**
    ****工具**![](../graphics/images/arrow.gif)**显示组**![](../graphics/images/arrow.gif)**创建****：切换 **显示链接视口之间的共同列表**
```
