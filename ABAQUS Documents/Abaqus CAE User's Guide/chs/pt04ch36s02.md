# 36.2 定义桁条加固





桁条加固定义粘合到现有部件边缘的桁条并指定其工程属性。您可以选择三维实体部件的边缘或二维平面部件的边缘。部件可以包括几何体和孤立网格单元；但是，必须分别为几何体和孤立单元定义桁条加固。您应该将桁条视为部件或区域的属性，就像截面是部件或区域的属性一样。

[图36-2](pt04ch36s02.md#prp-prop-stringer)所示的钢筋加固梁是如何在模型中使用桁条加固的示例。

**图36-2** 带钢筋桁条加固的混凝土梁。

![](../graphics/prp-prop-stringer-nls.png)

梁具有沿梁长度延伸的四个圆柱形挤压件的混凝土芯。在每个挤压件中创建了具有圆形轮廓的钢桁条，以沿梁的长度提供支撑。您可以创建表示梁的实体部件，并添加四个表示钢筋加固的桁条加固。在Mesh模块中，您为混凝土分配实体单元，为桁条分配线单元。实体和线单元共享相同的节点。

从Property模块的主菜单栏中选择****Special**![](../graphics/images/arrow.gif)**Stringer**![](../graphics/images/arrow.gif)**Create****以定义一个或多个桁条。从同一菜单中选择**Edit**以对现有定义进行更改。您创建的所有桁条也出现在Model Tree中部件下的**Stringers**容器中。不同的桁条可以共享相同的截面，并且可以在部件的边缘上放置多个桁条。默认情况下，桁条不在视口中显示，但您可以通过在视口中对它们进行颜色编码来使其可见。更多信息，请参见["对几何体和网格单元进行颜色编码"，第77.4节](pt07ch77s04.md)。

如果在几何体区域上创建桁条，Abaqus/CAE会在对底层几何体进行微小修改时更新桁条。如果使用桁条编辑孤立节点或单元，Abaqus/CAE会在您编辑或删除节点或单元时更新桁条；但是，如果在创建新节点或单元时不会更新桁条。

在后续建模操作中可能需要选择桁条；例如：
- 为桁条分配截面、梁截面方向、材料方向或切线方向。所有这些活动都在Property模块中进行。
- 在Load模块中为桁条施加体力或线载荷。
- 在Load模块中在桁条上施加热通量。实际上，您可以通过向底层边缘或边缘施加热通量载荷来执行此建模操作。Abaqus/CAE会在分析期间将该载荷应用于这些边缘上的所有桁条。
- 在Mesh模块中为桁条分配单元类型。
- 在Step模块中请求桁条的场数据输出或历史数据输出。
- 创建显示组以在Visualization模块中查看桁条单元上的应力值。虽然不能在**Create Display Group**对话框中按名称专门选择桁条，但可以通过搜索与模型中桁条单元具有相同单元类型、截面分配或其他属性的单元来在对话框中找到它们。此过程可以帮助您将单元列表缩小到所需的桁条。

当提示您为此类建模操作选择区域时，从**Selection**工具栏的对象类型列表中选择**Stringers**，然后从视口中选择桁条。更多信息，请参见["根据对象类型过滤您的选择"，第6.3.2节](pt01ch06s03s01.md)。

执行接触计算时，Abaqus/CAE仅在某些情况下考虑几何表面的桁条加固；例如，当您为所有外表面指定一般接触时，会在Abaqus中自动定义一个将考虑桁条加固的默认全包容表面。桁条加固可能会显著影响接触计算，这由于桁条厚度以及对接触惩罚刚度等数值量的潜在影响。要在接触定义中明确包含桁条加固，您应该在桁条加固上创建一个集合，然后在接触定义中使用该集合。

有关创建桁条的详细信息，请参见["创建和编辑桁条加固"，第36.9节](pt04ch36s09.md)。
