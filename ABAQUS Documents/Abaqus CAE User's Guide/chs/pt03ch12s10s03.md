# 12.10.4 指定焦耳热分数







当流过导体的电流所耗散的能量转换为热能时，就会产生焦耳加热。焦耳热分数是在耦合热-电问题中作为热量释放的耗散电能的分数。

有关更多信息，请参见以下章节：
- "耦合热-电分析"，Abaqus分析用户指南第6.7.3节
- "耦合热-电分析"中的"指定因电流产生的热能量"，Abaqus分析用户指南第6.7.3节

**指定焦耳热分数：**

1. 从**编辑材料**对话框的菜单栏中，选择**热**![](../graphics/images/arrow.gif)**焦耳热分数**。（有关显示**编辑材料**对话框的信息，请参见"创建或编辑材料"，第12.7.1节。）
2. 在**分数**字段中，输入作为热量释放的电能分数，包括任何单位转换因子。默认值为1.0。
3. 单击**确定**关闭**编辑材料**对话框。或者，您可以从**编辑材料**对话框的菜单中选择其他材料行为进行定义（有关更多信息，请参见"浏览和修改材料行为"，第12.7.2节）。
