# 17.8.2 什么是映射网格划分？






在有限元分析文献中，结构网格划分和映射网格划分这两个术语可以互换使用。然而，Abaqus/CAE 对这两个术语做了细微的区分。映射网格划分是结构网格划分的一个子集，仅指四边二维区域的规则网格划分——即[图 17-42](pt03ch17s08s01.md#mgn-mapped1) 所示的方形网格模式。

某些看起来非常复杂的模型实际上包含几何形状相对简单的面。当您使用自由网格划分或扫掠网格划分对这样的模型进行网格划分时，这些面上的单元质量可能很差。但是，如果您允许 Abaqus/CAE 在几何合适的地方应用映射网格划分技术，通常会生成高质量的单元，特别是在区域是狭长的矩形面时。

您不能直接应用映射网格划分。但是，您可以通过对区域进行网格划分并允许 Abaqus/CAE 在适当的地方应用映射网格划分来间接应用。例如，[图 17-43](pt03ch17s08s02.md#mgn-mapped-withandwithout) 展示了自由网格划分一个零件并允许 Abaqus/CAE 在适当的地方使用映射网格划分的效果。

**图 17-43** 允许映射网格划分的效果。

![](../graphics/mgn-mapped-withandwithout-nls.png)

默认情况下，当您执行以下操作时，Abaqus/CAE 会在适当的地方使用映射网格划分：
- 使用推进前沿算法对实体区域使用六面体或六面体主导单元进行扫掠网格划分。
- 使用推进前沿算法对壳区域使用四边形或四边形主导单元进行自由网格划分。
- 对实体区域使用四面体单元进行自由网格划分。
- 对壳区域使用三角形单元进行自由网格划分。
