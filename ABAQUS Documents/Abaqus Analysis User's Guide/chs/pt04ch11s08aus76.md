# 11.8 稳态检测

**产品：** Abaqus/Standard Abaqus/Explicit

##### **参考**

- [*STEADY STATE DETECTION](../key/key-link.md#usb-kws-msteadystatedetection)

### 概述

稳态检测是一种用于确定准静态分析何时达到稳态响应的技术。当结构响应从瞬态逐渐过渡到稳态时，稳态检测可以自动终止分析，提高计算效率。

### 稳态检测原理

准静态分析中，结构响应通常包含：
- **瞬态响应**：加载初期的高频振荡和快速变化
- **稳态响应**：长期加载下的渐近行为

稳态检测通过监控响应量（如位移、速度、加速度）的变化率来判断是否达到稳态。

### 检测方法

#### 基于能量的检测

监控能量指标的变化率：
- 内能变化率
- 动能变化率
- 外力功变化率

当这些指标的变化率低于指定阈值时，认为达到稳态。

#### 基于响应的检测

监控节点位移或应力等响应量的变化率。当响应量变化率低于指定阈值时，认为达到稳态。

#### 基于频率的检测

通过傅里叶分析检测响应中高频成分的衰减。当高频成分衰减到指定水平时，认为瞬态响应结束。

### 定义稳态检测

使用**STEADY STATE DETECTION**关键字定义稳态检测：

```
[*STEADY STATE DETECTION](../key/key-link.md#usb-kws-msteadystatedetection), TYPE=energy
*阈值
*检测间隔
```

### 参数设置

#### 阈值

阈值定义何时认为响应已稳定。较低的值要求更接近稳态，但可能需要更长的分析时间。

典型值：
- 能量变化率：1e-4 到 1e-6
- 位移变化率：1e-3 到 1e-5

#### 检测间隔

定义执行稳态检测的频率（以增量步或时间步为单位）。

#### 连续符合次数

定义连续多少次检测符合稳态条件才认为真正达到稳态。用于避免误判。

### 应用场景

#### 蠕变分析

材料蠕变响应最终会达到稳态（稳态蠕变阶段）。稳态检测可用于自动终止分析。

#### 粘弹性和粘塑性分析

粘性材料的响应会逐渐过渡到稳态。

#### 振动分析

结构在谐波激励下的响应最终达到稳态振动。

#### 疲劳分析

循环加载下响应最终达到稳态循环。

### 输出

稳态检测分析会产生以下输出：
- 稳态指标随时间/步数的变化
- 达到稳态的步数或时间
- 达到稳态时的响应量值

### 注意事项

1. **阈值选择**：阈值过低可能导致分析过早终止，阈值过高则可能延长分析时间

2. **初始瞬态**：对于有明显初始瞬态的问题，应允许足够的"预热"时间再开始检测

3. **非线性问题**：强非线性问题可能不会真正达到稳态，应谨慎使用

4. **验证**：首次使用稳态检测时应验证结果是否合理

### 与其他功能结合

稳态检测可以与以下功能结合使用：

- **自动时间步长**：达到稳态后可增大时间步长
- **结果输出**：稳态检测触发后可减少输出频率
- **重启动**：稳态检测可用于确定何时可以生成重启动数据
