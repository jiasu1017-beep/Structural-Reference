### 概述

率相关屈服：
- 当屈服强度取决于应变率且预期应变率显著时，需要准确定义材料的屈服行为；
- 仅适用于各向同性硬化金属塑性模型（Mises和Johnson-Cook）、非线性各向同性/运动塑性模型的各向同性分量、扩展Drucker-Prager塑性模型和可压碎泡沫塑性模型；
- 可以通过为各向同性硬化金属塑性模型、非线性各向同性/运动塑性模型的各向同性分量和扩展Drucker-Prager塑性模型提供表格数据，基于加工硬化参数和场变量方便地定义；
- 可以通过指定用户定义的过应力幂律参数、屈服应力比或Johnson-Cook率相关性参数来定义（最后一个选项不适用于可压碎泡沫塑性模型，是Johnson-Cook塑性模型可用的唯一选项）；
- 不能与任何Abaqus/Standard蠕变模型（金属蠕变、时间相关体积肿胀、Drucker-Prager蠕变或帽盖蠕变）一起使用，因为蠕变行为已经是率相关机制；和
- 在动态分析中应指定为屈服应力随应变率增加而增加。

### 加工硬化依赖性

通常，材料的屈服应力σ^0~（或可压碎泡沫模型的σ~s~）依赖于加工硬化，对于各向同性硬化模型，这通常用等效塑性应变的合适度量ε¯^p^表示；非弹性应变率ε¯˙^p^；温度θ；和预定义场变量f^α^：

σ^0^ = f(ε¯^p^, ε¯˙^p^, θ, f^α^)

许多材料表现出屈服强度随应变率增加而增加；当应变率在每秒0.1到1之间时，这一效应对许多金属和聚合物变得重要，对于每秒10到100之间的应变率可能非常重要，这是高能动态事件或制造过程的特征。

### 为各种材料模型定义硬化依赖性

应变率相关性可以通过直接输入不同应变率的硬化曲线或通过定义屈服应力比来独立指定率相关性来定义。

#### 直接输入测试数据

对于各向同性硬化Mises塑性模型、非线性各向同性/运动硬化模型的各向同性分量和扩展Drucker-Prager塑性模型，加工硬化依赖性可以相当一般地以表格数据给出。测试数据以不同等效塑性应变率下的屈服应力值与等效塑性应变的关系表形式输入。屈服应力必须给出为等效塑性应变以及温度和其他预定义场变量的函数。在有限应变下定义此依赖性时，应使用"真实"（Cauchy）应力和对数应变值。每个温度的硬化曲线必须始终从零塑性应变开始。对于理想塑性，每个温度只应定义一个屈服应力，塑性应变为零。也可以将材料定义为应变软化以及应变硬化。加工硬化数据根据需要重复多次以定义不同应变率下的应力-应变曲线。给定应变和应变率下的屈服应力直接从这些表中插值获得。

#### 使用屈服应力比

或者，作为为Johnson-Cook和可压碎泡沫塑性模型定义率相关屈服应力的唯一手段，假设应变率行为是可分离的，因此应力-应变依赖性在所有应变率水平下是相似的：

σ^0^(ε¯^p^, ε¯˙^p^) = R(ε¯˙^p^) σ^0~s~(ε¯^p^)

其中σ^0~s~（或泡沫模型中的σ~s~）是静态应力-应变行为，R是屈服应力与非零应变率下静态屈服应力之比（因此 R = 1当ε¯˙^p^ = 0时）。

Abaqus提供了三种方法来定义R：指定过应力幂律、直接将R定义为表格函数，或指定分析Johnson-Cook形式来定义R。

##### 过应力幂律

Cowper-Symonds过应力幂律具有以下形式：

R = 1 + (ε¯˙^p^/C)^1/p

其中C和p是材料参数，可以是温度以及可能是其他预定义场变量的函数。

##### Johnson-Cook率相关性

Johnson-Cook率相关性具有以下形式：

R = 1 + C ln(ε¯˙^p^/ε¯˙^p^0)

其中D和C是不依赖于温度的材料常数，假定也不依赖于预定义场变量。Johnson-Cook率相关性可与Johnson-Cook塑性模型、各向同性硬化金属塑性模型和扩展Drucker-Prager塑性模型结合使用（不能与可压碎泡沫塑性模型结合使用）。

这是Johnson-Cook塑性模型可用的唯一形式的率相关性。更多详情，请参见["Johnson-Cook塑性，" 第23.2.7节"](pt05ch23s02abm23.md)。
