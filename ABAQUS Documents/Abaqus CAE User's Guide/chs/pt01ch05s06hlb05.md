# 5.6.5 放大到视图的所选区域





使用**视图操作**工具栏中的框选缩放工具 ![](../graphics/ico_viewZoomRectangle.png) 放大视图，使所选区域填充视口。如果当前视口链接到其他视口，Abaqus/CAE 会以相同因子放大链接视口中的视图。有关更多信息，请参阅["链接视口以进行视图操作，" 第 4.5 节](pt01ch04s05.md)。

**要放大到视图的所选区域：**

1. 从**视图操作**工具栏中，单击框选缩放工具 ![](../graphics/ico_viewZoomRectangle.png) 进入缩放模式。**提示：**你也可以从主菜单中选择****View**![](../graphics/images/arrow.gif)**Box Zoom****，或按 **[F5]**。
2. 将光标定位在你要更改视图的视口中。光标形状变为十字线。
3. 将光标定位在要放大区域的角落之一。
4. 将光标拖动到对角角落。矩形表示要放大的区域。
5. 释放鼠标按钮 1。矩形定义的区域会放大以填充视口。
6. 根据需要重复步骤 2 到 5 多次，以达到所需的视图。
7. 要退出框选缩放模式，请执行以下操作之一：
   - 单击鼠标按钮 2。
   - 单击提示区域中的取消按钮 ![](../graphics/afxI_cancel.png)。
   - 单击框选缩放工具。
   - 单击任何其他视图操作工具。
   **提示：**使用[循环](pt01ch05s06hlb07.md)视图操作工具返回上一个视图。
