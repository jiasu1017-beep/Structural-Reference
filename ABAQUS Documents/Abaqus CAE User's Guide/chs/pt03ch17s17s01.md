# 17.17.1 创建网格





您可以对整个零件或零件实例创建网格，也可以在选定区域内创建网格。 要创建网格，请从主菜单栏选择 ****Mesh**![](../graphics/images/arrow.gif)**Part****、 ****Mesh**![](../graphics/images/arrow.gif)**Instance**** 或 ****Mesh**![](../graphics/images/arrow.gif)**Region****。

**创建网格：**

1. 从上下文栏的**Object**字段中，选择要网格化的零件或选择装配。
2. 从主菜单栏，选择 ****Mesh**![](../graphics/images/arrow.gif)**Part****、 ****Mesh**![](../graphics/images/arrow.gif)**Instance**** 或 ****Mesh**![](../graphics/images/arrow.gif)**Region****。 Abaqus/CAE在提示区域显示提示以指导您完成该过程。 **提示：**您也可以使用Mesh模块工具箱中网格工具所在的 ![](../graphics/ico_meshInstance.png) 和 ![](../graphics/ico_meshRegion.png) 工具对零件、零件实例或区域进行网格划分。 （有关更多信息，请参阅["使用Mesh模块工具箱，" 第17.15节](pt03ch17s15.md)。）
3. 如果您正在对装配进行网格划分，必须选择要进行网格划分的实例或区域。 单击鼠标按钮2表示您已完成选择。 （有关选择对象的更多信息，请参阅[第6章，"在视口中选择对象"](pt01ch06.md)。） 仅选择那些显示为绿色、粉红色或黄色的零件实例或区域，表示它们可以进行网格划分。 要使橙色区域可进行网格划分，必须使用划分工具细分它们，分配自底向上网格划分技术，或为它们分配四面体形状的单元。 已分配自底向上网格划分技术的区域显示为浅棕色；您必须使用自底向上技术对它们进行网格划分，或分配另一种网格划分技术。 （有关创建自底向上网格的更多信息，请参阅["自底向上网格划分，" 第17.11节](pt03ch17s11.md)。） 显示为白色的零件无法进行网格划分，因为它们与独立实例相关联。 **注意：**仅当选择**Mesh defaults**颜色映射时，零件实例才会根据其可网格划分性着色。 如果视口中的颜色与本步骤描述的颜色不匹配，请应用此颜色映射。
4. 执行以下操作之一： - 从提示区域，单击**Yes**在可网格划分的区域上生成网格。 - 如果装配包含将用四面体单元进行网格划分的实体区域，Abaqus/CAE会询问您是否要预览该区域外表面上的三角形网格。 1. 要预览网格，请从提示区域切换**Preview boundary mesh**并单击**Yes**创建边界网格。 Abaqus/CAE会在可网格划分的区域上生成三角形边界网格。 2. 如果某些区域网格划分失败或边界网格不可接受，Abaqus/CAE提供了多种工具来帮助您在所有区域生成网格并创建可接受的网格。 有关更多信息，请参阅["边界网格有什么用？" 第17.10.5节](pt03ch17s10s05.md)。 3. 如果边界网格可接受，从提示区域单击**Yes**继续对零件、实例或区域的内部进行网格划分。 Abaqus/CAE会在可网格划分的区域上生成网格。
