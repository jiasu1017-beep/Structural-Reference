# 16.9.17 定义表面热通量





您可以创建表面热通量载荷以定义基于表面的热通量。

**创建或编辑表面热通量：**

1. 使用以下方法之一显示表面热通量载荷编辑器：- 若要创建新的表面热通量载荷，请遵循"创建载荷"，第16.8.1节中概述的过程（**类别**：`Thermal`；**所选步骤的类型**：`Surface heat flux`）。- 若要使用菜单或管理器编辑现有的表面热通量载荷，请参见"编辑依赖于步骤的对象"，第3.4.13节。若要编辑应用载荷的区域，请参见"编辑应用规定条件的区域"，第16.8.4节。
2. 单击**分布**字段右侧的箭头，并从出现的列表中选择您的选项：- 选择**均匀**以定义在表面上均匀的载荷。对于此选项，您提供的数值必须是每单位面积的通量。- 选择**用户定义**以在用户子程序[`DFLUX`](../sub/sub-link.md#sub-xsl-dflux)中定义载荷幅值。（此选项仅在Abaqus/Standard分析中有效。）有关更多信息，请参见以下章节：- "指定常规作业设置"，第19.8.6节- "Abaqus User Subroutines Reference Guide"第1.1.3节"DFLUX"- 选择**总通量**以定义在表面上均匀的载荷。对于此选项，您提供的数值必须是将应用于表面的通量的总幅值（而不是每单位面积的通量）。- 选择分析场（标记为(A)）或离散场（标记为(D)）以定义空间变化的表面热通量。只有对这些载荷类型有效的分析场和离散场才会在选择列表中显示。或者，您可以单击![](../graphics/ico_analyticalField.png)创建新的分析场。（更多信息，请参见第58章"分析场工具集"。）
3. 如果您选择了**均匀**、**总通量**、分析场或离散场分布选项，请执行以下步骤：1. 在**幅值**文本字段中，输入表面热通量幅值。正幅值表示热量流入表面。对于**均匀**分布，输入所施加通量的总幅值除以施加通量的表面面积（单位[JT<sup>-1</sup>L<sup>-2</sup>](../popups/usb-int-iconventions-unitsym.md)）。对于**总通量**分布，输入通量的总幅值（单位[JT<sup>-1</sup>](../popups/usb-int-iconventions-unitsym.md)）。Abaqus/CAE根据输入的通量幅值计算恒定的均匀表面通量。2. 如果需要，单击**幅值**字段右侧的箭头，并从出现的列表中选择您选择的幅值。或者，您可以单击![](../graphics/ico_amplitudeSmall.png)创建新的幅值。（更多信息，请参见第57章"幅值工具集"。）3. 单击**确定**保存您的数据并退出编辑器。
4. 如果您选择了**用户定义**分布选项，请执行以下步骤：1. 如果需要，在**幅值**字段中输入表面热通量幅值（单位[JT<sup>-1</sup>L<sup>-2</sup>](../popups/usb-int-iconventions-unitsym.md)）。正幅值表示热量流入表面。您在编辑器中输入的幅值数据被传递到用户子程序中。2. 单击**确定**保存您的数据并退出编辑器。3. 进入作业模块并显示感兴趣的分析作业的作业编辑器。（更多信息，请参见"创建、编辑和操作作业"，第19.7节。）4. 在作业编辑器中，单击**常规**选项卡，并指定包含定义载荷幅值的用户子程序的文件。更多信息，请参见"指定常规作业设置"，第19.8.6节。**注意：**您只能在作业编辑器中指定一个用户子程序文件；如果您的分析涉及多个用户子程序，您必须将用户子程序合并到一个文件中，然后指定该文件。
