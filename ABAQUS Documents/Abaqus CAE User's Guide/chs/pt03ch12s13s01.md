# 12.13.6 创建均匀壳截面





壳截面行为根据壳截面对拉伸、弯曲、剪切和扭转的响应来定义。请参见[Abaqus Analysis User's Guide的第29.6.4节，"壳截面行为"](../usb/usb-link.md#usb-elm-eshellsectionbehavior)。

创建壳截面时，必须选择截面积分方法。您可以选择在分析前提供截面属性数据（预积分壳截面），或让Abaqus在分析期间从截面积分点计算（积分）横截面行为。

在分析期间积分的壳截面允许通过壳厚度的数值积分计算横截面行为，从而在材料建模中提供完全的通用性。可以通过厚度定义任意数量的材料点，材料响应可以从点到点变化。这种类型的壳截面通常与截面中的非线性材料行为一起使用。它必须与提供热传递的壳一起使用。请参见["使用在分析期间积分的壳截面来定义截面行为"，第29.6.5节Abaqus Analysis User's Guide](../usb/usb-link.md#usb-elm-eusingshellsection)。

可以使用预积分壳截面定义线性弯矩-弯曲和力-膜应变关系。在这种情况下，所有计算都以截面力和弯矩的形式进行。截面属性由弹性材料指定；可选择地，您还可以基于关于壳的预期行为或组成的假设来应用理想化。如果壳的响应是线弹性的，并且其行为不依赖于温度或预定义场变量的变化，请使用预积分壳截面。请参见["使用通用壳截面定义截面行为"，第29.6.6节Abaqus Analysis User's Guide](../usb/usb-link.md#usb-elm-eusingshellgensect)。

以下章节描述如何创建均匀壳截面：

### 创建均匀壳截面

**创建均匀壳截面：**

1. 从主菜单栏，选择****Section**![](../graphics/images/arrow.gif)**Create****。出现**Create Section**对话框。
   **提示：**您也可以在**Section Manager**中点击**Create**，或在Property模块工具箱中选择创建截面工具![](../graphics/ico_section.png)。
2. 输入截面名称。有关命名对象的更多信息，请参见["使用基本对话框组件，"第3.2.1节"](pt01ch03s02s01.md)。
3. 选择**Shell**作为截面**Category**，选择**Homogeneous**作为截面**Type**，然后点击**Continue**。出现壳截面编辑器。
4. 选择**Section integration**方法。选择**During analysis**以为在分析期间积分的均匀壳截面指定属性。选择**Before analysis**以为预积分均匀壳截面指定属性。

### 为均匀壳截面指定基本属性

在**Basic**标签页上：

**为均匀壳截面指定基本属性：**

1. 指定**Shell thickness**。
   - 选择**Value**，并输入壳厚度值。在连续壳中，此值用于估计某些截面属性，如小时刚度，这些属性稍后使用从元素几何计算的实际厚度计算。
   - 选择**Element distribution**；并选择解析场（标记为(A)）或基于元素的离散场（标记为(D)）以定义空间变化的基于元素的壳厚度。或者，您可以点击![](../graphics/ico_analyticalField.png)创建新解析场或点击![](../graphics/ico_discreteField.png)创建新离散场。请参见[第58章，"解析场工具集"](pt06ch58.md)和[第63章，"离散场工具集"](pt06ch63.md)，了解更多。
   - 选择**Nodal distribution**；并选择解析场（标记为(A)）或基于节点的离散场（标记为(D)）以定义空间变化的基于节点的壳厚度。或者，您可以点击![](../graphics/ico_analyticalField.png)创建新解析场或点击![](../graphics/ico_discreteField.png)创建新离散场。请参见[第58章，"解析场工具集"](pt06ch58.md)和[第63章，"离散场工具集"](pt06ch63.md)，了解更多。
2. 为壳截面选择一种材料。如有需要，点击![](../graphics/ico_material.png)创建材料；有关更多信息，请参见["创建或编辑材料，"第12.7.1节"](pt03ch12s07hlb01.md)。线性或非线性材料行为可以与截面定义相关联。但是，如果材料响应是线性的，更经济的方法是使用广义壳截面。
3. 如果您为在分析前积分的均匀壳截面指定属性，可以指定**Idealization**以基于关于壳预期行为的假设应用于截面。请参见"使用通用壳截面定义截面行为"中的"理想化截面响应"，第29.6.6节Abaqus Analysis User's Guide。
   - 选择**No idealization**以考虑由材料分配确定的壳截面的完整刚度。
   - 如果壳的主要响应是面内拉伸，请选择**Membrane only**；弯曲刚度项从壳刚度计算中消除。
   - 如果壳的主要响应是纯弯曲，请选择**Bending only**；膜刚度项从壳刚度计算中消除。
4. 如果您为在分析期间积分的均匀壳截面指定属性，请选择**Thickness integration rule**。
   - 选择**Simpson**以使用Simpson规则进行壳截面积分。
   - 选择**Gauss**以使用Gauss积分进行壳截面积分。请参见"使用在分析期间积分的壳截面来定义截面行为"中的"定义壳截面积分"，第29.6.5节Abaqus Analysis User's Guide。
5. 如果您为在分析期间积分的均匀壳截面指定属性，请输入厚度积分点的数量。对于Simpson规则积分，默认厚度积分点数量为5；对于Gauss积分，默认数量为3。要指定积分点数量的新值，您可以直接输入数字或点击**Thickness integration points**文本字段中的箭头。
   - 如果您使用**Simpson**积分规则，只能指定从3到15的奇数。
   - 如果您使用**Gauss**积分规则，可以指定从2到15的数字。

### 为均匀壳截面指定高级属性

在**Advanced**标签页上：

**为均匀壳截面指定高级属性：**

1. 指定**Section Poisson's ratio**以定义壳厚度行为。
   - 在允许大变形分析中有限膜应变的常规壳元素中，指定截面泊松比会导致壳厚度作为膜应变的函数而变化：
     - 切换**Use analysis default**以使用默认值。在Abaqus/Standard中，默认值为0.5，这将强制元素膜应变的不可压缩行为。在Abaqus/Explicit中，默认值基于元素材料定义中的厚度变化。
     - 切换**Specify value**，并输入泊松比值。此值必须介于1.0和0.5之间。值为0.0将强制恒定壳厚度，负值将导致壳厚度响应拉伸膜应变而增加。
   - 在连续壳元素中，指定截面泊松比定义小位移和大位移分析的厚度行为：
     - 切换**Use analysis default**以指示厚度变化基于元素材料定义。
     - 切换**Specify value**，并输入泊松比值以导致壳厚度作为膜应变的函数而变化。此值必须介于1.0和0.5之间。值0.5不能与连续壳一起使用。值为0.0将强制恒定壳厚度，负值将导致壳厚度响应拉伸膜应变而增加。
2. 对于连续壳元素，切换**Thickness modulus**，并输入有效厚度模量的值。如果不指定厚度模量，Abaqus将尝试基于初始弹性材料属性进行计算。
3. 如果您为在分析期间积分的均匀壳截面指定属性，请选择定义截面**Temperature variation**的方法：
   - 选择**Linear through thickness**以指示已指定参考表面温度和穿过截面的温度梯度。您可以使用Load模块指定这些温度。
   - 选择**Piecewise linear over *n* values**以在提供的文本字段中输入穿过截面的温度点（值）的数量。您可以使用Load模块指定每个这些点的温度。
4. 切换**Density**，并输入壳单位表面积的质量。壳的质量包括来自密度的贡献以及所选材料的任何贡献。
5. 对于大多数壳截面，Abaqus将计算元素公式中所需的横向剪切刚度值。如有需要，从**Transverse Shear Stiffnesses**选项切换**Specify values**以在截面定义中包含非默认横向剪切刚度效应，并输入以下值：![](../graphics/usi_eqn00369.gif)，截面在第一方向上的剪切刚度；![](../graphics/usi_eqn00370.gif)，截面剪切刚度中的耦合项；以及![](../graphics/usi_eqn00371.gif)，截面在第二方向上的剪切刚度。如果![](../graphics/usi_eqn00369.gif)或![](../graphics/usi_eqn00371.gif)中的任何一个被省略或给定为零，则将使用非零值作为两者的值。
6. 如果您为在分析期间积分的均匀壳截面指定属性，请点击壳截面编辑器底部的![](../graphics/ico_rebarLayers.png)以在壳截面中定义钢筋层，如["定义钢筋层，"第12.13.19节"](pt03ch12s13hlb15.md)中所述。
7. 点击**OK**保存更改并关闭壳截面编辑器。
