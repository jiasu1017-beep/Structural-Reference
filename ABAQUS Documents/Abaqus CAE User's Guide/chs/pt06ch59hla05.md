# 59.8 移除附件点和线





您可以从模型中移除附件点和线。移除的附件点或线出现在模型树中，作为零件或装配下**特征**容器中的**移除附件**特征，具体取决于创建附件点或线的模块。您可以抑制或删除**移除附件**特征以恢复移除的附件点或线。在 Part 模块或 Property 模块中创建的附件点只能从这两个模块中移除；在 Assembly 模块或 Interaction 模块中创建的附件点和线只能从这两个模块中移除。

**移除附件点和线的步骤：**

1. 从主菜单栏，选择 ****Tools**![](../graphics/images/arrow.gif)**Attachment**![](../graphics/images/arrow.gif)**Remove attachments****。

2. 从当前视口，选择要移除的附件点或线。您可以单击提示区域右侧的 **Sets** 移除先前定义的集合。

3. 如果您在 Part 模块或 Property 模块中，请单击提示区域的 **Done** 以指示您已完成选择要移除的附件点。选定的附件点从模型中移除，并在模型树中零件下的**特征**下创建一个**移除附件**特征。

4. 如果您在 Interaction 模块或 Assembly 模块中，请执行以下操作：
   1. 在提示区域，单击 **Done** 以指示您已完成选择要移除的附件点或线。
   2. 在出现的对话框中，执行以下操作之一：
      - 单击 **Yes** 继续并从模型中移除选定的附件点或线。将在模型树中装配下创建一个**移除附件**特征。
      - 单击 **No** 返回视口以修改要移除的选定的附件点或线。
      - 单击 **Cancel** 退出过程，不移除任何附件点或线。

5. 如果需要，选择其他要移除的附件特征。

6. 单击鼠标按钮 2 退出移除附件过程。
