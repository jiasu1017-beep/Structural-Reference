# 17.5.2 哪些类型的单元必须在网格模块外部生成？







Abaqus/CAE 为所有 Abaqus/CFD 单元以及 Abaqus/Standard 和 Abaqus/Explicit 使用的大多数单元提供支持。但是，某些单元不受支持，必须在网格模块外部生成。以下列表描述了 Abaqus/CAE 不支持的单元。如果您想将这些单元类型分配给模型，必须使用文本编辑器将它们添加到在 Job 模块中生成的输入文件。有关生成输入文件的信息，请参见"分析模型的基本步骤"第 19.2.1 节。

- 声学界面单元（ASI1）
- 具有非轴对称响应的轴对称单元（CAXA4*N*、CAXA8P*N* 等）
- 耦合热-电-结构单元（Q3D4、Q3D6 等）
- 分布式耦合单元（DCOUP2D 和 DCOUP3D）
- 拖链单元（DRAG2D 和 DRAG3D）
- 弹塑性连接单元（JOINT2D 和 JOINT3D）
- 框架单元（FRAME2D 和 FRAME3D）
- 间隙单元，耦合温度-位移和传热（GALUNIT 和 DGAP）
- 无限单元（CIN3D8、CINAX4 等）
- 线弹簧单元（LS3S 和 LS6）
- 薄膜单元，9 节点四边形（M3D9 和 M3D9R）
- 薄膜单元，柱面（MCL6 和 MCL9）
- 粒子单元（PC3D）
- 管-土相互作用单元（PSI24、PSI34 等）
- 孔隙压力粘聚单元（COH2D4P、COH3D6P 等）
- 滑线单元（ISL21A 和 ISL22A）
- 应力/位移变节点单元（C3D15V、C3D27 等） **注意：**提交分析执行后，如果 C3D20*(R)(H)* 单元与接触对中的从属表面相邻，Abaqus/Standard 会自动将其转换为相应的 C3D27*(R)(H)* 单元。（显式分析中这两种单元都不可用。）否则，无法使用 Abaqus/CAE 生成变节点六面体。
- 表面单元，柱面（SFMCL6 和 SFMCL9）
- 薄壳单元，9 节点双曲（S9R5）
- 管-管接触单元（ITT21 和 ITT31）

您无法在网格模块中分配某些单元（如 CONN2D2 和 SPRING1）；但是，您可以在 Interaction 模块中创建等效连接器，或在 Property 模块或 Interaction 模块中创建等效工程特性，如表 17-1 所示：

**表 17-1** Abaqus/CAE 对连接器和工程特性的支持。
| 单元 | Abaqus/CAE 支持 |
| --- | --- |
| CONN2D2, CONN3D2 | Interaction 模块中的等效连接器 |
| DASHPOTA, DASHPOT1, DASHPOT2 | Property 模块或 Interaction 模块中的工程特性（与场变量无关的线性行为） |
| 等效连接器在 Interaction 模块中 |
| GAPCYL, GAPSPHER, GAPUNI | Interaction 模块中的等效连接器 |
| HEATCAP | Property 模块或 Interaction 模块中的工程特性 |
| ITSCYL, ITSUNI | Interaction 模块中的等效连接器 |
| JOINTC | Interaction 模块中的等效连接器 |
| MASS | Property 模块或 Interaction 模块中的工程特性 |
| ROTARYI | Property 模块或 Interaction 模块中的工程特性 |
| SPRINGA, SPRING1, SPRING2 | Property 模块或 Interaction 模块中的工程特性（与场变量无关的线性行为） |
| 等效连接器在 Interaction 模块中 |

有关更多信息，请参见"理解连接器"第 15.7 节；第 33 章"
