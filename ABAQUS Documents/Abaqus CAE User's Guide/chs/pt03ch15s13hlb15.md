# 15.13.17 定义声阻抗







您可以通过提供声阻抗边界或非反射边界来模拟声阻抗，以用于声学和耦合声固分析。从主菜单栏中选择****相互作用**![](../graphics/images/arrow.gif)**创建****，然后选择形成声学边界的表面。声阻抗相互作用仅在使用声自由度的动态步骤中活动。您可以在静态步骤中创建声阻抗相互作用；Abaqus/CAE会忽略静态步骤中的声学效应，但会通过任何适用的后续线性扰动步骤传播该相互作用。如果您在线性扰动步骤中创建声阻抗相互作用，该相互作用不会传播到任何后续步骤。

有关声阻抗的简要概述，请参见"理解相互作用"，第15.3节。有关更详细的讨论，请参见Abaqus Analysis User's Guide的第34.4.6节"声学和冲击载荷"。

**定义声阻抗：**

1. 从主菜单栏中，选择****相互作用**![](../graphics/images/arrow.gif)**创建****。**提示：**您也可以使用相互作用模块工具箱中的![](../graphics/ico_interaction.png)工具创建声阻抗相互作用。
2. 在出现的**创建相互作用**对话框中，执行以下操作：- 命名该相互作用。有关命名对象的信息，请参见第3.2.1节"使用基本对话框组件"。- 选择步骤。声阻抗仅在使用声自由度的动态步骤中活动。- 选择**声阻抗**类型的相互作用。
3. 单击**继续**关闭**创建相互作用**对话框。
4. 使用以下方法之一选择表面：- 使用现有表面定义区域。在提示区域右侧单击**表面**。从出现的**区域选择**对话框中选择现有表面，然后单击**继续**。**注意：**默认选择方法基于您最近使用的选择方法。要恢复为另一种方法，请单击提示区域右侧的**在视口中选择**或**表面**。- 使用鼠标在视口中选择区域。（更多信息，请参见"在当前视口中选择对象"，第6.2节。）如果模型包含网格和几何的组合，请从提示区域单击以下之一：- 如果要从几何区域选择表面，请单击**几何**。- 如果要从本机或孤立网格选择表面，请单击**网格**。您可以使用角度方法从几何或网格中选择一组面或边或一组单元面。更多信息，请参见"使用角度和特征边缘方法选择多个对象"，第6.2.3节。
5. 在出现的**编辑相互作用**对话框中的**定义**字段中，选择以下之一：- 选择**表格**以使用声阻抗属性中的导纳或阻抗值表格定义阻抗。- 选择**非反射**以为阻抗定义非反射边界。
6. 如果您选择了**表格**定义选项，请选择声阻抗属性。如果需要，请单击![](../graphics/ico_interactionProperty.png)创建相互作用属性；有关更多信息，请参见第15.14.6节"定义声阻抗相互作用属性"。
7. 如果您选择了**非反射**定义选项，请单击**非反射类型**字段右侧的箭头，然后从出现的列表中选择以下选项之一来定义非反射几何。- 选择**平面**以指定适用于平面波垂直入射到平面边界的辐射条件。- 选择**改进平面**以指定适用于任意入射角平面波的辐射条件。在线性扰动步骤中，改进平面声阻抗相互作用默认为平面声阻抗相互作用。- 选择**圆形**以指定适用于二维圆形边界或三维正圆柱的辐射条件。- 选择**球形**以指定适用于球形边界的辐射条件。- 选择**椭圆形**以指定适用于二维椭圆边界或三维正椭圆柱的辐射条件。- 选择**长球形**以指定适用于长球形边界的辐射条件。
8. 如果您选择了**圆形**或**球形**非反射定义选项，请在**半径**字段中输入定义边界表面的圆或球的半径。
9. 如果您选择了**椭圆形**或**长球形**非反射定义选项，请执行以下步骤：1. 在**轴长度**字段中，输入定义表面的椭圆或长球体的半长轴长度*a*。*a*是椭圆或球体上两点之间最大距离的二分之一，类似于圆或球的半径。2. 在**离心率**字段中，输入椭圆或长球体的离心率，![](../graphics/usi_eqn00171.gif)。离心率是半短轴*b*与半长轴*a*之比的平方减一的平方根：![](../graphics/usi_eqn00573.gif)。3. 在**中心坐标**字段中，输入定义辐射表面的椭圆或长球体中心的*X*坐标、*Y*坐标和*Z*坐标。4. 在**方向余弦**字段中，输入定义辐射表面的椭圆或长球体长轴方向余弦的*X*分量、*Y*分量和*Z*分量。
