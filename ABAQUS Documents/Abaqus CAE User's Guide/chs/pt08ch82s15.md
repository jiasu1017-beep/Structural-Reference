# 82.15 找到一组元素的平均温度





**Volume Weighted Average Temperature** 插件让您可以计算分析中所有元素或任意元素子集的平均温度。可以在视口中选择元素，也可以从预定义的命名集中选择元素。如果装配中有多个零件实例，您只能在一个实例上选择元素。可以在当前时间为模型请求结果，该模型显示在视口中，也可以作为整个分析所有步长的 *X–Y* 时间曲线请求结果。

此插件只能与输出数据库（`.odb`）文件一起使用。输出数据库中必须包含整个模型的场输出变量 IVOL 和 TEMP。

**查找体积加权平均温度：**

1. 在 Visualization 模块中，从主菜单栏选择 ****Plug-ins**![](../graphics/images/arrow.gif)**Tools**![](../graphics/images/arrow.gif)**Volume Weighted Average Temperature****。出现 **Volume Weighted Average Temperature** 对话框。
2. 选择要计算平均温度的时间： - 选择 **Current step/frame** 以仅在视口中当前显示的步长和帧期间查找温度。 - 选择 **All steps** 以计算和绘制整个分析过程中的温度。
3. 选择要计算平均温度的元素。您可以选择模型的所有元素或一个子集。 - 点击 **Select From Viewport**，然后按照提示单独、按角度或按特征边选择元素。请参见 ["在当前视口中选择对象，" 第6.2节](pt01ch06s02.md) 获取说明。有关按角度或特征边选择的详细信息，请参见 ["使用角度和特征边方法选择多个对象，" 第6.2.3节](pt01ch06s02hlb03.md)。 - 点击 **Select Element Sets** 以从预定义的元素集列表中选择。完成元素选择后点击 **Complete**。
4. 计算平均温度。 - 如果您选择了 **Current step/frame**，请点击 **Calculate**。结果在窗口底部的消息区域中给出，例如：
```
Number of elements picked from viewport: 80
**Volume weighted average temperature for the selected elements
No. of elements used in calculation: 80
Step: Step-1, Frame: 15, Current time: 15.0
WtAvTemp: 199.883326817
Volume: 1575.0
```
 - 如果您选择了 **All steps**，请点击 **Plot**。当前视口中显示时间历史 *X--Y* 绘图。如果需要，您可以更改历史曲线的默认曲线名称。
