# 1.1 本指南概述









本指南是使用 Abaqus/CAE（包括 Abaqus/Viewer，Abaqus/CAE 的一个子集，仅包含 Visualization 模块）的完整参考资料。通常，本指南中对 Visualization 模块的任何引用同样适用于 Abaqus/Viewer。

Abaqus/CAE 用户界面非常直观，允许您开始工作而无需大量准备。但是，您可能会发现，在首次使用该产品之前，通读[Getting Started with Abaqus: Interactive Edition](../gsa/gsa-link.md#gsa) 末尾的教程很有用。如果您正在运行 Abaqus/Viewer，只有 [Getting Started with Abaqus: Interactive Edition 的「附录 D，查看分析输出」](../gsa/gsa-link.md#gsa-chp-caevismod) 适用。

本指南分为以下部分：
- [第一部分，「与 Abaqus/CAE 交互」](pt01.md)，包含用户界面的一般信息
- [第二部分，「使用 Abaqus/CAE 模型数据库、模型和文件」](pt02.md)，包含有关 Abaqus/CAE 创建和使用的各种文件的信息
- [第三部分，「使用 Abaqus/CAE 模块创建和分析模型」](pt03.md)，详细讨论除 Visualization 模块之外的每个 Abaqus/CAE 模块
- [第四部分，「建模技术」](pt04.md)，讨论如何在 Abaqus/CAE 模型中定义特殊工程特征，以及跨多个 Abaqus/CAE 模块的建模技术。
- [第五部分，「查看结果」](pt05.md)，详细讨论 Visualization 模块（Abaqus/Viewer）
- [第六部分，「使用工具集」](pt06.md)，包含除 Visualization 模块（[第五部分，「查看结果」](pt05.md) 中讨论）之外的所有 Abaqus/CAE 模块中工具集的信息
- [第七部分，「自定义模型显示」](pt07.md)，包含自定义信息
- [第八部分，「使用插件」](pt08.md)，讨论您如何使用插件和插件工具集扩展 Abaqus/CAE 的功能。

[附录 A，「关键字支持」](ap01.md) 提供了表格，您可以用它来确定哪个 Abaqus/CAE 模块体现了特定 Abaqus 关键字的功能，以及特定关键字是否受支持。[附录 B，「特殊单元类型」](ap02.md) 列出了 Abaqus 中用于模型特征的单元类型，这些特征不是网格的一部分。[附录 C，「特殊图形符号」](ap03.md) 解释如何理解 Abaqus/CAE 使用的特殊图形符号。[附录 D，「单元和输出变量支持」](ap04.md) 列出了 Visualization 模块不支持的 Abaqus 输出变量。
