# 58.4.2 映射场显示符号详情





要查看表示映射场的符号，您必须在应用载荷、相互作用或预定义场之前先对模型进行网格划分。

映射场上符号显示使用的点是分析中将使用的原生或孤立网格节点或单元。如果没有原生网格，符号将按非缩放状态显示。此外，如果由于错误（例如不支持的单元类型或不正确的容差）导致映射失败，符号也将按非缩放状态显示。

在这些情况下，消息区域中会出现一条消息，指示评估错误。作为视觉提示以指示存在错误，显示在视口中的符号不再缩放，并以关闭缩放时它们将所在的点显示。您可以尝试通过重新划分网格或更改符号密度设置来修复映射，这会提示 Abaqus/CAE 选择一组不同的点用于符号显示。有关更改符号密度设置的更多信息，请参见["控制属性的显示，" 第76.15节](pt07ch76hla14.md)。
