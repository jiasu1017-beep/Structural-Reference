# 15.17.11 定义失效

对于Abaqus/Standard分析，您可以为任何可用相对运动分量指定失效准则。对于Abaqus/Explicit分析，失效准则可以依赖于相对运动可用分量的运动，以及相对运动可用分量和约束分量中的力/力矩。当满足任何准则时发生失效，而不是当满足所有准则时发生失效。有关更多信息，请参见["连接器失效行为，"Abaqus分析用户指南第31.2.9节](../usb/usb-link.md#usb-elm-econnfailurebehav)。

当您定义依赖于力/力矩的失效准则时，来自所有行为选项的力/力矩之和将与失效准则进行比较。在Abaqus/Standard分析中，为了使失效准则有效，您必须定义一个行为选项（如弹性），以为用于定义失效准则的可用相对运动分量产生力/力矩。

**要创建或编辑失效：**

1. 使用以下方法之一显示连接器截面编辑器：
   - 要创建新的连接器截面，请按照["创建连接器截面，"第15.12.11节"](pt03ch15s12s01.md)中概述的过程进行操作。
   - 要编辑现有的连接器截面，请从主菜单栏选择**连接器**![](../graphics/images/arrow.gif)**截面**![](../graphics/images/arrow.gif)**管理器**，从出现的列表中选择连接器截面，然后点击**编辑**。
2. 在**编辑连接器截面**对话框中，执行以下操作之一：
   - 要定义新的失效选项，请点击**添加**并从出现的选项菜单中选择**失效**。
   - 要编辑现有的失效选项，请从**行为选项**列表中选择该选项以显示该选项的关联数据字段。
3. 在**分量**字段中，打开用于定义失效准则的相对运动分量的开关。如果您想为不同分量指定不同的限制，则必须定义多个失效选项。
4. 点击**释放**字段右侧的箭头，并从出现的列表中选择一个选项：
   - 选择**全部**以在满足任何失效准则时释放所有相对运动分量。在此点之后，连接器将对分析没有影响。
   - 选择**指定**以选择在满足任何失效准则时要释放的相对运动分量。
5. 要基于运动定义失效准则，请在编辑器的**位置失效准则**部分输入将导致失效的可用相对运动分量的相对位置限制。
6. 要基于力/力矩定义失效准则，请在编辑器的**力/力矩失效准则**部分输入将导致失效的可用相对运动分量的力/力矩限制。在Abaqus/Explicit分析中，您还可以输入将导致失效的约束相对运动分量的力/力矩限制。
7. 选择以下之一：
   - 如果您想继续定义行为选项，请点击**添加**，选择所需的行为选项，并继续连接器截面定义。有关定义其他行为选项的说明，请参见["使用连接器截面编辑器，"第15.17节"](pt03ch15s17.md)。
   - 如果您想查看或编辑现有的行为选项，请从**行为选项**列表中选择该选项。有关编辑行为选项的说明，请参见["使用连接器截面编辑器，"第15.17节"](pt03ch15s17.md)。
   - 如果您想保存连接器截面定义并退出编辑器，请点击**确定**。
