# 62.7.7 Creating a datum axis parallel to a line and passing through a point







You can position a datum axis by selecting an edge of the model and a point outside the edge. Abaqus/CAE creates a datum axis that is parallel to the edge and passes through the point.

![](../graphics/dtm-halp-nls.png)

**To create a datum axis parallel to a line and passing through a point:**

1. From the main menu bar, select ****Tools**![](../graphics/images/arrow.gif)**Datum****. The **Create Datum** dialog box appears. The dialog box outlines the types of datum geometry you can create. Abaqus/CAE displays prompts in the prompt area to guide you through the procedure. **Tip:**You can also create a datum axis using the ![](../graphics/ico_dtmAxisParallelToLine.png) tool, located in the module toolbox. For a diagram of the datum tools in the toolbox, see ["Using the Datum toolset," Section 62.2](pt06ch62s02.md).
2. From the list of types at the top of the dialog box, choose **Axis**. The **Method** list indicates the methods you can use to create a datum axis.
3. From the **Method** list, select **parallel to line, thru point**.
4. From the part or assembly in the current viewport, select an edge.
5. From the part or assembly in the current viewport, select a point. The datum axis appears. You cannot modify a datum axis created with this method; you must delete the old axis and create a new one.
