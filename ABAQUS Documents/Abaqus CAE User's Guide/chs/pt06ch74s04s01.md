# 74.4.1 自定义流颜色







您可以调整Abaqus/CAE用于在当前视口中显示活动流的流线的颜色。**Stream Plot Options**对话框使您能够以以下方式显示流线：
- 沿其整个长度单一的均匀颜色，
- 沿流线长度变化的连续轮廓光谱，或
- 沿流线长度变化的带状轮廓光谱。

连续轮廓显示渐变，没有任何纯色带，而带状轮廓显示颜色恒定的多个轮廓间隔。流线的轮廓反映当前主变量的值，并受**Contour Plot Options**对话框中指定的轮廓设置约束。更多关于轮廓选项的信息，请参见["自定义轮廓图," 第44.5节](pt05ch44s02.md)。

为均匀颜色显示选择的颜色也决定了Abaqus/CAE显示指示流动方向的流线箭头的颜色。更多关于切换箭头显示的信息，请参见["自定义流线粗细和箭头," 第74.4.2节](pt06ch74s04s02.md)。

**自定义流颜色的步骤：**

1. 找到**Color**选项。从Visualization模块的主菜单栏中选择****Options**![](../graphics/images/arrow.gif)**Stream****。**Color**选项出现在页面顶部。
2. 要指定颜色选项，请选择**Uniform color**、**Continuous contour**或**Banded contour**。
3. 如果您选择**Uniform color**，请执行以下额外步骤选择颜色：
   1. 单击颜色样本(![](../graphics/usi-colorsample.png))。Abaqus/CAE显示"选择颜色"对话框。
   2. 使用"选择颜色"对话框中的方法之一选择新颜色。更多信息，请参见["自定义颜色," 第3.2.9节"](pt01ch03s02s09.md)。
   3. 单击**OK**关闭"选择颜色"对话框。颜色样本和视口中的所有流线将更改为所选颜色。
4. 单击**OK**实现您的更改并关闭"Stream Plot Options"对话框。
