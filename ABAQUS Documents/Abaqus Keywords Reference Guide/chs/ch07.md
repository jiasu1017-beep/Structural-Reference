# *GAP CONDUCTANCE






### *GAP CONDUCTANCE在界面表面之间引入热传导。

此选项用于在紧密相邻（或接触）的表面之间提供传导热传递。它必须与 [*SURFACE INTERACTION](ch18abk50.md) 选项一起使用，或在 Abaqus/Standard 分析中与 [*GAP](ch07abk01.md) 选项一起使用。

**产品：** Abaqus/Standard  Abaqus/Explicit  Abaqus/CAE  

**类型：** Abaqus/Standard 中的模型数据；Abaqus/Explicit 中的历史数据  

**级别：** Abaqus/Standard 中的零件、零件实例、装配、模型；Abaqus/Explicit 中的步骤  

**Abaqus/CAE：** Interaction 模块

### **可选参数：**

DEPENDENCIES

将此参数设置为间隙热传导系数 *k* 所依赖的场变量数量。

PRESSURE

包含此参数以指示 *k* 是表面之间接触压力 *p* 的函数。省略此参数以将 *k* 定义为表面之间间隙 *d* 的函数。

USER

此参数仅适用于 Abaqus/Standard 分析。

包含此参数以在用户子程序 [`GAPCON`](../sub/sub-link.md#sub-xsl-gapcon) 中定义 *k*。使用此参数将导致 DEPENDENCIES 和 PRESSURE 参数以及任何数据行被忽略。

### **直接定义间隙热传导系数 (*k*) 的数据行：**

**第一行：**

根据需要重复此数据行，以定义间隙热传导对间隙间隙、间隙压力、平均表面温度、平均质量流量以及任何预定义场变量的依赖关系。必须至少指定两个数据行。当间隙热传导被定义为间隙的函数时，热传导系数在最后一个数据点之后立即降为零；因此，当间隙大于与最后一个数据点对应的值时，不存在热传导。

### **通过用户子程序定义间隙热传导系数 (*k*) 的数据行：**

使用 USER 参数时没有数据行。相反，在用户子程序 [`GAPCON`](../sub/sub-link.md#sub-xsl-gapcon) 中定义间隙热传导系数。
