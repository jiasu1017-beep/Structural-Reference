# 58.7 创建映射场





在 Property 模块、Interaction 模块或 Load 模块中，从主菜单栏选择 ****Tools**![](../graphics/images/arrow.gif)**Analytical Field**![](../graphics/images/arrow.gif)**Create****，以创建分析映射场。从外部数据源定义的分析场称为映射场。外部数据源可以是点云数据文件或 Abaqus 输出数据库（`.odb`）文件。有关映射场概述，请参见["使用分析映射场，" 第58.3节](pt06ch58s03.md)。
