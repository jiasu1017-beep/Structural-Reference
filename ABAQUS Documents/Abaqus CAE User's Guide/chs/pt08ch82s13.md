# 82.13 合并来自多个输出数据库的数据





**Combine ODBs** 插件使您能够将来自两个或多个 Abaqus 输出数据库（`.odb`）文件的模型数据和结果数据合并到一个新的输出数据库中。您可以使用此工具合并任何输出数据库文件，包括包含以下数据组合的模型数据和结果的输出数据库：
- 同一模型中不同子结构的输出。
- 来自两种不同分析产品（如 Abaqus/Standard 和 Abaqus/Explicit）的输出，作为协同仿真的一部分。

如果您想尝试不同输出数据库组合而不创建新的合并输出数据库，可以首先尝试使用叠加绘图一起显示来自不同输出数据库的数据。叠加绘图中的数据无法保存，但将来自不同输出数据库的结果数据一起绘制可以帮助您决定要在合并操作中包含哪些数据。有关叠加绘图的更多信息，请参见 [第79章 "Overlaying multiple plots](pt07ch79.md)"。

所选输出数据库的所有模型数据都包含在输出数据库中。但是，对于结果数据，您可以选择包含指定输出数据库中数据的子集。Abaqus/CAE 根据两个因素确定哪些结果数据包含在合并输出数据库中：您指定的过滤选项和您选择的主输出数据库。

**过滤器**

您可以过滤插件包含在合并输出数据库中的数据，仅包括所选步长或帧的数据、所选输出变量的数据，或这些选项的组合。例如，过滤器可以仅使结果数据包含在指定输出数据库的最后一步和最后一帧中，相同的过滤器可以规定仅 Mises 应力结果包含在合并输出数据库中。如果要在不同步骤中设置不同的过滤条件，您也可以建立多个过滤器。

**Combine ODBs** 插件还提供两级过滤：输出数据库特定过滤器（仅过滤来自单个输出数据库的结果）和默认过滤器（适用于整个作业）。输出数据库特定过滤器优先于默认过滤器，因此只有当您定义的默认过滤器与某个单独输出数据库的过滤器不冲突时，Abaqus/CAE 才使用默认过滤器中的设置。

过滤语法足够灵活，允许您指定多个步长、帧或输出变量值。您可以在逗号分隔的列表中指定多个步长名称，例如 `Step-1, Step-2, Step-4`。对于帧，您可以包含范围或单独的值；例如，输入 `1, 3, 5, 7:9` 会将帧 1、3、5、7、8 和 9 返回到合并输出数据库。

您也可以使用符号常量 `"ALL"`、`"FIRST"` 和 `"LAST"` 作为快捷方式来指定要包含的数据。这些选项使您能够包含来自所有步长或帧的结果数据，以及来自所有输出变量的数据，而不是一个或多个选定的变量。

**主输出数据库**

您在每个合并操作中指定一个输出数据库作为主输出数据库。**Combine ODBs** 首先将所有结果数据（根据过滤选择）从主输出数据库传输到合并输出数据库。然后插件在后续输出数据库中找到匹配步长和帧的结果数据，并仅将这些数据复制到合并输出数据库。此策略为合并结果数据提供了更连贯的结构。

**注意：**每次使用插件合并输出数据库时，Abaqus/CAE 都会以 XML 格式在配置文件中记录您的输出数据库和过滤选择。您可以将此文件上传到插件，以便进行后续合并操作。

**Combine ODBs** 插件从合并输出数据库中排除以下类型的数据：
- 历史数据
- 曲面数据
- 解析刚体零件实例
- 与场输出数据关联的局部坐标系

**合并来自多个输出数据库的数据：**

1. 在 Visualization 模块中，从主菜单栏选择 ****Plug-ins**![](../graphics/images/arrow.gif)**Tools**![](../graphics/images/arrow.gif)**Combine ODBs****。出现 **Combine ODBs** 对话框。
2. 在 **Job name** 字段中，为合并作业指定一个名称。
3. 如果要使用 XML 配置文件中的数据指定合并选项，请执行以下操作： 1. 在对话框中的任意位置点击鼠标按钮 3，然后从出现的列表中选择 **Read from XML File**。出现 **Select XML File** 对话框。 2. 导航到要使用的 XML 配置文件，然后点击 **OK**。有关更多信息，请参见 ["使用文件选择对话框，" 第3.2.10节](pt01ch03s02s10.md)。Abaqus/CAE 用配置文件中指定的输出数据库文件和过滤选择填充对话框中的表格。在运行合并操作之前，您可以根据需要进一步自定义这些设置。
4. 执行以下操作来指定主输出数据库： 1. 在对话框中的任意位置点击鼠标按钮 3，然后从出现的列表中选择 **Select Master ODB**。出现 **Select Master ODB** 对话框。 2. 导航到要使用的输出数据库，然后点击 **OK**。有关更多信息，请参见 ["使用文件选择对话框，" 第3.2.10节](pt01ch03s02s10.md)。
5. 如果要一次一个地向合并操作添加输出数据库，请执行以下操作： 1. 在表格中的任意位置点击鼠标按钮 3，然后从出现的列表中选择 **Add ODB**。Abaqus/CAE 向表格添加一个标有 **ODB** 的行，并显示 **Select ODB** 对话框。 2. 导航到要包含的输出数据库，然后点击 **OK**。有关更多信息，请参见 ["使用文件选择对话框，" 第3.2.10节](pt01ch03s02s10.md)。Abaqus/CAE 将输出数据库添加到所选行。 3. 对要添加到合并操作的每个输出数据库重复这些步骤。
6. 如果要一次向合并操作添加多个输出数据库，请执行以下操作： 1. 双击 **Name** 标题。出现 **Select ODBs** 对话框。 2. 导航到包含要包含的输出数据库的文件夹，选择输出数据库文件，然后点击 **OK**。有关更多信息，请参见 ["使用文件选择对话框，" 第3.2.10节](pt01ch03s02s10.md)。Abaqus/CAE 将选定的输出数据库作为新的 **ODB** 行添加到 **Combine ODBs** 对话框。
7. 如果需要，定义过滤选项以指定要包含在合并输出数据库中的步长、帧和输出变量。 - 为所有输出数据库定义单个步长、帧或输出变量过滤器： 1. 双击 **Step**、**Frame** 或 **Variable** 标题。 2. 在 **Set Values** 对话框中指定过滤条件。如果指定符号常量 `"ALL"`、`"FIRST"` 和 `"LAST"`，则必须包含周围的引号。 3. 点击 **OK**。Abaqus/CAE 将选定条件添加到默认过滤器的 **Step**、**Frame** 或 **Variable** 列，以及对话框中包含的每个输出变量的默认过滤器。 - 定义默认过滤选项或单个输出数据库的过滤选项： 1. 在 **Default** 行、**Master** 行或其中一个 **ODB** 行中，指定要用作默认过滤器或仅用于所选输出数据库的过滤器步长、帧和输出变量过滤器。 2. 如果需要，通过在相应行中点击鼠标按钮 3，选择 **Add filter** 从菜单中添加更多默认过滤器或输出数据库特定过滤器，并用其他过滤条件填充新行。
8. 切换 **Open the combined ODB after job completion** 以在作业完成后在视口中显示合并输出数据库。
9. 点击 **OK**。Abaqus/CAE 打开 **Odb Combine Log Message** 对话框，显示合并作业的进度并显示可能出现的任何错误。如果您选择的任何输出数据库来自早期版本，Abaqus/CAE 会升级输出数据库，将升级后的副本保存为新文件，并在合并作业中使用新文件。如果您选择了 **Open the combined ODB after job completion** 并且合并作业完成且没有错误，则 Abaqus/CAE 会在合并作业完成后在视口中打开合并输出数据库。
