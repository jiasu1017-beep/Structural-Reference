# *CONTACT CLEARANCE ASSIGNMENT






### *CONTACT CLEARANCE ASSIGNMENT在通用接触域中的表面之间分配接触间隙。

此选项用于定义接触表面之间的初始接触间隙，并控制如何在通用接触算法中解析初始接触过盈。

**产品：**Abaqus/Explicit  

**类型：**模型数据或历史数据  

**级别：**模型、步骤

### **此选项没有关联的参数。**

### **用于定义非默认接触间隙的数据行：**

**第一行：**

根据需要重复此数据行。如果接触间隙分配重叠，则最后一个分配适用于重叠区域。
