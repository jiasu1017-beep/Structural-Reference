# 65.4.3 Suppressing features







Select ****Feature**![](../graphics/images/arrow.gif)**Suppress**** from the main menu bar to suppress selected features. Suppressing features is equivalent to temporarily removing them from the part or assembly. Suppressed features remain suppressed when you regenerate a part or assembly. You can also suppress a feature by clicking mouse button 3 on the feature in the Model Tree and selecting **Suppress** from the menu that appears.

You cannot suppress the base feature. In addition, if you suppress parent features, all of their child features are also suppressed automatically. Suppressed features can be restored by selecting ****Feature**![](../graphics/images/arrow.gif)**Resume**** from the main menu bar or by clicking mouse button 3 on the feature in the Model Tree. When you resume a suppressed feature, its children are not resumed unless you select them. To help you select a feature's children, you can click mouse button 3 on the feature in the Model Tree and select **Show children** from the menu that appears.

**To suppress features:**

1. From the main menu bar, select ****Feature**![](../graphics/images/arrow.gif)**Suppress****. **Tip:**You can also suppress features using the ![](../graphics/ico_featureSuppress.png) tool located in the module toolbox.
2. Select one or more features to suppress. You can select features from the current viewport or from the Model Tree. For more information, see [Chapter 6, "Selecting objects within the viewport](pt01ch06.md)," and ["Working with the Model Tree and the Results Tree," Section 3.5](pt01ch03s05.md). Abaqus/CAE highlights the selected feature. If you selected features that have child features, Abaqus/CAE also highlights the children. If you selected features from the viewport, click **Done** in the prompt area when you are finished making selections.
3. If you selected features from the viewport, click **Yes** in the prompt area. If you selected features from the Model Tree, click mouse button 3 on one of the feature names and select **Suppress** from the menu that appears. The selected features and any child features disappear from the viewport.
4. To exit the suppress procedure, either - click the cancel button ![](../graphics/fts-cancel.gif) in the prompt area, or - click mouse button 2 anywhere in the Abaqus/CAE window, or - select another operation from the **Feature** menu.
