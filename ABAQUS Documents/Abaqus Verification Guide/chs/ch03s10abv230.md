# 3.10.5 热传递模型变化：稳态







**产品：**Abaqus/Standard  

### 测试的单元

DC2D4    DS3    

### 测试的功能

在稳态热传递分析期间移除和添加连续体和壳热传递单元。

### 问题描述

**模型：**

模型在*x*–*y*平面中的尺寸为5.0×2.0，平面外尺寸为1.0。

**材料：**

| 导热率 | 7.872×10^4 |
| --- | --- |
| 密度 | 0.2829 |

**载荷和边界条件：**

模型的左侧保持在![](../graphics/ver_eqn01255.gif) = 0.0。对于使用DC2D4单元的模拟，模型的右侧存在膜条件；对于使用DS3壳单元的模拟，存在温度边界条件。环境温度为![](../graphics/ver_eqn01255.gif) = 100.0，膜系数*h*为1.0。对于壳问题，温度边界条件在右侧边缘为100.0。获得稳态解。然后移除模型的三分之二。当移除单元时，沿新外部边界的温度保持固定。移除的单元在最后一步中重新添加到模型中，对于连续体模型在右侧施加新的膜条件，对于壳模型施加温度边界条件。新的环境温度为![](../graphics/ver_eqn01255.gif) = 200.0，使用相同的膜系数。温度边界条件为200.0。
