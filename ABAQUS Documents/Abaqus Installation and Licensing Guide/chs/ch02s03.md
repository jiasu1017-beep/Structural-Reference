# 2.3 Abaqus FLEXnet 许可证安装详情



Abaqus 6.14 使用 FLEXnet Licensing Version 11.6.1。许可证安装过程可用于以下场景：
- 安装 FLEXnet Licensing 管理工具并自动配置和启动 FLEXnet Abaqus 6.14 许可证服务器；或
- 仅安装 FLEXnet Licensing 管理工具。

如果以前使用 Abaqus 许可证安装程序安装和配置了以前版本的 FLEXnet Licensing 的现有 FLEXnet 许可证服务器，则在安装过程中会自动升级该服务器。如果安装程序检测到您的计算机上有来自高于 Version 11.6.1 的 FLEXnet Licensing 版本的现有许可证服务器，您可以选择覆盖现有服务器或终止安装过程。要在不覆盖现有许可证服务器的情况下更新 FLEXnet 许可证文件，请参阅["更新 Abaqus FLEXnet 许可证文件，"第 3.2 节](ch03s02.md)中的过程。

用于安装 Abaqus FLEXnet 许可证服务器或停止和删除现有许可证服务器的命令和选项列在["FLEXnet 许可证安装，"第 2.1.3 节](ch02s01.md#sgb-chp-install-glance-lic)中。

对于冗余许可证服务器配置，您可以安装许可证管理工具并配置许可证服务器，但需要手动启动许可证服务器（请参阅["冗余 FLEXnet 许可证服务器配置，"第 3.4 节](ch03s04.md)）。如果选择仅安装许可证管理工具，则必须使用 FLEXnet 许可证服务器管理器 `lmgrd` 手动启动许可证服务器（请参阅["FLEXnet 许可证服务器管理器 lmgrd，"第 3.5 节](ch03s05.md)。

如果您要安装 Abaqus 6.14 FLEXnet 许可证服务器，您必须将 Abaqus 6.14 许可证文件保存在安装许可证服务器的计算机上。FLEXnet 许可证文件是包含 Abaqus 6.14 许可证的文件，该文件附加在 SIMULIA 发送的电子邮件消息中（标识为"主题：LK: *站点 ID* Abaqus 6.14 许可证文件"，其中 *站点 ID* 是分配给您站点的客户编号），或由您当地的办事处或代表提供。

FLEXnet Abaqus 许可证只需安装在一台网络计算机上，除非您使用的是冗余许可证服务器配置。许可证实用程序的磁盘空间要求范围为 10-25 MB，具体取决于平台。您选择的许可证计算机应能被您计划运行 Abaqus 的所有计算机访问，并且应该是稳定的机器，不会频繁重启或关闭。有关许可证的更多信息，请参阅[第 3 章，"Abaqus 许可证"](ch03.md)。

执行安装过程时，会为请求您指定目录或文件位置的提示提供浏览功能。系统会提示您输入以下信息：
- FLEXnet 许可证文件的位置。安装过程会将许可证文件保存在 `*simulia_dir*/License` 目录中，使用默认名称 `simulialm.lic`。
- 安装目录。指定 SIMULIA 父目录 (`*simulia_dir*`)，即存储许可证目录的位置。安装目录的默认位置在 Linux 平台上是当前工作目录，在 Windows 平台上是 `C:\SIMULIA`。

**注意：**在 Windows 平台上，FLEXnet 许可证服务器由安装程序启动，将在重启后自动启动。在 Linux 平台上，许可证服务器由安装程序启动，但您应该将重启许可证服务器的命令包含在许可证服务器主机的系统启动文件中，以便 Abaqus 许可证服务器在系统重启后自动重启。重启许可证服务器的命令写入 `*simulia_dir*/License/installation_info` 目录中的 `license_startup.csh` 文件。请联系您的系统管理员或平台供应商，获取有关计算机系统启动文件的信息。

安装完成后，日志文件将写入 `*simulia_dir*/License/installation_info` 目录。日志文件有助于排除问题，不应删除。

安装 Abaqus 文档并安装和运行许可证服务器后，您应该继续在将用于运行产品的每台计算机上进行 Abaqus 产品安装。除非您会在该计算机上运行 Abaqus，否则无需在许可证服务器主机上安装 Abaqus 产品。
