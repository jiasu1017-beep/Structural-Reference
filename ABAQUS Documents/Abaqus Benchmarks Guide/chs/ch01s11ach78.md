# 1.11.3 呼吸模式下球体的声辐射阻抗







**产品：** Abaqus/Standard  Abaqus/Explicit

本示例计算球体在"呼吸"模式下的声辐射；即当运动由均匀径向速度组成时。本示例说明了一种简单的吸收边界条件与声学连续体单元、界面单元和声学无限单元结合使用的方法。结果与经典结果进行了比较。

在 Abaqus/Standard 中在 0 到 20 Hz 的频率范围内进行稳态动态分析。瞬态模拟也在 Abaqus/Explicit 中以 1 Hz 的激励频率进行。可以通过更改输入文件中定义的参数来测试不同的激励频率。

稳态分析也在 Abaqus/Standard 中使用三维和轴对称声学无限单元进行。这些声学无限单元使用基于球体辐射模式的无限方向基函数（参见["声学无限单元"，《Abaqus 理论指南》第 3.3.2 节](../stm/stm-link.md#stm-elm-acousticinfinite)）。因此，球形呼吸模式的阻抗被这些单元精确建模，使用声学无限单元获得的结果应与使用球形辐射阻抗条件获得的结果紧密匹配。然而，对于更复杂的波形状，预期声学无限单元将给出更准确的结果，因为用于模拟无限方向上声场变化的基函数更丰富。建模本问题所需的无限单元网格仅由球面上的声学无限单元组成，添加了声学-结构界面单元（具有相同的网格拓扑）以便于在球面表面施加速度边界条件。无限单元的参考点位于球心。对于本问题，单个输入文件 [acousticimpsphere_acin.inp](../eif/acousticimpsphere_acin.inp) 包含用三维声学无限单元 ACIN3D3、ACIN3D4、ACIN3D6 和 ACIN3D8 建模的球体部分。另外两个输入文件 [acousticimpsphere_acinax2.inp](../eif/acousticimpsphere_acinax2.inp) 和 [acousticimpsphere_acinax3.inp](../eif/acousticimpsphere_acinax3.inp) 分别使用 ACINAX2 和 ACINAX3 单元的半圆形网格对问题进行建模。

对于 Abaqus/Explicit 中的瞬态分析，Bayliss 等人边界条件使用球形辐射阻抗条件强制执行。在网格的内半径处，三维情况下的壳单元和轴对称情况下的梁单元通过绑定约束连接到声学域。瞬态分析也在 Abaqus/Explicit 中使用三维和轴对称声学无限单元进行，而不是定义阻抗。另外两个输入文件 [acousticimpsphere_acin3d4_xpl.inp](../eif/acousticimpsphere_acin3d4_xpl.inp) 和 [acousticimpsphere_acinax2_xpl.inp](../eif/acousticimpsphere_acinax2_xpl.inp) 分别对三维和轴对称问题进行建模。

### 输入文件

##### **Abaqus/Standard 输入文件**

[acousticimpsphere_acax8_bayliss.inp](../eif/acousticimpsphere_acax8_bayliss.inp)

使用 ACAX8 和 ASI3A 单元以及 Bayliss 等人边界条件的模型。

[acousticimpsphere_acax8_auto.inp](../eif/acousticimpsphere_acax8_auto.inp)

使用 ACAX8 和 ASI3A 单元以及 Bayliss 等人边界条件的模型，使用球形辐射条件定义。

[acousticimpsphere_acax8_planew.inp](../eif/acousticimpsphere_acax8_planew.inp)

具有平面波边界条件的 ACAX8 单元。

[acousticimpsphere_acax6_bayliss.inp](../eif/acousticimpsphere_acax6_bayliss.inp)

与 acousticimpsphere_acax8_bayliss.inp 相同的问题，但使用 ACAX6 单元。

[acousticimpsphere_ac3d15.inp](../eif/acousticimpsphere_ac3d15.inp)

使用 AC3D15 单元的三维球形模型版本。

[acousticimpsphere_ac3d20.inp](../eif/acousticimpsphere_ac3d20.inp)

与 acousticimpsphere_ac3d15.inp 相同的问题，但使用 AC3D20 单元。

[acousticimpsphere_ac3d10.inp](../eif/acousticimpsphere_ac3d10.inp)

与 acousticimpsphere_ac3d20.inp 相同的问题，但使用 AC3D10 单元。

[acousticimpsphere_acin.inp](../eif/acousticimpsphere_acin.inp)

与 acousticimpsphere_ac3d20.inp 相同的问题，但球形部分用 ACIN3D3、ACIN3D4、ACIN3D6 和 ACIN3D8 单元建模。

[acousticimpsphere_acinax2.inp](../eif/acousticimpsphere_acinax2.inp)

与 acousticimpsphere_acax.inp 相同的问题，但球形部分用 ACINAX2 单元建模。

[acousticimpsphere_acinax3.inp](../eif/acousticimpsphere_acinax3.inp)

与 acousticimpsphere_acax.inp 相同的问题，但球形部分用 ACINAX3 单元建模。

##### **Abaqus/Explicit 输入文件**

[acousticimpsphere_acax4r_xpl.inp](../eif/acousticimpsphere_acax4r_xpl.inp)

使用 ACAX4R 单元以及 Bayliss 等人边界条件的模型。

[acousticimpsphere_ac3d8r_xpl.inp](../eif/acousticimpsphere_ac3d8r_xpl.inp)

使用 AC3D8R 单元的三维球形模型版本。

[acousticimpsphere_acin3d4_xpl.inp](../eif/acousticimpsphere_acin3d4_xpl.inp)

使用 ACAX4R 和 ACINAX2 单元的模型。

[acousticimpsphere_acinax2_xpl.inp](../eif/acousticimpsphere_acinax2_xpl.inp)

使用 AC3D8R 和 ACIN3D4 单元的三维球形模型版本。
