# 3.2.31 将 PAM-CRASH 输入文件转换为部分 Abaqus 输入文件





**产品：**Abaqus/Explicit

##### **参考**

- [『Abaqus 执行程序：概述』，第 3.1.1 节](pt01ch03s01abo02.md)

### 概述

PAM-CRASH 到 Abaqus 的转换器将 PAM-CRASH 输入文件中的某些关键字转换为 Abaqus/Explicit 中的等效项。

### 使用转换器

转换器需要由 PAM-CRASH Version 2002 或更高版本创建的输入文件。输入文件可以是任何名称和扩展名。

转换的 PAM-CRASH 数据条目列于下表中。其他 PAM-CRASH 关键字和数据将被跳过并在日志文件中注明。

转换器创建一个仅包含模型数据的部分 Abaqus 输入文件。您必须提供历史数据（包括输出数据）才能完成输入。

#### 单元编号和分组

所有单元必须具有唯一的单元编号。分配了相同 PART 标识号的单元被分组到一个单元集中。

除了由 SPRING 和 KJOIN 转换而产生的连接器单元外，截面属性需要在 PART 部分而不是在单元部分中单独输入。具有不同材料或截面属性的单元应分配不同的 PART 标识号；即，相同的材料和截面属性必须适用于分组在同一单元集中的所有单元。

如果由 SPRING 和 KJOIN 转换而产生的单元具有不同的单元数据（如用于定义局部方向的框架编号），并且被分配了相同的 PART 标识号，转换器会自动将它们分到不同的单元集中。

#### 材料模型

转换器仅支持下表中所示的材料模型（第 3.2.31-3 表）。所有不支持的 1 到 99 类型的材料模型都作为双线性弹塑性材料进行转换，所有其他材料类型在需要应力-应变定律定义时都作为线弹性材料进行转换。在这些情况下，转换器会提供材料属性的标称值。

#### 历史部分数据

转换器根据 PAM-CRASH 文件控制部分中的关键字（TITLE 除外）部分创建历史部分，如表 3.2.31-1 所示（第 3.2.31-1 表）。其他控制数据不支持。

### 命令行选项

**job**

此选项用于指定转换器输出的 Abaqus 输入文件的名称。Abaqus 输入文件的名称必须不带 `.inp` 扩展名。转换器创建的诊断信息写入名为 `*job-name*_frompam.log` 的文件。

**input**

此选项用于指定包含 PAM-CRASH 数据的文件的名称。文件的名称必须带有文件扩展名。

**pLinkConnectors**

此选项用于指定是否在 PLINK 转换中包含连接器单元。默认值为 `ON`。

**splitAirbagElements**

此选项用于指定是否将 4 节点气囊薄膜单元拆分为两个 3 节点气囊薄膜单元。默认值为 `ON`。气囊薄膜单元由 MEMBR 和 MATER / 类型 150 和 151 的转换产生。此选项仅在 PAM-CRASH 输入文件中指定了关键字 BAGIN 时有效。

**autoKJoinStops**

此选项用于向所有 KJOIN 连接器单元的行为添加连接器止挡。如果在端点处插值的力-位移曲线刚度超过相邻点插值的刚度 10 倍，则在靠近端点的点处定义一个连接器止挡。默认值为 `OFF`。
