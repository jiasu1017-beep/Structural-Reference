If these are not specified, they are constructed as for all other shell elements—by averaging over the shell elements attached to each node.

### Element property definition

The only element property used is the thickness; the number of integration points is ignored, since the elements work on the basis of section properties.

### Element output

Nodes 1, 2, and 3 on the element define side *B* and nodes 4, 5, and 6 define side *A* (see [Figure 32.9.2--1](pt06ch32s09ael40.md#elinespring-strain-not)). The sign of the crack is defined by the surface of the shell from which the crack originates, which you identify when you define the depth of the crack (see ["Line spring elements for modeling part-through cracks in shells," Section 32.9.1](pt06ch32s09alm54.md)). If the crack originates from the positive surface of the shell, *sign*(crack)=1.0; if the crack originates from the negative surface of the shell, *sign*(crack)=1.0.

The vector ![](../graphics/usb_eqn01086.gif) is defined by the right-hand rule from the cross product of the tangent, ![](../graphics/usb_eqn01319.gif), which is positive going from node 1 to node 3 of the element, and the normal, ![](../graphics/usb_eqn00842.gif), defined when the coordinates are given (or by a user-defined normal definition). For element type LS3S the vector ![](../graphics/usb_eqn01086.gif) must point into the model (away from the symmetry plane). For element type LS6 the vector ![](../graphics/usb_eqn01086.gif) must point from side *A* to side *B*.

The *J*-integral is provided at each integration point. If elastic-plastic material behavior is defined, the elastic and plastic parts of *J* are provided. The stress intensity factors, *K*, are also provided corresponding to the elastic parts of *J*.
