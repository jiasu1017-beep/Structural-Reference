# 11.4.1 零件建模空间





创建新零件时，必须指定零件所在的建模空间。您可以分配以下三种类型的建模空间：

**三维**

Abaqus/CAE将零件嵌入*X*、*Y*、*Z*坐标系中。三维零件可以包含实体、壳、线框、切割、圆角和倒角特征的任何组合。您可以使用三维实体、壳、梁、桁架或膜单元对三维零件进行建模。

**二维平面**

Abaqus/CAE将零件嵌入*X*–*Y*平面中。二维平面零件只能包含平面壳和线框特征的组合，所有切割特征都定义为平面通孔切割。您可以使用二维实体连续单元以及桁架或梁单元对二维平面零件进行建模。

**轴对称**

Abaqus/CAE将零件嵌入*X*–*Y*平面中，*Y*轴表示旋转轴。轴对称零件只能包含平面壳和线框特征的组合，所有切割特征都定义为平面通孔切割。您可以使用轴对称实体连续单元或轴对称壳单元对轴对称零件进行建模。

建模空间是指零件嵌入的空间，而不是零件本身的拓扑。因此，您可以使用拓扑上二维的壳特征或一维的线框特征创建三维零件。创建零件后，您可以通过在模型树中右键单击零件并从出现的菜单中选择**编辑**来更改零件的建模空间。

Abaqus/CAE使用以下方法来确定导入零件的建模空间：
- 当您从包含第三方格式几何的文件导入零件时，可以指定零件的建模空间，前提是Abaqus/CAE确定它不必是三维的。
- 当您从输出数据库导入网格时，Abaqus/CAE根据输出数据库中存储的信息确定新零件的建模空间。
- 当您从输入文件导入网格时，Abaqus/CAE根据单元类型确定新零件的建模空间。
- 当您在网格模块中创建网格零件时，网格零件的建模空间与原始零件的建模空间相同。
