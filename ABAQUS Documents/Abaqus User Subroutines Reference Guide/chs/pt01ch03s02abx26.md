# 3.2.26 元素矩阵组装实用工具





**产品：** Abaqus/Standard  

##### **参考**

- ["生成矩阵，" 第10.3节](pt04ch10s03.md)

### 概述

**abaqus mtxasm**实用工具组装SIM文档中包含的元素矩阵，并且（可选）将组装后的矩阵写入文本文件。如果SIM文档中已有组装后的矩阵，您可以使用此实用工具将它们写入文本文件。

### 命令摘要

| **abaqus mtxasm** | [**job**](pt01ch03s02abx26.md#job1-usb-int-dmtxassemblyutil)=*name-of-assembled-mtx-sim-doc* |
| --- | --- |
|  | [[**oldjob**](pt01ch03s02abx26.md#oldjob2-usb-int-dmtxassemblyutil)=*name-of-element-mtx-sim-doc*] [[**text**](pt01ch03s02abx26.md#text3-usb-int-dmtxassemblyutil)] |

### 命令行选项

**job**

此选项用于指定包含组装矩阵的SIM文档的名称，或者将向其中写入组装矩阵的SIM文档的名称（取决于操作模式）。

**oldjob**

此选项用于指定包含要组装的元素矩阵的SIM文档的名称。如果未使用此选项，则使用**job**选项指定的SIM文档必须已存在并包含组装矩阵。

**text**

此选项用于将组装矩阵以矩阵输入格式写入文本文件。每个矩阵被写入一个文件，该文件遵循命名约定*jobname*_*matrix**N*.`mtx`，其中*jobname*是使用**job**选项指定的名称，*matrix*是四字母标识符（`STIF`、`MASS`、`DMPV`、`DMPS`或`LOAD`），指示与结构或热矩阵相关的矩阵类型，*N*是增量号。
