### 7.4.1 过程示例

过程的步骤使用以下方法显示：

- `getFirstStep`
- `getNextStep`
- `getLoopStep`

以下类型的步骤可用于过程：

- `AFXDialogStep`：此步骤提供对话框界面。
- `AFXPickStep`：此步骤提供允许在视口中选择实体的界面。

以下示例显示如何编写一个简单的单步过程模式，该模式使用对话框步骤。后续示例将扩展此示例以展示如何使用更多步骤。

```
from abaqusGui import *
from plateDB import PlateDB

class PlateProcedure(AFXProcedure):

    #~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
    def __init__(self, owner):

        AFXProcedure.__init__(self, owner)

        self.cmd = AFXGuiCommand(self, 'Plate', 'examples')
        self.nameKw = AFXStringKeyword(self.cmd, 'name', True)
        self.widthKw = AFXFloatKeyword(self.cmd, 'width', True)
        self.heightKw = AFXFloatKeyword(self.cmd, 'height', True)

    #~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
    def getFirstStep(self):

        self.cmd.setKeywordValuesToDefaults()
        db = PlateDB(self)
        return AFXDialogStep(self, db)
```

### 7.4.2 过程构造函数

您通过从`AFXProcedure`派生新类开始编写过程模式。在`AFXProcedure`构造函数的主体中，必须调用基类构造函数并传入owner，这是此过程所属的模块或工具集GUI。可选地，您可以为过程类型传递一个值。类型的默认值为NORMAL。类型定义了当新过程在另一个过程正在执行时被激活时会发生什么。

过程的类型可以是NORMAL或SUBPROCEDURE。当激活正常过程时，它会取消当前正在执行的任何过程。当激活子过程时，它会挂起正常过程或取消另一个子过程。如果过程被挂起，它会在子过程完成后恢复。

视图过程（例如，平移、旋转和缩放）是特殊类型的过程，无法被挂起。当另一个过程被激活时，视图过程总是被取消，当它们被激活时，它们总是挂起任何当前正在执行的过程。

默认情况下，基础设施使用类名来识别过程。如果您需要同时执行同一过程的多个实例，您需要通过调用`setModeName`方法向基础设施区分它们的名称。

从`AFXProcedure`派生新类后，定义模式所需的命令和关键字。关键字存储为模式的成员，以便步骤可以访问它们。如果您在`AFXGuiCommand`构造函数中设置*registerQuery*=True，模式将在激活时查询命令指定的kernel对象，并自动设置命令关键字的值。有关更多信息，请参阅["保持GUI和命令最新，" 第6.5.3节](pt04ch06s05.md#cus-app-commands-gui-uptodate)。如果没有与您的命令关联的kernel对象（例如，在创建新对象时），您可以通过在构造函数中指定默认值来设置关键字值。

如果您有一个想要用于重新建立对话框默认值的默认对象，可以使用模式的`registerDefaultsObject`方法注册一个对象，当用户按下对话框中的"默认值"按钮时将查询该对象的值。有关更多信息，请参阅["默认值对象，" 第6.5.16节](pt04ch06s05.md#cus-app-commands-gui-defaults)。

默认情况下，对话框显示为非模态的。您可以通过调用`self.setModal(True)`将对话框显示为模态。在大多数情况下，您只需在模式中设置一次模态；但是，您可以通过在`getFirstDialog`或`getNextDialog`方法中调用`setModal`方法根据需要更改模态。有关更多信息，请参阅["模态与非模态，" 第5.2节](pt03ch05s02.md)。

### 7.4.3 getFirstStep

您必须始终为模式编写`getFirstStep`方法。`getFirstStep`方法应返回模式的第一步。在["过程示例，" 第7.4.1节](pt04ch07s04.md#cus-mod-procedure-example)中，指向过程的指针被传递到对话框构造函数中。对话框将使用此指针访问模式的关键字。

如果您希望每次显示对话框时出现相同的默认值，必须在返回对话框之前调用`setKeywordValuesToDefaults()`方法，如["过程示例，" 第7.4.1节](pt04ch07s04.md#cus-mod-procedure-example)中所示。

### 7.4.4 getNextStep

如果您的模式包含多个步骤，除了`getFirstStep`方法外，还必须编写`getNextStep`方法。之前的步骤被传递到`getNextStep`方法中，这样您就可以确定用户在步骤序列中的位置并相应地采取行动。`getNextStep`方法应返回序列中的下一步，或者返回`None`以表示已完成从用户收集输入。以下示例是["过程示例，" 第7.4.1节](pt04ch07s04.md#cus-mod-procedure-example)中示例的修改版本，说明了如何通过三个步骤的序列而不是仅仅一个来收集用户的输入：

```
#~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
def getFirstStep(self):

    self.cmd.setKeywordValuesToDefaults()
    self.plateWidthDB = None
    self.plateHeightDB = None
    db = PlateNameDB(self)
    self.step1 = AFXDialogStep(self, db)
    return self.step1

#~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
def getNextStep(self, previousStep):

    if previousStep == self.step1:
        if not self.plateWidthDB:
            self.plateWidthDB = PlateWidthDB(self)
        self.step2 = AFXDialogStep(self, self.plateWidthDB)
        return self.step2

    elif previousStep == self.step2:
        if not self.plateHeightDB:
            self.plateHeightDB = PlateHeightDB(self)
        self.step3 = AFXDialogStep(self, self.plateHeightDB)
        return self.step3

    else:
        return None
```

### 7.4.5 getLoopStep

如果您希望过程循环，必须编写`getLoopStep`方法。`getLoopStep`方法在基类中定义为返回`None`，表示模式将单次运行。您可以重新定义`getLoopStep`方法并返回一个过程应循环回到的步骤。以下示例显示如何使上一节中显示的过程在完成最后一步后循环回第一步：

```
def getLoopStep(self):
    return self.step1
```

### 7.4.6 AFXDialogStep

`AFXDialogStep`类允许您在过程中显示对话框。要创建对话框步骤，必须提供过程、对话框，以及可选的提示行提示。如果您不提供提示，Abaqus使用默认提示`Fill out the *dialog box title* dialog`。以下是单步过程中对话框步骤的示例：

```
def getFirstStep(self):

    db = PlateDB(self)
    prompt = 'Enter plate dimensions in the dialog box'
    return AFXDialogStep(self, db, prompt)
```

在大多数情况下，过程会有多个步骤。由于过程能够返回到之前的步骤，您必须编写在过程中不重复构造对话框的过程。您可以通过初始化过程成员，然后在`getNextStep`期间检查成员来防止过程重复构造对话框，如以下示例所示：

```
def getFirstStep(self):

    self.plateWidthDB = None
    self.plateHeightDB = None
    db = PlateNameDB(self)
    self.step1 = AFXDialogStep(self, db)
    return self.step1

def getNextStep(self, previousStep):

    if previousStep == self.step1:
        if not self.plateWidthDB:
            self.plateWidthDB = PlateWidthDB(self)
        self.step2 = AFXDialogStep(self, self.plateWidthDB)
        return self.step2

    elif previousStep == self.step2:
        if not self.plateHeightDB:
            self.plateHeightDB = PlateHeightDB(self)
        self.step3 = AFXDialogStep(self, self.plateHeightDB)
        return self.step3

    else:
        return None
```
