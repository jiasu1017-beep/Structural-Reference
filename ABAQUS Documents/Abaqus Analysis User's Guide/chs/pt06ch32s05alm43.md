### 概述

本节描述的功能用于使用连续体方法对内聚单元进行建模，该方法假定内聚区包含有限厚度的材料，可以使用Abaqus中的常规材料模型进行建模。如果内聚区非常薄，实际上可以认为是零厚度，则本构响应通常以牵引-分离定律的形式描述；另一种方法在["使用牵引-分离描述定义内聚单元的本构响应，" 第32.5.6节](pt06ch32s05alm45.md)中讨论。

建模为连续体的内聚单元的本构响应：
- 可以使用常规材料模型以宏观材料特性（如刚度和强度）的形式定义；
- 可以指定为内置材料模型或用户定义材料模型；
- 可以在Abaqus/Explicit中包含材料损伤和失效的影响；和
- 也可以在Abaqus/Standard的低循环疲劳分析中包含材料损伤和失效的影响。

### 具有常规材料模型的内聚单元行为

Abaqus中内聚单元的常规材料模型（包括用户定义模型）的实现基于关于内聚层变形状态的某些假设。考虑两类不同的问题：有限厚度粘接层的建模和垫片的建模。

对于这些类型的問題，可以使用内聚单元在Abaqus/Standard和Abaqus/Explicit中进行损伤建模（详见["渐进损伤和失效，" 第24.1.1节](pt05ch24s01abo21.md)，了解损伤模型的详细信息）。您可能需要修改粘接材料的损伤模型，以考虑粘接失效可能发生在粘接剂与被粘物之间的界面而非粘接材料内部这一事实。

当与Abaqus中的常规材料模型一起使用时，内聚单元使用真实应力和应变测量。当与基于牵引-分离描述的材料模型一起使用时（详见["使用牵引-分离描述定义内聚单元的本构响应，" 第32.5.6节](pt06ch32s05alm45.md)，了解这种方法），内聚单元使用名义应力和应变测量。

内聚单元的频率特性通过Abaus/Explicit中自动选择时间增量的算法来考虑（["显式动态分析，" 第6.3.3节](pt03ch06s03at08.md)）。在许多涉及粘接剂或垫片的应用中，内聚单元可能比其他单元薄得多，这往往会减小稳定时间增量。关于使用内聚单元时如何避免稳定时间增量显著减小的问题，请在["使用内聚单元建模，" 第32.5.3节中的"Abaqus/Explicit中的稳定时间增量"](pt06ch32s05alm42.md#usb-elm-ecohesiveusage-stabletime)查看更多讨论。

#### 有限厚度粘接层的建模

对于有限厚度的粘接层，假定内聚层仅承受一个直接的应变分量，即厚度应变，以及两个横向剪切应变分量（二维问题为一个横向剪切应变分量）。应变的其他两个直接分量（直接膜应变）和面内（膜）剪切应变假定为零，用于本构计算。更具体地说，厚度应变和横向剪切应变是根据单元运动学计算的。然而，膜应变不是根据单元运动学计算的；它们只是假定为零用于本构计算。这些假设在粘接层相对较薄且 compliant（相对于粘接剂）的情况下是合适的。与粘接剂相比，刚性较大（与粘接剂相比）的两个部件粘合在一起时，上述运动学假设在内聚层内部除其边缘附近外几乎到处都成立。

可以定义额外的线性弹性横向剪切行为，以提供更大的稳定性给内聚单元，特别是在损伤发生后。横向剪切行为假定独立于常规材料响应，不经历任何损伤。

| **输入文件用法：** | 使用以下选项（仅在需要定义解耦横向剪切响应时才需要第二个选项）： |
| --- | --- |
|  | ``` [*COHESIVE SECTION](../key/key-link.md#usb-kws-mcohesivesection), RESPONSE=CONTINUUM [*TRANSVERSE SHEAR STIFFNESS](../key/key-link.md#usb-kws-mtransshearstiff) ``` |

| **Abaqus/CAE用法：** | 属性模块：**Create Section**：选择**Other**作为section **Category**和**Cohesive**作为section **Type**：**Response**：**Continuum** |
| --- | --- |
|  | Abaqus/CAE不支持内聚截面的横向剪切行为。 |

#### 垫片和/或小粘接贴片的建模

垫片和/或小粘接贴片的建模涉及内聚层没有横向约束的情况。因此，层可以自由地在侧向方向上以无应力方式膨胀。应用领域包括单个点焊和垫片。本构计算假定只有一个直接的应力分量，即厚度方向法向应力。所有其他应力分量，包括横向剪切应力分量，都假定为零。

使用此选项提供的垫片建模功能与Abaus/Standard中垫片单元系列相比有一些优点。内聚单元是完全非线性的（单元运动学正确考虑了有限应变以及有限旋转），可以在动态分析中贡献质量和阻尼，并且在Abaqus/Explicit中可用。以这种方式建模的垫片响应类似于使用仅具有厚度方向行为的特殊用途垫片单元进行建模（见["在模型中包含垫片单元，" 第32.6.3节](pt06ch32s06alm48.md)）。

如果需要，可以定义解耦的线性弹性横向剪切行为。横向剪切行为可以定义垫片和/或粘接贴片的响应，或在厚度方向响应中损伤发生后提供稳定性。横向剪切响应没有相关的损伤。

| **输入文件用法：** | 使用以下选项（仅在需要定义解耦横向剪切响应时才需要第二个选项）： |
| --- | --- |
|  | ``` [*COHESIVE SECTION](../key/key-link.md#usb-kws-mcohesivesection), RESPONSE=GASKET [*TRANSVERSE SHEAR STIFFNESS](../key/key-link.md#usb-kws-mtransshearstiff) ``` |

| **Abaqus/CAE用法：** | 属性模块：**Create Section**：选择**Other**作为section **Category**和**Cohesive**作为section **Type**：**Response**：**Gasket** |
| --- | --- |
|  | Abaqus/CAE不支持内聚截面的横向剪切行为。 |

### 输出

Abaqus中的所有标准输出变量（["Abaqus/Standard输出变量标识符，" 第4.2.1节](pt02ch04s02abv01.md)和["Abaqus/Explicit输出变量标识符，" 第4.2.2节](pt02ch04s02xbv01.md)）可用于具有常规材料模型的内聚单元。附加横向剪切响应产生的应力使用输出变量TSHR13和（在三维中）TSHR23单独报告。这些应力不会添加到使用输出变量S报告的常规材料点应力中。
