# 5.6.9 保存用户定义的视图





使用**视图**工具栏中的保存工具 ![](../graphics/ico_viewSave.png) 打开**保存视图**对话框并保存用户定义的视图。**保存视图**对话框如下图所示：

![](../graphics/viw-savetoolbox-nls.png)

使用**缩放和位置**选项来确定保存的视图是否包含缩放因子和位置信息。

**要保存用户定义的视图：**

1. 从主菜单栏中，选择****View**![](../graphics/images/arrow.gif)**Save****。**提示：**你也可以通过单击**视图**工具栏中的![](../graphics/ico_viewSave.png)工具来保存视图。如果你只有一个视口，Abaqus/CAE 会立即打开**保存视图**对话框。如果你有多个视口，请点击要保存的视图的视口；然后 Abaqus/CAE 打开**保存视图**对话框。
2. 从**保存视图**对话框中，选择所需的**缩放和位置**行为：
   - 选择**自动拟合**仅保存视图的方向。当你应用以此选项保存的视图时，会应用保存的方向，但会调整缩放因子和位置以使视图填充视口。
   - 选择**保存当前**保存视图的方向、缩放因子和位置。当你应用以此选项保存的视图时，会全部应用保存的方向、缩放因子和位置。
3. 在**保存视图**对话框中的**视图名称**列表中，单击你将用于回忆此视图的工具名称。如果你覆盖了六个自定义视图之一——前、后、顶、底、左、右——其他五个视图仍保留其原始定义；也就是说，它们不会旋转到与你保存视图正交的位置。
4. 从**保存视图**对话框中，单击**确定**。Abaqus/CAE 保存你选择视图的定义。视图仅在当前会话期间保存；下次你运行 Abaqus/CAE 时，保存的视图将不可用。
