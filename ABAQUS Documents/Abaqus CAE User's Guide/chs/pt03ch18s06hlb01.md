# 18.6.1 创建优化任务






您可以创建拓扑优化任务。优化任务包含您的优化的定义，如设计响应、目标、约束和几何限制。要运行优化，您需要在 Job 模块中创建优化过程。优化过程引用优化任务。

**创建拓扑优化任务：**

1. 从主菜单栏，选择****任务**![](../graphics/images/arrow.gif)**创建****。此时出现**创建优化任务**对话框。**提示：**您可以通过另外两种方式启动**创建**过程：- 在**优化任务管理器**中单击**创建**。（您可以通过从主菜单栏选择****任务**![](../graphics/images/arrow.gif)**管理器****来显示**优化任务管理器**。）- 在 Optimization 模块工具箱中单击 ![](../graphics/icoR_optTask.png) 工具。
2. 在出现的**创建优化任务**对话框中，输入任务的名称。
3. 选择优化**类型**（**拓扑优化**、**形状优化**或**尺寸优化**），然后单击**继续**。有关更多信息，请参阅 [Abaqus Analysis User's Guide 第 13.1.1 节「结构优化：概述」](../usb/usb-link.md#usb-anl-aoptover)。
4. 从视口中选择将进行优化的区域，或单击**完成**以优化整个模型。默认情况下，Abaqus/CAE 允许您选择整个模型。要选择面或单元，请使用**选择**工具栏将可选择的对象类型更改为**面**或**单元**。有关更多信息，请参阅[「根据对象类型过滤您的选择，第 6.3.2 节」](pt01ch06s03s01.md)。此时出现**编辑优化任务**对话框。如果您希望从现有集合列表中进行选择，请执行以下操作：1. 单击提示区域右侧的**集合**。Abaqus/CAE 显示**区域选择**对话框，其中包含可用集合的列表。2. 选择感兴趣的集合，然后单击**继续**。**注意：**默认选择方法基于您最近使用的选择方法。要恢复另一种方法，请单击提示区域右侧的按钮——**在视口中选择**或**集合**。
5. 完成选择优化区域后，在提示区域中单击**完成**。有关选择对象的更多信息，请参阅[第 6 章，「在视口内选择对象」](pt01ch06.md)。
6. 配置优化任务，如[「配置拓扑优化任务，第 18.6.2 节」](pt03ch18s06s01.md)、[「配置形状优化任务，第 18.6.3 节」](pt03ch18s06s02.md) 和[「配置尺寸优化任务，第 18.6.4 节」](pt03ch18s06s03.md) 中所述。
