# 29.102 UserDefinedField 对象

UserDefinedField 对象在材料点重新定义场变量。

**访问**

```
import material
mdb.models[*name*].materials[*name*].userDefinedField
import odbMaterial
session.odbs[*name*].materials[*name*].userDefinedField
```

### 29.102.1 UserDefinedField()

此方法定义 UserDefinedField 对象。

**路径**

```
mdb.models[*name*].materials[*name*].UserDefinedField
session.odbs[*name*].materials[*name*].UserDefinedField
```

**返回值**

UserDefinedField 对象。

**异常**

无。

### 29.102.2 成员

UserDefinedField 对象没有成员。
