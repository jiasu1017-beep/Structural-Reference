# 70.1 理解分区的作用







在建模过程中，您可能发现需要选择模型中不存在的特定区域。这些区域可用于定义材料边界、指示载荷和约束的位置，并帮助细化网格。您使用分区工具集将模型分区为区域。

分区可以在边、面和胞上创建。沿边的分区会创建新顶点，而穿过面和胞的分区分别创建新边和新面。在所有情况下，分区都用于细分被分区的几何。[图 70-1](pt06ch70s01.md#par-simple-example) 说明了分区边、面和胞。

**图 70-1** 分区边、面和胞。

![](../graphics/par-simple-example-nls.png)

您通过定义引用现有几何的分区来分区边、面和胞。您可以在 Part（部件）模块或 Property（属性）模块中分区部件，也可以在对装配操作的模块中分区装配。例如，您可以在 Mesh（网格）模块中分区装配的面，并且您可以在 resulting 内部边上播种以细化网格。分区是一个特征；而且，与所有特征一样，它可以编辑、删除、抑制、恢复和查询。类似地，当装配或部件被再生时，分区会被再生。
