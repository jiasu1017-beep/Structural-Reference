# 10.6 成功导入IGES文件的逻辑方法





如果从IGES文件导入零件到Abaqus/CAE时遇到问题，以下列表描述了您应遵循的步骤。相同的方法可用于导入其他文件类型，但导入选项会根据文件类型而有所不同。

**简化零件**

在从CAD系统导出零件之前，您应该知道要使用Abaqus分析什么类型的零件以及期望从分析中获得什么。例如，您可能希望删除会影响网格并主导分析执行时间的过度细节。更多信息，请参阅["了解您想要的最终产品，"第10.1.5节"](pt02ch10s01s05.md)。

**修改Abaqus/CAE中的导入选项**

如果您从IGES文件导入零件，可以选择以下选项：
- **修剪曲线偏好（Trim Curve Preference）**
- **比例（Scale）**
- **MSBO**

您还可以指定从IGES文件导入的级别。有关设置这些选项效果的更多信息，请参阅["Abaqus/CAE中的IGES选项是什么？，"第10.4.1节"](pt02ch10s04s01.md)。

**尝试修复零件**

如果Abaqus/CAE需要精确零件才能继续（例如，如果您需要分割零件），您可以使用"零件"模块中的几何编辑工具集来编辑零件。更多信息，请参阅["编辑技术概述，"第69.2节"](pt06ch69s02.md)。

**诊断和定位问题**

如果您不确定为什么零件仍然无法被Abaqus/CAE使用，可以尝试以下操作：
- 在**IGES选项（IGES Options）**对话框中查看**IGES实体过滤器（IGES Entity Filter）**列表以获取不支持的实体。
- 面上灰色或浅色的未连接线条是轮廓线，表示Abaqus/CAE在尝试使曲面更精确时引入的曲面波动。您应该手动删除该面或不使用**转换为精确表示（Convert to precise representation）**自动修复选项重新导入零件。
- 尝试对零件进行网格划分。Abaqus/CAE无法网格划分的面表明几何体较差。

**尝试重新导入零件**

如果修复工具无法产生有效零件，描述零件的文件可能无效或包含非法语句。您应该尝试将零件重新导入创建它的CAD系统。如果零件无法导入，您应该检查导出设置并重新生成文件。文件中与零件定义无关的实体会导致导入失败；例如，零件周围的边界或标题块。理想情况下，在知道零件可以导入回创建它的CAD系统之前，您不应该尝试将零件导入Abaqus/CAE。
