# 70.7.1 使用切割平面方法分区胞







您可以通过用平面切割所选胞来分区胞。您首先选择要分区的胞，然后选择三种方法之一来定义分区胞的平面。三种方法是：
- 选择一个点和任意直边。切割平面将穿过该点并垂直于该边。
- 选择三个点。切割平面将穿过所有三个点。
- 选择一条直边或曲线边以及边上的一个点。切割平面将穿过该点并在所选点处垂直于该边。

以下过程说明了三种方法。

**使用平面方法分区胞：**

1. 从主菜单栏中选择 ****Tools**![](../graphics/images/arrow.gif)**Partition****（工具→分区）。出现 **Create Partition**（创建分区）对话框。Abaqus/CAE 在提示区域显示提示以指导您完成过程。**提示：**您也可以使用位于模块工具箱分区胞工具处的 ![](../graphics/ico_ptnCellCuttingPlane.png) 工具来使用平面方法分区胞。 有关工具箱中分区工具的图表，请参阅["使用分区工具集，" 第 70.2 节"](pt06ch70s02.md)。
2. 从对话框顶部的 **Type**（类型）单选按钮中选择 **Cell**（胞）。**Method**（方法）列表显示可用于分区胞的方法。
3. 从方法列表中选择 **Define cutting plane**（定义切割平面）。
4. 如果部件或装配包含多个胞，选择要分区的胞。您可以使用拖动选择、**[Shift]****+点击** 和 **[Ctrl]****+点击** 的组合来选择多个要分区的胞。**提示：**如果您无法选择所需的胞，可以使用 **Selection**（选择）工具栏更改选择行为。 有关更多信息，请参阅["使用选择选项，" 第 6.3 节"](pt01ch06s03.md)。Abaqus/CAE 高亮显示所选胞。
5. 从提示区域点击以下方法之一来定义切割平面： - 使用 **Point & normal**（点和平面）方法沿穿过所选点并垂直于所选直边或基准轴的平面对所选胞进行分区（如下图所示）：![](../graphics/par-cell-plane1-nls.png) - 使用 **3 Points**（3 点）方法沿穿过三个所选点的平面对所选胞进行分区（如下图所示）：![](../graphics/par-cell-plane2-nls.png) 这三个点可以在部件或装配上的任意位置，但它们必须不同且不能共线。 - 使用 **Normal to edge**（垂直于边）方法沿垂直于所选边并穿过边上所选点的平面对所选胞进行分区。边可以是直的或曲的，并且不必属于被分区的胞，尽管您不能选择基准轴。当您想要沿曲线边法向分区胞时，此方法很有用（如下图所示）：![](../graphics/par-cell-plane3-nls.png)
6. 按照提示区域中的提示选择点和边。点可以是顶点、中点、弧中心或基准点。
7. 在提示区域中点击 **Create Partition**（创建分区）。Abaqus/CAE 创建分区。
