# 78.2.1 创建或编辑显示组





要创建显示组，首先选择您感兴趣的特定项目。然后，您可以对选择和当前视口的内容执行布尔运算。创建显示组时，您随时可以选择保存当前视口的内容或对话框中选择的项目（如果适用）。您必须保存显示组才能执行以下任一操作：
- 稍后在会话中绘制显示组。
- 将显示组应用于其他模型或其他视口。
- 编辑、复制或重命名显示组。
- 在可视化模块的同一视口中绘制多个显示组。

保存的显示组仅在当前会话期间可用，并且仅在您处于与创建显示组的模块相同类型的模块中时可用。

**注意：**保存的显示组不会更新以反映对模型所做的更改，例如添加、删除或抑制特征或对零件进行分区；此类更改可能会使基于原始模型创建的显示组失效。

您可以编辑先前保存的显示组中的模型组件组合。编辑显示组与创建显示组类似：在两种情况下，您都可以选择模型组件，然后对选择和当前视口的内容执行布尔运算。当您选择要编辑的显示组时，当前视口的内容会更新以显示所选显示组。您使用当前视口中的显示来确定要编辑的组中的模型组件；您无法列出模型组件。编辑完成后，使用显示组的所有视口都会更新。

**创建或编辑显示组的步骤：**

1. 找到用于创建或编辑显示组的选项。
   - 要创建显示组，请从主菜单栏中选择****工具**![](../graphics/images/arrow.gif)**显示组**![](../graphics/images/arrow.gif)**创建****。**提示：**您也可以通过点击**显示组**工具栏中的![](../graphics/ico_displayGroupCreate.png)工具来创建显示组。**创建显示组**对话框出现。
   - 要编辑先前保存的显示组，请从主菜单栏中选择****工具**![](../graphics/images/arrow.gif)**显示组**![](../graphics/images/arrow.gif)**编辑****。从出现的菜单中选择要编辑的显示组。**编辑显示组**对话框出现。**提示：**您也可以使用显示组管理器来创建或编辑显示组。从主菜单栏中选择****工具**![](../graphics/images/arrow.gif)**显示组**![](../graphics/images/arrow.gif)**管理器****可显示当前模块的管理器。要创建显示组，请点击**创建**。要编辑显示组，请从列表中选择它，然后双击它或点击**编辑**。
2. 从对话框左上角的**项目**列表中，选择用于显示组的模型组件类型（根据当前模块可用）：**单元**、**面**、**边**、**基准**、**装配线**、**参考点**、**单元**、**节点**、**集合**、**曲面**、**零件实例**、**显示组**、**内部集合**、**内部曲面**、**坐标系**、**绑定**、**壳到实体耦合**、**分布耦合**、**运动耦合**或**刚体**。您可以选择**全部**来指定当前零件、装配或输出数据库中的所有对象（当您创建新显示组时）。Abaqus/CAE会刷新**方法**列表和对话框右侧。如果选择**全部**，这些字段为空，不需要进一步指定项目。如果选择的項目只有一种选择方法，Abaqus/CAE会立即进入所需的选择模式。例如，如果选择**单元**，Abaqus/CAE会立即进入视口选择模式。
3. 从**方法**列表中选择一种选择方法；和/或通过从视口中选择、从对话框右侧出现的列表中选择项目，或在对话框右侧输入数据来选择显示组的特定项目。有关更多信息，请参见["用于创建或编辑显示组的选择方法，"第78.2.2节](pt07ch78s02hlb02.md)。**提示：**如果您的模型包含许多特定类型的项目，您可以使用筛选器来减少**创建显示组**对话框右侧显示的项目名称数量。点击**筛选器**字段旁边的**提示**按钮可查看有效筛选语法的示例。某些项目可以在视口中高亮显示以验证您的选择。切换**在视口中高亮显示项目**（如果可用）。切换**显示链接视口之间的公共列表**可在**方法**列表中显示所有链接视口中共有的项目。对于**零件实例**，包含模型数据库或输出数据库的视口会被考虑；对于其他模型组件，仅考虑包含输出数据库的视口。
4. 点击对话框底部图标中所需的布尔运算（有关更多信息，请参见["理解显示组布尔运算，"第78.1.2节](pt07ch78s01s02.md)）。Abaqus/CAE会对您在对话框中选择的模型组件和当前视口的内容执行所选的布尔运算。
5. 根据需要重复从步骤2开始的步骤以产生所需的显示组。
6. 如果您要创建新的显示组，可以随时选择保存它。
   1. 要将当前视口的内容保存为显示组，请点击**另存为**；然后在出现的对话框中输入显示组的名称，并点击**确定**。Abaqus/CAE会将当前视口的内容（无论对话框中选择了什么）保存为显示组。
   2. 如果在**创建显示组**对话框中选择了模型组件，您可以将所选组件保存为显示组。点击**将选择另存为**；然后在出现的对话框中输入显示组的名称，并点击**确定**。Abaqus/CAE仅将对话框中选择的组件（无论当前视口中显示了什么）保存为显示组。
7. 点击**解散**关闭**创建显示组**对话框，或点击**确定**关闭**编辑显示组**对话框。
