# 20.5.2 构造几何







您在 Sketcher 中创建构造几何是为了帮助您在草图中定位和对齐几何；您可能需要定位的对象包括孔、圆弧、槽或齿轮齿。Sketcher 允许您向草图添加构造线和圆；此外，使用孤立点工具![](../graphics/ico_sketchPoint.png)创建的点被视为构造几何。构造线、圆和点不会出现在您正在创建或修改的特征中。

当您添加构造几何并在草图中移动光标时，Abaqus/CAE 会在以下位置显示预选符号：
- 新构造几何与草绘线或曲线的交点。
- 新构造几何与现有构造几何的交点。
- 新构造几何与参考几何的交点。仅当参考几何是在被编辑特征之前创建时，Abaqus/CAE 才会显示预选符号。（有关参考几何行为的更多信息，请参见["参考几何，"第 20.5.1 节](pt03ch20s05s01.md)。）

构造几何生成的预选符号允许您精确对齐对象；例如，沿斜线或围绕圆对齐。例如，您可以创建一条斜构造线和几条垂直构造线来帮助对齐一组圆，如图 20-6](pt03ch20s05s02.md#ske-congeom-example)所示。

**图 20-6** 使用构造线对齐草图几何。

![](../graphics/ske-congeom-example-nls.png)

在草绘旋转实体和曲面时，构造线也定义旋转轴。有关构造线与旋转轴之间关系的更多信息，请参见["为轴对称零件和旋转特征定义旋转轴，"第 11.13.5 节](pt03ch11s13s05.md)。

构造几何使用虚线显示以区别于草绘几何。构造几何仅在您处理草图时可见；一旦退出 Sketcher，构造几何就会消失。Abaqus/CAE 会将构造几何与原始草图一起保存；如果调用 Sketcher 修改包含构造几何的草图，构造几何会与草图一起重新出现。

要创建构造几何，请从 Sketcher 工具箱中选择一个构造几何工具，或从主菜单栏中选择****添加**![](../graphics/images/arrow.gif)**构造****。

您还可以将草图组件转换为构造几何，并且可以反转已转换为构造几何的项目此过程。要将组件转换为构造几何，请选择该组件，然后从主菜单栏中选择****编辑**![](../graphics/images/arrow.gif)**设置为构造****；要反转此过程，请选择构造几何组件，然后从主菜单栏中选择****编辑**![](../graphics/images/arrow.gif)**取消设置构造****。
