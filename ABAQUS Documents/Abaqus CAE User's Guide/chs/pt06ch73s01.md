# 73.1 理解集和曲面工具集的作用







集和曲面工具集是一组工具，允许您创建和管理集和曲面。您可以使用这些工具执行以下任务：

**创建**

通过选择一组实体（如面和边）来创建集或曲面。

**编辑**

通过选择要添加或删除的实体来修改集或曲面。

**重命名**

用新名称替换现有集或曲面的当前名称。

**删除**

从模型中删除集或曲面。
