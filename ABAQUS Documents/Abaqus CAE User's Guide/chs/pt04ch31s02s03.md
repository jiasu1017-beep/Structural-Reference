# 31.2.3 定义裂缝尖端或裂缝线





定义裂缝前缘后，配置轮廓积分过程的下一步是通过从装配中选择实体来定义裂缝尖端或裂缝线。您的装配的建模空间决定了您是需要为轮廓积分分析定义裂缝尖端还是裂缝线。

**二维**

如果您的装配是二维的，您必须通过选择顶点或节点来定义裂缝尖端。裂缝尖端是裂缝前缘上定义裂缝扩展方向<img src="../graphics/usi_eqn00702.gif">的点。在某些情况下，所需的顶点或节点不存在，您必须通过分割裂缝前缘来创建它。

如果您选择了顶点或节点来定义裂缝前缘，则相同的顶点或节点定义裂缝尖端。

**三维**

如果您的部件是三维的，您必须通过选择形成连续线的边缘或单元边缘来定义裂缝线。裂缝线是沿裂缝前缘的一系列连接边缘，您可以在其中定义裂缝扩展方向<img src="../graphics/usi_eqn00702.gif">。在某些情况下，所需的边缘不存在，您必须通过分割裂缝前缘来创建它们。

如果您选择了边缘或单元边缘来定义裂缝前缘，则相同的边缘定义裂缝线。

所选边缘（来自Abaqus/CAE原生部件或孤立网格）必须连接，必须从裂缝前缘的一侧连接到另一侧，并且必须包含在裂缝前缘中。
