# 78.2.5 在可视化模块中自定义显示组





可视化模块中的**ODB显示组管理器**允许您锁定和解锁绘制的显示组以有选择地自定义绘图选项。此外，您可以同步所有绘制的显示组以使用相同的绘图选项。例如，您可以显示一组特定的单元，采用半透明着色渲染样式，同时以线框渲染样式显示模型的其余部分。[图78-1](pt07ch78s02hlb05.md#uss-dgp-custom-multi-plot)显示了在同一视口中使用不同绘图状态和绘图选项绘制的两个显示组的另一个示例。

**图78-1** 一个视口中的多个自定义显示组。

![](../graphics/uss-dgp-custom-multi-plot.png)

当您在可视化模块中绘制显示组时，显示组（连同其当前绘图状态）出现在**ODB显示组管理器**底部显示组实例列表中。您指定的与绘图状态相关和无关的选项适用于所有解锁的显示组实例。您可以锁定不想被绘图选项影响的显示组实例。

更改绘图状态不会影响视口中锁定的显示组实例。但是，视图操作和颜色编码更改会影响所有显示组实例（锁定和未锁定）。此外，可以编辑锁定显示组的内容。

显示组按它们在显示组实例列表中出现的顺序绘制在视口中；列表中的第一个实例是视口中最顶层的实例。因此，如果两个显示组实例包含相同的组件且这些组件在视口中重叠，则列表中第一个实例的绘图选项优先。您可以重新排列列表中实例的顺序来控制将显示哪些绘图选项。

**在可视化模块中自定义显示组的绘图选项的步骤：**

1. 按照["绘制显示组，"第78.2.4节](pt07ch78s02hlb04.md)中描述的步骤在同一视口中绘制多个显示组。
2. 在**ODB显示组管理器**中的显示组实例列表中，点击**锁定**列中显示组名称旁边的列，以锁定您不想自定义的实例。锁定的显示组在**锁定**列中由复选标记指示。
3. 根据需要自定义未锁定显示组实例的绘图选项。（有关更多信息，请参见[第55章，"自定义绘图显示](pt05ch55.md)。）
4. 完成自定义显示组实例的绘图选项后，点击显示组名称旁边的**锁定**列以防止您的设置被修改。
5. 根据需要重复步骤3和4，直到在视口中获得所需的显示。
6. 如果您希望所有显示组实例使用相同的绘图状态和绘图选项，请从管理器底部的列表中选择反映所需绘图状态和绘图选项的实例，然后点击**同步选项**。所有其他未锁定的显示组实例将同步到所选显示组的绘图状态和绘图选项。
7. 要重新排列显示组在视口中绘制的顺序，请从**ODB显示组管理器**底部列表中选择实例，然后点击**上移**将实例在列表中上移，或**下移**将实例在列表中下移。当前视口中的显示会更新以反映您的更改。
