# 14.13.1 定义集成输出截面





从主菜单栏选择****Output**![](../graphics/images/arrow.gif)**Integrated Output Sections****来创建集成输出截面。更多信息，请参见["集成输出截面定义，"Abaqus分析用户指南第2.5.1节](../usb/usb-link.md#usb-int-aintegratedoutputsect)。

**要创建集成输出截面：**

1. 从主菜单栏，选择****Output**![](../graphics/images/arrow.gif)**Integrated Output Sections**![](../graphics/images/arrow.gif)**Create****。出现**创建集成输出截面**对话框。
2. 在对话框中，为输出截面输入名称或接受默认名称。
3. 使用以下方法之一选择要定义集成输出截面的曲面：
   - 在视口中选择一个曲面。完成选择后，点击鼠标按钮2。如果模型包含网格和几何的组合，请从提示区域点击以下之一：
     - 点击**几何**在几何上定义截面。
     - 点击**网格**在原生或孤立网格面上定义截面。
   - 要从现有曲面列表中选择，请执行以下操作：
     1. 点击提示区域右侧的**曲面**。Abaqus/CAE显示包含可用曲面列表的**区域选择**对话框。
     2. 选择感兴趣的曲面，然后点击**继续**。
   **注意：**默认选择方法基于你最近使用的选择方法。要切换回另一种方法，请点击提示区域右侧的**在视口中选择**或**曲面**。出现集成输出截面编辑器。你正在定义截面的曲面在视口中高亮显示。
4. 默认情况下，集成输出截面锚定在全局原点。要定义截面将锚定在其上的参考点，选择**在参考点处锚定**，然后点击**编辑**![](../graphics/ico_selectBlue.png)。
5. 如果参考点在物理上未连接到模型，你可以调整其初始位置并指定它应该跟随与集成输出截面关联的曲面的平均运动。默认情况下，参考点不跟踪曲面的平均运动。
   - 要将参考点重新定位到初始配置中曲面的中心，启用**将点移动到曲面中心**。
   - 要指定参考点应该既平移又跟随曲面的平均运动旋转，选择**点运动**的**平均平移和旋转**。
   - 要指定参考点应该仅跟随曲面的平均平移移动而不旋转，选择**点运动**的**平均平移**。
6. 默认情况下，集成输出截面的坐标系初始化为全局坐标系。要选择局部坐标系代替，点击**CSYS**字段旁边的![](../graphics/ico_selectBlue.png)。
7. 如果需要，启用**将方向投影到曲面**以将坐标系（全局或局部）投影到截面曲面上。
8. 点击**确定**关闭集成输出截面编辑器。
