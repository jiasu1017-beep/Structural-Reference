# 17.18.8 应用网格堆栈方向





在网格划分包含连续壳单元、粘聚单元、圆柱单元或垫片单元的零件时，网格堆栈方向很重要，因为这些单元具有独特的方向性行为。 如果使用扫掠网格划分，默认堆栈方向遵循扫掠路径的方向，您可以通过选择不同的扫掠路径来更改堆栈方向。 但是，您必须为零件中的每个单元单独分配扫掠方向——这是一个耗时的过程——而且可用的扫掠方向可能不包括您所需的堆栈方向。

堆栈方向工具允许您为实体零件或零件实例中的所有单元或选定单元分配堆栈方向，只要这些单元未分配四面体单元即可。 您可以使用该工具在单个操作中快速为一组单元应用堆栈方向。 您根据所选面分配堆栈方向；堆栈的取向使得所选面成为堆栈的顶部。 Abaqus/CAE会不顾单元使用的网格划分技术而应用该方向；因此，扫掠方向（如果存在）无关紧要。

在[图 17-139](pt03ch17s18hlb01.md#mgn-stackdir) 中，使用堆栈方向工具通过零件厚度分配了方向（如图中红色箭头所示）； 单元的顶面用棕色着色。 扫掠实体零件的复杂轮廓不允许使用扫掠路径获得所需的堆栈方向——两个可用的扫掠方向由图中的黑色箭头指示。

**图 17-139** 堆栈方向不必遵循扫掠方向。

![](../graphics/mgn-stackdir.png)

新堆栈方向的应用可能因所选单元而异。 [图 17-140](pt03ch17s18hlb01.md#mgn-stackorient) 展示了一个示例，其中堆栈方向随着更多单元被网格划分而改变。 左上方面用于参考方向，因此中间图像中两个断开的单元的网格方向与此方向一致。 但是，当对中心单元进行网格划分时，右下单元中单元的方向会更新为与其他两个单元对齐，如图右侧所示。

**图 17-140** 如果您向网格添加单元，堆栈方向可能会改变。

![](../graphics/mgn-stackorient.png)

**应用网格堆栈方向：**

1. 从主菜单栏，选择 ****Mesh**![](../graphics/images/arrow.gif)**Orientation**![](../graphics/images/arrow.gif)**Stack****。 Abaqus/CAE在提示区域显示提示以指导您完成该过程。 **提示：**您也可以使用Mesh模块工具箱中的 ![](../graphics/ico_stackDir.png) 工具设置网格堆栈方向（有关更多信息，请参阅["使用Mesh模块工具箱，" 第17.15节](pt03ch17s15.md)），以及**Mesh Controls**对话框中的**Assign stack direction**按钮。
2. 如果您的零件包含多个单元，请选择要为其分配堆栈方向的单元。
3. 选择一个顶面来定义参考方向。 Abaqus/CAE会提示您在接受选择之前确认您的选择。
4. 单击**Yes**接受所选面及相应的方向，或单击**No**返回步骤3。 当您接受参考面时，Abaqus/CAE会将堆栈方向应用于所选单元。 如果任何所选单元包含圆柱单元，Abaqus/CAE会显示一条警告消息，指示必须删除圆柱单元才能应用新的堆栈方向。 单击**Yes**删除圆柱网格，或单击**No**取消堆栈方向过程。
