# 55.6 显示多个绘图状态







除了为单个绘图状态（如云图）叠加未变形和变形形状外，您还可以选择在视口中显示多个绘图状态。通过切换工具箱中的"允许多个绘图状态"![](../graphics/ico_plotMultipleStates.png)，您可以为当前输出数据库的所选步骤和帧选择未变形和变形绘图形状与云图、符号和材料取向的任意组合。Abaqus/CAE使用"通用绘图选项"显示变形图，使用"叠加绘图选项"显示未变形图，结合每个绘图状态的其他绘图状态无关和相关选项。

如果您关闭"允许多个绘图状态"，Abaqus/CAE会使用以下顺序以及最后显示的绘图状态来选择要显示的单个绘图状态：
- 未变形
- 变形
- 云图
- 符号
- 材料取向

例如，如果您显示云图和材料取向，当您关闭"允许多个绘图状态"时，Abaqus/CAE会保留云图。这与绘图状态在"绘图"菜单中的列出顺序以及在工具箱中的显示顺序相同。

要显示更复杂的绘图组合，您必须使用"叠加绘图"。叠加绘图允许您显示动画或X-Y图以及上面列出的静止绘图状态。您还可以使用叠加绘图来绘制来自多个步骤、帧或输出数据库的结果。有关更多信息，请参见[第79章，"覆盖多个绘图"](pt07ch79.md)。
