# 9.1 网格无关紧固件的新附着方法







**产品：**Abaqus/Standard  Abaqus/Explicit  

**优点：**您现在可以保留网格无关紧固件连接的位置和方向。

**说明：**以前，所有网格无关紧固件的附着方法都允许紧固点移动到位于第一个附着表面法线上的位置。新的附着方法（必须使用连接单元进行附着）保留连接的位置和方向。
