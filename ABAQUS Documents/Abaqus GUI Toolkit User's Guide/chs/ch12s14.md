# 12.14 小结





- 接触分析需要谨慎、逻辑的方法。如有必要，将分析分为几个步骤，并缓慢施加荷载，确保接触条件建立良好。
- 通常，在 Abaqus/Standard 中最好为分析的每个部分使用单独的步骤，即使只是将边界条件更改为荷载。您几乎肯定会最终得到比预期更多的步骤，但模型应该更容易收敛。如果您尝试在一个步骤中施加所有荷载，接触分析将更加难以完成。
- 在 Abaqus/Standard 中，在将工作荷载施加到结构之前，在所有组件之间建立稳定的接触条件。如有必要，应用可在后续阶段移除的临时边界条件。只要约束不产生永久变形，最终结果将不受影响。
- 不要在接触表面的节点上施加约束接触方向的边界条件。如果存在摩擦，不要约束这些节点的任何自由度：可能导致零枢轴消息。
- 始终尝试在 Abaqus/Standard 接触模拟中使用一阶单元。
- Abaqus/Standard 和 Abaqus/Explicit 都提供两种不同的建模接触的算法：通用接触（使用 [*CONTACT](../key/key-link.md#usb-kws-hcontact) 选项定义）和接触对（使用 [*CONTACT PAIR](../key/key-link.md#usb-kws-hcontactpair) 选项定义）。
- 通用接触相互作用允许您定义模型中许多或所有区域之间的接触；接触对相互作用描述两个表面之间或单个表面与其自身之间的接触。
- 通用接触可以在模型的模型部分或历史部分定义；接触对在输入文件的历史部分定义。
- 与通用接触算法一起使用的表面可以跨越多个未连接的体。多个表面面可以共享一条公共边。相反，接触对算法使用的所有表面必须是连续且单连通的。
- 使用 [*SURFACE](../key/key-link.md#usb-kws-msurface) 选项定义表面。可以通过在 [*SURFACE](../key/key-link.md#usb-kws-msurface) 选项上使用 TYPE=NODE 参数将单个节点包含在接触对中。解析刚性表面通过在 [*RIGID BODY](../key/key-link.md#usb-kws-mrigidbody) 选项上使用 ANALYTICAL SURFACE 参数分配给刚体。[*SURFACE](../key/key-link.md#usb-kws-msurface) 选项上的参数 TYPE=SEGMENTS、CYLINDRICAL 或 REVOLUTION 指定解析刚性表面的类型。
- 在 Abaqus/Explicit 中，壳单元、膜单元或刚性单元上的单面表面必须定义得使法线方向在表面遍历时不会"翻转"。
- Abaqus/Explicit 不会平滑刚性表面；它们像底层单元一样是面状的。粗网格的离散刚性表面会产生嘈杂的解与接触对算法。通用接触算法确实包括一些数值特征舍入。
- 绑定约束是 Abaqus 中网格细化的有用方法。
- Abaqus/Explicit 在第一个步骤之前调整节点坐标而不产生应变以移除任何初始过盈闭合。如果调整相对于单元尺寸较大，单元可能会严重扭曲。
- 在后续步骤中，Abaqus/Explicit 移除初始过盈闭合的任何节点调整会产生应变，可能导致严重的网格扭曲。
- 当您对可能包含高频振荡的结果感兴趣时（如撞击问题中的加速度），请以相对较高的输出率请求 Abaqus/Explicit 历史输出（如果输出率小于每增量），并应用抗混叠滤波器；然后，如果需要更强的过滤，请使用后处理过滤器。
