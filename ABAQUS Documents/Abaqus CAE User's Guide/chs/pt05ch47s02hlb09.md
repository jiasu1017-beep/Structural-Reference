# 47.2.9 将X–Y数据对象加载到当前会话









### 47.2.9 将X--Y数据对象加载到当前会话

当您创建*X–Y*数据对象时，必须保存它才能在会话中操作数据。然后您可以将*X–Y*数据对象复制到输出数据库文件，以便您可以从以后的会话访问它。**XY数据管理器**列出了可以从两个来源获取的*X–Y*数据对象：当前会话或当前输出数据库文件。如果您在以前的会话中创建了*X–Y*数据对象并将其复制到了输出数据库文件，则必须将*X–Y*数据对象加载到当前会话才能操作它。

**要将X–Y数据对象加载到当前会话：**

1. 从主菜单栏选择****工具**![](../graphics/images/arrow.gif)**XY数据**![](../graphics/images/arrow.gif)**管理器****。出现**XY数据管理器**。**提示：**您也可以从工具箱中的![](../graphics/ico_manager.png)工具访问**XY数据管理器**。
2. 切换**当前ODB：*odb_name***作为**数据源**以查看与当前打开的输出数据库文件关联的*X--Y*数据对象列表。
3. 选择要加载的*X--Y*数据对象。
4. 点击**加载到会话**。
5. 切换**当前会话**作为**数据源**。加载的*X--Y*数据对象将被列为可用；现在您可以编辑、复制、重命名或删除此*X--Y*数据对象。
