# 42.3.1 选择特定的结果步骤和帧





您可以选择Abaqus从输出数据库获取模型数据和结果的特定步骤和帧，也可以选择Abaqus从当前模型数据库中的模型获取模型数据的特定步骤。在输出数据库中，可用步骤和帧由已保存分析结果的步骤和帧组成；在模型数据库中，所有步骤都可用于数据显示。要查看在Abaqus/CAE会话期间创建的场输出变量（如果有），必须选择****Result**![](../graphics/images/arrow.gif)**Step/Frame**![](../graphics/images/arrow.gif)**Session Step****。要查看分析中定义的参数化形状变化（如果有），必须选择****Result**![](../graphics/images/arrow.gif)**Step/Frame**![](../graphics/images/arrow.gif)**Step 0****。

可以打开仍在进行分析的输出数据库。随着分析趋于完成，每当你关闭然后重新打开**Step/Frame**对话框时，已完成步骤和帧的列表都会更新。

有关逐步切换结果帧的信息，请参见["逐步切换帧，" 第42.3.2节](pt05ch42s01hlb02.md)。

**选择特定的结果步骤和帧：**

1. 找到**Step/Frame**选项。从主菜单栏，选择****Result**![](../graphics/images/arrow.gif)**Step/Frame****。**Step/Frame**对话框出现。对话框的**Step**部分列出了可用步骤，对话框的**Frame**部分列出了当前[输出数据库](pt02ch09s06hlb02.md)的可用帧（如果已选择输出数据库）。**注意：**您也可以通过点击**Field Output**对话框中的![](../graphics/ico_stepSmall.png)来访问这些选项。
2. 从**Step**列表中，点击要显示的步骤。选中的步骤高亮显示，Abaqus/CAE刷新**Frame**列表以仅显示所选步骤可用的帧。
3. 从**Frame**列表中，点击要显示的帧。选择**Frame 0**显示当前步骤的初始状态；例如，用于显示初始应力的云图。选中的帧高亮显示。
4. 当前视口中的模型图会更改以显示你在所选步骤和帧下的模型。如果处于活动状态，状态块中的文本会更改以标识所选的步骤和帧。有关状态块的更多信息，请参见["自定义状态块，" 第56.3节](pt05ch56s03.md)。Abaqus刷新[**Field Output**](pt05ch42s03hlb01.md)对话框以列出所选帧可用的变量。（您可以通过点击**Step/Frame**对话框底部的**Field Output**按钮来访问**Field Output**对话框。）Abaqus还会刷新所有标识当前步骤和帧的对话框。你的更改将保存到会话结束。
