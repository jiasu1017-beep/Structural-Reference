# 10.5.5 从ANSYS输入文件导入模型





您可以通过从主菜单栏选择****文件（File）**![](../graphics/images/arrow.gif)**导入（Import）**![](../graphics/images/arrow.gif)**模型（Model）****使用ANSYS输入文件将模型导入Abaqus/CAE。Abaqus/CAE将ANSYS输入文件中的许多常见实体导入您创建的模型中；有关支持的实体更多信息，请参阅["将ANSYS输入文件转换为部分Abaqus输入文件，"Abaqus Analysis User's Guide第3.2.30节"](../usb/usb-link.md#usb-int-dansysabaproc)。
