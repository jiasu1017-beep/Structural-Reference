# 28.5 体积分数工具






体积分数工具通过在欧拉零件实例与和欧拉实例相交的第二个零件实例（参考零件实例）之间执行布尔比较来创建标量离散场。该比较确定两个零件实例重叠的位置，然后根据欧拉实例中单元被参考实例占据的百分比为欧拉实例中的每个单元分配体积分数。体积分数指定为零到一之间的小数。

体积分数工具创建的离散场可用于向欧拉零件实例分配材料实例（请参见["向欧拉零件实例分配材料，" 第 28.4 节"](pt04ch28s04.md)）。所分配欧拉材料实例的拓扑结构对应于参考零件实例在欧拉零件实例内的形状。

[图 28-7](pt04ch28s05.md#usi-adv-volfrac-proc-nls) 和以下过程总结了使用体积分数工具的过程：

**图 28-7** 使用体积分数工具的过程。

![](../graphics/usi-adv-volfrac-proc-nls.png)

1. 使用 Abaqus/CAE 中的任何建模工具和技术，创建一个对应于所需欧拉材料区域几何形状的参考零件。
2. 在欧拉零件实例内实例化参考零件。参考零件实例应在空间上对应于所需的欧拉材料区域。
3. 使用体积分数工具基于参考零件实例和欧拉零件实例的比较创建离散场。
4. 使用体积分数工具创建的离散场为欧拉零件实例定义材料分配预定义场。

体积分数工具的一个选项控制计算的离散场是代表参考实例内部的空间（在与参考实例重叠的单元中体积分数非零）还是代表参考实例外部的空间（在与参考实例不重叠或部分重叠的单元中体积分数非零），如图 [图 28-8](pt04ch28s05.md#usi-adv-eulerian-volfrac-exa) 所示。

**图 28-8** 表示阴影参考区域内部（左侧）和外部（右侧）体积分数的离散场。

![](../graphics/usi-adv-eulerian-volfrac-exa.png)

通常，计算参考实例外部的体积分数用于在耦合欧拉-拉格朗日分析中在拉格朗日零件实例周围创建欧拉材料分配。计算参考实例内部的体积分数通常用于在纯欧拉零件实例内建模复杂的材料分配场；在这种情况下，欧拉材料分配后参考实例被抑制。您还可以计算参考实例内部的体积分数，以在耦合欧拉-拉格朗日分析中的封闭拉格朗日壳内部创建欧拉材料分配。

要使用体积分数工具，请从主菜单栏中选择****工具**![](../graphics/images/arrow.gif)**离散场**![](../graphics/images/arrow.gif)**体积分数工具****。有关使用该工具的分步说明，请参阅["为材料体积分数创建离散场，" 第 63.4 节"](pt06ch63hla03.md)。
