# 10.7.7 从IGES格式文件导入零件





许多计算机辅助建模和绘图应用程序可以读取和写入IGES格式文件；您可以使用这些文件在这些应用程序与Abaqus/CAE之间交换几何信息。从主菜单栏选择**文件**![](../graphics/images/arrow.gif)**导入**![](../graphics/images/arrow.gif)**零件**，可以从IGES格式文件导入零件。您可以将草图和零件从Abaqus/CAE导出为IGES格式文件，但不能将装配体导出为IGES格式文件。

如果IGES格式文件包含多个零件，Abaqus/CAE会将它们作为单个零件导入。导入的IGES零件在Abaqus/CAE中形成新零件的基础特征；您不能直接修改此基础特征，但可以向其添加其他特征，例如实体拉伸或盲孔切割。

**从IGES格式文件导入零件：**

1. 从主菜单栏，选择**文件**![](../graphics/images/arrow.gif)**导入**![](../graphics/images/arrow.gif)**零件**。出现**导入零件**对话框。
2. 从**导入零件**对话框底部的**文件过滤器**菜单中，选择**IGES (*.igs*.*.iges*)**。Abaqus/CAE列出当前目录中所有具有`.igs*`或`.iges*`文件扩展名的文件。
3. 选择包含要导入的零件的IGES文件，然后单击**确定**。Abaqus/CAE显示**从IGES文件创建零件**对话框。
4. 单击**名称-修复**选项卡设置以下内容：-新零件的名称。-Abaqus/CAE将在导入过程中应用的修复选项。在大多数情况下，您应该接受默认设置；更多信息，请参见"控制导入过程"，第10.3节。
5. 单击**零件属性**选项卡设置以下内容：-正在导入的零件的建模空间。导入零件时，Abaqus/CAE会扫描文件并尝试确定建模空间。-正在导入的零件的类型。Abaqus/CAE始终假定零件类型为可变形。或者，您可以选择**离散刚体**导入离散刚体零件，或选择**欧拉**导入欧拉零件。更多信息，请参见"零件类型"，第11.4.2节。同样，在大多数情况下您应该接受默认设置；更多信息，请参见"什么是零件属性？"，第10.3.2节。
6. 单击**缩放**选项卡以在导入过程中缩放零件。-选择**不缩放**以保持文件中存储的尺寸。-选择**使用文件中的变换，包括缩放**以从IGES文件读取缩放因子、旋转矩阵和平移矩阵。-选择**将所有长度乘以**，然后输入缩放因子。Abaqus/CAE将缩放因子应用于文件中的所有坐标。因此，距原点的任何偏移都将按比例缩放。
7. 单击**IGES选项**选项卡设置以下内容：-Abaqus/CAE如何将曲面和裁剪曲线转换为零件的内部表示。-Abaqus/CAE是否识别流形实体B-rep对象（MSBO，实体类型186）。在大多数情况下，Abaqus/CAE在扫描IGES格式文件并找到实体后自动设置MSBO选项。但是，如果IGES文件非常大，则必须手动设置此选项。-Abaqus/CAE将尝试导入的IGES文件中的级别。例如，如果**级别**字段包含`0:2,4`，Abaqus/CAE将尝试导入级别0、1、2和4。如果级别包含几何以外的内容，Abaqus/CAE将在导入期间忽略该级别。您可以从列表中删除级别，但不能添加级别。如果IGES文件只包含一个级别，则此字段不可用。更多信息，请参见"Abaqus/CAE中的IGES选项是什么？"，第10.4.1节，以及"什么是IGES实体？"，第10.4.2节。您还可以使用**IGES选项**对话框查看IGES文件中的标题信息以及文件中找到的实体列表。更多信息，请参见"什么是IGES实体？"，第10.4.2节。
8. 单击**确定**导入IGES零件并关闭**从IGES文件创建零件**对话框。Abaqus/CAE扫描IGES文件并根据所选选项启动修复过程。如果希望取消导入过程，请单击提示区域中的**停止**。
9. 导入零件后，Abaqus/CAE会显示一条消息，指示零件是否存在任何有效性或精度问题。如有必要，使用查询工具集定位无效和不精确几何的区域。更多信息，请参见"在零件模块中使用查询工具集"，第11.16.4节。
