# 64.6.2 删除单元





从主菜单栏选择****网格**![](../graphics/images/arrow.gif)**编辑****可以删除单元。您可以选择指示Abaqus/CAE删除在单元被删除后将变得不与任何单元关联的任何节点的选项。如果要删除的单元和关联节点属于现有单元集或节点集，这些集合将相应更新。

**删除单元的步骤：**

1. 进入"网格"模块。
2. 从上下文栏的**对象**字段中选择**零件**并从列表中选择一个零件。此工具仅适用于处理孤兒网格单元或使用自底向上网格划分程序创建的单元。
3. 从主菜单栏选择****网格**![](../graphics/images/arrow.gif)**编辑****。Abaqus/CAE显示**编辑网格**对话框。 **提示：**您也可以使用位于"网格"模块工具箱底部的![](../graphics/ico_meshEdit.png)工具显示**编辑网格**对话框。
4. 在对话框中执行以下操作： 1. 在**类别**字段中选择**单元**。 2. 从**方法**列表中选择**删除**。
5. 如果您想删除在所选单元被删除后将变得不与任何单元关联的所有节点，请在提示区域切换**删除关联的未引用节点**。
6. 选择要删除的单元。您可以**按住[Shift]键并点击**单元将其添加到您的选择中，或**按住[Ctrl]键并点击**所选单元将其从选择中移除。 **提示：**您可以使用**选择**工具栏更改选择行为。有关更多信息，请参见[第6章，"在视口中选择对象"](pt01ch06.md)。如果您希望从现有单元集列表中进行选择，请执行以下操作： 1. 点击提示区域右侧的**集合**。Abaqus/CAE显示**区域选择**对话框，其中包含您已创建的单元集列表。 2. 选择要删除的单元集，然后点击**继续**。 **注意：**默认选择方法基于您最近使用的选择方法。要切换到另一种方法，请点击提示区域右侧的**在视口中选择**或**集合**。Abaqus/CAE显示所选单元。
7. 完成要删除的单元选择后，点击鼠标按钮2。Abaqus/CAE会删除所选单元。此外，如果您切换了**删除关联的未引用节点**，Abaqus/CAE也会删除在所选单元被删除后将变得不与任何单元关联的节点。单元和节点也会从任何现有集合中移除。 **提示：**如果在删除单元时出错，点击**编辑网格**对话框中的**撤销**可以撤消删除。Abaqus/CAE会恢复已删除的单元。
8. 根据需要重复上述步骤以删除其他单元。
9. 完成单元删除后，点击鼠标按钮2或提示区域中的取消按钮![](../graphics/afxI_cancel.png)退出过程。
