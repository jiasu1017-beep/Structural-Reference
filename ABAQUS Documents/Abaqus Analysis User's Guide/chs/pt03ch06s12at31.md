**产品：** Abaqus/Standard 和 Abaqus/Explicit

### 概述

声学、冲击和耦合声学-结构分析能力：
- 用于模拟涉及声波传播的问题；
- 可用于模拟冲击载荷下的结构响应；
- 支持声学介质与结构的耦合分析；
- 可用于分析水下结构、隔音屏障、声波传感器等；以及
- 可用于稳态和瞬态分析。

### 声学分析

声学分析用于模拟介质中的声波传播。声学单元可以存在于二维和三维模型中。

#### 声学单元

Abaqus 支持以下声学单元：

| **单元类型** | **描述** |
| --- | --- |
| AC2D3 | 二维三角形声学单元 |
| AC2D4 | 二维四边形声学单元 |
| AC3D4 | 三维四面体声学单元 |
| AC3D8 | 三维六面体声学单元 |

#### 定义声学分析

| **输入文件用法：** | 使用以下选项定义声学分析： |
| --- | --- |
| | ``` [*ACOUSTIC ANALYSIS](../key/key-link.md#usb-kws-hacousticanalysis) ``` |

### 冲击分析

冲击分析用于模拟瞬态载荷作用于结构的情况。

#### 定义冲击分析

| **输入文件用法：** | 使用以下选项定义冲击分析： |
| --- | --- |
| | ``` [*SHOCK ANALYSIS](../key/key-link.md#usb-kws-hshockanalysis) ``` |

### 耦合声学-结构分析

耦合声学-结构分析用于模拟声学介质与弹性结构的相互作用。这种分析类型常见于：
- 水下结构物；
- 隔音结构；
- 声波传感器；
- 医学超声设备。

#### 定义耦合分析

| **输入文件用法：** | 在步骤定义中包含声学介质和结构单元： |
| --- | --- |
| | ``` [*COUPLED ACOUSTIC-STRUCTURAL](../key/key-link.md#usb-kws-hcoupledacousticstructural) ``` |

### 声学边界条件

声学分析支持以下边界条件：
- 声压边界条件；
- 法向加速度边界条件；
- 吸收边界条件（用于模拟无限域）；
- 对称和反对称条件。

| **输入文件用法：** | 使用以下选项定义声学边界条件： |
| --- | --- |
| | ``` [*BOUNDARY](../key/key-link.md#usb-kws-hboundary), ACOUSTIC PRESSURE *节点* *数值* ``` |

### 声学材料

声学介质需要定义以下属性：
- 密度；
- 体积模量；
- （可选）声吸收系数。

### 输出

声学分析的输出包括：
- 声压；
- 声压速度；
- 结构位移和应力；
- 界面力。

### 局限性

- 声学分析仅支持线性行为；
- 声学单元不能与实体单元直接耦合；
- 不支持流动声学效应。

### 输入文件模板

```
[*HEADING](../key/key-link.md#usb-kws-mheading)
…
[*ACOUSTIC INITIAL](../key/key-link.md#usb-kws-hacousticinitial)

[*STEP](../key/key-link.md#usb-kws-hstep)
[*DYNAMIC](../key/key-link.md#usb-kws-hdynamic)
[*ACOUSTIC ANALYSIS](../key/key-link.md#usb-kws-hacousticanalysis)
[*BOUNDARY](../key/key-link.md#usb-kws-hboundary)
*定义声学边界条件*
[*END STEP](../key/key-link.md#usb-kws-hendstep)
```
