# 12.15.2 管理截面分配







**截面分配管理器**允许您查看、创建、编辑、抑制、恢复和删除截面分配。管理器按创建顺序列出截面分配，并显示每个截面分配关联的截面名称、截面类型、材料名称和区域。在属性模块中，Abaqus/CAE将区域着色为绿色以表示该区域具有截面分配，或者如果存在重叠的截面分配，则将区域着色为黄色。

**注意：**分配的顺序可能是相关的；当截面分配重叠时，最后的分配将优先。

**创建、编辑、抑制、恢复或删除截面分配：**

1. 从主菜单栏，选择**截面**![](../graphics/images/arrow.gif)**截面分配管理器**。
2. 从管理器底部，选择以下选项：- 如果您想创建新的截面分配，请单击**创建**并遵循"分配截面"，第12.15.1节中描述的过程。新的截面分配出现在管理器中。- 如果您想编辑截面分配，请在管理器中选择截面分配，然后单击**编辑**。**编辑截面分配**对话框出现，并且当前分配截面的区域在视口中高亮显示。1. 如有需要，为截面分配选择不同的截面。2. 单击![](../graphics/ico_selectBlue.png)为截面分配选择新区域。3. 从视口或现有集合列表中选择区域，然后单击鼠标按钮2表示已完成选择。4. 单击**确定**关闭**编辑截面分配**对话框。- 如果您想删除截面分配，请在管理器中选择截面分配，然后单击**删除**。
3. 在管理器的左列中，单击绿色复选标记抑制截面分配，或单击红色"X"恢复被抑制的截面分配。**注意：**抑制和恢复连接器截面分配将抑制或恢复相应的连接器方向。Abaqus/CAE更新所选零件或区域的颜色以反映修订后的截面分配状态。
4. 单击**关闭**关闭**截面分配管理器**对话框。
