# 5.2.3 旋转视图工具





当你选择旋转工具 ![](../graphics/ico_viewRotate.png) 和要工作的视口时，Abaqus/CAE 进入旋转模式。在此模式下，光标变为两个弯曲箭头，视口中出现一个大圆。要定义旋转中心，你可以直接输入其坐标或从视口中选择一个点。否则，Abaqus/CAE 将围绕视口中心旋转视图。如果你通过从视口中选择位置或输入坐标来选择旋转中心，该旋转中心位置会覆盖视图中心，并保持选中状态，直到你选择新的旋转中心、显示不同对象或选择默认旋转中心。拖动光标时模型的视图会旋转，橡皮筋线条表示旋转量和方向。当你旋转模型视图时，视图 triad 指示全局坐标系统的方向。

**注意：**当 *X–Y* 绘图显示在当前视口中时，Abaqus/CAE 会禁用旋转工具。

当你进入旋转模式时绘制的圆代表围绕对象的假想球体的轮廓。当你在圆内拖动鼠标时，你可以想象你实际上是在旋转球体，就像使用轨迹球一样。你的模型连接在球体中心，因此旋转球体会导致模型视图也随之旋转。

你通过将光标移动到假想球体表面上来确定旋转轴。橡皮筋线条代表切割平面与球体表面的交线，旋转轴垂直于该切割平面。旋转角度等于橡皮筋线条在球体表面上形成的角度，因此在圆上完全拖动会产生 180 度旋转。[图 5-5](pt01ch05s02s03.md#viw-rotate-tool) 说明了这个假想球体和在其表面拖动的橡皮筋线条。

**图 5-5** 旋转工具。

![](../graphics/viw-rotate-tool-nls.png)

当你在圆外拖动时，橡皮筋线条叠加在圆的边缘上，你对对象的视图仅围绕穿过圆心的屏幕法向轴旋转。与在圆内拖动一样，橡皮筋线条代表对象旋转的角度。

使用默认视图旋转模式类似于围绕视图中心或所选旋转中心旋转相机，如图 [图 5-6](pt01ch05s02s03.md#viw-rotate1) 所示。

**图 5-6** 围绕目标或所选旋转中心旋转相机。

![](../graphics/viw-rotate1.png)

旋转工具的替代模式通过在执行操作时按住 **[Shift]** 来访问，它围绕自身旋转相机，如图 [图 5-7](pt01ch05s02s03.md#viw-rotate2) 所示。这会移动相机目标和平截头体，而不考虑原始视图中对象的位置。当你在电影模式且相机位于模型内部时，围绕自身旋转相机最有用。在这个位置，移动相机目标和平截头体会将模型内部的不同部分带入视图。

**图 5-7** 围绕自身旋转相机。

![](../graphics/viw-rotate2.png)

**注意：**如果你已选择一个点作为旋转中心，你的选择会覆盖旋转的替代模式。

在任何一种模式中，通常更容易通过执行一系列较小的旋转而不是一次大旋转来获得所需的旋转。如果你需要放弃旋转并返回已知方向，请使用**视图**工具栏中的预定义视图或循环视图工具 ![](../graphics/ico_viewCycle.png)。使用任何预定义视图也会将相机目标重置到模型中心。

由于 *X–Y* 绘图是二维的，当 *X–Y* 绘图显示在当前视口中时，Abaqus/CAE 会禁用旋转工具。
