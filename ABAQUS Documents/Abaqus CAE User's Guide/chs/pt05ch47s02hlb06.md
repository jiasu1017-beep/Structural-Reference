# 47.2.6 从键盘输入X–Y数据









### 47.2.6 从键盘输入X--Y数据

您可以直接从键盘指定*X–Y*数据。为此，请将*X*和*Y*值键入简单的表格编辑器。

**要从键盘输入X–Y数据：**

1. 找到**来自键盘的XY数据**选项：从主菜单栏选择****工具**![](../graphics/images/arrow.gif)**XY数据**![](../graphics/images/arrow.gif)**创建****。在出现的对话框中点击**键盘**；然后点击**继续**。**提示：**您也可以通过点击**XY数据管理器**中的**创建**或使用工具箱中的![](../graphics/ico_xyData.png)工具来指定*X--Y*数据。出现**来自键盘的XY数据**对话框。
2. 使用标准键盘和鼠标编辑技术在**来自键盘的XY数据**表格中插入、修改或删除*X*和*Y*值。对于特殊表格编辑选项或从ASCII文件读取数据，请点击鼠标按钮3。（有关更多信息，请参见["输入表格数据，" 第3.2.7节](pt01ch03s02s07.md)。）
3. 要为*X*或*Y*列中的值分配量类型，展开**X**或**Y**列的**量类型**列表，然后从出现的列表中选择量类型。
4. 要评估和显示数据，点击**绘图**。当前视口中出现*X--Y*图。该图表示您在对话框中配置的数据，Abaqus认为这是临时数据，无论您是否点击**另存为**保存它。
5. 要保存您配置的数据，点击**另存为**。**注意：**要绘制您保存的*X--Y*数据对象，请从主菜单栏选择****工具**![](../graphics/images/arrow.gif)**XY数据**![](../graphics/images/arrow.gif)**绘图****，然后从弹出菜单中选择*X--Y*数据对象。
6. 完成后，点击**取消**以关闭对话框。
