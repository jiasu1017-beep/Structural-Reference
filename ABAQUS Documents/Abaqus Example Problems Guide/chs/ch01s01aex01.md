# 1.1.1 螺栓管法兰连接的轴对称分析





**产品：** Abaqus/Standard  

螺栓管法兰连接是许多管道系统中的常见且重要的部件。此类连接通常由管道轮毂、管法兰（带螺栓孔）、螺栓和螺母组以及垫圈组成。这些部件在紧固过程中以及施加工作载荷（如内压和温度）时相互作用。实验和数值研究经常报告关于这些部件之间不同类型相互作用的研究。这些研究包括螺栓上紧过程分析，以获得均匀的螺栓应力（Bibel and Ezell, 1992）、螺纹接触分析（Fukuoka, 1992; Chaaban and Muzzo, 1991）以及整个管道接头组件的完整应力分析（Sawa et al., 1991）。为了建立最佳设计，完整应力分析需要确定以下因素：控制密封性能的接触应力、螺栓力与内压之间的关系、有效的垫圈密封宽度以及螺栓中产生的弯矩。本示例展示如何使用经济的轴对称模型执行此类设计分析，以及如何通过将结果与使用三维段模型模拟获得的结果进行比较来评估轴对称解的准确性。此外，还分析了使用多级子结构的三维模型，以演示具有大量保留自由度子结构的使用。最后，分析了包含刚度矩阵的三维模型，以演示矩阵输入功能的使用。

### 几何结构和模型

所分析的螺栓接头组件如图1.1.1-1所示（ch01s01aex01.md#sxmpipeflange-schematic）。各个部件的几何形状和尺寸取自Sawa等人（1991），并进行了轻微简化以简化建模。轮毂和垫圈的内壁半径均为25毫米。管道法兰和垫圈的外壁半径分别为82.5毫米和52.5毫米。垫圈厚度为2.5毫米。管道法兰有八个螺栓孔，以半径为65毫米的节圆等间距排列。螺栓孔半径在本分析中被修改为与螺栓半径相同：8毫米。螺栓头部（支承面）假定为圆形，其半径为12毫米。

螺栓和管道轮毂/法兰的杨氏模量为206 GPa，泊松比为0.3。垫圈可使用实体连续体单元或垫圈单元建模。当使用连续体单元时，垫圈的杨氏模量E等于68.7 GPa，泊松比![](../graphics/exa_eqn00001.gif)等于0.3。

当使用垫圈单元时，使用线性垫圈压力/闭合关系，有效"法向刚度"![](../graphics/exa_eqn00002.gif)等于材料杨氏模量除以厚度，因此![](../graphics/exa_eqn00003.gif) 27.48 GPa/mm。类似地，使用线性剪切应力/剪切运动关系，有效剪切刚度![](../graphics/exa_eqn00004.gif)等于材料剪切模量除以厚度，因此![](../graphics/exa_eqn00005.gif) 10.57 GPa/mm。薄膜行为指定杨氏模量为68.7 GPa，泊松比为0.3。所有接触区域假定为粘滞接触条件：支承面与法兰之间以及垫圈与轮毂之间。忽略螺栓杆与螺栓孔之间的接触。

管道接头对称半部的有限元理想化模型分别如图1.1.1-2和图1.1.1-3所示（ch01s01aex01.md#sxmpipeflange-axisym和ch01s01aex01.md#sxmpipeflange-3dmodel），分别对应轴对称分析和三维分析。轴对称分析使用的mesh由管道轮毂/法兰和垫圈的mesh以及螺栓的独立mesh组成。在图1.1.1-2中（ch01s01aex01.md#sxmpipeflange-axisym），上图显示了管道轮毂和法兰的mesh，螺栓孔区域以较浅颜色显示；下图显示了包含垫圈和螺栓的完整mesh。

对于轴对称模型，管道轮毂/法兰的整个mesh使用二阶减缩积分单元CAX8R。垫圈可使用CAX8R实体连续体单元或GKAX6垫圈单元建模。垫圈与管道轮毂/法兰之间的接触使用接触对建模，接触对由接触区域中元素面上定义的曲面或此类基于元素的曲面与基于节点的曲面之间的接触对定义。在轴对称分析中，必须正确建模螺栓和穿孔法兰。螺栓建模为平面应力单元，因为它们不承受环向应力。为此目的采用具有减缩积分的二阶平面应力单元CPS8R。与元素面相关的接触曲面定义自动考虑平面应力条件。为了考虑接头中使用的全部八个螺栓，必须计算螺栓杆和螺栓头部的组合横截面积，并使用实体单元的面积属性适当地重新分配到螺栓mesh。接触面积自动调整。

三维分析的mesh（无子结构）如图1.1.1-3所示（ch01s01aex01.md#sxmpipeflange-3dmodel），代表管道接头的22.5度段，采用减缩积分的二阶砖单元C3D20R用于管道轮毂/法兰和螺栓。垫圈使用C3D20R单元或GK3D18单元建模。上图显示了管道轮毂和法兰的mesh，下图显示了垫圈和螺栓（浅色）。接触通过将接触区域中特定元素面分组定义的接触曲面的相互作用建模。对于双侧均可变形的master和slave表面的三维接触，必须使用小滑动接触对公式来表明接触表面之间发生小的相对滑动。不需要对三维模型中使用的材料属性进行特殊调整，因为所有部件都已被正确建模。

测试了四种使用子结构建模法兰的不同mesh。为图1.1.1-3所示的整个22.5度法兰段创建了第一级子结构（ch01s01aex01.md#sxmpipeflange-3dmodel），而垫圈和螺栓的mesh与之前相同。与螺栓帽接触的法兰上的节点形成基于节点的曲面，而与垫圈接触的法兰上的节点形成另一个基于节点的曲面。这些基于节点的曲面将与螺栓帽上的master曲面和垫圈上的master曲面形成接触对，这些曲面使用曲面定义选项定义。子结构上保留的自由度包括这些基于节点曲面中节点的所有三个自由度，以及法兰0度和22.5度面上节点的所有三个自由度。在子结构使用级别指定适当的边界条件。

通过相对于22.5度平面反射第一级子结构来创建45度的第二级子结构。属于反射子结构的22.5度面上的节点与属于原始第一级子结构的22.5度面上的对应节点在所有三个自由度上被约束。属于反射子结构的半螺栓和垫圈扇区也通过反射构建。保留的自由度包括所有接触节点集的所有三个自由度以及法兰0度和45度面上节点的所有三个自由度。MPC类型CYCLSYM用于在这些两个面上施加循环对称边界条件。

通过相对于45度平面反射原始45度第二级子结构并将其连接到原始45度子结构来创建90度的第三级子结构。模型45-90度扇区中对应的垫圈和螺栓的剩余部分通过反射和适当约束创建。在这种情况下，不需要在法兰的0度和90度面上保留任何自由度，因为这个90度子结构不会连接到其他子结构，可以在校结构创建级别指定适当的边界条件。

最终子结构模型通过相对于垂直于y轴的垫圈对称平面镜像90度mesh来设置。因此，当不使用子结构时本应较大的分析（![](../graphics/exa_eqn00039.gif) 750,000个未知量）可以通过使用第三级子结构两次方便地求解（![](../graphics/exa_eqn00039.gif) 80,000个未知量）。使用稀疏求解器，因为它显著减少了此模型的运行时间。

最后，通过将图1.1.1-3所示的整个22.5度法兰段的单元替换为刚度矩阵来创建基于矩阵的三维模型，而垫圈和螺栓的mesh与之前相同。法兰与垫圈之间以及法兰与螺栓帽之间的接触使用与子结构模型相同的基于节点的slave曲面进行建模。应用与无子结构三维模型相同的适当边界条件。

### 载荷和边界条件

唯一的边界条件是对称边界条件。在轴对称模型中，![](../graphics/exa_eqn00040.gif) 0被施加到垫圈的对称平面和螺栓的底部。在三维模型中，![](../graphics/exa_eqn00041.gif) 0被施加到垫圈的对称平面以及螺栓的底部。![](../graphics/exa_eqn00042.gif)0和![](../graphics/exa_eqn00042.gif)22.5度平面也是对称平面。在![](../graphics/exa_eqn00042.gif)22.5度平面上，通过使用适当的节点变换并向该对称平面中的局部方向施加边界条件来强制执行对称边界条件。这些变换使用局部坐标系定义实现。在两个对称平面上，除与C BIQUAD MPC相关的从属节点和接触表面一侧上的节点外，对称边界条件![](../graphics/exa_eqn00040.gif) 0被施加到所有位置。第二个例外是为了避免过度约束问题，当在粗糙摩擦规范相关的Lagrange乘数约束的相同方向上存在边界条件时会出现此问题。

在使用子结构的模型中，边界条件根据所使用的子结构指定。对于第一级22.5度子结构，边界条件和约束方程与图1.1.1-3所示的三维模型相同（ch01s01aex01.md#sxmpipeflange-3dmodel）。对于45度第二级子结构，对称边界条件在![](../graphics/exa_eqn00043.gif)45度平面上强制执行，约束方程为![](../graphics/exa_eqn00044.gif) 0。也可以使用变换。对于90度第三级子结构，面![](../graphics/exa_eqn00045.gif)90度用边界条件![](../graphics/exa_eqn00046.gif) 0约束。

对于包含矩阵的三维模型，对称边界条件应用节点变换。这些节点的刚度矩阵条目也在局部坐标系中。

通过将预拉伸节点与预拉伸截面关联，对每个螺栓施加15 kN的夹紧力。预拉伸截面通过曲面定义来标识。然后通过向预拉伸节点施加集中载荷来指定预拉伸。在轴对称分析中，由于有八个螺栓，实际施加的载荷为120 kN。在无子结构的三维模型中，由于只建模了半个螺栓，实际施加的载荷为7.5 kN。在使用子结构的模型中，所有半螺栓都加载7.5 kN的力。对于所有模型，预拉伸截面指定在螺栓杆中间位置。

在所有分析中，所有表面相互作用假定为粘滞接触条件，并用粗糙摩擦和无分离接触模拟。

### 结果与讨论

所有分析均作为小位移分析进行。

图1.1.1-5（ch01s01aex01.md#sxmpipeflange-normsol）显示了当使用实体连续体单元建模垫圈时，轴对称分析（底部）和三维分析（顶部）预测的垫圈在垫圈与管道轮毂/法兰界面处的法向应力分布顶视图。该图显示，垫圈上的法向压应力在外边缘最高，向内径向减小，在约35毫米半径处从压应力变为拉应力，这与Sawa等人（1991）的发现一致。轴对称分析和三维分析之间整体解的非常接近表明，对于此类问题，轴对称分析提供了三维分析的简单而相当准确的替代方案。

图1.1.1-6（ch01s01aex01.md#sxmpipeflange-normgas）显示了当使用垫圈单元建模垫圈时，轴对称分析（底部）和三维分析（顶部）预测的垫圈在垫圈与管道轮毂/法兰界面处的法向应力分布顶视图。轴对称分析和三维分析之间整体解的接近程度在此情况下也很明显。垫圈在约40毫米半径处开始承受压缩载荷，与之前结果相差5毫米。这种差异是由于垫圈单元无法在其厚度方向上承受拉载荷。这个解决方案在物理上更真实，因为大多数情况下，垫圈在受到拉伸载荷时会与相邻部件分离。从输入文件中移除垫圈/法兰接触曲面定义中的无分离接触会产生与图1.1.1-6中获得的良好一致性（因为在那种情况下，垫圈中的实体连续体单元无法在垫圈厚度方向上承受拉伸载荷）。

本示例中的模型可以修改用于研究其他因素，例如垫圈的有效密封宽度或垫圈在工作载荷下的密封性能。垫圈单元具有允许在垫圈厚度方向上定义非常复杂行为的优势。垫圈单元还可以使用Abaqus提供的任何小应变材料模型，包括用户定义的材料模型。图1.1.1-7（ch01s01aex01.md#sxmpipeflange-isogas）显示了在为垫圈单元指定各向同性材料属性时，轴对称分析（底部）和三维分析（顶部）预测的垫圈在垫圈与管道轮毂/法兰界面处的法向应力分布比较。图1.1.1-7中的结果与使用实体和轴对称单元模拟垫圈的分析中获得的结果（图1.1.1-5）比较良好。

图1.1.1-8（ch01s01aex01.md#sxmpipeflange-zlinestress）显示了界面处垫圈在平面![](../graphics/exa_eqn00047.gif) 0中的法向应力分布。结果针对以下模型绘制：仅包含实体连续体单元且无子结构的三维模型、包含矩阵的三维模型，以及包含上述子结构的四个模型。

有一个执行过程可将两个子结构输出数据库中的模型和结果数据组合成单个输出数据库。有关更多信息，请参见Abaqus Analysis User's Guide的第3.2.22节"Combining output from substructures"（../usb/usb-link.md#usb-int-dsubstructurecombineproc）。

本示例也可用于演示准Newton非线性求解器的有效性。该求解器在几个连续平衡迭代中使用廉价的近似刚度矩阵更新，而不是像默认的完整Newton方法那样在每次迭代中执行完整的刚度矩阵分解。准Newton方法导致更多但较便宜的迭代次数增加，以及计算成本的净节省。
