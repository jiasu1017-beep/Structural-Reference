# 49.1.3 谐波动画





谐波动画仅对包含复数的场输出有效。在谐波动画中，Abaqus/CAE从输出数据库的单个步骤和帧创建变形形状、等值线或符号图序列中的所有图像。Abaqus/CAE生成根据您的规范从0到180或从180到180的角度，并显示每个角度的复数场输出值。这给出了场输出随时间演变的 appearance，尽管所有图像都是从您的结果的单个步骤和帧生成的。（要查看结果随时间的实际演变，您必须生成时间历史动画。）谐波动画对于模拟特征值分析计算的振动模式很有用。

如果您正在动画化等值线图，场输出值的显示取决于您选择的选项。默认行为是图例中的值基于实部的范围固定。要了解如何设置等值线限制，请参阅["设置等值线限制，"第44.5.7节](pt05ch44s02hlb07.md)。要了解如何自定义图例，请参阅["自定义图例，"第56.1节](pt05ch56s01.md)。要了解有关复数结果的更多信息，请参阅["理解复数结果，"第42.6.3节](pt05ch42s04s03.md)。
