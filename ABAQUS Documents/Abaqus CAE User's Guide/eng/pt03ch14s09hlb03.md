# 14.9.3 Editing a step







You can use the step editor to edit the analysis procedure settings associated with an existing step.

**To edit a step:**

1. From the main menu bar, select ****Step**![](../graphics/images/arrow.gif)**Edit**![](../graphics/images/arrow.gif)***step name*****. The step editor appears. **Tip:**You can also select the step name in the **Step Manager** and click **Edit**.
2. Use the tabs within the step editor to modify the settings. (For detailed help on a particular editor feature, select ****Help**![](../graphics/images/arrow.gif)**On Context**** from the main menu bar and then click the feature of interest.)
3. Click **OK** to close the step editor and save the new settings.
