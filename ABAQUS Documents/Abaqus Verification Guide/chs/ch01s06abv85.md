# 1.6.6 可变形体与刚性表面间的有限滑动接触





**产品：**Abaqus/Standard  Abaqus/Explicit

### 功能测试

接触对

可变形体上基于单元的表面或基于节点的表面，以及刚性表面

### 问题描述

Abaqus/Standard模型由放置在刚性表面上的实体单元或梁单元组成，或者对于三维实体单元，距离刚性表面一个单位。在后一种情况下，在第一步中施加位移以使物体与刚性表面接触。假设为无摩擦接触。接触建立后，向可变形单元施加向下的压力，导致接触压力和实体单元中的应力。执行两个压力载荷步。第一步是几何线性分析，而第二步调用几何非线性，这考虑了接触面积的增加。接触压力（CPRESS）应在两步中平衡施加的压力载荷。

对于带扭转的轴对称单元，测试由五个步骤组成。最初实体单元与刚性表面干涉。第一个步骤（非线性步骤）消除这种过闭合。接下来三个步骤是线性扰动步骤，在两个接触表面之间进行相对滑动和/或扭转。随后的非线性步骤结合了两个表面之间的相对滑动和扭转。

在最后三个步骤中，进行三个线性扰动分析：两个接触物体在谐波分布载荷作用下的直接求解稳态动态分析、固有频率提取分析，以及基于子空间的稳态动态分析。

对于圆柱膜单元，模型由两个同心圆柱体组成。可变形圆柱体用圆柱膜单元网格化，半径为一个单位。刚性圆柱体用解析刚性表面建模，半径为1.2个单位。测试由三个步骤组成。最初圆柱膜单元与刚性表面干涉。第一个步骤（线性步骤）消除这种过闭合。第二个非线性步骤中摩擦值发生变化。在最后一步中，两个接触表面之间进行相对滑动。

Abaqus/Explicit模型由单个梁单元与解析刚性表面接触组成。分析有两个步骤。在第一步中，使用可变形梁上基于节点的表面建立无摩擦接触对定义。在下一步中，使用摩擦重新定义接触对，并且使梁在解析刚性表面上滑动。对于梁和管道单元获得一致的接触应力。

**模型：**

| 梁的长度 | 1.0 |
| --- | --- |
| 二维实体单元尺寸 | 5×5 |
| 三维实体单元尺寸 | |

| C3D8（接触节点集） | 接触面积 = 5×1 |
| --- | --- |
| C3D6 | 不规则：接触面积 = 3 |
| C3D8 | 不规则：接触面积 = 4 |
| C3D27 | 接触面积 = 3×2 |

**材料：**

| 杨氏模量 | 30 106 |
| --- | --- |
| 泊松比 | 0.3 |

### 结果与讨论

对于施加压力测试，接触压力在可变形单元上精确平衡施加的向下压力载荷。
