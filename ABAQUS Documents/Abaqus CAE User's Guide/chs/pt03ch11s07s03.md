# 11.7.3 刚体零件和刚体约束有什么区别？





您可以通过创建零件并将其类型声明为离散或分析刚体来在零件模块中创建刚体零件。您可以创建参考点并将其分配给刚体参考点。然后，您应用于参考点的运动或约束将应用于整个刚体零件。

类似地，您可以在相互作用模块中创建刚体约束。刚体约束允许您将装配区域的运动约束到参考点的运动。作为刚体一部分的区域在分析过程中保持相对位置不变。此外，您可以从零件实例中选择区域，并使用刚体约束为完全耦合热应力分析指定等温刚体。有关定义刚体约束和分配刚体参考点的详细说明，请参见"定义刚体约束"，第15.15.2节。

您不必为零件创建参考点，即使零件类型是离散或分析刚体。但是，如果不为刚性零件创建参考点，则装配中该零件的每个实例都必须包含在刚体约束中。

刚体零件与零件关联；刚体约束与装配区域关联。例如，如果您将零件定义为刚体，则装配中该零件的每个实例都是刚性的。相反，如果您将零件定义为可变形，可以使用刚体约束仅使某些实例成为刚性的。如果您在零件模块中未创建参考点，则不能通过将刚性零件的实例与在装配模块中创建的参考点关联来创建刚体参考点。但是，您可以将实例与刚体约束以及在装配模块中创建的参考点关联。

如果您将零件定义为刚体，可以使用模型树将零件类型更改为可变形。为了检查基本模型是否正确，您可以先用定义为刚性的零件运行快速分析，然后将类型更改为可变形。类似地，如果您将零件定义为可变形并在装配中的零件实例上施加刚体约束，您可以稍后轻松移除约束。您可以使用刚体约束应用于零件实例来运行快速分析，然后移除约束并使用作为可变形体作用的零件实例运行完整分析。这两种方法非常相似。
