# 26.1 共仿真概述






在 Abaqus/CAE 中建模和运行共仿真涉及以下一般步骤：

1. 在单个模型数据库中创建模型。
2. （可选）链接模型之间的零件实例，并从分析中排除链接的实例。
3. （可选）确保接口区域处的节点匹配。
4. 在每个模型中，创建共仿真相互作用以指定接口区域和耦合方案。
5. 创建共执行以识别所涉及的两个模型，并指定每个分析的作业参数。
6. 提交共执行以执行共仿真。
7. 使用叠加图查看共仿真的结果。

Abaqus/CFD 到 Abaqus/Standard 共仿真的示例在["元件安装电子电路板的共轭热传递分析，" Abaqus 例题指南第 6.1.1 节](../exa/exa-link.md#exa-cfd-conjugateheattransfer)中显示。
