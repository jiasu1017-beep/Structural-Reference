# 49.4.4 显示动画帧计数器





对于基于对象的动画，您可以显示或抑制动画帧计数信息，该信息显示在视口的右上角，包括：
- 比例因子动画中每个图像的比例因子，
- 谐波动画中每个图像的相位角，或
- 时间历史动画中包含的每个图像的步骤和帧（步骤内的绘图编号）。

除了完全关闭之外，您无法自定义计数器的内容或外观。

与每个动画步骤和帧关联的步骤和增量由状态块给出。比例因子动画中的所有图像都基于单个步骤和帧。有关自定义状态块的更多信息，请参阅 ["自定义状态块，" 第 56.3 节"](pt05ch56s03.md)。

**显示动画帧计数器：**

1. 找到 **显示帧计数器（Show frame counter）** 选项。在提示区域中，点击 **动画选项（Animation Options）**；然后在出现的对话框中点击 **播放器（Player）** 选项卡。**显示帧计数器（Show frame counter）** 选项位于页面底部附近。
2. 切换 **显示帧计数器（Show frame counter）** 以显示或抑制动画帧计数信息。
3. 点击 **应用（Apply）** 实施您的更改。所有动画视口中的动画都会根据您的规范发生变化。当 **显示帧计数器（Show frame counter）** 开启时，Abaqus/CAE 在所有动画视口的右上角显示状态信息。该信息为动画序列中的每个图像更新。您可能需要 [停止](pt05ch49s04hlb01.md) 播放来读取它。您的更改将在会话期间保存，并将影响此状态中的所有后续绘图。

# 49.4.3 控制动画速度和重复





您可以控制基于对象动画播放的速度，并可以选择播放动画一次或重复播放。如果您选择重复播放动画，可以选择三种重复模式之一：循环、循环反向或摆动。**循环（Loop）** 通过从最后一帧跳转到电影开头来重复。**循环反向（Loop backward）** 反向播放动画并通过从第一帧跳转到电影结尾来重复。**摆动（Swing）** 通过从头到尾播放电影，然后从尾到头反向播放来平滑重复。

在创建基于图像的动画之前指定的速度控制设置决定在行业标准动画软件中重放 QuickTime 或 AVI 文件时使用的默认帧速率。

**自定义基于对象动画的动画速度和重复：**

1. 在提示区域中，点击 **动画选项（Animation Options）**；然后在出现的对话框中点击 **播放器（Player）** 选项卡。
2. 要控制动画的播放速度，请拖动 **帧速率（Frame rate）** 滑块到您想要的速率。
3. 要控制动画序列中图像的重复：- 点击 **播放一次（Play once）** 抑制重复。- 点击 **循环（Loop）** 按顺序从第一帧到最后一帧重复播放图像。- 点击 **循环反向（Loop backward）** 按顺序从最后一帧到第一帧重复播放图像。- 点击 **摆动（Swing）** 从第一帧到最后一帧播放图像，然后从最后一帧到第一帧反向播放，重复进行。
4. 点击 **应用（Apply）** 实施您的更改。所有动画视口中的动画都会根据您的规范发生变化。您的更改将在会话期间保存，并将影响此状态中的所有后续绘图。

# 49.4.6 控制多视口中的动画





默认情况下，Abaqus/CAE 在会话中的每个符合动画条件的视口中为数据制作动画。所有动画都是同步的，因此当您逐帧检查数据、为模型绘图制作动画或为随时间变化的 X–Y 绘图制作动画时，视口会一起播放、停止和递增。

您可以激活和停用各个视口的动画，这使您能够仅为会话中符合动画条件的视口子集显示动画。**动画选项（Animation Options）** 对话框的 **视口（Viewports）** 页面（如图 49–1](pt05ch49s04hlb06.md#usv-anm-viewport-tab) 所示）显示会话中所有符合动画条件的视口的列表。切换此页面上视口的复选框可将该视口包括在动画中或从中排除。

**图 49–1** **动画选项（Animation Options）** 对话框的 **视口（Viewports）** 页面。

![](../graphics/usv-anm-viewport-tab-nls.png)

**控制将为其制作数据动画的视口：**

1. 找到 **视口（Viewports）** 选项。在提示区域中，点击 **动画选项（Animation Options）**；然后在出现的对话框中点击 **视口（Viewports）** 选项卡。出现 **视口（Viewports）** 页面。
2. 为您想要制作动画的视口切换打开复选框。
3. 如果需要，通过从 **动画选项（Animation Options）** 对话框底部的字段中选择 **动画类型（Animation type）** 来切换动画类型。
4. 点击 **应用（Apply）**。

# 49.4.7 自定义同步视口的时间历史





时间历史设置控制当您显示时间历史动画时 Abaqus/CAE 是使用基于帧的历史还是基于时间的历史。基于帧的同步允许所有同步视口一次一帧地一起递增，而基于时间的同步允许视口根据公共时间线显示数据。

当您选择基于时间的动画时，您还可以自定义公共时间线的三个方面：
- 时间步长，控制每次递增数据显示时经过多少时间。
- 最小时间，这是包含数据的最早点。如果自动计算最小时间，Abaqus/CAE 从公共时间线的开头开始包含数据。
- 最大时间，这是包含数据的最晚点。如果自动计算最大时间，Abaqus/CAE 包含数据到公共时间线的结尾。

**自定义数据的时间历史：**

1. 找到 **时间历史（Time History）** 选项。在提示区域中，点击 **动画选项（Animation Options）**；然后在出现的对话框中点击 **时间历史（Time History）** 选项卡。
2. 从页面顶部的选项中，选择 **基于帧（Frame-based）** 或 **基于时间（Time-based）**。
3. 如果需要，更改 **时间增量（Time increment）** 值。时间增量仅适用于基于时间的动画。
4. 如果需要，通过在相应的 **指定（Specify）** 字段中输入值来自定义 **最小时间（Min time）** 和 **最大时间（Max time）**。这些值仅适用于基于时间的动画。
5. 点击 **应用（Apply）**。

# 49.4.5 自定义 X–Y 绘图动画





### 49.4.5 自定义 X–Y 绘图动画

当您为随时间变化的 X–Y 绘图制作动画时，Abaqus/CAE 会向绘图添加一条沿 X 轴移动的垂直线，以高亮显示当前时间。绘图还会在每个 X–Y 曲线上最接近当前时间发生的数据点处显示彩色符号；时间跟踪器符号不会出现在时间跟踪器线与 X–Y 曲线的确切交点处，除非该曲线在当前显示的时间有一个数据点。下图显示了带有时间跟踪器线和符号的 X–Y 绘图。

![](../graphics/usv-anm-animxy-timetracker.png)

默认情况下，时间跟踪器线和符号使用您为会话选择的突出显示方法显示；有关更多信息，请参阅 ["选择突出显示方法，" 第 7.4 节"](pt01ch07s04.md)。但是，您可以通过更改时间跟踪器线的颜色、粗细和样式以及时间跟踪器符号的颜色、形状和大小来自定义其外观。或者，您可以完全抑制任一时间跟踪器组件。默认的突出显示方法提供最佳性能。

**注意：**这些线条和符号选项仅控制时间跟踪器的外观。您还可以自定义用于显示 X–Y 曲线本身的线条和符号；请参阅 ["自定义 X–Y 曲线外观，" 第 47.6.4 节"](pt05ch47s05hlb03.md)。

**自定义时间跟踪器的外观：**

1. 找到 **XY** 选项。在提示区域中，点击 **动画选项（Animation Options）**；然后在出现的对话框中点击 **XY** 选项卡。
2. 如果您想自定义 X–Y 绘图时间跟踪器的外观，请关闭 **使用突出显示方法绘制（Draw using highlight method）**。
3. 切换 **显示线（Show line）** 以显示或抑制时间跟踪器线。当 **显示线（Show line）** 开启时，显示时间跟踪器线，线条属性选项被启用。
4. 选择时间跟踪器线样式：1. 点击 **样式（Style）** 按钮显示线条样式选项（实线、虚线等）。2. 从样式列表中，点击所需的线条样式。指定的线条样式出现在 **样式（Style）** 按钮上。
5. 选择时间跟踪器线粗细：1. 点击 **粗细（Thickness）** 按钮显示线条粗细选项。2. 从粗细列表中，点击所需的线条粗细。指定的粗细出现在 **粗细（Thickness）** 按钮上。
6. 选择时间跟踪器线颜色：1. 点击颜色样本 ![](../graphics/usi-colorsample.png)。Abaqus/CAE 显示 **选择颜色（Select Color）** 对话框。2. 使用 **选择颜色（Select Color）** 对话框中的方法选择新颜色。有关更多信息，请参阅 ["自定义颜色，" 第 3.2.9 节"](pt01ch03s02s09.md)。3. 点击 **确定（OK）** 关闭 **选择颜色（Select Color）** 对话框。颜色样本变为所选颜色。
7. 切换 **显示符号（Show symbol）** 以显示或抑制表示每个选定 X–Y 曲线数据点当前值的符号。当 **显示符号（Show symbol）** 开启时，显示符号，符号属性被启用。
8. 选择符号：1. 点击 **符号（Symbol）** 按钮显示符号选项。2. 从符号列表中，点击所需的符号。指定的符号出现在 **符号（Symbol）** 按钮上。
9. 选择时间跟踪器符号颜色：1. 点击颜色样本 ![](../graphics/usi-colorsample.png)。Abaqus/CAE 显示 **选择颜色（Select Color）** 对话框。2. 使用 **选择颜色（Select Color）** 对话框中的方法选择新颜色。有关更多信息，请参阅 ["自定义颜色，" 第 3.2.9 节"](pt01ch03s02s09.md)。3. 点击 **确定（OK）** 关闭 **选择颜色（Select Color）** 对话框。颜色样本变为所选颜色。
10. 选择时间跟踪器符号大小：1. 点击 **大小（Size）** 按钮显示符号大小选项。2. 从符号大小选项中，点击所需的符号大小。指定符号大小出现在 **大小（Size）** 按钮上。
11. 点击 **应用（Apply）**。当前视口显示您的颜色、粗细、样式和符号选择。
