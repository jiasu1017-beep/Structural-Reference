# 17.6.7 使用自适应重新网格划分改进网格





在许多情况下，您只有在执行了多次分析并评估了求解结果后才知道网格细化对于您的特定求解目标是否足够。在这些情况下通常会执行网格细化研究，即网格被逐步细化，并确认关键求解结果收敛。您可以通过向模型中有意义的区域应用重新网格划分规则，并使用Abaqus/CAE自适应重新网格划分过程基于一系列已执行的自动执行来自动进行逐步网格细化。

通过重新网格划分规则，您可以指定：
- 您想要细化网格的区域。
- 网格细化所基于的求解质量标准（例如Mises应力的误差指标）。
- 细化所基于的分析步或分析步。
- 最小和最大单元尺寸约束。
- 适合您的模拟的尺寸算法和参数。

有关更多信息，请参见["高级网格技术，"第17.14节](pt03ch17s14.md)；以及["创建、编辑和操作自适应过程，"第19.9节](pt03ch19s09.md)。
