# 47.1.4 理解量类型









量类型是描述*X–Y*数据对象某一列中数据的类别。Abaqus/CAE包含八十多种预定义量类型，描述Abaqus中每个变量的输出，包括常用类型如温度、时间、应力和应变。与每列关联的量类型出现在*X–Y*报告中，并作为该数据对象*X–Y*图的默认轴标签。

当您从场输出数据或历史输出数据创建*X–Y*数据对象时，Abaqus/CAE会自动使用输出数据库中指定的量类型。当您通过从ASCII文件加载数据或直接从键盘输入数据来创建*X–Y*数据对象时，可以在创建对象时为每列指定量类型。在任何一种情况下，您都可以通过在**编辑XY数据**对话框中编辑数据对象来更改与任一列关联的量类型。
