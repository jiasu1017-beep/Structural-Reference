# 17.3.11 显示原生网格







您可以通过点击 **Visible Objects** 工具栏中的 **Show native mesh** ![](../graphics/ico_mesh.png) 图标，在零件实例的几何形状和同一实例的网格化表示之间切换。

您可以在任何与装配体相关的模块中使用 **Show native mesh** 工具，在装配体的几何形状和装配体的网格化表示之间切换。Abaqus/CAE 在装配体中显示独立和从属零件实例的网格化表示（假设您已创建了相应的网格）。

使用 **Show native mesh** 工具在零件的几何形状及其网格化表示之间切换，可让您查看网格与几何形状的吻合程度。该工具还允许您查看 Abaqus/CAE 如何将虚拟拓扑合并到网格中。此外，您可能会发现，在 Job 模块中点击 **Show native mesh** 工具很有用。这样，您就可以在提交作业进行分析之前确认整个装配体已正确网格划分。

模型中任何孤立单元的显示不受此工具影响；无论您是显示零件实例的本机部分的几何形状还是单元，都会显示孤立单元。
