# 58.5 显示符号以可视化映射源数据





您可以单独绘制映射源数据，而无需绘制应用源数据的载荷、相互作用或预定义场。源数据的绘制仅支持从 **XYZ** 格式点云数据创建的分析映射场。

**显示映射场源数据符号的步骤：**

1. 在模型树中，展开**字段**容器和**分析场**容器，以显示映射场对象。
2. 用鼠标按钮 3 单击映射场对象，并从出现的菜单中选择**绘制源数据**。

显示的符号表示源数据点的位置和相对幅值（场值）。幅值使用彩虹色谱进行颜色编码，从红色（最大值）到蓝色（最小值），如图 [图 58–3](pt06ch58s05.md#fld-mapsrc-colors) 所示。
