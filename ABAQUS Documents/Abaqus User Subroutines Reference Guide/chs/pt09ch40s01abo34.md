# 40.1.1 使用单元的接触建模





Abaqus/Standard提供了各种接触单元，可用于当两个物体之间的接触无法用基于表面的接触方法模拟时（[第36章，"定义接触相互作用"](pt09ch36.md)"）。这些单元包括以下内容：

**间隙接触单元：** 使用间隙单元建模两个节点之间的机械和热接触（["间隙接触单元，" 第40.2.1节](pt09ch40s02alm64.md)）。例如，这些单元可用于建模管道系统与其支撑之间的接触。它们也可用于建模只能承受拉伸载荷的不可拉伸电缆。

**管-管接触单元：** 使用管-管接触单元（["管-管接触单元，" 第40.3.1节](pt09ch40s03alm65.md)）以及滑移线建模两个管道或管之间的接触。例如，这些单元可用于模拟将管状组件放入油井的过程（钻杆或J形管分析）。它们也可用于模拟插入血管的导管。

**滑移线接触单元：** 使用滑移线接触单元（["滑移线接触单元，" 第40.4.1节](pt09ch40s04alm66.md)）以及用户定义的滑移线，建模可能发生不对称变形的两个轴对称结构之间的有限滑动接触。滑移线单元可用于建模例如螺纹连接器。

**刚表面接触单元：** 使用刚表面接触单元（["刚表面接触单元，" 第40.5.1节](pt09ch40s05alm67.md)）建模可能发生不对称变形的解析刚表面与轴对称变形体之间的接触。例如，刚表面接触单元可用于建模橡胶密封件与更硬结构之间的接触。
