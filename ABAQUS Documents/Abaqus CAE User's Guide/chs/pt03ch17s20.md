# 17.20 创建网格零件





从主菜单栏选择 ****Mesh**![](../graphics/images/arrow.gif)**Create Mesh Part**** 可以从当前网格化零件或装配创建没有几何特征的网格零件。 您也可以从已进行网格划分的选定零件实例创建网格零件。 如果您只对装配或所选零件实例的某些区域进行了网格划分，Abaqus/CAE只会从那些网格划分的区域创建网格零件； 未网格划分的区域不包含在网格零件中。

网格零件不包含特征信息，由节点、单元、曲面和集的集合定义。 当您创建网格零件时，集合、曲面和截面分配从源零件或零件实例复制，因此应用于原始零件的载荷和相互作用也会应用于网格零件。 您可以向网格零件添加几何特征，也可以使用网格编辑工具修改其节点和单元。 有关更多信息，请参阅["Edit Mesh工具集有什么用途？" 第64.1节](pt06ch64s01.md)。

**创建网格零件：**

1. 从上下文栏的**Object**字段中，选择网格化零件或选择装配。
2. 从主菜单栏，选择 ****Mesh**![](../graphics/images/arrow.gif)**Create Mesh Part****。
3. 如果装配包含多个零件实例，必须选择要包含在网格零件中的零件实例。 在提示区域，单击**Done**表示您已完成选择零件实例。
4. 在提示区域，输入新零件的名称。 如果您从装配中选择了零件实例，可以切换**Replace part instances**用新网格零件的实例替换装配实例。 Abaqus/CAE创建网格零件。
