# 73.3.5 编辑集和曲面







您可以通过在视口中重新选择实体来编辑现有集或曲面。

**编辑集或曲面的步骤：**

1. 从主菜单栏，选择****Tools**![](../graphics/images/arrow.gif)**Set**![](../graphics/images/arrow.gif)**Edit**![](../graphics/images/arrow.gif)***您选择的集*****或****Tools**![](../graphics/images/arrow.gif)**Surface**![](../graphics/images/arrow.gif)**Edit**![](../graphics/images/arrow.gif)***您选择的曲面*****。**注意：**在**Set Manager**或**Surface Manager**中选择感兴趣的集或曲面的名称后，您也可以在其中单击**Edit**。集或曲面在视口中高亮显示，Abaqus/CAE提示您重新选择对象。
2. 在视口中，重新选择要包含在集或曲面中的实体。使用**[Shift]**+**单击**和**[Ctrl]**+**单击**来更改视口中选择的实体。**提示：**您可以使用**Selection**工具栏限制可以在视口中选择的实体类型。更多信息，请参见[第6章,"在视口中选择对象"](pt01ch06.md)。选择完实体后，单击鼠标按钮2。如果您正在编辑几何集，则只能为该集选择几何体。同样，如果您正在编辑节点或单元集，则只能分别选择节点或单元。如果您正在编辑从输出数据库导入的或从包含节点和元素的输入文件导入的集，或者使用布尔并集运算符创建的包含节点、元素和几何体的集，Abaqus/CAE会提示您选择要重新选择的实体类型。如果您重新选择节点，Abaqus/CAE会保留集中任何原始的元素和几何体。根据需要重复此步骤以编辑每种实体类型。
3. 如果需要，指定要包含在曲面中的区域的侧或端，然后单击鼠标按钮2。（更多信息，请参见["指定区域的特定侧或端," 第73.2.5节"](pt06ch73s02s05.md)。）
