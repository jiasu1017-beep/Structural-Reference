# 62.9.1 由三点定义的datum坐标系







您可以通过选择以下内容来定位直角、圆柱或球datum坐标系：
- 定义原点的点。
- *X*或*R*轴上的点。
- *X*–*Y*或*R*–![](../graphics/usi_eqn00065.gif)平面上的点。

![](../graphics/dtm-hc3pts-nls.png)

当Abaqus/CAE提示您选择一个点时，您可以选择datum坐标系的原点。当Abaqus/CAE提示您选择一个边时，您可以选择datum坐标系的轴之一。

**由三点创建datum坐标系：**

1. 从主菜单栏中，选择****Tools**![](../graphics/images/arrow.gif)**Datum****。此时将出现**Create Datum**对话框。该对话框概述了您可以创建的datum几何体类型。Abaqus/CAE在提示区域中显示提示以指导您完成此过程。**提示：**您也可以使用模块工具箱中的![](../graphics/ico_dtmCsysThreePoints.png)工具创建datum坐标系。有关工具箱中datum工具的图示，请参见["使用Datum工具集，" 第62.2节](pt06ch62s02.md)。
2. 从对话框顶部的类型列表中，选择**CSYS**。**Method**列表指示您可以使用的方法来创建datum坐标系。
3. 从**Method**列表中，选择**3 points**。Abaqus/CAE显示**Create Datum CSYS**对话框。
4. 在对话框中，输入datum坐标系的名称。为了帮助您跟踪datum坐标系，Abaqus/CAE会在模型树中显示其名称。此外，您可以使用模型树重命名datum坐标系。
5. 在对话框中，选择以下datum坐标系之一： - **直角**：*X*、*Y*和*Z*轴分别与1、2和3全局轴对齐。 - **圆柱**：*R*、![](../graphics/usi_eqn00065.gif)和*Z*轴分别与1、2和3全局轴对齐。 - **球面**：*R*、![](../graphics/usi_eqn00065.gif)和![](../graphics/usi_eqn00226.gif)轴分别与1、2和3全局轴对齐。
6. 从**Create Datum CSYS**对话框中，单击**Continue**。Abaqus/CAE显示一个默认datum坐标系。datum坐标系的原点位于零件或装配的原点。此外，默认datum坐标系的轴与零件或装配的坐标系对齐。
7. 如果您想接受默认datum坐标系，请从提示区域单击**Create Datum**。Abaqus/CAE创建datum坐标系。
8. 要在不同位置或以不同方向创建datum坐标系，请执行以下操作： 1. 从当前视口中的零件或装配中，选择将定义datum坐标系原点的点，或输入原点的坐标。Abaqus/CAE在选定位置显示一个临时datum坐标系。datum坐标系的轴与全局坐标系的轴对齐。 2. 从提示区域单击**Create Datum**接受临时datum坐标系的默认方向，或选择（或输入坐标）一个将位于*X*或*R*轴上的点。Abaqus/CAE重新计算临时datum坐标系的方向。 3. 从提示区域单击**Create Datum**接受临时datum坐标系的新方向，或选择（或输入坐标）一个将位于*X*–*Y*或*R*–![](../graphics/usi_eqn00065.gif)平面上的点。Abaqus/CAE创建datum坐标系。您无法移动或旋转datum坐标系。
