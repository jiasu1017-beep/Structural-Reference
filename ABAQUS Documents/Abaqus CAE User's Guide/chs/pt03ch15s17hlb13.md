# 15.17.13 定义时间积分

您可以在Abaqus/Explicit分析中为弹性、阻尼和摩擦指定隐式或显式时间积分。默认情况下，使用隐式时间积分。您只能定义一个积分选项。有关更多信息，请参见["连接器行为，"Abaqus分析用户指南第31.2.1节](../usb/usb-link.md#usb-elm-econnectorbehavior)。

**要创建或编辑时间积分：**

1. 使用以下方法之一显示连接器截面编辑器：
   - 要创建新的连接器截面，请按照["创建连接器截面，"第15.12.11节"](pt03ch15s12s01.md)中概述的过程进行操作。
   - 要编辑现有的连接器截面，请从主菜单栏选择**连接器**![](../graphics/images/arrow.gif)**截面**![](../graphics/images/arrow.gif)**管理器**，从出现的列表中选择连接器截面，然后点击**编辑**。
2. 在**编辑连接器截面**对话框中，执行以下操作之一：
   - 要定义新的积分选项，请点击**添加**并从出现的选项菜单中选择**积分**。
   - 要编辑现有的积分选项，请从**行为选项**列表中选择该选项以显示该选项的关联数据字段。
3. 点击**积分**字段右侧的箭头，并从出现的列表中选择一个选项：
   - 选择**隐式**以使用隐式时间积分。
   - 选择**显式**以使用显式时间积分。
4. 选择以下之一：
   - 如果您想继续定义行为选项，请点击**添加**，选择所需的行为选项，并继续连接器截面定义。有关定义其他行为选项的说明，请参见["使用连接器截面编辑器，"第15.17节"](pt03ch15s17.md)。
   - 如果您想查看或编辑现有的行为选项，请从**行为选项**列表中选择该选项。有关编辑行为选项的说明，请参见["使用连接器截面编辑器，"第15.17节"](pt03ch15s17.md)。
   - 如果您想保存连接器截面定义并退出编辑器，请点击**确定**。
