# 2.2.3 工具栏的组成





工具栏包含用于管理文件、过滤对象选择和查看模型的便捷工具集。工具栏中的项目是主菜单栏中可用功能的快捷方式。默认情况下，Abaqus/CAE会将所有工具栏显示在主菜单栏下方的一行中。根据您的显示分辨率和主窗口大小，Abaqus/CAE可能会将某些工具栏放置在第二行中。工具栏如下图所示：

![](../graphics/top-toolbar-nls.png)

您可以使用工具栏的握柄更改工具栏的位置，如上图所示。单击并拖动握柄可以将工具栏移动到主窗口周围。如果您将工具栏握柄释放到主窗口的四个可用停靠区域之一上（请参阅[图2-2](pt01ch02s02hlb03.md#usi-docking-regions)），Abaqus/CAE会"停靠"工具栏；停靠的工具栏没有标题栏，不会遮挡主窗口的其他部分。

**图2-2** 工具栏的可用停靠区域。

![](../graphics/usi-docking-regions-nls.png)

如果您将工具栏握柄释放到不在停靠区域附近的位置，Abaqus/CAE会创建一个带有标题栏的浮动工具栏。浮动工具栏会遮挡主窗口中的其他项目（请参阅[图2-3](pt01ch02s02hlb03.md#usi-floating-toolbar)）；但是，浮动工具栏可以定位在Abaqus/CAE主窗口之外。

**图2-3** 浮动工具栏。

![](../graphics/usi-floating-toolbar-nls.png)

在工具栏握柄上单击鼠标按钮3会显示一个菜单，允许您指定工具栏的位置和格式：
- 选择**顶部**可将工具栏停靠在顶部停靠区域。
- 选择**底部**可将工具栏停靠在底部停靠区域。
- 选择**左侧**可将工具栏停靠在左侧停靠区域。
- 选择**右侧**可将工具栏停靠在右侧停靠区域。
- 选择**浮动**可将停靠的工具栏变为浮动工具栏；此选项仅适用于停靠的工具栏。
- 选择**翻转**可将浮动工具栏的方向从水平变为垂直，反之亦然；此选项仅适用于浮动的工具栏。

您还可以隐藏工具栏并创建包含其他功能快捷方式的自定义工具栏。有关更多信息，请参阅[第61章，"自定义工具集"](pt06ch61.md)。

要获取工具栏中工具的简短描述，请将光标放在该工具上一会儿；会出现一个包含描述或"工具提示"的小框。要获取工具栏的名称，请将光标放在工具栏握柄上一会儿。

Abaqus/CAE工具栏包含以下功能：

**文件**

![](../graphics/top-filetools.png)

**文件**工具栏允许您创建、打开和保存模型数据库；打开输出数据库；打印视口；以及保存和加载会话对象和选项。有关更多信息，请参阅[第二部分，"使用Abaqus/CAE模型数据库、模型和文件"](pt02.md)”；[第8章，"打印视口"](pt01ch08.md)”；以及["管理会话对象和会话选项，" 第9.9节](pt02ch09s08.md)。

**视图操作**

![](../graphics/top-viwtools.png)

**视图操作**工具栏允许您指定模型或绘图的不同视图。例如，您可以使用这些工具平移、旋转或缩放模型或绘图。有关更多信息，请参阅[第5章，"操作视图和控制透视"](pt01ch05.md)。

**视图选项**

![](../graphics/top-pertools.png)

**视图选项**工具栏允许您指定是否为模型应用透视。有关更多信息，请参阅["控制透视，" 第5.5节](pt01ch05s05.md)。

**渲染样式**

![](../graphics/top-rstools.png)

**渲染样式**工具栏允许您指定使用线框、隐藏线还是阴影渲染样式来显示模型。在可视化模块中，**渲染样式**工具栏还包括填充渲染样式![](../graphics/ico_displayFilled.png)工具。有关更多信息，请参阅["选择渲染样式，" 第55.2.1节](pt05ch55s02hlb01.md)。

**可见对象**

![](../graphics/top-visobjtools.png)

**可见对象**工具栏允许您在Abaqus/CAE原生零件的几何体和同一零件的网格表示之间切换，打开和关闭种子显示，以及在网格表示和参考表示之间切换参考表示的显示（如果两者都存在）。有关更多信息，请参阅["显示原生网格，" 第17.3.11节"](pt03ch17s03s11.md)；["什么是网格种子？," 第17.4.1节"](pt03ch17s04s01.md)；以及["理解参考表示，" 第35.2节"](pt04ch35s02.md)。

**选择**

![](../graphics/obj-selection-toolbar-nls.png)

**选择**工具栏允许您通过切换箭头图标来启用或禁用对象选择。您可以使用箭头右侧的列表来限制可选择的对象类型。**选择**工具栏仅在视口中没有活动过程运行时才可用。有关更多信息，请参阅["选择对象后再选择过程，" 第6.3.7节"](pt01ch06s03s06.md)。

**查询**

![](../graphics/top-qrytools.png)

**查询**工具栏允许您获取有关模型几何体和特征的信息，探测模型和X-Y曲线的输出数据，以及对结果执行应力线性化。有关更多信息，请参阅[第71章，"查询工具集"](pt06ch71.md)”；[第51章，"探测模型"](pt05ch51.md)”；以及[第52章，"计算线性化应力"](pt05ch52.md)。

**显示组**

![](../graphics/top-dgtools.png)

**显示组**工具栏允许您有选择地绘制一个或多个模型或输出数据库项目。例如，您可以创建一个仅包含模型中指定集合元素的显示组。有关更多信息，请参阅[第78章，"使用显示组显示模型的子集"](pt07ch78.md)。

**颜色编码**

![](../graphics/top-ccdtools-nls.png)

**颜色编码**工具栏允许您自定义视口中项目的颜色并更改其半透明度。

对于颜色编码，您可以创建将独特颜色分配给显示不同元素的颜色映射。例如，当使用零件实例颜色映射时，模型中的每个零件实例将显示为不同的颜色。有关更多信息，请参阅[第77章，"几何体和网格元素的颜色编码"](pt07ch77.md)。

对于半透明度，您可以单击![](../graphics/ico_displayTranslucency.png)工具右侧的箭头显示滑块，您可以拖动滑块使显示颜色更透明或更不透明。有关更多信息，请参阅["更改半透明度，" 第77.3节"](pt07ch77s03.md)。

**场输出**

![](../graphics/viw-fieldout-toolbar-nls.png)

**场输出**工具栏允许您控制场输出变量显示的两个方面：
- 您可以选择要在当前视口中显示的场输出变量。选择包括场输出变量的类型（**主要**、**变形**或**符号**）、变量名称，以及可用时所选主要变量的不变量和分量。
- 对于变量类型的变化，您可以控制Abaqus/CAE是否自动将当前视口中的绘图状态与新的变量类型选择同步。如果![](../graphics/ico_synchLayers.png)工具被切换为开启，Abaqus/CAE会在新选择的场输出变量需要更改绘图状态时同步绘图状态；如果此选项被关闭，Abaqus/CAE仍会更新视口中显示的输出变量，但不会更改当前视口中的绘图状态。

工具栏中的选择是有限的，但如需要，![](../graphics/ico_fieldOutput.png)工具可以访问**场输出**对话框。有关工具栏中选项的更多信息，请参阅["使用场输出工具栏，" 第42.5.2节"](pt05ch42s03hlb02.md)。

**视口**

![](../graphics/top-vptools.png)

**视口**工具栏允许您创建和对齐视口，链接视口，以及创建视口注释。有关更多信息，请参阅["从视口工具栏管理视口和视口注释，" 第4.2.2节"](pt01ch04s02hlb02.md)。默认情况下不显示**视口**工具栏。

**视图剪切**

![](../graphics/top-cuttools.png)

**视图剪切**工具栏允许您在可视化模块以外的模块中切换视图剪切的显示，并自定义它们的定义和显示。有关更多信息，请参阅[第80章，"切割模型"](pt07ch80.md)。默认情况下显示**视图剪切**工具栏；在可视化模块中，视图剪切选项可在工具箱中获得。

**视图**

![](../graphics/top-vwstools.png)

**视图**工具栏允许您将自定义视图应用于视口中的模型。有关更多信息，请参阅["自定义视图，" 第5.2.8节"](pt01ch05s02s08.md)。默认情况下不显示**视图**工具栏。
