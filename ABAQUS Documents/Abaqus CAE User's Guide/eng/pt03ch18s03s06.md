# 18.3.6 About stop conditions







A global stop condition defines the maximum number of iterations an optimization should perform. A local stop condition specifies that the optimization should end when a local minimum (or maximum) has been reached. For more information, see ["Creating local stop conditions," Section 18.11](pt03ch18s11.md), and ["Objectives and constraints," Section 13.2.2 of the Abaqus Analysis User's Guide](../usb/usb-link.md#usb-anl-aoptobjectives).
