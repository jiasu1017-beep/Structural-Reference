# *MOLECULAR WEIGHT







### *MOLECULAR WEIGHTDefine the molecular weight of an ideal gas species.

This option is used to define the molecular weight of an ideal gas species. It can be used only in conjunction with the [*FLUID BEHAVIOR](ch06abk16.md) option.
