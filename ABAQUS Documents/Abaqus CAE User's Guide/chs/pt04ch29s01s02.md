# 29.1.2 关于离散紧固件





离散紧固件利用连接线在Abaqus/Standard和Abaqus/Explicit中的所选面之间创建连接器和耦合。[图29-3](pt04ch29s01s02.md#eng-fasteners-discretebeams-nls)显示了一个使用连接线和连接器（**Beam**连接类型）跨三个表面建模点焊的离散紧固件。用户首先在点焊位置创建了连接线。连接线用于定义离散紧固件的位置。

**图29-3** 使用离散紧固件跨三个表面建模的点焊。

![](../graphics/eng-fasteners-discretebeams-nls.png)

创建连接线的过程与创建基于点的紧固件的过程相似。您选择一个起点，并指定Abaqus/CAE将使用两种方法中的哪一种将点投影到最近的面。[图29-4](pt04ch29s01s02.md#eng-fasteners-discrete-nls)显示了用于定义离散紧固件的连接线的两种创建方法。

**图29-4** 创建用于定义离散紧固件的连接线。

![](../graphics/eng-fasteners-discrete-nls.png)

Abaqus/CAE沿着最近面的法线投影连接线。您可以使用以下两种方法确定连接线连接的面数：
- 指定投影或层的数量。
- 指定投影的最大长度。

对于每个连接线，Abaqus/CAE确定紧固点，并在每个紧固点及其对应表面之间应用分布耦合。连接线分配连接器截面后，Abaqus/CAE创建一个连接器，离散紧固件被认为是完全定义的。您可以通过在Part模块和Property模块中指定曲线细化级别来控制连接线在部件实例的面片表示上的位置精度。更多信息，请参见["控制曲线细化"，第76.4节](pt07ch76hla03.md)。

如果您的模型很复杂，Abaqus/CAE可以创建连接多个表面的连接线链，手动创建会很耗时。与基于点的紧固件不同，您可以在Visualization模块外部查看连接线及其连接器和耦合。

如果两个表面由两个连接线使用并共享一个公共面，则在您提交模型进行分析时，Abaqus/CAE会将这两个面合并为一个面。这会带来更好的Abaqus/Standard或Abaqus/Explicit性能，特别是当紧固件连接经过细化的孤立网格的面节点时。

当您提交包含具有离散紧固件的模型的作业进行分析时，Abaqus/CAE会将特殊注释行写入输入文件。这些特殊注释行被Abaqus求解器忽略，允许Abaqus/CAE在重新导入Abaqus/CAE时重新创建完全定义的离散紧固件。更多信息，请参见["从Abaqus输入文件导入模型"中的"导入相互作用、约束和紧固件"，第10.5.2节](pt02ch10s05s02.md#usi-imp-inp-int)。

创建离散紧固件的示例在["带点焊的柱屈曲"，Abaqus Example Problems Guide第1.2.3节](../exa/exa-link.md#exa-sta-bucklespotweld)中包含的Python脚本中进行了说明。
