# 78.2.3 复制、重命名和删除显示组





要复制、重命名或删除显示组，请使用以下任一方法：
- 列出在主菜单栏上****工具**![](../graphics/images/arrow.gif)**显示组****菜单下的**复制**、**重命名**和**删除**项目。**复制**、**重命名**和**删除**项目包含子菜单，列出您在当前模块同类型模块的会话期间保存的所有显示组。
- 当前模块的显示组管理器对话框。显示组管理器对话框包含与主菜单栏上****工具**![](../graphics/images/arrow.gif)**显示组****菜单下列出的相同功能，但带有方便的浏览器，列出当前模块和会话的所有显示组。要显示当前模块的显示组管理器对话框，请从主菜单栏中选择****工具**![](../graphics/images/arrow.gif)**显示组**![](../graphics/images/arrow.gif)**管理器****。

**注意：**如果您尝试删除在多个视口中显示的显示组，Abaqus/CAE会警告这些视口中的显示组将重置为整个模型。
