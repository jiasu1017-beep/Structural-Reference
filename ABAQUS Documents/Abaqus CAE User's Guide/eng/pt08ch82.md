# 82 Abaqus plug-ins







This section describes the plug-ins that are provided with Abaqus/CAE and appear in the **Plug-ins** menu. Select ****Plug-ins**![](../graphics/images/arrow.gif)**About Plug-ins**** to display a **View** button that displays help on each plug-in. The following plug-ins are described:
- ["Running the Getting Started with Abaqus examples," Section 82.1](pt08ch82s01.md)
- ["Creating a GUI plug-in," Section 82.2](pt08ch82s02.md)
- ["Creating a kernel plug-in," Section 82.3](pt08ch82s03.md)
- ["Creating dialog boxes with the **Really Simple GUI (RSG) Dialog Builder**," Section 82.4](pt08ch82s04.md)
- ["Upgrading a script," Section 82.5](pt08ch82s05.md)
- ["Displaying modal contribution factors," Section 82.6](pt08ch82s06.md)
- ["Displaying the history of the adaptive remeshing error indicator," Section 82.7](pt08ch82s07.md)
- ["Generating a report of postprocessing results," Section 82.8](pt08ch82s08.md)
- ["Exchanging data between Abaqus/CAE and Microsoft Excel," Section 82.9](pt08ch82s09.md)
- ["Importing a model from a file in stereolithography (STL) format," Section 82.10](pt08ch82s10.md)
- ["Exporting a part or assembly in stereolithography (STL) format," Section 82.11](pt08ch82s11.md)
- ["Plotting amplitude data," Section 82.12](pt08ch82s12.md)
- ["Combining data from multiple output databases," Section 82.13](pt08ch82s13.md)
- ["Finding the nearest node to a point," Section 82.14](pt08ch82s14.md)
- ["Finding the average temperature of a set of elements," Section 82.15](pt08ch82s15.md)
- ["Creating a planar constraint," Section 82.16](pt08ch82s16.md)

**Note:**All of the Abaqus plug-ins are stored in `*abaqus_dir*\code\python\lib\abaqus_plugins`, where *abaqus_dir* is the Abaqus parent directory.
