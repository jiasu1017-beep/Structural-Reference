# 54.7 格式化报告值





您可以指定要呈现在表格数据报告中的有效数字位数(对于X-Y数据报告、场输出报告和探测值报告)或小数位数(对于自由体切割报告)。对于X-Y数据报告、场输出报告和探测值报告,默认有效数字位数是6;对于自由体切割报告,默认小数位数是3。更多位数提供更高的精度但会占用更多空间并增加表格的宽度;所使用的有效数字或小数位数决定表中每列的宽度。此外,您可以为数据值选择以下数字格式之一:

**Automatic**

非常大的值和非常小的值以科学计数法表示。所有剩余值以无指数的形式表示,并使用指定的有效数字位数。此格式可用于X-Y数据报告、场输出报告和探测值报告。

**Engineering**

大于或等于1000(或小于或等于0.001)的值表示为1到999之间的数字乘以10的n次方,其中n是3的倍数(例如,`20.5E+03` 或 `17.76E+06`)。所有剩余值使用指定的有效数字位数表示。

**Fixed**

所有值都以不使用指数值的形式表示。当您选择此格式时,数字501,000在图例中显示为`501000.00`。此数字格式仅适用于自由体切割报告,如果您更改"**Decimal places**"选项,图例值可能会以不同方式显示。

**Scientific**

所有值表示为1到10之间的数字乘以适当的10的幂(例如,`2.05E+04` 或 `1.776E+07`)。

默认格式对于X-Y数据报告、场输出报告和探测值报告是"**Engineering**",对于自由体切割报告是"**Scientific**"。

**格式化报告值的步骤:**

1. 找到"**Output Format**"选项。 **对于X-Y数据或场输出的报告:** 从主菜单栏,选择 ****Report**![](../graphics/images/arrow.gif)**XY**** 或 ****Report**![](../graphics/images/arrow.gif)**Field Output****;然后在出现的对话框中点击"**Setup**"选项卡。"**Output Format**"选项在页面中间。 **对于自由体切割的报告:** 从主菜单栏,选择 ****Report**![](../graphics/images/arrow.gif)**Free Body Cut****。"**Output Format**"选项在页面中间。 **对于探测值的报告:** 从主菜单栏,选择 ****Tools**![](../graphics/images/arrow.gif)**Query****,然后在出现的对话框中点击"**Probe values**"。出现"**Probe Values**"对话框。将光标移动到当前视口中探测模型或X-Y曲线图,然后点击鼠标按钮1将感兴趣的值存储到对话框中。完成后,点击"**Write to File**"。出现"**Report Probe Values**"对话框;"**Output Format**"选项在对话框中间。
2. 指定报告中数值的精度级别: - 对于X-Y、场输出或探测值报告,在"**Number of significant digits**"文本字段中指定此值。 - 对于自由体切割报告,在"**Decimal places**"文本字段中输入此值。您也可以点击适用字段旁边的箭头,直到出现所需的数字。
3. 点击"**Number Format**"按钮,从出现的列表中选择您想要的数字格式。

要生成报告,请在对话框中配置剩余选项;然后点击"**Apply**"(对于X-Y、场输出或自由体切割报告)或"**OK**"(对于探测值报告)。完成后,点击"**Cancel**"关闭对话框。
