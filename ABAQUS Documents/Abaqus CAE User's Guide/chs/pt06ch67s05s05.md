# 67.5.5 自定义自由体切割中标签的显示







点击 **Labels**（标签）选项卡来自定义在自由体切割中与向量一起显示的数值标签的显示。您可以分别自定义力和力矩向量标签的外观；**Free Body Plot Options**（自由体绘图选项）对话框为 **Force**（力）和 **Moment**（力矩）向量标签包含相同的（但独立的）选项。

您可以开启和关闭数值标签的显示，并且可以为这些标签自定义以下设置：
- 标签的文本颜色。
- 标签的字体。
- 标签的格式（科学记数法、固定或工程）。
- 标签值显示的小数位数。

此外，您可以为标签显示设置阈值。如果向量值小于阈值，Abaqus/CAE 会隐藏其标签。

**自定义自由体切割中标签的显示：**

1. 找到 **Labels**（标签）选项。在可视化模块中从主菜单栏中选择 ****Options**![](../graphics/images/arrow.gif)**Free Body****（选项→自由体），然后在出现的对话框中点击 **Labels**（标签）选项卡。
2. 分别点击 **Force**（力）选项卡或 **Moment**（力矩）选项卡来自定义力向量或力矩向量的标签显示。
3. 切换开启 **Display value next to vector symbol**（在向量符号旁边显示值）以为所选类型的向量显示向量标签。
4. 为所选类型的向量选择向量标签的颜色。 1. 点击 **Text color**（文本颜色）颜色样本 ![](../graphics/usi-colorsample.png)。Abaqus/CAE 显示 **Select Color**（选择颜色）对话框。 2. 使用 **Select Color**（选择颜色）对话框中的方法之一来选择新颜色。 有关更多信息，请参阅["自定义颜色，" 第 3.2.9 节"](pt01ch03s02s09.md)。 3. 点击 **OK**（确定）关闭 **Select Color**（选择颜色）对话框。
5. 为所选类型的向量选择向量标签的字体。 1. 点击 **Set Font**（设置字体）。Abaqus/CAE 显示 **Select Font**（选择字体）对话框。 2. 使用 **Select Font**（选择字体）对话框中的方法之一来选择新字体、大小和样式。 有关更多信息，请参阅["自定义字体，" 第 3.2.8 节"](pt01ch03s02s08.md)。 3. 点击 **OK**（确定）关闭 **Select Font**（选择字体）对话框。
6. 要更改所选向量类型的向量标签的格式，请点击 **Number Format**（数字格式）箭头并选择以下格式之一： - 选择 **Scientific**（科学记数法）以科学记数法显示图例值，即将图例值表示为 1 到 10 之间的十进制值乘以 10 的幂。选择此格式时，数字 501,000 在图例中显示为 `5.01e5`。 - 选择 **Fixed**（固定）以不使用指数值的方式显示图例值。选择此格式时，数字 501,000 在图例中显示为 `501000.00`。 - 选择 **Engineering**（工程）以选择工程记数法，它以类似于科学记数法的格式表示值，但只允许是三的倍数的指数值。选择此格式时，数字 501,000 在图例中显示为 `501E3`。
7. 点击 **Decimal places**（小数位）箭头来指定每个标签所需的小数位数。
8. 为所选类型的向量标签指定 **Threshold**（阈值）值。值小于此阈值的向量将在视口中隐藏其标签。向量标签显示的默认阈值是 `1E-006`。
9. 点击 **OK**（确定）以实施您的更改并关闭 **Free Body Plot Options**（自由体绘图选项）对话框。默认情况下，您的更改会在会话期间保存，并将影响该视口中所有后续的自由体切割图。如果您想在后续会话中保留您的更改，请将它们保存到文件。有关更多信息，请参阅["保存自定义设置以供后续会话使用，" 第 55.1.1 节"](pt05ch55s01s01.md)。
