# 20.4.4 相对于草图重新对齐网格







当您草绘特征时，草图网格并不总是与草图的顶点和线或底层参考几何对齐。您可以使用以下技术重新对齐网格：
- 通过选择代表网格原点的顶点来相对于草图移动网格。
- 通过选择将与草图 *X* 轴平行的线来相对于草图旋转网格。
- 移除任何重新对齐并将网格重置为其原始方向。

重新对齐仅适用于特定草图，网格的对齐方式与草图一起存储在模型数据库中。新的和现有的草图保留其默认对齐方式。

更改原点和旋转草图网格允许您快速创建使用默认网格位置可能难以创建的特征。旋转网格具有以下效果：
- 当前光标位置的网格坐标显示在视口左上角，您可以使用这些坐标作为指导。显示的网格坐标相对于草图网格的当前对齐方式，如果重新对齐网格，这些坐标会发生变化。
- 当前光标位置的草图坐标显示在视口右上角。显示的草图坐标相对于草图网格的原始对齐方式，如果重新对齐网格，这些坐标不会发生变化。
- 当您在提示区域输入顶点的坐标时，这些坐标相对于草图网格的当前对齐方式。
- 旧的水平和垂直尺寸保持水平和垂直。
- 旧的水平和垂直构造线保持水平和垂直。
- 新的水平和垂直构造线以及预选点与旋转后的网格对齐。
- 矩形工具生成的线与旋转后的网格对齐。
