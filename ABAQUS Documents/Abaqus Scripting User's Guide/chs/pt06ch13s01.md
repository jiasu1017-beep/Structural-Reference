# 13.1 主窗口概述





交互式Abaqus产品由一个包含多个GUI基础设施组件的主窗口组成。主窗口本身只提供GUI基础设施支持。你通过向主窗口注册模块和工具集来向应用程序添加特定功能。向主窗口注册模块和工具集的详细讨论见["模块和工具集，"第14.1节"](pt06ch14s01.md)。

主窗口设计为与GUI模块的概念协同工作，GUI模块包含自己的菜单栏、工具栏和工具箱条目。主窗口一次只显示一个模块的组件。主窗口负责在用户访问应用程序的各个模块时将这些组件换入和换出。

以下语句显示用于创建主窗口的构造函数：

```
AFXMainWindow(app, title, icon=None, miniIcon=None, 
    opts=DECOR_ALL, x=0, y=0, w=0, h=0)
```
以下列表描述了`AFXMainWindow`构造函数的参数：

**app**

应用程序对象。

**title**

将显示在主窗口标题栏中的字符串。

**icon**

用于桌面上应用程序的32×32像素图标。

**miniIcon**

在Windows上用于标题栏和系统托盘中应用程序的16×16像素图标。

**opts**

控制各种窗口行为的标志。

**x,y,w,h**

窗口的*X-*、*Y-*位置，以及窗口的*宽度*和*高度*。默认值为零表示系统应自动计算这些数字。当应用程序退出时，主窗口大小和位置存储在`abaqus_v6.14.gpr`中，以便当再次启动应用程序时，它将出现在相同位置并具有相同大小。因此，建议你不要在主窗口构造函数中设置*x、y、w或h*；但是，如果你设置了这些值，它们将覆盖`abaqus_v6.14.gpr`中的设置。

以下语句显示如何访问主窗口：

```
mainWindow = getAFXApp().getAFXMainWindow()
```
