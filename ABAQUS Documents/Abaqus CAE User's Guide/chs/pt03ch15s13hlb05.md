# 15.13.5 为通用接触指定表面属性分配





通用接触相互作用的属性（如表面属性）在相互作用编辑器的「Attribute Assignments」部分独立于接触域指定。有关通用接触的简要概述，请参见「理解相互作用」，第15.3节。更详细的讨论，请参见「Abaqus/Standard中通用接触的表面属性」，Abaqus Analysis User's Guide第36.2.2节，以及「在Abaqus/Explicit中为通用接触分配表面属性」，Abaqus Analysis User's Guide第36.4.2节。您可以在通用接触相互作用中指定以下表面属性：
- **表面厚度分配**。为壳或膜表面分配非默认表面厚度。
- **壳/膜偏移分配**。分配表面偏移以定义从中面到参考面的距离（以表面厚度的分数表示）。
- **表面平滑分配**。为模型表面分配非默认接触平滑。接触平滑提高了弯曲表面的接触应力和压力精度（请参见「Abaqus/Standard中平滑接触表面」，Abaqus Analysis User's Guide第38.1.3节）。
- **特征边准则分配**。控制要包含在通用接触域中的特征边。模型的特征边包括壳周缘边缘和几何特征边。不允许对Abaqus/Standard分析进行边对边接触建模。

**指定表面厚度分配的步骤：**

1. 使用以下方法之一显示通用接触相互作用编辑器：- 要创建新的通用接触相互作用，请按照「定义通用接触」，第15.13.1节中的说明操作。- 要编辑现有的通用接触相互作用，请从主菜单栏选择 ****Interaction**![](../graphics/images/arrow.gif)**Edit**![](../graphics/images/arrow.gif)***相互作用名称*****。
2. 点击相互作用编辑器「Attribute Assignments」部分中的「Surface Properties」选项卡。表面厚度只能修改用于壳和膜上定义的表面。
3. 点击「Surface thickness assignments」旁边的![](../graphics/ico_edit.png)。此时会出现「Edit Surface Thickness Assignments」对话框。默认情况下，当您从列表或表中选择一个表面时，Abaqus/CAE会在视口中高亮显示该表面；但是，高亮不适用于「(Global)」。您可以关闭对话框底部的「Highlight selected regions」以关闭选择高亮。
4. 从左侧列表的现有表面列表中选择一个或多个表面。选择「(Global)」将壳/膜厚度分配给整个接触域。**提示：**您可以点击![](../graphics/ico_surface.png)定义新表面并将其添加到列表中。请参见「创建表面」，第73.3.3节，获取定义表面的说明。
5. 点击对话框中间的箭头![](../graphics/usi-itn-contactattribarrow.png)将您的选择转移到壳/膜厚度分配列表中。对话框右侧的表会更新以反映您的选择。
6. 在「Surface Thickness Assignments」表中间一列中指定每个表面的厚度。- 输入`ORIGINAL`将壳/膜厚度设置为原始壳或膜厚度（默认）；- 输入`THINNING`将壳/膜厚度设置为当前壳或膜厚度（此选项仅在Abaqus/Explicit中可用）；或- 指定壳/膜厚度的值。
7. （可选）在「Surface Thickness Assignments」表最后一列中为任何壳/膜厚度分配指定比例因子。
8. 根据需要重复上述步骤以完成壳/膜厚度分配。如果您想删除厚度分配，请选择行并点击![](../graphics/ico_delete.png)。**注意：**分配顺序可能是相关的；当壳/膜厚度分配重叠时，最后的分配将优先。
9. 点击「OK」保存您的选择并关闭「Edit Surface Thickness Assignments」对话框。相互作用编辑器重新出现，并更新了壳/膜厚度分配数量的信息。
10. 点击「OK」创建相互作用并关闭编辑器。

**指定壳/膜偏移分配的步骤：**

1. 使用以下方法之一显示通用接触相互作用编辑器：- 要创建新的通用接触相互作用，请按照「定义通用接触」，第15.13.1节中的说明操作。- 要编辑现有的通用接触相互作用，请从主菜单栏选择 ****Interaction**![](../graphics/images/arrow.gif)**Edit**![](../graphics/images/arrow.gif)***相互作用名称*****。
2. 点击相互作用编辑器「Attribute Assignments」部分中的「Surface Properties」选项卡。表面偏移只能修改用于壳和膜上定义的表面。
3. 点击「Shell/Membrane offset assignments」旁边的![](../graphics/ico_edit.png)。此时会出现「Edit Shell/Membrane Offset Assignments」对话框。默认情况下，当您从列表或表中选择一个表面时，Abaqus/CAE会在视口中高亮显示该表面；但是，高亮不适用于「(Global)」。您可以关闭对话框底部的「Highlight selected regions」以关闭选择高亮。
4. 从左侧列表的现有表面列表中选择一个或多个表面。选择「(Global)」将壳/膜偏移分配给整个接触域。**提示：**您可以点击![](../graphics/ico_surface.png)定义新表面并将其添加到列表中。请参见「创建表面」，第73.3.3节，获取定义表面的说明。
5. 点击对话框中间的箭头![](../graphics/usi-itn-contactattribarrow.png)将您的选择转移到壳/膜偏移分配列表中。对话框右侧的表会更新以反映您的选择。
6. 在「Shell/Membrane Offset Assignments」表第二列中指定每个表面的偏移分数。偏移分数定义从中面到参考面的距离（以厚度的分数表示）。- 输入`ORIGINAL`将壳/膜偏移设置为原始壳或膜偏移（默认）。- 输入`SPOS`将壳/膜的顶面指定为参考面。- 输入`SNEG`将壳/膜的底面指定为参考面。- 为偏移分数指定介于-0.5（表示壳/膜的底面）和0.5（表示壳/膜的顶面）之间的值。
7. 根据需要重复上述步骤以完成壳/膜偏移分配。如果您想删除偏移分配，请选择行并点击![](../graphics/ico_delete.png)。**注意：**分配顺序可能是相关的；当壳/膜偏移分配重叠时，最后的分配将优先。
8. 点击「OK」保存您的选择并关闭「Edit Shell/Membrane Offset Assignments」对话框。相互作用编辑器重新出现，并更新了壳/膜偏移分配数量的信息。
9. 点击「OK」创建相互作用并关闭编辑器。

**指定表面平滑分配的步骤：**

1. 使用以下方法之一显示通用接触相互作用编辑器：- 要创建新的通用接触相互作用，请按照「定义通用接触」，第15.13.1节中的说明操作。- 要编辑现有的通用接触相互作用，请从主菜单栏选择 ****Interaction**![](../graphics/images/arrow.gif)**Edit**![](../graphics/images/arrow.gif)***相互作用名称*****。在Abaqus/Standard通用接触相互作用中，表面平滑分配只能在初始步骤中指定。在Abaqus/Explicit通用接触相互作用中，表面平滑分配可以在任何分析步骤中指定或修改。
2. 点击相互作用编辑器「Attribute Assignments」部分中的「Surface Properties」选项卡。
3. 点击「Surface smoothing assignments」旁边的![](../graphics/ico_edit.png)。此时会出现「Edit Surface Smoothing Assignments」对话框。默认情况下，当您从列表或表中选择一个表面时，Abaqus/CAE会在视口中高亮显示该表面，并显示将用于平滑计算的检测到的曲率轴或曲率中心。您可以关闭对话框底部的「Highlight selected regions」以关闭选择高亮。
4. 切换「Automatically assign smoothing for geometric faces」以确定Abaqus/CAE是否自动将表面平滑应用于通用接触域中的所有适当表面。此选项默认情况下为Abaqus/Standard通用接触相互作用开启，为Abaqus/Explicit通用接触相互作用关闭。
5. 从左侧列表的现有表面列表中选择一个或多个表面。**提示：**您可以点击![](../graphics/ico_surface.png)定义新表面并将其添加到列表中。请参见「创建表面」，第73.3.3节，获取定义表面的说明。
6. 点击对话框中间的箭头![](../graphics/usi-itn-contactattribarrow.png)将您的选择转移到表面平滑分配列表中。对话框右侧的表会更新以反映您的选择。
7. 在「Surface Smoothing Assignments」表第二列中为每个表面指定要应用的平滑。为这些表面指定的平滑会覆盖应用的默认全局平滑（如果已应用）。- 选择「REVOLUTION」将圆周平滑应用于关于旋转轴对称的弯曲表面（或关于中心点对称的二维圆弧）。- 选择「SPHERICAL」将球面平滑应用于关于中心点对称的弯曲表面。- 选择「TOROIDAL」将环面平滑应用于关于旋转轴对称的弯曲表面。- 选择「NONE」防止将平滑应用于指定表面。
8. 根据需要重复上述步骤以完成表面平滑分配。如果您想删除平滑分配，请选择行并点击![](../graphics/ico_delete.png)。**注意：**分配顺序可能是相关的；当平滑分配重叠时，最后的分配将优先。
9. 点击「OK」保存您的选择并关闭「Edit Surface Smoothing Assignments」对话框。相互作用编辑器重新出现，并更新了表面平滑分配数量的信息。
10. 点击「OK」创建相互作用并关闭编辑器。

**指定特征边准则分配的步骤：**

1. 使用以下方法之一显示通用接触相互作用编辑器：- 要创建新的通用接触相互作用，请按照「定义通用接触」，第15.13.1节中的说明操作。- 要编辑现有的通用接触相互作用，请从主菜单栏选择 ****Interaction**![](../graphics/images/arrow.gif)**Edit**![](../graphics/images/arrow.gif)***相互作用名称*****。
2. 点击相互作用编辑器「Attribute Assignments」部分中的「Surface Properties」选项卡。
3. 点击「Feature edge criteria assignments」旁边的![](../graphics/ico_edit.png)。此时会出现「Edit Feature Edge Criteria Assignments」对话框。默认情况下，当您从列表或表中选择一个表面时，Abaqus/CAE会在视口中高亮显示该表面；但是，高亮不适用于「(Global)」。您可以关闭对话框底部的「Highlight selected regions」以关闭选择高亮。
4. 从左侧列表的现有表面列表中选择一个或多个表面。选择「(Global)」将表面特征分配给整个接触域。**提示：**您可以点击![](../graphics/ico_surface.png)定义新表面并将其添加到列表中。请参见「创建表面」，第73.3.3节，获取定义表面的说明。
5. 点击对话框中间的箭头![](../graphics/usi-itn-contactattribarrow.png)将您的选择转移到表面特征分配列表中。对话框右侧的表会更新以反映您的选择。
6. 在「Surface Feature Assignments」表第二列中通过执行以下操作之一为每个表面指定特征边准则：- 输入`PERIMETER`仅在通用接触域中包含周缘边缘。- 输入`ALL`在通用接触域中包含所有边缘（此选项仅适用于Abaqus/Explicit）。- 输入`PICKED`仅包含作为表面定义一部分明确选择的边缘（此选项仅适用于Abaqus/Explicit中的壳几何和单元）。- 输入`NONE`在通用接触域中不包含任何特征边缘。- 输入以度为单位的角度以在通用接触域中包含周缘边缘和特征角度大于或等于指定角度的几何边缘。指定值必须在0到180之间。
7. 根据需要重复上述步骤以完成表面特征分配。如果您想删除表面特征分配，请选择行并点击![](../graphics/ico_delete.png)。**注意：**分配顺序可能是相关的；当表面特征分配重叠时，最后的分配将优先。
8. 点击「OK」保存您的选择并关闭「Edit Feature Edge Criteria Assignments」对话框。相互作用编辑器重新出现，并更新了特征边准则分配数量的信息。
9. 点击「OK」创建相互作用并关闭编辑器。
