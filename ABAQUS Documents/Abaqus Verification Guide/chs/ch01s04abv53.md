# 1.4.5 内聚力元件载荷验证





**产品：**Abaqus/Standard  Abaqus/Explicit  

### 测试的元件

COH3D8、COH3D6、COH2D4、COHAX4、COH3D8P、COH3D6P、COH2D4P、COHAX4P。

### 测试的功能

在此验证问题中测试了以下功能：
- 基于元件的分布载荷。
- 基于表面的分布载荷。
- 使用预定义温度指定的热载荷。

### 问题描述

在此验证测试中，每个元件的所有节点都是固定的，载荷施加在节点上产生的反作用力用于验证元件载荷计算。此外，通过允许每个元件在厚度方向上随温度变化自由变形来验证热载荷的效果。厚度方向产生的热应变与解析结果进行比较。

**模型：**

- COH3D8、COH3D6、COH3D8P和COH3D6P：
  | 立方体尺寸 | 7×7×7 |
  | --- | --- |
  | 厚度 | 几何 |
  | 厚度方向 | 全局2 |
  | 响应 | 连续体 |
  | 旋转离心轴 | (1, 0, 0)，通过 (3.5, 1000, 3.5) |
  | 科里奥利旋转轴 | (1, 0, 0)，通过原点 |
  | 重力载荷向量 | (0, 1, 0) |
- COH2D4和COH2D4P：
  | 平面尺寸 | 7×7 |
  | --- | --- |
  | 厚度 | 几何 |
  | 厚度方向 | 全局2 |
  | 响应 | 连续体 |
  | 旋转离心轴 | (1, 0, 0)，通过 (3.5, 1000, 0.0) |
  | 科里奥利旋转轴 | (0, 0, 1)，通过原点 |
  | 重力载荷向量 | (0, 1, 0) |
- COHAX4和COHAX4P：
  | 平面尺寸 | 10×10 |
  | --- | --- |
  | 厚度 | 几何 |
  | 厚度方向 | 全局1；全局2 |
  | 响应 | 连续体 |
  | 旋转离心轴 | (0, 1, 0)，通过原点 |
  | 重力载荷向量 | (0, 1, 0) |

**材料：**

| 杨氏模量 | 3×10^6 |
| --- | --- |
| 泊松比 | 0.3 |
| 热膨胀系数 | 0.0001 |
| 密度 | 10.0 |

**初始条件：**

| 初始温度 | ALL, 0.0 |
| --- | --- |

### 结果与讨论

计算的反作用力与施加的载荷一致。此外，对于热载荷情况，厚度方向的热应力和热应变与解析结果一致。
