# 58.4.1 表达式场显示符号详情





网格上符号显示使用的点与分析过程中评估表达式场的位置一致。在几何体上，当符号沿模型边缘或表面均匀分布时，Abaqus/CAE 选择一组随机点用于符号显示。用于符号显示的点可能与分析过程中评估表达式场的位置一致，也可能不一致。

在某些情况下，由于错误（例如除以零），表达式场无法在给定点进行评估。当出现这种情况时，消息区域中会出现一条消息，指示评估表达式场时发生错误。作为视觉提示以指示存在错误，显示在视口中的符号不再缩放，即使基于表达式场值缩放符号的设置已打开。您可以尝试通过更改符号密度设置来消除评估错误，这会提示 Abaqus/CAE 选择一组不同的点用于符号显示。有关更改符号密度设置的更多信息，请参见["控制属性的显示，" 第76.15节](pt07ch76hla14.md)。
