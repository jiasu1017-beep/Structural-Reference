# 62.7.2 沿两个平面的交线创建datum轴







您可以通过选择两个不平行的平面来定位datum轴。Abaqus/CAE在两个平面（或两个平面的投影）相交处创建datum轴。

![](../graphics/dtm-haxn-nls.png)

**沿两个平面的交线创建datum轴：**

1. 从主菜单栏中，选择****Tools**![](../graphics/images/arrow.gif)**Datum****。此时将出现**Create Datum**对话框。该对话框概述了您可以创建的datum几何体类型。Abaqus/CAE在提示区域中显示提示以指导您完成此过程。**提示：**您也可以使用模块工具箱中的![](../graphics/ico_dtmAxisIntersectPlanes.png)工具创建datum轴。有关工具箱中datum工具的图示，请参见["使用Datum工具集，" 第62.2节](pt06ch62s02.md)。
2. 从对话框顶部的类型列表中，选择**Axis**。**Method**列表指示您可以使用的方法来创建datum轴。
3. 从**Method**列表中，选择**Intersection of 2 planes**。
4. 从当前视口中的零件或装配中，选择一个平面。
5. 选择与第一个不平行的第二个平面。datum轴会出现。您无法修改使用此方法创建的datum轴；您必须删除旧轴并创建一个新的。
