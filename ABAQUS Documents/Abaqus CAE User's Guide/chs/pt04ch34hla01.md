# 34.5 定义载荷工况





您使用Load模块中的**Load Case**菜单来定义载荷工况。您可以在静态扰动、直接稳态动态、基于SIM的稳态动态（基于模态和基于子空间）以及子结构生成分析中创建载荷工况。在创建载荷工况之前，您必须在Load模块中创建单独的载荷和边界条件。更多信息，请参见["创建载荷"，第16.8.1节](pt03ch16s08hlb01.md)和["创建边界条件"，第16.8.2节](pt03ch16s08hlb02.md)。

**要定义载荷工况：**

1. 从Load模块的主菜单栏中，选择****Load Case**![](../graphics/images/arrow.gif)**Create****。
   **提示：**您也可以使用Load模块工具箱中的![](../graphics/ico_loadCase.png)工具创建载荷工况。
2. 在出现的**Create Load Case**对话框中，执行以下操作：
   1. 命名载荷工况。有关命名对象的信息，请参见["使用基本对话框组件"，第3.2.1节](pt01ch03s02s01.md)。
   2. 选择步骤。您只能在以下步骤中定义载荷工况：静态、线性扰动；稳态动态、直接；基于SIM的稳态动态（模态和子空间）；和子结构生成。
   3. 单击**Continue**。载荷工况编辑器出现。
3. 如果要在载荷工况中包含载荷，请单击**Loads**标签页。使用以下方法之一指定载荷：
   - 单击页面底部的![](../graphics/ico_add.png)以显示所选步骤可用载荷的列表。
     1. 从**Load Selection**对话框中，从列表中选择载荷（有关更多信息，请参见["从列表和表中选择多个项目"，第3.2.11节](pt01ch03s02s11.md)）。所选载荷在视口中高亮显示。
     2. 在**Scale factor**字段中，输入非零值以缩放所选载荷的大小。输入负值以沿相反方向施加所选载荷。每个载荷的默认缩放因子为1.0。
     3. 单击**OK**保存您的选择并关闭**Load Selection**对话框。
   - 使用键盘在表中输入数据。
     1. 输入先前定义的载荷的名称。
     2. 输入非零值以缩放载荷的大小。输入负值以沿相反方向施加载荷。默认缩放因子为1.0。
     3. 重复前述步骤以指定要包含在载荷工况中的所有载荷。载荷工况编辑器显示载荷工况中包含的载荷和缩放因子。要在视口中高亮显示载荷，请单击表行标题以在编辑器中选择载荷（有关更多信息，请参见["从列表和表中选择多个项目"，第3.2.11节](pt01ch03s02s11.md)）并切换**Highlight selections in viewport**。抑制的载荷不会在视口中高亮显示。要从表中移除载荷，请单击表行标题以选择载荷，然后单击![](../graphics/ico_delete.png)。
4. 如果要在载荷工况中包含边界条件，请单击**Boundary Conditions**标签页以显示边界条件页面。默认情况下，Abaqus/CAE将包含从基态传播或修改的所有边界条件在您创建的每个载荷工况中。您可以关闭边界条件页面顶部的此行为。使用以下方法之一指定边界条件：
   - 单击页面底部的![](../graphics/ico_add.png)以显示所选步骤可用边界条件的列表。
     1. 从**Boundary Condition Selection**对话框中，从列表中选择边界条件（有关更多信息，请参见["从列表和表中选择多个项目"，第3.2.11节](pt01ch03s02s11.md)）。所选边界条件在视口中高亮显示。
     2. 在**Scale factor**字段中，输入非零值以缩放所选边界条件的大小。每个边界条件的默认缩放因子为1.0。
     3. 单击**OK**保存您的选择并关闭**Boundary Condition Selection**对话框。
   - 使用键盘在表中输入数据。
     1. 输入先前定义的边界条件的名称。
     2. 输入非零值以缩放边界条件的大小。默认缩放因子为1.0。
     3. 重复前述步骤以指定要包含在载荷工况中的所有边界条件。载荷工况编辑器显示载荷工况中包含的边界条件和缩放因子。缩放因子仅在边界条件具有非零值时相关。要在视口中高亮显示边界条件，请单击表行标题以在编辑器中选择边界条件（有关更多信息，请参见["从列表和表中选择多个项目"，第3.2.11节](pt01ch03s02s11.md)）并切换**Highlight selections in viewport**。抑制的边界条件不会在视口中高亮显示。要从表中移除边界条件，请单击表行标题以选择边界条件，然后单击![](../graphics/ico_delete.png)。
5. 单击**OK**保存您的载荷工况定义并关闭编辑器。
