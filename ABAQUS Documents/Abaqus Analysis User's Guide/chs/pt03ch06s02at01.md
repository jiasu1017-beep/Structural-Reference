### 概述

静态应力分析：
- 用于可以忽略惯性效应的情况；
- 可以是线性或非线性的；并且
- 忽略随时间变化的材料效应（蠕变、膨胀、粘弹性），但考虑率相关塑性和超弹性材料的滞后行为。

### 时间周期

在静态步骤中，您可以为分析分配一个时间周期。这对于振幅选项的交叉引用是必要的，振幅选项可用于确定步骤期间载荷和其他外部指定参数的变化（参见["幅值曲线，" 34.1.2节](pt07ch34s01aus115.md)）。在某些情况下，此时间尺度是相当真实的——例如，响应可能由基于先前瞬态热传递运行随时间变化的温度引起；或者材料响应可能与速率相关（率相关塑性），因此存在自然的时间尺度。其他情况没有这种自然的时间尺度；例如，当容器被加压至极限载荷而材料响应与速率无关时。如果您未指定时间周期，Abaqus/Standard默认为时间周期，其中"时间"在步骤中从0.0变化到1.0。"时间"增量只是步骤总周期的分数。

### 线性静态分析

线性静态分析涉及载荷工况和适当边界条件的指定。如果问题的全部或部分具有线性响应，子结构是降低大型分析计算成本的强大能力（参见["使用子结构，" 10.1.1节](pt04ch10s01aus58.md)）。

### 非线性静态分析

非线性可能来自大位移效应、材料非线性和/或边界非线性（如接触和摩擦）（参见["常规与线性扰动分析步骤，" 6.1.3节](pt03ch06s01aus44.md)），必须予以考虑。如果在步骤中预期几何非线性行为，应使用大位移formulation。在大多数非线性分析中，步骤期间的加载变化遵循规定的历史，如温度瞬态或指定位移。

| **输入文件用法：** | 使用以下选项指定静态步骤应使用大位移formulation： |
| --- | --- |
|  | ``` [*STEP](../key/key-link.md#usb-kws-hstep), NLGEOM ``` |

| **Abaqus/CAE用法：** | Step模块：**Create Step**：**General**：**Static, General**：**Basic**：**Nlgeom**：**On**（激活大位移formulation） |
| --- | --- |

#### 不稳定问题

一些静态问题可能天生不稳定，原因有多种。

##### 屈曲或坍塌

在某些几何非线性分析中，可能会发生屈曲或坍塌。在这些情况下，只有当载荷幅度不遵循规定历史时，才能获得准静态解；它必须是解的一部分。当加载可以被视为成比例的（整个结构上的加载可以用单个参数缩放）时，可以使用一种称为"修正Riks方法"的特殊方法，如["不稳定坍塌与后屈曲分析，" 6.2.4节](pt03ch06s02at03.md)中所述。

| **输入文件用法：** | ``` [*STATIC](../key/key-link.md#usb-kws-hstatic), RIKS ``` |
| --- | --- |

| **Abaqus/CAE用法：** | Step模块：**Create Step**：**General**：**Static, Riks** |
| --- | --- |

##### 局部不稳定

在其他不稳定分析中，不稳定是局部的（例如，表面起皱、材料不稳定或局部屈曲），在这种情况下，Riks方法等全局载荷控制方法不适用。Abaqus/Standard提供通过在整个模型中施加阻尼来稳定此类问题的选项，这样引入的粘性力足够大以防止即时屈曲或坍塌，但又足够小以在问题稳定时不会显著影响行为。可用的自动稳定化方案在["求解非线性问题，" 7.1.1节中的"不稳定问题的自动稳定化"](pt03ch07s01aus49.md#usb-anl-anonlineareqns-stabilize-over)中有详细描述。

#### 增量控制

Abaqus/Standard使用Newton法求解非线性平衡方程。许多问题涉及历史相关响应；因此，解通常作为一系列增量获得，并进行迭代以在每个增量内获得平衡。有时必须保持增量较小（从转动和应变增量必须较小的意义上），以确保正确建模历史相关效应。大多数情况下，增量大小的选择是计算效率的问题：如果增量太大，将需要更多迭代。此外，Newton法有有限的收敛半径；太大的增量可能无法获得任何解，因为初始状态离正在寻找的平衡状态太远——它在收敛半径之外。因此，对增量大小存在算法限制。

##### 自动增量控制

在大多数情况下，默认的自动增量控制方案是优选的，因为它将基于计算效率选择增量大小。

| **输入文件用法：** | ``` [*STATIC](../key/key-link.md#usb-kws-hstatic) ``` |
| --- | --- |

| **Abaqus/CAE用法：** | Step模块：**Create Step**：**General**：**Static, General**：**Incrementation**：**Type**：**Automatic**（默认） |
| --- | --- |

##### 直接增量控制

还提供直接用户控制增量大小，因为如果您对特定问题有相当多的经验，您可能能够选择更经济的方法。

| **输入文件用法：** | ``` [*STATIC](../key/key-link.md#usb-kws-hstatic), DIRECT ``` |
| --- | --- |

| **Abaqus/CAE用法：** | Step模块：**Create Step**：**General**：**Static, General**：**Incrementation**：**Type**：**Fixed** |
| --- | --- |

使用直接用户控制，在允许的最大迭代次数完成后（参见["常用控制参数，" 7.2.2节](pt03ch07s02aus50.md)中定义），可以接受增量的解，即使未满足平衡容差。不推荐此方法；仅在您对如何解释以这种方式获得的结果有透彻理解的特殊情况下才应使用。如果使用此选项，通常需要非常小的增量和至少两次迭代。

| **输入文件用法：** | ``` [*STATIC](../key/key-link.md#usb-kws-hstatic), DIRECT=NO STOP ``` |
| --- | --- |

| **Abaqus/CAE用法：** | Step模块：**Create Step**：**General**：**Static, General**：**Other**：**Accept solution after reaching maximum number of iterations** |
| --- | --- |

### 稳态摩擦滑动

在静态分析过程中，您可以通过将物体运动指定为预定义场来模拟两个可变形体之间或一个可变形体与一个以不同速度运动的刚体之间的稳态摩擦滑动。在这种情况下，假设滑移速度源于用户指定速度之间的差异，与节点位移无关，如["Coulomb摩擦，" Abaqus Theory Guide第5.2.3节](../stm/stm-link.md#stm-ifc-coulombfric)中所述。

由于此摩擦行为与不使用稳态摩擦滑动时的摩擦行为不同，在相对速度由预定义运动确定的分析步骤与先前步骤之间，解决方案可能出现不连续。一个例子是盘式制动器系统中盘片垫的初始预加载与后续制动分析（其中盘以指定转速旋转）之间发生的间断。为了确保解决方案的平滑过渡，建议在指定预定义运动的分析步骤之前的所有分析步骤使用零摩擦系数。然后您可以在稳态分析中修改摩擦属性以使用所需的摩擦系数（参见["摩擦行为，" 37.1.5节中的"在Abaqus/Standard分析期间更改摩擦属性"](pt09ch37s01aus169.md#usb-cni-afriction-change-std)）。

| **输入文件用法：** | ``` [*MOTION](../key/key-link.md#usb-kws-hmotion) ``` |
| --- | --- |

| **Abaqus/CAE用法：** | 预定义运动场在Abaqus/CAE中不支持。 |
| --- | --- |

### 初始条件

可以指定应力、温度、场变量、依赖求解的状态变量等的初始值。["Abaqus/Standard和Abaqus/Explicit中的初始条件，" 34.2.1节](pt07ch34s02aus116.md)描述了所有可用的初始条件。

### 边界条件

边界条件可以施加于任何位移或转动自由度（1-6）；开口截面梁单元的翘曲自由度7；或者，如果模型中包含静水压力流体单元，则施加于流体压力自由度8。如果边界条件施加于转动自由度，您必须了解Abaqus如何处理有限转动（参见["Abaqus/Standard和Abaqus/Explicit中的边界条件，" 34.3.1节](pt07ch34s03aus118.md)）。在分析期间，可以使用幅值定义来改变规定的边界条件（参见["幅值曲线，" 34.1.2节](pt07ch34s01aus115.md)）。

### 载荷

在静态应力分析中可以规定以下载荷：
- 集中节点力可以施加于位移自由度（1-6）；参见["集中载荷，" 34.4.2节](pt07ch34s04aus121.md)。
- 可以施加分布压力载荷或体力；参见["分布载荷，" 34.4.3节](pt07ch34s04aus122.md)。特定单元可用的分布载荷类型在[第六部分，"单元"](pt06.md)中描述。

### 预定义场

可以在静态应力分析中指定以下预定义场，如["预定义场，" 34.6.1节](pt07ch34s06aus128.md)中所述：
- 虽然温度在静态应力分析中不是自由度，但可以将节点温度指定为预定义场。如果为材料给出了热膨胀系数，则施加温度与初始温度之间的任何差异将导致热应变（["热膨胀，" 26.1.2节](pt05ch26s01abm52.md)）。指定温度也会影响温度依赖性材料属性（如果有）。
- 可以指定用户定义场变量的值。这些值仅影响场变量依赖性材料属性（如果有）。

### 材料选项

大多数描述机械行为的材料模型可用于静态应力分析。以下材料属性在静态应力分析期间不活跃：声学属性、热属性（热膨胀除外）、质量扩散属性、电导属性和孔隙流体流动属性。

率相关屈服（["率相关屈服，" 23.2.3节](pt05ch23s02abm19.md)）、滞后（["弹性体中的滞后，" 22.8.1节](pt05ch22s08abm14.md)）和双层粘塑性（["双层粘塑性，" 23.2.11节](pt05ch23s02abm27.md)）是静态分析期间活跃的唯一时间相关材料响应。率相关屈服响应在金属加工问题等快速过程中通常很重要。滞后模型在建模循环加载下表现出显著滞后的弹性体的大应变率相关响应时很有用。双层粘塑性模型在观察到显著的时间相关行为以及塑性时很有用，这对于金属通常发生在高温下。必须指定适当的时间尺度，以便Abaqus/Standard能够正确处理材料响应的率相关性。

静态蠕变和膨胀问题以及时域粘弹性模型通过准静态过程进行分析（["准静态分析，" 6.2.5节](pt03ch06s02at04.md)）。当这些时间相关材料模型中的任何一个用于静态分析时，获得与速率无关的弹性解，并且所选时间尺度对材料响应没有影响。对于蠕变和膨胀行为，这意味着加载是相对于蠕变效应发生的自然时间尺度瞬时施加的。

瞬时加载应用的相同概念适用于时域粘弹性行为。您还可以在静态过程中直接获得完全松弛的长期粘弹性解，而无需执行瞬态分析；此选择仅在定义时域粘弹性材料属性时有意义。如果请求长期粘弹性解，则与每个Prony系列项相关的内应力将从步骤开始时的值逐渐增加到步骤结束时的长期值。

对于双层粘塑性材料模型，您可以单独获得弹性-塑性网络的长期响应。

当定义频域粘弹性材料属性时（参见["频域粘弹性，" 22.7.2节](pt05ch22s07abm13.md)），相应的弹性模量必须指定为长期弹性模量。这意味着响应对应于长期弹性解，无论为步骤指定的时间周期如何。

| **输入文件用法：** | 使用以下选项获取时域粘弹性的完全松弛长期弹性解或双层粘塑性的长期弹性-塑性解： |
| --- | --- |
|  | ``` [*STATIC](../key/key-link.md#usb-kws-hstatic), LONG TERM ``` |

| **Abaqus/CAE用法：** | Step模块：**Create Step**：**General**：**Static, General**或**Static, Riks**：**Other**：**Obtain long-term solution with time-domain material properties** |
| --- | --- |

### 单元

Abaqus/Standard中任何应力/位移单元都可用于静态应力分析（参见["为分析类型选择适当的单元，" 27.1.3节](pt06ch27s01aus112.md))。虽然静态应力分析中速度不可用，但阻尼器仍然可以使用（它们可用于稳定不稳定问题）。相对速度将按["阻尼器，" 32.2.1节](pt06ch32s02alm38.md)中所述计算。

声学单元在静态步骤中不活跃。因此，如果声学-固体分析包含静态步骤，则只有固体单元会变形。如果变形很大，声学和固体网格可能不一致，后续声学-结构分析步骤可能产生误导性结果。有关使用自适应网格技术变形声学网格的信息，请参见["ALE自适应网格：概述，" 12.2.1节](pt04ch12s02abo14.md)。

### 输出

静态应力分析可用的单元输出包括应力；应变；能量；状态、场和用户定义变量的值；以及复合失效度量。可用的节点输出包括位移、反作用力和坐标。所有输出变量标识符在["Abaqus/Standard输出变量标识符，" 4.2.1节](pt02ch04s02abv01.md)中概述。
