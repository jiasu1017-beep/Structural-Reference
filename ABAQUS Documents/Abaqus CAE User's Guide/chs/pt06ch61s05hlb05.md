# 61.5.5 更改图标分配





从主菜单栏选择 ****Tools**![](../graphics/images/arrow.gif)**Customize**** 以更改与自定义工具栏上特定工具或功能关联的图标。

**更改图标分配的步骤：**

1. 在任何模块中，从主菜单栏选择 ****Tools**![](../graphics/images/arrow.gif)**Customize****。会出现 **Customize** 对话框。

2. 显示 **Functions** 选项卡页面。

3. 从 **Module/Toolset** 列表中，选择您的所需功能所属的 Abaqus/CAE 模块或工具集。Abaqus/CAE 使用所选模块或工具集中的可用功能填充 **Functions** 列表。

4. 从 **Functions** 列表中，选择您要更改其图标的功能。

5. 在对话框右下角的 **Icon** 字段中，单击 **Select**。会出现**选择图标**对话框。

6. 要使用现有的 Abaqus/CAE 图标，请从 **Abaqus Icons** 选项卡页面上的显示中选择图标图像，然后单击 **OK**。Abaqus/CAE 在所有自定义工具栏中更新所选功能的图标。

7. 要使用自定义图标：
   1. 显示 **User Icons** 选项卡页面。
   2. 要从 **User Icons** 选项卡页面的显示中移除自定义图标，请选择图标图像然后单击 **Delete**。
   3. 如果您想要使用的图标在显示中，请跳到步骤"e"。如果您的图标不在显示中，请单击 **Import** 导入图标图像。
   4. 在出现的**选择一个图标**对话框中，选择要用作图标的图像文件。有关可接受的图标图像文件的信息，请参见["配置自定义工具栏中的图标，" 第61.4节](pt06ch61s04.md)。有关使用**选择一个图标**对话框的信息，请参见["使用文件选择对话框，" 第3.2.10节](pt01ch03s02s10.md)。
   5. 从 **User Icons** 选项卡页面上的显示中选择图标图像，然后单击 **OK**。Abaqus/CAE 在所有自定义工具栏中更新所选功能的图标。
   **注意：**您无法修改默认 Abaqus/CAE 工具栏和工具箱中的图标。非默认图标分配仅适用于自定义工具栏中的工具。

8. 单击 **Dismiss** 关闭 **Customize** 对话框。
