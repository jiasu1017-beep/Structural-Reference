# 11.20 Adding a feature to a part







Use the **Shape** menu to add a feature to the current part. You can do the following:
- Use the **Solid** tools to add a solid feature to a to a three-dimensional solid part. For more information, see ["Adding a solid feature," Section 11.21](pt03ch11s21.md).
- Use the **Shell** tools to add a shell feature to a part. For more information, see ["Adding a shell feature," Section 11.22](pt03ch11s22.md).
- Use the **Wire** tools to add a wire feature to a part. For more information, see ["Adding a wire feature," Section 11.23](pt03ch11s23.md).
- Use the **Cut** tools to add a cut feature to a part. For more information, see ["Adding a cut feature," Section 11.24](pt03ch11s24.md).
- Use the **Blend** tools to add a blend feature to a three-dimensional solid part. For more information, see ["Blending edges," Section 11.27](pt03ch11s27.md).
- Use the **Transform** tool to mirror a part about a selected plane. For more information, see ["Mirroring a part," Section 11.28](pt03ch11s28.md).
