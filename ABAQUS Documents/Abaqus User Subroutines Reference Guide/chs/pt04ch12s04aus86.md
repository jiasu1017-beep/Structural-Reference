# 12.4.1 网格到网格的解映射







**产品：** Abaqus/Standard  

##### **参考文献**

- [*MAP SOLUTION](../key/key-link.md#usb-kws-mmapsolution)

### 概述

将解从一个网格映射到另一个网格是重新网格划分分析技术中的一个步骤，其中已从其原始配置显著变形的网格被更高质量的网格替换，分析继续进行。解映射技术：
- 当单元在分析期间变得严重变形以至于不再提供问题的良好离散化时使用；
- 将解从旧的变形网格映射到新网格，以便分析可以继续；和
- 只能与连续体单元一起使用。

有关将此方法与其他 Abaqus 自适应方法进行比较的高层讨论，请参阅 ["自适应技术，" 第 12.1.1 节"](pt04ch12s01aus77.md)。

###何时重新网格划分

Abaqus/Standard 使用拉格朗日公式：网格附着在材料上，因此随材料变形。当几何非线性分析中应变变得很大时，单元可能变得严重变形以至于不再提供问题的良好离散化。严重变形可能发生在橡胶弹性问题或塑性或粘塑性计算中，特别是在模拟制造过程时。当发生严重变形时，有必要重新网格划分：创建更好地设计用于继续分析的新网格，并将旧模型解映射到此网格上。

您必须决定何时需要重新网格划分。这一决定可以通过查看使用特定网格的分析阶段中发生的应变幅度来辅助，后文将讨论。当需要重新网格划分时，必须使用 Abaqus 中的网格生成能力或外部网格生成器为变形对象生成新网格。然后使用新网格作为新问题继续分析。在大多数情况下，需要将解从旧网格传输到新网格。

#### 解的不连续性

每当解从另一个网格映射时，由于网格的变化和解映射算法的结果，您可以预期解中会有一些不连续。如果不连续性显著，表明网格太粗糙或者重新网格划分应该在太变形之前更早的阶段完成。

重新网格划分技术工作良好，前提是网格对于问题足够精细，并且重新网格划分在单元变得太变形之前完成。

### 重新网格划分标准

重新网格划分的第一个要求是一些指示，表明网格正在变得扭曲的区域可能导致求解不准确。重新网格划分的一个可能标准是需要准确解析高应变梯度的区域中的极端单元变形。如果变形的单元已经移动到应变场变化均匀的区域，则不准确性不太重要；无论变形多么严重，单元都可以准确地表示恒定应变状态。然而，最终，重新网格划分的决定是一个判断问题。

### 生成新网格

一旦您确定当前网格不充分，必须通过使用 Abaqus 中的网格生成能力或外部网格生成器生成更适合问题当前状态的新网格。变形配置图可能有助于提供关于被建模对象当前形状的数据。通常可以从形成体表面的节点集在结果文件输出中定义用于网格生成器的外表面。见 ["油井中材料的侵蚀（出砂），" Abaqus 例题指南第 1.1.22 节"](../exa/exa-link.md#exa-sta-boreholeerosion) 和 ["圆柱坯料的镦粗：带有网格到网格解映射（Abaqus/Standard）和自适应网格划分（Abaqus/Explicit）的准静态分析，" Abaqus 例题指南第 1.3.1 节"](../exa/exa-link.md#exa-sta-rezonebillet)。

#### 重新网格划分接触问题

在接触区域，新网格必须与旧分析中的表面形状紧密一致。这一要求对于涉及两个可变形体之间接触的问题尤其重要；如果新网格定义的表面与旧分析中的表面即使略有不同，接触算法可能无法收敛。

### 指定要插值到新网格的解

通过将解从使用旧网格生成的输出数据库插值到新网格来继续模拟。

#### 指定必须读取解的时间

默认情况下，解传输将从有解变量可用的最后步骤和增量发生。或者，您可以指定将读取旧解的步骤和增量。

#### 平移和旋转旧作业网格

在执行映射之前，可以通过相对于全局原点给出平移和/或旋转来重新定位旧作业的网格。通过给出平移向量来指定平移。通过给出两个点来定义旋转轴加上绕该轴的右手角旋转来指定旋转。

| **输入文件用法：** | ``` [*MAP SOLUTION](../key/key-link.md#usb-kws-mmapsolution), STEP=*step*, INC=*increment* *translation vector data* *rotation axis and angular rotation data* ``` |

#### 识别旧作业

通过在运行 Abaqus 的命令中使用 **oldjob** 参数或通过响应命令过程的请求来指定将从中获取重启和结果数据的旧作业的名称（见 ["Abaqus/Standard、Abaqus/Explicit 和 Abaqus/CFD 执行，" 第 3.2.2 节"](pt01ch03s02abx02.md)）。旧作业所需的文件包括：重启文件（`.res`）、输出数据库（`.odb`）、模型数据库（`.mdl`）、状态数据库（`.stt`）和部件（`.prt`）文件。

### 解映射算法

解映射通过将结果从旧网格中的节点插值到新网格中的点（节点或积分点）来操作。因此，第一步涉及将解变量与旧网格中的节点关联。对于节点解变量（如节点温度或孔隙压力），关联已经完成。对于积分点变量，Abaqus 通过从积分点到每个单元的节点外推值，然后对围绕每个节点的所有相似单元平均这些值来获得旧网格节点处的解变量。

接下来，获得新网格中每个点相对于旧网格的位置。新网格点包括所有情况下的积分点，以及除了位移外还记录节点状态的过程中的节点（例如，耦合温度-位移过程中的节点温度）。

1. 找到点所在的（旧网格中的）单元，并获得该点在该单元中的位置。（此过程假定新网格中的所有点都在旧网格的边界内：如果不是这样，将发出警告消息，变量值设置为零。）
2. 然后将变量从旧单元的节点插值到新模型中的点。

所有必要的变量都以这种方式自动插值，以便可以使用新网格继续求解。

#### 解扩散

此算法在映射解中引入了一些扩散。扩散的影响随旧网格中的解梯度缩放；因此，即使在模型中网格从旧模型到新模型不变化的区域，当旧网格解梯度高时，映射导致的扩散也可能导致显著不同的映射量。您可以通过在解梯度高的区域细化旧网格或更早地重新网格划分来 moderating 此效果。

### 初始条件

从初始分析映射的解构成重新网格划分分析的初始条件。可以指定纯应力/位移分析的温度等初始条件。任何其他指定的初始条件将被忽略。

### 边界条件

边界条件不会从旧网格携带到新网格。在重新网格划分分析开始时施加的边界条件通常应该与在从初始分析选择的步骤和增量时有效的边界条件相同。虽然可以更改边界条件，但如果结构远离平衡状态，问题可能无法收敛。

在映射解分析中施加边界条件没有限制。边界条件可以以与在没有映射解的分析中相同的方式应用于所有可用自由度（见 ["Abaqus/Standard 和 Abaqus/Explicit 中的边界条件，" 第 34.3.1 节"](pt07ch34s03aus118.md)）。

### 载荷

在映射解分析中施加载荷没有限制。载荷可以以与在没有映射解的分析中相同的方式施加。

在重新网格划分分析开始时施加的载荷通常应该与初始分析结束时有效的载荷相同。虽然可以更改载荷，但如果结构远离平衡状态，问题可能无法收敛。

### 预定义场

温度和场变量从旧网格映射到新网格。如果在重新网格划分分析中更改了场变量的数量，则将传输两个分析共有的数量。预定义场可以以与在没有解映射的分析中修改它们相同的方式进行修改（见 ["预定义场，" 第 34.6.1 节"](pt07ch34s06aus128.md)）。

### 材料选项

Abaqus 中可用的任何机械本构模型都可以在映射解分析中使用（见 [第五部分，"材料"](pt05.md)"）。旧分析和新分析中的材料模型之间没有限制。解映射算法将传输两个模型共有的那些变量。您必须确保材料模型是兼容的。

### 单元

解映射能力只能与连续体单元一起使用（见 ["实体（连续体）单元，" 第 28.1.1 节"](pt06ch28s01alm01.md)）。

### 输出

映射解分析没有特定的输出。可以以与在没有映射解的分析中相同的方式请求输出。Abaqus 中可用的输出变量在 ["Abaqus/Standard 输出变量标识符，" 第 4.2.1 节"](pt02ch04s02abv01.md) 中列出。
