# 5.3 3D 罗盘





3D 罗盘是出现在视口右上角的视口标注（请参阅[图 5-16](pt01ch05s03.md#usi-3dcompass-over)）。

**图 5-16** 3D 罗盘。

![](../graphics/usi-3dcompass-over-nls.png)

Abaqus/CAE 中的 3D 罗盘基于 CATIA V5 中使用的 3D 罗盘。3D 罗盘指示模型在视口中的方向，类似于视图 triad。与视图 triad 不同，你可以通过点击和拖动来操作 3D 罗盘的方向。当你操作 3D 罗盘时，视口相机会平移或旋转以相应地更改视口方向。罗盘视图操作的行为与 CATIA V5 中的罗盘视图操作行为相同。

3D 罗盘是某些视图操作选项的有用快捷方式，因为它在所有模块和所有过程中都可用；你不需要进入视图操作模式即可使用 3D 罗盘更改视口方向。
