# 47.4.28 减小X–Y数据对象的样本大小







### 47.4.28 减小X–Y数据对象的样本大小

使用 **decimateFilter** 函数将大型的[之前保存的](pt05ch47s02hlb07.md) *X–Y* 数据对象（有序对的集合）缩减为更小的代表性样本。**decimateFilter** 函数是一种抗混叠滤波器，选择用户定义的采样频率的点以生成一个新的 *X–Y* 数据对象。

**decimateFilter** 函数需要两个参数：*X–Y* 数据对象的名称（*name*）和抽样因子（*decimationFactor*），即采样率要增加的因子。抽样因子必须是大于1的整数值。该函数还接受一个可选参数，该参数指定新采样率与 **decimateFilter** 函数在抽样前使用的截止频率的比率（*cutoffFactor*）。截止因子必须是2或更大的整数值。默认值为3。

**减小X–Y数据对象的样本大小：**

1. 找到 **Operate on XY Data** 对话框。从主菜单栏，选择 ****Tools**![](../graphics/images/arrow.gif)**XY Data**![](../graphics/images/arrow.gif)**Create****. 在出现的对话框中点击 **Operate on XY data**；然后点击 **Continue**。**Operate on XY Data** 对话框出现。
2. 从列出的 **Operators** 中，点击 **decimateFilter(X,I)**。**decimateFilter** 函数出现在表达式窗口中。
3. 从 **XY Data** 选项中，点击要操作的 *X–Y* 数据对象的名称，然后点击 **Add to Expression**。您可以从本次会话中之前保存的所有 *X–Y* 数据对象中选择（在 **XY Data** 字段中按字母顺序列出）。*X–Y* 数据对象名称出现在表达式窗口中 **decimateFilter** 函数的括号内。
4. 将光标定位在表达式窗口中第二个逗号之前，然后输入抽样因子的值。
5. 如果需要，将光标定位在表达式窗口中第三个逗号之前，然后输入截止因子的值。
6. 要评估并显示表达式，点击 **Plot Expression**。
7. 要[保存](pt05ch47s02hlb07.md)您的新 *X–Y* 数据对象，点击 **Save As**，然后在出现的对话框中提供名称。保存数据对象使其可用于本次会话中的未来操作以及包含多个数据对象的 *X–Y* 曲线图。
8. 完成后，点击 **Cancel** 关闭对话框。
