# 5.2.1 视图操作工具概述





相机位置、方向和缩放因子组合起来定义视口中对象的视图。你对装配体的视图以及每个部件的视图都相对于默认笛卡尔坐标系定位，此默认坐标系在视口中的方向由视图 triad 指示。默认情况下，当模块首次显示三维部件或装配体时使用等轴测视图。

你可以使用**视图操作**工具栏上的平移、旋转、放大、框选缩放和自动拟合工具来操作视图，以控制相机、相机目标和你正在查看的模型或结果之间的相对位置。例如，你可能想要平移和缩放轮廓图以查看应力集中区域。视图操作工具允许你执行以下操作：
- ![](../graphics/ico_viewPan.png) 水平或垂直移动视图；即[平移](pt01ch05s06hlb02.md)视图。
- ![](../graphics/ico_viewRotate.png) [旋转](pt01ch05s06hlb03.md)视图。
- ![](../graphics/ico_viewZoom.png) [放大](pt01ch05s06hlb04.md)或缩小视图。
- ![](../graphics/ico_viewZoomRectangle.png) [放大](pt01ch05s06hlb05.md)到视图的所选区域。
- ![](../graphics/ico_viewFit.png) [重新调整](pt01ch05s06hlb06.md)视图以填充视口；即自动拟合视图。
- ![](../graphics/ico_viewCycle.png) [循环](pt01ch05s06hlb07.md)浏览之前的视图。

使用 3D 罗盘可以执行其他类型的视图操作；有关更多信息，请参阅["3D 罗盘，" 第 5.3 节](pt01ch05s03.md)。

你可以单击鼠标右键 3 访问以下视图操作工具：
- **设置为旋转中心**：在鼠标点击位置设置旋转中心，可以在视口中的任何位置。
- **使用默认旋转中心**：清除先前设置的旋转中心。
- **居中视图**：在鼠标点击位置居中视图。

当视口中显示 *X–Y* 绘图时，你可以使用视图操作工具更改 *X–Y* 曲线的视图。由于 *X–Y* 绘图是二维的，当当前视口显示 *X–Y* 绘图时，旋转工具会被禁用。

单击视图操作工具会使你进入相应的视图操作模式。然后你通过将光标移动到该视口并根据需要进行拖动或点击来在特定视口中操作视图。此外，平移、旋转和放大工具具有替代模式，你可以通过在正常使用这些工具时按住 **[Shift]** 键来访问。这些工具的替代模式旨在用于电影相机模式，但也可以在默认模式中使用。有关相机术语和视图模式的更多信息，请参阅["理解相机模式和视图选项，" 第 5.1 节](pt01ch05s01.md)。要退出视图操作模式，请执行以下操作之一：
- 单击鼠标按钮 2。
- 单击提示区域中的取消按钮 ![](../graphics/afxI_cancel.png)。
- 再次单击视图操作工具。
- 单击任何其他视图操作工具。

你可以根据需要多次使用视图操作工具以达到所需的视图，你可以在任何视口中执行视图操作，而不管显示的是什么。Abaqus/CAE 存储每个视口最近的八个视图，你可以使用循环视图操作工具在这些视图之间向后和向前循环。

当你在链接到其他视口的视口中使用移动、旋转、放大、缩放或重新调整工具时，Abaqus/CAE 也会操作链接视口中的对象视图。有关更多信息，请参阅["链接视口以进行视图操作，" 第 4.5 节](pt01ch04s05.md)。

默认情况下，Abaqus/CAE 在你操作对象视图时使用当前渲染样式（线框、填充、消隐线或着色）显示图像。或者，你可以在**图形选项**对话框中更改**拖动模式**以在使用视图操作时将图像显示为简单线框；此模式允许在着色渲染样式下更快地操作大型模型。当你完成操作时，视图会恢复为原始渲染样式。

如果你更喜欢使用菜单而不是**视图操作**工具栏上的工具，你可以通过主菜单栏上的**视图**菜单访问所有视图操作工具。此外，你可以使用**视图**工具栏应用预定义和用户定义的视图，并且可以通过从主菜单栏中选择****View**![](../graphics/images/arrow.gif)**Specify****出现的对话框数值指定精确视图。有关自定义和数值指定视图的更多信息，请分别参阅["自定义视图，" 第 5.2.8 节](pt01ch05s02s08.md)和["数值指定视图，" 第 5.2.9 节](pt01ch05s02s09.md)。

或者，你可以通过使用键盘和鼠标组合操作进入三种视图操作模式之一。
- 要旋转视图，按 **[Ctrl]****+[Alt]**，并按住鼠标按钮 1。
- 要平移视图，按 **[Ctrl]****+[Alt]**，并按住鼠标按钮 2。
- 要放大或缩小视图，按 **[Ctrl]****+[Alt]**，并按住鼠标按钮 3。

在任何这些组合中添加 **[Shift]** 可以访问这些工具的替代模式。例如，按 **[Shift]****+[Ctrl]****+[Alt]** 并按住鼠标按钮 3 可以访问放大工具的替代模式，并将相机移近或远离视图中的对象。当你不在替代模式时，**[Shift]** 键对视图操作工具没有影响。使用前述操作之一进入视图操作模式后，只需释放鼠标按钮即可退出。

你可以通过从主菜单栏中选择****Tools**![](../graphics/images/arrow.gif)**Options****来重新配置这些键盘和鼠标组合，以模仿其他五种常见 CAD 应用程序使用的视图操作界面。有关更多信息，请参阅["使用视图操作快捷方式，" 第 68.2 节](pt06ch68s02.md)。
