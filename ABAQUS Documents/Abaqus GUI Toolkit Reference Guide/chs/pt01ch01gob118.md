### 全局标志

### **树形列表样式**

| **TREELIST_EXTENDEDSELECT** | 扩展选择模式允许拖动选择一系列项目。 |
| --- | --- |
| **TREELIST_SINGLESELECT** | 单选模式最多允许选择一个项目。 |
| **TREELIST_BROWSESELECT** | 浏览选择模式强制始终选择一个项目。 |
| **TREELIST_MULTIPLESELECT** | 多选模式用于选择单个项目。 |
| **TREELIST_AUTOSELECT** | 自动选择光标下的项目。 |
| **TREELIST_SHOWS_LINES** | 显示线条。 |
| **TREELIST_SHOWS_BOXES** | 显示展开框。 |
| **TREELIST_ROOT_BOXES** | 也显示根框。 |
| **TREELIST_CHECK_BOXES** | 显示复选框。 |
| **TREELIST_PROPAGATE_CHECKS** | 将选中状态传播到子项和父项。 |
