# 13.10.1 使用Instance菜单





### 13.10.1 使用**Instance**菜单

使用**Instance**菜单可以执行以下操作：
- [创建](pt03ch13s10hlb04.md)当前模型中零件的实例并将其添加到装配中。你也可以创建其他模型的实例以添加到当前装配中。
- 创建零件实例的[线性阵列](pt03ch13s10hlb05.md)。
- 创建零件实例的[径向阵列](pt03ch13s10hlb06.md)。
- [平移](pt03ch13s10hlb07.md)所选零件或模型实例沿指定矢量移动。
- [平移](pt03ch13s10hlb08.md)所选零件或模型实例沿指定矢量移动，直到它们与另一个实例的距离达到指定值。
- [旋转](pt03ch13s10hlb09.md)所选零件或模型实例沿指定轴旋转指定角度。
- [替换](pt03ch13s10hlb10.md)零件实例为第二个零件实例。
- [转换](pt03ch13s10hlb11.md)任何位置约束为绝对位置。
- [合并或切割](pt03ch13s10hlb12.md)零件实例。

你可能会发现使用装配模块工具箱访问实例工具更方便。有关装配工具箱中工具的图表，请参见["使用装配模块工具箱，"第13.9节](pt03ch13s09.md)。
