# 15.8.4 连接器导出分量和连接器势





您可以使用连接器导出分量和连接器势来定义连接器的复杂耦合行为。连接器导出分量是基于相对运动固有连接器分量函数的用户指定分量定义。您可以创建导出分量来指定连接器中作为连接器力和力矩复杂组合的摩擦产生法向力，或用作连接器势函数中的中间结果。

连接器势是相对运动固有分量或导出分量的用户定义数学函数。这些函数可以是二次的、椭圆的或最大范数。您可以使用连接器势来定义耦合摩擦、塑性和损伤连接器行为。

有关定义导出分量和势的详细说明，请参见「指定连接器导出分量」，第15.17.15节，以及「指定势项」，第15.17.16节。有关连接器函数的更多信息，请参见「耦合行为的连接器函数」，Abaqus Analysis User's Guide第31.2.4节。
