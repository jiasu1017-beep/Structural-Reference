# *CONNECTOR UNIAXIAL BEHAVIOR







### *CONNECTOR UNIAXIAL BEHAVIORDefine uniaxial behavior in connector elements.

This option is used to define uniaxial behavior in connector elements by specifying the loading and unloading response for the component of relative motion.

**Products: **Abaqus/Explicit  Abaqus/CAE  

**Type: **Model data  

**Level: **Model  

**Abaqus/CAE: **Interaction module

### **Required parameter: **

COMPONENT

Set this parameter equal to the connector's component of relative motion for which the uniaxial behavior is specified.

**There are no data lines associated with this option.**
