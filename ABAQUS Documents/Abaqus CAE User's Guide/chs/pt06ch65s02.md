# 65.2 使用模型树管理特征







模型树如图[图 65-2](pt06ch65s02.md#fts-snap4)所示，包含模型中所有可用特征的列表。您可以在模型树中的特征上点击鼠标按钮 3，并使用出现的菜单来管理特征。

**图 65-2** 模型树。

![](../graphics/adv-conn-featuremanager-nls.png)

您必须使用模型树来操作装配约束、已抑制的特征或失败的特征，因为它们在视口中不可见。您还可以同时选择多个特征进行删除、抑制或恢复。当您在模型树中选择特征时，它们会在视口中高亮显示（如果可见）。此外，任何特征的父特征或子特征都可以在视口中高亮显示。

模型树显示模型中每个特征的状态。每个特征名称前的图标表示状态：
- 黄色勾号表示特征已修改但尚未再生。如果部件被数据库升级锁定，所有活动特征都会显示此状态，直到部件解锁时再生。Abaqus/CAE 还会在您编辑但随后未再生的特征旁边显示黄色勾号。
- 挂锁表示部件或装配已被用户或数据库升级锁定。
- 红色"X"表示特征已被抑制。
- 红色"!"表示特征再生失败。部件名称旁边的红色"!"表示该部件无效。

要获取有关特征的更多信息，请在模型树中的特征上点击鼠标按钮 3，并从出现的菜单中选择 **Query**（查询）。信息显示在消息区域。![](../graphics/images/black4rule.gif) 有关相关信息，请点击以下任一项目：- ["使用特征操作工具集，" 第 65.1 节"](pt06ch65s01.md)
