# 17.19.2 获取网格信息





要获取网格信息，请从主菜单栏选择 ****Tools**![](../graphics/images/arrow.gif)**Query****。 您可以请求以下信息：
- 所选零件、零件实例或区域中的节点和单元总数，以及每种单元形状的单元数量。
- 所选单元的类型和连通性。
- 壳和膜面的正面和背面。
- 梁和桁架切线的方向。
- 网格堆栈方向。
- 边界面的边缘是否有不兼容的界面、裂纹或间隙，以及是否有边缘与其他面相交。
- 是否有自由或非流形外部单元边缘。
- 是否有未网格划分的区域。

**获取网格信息：**

1. 从上下文栏的**Object**字段中，选择零件或选择装配。
2. 从主菜单栏，选择 ****Tools**![](../graphics/images/arrow.gif)**Query****。 **提示：**您也可以选择**Query**工具栏中的 ![](../graphics/ico_info.png) 工具。 Abaqus/CAE显示**Query**对话框。
3. 从**Query**对话框中，选择以下查询之一： **Shell element normals** Abaqus/CAE使用着色渲染样式显示零件或装配。 表面法线与壳法线重合（顶面）的壳侧显示为棕色； 另一侧（底面）显示为紫色。 **Beam element tangents** Abaqus/CAE显示青色箭头，指示梁切线的方向。 **Mesh stack orientation** 对于六面体和楔形单元，Abaqus/CAE将顶面着色为棕色，底面着色为紫色。 类似地，箭头指示四边形单元的方向。 此外，Abaqus/CAE高亮显示任何具有不一致方向的单元面和边缘。 ****Mesh**** 对于装配、零件或零件实例、几何区域或单元，Abaqus/CAE显示以下内容： - 所选区域中的节点和单元总数 - 每种单元形状的单元数量 默认情况下，Abaqus/CAE在消息区域显示网格信息，但您可以通过在提示区域切换**Display detailed report**在**Mesh Statistics**对话框中以表格格式显示此信息。 **Mesh Statistics**对话框还允许您按零件实例或按单元类型显示网格信息。 **Element** 选择一个单元。 Abaqus/CAE在消息区域显示以下内容： - 单元标签 - 单元拓扑 - Abaqus/CAE将用于分析的单元类型 - 节点连通性 **Mesh gaps/intersections** 选择一个零件实例。 Abaqus/CAE在当前视口中高亮显示模型边界面的任何边缘，如下所示： - 不兼容的界面 - 裂纹或间隙 - 与其他面的交集 此外，如果发现相交的单元，Abaqus/CAE会显示一个对话框，允许您保存包含共享高亮边缘的单元的集。 如果模型几何可用，您可以保存包含与高亮单元边缘相关的单元、面或边缘的集。 **Free/Non-manifold edges** 如果当前视口包含具有多个零件实例的装配，请为查询选择一个或多个实体或壳实例。 Abaqus/CAE高亮显示两种类型的外部单元边缘： - 自由单元边缘是属于单个外部单元面的外部边缘。 - 非流形单元边缘是被两个以上相邻外部单元面共享的外部边缘。 如果Abaqus/CAE发现自由或非流形单元边缘，它会在消息区域显示这些边缘的合并总数。 Abaqus/CAE还显示一个对话框，您可以使用它保存包含所有具有高亮边缘的单元的单元集。 如果没有自由或非流形边缘，消息区域会指示所有外部单元边缘恰好由两个外部单元面共享。 [图 17-141](pt03ch17s19hlb02.md#mgn-freenonmanifold) 显示了一个拉伸壳零件，其中自由和非流形边缘以红色高亮显示。 三个平面脸上外部边界上的单元边缘是自由边缘。 三个面相交处的单元边缘是非流形边缘——每个由三个外部单元面共享。 **图 17-141** 自由单元边缘构成外部边缘，非流形边缘形成三个平面壳相交的中心。 ![](../graphics/mgn-freenonmanifold.png) 壳零件应在适当时包含自由边缘和非流形边缘以用于零件设计。 实体零件不应包含任何自由或非流形边缘。 **Unmeshed regions** Abaqus/CAE显示警告对话框并高亮显示模型中未进行网格划分的任何区域。 切换**Save regions in a set**并选择集名称以保存包含未网格划分区域的集。 如果没有未网格划分的区域，Abaqus/CAE会在消息区域显示一条消息，指示所有区域都已完全网格划分。 该查询不考虑不需要网格的区域，如显示体和解析刚性表面。 **Unassociated geometry** Abaqus/CAE显示**Query Unassociated Geometry**对话框，它使您能够高亮显示零件或模型中未与网格关联的单元、面、边缘或顶点。 切换**Create set**并选择集名称以保存包含未与网格关联的几何的集。 如果您选择的所有几何都与网格关联，Abaqus/CAE会显示一个对话框，指示所有选定的几何都与网格关联。 该查询不考虑不需要网格的区域，如显示体和解析刚性表面。
4. 获取网格信息完成后，关闭**Query**对话框。
