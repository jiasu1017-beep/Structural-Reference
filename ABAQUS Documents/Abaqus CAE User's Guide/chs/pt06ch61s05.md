# 61.5 使用 Customize 工具集





Customize 工具集的所有功能都可以通过 **Customize** 对话框访问。在任何模块中从主菜单栏选择 ****Tools**![](../graphics/images/arrow.gif)**Customize**** 或 ****View**![](../graphics/images/arrow.gif)**Toolbars**![](../graphics/images/arrow.gif)**Customize**** 打开 **Customize** 对话框。
