# 64.6.7 交换一对相邻三角形单元的对角线





从主菜单栏选择 ****网格**![](../graphics/images/arrow.gif)**编辑**** 可以交换一对相邻三角形单元的对角线，如下图所示：

![](../graphics/edt-meshedit-flipdiagonal.png)

相邻三角形单元必须具有一致的法线。如有需要，您可以使用"编辑网格"工具集翻转法线；有关更多信息，请参见 ["反转壳单元的表面法线方向，" 第64.6.3节](pt06ch64s06s03.md)。

**交换对角线的步骤：**

1. 进入"网格"模块，并执行以下操作之一： - 从上下文栏的 **对象** 字段中选择 **装配**。 - 从上下文栏的 **对象** 字段中选择 **零件** 并从列表中选择一个零件。
2. 从主菜单栏选择 ****网格**![](../graphics/images/arrow.gif)**编辑****。Abaqus/CAE 显示 **编辑网格** 对话框。 **提示：** 您也可以使用位于"网格"模块工具箱底部的 ![](../graphics/ico_meshEdit.png) 工具显示 **编辑网格** 对话框。
3. 在对话框中执行以下操作： 1. 在 **类别** 字段中选择 **单元**。 2. 从 **方法** 列表中选择 **交换对角线（三角）**。
4. 选择要交换的相邻三角形单元的边缘。您只能选择一个边缘。Abaqus/CAE 会高亮显示所选边缘，并提示您确认要交换所选边缘。
5. 点击 **是** 以交换单元边缘。Abaqus/CAE 会提示您选择下一个要交换的单元边缘。 **提示：** 如果在交换对角线时出错，点击 **编辑网格** 对话框中的 **撤销** 可以撤消更改。Abaqus/CAE 会恢复两个相邻单元中的原始对角线。
6. 完成交换对角线后，点击鼠标按钮2或提示区域中的取消按钮 ![](../graphics/afxI_cancel.png) 退出过程。
