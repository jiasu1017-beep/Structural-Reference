# 5.6.10 应用指定视图





从主菜单栏中选择****View**![](../graphics/images/arrow.gif)**Specify****来指定视图。你可以选择以下方法来指定视图：

****旋转角度****

你可以指定 Abaqus/CAE 围绕模型或屏幕 1、2 和 3 轴旋转模型视图的角度。你也可以选择从绝对位置（"前"视图）或从当前位置旋转模型视图。

****视点****

你可以指定观察者查看模型的向量坐标。你也可以通过指定一个代表"上"方向的向量来在视口中定位全局 1、2 和 3 轴。

****缩放****

你可以指定放大或缩小视图的缩放因子。你也可以选择相对于视口中对象的绝对大小（应用缩放因子为 1 的默认大小）或相对于对象当前大小来缩放视图。

****平移****

你可以指定沿 1 和 2 方向移动视图。这些值对应于视口水平和垂直尺寸的分数，并相对于当前视图。

有关更详细的说明，请参阅["数值指定视图，" 第 5.2.9 节](pt01ch05s02s09.md)。

如果当前视口链接到其他视口，Abaqus/CAE 也会将会话中所有链接视口中的指定视图应用。有关更多信息，请参阅["链接视口以进行视图操作，" 第 4.5 节](pt01ch04s05.md)。

**要指定视图：**

1. 从主菜单栏中，选择****View**![](../graphics/images/arrow.gif)**Specify****。**提示：**你也可以通过双击 3D 罗盘来指定视图。Abaqus/CAE 显示**指定视图**对话框。
2. 从**指定视图**对话框中，选择所需的方法，并执行以下操作之一：
   - 如果你选择了**旋转角度**方法，输入围绕 *X*、*Y* 和 *Z* 轴的旋转角度（![](../graphics/usi_eqn00004.gif)、![](../graphics/usi_eqn00005.gif)、![](../graphics/usi_eqn00006.gif)）；正数对应于围绕每个轴的逆时针旋转。使用**模式**按钮指定 Abaqus/CAE 如何应用你的旋转：
      - 选择**关于模型轴增量**将旋转应用到当前视图的模型轴。
      - 选择**关于屏幕轴增量**将旋转应用到当前视图的屏幕轴。屏幕 *X* 轴是水平的，*Y* 轴是垂直的，*Z* 轴是垂直于屏幕的。屏幕轴的原点是视口中心。
      - 选择**从 (0,0,1) 的总旋转**首先将视图旋转到默认位置（沿 3 轴向下看，1 和 2 轴在屏幕平面中的视图），然后应用旋转。
   - 如果你选择了**视点**方法，输入视点向量的 *X*、*Y* 和 *Z* 坐标以及上向量的坐标。
   - 如果你选择了**缩放**方法，输入缩放因子，并选择**绝对**或**相对**放大。大于 1 的缩放因子会扩展模型的视图，介于 0 和 1 之间的缩放因子会收缩模型的视图。
   - 如果你选择了**平移**方法，输入表示你希望相对于当前视图定位模型视图的值。这些值是视口尺寸的分数；第一个值代表水平运动，第二个值代表垂直运动。
3. 单击**确定**应用你指定的视图并关闭**指定视图**对话框。**提示：**使用[循环](pt01ch05s06hlb07.md)视图操作工具返回原始视图。
