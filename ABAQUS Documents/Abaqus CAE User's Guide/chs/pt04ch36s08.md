# 36.8 创建和编辑表皮加固





在初始创建表皮期间，您选择一个或多个要加固的面，如果需要，还可以选择要创建的表皮数量。创建表皮后，您可以以下列方式修改：
- 在Property模块中，您可以通过为其分配截面（均匀壳截面、复合壳截面、膜截面、表面截面或垫片截面）、材料方向或钢筋参考方向来修改表皮。您还可以通过为其底层面向分配法线方向来反转表皮的法线方向。当您反转面的法线方向时，分配给该面的所有表皮也会继承新的法线方向。
- 在Mesh模块中，您可以为表皮分配单元类型。当提示您选择要分配单元类型的区域时，必须从**Selection**工具栏的对象类型列表中选择**Skins**。如果您为表皮分配壳截面，则必须为表皮分配壳单元。同样，如果您分配膜、表面或垫片截面，则必须分配膜、表面或垫片单元。

Abaqus/CAE自动命名表皮，但您可以从**Skin Manager**、**Special**菜单或Model Tree中重命名它们。如果您一次创建多个表皮，Abaqus/CAE会附加唯一标识符到名称以识别每个表皮。

您可以通过使用Step模块中的场和历史输出请求编辑器从表皮获取数据。在编辑器的**Domain**部分，选择**Skin**并从出现的菜单中选择所需的表皮。更多信息，请参见["创建输出请求"，第14.12.1节](pt03ch14s12hlb01.md)。

**要创建表皮加固：**

1. 从Property模块的主菜单栏中，选择****Special**![](../graphics/images/arrow.gif)**Skin**![](../graphics/images/arrow.gif)**Create****。
   **提示：**您也可以单击Property模块工具箱中的![](../graphics/ico_skin.png)工具。
2. 使用以下方法之一选择要添加表皮加固的面：
   **选择各个面：**
   1. 在提示区域中单击**Select entities on which to create a skin**字段旁边的箭头，并从出现的列表中选择**individually**。
   2. 选择要添加表皮加固的面。
   3. **[Shift]**+单击其他面以将其添加到您的选择。
   4. 如有必要，**[Ctrl]**+单击所选面以取消选择。
   5. 完成选择面后，单击鼠标按钮2。
   **指定现有集：**
   1. 在提示区域右侧单击**Sets**。Abaqus/CAE显示**Region Selection**对话框，其中包含您创建的集列表。
   2. 选择要添加表皮加固的面集，然后单击**Continue**。
   **注意：**默认选择方法基于您最近使用的选择方法。要切换到另一种方法，请在提示区域右侧单击**Select in Viewport**或**Sets**。
   **使用角度方法选择面：**
   1. 在提示区域中单击**Select entities on which to create a skin**字段旁边的箭头，并从出现的列表中选择**by face angle**。
   2. 输入角度（0到90），然后选择一个面。Abaqus/CAE从所选面开始选择每个相邻面，直到面之间的角度等于或超过您输入的角度。（更多信息，请参见["使用角度和特征边缘方法选择多个对象"，第6.2.3节](pt01ch06s02hlb03.md)。）
   3. 使用**by face angle**方法后，您可以**[Shift]**+单击其他面以将其添加到您的选择，或**[Ctrl]**+单击所选面以将其从您的选择中移除。（更多信息，请参见["组合选择技术"，第6.2.8节](pt01ch06s02hlb08.md)。）
   **提示：**您可以使用**Selection**工具栏来限制可以在视口中选择的对象类型。更多信息，请参见[第6章，"在视口中选择对象"](pt01ch06.md)。
3. 如需，在提示区域中切换**Multiple skins**以为所选面创建多个表皮加固。
4. 单击**Done**创建表皮。如果您要创建多个表皮，请输入您要创建的表皮层数，然后单击鼠标按钮2。
