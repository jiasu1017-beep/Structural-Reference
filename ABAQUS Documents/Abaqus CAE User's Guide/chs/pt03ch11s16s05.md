# 11.16.2 在零件模块中使用特征操作工具集





以下被视为零件的特征：
- 几何特征，如拉伸实体、旋转壳、草图线框和圆角边缘
- 修复操作
- 分区
- 基准几何

当特征操作工具集要求您选择特征时，您可以从视口中选择它。或者，您可以从模型树中选择特征。

如果在模型树中右键单击特征，出现的新菜单允许您执行以下操作：

**编辑**

编辑特征时，Abaqus/CAE显示特征编辑器。您可以直接修改特征的参数，或者如果适用，可以修改形成特征二维轮廓或扫掠路径的草图。

**重新生成**

修改复杂零件中的特征时，延迟重新生成直到完成所有更改可能很方便，因为重新生成可能耗时。当您准备好重新生成零件时，选择**特征**![](../graphics/images/arrow.gif)**重新生成**。

**重命名**

重命名零件。

**抑制**

抑制特征会从零件定义中临时将其移除。被抑制的特征不可见，不能进行网格划分，也不包含在模型分析中。您不能抑制基础特征，抑制父特征将抑制其所有子特征。

**恢复**

恢复特征会将抑制的特征恢复到零件；恢复父特征会恢复其所有子特征。您可以选择恢复所有特征、最近抑制的一组特征或选定的特征。

**删除**

删除特征会将其从零件中移除。您无法恢复已删除的特征。

**查询**

查询零件时，Abaqus/CAE会在消息区域显示信息，并将相同信息以注释形式写入重放文件（`*abaqus*.rpy`）。

**选项**

**特征选项**对话框允许您调整当前模型的重新生成性能。
