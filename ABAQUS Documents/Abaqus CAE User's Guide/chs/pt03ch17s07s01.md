# 17.7.1 概述





Abaqus/CAE中的大多数网格划分以"自顶向下"的方式完成。这意味着网格被创建为与区域几何形状完全一致，并向下处理到单元和节点位置。Abaqus/CAE遵循以下基本步骤生成网格：

1. 使用当前分配给该区域的网络技术对每个自顶向下区域生成网格。默认情况下，Abaqus/CAE在整个过程中生成具有一阶线、四边形或六面体单元的网格。
2. 将所有区域的网格合并为单个网格。通常，Abaqus/CAE会将相邻区域公共边界上的节点合并为一组节点。但是，在某些情况下，Abaqus/CAE会创建绑定面相互作用而不是合并这些节点；例如，在六面体和四面体网格之间的公共界面上。有关更多信息，请参见["对多个三维实体区域进行网格划分，"第17.14.1节](pt03ch17s14s01.md)。

Abaqus/CAE生成的自顶向下网格符合它们划分的零件或零件实例的几何形状，如图17-31所示：

**图17-31** 网格符合零件实例的几何形状。

![](../graphics/mgn-overview-nls.png)

- 在每个几何顶点处生成一个节点。
- 沿每条几何边生成一组连通的单元边。
- 沿每个几何面生成一组连通的单元面。
- 边界上的节点（包括二阶单元的中边节点）也在几何边界上。
- 内部二阶单元的中边节点位于单元边端节点之间的中心位置。

有关创建自顶向下网格的详细分步说明，请参见["创建网格，"第17.17.1节](pt03ch17s17s01.md)。

直接依赖几何形状形成外部网格边界可能会影响网格质量，因为Abaqus/CAE会创建元素来填充小细节。在某些情况下，您可能无法实现允许您在复杂区域上应用自顶向下扫掠或结构化网格技术的分割策略。对于实体区域，您可以使用"自底向上"网格技术来替代自动自顶向下网格技术生成六面体网格。自底向上网格是一种手动、增量网格划分过程，从二维实体逐步构建三维网格。您定义将使用自底向上技术进行网格划分的区域，控制网格划分过程，判断生成的网格是否满足您的需求，并且——由于网格不需要符合几何形状——控制几何形状与网格的关联性。有关自底向上网格的更多信息，请参见["自底向上网格，"第17.11节](pt03ch17s11.md)。
