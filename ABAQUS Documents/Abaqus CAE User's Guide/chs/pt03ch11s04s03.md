# 11.4.3 零件尺寸





创建新零件时，必须选择零件的近似尺寸。您输入的尺寸用于Abaqus/CAE计算草图工作表及其网格的间距。您应将零件的近似尺寸设置为与完成零件的最大尺寸相匹配。如果随后发现零件超出了草图工作表的尺寸，请使用草图自定义选项增加工作表尺寸。创建零件后，您不能更改零件的近似尺寸。但是，您可以复制零件到新零件并在复制操作期间缩放零件。更多信息，请参见"复制零件"，第11.18.3节。

Abaqus/CAE使用几何引擎来建模零件和特征。建议的近似尺寸限制在0.001（10³）和10000（10⁴）单位之间。此尺寸范围应防止模型超出几何引擎的限制。例如，几何引擎支持的最小尺寸为10⁶，因此保持10³数量级的几何通常允许节点和单元尺寸保持在最小尺寸以上。超出建议限制的零件可能会出现几何缺陷。如果您发现自己需要指定超出建议范围的近似尺寸，应考虑采用不同的单位集。
