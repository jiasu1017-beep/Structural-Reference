# 15.12.3 自定义接触控制





接触控制编辑器允许您修改用于实施接触条件的算法。默认接触控制通常已经足够，但自定义这些控制可能会得到更经济有效的解决方案。

**警告：**接触控制适用于高级用户。这些控制的默认设置适合大多数分析。使用这些控制的非默认值可能会大大增加分析的计算时间或产生不准确的结果。在Abaqus/Standard分析中更改这些设置也可能导致收敛问题。

**自定义接触控制的步骤：**

1. 从主菜单栏选择 ****Interaction**![](../graphics/images/arrow.gif)**Contact Controls**![](../graphics/images/arrow.gif)**Create****。
2. 在出现的「Create Contact Controls」对话框中，执行以下操作：- 为接触控制命名。- 选择「Abaqus/Standard contact controls」或「Abaqus/Explicit contact controls」。
3. 点击「Continue」。会出现您指定的接触控制类型的编辑器。
4. 在编辑器中，输入自定义接触控制所需的数据。详细说明，请参见以下章节：- 「在Abaqus/Standard分析中指定接触控制」，第15.13.9节- 「在Abaqus/Explicit分析中指定接触控制」，第15.13.10节
5. 如果您想将编辑器中的值重置为默认值，请点击编辑器底部的「Defaults」。
6. 点击「OK」保存您的自定义设置并退出编辑器。
