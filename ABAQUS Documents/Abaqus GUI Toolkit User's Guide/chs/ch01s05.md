# 1.2 Abaqus入门

本指南是一本入门教材，旨在为新用户提供指导，帮助他们使用Abaqus/Standard和Abaqus/Explicit分析实体、壳、梁和桁架模型，并在Abaqus/Viewer或其他后处理器中查看结果。您无需任何Abaqus以前的知识即可从本指南中获益，尽管建议您以前接触过有限元方法。如果您已经熟悉Abaqus求解器产品（Abaqus/Standard或Abaqus/Explicit），但希望了解Abaqus/CAE界面入门知识，请参阅《Abaqus入门指南：交互版》../gsa/gsa-link.md#gsa)指南。

本文档主要涵盖应力/位移模拟，重点介绍线性和非线性静态分析以及动态分析。还包括CFD分析和流体结构耦合建模的简介。其他类型的模拟（如热传递和质量扩散）不在本指南范围内。

## 1.2.1 如何使用本指南

本指南的每一章都介绍一个或多个与使用Abaqus/Standard和Abaqus/Explicit相关的主题。在整个指南中，术语"Abaqus"统指Abaqus/Standard和Abaqus/Explicit；仅当信息适用于单一产品时，才使用各个产品名称。大多数章节都包含对所考虑主题的简短讨论和一个或两个教程示例。您应该仔细研究这些示例，因为它们包含大量关于使用Abaqus的实用建议。

Abaqus/Standard和Abaqus/Explicit的功能在这些示例中逐步引入。您可以使用文本编辑器创建输入文件；但是，使用交互式预处理器会促进这些示例的模型创建。您在每个示例中创建的所有输入文件的完整版本在[附录A"示例文件"](ap01.md)中。如果您可以访问Abaqus/CAE，您可以使用配套指南《Abaqus入门指南：交互版》../gsa/gsa-link.md#gsa)，通过详细的Abaqus/CAE教程执行所有预处理和分析步骤。

本章是对Abaqus和本指南的简短介绍。[第2章"Abaqus基础"](ch02.md)以一个简单示例为中心，涵盖使用Abaqus的基础知识。读完[第2章"Abaqus基础"](ch02.md)后，您将了解准备Abaqus仿真模型、检查数据、运行分析作业和查看结果的基础知识。

[第3章"有限元和刚体"](ch03.md)概述了Abaqus中可用的主要单元族。连续体（实体）单元、壳单元和梁单元的使用分别在[第4章"使用连续体单元"](ch04.md)）、[第5章"使用壳单元"](ch05.md)"和[第6章"使用梁单元"](ch06.md)中讨论。

[第7章"线性动力学"](ch07.md)讨论了线性动态分析。[第8章"非线性"](ch08.md)介绍了非线性的概念，特别是几何非线性，并包含第一个非线性Abaqus模拟。[第9章"非线性显式动力学"](ch09.md)讨论了非线性动态分析，[第10章"材料"](ch10.md)介绍了材料非线性。[第11章"多步骤分析"](ch11.md)介绍了多步骤模拟的概念，[第12章"接触"](ch12.md)讨论了接触分析中出现的许多问题。[第13章"使用Abaqus/Explicit进行准静态分析"](ch13.md)介绍了使用Abaqus/Explicit求解准静态问题。说明性示例是板材成型模拟，需要在Abaqus/Explicit和Abaqus/Standard之间导入，以高效执行成型和回弹分析。

## 1.2.2 本指南中使用的约定

本指南遵循以下约定：

**排版约定**

教程示例中使用不同的文本样式来指示特定操作或识别项目。
- 使用`COURIER FONT`的输入应按显示的精确方式键入到Abaqus/Viewer或您的计算机中。例如，要在计算机上运行Abaqus/Viewer，应键入``` abaqus viewer ```。
- Abaqus/Viewer中的菜单选择、对话框中的选项卡以及屏幕上的项目标签使用粗体表示：``` ****视图**![](../graphics/images/arrow.gif)**图形选项**** **等值线图选项** ```

**视图方向 triad**

默认情况下，Abaqus/Viewer使用字母选项*x-y-z*来标记视图方向triad。一般来说，本指南采用数字选项1-2-3，以便与自由度输出标记直接对应。

## 1.2.3 基本鼠标操作

[图1-2](ch01s02.md#gst-mouse)显示了左手和右手3键鼠标的鼠标按钮方向。

**图1-2** 鼠标按钮。

![](../graphics/cnv-mouse-nls.png)

以下术语描述了使用鼠标执行的操作：

**单击**

按下并快速释放鼠标按钮。除非另有说明，"单击"意味着您应该单击鼠标按钮1。

**拖动**

按下并按住鼠标按钮1，同时移动鼠标。

**指向**

移动鼠标直到光标在所需项目上。

**选择**

指向项目，然后单击鼠标按钮1。

****[Shift]****+单击****

按下并按住**[Shift]**键，单击鼠标按钮1，然后释放**[Shift]**键。

****[Ctrl]****+单击****

按下并按住**[Ctrl]**键，单击鼠标按钮1，然后释放**[Ctrl]**键。

Abaqus/Viewer设计为使用3键鼠标。因此，本指南按照[图1-2](ch01s02.md#gst-mouse)所示引用鼠标按钮1、2和3。但是，您可以通过以下方式将Abaqus/Viewer与2键鼠标一起使用：
- 两个鼠标按钮等同于3键鼠标上的鼠标按钮1和3。
- 同时按下两个鼠标按钮等同于在3键鼠标上按下鼠标按钮2。

**提示：**您将在整个本指南的过程中被指示单击鼠标按钮2。确保将鼠标按钮2（或滚轮按钮）配置为充当中间按钮单击。
