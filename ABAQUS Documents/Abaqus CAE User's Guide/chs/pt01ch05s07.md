# 5.7 将 3Dconnexion 运动控制器与 Abaqus/CAE 结合使用





[3Dconnexion](http://www.3dconnexion.com) 生产各种视图操作设备，这些设备在 CAE 和 CAD 系统用户中很受欢迎。其设备的一个例子是 [图 5-18](pt01ch05s07.md#viw-spaceball) 中所示的 SpaceBall。

**图 5-18** 3Dconnexion SpaceBall。

![](../graphics/viw-spaceball.png)

你可以将 3Dconnexion 运动控制器与鼠标结合使用，以更高效地与 Abaqus/CAE 交互。你可以使用运动控制器用一只手操作模型视图，同时用另一只手使用鼠标从模型中进行选择。运动控制器对光标下的任何内容进行操作；例如，视口中的部件或变形绘图或对话框中的滚动条。如果光标下既没有视口也没有滚动条，运动控制器会对当前视口进行操作。

如果将运动控制器与旋转视图操作工具结合使用，你可以更改对象的旋转中心。默认情况下，旋转工具和运动控制器都围绕视口中心旋转对象。但是，你可以通过将光标定位在视口中要用作旋转中心的位置，单击鼠标按钮 3，并选择**设置为旋转中心**来为旋转工具和运动控制器选择旋转中心。或者，如果你单击**视图操作**工具栏中的旋转工具 ![](../graphics/ico_viewRotate.png)，提示区域中会出现一个**选择**按钮。如果你单击此按钮，可以从视口中的顶点或节点选择旋转中心。即使你退出旋转模式，运动控制器仍继续使用指定的旋转中心。有关更多信息，请参阅["旋转视图工具，" 第 5.2.3 节](pt01ch05s02s03.md)。

此外，Abaqus 提供了一套通过标准 3Dconnexion 用户界面可用的"应用程序功能"。这些功能提供对视图操作、视图和显示工具的快捷方式。你可以将这些功能映射到内置于 3Dconnexion 运动控制器中的可编程按钮。

在 Windows 平台上，Abaqus 提供以下功能：

**电影模式**

在默认和替代旋转模式之间切换。默认模式类似于围绕相机目标或所选旋转中心旋转相机；替代模式围绕自身旋转相机，而不是围绕相机目标旋转。如果你按住 **[Shift]** 键同时使用旋转视图操作工具，你也可以切换到替代旋转模式。有关更多信息，请参阅["相机模式和视图术语，" 第 5.1.1 节](pt01ch05s01s01.md)；["使用视图选项控制相机，" 第 5.1.2 节](pt01ch05s01s02.md)；和["旋转视图工具，" 第 5.2.3 节](pt01ch05s02s03.md)。

**降低 Abaqus 灵敏度**

降低 3Dconnexion 运动控制器的灵敏度。此设置仅在 Abaqus/CAE 中操作视图时适用。

**提高 Abaqus 灵敏度**

提高 3Dconnexion 运动控制器的灵敏度。此设置仅在 Abaqus/CAE 中操作视图时适用。

**重置 Abaqus 灵敏度**

将 3Dconnexion 运动控制器的灵敏度恢复为其默认设置。

**自动拟合**

将模型拟合到视口。这与自动拟合 ![](../graphics/ico_viewFit.png) 视图操作工具的功能相同。有关更多信息，请参阅["自动拟合工具，" 第 5.2.6 节](pt01ch05s02s06.md)。

**保持在视图中**

在旋转和平移时保持模型在视图中。当使用运动控制器操作视图时，切换此功能可防止你将相机目标移出视口。同样，如果你已更改旋转中心，此选项可防止你将中心移出视口。

**缩放到光标**

用缩放替换旋转和平移。如果你按住鼠标按钮 2 并操作运动控制器，Abaqus/CAE 会禁用正常的平移和旋转模式，并用仅从鼠标光标下方区域放大或缩小的模式替换它们。如果你释放鼠标按钮 2，正常的平移和旋转模式会恢复。此行为默认启用。你可以使用此功能在两种模式之间切换。

**线框/着色**

在线框和着色渲染样式之间切换。这与**渲染样式**工具栏中的线框 ![](../graphics/ico_displayWire.png) 和着色 ![](../graphics/ico_displayShaded.png) 图标的功能相同。有关更多信息，请参阅["选择渲染样式，" 第 76.2 节](pt07ch76hla01.md)。

**透视**

在透视和平行视图之间切换。这与**视图选项**工具栏中的透视 ![](../graphics/ico_viewPerspective.png) 和平行 ![](../graphics/ico_viewParallel.png) 图标的功能相同。有关更多信息，请参阅["控制透视，" 第 5.5 节](pt01ch05s05.md)。

**操作层**

操作所有层或当前层。如果你在可视化模块中创建了结果叠加绘图，你可以使用此功能在将视图操作应用到所有层或仅当前层之间切换。有关更多信息，请参阅["为叠加绘图操作视图，" 第 79.2.3 节](pt07ch79s02hlb03.md)。

**向上翻页**

在可滚动对话框中向上翻页。如果对话框只允许水平滚动，此按钮将页面移动到右侧。

**向下翻页**

在可滚动对话框中向下翻页。如果对话框只允许水平滚动，此按钮将页面移动到左侧。

**设置旋转中心**

在鼠标位置设置旋转中心。

**清除旋转中心**

清除先前设置的旋转中心。

**设置视图中心**

在鼠标位置居中视图。

在 UNIX 平台上，Abaqus 还提供了一套通过标准 3Dconnexion 用户界面可用的"应用程序功能"。你应该从**应用程序**列表中选择 Abaqus。如果 Abaqus 未列出，你应该选择 **XWindow Driver Version 2.0/3.0**。

UNIX 平台上可编程按钮的默认映射如下表所示：

| 按钮 | 含义 |
| --- | --- |
| 4 | 电影模式 |
| 5 | 降低灵敏度 |
| 6 | 提高灵敏度 |
| 7 | 重置灵敏度 |
| 8 | 自动拟合 |
| 9 | 保持在视图中 |
| A | 缩放到光标 开/关 |
| B | 线框/着色 |

Abaqus 应用程序的按钮映射是固定的。但是，3Dconnexion 允许你重新分配按钮以使用其驱动程序提供的标准功能。有关更多信息，请参阅 3Dconnexion 文档的 UNIX/Linux 版本。
