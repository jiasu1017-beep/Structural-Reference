# 14.13.3 配置诊断打印





选择 ****Output**![](../graphics/images/arrow.gif)**Diagnostic Print**** 可以配置Abaqus写入消息文件（对于Abaqus/Standard）或状态文件（对于Abaqus/Explicit）的诊断消息，并在分析过程中报告给Job模块。如果您的模型分析失败或产生意外结果，您可以检查这些消息并查看分析说明。（更多信息，请参见「诊断打印」，第14.5.3节。）

Abaqus/Standard还会将大多数诊断信息写入输出数据库，以供可视化模块使用。但是，写入输出数据库的诊断信息不受诊断打印设置的影响。（更多信息，请参见第41章，「查看诊断输出」。）

**配置诊断打印的步骤：**

1. 从主菜单栏选择 ****Output**![](../graphics/images/arrow.gif)**Diagnostic Print****。此时会出现「Edit Diagnostic Print」对话框，其中显示当前模型中所有步骤的列表。
2. 在「Frequency」列中，指定要为每个步骤写入诊断打印数据的频率。（此选项不适用于Abaqus/Explicit分析。）
3. 点击其他列可以在特定步骤期间请求特定的诊断信息。会出现一个勾选标记，表示该类型的诊断打印在该步骤期间已启用。再次点击该列可以关闭诊断打印。
4. 完成诊断打印的配置后，点击「OK」退出「Edit Diagnostic Print」对话框。
