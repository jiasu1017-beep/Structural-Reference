# 3.2.3 使用暗淡的对话框和工具箱组件





某些对话框和工具箱中的对象仅在特定情况下可用。当对象不可用时，它会在对话框中显示为暗淡。项通常由于对话框中的其他设置而显示为暗淡。例如，如果未选择**使用以下设置**，其下方的图像大小选项不可用，会显示为暗淡，如下所示。

![](../graphics/int-dimmed-nls.png)

即使对于暗淡的选项，上下文相关帮助也是可用的，但工具提示不可用。
