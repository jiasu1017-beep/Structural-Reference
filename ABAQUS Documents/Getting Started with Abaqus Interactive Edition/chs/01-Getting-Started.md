# 1.2 ABAQUS入门

## 摘要

本指南是一本入门教材，旨在为新用户提供创建实体、壳、梁和桁架模型的指导，使用Abaqus/CAE进行分析，使用Abaqus/Standard和Abaqus/Explicit分析模型，并在可视化模块中查看结果。附录中包含了使用Abaqus/CFD的简要介绍。您不需要任何Abaqus的先前知识即可从本指南中获益，尽管建议您对有限元方法有一些初步了解。如果您已经熟悉Abaqus求解器产品（Abaqus/Standard或Abaqus/Explicit），但希望了解Abaqus/CAE界面，本指南的附录中提供了三个教程来引导您完成Abaqus/CAE中的建模过程。

本文档主要涵盖应力/位移模拟，重点关注线性和非线性静态分析以及动态分析。文档还包含了CFD分析和流体-结构耦合建模的介绍。其他类型的模拟（如传热和质量扩散）不在本指南的范围内。

## 1.2.1 如何使用本指南

本指南的不同部分针对不同类型的用户。

### 新Abaqus用户的教程

如果您是Abaqus的完全新手，我们建议您按照本指南中的每个自学教程进行学习。本指南的每一章和附录都介绍了一个或多个与使用Abaqus/Standard、Abaqus/Explicit或Abaqus/CFD相关的主题。在整个指南中，术语"Abaqus"用于统称这三个分析产品；仅在信息适用于单一产品时使用各个产品名称。大多数章节包含对所考虑主题的简要讨论以及一个或两个教程示例。您应该仔细阅读这些示例，因为它们包含大量使用Abaqus的实践经验。

在这些示例中，Abaqus/CAE的功能是逐渐介绍的。我们假定您将使用Abaqus/CAE来创建示例中使用的模型。您也可以使用脚本生成任何示例的模型，该脚本可以复制问题的完整分析模型。使用脚本创建的模型可能与按照本指南中的步骤创建的模型略有不同。这些差异（如材料名称或节点编号）是次要的，可以忽略。脚本位于两个位置：

1. **附录A中的Python脚本**：每个示例都提供了一个Python脚本。"示例文件"附录提供了关于如何获取脚本并在Abaqus/CAE中运行的说明。
2. **Abaqus/CAE插件脚本**：每个示例都在Abaqus/CAE插件工具集的"入门示例"对话框中提供了插件脚本。

如果您无法访问Abaqus/CAE或其他预处理器，您可以使用配套指南《Abaqus入门：关键字版》来手动创建大多数示例所需的输入文件。

本章是对Abaqus和本指南的简要介绍。第2章"Abaqus基础"以一个简单示例为中心，涵盖了使用Abaqus的基础知识。到第2章结束时，您将了解为Abaqus模拟准备模型、检查数据、运行分析作业和查看结果的基本方法。

第3章"有限元和刚体"介绍了Abaqus中可用的主要单元族。连续体（实体）单元、壳单元和梁单元的使用分别在第4章"使用连续体单元"、第5章"使用壳单元"和第6章"使用梁单元"中讨论。

第7章"线性动力学"讨论了线性动态分析。第8章"非线性"介绍了非线性的一般概念，特别是不确定性（非线性）几何，并包含第一个非线性Abaqus模拟。第9章"非线性显式动力学"讨论了非线性动态分析，第10章"材料"介绍了材料非线性。第11章"多步骤分析"介绍了多步骤模拟的概念，第12章"接触"讨论了接触分析中出现的许多问题。第13章"使用Abaqus/Explicit进行准静态分析"介绍了使用Abaqus/Explicit解决准静态问题的方法。说明性示例是板材成形模拟，需要在Abaqus/Explicit和Abaqus/Standard之间导入，以高效地执行成形和回弹分析。

您可能会发现，在PDF版本的指南中遵循这些教程示例会更容易。这种方法可以减少屏幕上的混乱，让您专注于当前任务。如果您确实遵循HTML中的教程，您应该调整大小并移动Abaqus/CAE窗口和Web浏览器，以便在完成教程时两者都可见。

### 有经验的Abaqus用户的Abaqus/CAE教程

提供了四个附录来向熟悉Abaqus分析产品的用户介绍Abaqus/CAE界面。在附录B"在Abaqus/CAE中创建和分析简单模型"中，您可以创建简单模型、分析它，然后查看结果。第二个教程（附录C"使用附加技术在Abaqus/CAE中创建和分析模型"）更复杂，说明了零件、草图、基准几何和分区如何协同工作，以及如何装配零件实例。附录D"查看分析输出"演示了如何使用可视化模块（也作为Abaqus/Viewer单独许可）以各种格式显示结果，以及如何自定义显示。附录E"弯管中的流动"说明了如何使用Abaqus/CFD对弯管中的流体流动进行建模，以及如何使用Abaqus/Standard对管中的结构变形进行建模。

## 1.2.2 本指南中使用的约定

本指南遵守以下约定：

### 排版约定

在教程示例中，不同的文本样式用于指示特定操作或识别项目。

- **等宽字体输入**：应按原样输入到Abaqus/CAE或您的计算机中。例如：
  ```
  abaqus cae
  ```
  应在您的计算机上键入以运行Abaqus/CAE。

- **菜单选择**：对话框中的选项卡和屏幕上项目的标签以粗体表示，例如：
  ```
  **View** → **Graphics Options**
  **Contour Plot Options**
  ```

### 草图

草图是形成定义Abaqus/CAE本机零件几何特征的二维轮廓。您使用草图编辑器创建这些草图（如图1-2所示）。草图编辑器以实线样式显示主要网格线，以虚线样式显示X轴和Y轴以及次要网格线。为了在视觉上区分零件草图和草图编辑器网格，本指南中的大多数草图图形都使用虚线网格线。

## 1.2.3 基本鼠标操作

图1-3显示了左手和右手3按钮鼠标的鼠标按钮方向。

![图1-3 鼠标按钮](../graphics/cnv-mouse-nls.png)

**图1-3 鼠标按钮**

以下术语描述了您使用鼠标执行的操作：

### Click（单击）

按下并快速释放鼠标按钮。除非另有说明，否则"单击"指令意味着您应该单击鼠标按钮1。

### Drag（拖动）

按下并按住鼠标按钮1的同时移动鼠标。

### Point（指向）

将鼠标移动到所需项目上方。

### Select（选择）

指向一个项目，然后单击鼠标按钮1。

### [Shift]+Click

按下并按住[Shift]键，单击鼠标按钮1，然后释放[Shift]键。

### [Ctrl]+Click

按下并按住[Ctrl]键，单击鼠标按钮1，然后释放[Ctrl]键。

Abaqus/CAE设计为使用3按钮鼠标。因此，本指南将鼠标按钮称为1、2和3（如图1-3所示）。但是，您可以通过以下方式将Abaqus/CAE与2按钮鼠标一起使用：

- 两个鼠标按钮等同于3按钮鼠标上的鼠标按钮1和3。
- 同时按下两个鼠标按钮等同于按下3按钮鼠标上的鼠标按钮2。

> **提示：**
> 在整个指南的过程说明中，您会被指示单击鼠标按钮2。请确保将鼠标按钮2（或滚轮按钮）配置为充当中间按钮单击。
