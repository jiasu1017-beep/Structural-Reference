# 6.2 公式与积分





Abaqus 中的所有梁单元均为"梁柱"单元，即允许轴向、弯曲和扭转变形。Timoshenko 梁单元还考虑横向剪切变形的影响。

### 6.2.1 剪切变形

线性单元（B21 和 B31）以及二次单元（B22 和 B32）为可剪切变形的 Timoshenko 梁；因此，它们适用于建模短粗构件（其中剪切变形很重要）和细长梁（其中剪切变形不重要）。这些单元的横截面行为与厚壳单元的横截面行为相同，如图 6-7（b）所示，并在"壳公式——厚壳或薄壳"第 5.2 节中讨论。

**图 6-7** （a）细长梁和（b）厚梁中横向梁截面的行为。

![](../graphics/gss-transhellsect-nls.png)

Abaqus 假定这些梁单元的横向剪切刚度为线性弹性和恒定。此外，这些梁的公式使横截面积可以作为轴向变形的函数而变化，这种效应仅在几何非线性模拟中考虑（参见第 8 章，"非线性"），其中梁截面属性选项上的 POISSON 参数具有非零值。只要横截面尺寸小于结构典型轴向尺寸的 1/10，这些单元就可以提供有用的结果，这通常被认为是梁理论适用性的极限；如果梁截面在弯曲变形下不保持平面，则梁理论不足以模拟变形。

Abaqus/Standard 中可用的三次单元——所谓的 Euler-Bernoulli 梁单元（B23 和 B33）——不模拟剪切柔性。这些单元的横截面保持与梁轴垂直（参见图 6-7（a））。因此，三次梁单元最适用于建模相对细长构件的结构。由于三次单元沿其长度建模位移的三次变化，因此静力分析中通常可以使用单个三次单元建模结构构件，而动力分析中可以使用少量单元。这些单元假定剪切变形可以忽略。一般来说，如果横截面尺寸小于结构典型轴向尺寸的 1/15，此假设是有效的。

### 6.2.2 扭转响应——翘曲

结构构件经常受到扭矩作用，这发生在几乎任何三维框架结构中。导致一个构件弯曲的载荷可能导致另一个构件扭转，如图 6-8 所示。

**图 6-8** 框架结构中的扭转。

![](../graphics/gss-torsion-nls.png)

梁对扭矩的响应取决于其横截面的形状。一般来说，梁中的扭矩会在横截面中产生翘曲或非均匀的面外位移。Abaqus 仅在三维单元中考虑扭矩和翘曲的影响。翘曲计算假定翘曲位移很小。以下横截面在扭矩下行为不同：实心横截面；闭合薄壁横截面；和开口薄壁横截面。

**实心横截面**

实心非圆形横截面在扭矩下不会保持平面；相反，截面会翘曲。Abaqus 使用 St. Venant 翘曲理论计算横截面中每个截面点处由翘曲引起的单个剪切应变分量。这种实心截面中的翘曲被认为是无约束的，会产生可忽略的轴向应力。（翘曲约束仅在约束端附近影响解。）具有实心截面的梁的扭转刚度取决于材料的剪切模量 *G* 和梁截面的扭转常数 *J*。扭转常数取决于梁横截面的形状和翘曲特性。在横截面中产生大量非弹性变形的扭矩无法通过此方法准确建模。

**闭合薄壁横截面**

具有闭合薄壁非圆形横截面（BOX 或 HEX）的梁具有显著的扭转刚度，因此，其行为类似于实心截面。Abaqus 假定这些截面中的翘曲也是无约束的。横截面的薄壁特性允许 Abaqus 考虑沿壁厚恒定的剪切应变。薄壁假设通常在壁厚为典型梁横截面尺寸的 1/10 时有效。薄壁横截面的典型横截面尺寸示例包括：
- 管截面的直径。
- 箱形截面边缘的长度。
- 任意截面的典型边缘长度。

**开口薄壁横截面**

当翘曲无约束时，开口薄壁横截面在扭转中非常柔软，而此类结构扭转刚度的主要来源是轴向翘曲应变的约束。约束开口薄壁梁的翘曲会引入轴向应力，这些应力会影响梁对其他载荷类型的响应。Abaqus/Standard 具有可变形梁单元 B31OS 和 B32OS，它们包括开口薄壁截面中的翘曲效应。在建模承受重大扭矩的开口薄壁横截面结构时——如槽形（定义为 ARBITRARY 截面）或 I 形截面——必须使用这些单元。

翘曲引起的轴向变形在梁横截面上的变化由截面的翘曲函数定义。该函数的幅度在开口截面梁单元中被视为额外的自由度 7。约束此自由度可以防止在施加约束的节点处发生翘曲。

为了让翘曲幅度在每个分支中不同，框架结构中开口截面梁之间的连接通常应该为每个分支使用独立的节点进行建模（参见图 6-9）。

**图 6-9** 连接开口截面梁。

![](../graphics/gss-opensectbeam-nls.png)

但是，如果连接设计为防止翘曲，所有分支应共享一个公共节点，并且应使用 [*BOUNDARY](../key/key-link.md#usb-kws-hboundary) 选项约束翘曲自由度。

不通过梁剪切中心的剪切力会产生扭矩。扭矩等于剪切力乘以其相对于剪切中心的偏心距。在开口薄壁梁截面中，质心和剪切中心通常不重合（参见图 6-10）。如果节点不在横截面的剪切中心处，截面可能在载荷下扭转。
