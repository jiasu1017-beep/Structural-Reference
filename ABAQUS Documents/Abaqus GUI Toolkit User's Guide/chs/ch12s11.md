# 12.11 Abaqus/Standard 和 Abaqus/Explicit 之间的兼容性





Abaqus/Standard 和 Abaqus/Explicit 中的机械接触算法存在根本差异。这些差异反映在接触条件的定义方式上。主要差异如下：
- 对于接触对，Abaqus/Standard 默认使用纯主-从关系进行接触约束（参见 ["在 Abaqus/Standard 中定义接触对，" Abaqus 分析用户指南第 36.3.1 节](../usb/usb-link.md#usb-cni-acontactpair)）；从属表面的节点被约束不穿透主表面。原则上，主表面的节点可以穿透从属表面。Abaqus/Explicit 包含此公式，但通常默认使用平衡主-从加权（参见 ["Abaqus/Explicit 中接触对的接触公式，" Abaqus 分析用户指南第 38.2.2 节](../usb/usb-link.md#usb-cni-aexpcontactpairform)）。
- Abaqus/Standard 和 Abaqus/Explicit 中的接触公式在许多方面有所不同。例如，Abaqus/Standard 提供表面-表面公式，而 Abaqus/Explicit 提供边-边公式。
- Abaqus/Standard 和 Abaqus/Explicit 中的约束强制方法在某些方面有所不同。例如，Abaqus/Standard 和 Abaqus/Explicit 都提供罚函数约束方法，但默认罚函数刚度不同。
- Abaqus/Standard 和 Abaqus/Explicit 都提供小滑动接触公式（参见 ["Abaqus/Standard 中的接触公式，" Abaqus 分析用户指南第 38.1.1 节](../usb/usb-link.md#usb-cni-acontactpairform)，和 ["Abaqus/Explicit 中接触对的接触公式，" Abaqus 分析用户指南第 38.2.2 节](../usb/usb-link.md#usb-cni-aexpcontactpairform)）。但是，Abaqus/Standard 中的小滑动接触公式根据从属节点的当前位置将荷载传递到主节点。Abaqus/Explicit 始终通过锚点传递荷载。

由于这些差异，在 Abaqus/Standard 分析中指定的接触定义不能导入到 Abaqus/Explicit 分析中，反之亦然（参见 ["在 Abaqus/Explicit 和 Abaqus/Standard 之间传递结果，" Abaqus 分析用户指南第 9.2.2 节](../usb/usb-link.md#usb-anl-aexptostd))。
