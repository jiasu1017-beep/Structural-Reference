# 4.1.8 日内瓦机构

**产品：** Abaqus/Standard   

本示例说明了使用连接器元件对日内瓦机构进行建模，该机构将连续旋转运动转换为间歇旋转运动。

### 几何与模型

日内瓦机构本质上是一种计时装置。它用于计数仪器和其他需要将连续旋转运动转换为间歇旋转运动的应用。日内瓦机构由一个带突出销的旋转体和另一个带槽的旋转体组成，销在槽中滑动。

机构由三个名为 PIN、SLOT1 和 SLOT2 的刚体组成。SLOT1 是覆盖在 SLOT2 上的解析刚性表面。SLOT1 和 SLOT2 在 A 点刚性连接。销被约束为刚性的。

### 模型相互作用

PIN 和 SLOT1 之间的接触通过位于 C（销突出部分中心）的单个从节点进行。A 和 B 处除绕 3 轴的旋转自由度外的所有自由度都被固定。在 B 处绕 3 轴规定旋转 720 度，分三个步骤。

### 结果与讨论

当 PIN 旋转时，销的突出部分进入 SLOT1 中的槽。PIN 和 SLOT1 之间的相互作用导致 SLOT1 和 SLOT2 旋转，每完成 PIN 的 360 度旋转，它们就旋转 90 度。

### 输入文件

- geneva_model.py - Python 回放文件
- geneva.inp - 日内瓦机构模型
