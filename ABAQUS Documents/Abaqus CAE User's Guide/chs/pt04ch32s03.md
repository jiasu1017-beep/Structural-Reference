# 32.3 为区域分配垫片单元





如果区域是壳，则必须分配扫掠网格技术；如果区域是实体，则可以分配任何网格技术。对于实体区域，您可以定义独立于网格技术的堆叠方向。（更多信息，请参见["应用网格堆叠方向，" 第17.18.8节](pt03ch17s18hlb01.md)。）扫掠网格技术确保单元以适合垫片建模的方式堆叠。当您扫掠网格壳区域或未分配堆叠方向的实体区域时，每个垫片单元的轴线与扫掠路径方向重合；因此，您可以通过指定适当的扫掠路径来控制垫片单元对齐。

例如，[图32-3](pt04ch32s03.md#mgn-gasketalign)中所示的部件实例有三条可能的扫掠路径，每条路径有两个可能的扫掠方向。如果您使用垫片单元对该部件实例进行网格划分，您可以从六种可能的垫片轴方向中选择。更多信息，请参见["扫掠网格划分，" 第17.9节](pt03ch17s09.md)和["指定扫掠路径，" 第17.18.6节](pt03ch17s18s06.md)。

**图32–3** 对于壳区域，您必须选择为模型提供适当垫片轴方向的扫掠路径和方向。

![](../graphics/mgn-gasketalign.png)

此外，您可以为孤立网格单元分配垫片单元。您可以使用查询工具集来验证垫片单元是否正确对齐，用于垫片轴有多个可能方向的区域。如有必要，您可以使用编辑网格工具集更改网格堆叠方向。更多信息，请参见["定向堆叠方向，" 第64.6.4节](pt06ch64s06s04.md)。您可以将这些工具与六面体、楔形和四边形单元一起使用，因为只有这些单元类型能够堆叠形成连续体壳或垫片网格。如果已为该区域分配了二次垫片单元，则必须首先将分配更改为常规单元，然后再重新定向堆叠方向。然后可以通过重新分配二次垫片单元将该区域转换回垫片网格。
