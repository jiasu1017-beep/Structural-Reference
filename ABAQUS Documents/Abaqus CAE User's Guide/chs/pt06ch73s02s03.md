# 73.2.3 什么是曲面？







曲面是面和边或单元面和边的集合。当过程需要面时（例如，在施加分布式载荷（如压力载荷）和定义接触相互作用时），您可以选择曲面。您可以选择内部曲面；例如，当您使用实体偏移网格工具时。

当您在壳模型上创建曲面时，必须选择曲面的哪一侧是所需的面；您也可以选择两个面。同样，如果在线模型上创建曲面，必须选择线的哪一端是所需的面；您也可以选择圆周。要了解更多信息，请参见["指定区域的特定侧或端," 第73.2.5节](pt06ch73s02s05.md)。

您可以定义包含壳边或单元边的曲面。您可以在Abaqus/Explicit通用接触相互作用中使用基于边的曲面。您还可以使用基于边的曲面沿两条边将两个壳绑定在一起，或沿两条相交的边将两个壳绑定形成"T"。

您可以创建两种不同类型的曲面：

**几何曲面**

通过从原生或导入的几何体中选择几何对象（面和边）来创建几何曲面。创建分析输入文件时，输入文件中的曲面定义指定了与曲面几何体关联的单元边（对于轴对称部件实例的情况）或面。

**网格曲面**

通过从网格中选择单元面（对于三维区域）或边（对于二维区域）来创建网格曲面。向网格添加曲面允许您请求输出或将载荷添加到特定区域，而无需删除网格和分割几何体。但是，集成的网格曲面中的原生单元面或边如果对网格进行任何更改（包括从重放或日志文件重新生成网格），可能会失效或内容更改。

如果您重命名或删除曲面，则与该曲面关联的对象（如载荷或相互作用）将变为无效。但是，如果将重命名的曲面的名称更改回其原始名称，或者使用其原始名称重新创建已删除的曲面，则与该曲面关联的对象将恢复。

如果使用"虚拟拓扑工具集"创建虚拟面和边，则在创建集和曲面时可以选择这些虚拟面和边。此外，如果现有集或曲面包含使用"虚拟拓扑工具集"忽略的边或顶点，则该边或顶点将从集或曲面中删除。类似地，如果现有集或曲面包含您合并的面或边，Abaqus/CAE会用新的合并面和边替换原始面和边。只有当您合并的所有面和边都属于同一集时，这才适用。有关更多信息，请参见["我对包含虚拟拓扑的部件或部件实例有什么操作？," 第75.3节](pt06ch75s03.md)。
