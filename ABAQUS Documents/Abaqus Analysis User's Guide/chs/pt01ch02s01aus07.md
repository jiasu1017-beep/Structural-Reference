# 2.1.4 节点处的法线定义





**产品：**Abaqus/Standard  Abaqus/Explicit

##### **参考**

- [*NORMAL*](../key/key-link.md#usb-kws-mnormal)
- [*NODE*](../key/key-link.md#usb-kws-mnode)

### 概述

可以在节点处定义法线：
- 使用用户指定的法线定义；
- 作为梁和壳单元节点定义的一部分跟随节点坐标；
- 在Abaqus/Standard中接触对使用的刚性主曲面上；
- 在梁和壳单元中；
- 用于线弹簧单元以给出结构中缺陷的法线方向；
- 用于垫片单元以给出单元的厚度方向；和
- 用于轮廓积分评估。

在节点处定义的法线不影响由单元连接性定义的单元面法线。它们不必是单位长度。

### Abaqus/Standard中的接触曲面

Abaqus/Standard中接触曲面的用户指定曲面法线仅在使用小滑动接触方法或使用构成主曲面的刚性单元的有限滑动接触方法时才相关。当使用有限滑动时，在接触对中定义在变形主曲面上的用户指定曲面法线会被忽略。

小滑动接触公式使用主曲面上每个节点处的曲面法线来定义在曲面上从点到点平滑变化的法向量。有关如何使用曲面法线为每个从节点构建"主平面"的详细讨论，请参见["Abaqus/Standard中的接触公式，"第38.1.1节"](pt09ch38s01aus177.md)。

对于由刚性单元组成的主曲面，Abaqus/Standard平滑刚性单元之间任何不连续的曲面法线过渡。节点处的曲面法线用于控制曲面法线插值。有关此类主曲面平滑的详细讨论，请参见["解析刚性曲面定义，"第2.3.4节"](pt01ch02s03aus19.md)。

要定义法线，请指定全局坐标系中法线的分量。

| **输入文件用法：** | ``` [*NORMAL*](../key/key-link.md#usb-kws-mnormal), TYPE=CONTACT SURFACE ``` |
| --- | --- |

### 单元

对于梁和壳单元、线弹簧单元、垫片单元或参与轮廓积分评估的单元，可能需要用户指定的法线。在这种情况下，请在全局坐标系中指定法线的分量。

| **输入文件用法：** | ``` [*NORMAL*](../key/key-link.md#usb-kws-mnormal), TYPE=ELEMENT ``` |
| --- | --- |

#### 梁和壳单元

可能需要用户指定的法线来定义壳曲面交点或梁交点处所需的法线方向，此时自动确定的法线可能不适合模型（参见["梁单元横截面方向，"第29.3.4节"](pt06ch29s03alm09.md)或["定义常规壳单元的初始几何，"第29.6.3节"](pt06ch29s06alm17.md)）。

节点法线也可以作为节点定义的一部分定义。虽然您可以为连接到节点的的所有单元定义单个法线作为节点定义的一部分，但用户指定的法线定义为节点处的特定单元定义法线，从而允许您为连接到节点的每个单元定义单独的法线。用户指定的法线定义覆盖作为节点定义一部分定义的法线。

| **输入文件用法：** | ``` [*NODE*](../key/key-link.md#usb-kws-mnode) *在数据行的第五、第六和第七个位置指定法线。* ``` |
| --- | --- |
|  | 例如，以下行将一些法线定义为节点定义的一部分；在单元2中节点7要使用的法线随后使用用户指定的法线定义重新定义： ``` [*NODE*](../key/key-link.md#usb-kws-mnode) 6, 5., 5., , -0.5, .8 7, 10., 8., , -0.5, .8 9, 14., 4., , .6, .6 [*NORMAL*](../key/key-link.md#usb-kws-mnormal) 2, 7, .6, .6 ``` |

#### 线弹簧单元

对于线弹簧单元，用户指定的法线可用于给出结构中缺陷的法线方向。有关这些单元的描述，请参见["用于建模壳中部分穿透裂纹的线弹簧单元，"第32.9.1节"](pt06ch32s09alm54.md)。

#### 垫片单元

对于垫片单元，用户指定的法线可用于指定单元的厚度方向。节点厚度方向也可以作为垫片截面定义的一部分定义。用户指定的法线定义的厚度方向覆盖作为垫片截面定义一部分定义的厚度方向。有关这些单元厚度方向定义的描述，请参见["定义垫片单元的初始几何，"第32.6.4节"](pt06ch32s06alm49.md)。

#### 轮廓积分评估

对于轮廓积分评估（["轮廓积分评估，"第11.4.2节"](pt04ch11s04aus68.md)），应在位于所请求轮廓范围内的所有曲面节点处指定曲面法线。这些节点在数据（`.dat`）文件的"轮廓积分"信息下打印。对于准确的轮廓积分评估，对于以下情况，虚拟裂纹扩展方向位于曲面平面内是很重要的：当裂纹前缘与三维实体的外表面相交时，当裂纹前缘与材料不连续表面相交时，或者当裂纹位于弯曲壳中时。如果未指定法线，Abaqus将自动计算法线。

作为节点定义一部分指定的节点法线数据将不会被激活用于实体单元，除非在模型中使用了用户指定的法线定义；仅在一个节点上包含用户指定的法线定义就足以激活作为节点定义一部分指定的节点法线数据的使用。

### 法线定义的坐标系

Abaqus模型可以以部件实例的装配体形式定义（参见["定义装配体，"第2.10.1节"](pt01ch02s10aus28.md)）。在部件（或部件实例）内定义的节点法线是相对于部件坐标系定义的。这些法线根据为每个部件实例给出的定位数据旋转。如有需要，可以在装配级别在参考节点处定义法线。在装配级别定义的法线是在全局坐标系中定义的。

对于不是以部件实例装配体形式定义的模型，法线在全局坐标系中定义。
