# 42.7.1 构建有效的场输出表达式

要通过对单个现有场输出变量进行运算来定义新的场输出变量，您可以在**Create Field Output**对话框的表达式文本字段中[构建算术表达式](pt05ch42s05hlb04.md)。

表达式由一个或多个现有场输出变量标签以及可选的一个或多个运算符、变换和标量组成。请参阅["场输出运算概述，" 第42.7.2节](pt05ch42s05hlb02.md)，了解支持的运算符的信息。表达式作为Python输入进行求值；包含语法错误的条目将生成非特定Python异常。

**Create Field Output**对话框按变量名称和由Abaqus/CAE创建的唯一"标签"列出场输出。给定场输出变量的标签由预先添加到输出变量名称的步骤和帧编号组成；例如，`s1f1_RF`是步骤1、帧1处的反作用力。此标签使您能够在单个表达式中组合来自不同步骤和帧的输出。

场输出变量可能包含可能不兼容的不同类型的数据。**Create Field Output**对话框将每个场输出变量的类型列为三种一般类型之一：标量、矢量和张量。张量类型进一步细分为五个子类型，描述张量的维度和可用分量。以下规则适用：
- 表达式中只能出现类似的数据对象类型和子类型。
- 复数数据的运算仅使用实部。
- 两个矢量对象之间或两个张量对象之间不支持乘法和除法运算。
- 张量运算在Abaqus/CAE从输出数据库读取的张量分量数据上执行。随后，显示结果场输出变量可能会为外推分量或计算的不变量给出意外值。例如，如果应力张量的分量为负，则对应力张量应用绝对值运算将产生正值的应力分量，但绝对值运算之后计算的压力（不变量）可能为负。类似地，对应力张量应用正弦运算将产生在范围{-1, 1}内的分量值，但用于计算此类分量等值线值的外推可能产生超出此范围的值。
- 不支持对与不同区域关联的场进行运算。

以下示例演示了有效的场输出表达式：

**示例1**

要通过查找两个增量之间的应力场差异来创建场输出变量，请在**Create Field Output**对话框的表达式文本字段中键入：

```
s1f2_S - s1f1_S
```

`s1f2_S`和`s1f1_S`是表示特定步骤两个不同增量处的应力的场输出变量。此方程的结果是表示两个增量之间应力场差异的场输出变量。

**示例2**

要通过声压的分贝来表示压力作为声压的函数来创建场输出变量，请在**Create Field Output**对话框的表达式文本字段中键入：

```
20.0 * log10(s1f1_POR/Pref)
```

`Pref`是参考压力。此方程的结果是表示作为分析变量POR函数的分贝压力的场输出变量。

**示例3**

要通过将张量结果变换到用户指定的坐标系来创建场输出变量：

1. 从**Create Field Output**对话框右侧的"Function"列表中选择**Transformation**。
2. 从场输出变量列表中选择张量结果；例如，`s1f10_S`。
3. 选择要应用的坐标系变换；例如，名为`CSYS-1`的固定坐标系。表达式``` s1f10_S.getTransformedField(datumCsys=o_CSYS_1) ```出现在表达式文本字段中。

此方程的结果是表示将应力张量`s1f10_S`变换到`CSYS-1`坐标系的场输出变量。
