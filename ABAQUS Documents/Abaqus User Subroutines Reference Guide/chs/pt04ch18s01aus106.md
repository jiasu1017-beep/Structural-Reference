# 18.1.3 可用的实用程序例程





**产品：** Abaqus/Standard  Abaqus/Explicit  Abaqus/Aqua

### 概述

有多种实用程序例程可用于协助编写用户子程序。调用时，实用程序例程将执行预定义的功能或操作，其输出或结果可以集成到用户子程序中。

### 可用的实用程序例程

以下实用程序例程可用于在Abaqus中编写用户子程序：
- `GETENVVAR`或`VGETENVVAR`可分别从任何Abaqus/Standard或Abaqus/Explicit用户子程序调用，以获取环境变量的值（["获取Abaqus环境变量，" Abaqus用户子程序参考指南第2.1.1节](../sub/sub-link.md#sub-utl-ugetenvvar)）。
- `GETJOBNAME`或`VGETJOBNAME`可分别从任何Abaqus/Standard或Abaqus/Explicit用户子程序调用，以获取当前分析作业的名称（["获取Abaqus作业名称，" Abaqus用户子程序参考指南第2.1.2节](../sub/sub-link.md#sub-utl-ugetjobname)）。
- `GETOUTDIR`或`VGETOUTDIR`可分别从任何Abaqus/Standard或Abaqus/Explicit用户子程序调用，以获取放置分析作业输出的目录名称（["获取Abaqus输出目录名称，" Abaqus用户子程序参考指南第2.1.3节](../sub/sub-link.md#sub-utl-ugetoutdir)）。
- `GETNUMCPUS`可从任何Abaqus/Standard用户子程序调用以获取MPI进程数；`VGETNUMCPUS`可从域并行运行中的任何Abaqus/Explicit用户子程序调用以获取用于并行运行的进程数（["获取并行进程信息，" Abaqus用户子程序参考指南第2.1.4节](../sub/sub-link.md#sub-utl-ugetnumcpus)）。
- `GETRANK`可从任何Abaqus/Standard用户子程序调用以获取调用该函数的MPI进程的rank；`VGETRANK`可从域并行运行中的任何Abaqus/Explicit用户子程序调用以获取单个进程rank（["获取并行进程信息，" Abaqus用户子程序参考指南第2.1.4节](../sub/sub-link.md#sub-utl-ugetnumcpus)）。
- `get_thread_id`可从任何Abaqus用户子程序调用以检索Abaqus分配给该线程的ID（["获取并行进程信息，" Abaqus用户子程序参考指南第2.1.4节](../sub/sub-link.md#sub-utl-ugetnumcpus)）。
- `GETPARTINFO`或`VGETPARTINFO`可分别从任何Abaqus/Standard或Abaqus/Explicit用户子程序调用，以检索与内部节点或单元编号对应的部件实例名称和局部节点或单元编号。`GETINTERNAL`或`VGETINTERNAL`可分别从任何Abaqus/Standard或Abaqus/Explicit用户子程序调用，以检索与给定部件实例名称和局部编号对应的内部节点或单元编号（["获取部件信息，" Abaqus用户子程序参考指南第2.1.5节](../sub/sub-link.md#sub-utl-ugetpartinfo)。）
- `GETVRM`提供对Abaqus/Standard用户子程序[`UVARM`](../sub/sub-link.md#sub-xsl-uvarm)、[`UDMGINI`](../sub/sub-link.md#sub-xsl-udmgini)或[`USDFLD`](../sub/sub-link.md#sub-xsl-usdfld)的材料点信息的访问（["在Abaqus/Standard分析中获取材料点信息，" Abaqus用户子程序参考指南第2.1.6节](../sub/sub-link.md#sub-utl-ugetvrm)）。
- `VGETVRM`提供对Abaqus/Explicit用户子程序[`VUSDFLD`](../sub/sub-link.md#sub-xsl-vusdfld)的材料点上选定输出变量的访问（["在Abaqus/Explicit分析中获取材料点信息，" Abaqus用户子程序参考指南第2.1.7节](../sub/sub-link.md#sub-utl-uvgetvrm)）。
- `GETVRMAVGATNODE`提供对材料点信息（外推并平均到节点处）的访问，适用于Abaqus/Standard用户子程序[`UMESHMOTION`](../sub/sub-link.md#sub-xsl-umeshmotion)（["在节点处获取平均的材料点信息，" Abaqus用户子程序参考指南第2.1.8节](../sub/sub-link.md#sub-utl-ugetvrmavgatnode)）。
- `GETVRN`提供对节点点信息的访问，适用于Abaqus/Standard用户子程序[`UMESHMOTION`](../sub/sub-link.md#sub-xsl-umeshmotion)（["获取节点点信息，" Abaqus用户子程序参考指南第2.1.9节](../sub/sub-link.md#sub-utl-ugetvrn)）。
- `GETNODETOELEMCONN`可从用户子程序[`UMESHMOTION`](../sub/sub-link.md#sub-xsl-umeshmotion)调用，以检索连接到特定节点的单元列表。然后可以将此单元列表与实用程序例程`GETVRMAVGATNODE`一起使用（["获取节点到单元的连接，" Abaqus用户子程序参考指南第2.1.10节](../sub/sub-link.md#sub-utl-ugetnodetolemconn)）。
- `SINV`确定Abaqus/Standard中给定应力张量的第一和第二应力不变量（["在Abaqus/Standard分析中获取应力不变量、主应力/应变值和方向以及旋转张量，" Abaqus用户子程序参考指南第2.1.11节](../sub/sub-link.md#sub-utl-utensor)）。
- `SPRINC`或`VSPRINC`分别确定Abaqus/Standard或Abaqus/Explicit中给定应力或应变张量的主值（["在Abaqus/Standard分析中获取应力不变量、主应力/应变值和方向以及旋转张量，" Abaqus用户子程序参考指南第2.1.11节](../sub/sub-link.md#sub-utl-utensor)，以及["在Abaqus/Explicit分析中获取主应力/应变值和方向，" Abaqus用户子程序参考指南第2.1.12节](../sub/sub-link.md#sub-utl-uvtensor)）。
- `SPRIND`或`VSPRIND`分别确定Abaqus/Standard或Abaqus/Explicit中给定应力或应变张量的主值和主方向（["在Abaqus/Standard分析中获取应力不变量、主应力/应变值和方向以及旋转张量，" Abaqus用户子程序参考指南第2.1.11节](../sub/sub-link.md#sub-utl-utensor)，以及["在Abaqus/Explicit分析中获取主应力/应变值和方向，" Abaqus用户子程序参考指南第2.1.12节](../sub/sub-link.md#sub-utl-uvtensor)）。
- `ROTSIG`可从Abaqus/Standard用户子程序[`UMAT`](../sub/sub-link.md#sub-xsl-umat)调用，以在执行大应变计算时执行张量旋转（["在Abaqus/Standard分析中获取应力不变量、主应力/应变值和方向以及旋转张量，" Abaqus用户子程序参考指南第2.1.11节](../sub/sub-link.md#sub-utl-utensor)）。
- `GETWAVE`确定与Abaqus/Aqua分析中应用的波浪理论相关的波浪运动数据（["在Abaqus/Aqua分析中获取波浪运动数据，" Abaqus用户子程序参考指南第2.1.13节](../sub/sub-link.md#sub-utl-uwavekinematic)）。
- `GETWAVEVEL`、`GETWINDVEL`和`GETCURRVEL`分别用于获取Abaqus/Aqua分析中给定点的波浪、风和稳态流速分量（["在Abaqus/Aqua分析中获取波浪运动数据，" Abaqus用户子程序参考指南第2.1.13节](../sub/sub-link.md#sub-utl-uwavekinematic)）。
- `STDB_ABQERR`或`XPLB_ABQERR`可分别从任何Abaqus/Standard或Abaqus/Explicit用户子程序调用，以将信息性、警告或错误消息打印到Abaqus/Standard的消息文件或Abaqus/Explicit的状态文件（["向消息或状态文件打印消息，" Abaqus用户子程序参考指南第2.1.14节](../sub/sub-link.md#sub-utl-uabqerr)）。
- `XIT`或`XPLB_EXIT`可分别从任何Abaqus/Standard或Abaqus/Explicit用户子程序调用，以终止分析（["终止分析，" Abaqus用户子程序参考指南第2.1.15节](../sub/sub-link.md#sub-utl-uxit)）。
- `IGETSENSORID`或`IVGETSENSORID`可分别从Abaqus/Standard用户子程序[`UAMP`](../sub/sub-link.md#sub-xsl-uamp)或Abaqus/Explicit用户子程序[`VUAMP`](../sub/sub-link.md#sub-xsl-vuamp)调用，以获取用户定义的传感器的ID。`GETSENSORVALUE`或`VGETSENSORVALUE`可分别从Abaqus/Standard用户子程序[`UAMP`](../sub/sub-link.md#sub-xsl-uamp)或Abaqus/Explicit用户子程序[`VUAMP`](../sub/sub-link.md#sub-xsl-vuamp)调用，以获取用户定义的传感器的值（["获取传感器信息，" Abaqus用户子程序参考指南第2.1.16节](../sub/sub-link.md#sub-utl-ugetsensor)）。
- `MATERIAL_LIB_MECH`可从Abaqus/Standard用户子程序[`UELMAT`](../sub/sub-link.md#sub-xsl-uelmat)调用以访问Abaqus材料库（["访问Abaqus材料，" Abaqus用户子程序参考指南第2.1.17节](../sub/sub-link.md#sub-utl-ugetmaterialmech)）。
- `MATERIAL_LIB_HT`可从Abaqus/Standard用户子程序[`UELMAT`](../sub/sub-link.md#sub-xsl-uelmat)调用以访问Abaqus热材料库（["访问Abaqus热材料，" Abaqus用户子程序参考指南第2.1.18节](../sub/sub-link.md#sub-utl-ugetmaterialht)）。
- `SMACfdUserSubroutineGetScalar`可从任何Abaqus/CFD用户子程序调用，以访问作为边界条件定义一部分的单元或表面面元的选定输出变量（["在Abaqus/CFD分析中获取标量状态信息，" Abaqus用户子程序参考指南第2.1.19节](../sub/sub-link.md#sub-utl-ucfdgetscalar)）。
- `SMACfdUserSubroutineGetVector`可从任何Abaqus/CFD用户子程序调用，以访问作为边界条件定义一部分的单元和表面面元的选定输出变量（["在Abaqus/CFD分析中获取矢量状态信息，" Abaqus用户子程序参考指南第2.1.20节](../sub/sub-link.md#sub-utl-ucfdgetvector)）。
- `SMACfdUserSubroutineGetMpiComm`可从任何Abaqus/CFD用户子程序内部调用，以获取在并行分析作业中使用的MPI通信器（["在Abaqus/CFD分析中获取MPI通信器，" Abaqus用户子程序参考指南第2.1.21节](../sub/sub-link.md#sub-utl-ucfdgetmpicomm)）。
- `MutexInit`、`MutexLock`和`MutexUnlock`可用于创建和操作互斥锁（["确保线程安全，" Abaqus用户子程序参考指南第2.1.22节](../sub/sub-link.md#sub-utl-ugetthreadid)）。
- 线程本地数组可用于分配存储。`SMALocalIntArrayCreate`和`SMALocalFloatArrayCreate`可用于创建或调整本地数组的大小。`SMALocalIntArrayAccess`和`SMALocalFloatArrayAccess`可用于定位现有的本地数组。`SMALocalIntArrayDelete`和`SMALocalFloatArrayDelete`可用于删除本地数组。`SMALocalIntArraySize`和`SMALocalFloatArraySize`可用于获取数组的大小（["可分配数组，" Abaqus用户子程序参考指南第2.1.23节](../sub/sub-link.md#sub-utl-localarrays)）。
- 全局数组也可用于分配存储。`SMAIntArrayCreate`和`SMAFloatArrayCreate`可用于创建或调整全局数组的大小。`SMAIntArrayAccess`和`SMAFloatArrayAccess`可用于定位现有的全局数组。`SMAIntArrayDelete`和`SMAFloatArrayDelete`可用于删除全局数组。`SMAIntArraySize`和`SMAFloatArraySize`可用于获取全局数组的大小（["可分配数组，" Abaqus用户子程序参考指南第2.1.23节](../sub/sub-link.md#sub-utl-localarrays)）。
