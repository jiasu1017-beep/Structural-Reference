# 16.11.5 Defining a fluid density field







You can create a fluid density field to define the initial value of the fluid density in an Abaqus/CFD analysis. For more information, see [Chapter 30, "Fluid dynamic analyses](pt04ch30.md).”

**To create or edit a fluid density field:**

1. Display the fluid density field editor using one of the following methods: - To create a new fluid density field, follow the procedure outlined in ["Creating predefined fields," Section 16.8.3](pt03ch16s08hlb03.md) (**Category**: `Fluid`; **Types for Selected Step**: `Fluid density`). - To edit an existing fluid density field using menus or managers, see ["Editing step-dependent objects," Section 3.4.13](pt01ch03s04hlb05.md). **Note:**You can create or edit a fluid density field only in the initial step.
2. By default, the fluid density field is applied to the whole model. If you want to change the region for which the fluid density field applies, do the following: 1. Click ![](../graphics/ico_selectBlue.png). 2. Select one or more regions or sets from the viewport. 3. From the prompt area, click **Done**. Abaqus/CAE updates the fluid density field editor to reflect your selection.
3. In the **Density** field, enter the initial value of the fluid density.
4. Click **OK** to save your data and to exit the editor.
