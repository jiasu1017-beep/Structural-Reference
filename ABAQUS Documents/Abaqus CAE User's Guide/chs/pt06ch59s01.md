# 59.1 了解附件点和线





您使用附件工具集向模型添加附件点和线。您使用附件点来定义模型中的点式紧固件、惯性、弹簧、阻尼器、载荷或边界条件，或连接定义的连接点；您使用附件线来定义离散紧固件。紧固件对点对点连接（如点焊、铆钉和螺栓）进行建模，相关描述见["关于紧固件，" 第29.1节](pt04ch29s01.md)。您可以在装配上或零件上创建附件点。您只能在装配上创建附件线。

您可以使用附件工具集通过从主菜单栏选择 ****Tools**![](../graphics/images/arrow.gif)**Attachment**** 并选择可用的菜单选项来创建附件点和线。在 Interaction 模块中，您可以使用模块工具箱中的工具来创建附件点和线。三个附件点工具也可从 Property 模块和 Part 模块工具箱中获得。附件工具集提供以下工具来创建附件点：

**![](../graphics/ico_attPtByPick.png) 通过拾取或从文件读取创建附件点**

此工具允许您通过从视口拾取每个点或从文件读取点的坐标来创建附件点。详细说明请参见["通过拾取或从文件读取创建附件点，" 第59.3节](pt06ch59hla01.md)。

**![](../graphics/ico_attPtByDir.png) 通过选择方向和间距创建附件点**

此工具允许您通过定义一条线并指定沿该线的点数或指定沿该线的点间距来创建附件点。详细说明请参见["通过选择方向和间距创建附件点，" 第59.4节](pt06ch59hla02.md)。

**![](../graphics/ico_attPtOffEdge.png) 通过选择边缘和偏移创建附件点**

此工具允许您通过选择单个边缘或相连边缘，然后定义图案参数，在边缘上、面或沿方向创建简单的附件点图案。详细说明请参见["基于边缘创建附件点图案，" 第59.5节](pt06ch59hla03.md)。

如果需要，您可以将点移动到指定的面，如["了解投影方法，" 第59.2节](pt06ch59s02.md) 中所述。

附件工具集提供以下工具来创建附件线：

**![](../graphics/ico_attLineByProj.png) 通过投影点创建附件线**

此工具允许您通过指定点然后通过多个面投影它们来创建附件线。附件线连接一个或多个面。您只能在装配上创建附件线；您不能在零件上创建附件线。创建附件线是一个三步过程：

1. 选择要投影的点。
2. 沿表面的法向或沿指定方向将点投影到源面上。
3. 从投影点与源面的交点开始，沿源面法向将附件线投影到目标面，投影指定距离。

详细说明请参见["通过投影点创建附件线，" 第59.6节](pt06ch59hla04.md)。

附件点和线是特征，当您更改模型时会重新生成；例如，在添加切割特征之后。重新生成可能会影响附件点和线的投影，并改变附件特征创建的紧固点和线的总数。此外，包含附件的集合可能会受到重新生成的影响。因此，在更改模型后，您应该确定附件点和线的行为符合预期。您还应该检查您已应用于附件点和线的任何规定条件（如载荷和边界条件）是否未发生变化。
