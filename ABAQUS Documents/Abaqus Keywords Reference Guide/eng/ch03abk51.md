# *CONNECTOR STOP







### *CONNECTOR STOPSpecify connector stops for connector elements.

This option is used to define connector stops for connector elements.

**Products: **Abaqus/Standard  Abaqus/Explicit  Abaqus/CAE  

**Type: **Model data  

**Level: **Model  

**Abaqus/CAE: **Interaction module

### **Required parameter: **

COMPONENT

Set this parameter equal to the connector's available component of relative motion number for which connector stops are defined.
