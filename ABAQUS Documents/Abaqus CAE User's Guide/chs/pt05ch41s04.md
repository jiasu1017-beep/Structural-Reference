# 41.4 访问诊断信息





您可以使用**作业诊断**对话框访问Abaqus在分析作业期间写入输出数据库的诊断信息。

**访问作业诊断：**

1. 从可视化模块的主菜单栏中，选择**工具**![](../graphics/images/arrow.gif)**作业诊断**。出现**作业诊断**对话框。
2. 单击**作业历史**树中的"+"符号以展开树，查看作业历史中的步骤、增量、尝试和迭代。
3. 单击**作业历史**树中的项目以高亮显示它。Abaqus/CAE在对话框右侧显示包含与所选项目关联的诊断信息的附加选项卡。
4. 单击对话框右侧的选项卡。Abaqus/CAE显示与树中选定的项目对应的各种类型的诊断信息。以下部分描述了每个选项卡页面中可用的信息：- ["诊断摘要"，第41.3.1节](pt05ch41s03hlb01.md)- ["警告和错误"，第41.3.3节](pt05ch41s03hlb03.md)- ["残差"，第41.3.4节](pt05ch41s03hlb04.md)- ["接触"，第41.3.5节](pt05ch41s03hlb05.md)- ["单元"，第41.3.6节](pt05ch41s03hlb06.md)- ["其他"，第41.3.7节](pt05ch41s03hlb07.md)
5. 单击**关闭**以关闭**作业诊断**对话框。
