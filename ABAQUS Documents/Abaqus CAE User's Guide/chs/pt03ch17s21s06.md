# 17.21.6 手动调整大小和重新网格划分





为了了解重新网格划分规则对 Abaqus/CAE 生成的网格的影响，您可以手动应用规则并查看其对生成网格的影响。但是，在重新网格划分模型之前，您必须运行分析并创建包含误差指示器输出变量的输出数据库。然后您可以更改尺寸方法和约束，并查看修改后的规则对生成网格的影响。当您确信重新网格划分规则捕获了您的意图时，可以使用相同的规则来控制由自动重新网格划分驱动的多次迭代。您还可以使用手动重新网格划分来继续过早结束的自适应过程。

有关更多信息，请参见["自动自适应重新网格划分和手动自适应重新网格划分有什么区别？，" 第 17.13.4 节](pt03ch17s13s04.md)和["何时需要使用手动自适应重新网格划分？，" 第 17.13.5 节](pt03ch17s13s05.md)。

**手动调整大小和重新网格划分的步骤：**

1. 从主菜单栏，选择 ****自适应**![](../graphics/images/arrow.gif)**手动自适应重新网格划分****。Abaqus/CAE 显示"手动自适应重新网格划分"对话框。**提示：**您也可以使用位于网格工具箱中 Mesh 模块工具箱中的 ![](../graphics/ico_adaptiveRemesh.png) 工具来手动调整大小和重新网格划分。（有关更多信息，请参见["使用 Mesh 模块工具箱，" 第 17.15 节](pt03ch17s15.md)。）
2. 在 **ODB** 字段中，输入包含误差指示器输出变量的输出数据库的名称。Abaqus/CAE 使用误差指示器来驱动重新网格划分算法。Abaqus/CAE 显示有关输出数据库中每个重新网格划分规则的以下信息：- 规则名称。- 规则使用的误差指示器输出变量的描述。- 百分比目标误差。这是创建重新网格划分规则时指定的值。如果您选择了固定目标，Abaqus/CAE 显示最大和/或最小基础解目标。如果您选择了**默认方法和参数**，Abaqus/CAE 显示破折号；如果您选择了**自动目标减少**，则显示**自动**。- 规则使用的尺寸方法。
3. 如果需要，单击**显示误差**。Abaqus/CAE 显示每个误差指示器输出变量的百分比**误差指示器结果**和**单元计数**。单元计数是应用重新网格划分规则的区域中的单元数。
4. 单击**手动自适应重新网格划分**。Abaqus/CAE 重新网格划分模型。所有活动（未抑制的）重新网格划分规则都会对模型的重新网格划分做出贡献。如果您将多个重新网格划分规则应用于同一区域，则产生最细网格的规则优先。
5. 如果需要，您可以从主菜单选择 ****自适应**![](../graphics/images/arrow.gif)**重新网格划分规则**![](../graphics/images/arrow.gif)**编辑**![](../graphics/images/arrow.gif)***规则名称***** 并修改定义规则的参数。然后您可以返回"手动自适应重新网格划分"对话框，查看修改后的规则对误差估计和生成网格的影响。**注意：**当您编辑重新网格划分规则时，您可以修改参数，例如重新网格划分规则尺寸方法和单元尺寸约束，并查看更改对生成网格的影响。但是，如果更改步骤或误差指示器输出变量，则必须重新运行分析。
6. 您可以修改重新网格划分规则并手动重新网格划分模型，直到对生成的网格满意为止。然后您可以使用相同的规则来驱动由 Abaqus/CAE 控制的迭代重新网格划分操作序列。有关更多信息，请参见["创建、编辑和操作自适应过程，" 第 19.9 节](pt03ch19s09.md)。
