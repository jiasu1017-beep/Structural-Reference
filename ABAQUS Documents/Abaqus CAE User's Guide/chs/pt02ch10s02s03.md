# 10.2.3 处理无效零件





如果几何错误阻止Abaqus/CAE为实体或壳零件重新创建封闭体积，则该零件被视为无效。在大多数情况下，零件无效是因为它们在导入Abaqus/CAE时存在错误。由于Abaqus/CAE不同版本之间对几何的ACIS解释发生变化，零件也可能改变有效性。如果您有对分析至关重要且无法修复的无效零件，则可以忽略无效零件的状态。如果有效零件在数据库升级期间变为无效，您可以在不解锁零件的情况下忽略无效几何体；这使您可以在需要时重新对零件进行网格划分而不重新生成它。

您可以使用两种方法来忽略零件无效的状态：
- 在**零件管理器（Part Manager）**中选择无效零件，然后单击**忽略无效性（Ignore Invalidity）**。
- 在模型树中选择无效零件或无效独立零件实例，单击鼠标按钮3，然后选择**忽略无效性（Ignore Invalidity）**。

当您选择忽略零件或零件实例的无效状态时，Abaqus/CAE会显示警告。警告表明，忽略零件的无效性允许您执行所有几何和网格划分功能，但忽略无效性可能导致某些零件功能失败或导致其他不良行为。如果您接受此警告，Abaqus/CAE会将零件状态更改为**无效（已忽略）（Invalid (ignored)）**。零件状态显示在**零件管理器（Part Manager）**和模型树中。[表10-1](pt02ch10s02s03.md#usi-imp-table-invalid)显示了用于指示无效零件状态的图标和文本。

无论您是尝试修复几何体还是忽略无效状态工作，Abaqus/CAE并不总是在您进行更改时重新计算零件的有效性。重新计算有效性，特别是对于复杂零件，可能需要大量时间。要重新计算并更新零件的有效性，请在**零件管理器（Part Manager）**中选择**更新有效性（Update Validity）**，或在模型树中无效零件或零件实例上单击鼠标按钮3时出现的菜单中选择。您也可以使用**几何诊断**查询来更新有效性。（有关几何诊断的更多信息，请参阅["使用几何诊断工具，"第71.2.4节"](pt06ch71s02hlb04.md)。)
