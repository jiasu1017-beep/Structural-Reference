# 64.6.10 重新编号单元







从主菜单栏中选择 ****Mesh（网格）**![](../graphics/images/arrow.gif)**Edit（编辑）**** 可对选定单元进行重新编号。您可以通过指定起始标签和增量来重新编号选定的单元，也可以通过指定一个值来偏移现有标签。（有关对 Abaqus/CAE 原生网格的单元进行重新编号的信息，请参见["更改所有节点和单元的标签，" 第 17.18.11 节](pt03ch17s18s09.md)。）

**重新编号单元：**

1. 进入 Mesh 模块。
2. 从上下文栏的 **Object** 字段中选择 **Part**，然后从列表中选择一个零件。此工具仅适用于不包含任何几何体的零件。
3. 从主菜单栏中选择 ****Mesh**![](../graphics/images/arrow.gif)**Edit****。Abaqus/CAE 显示 **Edit Mesh** 对话框。**提示：**您也可以使用位于 Mesh 模块工具箱底部的 ![](../graphics/ico_meshEdit.png) 工具来显示 **Edit Mesh** 对话框。
4. 在对话框中执行以下操作：1. 在 **Category** 字段中选择 **Element**。2. 从 **Method** 列表中选择 **Renumber**。Abaqus/CAE 显示 **Element Renumbering** 对话框。**Part Label Range** 显示整个零件的单元标签范围。
5. 执行以下任一操作来指定重新编号方法：- 选择 **By Start Label**，然后输入第一个单元标签的值以及用于对后续单元进行编号的增量。- 选择 **By Offset**，然后输入现有单元标签将被偏移的值。
6. 使用以下方法之一来选择要重新编号的单元：**选择所有单元** - 选择 **All** 以重新编号所有孤立网格单元。**使用无序方法选择单元** 1. 选择 **Specify** 和 **Unordered** 以任意顺序选择单元。2. 点击 **Select** 选择要重新编号的单元。3. 从提示区域选择选择方法，然后选择要重新编号的单元。有关更多信息，请参见["选择要编辑的单元，" 第 64.6.11 节](pt06ch64s06s11.md)。**使用有向路径方法选择单元** 1. 选择 **Specify** 和 **Directed Path** 以沿边从选定的起始单元到选定的结束单元对单元进行重新编号。2. 点击 **Select** 选择要重新编号的单元。3. 选择起始单元。Abaqus/CAE 显示沿特征边的所有单元。如果需要，可修改角度以重新定义 Abaqus/CAE 识别特征边的方式。4. 从提示区域选择 **Done**。5. 选择结束单元。起始单元和结束单元必须位于同一特征边上。**使用序列方法选择单元** 1. 选择 **Specify** 和 **Sequence** 以根据选择单元的顺序对单元进行重新编号。2. 点击 **Select** 选择要重新编号的单元。3. 选择起始单元。Abaqus/CAE 显示沿特征边的所有单元。如果需要，可修改角度以重新定义 Abaqus/CAE 识别特征边的方式。4. 从提示区域选择 **Done**。起始单元以紫色显示。5. 选择结束单元。起始单元和结束单元必须位于同一特征边上。**Part Label Range** 显示重新编号后单元的标签范围。
7. 点击 **OK** 或 **Apply** 以重新编号单元。如果您点击 **Apply**，Abaqus/CAE 将重新编号单元；但这些单元仍保持选中状态，您可以继续对它们进行重新编号。
8. 根据需要重复上述步骤以重新编号其他单元。
9. 完成重新编号单元后，点击 **Cancel** 以关闭 **Element Renumbering** 对话框。
