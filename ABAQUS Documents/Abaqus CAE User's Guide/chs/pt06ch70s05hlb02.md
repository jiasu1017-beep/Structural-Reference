# 70.5.2 使用输入参数方法分区边







您可以通过输入表示边长度分数的参数来沿所选边的任意长度分区边。沿所选边的箭头从起始顶点（对应于边参数值零）指向端顶点（对应于值一）。分区沿所选边在由参数决定的位置创建（如下图所示）：

![](../graphics/par-edge-parameter-nls.png)

要修改通过输入参数创建的分区，请使用特征操作工具集更改参数。

**使用输入参数方法分区边：**

1. 从主菜单栏中选择 ****Tools**![](../graphics/images/arrow.gif)**Partition****（工具→分区）。出现 **Create Partition**（创建分区）对话框。Abaqus/CAE 在提示区域显示提示以指导您完成过程。**提示：**您也可以使用位于模块工具箱分区边工具处的 ![](../graphics/ico_ptnEdgeParameter.png) 工具来使用输入参数方法分区边。 有关工具箱中分区工具的图表，请参阅["使用分区工具集，" 第 70.2 节"](pt06ch70s02.md)。
2. 从对话框顶部的 **Type**（类型）单选按钮中选择 **Edge**（边）。**Method**（方法）列表显示可用于分区边的方法。
3. 从方法列表中选择 **Enter Parameter**（输入参数）。
4. 如果部件或装配包含多个边，选择要分区的边。您可以使用拖动选择、**[Shift]****+点击**、**[Ctrl]****+点击** 和角度方法的组合来选择多个要分区的边。 有关更多信息，请参阅["在当前视口中选择对象，" 第 6.2 节"](pt01ch06s02.md)。**提示：**如果您无法选择所需的边，可以使用 **Selection**（选择）工具栏更改选择行为。 有关更多信息，请参阅["使用选择选项，" 第 6.3 节"](pt01ch06s03.md)。Abaqus/CAE 高亮显示所选边，箭头指示参数值增加的方向。
5. 在提示区域中点击 **Done**（完成）以表明您已完成选择边。
6. 在提示区域中输入所需的边参数，值介于零和一之间。
7. 在提示区域中点击 **Create Partition**（创建分区）。Abaqus/CAE 创建分区。
