# 11.23.3 圆化线框中的顶点







从主菜单栏选择**形状**![](../graphics/images/arrow.gif)**线框**![](../graphics/images/arrow.gif)**圆化**，对线框零件中两条边缘之间的顶点进行"圆化"或倒圆角。

**圆化线框顶点：**

1. 从主菜单栏，选择**形状**![](../graphics/images/arrow.gif)**线框**![](../graphics/images/arrow.gif)**圆化**。Abaqus/CAE在提示区域显示引导您完成该过程的提示。**提示：**您也可以使用零件模块工具箱中位于线框工具旁边的![](../graphics/ico_partWireRound.png)工具对线框零件中两条边缘之间的顶点进行圆化。有关零件模块工具箱中工具的图表，请参见"使用零件模块工具箱"，第11.17节。Abaqus/CAE会提示您选择要圆化的顶点，并在提示区域显示引导您完成该过程的提示。
2. 选择要圆化的顶点。您可以**[Shift]**+单击**添加更多顶点到您的选择中，并**[Ctrl]**+单击**已选择的顶点以取消选择。**提示：**如果您无法选择所需的顶点，可以使用**选择**工具栏更改选择行为。有关更多信息，请参见"使用选择选项"，第6.3节。
3. 完成选择顶点后，在提示区域单击**完成**。提示区域中会出现一个默认半径。
4. 如有必要，在提示区域的文本字段中输入新的半径。
5. 单击**完成**接受新半径并完成圆化过程。Abaqus/CAE使用所选顶点圆化后的零件重新绘制图形。
