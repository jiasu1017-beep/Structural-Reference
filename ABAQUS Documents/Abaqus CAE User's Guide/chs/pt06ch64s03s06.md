# 64.3.6 使用工具组合用四面体单元网格化导入的实体零件





在某些情况下，由于表面网格中存在非常薄的三角形单元或某些细长面无法用三角形进行网格划分，您可能无法用四面体单元对导入的实体零件进行网格划分。以下过程说明瞭如何使用"网格"模块中的工具组合成功对零件进行网格划分：

**用四面体单元对导入的实体零件进行网格划分的步骤：**

1. 执行以下操作之一： - 从四面体边界网格开始。转到"网格"模块，在实体上创建四面体边界网格。 - 从线性三角形网格开始。 1. 通过从主菜单栏选择****形状**![](../graphics/images/arrow.gif)**壳**![](../graphics/images/arrow.gif)**从实体****将实体零件转换为壳零件。 2. 转到"网格"模块，用线性三角形单元对壳零件进行网格划分。
2. 从主菜单栏选择****网格**![](../graphics/images/arrow.gif)**创建网格零件****创建一个新的孤兒网格零件。有关更多信息，请参见["创建网格零件，" 第17.20节](pt03ch17s20.md)。
3. 将显示的对象更改为孤兒网格。
4. 从主菜单栏选择****工具**![](../graphics/images/arrow.gif)**编辑网格****，并执行以下操作来清理网格： 1. 从**类别**字段中选择**网格**。 2. 从**方法**列表中选择**折叠边缘**。清理网格会自动合并短单元边缘上的节点并移除坍缩的单元。有关更多信息，请参见["折叠线性孤兒网格的短边缘，" 第64.7.4节](pt06ch64s07s04.md)。
5. 使用网格编辑工具集手动修复任何 remaining 的错误单元或间隙。有关更多信息，请参见["编辑单元，" 第64.6节](pt06ch64s06.md)。
6. 从主菜单栏选择****工具**![](../graphics/images/arrow.gif)**编辑网格****，并选择**转换**以将线性三角形壳网格替换为线性四面体实体网格。有关更多信息，请参见["将三角形壳网格转换为四面体实体网格，" 第64.7.6节](pt06ch64s07s06.md)。
7. 转到"装配"模块，并选择****实例**![](../graphics/images/arrow.gif)**替换****以用孤兒网格实例替换原始壳零件实例。
8. 如果您希望孤兒网格使用二阶四面体单元，则可以在"网格"模块中为孤兒网格分配二次单元类型。
