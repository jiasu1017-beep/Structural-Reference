# 11.21.2 添加旋转实体特征





从主菜单栏选择**形状**![](../graphics/images/arrow.gif)**实体**![](../graphics/images/arrow.gif)**旋转**，向当前视口中的零件添加旋转实体特征。您只能向三维零件添加旋转实体特征。

您通过在选定面上绘制二维横截面和构造线来添加旋转实体特征。构造线用作旋转轴，Abaqus/CAE通过使用指定的旋转角度绕轴旋转横截面来创建实体特征。此外，您可以指定pitch和沿旋转轴的方向，Abaqus/CAE在旋转轮廓时使用它来沿轴平移草图。

**添加旋转实体特征：**

1. 从主菜单栏，选择**形状**![](../graphics/images/arrow.gif)**实体**![](../graphics/images/arrow.gif)**旋转**。
2. 如果需要，指定要用于为旋转实体特征的草图选择原点的方法。
3. 选择实体将被旋转的平面。
4. 如果选择了**指定**作为**草图原点**方法，指定原点位置。
5. 选择一条边以及边在草图编辑器网格上的方向。
6. 使用水平、垂直、角度或斜交构造线工具绘制旋转轴。
7. 使用草图编辑器绘制旋转特征的二维轮廓。
8. 在提示区域，单击**完成**。
9. 在**编辑旋转**对话框中，输入所需的旋转角度或接受默认值。
10. 单击**旋转方向**旁边的以更改箭头方向和相关的旋转方向。
11. 如果需要，切换**包含平移**并输入pitch的正值。
12. 如果需要，切换**扫掠轮廓垂直于路径**。
13. 切换**保持内部边界**。
14. 单击**确定**接受指示的方向并创建旋转实体特征。
