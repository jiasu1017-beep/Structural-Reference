# 41.3.7 其他





**作业诊断**对话框中的**其他**页面仅在您从Abaqus/Standard分析的**作业历史**树中选择**步骤**时出现。它显示有关用于分析的矩阵求解器以及网格中特征单元长度的信息。根据程序类型，还提供其他相关信息，例如模型的质量和质心。此参考信息可以帮助您确定问题的可能原因或范围。例如，如果总质量或质心不正确，则属性模块中的材料定义、截面或截面分配存在问题。
