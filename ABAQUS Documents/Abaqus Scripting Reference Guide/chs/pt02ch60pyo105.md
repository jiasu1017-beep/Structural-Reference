# 60.105 VelocityDependence 对象





VelocityDependence 对象指定材料渗透率对流体流速的依赖性。

**访问**

```
materialApi.materials()[*name*].permeability().velocityDependence()
```

### 60.105.1 VelocityDependence(...)

此方法创建一个 VelocityDependence 对象。

**路径**

```
materialApi.materials()[*name*].permeability().VelocityDependence
```

**原型**

```
odb_VelocityDependence&
VelocityDependence(const odb_SequenceSequenceDouble& table);
```

**必需参数**

*table*

一个 odb_SequenceSequenceDouble，指定下述项目。

**可选参数**

无。

**表数据**

- ![](../graphics/ker_eqn00095.gif)。仅允许 ![](../graphics/ker_eqn00383.gif) 0.0。
- 孔隙比，![](../graphics/ker_eqn00289.gif)。

**返回值**

一个 VelocityDependence 对象。

**异常**

RangeError。
