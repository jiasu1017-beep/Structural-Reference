# 2.2.5 视口的组成





[图2-5](pt01ch02s02hlb04.md#cnv-view-components)显示了可视化模块中视口的组成。

**图2-5** 视口的组成。

![](../graphics/cnv-view-components-nls.png)

视口标题和视口周围的边框称为视口装饰。图例、状态块、标题块、视图方向三轴和3D罗盘称为视口注释。视图方向三轴和3D罗盘指示当前显示的模型的方向。您可以通过在3D罗盘上单击并拖动来更改模型的视图；视图方向三轴上的三个垂直轴会随着罗盘旋转以指示当前视图方向。有关更多信息，请参阅["3D罗盘，" 第5.3节"](pt01ch05s03.md)，以及["自定义视图三轴，" 第5.4节"](pt01ch05s04.md)。图例、状态块和标题块标识您使用可视化模块显示的结果。有关更多信息，请参阅[第56章，"自定义视口注释"](pt05ch56.md)。
