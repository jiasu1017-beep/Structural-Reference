# 79.2.3 操作叠加绘图的视图





当您处于叠加绘图状态时，可以选择视图操作是应用于所有现有层还是仅应用于当前层。

**操作叠加绘图中层的视图的步骤：**

1. 使用["生成叠加绘图，"第79.2.1节](pt07ch79s02hlb01.md)中描述的过程创建叠加绘图。
2. 在**叠加绘图层管理器**中，切换**全部**或**当前**在**视图操作层**旁边，以指定您想要的视图操作是影响所有层还是仅影响当前层。
3. 使用视图操作工具来定位、定向和放大视口中的对象。（有关视图操作工具的信息，请参见[第5章，"操作视图和控制透视](pt01ch05.md)。）视图操作将根据您在步骤2中的选择应用于所有可见层或仅应用于当前层。3D罗盘只能用于操作所有可见层；当您仅操作当前层时不可用。
4. 当叠加绘图显示在视口中时，您可以点击**叠加绘图层管理器**中的**绘制单个**退出叠加绘图状态。叠加绘图从视口中消失，显示返回到之前的绘图状态；但是**叠加绘图层管理器**保持打开。点击**解散**关闭**叠加绘图层管理器**。**提示：**您也可以随时点击工具箱中的![](../graphics/ico_overlayState.png)工具在单个绘图和叠加绘图状态之间切换。
