# 81.3 如何获取插件的相关信息？





Abaqus/CAE 主菜单栏中的 **Plug-ins** 菜单包含一个 **About Plug-ins** 项，该项会显示 **About Plug-ins** 对话框。此对话框列出了所有当前已安装的插件。当您点击树形列表中的每个插件时，Abaqus/CAE 会显示该插件的相关信息，如作者和版本。此外，您可以在 **About Plug-ins** 对话框中点击 **View** 来查看描述插件的文档。有关更多信息，请参见 ["如何提供插件的相关信息？，" 第81.13节](pt08ch81s13.md)。

此信息被指定为插件注册命令的可选参数。因此，如果插件作者选择不提供某些可选参数，则 **About Plug-ins** 对话框中将无法显示相应的信息。
