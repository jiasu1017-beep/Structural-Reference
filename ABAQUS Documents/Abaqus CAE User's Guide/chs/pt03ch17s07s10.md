# 17.7.10 我可以更改网格中单元的几何阶次吗？





如果您已经对零件、零件实例或区域进行了网格划分，Abaqus/CAE允许您更改单元阶次，而无需重新创建整个网格。如果您在线性和二次单元之间切换，Abaqus/CAE会根据需要简单地添加或移除中边节点。

**警告：**如果您更改零件或零件实例相邻区域中的单元阶次，则Abaqus/CAE无法在两个区域的边界处的线性节点和二次节点之间创建绑定接触。您必须创建两个单独的零件实例，并在它们之间创建绑定接触。

如果零件或零件实例包含孤立单元，您可以更改所有单元的阶次，也可以仅更改所选单元的阶次。孤立单元不包含底层几何信息；因此，在更改单元阶次时应小心。如果将孤立单元从二次更改为线性，则会丢失有关中边节点位置的所有信息。因此，如果您随后决定从线性更改为二次，您将无法返回到原始网格。有关更多信息，请参见["可以从Abaqus/CAE导入和导出哪些类型的文件？"第10.1.1节](pt02ch10s01s01.md)。
