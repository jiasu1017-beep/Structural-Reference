# 40.11 OdbDataFrame 对象

OdbDataFrame 对象。

**访问**

```
import visualization
session.odbData[*name*].steps[*i*].frames[*i*]
```

### 40.11.1 setValues(...)

此方法修改 OdbDataFrame 对象。

**必需参数**

*activateFrame*

 Boolean，指定是否激活帧。

**可选参数**

*update*

 Boolean，指定是否更新模型。默认值为 ON

**返回值**

无

**异常**

无。

### 40.11.2 成员

OdbDataFrame 对象没有成员。
