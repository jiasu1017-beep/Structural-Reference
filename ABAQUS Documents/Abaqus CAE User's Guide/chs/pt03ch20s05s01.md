# 20.5.1 参考几何







当您草绘特征的轮廓时，Abaqus/CAE 会将与草图平面相同的平面上的零件或装配的所有边和顶点投影到草图平面上。Abaqus/CAE 还会将所有基准点和基准轴投影到草图平面上，无论它们在零件或装配中的位置如何。这些投影的点 和线称为参考几何，您可以在草绘时使用它们作为参考。

您通常通过在草绘实体（如直线或顶点）和参考几何之间指定尺寸来精确地定位草图。您还可以使用参考几何来重新对齐草图网格。草图保留任何用于定位或标注草图的参考几何——当您使用 Sketcher 再次编辑特征时，参考几何会重新出现。Abaqus/CAE 会丢弃草图中未使用的任何参考几何。

如果您的零件很复杂，仅将草图平面中的边、顶点和基准点投影到草图平面上可以减少屏幕上的混乱，使选择过程更简单。但是，您的选择将仅限于草图平面中的边、顶点和基准点。如果您想使用其他边、顶点或基准点（包括孤立单元边和节点）作为参考，您必须使用**投影参考**工具![](../graphics/ico_sketchProjectReferences.png)将它们作为参考几何投影到草图平面上。或者，如果您想从现有模型边或孤立单元边创建新的草图边，可以使用**投影边**工具![](../graphics/ico_sketchProjectEdges.png)（有关更多信息，请参见["将边投影到草图上，"第 20.15 节](pt03ch20hla03.md)）。

当您草绘新特征时，您可以使用以下方法之一将草图约束到基础参考几何：
- 在草绘新特征时从参考几何中选择实体；例如，选择投影顶点来定义圆的中心或直线的一端。
- 在草图和任何投影参考几何之间创建约束、尺寸或参数。有关更多信息，请参见["控制草图几何，"第 20.7 节](pt03ch20s07.md)。

约束草图定义了它相对于参考几何的定位方式，以及如果您修改和重新生成零件或装配，它将如何重新定位。

当您退出 Sketcher 时，Abaqus/CAE 会确定您是否已将草图约束到参考几何。如果找到约束，Abaqus/CAE 会在新特征（子）和所选参考几何（父）之间创建父子关系。如果未找到约束，Abaqus/CAE 会在消息区域显示以下消息：`未在特征 *特征名称* 上指定草图放置约束`。您不必约束草图；但如果不这样做，如果您修改和重新生成零件或装配，其相对于零件或装配基础几何的位置可能会发生变化。

如果您保存独立草图，Abaqus/CAE 仅保存草图，不保存任何底层参考几何。因此，如果草图包含草图线和参考线之间的尺寸，该尺寸不会随独立版本的草图一起保存。

创建特征的顺序会影响可用的参考几何。基于特征的建模允许子特征依赖于父特征，但不允许父依赖于子。因此，如果特征是在您正在编辑的特征之后创建的，则无法将较新特征的边和顶点投影到草图平面上。您可以使用查询工具集来确定创建特征的顺序；有关更多信息，请参见["使用模型树获取特征信息，"第 65.4.8 节](pt06ch65s04hlb08.md)。
