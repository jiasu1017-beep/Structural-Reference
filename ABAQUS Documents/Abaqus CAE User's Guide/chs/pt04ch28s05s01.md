# 28.5.1 体积分数工具的要求





体积分数工具使用的欧拉部件实例必须与分配材料的实例相同。欧拉部件在使用体积分数工具之前必须进行网格划分。创建离散场后不应编辑部件网格，因为离散场中的单元编号可能与更新后网格中的单元不一致。

体积分数工具使用的参考部件实例可以包含未网格化的几何体、原始网格或孤立单元；允许使用可变形、欧拉和离散刚性部件。体积分数工具在计算离散场时始终使用参考部件实例的网格表示（如果可用）；如果参考实例已部分网格化，则在体积分数计算中仅考虑该实例的网格化部分。参考实例上的网格应足够精细，以捕获后续材料分配中的所有重要几何细节。

参考部件实例必须是三维实体或完全封闭的三维壳。壳封闭面的边缘必须定义单一连续表面；不允许使用与表面边缘产生T形交叉的特征（如肋或内部分隔板）（参见[图28-9](pt04ch28s05s01.md#usi-adv-volfrac-refshells)）。如果欧拉单元位于壳表面所封闭的体积内，则认为该单元在壳参考实例内部。
