# 3.1.2 在提示区域遵循指令和输入数据





要使用过程，只需遵循主窗口底部附近提示区域中出现的指示，如下所示：

![](../graphics/int-procedure-nls.png)

上图中标记为X的按钮是取消按钮；随时单击此按钮可取消整个过程。取消按钮左侧的箭头是上一步按钮；单击它可中止过程的当前步骤并返回上一个步骤。（在任何过程的第一步中，上一步按钮会显示为暗淡。）如果您愿意，可以将光标放在画布上并按下鼠标按钮3；然后从出现的菜单中选择**上一步**或**取消过程**。

在某些耗时的操作（如零件修复、网格划分或从大型模型的历史记录中提取X-Y数据）期间，提示区域会出现**停止**按钮。您可以单击**停止**来中断和取消操作。

许多过程需要文本或数字数据；例如，当使用草图模块创建圆角时，您必须首先指定圆角半径。当需要文本或数字数据时，Abaqus/CAE会在提示区域显示一个供您填写的文本字段；通常文本框已经包含默认值，如下所示：

![](../graphics/int-procedure-text-nls.png)

将光标放在视口上，并按如下方式在文本字段中输入数据：
- 要接受默认值，请按**[Enter]**或鼠标按钮2。
- 要替换默认值，只需开始键入；键入前不需要单击文本字段。一旦您开始键入，默认值就会消失。
- 要更改默认值的一部分，首先单击文本字段；然后使用**[Delete]**键和键盘上的其他键来更改值。
- 要提交任何更改，请按**[Enter]**或鼠标按钮2。
- 您也可以在提示区域的文本字段中输入表达式。有关更多信息，请参阅["输入表达式，" 第3.2.2节"](pt01ch03s02s02.md)。

某些过程要求您从多个选项中进行选择。例如，基准工具集可能会要求您选择主轴。这些选项由提示区域中的按钮表示，如下所示：

![](../graphics/int-procedure-choice-nls.png)

单击相应的按钮选择所需的选项。

在某些过程中，默认选项由相应按钮周围的边框表示；在上例中，边框绘制在**X轴**按钮周围。要选择默认选项，请单击鼠标按钮2。
