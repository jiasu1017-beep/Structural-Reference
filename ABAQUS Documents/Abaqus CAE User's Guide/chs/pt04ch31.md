# 31 断裂力学





您可以执行以下操作来使用Abaqus/CAE对断裂力学进行建模：
- 创建定义在分析过程中可以分离的具有重叠节点的边缘或面的裂缝。
- 使用轮廓积分估计来研究准静态问题中的开裂起始；然而，轮廓积分估计不能预测裂缝将如何扩展。您可以为二维或三维模型计算轮廓积分。
- 使用扩展有限元方法（XFEM）来研究沿任意、依赖解的路径的裂缝萌生和扩展，而无需对模型进行重新网格划分。XFEM仅适用于三维实体和平面模型以及孤立网格。
- 使用虚拟裂缝闭合技术（VCCT）来研究沿已知裂缝表面的裂缝萌生和扩展。
- 使用**Crack Manager**创建和管理裂缝。
