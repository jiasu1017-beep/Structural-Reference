# 58.7.2 从输出数据库网格数据创建映射场





您可以通过从 Abaqus 输出数据库（`.odb`）文件读取参数值来创建映射场。当映射数据源是 Abaqus 输出数据库时，场值必须在 Visualization 模块的等值线图中显示。请参见["从输出数据库文件进行网格到网格映射，" 第58.3.3节](pt06ch58s03s03.md)，了解网格到网格映射概述。

**从输出数据库网格数据创建映射场的步骤：**

1. 从输出数据库创建映射场时，您必须打开两个文件：
   - 包含映射主目标模型的模型数据库（`.cae`）文件，以及
   - 包含先前在同一（或类似）模型上运行的分析的场输出变量数据的输出数据库（`.odb`）文件。

2. 选择要使用的源数据。
   1. 在 Visualization 模块中，打开包含源数据的输出数据库文件。
   2. 在视口中显示未变形等值线图。您也可以使用变形图，但数据仍从未变形网格读取。
   3. 选择您要映射的值对应的场输出变量，在分析的特定步骤和帧处。请参见["生成等值线图，" 第44.3节](pt05ch44hla02.md)，了解等值线图详情。

3. 在包含主（目标）模型的模型数据库中，使用以下方法之一打开**创建映射场**对话框：
   - 从主菜单栏，选择 ****Tools**![](../graphics/images/arrow.gif)**Analytical Field**![](../graphics/images/arrow.gif)**Create****。
   **提示：**您也可以单击**分析场管理器**中的**创建**。
   - 在 Interaction 模块的相互作用编辑器中，单击 **Definition** 或 **Emissivity distribution** 字段旁边的 ![](../graphics/ico_analyticalField.png)。
   - 在 Load 模块的载荷、边界条件或预定义场编辑器中，单击 **Distribution** 字段旁边的 ![](../graphics/ico_analyticalField.png)。

4. 选择 **Mapped field** 作为**类型**，然后单击 **Continue**。

5. 在**名称**文本字段中，输入映射场的名称。有关命名对象的信息，请参见["使用基本对话框组件，" 第3.2.1节](pt01ch03s02s01.md)。在**描述**文本字段中，输入映射场的描述。

6. 选择 **ODB mesh** 作为**数据源**。

7. 如果需要，选择以下选项之一输入**比例因子**并缩放源数据坐标；例如，为了补偿单位不匹配（即米到毫米）：
   - 选择 **Uniform**，然后输入单个比例因子。
   - 选择 **Nonuniform**，然后为 *X*、*Y* 和 *Z* 坐标输入比例因子。

8. 输入**默认未映射场值**的值，或接受默认值零。Abaqus/CAE 为任何无法找到源点进行映射的目标点分配此值。请参见["映射场的搜索控制，" 第58.7.3节](pt06ch58s06s01.md)。

9. 在 **ODB Mesh Data** 选项卡页面上，从打开的视口列表中选择**要映射的视口**。Abaqus/CAE 使用所选视口的当前绘图状态作为要映射到模型数据库（`.cae`）文件中目标模型的值集；即，在分析的特定步骤和帧显示的特定场输出变量。

10. 在 **Mapper Controls** 选项卡页面上，您可以调整搜索容差参数。请参见["映射场的搜索控制，" 第58.7.3节](pt06ch58s06s01.md)。

11. 单击 **OK** 创建映射场并关闭对话框。

使用网格到网格映射创建映射场后，您无法编辑源数据网格规范；您只能编辑现有映射场以更改默认场值或映射器控制选项。

当您稍后使用映射场应用载荷、相互作用或预定义场时，您可以在视口中显示符号以可视化场值的位置和幅值。但是，您必须先对模型进行网格划分才能看到这些符号。
