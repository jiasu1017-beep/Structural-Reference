# 42.8.1 创建坐标系方法概述





在可视化模块中，从主菜单栏选择****工具**![](../graphics/images/arrow.gif)**坐标系**![](../graphics/images/arrow.gif)**创建****来定义局部坐标系。您选择坐标系类型，然后按照提示区域中的提示来定义坐标系轴。可用的坐标系类型如下：
- **固定系统**
- **跟随3节点的坐标系**
- **跟随圆上3节点的坐标系**
- **跟随单节点的坐标系**

您可以使用结果树中的**会话坐标系**容器或**坐标系管理器**来重命名或删除用户定义的坐标系，并将坐标系保存到输出数据库文件中，以便在以后的Abaqus/CAE会话中使用。
