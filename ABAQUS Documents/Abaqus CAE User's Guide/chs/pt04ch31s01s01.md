# 31.1.1 什么是裂缝？





裂缝定义了模型中最初闭合但在分析过程中可以打开的边缘或面。生成网格时，Abaqus/CAE沿着裂缝放置重叠的重复节点。裂缝不能沿部件边界延伸，必须嵌入二维部件的面内或实体部件的单元内。创建裂缝后，您可以使用轮廓积分分析来确定其裂缝属性。因为裂缝修改了网格，所以您不能在依赖部件实例上创建裂缝。

[图31-1](pt04ch31s01s01.md#eng-crack-seam2d)显示了嵌入在平面部件面上的裂缝以及向模型施加拉力后的效果。沿裂缝的重复节点彼此独立，可以自由移动。
