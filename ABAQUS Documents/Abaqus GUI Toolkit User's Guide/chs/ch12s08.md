# 12.8 在 Abaqus/Explicit 中定义接触





Abaqus/Explicit 为建模接触相互作用提供了两种算法。通用（"自动"）接触算法允许非常简单的接触定义，对所涉及表面类型几乎没有限制（参见 ["在 Abaqus/Explicit 中定义通用接触相互作用，" Abaqus 分析用户指南第 36.4.1 节](../usb/usb-link.md#usb-cni-acontactgeneral)）。接触对算法对所涉及的表面类型有更多限制，通常需要更仔细的接触定义；但是，它允许一些当前通用接触算法不可用的相互作用行为（参见 ["在 Abaqus/Explicit 中定义接触对，" Abaqus 分析用户指南第 36.5.1 节](../usb/usb-link.md#usb-cni-aexpcontactpair)）。通用接触相互作用通常通过为 Abaqus/Explicit 自动定义的全包容、基于单元的表面（包含模型中的所有体）指定自接触来定义。为了细化接触域，可以包括或排除特定的表面对。接触对相互作用通过指定可能相互作用的各个表面对来定义。

### 12.8.1 Abaqus/Explicit 接触公式

Abaqus/Explicit 中的接触公式包括约束强制方法、接触表面加权方法和滑动公式。

**约束强制方法**

对于通用接触（[*CONTACT](../key/key-link.md#usb-kws-hcontact)），Abaqus/Explicit 使用罚函数接触方法强制接触约束，该方法在当前配置中搜索节点-面和边-边穿透。罚函数刚度将接触力与穿透距离相关联，由 Abaqus/Explicit 自动选择，以使对时间增量的影响最小，同时穿透不显著。

接触对算法（[*CONTACT PAIR](../key/key-link.md#usb-kws-hcontactpair)）默认使用运动接触公式，该公式使用预测/校正方法实现与接触条件的精确一致性。增量最初在假设不发生接触的情况下进行。如果在增量结束时存在过盈闭合，则修改加速度以获得强制执行接触约束的校正配置。运动接触使用的预测/校正方法在 ["Abaqus/Explicit 中的接触约束强制方法，" Abaqus 分析用户指南第 38.2.3 节](../usb/usb-link.md#usb-cni-aexpcontactconstraints) 中有更详细的讨论；该方法的一些限制在 ["与在 Abaqus/Explicit 中使用接触对的接触建模相关的常见困难，" Abaqus 分析用户指南第 39.2.2 节](../usb/usb-link.md#usb-cni-aexpcontacttrouble) 中讨论。

接触对的法向接触约束可以选择使用罚函数接触方法强制执行，这可以模拟运动方法无法模拟的一些接触类型。例如，罚函数方法允许模拟两个刚性表面之间的接触（当两个表面都是解析刚性表面时除外）。当使用罚函数接触公式时，在穿透点对主节点和从节点施加大小等于罚函数刚度乘以穿透距离的相等且相反的接触力。罚函数刚度由 Abaqus/Explicit 自动选择，类似于通用接触算法使用的刚度。也可以指定罚函数比例因子。要为接触对分析选择罚函数方法，请在 [*CONTACT PAIR](../key/key-link.md#usb-kws-hcontactpair) 选项上将 MECHANICAL CONSTRAINT 参数设置为 PENALTY。

**接触表面加权**

在纯主-从方法中，一个表面是主表面，另一个是从属表面。当两个物体接触时，检测穿透，并根据约束强制方法（运动或罚函数）施加接触约束。纯主-从加权（无论约束强制方法如何）将仅抵抗从属节点穿透主面。主节点穿透从属表面可能检测不到（如图 12-35 所示），除非从属表面上的网格足够细化。

**图 12-35** 纯主-从接触中主节点穿透从属表面。

![](../graphics/gsx-pure-mast-slave-nls.png)

平衡主-从接触简单地应用纯主-从方法两次，在第二次传递时反转表面。获得一组表面 1 作为从属的接触约束，并获得另一组表面 2 作为从属的接触约束。通过对两个计算取加权平均来获得加速度校正或力。对于运动平衡主-从接触，进行第二次校正以解决任何剩余穿透，如 ["Abaqus/Explicit 中接触对的接触公式，" Abaqus 分析用户指南第 38.2.2 节](../usb/usb-link.md#usb-cni-aexpcontactpairform) 中所述。当使用运动一致性时，平衡主-从接触约束如图 12-36 所示。

平衡方法最小化接触体的穿透，因此，在大多数情况下提供更准确的结果。

通用接触算法尽可能使用平衡主-从加权；对于涉及基于节点表面的通用接触相互作用，仅当它们只能作为纯从属表面时，才使用纯主-从加权。对于接触对算法，Abaqus/Explicit 将根据所涉及的两个表面的性质和所使用的约束强制方法为给定接触对决定使用哪种类型的加权。

通用接触算法尽可能使用平衡主-从加权；对于涉及基于节点表面的通用接触相互作用，仅当它们只能作为纯从属表面时，才使用纯主-从加权。使用 [*CONTACT FORMULATION](../key/key-link.md#usb-kws-hcontformulation), TYPE=PURE MASTER-SLAVE 为其他通用接触相互作用指定纯主-从加权。

对于接触对算法，Abaqus/Explicit 将根据所涉及的两个表面的性质和所使用的约束强制方法为给定接触对决定使用哪种类型的加权。可以使用 [*CONTACT PAIR](../key/key-link.md#usb-kws-hcontactpair) 选项上的 WEIGHT 参数为接触对算法的平衡主-从接触指定平均权重。对于大多数单元类型，默认权重为 0.5，因此每个加速度校正使用相同的权重。将 WEIGHT 设置为 1.0 指定纯主-从关系，第一个表面是主表面。相反，权重为零意味着第二个表面是主表面。

**滑动公式**

定义接触对时，必须确定相对滑动的大小是小还是有限。默认（也是通用接触相互作用的唯一选项）是更通用的有限滑动公式。如果两个表面的相对运动小于单元面特征长度的一小部分，则小滑动是合适的。通过在 [*CONTACT PAIR](../key/key-link.md#usb-kws-hcontactpair) 选项上包含 SMALL SLIDING 参数来选择小滑动公式。在适用时使用小滑动公式将产生更高效的分析。
