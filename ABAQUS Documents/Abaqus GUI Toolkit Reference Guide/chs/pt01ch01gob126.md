# 辅助函数

本节列出各种辅助 GUI 工具包函数。

### addExitCallback(callback)

注册一个回调函数，在应用程序即将退出时调用。

| **参数** | **类型** | **默认值** | **描述** |
| --- | --- | --- | --- |
| callback | Function |  | 应用程序即将退出时要调用的函数。 |

### afxCreateIcon(fileName)

返回通过读取指定文件创建的图标，文件可以是以下格式：BMP、GIF、PNG、XPM。文件格式从文件扩展名推断（不区分大小写）。如果文件无法打开则返回 0。

| **参数** | **类型** | **默认值** | **描述** |
| --- | --- | --- | --- |
| fileName | String |  | 文件名。 |

### afxCreateBMPIcon(fileName)

返回通过读取 BMP 格式的指定文件创建的图标。如果文件无法打开则返回 0。

| **参数** | **类型** | **默认值** | **描述** |
| --- | --- | --- | --- |
| fileName | String |  | 文件名。 |

### afxCreateGIFIcon(fileName)

返回通过读取 GIF 格式的指定文件创建的图标。如果文件无法打开则返回 0。

| **参数** | **类型** | **默认值** | **描述** |
| --- | --- | --- | --- |
| fileName | String |  | 文件名。 |

### afxCreatePNGIcon(fileName)

返回通过读取 PNG 格式的指定文件创建的图标。如果文件无法打开则返回 0。

| **参数** | **类型** | **默认值** | **描述** |
| --- | --- | --- | --- |
| fileName | String |  | 文件名。 |

### afxCreateXPMIcon(fileName)

返回通过读取 XPM 格式的指定文件创建的图标。如果文件无法打开则返回 0。

| **参数** | **类型** | **默认值** | **描述** |
| --- | --- | --- | --- |
| fileName | String |  | 文件名。 |

### afxDisableWidgetTree(widget)

禁用指定的组件及其所有子组件。

| **参数** | **类型** | **默认值** | **描述** |
| --- | --- | --- | --- |
| widget | Widget |  | 父组件，该组件及其所有子组件都将被禁用。 |

### afxEnableWidgetTree(widget)

启用指定的组件及其所有子组件。

| **参数** | **类型** | **默认值** | **描述** |
| --- | --- | --- | --- |
| widget | Widget |  | 父组件，该组件及其所有子组件都将被启用。 |

### afxGetIcon(fileName, size)

使您可以在自定义中使用 Abaqus/CAE 图标。将 size 参数设置为 1 表示正常大小图标，设置为 0 表示小图标。

| **参数** | **类型** | **默认值** | **描述** |
| --- | --- | --- | --- |
| fileName | String |  | 文件名。 |
| size | String |  | 图标大小。 |

### displayURL(url)

在 Web 浏览器中显示指定的 URL。返回调用状态。此调用将使用打开的 Web 浏览器（如果有）。此方法可通过 uti 模块中的 webBrowser.displayURL 访问。另请参阅 openWithURL。

| **参数** | **类型** | **默认值** | **描述** |
| --- | --- | --- | --- |
| url | String |  | 要显示页面的 URL。 |

### getAFXApp()

返回应用程序对象。

### getAFXAliasMap().setPrefix(widget, prefix)

设置组件的前缀键——通常是对话框类名。

| **参数** | **类型** | **默认值** | **描述** |
| --- | --- | --- | --- |
| widget | FXWindow |  | 要设置前缀的组件。 |
| prefix | String |  | 组件的前缀键。 |

### getAFXAliasMap().setName(widget, name)

设置组件的名称键。

| **参数** | **类型** | **默认值** | **描述** |
| --- | --- | --- | --- |
| widget | FXWindow |  | 要设置名称的组件。名称键与前缀一起用于标识组件。 |
| name | String |  | 组件的名称。 |

### getAFXFont(opts=FONT_PROPORTIONAL)

返回指定的字体。

| **参数** | **类型** | **默认值** | **描述** |
| --- | --- | --- | --- |
| opts | Int | FONT_PROPORTIONAL | 要获取的字体类型。可能的选项有 FONT_PROPORTIONAL、FONT_MONOSPACE、FONT_REGULAR、FONT_BOLD、FONT_ITALIC、FONT_BASE 或 FONT_SMALL。 |

### afxGetColorHexSpecFromID(colorId)

返回指定颜色索引的等效十六进制字符串。

| **参数** | **类型** | **默认值** | **描述** |
| --- | --- | --- | --- |
| colorId | Integer |  | 要转换为十六进制字符串的颜色索引。例如，绿色为 FXRGB(0, 255,0)。有关详细信息，请参阅《Abaqus GUI Toolkit Reference Guide》的 Colors 附录。 |

### afxGetColorHexSpecFromName(colorName)

返回指定颜色名称的等效十六进制字符串。

| **参数** | **类型** | **默认值** | **描述** |
| --- | --- | --- | --- |
| colorName | String |  | 要转换为十六进制字符串的颜色名称。例如，Red。有关详细信息，请参阅《Abaqus GUI Toolkit Reference Guide》的 Colors 附录。 |

### getCurrentContext()

返回当前 GUI 上下文字典，其中包含以下键：mdbName、viewportName、objectPath、objectType、modelName 和 moduleName。您可以使用 registerCurrentContext 函数来接收上下文更改通知。

### getCurrentModuleGui()

返回当前 moduleGui 对象。您可以调用该对象的 getModuleName 方法来检查其名称。

### getCursorPosition()

返回 (status, x, y, buttonState) 元组。status（TRUE 或 FALSE）表示调用是否成功。x 和 y 值表示光标在窗口坐标系统中的位置（原点在左上角，正 Y 向下）。此方法在 FXWindow 中定义，因此可以在从 FXWindow 派生的任何对象上调用。

### getDisplayedObjectType()

返回当前视口中显示的对象类型。可能的返回值为：PART、ASSEMBLY、ODB、XY_PLOT、SKETCH 或 None。

### getSeparator(parent, count)

返回父组件的第 n 个（由 count 参数指定）分隔符组件。

| **参数** | **类型** | **默认值** | **描述** |
| --- | --- | --- | --- |
| parent | Widget |  | 要搜索的组件（不遍历子组件）。 |
| count | Int |  | 分隔符的编号（例如，指定 2 获取第二个分隔符）。 |

### getWidgetFromText(parent, text)

返回其标签或提示文本与指定文本匹配且也是指定组件子组件的组件。

| **参数** | **类型** | **默认值** | **描述** |
| --- | --- | --- | --- |
| parent | Widget |  | 开始搜索的组件（也会遍历子组件）。 |
| text | String |  | 要匹配的文本（必须是精确匹配）。 |

### openWithURL(url)

在 Web 浏览器中显示指定的 URL。返回调用状态。此调用将始终打开一个新的 Web 浏览器。此方法可通过 uti 模块中的 webBrowser.openWithURL 访问。另请参阅 displayURL。

| **参数** | **类型** | **默认值** | **描述** |
| --- | --- | --- | --- |
| url | String |  | 要显示页面的 URL。 |

### registerCurrentContext(callbackFunction)

注册对当前上下文的查询。当当前视口或显示的对象更改时，将调用指定的回调函数。使用 getCurrentContext 方法获取当前对象的值。请注意，当不再需要查询时，应使用相同的参数调用 unregisterCurrentContext 来取消注册查询。

| **参数** | **类型** | **默认值** | **描述** |
| --- | --- | --- | --- |
| callbackFunction | Function |  | 当前上下文更改时要调用的函数。 |

### registerPluginToolset()

将插件工具集注册到应用程序。

### objectPath.registerQuery(callbackFunction, callOnRegister=TRUE)

注册对对象的查询。当对象更改时，将调用指定的回调函数。有关更多信息，请参阅《Abaqus GUI Toolkit User's Guide》第 6.8 节"接收内核数据更改通知"。请注意，当不再需要查询时，应使用相同的参数调用 unregisterQuery 来取消注册查询。

| **参数** | **类型** | **默认值** | **描述** |
| --- | --- | --- | --- |
| callbackFunction | Function |  | 对象更改时要调用的函数。 |
| callOnRegister | Bool | TRUE | 如果为 TRUE，则在注册查询时将调用回调函数。 |

### objectPath.registerInclusive(callbackFunction, callOnRegister=TRUE)

注册对对象及其子对象的查询。当对象或其子对象更改时，将调用指定的回调函数。请注意，当不再需要查询时，应使用相同的参数调用 unregisterQuery 来取消注册查询。

| **参数** | **类型** | **默认值** | **描述** |
| --- | --- | --- | --- |
| callbackFunction | Function |  | 对象或其子对象更改时要调用的函数。 |
| callOnRegister | Bool | TRUE | 如果为 TRUE，则在注册查询时将调用回调函数。 |

### removeExitCallback(callback)

取消注册应用程序即将退出时要调用的回调函数。该函数必须先前已使用 addExitCallback 注册。

| **参数** | **类型** | **默认值** | **描述** |
| --- | --- | --- | --- |
| callback | Function |  | 要从回调列表中移除的函数。 |

### sendCommand(writeToJournal, command, writeToReplay=True, writeToJournal=False)

向内核发送命令字符串。

**注意：** writeToJournal 参数通常不是必需的。如果命令是内置的 Abaqus Scripting Interface 命令，它会自动写入日志文件。如果命令不是内置的 Abaqus Scripting Interface 命令，但它使用内置的 Abaqus Scripting Interface 命令更改了 mdb，则对 mdb 的更改会自动写入日志文件。但是，如果命令不是内置的 Abaqus Scripting Interface 命令，并且它对 mdb 进行了否则不会写入日志文件的更改，则该命令本身应使用 journalMethodCall 函数来写入日志文件。

| **参数** | **类型** | **默认值** | **描述** |
| --- | --- | --- | --- |
| writeToJournal |  |  |  |
| command | String |  | 命令字符串。 |
| writeToReplay | Bool | True | 如果为 True，则命令将写入回放文件。 |
| writeToJournal | Bool | False | 如果为 True，则命令将写入日志文件。 |

### setCurrentModel(modelName)

将当前模型设置为指定的模型名称。

| **参数** | **类型** | **默认值** | **描述** |
| --- | --- | --- | --- |
| modelName | String |  | 要设为当前的模型名称。 |

### setSwitchModuleHook(callbackFunction)

设置一个函数，每当用户切换到 GUI 模块时就会调用该函数。当用户切换到 GUI 模块时，将调用指定的函数，并向其传递新模块的名称（显示在 Module 组合框中）。请注意，setSwitchModuleHook 函数不接受关键字参数，如果需要，可以多次调用它。

| **参数** | **类型** | **默认值** | **描述** |
| --- | --- | --- | --- |
| callbackFunction | Function |  | 切换到 GUI 模块时要调用的函数。请注意，您不能从此函数内发出内核命令，否则会导致应用程序挂起。 |

### shutdown()

退出 Abaqus/CAE。这相当于选择 File->Exit。

### switchModule(moduleName)

切换到 GUI 模块。

| **参数** | **类型** | **默认值** | **描述** |
| --- | --- | --- | --- |
| moduleName | String |  | 要切换到的模块。 |

### unregisterCurrentContext(callbackFunction)

取消对当前上下文的查询的注册。指定的回调函数应与用于注册查询的参数相同。

| **参数** | **类型** | **默认值** | **描述** |
| --- | --- | --- | --- |
| callbackFunction | Function |  | 当前上下文更改时要调用的函数。 |

### objectPath.unregisterQuery(callbackFunction)

取消对对象查询的注册。指定的回调函数应与用于注册查询的参数相同。

| **参数** | **类型** | **默认值** | **描述** |
| --- | --- | --- | --- |
| callbackFunction | Function |  | 对象更改时要调用的函数。 |
