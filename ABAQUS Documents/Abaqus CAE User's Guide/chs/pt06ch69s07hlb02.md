# 69.7.2 用新面覆盖边







您可以通过选择新面将覆盖的边来创建面。所选的边必须形成闭合环。Abaqus/CAE 将新面创建为壳。如果新壳形成闭合部件，您可以使用 solid-from-shell 工具将部件转换为实体。 有关更多信息，请参阅["从壳创建实体特征，" 第 11.21.5 节"](pt03ch11s21hlb05.md)。覆盖边操作作为部件的特征存储；因此，您可以使用模型树删除您创建的面。Abaqus/CAE 通过在由所选边形成的平面上投影面来创建新面。因此，如果结果面非常凸起，您不应使用 **Cover edges**（覆盖边）工具。

**创建新面：**

1. 从主菜单栏中选择 ****Tools**![](../graphics/images/arrow.gif)**Geometry Edit****（工具→几何编辑）。Abaqus/CAE 显示 **Geometry Edit**（几何编辑）对话框。**提示：**您也可以使用位于 Part（部件）模块工具箱中编辑工具处的 ![](../graphics/ico_repairCreateFace.png) 工具来创建面。 有关工具箱中编辑工具的图表，请参阅["使用几何编辑工具集，" 第 69.1 节"](pt06ch69s01.md)。
2. 从对话框中选择 **Face**（面）类别和 **Cover edges**（覆盖边）方法。
3. 选择形成环并定义新面边缘的边。您可以从视口选择边或选择包含边的现有集合。您可以选择多个环，Abaqus/CAE 将创建多个面。但是，这些环不应是同心的。**注意：**默认选择方法基于您最近使用的选择方法。要恢复为其他方法，请点击提示区域右侧的 **Select in Viewport**（在视口中选择）或 **Sets**（集合）。
4. 从提示区域点击 **Done**（完成）。Abaqus/CAE 创建新面。
5. 如果您已完成编辑部件，您应该通过在模型树中的部件上点击鼠标按钮 3 并从出现的菜单中选择 **Update validity**（更新有效性）来检查其有效性。
