# 77.5 在可视化模块中对所有几何进行着色





本节介绍当您在可视化模块中工作时，如何对当前视口中的所有可视图几何应用颜色编码。要从其他 Abaqus/CAE 模块对几何和网格单元应用颜色编码，请参见["对几何和网格单元进行着色，" 第77.4节](pt07ch77s04.md)。要在可视化模块中按选定的节点或单元而不是整个几何进行颜色编码，请参见["在可视化模块中对节点或单元进行着色，" 第77.6节](pt07ch77s06.md)。

当在可视化模块中选择了输出数据库时，您可以在对所有几何应用颜色编码时选择以下任何颜色映射：
- **零件实例**
- **单元集**
- **材料**
- **截面**
- **单元类型**
- **平均区域**
- **内部集合**
- **复合铺层**
- **复合层**

当在可视化模块中选择了模型数据库中的模型时，只有 **零件实例** 颜色编码选项可用。

Abaqus/CAE 在 **颜色映射** 表中列出当前选择方法的所有项目。在 **颜色编码** 对话框中选择颜色映射后，您还可以自定义其单个项目的显示颜色。请参见["从列表和表中选择多个项目，" 第3.2.11节](pt01ch03s02s11.md)，了解更多信息。您可以直接从表中进行选择，并且可以选择多个单元格；请参见["从列表和表中选择多个项目，" 第3.2.11节](pt01ch03s02s11.md)，了解更多信息。

**从可视化模块对视口中的所有几何应用颜色映射：**

1. 点击 **颜色编码** 工具栏中的 ![](../graphics/ico_colorCode.png) 工具。Abaqus/CAE 显示 **颜色编码** 对话框，其中显示当前模块的默认颜色映射。
2. 从 **颜色编码** 列表中，选择 **全部**。
3. 从 **依据** 列表中，选择您要应用的颜色映射名称。
4. 点击 **应用**。Abaqus/CAE 根据 **颜色编码** 对话框中显示的颜色对当前视口进行颜色编码。
