# 1.4.11 压电单元






**产品：**Abaqus/Standard  

压电单元的位移和电势都是自由度。这些单元包括桁架、平面应力、平面应变、轴对称或三维连续体。单元与基本应力/位移单元相同，只是应力场与电势梯度之间存在耦合。这些单元也测试了机械载荷，但此处未予报告，因为它们与连续体应力/位移单元章节中报告的内容相同。本章节仅报告与体积和分布电荷相关的附加载荷。
