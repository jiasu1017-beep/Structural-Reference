# 49.3.2 选择动画文件格式





您可以选择将动画保存为以下格式之一的文件：Audio Video Interleave（AVI，默认）、QuickTime 或 Virtual Reality Modeling Language（VRML）格式。您还可以在这所有格式中选择压缩或未压缩的文件。AVI 和 QuickTime 格式都提供额外的图像大小和压缩选项。

在 Windows 系统上以 AVI 格式保存动画时，您还可以使用系统上的任何编解码器压缩文件，并可以设置文件的质量级别。这些选项在 Linux 系统上不可用。

**改进 VRML 动画的播放**

VRML 或压缩 VRML 格式的动画可能需要进一步的自定义才能在客户端计算机上正确播放。客户端计算机上的多个组件决定 VRML 播放的质量，包括图形硬件、VRML 插件软件和为插件软件选择的设置。当这些组件与创建 VRML 动画的计算机上的组件不同时，客户端计算机上的 VRML 播放可能会出现显示问题，例如视口注释中的文本模糊。您可以通过在客户端计算机上配置插件设置或使用不同的视口注释设置创建新版 VRML 动画来改进特定 VRML 文件的客户端播放。
- 要在客户端计算机上自定义 VRML 显示，请打开客户端计算机上 VRML 插件的设置，并切换任何抗锯齿设置。启用抗锯齿可以帮助澄清 VRML 图像或动画中的文本，但可能会降低性能。
- 要创建新的 VRML 动画，请返回 Abaqus/CAE，并为视口中的每个视口注释增加文本大小或选择不同的字体，然后保存新副本的 VRML 动画。有关更改字体和类型大小的更多信息，请参阅 [第 56 章，"自定义视口注释"](pt05ch56.md)。

**选择 VRML 或压缩 VRML 格式：**

1. 找到动画文件格式选项。从主菜单栏中，选择 ****动画（Animate）**![](../graphics/images/arrow.gif)**另存为（Save As）****；出现 **保存图像动画（Save Image Animation）** 对话框。
2. 从 **格式（Format）** 选项中，选择 **VRML** 或 **压缩 VRML（Compressed VRML）** 格式。对于这两种 VRML 格式，Abaqus/CAE 停用 **选择（Selection）** 和 **帧速率（Frame Rate）** 字段；您无法为 **VRML** 或 **压缩 VRML（Compressed VRML）** 动画自定义这些设置。

**选择 QuickTime 格式：**

1. 找到动画文件格式选项。从主菜单栏中，选择 ****动画（Animate）**![](../graphics/images/arrow.gif)**另存为（Save As）****；出现 **保存图像动画（Save Image Animation）** 对话框。
2. 从 **格式（Format）** 选项中，选择 **QuickTime**。
3. 要选择其他 QuickTime 格式选项，请点击 ![](../graphics/ico_selectionOptions.png)。出现 **QuickTime 选项（QuickTime Options）** 对话框。
4. 在 **QuickTime 选项（QuickTime Options）** 对话框中，选择以下方法之一来指定动画图像的大小（以像素为单位）：- 点击 **使用屏幕上的大小（Use size on screen）** 使用屏幕上图像的大小。（Abaqus/CAE 在 **QuickTime 选项（QuickTime Options）** 对话框中指示当前图像大小。）此方法是默认方法。- 点击 **使用以下设置（Use settings below）** 设置宽度或高度；然后在 **宽度（Width）** 或 **高度（Height）** 字段中输入您选择的值。您只需指定一个维度；Abaqus/CAE 计算另一个维度以维持视口的纵横比。
5. 在 **QuickTime 选项（QuickTime Options）** 对话框中指定您的文件是否应该被压缩（行程长度编码）或未压缩（原始）。从 **压缩（Compression）** 列表中，选择 **原始（24 位/像素）（Raw (24 bits/pixel)）** 或 **Rle（24 位/像素）（Rle (24 bits/pixel)）**。**Rle（24 位/像素）** 压缩是一种行程长度编码形式，提供无损压缩且图像无退化。
6. 点击 **确定（OK）** 关闭 **QuickTime 选项（QuickTime Options）** 对话框。您的选项设置将在会话期间保存。

**选择 **AVI** 格式：**

1. 找到动画文件格式选项。从主菜单栏中，选择 ****动画（Animate）**![](../graphics/images/arrow.gif)**另存为（Save As）****；出现 **保存图像动画（Save Image Animation）** 对话框。
2. 从 **格式（Format）** 选项中，选择 **AVI**。
3. 要选择其他 AVI 格式选项，请点击 ![](../graphics/ico_selectionOptions.png)。出现 **AVI 选项（AVI Options）** 对话框。
4. 在 **AVI 选项（AVI Options）** 对话框中，选择以下方法之一来指定动画图像的大小（以像素为单位）：- 点击 **使用屏幕上的大小（Use size on screen）** 使用屏幕上图像的大小。（Abaqus/CAE 在选项对话框中指示当前图像大小。）此方法是默认方法。- 点击 **使用以下设置（Use settings below）** 设置宽度或高度；然后在 **宽度（Width）** 或 **高度（Height）** 字段中输入您选择的值。您只需指定一个维度；Abaqus/CAE 计算另一个维度以维持视口的纵横比。
5. 对于 Linux 系统，在 **AVI 选项（AVI Options）** 对话框中指定您的文件是否应该被压缩（行程长度编码）或未压缩（原始）。从 **AVI 选项（AVI Options）** 对话框中的 **压缩（Compression）** 列表中，选择 **Rle（8 位/像素）（Rle (8 bits/pixel)）**、**原始（8 位/像素）（Raw (8 bits/pixel)）** 或 **原始（32 位/像素）（Raw (32 bits/pixel)）**。选择 **Rle（8 位/像素）** 压缩或 **原始（8 位/像素）** 可能导致颜色丢失。
6. 对于 Windows 系统，在 **AVI 选项（AVI Options）** 对话框中指定您的文件是否应该被压缩或未压缩（原始）。如果您选择压缩文件，您可以从 Abaqus/CAE 预定义的编解码器或系统上可用的视频编解码器中进行选择。您可以指定保存的动画文件的压缩质量。1. 从 **编解码器（Codec）** 列表中，选择以下之一：- 选择 **无 - 8 位/像素（None - 8 bits/pixel）** 使用减少的颜色集创建原始动画。选择此选项可能导致颜色丢失。- 选择 **无 - 24 位/像素（None - 24 bits/pixel）** 创建没有任何压缩的原始动画文件。- 选择视频编解码器之一。**编解码器（Codec）** 列表显示系统上可用的每个编解码器。某些视频编解码器包括可配置的选项，使您能够更改视频文件的某些方面。当您选择允许此配置的编解码器时，**配置（Configure）** 按钮变为活动状态。您还可以通过在选择编解码器时点击 **关于（About）** 按钮来了解有关特定编解码器的更多信息。2. 要配置编解码器，请点击 **配置（Configure）**，然后从出现的对话框中选择配置选项。每个可配置的视频编解码器提供不同的配置选项；例如，某些编解码器允许您控制动画的透明度或创建可缩放文件。3. 拖动 **质量（Quality）** 滑块来指定文件的压缩质量。最高值 100 会产生最高质量的动画文件，压缩程度最小。
7. 点击 **确定（OK）** 关闭 **AVI 选项（AVI Options）** 对话框。您的选项设置将在会话期间保存。
