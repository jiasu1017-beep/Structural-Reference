# 12.9.4 定义其他机械模型







您可以创建以下其他材料模型：
- **变形塑性**；请参见"定义变形塑性"
- **阻尼**；请参见"定义阻尼"
- **膨胀**；请参见"定义热膨胀"
- **脆裂**；请参见"定义脆裂"
- **状态方程**；请参见"定义状态方程"
- **黏性**；请参见"定义黏性"

### 定义变形塑性

Abaqus/Standard提供了变形理论Ramberg-Osgood塑性模型，用于开发韧性金属断裂力学应用中的完全塑性解。该模型最常用于小位移分析的静态加载，此时必须在模型的一部分中开发完全塑性解。

更多信息，请参见Abaqus Analysis User's Guide的第23.2.13节"变形塑性"。

**定义变形塑性模型：**

1. 在**编辑材料**对话框的菜单栏中，选择****机械**![](../graphics/images/arrow.gif)**变形塑性****。（有关显示**编辑材料**对话框的信息，请参见"创建或编辑材料"，第12.7.1节。）
2. 启用**使用温度相关数据**以定义依赖于温度的数据。**数据**表中会出现一个标有**温度**的列。
3. 在**数据**表中输入以下数据：**杨氏模量** 杨氏模量，*E*，定义为应力-应变曲线在零应力处的斜率。**泊松比** 泊松比，![](../graphics/usi_eqn00010.gif)。**屈服应力** 屈服应力，![](../graphics/usi_eqn00144.gif)。**指数** 塑性（非线性项）的硬化指数，*n*。**屈服偏移** 屈服偏移，![](../graphics/usi_eqn00007.gif)。**温度** 温度。您可能需要展开对话框以查看**数据**表中的所有列。有关输入数据的详细信息，请参见"输入表格数据"，第3.2.7节。
4. 单击**确定**创建材料并关闭**编辑材料**对话框。或者，您可以从**编辑材料**对话框的菜单中选择定义其他材料行为（有关更多信息，请参见"浏览和修改材料行为"，第12.7.2节）。

### 定义阻尼

您可以为Abaqus/Standard中的基于模态的分析和直接积分动态分析以及Abaqus/Explicit中的显式动态分析定义阻尼。更多信息，请参见Abaqus Analysis User's Guide的第6.3.1节"动态分析过程概述"和第26.1.1节"材料阻尼"。

**定义阻尼：**

1. 在**编辑材料**对话框的菜单栏中，选择****机械**![](../graphics/images/arrow.gif)**阻尼****。（有关显示**编辑材料**对话框的信息，请参见"创建或编辑材料"，第12.7.1节。）
2. 在**Alpha**字段中，输入![](../graphics/usi_eqn00272.gif)因子的值以创建Rayleigh质量比例阻尼。默认值为0。（单位：[T1](../popups/usb-int-iconventions-unitsym.md)。）
3. 在**Beta**字段中，输入![](../graphics/usi_eqn00273.gif)因子的值以创建Rayleigh刚度比例阻尼。默认值为0。（单位：T。）
4. 在**复合**字段中，输入临界阻尼分数的值，用于在计算模态复合阻尼因子时与该材料一起使用。默认值为0。（此值仅适用于Abaqus/Standard分析。）
5. 在**结构**字段中，输入![](../graphics/usi_eqn00274.gif)因子的值以创建虚刚度比例阻尼。默认值为0。
6. 单击**确定**创建材料并关闭**编辑材料**对话框。或者，您可以从**编辑材料**对话框的菜单中选择定义其他材料行为（有关更多信息，请参见"浏览和修改材料行为"，第12.7.2节）。

### 定义热膨胀

您可以通过在**编辑材料**对话框中输入热膨胀系数来定义热膨胀，或者，如果热应变是场和状态变量的复杂函数，则可以使用用户子程序[`UEXPAN`](../sub/sub-link.md#sub-xsl-uexpan)进行定义。更多信息，请参见Abaqus Analysis User's Guide的第26.1.2节"热膨胀"。

**定义热膨胀：**

1. 在**编辑材料**对话框的菜单栏中，选择****机械**![](../graphics/images/arrow.gif)**膨胀****。（有关显示**编辑材料**对话框的信息，请参见"创建或编辑材料"，第12.7.1节。）
2. 单击**类型**字段右侧的箭头，指定热膨胀的方向相关性。
3. 如果要使用用户子程序[`UEXPAN`](../sub/sub-link.md#sub-xsl-uexpan)定义热应变的增量，请启用**使用用户子程序UEXPAN**。
4. 如果您启用了**使用用户子程序UEXPAN**，请单击**确定**创建材料并关闭**编辑材料**对话框。或者，您可以从**编辑材料**对话框的菜单中选择定义其他材料行为（有关更多信息，请参见"浏览和修改材料行为"，第12.7.2节）。如果您选择在**编辑材料**对话框中直接指定热膨胀系数，请执行此过程的后续步骤。
5. 如果热膨胀依赖于温度或场变量，请在**参考温度**字段中输入值，![](../graphics/usi_eqn00275.gif)。
6. 启用**使用温度相关数据**以定义依赖于温度的数据。**数据**表中会出现一个标有**温度**的列。
7. 单击**场变量数量**字段右侧的箭头以增加或减少数据所依赖的场变量数量。
8. 在**数据**表中输入适用数据：**膨胀系数alpha** 各向同性热膨胀系数，![](../graphics/usi_eqn00276.gif)。（单位：[![](../graphics/usi_eqn00065.gif)1](../popups/usb-int-iconventions-unitsym.md)。）**alpha11, alpha22和alpha33** 三个值，用于定义正交热膨胀，![](../graphics/usi_eqn00277.gif)、![](../graphics/usi_eqn00278.gif)和![](../graphics/usi_eqn00279.gif)。（单位：[![](../graphics/usi_eqn00065.gif)1](../popups/usb-int-iconventions-unitsym.md)。）**alpha11, alpha22, alpha33, alpha12, alpha13和alpha23** 六个值，用于定义各向异性热膨胀，![](../graphics/usi_eqn00277.gif)、![](../graphics/usi_eqn00278.gif)、![](../graphics/usi_eqn00279.gif)、![](../graphics/usi_eqn00280.gif)、![](../graphics/usi_eqn00281.gif)和![](../graphics/usi_eqn00282.gif)。（单位：[![](../graphics/usi_eqn00065.gif)1](../popups/usb-int-iconventions-unitsym.md)。）**温度** 温度。**场变量*n*** 预定义场变量。您可能需要展开对话框以查看**数据**表中的所有列。有关输入数据的详细信息，请参见"输入表格数据"，第3.2.7节。
9. 单击**确定**创建材料并关闭**编辑材料**对话框。或者，您可以从**编辑材料**对话框的菜单中选择定义其他材料行为（有关更多信息，请参见"浏览和修改材料行为"，第12.7.2节）。

### 定义脆裂

您可以在Abaqus/Explicit中使用脆裂模型，用于混凝土行为以受拉开裂为主且受压失效不重要的应用。更多信息，请参见Abaqus Analysis User's Guide的第23.6.2节"混凝土断裂模型"。

#### 定义脆裂模型

Abaqus/Explicit中的脆裂模型在脆性行为占主导地位且假设材料在受压时为线性弹性的应用中最为准确。您可以将此模型用于素混凝土和其他材料（如陶瓷或脆性岩石），但它主要用于钢筋混凝土结构的分析。更多信息，请参见Abaqus Analysis User's Guide的第23.6.2节"混凝土断裂模型"。

**定义脆裂模型：**

1. 在**编辑材料**对话框的菜单栏中，选择****机械**![](../graphics/images/arrow.gif)**脆裂****。（有关显示**编辑材料**对话框的信息，请参见"创建或编辑材料"，第12.7.1节。）
2. 单击**类型**字段右侧的箭头，选择定义开裂后行为的方法：- 选择**应变**以通过直接输入失效后应力-应变关系来指定开裂后行为。- 选择**位移**以通过直接输入失效后应力/位移关系来定义开裂后行为。- 选择**GFI**以通过输入失效应力和I型断裂能来定义开裂后行为。
3. 启用**使用温度相关数据**以定义依赖于温度的数据。**数据**表中会出现一个标有**温度**的列。
4. 单击**场变量数量**字段右侧的箭头以增加或减少数据所依赖的场变量数量。
5. 如果从**类型**选项列表中选择了**应变**或**位移**，请在**数据**表中输入以下数据：**开裂后直接应力** 开裂后剩余的直接应力，![](../graphics/usi_eqn00283.gif)。（单位：[![](../graphics/usi_eqn00184.gif)](../popups/usb-int-iconventions-unitsym.md)。）**直接开裂应变** 直接开裂应变，![](../graphics/usi_eqn00284.gif)。（如果您从**类型**选项列表中选择了**应变**，请输入此值。）**直接开裂位移** 直接开裂位移，![](../graphics/usi_eqn00285.gif)。（单位：[L](../popups/usb-int-iconventions-unitsym.md)。）（如果您从**类型**选项列表中选择了**位移**，请输入此值。）**温度** 温度。**场变量*n*** 预定义场变量。您可能需要展开对话框以查看**数据**表中的所有列。有关输入数据的详细信息，请参见"输入表格数据"，第3.2.7节。
6. 如果从**类型**选项列表中选择了**GFI**，请在**数据**表中输入以下数据：**失效应力** 失效应力，![](../graphics/usi_eqn00286.gif)。（单位：[![](../graphics/usi_eqn00184.gif)](../popups/usb-int-iconventions-unitsym.md)。）**I型断裂能** I型断裂能，![](../graphics/usi_eqn00287.gif)。（单位：[![](../graphics/usi_eqn00193.gif)](../popups/usb-int-iconventions-unitsym.md)。）**温度** 温度。**场变量*n*** 预定义场变量。您可能需要展开对话框以查看**数据**表中的所有列。有关输入数据的详细信息，请参见"输入表格数据"，第3.2.7节。
7. 从**子选项**菜单中选择**脆性剪切**以定义材料的开裂后剪切行为。有关详细信息，请参见"定义脆性剪切"。
8. 如果需要，从**子选项**菜单中选择**脆性失效**以指定脆性失效准则。有关详细信息，请参见"定义脆性失效"。
9. 单击**确定**创建材料并关闭**编辑材料**对话框。或者，您可以从**编辑材料**对话框的菜单中选择定义其他材料行为（有关更多信息，请参见"浏览和修改材料行为"，第12.7.2节）。

#### 定义脆性剪切

断裂模型的一个重要特征是，虽然裂缝萌生仅基于I型断裂，但开裂后的行为包括II型和I型。

II型剪切行为基于剪切行为取决于裂缝开口量的常见观察结果。更具体地说，开裂剪切模量随着裂缝的张开而减小。因此，Abaqus/Explicit提供了一种剪切保留模型，其中开裂后剪切刚度定义为裂缝应变跨度的函数。

您必须提供开裂后剪切数据来完成脆裂模型定义。更多信息，请参见Abaqus Analysis User's Guide的第23.6.2节"混凝土断裂模型"中的"剪切保留模型"。

**定义脆性剪切：**

1. 如"定义脆裂模型"中所述创建材料模型。
2. 在**编辑材料**对话框的**子选项**菜单中，选择**脆性剪切**。将出现**子选项编辑器**。
3. 单击**类型**字段右侧的箭头，选择指定开裂后剪切行为的方法：- 选择**保留因子**以通过直接输入剪切保留因子-裂缝开口应变关系来指定开裂后剪切行为。- 选择**幂律**以通过输入幂律剪切保留模型的材料参数来指定开裂后剪切行为。
4. 启用**使用温度相关数据**以定义依赖于温度的数据。**数据**表中会出现一个标有**温度**的列。
5. 单击**场变量数量**字段右侧的箭头以增加或减少数据所依赖的场变量数量。
6. 如果从**类型**选项列表中选择了**保留因子**，请在**数据**表中输入以下数据：**剪切保留因子** 剪切保留因子，![](../graphics/usi_eqn00288.gif)。**裂缝开口应变** 裂缝开口应变，![](../graphics/usi_eqn00284.gif)。**温度** 温度。**场变量*n*** 预定义场变量。您可能需要展开对话框以查看**数据**表中的所有列。有关输入数据的详细信息，请参见"输入表格数据"，第3.2.7节。
7. 如果从**类型**选项列表中选择了**幂律**，请在**数据**表中输入以下数据：**e和p** 材料参数![](../graphics/usi_eqn00289.gif)和*p*。**温度** 温度。**场变量*n*** 预定义场变量。您可能需要展开对话框以查看**数据**表中的所有列。有关输入数据的详细信息，请参见"输入表格数据"，第3.2.7节。
8. 单击**确定**返回**编辑材料**对话框。

#### 定义脆性失效

当材料点上的一个、两个或所有三个局部直接开裂应变或位移分量达到定义为失效应变或位移的值时，该材料点失效，所有应力分量设置为零。如果元素中的所有材料点都失效，则该元素从网格中移除。更多信息，请参见Abaqus Analysis User's Guide的第23.6.2节"混凝土断裂模型"中的"脆性失效准则"。

**定义脆性失效：**

1. 如"定义脆裂模型"中所述创建材料模型。
2. 在**编辑材料**对话框的**子选项**菜单中，选择**脆性失效**。将出现**子选项编辑器**。
3. 从**失效准则**列表中选择一个选项：- 选择**单向**以表示当任何局部直接开裂应变（或位移）分量达到失效值时，将移除元素。- 选择**双向**以表示当任何两个直接开裂应变（或位移）分量达到失效值时，将移除元素。- 选择**三向**以表示当所有三个可能的直接开裂应变（或位移）分量达到失效值时，将移除元素。
4. 启用**使用温度相关数据**以定义依赖于温度的数据。**数据**表中会出现一个标有**温度**的列。
5. 单击**场变量数量**字段右侧的箭头以增加或减少数据所依赖的场变量数量。
6. 在**数据**表中输入以下数据：**直接开裂失效应变或位移** 您输入的值取决于在**编辑材料**对话框中选择指定开裂后行为的方法（如"定义脆裂模型"中所述）。如果您选择了**应变**，请输入直接开裂失效应变，![](../graphics/usi_eqn00290.gif)。如果您选择了**位移**或**GFI**，请输入直接开裂失效位移，![](../graphics/usi_eqn00291.gif)。（单位：[L](../popups/usb-int-iconventions-unitsym.md)。）**温度** 温度。**场变量*n*** 预定义场变量。
7. 单击**确定**返回**编辑材料**对话框。

6. 如果在步骤2中选择了**表格**，请在**数据**表中输入以下数据。体积应变值必须按降序排列（即从最拉伸到最压缩状态）。**f1** ![](../graphics/usi_eqn00300.gif)。（单位：[FL2](../popups/usb-int-iconventions-unitsym.md)。）**f2** ![](../graphics/usi_eqn00301.gif)。（无量纲。）**epsilon_vol** 体积应变![](../graphics/usi_eqn00302.gif)。（无量纲。）有关输入数据的详细信息，请参见"输入表格数据"，第3.2.7节。

8. 如果在步骤2中选择了**JWL**，请从**子选项**菜单中选择**起爆点**以为炸药材料定义起爆点。有关详细说明，请参见"为爆炸材料定义起爆点"。
9. 如果在步骤2中选择了**Us-Up**或**表格**，您可以从**子选项**菜单中选择**Eos压实**以为多孔韧性材料指定塑性压实行为。有关详细说明，请参见"为状态方程定义塑性压实行为"。
10. 单击**确定**关闭**编辑材料**对话框。或者，您可以从**编辑材料**对话框的菜单中选择定义其他材料行为（有关更多信息，请参见"浏览和修改材料行为"，第12.7.2节）。

#### 定义点火和生长状态方程

这种类型的状态方程模拟固体高能炸药的冲击起爆和爆轰波传播。更多信息，请参见Abaqus Analysis User's Guide的第25.2.1节"状态方程"中的"点火和生长状态方程"。

**定义点火和生长状态方程**

1. 如"定义状态方程"中所述定义点火和生长状态方程。
2. 在**爆轰能量**字段中，输入![](../graphics/usi_eqn00303.gif)的值。默认值为0。
3. 在**固相**选项卡页上，在**数据**表中输入未反应固体炸药的以下材料常数：**A和B** 材料常数*A*和*B*。（单位：[FL2](../popups/usb-int-iconventions-unitsym.md)。）**omega, R1和R2** 材料常数![](../graphics/usi_eqn00122.gif)、![](../graphics/usi_eqn00294.gif)和![](../graphics/usi_eqn00295.gif)。（无量纲。）有关输入数据的详细信息，请参见"输入表格数据"，第3.2.7节。
4. 在**气相**选项卡页上，在**数据**表中输入反应气体产物的以下材料常数：**A和B** 材料常数*A*和*B*。（单位：[FL2](../popups/usb-int-iconventions-unitsym.md)。）**omega, R1和R2** 材料常数![](../graphics/usi_eqn00122.gif)、![](../graphics/usi_eqn00294.gif)和![](../graphics/usi_eqn00295.gif)。（无量纲。）有关输入数据的详细信息，请参见"输入表格数据"，第3.2.7节。
5. 在**反应速率**选项卡页上，在**数据**表中输入以下数据：**I** 初始压力。（单位：[T1](../popups/usb-int-iconventions-unitsym.md)。）**a** 产物余容。（无量纲。）**b** 未反应分数的指数（点火项）。（无量纲。）**x** 指数（点火项）。（无量纲。）**G1** 第一次燃烧率系数。（单位：[T1](../popups/usb-int-iconventions-unitsym.md)。）**c** 未反应分数的指数（生长项）。（无量纲。）**d** 反应分数的指数（生长项）。（无量纲。）**y** 压力指数（生长项）。（无量纲。）**G2** 第二次燃烧率系数。（单位：[T1](../popups/usb-int-iconventions-unitsym.md)。）**e** 未反应分数的指数（完成项）。（无量纲。）**g** 反应分数的指数（完成项）。（无量纲。）**z** 压力指数（完成项）。（无量纲。）**Fig(max)** 初始反应分数，![](../graphics/usi_eqn00304.gif)。（无量纲。）**FG1(max)** 生长项的最大反应分数，![](../graphics/usi_eqn00305.gif)。（无量纲。）**FG2(min)** 完成项的最小反应分数，![](../graphics/usi_eqn00306.gif)。（无量纲。）有关输入数据的详细信息，请参见"输入表格数据"，第3.2.7节。
6. 在**气体比热**选项卡页上，输入反应气体产物的比热数据。1. 启用**使用温度相关数据**以定义依赖于温度的数据。**数据**表中会出现一个标有**温度**的列。2. 单击**场变量数量**字段右侧的箭头以增加或减少数据所依赖的场变量数量。3. 在**数据**表中输入以下数据：**比热** 反应气体产物的比热，按单位质量计。（单位：[JM1![](../graphics/usi_eqn00065.gif)1](../popups/usb-int-iconventions-unitsym.md)。）**温度** 温度。**场变量*n*** 预定义场变量。您可能需要展开对话框以查看**数据**表中的所有列。有关输入数据的详细信息，请参见"输入表格数据"，第3.2.7节。
7. 单击**确定**创建材料并关闭**编辑材料**对话框。或者，您可以从**编辑材料**对话框的菜单中选择定义其他材料行为（有关更多信息，请参见"浏览和修改材料行为"，第12.7.2节）。

#### 为状态方程定义塑性压实行为

如果您定义了线性*Us-Up*或表格状态方程（如"定义状态方程"中所述），您可以使用**子选项编辑器**为多孔韧性材料指定塑性压实行为。更多信息，请参见Abaqus Analysis User's Guide的第25.2.1节"状态方程"中的"*P--α*状态方程"。

**定义塑性压实：**

3. 在**多孔材料中的参考音速**字段中，输入![](../graphics/usi_eqn00307.gif)的值。（单位：[LT1](../popups/usb-int-iconventions-unitsym.md)。）
4. 在**卸载材料的孔隙率值**字段中，输入![](../graphics/usi_eqn00308.gif)的值。（无量纲。）
5. 在**初始化塑性行为所需的压力**字段中，输入![](../graphics/usi_eqn00309.gif)的值。（单位：[FL2](../popups/usb-int-iconventions-unitsym.md)。）
6. 在**所有孔隙被压碎时的压实压力**字段中，输入![](../graphics/usi_eqn00310.gif)的值。（单位：[FL2](../popups/usb-int-iconventions-unitsym.md)。）
7. 单击**确定**返回**编辑材料**对话框。

#### 为爆炸材料定义起爆点

您可以为爆炸材料定义任意数量的起爆点。必须定义点的坐标以及起爆延迟时间。每个材料点响应它看到的第一个起爆点。更多信息，请参见Abaqus Analysis User's Guide的第25.2.1节"状态方程"中的"JWL高能炸药状态方程"。

**定义起爆点：**

1. 如"定义状态方程"中所述定义Jones-Wilkins-Lee (JWL)状态方程。

3. 在**数据**表中输入以下数据：**X** 起爆点的坐标1。**Y** 起爆点的坐标2。**Z** 起爆点的坐标3。**起爆延迟时间** 起爆延迟时间（总时间，如Abaqus Analysis User's Guide的第1.2.2节"公约"中所定义）。默认值为0。有关输入数据的详细信息，请参见"输入表格数据"，第3.2.7节。
4. 单击**确定**返回**编辑材料**对话框。
