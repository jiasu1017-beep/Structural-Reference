# 4.1.26 UVARM





### 4.1.26 [`UVARM`](../sub/sub-link.md#sub-xsl-uvarm)

**产品：**Abaqus/Standard  

### 测试单元

B21、B31

C3D8、C3D8R、C3D8T、CAX4、CAX4E、CAXA42、CPE4R、CPE4T、CPE8RP、CPS4R

DC1D2、DC2D4、DC3D8

M3D4、M3D4R

S4、S4R、S8R、SAX1、SAXA12

T3D2、T3D3T

### 测试功能

用于将用户输出变量定义为标准输出变量的函数的用户子程序。

### 问题描述

这组验证问题测试传入[`UVARM`](../sub/sub-link.md#sub-xsl-uvarm)的许多变量，以及通过实用程序子程序`GETVRM`读取的积分点数量。大多数测试是单元素案例，将用户定义的输出变量直接设置为积分点数量（如应力和应变）。这些测试还验证用户定义的输出变量被正确传递到求解过程。

### 结果与讨论

结果验证子程序`GETVRM`拾取正确的变量并将它们正确传递到输出文件。
