# 10.3 为弹塑性问题选择单元







金属塑性变形的不可压缩性质对可用于弹塑性模拟的单元类型施加了限制。限制的产生是因为对不可压缩材料行为进行建模会向单元添加运动约束；在这种情况下，限制约束单元积分点处的体积保持恒定。在某些单元类别中，添加这些不可压缩性约束会使单元过度约束。当这些单元无法解析所有这些约束时，它们会遭受*体积锁定*，导致响应过于刚硬。体积锁定表现为静水压力应力从单元到单元或从积分点到积分点的快速变化。

在 Abaqus/Standard 中，完全积分的二阶实体单元在模拟不可压缩材料行为时非常容易受到体积锁定，因此不应在弹塑性模拟中使用。Abaqus/Standard 中的完全积分一阶实体单元不会遭受体积锁定，因为 Abaqus 实际上在这些单元中使用恒定体积应变。因此，它们可以安全地用于塑性问题。

减缩积分实体单元具有更少的积分点来满足不可压缩性约束。因此，它们不会被过度约束，可用于大多数弹塑性模拟。Abaqus/Standard 中的二阶减缩积分单元如果应变超过 20–40%，应谨慎使用，因为在这种幅度下它们可能遭受体积锁定。这种效果可以通过网格细化来减少。

如果您必须在 Abaqus/Standard 中使用完全积分的二阶单元，请使用混合版本，这些版本设计用于模拟不可压缩行为；但是，这些单元中的额外自由度会使分析在计算上更昂贵。

有一类改进的二阶三角形和四面体单元可用，与一阶三角形和四面体单元相比，它们提供更好的性能，并避免了一些存在于传统二阶三角形和四面体单元中的问题。特别是，这些单元表现出最小的剪切和体积锁定。这些单元在 Abaqus/Standard 中作为完全积分和混合单元的补充可用；它们是 Abaqus/Explicit 中唯一可用的二阶连续体（实体）单元。
