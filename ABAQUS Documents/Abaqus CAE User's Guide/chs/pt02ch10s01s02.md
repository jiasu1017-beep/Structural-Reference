# 10.1.2 关联接口可以做什么？





关联接口是可选的附加产品，可简化CAD系统和Abaqus/CAE之间传输模型数据的过程。关联接口使用CAD连接工具集在Abaqus/CAE和运行关联接口插件的CAD系统之间创建连接。关联接口插件可用于以下CAD系统：
- CATIA V6
- CATIA V5
- NX (Unigraphics)
- SolidWorks
- Pro/ENGINEER

关联接口允许从CAD系统到Abaqus/CAE的关联导入模型。您可以使用单击将整个装配从CAD系统导出到Abaqus/CAE，并且可以在CAD系统中修改模型并使用连接快速更新Abaqus/CAE中的模型。当您从CAD系统导入修改后的模型时，在Abaqus/CAE中创建的特征（如载荷、边界条件、集合和曲面）会更新。除了修改的零件外，对装配中实例位置的任何更改也会导出到Abaqus/CAE。

当您在同一台计算机上运行CAD系统和Abaqus/CAE，并根据分析结果迭代设计模型时，关联导入非常有用。[图10-1](pt02ch10s01s02.md#usi-catiacae)显示了使用关联导入在SolidWorks和Abaqus/CAE之间建立的连接。

在迭代设计时，您可以使用CAD系统或Abaqus/CAE修改模型。如果您将模型从CAD系统导入Abaqus/CAE，然后在Abaqus/CAE中向模型添加特征，则下次从CAD系统导入时，Abaqus/CAE会在模型中重新生成这些特征。您在Abaqus/CAE中做出的几何修改（如分区、倒角等）也将在每次从CAD系统导入模型时重新生成。如果您在CAD系统中做出的更改显著改变了模型的拓扑结构，Abaqus/CAE可能无法重新生成特征。在Abaqus/CAE中做出的几何修改不会传播到原始CAD模型，尽管Pro/ENGINEER关联接口确实提供了一种从Abaqus/CAE内部修订原始Pro/ENGINEER模型中某些几何实体的方法（请参阅["更新导入模型中的几何参数，"第60.2节"](pt06ch60s02.md))。

关联接口还允许您将CAD系统中的装配保存为装配文件格式，随后可以导入到Abaqus/CAE。例如，CATIA V5关联接口将您的CATIA V5装配（`.CATProduct`）保存为装配文件（`.eaf`）格式，您可以手动导入到Abaqus/CAE。同样，CATIA V6关联接口将您的CATIA V6产品转换为CATIA V5装配，然后将其保存为您可以手动导入到Abaqus/CAE的装配文件格式。

有关在Abaqus/CAE和CAD系统之间创建连接的更多信息，请参阅[第60章，"CAD连接工具集"](pt06ch60.md)。有关关联接口支持的CAD软件版本的信息，请参阅Dassault Systmes知识库，网址为[www.3ds.com/support/knowledge-base](http://www.3ds.com/support/knowledge-base)。
