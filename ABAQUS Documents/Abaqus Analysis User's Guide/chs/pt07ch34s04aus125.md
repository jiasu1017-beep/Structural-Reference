### 概述

声学和冲击载荷用于模拟动态声学环境和冲击波作用。这些载荷类型在航空航天、汽车、军工等领域有广泛应用。

### 声学载荷类型

| 类型 | 描述 | 应用 |
| --- | --- | --- |
| ACOUSTIC WAVE | 声波载荷 | 声学分析 |
| PRESSURE | 声压载荷 | 声学-结构耦合 |
| IMPEDANCE | 声阻抗边界 | 吸收边界 |

### 声压载荷

| **输入文件用法：** | ```
*LOAD, TYPE=PRESSURE
element_or_element_set, magnitude
``` |
| --- | --- |

### 使用幅值曲线定义时变声压

| **输入文件用法：** | ```
*LOAD, TYPE=PRESSURE, AMPLITUDE=name
element_or_element_set, magnitude
``` |
| --- | --- |

### 冲击载荷

冲击载荷（SHOCK）用于模拟瞬态冲击波：

| **输入文件用法：** | ```
*SHOCK
node_or_node_set, dof, magnitude, duration
``` |
| --- | --- |

### 冲击载荷参数

| 参数 | 描述 |
| --- | --- |
| magnitude | 冲击载荷幅值 |
| duration | 冲击持续时间 |
| rise_time | 上升时间 |

### 声学-结构耦合载荷

在声学-结构耦合分析中，声压作为等效载荷作用于结构表面：

| **输入文件用法：** | ```
*COUPLED ACOUSTIC-STRUCTURE
...
*LOAD, TYPE=PRESSURE
element_set, acoustic_pressure
``` |
| --- | --- |

### 吸收边界条件

用于模拟无限声学域：

| **输入文件用法：** | ```
*BOUNDARY, TYPE=IMPEDANCE
node_set, impedance_value
``` |
| --- | --- |
