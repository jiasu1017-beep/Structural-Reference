# 15.17.3 定义摩擦

您可以为可用的相对运动分量定义摩擦效应。有关更多信息，请参见["连接器摩擦行为，"Abaqus分析用户指南第31.2.5节](../usb/usb-link.md#usb-elm-econnfrictionbehav)。

**要创建或编辑摩擦行为：**

1. 显示连接器截面编辑器。
   - 要创建新的连接器截面，请按照["创建连接器截面，"第15.12.11节"](pt03ch15s12s01.md)中概述的过程进行操作。
   - 要编辑现有的连接器截面，请从主菜单栏选择**连接器**![](../graphics/images/arrow.gif)**截面**![](../graphics/images/arrow.gif)**管理器**，从出现的列表中选择连接器截面，然后点击**编辑**。
2. 在**编辑连接器截面**对话框中，执行以下操作之一：
   - 要定义新的摩擦行为，请点击**添加**并从出现的菜单中选择**摩擦**。
   - 要编辑现有的摩擦行为，请从**行为选项**列表中选择该行为以显示该选项的关联数据字段。
3. 选择**摩擦模型**。有关更多信息，请参见["有哪些类型的摩擦模型可用？，"第15.8.3节"](pt03ch15s08s03.md)。
   - 选择**预定义**以使用预定义的摩擦模型。Abaqus显示发生摩擦的滑动方向。
   - 选择**用户定义**以指定施加摩擦力或力矩的滑动方向，并定义摩擦产生的接触力贡献。
4. 如果您要指定用户定义的摩擦模型，请选择**滑动方向**。
   - 选择**指定方向**以指定施加摩擦力或力矩的可用相对运动分量。您可以定义多个摩擦行为来为多个分量建模摩擦，但Abaqus只允许每个分量有一个摩擦行为。
   - 选择**使用力势计算**以使用力势计算瞬时滑动方向。选择**力势**标签页，并定义至少一个力势项。有关更多信息，请参见["指定势项，"第15.17.16节"](pt03ch15s17hlb16.md)。
5. 选择**粘附刚度**。
   - 选择**使用默认值**以让Abaqus计算适当的粘附刚度。
   - 选择**指定**以输入与滑动方向指定的摩擦行为方向关联的粘附刚度。
6. 配置**切向行为**。有关更多信息，请参见["配置连接器摩擦的切向行为，"第15.17.4节"](pt03ch15s17hlb04.md)。
7. 如果您要指定预定义的摩擦模型，请指定**预定义摩擦参数**。如果您要指定用户定义的摩擦模型，请指定**接触力**。有关更多信息，请参见["指定预定义的摩擦参数或接触力，"第15.17.5节"](pt03ch15s17hlb05.md)。
8. 选择以下之一：
   - 如果您想继续定义行为，请点击**添加**，选择所需的行为，并继续连接器截面定义。有关定义其他行为的说明，请参见["使用连接器截面编辑器，"第15.17节"](pt03ch15s17.md)。
   - 如果您想查看或编辑现有行为，请从**行为选项**列表中选择该行为。有关编辑行为的说明，请参见["使用连接器截面编辑器，"第15.17节"](pt03ch15s17.md)。
   - 如果您想保存连接器截面定义并退出编辑器，请点击**确定**。
