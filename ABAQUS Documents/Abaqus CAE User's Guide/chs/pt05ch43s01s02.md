# 43.1.2 绘制未变形形状的部件实例





您可以绘制未变形形状的部件实例。当您绘制部件实例时，Abaqus/CAE会显示部件实例，而不是显示整个模型。每次您在模型中显示或隐藏部件实例时，Abaqus/CAE都会重新生成整个模型的绘图。

**要绘制未变形形状的部件实例：**

1. 确保您当前查看的是未变形形状绘图（有关更多信息，请参见"理解未变形形状绘图"，第43.1节）。
2. 在**零件实例**列表中，选择要绘制的部件实例。Abaqus/CAE会在当前视口中绘制所选部件实例。
3. 重复此过程以添加更多部件实例。
4. 要移除所有部件实例，请在**零件实例**列表中选择**无**，然后继续选择要绘制的部件实例。

**提示：**您还可以使用**显示组**工具集来选择要绘制的数据子集。更多信息，请参见第78章"使用显示组显示模型的子集"。
