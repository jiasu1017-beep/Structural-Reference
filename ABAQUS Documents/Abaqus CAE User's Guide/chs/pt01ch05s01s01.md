# 5.1.1 相机模式和视图术语





视图是在视口中显示的模型或分析结果的二维表示——相机图像。Abaqus/CAE 使用单个相机在每个视口中创建视图。你可以选择两种相机模式来创建模型或结果的所需视图。默认模式允许你将相机放置在模型外部的任何位置。电影模式允许你将同一相机放置在模型内部以及外部。此外，电影模式提供两个裁剪平面，你可以使用它们在对象太靠近或太远离相机时将其从视图中消除。视图深度在默认模式下不受限制；视图中的对象可以直接位于相机前方，也可以位于距离较远的地方，使其小到无法在视口中看到。

视图操作工具可用，使你能够充分利用这两种相机模式。所有视图操作工具都可以在任一相机模式中使用，但某些视图操作工具的"替代模式"主要用于电影模式。例如，放大工具允许你在不移动相机的情况下放大当前视图；此工具的替代模式会将相机移近模型。两种视图操作的效果看起来相同；但在默认相机模式下，如果相机"碰到"模型的外边缘，替代模式将停止工作。在电影模式下，你可以使用放大工具的替代模式将相机移入模型并穿过模型。视图操作工具在["理解视图操作工具，" 第 5.2 节](pt01ch05s02.md)中有描述。

以下相机术语用于描述你在视口中看到的视图：

**相机目标**

相机目标是空间中的一个点，控制大多数视图操作中相机的移动方式。对于所有默认视图，相机目标与视图中所有对象的中心点重合。当你使用平移、旋转和放大视图操作工具的替代模式时，相机目标会偏离所有对象的中心。

**平截头体**

平截头体是电影相机模式下可见的三维空间。相机位置形成由左、右、顶、底平面组成的金字塔的顶点（与默认模式相同）。为了创建平截头体，向默认视图添加了两个额外的平面：近平面和远平面。只有在平截头体内的对象（或对象的部分）才会在电影相机模式下可见。

**视场角**

视场角是形成视图侧面的左右平面或上下平面之间角度中较大的一个。使用的角度取决于平截头体的形状（实际上是视口的形状）；在[图 5-1](pt01ch05s01s01.md#viw-camera) 的两个图像中，左右平面之间的角度较大；因此，此角度被指定为视场角。视场角适用于默认相机模式和电影相机模式；更改角度类似于在固定相机上调整焦距来扩展或收缩相机图像。

放大、框选缩放和自动拟合视图操作工具都通过更改视场角来调整视口中的视图大小。有关这些工具的更多信息，请参阅["理解视图操作工具，" 第 5.2 节](pt01ch05s02.md)。

**近平面**

近平面垂直于相机方向，仅在电影相机模式下有效。从相机到近平面的距离是对象能够靠近相机且仍保留在视图中的最近距离。默认相机的视图包括任意距离的对象，就好像近平面位于相机镜头正前方。

近平面是一个裁剪平面；它从视图中移除模型表面和边缘，而不切割模型。在可视化模块中，你可以切割模型使内部表面可见；有关详细信息，请参阅[第 80 章，"切割模型"](pt07ch80.md)。

**远平面**

与近平面一样，远平面垂直于相机方向，仅在电影相机模式下有效。从相机到远平面的距离是对象能够远离相机且仍保留在视图中的最远距离。默认相机的视图包括任意距离的对象，就好像远平面位于距相机镜头无限远的位置。

远平面是一个裁剪平面；它从视图中移除模型表面和边缘，而不切割模型。在可视化模块中，你可以切割模型使内部表面可见；有关详细信息，请参阅[第 80 章，"切割模型"](pt07ch80.md)。

你使用视图选项、视图操作工具和透视工具来更改相机模式设置，或更改相机、相机目标和你要查看的对象之间的关系。这些工具和选项的当前设置定义了当前视图。
