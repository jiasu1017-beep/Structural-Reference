# 19.4.3 协同执行编辑器





您可以使用协同执行编辑器选择模型，在各个作业编辑器中定义初始作业参数设置，以及更改协同执行超时值。创建协同执行后，协同执行编辑器中的作业参数将无法编辑；您必须编辑各个作业。

您可以通过从主菜单栏选择 ****协同执行**![](../graphics/images/arrow.gif)**创建**** 或 ****协同执行**![](../graphics/images/arrow.gif)**编辑**![](../graphics/images/arrow.gif)***协同执行名称***** 来显示协同执行编辑器。（您也可以在协同执行管理器中单击**创建**或**编辑**。）

协同执行编辑器包含以下用于指定作业参数的选项卡页面：

**[提交](pt03ch19s08hlb02.md)**

使用**提交**选项卡页面配置作业的提交属性，例如作业类型、运行模式和提交时间。您还可以使用提交选项卡页面指定将作业提交到由本地 Abaqus 环境文件或系统管理员配置的远程队列。

**[常规](pt03ch19s08hlb06.md)**

使用**常规**选项卡页面配置作业设置，例如分析输入文件处理器打印输出和用于临时文件的目录名称。

**[内存](pt03ch19s08hlb07.md)**

使用**内存**选项卡页面配置分配给 Abaqus 分析的内存量。

**[并行化](pt03ch19s08s01.md)**

使用**并行化**选项卡页面配置 Abaqus 分析作业的并行执行，例如要使用的处理器数量和并行化方法。
