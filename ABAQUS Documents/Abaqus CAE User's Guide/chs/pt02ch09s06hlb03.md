# 9.7.3 升级模型数据库或输出数据库





来自先前版本的Abaqus的输出和模型数据库文件必须在打开时升级到当前版本。

要永久升级输出数据库，您必须以写权限打开文件并在提示时转换文件，或使用**abaqus upgrade**实用程序（请参阅["输出数据库升级实用程序，"Abaqus Analysis User's Guide第3.2.18节"](../usb/usb-link.md#usb-int-dupgradeproc))。您只能使用**abaqus upgrade**实用程序从数据库所在的系统升级远程输出数据库。

当打开先前版本的模型数据库或输出数据库时，Abaqus/CAE会执行以下操作之一：
- 如果Abaqus/CAE有权写入原始文件（即，您选择以写权限打开的输出数据库文件或模型数据库文件），系统会提示您将文件转换为当前版本。在转换期间，Abaqus/CAE会创建原始模型或输出数据库及与模型数据库关联的日志文件的备份；转换后的数据库文件和新日志文件使用原始文件名保存在当前目录（您打开Abaqus/CAE的目录）中。如果数据库是从当前目录以外的目录打开的，该目录仍将包含具有原始文件名的文件原始版本。转换完成后，Abaqus/CAE会创建一个名为`*file_name*-upgrade.log`的日志文件，指示转换结果。对于模型数据库文件的升级，Abaqus/CAE还会显示一个对话框，提供转换日志文件的名称，并包含**查看转换日志文件（View the conversion log file）**选项。切换此选项，然后单击**确定（OK）**以在Abaqus/CAE对话框中显示转换日志文件，您可以从该对话框浏览日志文件或搜索其内容以查找错误消息。
- 如果您以只读方式打开本地输出数据库文件，Abaqus/CAE会自动创建转换后的输出数据库版本，并保存到临时位置。转换后的输出数据库文件会保存到系统中`$TMPDIR`（UNIX）或`TEMP`（Windows）环境变量定义的目录；如果未定义此变量，新文件会保存到当前工作目录。当您退出Abaqus/CAE时，此临时版本的输出数据库文件会被删除。
- 如果您打开远程输出数据库文件，Abaqus/CAE会尝试创建转换后的输出数据库版本，并保存到临时位置。转换后的输出数据库文件会保存到远程系统上的`/tmp`目录或远程系统上`$TMPDIR`环境变量定义的目录。如果未定义临时目录，数据库升级会失败。当您退出Abaqus/CAE时，此临时版本的输出数据库文件会被删除。

当您升级较旧的模型数据库时，升级过程可能会将您手动添加到模型中的关键词放在升级后输入文件中的错误位置。因此，您在"作业（Job）"模块中提交模型进行分析时可能会遇到问题。如果是这种情况，您应该打开升级后的模型，返回**关键词编辑器（Keywords Editor）**，然后单击**放弃所有编辑（Discard All Edits）**来删除您添加的所有关键词。然后您可以在输入文件中正确位置重新创建关键词。
