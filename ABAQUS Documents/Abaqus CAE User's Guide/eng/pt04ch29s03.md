# 29.3 Creating point-based fasteners







You can create point-based fasteners in the Interaction module by picking each positioning point from the viewport or by reading the coordinates of the positioning points from a file. You can allow Abaqus/CAE to project the positioning point to the closest face along a normal to the surface, or you can specify the direction vector along which the point is projected to the closest face. For more information, see ["About fasteners," Section 29.1](pt04ch29s01.md).

**To create point-based fasteners:**

1. From the main menu bar in the Interaction module, select ****Special**![](../graphics/images/arrow.gif)**Fasteners**![](../graphics/images/arrow.gif)**Create****. Abaqus/CAE displays the **Create Fasteners** dialog box. **Tip:**You can also create point-based fasteners using the ![](../graphics/ico_fastener.png) tool in the Interaction module toolbox.
2. From the **Create Fasteners** dialog box that appears, select **Point-based** fasteners and click **Continue**.
3. Select the positioning points, and click **Done**. You can select attachment points, reference points, or nodes from an orphan mesh. For more information, see ["Creating attachment points by picking or reading from a file," Section 59.3](pt06ch59hla01.md); ["Creating attachment points by choosing a direction and a spacing," Section 59.4](pt06ch59hla02.md); and ["Creating patterns of attachment points based on edges," Section 59.5](pt06ch59hla03.md). Abaqus/CAE displays the **Edit Fasteners** dialog box.
4. On the **Domain**, **Criteria**, **Property**, **Formulation**, and **Adjust** tabbed pages, configure settings such as the region where the points will be created, the connector section, the coupling type, and the influence radius as described in the following procedures: - ["Defining the surfaces where fasteners will be connected," Section 29.3.1](pt04ch29s03hlb01.md) - ["Defining the fastener attachment zone," Section 29.3.2](pt04ch29s03hlb02.md) - ["Defining the fastener properties," Section 29.3.3](pt04ch29s03hlb03.md) - ["Defining the fastener formulation," Section 29.3.4](pt04ch29s03hlb04.md) - ["Adjusting the fastener definition," Section 29.3.5](pt04ch29s03hlb05.md)
5. Click **OK** to create the point-based fasteners. Symbols appear in the viewport that represent the positioning points of the point-based fasteners that you just created.
