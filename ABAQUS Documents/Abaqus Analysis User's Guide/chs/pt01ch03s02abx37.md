# 3.2.37 将 Moldflow 数据转换为 Abaqus 输入文件





**产品：**Abaqus/Standard

##### **参考**

- [『Abaqus 执行程序：概述』，第 3.1.1 节](pt01ch03s01abo02.md)
- [『Moldflow 转换示例』，Abaqus 例题指南第 1.3.19 节](../exa/exa-link.md#exa-sta-moldflow)

### 概述

Autodesk, Inc. 的 Moldflow Plastics Insight（本节中简称为 Moldflow）对塑料注射成型过程进行建模。Moldflow 模拟的结果包括塑料零件中材料属性和残余应力的计算。

**abaqus moldflow** 转换器将来自 Moldflow 分析的有限元模型信息转换为部分 Abaqus 输入文件。转换器需要由 Moldflow 分析创建的 Moldflow 接口文件。（有关更多信息，请参阅『Moldflow 接口文件』（第 3.2.37 节）。）

对于中面模拟，**abaqus moldflow** 转换器读取由 **abaqus moldflow** 转换器 Version MPI 3 或更高版本创建的接口（`.pat` 和 `.osp`）文件。

对于使用 Moldflow Version MPI 6 的三维实体模拟，转换器读取使用 Visual Basic 脚本 `mpi2abq.vbs` 创建的接口（`.inp` 和 `.xml`）文件。此脚本是 Abaqus 安装的一部分，通常位于 `*moldflow_install_dir*/Plastic Insight 6.0/data/commands` 目录中。

### 使用转换器

以下过程总结了 **abaqus moldflow** 转换器的典型用法：

1. 运行 Moldflow 模拟。
2. 导出数据，如下所示： - 对于中面 Moldflow 模拟，将有限元网格数据导出到名为 `*job-name*.pat` 的文件，将结果数据（材料属性和残余应力）导出到名为 `*job-name*.osp` 的文件。 - 对于使用 Moldflow Version MPI 6 的三维实体模拟，运行 Visual Basic 脚本 `mpi2abq.vbs`，将有限元网格数据导出到名为 `*job-name*_mesh.inp` 的文件，将结果数据导出到 `.xml` 文件。
3. 运行 **abaqus moldflow** 转换器，从 Moldflow 接口文件创建部分 Abaqus 输入文件。
4. 编辑 Abaqus 输入文件以添加适当的分析数据（例如，添加边界条件和平分析骤数据）。
5. 提交 Abaqus 输入文件进行分析。

#### 有限元网格数据

Moldflow 接口文件包含有限元网格数据。
- 对于中面模拟，网格数据位于 Patran 中性文件中，包含节点坐标、单元拓扑和单元属性。
- 对于三维实体模拟，网格数据位于 Abaqus 输入文件中，包含节点坐标、单元拓扑、单元属性以及消除结构刚体模态所需的边界条件。网格文件中的实体单元始终是 4 节点四面体。转换器有一个选项可将它们转换为 10 节点四面体。

#### 材料属性数据

Moldflow 接口材料属性数据文件包含每个单元的弹性和热膨胀系数。对于中面模拟，这些属性可以是各向同性或正交各向异性。对于填充模型的三维实体模拟，这些属性是各向异性的。对于未填充模型的三维实体模拟，数据文件包含经过调整以表示物理各向同性材料的各向异性数据。

#### 残余应力数据

**abaqus moldflow** 转换器计算塑料零件冷却后的残余应力。这些残余应力可以转换为 Abaqus 结构分析的初始应力。
- 对于中面模拟，平面应力初始应力状态在与材料属性相同的方向上定义。材料坐标中的应力状态以主应力表示（剪应力为零）。
- 对于三维实体模拟，`*job-name*_initStresses.xml` 中每个单元的残余应力位于全局坐标系中。转换器将这些坐标转换为与材料属性相同的方向。

### 用于转换中面模拟的 Moldflow 数据的假设

对于中面模拟，**abaqus moldflow** 转换器对数据的拓扑和属性做了许多假设。这些假设列于下文，以确保与当前版本的 Abaqus 中可用的选项兼容。
- Moldflow 网格可以由 3 节点平面三角形单元以及表示流道和筋等零件的 2 节点一维单元组成。**abaqus moldflow** 转换器将三角形单元转换为相同的 Abaqus S3R 壳单元网格。Moldflow 网格中的一维单元不会被转换。
- **abaqus moldflow** 转换器创建的 Abaqus S3R 壳单元的层数等于 Moldflow 传递的层数，即 20。因此，传递给转换器的力学属性和应力数据适用于穿过厚度的 20 层。
- **abaqus moldflow** 转换器创建的 Abaqus 输入数据取决于接口（`.osp`）文件中定义的材料类型，如下所示： - 对于未填充的各向同性材料，Abaqus 假定以下内容： - 均匀壳公式。 - 各向同性材料常数。 - Abaqus 截面点初始应力从 Moldflow 穿透厚度积分点的值进行插值。 - 对于未填充的横向各向同性材料，Abaqus 假定以下内容： - 均匀壳公式。 - 以截面材料主方向以及相对于局部 Abaqus 坐标系的方向定义的横向各向同性材料常数。 - Abaqus 截面点初始应力从 Moldflow 穿透厚度积分点的值进行插值。 - 对于纤维填充材料，Abaqus 假定以下内容： - 复合壳公式。 - 以每层材料主方向以及相对于每层局部 Abaqus 坐标系的方向定义的层材料常数。 - Moldflow 穿透厚度积分点被视为每个 Abaqus 层的中点。 - 材料属性在每层中是恒定的。 - Abaqus 截面点初始应力与 Moldflow 穿透厚度积分点处的值相同，并且在每层中是恒定的。

**abaqus moldflow** 转换器生成的 Abaqus 输入文件不包含边界条件和载荷数据。您必须手动将此信息添加到输入文件中。

### 用于转换三维实体模拟的 Moldflow 数据的假设

对于三维实体模拟，**abaqus moldflow** 转换器对数据的拓扑和属性做了许多假设。这些假设列于下文，以确保与当前版本的 Abaqus 中可用的选项兼容。
- **abaqus moldflow** 转换器将四面体单元转换为相同的 Abaqus C3D4 或 C3D10 实体单元网格（有关更多信息，请参阅下面的命令行选项）。
- 各向异性材料常数以材料主方向表示。
- 材料属性在每个单元中是恒定的。
- 方向在 `*job-name*_principalDirections.xml` 中通过给出定义局部 1 和 2 方向的向量来定义。
- Moldflow 的 WARP3D 模块计算的残余应力在 `*job-name*_initStresses.xml` 中从全局坐标转换为局部材料方向，并用作 Abaqus 中的初始应力。
- 表示使用载荷的载荷和边界条件必须手动添加到输入文件中。对于使用 Moldflow Version MPI 6 的模拟，转换器创建的 Abaqus 输入文件包含足够的边界条件以从模型中消除刚体模态，以便分析可以轻松求解初始应力引起的响应。

### 为中面模拟创建的文件

**abaqus moldflow** 转换器读取 Moldflow 接口文件并创建相关文件。创建的文件取决于您在执行转换器时在命令行上包含的选项。对于中面模拟，**abaqus moldflow** 转换器创建一个部分 Abaqus 输入文件、一个中性文件和一个初始应力文件。

#### 部分 Abaqus 输入（`.inp`）文件

部分 Abaqus 输入文件包含由节点坐标、单元拓扑和截面定义组成的模型数据。它还包含一个带有默认输出请求的 [*STATIC](../key/key-link.md#usb-kws-hstatic) 步骤。如果您使用各向同性材料，输入文件包含材料属性数据。每个输入文件都以一系列注释开头，这些注释总结了 Moldflow 接口文件提供的数据以及数据如何转换为 Abaqus 输入文件。必须手动将此文件添加到边界条件、载荷和非默认输出请求等附加数据中。

#### 包含分层、空间变化材料数据的中性（`.shf`）文件

材料数据被翻译成适当格式的 ASCII 中性文件。此文件包含每个单元每层的层材料属性数据。Abaqus 输入文件中的 Abaqus [*ELASTIC](../key/key-link.md#usb-kws-melastic), TYPE=SHORT FIBER 和 [*EXPANSION](../key/key-link.md#usb-kws-mexpansion), TYPE=SHORT FIBER 选项指示 Abaqus/Standard 在初始化步骤期间从此文件读取材料数据。

**中性文件中的数据行：**

1. `.shf` 文件中的单元数。
2. 每个壳截面中的层数。

1. 单元标签。
2. 层标识符。
3. ![](../graphics/usb_eqn00007.gif)。
4. ![](../graphics/usb_eqn00009.gif)。
5. ![](../graphics/usb_eqn00030.gif)。
6. ![](../graphics/usb_eqn00273.gif)。
7. ![](../graphics/usb_eqn00274.gif)。
8. ![](../graphics/usb_eqn00275.gif)。
9. ![](../graphics/usb_eqn00276.gif)。
10. ![](../graphics/usb_eqn00277.gif)。
11. 纤维取向角（度），相对于默认单元取向测量。

根据需要重复此数据行，以定义不同单元内不同壳截面的上述参数。

#### 初始应力（`.str`）文件

来自 Moldflow 分析的残余应力数据被翻译成适当格式的 ASCII 中性文件。这些数据以每个截面点处局部 Abaqus 坐标系表示。Abaqus 输入文件中的 Abaqus [*INITIAL CONDITIONS](../key/key-link.md#usb-kws-minitialcond), TYPE=STRESS, SECTION POINTS 选项指示 Abaqus/Standard 在初始化步骤期间从此文件读取初始应力数据。

### 为三维实体模拟创建的文件

**abaqus moldflow** 转换器读取 Moldflow 接口文件并创建相关文件。创建的文件取决于您在执行转换器时在命令行上包含的选项。

如果您使用的是未填充模型，**abaqus moldflow** 转换器仅创建如下所述的部分 Abaqus 输入文件。对于使用填充模型的三维实体模拟，转换器可以创建其他文件，如下所述。

#### 部分 Abaqus 输入文件

部分 Abaqus 输入文件包含由节点坐标、单元拓扑和截面定义组成的模型数据。必须手动将服务载荷、边界条件以及非默认输出请求等附加数据添加到此文件。

边界条件数据足以消除刚体模态，也包含在内。

#### 包含各向异性材料属性数据的材料（`.mpt`）文件

来自 Moldflow 分析的材料数据被收集并放置在二进制文件中。写入文件的数据格式与为 Abaqus [*ELASTIC](../key/key-link.md#usb-kws-melastic), TYPE=ENGINEERING CONSTANTS 选项提供的信息相同。这些以每个单元的局部 Abaqus 坐标系定义。

#### 包含单元方向数据的方向（`.opt`）文件

定义材料属性和初始应力方向的方向被计算并放置在此二进制文件中。

#### 包含单元热膨胀系数数据的热膨胀（`.tpt`）文件

来自 Moldflow 分析的各向异性热膨胀数据被收集并放置在二进制文件中。这些以每个单元的局部 Abaqus 坐标系定义。

### 准备用于分析的 Abaqus 输入文件

一旦 **abaqus moldflow** 转换器创建了 Abaqus 输入文件，您必须在提交分析之前手动完成输入文件（详见『在 Abaqus 中定义模型』，第 1.3.1 节（第 1.3.1 节））。

#### 准备进行收缩和翘曲分析

收缩和翘曲分析计算模型从模具中取出后由于残余应力而引起的变形。通常只需要消除刚体模态。

在这种情况下，您必须确保残余应力已被转换。对于三维实体 Moldflow 模拟，足以约束刚体模态的边界条件会自动转换到输入文件。在其他情况下，您需要添加适当的边界条件以消除模型的刚体模态。

在某些情况下，当您必须考虑几何非线性和存在较大初始应力时，可能会出现收敛问题。您可以通过两个分析步骤来克服这些问题：
- 在第一步中约束所有位移自由度。
- 在第二步中，使用 OP=NEW 参数应用消除刚体模态的边界条件。

#### 准备进行服务载荷分析

服务载荷分析（带有适当的边界条件）评估模型的性能。您可以在有或无初始应力的情况下执行此分析。您必须在 Abaqus 输入文件中将适当的边界条件和载荷指定为历史数据。

#### 准备进行其他分析类型

只要您在 Abaqus 输入文件中指定了正确的边界条件和载荷，就可以对翻译后的模型执行任何 Abaqus/Standard 分析过程。此外，某些分析类型可能要求您在输入文件中指定附加材料常数、模型数据和/或求解控制。

### 命令行选项

**job**

此选项指定结果转换期间使用的输入和输出文件名。*job-name* 值用于构造默认 SIM 数据库文件名 `*job-name*.sim`。输出模态中性文件被命名为 `*job-name*.mnf`。

如果此选项从命令行中省略，系统将提示您输入此值。

**input**

此选项用于在文件名与 *job-name* 不同时指定包含 Moldflow 接口数据的文件的名称。

**midplane**

此选项用于将中面模拟的结果转换为具有三维（壳）单元的 Abaqus 模型。

**3D**

此选项用于将三维实体模拟的结果转换为具有实体单元的 Abaqus 模型。

**element_order**

此选项用于指定为三维实体模拟在部分输入文件中创建的单元的阶数。可能的值为 `1`（创建一阶单元 C3D4）或 `2`（创建二阶单元 C3D10）。默认值为 `2`。此选项仅在使用 **3D** 选项时有效。

**initial_stress**

此选项指定是否在模型中包含初始应力。此选项仅在使用 **3D** 选项时有效。

如果未包含 **initial_stress** 选项或 **initial_stress**=`off`，则不会转换初始应力。

如果 **initial_stress**=`on`，则初始应力将被写入输入文件。

**material**

此选项用于指定材料属性的写入位置。如果 **material**=`traditional`，则材料属性将被写入输入文件。否则，材料属性将被写入（二进制）`.mpt` 文件。不建议对大型模型使用 **material**=`traditional`，因为性能原因，因为每个单元都将有自己的 [*MATERIAL](../key/key-link.md#usb-kws-mmaterial) 定义。

**orientation**

此选项用于指定方向的写入位置。如果 **orientation**=`traditional`，则方向将被写入输入文件。否则，方向将被写入（二进制）`.opt` 文件。不建议对大型模型使用 **orientation**=`traditional`，因为性能原因，因为每个单元都将有自己的 [*ORIENTATION](../key/key-link.md#usb-kws-morientation) 定义。
