# 42.5.3 选择主场输出变量





您可以为[云图](pt05ch44.md)、[模型探测](pt05ch51.md)、基于等值面的视图切割以及[沿模型路径获取结果](pt05ch48.md)选择要显示的变量；这个变量称为主场输出变量。Abaqus/CAE还使用主场输出变量来[形成显示组](pt07ch78.md)、基于结果[应用颜色编码](pt07ch77s05.md)，以及显示来自当前模型数据库中模型的载荷、预定义场或相互作用。

当选择输出数据库时，Abaqus/CAE默认列出当前[步骤和帧](pt05ch42s01hlb01.md)的[输出数据库](pt02ch09s06hlb02.md)中所有可用的变量供您选择。描述左侧的星号表示该变量包含复数结果。

当选择当前模型数据库中的模型时，Abaqus/CAE默认列出模型当前步骤中所有可用的载荷、预定义场、边界条件和相互作用供您选择。所有这些可选项目的名称前都有括号中的字母来区分类别：**(L)**表示载荷，**(P)**表示预定义场，**(B)**表示边界条件，**(I)**表示相互作用。复数载荷或预定义场仅显示实部。

使用**Field Output**对话框中的**Primary Variable**选项来选择您想要的变量，以及如果适用的话，选择不变量或分量。有关单个输出变量标识符的信息，请参见["输出变量，" Abaqus Analysis User's Guide第4.2节](../usb/usb-link.md#usbovsect)。

**选择主场输出变量：**

1. 找到控制主场输出变量的选项。从主菜单栏，选择****Result**![](../graphics/images/arrow.gif)**Field Output****。在出现的对话框中点击**Primary Variable**选项卡。**Primary Variable**选项出现。**提示：**您也可以通过点击任何出现此按钮的对话框中的**Field Output**按钮来访问这些选项。要查看列出变量的完整描述，请通过拖动角落来增加对话框的宽度。
2. 要控制**Name**和**Description**列表中显示哪些变量：
   1. 切换**List only variables with results**以按变量的存储位置限制显示列表。限制列表有助于您通过例如仅显示积分点量来选择变量。当**List only variables with results**开启时，下拉菜单中会出现筛选选项。
   2. 点击**List only variables with results**箭头显示筛选选项。
   3. 点击**Name**和**Description**列表中要包含的变量位置的文本。文本出现在**List only variables with results**框中，**Name**和**Description**列表会刷新以仅包含具有该位置的变量。
3. 从**Name**和**Description**列表中，点击您想要的分析变量名。列表中描述左侧的星号表示该变量包含复数结果。选中的变量高亮显示。如果适用，对话框底部的**Component**和**Invariant**列表会刷新以显示可用的分量或不变量。
4. 如果**Component**或**Invariant**列表中有项目，点击您想要的分量或不变量。选中的分量或不变量高亮显示。
5. 当处于活动状态时，当前视口中的云图会更改以显示您指定的分析变量的值。如果处于活动状态，图例和状态块中的文本会更改以标识与图关联的变量。有关图例和状态块的更多信息，请参见["自定义图例，" 第56.1节](pt05ch56s01.md)和["自定义状态块，" 第56.3节](pt05ch56s03.md)。此外，Abaqus会刷新所有标识当前主变量的对话框。您的更改将保存到会话结束。
