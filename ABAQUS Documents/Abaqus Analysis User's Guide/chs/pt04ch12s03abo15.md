# 12.3.1 自适应重网格划分：概述






Abaqus/CAE 提供了一个自动化的过程来自适应地重新网格划分您的模型。自适应重网格划分过程的目标是接近或达到所选误差指标的指定模型及其伴随载荷历史的目標。参见 ["自适应技术," 第 12.1.1 节](pt04ch12s01aus77.md)，了解将此过程与此方法与其他 Abaqus 自适应方法的比较。

### 概述

将自适应重网格划分纳入 Abaqus/CAE 模型需要以下步骤：
- 您确定要应用一个或多个自适应重网格划分规则的模型区域。重网格划分规则定义应用它的步骤、误差指标输出变量及其目标、网格划分方法以及单元尺寸的任何约束。参见 ["什么是重网格划分规则？," Abaqus/CAE 用户指南第 17.13.1 节](../usi/usi-link.md#usi-mgn-conc-adaptivity-rulerole)。
- 您定义一系列分析作业，一个"自适应过程"，Abaqus/CAE 将运行这些作业以尝试满足您的重网格划分规则目标。参见 ["什么是自适应过程？," Abaqus/CAE 用户指南第 19.3.1 节](../usi/usi-link.md#usi-ana-adaptivity-steps)。

 基于这些重网格划分规则和自适应过程定义，Abaqus/CAE 迭代地：
- 执行 Abaqus/Standard 分析，这将根据您的重网格划分规则设置写出选定的误差指标输出变量（参见 ["选择影响自适应重网格划分的误差指标," 第 12.3.2 节](pt04ch12s03aus84.md)），
- 使用误差指标变量在大小函数中计算新网格的单元尺寸，尊重您可能指定的任何尺寸约束（参见 ["基于解的网格划分," 第 12.3.3 节](pt04ch12s03aus85.md)），以及
- 根据计算的单元尺寸在指定区域生成新网格。相邻区域也将被重新网格划分。

 这些迭代继续，直到：
- 所有重网格划分规则目标都满足，或
- 达到最大重网格迭代次数。

 参见 ["我的网格自适应何时停止迭代？," Abaqus/CAE 用户指南第 19.3.2 节](../usi/usi-link.md#usi-ana-adaptivity-automatic)，了解更多详情。[图 12.3.1--1](pt04ch12s03abo15.md#aadaptivity-bigloop) 显示了此过程中 Abaqus 产品和文件的交互。

### 典型应用

自适应重网格划分可以提高模拟结果的质量。在以下情况下，自适应重网格划分可能有用：
- 您不确定需要多么精细的网格才能达到特定的精度水平，或者网格可以多么粗略而不会对解精度产生不可接受的影响；
- 难以在感兴趣区域（如应力集中区域附近）设计足够精细的网格；或者
- 您事先不知道感兴趣的位置，例如塑性区域的形成。

使用自适应重网格划分研究螺栓封闭压力容器的热和应力行为的示例在 ["反应堆压力容器螺栓封闭的热应力分析," Abaqus 例题指南第 5.1.6 节](../exa/exa-link.md#exa-htr-reactor) 中提供。该示例包括一个 Python 脚本，您可以从 Abaqus/CAE 运行它来创建模型和重网格划分规则。第二个脚本允许您提交自适应过程并查看网格变化，因为 Abaqus/CAE 计算新的单元尺寸。

### 为自适应重网格划分准备模型

在执行自适应重网格划分时，您使用 Abaqus/CAE 执行以下操作：
- 创建模型并指定边界条件和载荷历史，
- 创建重网格划分规则，
- 创建自适应过程，以及
- 启动和监控自适应过程的进度。

#### 创建模型

创建模型并指定边界条件和载荷历史时不必考虑自适应重网格划分；但是，在使用自适应重网格划分之前，您必须执行以下操作：
- 创建模型的几何形状——您不能使用孤立网格部件——以及
- 提供一个初始的、公称网格。此网格可以相当粗略。但是，提供极其粗略的网格可能导致更多自适应重网格迭代，因为早期重网格迭代误差指标计算质量差。在典型情况下，您可以通过使用 Abaqus/CAE 中的默认部件实例网格播种来定义合理的初始网格。

#### 创建重网格划分规则

您使用 Abaqus/CAE 中的网格模块创建和配置重网格划分规则。参见 ["创建重网格划分规则," Abaqus/CAE 用户指南第 17.21.1 节](../usi/usi-link.md#usi-mgn-adaptivity-rule)，了解定义重网格规则的详细信息。参考 ["选择影响自适应重网格划分的误差指标," 第 12.3.2 节](pt04ch12s03aus84.md) 和 ["基于解的网格划分," 第 12.3.3 节](pt04ch12s03aus85.md)，了解用于确定修订网格尺寸分布的方法的详细信息。

### 使用临时分析执行自适应重网格划分

在某些情况下，您希望在进行完全详细的分析之前确定模型的适当网格，这可能包括许多步骤和复杂行为。"临时"分析通常可以与自适应重网格划分一起使用，以有效地确定为模型提供良好网格。临时分析可能包括您的完全详细分析的各种简化，例如：
- 用一个线性扰动步骤替换您的步骤，该步骤具有充分反映更一般载荷情况的载荷，
- 移除塑性和其他材料非线性，以及
- 禁用几何非线性。

临时分析方法可能导致不完全适合您最终载荷选择的网格。但是，从临时模型获取网格的成本可能远低于您的自适应过程考虑完全详细分析的所有复杂性的情况，并且您可能会发现细化的网格适用于各种分析情况。

### 特殊注意事项

一般来说，Abaqus 自适应重网格划分过程会自动迭代到更好的网格质量；但是，您应该注意某些注意事项。

#### 奇异性

应力奇异性通常由几何抽象引起，如重入角和弹性材料中尖锐边缘的接触，以及点载荷或突然结束的分布载荷区域。在这些情况下，奇异区域附近的应力场是无界的，并且没有网格细化能够解析正确的解。如果将自适应重网格划分过程应用于包含奇异性的模型区域，该过程将驱动靠近奇异性的单元变得非常小。最终结果可能是分析成本高昂且不切实际。

您可以使用以下技术防止具有奇异性的模型成本高昂的分析：
- 从重网格划分过程中排除奇异区域。您通过分割模型并仅将重网格划分规则分配给远离奇异性的区域来排除区域。
- 在重网格划分规则中应用最小单元尺寸约束。Abaqus/CAE 默认分配最小单元尺寸，这是默认部件实例网格种子的一部分。您可以修改此约束以在奇异性附近实现优质解，同时避免过度细化的网格。您还可以使用重网格划分规则来控制 Abaqus/CAE 细化单元尺寸的速率。单元尺寸约束可能阻止自适应过程实现指定的误差指标目标。
- 为重网格划分规则区域指定最大单元数。Abaqus/CAE 调整网格划分，使得生成的单元总数近似满足此约束。

[图 12.3.1--6](pt04ch12s03abo15.md#aadaptivity-costanderror) 中的示例显示了理想的收敛曲线。解误差指标单调快速地下降到所需的 25% 误差指标目标。伴随此误差指标下降的是计算成本的适度增加，以模型自由度或 Abaqus/Standard 中的时间衡量。某些情况可能会干扰这种理想的收敛曲线，如下所述：
- 如果您的初始网格太粗，误差指标变量可能质量不足，导致在下次迭代中网格不够改进。自适应重网格划分过程最终通常会创建高质量网格，即使初始网格相当粗略。但是，通过合理细化的初始网格可以避免一些网格迭代。
- 您在创建重网格划分规则时指定的最小单元尺寸约束和最大单元数约束可能阻止网格实现足够的细化（在奇异性的极端情况下总是如此）以满足您的误差指标目标。您可以通过放宽这些约束来满足您的目标；例如，通过减小最小单元尺寸。有关更多信息，请参见 ["什么是重网格划分规则？," Abaqus/CAE 用户指南第 17.13.1 节](../usi/usi-link.md#usi-mgn-conc-adaptivity-rulerole)。
- 除了产生导致大量单元的小网格尺寸外，奇异性还可能导致自适应过程无法实现误差目标或需要更多重网格迭代。如 ["奇异性"](pt04ch12s03abo15.md#usb-anl-aadpover-singularities) 中所述，您可以通过指定最小单元尺寸约束或最大单元数来控制计算成本。无论奇异性存在于重网格划分规则区域内，您都可能在误差指标结果中看到较差的收敛。
- 线性单元（C3D4、CPS4 等）和修正单元（C3D10M、CPS6M 等）与二次单元（C3D10、CPS6 等）相比收敛较慢，需要相对较多的单元来实现给定的误差目标。因此，只要可能，您应该使用二次单元。

#### 继续停止的自适应重网格划分过程

自适应重网格划分过程被设计为自动的——Abaqus/CAE 执行一系列分析，因为它继续细化您的网格。但是，在某些情况下，过程会停止，您会希望从最新的网格继续自适应重网格划分：
- 当您想要更改后续重网格迭代的重网格划分规则时，或
- 当自适应网格过程由于机器资源问题而未能完成时。

您可以通过重新提交现有自适应过程、创建和提交新的自适应过程或执行手动重新网格划分来继续自适应重网格划分过程。参见 ["手动调整大小和重新网格划分," Abaqus/CAE 用户指南第 17.21.6 节](../usi/usi-link.md#usi-mgn-adaptivity-remeshing)。

### 限制

自适应重网格划分需要使用 Abaqus/CAE，并且仅支持 Abaqus/Standard 过程。其他特定限制也适用。

#### 单元类型

Abaqus/CAE 只能对以下形状的单元执行自适应重网格划分（参见 ["我可以使用哪些网格控制与自适应重网格划分？," Abaqus/CAE 用户指南第 17.13.2 节](../usi/usi-link.md#usi-mgn-conc-adaptivity-elements)）：
- 平面连续体三角形和四边形
- 壳三角形和四边形
- 四面体

#### 过程

Abaqus/CAE 可以使用以下 Abaqus/Standard 过程执行重新网格划分：
- ["静态应力分析," 第 6.2.2 节](pt03ch06s02at01.md)（一般和线性扰动）。
- ["准静态分析," 第 6.2.5 节](pt03ch06s02at04.md)。
- ["非耦合热传递分析," 第 6.5.2 节](pt03ch06s05at18.md)。
- ["完全耦合热应力分析," 第 6.5.3 节](pt03ch06s05at19.md)。
- ["耦合热电分析," 第 6.7.3 节](pt03ch06s07at22.md)。
- ["耦合孔隙流体扩散和应力分析," 第 6.8.1 节](pt03ch06s08at26.md)。
