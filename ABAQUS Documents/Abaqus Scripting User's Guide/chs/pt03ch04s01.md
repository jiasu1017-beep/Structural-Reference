# 4.2 内边距和间距







布局管理器（和大多数组件）提供一些默认内边距，使组件之间保持一定间距。这些值通常位于组件参数列表的末尾。例如：

```
FXPacker(…, pl, pr, pt, pb, …)
```
一般来说，您应该接受内边距的默认值。但是，如果有嵌套的布局管理器，应该将内边距值设置为零。

布局管理器还提供其子组件之间的间距。这些值通常位于组件参数列表的末尾。例如：

```
FXPacker(…, hs, vs)
```
一般来说，您应该接受间距的默认值。

某些"复合"组件（如 `AFXTextField`、`AFXComboBox` 和 `AFXSpinner`）有两个内边距值——一个用于内部文本字段组件的内边距，另一个用于包含标签的整个组件的内边距。您可以通过将内边距值传递给组件构造函数来设置内部文本字段组件的内边距。您可以通过在组件上调用内边距方法之一来设置整个组件的内边距；例如 `setPadLeft`。
