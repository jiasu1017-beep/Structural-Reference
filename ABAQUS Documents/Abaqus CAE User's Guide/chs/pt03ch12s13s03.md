# 12.13.10 创建通用壳刚度截面





壳截面行为根据壳截面对拉伸、弯曲、剪切和扭转的响应来定义。请参见[Abaqus Analysis User's Guide的第29.6.4节，"壳截面行为"](../usb/usb-link.md#usb-elm-eshellsectionbehavior)。

通用壳刚度截面允许您直接以刚度矩阵和热膨胀响应的形式指定壳截面属性。这些数据完全定义壳截面的力学响应，因此截面定义中不需要材料引用。当您知道适用刚度和热应力矩阵的项时，通用壳刚度截面提供了一种高效灵活的方法来定义壳响应。请参见"使用通用壳截面定义截面行为"中的"直接指定等效截面属性以用于常规壳"，第29.6.6节Abaqus Analysis User's Guide。

通用壳刚度截面不能与可变厚度壳或连续壳一起使用。与其他预积分壳截面一样，通用壳刚度截面不通过其厚度进行积分，也不提供热传递。在Abaqus/Standard分析中，无法从通用壳刚度截面输出应力和应变。

### 创建通用壳刚度截面

**创建通用壳刚度截面：**

1. 从主菜单栏，选择****Section**![](../graphics/images/arrow.gif)**Create****。出现**Create Section**对话框。
   **提示：**您也可以在**Section Manager**中点击**Create**，或在Property模块工具箱中选择创建截面工具![](../graphics/ico_section.png)。
2. 输入截面名称。有关命名对象的更多信息，请参见["使用基本对话框组件，"第3.2.1节"](pt01ch03s02s01.md)。
3. 选择**Shell**作为截面**Category**，选择**General shell stiffness**作为截面**Type**，然后点击**Continue**。出现通用壳刚度截面编辑器。
4. 在**Stiffness**标签页上，在数据表格中输入壳刚度矩阵的对称半部分。第一行包含矩阵项![](../graphics/usi_eqn00372.gif)到![](../graphics/usi_eqn00373.gif)，第二行包含矩阵项![](../graphics/usi_eqn00374.gif)到![](../graphics/usi_eqn00375.gif)，依此类推；最后一行包含矩阵项![](../graphics/usi_eqn00376.gif)。请参见["输入表格数据，"第3.2.7节"](pt01ch03s02s07.md)，了解更多。

### 为通用壳刚度截面指定依赖性

**Dependencies**标签页允许您在壳截面上定义热应力。您还可以向壳刚度矩阵和热应力应用温度依赖性和场变量依赖性缩放因子。有关用于定义壳刚度依赖性的方程的详细信息，请参阅"使用通用壳截面定义截面行为"中的"直接指定等效截面属性以用于常规壳"，第29.6.6节Abaqus Analysis User's Guide。

**为通用壳刚度截面指定依赖性：**

1. 如果您定义的 热膨胀系数是温度的函数，请切换**Specify reference temperature**，并在提供的字段中输入参考温度，![](../graphics/usi_eqn00275.gif)。
2. 要在壳截面上定义热应力，请切换**Apply thermal stress**，并在提供的表格中输入应力分量：![](../graphics/usi_eqn00377.gif)（**Sigma11**）、![](../graphics/usi_eqn00378.gif)（**Sigma22**）、![](../graphics/usi_eqn00379.gif)（**Gamma12**）、![](../graphics/usi_eqn00380.gif)（**K11**）、![](../graphics/usi_eqn00381.gif)（**K22**）和![](../graphics/usi_eqn00382.gif)（**K12**）。
3. 要定义依赖于温度的缩放因子，请切换**Use temperature-dependent data**。**Scaling factors**数据表格中出现一个标为**Temp**的列。
4. 要定义依赖于场变量的缩放因子，请点击**Number of field variables**右侧的箭头以增加或减少场变量数量。**Field**变量列出现在**Scaling factors**数据表格中。
5. 在**Scaling factors**数据表格中，为刚度矩阵输入**Scaling modulus**(*Y*)以及热应力的**Thermal Expansion Coefficient**（![](../graphics/usi_eqn00007.gif)）。

### 为通用壳刚度截面指定高级属性

**为通用壳刚度截面指定高级属性：**

1. 指定**Section Poisson's ratio**以定义壳厚度行为。指定截面泊松比会导致壳厚度作为膜应变的函数而变化。
   - 切换**Use analysis default**以使用默认值0.5，这将强制元素膜应变的不可压缩行为。
   - 切换**Specify value**，并输入泊松比值。此值必须介于1.0和0.5之间。值为0.0将强制恒定壳厚度，负值将导致壳厚度响应拉伸膜应变而增加。
2. 切换**Density**，并输入壳单位表面积的质量。壳的质量由此密度确定。
3. 对于大多数壳截面，Abaqus将计算元素公式中所需的横向剪切刚度值。如有需要，从**Transverse Shear Stiffnesses**选项切换**Specify values**以在截面定义中包含非默认横向剪切刚度效应，并输入以下值：![](../graphics/usi_eqn00369.gif)，截面在第一方向上的剪切刚度；![](../graphics/usi_eqn00370.gif)，截面剪切刚度中的耦合项；以及![](../graphics/usi_eqn00371.gif)，截面在第二方向上的剪切刚度。如果![](../graphics/usi_eqn00369.gif)或![](../graphics/usi_eqn00371.gif)中的任何一个被省略或给定为零，则将使用非零值作为两者的值。
4. 点击**OK**保存更改并关闭通用壳刚度截面编辑器。
