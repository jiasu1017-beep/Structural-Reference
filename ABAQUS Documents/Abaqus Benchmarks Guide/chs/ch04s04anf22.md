# 4.4.7 FV32：悬臂锥形薄膜







**产品：** Abaqus/Standard  

### 测试单元

CPS4    CPS4I    CPS8    CPS8R    CPS6    CPS6M  

### 问题描述

![](../graphics/bmkfv32.png)

**模型：**

板厚度 = 0.05 m。

**材料：**

弹性模量 = 200 GPa，泊松比 = 0.3，密度 = 8000 kg/m³。

**边界条件：**

沿y轴![](../graphics/bmk_eqn01324.gif) = 0。在所有节点处![](../graphics/bmk_eqn01281.gif) = 0。

### 参考解

这是英国国家有限元方法与标准机构（NAFEMS）推荐的测试：NAFEMS出版物TNSB第3版"The Standard NAFEMS Benchmarks"（1990年10月）中的测试FV32。
