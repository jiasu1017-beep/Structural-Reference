# 79.2.4 编辑叠加绘图中的层





您可以更改叠加绘图中每个层的模型、绘图状态、绘图选项、动画控制或场输出变量选项。

**编辑叠加绘图中层的步骤：**

1. 使用["生成叠加绘图，"第79.2.1节](pt07ch79s02hlb01.md)中描述的过程创建叠加绘图。
2. 点击要修改的层在**当前**列旁边的列。
3. 要更改模型，请打开新的输出数据库。（有关更多信息，请参见["打开模型数据库或输出数据库，"第9.7.2节](pt02ch09s06hlb02.md)。）
4. 要更改绘图类型，请从**绘图**菜单或工具箱中的绘图工具中选择新的绘图类型。
5. 要更改与绘图状态无关或与绘图状态相关的自定义选项，请从以下对话框中选择新选项：
   - **视口注释选项**
   - **ODB显示选项**
   - **结果选项**
   - **公共绘图选项**
   - **叠加绘图选项**
   - **轮廓图选项**
   - **符号图选项**
   - **材料方向图选项**
   有关更多信息，请参见["视口注释选项概述，"第56.4节](pt05ch56hla01.md)、["选择结果选项，"第42.6节](pt05ch42s04.md)，或[第55章，"自定义绘图显示](pt05ch55.md)。在每个对话框中点击**确定**或**应用**可将更改应用到叠加绘图的当前层。
6. 您可以通过点击**叠加绘图层管理器**中**层选项**下的任何![](../graphics/ico_synchLayers.png)图标来同步可见层：**视图操作层**、**绘图状态层**或**绘图选项层**。
7. 根据需要更改任何动画选项。有关更多信息，请参见["生成和自定义基于对象的动画，"第49.2节](pt05ch49s02.md)和["控制动画播放，"第49.4节](pt05ch49s04.md)。
8. 根据需要更改场输出变量选项。有关更多信息，请参见["场输出变量选择概述，"第42.5.1节](pt05ch42s03hlb01.md)。
9. 您可以通过点击**场输出层**旁边的![](../graphics/ico_synchLayers.png)图标来同步可见层的场输出变量选项。
10. 对每个要修改的层重复步骤2到步骤8。
11. 当叠加绘图显示在视口中时，您可以点击**叠加绘图层管理器**中的**绘制单个**退出叠加绘图状态。叠加绘图从视口中消失，显示返回到之前的绘图状态；但是**叠加绘图层管理器**保持打开。点击**解散**关闭**叠加绘图层管理器**。**提示：**您也可以随时点击工具箱中的![](../graphics/ico_overlayState.png)工具在单个绘图和叠加绘图状态之间切换。
