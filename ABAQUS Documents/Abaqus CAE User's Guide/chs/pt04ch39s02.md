# 39.2 生成子结构




子结构定义的第一步是在分析中添加**Substructure generate**步骤。子结构生成步骤使您能够在模型数据库中创建子结构，并可选择指定子结构相关选项，例如将恢复矩阵、刚度矩阵、质量矩阵和载荷情况向量写入文件。这些选项将在本节后面描述。

单个分析可以包含多个子结构生成步骤，Abaqus/CAE为每个步骤创建相应的输出数据库文件。每个子结构生成步骤之前可以有多个预加载步骤。如果要为子结构生成指定保留特征模态，还必须在分析中包含频率提取步骤。

**子结构标识符**

您必须为创建的每个子结构指定唯一标识符。子结构标识符必须以字母Z开头，后跟一个不超过9999的数字。

**恢复选项**

您可以在使用级分析期间恢复子结构的场输出数据，但必须在子结构生成期间指定恢复区域。子结构恢复只能在恢复区域中包含的集合上执行。您可以指定在整个模型上执行恢复，或针对单个节点集或单元集执行恢复。在使用模型中执行子结构恢复时，Abaqus/CAE必须能够访问子结构的`.mdl`、`.prt`、`.stt`和`.sup`文件。有关这些文件类型的更多信息，请参见["定义子结构，" Abaqus Analysis User's Guide第10.1.2节](../usb/usb-link.md#usb-anl-asuperelementdef)。

**生成选项**

您可以控制子结构生成过程的多个方面，包括重力载荷向量的计算、频率相关材料特性的评估以及缩减质量矩阵、缩减结构阻尼矩阵和粘性阻尼矩阵的生成。

**保留特征模态**

您可以为耦合声-结构子结构指定保留特征模态。当您选择指定保留特征模态时，Abaqus/CAE使您能够通过模态范围或频率范围指定特征模态。

**阻尼**

您可以指定多个全局阻尼控制和子结构阻尼控制。对于全局阻尼，您可以选择将阻尼设置应用于声学或力学选项；对于子结构阻尼，您可以分别为粘性阻尼和结构阻尼指定单独的控制。
