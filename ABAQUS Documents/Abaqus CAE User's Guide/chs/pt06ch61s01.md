# 61.1 配置工具栏的可见性





使用 Customize 工具集，您可以从 Abaqus/CAE 显示中移除各个工具栏。隐藏不常用的工具栏可以减少屏幕上的混乱。您始终可以使用替代方法（如主菜单栏、模块专用工具箱或键盘快捷键）访问隐藏工具栏中的功能。

有三种方法可以更改工具栏的可见性：

**关闭浮动工具栏**

您可以通过单击工具栏标题栏中的"X"来隐藏浮动工具栏。关于浮动工具栏与停靠工具栏的说明，请参见["工具栏的组件，" 第2.2.3节](pt01ch02s02hlb03.md)。

要恢复浮动工具栏的显示，您必须使用以下述技术之一。

**使用主菜单栏**

要隐藏可见的工具栏，请从主菜单栏选择 ****View**![](../graphics/images/arrow.gif)**Toolbars**![](../graphics/images/arrow.gif)***Toolbar Name*****。使用相同的过程来恢复隐藏工具栏的显示。当前正在显示的工具栏在菜单列表中其名称旁边有一个复选标记。

**使用 **Customize** 对话框**

使用 **Customize** 对话框，您可以一次更改多个工具栏的可见性。从主菜单栏选择 ****Tools**![](../graphics/images/arrow.gif)**Customize**** 打开 **Customize** 对话框。在 **Toolbars** 选项卡页面上，切换关闭您想要隐藏的任何工具栏；切换打开您想要显示的任何工具栏。
