# 2.2.16 Johnson-Cook塑性







**产品：**Abaqus/Standard  Abaqus/Explicit  

### 测试单元

T2D2    T3D2    B21    B31    SAX1    C3D8R    CPE4R    CPS4R    CAX4R    S4R    S4RS    S4RSW    M3D4R    

### 测试特性

Johnson-Cook塑性模型。

### 问题描述

此验证问题测试在简单加载条件（单轴拉伸、单轴压缩和简单剪切）下运行的单元素模型。本例的目的是通过与具有等效塑性硬化的Mises塑性模型比较来测试Johnson-Cook塑性模型。图2.2.16-1显示了分析中使用的26个单元的原始形状。顶行的单元使用Johnson-Cook材料模型建模；底行的单元使用具有等效硬化曲线的Mises塑性模型建模。弹性材料特性为杨氏模量 = 124 GPa，泊松比 = 0.34。塑性硬化选择为

![](../graphics/ver_eqn00768.gif)

其中 ![](../graphics/ver_eqn00769.gif) 是屈服应力（单位为MPa），![](../graphics/ver_eqn00770.gif) 是等效塑性应变。材料特性为[Johnson和Cook (1985)]报告的OFHC铜的特性。![](../graphics/ver_eqn00769.gif) 与 ![](../graphics/ver_eqn00770.gif) 的关系图如图2.2.16-2所示。

### 结果与讨论

使用Johnson-Cook材料模型获得的结果与使用具有等效硬化曲线的Mises塑性模型获得的相应结果一致。图2.2.16-3显示了使用C3D8R单元在单轴拉伸下Johnson-Cook和Mises塑性模型获得的Mises应力的比较；图2.2.16-4显示了使用CPE4R单元在单轴压缩下Johnson-Cook和Mises塑性模型获得的Mises应力的比较；图2.2.16-5显示了使用CPE4R单元在简单剪切下Johnson-Cook和Mises塑性模型获得的Mises应力的比较。
