# 73.3.1 创建、编辑、重命名和删除集和曲面







要创建、编辑、重命名和删除集或曲面定义，请使用以下任一方式：
- 主菜单栏中****Tools**![](../graphics/images/arrow.gif)**Set****和****Tools**![](../graphics/images/arrow.gif)**Surface****子菜单下列出的**Create**、**Edit**、**Rename**和**Delete**项。**Edit**、**Rename**和**Delete**菜单项包含列出当前模型中所有集或曲面的子菜单。**Set**子菜单因您所在的模块而异；在Part和Property模块中，仅列出部件集。在Mesh模块中，根据上下文栏中选择的是部件还是装配体，可能列出部件集或装配体集。在所有其他模块中，仅列出装配体集。
- 集和曲面管理器。这些管理器包含与**Set**和**Surface**子菜单下列出的相同功能，但带有方便的浏览器，显示集和曲面及其类型（如几何或网格）。**Set Manager**在Part和Property模块中显示部件集，在Mesh模块中根据上下文栏中的当前对象选择显示部件集或装配体集，在所有其他模块中显示装配体集。**注意：**由于特征或网格修改而变为空的集和曲面仍将显示在列表中。有关更多信息，请参见["几何集和曲面的重新生成," 第73.2.4节](pt06ch73s02s04.md)。模型树在任何空集或曲面名称旁边显示一个感叹号(!)。您还可以将光标放在模型树中的集或曲面名称上，查看包含项目数的工具提示。要显示管理器，请从主菜单栏选择****Tools**![](../graphics/images/arrow.gif)**Set**![](../graphics/images/arrow.gif)**Manager****或****Tools**![](../graphics/images/arrow.gif)**Surface**![](../graphics/images/arrow.gif)**Manager****。

**警告：**当您删除集或曲面时，与该集或曲面关联的任何对象（如载荷或相互作用）将变为无效。
