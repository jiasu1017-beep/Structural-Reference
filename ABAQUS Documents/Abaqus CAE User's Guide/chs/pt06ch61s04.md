# 61.4 配置自定义工具栏中的图标





您可以更改分配给自定义工具栏中工具的默认图标。如果您创建的工具没有默认图标分配，您可以向该工具分配新图标。如果工具出现在多个自定义工具栏中，则该工具的图标分配适用于所有自定义工具栏。默认 Abaqus/CAE 工具栏和工具箱中的图标无法更改，不受自定义图标分配的影响。

要更改或添加图标分配，请从主菜单栏选择 ****Tools**![](../graphics/images/arrow.gif)**Customize****。在出现的对话框的 **Functions** 选项卡页面上，选择工具的功能，然后单击 **Select** 为该工具分配新图标。

分配图标时，您可以从 Abaqus/CAE 中当前使用的任何图标中进行选择。您也可以导入用户图标。用户图标必须为以下格式之一：bitmap (`.bmp`)、GIF (`.gif`)、PNG (`.png`) 或 XPM (`.xpm`)。图标图像应适当调整大小——Abaqus/CAE 中工具栏图标的标准尺寸为 24 x 24 像素。您无法缩放用户图标在 Abaqus/CAE 中的显示尺寸；如果增加图标尺寸比例因子（如["缩放图标尺寸，" 第68.3节](pt06ch68s03.md) 中所述），只有 Abaqus/CAE 图标会增加尺寸。

Abaqus/CAE 在您的主目录中创建一个名为 `abaqus_icons` 的目录来存储导入的用户图标图像。如果从此目录中删除图像文件，Abaqus/CAE 将恢复使用已删除图像文件的任何工具的默认图标分配。

同一个图标可以分配给多个工具。要区分使用相同图标的工具的功能，请将光标放在工具上片刻；会出现一个包含工具功能描述的小框或"工具提示"。
