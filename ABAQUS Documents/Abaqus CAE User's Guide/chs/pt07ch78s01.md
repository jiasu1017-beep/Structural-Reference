# 78.1 理解显示组





显示组是所选模型组件的集合，可以包含整个模型或零件实例、几何图形（单元、面或边）、基准几何（点、轴、平面或坐标系）、单元、节点、曲面、约束和输出数据库坐标系的组合。显示组允许您减少屏幕上的杂乱并集中关注模型中的感兴趣区域，访问复杂模型中的"隐藏"组件，以及减少刷新当前视口中显示所需的时间。例如，您可以使用显示组来显示接触曲面但抑制单元，或生成显示模型内部单元的云纹图，否则这些单元会被遮挡。您可以绘制、保存、编辑、复制、重命名和删除显示组。

在可视化模块中，您可以在同一视口中绘制多个显示组，并且可以独立自定义每个显示组的绘图选项。
