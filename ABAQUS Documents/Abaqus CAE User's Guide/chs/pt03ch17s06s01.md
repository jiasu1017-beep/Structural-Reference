# 17.6.1 验证网格







网格划分操作完成后，Abaqus/CAE 会突出显示网格中任何不良单元。Abaqus/CAE 还在网格模块中提供了一组工具，允许您验证网格质量并获取有关网格中节点和单元的信息。您可以使用这些工具来隔离网格质量较差的区域，并在需要细化网格时指导您。要验证网格质量，请从上下文栏中选择 **Object**，然后从主菜单栏中选择 ****Mesh**![](../graphics/images/arrow.gif)**Verify****。然后您可以选择要验证的零件、零件实例、几何区域或单元。Abaqus/CAE 允许您在检查网格是否将通过分析产品中的质量测试与检查网格是否通过单独的质量检查（如检查具有大长宽比的单元）之间进行选择。任何不符合指定标准的单元都会在视口中突出显示，您可以选择创建并保存包含突出显示单元的集合，或者，如果适用的话，创建并保存与这些单元相关的单元、面或边缘的集合。有关使用网格验证工具的详细信息，请参见"验证单元质量"第 17.19.1 节。

您可以使用 **Analysis checks** 来验证网格中的单元是否将通过包含在 Abaqus/Standard 或 Abaqus/Explicit 输入文件处理器中的单元质量检查。Abaqus/CAE 会突出显示任何未通过网格质量测试的单元，并在消息区域显示测试的单元数量以及错误和警告数量。输入文件处理器中的网格质量测试是广泛的且特定于每种单元类型。至少，网格质量测试会对看起来不适当扭曲的单元发出警告，如果扭曲严重则发出错误。Abaqus/CAE 不支持对梁、垫片或粘聚单元进行分析检查。

您可以使用 **Shape metrics** 来突出显示不满足以下任一选择标准的所选形状的单元：

**形状因子**

Abaqus/CAE 突出显示归一化形状因子小于指定值的单元。形状因子标准仅适用于三角形和四面体单元。形状因子范围为 0 到 1，其中 1 表示最佳单元形状，0 表示退化单元。
- 对于三角形单元，归一化形状因子定义为 ![](../graphics/usi_eqn00645.gif) *最佳单元面积* 是与单元具有相同外接圆的等边三角形的面积。（外接圆是通过三角形三个顶点的圆的半径。）
- 对于四面体单元，归一化形状因子定义为 ![](../graphics/usi_eqn00646.gif) *最佳单元体积* 是与单元具有相同外接球的等面四面体的体积。（外接球是通过四面体四个顶点的球的半径。）

**小面角**

Abaqus/CAE 突出显示包含两条边缘以小于指定角度的角度相交的面的单元。

**大面角**

Abaqus/CAE 突出显示包含两条边缘以大于指定角度的角度相交的面的单元。

**长宽比**

Abaqus/CAE 突出显示长宽比大于指定值的单元。长宽比是单元最长边缘与最短边缘之比。

您可以使用 **Size metrics** 来突出显示不满足以下任一选择标准的单元：

**几何偏移因子**

Abaqus/CAE 突出显示边缘几何偏移因子大于指定值的单元。几何偏移因子是衡量单元边缘偏离原始几何程度的指标，Abaqus/CAE 通过将单元边缘与其父几何面或边缘之间的最大间隙除以单元边缘的长度来计算此值。默认情况下，Abaqus/CAE 突出显示几何偏移因子大于 0.2 的单元。

Abaqus/CAE 仅对原生网格中的单元计算几何偏移因子。如果您选择不包含几何形状的零件，Abaqus/CAE 会在 **Verify Mesh** 对话框中禁用此选项。如果您的选择同时包含原生和孤立单元，Abaqus/CAE 仅考虑用于几何偏移因子计算的原生单元。

**短边缘**

Abaqus/CAE 突出显示边缘短于指定值的单元。

**长边缘**

Abaqus/CAE 突出显示边缘长于指定值的单元。

**稳定时间增量**

Abaqus/CAE 突出显示计算稳定时间增量小于指定值的单元。稳定时间增量计算需要适当的材料定义和截面分配，仅对 Abaqus/Explicit 分析有意义。

Abaqus/CAE 中的稳定时间增量计算是 Abaqus/Explicit 对单元进行公式化计算的初始稳定时间增量计算的近似值。它不考虑以下任何条件：
- 质量缩放
- 点质量
- 转动惯量
- 非结构质量
- 增强（钢筋）

Abaqus/CAE 中稳定时间增量计算支持的材料行为包括弹性、超弹性、超泡沫（无用户定义的测试数据）和声学介质。不支持具有多种材料的复合截面。 有关更多信息，请参见《Abaqus Analysis User's Guide》第 6.3.3 节"显式动态分析"中的"稳定性"。

此外，对于形状和尺寸指标，Abaqus/CAE 在消息区域为每个所选零件、零件实例或区域显示以下信息：
- 零件或零件实例的名称。
- 所选区域中所选形状的单元总数。
- 突出显示的单元数量及其占正在验证的单元的百分比。
- 选择标准的平均值。对于几何偏移因子，Abaqus/CAE 仅通过考虑沿曲线或面的单元来计算平均值；体积中心的实体单元不包含在此值中。
- 选择标准的"最差"值——如果未超过标准，则为最接近标准的值；如果超过标准，则为最超过标准的值。
