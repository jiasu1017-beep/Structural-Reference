# 55.10 控制模型实体的显示





**ODB Display Options**对话框中的**Entity Display**选项允许您控制模型中以下实体表示符号的显示：
- 分析过程中应用的边界条件，
- 模型中使用的连接器（可以单独控制端点的高亮显示、与每个连接器关联的局部方向轴的显示以及连接器类型标签的显示），
- 写入当前输出数据库或在当前会话期间在可视化模块中创建的坐标系，以及
- 点元素，包括参考点、质量和惯性元素、弹簧和阻尼器、点焊和分布耦合（DCOUP*）元素、示踪粒子、连续粒子元素和离散粒子元素。

模型实体显示与绘图状态无关。Abaqus/CAE会为输出数据库的当前[步和帧](pt05ch42s01hlb01.md)显示模型实体符号。您只能通过更改符号大小来在可视化模块中自定义模型实体符号的外观。

有关边界条件符号的信息，请参阅["理解代表规定条件的符号，" 第16.5节](pt03ch16s05.md)；有关连接器符号的信息，请参阅["理解代表相互作用、约束和连接器的符号，" 第15.10节](pt03ch15s10.md)。有关在装配相关模块中控制这些实体显示的信息，请参阅["控制属性的显示，" 第76.15节](pt07ch76hla14.md)。

**提示：**您也可以使用显示组来控制连接器的显示；每个连接器都列为一个元素集。有关显示组的更多信息，请参阅["创建或编辑显示组，" 第78.2.1节](pt07ch78s02hlb01.md)。

默认情况下只显示会话坐标系和参考点。模型实体的显示会显著影响性能。为了最大化Abaqus/CAE的性能，请慎重选择要显示的实体。

**要控制模型实体的显示：**

1. 从主菜单栏，选择****View**![](../graphics/images/arrow.gif)**ODB Display Options****。在出现的对话框中单击**Entity Display**选项卡。实体显示选项变得可用。
2. 切换您想要显示的实体。只有您选择的实体（如果它们已为当前模型定义）才会出现在视口中。
3. 如果需要，将**Symbol size**滑块向左或向右拖动以修改视口中符号的大小。
4. 单击**Apply**来应用您的更改。当前视口中的绘图会更改以反映您的规范。默认情况下，您的更改会在会话期间保存，并会影响当前视口和从当前视口创建的任何新视口中的所有后续绘图。如果您希望保留所做的更改以供后续会话使用，请将它们保存到文件中。有关更多信息，请参阅["保存自定义设置以供后续会话使用，" 第55.1.1节](pt05ch55s01s01.md)。
