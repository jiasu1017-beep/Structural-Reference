# 19.11.4 Viewing the results of your co-execution







Once your co-execution is completed, Abaqus/CAE stores the results of each analysis job in an output database. You can use the Visualization module to view the combined results graphically.

**To view the results of a co-execution:**

1. From the **Co-execution Manager**, select the co-execution name and click **Results**. The Visualization module starts and presents you with an overlay plot of both models.
2. Use the Visualization module to create and customize different plots of your results. For more information on using overlay plots, see [Chapter 79, "Overlaying multiple plots](pt07ch79.md)."
