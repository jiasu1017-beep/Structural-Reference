# 17.6.6 使用虚拟拓扑改进网格





在某些情况下，零件或独立零件实例包含非常小的面和边等细节。"虚拟拓扑"工具集允许您通过将小面与相邻面合并或将小边与相邻边合并来移除这些小细节。您也可以忽略选定的边和顶点，这具有与合并面和边相同的效果。引入虚拟拓扑是创建干净、良好网格的便捷方法。"虚拟拓扑"工具集仅在网格模块中可用。

但是，向零件实例添加虚拟拓扑可能会限制您随后对其进行网格划分的能力。例如，您无法使用以下技术对包含虚拟拓扑的区域进行网格划分：
- 使用中轴算法的四边形或四边形主导单元进行二维自由网格划分。
- 使用中轴算法的三维扫掠网格划分。
- 如果要网格化的区域不是由四个角限定的，则无法进行二维结构化网格划分。
- 如果要网格化的区域不是由六面限定的，则无法进行三维结构化网格划分。

有关更多信息，请参见[第75章，"虚拟拓扑"工具集](pt06ch75.md)。

此外，您只能将虚拟拓扑应用于独立实例。如果您需要将虚拟拓扑应用于从属实例，可以创建原始零件的副本，然后创建该副本的独立实例。然后您可以用新的独立实例替换从属实例，并将虚拟拓扑应用于独立实例。有关更多信息，请参见["从属实例和独立实例有什么区别？"第13.3.2节](pt03ch13s03s02.md)。
