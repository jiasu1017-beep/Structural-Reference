### 概述

耦合约束（Coupling Constraints）用于将一个参考节点与一组从节点耦合。与运动耦合不同，分布耦合允许从节点有相对变形，同时通过权因子分配力。

### 耦合约束类型

| 类型 | 描述 | 用途 |
| --- | --- | --- |
| DISTRIBUTING | 分布耦合 | 一般用途 |
| KINEMATIC | 运动耦合 | 刚体运动 |

### 分布耦合（ DISTRIBUTING ）

分布耦合约束将参考节点的力和力矩分布到从节点上：

| **输入文件用法：** | ```
*COUPLING, CONSTRAINT=DISTRIBUTING, REF NODE=ref_node
slave_node_1
slave_node_2
...
``` |
| --- | --- |

### 分布耦合权因子

权因子定义了每个从节点分配载荷的比例：

| **输入文件用法：** | ```
*COUPLING, CONSTRAINT=DISTRIBUTING, REF NODE=ref_node
slave_node_1, weight_1
slave_node_2, weight_2
...
``` |
| --- | --- |

### 运动耦合（ KINEMATIC ）

运动耦合约束强制从节点与参考节点保持相同运动：

| **输入文件用法：** | ```
*COUPLING, CONSTRAINT=kinematic, REF NODE=ref_node
slave_node_1
slave_node_2
...
``` |
| --- | --- |

### 耦合约束参数

| 参数 | 描述 |
| --- | --- |
| REF NODE | 参考节点编号 |
| CONSTRAINT | 耦合类型 |
| SURFACE | 从节点所在表面 |

### 表面耦合

耦合约束可以施加于表面上的所有节点：

| **输入文件用法：** | ```
*COUPLING, REF NODE=ref_node, SURFACE=slave_surface
``` |
| --- | --- |

### 应用场景

- 加载板建模
- 连接件简化
- 点载荷分布
- 力矩施加
