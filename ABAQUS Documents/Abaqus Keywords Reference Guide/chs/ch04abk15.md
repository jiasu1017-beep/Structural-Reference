# *DESIGN PARAMETER





### *DESIGN PARAMETER指定要计算灵敏度的设计参数。

此选项用于为设计灵敏度分析指定设计参数。将根据这些设计参数计算在 [*DESIGN RESPONSE](ch04abk17.md) 选项下指定的响应的灵敏度。设计参数必须从 [*PARAMETER](ch16abk01.md) 选项上定义的现有参数集中选择。

**产品：**Abaqus/Design  

**类型：**模型数据  

**级别：**模型
