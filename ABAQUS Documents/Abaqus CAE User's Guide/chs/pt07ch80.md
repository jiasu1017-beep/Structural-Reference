# 80 切割模型





视图切割允许您通过模型切割平面或可变形截面，以查看模型内部。例如，[图80-1](pt07ch80s01.md#uss-cutplane-gearbox-example)显示了平面视图切割如何用于切割可视化模块中变速箱模型轮廓图的截面。

**图80-1** 变速箱模型轮廓图的平面视图切割。从左到右：切割下方、切割处和切割上方的模型；底部：整个模型。

![](../graphics/uss-cutplane-gearbox-example.png)

您可以在建模和后处理活动中定义和使用视图切割，尽管视图切割功能的某些方面仅在其中一个活动中可用。本节描述视图切割功能；除非另有说明，否则所述视图切割功能在建模和后处理中均可用。

**视图切割形状**

您可以基于平面创建视图切割。在可视化模块中，您还可以基于以下形状创建视图切割：
- 圆柱体，
- 球体，或
- 对应于场变量或模型属性常量的等值面。

视图切割创建形状的类型如图[图80-2](pt07ch80s01.md#uss-viewcut-shapes)所示。

**图80-2** 基于平面、圆柱、球面和等值面形状的视图切割。

![](../graphics/uss-viewcut-shapes.png)

对于等值面切割，结果值的计算方式与线型和带状类型轮廓的["理解轮廓值的计算方式，"第44.1.1节](pt05ch44s01s01.md)中所述相同。默认情况下，Abaqus/CAE对等值面切割应用100%的平均阈值，以确保切割位置结果的连续显示。您可以通过关闭**覆盖主变量平均**来应用小于100%的平均阈值来创建等值面切割。然后，Abaqus/CAE应用**结果选项**对话框中**平均**选项指定的平均阈值；有关更多信息，请参见["控制结果平均，"第42.6.6节](pt05ch42s04hlb02.md)。默认情况下，会沿X-、Y-和Z平面创建切割。

**注意：**等值面视图切割提供与等值面类型轮廓图略有不同的功能和行为。等值面视图切割始终反映创建视图切割时活动的主变量的结果；您无法更改等值面视图切割显示结果的变量。相比之下，等值面类型轮廓图始终显示当前为会话选择的主变量的数据。因为每个等值面视图切割绑定到单个变量，所以当您显示基于不同输出变量的多个等值面视图切割时，可以调查两个等值面视图切割相交的位置。

**显示切割截面**

要显示模型的切割截面，您需要激活切割并选择是在切割处、切割上方和/或切割下方显示模型（如图[图80-1](pt07ch80s01.md#uss-cutplane-gearbox-example)所示）。切割本身不可见。对于平面切割，切割下方定义为位于平面法线负侧的部分（相对于平面方向）。对于圆柱和球面切割，切割下方定义为位于切割形状半径以内的部分。对于等值面切割，切割下方定义为位于小于指定等值面的等值部分。默认情况下，Abaqus/CAE显示切割处和切割下方的模型。在可视化模块中，您可以同时在切割上方和切割下方显示模型；在所有其他模块中，这些显示选项一次只能使用一个。

**绘图选项**

您可以为切割下方、切割处和切割上方的模型部分选择不同的绘图选项；例如，在[图80-2](pt07ch80s01.md#uss-viewcut-shapes)中，模型的部分以激活的透明度显示，而其他部分以无透明度显示。

**显示多个视图切割（仅可视化模块）**

在其他模块中，一次只能有一个切割处于活动状态（用于在当前视口中显示模型）；但在可视化模块中，您可以同时显示多个视图切割。此外，在可视化模块中，您可以在未变形、变形、轮廓（仅纹理映射）、符号或材料方向图上激活视图切割。对于符号和材料方向图，Abaqus/CAE在包含在视图切割中的每个单元的所有积分点显示符号和材料方向 triad，即使单元被部分切割。

**视图切割的动画（仅结果数据）**

可以在可视化模块中为输出数据库数据对带有活动视图切割的绘图进行动画处理；视图切割将针对每个动画帧更新。无法在镶嵌轮廓图、扫描、收缩或拉伸绘图或高度细化绘图（中等、精细或超精细）上激活视图切割。

**结果缓存**

默认情况下，Abaqus/CAE缓存用于生成分割模型图像的结果值以提高性能。但是，如果需要，您可以禁用切割字段的结果缓存以减少内存使用；请参见["控制结果缓存，"第42.6.10节](pt05ch42s04hlb06.md)了解更多信息。

**跟随变形**

对于变形模型的绘图，您可以选择让切割跟随变形；即切割表面将相对于参考框架计算，切割变形将与模型的变形相匹配。

**重新定位视图切割**

您可以在模型上重新定位切割：平面切割可以旋转或平移，而其他切割形状只能平移。

**视图切割管理器**

您可以在**视图切割管理器**中选择切割（活动或非活动）来定位、编辑、复制、重命名或删除它。在管理器中选择切割与激活切割不同。切割可以同时处于选定和活动、取消选定和活动或选定和非活动状态；并且您可以显示或隐藏当前显示视图切割的自由体切割。[图80-3](pt07ch80s01.md#uss-cutplane-manager)显示了可视化模块中视图切割管理器中显示的信息示例；在其他模块中，此对话框中不包含自由体显示选项。

**图80-3** 可视化模块中的**视图切割管理器**对话框。

![](../graphics/uss-cutplane-manager-nls.png)

**视图切割上的合力和力矩（仅结果数据）**

对于可视化模块中输出数据库数据的视图切割，您还可以显示合力与力矩，并且可以基于整个模型或基于当前显示组来计算这些值。[图80-4](pt07ch80s01.md#uss-cutplane-freebody)所示的示例包括为起落架装配的视图切割显示的合力和力矩。

**图80-4** 显示合力和力矩的起落架装配平面视图切割。

![](../graphics/uss-cutplane-freebody-image.png)

您只能为实体几何、壳截面或梁截面显示视图切割上的合力和力矩；对于壳截面和梁截面，输出数据库必须包含截面力(SF)和截面矩(SM)输出，以便显示合力和力矩。Abaqus/CAE不支持为轴对称模型显示视图切割上合力和力矩。您可以显示会话中任何显示视图切割的合力和力矩。当您重新定位视图切割或对模型进行动画处理时，Abaqus/CAE会更新合力与力矩向量和求和点。

默认情况下，Abaqus/CAE仅在视图切割上显示向量，显示跨视图切割的合力和力矩。但是，您也可以显示一系列向量，在模型中以规则间隔显示合力和力矩数据。这些合力和力矩向量系列可以贯穿整个模型，也可以在用户指定区域内。

**基于XFEM的视图切割组件（仅结果数据）**

当您打开包含由扩展有限元方法(XFEM)计算的裂纹的输出数据库时，Abaqus/CAE自动创建并显示视图切割。视图切割在与有符号距离函数值为零的等值面对应的位置显示模型，这与XFEM裂纹的表面对应。激活XFEM裂纹视图切割时，不显示边界条件。

**视图切割管理器**中的工具对XFEM裂纹的操作如下：
- 切割下方![](../graphics/visG_CutBelow.png)复选框显示整个模型，XFEM裂纹除外。
- 切割处![](../graphics/visG_CutOn.png)复选框显示有符号距离函数值为零的等值面，这与XFEM裂纹的表面对应。
- 切割上方![](../graphics/visG_CutAbove.png)复选框不用于XFEM裂纹。

**基于优化的视图切割组件（仅结果数据）**

当您打开由优化过程创建的输出数据库时（拓扑优化和形状优化），Abaqus/CAE自动创建并显示视图切割。默认情况下，视图切割在与材料属性值为零的等值面对应的位置显示模型，这对应于密度和刚度接近零的单元，因此对模型强度贡献微小。激活优化视图切割时，不显示边界条件。

**视图切割管理器**中的工具对优化的操作如下：
- 切割下方![](../graphics/visG_CutBelow.png)复选框显示属性小于滑块值的材料。默认情况下，这是对模型强度没有贡献的材料。
- 切割处![](../graphics/visG_CutOn.png)复选框显示属性等于滑块值的等值面。
- 切割上方![](../graphics/visG_CutAbove.png)复选框显示属性大于滑块值的材料。默认情况下，这是继续对模型强度有贡献的材料。
