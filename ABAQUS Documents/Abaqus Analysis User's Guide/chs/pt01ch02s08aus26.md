### 概述

分布：
- 是定义在 Abaqus 模型中单元或节点上的空间变化场；
- 可用于在单元基础上定义壳厚度；
- 可用于在单元基础上定义壳刚度；
- 可用于在单元基础上在实体连续体和壳单元上定义局部坐标系；
- 可用于在复合壳单元层上定义方向角；
- 可用于为连接器单元定义方向角；
- 可用于在传统复合壳单元层上定义厚度；
- 可用于指定初始接触间隙；
- 可用于在 Abaqus/CFD 分析中指定随穿过表面的流体总体积而变化的压力；以及
- 在 Abaqus/Standard 分析中可用于为实体连续体单元定义质量密度、线性弹性材料行为和热膨胀；壳偏移；复合实体连续体单元层上的方向角；膜单元上的局部坐标系；以及在单元基础上定义膜厚度。

### 分布

分布是幅值定义的类比（请参阅 ["幅值曲线，" 第 34.1.2 节"](pt07ch34s01aus115.md)）。幅值定义用于提供载荷、位移和其他规定变量的任意时间变化。分布用于指定所选元素属性、材料属性、局部坐标系以及初始接触间隙的空间变化。

分布的两个主要组成部分是其*位置*和*场数据*。位置标识分布的定义位置（在单元上或节点上）。场数据是为分布中每个单元或节点定义的指定数量的浮点值。

要定义分布，您必须为其分配一个唯一名称。您还必须通过引用分布表来指定分布中每个数据值的数量和物理维度。

### 指定分布的位置

您可以在单元或节点上定义分布。目前，仅支持在节点上的分布用于定义初始接触间隙，如 ["在 Abaqus/Explicit 中控制通用接触的初始接触状态，" 第 36.4.4 节"](pt09ch36s04aus158.md) 中所述。对于与 Abaqus/CFD 中流体边界定义一起使用的分布，您指定不需要位置。所有其他分布应用都需要在单元上定义的分布。

对于给定单元或节点所属的分布数量没有限制。单元和节点不能在同一分布定义中组合。

#### 在单元上定义分布

在单元上定义分布需要您为分布定义中包含的每个单元或单元集指定场数据。所有单元上的分布都需要定义默认数据。默认数据用于未在分布中专门分配值的所有单元。

| **输入文件用法：** | ``` [*DISTRIBUTION](../key/key-link.md#usb-kws-mdistribution), LOCATION=ELEMENT *blank space*, *field data* *element set or element number*, *field data* ``` |
| --- | --- |
|  | 默认数据通过在分布定义的第一数据行上使用空白而不是单元号或单元集作为第一数据项来定义。一个分布只能定义一组默认数据。如果您仅指定默认数据，则所有引用该分布的单元都使用默认值。如果在给定分布定义中多次指定了一个单元，则使用最后给出的规范。 |

#### 在节点上定义分布

在节点上定义分布需要您为分布定义中包含的每个节点或节点集指定场数据。
