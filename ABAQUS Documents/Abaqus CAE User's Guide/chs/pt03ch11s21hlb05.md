# 11.21.5 从壳创建实体特征





从主菜单栏选择**形状**![](../graphics/images/arrow.gif)**实体**![](../graphics/images/arrow.gif)**从壳**，通过选择将形成闭合零件的面，从三维壳零件创建实体特征。Abaqus/CAE添加实体材料，将由选定面定义的区域从壳变为实体。

**从壳创建实体特征：**

1. 从主菜单栏，选择**形状**![](../graphics/images/arrow.gif)**实体**![](../graphics/images/arrow.gif)**从壳**。
2. 从壳中选择要转换为实体的面，然后单击鼠标按钮2表示完成选择面。
3. 如果选择了单个面，Abaqus/CAE高亮显示该面并显示一个箭头，指示将添加材料以创建实体的方向。如果需要，单击**翻转**以反转箭头的方向。
4. 单击鼠标按钮2确认箭头的方向。Abaqus/CAE沿指示的方向填充壳并创建实体区域。
5. 单击**完成**创建实体零件。
