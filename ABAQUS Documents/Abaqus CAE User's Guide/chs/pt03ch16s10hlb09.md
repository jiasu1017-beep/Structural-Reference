# 16.10.8 定义基础运动边界条件





您可以创建基础运动边界条件来指定模态动态步骤、稳态动态（基于模态或基于子空间）步骤或随机响应步骤中节点的运动。基础运动可以是加速度、速度或位移。

有关基础运动的更多信息，请参阅以下章节中的边界条件讨论：
- 《Abaqus Analysis User's Guide》第6.3.7节"瞬态模态动态分析"
- 《Abaqus Analysis User's Guide》第6.3.8节"基于模态的稳态动态分析"
- 《Abaqus Analysis User's Guide》第6.3.9节"基于子空间的稳态动态分析"
- 《Abaqus Analysis User's Guide》第6.3.11节"随机响应分析"

**要创建或编辑加速度、速度或位移的基础运动边界条件：**

1. 使用以下方法之一显示边界条件编辑器：- 要创建新的位移基础运动边界条件，请按照"创建边界条件"中的步骤操作（第16.8.2节）（**类别**：`Mechanical`；**所选步骤的类型**：`Acceleration base motion`或`Velocity base motion`或`Displacement base motion`）。- 要使用菜单或管理器编辑现有基础运动边界条件，请参见"编辑依赖于步骤的对象"（第3.4.13节）。
2. 在**Basic**选项卡中，选择**Degree-of-freedom**来指定定义基础运动的方向。此方向始终是全局方向：**U1**、**U2**、**U3**、**UR1**、**UR2**或**UR3**。
3. 对于模态动态步骤或稳态动态（基于模态或基于子空间）步骤，请指定以下选项：1. 如果要将基础运动应用于次级基础，请切换**Secondary base**并选择先前在特征频率提取步骤中定义的次级基础边界条件的名称。2. 选择定义基础运动的时间历史（模态动态步骤）或频谱（稳态动态步骤）的**Amplitude**曲线。3. 如果需要，为**Amplitude scale factor**输入新值。默认值为1.0。4. 基础运动始终在全局方向上指定，无论节点转换的使用如何。您指定定义基础运动的全局方向（**Degree-of-freedom**）。如果围绕不是坐标原点的原点指定了旋转，则必须通过单击![](../graphics/ico_selectBlue.png)并在视口中选择点或输入坐标来指定**Center of rotation**。5. 对于稳态动态步骤，如果指定的幅值曲线定义了基础运动的虚部（而非实部、同相部分），您可以切换**Define imaginary (out-of-phase) portion given by amplitude**。
4. 对于随机响应步骤，单击**Correlation**选项卡并切换**Specify correlation**来定义作为随机加载定义一部分的互相关。有关相关性选项的完整信息，请参见《Abaqus Analysis User's Guide》第6.3.11节"随机响应分析"中的"定义相关性"。要定义相关性，请填写表格行中的单元格，如下所示：1. 单击**Approach**列，然后选择以下方法之一：- 如果相关矩阵中的所有项都应该被包含，请选择**Correlated**。- 如果只应使用对角项，请选择**Uncorrelated**。- 选择**User**表示将调用用户子程序[`UCORR`](../sub/sub-link.md#sub-xsl-ucorr)来获取相关矩阵的缩放因子。2. 单击**PSD**列单元格，然后选择先前定义的PSD定义幅值。此幅值表示随机噪声源的功率谱密度函数。3. 在**Real**和**Imaginary**列中，输入缩放因子的实部和虚部。这些缩放因子指定是否在空间相关矩阵中包含实部和虚部。您可以在表格中添加行以创建额外的相关性来定义随机加载。
5. 单击**OK**保存数据并退出编辑器。
