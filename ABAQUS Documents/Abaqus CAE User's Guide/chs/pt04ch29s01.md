# 29.1 关于紧固件





您使用紧固件对两个或多个面之间的点对点连接进行建模，如点焊、螺栓或铆钉。您可以使用Attachment工具集创建有助于定义紧固件的连接；有关更多信息，请参见["理解连接点和连接线"，第59.1节](pt06ch59s01.md)。您可以创建基于点的紧固件、离散紧固件或组装紧固件。以下主题提供不同紧固件类型的概述：
- ["关于基于点的紧固件"，第29.1.1节](pt04ch29s01s01.md)
- ["关于离散紧固件"，第29.1.2节](pt04ch29s01s02.md)
- ["关于组装紧固件"，第29.1.3节](pt04ch29s01s03.md)

基于点和离散紧固件都可以使用连接器或梁MPC进行建模。如果您定义的连接器使用基本、组装或复杂连接类型，则可以通过连接器输出变量从基于点或离散紧固件请求输出。但是，如果您使用梁MPC，则无法从基于点或离散紧固件获取输出。
