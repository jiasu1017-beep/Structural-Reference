### 概述

钢筋：
- 用于在膜、壳和曲面单元中定义单向增强层（此类层被视为平滑层，其恒定厚度等于每个钢筋横截面积除以钢筋间距）；
- 可用于通过将增强曲面或膜单元嵌入"宿主"实体单元中来为实体增加增强层，如["嵌入单元，"第35.4.1节"](pt08ch35s04aus136.md)中所述；
- 可用于为模型增加额外的刚度、体积和质量；
- 可用于在Abaqus/Standard中的梁单元中添加离散轴向增强；
- 可用于耦合温度-位移分析，但不影响热传导率和比热；
- 可用于耦合热-电-结构分析，但不影响电导率、热传导率和比热；
- 不能用于热传递或质量扩散分析；和
- 具有与底层或宿主单元不同的材料属性。
- 不包括底层单元的质量或体积。

### 定义钢筋层

您可以在膜、壳或曲面单元中指定一层或多层增强。对于每一层，您可以指定钢筋属性，包括钢筋层名称；每个钢筋的横截面积；膜、壳或曲面单元平面中的钢筋间距；钢筋在厚度方向上的位置（仅适用于壳单元），相对于壳中面测量（正方向为壳法线正方向）；钢筋材料名称；相对于局部1方向测量的初始角度方向（以度为单位）；以及将用于测量钢筋角度输出的等参方向。

您可以通过在宿主连续体单元集中嵌入具有如上定义的钢筋层的曲面或膜单元集，来在实体（连续体）单元中建模钢筋层。

| **输入文件用法：** | 使用以下选项在膜单元中定义一层或多层钢筋： |

|  | ``` [*MEMBRANE SECTION*](../key/key-link.md#usb-kws-mmembranesection), ELSET=*memb_set_name* [*REBAR LAYER*](../key/key-link.md#usb-kws-mrebarlayer) ``` 使用以下选项在壳单元中定义一层或多层钢筋： ``` [*SHELL SECTION*](../key/key-link.md#usb-kws-mshellsection), ELSET=*shell_set_name* [*REBAR LAYER*](../key/key-link.md#usb-kws-mrebarlayer) ``` 使用以下选项在曲面单元中定义一层或多层钢筋： ``` [*SURFACE SECTION*](../key/key-link.md#usb-kws-msurfacesection), ELSET=*surf_set_name* [*REBAR LAYER*](../key/key-link.md#usb-kws-mrebarlayer) ``` 使用以下选项在实体（连续体）单元中建模钢筋层： ``` [*EMBEDDED ELEMENT*](../key/key-link.md#usb-kws-membeddedelement), HOST ELSET=*solid_set_name* *memb_set_name* or *surf_set_name* ``` |

#### 指定钢筋几何

钢筋几何始终相对于局部坐标系定义。定义适当的局部系统将在下一节中描述。钢筋几何可以是恒定的，作为柱坐标系统中径向位置的函数变化，或根据轮胎"提升"方程变化。在每种情况下，您必须指定间距*s*和面积*A*，它们用于确定的等效钢筋层的厚度，以及钢筋相对于此局部系统的角方向。

此外，对于壳单元，您必须指定钢筋在壳厚度方向上相对于壳中面的位置（正方向为壳法线正方向）。如果壳的厚度由节点厚度定义（["节点厚度，"第2.1.3节"](pt01ch02s01aus07.md)），则此距离将按节点厚度定义的厚度与截面定义的厚度的比率进行缩放。如果壳的厚度用分布定义（["分布定义，"第2.8.1节"](pt01ch02s08aus26.md)），则此距离按分布定义的单元厚度与默认厚度的比率进行缩放。

##### 定义恒定间距的钢筋

您可以指定几何在局部钢筋坐标系中为恒定。在这种情况下，间距*s*被指定为长度度量。

| **输入文件用法：** | ``` [*REBAR LAYER*](../key/key-link.md#usb-kws-mrebarlayer), GEOMETRY=CONSTANT ``` |
| --- | --- |

| **Abaqus/CAE用法：** | 属性模块：膜、壳或曲面截面编辑器：**钢筋层**：**钢筋几何：恒定** |
| --- | --- |

##### 将钢筋间距定义为径向位置的函数

您可以指定间距*s*，以角度间距（度）为单位表示。

角度间距值也可用于非径向钢筋以及具有相对于经向平面非零方向角度的钢筋。在这些情况下，钢筋的方向角度不会改变。角度间距选项仅用于通过将角度间距乘以钢筋上关切点距轴对称轴的径向距离来计算以长度为单位的钢筋间距。如果钢筋与三维单元关联，则必须为钢筋定义局部柱坐标系。

| **输入文件用法：** | ``` [*REBAR LAYER*](../key/key-link.md#usb-kws-mrebarlayer), GEOMETRY=ANGULAR ``` |
| --- | --- |

| **Abaqus/CAE用法：** | 属性模块：膜、壳或曲面截面编辑器：**钢筋层**：**钢筋几何：角度** |
| --- | --- |

##### 使用轮胎"提升"方程定义钢筋

结构轮胎分析通常使用硫化轮胎几何作为有限元模型的参考配置。然而，帘线几何更方便地相对于"生"或未硫化轮胎配置指定。轮胎提升方程从未硫化几何映射到硫化几何（参见[图2.2.3-2](pt01ch02s02aus13.md#erebarlayer-liftequation)）。

**图2.2.3-2** 未硫化和硫化轮胎钢筋几何之间的映射。

![](../graphics/erebarlayer-liftequation-nls.png)

您可以相对于未硫化配置指定钢筋帘线的间距和方向，并让Abaqus将这些属性映射到硫化轮胎的参考配置。使用柱坐标系，从硫化轮胎中的间距*s*和角方向获得。

如果钢筋与三维单元关联，则必须为钢筋定义局部柱坐标系。

| **输入文件用法：** | ``` [*REBAR LAYER*](../key/key-link.md#usb-kws-mrebarlayer), GEOMETRY=LIFT EQUATION ``` |

| **Abaqus/CAE用法：** | 属性模块：膜、壳或曲面截面编辑器：**钢筋层**：**钢筋几何：基于提升方程** |

#### 局部钢筋方向系统

钢筋几何（如钢筋方向和间距）是相对于局部方向系统定义的。此局部钢筋方向系统与底层单元分配使用的局部方向系统完全独立。

钢筋角度始终相对于局部1方向定义。

对于用角度间距或轮胎提升方程定义的钢筋，相对于柱方向系统指定。对于轴对称分析，全局坐标系用作柱系统。对于三维分析，您必须提供用户定义的柱方向定义。

##### 三维单元的局部方向系统

您可以通过引用用户定义的局部坐标系来定义局部系统。详见["方向，"第2.2.5节"](pt01ch02s02aus15.md)中关于如何从用户定义的方向计算壳、膜和曲面单元中钢筋定义的局部坐标系的描述。

如果您不指定用户定义的方向，局部1方向将基于默认投影局部坐标系。

##### 轴对称单元的局部方向系统

轴对称膜单元或轴对称曲面单元中的钢筋必须位于单元参考曲面，而轴对称壳中的钢筋可以位于壳参考曲面或可以偏离中面。轴对称膜、壳和曲面单元中的钢筋可以定义为相对于*r*–*z*平面具有任何角度方向。

您不能为轴对称膜、壳和曲面单元中的钢筋层指定用户定义的方向。相反，您可以在钢筋层定义中指定钢筋层相对于*r*–*z*平面的角度方向（以度为单位）；此方向相对于膜、壳或曲面单元的法线正方向正测量。

#### 大位移考虑

在大变形分析中，当钢筋增强单元变形时，初始定义的几何属性和钢筋层方向可能因有限应变效应而改变。钢筋层的变形从底层壳、膜或曲面单元的变形梯度确定。钢筋随实际变形旋转，而不是随底层单元材料点的平均刚体旋转。

例如，考虑在纯剪切变形下使用一阶单元建模的板，其中钢筋最初与单元等参方向对齐。

如果使用各向异性材料属性而非钢筋建模相同问题，且材料方向（1和2）最初与单元等参方向对齐，则在这种大剪切变形下，材料方向会旋转，不再与单元等参方向对齐。在这种情况下，材料方向基于材料点的平均刚体旋转确定。因此，如果材料不是真正的连续体，使用钢筋建模各向异性行为会更好。

### 在Abaqus/Standard梁单元中定义钢筋

您必须使用基于单元的钢筋（如["将钢筋定义为单元属性，"第2.2.4节"](pt01ch02s02aus14.md)中所述）在Abaqus/Standard梁单元中建模离散钢筋。您可以指定包含钢筋的单元、每个钢筋的横截面积以及相对于局部梁截面轴的每个钢筋的位置。

每个单独的钢筋必须在特定单元或单元集中分配一个单独的名称。此名称可用于定义钢筋预应力和输出请求。

| **输入文件用法：** | ``` [*REBAR*](../key/key-link.md#usb-kws-mrebar), ELEMENT=BEAM, MATERIAL=*mat*, NAME=*name* ``` |
| --- | --- |

| **Abaqus/CAE用法：** | Abaqus/CAE不支持Abaqus/Standard梁单元中的钢筋。 |
| --- | --- |

### 定义钢筋材料

钢筋的材料属性与底层单元的材料属性不同，由单独的材料定义（["材料数据定义，"第21.1.2节"](pt05ch21s01aus109.md)）。您必须将每个钢筋层（或对于Abaqus/Standard中的梁单元，每个钢筋定义）与一组材料属性关联。

以下材料行为不能在Abaqus/Standard中用于定义钢筋材料：
- ["多孔金属塑性，"第23.2.9节"](pt05ch23s02abm25.md)。

以下材料行为不能在Abaqus/Explicit中用于定义钢筋材料：
- ["线性弹性行为"中的"定义完全各向异性弹性，"第22.2.1节"](pt05ch22s02abm02.md#usb-mat-clinearelastic-anisotropic)；
- ["线性弹性行为"中的"通过指定弹性刚度矩阵项定义正交各向异性弹性，"第22.2.1节"](pt05ch22s02abm02.md#usb-mat-clinearelastic-orthoterms)；
- ["状态方程，"第25.2.1节"](pt05ch25s02abm50.md)；
- ["各向异性屈服/蠕变，"第23.2.6节"](pt05ch23s02abm22.md)；
- ["多孔金属塑性，"第23.2.9节"](pt05ch23s02abm25.md)；
- ["扩展Drucker-Prager模型，"第23.3.1节"](pt05ch23s03abm30.md)；
- ["改进的Drucker-Prager/Cap模型，"第23.3.2节"](pt05ch23s03abm31.md)；
- ["可压碎泡沫塑性模型，"第23.3.5节"](pt05ch23s03abm34.md)；或
- ["混凝土断裂模型，"第23.6.2节"](pt05ch23s06abm38.md)。

如果在钢筋层的材料中指定了非零密度，则钢筋的质量将在动态分析以及重力、离心和旋转加速度分布载荷中被考虑。

梁单元中钢筋的质量不被考虑（仅适用于Abaqus/Standard）；您应该调整梁材料的密度以考虑钢筋质量。

| **输入文件用法：** | ``` [*REBAR LAYER*](../key/key-link.md#usb-kws-mrebarlayer) *rebar layer name*, *A*, *s*, *distance of rebar from shell midsurface*, *rebar material name* ``` |
| --- | --- |

| **Abaqus/CAE用法：** | 属性模块：膜、壳或曲面截面编辑器：**钢筋层**：**材料** *rebar material name* |

### 初始条件

初始条件（["Abaqus/Standard和Abaqus/Explicit中的初始条件，"第34.2.1节"](pt07ch34s02aus116.md)）可用于定义钢筋的预应力或解相关值。

#### 在钢筋中定义预应力

对于定义了增强的结构（如钢筋混凝土结构），您可以使用初始条件来定义钢筋中的预应力。

在Abaqus/Standard这种情况下，必须在通过初始静态分析步骤（["静态应力分析，"第6.2.2节"](pt03ch06s02at01.md)）主动加载之前（或者可能仅带有"恒载"）使结构达到平衡状态——参见["Abaqus/Standard和Abaqus/Explicit中的初始条件，"第34.2.1节"](pt07ch34s02aus116.md)。

| **输入文件用法：** | ``` [*INITIAL CONDITIONS*](../key/key-link.md#usb-kws-minitialcond), TYPE=STRESS, REBAR *element number or element set name, rebar name, prestress value* ``` |
| --- | --- |

| **Abaqus/CAE用法：** | Abaqus/CAE不支持钢筋预应力。 |
| --- | --- |

#### 在Abaqus/Standard中保持钢筋中的预应力

如果在钢筋中定义了预应力，除非预应力保持固定，否则允许在平衡静态分析步骤期间发生变化；这是因为自平衡应力状态建立时结构应变的结果。一个例子是混凝土预应力的预张类型，其中增强腱最初被拉伸到所需张力，然后再被混凝土覆盖。混凝土固化并与钢筋粘合后，释放初始钢筋张力会将载荷传递到混凝土，在混凝土中引入压应力。混凝土中的 resulting 变形会减少钢筋中的应力。

或者，您可以在此初始平衡解期间保持一些或所有钢筋中定义的初始应力恒定。一个例子是混凝土后张预应力；允许钢筋穿过混凝土滑动（通常它们在管道中），预应力载荷由某些外部来源（预应力千斤顶）维持。钢筋中预应力的大小通常是设计要求的一部分，不应随着混凝土在预应力载荷下压缩而减小。通常，预应力仅在分析的第一步中保持恒定。这通常是预应力的更常见假设。

| **输入文件用法：** | ``` [*PRESTRESS HOLD*](../key/key-link.md#usb-kws-hprestresshold) ``` |
| --- | --- |

| **Abaqus/CAE用法：** | Abaqus/CAE不支持钢筋预应力。 |
| --- | --- |

### 输出

钢筋力输出在钢筋积分位置可用，输出变量为RBFOR。钢筋力等于钢筋应力乘以当前钢筋横截面积。对于膜、壳或曲面单元中的钢筋，输出变量RBANG和RBROT分别识别单元内钢筋的当前方向以及由于有限变形而导致的钢筋相对旋转。

详见["Abaqus/Standard输出变量标识符，"第4.2.1节"](pt02ch04s02abv01.md)和["Abaqus/Explicit输出变量标识符，"第4.2.2节"](pt02ch04s02xbv01.md)，了解应力、应变等其他输出量的信息。对于具有多个积分点的膜、壳或曲面单元中的钢筋，输出量可在积分点和单元质心处使用。

#### 指定钢筋角度输出的方向

输出量RBANG和RBROT可以从膜、壳或曲面单元平面中的任一等参方向测量。您可以指定所需的等参方向，从该方向测量钢筋角度（1或2）。钢筋角度从等参方向到钢筋测量，正角度定义为绕单元法线方向逆时针旋转。默认方向是第一个等参方向。

| **输入文件用法：** | ``` [*REBAR LAYER*](../key/key-link.md#usb-kws-mrebarlayer) *rebar layer name*, *A*, *s*, *distance of rebar from shell midsurface*, *rebar material name*, *angular orientation of rebar*, *isoparametric direction* ``` |
| --- | --- |

| **Abaqus/CAE用法：** | 您不能在Abaqus/CAE中指定钢筋角度输出的方向；始终使用第一个等参方向。 |
| --- | --- |

##### 示例

例如，用户定义的局部坐标系用于定义壳单元中的钢筋（=），RBANG的输出值为75。

```
[*REBAR LAYER*](../key/key-link.md#usb-kws-mrebarlayer), ORIENTATION=ORIENT
 Rbname, 0.01, 0.1, 0.0, Rbmat, 30., 2
[*ORIENTATION*](../key/key-link.md#usb-kws-morientation), SYSTEM=RECTANGULAR, NAME=ORIENT
 -0.7071, 0.7071, 0.0, -0.7071, -0.7071, 0.0
 3, 0.0
```

钢筋位于壳的中面。输出变量RBANG从第二个等参方向到钢筋测量。如果选择第一个等参方向，输出变量RBANG将报告角度为165。

#### 在Abaqus/CAE中可视化钢筋方向和结果

Abaqus/CAE支持钢筋层中钢筋方向和结果的可视化。仅当您请求钢筋的单元输出时，钢筋方向图才可用（参见["输出到输出数据库"中的"单元输出"，第4.1.3节"](pt02ch04s01aus40.md#usb-out-odboutput-elementoutput)）。钢筋的单元变量可以作为场输出进行等值线绘制或在可视化模块中绘制为历史输出。每一钢筋层将有一个唯一名称，代表膜、壳或曲面单元中的额外截面点。您可以在可视化模块中选择膜、壳或曲面单元中的命名钢筋层来显示其结果。Abaqus/CAE尚不支持梁中的钢筋。
