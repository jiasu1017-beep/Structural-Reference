# 34.4.4 集中载荷









**产品：** Abaqus/StandardAbaqus/ExplicitAbaqus/CFD

##### **参考资料**

- [*CLOAD*](#)
- ["施加载荷：概述," 第34.4.1节](pt07ch34s04aus120.md)

### 概述

集中载荷（CLOAD）用于向有限元模型中的节点或节点集施加集中力或力矩。集中载荷是最基本的载荷类型之一。

### 集中载荷定义

| **输入文件用法：** | ```
*CLOAD
node_or_node_set, degree_of_freedom, magnitude
``` |
| --- | --- |

### 参数说明

| 参数 | 描述 | 范围 |
| --- | --- | --- |
| node_or_node_set | 节点编号或节点集名称 | 有效节点 |
| degree_of_freedom | 自由度编号 | 取决于单元类型 |
| magnitude | 载荷大小 | 实数 |

### 自由度编号

| 自由度 | 描述 | 应用 |
| --- | --- | --- |
| 1 | X 方向平移 | 结构分析 |
| 2 | Y 方向平移 | 结构分析 |
| 3 | Z 方向平移 | 结构分析 |
| 4 | X 方向转动 | 梁/壳单元 |
| 5 | Y 方向转动 | 梁/壳单元 |
| 6 | Z 方向转动 | 梁/壳单元 |
| 11 | 温度 | 热分析 |

### 使用幅值曲线

可以使用幅值曲线定义集中载荷随时间的变化：

| **输入文件用法：** | ```
*CLOAD, AMPLITUDE=name
node_or_node_set, degree_of_freedom, magnitude
``` |
| --- | --- |

### 多个集中载荷

可以在同一行定义多个集中载荷：

| **输入文件用法：** | ```
*CLOAD
node_set, dof1, value1, dof2, value2, dof3, value3
``` |
| --- | --- |

### 示例

```
*CLOAD
NODE1, 1, 100.0
NODE1, 2, 50.0
NODE1, 3, 0.0
```

上述示例定义了作用于 NODE1 的三个方向的集中力。
