# 3.3 升级 FLEXnet 许可证服务器和管理工具



Abaqus 6.14 许可证需要 FLEXnet Licensing Version 11.6.1；如果许可证服务器是早期版本，则必须升级。您可以使用 FLEXnet Licensing 工具 `lmver` 或 `lmstat`（请参阅["lmstat，"第 3.6.7 节](ch03s06.md#sgb-chp-net-lic-lmstat)）检查许可证服务器的版本号。

如果需要升级许可证服务器，您应该使用许可证安装过程自动升级许可证服务器。如果您想手动配置许可证服务器，可以使用许可证安装过程仅安装 FLEXnet Licensing 管理工具。有关更多信息，请参阅["FLEXnet 许可证安装，"第 2.1.3 节](ch02s01.md#sgb-chp-install-glance-lic)。

如果您的 Abaqus 许可证文件使用组合许可证文件与 FLEXnet 其他产品的许可证文件集成，则必须确保 `lmgrd` 版本符合 FLEXnet 版本兼容性规则。有关更多信息，请参阅 FLEXnet Licensing End User Guide Version 11.6，可从 **Support** 页面上的 **Licensing** 部分下载，网址为 [www.3ds.com/simulia](http://www.3ds.com/simulia)。
