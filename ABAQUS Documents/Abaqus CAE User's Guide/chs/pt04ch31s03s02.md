# 31.3.2 选择XFEM分析的类型





Abaqus提供以下使用XFEM研究裂缝萌生和扩展的方法：

**牵引-分离内聚行为**

牵引-分离内聚行为方法在["使用富集特征模拟移动裂缝"中的"扩展有限元方法"，Abaqus Analysis User's Guide第10.7.1节](../usb/usb-link.md#usb-anl-aenrichment-cohesive)中详细描述。您可以指定定义导致最终失效的损伤演化的材料属性。您将材料应用于分配给裂缝域的截面。您可以选择将法向行为接触相互作用属性与XFEM裂缝关联，以定义裂缝单元表面的接触。详细信息请参见["为XFEM指定接触相互作用属性"，第31.3.6节](pt04ch31s03hlb03.md)。为帮助材料失效时的收敛，您可以使用粘性正则化技术引入局部阻尼。更多信息，请参见["定义损伤"中的"损伤稳定化"，第12.9.3节](pt03ch12s09s03.md#usi-prp-mechanical-damage-stabilization)。

**线弹性断裂力学（LEFM）**

线弹性断裂力学（LEFM）方法使用改进的虚拟裂缝闭合技术（VCCT）来计算裂缝尖端的应变能释放率。该方法更适用于脆性断裂问题，在["基于线弹性断裂力学（LEFM）和幻影节点原理的移动裂缝建模"中的"使用扩展有限元方法将不连续性建模为富集特征"，Abaqus Analysis User's Guide第10.7.1节](../usb/usb-link.md#usb-anl-aenrichment-lefm)中详细描述。要使用此方法，必须创建裂缝准则接触相互作用属性，如["定义接触相互作用属性"中的"为裂缝扩展指定裂缝准则属性"，第15.14.1节](pt03ch15s14s01.md#usi-itn-help-prop-contact-mech-fracture)中所述。如果您在接触相互作用属性中指定局部阻尼，Abaqus将使用粘性正则化技术来帮助材料失效时的收敛。更多信息，请参见["定义接触相互作用属性"中的"为裂缝扩展指定裂缝准则属性"，第15.14.1节](pt03ch15s14s01.md#usi-itn-help-prop-contact-mech-fracture)。
