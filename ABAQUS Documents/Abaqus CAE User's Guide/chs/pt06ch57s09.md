# 57.9 定义解相关振幅曲线







使用解相关定义方法根据解相关变量计算振幅值。有关更多信息，请参见 Abaqus Analysis User's Guide](../usb/usb-link.md#usb-prc-pamplitude-data-sol) 的"振幅曲线"第 34.1.2 节中的"为超塑性成形分析定义解相关振幅"。

**定义解相关振幅曲线：**

1. 如 ["选择要定义的振幅类型，" 第 57.3 节](pt06ch57s03.md) 中所述显示 **编辑振幅** 对话框。
2. 点击 **时间跨度** 字段旁边的箭头，并指定您希望如何将振幅定义为时间的函数：- 选择 **步骤时间** 表示从每个步骤开始测量的时间。- 选择 **总时间** 表示在所有非扰动分析步骤中累积的总时间。
3. 在 **初始振幅** 字段中，输入初始振幅值。振幅以初始值开始，然后根据解的进度进行修改。
4. 在 **最小振幅** 字段中，输入最小振幅值。
5. 在 **最大振幅** 字段中，输入最大振幅值。最大值通常是用于结束分析的控制机制。
6. 点击 **确定** 保存您的振幅定义并关闭 **编辑振幅** 对话框。
