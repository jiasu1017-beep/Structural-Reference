# 82.16 创建平面约束





**Plane Remains Plane** 插件让您可以使用以下技术之一对实体上的平面面施加约束：
- **Create Planar Constraint**——生成一个约束，使平面在分析过程中保持平面。平面可以旋转和扩展，但面上的所有节点将保持共面。此约束使用放置在面上且无摩擦的解析刚体表面来强制执行。还会在解析刚体表面参考节点和面上的所有节点之间创建分布式耦合，以约束解析表面的刚体运动。
- **Create Parallel Planar Constraint**——除了上述解析刚体表面约束外，还会生成一个约束，强制平面保持与其原始配置平行。
- **Create Beam Element Normal to Plane**——在垂直于平面的方向上生成任意长度的梁单元，并通过运动耦合连接到解析刚体表面参考节点。此技术要求您首先创建上述任一约束。

该插件有以下限制：
- 如果要选择多个平面进行约束，必须选择相互平行的平面。没有自动检查多个平面是否平行的功能。
- 应用温度场时，必须为实体和梁区域创建单独的场。
- 在大多数情况下，解析刚体表面将被调整大小以覆盖整个平面，但可能存在大小不足的情况。您应该目视确认解析刚体表面的大小；如果表面太小，您将不得不在 Part 模块中编辑其尺寸。
- 如果实体被编辑，必须删除并重新定义约束。

**创建平面约束：**

1. 在 Interaction 模块中，从主菜单栏选择 ****Plug-ins**![](../graphics/images/arrow.gif)**Tools**![](../graphics/images/arrow.gif)**Plane Remains Plane**![](../graphics/images/arrow.gif)**Create Planar Constraint**** 或 **Create Parallel Planar Constraint**。
2. 按照提示选择要保持平面的表面。选择表面可以单独进行，也可以按面角度或面曲率进行。请参见 ["在当前视口中选择对象，" 第6.2节](pt01ch06s02.md) 获取说明。完成表面选择后点击 **Done**。
3. 在 **Scaling Factor for ARS Plane** 对话框中，输入解析刚体表面的所需比例因子，然后点击 **OK**。
4. 如果需要，选择 ****Plug-ins**![](../graphics/images/arrow.gif)**Tools**![](../graphics/images/arrow.gif)**Plane Remains Plane**![](../graphics/images/arrow.gif)**Create Beam Element Normal to Plane**** 以添加垂直于平面的梁单元。按照提示选择解析刚体表面（以定义梁方向）、指定梁的端点并输入梁的长度。
