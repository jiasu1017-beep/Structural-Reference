# 15.8.3 有哪些类型的摩擦模型可用？





您可以对预定义或用户定义的摩擦行为进行建模。通常，对于预定义摩擦，您指定一组对摩擦建模的连接类型具有特征性的几何量。此外，您可以定义内部接触力贡献，例如来自连接的预紧力。Abaqus自动定义接触力贡献和发生摩擦的局部切向方向。

您可以为以下连接类型建模预定义摩擦：

**组装/复杂类型**

- 圆柱（槽 + 旋转）
- 铰链（连接 + 旋转）
- 平面（滑动平面 + 旋转）
- 滑环（复杂）
- 平移器（槽 + 对齐）
- U关节（连接 + 通用）

**基本类型**

- 滑动平面
- 槽

如果您定义的组合平移和旋转连接类型等效于这些组装类型之一，则预定义摩擦也可用。如果您正在建模预定义摩擦，对于给定的连接类型只能定义一种摩擦行为。

如果预定义摩擦模型不可用或不能充分描述正在分析的结构，您可以指定用户定义的摩擦模型（**滑环**连接类型除外，它不允许用户定义的摩擦模型）。您必须指定滑移方向信息、产生摩擦的法向力或法向力矩，以及摩擦定律。您可以使用多个连接器摩擦行为来表示连接器中的摩擦效应。

有关定义摩擦的详细说明，请参见「定义摩擦」，第15.17.3节。更多信息，请参见「连接器摩擦行为」，Abaqus Analysis User's Guide第31.2.5节。
