# 16.11.9 定义温度场





您可以为每个分析步骤定义所选区域的温度大小和时间变化；Abaqus会将温度插值到材料点。或者，您可以使用Abaqus在之前带有热组件的分析中计算出的温度值来定义当前模型中的温度。

**创建或编辑温度场的步骤：**

1. 使用以下方法之一显示温度场编辑器： - 要创建新的温度场，请按照["创建预定义场"章节第16.8.3节](pt03ch16s08hlb03.md)中概述的程序进行操作（**类别**：`其他`；**所选步骤的类型**：`温度`）。**注意：**如果您想直接在温度场编辑器中指定温度变化、在用户子程序[`UTEMP`](../sub/sub-link.md#sub-xsl-utemp)中指定，或使用解析场或离散场指定，则必须为温度场选择一个区域。如果您想指定任何其他温度分布，可以单击提示区域中的**完成**来使用计算温度。 - 要使用菜单或管理器编辑现有的温度场，请参见["编辑依赖于步骤的对象"章节第3.4.13节](pt01ch03s04hlb05.md)。
2. 如果您在创建该场之外的任何通用分析步骤中编辑温度场，请单击**状态**字段右侧的箭头，并从出现的列表中选择所需的选项： - 选择**传播**表示温度场在当前步骤中处于活跃状态，温度值从上一个步骤保持不变。不能指定任何数据；单击**确定**退出编辑器。如果您将当前步骤的状态从**修改**更改为**传播**并单击**确定**，则会移除对当前步骤的所有修改。 - 选择**修改**来编辑当前步骤的场定义。继续执行第3步。 - 选择**重置为初始**来将场定义重置为在初始步骤中规定的定义。不能指定任何数据；单击**确定**重置场并退出编辑器。
3. 如果您要创建新的温度场，请单击**分布**字段右侧的箭头，并从出现的列表中选择所需的选项。**注意：**如果您在初始步骤中创建温度场，则只有**直接指定**、**从结果或输出数据库文件**、解析场和离散场分布选项可用。 - 选择**直接指定**在温度场编辑器中指定所选区域上的温度变化。 - 选择**从结果或输出数据库文件**从先前带有热组件的Abaqus分析的结果或输出数据库文件读取温度值。 - 选择**用户自定义**在用户子程序[`UTEMP`](../sub/sub-link.md#sub-xsl-utemp)中定义温度值。此选项仅对Abaqus/Standard分析可用。有关更多信息，请参见以下章节： - ["指定常规作业设置"章节第19.8.6节](pt03ch19s08hlb06.md) - ["UTEMP"，《Abaqus User Subroutines Reference Guide》第1.1.51节](../sub/sub-link.md#sub-rtn-uutemp) - 选择**从结果或输出数据库文件和用户自定义**从先前带有热组件的Abaqus分析的结果或输出数据库文件读取温度值，并在用户子程序[`UTEMP`](../sub/sub-link.md#sub-xsl-utemp)中修改它们。此选项仅对Abaqus/Standard分析可用。 - 选择标记为(A)的解析场或标记为(D)的离散场来定义空间变化的温度。选择列表包含所有解析场和仅对温度场有效的离散场。或者，您可以单击![](../graphics/ico_analyticalField.png)来创建新的解析场。（有关更多信息，请参见[第58章，"解析场工具集"](pt06ch58.md)。）
4. 如果您选择了**直接指定**、解析场或离散场分布选项，请执行以下步骤： 1. 单击**截面变化**字段右侧的箭头，并从出现的列表中选择一个选项。只有对所选区域有效的选项才会出现在列表中。梁和壳的梯度值不受解析场或离散场的影响。您应选择与为该场选择的区域关联的截面定义中定义的温度变化方法一致的选项： - 选择**区域内恒定**来定义截面上的恒定温度。在**幅值**文本字段中，输入整个截面的温度大小。 - 选择**壳截面内的梯度**来定义壳截面内的温度变化。壳截面编辑器中的**温度变化**方法必须指定**厚度方向线性**。在**参考幅值**文本字段中，输入参考面上的温度大小。在**厚度梯度**文本字段中，输入通过截面的温度梯度。 - 选择**梁截面内的梯度**来定义梁截面内的温度变化。（此方法不适用于轴对称模型。）梁截面编辑器中的**温度变化**方法必须指定**按梯度线性**。在**参考幅值**文本字段中，输入横截面原点处的温度大小。在**N1梯度**和**N2梯度**文本字段中，分别输入梁的![](../graphics/usi_eqn00643.gif)和![](../graphics/usi_eqn00644.gif)方向通过截面的温度梯度。对于平面中的梁，只需指定温度大小和**N2梯度**。 - 选择**在壳/梁温度点定义**来定义通过壳或梁截面的分段线性温度变化。壳截面编辑器中的**温度变化**方法必须指定**在*n*值上分段线性**。梁截面编辑器中的**温度变化**方法必须指定**从温度点插值**。通过在**温度点**文本字段中输入整数或使用文本字段右侧的箭头选择数字来选择要指定的温度点数量，并在**截面数据**表中输入每个点的温度大小。截面定义中的值的数量应小于或等于为此场给出的温度数据点的数量。如果值的数量较少，最后一个值将被重复以匹配温度数据点的数量。 2. 如果需要（对于初始步骤以外的所有步骤），单击**幅值**文本字段右侧的箭头，并从出现的列表中选择所需的幅值。或者，您可以单击![](../graphics/ico_amplitudeSmall.png)来创建新的幅值。（有关更多信息，请参见[第57章，"幅值工具集"](pt06ch57.md)。）梁和壳的梯度值将被幅值定义修改。 3. 单击**确定**保存数据并退出编辑器。
5. 如果您选择了**从结果或输出数据库文件**分布选项，请执行以下步骤： 1. 在**文件名**文本字段中，输入要从中读取温度数据的结果或输出数据库文件的名称；或单击![](../graphics/ico_fileOpen.png)显示**选择结果或输出数据库文件**对话框并选择所需的文件。（有关更多信息，请参见["使用文件选择对话框"章节第3.2.10节](pt01ch03s02s10.md)。） 2. 如果您在初始步骤中创建温度场： - 在**步骤**文本字段中，输入要用作输入的分析的步骤编号，从中读取场数据。 - 在**增量**文本字段中，输入要用作输入的分析的增量编号，从中读取场数据。 3. （可选）如果您在初始步骤以外的分析步骤中创建温度场： - 在**开始步骤**文本字段中，输入要用作输入的分析的步骤编号，该分析开始读取场数据。默认情况下，将使用结果或输出数据库文件中可用的第一个步骤。 - 在**开始增量**文本字段中，输入要用作输入的分析的增量编号，该增量开始读取场数据。默认情况下，将使用结果或输出数据库文件中可用的第一个增量。 - 在**结束步骤**文本字段中，输入要用作输入的分析的步骤编号，该分析结束读取场数据。默认情况下，将使用**开始步骤**值。 - 在**结束增量**文本字段中，输入要用作输入的分析的增量编号，该增量结束读取场数据。默认情况下，将使用结果或输出数据库文件中**结束步骤**值可用的最后一个增量。 4. 选择**网格兼容性**。 - 如果原始分析和当前分析中的网格相同或仅在单元阶次上不同，请选择**兼容**。如果原始和当前网格仅在单元阶次上不同，则必须通过开启**插值中间节点**来表示将从角节点温度插值二阶单元的中间节点温度。 - 如果原始分析和当前分析中的网格不同，请选择**不兼容**。此选项仅在从输出数据库文件读取温度值时有效。 5. 如果原始和当前网格不同，您可以为**外部容差**指定一个值。 - 指定当前模型的节点可能位于原始模型元素区域外部的绝对值。如果未指定此值或为0.0，则将应用**相对**容差。 - 指定当前模型的节点可能位于原始模型元素区域外部的平均元素尺寸的相对分数。如果同时指定了两种容差，Abaqus使用更严格的容差。 6. 单击**确定**保存数据并退出编辑器。
6. 如果您选择了**用户自定义**分布选项，请执行以下步骤： 1. 单击**确定**退出编辑器。 2. 进入作业模块，并显示感兴趣分析作业的作业编辑器。（有关更多信息，请参见["创建、编辑和操作作业"章节第19.7节](pt03ch19s07.md)。） 3. 在作业编辑器中，单击**常规**选项卡，并指定包含定义温度场的用户子程序的文件。有关更多信息，请参见["指定常规作业设置"章节第19.8.6节](pt03ch19s08hlb06.md)。**注意：**您只能在作业编辑器中指定一个用户子程序文件；如果您的分析涉及多个用户子程序，则必须将用户子程序合并到一个文件中，然后指定该文件。
7. 如果您选择了**从结果或输出数据库文件和用户自定义**分布选项，请遵循第5步和第6步中概述的**从结果或输出数据库文件**和**用户自定义**分布选项的程序。
