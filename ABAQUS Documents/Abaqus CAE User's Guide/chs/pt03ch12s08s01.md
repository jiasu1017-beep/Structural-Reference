# 12.8.1 指定材料质量密度







您可以将密度定义为温度和场变量的函数。对于声学、热传递、耦合温度-位移和耦合热-电单元，密度会持续更新到与当前温度和场变量对应的值。但是，对于所有其他单元，密度是温度和场变量初始值的函数，仅随体积变化。Abaqus不会在分析过程中温度和场变量发生变化时更新密度值。

有关更多信息，请参见"密度"，Abaqus分析用户指南第21.2.1节。

**指定密度：**

1. 从**编辑材料**对话框的菜单栏中，选择**常规**![](../graphics/images/arrow.gif)**密度**。（有关显示**编辑材料**对话框的信息，请参见"创建或编辑材料"，第12.7.1节。）
2. 单击**分布**字段右侧的箭头，从出现的列表中选择您的选项：- 选择**均匀**定义在整个材料中均匀分布的密度。- 选择解析场（标记为(A)）或离散场（标记为(D)）来定义空间变化的密度。只有对密度有效的解析场和离散场才会在选择列表中可用。有关更多信息，请参见第58章"解析场工具集"和第63章"离散场工具集"。或者，您可以单击![](../graphics/ico_discreteField.png)创建新的离散场。
3. 切换**使用温度相关数据**将密度定义为温度的函数。**数据**表中会出现一列标为**温度**。
4. 单击**场变量数量**字段右侧的箭头增加或减少密度所依赖的场变量数量。
5. 在**数据**表中输入以下数据：**质量密度**质量密度。（单位为[ML3](../popups/usb-int-iconventions-unitsym.md)。）**温度**温度。**场变量* n ***预定义的场变量。有关如何输入数据的详细信息，请参见"输入表格数据"，第3.2.7节。
6. 单击**确定**关闭**编辑材料**对话框。或者，您可以从**编辑材料**对话框的菜单中选择其他材料行为进行定义（有关更多信息，请参见"浏览和修改材料行为"，第12.7.2节）。
