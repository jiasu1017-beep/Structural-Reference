# 17.4.4 对种子施加曲率控制






零件布种工具允许您在布种零件、零件实例或多条边时指定目标单元尺寸。如果几何形状相对规则，指定单一目标单元尺寸即可生成可接受的网格。然而，如果您指定了单一目标单元尺寸，但组成零件或边的几何特征尺寸不一致，则生成的网格可能过于粗糙，无法充分表达任何小特征，如图17-8所示。

**图17-8** 无曲率控制时的布种及生成的网格。

![](../graphics/mgn-seed-nocc.png)

为避免在小曲线特征周围布种不足的问题，Abaqus/CAE在布种零件、零件实例或边时会应用曲率控制。曲率控制允许Abaqus/CAE根据边的曲率以及目标单元尺寸来计算种子分布。如图17-9所示，启用曲率控制后对相同零件进行布种和网格划分。

**图17-9** 启用曲率控制时的布种及生成的网格。

![](../graphics/mgn-seed-cc.png)

您可以配置以下选项来指定曲率控制如何影响布种：

**偏差因子**

偏差因子用于衡量单元边与原始几何形状的偏离程度，如图17-10所示。

**图17-10** 偏差因子。

![](../graphics/mgn-seed-deviationfactor-nls.png)

为帮助您可视化偏差因子，Abaqus/CAE会显示它将在与您输入的设置对应的圆周上创建的大致单元数量。当您减小偏差因子时，Abaqus/CAE将在圆周上创建的单元数量会增加。此数字仅供参考；例如，如果您对样条曲线或椭圆进行布种，Abaqus/CAE会创建不同数量的单元，具体取决于沿边的局部曲率。

**指定最小尺寸因子**

指定最小尺寸因子可以防止Abaqus/CAE在您不感兴趣建模的高曲率区域创建非常精细的网格；例如，样条曲线中的折点或半径非常小的圆角。您输入的最小尺寸数字是全局种子尺寸的分数。因此，如果您更改全局种子尺寸，则不必更改最小尺寸因子。

有关应用曲率控制的详细说明，请参见["为整个零件或零件实例定义种子密度，"第17.16.1节](pt03ch17s16hlb01.md)。
