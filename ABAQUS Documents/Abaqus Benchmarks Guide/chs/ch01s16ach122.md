# 1.16.3 中心斜裂纹板在拉力作用下

**产品：** Abaqus/Standard

### 问题描述

![](../graphics/vercslantcrack.png)

模型由一个具有中心斜裂纹并受到远场拉力的矩形板组成。计算使用 CPE8 单元和线弹性材料进行。为板选择的几何比为 ![](../graphics/bmk_eqn01149.gif) = 0.4 和 ![](../graphics/bmk_eqn01150.gif) = 2。斜角为 ![](../graphics/bmk_eqn00198.gif) = 45°。在顶部和底部板表面上规定位移边界条件，以向板施加拉力，从而可以获得两个裂纹尖端的合理解。总的远端拉力荷载通过对顶部或底部平面上每个节点的 *y* 方向节点反作用力求和获得；远端应力 ![](../graphics/bmk_eqn00218.gif) 定义为总拉力除以板的横截面积。

### 结果与讨论

计算的应力强度因子与 Y. Murakami 编辑的《应力强度因子手册》第 909 页给出的解进行了比较。

**表 1.16.3-1** 各向同性弹性的无量纲应力强度因子结果。轮廓 1 从计算中省略。
|  | ![](../graphics/bmk_eqn01151.gif) | ![](../graphics/bmk_eqn01152.gif) |
| --- | --- | --- |
| 参考解 | 0.5719 | 0.5290 |
| Abaqus | 0.5540 | 0.5289 |

基于应力强度因子 ![](../graphics/bmk_eqn01130.gif) 和 ![](../graphics/bmk_eqn01133.gif)，Abaqus 可以自动预测裂纹扩展方向，这是相对于裂纹平面测量的角度 ![](../graphics/bmk_eqn01153.gif)。例如，如果使用最大切向应力准则，![](../graphics/bmk_eqn01154.gif) = 52.41；如果使用最大能量释放率准则，![](../graphics/bmk_eqn01154.gif) = 55.76；如果使用 ![](../graphics/bmk_eqn01155.gif) = 0 准则，![](../graphics/bmk_eqn01154.gif) = 56.12。

Abaqus 还输出由应力强度因子估计的 *J* 积分值，![](../graphics/bmk_eqn01156.gif) = 3.4517×10³，这与直接估计的 *J* 积分值 ![](../graphics/bmk_eqn01156.gif) = 3.4515×10³ 非常一致。

此外，对于使用四种不同各向异性线弹性材料的相同板评估了应力强度因子和 *J* 积分：由工程常数规定的正交弹性（用 ENGC 表示）、由刚度参数规定的正交弹性（ORTH）、完全各向异性弹性（ANIS）和层状弹性（LAMI）。具有层状弹性的模型使用平面应力 CPS8 单元进行网格划分。结果总结在[表 1.16.3-2](ch01s16ach123.md#table-k12jresults) 中。虽然没有可比较的已发布解，但应力强度因子的 *J* 积分与直接评估的 *J* 积分非常一致。
