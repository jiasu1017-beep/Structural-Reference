# 1.6.17 曲面初始过闭合的接触





**产品：**Abaqus/Standard  Abaqus/Explicit  

### 单元测试

C3D8    C3D8R    CPS4    CPS4R  

### 功能测试

接触阻尼

接触对

修正的接触压力-闭合关系

### 问题描述

该模型由两个具有小初始过闭合的同心环组成。在Abaqus/Standard中，初始过闭合在静态步骤期间被解析。在Abaqus/Explicit中，初始过闭合在动态模拟的第一步期间被解析。

二维模型由两个16单元环组成，三维模型由两个32单元环组成。内环和外环的单元完全对齐。

**材料：**

| 杨氏模量 | 206800 |
| --- | --- |
| 泊松比 | 0.32 |
| 密度 | 8.01×10⁶ |
| 界面摩擦系数 | 0.0 |

### 结果与讨论

干涉在使用硬接触的模型中被解析。对于软化接触的情况，干涉减少直到达到平衡；步骤结束时的任何残余过闭合可以通过增加压力-闭合关系的刚度来减少。
