# 57.12 定义用户定义振幅曲线







使用用户定义方法在用户子程序 [`UAMP`](../sub/sub-link.md#sub-xsl-uamp)（Abaqus/Standard）或 [`VUAMP`](../sub/sub-link.md#sub-xsl-vuamp)（Abaqus/Explicit）中定义振幅曲线。有关更多信息，请参见 Abaqus Analysis User's Guide](../usb/usb-link.md#usb-prc-pamplitude-data-user) 的"振幅曲线"第 34.1.2 节中的"通过用户子程序定义振幅"。

**定义用户定义振幅曲线：**

1. 如 ["选择要定义的振幅类型，" 第 57.3 节](pt06ch57s03.md) 中所述显示 **编辑振幅** 对话框。
2. 在 **变量数** 字段中，输入必须与此振幅定义一起存储的解相关状态变量的数量。
3. 在用户子程序 [`UAMP`](../sub/sub-link.md#sub-xsl-uamp)（用于 Abaqus/Standard）或 [`VUAMP`](../sub/sub-link.md#sub-xsl-vuamp)（用于 Abaqus/Explicit）中定义振幅。请参阅以下各节获取更多信息：- ["指定常规作业设置，" 第 19.8.6 节](pt03ch19s08hlb06.md)- ["UAMP，" Abaqus User Subroutines Reference Guide](../sub/sub-link.md#sub-rtn-uuamp) 第 1.1.19 节- ["VUAMP，" Abaqus User Subroutines Reference Guide](../sub/sub-link.md#sub-rtn-uexpamp) 第 1.2.8 节
4. 点击 **确定** 保存您的振幅定义并关闭 **编辑振幅** 对话框。
