# 47.6.3 自定义X–Y图图例





### 47.6.3 自定义X–Y图图例

*X–Y*图的图例提供了一个键，将图中包含的每条曲线与其表示的*X–Y*数据对象关联起来。每个图例条目包含一个以图中显示的颜色和样式绘制的线段。图例条目还包括文本，默认情况下说明曲线表示的*X–Y*数据的名称。您可以使用**Curve Options**自定义此文本。

图例的其余特征由**Viewport Annotation Options**对话框中的选项控制。有关自定义图例以及在图例中包含最小和最大*X–Y*值报告的信息，请参见["自定义图例，"第56.1节](pt05ch56s01.md)。

**要自定义X–Y图的图例文本：**

1. 找到**Attributes**选项。**Attributes**框架位于**Curve Options**对话框的底部。
2. 在**Curves**选项中，选择要修改其图例文本的一条或多条曲线。
3. 在**Legend Text**字段中，输入自定义图例文本。您也可以通过切换**Use default**恢复默认图例文本。
4. 点击**Dismiss**关闭**Curve Options**对话框。
