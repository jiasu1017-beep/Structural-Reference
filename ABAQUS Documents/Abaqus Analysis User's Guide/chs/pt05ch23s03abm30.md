### 概述

扩展德鲁克-普拉格模型：
- 可与线弹性模型（["Linear elastic behavior," Section 22.2.1](pt05ch22s02abm02.md)）和各向同性硬化塑性模型（见["Classical metal plasticity," Section 22.2.1](pt05ch22s02abm02.md)）结合使用；
- 考虑静水压力对屈服的影响；
- 支持率相关（粘性）响应；
- 支持温度依赖性；以及
- 包含可选的硬化定律。

### 硬化定律

扩展德鲁克-普拉格模型支持等向硬化。硬化数据定义了屈服应力作为等效塑性应变的函数。

| **输入文件用法：** | ``` [*DRUCKER-PRAGER](../key/key-link.md#usb-kws-mdruckerprager) [*HARDENING](../key/key-link.md#usb-kws-mhardening) ``` |
| --- | --- |

### 率相关响应

扩展德鲁克-普拉格模型支持率相关响应。Abaqus 使用过应力理论来考虑应变率对屈服应力的影响。

| **输入文件用法：** | ``` [*DRUCKER-PRAGER](../key/key-link.md#usb-kws-mdruckerprager), RATE=参数 ``` |
| --- | --- |

### 温度依赖性

扩展德鲁克-普拉格模型的参数可以定义为温度的函数。

| **输入文件用法：** | ``` [*DRUCKER-PRAGER](../key/key-link.md#usb-kws-mdruckerprager), TEMPERATURE ``` |
| --- | --- |

### 与其他材料模型的结合

扩展德鲁克-普拉格模型可与线弹性模型和各向同性硬化塑性模型结合使用。

### 元素

扩展德鲁克-普拉格模型可与 Abaqus/Standard 和 Abaqus/Explicit 中的连续体元素一起使用。
