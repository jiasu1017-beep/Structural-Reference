# 3.2.5 Dymola模型执行





**产品：** Abaqus/Standard  Abaqus/Explicit  

##### **参考**

- ["Abaqus执行程序：概述，" 第3.1.1节](pt01ch03s01abo02.md)
- ["SIMULIA协同仿真引擎director执行，" 第3.2.3节](pt01ch03s02abx03.md)

### 概述

您可以在Abaqus/Standard或Abaqus/Explicit模型与从Dymola导出的模型之间执行协同仿真。此过程要求您在当前工作目录中提供以下两个文件：`dymosim.dll`，其中包含模型的Dymola导出详细信息，以及`libdsdll.dll`，其中包含Dymola库的基本集合。有关创建或获取这些文件的详细信息，请参阅["结构到逻辑协同仿真，" 第17.4.1节](pt04ch17s04aus103.md)。

### 命令摘要

| **abaqus dymola** | [**input**](pt01ch03s02abx05.md#input1-usb-int-dcosimdymolaproc)=*Dymola map file name* |
| --- | --- |
|  | [**port**](pt01ch03s02abx05.md#port2-usb-int-dcosimdymolaproc)=*co-simulation port number* [**host**](pt01ch03s02abx05.md#host3-usb-int-dcosimdymolaproc)=*co-simulation host name* |

### 命令行选项

**input**

此选项用于指定映射文件名。映射文件具有`.sgn`文件扩展名。

**port**

此选项用于指定使用直接耦合接口的求解器之间协同仿真（包括Abaqus与某些第三方分析程序之间的协同仿真）的TCP/UDP端口号。将**port**设置为用于连接的端口号。默认值为`48000`。Abaqus用于启动通信的默认端口号可以使用环境文件中的**cosimulation_port**参数设置（请参阅["使用Abaqus环境设置，" 第3.3.1节](pt01ch03s03aus30.md)）。此选项与**host**选项结合使用。有关更多信息，请参阅["选择TCP/UDP端口号"在"Abaqus执行程序：概述，" 第3.1.1节](pt01ch03s01abo02.md#usb-int-doverview-port)。

**host**

此选项用于指定使用直接耦合接口的求解器之间协同仿真（包括Abaqus与某些第三方分析程序之间的协同仿真）的主机名。此选项指定托管连接的主机名称。请参考第三方程序文档以确定是否需要**host**选项。此选项与**port**选项结合使用。

### 示例

以下示例说明了在涉及Abaqus/Explicit和导出的Dymola模型的协同仿真中使用Dymola执行程序。

#### 运行Dymola模拟

使用以下命令启动Dymola模拟，该模拟将在端口44444上监听来自Abaqus/Explicit的连接：

```
abaqus dymola input=inverted_pend_map port=44444
```

#### 运行Abaqus/Explicit模拟

使用以下命令进行Abaqus/Explicit分析，该分析将通过端口44444连接到名为"mercury"的机器上的Dymola模拟：

```
abaqus job=inverted_pend_xpl host=mercury port=44444
```
