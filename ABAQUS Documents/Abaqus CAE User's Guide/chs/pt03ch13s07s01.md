# 13.7.1 合并和切割零件实例

从主菜单栏选择**实例**![](../graphics/images/arrow.gif)**合并/切割**来合并零件的多个实例。要合并的零件可以包含几何和孤儿网格节点及元素的任意组合；并且有用于合并几何、网格（孤儿和原生）或两者的选项。此外，您可以使用一个或多个零件实例的几何部分来切割零件实例的几何部分。合并和切割操作都会创建新的零件实例和新的零件。当您合并或切割零件实例时，您可以选择抑制或删除原始零件实例。以下更详细地描述了合并和切割操作。

**合并**

您可以选择多个零件实例并将它们合并为单个零件实例。例如，[图13-18](pt03ch13s07s01.md#asm-conc-position-merge)显示了两个对15针连接器建模的零件实例。这两个零件实例沿公共面定位，然后合并为可以网格划分和分析的单个零件实例。

**图13-18** 两个零件实例合并为单个零件实例。

![](../graphics/asm-conc-position-merge.png)

即使实例不接触或重叠，您也可以合并零件实例。您可以选择移除或保留合并零件实例之间的相交边界，如图13-19所示。如果需要，您可以使用**零件副本**对话框创建关于三个主平面之一的零件镜像图像。更多信息，请参见["复制零件，"第11.5节"](pt03ch11s05.md)。

**图13-19** 移除和保留相交边界的效果。

![](../graphics/asm-position-intersections.png)

如果您合并网格，可以指定**节点合并容差**，这是将被合并的节点之间的最大距离。Abaqus/CAE通过删除距离小于指定值的节点并用单个新节点替换它们来创建兼容网格。新节点的位置是已删除节点的平均位置。如果您为**节点合并容差**输入的值太大，Abaqus/CAE可能会检测到来自同一元素的重复节点。Abaqus/CAE不会合并来自同一元素的节点。但是，大的容差可能导致网格变形，Abaqus/CAE会询问您是否要继续或结束合并过程。如果没有节点彼此距离小于指定距离，Abaqus/CAE会询问您是要取消过程还是从所选实例创建单个实例。

当您合并相交的网格零件实例时，可以选择是否创建重复元素以及重复节点。重复元素具有与另一元素相同的连接。默认情况下，Abaqus/CAE删除重复元素，在大多数情况下您应该接受默认行为。但是，如果要使用Abaqus不支持的材料属性组合建模材料，必须保留重复元素，如["无压缩或无拉伸，"第22.2.2节，Abaqus分析用户指南"](../usb/usb-link.md#usb-mat-cnocompnoten)中的稳定性讨论中所述。

您可以在以下节点合并方法之间进行选择：

**仅边界**

默认情况下，Abaqus/CAE仅沿边界（对于三维实例由自由面定义，对于二维实例由自由边定义）合并网格零件实例。自由边和面是仅属于一个几何实体或元素的边和面。使用此设置，Abaqus/CAE不检查零件内部的重复节点，这加快了合并过程。如果三维零件实例仅在公共面相交或二维实例仅在公共边相交，则应保留此默认设置。

**全部**

如果零件实例重叠，您可能希望合并所选零件实例中的所有节点。

**无**

或者，您可以选择不合并任何节点，在这种情况下，Abaqus/CAE将零件实例合并为单个实例，但保留所有原始节点。

在许多情况下，您将合并不相交但共享公共面的零件实例；例如，[图13-20](pt03ch13s07s01.md#asm-conc-position-mergeorphan)所示的两个零件实例。

**图13-20** 两个网格零件实例合并为单个网格零件实例。

![](../graphics/asm-conc-position-mergeorphan.png)

您也可以使用编辑网格工具集合并网格零件的所选节点；更多信息，请参见["操作节点，"第64.1.1节"](pt06ch64s01s01.md)。

尽管结果合并网格可能在视口中看起来可以接受，但网格可能在节点和元素面之间包含不易察觉的小间隙。网格还可能包含具有不兼容网格模式的合并面。您可以使用查询工具集中的**网格间隙/相交**工具来检查小间隙和不兼容面。更多信息，请参见["获取模型常规信息，"第71.2.2节"](pt06ch71s02hlb02.md)。

当您合并零件实例时，原始零件和零件实例上的任何集或面都会映射到新的零件和零件实例。如果不同零件的集或面具有相同名称，它们将合并为合并零件和零件实例上的单个集或面。如果您选择移除合并零件实例之间的相交边界，则位于这些边界边和面上的集或面的部分将从映射的集和面中移除。

原始零件的截面分配也会映射到新零件。如果原始装配中的零件相交，Abaqus/CAE只能映射相交区域中的单个截面。类似地，如果零件完全接触或相交，并且在合并期间移除了相交的边界，Abaqus/CAE会将单个截面映射到整个合并零件。在这些相交情况下，被映射的截面取决于多种因素，可能与您的建模意图不符。合并相交区域时，应保留相交边界；边界保留非相交区域中的原始截面分配，并在需要时更容易修改映射的截面分配（详情请参见["管理截面分配，"第12.15.2节"](pt03ch12s15hlb02.md)）。

**注意：**梁截面分配和钢筋参考方向不会映射到合并零件。合并后，您必须重新创建它们和任何关联的属性。

您可能出于以下原因要合并零件实例：
- 如果独立实例中的几何接触或重叠但您不合并实例，Abaqus/CAE为每个实例创建单独的网格，您必须施加绑定约束以有效地合并节点。相反，当您合并零件实例时，Abaqus/CAE创建单个组合网格，您不需要施加计算成本昂贵的绑定约束。实际上，您已在零件实例之间创建了兼容网格。如果要保留独立零件实例的概念，可以在合并实例的公共界面处创建分区。
- 合并零件实例允许您将材料属性分配给由合并操作创建的单个零件，而不是分配给每个零件。
- 您可以将显示体约束应用于一组合并的零件实例，而不是分别应用于每个零件实例。
- 当您导入复杂装配时，装配可能在Abaqus/CAE中显示为大量将单独进行网格划分的单个零件实例。您可以将所有零件实例合并为单个零件实例，或者可以将零件实例组合并为几个独立的零件实例。

合并零件实例时，您有以下三个选项：

**几何**

仅合并几何。正在合并的实例的任何孤儿网格部分将从合并的零件和零件实例中删除。

**网格**

合并所有原生和孤儿网格组件。正在合并的实例的任何几何将从合并的零件和零件实例中删除。原始零件的原生网格部分成为新零件中孤儿网格的一部分。

**两者**

合并几何和孤儿网格。在此过程中删除任何原生网格。

**切割**

您可以选择要切割的单个零件实例的几何部分，然后您可以选择接触或重叠要切割零件实例的一个或多个零件实例的几何。Abaqus/CAE使用将进行切割的几何（模具）从要切割的实例的几何（坯料）上切割掉。几何必须接触或重叠才能创建切割的零件和零件实例。如果被切割的零件实例包含孤儿网格元素，它们不会受到切割操作的影响。

当您切割零件实例时，原始零件和零件实例上的集、面和截面分配将映射到新的零件和零件实例。原始几何的切割部分内的原始集和面的部分将从映射的集和面中移除。

如果您想从零件创建模具或反之，切割操作非常有用。[图13-21](pt03ch13s07s01.md#asm-conc-position-mergecut)显示了一个瓶子和一个矩形坯料，以及切割过程如何创建模具。

**图13-21** 使用切割操作从坯料和模具创建的模具。

![](../graphics/asm-conc-position-mergecut-nls.png)

您不能用壳零件实例进行切割。因此，在执行切割操作之前，瓶子已从壳转换为零件模块中的实体零件。更多信息，请参见["从壳创建实体特征，"第11.21.5节"](pt03ch11s21hlb05.md)。此外，原始零件实例（坯料和模具）在切割操作后被抑制。当您执行声学或冲击分析时，切割操作还可用于对结构和声学介质进行建模。

**注意：**您无法合并或切割包含虚拟拓扑的零件实例。
