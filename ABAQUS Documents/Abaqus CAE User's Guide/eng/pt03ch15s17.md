# 15.17 Using the connector section editors







After you specify a connection type for a connector section, you can define connector behaviors for connection types that have one or more available components of relative motion. If you change the connection type after you have defined connector behaviors, you will have to delete the connector behaviors and redefine them for the new connection type.
