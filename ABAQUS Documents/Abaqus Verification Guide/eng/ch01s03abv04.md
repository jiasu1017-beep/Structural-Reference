### Results and discussion

The results for generalized plane strain elements depend on the boundary constraints applied to the generalized plane strain reference node. In these tests the reference nodes in the lower-order generalized plane strain elements are constrained such that the results are the same as their plane strain counterparts. For the higher-order generalized plane strain elements these nodes are unconstrained, so the results are the same as their plane stress counterparts.

Elements using reduced integration may have additional boundary conditions to those specified above. All elements yield exact solutions.
