# 11.5.2 流体腔体定义

**产品：** Abaqus/Standard Abaqus/Explicit

##### **参考**

- ["Fluid cavity overview," Section 11.5.1](pt04ch11s05aus70.md)
- [*FLUID CAVITY](../key/key-link.md#usb-kws-mfluidcavity)
- [*FLUID PROPERTY](../key/key-link.md#usb-kws-mfluidproperty)

### 概述

流体腔体定义涉及创建腔体几何、定义流体属性和设置初始条件。本节详细介绍如何定义流体腔体及其相关参数。

### 定义流体腔体

#### 创建腔体表面

流体腔体由一组表面定义，这些表面形成了容纳流体的封闭或半封闭空间。表面可以来自：
- 壳单元表面
- 连续体单元表面
- 膜单元表面

#### 定义腔体属性

使用**FLUID CAVITY**关键字定义流体腔体的属性：

- **CAVITY DEFINITION**：腔体名称和标识
- **SURFACE**：形成腔体的表面
- **CAVITY VOLUME**：初始腔体体积
- **PRESSURE**：初始压力

### 流体属性定义

#### 气体属性

对于气体，需要定义理想气体状态方程参数：

```
[*FLUID PROPERTY](../key/key-link.md#usb-kws-mfluidproperty), TYPE=GAS
*气体常数
```

气体状态方程：PV = nRT

其中：
- P：压力
- V：体积
- n：摩尔数
- R：通用气体常数
- T：温度

#### 液体属性

对于液体，需要定义体积模量：

```
[*FLUID PROPERTY](../key/key-link.md#usb-kws-mfluidproperty), TYPE=LIQUID
*体积模量
```

液体状态方程：P = P0 + K(P0/P0 * (V0/V) - 1)

其中：
- K：体积模量
- P0：参考压力
- V0：参考体积

### 初始条件

#### 初始压力

可以指定腔体的初始压力状态。默认初始压力为0。

#### 初始体积

对于预定义形状的腔体，可以指定初始体积。对于变形后形成腔体的情况，初始体积由初始几何构型确定。

### 腔体与结构耦合

#### 体积交换

流体腔体与结构之间通过体积交换实现耦合。当结构变形时，腔体体积随之变化。

#### 压力作用

腔体内压力作用于腔体壁面，产生分布载荷。

### 接触定义

如果腔体表面与其他表面存在接触，需要定义接触对。接触属性包括：
- 法向行为：传递压力
- 切向行为：可能存在摩擦

### 示例：安全气囊腔体定义

```
*FLUID CAVITY, NAME=airbag
airbag_surface
*FLUID PROPERTY, TYPE=GAS
1.4
```

### 多腔体系统

可以定义多个相互连接的流体腔体，形成复杂的流体网络。腔体之间可以通过：
- **FLUID EXCHANGE**：定义腔体间的流体交换
- **ORIFICE**：定义节流孔，控制流量

### 诊断与调试

流体腔体定义时常见问题：
- 表面不封闭：检查腔体表面是否形成完整封闭
- 初始体积为零：确保初始几何构型有效
- 接触定义错误：检查接触对是否正确定义
