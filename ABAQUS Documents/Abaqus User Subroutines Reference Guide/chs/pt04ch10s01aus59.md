### 概述

本节描述如何定义单个子结构。有关如何在模型中使用它们的信息，请参阅["使用子结构，" 第 10.1.1 节"](pt04ch10s01aus58.md)。

子结构使用子结构生成过程定义。子结构创建和使用不能包括在同一分析中。可以在一个分析中生成多个子结构。任何子结构可以由一个或多个其他子结构组成；如果是这种情况，必须首先定义嵌套级别的子结构。子结构库不是按部件实例组织的；因此，不能从定义了装配的模型生成子结构。在定义了装配的模型中不支持任何子结构选项。

要定义典型的子结构生成步骤，请执行以下操作：
- 调用子结构生成过程。
- 定义当使用子结构时要保留为外部自由度的节点和自由度。
- 可选地，保留额外的动态模态以改善子结构在使用期间的动态行为。
- 可选地，指定子结构载荷情况。
- 可选地，将恢复矩阵、子结构的刚度矩阵、质量矩阵和/或载荷情况向量写入文件。

### 生成子结构

当您生成子结构时，您指定一个标识符，该标识符将在子结构库中分配给此子结构。标识符必须以字母 Z 开头，后跟一个不超过 9999 的数字。

库中的子结构标识符必须唯一。如果库中已存在具有此相同标识符的子结构，除非您已指定应覆盖现有子结构（如后所述），否则分析将终止并显示错误消息。

| **输入文件用法：** | ``` [*SUBSTRUCTURE GENERATE](../key/key-link.md#usb-kws-ssubgenerate), TYPE=Z*n* ``` |
| --- | --- |

| **Abaqus/CAE 用法：** | 步骤模块：**创建步骤**：**线性扰动**：**子结构生成**：*n* |
| --- | --- |

#### 子结构数据库

子结构数据库是描述子结构几何形状的文件集，Abaqus 在分析期间将所有子结构数据写入子结构数据库。子结构数据库可以包括具有以下扩展名的文件：`.sup`、`.sim`、`.prt`、`.mdl` 和 `.stt`；`.sup` 文件称为子结构库。默认情况下，子结构数据被写入名为 *jobname* 的子结构数据库，子结构文件被命名为 *jobname*`.sup`、*jobname_Zn*`.sim`、*jobname_Zn*`.prt`、*jobname_Zn*`.mdl` 和 *jobname_Zn*`.stt`。具有 `.sup` 和 `.sim` 扩展名的文件为所有子结构生成。具有 `.prt`、`.mdl` 和 `.stt` 扩展名的文件仅在可以完全或部分恢复子结构内部时生成。多个子结构可以共享一个子结构库文件，但其他文件对每个子结构是单独的。强烈建议不同的子结构使用不同的子结构库名称。

您可以选择将数据写入用户指定的子结构数据库。

如果您指定子结构库名称，文件将被命名为 *library_name_Zn*`.sim`、*library_name_Zn*`.prt`、*library_name_Zn*`.mdl` 和 *library_name_Zn*`.stt`。

| **输入文件用法：** | ``` [*SUBSTRUCTURE GENERATE](../key/key-link.md#usb-kws-ssubgenerate), TYPE=Z*n*, LIBRARY=*library_name* ``` |
| --- | --- |

| **Abaqus/CAE 用法：** | Abaqus/CAE 不支持子结构库的定义。 |
| --- | --- |

### 定义保留自由度

节点的自由度可以分为保留自由度（用于子结构的使用级别）和消除自由度（子结构内部）。Abaqus/Standard 允许保留任何节点上的任何自由度，但有一个例外：如果生成声学-结构子结构（基于耦合或非耦合模态），则只能保留结构自由度。您必须确保保留自由度的选择是合理的，以便子结构可以正确连接到模型的其余部分。

任何在使用子结构时可能需要重新指定运动学约束的自由度都应保留为保留自由度。

如果用于定义分布耦合元素的节点有任何自由度被保留，则与 Lagrange 乘子关联的内部节点的自由度会自动添加到子结构的保留自由度列表中。

要定义保留自由度，请指定节点编号或节点集标签，并可选地指定要保留的第一个和最后一个自由度。

默认情况下，与保留自由度关联的节点将按升序排序。

| **输入文件用法：** | ``` [*RETAINED NODAL DOFS](../key/key-link.md#usb-kws-sretainnodaldofs) ``` |
| --- | --- |

| **Abaqus/CAE 用法：** | 载荷模块：边界条件编辑器：**类别**：**机械**：**所选步骤的类型**：**保留节点自由度** |
| --- | --- |

#### 当子结构用于使用级别几何非线性分析时保留自由度

当子结构用于几何非线性分析时，建议保留特定节点的所有平移和/或所有旋转自由度。即使在特定节点的单个平移/旋转自由度被认为在使用级别所需的情况下，您也应该保留与该节点关联的所有平移/旋转自由度。否则，当子结构在几何非线性分析期间旋转时，可能会出现局部数值不稳定（负特征值），因为旋转的子结构可能在特定自由度上没有刚度。

您必须选择适当数量的节点，以允许计算子结构的等效刚体运动。在二维或轴对称分析中，保留两个具有所有平移自由度的节点或一个具有所有平移和旋转自由度的节点足以计算使用级别子结构的等效刚体运动。在三维分析中，需要三个不共线的具有所有保留平移自由度的节点，或一个具有所有平移和旋转的节点。如果保留节点共线或少于三个节点保留，您必须保留至少一个具有所有旋转自由度的节点。当 Abaqus/Standard 在使用级别分析期间无法计算子结构的等效刚体运动时（因为保留自由度的数量不合适），会发出警告消息，并且忽略与子结构相关的任何几何非线性效应。

#### 定义运动学约束

运动学约束按照["运动学约束：概述，" 第 35.1.1 节"](pt08ch35s01abo32.md)中的描述定义。适用以下规则：
- 与未保留自由度关联的所有运动学边界条件必须在生成子结构时指定。这些条件内置于子结构中，并在任何时候使用时都保持施加。一旦生成了子结构，就不能重新指定内部变量上的运动学约束；只能通过擦除和在库中重新创建子结构来修改或移除。施加到内部自由度的规定边界条件的幅度可以与子结构载荷情况关联，并可以在使用级别更改。约束本身内置于子结构中，不能通过忽略对载荷情况的引用来移除。
- 在子结构生成期间，必须避免某些子结构保留自由度有利于内部自由度消除的多点约束。如果希望保留被多点约束消除的某些自由度，则必须将出现在多点约束中的所有变量重新分配为保留自由度，并在使用级别施加约束。

### 定义广义自由度

建模子结构动态行为的一种有效技术是通过包括与动态模态相关的广义自由度来增强子结构内部的响应。您可以选择要保留的模态，这些模态必须在先前的特征频率提取步骤中计算（["自然频率提取，" 第 6.3.5 节"](pt03ch06s03at10.md)）。对于某些子结构生成情况，必须完全恢复动态模态；如果它们是用 AMS 特征求解器计算的且仅部分恢复，则会在此类情况下发出错误消息。例如，如果子结构包括子结构载荷情况或结构-声学耦合（或将用于柔性体生成），则必须完全恢复特征模态。模态将包括特征模态，如果它们在特征频率提取步骤中被激活，还包括残余模态。如果在特征频率提取步骤中约束了子结构的所有保留自由度，这种技术通常称为 Craig-Bampton 方法。如果在特征频率提取步骤中未约束子结构的所有保留自由度，这种技术通常称为 Craig-Chang 方法。Craig-Bampton 方法中的子结构动态模态通常称为固定界面模态，Craig-Chang 方法中的子结构动态模态通常称为自由界面模态。如果在特征频率提取步骤中约束了子结构的部分保留自由度而未约束其他保留自由度，则动态模态称为混合界面模态。如果选择了自由界面或混合界面动态模态，与使用相同数量的固定界面动态模态相比，子结构生成时间可能大幅增加。Abaqus 在这种情况下会发出警告消息。然而，有时可以使用显著更少数量的自由界面或混合界面动态模态来获得比使用固定界面模态更好的解精度。

应选择足够数量的动态模态以提供子结构充分的动态表示。您应该检查载荷频率和结构的频率内容以确定此范围。在特征频率提取步骤定义中指定移位点和/或截止频率，以仅获取所需频率范围内的模态。添加广义自由度会增加子结构生成步骤中频率提取的成本，但如果子结构用于后续动力学（["使用直接积分的隐式动力学分析，" 第 6.3.2 节"](pt03ch06s03at07.md)）、稳态动力学（["直接求解稳态动力学分析，" 第 6.3.4 节"](pt03ch06s03at09.md)）或频率提取（["自然频率提取，" 第 6.3.5 节"](pt03ch06s03at10.md)）分析，会大大改善解的准确性。

在特征频率提取分析中特征向量的位移归一化的情况下，子结构必须至少有一个在使用级别活跃的物理自由度；否则，模态不能被正确归一化。请参阅["子结构化和子结构分析，" Abaqus 理论指南第 2.14.1 节](../stm/stm-link.md#stm-anl-superelements)，了解更多详细信息。

当生成声学-结构子结构时，必须选择保留特征模态。

声学-结构耦合效应可以在自然频率提取过程期间包括在保留特征模态中。要计算耦合结构-声学特征模态，请在自然频率提取过程期间使用默认 Lanczos 特征求解器并包括声学-结构耦合效应（["自然频率提取，" 第 6.3.5 节"](pt03ch06s03at10.md)）。

Abaqus 还可以使用非耦合特征模态（从基于 SIM 的 Lanczos 或 AMS 特征求解器生成）来生成耦合声学-结构子结构。在这种情况下，声学-结构耦合效应在子结构生成期间包括。结构特征模态和声学特征模态都必须为子结构生成保留，并且需要选择声学零频率模态（如果存在此类模态）以获得准确的子结构。

### 预加载子结构

子结构可用于表现出非线性响应（与标准 Abaqus 单元或接触定义相关）的模型，但子结构内部的响应假定为线性小变形。然而，子结构的响应可能是在其预加载历史期间基于子结构内非线性响应定义的预变形（可能旋转和平移）基态上的线性扰动。

当子结构用于几何非线性分析时，子结构预加载应限于仅产生自平衡应力的载荷（如热应力或过盈配合）。在大多数情况下，预加载应力不是自平衡的（如来自规定边界条件或施加载荷的应力）。如果存在非自平衡预应力且子结构在使用级别经历刚体运动，则会在子结构中产生额外应力。此类使用级别应力是非物理的，将导致收敛问题并产生难以解释的结果。因此，在预加载用于几何非线性分析的子结构时，应格外小心。

此预加载概念允许将应力刚化等效应包括在子结构中。预加载是子结构状态的一部分：预加载是自平衡的，因此在子结构使用时不会产生载荷向量。在模型中使用子结构期间的任何载荷加载是预加载的附加项。

区分预加载和载荷情况之间的区别很重要。两者都在子结构生成分析期间允许，但只有预加载在实际生成期间应用于子结构。载荷情况（在子结构生成期间定义）只能在使用级别应用（请参阅["使用子结构"中的"向子结构施加载荷，" 第 10.1.1 节"](pt04ch10s01aus58.md#usb-anl-asuperelements-loads)）。载荷情况将在后面更详细地讨论。

#### 计算变量总响应

子结构内部任何恢复的响应变量（如应力或位移）被定义为相对于预加载基态的扰动（对于几何非线性分析有一些例外）。对于几何非线性分析，位移输出包括与子结构相关的等效刚体旋转和平移以及应变诱导的小位移扰动。如果需要变量的总响应，可以通过将扰动结果与子结构预加载期间计算的最终结果相加来计算。

计算预加载子结构刚度矩阵的规则与静态线性扰动步骤的规则相同。请参阅["通用和线性扰动过程，" 第 6.1.3 节"](pt03ch06s01aus44.md)，了解规则详细描述。

#### 在预加载步骤期间在保留自由度上规定边界条件

在子结构预加载期间，可以在保留自由度上规定边界条件。当随后在子结构生成步骤中创建预加载子结构时，您必须释放所有保留自由度（请参阅["Abaqus/CFD 中的边界条件"中的"移除边界条件，" 第 34.3.2 节"](pt07ch34s03aus119.md#usb-prc-pboundary-remove)）。如果某些保留自由度未释放，将发出错误消息。释放自由度处的反作用力成为与子结构内应力平衡的集中载荷。这些集中载荷不能在不更改预加载的情况下移除。

### 定义用于后续分析加载的子结构载荷情况

在子结构生成期间定义的载荷情况在使用级别被激活，相当于 Abaqus 中常规单元可用的元素载荷类型。它们可以由任何组合的载荷（分布式载荷、集中节点载荷、热膨胀以及为可能用作此子结构定义一部分的任何子结构定义的载荷情况）组成。

需要载荷情况，以便当子结构随后在模型中使用时，只需按特定载荷的适当幅度缩放保留自由度上的一致的载荷：无需进入子结构并重复基本单元计算来分布载荷。

每个这样的载荷情况可以通过将其与幅值/时间曲线和幅度关联在使用子结构时应用（["幅值曲线，" 第 34.1.2 节"](pt07ch34s01aus115.md)）。当使用子结构时，只能使用在生成子结构时创建的子结构载荷情况载荷。除了重力载荷，使用子结构时，不能向组成任何子结构的单元施加分布式载荷、温度载荷等。这些载荷必须在创建期间内置于子结构中。

您可以在子结构生成期间定义多个子结构载荷情况，以定义子结构的不同载荷。每个载荷情况被分配一个名称，该名称将在使用级别应用载荷情况时使用。

您可以使用集中载荷、分布式载荷、子结构载荷和温度场（["集中载荷，" 第 34.4.2 节"](pt07ch34s04aus121.md)；和["分布式载荷，" 第 34.4.3 节"](pt07ch34s04aus122.md)）的任何组合来定义每个载荷情况。

您为每个基本载荷分配一个参考幅度，然后在使用子结构载荷应用时将按实际幅度缩放。必须为每个基本载荷定义的参考幅度是载荷或边界条件相对于基态的变化，而不是基态加扰动值的总和。子结构生成内应用的初始条件不包括为载荷情况定义的一部分。

对于温度载荷，子结构载荷情况的载荷向量将仅包含热膨胀的贡献。如果存在温度相关属性，则在预加载状态中指定的温度下评估它们。因此，要考虑规定为初始条件（["Abaqus/Standard 和 Abaqus/Explicit 中的初始条件，" 第 34.2.1 节"](pt07ch34s02aus116.md)）的非零初始温度场，必须在创建子结构之前预加载结构。当在子结构载荷情况中使用温度载荷时，不能从结果文件读取数据。规定的温度必须定义为相对于基态的温度变化。

当在子结构生成过程中指定了保留模态的子结构生成过程期间，载荷情况定义包括声学载荷时，Abaqus/Standard 当前有一个限制：奇异（恒定压力）声学模态（["声学、冲击和耦合声学-结构分析，" 第 6.10.1 节"](pt03ch06s10at29.md)）的贡献在生成载荷情况中不被考虑。由于该模态的贡献对于低频响应是显著的，因此在这些情况下生成的载荷情况将无法充分表示指定的声学载荷。如果耦合声学-结构子结构中不存在奇异声学区域，则声学载荷被准确地表示。

区分载荷情况和预加载之间的区别很重要。两者都在子结构生成期间定义，但只有预加载在实际生成级别应用于子结构；在生成级别定义的载荷情况只能在使用级别应用，并且它们作用于预加载基态（如果已指定）。（预加载已在前面讨论过。）

在一般分析步骤和扰动步骤中，子结构载荷的处理方式与其他载荷（如集中载荷和分布式载荷（["集中载荷，" 第 34.4.2 节"](pt07ch34s04aus121.md)和["分布式载荷，" 第 34.4.3 节"](pt07ch34s04aus122.md)）相同。例如，如果一般分析步骤之后是另一个一般分析步骤，则子结构载荷将在第二个步骤中保留，其幅度等于前一个一般分析步骤结束时的幅度，除非子结构载荷被修改或移除。在线性扰动步骤中，子结构载荷表示增量载荷。

如果子结构载荷用于在直接求解稳态动力学分析中施加 Coriolis 载荷，则不考虑非对称载荷刚度贡献。

#### 定义边界条件

要内置于子结构矩阵的所有边界条件必须使用边界条件定义指定。这些不能是子结构载荷情况规范的一部分。一旦在特定节点自由度上指定了运动学边界条件，它就被内置于子结构矩阵中，对所有载荷情况有效，不能移除（或在使用级别重新定义）。作为预加载历史一部分指定的边界条件被内置于子结构矩阵中。

如果对约束是否永久存在有任何疑问，最好将自由度设为保留自由度，而不是在子结构定义中指定任何约束。然后可以根据需要在每个分析步骤中按需包括约束。

#### 子结构用于几何非线性分析时的载荷情况

在生成级别包括在使用级别作为子结构载荷应用的子结构载荷情况中的所有载荷和边界条件都应用于与子结构关联的局部系统。由于当存在大运动时此系统随子结构旋转，这些载荷和边界条件也将旋转。因此，在几何非线性分析中使用子结构载荷情况时要小心，以确保载荷在使用级别处于适当方向。这种情况类似于使用子结构属性定义旋转子结构。

#### 重力载荷

要施加重力载荷，必须为至少包含在子结构中的某些单元定义密度。可以用两种不同的解释以两种不同的方式向子结构施加重力载荷。如果在子结构生成期间将分布式载荷定义用作子结构载荷情况的一部分（如上所述["定义用于后续分析加载的子结构载荷情况"](pt04ch10s01aus59.md#usb-anl-asuperelementdef-loads)"中所述），则重力载荷成为子结构载荷情况的一部分，因此在使

用期间旋转以跟随子结构的局部系统（局部系统可能通过子结构属性定义旋转子结构或由于几何非线性响应而旋转）。

要定义在使用级别以固定全局方向作用的重力载荷，您可以请求在子结构生成期间计算子结构的重力载荷向量。在这种情况下，重力载荷不应作为子结构载荷情况的一部分定义。当计算重力载荷向量时，Abaqus/Standard 为每个全局方向生成重力载荷向量（三维分析为三个，二维/轴对称分析为两个）。在使用级别，可以使用分布式载荷定义（请参阅["使用子结构"中的"重力载荷，" 第 10.1.1 节"](pt04ch10s01aus58.md#usb-anl-asuperelements-loads-grav)）来指定在指定幅度下以固定全局方向作用于子结构的重力载荷。

| **输入文件用法：** | ``` [*SUBSTRUCTURE MATRIX OUTPUT](../key/key-link.md#usb-kws-ssubmatrixoutput), OUTPUT FILE=RESULTS FILE ``` |
| --- | --- |

| **Abaqus/CAE 用法：** | 当您使用 **子结构生成** 步骤运行分析时，Abaqus/CAE 自动将请求的矩阵写入 Abaqus 结果文件。 |
| --- | --- |

#### 将记录写入用户定义的文件

您可以指定要写入数据的文件名（不带扩展名）。记录的写入格式与线性用户定义单元兼容。记录格式在["用户定义单元，" 第 32.15.1 节"](pt06ch32s15alm60.md)中描述。将向指定的文件名添加 `.mtx` 扩展名。

| **输入文件用法：** | ``` [*SUBSTRUCTURE MATRIX OUTPUT](../key/key-link.md#usb-kws-ssubmatrixoutput), OUTPUT FILE=USER DEFINED, FILE NAME=*file_name* ``` |
| --- | --- |

| **Abaqus/CAE 用法：** | 作业模块：作业编辑器：**名称** |

### 管理库内的子结构

子结构存储在库集合中。提供管家功能以帮助维护大量库；例如，可以从库中删除子结构或将子结构移动到不同的库。

一旦生成了子结构库，可以将磁盘文件设为只读以保护库免受意外删除或修改。子结构库在子结构生成期间以及使用子结构管家功能添加或删除子结构时必须是可写的。

当使用多个分析来生成子结构库时，这些分析必须按顺序运行；它们不能同时运行。Abaqus 可能无法提供任何指示，表明正在写入的子结构库可能已被另一个 Abaqus 分析使用。如果多个分析同时写入同一库，库可能会损坏。如果发生这种情况并且库在后续分析中使用，结果可能是大量预处理器内存需求。

| **输入文件用法：** | 使用以下任何选项（详细描述如下）对子结构库执行管家功能： |
| --- | --- |
|  | ``` [*SUBSTRUCTURE COPY](../key/key-link.md#usb-kws-ssubcopy) [*SUBSTRUCTURE DELETE](../key/key-link.md#usb-kws-ssubdelete) [*SUBSTRUCTURE DIRECTORY](../key/key-link.md#usb-kws-ssubdirectory) ``` 管家选项可以出现在输入文件的模型部分中的任何位置（["在 Abaqus 中定义模型，" 第 1.3.1 节"](pt01ch01s03aus03.md)）。输入文件可以仅由 [*HEADING](../key/key-link.md#usb-kws-mheading) 选项和一个或多个管家选项组成。在这种情况下，管家选项引用的文件和子结构必须在分析开始时存在。 |

| **Abaqus/CAE 用法：** | Abaqus/CAE 不支持子结构库。 |
| --- | --- |

#### 列出存储在子结构库中的子结构

您可以获得存储在子结构库中的子结构的信息摘要。如有必要，您可以为库指定非默认名称（默认名称是 *jobname*）。

| **输入文件用法：** | ``` [*SUBSTRUCTURE DIRECTORY](../key/key-link.md#usb-kws-ssubdirectory), LIBRARY=*substructure_library_name* ``` |
| --- | --- |

| **Abaqus/CAE 用法：** | Abaqus/CAE 不支持子结构库。 |
| --- | --- |

#### 从子结构库中移除子结构

您可以从子结构库中移除指定的子结构。如有必要，您可以指定库的名称。

| **输入文件用法：** | ``` [*SUBSTRUCTURE DELETE](../key/key-link.md#usb-kws-ssubdelete), TYPE=Z*n*, LIBRARY=*substructure_library_name* ``` |
| --- | --- |

| **Abaqus/CAE 用法：** | Abaqus/CAE 不支持子结构库。 |
| --- | --- |

#### 复制或移动子结构定义

您可以将子结构定义从一个库复制到另一个库，或在同一库中从一子结构复制到另一子结构。您必须标识要复制的子结构，并为正在创建的子结构分配名称。

将子结构从库复制到库时，您可以标识包含要复制的子结构的库的名称。同样，您可以标识要将子结构复制到的新库的名称。此新库在复制子结构之前不需要存在；在这种情况下将被创建。

如果原始子结构要被删除，您可以在复制后跟随删除（见上文）。

| **输入文件用法：** | ``` [*SUBSTRUCTURE COPY](../key/key-link.md#usb-kws-ssubcopy), OLD TYPE=Z*n*, NEW TYPE=Z*n*, OLD LIBRARY=*substructure_library_name*, NEW LIBRARY=*substructure_library_name* ``` |
| --- | --- |

| **Abaqus/CAE 用法：** | Abaqus/CAE 不支持子结构库。 |
| --- | --- |

#### 重命名子结构库

一旦生成了子结构库，不应手动重命名磁盘文件。要重命名子结构库，请将现有子结构复制到新库。新库在复制第一个子结构之前不需要存在。然后，如果不再需要原始磁盘文件，您可以手动删除它。
