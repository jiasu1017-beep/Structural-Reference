# *RESPONSE SPECTRUM







### *RESPONSE SPECTRUM基于用户提供的响应谱计算响应。

此选项用于基于用户提供的响应谱（使用[*SPECTRUM](ch18abk28.md)选项定义）和系统的固有模态，计算节点和单元响应的峰值估计。

**产品：**Abaqus/Standard  Abaqus/CAE  

**类型：**历史数据  

**级别：**步骤  

**Abaqus/CAE：**步骤模块

##### **参考：**

- ["响应谱分析，" Abaqus Analysis User's Guide第6.3.10节](../usb/usb-link.md#usb-anl-aresponsespectrum)
- [*SPECTRUM](ch18abk28.md)

### **可选参数：**

COMP

设置COMP=ALGEBRAIC以代数方式求和方向激励分量。如果使用此参数，则首先对方向激励分量求和，然后进行模态求和。

设置COMP=SRSS（默认）以使用平方和的平方根。如果使用此参数，则首先执行模态求和，然后对方向激励分量求和。

设置COMP=R40以使用ASCE 4-98指南推荐的40%规则，该规则假定对于一个分量的最大响应，其他两个分量的响应为其最大值的40%。

设置COMP=R30以使用30%规则。该规则假定对于一个分量的最大响应，其他两个分量的响应为其最大值的30%。它遵循ASCE 4-98指南中描述的40%规则的表达式。

SUM

设置SUM=ABS（默认）以对每个固有模态中的响应绝对值求和。

设置SUM=CQC以使用完全二次组合方法对每个固有模态中的响应求和。

设置SUM=DSC以使用双和组合。这是基于随机振动理论评估模态相关性的首次尝试。它利用强地震运动的持续时间![](../graphics/key_eqn01032.gif)。

设置SUM=GRP以使用USNRC监管指南1.92（1976年2月）中描述的分组方法。

设置SUM=NRL以使用海军研究实验室方法。

设置SUM=SRSS以使用平方和平方根求和。

设置SUM=TENP以使用10%方法。
