# 9.8.4 指定模型属性





您可以指定以下描述模型特征的模型属性：
- 名称。
- 模型类型。
- 模型描述。
- 将模型写入输入文件时是否包含零件和装配。
- 模型的物理常数。
- 如果需要，重新启动信息，将使用先前分析的数据启动分析。有关更多信息，请参阅["重启分析，"第19.6节"](pt03ch19s06.md)和["重启分析，"Abaqus Analysis User's Guide第9.1.1节"](../usb/usb-link.md#usb-anl-arestart)。
- 是否使用将驱动子模型边界条件或载荷的全局模型。您还可以指定全局模型是驱动实体子模型的壳模型。有关更多信息，请参阅[第38章，"子模型"](pt04ch38.md)。

**要指定模型属性：**

1. 从主菜单栏，使用以下方法之一显示**编辑模型属性（Edit Model Attributes）**对话框：- 要在新模型中指定模型属性，请从主菜单栏选择****模型（Model）**![](../graphics/images/arrow.gif)**创建（Create）****。- 要在现有模型中指定模型属性，请从主菜单栏选择****模型（Model）**![](../graphics/images/arrow.gif)**编辑属性（Edit Attributes）**![](../graphics/images/arrow.gif)***模型名称*****。
2. 如果要创建新模型，请选择模型类型：- 选择**Standard和Explicit**（默认）为Abaqus/Standard或Abaqus/Explicit分析创建模型。- 选择**CFD**为Abaqus/CFD分析创建模型。- 选择**电磁（Electromagnetic）**为电磁分析创建模型。您无法更改现有模型的模型类型。
3. 如果需要，输入或修改模型的描述。1. 在**编辑模型属性（Edit Model Attributes）**对话框中单击![](../graphics/ico_edit.png)。出现模型描述编辑器。2. 在模型描述编辑器中，输入您要记录的有关模型的信息。3. 单击**确定（OK）**存储描述并关闭模型描述编辑器。您输入的描述保存在模型数据库中，并在提交模型进行分析时写在输入文件的标题上方；描述不会写入输出数据库。有关更多信息，请参阅["为Abaqus/CAE模型添加描述，"第9.10.2节"](pt02ch09s09hlb02.md)。
4. 如果您希望Abaqus/CAE在写入输入文件时不使用零件和装配，请切换**不在输入文件中使用零件和装配（Do not use parts and assemblies in input files）**。有关此选项的更多信息，请参阅["在输入文件中不使用零件和装配，"第9.10.4节"](pt02ch09s09hlb04.md)。
5. 在对话框的**物理常数（Physical Constants）**部分中，执行以下操作：- 要在热传递分析中指定表面发射率和辐射条件，请输入绝对零温度和斯特藩-玻尔兹曼常数的值。- 要指定通用气体常数，请在**通用气体常数（Universal gas constant）**字段中输入值。- 要识别声学分析中入射波相互作用的人射波加载类型，请切换**指定声波公式（Specify acoustic wave formulation）**，单击文本字段右侧的箭头，然后选择公式。- 选择**散射波（Scattered wave）**以获得将由入射波加载产生的散射波场解。- 选择**总波（Total wave）**以获得总声压波解。
6. 如果需要，单击**重启（Restart）**选项卡以指定将使用先前分析的数据启动分析的重新启动信息。切换**从作业读取数据（Read data from job）**并执行以下操作：- 输入Abaqus/CAE将从中读取重启信息的作业名称。- 输入Abaqus/CAE将从中重启分析的步骤名称。- 选择Abaqus/CAE将从中重启分析的步骤的增量、间隔、迭代或循环。
7. 如果需要，单击**子模型（Submodel）**选项卡并执行以下操作：- 切换**从作业读取数据（Read data from job）**并输入将用于驱动子模型边界条件或载荷的全局解决方案的输出数据库名称。如果输出数据库不可用，您也可以输入结果文件的名称。- 指定子模型是否是将由全局壳模型驱动的实体。有关更多信息，请参阅["创建子模型，"第38.2节"](pt04ch38s02.md)。
8. 单击**确定（OK）**保存您的数据并关闭对话框。
