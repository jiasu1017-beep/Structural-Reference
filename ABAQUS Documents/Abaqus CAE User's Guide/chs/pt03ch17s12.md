# 17.12 网格-几何关联







Abaqus/CAE 自动将自顶向下网格与底层几何体关联。Abaqus/CAE 可以建立这种关联，因为网格完全符合几何体。相反，自底向上网格不必符合几何体，因此 Abaqus/CAE 可能不会将网格的某些部分与几何体关联。类似地，孤立网格实体不与几何体关联，因为它们要么是在独立于几何体的情况下创建的，要么是创建它们的几何体在当前模型中不可用。

在孤立或自底向上网格实体（单元、单元面、单元边缘和节点）与相邻几何体之间创建关联允许将载荷、相互作用和边界条件从几何体传递到网格。如果您将孤立或自底向上网格实体与相邻几何面完全关联，您可以使用该面为与您开始使用的孤立或自底向上网格兼容的几何区域创建本地网格。完全关联意味着：
- 所选几何面与覆盖整个面的单元面关联。
- 几何面的所有边缘与跨越整个边缘的单元边缘关联。
- 面的所有顶点与节点关联。

图 17-122 显示了二维孤立单元区域与相邻几何区域之间网格-几何关联使用的示例。将两个区域之间的几何边缘——左图中的黄线——与孤立单元边缘和节点关联会产生兼容的混合网格。

相反，如果模型有一个您需要保留的网格——因为它包含大量编辑或非常适合分析——而您不想为整个零件创建孤立网格，您可以删除网格与所选实体之间的关联，以在所选区域中创建孤立节点和单元。删除网格关联性可防止 Abaqus/CAE 在您编辑几何体时删除区域的网格。完成编辑后，您可以在几何面和网格的表面实体之间重新建立关联。

**注意：**如果您删除实体区域的网格-几何关联性，重新建立本地网格的唯一方法是使用 **Edit Mesh** 对话框中的 **Undo** 功能，或为区域分配自底向上网格划分技术并重新创建关联性。

当您处理实体几何区域时，以下规则适用于自底向上单元的关联：
- Abaqus/CAE 始终将自底向上单元与所选区域关联。
- 当底层几何体用于定义网格形状时，Abaqus/CAE 将网格与该几何体关联。
- 当几何体的部分未用于定义自底向上网格时，Abaqus/CAE 不会将它们关联，即使网格和几何体位于同一位置。
- 您可以编辑任何基于几何的自底向上网格区域的网格-几何关联。
- 即使您编辑生成的网格以使其与几何体匹配，您仍必须手动将网格与几何体关联。

以下三个注意事项使网格-几何关联性对于创建良好的分析模型至关重要：

1. 当您使用几何体工作时，载荷和边界条件等属性应用于几何体。正确的网格-几何关联性确保这些属性在分析期间正确传递到网格。
2. 如果您选择几何面作为自底向上网格的源或连接侧，Abaqus 仅在该面网格实体与所选面完全关联时才会重用该面上现有的网格实体来创建兼容网格。
3. Abaqus 会尝试合并与相同几何实体关联的网格。未关联的网格可能需要您使用编辑网格工具集沿网格边界合并节点。

在使用自底向上网格或编辑任何网格的关联性时，您应始终检查网格-几何关联性是否正确。如果您提交作业且属性应用于没有关联网格的几何体，Abaqus/CAE 会在 Job 模块中发出错误。但是，Abaqus/CAE 无法确定关联是否正确。例如，如果载荷应用于应该具有数百个单元的几何面，但只有一个单元与该面关联，Abaqus/CAE 将尝试使用整个载荷应用于单个关联单元来分析模型。错误的关联会产生错误的分析结果。

您可以使用网格模块工具箱中的网格-几何关联 ![](../graphics/ico_meshBottomUpAssociate.png) 和删除网格关联性 ![](../graphics/icoR_meshDeleteAssociativity.png) 工具来查看和编辑或删除网格-几何关联。 有关详细说明，请参见"查看和编辑网格-几何关联性"第 64.7.11 节和"删除网格-几何关联性"第 64.7.12 节。
