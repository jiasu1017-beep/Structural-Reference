# 15.13.23 定义集中薄膜条件相互作用







您可以通过创建集中薄膜条件相互作用来模拟装配件中由于对流引起的一个或多个点的热传递。从主菜单栏中选择****相互作用**![](../graphics/images/arrow.gif)**创建****，然后选择一个或多个节点或顶点或保存的节点或顶点集合。有关薄膜条件的简要概述，请参见"理解相互作用"，第15.3节。有关更详细的讨论，请参见Abaqus Analysis User's Guide的第34.4.4节"热载荷"。

**定义集中薄膜条件相互作用：**

1. 从主菜单栏中，选择****相互作用**![](../graphics/images/arrow.gif)**创建****。**提示：**您也可以使用相互作用模块工具箱中的![](../graphics/ico_interaction.png)工具创建集中薄膜条件相互作用。
2. 在出现的**创建相互作用**对话框中，执行以下操作：- 命名该相互作用。有关命名对象的信息，请参见第3.2.1节"使用基本对话框组件"。- 选择步骤。您只能在热传递、耦合温度-位移或耦合热-电步骤期间定义节点区域的对流。- 选择**集中薄膜条件**类型的相互作用。
3. 单击**继续**关闭**创建相互作用**对话框。
4. 使用以下方法之一选择点：- 使用节点或顶点的现有集合定义区域。在提示区域右侧单击**集合**。从出现的**区域选择**对话框中选择现有集合，然后单击**继续**。**注意：**默认选择方法基于您最近使用的选择方法。要恢复为另一种方法，请单击提示区域右侧的**在视口中选择**或**集合**。- 使用鼠标在视口中选择节点或顶点。（更多信息，请参见"在当前视口中选择对象"，第6.2节。）如果模型包含网格和几何的组合，请从提示区域单击以下之一：- 如果要从几何区域选择顶点，请单击**几何**。- 如果要从本机或孤立网格选择节点，请单击**网格**。您可以使用角度方法从网格中选择一组节点。更多信息，请参见"使用角度和特征边缘方法选择多个对象"，第6.2.3节。
5. 在出现的**编辑相互作用**对话框中，单击**定义**字段右侧的箭头，然后从出现的列表中选择一个选项：- 选择**嵌入系数**以在此对话框中指定薄膜系数。- 选择**属性参考**以使用薄膜条件相互作用属性将薄膜系数定义为温度和场变量的函数。- 选择**用户定义**以在用户子程序[`FILM`](../sub/sub-link.md#sub-xsl-film)中定义非均匀薄膜系数。（此选项仅在Abaqus/Standard分析中有效。）请参见以下章节获取更多信息：- 第19.8.6节"指定常规作业设置"- Abaqus User Subroutines Reference Guide的第1.1.6节"FILM"- 选择解析场以定义空间变化的薄膜系数。解析场不影响汇温度。只有对该相互作用类型有效的解析场才显示在选择列表中。或者，您可以单击![](../graphics/ico_analyticalField.png)创建新解析场。（请参见第58章"解析场工具集"获取更多信息。）
6. （如果需要）指定集中薄膜条件如何应用于自适应网格域的边界。此选项仅对Abaqus/Explicit分析有效。单击**自适应网格边界类型**字段右侧的箭头，然后从出现的列表中选择一个选项。更多信息，请参见Abaqus Analysis User's Guide的第12.2.2节"在Abaqus/Explicit中定义ALE自适应网格域"。- 选择**拉格朗日**以将集中薄膜应用于跟随材料的节点（非自适应）。- 选择**滑动**以将集中薄膜应用于可以滑动越过材料的节点。网格约束通常应用于节点以将其固定在空间上。- 选择**欧拉**以将集中薄膜应用于可以独立于材料移动的节点。此选项仅用于材料可以流入或流出自适应网格域的边界区域。必须对欧拉边界区域应用垂直于区域的网格约束，以允许材料流过该区域。如果未应用网格约束，则欧拉边界区域的行为将与滑动边界区域相同。
7. 在**关联节点面积**字段中，输入应用集中薄膜条件的节点的关联面积。
8. 如果您选择了**嵌入系数**或解析场定义选项，请执行以下步骤：1. 在**薄膜系数**字段中，输入薄膜系数*h*。2. 如果要使薄膜系数随时间变化，请单击**薄膜系数幅值**字段右侧的箭头，然后从出现的列表中选择幅值。如果需要，请单击![](../graphics/ico_amplitudeSmall.png)创建新幅值；请参见第57.3节"选择要定义的幅值类型"获取更多信息。3. 在**汇温度**字段中，输入汇温度，![](../graphics/usi_eqn00275.gif]。4. 如果要定义空间变化的汇温度，请单击**汇定义**字段右侧的箭头，然后选择标记为（A）的解析场或标记为（D）的离散场。只有对温度有效的解析场和离散场才在选择列表中可用。请参见第58章"解析场工具集"和第63章"离散场工具集"获取更多信息。或者，您可以单击![](../graphics/ico_discreteField.png)创建新离散场。5. 如果要使汇温度随时间变化，请单击**汇幅值**字段右侧的箭头，然后从出现的列表中选择幅值。如果需要，请单击![](../graphics/ico_amplitudeSmall.png)创建新幅值；请参见第57.3节"选择要定义的幅值类型"获取更多信息。
9. 如果您选择了**属性参考**定义选项，请执行以下步骤：1. 选择薄膜相互作用属性。如果需要，请单击![](../graphics/ico_interactionProperty.png)创建相互作用属性；有关更多信息，请参见第15.14.2节"定义薄膜条件相互作用属性"。2. 在**汇温度**字段中，输入汇温度，![](../graphics/usi_eqn00275.gif]。3. 如果要定义空间变化的汇温度，请单击**汇定义**字段右侧的箭头，然后选择标记为（A）的解析场或标记为（D）的离散场。只有对温度有效的解析场和离散场才在选择列表中可用。请参见第58章"解析场工具集"和第63章"离散场工具集"获取更多信息。或者，您可以单击![](../graphics/ico_discreteField.png)创建新离散场。4. 如果要使汇温度随时间变化，请单击**汇幅值**字段右侧的箭头，然后从出现的列表中选择幅值。如果需要，请单击![](../graphics/ico_amplitudeSmall.png)创建新幅值；请参见第57.3节"选择要定义的幅值类型"获取更多信息。
10. 如果您选择了**用户定义**定义选项，请执行以下步骤：1. 在**薄膜系数**字段中，输入薄膜系数*h*。2. 在**汇温度**字段中，输入汇温度，![](../graphics/usi_eqn00275.gif]。3. 进入作业模块，并显示感兴趣的分析作业的作业编辑器。（更多信息，请参见第19.7节"创建、编辑和操作作业"。）4. 在作业编辑器中，单击**常规**选项卡，然后指定包含用户子程序[`FILM`](../sub/sub-link.md#sub-xsl-film)的文件。更多信息，请参见第19.8.6节"指定常规作业设置"。**注意：**您只能在作业编辑器中指定一个用户子程序文件；如果您的分析涉及多个用户子程序，则必须将用户子程序组合到一个文件中，然后指定该文件。
11. 单击**确定**创建相互作用并关闭编辑器。
