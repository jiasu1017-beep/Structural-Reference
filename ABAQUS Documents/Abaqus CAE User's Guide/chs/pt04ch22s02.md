# 22.2 创建和编辑螺栓载荷





从主菜单栏选择 ****Load**![](../graphics/images/arrow.gif)**Create**** 可以对螺栓或紧固件中的紧固力或长度调整进行建模。

您可以通过使用Step模块中的场和历史输出请求编辑器从螺栓载荷获取数据。 在编辑器的**Domain**部分中，选择**Bolt load**并从出现的菜单中选择所需的螺栓载荷。 有关更多信息，请参阅["创建输出请求，" 第14.12.1节](pt03ch14s12hlb01.md)。

**定义螺栓载荷：**

1. 如果您使用原生或导入几何，请创建指示螺栓载荷所需位置的划分。 有关更多信息，请参阅[第70章，"The Partition toolset"](pt06ch70.md)。
2. 如果您使用实体零件实例，请创建指示螺栓轴线所需方向的基准轴。 您也可以创建基准坐标系统并使用其一个轴来指示螺栓轴线的所需方向。 （有关更多信息，请参阅["创建基准轴，" 第62.7节](pt06ch62s07.md)。
3. 从主菜单，选择 ****Load**![](../graphics/images/arrow.gif)**Create****。 Abaqus/CAE显示**Create Load**对话框。
4. 在**Create Load**对话框中，执行以下操作： 1. 从**Category**列表中，选择**Mechanical**。 2. 从**Types for Selected Step**列表中，选择**Bolt Load**，然后单击**Continue**。
5. 选择指示螺栓载荷位置的内部表面或线段。 - 如果您使用原生或导入几何，请使用鼠标在视口中选择内部表面或线段。 您可以使用拖动选择、**[Shift]****+Click**、**[Ctrl]****+Click**和角度方法的组合来选择多个面或边。 有关更多信息，请参阅["在当前视口中选择对象，" 第6.2节](pt01ch06s02.md)。 **提示：**如果您无法选择所需的面或边，可以使用**Selection**工具栏更改选择行为。 有关更多信息，请参阅["使用选择选项，" 第6.3节](pt01ch06s03.md)。 - 如果您使用孤儿网格单元，必须选择单元面以指定内部表面。 您可以使用显示组从视口中移除所选单元以显示横截面表面的单元面。 有关更多信息，请参阅[第78章，"使用显示组显示模型的子集"](pt07ch78.md)。 完成选择后，单击鼠标按钮2。
6. 使用["指定区域的特定侧面或端部，" 第73.2.5节](pt06ch73s02s05.md)中所述的技术选择表面定义侧的哪一侧。 您选择的侧面决定了哪些单元被调整以产生所需的紧固载荷或长度调整（请参阅["预定义的装配载荷，" Abaqus Analysis User's Guide第34.5.1节](../usb/usb-link.md#usb-prc-ppretension)）。 如果螺栓用线零件实例建模，当您完成侧面选择时，Abaqus/CAE会显示螺栓载荷编辑器。 如果螺栓用实体零件实例建模，系统会提示您选择一个基准轴。
7. 如果螺栓用实体零件实例建模，请选择指示螺栓轴线所需方向的基准轴。 您也可以选择基准坐标系统的轴之一。 Abaqus/CAE显示螺栓载荷编辑器。
8. 单击**Method**字段旁边的箭头，并从出现的列表中选择您想要的加载方法。
9. 在**Magnitude**字段中，输入力的大小（对于**Apply force**方法）或长度变化（对于**Adjust length**方法）。 **注意：**如果您在创建载荷的步骤之后的步骤中编辑载荷，**Fix at current length**方法变为可用。 如果在编辑载荷时将方法更改为**Fix at current length**，则**Magnitude**字段将变为不可用。
10. 如果需要，指定振幅。 （有关更多信息，请参阅[第57章，"The Amplitude toolset"](pt06ch57.md)。）
11. 如果您在实体零件实例上创建螺栓载荷，或者如果您在第一步分析中编辑实体零件实例上的螺栓载荷，编辑器底部会出现一个**Edit axis**按钮。 如果您想更改基准轴选择，请单击**Edit axis**。
12. 单击**OK**创建载荷并关闭**Create Bolt Load**对话框。 视口中会出现代表您刚刚创建的螺栓载荷的箭头。 有关更多信息，请参阅["理解代表预定义条件的符号，" 第16.5节](pt03ch16s05.md)。
