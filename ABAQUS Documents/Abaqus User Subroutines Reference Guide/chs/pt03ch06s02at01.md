### 概述

静态应力分析：
- 用于可以忽略惯性效应的情况；
- 可以是线性的或非线性的；以及
- 忽略随时间变化的材料效应（蠕变、膨胀、粘弹性），但考虑率相关塑性和超弹性材料的滞后行为。

### 时间周期

在静态步骤期间，您可以为分析分配时间周期。这对于幅值选项的交叉引用是必要的，幅值选项可用于确定步骤期间载荷和其他外部规定参数的变化（请参见["幅值曲线，" 第34.1.2节"](pt07ch34s01aus115.md)）。在某些情况下，此时间尺度是相当真实的——例如，响应可能由基于先前瞬态热传递运行随时间变化的温度引起；或者材料响应可能是率相关的（率相关塑性），因此存在自然的时间尺度。其他情况没有这样的自然时间尺度；例如，当容器在率无关材料响应下被加压至极限载荷时。如果您不指定时间周期，Abaqus/Standard默认为"时间"在步骤中从0.0到1.0变化的时间周期。然后，"时间"增量就是步骤总周期的简单分数。

### 线性静态分析

线性静态分析涉及载荷工况的规范和适当的边界条件。如果问题的全部或部分是线性响应，子结构是降低大型分析计算成本的有效方法（请参见["使用子结构，" 第10.1.1节"](pt04ch10s01aus58.md)）。

### 非线性静态分析

非线性可能来自大位移效应、材料非线性和/或边界非线性（如接触和摩擦）（请参见["一般和线性扰动过程，" 第6.1.3节"](pt03ch06s01aus44.md)），必须考虑。如果预期步骤中会出现几何非线性行为，应使用大位移公式。在大多数非线性分析中，步骤上的载荷变化遵循规定的历史，如温度瞬态或规定位移。

| **输入文件用法：** | 使用以下选项指定应为静态步骤使用大位移公式： |
| --- | --- |
|  | ``` [*STEP](../key/key-link.md#usb-kws-hstep), NLGEOM ``` |

| **Abaqus/CAE用法：** | 步骤模块：**创建步骤**：**一般**：**静态，一般**：**基本**：**Nlgeom**：**打开**（激活大位移公式） |

#### 不稳定问题

一些静态问题可能由于各种原因而自然不稳定。

##### 屈曲或坍塌

在某些几何非线性分析中，可能发生屈曲或坍塌。在这些情况下，只有当载荷幅度不遵循规定历史时，才能获得准静态解；它必须是解的一部分。当载荷可以被认为是成比例的（整个结构上的载荷可以用单个参数缩放）时，可以使用一种称为"改进Riks方法"的特殊方法，如["不稳定坍塌和后屈曲分析，" 第6.2.4节"](pt03ch06s02at03.md)中所述。

| **输入文件用法：** | ``` [*STATIC](../key/key-link.md#usb-kws-hstatic), RIKS ``` |
| --- | --- |

| **Abaqus/CAE用法：** | 步骤模块：**创建步骤**：**一般**：**静态，Riks** |
| --- | --- |

##### 局部不稳定

在其他不稳定分析中，不稳定是局部的（例如，表面起皱、材料不稳定或局部屈曲），在这种情况下，全局载荷控制方法（如Riks方法）不合适。Abaqus/Standard提供通过在整个模型中施加阻尼来稳定此类问题的选项，引入的粘性力足够大以防止即时屈曲或坍塌，但又足够小以在问题稳定时不会显著影响行为。["求解非线性问题，" 第7.1.1节中的"不稳定问题的自动稳定"](pt03ch07s01aus49.md#usb-anl-anonlineareqns-stabilize-over)中详细描述了可用的自动稳定方案。

#### 增量

Abaqus/Standard使用Newton法求解非线性平衡方程。许多问题涉及历史相关的响应；因此，解通常作为一系列增量获得，并迭代以在每个增量内获得平衡。有时必须保持增量较小（从旋转和应变增量必须较小的意义上），以确保正确建模历史相关的效应。最常见的是，增量大小的选择是计算效率的问题：如果增量太大，将需要更多迭代。此外，Newton法具有有限的收敛半径；太大的增量可能无法获得任何解，因为初始状态离正在寻求的平衡状态太远——它在收敛半径之外。因此，增量大小存在算法限制。

##### 自动增量

在大多数情况下，默认的自动增量方案是优选的，因为它将基于计算效率选择增量大小。

| **输入文件用法：** | ``` [*STATIC](../key/key-link.md#usb-kws-hstatic) ``` |
| --- | --- |

| **Abaqus/CAE用法：** | 步骤模块：**创建步骤**：**一般**：**静态，一般**：**增量**：**类型**：**自动**（默认） |
| --- | --- |

##### 直接增量

还提供了直接用户对增量大小的控制，因为如果您对特定问题有相当多的经验，您可能能够选择更经济的方法。

| **输入文件用法：** | ``` [*STATIC](../key/key-link.md#usb-kws-hstatic), DIRECT ``` |
| --- | --- |

| **Abaqus/CAE用法：** | 步骤模块：**创建步骤**：**一般**：**静态，一般**：**增量**：**类型**：**固定** |
| --- | --- |

使用直接用户控制，在完成允许的最大迭代次数后（定义在["常用控制参数，" 第7.2.2节"](pt03ch07s02aus50.md)中），即使未满足平衡容差，也可以接受增量的解。不推荐这种方法；只有当您对如何解释以这种方式获得的结果有透彻理解时，才应在特殊情况下使用。如果使用此选项，通常需要非常小的增量和至少两次迭代。

| **输入文件用法：** | ``` [*STATIC](../key/key-link.md#usb-kws-hstatic), DIRECT=NO STOP ``` |
| --- | --- |

| **Abaqus/CAE用法：** | 步骤模块：**创建步骤**：**一般**：**静态，一般**：**其他**：**达到最大迭代次数后接受解** |
| --- | --- |

### 稳态摩擦滑动

在静态分析过程中，您可以通过将物体运动规定为预定义场来模拟两个可变形体之间或以不同速度运动的可变形体与刚性体之间的稳态摩擦滑动。在这种情况下，假设滑移速度由用户指定的速度差异得出，与节点位移无关，如["Coulomb摩擦，" Abaqus Theory Guide第5.2.3节"](../stm/stm-link.md#stm-ifc-coulombfric)中所述。

由于这种摩擦行为与不使用稳态摩擦滑动时的摩擦行为不同，在相对速度由预定义运动确定的分析步骤与先前步骤之间，解决方案可能会出现不连续。一个例子是盘式制动系统中介质的初始预加载与随后制动分析（其中盘以规定旋转速度旋转）之间发生的连续性。为了确保解决方案的平滑过渡，建议在指定预定义运动的分析步骤之前的所有分析步骤使用零摩擦系数。然后，您可以在稳态分析中修改摩擦特性以使用所需的摩擦系数（请参见["摩擦行为，" 第37.1.5节中的"在Abaqus/Standard分析期间更改摩擦特性"](pt09ch37s01aus169.md#usb-cni-afriction-change-std)）。

| **输入文件用法：** | ``` [*MOTION](../key/key-link.md#usb-kws-hmotion) ``` |
| --- | --- |

| **Abaqus/CAE用法：** | 不支持在Abaqus/CAE中预定义运动场。 |
| --- | --- |

### 初始条件

可以指定应力、温度、场变量、依赖于求解的状态变量等的初始值。["Abaqus/Standard和Abaqus/Explicit中的初始条件，" 第34.2.1节"](pt07ch34s02aus116.md)描述了所有可用的初始条件。

### 边界条件

边界条件可以施加于任何位移或旋转自由度（1-6）；开口截面梁单元的翘曲自由度7；或者，如果模型中包含静水压力流体单元，则施加于流体压力自由度8。如果边界条件施加于旋转自由度，您必须了解Abaqus如何处理有限旋转（请参见["Abaqus/Standard和Abaqus/Explicit中的边界条件，" 第34.3.1节"](pt07ch34s03aus118.md)）。在分析期间，可以使用幅值定义来改变规定的边界条件（请参见["幅值曲线，" 第34.1.2节"](pt07ch34s01aus115.md)）。

### 载荷

可以在静态应力分析中规定以下载荷：
- 集中节点力可以施加于位移自由度（1-6）；请参见["集中载荷，" 第34.4.2节"](pt07ch34s04aus121.md)。
- 分布压力载荷或体积力可以施加；请参见["分布载荷，" 第34.4.3节"](pt07ch34s04aus122.md)。特定单元可用的分布载荷类型在["单元，" 第VI部分](pt06.md)中描述。

### 预定义场

可以在静态应力分析中指定以下预定义场，如["预定义场，" 第34.6.1节"](pt07ch34s06aus128.md)中所述：
- 虽然温度在静态应力分析中不是自由度，但可以将节点温度规定为预定义场。如果为材料指定了热膨胀系数，则施加温度与初始温度之间的任何差异将导致热应变（["热膨胀，" 第26.1.2节"](pt05ch26s01abm52.md)）。指定温度也会影响温度依赖性材料属性（如果有的话）。
- 可以指定用户定义场变量的值。这些值仅影响场变量依赖性材料属性（如果有的话）。

### 材料选项

大多数描述机械行为的材料模型可用于静态应力分析。以下材料属性在静态应力分析期间不活跃：声学属性、热属性（热膨胀除外）、质量扩散属性、电导率属性和孔隙流体流动属性。

率相关屈服（["率相关屈服，" 第23.2.3节"](pt05ch23s02abm19.md)）、滞后（["弹性体中的滞后，" 第22.8.1节"](pt05ch22s08abm14.md)）和双层粘塑性（["双层粘塑性，" 第23.2.11节"](pt05ch23s02abm27.md)）是静态分析期间活跃的唯一时间相关材料响应。率相关屈服响应在快速过程中通常很重要，如金属加工问题。滞后模型在模拟在循环载荷下表现出显著滞后的弹性体的大应变率相关响应时很有用。双层粘塑性模型在观察到显著的时间相关行为以及塑性时很有用，这对于金属通常发生在高温下。必须指定适当的时间尺度，以便Abaqus/Standard能够正确处理材料响应的率依赖性。

静态蠕变和膨胀问题以及时域粘弹性模型通过准静态过程进行分析（["准静态分析，" 第6.2.5节"](pt03ch06s02at04.md)）。当在静态分析中使用这些时间相关材料模型时，获得与率无关的弹性解，并且所选时间尺度对材料响应没有影响。对于蠕变和膨胀行为，这意味着载荷与蠕变效应发生的自然时间尺度相比是瞬时施加的。

瞬时载荷施加的相同概念适用于时域粘弹性行为。您也可以在静态过程中直接获得完全松弛的长期粘弹性解，而无需执行瞬态分析；此选择仅在定义时域粘弹性材料属性时有意义。如果请求长期粘弹性解，则与每个Prony级数项相关的内应力从步骤开始时的值逐渐增加到步骤结束时的长期值。

对于双层粘塑性材料模型，您可以获得纯弹性-塑性网络的长期响应。

当定义频域粘弹性材料属性时（请参见["频域粘弹性，" 第22.7.2节"](pt05ch22s07abm13.md)），相应的弹性模量必须指定为长期弹性模量。这意味着响应对应于长期弹性解，无论为步骤指定的时间周期如何。

| **输入文件用法：** | 使用以下选项获得具有时域粘弹性的完全松弛长期弹性解或双层粘塑性的长期弹塑性解： |
| --- | --- |
|  | ``` [*STATIC](../key/key-link.md#usb-kws-hstatic), LONG TERM ``` |

| **Abaqus/CAE用法：** | 步骤模块：**创建步骤**：**一般**：**静态，一般**或**静态，Riks**：**其他**：**使用时域材料属性获得长期解** |
| --- | --- |

### 单元

Abaqus/Standard中任何应力/位移单元都可以在静态应力分析中使用（请参见["为分析类型选择适当的单元，" 第27.1.3节"](pt06ch27s01aus112.md)）。虽然在静态应力分析中速度不可用，但阻尼器仍然可以使用（它们可用于稳定不稳定问题）。相对速度将按["阻尼器，" 第32.2.1节"](pt06ch32s02alm38.md)中的描述计算。

声学单元在静态步骤中不活跃。因此，如果声学-固体分析包含静态步骤，则只有固体单元会变形。如果变形很大，声学和固体网格可能不一致，后续的声学-结构分析步骤可能产生误导性结果。关于使用自适应网格技术变形声学网格的信息，请参见["ALE自适应网格：概述，" 第12.2.1节"](pt04ch12s02abo14.md)。

### 输出

静态应力分析可用的单元输出包括应力；应变；能量；状态、场和用户定义变量的值；以及复合失效量度。可用的节点输出包括位移、反作用力和坐标。所有输出变量标识符在["Abaqus/Standard输出变量标识符，" 第4.2.1节"](pt02ch04s02abv01.md)中概述。
