# 12.8.4 定义用户材料的常量







以下用户子程序允许您创建用户定义的材料模型：
- [`UMAT`](../sub/sub-link.md#sub-xsl-umat)，用于Abaqus/Standard中的机械材料模型。
- [`VUMAT`](../sub/sub-link.md#sub-xsl-vumat)，用于Abaqus/Explicit中的机械材料模型。
- [`UMATHT`](../sub/sub-link.md#sub-xsl-umatht)，用于Abaqus/Explicit中的热材料模型。

您可以在**编辑材料**对话框中输入供这些子程序使用的常量。请参见以下章节获取更多信息：- "用户定义的机械材料行为"，Abaqus分析用户指南第26.7.1节- "用户定义的热材料行为"，Abaqus分析用户指南第26.7.2节

**为用户材料定义材料常量：**

1. 从**编辑材料**对话框的菜单栏中，选择**常规**![](../graphics/images/arrow.gif)**用户材料**。（有关显示**编辑材料**对话框的信息，请参见"创建或编辑材料"，第12.7.1节。）
2. 如果您正在执行Abaqus/Standard分析，单击**用户材料类型**字段右侧的箭头，选择您正在为其定义常量的用户材料类型。
3. 如果您正在执行Abaqus/Standard分析，如果材料刚度矩阵![](../graphics/usi_eqn00008.gif)不是对称的，或者如果是热本构模型，![](../graphics/usi_eqn00009.gif)不是对称的，请切换**使用非对称材料刚度矩阵**。此参数导致Abaqus/Standard使用其非对称方程求解过程。
4. 如果您正在定义机械或热机械材料，请在**数据**表中输入**机械常量**。有关如何输入数据的详细信息，请参见"输入表格数据"，第3.2.7节。
5. 如果您正在定义热或热机械材料，请在**数据**表中输入**热常量**。有关如何输入数据的详细信息，请参见"输入表格数据"，第3.2.7节。
6. 单击**确定**关闭**编辑材料**对话框。或者，您可以从**编辑材料**对话框的菜单中选择其他材料行为进行定义（有关更多信息，请参见"浏览和修改材料行为"，第12.7.2节）。
