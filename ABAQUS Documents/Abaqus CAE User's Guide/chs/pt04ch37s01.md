# 37.1 定义弹簧和阻尼器





您可以定义表现出相同线性行为的弹簧和阻尼器。您也可以在同一组点上定义弹簧和阻尼器行为。如果您同时定义弹簧和阻阻器行为，它们将并行作用。您可以使用以下连接类型对弹簧和阻尼器进行建模：
- 连接两点并遵循点之间的作用线
- 连接两点并沿固定方向作用（仅适用于Abaqus/Standard分析）
- 将点连接到地（仅适用于Abaqus/Standard分析）

更多信息，请参见["弹簧"，Abaqus Analysis User's Guide第32.1.1节](../usb/usb-link.md#usb-elm-espring)和["阻尼器"，Abaqus Analysis User's Guide第32.2.1节](../usb/usb-link.md#usb-elm-edashpot)。

从Property模块或Interaction模块的主菜单栏中选择****Special**![](../graphics/images/arrow.gif)**Springs/Dashpots**![](../graphics/images/arrow.gif)**Create****来定义弹簧和阻阻器。从同一菜单中选择**Edit**来对现有定义进行更改。在部件上创建的弹簧和阻阻器会随部件一起实例化。

如果您希望在分析生成的输出数据库中包含弹簧和阻阻器结果数据，必须使用Step模块中的**History Output Request**编辑器。弹簧和阻阻器出现在Model Tree的**Engineering Features**容器下的部件下（如果在Property模块中创建）或装配下（如果在Interaction模块中创建）。
