# 74.4.2 自定义流线粗细和箭头







您可以通过更改流线的粗细、添加指示流体流动方向的定向箭头以及自定义启用显示时定向箭头的数量来调整当前视口中流线的显示。Abaqus/CAE使用**Color**选项中选择的均匀颜色显示定向箭头；更多信息，请参见["自定义流颜色," 第74.4.1节](pt06ch74s04s01.md)。当您增加或减少流线粗细时，箭头大小会相应变化。

**自定义流线粗细和箭头的步骤：**

1. 找到**Style**选项。从Visualization模块的主菜单栏中选择****Options**![](../graphics/images/arrow.gif)**Stream****。**Style**选项出现在对话框底部。
2. 拖动**Line thickness**滑块使流线变细或变粗。
3. 切换**Uniform thickness across streams**以在当前视口中以均匀粗细显示所有流线。
4. 切换**Show direction arrow**以沿流线显示定向箭头。
5. 拖动**Approx. number of arrows on each streamline**滑块来增加或减少每条流线上显示的定向箭头数量。
6. 单击**OK**实现您的更改并关闭"Stream Plot Options"对话框。
