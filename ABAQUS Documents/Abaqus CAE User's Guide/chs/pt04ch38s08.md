# 38.8 检查子模型的结果





分析完成后，您可以使用Visualization模块来叠加子模型和全局模型的等值线图。为了进行有意义的比较，叠加图的图层应使用相同的图例比例和相同的变形比例因子。更多信息，请参见[第79章，"叠加多个绘图"](pt07ch79.md)"。

您应检查以下内容：

**子模型位置**

您应检查子模型相对于全局模型的位置是否正确。您可以使用全局模型和子模型输出数据库的叠加图来检查相对位置。或者，您可以在提交子模型进行分析之前在Assembly模块中检查相对位置，方法是在全局模型中创建部件实例的临时实例。您可以查看全局模型中装配相对于子模型中装配的位置。然后，您可以在对子模型进行网格划分和分析之前，删除或抑制全局模型中部件的实例。

**子模型响应不影响全局响应**

您应检查子模型响应对全局响应的影响可以忽略不计。这是子建模的基本假设。您可以通过创建诸如应力和应变等变量的叠加等值线图来检查。轮廓应在子模型边界上相当连续。
