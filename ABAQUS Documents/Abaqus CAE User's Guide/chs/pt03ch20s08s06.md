# 20.8.6 偏移对象







您可以偏移草图中的对象以创建更大或更小的类似对象。您选择要偏移的边和偏移距离；Abaqus/CAE 显示预览，以便您可以选择偏移方向。您可以偏移任何不包含分支的连续开环或闭环。[图 20-25](pt03ch20s08s06.md#ske-offset)显示了几种边选择可以创建的两种可能的偏移。（为清晰起见，图中未显示顶点和基准点。）

**图 20-25** 样条曲线、正方形和圆的偏移。

![](../graphics/ske-offset-nls.png)

Abaqus/CAE 以与您用绘图圆规在纸上偏移边相同的方式偏移每个对象。使用圆规，您会沿着原始边移动圆规点，用铅笔在所需的垂直（偏移）距离处追踪一条新边。在任何尖角处，您都会旋转圆规使其与该角处的第二条边垂直。在此过程结束时，您会修剪与任何其他偏移边相交的偏移边。

如图 20-25](pt03ch20s08s06.md#ske-offset)所示，Abaqus/CAE 完成类似的过程：
- 复制直线边。
- 修剪重叠的偏移边（如向"内部"偏移的角）以形成新角。
- 对于向"外部"偏移的角，您可以选择创建尖角或半径等于偏移距离的圆角。
- 现有曲线的半径会增加或减少偏移距离。

如果曲线的半径因偏移而减小到零或更小，Abaqus/CAE 会创建一个新顶点来连接先前由曲线连接的两条边。详细说明请参见["偏移 Sketcher 对象的边，"第 20.19.3 节](pt03ch20s16hlb03.md)。
