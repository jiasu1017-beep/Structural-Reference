# 1.2 Abaqus 入门

本指南是一本入门教程，旨在为新用户提供指导，帮助用户使用 Abaqus/Standard 和 Abaqus/Explicit 分析实体、壳、 梁和桁架模型，并在 Abaqus/Viewer 或其他后处理器中查看结果。学习本指南不需要任何 Abaqus 基础知识，但建议您预先了解一些有限元知识。如果您已经熟悉 Abaqus 求解器产品（Abaqus/Standard 或 Abaqus/Explicit），但希望了解 Abaqus/CAE 界面简介，请参阅《Abaqus 入门：交互版》。

本文档主要涵盖应力/位移模拟，重点介绍线性静态和非线性静态分析以及动态分析。本文档还包含了 CFD 分析和流体-结构耦合模拟的介绍。其他类型的模拟（如传热和质量扩散）不在本指南范围内。

## 1.2.1 如何使用本指南

本指南的每一章都介绍一个或多个与使用 Abaqus/Standard 和 Abaqus/Explicit 相关的主题。在整个指南中，"Abaqus"一词用于统称 Abaqus/Standard 和 Abaqus/Explicit；当信息仅适用于其中一个产品时，则使用单独的产品名称。大多数章节都包含对所讨论主题的简要说明以及一到两个教程示例。您应该仔细学习这些示例，因为其中包含大量使用 Abaqus 的实用建议。

在这些示例中，Abaqus/Standard 和 Abaqus/Explicit 的功能会逐步介绍。您可以使用文本编辑器创建输入文件；但是，使用交互式前处理器可以更方便地为这些示例创建模型。您在每个示例中创建的输入文件的完整版本位于附录 A"示例文件"中。如果您可以访问 Abaqus/CAE，可以使用配套指南《Abaqus 入门：交互版》，通过详细的 Abaqus/CAE 教程来执行所有预处理和分析步骤。

本章是 Abaqus 和本指南的简短介绍。第 2 章"Abaqus 基础"以一个简单示例为中心，介绍使用 Abaqus 的基础知识。读完第 2 章"Abaqus 基础"后，您将了解为 Abaqus 模拟准备模型、检查数据、运行分析作业以及查看结果的基本方法。

第 3 章"有限元和刚体"介绍了 Abaqus 中可用的主要单元族。连续体（实体）单元、壳单元和梁单元的用法分别在第 4 章"使用连续体单元"、第 5 章"使用壳单元"和第 6 章"使用梁单元"中讨论。

线性动态分析在第 7 章"线性动力学"中讨论。第 8 章"非线性"主要介绍非线性概念，特别是几何非线性，并包含第一个非线性 Abaqus 模拟。非线性动态分析在第 9 章"非线性显式动力学"中讨论，材料非线性在第 10 章"材料"中介绍。第 11 章"多步骤分析"介绍了多步骤模拟的概念，第 12 章"接触"讨论了接触分析中出现的许多问题。第 13 章"使用 Abaqus/Explicit 进行准静态分析"介绍了使用 Abaqus/Explicit 解决准静态问题的方法。说明性示例是钣金成形模拟，该模拟需要在 Abaqus/Explicit 和 Abaqus/Standard 之间进行数据传递，以高效地执行成形和回弹分析。

## 1.2.2 本指南使用的约定

本指南遵循以下约定：

**排版约定**

教程示例中使用不同的文本样式来表示特定操作或标识项目。

- 使用 `COURIER FONT` 输入的代码应按显示的内容准确输入到 Abaqus/Viewer 或您的计算机中。例如：

  ```
  abaqus viewer
  ```

  应在计算机上键入此命令来运行 Abaqus/Viewer。

- Abaqus/Viewer 中的菜单选择、对话框中的选项卡以及屏幕上项目的标签使用**粗体**表示：

  ```
  **View** → **Graphics Options**
  **Contour Plot Options**
  ```

**视图方向三轴**

默认情况下，Abaqus/Viewer 使用字母选项 x-y-z 来标记视图方向三轴。一般来说，本指南采用数字选项 1-2-3，以便与自由度标记和输出标记直接对应。

## 1.2.3 基本鼠标操作

图 1-2 显示了左手和右手 3 键鼠标的鼠标按钮方向。

**图 1-2** 鼠标按钮。

![Mouse buttons](../graphics/cnv-mouse-nls.png)

以下术语描述了使用鼠标执行的操作：

**点击（Click）**

按下并快速释放鼠标按钮。除非另有说明，"点击"指令表示您应该点击鼠标按钮 1。

**拖动（Drag）**

按下并按住鼠标按钮 1，同时移动鼠标。

**指向（Point）**

移动鼠标直到光标位于所需项目上方。

**选择（Select）**

指向一个项目，然后点击鼠标按钮 1。

**[Shift]+点击**

按住 **\[Shift\]** 键，点击鼠标按钮 1，然后释放 **\[Shift\]** 键。

**[Ctrl]+点击**

按住 **\[Ctrl\]** 键，点击鼠标按钮 1，然后释放 **\[Ctrl\]** 键。

Abaqus/Viewer 专为使用 3 键鼠标而设计。因此，本指南按照图 1-2 所示的方式将鼠标按钮称为 1、2 和 3。但是，您也可以使用 2 键鼠标操作 Abaqus/Viewer，如下所示：

- 两个鼠标按钮分别相当于 3 键鼠标上的鼠标按钮 1 和 3。
- 同时按下两个鼠标按钮相当于按下 3 键鼠标上的鼠标按钮 2。

> **提示：**
> 在整个本指南的程序中，您会被指示点击鼠标按钮 2。请确保将鼠标按钮 2（或滚轮按钮）配置为充当中键点击。
