# 38.2 创建子模型





当您成功分析了全局模型并生成了包含全局模型结果的输出数据库或结果文件后，您就可以创建子模型了。首先将全局模型复制到新模型中，您将使用该模型来定义子模型。要复制模型，从主菜单栏中选择****Model**![](../graphics/images/arrow.gif)**Copy**![](../graphics/images/arrow.gif)***全局模型名称*****。在出现的**Copy Model**对话框中输入子模型的名称，然后单击**OK**。复制的模型成为当前模型。

您必须选择将用于驱动子模型的全局分析的输出数据库或结果文件。从主菜单栏中选择****Model**![](../graphics/images/arrow.gif)**Edit Attributes**![](../graphics/images/arrow.gif)***子模型名称*****。从出现的**Edit Model Attributes**对话框中，单击**Submodel**标签页，并执行以下操作：
- 切换**Read data from job**，并输入包含将驱动子模型的全局模型的输出数据库或结果文件的名称。如果输出数据库和结果文件存在且使用相同名称，则必须提供文件扩展名（`.odb`或`.fil`）。
- 此外，如果您的壳全局模型正在驱动实体子模型，请切换**Shell global model drives a solid submodel**。
- 单击**OK**关闭**Edit Model Attributes**对话框。
