# 63.3 评估离散场






如果您为边界条件指定了离散场，Abaqus/CAE会在写入输入文件时评估该离散场。
- 如果在边界条件所选区域和离散场中都找到了某个单元或节点，Abaqus/CAE会将边界条件的大小乘以该单元或节点处的值，以确定提交给分析的最新值。材料分配预定义场是唯一的例外；此边界条件没有大小。
- 如果在边界条件所选区域中找到了某个单元或节点，但在离散场中未找到，则边界条件的大小将乘以默认值并提交给分析。

[表63-1](pt06ch63hla02.md#value-table)总结了对标量离散场写入输入文件值的确定方式。

**表63-1** 评估标量离散场。
| 单元或节点在边界条件所选区域中？ | 单元或节点在离散场中？ | 写入输入文件的值 |
| --- | --- | --- |
| 是 | 是 | 边界条件大小乘以离散场中指定的值 |
| 是 | 否 | 边界条件大小乘以离散场中指定的默认值 |
| 否 | 是 | 不写入输入文件 |

对于具有多个自由度的边界条件（如位移），使用边界条件离散场；评估离散场变得更加复杂。对于可以激活单个自由度的边界条件，仅考虑边界条件中激活的自由度；边界条件中未激活的任何自由度将被忽略。边界条件中为每个自由度指定的大小将乘以离散场中指定的大小或该自由度在离散场中的默认值。

例如，如果您定义了位移边界条件，如[图63-1](pt06ch63hla02.md#usi-dfl-bc)所示，对于`节点集-1`（包含节点`Part-2-1.10`、`Part-2-1.20`和`Part-2-1.30`）和边界条件离散场，如[图63-2](pt06ch63hla02.md#usi-dfl-df)所示，提交给输入文件的值如[表63-2](pt06ch63hla02.md#usi-dfl-input)所示。

**表63-2** 评估边界条件离散场的结果。
| 节点ID | 自由度 | 写入输入文件的值 |
| --- | --- | --- |
| `Part-2-1.10` | 3 (**U3**) | 30 |

| `Part-2-1.40` | 3 (**U3**) | 无 |

导致零大小的边界条件和预定义场的自由度将按原样写入输入文件；但是，导致零大小的载荷的自由度不会写入输入文件。

在为装配级和历史对象（如载荷或相互作用）创建离散场时，必须指定节点或单元编号的完整名称，如["定义装配中的命名约定" Abaqus Analysis User's Guide第2.10.1节](../usb/usb-link.md#usb-int-ipartassy-naming)中所述。
