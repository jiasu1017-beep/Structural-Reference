# 13.11.5 用相隔指定距离的平行边约束两个实例





从主菜单栏选择**约束**![](../graphics/images/arrow.gif)**边对边**来施加约束，将可动实例定位，使所选边平行于固定实例的所选边。此外，如果实例是二维的，您必须指定所选边之间的距离；否则，Abaqus/CAE会使它们重合。所有位置约束都是装配的特征，可以使用特征操作工具集来抑制或删除。此外，您可以编辑两条所选边之间的间隙（如适用）。更多信息，请参见["位置约束方法有何不同，"第13.5.2节](pt03ch13s05s02.md)。

**要用相隔指定距离的平行边约束两个零件或模型实例：**

1. 从主菜单，选择**约束**![](../graphics/images/arrow.gif)**边对边**。**提示：**您也可以使用装配模块工具箱中的![](../graphics/ico_constraintEdgeToEdge.png)工具来施加边对边约束。有关装配工具箱中工具的图表，请参见["使用装配模块工具箱，"第13.9节](pt03ch13s09.md)。Abaqus/CAE在提示区域显示提示来引导您完成过程。
2. 从将移动的实例中选择一条直边，并从将保持固定的实例中选择一条直边，如下图所示：![](../graphics/asm-help-paralleledge-select-nls.png)除了选择边或基准轴外，您也可以选择基准坐标系的一个轴。Abaqus/CAE沿所选边显示箭头。当Abaqus/CAE提示您从固定实例中选择边时，您可以选择在零件或装配模块中创建的基准轴。相比之下，当您从可动零件实例中选择边时，您只能选择在零件模块中创建的基准轴。
3. 从提示区域中的按钮，执行以下操作之一：
   - 点击**确定**接受可动实例边上箭头的方向。
   - 点击**翻转**反转可动实例边上箭头的方向，然后点击**确定**。更改箭头方向的效果在下一步中说明。如果实例是三维的，Abaqus/CAE将可动实例定位，使所选边平行且重合。
4. 如果实例是二维的，您必须指定所选边之间的间隙。在提示区域出现的文本字段中，输入从可动实例的边到固定实例的边的距离，沿着固定实例的边的法线方向为正。Abaqus/CAE将可动实例定位，使两条边平行且箭头指向相同方向。此外，可动实例被平移以满足指定的间隙。固定实例的方向保持不变。指定距离和更改箭头方向的效果用二维实例说明如下图：![](../graphics/asm-help-edgetoedge.png)
   如果边对边约束与现有约束冲突，Abaqus/CAE显示错误消息并取消操作。要避免冲突，您可以尝试反转将移动的实例和将保持固定的实例的选择。或者，您可以先将现有约束转换为绝对位置，然后重新施加边对边约束。
