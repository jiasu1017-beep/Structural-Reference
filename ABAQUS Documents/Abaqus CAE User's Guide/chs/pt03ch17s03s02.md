# 17.3.2 网格属性和控制







Abaqus/CAE 为您提供了多种用于控制网格特征的工具：
- 您可以通过沿模型边缘创建种子来指定网格的密度，以指示单元角节点应放置的位置。例如，图 17-1 显示了在顶部和左侧边缘具有偏差种子的模型。**图 17-1** 具有偏差种子的模型。![](../graphics/mgn-bias.png) 有关更多信息，请参见"理解种子"第 17.4 节。
- 您可以选择网格单元的形状。例如，图 17-2 显示了一个模型，该模型首先使用四边形单元进行网格划分，然后使用三角形单元进行网格划分。**图 17-2** 具有不同单元形状的两个网格。![](../graphics/mgn-elementshapes.png) 有关更多信息，请参见"分配 Abaqus 单元类型"第 17.5 节。
- 您可以选择网格划分技术——自由、结构化或扫掠——并且，在适用的情况下，可以选择网格划分算法——中轴或前进前缘。有关更多信息，请参见"网格生成"第 17.3.3 节。
- 您可以通过选择单元族、几何阶次和形状以及特定单元控制（如沙漏）来选择分配给网格的单元类型。有关更多信息，请参见"理解网格生成"第 17.7 节。
