# 21.1.1 材料库：概述





本章描述如何在Abaqus中定义材料，并包含对每个材料行为的简要说明。更高级行为的详细讨论在[Abaqus理论指南](../stm/stm-link.md#stm)中提供。

### 定义材料

材料通过以下方式定义：
- 选择材料行为并定义它们（["材料数据定义，" 第21.1.2节](pt05ch21s01aus109.md)）；和
- 组合互补的材料行为，如弹性和塑性（["组合材料行为，" 第21.1.3节](pt05ch21s01aus110.md)）。

材料计算可以使用局部坐标系（["方向，" 第2.2.5节](pt01ch02s02aus15.md)）。任何各向异性属性必须在此局部系统中给出。

### 可用的材料行为

Abaqus中的材料库旨在提供线性和非线性、各向同性和各向异性材料行为的全面覆盖。单元中数值积分的使用，包括壳和梁截面的数值积分，提供了分析最复杂复合结构的灵活性。

材料行为分为以下几大类：
- 通用属性（材料阻尼、密度、热膨胀）；
- 弹性力学属性；
- 非弹性力学属性；
- 热属性；
- 声学属性；
- 静水压流体属性；
- 状态方程；
- 质量扩散属性；
- 电属性；和
- 孔隙流体流动属性。

某些力学行为是互斥的：这些行为不能出现在单个材料定义中。某些行为需要其他行为的存在；例如，塑性需要线性弹性。本章末尾每个材料行为描述以及["组合材料行为，" 第21.1.3节](pt05ch21s01aus110.md)中讨论了此类要求。

### 将材料行为与各种单元类型一起使用

将特定材料行为与实体、壳、梁和管道单元一起使用没有一般性限制。任何有意义的组合都是可以接受的。存在的少数限制在以下页面描述该特定行为时提到。在每个材料行为描述的末尾会显示有关可与材料行为一起使用的单元的部分。

### 使用完整材料定义

材料定义可以包含对使用该材料的单元或分析没有意义的行为。此类行为将被忽略。例如，材料定义可以包含热传递属性（电导率、比热容）以及应力-应变属性（弹性模量、屈服应力等）。当此材料定义与非耦合应力/位移单元一起使用时，Abaqus会忽略热传递属性；当与热传递单元一起使用时，力学强度属性将被忽略。此功能允许您开发完整的材料定义并在任何分析中使用它们。

### 在Abaqus/Standard中使用分布为均匀实体连续单元定义空间变化材料行为

在Abaqus/Standard中，可以为均匀实体连续单元使用分布定义空间变化的 mass density（["密度，" 第21.2.1节](pt05ch21s02abm01.md)）、线性弹性行为（["线性弹性行为，" 第22.2.1节](pt05ch22s02abm02.md)）和热膨胀（["热膨胀，" 第26.1.2节](pt05ch26s01abm52.md)）（["分布定义，" 第2.8.1节](pt01ch02s08aus26.md)）。在材料行为显著变化的模型中使用分布可以极大地简化前后处理，并通过允许单个材料定义定义空间变化的材料行为来提高分析性能。如果没有分布，此类模型可能需要许多材料定义和相关的截面分配。
