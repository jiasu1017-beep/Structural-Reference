# 82.5.2 使用升级脚本插件





通过从主菜单栏选择 ****Plug-ins**![](../graphics/images/arrow.gif)**Abaqus**![](../graphics/images/arrow.gif)**Upgrade Scripts**** 来启动插件。在 **Upgrade Scripts** 对话框中输入以下内容：
- 执行以下操作之一来选择要升级的文件： - 选择 **Files**，然后输入要升级的文件名。 - 选择 **Directory**，然后输入要升级的目录名。Abaqus/CAE 会升级目录及任何子目录中的所有脚本。
- 切换 **Create backups** 以保留原始脚本的副本。
- 执行以下操作之一来选择要升级的命令类型： - 选择 **Kernel** 仅升级 Abaqus 脚本接口命令。 - 选择 **GUI** 仅升级 Abaqus GUI Toolkit 命令。 - 选择 **Both** 升级 Abaqus 脚本接口和 Abaqus GUI Toolkit 命令。在大多数情况下，您应该选择同时升级两种类型的命令。
- 输入日志文件的名称。该文件将包含升级命令的日志，以及对脚本的更改以及任何错误或警告的列表。Abaqus/CAE 将日志文件存储在当前目录中，并在升级过程中在屏幕上显示其内容。
- 选择原始脚本的版本和您要升级到的版本。如果您不确定脚本是何时创建的，应该选择可用的最早版本。原始脚本的默认版本是当前版本之前的三个版本。
- 执行以下操作之一来选择当您点击 **Preview Changes** 时显示更新脚本预览的应用程序： - 选择 **Use web browser** 在默认 Web 浏览器中预览您的更改。 - 选择 **Use specified executable** 并输入将显示脚本与升级脚本之间差异的可执行文件的路径；例如，`windiff`。

在对话框底部的按钮中，执行以下操作：
- 点击 **Preview Changes** 以使用选定的应用程序预览脚本的更改。
- 如果更改可接受，点击 **Upgrade Scripts** 执行升级并关闭 **Upgrade Scripts** 对话框。
