# 41.3.3 警告和错误





**作业诊断**对话框中的**警告**和**错误**页面都显示有关分析期间遇到的不良条件的详细信息。警告包括可能导致可疑分析结果的条件的信息。错误包括导致Abaqus提前终止分析的条件的信息。

**警告**页面的内容取决于**作业历史**树中高亮显示的项目；如果作业被高亮显示，**警告**页面显示为整个作业保存到输出数据库的所有警告。如果您在**作业历史**树中高亮显示步骤、增量、尝试或迭代，Abaqus/CAE仅显示与高亮显示项目关联的警告信息。**错误**页面仅在作业项目被高亮显示时可用；它显示为整个作业保存到输出数据库的所有错误的信息。

**注意：**只有Abaqus/Standard和Abaqus/Explicit写入消息和状态文件的警告和错误的一个子集被保存到输出数据库以通过**作业诊断**对话框访问。

每个**警告**或**错误**页面包括一个**摘要**表，该表提供与您在**作业历史**树中高亮显示的项目关联的所有问题的简明描述。如果有一种以上类型的问题，您可以根据常见的**类别**过滤问题列表。

**警告**和**错误**页面的**详细信息**区域显示与您在**摘要**表中高亮显示的项目对应的更多信息。根据您选择的警告或错误类型，Abaqus/CAE显示提供问题详细信息的陈述或表格。如果**详细信息**区域显示包含节点和元素的表格信息，您可以切换**在视口中高亮显示选择**并选择项目以在当前视口中查看它们。
