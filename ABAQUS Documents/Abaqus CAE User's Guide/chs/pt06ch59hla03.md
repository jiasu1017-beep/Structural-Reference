# 59.5 基于边缘创建附件点图案





您可以沿单个边缘或相连边缘以简单图案定位附件点。然后，您可以在相邻面上或沿指定方向将点定位成行图案。如果需要，您可以将指定的点投影到选定的面上。您可以使用附件点来定义点式紧固件的位置。有关更多信息，请参见["关于紧固件，" 第29.1节](pt04ch29s01.md)。

**基于边缘创建附件点图案的步骤：**

1. 从主菜单栏，选择 ****Tools**![](../graphics/images/arrow.gif)**Attachment**![](../graphics/images/arrow.gif)**Points Offset From Edges****。
   **提示：**您也可以使用 ![](../graphics/ico_attPtOffEdge.png) 工具通过选择边缘和偏移量来创建附件点，该工具位于 Property 模块或 Interaction 模块工具箱的附件工具中。有关工具箱中附件工具的更多信息，请参见["了解附件点和线，" 第59.1节](pt06ch59s01.md)。

2. 从当前视口，选择要分配附件点的边缘。您可以选择多个边缘，只要这些边缘相连且不分叉。您可以单击提示区域右侧的 **Sets** 以使用先前定义的集合。如果边缘选择形成闭合回路，则从视口选择点生成的起始顶点。会出现红色箭头以指示从起始点生成点的方向。

3. 在提示区域，根据需要单击 **Flip** 反转点生成的方向，然后单击 **Yes**。Abaqus/CAE 显示**从边缘偏移创建附件点**对话框。

4. 在对话框中，指定沿边缘的点。
   - 选择 **by number** 以定义沿边缘选择的附件点数量。Abaqus/CAE 会自动沿边缘分布点。
     1. 输入沿边缘的点数。
     2. 输入从起始点的第一个附件点的所需偏移量。
     3. 输入从终点的最后一个附件点的所需偏移量。
   - 选择 **by spacing** 以定义附件点之间的距离。
     1. 输入点之间的间距。
     2. 输入从起始点的第一个附件点的所需偏移量。
     3. 指定沿边缘的点数，或选择 **Auto-fit** 以让 Abaqus/CAE 确定附件点的数量。如果您指定点数，Abaqus/CAE 将仅创建起点和终点之间的点；延伸到终点之外的任何点都不会被创建。

5. 如果需要，通过指定以下内容来定义附件点行的图案：
   - 输入图案中的行数。默认值为 1 会创建单行附件点。
   - 输入间距值以定义图案行之间的距离。
   - 输入图案第一行从边缘的偏移距离。默认值为 0 会在边缘上创建图案的第一行。
   - 选择 **Orthogonal to faces** 作为偏移方法，然后单击 ![](../graphics/ico_selectBlue.png) 选择要定位图案的相邻面。
   - 选择 **Along direction** 作为偏移方法，然后单击 ![](../graphics/ico_selectBlue.png) 指定方向向量的起点和终点。

6. 要将指定的点投影到选定的面上，请显示 **Projection** 选项卡页面，然后执行以下操作：
   1. 切换打开 **Project onto faces**。
   2. 单击 ![](../graphics/ico_selectBlue.png)，然后选择要将点投影到的面。这些面可以是平面或非平面。
   3. 选择投影点的方法。
      - 选择 **Proximity** 以允许 Abaqus/CAE 从每个点到所选面上最近的点绘制向量。
      - 选择 **Direction** 以定义沿其投影点的向量。单击 ![](../graphics/ico_selectBlue.png) 以选择向量的**起点**和**终点**。
      有关更多信息，请参见["了解投影方法，" 第59.2节](pt06ch59s02.md)。

7. 默认情况下，Abaqus/CAE 创建一个包含附件点的集合。如果需要，您可以修改集合名称。如果不想创建包含附件点的集合，请切换关闭 **Create set with name**。

8. 单击 **OK** 创建附件点。Abaqus/CAE 显示附件点。有关编辑附件点的信息，请参见["编辑基于边缘图案创建的附件点，" 第59.7.2节](pt06ch59s03hlb02.md)。
