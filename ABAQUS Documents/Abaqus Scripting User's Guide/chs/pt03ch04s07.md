# 4.8 轮换区域







`FXSwitcher` 组件管理堆叠在一起的子组件。`FXSwitcher` 允许您通过发送消息或调用其 `setCurrent` 方法来选择应显示哪个子组件。发送消息时，必须将消息 ID 设置为 `FXSwitcher.ID_OPEN_FIRST` 作为第一个子组件。然后您必须从该值开始递增后续子组件的消息 ID，如下例所示。有关消息的更多信息，请参阅"目标和消息"，第 6.5.4 节。要使用 `setCurrent` 方法，您应该提供要显示的子组件的从零开始的索引。例如，要显示第一个子组件，您应该使用索引值零调用 `setCurrent` 方法。

例如：

```
sw = FXSwitcher(parent)
FXRadioButton(hf, 'Option 1', sw, FXSwitcher.ID_OPEN_FIRST)
FXRadioButton(hf, 'Option 2', sw, FXSwitcher.ID_OPEN_FIRST+1)
hf1 = FXHorizontalFrame(sw)     
FXButton(hf1, 'Button 1')
FXButton(hf1, 'Button 2')
hf2 = FXHorizontalFrame(sw)
FXButton(hf2, 'Button 3')
FXButton(hf2, 'Button 4') 
```
