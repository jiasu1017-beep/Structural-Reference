# 5.5 示例：斜板







您被要求对如图 [图 5-10](ch05s05.md#gss-skewed-plate) 所示的板进行建模。 

**图 5-10** 斜板草图。

![](../graphics/gss-skewed-plate-nls.png)

它偏离全局 1 轴 30°，一端固定，另一端约束在与板轴平行的导轨上移动。当板承受均匀压力时，您需要确定跨中挠度。您还需要评估此问题线性分析是否有效。您将使用 Abaqus/Standard 进行分析。

### 5.5.1 坐标系

结构在全局坐标系中的方向和建议的原点位置如图 [图 5-10](ch05s05.md#gss-skewed-plate) 所示。板位于全局 1-2 平面中。如果您在此模型中使用壳单元的默认材料方向，结果容易解释吗？

### 5.5.2 网格设计

[图 5-11](ch05s05.md#gss-mesh-design) 显示了此模拟的建议网格设计。

**图 5-11** 斜板模拟的建议网格设计。

![](../graphics/gss-platemes的-c.png)

在选择单元类型之前，您必须回答以下问题：板是薄还是厚？应变小还是大？板相当薄，厚度与最小跨度比为 0.02。（厚度为 0.8 cm，最小跨度为 40 cm。）虽然我们不能直接预测结构中的应变大小，但我们认为应变很小。基于此信息，您选择二次壳单元（S8R5），因为它们对小应变模拟中的薄壳给出准确结果。有关壳单元选择的更多详细信息，请参阅 ["Choosing a shell element," Abaqus Analysis User's Guide 第 29.6.2 节](../usb/usb-link.md#usb-elm-eshellelem)。

### 5.5.3 预处理——创建模型

斜板示例的输入文件是 `skew.inp`，可在 ["Skew plate," A.3 节](ap01s03.md) 中获得。此示例使用通过创建如图 [图 5-12](ch05s05.md#gss-nodeset) 所示的节点集创建的网格，并存储在名为 `PLATE` 的单元集中。 

**图 5-12** 斜板模拟所需的节点集。

![](../graphics/gss-nodeset-nls.png)

此示例中的以下步骤描述了如何在此输入文件中定义材料和历史信息。此练习将帮助您更好地理解各种选项块如何组合以定义 Abaqus 模型。如果您希望使用 Abaqus/CAE 创建整个模型，请参阅 ["示例：斜板," Getting Started with Abaqus: Interactive Edition 第 5.5 节](../gsa/gsa-link.md#gsa-shl-example-skewplate)。

在开始构建模型之前，请决定单位制。尺寸以 cm 给出，但载荷和材料属性以 MPa 和 GPa 给出。由于这些不是一致单位，您必须选择在模型中使用的一致单位系统并转换必要的输入数据。

### 5.5.4 检查输入文件——模型数据

此时，我们假设您已使用预处理器创建了基本网格。在本节中，您将检查并更正输入文件，以及添加其他信息（如材料数据）。

**模型描述**

对于此模拟，以下 [*HEADING](../key/key-link.md#usb-kws-mheading) 选项中的描述是合适的：

```
*HEADING
Linear Elastic Skew Plate. 20 kPa Load.
S.I. Units (meters, newtons, sec, kilograms)
```
它清楚解释了您正在建模的内容以及您使用的单位。

**单元连接性**

检查以确保您使用正确的单元类型（S8R5）。您可能在预处理器中指定了错误的单元类型，或者翻译器在生成输入文件时犯了错误。模型中的 [*ELEMENT](../key/key-link.md#usb-kws-melement) 选项块应以下列内容开头：

```
*ELEMENT, TYPE=S8R5, ELSET=PLATE
```
在一些示例中，给 ELSET 参数的名称不是像 `PLATE` 这样的描述性名称。如果需要，您可以更改这些值，因为节点和单元集的有意义的名称使输入文件易于理解。

**节点集**

[图 5-12](ch05s05.md#gss-nodeset) 中显示的三个节点集在完成板的模型时很有用。这些节点集使用 [*NSET](../key/key-link.md#usb-kws-mnset) 选项块在输入文件中描述。

**定义替代材料方向**

如果使用默认材料方向，材料 1 方向上的直接应力将包含轴向应力（由板的弯曲产生）和垂直于板轴向的应力的贡献。如果材料方向与板的轴向和横向对齐，结果将更容易解释。因此，需要一个局部直角坐标系，其中局部 ![](../graphics/gsk_eqn00042.gif) 方向沿板轴向（即与全局 1 轴成 30°），局部 ![](../graphics/gsk_eqn00043.gif) 方向也在板平面内。

正如您在["壳材料方向，" 5.3 节"](ch05s03.md) 中学到的，[*ORIENTATION](../key/key-link.md#usb-kws-morientation) 选项定义这样的局部坐标系。选择点 *a*（见[图 5-8](ch05s03.md#gss-coord-system)）使坐标为 (10.0E2, 5.77E2, 0.)——即 ![](../graphics/gsk_eqn00052.gif) = tan 30——点 *b* 坐标为 (5.77E2, 10.E2, 0.)。您还必须指定不投影到壳表面上的轴（此模型中的 ![](../graphics/gsk_eqn00044.gif) 方向）以及额外旋转（使用此方法为零）。以下 [*ORIENTATION](../key/key-link.md#usb-kws-morientation) 选项块创建了名为 `SKEW` 的适当局部坐标系：

```
*ORIENTATION, NAME=SKEW, SYSTEM=RECTANGULAR
10.0E-2,5.77E-2,0.0, -5.77E-2,10.0E-2,0.0
3, 0.0
```
或者，您可以通过选择点 *a* 和点 *b* 位于全局坐标 1 和 2 轴上，并指定额外旋转 30°，来定义完全相同的局部坐标系：
```
*ORIENTATION, NAME=SKEW, SYSTEM=RECTANGULAR
1., 0., 0., 0., 1., 0.
3, 30.
```

**截面属性**

由于结构由单一材料制成且厚度恒定，因此所有单元的截面属性相同。因此，您可以使用单元集 `PLATE`（包含所有单元）为单元分配物理和材料属性。由于您假设板是线性弹性的，[*SHELL GENERAL SECTION](../key/key-link.md#usb-kws-mshellgensect) 选项比使用 [*SHELL SECTION](../key/key-link.md#usb-kws-mshellsection) 选项更高效。以下单元属性选项块定义此示例的截面属性：

```
*SHELL GENERAL SECTION, ELSET=PLATE, MATERIAL=MAT1,
ORIENTATION=SKEW
0.8E-2,
```

ORIENTATION 参数告诉 Abaqus 使用名为 `SKEW` 的局部坐标系来定义单元集 `PLATE` 中壳的材料方向。所有单元变量将在 `SKEW` 坐标系中定义。

**材料属性**

板由各向同性线性弹性材料制成，杨氏模量为 30.0 GPa，泊松比为 0.3。以下材料选项块指定此材料数据：

```
*MATERIAL, NAME=MAT1
*ELASTIC
30.0E9, 0.3
```

**节点处的局部方向**

虽然 [*ORIENTATION](../key/key-link.md#usb-kws-morientation) 选项定义元素的局部坐标系，但您必须使用 [*TRANSFORM](../key/key-link.md#usb-kws-mtransform) 选项来定义节点的局部坐标系。这两个选项完全独立。如果节点引用使用 [*TRANSFORM](../key/key-link.md#usb-kws-mtransform) 定义的局部坐标系，与节点相关的所有数据（如边界条件、集中载荷或节点输出变量（位移、速度、反作用力等））都在变换坐标系中定义。

[*TRANSFORM](../key/key-link.md#usb-kws-mtransform) 选项格式如下：

```
[*TRANSFORM](../key/key-link.md#usb-kws-mtransform), NSET=*<node set name>*, TYPE=*<axis type>*
<![](../graphics/gsk_eqn00045.gif)>, <![](../graphics/gsk_eqn00046.gif)>, <![](../graphics/gsk_eqn00047.gif)>, <![](../graphics/gsk_eqn00048.gif)>, <![](../graphics/gsk_eqn00049.gif)>, <![](../graphics/gsk_eqn00050.gif)>
```
数据行指定两点 *a* 和 *b* 的坐标，与 [*ORIENTATION](../key/key-link.md#usb-kws-morientation) 选项的方式大致相同。使用 [*TRANSFORM](../key/key-link.md#usb-kws-mtransform) 定义的坐标系在物体变形时不旋转；它在模拟开始时定义的方向固定。可以指定直角（TYPE=R）、圆柱（TYPE=C）和球（TYPE=S）坐标系。使用 NSET 参数指定使用此局部坐标系的节点集。

如图 [图 5-10](ch05s05.md#gss-skewed-plate) 所示，板的一端约束在与板轴平行的导轨上移动。由于此边界条件与全局轴不一致，您必须将板此端的节点变换为具有与板对齐的轴的局部坐标系。以下 [*TRANSFORM](../key/key-link.md#usb-kws-mtransform) 选项实现此变换：

```
*TRANSFORM, NSET=ENDB, TYPE=R
10.0E-2,5.77E-2,0.0, -5.77E-2,10.0E-2,0.0
```
此选项块为节点集 `ENDB` 定义了自由度在局部坐标系中的值，该坐标系具有与板长轴对齐的 ![](../graphics/gsk_eqn00042.gif) 轴（即局部系统绕全局 3 轴旋转 30°）。

### 5.5.5 检查输入文件——历史数据

现在我们检查输入文件的历史定义部分。需要一个步骤来定义此模拟。

**步骤定义**

[*STEP](../key/key-link.md#usb-kws-hstep) 定义指定线性静态模拟：

```
*STEP, PERTURBATION
Uniform pressure (20.0 kPa) load
*STATIC
```

[*STEP](../key/key-link.md#usb-kws-hstep), PERTURBATION 后的行包含对此步骤施加载荷的清晰描述。

**边界条件**

板左端的节点（节点集 `ENDA`）需要通过以下边界条件完全约束：

```
*BOUNDARY
ENDA, ENCASTRE
```

板右端的节点需要约束以模拟其"导轨"边界条件。由于您已使用 [*TRANSFORM](../key/key-link.md#usb-kws-mtransform) 变换了此端的节点，因此必须在局部坐标系中施加边界条件。要允许这些节点仅在局部 1 方向（沿板轴）移动，所有其他自由度必须按如下方式约束：

```
ENDB, 2,6
```
如果您没有定义节点集 `ENDA` 和 `ENDB`，您必须为每个节点创建一个数据行。

**载荷**

在此模拟中，20.0 kPa 的分布压力载荷施加到板上。如图 [图 5-10](ch05s05.md#gss-skewed-plate) 所示，压力在负全局 3 方向上作用。压力载荷使用 [*DLOAD](../key/key-link.md#usb-kws-hdload) 选项施加到单元面（[*DLOAD](../key/key-link.md#usb-kws-hdload) 在 [第 4 章，"使用连续体单元"](ch04.md) 中的耳片模型示例中有描述）。壳单元只有一个面；因此，压力载荷标识符仅为"P"。壳上的正压力在正元素法线方向上作用。["Skew plate," A.3 节](ap01s03.md) 中输入文件中的壳单元具有与正全局 3 轴对齐的法线。因此，以下输入定义了正确压力载荷：

```
*DLOAD
PLATE, P, -20000.0
```
由于单元集 `PLATE` 包含模型中的所有单元，此选项块将压力载荷施加到模型中的所有单元。 

**输出请求**

如果预处理器生成了默认输出请求选项，您应该删除它们。要创建用于 Abaqus/Viewer 的输出数据库（`.odb`）文件和单元应力、节点反作用力和力矩以及板跨中位移的打印表，请在输入文件中包含以下输出请求：

```
*OUTPUT, FIELD, OP=NEW
*NODE OUTPUT
U, RF
*ELEMENT OUTPUT
S, E
*OUTPUT, HISTORY, OP=NEW
*NODE OUTPUT, NSET=MIDSPAN
U,
*EL PRINT
S,
E,
*NODE PRINT, SUMMARY=NO, TOTALS=YES, GLOBAL=YES
RF,
*NODE PRINT, NSET=MIDSPAN
U,
```

指定 [*OUTPUT](../key/key-link.md#usb-kws-houtput) 选项会覆盖前几章中注意到的默认输出选择。该选项与 FIELD 和 HISTORY 参数一起使用，请求场和历史输出到输出数据库文件。通常，场输出用于生成云图、符号图和变形形状图；历史输出用于 X–Y 绘图。结合 [*OUTPUT](../key/key-link.md#usb-kws-houtput) 选项，[*NODE OUTPUT](../key/key-link.md#usb-kws-hnodeoutput) 选项用于请求节点变量的输出，[*ELEMENT OUTPUT](../key/key-link.md#usb-kws-helementoutput) 选项用于单元变量的输出。

### 5.5.6 运行分析

将输入存储在名为 `skew.inp` 的文件中后，交互运行分析。如果您不记得如何运行分析，请参阅 ["运行分析，" 4.3.6 节](ch04s03.md#gsk-gen-ctm-runninganal)。如果您的分析未完成，检查数据文件 `skew.dat` 中的错误消息。更改输入文件以删除错误；如果您仍有问题运行模型，请将您的输入文件与 ["Skew plate," A.3 节](ap01s03.md) 中给出的文件进行比较。

### 5.5.8 后处理

本节讨论使用 Abaqus/Viewer 进行后处理。云图和符号图都可用于可视化壳分析结果。由于在 [第 4 章，"使用连续体单元"](ch04.md) 中详细讨论了云图绘制，我们在此使用符号图。

通过在操作系统提示符下键入以下命令启动 Abaqus/Viewer：

```
abaqus viewer odb=skew
```

默认情况下，Abaqus/Viewer 绘制模型的未变形形状。

**单元法线**

使用未变形形状图检查模型定义。检查斜板模型的单元法线是否正确定义并指向正 3 方向。

**显示单元法线：**

1. 从主菜单栏中，选择 ****Options**![](../graphics/images/arrow.gif)**Common****；或使用工具箱中的 ![](../graphics/ico_plotCommonOptions.png) 工具。出现 **Common Plot Options** 对话框。
2. 点击 **Normals** 选项卡。
3. 切换 **Show normals**，并接受 **On elements** 的默认设置。
4. 点击 **OK** 应用设置并关闭对话框。

默认视图是等轴测的。您可以使用视图菜单中的选项或视图操作工具栏中的视图工具（如 ![](../graphics/ico_viewRotate.png)）更改视图。

**更改视图：**

1. 从主菜单栏中，选择 ****View**![](../graphics/images/arrow.gif)**Specify****。出现 **Specify View** 对话框。
2. 从可用方法列表中，选择 **Viewpoint**。
3. 输入视点向量的 ![](../graphics/gsk_eqn00054.gif)-、![](../graphics/gsk_eqn00055.gif)- 和 ![](../graphics/gsk_eqn00056.gif) 坐标为 `0.2, 1, 0.8`，向上向量的坐标为 `0, 0, 1`。
4. 点击 **OK**。Abaqus/Viewer 以指定视图显示您的模型，如图 [图 5-13](ch05s05.md#gss-normals-c) 所示。**图 5-13** 斜板模型中的壳单元法线。![](../graphics/gss-normals-c.png)

**符号图**

符号图将指定变量显示为从节点或单元积分点发出的矢量。您可以生成大多数张量值和矢量值变量的符号图。例外主要是非机械输出变量和存储在节点处的单元结果（如节点力）。箭头相对大小表示结果的相对大小，矢量沿结果全局方向定向。符号图图例显示每个箭头颜色如何对应特定范围的值。您可以为合量（如位移 (U)、反作用力 (RF) 等）的结果绘图；或者您可以绘制这些变量的各个分量。

在继续之前，抑制单元法线的可见性。

**生成位移的符号图：**

1. 从 **Field Output** 工具栏左侧的变量类型列表中，选择 **Symbol**。
2. 从工具栏中央的输出变量列表中，选择 **U**。
3. 从向量量和选定分量列表中，选择 **U3**。Abaqus/Viewer 在变形模型形状上显示位移矢量合量的符号图。
4. 默认着色渲染样式遮挡了箭头。可以通过使用**通用绘图选项**对话框将渲染样式更改为 **Wireframe** 来获得箭头的无遮挡视图。如果单元法线仍然可见，您应该立即关闭它们。
5. 符号图也可以基于未变形模型形状。从主菜单栏中，选择 ****Plot**![](../graphics/images/arrow.gif)**Symbols**![](../graphics/images/arrow.gif)**On Undeformed Shape****。出现未变形模型形状上的符号图，如图 [图 5-14](ch05s05.md#gss-displacement) 所示。

**图 5-14** 位移符号图。

![](../graphics/gss-displacement-c.png)

您可以使用符号图绘制张量变量（如应力）的主值。应力主值的符号图在每个积分点产生三个矢量，每个对应一个主值，方向沿相应主方向。压缩值用指向积分点的箭头表示，拉伸值用远离积分点的箭头表示。您也可以绘制各个主值。

**生成主应力的符号图：**

1. 从 **Field Output** 工具栏左侧的变量类型列表中，选择 **Symbol**。
2. 从工具栏中央的输出变量列表中，选择 **S**。
3. 从张量量和分量列表中，选择 **ALL_PRINCIPAL_COMPONENTS** 作为张量量。Abaqus/Viewer 显示主应力符号图。
4. 从主菜单栏中，选择 ****Options**![](../graphics/images/arrow.gif)**Symbol****；或使用工具箱中的 **Symbol Options** ![](../graphics/ico_plotSymbolOptions.png) 工具更改箭头长度。出现 **Symbol Plot Options** 对话框。
5. 在 **Color & Style** 页面中，点击 **Tensor** 选项卡。
6. 拖动 **Size** 滑块选择 `2` 作为箭头长度。
7. 点击 **OK** 应用设置并关闭对话框。出现如图 [图 5-15](ch05s05.md#gss-vector-plot) 所示的符号图。**图 5-15** 板底面上的主应力符号图。![](../graphics/gss-vector-plot-c.png)
8. 默认显示截面点 1 处的主应力。要在非默认截面点绘图，从主菜单栏中选择 ****Result**![](../graphics/images/arrow.gif)**Section Points**** 打开 **Section Points** 对话框。
9. 选择用于绘图的非默认截面点。
10. 在复杂模型中，单元边缘可能遮挡符号图。要抑制单元边缘的显示，在 **Common Plot Options** 对话框的 **Basic** 选项卡页面中选择 **Feature edges**。图 [图 5-16](ch05s05.md#gss-perimeter) 显示了默认截面点处仅可见特征边缘的主应力符号图。**图 5-16** 使用特征边缘的符号图。![](../graphics/gss-perimeter-c.png)

**材料方向**

Abaqus/Viewer 还允许您可视化单元材料方向。如果，您希望验证材料方向是否在模拟中正确分配，此功能特别有用。

**绘制材料方向：**

1. 从主菜单栏中，选择 ****Plot**![](../graphics/images/arrow.gif)**Material Orientations**![](../graphics/images/arrow.gif)**On Undeformed Shape****；或使用工具箱中的 ![](../graphics/ico_plotMatOrientUndeformed.png) 工具。在未变形形状上绘制材料方向。默认情况下，绘制材料方向时不带箭头。
2. 从主菜单栏中，选择 ****Options**![](../graphics/images/arrow.gif)**Material Orientation****；或使用 **Material Orientation Options** ![](../graphics/ico_plotMatOrientOptions.png) 工具显示带箭头的 triad。出现 **Material Orientation Plot Options** 对话框。
3. 在 triad 中将 **Arrowhead** 选项设置为使用填充箭头。
4. 点击 **OK** 应用设置并关闭对话框。
5. 使用 **Views** 工具栏中的预定义视图显示如图 [图 5-17](ch05s05.md#gss-orientation) 所示的板。此图中透视已关闭。要关闭透视，请在 **View Options** 工具栏中点击 ![](../graphics/ico_viewParallel.png) 工具。**提示：**如果 **Views** 工具栏不可见，从主菜单栏中选择 ****View**![](../graphics/images/arrow.gif)**Toolbars**![](../graphics/images/arrow.gif)**Views****。默认情况下，材料 1 方向为蓝色，材料 2 方向为黄色，如果存在，材料 3 方向为红色。**图 5-17** 板中材料方向图的绘制。![](../graphics/gss-orientation-c.png)
