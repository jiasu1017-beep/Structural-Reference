### 概述

临界状态（粘土）塑性模型：
- 可与线弹性模型（["Linear elastic behavior," Section 22.2.1](pt05ch22s02abm02.md)）结合使用；
- 基于临界状态理论和帽子屈服面；
- 考虑材料的应变软化行为；
- 适用于粘土、土壤和软岩；以及
- 支持率相关（粘性）响应。

### 硬化定律

临界状态塑性模型使用塑性体积应变来定义硬化/软化行为。

| **输入文件用法：** | ``` [*CLAY PLASTICITY](../key/key-link.md#usb-kws-mclayplasticity) [*HARDENING](../key/key-link.md#usb-kws-mhardening) ``` |
| --- | --- |

### 率相关响应

临界状态塑性模型支持率相关响应。

| **输入文件用法：** | ``` [*CLAY PLASTICITY](../key/key-link.md#usb-kws-mclayplasticity), RATE=参数 ``` |
| --- | --- |

### 与其他材料模型的结合

临界状态塑性模型可与线弹性模型结合使用。

### 元素

临界状态塑性模型可与 Abaqus/Standard 和 Abaqus/Explicit 中的连续体元素一起使用。
