# D.1 从远程安装 DVD 访问安装文件





本节描述从远程 DVD 设备访问安装文件的过程。DVD 设备必须正确挂载。如果设备未自动挂载，请检查设备所在平台的 DVD 挂载命令。

完成以下步骤后，您可以使用安装过程安装 Abaqus：

**Linux 平台**

1. 找到 Abaqus 产品 DVD。
2. 将以下项目从 DVD 顶层复制到目标计算机可访问的位置：- 与您要安装的平台对应的目录（及其全部内容）- `common` 目录（及其全部内容）- `DSLS` 目录- `setup` 可执行文件 UNIX 实用程序 `cp` 和 `rcp` 便于复制这些项目。在不同平台之间复制文件时（特别是 UNIX 和 Windows 之间），可能需要特别注意保持原始文件权限和文件名大小写。通常，所有文件的读取和执行权限足以确保正常执行。
3. 在要安装 Abaqus 的计算机上创建 Abaqus 父目录（`SIMULIA/Abaqus`）；有关详细信息，请参阅["安装过程，"第 2.1 节](ch02s01.md)。
4. 从 Abaqus 父目录，执行 `*prod_dir*/setup` 启动安装过程，其中 *prod_dir* 是复制的安装文件的位置。

**Windows 平台**

如果 DVD 设备已定义为共享，则可以从远程 DVD 设备安装 Abaqus。设置 DVD 设备为共享必须由具有管理员权限的用户执行。要从共享的远程 DVD 安装 Abaqus，请连接到该驱动器，然后执行与本地驱动器相同的安装（请参阅["安装过程，"第 2.1 节](ch02s01.md)）。通用命名约定 (UNC) 路径不能用于运行安装程序可执行文件（`setup.exe`），但可以在安装过程中作为目标安装目录输入。请参阅["产品安装期间要输入的信息，"第 2.4.2 节](ch02s04.md#sgb-chp-install-prod-prompts)。
