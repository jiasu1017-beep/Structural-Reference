# 14.10.2 **增量**选项卡





### 14.10.2 **增量**选项卡

当您配置通用过程时，使用步骤编辑器中的**基本**选项卡输入步骤的总时间周期。您使用**增量**选项卡来配置Abaqus用于将步骤的总时间周期划分为增量方法。对于通用静态步骤以及许多其他类型的步骤，您可以在**增量**选项卡页面上设置以下选项：

**时间增量**

- 当您选择**自动**时间增量时，Abaqus使用为初始增量大小输入的值开始增量。随后时间增量的大小会根据求解收敛的速度进行调整。此选项是默认选择。
- 当您选择**固定**时间增量时，Abaqus在整个步骤中使用为初始增量大小输入的值。**警告：**选择**固定**时间增量可能会阻止求解收敛，不推荐使用。

**最大增量数**

Abaqus将步骤中的增量数限制为您为最大增量数输入的值。如果步骤超过此增量数，分析停止，并将诊断信息报告给Job模块并写入消息文件。默认情况下，Abaqus/CAE将最大增量数设置为100。

**初始增量大小**

Abaqus使用为初始增量大小输入的值开始步骤。

**最小增量大小**

只有当您使用自动时间增量分析模型时，Abaqus才检查最小增量大小。如果Abaqus需要比此值更小的时间增量来达到收敛求解，它将终止分析，向Job模块报告，并将诊断信息写入消息文件。如果您没有输入最小增量大小，Abaqus使用总时间周期的10-5倍。

****最大**增量大小**

只有当您使用自动时间增量分析模型时，Abaqus才检查最大增量大小。在分析期间，Abaqus不会将增量大小增加到此值以上。如果您没有指定此值，Abaqus/CAE将其设置为总时间周期的值（动态隐式过程除外，其默认最大增量大小取决于多种分析设置；请参见["配置动态隐式过程"，"配置通用分析过程，"第14.11.1节](pt03ch14s11s01.md#usi-sim-configure-dynamicimplicit)）。

**注意：**必须为上述每个增量选项输入值。如果您删除了增量选项的默认值但未能提供另一个值，Abaqus/CAE不允许您创建步骤。
