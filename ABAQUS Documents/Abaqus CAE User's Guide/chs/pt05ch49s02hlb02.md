# 49.2.2 生成比例因子动画





通过比例因子进行动画处理会生成从输出数据库的单个步骤和帧创建的图像序列。这些图像可以由变形形状、云图或符号绘图组成。对于变形形状绘图以及变形形状上的云图和符号绘图，图像在应用于变形的比例因子方面有所不同。对于云图和符号绘图，图像在应用于场输出的比例因子方面也有所不同。

**生成比例因子动画：**

1. 使用 **文件（File）** 菜单 [打开包含您的分析结果的输出数据库](pt02ch09s06hlb02.md)。
2. 使用 **结果（Result）** 菜单选择要绘制和分析结果，包括步骤和帧、变形场输出变量，以及对于云图或符号绘图动画，要绘制的 [主场输出变量](pt05ch42s03hlb03.md) 或 [符号场输出变量](pt05ch42s03hlb05.md)。有关场输出变量选择的更多信息，请参阅 ["场输出变量选择概述，" 第 42.5.1 节"](pt05ch42s03hlb01.md)。
3. 使用 **选项（Options）** 和 **视图（View）** 菜单来选择您想要的 [自定义选项](pt05ch40s03s03.md)。
4. 要启动动画，请从主菜单栏中选择 ****动画（Animate）**![](../graphics/images/arrow.gif)**比例因子（Scale Factor）****。**提示：**您还可以使用工具箱中的 ![](../graphics/ico_animScaleFactor.png) 工具来生成比例因子动画。动画在所有动画视口中开始，动画控件出现在上下文栏的右侧。Abaqus 现在处于比例因子动画绘图状态。在此状态下，每次您在 **结果（Results）**、**选项（Options）** 或 **视图（View）** 菜单对话框中点击 **应用（Apply）** 或 **确定（OK）** 时，Abaqus 会自动刷新您的绘图。所有动画视口将保持比例因子动画状态，直到您在某种其他状态（如未变形或变形状态）中生成绘图。有关自定义底层模型形状的更多信息，请参阅 ["自定义模型形状，" 第 55.4 节"](pt05ch55s04.md)。

# 49.2.4 自定义比例因子或谐波动画





您可以通过在 **半周期（Half cycle）** 比例因子范围和扩展范围 **全周期（Full cycle）** 之间进行选择来自定义动画模型的表观运动。扩展范围很有用，例如，可视化通过模态分析获得的振动模式。您还可以改变动画中的帧（单独图像）数量。更多的帧数会产生更平滑的动画。

在比例因子动画中，Abaqus/CAE 通过将比例因子范围除以帧数来计算动画比例因子值。这些值被应用于每个坐标方向中的 [变形比例因子](pt05ch55s04hlb01.md) 和 [场输出变量值](pt05ch42s03hlb01.md)，以生成动画序列中的每个图像。

在谐波动画中，Abaqus/CAE 通过将角度范围除以帧数来计算动画角度值。这些值被应用于计算该角度处的场输出变量值，并生成动画序列中的每个图像。

**自定义比例因子或谐波动画：**

1. 找到 **比例因子/谐波（Scale Factor/Harmonic）** 选项。从主菜单栏中，选择 ****选项（Options）**![](../graphics/images/arrow.gif)**动画（Animation）****；然后在出现的对话框中点击 **比例因子/谐波（Scale Factor/Harmonic）** 选项卡。出现 **比例因子/谐波（Scale Factor/Harmonic）** 选项。
2. 要控制范围，请点击您想要的 **相对缩放（Relative Scaling）**：- **全周期（Full cycle）**：对于比例因子动画（实数），此选择对应范围 1 到 1；对于谐波动画（具有实部和虚部的复数），此选择对应范围 180 到 180。- **半周期（Half cycle）**：对于比例因子动画（实数），此选择对应范围 0 到 1；对于谐波动画（具有实部和虚部的复数），此选择对应范围 0 到 180。
3. 要选择动画中的单独图像数量，请点击 **帧（Frames）** 箭头。指定数量的帧出现在 **帧（Frames）** 框中。
4. 点击 **应用（Apply）** 实施您的更改。所有动画视口中的比例因子或谐波动画都会更改以反映您指定的相对缩放和帧数。您的更改将在会话期间保存，并会影响谐波或比例因子动画状态中的所有后续绘图。

# 49.2.3 生成谐波动画





谐波动画生成从输出数据库的单个步骤和帧创建的图像序列。这些图像可以由变形形状、云图或符号绘图组成。对于变形形状绘图以及变形形状上的云图和符号绘图，图像在应用于变形值的角度方面有所不同。对于云图和符号绘图，图像在应用于场输出的角度方面有所不同；结果的实际和虚部都被使用。

默认情况下，Abaqus/CAE 在谐波动画期间显示帧计数器。帧计数器指示查看复数结果时的当前角度。

**生成谐波动画：**

1. 使用 **文件（File）** 菜单 [打开包含您的分析结果的输出数据库](pt02ch09s06hlb02.md)。
2. 使用 **结果（Result）** 菜单选择要绘制和分析结果，包括步骤和帧、变形场输出变量，以及对于云图或符号绘图动画，要绘制的 [主场输出变量](pt05ch42s03hlb03.md) 或 [符号场输出变量](pt05ch42s03hlb05.md)。有关场输出变量选择的更多信息，请参阅 ["场输出变量选择概述，" 第 42.5.1 节"](pt05ch42s03hlb01.md)。
3. 使用 **选项（Options）** 和 **视图（View）** 菜单来选择您想要的 [自定义选项](pt05ch40s03s03.md)。
4. 要启动动画，请从主菜单栏中选择 ****动画（Animate）**![](../graphics/images/arrow.gif)**谐波（Harmonic）****。**提示：**您还可以使用工具箱中的 ![](../graphics/ico_animHarmonic.png) 工具来生成谐波动画。动画在所有动画视口中开始，动画控件出现在提示区域。Abaqus 现在处于谐波动画绘图状态。在此状态下，每次您在 **结果（Results）**、**选项（Options）** 或 **视图（View）** 菜单对话框中点击 **应用（Apply）** 或 **确定（OK）** 时，Abaqus 会自动刷新您的绘图。所有动画视口将保持谐波动画状态，直到您在某种其他（例如，未变形、变形）状态中生成绘图。有关自定义底层模型形状的更多信息，请参阅 ["自定义模型形状，" 第 55.4 节"](pt05ch55s04.md)。

# 49.2.5 自定义动画的底层绘图





时间历史、比例因子和谐波动画都以快速连续的方式显示一系列变形形状、云图、符号或（仅限时间历史动画）材料取向绘图。在动画播放之前或播放期间的任何时候，您都可以自定义动画底层的变形形状、云图、符号或材料取向绘图。例如，您可以切换图例、调整变形比例因子或选择不同的场输出变量。使用 **视图（View）**、**结果（Result）** 和 **选项（Options）** 菜单来自定义动画的底层绘图。

**自定义底层绘图：**

1. 找到适用于当前绘图状态的自定义选项。- 对于云图绘图动画，请从主菜单栏中选择 ****选项（Options）**![](../graphics/images/arrow.gif)**云图（Contour）****。- 对于符号绘图动画，请从主菜单栏中选择 ****选项（Options）**![](../graphics/images/arrow.gif)**符号（Symbol）****。- 对于材料取向绘图动画，请从主菜单栏中选择 ****选项（Options）**![](../graphics/images/arrow.gif)**材料取向（Material Orientation）****。
2. 从出现的对话框中，选择您想要的绘图状态相关自定义选项。有关更多信息，请参阅 ["自定义您的绘图，" 第 40.3.3 节"](pt05ch40s03s03.md)。
3. 通过选择 ****选项（Options）**![](../graphics/images/arrow.gif)**通用（Common）****、****选项（Options）**![](../graphics/images/arrow.gif)**叠加（Superimpose）****、****视口（Viewport）**![](../graphics/images/arrow.gif)**视口注释选项（Viewport Annotation Options）****、****视图（View）**![](../graphics/images/arrow.gif)**图形选项（Graphics Options）**** 或 ****视图（View）**![](../graphics/images/arrow.gif)**ODB 显示选项（ODB Display Options）**** 来设置您想要的绘图状态无关自定义选项。
4. 通过选择 ****结果（Result）**![](../graphics/images/arrow.gif)**场输出（Field Output）**** 来选择要动画的 [结果变量](pt05ch42s03hlb01.md)。
5. 选择要动画的结果步骤和帧：- 对于比例因子或谐波动画，请通过选择 ****结果（Result）**![](../graphics/images/arrow.gif)**步骤/帧（Step/Frame）**** 来选择要动画的单个步骤和帧。- 对于时间历史动画，请通过选择 ****结果（Result）**![](../graphics/images/arrow.gif)**活动步骤/帧（Active Steps/Frames）**** 来选择要动画的步骤和帧。
