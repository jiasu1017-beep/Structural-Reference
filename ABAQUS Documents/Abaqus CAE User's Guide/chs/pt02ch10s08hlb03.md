# 10.9.3 将装配体导出为ACIS格式文件





从主菜单栏选择**文件**![](../graphics/images/arrow.gif)**导出**![](../graphics/images/arrow.gif)**装配**，可以将装配体导出为ACIS格式文件。许多计算机辅助建模和绘图应用程序可以读取和写入ACIS文件；因此，您可以在Abaqus/CAE和这些应用程序之间传输零件。您只能导出零件的几何体；如果存在孤儿元素，则不会将其包含在导出的零件中。

您可以在Abaqus/CAE会话期间随时导出装配体。Abaqus/CAE分别导出每个零件及其位置。如果随后导入包含装配体的ACIS格式文件，Abaqus/CAE会为每个实例创建一个零件，同时保留每个零件的原始位置，以便以后在装配模块中实例化零件时使用。当您创建导入的ACIS零件的实例时，Abaqus/CAE使用位置信息重新创建原始装配体。您不能直接导入装配体。

**将装配体导出为ACIS格式文件：**

1. 从主菜单栏，选择**文件**![](../graphics/images/arrow.gif)**导出**![](../graphics/images/arrow.gif)**装配**。出现**导出至ACIS文件**对话框。Abaqus/CAE列出所选目录中所有具有`.sat`文件扩展名的文件。
2. 选择要导出装配体的文件，或在**文件名**文本字段中输入新文件的名称。
3. 单击**确定**导出装配体并关闭**导出至ACIS文件**对话框。
