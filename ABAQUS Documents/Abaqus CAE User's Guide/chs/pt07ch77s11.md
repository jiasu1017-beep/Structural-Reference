# 77.11 保存和恢复自定义颜色映射





颜色映射是特定于会话的设置，默认情况下不会保存到模型数据库或输出数据库。Abaqus/CAE提供了两种保存自定义颜色编码定义的方法。
- 您可以将颜色映射定义保存为会话选项。会话选项可以保存到模型数据库、输出数据库或XML文件。当您使用此过程保存颜色映射时，Abaqus/CAE仅记录**颜色编码**对话框中当前所选项目的颜色映射。例如，当显示零件实例的颜色映射时，那些颜色映射定义是写入所选文件的唯一内容。
- 您可以创建颜色宏或将所选颜色映射写入ASCII文件。颜色宏记录所有颜色映射和您的初始颜色选择，而ASCII文件仅记录当前颜色映射。您可以像运行任何其他宏一样运行颜色宏；请参见["管理宏，"第9.11节](pt02ch09hla02.md)。颜色宏记录您选择的所有颜色映射，而不仅仅是**颜色编码**对话框中当前显示的颜色映射。

本节介绍用于保存颜色映射定义的基于宏的方法。有关将颜色映射保存为会话选项的更多信息，请参见["管理会话对象和会话选项，"第9.9节](pt02ch09s08.md)。

**保存颜色宏的步骤：**

1. 点击**颜色编码**工具栏中的![](../graphics/ico_colorCode.png)工具。Abaqus/CAE显示**颜色编码**对话框。
2. 点击**高级**，然后从出现的列表中选择**保存颜色宏**。Abaqus/CAE显示**创建宏**对话框，其中指示保存宏的位置。
3. 为此宏输入一个**名称**，然后点击**确定**。Abaqus/CAE保存该宏，使其可从**宏管理器**对话框运行。

**读取或写入颜色映射的ASCII文件的步骤：**

1. 点击**颜色编码**工具栏中的![](../graphics/ico_colorCode.png)工具。Abaqus/CAE显示**颜色编码**对话框。
2. 将鼠标指针放在**颜色映射**表上，然后点击鼠标按钮3，从出现的列表中选择以下选项之一：
   - 选择**写入文件**可选择文件名并保存当前颜色映射。
   - 选择**从文件读取**可选择文件名并读取已保存颜色映射的内容。
