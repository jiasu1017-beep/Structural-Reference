# 14.4.5 创建和修改输出请求





要创建输出请求，请从主菜单栏选择****Output**![](../graphics/images/arrow.gif)**Field Output Requests**![](../graphics/images/arrow.gif)**Create****或****Output**![](../graphics/images/arrow.gif)**History Output Requests**![](../graphics/images/arrow.gif)**Create****。出现的编辑器中可以输入定义输出请求所需的全部信息。编辑器顶部显示以下内容：
- 输出请求的名称。
- 你正在创建或修改输出请求的步骤名称。
- 与步骤关联的分析过程名称。

例如，[图14-3](pt03ch14s04s05.md#sim-odbeditor)显示了**场输出请求**编辑器。

**图14-3****场输出请求**编辑器。

![](../graphics/sim-odbeditor-nls.png)

编辑器的**域**部分允许你选择将从中生成输出的区域。你可以请求Abaqus将场数据写入输出数据库，用于以下内容：
- 整个模型
- 整个模型，仅外部节点和单元（Abaqus/Standard或Abaqus/Explicit分析中的三维模型）
- 一个集合
- 一个螺栓载荷
- 一个蒙皮
- 一个纵梁
- 一个紧固件
- 一个组装紧固件集
- 一个相互作用
- 一个复合铺层
- 一个子结构

类似地，你可以请求Abaqus将历史数据写入输出数据库，用于以下内容：
- 整个模型
- 一个集合
- 一个螺栓载荷
- 一个蒙皮
- 一个纵梁
- 一个紧固件
- 一个组装紧固件集
- 一个轮廓积分
- 一个通用接触曲面（仅限Abaqus/Explicit步骤）
- 一个集成输出截面（仅限Abaqus/Explicit步骤）
- 一个相互作用
- 弹簧/阻尼器
- 一个复合铺层

编辑器的**频率**部分允许你指定将输出写入输出数据库的频率。选择以下之一：
- **最后增量**仅在步骤的最后增量后请求输出。此输出频率仅在你选择Abaqus/Standard分析过程时可用。
- **每*n*个增量**在指定数量的增量后请求输出。如果你以增量指定频率，Abaqus也会在步骤的最后增量后写入输出。此输出频率在你选择Abaqus/Standard或Abaqus/CFD分析过程时可用。
- **每个时间增量**在每个时间增量时请求输出。此输出频率在你选择Abaqus/Explicit分析过程用于场输出时可用。
- **每*n*个时间增量**在指定数量的时间增量时请求输出。此输出频率在你选择Abaqus/Explicit分析过程用于历史输出时可用。
- **等间距时间间隔**在多个等间距时间间隔时请求输出。
- **每*x*单位时间**在经过特定时间段后请求输出。
- **从时间点**根据你指定的一组时间点请求输出。此输出频率在你选择Abaqus/Standard分析过程用于场和历史输出时可用，以及在你选择Abaqus/Explicit分析过程用于场输出时可用。

编辑器的**输出变量**部分包含适用于步骤过程和所选域的变量类别列表。选择以下之一：
- **从下方列表选择**从下方的复选框列表中请求变量。你可以点击类别名称旁边的复选框来选择该类别中的所有变量，或者点击类别名称旁边的箭头来显示该类别中的变量列表，然后选择各个变量。
- **预选默认值**请求过程的默认输出变量。
- **全部**请求该过程的所有输出变量。
- **编辑变量**从下方的文本字段请求变量。你可以手动编辑此字段并输入或删除变量名称。

**注意：**除了当前分析过程外，模型的其他方面可能会影响预选默认输出变量。例如，如果输出变量对分析过程有效但对网格中使用的单元类型无效，Abaqus将在分析期间删除该变量。

如果你使用**场输出请求**编辑器选择要包含在场输出请求中的向量或张量变量，Abaqus会自动在该步骤期间将该变量的所有分量写入输出数据库。例如，如果你在三维模型中选择向量U，Abaqus会将三个位移分量U1、U2和U3以及三个旋转分量UR1、UR2和UR3输出到输出数据库。

相比之下，如果你使用**历史输出请求**编辑器选择要包含在历史输出请求中的向量或张量变量，**历史输出请求**编辑器允许你选择该变量的各个分量。在历史输出请求中指定各个分量很有用，因为这些变量通常输出非常频繁——可能每个增量都输出。

如果你的模型包含钢筋，你必须启用**钢筋输出**才能将钢筋输出包含在Abaqus写入输出数据库的数据中，并在可视化模块中查看钢筋方向的图表。更多信息，请参见["理解壳截面中的钢筋，"第12.2.5节](pt03ch12s02s05.md)。

编辑器还允许你指定将从中获取输出的截面点。如果你从复合铺层请求输出，你可以指定为铺层的每个层获取输出的截面点。更多信息，请参见["从复合铺层请求输出，"第23.5节](pt04ch23s05.md)。

例如，在[图14-3](pt03ch14s04s05.md#sim-odbeditor)中，用户正在编辑与**Static, General**分析过程关联的场输出请求。用户已选择**应力**类别中的所有变量。这些变量将包含在名为`Side Load`的步骤期间的输出请求中。Abaqus将在每个增量从默认截面点写入输出。

有关选择输出变量和分量的详细说明，请参见以下各节：
- ["修改场输出请求，"第14.12.2节](pt03ch14s12hlb02.md)
- ["修改历史输出请求，"第14.12.3节](pt03ch14s12hlb03.md)

创建输出请求后，可以通过以下方式修改它：
- 选择****Output**![](../graphics/images/arrow.gif)**Field Output Requests**![](../graphics/images/arrow.gif)**Edit****或****Output**![](../graphics/images/arrow.gif)**History Output Requests**![](../graphics/images/arrow.gif)**Edit****显示场或历史输出请求编辑器。
- 选择****Output**![](../graphics/images/arrow.gif)**Field Output Requests**![](../graphics/images/arrow.gif)**Manager****或****Output**![](../graphics/images/arrow.gif)**History Output Requests**![](../graphics/images/arrow.gif)**Manager****显示场或历史输出请求管理器。使用管理器修改输出请求的步骤历史。（更多信息请参见["什么是步骤相关管理器？"第3.4.2节](pt01ch03s04s02.md)。）

如果你在创建请求的步骤中修改输出请求，你可以修改域、输出变量、钢筋输出选项、截面点和输出频率。但是，如果你修改传播到的步骤中的输出请求，则只能修改输出变量和输出频率。

当你从轮廓积分请求输出时，**历史输出请求**编辑器允许你仅选择输出频率、轮廓积分数量和轮廓积分计算类型。更多信息，请参见["请求轮廓积分输出，"第31.2.11节](pt04ch31s02hlb04.md)。
