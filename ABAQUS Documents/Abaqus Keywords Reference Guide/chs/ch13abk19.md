# *MODAL FILE









### *MODAL FILE在基于模式的动力学或特征值提取程序期间将广义坐标（模态幅度）数据或特征数据写入结果文件。

此选项用于在基于模式的动力学或特征值提取程序期间控制将广义坐标（模态幅度和相位）值或特征数据写入Abaqus/Standard结果文件。

**产品：**Abaqus/Standard  Abaqus/CAE  

**类型：**历史数据  

**级别：**步骤

**Abaqus/CAE：**不支持；Abaqus/CAE仅从输出数据库文件读取输出。

##### **参考：**

- ["输出到数据和结果文件，" Abaqus Analysis User's Guide第4.1.2节](../usb/usb-link.md#usb-out-oprintfile)

### **可选参数：**

FREQUENCY

此参数仅在基于模式的动力学程序中有效。

将此参数设置为输出频率，以增量计。除非FREQUENCY=0，否则输出将始终在每个步骤的最后一个增量写入结果文件。默认值为FREQUENCY=1。设置FREQUENCY=0以抑制输出。

### **在基于模式的动力学程序期间在结果文件中请求模态输出的数据行：**

**第一行：**

根据需要重复此数据行。

### **在特征值提取程序期间写入特征数据：**

不需要数据行；特征数据会自动写入。
