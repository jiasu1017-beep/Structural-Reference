# 59.6 通过投影点创建附件线





定位附件线是一个三步过程：

1. 选择要投影的点。
2. 沿指定方向将点投影到源面上。
3. 从投影点与源面的交点开始，沿源面法向将附件线投影到目标面，投影指定距离或到指定数量的目标面。

您可以使用附件线来定义离散紧固件的位置。有关更多信息，请参见["关于紧固件，" 第29.1节](pt04ch29s01.md)。

**通过投影点创建附件线的步骤：**

1. 从主菜单栏，选择 ****Tools**![](../graphics/images/arrow.gif)**Attachment**![](../graphics/images/arrow.gif)**Lines by Projecting Points****。
   **提示：**您也可以使用 ![](../graphics/ico_attLineByProj.png) 工具通过投影点来创建附件线，该工具位于 Interaction 模块工具箱的附件工具中。有关工具箱中附件工具的图表，请参见["了解附件点和线，" 第59.1节](pt06ch59s01.md)。

2. 从当前视口，选择要投影的点，然后在提示区域单击 **Done**。您可以选择顶点、参考点或附件点。

3. 从当前视口，选择要将点投影到的源面，然后在提示区域单击 **Done**。附件线从方向向量与源面相交的位置开始。

4. 从当前视口，选择将被附件线相交的目标面，然后在提示区域单击 **Done**。附件线从起点沿源面法向绘制，结束于最远的目标面。Abaqus/CAE 显示**通过投影点创建附件线**对话框。如果需要，您可以使用对话框编辑点以及源面和目标面。

5. 在对话框中，选择投影点的方法：
   - 选择 **Proximity** 以允许 Abaqus/CAE 从每个点到所选面上最近的点绘制向量。
   - 选择 **Direction** 以定义沿其投影点的向量。单击 ![](../graphics/ico_selectBlue.png) 以选择向量的**起点**和**终点**。

6. 要更改目标面，请显示 **Target** 选项卡页面，然后执行以下操作：
   1. Abaqus/CAE 显示一个箭头，指示源面上的点将投影到目标面的方向。单击 ![](../graphics/ico_flipArrow.png) 反转箭头方向。
   2. 选择确定投影向量长度的方法：
      - 选择 **Maximum projections along direction**，然后输入附件线将相交的目标面的最大数量。
      - 选择 **Maximum length of projection vector**，然后输入附件线的最大长度。

7. 默认情况下，Abaqus/CAE 创建一个将包含附件线端点的集合。如果需要，您可以修改集合名称。如果不想创建包含端点的集合，请切换关闭 **Create set with name**。

8. 单击 **OK** 创建附件线。Abaqus/CAE 显示附件线。有关编辑附件线的信息，请参见["编辑通过投影点创建的附件线，" 第59.7.3节](pt06ch59s03hlb03.md)。
