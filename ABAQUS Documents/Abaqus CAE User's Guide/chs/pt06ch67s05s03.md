# 67.5.3 显示或隐藏单个分量向量







当自由体切割配置为在 1、2 或 3 方向上显示单个分量向量而不是合向量时，您可以显示或隐藏这些单个分量向量。这些设置针对力向量和力矩向量独立提供；例如，您可以隐藏 3 方向上力的分量向量，但显示同方向上力矩的分量向量。

切换单个分量向量的显示不会更改合向量显示。合向量仍使用所有三个分量显示。

**按分量自定义向量显示：**

1. 找到 **Color & Style**（颜色和样式）选项。在可视化模块中从主菜单栏中选择 ****Options**![](../graphics/images/arrow.gif)**Free Body****（选项→自由体），然后在出现的对话框中点击 **Color & Style**（颜色和样式）选项卡。
2. 分别点击 **Force**（力）选项卡或 **Moment**（力矩）选项卡来显示或隐藏力向量或力矩向量的单个分量向量。**Components**（分量）选项出现在页面中间。
3. 切换开启 **1**、**2** 或 **3** 选项以将所选分量包含在向量显示中。
4. 点击 **OK**（确定）以实施您的更改并关闭 **Free Body Plot Options**（自由体绘图选项）对话框。默认情况下，您的更改会在会话期间保存，并将影响该视口中所有后续的自由体切割图。如果您想在后续会话中保留您的更改，请将它们保存到文件。有关更多信息，请参阅["保存自定义设置以供后续会话使用，" 第 55.1.1 节"](pt05ch55s01s01.md)。
