# 16.11.10 定义材料分配场





材料分配场用于指定欧拉部件实例初始状态中不同材料的几何组成。可以分配的可用水材料取决于分配给该部件的欧拉截面中定义的材料实例。有关欧拉部件材料分配的概述，请参见["为欧拉部件实例分配材料"章节第28.4节](pt04ch28s04.md)。

模型中的每个欧拉部件实例只能有一个材料分配场应用于它。相反，一个材料分配场只能应用于一个欧拉部件实例；如果您的模型中有多个欧拉部件实例，则必须为每个实例创建一个材料分配场。

材料通过选择欧拉部件实例的区域，然后指定该区域中存在的材料实例来分配。每个材料实例被分配一个百分比，表示该材料实例在所选区域中占据的体积分数。如果区域内材料实例的百分比加起来不等于100%，则区域中的剩余体积分数由没有材料属性的空隙占据。默认情况下，未明确指定材料分配的区域也会被分配空隙。有关Abaqus如何解释材料体积分数的详细讨论，请参阅["欧拉分析"章节，《Abaqus Analysis User's Guide》第14.1.1节](../usb/usb-link.md#usb-anl-aeuleriananalysis)。

**创建或编辑材料分配场的步骤：**

1. 使用以下方法之一显示材料分配场编辑器： - 要创建新的材料分配场，请按照["创建预定义场"章节第16.8.3节](pt03ch16s08hlb03.md)中概述的程序进行操作（**类别**：`其他`；**所选步骤的类型**：`材料分配`）。您只能在初始步骤期间创建材料分配场。 - 要使用菜单或管理器编辑现有的材料分配场，请参见["编辑依赖于步骤的对象"章节第3.4.13节](pt01ch03s04hlb05.md)。您只能在初始步骤期间修改材料分配场，并且不能编辑应用材料分配场的区域。
2. 指定用于定义每个材料实例体积分数的方法： - 选择**均匀**来定义在每个区域上均匀的体积分数。 - 选择**离散场**来使用离散场定义体积分数。离散场可用于定义在区域内空间变化的体积分数。
3. 为材料分配选择一个区域。该区域必须包含在步骤1中选择的部件实例内。如果欧拉部件由单个单元组成（没有内部划分），Abaqus/CAE自动选择整个部件。 1. 在**体积分数**表的**区域**列中单击鼠标按钮3，并从出现的菜单中选择**编辑区域**。或者，您可以在**区域**列中双击鼠标按钮1。 2. 使用以下方法之一在部件实例中选择单元： - 在视口中选择和取消选择单元。完成选择单元后，单击鼠标按钮2。（有关更多信息，请参见[第6章，"在视口中选择对象"](pt01ch06.md)。） - 要从现有集合列表中进行选择，请在提示区域右侧单击**集合**，在**区域选择**对话框中选择感兴趣的集合，然后单击**继续**。**注意：**默认选择方法基于您最近使用的选择方法。要恢复为另一种方法，请在提示区域右侧单击**在视口中选择**或**集合**。
4. **体积分数**表中的其余列对应于欧拉截面中的材料实例。材料实例名称被附加到列标题中的部件实例名称之后（例如，标记为**Bottle-1.Water**的列对应于名为`Water`的材料实例在名为`Bottle-1`的部件实例中）。如果您正在定义均匀体积分数，则还会出现**空隙**列；您不能编辑**空隙**列中的值。在与每个材料实例对应的列中，输入该区域的适当体积分数： - 如果您正在定义均匀体积分数，请以小数形式输入每个材料实例的体积分数作为百分比。例如，输入`0.45`表示45%的体积分数。大于`1`的值会自动重置为`1`。要将所有材料实例平均分配给一个区域，请在**体积分数**表中的任意位置单击鼠标按钮3，并从出现的菜单中选择**均等化行**。**注意：**当您输入材料体积分数时，Abaqus/CAE自动调整**空隙**列中的值，使表行中的所有值加起来等于1。如果输入值的总和大于1，Abaqus/CAE自动将**空隙**设置为零，并对其余值进行归一化，使它们加起来等于1。 - 如果您正在使用离散场定义体积分数，请单击材料实例列中的位置；然后单击出现的箭头并从可用离散场列表中选择一个名称。如果离散场尚未定义，请在**体积分数**表中的任意位置单击鼠标按钮3，并从出现的菜单中选择**创建离散场**以访问离散场工具集（请参见["创建离散场"章节第63.2节](pt06ch63hla01.md)）。离散场定义中的体积分数应以小数形式指定为百分比；例如，输入`0.45`表示45%的体积分数。不在所选区域内的元素在离散场定义中的数据将被忽略。**警告：**由于离散场依赖于网格，因此对欧拉部件网格的后续修改将使基于离散场的材料分配失效。
5. 要清除一行中的所有材料体积分数值，请在行中的任意位置单击鼠标按钮3，并从出现的菜单中选择**清除行**。
6. 要在部件实例的另一区域中定义材料体积分数，请在**体积分数**表中单击鼠标按钮3，并从出现的菜单中选择**添加行**。Abaqus/CAE向**体积分数**表添加新行。重复上述步骤来定义新区域及其相关的材料体积分数。
7. 将材料实例适当分配给部件实例的所有区域后，单击**确定**保存数据并退出材料分配场编辑器。如果材料实例未分配给部件实例的某些区域，则这些区域最初被分配为没有材料属性的空隙。
