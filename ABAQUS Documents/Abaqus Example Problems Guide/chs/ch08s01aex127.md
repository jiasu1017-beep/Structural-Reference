# 8.1.1 氢在容器壁截面中的扩散







**产品：** Abaqus/Standard

这个一维问题为Abaqus中的质量扩散功能提供了简单的验证。Abaqus中使用的非耦合质量扩散 formulations 在 ["质量扩散分析，" Abaqus Analysis User's Guide 第6.9.1节](../usb/usb-link.md#usb-anl-amassdiffusion) 和 ["质量扩散分析，" Abaqus Theory Guide 第2.13.1节](../stm/stm-link.md#stm-anl-massdiffusion) 中描述。

这里考虑的物理问题是关于一个压力容器壳壁，由2 1/4 Cr-1 Mo钢合金基体金属制成，内覆层为347型不锈钢堆焊层。这些容器通常在高温和高压条件下使用。在这样的工作条件下，氢溶解到合金中（Fujii等人，1982），在冷却过程中可能导致堆焊层从基体金属剥离，甚至可能由于氢脆导致基体金属中的裂纹萌生和扩展。本例关注的是问题的氢扩散方面。

### 问题描述

问题如图8.1.1-1所示（[ch08s01aex127.md#sxmh2odiff-wallsection](ch08s01aex127.md#sxmh2odiff-wallsection)），由200毫米厚的基体金属和5毫米厚的堆焊金属组成的容器壁截面。问题是一维的，唯一的梯度是通过壁厚。分析的目的是预测在停机冷却过程中氢浓度沿壁厚的演变。

### 几何与模型

由于问题是一维的，我们使用平面网格，在*y*方向上只有一个单元（参见图8.1.1-2）。网格进行了加密，在两种材料的界面附近布置更多单元，因为我们预计此处的浓度梯度会非常高。

壁体初始温度均匀为727.5 K（454.4 C），在21.5小时内以恒定速率冷却至298.15 K（25.0 C）。

边界条件如下。在初始稳态条件下，堆焊金属外侧的氢浓度为35.85 ppm，对应归一化浓度为0.1225 N1/2mm1。归一化浓度作为主要求解变量使用（在整个离散化域上连续），定义为浓度除以溶解度。基体金属外侧的氢浓度为零。当冷却期开始时，堆焊金属外侧的氢浓度假定瞬间降为零。

### 时间步长

问题分两部分运行。第一部分是一个步骤，执行单次增量的稳态非耦合质量扩散分析，使用任意时间步长建立对应于初始温度的初始稳态氢浓度分布。

然后在四个后续的质量扩散瞬态分析步骤中分析冷却期间的氢扩散，使用自动时间步长。这不必分成四个单独的步骤。我们这里这样做是因为Fujii等人（1982）给出的结果（在我们的Abaqus结果中进行了比较）是在瞬态过程中四个特定时刻给出的：2.7 h（673.15 K，400.0 C）、5.2 h（623.15 K，350.0 C）、10.2 h（523.15 K，250.0 C）和21.5 h（298.15 K，25.0 C）。

瞬态分析步骤（冷却发生期间）时间积分的精度由DCMAX参数控制。该参数指定每个时间步长的允许归一化浓度变化。即使在这样的线性问题中，DCMAX也控制解的精度，因为时间积分算子不是精确的（使用后差分规则）。这里选择DCMAX为0.01 N1/2mm1，这是一个非常严格的值。这是必要的，因为材料的溶解度随着温度降低而显著降低（基体金属中降低超过两个数量级），因此对于给定的归一化浓度变化，浓度的变化变得更大。

瞬态扩散问题中的一个重要问题是初始时间步长的选择。与任何瞬态问题一样，空间单元尺寸和时间步长相关，以至于小于某一尺寸的时间步长可能导致解中出现伪振荡，因此不会提供有用的信息。这种空间和时间近似的耦合在扩散问题的开始阶段总是最明显的，特别是在边界值规定变化之后。对于质量扩散情况，选择初始时间增量的建议准则（参见 ["质量扩散分析，" Abaqus Analysis User's Guide 第6.9.1节](../usb/usb-link.md#usb-anl-amassdiffusion)）是

### 结果与讨论

图8.1.1-3（[ch08s01aex127.md#sxmh2odiff-dist-weld](ch08s01aex127.md#sxmh2odiff-dist-weld)）显示了初始稳态条件和冷却期间四个不同时间点的堆焊金属中的氢浓度分布。图8.1.1-4（[ch08s01aex127.md#sxmh2odiff-dist-base](ch08s01aex127.md#sxmh2odiff-dist-base)）显示了基体金属中相应的氢浓度分布。这些结果与Fujii等人（1982）给出的结果进行了很好的比较，此处未绘制，因为它们与Abaqus结果几乎无法区分。

可以观察到，尽管主要求解变量（归一化浓度）在瞬态过程中在材料界面上保持连续，但氢浓度在界面上变得越来越不连续。在冷却过程中，基体金属中的氢浓度降低，而堆焊金属中的氢浓度显著增加，在界面处的堆焊金属侧达到峰值。
