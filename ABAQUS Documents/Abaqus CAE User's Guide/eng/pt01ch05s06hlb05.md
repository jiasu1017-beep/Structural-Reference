# 5.6.5 Zooming in to a selected area of the view







Use the box zoom tool ![](../graphics/ico_viewZoomRectangle.png) from the **View Manipulation** toolbar to enlarge the view so that a selected area fills the viewport. If the current viewport is linked to other viewports, Abaqus/CAE enlarges the view in the linked viewports by the same factor. For more information, see ["Linking viewports for view manipulation," Section 4.5](pt01ch04s05.md).

**To zoom in to a selected area of the view:**

1. From the **View Manipulation** toolbar, click the box zoom tool ![](../graphics/ico_viewZoomRectangle.png) to enter zoom mode. **Tip:**You can also select ****View**![](../graphics/images/arrow.gif)**Box Zoom**** from the main menu or press **[F5]**.
2. Position the cursor in the viewport whose view you want to change. The cursor shape changes to crosshairs.
3. Position the cursor at one corner of the area to be enlarged.
4. Drag the cursor to the opposite corner. A rectangle indicates the area to be enlarged.
5. Release mouse button 1. The area defined by the rectangle enlarges to fill the viewport.
6. Repeat Steps 2 through 5 as many times as necessary to achieve the desired view.
7. To exit box zoom mode, do one of the following: - Click mouse button 2. - Click the cancel button ![](../graphics/afxI_cancel.png) in the prompt area. - Click the box zoom tool. - Click any other view manipulation tool. **Tip:**Use the [cycle](pt01ch05s06hlb07.md) view manipulation tool to return to the previous view.
