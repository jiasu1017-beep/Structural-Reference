# 82.7 Displaying the history of the adaptive remeshing error indicator







The adaptivity plotter plug-in allows you to review the history of selected error indicators and the element count in a sequence of adaptive remeshing jobs. 

**Note:**All of the Abaqus plug-ins are stored in `*abaqus_dir*\code\python\lib\abaqus_plugins`, where *abaqus_dir* is the Abaqus parent directory.
