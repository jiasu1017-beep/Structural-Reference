# 45.2 符号图选项概述





您可以使用符号图选项来自定义符号图的外观。对于来自输出数据库的数据，使用**Field Output**对话框中的[**Symbol Variable**](pt05ch42s03hlb05.md)选项来选择要绘制的向量或张量量以及可选分量。

**注意：**如果您从当前模型数据库中选择模型，符号图显示主场变量，而不是符号变量。除非您从输出数据库绘制数据，否则更改符号变量无效。

从主菜单栏中选择****Options**![](../graphics/images/arrow.gif)**Symbol****，或单击工具箱中的![](../graphics/ico_plotSymbolOptions.png)访问**Symbol Plot Options**对话框。单击以下选项卡来自定义当前视口中符号图的外观：
- **Color & Style**：**Color & Style**页面包含以下选项卡：
  - **Vector**：选择自定义表示向量变量合量或所选分量值的箭头外观。
  - **Tensor**：选择自定义表示张量变量所有主分量、所有直接分量或所选分量的箭头外观。
- **Limits**：**Limits**页面包含以下选项卡：
  - **Vector**：选择指定最小和最大向量值或让Abaqus/CAE自动计算这些值。
  - **Tensor**：选择指定最小和最大张量值或让Abaqus/CAE自动计算这些值。
- **Labels**：**Labels**页面包含以下选项卡：
  - **Vector**：选择显示向量值的数值标签，并自定义向量值的颜色、字体、类型大小、数字格式和小数位数。
  - **Tensor**：选择显示张量值的数值标签，并自定义张量值的颜色、字体、类型大小、数字格式和小数位数。

渲染样式、边缘可见性、变形比例因子和透明度等其他选项位于[**Common Plot Options**](pt05ch43hla01.md)对话框中。如果您选择在未变形和变形形状上绘制符号，可以使用[**Superimpose Plot Options**](pt05ch43hla02.md)来自定义未变形形状的外观。要了解如何自定义符号图的渲染样式和其他显示特征，请参见[第55章，"自定义绘图显示"](pt05ch55.md)。有关张量和场输出变量的更多信息，请参见["场输出变量选择概述，" 第42.5.1节](pt05ch42s03hlb01.md)。有关选择结果步骤的更多信息，请参见["选择特定结果步骤和帧，" 第42.3.1节](pt05ch42s01hlb01.md)。
