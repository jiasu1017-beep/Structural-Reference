# 9.7.1 创建新模型数据库





您可以在计算机上创建和存储多个模型数据库，但一次只能打开一个模型数据库。选择以下选项之一来创建新的模型数据库：
- 单击**文件**工具栏中的![](../graphics/ico_fileNew.png)或选择****文件（File）**![](../graphics/images/arrow.gif)**新建模型数据库（New Model Database）**![](../graphics/images/arrow.gif)**使用Standard/Explicit模型（With Standard/Explicit Model）****来为Abaqus/Standard或Abaqus/Explicit分析创建新的模型数据库。
- 选择****文件（File）**![](../graphics/images/arrow.gif)**新建模型数据库（New Model Database）**![](../graphics/images/arrow.gif)**使用CFD模型（With CFD Model）****来为Abaqus/CFD分析创建新的模型数据库。
- 选择****文件（File）**![](../graphics/images/arrow.gif)**新建模型数据库（New Model Database）**![](../graphics/images/arrow.gif)**使用电磁模型（With Electromagnetic Model）****来为电磁分析创建新的模型数据库。

如果您对当前模型数据库做了任何更改，Abaqus/CAE会在关闭当前模型数据库并创建新模型数据库之前询问您是否要保存更改。然后新数据库将成为当前数据库。要保存新的模型数据库，请从主菜单栏选择****文件（File）**![](../graphics/images/arrow.gif)**保存（Save）****并输入数据库的名称。保存模型数据库后，Abaqus/CAE会在主窗口的标题栏中显示其名称。
