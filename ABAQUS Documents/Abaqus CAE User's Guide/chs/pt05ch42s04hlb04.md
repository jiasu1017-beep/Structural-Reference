# 42.6.8 将结果变换到新坐标系

默认情况下，Abaqus/CAE在预处理期间定义的坐标系中显示基于单元的场输出结果，在全局坐标系中显示基于节点的结果。如果您在预处理期间定义了节点变换，可以选择将这些变换应用于基于节点的结果。或者，您可以选择将基于单元和基于节点的结果都变换到用户指定的坐标系，或对基于坐标和距离的节点矢量结果应用角度变换。

对于位移、速度和加速度等矢量量（源自节点坐标），Abaqus/CAE使用最终量（保存的矢量）而不是原始坐标将结果变换到请求的坐标系。例如，圆柱位移分量是通过沿当前*r*-或![](../graphics/usi_eqn00065.gif)方向投影位移矢量来计算的。相比之下，取变换节点坐标之间的差分将产生不同的结果。

当您将结果变换到局部坐标系时，Abaqus/CAE会旋转全局直角坐标系，以使其在感兴趣的点处与局部系统对齐。变换后的结果在旋转后的全局直角坐标中报告。

对于到用户指定坐标系的变换，默认情况下Abaqus/CAE在变形效果可用时在变换中包含当前变形的影响。如果您希望变换计算仅考虑未变形状态，可以排除这些效果。变形效果不会缩放；Abaqus/CAE使用变形比例因子1.0执行这些计算。在变换中包含变形效果可以改变基于节点的坐标系统的方向、壳和膜单元上坐标系的投影，以及位置相关的圆柱和位置相关的球面坐标系统的方向。

当您选择用户指定的坐标系进行变换时，您还可以调整主变量结果或主变量和变形变量两者的结果的显示，以考虑坐标系的刚体变换。显示考虑刚体变换的变形使您能够可视化模型相对于移动的、用户定义的坐标系的相对位移。有关的详细信息选择结果变量，请参阅["选择主场输出变量，" 第42.5.3节](pt05ch42s03hlb03.md)和["选择变形场输出变量，" 第42.5.4节](pt05ch42s03hlb04.md)。

您可以对基于坐标和距离的节点矢量结果应用角度变换。角度变换计算圆柱坐标系的*R*、![](../graphics/usi_eqn00065.gif)和*Z*分量，以及球面坐标系的*R*、![](../graphics/usi_eqn00065.gif)和![](../graphics/usi_eqn00226.gif)分量。

您可以对包含场输出变量SORIENT输出的结果和复合截面应用铺层方向变换。此变换使用每个单独铺层上单元的方向而不是整个复合铺层的单一方向来计算张量和矢量场。

**将结果变换到新坐标系：**

1. 找到"Transform Type"选项。从主菜单栏中，选择****Result**![](../graphics/images/arrow.gif)**Options****；然后在出现的对话框中单击**Transformation**标签。将出现"Transform Type"选项。
2. 选择要用于结果的变换类型。
   - 选择**默认**以使用为模型定义的默认坐标系。节点结果显示在全局系统中。基于单元的结果显示在为模型定义的局部方向中；如果未定义，则使用全局系统。二维连续体单元、壳单元和膜单元使用全局系统或局部方向的投影。
   - 选择**节点**以考虑在节点处定义的局部方向。基于单元的结果显示在默认坐标系（如上所述）中。模型的节点变换应用于节点结果；如果未定义，则使用全局系统。
   - 选择**用户指定**将整个模型的结果变换到指定坐标系。要限制列表以便更轻松地选择，请在**Name filter**字段中键入筛选模式，然后按**[Enter]**键应用筛选器。从出现的坐标系列表中选择要应用的变换，或单击![](../graphics/ico_selectBlue.png)并从视口中选择坐标系。在模型生成或后处理期间定义的坐标系都可用。带有星号标记的系统已保存到当前输出数据库。
   - 选择**角度**以对基于坐标和距离的节点矢量结果进行变换，结果以*R*、![](../graphics/usi_eqn00065.gif)和*Z*或*R*、![](../graphics/usi_eqn00065.gif)和![](../graphics/usi_eqn00226.gif)表示。从出现的圆柱和球面坐标系列表中选择要应用的变换，或单击![](../graphics/ico_selectBlue.png)并从视口中选择坐标系。在模型生成或后处理期间定义的坐标系都可用。带有星号标记的系统已保存到当前输出数据库。
   - 选择**铺层方向**将张量和矢量场变换为复合截面定义中定义的铺层方向。您的输出数据库必须包含场输出变量SORIENT才能执行此变换。
3. 如果您选择了用户指定的坐标系，您还可以在变换中包含变形的影响。切换打开**包括变形的影响（当可用时）**以包括变形的影响，或切换关闭此选项以排除变形的影响，并在变换中仅考虑未变形状态。
4. 如果您选择了用户指定的坐标系，您还可以将刚体变换应用于主变量和变形变量的显示。从**刚体变换**选项中，执行以下操作之一：
   - 切换打开**主变量**以将刚体变换应用于主变量结果。
   - 切换打开**主变量**和**变形变量**以将刚体变换应用于两个变量的结果。
5. Abaqus在当前视口中以指定坐标系或多个坐标系显示结果。角度分量以弧度给出。角度变换分量结果用(AT: *CSYS-name*)表示，刚体变换分量结果用(RT: *CSYS-name*)表示，以区别于使用默认变换类型获得的结果。默认情况下，您的更改会在会话期间保存，并会影响后续的所有结果显示。如果您希望将更改保留到后续会话中，请将其保存到文件。有关的详细信息，请参阅["保存自定义设置以供后续会话使用，" 第55.1.1节](pt05ch55s01s01.md)。
