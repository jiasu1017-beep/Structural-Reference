# 55.2.2 自定义透明度





透明度选项控制模型的透明程度,它们适用于除线框外的所有渲染样式中的曲线图。透明度是曲线图状态无关的;也就是说,您为所有未变形、变形、云图、符号和材料取向曲线图设置一次。您也可以设置Abaqus/CAE在变形形状叠加在未变形形状上时将应用于未变形形状的透明度。

对于着色符号和材料取向曲线图,如果打开透明度,则更容易查看位于模型内部的符号曲线图箭头和材料取向三轴。

**自定义透明度的步骤:**

1. 找到公共或叠加"**Translucency**"选项。公共透明度设置始终应用于变形形状;当在任何曲线图状态(未变形、变形、云图、符号或材料取向)中单独绘制未变形形状时,它也适用于未变形形状。当未变形形状与变形形状一起绘制时,应用叠加透明度样式。从主菜单栏,选择 ****Options**![](../graphics/images/arrow.gif)**Common**** 或 **Superimpose**。在出现的对话框中点击"**Other**"选项卡;然后点击"**Translucency**"选项卡。"**Translucency**"选项出现。
2. 切换打开"**Apply translucency**"以激活透明度选项。
3. 选择透明度百分比。拖动滑块到您想要的透明度百分比。值为.00表示透明绘制;1.00表示不透明绘制。默认透明度是.30。
4. 点击"**Apply**"实施您的更改。透明度会根据您的规格进行更改。默认情况下,您的更改会在会话期间保存,并会影响当前视口和从当前视口创建的任何新视口中的所有后续曲线图。如果要为后续会话保留您应用的更改,请将它们保存到文件。更多信息,请参阅 ["为后续会话保存自定义," 第55.1.1节](pt05ch55s01s01.md)。
