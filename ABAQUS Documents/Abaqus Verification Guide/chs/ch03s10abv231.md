### 测试的功能

在稳态分析期间移除和添加连续体耦合温度-位移单元。

### 问题描述

**模型：**

模型在*x*–*y*平面中的尺寸为5.0×2.0，平面外尺寸为1.0。在轴对称情况下，模型在*r*–*z*平面中的尺寸为2.0×5.0，内半径![](../graphics/ver_eqn00176.gif)等于10^5。内半径很大以确保周向应变大致均匀，这允许将分析中获得的结果与分析获得的解进行比较。

**材料：**

| 导热率 | 7.872×10^4 |
| --- | --- |
| 密度 | 0.2829 |
| 热膨胀系数 | 1.0×10^-6 |
| 弹性模量 | 100×10^4 |
| 泊松比 | 0.25 |

**载荷和边界条件：**

模型的左侧保持在![](../graphics/ver_eqn00001.gif)=0.0。模型的右侧存在膜条件。环境温度为![](../graphics/ver_eqn01255.gif)=100.0，膜系数*h*为1.0。获得稳态解后，移除模型中的一些单元。沿新外部边界的温度保持固定。移除的单元在最后一步中重新添加到模型中，在右侧施加新的膜条件。新的环境温度为![](../graphics/ver_eqn01255.gif)=200.0，使用相同的*h*。

在所有三个步骤中保持以下机械边界条件：在*y*=0的所有点上![](../graphics/ver_eqn00272.gif)=0.0；在点(0,0)处![](../graphics/ver_eqn00107.gif)=0.0。
