## 4.4. cuBLASXt API 数学函数参考

在本章中，我们描述 cuBLASXt API 所支持的实际线性代数例程。我们将使用缩写 `<type>` 表示类型，使用 `<t>` 表示相应的简短类型，以便更简洁清晰地展示所实现的函数。除非另有说明，`<type>` 和 `<t>` 具有以下含义：

| <type> | <t> | 含义 |
| --- | --- | --- |
| float | 's' 或 'S' | 实数单精度 |
| double | 'd' 或 'D' | 实数双精度 |
| cuComplex | 'c' 或 'C' | 复数单精度 |
| cuDoubleComplex | 'z' 或 'Z' | 复数双精度 |

缩写 \(\mathbf{Re}(\cdot)\) 和 \(\mathbf{Im}(\cdot)\) 分别表示一个数的实部和虚部。由于实数不存在虚部，我们将其视为零，通常可以简单地从使用它的方程中忽略。同样，\(\bar{\alpha}\) 将表示 \(\alpha\) 的复共轭。

通常在整个文档中，小写希腊符号 \(\alpha\) 和 \(\beta\) 表示标量，斜体小写英文字母 \(\mathbf{x}\) 和 \(\mathbf{y}\) 表示向量，大写英文字母 \(A\)、\(B\) 和 \(C\) 表示矩阵。

```c++
cublasStatus_t cublasXtDgemm(cublasXtHandle_t handle,
                           cublasOperation_t transa, cublasOperation_t transb,
                           int m, int n, int k,
                           const double          *alpha,
                           const double          *A, int lda,
                           const double          *B, int ldb,
                           const double          *beta,
                           double          *C, int ldc)
cublasStatus_t cublasXtCgemm(cublasXtHandle_t handle,
                           cublasOperation_t transa, cublasOperation_t transb,
                           int m, int n, int k,

                           const cuComplex       *A, int lda,
                           const cuComplex       *B, int ldb,

                           const cuDoubleComplex *B, int ldb,

```

此函数执行矩阵-矩阵乘法

\(C = \alpha\text{op}(A)\text{op}(B) + \beta C\)

其中 \(\alpha\) 和 \(\beta\) 是标量，\(A\)、\(B\) 和 \(C\) 是按列主序存储的矩阵，维度分别为 \(\text{op}(A)\) \(m \times k\)、\(\text{op}(B)\) \(k \times n\) 和 \(C\) \(m \times n\)。同样，对于矩阵 \(A\)：

\(\text{op}(A) = \left\{ \begin{matrix}
A & {\text{if }\textsf{transa == CUBLAS\_OP\_N}} \\
A^{T} & {\text{if }\textsf{transa == CUBLAS\_OP\_T}} \\
A^{H} & {\text{if }\textsf{transa == CUBLAS\_OP\_C}} \\
\end{matrix} \right.\)

而 \(\text{op}(B)\) 对矩阵 \(B\) 的定义类似。

| 参数 | 内存 | 输入/输出 | 含义 |
| --- | --- | --- | --- |
| handle |  | 输入 | cuBLASXt API 上下文的句柄。 |
| transa |  | 输入 | 运算 op(A)，为非转置或（共轭）转置。 |
| transb |  | 输入 | 运算 op(B)，为非转置或（共轭）转置。 |
| m |  | 输入 | 矩阵 op(A) 和 C 的行数。 |
| n |  | 输入 | 矩阵 op(B) 和 C 的列数。 |
| k |  | 输入 | op(A) 的列数和 op(B) 的行数。 |
| alpha | 主机 | 输入 | 用于乘法的 <type> 标量。 |
| A | 主机或设备 | 输入 | <type> 数组，维度为 lda x k，当 transa == CUBLAS_OP_N 时 lda >= max(1, m)，否则为 lda x m，lda >= max(1, k)。 |
| lda |  | 输入 | 用于存储矩阵 A 的二维数组的前导维度。 |
| B | 主机或设备 | 输入 | <type> 数组，维度为 ldb x n，当 transb == CUBLAS_OP_N 时 ldb >= max(1, k)，否则为 ldb x k，ldb >= max(1, n)。 |
| ldb |  | 输入 | 用于存储矩阵 B 的二维数组的前导维度。 |
| beta | 主机 | 输入 | 用于乘法的 <type> 标量。如果 beta == 0，则 C 不必是有效输入。 |
| C | 主机或设备 | 输入/输出 | <type> 数组，维度为 ldc x n，ldc >= max(1, m)。 |
| ldc |  | 输入 | 用于存储矩阵 C 的二维数组的前导维度。 |

该函数执行对称压缩矩阵-矩阵乘法


$C = \left\{ \begin{matrix}
{\alpha AB + \beta C} & {\text{if }\textsf{side == CUBLAS\_SIDE\_LEFT}} \\
{\alpha BA + \beta C} & {\text{if }\textsf{side == CUBLAS\_SIDE\_RIGHT}} \\
\end{matrix} \right.$


其中 $A$ 是以压缩格式存储的 $n \times n$ 对称矩阵，$B$ 和 $C$ 是 $m \times n$ 矩阵，$\alpha$ 和 $\beta$ 是标量。

\end{matrix} \right.$
