# 5.6.1 居中视图





在视口中单击鼠标按钮 3 以访问居中视图的选项。你也可以使用**视图操作**工具栏中的自动拟合工具 ![](../graphics/ico_viewFit.png) 来快速平移和放大或缩小视图，使视图填充视口并在视口中居中。

**要居中视图：**

1. 将光标定位在视口中要用于居中视图的位置，然后单击鼠标按钮 3。
2. 从出现的菜单中，选择**居中视图**。Abaqus/CAE 会将你选择的位置移动到视口中心。
