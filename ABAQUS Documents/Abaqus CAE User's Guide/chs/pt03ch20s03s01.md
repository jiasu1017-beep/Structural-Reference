# 20.3.1 独立草图







您可以使用草图模块创建和管理与特征无关的草图；这些草图称为独立草图。独立草图存储在模型中；与模型中的其他对象（如零件和载荷）一样，独立草图可以被编辑、复制、重命名和删除。要创建独立草图，请点击草图模块工具箱中的草图创建工具![](../graphics/ico_sketch.png)。

当您使用 Sketcher 时（例如，在零件模块中绘制实体拉伸的轮廓时），您可以通过点击 Sketcher 工具箱中的草图保存工具![](../graphics/ico_sketchSaveAs.png)将您的工作保存为独立草图。同样，您可以通过点击 Sketcher 工具箱中的草图检索工具![](../graphics/ico_sketchAdd.png)将一个独立草图合并到另一个草图中。当您检索独立草图时，Abaqus/CAE 会执行以下操作：
- 定位检索的草图，使其原点与当前草图的原点重合。
- 调整视口大小以显示当前草图和检索的草图。
- 询问您是否要从默认大小和位置对检索的草图进行平移、旋转、镜像或缩放。
