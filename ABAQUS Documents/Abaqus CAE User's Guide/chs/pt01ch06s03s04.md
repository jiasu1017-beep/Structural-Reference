# 6.3.4 在选择之前高亮对象





当你从视口中选择对象并停止移动光标时，Abaqus/CAE 会高亮显示在光标位置将被选择的对象。这种行为称为"预选"，允许你在进行选择之前准确看到将选择哪个对象。如果当前选择选项使多个对象可用，Abaqus/CAE 会更改光标，在箭头右侧添加省略号 (...)，并高亮显示所有可能的选择。你为选择选择的位置和类型过滤也适用于预选。

在**选择**工具栏中关闭**允许拾取期间的预选**工具 ![](../graphics/ico_filterPreselect.png) 可以关闭当前会话中过程的预选高亮。此工具默认是开启的。

**注意：**对于大型模型，预选高亮可能会有延迟；关闭预选可能会提高显示速度。
