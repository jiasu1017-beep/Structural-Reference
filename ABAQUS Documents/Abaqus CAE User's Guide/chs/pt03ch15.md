# 15 相互作用模块





您可以使用「相互作用」模块来定义和管理以下对象：
- 模型区域之间或模型区域与其周围环境之间的机械和热相互作用。
- Abaqus/Standard到Abaqus/Explicit共仿真接口区域和耦合方案。
- 流固耦合仿真（介于Abaqus/CFD与Abaqus/Standard或Abaqus/Explicit之间）的接口区域和耦合步骤周期。
- 模型区域之间的分析约束。
- 装配级线特征、连接器截面以及将连接器截面分配给模型连接器。
- 模型区域上的惯性（点质量、旋转惯性和热容）。
- 模型区域上的裂纹。
- 模型两点之间或模型点与地面之间的弹簧和阻尼器。
