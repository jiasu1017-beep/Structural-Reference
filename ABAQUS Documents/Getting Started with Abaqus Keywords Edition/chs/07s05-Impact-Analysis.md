# 7.5 示例：动态载荷作用下的货物起重机

本示例使用的货物起重机与["示例：货物起重机，"第6.4节](ch06s04.html)中分析过的相同，但现在需要研究当10 kN的载荷在0.2秒内落到吊钩上时会发生什么情况。连接点A、B、C和D（参见[图7-5](#gss-cargo)）只能承受最大100 kN的拉拔力。需要判断这些连接是否会断裂。

![图7-5]((../graphics/gss-cargo.png))货物起重机。

载荷的持续时间很短，这意味着惯性效应可能很重要，因此必须进行动态分析。没有给出关于结构阻尼的任何信息。由于桁架和交叉支撑之间采用螺栓连接，摩擦效应引起的能量吸收可能很显著。因此，根据经验，选择每种模态的临界阻尼的5%。

施加载荷的大小与时间的关系如[图7-6](#gss-load-time-v)所示。

![图7-6]((../graphics/gss-load-time-v-nls.png))载荷-时间特性。

接下来的步骤假设您可以访问此示例的完整输入文件。该输入文件`dynamics.inp`在["货物起重机——动态载荷，"附录A.5节](ap01s05.html)中提供。获取和运行脚本的说明在[附录A，"示例文件"](ap01.html)中给出。

如果您希望使用Abaqus/CAE以交互方式创建此示例，请参阅["示例：动态载荷作用下的货物起重机，"Getting Started with Abaqus: Interactive Edition第7.5节](../gsa/gsa-link.htm#gsa-dyn-exacargodyn)。

## 7.5.1 输入文件的修改——模型数据

模型数据与静态分析相同，但进行了如下修改。这些修改最容易使用编辑器完成，尽管您也可以在预处理器中更改模型。

**材料**

在动态模拟中，必须指定每种材料的密度，以便形成质量矩阵。起重机的钢密度为7800 kg/m³。使用[*BEAM GENERAL SECTION*](../key/key-link.htm#usb-kws-mbeamgensect)选项定义了梁单元属性，因此此输入文件中没有材料属性定义。密度必须使用[*BEAM GENERAL SECTION*](../key/key-link.htm#usb-kws-mbeamgensect)选项上的`DENSITY`参数指定。例如：

```abaqus
*BEAM GENERAL SECTION, SECTION=BOX, ELSET=OUTA, DENSITY=7800.
0.10,0.05,0.005,0.005,0.005,0.005
-0.1118, 0.0, -0.9936
200.E9,80.E9
```

`DENSITY`参数已添加到所有单元属性选项。

如果使用[*MATERIAL*](../key/key-link.htm#usb-kws-mmaterial)选项定义材料数据，则通过使用[*DENSITY*](../key/key-link.htm#usb-kws-mdensity)选项并在数据行上给出质量密度来包含密度。例如：

```abaqus
*MATERIAL, NAME=STEEL
*ELASTIC
<杨氏模量>,<泊松比>
*DENSITY
<密度>,
```

**初始条件**

在此示例中，结构没有初始速度或加速度，这是默认值。但是，如果要定义初始速度，可以按以下方式操作：

```abaqus
*INITIAL CONDITIONS, TYPE=VELOCITY
```

节点（或节点集）、方向和速度大小在数据行上指定，如下所示：

```abaqus
<节点或节点集>,<自由度>, <速度>
```

例如：

```abaqus
*INITIAL CONDITIONS, TYPE=VELOCITY
NALL, 1, 10.0
```

这将节点集`NALL`中所有节点在1方向上的速度设置为10 m/s。

**载荷的时间变化**

施加到起重机顶端的载荷大小随时间变化，如[图7-6](#gss-load-time-v)所示。载荷的时间依赖性使用[*AMPLITUDE*](../key/key-link.htm#usb-kws-mamplitude)选项定义。[*AMPLITUDE*](../key/key-link.htm#usb-kws-mamplitude)选项必须作为模型数据的一部分出现，即使引用它的[*CLOAD*](../key/key-link.htm#usb-kws-hcload)选项是历史数据的一部分。

[*AMPLITUDE*](../key/key-link.htm#usb-kws-mamplitude)选项的每个数据行上给出四对时间和大小数据，并使用`NAME`参数为幅值曲线指定名称。对于您的模拟，定义幅值曲线的选项块应类似于以下内容：

```abaqus
*AMPLITUDE, NAME=BOUNCE, VALUE=RELATIVE, SMOOTH=0.25
0.0, 0.0, 0.01, 1.0, 0.2, 1.0, 0.21, 0.0
```

曲线名称`BOUNCE`将用于将载荷选项（[*CLOAD*](../key/key-link.htm#usb-kws-hcload)）引用到此幅值曲线。实际施加的载荷将是载荷选项上的大小与`BOUNCE`曲线上幅值的乘积。参数`VALUE`=`RELATIVE`用于指示此方法。您可以选择在[*AMPLITUDE*](../key/key-link.htm#usb-kws-mamplitude)选项上使用`VALUE`=`ABSOLUTE`来定义载荷的绝对大小。

## 7.5.2 输入文件的修改——历史数据

历史定义与静态分析中的历史定义完全不同。因此，删除整个静态步骤，并按如下所述添加新的历史部分。

此分析需要两个步骤。第一步计算结构的固有频率和振型。第二步然后使用这些数据计算起重机的瞬态动态响应。如果要在此模拟中对任何非线性进行建模，必须使用[*DYNAMIC*](../key/key-link.htm#usb-kws-hdynamic)过程。在此分析中，我们假设所有内容都是线性的。

**步骤1——模态和频率**

[*FREQUENCY*](../key/key-link.htm#usb-kws-hfrequency)过程用于计算固有频率和振型。Abaqus提供Lanczos和子空间迭代特征值提取方法。Lanczos方法是默认方法；当需要大量特征模态且系统具有许多自由度时，它通常更快。当只需要少量（少于20个）特征模态时，子空间迭代方法可能更快。

我们在此分析中使用默认的Lanczos特征解算器。所需的模态数在[*FREQUENCY*](../key/key-link.htm#usb-kws-hfrequency)选项的数据行上指定。或者，可以指定感兴趣的最小和最大频率，这样一旦Abaqus找到指定范围内的所有特征值，步骤就会完成。也可以指定一个偏移点，以便提取最接近偏移点的特征值。默认情况下，不使用最小或最大频率或偏移。如果结构没有相对于刚体模态进行约束，则应将偏移值设置为一个小的负值，以消除与刚体运动相关的数值问题。

[*FREQUENCY*](../key/key-link.htm#usb-kws-hfrequency)选项块的形式为：

```abaqus
*FREQUENCY
<特征值数量>,<最小频率>, <最大频率>,<偏移点>
```

此模拟的步骤和过程选项块为：

```abaqus
*STEP, PERTURBATION
Frequency extraction of the first 30 modes
*FREQUENCY
30,
```

在结构动力学分析中，响应通常与较低阶模态相关。但是，应该提取足够的模态以提供结构动态响应的良好表示。检查是否提取了足够数量的特征值的一种方法是查看每个自由度中的总有效质量，这表明每个提取模态在每个方向上有多少质量是活跃的。有效质量在数据文件中的特征值输出下以表格形式列出。理想情况下，每个方向上每个模态的模态有效质量之和应至少为总质量的90%。这在["模态数量的影响，"第7.6节](ch07s06.html)中有更详细的讨论。

**边界条件**

边界条件与静态分析中的相同。

**输出**

默认情况下，Abaqus将振型写入输出数据库（`.odb`）文件，以便可以使用Abaqus/Viewer绘制它们。每个振型的节点位移被归一化，使得最大位移为单位1。因此，这些结果以及相应的应力和应变没有物理意义：它们应该只用于相对比较。

步骤以以下内容结束：

```abaqus
*END STEP
```

**步骤2——瞬态动力学**

[*MODAL DYNAMIC*](../key/key-link.htm#usb-kws-hmodaldyn)过程用于瞬态模态动态分析。固定时间增量和平共step总时间在此选项的数据行上给出。模拟的总时间为0.5秒，固定增量为0.005秒。此数据行的格式与[*STATIC*](../key/key-link.htm#usb-kws-hstatic)基本相同。但是，在这种情况下，我们必须小心确保给出真实的时间值；在动态分析中，时间是一个真实的物理量。

此模拟的[*STEP*](../key/key-link.htm#usb-kws-hstep)和[*MODAL DYNAMIC*](../key/key-link.htm#usb-kws-hmodaldyn)选项块的形式为：

```abaqus
*STEP, PERTURBATION
Crane Response to Dropped Load
*MODAL DYNAMIC
0.005, 0.5
```

**阻尼**

应在第一步中提取的所有30个模态中使用临界阻尼的5%。此输入在以下[*MODAL DAMPING*](../key/key-link.htm#usb-kws-hmodaldamp)选项块中指定：

```abaqus
*MODAL DAMPING, VISCOUS=FRACTION OF CRITICAL DAMPING
1, 30, 0.05
```

**选择特征模态**

如果使用了[*MODAL DAMPING*](../key/key-link.htm#usb-kws-hmodaldamp)，则必须使用[*SELECT EIGENMODES*](../key/key-link.htm#usb-kws-hselecteigenmodes)选项来选择基于模态的动态过程中使用的特征模态。对于此示例，此选项的形式为：

```abaqus
*SELECT EIGENMODES, GENERATE
1, 30, 1
```

**载荷**

在节点104的负全局2方向上向起重机顶端施加集中力。[*EQUATION*](../key/key-link.htm#usb-kws-mequation)约束在节点104和204的自由度2之间，这意味着载荷将由两个节点平等地承担，从而由起重机的两半承担。集中力使用[*CLOAD*](../key/key-link.htm#usb-kws-hcload)选项定义。此示例使用参数`AMPLITUDE`=`BOUNCE`来指示应使用名为`BOUNCE`的幅值曲线（先前定义为模型数据的一部分）来定义步骤期间载荷随时间变化的大小：

```abaqus
*CLOAD, AMPLITUDE=BOUNCE
104, 2, -1.0E4
```

在任何时刻施加的载荷的实际大小是通过将[*CLOAD*](../key/key-link.htm#usb-kws-hcload)选项上的大小（-10,000 N）与该时刻`BOUNCE`幅值曲线的值相乘得到的。

**边界条件**

步骤1中应用的相同边界条件对此步骤仍然有效。由于[*FREQUENCY*](../key/key-link.htm#usb-kws-hfrequency)步骤与任何后续模态动态步骤之间不能更改边界条件，因此不应指定任何边界条件。

**输出**

动态分析通常比静态分析需要更多的增量来完成。因此，动态分析的输出量可能非常大，您应该控制输出请求以将输出文件保持在合理的大小。

您可以使用**datacheck**分析期间在数据文件底部附近给出的近似大小来估计重启文件的大小。

在此示例中，请求在每第五个增量结束时将变形形状输出到输出数据库文件。步骤中将有100个增量（0.5/0.005）；因此，将有20帧输出。

```abaqus
*OUTPUT, FIELD, FREQUENCY=5, VARIABLE=PRESELECT
```

独立尖端节点（分配给名为`TIP`的节点集）的位移和固定节点（分组到名为`ATTACH`的节点集中）的反力作为历史数据写入输出数据库文件，每个增量都写入，以便可以获得这些数据的高分辨率。在动态分析中，我们还关心模型中的能量分布以及能量采取的形式。动能作为质量运动的结果存在于模型中；应变能作为结构位移的结果存在；能量也通过阻尼耗散。我们可以输出动能（`ALLKE`）、应变能（`ALLSE`）、通过阻尼耗散的能量（`ALLVD`）、整个模型上的外力功（`ALLWK`）以及模型中的总能量平衡（`ETOTAL`）。输出请求的历史部分编写如下：

```abaqus
*NSET, NSET=TIP
104,
*OUTPUT, HISTORY, FREQUENCY=1
*NODE OUTPUT, NSET=TIP
U,
*NODE OUTPUT, NSET=ATTACH
RF,
*ENERGY OUTPUT
ALLKE, ALLSE, ALLVD, ALLWK, ETOTAL
```

步骤以以下内容结束：

```abaqus
*END STEP
```

## 7.5.3 运行分析

输入文件名为`dynamics.inp`（示例列在["货物起重机——动态载荷，"附录A.5节](ap01s05.html)中）。使用以下命令在后台运行分析：

```bash
abaqus job=dynamics
```

为确保使用了足够的模态，每个方向上的总有效质量应该是模型质量的较大部分（比如90%）。但是，模型的一些质量与被约束的节点相关联。此约束质量约为附加到约束节点的所有单元质量的三分之一，在本例中约为28 kg。因此，模型可以运动的质量为385 kg。*x*、*y*和*z*方向的有效质量分别占可运动质量的6%、98%和97%。2和3方向的总有效质量远高于之前推荐的90%；1方向的总有效质量低得多。然而，由于载荷施加在2方向，1方向的响应不显著。

数据文件不包含模态动力学步骤的任何结果，因为所有数据文件输出请求都被关闭了。

abaqus viewer odb=dynamics

**结果动画**

您将为分析结果创建动画。首先创建第三特征模态的比例因子动画。然后创建瞬态结果的时间历史动画。

**创建特征模态的比例因子动画：**

1. 从主菜单栏中，选择**动画**→**比例因子**；或者使用工具箱中的 ![](graphics/ico_animScaleFactor.png) 工具。

   Abaqus/Viewer显示第三振型，并在0到1之间的不同变形比例因子之间逐步显示。

   Abaqus/Viewer还在上下文栏的右侧显示电影播放控件。

2. 在上下文栏中，单击 ![](graphics/ico_framePause.png) 暂停动画。

**创建瞬态结果的时间历史动画：**

1. 从主菜单栏中，选择**结果**→**活动步骤/帧**以选择将在历史动画中活动的帧。

   Abaqus/Viewer显示**活动步骤/帧**对话框。

2. 切换步骤名称，使仅选择第二步（**步骤-2**）。

3. 单击**确定**接受选择并关闭对话框。

4. 从主菜单栏中，选择**动画**→**时间历史**；或者使用工具箱中的 ![](graphics/ico_animTimeHistory.png) 工具。

   Abaqus/Viewer逐步显示第二步的每个可用帧。状态块在动画过程中指示当前步骤和增量。到达此步骤的最后一个增量后，动画过程会重复自身。

5. 您可以在动画运行时自定义变形形状图。

   a. 显示**通用绘图选项**对话框。

   b. 从**变形比例因子**字段中选择**统一**。

   c. 输入`15.0`作为变形比例因子值。

   d. 单击**应用**以应用您的更改。

   e. 从**变形比例因子**字段中选择**自动计算**。

   f. 单击**确定**应用您的更改并关闭**通用绘图选项**对话框。

   Abaqus/Viewer显示选定的曲线。
