# 64.6.11 选择要编辑的单元







当您选择要编辑的单元时，Abaqus/CAE 会在提示区域显示一个菜单，允许您选择用于选择单元的技术。这些技术允许您选择单个节点或一组相邻单元。

**选择要编辑的单元：**

1. 使用以下方法之一来选择要编辑的单元：**单独选择单元** 1. 从提示区域的菜单中选择 **Individually**。2. 选择要重新编号的单元，或拖动选择一组单元。3. 按 **[Shift]**+**Click** 可将其他单元添加到您的选择中。4. 如有必要，按 **[Ctrl]**+**Click** 可从选择中移除已选的单元。**使用角度方法选择单元** 1. 从提示区域的菜单中选择 **by angle**。2. 输入角度（0 到 90），然后选择一个单元。Abaqus/CAE 会选择与所选单元相邻的所有单元面，直到相邻单元面之间的角度等于或超过您输入的角度。（有关更多信息，请参见["使用角度和特征边方法选择多个对象，" 第 6.2.3 节](pt01ch06s02hlb03.md)。）3. 使用 **by angle** 方法后，您可以按 **[Shift]**+**Click** 可将其他单元添加到选择中，或按 **[Ctrl]**+**Click** 从选择中移除已选的单元。（有关更多信息，请参见["组合选择技术，" 第 6.2.8 节](pt01ch06s02hlb08.md)。）**使用特征边方法选择单元** 1. 从提示区域的菜单中选择 **by feature edge**。2. 输入角度（0 到 90）。Abaqus/CAE 通过查找所有单元边（其中两个相邻单元面之间的角度大于指定角度）来识别模型中的所有特征边。3. 选择一个单元边。Abaqus/CAE 会沿着穿过所选单元边或节点的特征边进行追踪。如果另一个特征边以大于上一步中指定的角度与之相交，则特征边会被截断。4. Abaqus/CAE 会选择沿特征边的所有单元。（有关更多信息，请参见["使用角度和特征边方法选择多个对象，" 第 6.2.3 节](pt01ch06s02hlb03.md)。）或者，您可以选择多条特征边交点处的节点，Abaqus/CAE 会选择沿所有这些特征边的单元。5. 使用 **by feature edge** 方法后，您可以按 **[Shift]**+**Click** 可将其他单元添加到选择中，或按 **[Ctrl]**+**Click** 从选择中移除已选的单元。（有关更多信息，请参见["组合选择技术，" 第 6.2.8 节](pt01ch06s02hlb08.md)。）**指定现有单元集** 1. 点击提示区域右侧的 **Sets**。Abaqus/CAE 显示 **Region Selection** 对话框，其中包含您已创建的单元集列表。2. 选择要重新编号的单元集，然后点击 **Continue**。**注意：**默认选择方法基于您最近使用的选择方法。如需恢复为其他方法，请点击提示区域右侧的 **Select in Viewport** 或 **Sets**。
2. 完成选择单元后，从提示区域选择 **Done**。
