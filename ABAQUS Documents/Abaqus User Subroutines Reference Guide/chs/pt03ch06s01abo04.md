# 6.1.1 求解分析问题：概述







### 概述

一大类应力分析问题可以使用Abaqus/Standard和Abaqus/Explicit求解。这类问题的基本划分是静态或动态响应；动态问题是惯性效应显著的问题。Abaqus/CFD求解范围广泛的不可压缩流动问题。

分析问题历史使用Abaqus中的步骤来定义（["定义分析，" 第6.1.2节"](pt03ch06s01abo05.md)）。对于每个步骤，您选择一个分析过程，该过程定义在该步骤期间要执行的的分析类型。可用的分析过程列在下面，并在参考节中详细描述。

Abaqus使用内置完全耦合过程、顺序耦合和协同仿真作为多物理场模拟的解决方案技术，提供多物理场能力。大量额外的分析技术提供了强大的工具，可以更高效、更有效地执行Abaqus分析；请参见[第四部分，"分析技术"](pt04.md)。"

### Abaqus/Standard分析

Abaqus/Standard在区分静态和动态响应方面提供完全的灵活性；相同的分析可以包含多个静态和动态阶段。因此，可以施加静态预载荷，然后计算线性或非线性动态响应（如旋转机器部件的振动响应，或者最初在浮力和稳态水流载荷作用下移动到平衡位置的柔性海上系统，然后受到波浪载荷激励的响应）。类似地，可以在动态事件之后寻求静态解（通过在动态分析步骤之后跟随静态载荷步骤）。有关这些过程类型的详细信息，请参见["静态应力/位移分析，" 第6.2节"](pt03ch06s02.md)和["动态应力/位移分析，" 第6.3节"](pt03ch06s03.md)。除了静态和动态应力分析外，Abaqus/Standard还提供以下分析类型：
- ["稳态传输分析，" 第6.4节"](pt03ch06s04.md)
- ["热传递和热应力分析，" 第6.5节"](pt03ch06s05.md)
- ["电磁分析，" 第6.7节"](pt03ch06s07.md)
- ["耦合孔隙流体流动和应力分析，" 第6.8节"](pt03ch06s08.md)
- ["质量扩散分析，" 第6.9节"](pt03ch06s09.md)
- ["声学和冲击分析，" 第6.10节"](pt03ch06s10.md)
- ["Abaqus/Aqua分析，" 第6.11节"](pt03ch06s11.md)

### Abaqus/Explicit分析

Abaqus/Explicit使用显式直接积分程序求解动态响应问题。有关Abaqus中可用的显式动态过程的更多信息，请参见["动态应力/位移分析，" 第6.3节"](pt03ch06s03.md)。Abaqus/Explicit还提供热传递、声学和退火分析能力：有关详细信息，请参见["热传递和热应力分析，" 第6.5节"](pt03ch06s05.md)；["声学和冲击分析，" 第6.10节"](pt03ch06s10.md)；以及["退火，" 第6.12节"](pt03ch06s12.md)。

### Abaqus/CFD分析

Abaqus/CFD使用二阶投影方法求解范围广泛的不可压缩流动问题。有关Abaqus中可用的不可压缩流动过程的详细信息，请参见["流体动力学分析，" 第6.6节"](pt03ch06s06.md)。

### 多物理场分析

多物理场是一种数值方法，用于求解多个相互作用的物理域的耦合问题。Abaqus提供内置完全耦合过程、顺序耦合和协同仿真作为多物理场模拟的解决方案技术。

#### 内置完全耦合过程

原生Abaqus多物理场能力通过添加表示每个物理场的自由度并使用单一求解器来求解物理问题。Abaqus提供以下内置完全耦合过程来求解多学科模拟，其中所有物理场都由Abaqus计算：
- ["完全耦合热应力分析，" 第6.5.3节"](pt03ch06s05at19.md)
- ["耦合热电分析，" 第6.7.3节"](pt03ch06s07at22.md)
- ["完全耦合热电结构分析，" 第6.7.4节"](pt03ch06s07at23.md)
- ["压电分析，" 第6.7.2节"](pt03ch06s07at21.md)（电-机械耦合）
- ["涡流分析，" 第6.7.5节"](pt03ch06s07at24.md)（电磁）
- ["耦合孔隙流体扩散和应力分析，" 第6.8.1节"](pt03ch06s08at26.md)
- ["声学、冲击和耦合声-结构分析，" 第6.10.1节"](pt03ch06s10at29.md)
- ["欧拉分析，" 第14.1.1节"](pt04ch14s01aus90.md)

#### 顺序耦合

当模型中一个或多个物理场之间的耦合仅在一个方向上重要时，可以使用顺序耦合多物理场分析。一个常见的例子是热应力分析，其中温度场对应力场的依赖性不强。典型的顺序耦合热应力分析包括两次Abaqus/Standard运行：热传递分析和后续应力分析。

您可以按照以下节中描述的那样在Abaqus/Standard中执行顺序耦合多物理场分析：
- ["用于顺序耦合的预定义场，" 第16.1.1节"](pt04ch16s01at38.md)
- ["顺序耦合热应力分析，" 第16.1.2节"](pt04ch16s01at39.md)
- ["用于顺序耦合的预定义载荷，" 第16.1.3节"](pt04ch16s01aus97.md)

#### 协同仿真

协同仿真技术是一种多物理场能力，用于运行时耦合Abaqus和另一个分析程序。Abaqus分析可以耦合到另一个Abaqus分析或第三方分析程序，以执行多学科模拟和多域（多模型）耦合。
