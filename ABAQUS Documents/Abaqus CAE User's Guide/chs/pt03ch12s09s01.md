# 12.9.1 定义弹性

您使用**编辑材料**对话框来创建弹性材料并指定其弹性材料属性。您可以创建以下弹性材料模型：
- **弹性**；请参见["创建线性弹性材料模型"](pt03ch12s09s01.md#usi-prp-mechanical-elastic-elastic)
- **各向同性超弹性**；请参见["创建各向同性超弹性材料模型"](pt03ch12s09s01.md#usi-prp-mechanical-elastic-hyperelastic)
- **各向异性超弹性**；请参见["创建各向异性超弹性材料模型"](pt03ch12s09s01.md#usi-prp-mechanical-elastic-hyperelastic-aniso)
- **超泡沫**；请参见["创建超泡沫材料模型"](pt03ch12s09s01.md#usi-prp-mechanical-elastic-hyperfoam)
- **低密度泡沫**；请参见["创建低密度泡沫材料模型"](pt03ch12s09s01.md#usi-prp-mechanical-elastic-lowdensfoam)
- **亚弹性**；请参见["创建亚弹性材料模型"](pt03ch12s09s01.md#usi-prp-mechanical-elastic-hypoelastic)
- **多孔弹性**；请参见["创建多孔弹性材料模型"](pt03ch12s09s01.md#usi-prp-mechanical-elastic-porouselastic)
- **粘弹性**；请参见["创建粘弹性材料模型"](pt03ch12s09s01.md#usi-prp-mechanical-elastic-viscoelastic)

有关更多信息，请参见["弹性行为概述，"Abaqus分析用户指南第22.1.1节](../usb/usb-link.md#usb-mat-celasticbehav)。

### 创建线性弹性材料模型

线性弹性是Abaqus中最简单的弹性形式。线性弹性模型可以定义各向同性、正交各向异性或各向异性材料行为，适用于小弹性应变。请参见["指定弹性材料属性"]("pt03ch12s09s01.md#usi-prp-mechanical-elastic-elastic-overview)"以获取有关如何定义线性弹性材料模型的详细信息。

为线性弹性提供了失效理论。它们可用于获取后处理输出请求。以下部分描述了如何指定这些失效模型：
- ["为弹性模型定义基于应力的失效准则"](pt03ch12s09s01.md#usi-prp-mechanical-elastic-elastic-stressfailure)
- ["为弹性模型定义基于应变的失效准则"](pt03ch12s09s01.md#usi-prp-mechanical-elastic-elastic-strainfailure)

#### 指定弹性材料属性

线性弹性材料模型适用于小弹性应变（通常小于5%）；可以是各向同性、正交各向异性或完全各向异性；并且可以具有依赖于温度和/或其他场变量的属性。有关更多信息，请参见["线性弹性行为，"Abaqus分析用户指南第22.2.1节](../usb/usb-link.md#usb-mat-clinearelastic)。

**要指定弹性材料属性：**

1. 从**编辑材料**对话框的菜单栏中，选择**机械**![](../graphics/images/arrow.gif)**弹性**![](../graphics/images/arrow.gif)**弹性**。（有关显示**编辑材料**对话框的信息，请参见["创建或编辑材料，"第12.7.1节"](pt03ch12s07hlb01.md)。）
2. 从**类型**字段中，选择将用于指定弹性材料属性的数据类型。
   - 选择**各向同性**以指定各向同性弹性属性，如["在Abaqus分析用户指南线性弹性行为第22.2.1节中定义各向同性弹性"](../usb/usb-link.md#usb-mat-clinearelastic-isotropic)中所述。
   - 选择**工程常数**以通过给出工程常数来指定正交各向异性弹性属性，如["通过指定工程常数定义正交各向异性弹性"中"线性弹性行为，"Abaqus分析用户指南第22.2.1节](../usb/usb-link.md#usb-mat-clinearelastic-engconst)中所述。
   - 选择**层压板**以在平面应力中指定正交各向异性弹性属性，如["在平面应力中定义正交各向异性弹性"中"线性弹性行为，"Abaqus分析用户指南第22.2.1节](../usb/usb-link.md#usb-mat-clinearelastic-planestress)中所述。
   - 选择**正交各向异性**以直接指定正交各向异性弹性属性，如["通过指定弹性刚度矩阵项定义正交各向异性弹性"中"线性弹性行为，"Abaqus分析用户指南第22.2.1节](../usb/usb-link.md#usb-mat-clinearelastic-orthoterms)中所述。
   - 选择**各向异性**以指定各向异性弹性属性，如["定义完全各向异性弹性"中"线性弹性行为，"Abaqus分析用户指南第22.2.1节](../usb/usb-link.md#usb-mat-clinearelastic-anisotropic)中所述。
   - 选择**牵引**以用于翘曲元素的正交各向异性弹性属性，如["为翘曲元素定义正交各向异性弹性"中"线性弹性行为，"Abaqus分析用户指南第22.2.1节](../usb/usb-link.md#usb-mat-clinearelastic-orthowarp)中所述，或定义内聚元素的非耦合弹性属性，如["为内聚元素以牵引和分离方式定义弹性"中"线性弹性行为，"Abaqus分析用户指南第22.2.1节](../usb/usb-link.md#usb-mat-clinearelastic-traction)中所述。
   - 选择**耦合牵引**以为内聚元素指定耦合弹性属性，如["为内聚元素以牵引和分离方式定义弹性"中"线性弹性行为，"Abaqus分析用户指南第22.2.1节](../usb/usb-link.md#usb-mat-clinearelastic-traction)中所述。
   - 选择**剪切**以指定线性各向同性偏量材料模型。有关更多信息，请参见["状态方程中的偏量行为，"Abaqus分析用户指南第25.2.1节](../usb/usb-link.md#usb-mat-ceos-deviatoric)。
3. 要定义依赖于温度的行为数据，请打开**使用与温度相关的数据**。**数据**表中会出现一个标有**温度**的列。
4. 要定义依赖于场变量的行为数据，请点击**场变量数量**字段右侧的箭头以增加或减少场变量的数量。**场**变量列出现在**数据**表中。
5. 如果您要定义粘弹性材料的弹性行为，请点击**模量时间尺度（用于粘弹性）**字段右侧的箭头以指定长期或瞬时弹性响应。
6. 如果您想修改弹性材料响应使其不产生压缩应力，请打开**无压缩**。有关详细信息，请参见["无压缩或无拉伸，"Abaqus分析用户指南第22.2.2节](../usb/usb-link.md#usb-mat-cnocompnoten)。
7. 如果您想修改弹性材料响应使其不产生拉伸应力，请打开**无拉伸**。有关详细信息，请参见["无压缩或无拉伸，"Abaqus分析用户指南第22.2.2节](../usb/usb-link.md#usb-mat-cnocompnoten)。
8. 在**数据**表中输入材料属性。
   - 对于**各向同性**数据，输入杨氏模量，*E*，和泊松比，![](../graphics/usi_eqn00010.gif)。
   - 对于**工程常数**数据，输入主方向的广义杨氏模量，![](../graphics/usi_eqn00011.gif)，![](../graphics/usi_eqn00012.gif)，![](../graphics/usi_eqn00013.gif)；主方向的泊松比，![](../graphics/usi_eqn00014.gif)，![](../graphics/usi_eqn00015.gif)，![](../graphics/usi_eqn00016.gif)；和主方向的剪切模量，![](../graphics/usi_eqn00017.gif)，![](../graphics/usi_eqn00018.gif)，![](../graphics/usi_eqn00019.gif)。
   - 对于**层压板**数据，输入杨氏模量，![](../graphics/usi_eqn00011.gif)，![](../graphics/usi_eqn00012.gif)；泊松比，![](../graphics/usi_eqn00014.gif)；和剪切模量，![](../graphics/usi_eqn00017.gif)，![](../graphics/usi_eqn00018.gif)，![](../graphics/usi_eqn00019.gif)。![](../graphics/usi_eqn00018.gif)和![](../graphics/usi_eqn00019.gif)剪切模量用于定义壳中的横向剪切行为。
   - 对于**正交各向异性**数据，输入9个弹性刚度参数：![](../graphics/usi_eqn00020.gif)，![](../graphics/usi_eqn00021.gif)等（单位为[FL2](../popups/usb-int-iconventions-unitsym.md)）。
   - 对于**各向异性**数据，输入21个弹性刚度参数：![](../graphics/usi_eqn00020.gif)，![](../graphics/usi_eqn00021.gif)等（单位为[FL2](../popups/usb-int-iconventions-unitsym.md)）。
   - 对于**牵引**数据，您的输入取决于您建模的元素类型。
     - 对于用翘曲元素建模的实心截面Timoshenko梁单元，输入杨氏模量，![](../graphics/usi_eqn00022.gif)，和材料方向的剪切模量，![](../graphics/usi_eqn00023.gif)和![](../graphics/usi_eqn00024.gif)。
     - 对于具有非耦合牵引的内聚元素，输入法向和两个局部剪切方向的弹性模量，![](../graphics/usi_eqn00025.gif)，![](../graphics/usi_eqn00026.gif)，和![](../graphics/usi_eqn00027.gif)。
   - 对于**耦合牵引**数据，输入六个弹性模量：![](../graphics/usi_eqn00025.gif)，![](../graphics/usi_eqn00026.gif)，![](../graphics/usi_eqn00027.gif)，![](../graphics/usi_eqn00028.gif)，![](../graphics/usi_eqn00029.gif)，和![](../graphics/usi_eqn00030.gif)。
   - 对于**剪切**数据，输入**剪切模量**。
9. 如果需要，要定义材料的平面应力正交各向异性失效准则，请点击**子选项**。有关详细信息，请参见以下章节：
   - ["为弹性模型定义基于应力的失效准则"](pt03ch12s09s01.md#usi-prp-mechanical-elastic-elastic-stressfailure)
   - ["为弹性模型定义基于应变的失效准则"](pt03ch12s09s01.md#usi-prp-mechanical-elastic-elastic-strainfailure)
10. 点击**确定**创建材料并关闭**编辑材料**对话框。或者，您可以从**编辑材料**对话框的菜单中选择另一种材料行为来定义（请参见["浏览和修改材料行为，"第12.7.2节"](pt03ch12s07hlb02.md)，了解更多信息）。

#### 为弹性模型定义基于应力的失效准则

使用**子选项编辑器**为弹性材料模型定义基于应力失效准则的应力极限。有关更多信息，请参见["平面应力正交各向异性失效准则，"Abaqus分析用户指南第22.2.3节](../usb/usb-link.md#usb-mat-cfailuremeasures)。

**要为弹性模型定义基于应力的失效准则：**

1. 按照["指定弹性材料属性"](pt03ch12s09s01.md#usi-prp-mechanical-elastic-elastic-overview)中所述创建线性弹性材料模型。
2. 从**编辑材料**对话框的**子选项**菜单中，选择**失效应力**。**子选项**编辑器出现。
3. 要定义依赖于温度的行为数据，请打开**使用与温度相关的数据**。**数据**表中会出现一个标有**温度**的列。
4. 要定义依赖于场变量的行为数据，请点击**场变量数量**字段右侧的箭头以增加或减少场变量的数量。**场**变量列出现在**数据**表中。
5. 在**数据**表中，输入应力极限：
   - **纤维方向拉伸应力** 纤维方向的拉伸应力极限，![](../graphics/usi_eqn00031.gif)。
   - **纤维方向压缩应力** 纤维方向的压缩应力极限，![](../graphics/usi_eqn00032.gif)。
   - **横向拉伸应力** 横向拉伸应力极限，![](../graphics/usi_eqn00033.gif)。
   - **横向压缩应力** 横向压缩应力极限，![](../graphics/usi_eqn00034.gif)。
   - **剪切强度** *X*–*Y*平面中的剪切强度，*S*。
   - **交叉项系数** 交叉乘积项系数，![](../graphics/usi_eqn00035.gif)（![](../graphics/usi_eqn00036.gif)）。此值仅用于Tsai-Wu理论，如果提供了![](../graphics/usi_eqn00037.gif)则被忽略。默认值为零。
   - **应力极限** 双轴应力极限，![](../graphics/usi_eqn00037.gif)。此值仅用于Tsai-Wu理论。如果此项非零，则忽略![](../graphics/usi_eqn00035.gif)。
   您可能需要扩展对话框以查看**数据**表中的所有列。有关如何输入数据的详细信息，请参见["输入表格数据，"第3.2.7节"](pt01ch03s02s07.md)。
6. 点击**确定**返回**编辑材料**对话框。

#### 为弹性模型定义基于应变的失效准则

使用**子选项编辑器**为弹性材料模型定义基于应变失效准则的应变极限。有关更多信息，请参见["平面应力正交各向异性失效准则，"Abaqus分析用户指南第22.2.3节](../usb/usb-link.md#usb-mat-cfailuremeasures)。

**要定义基于应变的失效准则：**

1. 按照["指定弹性材料属性"](pt03ch12s09s01.md#usi-prp-mechanical-elastic-elastic-overview)中所述创建线性弹性材料模型。
2. 从**编辑材料**对话框的**子选项**菜单中，选择**失效应变**。
3. 要定义依赖于温度的行为数据，请打开**使用与温度相关的数据**。**数据**表中会出现一个标有**温度**的列。
4. 要定义依赖于场变量的行为数据，请点击**场变量数量**字段右侧的箭头以增加或减少场变量的数量。**场**变量列出现在**数据**表中。
5. 在**数据**表中，输入应变极限：
   - **纤维方向拉伸应变** 纤维方向的拉伸应变极限，![](../graphics/usi_eqn00038.gif)。
   - **纤维方向压缩应变** 纤维方向的压缩应变极限，![](../graphics/usi_eqn00039.gif)。
   - **横向拉伸应变** 横向拉伸应变极限，![](../graphics/usi_eqn00040.gif)。
   - **横向压缩应变** 横向压缩应变极限，![](../graphics/usi_eqn00041.gif)。
   - **剪切应变** *X*–*Y*平面中的剪切应变极限，![](../graphics/usi_eqn00042.gif)。
   您可能需要扩展对话框以查看**数据**中的所有列。有关如何输入数据的详细信息，请参见["输入表格数据，"第3.2.7节"](pt01ch03s02s07.md)。
6. 点击**确定**返回**编辑材料**对话框。
