# 4.2.3 注释管理器





**注释管理器**类似于Abaqus/CAE中的其他管理器对话框。它允许您执行以下操作：
- 创建[箭头](pt01ch04s04hlb01.md)、[文本](pt01ch04s04hlb02.md)或[组合箭头和文本](pt01ch04s04hlb03.md)注释。
- 编辑[箭头](pt01ch04s04hlb05.md)或[文本](pt01ch04s04hlb06.md)注释。
- 复制或重命名注释。
- 删除注释。

此外，**注释管理器**允许您执行以下**视口**菜单或工具栏中不可用的任务：- 选择要管理的注释来源——模型数据库（**MDB**）或输出数据库（**ODB**）。
- 在当前视口中绘制模型数据库或输出数据库注释。
- 在当前视口中隐藏模型数据库或输出数据库注释。
- 将注释从模型数据库复制到输出数据库，反之亦然。
- 在视口中高亮显示注释。
- 重新排列列表中箭头和文本注释的顺序。

您可以通过从主菜单栏选择****视口（Viewport）**![](../graphics/images/arrow.gif)**注释管理器（Annotation Manager）****或![](../graphics/ico_manager.png)**视口**工具栏中的来显示**注释管理器**。**注释管理器**如图[图4-5](pt01ch04s02hlb03.md#vp-annotmgr)所示。
