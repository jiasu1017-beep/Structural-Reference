# *PLASTIC TORQUE







### *PLASTIC TORQUE定义框架单元的塑性扭矩行为。

此选项只能与[*FRAME SECTION](ch06abk34.md)选项结合使用，且仅适用于FRAME3D单元。它描述框架单元中的扭矩作为关于单元轴线塑性旋转的函数。

**产品：**Abaqus/Standard  

**类型：**模型数据  

**级别：**零件，零件实例  

##### **参考：**

- ["框架截面行为，" Abaqus Analysis User's Guide第29.4.2节](../usb/usb-link.md#usb-elm-eusingframesection)
- [*FRAME SECTION](ch06abk34.md)

**此选项没有关联的参数。**

### **定义扭矩屈服矩随塑性旋转变化的数据行：**

**第一行：**

根据需要重复此数据行以定义扭矩与塑性旋转之间的关系。至少需要三对数据。
