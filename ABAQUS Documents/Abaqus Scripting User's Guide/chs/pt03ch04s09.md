# 4.10 布局提示







`FXPacker`、`FXTopWindow` 和 `FXGroupBox` 组件在其子组件中接受以下布局提示：

**LAYOUT_SIDE_TOP**

将组件附加到空腔的顶部。LAYOUT_SIDE_TOP 是默认的布局提示。

**LAYOUT_SIDE_BOTTOM**

将组件附加到空腔的底部。

**LAYOUT_SIDE_LEFT**

将组件附加到空腔的左侧。

**LAYOUT_SIDE_RIGHT**

将组件附加到空腔的右侧。

您应该只为每个子组件指定一个 LAYOUT_SIDE_* 提示。顶部和底部提示有效地减少了用于放置其他子组件的可用空间的高度。左侧和右侧提示有效地减少了用于放置其他子组件的可用空间的宽度。

所有布局管理器支持以下布局提示：
- LAYOUT_LEFT（默认）和 LAYOUT_RIGHT。布局管理器将组件放置在容器中剩余空间的左侧或右侧。
- LAYOUT_TOP（默认）和 LAYOUT_BOTTOM。布局管理器将组件放置在容器中剩余空间的顶部或底部。
- LAYOUT_CENTER_X 和 LAYOUT_CENTER_Y。布局管理器在父组件的 *X* 或 *Y* 方向上将组件居中。管理器在组件周围添加额外间距，以将其放置在其可用空间的中心。组件的大小将为其默认大小，除非您指定 LAYOUT_FIX_WIDTH 或 LAYOUT_FIX_HEIGHT。
- LAYOUT_FILL_X 和 LAYOUT_FILL_Y。您可以指定这些布局提示中的无、一个或两个。LAYOUT_FILL_X 导致父布局管理器拉伸或收缩组件以适应可用空间。如果并排放置多个带有此选项的子组件，管理器会根据子组件的默认大小按比例细分可用空间。LAYOUT_FILL_Y 在垂直方向上具有相同的效果。

`FXPacker`、`FXTopWindow` 和 `FXGroupBox` 必须将 LAYOUT_LEFT 和 LAYOUT_RIGHT 与 LAYOUT_SIDE_TOP 和 LAYOUT_SIDE_BOTTOM 一起使用。Abaqus GUI Toolkit 会忽略没有意义的提示；例如，`FXHorizontalFrame` 会忽略 LAYOUT_TOP 和 LAYOUT_BOTTOM。其他提示也有类似的规则。

Abaqus GUI Toolkit 中的大多数组件在其构造函数中都有宽度和高度参数。在大多数情况下，您可以接受这些参数的默认值零，这允许应用程序确定组件的正确大小。但在某些情况下，您需要为组件的宽度和高度设置特定值。要设置宽度和高度，必须将 LAYOUT_FIX_WIDTH 和 LAYOUT_FIX_HEIGHT 标志传递给组件的 options 参数。如果不将这些标志传递给 options 参数，工具包将忽略您为宽度和高度指定的值。

布局提示的详细说明请参见[附录 C，"布局提示"](ap03.md)。
