# 3.8.9 表面单元子模型





**产品：**Abaqus/Standard  

### 测试的单元

SFM3D3    SFM3D4    SFM3D6    SFM3D8R    

### 测试的功能

子模型功能应用于具有表面单元的模型。全局输入文件使用["膜片测试，"第1.5.1节](ch01s05abv69.md)中描述的膜片测试中的模型。子模型输入文件用16个SFM3D8单元建模全局模型的右半部分。子模型驱动节点位于与模型y轴平行的线上。测试了静态摄动和静态一般程序。

### 问题描述

全局模型的尺寸与["膜片测试，"第1.5.1节](ch01s05abv69.md)中使用的片测试相同。节点文件请求已更改，以能够对驱动变量的值或驱动节点的历史幅值进行插值。有三层钢筋，分别与x轴成0度、45度和90度方向。钢筋的材料特性和尺寸如下：

| 杨氏模量 | 1×10^6 |
| --- | --- |
| 泊松比 | 0.25 |
| 钢筋面积 | 0.3 |
| 钢筋间距 | 0.5 |

子模型材料特性与全局模型特性相同。

### 结果与讨论

所有驱动变量都从全局分析正确插值。由于规定位移和/或集中力模式被带到子模型上的物理位置，应力场在两个模型中不匹配。
