# *LOAD CASE









### *LOAD CASE开始多荷载工况分析的荷载工况定义。

此选项用于开始每个荷载工况定义。

**产品：**Abaqus/Standard  Abaqus/CAE  

**类型：**历史数据  

**级别：**步骤

**Abaqus/CAE：**荷载模块

##### **参考：**

- ["多荷载工况分析，" Abaqus Analysis User's Guide第6.1.4节](../usb/usb-link.md#usb-anl-amultiloadcase)
- [*END LOAD CASE](ch05abk18.md)

### **必需参数：**

NAME

将此参数设置为用于引用荷载工况的标签。

**此选项没有关联的数据行。**
