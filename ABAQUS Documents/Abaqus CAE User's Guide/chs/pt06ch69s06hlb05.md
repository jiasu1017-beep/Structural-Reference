# 69.6.5 修复无效边







在极少数情况下，导入部件后，Abaqus 会报告其所有边都无效。**Repair invalid**（修复无效）工具尝试通过重新计算定义边的数据来修复无效边，这可以帮助修复某些类型的无效性。如果自动修复部件后仅剩无效边，您也应该使用此工具。

查询工具集提供了一组几何诊断工具，用于确定部件是否包含无效边。 有关更多信息，请参阅["在 Part 模块中使用查询工具集，" 第 11.16.4 节"](pt03ch11s16s04.md)。修复无效边的操作作为部件的特征存储；因此，您可以使用模型树删除或抑制操作。

**修复无效边：**

1. 从主菜单栏中选择 ****Tools**![](../graphics/images/arrow.gif)**Geometry Edit****（工具→几何编辑）。Abaqus/CAE 显示 **Geometry Edit**（几何编辑）对话框。**提示：**您也可以使用位于 Part（部件）模块工具箱中编辑工具处的 ![](../graphics/ico_repairInvalidEdges.png) 工具来修复无效边。 有关工具箱中编辑工具的图表，请参阅["使用几何编辑工具集，" 第 69.1 节"](pt06ch69s01.md)。
2. 从对话框中选择 **Edge**（边）类别和 **Repair invalid**（修复无效）方法。Abaqus/CAE 修复无效边并提示您更新部件的有效性。
3. 执行以下操作之一： - 点击 **Yes**（是）更新部件的有效性。（每当您修复无效边时都应该更新部件的有效性。） - 点击 **No**（否）跳过更新并继续您的修复。 - 点击 **Cancel**（取消）取消当前修复操作中的更改。您也可以随时通过在模型树中的部件上点击鼠标按钮 3 并从出现的菜单中选择 **Update validity**（更新有效性）来更新部件的有效性。
