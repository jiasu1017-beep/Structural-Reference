# 17.16.8 删除边种子





通过从主菜单栏选择 ****Seed**![](../graphics/images/arrow.gif)**Delete Edge Seeds****，然后选择要删除边种子的边、面或单元来删除边种子。

**删除边种子：**

1. 从主菜单栏，选择 ****Seed**![](../graphics/images/arrow.gif)**Delete Edge Seeds****。 Abaqus/CAE在提示区域显示提示以指导您完成该过程。 **提示：**您也可以使用Mesh模块工具箱中种子工具所在的 ![](../graphics/ico_meshSeedDeleteEdge.png) 工具删除种子。 （有关更多信息，请参阅["使用Mesh模块工具箱，" 第17.15节](pt03ch17s15.md)。）
2. 选择要删除其边种子的边。 默认情况下，Abaqus/CAE允许您从所有项目中选择以删除边种子。 要将对像选择限制为仅边、面或单元，请使用**Selection**工具栏将可选择的对象类型更改为**Edges**、**Faces**或**Cells**。 有关更多信息，请参阅["根据对象类型过滤您的选择，" 第6.3.2节](pt01ch06s03s01.md)。
3. 完成选择边、面或单元后，按鼠标按钮2提交您的选择。 有关选择对象的更多信息，请参阅[第6章，"在视口中选择对象"](pt01ch06.md)。 所选对象的所有边上的种子都会消失。
