# 68.3 缩放图标大小





工具栏和工具箱中图标的默认大小为24×24像素。模型树和结果树中图标的默认大小为16×16像素。您可以指定一个缩放因子来增加或减小这些图标的大小，使它们以适合您的显示分辨率的大小显示。例如，将图标大小缩放因子增加到1.5会提示Abaqus/CAE以36×36像素显示工具栏和工具箱图标，以24×24像素显示模型树和结果树图标。

图标大小的更改在您重新启动Abaqus/CAE时生效。您选择的缩放因子会保存以供将来的Abaqus/CAE会话使用。

**要缩放图标大小：**

1. 从任何模块的主菜单栏中，选择****工具**![](../graphics/images/arrow.gif)**选项****。将出现**选项**对话框。
2. 显示**图标**选项卡页面。
3. 单击**缩放因子**字段右侧的箭头来增加或减小Abaqus/CAE中图标的大小。最大缩放因子为3；最小缩放因子为0.5。
4. 单击**确定**应用您指定的配置并关闭**选项**对话框。您必须重新启动Abaqus/CAE会话才能查看更改。
