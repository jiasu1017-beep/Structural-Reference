# 69.1 使用几何编辑工具集





您可以在零件模块中使用几何编辑工具集来编辑或修复零件的几何形状。几何编辑工具集使您能够编辑导入的零件以提高其精度和有效性；但是，Abaqus/CAE不要求零件完全精确。此外，您可以使用几何编辑工具集从零件中移除小特征。您可以使用几何编辑工具集来编辑边、面或整个零件。["编辑技术概述," 第69.2节](pt06ch69s02.md)提供了每种类型分割可用方法的概述。Abaqus/CAE将大多数编辑操作存储为特征。因此，您可以通过使用特征操作工具集删除或抑制相应的特征来撤销编辑。

您可以通过从主菜单中选择****工具**![](../graphics/images/arrow.gif)**几何编辑****来访问编辑工具，并从出现的**几何编辑**对话框中选择所需的工具。您也可以通过零件模块工具箱访问几何编辑工具。[图69-1](pt06ch69s01.md#rep-toolbox)显示了零件模块工具箱中所有几何编辑工具的隐藏图标。

![](../graphics/rep-toolbox-nls.png)
