# 62.5.2 创建datum轴的方法概述







当您从**Create Datum**对话框中选择**Axis**时，**Method**列表显示以下创建datum轴的方法：

**![](../graphics/ico_dtmAxisPrincipal.png) 主轴**

选择datum轴必须与其共线的三个主轴之一，如[图62--13](pt06ch62s05s02.md#dtm-hapa)所示。有关详细说明，请参见["沿主轴创建datum轴，" 第62.7.1节](pt06ch62s07hlb01.md)。

**图62–13** 将datum轴定义为三个主轴之一。

![](../graphics/dtm-hapa-nls.png)

**![](../graphics/ico_dtmAxisIntersectPlanes.png) 两个平面的交线**

选择两个不平行的平面。Abaqus/CAE在两个平面（或两个平面的延伸）相交处创建datum轴，如[图62--14](pt06ch62s05s02.md#dtm-haxn)所示。有关详细说明，请参见["沿两个平面的交线创建datum轴，" 第62.7.2节](pt06ch62s07hlb02.md)。

**图62–14** 将datum轴定义为两个平面的交线。

![](../graphics/dtm-haxn-nls.png)

**![](../graphics/ico_dtmAxisThruEdge.png) 直边**

选择模型上datum轴必须与其共线的一条直边，如[图62--15](pt06ch62s05s02.md#dtm-har)所示。有关详细说明，请参见["沿直边创建datum轴，" 第62.7.3节](pt06ch62s07hlb03.md)。

**图62–15** 将datum轴定义为模型上的一条直边。

![](../graphics/dtm-har-nls.png)

**![](../graphics/ico_dtmAxisTwoPoints.png) 两点**

选择datum轴必须通过的模型上的任意两点，如[图62--16](pt06ch62s05s02.md#dtm-ha2p)所示。有关详细说明，请参见["通过两点创建datum轴，" 第62.7.4节](pt06ch62s07hlb04.md)。

**图62–16** 通过选择两点定义datum轴。

![](../graphics/dtm-ha2p-nls.png)

**![](../graphics/ico_dtmAxisCylinderAxis.png) 圆柱轴线**

在模型上选择一个圆柱面。Abaqus/CAE创建一条沿该圆柱面轴线的datum轴，如[图62--17](pt06ch62s05s02.md#dtm-hacyl)所示。有关详细说明，请参见["沿圆柱轴线创建datum轴，" 第62.7.5节](pt06ch62s07hlb05.md)。

**图62–17** 将datum轴定义为圆柱的轴线。

![](../graphics/dtm-hacyl-nls.png)

**![](../graphics/ico_dtmAxisNormalToPlane.png) 垂直于平面，穿过点**

选择一个平面和一个不在平面上的点。Abaqus/CAE创建一条垂直于该平面并穿过该点的datum轴，如[图62--18](pt06ch62s05s02.md#dtm-happ)所示。有关详细说明，请参见["创建垂直于平面并穿过点的datum轴，" 第62.7.6节](pt06ch62s07hlb06.md)。

**图62–18** 通过选择一个点和一个平面定义datum轴。

![](../graphics/dtm-happ-nls.png)

**![](../graphics/ico_dtmAxisParallelToLine.png) 平行于线，穿过点**

选择模型的一条边和一个不在边上的点。Abaqus/CAE创建一条平行于该边并穿过该点的datum轴，如[图62--19](pt06ch62s05s02.md#dtm-halp)所示。有关详细说明，请参见["创建平行于直线并穿过点的datum轴，" 第62.7.7节](pt06ch62s07hlb07.md)。

**图62–19** 通过选择一个点和一个边定义datum轴。

![](../graphics/dtm-halp-nls.png)

**![](../graphics/ico_dtmAxisThreePointCircle.png) 圆上三点**

选择在模型上定义圆的三个点。Abaqus/CAE沿圆的轴线创建一条datum轴，如[图62--20](pt06ch62s05s02.md#dtm-hacir)所示。有关详细说明，请参见["沿由三点定义的圆的轴线创建datum轴，" 第62.7.8节](pt06ch62s07hlb08.md)。

**图62–20** 将datum轴定义为圆的轴线。

![](../graphics/dtm-hacir-nls.png)

**![](../graphics/ico_dtmAxisRotateLine.png) 从线旋转**

选择一个旋转边和一个旋转轴，并指定边将旋转的角度。Abaqus/CAE通过将边绕轴旋转指定角度来创建datum轴，如[图62--21](pt06ch62s05s02.md#dtm-harot)所示。有关详细说明，请参见["通过将现有边旋转指定角度创建datum轴，" 第62.7.9节](pt06ch62s07hlb09.md)。

**图62–21** 通过将边旋转指定角度定义datum轴。

![](../graphics/dtm-harot-nls.png)
