# 36.9 创建和编辑桁条加固





您可以在定义部件后立即创建桁条；桁条最初不需要材料或截面分配。创建桁条后，您可以使用Property模块中**Special**菜单下的选项为其分配截面、梁截面方向、材料方向和切线方向。您还可以在Mesh模块中为桁条分配单元类型。当提示您选择要分配单元类型的区域时，必须从**Selection**工具栏的对象类型列表中选择**Stringers**。

您可以在同一边缘上一次创建多个桁条。Abaqus/CAE将这些桁条视为桁条加固的不同层，您可以为每个桁条分配不同的属性。

Abaqus/CAE自动命名桁条，但您可以从**Stringer Manager**、**Special**菜单或Model Tree中重命名它们。如果您一次创建多个桁条，Abaqus/CAE会附加唯一标识符到名称以识别每个桁条。

您可以通过使用Step模块中的场和历史输出请求编辑器从桁条获取数据。在编辑器的**Domain**部分，选择**Stringer**并从出现的菜单中选择所需的桁条。更多信息，请参见["创建输出请求"，第14.12.1节](pt03ch14s12hlb01.md)。

**要创建桁条加固：**

1. 从Property模块的主菜单栏中，选择****Special**![](../graphics/images/arrow.gif)**Stringer**![](../graphics/images/arrow.gif)**Create****。
   **提示：**您也可以单击Property模块工具箱中的![](../graphics/ico_stringer.png)工具。
2. 使用以下方法之一选择要添加桁条加固的边缘：
   **选择各个边缘：**
   1. 在提示区域中单击**Select entities on which to create a stringer**字段旁边的箭头，并从出现的列表中选择**individually**。
   2. 选择要添加桁条加固的边缘。
   3. **[Shift]**+单击其他边缘以将其添加到您的选择。
   4. 如有必要，**[Ctrl]**+单击所选边缘以取消选择。
   5. 完成选择边缘后，单击鼠标按钮2。
   **指定现有集：**
   1. 在提示区域右侧单击**Sets**。Abaqus/CAE显示**Region Selection**对话框，其中包含您创建的单元集列表。
   2. 选择要添加桁条加固的实体集，然后单击**Continue**。
   **注意：**默认选择方法基于您最近使用的选择方法。要切换到另一种方法，请在提示区域右侧单击**Select in Viewport**或**Sets**。
   **使用角度方法选择边缘：**
   此方法仅用于选择几何边缘。此方法不可用于选择孤立网格单元边缘。
   1. 在提示区域中单击**Select entities on which to create a stringer**字段旁边的箭头，并从出现的列表中选择**by edge angle**。
   2. 输入角度（0到90），然后选择一个边缘或其他实体。Abaqus/CAE从所选边缘开始选择每个相邻边缘，直到边缘之间的角度等于或超过您输入的角度。（更多信息，请参见["使用角度和特征边缘方法选择多个对象"，第6.2.3节](pt01ch06s02hlb03.md)。）
   3. 使用**by edge angle**方法后，您可以**[Shift]**+单击其他边缘以将其添加到您的选择，或**[Ctrl]**+单击所选边缘以将其从您的选择中移除。（更多信息，请参见["组合选择技术"，第6.2.8节](pt01ch06s02hlb08.md)。）
   **提示：**您可以使用**Selection**工具栏来限制可以在视口中选择的对象类型。更多信息，请参见[第6章，"在视口中选择对象"](pt01ch06.md)。
3. 如需，切换**Multiple stringers**以为所选边缘创建多个桁条。
4. 单击**Done**创建桁条。如果您要创建多个桁条，请输入您要创建的桁条数量，然后单击鼠标按钮2。
