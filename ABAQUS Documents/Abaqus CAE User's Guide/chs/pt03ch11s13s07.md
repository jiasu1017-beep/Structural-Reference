# 11.13.7 控制带pitch的旋转特征的横截面





创建不带pitch的旋转特征时，草图轮廓围绕由旋转轴和草图之间半径描述的圆形路径扫掠。旋转特征的横截面是草图；横截面始终平行于旋转轴并垂直于圆形路径。

当在旋转特征中包含pitch时，草图的路径变为螺旋而不是圆形。您可以选择使草图平行于旋转轴，也可以选择使草图垂直于螺旋路径旋转。

要使草图轮廓垂直于带pitch的旋转特征的螺旋路径，请在**编辑旋转**对话框中切换**扫掠草图垂直于路径**。当Abaqus/CAE创建旋转特征时，它会旋转草图轮廓，使其在起点处垂直于旋转路径。在整个特征创建过程中，轮廓保持垂直于路径。无论pitch值如何，横截面都将与草图轮廓匹配。使用此选项，您可以创建螺旋弹簧或其他横截面为草图轮廓的特征。

如果您不选择**扫掠草图垂直于路径**，草图轮廓将保持平行于旋转轴，旋转特征的横截面将与轮廓不同。轮廓与横截面之间的差异将随着pitch值的增加而增加。例如，如果没有pitch，圆形草图轮廓会创建圆形横截面。如果增加pitch，横截面将变得越来越椭圆。您可以创建螺纹或其他横截面平行于旋转轴的特征是您的草图轮廓。

要在特征创建后更改横截面行为，可以在**编辑特征**对话框中切换**移动草图垂直于路径**。![](../graphics/images/black4rule.gif)有关相关信息，请单击以下任一项：- "添加旋转实体特征"，第11.21.2节- "添加旋转壳特征"，第11.22.2节- "创建旋转切割"，第11.24.3节- "使用六面体单元网格划分复杂实体"，第17.14.5节
