# 53.5 用户输入命令







以下命令用于请求用户输入数据。

### 53.5.1 getInput(...)

此方法用于从对话框获取用户输入。该方法显示一个模态对话框，其中包含一个以指定标签为前缀的文本字段。用户可以在文本字段中输入数据，或接受显示在文本字段中的可选默认值。当用户点击 **OK** 按钮时，`getInput` 方法从对话框读取数据并关闭对话框。可以在标签参数中使用 "\t" 分隔符来提供对话框的工具提示；例如：

```
getInput('Enter a number:\tEnter the number of nodes to delete')
```

**注意：**如果从命令行运行脚本并将脚本名称传递给命令行选项 **-start**、**-script**、**-replay** 或 **-noGUI**，则不能使用包含 `getInput` 的脚本。

**路径**

```
getInput
```

**必需参数**

*prompt*

一个 String，指定对话框中的文本字段。

**可选参数**

*default*

一个 String，指定显示在对话框文本字段中的默认值。

**返回值**

一个 String；如果用户点击 **Cancel**，则返回 `None`。

**异常**

None。

### 53.5.2 getInputs(...)

此方法用于从对话框获取多个用户输入。该方法显示一个模态对话框，其中包含一列以指定标签为前缀的文本字段。用户可以在文本字段中输入数据，或接受显示在文本字段中的可选默认值。当用户点击 **OK** 按钮时，`getInputs` 方法从对话框读取数据并关闭对话框。

**注意：**如果从命令行运行脚本并将脚本名称传递给命令行选项 **-start**、**-script**、**-replay** 或 **-noGUI**，则不能使用包含 `getInputs` 的脚本。

**路径**

```
getInputs
```

**必需参数**

*fields*

一个 String 序列的序列，指定对话框中的文本字段。每个内部序列是一对 String，指定描述文本字段的标签以及显示在文本字段中的默认值。如果字段没有默认值，应在对的第二个字符串中输入空字符串。

**可选参数**

*label*

一个 String，指定放置在对话框顶部的标签。默认值为空字符串，表示不显示标签。

*dialogTitle*

一个 String，指定在对话框标题栏中显示的文本。默认值为 "Get Inputs"。

**返回值**

一个 String 序列，表示对话框中每个文本字段中的数据。如果用户点击 **Cancel** 按钮，该方法返回一个 `None` 对象序列。

**异常**

None。

### 53.5.3 getWarningReply(...)

此方法用于从警告对话框获取用户的回复。该方法显示一个带有消息和标准按钮的模态警告对话框。用户点击其中一个标准按钮后，`getWarningReply` 返回相应的按钮值并关闭对话框。可以在消息参数中使用 "\n" 分隔符在警告对话框中提供多行消息；例如：

```
getWarningReply('Out of disk space!\nOkay to continue', (YES,NO))
```

**注意：**如果从命令行运行脚本并将脚本名称传递给命令行选项 **-start**、**-script**、**-replay** 或 **-noGUI**，则不能使用包含 `getWarningReply` 的脚本。

**路径**

```
getWarningReply
```

**必需参数**

*message*

一个 String，指定警告对话框中的消息。

*buttons*

一个序列，指定显示在警告对话框中的标准按钮。可取值为 YES、NO、YES_TO_ALL 和 CANCEL。

**可选参数**

None。

**返回值**

用户点击的按钮。

**异常**

None。
