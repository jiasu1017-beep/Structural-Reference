# 42.5.7 选择流场输出变量





您可以指定计算流数据的变量；这个变量称为流场输出变量。对于大多数分析，速度（**V**）可用于流数据；但是，对于某些分析，您可以使用**VORTICITY**作为流场输出变量。只能将节点中心的向量结果与流线跟踪关联。

选择流场输出变量时，Abaqus/CAE列出存储在节点位置且在当前[步骤和帧](pt05ch42s01hlb01.md)的[输出数据库](pt02ch09s06hlb02.md)中可用的所有向量变量。有关单个输出变量标识符的信息，请参见["Abaqus/CFD输出变量标识符，" Abaqus Analysis User's Guide第4.2.3节](../usb/usb-link.md#usb-out-hcfdoutputvar)。

**选择流场输出变量：**

1. 找到控制流场输出变量的选项。从主菜单栏，选择****Result**![](../graphics/images/arrow.gif)**Field Output****。在出现的对话框中点击**Stream Variable**选项卡。**提示：**您也可以通过点击任何出现此按钮的对话框中的**Field Output**按钮来访问这些选项。要查看列出变量的完整描述，请通过拖动角落来增加对话框的宽度。
2. 从**Name**和**Description**列表中，点击您想要的分析变量名。选中的变量高亮显示。
3. 点击**Apply**实现您的更改。Abaqus/CAE基于所选流输出变量重新计算当前视口中的流线。您对流输出变量的选择也会保存到会话结束。
