# 55.12.1 扫掠和拉伸模型





您可以将轴对称模型显示为平面二维形状，或将模型"扫掠"通过指定角度，产生三维视觉效果。同样，您可以仅查看循环对称结构的建模扇区，或"扫掠"该扇区以查看整个模型的指定部分。（有关循环对称结构的更多信息，请参阅["分析表现出循环对称性的模型，" Abaqus Analysis User's Guide第10.4.3节](../usb/usb-link.md#usb-anl-acyclicsymmetry)。）此外，您可以将二维模型（或三维圆柱形解析刚性表面）显示为平面或将它们拉伸到指定深度，产生三维视觉效果。扫掠和拉伸对于显示二维元素和接触表面的轮廓图特别有用。

轴对称解析刚性表面可以绕轴旋转（扫掠），以下元素也可以：ACAX*n*、CAX*n*、CAXA*n*、CGAX*n*、DCAX*n*、DCCAX*n*、DSAX*n*、FAX*n*、MAX*n*、MGAX*n*、RAX*n*、SAX*n*和SAXA*n*。包含三维轴对称解析刚性表面或CAXA元素的模型默认会被扫掠（默认起始角、终止角和扇区数因模型类型而异）。如果模型同时包含解析刚性表面和CAXA元素，则默认只扫掠CAXA元素。当您显示扫掠轴对称元素的边界条件时，符号仅出现在原始节点上，而不是扫掠节点上。

循环对称模型默认不扫掠。当您在扫掠的循环对称模型上显示轮廓时，只能在扫掠扇区上对节点量进行轮廓显示；标量、向量和张量在积分点处的数量只能在整个原始扇区上进行轮廓显示。

您可以沿*Z*方向拉伸解析刚性表面和Abaqus单元库中的所有平面二维实体单元。有关这些单元的列表，请参阅["二维实体单元库，" Abaqus Analysis User's Guide第28.1.3节](../usb/usb-link.md#usb-elm-e2delem)。

**要在轴对称模型中扫掠元素：**

1. 找到**General Sweep**选项。从主菜单栏选择****View**![](../graphics/images/arrow.gif)**ODB Display Options****。在出现的对话框中单击**Sweep/Extrude**选项卡。**General Sweep**选项位于页面上部。**注意：**如果模型中没有可以扫掠的元素或解析刚性表面，则**General Sweep**选项将不可用。
2. 切换**Sweep elements**。
3. 通过在**Sweep from**字段中输入数字（度）（默认值为0）和在**To**字段中输入数字（度）（默认值为180）来指定扫掠角。Abaqus/CAE将二维模型围绕旋转轴从第一个指定角度逆时针扫掠到第二个角度。
4. 通过在**Number of segments**字段中输入正整数或单击字段旁边的箭头来指定沿圆周方向扫掠时使用的段数。默认使用的段数因模型类型而异。段之间的角度会在对话框中自动更新。当您增加段数时，模型中的曲线看起来会更平滑（它们之间的角度减小）。但是，较少的段数绘图更快。
5. 单击**Apply**来应用您的更改。您的扫掠规范会在所有绘图状态中反映，默认情况下会在会话期间保存。如果您希望保留所做的更改以供后续会话使用，请将它们保存到文件中。有关更多信息，请参阅["保存自定义设置以供后续会话使用，" 第55.1.1节](pt05ch55s01s01.md)。如果显示轮廓图，轮廓会出现在所有扫掠面上。如果显示符号图，模型会按指定进行扫掠，但符号仅出现在原始平面模型面上。

**要在轴对称模型中扫掠解析刚性表面：**

1. 找到**General Sweep**选项。从主菜单栏选择****View**![](../graphics/images/arrow.gif)**ODB Display Options****。在出现的对话框中单击**Sweep/Extrude**选项卡。**General Sweep**选项位于页面上部。**注意：**如果模型中没有可以扫掠的元素或解析刚性表面，则**General Sweep**选项将不可用。
2. 如果当前未激活，则切换打开**Sweep analytical rigid surfaces**。
3. 通过在**Sweep from**字段中输入数字（度）和在**To**字段中输入数字（度）来指定扫掠角（首次打开轴对称模型时默认值为180，首次打开三维模型时默认值为360）。Abaqus/CAE将解析刚性表面围绕旋转轴从第一个指定角度逆时针扫掠到第二个角度。
4. 通过在**Number of segments**字段中输入正整数或单击字段旁边的箭头，指定在扫掠解析刚性表面时沿圆周方向使用的段数。默认情况下，Abaqus/CAE对轴对称模型使用10段，对三维模型使用20段。段之间的角度会在对话框中自动更新。当您增加段数时，模型中的曲线看起来会更平滑（它们之间的角度减小）。但是，较少的段数绘图更快。
5. 单击**Apply**来应用您的更改。您的扫掠规范会在所有绘图状态中反映并会在会话期间保存。如果您希望保留所做的更改以供后续会话使用，请将它们保存到文件中。有关更多信息，请参阅["保存自定义设置以供后续会话使用，" 第55.1.1节](pt05ch55s01s01.md)。

**要扫掠循环对称模型：**

1. 找到**Sector Sweep**选项。从主菜单栏选择****View**![](../graphics/images/arrow.gif)**ODB Display Options****。在出现的对话框中单击**Sweep/Extrude**选项卡。**Sector Sweep**选项位于页面中部。**注意：**如果模型中没有循环对称元素，则**Sector Sweep**选项将不可用。
2. 切换**Sweep cyclic symmetry sectors**。
3. 单击**Sector selection**字段旁边的箭头，从以下可用的扫掠方法中选择：
   **By Number** 这是默认的扫掠方法。模型中的总扇区数显示在对话框中。您可以通过在**Sectors**字段中输入1到总扇区数之间的正整数（用逗号分隔）来指定应显示哪些扇区。扇区从原始扇区开始逆时针编号。Abaqus/CAE仅显示指定的扇区（无论它们是否相邻）。
   **By Angle** 扇区角度（每个扇区之间的角度）显示在对话框中。您可以通过在**Sweep from**字段中输入数字（度）（默认值为0）和在**To**字段中输入数字（度）（默认值为扇区角度）来以扇区角度的增量指定扫掠角度。两个数字都应该能被扇区角度整除。**提示：**您也可以使用文本字段旁边的箭头指定角度。Abaqus/CAE将二维模型围绕旋转轴扫掠指定的度数。如果您输入的角度不能被扇区角度整除，Abaqus/CAE会四舍五入到下一个可以整除的数字。
   **All Sectors** 如果选择此扫掠方法，将显示所有扇区；换句话说，模型将围绕旋转轴扫掠360度。
4. 单击**Apply**来应用您的更改。您的扫掠规范会在所有绘图状态中反映并会在会话期间保存。如果您希望保留所做的更改以供后续会话使用，请将它们保存到文件中。有关更多信息，请参阅["保存自定义设置以供后续会话使用，" 第55.1.1节](pt05ch55s01s01.md)。如果显示轮廓图，节点量的轮廓会出现在所有扫掠面上；积分点处的标量、向量和张量只会出现在原始扇区上。如果显示符号图，模型会按指定进行扫掠，但符号仅出现在原始平面模型面上。

**要在模型中拉伸元素：**

1. 找到**Extrude**选项。从主菜单栏选择****View**![](../graphics/images/arrow.gif)**ODB Display Options****。在出现的对话框中单击**Sweep/Extrude**选项卡。**Extrude**选项位于页面下部。**注意：**如果模型中没有可以拉伸的元素，则**Extrude**选项将不可用。
2. 切换**Extrude elements**。当**Extrude elements**打开时，**Depth**选项变得可用。
3. 通过在**Depth**字段中输入正数（模型单位）来指定拉伸深度。默认深度为1.0。Abaqus/CAE将模型在负*z*方向上延伸指定的单位数。使用视图操作旋转工具在三维中查看拉伸后的模型。
4. 单击**Apply**来应用您的更改。您的拉伸规范会在所有绘图状态中反映并会在会话期间保存。如果您希望保留所做的更改以供后续会话使用，请将它们保存到文件中。有关更多信息，请参阅["保存自定义设置以供后续会话使用，" 第55.1.1节](pt05ch55s01s01.md)。如果显示轮廓图，轮廓会出现在所有拉伸面上。如果显示符号图，模型会按指定进行拉伸，但符号仅出现在原始平面模型面上。

**要在模型中拉伸解析刚性表面：**

1. 找到**Extrude**选项。从主菜单栏选择****View**![](../graphics/images/arrow.gif)**ODB Display Options****。在出现的对话框中单击**Sweep/Extrude**选项卡。**Extrude**选项位于页面下部。**注意：**如果模型中没有可以拉伸的元素或解析刚性表面，则**Extrude**选项将不可用。
2. 如果当前未激活，则切换打开**Extrude analytical rigid surfaces**。当**Extrude analytical rigid surfaces**打开时，**Depth**选项变得可用。
3. 选择拉伸的深度。您可以通过选择**Auto-compute**接受默认深度，或者通过在**Specify**字段中输入正数（模型单位）来指定自定义深度。解析刚性表面的默认拉伸深度是不会拉伸边界框的最大值。Abaqus/CAE将模型在负*z*方向上延伸指定的单位数。使用视图操作旋转工具在三维中查看拉伸后的模型。
4. 单击**Apply**来应用您的更改。您的拉伸规范会在所有绘图状态中反映并会在会话期间保存。如果您希望保留所做的更改以供后续会话使用，请将它们保存到文件中。有关更多信息，请参阅["保存自定义设置以供后续会话使用，" 第55.1.1节](pt05ch55s01s01.md)。
