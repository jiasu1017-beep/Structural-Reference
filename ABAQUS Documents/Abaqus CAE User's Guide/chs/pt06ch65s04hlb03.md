# 65.4.3 抑制特征







选择 ****Feature**![](../graphics/images/arrow.gif)**Suppress****（特征→抑制）从主菜单栏抑制所选特征。抑制特征相当于从部件或装配中临时移除它们。当您再生部件或装配时，被抑制的特征保持抑制状态。您也可以通过在模型树中的特征上点击鼠标按钮 3 并从出现的菜单中选择 **Suppress**（抑制）来抑制特征。

您无法抑制基础特征。此外，如果您抑制父特征，其所有子特征也会自动被抑制。被抑制的特征可以通过从主菜单栏中选择 ****Feature**![](../graphics/images/arrow.gif)**Resume****（特征→恢复）或通过在模型树中的特征上点击鼠标按钮 3 来恢复。当您恢复被抑制的特征时，其子特征不会被恢复，除非您选择它们。为了帮助您选择特征的子特征，您可以在模型树中的特征上点击鼠标按钮 3，并从出现的菜单中选择 **Show children**（显示子特征）。

**抑制特征：**

1. 从主菜单栏中选择 ****Feature**![](../graphics/images/arrow.gif)**Suppress****（特征→抑制）。**提示：**您也可以使用模块工具箱中位于的 ![](../graphics/ico_featureSuppress.png) 工具来抑制特征。
2. 选择一个或多个要抑制的特征。您可以从当前视口或从模型树中选择特征。有关更多信息，请参阅[第 6 章，"在视口中选择对象"](pt01ch06.md)"和["使用模型树和结果树，" 第 3.5 节"](pt01ch03s05.md)。Abaqus/CAE 高亮显示所选特征。如果您选择了具有子特征的特征，Abaqus/CAE 也会高亮显示子特征。如果您从视口选择特征，请在完成选择后点击提示区域中的 **Done**（完成）。
3. 如果您从视口选择了特征，请在提示区域中点击 **Yes**（是）。如果您从模型树选择了特征，请在其中一个特征名称上点击鼠标按钮 3，并从出现的菜单中选择 **Suppress**（抑制）。所选特征和任何子特征从视口中消失。
4. 要退出抑制过程，请执行以下操作之一 - 点击提示区域中的取消按钮 ![](../graphics/fts-cancel.gif)，或 - 在 Abaqus/CAE 窗口中的任意位置点击鼠标按钮 2，或 - 从 **Feature**（特征）菜单中选择其他操作。
