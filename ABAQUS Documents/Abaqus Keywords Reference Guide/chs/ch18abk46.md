# *SUBSTRUCTURE PROPERTY





### *SUBSTRUCTURE PROPERTY平移、旋转和/或反射子结构。

此选项用于为子结构定义属性。对于模型中的所有子结构，这是必需的。

**产品：**Abaqus/Standard

**类型：**模型数据

**级别：**此选项在以部件实例装配定义的模型中不受支持。

##### **参考：**

- ["Using substructures," Section 10.1.1 of the Abaqus Analysis User's Guide](../usb/usb-link.md#usb-anl-asuperelements)

### **必需参数：**

ELSET

将此参数设置为包含正在定义属性的子结构的元素集名称。

### **可选参数：**

POSITION TOL

将此参数设置为使用级别节点与相应子结构节点之间距离的容差。如果省略此参数，则默认容差为子结构内最大整体尺寸的10^4倍。如果参数给出的值为0.0，则不检查保留节点的位置。
