# 5.3.1 使用 3D 罗盘旋转视图





3D 罗盘允许你使用两种不同方法旋转模型视图：你可以自由地向所有方向旋转，或者你可以将操作限制为绕特定轴旋转。在这两种情况下，模型视图都围绕视口的当前旋转中心旋转，如旋转工具所定义（请参阅["旋转视图工具，" 第 5.2.3 节](pt01ch05s02s03.md)）。

**自由旋转**

要向任意方向旋转模型，请点击并拖动 3D 罗盘上的自由旋转手柄：

![](../graphics/usi-compass-rotation-nls.png)

当你拖动鼠标时，罗盘围绕其支点沿鼠标移动方向旋转（支点与罗盘操作手柄重合）。旋转取决于鼠标移动的方向，而不是指针的位置；换句话说，只要你继续拖动，罗盘就会继续旋转。随着罗盘方向的变化，模型的视图也会相应变化。

**绕轴旋转**

你也可以绕指定轴旋转模型，从而在操作期间保持特定方向的恒定方向。要绕轴旋转，请点击并拖动 3D 罗盘圆周上的三个圆弧之一：

![](../graphics/usi-compass-arc-nls.png)

当你拖动鼠标时，罗盘围绕垂直于所选圆弧所张平面的轴旋转（上例中的 *X* 轴）。旋转取决于指针的位置。当你拖动鼠标时，视口中指针的路径被投影到所选的罗盘圆弧上。罗盘根据此投影的路径旋转。随着罗盘方向的变化，模型的视图也会相应变化。
