# 3.2.16 获取示例输入文件





**产品：** Abaqus/Standard  Abaqus/Explicit  

##### **参考**

- ["Abaqus执行程序：概述，" 第3.1.1节](pt01ch03s01abo02.md)

### 概述

**abaqus fetch**实用工具用于从随版本提供的压缩档案文件中提取示例Abaqus输入文件、用户子程序文件、日志文件、参数化研究脚本文件或后处理程序（用于[Abaqus示例问题指南](../exa/exa-link.md#exa)、[Abaqus基准指南](../bmk/bmk-link.md#bmk)和[Abaqus验证指南](../ver/ver-link.md#ver)中的问题）。指南中指定了文件名。如果未指定文件扩展名，则将提取与给定名称对应的所有文件。

指定文件名时可以使用通配符表达式，包括：
- 星号(*)匹配零个或多个字符的序列。
- 问号(?)恰好匹配一个字符。
- 方括号项([...])匹配括号内的任何单个字符；范围由起始字符、连字符和结束字符指定。如果左括号后面跟有感叹号(!)或脱字号(^)，则括号内的字符范围是补集；也就是说，除了括号内的字符之外的任何内容都被认为是匹配项。

任何可能被操作系统解释或修改的字符（特别是在UNIX平台上）都应放在引号内。如果使用通配符表达式找不到匹配项，**abaqus fetch**实用工具将尝试提取具有指定名称的文件。

### 命令摘要

| **abaqus fetch** | [**job**](pt01ch03s02abx16.md#job1-usb-int-dfetchproc)=*job-name* |
| --- | --- |
|  | **[**[**input**](pt01ch03s02abx16.md#input2-usb-int-dfetchproc)=*input-file***]** |

### 命令行选项

**job**

此选项用于指定要获取的输入文件或文件的输出文件名。它也是要获取的输入文件的默认名称。

如果命令行上省略了此选项，系统将提示您输入此值。

**input**

此选项用于指定要获取的输入文件或文件的名称（如果与*job-name*不同）。

### 示例

要从档案文件中获取示例输入文件`c2.inp`，请使用以下命令：

```
abaqus fetch job=c2.inp
```

要从档案文件中获取与作业`c8`关联的所有文件，请不要指定文件扩展名。以下命令将提取输入文件(`c8.inp`)和用户子程序文件(`c8.f`)：

```
abaqus fetch job=c8
```

要从档案文件中获取名为`parstudy.psf`的示例参数化研究脚本文件，请使用以下命令：

```
abaqus fetch job=parstudy.psf
```
