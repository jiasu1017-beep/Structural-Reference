# 2.5 小结

- Abaqus 输入文件包含分析模型的完整描述。它是前处理器（如 Abaqus/CAE）与分析产品（Abaqus/Standard 或 Abaqus/Explicit）之间的通信媒介。
- 输入文件包含两个部分：定义被分析结构的数据（模型数据）和定义结构发生什么的历史数据（历史数据）。
- 输入文件的每个部分由若干选项块组成，每个选项块由一个关键字行组成，该关键字行后面可以跟随数据行。
- 创建输入文件后，您可以执行 **datacheck** 分析。错误和警告消息会打印到数据文件中。**datacheck** 分析成功完成后，会将模拟所需的计算机资源估算值打印到数据文件中。
- 使用 Abaqus/Viewer 可以通过在 **datacheck** 阶段生成的结果数据库文件以图形方式验证模型几何形状和边界条件。
- 通常最容易在数据（`.dat`）文件中检查材料属性的错误；而几何形状、载荷和边界条件更容易使用 Abaqus/Viewer 等图形后处理器进行检查。
- 始终检查结果是否满足基本工程原理，如平衡条件。
- Abaqus/Viewer 允许您以多种方式图形化地可视化分析结果，也允许您编写表格数据报告。
- 使用隐式方法还是显式方法的选择很大程度上取决于问题的性质。
