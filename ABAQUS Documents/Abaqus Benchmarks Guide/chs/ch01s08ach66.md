# 1.8.4 TEAM 2：振荡磁场中长圆柱导体的涡流模拟

**产品：** Abaqus/Standard

此基准问题是设计用于测试电磁分析方法（TEAM）的问题标准套件的一部分。要解决的问题是浸入时间谐均匀磁场中的无限导电圆柱壳。目标是计算由随时间变化的磁场在圆柱壳中感应的涡流。导体中的洛伦兹力和焦耳加热也是感兴趣的。

### 问题描述

问题设置如图 1.8.4-1（[图 1.8.4-1](ch01s08ach66.md#bmk-em-team2-geom)）所示。它描述了浸入时间谐均匀磁场中的无限导电圆柱壳。导电圆柱壳的内半径和外半径分别为 ![](../graphics/bmk_eqn00801.gif) m 和 ![](../graphics/bmk_eqn00802.gif) m。其电阻率和相对磁导率假定为 ![](../graphics/bmk_eqn00803.gif) Ω·m 和 ![](../graphics/bmk_eqn00804.gif)。磁通量密度假定具有幅值 ![](../graphics/bmk_eqn00805.gif) T，并以频率 ![](../graphics/bmk_eqn00806.gif) Hz 振荡。磁场假定沿 ![](../graphics/bmk_eqn00807.gif) 方向。我们假定圆柱壳浸入的介质具有类似于真空的特性。对于这些参数，导体的集肤深度约为 ![](../graphics/bmk_eqn00808.gif) mm，与壳厚度 ![](../graphics/bmk_eqn00809.gif) mm 相当。

### 模型和边界条件

使用磁矢量势公式来解决此问题。由于几何沿 ![](../graphics/bmk_eqn00619.gif) 方向的不变性以及磁通量密度位于 *x*–*y* 平面内的事实，只有磁矢量势的 ![](../graphics/bmk_eqn00619.gif) 分量非零。虽然几何本质上是二维的，但二维中的磁矢量势公式只能表示磁矢量势的 ![](../graphics/bmk_eqn00620.gif) 和 ![](../graphics/bmk_eqn00807.gif) 分量。因此，使用了沿 ![](../graphics/bmk_eqn00619.gif) 方向包含一个单元的三维几何。

由于问题的对称性，仅需在 *x*–*y* 平面中建模问题域的第一象限。在对称平面 ![](../graphics/bmk_eqn00810.gif) 和 ![](../graphics/bmk_eqn00811.gif) 上施加适当的边界条件。由于磁矢量势 ![](../graphics/bmk_eqn00812.gif) 沿 ![](../graphics/bmk_eqn00619.gif) 方向，对称论证要求它在 ![](../graphics/bmk_eqn00104.gif) 平面上恒为零。因此，在对称平面 ![](../graphics/bmk_eqn00104.gif) 上施加齐次 Dirichlet 边界条件 ![](../graphics/bmk_eqn00813.gif)。这里 ![](../graphics/bmk_eqn00814.gif) 表示垂直于边界面的单位法向量。类似地，在平面 ![](../graphics/bmk_eqn00105.gif) 上，对称论证要求磁场 ![](../graphics/bmk_eqn00815.gif) 垂直于该平面。因此，在对称平面 ![](../graphics/bmk_eqn00105.gif) 上施加齐次 Neumann 边界条件 ![](../graphics/bmk_eqn00816.gif)。

由于问题域是无界的，必须以某种方式截断。Abaqus 不支持吸收边界条件；因此，截断边界应选择远离导体。远离导体的边界面选择使其平行于 ![](../graphics/bmk_eqn00104.gif) 或 ![](../graphics/bmk_eqn00105.gif) 平面。远离导体的磁矢量势和磁通量密度由 ![](../graphics/bmk_eqn00817.gif) 和 ![](../graphics/bmk_eqn00818.gif) 给出，其中 ![](../graphics/bmk_eqn00819.gif) 和 ![](../graphics/bmk_eqn00279.gif) 是沿 ![](../graphics/bmk_eqn00807.gif) 和 ![](../graphics/bmk_eqn00619.gif) 坐标轴的单位向量。由于磁矢量势在平行于 ![](../graphics/bmk_eqn00810.gif) 平面的远边界面上的投影是恒定的，在此边界面上施加非齐次 Dirichlet 边界条件 ![](../graphics/bmk_eqn00820.gif)。由于磁通量密度垂直于平行于 ![](../graphics/bmk_eqn00811.gif) 平面的远边界面，因此在此边界面上施加齐次 Neumann 边界条件。

最后，由于磁矢量势沿 ![](../graphics/bmk_eqn00619.gif) 轴方向，在平行于 ![](../graphics/bmk_eqn00133.gif) 平面的边界面上施加齐次 Dirichlet 边界条件。

### 结果与讨论

[图 1.8.4-2](ch01s08ach66.md#bmk-em-team2-xyplotb) 显示了使用 Abaqus/Standard 计算的磁通量密度 ![](../graphics/bmk_eqn00807.gif) 分量的幅值与解析解的比较。图例中的标签 `EMC3D8' 和 `EMC3D4' 对应于使用这些单元进行的分析。图例中的标签 `Analytical Truncated' 和 `Analytical True' 对应于解析解，该解析解分别假定在有限距离和无穷远处的外圆柱边界面上施加 Dirichlet 边界条件，如前一节所述。该图清楚地表明，分析结果与解析结果比较非常好，并且外边界面距离圆柱壳足够远，以至于截断引入的误差很小。

[图 1.8.4-3](ch01s08ach66.md#bmk-em-team2-contoure) 显示了电场幅值的等值线图。对于时间谐分析，电场的幅值与按弧度频率缩放的磁矢量势幅值相同。该图清楚地表明，导体的存在使其附近区域中的场发生畸变。最后，[图 1.8.4-4](ch01s08ach66.md#bmk-em-team2-contourj) 描述了由于磁场在导体中感应的电流密度。该图显示，导体中的电流密度沿 ![](../graphics/bmk_eqn00620.gif) 轴较大，沿 ![](../graphics/bmk_eqn00807.gif) 轴消失。因此，导体中产生的焦耳热沿 ![](../graphics/bmk_eqn00620.gif) 轴较大。
