# 5.2.2 平移视图工具





当你选择平移工具 ![](../graphics/ico_viewPan.png) 和要工作的视口时，Abaqus/CAE 进入平移模式，如 ![](../graphics/crs-pan.gif) 光标所示。

当模型显示在视口中时，点击然后拖动光标时模型的视图位置会发生变化，橡皮筋线条表示平移量。平移视图类似于在模型快照上移动相机，如图 [图 5-3](pt01ch05s02s02.md#viw-pan1) 所示；快照在视口中移动，但模型中任何在原始相机位置隐藏的面在平移时保持隐藏。

**图 5-3** 平移视图。

![](../graphics/viw-pan1-nls.png)

平移工具的替代模式通过在执行操作时按住 **[Shift]** 来访问，它创建更真实的相机视图。不是移动模型快照，而是在真实模型上移动相机。当你在模型上移动相机时，模型中在原始相机位置隐藏的面会暴露出来，如图 [图 5-4](pt01ch05s02s02.md#viw-pan2) 所示。

**图 5-4** 在替代模式中平移视图。

![](../graphics/viw-pan2.png)

**注意：**如果透视未开启，平移的替代模式与标准平移工具工作方式相同。

对于平移工具的两种模式，光标的初始位置不重要，只要你将其放置在视口内。光标移动仅受显示器物理边界的限制，即使你将光标移动到视口或窗口外部，平移也会继续。

当 *X–Y* 绘图显示在视口中时，你可以通过在网格中点击和拖动光标来更改绘图中 *X–Y* 曲线的视图。Abaqus/CAE 会在你操作 *X–Y* 数据视图时更新轴中的值。
