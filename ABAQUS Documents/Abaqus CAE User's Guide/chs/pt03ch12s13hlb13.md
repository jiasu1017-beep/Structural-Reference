# 12.13.17 创建声学无限截面





声学无限截面用于定义建模承受小压力变化的声学介质的二维、三维和轴对称区域的截面属性。当您想提高涉及外部域的分析的准确性时，可以使用声学无限截面。您必须创建一个参考点来指示声学无限元素所需的参考节点。请参见[Abaqus Analysis User's Guide的第28.3.1节，"无限元素"](../usb/usb-link.md#usb-elm-einfinite)。

**指定声学无限截面属性：**

1. 从主菜单栏，选择****Section**![](../graphics/images/arrow.gif)**Create****。出现**Create Section**对话框。
   **提示：**您也可以在**Section Manager**中点击**Create**，或在Property模块工具箱中选择创建截面工具![](../graphics/ico_section.png)。
2. 输入截面名称。有关命名对象的更多信息，请参见["使用基本对话框组件，"第3.2.1节"](pt01ch03s02s01.md)。
3. 选择**Other**作为截面**Category**，选择**Acoustic infinite**作为截面**Type**，然后点击**Continue**。出现截面编辑器。
4. 为声学无限截面选择一种声学介质材料。如有需要，点击![](../graphics/ico_material.png)创建材料；有关更多信息，请参见["创建或编辑材料，"第12.7.1节"](pt03ch12s07hlb01.md)。
5. 为截面**Plane stress/strain thickness**输入值。如果截面将用于二维区域，则必须指定截面厚度。如果区域类型不需要厚度信息，Abaqus/CAE会忽略厚度信息。
6. 要定义用于解析无限方向声学场变化的九阶多项式的数量，请点击**Order**字段右侧的箭头以减少将使用的多项式数量（仅适用于Abaqus/Explicit分析）。默认值为10，这是Abaqus/Standard中始终使用的值。
7. 点击**OK**保存更改并关闭截面编辑器。

**为与声学无限元素一起使用创建参考点：**
- 您必须创建一个参考点，用于确定声学无限元素每个节点处的"半径"和"节点射线"。您可以使用["创建参考点，"第72.3节"](pt06ch72hla01.md)中所述的过程在Part模块或Property模块中创建与部件关联的参考点。请参见Abaqus Analysis User's Guide的第28.3.1节"无限元素"中的"为声学无限元素定义参考点和厚度"。
