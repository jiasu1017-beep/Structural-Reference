# 42.5.4 选择变形场输出变量





您可以通过生成变形图或选择变形模型作为云图或符号图的底层形状来显示变形模型。您变形模型的形状基于您选择的特定变形场输出变量的值。**Output Variable**列表中描述左侧的星号表示该变量包含复数结果。有关单个输出变量标识符的信息，请参见["输出变量，" Abaqus Analysis User's Guide第4.2节](../usb/usb-link.md#usbovsect)。

**选择变形场输出变量：**

1. 找到控制变形场输出变量的选项。从主菜单栏，选择****Result**![](../graphics/images/arrow.gif)**Field Output****。在出现的对话框中点击**Deformed Variable**选项卡。**Deformed Variable**选项变为可用。**提示：**您也可以通过点击任何出现此按钮的对话框中的**Field Output**按钮来访问这些选项。Abaqus/CAE按名称和描述列出当前[步骤和帧](pt05ch42s01hlb01.md)的[输出数据库](pt02ch09s06hlb02.md)中可用于变形图（节点向量量）的所有变量。要查看列出变量的完整描述，请通过拖动角落来增加对话框的宽度。
2. 从**Name**和**Description**列表中，点击您想要的变形场变量。列表中描述左侧的星号表示该变量包含复数结果。选中的变量高亮显示。
3. 当前视口中的变形模型形状会更改以反映您指定的变形场输出变量的值。如果处于活动状态，状态块中的文本会更改以标识与图关联的变量。有关状态块的更多信息，请参见["自定义状态块，" 第56.3节](pt05ch56s03.md)。您的更改将保存到会话结束。
