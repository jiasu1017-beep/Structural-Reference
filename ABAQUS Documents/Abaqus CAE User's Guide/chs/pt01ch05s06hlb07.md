# 5.6.7 循环浏览视图





使用**视图操作**工具栏中的循环工具 ![](../graphics/ico_viewCycle.png) 循环浏览之前的视图；Abaqus/CAE 为每个视口保存最近的八个视图。

如果当前视口链接到其他视口，Abaqus/CAE 也会循环浏览会话中所有链接视口中的视图。有关更多信息，请参阅["链接视口以进行视图操作，" 第 4.5 节](pt01ch04s05.md)。

**要循环浏览之前的视图：**

1. 从**视图操作**工具栏中，单击循环工具 ![](../graphics/ico_viewCycle.png) 进入循环模式。**提示：**你也可以从主菜单中选择****View**![](../graphics/images/arrow.gif)**Previous Views****，或按 **[F7]**。
2. 将光标定位在你要更改视图的视口中（光标变为双向箭头）；然后点击。
3. 要控制循环方向，请在提示区域中单击**向后**或**向前**。默认是向后循环。
4. 根据需要重复步骤 2 和 3 多次，以达到所需的视图。向后循环到最早的可用视图后，继续点击无效。同样，向前循环到最新的视图后，继续点击无效。
5. 要退出循环模式，请执行以下操作之一：
   - 单击鼠标按钮 2。
   - 单击提示区域中的取消 ![](../graphics/afxI_cancel.png)或**完成**按钮。
   - 单击循环视图工具。
   - 单击任何其他视图操作工具。
