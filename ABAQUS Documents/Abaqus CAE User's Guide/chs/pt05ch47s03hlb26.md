# 47.4.27 对X–Y数据对象应用Chebyshev I型或II型滤波







### 47.4.27 对X–Y数据对象应用Chebyshev I型或II型滤波

使用 **chebyshev1Filter** 或 **chebyshev2Filter** 函数对[之前保存的](pt05ch47s02hlb07.md) *X–Y* 数据对象（有序对的集合）应用Chebyshev I型或II型滤波运算，生成一个新的 *X–Y* 数据对象。例如，此滤波运算可用于去除高频噪声。

Chebyshev滤波器的传递函数显示在Abaqus Analysis User's Guide的["Output to the output database"中的"Filtering output and operating on output in Abaqus/Explicit"第4.1.3节](../usb/usb-link.md#usb-out-odboutput-filter)中。

[图47-24](pt05ch47s03hlb25.md#usv-xyplot-cfilter) 说明了可以使用 **chebyshev1Filter** 运算生成的 *X–Y* 曲线图类型。

**图47-24** 使用 **chebyshev1Filter** 运算生成的 *X–Y* 曲线图。

![](../graphics/usv-xyplot-cfilter.png)

**chebyshev1Filter** 和 **chebyshev2Filter** 函数使用相同的语法并需要相同的参数集。以下参数是必需的：*X–Y* 数据对象的名称（*name*）；截止频率（*cutoffFrequency*），即滤波器衰减至少一半输入信号的频率；以及纹波因子（*rippleFactor*），这是一个浮点数，表示您愿意用多少振荡来换取改善的滤波器响应。Chebyshev I型和II型滤波都需要大于0的纹波因子；此外，Chebyshev II型滤波要求该值小于1。Chebyshev I型滤波对纹波因子值没有上限。

两种类型的Chebyshev滤波器在响应纹波出现的位置以及纹波因子值的处理上有所不同；请参见Abaqus Analysis User's Guide的["Output to the output database"中的"Filtering output and operating on output in Abaqus/Explicit"第4.1.3节](../usb/usb-link.md#usb-out-odboutput-filter)，了解I型和II型Chebyshev输出与典型Butterworth滤波器输出的比较。

可选参数的说明如下：
- 要使用的滤波器阶数（*filterOrder*）。此参数必须是正偶数值；默认值为2。
- 指定在数据信号起点处计算的投影和预充电方法的符号常数（*startCondition*）。此参数的有效值为 ZERO（应用零常投影和预充电）；CONSTANT（应用等于 *X–Y* 数据对象中第一个数据点的常投影和预充电）；MIRROR（应用相当于围绕第一个数据点处的垂直线反射 *X–Y* 数据对象的投影和预充电）；REVERSE_MIRROR（应用相当于围绕第一个数据点处的垂直线和水平线反射 *X–Y* 数据对象的投影和预充电）；以及 TANGENTIAL（应用与前两个数据点相切的线性投影和预充电）。默认值为 CONSTANT。
- 指定在数据信号终点处计算的投影和预充电方法的符号常数（*endCondition*）。此参数的有效值为 ZERO（应用零常投影和预充电）；CONSTANT（应用等于 *X–Y* 数据对象中最后一个数据点的常投影和预充电）；MIRROR（应用相当于围绕最后一个数据点处的垂直线反射 *X–Y* 数据对象的投影和预充电）；REVERSE_MIRROR（应用相当于围绕最后一个数据点处的垂直线和水平线反射 *X–Y* 数据对象的投影和预充电）；以及 TANGENTIAL（应用与最后两个数据点相切的线性投影和预充电）。默认值为 CONSTANT。
- 指定插值方案的符号常数（*interpolation*）。此参数的有效值为 QUADRATIC（指定拉格朗日二阶插值方案）；CUBIC_SPLINE（指定三次样条插值方案）；和 LINEAR（指定线性插值方案）。默认值为 QUADRATIC。
- 到达第一个数据点之前的原始数据曲线的斜率（*startslope*）。此参数的默认值为0.0（水平斜率），仅在 interpolation=CUBIC_SPLINE 时使用。
- 越过最后一个数据点之后的原始数据曲线的斜率（*endslope*）。此参数的默认值为0.0（水平斜率），仅在 interpolation=CUBIC_SPLINE 时使用。
- 指定是否对滤波后数据执行反向传递（*backwardPass*）的布尔值。此参数的默认值为 True。当此参数设置为 False 时，*endCondition* 参数将被忽略。

您的 *X–Y* 数据对象必须具有恒定的时间步长才能被滤波。如果时间步长不是常数，Abaqus/CAE 通过[插值](pt05ch47s03s02.md)以恒定间隔计算额外点。Chebyshev I型或II型滤波的恒定时间步长由要滤波的 *X–Y* 数据对象中的最小时间步长定义。

**对X–Y数据对象应用Chebyshev I型或II型滤波：**

1. 找到 **Operate on XY Data** 对话框。从主菜单栏，选择 ****Tools**![](../graphics/images/arrow.gif)**XY Data**![](../graphics/images/arrow.gif)**Create****. 在出现的对话框中点击 **Operate on XY data**；然后点击 **Continue**。**Operate on XY Data** 对话框出现。
2. 从列出的 **Operators** 中，点击 **chebyshev1Filter(X,F,F)** 或 **chebyshev2Filter(X,F,F)**，分别用于Chebyshev I型或II型滤波。**chebyshev1Filter** 或 **chebyshev2Filter** 函数出现在表达式窗口中。
3. 从 **XY Data** 选项中，点击要操作的 *X–Y* 数据对象的名称，然后点击 **Add to Expression**。您可以从本次会话中之前保存的所有 *X–Y* 数据对象中选择（在 **XY Data** 字段中按字母顺序列出）。*X–Y* 数据对象名称出现在表达式窗口中 **chebyshev1Filter** 或 **chebyshev2Filter** 函数的括号内。
4. 将光标定位在表达式窗口中第二个逗号之前，然后输入截止频率的值。
5. 将光标定位在表达式窗口中第三个逗号之前，然后输入纹波因子的正值。对于 **chebyshev2Filter** 函数，此值必须小于1。
6. 要继续构建表达式，将光标定位在表达式窗口中，然后输入或选择要包含的函数、运算符和 *X–Y* 数据。
7. 要评估并显示表达式，点击 **Plot Expression**。
8. 要[保存](pt05ch47s02hlb07.md)您的新 *X–Y* 数据对象，点击 **Save As**，然后在出现的对话框中提供名称。保存数据对象使其可用于本次会话中的未来操作以及包含多个数据对象的 *X–Y* 曲线图。
9. 完成后，点击 **Cancel** 关闭对话框。
