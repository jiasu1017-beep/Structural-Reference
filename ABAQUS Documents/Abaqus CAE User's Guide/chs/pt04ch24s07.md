# 24.7 参考点和连接器之间是什么关系？






您可用于定义装配级线框特征的点可以是以下之一：
- 零件实例的顶点或节点
- 零件实例或装配的参考点
- 地面

使用参考点通常比使用装配上的点或节点更方便，因为同一参考点可用于刚性体约束或耦合约束。在"零件"模块中，您可以使用参考点工具集在每个零件上创建一个参考点；可以在"装配"模块、"相互作用"模块和"载荷"模块中创建其他参考点。

您可以使用模型树将参考点重命名为更具描述性的特征名称。描述性名称使您更容易在复杂模型中将参考点和线框关联起来，如图 [图 24-3](pt04ch24s07.md#adv-conn-featuremanager) 所示。

**图 24-3** 曲柄机构的模型树。

![](../graphics/adv-conn-featuremanager-nls.png)

如果您需要对同一零件的多个实例上的参考点使用不同的特征名称和标签，则必须在装配相关模块中创建参考点。您可以使用"编辑网格"工具集在网格上创建节点，然后选择该节点来定义装配级线框的端点；但是，该节点不会列在模型树中，在视口中也没有标签。有关创建和重命名参考点的更多信息，请参见[第 72 章，"参考点工具集"](pt06ch72.md)。有关在网格上创建节点的更多信息，请参见["操作节点，" 第 64.1.1 节"](pt06ch64s01s01.md)。

每个定义装配级线框端点的参考点或节点都需要与模型几何体关联，即使它们是在"零件"模块中创建的。在许多情况下，您将在参考点或节点与零件实例之间创建刚性体约束或耦合约束（将力分布到区域而不是单个点）。因此，线框端点的运动将被约束为零件实例的运动。您在"相互作用"模块中创建约束。有关更多信息，请参见["理解约束，" 第 15.5 节"](pt03ch15s05.md)。
