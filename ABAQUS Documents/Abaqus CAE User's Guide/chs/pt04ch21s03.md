# 21.3 使用几何和网格工具创建具有粘聚单元的模型





您可以使用以下过程使用几何和网格工具创建具有粘聚单元的模型：

1. 在Part模块中，定义模型的几何。 即使层的厚度接近零，您也应该将代表粘聚层的几何区域定义为实体。 为避免数值问题，建议您使用104或更大的厚度值来对几何进行建模。 如果层的实际厚度小于此值，您应在粘聚截面编辑器中的**Initial thickness**字段中指定实际厚度，如["创建粘聚截面，" 第12.13.16节](pt03ch12s13hlb12.md)中所述。
2. 在Mesh模块中，对周围散体材料进行网格划分。 您可以使用任何网格工具对周围散体材料进行网格划分。 有关更多信息，请参阅["网格生成，" 第17.3.3节](pt03ch17s03s03.md)。
3. 在Mesh模块中，使用以下方法之一对粘聚区域进行网格划分： **二维和三维模型** 自顶向下扫掠或自底向上网格划分技术。 您可以分配自顶向下扫掠网格划分技术或自底向上网格划分技术来对粘聚区域进行网格划分。 自底向上网格划分技术仅适用于三维模型（有关更多信息，请参阅["自底向上网格划分，" 第17.11节](pt03ch17s11.md)）。 无论您选择哪种网格划分技术，都必须沿单元厚度方向扫掠、拉伸或旋转网格以产生正确的单元方向。 对于复杂的粘聚区域，您可能需要划分模型以创建一组可以一致对齐的扫掠区域。 有关更多信息，请参阅["选择网格划分技术，" 第17.18.3节](pt03ch17s18s03.md)和["指定扫掠路径，" 第17.18.6节](pt03ch17s18s06.md)。 **三维模型** 将粘聚区域转换为壳区域，并使用偏移网格划分技术。 1. 使用Part模块中的**From solid**壳工具将实体零件转换为壳。 2. 使用Geometry Edit工具集中的**Remove faces**工具隔离代表零件理想化壳的面集合。 3. 使用壳单元对简化模型进行网格划分，并创建网格零件。 4. 使用网格零件生成从现有网格偏移的实体六面体或楔形单元网格。 单元将通过零件厚度定向，您可以使用Query工具集进行验证。 有关详细说明，请参阅["从现有网格生成偏移的实体单元层，" 第64.7.1节](pt06ch64s07s01.md)。
4. 在Mesh模块中，使用单元类型分配工具将粘聚单元类型分配给粘聚区域。 有关更多信息，请参阅["单元类型分配，" 第17.5.3节](pt03ch17s05s03.md)。
