# 5.1.3 排气歧管组件

**产品：** Abaqus/Standard   

发动机排气歧管通常在运行期间和关闭期间承受严重的热循环。歧管的热膨胀和收缩受到与其螺栓连接的气缸盖的相互作用约束。这些约束决定了歧管的热机械疲劳寿命。

初始装配程序包括用规定的螺栓力将歧管法兰螺栓连接到气缸盖，这会产生均匀的轴向螺栓应力。在随后的运行条件（如热循环和蠕变）下，这些螺栓力可能会增加或松弛，从而可能改变法向压力并导致气缸盖和歧管法兰之间的横向滑动。因此，歧管法兰上的边界约束是整个组件对其运行条件响应的函数。因此，这些边界约束不能预先规定。本示例展示了如何使用 Abaqus 的规定装配载荷功能来模拟这些变化的边界约束。

问题场景包括三个步骤：
1. 施加规定的螺栓载荷以将排气歧管紧固到气缸盖。
2. 使组件承受稳态运行温度分布。
3. 将组件返回到环境温度条件。

### 几何与模型

分析的排气歧管组件由一个四管排气歧管和三个法兰组成，用七个螺栓紧固到气缸盖的一小部分。

歧管由灰铸铁铸造而成，杨氏模量为 138 GPa，泊松比为 0.283，热膨胀系数为 13.8 x 10-6 /C。

### 载荷与边界约束

在分析的第一步中，每个七个螺栓被紧固到 20 kN 的均匀螺栓力。在后续步骤中，监测螺栓载荷的变化，因为螺栓响应于组件上的热载荷。

### 结果与讨论

分析以小位移分析的形式执行。问题中的非线性来自变化的接触条件、摩擦滑动和粘滞以及温度依赖性塑性。

### 输入文件

- manifold.inp - 分析的输入数据
- manifold_node_elem.inp - 节点和单元定义
- manifold_nodaltemp.inp - 节点温度数据
