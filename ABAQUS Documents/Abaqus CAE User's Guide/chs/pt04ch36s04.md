# 36.4 在表皮或桁条加固上生成单元





您可以使用Mesh模块在三维部件实例的面上的表皮加固上生成二维单元。类似地，您可以在轴对称部件实例的边缘上的桁条加固上生成一维单元。当底层几何体被网格划分时，Abaqus/CAE会生成表皮或桁条单元；您不能独立于底层几何体对表皮或桁条加固进行网格划分。

[图36-3](pt04ch36s04.md#mgn-basics-skins)显示了在上表面带有表皮加固的三维部件实例。

**图36-3** 带表皮加固的三维部件实例。

![](../graphics/mgn-basics-skins-nls.png)

当部件实例被网格划分时，表皮或桁条单元与三维单元共享相同的节点和网格拓扑。如果您为表皮单元分配不同的几何阶，您还应该更改底层单元的阶。更多信息，请参见["为表皮或桁条加固分配单元类型"，第36.5节](pt04ch36s05.md)。

如果您在孤立网格上创建表皮或桁条，表皮或桁条单元与底层孤立网格单元共享相同的节点和网格拓扑。如果删除孤立网格单元，Abaqus/CAE会自动再生与已删除单元关联的任何表皮或桁条。再生后，这些表皮或桁条具有新的单元ID号，因此此再生会使包含先前与已删除单元关联的表皮或桁条的任何集合失效。因此，如果从孤立网格中删除单元，您应该更新包含先前与已删除单元关联的表皮或桁条的任何集合定义。
