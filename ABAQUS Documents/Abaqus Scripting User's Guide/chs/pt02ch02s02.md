# 2.2 Abaqus GUI应用程序的组件是什么？





创建GUI应用程序涉及许多组件。[图2-1](pt02ch02s02.md#cus-gst-overview-figure)显示了这些组件及其连接方式的概述。本节简要概述每个组件。这些组件在后续章节中有更详细的讨论。

**图2-1** Abaqus GUI应用程序概述。

![](../graphics/cus-gst-overview-nls.png)

**Widget**

在应用程序的最低层级，您使用widget通过图形用户界面从用户那里收集输入。例如，文本字段widget呈现一个用户可以在其中输入数字的框。类似地，复选按钮widget呈现一个小框，用户可以点击它来打开或关闭选项。

**布局管理器**

布局管理器通过提供对齐选项来排列widget。例如，水平框架将widget排列成一行。垂直框架将widget排列成一列。

**对话框**

对话框将widget分组在布局管理器中，并呈现特定功能所需的所有输入。例如，**Print**对话框呈现允许用户指定应打印什么以及如何打印的所有控件。

**模式**

模式是控制特定用户界面显示的GUI机制。模式还负责发出与该用户界面关联的命令。例如，当您选择****File**![](../graphics/images/arrow.gif)Print****时，模式启动。该模式显示**Print**对话框，并在用户点击**OK**时发出打印命令。

**模块和工具集**

模块和工具集将功能分组在一起。GUI模块是类似功能的分组，例如Abaqus/CAE中的零件模块。GUI工具集与GUI模块类似，因为它将类似功能分组，但它通常包含可能被一个或多个GUI模块使用的更具体的功能。Abaqus/CAE中的基准工具是GUI工具集的示例。

**应用程序**

应用程序负责高级活动，例如管理应用程序使用的GUI进程和更新widget的状态。此外，应用程序负责与桌面窗口管理器的交互。
