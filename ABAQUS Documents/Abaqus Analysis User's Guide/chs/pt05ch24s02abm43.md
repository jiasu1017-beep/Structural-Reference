### 概述

损伤演化模型描述材料在损伤起始后的软化行为。该模型：
- 基于损伤变量 D（从 0 到 1）；
- 支持多种软化方式；
- 考虑能量耗散；以及
- 与损伤起始模型结合使用。

### 稳定化

数值稳定化用于控制损伤演化过程中的收敛：

| **输入文件用法：** | ``` [*DAMAGE EVOLUTION](../key/key-link.md#usb-kws-mdamageevolution), STABILIZATION ``` |
| --- | --- |

### 与损伤起始的结合

损伤演化必须与损伤起始结合使用：

| **输入文件用法：** | ``` [*DAMAGE INITIATION](../key/key-link.md#usb-kws-mdamageinitiation) [*DAMAGE EVOLUTION](../key/key-link.md#usb-kws-mdamageevolution) ``` |
| --- | --- |

### 元素删除

当损伤变量达到临界值时，元素将被删除：

| **输入文件用法：** | ``` [*DAMAGE EVOLUTION](../key/key-link.md#usb-kws-mdamageevolution), ELEMENT_DELETION=参数 ``` |
| --- | --- |

### 元素

损伤演化模型可与 Abaqus/Standard 和 Abaqus/Explicit 中的连续体元素和壳元素一起使用。
