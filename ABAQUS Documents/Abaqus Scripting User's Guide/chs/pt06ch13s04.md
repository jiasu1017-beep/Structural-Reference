# 13.4 工具栏





默认情况下，Abaqus/CAE将所有工具栏显示在主菜单栏下方的同一行中（参见[图13-2](pt06ch13s04.md#cus-top-toolbar)）：

**图13-2** Abaqus/CAE工具栏。

![](../graphics/top-toolbar-nls.png)

你可以使用以下语句从定义工具栏组的模块或工具集访问工具栏组：
```
toolbar = self.getToolbarGroup(toolbarName) 
```
其中*self*是模块或工具集，*toolbarName*是Abaqus/CAE构建工具栏时赋予工具栏的名称。你可以通过从主菜单栏选择****工具**![](../graphics/images/arrow.gif)**自定义****并查看出现的对话框来确定工具栏的名称。
