# 4.1.41 VWAVE





### 4.1.41 [`VWAVE`](../sub/sub-link.md#sub-xsl-vwave)

**产品：**Abaqus/Explicit  Abaqus/Aqua  

### 测试的功能

用户子程序 [`VWAVE`](../sub/sub-link.md#sub-xsl-vwave) 用于通过 Abaqus/Explicit 指定 Abaqus/Aqua 分析中的波浪运动。

### I. PB_VWAVE-AIRY-PERIOD_P31_XPL.inp

### 测试的单元

B31    PIPE31    

### 问题描述

验证问题 [pb_vwave-airy-period_p31_xpl.inp](../eif/pb_vwave-airy-period_p31_xpl.inp) 与 [pb_airy_p31_xpl.inp](../eif/pb_airy_p31_xpl.inp) 完全相同，详见 ["动态压力，闭端浮力载荷"中的"Aqua 载荷情况，" 第 3.12.1 节](ch03s12abv237.md#ver-prc-dynpressbuoy)。它测试了用户子程序 [`VWAVE`](../sub/sub-link.md#sub-xsl-vwave) 中编码的 Airy 波的动态压力实现和闭端浮力载荷。在这个问题中，一个垂直桩被完全约束并承受浮力载荷。Airy 波理论用于计算动态分析中结构上的总反作用力。分布式载荷类型 PB 用于 50 单元（PIPE31）模型，集中载荷类型 TSB 用于单单元（B31）模型。

**模型：**

| 结构高度 | 175.0（平均水位以下 100.0，以上 75.0） |
| --- | --- |
| 管道截面数据 | *r* = 1.0，*t* = 0.25 |

**材料：**

| 杨氏模量 | 1×10⁶ |
| --- | --- |

**Aqua—环境：**

| 海底高程 | 100.0 |
| --- | --- |
| 平均水位 | 1100.0 |
| 重力常数 | 32.2 |
| 流体质量密度 | 2.0 |

### 结果与讨论

结果与 [pb_airy_p31_xpl.inp](../eif/pb_airy_p31_xpl.inp) 中计算的结果完全相同，并与解析确定的峰值总反作用力非常吻合。

### 输入文件

[pb_vwave-airy-period_p31_xpl.inp](../eif/pb_vwave-airy-period_p31_xpl.inp)

此分析的输入文件。

[pb_vwave-airy-length_p31_xpl.inp](../eif/pb_vwave-airy-length_p31_xpl.inp)

相同的分析，但指定了相应的波长而不是波周期。

[vwave_airy.f](../eif/vwave_airy.f)

用于 pb_vwave-airy-period_p31_xpl.inp 和 pb_vwave-airy-length_p31_xpl.inp 的用户子程序 [`VWAVE`](../sub/sub-link.md#sub-xsl-vwave)。

[pb_vwave-airy-period-opt_p31_xpl.inp](../eif/pb_vwave-airy-period-opt_p31_xpl.inp)

从波周期计算波长可能代价较高。此优化输入文件最初计算一次波长，并将波参数适当存储以供后续使用。

[vwave_airy-opt.f](../eif/vwave_airy-opt.f)

用于 pb_vwave-airy-period-opt_p31_xpl.inp 的用户子程序 [`VWAVE`](../sub/sub-link.md#sub-xsl-vwave)。
