# W







### wedge angle

An angle used to define initial curvature of a model when you create a generalized plane strain section. A wedge angle is given about the global 1- and 2-axes at the reference point indicating the reference node required by generalized strain plane elements.
For more information:- ["Generalized plane strain elements" in "Choosing the element's dimensionality," Section 27.1.2 of the Abaqus Analysis User's Guide](../usb/usb-link.md#usb-elm-edimension-genplanestrain)

- ["Creating generalized plane strain sections," Section 12.13.2 of the Abaqus/CAE User's Guide](../usi/usi-link.md#usi-prp-section-genplanestrain)

### wire edge

A geometric region in Abaqus/CAE represented by a line or curve, as opposed to the edge of a solid or shell feature.
For more information:- ["Removing wire edges," Section 69.6.6 of the Abaqus/CAE User's Guide](../usi/usi-link.md#usi-rep-help-repair-wireedges)

### work directory

The directory into which Abaqus/CAE writes files during a session and the directory from which you started Abaqus/CAE unless you specified the directory using ****File**![](../graphics/images/arrow.gif)**Set Work Directory****.
For more information:- ["Setting the work directory," Section 9.7.8 of the Abaqus/CAE User's Guide](../usi/usi-link.md#usi-dbs-mdb-changework)

- ["Using file selection dialog boxes," Section 3.2.10 of the Abaqus/CAE User's Guide](../usi/usi-link.md#uss-int-dialog-selection)
