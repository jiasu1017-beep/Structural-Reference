# 64.1.4 细化网格





网格编辑工具集提供以下工具，允许您细化平面三角形单元网格：
- 设置尺寸。您可以为重新网格划分过程指定全局单元尺寸。Abaqus/CAE会更改新网格的密度以反映新的目标单元尺寸。如果网格包含线性或二次单元，您可以更改单元尺寸。
- 移除尺寸。您可以在重新网格划分过程中保持全局单元尺寸。Abaqus/CAE会在改善内部网格质量的同时，保持零件边界上单元的边缘。如果网格包含线性或二次单元，您可以细化网格。
- 重新网格划分。您可以重新网格划分包含线性或二次三角形单元的平面网格。

**提示：**网格编辑撤销功能可以回滚任何网格细化更改。有关更多信息，请参见["撤销或重做网格编辑工具集中的更改，" 第64.9节](pt06ch64s09.md)。

有关网格细化的详细说明，请参见["编辑和细化孤兒网格，" 第64.8节](pt06ch64s08.md)。
