# 26.1.3 场膨胀

**产品：** Abaqus/Standard  

##### **参考文献**

- ["Material library: overview," Section 21.1.1](pt05ch21s01abo18.md)
- ["UEXPAN," Section 1.1.30 of the Abaqus User Subroutines Reference Guide](../sub/sub-link.md#sub-rtn-uuexpan)
- [*EXPANSION](../key/key-link.md#usb-kws-mexpansion)

### 概述

场膨胀效应：
- 可以通过指定场膨胀系数来定义，以便 Abaqus/Standard 可以计算由预定义场变量变化驱动的场膨胀应变；
- 可以是各向同性、正交各向异性或完全各向异性；
- 定义为从预定义场变量的参考值开始的总膨胀；
- 可以指定为温度和/或预定义场变量的函数；
- 可以直接在用户子程序 [`UEXPAN`](../sub/sub-link.md#sub-sxl-uexpan) 中指定（如果场膨胀应变是场变量和状态变量的复杂函数）；以及
- 可以为多个预定义场变量定义。

### 定义场膨胀系数

场膨胀是包含在材料定义中的材料属性（见["Material data definition," Section 21.1.2](pt05ch21s01aus109.md)），除非它指的是其材料属性未作为材料定义一部分定义的垫片的膨胀。在这种情况下，必须与垫片行为定义结合使用场膨胀（见["Defining the gasket behavior directly using a gasket behavior model," Section 32.6.6](pt06ch32s06alm51.md)）。

| **输入文件用法：** | ``` [*MATERIAL](../key/key-link.md#usb-kws-mmaterial) [*EXPANSION](../key/key-link.md#usb-kws-mexpansion), FIELD=n ``` |
| --- | --- |

### 定义方向依赖的场膨胀

可以定义各向同性、正交各向异性或完全各向异性场膨胀。

正交各向异性和各向异性场膨胀只能与使用局部方向定义材料方向的材料一起使用（见["Orientations," Section 2.2.5](pt01ch02s02aus15.md)）。

只有超弹性和超泡沫材料模型允许各向同性场膨胀。

#### 各向同性膨胀

如果场膨胀系数是直接定义的，则每个温度和/或预定义场变量需要一个值。如果使用用户子程序 [`UEXPAN`](../sub/sub-link.md#sub-sxl-uexpan)，则只需定义一个各向同性场膨胀应变增量。

| **输入文件用法：** | ``` [*EXPANSION](../key/key-link.md#usb-kws-mexpansion), FIELD=n, TYPE=ISO ``` |
| --- | --- |

#### 正交各向异性膨胀

如果场膨胀系数是直接定义的，则应给出主材料方向上的三个膨胀系数作为温度和/或预定义场变量的函数。如果使用用户子程序 [`UEXPAN`](../sub/sub-link.md#sub-sxl-uexpan)，则必须在主材料方向上定义三个场膨胀应变增量分量。

| **输入文件用法：** | ``` [*EXPANSION](../key/key-link.md#usb-kws-mexpansion), FIELD=n, TYPE=ORTHO ``` |
| --- | --- |

#### 各向异性膨胀

如果场膨胀系数是直接定义的，则必须给出所有六个分量作为温度和/或预定义场变量的函数。如果使用用户子程序 [`UEXPAN`](../sub/sub-link.md#sub-sxl-uexpan)，则必须定义所有六个场膨胀应变增量分量。

| **输入文件用法：** | ``` [*EXPANSION](../key/key-link.md#usb-kws-mexpansion), FIELD=n, TYPE=ANISO ``` |
| --- | --- |

### 元素

场膨胀可用于 Abaqus/Standard 中除使用一般截面行为的梁和壳元素外的任何应力/位移元素。
