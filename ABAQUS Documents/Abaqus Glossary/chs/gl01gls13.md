# T







### tab

Abaqus/CAE 对话框中带标签的选项组。
更多信息：- [「使用由标签分隔的对话框，」Abaqus/CAE User's Guide 第 3.2.6 节](../usi/usi-link.md#uss-int-dialog-tab)

### text field

Abaqus/CAE 对话框中可以输入信息的区域。
更多信息：- [「使用基本对话框组件，」Abaqus/CAE User's Guide 第 3.2.1 节](../usi/usi-link.md#uss-int-dialog-basic)

### thread-based execution

使用多个同时执行线程的并行操作模式，使 Abaqus 能够将计算密集型操作拆分为多个同时任务。在多处理器或多核系统上，线程同时在不同处理器或内核上运行。Abaqus 支持使用 SMP 的基于线程的执行。
更多信息：- [「并行执行：概述，」Abaqus Analysis User's Guide 第 3.5.1 节](../usb/usb-link.md#usb-int-aparallelmodes)

### through hole

完全穿透三维实体的孔。孔的路径继续延伸到无穷远，并在其交叉的任何位置切割物体。
更多信息：- [「添加切割特征，」Abaqus/CAE User's Guide 第 11.24 节](../usi/usi-link.md#usi-prt-help-addcut)

### TIFF（Tag Image File Format）

一种标准位图图形文件格式，常用于存储图形图像。根据显示设备的不同，TIFF 格式提供调整彩色和灰度图像的选项，并且可以编码非常高质量的图像。Abaqus/CAE 允许您将选定 viewport 的图像保存为 TIFF 格式文件。Abaqus/CAE 不会压缩 TIFF 格式文件中存储的数据；因此，这些文件可能会占用大量磁盘空间。
更多信息：- [「打印图像格式，」Abaqus/CAE User's Guide 第 8.1.1 节](../usi/usi-link.md#uss-pri-conc-gif)

### time history

根据实际分析结果随时间变化的绘图序列。如果时间历史是基于帧的，动画会逐帧递增；如果时间历史是基于时间的，动画会沿公共时间线递增。
更多信息：- [「自定义同步 viewports 的时间历史，」Abaqus/CAE User's Guide 第 49.4.7 节](../usi/usi-link.md#usv-anm-timehist)

### toolbar

Abaqus/CAE 中包含用于管理文件和查看模型的一组便捷工具的区域。工具栏中的项目是也可从主菜单栏使用的功能的快捷方式。默认情况下，所有标准工具栏以一行形式直接出现在 Abaqus/CAE 主菜单栏下方；但是，您可以将工具栏配置为出现在屏幕上的任何位置。
更多信息：- [「工具栏的组件，」Abaqus/CAE User's Guide 第 2.2.3 节](../usi/usi-link.md#uss-top-toolbar)

### toolbox

提供快速访问常用 Abaqus/CAE 功能的图标集合。工具箱中的工具也可从主菜单栏使用。
更多信息：- [「了解和使用的工具箱和工具栏，」Abaqus/CAE User's Guide 第 3.3 节](../usi/usi-link.md#uss-int-toolbox)

### toolset

允许您在 Abaqus/CAE 中执行特定建模任务的功能单元。
更多信息：- [「什么是 toolset？，」Abaqus/CAE User's Guide 第 2.4 节](../usi/usi-link.md#uss-top-menu-toolset)

### top-down meshing

Abaqus/CAE 用于通过从区域几何向下工作到元素和节点来创建网格的自动网格划分过程。另请参阅 [bottom-up meshing](gl01gls03.md#gls-bottomup)。
更多信息：- [第 17 章，「The Mesh module，」Abaqus/CAE User's Guide](../usi/usi-link.md#usi-mgn)

### triad

请参阅 [view orientation triad](gl01gls23.md#gls-xyztriad) 或 [material orientation triad](gl01gls14.md#gls-mattriad)。

### truss sections

桁架区域截面属性。桁架对细长的杆状结构建模，提供轴向强度但与梁不同，不提供弯曲刚度。
更多信息：- [「桁架单元，」Abaqus Analysis User's Guide 第 29.2.1 节](../usb/usb-link.md#usb-elm-etruss)

- [「定义截面，」Abaqus/CAE User's Guide 第 12.2.3 节](../usi/usi-link.md#usi-prp-prop-section)

- [「创建桁架截面，」Abaqus/CAE User's Guide 第 12.13.12 节](../usi/usi-link.md#usi-prp-section-truss)
