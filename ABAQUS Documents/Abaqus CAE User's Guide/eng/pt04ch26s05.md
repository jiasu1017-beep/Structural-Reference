# 26.5 Specifying the interface region and coupling schemes







In each model, you define a co-simulation interaction to specify the interface region (region for exchanging data) and coupling schemes for the co-simulation. Only one co-simulation interaction can be active in each model, and the settings in each co-simulation interaction must be the same in each model. For more information, see ["Defining a Standard-Explicit co-simulation interaction," Section 15.13.14](pt03ch15s13hlb12.md), and ["Defining a fluid-structure co-simulation interaction," Section 15.13.15](pt03ch15s13hlb13.md).
