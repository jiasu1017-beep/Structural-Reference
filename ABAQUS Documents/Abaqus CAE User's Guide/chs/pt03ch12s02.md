# 12.2 理解属性





您可以通过创建截面并将其分配给部件来指定部件或部件区域的属性。在大多数情况下，截面引用您已定义的材料。您定义的梁截面也引用您已定义的轮廓。本指南的本节解释材料、轮廓、截面、钢筋和截面分配。您可以使用Property模块编辑器创建材料、轮廓和截面，如["理解Property模块编辑器，"第12.4节](pt03ch12s04.md)中所述。涵盖以下主题：

- ["定义轮廓，"第12.2.2节](pt03ch12s02s02.md)
- ["定义截面，"第12.2.3节](pt03ch12s02s03.md)
- ["定义复合铺层，"第12.2.4节](pt03ch12s02s04.md)
- ["理解壳截面中的钢筋，"第12.2.5节](pt03ch12s02s05.md)
