# 21.2 在现有三维网格中嵌入粘聚单元





您可以使用以下过程嵌入一层粘聚单元：

1. 在Mesh模块中，使用Edit Mesh工具集中的实体偏移网格工具将单元嵌入现有网格（请参阅["编辑整个网格，" 第64.7节](pt06ch64s07.md)）。 此方法生成一层与周围散体材料共享节点的六面体或楔形单元。 偏移网格工具生成的单元方向与沿偏移方向对齐的堆栈方向一致。 当提示您选择将从中生成偏移网格的单元面时，使用["选择内部表面，" 第6.2.12节](pt01ch06s02hlb11.md)中描述的过程选择内部单元面。 **注意：**当生成嵌入单元层时，厚度应该远小于相邻单元的厚度，因为生成偏移层时会移动节点。 您只能从三维单元面创建偏移网格。 因此，您只能使用偏移网格创建六面体和楔形粘聚单元。 例如，您不能通过从单元边缘偏移来创建四边形粘聚单元。
2. 在Mesh模块中，使用单元类型分配工具将粘聚单元类型分配给粘聚区域。 有关更多信息，请参阅["单元类型分配，" 第17.5.3节](pt03ch17s05s03.md)。

如果您的现有网格是原生网格，在添加粘聚单元之前应创建网格零件。 有关更多信息，请参阅["创建网格零件，" 第17.20节](pt03ch17s20.md)。

如果您想在粘聚层中使用更细的网格，应将粘聚层构建为单独的零件。 您应该将周围散体材料中的网格分成两个区域，并具有适当的间距以容纳粘聚层。 您可以使用["使用几何和网格工具创建具有粘聚单元的模型，" 第21.3节](pt04ch21s03.md)中所述的方法对粘聚层进行网格划分，并可将粘聚层绑定到周围的散体材料。 有关更多信息，请参阅["定义粘聚层与周围散体材料之间的绑定约束，" 第21.4节](pt04ch21s04.md)。
