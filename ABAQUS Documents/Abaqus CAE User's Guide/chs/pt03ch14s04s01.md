# 14.4.1 什么是输出请求？





Abaqus分析产品在步骤的每个增量中计算许多变量的值。通常你只对所有这些计算数据的一小部分感兴趣。你可以通过创建输出请求来指定要写入输出数据库的数据。输出请求包含以下信息：
- 感兴趣的变量或变量分量。
- 将值写入输出数据库的模型区域和积分点。
- 变量或分量值写入输出数据库的速率。

当你创建第一个步骤时，Abaqus/CAE选择对应于步骤分析过程的默认输出变量集。默认情况下，从模型中的每个节点或积分点以及默认截面点请求输出。此外，Abaqus/CAE选择将变量写入输出数据库的默认速率。你可以编辑这些默认输出请求或创建和编辑新的输出请求。

默认输出请求和你修改的输出请求会被传播到分析中的后续步骤。如果你有一个包含默认输出请求并从大量帧请求输出的大型模型，生成的输出数据库将非常大。你可以使用C++程序从大型输出数据库中提取数据，仅将选定的帧复制到第二个输出数据库。更多信息，请参见["通过在特定帧处保留数据来减少输出数据库中的数据量，"Abaqus脚本用户指南第10.15.4节](../cmd/cmd-link.md#cmd-odb-intro-exa-odbfilter-cpp)。

分析完成后，使用可视化模块读取输出数据库并以图形方式显示写入的数据。
