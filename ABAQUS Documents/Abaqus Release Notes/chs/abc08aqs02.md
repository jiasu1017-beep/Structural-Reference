# 9.2 增强型热绑定连接的区域分解







**产品：**Abaqus/Explicit  

**优点：**热绑定连接的区域分解已得到改进。改进使得热绑定连接以及参与绑定约束的面单元的区域分解更加均衡。对于具有大型热绑定连接的模型，性能得到了显著提升，并且在使用大量处理器执行时观察到更好的扩展性。

**说明：**关键字接口没有变化。改进的区域分解是默认行为。
