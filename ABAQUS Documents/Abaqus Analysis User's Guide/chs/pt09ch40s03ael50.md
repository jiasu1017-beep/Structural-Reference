### 概述

本节提供Abaqus/Standard中可用管对管接触单元的参考。

### 单元类型

| ITT21 | 用于二维梁和管单元的管对管单元 |
| --- | --- |

| ITT31 | 用于三维梁和管单元的管对管单元 |
| --- | --- |

##### 激活的自由度

ITT21: 1, 2

ITT31: 1, 2, 3

##### 附加解变量

ITT21: 两个与接触力相关的附加变量。

ITT31: 三个与接触力相关的附加变量。

### 需要的节点坐标

ITT21: *X*, *Y*

ITT31: *X*, *Y*, *Z*

### 单元属性定义

| **输入文件用法：** | 使用以下选项来标识第二个（外部）管，指定ITT接触单元可以在其上相互作用的第一（内部）管： |
| --- | --- |
|  | ``` [*SLIDE LINE](../key/key-link.md#usb-kws-mslideline) ``` 使用以下选项给出管的径向间隙作为正数，用于在一个管内建模另一个管内的管滑动： ``` [*INTERFACE](../key/key-link.md#usb-kws-minterface) ``` 当单元对两个管的外表面之间的接触建模时，将两个管的外半径之和指定为负数。 |

### 基于单元的加载

无。

### 单元输出

#### 应力分量

| S11 | 两个管之间的法向力分量。 |
| --- | --- |

| S12 | 两个管之间的剪切力，平行于第二个（外部）管的轴。 |
| --- | --- |

| S13 | 两个管之间的剪切力，垂直于接触方向和第二个（外部）管的轴（仅适用于ITT31）。 |
| --- | --- |

#### 应变分量

| E11 | 垂直于第二个（外部）管中心线切线的表面过闭。 |
| --- | --- |

| E12 | 两个管之间的累积相对切向运动，平行于第二个（外部）管的轴。 |
| --- | --- |

| E13 | 两个管之间的累积相对切向运动，垂直于接触方向和第二个（外部）管的轴（仅适用于ITT31）。 |
| --- | --- |
