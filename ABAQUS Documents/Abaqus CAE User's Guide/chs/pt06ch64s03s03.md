# 64.3.3 减少网格偏移期间的单元畸变和坍缩





Abaqus/CAE通过以下方式创建偏移网格：
- 沿着现有网格表面的边界法向偏移节点，以及
- 沿法向扩展的单元。

当您从凹面网格表面偏移创建网格时，单元往往会汇聚，如[图64-11](pt06ch64s03s03.md#edt-offset-fanin)所示。

**图64-11** 从凹面偏移时单元汇聚。

![](../graphics/edt-offset-fanin-nls.png)

因此，单元可能会坍缩或倒置，偏移量的大小受曲率和单元尺寸的组合限制，如[图64-12](pt06ch64s03s03.md#edt-offset-limit)所示。

**图64-12** 当您在凹面区域偏移小单元时，单元可能会坍缩或倒置。

![](../graphics/edt-offset-limit-nls.png)

您可以尝试通过从**偏移网格**对话框中选择**恒定厚度环绕角落**来避免单元坍缩和倒置。选择此选项后，Abaqus/CAE会逐渐变细从尖锐角落延伸的偏移单元。 resulting 单元具有恒定的节点偏移距离，而不是单元面层之间的恒定距离。[图64-13](pt06ch64s03s03.md#edt-offset-compare)说明瞭请求恒定厚度环绕角落对偏移网格图案的影响。

**图64-13** 请求恒定厚度环绕角落对偏移网格图案的影响。

![](../graphics/edt-offset-compare-nls.png)

对偏移网格图案的影响对于实体和壳偏移网格是相似的。您还可以通过在凹面区域使用较大的单元尺寸来减少单元畸变。但是，只有当您可以重新生成要偏移的网格时，才能控制单元尺寸。

相反，当您从凸面网格表面偏移创建网格时，单元往往会散开，如[图64-14](pt06ch64s03s03.md#edt-offset-fanout)所示。

**图64-14** 从凸面偏移时单元发散。

![](../graphics/edt-offset-fanout-nls.png)
