# 55.1 曲线图显示自定义概述





您可以自定义模型在未变形、变形、云图、符号和材料取向曲线图中的外观。对于未变形和变形曲线图状态,应用于模型形状的所有自定义选项都是曲线图状态无关的。也就是说,Abaqus/CAE将自定义应用于所有未变形和变形曲线图,包括那些也显示云图、符号或材料取向的曲线图。曲线图状态无关选项控制渲染样式和填充颜色;单元和表面边缘;单元颜色;模型标签、节点符号和显示体的外观;相机移动;各种模型实体的显示;曲线边缘的平滑度;无结果的单元颜色;以及模型扫略和拉伸。使用主菜单栏中的"**Options**"菜单可从中选择这些曲线图状态无关的自定义选项。

从主菜单栏选择 ****Tools**![](../graphics/images/arrow.gif)**Color Code**** 可为单个单元着色。从主菜单栏选择 ****Options**![](../graphics/images/arrow.gif)**Display Body**** 可自定义模型中所有显示体的外观。从主菜单栏选择 ****View**![](../graphics/images/arrow.gif)**View Options**** 可自定义相机的移动。要选择剩余的曲线图状态无关选项,从主菜单栏选择 ****View**![](../graphics/images/arrow.gif)**ODB Display Options****。如果您选择为同一曲线图状态将变形模型形状叠加在未变形形状上,请从主菜单栏选择 ****Options**![](../graphics/images/arrow.gif)**Superimpose**** 可独立于变形形状自定义未变形形状。与其他曲线图状态无关的自定义一样,这些曲线图选项也应用于所有曲线图状态中的叠加曲线图。

**注意:**在视口中显示模型标签和/或符号会增加刷新曲线图和动画显示所需的时间。

除了曲线图状态无关的自定义选项外,您还可以为云图、符号和材料取向曲线图状态选择曲线图状态相关的选项。曲线图状态相关选项对于每种曲线图类型都是唯一的。例如,您可以自定义云图颜色谱、符号的大小和样式,以及材料取向三轴的显示。如果可以在公共和云图曲线图选项中使用独立和相关曲线图选项进行类似的自定义(如模型边缘的颜色),则曲线图状态相关的选项会覆盖独立选项。
