# 14.3.5 用Abaqus/Explicit过程替换Abaqus/Standard过程，反之亦然





如果您想用Abaqus/Explicit分析过程替换Abaqus/Standard分析过程，反之亦然，您的模型中必须只有一个分析步骤，所需的过程类型才会出现在**替换步骤**对话框中。如果您的模型包含多个步骤，您可以使用步骤相关管理器将对象移动到单个步骤。然后您可以删除其他步骤，并用新的分析过程替换剩余的步骤。

例如，如果您想将包含四个**静态，通用**过程的模型从Abaqus/Standard分析更改为Abaqus/Explicit分析，您可以使用**载荷管理器**将所有载荷移动到四个步骤之一。类似地，您可以使用**相互作用管理器**移动相互作用。然后您可以删除其他三个步骤，并用**动态，显式**过程替换剩余的步骤。如果需要，您可以创建额外的Abaqus/Explicit步骤，并使用步骤相关管理器将步骤替换期间复制的对象移动到适当的Abaqus/Explicit过程。

更多信息，请参见["修改步骤相关对象的历史，"第3.4.6节](pt01ch03s04s06.md)。
