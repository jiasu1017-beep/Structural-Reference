# 35.3.6 网格无关紧固件









**产品：** Abaqus/StandardAbaqus/ExplicitAbaqus/CAE

##### **参考资料**

- ["绑定约束," Section 35.3.1](pt08ch35s03aus132.md)
- ["Surfaces: overview," Section 2.3.1](pt01ch02s03aus16.md)
- [*FASTENER*](#)

### 概述

网格无关紧固件（Mesh-Independent Fasteners）用于连接两个或多个表面，不依赖于网格划分。紧固件可以模拟螺栓、铆钉、焊点等连接件。

### 紧固件类型

| 类型 | 描述 | 应用 |
| --- | --- | --- |
| BEAM | 梁式紧固件 | 螺栓、铆钉 |
| SPRING | 弹簧式紧固件 | 弹性连接 |
| CARTESIAN | 笛卡尔紧固件 | 规则排列 |
| GENERAL | 一般紧固件 | 用户定义 |

### 梁式紧固件（ BEAM ）

梁式紧固件模拟刚性或柔性连接：

| **输入文件用法：** | ```
*FASTENER, TYPE=BEAM, BEHAVIOR=behavior
surface_1
surface_2
``` |
| --- | --- |

### 弹簧式紧固件（ SPRING ）

弹簧式紧固件模拟弹性连接：

| **输入文件用法：** | ```
*FASTENER, TYPE=SPRING, SPRING BEHAVIOR=stiffness
surface_1
surface_2
``` |
| --- | --- |

### 笛卡尔紧固件（ CARTESIAN ）

笛卡尔紧固件用于定义规则排列的紧固件：

| **输入文件用法：** | ```
*FASTENER, TYPE=CARTESIAN
surface_1, surface_2
x_coord, y_coord, z_coord
``` |
| --- | --- |

### 紧固属性

紧固件行为定义：

| **输入文件用法：** | ```
*FASTENER BEHAVIOR, NAME=name
spring_stiffness, damping
``` |
| --- | --- |

### 网格无关性

网格无关紧固件的主要优势：

- 不依赖于表面网格节点位置
- 自动插值计算紧固件位置
- 可以在任意位置添加紧固件
- 简化网格划分过程

### 应用场景

- 薄壁结构连接
- 复合材料层合板
- 车身结构
- 航空航天结构
