# 17.21.1 创建重新网格划分规则






重新网格划分规则使 Abaqus/CAE 能够使您的网格自适应迭代，以满足您指定的误差指示器目标和尺寸标准。有关更多信息，请参阅[「什么是重新网格划分规则？，」第 17.13.1 节](pt03ch17s13s01.md)。

**创建重新网格划分规则：**

1. 从主菜单栏，选择****自适应**![](../graphics/images/arrow.gif)**重新网格划分规则**![](../graphics/images/arrow.gif)**创建****。**提示：**您也可以使用位于网格模块工具箱中与网格工具一起的 ![](../graphics/ico_adaptiveRemeshRules.png) 工具来创建重新网格划分规则。（有关更多信息，请参阅[「使用网格模块工具箱，」第 17.15 节](pt03ch17s15.md)。）
2. 选择要应用重新网格划分规则的区域，或单击**完成**以选择整个模型。如果您将重新网格划分规则分配给从属实例，Abaqus/CAE 会对原始零件进行重新网格划分，并且每个从属实例继承相同的网格。区域的建模空间必须是同构的。例如，Abaqus/CAE 不允许您选择同时包含实体和壳的区域。如有必要，您应该使用分区工具集来隔离将产生应力奇异性的区域，并从重新网格划分规则中排除这些区域。有关更多信息，请参阅 [Abaqus Analysis User's Guide 第 12.3.1 节「自适应重新网格划分：概述」中的「奇异性」](../usb/usb-link.md#usb-anl-aadpover-singularities)。
3. 此时会出现**创建重新网格划分规则**对话框。
4. 如果需要，使用**名称**文本字段更改新规则的名称。
5. 如果需要，使用**描述**文本字段输入重新网格划分规则的描述。您可以使用描述来帮助您跟踪重新网格划分规则的范围和目的。**重新网格划分规则管理器**显示重新网格划分规则的名称和描述。
6. 单击**步骤和指示器**选项卡以选择以下内容：- 将应用重新网格划分规则的步骤。- Abaqus/CAE 将写入输出数据库的误差指示器输出变量，以及写入它们的频率。有关更多信息，请参阅[「为重新网格划分规则选择步骤和误差指示器输出变量，」第 17.21.2 节](pt03ch17s21s02.md)。
7. 单击**尺寸方法**选项卡以选择以下内容：- Abaqus/CAE 将在重新网格划分过程中用于计算单元尺寸的方法。- 是否对自适应重新网格划分过程使用误差指示器目标的自动减少，还是指定误差指示器目标。有关更多信息，请参阅[「选择重新网格划分规则尺寸方法，」第 17.21.3 节](pt03ch17s21s03.md)。
8. 单击**约束**选项卡以选择重新网格划分过程中对单元尺寸的约束。有关更多信息，请参阅[「选择重新网格划分规则约束，」第 17.21.4 节](pt03ch17s21s04.md)。
9. 单击**确定**以创建重新网格划分规则并关闭**创建重新网格划分规则**对话框。
