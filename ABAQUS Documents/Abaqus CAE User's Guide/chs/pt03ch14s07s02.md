# 14.7.2 求解器控制





您可以自定义控制迭代线性方程求解器的变量。

**注意：**您只能将迭代线性方程求解器用于**静态，通用**；**静态，线性扰动**；**粘弹性**；**热传递**；**地静力**；和**土**分析步骤。

您可以通过从主菜单栏选择**其他**![](../graphics/images/arrow.gif)**求解器控制**来访问求解器控制。更多信息，请参见["迭代线性方程求解器，"第6.1.6节，Abaqus分析用户指南](../usb/usb-link.md#usb-anl-aitrsolveroverview)。

有关设置求解器控制的详细说明，请参见["自定义求解器控制，"第14.15.2节](pt03ch14s15hlb02.md)。
