# 46 绘制材料取向

材料取向图显示模型单元在单元积分点处的材料方向。材料取向在分析的指定步骤和帧以及壳单元的指定截面点绘制。材料取向图也可以创建来显示膜单元、壳单元和表面单元中增强材料（钢筋方向）的材料方向。Abaqus/CAE将材料取向表示为三轴。本章解释材料取向绘制。涵盖以下主题：
