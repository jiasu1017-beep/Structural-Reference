# *CENTROID






### *CENTROID定义梁截面质心的位置。

此选项只能与 [*BEAM GENERAL SECTION](ch02abk05.md)、SECTION=GENERAL 或 [*BEAM GENERAL SECTION](ch02abk05.md)、SECTION=MESHED 选项结合使用。用于定义截面相对于局部（1, 2）轴系的位置。

**产品：**Abaqus/Standard  Abaqus/Explicit  Abaqus/CAE  

**类型：**模型数据  

**级别：**部件、部件实例  

**Abaqus/CAE: **属性模块
