# 47.6.3 自定义 X–Y 绘图图例





### 47.6.3 自定义 X–Y 绘图图例

X–Y 绘图的图例提供了图例，用于关联绘图中包含的每条曲线及其所代表的 X–Y 数据对象。每个图例条目包含一个线段，其颜色和样式与绘图中的显示相同。图例条目还包括文本，默认情况下，该文本说明曲线所表示的 X–Y 数据的名称。您可以使用 **曲线选项（Curve Options）** 自定义此文本。

图例的其他特征由 **视口注释选项（Viewport Annotation Options）** 对话框中的选项控制。有关自定义图例以及在图例中包含 X–Y 最小值和最大值报告的信息，请参阅 ["自定义图例，" 第 56.1 节"](pt05ch56s01.md)。

**要自定义 X–Y 绘图的图例文本：**

1. 找到 X–Y 曲线的 **属性（Attributes）** 选项。**属性（Attributes）** 框架位于 **曲线选项（Curve Options）** 对话框的底部。
2. 在 **曲线（Curves）** 选项中，选择要修改其图例文本的一条或多条曲线。
3. 在 **图例文本（Legend Text）** 字段中，输入自定义图例文本。您也可以通过切换 **使用默认（Use default）** 来恢复默认图例文本。
4. 点击 **关闭（Dismiss）** 关闭 **曲线选项（Curve Options）** 对话框。

# 47.5.9 自定义 X–Y 绘图坐标轴标签





### 47.5.9 自定义 X–Y 绘图坐标轴标签

点击 **坐标轴（Axes）** 选项卡来自定义 X–Y 绘图中 X 轴和 Y 轴上主 [刻度线](pt05ch47s04hlb06.md) 旁边的数字标签。坐标轴标签示例如 [图 47–41](pt05ch47s04hlb09.md#viw-axislabel) 所示。

**图 47–41** 带坐标轴标签的 X–Y 绘图。

![](../graphics/viw-axislabel-nls.png)

您可以自定义以下内容：
- 标签的格式（自动、十进制、工程或科学）。
- 标签中的小数位数或有效数字位数。
- 标签相对于主刻度线的频率。
- 标签的字体。

标签的颜色由坐标轴的颜色控制。有关更多信息，请参阅 ["自定义 X–Y 绘图坐标轴颜色和样式，" 第 47.5.10 节"](pt05ch47s04hlb10.md)。

**要控制 X–Y 绘图的坐标轴标签：**

1. 找到 **标签（Labels）** 选项。选择 ****选项（Options）**![](../graphics/images/arrow.gif)**XY 选项（XY Options）**![](../graphics/images/arrow.gif)**坐标轴（Axis）****；或点击 ![](../graphics/ico_xyAxisOptions.png)，该图标位于可视化模块工具箱的 X–Y 绘图工具中。点击出现的对话框中的 **坐标轴（Axes）** 选项卡。坐标轴的 **标签（Labels）** 选项出现在 **坐标轴（Axes）** 页面的中间。
2. 从 **X 轴（X Axis）** 或 **Y 轴（Y Axis）** 字段中，高亮显示一个或多个轴。
3. 指定坐标轴标签的放置位置。从 **标签（Labels）** 选项中的 **放置位置（Placement）** 列表中，选择以下之一：**无（None）** Abaqus/CAE 隐藏所选轴的坐标轴标签。**内部（Inside）** Abaqus/CAE 在与 X–Y 曲线相同的轴侧显示坐标轴标签。**外部（Outside）** Abaqus/CAE 在与 X–Y 曲线相对的轴侧显示坐标轴标签。
4. 点击 **频率（Frequency）** 箭头指定标签相对于每个主刻度线的打印频率。例如，如果选择频率为 2，则标签将沿轴显示在每隔一个主刻度线处。如果选择频率为 0，则不会显示任何标签。您可以请求的标签频率最多比主刻度线数多 1。
5. 从 **标签（Labels）** 选项中，选择所选标签的格式。1. 点击 **格式（Format）** 按钮显示 X 轴标签格式选项。2. 选择以下之一：**自动（Automatic）** 非常大的值和非常小的值以科学计数法表示。所有其他值均以无指数形式表示，并使用指定的有效数字位数。**十进制（Decimal）** 所有值均以无指数形式表示，并使用指定的有效数字位数。**工程（Engineering）** 大于或等于 1000（或小于或等于 0.001）的值表示为 1 到 999 之间的数字乘以 10 的 n 次方，其中 n 是 3 的倍数（例如，`20.5E+03` 或 `17.76E+06`）。所有其他值使用指定的有效数字位数表示。**科学（Scientific）** 所有值表示为 1 到 10 之间的数字乘以 10 的适当次方（例如，`2.05E+04` 或 `1.776E+07`）。
6. 点击 **精度（Precision）** 箭头指定每个标签中有效数字（非十进制格式）或小数位（十进制格式）的所需数量。
7. 选择坐标轴标签的字体。1. 点击 ![](../graphics/ico_text16.png)。Abaqus/CAE 显示 **选择字体（Select Font）** 对话框。2. 使用 **选择字体（Select Font）** 对话框中的方法选择新字体、大小和样式。有关更多信息，请参阅 ["自定义字体，" 第 3.2.8 节"](pt01ch03s02s08.md)。3. 点击 **应用（Apply）** 查看字体选择的效果。4. 点击 **确定（OK）** 关闭 **选择字体（Select Font）** 对话框。坐标轴标签变为所选的字体、大小和样式。
8. 选择坐标轴标签的颜色。1. 点击颜色样本 ![](../graphics/usi-colorsample.png)。Abaqus/CAE 显示 **选择颜色（Select Color）** 对话框。2. 使用 **选择颜色（Select Color）** 对话框中的方法选择新颜色。有关更多信息，请参阅 ["自定义颜色，" 第 3.2.9 节"](pt01ch03s02s09.md)。3. 点击 **确定（OK）** 关闭 **选择颜色（Select Color）** 对话框。坐标轴标签变为所选颜色。
9. 点击 **关闭（Dismiss）** 关闭 **坐标轴选项（Axis Options）** 对话框。

# 47.5.10 自定义 X–Y 绘图坐标轴颜色和样式





### 47.5.10 自定义 X–Y 绘图坐标轴颜色和样式

点击 **坐标轴（Axes）** 选项卡来自定义 X–Y 绘图的坐标轴颜色和粗细。您的选择适用于 X 轴和 Y 轴；无法单独自定义坐标轴。

**要自定义 X–Y 绘图中坐标轴的颜色和样式：**

1. 找到 **样式（Style）** 选项。选择 ****选项（Options）**![](../graphics/images/arrow.gif)**XY 选项（XY Options）**![](../graphics/images/arrow.gif)**坐标轴（Axis）****；或点击 ![](../graphics/ico_xyAxisOptions.png)，该图标位于可视化模块工具箱的 X–Y 绘图工具中。点击出现的对话框中的 **坐标轴（Axes）** 选项卡。坐标轴的 **样式（Style）** 选项出现在 **坐标轴（Axes）** 页面的底部。
2. 从 **X 轴（X Axis）** 或 **Y 轴（Y Axis）** 字段中，高亮显示一个或多个轴。
3. 选择线条样式。从 **样式（Style）** 列表中，点击所需的线条样式（实线、虚线等）。所选轴以及 **样式（Style）** 列表中会显示指定的线条样式。
4. 选择线条粗细。从 **粗细（Thickness）** 列表中，点击所需的线条粗细。所选轴以及 **粗细（Thickness）** 列表中会显示指定的线条粗细。
5. 选择线条颜色：1. 点击颜色样本 ![](../graphics/usi-colorsample.png)。Abaqus/CAE 显示 **选择颜色（Select Color）** 对话框。2. 使用 **选择颜色（Select Color）** 对话框中的方法选择新颜色。有关更多信息，请参阅 ["自定义颜色，" 第 3.2.9 节"](pt01ch03s02s09.md)。3. 点击 **确定（OK）** 关闭 **选择颜色（Select Color）** 对话框。坐标轴线变为所选颜色。
6. 点击 **关闭（Dismiss）** 关闭 **坐标轴选项（Axis Options）** 对话框。

# 47.6.1 自定义 X–Y 曲线外观概述





### 47.6.1 自定义 X–Y 曲线外观概述

要自定义 X–Y 曲线的外观，请在结果树中展开 **曲线（Curves）** 容器，高亮显示要修改的曲线，然后点击鼠标按钮 3 并从出现的列表中选择 **曲线选项（Curve Options）**。您可以自定义每条曲线的线条样式、符号样式和图例文本。

只有最新的 X–Y 绘图中显示的曲线才能进行自定义。您可以自定义以下内容：
- 图例中描述数据的文本。有关更多信息，请参阅 ["自定义 X–Y 绘图图例，" 第 47.6.3 节"](pt05ch47s05hlb02.md)。
- 表示数据（曲线）的线条外观。有关更多信息，请参阅 ["自定义 X–Y 曲线外观，" 第 47.6.4 节"](pt05ch47s05hlb03.md)。
- 曲线沿线符号的外观。有关更多信息，请参阅 ["自定义 X–Y 曲线上使用的符号，" 第 47.6.5 节"](pt05ch47s05hlb04.md)。

**要设置曲线选项：**

1. 找到 **曲线选项（Curve Options）** 对话框。您可以使用以下任何方法打开此对话框：- 从主菜单栏中，选择 ****选项（Options）**![](../graphics/images/arrow.gif)**XY 选项（XY Options）**![](../graphics/images/arrow.gif)**曲线（Curve）****。- 从可视化模块工具箱中，点击 ![](../graphics/ico_xyCurveOptions.png)。- 从结果树中，展开 **图表（Charts）** 容器，高亮显示要修改的绘图或多个绘图，点击鼠标按钮 3，并从出现的列表中选择 **图表选项（Chart Options）**。
2. 使用曲线选项来选择然后自定义当前视口中 X–Y 曲线的外观。
3. 完成自定义 X–Y 曲线的外观后，点击 **关闭（Dismiss）** 关闭 **曲线选项（Curve Options）** 对话框。
