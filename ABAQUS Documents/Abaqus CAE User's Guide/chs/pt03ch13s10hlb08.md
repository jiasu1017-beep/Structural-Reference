# 13.10.8 将零件或模型实例平移到另一个实例





从主菜单栏选择****Instance**![](../graphics/images/arrow.gif)**Translate To****可以通过沿定义运动方向的矢量平移一个实例来定位两个实例，直到可动实例的所选曲面或边缘与固定实例的所选曲面或边缘之间的距离达到指定值。

Abaqus/CAE仅在你模型大小相关的容差范围内计算接触。因此，除非在两个平面之间应用，否则接触可能不精确。如果所选曲面或边缘在Abaqus/CAE平移可动实例时永远不会接触，则不会应用平移。

**要将零件或模型实例平移到另一个实例：**

1. 从主菜单，选择****Instance**![](../graphics/images/arrow.gif)**Translate To****。
   **提示：**你也可以使用装配模块工具箱中的![](../graphics/ico_instanceTranslateTo.png)工具来定义**Translate To**约束。有关装配工具箱中工具的图表，请参见["使用装配模块工具箱，"第13.9节](pt03ch13s09.md)。Abaqus/CAE在提示区域显示提示来指导你完成过程。
2. 从将移动的零件或模型实例以及将保持固定的实例中选择曲面（对于三维零件实例）或边缘（对于二维零件实例）。你可以从固定和可动实例中选择多个曲面或边缘。如果你不确定可动实例沿选定矢量移动时模型的哪部分会接触，选择多个曲面或边缘会很有用。但是，为了加快处理速度，你应该尽可能少地选择曲面或边缘。你不能选择基准平面。
3. 选择定义运动方向的矢量的起点和终点。你可以选择的任意现有顶点或基准点，或者可以在提示区域的文本框中输入坐标。如果实例是轴对称的，平移矢量必须平行于旋转轴。
4. 在提示区域出现的文本框中，输入两个所选曲面之间的间隙值；负值表示过盈。平移不保存为特征，完成过程后你不能更改间隙。
5. 从提示区域，点击**Preview**。Abaqus/CAE沿平移矢量移动可动零件或模型实例，直到可动实例的所选曲面与固定实例的所选曲面之间被指定间隙分隔。
6. 如果实例的新位置正确，点击提示区域中的**Done**。尝试平移实例可能会与现有位置约束冲突。Abaqus/CAE仅沿未约束自由度分量应用平移分量。如果所有自由度都被约束，Abaqus/CAE显示错误消息，平移失败。要避免冲突，你可以尝试反转将移动的实例和将保持固定的实例的选择。或者，你可以将现有约束转换为绝对位置并重新应用平移。
