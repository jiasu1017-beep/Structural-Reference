# 12.13.8 创建膜截面





膜截面用于定义空间中提供面内强度但没有弯曲刚度的薄表面的截面属性。请参见[Abaqus Analysis User's Guide的第29.1.1节，"膜元素"](../usb/usb-link.md#usb-elm-emembrane)。

**指定膜截面属性：**

1. 从主菜单栏，选择****Section**![](../graphics/images/arrow.gif)**Create****。出现**Create Section**对话框。
   **提示：**您也可以在**Section Manager**中点击**Create**，或在Property模块工具箱中选择创建截面工具![](../graphics/ico_section.png)。
2. 输入截面名称。有关命名对象的更多信息，请参见["使用基本对话框组件，"第3.2.1节"](pt01ch03s02s01.md)。
3. 选择**Shell**作为截面**Category**，选择**Membrane**作为截面**Type**，然后点击**Continue**。出现膜截面编辑器。
4. 为膜截面选择一种材料。如有需要，点击**Create**创建材料；有关更多信息，请参见["创建或编辑材料，"第12.7.1节"](pt03ch12s07hlb01.md)。
5. 指定**Membrane thickness**。
   - 选择**Value**，并输入膜厚度值。
   - 选择**Element distribution**；并选择解析场（标记为(A)）或基于元素的离散场（标记为(D)）以定义空间变化的基于元素的膜厚度。或者，您可以点击![](../graphics/ico_analyticalField.png)创建新解析场或点击![](../graphics/ico_discreteField.png)创建新离散场。请参见[第58章，"解析场工具集"](pt06ch58.md)和[第63章，"离散场工具集"](pt06ch63.md)，了解更多。
6. 指定**Section Poisson's ratio**以定义膜厚度如何随变形变化。
   - 切换**Use analysis default**以使用默认值。在Abaqus/Standard中，默认值为0.5，这将强制元素不可压缩行为。在Abaqus/Explicit中，默认值基于元素材料定义中的厚度变化。
   - 切换**Specify value**，并输入泊松比值。此值必须介于1.0和0.5之间。值为0.0将强制恒定厚度，负值将导致膜拉伸应变时厚度增加。
7. 点击膜截面编辑器底部的![](../graphics/ico_rebarLayers.png)以在膜截面中定义钢筋层，如["定义钢筋层，"第12.13.19节"](pt03ch12s13hlb15.md)中所述。
8. 点击**OK**保存更改并关闭膜截面编辑器。
