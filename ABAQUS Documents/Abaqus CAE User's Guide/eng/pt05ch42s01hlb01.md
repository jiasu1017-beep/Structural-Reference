# 42.3.1 Selecting a specific results step and frame







You can select a specific step and frame at which Abaqus obtains model data and results from an output database, and you can select a specific step at which Abaqus obtains model data from a model in the current model database. In an output database, available steps and frames consist of those for which analysis results have been saved; in a model database, all steps are available for data display. To view field output variables that you have created during the Abaqus/CAE session, if any, you must select ****Result**![](../graphics/images/arrow.gif)**Step/Frame**![](../graphics/images/arrow.gif)**Session Step****. To view parametric shape variations defined in your analysis, if any, you must select ****Result**![](../graphics/images/arrow.gif)**Step/Frame**![](../graphics/images/arrow.gif)**Step 0****.

It is possible to open an output database for an analysis that is still in progress. As the analysis moves toward completion, the lists of completed steps and frames are updated every time you close and then reopen the **Step/Frame** dialog box.

For information on stepping through results frames, see ["Stepping through frames," Section 42.3.2](pt05ch42s01hlb02.md).

**To select a specific results step and frame:**

1. Locate the **Step/Frame** options. From the main menu bar, select ****Result**![](../graphics/images/arrow.gif)**Step/Frame****. The **Step/Frame** dialog box appears. The **Step** portion of the dialog lists the available steps, and the **Frame** portion of the dialog lists the available frames for the current [output database](pt02ch09s06hlb02.md) if an output database is selected. **Note:**You can also access these options by clicking ![](../graphics/ico_stepSmall.png) in the **Field Output** dialog box.
2. From the **Step** list, click the step to display. The selected step is highlighted, and Abaqus/CAE refreshes the **Frame** list to show only frames available for the selected step.
3. From the **Frame** list, click the frame to display. Select **Frame 0** to display the base state of the current step; for example, to contour initial stresses. The selected frame is highlighted.
4. The model plot in the current viewport changes to show your model at the step and frame you have selected. If active, the text in the state block changes to identify the selected step and frame. For more information on the state block, see ["Customizing the state block," Section 56.3](pt05ch56s03.md). Abaqus refreshes the [**Field Output**](pt05ch42s03hlb01.md) dialog box to list variables available for the selected frame. (You can access the **Field Output** dialog box by clicking the **Field Output** button on the bottom of the **Step/Frame** dialog box.) Abaqus also refreshes all dialog boxes in which the current step and frame are identified. Your changes are saved for the duration of the session.
