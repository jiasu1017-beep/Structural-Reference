# 12.2.3 定义截面





截面包含关于部件或部件区域属性的信息。截面定义中所需的信息取决于所讨论的区域类型。例如，如果区域是可变形线、壳或二维实体，您必须为该区域分配一个提供其横截面几何信息的截面。同样，刚性区域需要一个描述其质量属性的截面。大多数截面必须引用材料名称。梁截面还必须引用轮廓名称。

当您将截面分配给部件时，Abaqus/CAE会自动将该截面分配给该部件的每个实例。因此，当您对这些部件实例进行网格划分时创建的元素将具有该截面中指定的属性。

截面是独立于任何特定区域、部件或装配进行命名和创建的。您可以将单个截面分配给任意多个不同区域。您可以使用Property模块创建实体截面、壳截面、梁截面、流体截面和其他截面。

**实体截面**

实体截面定义二维、三维和轴对称实体区域的截面属性。
- **均匀实体截面**。均匀实体截面由材料名称组成。此外，如果截面将用于二维区域，您还必须指定截面厚度。（即使截面将分配给三维区域，您也可以选择指定平面应力或平面应变厚度。如果区域类型不需要厚度信息，Abaqus/CAE会忽略厚度信息。）有关更多信息，请参见["创建均匀实体截面，"第12.13.1节](pt03ch12s13hlb01.md)。
- **广义平面应变截面**。广义平面应变截面由材料名称、厚度和关于全局1轴和2轴的楔角组成。您只能将广义平面应变截面分配给二维平面区域。有关更多信息，请参见["创建广义平面应变截面，"第12.13.2节](pt03ch12s13hlb02.md)。
- **欧拉截面**。欧拉截面由材料名称列表组成。此列表指定欧拉域中可能存在的所有材料。您只能将欧拉截面分配给欧拉部件。有关更多信息，请参见["创建欧拉截面，"第12.13.3节](pt03ch12s13hlb03.md)。有关欧拉分析的概述，请参见[第28章，"欧拉分析"](pt04ch28.md)。
- **复合实体截面**。复合实体截面由材料层组成。对于每一层材料，您必须指定材料名称、厚度和取向。有关更多信息，请参见["创建复合实体截面，"第12.13.4节](pt03ch12s13hlb04.md)。
- **电磁实体截面**。电磁实体截面对电磁模型有效，由材料名称组成。此外，如果截面将用于二维区域，您还必须指定截面厚度。（即使截面将分配给三维区域，您也可以选择指定平面应力或平面应变厚度。如果区域类型不需要厚度信息，Abaqus/CAE会忽略厚度信息。）有关更多信息，请参见["创建电磁实体截面，"第12.13.5节](pt03ch12s13hlb05.md)。

**壳截面**

壳截面定义壳区域的截面属性。壳用于对一维（厚度）明显小于其他两维且厚度方向应力可忽略不计的结构进行建模。您可以在壳截面中定义一层或多层增强（钢筋）。有关更多信息，请参见["理解壳截面中的钢筋，"第12.2.5节](pt03ch12s02s05.md)。
- **均匀壳截面**。均匀壳截面由壳厚度、材料名称、截面泊松比和可选的钢筋层组成。您可以选择在分析前提供截面属性数据，或让Abaqus在分析期间从截面积分点计算（积分）横截面行为。如果选择后者，则提供选项来控制截面积分和厚度方向的温度变化。有关更多信息，请参见["创建均匀壳截面，"第12.13.6节](pt03ch12s13s01.md)。
- **复合壳截面**。复合壳截面由材料层、截面泊松比和可选的钢筋层组成。对于每一层材料，您必须指定材料名称、厚度和取向。您可以选择在分析前提供截面属性数据，或让Abaqus在分析期间从截面积分点计算（积分）横截面行为。如果选择后者，则提供选项来控制截面积分和厚度方向的温度变化。有关更多信息，请参见["创建复合壳截面，"第12.13.7节](pt03ch12s13s02.md)。
- **膜截面**。膜表示空间中提供面内强度但没有弯曲刚度的薄表面。膜截面由材料名称、膜厚度、截面泊松比和可选的钢筋层组成。有关更多信息，请参见["创建膜截面，"第12.13.8节](pt03ch12s13hlb06.md)。
- **表面截面**。表面截面表示空间中本身没有固有刚度且行为类似于零厚度膜元素的表面。表面截面由可选的钢筋层组成。有关更多信息，请参见["创建表面截面，"第12.13.9节](pt03ch12s13hlb07.md)。
- **通用壳刚度截面**。通用壳刚度截面允许您通过直接指定刚度矩阵和热膨胀响应来定义壳的力学响应。通用壳刚度截面由截面刚度矩阵和缩放模量组成。可选地，您还可以指定热膨胀系数和截面中的热应力。有关更多信息，请参见["创建通用壳刚度截面，"第12.13.10节](pt03ch12s13s03.md)。

**梁截面**

梁用于二维和三维中建模细长的杆状结构，提供轴向强度和弯曲刚度。梁表示横截面被认为比长度小得多的结构。您只能将梁截面分配给线区域。此外，您必须为所有具有梁截面的区域分配梁截面取向。
- **梁截面**。梁截面由截面泊松比和对轮廓的引用组成。根据您选择在分析前还是分析期间计算（积分）截面刚度，需要附加信息。有关轮廓信息，请参见["定义轮廓，"第12.2.2节](pt03ch12s02s02.md)。有关梁截面的更多信息，请参见["创建梁截面，"第12.13.11节](pt03ch12s13s04.md)。
- **桁架截面**。与梁一样，桁架用于二维和三维中建模细长的杆状结构，提供轴向强度但没有弯曲刚度。桁架截面由材料名称和横截面积组成。有关更多信息，请参见["创建桁架截面，"第12.13.12节](pt03ch12s13hlb08.md)。

您可以使用部件显示选项来查看沿线区域的梁或桁架轮廓的理想化表示。有关更多信息，请参见["控制梁轮廓显示，"第76.7节](pt07ch76hla06.md)。

**流体截面**

流体截面定义三维流体区域的材料属性。这些截面仅在Abaqus/CFD模型中使用。
- **均匀流体截面**。均匀流体截面由材料名称组成。有关更多信息，请参见["创建均匀流体截面，"第12.13.13节](pt03ch12s13hlb09.md)。

**其他截面**

您可以创建的其他截面包括垫片截面、内聚截面、声学无限截面和声学界面截面。
- **垫片截面**（仅限Abaqus/Standard分析）。垫片对位于结构组件之间的薄密封组件进行建模。垫片截面用于为密封组件提供压力-闭合行为。垫片截面由材料名称、初始垫片厚度、初始间隙、初始空隙和横截面积组成。有关更多信息，请参见["创建垫片截面，"第12.13.15节](pt03ch12s13hlb11.md)和[第32章，"垫片"](pt04ch32.md)。
- **内聚截面**。内聚截面用于建模有限厚度的粘合剂、可忽略厚度的粘合层（用于脱粘应用）以及垫片。不可使用专门的热圈行为（通常以压力与闭合的关系定义）。内聚截面由材料名称、响应、初始厚度和面外厚度组成。有关更多信息，请参见["创建内聚截面，"第12.13.16节](pt03ch12s13hlb12.md)和[第21章，"粘接接头和粘接界面"](pt04ch21.md)。
- **声学无限截面**。声学无限截面用于对涉及外部域的小压力变化的声学介质进行建模。声学无限截面由声学介质材料名称组成。此外，如果截面将用于二维区域，您还必须指定截面厚度。（即使截面将分配给三维区域，您也可以选择指定平面应力或平面应变厚度。如果区域类型不需要厚度信息，Abaqus/CAE会忽略厚度信息。）有关更多信息，请参见["创建声学无限截面，"第12.13.17节](pt03ch12s13hlb13.md)。
- **声学界面截面**。声学界面截面用于将声学介质与结构模型耦合。声学界面截面由声学介质材料名称组成。此外，如果截面将用于二维区域，您还必须指定截面厚度。（即使截面将分配给三维区域，您也可以选择指定平面应力或平面应变厚度。如果区域类型不需要厚度信息，Abaqus/CAE会忽略厚度信息。）有关更多信息，请参见["创建声学界面截面，"第12.13.18节](pt03ch12s13hlb14.md)。

**警告：**分配给部件的截面类型必须与您在Mesh模块中分配给该部件实例的元素类型一致。例如，如果在Property模块中为线部件分配了桁架截面，则应在Mesh模块中为该部件的任何实例分配桁架元素类型（而不是梁元素类型）。
