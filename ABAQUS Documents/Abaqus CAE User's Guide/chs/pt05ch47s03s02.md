# 47.4.42 从X–Y数据对象的末端截断数据





### 47.4.42 从X–Y数据对象的末端截断数据

使用**truncate**函数对[之前保存的](pt05ch47s02hlb07.md) *X–Y*数据对象（有序对的集合）进行运算，生成一个新的*X–Y*数据对象，该对象排除*X*坐标值小于用户指定最小值的数据点或*X*坐标值大于用户指定最大值的数据点。

[图47-37](pt05ch47s03hlb40.md#usv-xyplot-truncate)展示了一个通过使用**truncate**函数在最大值为7处截断的*X–Y*图。

**图47–37** 使用**truncate**函数生成的*X–Y*图。

![](../graphics/usv-xyplot-truncate.png)

**要从X–Y数据对象的末端截断数据：**

1. 找到**Operate on XY Data**对话框。从主菜单栏，选择****Tools**![](../graphics/images/arrow.gif)**XY Data**![](../graphics/images/arrow.gif)**Create****。在出现的对话框中点击**Operate on XY data**；然后点击**Continue**。**Operate on XY Data**对话框出现。
2. 从列出的**Operators**中，点击**truncate(X,F)**。**truncate**函数出现在表达式窗口中。
3. 从**XY Data**选项中，点击要运算的*X–Y*数据对象的名称，然后点击**Add to Expression**。您可以从本会话中之前保存的所有*X–Y*数据对象中选择（按**XY Data**字段中的字母顺序列出）。*X–Y*数据对象名称出现在表达式窗口中**truncate**函数括号内。
4. 指定应从*X–Y*数据对象中移除数据点的*X*坐标值上限或下限。要指定最大值，将光标放在表达式窗口中第二个逗号之前，然后输入最大值的数字。要指定最小值，请在表达式窗口中的表达式后附加`startX=*value*`。
5. 如果需要，自定义截断顺序值，该值决定Abaqus/CAE是否应在您指定的截断边界或边界处创建新数据点，如果创建，这些新数据点应如何计算。选择以下之一： - 如果`truncOrder=2`（默认值），Abaqus/CAE对数据进行分段线性拟合，以在每个边界处确定一个新数据点，并将新点或点包含在新的*X–Y*数据对象中。 - 如果`truncOrder=3`，Abaqus/CAE对数据进行样条插值，以在每个边界处确定一个新数据点，并将新点或点包含在新的*X–Y*数据对象中。 - 如果`truncOrder=1`，Abaqus/CAE不会在边界值处创建任何新点。
6. 如果需要，自定义截断容差值。如果您选择在截断边界处包含插值值，Abaqus/CAE会在找到已非常接近边界的现有数据点时覆盖这些新数据点的创建。默认情况下，截断容差（`truncTol`）为0.005，或*X–Y*数据对象中相邻数据点之间*X*方向平均距离的百分之一。如果Abaqus/CAE发现现有数据点比截断容差值更接近截断边界，则不会在该边界处创建新的插值数据点。
7. 要继续构建表达式，将光标放在表达式窗口中，然后输入或选择要包含的函数、运算符和*X–Y*数据。
8. 要计算并显示表达式，点击**Plot Expression**。
9. 要[保存](pt05ch47s02hlb07.md)新的*X–Y*数据对象，点击**Save As**，然后在出现的对话框中提供名称。保存数据对象使其可用于本会话中的后续操作，以及包含多个数据对象的*X–Y*图中。
10. 完成后，点击**Cancel**关闭对话框。
