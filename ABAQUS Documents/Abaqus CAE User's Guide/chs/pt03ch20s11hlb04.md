# 20.11.4 创建倾斜构造线







使用 Sketcher 工具箱中的倾斜构造线工具帮助沿相对于 Sketcher 网格 *X* 轴指定角度的线定位和对齐对象。旋转 Sketcher 网格不会影响现有的倾斜构造线，但新的倾斜构造线从旋转后网格的 *X* 轴测量。下图说明了倾斜构造线和一对垂直构造线如何用于定位矩形的顶点。（虚线表示构造几何。）

![](../graphics/ske-congeom-angle.png)

您还可以使用倾斜构造线来定义旋转实体和曲面的旋转轴。

**草绘倾斜构造线：**

1. 从 Sketcher 工具箱中的构造工具中，选择倾斜构造线工具![](../graphics/ico_sketchConstLineAngular.png)。有关 Sketcher 工具箱中工具的图表，请参见["Sketcher 工具，"第 20.4.1 节](pt03ch20s04s01.md)。Abaqus/CAE 在提示区域显示提示以指导您完成该过程。
2. 在提示区域出现的文本框中，输入以度为单位的角度值。正角度从水平轴逆时针测量，负角度从水平轴顺时针测量。当您在 Sketcher 工作表上移动光标时，构造线会移动。
3. 点击将位于倾斜构造线上的点。或者，您可以在提示区域出现的文本字段中输入其 *X-Y* 坐标。
4. 要创建其他倾斜构造线，重复从第 2 步开始的上述步骤。
5. 完成创建倾斜构造线后，执行以下操作之一：- 在 Abaqus/CAE 窗口中的任意位置点击鼠标按钮 2。- 在 Sketcher 工具箱中选择任何其他工具。- 点击提示区域中的取消按钮![](../graphics/afxI_cancel.png)。
