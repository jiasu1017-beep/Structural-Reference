### 概述

不对称轴对称问题中的接触建模：
- 需要使用接触单元（ISL或IRS）；
- 需要在每个周向平面上有独立的接触单元；和
- 只能发生在某些周向平面上。

### 不对称轴对称问题中的接触建模

CAXA或SAXA单元（见["具有非线性、非对称变形的轴对称实体单元，" 第28.1.7节"](pt06ch28s01ael06.md)，和["具有非线性、非对称变形的轴对称壳单元，" 第29.6.10节"](pt06ch29s06ael20.md)）用于建模初始轴对称结构可能发生非对称变形的问题。这些非对称变形可能包括非对称接触条件。基于表面的接触能力不能用于建模此类问题；必须使用接触单元（ISL或IRS）。

必须为CAXA或SAXA单元中的每个周向平面创建独立的二维接触单元集。您必须指定每个接触单元集关联的周向平面的角度![](../graphics/usb_eqn00123.gif)，以及与底层CAXA或SAXA单元一起使用的傅里叶模式数*n*。

| **输入文件用法：** | 同时使用以下两个选项： |
| --- | --- |
| | ``` [*INTERFACE*](../key/key-link.md#usb-kws-minterface), ELSET=*element_set_name* [*ASYMMETRIC-AXISYMMETRIC*](../key/key-link.md#usb-kws-masymmaxisymm), MODE=*n*, ANGLE=![](../graphics/usb_eqn00123.gif) ``` 其中ELSET参数引用一组ISL或IRS型接触单元。 |

#### 不对称轴对称问题中接触的限制

如果不对称轴对称问题中的周向平面旋转超过几度，Abaqus/Standard仅能在![](../graphics/usb_eqn00123.gif)=0和180的周向平面上正确建模接触条件。不对称轴对称单元具有周向平面旋转和面外运动的内部自由度，但这些自由度在接触单元中未被考虑。忽略这些自由度意味着Abaqus/Standard将接触方向固定在初始周向平面中，并将节点位置投影回这些初始平面进行接触计算。如果节点相对于这些初始平面的旋转和运动很小，则此方法引起的误差很小。如果很大，误差将变得非常大，使结果不现实。
