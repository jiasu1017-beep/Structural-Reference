# 82.7.1 简介





当您执行自适应分析时，Abaqus/CAE 会提交一系列 Abaqus/Standard 作业，因为它试图减少模型中的网格离散化误差。该误差由 Abaqus/Standard 计算的误差指示器度量表征。Abaqus/CAE 计算这些指示器的全局范数，您可以使用这些范数来评估自适应重网格过程的性能和有效性。有关更多信息，请参阅 ["选择影响自适应重网格的误差指示器，" Abaqus Analysis User's Guide 第12.3.2节](../usb/usb-link.md#usb-anl-aadperrorindicators)。

自适应绘图器插件使您能够绘制这些误差指示器范数，并查看自适应重网格过程有效性的图形表示。您可以从任何模块中的 **Plug-ins** 菜单访问该插件。插件读取与自适应重网格作业关联的模型数据库（`.cae`）文件；因此，该文件必须可用。
