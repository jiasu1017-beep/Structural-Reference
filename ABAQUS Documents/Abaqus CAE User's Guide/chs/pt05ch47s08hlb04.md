# 47.9.4 自定义图例的位置







您可以通过在视口中拖动来移动*X–Y*曲线图图例，或者使用**Chart Legend Options**对话框中的**Position**选项指定视口位置。只有当Abaqus/CAE没有活动过程时，您才能拖动图例；如果您无法拖动图例或从视口中选择任何其他*X–Y*曲线图组件，请取消当前过程。

**要使用Chart Legend Options对话框自定义X–Y曲线图图例的位置：**

1. 找到**Chart Legend Options**。选择****Options**![](../graphics/images/arrow.gif)**XY Options**![](../graphics/images/arrow.gif)**Chart Legend****；或点击![](../graphics/ico_xyChartLegendOptions.png)，该图标位于可视化模块工具箱的*X--Y*绘图工具中。**提示：**当显示*X--Y*曲线图时，您也可以通过在视口中双击图例直接打开此对话框。当活动过程生效时，您不能直接从视口中选择*X--Y*曲线图中的组件；在从*X--Y*曲线图中选择项目之前，请取消当前过程。
2. 从**Charts**字段中，选择要移动其图例的一个或多个*X--Y*曲线图。
3. 点击**Area**选项卡（如果尚未选择）。
4. 切换**Inset**以在*X--Y*曲线图网格内显示*X--Y*曲线图图例。如果**Inset**关闭，Abaqus/CAE会将图例和*X--Y*曲线图保持在视口的独立区域中。
5. 选择以下定位选项之一：**Auto-align**此选项使您可以将图例对齐到视口中的九个位置之一：中心、四个角落中的任何一个，或四条边中任意一条的中心。展开**Alignment**列表，然后选择列表中以图形方式显示的选项之一。**Manual**此选项使您可以将图例定位在视口中的特定*X--Y*位置。拖动**X**和**Y**滑块可以在视口中水平或垂直移动图例。当这些滑块的值设置为`0`时，图例位于视口的左下角。
6. 点击**Dismiss**关闭**Chart Legend Options**对话框。
