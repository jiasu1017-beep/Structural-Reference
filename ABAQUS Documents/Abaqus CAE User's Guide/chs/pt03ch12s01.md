# 12.1 进入和退出Property模块





您可以在Abaqus/CAE会话的任何时候通过点击上下文栏中**Module**列表里的**Property**来进入Property模块。当您进入Property模块时，**Material**、**Section**、**Profile**、**Assign**、**Special**、**Feature**和**Tools**菜单会出现在主菜单栏中。**Part**列表会出现在上下文栏中，允许您选择要分配属性的部件。

要退出Property模块，请从**Module**列表中选择另一个模块。在退出模块之前，您不必采取任何特定操作来保存您的材料、截面和其他定义；当您通过从主菜单栏选择****File**![](../graphics/images/arrow.gif)**Save****或****File**![](../graphics/images/arrow.gif)**Save As****来保存整个模型时，它们会自动保存。
