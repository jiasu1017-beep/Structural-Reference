# 41.3.6 单元





**作业诊断**对话框中的**单元**页面显示模型中单元和材料点计算问题的区域的信息，这些问题可能阻止Abaqus为Abaqus/Standard步骤找到收敛解。Abaqus/Explicit显示关键单元的信息——具有最低稳定时间限制的单元。

**注意：**只有Abaqus/Standard和Abaqus/Explicit写入消息和状态文件的单元和材料点诊断的一个子集可以通过**作业诊断**对话框访问。

选择**在视口中高亮显示选择**以在当前视口中定位单元。您在**单元诊断**和**详细信息**字段中的选择决定Abaqus/CAE将高亮显示的单元。
