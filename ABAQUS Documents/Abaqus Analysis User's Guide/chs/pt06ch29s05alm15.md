# 29.5.1 具有变形横截面的管道和弯头：弯头单元

**产品：** Abaqus/Standard

##### **参考资料**

- ["弯头单元库，" 第29.5.2节](pt06ch29s05ael16.md)
- [*BEAM SECTION](../key/key-link.md#usb-kws-mbeamsection)

### 概述

弯头单元：
- 旨在为初始圆形管道和弯头的非线性响应提供准确建模，当横截面的椭圆化和翘曲变形主导行为时；
- 外观像梁，但实际上是壳，允许相当复杂的变形模式；
- 使用平面应力理论模拟管道壁的变形；
- 无法提供应力、应变和其他本构结果的节点值。

### 典型应用

在线性分析的常用方法中，响应预测基于半分析结果，用作"柔性因子"来校正简单梁理论获得的结果。这种因子不适用于非线性情况，必须将管道建模为壳才能准确预测响应（例如，参见["线性弹性管道在面内弯曲下的参数化研究，" Abaqus例题指南第1.1.3节](../exa/exa-link.md#exa-sta-elbowtest)）。虽然弯头单元看起来像梁，但它们实际上是壳，允许相当复杂的变形模式。在薄壁弯头中，弯头与相邻直段之间的相互作用是弯头建模的一个重要方面，以及横截面变形中容易发生的大旋转，即使管道轴线本身的相对旋转很小。所有这些效应（包括内压的强化效应）都可以用这些单元进行建模。

弯头单元旨在为初始圆形管道和弯头的非线性响应提供准确建模，当横截面的椭圆化和翘曲变形主导行为时。这种行为出现在两种情况下：在弯头中，管道的初始曲率加上管道壁的薄度导致椭圆化主导响应；在直管段中，过度弯曲可能导致薄壁圆形截面的屈曲坍塌（"Brazier屈曲"）。

由于弯头单元在圆周方向使用完整的壳公式，每个单元的 degrees of freedom 数量很高。使用所有傅里叶模式（见下文）对椭圆化和截面翘曲进行建模的弯头单元的计算成本比梁单元高得多，但它们的成本与可用于对截面进行建模的粗壳模型相当。

如果分析需要将管道单元连接到弯头，将弯头单元连接到管道单元比将壳单元连接到管道单元更容易。

### 选择适当的单元

弯头单元沿其长度使用多项式插值（根据单元类型为线性或二次），以及沿管道周围的傅里叶插值来模拟截面的椭圆化和翘曲。然后使用壳理论来模拟行为。

提供了两种类型的弯头单元。

#### ELBOW31 和 ELBOW32

单元类型ELBOW31和ELBOW32是最完整的弯头单元。在这些单元中，管道壁的椭圆化从一个单元到下一个单元是连续的，从而模拟诸如弯头与管道相邻直段之间的相互作用等效应。

ELBOW31和ELBOW32不应用于分析未连接的直管，除非在管道的某个点限制了翘曲和椭圆化。

#### ELBOW31B 和 ELBOW31C

单元类型ELBOW31B和ELBOW31C使用公式的简化版本，其中仅考虑椭圆化（无翘曲），忽略椭圆化的轴向梯度。这些近似通常是令人满意的，实际上它们构成了线性分析中使用的标准柔性因子方法的基础。它们提供了相当低成本的能力。ELBOW31C包括更进一步的近似，即忽略了管道周围傅里叶插值中的奇数项（第一项除外）。对于管道半径相对于管道轴线曲率半径较小的情况，此公式提供了稍微便宜一点的模型。

### 定义单元的截面特性

您可以使用在分析过程中积分的梁截面定义来定义弯头单元的截面特性。给出管道外半径*r*；管道壁厚*t*；以及弯头圆环半径，测量到管道轴线*R*。对于直管道，将*R*设为零。

您必须将这些特性与一组弯头单元相关联。

| **输入文件用法：** | ``` [*BEAM SECTION](../key/key-link.md#usb-kws-mbeamsection), SECTION=ELBOW, ELSET=*name* *r*, *t*, *R* ``` |
| --- | --- |

#### 定义截面方向

对于所有弯头单元，必须通过指定一个点来在空间中定位截面，该点与单元的节点一起定义![](../graphics/usb_eqn04672.gif)轴平面（见图29.5.1-1）。对于弯管，此点应位于弯头外侧（管道的远离弯曲外侧的一面称为*外侧*）。对于小于180度的弯管，可以将此点设置为相邻直管段的切线交点。如果弯管张角大于或等于180度，应将弯头分成小于180度的段，并为每个段定义单独的梁截面，以便用于定义![](../graphics/usb_eqn04672.gif)轴平面的点可以位于外侧的外部。当使用单元对直管道进行建模时，该点可以是管道轴线外的任意点。

#### 定义积分点和傅里叶模式的数量

您可以为弯头截面指定积分点和傅里叶模式的数量。经验表明，对于相对厚壁的情况，使用4个傅里叶模式和沿管道周围的12个积分点就足够了。对于薄壁弯头，需要6个傅里叶模式和沿管道周围的18个积分点。作为一般规则，沿管道周围的积分点数量不应小于所使用的傅里叶模式数量的三倍；否则，刚度矩阵可能出现奇异。当与零傅里叶模式一起使用时，单元变为简单的管道单元，包括环向应变和应力：当泊松比设为零时，它们表现出与Abaqus中PIPE单元相似的行为（见["选择梁单元，" 第29.3.3节](pt06ch29s03alm08.md)）。

| **输入文件用法：** | ``` [*BEAM SECTION](../key/key-link.md#usb-kws-mbeamsection), SECTION=ELBOW *first data line* *second data line* *number of int. pts. through thickness*, *number of int. pts. around pipe*, *number of Fourier modes* ``` |

### 指定温度和场变量

温度和场变量可以通过定义截面各点的值来指定，也可以通过定义管道壁中部的值并指定沿管道壁厚度的梯度来指定。

#### 通过定义截面各点的值来指定

您可以通过给出下面显示的三个点处的值来定义温度和场变量。

![](../graphics/eelbow-tfv-pts-nls.png)

无论截面厚度方向有多少个截面点，只指定这三个点的值。这三个值应用于圆周方向的所有积分点，因此唯一允许的变化在径向方向。

| **输入文件用法：** | ``` [*BEAM SECTION](../key/key-link.md#usb-kws-mbeamsection), SECTION=ELBOW, TEMPERATURE=VALUES ``` |
| --- | --- |

#### 通过定义管道壁中部的值和沿厚度的梯度来指定

或者，您可以通过给出管道壁中面处的值以及沿管道壁厚度位置相对于温度的梯度来定义温度和场变量，当外表面比内表面更热时为正。

| **输入文件用法：** | ``` [*BEAM SECTION](../key/key-link.md#usb-kws-mbeamsection), SECTION=ELBOW, TEMPERATURE=GRADIENTS ``` |

### 在大位移分析中使用弯头单元

当弯头单元在大位移分析（["一般和线性扰动过程，" 第6.1.3节](pt03ch06s01aus44.md)）中承受管道压力载荷（载荷类型PI、PE、HPI或HPE）时，会考虑对载荷刚度最主要的贡献。

### 在弯头单元上定义运动边界条件

弯头单元节点处标准自由度（即自由度1-6）上的运动边界条件应按常规方式处理。

此外，这些单元具有内部存储的椭圆化和翘曲项。对于ELBOW31B和ELBOW31C单元，这不需要额外的考虑。对于ELBOW31或ELBOW32单元，您通常需要为这些额外的自由度提供运动边界条件。例如，通常用弯头和相邻直管道段中的椭圆化和翘曲来对管道线进行建模，但在长直管道段的中段没有椭圆化（见图29.5.1-3）。

**图29.5.1-3** 管道线示意图。

![](../graphics/eelbow-pipeline-nls.png)

（后者通常通过使用零模式的ELBOW31元素或PIPE31元素来实现，这样通常的弯曲项和与管道中压力相关的均匀径向膨胀项被包括在内；如果内压不重要，可以使用简单的梁单元B31代替。）在具有椭圆化和翘旋的段结束处，必须限制翘曲；如果在该点存在刚性法兰或容器，也应限制椭圆化。为此，请为该节点指定NOWARP和/或NOOVAL或NODEFORM边界条件（["Abaqus/Standard和Abaqus/Explicit中的边界条件，" 第34.3.1节](pt07ch34s03aus118.md)）。

NOWARP表示该节点不允许翘曲，但允许椭圆化和均匀径向膨胀；NOOVAL表示该节点不能有椭圆化，但允许翘曲和均匀径向膨胀；NODEFORM表示根本不能有横截面变形——无翘曲、椭圆化或均匀径向膨胀。

通常，NOWARP将在用ELBOW31建模的弯头段端部指定，而NOWARP和NOOVAL将在刚性法兰或容器连接点指定。NODEFORM限制所有横截面变形，包括均匀径向膨胀项：如果发生热膨胀，这将导致大应力。NODEFORM应在固定端等处使用。

### 可视化横截面变形

当前版本的Abaqus/Standard没有提供直接可视化横截面椭圆化的方法。但是，实用程序例程`felbow.f`（["创建便于后处理弯头单元结果的数据文件：FELBOW，" Abaqus例题指南第15.1.6节](../exa/exa-link.md#exa-pst-felbow)）创建一个数据文件，可在Abaqus/CAE中使用该文件绘制感兴趣弯头截面周围积分点的当前坐标。该例程使用输出变量COORD（["Abaqus/Standard输出变量标识符，" 第4.2.1节](pt02ch04s02abv01.md)）获取积分点的当前坐标。只有在步骤中考虑了几何非线性时，这些值才可用。您必须确保变量COORD被写入结果（`.fil`）文件以用于此目的。

该例程适用于任意方向放置在空间中的弯头单元：弯头截面的积分点被适当地投影到适合绘制横截面的坐标系中。绘图输入数据被写入一个文件，可以读入Abaqus/CAE。可以使用可视化模块中的**XY Data Manager**显示弯头单元变形横截面的X-Y图。

除了便于可视化横截面椭圆化之外，该程序还允许您创建数据文件，以沿一排弯头单元和一个给定弯头单元周围绘制变量的变化。

提供了类似的C++和Python实用程序例程，`felbow.C`（["FELBOW的C++版本，" Abaqus脚本用户指南第10.15.6节](../cmd/cmd-link.md#cmd-odb-intro-felbow-cpp)）和`felbow.py`（["FELBOW的Abaqus脚本接口版本，" Abaqus脚本用户指南第9.10.12节](../cmd/cmd-link.md#cmd-odb-intro-felbow-pyc)），用于处理写入输出数据库（`.odb`）文件的相应弯头单元结果输出。执行这些程序时，它们会将数据写入ASCII格式文件和/或输出数据库文件，该文件可在Abaqus/CAE中用于绘制感兴趣弯头截面周围积分点的当前坐标。这两个例程还可用于可视化弯头截面周围变量输出的变化。
