# 11.16.4 在零件模块中使用查询工具集





从主菜单栏选择**工具**![](../graphics/images/arrow.gif)**查询**，或单击**查询**工具栏中的工具以启动查询工具集。

您可以使用查询工具集请求常规信息或特定于模块的信息。关于常规查询显示的信息的讨论，请参见"获取关于模型的一般信息"，第71.2.2节。

以下查询特定于零件模块：

**零件属性**

Abaqus/CAE在消息区域显示零件名称、建模空间和类型，以及形状（实体、壳、线框或点）和实体数量（单元、面、边和顶点）。

**重新生成警告**

如果所选零件中的某些集合或曲面由于其底层几何被修改或删除而无法重新生成，Abaqus/CAE会显示集合或曲面名称、原始面数以及查询期间找到的面数。

**子结构统计**

Abaqus/CAE显示有关所选子结构零件的以下信息：零件中保留的节点、特征模态和子结构载荷的数量；子结构中恢复矩阵、重力载荷向量、缩减质量矩阵、缩减结构阻尼矩阵和缩减粘性阻尼矩阵的可用性；以及子结构的质量特性。
