### 概述

离散粒子单元：
- 用刚性的球形表示单个粒子；
- 通常用于大量离散粒子单元相互作用的分析以及与其他物体的相互作用；
- 仅能用于显式动态分析；并且
- 必须只有一个节点。

### 典型应用

离散粒子单元（PD3D）适用于涉及砾石等不连续介质的模拟。关于离散元素方法（DEM）及其典型应用的描述，请参阅["离散元素方法，" 第15.1.1节](pt04ch15s01aus94.md)。

### 定义单元的截面属性

您必须将离散截面定义与一组离散粒子单元相关联。截面定义提供了与 PD3D 单元关联的密度和粒子半径。

| **输入文件用法：** | ``` [*DISCRETE SECTION](../key/key-link.md#usb-kws-mdiscretesection), ELSET=*element_set_name* ``` |
| --- | --- |

### 粒子几何形状、质量和转动惯量

PD3D 单元呈球形，每个粒子具有均匀的密度。粒子半径和密度在离散截面定义中指定。Abaqus 使用这些量来计算 PD3D 单元的质量和转动惯量。每个离散单元都很简单，但大量此类粒子和有限单元的相互作用能够模拟复杂现象，如["离散元素方法，" 第15.1.1节](pt04ch15s01aus94.md)中所讨论的。

分配给单个离散截面定义的所有粒子的粒度尺寸和密度通常相同，但可以通过为半径和/或密度分配分布名称而不是标量值来指定与单个离散截面定义关联的粒子之间这些量的变化。请参阅["分布定义，" 第2.8.1节](pt01ch02s08aus26.md)，了解分布的讨论。例如，如果要考虑两种粒度尺寸，使用两个各包含单一粒子尺寸的离散截面定义可能最方便。

| **输入文件用法：** | 使用以下选项定义粒子形状、密度和半径： |
| --- | --- |
|  | ``` [*DISCRETE SECTION](../key/key-link.md#usb-kws-mdiscretesection), SHAPE=SPHERE, DENSITY=*particle density * *or a particle densities distribution table name* *particle radius or particle radii distribution table name* ``` |

### Alpha 阻尼

您可以为 PD3D 单元定义质量和转动惯量比例阻尼，类似于点质量和转动惯量单元（请参阅["点质量，" 第30.1.1节](pt06ch30s01alm21.md)和["转动惯量，" 第30.2.1节](pt06ch30s02alm22.md)）。此阻尼作用于单个粒子的平移和旋转速度（相对于"地面"），独立于接触阻尼（作用于附近粒子对的相对速度）。与 Abaqus/Explicit 中的其他类型的可变形有限单元不同，没有作用于 PD3D 单元的全局粘性阻尼。少量的质量比例阻尼有助于减少由大量打开和关闭接触条件产生的解决方案中的噪声。

| **输入文件用法：** | 使用以下选项指定质量和转动惯量比例阻尼： |
| --- | --- |
|  | ``` [*DISCRETE SECTION](../key/key-link.md#usb-kws-mdiscretesection), ALPHA=*damping factor, α* ``` |
