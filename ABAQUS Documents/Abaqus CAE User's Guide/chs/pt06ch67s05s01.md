# 67.5.1 自定义自由体切割的常规显示选项







**Free Body Plot Options**（自由体绘图选项）对话框的 **General**（常规）选项卡页面上的选项使您能够切换自由体切割的力向量和力矩向量的显示。您还可以通过仅显示合力向量或在 x-、y- 或 z- 方向上显示分量向量来自定义向量显示，以显示力和力矩值。

**自定义当前视口中所有活动自由体切割的常规显示选项：**

1. 找到 **General**（常规）选项。在可视化模块中从主菜单栏中选择 ****Options**![](../graphics/images/arrow.gif)**Free Body****（选项→自由体）。
2. 切换开启 **Show forces**（显示力）以在当前视口中显示所有自由体切割的力向量。
3. 切换开启 **Show moments**（显示力矩）以在当前视口中显示所有自由体切割的力矩向量。
4. 切换开启 **Use constant-length arrows**（使用固定长度箭头）以相同长度显示所有力和力矩值的箭头。未选择此选项时，力和力矩向量根据其大小显示长度。
5. 从 **Vector display**（向量显示）选项中，选择以下之一： - 选择 **Resultant**（合向量）仅显示合力和合力矩向量。 - 选择 **Component**（分量）以在 x-、y- 或 z- 方向上显示分力向量和力矩向量。
6. 点击 **OK**（确定）以在当前视口中实施您的更改并关闭 **Free Body Plot Options**（自由体绘图选项）对话框。默认情况下，您的更改会在会话期间保存，并将影响该视口中所有后续的自由体切割图。如果您想在后续会话中保留您的更改，请将它们保存到文件。有关更多信息，请参阅["保存自定义设置以供后续会话使用，" 第 55.1.1 节"](pt05ch55s01s01.md)。
