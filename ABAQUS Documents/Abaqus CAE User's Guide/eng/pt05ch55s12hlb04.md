# 55.12.4 Coloring elements with no results







To produce a contour plot, Abaqus/CAE reads results for the variable and frame you specify from the output database. Results for a particular variable and frame may not be available or may not be applicable for one or more elements included in the plot. You can specify the display color for elements that have no results. The default color is white.

For example, the contour plot in [Figure 55--18](pt05ch55s12hlb04.md#viw-noresults) shows the von Mises stress generated when a blank sheet is deformed using a die and punch (the punch is not shown in the figure). In this case the color gray was selected for elements with no results. The die elements appear gray because the die is a rigid surface for which no stress results exist.

**Figure 55–18** Contour plot of blank and die.

![](../graphics/viw-noresults.gif)

**To color elements with no results:**

1. Locate the **Elements with No Results** options. From the main menu bar, select ****View**![](../graphics/images/arrow.gif)**ODB Display Options****. Click the **General** tab in the dialog box that appears. The **Elements with No Results** options are in the center of the **General** page.
2. Click the color sample ![](../graphics/usi-colorsample.png). Abaqus/CAE displays the **Select Color** dialog box.
3. Use one of the methods in the **Select Color** dialog box to select a new color. For more information, see ["Customizing colors," Section 3.2.9](pt01ch03s02s09.md).
4. Click **OK** to close the **Select Color** dialog box. The color sample changes to the selected color.
5. Click **Apply** to implement your changes. Elements with no results in contour plots (including display bodies) change to reflect your coloring specification, which is saved for the duration of the session. If you want to retain the changes you applied for subsequent sessions, save them to a file. For more information, see ["Saving customizations for use in subsequent sessions," Section 55.1.1](pt05ch55s01s01.md).
