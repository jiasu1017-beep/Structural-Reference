# 58.7.1 从点云数据创建映射场





您可以通过读取由第三方 CAE 应用程序生成的点云数据文件中的参数值来创建映射场。在使用以下过程创建映射场之前，您必须准备好点云数据文件并可供导入。

**从点云数据创建映射场的步骤：**

1. 使用以下方法之一打开**创建映射场**对话框：
   - 从主菜单栏，选择 ****Tools**![](../graphics/images/arrow.gif)**Analytical Field**![](../graphics/images/arrow.gif)**Create****。
   **提示：**您也可以单击**分析场管理器**中的**创建**。
   - 在 Interaction 模块的相互作用编辑器中，单击 **Definition** 或 **Emissivity distribution** 字段旁边的 ![](../graphics/ico_analyticalField.png)。
   - 在 Load 模块的载荷或预定义场编辑器中，单击 **Distribution** 字段旁边的 ![](../graphics/ico_analyticalField.png)。

2. 选择 **Mapped field** 作为**类型**，然后单击 **Continue**。

3. 在**名称**文本字段中，输入映射场的名称。有关命名对象的信息，请参见["使用基本对话框组件，" 第3.2.1节](pt01ch03s02s01.md)。在**描述**文本字段中，输入映射场的描述。

4. 选择 **Point cloud** 作为**数据源**，以导入您从另一个 CAE 应用程序生成的点云数据文件。这些文件必须包含 **XYZ** 格式或 **Grid** 格式的场值和坐标，如["用于映射的点云数据文件格式，" 第58.3.1节](pt06ch58s03s01.md) 中所述。

5. 对于点云数据源，您可以选择为源数据区域分配局部坐标系，以简化空间中点的三维定义。单击 ![](../graphics/ico_selectBlue.png)，然后使用以下方法之一：
   - 在视口中选择现有的基准坐标系。
   - 通过名称选择现有的基准坐标系。
     1. 从提示区域，单击 **Datum CSYS List** 以显示基准坐标系列表。
     2. 从列表中选择一个名称，然后单击 **OK**。
   - 从提示区域单击 **Use Global CSYS** 以恢复到全局坐标系。
   单击 ![](../graphics/ico_dtmCsys.png) 创建新的基准坐标系。当您分配局部坐标系时，源数据坐标将在该坐标系中定义，该坐标系是装配级（不是零件级）坐标系。有关更多信息，请参见["点云映射场的坐标系，" 第58.3.2节](pt06ch58s03s02.md)。

6. 如果需要，切换打开 **Supplied data are defined in part space** 以指示源数据及其局部坐标系都在目标模型区域零件级的坐标系中定义。

7. 如果需要，选择以下选项之一输入**比例因子**并缩放源数据坐标；例如，为了补偿单位不匹配（即米到毫米）：
   - 选择 **Uniform**，然后输入单个比例因子。
   - 选择 **Nonuniform**，然后为 *X*、*Y* 和 *Z* 坐标输入比例因子。

8. 输入**默认未映射场值**的值，或接受默认值零。Abaqus/CAE 为任何无法找到源点进行映射的目标点分配此值。请参见["映射场的搜索控制，" 第58.7.3节](pt06ch58s06s01.md)。

9. 在 **Point Data** 选项卡页面上，选择以下之一作为数据格式：
   - 如果您的数据文件包含直角 *X*、*Y* 和 *Z* 坐标以及相应的场值，请选择 **XYZ**。有关此格式的详细信息，请参见["**XYZ** 格式" 在"用于映射的点云数据文件格式，" 第58.3.1节](pt06ch58s03s01.md#usi-fld-usingmap-point-xyz)。单击 ![](../graphics/icoR_tableImport.png) 浏览并选择要导入的数据文件。有关从文件读取数据的更多信息，请参见["输入表格数据，" 第3.2.7节](pt01ch03s02s07.md)。如果您的局部坐标系是圆柱坐标系或球坐标系而不是直角坐标系，Abaqus/CAE 期望您的数据文件包含相应的坐标类型。请参见["点云映射场的坐标系，" 第58.3.2节](pt06ch58s03s02.md)。
   - 如果您的数据文件包含在三维点网格中定义的场值，请选择 **Grid**。有关此格式的详细信息，请参见["**Grid** 格式" 在"用于映射的点云数据文件格式，" 第58.3.1节](pt06ch58s03s01.md#usi-fld-usingmap-point-grid)。在 **Field Data** 表中，指定平面，然后按如下方式导入场值：
     1. 选择数据所在的网格平面，例如对于直角坐标系为 **XY**、**YZ**、**XZ**、**YX**、**ZY** 或 **ZX**。
     2. 单击 ![](../graphics/ico_add.png) 在左侧框架中添加一个平面。输入平面高度，单击 **OK**，然后在表中输入场值或单击 ![](../graphics/icoR_tableImport.png) 从文件导入。例如，在 *Z*=2 处添加一个平面，然后导入该平面内各种 *X* 和 *Y* 坐标处的场值。
     3. 对于同一网格平面内的其他平面高度重复上述操作；例如，在 *Z*=3、*Z*=4 等处。您可以在 **Data Values** 表中使用以下编辑工具：
        | ![](../graphics/ico_add.png) | **Add Plane** |
        | --- | --- |
        | ![](../graphics/ico_edit.png) | **Edit Plane** |
        | ![](../graphics/ico_tableCopyRowAfter.png) | **Copy Plane** |
        | ![](../graphics/ico_delete.png) | **Delete Plane** |
     有关从文件读取数据的更多信息，请参见["输入表格数据，" 第3.2.7节](pt01ch03s02s07.md)。

10. 在 **Mapper Controls** 选项卡页面上，您可以调整搜索容差参数。请参见["映射场的搜索控制，" 第58.7.3节](pt06ch58s06s01.md)。

11. 单击 **OK** 创建映射场并关闭对话框。

当您稍后使用映射场应用载荷、相互作用或预定义场时，您可以在视口中显示符号以可视化场值的位置和幅值。但是，您必须先对模型进行网格划分才能看到这些符号。
