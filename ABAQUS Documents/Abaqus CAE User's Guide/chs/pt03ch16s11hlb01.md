# 16.11.1 定义初始速度场





您可以创建此类预定义场来定义所选区域的初始速度。有关更多信息，请参见["Abaqus/Standard和Abaqus/Explicit中的初始条件"章节，《Abaqus Analysis User's Guide》第34.2.1节](../usb/usb-link.md#usb-prc-pinitialcond)。

**创建或编辑初始速度场的步骤：**

1. 使用以下方法之一显示速度场编辑器： - 要创建新的速度场，请按照["创建预定义场"章节第16.8.3节](pt03ch16s08hlb03.md)中概述的程序进行操作（**类别**：`机械`；**所选步骤的类型**：`速度`）。 - 要使用菜单或管理器编辑现有的速度场，请参见["编辑依赖于步骤的对象"章节第3.4.13节](pt01ch03s04hlb05.md)。**注意：**您只能在初始步骤中创建或编辑初始速度。
2. 单击**分布**字段右侧的箭头，然后从出现的列表中选择所需的选项： - 选择**均匀**来定义在该区域上均匀的初始速度。 - 选择解析场来定义空间变化的初始速度。或者，您可以单击![](../graphics/ico_analyticalField.png)来创建新的解析场。（有关更多信息，请参见[第58章，"解析场工具集"](pt06ch58.md)。）
3. 单击**定义**字段右侧的箭头，然后从出现的列表中选择所需的选项： - 选择**仅平移**仅定义初始平移速度。 - 选择**仅旋转**仅定义初始旋转速度。 - 选择**平移和旋转**来同时定义初始平移速度和初始旋转速度。
4. 如果您选择了**仅平移**定义选项，请在**V1**、**V2**和（如果适用）**V3**文本字段中输入初始平移速度的大小，分别对应1、2和（如果使用三维空间）3方向。（如果将文本字段留空，则默认情况下该自由度被分配为零初始平移速度；必须至少在一个方向上定义速度。）
5. 如果您选择了**仅旋转**定义选项： 1. 在**角速度**文本字段中输入角速度大小（以弧度/时间为单位）。 2. 定义旋转轴。 - 对于三维模型，请在**轴点1**和**轴点2**文本字段中输入点的*x*-、*y*-和*z*-坐标，用逗号分隔。 - 对于二维模型，请在**轴点**文本字段中输入点的*x*-和*y*-坐标，用逗号分隔。假设旋转围绕指定点处的正*z*轴进行。 - 对于轴对称模型，请在**轴半径**文本字段中输入半径。假设旋转围绕指定半径处的正*z*轴进行。
6. 如果您选择了**平移和旋转**定义选项，请同时遵循第3步和第4步中的说明。
7. 单击**确定**保存数据并退出编辑器。
