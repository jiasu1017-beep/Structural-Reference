# 10.7.5 从CATIA V4或V5格式文件导入零件





从主菜单栏选择****文件（File）**![](../graphics/images/arrow.gif)**导入（Import）**![](../graphics/images/arrow.gif)**零件（Part）****直接从CATIA V4或V5格式文件导入零件到Abaqus/CAE。您可以导入CATIA格式的零件，但无法导出CATIA格式的零件。有关Abaqus/CAE支持的CATIA V4和CATIA V5版本的信息，请参阅Dassault Systmes知识库，网址为[www.3ds.com/support/knowledge-base](http://www.3ds.com/support/knowledge-base)。

**注意：**仅当您拥有可选附加组件CATIA V5关联接口的许可证时，才能直接导入CATIA V5格式文件。更多信息，请参阅["关联接口可以做什么？，"第10.1.2节"](pt02ch10s01s02.md)。

导入的CATIA零件在Abaqus/CAE中形成新零件的基础特征；您无法直接修改此基础特征，但可以向其添加其他特征，如实体拉伸或盲孔切割。

**要从CATIA格式文件导入零件：**

1. 从主菜单栏，选择****文件（File）**![](../graphics/images/arrow.gif)**导入（Import）**![](../graphics/images/arrow.gif)**零件（Part）****。出现**导入零件（Import Part）**对话框。
2. 从**导入零件**对话框底部的**文件过滤器（File Filter）**菜单中，选择**CATIA V4 (*.model, *.catdata, *.exp)**或**CATIA V5 (*.CATPart, *.CATProduct)**。Abaqus/CAE会列出所选目录中具有可接受文件扩展名的所有文件。
3. 选择包含要导入零件的CATIA文件，然后单击**确定（OK）**。Abaqus/CAE会显示**从CATIA文件创建零件（Create Part from CATIA File）**对话框。
4. 单击**名称-修复（Name-Repair）**选项卡设置以下内容：**名称（Name）**新零件的名称。更多信息，请参阅["使用基本对话框组件，"第3.2.1节"](pt01ch03s02s01.md)。**修复选项（Repair Options）**Abaqus/CAE在导入过程中自动修复CATIA零件。**零件过滤器（Part Filter）**CATIA文件可以包含多个零件，默认情况下Abaqus/CAE会导入文件中的所有零件。如果您导入CATIA文件中的所有零件，可以为每个零件创建单独的Abaqus零件，也可以将它们组合成单个Abaqus零件；此外，如果您组合这些零件，Abaqus/CAE使您能够使用用户指定的缝合容差值将这些导入的零件缝合在一起。如果要从CATIA文件导入单个零件，可以切换**导入零件号（Import part number）**并输入要从文件导入的单个零件的编号。
5. 单击**零件属性（Part Attributes）**选项卡设置以下内容：- 所导入零件的建模空间。导入零件时，Abaqus/CAE会扫描文件并尝试确定建模空间。- 所导入零件的类型。Abaqus/CAE始终假定零件类型是可变形的。或者，您可以选择**离散刚体（Discrete rigid）**导入离散刚体零件，或选择**欧拉（Eulerian）**导入欧拉零件。更多信息，请参阅["零件类型，"第11.4.2节"](pt03ch11s04s02.md)。同样，在大多数情况下，您应该接受默认设置；更多信息，请参阅["零件属性是什么？，"第10.3.2节"](pt02ch10s03s02.md)。
6. 单击**比例（Scale）**选项卡在导入过程中缩放零件。- 选择**不缩放（No scale）**保持文件中存储的尺寸。此选项仅在导入CATIA V4格式零件时可用。- 选择**使用文件中的变换，包括比例（Use transform from file, including scale）**从CATIA文件读取比例因子、旋转矩阵和平移矩阵。此选项仅在导入CATIA V4格式零件时可用。- 选择**将所有长度乘以（Multiply all lengths by）**，然后输入比例因子。Abaqus/CAE会将比例因子应用于文件中的所有坐标。因此，距原点的任何偏移都将按比例缩放。
7. 单击**确定（OK）**导入零件并退出**从CATIA文件创建零件**对话框。Abaqus/CAE会扫描CATIA文件并根据所选选项启动修复过程。如果希望取消导入过程，请在提示区域单击**停止（Stop）**。
8. 导入单个零件时，Abaqus/CAE会显示消息，指示零件是否包含任何有效性或精度问题。如果您从单个CATIA文件导入多个零件，可以使用**零件管理器（Part Manager）**检查任何零件是否无效或不精确。如果需要，使用查询工具集定位无效和不精确几何的区域。更多信息，请参阅["在零件模块中使用查询工具集，"第11.16.4节"](pt03ch11s16s04.md)。
