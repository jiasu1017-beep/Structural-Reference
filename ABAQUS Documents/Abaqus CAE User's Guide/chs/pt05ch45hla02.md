# 45.3 生成结果符号图





符号图显示输出数据库在指定步骤和帧的向量或张量变量的大小和方向，或显示当前模型数据库中模型在指定步骤的属性的大小和方向。Abaqus/CAE将值表示为模型上特定位置的符号（例如箭头）。有关张量和场输出变量的更多信息，请参见["场输出变量选择概述，" 第42.5.1节](pt05ch42s03hlb01.md)。有关选择结果步骤的更多信息，请参见["选择特定结果步骤和帧，" 第42.3.1节](pt05ch42s01hlb01.md)。要了解如何显示与符号图关联的最小值和最大值，请参见["自定义图例，" 第56.1节](pt05ch56s01.md)。

**要生成符号图：**

1. [打开包含模型数据或分析结果的模型数据库或输出数据库](pt02ch09s06hlb02.md)。
2. 如果您正在从模型数据库绘制数据，请切换到可视化模块，展开Results Tree的**Model Database**容器，然后选择要使用的模型。
3. 选择要显示的结果[步骤和帧](pt05ch42s01hlb01.md)（如果选择了当前模型数据库，则只需步骤）。
4. 对于输出数据库，选择要显示的[符号场输出变量](pt05ch42s03hlb05.md)。对于模型数据库，选择要显示的[主场输出变量](pt05ch42s03hlb03.md)作为变量。符号图状态被激活。
5. 选择您想要的绘图状态独立和符号图[自定义](pt05ch40s03s03.md)选项。
6. 如果您选择的场输出包含复数，请在**Result Options**对话框中选择[**Complex Form**](pt05ch42s04hlb05.md)选项卡来控制要显示的数值形式。此选项仅适用于输出数据库的绘图。
7. 对于输出数据库数据的绘图，从主菜单栏中选择****Plot**![](../graphics/images/arrow.gif)**Symbols****来修改是在变形模型形状、未变形形状还是两者上绘制符号。**提示：**您还可以使用工具箱中的变形![](../graphics/ico_plotSymbolDeformed.png)、未变形![](../graphics/ico_plotSymbolUndeformed.png)或叠加![](../graphics/ico_plotSymbolDefUndef.png)符号工具生成符号图。当前视口更改为显示当前输出数据库在指定步骤和帧的指定场输出变量的自定义符号图。每次您在步骤和帧选择器对话框、绘图状态独立选项、叠加绘图选项（如果适用）或符号图选项对话框中单击**Apply**时，Abaqus会自动刷新您的符号图。
