# 49.2.5 自定义动画的基础图





时间历史、比例因子和谐波动画都以快速连续方式显示一系列变形形状、等值线、符号或（仅适用于时间历史动画）材料方向图。在动画播放之前或期间的任何时候，您可以自定义支撑动画的变形形状、等值线、符号或材料方向图。例如，您可以切换图例、调整变形比例因子或选择不同的场输出变量。使用"视图"、"结果"和"选项"菜单来自定义动画的基础图。

**自定义基础图：**

1. 找到适用于当前绘图状态的自定义选项。- 对于等值线图动画，从主菜单栏中选择****选项**![](../graphics/images/arrow.gif)**等值线****。- 对于符号图动画，从主菜单栏中选择****选项**![](../graphics/images/arrow.gif)**符号****。- 对于材料方向图动画，从主菜单栏中选择****选项**![](../graphics/images/arrow.gif)**材料方向****。
2. 从出现的对话框中，选择您想要的绘图状态相关自定义选项。有关详细信息，请参阅["自定义您的绘图，"第40.3.3节](pt05ch40s03s03.md)。
3. 通过选择****选项**![](../graphics/images/arrow.gif)**常规****、****选项**![](../graphics/images/arrow.gif)**叠加****、****视口**![](../graphics/images/arrow.gif)**视口注释选项****、****视图**![](../graphics/images/arrow.gif)**图形选项****或****视图**![](../graphics/images/arrow.gif)**ODB显示选项****来设置您想要的绘图状态无关自定义选项。
4. 通过选择****结果**![](../graphics/images/arrow.gif)**场输出****来选择要动画化的[结果变量](pt05ch42s03hlb01.md)。
5. 选择要动画化的结果步骤和帧：- 对于比例因子或谐波动画，通过选择****结果**![](../graphics/images/arrow.gif)**步骤/帧****来选择要动画化的单个步骤和帧。- 对于时间历史动画，通过选择****结果**![](../graphics/images/arrow.gif)**活动步骤/帧****来选择要动画化的步骤和帧。
