# 5.2.2 子结构矩阵输出请求





**产品：**Abaqus/Standard   

### 测试单元

C3D8

### 测试功能

测试子结构矩阵和载荷工况向量的输出和输入。

### 问题描述

这些测试验证子结构矩阵输出过程写入的矩阵和载荷向量是有效的，并且可以输入到分析中再次使用。通过使用矩阵和载荷向量求解线性问题的分析来测试结果的有效性。

### 结果与讨论

**xsmon2so.inp, xsmon2su.inp**

最大位移2.0E4出现在此问题的节点3。两个运行具有相同的位移场。

**xsmop1so.inp, xsmop1su.inp**

最大位移2.0E4出现在此问题的节点3。两个运行具有相同的位移场。

### 输入文件

#### 单元矩阵输出：

[xsmon2so.inp](../eif/xsmon2so.inp)

一个由一个C3D8单元组成的子结构，在一个[*SUBSTRUCTURE LOAD CASE](../key/key-link.md#usb-kws-ssubloadcase)中有分布载荷，在另一个中有集中载荷。使用静态步骤测试矩阵和载荷向量。

[xsmop1so.inp](../eif/xsmop1so.inp)

一个由一个C3D8单元组成的子结构，在预加载步骤中施加集中载荷，在[*SUBSTRUCTURE LOAD CASE](../key/key-link.md#usb-kws-ssubloadcase)中施加分布载荷。使用静态步骤测试矩阵和载荷向量。

#### 单元矩阵输入：

[xsmon2su.inp](../eif/xsmon2su.inp)

一个由一个C3D8单元组成的子结构，在一个[*SUBSTRUCTURE LOAD CASE](../key/key-link.md#usb-kws-ssubloadcase)中有分布载荷，在另一个中有集中载荷。使用静态步骤测试矩阵和载荷向量。

[xsmop1su.inp](../eif/xsmop1su.inp)

一个由一个C3D8单元组成的子结构，在预加载步骤中施加集中载荷，在[*SUBSTRUCTURE LOAD CASE](../key/key-link.md#usb-kws-ssubloadcase)中施加分布载荷。使用静态步骤测试矩阵和载荷向量。
