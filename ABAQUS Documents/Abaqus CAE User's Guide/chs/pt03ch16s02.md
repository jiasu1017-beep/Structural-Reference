# 16.2 进入和退出载荷模块







您可以在Abaqus/CAE会话期间通过单击上下文栏中**模块**列表中的**载荷**来进入载荷模块。主菜单栏上会出现**载荷**、**边界条件**、**预定义场**、**载荷工况**、**特征**和**工具**菜单。上下文栏中会出现**步骤**列表。

要退出载荷模块，请在上下文栏的**模块**列表中指定另一个模块。在退出模块之前，您无需采取任何特定操作来保存您的规定条件；在通过从主菜单栏中选择****文件**![](../graphics/images/arrow.gif)**保存****或****文件**![](../graphics/images/arrow.gif)**另存为****保存整个模型时，它们会自动保存。
