# 3.2.31 将PAM-CRASH输入文件翻译为部分Abaqus输入文件





**产品：** Abaqus/Explicit  

##### **参考**

- ["Abaqus执行程序：概述，" 第3.1.1节](pt01ch03s01abo02.md)

### 概述

从PAM-CRASH到Abaqus的翻译器将PAM-CRASH输入文件中的某些关键字转换为其在Abaqus/Explicit中的对应物。

### 使用翻译器

翻译器需要由PAM-CRASH版本2002或更高版本创建的输入文件。输入文件可以有任意名称和可选扩展名。

翻译的PAM-CRASH数据条目列在下面的表格中。其他PAM-CRASH关键字和数据被跳过并记录在日志文件中。

翻译器创建一个仅包含模型数据的部分Abaqus输入文件。您必须提供历史数据（包括输出数据）来完成输入。

#### 元素编号和分组

所有元素必须具有唯一的元素编号。分配了相同PART标识号的元素被分组在一个元素集中。

除从SPRING和KJOIN翻译而来的连接器元素外，截面属性需要在PART部分而不是在元素部分中单独输入。具有不同材料或截面属性的元素应分配不同的PART标识号；即，相同的材料和截面属性必须适用于组合在同一元素集中的所有元素。

如果从SPRING和KJOIN翻译而来的元素具有不同的元素数据（例如，用于定义局部方向的框架编号），并且它们被分配了相同的PART标识号，翻译器会自动将它们分离到不同的元素集中。

#### 材料模型

翻译器仅支持[表3.2.31-3](pt01ch03s02abx31.md#table-usb-int-dpamabaproc-materialsectdata)中所示的材料模型。类型1到99之间的所有不支持的材料模型都被翻译为双线性弹塑性，所有其他材料类型在需要应力-应变律定义时被翻译为线性弹性。在这些情况下，翻译器会提供材料属性的标称值。

#### 历史部分数据

翻译器根据PAM-CRASH文件控制部分中的关键字（TITLE除外）部分创建历史部分，如[表3.2.31-1](pt01ch03s02abx31.md#table-usb-int-dpamabaproc-controlsectdata)所示。其他控制数据不支持。

### 命令行选项

**job**

此选项用于指定要由翻译器输出的Abaqus输入文件的名称。Abaqus输入文件的名称必须不带`.inp`扩展名。翻译器生成的诊断信息被写入名为`*job-name*_frompam.log`的文件。

**input**

此选项用于指定包含PAM-CRASH数据的文件的名称。必须带有文件扩展名给出文件的名称。

**pLinkConnectors**

此选项用于指定PLINK翻译中连接器元素的包含。默认值为`ON`。

**splitAirbagElements**

此选项用于指定将4节点安全气囊膜元素拆分为两个3节点安全气囊膜元素。默认值为`ON`。安全气囊膜元素由具有厚度方向一个积分点的SHELL或SH3N的翻译产生。此选项仅在PAM-CRASH输入文件中指定了关键字MONVOL/AIRBAG时有效。

**autoKJoinStops**

此选项用于向所有KJOIN连接器元素的行为添加连接器止挡。如果在端点处插值的刚度超过相邻点插值刚度的10倍，则在端点相邻点处定义连接器止挡。默认值为`OFF`。
