# 20.12.3 添加单个约束







约束是草图中实体之间的逻辑关系。约束通过从草图中移除自由度来控制 Sketcher 几何。各种约束类型在["使用约束控制草图几何，"第 20.7.1 节](pt03ch20s07s01.md)中描述。要更改受约束的几何，您必须要么在定义的约束范围内工作，要么修改约束以接受您的更改。您也可以使用**自动约束**工具自动约束草图。有关更多信息，请参见["自动约束草图，"第 20.12.1 节](pt03ch20s12hlb01.md)。

**添加约束：**

1. 从 Sketcher 工具箱中的工具中，选择约束工具![](../graphics/ico_sketchConstraint.png)。有关 Sketcher 工具箱中工具的图表，请参见["Sketcher 工具，"第 20.4.1 节](pt03ch20s04s01.md)。Abaqus/CAE 打开**添加约束**对话框。
2. 选择您要添加到草图的约束类型，然后点击**应用**。Abaqus/CAE 在提示区域显示提示以指导您完成该过程。
3. 选择要创建约束的实体或实体。Abaqus/CAE 创建约束并在草图上显示适当的约束符号。草图修改工具（如平移、旋转、缩放和镜像）可以覆盖约束。
4. 要创建其他约束，重复从第 2 步开始的上述步骤。
5. 完成创建约束后，点击**取消**关闭**添加约束**对话框。
