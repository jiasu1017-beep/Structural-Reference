# 48.2.4 创建或编辑圆形路径





路径是通过在模型中指定一系列点来定义的线。在圆形路径中，所有点都是位于圆周或圆半径上的模型坐标位置。有关路径规格的替代形式的信息，请参阅["创建穿过模型的路径，"第48.2节](pt05ch48s02.md)。

您可以通过选择节点或输入坐标来定义圆，从而创建圆形路径。定义圆后，使用圆的边缘或半径来定义曲线周向路径或直线径向路径。

**创建或编辑圆形路径：**

1. 找到路径创建和编辑选项。**要创建新路径：**1. 从主菜单栏中，选择****工具**![](../graphics/images/arrow.gif)**路径**![](../graphics/images/arrow.gif)**创建****。"创建路径"对话框出现。2. Abaqus在"名称"文本字段中显示路径的默认名称。要提供更有意义的名称，请将此默认值替换为您选择的名称（包括空格如果您希望的话）。3. 在"创建路径"对话框中，单击"圆形"；然后单击"继续"。**要编辑现有路径：**- 从主菜单栏中，选择****工具**![](../graphics/images/arrow.gif)**路径**![](../graphics/images/arrow.gif)**编辑****。从出现的菜单中，选择要编辑的路径。**提示：**您也可以使用"路径管理器"来编辑路径。从主菜单栏中，选择****工具**![](../graphics/images/arrow.gif)**路径**![](../graphics/images/arrow.gif)**管理器****以显示管理器。选择要编辑的路径，然后单击管理器右侧的"编辑"按钮。"编辑圆形路径"对话框出现。
2. 选择路径类型，**周向**或**径向**。Abaqus/CAE更新对话框以反映您正在创建的路径类型。周向路径沿着圆的边缘，需要半径、起始角度和结束角度来定义其位置和长度。径向路径沿着圆的半径，需要径向角度、起始值和结束值来定义其位置和长度。
3. 在对话框的"圆定义"部分，选择"原点和轴"或"弧上的3个点"作为定义圆的方法。原点和轴方法使用两个点定义旋转轴，使用圆平面上的一点定义默认半径长度和旋转起始点。弧上三点方法拟合三个边缘点成一个圆。
4. 单击"从视口选择"以选择节点或输入坐标来定义圆。Abaqus/CAE关闭"编辑圆形路径"对话框，并在提示区域中显示指导您完成该过程的提示。
5. 从视口中选择节点，或使用提示区域中的文本字段将每个点指定为X、Y、Z坐标，用空格或逗号分隔；例如，1.0, 3.4, 2.0。Abaqus/CAE打开"编辑圆形路径"对话框；您选择的坐标出现在圆定义表中。如果您选择了节点，使用的坐标对应于模型的当前变形状态。在当前视口中的模型图上，Abaqus高亮显示路径中包含的所有点，并显示路径连接。
6. 您可以按如下方式编辑"圆定义"中的单个点：1. 将光标放在点的坐标上。2. 单击鼠标按钮3，然后选择"编辑选择"。3. 在视口中选择一个节点，或在提示区域中编辑坐标。当您进行选择或输入新坐标时，Abaqus/CAE返回"编辑圆形路径"对话框，并在视口中显示重新定义的圆。
7. 选择要包含在路径中的段数。Abaqus/CAE修改视口中的路径以显示所选数量的等长段。
8. 按如下方式完成圆形路径的编辑或创建：**周向路径**选择**半径**，然后选择旋转的**起始角度**和**结束角度**。默认情况下，Abaqus使用您定义的圆的半径，并从您在圆上选择的第一个点开始旋转完整360度——如果您选择了原点和轴方法，这是圆上唯一的点。**径向路径**选择**径向角度**、**起始半径**和**结束半径**。默认径向角度（0）是从圆心到您在圆周上选择的第一个点的连线——如果您选择了原点和轴方法，这是圆上唯一的点。默认起始半径是`0.0`（圆心），默认结束半径等于圆的半径。
9. 完成后，单击"确定"。如果创建了新路径，Abaqus/CAE会将路径名称添加到"路径管理器"列表中。默认情况下，Abaqus/CAE保存您在会话期间指定的路径。如果要保留路径以供后续会话使用，请将其保存到XML文件、模型数据库或输出数据库；有关详细信息，请参阅["管理会话对象和会话选项，"第9.9节](pt02ch09s08.md)。
