# 19.7.5 在数据检查后继续分析作业







当您执行数据检查分析时（请参见["对模型执行数据检查，"第 19.7.3 节](pt03ch19s07hlb03.md)），Abaqus 会创建并保存所有必要的文件，以便稍后继续进行完整分析。要继续分析，请从主菜单中选择****作业**![](../graphics/images/arrow.gif)**继续**![](../graphics/images/arrow.gif)***您选择的作业*****。您也可以通过在**作业管理器**中选择作业名称，然后点击**继续**来继续分析。Abaqus/CAE 会使用作业编辑器中定义的作业设置继续分析。

有关监视已提交作业的信息，请参见["监视分析作业的进度，"第 19.2.6 节](pt03ch19s02hlb01.md)。

![](../graphics/images/black4rule.gif)有关相关主题的信息，请点击以下任一项目：- ["理解分析作业，"第 19.2 节](pt03ch19s02.md)
- ["创建、编辑和操作作业，"第 19.7 节](pt03ch19s07.md)
