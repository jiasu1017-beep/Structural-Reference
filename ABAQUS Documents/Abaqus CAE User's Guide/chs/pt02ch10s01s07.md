# 10.1.7 实体建模器如何表示实体？





Abaqus/CAE提供了允许您将草图、零件或装配导入当前模型的工具。当您导入草图或平面零件或装配时，过程通常很简单。但是，当您导入实体时，您可能会发现必须执行一些额外的步骤才能获得满意的结果。要理解这些步骤完成什么以及为什么需要执行它们，您需要了解实体建模器如何表示实体。

在过去30年中，随着建模系统的发展，表示实体的技术也在发展。每一代新的建模器都包含更多关于体构建的知识，并且越来越少地依赖大量数据点来描述实体。

**线框模型**

原始CAD系统使用二维线框来复制传统机械图纸。后来的版本以等轴测视图和透视视图的形式引入了三维性。在线框中，实体体由定义体边缘的一组曲线表示；但是，系统没有关于边缘之间曲面的信息。线框模型通过其边缘和顶点定义对象；因此，实体的线框表示用途有限。例如，您无法计算实体的体积，也无法对实体进行网格划分。

**修剪曲面**

后来的系统引入了修剪曲面的概念，如图[图10-2](pt02ch10s01s07.md#dbs-conc-trim-surface)所示。

**图10-2** 修剪曲面。

![](../graphics/dbs-conc-trim-surface-nls.png)

修剪曲面由曲面几何和修剪曲线的组合定义。曲面几何是曲面的通用表达式；修剪曲线在曲面几何上形成封闭环并定义曲面的边界。曲面也可以有多个内部修剪曲线曲面。实体被定义为一组形成封闭体积的修剪曲面。

例如，圆柱体可以从三个修剪曲面定义，如图[图10-3](pt02ch10s01s07.md#dbs-conc-trim-cylinder)所示。

**图10-3** 三个修剪曲面定义圆柱体。

![](../graphics/dbs-conc-trim-cylinder-nls.png)

每个曲面都包含一组定义曲面边界的边缘。圆柱体顶部的平坦修剪曲面包含修剪曲线*A1*。圆柱体底部的平坦修剪曲面包含修剪曲线*B1*。圆柱形修剪曲面以两条修剪曲线*A2*和*B2*为边界。当两个修剪曲面相交时，生成边缘的定义由每个曲面复制。没有信息指示一组修剪曲面构成实体。

当相交的曲面是平面、圆柱形或球形时，实体的边缘定义良好。但是，当更复杂的曲线相交时，生成的边缘必须由近似两个面相交的多项式表达式描述。近似的精度取决于多项式表达式的阶数。如果边缘不在任一曲面上，则会产生间隙，实体被视为无效。[图10-4](pt02ch10s01s07.md#dbs-conc-trim-gap)说明修剪曲面之间的间隙。

**图10-4** 修剪曲面之间的间隙。

![](../graphics/dbs-conc-trim-gap-nls.png)

您可以使用几何编辑工具集来缝合间隙。更多信息，请参阅["什么是缝合？，"第69.3节"](pt06ch69s03.md)。

IGES和VDA-FS标准都使用修剪曲面。导入期间可能发生的大多数问题都源于将修剪曲面从原始文件转换为Abaqus/CAE可识别的形式。

**B-rep**

最近，实体建模器引入了"边界表示"或"B-rep"的概念来定义实体对象。B-rep实体类似于由一组修剪曲面定义的实体；但是，B-rep实体包含有关在曲面相交形成实体时生成的面、边缘和顶点的附加信息。Abaqus/CAE使用ACIS存储几何实体，ACIS使用B-rep的概念。[图10-5](pt02ch10s01s07.md#dbs-conc-trim-brep)说明了定义为B-rep实体的圆柱体。

**图10-5** B-rep实体。

![](../graphics/dbs-conc-trim-brep-nls.png)

与仅由修剪曲面表示的实体不同，B-rep实体不会复制两个曲面共享的边缘。在B-rep实体中，一个修剪曲面定义共享边缘，第二个边缘引用该定义。要正确重新创建B-rep实体，必须能够沿着第二个曲面的几何形状复制由第一个曲面定义的修剪曲线。在某些情况下，ACIS会缝合相邻边缘以创建B-rep实体。

IGES和STEP标准包括B-rep的概念，尽管IGES称它们为流形实体B-rep对象或MSBO。如果您导入包含两个或多个足够紧密可以缝合在一起形成B-rep实体的修剪曲面的IGES或STEP文件，Abaqus/CAE会将这些曲面分组为单个实体。VDA-FS标准不包括B-rep实体的概念；VDA-FS仅使用修剪曲面来定义实体。
