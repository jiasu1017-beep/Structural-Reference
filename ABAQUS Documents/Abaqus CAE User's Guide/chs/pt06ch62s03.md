# 62.3 为什么datum坐标系如此重要？







datum坐标系在整个Abaqus/CAE中使用；例如：
- 定义材料取向。
- 定义连接器取向。
- 定义耦合约束的取向。
- 定义惯性relief载荷的取向。
- 定义边界条件的取向。
- 定位和对齐零件实例。

此外，当系统提示您选择一个点时，您可以选择datum坐标系的原点；当系统提示您选择一个边时，您可以选择datum坐标系的轴。

Abaqus/CAE提供了三种创建datum坐标系的方法：
- 三点
- 从坐标系偏移
- 两条线
