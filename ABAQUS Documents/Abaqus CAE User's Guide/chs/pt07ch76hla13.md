# 76.12 自定义网格显示




您可以使用 ****视图**![](../graphics/images/arrow.gif)**零件显示选项**** 和 ****视图**![](../graphics/images/arrow.gif)**装配显示选项**** 菜单项来指定是否在网格化零件或装配上显示节点和单元标签。您可以选择显示或抑制原生网格，并且如果显示，可以选择仅在网格模块中或所有零件相关模块或所有装配相关模块中显示。如果您显示原生网格，可以选择同时显示自底向上网格划分截面的几何、非自底向上截面或两者。显示几何可以直观地显示网格与几何的拟合程度。

**自定义网格显示：**

1. 找到网格显示选项。从主菜单栏，选择 ****视图**![](../graphics/images/arrow.gif)**零件显示选项**** 或 ****视图**![](../graphics/images/arrow.gif)**装配显示选项****。在出现的对话框中，点击 **网格** 选项卡。
2. 切换 **显示原生网格** 以显示或抑制原生网格。当 **显示原生网格** 打开时，您还可以控制以下选项： 1. 选择以下一个或两个选项以使用原生网格显示几何： - 切换 **显示自底向上几何** 以显示已分配自底向上网格划分技术的区域的几何。此选项默认打开。 - 切换 **显示非自底向上几何** 以显示分配了所有其他网格划分技术的区域的几何。此选项默认关闭。 2. 选择以下选项之一来控制可以显示原生网格的模块： - 选择 **仅在网格模块中** 以仅在网格模块中显示原生网格。 - 从 **零件显示选项** 对话框中选择 **在所有零件相关模块中** 或从 **装配显示选项** 对话框中选择 **在所有装配相关模块中**，以在支持零件或装配显示的所有模块中分别显示原生网格。
3. 切换 **显示节点标签** 和 **显示单元标签** 以影响这些项目的显示。
4. 点击 **确定** 实施更改并关闭对话框。更改仅适用于当前视口，并在会话期间保存。
