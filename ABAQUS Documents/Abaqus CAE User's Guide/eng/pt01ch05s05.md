# 5.5 Controlling perspective







Perspective representation accurately depicts the spatial relationship of three-dimensional objects in a two-dimensional plane. In other words, a three-dimensional model on your screen appears more realistic when perspective is turned on. Alternatively, parallel lines in the model appear parallel when perspective is turned off. Perspective affects all plots except *X–Y* plots, applies in all modules, and is turned on by default.

- To turn perspective on, select the ![](../graphics/ico_viewPerspective.png) icon located in the **View Options** toolbar or select ****View**![](../graphics/images/arrow.gif)**Perspective**** from the main menu bar.
- To turn perspective off, select the ![](../graphics/ico_viewParallel.png) icon located in the **View Options** toolbar or select ****View**![](../graphics/images/arrow.gif)**Parallel**** from the main menu bar.

Your changes apply only to the current viewport and are saved for the duration of the session.![](../graphics/images/black4rule.gif)For information on related topics, click the following item: - [Chapter 5, "Manipulating the view and controlling perspective](pt01ch05.md)"
