# 64.7.6 将三角形壳网格转换为实体四面体网格







从主菜单栏中选择 ****Mesh**![](../graphics/images/arrow.gif)**Edit**** 可将闭合的三维三角形单元壳转换为四面体单元的实体网格。原始壳零件必须用三角形单元网格化，这些单元完全包围一个没有间隙的体积。

**将壳网格转换为实体网格：**

1. 进入 Mesh 模块。
2. 从上下文栏的 **Object** 字段中选择 **Part**，然后从列表中选择一个零件。此工具仅适用于孤立网格单元。
3. 从主菜单栏中选择 ****Mesh**![](../graphics/images/arrow.gif)**Edit****。Abaqus/CAE 显示 **Edit Mesh** 对话框。**提示：**您也可以使用位于 Mesh 模块工具箱底部的 ![](../graphics/ico_meshEdit.png) 工具来显示 **Edit Mesh** 对话框。
4. 从 **Category** 字段中选择 **Mesh**。
5. 从 **Method** 列表中选择 **Convert tri to tet**。
6. 从提示区域出现的按钮中点击 **Yes**。Abaqus/CAE 将三角形网格转换为四面体网格。**提示：**您可以通过点击 **Edit Mesh** 对话框中的 **Undo** 来恢复转换之前的三角形网格状态。

编辑孤立网格后，您应始终验证基于节点、单元和表面的特征（如截面分配、载荷和相互作用），以确保它们正确应用于修改后的网格。
