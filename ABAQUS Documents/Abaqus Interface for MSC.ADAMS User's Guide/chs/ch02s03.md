# 2.3 设置 Abaqus 模型以创建包含应力或应变的模态中性文件





如果您希望 Abaqus Interface for MSC.ADAMS 将应力或应变转换到模态中性文件，则必须修改上一节中的模板来准备第一次 Abaqus 分析的输入文件。您必须在 [*FREQUENCY](../key/key-link.md#usb-kws-hfrequency) 步骤中包含应力或应变输出请求，如下例所示：

```
********************
*EL FILE, POSITION=NODES, DIRECTIONS=YES
1,
S,
E,
********************
```

关于输出请求注意以下几点：
- POSITION=NODES 参数是必需的。
- 对于所有模型，建议使用 DIRECTIONS=YES 参数。对于包含壳单元的模型，此参数是必需的。
- 截面点编号（在此例中为 `*EL FILE` 下一行的 `1`）对于包含壳单元的模型是必需的。对于实体单元，截面点编号将被忽略。只有单个截面点的应力或应变可以转换到模态中性文件。
- 应力（S）、应变（E）或两者都可以写入结果文件。

此外，您必须运行第二次 Abaqus 分析来恢复静态约束模态的子结构应力或应变。您可以使用 ["创建第二个输入文件" 第 3.3.1 节](ch03s03.md#adm-chp-mnf-command-input) 中描述的过程创建第二次 Abaqus 分析的输入文件。关于第一次 Abaqus 分析中的 [*SUBSTRUCTURE GENERATE](../key/key-link.md#usb-kws-ssubgenerate) 步骤注意以下几点：
- 如果在 [*RETAINED NODAL DOFS](../key/key-link.md#usb-kws-sretainnodaldofs) 选项上使用 SORTED=NO 参数，并且相同的节点编号（或节点集）在数据行上出现多次，则必须编辑第二个输入文件，使得使用级别上对应的节点出现相同次数。更多信息，请参阅 Abaqus Analysis User's Guide ["使用子结构" 第 10.1.1 节](../usb/usb-link.md#usb-anl-asuperelements-ordering) 中的"使用级别上子结构节点的排序"。
