# 19.12.4 提交优化过程







要提交优化过程，请从主菜单栏中选择****优化**![](../graphics/images/arrow.gif)**提交**![](../graphics/images/arrow.gif)***优化过程名称*****。（您也可以通过在**优化过程管理器**中选择优化过程名称，然后点击**提交**来提交优化过程。）Abaqus/CAE 使用优化过程编辑器中定义的设置提交优化过程。点击**优化过程管理器**中的**监视器**以监视优化过程。有关更多信息，请参见["监视您的优化过程，"第 19.12.6 节](pt03ch19s12hlb06.md)。

优化过程出现在模型树中。**优化过程**容器包含一个**作业**容器，其中包含优化过程中执行的 Abaqus 作业。您可以通过从优化过程监视器顶部点击**监视器**按钮来监视正在运行的 Abaqus 作业。如果您的优化过程包含多个模型，优化模块会为每个模型创建一个分析作业，您必须首先选择要监视的所需作业。您还可以通过在模型树中右键点击作业并选择**监视器**来查看先前设计循环期间完成的 Abaqus 作业的监视器。要终止优化过程，您必须终止当前的 Abaqus 作业。有关更多信息，请参见["监视分析作业的进度，"第 19.2.6 节](pt03ch19s02hlb01.md)。
