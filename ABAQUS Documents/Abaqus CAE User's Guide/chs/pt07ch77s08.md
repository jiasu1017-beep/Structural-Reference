# 77.8 自定义单个对象的显示颜色





本节介绍如何自定义各个对象的显示；这些示例适用于整个Abaqus/CAE，包括可视化模块。一旦Abaqus/CAE为您的视口创建了自动颜色映射，您可以通过更改颜色映射中分配给各个对象的颜色来自定义颜色显示。**颜色编码**对话框在**颜色映射**表的同一行中显示每个对象及其分配的颜色。对于每个对象，您可以选择不同的颜色，激活或停用其显示，以及设置（或恢复）所选对象的默认颜色。**颜色编码**对话框还提供支持管理每个对象颜色的选项：您可以按对象名称筛选，排序表列，以及高亮显示所选对象。

在可视化模块中，您可以对整个视口或仅对节点或单元应用颜色编码。您无法在其他Abaqus/CAE模块中的模型数据库中对节点或单元进行颜色编码。

以下选项可用于自定义各个对象的显示颜色：

**更改单个项目的填充颜色**

**颜色**列显示分配给每个对象的填充颜色。要选择不同的颜色，请双击该行中的颜色样本![](../graphics/usi-colorsample.png)或高亮显示颜色样本并点击**编辑颜色**颜色样本。Abaqus/CAE打开**选择颜色**对话框，您可从中选择新的填充颜色。（有关选择自定义颜色的更多信息，请参见["自定义颜色，"第3.2.9节](pt01ch03s02s09.md)。）当您点击**应用**时，Abaqus/CAE会用新的填充颜色刷新当前视口。

**更改单个项目的边缘颜色（仅可视化模块）**

当您从可视化模块打开**颜色编码**对话框时，对话框包含**边缘**列，显示分配给每个对象的边缘颜色。您无法在其他任何模块中自定义边缘颜色。

要更改单个项目的边缘颜色，请双击该行中的颜色样本![](../graphics/usi-colorsample.png)或高亮显示颜色样本并点击并按住**编辑颜色**符号（`=`、`*`或![](../graphics/usi-colorsample.png)）。Abaqus/CAE打开**选择颜色**对话框，您可从中选择新的边缘颜色。（有关选择自定义颜色的更多信息，请参见["自定义颜色，"第3.2.9节](pt01ch03s02s09.md)。）当您点击**应用**时，Abaqus/CAE会用新的边缘颜色刷新当前视口。

**激活和停用表行**

Abaqus/CAE仅对活动对象显示颜色编码；非活动对象使用初始颜色呈现。停用颜色映射中多个对象的颜色编码可以简化显示，帮助您检查或调试模型。

**注意：**您无法停用模块默认颜色映射中对象的颜色编码。

要切换对象颜色编码的激活状态，请点击**状态**并从出现的列表中选择激活或停用选项。如果您选择**全部激活**或**全部停用**，Abaqus/CAE会切换**颜色映射**表中所有行的**活动**列的状态。如果您选择**激活所选**或**停用所选**，Abaqus/CAE仅切换高亮显示行的**活动**列的状态。您还可以选择和取消选择**活动**列中的复选框来切换这些行中对象的颜色编码状态。当您在**颜色编码**对话框中点击**应用**时，Abaqus/CAE仅对活动行在当前视口中应用颜色编码。

**高亮显示所选对象**

选择**高亮显示项目**复选框可在**颜色映射**表中任何所选行周围显示高亮边框。Abaqus/CAE仅对以下颜色映射启用高亮显示：**零件实例**、**集合**、**曲面**、**内部集合**和**内部曲面**。当您对节点或单元应用颜色编码时，无法高亮显示对象。

**恢复默认颜色并设置新默认值**

更改各个对象的显示颜色后，您可以通过点击**颜色映射**，然后从出现的列表中选择**恢复默认**来恢复到当前颜色映射的默认颜色设置。或者，如果您希望将更改设置为会话的新默认颜色映射，请从同一列表中选择**设为默认**。

**自动为各个对象着色**

您可以对颜色映射中所选对象应用自动着色。选择要更改的各个对象，然后从**颜色映射**列表中选择**自动着色所选**。Abaqus/CAE根据所选行中名称的字母顺序从自动颜色列表应用颜色编码。

**按列排序**

点击**颜色映射**表中的列标题可按所选列的内容对表进行排序。第二次点击同一标题可反转排序顺序。

**注意：**当您对节点或单元进行颜色编码时，无法按列排序。

更改**颜色映射**表中对象的排序顺序仅用于导航；它们在对话框中的顺序不影响为颜色编码选择的颜色。Abaqus/CAE根据颜色映射中项目名称的字母顺序从自动颜色列表为模型定义分配颜色。

**按名称筛选行**

如果所选颜色映射包含许多行，您可以使用**名称筛选器**来减少显示的行数。点击**名称筛选器**字段旁边的![](../graphics/ico_tipSmall.png)可查看有效筛选语法的示例。
