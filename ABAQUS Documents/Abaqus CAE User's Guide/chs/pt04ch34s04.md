# 34.4 查看载荷工况输出





载荷工况输出存储在输出数据库（`.odb`）中的单独帧中。您使用Visualization模块查看和操作载荷工况输出。载荷工况名称显示在包含载荷工况输出的每个帧的状态块中。例如，[图34-5](pt04ch34s04.md#usi-adv-lc-odb)显示了悬臂梁模型的`Pressure Load w End Load Up`载荷工况的变形图（参见["载荷工况编辑器"，第34.3节](pt04ch34s03.md)）。

**图34-5** 载荷工况分析的结果。

![](../graphics/adv-lc-odb.png)

在Visualization模块中，您可以线性组合多个载荷工况的输出以表示实际加载环境。您还可以获得所选场变量在某些或所有载荷工况上的最小值或最大值。

例如，您可以组合悬臂梁模型的`End Load Only`和`Pressure Load Only`载荷工况的输出，以创建名为`Both loads`的新场输出，如图34-6所示。

**图34-6** 通过组合两个载荷工况（帧）的输出创建新的场输出。

![](../graphics/adv-odb-createoutputdb-nls.png)

新的场输出包含在会话步骤的帧中，可从**Field Output**对话框中获得。[图34-7](pt04ch34s04.md#usi-adv-lc-odbboth)显示了组合变形结果图。

**图34-7** 两个载荷工况的组合变形结果。

![](../graphics/adv-lc-odbboth.png)

有关操作载荷工况输出的详细信息，请参见["组合多个帧的结果"，第42.7.3节](pt05ch42s05hlb03.md)。
