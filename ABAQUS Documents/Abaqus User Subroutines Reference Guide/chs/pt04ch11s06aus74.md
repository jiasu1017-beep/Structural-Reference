### 概述

质量缩放通常用于 Abaqus/Explicit 中，以提高准静态分析和某些包含少数控制稳定时间增量的小单元的动态分析的计算效率。质量缩放可用于：
- 缩放整个模型的质量或缩放单个单元和/或单元集的质量；
- 在多步骤分析中按步骤缩放质量；以及
- 在步骤开始时和/或整个步骤中缩放质量。

质量缩放可以通过以下方式执行：
- 通过用户提供的常数因子缩放所有指定单元的质量；
- 通过相同的值缩放所有指定单元的质量，使得单元集中任何单元的最小稳定时间增量等于用户提供的时间增量；
- 仅缩放其单元稳定时间增量小于用户提供的时间增量的单元集中的单元的质量，使得这些单元的单元稳定时间增量变得等于用户提供的时间增量；
- 缩放所有指定单元的质量，使其单元稳定时间增量各自变得等于用户提供的时间增量；以及
- 基于网格几何和初始条件自动缩放，用于 bulk 金属轧制分析。

### 简介

显式动力学过程通常用于解决两类问题：瞬态动态响应计算和涉及复杂非线性效应的准静态模拟（最常见的是涉及复杂接触条件的问题）。因为使用显式中心差分法来积分时间中的方程（参见 ["显式动态分析," 第 6.3.3 节"](pt03ch06s03at08.md)），所以平衡方程中使用的离散质量矩阵在两类问题的计算效率和准确性中都起着至关重要的作用。如果使用得当，质量缩放通常可以在保持特定问题类别所需准确程度的同时提高计算效率。然而，对准静态模拟最适用的质量缩放技术可能与应该用于动态分析的技术完全不同。

#### 准静态分析

对于包含与率无关材料行为的准静态模拟，自然时间尺度通常不重要。为了获得经济的解决方案，缩短分析的时间周期或人为增加模型的质量（"质量缩放"通常是有用的。对于与率无关的材料，这两种方法会产生相似的结果，尽管如果模型中包含率依赖性，则质量缩放是减少求解时间的优选方法，因为自然时间尺度被保留。

准静态分析的质量缩放通常对整个模型执行。然而，当模型的不同部分具有不同的刚度和质量属性时，仅缩放模型的选定部分或独立缩放每个部分可能是有用的。在任何情况下，都不必将模型的质量从其物理值降低，而且通常不可能在不降低准确性的情况下随意增加质量。有限的缩放量通常对大多数准静态情况是可能的，这将导致 Abaqus/Explicit 使用的时间增量相应增加，并减少计算时间。但是，您必须确保质量的改变和由此产生的惯性力的增加不会显著改变解决方案。

虽然可以通过修改模型中材料的密度来实现质量缩放，但本节描述的方法提供了更大的灵活性，特别是在多步骤分析中。

有关在准静态分析中使用质量缩放的讨论，请参见 ["厚板轧制," Abaqus 例题指南第 1.3.6 节](../exa/exa-link.md#exa-sta-rolling)。

#### 动态分析

自然时间尺度在动态分析中始终很重要，需要准确表示模型中的物理质量和惯性以捕获瞬态响应。然而，许多复杂的动态模型包含少数非常小的单元，这将迫使 Abaqus/Explicit 使用小的时间增量来积分整个模型。这些小单元通常是困难的网格生成任务的结果。通过在步骤开始时缩放这些控制单元的质量，可以显著增加稳定时间增量，而对模型整体动态行为的影响可以忽略不计。

在冲击分析中，靠近冲击区域的单元通常会经历大量变形。这些单元减小的特征长度导致更小的全局时间增量。在整个模拟过程中根据需要缩放这些单元的质量可以显著减少计算时间。对于压缩单元冲击静止刚体的情况，模拟过程中这些小单元的质量增加对整体动态响应的影响非常小。

对于真正的动态事件，质量缩放几乎总是只应对有限数量的单元进行，并且不应显著增加模型的整体质量属性，否则会降低动态解决方案的准确性。

有关在动态分析中使用质量缩放的讨论，请参见 ["铜棒的冲击," Abaqus 基准指南第 1.3.10 节](../bmk/bmk-link.md#bmk-anl-rodimpact)。

#### 稳定时间增量

在本节中，"单元稳定时间增量"一词指的是单个单元的稳定时间增量。"逐单元稳定时间增量"一词指的是特定单元集中的最小单元稳定时间增量。"稳定时间增量"一词指的是整个模型的稳定时间增量，无论使用全局估计器还是逐单元估计器。

### 将质量缩放引入模型

Abaqus/Explicit 中有两种质量缩放可用：固定质量缩放和可变质量缩放。这两种质量缩放可以分别应用，也可以一起应用以定义整体质量缩放策略。质量缩放也可以全局应用于整个模型，或者可选地基于单元集逐单元集应用。

#### 固定质量缩放

固定质量缩放在为其指定的步骤开始时执行一次。固定质量缩放有两种基本方法：您可以直接定义质量缩放因子，或者可以定义所需的最稳定时间增量，由 Abaqus/Explicit 确定质量缩放因子。

如果在步骤中同时指定了可变质量缩放和固定质量缩放，则单元原始质量基于指定的固定质量缩放在该步骤开始时缩放一次。然后在该步骤开始和定期根据指定的可变质量缩放进一步缩放。

固定质量缩放提供了一种在分析开始时修改准静态模型质量属性或在动态模型中修改少数小单元质量的简单方法，使得它们不控制稳定时间增量大小。由于缩放操作仅在定义质量缩放的步骤开始时执行一次，固定质量缩放在计算上是高效的。

| **输入文件用法：** | ``` [*FIXED MASS SCALING](../key/key-link.md#usb-kws-hfixedmassscaling) ``` |
| --- | --- |

| **Abaqus/CAE 用法：** | 步骤模块：**Create Step**：**General**，**Dynamic, Explicit** 或 **Dynamic, Temp-disp, Explicit**：**Mass scaling**：**Use scaling definitions below**：**Create**：**Semi-automatic mass scaling**，**Scale: At beginning of step** |
| --- | --- |

#### 可变质量缩放

可变质量缩放用于在步骤开始时和该步骤期间定期缩放单元的质量。使用这种类型质量缩放时，您需要定义所需的最稳定时间增量：质量缩放因子将自动计算并在整个步骤中根据需要应用。

如果在步骤中同时指定了可变质量缩放和固定质量缩放，则单元原始质量基于指定的固定质量缩放在该步骤开始时缩放一次。然后在该步骤开始和定期根据指定的可变质量缩放进一步缩放。

当控制稳定时间增量的刚度属性在步骤中剧烈变化时，可变质量缩放最有用。这种情况可能出现在准静态 bulk 成型和高度压缩或压碎单元的动态模拟中。

| --- | --- |

#### 直接定义缩放因子

直接定义缩放因子对于模型中动能应保持较小的准静态分析是有用的。您可以定义应用于指定单元集中所有单元原始质量的固定质量缩放因子。单元的质量将在步骤开始时缩放，并在整个步骤中保持不变，除非被可变质量缩放进一步修改。

| **输入文件用法：** | ``` [*FIXED MASS SCALING](../key/key-link.md#usb-kws-hfixedmassscaling), FACTOR=*scale_factor* ``` |
| --- | --- |
|  | 例如，以下选项将包含在单元集 *elset* 中的单元的质量缩放因子为 10： ``` [*FIXED MASS SCALING](../key/key-link.md#usb-kws-hfixedmassscaling), FACTOR=10., ELSET=*elset* ``` |

| **Abaqus/CAE 用法：** | 步骤模块：**Create Step**：**General**，**Dynamic, Explicit** 或 **Dynamic, Temp-disp, Explicit**：**Mass scaling**：**Use scaling definitions below**：**Create**：**Semi-automatic mass scaling**，**Scale: At beginning of step**，**Scale by factor:** *scale_factor* |
| --- | --- |

#### 定义所需的逐单元稳定时间增量

您可以为固定或可变质量缩放定义单元集的所需逐单元稳定时间增量。然后 Abaqus/Explicit 将确定必要的质量缩放因子。当定义所需的逐单元稳定时间增量时，有三种互斥的方法可用于缩放模型的质量。每种方法将在本节后面详细描述。

为了确定增量期间使用的稳定时间增量，Abaqus/Explicit 首先逐单元确定最小的稳定时间增量。然后，全局估计算法基于模型的最高频率确定稳定时间增量。两个估计中较大的一个决定所使用的稳定时间增量。通常，全局估计器确定的稳定时间增量将大于逐单元估计器确定的稳定时间增量。当使用指定逐单元稳定时间增量的固定或可变质量缩放来缩放一组单元的质量时，逐单元稳定时间增量估计直接受到影响。如果模型中的所有单元通过单一质量缩放定义进行缩放，则逐单元估计将等于分配给逐单元稳定时间增量的值，除非使用罚方法强制接触约束。罚接触可能导致逐单元估计略低于分配给逐单元稳定时间增量的值（参见 ["Abaqus/Explicit 中通用接触的接触控制," 第 36.4.5 节"](pt09ch36s04aus159.md)，和 ["Abaqus/Explicit 中的接触约束 enforcement 方法," 第 38.2.3 节"](pt09ch38s02aus182.md)。由于全局估计器的使用，实际使用的稳定时间增量可能大于分配给逐单元稳定时间增量的值。如果仅对模型的一部分执行质量缩放，则未缩放的单元可能具有小于分配给逐单元稳定时间增量的单元稳定时间增量，在这种情况下将控制逐单元稳定时间增量估计。因此，如果仅对模型的部分进行缩放，则使用的时间增量通常不会等于分配给逐单元稳定时间增量的值。

如果显式动态步骤的固定时间增量大小基于初始逐单元稳定性限制或直接指定，则使用的时间增量将根据 ["显式动态分析," 第 6.3.3 节"](pt03ch06s03at08.md) 中描述的规则计算。

##### 统一缩放质量

统一缩放质量对于模型中动能应保持较小的准静态分析是有用的。这种方法类似于直接定义缩放因子。在这两种情况下，所有指定单元的质量都通过单一因子统一缩放。但是，使用此方法，质量缩放因子由 Abaqus/Explicit 确定，而非用户指定。单一质量缩放因子统一应用于所有单元，使得这些单元中的最小稳定时间增量等于分配给逐单元稳定时间增量的值 *dt*。

| **输入文件用法：** | 使用以下任一选项： |
| --- | --- |
|  | ``` [*FIXED MASS SCALING](../key/key-link.md#usb-kws-hfixedmassscaling), TYPE=UNIFORM, DT=*dt* [*VARIABLE MASS SCALING](../key/key-link.md#usb-kws-hvariablemassscaling), TYPE=UNIFORM, DT=*dt* ``` |

| **Abaqus/CAE 用法：** | 步骤模块：**Create Step**：**General**，**Dynamic, Explicit** 或 **Dynamic, Temp-disp, Explicit**：**Mass scaling**：**Use scaling definitions below**：**Create**：**Semi-automatic mass scaling**，**Scale: At beginning of step** 或 **Throughout step**，**Scale to target time increment of:** *dt*，**Scale element mass: Uniformly to satisfy target** |
| --- | --- |

##### 仅缩放稳定时间增量低于指定逐单元稳定时间增量的单元

缩放稳定时间增量低于用户指定值的单元适用于准静态和动态分析。它对于增加最关键单元的单元稳定时间增量很有用。

当分析或步骤开始时的网格包含少数控制稳定时间增量大小的非常小的单元时，使用固定质量缩放来缩放这些单元的质量，并以所需的时间增量值开始步骤。仅增加这些控制单元的质量意味着稳定时间增量可以显著增加，而对模型整体行为的影响可以忽略不计。

对于变形演化产生有限数量小单元的分析，使用可变质量缩放来缩放这些单元的质量，从而限制稳定时间增量的减小。

| --- | --- |
|  | ``` [*FIXED MASS SCALING](../key/key-link.md#usb-kws-hfixedmassscaling), TYPE=BELOW MIN, DT=*dt* [*VARIABLE MASS SCALING](../key/key-link.md#usb-kws-hvariablemassscaling), TYPE=BELOW MIN, DT=*dt* ``` |

| **Abaqus/CAE 用法：** | 步骤模块：**Create Step**：**General**，**Dynamic, Explicit** 或 **Dynamic, Temp-disp, Explicit**：**Mass scaling**：**Use scaling definitions below**：**Create**：**Semi-automatic mass scaling**，**Scale: At beginning of step** 或 **Throughout step**，**Scale to target time increment of:** *dt*，**Scale element mass: If below minimum target** |
| --- | --- |

##### 缩放所有单元使其具有相等的单元稳定时间增量

缩放所有单元使其具有相同的稳定时间增量有效地收缩了模型的特征谱；也就是说，它降低了模型最低和最高自然频率之间的范围。由于质量属性的剧烈变化，此方法仅适用于准静态分析。它意味着某些单元的质量缩放因子可能小于一。

### 全局和局部质量缩放

为固定或可变质量缩放指定单元集可缩放模型的局部区域的质量。省略单元集意味着将对所有单元执行质量缩放。可以通过使用指定的单元集重复质量缩放定义来覆盖给定单元集的全局定义。

| **输入文件用法：** | 使用以下任一选项： |
| --- | --- |
|  | ``` [*FIXED MASS SCALING](../key/key-link.md#usb-kws-hfixedmassscaling), ELSET=*elset* [*VARIABLE MASS SCALING](../key/key-link.md#usb-kws-hvariablemassscaling), ELSET=*elset* ``` |

| **Abaqus/CAE 用法：** | 步骤模块：**Create Step**：**General**，**Dynamic, Explicit** 或 **Dynamic, Temp-disp, Explicit**：**Mass scaling**：**Use scaling definitions below**：**Create**：**Semi-automatic mass scaling**，**Scale: At beginning of step** 或 **Throughout step**，**Region: Set:** *elset* |
| --- | --- |

#### 示例 1

当分析中存在具有截然不同波速或网格细化的材料时，不同的质量缩放因子可能有用。在此示例中，对于准静态分析中除少数单元外的所有单元的质量，可能需要 50 的缩放因子，而对于这些少数单元，使用 500 的质量缩放因子。

```
[*FIXED MASS SCALING](../key/key-link.md#usb-kws-hfixedmassscaling), FACTOR=50.0
[*FIXED MASS SCALING](../key/key-link.md#usb-kws-hfixedmassscaling), FACTOR=500.0, ELSET=*elset1*
```

第一个固定质量缩放定义将模型中所有单元的质量缩放因子为 50。第二个固定质量缩放定义通过将包含在单元集 *elset1* 中的单元的质量缩放因子为 500 来覆盖第一个定义。

#### 示例 2

缩放 *elset1* 中单元质量的替代方法是为其分配稳定时间增量，并允许 Abaqus/Explicit 确定质量缩放因子。

```
[*FIXED MASS SCALING](../key/key-link.md#usb-kws-hfixedmassscaling), FACTOR=50.0
[*FIXED MASS SCALING](../key/key-link.md#usb-kws-hfixedmassscaling), DT=.5E-6, TYPE=BELOW MIN, ELSET=*elset1*
```

第一个固定质量缩放定义将整个模型的质量缩放因子为 50。第二个固定质量缩放定义通过缩放稳定时间增量小于 0.5×10⁶ 的 *elset1* 中任何单元的质量来覆盖第一个定义。

### 步骤开始时的质量缩放

固定质量缩放用于仅在步骤开始时规定质量缩放，并且始终缩放原始单元质量。当直接定义缩放因子时，质量按缩放因子分配的值进行缩放。如果指定了逐单元稳定时间增量 *dt*，则质量缩放基于此值。如果同时指定了缩放因子和逐单元稳定时间增量，则质量首先按缩放因子分配的值进行缩放，然后可能根据分配给逐单元稳定时间增量的值和所选固定质量缩放的类型再次缩放。

可以为特定单元集定义局部质量缩放。如果未指定单元集，则固定质量缩放定义将应用于模型中的所有单元。每个单元集只允许一个固定质量缩放定义。多个固定质量缩放定义不能包含重叠的单元集。局部质量缩放定义将覆盖指定单元集的全局定义。

| **输入文件用法：** | ``` [*FIXED MASS SCALING](../key/key-link.md#usb-kws-hfixedmassscaling), FACTOR=*factor*, DT=*dt*, TYPE=*type*, ELSET=*elset* ``` |
| --- | --- |

| **Abaqus/CAE 用法：** | 步骤模块：**Create Step**：**General**，**Dynamic, Explicit** 或 **Dynamic, Temp-disp, Explicit**：**Mass scaling**：**Use scaling definitions below**：**Create**：**Semi-automatic mass scaling**，**Scale: At beginning of step**，**Scale by factor:** *factor*，**Scale to target time increment of:** *dt* |
| --- | --- |

#### 示例

假设对于准静态分析，质量缩放因子 50 应用于模型中的所有单元。此外，假设即使按因子 50 缩放后，一些极小或形状不良的单元仍导致稳定时间增量小于所需的最小值。为了增加稳定时间增量，使用以下选项：

```
[*FIXED MASS SCALING](../key/key-link.md#usb-kws-hfixedmassscaling), FACTOR=50., TYPE=BELOW MIN, DT=.5E-6
```

指定的缩放因子导致模型中所有单元的质量按因子 50 进行缩放。如果任何单元的稳定时间增量在按因子 50.0 缩放后仍低于 0.5×10⁶，则其质量将进行缩放，使其稳定时间增量等于 0.5×10⁶。

### 步骤期间的质量缩放

具有指定逐单元稳定时间增量的可变质量缩放用于定义在步骤开始时和整个步骤中执行的质量缩放。必须指定增量频率或间隔数来定义质量缩放的执行频率。在除执行质量缩放外的其他增量中，使用的时间增量通常与分配给逐单元稳定时间增量的值不同。

可以为特定单元集定义局部质量缩放。如果未指定单元集，则可变质量缩放定义将应用于模型中的所有单元。每个单元集只允许一个可变质量缩放定义。多个可变质量缩放定义不能包含重叠的单元集。局部质量缩放定义将覆盖指定单元集的全局定义。

| **输入文件用法：** | ``` [*VARIABLE MASS SCALING](../key/key-link.md#usb-kws-hvariablemassscaling), DT=*dt*, TYPE=*type*, ELSET=*elset* ``` |

#### 示例

假设在动态冲击分析中，网格中存在一些极小或形状不良的单元，因此控制稳定时间增量。为了防止这些单元控制稳定时间增量，希望在步骤开始时缩放它们的质量。此外，网格某个区域的单元将因与固定刚性表面碰撞而产生严重变形。因此，冲击区域中的单元最终可能控制稳定时间增量。

由于冲击区域中的单元基本上相对于刚性表面静止，选择性地缩放它们的质量将保证整体动态响应不会受到不利影响。通过规定时间增量来缩放这些单元的质量以限制逐单元稳定时间增量的减小可能会大大减少运行时间。

例如，为模型中稳定时间增量低于 1.0×10⁶ 的所有单元指定固定质量缩放。此外，为冲击区域（*elset1*）中稳定时间增量低于 0.5×10⁶ 的单元指定可变质量缩放。在这种情况下，在步骤开始时检查模型中的所有单元。如果任何单元的稳定时间增量小于 1.0×10⁶，则（独立地）缩放其质量，使逐单元稳定时间增量等于 1.0×10⁶。此缩放在整个步骤中保持有效，不会被进一步修改，但 *elset1* 中的单元除外。可变质量缩放定义导致包含在 *elset1* 中的单元在整个步骤中被缩放，使其稳定时间增量不会变得小于 0.5×10⁶。因为只有在 *elset1* 中的单元在步骤期间被缩放，所以可能产生小于 0.5×10⁶ 的稳定时间增量。

### 多步骤分析中的质量缩放

一个步骤结束时的缩放单元质量以及在该步骤中指定的可变质量缩放方法会自动结转到后续步骤，确保步骤边界处质量矩阵的连续性以及可变质量缩放方法的持续应用。但是，您可以在后续步骤开始时将单元质量重置为其原始值，或使用新的固定质量缩放方法重新计算单元质量。您也可以删除从前一步骤继承的可变质量缩放方法，或用新的可变质量缩放方法替换继承的方法。

要重置初始质量矩阵，请在后续步骤中指定固定质量缩放方法。类似地，在后续步骤中指定可变质量缩放方法以停止前一步骤的所有可变质量缩放方法。以下示例说明以下特殊情况：（a）没有进一步质量缩放的连续质量矩阵，以及（b）将质量矩阵恢复到原始状态而没有进一步的质量缩放。

由于质量缩放导致步骤之间单元质量的非常大的变化可能导致质量计算中的精度问题。这些精度问题可能导致错误或误导性结果。当在这种情况下需要单元质量的大变化时，建议在新步骤中使用固定质量缩放将其恢复到原始值，然后再根据需要使用附加质量缩放定义，将其缩放到所需值。

#### 没有进一步缩放的连续质量矩阵

要定义没有进一步缩放的连续质量矩阵，请通过重新定义新的可变质量缩放定义来删除从前一步骤继承的任何可变质量缩放定义。

| **输入文件用法：** | 在新步骤中使用不带任何参数的以下选项： |
| --- | --- |
|  | ``` [*VARIABLE MASS SCALING](../key/key-link.md#usb-kws-hvariablemassscaling) ``` |

| --- | --- |

##### 示例

假设在准静态分析的第一步中，单元将经历导致稳定时间增量急剧减小的变形。此外，假设第二步期间的变形不会大到对稳定时间增量产生任何进一步影响。

```
[*HEADING](../key/key-link.md#usb-kws-mheading)
…
[*STEP](../key/key-link.md#usb-kws-hstep)
…

[*VARIABLE MASS SCALING](../key/key-link.md#usb-kws-hvariablemassscaling), TYPE=BELOW MIN, DT=1.E-5, FREQUENCY=10
…
[*END STEP](../key/key-link.md#usb-kws-hendstep)
[*STEP](../key/key-link.md#usb-kws-hstep)
…
[*VARIABLE MASS SCALING](../key/key-link.md#usb-kws-hvariablemassscaling)
…
[*END STEP](../key/key-link.md#usb-kws-hendstep)
```

在第一步中，固定质量缩放将单元质量增加因子 1.1。可变质量缩放定义在步骤开始时和每十个增量对整个模型的质量进行缩放，使逐单元稳定时间增量至少等于 1×10⁵。第二步中的可变质量缩放定义替换了从第一步延续的定义。第二步中这个没有参数的可变质量缩放定义也防止了第二步中的任何进一步质量缩放。第一步的缩放质量矩阵被结转用于整个第二步。

##### 示例

假设分析包含准静态步骤，然后是动态步骤。质量缩放可以在准静态步骤期间执行，但在动态步骤期间关闭。

#### 通过共仿真连接到 Abaqus 的外部程序的质量贡献

共仿真可能导致外部程序的质量和/或转动惯量在步骤期间添加到 Abaqus 模型中。但是，一旦共仿真步骤完成，必须移除该贡献以及从外部程序导入的其他量。如果预期共仿真会向 Abaqus 模型添加质量和/或转动惯量，Abaqus 会自动在这种共仿真步骤完成后将质量矩阵恢复为原始状态。您需要重新指定必须延续到共仿真步骤之后的任何质量缩放。

### 使用或不使用质量缩放的情况

以下实体不受质量缩放的影响：
- 完全耦合热应力分析中的热求解响应
- 重力载荷、粘性压力载荷
- 绝热热计算
- 状态方程材料
- 流体和流体链接单元
- 基于表面的流体空腔
- 弹簧和阻尼单元

此列表中任何相关项的密度将保持不变。质量、转动惯量、无限元和刚性单元可以缩放。但是，由于没有任何单元具有相关的稳定时间增量，它们只能使用用户指定的缩放因子或统一应用的逐单元稳定时间增量进行缩放。如果指定了逐单元稳定时间增量，则质量缩放定义中必须至少包含一个具有稳定时间增量的单元。壳、 梁和管道单元中的转动惯量基于缩放后的质量。

无限元的质量可以缩放；但是，除非每个相邻可变形单元的密度通过相同因子缩放，否则无限元不会充当静默边界。如果将两个单元都包含在同一固定或可变质量缩放定义中，则两个单元的质量将通过相同因子缩放。

### 体积金属轧制分析的自动质量缩放

体积金属轧制通常被认为是准静态过程，但由于其处理接触问题的能力，该过程通常使用 Abaqus/Explicit 进行建模。为了使用 Abaqus/Explicit 获得经济的解决方案，人为增加产品质量的质量通常是有用的。但是，必须选择质量缩放因子，使得质量的改变和相应惯性力的改变不会显著改变解决方案。选择过高的缩放因子不会产生准静态结果。选择过低的缩放因子，虽然保守，但会导致很长的运行时间。轧制可变质量缩放可用于使此过程的最优缩放因子选择自动化。

自动策略基于将所有单元缩放为具有相等单元稳定时间增量的半自动方法。该方法通过从轧制过程的几个参数确定目标稳定时间增量的适当值来实现自动化。目标稳定时间增量使用的值 ![](../graphics/usb_eqn00456.gif) 基于轧制方向的平均单元长度 ![](../graphics/usb_eqn00628.gif)；进料速率 *V*；以及产品横截面中的节点数 *n*。进料速率定义为稳态条件下产品在轧制方向上的平均速度。在分析过程中调整 ![](../graphics/usb_eqn00456.gif) 的值以考虑进料速率的实际值。您必须为平均速度、轧制方向的平均单元长度以及产品横截面中的节点数指定估计值。

任何单元的质量永远不会降至其原始质量以下。这与将所有单元缩放为具有相等单元稳定时间增量的方法不同。施加此限制意味着具有显著惯性效应的轧制问题在为准静态分析时不会自动调整其质量。

为获得良好结果，建议：
- 通过挤压产品的二维横截面来网格化产品；
- 轧制方向的平均单元长度沿产品长度不应显著变化；
- 产品在轧制方向上的初始速度大致等于稳态进料速率；
- 横截面中的单元尺寸等于或小于轧制方向的尺寸；以及
- 不要对使用轧制自动可变质量缩放缩放的单元使用其他质量缩放。

| --- | --- |

| **Abaqus/CAE 用法：** | 步骤模块：**Create Step**：**General**，**Dynamic, Explicit** 或 **Dynamic, Temp-disp, Explicit**：**Mass scaling**：**Use scaling definitions below**：**Create**：**Automatic mass scaling**，**Feed rate:** *V*，**Extruded element length:** ![](../graphics/usb_eqn00628.gif)，**Nodes in cross section:** *n* |

### 输出

输出变量 EMSF 提供单元质量缩放因子。Abaqus/CAE 可用于获取 EMSF 的轮廓图和历史图。输出变量 DMASS 提供由于质量缩放导致的模型总质量变化百分比，可用于 Abaqus/CAE 中的历史绘制。DMASS 不能基于单元集获得。

输出变量 EDT 提供单元稳定时间增量。单元稳定时间增量包括质量缩放的影响。Abaqus/CAE 可用于获取 EDT 的历史图。
