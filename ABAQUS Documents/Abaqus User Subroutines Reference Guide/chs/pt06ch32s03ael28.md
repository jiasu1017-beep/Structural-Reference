# 32.3.2 柔性关节单元库





**产品：** Abaqus/Standard  

##### **参考文献**

- ["柔性关节单元，" 第32.3.1节](pt06ch32s03alm39.md)
- [*JOINT](../key/key-link.md#usb-kws-mjoint)

### 概述

本节提供Abaqus/Standard中可用柔性关节单元的参考。

### 单元类型

| JOINTC | 关节相互作用单元 |
| --- | --- |
|  |

##### 活动自由度

1、2、3、4、5、6

##### 附加解变量

无。

### 所需节点坐标

无。元素节点不需要定义坐标，因为与这些元素相关的作用是通过指定所涉及的自由度来定义的。

### 单元属性定义

| **输入文件用法：** | ``` [*JOINT](../key/key-link.md#usb-kws-mjoint) ``` |
| --- | --- |

### 基于单元的加载

无。

### 单元输出

| S11 | 第一个局部方向的总直接力。 |
| --- | --- |

| S22 | 第二个局部方向的总直接力。 |
| --- | --- |

| S33 | 第三个局部方向的总直接力。 |
| --- | --- |

| S12 | 关于第一个局部方向的总力矩。 |
| --- | --- |

| S13 | 关于第二个局部方向的总力矩。 |
| --- | --- |

| S23 | 关于第三个局部方向的总力矩。 |
| --- | --- |

上述力和力矩对应的相对位移和转动通过请求相应的"应变"来选择。

### 与单元关联的节点

两个节点。元素第一个节点处的转动定义了局部轴系统的转动。
