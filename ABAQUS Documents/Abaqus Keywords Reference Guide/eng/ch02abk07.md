# *BEAM SECTION GENERATE







### *BEAM SECTION GENERATEGenerate beam section properties for a meshed cross-section.

This option is used to calculate the cross-section warping function, to define the centroid and shear center, and to generate the stiffness and inertia properties for a meshed cross-section. These properties are written to the file `*jobname*.bsp` for use in a subsequent beam analysis using the [*BEAM GENERAL SECTION](ch02abk05.md), SECTION=MESHED option.

**Product: **Abaqus/Standard  

**Type: **History data 

**Level: **Step
