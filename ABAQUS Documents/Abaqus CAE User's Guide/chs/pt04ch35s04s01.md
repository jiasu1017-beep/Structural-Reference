# 35.4.1 冲压支架的中面建模





此示例展示如何使用Abaqus/CAE中的中面建模工具将支架的实体模型转换为相同部件的壳表示。

**实体模型**

本示例中的模型是由3.0毫米低碳钢板冲压而成的175.0毫米长支架，重量如图35-3所示。实体部件的默认网格划分导致支架厚度方向有一到两个单元，这将不会良好地承受弯曲。对支架进行壳表示建模将提供更准确的弯曲响应。

**图35-3** 支架的实体模型。

![](../graphics/adv-midsurface-basicmodel.png)

**分配中面区域**

使用Part模块中的**Assign Midsurface Region**工具从模型的活动表示中移除几何体，并创建原始实体几何体的参考表示，重量如图35-4所示。

**图35-4** 支架的参考表示。

![](../graphics/adv-midsurface-refrep.png)

参考表示是原始支架的抽象表示。它保留支架的原始几何体，但不能用于分析。参考表示默认显示在Part模块中；您可以使用主工具栏中与可见对象工具一起的**Show Reference Representation**工具![](../graphics/ico_referenceRepToggle.png)切换关闭和开启。更多信息，请参见["理解参考表示"，第35.2节](pt04ch35s02.md)和["分配中面区域"，第35.5.1节](pt04ch35s05s01.md)。

**创建壳表示**

您必须创建可由Abaqus分析的支架壳表示。用于创建中面壳模型的最常用工具是位于Geometry Edit工具集中与其他面工具一起的offset face工具。偏移面工具允许您从参考表示中选择一个或多个面，并创建与原始面偏移的新面。您可以输入固定距离，或者您可以选择Abaqus/CAE用于计算距离的目标面。如果您选择目标面，则如果面与目标面之间的距离不是常数，则生成的壳厚度会变化。当您分析模型时，Abaqus/CAE会计算每个节点和单元的节点厚度和单元偏移。**Offset Faces**对话框如图35-5所示。

**图35-5** **Offset Faces**对话框。

![](../graphics/adv-midsurface-offsetdb-nls.png)

更多信息，请参见["编辑面的方法概述"，第69.2.2节](pt06ch69s02s02.md)和["添加壳特征"，第11.22节](pt03ch11s22.md)。

[图35-6](pt04ch35s04s01.md#adv-midsurface-selectfaces-nls)显示了选择偏移的面和目标面。**by angle**选择方法用于选择内部面组作为偏移的面，选择外部面组作为目标面。

**图35-6** 要偏移的面和目标面。

![](../graphics/adv-midsurface-selectfaces-nls.png)

在此示例中，您从**Offset Faces**对话框中选择默认选项**Half the average distance**，Abaqus/CAE创建距离偏移面和目标面之间平均值一半的面。

默认情况下，**Auto trim to reference representation**在**Offset Faces**对话框中关闭。但是，此步骤的自动修剪选项处于活动状态，因此Abaqus/CAE将偏移面修剪为与参考表示对齐。在某些情况下，修剪操作失败，导致新面与参考表示之间轻微不对齐，并显示来自Abaqus/CAE的警告消息。即使修剪失败，生成的壳面通常仍然是对参考表示的合理近似；因此您可能可以忽略警告消息。更多信息，请参见["为中面建模使用偏移面工具"，第35.3.1节](pt04ch35s03s01.md)。

**分配壳截面**

您使用Property模块创建壳截面并将其分配给新面。当您创建壳截面时，可以输入壳厚度的任意值。当您随后将截面分配给壳时，您可以在**Edit Section Assignment**对话框中指定厚度和壳偏移从几何体计算，Abaqus/CAE忽略您为壳截面输入的厚度值。

**图35-7** **Edit Section Assignment**对话框。

![](../graphics/adv-midsurface-editsectiondb-nls.png)

更多信息，请参见["分配截面"，第12.15.1节](pt03ch12s15hlb01.md)。

如果需要，您可以从**Part Display Options**开启**Render shell thickness**以查看壳的厚度，如图35-8所示。

**图35-8** 查看壳厚度。

![](../graphics/adv-midsurface-shellthickness.png)

**为部件划分网格**

Abaqus/CAE在Mesh模块中将壳部件显示为粉色，以指示可以使用自由网格划分技术对其进行网格划分，重量如图35-9所示。

**图35-9** 自由网格划分可以应用于部件。

![](../graphics/adv-midsurface-freemeshing.png)

您将自动虚拟拓扑应用于部件，以从原始部件和面修剪操作期间创建的细节中移除小细节，如["自动创建虚拟拓扑"，第75.6.1节](pt06ch75s06hlb01.md)中所述。

您应用默认种子和网格控制，生成的网格如图35-10所示。

**图35-10** 生成的网格。

![](../graphics/adv-midsurface-meshed.png)
