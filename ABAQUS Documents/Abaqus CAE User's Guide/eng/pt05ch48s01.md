# 48.1 Understanding results along a path







To view results along a path through your model, you first select ****Tools**![](../graphics/images/arrow.gif)**Path**** to specify the path and then select ****Tools**![](../graphics/images/arrow.gif)**XY Data**** to obtain *X–Y* data along the path.
