# 3.1.3 均匀应变的粘塑性桁架





**产品：** Abaqus/Standard

本例旨在为Abaqus中用于弹性-塑性材料在高应变率下响应的粘塑性本构模型的验证提供基本验证。Abaqus中实现的粘塑性模型形式在["金属塑性，" Abaqus理论指南第4.3节](../stm/stm-link.md#stmmetalplastchap)中有详细描述。

问题是一个单自由度系统，由一个均匀应变的桁架和一个点质量组成（见图3.1.3-1）。桁架假定由弹性、粘塑性材料制成。系统在桁架未应变时施加初始速度而激发，速度足以引起相当广泛的屈服。由于系统的简单性，可以容易地得出精确响应，从而为Abaqus中本构模型实现的验证提供基本检验案例。

该问题已被Symonds和Ting（1964）用刚粘塑性材料模型分析。在该论文中，作者提供了确保刚粘塑性分析能够准确预测最终应变的标准。本质上，要求是初始速度足够大，以至于初始动能超过杆中可存储的最大应变能。这里分析的案例所选择的参数轻松满足此标准。

### 问题描述

单自由度模型如图3.1.3-1所示。尺寸为：

| 桁架长度 | 25.4 mm（1 in） |
| --- | --- |
| 桁架横截面积 | 64.52 mm²（0.1 in²） |

桁架假定具有以下材料属性：

| 杨氏模量 | 207 GPa（30×10⁶ lb/in²） |
| --- | --- |
| 静态屈服应力（![](../graphics/bmk_eqn00218.gif)₀） | 276 MPa（40×10³ lb/in²）（无应变硬化） |
| 质量密度 | 7827 kg/m³（7.324×10⁻⁴ lb-s²/in⁴） |
| 粘塑性参数 | ![](../graphics/bmk_eqn01441.gif) = 40/s，![](../graphics/bmk_eqn00392.gif) = 5 |

桁架端部的质量为5.254 kg（0.030 lb-s²/in），初始速度为5.08 m/s（200 in/s）。

桁架用单个桁架单元建模。相同的案例也使用平面应力单元建模作为附加验证练习。集中质量用质量单元建模。平面应力通常是此类塑性模型实现中最困难的情况，因为本构计算必须在具有一个直接应力分量为零约束的多维应力空间中完成。

图3.1.3-2显示数值解准确预测精确解，直到图表非常末尾，此时数值解开始出现小的相位误差。

有趣的是观察解的形式：在第一个半周期中，应力非常快速地增加到约静态屈服的2.38倍，然后卸载，开始缓慢，然后更快。在第一个半周期中，约99%的初始动能消耗在塑性工作中（此结果可通过将总能量内容摘要打印到数据文件获得）。分析的剩余响应（长达5 ms）在每个半周期的峰值应力处继续有一些耗散。由于与杆存储应变能的能力相比系统中剩余的能量非常少，因此每个周期中响应幅值的阻尼不大。Symonds和Ting（1964）的刚粘塑性解因为初始能量与可存储的应变能相比非常大，所以能准确跟随弹性粘塑性曲线。刚粘塑性解估计杆中最终应变为6.75%。数值分析显示在0.7 ms时总塑性应变为6.59%，在每个半周期中峰值应力继续超过静态屈服时此值仍有轻微增加。
