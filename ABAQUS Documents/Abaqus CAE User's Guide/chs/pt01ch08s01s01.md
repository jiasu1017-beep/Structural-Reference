# 8.1.1 打印图像格式





Abaqus/CAE 允许你直接将图像打印到 Windows 打印机。打印机驱动程序以所需格式创建并发送必要的信息到打印机。

如果 Windows 打印机驱动程序不可用，或者你使用其他平台，你可以使用打印命令直接创建并发送 PostScript 文件到 PostScript 打印机。你也可以将图像保存为 PostScript (PS)、封装的 PostScript (EPS)、标记图像文件格式 (TIFF)、便携式网络图形 (PNG) 或可缩放矢量图形 (SVG) 文件。以下列表描述了这些文件格式：

**PostScript**

PostScript 是桌面出版的公认标准。PostScript 实际上是一种编程语言，其指令和数据通常以 ASCII 格式存储，可以轻松地在操作系统之间传输。当你使用打印命令打印到 PostScript 打印机或将图像保存到 PostScript 文件时使用 PostScript 格式。当你选择 PostScript 格式时，Abaqus/CAE 会生成图像的压缩光栅表示或矢量表示。为提高生成光栅图像的效率，你应该最小化图像大小，并将图像分辨率限制在最多图像打印或显示设备的分辨率。

**封装的 PostScript**

封装的 PostScript (EPS) 是 PostScript 的变体，描述了设计为包含在更大文档中而不被修改的单个图形。EPS 文件与 PostScript 文件相同，只是包含一些描述图像大小和定位的信息。因此，上述关于图像矢量和光栅表示的讨论同样适用于 EPS 格式。大多数文字处理和图形应用程序都支持包含 EPS 文件。

**TIFF**

标记图像文件格式 (TIFF) 是一种被许多软件应用程序识别的成熟光栅图像格式。TIFF 格式支持颜色和灰度。默认情况下，Abaqus/CAE 将视口的 TIFF 图像限制为 8 位颜色（256 色）。但是，你也可以使用 24 位颜色（在 Windows 上）或系统颜色设置（在 Linux 上）。

**PNG**

便携式网络图形 (PNG) 是存储光栅图像的行业标准。PNG 文件的使用因万维网而普及，大多数流行的 Web 浏览器都可以在各种操作系统上显示 PNG 图像。PNG 文件由颜色信息和图像的压缩光栅表示组成。默认情况下，Abaqus/CAE 将视口的 PNG 图像限制为 8 位颜色（256 色）。但是，你也可以使用 24 位颜色（在 Windows 上）或系统颜色设置（在 Linux 上）。

**SVG**

可缩放矢量图形 (SVG) 是用 XML 编写的行业标准矢量图形语言。
