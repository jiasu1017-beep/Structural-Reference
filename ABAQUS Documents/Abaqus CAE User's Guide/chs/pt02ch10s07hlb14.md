# 10.7.11 从VDA-FS格式文件导入零件





许多计算机辅助建模和绘图应用程序可以读取和写入VDA-FS格式文件；您可以使用这些文件在这些应用程序与Abaqus/CAE之间交换几何信息。从主菜单栏选择**文件**![](../graphics/images/arrow.gif)**导入**![](../graphics/images/arrow.gif)**零件**，可以从VDA-FS格式文件导入零件。您可以将零件从Abaqus/CAE导出为VDA-FS格式文件，但不能将装配体导出为VDA-FS格式文件。

如果VDA-FS格式文件包含多个零件，Abaqus/CAE会将它们作为单个零件导入。导入的VDA-FS零件在Abaqus/CAE中形成新零件的基础特征；您不能直接修改此基础特征，但可以向其添加其他特征，例如实体拉伸或盲孔切割。

**从VDA-FS格式文件导入零件：**

1. 从主菜单栏，选择**文件**![](../graphics/images/arrow.gif)**导入**![](../graphics/images/arrow.gif)**零件**。出现**导入零件**对话框。
2. 从**导入零件**对话框底部的**文件过滤器**菜单中，选择**VDA-FS (*.vda*)**。Abaqus/CAE列出所选目录中所有具有`.vda*`文件扩展名的文件。
3. 选择包含要导入的零件的VDA-FS文件，然后单击**确定**。Abaqus/CAE显示**从VDA文件创建零件**对话框。
4. 单击**名称-修复**选项卡设置以下内容：-新零件的名称。-Abaqus/CAE将在导入过程中应用的修复选项。更多信息，请参见"控制导入过程"，第10.3节。
5. 单击**零件属性**选项卡设置以下内容：-正在导入的零件的建模空间。导入零件时，Abaqus/CAE会扫描文件并尝试确定建模空间。-正在导入的零件的类型。Abaqus/CAE始终假定零件类型为可变形。或者，您可以选择**离散刚体**导入离散刚体零件，或选择**欧拉**导入欧拉零件。更多信息，请参见"零件类型"，第11.4.2节。在大多数情况下您应该接受默认设置；更多信息，请参见"什么是零件属性？"，第10.3.2节。
6. 单击**缩放**选项卡以在导入过程中缩放零件。-选择**不缩放**以保持文件中存储的尺寸。-选择**使用文件中的变换，包括缩放**以从VDA-FS文件读取缩放因子、旋转矩阵和平移矩阵。-选择**将所有长度乘以**，然后输入缩放因子。Abaqus/CAE将缩放因子应用于文件中的所有坐标。因此，距原点的任何偏移都将按比例缩放。
7. 单击**确定**导入VDA-FS零件并关闭**从VDA文件创建零件**对话框。Abaqus/CAE扫描VDA-FS文件并根据所选选项启动修复过程。如果希望取消导入过程，请单击提示区域中的**停止**。导入零件后，Abaqus/CAE会显示一条消息，指示零件是否存在任何有效性或精度问题。如有必要，使用查询工具集定位无效和不精确几何的区域。更多信息，请参见"在零件模块中使用查询工具集"，第11.16.4节。
