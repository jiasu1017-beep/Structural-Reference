# 32.4.2 分布耦合单元库





**产品：** Abaqus/Standard  

##### **参考文献**

- ["分布耦合单元，" 第32.4.1节](pt06ch32s04alm40.md)
- [*DISTRIBUTING COUPLING](../key/key-link.md#usb-kws-mdistcoupling)

### 概述

本节提供Abaqus/Standard中可用分布耦合单元的参考。

### 单元类型

| DCOUP2D | 二维分布耦合单元 |
| --- | --- |
|  |

| DCOUP3D | 三维分布耦合单元 |
| --- | --- |
|  |

##### 活动自由度

DCOUP2D：1、2、6

DCOUP3D：1、2、3、4、5、6

##### 附加解变量

无。

### 所需节点坐标

DCOUP2D：*X*、*Y*

DCOUP3D：*X*、*Y*、*Z*

### 单元属性定义

您必须识别分布耦合单元分配载荷和质量的至少两个节点；此外，您可以指定单元质量。

| **输入文件用法：** | ``` [*DISTRIBUTING COUPLING](../key/key-link.md#usb-kws-mdistcoupling) ``` |
| --- | --- |

### 基于单元的加载

无。

### 单元输出

| ELKE | 单元动能。 |
| --- | --- |

| NFORC | 单元节点力。 |
| --- | --- |

### 与单元关联的节点

使用元素定义1个节点。形成耦合的附加节点在单元属性定义中定义。
