# 34.4 Viewing load case output







Load case output is stored in separate frames in the output database (`.odb`). You use the Visualization module to view and manipulate load case output. The load case name is displayed in the state block for each frame containing load case output. For example, [Figure 34--5](pt04ch34s04.md#usi-adv-lc-odb) shows the deformed plot for the `Pressure Load w End Load Up` load case of the cantilever beam model (see ["Load case editors," Section 34.3](pt04ch34s03.md)).

**Figure 34–5** Results from a load case analysis.

![](../graphics/adv-lc-odb.png)

In the Visualization module you can linearly combine output from several load cases to represent an actual loading environment. You can also obtain the minimum or maximum value of a selected field variable over some or all load cases.

For example, you can combine the output from the `End Load Only` and `Pressure Load Only` load cases for the cantilever beam model to create new field output with a load case name of `Both loads`, as shown in [Figure 34--6](pt04ch34s04.md#usi-adv-createoutput). 

**Figure 34–6** Creating new field output by combining the output from two load cases (frames).

![](../graphics/adv-odb-createoutputdb-nls.png)

The new field output is contained in a frame of the session step and is available from the **Field Output** dialog box. [Figure 34--7](pt04ch34s04.md#usi-adv-lc-odbboth) shows a plot of the combined deformation results.

**Figure 34–7** Combined deformation results from both load cases.

![](../graphics/adv-lc-odbboth.png)

For detailed information on manipulating load case output, see ["Combining results from several frames," Section 42.7.3](pt05ch42s05hlb03.md).
