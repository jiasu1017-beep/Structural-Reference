# 18.2 进入和退出Optimization模块





您可以在Abaqus/CAE会话的任何时候通过单击上下文栏**Module**列表中的**Optimization**来进入Optimization模块。 **Task**、**Design Response**、**Objective Function**、**Constraint**、**Geometric Restriction**、**Stop Condition**和**Tools**菜单出现在主菜单栏上。 如果当前视口包含除装配以外的内容，当您启动Step模块时，视口的内容会消失。

要退出Optimization模块，请从**Module**列表中选择任何其他模块。 退出模块前无需保存您的优化定义； 当您通过从主菜单栏选择 ****File**![](../graphics/images/arrow.gif)**Save**** 或 ****File**![](../graphics/images/arrow.gif)**Save As**** 保存模型数据库时，它会自动保存。
