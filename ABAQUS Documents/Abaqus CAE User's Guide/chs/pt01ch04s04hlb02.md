# 4.4.3 创建文本注释





您可以创建一行或多行文本来注释视口的内容，并将其放置在视口上的任何位置。通常，您可以使用[组合箭头和文本注释](pt01ch04s04hlb03.md)将文本注释连接到视口中的对象，但您也可以添加文本注释来提供标题或其他适用于整个模型或视口中结果的常规信息。

**要创建文本注释：**

1. 从主菜单栏，选择****视口（Viewport）**![](../graphics/images/arrow.gif)**创建注释（Create Annotation）**![](../graphics/images/arrow.gif)**文本（Text）****。**提示：**您也可以通过点击[**视口**工具栏](pt01ch04s02hlb02.md)中的![](../graphics/ico_annotationText.png)来创建文本注释。锚点符号会出现在视口中，注释为`Text-*n*`。默认锚点在您会话中创建新注释时会更改。如果是第一个文本注释，锚点位于视口的左下角。
2. 如果需要，在提示区域单击**更改锚点（Change Anchor）**，然后执行以下操作之一：- 在视口中选择一个新锚点。- 从提示区域中的列表中选择视口角或中心点。- 在提示区域单击**从模型中拾取（Pick From Model）**，然后选择节点或顶点，或输入所需锚点的坐标。**注意：**如果您将注释锚定到节点或顶点，Abaqus/CAE使用所选点的模型坐标；注释不会在可视化模块中跟随模型变形。
3. 选择以下选项之一来在视口中定位文本注释：- 要将文本定位在当前锚点上，请在提示区域单击**在锚点上（On Anchor）**。- 要将文本放置在视口上的任何位置，请将光标移动到所需位置，然后单击鼠标按钮1。Abaqus/CAE会打开**文本编辑器（Text Editor）**对话框。
4. 完成**文本编辑器**对话框中的信息以最终确定文本注释。有关更多信息，请参阅["编辑文本注释属性，"第4.4.7节"](pt01ch04s04hlb06.md)。
5. 单击**预览（Preview）**以在视口中查看您的更改。
6. 要完成文本注释的创建，请单击**确定（OK）**。Abaqus/CAE会在视口中更新您的更改，关闭编辑器，并从步骤2开始创建另一个文本注释。
7. 要停止创建文本注释，请在视口中单击鼠标按钮2或单击提示区域中的![](../graphics/afxI_cancel.png)按钮。
