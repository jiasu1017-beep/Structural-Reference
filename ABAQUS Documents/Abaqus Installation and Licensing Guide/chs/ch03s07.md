# 3.7 使用 FLEXnet 选项文件



选项文件允许许可证管理员控制 FLEXnet Licensing 的各种操作参数。可以使用 `EXCLUDE`、`INCLUDE`、`MAX` 和 `RESERVE` 等参数来控制 Abaqus 许可证使用情况。使用情况可以通过用户名、主机名、显示、IP 地址或项目来控制。在指定要控制的功能时，您可以指定许可证文件中 `FEATURE` 行上列出的功能。使用 `PACKAGE` 名称代替 `FEATURE` 名称会将选项应用于包中的所有组件。您不应指定 `PACKAGE` 行上列出的单个 `COMPONENTS`，因为它可能导致意外行为。请参阅 FLEXnet Licensing End User Guide Version 11.6 第 5 章，了解有关创建选项文件的详细信息。（此指南可从 **Support** 页面上的 **Licensing** 部分下载，网址为 [www.3ds.com/simulia](http://www.3ds.com/simulia)。）

以下是示例选项文件：

```
INCLUDEALL INTERNET 192.168.0.*
RESERVE 5 abaqus USER robert
RESERVE 10 abaqus HOST demeter
MAX 5 abaqus USER jim
EXCLUDE abaqus USER khan
EXCLUDE viewer HOST guardian
```
在此选项文件中：- 仅接受来自 IP 地址为 192.168.0.* 的计算机的连接。
- 为用户 *robert* 保留五个功能 `abaqus` 的令牌。
- 为主机 *demeter* 上的用户保留十个功能 `abaqus` 的令牌。
- 用户 *jim* 一次限制使用五个功能 `abaqus` 的令牌。
- 拒绝用户 *khan* 访问功能 `abaqus`。
- 拒绝主机 *guardian* 上的用户访问功能 `viewer` (Abaqus/Viewer)。
