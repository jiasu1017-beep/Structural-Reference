# 4.7.2 测试1.2：带热载荷的中心裂纹板





**产品：** Abaqus/Standard  

### 测试单元

CPS8    CPS8R    

### 问题描述

![](../graphics/bmknlf12.png)

**网格：**

在裂纹尖端使用带有1/4点中间节点的收缩单元。建模了测试几何体的四分之一。

**材料：**

弹性模量 = 207 GPa，泊松比 = 0.3，热膨胀系数 = 1.35×10⁻⁵。

**边界条件：**

沿AB边施加![](../graphics/bmk_eqn01322.gif)，沿DE边施加![](../graphics/bmk_eqn01321.gif)。

**载荷：**

二次热分布 T = T0![](../graphics/bmk_eqn01762.gif) = 0.01×x²，其中 T0 = 100，c = 100。

### 参考解

这是英国国家有限元方法与标准机构（NAFEMS）推荐的测试：NAFEMS出版物"2D Test Cases in Linear Elastic Fracture Mechanics"，R0020中的测试1.2。

目标解：K![](../graphics/bmk_eqn01757.gif)/K![](../graphics/bmk_eqn01758.gif) = 1.000，K![](../graphics/bmk_eqn01758.gif) = ![](../graphics/bmk_eqn01763.gif)

### 备注

K![](../graphics/bmk_eqn01757.gif) = ![](../graphics/bmk_eqn01764.gif)。报告中使用了Abaqus计算的J值的平均值，不包括第一个轮廓。经验表明，裂纹尖端单元没有给出足够准确的结果，无法为第一个轮廓的J积分提供良好的估计。热载荷通过用户子程序[`UTEMP`](../sub/sub-link.md#sub-xsl-utemp)施加。
