# 1.7.3 耦合热-电表面相互作用







**产品：**Abaqus/Standard

### 问题描述

固体材料放置在温度和电势保持恒定的热源附近。跨固体和热源之间间隙的热传递可以通过间隙传导率或间隙辐射进行（因此，每种单元类型有两个测试）。电流在形成间隙的两个紧密相邻表面之间传导。这种传导产生的电能有二分之一以热的形式释放，并平均分配到两个相邻表面。由于电传导，模型中不会发生焦耳热；因此，电能不在连续体单元内作为内部热源。为简单起见，我们通过向远离热源的表面施加更高的温度和电势来启动固体中的单向热流和电流。近热源固体表面的稳态温度和电势通过分析解进行验证。
