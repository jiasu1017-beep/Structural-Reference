# 3.8.5 轴对称连续体应力/位移子模型





**产品：**Abaqus/Standard  Abaqus/Explicit  

### 测试的单元

CAX3    CAX4I    CAX4R    CAX6    CAX6M    CAX8H    CAX8R    

### 测试的功能

子模型功能应用于轴对称应力/位移单元。子模型沿与全局模型规则几何成对角线的切割线切割全局模型。在Abaqus/Standard中测试静态和稳态动态程序。在Abaqus/Explicit中，分析是准静态的；每个子模型可以从任何全局模型驱动。

### 问题描述

**模型：**

全局模型在r-z平面中的尺寸为8.0×1.5。

**材料：**

| 杨氏模量 | 30×10^6 |
| --- | --- |
| 泊松比 | 0.3 |
| 密度 | 10.0 |

**边界条件：**

沿着模型的左面，![](../graphics/ver_eqn00099.gif)=0。一个节点进一步受到约束，使![](../graphics/ver_eqn00123.gif)=0。

**载荷：**

在所有模型中，1.0×10^6的压力施加在右侧面。（具有相同几何和载荷的图形显示在["二维连续体应力/位移子模型，"第3.8.2节](ch03s08abv205.md)中，只是图形使用矩形轴x和y而不是轴对称轴r和z。）

### 结果与讨论

在所有情况下，子模型中驱动节点的节点位移都从全局模型结果正确插值。在所示情况下，子模型中的单元和节点响应与全局模型中的响应匹配。
