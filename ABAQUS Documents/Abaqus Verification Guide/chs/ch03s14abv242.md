### 问题描述

本节概述的验证测试针对所列出的每种单元类型进行。有限元模型由两个承受递增拉伸载荷的单元组成。第一个分析包括两个静态步骤。在第一个步骤中，一个单元承受施加到另一个单元的载荷的一半。在第二个步骤中，两个单元承受相同的最终载荷。将此分析第一个步骤结束时的结果传输到第二个分析，在该分析中对两个单元施加与第一个分析第二个步骤中相同的载荷。针对重置参考构型为导入构型选项和导入特征的材料状态选项的所有组合执行测试。当材料状态被导入且参考构型未被更新为导入构型时，第二个分析结束时的结果应与第一个分析结束时的结果相同。

包括使用增强沙漏控制的一阶减缩积分单元对两个Abaqus/Standard分析进行验证测试，以测试沙漏力的导入。

对于除具有牵引分离的内聚单元之外的所有测试，材料模型使用各向同性线弹性，以及Mises塑性。使用的材料属性如下：

| 弹性模量 = 200 109 |
| --- |
| 泊松比 = 0.3 |
| 密度 = 7800. |
| 屈服应力 = 300 106 |

对于使用牵引分离的内聚单元的测试，材料模型使用非耦合牵引的线弹性，以及损伤模型。

### 结果和讨论

当材料状态被导入且参考构型未被更新时，导入分析的结果与第一个分析结束时的结果相同。在所有材料状态被导入的情况下，应力、弹性应变和等效塑性应变在从第一个分析到第二个分析传输过程中是连续的。当参考构型未被更新时，位移、应变和诸如可恢复应变能的能量量在两个分析之间是连续的。在第二个Abaqus/Standard分析开始时，如果参考构型被更新为导入构型，则位移和应变从零开始；如果材料状态未被导入，则弹性应变、应力和等效塑性应变被设置为零。

### 问题描述

本节概述的验证测试针对所列出的每种单元类型进行。有限元模型由两个承受递增拉伸载荷的单元组成。第一个分析包括两个动态步骤。在第一个步骤中，一个单元承受施加到另一个单元的载荷的一半。在第二个步骤中，两个单元承受相同的最终载荷。将此分析第一个步骤结束时的结果传输到第二个分析，在该分析中对两个单元施加与第一个分析第二个步骤中相同的载荷。针对重置参考构型为导入构型选项和导入特征的材料状态选项的所有组合执行测试。当材料状态被导入且参考构型未被更新为导入构型时，第二个分析结束时的结果应与第一个分析结束时的结果相同。

包括使用增强沙漏控制的一阶减缩积分单元对两个Abaqus/Standard分析进行验证测试，以测试沙漏力的导入。

对于除具有牵引分离的内聚单元之外的所有测试，材料模型使用各向同性线弹性，以及Mises塑性。使用的材料属性如下：

| 弹性模量 = 200 109 |
| --- |
| 泊松比 = 0.3 |
| 密度 = 7800. |
| 屈服应力 = 300 106 |

对于使用牵引分离的内聚单元的测试，材料模型使用非耦合牵引的线弹性，以及损伤模型。

### 结果和讨论

当材料状态被导入且参考构型未被更新时，导入分析的结果与第一个分析结束时的结果相同。在所有材料状态被导入的情况下，应力、弹性应变和等效塑性应变在从第一个分析到第二个分析传输过程中是连续的。当参考构型未被更新时，位移、应变和诸如可恢复应变能的能量量在两个分析之间是连续的。在第二个Abaqus/Standard分析开始时，如果参考构型被更新为导入构型，则位移和应变从零开始；如果材料状态未被导入，则弹性应变、应力和等效塑性应变被设置为零。

| 弹性模量 = 200 109 |
| --- |
| 泊松比 = 0.3 |
| 密度 = 7800. |
| 屈服应力 = 300 106 |
