# 79.1 理解如何叠加绘图





您可以创建在单个视口中包含多个绘图的显示。例如，您可能需要执行以下任一操作：
- 合并轮廓图和X-Y图
- 在同一视口中比较来自两个不同输出数据库文件的变形绘图形状
- 将时间历史动画与显示模型中多个变量随时间变化的动画X-Y图合并

叠加绘图非常有用，例如，用于在同一视口中显示协同仿真中两个输出数据库的数据。

叠加绘图由层组成；每层包含一个绘图，各层堆叠在一起创建组合绘图。[图79-1](pt07ch79s01.md#uss-ovr-layerplot)显示了一个绘图，其中包含分析四个不同增量的变形形状绘图以及模型应变能随时间变化的X-Y图。

**图79-1** 叠加绘图。

![](../graphics/uss-ovr-layerplot.png)

默认情况下，视口不包含任何层；一次只显示一个绘图。要叠加多个绘图，您需要在与Abaqus/CAE交互时为每个单独绘图创建一个层。然后，您可以选择在当前视口中显示的层。您可以创建任意数量的层；任意数量的层可以在同一视口中显示。此外，您可以打开多个输出数据库并自动在单个视口中的叠加绘图中显示组合内容。

使用**叠加绘图层管理器**创建、显示、定位和删除层。要访问管理器，请从主菜单栏中选择****视图**![](../graphics/images/arrow.gif)**叠加绘图****或点击工具箱中的**叠加绘图层管理器**![](../graphics/ico_manager.png)工具。

创建层时，它包含当前视口中所有可见的内容。您可以更改层的内容，操作其视图，根据叠加绘图中其他层重新排序层，以及更改变应用于内容的各种显示选项。默认情况下，层直接叠加在一起。有时直接叠加的线条会产生不良的视觉效果。您可以相对于彼此偏移层以避免此类显示异常。

**叠加绘图层管理器**中的列显示每个层的以下信息：

**可见**

此列中的复选标记表示当您处于叠加绘图状态时该层在视口中可见。

**当前**

此列中的复选标记表示该层是当前层。一次只能有一个层是当前层，尽管每个层可以包含多个绘图状态（有关更多信息，请参见["显示多个绘图状态，"第55.6节](pt05ch55s06.md)）。当您处于叠加绘图状态时，绘图选项仅应用于当前层；您可以选择视图操作选项是应用于所有现有层还是仅应用于当前层。当前层不一定是视口中最前面的层。

**名称**

层的名称。

**对象**

层包含的对象的名称；例如，输出数据库或X-Y绘图。
