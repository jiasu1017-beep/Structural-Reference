# 17.11.4 为自底向上网格选择参数





您用于创建自底向上网格的参数与Abaqus/CAE用于创建类似自顶向下扫掠网格的参数相似。在自顶向下网格方法中，Abaqus/CAE自动为区域选择所需的几何边。自顶向下扫掠网格（包括挤出和旋转网格）在["三维实体的扫掠网格，"第17.9.3节](pt03ch17s09s03.md)中讨论。自底向上网格方法中使用的参数如下所述。

**源侧**

**源侧**是必需参数，定义Abaqus/CAE将创建二维网格的面或面的组合，该网格将被扫掠、挤出或旋转以创建三维网格。源侧可以是几何面、单元面和二维单元的组合；您也可以选择保存的面而不是从视口中选择。对于自顶向下网格，Abaqus/CAE限制其选择作为源侧的多个面之间的角度（更多信息，请参见["几何特征可能阻止零件进行扫掠网格划分，"第17.9.5节](pt03ch17s09s05.md)）。对于自底向上网格的源侧，您选择的面之间的角度没有限制。但是，源侧面之间或源侧与目标侧之间的角度增大会导致三维单元质量下降。

**连接边**

**连接边**定义扫掠区域的方向。连接边可以是几何面、单元面和二维单元的组合。每个连接边必须只有一个四边形几何面，或由形成规则网格图案的四边形组合几何面、单元面和二维单元组成。

在自顶向下扫掠网格中，Abaqus/CAE沿源侧的所有边创建连接边。连接边从源侧完全延伸到目标侧。在自底向上扫掠网格中，连接边对于几何体是可选的，但建议您尽可能多地包含连接边。连接边有助于控制网格，因为它通过三维区域被扫掠，并强制结果网格的[关联性](pt03ch17s12.md)。包含连接边可以减少清理和将自底向上扫掠网格与几何体关联所需的工作量。自底向上网格的连接边可以从源侧完全延伸到目标侧，也可以仅覆盖部分距离。它们也可以在没有目标侧的情况下使用，在这种情况下，连接边的末端定义自底向上扫掠网格的末端。更多信息，请参见["为自底向上扫掠网格定义连接边，"第17.11.7节](pt03ch17s11s07.md)。

**目标侧**

**目标侧**定义扫掠网格或挤出网格的末端。您不能为旋转或偏移自底向上网格选择目标侧。扫掠网格的目标侧必须是单个几何面；挤出网格的目标侧可以是一个或多个几何面、一组单元面或基准平面。目标侧对于扫掠网格不是必需的，除非您没有包含任何连接边。但是，包含目标侧有助于您控制网格，并且如果适用的话，强制网格与几何体的[关联性](pt03ch17s12.md)。如果未使用连接边，Abaqus/CAE通过从源侧投影节点来为目标侧创建网格；如果使用了连接边，则从最后一层节点投影。投影的节点可能定位在所选几何目标侧的外部。对于挤出网格，目标侧是可选的，仅用于确定挤出距离；为目标侧选择网格面不会改变挤出网格以符合现有节点和单元面。

**层数**

**层数**参数定义Abaqus/CAE沿扫掠、挤出或旋转方向生成的单元层数。如果您选择自底向上扫掠方法并选择连接边，则单元层数由连接边的网格或布种驱动，类似于自顶向下网格技术。如果您没有为扫掠方法选择连接边，或者如果您选择挤出或旋转方法，则必须指定Abaqus/CAE应生成的单元层数。

**向量**

**向量**参数为挤出网格定义挤出方向和可选的距离。您通过从视口中的节点、顶点、基准点和有意义的点中选择起点和终点来选择挤出向量。Abaqus/CAE沿向量的方向和距离（如果适用）从源侧挤出网格，无论您在视口中的何处定义向量。

**轴线**

**轴线**参数为旋转网格定义旋转轴。您可以通过从视口中选择两个顶点、节点、基准点或有趣的点来定义旋转轴。

**角度**

**角度**参数为旋转网格定义旋转角度。您必须为自底向上旋转技术指定旋转角度，以指示Abaqus/CAE应在何处结束旋转网格。

**偏差比**

**偏差比**参数定义在创建多层挤出自底向上网格中，源侧与挤出网格末端之间单元厚度的变化。偏差比是挤出网格中第一层单元厚度与最后一层单元厚度的比值。小于一的偏差比在源侧附近创建更薄的层，等于一正好创建相等厚度的层，大于一在源侧附近创建更厚的层。Abaqus/CAE在挤出层数中均匀分布偏差。

当您选择参数来定义自底向上网格时，不仅要考虑当前自底向上网格的形状，还要考虑所需的最终网格形状。许多复杂的零件将需要多次自底向上网格迭代来生成完整网格。例如，您可以使用自底向上扫掠网格的连接边或单元面作为自底向上挤出网格的源侧。如果您无法在当前迭代中完全网格化选定区域，请考虑如何添加另一个自底向上网格或使用"编辑网格"工具集来完成网格。有关将自顶向下和自底向上技术结合使用对零件进行网格划分的详细分步示例，请参见["包含自底向上网格技术的示例，"第17.11.9节](pt03ch17s11s08.md)。
