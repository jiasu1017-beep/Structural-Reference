# 8.3 注册GUI模块

为了使GUI模块对GUI基础设施可见，你必须在主窗口代码中注册该模块。注册命令接受两个参数：一个用于在CAE的"模块"组合框中显示的名称，第二个指定要导入的模块的名称。有关更多信息，请参阅["注册模块，" 第14.2.5节](pt06ch14s02.md#cus-app-custom-main-reg-modules)。在大多数情况下，你通过在模块GUI文件末尾调用模块构造函数来注册模块。

如果["GUI模块示例，" 第8.2节](pt05ch08s02.md)所示的示例位于名为`myModuleGui.py`的文件中，则必须使用`myModuleGui`作为`registerModule`方法的第二个参数，如下语句所示：

```
# 注册模块
#
self.registerModule(
    displayedName='My Module', moduleImportName='myModuleGui')
```
