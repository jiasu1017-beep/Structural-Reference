# 55.8 DefaultPlot object







The DefaultPlot object is used to hold on default plot attributes. The DefaultPlot object attributes are used whenever an [XYPlot](pt01ch55pyo16.md) object is created. A DefaultPlot object is automatically created when opening a session.

**Access**

```
import visualization
session.defaultPlot
```

### 55.8.1 Members

The DefaultPlot object can have the following members:

*area*

An [Area](pt01ch55pyo02.md) object specifying an [Area](pt01ch55pyo02.md) used to hold on to the default display properties for the plot area.

*title*

A [Title](pt01ch55pyo14.md) object specifying a [Title](pt01ch55pyo14.md) object used to hold on to the default properties of the XY-Plot title.
