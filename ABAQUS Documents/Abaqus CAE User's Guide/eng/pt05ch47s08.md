# 47.9 Customizing the appearance of the X–Y plot legend







### 47.9 Customizing the appearance of the X--Y plot legend

This section explains how you use the **Chart Legend Options** dialog box to customize the appearance and contents of the legend for an *X–Y* plot. You can also display a title on the *X–Y* plot legend, and you can include on the legend a report of the minimum and maximum *X–Y* values. The following topics are covered:

- ["Adding minimum and maximum values to the X--Y plot legend," Section 47.9.3](pt05ch47s08hlb03.md)
- ["Customizing the location of the legend," Section 47.9.4](pt05ch47s08hlb04.md)
- ["Customizing the border and fill for the X--Y plot legend," Section 47.9.5](pt05ch47s08hlb05.md)
