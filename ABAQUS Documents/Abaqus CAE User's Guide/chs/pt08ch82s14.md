# 82.14 找到离点最近的节点





**Find Nearest Node** 插件使您可以定位网格化模型或未变形绘图中离给定店最近的节点。您输入任何点的 x、y 和 z 坐标，Abaqus/CAE 会在网格化模型或未变形绘图中向您显示最近的节点。此外，您可以选择将搜索限制在模型的特定区域，这在大型模型中很有用。

此插件可用于模型数据库（`.cae`）文件或输出数据库（`.odb`）文件的未变形绘图。它不能用于变形绘图。

**找到离指定点最近的节点：**

1. 在 Mesh 模块或 Visualization 模块中，从主菜单栏选择 ****Plug-ins**![](../graphics/images/arrow.gif)**Tools**![](../graphics/images/arrow.gif)**Find Nearest Node****。出现 **Find Nearest Node** 对话框。
2. 输入所需点的 x、y 和 z 坐标，然后点击 **Find**。Abaqus/CAE 在视口中添加一个注释，箭头指向网格中最近的节点。节点编号和坐标也显示在窗口底部的消息区域中；例如：
```
Node 15 of Instance Hinge-solid-1 is nearest to the defined point (1,1,1)
Coordinates of node are (0.104999999665,0.999999977648,0.04), distance from defined point is 1.64399665481
```
3. 如果您有一个大型模型，并希望将搜索限制在较小的区域，请点击 **Select Nodes for Reduced Search**。按照提示在视口中选择节点的任意子集，可以单独选择、按角度选择或按特征边选择。说明请参见 ["在当前视口中选择对象，" 第6.2节](pt01ch06s02.md)。有关按角度或特征边选择的详细信息，请参见 ["使用角度和特征边方法选择多个对象，" 第6.2.3节](pt01ch06s02hlb03.md)。完成节点选择后点击 **Complete**。
**注意：**如果装配中有多个零件实例，您只能在一个实例上选择节点进行缩小搜索。
