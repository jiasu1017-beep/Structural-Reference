# 73.3.4 对集或曲面执行布尔运算







您可以从模型树中选择一组集或曲面，并使用布尔运算符将它们组合成单个集或曲面。以下运算符可用：

**并集**

使用**Union**运算符创建合并您选择内容的集或曲面。

**交集**

使用**Intersection**运算符创建包含您所有选择中共有项目的集或曲面。

**差集**

使用**Difference**运算符通过减法创建集或曲面。您必须指定"第一个"集或曲面，Abaqus/CAE从中减去剩余的选择。

例如，您可以使用**Union**运算符将线性图案中所有部件实例的顶面组合成一个曲面。

**对集或曲面执行布尔运算的步骤：**

1. 从模型树中，展开与部件或装配体关联的**Sets**或**Surfaces**容器。
2. 选择所有所需的集或曲面。您可以使用拖动选择、**[Shift]**+**单击**和**[Ctrl]**+**单击**的组合来选择所需的集或曲面。您不能将集与曲面组合。更多信息，请参见["拖动选择多个对象," 第6.2.2节](pt01ch06s02hlb02.md)。
3. 在模型树中，在所选内容上单击鼠标按钮3，然后从出现的菜单中选择**Boolean**。Abaqus/CAE显示"集布尔"或"曲面布尔"对话框。
4. 在对话框中，输入新集或曲面的名称并选择所需的运算符。
5. 如果您选择了**Difference**运算符，请单击**First set**或**First surface**字段右侧的箭头，并指定Abaqus/CAE将从中减去剩余选择的集或曲面。
6. 单击**OK**。Abaqus/CAE创建新的集或曲面，它会出现在模型树中。
