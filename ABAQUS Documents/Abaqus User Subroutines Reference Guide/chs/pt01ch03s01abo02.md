# 3.1.1 Abaqus执行过程：概述





### 概述

Abaqus通过使用Abaqus执行过程执行。在以下讨论中，运行执行过程的命令假定为**abaqus**。但是，您可以自定义执行过程以使用您选择的任何别名运行Abaqus。（详细信息请参阅[Abaqus安装和许可指南](../sgb/sgb-link.md#sgb)。）

- ["在Abaqus中管理内存和磁盘使用，" 第3.4.1节](pt01ch03s04aus31.md)
- ["并行执行，" 第3.5节](pt01ch03s05.md)

### 约定

以下约定在这些部分中使用：
- 每个讨论都包含一个"命令摘要"部分，在左列提供命令的语法，在右列提供其选项的语法。完整命令必须首先出现，然后是选项。在某些情况下，命令有多个单词，例如**abaqus cae**；您必须在发出任何选项语句之前输入命令的所有单词。
- 选项以**粗体**显示。它们可以以任何顺序出现，可以缩写。
- 默认选项带下划线（__）。
- 方括号（[ ]）括起来的项目是可选的。
- 用竖线（|）分隔的列表中出现的项目是互斥的。
- 必须从括在花括号（{ }）中的值列表中选择一个值。
- 您必须提供*斜体*中的值。
- 空格用作选项之间的分隔符，不得在等号之前或之后出现。
- 可以使用**-option** *value*的替代语法而不是**option**=*value*格式。

如果命令行未提供所需信息，**abaqus**过程将提示输入任何所需信息。如果输入**abaqus**时没有选项，则会为所有选项发出提示。

### 环境设置

Abaqus执行过程使用"环境"设置来自定义作业的执行。这些设置可以使用Abaqus环境文件`abaqus_v6.env`更改。执行过程在运行作业时在安装位置之外的两个地方查找此文件。首先在主目录中查找。如果存在，该文件中的设置将应用于您运行的所有作业。执行过程查找的第二个地方是当前目录。如果文件存在，则其中定义的设置将应用于从该目录运行的所有作业。

如果同一作业参数在多个环境文件中定义，或在同一环境文件中定义多次，则将使用最后遇到的定义。此规则的某些例外在["使用Abaqus环境设置，" 第3.3.1节](pt01ch03s03aus30.md)中注明。这些环境文件可用于自定义Abaqus的行为，包括修改默认选项。有关环境文件的更多信息，请参阅["使用Abaqus环境设置，" 第3.3.1节](pt01ch03s03aus30.md)。

### 选择TCP/UDP端口号

多个执行过程命令行选项（如**port**和**listenerport**）要求您指定端口号。TCP/UDP端口号范围可以从0到65535。

端口0到1023是由系统进程（如FTP、SSH、SMTP等）使用的知名端口，不应使用。端口1024到49151是由软件供应商在互联网分配号码 Authority（IANA）注册的注册端口。这些端口可以使用，但您应小心不要与系统中可能正在使用此端口的任何软件冲突。端口49152到65535是未预留的，只要没有其他应用程序使用它们，就可以自由使用。

端口可能被防火墙阻止。请联系您的系统管理员以确保您要指定的端口未被阻止。

您可以使用`netstat`命令获取有关TCP/UDP网络连接的信息。
