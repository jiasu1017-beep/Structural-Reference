# 62.5.3 创建datum平面的方法概述







当您从**Create Datum**对话框中选择**Plane**时，**Method**列表显示以下创建datum平面的方法：

**![](../graphics/ico_dtmPlaneOffsetPrincipal.png) 从主平面偏移**

选择三个主平面之一；以相对于选定平面的偏移形式提供datum平面的位置，如[图62--22](pt06ch62s05s03.md#dtm-hpopp)所示。正值表示沿垂直于选定平面的轴的正方向偏移；例如，沿垂直于*Y*–*Z*平面的*X*轴方向偏移。有关详细说明，请参见["从主平面偏移创建datum平面，" 第62.8.1节](pt06ch62s08hlb01.md)。

**图62–22** 通过从三个主平面之一偏移创建datum平面。

![](../graphics/dtm-hpopp-nls.png)

**![](../graphics/ico_dtmPlaneOffsetPlane.png) 从平面偏移**

在模型上选择任何平面；通过指定法线方向和沿法线从选定平面的偏移来提供datum平面的位置，如[图62--23](pt06ch62s05s03.md#dtm-hpop)所示。您可以通过输入值或选择一个点来指定偏移。有关详细说明，请参见["在距选定平面的偏移处创建datum平面，" 第62.8.2节](pt06ch62s08hlb02.md)。

**图62–23** 通过从任意平面偏移创建datum平面。

![](../graphics/dtm-hpop-nls.png)

**![](../graphics/ico_dtmPlaneThreePoints.png) 三点**

选择datum平面必须通过的三个点，如[图62--24](pt06ch62s05s03.md#dtm-hp3pts)所示。有关详细说明，请参见["通过三点创建datum平面，" 第62.8.3节](pt06ch62s08hlb03.md)。

**图62–24** 通过选择三点创建datum平面。

![](../graphics/dtm-hp3pts-nls.png)

**![](../graphics/ico_dtmPlaneLinePoint.png) 线和点**

选择一条边和datum平面必须通过的一个点，如[图62--25](pt06ch62s05s03.md#dtm-hplp)所示。有关详细说明，请参见["通过一条线和一个点创建datum平面，" 第62.8.4节](pt06ch62s08hlb04.md)。

**图62–25** 通过选择一个点和一个边创建datum平面。

![](../graphics/dtm-hplp-nls.png)

**![](../graphics/ico_dtmPlanePointNormal.png) 点和法线**

选择一个点和一个边；datum平面通过该点并垂直于选定的边，如[图62--26](pt06ch62s05s03.md#dtm-hppn)所示。有关详细说明，请参见["通过一点并垂直于边创建datum平面，" 第62.8.5节](pt06ch62s08hlb05.md)。

**图62–26** 通过选择一个点和一个法向边创建datum平面。

![](../graphics/dtm-hppn-nls.png)

**![](../graphics/ico_dtmPlaneMidpoint.png) 两点之间的中点**

选择两点。Abaqus/CAE在两点之间的中点处创建一个datum平面，并垂直于连接两点的直线，如[图62--27](pt06ch62s05s03.md#dtm-hpmid2)所示。有关详细说明，请参见["在两点之间的中点处创建datum平面，并垂直于连接两点的直线，" 第62.8.6节](pt06ch62s08hlb06.md)。

**图62–27** 在两点之间的中点处定位datum平面。

![](../graphics/dtm-hpmid2-nls.png)

**![](../graphics/ico_dtmPlaneRotatePlane.png) 从平面旋转**

选择一个面和一个旋转轴，并指定面将旋转的角度。Abaqus/CAE通过将面绕轴旋转指定角度来创建datum平面，如[图62--28](pt06ch62s05s03.md#dtm-hprot)所示。有关详细说明，请参见["通过将现有面旋转指定角度创建datum平面，" 第62.8.7节](pt06ch62s08hlb07.md)。

**图62–28** 通过将面旋转指定角度定义datum平面。

![](../graphics/dtm-hprot-nls.png)
