# 76.16 保存显示选项设置




如果更改了显示选项设置（例如，如果您更改了渲染样式或关闭了基准面的显示），您可以存储新设置以供将来的会话使用。Abaqus/CAE 将您的设置保存在一个名为 `abaqus_v6.14.gpr` 的文件中。有关更多信息，请参见["使用 `abaqus_v6.14.gpr` 文件，" 第2.1.3节](pt01ch02s01s03.md)。

从主菜单栏，选择 ****文件**![](../graphics/images/arrow.gif)**保存显示选项**** 可以保存所有当前显示选项设置。**保存显示选项** 对话框出现，允许您将选项保存在当前目录或主目录中。

您可以使用 Abaqus Scripting Interface 中的API命令编辑 `abaqus_v6.14.gpr` 文件；有关更多信息，请参见["编辑显示首选项和GUI设置，" Abaqus Scripting User's Guide 第8.4节](../cmd/cmd-link.md#cmd-int-exa-caeprefsaccess)。您也可以删除该文件以恢复默认的GUI和显示选项设置，并且可以将该文件复制到计算机上的其他目录或传输到不同的计算机。当您从 `abaqus_v6.14.gpr` 保存设置时，Abaqus/CAE 始终保存所有当前设置，并始终覆盖之前保存的所有设置。您不能仅保存选定的设置。您可以使用 **noSavedOptions** 命令行选项来启动 Abaqus/CAE 而不加载 `abaqus_v6.14.gpr` 中的设置。有关更多信息，请参见["启动 Abaqus/CAE（或Abaqus/Viewer），" 第2.1.1节](pt01ch02s01s01.md)。

Abaqus/CAE 在 `abaqus_v6.14.gpr` 中保存以下显示选项设置：
- 零件和装配显示选项；例如，渲染样式、规定条件的可见性和各种类型的基准几何，以及网格、单元和节点标签的显示。
- 图形选项和视口注释选项。Abaqus/CAE 还会保存透视设置。
- 打印选项。
- 可视化模块中的显示选项；例如，云图绘制中的等值线类型以及通用绘图选项中的渲染样式和填充颜色。
- 可视化模块中的动画选项。
- 可视化模块中的其他选项，例如探针、场报告、*X-Y* 绘图和 *X-Y* 报告选项。
