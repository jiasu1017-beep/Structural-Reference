# 62.5.4 创建datum坐标系的方法概述







datum坐标系在整个Abaqus/CAE中使用；例如，定义材料取向和定义连接器取向。为了帮助您跟踪datum坐标系，您可以在创建时为系统命名，该名称会显示在模型树中其条目的旁边。当您从**Create Datum**对话框中选择**CSYS**时，**Method**列表显示以下创建datum坐标系的方法：

**![](../graphics/ico_dtmCsysThreePoints.png) 三点**

通过选择原点以及（可选）两个附加点来定义直角、圆柱或球坐标系。对于直角系统，第二点定义*X*轴，*X*–*Y*平面通过第二点和第三点，如[图62--29](pt06ch62s05s04.md#dtm-hc3pts)所示。这是创建datum坐标系最通用的工具，在可能的情况下应使用它。有关详细说明，请参见["由三点定义的datum坐标系，" 第62.9.1节](pt06ch62s09hlb01.md)。

**图62–29** 通过选择原点和两点来定位直角datum坐标系。

![](../graphics/dtm-hc3pts-nls.png)

**![](../graphics/ico_dtmCsysOffset.png) 从CSYS偏移**

选择一个坐标系；通过指定偏移来提供直角、圆柱或球datum坐标系的位置，如[图62--30](pt06ch62s05s04.md#dtm-hcofd)中涉及直角系统的示例所示。您可以通过输入值或选择一个点来指定偏移。有关详细说明，请参见["在距另一坐标系的偏移处创建datum坐标系，" 第62.9.2节](pt06ch62s09hlb02.md)。

**图62–30** 通过从另一坐标系偏移来定位直角datum坐标系。

![](../graphics/dtm-hcofd-nls.png)

**![](../graphics/ico_dtmCsysTwoLines.png) 两条线**

选择定义直角、圆柱或球坐标系统的两条边。对于直角系统，第一条边定义*X*轴，*X*–*Y*平面通过第二条边，如[图62--31](pt06ch62s05s04.md#dtm-hc2l)所示。有关详细说明，请参见["由两条线定义的datum坐标系，" 第62.9.3节](pt06ch62s09hlb03.md)。

**图62–31** 通过选择两条边来定位直角datum坐标系。

![](../graphics/dtm-hc2l-nls.png)
