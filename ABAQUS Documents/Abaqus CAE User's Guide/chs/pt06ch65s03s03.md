# 65.3.3 什么是缓存？







缓存是内存的一部分。Abaqus/CAE 使用以下与特征相关的缓存来存储几何状态：
- **Current part cache**（当前部件缓存）包含当前部件几何状态的快照。Abaqus 为模型中的每个部件创建一个部件缓存。
- **All part caches**（所有部件缓存）包含当前模型中所有部件几何状态的快照。
- **Assembly cache**（装配缓存）包含装配几何状态的快照。Abaqus 为模型创建一个装配缓存。

如果您发现 Abaqus/CAE 在您的工作站上消耗了过多内存并影响了系统性能，您可以使用 **Options**（选项）对话框中的 **Clear**（清除）按钮来擦除与部件和装配缓存关联的内存。清除此内存将提高性能；但是，几何状态的快照将被擦除，再生将变慢。如果关闭缓存几何状态的选项，Abaqus/CAE 不会清除已存储的状态。此外，Abaqus/CAE 为每个模型存储一个单独的缓存。因此，您可以通过删除当前不在工作的任何模型的缓存来释放更多内存。

如果您花费大量时间对部件进行了一系列更改，并且确信您的设计是正确的，您可以通过在 **Options**（选项）对话框中点击 **Cache current state**（缓存当前状态）来手动保存内存中的当前状态。因此，Abaqus/CAE 可以返回此状态并再生任何后续的添加或修改。但是，如果您继续自动缓存几何状态，Abaqus/CAE 可能会在保存了最大数量的几何状态后擦除您手动保存的状态。或者，您可以关闭自动缓存几何状态的选项，并继续以重要间隔手动保存状态。

**Options**（选项）对话框中的 **Geometry Caching for Fast Regeneration**（快速再生的几何缓存）选项指示在三个与特征相关的缓存中存储了多少几何状态。缓存可以包含您手动保存的几何状态和 Abaqus/CAE 自动保存的状态的组合。您可以查询 **Current part cache**（当前部件缓存）和 **Assembly cache**（装配缓存）来确定几何状态相对于特征序列的位置。
