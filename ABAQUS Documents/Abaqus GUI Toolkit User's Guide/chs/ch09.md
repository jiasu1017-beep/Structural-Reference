# 9 非线性显式动力学





在前几章中，您探索了显式动力学分析的基本原理；在本章中，您将更深入地了解这一主题。显式动力学分析方法是解决各种非线性固体和结构力学问题的有效工具。它通常与像 Abaqus/Standard 这样的隐式求解器互补。从用户的角度来看，显式方法和隐式方法的显著特征是：
- 显式方法需要很小的时间增量，大小仅取决于模型的最高固有频率，与加载的类型和持续时间无关。模拟通常需要大约 10,000 到 1,000,000 个增量，但每个增量的计算成本相对较小。
- 隐式方法对时间增量大小没有固有的限制；增量大小通常由精度和收敛性考虑决定。隐式模拟通常比显式模拟需要的增量数量少几个数量级。然而，由于每个增量必须求解一组全局方程，隐式方法每个增量的成本远高于显式方法。

了解这两种方法的特性可以帮助您决定哪种方法适合您的问题。
