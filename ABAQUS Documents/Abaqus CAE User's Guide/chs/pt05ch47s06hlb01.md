# 47.7.2 为 X–Y 绘图自定义边框和填充颜色





### 47.7.2 为 X–Y 绘图自定义边框和填充颜色

点击 **网格显示（Grid Display）** 选项卡访问 X–Y 绘图的 **网格区域（Grid Area）** 选项。这些选项使您可以自定义 X–Y 绘图中的网格，即绘制曲线的矩形区域。您可以在网格周围包裹边框或用背景色填充网格。

**注意：**您还可以控制整个 X–Y 绘图的边框和背景颜色设置，这包括 X–Y 绘图的网格和所有坐标轴覆盖的视口部分。Abaqus/CAE 在整个 X–Y 绘图的背景之上显示网格的背景颜色。有关更多信息，请参阅 ["为 X–Y 绘图自定义边框和填充颜色，" 第 47.10 节"](pt05ch47s09.md)。

**要为 X–Y 绘图自定义边框和填充颜色：**

1. 找到 **网格区域（Grid Area）** 选项。选择 ****选项（Options）**![](../graphics/images/arrow.gif)**XY 选项（XY Options）**![](../graphics/images/arrow.gif)**图表（Chart）****；或点击 ![](../graphics/ico_xyChartOptions.png)，该图标位于可视化模块工具箱的 X–Y 绘图工具中。点击出现的对话框中的 **网格显示（Grid Display）** 选项卡。**网格区域（Grid Area）** 选项可在 **网格显示（Grid Display）** 页面的底部找到。
2. 从窗口顶部选择一个或多个 X–Y 绘图。
3. 切换 **显示边框（Show border）** 以在整个网格区域周围显示边框。
4. 如果需要，通过执行以下步骤更改边框颜色：1. 点击 **显示边框（Show border）** 标签右侧的颜色样本 ![](../graphics/usi-colorsample.png)。Abaqus/CAE 显示 **选择颜色（Select Color）** 对话框。2. 使用 **选择颜色（Select Color）** 对话框中的方法选择新颜色。有关更多信息，请参阅 ["自定义颜色，" 第 3.2.9 节"](pt01ch03s02s09.md)。3. 点击 **确定（OK）** 关闭 **选择颜色（Select Color）** 对话框。颜色样本和网格边框变为所选颜色。
5. 切换 **填充（Fill）** 以用 **填充（Fill）** 标签右侧显示的颜色填充网格区域。
6. 如果需要，通过执行以下步骤更改网格区域填充颜色：1. 点击 **填充（Fill）** 标签右侧的颜色样本 ![](../graphics/usi-colorsample.png)。Abaqus/CAE 显示 **选择颜色（Select Color）** 对话框。2. 使用 **选择颜色（Select Color）** 对话框中的方法选择新颜色。有关更多信息，请参阅 ["自定义颜色，" 第 3.2.9 节"](pt01ch03s02s09.md)。3. 点击 **确定（OK）** 关闭 **选择颜色（Select Color）** 对话框。网格区域变为所选颜色。
7. 点击 **关闭（Dismiss）** 关闭 **图表选项（Chart Options）** 对话框。

# 47.7.4 自定义 X–Y 绘图的位置





### 47.7.4 自定义 X–Y 绘图的位置

使用 **图表选项（Chart Options）** 对话框中的 **位置（Position）** 选项来自定义 X–Y 绘图在视口中的位置。

**要自定义 X–Y 绘图的位置：**

1. 找到 **位置（Position）** 选项。选择 ****选项（Options）**![](../graphics/images/arrow.gif)**XY 选项（XY Options）**![](../graphics/images/arrow.gif)**图表（Chart）****；或点击 ![](../graphics/ico_xyChartOptions.png)，该图标位于可视化模块工具箱的 X–Y 绘图工具中。点击出现的对话框中的 **网格位置（Grid Position）** 选项卡。
2. 从 **图表（Charts）** 字段中，选择一个或多个要自定义其位置或形状的 X–Y 绘图。
3. 选择以下 X–Y 绘图定位选项之一：**自动对齐（Auto-align）** 此选项使您可以将所选 X–Y 绘图对齐到视口中九个位置之一：其中心、四个角中的任意一个，或沿四个边缘中的任意一个居中。展开 **对齐（Alignment）** 列表，然后选择列表中以图形方式描绘的选项之一。**手动（Manual）** 此选项使您可以将 X–Y 绘图原点定位在视口中的特定 X–Y 位置。拖动 **X** 和 **Y** 滑块可以在视口中水平或垂直移动 X–Y 绘图的左下角。当这些滑块的值设置为 `0` 时，X–Y 绘图原点放置在视口的左下角。
4. 点击 **关闭（Dismiss）** 关闭 **图表选项（Chart Options）** 对话框。

![](../graphics/images/black4rule.gif)有关相关信息，请点击以下项目：- ["选择要自定义的一条或多条 X–Y 曲线，" 第 47.6.2 节"](pt05ch47s05s01.md)





# 47.7.1 为 X–Y 绘图自定义网格线





### 47.7.1 为 X–Y 绘图自定义网格线

点击 **网格显示（Grid Display）** 选项卡来控制 X–Y 绘图的主网格线和次网格线的外观。主网格线和次网格线示例如 [图 47–44](pt05ch47s06hlb01.md#viw-gridlines) 所示。

**图 47–44** 带主网格线和次网格线的 X–Y 绘图。

![](../graphics/viw-gridlines-nls.png)

如果您的 X–Y 绘图在同一方向上使用多个坐标轴，Abaqus/CAE 仅为绘图中第一条曲线的坐标轴绘制网格线。

**网格显示（Grid Display）** 选项对四组网格线中的每一组提供独立控制；即对任一方向上的主网格线和次网格线提供独立控制。您可以显示或隐藏四组中的任意一组，并且可以修改任意一组网格线的颜色、样式或粗细。

**注意：**主网格线源于主刻度线，次网格线源于次刻度线。要控制网格线之间的间隔，您必须调整 [刻度线的间距](pt05ch47s04hlb06.md)。

**要自定义 X–Y 绘图中的网格显示：**

1. 找到坐标轴 **网格显示（Grid Display）** 选项。选择 ****选项（Options）**![](../graphics/images/arrow.gif)**XY 选项（XY Options）**![](../graphics/images/arrow.gif)**图表（Chart）****；或点击 ![](../graphics/ico_xyChartOptions.png)，该图标位于可视化模块工具箱的 X–Y 绘图工具中。点击出现的对话框中的 **网格显示（Grid Display）** 选项卡。
2. 从 **图表（Charts）** 字段中选择一个或多个要自定义的 X–Y 绘图。
3. 从 **X 网格线（X Gridlines）** 选项中，切换 **主（Major）** 或 **次（Minor）** 选项分别显示主或次垂直网格线。当 **主（Major）** 或 **次（Minor）** 开启时，该类型网格线的网格线样式、颜色和粗细选项变为可用。
4. 从 **Y 网格线（Y Gridlines）** 选项中，切换 **主（Major）** 或 **次（Minor）** 分别显示主或次水平网格线。
5. 重复以下步骤来自定义水平主网格线和次网格线以及垂直主网格线和次网格线：1. 选择网格线的颜色：1. 点击颜色样本 ![](../graphics/usi-colorsample.png)。Abaqus/CAE 显示 **选择颜色（Select Color）** 对话框。2. 使用 **选择颜色（Select Color）** 对话框中的方法选择新颜色。有关更多信息，请参阅 ["自定义颜色，" 第 3.2.9 节"](pt01ch03s02s09.md)。3. 点击 **确定（OK）** 关闭 **选择颜色（Select Color）** 对话框。所选网格线变为指定颜色。2. 选择线条样式：1. 展开样式列表显示线条样式选项（实线、虚线等）。样式列表是两个未标记列表选项中的上一个。2. 从样式列表中，点击所需的线条样式。所选网格线变为指定样式。3. 选择线条粗细：1. 展开粗细列表显示线条样式选项（实线、虚线等）。粗细列表是两个未标记列表选项中的下一个。2. 从粗细列表中，点击所需的线条粗细。所选网格线变为指定粗细。
6. 点击 **关闭（Dismiss）** 关闭 **图表选项（Chart Options）** 对话框。

# 47.7.3 自定义 X–Y 绘图的大小和形状





### 47.7.3 自定义 X–Y 绘图的大小和形状

使用 **图表选项（Chart Options）** 对话框中的 **大小（Size）** 选项来自定义 X–Y 绘图的大小和形状。

**要自定义 X–Y 绘图的大小或形状：**

1. 找到 **图表选项（Chart Options）**。选择 ****选项（Options）**![](../graphics/images/arrow.gif)**XY 选项（XY Options）**![](../graphics/images/arrow.gif)**图表（Chart）****；或点击 ![](../graphics/ico_xyChartOptions.png)，该图标位于可视化模块工具箱的 X–Y 绘图工具中。点击出现的对话框中的 **网格位置（Grid Position）** 选项卡。
2. 从 **图表（Charts）** 字段中，选择一个或多个要自定义其大小或形状的 X–Y 绘图。
3. 选择以下 X–Y 绘图大小调整和整形选项之一：**适应图表（Fit to chart）** Abaqus/CAE 用所选 X–Y 绘图填充整个视口，调整绘图的形状以匹配视口尺寸。**正方形（Square）** **大小（Size）** 滑块变为可用，您可以拖动它来增加或减小正方形 X–Y 绘图的大小。**手动（Manual）** **X** 和 **Y** 滑块变为可用，您可以拖动它们来沿任一轴增加或减小 X–Y 绘图的大小。
4. 点击 **关闭（Dismiss）** 关闭 **图表选项（Chart Options）** 对话框。
