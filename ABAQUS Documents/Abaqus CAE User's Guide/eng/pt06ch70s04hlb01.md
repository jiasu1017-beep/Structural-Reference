# 70.4.1 An overview of the methods for partitioning edges







Select ****Tools**![](../graphics/images/arrow.gif)**Partition**** from the main menu bar to display the **Create Partition** dialog box. When you choose **Edge** from the **Create Partition** dialog box, the **Method** list displays the following methods for partitioning edges:

**![](../graphics/ico_ptnEdgeLocation.png) **Specify parameter by location****

Pick a point anywhere along the edge. For detailed instructions, see ["Using the specify parameter by location method to partition edges," Section 70.5.1](pt06ch70s05hlb01.md).

**![](../graphics/ico_ptnEdgeParameter.png) **Enter parameter****

Enter a parameter in the prompt area, as shown in [Figure 70--5](pt06ch70s04hlb01.md#par-edge-parameter). An arrow along the edge indicates the direction of increasing parameter value from the start vertex (corresponding to an edge parameter value of zero) to the end vertex (corresponding to a value of one). For detailed instructions, see ["Using the enter parameter method to partition edges," Section 70.5.2](pt06ch70s05hlb02.md).

**Figure 70–5** Entering a parameter to partition an edge.

![](../graphics/par-edge-parameter-nls.png)

**![](../graphics/ico_ptnEdgeMidpoint.png) **Select midpoint/datum point****

Select the midpoint of the edge or a datum point along the edge, as shown in [Figure 70--6](pt06ch70s04hlb01.md#par-edge-pick). For detailed instructions, see ["Using the pick midpoint/datum point method to partition an edge," Section 70.5.3](pt06ch70s05hlb03.md).

**Figure 70–6** Selecting the midpoint or a datum to partition an edge.

![](../graphics/par-edge-pick-nls.png)

**![](../graphics/ico_ptnEdgeDatumPlane.png) **Use datum plane****

Select a datum plane. Abaqus/CAE creates the partition where the datum plane intersects the edges, as shown in [Figure 70--7](pt06ch70s04hlb01.md#par-edge-datum). Partitioning a group of selected edges with a datum plane is a useful technique for aligning a group of partitions. For detailed instructions, see ["Using the datum plane method to partition edges," Section 70.5.4](pt06ch70s05hlb04.md).

**Figure 70–7** Selecting a datum plane to partition edges.

![](../graphics/par-edge-datum-nls.png)
