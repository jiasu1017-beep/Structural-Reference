### 对不可网格化区域进行分割并对自顶向下区域进行网格划分

为了对零件进行网格划分，我们将首先创建三个分割。前两个分割在不可网格化区域的外边缘附近创建另一个自顶向下可扫掠网格区域。第三个分割是一个面分割，您稍后将其用作自底向上挤出方法的向量。您可以对整个不可网格化区域应用自底向上网格技术，而不需要分割。但是，生成的网格在外边缘附近会包含一些形状不佳的单元。在实践中，您可能需要创建多个自底向上网格，然后才能为整个区域生成满意的网格。对于更复杂的零件，您可能需要保存具有不同网格划分方法的零件副本，直到您确定哪种方法产生最佳网格。

**对不可网格化区域进行分割：**

1. 旋转零件，使其底部朝向您，如图17-105所示。
**图17-105** 在底面上创建偏移分割。
![](../graphics/mgn-buoffsetpartition.png)
2. 完成以下步骤来分割面：
   1. 从模块工具箱中的分割面工具中，选择草图方法工具![](../graphics/ico_ptnFaceSketch.png)。
   2. 选择零件的底面，并选择一条直边作为垂直边且位于右侧。Abaqus/CAE打开"草图器"。
   3. 从"草图器"工具箱中，选择偏移工具![](../graphics/ico_sketchOffset.png)。
   4. 选择弯曲边，然后单击鼠标按钮2接受选定的边。
   5. 在提示区域输入`2.5`作为偏移距离。Abaqus/CAE显示偏移分割的预览。
   6. 如果偏移显示正确（朝向零件内部），则单击**确定**；如果偏移显示在零件外部，则单击**翻转**。
   7. 在提示区域中单击鼠标按钮2或单击**完成**来分割面。Abaqus/CAE返回"网格"模块，并以图17-105所示方式显示分割面。
3. 将面分割挤出穿过区域。
   1. 从模块工具箱中的分割单元工具中，选择挤出/扫掠方法工具![](../graphics/ico_ptnCellSweepEdges.png)。
   2. 选择在步骤1中创建的面分割作为要挤出的边。
   3. 在提示区域中单击**沿方向挤出**，然后选择图17-106中所示的边。
**图17-106** 将面分割挤出穿过单元。
![](../graphics/mgn-bupartitioncell.png)
   4. 如果挤出方向显示正确（穿过区域），则单击**确定**；如果需要更改方向，则单击**翻转**。
   5. 单击**创建分割**来分割单元。Abaqus/CAE以黄色显示零件的外侧区域，表明现在可以使用自顶向下扫掠网格进行网格划分。
4. 按图17-107所示分割不可网格化区域的前表面。
**图17-107** 创建面分割。
![](../graphics/mgn-bupartitionface.png)
   1. 从模块工具箱中的分割面工具中，选择草图方法工具![](../graphics/ico_ptnFaceSketch.png)。
   2. 选择不可网格化区域的前表面，并选择一条边作为垂直边且位于右侧。Abaqus/CAE打开"草图器"。
   3. 使用垂直构造线工具![](../graphics/ico_sketchConstLineVertical.png)创建一条构造线，如图17-108所示。
**图17-108** 使用垂直构造线分割面。
![](../graphics/mgn-buconstline.png)
   4. 使用连接线工具![](../graphics/ico_sketchLines.png)创建一条连接垂直构造线与零件面相交的的两个点的线。
   5. 创建第二条线，连接垂直线上方的点与圆角在前表面结束的点——该线应接近水平。
   6. 在提示区域中单击鼠标按钮2或单击**完成**来分割面。现在零件中有三个区域可以使用自顶向下网格技术进行网格划分。
5. 使用近似尺寸`0.9`和曲率控制的默认设置为整个零件分配全局边缘种子。
6. 从主菜单栏中选择****网格**![](../graphics/images/arrow.gif)**零件****对自顶向下区域进行网格划分。Abaqus/CAE高亮显示不可网格化区域并显示警告，指出无法自动对其进行网格划分。
7. 单击**确定**对三个自顶向下区域进行网格划分。图17-109显示了生成的局部网格。

**图17-109** 自顶向下区域的自动网格划分。

![](../graphics/mgn-buautomesh.png)

### 开始自底向上网格

现在自顶向下网格已完成，您可以为剩余部分分配自底向上网格技术并开始创建六面体网格。

**创建自底向上扫掠网格：**

1. 使用网格控制工具![](../graphics/ico_meshControls.png)为不可网格化区域分配自底向上网格技术。Abaqus/CAE将该区域着色为浅棕色。
2. 从主菜单栏中选择****网格**![](../graphics/images/arrow.gif)**创建自底向上网格****。
**提示：**您也可以单击"网格"模块工具箱中的![](../graphics/ico_meshBottomUp.png)工具。（更多信息，请参见["使用网格模块工具箱，"第17.15节](pt03ch17s15.md)。）Abaqus/CAE显示**创建自底向上网格**对话框，并使未网格化的区域半透明。半透明度由**颜色编码**工具栏中的半透明度滑块决定（更多信息，请参见["更改半透明度，"第77.3节](pt07ch77s03.md)）。在更复杂的零件中，半透明度使您更容易选择内部区域。
3. 选择**扫掠**方法，然后单击**源侧**右侧的**选择**。
4. 选择如图17-110中以品红色显示的凸起底部的半圆面作为第一个自底向上网格的源侧；然后在提示区域中单击**完成**。您必须更改默认选择选项以选择内部面（更多信息，请参见["使用选择选项，"第6.3节](pt01ch06s03.md)）。
**图17-110** 选择几何面作为源侧。
![](../graphics/mgn-busourceside1.png)
**提示：**您也可以选择二维单元或单元面作为自底向上网格的边。在这种情况下，替代的源侧选择将是扫掠网格底部上的所有单元面。
5. 切换**连接边**，然后单击**选择**。根据需要旋转零件，选择代表顶部凸起与零件主体之间圆角的三个面，以及零件前部对应的扁平面，如图17-111中以黄色显示。
**图17-111** 选择连接边。
![](../graphics/mgn-buconnecting1.png)
在提示区域中单击**完成**以结束选择。
6. 在**创建自底向上网格**对话框中单击**网格**以创建网格。
**图17-112** 圆角区域的自底向上扫掠网格。
![](../graphics/mgn-bumesh1.png)
网格延伸到所选连接边大致相同的距离，自底向上区域保持选中状态，**创建自底向上网格**对话框保持打开状态以进行下一步。

### 继续自底向上网格

至少有三种方法可以完成零件的网格划分。最简单的方法是创建另一个扫掠网格，使用刚完成的网格底部和从未网格化区域顶部的圆角延伸出的两个面作为源侧，三个几何垂直面和一组垂直单元面作为连接边，底面未网格化部分作为目标侧。此方法将在单个自底向上网格步骤中完成零件网格，单元将与所选面完全关联。您也可以使用以零件底面作为挤出距离的自底向上挤出方法。但是，出于演示目的，我们将使用一个更长的过程，结合自底向上挤出方法、关联性工具和"编辑网格"工具集。

**创建自底向上挤出网格：**

1. 选择**挤出**方法，然后单击**源侧**右侧的**选择**。
2. 选择前一个自底向上网格底部的单元面，如图17-113所示，作为第二个自底向上网格的源侧。
**图17-113** 选择单元面作为源侧。
![](../graphics/mgn-busource2.png)
3. 在提示区域中单击**完成**以接受选定的面。**创建自底向上网格**对话框重新出现。
4. 单击**选择**按钮为挤出网格选择向量。
5. 选择分割的上端点作为向量的起点，选择分割的下端点作为向量的终点。图17-114显示了挤出向量。
**图17-114** 为自底向上挤出网格选择向量。
![](../graphics/mgn-buvector.png)
6. 为**层数**输入`10`。自顶向下扫掠网格的内面有10个单元。为挤出网格使用相同数量的单元将在创建第三个也是最后一个自底向上网格时提供更好的匹配。
7. 验证挤出深度是否使用默认的**使用向量长度**方法设置，然后在**创建自底向上网格**对话框中单击**网格**以创建网格。
**图17-115** 自底向上挤出网格。
![](../graphics/mgn-buextrudedmesh.png)

挤出的自底向上网格在非平面源侧和挤出向量长度的指导下终止于区域底面附近。您可以编辑最后一个挤出单元层中的节点，使它们正好终止于区域的底面上。

**扩展挤出网格：**

1. 选择"编辑网格"工具集![](../graphics/ico_meshEdit.png)，位于"网格"模块工具箱的底部。
2. 选择**节点**类别，然后在**编辑网格**对话框中单击**投影**。
3. 使用[角度方法](pt06ch64s05s02.md)选择自底向上挤出网格底部的所有节点，如图17-116所示；然后在提示区域中单击**完成**。
**图17-116** 选择要投影的节点。
![](../graphics/mgn-buprojectnodes.png)
4. 选择自底向上区域的底面，然后在提示区域中单击**是**将节点投影到该面上。

### 完成自底向上区域的网格

我们现在将创建一个最终的扫掠网格来完成零件的网格划分。

**完成网格：**

1. 从主菜单栏中选择****网格**![](../graphics/images/arrow.gif)**创建自底向上网格****。
2. 选择自底向上区域。Abaqus/CAE显示**创建自底向上网格**对话框。
3. 选择**扫掠**方法，然后单击**源侧**右侧的**选择**。
4. 选择零件顶部剩余的两个未网格化的面，如图17-117所示；然后在提示区域中单击**完成**。
**图17-117** 为最终自底向上网格选择源侧。
![](../graphics/mgn-busource3.png)
5. 切换**连接边**，然后单击**选择**。旋转零件，并使用角度方法选择外部扫掠网格的内部单元面和挤出网格的外部单元面，如图17-118所示。
**图17-118** 选择单元面作为连接边。
![](../graphics/mgn-buconnecting3.png)
在提示区域中单击**完成**以结束选择。
6. 切换**目标侧**，然后单击**选择**。Abaqus/CAE提示您为扫掠网格选择目标侧。
7. 选择区域的底面。
8. 在**创建自底向上网格**对话框中单击**网格**以创建网格。

该零件现在已完全使用六面体单元进行网格划分，如图17-119所示。

**图17-119** 最终网格化的零件。

![](../graphics/mgn-bucompletemesh.png)

### 检查自底向上网格与几何体的关联

在使用完成的自底向上网格进行分析之前，您应该检查区域几何体与自底向上网格单元之间的关联。Abaqus中的载荷、边界条件和其他属性应用于几何体，除非网格正确关联到几何体，否则它们不会正确传递到自底向上网格的单元。至少，您应该检查自底向上网格中施加载荷和边界条件的区域的关联。

在大多数情况下，如果您选择几何特征（如面）来定义自底向上网格，Abaqus/CAE会自动将适当的单元与该面关联。然而，在某些情况下不使用几何体，例如示例零件中心的挤出自底向上网格，单元仅与区域关联，而不与可能施加载荷的附近面关联。（更多信息，请参见["网格-几何体关联，"第17.12节](pt03ch17s12.md)。）以下过程将零件底面和前面上的单元与几何面关联：

**将自底向上网格与几何体关联：**

1. 从主菜单栏中选择****网格**![](../graphics/images/arrow.gif)**将网格与几何体关联****。
2. 选择自底向上区域的底面，如图17-120所示。在最终自底向上扫掠网格步骤中创建的单元着色为黄色，因为当该面用作最终自底向上扫掠网格的目标侧时，它们已经关联。然而，半圆形挤出网格中的单元未与该面关联。
**图17-120** 底面上现有的网格关联。
![](../graphics/mgn-buassociatebottom.png)
3. 移除从自底向上区域延伸到零件外曲面的自顶向下网格化单元。
   1. 选择****工具**![](../graphics/images/arrow.gif)**显示组**![](../graphics/images/arrow.gif)**创建****。Abaqus/CAE显示**创建显示组**对话框。
   2. 从项目列表中选择**单元**，然后单击**编辑选择**。
   3. 从视口中选择沿着零件弯曲外边缘延伸的自顶向下扫掠区域，然后在提示区域中单击**完成**。
   4. 在**创建显示组**对话框中，单击**移除**按钮。Abaqus/CAE从视口中移除选定的单元。更多信息，请参见[第78章，"使用显示组显示模型的子集"](pt07ch78.md)。
4. 使用角度方法选择自底向上区域底部的所有面。完成后，自底向上区域底部的所有单元面都应着色为黄色，如图17-121所示。
**图17-121** 底面上的最终网格关联。
![](../graphics/mgn-buassociatebottom2.png)
**注意：**如果您没有移除外部自顶向下网格化单元，使用角度方法也会选择自顶向下单元面进行关联以及自底向上面，这会导致在尝试关联面时出现错误消息。
5. 在提示区域中单击**完成**以将选定单元的面与区域面关联。

### 关联自底向上网格的其余部分

要完全关联自底向上网格，您应该验证并编辑自底向上区域中前面、右侧以及每个边界边和顶点的关联。您以与上一节中关联单元面相同的方式关联边和顶点，只是您分别选择边和单元边或顶点和节点。Abaqus/CAE尝试根据接近度将节点与顶点以及单元边与几何边关联。如果沿边的所有单元都与该边所限制的面关联，Abaqus/CAE会自动将单元边与几何边关联。

一旦您完全网格化了零件并检查了自底向上网格的关联，就应该保存模型。在运行分析之前，您还应该验证网格的质量。网格验证有助于确保没有隐藏的问题，无论您是自动创建网格（自顶向下）还是使用自底向上技术和网格编辑来构建网格。更多信息，请参见["验证单元质量，"第17.19.1节](pt03ch17s19hlb01.md)。
