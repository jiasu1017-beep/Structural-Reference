# 45.5.2 设置向量和张量限制





默认情况下，Abaqus/CAE在符号图中包含变量的所有结果值。但是，您可以使用**Symbol Plot Options**对话框限制图中出现的值范围。

当您指定特定最大值时，图中仅显示表示小于或等于该最大值的箭头。当您指定特定最小值时，图中仅显示表示大于该最小值的箭头。更改最大值或最小值也会反映在符号图图例中。

要了解如何显示与符号图关联的最小值和最大值，请参见["自定义图例，" 第56.1节](pt05ch56s01.md)。

**要设置向量或张量限制：**

1. 找到**Limits**选项。从主菜单栏中选择****Options**![](../graphics/images/arrow.gif)**Symbol****，或单击工具箱中的![](../graphics/ico_plotSymbolOptions.png)；然后在出现的对话框中单击**Limits**选项卡。单击**Vector**选项卡或**Tensor**选项卡；您选择的变量类型的符号图**Limits**选项变得可用。
2. 在页面顶部标记为**Max**的区域中，选择以下选项之一：- 选择**Auto-compute**以使用最大结果值作为最大值。- 选择**Specify**以自己指定最大值；然后在**Specify**文本字段中输入您选择的最大值。
3. 在页面底部标记为**Min**的区域中，选择以下选项之一：- 选择**Auto-compute**以使用最小结果值作为最小值。- 选择**Specify**以自己指定最小值；然后在**Specify**文本字段中输入您选择的最小值。
4. 单击**Apply**实施您的更改。当前视口中的符号图更改为仅显示落在您指定的限制范围内的变量值的箭头。默认情况下，您对符号图选项的更改在会话期间保存，并将影响当前视口和从当前视口创建的任何新视口中的所有后续符号图。如果您想在后续会话中保留更改，请将它们保存到文件。有关更多信息，请参见["保存自定义设置以供后续会话使用，" 第55.1.1节](pt05ch55s01s01.md)。
