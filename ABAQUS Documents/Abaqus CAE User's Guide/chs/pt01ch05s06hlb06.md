# 5.6.6 将视图重新调整以适应视口





使用**视图操作**工具栏中的自动拟合工具 ![](../graphics/ico_viewFit.png) 可以快速平移和放大或缩小视图，使视图填充视口并在视口中居中。当你拟合视图时，方向保持不变，如视图 triad 所示。如果重新调整的视口链接到其他视口，Abaqus/CAE 会以相同因子在那些视口中放大或缩小视图。有关更多信息，请参阅["链接视口以进行视图操作，" 第 4.5 节](pt01ch04s05.md)。

从**视图操作**工具栏中，单击自动拟合工具 ![](../graphics/ico_viewFit.png) 进入自动拟合模式。

**提示：**你也可以从主菜单中选择****View**![](../graphics/images/arrow.gif)**Auto-fit****，或按 **[F6]**。

如果你只有一个视口，Abaqus/CAE 会立即将视图缩放以适应视口而不改变方向，在视口中居中视图，然后退出拟合模式。如果你有多个视口，请选择自动拟合工具，然后将光标放在要重新调整的视口上。点击视口以自动拟合；Abaqus/CAE 会重新调整视图并退出拟合模式。

**提示：**使用[循环](pt01ch05s06hlb07.md)视图操作工具返回上一个视图。

有关在旋转期间自动将视图重新调整以适应视口的方法，请参阅["旋转视图，" 第 5.6.3 节](pt01ch05s06hlb03.md)。
