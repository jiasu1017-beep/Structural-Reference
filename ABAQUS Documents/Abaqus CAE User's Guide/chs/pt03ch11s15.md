# 11.15 将草图编辑器与零件模块结合使用





草图是形成定义Abaqus/CAE本机零件特征的二维轮廓。您使用草图编辑器创建这些草图；在零件模块中，您直接使用它们来定义平面零件或梁，或者通过拉伸、扫掠或旋转它们来形成三维或轴对称零件。每当需要创建新零件的基础特征、向零件添加特征或修改现有特征时，零件模块会自动进入草图编辑器，您可以在形成特征二维轮廓的草图上操作。完成草图绘制后，Abaqus/CAE会自动返回零件模块。

如果您正在添加特征或修改现有特征，必须选择要绘制草图的平面。关于Abaqus/CAE如何确定零件相对于草图平面的方向的详细说明，请参见"Abaqus/CAE如何定向您的草图"，第20.4.3节。![](../graphics/images/black4rule.gif)有关相关信息，请单击以下项目：- 第20章"草图模块"
