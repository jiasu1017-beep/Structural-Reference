# 15.13.7 定义表面对表面接触





表面对表面接触定义可用作通用接触的替代方案，对模型中特定表面之间的接触相互作用进行建模。某些相互作用行为只能通过使用表面对表面接触来定义。简要概述表面对表面接触和Abaqus中可用的其他类型的相互作用，请参见「理解相互作用」，第15.3节，以及「接触相互作用分析概述」，Abaqus Analysis User's Guide第36.1.1节。

您可以在任何步骤（包括初始步骤）中定义表面对表面接触。从主菜单栏选择 ****Interaction**![](../graphics/images/arrow.gif)**Create****，并选择主表面和从表面。您可以定义线边缘之间或实体或壳面之间的接触。某些连接限制适用于接触表面，具体取决于接触公式的类型。您可以在一个步骤中停用表面对表面接触相互作用，并根据需要在其后续步骤中重新激活此相互作用。如果在分析中不再需要该相互作用，您可以在一个步骤中停用它。

如果您要创建多个表面对表面接触相互作用，您可能需要使用接触检测工具。此工具自动选择表面的过程，并允许您同时创建多个相互作用。更多信息，请参见「使用接触和约束检测」，第15.16节。

您可以使用「Step」模块中的场和历史输出请求编辑器获取特定表面对表面接触相互作用的接触数据。在编辑器的「Domain」部分，选择「Interaction」并从出现的菜单中选择表面对表面接触相互作用的名称。更多信息，请参见「创建输出请求」，第14.12.1节。

定义表面对表面接触的过程取决于您是使用Abaqus/Standard还是Abaqus/Explicit进行分析。本节提供使用相互作用编辑器定义不同表面对表面接触选项的说明。以下主题已涵盖：
- 「在Abaqus/Standard分析中定义表面对表面接触」
- 「在Abaqus/Explicit分析中定义表面对表面接触」
- 「指定干涉配合选项」

### 在Abaqus/Standard分析中定义表面对表面接触

某些相互作用行为只能通过使用表面对表面接触在Abaqus/Standard中定义；更多信息，请参见「Abaqus/Standard中的接触仿真能力」，接触相互作用分析概述，第36.1.1节。

**在Abaqus/Standard分析中定义表面对表面接触的步骤：**

1. 从主菜单栏选择 ****Interaction**![](../graphics/images/arrow.gif)**Create****。**提示：**您也可以使用「相互作用」模块工具箱中的![](../graphics/ico_interaction.png)工具创建表面对表面接触相互作用。
2. 在出现的「Create Interaction」对话框中，执行以下操作：- 为相互作用命名。有关命名对象的更多信息，请参见「使用基本对话框组件」，第3.2.1节。- 选择创建相互作用的步骤。- 选择「Surface-to-surface contact (Standard)」类型的相互作用。
3. 点击「Continue」关闭「Create Interaction」对话框。
4. 使用以下方法之一选择主表面：- 使用现有表面定义区域。在提示区域右侧点击「Surfaces」。从出现的「Region Selection」对话框中选择一个现有表面，然后点击「Continue」。**注意：**默认选择方法基于您最近使用的选择方法。如需恢复为另一种方法，请点击提示区域右侧的「Select in Viewport」或「Surfaces」。- 使用鼠标在视口中选择一个区域。（更多信息，请参见「在当前视口中选择对象」，第6.2节。）点击鼠标按钮2表示您已完成选择。某些连接限制适用于接触表面，具体取决于接触公式的类型。详细说明，请参见「在Abaqus/Standard中定义接触对」，Abaqus Analysis User's Guide第36.3.1节。如果模型包含网格和几何的组合，请从提示区域点击以下之一：- 如果您想从几何区域选择表面，请点击「Geometry」。- 如果您想从本机或孤儿网格选择中选择表面，请点击「Mesh」。您可以使用角度方法从几何中选择一组面或边，或从网格中选择一组单元面。更多信息，请参见「使用角度和特征边方法选择多个对象」，第6.2.3节。您选择的主表面会在视口中以红色高亮显示。
5. 选择从表面。1. 在提示区域中选择以下之一：- 如果您想选择一个表面，请选择「Surface」。- 如果您想选择一个区域以创建接触节点集，请选择「Node Region」。2. 使用前面描述的相同方法之一选择从表面或区域。您选择的从表面或区域会在视口中以品红色高亮显示。此时会出现「Edit Interaction」对话框。
6. 「Switch Surfaces」选项允许您交换主表面和从表面选择，而无需重新开始。「Switch Surfaces」![](../graphics/ico_switchSurfaces.png)图标仅在您在上一步骤中选择了「Surface」时才可用。
7. 选择滑动公式。- 选择「Finite sliding」使用有限滑动公式，这是最通用的，允许表面任意运动。- 选择「Small sliding」使用小滑动公式，它假定虽然两个物体可能经历大运动，但一个表面沿另一个表面的滑动相对较小。更多信息，请参见「Abaqus/Standard中的接触公式」，Abaqus Analysis User's Guide第38.1.1节。
8. 选择离散化方法。- 选择「Node to surface」使用节点对表面离散化方法。- 选择「Surface to surface」使用表面对表面离散化方法。更多信息，请参见「Abaqus/Standard中的接触公式」，Abaqus Analysis User's Guide第38.1.1节中的「接触对表面的离散化」。
9. 根据滑动公式和离散化方法选择的组合，不同的字段会变得可用。- 默认情况下，对于以下组合，壳和膜厚度包含在接触计算中：**Small sliding**和「Node to surface」，「Small sliding」和「Surface to surface」，以及「Finite sliding」和「Surface to surface」。您可以切换「Exclude shell/membrane element thickness」以忽略任何这些组合的壳和膜厚度。使用「Finite sliding」和「Node to surface」的接触相互作用不考虑表面厚度。更多信息，请参见「在Abaqus/Standard中为接触对分配表面属性」，Abaqus Analysis User's Guide第36.3.2节中的「考虑壳和膜厚度」。- 对于使用「Node to surface」离散化方法的接触相互作用，您可以在「Degree of smoothing for master surface」字段中指定平滑因子。更多信息，请参见「Abaqus/Standard中的接触公式」，Abaqus Analysis User's Guide第38.1.1节中的「有限滑动、节点对表面公式的主表面平滑」。- 默认情况下，对于以下组合使用补充接触约束的选择性方案：**Finite sliding**和「Node to surface」，「Small sliding」和「Node to surface」，以及「Small sliding」和「Surface to surface」。对于这些组合，您可以按如下方式指定何时「Use supplementary contact points」：- 选择「Selectively」使用补充接触约束的选择性方案。- 选择「Never」放弃使用补充接触约束。- 选择「Always」在适用时添加补充接触约束。更多信息，请参见「Abaqus/Standard中调整接触控制」，Abaqus Analysis User's Guide第36.3.6节中的「补充接触约束」。- 对于使用「Finite sliding」和「Surface to surface」的接触相互作用，您可以选择「Contact tracking」方法。- 选择「Single configuration (state)」使用基于状态的跟踪算法。- 选择「Two configurations (path)」使用基于路径的跟踪算法。更多信息，请参见「Abaqus/Standard中的接触公式」，Abaqus Analysis User's Guide第38.1.1节中的「基于路径与基于状态的跟踪算法」。**注意：**如果您的接触相互作用使用表面对表面离散化方法，并且接触相互作用中的一个或多个表面是解析刚性表面，您应该选择基于状态的跟踪算法。
10. 指定从节点调整选项。更多信息，请参见「在Abaqus/Standard接触对中调整初始表面位置和指定初始间隙」，Abaqus Analysis User's Guide第36.3.5节，以及「在Abaqus/Standard中定义绑定接触」，Abaqus Analysis User's Guide第36.3.7节。
11. 对于使用「Surface to surface」离散化方法的接触相互作用，您可以将对接触表面应用平滑，以减少由弯曲几何上网格离散化引起的接触压力不准确。点击「Surface Smoothing」选项卡，并选择以下选项之一：- 选择「Do not smooth」防止应用平滑。- 选择「Automatically smooth 3D geometry surfaces when applicable」对由Abaqus/CAE自动识别的轴对称或球面表面（或部分表面）应用平滑。自动平滑对网格零件或二维模型没有影响。有关接触平滑技术的更多信息，请参见「Abaqus/Standard中平滑接触表面」，Abaqus Analysis User's Guide第38.1.3节。
12. 对于使用「Small sliding」公式的接触相互作用，您可以指定从表面节点与主表面之间的初始间隙。点击「Clearance」选项卡，从「Initial clearance」字段中选择间隙类型，并输入定义间隙和接触方向所需的所有数据。更多信息，请参见「在Abaqus/Standard接触对中调整初始表面位置和指定初始间隙」，Abaqus Analysis User's Guide第36.3.5节中的「为小滑动接触定义精确的初始间隙或过盈」。
13. 如果您为接触相互作用指定了节点对表面离散化，您还可以将绑定限制为特定子集中的从节点。点击「Bonding」选项卡，切换「Limit bonding to slave nodes in subset」，并从列表中选择一个节点集。您可以限制以下任一情况的绑定：- 当您想指定应体验内聚力的初始从节点的子集时。那些最初未接触但指定在节点集中的节点将进行无应变调整。此集合之外的所有从节点（包括那些最初接触主表面的节点）将在分析过程中仅体验压缩接触力。更多信息，请参见「定义接触相互作用属性」，第15.14.1节中的「为机械接触属性选项指定内聚行为属性」。- 当您想识别VCCT裂纹中从表面的初始绑定区域时。从表面的未绑定部分表现为常规接触表面。预定裂纹表面假定为初始部分绑定，以便在分析过程中可以明确识别裂纹尖端。更多信息，请参见「裂纹扩展分析」，Abaqus Analysis User's Guide第11.4.3节中的「在Abaqus/Standard中定义初始绑定裂纹表面」。
14. 选择接触相互作用属性。如有需要，点击![](../graphics/ico_interactionProperty.png)创建相互作用属性。更多信息，请参见「定义接触相互作用属性」，第15.14.1节，以及「Abaqus/Standard中的接触约束实施方法」，Abaqus Analysis User's Guide第38.1.2节。
15. 要指定干涉配合选项，请点击「Interference Fit」。干涉配合选项不能在初始步骤中指定。请参见下文「指定干涉配合选项」中的更详细说明。
16. 如有需要，点击「Contact controls」字段旁边的箭头并选择要用于此相互作用的自定义接触控制。只有先前创建的Abaqus/Standard接触控制才会出现在列表中。更多信息，请参见「在Abaqus/Standard分析中指定接触控制」，第15.13.9节。
17. 要在一个步骤中停用和重新激活接触相互作用，请切换「Active in this step」。接触相互作用在创建它的步骤中处于活动状态。更多信息，请参见「在Abaqus/Standard中定义接触对」，Abaqus Analysis User's Guide第36.3.1节中的「移除和重新激活接触对」。
18. 点击「OK」创建相互作用并关闭编辑器。

- 「相互作用编辑器」，第15.9.2节
- 「自定义接触控制」，第15.12.3节

### 在Abaqus/Explicit分析中定义表面对表面接触

某些相互作用行为只能通过使用表面对表面接触在Abaqus/Explicit中定义；更多信息，请参见「Abaqus/Explicit中的接触仿真能力」，接触相互作用分析概述，第36.1.1节。

**在Abaqus/Explicit分析中定义表面对表面接触的步骤：**

1. 从主菜单栏选择 ****Interaction**![](../graphics/images/arrow.gif)**Create****。**提示：**您也可以使用「相互作用」模块工具箱中的![](../graphics/ico_interaction.png)工具创建表面对表面接触相互作用。
2. 在出现的「Create Interaction」对话框中，执行以下操作：- 为相互作用命名。- 选择创建相互作用的步骤。- 选择「Surface-to-surface contact (Explicit)」类型的相互作用。
3. 点击「Continue」关闭「Create Interaction」对话框。
4. 使用以下方法之一选择主表面：- 使用现有表面定义区域。在提示区域右侧点击「Surfaces」。从出现的「Region Selection」对话框中选择一个现有表面，然后点击「Continue」。- 使用鼠标在视口中选择一个区域。点击鼠标按钮2表示您已完成选择。某些连接限制适用于接触表面，具体取决于接触公式的类型。如果模型包含网格和几何的组合，请从提示区域点击以下之一：- 点击「Geometry」。- 点击「Mesh」。您选择的主表面会在视口中以红色高亮显示。
5. 选择从表面。1. 在提示区域中选择以下之一：- 选择「Surface」。- 选择「Node Region」。2. 使用前面描述的相同方法之一选择从表面或区域。此时会出现「Edit Interaction」对话框。
6. 「Switch Surfaces」选项允许您交换主表面和从表面选择。
7. 选择机械约束公式。- 选择「Kinematic contact method」使用运动预测/校正器接触算法。- 选择「Penalty contact method」使用罚接触算法。更多信息，请参见「Abaqus/Explicit中的接触约束实施方法」，Abaqus Analysis User's Guide第38.2.3节。
8. 选择滑动公式。- 选择「Finite sliding」。- 选择「Small sliding」。「Small sliding」公式只能为在初始步骤或第一个通用分析步骤中创建的相互作用指定。
9. 对于使用「Small sliding」公式的接触相互作用，您可以指定从表面节点与主表面之间的初始间隙。选择「Initial clearance」字段中的间隙类型，并输入定义间隙和接触方向所需的所有数据。
10. 选择接触相互作用属性。
11. 选择权重因子。
12. 如有需要，选择自定义接触控制。
13. 要停用和重新激活接触相互作用，请切换「Active in this step」。
14. 点击「OK」创建相互作用并关闭编辑器。

### 指定干涉配合选项

当您为Abaqus/Standard定义表面对表面接触时，您可以指定干涉配合选项，以帮助Abaqus/Standard解析模型初始配置中表面之间的过度过盈。更多信息，请参见「在Abaqus/Standard中建模接触干涉配合」，Abaqus Analysis User's Guide第36.3.4节。

**指定干涉配合选项的步骤：**

1. 在「Interference Fit Options」对话框中，选择「Gradually remove slave node overclosure during the step」以规定允许的干涉。
2. 选择以下选项之一：- 如果您希望Abaqus/Standard为每个从节点分配不同的允许干涉（等于该节点的初始穿透），请选择「Automatic shrink fit (first general analysis step only)」。如果您选择此选项，请跳至步骤6。- 选择「Uniform allowable interference」以指定将应用于每个从节点的单一允许干涉。
3. 点击「Amplitude」字段右侧的箭头以选择定义步骤期间规定干涉幅值的幅值曲线名称。或者，您可以选择「(Ramp)」以在步骤开始时立即应用规定干涉，并在步骤中线性降至零。如有需要，您可以点击![](../graphics/ico_amplitudeSmall.png)定义新的幅值曲线。
4. 在「Magnitude at start of step」字段中，输入步骤开始时允许干涉的幅值。
5. 如有需要，选择「Interference Direction」选项「Along direction」以指定偏移方向向量。
6. 点击「OK」保存您指定的干涉配合选项并返回到「Edit Interaction」对话框。
