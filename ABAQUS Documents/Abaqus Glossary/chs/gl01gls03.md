# J







### job

您提交以在任何计算机或网络上执行的进程。您以分析作业的形式向 Abaqus/CAE 提交 Abaqus 模型进行分析。
更多信息：- [「Abaqus/Standard、Abaqus/Explicit 和 Abaqus/CFD 执行，」Abaqus Analysis User's Guide 第 3.2.2 节](../usb/usb-link.md#usb-int-danalysisproc)

- [第 19 章，「The Job module，」Abaqus/CAE User's Guide](../usi/usi-link.md#usi-ana)

### Job module

Abaqus/CAE 模块，用于创建作业、提交分析并监控其进度。
更多信息：- [第 19 章，「The Job module，」Abaqus/CAE User's Guide](../usi/usi-link.md#usi-ana)

### job monitor

使用作业监视器对话框在模型数据库中观察 Abaqus/CAE 分析作业状态的过程。
更多信息：- [「输出，」Abaqus Analysis User's Guide 第 4.1.1 节](../usb/usb-link.md#usb-out-ooutput)

- [「监控分析作业的进度，」Abaqus/CAE User's Guide 第 19.2.6 节](../usi/usi-link.md#usi-ana-conc-monitor)

### journal（`.jnl`）file

包含命令的文件（`*model_database_name*.jnl`），如果模型数据库损坏，Abaqus/CAE 可使用这些命令重新创建模型数据库。当您保存模型数据库时，Abaqus/CAE 也会自动保存日志文件。
更多信息：- [「重新创建未保存的模型数据库，」Abaqus/CAE User's Guide 第 9.5.3 节](../usi/usi-link.md#usi-dbs-files-recover)
