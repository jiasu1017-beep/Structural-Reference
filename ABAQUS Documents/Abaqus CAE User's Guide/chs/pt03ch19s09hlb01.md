# 19.9.1 创建新的自适应过程







自适应过程是连续的分析作业。Abaqus/CAE 在每个分析作业中根据从上次分析最终增量写入输出数据库的错误估计来修改网格。有关更多信息，请参见["什么是自适应过程？"，第 19.3.1 节](pt03ch19s03s01.md)。要创建新的自适应过程，请从主菜单栏中选择****自适应**![](../graphics/images/arrow.gif)**创建****。

**创建新的自适应过程：**

1. 从主菜单栏中，选择****自适应**![](../graphics/images/arrow.gif)**创建****。出现**编辑自适应过程**对话框。**提示：**您也可以通过点击模块工具箱中的![](../graphics/ico_adaptiveRemesh.png)图标或点击**自适应过程管理器**中的**创建**来创建新的自适应过程。
2. 在**名称**文本字段中输入新的自适应过程的名称。您指定的名称必须遵守操作系统的文件命名规则。
3. 从**模型**字段中选择要与自适应过程关联的模型。
4. 如有需要，在**描述**文本字段中输入自适应过程的描述。描述存储在模型数据库和输出数据库中，您可以使用此描述来帮助管理您的自适应过程。
5. 在**作业前缀**文本字段中输入作业前缀。自适应过程每次迭代的作业名称将以此前缀开头。您指定的名称必须遵守操作系统的文件命名规则。如果您不提供作业前缀，Abaqus/CAE 将使用自适应过程的名称。
6. 在编辑器中输入定义过程所需的所有数据，然后点击**确定**。（有关更多信息，请参见["使用自适应过程编辑器，"第 19.10 节](pt03ch19s10.md)。）
