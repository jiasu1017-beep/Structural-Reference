# 3.1.1 Abaqus执行过程概述





### 概述

Abaqus通过使用Abaqus执行过程执行。在下述讨论中，运行执行过程的命令假定为**abaqus**。但是，您可以自定义执行过程以使用您选择的任何别名运行Abaqus。（详细信息请参阅[Abaqus安装和许可指南](../sgb/sgb-link.md#sgb)。）

### 约定

以下约定在这些章节中使用：
- 每个讨论都包含一个"命令摘要"部分，提供左侧列中命令的语法，右侧列中选项的语法。完整命令必须首先出现，然后是选项。在某些情况下，命令有多个单词，如**abaqus cae**；在发出任何选项语句之前，您必须输入命令的所有单词。
- 选项以**粗体**显示。它们可以以任何顺序出现，可以缩写。
- 默认选项带下划线（ __ ）。
- 方括号（[ ]）括起来的项目是可选的。
- 用竖杠（|）分隔的列表中出现的项目是互斥的。
- 必须从括在花括号（{ }）中的值列表中选择一个值。
- 您必须以*斜体*提供值。
- 空格用作选项之间的分隔符，不得在等号前后出现。
- 可以使用**-option** *value*的替代语法代替**option**=*value*格式。

如果命令行上未提供所需信息，**abaqus**过程将提示输入。如果**abaqus**未带选项键入，则会提示输入所有选项。

### 环境设置

Abaqus执行过程使用"环境"设置来自定义作业的执行。这些设置可以使用Abaqus环境文件`abaqus_v6.env`更改。运行作业时，执行过程除了在安装位置外还会两个位置查找此文件。首先查找的位置是您的主目录。如果存在，此文件中的设置将应用于您运行的所有作业。执行过程查找的第二个位置是当前目录。如果文件存在，则其中定义的设置将应用于从该目录运行的所有作业。

如果同一个作业参数在多个环境文件中定义，或在同一个环境文件中定义多次，则使用最后遇到的定义。此规则的某些例外情况在["使用Abaqus环境设置，"第3.3.1节"](pt01ch03s03aus30.md)中注明。这些环境文件可用于自定义Abaqus的行为，包括修改默认选项。有关环境文件的更多信息，请参见["使用Abaqus环境设置，"第3.3.1节"](pt01ch03s03aus30.md)。

### 选择TCP/UDP端口号

多个执行过程命令行选项（如**port**和**listenerport**）要求您指定端口号。TCP/UDP端口号范围可以从0到65535。

端口0到1023是由系统进程使用的常用端口（如FTP、SSH、SMTP等），不应使用。端口1024到49151是向互联网分配号码授权机构（IANA）注册的软件供应商注册端口。这些端口可以使用，但您应注意不要与可能使用此端口的系统上安装的任何软件冲突。端口49152到65535是未保留的，只要没有其他应用程序使用它们，就可以自由使用。

端口可能被防火墙阻止。请联系您的系统管理员以确保要指定的端口未被阻止。

您可以使用`netstat`命令获取有关TCP/UDP网络连接的信息。
