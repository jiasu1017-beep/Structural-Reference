# 1.1 Abaqus GUI Toolkit能做什么？





有多种方式可以定制Abaqus产品：
- 用户子程序允许您改变Abaqus/Standard和Abaqus/Explicit计算分析结果的方式。用户子程序的信息可以在[Abaqus用户子程序参考指南](../sub/sub-link.md#sub)中找到。
- 环境文件允许您更改各种默认设置。环境文件的信息可以在[Abaqus分析用户指南](../usb/usb-link.md#usb)中找到。
- kernel脚本允许您创建新功能来执行建模或后处理任务。kernel脚本的信息可以在[Abaqus脚本用户指南](../cmd/cmd-link.md#cmd)中找到。
- GUI脚本允许您创建新的图形用户界面。GUI脚本在本指南中有描述。

Abaqus GUI Toolkit提供了编程例程，允许您创建或修改GUI的组件。该工具包允许您执行以下操作：
- 创建新的GUI模块。GUI模块是类似功能的分组，例如Abaqus/CAE中的零件模块。
- 创建新的GUI工具集。GUI工具集与GUI模块类似，因为它将类似功能分组，但它通常包含可能被一个或多个GUI模块使用的更具体的功能。Abaqus/CAE中的基准工具是GUI工具集的示例。
- 创建新的对话框。Abaqus GUI Toolkit提供了完整的widget库，您可以从中构建自己的对话框。但是，Abaqus GUI Toolkit不能用于修改现有的Abaqus/CAE对话框。
- 移除Abaqus/CAE GUI模块和工具集。您可以选择在应用程序中包含哪些GUI模块和排除哪些GUI模块。例如，Abaqus/Viewer应用程序不包含建模相关的GUI模块；它只包含可视化模块。
- 移除某些顶级菜单或这些顶级菜单中的某些项目。例如，您可以移除整个**Viewport**顶级菜单以防止用户操作视口，或者可以从**File**菜单中移除**Import**和**Export**菜单项。
- 对Abaqus/CAE GUI模块和工具集执行有限的更改。有关更多信息，请参见["修改和访问Abaqus/CAE GUI模块和工具集"，第10.1节](pt05ch10s01.md)。

Abaqus GUI Toolkit不能独立于Abaqus/CAE运行；它必须与Abaqus/CAE一起使用才能使基础设施正常运行。
