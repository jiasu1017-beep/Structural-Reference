# 17.16.2 通过指定单元数量对边进行种子布置





您可以通过输入要创建的单元数量来定义沿所选边的单元尺寸。 Abaqus/CAE允许您选择边、面或单元进行种子布置。 但是，种子仅沿边放置——您选择的边或您选择的面和单元的边。

所有边种子工具都会生成边种子，以品红色显示。 边种子覆盖您指定的任何零件或实例种子。 您应该对所有边施加种子。

**通过指定单元数量对边进行种子布置：**

1. 从主菜单栏，选择 ****Seed**![](../graphics/images/arrow.gif)**Edges****。 Abaqus/CAE在提示区域显示提示以指导您完成该过程。 **提示：**您也可以使用Mesh模块工具箱中种子工具所在的 ![](../graphics/ico_meshSeedEdges.png) 工具通过指定单元数量对边进行种子布置。 （有关更多信息，请参阅["使用Mesh模块工具箱，" 第17.15节](pt03ch17s15.md)。）
2. 选择要种子布置的边、面或单元。 默认情况下，Abaqus/CAE仅允许您选择边进行种子布置。 要选择面或单元进行种子布置，请使用**Selection**工具栏将可选择的对象类型更改为**Face**、**Cells**或**All**。 有关更多信息，请参阅["根据对象类型过滤您的选择，" 第6.3.2节](pt01ch06s03s01.md)。
3. 完成选择边、面或单元后，在提示区域单击**Done**。 有关选择对象的更多信息，请参阅[第6章，"在视口中选择对象"](pt01ch06.md)。
4. 从出现的**Local Seeds**对话框中，选择**By number**。 **注意：**如果您选择的边之前使用多种种子布置方法的组合进行了种子布置，Abaqus/CAE会提供**As Is**选项，允许您保留所选边上的种子布置方法。
5. 输入沿每条边Abaqus应生成的单元数量。
6. 选择偏置控制为**None**。
7. 如果需要，在**Constraints**标签页上更改默认种子约束。 有关设置种子约束的更多信息，请参阅["对边种子施加约束，" 第17.16.5节](pt03ch17s16hlb05.md)。
8. 如果需要，切换**Create set with name**以创建包含您选择的边、面和单元的集合，并输入集合的名称。 如果您随后想要更改种子布置，可以在步骤2中选择该集合，而无需重新选择边、面和单元。
9. 单击**Apply**查看Abaqus/CAE将使用的种子布置。 品红色种子出现在所选边上。
10. 如果需要，调整**Local Seeds**对话框中输入的值。
11. 单击**OK**提交单元种子布置并关闭对话框。
