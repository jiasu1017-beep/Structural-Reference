# 11.14.2 定义放样路径







您创建的每个放样特征都需要至少一条放样路径。定义放样截面后，您可以选择修改放样路径。放样特征的路径将起始截面上的一个点连接到终止截面上的一个点。如果定义了超过两个放样截面，每条路径也会通过每个中间截面上的一个点。您可以使用**编辑放样**对话框的**过渡**选项卡页面上的选项来定义放样路径。

创建放样特征时，您可以选择以下方法来定义放样路径：

**指定切线**

**指定切线**是默认的放样路径定义方法。如果您选择**指定切线**，Abaqus/CAE会创建一条通过每个放样截面中心的单一平滑路径，如图11-51所示。您可以应用切线条件来修改放样特征在起始和终止放样截面附近区域的形状。有关放样切线的更多信息，请参见"定义放样切线"，第11.14.3节。

**图11-51** 由Abaqus/CAE定义路径的放样特征。

![](../graphics/prt-loft-auto.png)

**选择路径**

如果您选择**选择路径**，可以从现有边缘中选择来定义放样路径。此方法还允许您定义多条放样路径。放样特征通过沿路径连接一个放样截面到下一个放样截面来创建，如图11-52所示。具有单一选择路径的放样特征的行为与扫掠特征类似，只是放样的横截面不断变化以沿路径匹配下一个放样截面的位置和形状。

**图11-52** 具有单一用户定义路径的放样特征。

![](../graphics/prt-loft-userdef.png)

您必须从视口中的现有线段中进行选择，以创建连接所有放样截面的路径。每条路径必须是平滑曲线，并且必须按放样创建时将连接的相同顺序连接各截面。您可以使用![](../graphics/ico_partWirePoint.png)工具创建定义三维路径的样条线框。有关更多信息，请参见"添加点对点线框特征"，第11.23.2节。

一旦创建了放样特征，无论您选择哪种路径定义方法，都无法直接编辑路径。但是，如果您使用了**选择路径**，则可以使用特征操作工具集通过编辑创建线框顶点的特征来编辑创建每条样条线框的点。
