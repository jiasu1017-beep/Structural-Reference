# 58.6 创建表达式场







在属性模块、相互作用模块或载荷模块中，从主菜单栏选择 ****工具**![](../graphics/images/arrow.gif)**解析场**![](../graphics/images/arrow.gif)**创建****，使用表达式创建解析场。使用数学表达式定义的解析场称为表达式场。

**创建表达式场：**

1. 使用以下方法之一打开 **创建表达式场** 对话框：- 从主菜单栏，选择 ****工具**![](../graphics/images/arrow.gif)**解析场**![](../graphics/images/arrow.gif)**创建****。**提示：**您还可以点击 **解析场管理器** 中的 **创建**。- 从相互作用模块中的相互作用编辑器，点击 **定义** 或 **发射率分布** 字段旁边的 ![](../graphics/ico_analyticalField.png)。- 从载荷模块中的载荷、边界条件或预定义字段编辑器，点击 **分布** 字段旁边的 ![](../graphics/ico_analyticalField.png)。
2. 选择 **表达式场** 作为 **类型**，然后点击 **继续**。
3. 在 **名称** 文本字段中，输入表达式场的名称。有关命名对象的信息，请参见 ["使用基本对话框组件，" 第 3.2.1 节](pt01ch03s02s01.md)。
4. 在 **描述** 文本字段中，输入表达式场的描述。
5. 如果您想更改表达式场的局部坐标系，请点击 **编辑**，并使用以下方法之一：- 在视口中选择现有的基准坐标系。- 按名称选择现有的基准坐标系。1. 从提示区域，点击 **基准 CSYS 列表** 以显示基准坐标系列表。2. 从列表中选择一个名称，然后点击 **确定**。- 从提示区域点击 **使用全局 CSYS** 以恢复到全局坐标系。
6. 在编辑器的表达式窗口中使用以下程序组合来构建您的表达式：- 从编辑器左侧的列表中选择参数名称。可用的参数名称取决于您选择的坐标系类型：- **直角坐标**：**X**、**Y** 和 **Z** 分别是 *X*、*Y* 和 *Z* 轴上的值。- **圆柱坐标**：**R**、**Th** 和 **Z** 分别是 *R*、![](../graphics/usi_eqn00065.gif) 和 *Z* 轴上的值。- **球坐标**：**R**、**Th** 和 **P** 分别是 *R*、![](../graphics/usi_eqn00065.gif) 和 ![](../graphics/usi_eqn00226.gif) 轴上的值。参数名称出现在表达式窗口中。- 从编辑器右侧的 **运算符** 列表中选择操作、函数或常数。有关更多信息，请参见 ["表达式中的运算和函数概述，" 第 58.2.2 节](pt06ch58s02s02.md)。所选的操作、函数或常数出现在表达式窗口中。- 在表达式窗口中使用标准的鼠标和键盘编辑技术来定位光标和配置您的表达式。参数名称和运算符区分大小写。- 如有必要，调整您的 [表达式的语法](pt06ch58s01.md)；可能需要括号。- 点击 **清除表达式** 以清除表达式窗口中的整个表达式。
7. 点击 **确定** 创建表达式场并退出编辑器。
