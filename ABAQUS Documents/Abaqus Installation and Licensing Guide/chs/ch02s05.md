# 2.5 卸载 Abaqus 产品或许可证



本节介绍如何从计算机中删除 Abaqus FLEXnet 许可证或产品版本。

有关如何删除 Dassault Systèmes 许可证服务器的说明，请参阅*Dassault Systèmes 许可证服务器安装和配置指南*（`DSLS.pdf`）中的"卸载 Dassault Systèmes 许可证服务器"。

### 2.5.1 Linux 平台

以下过程说明如何从 Linux 平台删除 Abaqus 产品版本或 FLEXnet 许可证。

**删除 Abaqus FLEXnet 许可证：**

此过程必须在许可证服务器主机上执行，并且您必须以 root 或许可证服务器所有者的身份登录。

1. 在删除许可证服务器和许可证文件之前，必须使用 FLEXnet Licensing 工具 `lmdown` 终止许可证服务器（请参阅["lmdown，"第 3.6.2 节](ch03s06.md#sgb-chp-net-lic-lmdown)）。
2. 备份您希望保留的任何文件，例如您的 Abaqus 许可证文件（通常名为 `abaquslm.lic` 或 `simulialm.lic`）和 FLEXnet Licensing 服务器调试日志文件（通常名为 `simulialm.log`）。
3. 使用以下过程删除文件： ``` cd *simulia_dir* rm -rf License ```
4. 从计算机的系统启动文件中删除许可证服务器重启命令。请联系您的系统管理员或平台供应商，获取有关计算机系统启动文件的信息。

**删除 Abaqus 产品：**

您可以通过删除版本特定的目录及其所有从属文件来删除 Abaqus 版本。您的用户帐户必须具有删除 Abaqus 版本特定目录的权限。

1. 键入命令 ``` cd *abaqus_dir* ``` 其中 *abaqus_dir* 是 Abaqus 父目录。
2. 通过输入命令更改目录和文件的权限 ``` chmod -R 755 6.14-*n* ``` 其中 *n* 是您希望删除的更新版本。
3. 通过键入命令删除版本或维护交付 ``` rm -rf 6.14-*n* ``` 其中 *n* 是您希望删除的更新版本。
4. 指向已卸载 Abaqus 版本的链接将不再有效。您可以通过键入命令从 `Commands` 目录中删除无效链接 ``` rm *link* ``` 其中 *link* 是无效链接名称。或者，您可以通过键入命令从 Abaqus 父目录 (*abaqus_dir*) 中删除整个 `Commands` 目录 ``` rm -rf `Commands` ```

### 2.5.2 Windows 平台

您可以使用 Windows 提供的卸载快捷方式、**添加或删除程序**工具或 Abaqus 无提示卸载程序批处理文件来删除 Abaqus 产品或 FLEXnet 许可证。无提示卸载程序允许您从另一个批处理/脚本文件自动执行这些任务。无提示卸载程序仅在 Windows 平台上可用。您必须具有 Windows 管理员权限才能卸载 Abaqus 组件。

Windows 卸载快捷方式和无提示卸载程序批处理文件都会生成一个可查看的日志文件。日志文件指示卸载是否成功，以及是否应该重启/重新启动计算机。此信息显示在日志文件的最后两行。例如，

```
Uninstall Status: SUCCESS
Restart Needed: YES_RECOMMENDED
```
`Uninstall Status` 行将指示 `SUCCESS` 或 `INCOMPLETE`。`Restart Needed` 行将指示 `YES_RECOMMENDED`、`YES_REQUIRED` 或 `NO`。

卸载程序可能无法删除某些文件；例如，在其他应用程序中打开的文件。此外，卸载程序仅删除 Abaqus 安装程序写入的目录和文件。您可能希望手动删除任何剩余的目录和文件。

**删除 Abaqus FLEXnet 许可证：**

可以通过以下任何方式删除 FLEXnet 许可证：
- 从**开始**菜单，选择****程序**![](../graphics/images/arrow.gif)**SIMULIA FLEXnet 许可证**![](../graphics/images/arrow.gif)**卸载 SIMULIA FLEXnet 许可证****。
- 从**开始**菜单，选择****设置**![](../graphics/images/arrow.gif)**控制面板**![](../graphics/images/arrow.gif)**添加或删除程序****。选择 Abaqus 许可证，然后单击**更改或删除程序**。
- 要使用 FLEXnet 许可证无提示卸载程序，请从命令行或从另一个批处理/脚本文件执行以下批处理文件： ``` *simulia_dir*\License\installation_info\uninstaller\ silent_license_uninstall.bat ```

FLEXnet 许可证卸载程序生成以下日志文件：
```
*simulia_dir*\License\installation_info\uninstall_log.log
```
您应该备份您希望保留的任何文件，例如您的 Abaqus 许可证文件（通常名为 `abaquslm.lic` 或 `simulialm.lic`）和 FLEXnet Licensing 服务器调试日志文件（通常名为 `simulialm.log`）。

**删除 Abaqus 产品：**

可以通过以下任何方式删除 Abaqus 产品：
- 从**开始**菜单，选择****程序**![](../graphics/images/arrow.gif)**Abaqus *release***![](../graphics/images/arrow.gif)**卸载 Abaqus *release*****，其中 *release* 指 Abaqus 版本。
- 从**开始**菜单，选择****设置**![](../graphics/images/arrow.gif)**控制面板**![](../graphics/images/arrow.gif)**添加或删除程序****。选择 Abaqus，然后单击**更改或删除程序**。
- 要使用 Abaqus 产品无提示卸载程序，请从命令行或从另一个批处理/脚本文件执行以下批处理文件： ``` *simulia_dir*\*abaqus_dir*\*release*\installation_info\uninstaller\ silent_product_uninstall.bat ```

产品卸载程序生成以下日志文件：
```
*simulia_dir*\*abaqus_dir*\*release*\installation_info\uninstall_log.log
```
卸载程序会从所有用户配置文件中删除 Abaqus 快捷方式。
