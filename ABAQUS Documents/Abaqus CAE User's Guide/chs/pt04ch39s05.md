# 39.5 在装配中使用子结构部件实例




将子结构部件导入模型数据库后，您可以通过实例化它们将其添加到装配中，方式与为任何部件实例化相同。子结构部件实例在视口中以半透明颜色显示。

您可以移动子结构部件实例并对其应用约束；但是，子结构部件实例具有以下建模限制：
- 您不能为子结构部件实例分配截面。
- 您不能为子结构部件实例应用属性。
- 子结构部件实例不符合接触对定义条件。
- 重力载荷是唯一可以应用于子结构部件实例的载荷定义。
