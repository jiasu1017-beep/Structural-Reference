# 30.3 定义流体材料属性





流体材料及其物理属性决定了流体响应。您可以在属性模块中创建流体材料。根据分析的不同，您可能需要指定以下属性：

**密度**

必须在Abaqus/CFD分析的流体中定义密度；它是瞬态Navier-Stokes计算所必需的。对于不可压缩流动，流体密度被视为常数。

**粘度**

必须为粘性流动指定粘度，仅支持恒定粘度。

**比热容**

对于包含能量方程的不可压缩流体动力学分析，您必须定义比热容和热导率。需要使用恒压比热容；您可以定义恒压比热容，或者可以定义恒容比热容并指定理想气体常数。

**热导率**

仅支持恒定各向同性热导率。

**热膨胀系数**

热膨胀系数与自然对流的Boussinesq近似一起使用。

**渗透率**

您可以指定各向同性渗透率（仅依赖于孔隙度），或使用Carman-Kozeny关系指定渗透率。
