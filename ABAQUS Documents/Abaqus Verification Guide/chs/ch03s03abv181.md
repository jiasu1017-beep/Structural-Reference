# 3.3.2 二维湍流通道流动







**产品：**Abaqus/CFD  

### 测试的单元

FC3D8

### 测试的功能

Spalart-Allmaras湍流模型。

### 问题描述

使用平面通道中的二维湍流流动来验证Spalart-Allmaras模型。这个典型问题使用简单几何形状，允许与"壁面定律"进行直接比较。

实验证据和平板边界层和通道流动的量纲分析表明，当用适当的粘性单位归一化时，流体动力学完全发展的速度剖面塌陷为通用速度剖面。这个结果被称为壁面定律，由通道流动中解释的三个主要区域组成（见[Pope 2000](ch03s03abv181.md#ver-ref-pope)）。

| 内层： | 在这个区域，粘性应力占主导地位，平均速度剖面呈线性分布。 |
| --- | --- |
| 外层： | 在这个区域，湍流应力占主导地位，平均速度剖面呈对数分布。 |
| 缓冲区： | 在这个区域——一个过渡区——粘性应力和湍流应力都很重要。速度剖面呈平滑过渡，连接线性和对数区域。 |

### 总结

使用Abaqus/CFD成功计算了平面通道中的稳态不可压缩湍流流动。平均速度剖面与著名的壁面定律解非常好地一致。此外，测量了Abaqus/CFD摩擦速度的估计收敛率，发现与代码的理论二阶空间精度非常接近。
