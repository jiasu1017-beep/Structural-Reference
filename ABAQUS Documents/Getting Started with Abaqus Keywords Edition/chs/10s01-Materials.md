# 10.1 在 Abaqus 中定义材料

你可以在模拟中使用任意数量的不同材料。每个材料定义都以 `*MATERIAL` 选项开始。`NAME` 参数用于标识所定义材料的名称。此名称用于将材料定义分配给模型中的特定单元。

材料定义是 Abaqus 输入文件中少数几个选项块位置重要的情形之一。所有定义材料行为特定方面的选项块（如弹性模量或密度）必须直接跟在 `*MATERIAL` 选项之后。此外，定义特定材料行为的材料选项块不能被其他非材料选项打断。如果 Abaqus 无法将材料行为选项块（如 `*ELASTIC`）与之前的 `*MATERIAL` 选项关联起来，它会发出错误消息。

例如，考虑一种材料描述（如承受重力载荷的弹塑性金属），它需要多个材料行为选项块来向 Abaqus 提供必要的数据。除了弹性和塑性属性选项块之外，Abaqus 还需要材料的密度来计算重力载荷。因此，完整的材料描述如下所示。![材料输入示例](../graphics/gss-c8-property-input-nls.png)
