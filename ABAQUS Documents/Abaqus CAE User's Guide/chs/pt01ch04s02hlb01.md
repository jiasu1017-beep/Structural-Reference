# 4.2.1 从主菜单栏管理视口和视口注释





使用主菜单栏上的**视口（Viewport）**菜单可以创建、删除、修改或重新排列视口，以及创建或编辑视口注释——包括您自己创建的注释和由Abaqus/CAE生成的注释。如果愿意，您可以选择****视图（View）**![](../graphics/images/arrow.gif)**工具栏（Toolbars）**![](../graphics/images/arrow.gif)**视口（Viewport）****从主菜单栏显示一个工具栏，其中包含**视口**菜单中大部分的功能。

**视口**菜单和工具栏允许您执行以下操作：
- [创建](pt01ch04s03hlb01.md)视口。
- [编辑](pt05ch56.md)生成的视口注释的选项（三轴图例、图例、标题块和状态块）。
- 创建[箭头](pt01ch04s04hlb01.md)注释。
- 创建[文本](pt01ch04s04hlb02.md)注释。
- 创建[组合箭头和文本](pt01ch04s04hlb03.md)注释。
- 编辑[箭头](pt01ch04s04hlb05.md)和[文本](pt01ch04s04hlb06.md)注释。
- 打开**注释管理器**以操作箭头和文本注释。**注意：**注释管理器提供了几种独特的管理功能；有关更多信息，请参阅["注释管理器，"第4.2.3节"](pt01ch04s02hlb03.md)。
- 打开**视口注释选项**以显示或隐藏所有注释，并操作由Abaqus/CAE生成的视口注释。

此外，**视口**菜单列出会话中的所有视口，并允许您删除当前视口。![](../graphics/images/black4rule.gif)有关相关主题的信息，请单击以下项目：- [第4章，"在画布上管理视口"](pt01ch04.md)
