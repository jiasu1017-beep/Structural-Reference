# 3.1 FLEXnet 网络许可证



Abaqus 6.14 FLEXnet 许可证以网络许可证格式提供。网络许可证允许有限数量的分析作业和交互会话在连接到网络的任何受支持计算机上同时运行。Abaqus 网络许可证使用来自 Flexera Software（前身为 Acresso Software）的 FLEXnet 网络许可证管理器来控制许可证服务器，该服务器是网络上单台计算机（许可证服务器主机）上运行的一个进程。只要有可用的令牌，Abaqus 产品就可以在网络上的任何受支持计算机上运行，包括许可证服务器主机。

**注意：**计算机需要网卡或板载网络芯片才能作为许可证服务器运行。

可以选择多个 FLEXnet 许可证服务器以在许可证服务器或其主机发生故障时提供冗余。有关冗余许可证服务器的信息，请参阅["冗余 FLEXnet 许可证服务器配置，"第 3.4 节](ch03s04.md)。

网络上的任何受支持计算机都可以用于运行 FLEXnet 许可证服务器，只要运行 Abaqus 的计算机能够与许可证服务器通信即可。运行 Windows 的计算机可以作为 Linux 网络的许可证服务器主机，反之亦然。

每个 Abaqus 分析或交互会话必须在开始执行之前联系许可证服务器，并在 Abaqus 运行期间定期联系。因此，在整个分析或交互会话期间，Abaqus 必须与许可证服务器保持不间断的通信。如果许可证服务器或其运行的计算机发生故障，应尽快重新启动，以确保证书服务器的运行中断最小。

许可证活动记录在称为服务器日志的日志文件中。管理许可证活动的工具在["FLEXnet Licensing 管理工具，"第 3.6 节](ch03s06.md)和 FLEXnet Licensing End User Guide Version 11.6 中进行了描述；后者可从 **Support** 页面上的 **Licensing** 部分下载，网址为 [www.3ds.com/simulia](http://www.3ds.com/simulia)。

可以对令牌应用限制以控制对 Abaqus 的访问。可以为某些用户保留一些令牌，或拒绝特定用户和计算机的访问。这些限制在网络许可证选项文件中定义。请参阅["使用 FLEXnet 选项文件，"第 3.7 节](ch03s07.md)，了解更多信息。您还可以通过 Abaqus 环境文件中的设置自定义许可证服务器的行为。请参阅["许可证管理参数，"第 4.1.6 节](ch04s01.md#sgb-chp-env-net)，了解环境文件设置的信息。

FLEXnet 许可证服务器持续跟踪网络上哪些令牌"可用"哪些令牌"正在使用"。在开始分析或交互会话之前，Abaqus 作业（客户端）从许可证服务器请求令牌。如果池中有可用令牌，许可证服务器会将令牌从可用池转移到正在使用池，并且令牌将保留在正在使用池中直到分析或交互会话结束。

如果没有可用令牌，分析作业或交互会话可以根据 **lmlicensequeuing** 和 **lminteractivequeuing** 参数的设置进入许可证队列等待令牌；请参阅["许可证管理参数，"第 4.1.6 节](ch04s01.md#sgb-chp-env-net)。如果没有可用令牌且 **lmlicensequeuing** 和 **lminteractivequeuing** 参数设置为 `OFF`，则分析作业或交互会话将退出并显示错误消息。在分析或交互会话正常完成时，Abaqus 客户端会通知许可证服务器，服务器会将令牌返回到可用池。如果分析或交互会话中止，令牌会立即返回。

有关解决 FLEXnet 许可证相关问题的帮助，请参阅[附录 F，"排除 Abaqus FLEXnet 许可证故障"](ap06.md)。
