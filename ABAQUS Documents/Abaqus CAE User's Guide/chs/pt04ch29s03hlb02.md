# 29.3.2 定义紧固件连接区域





您可以使用**Criteria**标签页指定Abaqus在分析模型时用于确定紧固件投影位置的材料。

**要定义紧固件连接区域：**

1. 从**Edit Fasteners**对话框中，显示**Criteria**标签页。
2. 选择**Attachment method**。一般来说，定位点应尽可能靠近要连接的面。Abaqus通过首先将定位点投影到最近的面来确定紧固件的实际紧固点。您可以从以下投影方法中选择：
   - **Face-to-Face**
   - **Face-to-Edge**
   - **Edge-to-Face**
   - **Edge-to-Edge**
   您选择的方法取决于面的相对方向。更多信息，请参见["指定定位点、投影方法和紧固点"中的"独立于网格的紧固件"，Abaqus Analysis User's Guide第35.3.4节](../usb/usb-link.md#usb-cni-afastener-points)。
3. 选择指定紧固点必须位于定位点距离内的方法。
   - 选择**Facet-based default**以允许Abaqus根据每个定位点附近的面片厚度（对于壳单元）或特征面片长度（对于非壳单元）计算默认搜索半径。
   - 选择**Specify**，然后输入紧固点必须位于其内的距定位点的最大距离。更多信息，请参见["在用户指定的搜索区域内表面上形成紧固件"中的"独立于网格的紧固件"，Abaqus Analysis User's Guide第35.3.4节](../usb/usb-link.md#usb-cni-afastener-zone)。
4. 选择紧固点将被投影的最大层数。
   - 选择**All**以允许Abaqus使用最大可能数量的层。
   - 选择**Specify**，然后输入层数。
