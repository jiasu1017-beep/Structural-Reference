# 3.9.1 弯管单元

### 3.9.1 弯管单元

**产品：** Abaqus/Standard

管道系统的设计严重依赖使用弯管（管弯头）来提供对热应变的柔性响应。出于这个原因，弯管本身是设计中的关键部件，准确预测其响应对于管道结构的资格认证至关重要。在事故条件下尤其如此，此时非弹性行为——塑性或粘塑性——主导响应。众所周知，管道弯头通过壳型行为实现其柔性——通过管道横截面的显著椭圆化来响应弯曲载荷——这与直管的梁响应形成对比，在直管中横截面不会发生任何显著变形，直到Brazier屈曲发生。椭圆化赋予弯头在弯曲中比相同尺寸直管高5-20倍的弹性柔性（对于核反应堆典型主管道回路弯头）。这种柔性伴随着通常在相同载荷下直管3-12倍的应力和应变——当然——由横截面椭圆化引起的管道周围和穿透其厚度的高应变梯度。出于线性弹性分析的目的，这些影响几乎没有困难，因为柔度因子和应力强度因子已经被精确计算并通过实验验证，因此直接用于使用三维梁理论的响应预测（见[Dodge和Moore，1972](07s01a01-References.md)，对该工作的综述）。

对于非线性分析或非圆形截面管道，这些解是无用的。对于非线性情况提出了两种可能的方法：可以寻求线性弹性柔度和应变强度因子的非线性等价物，或者可以进行基于管道弯头壳型行为的详细分析。许多研究者已经研究了线性弹性柔度和应力强度因子的非线性等价物概念（例如，[Spence，1972](07s01a01-References.md)），尽管这种方法的成功尚不能保证其在设计中的使用（[Scheller和Mallett，1979](07s01a01-References.md)）。因此，详细非弹性分析仍然是作为不能基于弹性基础设计的管道系统资格认证的明显方法。这种分析的主要困难在于工程时间和计算机时间方面的成本。结构是一个表现出严重应力梯度的壳，因此提供足够精度详细分析所需的数值模型是大型且复杂的。已经进行了许多尝试来提供这种模型。[Hibbitt（1974）](07s01a01-References.md)描述了一种壳型有限元的公式，该元使用沿管道分段三次位移插值和沿管道分段常数应变。后者的简化导致了沿管道的不良位移不连续性。然而，该方法具有记录在案的精度（[Sobel，1977](07s01a01-References.md)），已被用于设计评估（[Chen和Weiner，1978](07s01a01-References.md)）。[Ohtsubo和Watanabe（1976）](07s01a01-References.md)基于薄壁圆环截面小应变的虚功原理开发了"环"几何形状的壳单元。他们用完整的Fourier级数——截断M项后——沿管道插值壳中面法线和两个切向位移分量，并沿管道分段三次。他们的模型允许横截面的椭圆化和翘曲，因此可能高精度地描述管道行为。他们的结果表明，对于典型主管道弯头，需要沿管道使用四到六个Fourier模式来获得收敛解，在较不柔性的弯头中需要更少的模式。然而，他们的公式不适合于多弯头管道中的实际应用，因为应变来自于弯头的刚体运动。这是由于在曲线坐标系中使用多项式和Fourier位移分量插值的圆柱壳理论。

[Takeda等人（1979）](07s01a01-References.md)开发了Ohtsubo和Watanabe壳单元的扩展，沿管道使用Fourier和分段三次插值。该单元已被应用于一些有据可查的非线性基准问题（例如，[Kwata，1978](07s01a01-References.md)），并被证明是准确的但相对昂贵。在[Kwata（1978）](07s01a01-References.md)中，Takeda等人将该单元与Hughes等人（1977）的单点积分双线性插值壳单元进行比较，得出结论：对于给定的精度，后者更经济。在很大程度上，这个结论似乎取决于实现该单元的计算机代码的架构：除非代码架构允许经济地计算这种单元，否则它们将非常昂贵。在[Kwata（1978）](07s01a01-References.md)中，简单的双线性壳被显示需要沿管道约40个单元（因此40个本构点）来提供准确的应力（该论文实际上显示了在面内弯曲情况下沿半管20个单元），因此每个段引入约240个自由度；而在[Ohtsubo和Watanabe（1976）](07s01a01-References.md)中，六个Fourier模式（因此对于面内建模12个自由度，对于面外建模每个段24个自由度）给出相同水平的精度——同样约40个本构计算点。使用Abaqus中的公式，我们还得出结论，需要沿管道32-48个本构计算站。每个段自由度的数量对比表明——尽管像Ohtsubo和Watanabe的单元相对复杂——给定设计用于有效处理这种单元的合适代码架构，利用弯头的特定几何和行为优势的单元应该比大多数简单壳单元（如Hughes等人的）明显更便宜。

[Bathe和Almeida（1980）](07s01a01-References.md)的单元是经典von Karmann方法的直接有限元实现。截面的翘曲被忽略，在横截面平面内只允许管道壁的纯弯曲。后者的限制排除了将热载荷直接应用于该单元。此外，它不允许Gross添加到von Karmann理论的"周向膜力修正"（见[Dodge和Moore，1972](07s01a01-References.md)）。作为额外的简化，沿管道壁厚度的轴向和剪切应变梯度被忽略。沿管道，使用Fourier级数表示管道壁中面在管道截面平面内的切向位移分量。在该单元中只使用偶数Fourier项![](../graphics/stm_eqn05093.gif)，如von Karmann分析中所做的那样。这意味着这些位移关于弯头的冠部是对称的（关于[图3.9.1-1](03s09a92.md)中的![](../graphics/stm_eqn05094.gif)）。当管道半径相对于弯头环面半径较小时这是令人满意的，但当管道半径与环面半径相当时可能不足够。该单元沿管道的插值是分段Lagrange三次式，而不是[Takeda等人（1979）](07s01a01-References.md)的Hermite三次式，因此每个段引入了相当多的自由度。这个选择的原因在[Bathe和Almeida（1980）](07s01a01-References.md)中不明显。作者将自己限制在3个Fourier模式和沿管道24个积分站（本构计算点）；该单元可能需要一些扩展以允许更多的Fourier模式和沿管道的本构计算点，以应用于更柔性的弯头。

在Abaqus/Standard中，我们在上述经验基础上构建，以合理的成本提供管道详细非弹性分析的能力。在合理的计算机时间内实现这种复杂响应预测需要几何建模尽可能高效——即，模型被"调整"到特定结构几何和响应的特征。管道弯头表现为壳；然而，它们有一个简化特征，即沿管道的应变梯度通常比沿管道周围的梯度要轻。Abaqus中用于弯管单元的几何建模选择利用这个简化来最小化与管道非线性分析相关的计算机成本。
