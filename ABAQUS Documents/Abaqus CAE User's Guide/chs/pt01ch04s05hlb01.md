# 4.5.1 使用链接视口





链接视口使你能够同时在不同的视口中操作对象的视图。当你在一个链接视口中使用[第 5 章，"操作视图和控制透视"](pt01ch05.md)中所述的任何视图操作工具操作对象时，Abaqus/CAE 会在会话中的所有链接视口中执行相同的操作。

Abaqus/CAE 只允许一组链接视口，所以会话中的每个视口要么是独立的，要么是链接视口组的一部分。当你在独立视口中更改视图时，Abaqus/CAE 只更改该视口中的视图。

当使用标准视图操作工具时（请参阅["理解视图操作工具，" 第 5.2 节](pt01ch05s02.md)），链接视口中的操作不依赖于每个视口中的视图方向。例如，向左平移视图会在所有链接视口中向左平移，而不管每个视口中的视图方向如何。当使用 3D 罗盘操作视图时（请参阅["3D 罗盘，" 第 5.3 节](pt01ch05s03.md)），链接视口中的操作依赖于每个视口中的视图方向。例如，沿 *X* 轴平移视图会在所有链接视口中沿 *X* 轴平移；每个视口中的视图方向决定了运动的方向。

你可以从**链接视口**对话框控制视口链接，该对话框提供以下选项：
- 你可以为会话启用或禁用视口链接。如果此选项被禁用，会话中的所有视口都是独立的。如果此选项被启用，对话框底部选择的视口将被链接，未选择的视口保持独立。

- 你可以将任何视口包含在链接视口组中或从中排除。切换视口的复选框以将其链接到其他链接视口。

- 你可以控制链接视口共享的特性。某些链接视口选项在所有模块中都可用；有些选项（如在所有链接视口中显示相同的绘图状态）仅适用于可视化模块。
