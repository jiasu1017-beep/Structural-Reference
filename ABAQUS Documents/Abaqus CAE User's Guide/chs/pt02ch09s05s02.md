# 9.5.2 重新创建已保存的模型数据库





当您保存模型数据库（通过从主菜单栏选择****文件（File）**![](../graphics/images/arrow.gif)**保存（Save）****或****文件（File）**![](../graphics/images/arrow.gif)**另存为（Save As）****）时，Abaqus/CAE还会保存一个模型数据库日志文件（`*model_database_name*.jnl`），其中包含将重新创建模型数据库的Abaqus Scripting Interface命令。如果保存的模型数据库损坏，您可以通过使用恢复选项启动Abaqus/CAE来重新创建它。（输入`abaqus cae recover=`*model_database_name*.jnl``。）恢复选项执行指定的模型数据库日志文件中的命令。

模型数据库日志文件与重放文件的不同之处在于，它不包含会话期间执行的每个操作。模型数据库日志文件仅包含更改保存的模型数据库的命令；例如，创建或编辑零件、更改分析步骤的时间增量或修改网格的命令。更改模型数据库的操作不会保存在日志文件中；例如，将图像发送到打印机、创建视口、旋转模型或在可视化模块中查看结果。

当您继续处理模型时，内存中的模型数据库将与最近保存的模型数据库不同。模型数据库日志文件仅在您使用****文件（File）**![](../graphics/images/arrow.gif)**保存（Save）****或****文件（File）**![](../graphics/images/arrow.gif)**另存为（Save As）****对模型数据库执行明确保存时才会更新。如果您将模型数据库复制到其他位置，也应该复制关联的模型数据库日志文件。否则，您将无法重新创建模型数据库。
