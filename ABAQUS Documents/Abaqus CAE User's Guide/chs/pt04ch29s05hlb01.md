# 29.5.1 将模板模型表面映射到主模型表面





您可以使用**Surfaces**标签页将模板模型中的约束表面分配给将在您的主模型中紧固的相应目标表面。约束表面必须映射到主模型中以启用主模型中的约束行为。

参与模板模型约束的所有表面必须已分配名称。主模型中对应的目标表面也必须具有名称。

**要将模板模型表面映射到主模型：**

1. 从**Edit Fasteners**对话框中，显示**Surfaces**标签页。
2. 在**Constraint Surface Substitutions**表中，将每个**Template Surface**分配给主模型中的**Assignment Surface**。**Template Surface**列将使用模板模型中参与绑定约束、耦合约束或调整点约束的任何模板模型表面名称进行预填充。如果模板表面顺序正确，您可以简单地选择在**Assignment Surface**列中主模型表面名称。如果您需要更改要分配的模板模型表面的顺序，请单击每个表格单元格，然后单击出现的箭头以显示可用表面名称列表。模板表面最初按升序*Z*轴顺序列出，但您可以更改顺序以与模板紧固件设计重合。但是，除了第一个表面的选择外，表面的顺序并不重要，因为相应分配表面法线用于定向组装紧固件。要为每个模板表面选择主模型分配表面，请单击相应表格单元格，然后单击出现的箭头以从主模型中的可用表面列表中选择。如果您还没有为需要使用的任何表面分配名称，请单击![](../graphics/ico_surface.png)（**Create Assignment Surface**）来定义新表面并将其添加到列表中。
3. 如果需要，可以切换**Render template surfaces**以在主模型中显示表面。
