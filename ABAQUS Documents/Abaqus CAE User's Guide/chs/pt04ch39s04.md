# 39.3 为子结构指定保留的节点自由度和载荷情况




在为分析定义了子结构生成步骤或多个步骤之后，必须为子结构定义**Retained nodal dofs**边界条件。子结构节点的保留自由度是外部的且可用于分析的自由度；指定节点的所有其他自由度被认为是子结构内部的，不计入分析。当您从此分析将子结构导入模型以进行子结构使用时，Abaqus/CAE将这些节点显示为浅蓝色十字准线，使您可以轻松地从部件实例或装配中选择它们。

如果您想要在与子结构的保留自由度不同的位置应用载荷，可以在子结构生成步骤中定义载荷情况。
