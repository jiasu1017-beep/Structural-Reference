# 1.19.4 使用 XFEM 模拟单边缺口的动态剪切失效

**产品：** Abaqus/Standard

### 问题描述

此示例验证并说明了在 Abaqus/Standard 中使用扩展有限元方法（XFEM）预测具有边缘裂纹的板的动态裂纹扩展。标本受到高应变速率剪切冲击载荷。呈现的裂纹路径和裂纹起始角度与 Kalthoff 和 Winkler（1987）的实验结果进行了比较。

### 几何和模型

研究了具有单边裂纹的板。标本如图 1.19.4-1（[图 1.19.4-1](ch01s19ach136.md#xfem-shearbar-geometry)）所示，尺寸 L = 0.003 m，W = 0.0015 m，初始裂纹长度 a = 0.0015 m。标本的下部沿水平方向受到脉冲载荷，模拟为规定速度：

![](../graphics/bmk_eqn01257.gif)

其中 ![](../graphics/bmk_eqn01258.gif) = 25 m/s，![](../graphics/bmk_eqn01259.gif) = 1.0×10⁻⁷ s。

### 材料

富集单元中体积材料特性的材料数据为 ![](../graphics/bmk_eqn01065.gif) = 3.24 GPa，![](../graphics/bmk_eqn00406.gif) = 1190 kg/m³，![](../graphics/bmk_eqn00114.gif) = 0.35。

指定了模型中富集单元的内聚力行为响应。选择最大主应力失效准则用于损伤起始，选择基于幂律断裂准则的基于能量的损伤演化规律用于损伤扩展。相关材料数据如下：![](../graphics/bmk_eqn01260.gif) = 100.0 MPa，![](../graphics/bmk_eqn01261.gif) = 700 N/m，![](../graphics/bmk_eqn01262.gif) = 700 N/m，![](../graphics/bmk_eqn01263.gif) = 700 N/m，![](../graphics/bmk_eqn01264.gif) = 1.0，![](../graphics/bmk_eqn01265.gif) = 1.0，![](../graphics/bmk_eqn01266.gif) = 1.0。

[图 1.19.4-2](ch01s19ach136.md#xfem-shearbar-crack-profile) 显示了 ![](../graphics/bmk_eqn01105.gif) = 6.0×10⁻⁶ s 时的裂纹轮廓。裂纹以 62° 的角度扩展，与 65° 的实验结果合理一致。
