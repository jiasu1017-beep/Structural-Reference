# *LATENT HEAT









### *LATENT HEAT指定潜热。

此选项用于指定材料的潜热。

**产品：**Abaqus/Standard  Abaqus/Explicit  Abaqus/CAE  

**类型：**模型数据  

**级别：**模型  

**Abaqus/CAE：**属性模块

##### **参考：**

- ["潜热，" Abaqus Analysis User's Guide第26.2.4节](../usb/usb-link.md#usb-mat-clatentheat)

### **可选参数：**

PORE FLUID

此参数仅适用于Abaqus/Standard分析。

如果正在定义多孔介质中孔隙流体的潜热，请包含此参数。

### **定义材料潜热的数据行：**

**第一行：**

根据需要重复此数据行，以定义材料中的相变；每行一个相变。潜热值必须按温度升序给出。
