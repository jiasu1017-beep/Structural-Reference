# 29.100 Trs 对象

Trs 对象定义时间历史粘弹性分析的温度-时间偏移。

**访问**

```
import material
mdb.models[*name*].materials[*name*].viscoelastic.trs
mdb.models[*name*].materials[*name*].viscosity.trs
import odbMaterial
session.odbs[*name*].materials[*name*].viscoelastic.trs
session.odbs[*name*].materials[*name*].viscosity.trs
```

### 29.100.1 Trs(...)

此方法创建 Trs 对象。

**路径**

```
mdb.models[*name*].materials[*name*].viscoelastic.Trs
mdb.models[*name*].materials[*name*].viscosity.Trs
session.odbs[*name*].materials[*name*].viscoelastic.Trs
session.odbs[*name*].materials[*name*].viscosity.Trs
```

**必需参数**

无。

**可选参数**

*definition*

SymbolicConstant，指定偏移函数的定义。可能的值为 WLF、ARRHENIUS 和 USER。默认值为 WLF。

*table*

Float 元组序列，指定下述项目。默认值为空序列。

此参数仅在 *definition*=WLF 时有效。

**表格数据**

- 参考温度，![](../graphics/ker_eqn00376.gif)。
- 校准常数，![](../graphics/ker_eqn00338.gif)。
- 校准常数，![](../graphics/ker_eqn00340.gif)。

**返回值**

Trs 对象。

**异常**

无。

### 29.100.2 setValues(...)

此方法修改 Trs 对象。

**必需参数**

无。

**可选参数**

`setValues` 的可选参数与 [Trs](pt01ch29pyo100.md#ker-trs-trs-pyc) 方法的参数相同。

**返回值**

无

**异常**

无。
