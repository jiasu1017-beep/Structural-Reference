# 31.2.1 轮廓积分分析概述





您可以使用轮廓积分估计来研究准静态问题中的开裂起始；但是，轮廓积分估计不能预测裂缝将如何扩展。您可以为二维或三维模型计算轮廓积分，并且您可以选择执行以下类型的轮廓积分计算：
- *J*积分
- <img src="../graphics/usi_eqn00701.gif">积分（用于蠕变）
- *T*应力（用于线性材料）
- 线性均匀材料和位于两个线性均匀材料界面上的界面裂缝的应力强度因子

更多信息，请参见["轮廓积分评估"，Abaqus Analysis User's Guide第11.4.2节](../usb/usb-link.md#usb-anl-acontintegral)。

您在Interaction模块中定义裂缝。要执行轮廓积分分析，您必须选择裂缝前缘、裂缝尖端或裂缝线以及裂缝扩展方向，如以下各节所述。您可以选择的实体取决于部件是二维还是三维，以及您是使用几何体还是使用来自孤立网格的单元和节点定义部件。在某些情况下，裂缝尖端或裂缝线与您选择的裂缝前缘相同，Abaqus/CAE为您选择裂缝尖端或裂缝线。

二维模型中的裂缝是包含可以自由分离的边缘的区域。三维模型中的裂缝是包含可以自由分离的面的区域。执行轮廓积分分析的最简单方法是使用已经包含可以自由分离的边缘或面的区域作为裂缝分离。或者，您可以将裂缝建模为嵌入在二维模型面中的线或嵌入在三维模型单元中的面。嵌入的线或面称为裂缝，您可以使用裂缝执行轮廓积分分析。当您对模型进行网格划分时，Abaqus/CAE在裂缝上创建重复的重叠节点；这些重合节点可以自由分离，因为裂缝会分离。有关裂缝的更多详细信息，请参见["什么是裂缝？"，第31.1.1节](pt04ch31s01s01.md)。[图31-3](pt04ch31s02s01.md#eng-crack-overview)显示了一个具有可以自由分离的现有边缘的二维模型和一个具有可以自由分离的嵌入裂缝的类似模型。

**图31-3** 您可以使用现有区域或使用裂缝定义裂缝。

![](../graphics/eng-crack-overview-nls.png)

您可以指定裂缝前缘位于对称平面上，在这种情况下，您只需对结构的一半进行建模。Abaqus将轮廓积分计算的值加倍以获得正确的值。在大多数情况下，您需要在裂缝周围使用细化网格。创建裂缝后，您必须使用**History Output Request**编辑器来请求Abaqus将轮廓积分信息写入输出数据库。更多信息，请参见["轮廓积分输出"，第31.2.6节](pt04ch31s02s06.md)。

详细说明请参见["创建轮廓积分裂缝"，第31.2.9节](pt04ch31s02hlb02.md)。
