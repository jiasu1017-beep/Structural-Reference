# 5.1.13 定义切割面







**产品：**Abaqus/Standard、Abaqus/Explicit  

### 测试单元

B21、B22、B31、B32、PIPE21、PIPE31

C3D4、C3D6、C3D8、C3D8T、C3D20R、C3D20RT、C3D10M、C3D10MT

### 测试功能

切割面被定义为通过平面切割单元集来创建类横截面表面。产生的表面形成在单元小面、边缘或端部上，以便成为指定平面的近似。该内部表面可用于预紧和请求输出跨该表面传递的力和扭矩等目的。

### 问题描述

这些测试通过定义切割面来验证在各种单元类型（连续体、结构、热传递和刚性单元）上创建类横截面表面。

### 结果与讨论

产生的表面通过Abaqus/CAE的Visualization模块进行后处理可视化验证。
