# 61.5.1 Hiding toolbars







Select ****Tools**![](../graphics/images/arrow.gif)**Customize**** from the main menu bar to set the visibility of toolbars in Abaqus/CAE. For other methods of hiding toolbars, see ["Configuring the visibility of toolbars," Section 61.1](pt06ch61s01.md).

**To set the visibility of toolbars:**

1. From the main menu bar in any module, select ****Tools**![](../graphics/images/arrow.gif)**Customize****. The **Customize** dialog box appears.
2. On the **Toolbars** tabbed page, toggle on the name of any toolbar you want to display; toggle off the name of any toolbar you want to hide.
3. When you have set the visibility for all toolbars as desired, click **Dismiss** to close the **Customize** dialog box.
