# 70.5.1 使用通过位置指定参数方法分区边







您可以通过在分区位置点击边上任意位置来沿所选边的长度分区边。Abaqus/CAE 将位置转换为参数。因此，您可以使用特征操作工具集修改分区的位置。

**使用通过位置指定参数方法分区边：**

1. 从主菜单栏中选择 ****Tools**![](../graphics/images/arrow.gif)**Partition****（工具→分区）。出现 **Create Partition**（创建分区）对话框。Abaqus/CAE 在提示区域显示提示以指导您完成过程。**提示：**您也可以使用位于模块工具箱分区边工具处的 ![](../graphics/ico_ptnEdgeLocation.png) 工具来使用通过位置指定参数方法分区边。 有关工具箱中分区工具的图表，请参阅["使用分区工具集，" 第 70.2 节"](pt06ch70s02.md)。
2. 从对话框顶部的 **Type**（类型）单选按钮中选择 **Edge**（边）。**Method**（方法）列表显示可用于分区边的方法。
3. 从方法列表中选择 **Specify parameter by location**（通过位置指定参数）。
4. 在边上任意位置点击。Abaqus/CAE 高亮显示边和您选择的位置。
5. 如果分区位置不正确，再次在边上点击。
6. 如果分区位置正确，点击提示区域中的 **Create Partition**（创建分区）。Abaqus/CAE 创建分区。
