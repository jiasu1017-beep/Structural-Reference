# 11.3.1 零件与特征的关系





在Abaqus/CAE中创建的零件具有基于特征的表示。特征是设计的有意义的部分，为工程师提供了一种方便自然的方式来构建和修改零件。在Abaqus/CAE中创建的零件由有序的特征列表和定义每个特征几何的参数构建而成。您可以选择以下形状特征在零件模块中构建零件：
- 实体
- 壳
- 线框
- 切割
- 混合

使用零件模块中的工具，您可以创建和编辑描述模型中每个零件所需的所有特征。Abaqus/CAE存储每个特征并使用此信息来定义整个零件，在您修改零件时重新生成零件，并在装配模块中生成零件的实例。有关零件与零件实例之间关系的更多信息，请参见"什么是零件实例？"，第11.3.4节。

以下序列说明了如何利用Abaqus/CAE中可用的特征构建[图11-1]中的三维零件：

**图11-1** 使用实体、壳、线框、切割和混合特征构建的零件。

![](../graphics/prt-all-features-round.png)

1. 构建零件时创建的第一个特征称为基础特征；您通过添加更多特征来构建零件的其余部分，这些特征可以修改或向基础特征添加细节。在此示例中，基础特征是一个U形零件；用户绘制了一个二维轮廓并将其拉伸以形成基础特征，如图所示。**图11-2** 基础特征。![](../graphics/prt-all-features-base.png)草图和拉伸深度(*a*)是定义基础特征的可修改参数。您可以使用特征操作工具集重新访问基础特征并更改其尺寸或形状，以修改截面草图或拉伸距离。如果需要，您可以删除基础特征并绘制新形状。
2. 加强腹板作为壳特征添加。用户在内表面之一上绘制了一条线，并将草图拉伸到相对的面，如图所示。草图是定义壳特征的唯一可修改参数。**图11-3** 壳特征。![](../graphics/prt-all-features-shell-nls.png)
3. 杆作为线框特征添加到角落。线框是通过连接用户选择的两个点创建的，如图所示。以这种方式创建的线框没有可修改的参数；如果需要更改，必须将其删除并重新创建。**图11-4** 线框特征。![](../graphics/prt-all-features-wire.png)
4. 盲孔切割到夹具顶部。用户绘制了一个二维轮廓，轮廓通过指定距离拉伸到夹具中，如图所示。草图和槽深度是定义盲孔切割特征的可修改参数。**图11-5** 切割特征。![](../graphics/prt-all-features-cut-nls.png)
5. 切割的边缘被圆角化。用户选择了要圆角的边缘并提供了圆角的半径，如图所示。半径是定义圆角特征的可修改参数。**图11-6** 圆角特征。![](../graphics/prt-all-features-round.png)

如果新特征的几何依赖于现有特征，Abaqus/CAE会在特征之间创建父子关系。新特征是子级，它所依赖的特征是父级。例如，在上述零件中，圆角特征是切割特征的子级。如果更改切割的位置或尺寸，边缘将保持圆角。类似地，如果删除切割，Abaqus/CAE也会删除圆角。

如果修改父特征，修改可能会使父特征的子特征失效。例如，在上述零件中，如果您增加切割深度使其成为通孔，您将失去沿其边缘的圆角；即，圆角将在修改后无法重新生成。Abaqus/CAE为您提供以下两种选择：
- 保留对父特征的更改，但抑制未能重新生成的特征。被抑制特征的子特征也将被抑制。
- 中止对父特征的修改并返回到最后一次成功重新生成的状态。
