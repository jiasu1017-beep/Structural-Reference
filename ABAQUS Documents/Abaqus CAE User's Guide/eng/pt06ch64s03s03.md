# 64.3.3 Reducing element distortion and collapse during mesh offsetting







Abaqus/CAE creates an offset mesh by 
- offsetting the nodes normal to the boundary of an existing mesh surface, and
- building elements that propagate out in the normal direction.

 When you create a mesh that is offset from a concave mesh surface, the elements tend to converge, as shown in [Figure 64--11](pt06ch64s03s03.md#edt-offset-fanin).

**Figure 64–11** Elements converge when offset from a concave surface.

![](../graphics/edt-offset-fanin-nls.png)

 As a result, elements may collapse or become inverted, and the magnitude of the offset is limited by a combination of the degree of curvature and the element size, as shown in [Figure 64--12](pt06ch64s03s03.md#edt-offset-limit). 

**Figure 64–12** When you offset small elements in a concave region, the elements may collapse or become inverted.

![](../graphics/edt-offset-limit-nls.png)

You can try to avoid element collapse and element inversion by selecting **Constant thickness around corners** from the **Offset Mesh** dialog box. When this option is selected, Abaqus/CAE tapers the offset elements that propagate from sharp corners. The resulting elements have a constant nodal offset distance as opposed to a constant distance between the layers of the element faces. [Figure 64--13](pt06ch64s03s03.md#edt-offset-compare) illustrates the effect on the offset mesh pattern of requesting a constant thickness around corners. 

**Figure 64–13** The effect on the offset mesh pattern of requesting a constant thickness around corners.

![](../graphics/edt-offset-compare-nls.png)

The effect on the offset mesh pattern is similar for both solid and shell offset meshes. You can also reduce element distortion by using a larger element size in the location of the concave region. However, you can control the element size only if you can regenerate the mesh from which you are offsetting.

In contrast, when you create a mesh that is offset from a convex mesh surface, the elements tend to fan out, as shown in [Figure 64--14](pt06ch64s03s03.md#edt-offset-fanout).

**Figure 64–14** Elements diverge when offset from a convex surface.

![](../graphics/edt-offset-fanout-nls.png)
