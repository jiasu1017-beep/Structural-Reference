# 55.11 在可视化模块中控制约束的显示





在可视化模块中，您可以使用显示组和**ODB Display Options**对话框选择性地控制分析约束的显示。这些控件在调试包含大量约束的模型时非常有用。通过打开和关闭某些约束的显示，您可以更轻松地理解复杂模型。

显示或隐藏约束有两个级别的控制：
- 您必须首先将所需的约束添加到显示组中，以便它们在视口中可见。您可以创建和编辑显示组以包含或排除不同的约束。有关显示组的更多信息，请参阅["创建或编辑显示组，" 第78.2.1节](pt07ch78s02hlb01.md)。
- 可以在**ODB Display Options**对话框的**Constraints**选项卡中选择或隐藏不同类型的约束。从主菜单栏选择****View**![](../graphics/images/arrow.gif)**ODB Display Options****来访问这些选项。

可以显示或隐藏以下类型的约束：
- 绑定约束
- 刚体约束（仅绑定和销钉）
- 壳-实体耦合
- 分布耦合
- 运动耦合
- 多点约束

**注意：**这些类型的约束是在"相互作用"模块中创建的，而不是在"装配"模块中创建的。请参阅["理解约束，" 第15.5节](pt03ch15s05.md)获取更多信息。

[表55-1](pt05ch55s11.md#usv-custom-constraints-table)显示了Abaqus/CAE如何显示每种类型的约束。

**表55-1** 约束在视口中的显示方式。
| 约束类型 | 在视口中的显示 |
| --- | --- |
| Tie | 在约束的主面和从面上的节点之间绘制红色线条的蛛网状显示。 |
| Rigid body | 刚体区域（元素、节点或表面）在视口中高亮显示。 |
| Shell-to-solid coupling | 在壳边缘上的节点和实体面之间绘制黄色线条的蛛网状显示。 |
| Distributing coupling | 在控制点和约束区域中的耦合点（节点）之间绘制橙色线条的蛛网状显示。 |
| Kinematic coupling | 在控制点和耦合点（节点）之间绘制紫色线条的蛛网状显示。 |
| Multi-point constraint (仅Abaqus/Explicit) | 组成多点约束的节点用黄色线条连接。 |

您可以在**Color Code**对话框中更改约束蛛蛛丝线的默认颜色。有关更多信息，请参阅["在可视化模块中对约束进行颜色编码，" 第77.7节](pt07ch77s07.md)。
