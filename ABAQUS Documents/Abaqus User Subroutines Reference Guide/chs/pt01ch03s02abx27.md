# 3.2.27 固定格式转换实用工具





**产品：** Abaqus/Standard  Abaqus/Explicit  

##### **参考**

- ["Abaqus执行程序：概述，" 第3.1.1节](pt01ch03s01abo02.md)

### 概述

**abaqus free**实用工具将Abaqus 5.8使用的固定格式输入文件转换为后续Abaqus版本使用的自由格式输入文件。

### 命令摘要

| **abaqus free** | [**job**](pt01ch03s02abx27.md#job1-usb-int-dfreeproc)=*job-name* |
| --- | --- |
|  | [**input**](pt01ch03s02abx27.md#input2-usb-int-dfreeproc)=*input-file* |

### 命令行选项

**job**

此选项用于指定要由实用工具输出的自由格式输入文件的名称。

**input**

此选项用于指定要转换的固定格式输入文件的名称。
