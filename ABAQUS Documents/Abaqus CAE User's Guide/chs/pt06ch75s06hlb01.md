# 75.6.1 自动创建虚拟拓扑





您可以选择 ****Tools**![](../graphics/images/arrow.gif)**Virtual Topology**![](../graphics/images/arrow.gif)**Automatic Create**** 从主菜单栏打开"创建虚拟拓扑"对话框。使用对话框中的控件来合并面、合并边或忽略模型中的实体。

**自动创建虚拟拓扑的步骤：**

1. 从上下文栏的 **Object** 字段中，选择一个零件或装配。
2. 从主菜单栏，选择 ****Tools**![](../graphics/images/arrow.gif)**Virtual Topology**![](../graphics/images/arrow.gif)**Automatic Create****。**提示：**您也可以使用位于 Mesh 模块工具箱中的 ![](../graphics/ico_virtTopoAuto.png) 工具自动创建虚拟拓扑。
3. 选择要考虑虚拟拓扑的目标区域。默认选择方法取决于您在步骤1中选择的是零件还是装配。如果需要，您可以在提示区域更改选择方法以选择面。您可以使用拖选、**[Shift]****+点击** 和 **[Ctrl]****+点击** 的组合来选择装配中的多个零件实例或零件或装配中的多个面。您也可以使用角度方法或面曲率方法来选择多个面。有关更多信息，请参见 ["在当前视口中选择对象，" 第6.2节](pt01ch06s02.md)。Abaqus/CAE 打开"创建虚拟拓扑"对话框。
4. 切换关闭您不希望 Abaqus/CAE 为虚拟拓扑考虑的任意条件。
5. 如果需要，点击位于每个包含大小参数的控件旁边的 ![](../graphics/ico_measure.png) 工具来测量视口中的实体。按照提示区域的说明进行选择。您可以在单次操作中测量多条边的长度或多个面的面积。完成后，在提示区域点击 **Done**（如有必要）。Abaqus/CAE 在消息区域显示测量值。如果您选择了多个实体，Abaqus/CAE 显示平均值、最小值和最大值。使用测量值来验证或调整对话框中的虚拟拓扑控件。
6. 点击对话框右侧的高亮工具，查看可能根据相应参数被虚拟拓扑替换的实体。例如，![](../graphics/ico_highlightEdge.png) 工具高亮显示目标区域中小于当前短边长度设置的边。如果高亮的实体是模型的组成部分或合并它们会违反曲率控制参数，则这些实体不会被替换。**注意：**曲率控件（拐角和圆角）的高亮工具指示根据当前设置将被*保留*的实体。
7. 当您对控件设置满意时，点击 **Preview** 来高亮显示所有将被虚拟拓扑替换的实体。**注意：**对于大型零件或装配，或包含大量待替换项目的任何选择集，preview 功能可能需要一些时间。相反，您可以使用步骤6中提到的高亮工具来快速查看满足参数设置的实体。
8. 点击 **Create** 根据选定的参数设置创建虚拟拓扑。Abaqus/CAE 高亮显示视口中使用当前选择将变得冗余的任何边或顶点，并提示您选择是否也要合并它们。Abaqus/CAE 在模型树中创建一个 **Auto virtual topology** 特征来合并或忽略选定的实体。
9. 如果在步骤3中选择的目標区域仍然有效，您可以重复步骤4到8在同一区域上创建更多虚拟拓扑，或点击 **Dismiss** 关闭"创建虚拟拓扑"对话框。如果选定的目标区域因虚拟拓扑而失效，Abaqus/CAE 关闭"创建虚拟拓扑"对话框并返回到步骤3选择新的目标区域。

您可以使用模型树通过删除或抑制 **Auto virtual topology** 对象来恢复选定的实体；您可以使用 **Restore Entities** 工具在保留虚拟拓扑特征的同时恢复某些实体。有关恢复单个实体的更多信息，请参见 ["恢复被虚拟拓扑替换的实体，" 第75.6.5节](pt06ch75s06hlb05.md)。
