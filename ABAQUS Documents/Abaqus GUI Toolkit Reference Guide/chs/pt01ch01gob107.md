# FXScrollArea







滚动区域窗口通过视口区域管理内容区域，可以查看内容。当内容区域变得大于视口区域时，会放置滚动条以允许通过滚动查看整个内容。根据模式，滚动条可以按需显示、始终显示或从不显示。通常，滚动区域的大小和内容的大小是独立的；但是，可以禁用水平（垂直）方向的滚动。在这种情况下，内容宽度（高度）将影响滚动区域窗口的宽度（高度）。对于重绘耗时的内容，可以关闭连续滚动。

### 全局标志

### **滚动条选项**

| **SCROLLERS_NORMAL** | 需要时显示滚动条。 |
| --- | --- |
| **HSCROLLER_ALWAYS** | 始终显示水平滚动条。 |
| **HSCROLLER_NEVER** | 从不显示水平滚动条。 |
| **VSCROLLER_ALWAYS** | 始终显示垂直滚动条。 |
| **VSCROLLER_NEVER** | 从不显示垂直滚动条。 |
| **HSCROLLING_ON** | 水平滚动打开（默认）。 |
| **HSCROLLING_OFF** | 水平滚动关闭。 |
| **VSCROLLING_ON** | 垂直滚动打开（默认）。 |
| **VSCROLLING_OFF** | 垂直滚动关闭。 |
| **SCROLLERS_TRACK** | 滚动条连续跟踪以实现平滑滚动。 |
| **SCROLLERS_DONT_TRACK** | 滚动条不连续跟踪。 |
