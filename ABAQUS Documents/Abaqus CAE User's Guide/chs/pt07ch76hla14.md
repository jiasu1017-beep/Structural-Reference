# 76.15 控制属性显示




**装配显示选项** 对话框中的 **属性** 显示选项允许您控制表示以下内容的符号显示：
- 您在"相互作用"模块中创建的相互作用、约束和连接器，
- 您在"载荷"模块中创建的载荷、边界条件和预定义场，
- 您在"属性"模块和"相互作用"模块中创建的工程特征，以及
- 您在"优化"模块中创建的优化属性。

您可以控制这些属性的显示时间和方式，您可以点击 **全部打开** 或 **全部关闭** 分别显示或隐藏所有属性。有关表示每个属性的符号的信息，请参见[附录C，"特殊图形符号"](ap03.md)。

有关控制在可视化模块中边界条件、耦合约束、连接器和点元素的显示的信息，请参见["控制模型实体的显示，" 第55.10节](pt05ch55s10.md)。

**控制属性显示：**

1. 找到 **属性** 显示选项。从主菜单栏，选择 ****视图**![](../graphics/images/arrow.gif)**装配显示选项****。在出现的对话框中点击 **属性** 选项卡。**属性** 选项变得可用。
2. 点击 **主** 选项卡来指定您要显示哪些属性以及要在哪些模块中显示它们。 1. 选择 **在以下位置显示属性** 选项。 - 选择 **创建属性的模块** 仅在创建属性的模块中显示属性。例如，载荷仅出现在载荷模块中，相互作用仅出现在相互作用模块中。 - 选择 **所有装配相关模块** 在支持装配显示的所有模块中显示属性。 2. 从 **显示** 列表中，选择您要显示的属性。只有您选择的属性才会出现在视口中；例如，如果您切换打开 **载荷** 和 **边界条件**，则只有载荷和边界条件符号会出现在视口中。您也可以通过点击 **全部打开** 选择所有类别和每个类别内的所有类型，或通过点击 **全部关闭** 取消选择所有类别和每个类别内的所有类型。
3. 点击 **符号** 选项卡来控制属性符号的大小和密度。您还可以将孤儿网格区域上显示的属性符号数量减少到最大允许数量的一部分。 1. 指定您的 **大小** 偏好。大小设置越高，符号在视口中显示得越大。 - 拖动 **箭头** 滑块来指定箭头符号的大小。 - 拖动 **其他符号** 滑块来指定除箭头外所有符号的大小。 - 关闭 **根据解析场值缩放符号** 以移除对指定解析场的属性的符号缩放。有关更多信息，请参见["显示具有解析场的相互作用和规定条件的符号，" 第58.4节](pt06ch58s04.md)。 2. 如果您正在处理几何，请指定属性符号的所需密度。密度设置越高，Abaqus 用于表示每个属性的符号就越多。 - 拖动 **面密度** 滑块来控制出现在面上的符号密度。 - 拖动 **边密度** 滑块来控制出现在边上的符号密度。更改符号密度设置的效果可能会因视口中区域的大小而异。 3. 如果您想减少显示在孤儿网格区域上的属性符号密度，请在 **孤儿网格区域上显示的符号分数** 字段中输入0到1之间的值。分数值越高，Abaqus/CAE 用于表示每个属性的符号就越多。选择默认密度1会提示Abaqus/CAE 在区域内的每个网格实体上绘制符号。
4. 点击感兴趣属性的选项卡来指定您希望在视口中显示的特定属性类别和类型。例如，如果您点击 **载荷** 选项卡，则会出现载荷类别列表。如果您点击其中一个类别旁边的箭头，则会出现该类别内所有载荷类型的列表。使用以下技术来指定您要显示的属性类别和类型： - 点击感兴趣类别旁边的箭头。从出现的类型列表中，选择您要显示的那些类型。 - 切换所需的类别。此操作选择或取消选择该类别中的所有类型。 - 点击 **全部打开** 以选择所有类别和每个类别内的所有类型。 - 点击 **全部关闭** 以取消选择所有类别和每个类别内的所有类型。当某个类别中的所有类型都被选中时，该类别标签旁边的复选框会变成白色背景上的黑色复选标记。当仅选择了该类别中的某些类型时，复选框变为浅灰色，复选标记变为深灰色。 **注意：**您指定的属性类别和类型仅在您在步骤2中切换打开该属性时才会出现在视口中。
5. 根据需要重复上一步，以显示其他感兴趣属性的特定类别和类型。
6. 在 **装配显示选项** 对话框底部，点击 **确定** 以在当前视口中实施您的显示设置并关闭对话框。
