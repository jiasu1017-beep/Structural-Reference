# 30.8 查看流体分析结果





您可以在Visualization模块中显示流体分析的结果。动画、线型等值线图、等值面型等值线图以及沿路径显示的结果对于查看流体分析结果特别有用。此外，您可以查看协同仿真的结果，如使用Abaqus/CFD和Abaqus/Standard的共轭热传递分析的结果（如图30-1所示）。
