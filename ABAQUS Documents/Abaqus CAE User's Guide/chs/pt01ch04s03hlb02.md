# 4.3.2 选择视口





大多数与模型的交互（如绘制零件、放置载荷、装配零件实例、生成网格以及自定义绘图状态）都在当前视口中进行。此外，如果在画布上显示多个视口，当前视口会指示您正在操作的模型（当前模型）和正在工作的模块（当前模块）。

有几种方法可以选择新视口作为当前视口：
- 单击现有视口的边框或标题栏。
- 从**视口**菜单中的列表中选择现有视口。
- 使用**[Ctrl]****+[Tab]**——或从**视口**菜单中选择**下一个**或**上一个**——循环遍历画布上的所有视口。**注意：**在Linux平台上，**[Ctrl]****+[Tab]**用于切换应用程序；**[Ctrl]****+[F6]**是替代键盘快捷键。
- 创建新视口。

当前视口的标题栏为深灰色，如下图所示：

![](../graphics/cnv-selected-view.png)

**要选择视口：**

1. 将光标移到视口的边框上。如果视口被隐藏，您可能需要移动其他视口以露出要选择的视口。有关更多信息，请参阅["移动视口，"第4.3.3节"](pt01ch04s03hlb03.md)。
2. 单击鼠标按钮1。视口变为当前视口；视口也被选中，其标题栏变为蓝色以指示它是选定的视口。如果您单击工具或菜单，视口保持为当前视口，但标题栏恢复为深灰色。**注意：**在Windows平台上，您可以自定义Abaqus/CAE使用的颜色。有关更多信息，请参阅["Windows平台上的常见自定义，"Abaqus安装和许可指南第5.1.2节](../sgb/sgb-link.md#sgb-chp-view-cust-common)。
