# 17.4.1 什么是网格种子？







种子是您沿区域边缘放置的标记，用于指定该区域中的目标网格密度。区域边界上的网格密度和区域内部的网格密度都由区域边缘上的种子决定。

您可以使用网格模块主菜单栏中的 **Seed** 菜单创建和控制种子。Abaqus/CAE 生成的网格会尽可能接近您的种子。Abaqus/CAE 可以使用以下方法来控制种子的分布：
- 在零件或零件实例的所有边缘上均匀放置种子
- 在边缘上均匀放置种子
- 放置有偏差的种子，使网格密度向边缘的一端增加
- 放置有偏差的种子，使网格密度从每个端点向边缘中心增加
- 放置有偏差的种子，使网格密度从中心向边缘的每个端点增加

您应该将种子应用到所有边缘。如果均匀种子分布足够，建议的方法是对整个零件或零件实例进行布种子。如果您想更好地控制网格，可以对区域进行分区，然后沿您创建的分界线提供种子。此技术在"验证和改进网格"第 17.6 节中有更详细的描述。

网格种子仅指定目标网格密度。如果您使用六面体或四边形单元，Abaqus/CAE 通常会更改单元分布，以便成功生成网格。您可以通过约束沿边缘的种子数量来防止此类调整。当您约束种子时，主要是规定沿边缘的单元数量，在较小程度上规定节点的精确位置；如有必要，Abaqus/CAE 会调整节点位置以减少单元扭曲。此外，您应该谨慎使用这些约束，因为它们可能使网格生成器更难获得网格。

默认情况下，Abaqus/CAE 仅在您定义或修改种子放置时显示零件或装配体上的种子。如果您希望在进行网格模块中的其他操作时显示种子，可以启用持久的种子显示；在 **Visible Objects** 工具栏中切换 ![](../graphics/icoR_meshSeeds.png) 以保持种子显示。
