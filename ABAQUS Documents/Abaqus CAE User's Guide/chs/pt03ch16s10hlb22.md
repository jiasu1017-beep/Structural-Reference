# 16.10.22 定义欧拉网格运动边界条件





欧拉网格运动允许欧拉网格——通常是刚性且不可变形的——在分析过程中扩展、收缩和平移。作为网格运动定义的一部分，您可以识别要跟随的对象，该对象可以是拉格朗日表面或欧拉材料实例；欧拉网格会在分析过程中调整大小和移动，以便始终围绕该对象。

欧拉网格运动基于由Abaqus构造的矩形"包围盒"计算。包围盒无法在Abaqus/CAE中可视化，但它通常与欧拉网格相当接近。包围盒的创建和行为在["欧拉网格运动"章节，《Abaqus Analysis User's Guide》第14.1.3节](../usb/usb-link.md#usb-anl-aeulerianmeshmotion)中描述。您可以对该包围盒的收缩、扩展和平移设置约束和限制。这些限制可能会阻止欧拉网格在分析过程中的所有时刻完全包含指定的跟随对象。

欧拉网格运动边界条件只能应用于具有欧拉截面分配的部件，并且不能应用于分析的初始步骤。

**指定欧拉网格运动边界条件的步骤：**

1. 使用以下方法之一显示欧拉网格运动边界条件编辑器： - 要创建新的欧拉网格运动边界条件，请按照["创建边界条件"章节第16.8.2节](pt03ch16s08hlb02.md)中概述的程序进行操作（**类别**：`其他`；**所选步骤的类型**：`欧拉网格运动`）。 - 要使用菜单或管理器编辑现有的欧拉网格运动边界条件，请参见["编辑依赖于步骤的对象"章节第3.4.13节](pt01ch03s04hlb05.md)。要编辑网格运动定义所适用的部件实例，请参见["编辑应用规定条件的区域"章节第16.8.4节](pt03ch16s08hlb04.md)。
2. 指定**要跟随的对象**： - 要跟随拉格朗日表面，请选择**表面区域**并单击![](../graphics/ico_selectBlue.png)选择表面： - 在视口中选择一个表面。您可以使用角度方法从几何体中选择一组面或从网格中选择一组单元面。有关更多信息，请参见["使用角度和特征边缘方法选择多个对象"章节第6.2.3节](pt01ch06s02hlb03.md)。完成选择后，单击鼠标按钮2。 - 要从现有表面列表中进行选择，请执行以下操作： 1. 在提示区域右侧单击**表面**。Abaqus/CAE显示**区域选择**对话框，其中包含可用表面列表。 2. 选择感兴趣的表面，然后单击**继续**。**注意：**默认选择方法基于您最近使用的选择方法。要恢复为另一种方法，请在提示区域右侧单击**在视口中选择**或**表面**。 - 要跟随欧拉材料，请选择**欧拉材料实例**并选择一个实例。
3. 默认情况下，使用全局坐标系来构造网格包围盒。要更改构造网格包围盒的坐标系，请单击**坐标系**选项的![](../graphics/ico_selectBlue.png)并执行以下操作之一： - 在视口中选择一个现有的基准坐标系。 - 按名称选择一个现有的基准坐标系。 1. 在提示区域中单击**基准坐标系列表**以显示基准坐标系列表。 2. 从列表中选择一个名称，然后单击**确定**。 - 在提示区域中单击**使用全局坐标系**以恢复为全局坐标系。或者，单击![](../graphics/ico_dtmCsys.png)来定义新的基准坐标系。此坐标系编辑选项仅在创建边界条件的步骤中可用。
4. 在**控制**选项卡页上，指定对网格包围盒运动的通用约束和限制： 1. 要防止网格在分析过程中收缩，请关闭**允许网格收缩**。此选项默认情况下为开启状态。 2. 指定包围盒三个方向中任一方向的长度与宽度比率允许的最大变化的**长宽比限制**。该值必须大于或等于1。 3. 指定**网格速度因子**。此因子将欧拉网格的最大速度定义为被跟随对象速度的分数。该值必须大于或等于零。 4. 如果被跟随的对象是欧拉材料，请指定**体积分数阈值**。体积分数低于此阈值的欧拉材料单元被视为空，用于网格运动目的，网格不会扩展或移动以考虑这些单元中的材料。该值必须大于零且小于或等于1。 5. 指定**缓冲区大小**，这在被跟踪对象和欧拉网格边界之间保留一定距离： - 选择**指定**并输入一个值来定义缓冲区，该值等于给定值乘以最大欧拉单元尺寸。该值必须大于或等于零。 - 选择**初始**来定义缓冲区，等于对象与网格边界之间的初始距离。
5. 在每个**轴**选项卡页上，指定包围盒在其三个方向上运动的限制： 1. 选择原始**中心位置**沿选定轴是**自由**还是**固定**。 2. 选择原始**正平面位置**是**自由**还是**固定**。正平面是法线与选定轴正方向对齐的包围盒面。 3. 选择原始**负平面位置**是**自由**还是**固定**。负平面是法线与选定轴负方向对齐的包围盒面。 4. 指定沿选定轴的**扩展比率**： - 选择**无限制**以在必要时允许包围盒无限缩放。 - 选择**指定最大值**并输入一个值来定义最大缩放比率。该值必须大于或等于1。 5. 指定沿选定轴的最小**收缩比率**。该值必须大于或等于零且小于或等于1。
