# 11.8.1 稳态检测

**产品：** Abaqus/Explicit

##### **参考文献**

- ["输出," 第 4.1.1 节"](pt02ch04s01aus38.md)
- [*STEADY STATE DETECTION](../key/key-link.md#usb-kws-hsteadystatedetection)
- [*STEADY STATE CRITERIA](../key/key-link.md#usb-kws-hsteadystatecriteria)

### 概述

稳态检测：
- 可用于检测准静态单向 Abaqus/Explicit 模拟中达到稳态条件的时间，然后终止模拟；
- 可用于输出有助于跟踪单向 Abaqus/Explicit 模拟进度的量；以及
- 仅适用于三维分析。

### 简介

许多类型的单向过程用于将预成形形状转化为更适合进一步加工的形式。最常见的例子是轧制、线材拉拔和挤压过程。由于这些过程通常以低速进行，因此经常使用诸如 Abaqus/Explicit 中的显式动态过程将过程建模为准静态。分析通常由通过任意数量的轧辊或其他成形表面沿主方向形成的工件组成。成形表面通常建模为刚体。对于轧制模拟，刚体参考节点通常定义在轧辊中心。工件的网格通常被挤压并可能由多层材料构建。当工件通过成形表面时，形状最终达到恒定状态。工件离开最终成形表面的位置称为出口平面，通常与最终成形表面的刚体参考节点对齐。一旦达到此恒定形状，分析就被认为已达到稳态。此时，最终成形表面上的力和扭矩也达到了恒定值或围绕恒定值振荡。通过检测稳态条件并立即或在稳态横截面超出出口平面到达称为切割平面的位置后终止分析，可以实现显著的计算节省。

#### 网格要求

工件网格需要满足某些条件才能使用稳态检测功能。首先，网格在主方向上必须是拓扑规则的。换句话说，网格应由多排单元平面组成，每排平面与其相邻的前后平面相似，因为它包含相同数量的单元和横截面上相同的单元拓扑。此外，平面上的每个单元连接到引用相同材料和截面属性的前后平面上的单元。因此，允许具有多种材料和截面属性的网格，但主方向上的任何单元行必须具有相同的类型，并且必须引用相同的材料和截面属性（参见 [图 11.8.1-1](pt04ch11s08aus76.md#ssdetection-deformed)）。

### 稳态检测标准采样

为了确定是否达到稳态，计算稳态检测"范数"，它表示当材料通过沿主方向的给定位置时，对感兴趣变量在工件横截面上的平均值。此位置称为出口平面，通常与工件通过的最后一个刚性成形工具（例如轧辊）的位置重合。出口平面的法线根据定义与主方向重合。范数采样的时间间隔取决于轧制分析是以欧拉方式还是拉格朗日方式建模。

#### 拉格朗日分析中的采样

在基于拉格朗日的分析中（包括在拉格朗日域上采用自适应网格划分），当每排单元平面的尾随控制节点通过出口平面时计算稳态范数。[图 11.8.1-2](pt04ch11s08aus76.md#ssdetection-controlnode) 说明了控制节点定义。

**图 11.8.1-2** 控制节点定位。

![](../graphics/ssdetection-controlnode-nls.png)

因此，范数采样的时间周期基于单元平面穿过出口平面的频率。对于输出，假定范数的值在连续控制节点通过出口平面的时间之间保持恒定。

#### 欧拉分析中的采样

欧拉分析采用控制体方法，其中材料从流入欧拉边界吸入，并通过流出边界推出或拉出。在工件上定义自适应网格域，并定义滑动边界区域以对工件与轧辊等成形工具之间的接触进行建模。有关自适应网格划分技术的详细信息，请参见 ["ALE 自适应网格划分：概述," 第 12.2.1 节"](pt04ch12s02abo14.md)。网格保持相对静止，而材料通过出口平面移动。因此，采样之间的时间周期基于材料通过出口平面的进度。为了以与拉格朗日情况一致的方式确定时间周期，采样周期通过将工件的特征单元长度除以材料流速来确定。这个周期大致是材料通过典型尺寸单元所需的时间。

### 稳态检测范数定义

如果单个范数在三个连续平面上的值的相对变化不超过容差，则认为该范数已达到稳态。当您定义稳态标准时，可以提供范数容差，或者可以选择由 Abaqus/Explicit 提供的默认值。可以通过请求以下定义中列出的其标识符来输出范数。

#### 等效塑性应变范数

单元平面的塑性应变范数定义为将平面每个单元的等效塑性应变与单元体积的乘积求和，然后除以平面上单元的总体积。此范数提供了平面等效塑性应变的加权平均值。等效塑性应变范数的标识符为 SSPEEQ。

#### 扩展范数

平面单元的扩展范数计算为平面横截面积惯性矩的最大值。在确定扩展范数时，平面单元的横截面通过将法线最初与主方向重合的单元面投影到出口平面上来确定。然后，关于横截面原始主轴方向的形心确定关于截面形心的面积惯性矩。扩展范数的标识符为 SSSPRD。

#### 力范数

力范数通过在采样点之间的时间周期内对成形工具（例如出口轧辊）的刚体参考节点处力的大小进行平均来计算。您提供刚体参考节点和力的方向。力范数的标识符为 SSFORC。

#### 扭矩范数

扭矩范数通过在采样点之间的时间周期内对成形工具（例如出口轧辊）的刚体参考节点处扭矩的大小进行平均来计算。您提供刚体参考节点和扭矩方向。扭矩范数的标识符为 SSTORQ。

### 在分析期间请求稳态检测

您必须定义用于确定是否达到稳态的标准。Abaqus/Explicit 将根据稳态的实现来终止分析。

#### 稳态检测

稳态检测定义用于定义工件中的单元、工件的主方向、切割位置和使用的采样类型。主方向通过指定相对于全局笛卡尔坐标系的_direction cosines_来定义。切割位置通过指定切割平面上一点的全局坐标来定义。切割平面的法线假定与主方向重合。一旦检测到稳态，当首次检测到已达到稳态的工件平面进展到切割平面时，分析终止。您可以选择使用的采样方法，如下所述。

##### 请求在拉格朗日分析中当单元通过出口平面时进行采样

您可以请求在每排单元平面穿过出口平面时计算所有稳态范数。

| **输入文件用法：** | ``` [*STEADY STATE DETECTION](../key/key-link.md#usb-kws-hsteadystatedetection), ELSET=*elset*, SAMPLING=PLANE BY PLANE ``` |
| --- | --- |

##### 为欧拉分析请求均匀间隔采样

或者，您可以请求基于材料进展一个平均单元长度所需的时间以均匀间隔计算所有稳态范数。

| **输入文件用法：** | ``` [*STEADY STATE DETECTION](../key/key-link.md#usb-kws-hsteadystatedetection), ELSET=*elset*, SAMPLING=UNIFORM ``` |
| --- | --- |

#### 稳态标准

可以指定任意数量的稳态标准定义。只有当任何单个稳态标准定义下指定的所有标准都满足时，分析才被认为是已达到稳态。

要定义标准，您需要指定范数类型标识符、范数容差和出口平面上一点的全局坐标。对于力和扭矩范数，您还需要指定出口平面上成形工具的刚体参考节点以及力或扭矩的方向余弦。可以为每个范数定义单独定义出口平面。

| **输入文件用法：** | 使用以下选项定义达到稳态所需的标准： |
| --- | --- |
|  | ``` [*STEADY STATE DETECTION](../key/key-link.md#usb-kws-hsteadystatedetection), ELSET*=elset*, SAMPLING=PLANE BY PLANE or UNIFORM [*STEADY STATE CRITERIA](../key/key-link.md#usb-kws-hsteadystatecriteria) [*STEADY STATE CRITERIA](../key/key-link.md#usb-kws-hsteadystatecriteria) ... ``` 例如，假设有两组标准并且一旦满足其中任一标准即可终止分析。输入可能如下： ``` [*STEADY STATE DETECTION](../key/key-link.md#usb-kws-hsteadystatedetection), ELSET=sheet, SAMPLING=PLANE BY PLANE 1.0, 0.0, 0.0, 6.0, 0.0, 0.0 [*STEADY STATE CRITERIA](../key/key-link.md#usb-kws-hsteadystatecriteria) SSPEEQ,.002, 5.0, 0.0, 0.0 SSSPRD,.002, 5.0, 0.0, 0.0 SSFORC,.005, 5.0, 0.0, 0.0, 1000, 1.0, 0.0, 0.0 SSFORC,.005, 5.0, 0.0, 0.0, 1000, 0.0, 1.0, 0.0 SSTORQ,.005, 5.0, 0.0, 0.0, 1000, 0.0, 0.0, 1.0 [*STEADY STATE CRITERIA](../key/key-link.md#usb-kws-hsteadystatecriteria) SSPEEQ,.001, 5.0, 0.0, 0.0 SSSPRD,.001, 5.0, 0.0, 0.0 SSFORC,.010, 5.0, 0.0, 0.0, 1000, 0.0, 1.0, 0.0 ``` |

### 材料

稳态检测旨在与基于塑性的材料一起使用，因为对于非塑性材料模型，等效塑性应变范数为零。

### 过程

每个分析只允许一个稳态检测定义。该定义可以在任何步骤中输入，并在分析中延续到后续步骤。稳态检测定义不能在退火步骤中输入或延续到退火步骤。

### 单元

当前稳态检测功能仅支持使用 C3D8R 和 C3D8RT 单元。

### 输出

输出变量 SSPEEQ*n*、SSSPRD*n*、SSFORC*n* 和 SSTORQ*n* 分别用于输出等效塑性应变、扩展、力和扭矩范数。Abaqus/CAE 可用于获取每个稳态检测范数变量的历史图。可以通过请求基于范数指定顺序的范数编号 *n* 来输出各个范数。参考上面的示例，如果请求第二个稳态标准定义的力范数，则输出标识符为 SSFORC3。如果请求不包含范数编号的稳态检测范数（例如 SSFORC），则输出该类型的所有范数。

一旦检测到稳态，Abaqus/Explicit 会自动创建一个单元集并将其写入由首次满足稳态标准的单元平面组成的输出数据库。创建的单元集名为 *SteadyStatePlane-StepN*，其中 *N* 是步骤编号；可以使用 Abaqus/CAE 查看。如果未对输出数据库提出任何输出请求，则不会创建单元集 *SteadyStatePlane-StepN*。
