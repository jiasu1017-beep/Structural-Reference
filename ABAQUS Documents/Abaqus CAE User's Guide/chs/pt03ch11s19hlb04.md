# 11.19.5 设置新零件的近似尺寸





使用**创建零件**对话框底部的**近似尺寸**文本字段来设置新零件的近似尺寸。您输入的尺寸用于Abaqus/CAE计算草图工作表及其网格的间距。近似零件尺寸必须在10000（10⁴）和0.001（10⁻³）单位之间。

退出**创建零件**对话框时，Abaqus/CAE进入草图编辑器，您绘制基础特征的轮廓。草图编辑器显示带有叠加网格的方形工作表，并调整工作表的尺寸以接近零件的尺寸。

如果随后编辑零件，Abaqus/CAE仍根据创建基础特征时使用的相同尺寸确定草图工作表的尺寸。因此，您应将零件的近似尺寸设置为与完成零件的最大尺寸相匹配。

**设置新零件的近似尺寸：**

1. 在**创建零件**对话框底部的**近似尺寸**文本字段中输入新零件的近似尺寸。
2. 完成选项选择后，单击**继续**关闭**创建零件**对话框。草图编辑器启动，显示基于新零件近似尺寸的工作表尺寸和网格间距。
