# 4.2 选择连续体单元

为特定模拟选择正确的单元类型对于以合理的成本获得准确结果至关重要。随着您在使用Abaqus方面越来越有经验，您无疑会为自己的特定应用制定选择单元的指南。但是，当您开始使用Abaqus时，此处提供的指南可能会有所帮助。

以下建议适用于Abaqus/Standard和Abaqus/Explicit：

-   尽可能减少网格扭曲。扭曲的线性单元粗网格可能会产生非常差的结果。
-   对于涉及非常大网格扭曲（大应变分析）的模拟，请使用细化的线性减缩积分单元网格（CAX4R、CPE4R、CPS4R、C3D8R等）。
-   在三维空间中，尽可能使用六面体（砖形）单元。它们以最低的成本提供最佳结果。复杂的几何形状可能很难完全用六面体进行网格划分；因此，可能需要使用楔形单元和四面体单元。这些单元的线性版本C3D4和C3D6是较差的单元（需要细网格才能获得准确的结果）；因此，这些单元通常应仅在完成网格需要时使用，而且即使使用，也应该远离任何需要精确结果的区域。
-   一些前处理器包含可以用四面体单元对任意几何形状进行自由网格划分的算法。Abaqus/Standard中的二次四面体单元（C3D10或C3D10I）适用于一般用途；但在与接触一起使用时，应仅使用"面-面"接触离散化。这些单元的替代方案是两个分析产品中都提供的改进型二次四面体单元（C3D10M）。该单元对于大变形问题和接触问题（无论是使用传统的"节点-面"还是"面-面"接触离散化）都是稳健的，并且表现出最小的剪切和体积锁定。然而，无论使用哪种单元，分析都比等效的六面体单元网格花费更长时间。您不应使用仅包含线性四面体单元（C3D4）的网格：除非您使用非常大量的单元，否则结果将不准确。

Abaqus/Standard用户还应考虑以下建议：

-   对于一般分析工作，使用二次减缩积分单元（CAX8R、CPE8R、CPS8R、C3D20R等），除非您需要模拟非常大的应变或具有复杂、变化的接触条件。
-   在可能存在应力集中的局部区域使用二次完全积分单元（CAX8、CPE8、CPS8、C3D20等）。它们以最低的成本提供应力梯度的最佳分辨率。
-   对于接触问题，请使用细化的线性减缩积分单元或不相容模式单元网格（CAX4I、CPE4I、CPS4I、C3D8I等）。请参阅第12章"接触"。
