**产品：** Abaqus/Standard

### 概述

框架截面行为：
- 需要定义截面的形状及其材料响应；
- 在框架单元内部使用线性弹性行为；
- 可包括单元端部的"集中"塑性以建模塑性铰的形成；
- 可以仅为单轴，响应由现象学屈曲 strut 模型控制，结合线弹性和拉伸塑性屈服；和
- 仅适用于管道截面，可在分析过程中切换到屈曲 strut 响应。

### 定义弹塑性截面行为

要包括弹塑性响应，直接指定 *N*、![](../graphics/usb_eqn04588.gif)、![](../graphics/usb_eqn04589.gif) 和 *T* 作为其共轭塑性变形变量的函数，或使用基于材料屈服应力的 *N*、![](../graphics/usb_eqn04588.gif)、![](../graphics/usb_eqn04589.gif) 和 *T* 的默认塑性响应。Abaqus/Standard 使用指定或默认值来定义非线性运动硬化模型，该模型"集中"在单元端部的塑性铰中。由于塑性集中在单元端部，铰没有关联长度维度。广义力与广义塑性位移相关，而非应变。实际上，塑性铰将具有由构件长度和加载决定的有限尺寸，这将影响硬化速率，但不影响极限载荷。例如，纯弯曲下的屈服（构件上恒定的弯矩）将产生等于构件长度的铰长度，而具有横向尖端载荷的悬臂屈服（构件上线性变化的弯矩）将产生更加局部的铰。因此，如果给定载荷的硬化速率和因此的塑性变形很重要，您应该对不同的长度和不同的加载情况适当校准塑性响应。

在塑性范围内，唯一可用的塑性表面是椭球体。此屈服面仅对管道截面相当准确。箱形、圆形、工字和矩形截面可在您的判断下使用，但需理解椭圆屈服面可能无法准确近似弹塑性响应。通用截面类型不能与塑性一起使用。

### 定义可选的单轴 strut 行为

框架单元可选地仅允许单轴响应（strut 行为）。在这种情况下，单元的两端都不承受垂直于轴线的弯矩或力；因此，只存在沿单元轴线的力。此外，此轴向力沿单元长度恒定，即使切向分布加载施加于单元轴线。单元的单轴响应是线弹性的或非线性的，在后者情况下包括压缩中的屈曲和后屈曲以及拉伸中的各向同性硬化塑性。

#### 定义线弹性单轴行为

线弹性单轴框架单元的行为类似于具有恒定刚度 ![](../graphics/usb_eqn04649.gif) 的轴向弹簧，其中 *E* 是杨氏模量，*A* 是横截面积，*L* 是原始单元长度。应变量度是单元长度变化除以单元原始长度。

### 在分析过程中切换到可选的单轴 strut 行为

框架单元允许在分析过程中切换到单轴屈曲 strut 响应。切换标准是"ISO"方程以及"强度"方程（见["框架单元的屈曲 strut 响应，" Abaqus 理论指南第 3.9.3 节](../stm/stm-link.md#stm-elm-strut)）。当满足 ISO 方程时，弹性或弹塑性框架单元发生一次性切换到屈曲 strut 响应。引入强度方程是为了在没有显著轴向力的情况下防止切换。

当框架单元切换到屈曲 strut 响应时，会发生结构刚度的显着损失。切换后的单元不再承受弯曲、扭转或剪切加载。如果切换导致全局结构不稳定（即结构将在施加的载荷下坍塌），分析可能无法收敛。

要允许单元响应切换，请使用默认屈曲包络线或定义屈曲包络线并提供屈服应力，但不要为框架单元激活线弹性单轴行为。

ISO 方程是基于与细长管道状（管状）构件的实验的经验关系。由于方程以管道外径和厚度明确写出，因此只有管道截面允许屈曲 strut 响应。ISO 方程包含几个可以定义的因子。有效和附加长度因子考虑单元端部固定，屈曲折减因子考虑弯矩对屈曲的影响。您可以在每个局部横截面方向上为这些因子定义非默认值。

| **输入文件用法：** | 要允许使用 ISO 方程的默认系数和默认屈曲包络线切换到屈曲 strut 响应，请使用以下选项： |
| --- | --- |
|  | ``` [*FRAME SECTION](../key/key-link.md#usb-kws-mframesection), SECTION=PIPE, ELSET=*name*, BUCKLING, YIELD STRESS=![](../graphics/usb_eqn00310.gif) ``` 要允许使用 ISO 方程的非默认系数和默认屈曲包络线切换到屈曲 strut 响应，请使用以下所有选项： ``` [*FRAME SECTION](../key/key-link.md#usb-kws-mframesection), SECTION=PIPE, ELSET=*name*, BUCKLING, YIELD STRESS=![](../graphics/usb_eqn00310.gif) [*BUCKLING LENGTH](../key/key-link.md#usb-kws-mbucklinglength) [*BUCKLING REDUCTION FACTORS](../key/key-link.md#usb-kws-mbucklingreductfact) ``` 要允许使用 ISO 方程的非默认系数和用户定义的屈曲包络线切换到屈曲 strut 响应，请使用以下所有选项： ``` [*FRAME SECTION](../key/key-link.md#usb-kws-mframesection), SECTION=PIPE, ELSET=*name*, YIELD STRESS=![](../graphics/usb_eqn00310.gif) [*BUCKLING ENVELOPE](../key/key-link.md#usb-kws-mbucklingenvelope) [*BUCKLING LENGTH](../key/key-link.md#usb-kws-mbucklinglength) [*BUCKLING REDUCTION FACTORS](../key/key-link.md#usb-kws-mbucklingreductfact) ``` |
