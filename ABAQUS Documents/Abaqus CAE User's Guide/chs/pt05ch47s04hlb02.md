# 47.5.3 自定义X–Y图轴刻度





### 47.5.3 自定义X–Y图轴刻度

点击**Scale**选项卡可为*X–Y*图的*X*轴和*Y*轴选择线性、对数或分贝刻度。您显示的所有*X*轴和*Y*轴的刻度是独立的。默认情况下，Abaqus/CAE对所有轴使用线性刻度。

**基于分贝的刻度**

您可以使用两种基于分贝的刻度之一沿特定轴重新调整数据刻度。每种刻度都适合特定类型的波动数据：**10 dB**刻度推荐用于显示波功率或强度数据；**20 dB**刻度推荐用于波幅度数据。

当您选择基于分贝的轴刻度时，您还可以调整分贝参考值。调整此值使您能够根据此*X–Y*数据对象中数据的环境条件和模型中的单位重新调整数据对象。默认值为`1`。您可能希望调整此值以显示不同环境或通过不同介质（如水）传播的波的数据。

**要自定义X–Y图的轴刻度：**

1. 找到**Scale**选项。选择****Options**![](../graphics/images/arrow.gif)**XY Options**![](../graphics/images/arrow.gif)**Axis****；或点击![](../graphics/ico_xyAxisOptions.png)，它位于Visualization模块工具箱中的*X–Y*绘图工具中。在出现的对话框中点击**Scale**选项卡。
2. 从**X Axis**或**Y Axis**字段中，高亮显示一个或多个轴。
3. 选择以下轴刻度选项之一：**Linear**所选轴以线性进度显示。**Log**所选轴以10为底的对数进度显示。**10 dB**所选轴以10 dB进度显示。**dB Reference**字段变得可用，您可以输入此轴上10 dB数据的分贝参考值。**20 dB**所选轴以20 dB进度显示。**dB Reference**字段变得可用，您可以输入此轴上20 dB数据的分贝参考值。
4. 点击**Dismiss**关闭**Axis Options**对话框。
