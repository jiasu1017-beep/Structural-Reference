# 53.2.2 自定义基本铺层堆叠图选项





点击"**Basic**"选项卡来选择铺层堆叠图中显示哪些铺层以及铺层的排列方式。如果您的复合铺层包含许多铺层,铺层堆叠图可能会变得混乱且难以解释。为了使铺层堆叠图更易于阅读,基本铺层堆叠图选项允许您仅查看可管理的铺层数量。

**自定义基本铺层堆叠图选项的步骤:**

1. 找到基本选项。 1. 点击提示区域中的"**铺层堆叠图选项**"按钮,或执行以下操作之一: - 在"属性"模块中,选择 ****View**![](../graphics/images/arrow.gif)**Ply Stack Plot Options****。 - 在"可视化"模块中,选择 ****Options**![](../graphics/images/arrow.gif)**Ply Stack Plot**** 或点击工具箱中的 ![](../graphics/ico_plotPlyStackOptions.png) 工具。 2. 在出现的"铺层堆叠图选项"对话框中点击"**Basic**"选项卡。
2. 从"**Display**"选项中,执行以下操作: - 选择渲染样式——着色、填充或线框。 - 选择是否显示每个铺层的边缘以及边缘的样式和粗细。 - 选择是将铺层显示为从最后一个铺层到第一个铺层的"阶梯"图像,还是显示为关于中心铺层或多个铺层对称的图像。
3. 从"**Visible Plies**"选项中,选择要显示的铺层。当您修改显示参数时,Abaqus/CAE会更新铺层堆叠图,这允许您快速方便地显示复合层中的铺层。您可以选择以下选项: - 显示从"**Start**"铺层开始到"**End**"铺层结束的指定铺层。 - 显示从"**Start**"铺层开始的"**Number**"个铺层。点击"**Increment by Number**"按铺层数增加**Start**值,而不是增加1。这允许您的铺层堆叠图是一个在铺层上增量移动的窗口,每次增量显示下一组相邻的铺层。 - 显示以"**Center**"铺层为中心的"**Number**"个铺层。点击"**Increment by Number**"按铺层数增加**Center**值,而不是增加1。这允许您的铺层堆叠图是一个在铺层上增量移动的窗口,每次增量显示下一组相邻的铺层。 - 显示以"**Center**"铺层为中心的"**Number**"个铺层。拖动"**Center**"滑块可以在铺层上移动。
4. 选择"**Even**"和"**Odd**"铺层的颜色: 1. 点击颜色样本 ![](../graphics/usi-colorsample.png)。Abaqus/CAE显示"选择颜色"对话框。 2. 使用"选择颜色"对话框中的方法之一来选择新颜色。有关更多信息,请参阅 ["自定义颜色," 第3.2.9节](pt01ch03s02s09.md)。 3. 点击"**OK**"关闭"选择颜色"对话框。颜色样本变为所选颜色。Abaqus/CAE相对于显示的铺层应用您选择的颜色,而不是实际铺层号。例如,如果您显示铺层2、3和4;Abaqus/CAE会将"**Odd**"颜色应用于铺层2和4,将"**Even**"颜色应用于铺层3。
5. 点击"**Apply**"实施您的更改。铺层堆叠图会根据您的规格进行更改。默认情况下,您的更改会在会话期间保存,并会影响该视口中所有后续的铺层堆叠图。如果要为后续会话保留更改,请将它们保存到文件。有关更多信息,请参阅 ["为后续会话保存自定义," 第55.1.1节](pt05ch55s01s01.md)。
