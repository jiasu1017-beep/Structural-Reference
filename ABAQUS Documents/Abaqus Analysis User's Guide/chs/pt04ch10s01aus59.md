# 12.3.2 定义子结构

**产品：** Abaqus/Standard

##### **参考资料**

- ["使用子结构概述，" 第 12.3.1 节](pt04ch10s01aus58.md)
- [*SUBSTRUCTURE GENERATE](../key/key-link.md#usb-kws-hsubstructuregenerate)
- [*SUBSTRUCTURE](../key/key-link.md#usb-kws-hsubstructure)

### 概述

子结构定义涉及选择单元区域、生成超单元矩阵以及在后续分析中使用子结构。本节详细说明如何定义子结构。

### 定义子结构区域

子结构区域由单元集定义：

| **输入文件用法：** | ``` [*ELSET](../key/key-link.md#usb-kws-melset), ELSET=*name* *element_list* ``` |

### 生成子结构

子结构生成步骤创建超单元矩阵并定义界面：

| **输入文件用法：** | ``` [*SUBSTRUCTURE GENERATE](../key/key-link.md#usb-kws-hsubstructuregenerate), ELSET=*element_set* ``` |

### 界面定义

界面节点是子结构与模型其他部分交换力的节点：

| **输入文件用法：** | ``` [*SUBSTRUCTURE GENERATE](../key/key-link.md#usb-kws-hsubstructuregenerate), ELSET=*element_set*, INTERFACE=*node_set* ``` |

### 内部输出控制

可以选择输出子结构内部的详细结果：

| **输入文件用法：** | 使用以下选项请求内部输出： |
| --- | --- |
| | ``` [*SUBSTRUCTURE GENERATE](../key/key-link.md#usb-kws-hsubstructuregenerate), ELSET=*element_set*, INTERNAL=*element_set* ``` |

### 子结构矩阵

子结构生成创建以下矩阵：

| **矩阵** | **描述** |
| --- | --- |
| 刚度矩阵 | K_ss |
| 质量矩阵 | M_ss |
| 阻尼矩阵 | C_ss |
| 载荷向量 | F_s |

### 模态分析

子结构支持模态分析以提取内部振型：

| **输入文件用法：** | ``` [*SUBSTRUCTURE GENERATE](../key/key-link.md#usb-kws-hsubstructuregenerate), ELSET=*element_set*, MODES=*number* ``` |

### 子结构使用

在主分析中使用子结构：

| **输入文件用法：** | ``` [*SUBSTRUCTURE](../key/key-link.md#usb-kws-hsubstructure), ELSET=*element_set* ``` |

### 载荷应用

载荷可以应用到子结构界面：

| **载荷类型** | **描述** |
| --- | --- |
| 位移边界条件 | 作用在界面节点上 |
| 力载荷 | 作用在界面节点上 |
| 分布载荷 | 通过界面传递 |
| 热载荷 | 通过界面传递 |

### 边界条件处理

子结构界面节点的边界条件处理：

| **输入文件用法：** | ``` [*BOUNDARY](../key/key-link.md#usb-kws-hboundary), OP=NEW *node_set*, *dof*, *value* ``` |

### 结果提取

从子结构分析中提取结果：

| **输入文件用法：** | ``` [*OUTPUT](../key/key-link.md#usb-kws-houtput) [*NODE OUTPUT](../key/key-link.md#usb-kws-hnodeoutput) *node_set* [*ELEMENT OUTPUT](../key/key-link.md#usb-kws-helementoutput) *element_set* ``` |

### 内部结果恢复

提取子结构内部详细结果：

| **输入文件用法：** | ``` [*SUBSTRUCTURE GENERATE](../key/key-link.md#usb-kws-hsubstructuregenerate), ELSET=*element_set*, RECOVERY=*element_set* ``` |

### 局限性

- 仅适用于线性分析
- 材料必须是线性或非线性（但分析是线性的）
- 接触不能跨越子结构界面
- 子结构不能嵌套

### 输入文件模板

```
*-heading
Substructure generation and use
**
*NODE, NSET=NALL
*ELEMENT, TYPE=C3D8, ELSET=SUBREGION
*SURFACE, NAME=S1
*ELSET, ELSET=SUBREGION
*SUBSTRUCTURE GENERATE, ELSET=SUBREGION, INTERFACE=NINT
*STEP
*STATIC
*CLOAD
NINT, 3, 10.0
*END STEP
**
*STEP
*STATIC
*SUBSTRUCTURE, ELSET=SUBREGION
*BOUNDARY
NINT, 3, 0.0
*END STEP
```
