# 59.3 通过拾取或从文件读取创建附件点





您可以通过从视口拾取每个点或从文件读取点的坐标来定位附件点。如果需要，您可以通过将指定的点投影到选定的面上来创建附件点。您可以使用附件点来定义点式紧固件的位置。有关更多信息，请参见["关于紧固件，" 第29.1节](pt04ch29s01.md)。

**通过拾取或从文件读取创建附件点的步骤：**

1. 从主菜单栏，选择 ****Tools**![](../graphics/images/arrow.gif)**Attachment**![](../graphics/images/arrow.gif)**Points From File/By Picking****。Abaqus/CAE 显示**创建附件点**对话框。
   **提示：**您也可以使用 ![](../graphics/ico_attPtByPick.png) 工具创建附件点，该工具位于 Interaction 模块或 Property 模块工具箱的附件工具中。有关工具箱中附件工具的图表，请参见["了解附件点和线，" 第59.1节](pt06ch59s01.md)。

2. 在对话框中，使用以下方法之一在表中输入每个附件点的坐标：
   - 选择 ![](../graphics/ico_selectBlue.png) 从视口中的零件或装配拾取点。您可以从零件中选择任何现有点，包括顶点、基准点、兴趣点、参考点和孤立网格节点。您也可以在提示区域出现的文本字段中输入点的 *X*、*Y* 和 *Z* 坐标。
   - 单击 ![](../graphics/ico_fileOpen.png) 读取包含每个点的 *X*、*Y* 和 *Z* 坐标的 ASCII 文件。
   - 单击 ![](../graphics/ico_delete.png) 从表中删除所选行。
   有关更多信息，请参见["输入表格数据，" 第3.2.7节](pt01ch03s02s07.md)。如果在零件上创建附件点，您提供的坐标相对于零件的坐标系。如果在装配上创建附件点，则坐标相对于装配的全局坐标系。

3. 要将指定的点投影到选定的面上，请显示 **Projection** 选项卡页面，然后执行以下操作：
   1. 切换打开 **Project onto faces**。
   2. 单击 ![](../graphics/ico_selectBlue.png)，然后选择要将点投影到的面。这些面可以是平面或非平面。
   3. 选择投影点的方法。
      - 选择 **Proximity** 以允许 Abaqus/CAE 从每个点到所选面上最近的点绘制向量。
      - 选择 **Direction** 以定义沿其投影点的向量。单击 ![](../graphics/ico_selectBlue.png) 以选择向量的**起点**和**终点**。
      有关更多信息，请参见["了解投影方法，" 第59.2节](pt06ch59s02.md)。

4. 默认情况下，Abaqus/CAE 创建一个将包含附件点的集合。如果需要，您可以修改集合名称。如果不想创建包含附件点的集合，请切换关闭 **Create set with name**。

5. 单击 **OK** 创建附件点。Abaqus/CAE 显示附件点。您无法修改附件点；必须删除这些点并创建新点。
