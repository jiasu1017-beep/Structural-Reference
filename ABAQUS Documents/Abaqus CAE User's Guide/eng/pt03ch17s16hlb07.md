# 17.16.7 Deleting part or instance seeds







Delete part or instance seeds by selecting ****Seed**![](../graphics/images/arrow.gif)**Delete Part Seeds**** or ****Seed**![](../graphics/images/arrow.gif)**Delete Instance Seeds**** from the main menu bar. 

**To delete part or instance seeds:**

1. From the main menu bar, select ****Seed**![](../graphics/images/arrow.gif)**Delete Part Seeds**** or ****Seed**![](../graphics/images/arrow.gif)**Delete Instance Seeds****. Abaqus/CAE displays prompts in the prompt area to guide you through the procedure. **Tip:**You can also delete seeds using the ![](../graphics/ico_meshSeedDeleteInstance.png) tool, located with the seed tools in the Mesh module toolbox. (For more information, see ["Using the Mesh module toolbox," Section 17.15](pt03ch17s15.md).)
2. Do one of the following: - If you are deleting part seeds, click **Yes** in the prompt area to confirm the deletion. - If your assembly contains only one part instance, click **Yes** in the prompt area to confirm the deletion. - If your assembly contains more than one part instance, select the instances whose seeds you want to delete and click mouse button 2 to commit your selection. Abaqus/CAE deletes the seeds.
