# 4.1.4 襟翼机构

**产品：** Abaqus/Standard  Abaqus/Explicit   

本示例说明了使用连接器元件对三维后缘机构进行建模。

### 几何与模型

襟翼的完整模型由致动器通过致动器臂的展开旋转钟形摇臂。钟形摇臂推动和拉动连接到襟翼臂的连杆。

襟翼通过 E 和 I 点连接到机翼结构上的刚性轴。钟形摇臂在 B 点旋转，旋转轴平行于全局 Z 轴。致动器系统在 C 点连接到钟形摇臂。

载荷施加在襟翼的重心处，使其沿全局 Z 轴同向且方向一致。

### 模型相互作用

- ACTUATOR 使用 HINGE 连接器元件在 A 点连接到地面。
- BELL CRANK 在 B 点用铰链连接物理连接到地面。
- ROD 使用 JOIN 连接器元件连接。
- FLAP 和 ARM 使用 WELD 连接器元件连接。

### 结果与讨论

当致动系统展开时，襟翼绕 Z 轴旋转以改变机翼的空气动力学特性。

### 输入文件

- flap_model.py - Python 回放文件

- flap_exp_plas.inp - 带塑性的 Abaqus/Explicit 模型
