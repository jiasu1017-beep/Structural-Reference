# 4.4.10 将视口标注复制到另一个数据库





你可以将创建的视口标注从模型数据库（**MDB**）复制到输出数据库（**ODB**），反之亦然。你不能将标注从一个输出数据库复制到另一个输出数据库。Abaqus/CAE 生成的标注包含模型和结果特定的信息；此信息会自动在每个数据库中可用，无法复制（有关详细信息，请参阅[第 56 章，"自定义视口标注"](pt05ch56.md)"）。

**要将视口标注复制到另一个数据库：**

1. 从主菜单栏中，选择****File**![](../graphics/images/arrow.gif)**Open****打开所需的模型数据库和输出数据库（`.cae` 和 `.odb`）文件。**注意：**你不需要对数据库文件具有写权限即可复制标注；但是，如果你没有写权限，则无法保存复制的标注。
2. 从主菜单栏中，选择****Viewport**![](../graphics/images/arrow.gif)**Annotation Manager****。**提示：**你也可以通过单击[**视口**工具栏](pt01ch04s02hlb02.md)中的![](../graphics/ico_manager.png)来显示**标注管理器**。
3. 单击**MDB**或**ODB**单选按钮来选择要复制的标注来源。
   1. 如果你打开了多个输出数据库，或不确定涉及的数据库文件，请将光标悬停在**MDB**或**ODB**单选按钮上，以显示包含数据库路径和文件名的工具提示。
   2. 如有需要，通过在当前视口中显示所需数据库来更改活动输出数据库。或者，你可以打开新视口来显示数据库，或关闭所有其他输出数据库。Abaqus/CAE 会列出所选数据库中可用的标注。
4. 从对话框列表中选择标注（有关详细信息，请参阅["从列表和表中选择多个项目，" 第 3.2.11 节](pt01ch03s02s11.md)）。**提示：**切换**在视口中高亮显示选择**可以预览所选标注。
5. 单击**复制到 MDB**或**复制到 ODB**按钮。Abaqus/CAE 会复制所选标注。
