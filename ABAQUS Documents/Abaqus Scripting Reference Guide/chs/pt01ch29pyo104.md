# 29.104 UserOutputVariables 对象

UserOutputVariables 对象指定用户定义输出变量的数量。

**访问**

```
import material
mdb.models[*name*].materials[*name*].userOutputVariables
import odbMaterial
session.odbs[*name*].materials[*name*].userOutputVariables
```

### 29.104.1 UserOutputVariables(...)

此方法创建 UserOutputVariables 对象。

**路径**

```
mdb.models[*name*].materials[*name*].UserOutputVariables
session.odbs[*name*].materials[*name*].UserOutputVariables
```

**必需参数**

无。

**可选参数**

*n*

Int，指定每个材料点所需的用户定义变量数量。默认值为 0。

**返回值**

UserOutputVariables 对象。

**异常**

RangeError。

### 29.104.2 setValues(...)

此方法修改 UserOutputVariables 对象。

**必需参数**

无。

**可选参数**

`setValues` 的可选参数与 [UserOutputVariables](pt01ch29pyo104.md#ker-useroutputvariables-useroutputvariables-pyc) 方法的参数相同。

**返回值**

无

**异常**

RangeError。
