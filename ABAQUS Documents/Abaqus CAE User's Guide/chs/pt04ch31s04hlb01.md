# 31.4.2 为Abaqus/Standard创建VCCT裂缝





您可以通过执行以下操作来创建可由Abaqus/Standard分析的虚拟裂缝闭合技术（VCCT）裂缝：
- 创建指定裂缝准则的接触相互作用属性。裂缝准则相互作用属性指定沿最初部分粘合表面的裂缝扩展准则。您可以在以下裂缝扩展准则之间进行选择：
   - 虚拟裂缝闭合技术（**VCCT**）准则，它使用线弹性断裂力学原理。更多信息，请参见["裂缝扩展分析"中的"VCCT准则"，Abaqus Analysis User's Guide第11.4.3节](../usb/usb-link.md#usb-anl-acrackpropagation-vcct)。
   - 增强型虚拟裂缝闭合技术（**Enhanced VCCT**）准则，您可以使用两个不同的临界裂缝能量释放率来控制裂缝的萌生和扩展。增强型虚拟裂缝闭合技术仅在Abaqus/Standard分析中可用。更多信息，请参见["裂缝扩展分析"中的"增强型VCCT准则"，Abaqus Analysis User's Guide第11.4.3节](../usb/usb-link.md#usb-anl-acrackpropagation-enhancedvcct)。
- 创建面-面接触相互作用，它使用主从接触表面建模潜在裂缝表面。最初粘合的区域定义从表面上与主表面最初粘合的区域。从表面的未粘合部分作为常规接触表面。详细信息，请参见["裂缝扩展分析"中的"在Abaqus/Standard中定义最初粘合的裂缝表面"，Abaqus Analysis User's Guide第11.4.3节](../usb/usb-link.md#usb-anl-acrackpropagation-initcond)。
- 激活裂缝扩展能力，以指定可以在最初部分粘合的两个表面之间发生裂缝扩展。裂缝继续沿主表面和从表面之间的界面扩展。虚拟裂缝闭合技术无法对尚未裂缝的表面上的裂缝萌生进行建模；因此，您必须在裂缝表面开始时指定从表面和主表面之间的间隙，以模拟预存裂缝。更多信息，请参见["裂缝扩展分析"，Abaqus Analysis User's Guide第11.4.3节](../usb/usb-link.md#usb-anl-acrackpropagation)。

**要创建VCCT裂缝：**

1. 如下所述创建定义机械断裂准则的接触相互作用属性，请参见["定义接触相互作用属性"中的"为裂缝扩展指定裂缝准则属性"，第15.14.1节](pt03ch15s14s01.md#usi-itn-help-prop-contact-mech-fracture)。
   1. 选择裂缝扩展准则（**VCCT**或**Enhanced VCCT**）。
   **注意：****Direction of crack growth relative to 1--direction**选项仅适用于将由扩展有限元方法（XFEM）用于裂缝扩展的富集区域。
   2. 指定裂缝萌生和扩展的临界能量释放率。
2. 如下所述创建指定主表面和从表面（包含潜在裂缝表面的区域）的面-面接触相互作用，请参见["在Abaqus/Standard分析中定义面-面接触"中的"定义面-面接触"，第15.13.7节](pt03ch15s13s01.md#usi-itn-help-surftosurf-std)。
   1. 选择**Surface to surface**的离散化方法。
   2. 从**Bonding**标签页中，切换**Limit bonding to slave nodes in subset**并选择指定从表面最初粘合节点的集合。从表面的未粘合部分作为常规接触表面。
   3. 选择定义机械断裂准则的接触相互作用属性。
3. 激活**VCCT**裂缝扩展能力。
   1. 从Interaction模块主菜单中，选择****Special**![](../graphics/images/arrow.gif)**Crack**![](../graphics/images/arrow.gif)**Create****。
   2. 从出现的**Create Crack**对话框中，选择**Debond using VCCT**。
   3. 输入裂缝名称，然后单击**Continue**关闭对话框。**Edit Crack**对话框出现。
   4. 从**Edit Crack**对话框中，执行以下操作：
      1. 选择启动裂缝扩展的步骤。
      2. 选择指定主表面和从表面的面-面接触相互作用。
      3. 指定在脱粘发生后如何释放定义接触对相互作用的面之间的牵引力：
         - 选择**Step**（默认）在脱粘后的增量期间释放牵引力。
         - 要避免突然失去稳定性，选择**Ramp**以在脱粘后相继增量中逐渐释放牵引力。
      4. 您可以指定将裂缝扩展信息写入数据（`.dat`）文件的速率。如果步骤使用自动时间增量，则该值是脱粘开始后第一个增量的建议时间增量。如果步骤使用固定时间增量，则该值是脱粘开始后的时间增量（如果Abaqus/Standard发现它需要的当前值更小）。时间增量大小将被修改直到脱粘完成。
      5. 单击**OK**配置VCCT裂缝并关闭编辑器。
4. 如果需要，编辑通用解控制并调整**VCCT Linear Scaling**以加速收敛。对于使用VCCT或增强型VCCT准则的大多数裂缝扩展模拟，变形在裂缝扩展起始点之前可能几乎是线性的；过去此点分析变得非常非线性。在这种情况下，线性缩放可用于有效减少到达裂缝萌生点所需的求解时间。更多信息，请参见["自定义通用解控制"，第14.15.1节](pt03ch14s15hlb01.md)。
