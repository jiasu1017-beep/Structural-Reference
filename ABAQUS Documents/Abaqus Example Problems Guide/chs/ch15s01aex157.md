# 15.1.3 计算主应力和应变及其方向：FPRIN





**产品：** Abaqus/Standard

本示例说明了使用FORTRAN程序从Abaqus结果文件读取应力和应变记录，并计算主应力和应变值及其方向。

### 概述

本程序展示了如何从Abaqus结果文件检索积分点和节点平均的应力和应变分量，然后使用Abaqus实用程序例程`SPRIND`计算主值和方向。该实用程序例程的使用在为本问题提供的输入文件中有所记录，关于此子程序接口的更多详细信息在["在Abaqus/Standard分析中获取应力不变量、主应力/应变值和方向，以及旋转张量，" Abaqus User Subroutines Reference Guide的第2.1.11节](../sub/sub-link.md#sub-utl-utensor)中讨论。[FJOIN程序，"合并多个结果文件数据并转换文件格式：FJOIN，"第15.1.2节](ch15s01aex155.md)中创建的`FJOIN`程序创建的结果文件在此用于验证组合在一起的记录是可检索的。之前生成的结果文件名为`fjoinxxx.fin`。要将其用作后处理程序`FPRIN`的输入文件，必须更改文件扩展名。此程序将假定结果文件具有默认的`.fil`扩展名，对应于FORTRAN单元`8`。

### 编程细节

用户应首先审查["Abaqus结果文件用户后处理概述，"第15.1.1节](ch15s01abo02.md)中关于编程概念和Abaqus FORTRAN接口的一般讨论，以及Abaqus Analysis User's Guide第5章"文件输出格式"中给出的后处理详细讨论。

运行程序`FPRIN`时（此程序在Abaqus发布媒体上命名为`fprin.f`），将提示用户输入初始化`FNAME`的文件名。其他变量（如`LOUTF`、`NRU`、`LRUNIT(1,NRU)`和`LRUNIT(2,NRU)`）在程序内部初始化。然后调用`INITPF`和`DBRNU`来完成必要的初始化和文件连接。数据处理从双重`DO`循环开始，通过调用`DBFILE`逐一读取所有要处理的记录。每条记录由其记录键标识，记录键存储在记录的第二条目中。当程序`FPRIN`处理记录`1922`和`2000`时，会写出标题和当前步和增量号，以便提供一种识别每个分析中数据开始的方法。然后检查记录类型`1`，以确定应力和应变的输出位置、直接和剪切应力和应变分量的数量，以及写出记录的单元号或节点号。应力和应变记录（分别为`11`和`21`）将被过滤出来，由Abaqus实用程序例程`SPRIND`处理。当应力和应变记录传入`SPRIND`时，主应力或应变以及相应的主方向以未排序的顺序计算并返回。

### 程序编译和链接

在程序执行之前，FORTRAN程序必须被编译和链接。**abaqus make**工具的单一执行执行这两个操作以及`aba_param.inc`文件的包含：

```
abaqus make job=fprin
```

必须重复此操作，直到所有FORTRAN错误被纠正。成功编译后，程序的目标代码会自动与存储在共享程序库和接口库中的Abaqus目标代码链接，以构建可执行程序。请参阅[Abaqus Installation and Licensing Guide](../sgb/sgb-link.md#sgb-chp-environment)的第4章"自定义Abaqus环境"，以查看特定计算机使用的编译和链接命令。

### 程序执行

在程序执行之前，必须已创建结果文件。本示例使用[FJOIN程序，"合并多个结果文件数据并转换文件格式：FJOIN，"第15.1.2节](ch15s01aex155.md)中讨论的`FJOIN`程序创建的结果文件`fjoinxxx.fin`。必须将此文件重命名为`fjoinxxx.fil`，因为程序中使用FORTRAN单元`8`（与`.fil`文件扩展名关联）来读取文件。当使用命令`abaqus fprin`执行程序时，会出现提示

```
Enter the name of the input file (w/o .fil):
```

输入`fjoinxxx`以定义`FNAME`。程序处理数据并生成名为`pvalue.dat`的文件，其中包含有关主应力和应变及其方向的信息。
