# 10.7.1 导入草图





从主菜单栏选择****文件（File）**![](../graphics/images/arrow.gif)**导入（Import）**![](../graphics/images/arrow.gif)**草图（Sketch）****从以下来源导入草图：
- IGES格式文件（`.igs`文件）
- AutoCAD格式文件（`.dxf`文件）
- ACIS格式文件（`.sat`文件）
- STEP格式文件（`.stp`文件）

如果您要导入草图，文件必须包含可直接映射到草图平面的二维平面轮廓。如果文件包含三维几何，Abaqus/CAE会执行以下操作：
- 如果您要导入AutoCAD文件，Abaqus/CAE仅使用X-Y平面中的几何创建草图。
- 如果您要导入IGES、ACIS或STEP文件，Abaqus/CAE会显示错误消息并取消导入过程。

您可以从仅包含简单几何的文件导入草图，因为Abaqus/CAE必须能够将几何转换为相应的草图器实体，如直线、圆、弧或样条曲线。如果Abaqus/CAE发现无法翻译的几何，它会忽略该几何。有关Abaqus/CAE支持的IGES和AutoCAD实体列表，请参阅["导入的草图，"第20.3.2节"](pt03ch20s03s02.md)。

**要导入草图：**

1. 从主菜单栏，选择****文件（File）**![](../graphics/images/arrow.gif)**导入（Import）**![](../graphics/images/arrow.gif)**草图（Sketch）****。Abaqus/CAE会显示**导入草图（Import Sketch）**对话框。
2. 从**导入草图**对话框底部的**文件过滤器（File Filter）**菜单中，选择以下之一：- **ACIS SAT (*.sat)**- **IGES (*.igs)**- **STEP (*.stp)**- **AutoCAD DXF (*.dxf)**
3. 选择包含要导入的草图的文件，然后单击**确定（OK）**。Abaqus/CAE会将文件中的轮廓转换为草图。草图现在出现在模型的草图列表中，可以在您进入"草图"模块时检索到有关如何使用导入的草图的信息，请参阅["独立草图，"第20.3.1节"](pt03ch20s03s01.md)。
