# 10.1.6 供应商对标准的解释不同





使用公认的行业标准（如IGES和VDA-FS）在CAD系统和Abaqus/CAE之间交换几何信息并不能保证成功。当CAD系统导出零件时，该系统会将其专有的零件表示映射到标准中可用的实体数组。同样，当Abaqus/CAE导入文件时，它会将标准定义的实体转换为其内部表示——ACIS。

ACIS仅识别IGES和VDA-FS标准中定义的一些实体，并且还期望在修剪曲面中具有一定的平滑度或连续性。尽管导出CAD系统不了解Abaqus/CAE的要求，但设置正确的导出选项将增加成功的机会。更多信息，请参阅["实体建模器如何表示实体？，"第10.1.7节"](pt02ch10s01s07.md)和["理解IGES文件的内容，"第10.4节"](pt02ch10s04.md)。

此外，您可能会遇到问题，因为CAD系统对行业标准的解释不同。在许多情况下，定义几何实体有多于一种方式，导致特定"风格"的文件格式。在更极端的情况下，供应商会违反标准，特别是在创建修剪曲面时。
