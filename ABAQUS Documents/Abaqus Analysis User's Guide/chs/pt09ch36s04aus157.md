### 概述

接触属性：
- 定义了控制表面在接触时行为的机械表面相互作用模型；和
- 可选择性地应用于通用接触域内的特定区域。

### 分配接触属性

Abaqus/Explicit 中的默认接触属性模型假设法向"硬"接触、无摩擦、无热相互作用等。您可以为通用接触域的指定区域分配非默认接触属性定义（表面相互作用）。

接触属性分配在通用接触相互作用活跃的所有分析步骤中传播。

用于指定应分配非默认接触属性的区域的表面名称不必与用于指定通用接触域的表面名称对应。在许多情况下，接触相互作用被定义为一个大的域，而非默认接触属性被分配给该域的子集。任何落在通用接触域之外的区域的接触属性分配将被忽略。如果指定的区域重叠，最后的分配优先。

| **输入文件用法：** | ``` [*CONTACT PROPERTY ASSIGNMENT](../key/key-link.md#usb-kws-hcontpropassign) *surface_1*, *surface_2*, *interaction_property_name* ``` |
| --- | --- |
|  | 此选项必须与 [*CONTACT](../key/key-link.md#usb-kws-hcontact) 选项一起使用。它每个步骤最多可出现一次；数据行可以根据需要重复多次以将接触属性分配给不同区域。如果第一个表面名称被省略，则假定一个包含整个通用接触域的默认表面。如果第二个表面名称被省略或与第一个表面名称相同，则假定第一个表面与自身之间的接触。请记住，表面可以定义为跨越多个未连接的体，因此自接触不限于单体检愈与自身的接触。如果省略了相互作用属性名称，则假定 Abaqus/Explicit 中默认接触属性的未命名集。如果指定了相互作用属性名称，它也必须作为输入文件模型部分中 [*SURFACE INTERACTION](../key/key-link.md#usb-kws-hsurfaceinteraction) 选项上 NAME 参数的值出现。 |

| **Abaqus/CAE 用法：** | Interaction 模块： **Create Interaction**: **General contact (Explicit)**: **Contact Properties**: **Individual property assignments: Edit**：在左侧列中选择表面和接触属性，点击中间的箭头将它们转移到接触属性分配列表，或 **Global property assignment:** *interaction_property_name* |
| --- | --- |
|  | 在 Abaqus/CAE 中，您必须为每个通用接触相互作用分配一个接触属性定义；Abaqus/CAE 不假定默认接触相互作用属性。 |

#### 示例

以下接触属性分配在通用接触分析的第一步中指定：
- 对整个通用接触域的全局分配 `contProp1`；
- 对 `surf1` 的自接触的局部分配 `contProp2`；
- 对 `surf2` 和 `surf3` 之间接触的默认 Abaqus 接触属性的局部分配；和
- 对整个接触域与 `surf4` 之间接触的 `contProp3` 的局部分配。

```
[*SURFACE INTERACTION](../key/key-link.md#usb-kws-hsurfaceinteraction), NAME=contProp1
[*FRICTION](../key/key-link.md#usb-kws-hfriction)
0.1
[*SURFACE INTERACTION](../key/key-link.md#usb-kws-hsurfaceinteraction), NAME=contProp2
[*FRICTION](../key/key-link.md#usb-kws-hfriction)
0.15
[*SURFACE INTERACTION](../key/key-link.md#usb-kws-hsurfaceinteraction), NAME=contProp3
[*FRICTION](../key/key-link.md#usb-kws-hfriction)
0.20
[*STEP](../key/key-link.md#usb-kws-hstep)
Step1
[*DYNAMIC](../key/key-link.md#usb-kws-hdynamic), EXPLICIT
…
[*CONTACT](../key/key-link.md#usb-kws-hcontact)
[*CONTACT INCLUSIONS](../key/key-link.md#usb-kws-hcontactinclusions), ALL EXTERIOR
[*CONTACT PROPERTY ASSIGNMENT](../key/key-link.md#usb-kws-hcontpropassign)
 , , contProp1 
surf1, surf1, contProp2 
surf2, surf3,
 , surf4, contProp3 
```

### 更改接触属性

通用接触相互作用的接触属性模型独立于使用它们的步骤，不能从步骤到步骤修改。要更改给定步骤中使用的接触属性，您必须指定引用不同接触属性模型的新接触属性分配。

#### 示例

例如，以下输入可用于更改在前一个示例启动的分析的第二步中用于整个通用接触域与 `surf4` 之间接触的摩擦系数：

```
[*STEP](../key/key-link.md#usb-kws-hstep)
Step2
[*DYNAMIC](../key/key-link.md#usb-kws-hdynamic), EXPLICIT
…
[*CONTACT](../key/key-link.md#usb-kws-hcontact)
[*CONTACT PROPERTY ASSIGNMENT](../key/key-link.md#usb-kws-hcontpropassign)
 , surf4, contProp2
```
