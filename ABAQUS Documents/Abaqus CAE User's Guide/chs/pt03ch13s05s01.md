# 13.5.1 装配模块中的位置工具







每个零件存在于其在零件模块自己的坐标系中，模型实例也在其自己的坐标系中创建。您使用装配模块在全局坐标系中相对彼此定位和定向这些零件和模型的实例。Abaqus/CAE提供以下工具来定位零件和模型实例：

**自动偏移**

当您在装配模块中创建第一个零件或模型实例时，Abaqus/CAE显示一个指示全局坐标系原点和方向的三角坐标。Abaqus/CAE定位第一个实例，使零件或模型的原点与全局坐标系的原点对齐，轴也对齐。如果您创建更多实例，Abaqus/CAE继续定位新实例，使其坐标系与全局坐标系对齐。由于这通常导致新实例与现有实例重叠，Abaqus/CAE允许您在创建实例之前应用偏移。偏移沿*X*轴应用于三维和二维实例，沿*Y*轴应用于轴对称实例。

**基本定位工具**

Abaqus/CAE提供以下基本方法来定位零件和模型实例：
- 您可以通过指定平移向量的起点和终点坐标来沿向量平移所选实例。您可以使用以下方法来确定所选实例移动的距离：- 所选实例沿平移向量从起点移动到终点。- 所选实例沿平移向量从起点向终点移动，并继续移动，直到所选面或边缘与从固定实例选择的面或边缘相距指定距离。有关更多信息，请参见"使用**平移至**工具定位零件或模型实例"，第13.5.4节。
- 您可以绕轴旋转所选实例。您指定旋转轴的起点的X、Y和Z坐标以及旋转轴的终点和旋转角度。

**位置约束工具**

位置约束定义两个实例之间的关系。与简单平移或旋转不同，您不直接指定位置。位置约束定义了一组规则，装配中的零件或模型实例必须始终满足这些规则；例如，必须平行于另一个面的面。

在装配模块中定义的位置约束仅对实例的初始位置创建约束，而在交互模块中定义的约束对分析自由度进行约束。在装配模块中，约束存储为装配的特征。如果您修改零件或移动零件或模型实例，Abaqus/CAE会在重新生成装配时尝试应用所有现有位置约束。每个位置约束都在"位置约束方法如何不同"，第13.5.2节中描述。

创建最终装配是创建实例、应用位置约束以及应用平移和旋转的迭代过程。每次重新定位后，Abaqus/CAE显示指示操作结果的临时图像。您可以接受新位置、取消操作，或通过单击提示区域中的**上一步**按钮![](../graphics/afxI_backup.png)逐步返回重新定位过程。

您可以使用查询工具集获取顶点坐标并测量所选顶点之间的距离。这可以帮助您确定需要沿其平移实例的向量或需要旋转的角度。"使用查询工具集查询装配"，第13.12节包含有关如何获取装配信息的详细说明。
