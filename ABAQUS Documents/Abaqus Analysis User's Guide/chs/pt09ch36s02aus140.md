### 概述

表面属性分配：
- 可用于为表面区域指定几何修正；
- 可用于更改基于结构元素的表面区域使用的接触厚度，或为基于实体元素的表面区域添加接触厚度；
- 可用于为基于壳、膜、刚体和表面元素的表面区域指定表面偏移；
- 可有选择性地应用于一般接触域内的特定区域；以及
- 不能应用于解析刚体表面。

### 分配表面属性

您可以为涉及一般接触交互的表面分配非默认表面属性。这些属性仅在表面涉及一般接触交互时才被考虑；当表面涉及其他交互（如接触对）时，不考虑这些属性。一般接触算法不考虑作为表面定义一部分指定的表面属性。

Abaqus/Standard中的一般接触表面属性在分析开始时分配，在步骤之间不能修改。

用于指定具有非默认表面属性的区域的表面名称不必与用于指定一般接触域的表面名称对应。在许多情况下，接触交互将为一个大域定义，而非默认表面属性将分配给该域的子集。位于一般接触域外的区域的任何表面属性分配都将被忽略。如果指定区域重叠，最后的分配优先。

| **输入文件用法：** | ``` [*SURFACE PROPERTY ASSIGNMENT](../key/key-link.md#usb-kws-hsurfpropassign), PROPERTY ``` |
| --- | --- |
|  | 此选项必须与[*CONTACT](../key/key-link.md#usb-kws-hcontact)选项结合使用，对于下面讨论的PROPERTY参数的每个值最多应出现一次；数据行可以根据需要重复，以将表面属性分配给不同的区域。 |

| **Abaqus/CAE用法：** | Interaction模块：**Create Interaction**：**General contact (Standard)**：**Surface Properties** |

### 表面几何修正

默认情况下，接触计算基于一般接触域中有限元表面的未平滑化 faceted 表示。可选的接触平滑技术模拟接触计算中曲面的更真实表示，从而提高接触应力和压力精度。此接触平滑技术在["Abaqus/Standard中的接触表面平滑，" 第38.1.3节](pt09ch38s01aus179.md)中讨论。

### 表面厚度

默认表面厚度等于原始父单元厚度。或者，您可以指定表面厚度的值或厚度缩放因子。可以为实体单元表面分配非零厚度；例如，用于模拟有限厚度表面涂层的效果。

#### 使用原始父单元厚度

默认表面厚度等于原始父单元厚度。

| **输入文件用法：** | ``` [*SURFACE PROPERTY ASSIGNMENT](../key/key-link.md#usb-kws-hsurfpropassign), PROPERTY=THICKNESS *surface*, ORIGINAL (default) ``` |
| --- | --- |
|  | 如果省略表面名称，则假定为包含整个一般接触域的默认表面。 |

| **Abaqus/CAE用法：** | Interaction模块：**Create Interaction**：**General contact (Standard)**：**Surface Properties**：**Surface thickness assignments: Edit**：选择表面，单击箭头将表面转移到厚度分配列表，并在**Thickness**列中输入ORIGINAL。 |
| --- | --- |

#### 指定表面厚度的值

您可以直接指定表面厚度值。

| **输入文件用法：** | ``` [*SURFACE PROPERTY ASSIGNMENT](../key/key-link.md#usb-kws-hsurfpropassign), PROPERTY=THICKNESS *surface*, *value* ``` |
| --- | --- |
|  | 如果省略表面名称，则假定为包含整个一般接触域的默认表面。 |

| **Abaqus/CAE用法：** | Interaction模块：**Create Interaction**：**General contact (Standard)**：**Surface Properties**：**Surface thickness assignments: Edit**：选择表面，单击箭头将表面转移到厚度分配列表，并在**Thickness**列中输入表面厚度幅值。 |
| --- | --- |

#### 对表面厚度应用缩放因子

您可以对任何表面厚度值应用缩放因子。例如，如果指定对`surf1`使用原始父单元厚度，并应用0.5的缩放因子，则当`surf1`涉及一般接触交互时，将使用原始父单元厚度一半的值（一般接触域中所有其他表面将使用默认的原始父单元厚度）。以这种方式缩放表面厚度可在某些情况下避免初始过盈接触。Abaqus/Standard将自动调整表面位置以解决与一般接触相关的初始过盈接触（请参见["控制Abaqus/Standard中的初始接触状态，" 第36.2.4节](pt09ch36s02aus142.md)）。但是，如果不需要节点位置调整（例如，如果它们会在原本平坦的零件中引入缺陷，导致不现实的屈曲模式），您可能希望减少表面厚度并完全避免过盈接触。

| **输入文件用法：** | ``` [*SURFACE PROPERTY ASSIGNMENT](../key/key-link.md#usb-kws-hsurfpropassign), PROPERTY=THICKNESS *surface*, *value or label*, *scale_factor* ``` |
| --- | --- |
|  | 如果省略表面名称，则假定为包含整个一般接触域的默认表面。 |

| **Abaqus/CAE用法：** | Interaction模块：**Create Interaction**：**General contact (Standard)**：**Surface Properties**：**Surface thickness assignments: Edit**：选择表面，单击箭头将表面转移到厚度分配列表，并输入**Scale Factor**。 |
| --- | --- |

### 表面偏移

表面偏移是薄体中面与其参考平面（由节点坐标和单元连接性定义）之间的距离。它是通过将偏移分数（指定为表面厚度的分数）乘以表面厚度和单元面法线来计算的。这定义了中面的位置，从而定义了身体相对于参考表面的位置；参考表面上节点的坐标不会被修改。表面偏移只能为在壳和类似元素上定义的表面指定（即膜、刚体和表面元素）。为其他元素（如实体或梁元素）指定的表面偏移将被忽略。默认情况下，元素截面定义中指定的表面偏移将在一般接触算法中使用。

您可以将表面偏移指定为表面厚度的分数。表面偏移分数可以设置为与表面父元素使用的偏移分数相等，或设置为指定值。为一般接触指定的表面偏移不会改变单元积分。

| **输入文件用法：** | 使用以下选项使用来自表面父元素的表面偏移分数（默认）： |
| --- | --- |
|  | ``` [*SURFACE PROPERTY ASSIGNMENT](../key/key-link.md#usb-kws-hsurfpropassign), PROPERTY=OFFSET FRACTION *surface*, ORIGINAL ``` 使用以下选项指定表面偏移分数的值： ``` [*SURFACE PROPERTY ASSIGNMENT](../key/key-link.md#usb-kws-hsurfpropassign), PROPERTY=OFFSET FRACTION *surface*, *offset* ``` 偏移可以指定为值或标签（SPOS或SNEG）。指定SPOS等效于指定0.5的值；指定SNEG等效于指定0.5的值。 |

| **Abaqus/CAE用法：** | Interaction模块：**Create Interaction**：**General contact (Standard)**：**Surface Properties**：**Shell/Membrane offset assignments: Edit**：选择表面，单击箭头将表面转移到偏移分配列表。在**Offset Fraction**列中，输入ORIGINAL以使用来自表面父元素的表面偏移分数，输入SPOS以使用0.5的表面偏移分数，输入SNEG以使用0.5的表面偏移分数，或输入表面偏移分数的值。 |
| --- | --- |

### 特征边

模型的特征边在梁和桁架元素上以及实体和结构元素的面的边缘（周长和其他）上定义。Abaqus/Standard中的一般接触包括边对表面接触公式和边对边接触公式（作为表面对表面公式的补充），如["在Abaqus/Standard中定义一般接触交互，" 第36.2.1节](pt09ch36s02aus139.md)中所讨论。默认情况下，边对表面接触公式考虑梁和桁架元素的"边缘"、周长边缘以及初始几何特征角为45及更高的边缘。您可以全局或局部控制边对表面和边对边接触的特征边准则。特征边准则对梁和桁架元素的"边缘"没有影响——它们通过包含在接触域中被激活。

某些接触属性分配选项仅适用于表面对表面公式（有关一般接触的接触属性的进一步讨论，请参见["Abaqus/Standard中一般接触的接触属性，" 第36.2.3节](pt09ch36s02aus141.md)）。边对表面和边对边公式始终使用惩罚 enforcement 方法，仅涉及位移自由度。例如，边对表面公式或交叉边对边公式不会对接触界面的热间隙电导做出贡献。

#### 指定截止特征角

特征角是两个连接边缘的 facet 法线之间形成的角度。facet之间的角度基于初始构型。在facet的凹形连接处产生负角度；因此，这些边缘不会被包含在接触域中。[图36.2.2-1](pt09ch36s04aus156.md#feature-angles)显示了一些如何计算不同边缘特征角的示例。

**图36.2.2-1** 计算特征角。

![](../graphics/feature-angles-nls.png)

边缘A的特征角为90（![](../graphics/usb_eqn00124.gif)和![](../graphics/usb_eqn00125.gif)之间的角度）；边缘B的特征角为25（![](../graphics/usb_eqn00125.gif)和![](../graphics/usb_eqn06156.gif)之间的角度）。边缘C与三个facet形成T形交叉（如图所示，以二维形式显示）；其特征角为0、90和90。

**图36.2.2-2** T形交叉的特征角（例如，[图36.2.2-1](pt09ch36s04aus156.md#feature-angles)中的边缘C）。

![](../graphics/t-intersection-angles-nls.png)

周长边缘（例如，[图36.2.2-1](pt09ch36s04aus156.md#feature-angles)中的边缘D）可以被认为是特征边的一种特殊类型，其特征角为180。

如果特征角准则生效（默认或您指定），则特征角大于或等于指定角度的实体和壳体的几何边缘被包含在一般接触域中。接触包含和排除选项（在["在Abaqus/Standard中定义一般接触交互，" 第36.2.1节](pt09ch36s02aus139.md)中讨论）适用于表面对表面接触公式、边对表面接触公式和边对边接触公式（并进一步控制哪些表面部分可能与任一公式交互）。在确定是否应将几何特征边包含在一般接触域中时，会考虑特征角的符号。例如，如果指定了20的截止特征角，则边缘A将被激活为接触模型中的特征边（因为90的特征角大于20的截止值），但边缘B和C不会被激活（因为边缘B处的特征角为25，边缘C处的最大特征角为0，都小于20的截止值）。截止特征角不能设置为小于0或大于180。指定小的截止特征角（例如小于20）可能会大大增加运行时间，而对结果的影响与较大的截止角（>20）相比不大。边对表面接触的默认特征角截止值为45，而默认不将特征边包含在边对边接触中。用于边对表面接触的包含边缘的准则可以与用于边对边接触的包含边缘的准则不同；两者完全独立。

[图36.2.2-3](pt09ch36s04aus156.md#feature-edges)进一步说明了如何使用特征角来确定一般接触域中激活的几何特征边。

**图36.2.2-3** 对于45的截止特征角，在一般接触域中激活的特征边。

![](../graphics/feature_edges-nls.png)

图右侧的表格列出了模型中各个边缘的特征角值。连接到壳facet但不在壳周长上的边缘有多个对应的特征角。边缘处的最大特征角与默认或指定的截止特征角进行比较。例如，如果默认的截止特征角45生效，则边缘A、D和E将被考虑用于边对表面接触，而边缘B、C和F将被忽略，不用于边对表面接触。

| **输入文件用法：** | ``` [*SURFACE PROPERTY ASSIGNMENT](../key/key-link.md#usb-kws-hsurfpropassign), PROPERTY=FEATURE EDGE CRITERIA *surface*, *feature_angle_value_edge_to_surface*, , *feature_angle_value_edge_to_edge* ``` |
| --- | --- |
|  | 如果省略表面名称，则假定为包含整个一般接触域的默认表面。 |

#### 指定应仅激活周长边缘

您可以指定仅应由边对表面和/或边对边公式考虑周长边缘。周长边缘出现在壳元素的"物理"周长上，也出现在当暴露在身体上的facet子集被包含在一般接触域中时出现的"人工"边缘上。边缘被分类为接触域周长上的边缘（或具有特定特征角的几何边缘）基于接触包含和接触排除定义以及网格特征。当结构元素与连续体单元共享节点时，周长边缘将不会在结构元素上被激活，因为满足将其指定为周长边缘的准则不再成立。

| **输入文件用法：** | ``` [*SURFACE PROPERTY ASSIGNMENT](../key/key-link.md#usb-kws-hsurfpropassign), PROPERTY=FEATURE EDGE CRITERIA *surface*, PERIMETER EDGES, , PERIMETER EDGES ``` |
| --- | --- |
|  | 如果省略表面名称，则假定为包含整个一般接触域的默认表面。 |

| **Abaqus/CAE用法：** | 使用以下选项指定仅应包含周长边缘用于边对表面接触： |
| --- | --- |
|  | Interaction模块：**Create Interaction**：**General contact (Standard)**：**Surface Properties**：**Feature edge criteria assignments: Edit**：选择表面，单击箭头将表面转移到特征分配列表，并在**Feature Edge Criteria**列中输入PERIMETER。在Abaqus/CAE中不支持指定仅应包含周长边缘用于边对边接触。 |

#### 指定不应包含特征边

您可以指定边对表面公式不应考虑任何边缘（全局或局部）。但是，这样做不会停用与梁和桁架单元关联的"接触边缘"。默认情况下，特征边不包含用于边对边接触。

| **输入文件用法：** | ``` [*SURFACE PROPERTY ASSIGNMENT](../key/key-link.md#usb-kws-hsurfpropassign), PROPERTY=FEATURE EDGE CRITERIA *surface*, NO FEATURE EDGES ``` |
| --- | --- |
|  | 如果省略表面名称，则假定为包含整个一般接触域的默认表面。 |

| **Abaqus/CAE用法：** | 使用以下选项指定不包含特征边用于边对表面接触： |
| --- | --- |
|  | Interaction模块：**Create Interaction**：**General contact (Standard)**：**Surface Properties**：**Feature edge criteria assignments: Edit**：选择表面，单击箭头将表面转移到特征分配列表，并在**Feature Edge Criteria**列中输入NONE。 |

#### 控制平滑程度

您可以通过指定分数*f*来控制平滑程度。默认值为0.2。

![](../graphics/usb_eqn06157.gif)

其中![](../graphics/usb_eqn01883.gif)和![](../graphics/usb_eqn06158.gif)是连接到表面节点的两个梁单元的长度，![](../graphics/usb_eqn06159.gif)（见[图36.2.2-4](pt09ch36s02aus140.md#beam-lin-smoothing)和[图36.2.2-5](pt09ch36s02aus140.md#beam-quad-smoothing)）。Abaqus/Standard在距离不连续存在节点![](../graphics/usb_eqn04646.gif)和![](../graphics/usb_eqn04647.gif)处构建抛物线或三次段；此平滑段用于接触计算。因此，接触几何与梁单元几何不同。这种平滑不会发生在三个或更多梁共享同一节点的地方。

| **输入文件用法：** | ``` [*SURFACE PROPERTY ASSIGNMENT](../key/key-link.md#usb-kws-hsurfpropassign), PROPERTY=BEAM SMOOTHING *surface*, *value* ``` |
| --- | --- |
|  | 如果省略表面名称，则假定为包含整个一般接触域中所有梁段的表面。 |

| **Abaqus/CAE用法：** | 在Abaqus/CAE中不支持控制梁段的平滑程度。 |
