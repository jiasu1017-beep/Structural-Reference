# 17.21.3 选择重新网格划分规则尺寸方法





您可以允许 Abaqus/CAE 根据误差指示器输出变量选择默认尺寸方法，也可以指定将用于重新网格划分规则中所有误差指示器的尺寸方法。如果您指定尺寸方法，您还可以指定自适应目标。有关更多信息，请参见["基于解的网格尺寸，" Abaqus Analysis User's Guide 第 12.3.3 节](../usb/usb-link.md#usb-anl-aadpsizing)。

**选择重新网格划分规则尺寸方法的步骤：**

1. 在"创建重新网格划分规则"对话框中，单击**尺寸方法**选项卡。
2. 单击**方法**字段右侧的箭头，然后选择以下网格尺寸算法之一：- 选择**默认方法和参数**以允许 Abaqus/CAE 为每个误差指示器输出变量选择默认计算方法。默认情况下，除元素能量（ENDENERI）和热通量（HFLERI）外的所有误差指示器都使用**最小/最大控制**网格尺寸算法。ENDENERI 和 HFLERI 使用**均匀误差分布**算法。默认误差目标是**自动目标减少**，具有中等网格偏差。- 选择**均匀误差分布**网格尺寸算法，强制 Abaqus/CAE 应用一种尺寸方法，该方法尝试满足模型中每个元素的指定误差目标。在大多数情况下，这种方法会导致全局收敛的网格。- 选择**最小/最大控制**网格尺寸算法，以控制基础解最小值和最大值位置处的网格密度。有关更多信息，请参见["基于解的网格尺寸，" Abaqus Analysis User's Guide 第 12.3.3 节](../usb/usb-link.md#usb-anl-aadpsizing)。
3. 如果您选择了**均匀误差分布**，请指定 Abaqus/CAE 用于确定误差指示器目标的方法：- 选择**自动目标减少**以允许 Abaqus/CAE 生成 successive mesh refinements，这些细化尝试将解误差从自适应过程中先前的作业减少固定量。您在 Job 模块中创建自适应过程时选择最大网格迭代次数。如果您选择**自动目标减少**，Abaqus/CAE 将认为该规则在误差指示器达到 1% 时满足。1% 的标准旨在仅为防止过于昂贵的分析作业提供保护。在大多数情况下，Abaqus/CAE 将完成您在 Job 模块中创建自适应过程时指定的所有重新网格迭代。- 选择**固定目标**，然后输入百分比误差目标。Abaqus/CAE 使用此值应用一种尺寸方法，该方法尝试满足模型中每个元素的误差目标。这种方法确保了全局收敛的网格。Abaqus/CAE 将认为该规则在误差指示器达到误差指示器目标时满足。
4. 如果您选择了**最小/最大控制**，请从**误差指示器目标**区域执行以下操作：1. 指定 Abaqus/CAE 用于确定误差目标的方法：- 选择**自动目标减少**以生成 successive mesh refinements，这些细化尝试将解误差从先前的分析减少固定量。您在 Job 模块中选择最大网格迭代次数。如果您选择**自动目标减少**，Abaqus/CAE 将认为该规则在误差指示器达到 1% 时满足。1% 的标准旨在仅为防止过于昂贵的分析作业提供保护。在大多数情况下，Abaqus/CAE 将完成您在 Job 模块中创建自适应过程时指定的所有重新网格迭代。- 选择**固定目标**，然后在基础解最大值和最小值位置输入百分比误差指示器目标。Abaqus/CAE 使用这些值应用一种异构尺寸方法，该方法尝试在各自的位置满足两个目标。Abaqus/CAE 将认为该规则在对应于基础解最大值的误差指示器达到时满足。此外，两个误差目标帮助 Abaqus/CAE 确保网格细化集中在感兴趣的区域。相反，它们还帮助 Abaqus/CAE 确保不会对基础解结果值较低的区域应用不成比例的细化。2. 指定**网格偏差**。网格偏差进一步调整最大和最小基础解位置之间的尺寸分布。当您选择**强**时，尺寸方法更具攻击性，在基础解强度较高的位置附近聚焦更多单元。当您选择**弱**时，尺寸方法攻击性较低，在基础解强度较高的位置附近生成较少的单元。图 17-142 说明了偏差因子的影响。**图 17-142** 偏差因子对单元尺寸分布的影响。![](../graphics/aadapting-globallocal-targets-nls.png)
