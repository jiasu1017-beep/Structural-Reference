### 3.1.1 标签和按钮概述

Abaqus GUI Toolkit中的多个widget支持标签。例如，如果您想在文本字段前放置标签，应该使用`AFXTextField`而不是创建水平框架并添加标签widget和文本字段widget。以下部分描述了支持标签的特定widget。

标签和按钮构造函数都接受文本字符串参数。此文本字符串可以由三部分组成，每部分用`\t`分隔。文本字符串的三部分是：

**文本**

widget显示的文本。

**提示文本**

将鼠标悬停在widget上一小段时间后显示的文本。如果widget只有一个关联的图标，则必须提供提示文本。

**帮助文本**

在应用程序状态栏中显示的文本（假设应用程序有状态栏）。

此外，您可以通过在文本中的某个字符前加一个与号（`&`）字符来为widget定义键盘加速器。例如，如果为按钮指定字符串`&Calculate`，按钮标签将显示为[图3-1](pt03ch03s01.md#cus-wgt-accelerate)所示。您可以使用加速器按下**[Alt]**键和**[C]**键来调用按钮。

**图3-1** 应用于按钮的键盘加速器。

![](../graphics/cus-wgt-accelerate-nls.png)

### 3.1.2 标签

`FXLabel`widget显示只读字符串。`FXLabel`也可以显示可选图标。

```
FXLabel(parent, 'This is an FXLabel.\tThis is\nthe tooltip')
```

### 3.1.3 按钮

`FXButton`widget包含标签和/或图标。当用户点击按钮时，会调用立即操作。

```
FXButton(parent, 'This is an FXButton')
```

**图3-3** 来自`FXButton`的按钮示例。

![](../graphics/wgt-widget-button-nls.png)

### 3.1.4 复选按钮

`FXCheckButton`widget提供"开/关"切换功能。该按钮还支持第三种"Maybe"或"Some"状态。"Maybe"状态通常用于表示部分选择；例如，`AFXOptionTreeList`widget使用"Maybe"状态。您只能以编程方式设置"Maybe"状态；用户不能将按钮切换到此状态。

```
FXCheckButton(parent, 'This is an FXCheckButton')
```

**图3-4** 来自`FXCheckButton`的复选按钮和标签示例。

![](../graphics/wgt-widget-checkbutton-nls.png)

### 3.1.6 菜单按钮

菜单由以下部分组成：
- 由`AFXMenuTitle`创建的菜单标题。
- 由`AFXMenuPane`创建的菜单窗格。
- 由`AFXMenuCommand`创建的菜单命令。

菜单标题位于菜单栏中，控制与菜单标题关联的菜单窗格的显示。菜单窗格包含菜单命令。菜单命令是通常调用某些操作的按钮。菜单窗格也可以包含由`AFXMenuCascade`创建的级联菜单。级联菜单在菜单中提供子菜单。[图3-6](pt03ch03s01.md#wgt-widget-menu-terms)说明了菜单的组件。

**图3-6** 菜单的组件。

![](../graphics/wgt-widget-menu-terms-nls.png)

以下示例说明了级联菜单的使用：

```
menu = AFXMenuPane(self)
AFXMenuTitle(self, '&Menu1', None, menu)
AFXMenuCommand(self, menu, '&Item 1', None, form1,
    AFXMode.ID_ACTIVATE)
subMenu = AFXMenuPane(self)
AFXMenuCascade(self, menu, '&Submenu', None, subMenu)
AFXMenuCommand(self, subMenu, '&Subitem 1', None,
    form2, AFXMode.ID_ACTIVATE)
```

**图3-7** 来自`AFXMenuCascade`的级联菜单按钮示例。

![](../graphics/wgt-widget-menu-nls.png)

除了使用`&`语法指定助记符（如["标签和按钮"，第3.1节](pt03ch03s01.md)中所述），您还可以在菜单项标签中指定加速器。您可以通过`\t`将加速器与按钮文本分开来指定加速器。例如：

```
AFXMenuCommand(self, menu, 'Graphics Options...\tCtrl+G', None,
    GraphicsOptionsForm(self), AFXMode.ID_ACTIVATE)
```

### 3.1.7 弹出菜单

您可以创建弹出菜单，当用户在widget上点击鼠标按钮3时出现。例如，以下语句说明了如何创建弹出菜单，当用户在树widget上点击鼠标按钮3时显示两个按钮：

```
# In the dialog box constructor:

    def __init__(self, form):

        ...

        FXMAPFUNC(self, SEL_RIGHTBUTTONPRESS, self.ID_TREE,
            MyDB.onCmdPopup)
        FXMAPFUNC(self, SEL_COMMAND, self.ID_TEST1,
            MyDB.onCmdTest1)
        FXMAPFUNC(self, SEL_COMMAND, self.ID_TEST2,
            MyDB.onCmdTest2)

        self.menuPane = None

        FXTreeList(self, 5, self, self.ID_TREE,
            LAYOUT_FILL_X|LAYOUT_FILL_Y|
            TREELIST_SHOWS_BOXES|TREELIST_SHOWS_LINES|
            TREELIST_ROOT_BOXES|TREELIST_BROWSESELECT)
        ...

    def onCmdPopup(self, sender, sel, ptr):

        if not self.menuPane:
            self.menuPane = FXMenuPane(self)
            FXMenuCommand(self.menuPane, 'Test1', None, self,
                self.ID_TEST1)
            FXMenuCommand(self.menuPane, 'Test2', None, self,
                self.ID_TEST2)
            self.menuPane.create()

        status, x, y, buttons = self.getCursorPosition()
        x, y = self.translateCoordinatesTo(self.getRoot(), x, y )
        self.menuPane.popup (None, x, y)

        return 1
```

**注意：**`AFXTable`有自己的弹出菜单命令，您应该使用这些命令而不是本节描述的方法。

### 3.1.8 工具栏和工具箱按钮

`AFXToolButton`widget不在其按钮中显示文本，但按钮通常有工具提示。您可以使用`AFXToolbarGroups`或`AFXToolboxGroups`将通过`AFXToolButton`创建的按钮分组到工具栏或工具箱中。`AFXToolbarGroups`和`AFXToolboxGroups`在工具栏或工具箱中的按钮之间提供视觉分组。例如：

```
# Create toolbar icons
#
group = AFXToolbarGroup(self)
AFXToolButton(group, '\tMy Module\nToolbar Button',
    icon, sel)

# Create toolbox icons
#
group = AFXToolboxGroup(self)
AFXToolButton(group, '\tMy Module\nToolbox Button',
    icon, sel)
```

### 3.1.9 飞出按钮

`AFXFlyoutButton`widget显示一个飞出弹出窗口。飞出弹出窗口包含`AFXFlyoutItem`widget，当用户按下鼠标按钮1并在某个时间跨度内按住鼠标按钮1时会出现。如果用户只是快速点击鼠标按钮1，飞出弹出窗口将不会显示，飞出按钮将像常规按钮一样工作。`AFXFlyoutButton`widget显示当前目标的图标以及右下角的右三角形，以指示可以调用飞出弹出窗口。例如：

```
group = AFXToolbarGroup(self)
popup = FXPopup(getAFXApp().getAFXMainWindow())
AFXFlyoutItem(popup, '\tFlyout Button 1', squareIcon)
AFXFlyoutItem(popup, '\tFlyout Button 2', circleIcon)
AFXFlyoutItem(popup, '\tFlyout Button 3', triangleIcon)
AFXFlyoutButton(group, popup)
popup.create()
```

**图3-8** 来自`AFXFlyoutItem`的飞出按钮示例。

![](../graphics/wgt-widget-flyoutmenu-nls.png)

### 3.1.10 颜色按钮

`AFXColorButton`widget显示一个显示颜色的按钮。点击按钮会显示颜色选择对话框，用户可以使用它来更改按钮的颜色值。例如：

```
AFXColorButton(parent, 'Color:')
```

**图3-9** `AFXColorButton`示例。

![](../graphics/cus-wgt-colorbutton-nls.png)

当连接到`AFXStringKeyword`时，此widget将按钮当前颜色的值以十六进制格式分配给关键字；例如，"#FF0000"。
