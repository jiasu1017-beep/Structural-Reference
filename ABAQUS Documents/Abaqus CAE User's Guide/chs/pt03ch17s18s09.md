# 17.18.11 更改所有节点和单元的标签





从主菜单栏选择 ****Mesh**![](../graphics/images/arrow.gif)**Global Numbering Control**** 可以更改原生区域或装配中选定独立零件实例的节点和/或单元标签。 输入起始标签，Abaqus/CAE会更改节点和单元标签，同时保持原始顺序和增量。 您可以在Abaqus/CAE生成网格之前或之后更改标签。 （有关重新编号孤儿节点的信息，请参阅["节点重新编号，" 第64.5.9节](pt06ch64s05s09.md)和["单元重新编号，" 第64.6.10节](pt06ch64s06s10.md)。）

**更改所有节点和/或单元的标签：**

1. 进入Mesh模块。
2. 从主菜单栏，选择 ****Mesh**![](../graphics/images/arrow.gif)**Global Numbering Control****。
3. 如果您正在处理装配，请选择要修改的独立零件实例。 Abaqus/CAE显示**Global Numbering Control**对话框以及节点和单元的起始标签。 如果您选择了单个零件实例，则显示的值是所选零件实例的节点和单元的当前起始标签。 如果您选择了多个具有相同节点和单元起始标签的零件实例，则显示这些值；否则，不显示任何值。
4. 在**Global Numbering Control**对话框中，输入节点和/或单元的起始标签。 Abaqus/CAE将相同的起始标签应用于您选择的每个独立零件实例。
5. 单击**OK**重新编号节点和/或单元。 如果网格存在，Abaqus/CAE会重新编号节点和/或单元，同时保持原始顺序和增量。 如果尚未创建网格，则在Abaqus/CAE对零件或零件实例进行网格划分时应用新的起始标签。
