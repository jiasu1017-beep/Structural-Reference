# 81.5 我可以用 GUI 插件做什么？





GUI 插件使用 Abaqus GUI Toolkit 编写。有关编写 GUI 脚本的更多详细信息，请参阅 [Abaqus GUI Toolkit User's Guide](../cus/cus-link.md#cus) 和 [Abaqus GUI Toolkit Reference Guide](../gui/gui-link.md#gui)。除了 GUI 命令外，GUI 插件通常还包含一个内核模块，该模块定义了在 GUI 向内核发出命令时将被执行的函数。例如，您可以编写一个 GUI 插件，向用户显示一个用于输入板尺寸的对话框，如 [图81-2](pt08ch81s05.md#usi-plg-dimensionsdb) 所示。

**图81-2** 用于输入板尺寸的对话框。

![](../graphics/usi-plg-dimensionsdb.png)

当用户点击 **OK** 按钮关闭对话框时，GUI 插件会构造一个发送到内核执行的命令。内核函数使用在对话框中输入的尺寸来构建零件。

GUI 插件与内核插件类似，您必须先向 Plug-in 工具集注册它，才能从 Abaqus/CAE 主菜单栏的 **Plug-ins** 菜单执行它。
