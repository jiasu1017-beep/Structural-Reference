# 55.12.2 使用镜像和图案显示结果





在Sketch模块中，您可以使用镜像和图案来复制公共特征并完成特征的草图。同样，在可视化模块中，您可以复制代表模型重复部分的分析结果，以可视化整个模型的结果。要镜像结果，您可以选择坐标系以及多达三个对应于选定系统主平面的镜像。要图案化结果，您可以选择坐标系并应用矩形或圆形图案。您可以将镜像、矩形图案和圆形图案结合使用；并且您可以选择它们的应用顺序以产生所需的视图。

**注意：**没有任何限制来防止复制的结果遮挡原始数据或其他副本在视口中的部分。

镜像和图案仅是可视化辅助手段。任何结果的数值表示（例如轮廓图例）仅指示被分析的部分模型。在选择模型和镜像及图案设置时要非常小心，以确保显示的结果能够准确地模拟完整模型。

**要镜像您的模型：**

1. 找到**Mirror**选项。从主菜单栏选择****View**![](../graphics/images/arrow.gif)**ODB Display Options****。在出现的对话框中单击**Mirror/Pattern**选项卡。**Mirror**选项位于页面上部。
2. 接受默认全局坐标系，或从**Mirror CSYS**列表中选择坐标系。您可以选择输出数据库中存在的任何坐标系。如果合适的坐标系不存在，您可以使用["在后处理期间创建坐标系，" 第42.8节](pt05ch42s06.md)中描述的方法之一创建新的坐标系。
3. 切换打开您希望Abaqus/CAE用于复制模型结果的镜像平面。
4. 如果您还希望Abaqus/CAE复制显示体，则切换打开**Mirror display bodies**。
5. 如果您将镜像与矩形或圆形图案结合使用，请选择Abaqus/CAE将用于创建副本的**Order of Operations**。
6. 单击**Apply**来应用您的更改。您的规范会在所有绘图状态中反映并会在会话期间保存。如果您希望保留所做的更改以供后续会话使用，请将它们保存到文件中。有关更多信息，请参阅["保存自定义设置以供后续会话使用，" 第55.1.1节](pt05ch55s01s01.md)。如果显示轮廓图，轮廓会出现在所有面上。如果显示符号或材料方向图，符号或材料方向仅出现在原始模型面上。

**要图案化您的模型：**

1. 找到**Pattern**选项。从主菜单栏选择****View**![](../graphics/images/arrow.gif)**ODB Display Options****。在出现的对话框中单击**Mirror/Pattern**选项卡。**Pattern**选项位于页面中部。
2. 接受默认全局坐标系，或从**Pattern CSYS**列表中选择坐标系。您可以选择输出数据库中存在的任何坐标系。如果合适的坐标系不存在，您可以使用["在后处理期间创建坐标系，" 第42.8节](pt05ch42s06.md)中描述的方法之一创建新的坐标系。
3. 按如下方式编辑矩形和圆形图案参数：
   - 矩形图案 - 在选定坐标系的*X*、*Y*和*Z*方向上，输入您想要显示的结果**数量**。
   - 输入每个方向上副本之间的**偏移**距离（模型单位）。要将副本紧挨着放置，每个方向的偏移必须与该方向上的模型尺寸相同。**提示：**如果您不知道模型尺寸，请使用查询工具![](../graphics/ico_info.png)来确定所需方向上第一个和最后一个模型节点之间的距离。（有关更多信息，请参阅["可视化模块中的查询工具集概述，" 第50.1节](pt05ch50s01.md)。）
   - 圆形图案 - 选择旋转轴。
   - 输入您想要围绕选定轴显示的结果**数量**。
   - 如果需要，编辑Abaqus/CAE将用于定位副本的**Total angle**。Abaqus/CAE将原始结果放在0度位置。所有后续副本根据**Total angle**除以**Number** of copies来定位。
4. 如果您同时使用矩形和圆形图案，或者将图案与镜像结合使用，请选择Abaqus/CAE将用于创建副本的**Order of Operations**。
5. 单击**Apply**来应用您的更改。您的规范会在所有绘图状态中反映并会在会话期间保存。如果您希望保留所做的更改以供后续会话使用，请将它们保存到文件中。有关更多信息，请参阅["保存自定义设置以供后续会话使用，" 第55.1.1节](pt05ch55s01s01.md)。如果显示轮廓图，轮廓会出现在所有面上。如果显示符号或材料方向图，符号或材料方向仅出现在原始模型面上。
