# 13.10.6 创建零件实例的径向阵列





要在径向阵列中创建所选零件实例的多个副本，请从主菜单栏选择****Instance**![](../graphics/images/arrow.gif)**Radial Pattern****。创建阵列后你不能编辑它。你可以指定以下内容：
- 径向阵列中要创建的副本数量，包括选定实例。你可以创建任意数量的实例。
- 原始实例与阵列中最后一个副本之间的总角度。
- 圆形阵列轴的位置。

更多信息，请参见["创建零件实例阵列，"第13.6节](pt03ch13s06.md)。你不能将径向阵列工具与模型实例一起使用。

**要创建实例的径向阵列：**

1. 从主菜单栏，选择****Instance**![](../graphics/images/arrow.gif)**Radial Pattern****。
   **提示：**你也可以使用装配模块工具箱中的![](../graphics/ico_patternRadial.png)工具来创建零件实例的径向阵列。有关装配工具箱中工具的图表，请参见["使用装配模块工具箱，"第13.9节](pt03ch13s09.md)。
2. 选择要复制的零件实例。
   **提示：**要选择多个实例，请在点击每个实例时按住**[Shift]**键，或在实例周围拖动一个矩形。要取消选择实例，请使用**[Ctrl]****+Click**。更多信息，请参见[第6章"在视口内选择对象"](pt01ch06.md)。
3. 点击鼠标按钮2表示已完成选择实例。Abaqus/CAE显示**Radial Pattern**对话框。
4. 在**Radial Pattern**对话框中，配置径向阵列：
   1. 点击**Number**右侧的箭头增加或减少要创建的副本数量，包括选定实例。当你点击箭头时，装配中的副本数量会更新并提供设置的预览。或者，你可以输入数字并按**[Enter]**预览设置。你可以输入任何大于或等于2的数字。
   2. 输入你所选原始实例与最终副本之间的**Total angle**。角度必须在-360到+360之间。正角度对应逆时针方向。
   3. 默认情况下，Abaqus/CAE绕*Z*轴旋转选定实例来创建阵列。要定义新的旋转轴，点击![](../graphics/ico_selectBlue.png)并从装配中选择一条线来表示新的旋转轴。你也可以从全局坐标系三角轴中选择一个轴。
5. 在大多数情况下，你会在**Radial Pattern**对话框中输入值时预览Abaqus/CAE将创建的径向阵列。但是，如果你选择创建大量副本，预览功能可能会影响Abaqus/CAE的性能。在这种情况下，你应该关闭**Preview**按钮。
6. 要复制更多实例，从第1步开始重复上述步骤。
