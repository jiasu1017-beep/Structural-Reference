# 42.7.2 场输出运算概述

本节列出了可以使用**Create Field Output**对话框对单个场输出变量执行的运算，以及每个运算接受的参数。三角函数的参数假定以弧度为单位。以下按键用于对函数参数进行分类：

**场输出运算的文本按键：**

| A | 参数可以是场输出变量、浮点数或整数。 |
| --- | --- |
| F | 参数必须是浮点数。 |
| FO | 参数必须是场输出变量。 |

**场输出运算：**

| **+** | 执行加法。 |
| --- | --- |
| **** | 执行减法或一元取反。 |
| ***** | 执行乘法。 |
| **/** | 执行除法。 |
| **abs(A)** | 取绝对值。 |
| **acos(A)** | 取反余弦。 |
| **asin(A)** | 取反正弦。 |
| **atan(A)** | 取反正切。 |
| **cos(A)** | 取余弦。 |
| **degreeToRadian(A)** | 将度转换为弧度。 |
| **exp(A)** | 取自然指数。 |
| **exp10(A)** | 取10为底的指数。 |
| **log(A)** | 取自然对数。 |
| **log10(A)** | 取10为底的对数。 |
| **power(FO,F)** | 将场输出对象提升为幂。 |
| **radianToDegree(A)** | 将弧度转换为度。 |
| **sin(A)** | 取正弦。 |
| **sqrt(A)** | 取平方根。 |
| **tan(A)** | 取正切。 |
