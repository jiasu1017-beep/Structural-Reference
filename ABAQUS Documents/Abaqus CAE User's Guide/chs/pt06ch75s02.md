# 75.2 我可以使用虚拟拓扑工具集做什么？







"虚拟拓扑工具集"仅在Mesh模块中可用。您可以将虚拟拓扑应用于部件，也可以将其应用于装配体中的独立实例。"虚拟拓扑工具集"允许您执行以下操作：

**合并面或边**

您可以选择两个或多个面或边，Abaqus/CAE将把它们合并成单个虚拟面或虚拟边。如果您合并两个面，则生成网格时面之间的边将被忽略。同样，如果您合并两条边，则生成网格时边之间的顶点将被忽略。

**忽略边或顶点**

您可以选择Abaqus/CAE将忽略的边或顶点。在两条边之间忽略一条边等同于合并这些面。类似地，在两个顶点之间忽略一个顶点等同于合并这些边；但是，当您有大量面和边要组合时，您会发现组合工具很有用。使用忽略创建虚拟拓扑与使用组合创建虚拟拓扑具有相同的效果——生成网格时，Abaqus/CAE不考虑忽略的边或顶点。

**自动组合或忽略实体**

您可以指定一组几何参数，Abaqus/CAE将使用这些参数创建虚拟拓扑。您还可以选择Abaqus/CAE将应用所选参数的区域——面、部件、部件实例或整个装配体。虚拟拓扑参数位于"创建虚拟拓扑"对话框中，该对话框还包括可用于测量和高亮显示实体以及预览虚拟拓扑的工具。完成后，Abaqus/CAE会创建一个包含所有更改的虚拟拓扑特征。

**恢复实体**

您可以使用**Restore Entities**工具高亮显示所有已被忽略的实体，并选择要重新激活的实体，而不是删除或抑制可能包含您要忽略以及您需要使用的实体的虚拟拓扑特征。

为了保持网格质量，要组合的边和面应具有接近180度的浅包含角度。建议角度在120到240度之间。您可以使用角度方法仅选择具有浅包含角度的相邻面。更多信息，请参见["使用角度和特征边方法选择多个对象," 第6.2.3节"](pt01ch06s02hlb03.md)。如果某些边或顶点处的包含角度在此范围之外，Abaqus/CAE会显示警告并允许您从选择中移除那些边或顶点。如果您选择保留尖锐边，则生成的网格可能不在平滑表面上。[图75-4](pt06ch75s02.md#vto-sharp-edges)展示了虚拟表面上不平滑的网格。

**图75-4** 当您将虚拟拓扑应用于尖锐边和顶点时，生成的网格可能与原始表面有很大偏差。

![](../graphics/vto-sharp-edge.png)

您只能忽略被两个相邻面共享的边或嵌入在面中的边。例如，您不能忽略壳的自由边或被三个面共享的边，如[图75-5](pt06ch75s02.md#vto-edge-what)所示。

**图75-5** 某些边不能被忽略。

![](../graphics/vto-edge-what-nls.png)

类似地，您只能忽略被两条相邻边共享的顶点或嵌入在面中的顶点。例如，您不能忽略线的自由端或线与面相交处的顶点，如[图75-6](pt06ch75s02.md#vto-vertex-what)所示。

**图75-6** 某些顶点不能被忽略。

![](../graphics/vto-vertex-what-nls.png)

Abaqus/CAE不支持在包含虚拟拓扑（组合面或组合边）的区域上使用所有网格划分技术。具体来说，Abaqus/CAE不支持以下内容：
- 使用中轴算法对四边形或四边形主导单元进行二维自由网格划分。
- 使用中轴算法的三维扫掠网格划分。
- 如果要网格划分的区域不是由四个角界定的，则不支持二维结构网格划分。
- 如果要网格划分的区域不是由六面界定的，则不支持三维结构网格划分。

如果您需要将虚拟拓扑应用于从属实例，可以创建原始部件的副本，然后创建该副本的独立实例。然后，您可以用新的独立实例替换从属实例，并对其应用虚拟拓扑。更多信息，请参见["从属和独立部件实例之间有什么区别？," 第13.3.2节"](pt03ch13s03s02.md)。

通过将目标侧上的多个面组合成单个面，您可能能够使部件进行扫掠网格划分；更多信息，请参见["三维实体的扫掠网格划分," 第17.9.3节"](pt03ch17s09s03.md)。但是，如果将许多面组合成单个面，网格划分过程可能会变慢，生成的网格可能不可接受。如果您尝试将大量面组合成单个面，Abaqus/CAE会显示警告并允许您更改选择。Abaqus/CAE从您选择中的面的累计数量计算正在组合的面数。例如，如果您尝试组合已经各自包含五个组合面的两个面，则累计面数将为十。
