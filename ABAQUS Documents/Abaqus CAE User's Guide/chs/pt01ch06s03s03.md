# 6.3.5 修改拖动选择区域的形状





选择工具允许你更改拖动选择区域的形状。从**选择**工具栏中，选择以下之一：

**矩形**

![](../graphics/ico_dragRectangle.png) 点击以指示矩形的一个角落，然后将光标拖动到第二个角落。此工具默认是选中的。

**圆形**

![](../graphics/ico_dragCircle.png) 点击以指示圆心，然后将光标拖动到圆周上的一个点。

**多边形**

![](../graphics/ico_dragPolygon.png) 点击以指示多边形的一个顶点，然后将光标拖动到第二个顶点。然后你继续点击多边形的每个顶点。单击鼠标按钮 2 指示你已完成输入顶点。多边形的顶点数没有限制。
