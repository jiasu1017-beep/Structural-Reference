# 14.7.2 Solver controls







You can customize the variables that control the iterative linear equation solver.

**Note:**You can use the iterative linear equation solver only for **Static, General**; **Static, Linear perturbation**; **Visco**; **Heat transfer**; **Geostatic**; and **Soils** analysis steps.

You can access the solver controls by selecting ****Other**![](../graphics/images/arrow.gif)**Solver Controls**** from the main menu bar. For more information, see ["Iterative linear equation solver," Section 6.1.6 of the Abaqus Analysis User's Guide](../usb/usb-link.md#usb-anl-aitrsolveroverview).

For detailed instructions on setting solver controls, see ["Customizing solver controls," Section 14.15.2](pt03ch14s15hlb02.md).
