# *PLASTIC AXIAL







### *PLASTIC AXIAL定义框架单元的塑性轴向力。

此选项只能与[*FRAME SECTION](ch06abk34.md)选项结合使用。它描述框架单元中的轴向力作为轴向塑性位移的函数。

**产品：**Abaqus/Standard  

**类型：**模型数据  

**级别：**零件，零件实例  

##### **参考：**

- ["框架截面行为，" Abaqus Analysis User's Guide第29.4.2节](../usb/usb-link.md#usb-elm-eusingframesection)
- [*FRAME SECTION](ch06abk34.md)

**此选项没有关联的参数。**

### **定义轴向屈服力随塑性位移变化的数据行：**

**第一行：**

根据需要重复此数据行以定义轴向力与轴向塑性位移之间的关系。至少需要三对数据。
