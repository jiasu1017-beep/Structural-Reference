# 81.6.5 向 Plug-ins 菜单添加 GUI 插件的示例







### 81.6.5 向 **Plug-ins** 菜单添加 GUI 插件的示例

您使用 `registerGuiMenuButton()` 命令使 GUI 插件从 **Plug-ins** 菜单可用。以下示例显示了如何使用注册命令添加一个 GUI 插件，该插件创建一个表单，显示一个对话框提示用户输入。当用户选择 ****Plug-ins**![](../graphics/images/arrow.gif)**My Utility**** 时，Abaqus/CAE 会激活该表单。

内核插件的注册命令必须位于与内核插件代码分开的文件中。相比之下，GUI 插件的注册命令可以位于单独的文件中，也可以位于 GUI 插件文件本身中。此示例显示了如何将注册命令和 GUI 插件代码组合到一个文件中。

```
from abaqusGui import AFXForm, getAFXApp

class MyForm(AFXForm):

    *form code goes here*

toolset = getAFXApp().getAFXMainWindow().getPluginToolset()
toolset.registerGuiMenuButton(buttonText='My Utility', 
    object=MyForm(toolset) )
```

当您使 GUI 插件从 **Plug-ins** 菜单或工具箱可用时，注册命令会引用一个 GUI 对象，该对象在选择插件时接收消息。消息的选择器通过将您指定的消息 ID 与 `SEL_COMMAND` 消息类型组合来生成。在大多数情况下，您将提供一个表单或过程作为 GUI 对象，并使用 `ID_ACTIVATE`（默认值）作为消息 ID。表单和过程会提示用户输入，在大多数情况下使用对话框或提示行。在从用户那里获得输入后，表单和过程会向内核发出命令。有关创建表单、过程和对话框的详细信息，请参阅 [Abaqus GUI Toolkit User's Guide 第7章 "Modes"](../cus/cus-link.md#cus-mod-modes) 和 [Abaqus GUI Toolkit User's Guide 第5章 "Dialog boxes"](../cus/cus-link.md#cus-dlg-dialogs)。您也可以使用 **Really Simple GUI Builder** 插件创建简单的对话框；有关更多信息，请参见 ["使用 **Really Simple GUI (RSG) Dialog Builder** 创建对话框，" 第82.4节](pt08ch82s04.md)。
