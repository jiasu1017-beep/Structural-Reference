# 2.1.3 使用 abaqus_v6.14.gpr 文件







### 2.1.3 使用 `abaqus_v6.14.gpr` 文件

主目录中的 `abaqus_v6.14.gpr` 文件存储 GUI 设置（例如主窗口的大小）以及显示选项设置（例如渲染样式）。您也可以在与主目录不同的目录中存储显示选项设置在 `abaqus_v6.14.gpr` 文件中。如果您使用 **noSavedOptions** 启动 Abaqus/CAE，Abaqus/CAE 不会应用存储在 `abaqus_v6.14.gpr` 文件中的显示选项设置（例如，渲染样式和基准平面的显示）。有关更多信息，请参阅[「启动 Abaqus/CAE（或 Abaqus/Viewer），第 2.1.1 节」](pt01ch02s01s01.md)。

**当您启动 Abaqus/CAE 时**

- GUI 设置从主目录中的 `abaqus_v6.14.gpr` 文件读取。
- 显示选项设置从您启动 Abaqus/CAE 的目录中的 `abaqus_v6.14.gpr` 文件读取。
- 如果该目录中不存在 `abaqus_v6.14.gpr` 文件，但存在较早版本的 `.gpr` 文件，Abaqus/CAE 会尝试应用该文件中指定的设置，并创建一个 `abaqus_v6.14.gpr` 文件来存储设置。
- 如果该目录中不存在 `.gpr` 文件，则显示选项设置从主目录中的 `abaqus_v6.14.gpr` 文件读取。

**在 Abaqus/CAE 会话期间**

您可以使用****文件**![](../graphics/images/arrow.gif)**保存显示选项****将显示选项设置保存到主目录或当前目录中的 `abaqus_v6.14.gpr` 文件中。有关更多信息，请参阅[「保存您的显示选项设置，第 76.16 节」](pt07ch76hla15.md)。此保存选项不适用于 GUI 设置。

**当您退出 Abaqus/CAE 时**

您的 GUI 设置会自动保存到主目录中的 `abaqus_v6.14.gpr` 文件中。有关更多信息，请参阅[「理解 Abaqus/CAE GUI 设置，第 3.6 节」](pt01ch03s06.md)。

您可以使用 Abaqus Scripting Interface 中的 API 命令编辑 `abaqus_v6.14.gpr` 文件；有关更多信息，请参阅[「编辑显示首选项和 GUI 设置，Abaqus Scripting User's Guide 第 8.4 节」](../cmd/cmd-link.md#cmd-int-exa-caeprefsaccess)。您也可以删除该文件以恢复默认的 GUI 和显示选项设置。
