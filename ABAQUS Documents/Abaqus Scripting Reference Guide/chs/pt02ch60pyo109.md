# 60.109 VoidNucleation 对象





VoidNucleation 对象定义多孔材料中孔洞的形核。

**访问**

```
materialApi.materials()[*name*].porousMetalPlasticity().voidNucleation()
```

### 60.109.1 VoidNucleation(...)

此方法创建一个 VoidNucleation 对象。

**路径**

```
materialApi.materials()[*name*].porousMetalPlasticity().VoidNucleation
```

**原型**

```
odb_VoidNucleation&
VoidNucleation(const odb_SequenceSequenceDouble& table,
               bool temperatureDependency,
               int dependencies);
```

**必需参数**

*table*

一个 odb_SequenceSequenceDouble，指定下述项目。

**可选参数**

*temperatureDependency*

一个布尔值，指定数据是否依赖于温度。默认值为 false。

*dependencies*

一个 Int，指定场变量依赖项的数量。默认值为 0。

**表数据**

- ![](../graphics/ker_eqn00398.gif)，成核应变正态分布的平均值。
- ![](../graphics/ker_eqn00399.gif)，成核应变正态分布的标准差。
- ![](../graphics/ker_eqn00400.gif)，成核孔洞的体积分数。
- 温度（如果数据依赖于温度）。
- 第一个场变量的值（如果数据依赖于场变量）。
- 第二个场变量的值。
- 依此类推。

**返回值**

一个 VoidNucleation 对象。

**异常**

RangeError。
