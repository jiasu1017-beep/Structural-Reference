# 16.10.11 定义流体入口/出口边界条件





您可以创建流体入口/出口边界条件来指定模型中特定表面位置的流动条件。有关更多信息，请参见第30章"流体动力学分析"。

**要创建或编辑流体入口/出口边界条件：**

1. 使用以下方法之一显示编辑器：- 要创建新的流体入口/出口边界条件，请按照"创建边界条件"中的步骤操作（第16.8.2节）（**类别**：`Fluid`；**所选步骤的类型**：`Fluid inlet/outlet`）。- 要使用菜单或管理器编辑现有流体入口/出口边界条件，请参见"编辑依赖于步骤的对象"（第3.4.13节）。要编辑应用边界条件的区域，请参见"编辑应用指定条件的区域"（第16.8.4节）。
2. 要基于压力指定动量，请单击**Momentum**选项卡，然后执行以下操作：1. 切换**Specify**。2. 选择**Pressure**。3. 输入入口或出口处的流体压力。4. 如果需要，单击**Amplitude**字段右侧的箭头，然后从出现的列表中选择所需的幅值。或者，您可以单击![](../graphics/ico_amplitudeSmall.png)创建新的幅值。（有关更多信息，请参见第57章"幅值工具集"。）
3. 要基于速度指定动量，请单击**Momentum**选项卡，然后执行以下操作：1. 切换**Specify**。2. 选择**Velocity**。3. 默认情况下，使用全局坐标系来定义边界条件。您可以选择另一个坐标系来指定边界条件；只能选择矩形坐标系。为了避免因有限精度算术而导致的精度损失，当在全局坐标系以外的坐标系中应用流体速度边界条件时，必须输入所有三个分量的值。Abaqus/CAE转换这些值并在全局坐标系中应用它们。单击**CSYS**选项的![](../graphics/ico_selectBlue.png)，然后执行以下操作之一：- 在视口中选择现有的基准坐标系。- 按名称选择现有的基准坐标系。从提示区域，单击**Datum CSYS List**显示基准坐标系列表，从列表中选择一个名称，然后单击**OK**。- 从提示区域单击**Use Global CSYS**恢复为全局坐标系。或者，单击![](../graphics/ico_dtmCsys.png)定义新的基准坐标系。此坐标系编辑选项仅在创建边界条件的步骤中可用。4. 单击**Distribution**字段右侧的箭头，然后从出现的列表中选择所需的选项：- 选择**Uniform**来定义在整个区域上均匀的流体边界条件。- 选择解析场来定义空间变化的流体边界条件。只有对该边界条件类型有效的解析场才显示在选择列表中。或者，您可以单击![](../graphics/ico_analyticalField.png)创建新的解析场。（有关更多信息，请参见第58章"解析场工具集"。）5. 切换您要指定的自由度，然后输入速度值。如果您在全局坐标系以外的坐标系中应用边界条件，则必须输入所有速度分量。6. 如果需要，单击**Amplitude**字段右侧的箭头，然后从出现的列表中选择所需的幅值。或者，您可以单击![](../graphics/ico_amplitudeSmall.png)创建新的幅值。（有关更多信息，请参见第57章"幅值工具集"。）
4. 要指定热能设置，请单击**Thermal Energy**选项卡，然后执行以下操作：1. 切换**Temperature**，然后输入入口或出口处的流体温度。2. 如果需要，单击**Amplitude**字段右侧的箭头，然后从出现的列表中选择所需的幅值。或者，您可以单击![](../graphics/ico_amplitudeSmall.png)创建新的幅值。（有关更多信息，请参见第57章"幅值工具集"。）
5. 要指定湍流设置，请单击**Turbulence**选项卡。- 如果您在当前流动步骤中使用Spalart-Allmaras湍流模型，请执行以下操作：1. 切换**Kinematic eddy viscosity**，然后输入流体的运动涡粘度。2. 如果需要，单击**Amplitude**字段右侧的箭头，然后从出现的列表中选择所需的幅值。或者，您可以单击![](../graphics/ico_amplitudeSmall.png)创建新的幅值。（有关更多信息，请参见第57章"幅值工具集"。）- 如果您在当前流动步骤中使用*k*--![](../graphics/usi_eqn00417.gif) RNG湍流模型，请执行以下操作：1. 切换**Turbulent kinetic energy**；然后输入流体的湍流动能*k*。2. 如果需要，单击**Amplitude**字段右侧的箭头，然后从出现的列表中选择所需的幅值。或者，您可以单击![](../graphics/ico_amplitudeSmall.png)创建新的幅值。（有关更多信息，请参见第57章"幅值工具集"。）3. 切换**Dissipation rate**；然后输入流体的耗散率![](../graphics/usi_eqn00417.gif)。4. 如果需要，单击**Amplitude**字段右侧的箭头，然后从出现的列表中选择所需的幅值。或者，您可以单击![](../graphics/ico_amplitudeSmall.png)创建新的幅值。
6. 单击**OK**保存数据并退出编辑器。
