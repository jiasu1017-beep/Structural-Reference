# 15 Datum commands







Datum commands return Feature objects and inherit the methods of Feature objects; for more details, see [Chapter 20, "Feature commands](pt01ch20.md).” Datums can be created using methods on a Part or Assembly object. 

Each command also creates a Datum object in the corresponding `datum` repository. The Datum object is used as an argument to other commands, such as Part and Partition commands.
