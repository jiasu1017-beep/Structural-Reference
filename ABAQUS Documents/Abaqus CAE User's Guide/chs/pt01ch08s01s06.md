# 8.1.6 将 Abaqus/CAE 图像导入其他软件产品





许多流行的软件应用程序（如文字处理器）允许你导入包含由 Abaqus/CAE 生成的图形图像的文件；大多数这些应用程序允许你预览导入的图像。此外，如果你使用 Windows 系统，你可以使用 **[Ctrl]****+C** 将当前视口中的图像复制到系统剪贴板，并使用 **[Ctrl]****+V** 将其粘贴到另一个应用程序中。Windows 以位图（`.bmp`）格式将图像存储在剪贴板中，分辨率为屏幕分辨率。虽然图像质量在在线查看时会令人满意，但打印时可能不可接受。如果你期望打印图像，应该将其保存为 PostScript 或封装的 PostScript 格式。
