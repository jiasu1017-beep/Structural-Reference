# FXSwitcher





Switcher 布局管理器自动排列其子窗口，使其中的一个位于顶部；所有其他子窗口都被隐藏。Switcher 提供了一种方便的方法来节省屏幕空间，通过排列多个 GUI 面板使其出现在同一空间中，具体取决于上下文。Switcher 忽略其子窗口的所有布局提示——所有子窗口都根据 switcher 布局管理器自己的大小进行拉伸。当使用 SWITCHER_HCOLLAPSE 或 SWITCHER_VCOLLAPSE 选项时，Switcher 的默认大小基于当前子窗口的宽度或高度，而不是所有子窗口的最大宽度或高度。

| --- | --- | --- | --- |

| opts | Int | 0 |  |

| y | Int | 0 |  |
| w | Int | 0 |  |
| h | Int | 0 |  |
| pl | Int | DEFAULT_SPACING |  |
| pr | Int | DEFAULT_SPACING |  |

### getCurrent()
