# A 图标





## 附录A：图标

Abaqus GUI Toolkit支持以下格式来创建图标：
- XPM
- BMP
- GIF
- PNG

您可以使用大多数图像编辑程序来生成支持格式的图标图像。创建图像文件后，通过调用适当的方法来构造图标，如下例所示：
```
icon = afxCreatePNGIcon('myIcon.png')
FXLabel(self, 'A label with an icon', icon)
```

在某些情况下，您可能需要在widget中使用图标之前调用图标的`create`方法。在上一个示例中，不需要调用图标的`create`方法，因为标签widget在创建标签时会创建图标。但是，如果在调用widget的`create`方法之后构造图标，则必须在widget中使用图标之前调用图标的`create`方法。有关更多信息，请参见["create方法，"第3.8节](pt03ch03s08.md)。

XPM图标格式简单；您可以使用任何像素图编辑器，甚至文本编辑器来创建图标数据。有关XPM格式的更多详细信息，请访问[XPM](http://koala.ilog.fr/lehors/xpm.md)网站。以下图像编辑程序支持XPM格式：
- ImageMagick（[www.imagemagick.org](http://www.imagemagick.org)）
- The GIMP（[www.gimp.org](http://www.gimp.org)）

您还可以在XPM网站的FAQ页面中找到像素图编辑器的参考。

作为使用`afxCreateXPMIcon`方法的替代方法，您可以将XPM图像数据定义为Python字符串列表，并使用`FXXPMIcon`方法创建图标，如下例所示。

**注意：**有关有效颜色名称及其对应RGB值的列表，请参见[附录B，"颜色与RGB值"](ap02.md)。要定义透明色，必须将其定义为`"c None s None"`，而不仅仅是`"c None"`。

```
blueIconData = [
"12 12 2 1",
".  c None  s None",
"   c blue",
"            ",
"            ",
"            ",
"            ",
"    ....    ",
"    ....    ",
"    ....    ",
"    ....    ",
"            ",
"            ",
"            ",
"            "
]
blueIcon = FXXPMIcon(getAFXApp(),blueIconData)
```
[图A-1](ap01.md#wgt-icons-bluesquare)显示了该示例创建的蓝色方块图标。
