# 10.2 Valid parts, precise parts, and tolerance







A part that you import into Abaqus/CAE must be valid if you wish to analyze the part with Abaqus/Standard or Abaqus/Explicit. The following sections describe valid and precise parts and how Abaqus/CAE uses imprecise modeling and tolerance to construct an imported part. The following topics are covered:

- ["How are precision and tolerance related?," Section 10.2.2](pt02ch10s02s02.md)
- ["Working with invalid parts," Section 10.2.3](pt02ch10s02s03.md)
