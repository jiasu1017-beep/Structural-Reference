# 70.7.2 使用基准面方法分割单元





您可以通过用基准面切割选定的单元来分割它们，如下图所示：

![](../graphics/par-cell-datum-nls.png)

**使用基准面方法分割单元：**

1. 从主菜单栏中，选择****工具**![](../graphics/images/arrow.gif)**分割****。将出现**创建分割**对话框。Abaqus/CAE在提示区域显示提示以指导您完成过程。**提示：**您也可以使用 ![](../graphics/ico_ptnCellDatumPlane.png) 工具通过基准面方法分割单元，该工具位于模块工具箱的分割单元工具中。有关工具箱中分割工具的图示，请参见["使用分割工具集," 第70.2节](pt06ch70s02.md)。
2. 从对话框顶部的**类型**单选按钮中，选择**单元**。**方法**列表显示您可以用来分割单元的方法。
3. 从方法列表中，选择**使用基准面**。
4. 如果零件或装配体包含多个单元，请选择要分割的单元。您可以使用拖动选择、**[Shift]**+点击和**[Ctrl]**+点击的组合来选择多个单元进行分割。**提示：**如果您无法选择所需的单元，可以使用**选择**工具栏更改选择行为。有关更多信息，请参见["使用选择选项," 第6.3节](pt01ch06s03.md)。Abaqus/CAE高亮显示选定的单元。
5. 选择一个现有的基准面。如果所需位置不存在基准面，请使用基准工具集创建基准。Abaqus/CAE高亮显示选定的基准面。
6. 在提示区域中，单击**创建分割**。Abaqus/CAE创建分割。
