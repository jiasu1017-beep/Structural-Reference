# 57.3 选择要定义的振幅类型







从主菜单栏选择 ****工具**![](../graphics/images/arrow.gif)**振幅**![](../graphics/images/arrow.gif)**创建**** 可创建振幅。有关振幅的详细信息，请参见 Abaqus Analysis User's Guide](../usb/usb-link.md#usb-prc-pamplitude) 的"振幅曲线"第 34.1.2 节，以及 Abaqus Analysis User's Guide](../usb/usb-link.md#usb-anl-aresponsespectrum-spec) 的"响应谱分析"第 6.3.10 节中的"指定频谱"。

**定义振幅曲线：**

1. 从主菜单栏，选择 ****工具**![](../graphics/images/arrow.gif)**振幅**![](../graphics/images/arrow.gif)**创建****。**提示：**您还可以通过在模型树中的 **振幅** 容器上点击鼠标按钮 3，或在 **振幅管理器** 中点击 **创建** 来创建振幅。出现 **创建振幅** 对话框。
2. 在 **名称** 字段中，输入振幅的名称。有关命名对象的信息，请参见 ["使用基本对话框组件，" 第 3.2.1 节](pt01ch03s02s01.md)。
3. 选择您要创建的振幅 **类型**：- 选择 **表格** 将振幅曲线定义为时间刻度上方便点处的值表。Abaqus 在需要时在这些值之间进行线性插值。有关更多信息，请参见 Abaqus Analysis User's Guide](../usb/usb-link.md#usb-prc-pamplitude-data-tabular) 的"振幅曲线"第 34.1.2 节中的"定义表格数据"。- 选择 **等间距** 以在从指定时间值开始的时间间隔处给出振幅值列表。Abaqus 在每个时间间隔之间进行线性插值。有关更多信息，请参见 Abaqus Analysis User's Guide](../usb/usb-link.md#usb-prc-pamplitude-data-eqspaced) 的"振幅曲线"第 34.1.2 节中的"定义等间距数据"。- 选择 **周期** 将振幅 *a* 定义为傅里叶级数：![](../graphics/usi_eqn00777.gif)，其中 ![](../graphics/usi_eqn00778.gif)、*N*、![](../graphics/usi_eqn00122.gif)、![](../graphics/usi_eqn00779.gif)、![](../graphics/usi_eqn00780.gif) 和 ![](../graphics/usi_eqn00781.gif)、![](../graphics/usi_eqn00782.gif) 是用户定义的常数。有关更多信息，请参见 Abaqus Analysis User's Guide](../usb/usb-link.md#usb-prc-pamplitude-data-periodic) 的"振幅曲线"第 34.1.2 节中的"定义周期数据"。- 选择 **调制** 将振幅 *a* 定义为 ![](../graphics/usi_eqn00783.gif)，其中 ![](../graphics/usi_eqn00779.gif)、*A*、![](../graphics/usi_eqn00778.gif)、![](../graphics/usi_eqn00784.gif) 和 ![](../graphics/usi_eqn00785.gif) 是用户定义的常数。有关更多信息，请参见 Abaqus Analysis User's Guide](../usb/usb-link.md#usb-prc-pamplitude-data-mod) 的"振幅曲线"第 34.1.2 节中的"定义调制数据"。- 选择 **衰减** 将振幅 *a* 定义为 ![](../graphics/usi_eqn00786.gif)，其中 ![](../graphics/usi_eqn00779.gif)、*A*、![](../graphics/usi_eqn00778.gif) 和 ![](../graphics/usi_eqn00787.gif) 是用户定义的常数。有关更多信息，请参见 Abaqus Analysis User's Guide](../usb/usb-link.md#usb-prc-pamplitude-data-decay) 的"振幅曲线"第 34.1.2 节中的"定义指数衰减"。- 选择 **解相关** 以根据解相关变量计算振幅值。有关更多信息，请参见 Abaqus Analysis User's Guide](../usb/usb-link.md#usb-prc-pamplitude-data-sol) 的"振幅曲线"第 34.1.2 节中的"为超塑性成形分析定义解相关振幅"。- 选择 **平滑步骤** 将两个连续数据点 ![](../graphics/usi_eqn00788.gif) 和 ![](../graphics/usi_eqn00789.gif) 之间的振幅 *a* 定义为 ![](../graphics/usi_eqn00790.gif)，其中 ![](../graphics/usi_eqn00791.gif)。此定义旨在从一个振幅值平滑地升高或降低到另一个振幅值。有关更多信息，请参见 Abaqus Analysis User's Guide](../usb/usb-link.md#usb-prc-pamplitude-data-smooth) 的"振幅曲线"第 34.1.2 节中的"定义平滑步骤数据"。- 选择 **执行器** 以在任意给定时间从与逻辑建模程序的协同仿真中导入执行器振幅的当前值。有关更多信息，请参见 Abaqus Analysis User's Guide](../usb/usb-link.md#usb-prc-pamplitude-data-actuator) 的"振幅曲线"第 34.1.2 节中的"通过协同仿真定义执行器振幅"。不需要额外数据来定义振幅曲线。- 选择 **频谱** 以定义要在响应谱分析中使用的频谱。有关更多信息，请参见 Abaqus Analysis User's Guide](../usb/usb-link.md#usb-anl-aresponsespectrum-spec) 的"响应谱分析"第 6.3.10 节中的"指定频谱"。- 选择 **用户** 在用户子程序 [`UAMP`](../sub/sub-link.md#sub-xsl-uamp)（Abaqus/Standard）或 [`VUAMP`](../sub/sub-link.md#sub-xsl-vuamp)（Abaqus/Explicit）中定义振幅曲线。有关更多信息，请参见 Abaqus Analysis User's Guide](../usb/usb-link.md#usb-prc-pamplitude-data-user) 的"振幅曲线"第 34.1.2 节中的"通过用户子程序定义振幅"。- 选择 **PSD 定义** 以定义频率函数，该函数定义随机响应分析步骤中随机载荷的频率相关性。此振幅曲线表示随机噪声源的功率谱密度函数。PSD 振幅可以在随机响应步骤中的基础运动边界条件的相关定义中引用。有关更多信息，请参见 Abaqus Analysis User's Guide](../usb/usb-link.md#usb-anl-arandomresponse-psdfunction) 的"随机响应分析"第 6.3.11 节中的"定义频率函数"。
4. 点击 **继续**。出现 **编辑振幅** 对话框，您可以在其中输入定义振幅曲线所需的所有数据。请参阅以下各节获取详细说明：- ["使用表格数据定义振幅曲线，" 第 57.4 节](pt06ch57s04.md)- ["使用等间距数据定义振幅曲线，" 第 57.5 节](pt06ch57s05.md)- ["使用周期数据定义振幅曲线，" 第 57.6 节](pt06ch57s06.md)- ["使用调制数据定义振幅曲线，" 第 57.7 节](pt06ch57s07.md)- ["使用指数衰减数据定义振幅曲线，" 第 57.8 节](pt06ch57s08.md)- ["定义解相关振幅曲线，" 第 57.9 节](pt06ch57s09.md)- ["使用平滑步骤数据定义振幅曲线，" 第 57.10 节](pt06ch57s10.md)- ["定义频谱，" 第 57.11 节](pt06ch57s11.md)- ["定义用户定义振幅曲线，" 第 57.12 节](pt06ch57s12.md)- ["定义 PSD 定义，" 第 57.13 节](pt06ch57s13.md)
