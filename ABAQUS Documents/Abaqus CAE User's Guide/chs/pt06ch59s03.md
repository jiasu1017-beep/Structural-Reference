# 59.7 编辑附件点和线





您可以编辑通过选择方向和间距创建的附件点、按边缘偏移创建的附件点图案以及通过投影点创建的附件线的多个参数。您无法编辑通过拾取或从文件读取创建的附件点。在 Part 模块或 Property 模块中创建的附件特征只能从这两个模块中编辑；在 Assembly 模块或 Interaction 模块中创建的附件特征只能从这两个模块中编辑。默认情况下，Abaqus/CAE 在编辑过程完成时重新生成模型。
