# 11.19.1 使用创建零件对话框定义零件的属性





### 11.19.1 使用**创建零件**对话框定义零件的属性

创建零件时，首先使用**创建零件**对话框定义零件的属性，然后使用**草图**绘制基础特征的二维轮廓。您可以使用**创建零件**对话框定义以下内容：

**名称**

使用**创建零件**对话框顶部的**名称**文本字段命名您正在创建的零件。要重命名零件，请从主菜单栏选择**零件**![](../graphics/images/arrow.gif)**重命名**。

创建零件后，Abaqus/CAE会在当前视口的标题栏中显示新零件的名称。

**建模空间**

使用**建模空间**单选按钮选择新零件的建模空间。您可以定义零件为三维、二维（平面）或轴对称。在Abaqus/CFD模型中，只能创建三维零件。

如果您创建轴对称可变形零件，可以在**创建零件**对话框中切换**包含扭转**以在模型中包含扭转自由度。

**类型**

在Abaqus/Standard或Abaqus/Explicit模型中，使用**类型**单选按钮选择新零件的类型。您可以定义零件为可变形、离散刚体、分析刚体或欧拉。在Abaqus/CFD模型中，只能创建流体零件。

**基础特征**

使用**基础特征**字段定义新零件基础特征的形状和类型。Abaqus/CAE显示的形状和类型选项取决于零件的建模空间和类型。创建零件后，您不能更改零件基础特征的类型。

**近似尺寸**

使用近似尺寸文本字段输入零件的尺寸。您输入的尺寸用于Abaqus/CAE计算草图工作表及其网格的间距。
