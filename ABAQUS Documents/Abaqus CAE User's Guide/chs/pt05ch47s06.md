# 47.7.1 自定义X–Y图的网格线





### 47.7.1 自定义X–Y图的网格线

点击**Grid Display**选项卡来控制*X–Y*图的主网格线和次网格线的外观。[图47-44](pt05ch47s06hlb01.md#viw-gridlines)显示了主网格线和次网格线的示例。

**图47–44** 显示主网格线和次网格线的*X–Y*图。

![](../graphics/viw-gridlines-nls.png)

如果您的*X–Y*图在同一方向使用多个轴，Abaqus/CAE仅为图中第一条曲线的轴绘制网格线。

**Grid Display**选项对四组网格线提供独立控制；即，对任一方向上的主网格线和次网格线的独立控制。您可以显示或隐藏四组中的任何一组，并且可以修改任何一组的线条颜色、样式或粗细。

**注意：**主网格线源于主刻度线，次网格线源于次刻度线。要控制网格线之间的间隔，您必须调整[刻度线的间距](pt05ch47s04hlb06.md)。

**要自定义X–Y图中的网格显示：**

1. 找到轴**Grid Display**选项。选择****Options**![](../graphics/images/arrow.gif)**XY Options**![](../graphics/images/arrow.gif)**Chart****；或点击![](../graphics/ico_xyChartOptions.png)，它位于Visualization模块工具箱中的*X–Y*绘图工具中。在出现的对话框中点击**Grid Display**选项卡。
2. 从**Charts**字段中选择一个或多个要自定义的*X–Y*图。
3. 从**X Gridlines**选项中，切换打开**Major**或**Minor**选项以分别显示主垂直网格线或次垂直网格线。当**Major**或**Minor**打开时，该类型网格线的网格线样式、颜色和粗细选项变得可用。
4. 从**Y Gridlines**选项中，切换打开**Major**或**Minor**以分别显示主水平网格线或次水平网格线。
5. 重复以下步骤来自定义水平主网格线和次网格线以及垂直主网格线和次网格线：1. 选择网格线的颜色：1. 点击颜色样本![](../graphics/usi-colorsample.png)。Abaqus/CAE显示**Select Color**对话框。2. 使用**Select Color**对话框中的方法之一选择新颜色。有关更多信息，请参见["自定义颜色，"第3.2.9节](pt01ch03s02s09.md)。3. 点击**OK**关闭**Select Color**对话框。所选网格线变为指定颜色。2. 选择线条样式：1. 展开样式列表显示线条样式选项（实线、虚线等）。样式列表是两个未标记列表选项中的上一个。2. 从样式列表中，点击所需的线条样式。所选网格线变为指定样式。3. 选择线条粗细：1. 展开粗细列表显示线条粗细选项（实线、虚线等）。粗细列表是两个未标记列表选项中的下一个。2. 从粗细列表中，点击所需的线条粗细。所选网格线变为指定粗细。
6. 点击**Dismiss**关闭**Chart Options**对话框。
