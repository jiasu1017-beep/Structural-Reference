# 17.4.2 我可以在面上或单元上布置种子吗？







您可以选择面或单元进行布种子；但是，Abaqus/CAE 仅沿边缘创建种子。当您选择面或单元进行布种子时，Abaqus/CAE 仅沿面或单元的边缘创建种子。此外，您可以选择集合或曲面进行布种子；因此，Abaqus/CAE 会沿集合或曲面中包含的几何体的边缘创建种子。

当您应用均匀种子时，可以使用以下方法的组合来选择 Abaqus/CAE 将向其应用种子的区域：

**单独/按角度**

您可以单独选择边缘、面或单元；或者您可以使用角度方法选择一组边缘或面。例如，如果您选择角度方法并选择一个边缘，Abaqus/CAE 会选择每个相邻边缘，直到边缘之间的角度等于或超过您输入的角度。 有关更多信息，请参见"使用角度和特征边缘方法选择多个对象"第 6.2.3 节。

**选择过滤器**

您可以使用过滤器选择要选择的对象类型——**Edges**、**Faces**、**Cells** 或 **All**。默认情况下，Abaqus/CAE 允许您选择所有类型的对象。只有在从选择过滤器中选择 **Edges** 或 **Faces** 后，按角度选择选项才可用。 有关更多信息，请参见"根据对象类型过滤您的选择"第 6.3.2 节。

**集合或曲面**

默认情况下，Abaqus/CAE 允许您从视口中选择的边缘、面和单元上应用种子。或者，您可以点击提示区域右侧的 **Sets/Surfaces** 并从符合条件的集合（或曲面）中选择。当您选择集合（或曲面）时，Abaqus/CAE 会将种子应用到集合（或曲面）中的每个边缘，包括所有单元和面的每个边缘。Abaqus/CAE 忽略集合（或曲面）中的任何顶点。
