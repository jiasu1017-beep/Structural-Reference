# 26.4 确保接口区域的节点匹配


您可能在模型定义中共享的区域有不相似的网格。对于 Abaqus/Standard 到 Abaqus/Explicit 的协同仿真,在某些情况下,您可以通过确保接口处的节点匹配来提高解的稳定性和准确性(请参见["结构到结构协同仿真," Abaqus Analysis User's Guide 第 17.3.1 节中的"不相似的网格相关限制"](../usb/usb-link.md#usb-anl-acosimabqtoabq-dissimilar))。本节介绍确保接口区域节点匹配的建议建模实践。

通常,您将在包含接口区域的零件上创建蒙皮或纵梁(取决于接口区域是面还是边缘)、执行各种建模技术,并获取用于在 Abaqus/Standard 模型中定义绑定约束的零件实例。

以下提供了详细步骤。

**确保接口区域节点匹配的步骤:**

1. 在 Abaqus/Explicit 模型中: 1. 在 Property 模块中,显示包含接口区域的零件。如果接口区域是面,则在面上创建蒙皮。如果接口区域是边缘,则在边缘上创建纵梁。更多信息,请参见[第 36 章,"蒙皮和纵梁增强"](pt04ch36.md)。 2. 如果零件是基于几何的,则对零件进行网格划分。 3. 创建一个网格零件(即使您使用的是网格零件)。 4. 删除新创建的网格零件中除蒙皮或纵梁上之外的所有单元。此外,使用 Edit Mesh 工具集删除相关的未引用节点。
2. 在 Abaqus/Standard 模型中: 1. 从 Abaqus/Explicit 模型复制包含蒙皮或纵梁的网格零件,并创建新复制零件的实例。 2. 为简化区域选择过程,创建包含网格零件的命名集或曲面。 3. 在 Interaction 模块中,创建绑定约束,指定复制的网格零件(使用命名集或曲面)作为主区域,将 Abaqus/Standard 模型上的接口区域作为从区域。 4. 在 Interaction 模块中,定义 Standard-Explicit 协同仿真交互,并指定网格零件(使用命名集或曲面)作为接口区域。更多信息,请参见["指定接口区域和耦合方案," 第 26.5 节](pt04ch26s05.md)。
3. 在 Abaqus/Explicit 模型中: 1. 删除包含蒙皮或纵梁的网格零件。 2. 从零件几何中删除蒙皮或纵梁。 3. 如果零件是基于几何的,则重新对零件进行网格划分。 4. 在 Interaction 模块中,定义 Standard-Explicit 协同仿真交互,并指定原始 Abaqus/Explicit 零件中的接口区域作为接口区域。
4. 继续执行["协同仿真概述," 第 26.1 节](pt04ch26s01.md)中所述的协同仿真过程。
