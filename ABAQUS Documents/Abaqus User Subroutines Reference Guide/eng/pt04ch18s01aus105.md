### Overview

User subroutines allow advanced users to customize a wide variety of Abaqus capabilities. Information on writing user subroutines and detailed descriptions of each subroutine appear online in the [Abaqus User Subroutines Reference Guide](../sub/sub-link.md#sub). A listing and explanations of associated utility routines also appear in that guide.

### Available user subroutines for Abaqus/Standard

 The available user subroutines for Abaqus/Standard are as follows:
- [`CREEP`](../sub/sub-link.md#sub-xsl-creep): Define time-dependent, viscoplastic behavior (creep and swelling).
- [`DFLOW`](../sub/sub-link.md#sub-xsl-dflow): Define nonuniform pore fluid velocity in a consolidation analysis.
- [`DFLUX`](../sub/sub-link.md#sub-xsl-dflux): Define nonuniform distributed flux in a heat transfer or mass diffusion analysis.
- [`DISP`](../sub/sub-link.md#sub-xsl-disp): Specify prescribed boundary conditions.
- [`DLOAD`](../sub/sub-link.md#sub-xsl-dload): Specify nonuniform distributed loads.
- [`FILM`](../sub/sub-link.md#sub-xsl-film): Define nonuniform film coefficient and associated sink temperatures for heat transfer analysis.
- [`FLOW`](../sub/sub-link.md#sub-xsl-flow): Define nonuniform seepage coefficient and associated sink pore pressure for consolidation analysis.
- [`FRIC`](../sub/sub-link.md#sub-xsl-fric): Define frictional behavior for contact surfaces.
- [`FRIC_COEF`](../sub/sub-link.md#sub-xsl-fric_coef): Define frictional coefficient for contact surfaces.
- [`GAPCON`](../sub/sub-link.md#sub-xsl-gapcon): Define conductance between contact surfaces or nodes in a fully coupled temperature-displacement analysis, a fully coupled thermal-electrical-structural analysis, or a pure heat transfer analysis.
- [`GAPELECTR`](../sub/sub-link.md#sub-xsl-gapelectr): Define electrical conductance between surfaces in a coupled thermal-electric analysis or a fully coupled thermal-electrical-structural analysis.
- [`HARDINI`](../sub/sub-link.md#sub-xsl-hardini): Define initial equivalent plastic strain and initial backstress tensor.
- [`HETVAL`](../sub/sub-link.md#sub-xsl-hetval): Provide internal heat generation in heat transfer analysis.
- [`MPC`](../sub/sub-link.md#sub-xsl-mpc): Define multi-point constraints.
- [`ORIENT`](../sub/sub-link.md#sub-xsl-orient): Provide an orientation for defining local material directions or local directions for kinematic coupling constraints or local rigid body directions for inertia relief.
- [`RSURFU`](../sub/sub-link.md#sub-xsl-rsurfu): Define a rigid surface.
- [`SDVINI`](../sub/sub-link.md#sub-xsl-sdvini): Define initial solution-dependent state variable fields.
- [`SIGINI`](../sub/sub-link.md#sub-xsl-sigini): Define an initial stress field.
- [`UAMP`](../sub/sub-link.md#sub-xsl-uamp): Specify amplitudes.
- [`UANISOHYPER_INV`](../sub/sub-link.md#sub-xsl-uanisohyper_inv): Define anisotropic hyperelastic material behavior using the invariant formulation.
- [`UANISOHYPER_STRAIN`](../sub/sub-link.md#sub-xsl-uanisohyper_strain): Define anisotropic hyperelastic material behavior based on Green strain.
- [`UCORR`](../sub/sub-link.md#sub-xsl-ucorr): Define cross-correlation properties for random response loading.
- [`UCREEPNETWORK`](../sub/sub-link.md#sub-xsl-ucreepnetwork): Define time-dependent behavior (creep) for models defined within the parallel rheological framework.
- [`UDECURRENT`](../sub/sub-link.md#sub-xsl-udecurrent): Define nonuniform volume current density in an eddy current or magnetostatic analysis.
- [`UDEMPOTENTIAL`](../sub/sub-link.md#sub-xsl-udempotential): Define nonuniform magnetic vector potential on a surface in an eddy current or magnetostatic analysis.
- [`UDMGINI`](../sub/sub-link.md#sub-xsl-udmgini): Define the damage initiation criterion.
- [`UDSECURRENT`](../sub/sub-link.md#sub-xsl-udsecurrent): Define nonuniform surface current density in an eddy current or magnetostatic analysis.
- [`UEL`](../sub/sub-link.md#sub-xsl-uel): Define an element.
- [`UELMAT`](../sub/sub-link.md#sub-xsl-uelmat): Define an element with access to Abaqus materials.
- [`UEXPAN`](../sub/sub-link.md#sub-xsl-uexpan): Define incremental thermal strains.
- [`UEXTERNALDB`](../sub/sub-link.md#sub-xsl-uexternaldb): Manage user-defined external databases and calculate model-independent history information.
- [`UFIELD`](../sub/sub-link.md#sub-xsl-ufield): Specify predefined field variables.
- [`UFLUID`](../sub/sub-link.md#sub-xsl-ufluid): Define fluid density and fluid compliance for hydrostatic fluid elements.
- [`UFLUIDLEAKOFF`](../sub/sub-link.md#sub-xsl-ufluidleakoff): Define the fluid leak-off coefficients for pore pressure cohesive elements.
- [`UGENS`](../sub/sub-link.md#sub-xsl-ugens): Define the mechanical behavior of a shell section.
- [`UHARD`](../sub/sub-link.md#sub-xsl-uhard): Define the yield surface size and hardening parameters for isotropic plasticity or combined hardening models.
- [`UHYPEL`](../sub/sub-link.md#sub-xsl-uhypel): Define a hypoelastic stress-strain relation.
- [`UHYPER`](../sub/sub-link.md#sub-xsl-uhyper): Define a hyperelastic material.
- [`UINTER`](../sub/sub-link.md#sub-xsl-uinter): Define surface interaction behavior for contact surfaces.
- [`UMASFL`](../sub/sub-link.md#sub-xsl-umasfl): Specify prescribed mass flow rate conditions for a convection/diffusion heat transfer analysis.
- [`UMAT`](../sub/sub-link.md#sub-xsl-umat): Define a material's mechanical behavior.
- [`UMATHT`](../sub/sub-link.md#sub-xsl-umatht): Define a material's thermal behavior.
- [`UMESHMOTION`](../sub/sub-link.md#sub-xsl-umeshmotion): Specify mesh motion constraints during adaptive meshing.
- [`UMOTION`](../sub/sub-link.md#sub-xsl-umotion): Specify motions during cavity radiation heat transfer analysis or steady-state transport analysis.
- [`UMULLINS`](../sub/sub-link.md#sub-xsl-umullins): Define damage variable for the Mullins effect material model.
- [`UPOREP`](../sub/sub-link.md#sub-xsl-uporep): Define initial fluid pore pressure.
- [`UPRESS`](../sub/sub-link.md#sub-xsl-upress): Specify prescribed equivalent pressure stress conditions.
- [`UPSD`](../sub/sub-link.md#sub-xsl-upsd): Define the frequency dependence for random response loading.
- [`URDFIL`](../sub/sub-link.md#sub-xsl-urdfil): Read the results file.
- [`USDFLD`](../sub/sub-link.md#sub-xsl-usdfld): Redefine field variables at a material point.
- [`UTEMP`](../sub/sub-link.md#sub-xsl-utemp): Specify prescribed temperatures.
- [`UTRACLOAD`](../sub/sub-link.md#sub-xsl-utracload): Specify nonuniform traction loads.
- [`UTRS`](../sub/sub-link.md#sub-xsl-utrs): Define a reduced time shift function for a viscoelastic material.
- [`UTRSNETWORK`](../sub/sub-link.md#sub-xsl-utrsnetwork): Define a reduced time shift function for models defined within the parallel rheological framework.
- [`UVARM`](../sub/sub-link.md#sub-xsl-uvarm): Generate element output.
- [`UWAVE`](../sub/sub-link.md#sub-xsl-uwave): Define wave kinematics for an Abaqus/Aqua analysis.
- [`UXFEMNONLOCALWEIGHT`](../sub/sub-link.md#sub-xsl-uxfemnonlocalweight): Define the weight function used to compute the average stress/strain to determine the crack propagation direction.
- [`VOIDRI`](../sub/sub-link.md#sub-xsl-voidri): Define initial void ratios.

### Available user subroutines for Abaqus/Explicit

The available user subroutines for Abaqus/Explicit are as follows:
- [`VDISP`](../sub/sub-link.md#sub-xsl-vdisp): Specify prescribed boundary conditions.
- [`VDLOAD`](../sub/sub-link.md#sub-xsl-vdload): Specify nonuniform distributed loads.
- [`VEXTERNALDB`](../sub/sub-link.md#sub-xsl-vexternaldb): Control analyses to exchange data among Abaqus user subroutines and external programs or files.
- [`VFABRIC`](../sub/sub-link.md#sub-xsl-vfabric): Define fabric material behavior.
- [`VFRIC`](../sub/sub-link.md#sub-xsl-vfric): Define contact frictional behavior between surfaces defined with the contact pair algorithm.
- [`VFRIC_COEF`](../sub/sub-link.md#sub-xsl-vfric_coef): Define contact frictional coefficient between surfaces defined with the general contact algorithm.
- [`VFRICTION`](../sub/sub-link.md#sub-xsl-vfriction): Define contact frictional behavior between surfaces defined with the general contact algorithm.
- [`VUAMP`](../sub/sub-link.md#sub-xsl-vuamp): Specify amplitudes.
- [`VUANISOHYPER_INV`](../sub/sub-link.md#sub-xsl-vuanisohyper_inv): Define anisotropic hyperelastic material behavior using the invariant formulation.
- [`VUANISOHYPER_STRAIN`](../sub/sub-link.md#sub-xsl-vuanisohyper_strain): Define anisotropic hyperelastic material behavior based on Green strain.
- [`VUCHARLENGTH`](../sub/sub-link.md#sub-xsl-vucharlength): Define characteristic element length at a material point.
- [`VUEL`](../sub/sub-link.md#sub-xsl-vuel): Define an element.
- [`VUEOS`](../sub/sub-link.md#sub-xsl-vueos): Define equation of a state material model.
- [`VUFIELD`](../sub/sub-link.md#sub-xsl-vufield): Specify predefined field variables.
- [`VUFLUIDEXCH`](../sub/sub-link.md#sub-xsl-vufluidexch): Define mass/heat energy flow rates for fluid exchange.
- [`VUFLUIDEXCHEFFAREA`](../sub/sub-link.md#sub-xsl-vufluidexcheffarea): Define effective area for fluid exchange.
- [`VUHARD`](../sub/sub-link.md#sub-xsl-vuhard): Define the yield surface size and hardening parameters for isotropic plasticity or combined hardening models.
- [`VUINTER`](../sub/sub-link.md#sub-xsl-vuinter): Define the contact interaction between surfaces defined with the contact pair algorithm.
- [`VUINTERACTION`](../sub/sub-link.md#sub-xsl-vuinteraction): Define the contact interaction between surfaces defined with the general contact algorithm.
- [`VUMAT`](../sub/sub-link.md#sub-xsl-vumat): Define material behavior.
- [`VUMULLINS`](../sub/sub-link.md#sub-xsl-vumullins): Define damage variable for the Mullins effect material model.
- [`VUSDFLD`](../sub/sub-link.md#sub-xsl-vusdfld): Redefine field variables at a material point.
- [`VUTRS`](../sub/sub-link.md#sub-xsl-vutrs): Define a reduced time shift function for a viscoelastic material.
- [`VUVISCOSITY`](../sub/sub-link.md#sub-xsl-vuviscosity): Define the shear viscosity for equation of state models.
- [`VWAVE`](../sub/sub-link.md#sub-xsl-vwave): Define wave kinematics for an Abaqus/Aqua analysis.

### Available user subroutines for Abaqus/CFD

The available user subroutines for Abaqus/CFD are as follows:
- [`SMACfdUserPressureBC`](../sub/sub-link.md#sub-xsl-smacfduserpressurebc): Specify prescribed pressure boundary conditions.
- [`SMACfdUserVelocityBC`](../sub/sub-link.md#sub-xsl-smacfduservelocitybc): Specify prescribed velocity boundary conditions.
