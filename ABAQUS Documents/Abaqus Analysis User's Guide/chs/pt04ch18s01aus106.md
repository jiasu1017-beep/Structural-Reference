# 18.1.3 可用的实用程序





**产品：** Abaqus/Standard  Abaqus/Explicit  Abaqus/Aqua  

##### **参考文献**

- ["用户子程序：概述，" 第 18.1.1 节](pt04ch18s01aus104.md)
- ["实用程序，" Abaqus 用户子程序参考指南第 2.1 节](../sub/sub-link.md#sub-utl-utilities)

### 概述

提供了多种实用程序来协助编写用户子程序。调用时，实用程序将执行预定义的功能或操作，其输出或结果可以集成到用户子程序中。

### 可用的实用程序

以下实用程序可用于 Abaqus 中用户子程序的编写：
- `GETENVVAR` 或 `VGETENVVAR` 可以从任何 Abaqus/Standard 或 Abaqus/Explicit 用户子程序中调用，分别用于获取环境变量的值（["获取 Abaqus 环境变量，" Abaqus 用户子程序参考指南第 2.1.1 节](../sub/sub-link.md#sub-utl-ugetenvvar)）。
- `GETJOBNAME` 或 `VGETJOBNAME` 可以从任何 Abaqus/Standard 或 Abaqus/Explicit 用户子程序中调用，分别用于获取当前分析作业的名称（["获取 Abaqus 作业名称，" Abaqus 用户子程序参考指南第 2.1.2 节](../sub/sub-link.md#sub-utl-ugetjobname)）。
- `GETOUTDIR` 或 `VGETOUTDIR` 可以从任何 Abaqus/Standard 或 Abaqus/Explicit 用户子程序中调用，分别用于获取放置分析作业输出的目录名称（["获取 Abaqus 输出目录名称，" Abaqus 用户子程序参考指南第 2.1.3 节](../sub/sub-link.md#sub-utl-ugetoutdir)）。
- `GETNUMCPUS` 可以从任何 Abaqus/Standard 用户子程序中调用，以获取 MPI 进程数；`VGETNUMCPUS` 可以从任何 Abaqus/Explicit 用户子程序中的域并行运行中调用，以获取并行运行使用的进程数（["获取并行进程信息，" Abaqus 用户子程序参考指南第 2.1.4 节](../sub/sub-link.md#sub-utl-ugetnumcpus)）。
- `GETRANK` 可以从任何 Abaqus/Standard 用户子程序中调用，以获取调用函数的 MPI 进程的 rank；`VGETRANK` 可以从任何 Abaqus/Explicit 用户子程序中的域并行运行中调用，以获取各个进程的 rank（["获取并行进程信息，" Abaqus 用户子程序参考指南第 2.1.4 节](../sub/sub-link.md#sub-utl-ugetnumcpus)）。
- `get_thread_id` 可以从任何 Abaqus 用户子程序中调用，以检索 Abaqus 分配给该线程的 ID（["获取并行进程信息，" Abaqus 用户子程序参考指南第 2.1.4 节](../sub/sub-link.md#sub-utl-ugetnumcpus)）。
- `GETPARTINFO` 或 `VGETPARTINFO` 可以分别从任何 Abaqus/Standard 或 Abaqus/Explicit 用户子程序中调用，以检索部件实例名称以及与内部节点或单元编号对应的本地节点或单元编号。`GETINTERNAL` 或 `VGETINTERNAL` 可以分别从任何 Abaqus/Standard 或 Abaqus/Explicit 用户子程序中调用，以检索与给定部件实例名称和本地编号对应的内部节点或单元编号（["获取部件信息，" Abaqus 用户子程序参考指南第 2.1.5 节](../sub/sub-link.md#sub-utl-ugetpartinfo)。）
- `GETVRM` 为 Abaqus/Standard 用户子程序 [`UVARM`](../sub/sub-link.md#sub-xsl-uvarm)、[`UDMGINI`](../sub/sub-link.md#sub-xsl-udmgini) 或 [`USDFLD`](../sub/sub-link.md#sub-xsl-usdfld) 提供对材料点信息的访问（["在 Abaqus/Standard 分析中获取材料点信息，" Abaqus 用户子程序参考指南第 2.1.6 节](../sub/sub-link.md#sub-utl-ugetvrm)）。
- `VGETVRM` 为 Abaqus/Explicit 用户子程序 [`VUSDFLD`](../sub/sub-link.md#sub-xsl-vusdfld) 提供对材料点选定输出变量的访问（["在 Abaqus/Explicit 分析中获取材料点信息，" Abaqus 用户子程序参考指南第 2.1.7 节](../sub/sub-link.md#sub-utl-uvgetvrm)）。
- `GETVRMAVGATNODE` 为 Abaqus/Standard 用户子程序 [`UMESHMOTION`](../sub/sub-link.md#sub-xsl-umeshmotion) 提供对材料点信息的访问，并外推和平均到节点（["在节点处获取平均的材料点信息，" Abaqus 用户子程序参考指南第 2.1.8 节](../sub/sub-link.md#sub-utl-ugetvrmavgatnode)）。
- `GETVRN` 为 Abaqus/Standard 用户子程序 [`UMESHMOTION`](../sub/sub-link.md#sub-xsl-umeshmotion) 提供对节点点信息的访问（["获取节点点信息，" Abaqus 用户子程序参考指南第 2.1.9 节](../sub/sub-link.md#sub-utl-ugetvrn)）。
- `GETNODETOELEMCONN` 可以从用户子程序 [`UMESHMOTION`](../sub/sub-link.md#sub-xsl-umeshmotion) 中调用，以检索连接到特定节点的单元列表。然后可以将此单元列表与实用程序 `GETVRMAVGATNODE` 一起使用（["获取节点到单元的连接，" Abaqus 用户子程序参考指南第 2.1.10 节](../sub/sub-link.md#sub-utl-ugetnodetolemconn)）。
- `SINV` 确定 Abaqus/Standard 中给定应力张量的第一和第二应力不变量（["在 Abaqus/Standard 分析中获取应力不变量、主应力/应变值和方向以及旋转张量，" Abaqus 用户子程序参考指南第 2.1.11 节](../sub/sub-link.md#sub-utl-utensor)）。
- `SPRINC` 或 `VSPRINC` 分别确定 Abaqus/Standard 或 Abaqus/Explicit 中给定应力或应变张量的主值（["在 Abaqus/Standard 分析中获取应力不变量、主应力/应变值和方向以及旋转张量，" Abaqus 用户子程序参考指南第 2.1.11 节](../sub/sub-link.md#sub-utl-utensor)，以及 ["在 Abaqus/Explicit 分析中获取主应力/应变值和方向，" Abaqus 用户子程序参考指南第 2.1.12 节](../sub/sub-link.md#sub-utl-uvtensor)）。
- `SPRIND` 或 `VSPRIND` 分别确定 Abaqus/Standard 或 Abaqus/Explicit 中给定应力或应变张量的主值和主方向（["在 Abaqus/Standard 分析中获取应力不变量、主应力/应变值和方向以及旋转张量，" Abaqus 用户子程序参考指南第 2.1.11 节](../sub/sub-link.md#sub-utl-utensor)，以及 ["在 Abaqus/Explicit 分析中获取主应力/应变值和方向，" Abaqus 用户子程序参考指南第 2.1.12 节](../sub/sub-link.md#sub-utl-uvtensor)）。
- `ROTSIG` 可以从 Abaqus/Standard 用户子程序 [`UMAT`](../sub/sub-link.md#sub-xsl-umat) 中调用，以在大应变计算时执行张量旋转（["在 Abaqus/Standard 分析中获取应力不变量、主应力/应变值和方向以及旋转张量，" Abaqus 用户子程序参考指南第 2.1.11 节](../sub/sub-link.md#sub-utl-utensor)）。
- `GETWAVE` 确定 Abaqus/Aqua 分析中与应用波浪理论相关的波浪运动数据（["在 Abaqus/Aqua 分析中获取波浪运动数据，" Abaqus 用户子程序参考指南第 2.1.13 节](../sub/sub-link.md#sub-utl-uwavekinematic)）。
- `GETWAVEVEL`、`GETWINDVEL` 和 `GETCURRVEL` 用于分别获取 Abaqus/Aqua 分析中给定点的波浪、风和稳态流速度分量（["在 Abaqus/Aqua 分析中获取波浪运动数据，" Abaqus 用户子程序参考指南第 2.1.13 节](../sub/sub-link.md#sub-utl-uwavekinematic)）。
- `STDB_ABQERR` 或 `XPLB_ABQERR` 可以分别从任何 Abaqus/Standard 或 Abaqus/Explicit 用户子程序中调用，以将信息性、警告或错误消息打印到 Abaqus/Standard 的消息文件或 Abaqus/Explicit 的状态文件（["向消息文件或状态文件打印消息，" Abaqus 用户子程序参考指南第 2.1.14 节](../sub/sub-link.md#sub-utl-uabqerr)）。
- `XIT` 或 `XPLB_EXIT` 可以分别从任何 Abaqus/Standard 或 Abaqus/Explicit 用户子程序中调用，以终止分析（["终止分析，" Abaqus 用户子程序参考指南第 2.1.15 节](../sub/sub-link.md#sub-utl-uxit)）。
- `IGETSENSORID` 或 `IVGETSENSORID` 可以分别从 Abaqus/Standard 用户子程序 [`UAMP`](../sub/sub-link.md#sub-xsl-uamp) 或 Abaqus/Explicit 用户子程序 [`VUAMP`](../sub/sub-link.md#sub-xsl-vuamp) 中调用，以获取用户定义传感器的 ID。`GETSENSORVALUE` 或 `VGETSENSORVALUE` 可以分别从 Abaqus/Standard 用户子程序 [`UAMP`](../sub/sub-link.md#sub-xsl-uamp) 或 Abaqus/Explicit 用户子程序 [`VUAMP`](../sub/sub-link.md#sub-xsl-vuamp) 中调用，以获取用户定义传感器的值（["获取传感器信息，" Abaqus 用户子程序参考指南第 2.1.16 节](../sub/sub-link.md#sub-utl-ugetsensor)）。
- `MATERIAL_LIB_MECH` 可以从 Abaqus/Standard 用户子程序 [`UELMAT`](../sub/sub-link.md#sub-xsl-uelmat) 中调用，以访问 Abaqus 材料库（["访问 Abaqus 材料，" Abaqus 用户子程序参考指南第 2.1.17 节](../sub/sub-link.md#sub-utl-ugetmaterialmech)）。
- `MATERIAL_LIB_HT` 可以从 Abaqus/Standard 用户子程序 [`UELMAT`](../sub/sub-link.md#sub-xsl-uelmat) 中调用，以访问 Abaqus 热材料库（["访问 Abaqus 热材料，" Abaqus 用户子程序参考指南第 2.1.18 节](../sub/sub-link.md#sub-utl-ugetmaterialht)）。
- `SMACfdUserSubroutineGetScalar` 可以从任何 Abaqus/CFD 用户子程序中调用，以访问属于边界条件定义一部分的单元或表面小平面的选定输出变量（["在 Abaqus/CFD 分析中获取标量状态信息，" Abaqus 用户子程序参考指南第 2.1.19 节](../sub/sub-link.md#sub-utl-ucfdgetscalar)）。
- `SMACfdUserSubroutineGetVector` 可以从任何 Abaqus/CFD 用户子程序中调用，以访问属于边界条件定义一部分的单元和表面小平面的选定输出变量（["在 Abaqus/CFD 分析中获取矢量状态信息，" Abaqus 用户子程序参考指南第 2.1.20 节](../sub/sub-link.md#sub-utl-ucfdgetvector)）。
- `SMACfdUserSubroutineGetMpiComm` 可以从任何 Abaqus/CFD 用户子程序中调用，以获取并行分析作业中使用的 MPI 通信器（["在 Abaqus/CFD 分析中获取 MPI 通信器，" Abaqus 用户子程序参考指南第 2.1.21 节](../sub/sub-link.md#sub-utl-ucfdgetmpicomm)）。
- `MutexInit`、`MutexLock` 和 `MutexUnlock` 可用于创建和操作互斥锁（["确保线程安全，" Abaqus 用户子程序参考指南第 2.1.22 节](../sub/sub-link.md#sub-utl-ugetthreadid)）。
- 线程本地数组可用于分配存储。`SMALocalIntArrayCreate` 和 `SMALocalFloatArrayCreate` 可用于创建或调整本地数组的大小。`SMALocalIntArrayAccess` 和 `SMALocalFloatArrayAccess` 可用于定位现有本地数组。`SMALocalIntArrayDelete` 和 `SMALocalFloatArrayDelete` 可用于删除本地数组。`SMALocalIntArraySize` 和 `SMALocalFloatArraySize` 可用于获取数组的大小（["可分配数组，" Abaqus 用户子程序参考指南第 2.1.23 节](../sub/sub-link.md#sub-utl-localarrays)）。
- 全局数组也可用于分配存储。`SMAIntArrayCreate` 和 `SMAFloatArrayCreate` 可用于创建或调整全局数组的大小。`SMAIntArrayAccess` 和 `SMAFloatArrayAccess` 可用于定位现有全局数组。`SMAIntArrayDelete` 和 `SMAFloatArrayDelete` 可用于删除全局数组。`SMAIntArraySize` 和 `SMAFloatArraySize` 可用于获取全局数组的大小（["可分配数组，" Abaqus 用户子程序参考指南第 2.1.23 节](../sub/sub-link.md#sub-utl-localarrays)）。
