# 20.2 进入和退出Sketch模块





您可以在Abaqus/CAE会话期间的任何时候通过单击上下文栏中**Module**列表内的**Sketch**来进入Sketch模块。 Sketch模块工具出现在模块工具箱中，**Sketch**菜单出现在主菜单栏上。

要退出Sketch模块，首先通过单击鼠标按钮2退出当前绘图工具。 然后单击提示区域出现的**Done**按钮，并从**Module**列表中选择任何其他模块。 退出模块前无需采取任何特定操作来保存草图； 当您通过从主菜单栏选择 ****File**![](../graphics/images/arrow.gif)**Save**** 或 ****File**![](../graphics/images/arrow.gif)**Save As**** 保存整个模型时，草图会自动保存。

您应该认识到Sketch模块和Sketcher之间的区别。 当您创建或编辑独立草图时，Sketch模块会启动Sketcher并在模块工具箱中显示Sketcher工具。 此外，每当你执行以下操作之一时，Abaqus/CAE都会启动Sketcher：
- 在Part模块中定义零件时创建或编辑特征。
- 在Part模块中工作时在面或单元上绘制划分。
- 在Assembly模块或Mesh模块中工作时在装配的面或单元上绘制划分。

要退出Sketcher并将草图合并到零件或装配中，首先通过单击鼠标按钮2退出当前绘图工具。 然后，单击提示区域出现的**Done**按钮。 在适用的情况下，按照其他提示拉伸、旋转或扫掠特征。 Abaqus/CAE退出Sketcher并返回调用它的模块。 此外，Abaqus/CAE会恢复零件或装配的原始视图。
