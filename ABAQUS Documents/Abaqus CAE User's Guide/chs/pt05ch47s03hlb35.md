# 47.4.37 查找两个或多个X–Y数据对象的当前范围





### 47.4.37 查找两个或多个X–Y数据对象的当前范围

使用**rng**函数对两个或多个[之前保存的](pt05ch47s02hlb07.md) *X–Y*数据对象（每个对象是有序对的集合）进行运算，生成一个新的*X–Y*数据对象。新的*X–Y*数据对象的*X*坐标包含所有输入数据对象的*X*坐标以及为对齐输入数据对象所需的任何其他*X*坐标。Abaqus/CAE通过[插值和外推](pt05ch47s03s02.md)计算额外的*X–Y*数据对。Abaqus/CAE将新数据对象的*Y*坐标计算为当前*X*坐标处最小*Y*坐标与最大*Y*坐标之间的差值。此操作的参数是可交换的。

[图47-33](pt05ch47s03hlb35.md#usv-xyplot-rangeof2)展示了一种可以使用**rng**函数生成的*X–Y*图。

**图47–33** 使用**rng**函数生成的*X–Y*图。

![](../graphics/usv-xyplot-rangeof2.png)

有关如何查找单个*X–Y*数据对象的当前范围的信息，请参见["查找X–Y数据对象的当前范围，"第47.4.34节](pt05ch47s03hlb32.md)。

**要查找两个或多个X–Y数据对象的当前范围：**

1. 找到**Operate on XY Data**对话框。从主菜单栏，选择****Tools**![](../graphics/images/arrow.gif)**XY Data**![](../graphics/images/arrow.gif)**Create****。在出现的对话框中点击**Operate on XY data**；然后点击**Continue**。**Operate on XY Data**对话框出现。
2. 从列出的**Operators**中，点击**rng(A,A,...)**。**rng**函数出现在表达式窗口中。
3. 从**XY Data**选项中，点击要运算的一个或多个*X–Y*数据对象的名称，然后点击**Add to Expression**。选项包括本会话中之前保存的所有*X–Y*数据对象（按字母顺序列出）。*X–Y*数据对象名称以逗号分隔，出现在表达式窗口中**rng**函数括号内。
4. 要继续构建表达式，将光标放在表达式窗口中，然后输入或选择要包含的函数、运算符和*X–Y*数据。
5. 要计算并显示表达式，点击**Plot Expression**。
6. 要[保存](pt05ch47s02hlb07.md)新的*X–Y*数据对象，点击**Save As**，然后在出现的对话框中提供名称。保存数据对象使其可用于本会话中的后续操作，以及包含多个数据对象的*X–Y*图中。
7. 完成后，点击**Cancel**关闭对话框。
