# 44.1.1 理解云图值的计算方式





从输出数据库中存储的结果生成云图所需的计算取决于云图是显示基于节点的数量（如位移或速度）还是基于单元的数量（如应力或应变）的值。本节仅适用于输出数据云的云图。

**基于节点的场输出变量云图值如何计算**

对于基于节点的场输出变量的云图，云图值直接从输出数据库中的量获得。然后直接使用这些值（无需进一步计算）在模型表面生成彩色线或彩色带。

**基于单元的场输出变量云图值如何计算**

对于基于单元的场输出变量的云图，Abaqus/CAE对输出数据库结果应用计算以形成云图值。计算根据以下标准而变化：
- 您选择要绘制的内容（场输出或间断），
- 您选择的平均选项，
- 您选择的[结果区域](pt05ch42s04hlb03.md)，
- 包含在绘图中的单元的兼容性，以及
- 您请求的云图类型（线型、带型或拼布型）。

从主菜单栏中选择****Result**![](../graphics/images/arrow.gif)**Field Output****以选择场输出变量。要了解更多关于此主题的信息，请参见["选择要显示的场输出，" 第42.5节](pt05ch42s03.md)。

从主菜单栏中选择****Result**![](../graphics/images/arrow.gif)**Options****以选择要绘制的内容、平均选项、结果区域，以及控制计算基于单元的云图值时的不兼容单元处理方式。要了解更多关于这些主题的信息，请参见["选择结果选项，" 第42.6节](pt05ch42s04.md)。

从主菜单栏中选择****Options**![](../graphics/images/arrow.gif)**Contour**![](../graphics/images/arrow.gif)**Basic****以选择您想要的云图类型。[ "选择线型、带型、拼布型或等值面型云图，" 第44.5.1节](pt05ch44s02hlb01.md) 包含选择云图类型的详细说明。

您请求的云图类型控制应用于从输出数据库读取的结果的外推。对于基于单元变量值的线型、带型和等值面型云图，Abaqus/CAE将结果外推到节点。默认情况下，在Abaqus/CAE执行任何必要的变换并计算任何请求的标量分量或不变 量之后，对结果进行条件平均；如果需要，您可以在结果平均后计算不变量。对于基于单元变量值的拼布型云图，Abaqus/CAE将结果外推到模型表面的单元面，然后计算加权和以产生每个面的单个值。由于拼布云图值为每个单元面单独计算，并且在单元边界之间不进行平均，因此拼布云图是按单元显示结果的有效方法。您只能为基于单元的场输出变量选择拼布型云图。
