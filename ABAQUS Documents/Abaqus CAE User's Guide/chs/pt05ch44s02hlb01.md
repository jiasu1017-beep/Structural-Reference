# 44.5.3 自定义带型和等值面型云图





Abaqus/CAE在模型表面显示带型云图作为彩色带，并通过将线型云图延伸到模型中创建表面来显示等值面云图。您可以显示或抑制每个云图带或等值面边缘处的线条；这些线条称为云图边缘。云图边缘在视觉上分隔带型云图中用于表示云图值的颜色；对于等值面，由于分隔通常更明显，云图边缘的显示更具装饰性。您可以进一步自定义云图边缘的颜色、样式和厚度。例如，在[图44-6](pt05ch44s02hlb03.md#usv-contbandedges)中，左侧的云图抑制了云图边缘，而右侧的云图显示了云图边缘。

**注意：**云图边缘不适用于[连续区间](pt05ch44s02hlb12.md)云图。

**图44-6** 带型云图，边缘抑制和显示。

![](../graphics/usv-cont-bandedges.png)

**要自定义带型和等值面型云图：**

1. 找到**Banded / Isosurface**选项。从主菜单栏中选择****Options**![](../graphics/images/arrow.gif)**Contour****，或单击工具箱中的![](../graphics/ico_plotContourOptions.png)。在出现的对话框中单击**Color & Style**选项卡；然后单击**Banded / Isosurface**选项卡。**Banded / Isosurface**选项出现。
2. 切换**Show contour edges**以显示或抑制云图边缘。当**Show contour edges**打开时，云图边缘选项变得可用。
3. 选择云图边缘的颜色：1. 单击颜色样本![](../graphics/usi-colorsample.png)。Abaqus/CAE显示**Select Color**对话框。2. 使用**Select Color**对话框中的方法之一选择新颜色。有关更多信息，请参见["自定义颜色，" 第3.2.9节](pt01ch03s02s09.md)。3. 单击**OK**关闭**Select Color**对话框。颜色样本更改为所选颜色。在您单击**Contour Plot Options**对话框中的**OK**或**Apply**之前，新颜色不会出现在视口中。
4. 选择云图边缘的样式：1. 单击**Style**按钮显示云图边缘样式选项。2. 单击您想要的边缘样式。指定边缘样式出现在**Style**按钮上。
5. 选择云图边缘的厚度：1. 单击**Thickness**按钮显示云图边缘厚度选项。2. 单击您想要的边缘厚度。指定边缘厚度出现在**Thickness**按钮上。
6. 单击**Apply**实施您的更改。当前视口中带型云图的云图边缘更改为反映您的规格。默认情况下，您的更改在会话期间保存，并将影响所有后续云图。如果您想在后续会话中保留更改，请将它们保存到文件。有关更多信息，请参见["保存自定义设置以供后续会话使用，" 第55.1.1节](pt05ch55s01s01.md)。
