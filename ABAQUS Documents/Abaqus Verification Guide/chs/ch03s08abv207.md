# 3.8.4 圆柱连续体应力/位移子模型





**产品：**Abaqus/Standard  

### 测试的单元

CCL9    CCL12    CCL18    CCL24    

### 测试的功能

子模型功能应用于圆柱单元。一般静态程序用于全局分析和子模型分析的各种组合。

### 问题描述

**模型：**

所有全局模型由外半径为2、内半径为1的180度圆柱段组成。子模型输入文件建模全局模型的右半部分。子模型驱动节点位于全局模型的对称平面沿线。

**材料：**

| 杨氏模量 | 6×10^6 |
| --- | --- |
| 泊松比 | 0 |

**载荷和边界条件：**

两端的所有节点在圆周方向上固定，内表面上的所有节点在所有自由度上受到约束。分布载荷或规定位移施加在圆柱单元的外表面上。

### 结果与讨论

子模型分析中所有驱动变量的幅值在全局分析文件输出中被正确识别，并应用于子模型分析中的驱动节点。
