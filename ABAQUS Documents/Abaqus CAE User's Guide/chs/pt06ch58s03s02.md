# 58.3.2 点云映射场的坐标系





映射场使用全局坐标系或局部坐标系的坐标来表示空间中的点。对于点云数据源，在解释和应用映射数据坐标时可能涉及不同的坐标系，这些坐标被应用于属性、载荷、预定义场或相互作用。

**源数据局部坐标系**

三维坐标和场值都在局部坐标系中定义。您可以在**创建映射场**对话框中指定要使用的坐标系。这是一个装配级（不是零件级）坐标系。

如果使用直角坐标系（默认），Abaqus/CAE 使用 *X*、*Y* 和 *Z* 坐标来解释源数据点。如果指定圆柱坐标系或球坐标系，Abaqus/CAE 会按照[表 58–4](pt06ch58s03s02.md#usi-fld-usingmap-csystable) 所示解释坐标。**创建映射场**对话框的数据表中显示的列取决于您选择的局部坐标系类型。对于圆柱坐标系和球坐标系，**Th** 和 **P** 的值以度为单位输入，在分析期间会转换为从 ![](../graphics/usi_eqn00792.gif) 到 ![](../graphics/usi_eqn00793.gif) 范围内的弧度。

**表 58–4** 映射场的坐标系类型和坐标名称。
| 坐标系类型 | 坐标 | 数据表中显示的列 |
| --- | --- | --- |
| 直角坐标系或全局坐标系（默认） | *X*、*Y* 和 *Z* 轴 | **X**、**Y**、**Z** |
| 圆柱坐标系 | *R*、![](../graphics/usi_eqn00065.gif)- 和 *Z* 轴 | **R**、**Th**、**Z** |
| 球坐标系 | *R*、![](../graphics/usi_eqn00065.gif)- 和 ![](../graphics/usi_eqn00226.gif)-轴 | **R**、**Th**、**P** |

**在零件空间中定义的源数据**

您可以通过在**创建映射场**对话框中切换**提供的源数据在零件空间中定义**来选择此选项。源数据及其局部坐标系都在目标模型区域零件级的坐标系中解释。
