# 55.5.2 Customizing element labels







Element labels are numeric labels (element numbers) that identify each element. For example, the elements are labeled in [Figure 55--7](pt05ch55s05hlb02.md#viw-labelelem).

**Figure 55–7** Model showing element labels.

![](../graphics/viw-labelelem.gif)

Element labels are plot state–independent; that is, you set them once for all undeformed, deformed, contour, symbol, and material orientation plots. You can also set the element labels that Abaqus/CAE will display on the undeformed shape when the deformed shape is superimposed on it. Toggle **Show element labels** to display or suppress element labels and to choose their color.

**To customize element labels:**

1. Locate the common or superimposed element label options. The common element label options always apply to the deformed shape; they apply to the undeformed shape when it is plotted individually in any plot state (undeformed, deformed, contour, symbol, or material orientation). When the undeformed shape is plotted with the deformed shape, the superimpose element label options apply. From the main menu bar, select ****Options**![](../graphics/images/arrow.gif)**Common**** or **Superimpose**; then click the **Labels** tab in the dialog box that appears.
2. Toggle **Show element labels** to display or suppress numeric element labels. When **Show element labels** is on, the element label color options become available.
3. Choose the color of the element labels: 1. Click the color sample ![](../graphics/usi-colorsample.png) . Abaqus/CAE displays the **Select Color** dialog box. 2. Use one of the methods in the **Select Color** dialog box to select a new color. For more information, see ["Customizing colors," Section 3.2.9](pt01ch03s02s09.md). 3. Click **OK** to close the **Select Color** dialog box. The color sample changes to the selected color
4. Click **Apply** to implement your changes. The labels change according to your specification. By default, your changes are saved for the duration of the session and will affect all subsequent plots in the current viewport and in any new viewports created from the current viewport. If you want to retain the changes you applied for subsequent sessions, save them to a file. For more information, see ["Saving customizations for use in subsequent sessions," Section 55.1.1](pt05ch55s01s01.md).
