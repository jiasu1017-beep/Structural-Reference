# 11.16.3 在零件模块中使用分区工具集





在零件模块中，您可以使用分区工具集将零件分区为其他区域。分区零件后，可以使用属性模块向结果区域分配不同的截面；例如，您可以使用分区来划定由不同材料组成的零件区域。

您创建的分区是与零件关联的特征，因此该零件在装配中的每个实例都将包含在零件模块中创建的所有分区。您可以在其他模块中使用这些区域处理装配；例如，您可以在载荷模块中向区域施加载荷。如果您不希望将分区与零件的每个实例关联，请在装配模块中创建独立实例并对独立实例进行分区。更多信息，请参见"对装配进行分区"，第13.8.3节。

如果您已创建装配并向其应用了属性（如载荷、边界条件和集合），则应小心，因为如果您随后决定返回零件模块并对原始零件之一进行分区，属性分配到的区域可能会意外更改。一般来说，在开始创建零件实例并将集合、载荷和网格应用到装配之前，应尝试在零件模块中完成零件的创建。如果您确实返回零件模块创建分区，至少应检查分配了属性的装配中的区域是否仍然有效。
