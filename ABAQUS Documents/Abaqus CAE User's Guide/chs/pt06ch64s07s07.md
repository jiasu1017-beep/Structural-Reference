# 64.7.9 细分单元







选择 ****Mesh**![](../graphics/images/arrow.gif)**Edit****（网格→编辑）从主菜单栏细分一个或多个选定的孤网格单元。要细分的单元必须相互连接。您可以选择边、面（实体单元）或**All Directions**（所有方向）来沿一个、两个或所有方向划分单元——壳单元两个方向，实体单元三个方向。您还必须指定要进行的分割数。Abaqus/CAE 将单元均匀地分割为每个方向上指定数量的分段。新单元与原始单元具有相同的单元类型。只有当单元域仅包含线性单元时，才能在网格中细分单元。

**细分线性网格的单元：**

1. 进入 Mesh（网格）模块。从上下文栏的 **Object**（对象）字段中选择 **Part**（部件），然后从列表中选择包含孤网格单元的部件。
2. 从主菜单栏中选择 ****Mesh**![](../graphics/images/arrow.gif)**Edit****（网格→编辑）。Abaqus/CAE 显示 **Edit Mesh**（编辑网格）对话框。**提示：**您也可以使用位于 Mesh 模块工具箱底部的 ![](../graphics/ico_meshEdit.png) 工具来显示 **Edit Mesh**（编辑网格）对话框。
3. 从 **Category**（类别）字段中选择 **Mesh**（网格）。
4. 从 **Method**（方法）列表中选择 **Subdivide layers**（细分层）。
5. 选择要细分的单元。您可以使用以下任何方法： - 从视口中选择单元。您可以单独、按角度、按特征边或按拓扑选择单元（有关更多信息，请参阅["在当前视口中选择对象，" 第 6.2 节](pt01ch06s02.md)）。完成选择后点击鼠标按钮 2。 - 选择一个或多个现有单元集。点击提示区域右侧的 **Sets**（集合）以显示包含可用集合列表的 **Region Selection**（区域选择）对话框。从列表中选择一个或多个集合，然后点击 **Continue**（继续）。所选单元必须形成单个连接集。**注意：**默认选择方法基于您最近使用的选择方法。要恢复为其他方法，请点击提示区域右侧的 **Select in Viewport**（在视口中选择）或 **Sets**（集合）。
6. 选择单元边以指示单个细分方向；选择实体单元的面以指示两个方向；在提示区域中选择 **All Directions**（所有方向）以指示所有方向——壳单元两个方向或实体单元三个方向。Abaqus/CAE 尝试细分单元。如果分割单元将在网格中造成不兼容性，Abaqus/CAE 显示一个对话框，指示生成的网格将是不协调的。点击 **Yes**（是）分割单元，或点击 **No**（否）清除选择并返回第 4 步。**提示：**如果在分割网格中的单元时出错，点击 **Edit Mesh**（编辑网格）对话框中的 **Undo**（撤销）可以撤销更改。Abaqus/CAE 恢复原始网格。

编辑孤网格后，您应始终验证基于节点、单元和曲面的特征（如截面分配、载荷和相互作用），以确保它们正确应用于修改后的网格。
