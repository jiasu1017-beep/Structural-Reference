# 64.6.4 定向堆叠方向





从主菜单栏选择 ****网格**![](../graphics/images/arrow.gif)**编辑**** 可以定向连续壳、相粘、圆柱形或垫片网格中单元的堆叠方向。您只能定向六面体、楔形和四边形单元，因为这些是唯一可以堆叠形成连续壳、相粘、圆柱形或垫片网格的单元。如果您已为区域分配了圆柱单元或二阶垫片单元，则必须首先将分配更改为常规单元，然后才能重新定向堆叠方向。然后，您可以通过重新分配圆柱单元或二阶垫片单元将区域转换回圆柱形或垫片网格。如果更改了区域中单元的堆叠方向方向，Abaqus/CAE 不会更改应用于该区域的预定义条件或命名曲面的方向。

**定向堆叠方向的步骤：**

1. 进入"网格"模块。
2. 从上下文栏的 **对象** 字段中选择 **零件** 并从列表中选择一个零件。此工具仅适用于处理孤岛网格单元。要更改本地单元的堆叠方向，请参见 ["应用网格堆叠方向，" 第17.18.8节](pt03ch17s18hlb01.md)。
3. 从主菜单栏选择 ****网格**![](../graphics/images/arrow.gif)**编辑****。Abaqus/CAE 显示 **编辑网格** 对话框。 **提示：** 您也可以使用位于"网格"模块工具箱底部的 ![](../graphics/ico_meshEdit.png) 工具显示 **编辑网格** 对话框。
4. 在对话框中执行以下操作： 1. 在 **类别** 字段中选择 **单元**。 2. 从 **方法** 列表中选择 **定向堆叠方向**。对于六面体和楔形单元，Abaqus/CAE 将网格堆叠的底面着为紫色，顶面着为棕色。同样，箭头指示四边形单元的方向。此外，Abaqus/CAE 以红色高亮显示具有不一致方向的任何单元面和边缘。有关更多信息，请参见 [第25章，"连续壳"](pt04ch25.md)；["使用几何和网格工具创建带相粘单元的模型，" 第21.3节](pt04ch21s03.md)；[第32章，"垫片"](pt04ch32.md)；以及 ["圆柱实体的扫掠网格划分，" 第17.9.4节](pt03ch17s09s04.md)。
5. 在视口中选择要定向的单元。您可以 **按住[Shift]键并点击** 各个单元将其添加到您的选择中，或 **按住[Ctrl]键并点击** 所选单元将其从选择中移除。 **提示：** 您可以使用 **选择** 工具栏更改选择行为。有关更多信息，请参见 [第6章，"在视口中选择对象"](pt01ch06.md)。
6. 完成选择要定向的单元后，点击鼠标按钮2。
7. 如果您的零件用六面体或楔形单元网格化，请从所选单元中选择一个面。Abaqus/CAE 使用所选面来表示参考方向的顶面。如果您的零件用四边形单元网格化，请从所选单元中选择一条边缘。Abaqus/CAE 会将单元定向到所选边缘。 **提示：** 如果在定向单元时出错，点击 **编辑网格** 对话框中的 **撤销** 可以撤消更改。Abaqus/CAE 会恢复这些单元的原始方向。
8. 根据需要重复上述步骤以定向其他单元。
9. 完成单元定向后，点击鼠标按钮2或提示区域中的取消按钮 ![](../graphics/afxI_cancel.png) 退出过程。
