# 4.3.1 创建新视口





您可以随时创建新视口；对视口数量或其 在画布上的位置没有限制。

**要创建新视口：**
- 从主菜单栏，选择****视口（Viewport）**![](../graphics/images/arrow.gif)**创建（Create）****。Abaqus/CAE会在绘图区域创建一个新视口。此视口变为当前视口。新视口的大小和位置取决于当前视口和绘图区域的大小。如果当前视口已最大化，Abaqus/CAE会自动最大化新视口。**提示：**您也可以通过点击**视口**工具栏中的![](../graphics/ico_viewport.png)来创建新视口。
