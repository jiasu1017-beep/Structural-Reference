# 2.6.2 浏览和搜索HTML指南





您可以通过选择****帮助**![](../graphics/images/arrow.gif)**搜索和浏览指南****来浏览和搜索整个HTML集合。出现的集合窗口包含文档集合中所有书名的列表。要查看特定指南，请单击感兴趣的书名；该指南将出现在新的浏览器窗口中。（有关详细信息，请参阅[使用Abaqus在线文档](../hhp/hhp-link.md#hhp)。）

**显示和搜索HTML指南：**

1. 从主菜单栏，选择****帮助**![](../graphics/images/arrow.gif)**搜索和浏览指南****。集合窗口会出现在您的Web浏览器中，其中包含文档集合中所有书名的列表，按类别分组。
2. 单击感兴趣的书名。包含您所选指南的书窗口会在新的浏览器窗口中打开。书窗口包含四个框架：导航框架、目录控制框架、目录框架和文本框架，如图2-6所示。

**图2-6** 书窗口。

![](../graphics/hhp-book-nls.png)

3. 使用以下任何技术浏览指南内容：
   **目录控制**使用目录控制框架中的按钮可更改目录框架中显示的详细程度或更改框架的大小。单击![](../graphics/hhp-toc-expand.png)可展开在线书籍目录中的多个级别。单击![](../graphics/hhp-toc-collapse.png)可折叠目录中所有展开的部分。分别单击![](../graphics/hhp-toc-narrow.gif)和![](../graphics/hhp-toc-wide.gif)可缩小或加宽目录框架。
   **浏览**使用文本框架中的和箭头可按顺序浏览文本。您也可以使用Web浏览器功能返回最近查看的页面。
   **搜索**使用位于导航框架中的搜索面板搜索特定的单词或短语。有关更多信息，请参阅[第4章，"搜索Abaqus HTML文档，"使用Abaqus在线文档](../hhp/hhp-link.md#hhp-chp-html-search)。
   **使用超链接**使用超链接可从书籍的一部分移动到另一部分，或从一本书移动到另一本书。
