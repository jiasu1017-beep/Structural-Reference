# 22.1.1 弹性行为：概述





Abaqus中的材料库包含几种弹性行为模型：

**线性弹性**：线性弹性（["线性弹性行为，" 第22.2.1节](pt05ch22s02abm02.md)）是Abaqus中最简单的弹性形式。线性弹性模型可以定义各向同性、正交各向异性或完全各向异性的材料行为，适用于小弹性应变。

**平面应力正交各向异性失效**：为线性弹性提供了失效理论（["平面应力正交各向异性失效度量，" 第22.2.3节](pt05ch22s02abm04.md)）。它们可用于获得后处理的输出请求。

**多孔弹性**：Abaqus/Standard中的多孔弹性模型（["多孔材料的弹性行为，" 第22.3.1节](pt05ch22s03abm05.md)）用于体积弹性应变部分随等效压力应力对数变化的多孔材料。这种非线性弹性形式适用于小弹性应变。

**次弹性**：Abaqus/Standard中的次弹性模型（["次弹性行为，" 第22.4.1节](pt05ch22s04abm06.md)）用于应力变化率由弹性应变变化率乘以弹性矩阵定义的材料，其中弹性矩阵是总弹性应变的函数。这种通用非线性弹性适用于小弹性应变。

**类橡胶超弹性**：对于有限应变的类橡胶材料，超弹性模型（["类橡胶材料的超弹性行为，" 第22.5.1节](pt05ch22s05abm07.md)）提供了通用的应变能势来描述几乎不可压缩的弹性体材料行为。这种非线性弹性模型适用于大弹性应变。

**泡沫超弹性**：超泡沫模型（["弹性泡沫中的超弹性行为，" 第22.5.2节](pt05ch22s05abm08.md)）为有限应变的可压缩弹性泡沫提供了通用能力。这种非线性弹性模型适用于大应变（特别是大体积变化）。Abaqus/Explicit中的低密度泡沫模型（["低密度泡沫，" 第22.9.1节](pt05ch22s09abm16.md)）是一种非线性黏弹性模型，适用于指定低密度弹性泡沫的应变率敏感行为，如碰撞和冲击应用中使用的那些。泡沫塑性模型（["可压碎泡沫塑性模型，" 第23.3.5节](pt05ch23s03abm34.md)）应用于经历永久变形的泡沫材料。

**各向异性超弹性**：各向异性超弹性模型（["各向异性超弹性行为，" 第22.5.3节](pt05ch22s05abm09.md)）为建模表现出高度各向异性和非线性弹性行为的材料（如生物医学软组织、纤维增强弹性体等）提供了通用能力。该模型适用于大弹性应变，并捕捉材料优先方向（或纤维方向）随变形的変化。

**织物材料**：Abaqus/Explicit中的织物模型（["织物材料行为，" 第23.4.1节](pt05ch23s04abm35.md)）用于机织物，捕捉沿纬纱和经纱方向刚度的方向性质。它还捕捉纱线方向相对旋转时的剪切响应。该模型考虑了包括大剪切旋转的有限应变。它通过使用测试数据或用户子程序[`VFABRIC`](../sub/sub-link.md#sub-xsl-vfabric)（参见["VFABRIC，" Abaqus用户子程序参考指南第1.2.4节](../sub/sub-link.md#sub-rtn-uexpfabric)）进行材料表征，捕捉织物的高度非线性弹性响应。基于测试数据的织物行为可以包括非线性弹性、永久变形、率相关响应和损伤累积。

**黏弹性**：黏弹性模型用于指定随时间变化的材料行为（["时域黏弹性，" 第22.7.1节](pt05ch22s07abm12.md)）。在Abaqus/Standard中，它还用于指定频率相关材料行为（["频域黏弹性，" 第22.7.2节](pt05ch22s07abm13.md)）。它必须与线性弹性、类橡胶超弹性或泡沫超弹性结合使用。

**并行流变框架**：并行流变框架（["并行流变框架，" 第22.8.2节](pt05ch22s08abm15.md)）旨在为承受大应变的材料（如弹性体和聚合物）建模非线性行为。在此框架内定义的模型由多个并行黏弹性网络组成，可选择地包括弹塑性网络，以允许使用Mullins效应建模永久变形和材料软化。弹性响应使用超弹性材料模型定义；塑性响应基于不可压缩等向硬化塑性理论；黏性响应使用从蠕变势导出的流动法则指定。

**滞后**：Abaqus/Standard中的滞后模型（["弹性体中的滞后，" 第22.8.1节](pt05ch22s08abm14.md)）用于指定弹性体的率相关行为。它与超弹性结合使用。

**Mullins效应**：Mullins效应模型（["Mullins效应，" 第22.6.1节](pt05ch22s06abm10.md)）用于指定由损伤引起的填充橡胶弹性体的应力软化，这是一种称为Mullins效应的现象。该模型还可用于包括弹性泡沫中的永久能量耗散和应力软化效应（["弹性泡沫中的能量耗散，" 第22.6.2节](pt05ch22s06abm11.md)）。它与类橡胶超弹性或泡沫超弹性结合使用。

**无压缩或无张力弹性**：Abaqus/Standard中的无压缩或无张力模型（["无压缩或无张力，" 第22.2.2节](pt05ch22s02abm03.md)）可在不应产生压缩或拉伸主应力时使用。这些选项只能与线性弹性结合使用。

### 热应变

可以为任何弹性或织物模型引入热膨胀（["热膨胀，" 第26.1.2节](pt05ch26s01abm52.md)）。

### 弹性应变幅度

除超弹性和织物材料模型外，应力始终假定为与弹性关系切线模量相比很小；即弹性应变必须很小（小于5%）。如果材料定义中包含金属塑性等非弹性响应，总应变可以任意大。

对于纯弹性的大应变有限应变计算，应使用织物模型（用于机织物）、超弹性模型（用于类橡胶行为）或泡沫超弹性模型（用于弹性泡沫）。超弹性和织物模型是唯一能在较大弹性应变下对实际材料行为给出真实预测的模型。线性或在Abaqus/Standard中的多孔弹性模型适用于其他大应变是非弹性情况。

在Abaqus/Standard中，如果应力达到弹性模量50%或更高的水平，线性弹性、多孔弹性和次弹性模型将表现出较差的收敛特性；在实际案例中这不是严重问题，因为这些材料模型对产生的大应变无效。
