# 20.2.8 aStudy=ParStudy()





### 20.2.8 ***aStudy*=ParStudy()**：创建参数化研究。

**产品：** Abaqus/Standard  Abaqus/Explicit  

此命令用于创建参数化研究。它必须先于任何引用参数化研究的其他脚本命令。

##### **参考**

- ["参数化研究脚本，" 第20.1.1节](pt04ch20s01aus108.md)

### **命令：**

***aStudy*=ParStudy** (**par**= , **name**= , **verbose**= , **directory**= )

### **必需数据：**

**par**

将**par**设置为为参数化研究选择的独立输入参数序列。此序列必须放在括号或方括号内，并且必须包含用匹配引号括起来并用逗号分隔的独立参数名称；例如，`('par1', 'par2', 'par3')`或`['par1', 'par2', 'par3']`。如果要列出的只有一个参数，其名称可以用匹配引号括起来；例如，`'par1'`。

### **可选数据：**

**name**

将**name**设置为参数化研究的名称；此名称必须用匹配引号括起来。如果未指定名称，其值默认为包含参数化研究命令的Python脚本文件的名称。

**verbose**

将**verbose**设置为符号常量OFF，以禁止打印注释和警告消息。默认值为**verbose**=ON。

**directory**

将**directory**设置为符号常量ON，以选择当前目录的子目录用于组织参数化研究的文件。将为分析的每个设计创建一个子目录。默认值为**directory**=OFF。
