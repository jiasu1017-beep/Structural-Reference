# 69.7.6 修复面法向







您可以修复壳和实体导入部件的面法向，您可以对这些修复在流形部件（每个边仅被一个或两个面共享的部件）和非流形部件上执行。

如果 Abaqus 报告导入部件的体积为负值，则意味着实体已被翻转。**Repair normals**（修复法向）工具将翻转法向并将实体翻正。

导入的壳部件可能包含法向指向相反方向的面。**Repair normals**（修复法向）工具将对齐壳部件上所有所选的法向。如果面法向已经对齐，此工具将翻转所有所选的法向，使它们保持对齐但指向相反方向。

**注意：**Abaqus/CAE 不会翻转使用 Property（属性）模块中的 **Element Normal**（元素法向）分配选项手动分配给壳面的法向。

当您对非流形壳部件执行修复时，您无法在单个操作中翻转部件中所有面的法向；您必须单独选择每个面以翻转其法向方向。使用 **Query**（查询）对话框中的 **Shell element normals**（壳元素法向）选项执行查询可以帮助您在修复壳部件之前评估面法向的方向。

查询工具集提供了一组几何诊断工具，用于计算实体部件的体积并显示壳和膜的法向。 有关更多信息，请参阅["在 Part 模块中使用查询工具集，" 第 11.16.4 节"](pt03ch11s16s04.md)。修复面法向的操作作为部件的特征存储；因此，您可以使用模型树删除或抑制操作。

**修复法向：**

1. 从主菜单栏中选择 ****Tools**![](../graphics/images/arrow.gif)**Geometry Edit****（工具→几何编辑）。Abaqus/CAE 显示 **Geometry Edit**（几何编辑）对话框。**提示：**您也可以使用位于 Part（部件）模块工具箱中编辑工具处的 ![](../graphics/ico_repairFaceNormals.png) 工具来修复面法向。 有关工具箱中编辑工具的图表，请参阅["使用几何编辑工具集，" 第 69.1 节"](pt06ch69s01.md)。
2. 从对话框中选择 **Face**（面）类别和 **Repair normals**（修复法向）方法。
3. 从提示区域选择以下选项之一： - 点击 **Entire Part**（整个部件）修复部件中每个方向错误的面。 - 点击 **Select Faces**（选择面）选择部件中的单个面。
4. 如果您在部件中选择单个面，请从视口中选择要修复的面法向，然后从提示区域点击 **Done**（完成）。您可以使用拖动选择、**[Shift]****+点击** 和 **[Ctrl]****+点击** 的组合来选择部件中的面，您还可以使用角度方法或面曲率方法来选择面。 有关更多信息，请参阅["在当前视口中选择对象，" 第 6.2 节"](pt01ch06s02.md)。Abaqus/CAE 修复面法向。
