# *DESIGN PARAMETER







### *DESIGN PARAMETERSpecify design parameters with respect to which sensitivities are calculated.

This option is used to specify design parameters for design sensitivity analysis. Sensitivities of responses specified under the [*DESIGN RESPONSE](ch04abk17.md) option will be calculated with respect to these design parameters. The design parameters must be chosen from an existing set of parameters defined on the [*PARAMETER](ch16abk01.md) option.

**Product: **Abaqus/Design  

**Type: **Model data  

**Level: **Model  

##### **References:**

- ["Design sensitivity analysis," Section 19.1.1 of the Abaqus Analysis User's Guide](../usb/usb-link.md#usb-anl-adsa)
- [*PARAMETER](ch16abk01.md)

**There are no parameters associated with this option.**
