# 20.4.5 Sketcher 光标和预选







Sketcher 中有两个活动光标：主光标和辅助光标。两个光标如图 20-4](pt03ch20s04s05.md#ske-twocursors)所示。

**图 20-4** Sketcher 光标。

![](../graphics/ske-twocursors-nls.gif)

主光标是您在与大多数计算机应用程序（包括 Abaqus/CAE）一起使用时使用的光标。主光标通常显示为箭头指针或指向手；您通过移动鼠标来定位此光标。辅助光标仅在 Sketcher 中处于活动状态；当 Sketcher 提示您选择一个点时，它会出现在主光标附近。辅助光标的位置允许您在确认选择之前准确查看选择了哪个点。如果您将主光标移近符合条件的点，辅助光标会直接跳转到该点，而主光标保持固定。这种行为称为预选。如果您点击鼠标按钮，Abaqus/CAE 会选择辅助光标下的点。

辅助光标的外观会随着您在草图中的移动而改变。[表 20-3](pt03ch20s04s05.md#ske-cursors)显示了当主光标接近所列 Sketcher 实体时辅助光标呈现的形状。

**表 20-3** 预选光标。
| 预选光标 | Sketcher 实体 |
| --- | --- |
| ![](../graphics/ske-square.png) | 顶点 |
| ● | 直线或曲线的中点 |
|  | 直线和曲线的交点 |
| ------ | 直线投影到其他直线和曲线上的点 |
| ○ | 沿现有直线或曲线的点 |
| + | 网格线之间或网格线与其他直线和曲线之间的交点 |

辅助光标与草图中的其他预选符号配合使用，以指示您可以选择特殊点。[表 20-4](pt03ch20s04s05.md#ske-othersym)显示了 Sketcher 中使用的其他预选符号及其含义。

**表 20-4** 预选符号。
| 预选符号 | 位置和含义 |
| --- | --- |
| ![](../graphics/ske-square.png) | 在新线段起点处，当线段在现有草图、参考或构造几何上开始时。新线段垂直于现有几何。 |
| || | 在现有边上开始创建新线时，在起点附近；符号与现有边对齐。新线段与现有边相切。 |
| **H** | 在创建新线时，在辅助光标附近。新线段是水平的。 |
| **V** | 在创建新线时，在辅助光标附近。新线段是垂直的。 |

预选光标和符号还指示如果您选择预选点 Abaqus/CAE 将应用于草图的约束。重合、垂直和相切等约束有助于控制草图几何并保留您的设计意图。

预选适用于草图中任何有效选择的对象。例如，当您在草图中移动光标时，预选会高亮显示以下内容以指示有效选择：
- 当您草绘直线时，顶点、交点、投影和切点会高亮显示，如[表 20-3](pt03ch20s04s05.md#ske-cursors)所示。
- 当您添加尺寸时，草图、构造和参考几何会高亮显示。
- 当您修改尺寸时，尺寸会高亮显示。

您可以按如下方式自定义光标的行为：- 默认情况下，辅助光标会捕捉到主光标附近的网格点。如果您关闭此捕捉，辅助光标会跟随主光标，并且可以在 Sketcher 工作表上的任何位置定位。
- 您可以关闭预选。

有关自定义辅助光标的更多信息，请参见["打开或关闭捕捉，"第 20.9.2 节](pt03ch20s09hlb02.md)和["打开或关闭预选，"第 20.9.3 节](pt03ch20s09hlb03.md)。辅助光标仅在 Sketcher 中可用，但您可以使用另一种形式的预选来帮助您在其他 Abaqus/CAE 模块中选择视口对象。有关选择视口对象的更多信息，请参见["选择和取消选择单个对象，"第 6.2.1 节](pt01ch06s02hlb01.md)。![](../graphics/images/black4rule.gif)有关相关主题的信息，请点击以下任一项目：- ["控制草图几何，"第 20.7 节](pt03ch20s07.md)
