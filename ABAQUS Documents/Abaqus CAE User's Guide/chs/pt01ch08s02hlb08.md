# 8.2.8 自定义保存到 TIFF、PNG 或 SVG 文件中的图像





当你将视口打印到 TIFF、PNG 或 SVG 格式文件时，你可以自定义结果图像。你可以保存与屏幕上图像大小相同的图像，或者你可以指定图像大小（以像素为单位）。

更多信息，请参阅["EPS、TIFF、PNG 和 SVG 图像大小，" 第 8.1.4 节](pt01ch08s01s04.md)和["打印图像格式，" 第 8.1.1 节](pt01ch08s01s01.md)。

**要自定义保存到 TIFF、PNG 或 SVG 文件中的图像：**

1. 从主菜单栏中，选择****File**![](../graphics/images/arrow.gif)**Print****。**注意：**你也可以单击**文件**工具栏中的![](../graphics/ico_filePrint.png)。**打印**对话框出现。
2. 在**文件名**文本字段中，输入文件名或点击![](../graphics/ico_fileOpen.png)从标准文件浏览器中选择文件名。
3. 从**格式**字段中，选择 **TIFF**、**PNG** 或 **SVG**。
4. 从**打印**对话框底部，点击 **TIFF 格式选项**、**PNG 格式选项**或 **SVG 格式选项**按钮。相应的对话框出现。
5. 从**图像大小**字段中，选择以下之一：
   - 选择**使用屏幕大小**保存与你要打印所选视口的整体宽度和高度相同的图像。Abaqus/CAE 在**使用屏幕大小**单选按钮右侧显示结果大小。
   - 选择**使用下方设置**以像素为单位指定结果图像的宽度或高度。允许的最大图像大小为 1280 1024 像素。
6. 点击**确定**保存你的自定义设置并关闭对话框。
7. 对于彩色 PNG 和 TIFF 图像，你可以使用默认的 8 位颜色深度（256 色）或关闭**减少到 256 色**以使用更多颜色，这会增加文件大小但使图像更接近显示。如果你不使用默认设置，图像可用颜色数量取决于系统类型和设置。对于 Windows 系统，Abaqus/CAE 使用 24 位颜色（167 万种颜色）。对于 Linux 系统，Abaqus/CAE 使用与显示相同的颜色设置。
8. 当你完成**打印**对话框时，单击**确定**生成所需的输出。Abaqus/CAE 生成输出并关闭**打印**对话框。你的**打印**对话框设置会在会话期间保存。
