# 5.6.1 Centering the view







Click mouse button 3 in the viewport to access the option to center the view. You can also use the auto-fit tool ![](../graphics/ico_viewFit.png) from the **View Manipulation** toolbar to quickly pan and magnify or reduce a view so that the view fills the viewport and is centered within it.

**To center the view:**

1. Position the cursor in the viewport at the location to be used to center the view, and click mouse button 3.
2. From the menu that appears, select **Center View**. Abaqus/CAE shifts the position that you selected to the center of the viewport.
