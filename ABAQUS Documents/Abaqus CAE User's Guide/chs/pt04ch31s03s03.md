# 31.3.3 查看XFEM裂缝





创建模型时，必须为签名距离函数PHILSM请求场输出。当您打开包含PHILSM输出变量的输出数据库文件时，Abaqus/CAE会沿XFEM裂缝（签名距离函数值为零的位置）创建视图切割。因此，当您显示变形或等值线图时，富集区域中的裂缝单元可见，如图31-15所示。

**图31-15** 查看XFEM裂缝。

![](../graphics/usi-eng-xfem-viewer.png)

您可以执行等值线图的时程动画并查看XFEM计算的损伤萌生。动画还允许您查看分析进行中损伤在富集区域中的扩展。开启半透明度可让您看到裂缝穿过内部单元的进程。更多信息，请参见["更改半透明度"，第77.3节](pt07ch77s03.md)。

要查看初始裂缝前缘，您可以创建以PSILSM作为当前主场输出变量的等值面切割形状。更多信息，请参见["创建或编辑视图切割"，第80.2.1节](pt07ch80s02hlb01.md)。

如果您想更详细地研究裂缝，可以创建签名距离函数（PHILSM）的等值线图，并确定哪些单元中的签名距离值为负或为正。裂缝表面位于PHILSM值从负数过渡到正数的单元中。此外，您可以使用**View Cut Manager**中的![](../graphics/visG_CutOn.png)选项仅查看签名距离函数值为零的表面，这对应于XFEM裂缝的表面。更多信息，请参见["理解视图切割"，第80.1节](pt07ch80s01.md)。
