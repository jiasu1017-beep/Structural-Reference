### 概述

组合曲面：
- 通过对现有曲面执行布尔运算（并集、交集或差集）创建；
- 可以由基于单元的曲面或基于节点的曲面形成；
- 不能由欧拉曲面形成；
- 可在Abaqus/Standard中以与其他基于单元或基于模式的曲面相同的方式使用；以及
- 不能与Abaqus/Explicit中的接触对一起使用（但可与Abaqus/Explicit中的通用接触一起使用）。

裁剪曲面：
- 通过裁剪现有曲面并仅保留曲面中包含在指定矩形框内的部分来创建；
- 可以由基于单元的曲面或基于节点的曲面形成；
- 不能由欧拉曲面形成；
- 可在Abaqus/Standard中以与其他基于单元或基于模式的曲面相同的方式使用；以及
- 不能与Abaqus/Explicit中的接触对一起使用（但可与Abaqus/Explicit中的通用接触一起使用）。

### 创建组合曲面

您必须为组合曲面分配一个名称；此名称可与其他引用曲面的功能一起使用。

在以部件实例装配体形式定义的模型中，所有曲面必须属于部件、部件实例或装配。曲面可以在部件级别创建并在装配级别组合。额外规则在["定义装配，"第2.10.1节"](pt01ch02s10aus28.md)中给出。

要组合的曲面必须是相同类型；即，基于单元的曲面可以与另一个基于单元的曲面组合，但不能与基于节点的曲面组合。组合曲面可用于创建另一个组合曲面。

#### 现有曲面的并集

任意数量的现有曲面可以组合创建新曲面。如果要组合的曲面是基于单元的曲面，则新曲面也将是基于单元的曲面，并且任何曲面之间的重叠将被合并。同样，如果要组合的曲面是基于节点的曲面，则新曲面将是基于节点的曲面，并且任何曲面之间的重叠将被合并。

| **输入文件用法：** | ``` [*SURFACE*](../key/key-link.md#usb-kws-msurface), NAME=*name*, COMBINE=UNION *list of surface names* ``` |
| --- | --- |

#### 现有曲面的交集或差集

两个现有曲面的交集或差集可用于创建新曲面。差集操作从第一个曲面减去第二个曲面。当对基于单元的曲面执行交集或差集操作时，它们仅作用于小平面。如果交集操作导致空曲面，则会发出警告消息。

| **输入文件用法：** | 使用以下选项基于两个现有曲面的交集创建新曲面： |
| --- | --- |
|  | ``` [*SURFACE*](../key/key-link.md#usb-kws-msurface), NAME=*name*, COMBINE=INTERSECTION *first surface name, second surface name* ``` 使用以下选项基于两个现有曲面的差集创建新曲面： ``` [*SURFACE*](../key/key-link.md#usb-kws-msurface), NAME=*name*, COMBINE=DIFFERENCE *first surface name, second surface name* ``` |

### 创建裁剪曲面

您可以创建一个新曲面，该曲面仅包含现有曲面中位于指定裁剪框内的面。对于基于节点的曲面，新曲面将仅包含裁剪框内的那些节点。如果面至少有一个节点在框内，则整个面被视为有效。您必须为新曲面分配名称，并指定要从中生成新曲面的现有曲面的名称。只能指定一个曲面。
