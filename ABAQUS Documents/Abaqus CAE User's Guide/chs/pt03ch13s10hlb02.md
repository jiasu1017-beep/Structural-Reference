# 13.10.2 使用模型树操作零件实例





您可以使用模型树访问操作零件实例的附加功能。在模型树中选择一个零件实例或多个实例后，您可以点击鼠标按钮3并从以下选项中选择：
- 要将零件实例从依赖更改为独立，请从出现的菜单中选择**Make Independent**。类似地，您可以从菜单中选择**Make Dependent**将零件实例从独立更改为依赖。
- 要链接零件实例，请从出现的菜单中选择**Link Instances**以显示**Link Instances**对话框，然后执行以下操作：
  1. 对于每个子零件实例，指定要链接到的模型和零件实例。默认情况下，对话框显示模型数据库中除当前模型之外的所有模型，这些模型具有与选定实例同名的零件实例。你可以通过点击表格的**Model**或**Instance**列并从显示的名称列表中进行选择来指定其他值。
  2. 如果您希望子零件实例被排除在Abaqus/CAE生成提交分析作业时生成的输入文件之外，请启用**Also exclude child instances from simulation**。
  3. 点击**Link**。图标出现在模型树中以指示零件实例的链接状态。默认情况下，链接的零件实例在视口中以灰色显示。类似地，您可以从菜单中选择**Unlink Instances**来取消链接先前链接的零件实例。更多信息，请参见["在模型之间链接零件实例，"第13.3.5节](pt03ch13s03s05.md)。
- 要从分析中排除零件实例，请从出现的菜单中选择**Exclude from Simulation**。图标出现在模型树中以指示零件实例的排除状态。默认情况下，从分析中排除的零件实例在视口中以深灰色显示。类似地，您可以从菜单中选择**Include in Simulation**来将零件实例包含在先前已排除的分析中。更多信息，请参见["从分析中排除零件实例，"第13.3.6节](pt03ch13s03s06.md)。
- 要忽略零件实例的无效状态，请选择**Ignore Invalidity**。更多信息，请参见["处理无效零件，"第10.2.3节](pt02ch10s02s03.md)。
