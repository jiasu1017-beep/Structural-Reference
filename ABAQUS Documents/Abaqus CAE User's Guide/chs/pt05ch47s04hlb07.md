# 47.5.8 自定义X–Y图轴的放置





### 47.5.8 自定义X–Y图轴的放置

点击**Axes**选项卡来自定义*X–Y*图的*X*轴和*Y*轴的放置。本节描述**Placement**选项，它们控制轴本身的位置；要自定义轴标签的位置，请参见["自定义X–Y图轴标签，"第47.5.9节](pt05ch47s04hlb09.md)

**要控制X–Y图中轴的放置：**

1. 找到**Placement**选项。选择****Options**![](../graphics/images/arrow.gif)**XY Options**![](../graphics/images/arrow.gif)**Axis****；或点击![](../graphics/ico_xyAxisOptions.png)，它位于Visualization模块工具箱中的*X–Y*绘图工具中。在出现的对话框中点击**Axes**选项卡。轴的**Placement**选项出现在**Axes**页面的顶部。
2. 从**X Axis**或**Y Axis**字段中，高亮显示一个或多个轴。
3. 从**Axes**页面顶部的**Placement**选项中，选择以下之一：**Min Edge**Abaqus/CAE仅在*X–Y*图的最小边缘放置一个轴。对于*X*轴，最小边缘是视口底部；对于*Y*轴，最小边缘是视口左侧。**Max Edge**Abaqus/CAE仅在*X–Y*图的最大边缘放置一个轴。对于*X*轴，最大边缘是视口顶部；对于*Y*轴，最大边缘是视口右侧。**Min/Max Edge**Abaqus/CAE将所选轴放置在*X–Y*图的最大边缘和最小边缘上。对于*X*轴，Abaqus/CAE将所选轴放置在视口的顶部和底部；对于*Y*轴，Abaqus/CAE将所选轴放置在视口的左侧和右侧。**Center**Abaqus/CAE将轴放置在*X–Y*图的中心。
4. 点击**Dismiss**关闭**Axis Options**对话框。
