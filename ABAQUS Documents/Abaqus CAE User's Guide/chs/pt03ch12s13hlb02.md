# 12.13.2 创建广义平面应变截面





广义平面应变截面用于定义二维平面区域的截面属性。您必须创建一个参考点来指示广义平面应变元素所需的参考节点。除了参考点处的材料和截面厚度外，您还可以指定关于全局1轴和2轴的楔角，如图[12-17](pt03ch12s13hlb02.md#prp-wedgeangle)所示。请参见[Abaqus Analysis User's Guide的第27.1.2节"选择元素的维度"中的"广义平面应变元素"](../usb/usb-link.md#usb-elm-edimension-genplanestrain)。

**图12-17** 广义平面应变截面定义。

![](../graphics/prp-wedgeangle-nls.png)

**指定广义平面应变截面属性：**

1. 从主菜单栏，选择****Section**![](../graphics/images/arrow.gif)**Create****。出现**Create Section**对话框。
   **提示：**您也可以在**Section Manager**中点击**Create**，或在Property模块工具箱中选择创建截面工具![](../graphics/ico_section.png)。
2. 输入截面名称。有关命名对象的更多信息，请参见["使用基本对话框组件，"第3.2.1节"](pt01ch03s02s01.md)。
3. 选择**Solid**作为截面**Category**，选择**Generalized plane strain**作为截面**Type**，然后点击**Continue**。出现广义平面应变截面编辑器。
4. 为截面选择一种材料。如有需要，点击![](../graphics/ico_material.png)创建材料；有关更多信息，请参见["创建或编辑材料，"第12.7.1节"](pt03ch12s07hlb01.md)。
5. 为**Thickness at the reference point**输入值。
6. 为**Wedge angle about global 1-axis**输入值，即参考点处关于全局1轴的旋转（弧度）。
7. 为**Wedge angle about global 2-axis**输入值，即参考点处关于全局2轴的旋转（弧度）。
8. 点击**OK**保存更改并关闭广义平面应变截面编辑器。
