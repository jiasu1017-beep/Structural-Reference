# 14.11.2 配置线性扰动分析过程







您可以配置线性扰动过程来分析线性问题。线性扰动过程仅在Abaqus/Standard中可用。线性扰动方法允许在在线性响应依赖于预加载或模型的非线性响应历史的情况下通用地应用线性分析技术。更多信息，请参见Abaqus Analysis User's Guide的第6.1.3节"常规和线性扰动过程"。

本节提供有关使用步骤编辑器配置不同类型线性扰动过程的说明。涵盖以下主题：
- "配置静态、线性扰动过程"
- "配置频率过程"
- "配置屈曲过程"
- "配置复频率过程"
- "配置模态动力学过程"
- "配置随机响应过程"
- "配置响应谱过程"
- "配置直接解稳态动态过程"
- "配置基于模态的稳态动态分析"
- "配置基于子空间的稳态动态分析"
- "配置子结构生成过程"
- "配置时谐电磁分析"
