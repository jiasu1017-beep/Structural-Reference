# 20.10.3 草绘矩形







使用 Sketcher 工具箱中的矩形工具绘制矩形。要绘制矩形，请点击下图中编号所示的任意两个对角。

![](../graphics/ske-rectbtn.png)

在草绘时，您应该注意定位点，因为这会影响网格的质量。草图中的点会成为您正在创建或修改的零件的顶点。反过来，当您在网格模块中对模型进行网格划分时，Abaqus/CAE 会将这些顶点转换为完全约束的种子，并将节点放置在其位置。有关如何随后移动顶点的信息，请参见["拖动 Sketcher 对象，"第 20.17.1 节](pt03ch20s14hlb01.md)。

**草绘矩形：**

1. 从 Sketcher 工具箱中的直线工具中，选择矩形工具![](../graphics/ico_sketchRectangle.png)。有关 Sketcher 工具箱中工具的图表，请参见["Sketcher 工具，"第 20.4.1 节](pt03ch20s04s01.md)。Abaqus/CAE 在提示区域显示提示以指导您完成该过程。
2. 点击矩形任意两个对角的所需位置。**提示：**如有必要，您可以使用提示区域中的文本框输入矩形角的精确坐标。有关精确定义矩形的更多信息，请参见["指定精确几何，"第 20.6 节](pt03ch20s06.md)。当您移动光标或输入坐标时，矩形会出现，并与当前 Sketcher 网格对齐。
3. 要创建更多矩形，重复上一步。
4. 完成创建矩形后，执行以下操作之一：- 在 Abaqus/CAE 窗口中的任意位置点击鼠标按钮 2。- 在 Sketcher 工具箱中选择任何其他工具。- 点击提示区域中的取消按钮![](../graphics/afxI_cancel.png)。
