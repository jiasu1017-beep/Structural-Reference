### 概述

ALE自适应网格划分包括两个基本任务：
- 创建新网格
- 使用称为平流的过程将解变量从旧网格重新映射到新网格

自适应网格划分技术的成功取决于每个任务所使用方法的选择。创建新网格和重新映射解变量的默认方法经过仔细选择，适用于各种问题。但是，您可能希望覆盖默认选择以平衡自适应网格划分的稳健性和效率，或将自适应网格划分扩展到更困难或不常见的应用。

### 网格划分

新网格：
- 为每个自适应域以指定频率创建
- 通过在自适应网格域上迭代扫描并移动节点来平滑网格来找到
- 可以保持原始网格的初始梯度

### 重新映射

用于将解变量平流到新网格的方法：
- 是一致的、单调的，并且（默认）二阶精度
- 保持质量、动量和能量

### 控制ALE自适应网格划分的频率

在大多数情况下，自适应网格划分的频率是影响网格质量和自适应网格划分计算效率的参数。没有欧拉边界的典型自适应网格应用通常每5-100个增量执行一次自适应网格划分。相比之下，使用欧拉边界的稳态过程模拟通常应以更高的频率执行自适应网格划分。因此，如果为自适应网格域定义了空间自适应网格约束或欧拉边界区域，默认频率为1；否则，默认频率为10。

### 控制ALE自适应网格划分的强度

在每个自适应网格划分增量期间，新网格是通过执行一次或多次网格扫描然后将解变量平流到新网格来创建的。

#### 网格扫描

在自适应网格划分增量期间，通过在自适应网格域上迭代扫描来创建新的、更平滑的网格。在每次网格扫描期间，域中的节点被重新定位——基于相邻节点和单元的当前位置——以减少单元变形。在典型扫描中，节点移动了围绕该节点的任何单元特征长度的一部分。增加扫描次数会增加每个自适应网格划分增量的网格划分强度。默认的网格扫描次数为一。

#### 平流扫描

将解变量从旧网格映射到新网格的过程称为平流扫描。至少在每个自适应网格划分增量中执行一次平流扫描。理想情况下，平流扫描只在所有网格扫描完成后执行一次。然而，只有当旧网格和新网格之间的差异较小时，平流扫描的数值稳定性才能保持。因此，如果在网格扫描后域中任何节点的累计移动大于任何相邻单元特征长度的50%，则执行平流扫描以将解变量从旧网格重新映射到中间网格。网格扫描将继续，直到达到指定的次数，或者直到任何节点的移动再次超过50%阈值。此时，再次执行平流扫描以将变量从最后一个中间网格映射到新的中间网格。循环将继续直到达到网格扫描次数。

每个自适应网格划分增量所需的每个自适应网格域的平流扫描次数由Abaqus/Explicit自动确定；您不能覆盖此自动计算。平流扫描次数默认打印到消息文件（`.msg`）（见["Abaqus/Explicit中ALE自适应网格划分的输出和诊断，" Section 12.2.5](pt04ch12s02aus81.md)）。

### ALE自适应网格划分的计算成本

自适应网格划分的成本取决于重划分的频率、执行的网络和平流扫描次数，以及自适应网格域的大小。与纯拉格朗日分析相比，仅在自适应网格划分增量内才会产生额外的计算成本。

一般来说，一次平流扫描的成本是单次网络扫描成本的几倍。当自适应网格划分执行得太不频繁和/或指定了高数量的网络扫描时，会触发多次平流扫描。更频繁地执行自适应网格划分并在每个自适应网格划分增量中执行1-5次网络扫描通常只会产生一次平流扫描，从而最小化计算成本。

自适应网格划分产生的相对平滑的网格和改进的单元纵横比可能会增加稳定时间增量，与类似的纯拉格朗日分析相比。在某些情况下，这种增加可以完全抵消自适应网格划分的成本。

尽管计算成本可能因应用类型而有很大差异，但在每个增量中对整个问题域执行自适应网格划分通常会将分析成本增加到类似拉格朗日分析成本的3-5倍。定义仅覆盖整个问题域一部分的自适应网格域将按比例降低成。本。将频率改为每10-25个增量一次将导致CPU时间仅比纯拉格朗日分析适度增加。

### 控制ALE自适应网格划分频率和强度的指南

尽管默认值适用于许多问题，但困难的分析可能需要更频繁的自适应网格划分频率或更高强度的网格划分。

#### 瞬态分析指南

对于没有空间自适应网格约束或欧拉边界区域的问题，自适应网格划分的默认频率为10，默认网络扫描次数为1。默认值通常适用于低至中等速率的动力学问题和经历中等变形的准静态过程模拟。如果频率或网络扫描次数太低，过度的单元变形可能导致分析在实际网格适配之前终止；或者，如果可以获得解，它可能不如具有更高质量网格的解准确。然而，实际上，在任何频率下执行自适应网格划分都会减少单元变形（并因此提高解质量），与纯拉格朗日分析相比。

对于经历大量变形的高速冲击问题，可能需要增加自适应网格划分的频率或网络扫描次数。通常，在增加频率之前稍微增加网络扫描次数是较便宜的，只要平流扫描次数保持较小。

对于仅持续几百个增量的爆炸问题，通常需要在每个增量执行自适应网格划分。对于在每个增量涉及大量流动的准静态过程模拟，也可能需要增加自适应网格划分的频率。

对于每个增量变形较小的问题，可以通过仅每25-100个增量执行一次自适应网格划分来保持高质量网格。对于这些问题，自适应网格划分的额外成本可以忽略不计。

#### 稳态分析指南

当自适应网格域包含欧拉边界区域或具有空间自适应网格约束时，自适应网格划分的默认频率为1。此默认频率是保守的，主要选择是因为空间网格约束仅在自适应网格划分增量期间应用。因此，在自适应网格划分增量之间，网格可能从其规定位置漂移，这可能影响解。然而，在下一个自适应网格划分增量中将始终消除来自自适应网格约束的漂移：它不会累积。

对于变形速度或材料从单元到单元的流动速度远小于材料波速的问题，频率通常可以增加到5或更高。这类问题包括大多数稳态过程模拟，其中网格从规定位置的漂移在几个增量中可以忽略不计。通过减少执行自适应网格划分的频率，稳态模拟变得与其对应的瞬态模拟具有竞争力。对于变形或材料流动速度较高的欧拉域（如动态冲击问题），应使用默认频率1。

### 网格平滑方法

Abaqus/Explicit中新网格的确定基于四个方面。您可以通过定义自适应网格控制来控制每个方面。已选择默认值使整体算法适用于大多数问题。

首先，Abaqus/Explicit中新网格的计算基于三种基本平滑方法的某种组合：体积平滑、拉普拉斯平滑和等势平滑。平滑方法应用于自适应网格域中的每个节点，以基于围绕节点的周围节点或单元的位置确定节点的新位置。尽管所有平滑方法都倾向于平滑网格和减少单元变形，但根据所使用的方法，生成的网格将有所不同。

其次，如果需要，可以在牺牲单元变形的代价下保持初始单元梯度。第三，在应用基本平滑方法之前对节点进行最佳定位可以提高网格质量并最小化所需的自适应网格划分频率。

最后，使用解相关的网格划分来将网格细化集中在演化边界曲率的区域。这抵消了基本平滑方法在解精度重要的凹边界附近减少网格细化的趋势。

#### 体积平滑

体积平滑通过计算围绕节点的单元中心的体积加权平均值来重新定位节点。在[图12.2.3-1](pt04ch12s02aus79.md#aaleremesh-smooth-method)中，节点M的新位置由四个周围单元的中心位置C的体积加权平均值确定。体积加权将倾向于将节点M从单元中心C1推开，朝向单元中心C3，从而减少单元变形。

**图12.2.3-1** 网格扫描期间节点的重新定位。

![](../graphics/aaleremesh-smooth-method.png)

体积平滑非常稳健，是Abaqus/Explicit中的默认方法。它适用于结构化和高度非结构化的域。（结构化域是包含没有退化单元的域，其中每个节点在二维中由四个单元包围，在三维中由八个单元包围。）

#### 拉普拉斯平滑

拉普拉斯平滑通过计算与所讨论节点由单元边连接的每个相邻节点的位置平均值来重新定位节点。在[图12.2.3-1](pt04ch12s02aus79.md#aaleremesh-smooth-method)中，节点M的新位置由连接到M的四个节点L的位置平均值确定。节点L2和L3的位置将把节点M向上和向右拉以减少单元变形。

拉普拉斯平滑是最便宜的平滑算法，常用于网格预处理器。对于低至中等变形的网格域，拉普拉斯平滑的结果与体积平滑的结果相似。对于具有复杂曲率边界的域，体积平滑通常会产生更平衡的网格。

#### 等势平滑

等势平滑是一种更高阶的方法，通过计算节点在二维中八个最近邻居节点（或在三维中十八个最近邻居节点）位置的高阶加权平均值来重新定位节点。在[图12.2.3-1](pt04ch12s02aus79.md#aaleremesh-smooth-method)中，节点M的新位置基于所有围绕节点L和E的位置。

等势平滑的加权平均相当复杂，基于拉普拉斯方程的求解。等势平滑倾向于最小化跨多个单元的网格线的局部曲率。虽然这种趋势对于平缓弯曲的域可能是可取的，但它可能抑制等势平滑在高度变形和局部弯曲域中减少单元变形的能力。

等势平滑只能对被局部结构化网格包围的节点执行。当选择等势平滑时，被非结构化网格包围的节点将使用等效的体积平滑进行移动。

#### 组合平滑方法

Abaqus/Explicit中的默认平滑方法是体积平滑。选择替代平滑方法或组合平滑方法时，请为每种方法指定加权因子。当使用多种平滑方法时，通过计算每种所选方法预测的位置的加权平均值来重新定位节点。所有权重必须为正，它们的和通常应为1.0。如果所选权重的和小于1.0，则网格平滑算法在每个自适应网格划分增量中不那么积极。如果所选权重的和大于1.0，则对其值进行归一化，使其和为1.0。

|  | 例如，可以使用以下选项定义体积平滑和等势平滑的等量混合，不使用拉普拉斯平滑： ``` [*ADAPTIVE MESH CONTROLS](../key/key-link.md#usb-kws-hadaptivemeshcontrols), NAME=*name* 0.5, 0.0, 0.5 ``` |

### 基于凹边界曲率的自适应网格划分

仅基于最小化单元变形的网格平滑算法倾向于在凹曲率区域减少网格细化，特别是在曲率演化时。在高度弯曲边界附近保持足够的网格细化通常对同时建模边界形状和体积都很重要。为防止这些区域网格细化的自然减少，Abaqus/Explicit使用解相关的网格划分来自动将网格梯度聚焦到这些区域。

虽然解相关的网格划分可能会将更多单元"拉"到高曲率区域，但其主要目的是在这些区域保持名义细化。因此，当期望高度弯曲的边界时，应始终使用细网格，并且解相关的网格划分通常不应作为更多单元的直接替代。

### 在步开始时平滑变形网格

当自适应网格域包含均匀密度的结构化网格时，网格仅在域变形时独立于材料移动。如果网格最初不均匀，则Abaqus/Explicit中的网格算法将在没有变形或材料传输的情况下平滑网格。

当初始网格包含高度变形的单元时，在步开始前平滑网格通常是有用的，以便在整个步中使用尽可能好的网格。当使用均匀平滑目标时，默认在步开始时执行五次网格扫描。对于分级平滑目标，默认在步开始时执行两次网格扫描，不考虑梯度。在所有后续网格扫描中用于梯度的纵横比基于此局部平滑网格。

当初始网格扫描时，初始条件被平流到新网格。

### 边界区域上的网格划分

拉格朗日和滑移边界区域上的自适应网格划分受以下约束：网格和材料必须在垂直于边界的方向上一起移动。边界区域内部的节点允许在边界区域内自由地在材料上滑动，这最大限度地提高了可以执行的网格平滑量。在每次网格扫描中，通过在约束节点位于边界区域上的同时应用基本平滑方法来定位节点。在三维中，拉格朗日边界区域上的一些节点将在拉格朗日边上。在每次网格扫描中，通过在约束节点沿着离散拉格朗日边的同时应用基本平滑方法来定位这些节点。

对于沿着边界从单元到单元的材料流动相对于变形很重要的问题，如果对称地应用这些约束（相对于材料流的上游和下游方向），边界网格可能会产生振荡。Abaqus/Explicit使用Petrov-Galerkin加权的自由边界约束来抑制任何振荡。该算法是保容的，上 wind的程度是自动选择的。

### 将解变量平流到新网格

Abaqus/Explicit中自适应网格划分的框架是任意拉格朗日-欧拉方法，该方法将平流项引入动量平衡和质量守恒方程中，以考虑独立的网格和材料运动。有两种基本方法可以求解这些修改后的方程：直接求解非对称方程组，或使用算子分裂将拉格朗日（材料）运动与额外的网格运动解耦。由于其计算效率，在Abaqus/Explicit中使用算子分裂方法。此外，这种技术在显式设置中是适当的，因为小时间增量限制了单个增量内的运动量。

在自适应网格划分增量中，首先以与纯拉格朗日分析一致的方式处理单元公式、边界条件、外部载荷、接触条件等。一旦更新了拉格朗日运动并执行了网格扫描找到了新网格，就通过执行平流扫描重新映射解变量。平流扫描解释了动量平衡和连续性方程中的平流项。

### 单元变量的平流方法

在每次平流扫描中，必须将单元和材料状态变量从旧网格传输到新网格。要平流的变量数量取决于材料模型和单元公式；但是，应力、历史变量、密度和内能始终是解变量。有两种可用于单元变量平流的方法：基于Van Leer工作的默认二阶方法（Van Leer，1977）和基于donor cell差分的一阶方法。

两种平流方法都包含上 wind的概念。它们还在从旧网格映射到新网格时在积分意义上保持单元变量（即，域上任何解变量的积分在自适应网格划分时保持不变）。当没有欧拉边界区域的自适应网格域时，使用保守算法平流单元密度和内能自动确保质量和能量守恒。

两种平流方法也都是单调的和一致的。如果单元量在旧网格的一部分上具有单调递增的空间分布，则方法单调意味着它在新区网格上保持如此。当将解变量平流到与旧网格相同的新网格时，一致性意味着所有单元量保持不变。

#### 二阶平流

二阶平流是所有自适应网格域的默认设置。它推荐用于从准静态到瞬态动态冲击的所有问题。单元变量![](../graphics/usb_eqn00092.gif)通过首先在每个旧单元中确定变量的线性分布从旧网格重新映射到新网格，如图12.2.3-2](pt04ch12s02aus79.md#aaleremesh-2ndord-advection)对于简单一维网格所示。

**图12.2.3-2** 二阶平流。

![](../graphics/aaleremesh-2ndord-advection-nls.png)

中间单元中![](../graphics/usb_eqn00092.gif)的线性分布取决于相邻单元中![](../graphics/usb_eqn00092.gif)的值。为了构建线性分布：

1. 从中间单元及其相邻单元的积分点处的常数值构建二次插值。
2. 通过对二次函数求导找到中间单元积分点处的斜率，找到试验线性分布![](../graphics/usb_eqn02045.gif)。
3. 通过将其斜率减小直到其最小值和最大值在相邻单元中的原始常数值范围内来限制试验线性分布。这个过程称为通量限制，对于确保平流单调是必不可少的。

一旦为旧网格中的所有单元确定了通量限制的线性分布，就对每个新单元进行积分。这些分布在旧网格上每个新单元上积分。新变量的值通过将每个积分的值除以新单元体积来找到。（有关此计算的一阶示例，请参见[图12.2.3-3](pt04ch12s02aus79.md#aaleremesh-1stord-advection)。）

**图12.2.3-3** 一阶平流。

### 动量平流

通过首先平流动量，然后使用新网格上的质量分布计算速度场，来计算新网格上的节点速度。直接平流动量确保在重新映射期间正确保持自适应网格域中的动量。有两种可用于平流动量的方法：默认的单元中心投影法和半索引偏移法（Benson，1992）。两种方法都适用于所有自适应网格应用。

#### 单元中心投影法

单元中心投影法是平流动量的默认方法，需要最少的数值操作。首先基于单元节点的质量和速度计算旧网格的单元动量。然后使用与用于平流单元变量相同的算法将单元动量从旧网格平流到新网格。最后，使用投影将在新网格上的单元动量组装到节点上。单元中心投影方法仅需要二维中2-3个额外变量或三维中3个额外变量的平流。

| **输入文件用法：** | 使用以下选项请求计算效率最高的动量平流方法： |
| --- | --- |
|  | ``` [*ADAPTIVE MESH CONTROLS](../key/key-link.md#usb-kws-hadaptivemeshcontrols), NAME=*name*, MOMENTUM ADVECTION=ELEMENT CENTER PROJECTION ``` |

#### 半索引偏移法

半索引偏移法在计算上比单元中心投影法更密集，但对于某些问题可能会减少波分散。此方法首先将每个节点动量变量从围绕单元的节点移动到单元中心。然后将偏移的动量变量使用与用于平流单元变量相同的算法从旧网格平流到新网格，从而在新单元中心提供动量变量。最后，将新网格单元中心的动量变量偏移回节点。半索引偏移法需要在二维中平流8个额外变量或在三维中平流24个额外变量，这可能显著增加每次平流扫描的成本。
