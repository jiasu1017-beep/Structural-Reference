# 15.14.7 定义入射波相互作用属性







您可以定义入射波的速度和波载荷的其他特性。对于球形入射波载荷，您可以选择指定由于UNDEX气泡或入射波场空间衰减引起的载荷效应。入射波相互作用属性只能由入射波相互作用引用。指定UNDEX数据的入射波相互作用属性只能由具有UNDEX定义的入射波相互作用引用。更多信息，请参见第15.13.18节"定义入射波"。

**定义入射波相互作用属性：**

1. 从主菜单栏中，选择****相互作用**![](../graphics/images/arrow.gif)**属性**![](../graphics/images/arrow.gif)**创建****。
2. 在出现的**创建相互作用属性**对话框中，执行以下操作：- 命名该相互作用属性。有关命名对象的信息，请参见第3.2.1节"使用基本对话框组件"。- 选择**入射波**类型的相互作用属性。
3. 单击**继续**关闭**创建相互作用属性**对话框。
4. 在**流体中的声速**字段中，输入流体中的声速，![](../graphics/usi_eqn00610.gif]。
5. 在**流体密度**字段中，输入流体质量密度，![](../graphics/usi_eqn00611.gif]。
6. 从**定义**字段中，选择要定义的波类型以定义入射波属性。- 选择**平面**以指定平面入射波。- 选择**球形**以指定球形入射波。- 选择**扩散**以指定从多个角度入射的平面波场。- 选择**空气爆炸**以使用CONWEP模型指定结构上的空气爆炸载荷。- 选择**表面爆炸**以使用CONWEP模型指定结构上的表面爆炸载荷。
7. 如果您要指定球形入射波，请选择传播模型。更多信息，请参见Abaqus Analysis User's Guide的第34.4.6节"声学和冲击载荷"中的"描述入射波载荷"。- 选择**声学**以指定振幅与距源距离成反比的入射波。- 选择**UNDEX电荷**以指定UNDEX气泡数据。- 选择**广义衰减**以指定入射波场的空间衰减。
8. 如果您选择了**UNDEX电荷**传播模型，请单击**物理**、**材料**和**气泡模型**选项卡，然后指定以下值：****物理数据****- 气体比热比，![](../graphics/usi_eqn00102.gif]。- 重力加速度，*g*。- 大气压力，![](../graphics/usi_eqn00612.gif]。- 流动阻力系数，![](../graphics/usi_eqn00613.gif]。- 流动阻力指数，![](../graphics/usi_eqn00614.gif]。- 切换打开**忽略流体和气体中的波效应**以忽略波效应。****材料数据****- 电荷材料常数，*K*。- 电荷材料常数，*k*。- 电荷材料常数，*A*。- 电荷材料常数，*B*。- 绝热电荷常数，![](../graphics/usi_eqn00175.gif]。- 电荷材料密度，![](../graphics/usi_eqn00615.gif]。- 电荷材料质量，![](../graphics/usi_eqn00616.gif]。****气泡模型步骤数据****- 时间持续，![](../graphics/usi_eqn00617.gif]。- 气泡模拟的最大时间步数，![](../graphics/usi_eqn00618.gif]。当步数达到![](../graphics/usi_eqn00618.gif]或达到时间持续![](../graphics/usi_eqn00617.gif]时，气泡振幅模拟停止。- 相对步长控制参数，![](../graphics/usi_eqn00619.gif]。- 绝对步长控制参数，![](../graphics/usi_eqn00620.gif]。- 步长控制指数，![](../graphics/usi_eqn00621.gif]。步长，![](../graphics/usi_eqn00622.gif]，根据误差估计减少或增加：![](../graphics/usi_eqn00623.gif]。
9. 如果您选择了**广义衰减**传播模型，请输入无量纲常数**A**、**B**和**C**的值，其中![](../graphics/usi_eqn00624.gif]、![](../graphics/usi_eqn00625.gif]和![](../graphics/usi_eqn00626.gif]。
10. 如果您要定义扩散入射波，请在**种子数**字段中指定扩散源计算的种子数。
11. 如果您选择了**空气爆炸**或**表面爆炸**定义，请在对话框的**CONWEP电荷**部分指定以下设置：- 在**TNT当量质量**字段中，以任何首选质量单位指定TNT当量质量。- 在**质量到千克的转换**字段中，指定将首选质量单位转换为千克的乘法因子。- 在**长度到米的转换**字段中，指定将分析长度单位转换为米的乘法因子。- 在**时间到秒的转换**字段中，指定将分析时间单位转换为秒的乘法因子。- 在**压力到帕斯卡的转换**字段中，指定将分析压力单位转换为帕斯卡的乘法因子。
12. 单击**确定**创建入射波相互作用属性并退出编辑器。
