# 11.3 示例：管道系统的振动







在此示例中，您将研究 5 m 长管道段的振动频率。管道由钢制成，外径为 18 cm，壁厚为 2 cm（参见[图 11--8](ch11s03.md#gss-pipingsystem)）。

**图 11–8** 被分析管道系统的一部分。

![](../graphics/gss-pipingsystem-nls.png)

它一端牢固夹紧，另一端只能轴向移动。此 5 m 管道段可能承受高达 50 Hz 的谐载荷。卸载结构的最低振动模式为 40.1 Hz，但此值未考虑施加到管道结构的载荷如何影响其响应。为确保该段不发生共振，您被要求确定使其最低振动模式高于 50 Hz 所需的服役载荷幅度。您被告知管道段在服役时将承受轴向拉伸。首先考虑 4 MN 的载荷幅度。

管道的最低振动模式在垂直于管道轴线的任何方向上都是正弦波变形，这是由于结构横截面对称性造成的。您将使用三维梁单元对管道段进行建模。

分析需要固有频率提取。因此，您将使用 Abaqus/Standard 作为您的分析产品。

### 11.3.1 坐标系

使用默认的全局坐标系。将原点放在管道段的左端，使管道轴线与全局 1 轴重合，如图 [Figure 11--8](ch11s03.md#gss-pipingsystem) 所示。

### 11.3.2 网格设计

使用均匀分布的网格对管道段进行建模，使用 30 个二阶管道单元（PIPE32）。[图 11--9](ch11s03.md#gss-node-elem1) 中显示了模型的节点和单元编号。

**图 11–9** 节点和单元编号（两者都从左到右增加 1）。

![](../graphics/gss-node-elem1-nls.png)

### 11.3.3 预处理——创建模型

您可以使用预处理器创建此示例的网格，或者如果您愿意，可以使用 ["管道系统振动，" 第 A.12 节](ap01s12.md) 中显示的 Abaqus 网格生成选项。如果您希望使用 Abaqus/CAE 创建整个模型，请参阅 ["示例：管道系统振动，" Getting Started with Abaqus: Interactive Edition 第 11.3 节](../gsa/gsa-link.md#gsa-stp-exapipingsys)。

### 11.3.4 检查输入文件——模型数据

以下步骤假设您可以访问此示例的完整输入文件。此输入文件 `pipe-2.inp` 在 ["管道系统振动，" 第 A.12 节](ap01s12.md) 中提供。获取和运行脚本的说明在 [附录 A，"示例文件"](ap01.md) 中给出。

模型定义——包括模型描述、节点和单元定义、截面特性和材料特性——将在接下来讨论。

**模型描述**

[*HEADING*](../key/key-link.md#usb-kws-mheading) 选项应在数据行中包含合适的标题。在示例输入文件中，此选项如下所示：

```
*HEADING
Analysis of a 5 meter long pipe under tensile load
Pipe has OD of 180 mm and ID of 140 mm
S.I. Units
```

**节点坐标和单元连接性**

检查正确的单元类型（PIPE32）已被使用，并且单元集名称具有描述性。

```
*ELEMENT, TYPE=PIPE32, ELSET=PIPE
```

创建包含管道段任一端节点的节点集。以下选项块为 [Figure 11--9](ch11s03.md#gss-node-elem1) 中所示的模型创建节点集：

```
*NSET, NSET=LEFT
1
*NSET, NSET=RIGHT
61
```

**梁特性**

[*BEAM SECTION*](../key/key-link.md#usb-kws-mbeamsection)，SECTION=PIPE 选项将与 PIPE32 单元一起使用。外半径（90 mm）和壁厚（20 mm）是几何定义此梁截面类型所需的。定义此模型的梁截面几何方向比前面章节中的货物起重机模型更容易，因为管道段是对称的。将近似 ![](../graphics/gsk_eqn00300.gif) 方向定义为向量（0.，0.，–1.0）。在此模型中，实际 ![](../graphics/gsk_eqn00301.gif) 向量将与此近似向量重合。

```
*BEAM SECTION, ELSET=PIPE, MATERIAL=STEEL, SECTION=PIPE
0.09, 0.02
0.0, 0.0, -1.0
```

**材料数据**

钢铁管道材料行为定义的选项块包含在以下行中：

```
*MATERIAL, NAME=STEEL
*ELASTIC
200.E9, 0.3
```

您必须定义钢材料的密度（7800 kg/m³），因为此模拟中正在提取特征模态和特征频率，并且此过程需要质量矩阵。因此，以下选项块必须跟在 [*ELASTIC*](../key/key-link.md#usb-kws-melastic) 选项块之后：

```
*DENSITY
7800.,
```

### 11.3.5 检查输入文件——历史数据

在此模拟中，您需要研究施加 4 MN 拉伸载荷时钢管道段的特征模态和特征频率。因此，载荷历史数据将分为两个步骤：

| 步骤 1. 一般步骤： | 施加 4 MN 拉伸力。 |
| --- | --- |
| 步骤 2. 线性摄动步骤： | 计算模态和频率。 |

这些步骤中的实际时间量不会影响结果；除非模型包含阻尼或率相关材料特性，"时间"在静态分析过程中没有物理意义。因此，在一般分析步骤中使用 1.0 的步骤时间。

**步骤 1 – 施加 4 MN 拉伸力**

审查定义第一个分析步骤所需的选项——包括过程定义、边界条件、载荷和输出请求。

**步骤和分析过程定义**

第一个步骤是包含几何非线性效应的通用静态步骤。指定初始增量大小为总步骤时间的 1/10，使 Abaqus 在第一个增量中施加 10% 的载荷。以下选项块定义分析过程，它们包括对步骤的有意义描述，以使审查载荷历史更容易：

```
*STEP, NLGEOM=YES
Apply axial tensile load of 4.0 MN
*STATIC
0.1, 1.0
```

**边界条件**

管道段在其左端（[Figure 11--9](ch11s03.md#gss-node-elem1) 中模型的节点 1）夹紧。它也在另一端夹紧；但是，必须在此端施加轴向力，因此仅约束自由度 2 到 6。

```
*BOUNDARY
LEFT, 1, 6
RIGHT, 2, 6
```

**拉伸载荷**

施加 4 MN 拉伸力到管道段右端，使其在正轴向（全局 1）方向变形。默认情况下，力在全局坐标系中施加。因此，[*CLOAD*](../key/key-link.md#usb-kws-hcload) 选项块看起来像

```
*CLOAD
RIGHT, 1, 4.0E6
```
在这种情况下，力直接施加到之前定义的节点集上。使用您的模型中的节点集名称或输入文件中 [*CLOAD*](../key/key-link.md#usb-kws-hcload) 选项中的节点编号。

**输出请求**

每 10 个增量向重启文件写入数据。此外，每 10 个增量也将预选场数据写入输出数据库文件，以及单元 25 的应力分量和应力不变量作为历史数据。以下选项块定义这些输出请求：

```
*ELSET, ELSET=ELEMENT25
25
*RESTART, WRITE, FREQUENCY=10
*OUTPUT, FIELD, FREQUENCY=10, VARIABLE=PRESELECT
*OUTPUT, HISTORY
*ELEMENT OUTPUT, ELSET=ELEMENT25
S, SINV

```
使用 [*END STEP*](../key/key-link.md#usb-kws-hendstep) 选项结束步骤。

**步骤 2 – 提取模态和频率**

第二个步骤提取扩展管道的固有频率。所需的选项将在下面讨论。

**步骤和分析过程定义**

在第二个步骤中，您需要计算扩展管道在其加载状态下的特征模态和特征频率。此步骤中使用的特征频率提取过程（[*FREQUENCY*](../key/key-link.md#usb-kws-hfrequency) 选项）是线性摄动过程。虽然只对第一个（最低）特征模态感兴趣，但提取模型的前八个特征模态。在 [*FREQUENCY*](../key/key-link.md#usb-kws-hfrequency) 选项块的数据行上指定此数字。定义分析过程的选项块应类似于以下内容：

```
*STEP, PERTURBATION
Extract modes and frequencies
*FREQUENCY
8,
```

**载荷**

您需要扩展管道段的固有频率。这不涉及任何摄动载荷的施加，固定边界条件从前一个一般步骤延续。因此，您不需要在此步骤中指定任何载荷或边界条件。

**输出请求**

任何线性摄动步骤中所需的输出请求必须重新定义，因为先前一般步骤的请求不会延续。您需要将数据写入重启文件和输出数据库文件。以下选项块定义这些请求：

```
*RESTART, WRITE
*OUTPUT, FIELD, VARIABLE=PRESELECT
```
再次，使用 [*END STEP*](../key/key-link.md#usb-kws-hendstep) 选项标记步骤定义的结束。

### 11.3.6 运行分析

将输入选项块存储在名为 `pipe.inp` 的文件中。使用以下命令在后台运行分析

```
abaqus job=pipe
```

### 11.3.8 后处理

使用命令

```
abaqus viewer odb=pipe
```

运行 Abaqus/Viewer。

**来自线性摄动步骤的变形形状**

当 Abaqus/Viewer 启动时，它自动使用输出数据库文件上最后一个可用的帧。此模拟第二步的结果是管道的自然模态形状和相应的固有频率。绘制第一模态形状。

**绘制第一模态形状：**

1. 从主菜单栏，选择 ****Result**![](../graphics/images/arrow.gif)**Step/Frame****。出现 **Step/Frame** 对话框。
2. 选择步骤 `Step-2` 和帧 `Mode 1`。
3. 点击 **OK**。
4. 从主菜单栏，选择 ****Plot**![](../graphics/images/arrow.gif)**Deformed Shape****。
5. 点击工具箱中的 ![](../graphics/ico_plotMultipleStates.png) 工具以允许视口中的多个绘图状态；然后点击 ![](../graphics/ico_plotUndeformed.png) 工具或选择 ****Plot**![](../graphics/images/arrow.gif)**Undeformed Shape****，以将未变形形状图添加到视口中现有的变形图中。
6. 在两个图上包含节点符号（叠加选项控制显示多个绘图状态时未变形形状的外观）。将节点符号颜色更改为绿色，符号形状更改为实心圆。
7. 点击自动拟合工具 ![](../graphics/ico_viewFit.png)，以便整个图被重新缩放以适合视口。

默认视图是等轴测。尝试旋转模型以找到第一特征模态的更好视图，类似于如图 [Figure 11--10](ch11s03.md#gss-pipesection) 所示。

**图 11–10** 拉伸载荷下管道段的第一和第二特征模态形状（模态位于相互正交的平面中）。

![](../graphics/gss-pipesection-v-nls.png)

由于这是线性摄动步骤，未变形形状是基态下的形状。这使得相对于基态的运动很容易看到。使用 **Frame Selector** 绘制其他模态形状。您会发现此模型对于每个自然频率都有多个重复特征模态。这是管道横截面对称性的结果，对于每个自然频率产生两个特征模态，对应于 1–2 和 1–3 平面。第二特征模态形状如图 [Figure 11--10](ch11s03.md#gss-pipesection) 所示。一些更高的振动模态形状如图 [Figure 11--11](ch11s03.md#gss-eigenmodes) 所示。

**图 11–11** 特征模态 3 到 6 的形状；相应的模态形状位于相互正交的平面中。

![](../graphics/gss-eigenmodes-v-nls.png)

与每个特征模态相关的固有频率报告在图的标题中。当施加 4 MN 拉伸载荷时，管道段的最低固有频率为 47.1 Hz。拉伸载荷增加了管道的刚度，从而增加了管道段的振动频率。此最低固有频率在谐载荷的频率范围内；因此，当与此载荷一起使用时，管道可能发生共振是一个问题。

因此，您需要继续模拟并向管道段施加额外的拉伸载荷，直到找到将管道段固有频率提高到可接受水平的幅度。与其重复分析并增加施加的轴向载荷，不如使用 Abaqus 中的重启功能来继续先前模拟的载荷历史，在新分析中计算模型的附加载荷历史响应。
