# *CONTACT FILE






### *CONTACT FILE定义接触变量的结果文件请求。

此选项用于控制将接触变量（用于接触表面对）写入 Abaqus/Standard 结果（`.fil`）文件。

**产品：**Abaqus/Standard  Abaqus/CAE  

**类型：**历史数据 

**级别：**步骤

**Abaqus/CAE: **不支持；仅从输出数据库文件读取输出。

##### **参考：**

- ["输出到数据和结果文件，" Abaqus 分析用户指南第 4.1.2 节](../usb/usb-link.md#usb-out-oprintfile)

### **可选参数：**

FREQUENCY

将此参数设置为输出频率（以增量为单位）。除非 FREQUENCY=0，否则输出将始终在每个步骤的最后增量写入。默认值为 FREQUENCY=1。设置 FREQUENCY=0 以禁止输出。

MASTER

将此参数设置为主表面的名称，为其发出此输出请求。

NSET

将此参数设置为节点集的名称，为其发出此输出请求。

SLAVE

将此参数设置为从表面的名称，为其发出此输出请求。

### **用于在结果文件中请求接触变量输出的数据行：**

**第一行：**

根据需要重复此数据行，以定义要写入的变量列表。如果省略此行，则将输出默认变量。
