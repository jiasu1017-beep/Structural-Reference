# 5 Customizing Abaqus/CAE and Abaqus/Viewer







This chapter describes user interface customization, printer configuration, and graphics card tuning for Abaqus/CAE and Abaqus/Viewer.

Platform dependencies sometimes exist for Abaqus/CAE and Abaqus/Viewer. These dependencies can change between releases; therefore, they are listed on the **Support** page at [www.3ds.com/simulia](http://www.3ds.com/simulia), where the latest information is published.
