# 28.2 在 Abaqus/CAE 中组装耦合欧拉-拉格朗日模型






Abaqus 允许您创建同时包含欧拉零件实例和拉格朗日零件实例的模型。在耦合欧拉-拉格朗日分析期间，拉格朗日网格与欧拉零件中的材料相互作用。耦合欧拉-拉格朗日分析通常比纯欧拉分析能更好地解释接触条件，特别是对于流体和固体材料之间的相互作用。在后处理期间，耦合欧拉-拉格朗日分析中的实体拉格朗物体看起来比纯欧拉分析中的类似物体能更好地保持其形状。

[图 28-2](pt04ch28s02.md#usi-eulerian-compare) 比较了纯欧拉分析（顶部）和耦合欧拉-拉格朗日分析（底部）中钢砖穿过水柱的情况。

**图 28-2** 纯欧拉分析（顶部）和耦合欧拉-拉格朗日分析（底部）的比较。

![](../graphics/usi-eulerian-compare.png)

在欧拉分析中，水倾向于粘附在砖的侧面，砖在最终配置中看起来变形了。这种明显的变形是欧拉材料使用的材料体积分数计算的结果，如["材料界面"中的"欧拉分析，" Abaqus 分析用户指南第 14.1.1 节](../usb/usb-link.md#usb-anl-aeuleriananal-mat)中所讨论的。另一方面，拉格朗日砖在穿过欧拉水时保持其形状，水绕着砖流动。

耦合欧拉-拉格朗日分析还允许您利用两种分析技术的优势；例如，您可以使用穿过欧拉材料的拉格朗日物体上的载荷来驱动拉格朗日物体的详细子模型。

要在 Abaqus/CAE 中组装耦合欧拉-拉格朗日模型，只需在同一装配中实例化欧拉和拉格朗日零件。耦合欧拉-拉格朗日分析仅可在**动态、显式**步骤中执行。您必须创建通用接触定义以启用拉格朗日和欧拉零件之间的接触。通用接触定义允许拉格朗日表面与模型中的欧拉材料实例之间的相互作用（请参见["在欧拉-拉格朗日模型中定义接触，" 第 28.3 节"](pt04ch28s03.md)，了解更多详细信息）。其他相互作用、载荷、边界条件和预定义场以常规方式应用于拉格朗日和欧拉零件。

在大多数情况下，拉格朗日零件被组装在欧拉零件实例内部。虽然拉格朗日单元和节点可以重叠，但三维拉格朗日单元不能占据与欧拉材料实例相同的空间。因此，拉格朗日零件必须实例化在欧拉零件实例内的空隙区域（即没有材料分配的区域）。要建模完全被欧拉材料包围的三维拉格朗日零件实例，请使用体积分数工具创建包含对应于拉格朗日零件实例的空隙区域的欧拉材料分配场（请参见["体积分数工具，" 第 28.5 节"](pt04ch28s05.md)，了解详细信息）。
