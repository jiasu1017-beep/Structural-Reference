# 5.1.3 访问结果文件信息

**产品：** Abaqus/Standard  Abaqus/Explicit

##### **参考文献**

- ["访问结果文件：概述，" 第5.1.1节](pt02ch05s01abo03.md)
- ["结果文件输出格式，" 第5.1.2节](pt02ch05s01afi01.md)
- ["访问结果文件的实用程序，" 第5.1.4节](pt02ch05s01abu01.md)

### 概述

Abaqus结果（`.fil`）文件使用内部数据管理程序写入，以最小化I/O成本。后处理程序必须使用这些相同的Abaqus数据管理程序来读取结果文件。必须调用以下实用程序来从Abaqus结果文件获取数据：
- `INITPF`
- `DBRNU`
- `DBFILE`
- `POSFIL`

您还可以通过使用以下实用子程序以Abaqus结果文件的格式写入文件：
- `INITPF`
- `DBFILW`

这些实用子程序的语法在["访问结果文件的实用程序，" 第5.1.4节](pt02ch05s01abu01.md)中描述。

### 读取浮点和整数变量

要在记录中读取浮点和整数变量，后处理程序中可以使用以下编码：

```
INCLUDE 'aba_param.inc'
      DIMENSION ARRAY(513), JRRAY(NPRECD,513)
      EQUIVALENCE (ARRAY(1),JRRAY(1,1))
```

使用此技术，例如，在每次调用`DBFILE` with `LOP`=0后，记录键可用为
```
KEY =  JRRAY (1,2)
```

使用`aba_param.inc`消除了对单精度和双精度使用不同版本代码的需要。文件`aba_param.inc`定义了适当的`IMPLICIT REAL`语句，并根据机器使用单精度还是双精度将`NPRECD`的值设置为1或2。当使用**abaqus make**实用程序编译和链接后处理程序时，从Abaqus安装的`site`子目录引用文件`aba_param.inc`（如下所述）。

### 链接后处理程序

运行Abaqus执行过程时，必须使用**make**参数链接后处理程序（参见["制作用户定义的可执行文件和子程序，" 第3.2.17节](pt01ch03s02abx17.md)）。为了正确链接，后处理程序不能包含FORTRAN PROGRAM语句。相反，程序必须以名为ABQMAIN的FORTRAN SUBROUTINE开始。

编译、链接和运行后处理程序包括两个步骤。例如，如果后处理程序的名称是`postproc.f`，请使用以下命令编译和链接`postproc.f`：

```
`abaqus make job=postproc`
```

然后必须使用以下命令运行程序：
```
`abaqus postproc`
```

### 调用实用子程序读取结果文件

在访问任何结果文件之前必须调用子程序`INITPF`。此子程序包含用于所有FORTRAN单元的FORTRAN OPEN语句，这些单元通过调用`INITPF`分配给结果文件；因此，您的代码不能包含这些单元的任何OPEN语句。Abaus根据提供的信息为给定单元构造文件名，作为`LRUNIT(1,K1)`和`FNAME`，如["访问结果文件的实用程序，" 第5.1.4节](pt02ch05s01abu01.md)中所述。

在读取第一个结果文件之前，还必须调用子程序`DBRNU`，然后在需要切换到读取另一个结果文件时再次调用。此子程序仅建立正在读取的结果文件的FORTRAN单元编号；不返回信息。`DBRNU`可以在`INITPF`之前或之后调用，但必须在`DBFILE`之前调用。

子程序`DBFILE`用于从结果文件读取每个记录。此子程序将以["结果文件输出格式，" 第5.1.2节](pt02ch05s01afi01.md)所述格式每次返回一个记录。

#### 示例

以下程序读取结果文件中的所有von Mises应力并获取最大值。然后，它打印此值以及发生该值的单元、截面点和积分点编号。

在此示例中，FORTRAN单元8用于读取结果文件，结果文件的名称假定为`TEST.fil`。结果文件假定为二进制文件，并且只读取一个结果文件。因此，`LRUNIT`维数为`LRUNIT(2,1)`；在调用`INITPF`程序时，`NRU`设置为1，`LRUNIT(1,1)`设置为8，`LRUNIT(2,1)`设置为2。不会写入新结果文件，因此`LOUTF`设置为零。

```
      SUBROUTINE ABQMAIN
C     Calculate the maximum von Mises stress and its location
C
      INCLUDE 'aba_param.inc'
      CHARACTER*80 FNAME
      DIMENSION ARRAY(513),JRRAY(NPRECD,513),LRUNIT(2,1)
      EQUIVALENCE (ARRAY(1),JRRAY(1,1))
C
C     File initialization
C
      FNAME='TEST'
      NRU=1
      LRUNIT(1,1)=8
      LRUNIT(2,1)=2
      LOUTF=0
      CALL INITPF(FNAME,NRU,LRUNIT,LOUTF)
      JUNIT=8
      CALL DBRNU(JUNIT)
C
C     Loop on all records in results file
C
      STRESS=0.
      DO 100 K1=1,99999
C
        CALL DBFILE(0,ARRAY,JRCD)
        IF(JRCD.NE.0)GO TO 110
        KEY=JRRAY(1,2)
C
        IF(KEY.EQ.1) THEN
C
C          Element header record:
C          extract element, sec pt, int pt numbers
C
           JEL=JRRAY(1,3)
           JPNT=JRRAY(1,4)
           JSPNT=JRRAY(1,5)
C
C     Stress invariant record for Abaqus/Standard
        ELSE IF(KEY.EQ.12)THEN
C     Stress invariant record for Abaqus/Explicit
        ELSE IF(KEY.EQ.75)THEN
C
C          Extract von Mises stress
C
           IF(ARRAY(3).GT.STRESS)THEN
              STRESS=ARRAY(3)
              KEL=JEL
              KPNT=JPNT
              KSPNT=JSPNT
           END IF
        END IF
C
 100  CONTINUE
 110  CONTINUE
C
      WRITE(6,120) KEL,KPNT,KSPNT,STRESS
 120  FORMAT(5X,'ELEMENT',I5,5X,'POINT',I4,5X,'SECTION POINT',
     1 I4,5X,'STRESS',1PG12.3)
      STOP
      END
```

有关其他示例，请参见[Abaqus Example Problems Guide第15章，"Abaqus结果的后处理"](../exa/exa-link.md#exa-pst-userpostprocchap)。

### 以结果文件格式写入文件

子程序`DBFILW`可用于以Abaqus结果文件的格式写入文件，以修改文件信息或在后处理之前添加其他信息。必须在`DBFILW`之前调用子程序`INITPF`。

文件将写入FORTRAN单元9，扩展名为`.fin`。当首次调用`DBFILW`时，Abaqus打开单元9；您的编码不能打开或重新定义单元9，但您必须确保在作业后保存FORTRAN单元9。

["连接多个结果文件的数据和转换文件格式：FJOIN，" Abaqus Example Problems Guide第15.1.2节](../exa/exa-link.md#exa-pst-fjoin)，包含使用子程序`DBFILW`合并不连续结果文件特定记录的示例。后处理需要连续的结果文件；如果您在分析期间写入了一个结果文件，然后在重启分析时写入了新结果文件而没有使文件连续，则必须在后处理之前使它们连续。["地震运动下悬臂梁的分析，" Abaqus Benchmarks Guide第1.4.13节](../bmk/bmk-link.md#bmk-anl-cantilever)，也展示了使用`DBFILW`合并结果文件。或者，可以使用**abaqus append**实用程序合并结果文件，如["连接结果（`.fil`）文件，" 第3.2.14节](pt01ch03s02abx14.md)中所述。

`DBFILW`子程序还可用于将Abaqus结果文件从二进制转换为ASCII格式，以将其从一个计算机系统传输到另一个计算机系统。或者，可以使用**abaqus ascfil**执行过程自动完成此转换，如["结果（`.fil`）文件的ASCII转换，" 第3.2.13节](pt01ch03s02abx13.md)中所述。
