# 4.1.3 DLOAD





### 4.1.3 [`DLOAD`](../sub/sub-link.md#sub-xsl-dload)

**产品：**Abaqus/Standard  

### 测试功能

用于指定非均匀分布载荷的用户子程序。

### 一般说明

用户子程序[`DLOAD`](../sub/sub-link.md#sub-xsl-dload)可用于定义多种类型的非均匀分布载荷。本指南其他章节中描述的一些测试给出了使用此子程序的示例。此子程序的使用不限于这些测试中所示的应用。["纯弯曲圆柱体：CAXA单元，" 第1.3.33节](ch01s03abv36.md)和["承受非对称压力载荷的圆柱体：CAXA单元，" 第1.3.35节](ch01s03abv38.md)说明了使用子程序[`DLOAD`](../sub/sub-link.md#sub-xsl-dload)向CAXA非对称轴对称连续单元施加非对称载荷。请参阅["轴对称单元的补丁测试，" 第1.5.4节](ch01s05abv72.md)中的问题描述，了解在轴对称连续应力-位移单元的补丁测试中使用子程序定义非均匀体力的示例。子程序[`DLOAD`](../sub/sub-link.md#sub-xsl-dload) also也用于["非均匀裂纹面载荷和J积分，" Abaqus基准指南第1.16.7节](../bmk/bmk-link.md#bmk-anl-crackloadingjint)中的测试，以向裂纹面施加均匀和非均匀载荷。
