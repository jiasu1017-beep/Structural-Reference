# 10.4.1 Abaqus/CAE中的IGES选项是什么？





IGES选项允许您控制以下内容：

**修剪曲线偏好（Trim Curve Preference）**

IGES文件可以包含使用实空间、参数空间或两者定义的曲线。当您将零件从IGES文件导入Abaqus/CAE时，曲面和修剪曲线将转换为零件的内部表示。默认情况下，Abaqus/CAE使用IGES文件中存储的信息来决定修剪曲线的定义方式；或者，您可以强制Abaqus/CAE始终使用实空间或参数空间。
- **按IGES文件（As per IGES file）**。这是默认选项。选中此选项时，Abaqus/CAE使用**始终使用参数数据（Always use parametric data）**选项或**始终使用3D数据（Always use 3D data）**选项来决定修剪曲线的定义方式。IGES文件中的信息决定使用两个选项中的哪一个。

- **始终使用参数数据（Always use parametric data）**。此选项使用曲线所在的曲面对修剪曲线进行参数化计算。修剪曲线上的每个数据点都由曲面参数（*u*、*v*）定位。Abaqus/CAE评估与数据点对应的曲面并为该点生成三维坐标。如果底层曲面有太多无法精确参数化的急剧偏转，当Abaqus/CAE尝试重建零件时，修剪曲线可能不在曲面上。这会产生修剪错误，并可能导致边缘之间出现间隙。

- **始终使用3D数据（Always use 3D data）**。此选项从空间中的三维坐标——零件的坐标系——以及修剪曲线位于参数曲面上的指示来计算修剪曲线。每个数据点都有自己的三维几何点；因此，每次移动曲面时都必须重新评估修剪曲线。修剪曲线只能沿曲面移动。**始终使用3D数据**选项应允许修剪曲线与其底层曲面相符；但这不能保证。如果选择**始终使用3D数据**选项，导入将需要更长的时间完成。

**MSBO**

流形实体B-rep对象（MSBO，实体类型186）是B-rep实体的IGES术语。与所有B-rep实体一样，MSBO实体通过引用定义实体的所有修剪曲面来指示实体的整体拓扑。Abaqus/CAE在扫描IGES格式文件并找到实体后自动设置MSBO选项。

对于包含MSBO实体的IGES文件，创建文件的CAD包必须在导出过程中包含MSBO。CATIA V5允许MSBO实体；SolidWorks不允许。

**级别（Levels）**

CAD应用程序可以将实体以级别序列（也称为层）存储在IGES格式文件中。例如，不同的级别可以包含几何图形、尺寸、文本注释、构造线、注释或图例。当您从IGES格式文件导入零件时，Abaqus/CAE会扫描文件，**从IGES文件创建零件（Create Part from IGES File）**对话框会显示在IGES文件中找到的级别列表。

如果级别包含几何图形以外的内容（如尺寸），Abaqus/CAE会在导入期间忽略该级别。但是，如果级别确实包含几何图形，您可以从对话框的列表中删除该级别，Abaqus/CAE将不会导入该级别。您必须至少导入一个级别。因此，如果IGES文件仅包含一个级别，**级别（Level）**字段将不可用。Abaqus/CAE会忽略您在IGES文件中不存在的任何级别。

如果在导入过程中遇到问题，您可能需要修改默认设置。
