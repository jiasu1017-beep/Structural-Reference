# 20.9.6 显示和隐藏构造几何







您可以创建[构造几何](pt03ch20s05s02.md)来帮助您在草图中定位和对齐对象。例如，下图显示了一系列沿斜构造线对齐的孔。孔中心位于垂直构造线与斜构造线的交点处：

![](../graphics/ske-congeom-example-nls.png)

如果构造几何造成干扰，可以使用**Sketcher 选项**对话框中的**显示构造几何**选项将其隐藏。（默认情况下，启动 Sketcher 时会显示构造几何。）如果启用了预选，光标仍会捕捉到与隐藏构造几何关联的项目，例如直线与构造线的交点。构造几何显示的自定义适用于当前草图，并与草图一起存储。

**显示或隐藏构造几何：**

1. 从 Sketcher 工具箱底部，选择自定义工具![](../graphics/ico_sketchOptions-nls.png)。出现**Sketcher 选项**对话框。
2. 从**常规**选项卡页面，切换**显示构造几何**。当**显示构造几何**打开时，Abaqus/CAE 在草图中显示构造几何。
3. 选择完所需的自定义选项后，点击**确定**应用您的更改并关闭**Sketcher 选项**对话框。
