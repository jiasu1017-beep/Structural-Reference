# *PRE-TENSION SECTION







### *PRE-TENSION SECTION将预拉伸节点与预拉伸截面关联。

此选项用于将预拉伸节点与预拉伸截面关联。

**产品：**Abaqus/Standard  Abaqus/CAE  

**类型：**模型数据  

**级别：**零件，零件实例，装配  

**Abaqus/CAE：**载荷模块

##### **参考：**

- ["预定义装配载荷，" Abaqus Analysis User's Guide第34.5.1节](../usb/usb-link.md#usb-prc-ppretension)

### **必需参数：**

NODE

将此参数设置为预拉伸节点编号或包含预拉伸节点的节点集名称。如果选择节点集名称，则该节点集必须恰好包含一个节点。

### **必需的互斥参数：**

ELEMENT

将此参数设置为用于定义预拉伸截面的桁架或梁单元的单元编号，或包含用于定义预拉伸截面的桁架或梁单元的元素集名称。如果选择元素集名称，则该元素集必须恰好包含一个单元。

SURFACE

将此参数设置为定义预拉伸截面的表面定义（[*SURFACE](ch18abk47.md)）的名称（当使用连续单元时）。

### **定义截面法线的数线（可选）：**

**第一行（也是唯一数据行）：**

如果省略数据行，Abaqus/Standard将为连续单元计算预拉伸截面的平均法线。对于桁架或梁单元，默认法线从单元连接中的第一个节点指向最后一个节点。
