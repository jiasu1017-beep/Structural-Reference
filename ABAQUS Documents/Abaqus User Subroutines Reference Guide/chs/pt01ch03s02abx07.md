# 3.2.7 Abaqus/Viewer执行





**产品：** Abaqus/Viewer  

##### **参考**

- ["Abaqus执行程序：概述，" 第3.1.1节](pt01ch03s01abo02.md)

### 概述

Abaqus/Viewer是Abaqus/CAE的子集，仅包含可视化模块的后处理功能，通过运行Abaqus执行程序并指定**viewer**参数来执行。

### 命令行选项

**database**

此选项指定要使用的输出数据库文件的名称（如果与*job-name*不同）。该过程在命令行上按输入的*database-file*搜索，并带有`.odb`文件扩展名。

**replay**

此选项指定要读取Abaqus/Viewer命令的文件的名称。*replay-file*中的命令将在Abaqus/Viewer启动时立即执行。如果未给出文件扩展名，则默认扩展名为`.rpy`。您不能使用`replay`选项来执行包含控制流语句的脚本。

**startup**

此选项指定包含在应用程序启动时运行的Python配置命令的文件的名称。此文件中的命令在任何已在环境文件中设置的配置命令之后运行。Abaqus/Viewer在执行时不将这些命令回放到重放文件中。

**script**

此选项指定包含在应用程序启动时运行的Python配置命令的文件的名称。此文件中的命令在任何已在环境文件中设置的配置命令之后运行。

**noGUI**

此选项指定在不显示图形用户界面(GUI)的情况下运行Abaqus/Viewer。如果未指定文件名，则会检出Abaqus/Viewer许可证，并初始化Python解释器以允许交互式输入Python或Abaqus脚本界面命令。

如果指定了文件名，Abaqus/Viewer将运行该文件中的命令，并在完成后退出。如果未给出文件扩展名，则默认扩展名为`.py`。此选项对于自动执行后处理分析任务非常有用，而无需运行显示界面。由于不提供界面，脚本不能包含任何用户交互。

**noenvstartup**

此选项指定在应用程序启动时不应运行环境文件中的所有配置命令。此选项可与**script**命令结合使用，以禁止**script**文件中除那些之外的 所有配置命令。

**noSavedOptions**

此选项指定Abaqus/Viewer不应应用存储在`abaqus_v6.14.gpr`中的显示选项设置（例如，渲染样式和边界条件的显示）。有关更多信息，请参阅["保存您的显示选项设置，" Abaqus/CAE用户指南第76.16节](../usi/usi-link.md#uss-dsp-preferences)。

**noSavedGuiPrefs**

此选项指定Abaqus/Viewer不应应用存储在`abaqus_v6.14.gpr`中的GUI设置（例如，Abaqus/CAE主窗口或其对话框的大小和位置）。

**noStartupDialog**

此选项指定不应显示Abaqus/Viewer的**Start Session**对话框。

**custom**

此选项指定包含Abaqus GUI工具包命令的文件的名称。此选项执行Abaqus/Viewer的定制版本应用程序。有关更多信息，请参阅[Abaqus GUI工具包用户指南第1章"简介"](../cus/cus-link.md#cus-int-int)。

**guiTester**

此选项启动一个包含Python开发环境以及Abaqus/Viewer的单独用户界面。Python开发环境允许您创建、编辑、单步执行和调试Python脚本。有关更多信息，请参阅[Abaqus脚本用户指南第三部分"Abaqus Python开发环境"](../cmd/cmd-link.md#cmd-pde)。

您可以将脚本指定为此选项的参数，这会提示Abaqus/Viewer运行GUI脚本。当到达脚本末尾时，Abaqus/Viewer关闭。

**guiRecord**

此选项使您能够在名为`abaqus.guiLog`的文件中记录Abaqus/Viewer用户界面中的操作。您也可以通过使用环境变量`ABQ_CAE_GUIRECORD`在启动时设置此选项。**guiRecord**选项不能与**guiTester**选项一起使用。

**guiNoRecord**

此选项使您能够在设置环境变量`ABQ_CAE_GUIRECORD`时禁用用户界面记录。
