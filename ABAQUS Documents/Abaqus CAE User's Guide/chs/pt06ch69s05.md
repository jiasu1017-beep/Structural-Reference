# 69.5 从孤立单元创建零件





您可以使用几何编辑工具集中的工具从孤立网格单元的面创建几何。此技术在您拥有需要修改的网格化零件但没有部分或全部几何可用时很有用。向孤立网格添加几何特征、从头重新创建零件几何或从孤立单元逆向工程几何的决定取决于多种因素，包括：
- 现有零件的复杂性
- 要添加的特征的复杂性
- 修改现有孤立网格的难度
- 为整个零件创建新网格的愿望

其他因素，如项目的时间限制和导出模型几何的能力，也可能影响您的决定。

从孤立网格创建零件的目的是创建与原始设计意图相匹配的几何，同时为当前和未来的修改提供平台。几何还必须适合通过与当前网格完全关联或创建新网格来创建本地网格。

以下准则将帮助您为孤立网格创建几何：

**创建新的几何面**

使用**从单元面创建面**工具 ![](../graphics/icoR_elementFacesToFace.png) 为模型创建外表面几何。每次使用该工具都会从选定的外部孤立单元面创建单个壳面。该工具包含多种选择多个孤立单元边界面的方法，它会自动将新创建的面缝合到任何相邻的几何上。有关更多信息，请参见["从单元面创建面," 第69.7.10节](pt06ch69s07hlb10.md)。当您完成创建面后，所有外部孤立单元面都应被几何面覆盖。

**移除多余的边和面**

使用几何编辑工具集中的其他工具移除多余的边、小面以及会限制良好网格生成的任何其他特征。如有必要，将零件转换为精确几何。有关更多信息，请参见["修复几何的策略," 第69.4节](pt06ch69s04.md)。

**转换为实体**

如果您正在处理实体零件，请使用**从壳创建实体**工具 ![](../graphics/ico_repairSolidFromShell.png) 来填充创建的面内部的空间。

**分割和网格划分零件**

抑制或删除孤立网格，并为零件创建新网格。

**修复后创建分割**

如果您的零件包含分割，几何编辑工具集可能会在修复操作期间删除它们。为避免此问题，您应该等到完成编辑后再分割零件。
