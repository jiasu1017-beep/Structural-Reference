# 49.4 控制动画播放





在基于对象的动画期间，您可以控制播放的速度和重复，您可以停止、重新启动和步进浏览动画。大多数行业标准动画软件支持基于图像动画的相同功能，但可用选项基于您选择的应用程序。对于基于对象的动画，提供了一个附加选项：您可以在动画播放时显示或抑制状态信息。

如果您使用多个视口来显示动画，您可以包括或排除公共动画时间线中的各个视口。默认情况下，您创建的每个新视口都将被包括在同步中。
