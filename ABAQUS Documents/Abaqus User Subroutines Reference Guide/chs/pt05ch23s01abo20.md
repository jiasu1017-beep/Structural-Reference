# 23.1.1 非弹性行为





Abaqus中的材料库包含多个非弹性行为模型：

**经典金属塑性**：金属在相对低温下的屈服和非弹性流动，其中加载相对单调且蠕变效应不重要，通常可以用经典金属塑性模型（["经典金属塑性，" 第23.2.1节"](pt05ch23s02abm17.md)）来描述。在Abaqus中，这些模型使用标准Mises或Hill屈服面及相关塑性流动。经典金属塑性模型中均提供了理想塑性和各向同性硬化定义。常见应用包括碰撞分析、金属成形和一般坍塌研究；这些模型简单且足以处理此类情况。

**承受循环加载的金属模型**：线性运动硬化模型或非线性各向同性/运动硬化模型（["承受循环加载的金属模型，" 第23.2.2节"](pt05ch23s02abm18.md)）可用于Abaqus中模拟承受循环加载的材料行为。这些模型中的演化定律由运动硬化分量（描述屈服面在应力空间中的平移）和非线性各向同性/运动硬化模型的各向同性分量（描述弹性范围的变化）组成。两种模型都可以模拟包辛格效应和塑性安定，但非线性各向同性/运动硬化模型提供更准确的预测。只有非线性各向同性/运动硬化模型才能考虑棘轮效应和平均应力松弛。除这两个模型外，Abaqus/Standard中的ORNL模型可用于对承受低循环疲劳和蠕变疲劳的不锈钢进行简单寿命估算（见下文）。

**率相关屈服**：随着应变率的增加，许多材料表现出屈服强度的增加。率相关性（["率相关屈服，" 第23.2.3节"](pt05ch23s02abm19.md)）可以在Abaqus中为多个塑性模型定义。率相关性可用于静态和动态过程。适用模型包括经典金属塑性、扩展Drucker-Prager塑性和可压碎泡沫塑性。

**蠕变和肿胀**：Abaqus/Standard提供了经典金属蠕变行为和时间相关体积肿胀行为的材料模型（["率相关塑性：蠕变和肿胀，" 第23.2.4节"](pt05ch23s02abm20.md)）。该模型适用于相对缓慢（准静态）的非弹性变形，如金属或玻璃的高温蠕变流动。蠕变应变率假定为纯偏量的，意味着与非弹性应变相关的体积变化为零。蠕变可与经典金属塑性模型、ORNL模型一起使用，也可用于定义率相关垫片行为（["直接使用垫片行为模型定义垫片行为，" 第32.6.6节"](pt06ch32s06alm51.md)）。肿胀可与经典金属塑性模型一起使用。（与Drucker-Prager模型的使用说明见下文。）

**退火或熔化**：Abaqus提供了一种建模能力，用于模拟在高于某个用户定义温度（称为退火温度）时发生的与硬化相关的记忆丧失（["退火或熔化，" 第23.2.5节"](pt05ch23s02abm21.md)）。它适用于承受高温变形过程的金属，其中材料可能经历熔化和可能的再凝固或其他形式的退火。在Abaqus中，退火或熔化可以用经典金属塑性（Mises和Hill）建模；在Abaqus/Explicit中，退火或熔化也可以用Johnson-Cook塑性建模。退火温度假定为材料属性。有关在Abaqus/Explicit中模拟退火的替代方法，请参见["退火过程，" 第6.12.1节"](pt03ch06s12at31.md)。

**各向异性屈服和蠕变**：Abaqus提供了一个各向异性屈服模型（["各向异性屈服/蠕变，" 第23.2.6节"](pt05ch23s02abm22.md)），可用于与经典金属塑性（["经典金属塑性，" 第23.2.1节"](pt05ch23s02abm17.md)）、运动硬化（["承受循环加载的金属模型，" 第23.2.2节"](pt05ch23s02abm18.md)）和/或蠕变（["率相关塑性：蠕变和肿胀，" 第23.2.4节"](pt05ch23s02abm20.md)）一起使用的材料，这些材料在不同方向表现出不同的屈服应力。Abaqus/Standard模型包含蠕变；Abaqus/Explicit中蠕变行为不可用。该模型允许为每个应力分量指定不同的应力比以定义初始各向异性。该模型不适用于因加载而各向异性发生显著变化的案例。

**Johnson-Cook塑性**：Abaqus/Explicit中的Johnson-Cook塑性模型（["Johnson-Cook塑性，" 第23.2.7节"](pt05ch23s02abm23.md)）特别适用于模拟金属的高应变率变形。该模型是一种特定类型的Mises塑性，包含硬化定律和分析形式的率相关性。它通常用于绝热瞬态动力学分析。

**动态失效模型**：Abaqus/Explicit为Mises和Johnson-Cook塑性模型提供了两种类型的动态失效模型（["动态失效模型，" 第23.2.8节"](pt05ch23s02abm24.md)）。一种是剪切失效模型，其中失效准则基于累积等效塑性应变。另一种是拉伸失效模型，它使用静水压力应力作为失效度量来模拟动态层裂或压力截止。这两种模型都提供多种失效选项，包括单元删除，主要适用于真正的动态情况。相比之下，逐步失效和损伤模型（[第24章"逐步损伤和失效"](pt05ch24.md)"）适用于准静态和动态情况，并具有其他显著优势。

**多孔金属塑性**：多孔金属塑性模型（["多孔金属塑性，" 第23.2.9节"](pt05ch23s02abm25.md)）用于模拟以孔洞萌生和生长形式表现损伤的材料，也可用于一些在相对高相对密度（相对密度定义为固体材料体积与材料总体积之比）下的粉末金属工艺模拟。该模型基于Gurson多孔金属塑性理论及孔洞形核，适用于相对密度大于0.9的材料。该模型适用于相对单调的加载。

**铸铁塑性**：铸铁塑性模型（["铸铁塑性，" 第23.2.10节"](pt05ch23s02abm26.md)）用于模拟灰口铸铁，其在拉伸和压缩中表现出明显不同的非弹性行为。灰口铸铁的微观结构由分布在钢基体中的石墨片组成。在拉伸时，石墨片起到应力集中的作用；而在压缩时，石墨片有助于传递应力。产生的材料在拉伸时是脆性的，但在压缩时其行为类似于钢。拉伸和压缩塑性响应的差异包括：（i）拉伸屈服应力比压缩屈服应力低三到五倍；（ii）拉伸时体积永久增加，但压缩时非弹性体积变化可忽略不计；（iii）拉伸和压缩中硬化行为不同。该模型适用于相对单调的加载。

**双层黏塑性**：Abaqus/Standard中的双层黏塑性模型（["双层黏塑性，" 第23.2.11节"](pt05ch23s02abm27.md)）适用于模拟观察到显著时间相关行为以及塑性的材料。对于金属，这通常发生在高温下。该模型已被证明在热机械加载中提供良好的结果。

**ORNL本构模型**：Abaqus/Standard中的ORNL塑性模型（["ORNL -- 橡树岭国家实验室本构模型，" 第23.2.12节"](pt05ch23s02abm28.md)）旨在用于304和316不锈钢的循环加载和高温蠕变。塑性和蠕变计算根据核标准NEF 9-5T"一级高温核系统组件设计指南和程序"提供。该模型是线性运动硬化模型（见上文）的扩展，旨在为低循环疲劳和蠕变疲劳成为关键问题时提供简单的寿命估算。

**变形塑性**：Abaqus/Standard提供了一个变形理论Ramberg-Osgood塑性模型（["变形塑性，" 第23.2.13节"](pt05ch23s02abm29.md)），用于开发延性金属断裂力学应用中的完全塑性解。该模型最常用于小位移分析的静态加载，其中必须在模型的一部分中开发完全塑性解。

**扩展Drucker-Prager塑性和蠕变**：扩展Drucker-Prager塑性模型系列（["扩展Drucker-Prager模型，" 第23.3.1节"](pt05ch23s03abm30.md)）描述了屈服行为取决于等效压力应力的粒状材料或聚合物行为。非弹性变形有时可能与摩擦机制相关，如颗粒相互滑动。该类模型提供了三种不同屈服准则的选择。准则的差异基于子午面上屈服面的形状，可以是线性形式、双曲线形式或一般指数形式。在Abaqus/Standard中，线性形式模型的塑性行为也与非弹性时间相关（蠕变）行为耦合；Abaqus/Explicit中蠕变行为不可用。

**改进的Drucker-Prager/帽盖塑性和蠕变**：改进的Drucker-Prager/帽盖塑性模型（["改进的Drucker-Prager/帽盖模型，" 第23.3.2节"](pt05ch23s03abm31.md)）可用于模拟表现出压力相关屈服的地质材料。添加帽盖屈服面有助于控制材料在剪切屈服时的体积膨胀，并提供非弹性硬化机制来表示塑性压实。在Abaqus/Standard中，该模型的塑性行为也与非弹性时间相关（蠕变）行为耦合；可能的两种蠕变机制：内聚力、Drucker-Prager样机制和固结、帽盖样机制。

**Mohr-Coulomb塑性**：Mohr-Coulomb塑性模型（["Mohr-Coulomb塑性，" 第23.3.3节"](pt05ch23s03abm32.md)）可用于岩土工程领域的设计应用。该模型使用经典的Mohr-Coloumb屈服准则：子午面上的一条直线和偏平面上的不规则六边形截面。然而，Abaqus的Mohr-Coulomb模型具有完全光滑的流动势，而不是经典的六角锥：流动势在子午面上是一条双曲线，并使用Mentrey和Willam提出的光滑偏平面截面。

**临界状态（黏土）塑性**：黏土塑性模型（["临界状态（黏土）塑性模型，" 第23.3.4节"](pt05ch23s03abm33.md)）描述无黏性土壤的非弹性响应。该模型对饱和黏土的实验观察行为提供了合理的匹配。该模型通过依赖于三个应力不变量的屈服函数、可定义塑性应变率的关联流动假设以及根据非弹性体积应变改变屈服面大小的应变硬化理论来定义材料的非弹性行为。

**可压碎泡沫塑性**：泡沫塑性模型（["可压碎泡沫塑性模型，" 第23.3.5节"](pt05ch23s03abm34.md)）旨在模拟通常用作能量吸收结构的可压碎泡沫；然而，其他可压碎材料（如巴沙木）也可以用该模型模拟。该模型最适用于相对单调的加载。具有各向同性硬化的可压碎泡沫模型适用于聚合物泡沫和金属泡沫。

**节理材料**：Abaqus/Standard中的节理材料模型（["节理材料模型，" 第23.5.1节"](pt05ch23s05abm36.md)）旨在为包含大量不同方向平行节理面的材料（如沉积岩）提供一个简单的连续体模型。该模型适用于主要承受压应力的应用，并在节理法向应力试图变为拉伸时提供节理张开能力。

**混凝土**：Abaqus为低围压下的混凝土分析提供了三种不同的本构模型：Abaqus/Standard中的 smeared crack混凝土模型（["混凝土smeared crack模型，" 第23.6.1节"](pt05ch23s06abm37.md)）；Abaqus/Explicit中的脆性断裂模型（["混凝土断裂模型，" 第23.6.2节"](pt05ch23s06abm38.md)）；以及Abaqus/Standard和Abaqus/Explicit中的混凝土损伤塑性模型（["混凝土损伤塑性，" 第23.6.3节"](pt05ch23s06abm39.md)）。每个模型都设计用于为所有类型结构（梁、桁架、壳和实体）中的素混凝土和钢筋混凝土（以及其他类似的准脆性材料）提供一般能力。smeared crack混凝土模型适用于混凝土承受基本单调应变且材料点表现出拉伸裂缝或压缩压碎的应用。塑性应变在压缩中由"压缩"屈服面控制。裂缝被假定为行为中最重要的方面，裂缝和裂缝后各向异性行为的表示主导了建模。脆性断裂模型在Abaqus/Explicit中适用于混凝土行为以拉伸裂缝为主且压缩失效不重要的应用。该模型包括对裂缝引起的各向异性的考虑。在压缩中，模型假定弹性行为。可以使用简单的脆性失效准则来允许从网格中删除单元。混凝土损伤塑性模型在Abaqus/Standard和Abaqus/Explicit中基于标量（各向同性）损伤假设，适用于混凝土承受任意加载条件（包括循环加载）的应用。该模型考虑了拉伸和压缩中塑性应变引起的弹性刚度退化。它还考虑了循环加载下的刚度恢复效应。

**逐步损伤和失效**：Abaqus/Explicit为延性金属和纤维增强复合材料提供了逐步损伤和失效建模的一般能力（[第24章"逐步损伤和失效"](pt05ch24.md)"）。

### 塑性理论

大多数工程材料的初始响应都是弹性的。弹性行为意味着变形是完全可恢复的：当移除负载时，标本恢复到其原始形状。如果负载超过某个极限（"屈服负载"），变形将不再完全可恢复。当移除负载时，部分变形将保留，例如当回形针被弯曲过多时，或当金属坯料在制造过程中被轧制或锻造时。塑性理论对材料在发生这种延性不可恢复变形时的力学响应进行建模。这些理论为金属开发得最为密集，但也应用于土壤、混凝土、岩石、冰、可压碎泡沫等。这些材料的行为非常不同。例如，大的纯静水压力在金属中引起很小的非弹性变形，但在土壤样本中相当小的静水压力值可能引起显著的非恢复体积变化。尽管如此，塑性理论的基本概念足够通用，以至于基于这些概念开发的模型已成功应用于各种材料。

Abaqus中的大多数塑性模型是"增量"理论，其中力学应变率分解为弹性部分和塑性（非弹性）部分。增量塑性模型通常以以下形式表述：
- 一个*屈服面*，它将"屈服负载"的概念推广为一个测试函数，可用于确定材料在特定应力、温度等状态下是否纯粹弹性响应；
- 一个*流动法则*，它定义了如果材料点不再纯粹弹性响应时发生的非弹性变形；和
- 定义*硬化*的演化定律——即屈服和/或流动定义如何随非弹性变形而变化。

Abaqus/Standard还有一个"变形"塑性模型，其中应力由总力学应变定义。这是一个Ramberg-Osgood模型（["变形塑性，" 第23.2.13节"](pt05ch23s02abm29.md)），主要用于延性断裂力学应用，通常需要完全塑性解。

### 弹性响应

Abaqus塑性模型还需要弹性定义来处理应变的可恢复部分。在Abaqus中，弹性通过在同一材料定义中包含线性弹性行为或相关塑性模型的多孔弹性行为来定义（见["材料数据定义，" 第21.1.2节"](pt05ch21s01aus109.md)）。对于Abaqus/Explicit中的Mises和Johnson-Cook塑性模型，弹性也可以使用状态方程和相关偏量行为来定义（见["状态方程，" 第25.2.1节"](pt05ch25s02abm50.md)）。

在有限应变的弹塑性分析中，Abaqus假定塑性应变主导变形，弹性应变为小应变。这一限制是由Abaqus使用的弹性模型强加的。这是合理的，因为大多数材料都有明确定义的屈服点，该屈服点仅是杨氏模量的很小一部分；例如，金属的屈服应力通常小于材料杨氏模量的1%。因此，弹性应变也将小于该百分比，材料的弹性响应可以非常准确地建模为线性的。

在Abaqus/Explicit中，弹性应变能是增量更新的。弹性应变能的增量变化（ΔU~e~）计算为ΔU~e~ = ΔU - ΔU~p~，其中ΔU是总应变能的增量变化，ΔU~p~是塑性能量耗散的增量变化。对于几乎全是塑性的变形增量，ΔU~e~远小于ΔU和ΔU~p~。ΔU和ΔU~p~计算中的近似导致与真实解的偏差，这些偏差与ΔU和ΔU~p~相比不显著，但可能与ΔU~e~相比显著。通常，弹性应变能解相当准确，但在极少数情况下，ΔU和ΔU~p~计算中的近似可能导致弹性应变能报告负值。这些负值最可能发生在使用率相关塑性的分析中。只要弹性应变能的绝对值与总应变能相比非常小，弹性应变能的负值不应被视为严重解决方案问题的指示。

### 应力和应变度量

大多数表现出延性行为（大非弹性应变）的材料在远低于材料弹性模量的应力水平下屈服，这意味着相关应力和应变度量是"真实"应力（Cauchy应力）和对数应变。因此，所有这些模型的材料数据都应以这些度量给出。

如果您有单轴测试的名义应力-应变数据且材料是各向同性的，真实应力和对数塑性应变的简单转换为：

σ~true~ = σ~nom~ (1 + ε~nom~)

ε~pl~ = ln(1 + ε~nom~) - σ~true~/E

其中E是杨氏模量。

#### 指定初始等效塑性应变

可以在Abaqus中为使用经典金属塑性（["经典金属塑性，" 第23.2.1节"](pt05ch23s02abm17.md)）或Drucker-Prager塑性（["扩展Drucker-Prager模型，" 第23.3.1节"](pt05ch23s03abm30.md)）的单元指定初始等效塑性应变值，方法为定义初始硬化条件（["Abaqus/Standard和Abaqus/Explicit中的初始条件，" 第34.2.1节"](pt07ch34s02aus116.md)）。等效塑性应变（输出变量PEEQ）然后包含初始等效塑性应变加上分析期间因塑性应变而产生的任何附加等效塑性应变。然而，塑性应变张量（输出变量PE）仅包含分析期间因变形而产生的应变。
