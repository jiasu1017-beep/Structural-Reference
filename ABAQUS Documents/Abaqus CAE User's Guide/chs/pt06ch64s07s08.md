# 64.7.7 将实体孤网格转换为壳孤网格







选择 ****Mesh**![](../graphics/images/arrow.gif)**Edit****（网格→编辑）从主菜单栏将实体孤网格转换为壳孤网格。**Convert solid to shell**（实体转壳）工具在部件的自由单元面上创建三角形或四边形壳单元；即与其他面不共享所有节点的单元面。如果您的部件包含具有不同单元形状或不同几何阶单元的连接面，Abaqus/CAE 将在接口两侧创建壳单元。同样，如果您的实体网格部件包含引用不同重合节点的相邻单元面（如表示裂纹的节点），Abaqus/CAE 将在接口两侧创建壳单元。

**将实体孤网格转换为壳孤网格：**

1. 进入 Mesh（网格）模块。
2. 从上下文栏的 **Object**（对象）字段中选择 **Part**（部件），然后从列表中选择一个部件。此工具仅适用于处理孤网格单元。
3. 从主菜单栏中选择 ****Mesh**![](../graphics/images/arrow.gif)**Edit****（网格→编辑）。Abaqus/CAE 显示 **Edit Mesh**（编辑网格）对话框。**提示：**您也可以使用位于 Mesh 模块工具箱底部的 ![](../graphics/ico_meshEdit.png) 工具来显示 **Edit Mesh**（编辑网格）对话框。
4. 从 **Category**（类别）字段中选择 **Mesh**（网格）。
5. 从 **Method**（方法）列表中选择 **Convert solid to shell**（实体转壳）。
6. 点击提示区域中出现的 **Yes**（是）按钮。Abaqus/CAE 将实体网格转换为壳网格。**提示：**您可以通过点击 **Edit Mesh**（编辑网格）对话框中的 **Undo**（撤销）来恢复转换前的实体网格状态。

编辑孤网格后，您应始终验证基于节点、单元和曲面的特征（如截面分配、载荷和相互作用），以确保它们正确应用于修改后的网格。
