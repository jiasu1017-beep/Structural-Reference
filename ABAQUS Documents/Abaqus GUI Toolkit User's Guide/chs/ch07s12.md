# 7.12 小结





- 动力学分析包括结构惯性的影响。
- [*FREQUENCY](../key/key-link.md#usb-kws-hfrequency) 过程提取结构的固有频率和振型。
- 然后可以使用模态叠加技术利用振型来确定线性系统的动力学响应。此技术是有效的，但不能用于非线性问题。
- Abaqus/Standard 中提供了线性动力学过程，用于计算瞬态载荷的瞬态响应、谐波载荷的稳态响应、基础运动的峰值响应以及对随机载荷的响应。
- 您应提取足够的模态以获得结构动力学行为的准确表示。运动方向上可能运动的质量的总模态有效质量应至少达到 90%，以产生准确的结果。
- 您可以在 Abaqus/Standard 中定义直接模态阻尼、Rayleigh 阻尼和复合模态阻尼。但是，由于固有频率和振型基于无阻尼结构，因此被分析的结构应该只是轻度阻尼。
- 模态技术不适合非线性动力学模拟。在这种情况下必须使用直接时间积分（[*DYNAMIC](../key/key-link.md#usb-kws-hdynamic)）方法。
- [*AMPLITUDE](../key/key-link.md#usb-kws-mamplitude) 选项允许定义载荷或规定边界条件的任何时间变化。
- 振型和瞬态结果可以在 Abaqus/Viewer 中制作动画。这是理解动力学和非线性静力分析响应的一种有用方法。
