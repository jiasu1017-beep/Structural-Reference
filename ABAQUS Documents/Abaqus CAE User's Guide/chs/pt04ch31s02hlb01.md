# 31.2.8 控制裂缝尖端的奇异性





如果裂缝区域的几何体定义了尖锐裂缝，则应变场在裂缝尖端变得奇异，如["传统有限元方法小应变分析的断裂力学网格构建"中的"轮廓积分评估"，Abaqus Analysis User's Guide第11.4.2节](../usb/usb-link.md#usb-anl-acontintegral-singularity)中所述。在模型中包含小应变分析的奇异性可提高轮廓积分以及应力和应变计算的准确性。

**要控制裂缝尖端的奇异性：**

1. 使用以下方法之一显示**Crack**编辑器：
   - 要配置新的轮廓积分，请按照["创建轮廓积分裂缝"，第31.2.9节](pt04ch31s02hlb02.md)中的说明进行操作。
   - 要编辑现有轮廓积分，请从Interaction模块的主菜单栏中选择****Special**![](../graphics/images/arrow.gif)**Crack**![](../graphics/images/arrow.gif)**Edit**![](../graphics/images/arrow.gif)***裂缝名称*****。
2. 要在轮廓积分估计中包含应变奇异性，请在**Crack**编辑器中单击**Singularity**标签页，并执行以下操作之一：
   - 要为弹性断裂力学应用创建<img src="../graphics/usi_eqn00705.gif>应变奇异性：
     1. 从**Second-order Mesh Options**字段中，为**Midside node parameter**输入值0.25，以将中间节点移动到<img src="../graphics/usi_eqn00706.gif>点。
     2. 从**Degenerate Element Control at Crack Tip/Line**字段中，选择**Collapsed element side, single node**。
   - 要为理想塑性断裂力学应用创建<img src="../graphics/usi_eqn00707.gif>应变奇异性：
     1. 从**Second-order Mesh Options**字段中，为**Midside node parameter**输入值0.5，以将中间节点保留在中间点位置。
     2. 从**Degenerate element control at Crack Tip/Line**字段中，选择**Collapsed element side, duplicate nodes**。
   - 要为幂律硬化材料创建组合的<img src="../graphics/usi_eqn00705.gif>和<img src="../graphics/usi_eqn00707.gif>应变奇异性：
     1. 从**Second-order Mesh Options**字段中，为**Midside node parameter**输入值0.25。
     2. 从**Degenerate Element Control at Crack Tip/Line**字段中，选择**Collapsed element side, duplicate nodes**。
3. 单击**OK**将奇异性包含在轮廓积分估计中并关闭编辑器。
**注意：**在大多数情况下，您必须为裂缝前缘区域分配二阶三角形或楔形单元，以在轮廓积分估计中包含奇异性。如果您分配一阶单元，Abaqus会忽略**Singularity**标签页中的中间节点设置值。但是，如果您分配一阶单元并从**Degenerate Element Control at Crack Tip/Line**字段中选择**Collapsed element side, duplicate nodes**，则将创建<img src="../graphics/usi_eqn00707.gif>应变奇异性。
