# 64.5.5 删除节点





从主菜单栏选择****网格**![](../graphics/images/arrow.gif)**编辑****可以删除孤兒网格节点。Abaqus/CAE还会删除与已删除节点关联的任何单元。如果要删除的节点和关联单元属于现有节点集或单元集，这些集合将相应更新。

**删除节点的步骤：**

1. 进入"网格"模块。
2. 从上下文栏的**对象**字段中选择**零件**，然后从列表中选择一个零件。此工具仅适用于删除孤兒网格节点。
3. 从主菜单栏选择****网格**![](../graphics/images/arrow.gif)**编辑****。Abaqus/CAE显示**编辑网格**对话框。 **提示：**您也可以使用位于"网格"模块工具箱底部的![](../graphics/ico_meshEdit.png)工具显示**编辑网格**对话框。
4. 在对话框中执行以下操作： 1. 在**类别**字段中选择**节点**。 2. 从**方法**列表中选择**删除**。如果未显示**删除**方法，则所选零件中没有孤兒节点。Abaqus/CAE显示现有节点。在隐藏和着色模式下，仅显示可见节点。
5. 在视口中选择要删除的节点。您可以**按住[Shift]键并点击**各个节点将其添加到您的选择中，或**按住[Ctrl]键并点击**所选节点将其从选择中移除。 **提示：**您可以使用**选择**工具栏更改选择行为。有关更多信息，请参见[第6章，"在视口中选择对象"](pt01ch06.md)。如果您希望从现有节点集列表中进行选择，请执行以下操作： 1. 点击提示区域右侧的**集合**。Abaqus/CAE显示**区域选择**对话框，其中包含您已创建的节点集列表。 2. 选择要删除的节点集，然后点击**继续**。 **注意：**默认选择方法基于您最近使用的选择方法。要切换到另一种方法，请点击提示区域右侧的**在视口中选择**或**集合**。Abaqus/CAE显示所选节点以及与所选节点关联的单元。（您选择的节点以及与这些节点关联的任何单元都将被删除。）
6. 完成要删除的节点选择后，点击鼠标按钮2。
7. 与所选节点关联的单元也可能与其他未选节点关联。如果在删除节点和关联单元后，这些未选节点将变得不与任何单元关联，请在提示区域点击**是**，让Abaqus/CAE也删除这些未选节点。 **提示：**如果在删除节点时出错，点击**编辑网格**对话框中的**撤销**可以撤消删除。Abaqus/CAE会恢复已删除的节点。
8. 根据需要重复上述步骤以删除其他节点。
9. 完成节点删除后，点击鼠标按钮2或提示区域中的取消按钮![](../graphics/afxI_cancel.png)退出过程。
