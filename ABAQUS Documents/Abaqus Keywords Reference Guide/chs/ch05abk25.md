# *ENRICHMENT









### *ENRICHMENT指定富集特征及其属性。

此选项用于使用扩展有限元法（XFEM）定义富集特征。富集特征有效地模拟不连续性（如裂纹），而无需使网格符合不连续几何形状。只有实体（连续体）单元可以与富集特征关联。

**产品：**Abaqus/Standard  Abaqus/CAE

**类型：**模型数据

**级别：**部件，部件实例

**Abaqus/CAE：**相互作用模块

##### **参考：**

- ["使用扩展有限元法将不连续性建模为富集特征，" Abaqus分析用户指南第10.7.1节](../usb/usb-link.md#usb-anl-aenrichment)

### **必需参数：**

ELSET

将此参数设置为包含其自由度被特殊函数富集的单元的单元集名称。单元集应包括当前被裂纹穿透的所有单元以及在裂纹穿过模型传播时可能被穿透的单元。

NAME

将此参数设置为将用于在模型中引用富集特征名称的标签。

### **可选参数：**

ENRICHMENT RADIUS

此参数仅在TYPE=STATIONARY CRACK时相关。

将此参数设置为距裂纹尖端的一个小半径，在该半径内使用单元进行裂纹奇异性计算。小半径内的单元应作为由ELSET参数指定的单元集的一部分包含。默认富集半径是富集区域中典型单元特征长度的三倍。

INTERACTION

将此参数设置为与基于小滑动公式的裂纹单元表面接触相互作用关联的[*SURFACE INTERACTION](ch18abk50.md)属性定义名称。

TYPE

设置 TYPE=PROPAGATION CRACK（默认）以基于扩展有限元法沿任意解相关路径模拟离散裂纹扩展。

设置 TYPE=STATIONARY CRACK 以基于扩展有限元方法模拟任意静止裂纹。
