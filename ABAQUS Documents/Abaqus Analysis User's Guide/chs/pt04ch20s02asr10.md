# 20.2.10 aStudy.sample()





### 20.2.10 **_aStudy_.sample()**: 参数化研究的参数采样。

**产品:** Abaqus/Standard  Abaqus/Explicit

此命令用于创建研究参数值的样本。

##### **参考**

- ["参数化研究脚本," 第20.1.1节](pt04ch20s01aus108.md)

### **命令:**

**_aStudy_.sample** (*token*, *additional data*)

### **标记:**

INTERVAL

使用此标记以等间隔对参数进行采样。

NUMBER

使用此标记对参数的给定数量值进行采样。

PRINT

使用此标记打印参数样本。

REFERENCE

使用此标记相对于参考参数值对参数值进行采样。

VALUES

使用此标记对参数的特定值进行采样。

##### **INTERVAL的附加数据:**

### **必需数据:**

**interval**

设置 **interval** 等于采样间隔。对于连续值参数，根据参数的数值以等间隔采样值。对于离散值参数，根据参数值序列的索引以等间隔采样值。

**par**

设置 **par** 等于要打印其样本的参数或参数序列的名称。如果指定单个参数，必须用匹配引号括起来；例如，`'par1'`。如果指定参数序列，必须将其放在括号或方括号内，并且必须包含用匹配引号括起来并用逗号分隔的参数名称；例如，`('par1', 'par2', 'par3')` 或 `['par1', 'par2', 'par3']`。

### **可选数据:**

**domain**

对于连续值参数，设置 **domain** 等于参数的最小值和最大值，用逗号分隔并用括号或方括号括起来；例如，`(10., 20.)` 或 `[10., 20.]`。对于离散值参数，设置 **domain** 等于参数可能具有的值序列，用逗号分隔并用括号或方括号括起来；例如，`(1., 2., 5., 3.)` 或 `[1., 2., 5., 3.]`。

如果在此命令以及 **define** 命令中都为参数指定了 **domain**，则此命令中的 **domain** 规范用于采样。

如果从此命令中省略 **domain**，则必须在 **define** 命令中指定。

##### **NUMBER的附加数据:**

### **必需数据:**

**number**

设置 **number** 等于要采样的等间隔值数量。

**par**

设置 **par** 等于要打印其样本的参数或参数序列的名称。如果指定单个参数，必须用匹配引号括起来；例如，`'par1'`。如果指定参数序列，必须将其放在括号或方括号内，并且必须包含用匹配引号括起来并用逗号分隔的参数名称；例如，`('par1', 'par2', 'par3')` 或 `['par1', 'par2', 'par3']`。

### **可选数据:**

**domain**

对于连续值参数，设置 **domain** 等于参数的最小值和最大值，用逗号分隔并用括号或方括号括起来；例如，`(10., 20.)` 或 `[10., 20.]`。对于离散值参数，设置 **domain** 等于参数可能具有的值序列，用逗号分隔并用括号或方括号括起来；例如，`(1., 2., 5., 3.)` 或 `[1., 2., 5., 3.]`。

如果在此命令以及 **define** 命令中都为参数指定了 **domain**，则此命令中的 **domain** 规范用于采样。

如果从此命令中省略 **domain**，则必须在 **define** 命令中指定。

##### **PRINT的附加数据:**

### **可选数据:**

**par**

设置 **par** 等于要打印其样本的参数或参数序列的名称。如果指定单个参数，必须用匹配引号括起来；例如，`'par1'`。如果指定参数序列，必须将其放在括号或方括号内，并且必须包含用匹配引号括起来并用逗号分隔的参数名称；例如，`('par1', 'par2', 'par3')` 或 `['par1', 'par2', 'par3']`。

如果省略 **par**，则打印参数化研究中所有参数的采样。

##### **REFERENCE的附加数据:**

### **必需数据:**

**interval**

设置 **interval** 等于采样间隔。对于连续值参数，根据参数的数值相对于参考值以等间隔采样。对于离散值参数，根据参数值序列的索引相对于参考值以等间隔采样。

**numSymPairs**

设置 **numSymPairs** 等于要相对于参数参考值对称采样的参数值对数量。

**par**

设置 **par** 等于正在采样的参数名称。此名称必须用匹配引号括起来；例如，`'par1'`。

### **可选数据:**

**domain**

对于连续值参数，设置 **domain** 等于参数的最小值和最大值，用逗号分隔并用括号或方括号括起来；例如，`(10., 20.)` 或 `[10., 20.]`。对于离散值参数，设置 **domain** 等于参数可能具有的值序列，用逗号分隔并用括号或方括号括起来；例如，`(1., 2., 5., 3.)` 或 `[1., 2., 5., 3.]`。

如果在此命令以及 **define** 命令中都为参数指定了 **domain**，则此命令中的 **domain** 规范用于采样。

对于离散值参数，如果从此命令中省略 **domain**，则必须在 **define** 命令中指定。

**reference**

对于连续值参数，设置 **reference** 等于参数的参考值。对于离散值参数，设置 **reference** 等于参数值序列中的索引；索引从零开始，因此序列的第一个值对应索引零，序列的最后一个值对应等于序列中值数量减一的索引。

如果在此命令以及 **define** 命令中都为参数指定了 **reference**，则此命令中的 **reference** 规范用于采样。

如果从此命令中省略 **reference**，则必须在 **define** 命令中指定。

##### **VALUES的附加数据:**

### **必需数据:**

**par**

设置 **par** 等于要打印其样本的参数或参数序列的名称。如果指定单个参数，必须用匹配引号括起来；例如，`'par1'`。如果指定参数序列，必须将其放在括号或方括号内，并且必须包含用匹配引号括起来并用逗号分隔的参数名称；例如，`('par1', 'par2', 'par3')` 或 `['par1', 'par2', 'par3']`。

**values**

设置 **values** 等于构成样本的参数值序列。此序列必须放在括号或方括号内，并且必须包含用逗号分隔的值；例如，`('CAX4', 'CAX4R', 'CAX4H')` 或 `[10., 20., 40.]`。
