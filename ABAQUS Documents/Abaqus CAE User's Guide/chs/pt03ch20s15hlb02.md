# 20.18.2 修复短边缘


使用 Sketcher 工具箱中的**修复短边缘**工具从草图中移除选定的短边缘。Abaqus/CAE 提供两种边缘移除方法:您可以移除您选择的每个边缘,或者您可以选择多个边缘作为移除候选并指定长度公差值。长度小于公差值的边缘会从草图中移除。

如果您通过指定长度公差值来移除短边缘,您应该使用迭代过程执行移除。首先选择要移除的边缘,并在第一次尝试时使用 0.001 的默认长度公差值。如果短边缘没有被移除,请重新选择它们并增加公差。重复此过程,直到短边缘从草图中移除。

您应该仅在移除草图的间隙和重叠之后再修复短边缘。更多信息,请参见["移除间隙和重叠," 第 20.18.1 节](pt03ch20s15hlb01.md)。

**修复草图中短边缘的步骤:**

1. 从 Sketcher 工具箱的修改工具中,选择**修复短边缘**工具 ![](../graphics/ico_repairSmallEdges.png)。有关 Sketcher 工具箱中工具的图示,请参见["Sketcher 工具," 第 20.4.1 节](pt03ch20s04s01.md)。Abaqus/CAE 会在提示区域显示提示,引导您完成操作。
2. 指定要移除的边缘,或者如果您要指定长度公差值,则指定要考虑的移除边缘。您可以 **[Shift]****+单击**或 **[Ctrl]****+单击**单独指定边缘,或者您可以拖动选择来指定草图中某个区域内的所有边缘。Abaqus/CAE 会将所选边缘着色为红色。
3. 如果您在不指定公差的情况下移除短边缘,请在提示区域单击**完成**。Abaqus/CAE 会从草图中移除所选边缘。
4. 如果您要移除短于指定长度公差值的边缘,请在提示区域切换打开**指定公差**并单击**完成**。对于第一次尝试,接受默认公差值并在提示区域单击**完成**。Abaqus/CAE 会从草图中移除所有长度小于长度公差值的所选边缘。
5. 要继续通过长度公差值移除短边缘,请执行以下步骤: 1. 重新选择要考虑的移除边缘,然后单击**完成**。 2. 保持**指定公差**切换打开,再次单击**完成**。 3. 在提示区域增加长度公差值。 4. 在 Abaqus/CAE 窗口任意位置单击鼠标按钮 2。Abaqus/CAE 会从草图中移除所有长度小于长度公差值的所选边缘。 5. 继续选择边缘并增加公差,直到 Abaqus/CAE 移除了您想要从模型中移除的所有短边缘。
6. 完成修复草图中的短边缘后,执行以下操作之一: - 在 Abaqus/CAE 窗口任意位置单击鼠标按钮 2。 - 选择 Sketcher 工具箱中的任何其他工具。 - 单击提示区域中的取消按钮 ![](../graphics/afxI_cancel.png)。
