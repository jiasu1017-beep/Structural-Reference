# 11.4.2 零件类型





创建新零件或从包含第三方格式几何的文件导入零件时，必须选择零件的类型。对于Abaqus/Standard和Abaqus/Explicit，可能的类型有：

**可变形**

您可以创建或导入的任何任意形状的轴对称、二维或三维零件都可以指定为可变形零件。可变形零件表示可以在载荷下变形的零件；载荷可以是机械、热或电的。默认情况下，Abaqus/CAE创建可变形零件。

**离散刚体**

离散刚体零件与可变形零件类似，因为它可以是任意形状。但是，离散刚体零件被认为是刚性的，用于接触分析中建模不能变形的物体。

**分析刚体**

分析刚体零件与离散刚体零件类似，因为它用于表示接触分析中的刚性表面。但是，分析刚体零件的形状不是任意的，必须由一组草图线、圆弧和抛物线形成。

**欧拉**

欧拉零件用于定义材料可以在欧拉分析中流动的域。欧拉零件在分析过程中不会变形；相反，零件内的材料在载荷下变形，可以跨越刚性单元边界流动。有关欧拉分析的更多信息，请参见"欧拉分析"，第28章。

**电磁**

电磁零件类型仅在电磁模型中使用。更多信息，请参见"涡流分析"，Abaqus分析用户指南第6.7.5节。

Abaqus/CFD可用的零件类型为：

**流体**

流体零件类型仅在Abaqus/CFD模型中使用。

您可以在装配模块中组装可变形体、离散刚体零件、分析刚体零件、欧拉零件、流体零件和电磁零件。如果允许，您可以通过在模型树中右键单击零件并从出现的菜单中选择**编辑**来更改已创建零件的类型。

Abaqus/CAE使用以下方法来确定导入零件的类型：
- 当您从包含第三方格式几何的文件导入零件时，可以将零件的类型指定为可变形、离散刚体或欧拉。
- 当您从输出数据库导入网格时，Abaqus/CAE根据输出数据库中存储的信息确定新零件的类型。
- 当您从输入文件导入网格时，Abaqus/CAE根据单元类型确定新零件的类型。
- 当您在网格模块中创建网格零件时，网格零件的类型与原始零件的类型相同。
