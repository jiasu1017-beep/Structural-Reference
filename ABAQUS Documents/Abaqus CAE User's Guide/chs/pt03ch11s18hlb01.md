# 11.18.2 创建新零件





从主菜单栏选择**零件**![](../graphics/images/arrow.gif)**创建**以在当前视口中创建新零件。

模型可以包含多个零件；每个零件存在于局部坐标系中，您使用装配模块来创建零件实例并相对于全局坐标系定位这些实例。创建零件时，您需要命名零件并选择其类型、建模空间、基础特征和近似尺寸；然后绘制零件基础特征的轮廓。

**创建新零件：**

1. 从主菜单栏，选择**零件**![](../graphics/images/arrow.gif)**创建**。出现**创建零件**对话框。更多信息，请参见"使用**创建零件**对话框"，第11.19节。
2. 输入零件的名称。
3. 选择新零件的建模空间、类型、基础特征和近似尺寸。更多信息，请参见"Abaqus/CAE中如何定义零件？"，第11.4节。
4. 单击**继续**关闭**创建零件**对话框。草图编辑器启动，**草图**网格出现在当前视口中。
5. 使用草图编辑器绘制基础特征的二维轮廓。更多信息，请参见第20章"草图模块"。
6. 完成基础特征的绘制后，单击鼠标按钮2退出当前**草图**工具。
7. 在提示区域，单击**完成**退出草图编辑器。
8. 如有必要，使用零件模块工具向基础特征添加其他特征。更多信息，请参见"什么是基于特征的建模？"，第11.3节。
