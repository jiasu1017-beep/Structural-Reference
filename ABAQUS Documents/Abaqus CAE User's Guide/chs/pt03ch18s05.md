# 18.5 查看和排除优化故障





您可以使用由Abaqus生成的场和历史输出查看优化过程生成的结果。 您也可以使用输出来诊断优化中的任何问题；例如，确定优化是否正在收敛到目标或研究收敛速率。 您可以通过单击**Optimization Process Manager**中的**Results**查看结果。

当您提交优化过程进行分析时，Abaqus为优化的每个设计周期创建一个输出数据库（`.odb`)文件。 输出数据库文件存储在`*jobname*\SAVE.odb`目录中。 在Visualization模块中查看优化之前，您必须将单独的输出数据库文件合并为单个输出数据库文件。 Visualization模块的行为取决于输出数据库文件是从拓扑优化、形状优化还是尺寸优化创建的。

**拓扑优化**

当您查看拓扑优化的结果时，Abaqus/CAE会自动在当前视图上显示一个视图切割，该切割表示优化设计表面。 视图切割的等值面变量是Optimization模块用于从分析中"添加"或"移除"单元的归一化材料属性。 默认情况下，Abaqus/CAE以归一化材料属性设置为0.3显示视图切割。 您可以使用视图切割管理器修改等值面变量的值并查看等值面的结果边界。 当优化视图切割处于活动状态时，不显示边界条件。 有关更多信息，请参阅["管理视图切割，" 第80.2节](pt07ch80s02.md)。

**形状优化**

当您查看形状优化的结果时，Abaqus/CAE使用表面节点的新位置以其优化形状显示模型。

**尺寸优化**

当您查看尺寸优化的结果时，Abaqus/CAE显示壳模型和优化壳厚度，随着优化的进行，壳厚度会变化。 （当您查看壳模型的非优化分析结果时，壳厚度从模型数据中读取，在分析过程中不会变化。）

将单独的输出数据库文件合并为单个输出数据库文件后，优化过程中的每个设计周期都会作为一帧出现在输出数据库中，您可以打开**Step/Frame**对话框显示每个设计周期的结果。 您可以在优化过程仍在进行时查看优化过程的结果。 当优化过程趋于完成时，每次关闭并重新打开**Step/Frame**对话框时，都会更新已完成步骤和帧的列表。 有关更多信息，请参阅["选择特定结果步骤和帧，" 第42.3.1节](pt05ch42s01hlb01.md)和["逐步浏览帧，" 第42.3.2节](pt05ch42s01hlb02.md)。

您可以对变形或轮廓图执行时间历史动画，并查看Abaqus在尝试满足目标函数同时遵守约束时优化的进度。 对于拓扑优化，您可以查看从分析中逐步移除单元的情况以及对模型机械行为的影响，例如变形或应力的变化。 开启半透明度允许您通过内部单元查看优化的进度；例如，移除内部单元以创建空隙。 有关更多信息，请参阅["更改半透明度，" 第77.3节](pt07ch77s03.md)。 对于形状优化，您可以查看表面节点位置随优化进行时的增量变化；并且与拓扑优化类似，您可以查看对模型机械行为的影响。

此外，优化过程将数据文件（`optimization_report.csv`和`optimization_status_all.csv`）写入`*jobname*`目录，您可以用它来跟踪设计变量。 例如，您可以创建一个X-Y图，显示每个设计周期后目标函数和约束的变化。 结果图指示Optimization模块如何在满足约束的同时努力满足指定的目标函数。 您可以使用目标函数和约束的X-Y图作为诊断工具，查看每个设计周期后优化的进度，并确定优化是否正在收敛到解，如图 18-2](pt03ch18s05.md#opz-designandconstraint)所示。

您还可以通过从优化过程管理器监控优化过程的进度来查看显示每个设计周期后目标函数和约束如何变化的X-Y图。 有关更多信息，请参阅["监控您的优化过程，" 第19.12.6节](pt03ch19s12hlb06.md)。

Optimization模块还在`*jobname*\SAVE.odb`目录中保存输出数据库。 这些输出数据库包含Abaqus分析在每个设计周期期间生成的数据； 但是，它们不包含与优化相关的数据，例如归一化材料属性的值。
