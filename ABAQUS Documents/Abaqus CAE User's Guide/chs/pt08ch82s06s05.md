# 82.6.5 显示模态贡献因子





第一个插件分析数据并将结果写回输出数据库后，您可以查看模态贡献因子。通过在 Visualization 模块中从主菜单栏选择 ****Plug-ins**![](../graphics/images/arrow.gif)**NVH**![](../graphics/images/arrow.gif)**Plot modal Contribution Factors**** 来启动第二个插件。插件显示 **Open ODB** 对话框，您输入要从中读取的输出数据库的名称。然后插件显示 **Select Steps** 对话框，如 [图82-2](pt08ch82s06s05.md#usi-plg-selectsteps) 所示。

**图82-2** **Select Steps** 对话框。

![](../graphics/usi-plg-selectsteps.png)

您必须选择 **Valid Frequency Step** 和 **Valid SSD Step** 来从中生成绘图。此外，您必须选择以下变量类型之一进行绘图：
- MCF 的幅值
- MCF 的相位
- 求和的幅值
- 求和的相位
- 排序

求和的幅值或相位表示所有模态的模态贡献因子之和。这与 Abaqus 输出变量（POR、U、V 或 A）相同。

当您点击 **Continue** 时，插件显示 **Plot MCF** 对话框。点击 **Frequency Step/SSD Step** 按钮可以选择不同的频率和稳态动态步长。

点击 **Selection** 选项卡选择以下内容：
- 响应点。
- 感兴趣的频率，如果要绘制条形图、极坐标图或矢量图。
- 任意数量的要绘制的 **Y** 分量。点击 **>** 添加 **Y** 分量；点击 **<** 删除 **Y** 分量。分别点击 **>>** 或 **<<** 按钮可以添加或删除所有 **Y** 分量。

**Selection** 选项卡页面如 [图82-3](pt08ch82s06s05.md#usi-plg-frerquencystp) 所示。

**图82-3** **Selection** 选项卡页面。

![](../graphics/usi-plg-frequencystp.png)

点击 **Options** 选项卡选择常规绘图选项。此外，如果选择基于 POR 的结果，您可以为 XY 和投影绘图选择 **P Reference** 值。从 **Plot MCF** 对话框右侧的按钮中，选择要创建的绘图类型。
- 点击 **XY Plot** 生成 MCF 幅值或相位的频谱。对于模态排序选择，插件会为相应排序的模态创建 *X–Y* 绘图。
- 点击 **Projection Plot** 将选定的 MCF 绘制到整个频谱上相应的总声压级投影上。您可以使用此绘图来查看模态对总声压级的贡献（正或负）。
- 点击 **Bar Graph** 绘制每个模态的 MCF 条形图。
- 点击 **Polar Graph** 绘制每个模态的 MCF 极坐标图。极坐标图在复平面上显示每个单独 MCF 的幅值和相位信息。当与总声压矢量一起绘制时，您可以轻松识别对总声压贡献最大的模态。
- 点击 **Vector Graph** 绘制每个模态的 MCF 矢量图。每个模态的矢量按顺序绘制。随着绘制的模态越来越多，矢量图趋向于总声压矢量的幅值。

**Options** 选项卡页面如 [图82-4](pt08ch82s06s05.md#usi-plg-plotmcf) 所示。

**图82-4** **Options** 选项卡页面。

![](../graphics/usi-plg-plotmcf.png)

插件在单独的视口中创建每种类型的绘图，并相应地命名视口。因此，您可以选择 ****Viewport**![](../graphics/images/arrow.gif)**Tile Vertically**** 从主菜单栏同时查看所有绘图。以下图形是使用 ["皮卡车耦合声固分析，" Abaqus Example Problems Guide 第9.1.7节](../exa/exa-link.md#exa-aco-acouststructpickup) 生成的输出数据库由插件创建的。选择的变量为 POR，条形图、极坐标图、矢量图和排序是在 35 Hz 频率下生成的。分析检查了前五个排序模态（共180个）：

[图82-5](pt08ch82s06s05.md#usi-plg-plot) 显示了分析的模型。条形图表明 Mode 36 在该频率下是主要模态。

**图82-5** 皮卡车间耦合声固分析的 MCF 条形图。

![](../graphics/usi-plg-plot.png)

[图82-6](pt08ch82s06s05.md#usi-plg-xyplot) 显示了 *X–Y* 绘图和投影绘图。*X–Y* 绘图显示所有频率下模态贡献因子的幅值。投影绘图显示所有频率下模态贡献因子对总响应的投影。例如，模态169在 110 Hz 时变得更加重要。

**图82-6** 皮卡车间耦合声固分析的 MCF *X–Y* 和投影绘图。

![](../graphics/usi-plg-xyplot.png)

[图82-7](pt08ch82s06s05.md#usi-plg-polar) 显示了极坐标图和矢量图。条形图显示模态36的幅值小于模态8的幅值；然而，极坐标图显示模态36与总响应（所有模态贡献的矢量和）更加同相位。因此，模态36在 35 Hz 时对总幅值的贡献大于模态8，因此排名更高。

**图82-7** 皮卡车间耦合声固分析的 MCF 极坐标图和矢量图。

![](../graphics/usi-plg-polar.png)
