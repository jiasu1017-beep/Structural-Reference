# 75.3 我可以对包含虚拟拓扑的零件或零件实例做什么？





Abaqus/CAE 将虚拟拓扑存储为相应零件或装配的特征。因此，您可以使用模型树来重命名、抑制、恢复和删除虚拟拓扑特征。您不能编辑虚拟拓扑特征。您可以恢复被忽略或通过虚拟拓扑合并的实体。恢复实体会创建更多新的虚拟拓扑特征在模型树中——原始虚拟拓扑特征保持不变。

您可以创建包含虚拟面和边的集合和曲面。此外，如果现有集合或曲面包含您忽略的边或顶点，则该边或顶点将从集合或曲面中移除。同样，如果现有集合或曲面包含您合并的面或边，Abaqus/CAE 将用新的合并面和边替换原始面和边。只有当您合并的所有面和边属于同一集合时，上述情况才适用。

虚拟拓扑不会被导出；例如，如果将包含虚拟拓扑的装配导出到 ACIS 文件，Abaqus/CAE 会从导出的模型中移除虚拟拓扑。当您在装配模块中创建装配时，不能合并或剪切包含虚拟拓扑的零件实例。




# 75.4 如果我可以使用虚拟拓扑，为什么还要修复零件？





引入虚拟拓扑是一种创建干净、良好网格的便捷方法。重点是忽略小的几何细节，这些细节会在分析结果不太重要的区域产生扭曲的单元。虚拟拓扑放宽了网格必须符合每条边或每个顶点的要求，让您能够获得更简单、细节更少的模型。但是，虚拟拓扑不会更改零件实例的底层几何。

当您合并两个面形成虚拟面时，Abaqus/CAE 会更新显示以反映被忽略的边。然而，为了形成新虚拟面的基础，Abaqus/CAE 维护着被合并面的底层几何。网格中的节点将放置在底层几何上。因此，粗糙网格将在虚拟面上进行插值，而细密网格将趋向于收敛到被合并面的底层几何。虚拟拓扑允许通过网格参数（如种子大小）来控制网格定义，它有助于将网格划分从几何约束（如小面）中解放出来。

相比之下，您使用几何编辑工具集来纠正通常在从第三方应用程序导入零件时引入的小缺陷。您使用几何编辑工具集来纠正底层几何；例如，缝合小缝隙并删除自相交的面并重新创建有效的替代面。您还使用几何编辑工具集来移除冗余拓扑，例如多余的顶点和边。几何编辑工具集会修改实际几何，几何限制会影响您应用修复工具的方式；例如，只有当顶点位于两条边共用的曲线上时，才能移除冗余顶点。

当您导出虚拟零件实例时，虚拟拓扑会丢失，因为虚拟面、边或顶点不是真实的几何。例如，虚拟面是对被合并面的底层集合的抽象引用。因此，如果需要从零件中移除小细节并希望以保留更改的方式导出零件，应使用几何编辑工具集。
