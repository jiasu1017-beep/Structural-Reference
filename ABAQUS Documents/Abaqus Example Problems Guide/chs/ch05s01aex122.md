# 5.1.6 反应堆压力容器螺栓封闭的热应力分析

**产品：** Abaqus/Standard  Abaqus/CAE   

### 目标

本示例演示了以下 Abaqus 功能和技巧，用于传热和静态应力分析：
- 在特定分析步骤中在模型的不同区域指定自适应重网格规则；
- 基于指定的重网格规则使用自动化流程自适应重网格模型；以及
- 查看误差指示器结果作为评估网格质量的一种手段。

### 应用描述

本示例检查核反应堆容器组件螺栓封闭区域的 thermal 和 stress 行为。容器组件形成围绕燃料芯的压力边界。本示例考虑承受以下载荷条件的能力：
- 螺栓中的预拉伸载荷，
- 恒定内部压力，以及
- 规定的加热/冷却速率。

### 几何

问题域包括圆柱形容器壳体、半球形底部封头、圆顶形封闭头以及封闭和密封组件。整体高度为 7650 mm。底部封头内半径为 1410 mm，厚度为 157 mm。容器壳体内半径为 1380 mm，厚度为 213 mm。封闭头高度为 2330 mm，内半径为 1310 mm，厚度为 210 mm。

### 边界条件和载荷

容器外部暴露在恒定温度为 21 C 的空气中。容器内部充满工作温度为 320 C 的热水。在冷却过程中，内部温度在两小时内降低 38 C。内部水对容器内表面施加 1.38 x 10^7 Pa 的恒定压力。

### Abaqus 建模方法和模拟技术

分析的目的是理解 vessel-to-head 接口附近的应力。顺序耦合热应力分析在反应堆容器上执行。热分析首先获得温度场分布，然后通过使用热分析结果指定温度场进行静态应力分析来获得容器的力学响应。

### 热传递分析

示例首先执行热传递分析以获得压力容器在热载荷下的温度分布。分析包括稳态热传递步骤和瞬态热传递步骤。

### 结构分析

温度分布计算将增强螺栓和压力载荷以定义容器组件的结构载荷。一系列静态步骤用于模拟模型在热和力载荷下的力学响应。

### 结果与讨论

沿 vessel-to-head 接口特定路径的 Mises 应力幅度。热和结构案例是互补的；结构案例依赖于热案例。
