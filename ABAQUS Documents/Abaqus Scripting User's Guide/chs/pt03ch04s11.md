# 4.11 布局示例







以下示例使用默认布局提示一次创建一个按钮。每个按钮创建时，这些图显示了布局空腔中剩余空间的效果。

**示例 1**

第一个示例首先在空腔左侧创建一个按钮。垂直位置的默认值是 LAYOUT_TOP，因此该示例将按钮放置在可用空间的左侧和顶部。

```
gb = FXGroupBox(parent, '')
FXButton(gb, 'Button 1', opts=LAYOUT_SIDE_LEFT|BUTTON_NORMAL)
```

**图 4–8** 在布局空腔左侧和顶部创建按钮。

![](../graphics/wgt-layout-hints-1button-nls.png)

以下语句在可用空间左侧顶部添加第二个按钮：
```
FXButton(gb, 'Button 2', opts=LAYOUT_SIDE_LEFT|BUTTON_NORMAL) 
```

**图 4–9** 在布局空腔左侧顶部添加第二个按钮。

![](../graphics/wgt-layout-hints-2button-nls.png)

以下语句在可用空间左侧顶部添加第三个按钮：
```
FXButton(gb, 'Button 3',
    opts=LAYOUT_SIDE_LEFT|BUTTON_NORMAL) 
```

**图 4–10** 在布局空腔左侧顶部添加第三个按钮。

![](../graphics/wgt-layout-hints-3button-nls.png)

[图 4-11](pt03ch04s11.md#wgt-layout-hints-3button-nospace) 显示了三个按钮的最终配置。

**图 4–11** 按钮的最终配置。

![](../graphics/wgt-layout-hints-3button-nospace-nls.png)

**示例 2**

第二个示例说明如何使用非默认布局提示。该示例首先使用默认提示将按钮放置在可用空间的顶部和左侧。

```
gb = FXGroupBox(p,'') 
FXButton(gb,'Button 1')
```

**图 4–12** 在布局空腔左侧和顶部创建按钮。

![](../graphics/wgt-layout-hints-1buttonabove-nls.png)

然后该示例将第二个按钮放置在布局空腔底部的右侧。
```
FXButton(gb, 'Button 2',    
    opts=LAYOUT_SIDE_BOTTOM|LAYOUT_RIGHT|BUTTON_NORMAL) 
```

**图 4–13** 在布局空腔右侧底部添加第二个按钮。

![](../graphics/wgt-layout-hints-2buttonabove-nls.png)

最后，该示例将第三个按钮放置在可用空间的底部，并在 *X* 方向上居中。
```
FXButton(gb, 'Button 3',    
    opts=LAYOUT_SIDE_BOTTOM|LAYOUT_CENTER_X|BUTTON_NORMAL)
```

**图 4–14** 在布局空腔底部居中添加第三个按钮。

![](../graphics/wgt-layout-hints-3buttonabove-nls.png)

[图 4-15](pt03ch04s11.md#wgt-layout-hints-3button-above-nospace) 显示了三个按钮的最终配置。

**图 4–15** 三个按钮的最终配置。

![](../graphics/wgt-layout-hints-3button-above-nospace-nls.png)
