# 14.4.2 场输出和历史输出有什么区别？





创建输出请求时，可以选择场输出或历史输出。

**场输出**

Abaqus从分布在整个模型或部分模型上的空间数据生成场输出。在大多数情况下，使用可视化模块通过变形形状、等高线或符号图来查看场输出数据。Abaqus在分析期间生成的场输出量通常很大。因此，通常会请求Abaqus以较低的速率将场数据写入输出数据库；例如，在每个步骤后或分析结束时。

创建场输出请求时，可以指定以等间距时间间隔或每次经过特定时间段来输出。对于Abaqus/Standard分析过程，你也可以指定以增量为单位的输出频率，请求在每个步骤的最后增量后输出，或根据一组时间点请求输出。对于Abaqus/Explicit分析过程，你也可以请求每个时间增量的场输出或根据一组时间点请求输出。对于Abaqus/CFD分析过程，你也可以指定以增量为单位的输出频率。

创建场输出请求时，Abaqus会将所选变量的每个分量写入输出数据库。例如，如果你使用实体单元对端部受载的悬臂梁进行建模，可以请求在加载步骤的最后增量后从整个模型中获取应力（所有六个分量）和位移（所有六个分量）数据。然后你可以使用可视化模块查看最终加载状态下应力和挠度的等高线图。

**历史输出**

Abaqus从模型中特定点的数据生成历史输出。在大多数情况下，使用可视化模块通过*X-Y*图显示历史输出。输出速率取决于你想如何使用分析生成的数据，速率可能非常高。例如，为诊断目的生成的数据可能在每个增量后写入输出数据库。你也可以将历史输出用于与整个模型或模型部分相关的数据；例如，整个模型能量。

创建历史输出请求时，可以指定以等间距时间间隔或每次经过特定时间段来输出。对于Abaqus/Standard分析过程，你也可以指定以增量为单位的输出频率，请求在每个步骤的最后增量后输出，或根据一组时间点请求输出。对于Abaqus/Explicit分析过程，你也可以请求以时间增量输出历史输出。对于Abaqus/CFD分析过程，你也可以指定以增量为单位的输出频率。

创建历史输出请求时，可以指定Abaqus/CAE将写入输出数据库的变量的各个分量。例如，如果你对端部受载的悬臂梁的响应进行建模，可能会在加载步骤的每个增量后请求以下输出：
- 梁根部单个节点的主应力。
- 梁端部单个节点的垂直位移。

然后你可以使用可视化模块查看随着载荷增加根部应力与端部位移的*X-Y*图。
