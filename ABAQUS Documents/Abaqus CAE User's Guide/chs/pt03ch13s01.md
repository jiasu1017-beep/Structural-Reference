# 13.1 理解装配模块的角色







创建零件时，它存在于自己的坐标系中，独立于模型中的其他零件。相比之下，您使用装配模块来创建零件实例并在全局坐标系中相对于彼此定位各实例，从而创建装配。您通过顺序应用位置约束来定位零件实例，这些约束对齐所选的面、边缘或顶点，或者通过应用简单的平移和旋转。

您还可以在主模型中创建其他模型的实例，从而能够添加完整的子装配以及单个零件。模型实例的创建方式与零件实例完全相同，并且可以以类似的方式进行定位和操作。

实例维护与其原始零件或模型的关联。如果零件或模型的几何发生变化，Abaqus/CAE会自动更新该零件或模型的所有实例以反映这些变化。您不能直接编辑实例的几何。

主模型可以包含许多零件和模型子装配，并且一个零件或模型可以在主模型装配中被实例化多次；但是，一个模型只包含一个顶层装配。载荷、边界条件、预定义场和网格都被应用于完整装配。即使您的模型仅由单个零件组成，您仍然必须创建一个仅包含该零件单个实例的装配。

零件实例可以被视为原始零件的表示。您可以创建从属或独立零件实例。独立实例实际上是零件的副本。从属实例仅是指向零件、分区或虚拟拓扑的指针；因此，您不能对从属实例进行网格划分。但是，您可以对从中派生实例的原始零件进行网格划分，在这种情况下，Abaqus/CAE会将相同的网格应用到该零件的每个从属实例。

模型实例始终是从属的，而不是独立的。
