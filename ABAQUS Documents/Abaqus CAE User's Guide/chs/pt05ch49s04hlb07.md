# 49.4.7 自定义同步视口的时间历史





时间历史设置控制当您显示时间历史动画时Abaqus/CAE使用基于帧的历史还是基于时间的历史。帧同步允许所有同步视口一次一帧地一起递增，而时间同步允许视口根据通用时间线显示数据。

当您选择基于时间的动画时，您还可以自定义通用时间线的三个方面：
- 时间步长，控制每次递增数据显示时经过多少时间。
- 最小时间，这是包含数据的最早时间点。如果自动计算最小时间，Abaqus/CAE从通用时间线的开始处开始包含数据。
- 最大时间，这是包含数据的最晚时间点。如果自动计算最大时间，Abaqus/CAE包含数据直到通用时间线的结束。

**自定义数据的时间历史：**

1. 找到"时间历史"选项。在提示区域中，单击"动画选项"；然后在出现的对话框中单击"时间历史"选项卡。
2. 从页面顶部的选项中，选择"基于帧"或"基于时间"。
3. 如果需要，更改"时间增量"值。时间增量仅适用于基于时间的动画。
4. 如果需要，通过在适当的"指定"字段中输入值来自定义"最小时间"和"最大时间"。这些值仅适用于基于时间的动画。
5. 单击"应用"。
