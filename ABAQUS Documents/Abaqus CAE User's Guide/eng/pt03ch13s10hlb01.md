# 13.10.1 Using the Instance menu







### 13.10.1 Using the **Instance** menu

Use the **Instance** menu to do the following:
- [Create](pt03ch13s10hlb04.md) instances of parts from the current model and add them to the assembly. You can also create instances of other models to add to the current assembly.
- Create a [linear pattern](pt03ch13s10hlb05.md) of part instances.
- Create a [radial pattern](pt03ch13s10hlb06.md) of part instances.
- [Translate](pt03ch13s10hlb07.md) selected part or model instances along a specified vector.
- [Translate](pt03ch13s10hlb08.md) selected part or model instances along a specified vector until they are a specified distance from another instance.
- [Rotate](pt03ch13s10hlb09.md) selected part or model instances through a specified angle about a specified axis.
- [Replace](pt03ch13s10hlb10.md) a part instance with a second part instance.
- [Convert](pt03ch13s10hlb11.md) any position constraints to an absolute position.
- [Merge or cut](pt03ch13s10hlb12.md) part instances.

You may find it more convenient to access the instance tools using the Assembly module toolbox.
