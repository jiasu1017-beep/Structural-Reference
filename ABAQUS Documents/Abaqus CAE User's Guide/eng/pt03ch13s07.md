# 13.7 Performing Boolean operations on part instances







You can select instances of parts that you created using Abaqus/CAE and merge them into a single instance. In addition, you can cut away the geometric portion of a part instance using the geometric portion of other part instances to make the cut. You can also merge instances of parts containing both geometry and orphan elements. This section describes how you merge and cut part instances. The following topics are discussed:
- ["Merging and cutting part instances," Section 13.7.1](pt03ch13s07s01.md)
- ["Merging and cutting independent and dependent part instances," Section 13.7.2](pt03ch13s07s02.md)
