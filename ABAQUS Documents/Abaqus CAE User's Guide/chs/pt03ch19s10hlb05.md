# 19.10.6 控制自适应过程精度







要找到自适应过程编辑器，请从主菜单栏中选择****自适应**![](../graphics/images/arrow.gif)**编辑**![](../graphics/images/arrow.gif)***自适应过程名称*****。使用**精度**选项卡页面控制节点输出的精度。您可以选择单精度或完整节点输出精度。此选项等同于 Abaqus 执行过程中的 `output_precision` 选项。有关更多信息，请参见["Abaqus/Standard、Abaqus/Explicit 和 Abaqus/CFD 执行，"《Abaqus 分析用户手册》第 3.2.2 节](../usb/usb-link.md#usb-int-danalysisproc)。
