# 12.13.13 创建均匀流体截面





流体截面用于在Abaqus/CFD模型中定义流体材料的截面属性。请参见[第30章，"流体动力学分析"](pt04ch30.md)。

**指定均匀流体截面属性：**

1. 从主菜单栏，选择****Section**![](../graphics/images/arrow.gif)**Create****。出现**Create Section**对话框。
   **提示：**您也可以在**Section Manager**中点击**Create**，或在Property模块工具箱中选择创建截面工具![](../graphics/ico_section.png)。
2. 输入截面名称。有关命名对象的更多信息，请参见["使用基本对话框组件，"第3.2.1节"](pt01ch03s02s01.md)。
3. 选择**Fluid**作为截面**Category**，选择**Homogeneous**作为截面**Type**，然后点击**Continue**。出现流体截面编辑器。
4. 为流体截面选择一种材料。如有需要，点击![](../graphics/ico_material.png)创建材料；有关更多信息，请参见["创建或编辑材料，"第12.7.1节"](pt03ch12s07hlb01.md)。
5. 点击**OK**保存更改并关闭流体截面编辑器。
