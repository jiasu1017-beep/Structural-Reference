# 20.22.2 创建独立草图


独立草图存储在您的模型中,独立于任何特定零件或装配。要在当前视口中创建新的独立草图,请从主菜单栏选择****Sketch**![](../graphics/images/arrow.gif)**Create****。

**创建独立草图的步骤:**

1. 从主菜单栏,选择****Sketch**![](../graphics/images/arrow.gif)**Create****。**创建草图**对话框会出现。**提示:**您也可以通过单击 Sketch 模块工具箱中的草图创建工具 ![](../graphics/ico_sketch.png) 来创建独立草图。
2. 在**创建草图**对话框中,输入草图的名称。有关有效名称的信息,请参见["使用基本对话框组件," 第 3.2.1 节](pt01ch03s02s01.md)。
3. 在**创建草图**对话框中,输入草图的大致尺寸。您输入的尺寸用于 Abaqus/CAE 计算图纸的尺寸及其网格的间距。大致尺寸应反映草图的最大尺寸,必须介于 105 和 103 单位之间。Abaqus/CAE 不要求特定单位,但整个模型中的单位必须一致。
4. 单击**继续**关闭**创建草图**对话框。Abaqus/CAE 会启动 Sketcher 并显示一个方形图纸,您可以在上面绘制;图纸的宽度和高度大约等于您在上一步中输入的值。如果您之后发现草图超出了图纸边缘,可以使用 Sketcher [自定义选项](pt03ch20s09.md)更改图纸尺寸。
5. 完成绘制后,在提示区域单击**完成**。
