- ["解析刚性表面定义，" 第2.3.4节](pt01ch02s03aus19.md)
- ["刚性表面接触单元，" 第40.5.1节](pt09ch40s05alm67.md)
- [*RIGID SURFACE](../key/key-link.md#usb-kws-mrigidsurf)
- [*INTERFACE](../key/key-link.md#usb-kws-minterface)

### 概述

本节提供Abaqus/Standard中可用的轴对称刚性表面接触单元的参考。

### 单元类型

| IRS21A | 用于一阶轴对称单元的轴对称刚性表面接触单元 |
| --- | --- |

| IRS22A | 用于二阶轴对称单元的轴对称刚性表面接触单元 |
| --- | --- |

**活动自由度**

除最后一个节点外，每个节点处为1、2

最后一个节点处为1、2、6（刚性体参考节点的运动）

**额外求解变量**

每个节点有两个与接触应力相关的额外变量。

**所需节点坐标**

*r*、*z*

**单元属性定义**

| **输入文件用法：** | 使用以下选项定义单元所交互的表面： |
| --- | --- |
| | ``` [*RIGID SURFACE](../key/key-link.md#usb-kws-mrigidsurf) ``` 使用以下选项定义刚性表面单元的截面属性： ``` [*INTERFACE](../key/key-link.md#usb-kws-minterface) ``` |

**基于单元的加载**

无。

**单元输出**

| S11 | 单元与沿刚性表面法向方向的刚性表面之间的压力。 |
| --- | --- |

| S12 | 单元与沿刚性表面切向方向的刚性表面之间的应力剪切分量。 |
| --- | --- |

| E11 | 在最接近单元积分点的表面上，沿垂直于刚性表面法向方向的表面分离。 |
| --- | --- |

| E12 | 表面的累积相对切向位移。 |
| --- | --- |

**单元上的节点排序**

IRS21A的前两个节点和IRS22A的前三个节点位于变形mesh上。最后一个节点是定义刚性体运动的刚性体参考节点。

**输出积分点编号**

积分点位于位于变形模型表面上的节点处，并相应地进行编号。
