# 23.4 理解复合铺层和分布






离散场是一种空间变化的场，其值与节点或单元相关联。离散场工具集允许您在 Abaqus/CAE 中创建和管理离散场。由复合铺层使用的离散场称为分布。由于分布应用于特定的单元和节点，因此您只能在完成零件网格划分后才能使用分布。在大多数情况下，您将使用对网格化零件进行操作的第三方预处理器来创建可应用于复合铺层的分布。有关更多信息，请参见[第63章，"离散场工具集"](pt06ch63.md)。

您可以使用分布执行以下操作：
- 定义空间变化的局部坐标系，该坐标系指定复合铺层的整体方向。
- 定义铺层方向的额外旋转，该旋转在铺层上空间变化。
- 定义层方向的额外旋转，该旋转在层上空间变化。您只能在 Abaqus/Standard 分析中使用分布来定义层方向。
- 定义在常规壳复合铺层上空间变化的整体壳厚度。
- 定义在常规壳复合铺层中层上空间变化的层厚度。您只能在 Abaqus/Standard 分析中使用分布来定义层厚度。
- 定义在常规壳复合铺层上空间变化的节点厚度。
- 定义在常规壳复合铺层上空间变化的偏移。您只能在 Abaqus/Standard 分析中使用分布来定义偏移。
