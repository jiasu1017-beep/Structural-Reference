### 概述

线性方程求解用于线性和非线性分析。在非线性分析中，Abaqus/Standard使用Newton法或其变体（如Riks法），在其中需要在每次迭代时求解一组线性方程。直接线性方程求解器找到该线性方程组的精确解（达到机器精度）。Abaqus/Standard中的直接线性方程求解器：
- 使用稀疏、直接的高斯消元法；并且
- 通常代表分析中最耗时的部分（尤其是大型模型）——方程的存储在计算过程中占用磁盘空间的最大部分。

### 稀疏求解器

直接稀疏求解器使用"多波前"技术，如果方程系统具有稀疏结构，可以显著减少求解方程的计算时间。这种矩阵结构通常出现在物理模型由多个连接在一起的部件或分支组成时；辐条轮是具有稀疏刚度矩阵的结构的一个很好的例子。使用梁、桁架和壳建模的空间框架和其他结构通常具有稀疏刚度矩阵。相比之下，*块状*结构——如单个实体三维块（参见["具有部分贯穿轴向裂纹的有限长度圆筒的弹塑性线弹簧建模，" Abaqus Example Problems Guide第1.4.3节](../exa/exa-link.md#exa-sta-inelasticlinespring)）——几乎没有机会让稀疏求解器减少计算时间。对于大型块状结构，迭代线性方程求解器可能更高效（参见["迭代线性方程求解器，" 6.1.6节](pt03ch06s01aus47.md)）。

| **输入文件用法：** | 使用以下选项使用默认直接稀疏求解器： |
| --- | --- |
|  | ``` [*STEP](../key/key-link.md#usb-kws-hstep) ``` |

| **Abaqus/CAE用法：** | Step模块：step编辑器：**Other**：**Method: Direct** |
| --- | --- |

#### 设置直接线性求解器的控制

线性方程求解器可以优化与硬接触和混合单元相关的约束方程的消除。此选项有两个潜在的不良副作用：
- 可能的小幅降低解精度可能对非线性收敛行为产生不利影响。
- 对于没有硬接触约束和/或混合单元的模型，可能存在轻微的性能下降。

| **输入文件用法：** | 使用以下选项开启约束优化： |
| --- | --- |
|  | ``` [*SOLVER CONTROLS](../key/key-link.md#usb-kws-hsolvercontrols), CONSTRAINT OPTIMIZATION ``` |

| **Abaqus/CAE用法：** | 您不能在Abaqus/CAE中指定约束优化。 |
| --- | --- |
