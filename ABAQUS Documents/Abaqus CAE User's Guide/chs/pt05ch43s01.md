# 43.1 理解未变形形状绘图





Abaqus/CAE从输出数据库获取生成未变形绘图所需的信息。在打开输出数据库后立即，Abaqus/CAE显示模型的未变形绘图状态。您可以使用不依赖于绘图状态的选项来自定义您的绘图。选择主菜单栏中的**选项**![](../graphics/images/arrow.gif)**通用**来选择不依赖于绘图状态的自定义选项。

默认情况下，Abaqus/CAE在输出数据库中最后一步的最后一帧显示您的未变形模型。如果您的模型在步骤之间发生变化，可能是因为添加或删除了接触表面或施加了载荷或边界条件，您可能希望在默认步骤以外的步骤查看您的未变形模型。您可以通过从主菜单栏中选择**结果**![](../graphics/images/arrow.gif)**步骤/帧**来选择查看未变形模型的步骤。更多信息，请参见"选择结果步骤和帧"，第42.3节。

您也可以使用Abaqus/CAE将您的未变形和变形模型形状组合成单个绘图。更多信息，请参见"叠加变形和未变形模型绘图"，第43.6节。将形状组合可以提供评估变形的上下文。
