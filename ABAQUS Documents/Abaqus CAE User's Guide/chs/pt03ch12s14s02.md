# 12.14.2 创建常规壳复合铺层





常规壳复合铺层由不同材料以不同取向组成的铺层构成。铺层在不同区域可以包含不同数量的铺层。请参见[第23章，"复合铺层"](pt04ch23.md)。Abaqus使用仅离散每个铺层参考表面的常规壳元素对壳复合铺层进行建模。壳截面行为根据壳截面对拉伸、弯曲、扭转和横向剪切的响应来定义。请参见[Abaqus Analysis User's Guide的第29.6.4节，"壳截面行为"](../usb/usb-link.md#usb-elm-eshellsectionbehavior)。

创建常规壳复合铺层时，必须选择截面积分方法。您可以选择在分析前提供截面属性数据（预积分壳截面），或让Abaqus在分析期间从截面积分点计算（积分）横截面行为。

在分析期间积分的常规壳复合铺层允许通过壳厚度的数值积分计算横截面行为，从而在材料建模中提供完全的通用性。可以通过厚度定义任意数量的材料点，材料响应可以从点到点变化。当复合铺层包含非线性材料行为时，通常使用在分析期间积分的壳元素。必须使用在分析期间积分的壳元素来建模热传递。请参见["使用在分析期间积分的壳截面来定义截面行为"，第29.6.5节Abaqus Analysis User's Guide](../usb/usb-link.md#usb-elm-eusingshellsection)。

可以使用预积分复合铺层定义线性弯矩-弯曲和力-膜应变关系。在这种情况下，所有计算都以截面力和弯矩的形式进行。截面属性由弹性材料指定；可选择地，您还可以基于关于铺层预期行为或组成的假设来应用理想化。如果铺层的响应是线弹性的，并且其行为不依赖于温度或预定义场变量的变化，应使用预积分复合铺层。请参见["使用通用壳截面定义截面行为"，第29.6.6节Abaqus Analysis User's Guide](../usb/usb-link.md#usb-elm-eusingshellgensect)。

创建常规壳复合铺层后，您可以使用铺层堆叠图查看铺层区域的核心样本的图形表示。请参见[第53章，"查看铺层堆叠图"](pt05ch53.md)。

### 创建常规壳复合铺层

您可以定义由一种或多种材料制成的铺层组成的常规壳复合铺层。对于铺层的每个铺层，您指定名称、材料、厚度、取向和积分点数量。此外，您选择分配铺层的区域。请参见[Abaqus Analysis User's Guide的第29.6.4节，"壳截面行为"](../usb/usb-link.md#usb-elm-eshellsectionbehavior)。

**创建常规壳复合铺层：**

1. 从主菜单栏，选择****Composite**![](../graphics/images/arrow.gif)**Create****。出现**Create Composite Layup**对话框。
   **提示：**您也可以在**Composite Layup Manager**中点击**Create**，或在Property模块工具箱中选择创建复合铺层工具![](../graphics/ico_compositeLayup.png)。
2. 输入复合铺层名称。Abaqus/CAE此名称显示在铺层堆叠图中。
3. 指定初始铺层数量。当复合铺层编辑器出现时，它将包含每一行的一个铺层；但是，您可以使用编辑器随后添加或删除铺层。铺层表格的第一行对应于铺层底部的铺层。
4. 选择**Conventional Shell**作为**Element Type**，然后点击**Continue**。出现复合铺层编辑器。
5. 输入铺层的描述。Abaqus/CAE此描述显示在复合铺层管理器中。
6. 执行以下操作之一来指定铺层取向：
   - 选择**Part global**以使用部件的坐标系，并选择代表**Normal direction**的轴。
   - 选择**Coordinate system**以选择现有坐标系（或创建新坐标系并选择它），然后执行以下操作：
     1. 选择代表**Normal direction**的轴。
     2. 指定额外旋转。通过此角度绕选定轴旋转所选坐标系。您可以指定角度，或可以选择定义在铺层上空间变化的标量离散场。Abaqus/CAE仅允许您选择有效的离散场，对于额外旋转，这些是应用于元素的标量离散场。您还可以通过点击![](../graphics/ico_discreteField.png)创建新的离散场。请参见[第63章，"离散场工具集"](pt06ch63.md)。
   - 选择**Discrete**以定义离散取向，然后执行以下操作：
     1. 点击![](../graphics/ico_edit.png)。
     2. 在出现的**Edit Discrete Orientation**对话框中，使用["使用离散取向进行材料取向和复合铺层取向，"第12.16节"](pt03ch12hla01.md)中所述的过程定义法线轴和主轴。
     3. 指定额外旋转。通过此角度绕所选法线轴旋转取向。您可以指定角度，或可以选择或创建标量离散场以定义在铺层上空间变化的角度。Abaqus/CAE仅允许您选择有效的离散场，对于额外旋转，这些是应用于元素的标量离散场。您还可以通过点击![](../graphics/ico_discreteField.png)创建新的离散场。请参见[第63章，"离散场工具集"](pt06ch63.md)。
   - 选择**User-defined**以在用户子程序[`ORIENT`](../sub/sub-link.md#sub-xsl-orient)中定义取向。此选项仅对Abaqus/Standard分析有效。请参阅以下章节获取更多信息：
     - ["指定常规作业设置，"第19.8.6节"](pt03ch19s08hlb06.md)
     - [Abaqus User Subroutines Reference Guide的第1.1.15节，"ORIENT"](../sub/sub-link.md#sub-rtn-uorient)
   - 选择取向离散场的名称以指定在铺层上空间变化的坐标系。您还可以通过点击![](../graphics/ico_discreteField.png)在**Definition**字段右侧创建新的离散场。请参见[第63章，"离散场工具集"](pt06ch63.md)。选择离散场后，您必须执行以下操作：
     1. 选择代表**Normal direction**的轴。
     2. 指定额外旋转。通过此角度绕选定轴旋转所选坐标系。您可以指定角度，或可以选择或创建标量离散场以定义在铺层上空间变化的角度。
   铺层取向是任何使用默认取向系统的铺层的参考取向（由铺层表格**CSYS**列中的**<Layup>**指示）。此取向用于单个铺层中的材料计算和应力输出、截面力输出以及横向剪切刚度。您可以通过指定参考取向和/或旋转角度为连续壳复合铺层的单个铺层指定不同的取向。请参见["理解复合铺层和取向，"第23.3节"](pt04ch23s03.md)。
7. 选择以下**Section integration**方法之一：
   - 选择**During analysis**以为在分析期间积分的壳复合铺层指定属性。
   - 选择**Before analysis**以为预积分壳复合铺层指定属性。
8. 如果您为在分析前积分的复合铺层指定属性，请指定**Idealization**以基于关于铺层预期行为或组成的假设应用于壳。请参见"使用通用壳截面定义截面行为"中的"理想化截面响应"，第29.6.6节Abaqus Analysis User's Guide。
   - 选择**No idealization**以考虑由材料分配和铺层组成确定的壳的完整刚度。
   - 如果您不知道复合铺层中铺层的确切堆叠顺序，请选择**Smeared properties**。每个指定铺层的贡献被 smeared 到整个铺层厚度上，产生与堆叠顺序无关的一般响应。
   - 如果壳的主要响应是面内拉伸，请选择**Membrane only**；弯曲刚度项从壳刚度计算中消除。
   - 如果壳的主要响应是纯弯曲，请选择**Bending only**；膜刚度项从壳刚度计算中消除。
9. 如果您为在分析期间积分的复合铺层指定属性，请选择**Thickness integration rule**。
   - 选择**Simpson**以使用Simpson规则进行壳截面积分。
   - 选择**Gauss**以使用Gauss积分进行壳截面积分。请参见"使用在分析期间积分的壳截面来定义截面行为"中的"定义壳截面积分"，第29.6.5节Abaqus Analysis User's Guide。
10. 完成定义壳复合铺层后，点击**OK**保存更改并关闭编辑器。

### 指定常规壳复合铺层的铺层

复合铺层由一系列铺层组成。您选择分配铺层的区域，并指定每个铺层的名称、材料、厚度、取向和积分点数量。使用铺层表格上方的图标或点击铺层表格上的鼠标按钮3查看允许您编辑表格单元格内容和操作表格数据的菜单；例如，您可以添加和删除铺层、图案铺层和反转铺层。您还可以从文件读取数据到表格或从表格写入数据到文件。请参见["定义复合铺层时使用铺层表格，"第12.14.1节"](pt03ch12s14s01.md)。

**指定常规壳复合铺层的铺层：**

1. 从**Composite Layup**编辑器，点击**Plies**标签。
2. 如果复合铺层中的铺层关于中心芯对称，请切换**Make calculated sections symmetric**。在铺层表格中输入铺层，从第一行的底部铺层开始，到中心铺层结束。在分析过程中，Abaqus通过以相反顺序重复输入的铺层（包括中心铺层）将铺层附加到铺层定义到铺层的顶部。每个生成的铺层通过在重复铺层的原始名称前面添加`Sym_`来在铺层堆叠图和输出数据库中标记。如果任何铺层的厚度或旋转角度使用离散场定义，则此选项不能使用。
3. 对于每个铺层，在铺层表格中输入以下数据：
   **Ply Name** 铺层的名称。Abaqus/CAE在Visualization模块中查看复合铺层时以及在铺层堆叠图中显示此名称。
   **Region** 选择分配铺层的区域。您可以从视口中选择面，或选择引用面的集。如果您正在显示依赖网格的部件，可以从视口中选择壳单元或选择单元集。要从视口中选择单元，必须显示本地网格，并且必须使用**Selection**工具栏启用**2D Elements**的选择。您也可以从网格部件中选择壳单元。请参见["显示本地网格，"第17.3.11节"](pt03ch17s03s11.md)和["根据对象类型过滤您的选择，"第6.3.2节"](pt01ch06s03s01.md)。
   **Material** 此铺层的材料名称。点击鼠标按钮3，从出现的菜单中选择**Edit Material**，然后执行以下操作之一：
   - 从可用材料列表中选择所需的材料。
   - 点击![](../graphics/ico_material.png)创建新材料。
   **Thickness** 铺层厚度。直接在表格中输入均匀厚度，或点击鼠标按钮3并选择**Edit Thickness**以执行以下操作：
   - 选择**Specify Value**为铺层输入均匀厚度。
   - 选择**Distribution**并选择标量离散场以指定在铺层上空间变化的厚度。
   **Coordinate system ** 要定义为铺层参考取向基础的坐标系，请执行以下操作：
   1. 点击鼠标按钮3，并从出现的菜单中选择**Edit CSYS**。
   **注意：**如果您从菜单中选择**Edit Orientation**，可以定义坐标系和旋转角度。
   2. 选择基础取向。您可以选择基础铺层取向，或选择坐标系。如果您选择坐标系，必须选择定义法线方向的轴。
   **Rotation Angle** 铺层参考取向的额外旋转（绕法线方向逆时针）。直接在表格中输入均匀旋转，或点击鼠标按钮3并选择**Edit Rotation Angle**以执行以下操作：
   - 选择预定义角度（0、45、90或45度）以定义均匀旋转。
   - 选择**Uniform**，并输入**Angle**以定义均匀旋转。
   - 选择标量离散场以定义在铺层上空间变化的旋转。您还可以通过点击![](../graphics/ico_discreteField.png)创建新的离散场。
   **Integration Points** 积分点的数量，如果您为在分析期间积分的复合铺层指定属性。

### 指定常规壳复合铺层的偏移

在大多数情况下，您可以使用元素的中面来指示参考表面。但是，在某些情况下，您需要将参考表面定义为与元素的中面偏移。例如，从CAD系统导入的模型可能假定壳位于元素的顶部或底部表面。此外，您可以指定壳偏移以定义更精确的表面几何形状，这对于壳厚度很重要的接触问题很重要。偏移值定义为从中间表面到参考表面的总厚度的分数。

偏移的正值在正元素法线方向上。当偏移设置为0.5时，元素的顶面是参考表面。当偏移设置为-0.5时，底面是参考表面。默认偏移为0，表示元素的中面是参考表面。[图12-18](pt03ch12s14s02.md#usi-shelloffset)显示了到元素顶面的偏移。

**图12-18** 到元素顶面的偏移。

![](../graphics/shelloffset-nls.png)

有关更多信息，请参见[Abaqus Analysis User's Guide的第29.6.3节，"定义常规壳元素的初始几何"](../usb/usb-link.md#usb-elm-eshellgeometry)。您可以使用离散场对具有连续变化偏移的元素进行建模。请参见[第63章，"离散场工具集"](pt06ch63.md)。

**指定常规壳复合铺层的偏移：**

1. 从**Composite Layup**编辑器，点击**Offset**标签。
2. 执行以下操作之一：
   - 选择**Middle surface**、**Top surface**或**Bottom surface**来表示参考表面。
   - 选择**Specify offset ratio**，并输入数字。
   - 选择**Distribution**，并选择定义在铺层上空间变化的偏移的标量离散场。Abaqus/CAE仅允许您选择有效的离散场，对于偏移，这些是应用于元素的标量离散场。您还可以通过点击![](../graphics/ico_discreteField.png)创建新的离散场。请参见[第63章，"离散场工具集"](pt06ch63.md)。

### 指定常规壳复合铺层的壳参数

**指定常规壳复合铺层的壳参数：**

1. 从**Composite Layup**编辑器，点击**Shell Parameters**标签。
2. 指定**Shell thickness**。
   - 选择**Use section thickness**以使用从各个铺层厚度计算的厚度。
   - 选择**Element distribution**；并选择解析场（标记为(A)）或基于元素的离散场（标记为(D)）以定义空间变化的基于元素的壳厚度。或者，您可以点击![](../graphics/ico_analyticalField.png)创建新解析场或点击![](../graphics/ico_discreteField.png)创建新离散场。请参见[第58章，"解析场工具集"](pt06ch58.md)和[第63章，"离散场工具集"](pt06ch63.md)，了解更多。
   - 选择**Nodal distribution**；并选择解析场（标记为(A)）或基于节点的离散场（标记为(D)）以定义空间变化的基于节点的壳厚度。或者，您可以点击![](../graphics/ico_analyticalField.png)创建新解析场或点击![](../graphics/ico_discreteField.png)创建新离散场。请参见[第58章，"解析场工具集"](pt06ch58.md)和[第63章，"离散场工具集"](pt06ch63.md)，了解更多。
3. 指定**Section Poisson's ratio**以定义壳厚度行为。在允许大变形分析中有限膜应变的常规壳元素中，指定截面泊松比会导致壳厚度作为膜应变的函数而变化。
   - 选择**Use analysis default**以使用默认值。在Abaqus/Standard中，默认值为0.5，这将强制元素膜应变的不可压缩行为。在Abaqus/Explicit中，默认值基于元素材料定义中的厚度变化。
   - 选择**Specify value**，并输入泊松比值。此值必须介于1.0和0.5之间。值为0.0将强制恒定壳厚度，负值将导致壳厚度响应拉伸膜应变而增加。
4. 如果您为在分析期间积分的复合铺层指定属性，请选择定义截面**Temperature variation**的方法：
   - 选择**Linear through thickness**以指示已指定参考表面温度和穿过铺层的温度梯度。您可以使用Load模块指定这些温度。
   - 选择**Piecewise linear over *n* values**以在提供的文本字段中输入穿过铺层的温度点（值）的数量。您可以使用Load模块指定每个这些点的温度。
5. 切换**Density**，并输入密度值。铺层的质量包括来自密度的贡献以及所选材料的任何贡献。
6. 对于大多数壳截面，Abaqus计算元素公式中所需的横向剪切刚度值。如有需要，从**Transverse Shear Stiffnesses**选项切换**Specify values**以在截面定义中包含非默认横向剪切刚度效应，并输入以下值：![](../graphics/usi_eqn00369.gif)，截面在第一方向上的剪切刚度；![](../graphics/usi_eqn00370.gif)，截面剪切刚度中的耦合项；以及![](../graphics/usi_eqn00371.gif)，截面在第二方向上的剪切刚度。如果![](../graphics/usi_eqn00369.gif)或![](../graphics/usi_eqn00371.gif)中的任何一个被省略或给定为零，则将使用非零值作为两者的值。请参见"使用在分析期间积分的壳截面来定义截面行为"中的"定义横向剪切刚度"，第29.6.5节Abaqus Analysis User's Guide。

### 指定常规壳复合铺层所选铺层的显示

**指定所选铺层的显示：**

1. 从**Composite Layup**编辑器，点击**Display**标签。
2. 选择Abaqus/CAE高亮所选铺层的方式：
   - 选择**On**以高亮所选铺层。
   - 选择**Off**以停止高亮所选铺层。如果模型中有大量铺层，您可能希望关闭高亮以提高性能。
3. 选择Abaqus/CAE在所选铺层上显示的取向：
   - 选择**Ply**以在所选铺层上显示铺层取向。
   - 选择**Layup**以在所选铺层上显示铺层取向。
   - 选择**None**以停止显示取向。
4. 指定您要在所选铺层上显示的取向向量。您可以切换1方向、2方向和3（或法线）方向向量的显示，以及显示铺层材料方向旋转前1方向的参考方向。
