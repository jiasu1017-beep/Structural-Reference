# 1.1.1 简介：概述






### Abaqus有限元系统概述

Abaqus有限元系统包括：
- Abaqus/Standard，一个通用有限元程序；
- Abaqus/Explicit，一个显式动力学有限元程序；
- Abaqus/CFD，一个通用计算流体动力学程序；
- Abaqus/CAE，一个交互式环境，用于创建有限元模型、提交Abaqus分析、监控和诊断作业、评估结果；以及
- Abaqus/Viewer，Abaqus/CAE的一个子集，仅包含可视化模块的后处理功能。

多个附加选项可用于进一步扩展Abaqus/Standard和Abaqus/Explicit的功能。Abaqus/Aqua选项与Abaqus/Standard和Abaqus/Explicit配合使用。Abaqus/Design和Abaqus/AMS选项与Abaqus/Standard配合使用。Abaqus/Aqua包含专门为水下安装的梁状结构分析而设计的可选功能，这些结构承受水流和波浪作用。Abaqus/Design选项使您能够执行设计灵敏度分析（DSA）。Abaqus/AMS是一个可选的特征值求解器，在Abaqus/Standard内工作，提供大型对称特征值问题的快速解。Abaqus协同仿真技术提供了多种应用，作为单独的附加功能，用于Abaqus与第三方分析程序之间的耦合。Abaqus/Foundation是Abaqus/Standard的一个可选子集，以更经济的方式提供Abaqus/Standard中的线性静态和动态分析功能。这些选项仅在您的许可证包含它们时才可用。

有关Abaqus产品、实用程序和附加选项的完整列表，请参阅["Abaqus产品"，Abaqus发行说明第1.2节](../rnb/rnb-link.md#rnb-chp-int-products)。

### 本指南概述

本指南是使用Abaqus/Standard（包括Abaqus/Aqua、Abaqus/Design和Abaqus/Foundation）、Abaqus/Explicit（包括Abaqus/Aqua）和Abaqus/CFD的参考。Abaqus/Standard在每个求解"增量"中隐式求解方程组。相比之下，Abaqus/Explicit以小时间增量向前推进解，而无需在每个增量中求解耦合方程组（甚至不形成全局刚度矩阵）。Abaqus/CFD提供计算流体动力学能力，在Abaqus/CAE中全面支持前处理、模拟和后处理。

在整个指南中，术语"Abaqus"最常用于统指Abaqus/Standard和Abaqus/Explicit，以及适用的Abaqus/CFD；个别产品名称用于表示信息仅适用于该产品。产品标识符出现在指南每个章节的开头（不包括概述章节），指示章节中的信息适用于哪些产品。

本指南分为几个部分：
- [第一部分，"简介、空间建模和执行"](pt01.md)，讨论Abaqus中的基本建模概念，例如定义节点、单元和表面；使用Abaqus时应遵循的约定和输入格式；以及Abaqus/Standard、Abaqus/Explicit、Abaqus/CFD、Abaqus/CAE和Abaqus系统提供的几个实用程序的执行过程。
- [第二部分，"输出"](pt02.md)，描述如何从Abaqus获取输出以及结果（`.fil`）文件的格式。它还描述了可用的输出变量标识符。
- [第三部分，"分析过程、求解和控制"](pt03.md)，描述可用的分析类型（静态应力分析、动力学、特征值提取等）。本章提供了Abaqus/Standard和Abaqus/Explicit如何求解有限元分析的详细讨论。
- [第四部分，"分析技术"](pt04.md)，讨论Abaqus中可用的各种分析技术，例如子模型、移除单元或表面，以及从先前模拟导入结果以定义当前模型的初始条件。
- [第五部分，"材料"](pt05.md)，描述材料建模选项以及如何校准一些更高级的材料模型。
- [第六部分，"单元"](pt06.md)，描述Abaqus中可用的单元。
- [第七部分，"规定条件"](pt07.md)，描述规定条件（如分布载荷和节点速度）的使用。
- [第八部分，"约束"](pt08.md)，讨论约束（如多点约束）的使用。
- [第九部分，"相互作用"](pt09.md)，讨论Abaqus中可用的接触和相互作用模型。

本指南还包括Abaqus/Standard、Abaqus/Explicit和Abaqus/CFD中所有输出变量和单元的[索引](pt10.md)。

### 使用Abaqus

Abaqus/Standard、Abaqus/Explicit和Abaqus/CFD可以作为批处理应用程序运行（详见["Abaqus/Standard、Abaqus/Explicit和Abaqus/CFD执行"，第3.2.2节"](pt01ch03s02abx02.md)），或通过交互式Abaqus/CAE环境运行（详见["Abaqus/CAE执行"，第3.2.6节"](pt01ch03s02abx06.md)，了解如何启动Abaqus/CAE）。Abaqus/Standard、Abaqus/Explicit和Abaqus/CFD分析产品的主要输入是一个包含模拟所需选项和这些选项相关数据的文件。也可能有补充文件，例如先前分析的重启或结果文件，或辅助数据文件，例如包含加速度记录或地震记录的动态分析文件。输入文件通常由Abaqus/CAE或其他预处理器创建。本指南提供了输入文件使用和Abaqus/CAE使用的信息。

如["在Abaqus中定义模型"，第1.3.1节"](pt01ch01s03aus03.md)中所述，主输入文件由两部分组成：模型输入和历史输入。输入围绕一些自然概念和约定组织，这意味着即使复杂模拟的输入文件可能很大，也可以毫无困难地管理。管理Abaqus输入文件的基本语法规则在["输入语法规则"，第1.2.1节"](pt01ch01s02aus01.md)中讨论。[Abaqus关键词参考指南](../key/key-link.md#key)包含Abaqus/Standard、Abaqus/Explicit和Abaqus/CFD中所有可用输入选项的完整描述。有关使用Abaqus进行分析的详细入门，建议您按照[《Abaqus入门：交互版》](../gsa/gsa-link.md#gsa)中的自助教程进行学习。有关使用Abaqus/CAE的详细信息，请参阅[Abaqus/CAE用户指南](../usi/usi-link.md#usi)。

此外，[Abaqus示例问题指南](../exa/exa-link.md#exa)、[Abaqus基准指南](../bmk/bmk-link.md#bmk)和[Abaqus验证指南](../ver/ver-link.md#ver)中讨论了许多展示Abaqus众多功能的分析。作为[Abaqus分析用户指南](book01.md)的补充，这些示例可以帮助您熟悉Abaqus提供的功能以及Abaqus输入文件的结构。例如，["梁撞击圆柱体"，Abaqus验证指南第1.6.12节](../ver/ver-link.md#ver-elm-beamimpact)讨论了可用于分析悬臂梁动态响应的各种建模技术。

### 审查Abaqus模拟的结果

请求Abaqus模拟输出的信息在["输出"，第4.1.1节"](pt02ch04s01aus38.md)中讨论。通过Abaqus/CAE中的可视化模块（也作为Abaqus/Viewer单独授权）查看Abaqus模拟的请求结果。可视化模块读取输出数据库文件，以创建Abaqus结果的等高线图、动画、*X-Y*图和表格输出。有关使用Abaqus/CAE中可视化模块的详细信息，请参阅[Abaqus/CAE用户指南](../usi/usi-link.md#usi-vismod)的["查看结果"第五部分](../usi/usi-link.md#usi-vismod)。
