# 17.7.3 确定哪些区域可进行网格划分





当选择**网格默认值**颜色映射时，网格模块中区域的颜色表示当前分配给该区域的网络技术。颜色编码如下：
- 结构化网格技术：绿色
- 自由网格技术：粉红色
- 扫掠网格技术：黄色
- 不可网格化：橙色
- 自底向上网格技术：浅棕色

请参阅["结构化网格和映射网格，"第17.8节](pt03ch17s08.md)；["自由网格，"第17.10节](pt03ch17s10.md)；["扫掠网格，"第17.9节](pt03ch17s09.md)；以及["自底向上网格，"第17.11节](pt03ch17s11.md)，了解每种网格技术的详细信息。请参阅[第77章，"几何和网格单元的颜色编码"](pt07ch77.md)，了解有关颜色映射的更多信息。

在许多情况下，Abaqus/CAE可以使用多种技术对区域进行网格划分；在这些情况下，您可以接受默认技术，也可以使用**网格控制**对话框选择替代技术。此外，您可以通过向区域添加分割或将不同的单元形状分配给区域来更改对区域有效的网格技术。例如，如果您将不可网格化的三维零件实例的单元形状分配从六面体更改为四面体，则该零件实例可以使用自由网格技术进行网格划分。有关更多信息，请参见["为什么需要在网格模块中进行分割？"第17.6.3节](pt03ch17s06s03.md)。

**注意：**您必须使用**网格控制**对话框将自底向上网格技术分配给区域。要取消分配自底向上技术，您可以选择其他技术或单击**网格控制**对话框中的**默认值**，让Abaqus/CAE对该区域使用默认单元形状和网格技术。

二维模型的默认网格技术是自由网格技术。如果您对自由网格技术生成的网格质量不满意，或者您更喜欢更规则的网格状图案，则可以将结构化网格技术分配给模型的简单区域。但是，如果您的模型很大且复杂，识别适用于结构化网格的简单区域可能非常耗时。为加快此过程，您可以将结构化网格技术应用于整个模型，Abaqus/CAE将执行以下操作：
- 确定是否有任何面太复杂而无法进行结构化网格划分，并询问您是否希望将其从选择中移除。
- 确定是否有任何面形状不佳并会导致不可接受的网格质量，并询问您是否希望将其从选择中移除。

如果Abaqus/CAE从您的选择中移除了任何面，则将它们着色为粉红色，以表示将使用自由网格进行网格划分。剩余的面着色为绿色，表示Abaqus/CAE将使用结构化网格进行网格划分。
