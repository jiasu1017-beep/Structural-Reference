# 64.2 编辑孤兒网格、网格化零件和装配中网格化零件实例有什么区别？





您可以在"网格"模块中显示包含孤兒和原生网格组件组合的零件或装配。您可以使用网格编辑工具集中的所有工具来编辑孤兒网格；但是，您只能使用以下工具来编辑装配中的网格化零件实例：
- 编辑、拖动、投影、合并和平滑节点
- 删除单元
- 折叠单元边缘
- 分割四边形或三角形单元的边缘
- 交换一对相邻三角形单元的对角线
- 将四边形单元分割成两个三角形单元
- 将两个三角形单元合并为一个四边形单元

如果您在装配中选择零件而不是零件实例，则可以使用上述网格编辑工具以及以下附加工具：- 创建节点和单元
- 偏移网格层以创建实体或壳层
- 将网格与几何体关联
- 删除网格关联性
- 复制网格图案

通常，您不能使用网格编辑工具集修改从属零件实例的网格。但是，您可以编辑、拖动和投影从属零件实例的节点；Abaqus/CAE修改原始网格化零件，您的修改会显示在该零件的所有从属实例上。

您可以在"网格"模块中创建孤兒网格，也可以从输出数据库或Abaqus输入文件导入孤兒网格。您可以在"零件"模块中向孤兒网格添加几何特征。孤兒网格的实例始终是从属实例；因此，您不能使用网格编辑工具集来编辑装配中孤兒网格的实例。但是，您可以显示用于创建实例的原始孤兒网格并编辑零件。有关更多信息，请参见["从属零件实例和独立零件实例有什么区别？" 第13.3.2节](pt03ch13s03s02.md)。

网格编辑撤销功能可以回滚您对网格所做的任何更改。有关更多信息，请参见["撤销或重做网格编辑工具集中的更改，" 第64.9节](pt06ch64s09.md)。
