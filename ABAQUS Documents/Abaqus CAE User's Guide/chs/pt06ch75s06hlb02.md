# 75.6.2 合并面





您可以选择 ****Tools**![](../graphics/images/arrow.gif)**Virtual Topology**![](../graphics/images/arrow.gif)**Combine Faces**** 从主菜单栏来合并选定的面。在网格生成过程中，Abaqus/CAE 将合并的面视为单个面，并忽略被移除的所有边和顶点。

**合并面的步骤：**

1. 从上下文栏的 **Object** 字段中，选择一个零件或装配。
2. 从主菜单栏，选择 ****Tools**![](../graphics/images/arrow.gif)**Virtual Topology**![](../graphics/images/arrow.gif)**Combine Faces****。**提示：**您也可以使用位于 Mesh 模块工具箱中的 ![](../graphics/ico_virtTopoCombineFaces.png) 工具来合并面。
3. 选择要合并的面。您可以使用拖选、**[Shift]****+点击**、**[Ctrl]****+点击** 和角度方法的组合来选择多个要合并的面。有关更多信息，请参见 ["在当前视口中选择对象，" 第6.2节](pt01ch06s02.md)。**提示：**如果您无法选择所需的面，可以使用 **Selection** 工具栏更改选择行为。有关更多信息，请参见 ["使用选择选项，" 第6.3节](pt01ch06s03.md)。
4. 如果所选面的任意边的端点处的顶点也可以被忽略，Abaqus/CAE 会高亮显示这些顶点，并询问您是否要将它们包含在您的选择中。点击 **Yes** 忽略高亮的顶点。点击 **No** 保留高亮的顶点。Abaqus/CAE 合并选定的面和顶点。您可以使用模型树通过删除或抑制虚拟拓扑特征来恢复选定的面和顶点；您可以使用 **Restore Entities** 工具选择要恢复的面和顶点，同时保留虚拟拓扑特征的其余部分。有关恢复单个实体的更多信息，请参见 ["恢复被虚拟拓扑替换的实体，" 第75.6.5节](pt06ch75s06hlb05.md)。
