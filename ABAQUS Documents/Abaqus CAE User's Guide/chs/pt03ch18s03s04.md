# 18.3.4 关于约束





约束也从设计响应中提取。 约束限制设计响应的值； 例如，您可以指定体积必须减少45%，或者区域中的绝对位移不能超过1mm。 您还可以应用独立于优化的制造和几何约束； 例如，结构必须能够被铸造或冲压，或者支承表面的直径不能被更改。

满足约束优先于目标函数的最小化或最大化。 优化算法仅在约束被满足后才开始最大化或最小化目标。

对于基于条件的拓扑或形状优化，您只能指定体积约束，并且体积约束必须等于固定值或优化开始前值的一部分。 如果请求的体积与初始体积差异很大，Optimization模块可能需要多个设计周期来满足体积约束。 一般拓扑优化提供更大的灵活性； 您可以选择任何设计响应作为约束，但使用Kreisselmaier-Steinhauser公式计算固有频率的设计响应除外。 一般拓扑优化中的约束可以小于、大于或等于固定值或优化开始前值的一部分。 有关更多信息，请参阅["创建约束，" 第18.9节](pt03ch18s09.md)和["目标和约束，" Abaqus Analysis User's Guide第13.2.2节](../usb/usb-link.md#usb-anl-aoptobjectives)。
