# 10.7.16 从Elysium中性文件导入装配





从主菜单栏选择**文件**![](../graphics/images/arrow.gif)**导入**![](../graphics/images/arrow.gif)**装配**，可以从Elysium中性文件导入装配。您可以导入由以下软件生成的Elysium中性文件：
- 使用CATIA V6关联接口的CATIA V6。
- 使用Elysium翻译器插件的CATIA V5。
- 使用Pro/ENGINEER关联接口的NX。
- 使用Pro/ENGINEER关联接口的Pro/ENGINEER。

有关Abaqus/CAE支持的CATIA V6、CATIA V5和Pro/ENGINEER版本的详细信息，请参见Dassault Systmes知识库，网址为[www.3ds.com/support/knowledge-base](http://www.3ds.com/support/knowledge-base)。

**从Elysium中性文件导入装配：**

1. 从主菜单栏，选择**文件**![](../graphics/images/arrow.gif)**导入**![](../graphics/images/arrow.gif)**装配**。出现**导入装配**对话框。
2. 从**导入装配**对话框底部的**文件过滤器**菜单中，选择**ProE/NX/CATIA V5 Elysium中性 (*.enf*)**。Abaqus/CAE列出当前目录中所有具有相应文件扩展名的文件。
3. 选择包含要导入的装配的文件，然后单击**确定**。Abaqus/CAE显示**从ENF文件导入装配**对话框。
4. 执行以下任一操作：-单击装配名称以导入装配中的所有零件实例。-单击装配名称左侧的箭头以显示装配中的所有零件实例。单击要导入的零件实例的名称。
5. 单击**确定**导入装配或选定的零件实例，并退出**从ENF文件导入装配**对话框。Abaqus/CAE显示一条消息，指示任何零件实例是否无效或不精确。如有必要，使用查询工具集定位无效和不精确几何的区域。更多信息，请参见"在零件模块中使用查询工具集"，第11.16.4节。
