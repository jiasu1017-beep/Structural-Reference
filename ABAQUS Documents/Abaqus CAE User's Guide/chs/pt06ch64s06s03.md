# 64.6.3 反转壳单元的表面法线方向





从主菜单栏选择 ****网格**![](../graphics/images/arrow.gif)**编辑**** 可以反转所选壳单元的表面法线。单元可以是四边形或三角形。在反转表面法线的过程中，Abaqus/CAE 使用着色渲染样式显示零件，壳单元的前面和背面以不同颜色显示。如果您在某个区域更改了表面法线的方向，Abaqus/CAE 不会更改应用于该区域的预定义条件或命名曲面的方向。

**反转表面法线的步骤：**

1. 进入"网格"模块。
2. 从上下文栏的 **对象** 字段中选择 **零件** 并从列表中选择一个零件。
3. 从主菜单栏选择 ****网格**![](../graphics/images/arrow.gif)**编辑****。Abaqus/CAE 显示 **编辑网格** 对话框。 **提示：** 您也可以使用位于"网格"模块工具箱底部的 ![](../graphics/ico_meshEdit.png) 工具显示 **编辑网格** 对话框。
4. 在对话框中执行以下操作： 1. 在 **类别** 字段中选择 **单元**。 2. 从 **方法** 列表中选择 **翻转法线**。Abaqus/CAE 使用着色渲染样式显示零件。每个单元的正面着为棕色，背面着为紫色。
5. 从提示区域选择区域类型 **几何体** 或 **网格**，并切换 **高亮显示冲突边缘** 以显示或隐藏具有冲突法线的连接单元之间的黄色边框。
6. 从提示区域选择选择方法，然后选择要反转其法线的壳区域或孤岛壳单元。有关更多信息，请参见 ["选择要编辑的单元，" 第64.6.11节](pt06ch64s06s11.md)。
7. 完成选择区域或单元后，从提示区域选择 **完成**。如果您选择了几何区域，Abaqus/CAE 会更改单元的法线并返回到步骤5。
8. 如果您选择了孤岛单元，请选择反转表面法线的方法： - 点击 **全部翻转** 以反转所有所选单元的法线。 - 点击 **选择法线** 以更改所选单元的法线，使其指向您指定的参考单元法线相同的方向。
9. 如果在上一步中选择了 **选择法线**，请执行以下操作： 1. 在视口中选择参考单元。 2. 在提示区域点击 **确定**。Abaqus/CAE 会更改所选单元的法线，使其指向与参考单元法线相同的方向，并返回到步骤5，以便您进行更多选择。
10. 点击鼠标按钮2退出过程。Abaqus/CAE 会恢复到开始反转表面法线过程之前选择的渲染样式。

**提示：** 如果在反转孤岛单元的表面法线时出错，点击 **编辑网格** 对话框中的 **撤销** 可以撤消更改。Abaqus/CAE 会为所选单元恢复原始表面法线。如果在反转几何区域中本地单元的表面法线时出错，请重新选择该区域并再次反转法线。
