# 15.12.11 创建连接器截面





您可以通过选择连接类型来定义连接器功能，从而创建连接器截面。
- 「为组装、复杂和基本连接类型创建连接器截面」
- 「为MPC类型创建连接器截面」

### 为组装、复杂和基本连接类型创建连接器截面

连接器截面编辑器允许您指定组装、复杂或基本连接类型；连接器行为；以及要包含在截面定义中的截面数据。更多信息，请参见「连接器单元」，Abaqus Analysis User's Guide第31.1.2节。

**为组装、复杂或基本连接类型创建连接器截面的步骤：**

1. 从主菜单栏选择 ****Connector**![](../graphics/images/arrow.gif)**Section**![](../graphics/images/arrow.gif)**Create****。**提示：**您也可以使用「相互作用」模块工具箱中的![](../graphics/ico_connectorSection.png)工具创建连接器截面。此时会出现「Create Connector Section」对话框。
2. 输入截面名称。有关命名对象的更多信息，请参见「使用基本对话框组件」，第3.2.1节。
3. 选择以下连接类别之一：- 选择「Assembled/Complex」使用预定义的基本连接类型组合或复杂连接类型。点击「Assembled/Complex type」文本字段旁边的箭头，并从出现的列表中选择所需的连接类型。- 选择「Basic」使用平移和旋转连接类型。1. 如有需要，点击「Translational type」文本字段旁边的箭头，并从出现的列表中选择连接类型。2. 如有需要，点击「Rotational type」文本字段旁边的箭头，并从出现的列表中选择连接类型。您可以选择一个平移类型、一个旋转类型，或一个平移和一个旋转类型来定义连接器截面。Abaqus/CAE显示所选连接类型的可用和约束相对运动分量（CORM）。此外，您可以点击![](../graphics/ico_tipSmall.png)查看连接类型的示意图以及Abaqus对连接的理想化。有关组装、复杂和基本连接类型的描述，请参见「连接类型库」，Abaqus Analysis User's Guide第31.1.5节。
4. 点击「Continue」。连接器截面编辑器会出现。**注意：**您可以通过从主菜单栏选择 ****Help**![](../graphics/images/arrow.gif)**On Context****，然后点击感兴趣的编辑器功能来显示该功能的帮助。有关编辑器功能的更多信息，请参见「使用连接器截面编辑器」，第15.17节。
5. 要编辑连接类型，请点击连接类型右侧的![](../graphics/ico_edit.png)以显示连接器截面类型编辑器。按上述方式选择连接类型。在编辑连接类型之前，您必须从编辑器中删除所有行为。
6. 在「Behavior Options」选项卡页面上，您可以按如下方式添加、删除和更改行为：**添加行为**点击编辑器右侧的「Add」以显示可用行为列表。选择定义连接器截面所需的行为。您可以为某些行为定义多个相同类型的行为。当您选择行为时，其名称会出现在「Behavior Options」列表中，与该行为关联的数据字段会出现在编辑器底部的数据区域中。使用数据字段输入当前所选行为的信息。更多信息，请参见「使用连接器截面编辑器」，第15.17节。**删除行为**在「Behavior Options」列表中，选择要删除的行为，并点击编辑器右侧的「Delete」。此过程会从行为选项列表和连接器截面定义中删除该行为。**更改行为数据**在「Behavior Options」列表中，选择要更改其数据的行为。当与该行为关联的数据字段出现在窗口底部时，根据需要更改您为该行为输入的信息。
7. 在「Table Options」选项卡页面上，您可以为连接器截面中所有行为选项的规则化（Abaqus/Explicit分析仅）和表格数据的外推指定行为设置。或者，您可以为单个行为选项指定行为设置。「Table Options」按钮可用于所选行为选项的「Behavior Options」选项卡页面上。单个行为选项的行为设置优先于连接器截面行为设置。更多信息，请参见「为表格数据指定行为设置」，第15.17.14节，以及「使用表格数据定义连接器行为」，Abaqus Analysis User's Guide第31.2.1节中的「连接器行为」。在「Table Options」选项卡页面上指定行为设置如下：1. 在页面的「Regularization」部分，指定Abaqus/Explicit分析中表格数据规则化的设置。默认情况下，Abaqus/Explicit将数据规则化为以自变量偶数间隔定义的表。- 切换「Regularize data」以规则化表格数据。- 关闭「Regularize data」以关闭表格数据的规则化并直接使用您定义的数据。2. 如果您想规则化表格数据，请指定误差容差。- 选择「Use default」使用默认值0.03。- 选择「Specify」并输入误差容差值。3. 在页面的「Extrapolation」部分，指定表格数据的外推方法。您输入的数据点构成构成空间中的非线性曲线。默认情况下，Abaqus将因变量外推为对应于自变量指定范围末端点之外曲线端点值的常数值。- 选择「Constant」在自变量指定范围之外使用因变量的常外推。- 选择「Linear」在自变量指定范围之外使用因变量的线性外推。
8. 「Section Data」选项卡页面对以下连接类型可用：- 对于「Flow-Converter」或「Retractor」连接类型，输入以下截面数据：**Node b material flow scaling factor**输入与节点b处材料流相关的缩放因子，![](../graphics/usi_eqn00572.gif)。节点b指的是用于在Abaqus/CAE中对连接器建模的线的第二个点。默认值为1。更多信息，请参见「FLOW-CONVERTER」，Abaqus Analysis User's Guide第31.1.5节中的「连接类型库」，以及「RETRACTOR」，Abaqus Analysis User's Guide第31.1.5节中的「连接类型库」。- 对于「Slip Ring」连接类型，输入以下截面数据：**Mass per unit reference length**输入每单位参考长度的皮带质量。**Contact angle around node b**接触角指的是皮带缠绕节点b所成的角度。节点b指的是用于在Abaqus/CAE中对连接器建模的线的第二个点。对于Abaqus/Standard分析，您必须直接指定接触角。对于Abaqus/Explicit分析，您可以直接指定接触角，也可以让Abaqus根据模型的构型计算接触角：- 要让Abaqus计算接触角，请选择「Compute」。- 要直接指定接触角，请选择「Specify」并输入以度为单位的接触角。更多信息，请参见「SLIPRING」，Abaqus Analysis User's Guide第31.1.5节中的「连接类型库」。
9. 点击「OK」保存数据并退出编辑器。

### 为MPC类型创建连接器截面

连接器截面编辑器允许您指定MPC连接类型。更多信息，请参见「通用多点约束」，Abaqus Analysis User's Guide第35.2.2节。

**为MPC类型创建连接器截面的步骤：**

1. 从主菜单栏选择 ****Connector**![](../graphics/images/arrow.gif)**Section**![](../graphics/images/arrow.gif)**Create****。**提示：**您也可以使用「相互作用」模块工具箱中的![](../graphics/ico_connectorSection.png)工具创建连接器截面。此时会出现「Create Connector Section」对话框。
2. 输入截面名称。有关命名对象的更多信息，请参见「使用基本对话框组件」，第3.2.1节。
3. 选择「MPC」连接类别。
4. 点击「MPC type」文本字段旁边的箭头，并从出现的列表中选择所需的连接类型。更多信息，请参见以下章节：- 「通用多点约束」，Abaqus Analysis User's Guide第35.2.2节- 「MPC」，Abaqus User Subroutines Reference Guide第1.1.14节
5. 点击「OK」保存截面定义并关闭对话框。
