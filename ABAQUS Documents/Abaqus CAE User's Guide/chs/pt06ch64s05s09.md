# 64.5.9 重新编号节点





从主菜单栏选择 ****网格**![](../graphics/images/arrow.gif)**编辑**** 可以对孤岛网格的所选节点重新编号。您可以通过指定起始节点标签和增量来重新编号所选节点，也可以通过指定值来偏移现有节点标签。（有关对 Abaqus/CAE 本地网格的节点重新编号的信息，请参见 ["更改所有节点和单元的标签，" 第17.18.11节](pt03ch17s18s09.md)。）

**重新编号节点的步骤：**

1. 进入"网格"模块。
2. 从上下文栏的 **对象** 字段中选择 **零件**，然后从列表中选择一个零件。此工具仅适用于不包含任何几何体的零件。
3. 从主菜单栏选择 ****网格**![](../graphics/images/arrow.gif)**编辑****。Abaqus/CAE 显示 **编辑网格** 对话框。 **提示：** 您也可以使用位于"网格"模块工具箱底部的 ![](../graphics/ico_meshEdit.png) 工具显示 **编辑网格** 对话框。
4. 在对话框中执行以下操作： 1. 在 **类别** 字段中选择 **节点**。 2. 从 **方法** 列表中选择 **重新编号**。Abaqus/CAE 显示 **节点重新编号** 对话框。**零件标签范围** 显示整个零件的节点标签范围。
5. 执行以下操作之一来指定重新编号方法： - 选择 **按起始标签**，然后输入第一个节点标签的值以及用于对后续节点进行编号的增量。 - 选择 **按偏移**，然后输入现有节点标签将被偏移的值。
6. 使用以下方法之一来选择要重新编号的节点：**选择所有节点** - 选择 **全部** 以对孤岛网格中的所有节点重新编号。**使用无序方法选择节点** 1. 选择 **指定** 和 **无序** 以任意顺序选择节点。 2. 点击 **选择** 以选择要重新编号的节点。 3. 从提示区域选择选择方法，然后选择要重新编号的节点。有关更多信息，请参见 ["选择要编辑的节点，" 第64.5.10节](pt06ch64s05s10.md)。**使用有向路径方法选择节点** 1. 选择 **指定** 和 **有向路径** 以沿特征边从选定的起始节点到选定的结束节点对节点重新编号。 2. 点击 **选择** 以选择要重新编号的节点。 3. 选择起始节点。Abaqus/CAE 显示沿特征边的所有节点。如有需要，可修改角度以重新定义 Abaqus/CAE 识别特征边的方式。 4. 从提示区域选择 **完成**。起始节点以紫色显示。 5. 选择结束节点。起始节点和结束节点必须位于同一特征边上。**使用序列方法选择节点** 1. 选择 **指定** 和 **序列** 以根据选择节点的顺序对节点重新编号。 2. 点击 **选择** 以选择要重新编号的节点。 3. 选择起始节点和每个其他节点（按顺序）。 4. 完成节点选择后，点击鼠标按钮2。**零件标签范围** 显示重新编号后节点的标签范围。
7. 点击 **确定** 或 **应用** 以重新编号节点。如果您点击 **应用**，Abaqus/CAE 将重新编号节点；但这些节点仍保持选中状态，您可以继续对它们进行重新编号。
8. 根据需要重复上述步骤以重新编号其他节点。
9. 完成重新编号节点后，点击 **取消** 以关闭 **节点重新编号** 对话框。
