# 11.7.2 绘制分析刚体零件的轮廓





Abaqus/CAE使用由一系列线、圆弧和抛物线组成的轮廓来表示分析刚体零件。草图编辑器中有几个工具可帮助您构建刚性零件轮廓的每个部分：

**线**

您使用草图编辑器的**线**工具绘制直线。

**圆弧和圆角**

您使用草图编辑器的**圆弧**和**圆角**工具绘制圆弧或对两条线进行圆角。任何生成的圆弧所对的角必须小于180度；如果要构造大于180度的圆弧，应创建两个相邻的圆弧。如果在绘制分析刚体表面轮廓时创建大于180度的圆弧，Abaqus/CAE会显示错误消息。

**样条线**

您使用草图编辑器的**样条线**工具绘制抛物线。您通过定义三点样条线来创建抛物线，这三个点是样条线的起点、沿样条线任意位置的一点和终点。只有由恰好三点组成的样条线才能生成分析刚体零件定义所需的抛物线；因此，如果在绘制分析刚体零件轮廓时使用超过三点创建样条线，Abaqus/CAE会显示错误消息。

您可以使用线、圆弧和抛物线的任意组合来构建分析刚体零件；但是，生成的轮廓必须是单个连接的（但不一定闭合）曲线。此外，为了在Abaqus/Standard或Abaqus/Explicit中获得收敛解，曲线必须平滑。您可能需要应用一系列小的线、圆弧或抛物线来消除任何表面不连续性（Abaqus/CAE没有Abaqus/Standard和Abaqus/Explicit的[*SURFACE*](../key/key-link.md#usb-kws-msurface)选项上的FILLET RADIUS参数的等效项）。有关创建抛物线并保持切线性的更多信息，请参见"绘制样条线"，第20.10.10节。有关分析刚体表面规则的更多信息，请参见"分析刚性表面定义"，Abaqus分析用户指南第2.3.4节。

分析刚体零件完全由您使用草图编辑器创建的基础特征的二维轮廓定义；因此，当您从草图编辑器返回零件模块时，不能使用零件模块工具添加特征。您只能通过编辑原始草图来修改零件。

创建分析刚体表面后，必须为其分配刚体参考点。您通过约束或指定参考点的运动来控制分析刚体表面的运动。更多信息，请参见"参考点"，第11.8.1节。![](../graphics/images/black4rule.gif)有关相关信息，请单击以下项目：- "绘制简单对象"，第20.10节
