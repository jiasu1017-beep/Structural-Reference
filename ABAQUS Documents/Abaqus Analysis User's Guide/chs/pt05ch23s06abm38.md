### 概述

低循环疲劳损伤模型：
- 适用于高应变、低循环次数的疲劳分析；
- 可与塑性模型结合使用；
- 基于应变范围参数；以及
- 支持疲劳寿命预测。

### 疲劳损伤机理

低循环疲劳损伤发生在材料承受高应变循环时。损伤累积导致疲劳裂纹的形成和扩展，最终导致失效。

### 与塑性模型的结合

低循环疲劳损伤模型可与各向同性硬化塑性模型结合使用。

| **输入文件用法：** | ``` [*PLASTICITY](../key/key-link.md#usb-kws-mplasticity) [*FATIGUE](../key/key-link.md#usb-kws-mfatigue) ``` |

### 元素

低循环疲劳损伤模型可与 Abaqus/Standard 和 Abaqus/Explicit 中的连续体元素和壳元素一起使用。
