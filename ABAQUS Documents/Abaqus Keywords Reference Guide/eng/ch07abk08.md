# *GASKET BEHAVIOR







### *GASKET BEHAVIORBegin the specification of a gasket behavior.

This option is used to indicate the start of a gasket behavior definition.

**Products: **Abaqus/Standard  Abaqus/CAE  

**Type: **Model data  

**Level: **Model  

**Abaqus/CAE: **Property module

##### **Reference:**

- ["Defining the gasket behavior directly using a gasket behavior model," Section 32.6.6 of the Abaqus Analysis User's Guide](../usb/usb-link.md#usb-elm-egasketbehavior)

### **Required parameter: **

NAME

Set this parameter equal to a label that will be used to refer to the behavior when it is referenced in the [*GASKET SECTION](ch07abk11.md) option. Gasket behavior names in the same input file must be unique.

**There are no data lines associated with this option.**
