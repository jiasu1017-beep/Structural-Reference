# 47.4.1 理解如何对保存的X–Y数据对象进行运算





### 47.4.1 理解如何对保存的X–Y数据对象进行运算

您可以通过对[之前保存的](pt05ch47s02hlb07.md) *X–Y*数据对象进行运算来派生新的*X–Y*数据。您通过构建数学表达式来定义新数据。表达式可以包括之前保存的*X–Y*数据对象的名称、内置函数和数学运算符。例如：`currentMax("XYData-1")+2.5`。Abaqus/CAE计算表达式以派生新的*X–Y*数据。

要构建表达式，您使用[**Operate on XY Data**](pt05ch47s03hlb01.md)对话框。此对话框列出表达式可用的数据对象名称和运算符。您可以点击选择列表中的数据对象和运算符；使用键盘输入值；并使用标准的鼠标和键盘编辑技术（如退格、复制和粘贴）来配置表达式。表达式可以包含任何语法有效的支持操作序列。以下语法规则适用：
- 函数的多个参数必须用逗号分隔。
- 数据对象名称必须用引号括起来。
- 必须使用括号来分组函数参数。

括号也可用于控制数学计算或增强可读性。如果您的表达式包含无效语法或由于数学原因无法计算（例如，如果计算需要除以零），Abaqus/CAE会通知您。

**提示：**当您在**History Output**对话框中选择多个变量并点击**Save As**时，您可以选择在保存之前对*X–Y*数据应用其中一个更常见的运算符。

默认情况下，Abaqus/CAE执行检查以验证您创建的数学表达式。如果您的表达式包含大量*X–Y*数据点，此验证可能非常耗时。您可以切换**Skip checks**以跳过这组检查程序。
