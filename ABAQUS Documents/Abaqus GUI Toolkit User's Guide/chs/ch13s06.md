# 13.6 小结





- 如果准静态分析以其自然时间尺度执行，解应该与真正的静态解几乎相同。
- 通常需要使用荷载速率缩放或质量缩放来使用更少的 CPU 时间获得准静态解。
- 加载速率通常可以提高一些，只要解不局部化。如果加载速率提高太多，惯性力会对解产生不利影响。
- 质量缩放是增加加载速率的替代方案。当使用速率相关材料时，质量缩放是优选的，因为增加加载速率会人为改变材料属性。
- 在静态分析中，结构的最低模式主导响应。知道最低自然频率，相应地，知道最低模式的周期，您可以估计获得适当静态响应所需的时间。
- 可能需要以不同的加载速率运行一系列分析，以确定可接受的加载速率。
- 在大多数过程中，变形材料的动能不应超过内能的一小部分（通常为 5% 到 10%）。
- 在准静态分析中，使用带有 DEFINITION=SMOOTH STEP 参数的 [*AMPLITUDE](../key/key-link.md#usb-kws-mamplitude) 选项是规定位移的最有效方法。
- 将模型从 Abaqus/Explicit 导入到 Abaqus/Standard 以执行有效的回弹分析。
