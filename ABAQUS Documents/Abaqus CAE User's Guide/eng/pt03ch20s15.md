# 20.18 Repairing short edges, gaps, and overlaps







This section describes how to use the Sketcher tools to repair short edges in a sketch and to remove gaps and overlaps from a sketch. These tools are intended for the repair of imported sketches, but you can use them to repair geometry in a native sketch as well. You should remove the gaps and overlaps in your sketch before repairing its short edges.
