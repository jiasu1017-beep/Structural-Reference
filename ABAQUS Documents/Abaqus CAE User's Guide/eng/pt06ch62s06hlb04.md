# 62.6.4 Creating a datum point at a specified distance from two edges







You can position a datum point by selecting two edges on the model and entering the distance from the datum point to each edge. 

![](../graphics/dtm-hpted-nls.png)

**To create a datum point by specifying the distance from two edges:**

1. From the main menu bar, select ****Tools**![](../graphics/images/arrow.gif)**Datum****. The **Create Datum** dialog box appears. The dialog box outlines the types of datum geometry you can create. Abaqus/CAE displays prompts in the prompt area to guide you through the procedure. **Tip:**You can also create a datum point using the ![](../graphics/ico_dtmPtOffsetEdges.png) tool, located in the module toolbox. For a diagram of the datum tools in the toolbox, see ["Using the Datum toolset," Section 62.2](pt06ch62s02.md).
2. From the list of types at the top of the dialog box, choose **Point**. The **Method** list indicates the methods you can use to create a datum point.
3. From the **Method** list, select **Offset from 2 edges**.
4. From the part or assembly in the current viewport, select the plane on which to create the datum point.
5. Select a straight edge on the specified plane and enter the distance of the datum point from this edge.
6. Select a second straight edge on the plane and enter the distance of the datum point from this edge. The datum point appears. You can modify the position of the datum point by selecting ****Feature**![](../graphics/images/arrow.gif)**Edit**** from the main menu bar and modifying the distance from the datum to each edge.
