# 5.7 Abaqus 复合材料压溃模拟的 CZone 增强





**产品：** Abaqus/Explicit

**优势：** Abaqus 的 CZone 插件功能已增强，可在 Abaqus/Explicit 中为复合材料压溃建模提供更完整的解决方案。

**说明：** Abaqus 的 CZone 是一款结合了复合材料压溃 CZone 方法和 Abaqus/Explicit 分析功能的插件产品。这种组合使得在受到冲击的复合材料结构的 FEA 模拟中包含压溃行为成为可能。Abaqus 的 CZone 中的局部加载行为使用面-面接触相互作用建模。

Abaqus 的 CZone 功能已增强，可在 Abaqus/Explicit 中为复合材料压溃模拟提供更完整的解决方案。增强功能包括扩展 Abaqus 功能覆盖范围以及用户界面和执行过程的重大改进：
- 现在支持使用在各层上定义的方向或使用分布定义的复合壳截面。
- 定义和报告压溃接触状态不再需要场变量。
- 材料压溃应力的定义新界面。
- CZone 压溃接触相互作用的定义新界面。
- CZone 压溃状态的新输出变量。
- 简化执行过程，无需使用预编译的 Python 脚本（`fileReader.py`）来运行 Abaqus 的 CZone 分析。
