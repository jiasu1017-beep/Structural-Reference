# 9.7.11 将当前模型数据库保存为不同名称的新文件





从主菜单栏选择****文件（File）**![](../graphics/images/arrow.gif)**另存为（Save As）****将当前模型数据库保存为不同名称的新文件。如果您在当前会话期间从模型中删除了项目，使用****文件（File）**![](../graphics/images/arrow.gif)**另存为（Save As）****可能会减小文件大小（有关压缩文件的更多信息，请参阅["压缩当前模型数据库的文件大小，"第9.7.12节"](pt02ch09s06hlb12.md))。从出现的**另存模型数据库为（Save Model Database As）**对话框中，输入模型数据库的新名称，然后单击**确定（OK）**。如果您省略文件扩展名，Abaqus/CAE会将`.cae`附加到文件名。请参阅["保存当前模型数据库，"第9.7.9节"](pt02ch09s06hlb09.md)，了解使用相同名称保存模型数据库的信息。

使用****文件（File）**![](../graphics/images/arrow.gif)**另存为（Save As）****并使用相同的文件名不会减小文件大小。

**注意：**您不能使用名称`abaqus`来保存模型数据库。
