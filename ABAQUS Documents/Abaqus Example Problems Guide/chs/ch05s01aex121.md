# 5.1.5 排气歧管中的传导、对流和辐射热传递

**产品：** Abaqus/Standard   

### 目标

本示例演示了以下 Abaqus 功能和技巧：
- 计算排气歧管中的稳态热传递，
- 比较使用空腔辐射和平均温度辐射条件的辐射热传递公式的结果，以及
- 使用薄膜条件模拟从排气废气的对流热传递。

### 应用描述

发动机排气歧管中的热传递受三个效应支配：通过金属的传导、从热排气的对流以及金属表面不同部分之间的辐射交换。本示例说明了计算歧管在这些效应下的平衡热状态的计算。

### 材料

歧管由灰铸铁铸造而成，导热系数为 4.5 x 10^2 W/mm/C，密度为 7.8 x 10^(-9) kg/mm^3，比热为 460 J/kg/C。歧管在分析开始时的初始温度为 20 C。

### 边界条件和载荷

热排气在内部管表面产生热通量。在本示例中，此效应使用基于表面的薄膜条件建模，恒定温度为 816 C，薄膜条件为 500 x 10^6 W/mm^2/C。在连接到气缸盖的法兰表面上施加 355 C 的温度边界条件，在连接到排气的法兰表面上施加 122 C 的温度边界条件。

### Abaqus 建模方法和模拟技术

歧管管内部表面之间的辐射传递使用几种方法建模进行比较：通过空腔辐射方法，有无空腔并行分解启用，以及通过平均温度辐射条件。

### 分析案例总结

| 案例 1 | 稳态热传递，带薄膜和辐射效应；辐射使用空腔辐射方法建模，未启用并行分解。 |
| --- | --- |
| 案例 2 | 稳态热传递，带薄膜和辐射效应；辐射使用空腔辐射方法建模，启用了并行分解。 |
| 案例 3 | 稳态热传递，带薄膜和辐射效应；辐射使用平均温度辐射条件建模。 |

### 结果讨论

使用空腔辐射方法（左）和平均温度辐射条件（右）的歧管中的平衡温度场。空腔辐射方法获得的峰值温度更高。

### 输入文件

- heattransfermanifold.inp - 使用平均温度辐射条件的分析输入数据
- heattransfermanifold_cavity.inp - 使用空腔辐射方法（无并行分解）的分析输入数据
- heattransfermanifold_cavity_parallel.inp - 使用空腔辐射方法（启用了并行分解）的分析输入数据
