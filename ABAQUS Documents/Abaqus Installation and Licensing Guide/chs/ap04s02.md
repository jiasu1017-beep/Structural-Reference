# D.2 Windows 客户端安装





您可以在已挂载的网络驱动器上安装 Abaqus，并配置用户的计算机将 Abaqus 作为客户端运行。请参阅 Dassault Systmes Knowledge Base 中的"Windows 客户端安装"，网址为 [www.3ds.com/support/knowledge-base](http://www.3ds.com/support/knowledge-base)。

您可以使用此技术配置多台计算机从中央服务器运行 Abaqus，而无需在每台计算机上安装 Abaqus。客户端计算机必须能够访问网络上包含 Abaqus 安装的目录以及 Abaqus 许可证服务器。
