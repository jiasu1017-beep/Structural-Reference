# 30.1 流体动力学分析概述





Abaqus/CFD提供了先进的计算流体动力学（CFD）功能，用于执行流体动力学分析，可用于解决不可压缩流动问题。不可压缩流体动力学分析可能涉及层流或湍流、热对流问题以及变形网格分析。更多信息，请参见["不可压缩流体动力学分析"，Abaqus Analysis User's Guide第6.6.2节](../usb/usb-link.md#usb-anl-aifluiddyn)。

在Abaqus/CAE中创建流体模型的过程涉及以下常规步骤：

1. 创建新模型数据库时，选择**CFD**模型类型以指定您正在建模Abaqus/CFD分析。然后，Abaqus/CAE界面中呈现的大部分功能将被过滤，仅显示对Abaqus/CFD分析有效的功能。更多信息，请参见["指定模型属性"，第9.8.4节](pt02ch09s07hlb04.md)。
2. 在Part模块中，创建定义流体域（流体的几何区域）的流体部件。更多信息，请参见["建模流体域"，第30.2节](pt04ch30s02.md)。
3. 在Property模块中，定义模型中的流体材料属性。更多信息，请参见["定义流体材料属性"，第30.3节](pt04ch30s03.md)。
4. 在Property模块中，为模型创建和分配流体截面。流体截面决定哪些材料可以存在于流体部件中。更多信息，请参见["创建均匀流体截面"，第12.13.13节](pt03ch12s13hlb09.md)和["为多孔介质创建流体截面"，第12.13.14节](pt03ch12s13hlb10.md)。
5. 在Assembly模块中，创建流体部件的实例。
6. 在Step模块中，定义流体分析步骤、求解器控制和湍流模型。更多信息，请参见["配置流动过程"中的"配置常规分析过程"，第14.11.1节](pt03ch14s11s01.md#usi-sim-configure-flow)。
7. 在Step模块中，创建场和历史输出请求。更多信息，请参见["Abaqus/CFD输出变量标识符"，Abaqus Analysis User's Guide第4.2.3节](../usb/usb-link.md#usb-out-hcfdoutputvar)。
8. 在Load模块中，定义作用于流体模型的任何载荷、边界条件或预定义场（参见["为流体模型指定预定条件"，第30.4节](pt04ch30s04.md)）。
9. 在Mesh模块中，为流体部件创建网格。更多信息，请参见["为流体模型划分网格"，第30.5节](pt04ch30s05.md)。
10. 在Job模块中，创建新作业并提交流体分析。更多信息，请参见["运行流体分析"，第30.6节](pt04ch30s06.md)。
