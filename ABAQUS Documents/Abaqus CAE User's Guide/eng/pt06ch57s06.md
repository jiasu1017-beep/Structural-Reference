**To define an amplitude curve using periodic data:**

1. Display the **Edit Amplitude** dialog box as described in ["Selecting an amplitude type to define," Section 57.3](pt06ch57s03.md).
2. Click the arrow to the right of the **Time span** field, and specify how you want to define the amplitude as a function of time: - Select **Step time** for time that is measured from the beginning of each step. - Select **Total time** for total time accumulated over all non-perturbation analysis steps.
3. In the **Circular frequency** field, enter ![](../graphics/usi_eqn00122.gif), the circular frequency, in radians per time.
4. In the **Starting time** field, enter ![](../graphics/usi_eqn00778.gif).
5. In the **Initial amplitude** field, enter ![](../graphics/usi_eqn00779.gif), the constant term in the Fourier series.
6. In the data table, enter values for **A**, the coefficient of the cosine terms, and **B**, the coefficient of the sine terms. For detailed information on how to enter data, see ["Entering tabular data," Section 3.2.7](pt01ch03s02s07.md).
7. Click **OK** to save your amplitude definition and to close the **Edit Amplitude** dialog box.
