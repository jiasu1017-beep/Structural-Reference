# 19.6 重启分析





如果您的模型包含多个步骤，您不必在单个分析作业中分析所有步骤。 实际上，通常希望分阶段运行复杂分析。 这允许您检查结果并确认分析是否按预期执行，然后再继续下一阶段。 Abaqus分析生成的重启文件允许您从指定步骤继续分析。 有关更多信息，请参阅["重启分析，" Abaqus Analysis User's Guide第9.1.1节](../usb/usb-link.md#usb-anl-arestart)。
