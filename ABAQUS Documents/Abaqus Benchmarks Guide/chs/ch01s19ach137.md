# 1.19.5 使用 XFEM 的水力驱动断裂扩展

**产品：** Abaqus/Standard

### 问题描述

此示例验证并说明了在 Abaqus/Standard 中使用扩展有限元方法（XFEM）建模可渗透多孔介质中水力驱动裂纹扩展。呈现的结果与使用内聚力单元获得的结果进行了比较。

### 几何和模型

此示例中考虑的问题域是厚度为 1 m 的含油岩石圆形切片，建模了一个小的初始裂纹。域的直径为 160 m。

由于对称性，仅建模域的一半。[图 1.19.5-1](ch01s19ach137.md#bmk-hydraulicfracture-mesh) 显示了有限元模型。岩石用二维 CPE4P 单元建模。研究了相同几何的四种不同网格离散化。第二个网格的元素数量是原始网格的两倍，第三个网格的元素数量是原始网格的四倍，而第四个网格的元素数量是原始网格的八倍。使用 C3D8P、C3D8RP、C3D4P、CAX4P、CAX4RP 和 CPE4RP 单元执行了类似的分析。

执行两步分析，模拟裂纹扩展。执行地静步骤，在对地层施加初始孔隙压力和初始原地应力后达到平衡。下一步代表水力压裂阶段，其中主要体积的流体被注入井中。以每秒 5.0×10⁻⁶ m³ 的速率将流体注入模型中的目标地层，邻近井眼的富集单元被定义为最初开放以允许流体进入。此阶段持续 140 秒。

### 材料

富集单元中体积材料特性的材料数据为 ![](../graphics/bmk_eqn01267.gif) = 19.32 GPa，![](../graphics/bmk_eqn00114.gif) = 0.25。

指定了模型中富集单元的内聚力行为响应。选择最大主应力失效准则用于损伤起始，选择基于 BK 定律准则的基于能量的损伤演化规律用于损伤扩展。相关材料数据如下：![](../graphics/bmk_eqn01268.gif) = 5.28 MPa，![](../graphics/bmk_eqn01269.gif) = 5.47×10³ N/m，![](../graphics/bmk_eqn01270.gif) = 5.47×10³ N/m，![](../graphics/bmk_eqn01271.gif) = 5.47×10³ N/m，![](../graphics/bmk_eqn01272.gif) = 2.0。

在富集单元的断裂区同时建模切向和法向流动。指定以下参数：
- 间隙流动被指定为牛顿流体，黏度为 1×10⁻⁶ kPa·s，大约是水的黏度。
- 流体泄漏被指定为 5.879×10⁻¹⁰ m/(kPa·s)。

### 结果与讨论

在第二步（泵送阶段）期间注入的流体启动并生长了从井眼向外延伸的裂纹。[图 1.19.5-2](ch01s19ach137.md#bmk-hydraulicfracture-fracturegeometry) 显示了 140 秒泵送期结束时的断裂几何。[图 1.19.5-3](ch01s19ach137.md#bmk-hydraulicfracture-fracturepressure) 显示了使用 XFEM 方法获得的不同网格离散化在裂纹口处孔隙压力随时间的变化，与使用内聚力单元获得的结果进行比较。此图清楚地表明流体流动已经稳定。使用 XFEM 方法获得的结果随网格细化而收敛，并与使用内聚力单元获得的结果良好一致。差异主要是因为使用 XFEM 方法时输出的是单元平均的孔隙压力，而使用内聚力单元时在节点处输出孔隙压力。[图 1.19.5-4](ch01s19ach137.md#bmk-hydraulicfracture-fractureleakoff) 说明裂纹口处泄漏流率的类似历史。

### 输入文件

[hydrfract_xfem_cpe4p.inp](../eif/hydrfract_xfem_cpe4p.inp)

使用 CPE4P 单元的 Abaqus/Standard 二维平面应变模型。

[hydrfract_xfem_cpe4p_2.inp](../eif/hydrfract_xfem_cpe4p_2.inp)

与 hydrfract_xfem_cpe4p.inp 相同，但元素数量加倍。

[hydrfract_xfem_cpe4p_3.inp](../eif/hydrfract_xfem_cpe4p_3.inp)

与 hydrfract_xfem_cpe4p.inp 相同，但元素数量翻两番。

[hydrfract_xfem_cpe4p_4.inp](../eif/hydrfract_xfem_cpe4p_4.inp)

与 hydrfract_xfem_cpe4p.inp 相同，但元素数量翻三番。

[hydrfract_xfem_cpe4rp.inp](../eif/hydrfract_xfem_cpe4rp.inp)

使用 CPE4RP 单元的二维平面应变模型。

[hydrfract_xfem_cax4p.inp](../eif/hydrfract_xfem_cax4p.inp)

使用 CAX4P 单元的轴对称模型。

[hydrfract_xfem_cax4rp.inp](../eif/hydrfract_xfem_cax4rp.inp)

使用 CAX4RP 单元的轴对称模型。

[hydrfract_xfem_c3d8rp.inp](../eif/hydrfract_xfem_c3d8rp.inp)

使用 C3D8RP 单元的三维模型。
