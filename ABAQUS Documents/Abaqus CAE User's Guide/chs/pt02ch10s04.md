# 10.4 理解IGES文件的内容





IGES中性文件格式是一种国际标准，允许您在Abaqus/CAE和其他CAD应用程序之间传输几何数据。您可以使用IGES格式文件导入和导出草图和零件。本节描述IGES格式以及在导入和导出IGES文件时可用的选项。涵盖以下主题：
- ["Abaqus/CAE中的IGES选项是什么？，"第10.4.1节"](pt02ch10s04s01.md)
- ["什么是IGES实体？，"第10.4.2节"](pt02ch10s04s02.md)
- ["IGES日志文件包含什么？，"第10.4.3节"](pt02ch10s04s03.md)
- ["导出到IGES文件，"第10.4.4节"](pt02ch10s04s04.md)

有关导入和导出IGES格式文件的详细说明，请参阅["从IGES格式文件导入零件，"第10.7.7节"](pt02ch10s07hlb07.md)和["导出几何、模型和网格数据，"第10.9节"](pt02ch10s08.md)。
