# *CYCLED PLASTIC





### *CYCLED PLASTIC为*ORNL模型指定循环屈服应力数据。

此选项用于指定ORNL本构模型的第十循环屈服应力和硬化值。此选项仅在使用了[*ORNL](ch15abk02.md)选项时才相关。

**产品：**Abaqus/Standard  Abaqus/CAE  

**类型：**模型数据  

**级别：**模型  

**Abaqus/CAE: **属性模块

##### **参考：**

- [*ORNL](ch15abk02.md)
- ["ORNL -- 橡树岭国家实验室本构模型，" Abaqus 分析用户指南第 23.2.12 节](../usb/usb-link.md#usb-mat-cornl)

**此选项没有关联的参数。**

### **用于指定循环屈服应力数据的数据行：**

**第一行：**

根据需要重复此数据行，以定义屈服应力对塑性应变的依赖性，以及在需要时对温度的依赖性。
