# 28.2 在Abaqus/CAE中组装欧拉-拉格朗日耦合模型





Abaqus允许您创建同时包含欧拉部件实例和拉格朗日部件实例的模型。在耦合欧拉-拉格朗日分析过程中，拉格朗日网格与欧拉部件中的材料相互作用。耦合欧拉-拉格朗日分析通常比纯欧拉分析能更好地解释接触条件，特别是对于流体和固体材料之间的相互作用。在后处理过程中，耦合欧拉-拉格朗日分析中的固体拉格朗日体似乎比纯欧拉分析中的类似物体能更好地保持其形状。

[图28-2](pt04ch28s02.md#usi-eulerian-compare)比较了纯欧拉分析（顶部）和耦合欧拉-拉格朗日分析（底部）中一块钢砖穿过水柱的情况。

**图28–2** 纯欧拉分析（顶部）和耦合欧拉-拉格朗日分析（底部）的比较。

![](../graphics/usi-eulerian-compare.png)

在欧拉分析中，水倾向于粘附在砖的侧面，并且砖在最终构型中看起来发生了变形。这种明显的变形是欧拉材料使用的材料体积分数计算的结果，如《Abaqus分析用户指南》第14.1.1节"欧拉分析"中的"材料界面"所述。另一方面，拉格朗日砖在穿过欧拉水时保持其形状，水流绕过砖流动。

耦合欧拉-拉格朗日分析还允许您利用两种分析技术的优势；例如，您可以使用穿过欧拉材料的拉格朗日物体上的载荷来驱动拉格朗日物体的详细子模型。

要在Abaqus/CAE中组装耦合欧拉-拉格朗日模型，只需在同一装配中实例化欧拉和拉格朗日部件。耦合欧拉-拉格朗日分析只能在**Dynamic, Explicit**步骤中执行。您必须创建通用接触定义以启用拉格朗日部件和欧拉部件之间的接触。通用接触定义允许模型中拉格朗日表面和欧拉材料实例之间的相互作用（请参见["定义欧拉-拉格朗日模型中的接触，" 第28.3节](pt04ch28s03.md)，了解更多信息）。其他相互作用、载荷、边界条件和预定义场以常规方式应用于拉格朗日和欧拉部件。

在大多数情况下，拉格朗日部件组装在欧拉部件实例内部。虽然拉格朗日单元和节点可以重叠，但三维拉格朗日单元不能占据与欧拉材料实例相同的空间。因此，拉格朗日部件必须实例化在欧拉部件实例内的空隙区域（即没有材料分配的区域）。要建模完全被欧拉材料包围的三维拉格朗日部件实例，请使用体积分数工具创建欧拉材料分配字段，该字段包括对应于拉格朗日部件实例的空隙区域（请参见["体积分数工具，" 第28.5节](pt04ch28s05.md)，了解详细信息）。
