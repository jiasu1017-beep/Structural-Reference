### 全局标志

### **文本组件选项**

| **TEXT_READONLY** | 文本不可编辑。 |
| --- | --- |
| **TEXT_WORDWRAP** | 在单词边界处换行。 |
| **TEXT_OVERSTRIKE** | 重写模式。 |
| **TEXT_FIXEDWRAP** | 固定换行列。 |
| **TEXT_NO_TABS** | 为制表符插入空格。 |
| **TEXT_AUTOINDENT** | 自动缩进。 |
| **TEXT_SHOWACTIVE** | 显示活动行。 |

### **选择模式**

| **SELECT_CHARS** | 选择字符。 |
| --- | --- |
| **SELECT_WORDS** | 选择单词。 |
| **SELECT_LINES** | 选择行。 |
