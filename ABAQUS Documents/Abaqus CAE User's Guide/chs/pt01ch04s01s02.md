# 4.1.2 什么是箭头和文本注释？







箭头和文本注释是您在视口中创建的箭头和文本字符串，用于增强显示模型或结果的外观和清晰度。您可以独立创建箭头注释和文本注释，也可以一起创建箭头和文本以自动将文本定位在箭头末端。视口中注释的位置由锚点控制。您基于视口几何或模型坐标定义每个锚点；您选择的方法决定了 Abaqus/CAE 如何移动注释。如果您操作视口，Abaqus/CAE 会重新定位任何锚定到视口几何的注释；类似地，如果您操作模型，Abaqus/CAE 会移动任何锚定到模型的注释。[图 4-2](pt01ch04s01s02.md#vp-annotation) 显示了使用箭头和文本描述模型细节。

**图 4-2** 箭头和文本注释。

![](../graphics/vp-annotation.png)

注释编辑操作要求您首先选择一个或多个注释。使用**视口**工具栏中的**编辑注释**工具![](../graphics/ico_select.png)从当前视口中选择箭头或文本注释。Abaqus/CAE 会高亮显示所选的箭头或文本注释及其锚点，如[图 4-3](pt01ch04s01s02.md#vp-selected-ann)所示。

**图 4-3** 选中的注释：无偏移量的箭头、带偏移量的文本、两端有间隙的箭头以及尾部与其锚点之间有偏移量的箭头。

![](../graphics/vp-selected-arrow-nls.png)

锚点显示为一个小点，周围放置一个锚点符号。虚线表示锚点和注释之间的偏移；圆形「手柄」是锚点连接点——如果没有偏移，连接点和锚点是同一个点。

箭头注释有两个锚点（您可以对两个点使用相同的坐标）。您可以在箭头末端和连接点之间添加间隙。添加间隙类似于在 Sketcher 或 CAD 绘图中在尺寸线和对象线之间留出空间；它可以增加注释的清晰度。文本注释只有一个锚点。您可以通过在视口中拖动连接点或整个注释来更改偏移量。

不要将您可以创建的视口注释与 Abaqus/CAE 生成的视口注释混淆。生成的视口注释包括视图方向三轴组；3D 罗盘；以及在 Visualization 模块中的图例、标题块和状态块。您可以修改生成注释的某些显示方面，但不能修改其内容。有关更多信息，请参阅[第 56 章，「自定义视口注释」](pt05ch56.md)。相反，您完全控制与箭头和文本注释相关的所有属性，包括它们的颜色、线型、线宽、箭头、字体、锚点以及锚点和注释之间的任何偏移。

Abaqus/CAE 在模型和输出数据库中保存箭头和文本注释；但是，视口不会被保存。因此，数据库中的箭头和文本注释不与视口关联。当您随后打开包含注释的数据库时，必须使用**注释管理器**在当前视口中显示选定的注释。**注释管理器**还允许您将注释从模型数据库复制到输出数据库，反之亦然。您不能从 Visualization 模块在模型数据库中创建注释；如果要为模型创建注释，请在其他模块中打开模型数据库。
