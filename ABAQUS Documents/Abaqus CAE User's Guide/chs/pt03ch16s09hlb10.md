# 16.9.10 定义广义平面应变载荷





您可以创建广义平面应变载荷以定义应用于使用广义平面应变单元建模的区域的参考点的轴向载荷。有关更多信息，请参见Abaqus Analysis User's Guide第27.1.2节"Choosing the element's dimensionality"中的"Generalized plane strain elements"。

**创建或编辑广义平面应变载荷：**

1. 使用以下方法之一显示广义平面应变载荷编辑器：- 若要创建新的广义平面应变载荷，请遵循"创建载荷"，第16.8.1节中概述的过程（**类别**：`Mechanical`；**所选步骤的类型**：`Generalized plane strain`）。- 若要使用菜单或管理器编辑现有的广义平面应变载荷，请参见"编辑依赖于步骤的对象"，第3.4.13节。若要编辑应用载荷的区域，请参见"编辑应用规定条件的区域"，第16.8.4节。
2. 单击**分布**字段右侧的箭头，并从出现的列表中选择您的选项：- 选择**均匀**以定义在区域上均匀的载荷。- 选择分析场以定义空间变化载荷。只有对该载荷类型有效的分析场才会在选择列表中显示。或者，您可以单击![](../graphics/ico_analyticalField.png)创建新的分析场。（更多信息，请参见第58章"分析场工具集"。）
3. 在**轴向力**文本字段中，输入轴向力（单位[F](../popups/usb-int-iconventions-unitsym.md)）。
4. 在**关于X的力矩**字段中，输入在参考点处关于*X*轴施加的力矩。
5. 在**关于Y的力矩**字段中，输入在参考点处关于*Y*轴施加的力矩。
6. 如果需要，单击**幅值**字段右侧的箭头，并从出现的列表中选择您选择的幅值。或者，您可以单击![](../graphics/ico_amplitudeSmall.png)创建新的幅值。（更多信息，请参见第57章"幅值工具集"。）
7. 单击**确定**保存您的数据并退出编辑器。
