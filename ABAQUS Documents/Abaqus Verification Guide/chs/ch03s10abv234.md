# 3.10.9 孔隙-热模型变化







**产品：**Abaqus/Standard  

### 测试的单元

C3D8PT    C3D8RPT    

### 测试的功能

移除耦合孔隙-热单元。

### 问题描述

**模型：**

模型在*x*–*y*平面中的尺寸为40.0×40，平面外尺寸为5.0。

**材料：**

| 模量 | 88.7×10^9 |
| --- | --- |
| 密度 | 1922 |
| 膨胀系数 | 5.6×10^-6 |
| 比热容 | 3000 |
| 导热率 | 3 |
| 孔隙流体密度 | 1000 |
| 孔隙流体膨胀系数 | 69.×10^-6 |
| 孔隙流体比热容 | 4186 |
| 孔隙流体导热率 | 0.58 |
| 渗透率 | 0.001 |
| 流体比重 | 9800 |

**载荷和边界条件：**

在顶面施加10个单位的孔隙压力和2个单位的表面压力。顶面温度设置为100个单位。三个面上约束法向位移。块体中心的单元在各步骤中被移除和添加。

### 结果与讨论

模型产生的结果与预期值一致。
