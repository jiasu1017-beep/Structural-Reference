# 56.4 视口注释选项概述







使用视口注释选项自定义所有绘图状态的图例、标题块、状态块、三维指南针和视图三轴。此外，视口注释选项控制箭头和文本注释的显示。

**设置视口注释选项：**

1. 从主菜单栏，选择 ****视口**![](../graphics/images/arrow.gif)**视口注释选项****。出现 **视口注释选项** 对话框。它包含以下选项卡：- **常规**：控制所有视口注释的显示---由 Abaqus/CAE 生成的注释以及您创建的任何箭头和文本注释。- **三轴**：控制视图三轴的外观和位置。- **图例**：控制图例的外观和位置，以及是否显示最小值和最大值。- **标题块**：控制标题块的外观和位置。- **状态块**：控制状态块的外观和位置。
2. 使用 **常规** 选项卡控制当前视口中所有绘图的视口注释。您可以通过点击 **全部打开** 来显示当前视口中的所有视口注释，或通过点击 **全部关闭** 来隐藏当前视口中的所有视口注释。
3. 使用对话框中的其余选项卡自定义由 Abaqus/CAE 生成的视口注释。要创建和自定义箭头和文本注释，请参见 ["使用视口箭头和文本注释，" 第 4.4 节](pt01ch04s04.md)。
