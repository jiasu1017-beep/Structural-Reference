# 4.6.3 显示和自定义背景电影





你可以在视口背景中显示电影文件的全部或部分。背景电影仅出现在可视化模块中，当处于活动状态时，它们显示在视口背景图像之上。如果你打印视口时背景电影处于活动状态，Abaqus/CAE 会在打印图像的背景中显示电影的当前帧。

本节介绍如何使用**图像/电影选项**对话框显示图像。或者，你也可以从**结果树**中的**电影**容器或**电影管理器**对话框显示和自定义背景电影。要显示电影管理器，请选择****Tools**![](../graphics/images/arrow.gif)**Movie**![](../graphics/images/arrow.gif)**Manager****。

**要在视口中显示背景电影：**

1. 从主菜单栏中，选择****View**![](../graphics/images/arrow.gif)**Image/Movie Options****。在出现的**图像/电影选项**对话框中，单击**电影**选项卡。
2. 切换**动画期间显示电影**以在动画播放时显示或隐藏背景电影。
3. 选择要显示的电影文件：
   - 要显示为会话定义的电影，请展开**电影名称**列表并选择电影名称。
   - 要添加新电影，请单击![](../graphics/ico_movie.png)；然后在出现的对话框中输入名称并指定文件位置。你可以直接在**文件名**字段中输入文件位置，或单击![](../graphics/ico_fileOpenBlue.png)在**选择电影文件**对话框中导航到它。对于新添加的电影，**编辑电影**对话框会出现。
4. 如有需要，从**编辑电影**对话框自定义电影的激活帧：
   - 在电影的开始和结束处激活或取消激活帧。你可以拖动**激活帧**滑块，或在**开始**和**结束**字段中输入帧号来更改这些值。
   - 切换**在视口中预览更改**以在调整激活帧时动态更改背景电影的长度。当未选择此选项时，背景电影的激活帧只有在单击**应用**或**确定**后才会更改。
   - 切换**仅显示电影**以隐藏视口中的模型数据。在某些情况下，模型数据可能会遮挡背景电影，使电影编辑变得困难。
5. 如有需要，从**编辑电影**对话框自定义电影的时间线。如果你选择基于帧的动画作为背景电影，你可以更改电影的起始帧和结束帧；如果你选择基于时间的动画，你可以更改电影的开始时间和结束时间。要更改结束值，请展开**结束**列下的相应字段并选择**指定**。然后 Abaqus/CAE 允许你在此字段中输入值。
6. 单击**确定**应用更改并关闭**图像/电影选项**对话框。如有需要，你也可以自定义背景电影的比例、位置或透明度；请参阅["自定义背景图像和电影的外观，" 第 4.6.4 节](pt01ch04s06hlb03.md)。
