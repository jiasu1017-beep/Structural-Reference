# 3.2.8 自定义字体





**选择字体**对话框允许您自定义某些类型文本的字体；例如，您可以使用此对话框自定义视口注释中出现的字体。类似的对话框用于自定义可视化模块标签和标题的字体。

**选择字体**对话框允许您指定和预览以下内容：
- 比例字体或等宽字体。
- 字体系列。
- 字体大小（以磅为单位）。
- 常规、粗体或斜体字体。

可用选项取决于系统中安装的字体。

**自定义视口字体：**

1. 显示要自定义文本的**选择字体**对话框。有关更多信息，请参阅以下章节：
   - ["自定义X-Y绘图轴，" 第47.5节"](pt05ch47s04.md)
   - ["自定义X-Y绘图图例，" 第47.6.3节"](pt05ch47s05hlb02.md)
   - [第56章，"自定义视口注释"](pt05ch56.md)
   - ["设置标签字体，" 第55.5.1节"](pt05ch55s05hlb01.md)
2. 选择所需的字体和属性。所选字体的预览会出现在**选择字体**对话框的**示例**区域中。
3. 在**选择字体**对话框的**应用于**字段中，切换要将所选字体应用到的项目。**应用于**字段仅在有多个可应用字体的项目时才会出现。
4. 单击**确定**接受更改并关闭**选择字体**对话框。
