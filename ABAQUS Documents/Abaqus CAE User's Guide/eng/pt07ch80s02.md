# 80.2 Managing view cuts







The view cut manager allows you to create a view cut and to activate, reposition, edit, copy, rename, delete, or customize a previously created view cut. When you display view cuts for output database data in the Visualization module, the manager also enables you to activate and deactivate display of the resultant forces and moments for the displayed view cut.
