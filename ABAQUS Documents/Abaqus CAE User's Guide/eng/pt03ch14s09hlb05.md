# 14.9.5 Resetting the default values in the step editor







When you create, edit, or replace a step, you use the step editor to configure the analysis procedure settings. You can use the replace function to reset the settings in the step editor to their default values by replacing an existing step with a step of the same procedure type.

**To reset the default values for procedure settings:**

1. From the main menu bar, select ****Step**![](../graphics/images/arrow.gif)**Replace**![](../graphics/images/arrow.gif)***step name*****. The **Replace Step** dialog box appears with the current procedure highlighted in the list of available procedures. **Tip:**You can also select the step name in the **Step Manager** and click **Replace**.
2. Click **Continue**. The **Edit Step** dialog box appears with default values for the procedure settings.
3. Use the **Edit Step** dialog box to modify the settings from their default values and to provide values for optional settings. (For detailed help on a particular editor feature, select ****Help**![](../graphics/images/arrow.gif)**On Context**** from the main menu bar and then click the feature of interest.)
4. Click **OK**. Abaqus/CAE copies step-dependent objects to the new step and closes the **Edit Step** dialog box.
