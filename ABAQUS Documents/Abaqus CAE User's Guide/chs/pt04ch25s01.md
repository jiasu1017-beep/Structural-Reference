# 25.1 连续壳建模概述






您使用常规壳零件对厚度显著小于其他尺寸的结构进行建模，并在创建截面时在"属性"模块中定义厚度。相反，您将连续壳单元分配给实体零件，Abaqus 根据零件的几何体确定厚度。从建模角度来看，连续壳单元看起来像三维连续实体，但其运动学和本构行为与常规壳单元类似。例如，常规壳单元具有位移和旋转自由度，而连续实体单元和连续壳单元仅具有位移自由度。有关更多信息，请参见["壳单元：概述，" Abaqus 分析用户指南第 29.6.1 节](../usb/usb-link.md#usb-elm-eshelloverview)，和["选择壳单元，" Abaqus 分析用户指南第 29.6.2 节](../usb/usb-link.md#usb-elm-eshellelem)。[图 25-1](pt04ch25s01.md#adv-shellvsshellcontinuum) 展示了常规壳和连续壳单元之间的差异。

在三维空间中建模连续壳的一般过程涉及以下步骤：

1. 在"零件"模块中，定义实体几何体。
2. 在"属性"模块中，向将在"网格"模块中分配连续壳单元的任何实体区域分配壳截面。您必须指定壳截面的厚度；但是，Abaqus 仅使用此厚度来估算某些截面属性，例如沙漏刚度。Abaqus 在分析过程中使用基于单元节点几何体的实际厚度。如果实体区域的厚度沿其长度变化，您应该提供一个厚度的近似值。有关更多信息，请参见["使用在分析期间集成的壳截面来定义截面行为，" Abaqus 分析用户指南第 29.6.5 节](../usb/usb-link.md#usb-elm-eusingshellsection)。
3. 在"网格"模块中，查询网格堆积方向。如有需要，分配堆积方向，以使连续单元从堆叠底部到顶部一致对齐。有关更多信息，请参见["施加网格堆积方向，" 第 17.18.8 节"](pt03ch17s18hlb01.md)。
4. 在"网格"模块中，向区域分配连续壳单元类型，并用六面体或楔形单元对区域进行网格划分。这些是可用于形成连续壳网格的唯一单元。
