# 64.9.1 撤销或重做网格编辑操作







**Edit Mesh**（编辑网格）和 **Create Bottom-Up Mesh**（创建自底向上网格）对话框的 **Undo**（撤销）部分中出现的撤销和重做选项特定于所选部件、部件实例或装配。当您切换到不同项目并编辑其网格时，撤销选项会更改以反映所选项目的历史。Abaqus/CAE 更新按钮的状态及其描述以反映新选定的部件、部件实例或装配的历史。

**撤销或重做 Edit Mesh 或自底向上网格划分操作：**

1. 进入 Mesh（网格）模块。
2. 从主菜单栏中选择 ****Mesh**![](../graphics/images/arrow.gif)**Edit****（网格→编辑）。Abaqus/CAE 显示 **Edit Mesh**（编辑网格）对话框。**提示：**您也可以使用位于 Mesh 模块工具箱底部的 ![](../graphics/ico_meshEdit.png) 工具来显示 **Edit Mesh**（编辑网格）对话框。此对话框的 **Undo**（撤销）部分包括 **Undo**（撤销）和 **Redo**（重做）按钮，每个按钮旁边都有将撤销或重做的网格编辑操作的描述。
3. 点击 **Undo**（撤销）撤销指示的网格编辑操作，或点击 **Redo**（重做）重做指示的网格编辑操作。Abaqus/CAE 执行指示的更改。
4. 继续编辑网格或撤销或重做更改；完成后关闭 **Edit Mesh**（编辑网格）对话框。

**注意：**当您执行新的网格编辑操作或任何其他更改网格的操作时，受影响部件或部件实例的任何重做信息都将丢失。如果您删除区域网格、应用自顶向下网格划分或添加分区（如果添加的分区需要删除部分网格），撤销和重做信息也会被删除。
