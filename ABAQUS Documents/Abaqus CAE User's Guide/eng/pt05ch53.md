# 53 Viewing a ply stack plot







A ply stack plot shows the plies from the selected region of a composite model. You can create a ply stack plot in either the Property module or the Visualization module. This chapter describes ply stack plotting. The following topics are covered:
- ["Overview of ply stack plots," Section 53.1](pt05ch53hla01.md)
- ["Customizing ply stack plots," Section 53.2](pt05ch53s01.md)
