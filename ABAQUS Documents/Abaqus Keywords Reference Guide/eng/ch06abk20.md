# *FLUID DENSITY







### *FLUID DENSITYSpecify hydrostatic fluid density.

This option is used to define the reference fluid density for fluid cavities. It is applicable only for hydraulic fluids and should not be used for  pneumatic fluids and user-defined fluids. The [*FLUID DENSITY](ch06abk20.md) option can be used only in conjunction with the [*FLUID BEHAVIOR](ch06abk16.md) option.
