# 47.2.2 从输出数据库场输出读取X–Y数据









### 47.2.2 从输出数据库场输出读取X--Y数据

您可以从输出数据库中的场输出读取*X–Y*数据。可用数据由在分析期间保存的场输出组成。从这些输出中，您可以选择以下内容：
- 某输出位置处的变量——例如，积分点处的S11（11方向的应力）；以及
- 单一位置或一组位置（元素或节点）。

然后，Abaqus/CAE会在当前活动的步和帧处从输出数据库读取*X–Y*数据对。如果您选择多个变量或多个位置，Abaqus/CAE会读取数据对并为每个变量/位置对创建单独的*X–Y*数据对象。对于所有基于时间的分析，*X*值取为从分析开始的总时间。*Y*值取为指定位置的变量的相应值。

在节点处提取的基于元素的场数据受当前结果平均设置的控制。有关更多信息，请参见["理解结果值平均，" 第42.6.2节](pt05ch42s04s02.md)。

**要从输出数据库场输出读取X–Y数据：**

1. 找到**来自ODB场输出的XY数据**选项。从主菜单栏选择****工具**![](../graphics/images/arrow.gif)**XY数据**![](../graphics/images/arrow.gif)**创建****，然后选择**ODB场输出**。出现**来自ODB场输出的XY数据**对话框。**注意：****来自ODB场输出的XY数据**对话框列出了分析期间保存的所有场输出（除了四元数变量）。如果此列表为空，则场输出不可用。
2. 点击**位置**箭头以显示可为选择列出变量的可能位置；然后选择所需位置。变量列表会刷新，仅显示可以在所选位置读取的变量。
3. 使用变量列表中每个变量旁边的复选框或页面底部的**编辑**文本字段来选择要读取的场输出变量。**使用复选框方法：**1. 要选择变量及其所有分量，请点击该变量的复选框。2. 要在变量的各个分量中进行选择，请点击该变量复选框旁边的箭头以列出其分量；然后点击各个分量复选框以选择它们。**使用编辑方法：**在**编辑**文本字段中，输入要读取的变量和分量的名称。要使变量有效，它必须在当前位置上可用于输出数据库。
4. 点击**元素/节点**选项卡。出现**元素/节点**选项。
5. 从对话框左上角的**项目**列表中，选择**元素**或**节点**。Abaqus刷新对话框底部的**选择方法**列表和右侧的项目列表。
6. 选择要读取输出数据的特定元素或节点。- 要通过直接从视口中选择来指定元素或节点：1. 从**选择方法**列表中选择**从视口选择**。Abaqus/CAE进入选择模式。提示区域出现**为显示组选择*项目*（其中*项目*是元素或节点）**。2. 从视口中选择一个或多个项目（有关更多信息，请参见[第6章，"在视口中选择对象"](pt01ch06.md)）。项目在视口中高亮显示。**注意：**如果您点击提示区域中的**完成**，Abaqus/CAE会退出选择模式，您的选择会消失；您必须从列表中选择一个**选择方法**。- 要通过编号指定元素或节点：1. 从**选择方法**列表中选择**元素标签**或**节点标签**。出现**部件实例**和**标签**字段。2. 从**部件实例**字段的列表中选择要获取结果的部件实例的名称。在**标签**字段中输入用逗号分隔的元素或节点编号列表，或输入编号范围，如1:4。3. 通过点击**在视口中高亮显示项目**来验证您的选择。- 要通过集合名称指定元素或节点：1. 从**选择方法**列表中选择**元素集**或**节点集**。Abaqus刷新右侧的项目列表。如果此列表为空，则没有满足您选择条件的项目。**注意：**此列表不包括Abaqus/CAE生成的内部集合。2. 从项目列表中选择一个或多个集合名称。（有关更多信息，请参见["从列表和表格中选择多个项目，" 第3.2.11节](pt01ch03s02s11.md)。）如果您的模型包含许多集合，可以使用过滤器减少显示的集合名称数量。点击**过滤器**字段旁边的![](../graphics/ico_tipSmall.png)查看有效过滤语法的示例。3. 通过切换**在视口中高亮显示项目**来验证您的选择。- 要指定属于内部集合（Abaqus/CAE创建的集合）的元素或节点：1. 从**选择方法**列表中选择**内部集合**。Abaqus刷新右侧的项目列表。如果此列表为空，则没有满足您选择条件的项目。2. 从项目列表中选择一个或多个集合名称。（有关更多信息，请参见["从列表和表格中选择多个项目，" 第3.2.11节](pt01ch03s02s11.md)。）如果您的模型包含许多内部集合，可以使用过滤器减少显示的集合名称数量。点击**过滤器**字段旁边的![](../graphics/ico_tipSmall.png)查看有效过滤语法的示例。3. 通过切换**在视口中高亮显示项目**来验证您的选择。下表总结了**变量**页面上指定的输出位置对元素或节点数据可用性的限制：

| 输出位置 | 可用于 | | --- | --- | | 积分点 | 元素 | | 质心 | 元素 | | 元素节点 | 元素或节点 | | 唯一节点 | 节点 |
7. 点击**活动步/帧**以更改Abaqus/CAE从中读取*X--Y*数据的步或帧。有关更多信息，请参见["激活和停用步和帧，" 第42.4.1节](pt05ch42s02s01.md)。
8. 如果数据包含复数，可以使用**结果选项**对话框选择显示形式。有关Abaqus/CAE中复数形式的更多信息，请参见["控制复数结果的形式，" 第42.6.9节](pt05ch42s04hlb05.md)。
9. 要评估和显示数据，点击**绘图**。当前视口中出现*X--Y*图。该图表示您在对话框中配置的数据，Abaqus认为这是临时数据，无论您是否点击**保存**保存它。
10. 要保存您配置的数据，点击**保存**。任何复数数据都保存为当前的复数形式，而不是完整的复数。**注意：**要绘制您保存的*X--Y*数据，请从主菜单栏选择****工具**![](../graphics/images/arrow.gif)**XY数据**![](../graphics/images/arrow.gif)**绘图****，然后从弹出菜单中选择*X--Y*数据。
11. 完成后，点击**关闭**以关闭对话框。
