# 8.4 示例：非线性斜板





此示例是第 5 章"使用壳单元"中描述的线性斜板模拟的延续（参见[图 8-11](ch08s04.md#gss-skewplate)）。

**图 8-11** 斜板。

![](../graphics/gss-skewplate-nls.png)

您现在将在 Abaqus/Standard 中重新分析板以包含几何非线性的影响。这些分析结果将使您能够确定几何非线性效应的重要性，从而确定线性分析的有效性。

您只需要修改历史数据即可将此模型从线性模拟转换为非线性模拟。

如果愿意，您可以按照此示例末尾的指南进行扩展，以使用 Abaqus/Explicit 执行动力学分析。

以下步骤假定您可以访问此示例的完整输入文件。此输入文件 `skew_nl.inp` 可在"非线性斜板"第 A.6 节中找到。获取和运行脚本的说明在附录 A"示例文件"中给出。如果您希望使用 Abaqus/CAE 创建整个模型，请参阅《Abaqus 入门：交互版》第 8.4 节的"示例：非线性斜板"。

### 8.4.1 修改输入文件——历史数据**

此示例不会更改原始斜板示例中的任何模型数据；只包含历史数据更改。

**将 NLGEOM 应用于步**

在 [*STEP](../key/key-link.md#usb-kws-hstep) 选项上将 NLGEOM 参数设置为 YES，并删除 PERTURBATION 参数。这表明分析现在包含非线性几何效应。默认最大增量数为 100；Abaqus 可能使用的增量少于这个上限，但如果需要更多增量，它将停止分析。

修改后的 [*STEP](../key/key-link.md#usb-kws-hstep) 选项如下：

```
*STEP, NLGEOM=YES
```

您可能还希望修改步的描述以反映这是非线性分析。

**定义步时间**

此分析需要在 [*STATIC](../key/key-link.md#usb-kws-hstatic) 选项上添加一行数据，用于指定分析的初始时间增量 ![](../graphics/gsk_eqn00142.gif) 和模拟的总步时间。使用 1.0 的总步时间，并指定 ![](../graphics/gsk_eqn00142.gif) 使得 Abaqus 在第一个增量中施加 10% 的载荷。因此，完整的 [*STATIC](../key/key-link.md#usb-kws-hstatic) 选项块为

```
*STATIC
0.1, 1.0
```

**输出控制**

在线性分析中，Abaqus 求解平衡方程一次并计算此单一解的结果。非线性分析可以产生更多输出，因为可以在每个收敛增量结束时请求结果。如果您没有仔细选择输出请求，输出文件会变得非常大，可能占满您计算机的磁盘空间。

如前所述，输出有四种不同文件：
- 输出数据库（`.odb`）文件，包含以中性二进制格式的数据，用于使用 Abaqus/Viewer 后处理结果；
- 数据（`.dat`）文件，包含所选结果的打印表格；
- 重启（`.res`）文件，用于继续分析；和
- 结果（`.fil`）文件，用于第三方后处理器。

这里只讨论输出数据库（`.odb`）和打印输出（`.dat`）文件的选项。如果选择得当，您可以在模拟过程中频繁保存数据而不会过多占用磁盘空间。

从输入文件中删除现有的输出请求；并添加以下输出选项，确保在非线性分析期间仅保存选定的输出。

为减小输出数据库文件的大小，在 [*OUTPUT](../key/key-link.md#usb-kws-houtput)、FIELD 选项上使用 FREQUENCY 参数；在本模拟中，字段输出每两个增量写入一次。因此，在 [*OUTPUT](../key/key-link.md#usb-kws-houtput) 选项上将 FREQUENCY 参数设置为 2：

```
*OUTPUT, FIELD, FREQUENCY=2, VARIABLE=PRESELECT
```
参数 VARIABLE=PRESELECT 表示将一组合选的常用字段变量写入输出数据库（`.odb`）文件。如果您只对最终结果感兴趣，请将 FREQUENCY 参数设置为一个大数。结果总是在每个步结束时存储，无论 FREQUENCY 参数的值如何；因此，使用大值只会保存最终结果。

请求将中点节点（节点集 `MIDSPAN`）的位移保存到输出数据库文件。这些结果稍后将用于演示 Abaqus/Viewer 中的 X-Y 绘图功能。对于此历史输出请求，对输出数据库文件使用默认 FREQUENCY 值（FREQUENCY=1）。这里，[*NODE OUTPUT](../key/key-link.md#usb-kws-hnodeoutput) 选项必须紧跟在历史输出请求之后。请记住，当限制要请求输出的模型子集时，使用 NSET 或 ELSET 参数；否则，将使用默认子集，即整个模型。

最后，请求打印模型中所有节点的支反力（RF）以及中点（节点集 `MIDSPAN`）节点的位移（U）。您将需要两个 [*NODE PRINT](../key/key-link.md#usb-kws-hnodeprint) 选项，因为您请求了两个不同模型子集的结果。同样，使用 FREQUENCY 参数来减少输出量；每两个增量打印一次数据。

新的输出请求选项块列表如下：

```
*OUTPUT, FIELD, FREQUENCY=2, VARIABLE=PRESELECT
*OUTPUT, HISTORY, FREQUENCY=1
*NODE OUTPUT, NSET=MIDSPAN
U,
*NODE PRINT, NSET=MIDSPAN, FREQUENCY=2
U,
*NODE PRINT, SUMMARY=NO, TOTALS=YES, FREQUENCY=2
RF,
```
最后，使用 [*END STEP](../key/key-link.md#usb-kws-hendstep) 选项完成步定义。
```
*END STEP
```

### 8.4.2 运行分析

将修改后的输入存储在名为 `skew_nl.inp` 的文件中（示例输入文件列在"非线性斜板"第 A.6 节中）。使用以下命令运行分析：

```
abaqus job=skew_nl interactive
```

### 8.4.3 结果

在非线性分析期间，两个额外的输出文件变得非常重要。它们是状态文件（`skew_nl.sta`）和消息文件（`skew_nl.msg`）。随着分析的进行，Abaqus 会向这两个文件写入数据。您甚至可以在 Abaqus 继续分析时查看数据。您需要学习如何使用这些文件中的数据来评估 Abaqus 对模拟的进展。在某些情况下，您可能基于这些文件中的信息决定提前终止分析。更可能的情况是，您可能需要使用这些文件来了解导致 Abaqus 提前终止分析的原因；即导致收敛问题的是什么。

在此示例中，初始时间增量为 0.1，如输入文件中指定的。该增量的平均力为 12.2 N，而 ![](../graphics/gsk_eqn00152.gif) 具有相同的值，因为这是第一个增量。模型中的最大残差力 ![](../graphics/gsk_eqn00149.gif) 为 749 N，显然大于 0.005 × ![](../graphics/gsk_eqn00153.gif)。![](../graphics/gsk_eqn00149.gif) 发生在节点 1051 的自由度 1 处。Abaqus 还必须检查模型中力矩的平衡，因为此模型包含壳单元。力矩/旋转场也未能满足平衡检查。

虽然未能满足平衡检查足以导致 Abaqus 尝试另一次迭代，但您还应检查位移修正。在第一个步的第一个增量的第一次迭代中，最大位移增量 ![](../graphics/gsk_eqn00154.gif) 和最大位移修正 ![](../graphics/gsk_eqn00155.gif) 都是 5.576×10⁻³ m；最大旋转增量和旋转修正都是 1.598×10⁻² 弧度。由于在第一个步的第一个增量的第一次迭代中，增量值和修正值总是相等的，因此最大节点变量修正小于最大增量值 1% 的检查将始终失败。然而，如果 Abaqus 判断解是线性的（判断基于残差 ![](../graphics/gsk_eqn00149.gif) < 10⁸×![](../graphics/gsk_eqn00152.gif) 的大小），它将忽略此标准。

板变形为曲线形状：这种几何形状变化在非线性模拟中被考虑在内。因此，膜效应导致部分载荷通过膜作用承载，而不仅仅通过弯曲承载。这使得板更刚。此外，载荷（始终垂直于板表面）在板变形时开始在 1 和 2 方向上有分量。非线性分析包括这种刚化效应和压力方向变化的影响。线性模拟中都不包含这些效应。

线性和非线性模拟之间的差异足够大，表明对于这种特定载荷条件下的板，线性模拟是不充分的。

对于五自由度壳（如本分析中使用的 S8R5 单元），Abaqus 不输出节点处的总旋转。

**创建中点位移的 X-Y 图：**

1. 首先，仅显示名为 `PART-1--1.MIDSPAN` 的节点集中的节点：在结果树中，展开名为 `skew_nl.odb` 的输出数据库文件下的**节点集**容器。在名为 **PART-1-1.MIDSPAN** 的集上点击鼠标按钮 3，并从出现的菜单中选择**替换**。

### 8.4.5 在 Abaqus/Explicit 中运行分析
