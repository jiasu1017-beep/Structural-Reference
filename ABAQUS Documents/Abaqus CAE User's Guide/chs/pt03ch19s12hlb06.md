# 19.12.6 监视您的优化过程







您可以通过选择****优化**![](../graphics/images/arrow.gif)**监视器**![](../graphics/images/arrow.gif)***优化过程名称*****来监视优化过程运行时的进度。您也可以通过在**优化过程管理器**中选择优化过程名称，然后点击**监视器**来监视优化过程。此外，您可以通过在模型树中右键点击作业并选择**监视器**来监视先前设计循环期间完成的 Abaqus 作业。

优化过程监视器显示一个表格，显示每个设计循环后的优化值，如["理解优化过程生成的文件，"第 19.5.2 节](pt03ch19s05s02.md)中所述。默认情况下，表格仅包括目标函数和优化过程中的任何约束。当设计循环完成时，Abaqus/CAE 会向表格添加一行。您可以右键点击表格中的单元格并执行以下操作：
- 选择**隐藏**从表格中移除选定的列。
- 选择**编辑可见列**并选择要显示的变量。[表 19-2](pt03ch19s12hlb06.md#usi-ana-manageoptimization-categories)描述了可以显示的变量。**表 19-2** 变量类别 | 类别 | 描述 | | --- | --- | | 目标函数 | 目标函数的值——单个设计响应或设计响应和参考值的加权组合。 | | 目标函数-设计响应 | 目标函数使用的设计响应的值。 | | 目标函数项 | 设计响应及其参考值的加权值。 | | 约束 | 约束的值。 | | 变量 | 每个设计响应的值。 | 有关从加权设计响应和参考值组合形成目标函数的公式的更多信息，请参见["目标和约束"，《Abaqus 分析用户手册》第 13.2.2 节](../usb/usb-link.md#usb-anl-aoptobjectives)。

要查看显示每个设计循环后设计响应值变化的 *X–Y* 图，请点击优化过程监视器左上角的**绘图**按钮。*X–Y* 图出现在新的视口中，包括表格中列出的每个设计响应。您可以使用 *X–Y* 绘图选项自定义绘图的外观，如[第 47 章，"X–Y 绘图](pt05ch47.md)中所述。

表格或 *X–Y* 图中显示的数据是从文件 `optimization_status_all.csv` 和 `optimization_report.csv` 中读取的。优化模块必须在优化过程执行时写入这些文件。例如，您不能在优化过程执行时在 Microsoft Excel 等应用程序中打开这些文件。

优化过程监视器还显示以下信息：
- **日志**选项卡页面显示优化过程的开始和结束时间以及过程进度日志。
- **错误**和**警告**选项卡页面显示与优化过程关联的错误或警告。Abaqus/CAE 通过在**错误**和**警告**选项卡前添加感叹号来指示错误或警告的存在。作业监视器中出现的错误和警告消息的数量分别受环境参数 **cae_error_limit** 和 **cae_warning_limit** 的限制（详细信息请参见["作业自定义参数"，《Abaqus 安装和许可指南》第 4.1.3 节](../sgb/sgb-link.md#sgb-chp-env-jobcustom)）。如果错误或警告的数量超过优化过程限制，请查阅数据、消息或状态文件以获取完整的消息列表。
- **输出文件**选项卡页面显示每个输出数据条目写入输出文件的详细记录，包括状态、警告和错误消息。输出文件称为 `atom.out`，并存储在优化过程目录中（与优化过程同名的目录）。

要监视与优化过程关联的 Abaqus 分析作业，请点击优化过程监视器右上角的**监视器**按钮。（如果您的优化过程包含多个模型，优化模块会为每个模型创建一个分析作业，您必须首先选择要监视的所需作业。）Abaqus/CAE 显示作业监视器并继续为优化的每次迭代更新作业监视器。如果需要，您可以从作业监视器终止 Abaqus 分析作业，从而终止优化过程。有关更多信息，请参见["继续已终止的优化过程，"第 19.12.5 节](pt03ch19s12hlb05.md)和["监视分析作业的进度，"第 19.2.6 节](pt03ch19s02hlb01.md)。

优化过程向优化过程监视器输出大量信息，特别是**输出文件**选项卡页面。为了帮助您找到特定的文本字符串，您可以使用优化过程监视器下半部分的**搜索文本**字段。选择所需的选项卡文件，在**查找文本**字段中输入搜索字符串，然后点击**下一个**或**上一个**逐步查看输出中的匹配项。切换**区分大小写**以执行区分大小写的搜索。

优化过程监视器中显示的信息会随着优化的进展而不断更新。如果优化失败，**错误**选项卡页面会自动出现在其他选项卡页面前面，以帮助您确定失败的原因。此外，如果输出任何错误或警告消息，选项卡上会出现一个感叹号。

如果您开始监视优化过程，然后退出 Abaqus/CAE、关闭当前模型数据库或打开新的模型数据库，优化过程将继续；但您将无法再从 Abaqus/CAE 监视优化过程。
