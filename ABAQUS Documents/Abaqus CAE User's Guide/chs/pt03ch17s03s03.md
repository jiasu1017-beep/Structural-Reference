# 17.3.3 网格生成







Abaqus/CAE 可以使用多种网格划分技术对不同拓扑的模型进行网格划分。在某些情况下，您可以选择用于对模型或模型区域进行网格划分的技术。在其他情况下，只有一种技术有效。不同的网格划分技术提供不同程度的自动化和用户控制。Abaqus/CAE 中有两种网格划分方法可用：自顶向下和自底向上。

自顶向下网格划分通过从零件或区域的几何形状向下工作到各个网格节点和单元来生成网格。您可以使用自顶向下网格划分技术，使用任何可用的单元类型对一维、二维或三维几何形状进行网格划分。生成的网格完全符合原始几何形状。对几何形状的刚性一致使得自顶向下网格划分主要是自动化过程，但可能难以在具有复杂形状的区域上产生高质量的网格。

自底向上网格划分通过从二维实体（几何面、单元面或二维单元）向上工作来创建三维网格。您可以使用自底向上网格划分技术，仅使用全部或几乎全部六面体单元对实心三维几何形状进行网格划分。使用自底向上网格划分技术生成网格是一个手动过程，生成的网格可能与原始几何形状显著不同。然而，允许网格与几何形状不同可能允许您在具有复杂形状的区域上产生高质量的六面体网格。
