# 20.10.1 草绘孤立点







使用 Sketcher 工具箱中的点工具绘制单个孤立点。您可以使用生成的点作为参考，并且可以在该点和草图上的顶点之间创建尺寸。

**草绘孤立点：**

1. 从 Sketcher 工具箱中，选择点工具![](../graphics/ico_sketchPoint.png)。有关 Sketcher 工具箱中工具的图表，请参见["Sketcher 工具，"第 20.4.1 节](pt03ch20s04s01.md)。Abaqus/CAE 在提示区域显示提示以指导您完成该过程。
2. 点击所需的点位置。点会出现。
3. 要创建更多点，重复上一步。
4. 完成创建点后，执行以下操作之一：- 在 Abaqus/CAE 窗口中的任意位置点击鼠标按钮 2。- 在 Sketcher 工具箱中选择任何其他工具。- 点击提示区域中的取消按钮![](../graphics/afxI_cancel.png)。
