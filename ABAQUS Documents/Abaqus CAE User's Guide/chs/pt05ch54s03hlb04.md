# 54.3.4 选择自由体切割报告的选项和阈值





您可以创建会话中所有活动自由体切割的合力和力矩的表格报告。您可以通过在"**Free Body Cut Manager**"中切换其"**Show**"复选框来激活或抑制单个自由体切割;更多信息,请参阅 ["显示、隐藏和高亮自由体切割," 第67.4节](pt06ch67s04.md)。Abaqus/CAE报告相对于全局坐标系的合力 和力矩数据,无论"自由体切割"或"视图切割"的"**Component Resolution**"选项设置如何。

您可以为当前帧或多个帧生成报告。您还可以指定报告中力数据和力矩数据的自定义阈值;小于这些阈值的力和力矩在报告中报告为零。选择适当的阈值使您能够生成更清晰的报告数据;例如,将`1.234E-17`这样的小报告值替换为`0`使报告数据更易于阅读。力和力矩值的默认阈值都是`1E-006`。

**选择自由体切割报告的选项和阈值的步骤:**

1. 找到自由体切割报告的选项。从主菜单栏,选择 ****Report**![](../graphics/images/arrow.gif)**Free Body Cut****。
2. 选择Abaqus/CAE从中读取自由体切割数据的帧。 - 选择"**Specify**"使用当前步骤/帧。要更改当前步骤或帧,点击 ![](../graphics/ico_stepSmall.png)。更多信息,请参阅 ["选择结果步骤和帧," 第42.3节](pt05ch42s01.md)。 - 选择"**All active steps/frames**"使用所有活动步骤/帧。点击"**Active Steps/Frames**"更改Abaqus/CAE从中读取自由体切割数据的步骤或帧。更多信息,请参阅 ["激活和取消激活步骤和帧," 第42.4.1节](pt05ch42s02s01.md)。
3. 从"**File**"选项中,指定报告文件名。具体说明,请参阅 ["指定报告文件名," 第54.4节](pt05ch54hla01.md)。
4. 从"**Output Format**"选项中,指定报告数据的呈现格式。具体说明,请参阅 ["控制报告布局、宽度、格式和坐标系," 第54.5节](pt05ch54hla02.md)。
5. 如果需要,从"**Thresholds**"选项中指定力和力矩的阈值。Abaqus/CAE会将小于这些阈值的力和力矩值报告为零。

要生成报告,点击"**Apply**"。完成后,点击"**Cancel**"关闭对话框。
