# 17.18.12 在四面体网格边界添加楔形单元层





如果您对区域应用自由网格划分技术和四面体单元形状，**Mesh Controls**对话框中会出现**Insert boundary layer**切换按钮和**Assign Controls**按钮。

向四面体网格区域添加由多层楔形单元组成的边界层对Abaqus/CFD分析最有用。 边界层在边界壁处及其附近创建密网格，其中流体流动和温度深受与壁和周围条件的接触和热传递的影响。

如果您向多个网格划分单元添加边界层，Abaqus/CAE会临时固定这些单元之间任何内部面上的种子，以简化在这些面上创建边界层的过程。

**添加楔形单元边界层：**

1. 从主菜单栏，选择 ****Mesh**![](../graphics/images/arrow.gif)**Controls****。 （有关详细说明，请参阅["分配网格控制，" 第17.18.1节](pt03ch17s18s01.md)。） **注意：**您必须选择自由四面体网格才能包含楔形单元边界层。
2. 在**Mesh Controls**对话框底部附近切换**Insert boundary layer**，然后单击**Assign Controls**。 Abaqus/CAE显示**Boundary Layer**对话框。
3. 输入**Wall element height**。 壁单元高度设置沿边界壁的第一层楔形单元的高度或厚度。
4. 输入**Growth factor**。 从壁单元向内移动，每层的厚度由前一层的厚度乘以增长因子决定。 增长因子必须在`1.0`和`2.0`之间（含），其中`1.0`表示无生长，`2.0`使每层厚度加倍。
5. 输入边界层中楔形单元层的数量。 基于您在步骤3到步骤5中的输入，Abaqus/CAE显示总**Boundary layer thickness**。
6. 如果需要，切换**Inactive faces**，然后单击**Edit**选择不想包含在边界层中的面。 非活动面应该是近似平面的并且垂直于局部流动方向或沿着对称平面。 **注意：**默认选择方法基于您最近使用的选择方法。 要恢复为另一种方法，请单击提示区域右侧的**Select in Viewport**或**Sets**。
7. 如果需要，切换**Create set**并输入集名称以保存包含所有边界层单元的集。
8. 单击**OK**关闭**Boundary Layer**对话框。 边界层将在您对区域进行网格划分时创建。 如果由于边界层问题导致网格生成失败，Abaqus/CAE会显示边界层网格的预览以及警告对话框。 在关闭警告对话框并删除网格预览之前，寻找诸如尖锐角落附近自相交的层等问题； 可能的失败模式类似于["在网格偏移期间减少单元畸变和坍塌，" 第64.3.3节](pt06ch64s03s03.md)中描述的那些。 对边界层控制进行必要的更正，然后再次尝试对区域进行网格划分。
