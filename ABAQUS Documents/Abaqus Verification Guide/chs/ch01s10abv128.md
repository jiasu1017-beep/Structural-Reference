# 1.10.4 拖链单元









**产品：**Abaqus/Standard  

### 测试单元

B21    B33H    DRAG2D    DRAG3D    

### 问题描述

**模型：**

每个测试系统包含附着在梁单元上的拖链单元，梁单元的另一端完全约束。对于二维情况，B21单元在第二个节点处连接有DRAG2D单元。集中力在自由端*y*方向施加。为了测试三维情况，DRAG3D单元连接到B33H梁单元。在三维情况中，位于全局*x*–*y*平面的海底建模为刚性表面。

| DRAG2D： |
| --- |
| 摩擦极限 | 125.0 |
| 滑动时水平长度 | 0.5 |
| DRAG3D： |
| 链的总长度 | 131.0 |
| 摩擦系数 | 0.3 |
| 单位长度链的重量 | 4.0 |
| 位于海底的链长度 | 104.0 |
| 梁高于海底的高度 | 10.0 |

### 结果与讨论

计算的反作用力与施加的载荷一致：施加的力从链单元中的力和梁约束节点处的反作用力恢复。
