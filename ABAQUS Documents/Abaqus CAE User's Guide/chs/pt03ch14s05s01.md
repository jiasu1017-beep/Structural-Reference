# 14.5.1 Abaqus/Explicit的集成输出





要获取变量的历史输出，例如在接触外表面上求和的力或通过表面之间的绑定约束传递的力（请参见["Abaqus/Explicit中的集成输出"在"输出到输出数据库，"Abaqus分析用户指南第4.1.3节](../usb/usb-link.md#usb-out-odboutput-integrated)；此类输出仅适用于Abaqus/Explicit步骤），你必须引用集成输出截面来识别需要输出的表面。此外，集成输出截面定义可以提供一个局部坐标系来表达向量输出量和/或一个参考节点作为计算表面上总力矩的锚点。

默认情况下，集成输出截面锚定在全局原点，不跟随其上定义的表面的运动。你可以定义一个参考点，输出截面锚定在该点上，并指定此参考点如何跟随表面的平均运动。参考点不能连接到有限元模型的任何其他部分。

与坐标系和/或参考节点关联的集成输出截面可以独立于集成输出请求使用，以跟踪表面的平均运动。

你通过从主菜单栏选择****Output**![](../graphics/images/arrow.gif)**Integrated Output Sections**![](../graphics/images/arrow.gif)**Create****来定义集成输出截面。详细说明，请参见["定义集成输出截面，"第14.13.1节](pt03ch14s13hlb01.md)。有关为集成输出截面请求输出的信息，请参见["修改历史输出请求，"第14.12.3节](pt03ch14s12hlb03.md)。
