# 69.7.3 替换面







如果导入部件的面看起来没有正确形成，您可以用基于底层几何由 Abaqus/CAE 生成的新面替换这些面。因为 Abaqus/CAE 依赖底层几何，如果结果面非常凸起，您不应使用 **Replace**（替换）工具。如果您选择要替换多个面，这些面必须相连。

或者，您可以通过扩展相邻面来替换面。此工具对于从部件中移除凸台或小凸起很有用。

**替换面：**

1. 从主菜单栏中选择 ****Tools**![](../graphics/images/arrow.gif)**Geometry Edit****（工具→几何编辑）。Abaqus/CAE 显示 **Geometry Edit**（几何编辑）对话框。**提示：**您也可以使用位于 Part（部件）模块工具箱中编辑工具处的 ![](../graphics/ico_repairReplaceFace.png) 工具来替换面。 有关工具箱中编辑工具的图表，请参阅["使用几何编辑工具集，" 第 69.1 节"](pt06ch69s01.md)。
2. 从对话框中选择 **Face**（面）类别和 **Replace**（替换）方法。
3. 选择要替换的面。这些面必须相连。您可以从视口选择面或点击鼠标按钮 3 从临时选择组中粘贴面到您的选择中（有关详细信息，请参阅["什么是选择组？" 第 6.1.2 节"](pt01ch06s01s02.md)）。您可以使用拖动选择、**[Shift]****+点击**、**[Ctrl]****+点击** 和角度方法的组合来选择多个要替换的面。 有关更多信息，请参阅["在当前视口中选择对象，" 第 6.2 节"](pt01ch06s02.md)。**提示：**如果您无法选择所需的面，可以使用 **Selection**（选择）工具栏更改选择行为。 有关更多信息，请参阅["使用选择选项，" 第 6.3 节"](pt01ch06s03.md)。
4. 从提示区域切换开启 **Extend neighboring faces**（扩展相邻面）以用通过扩展相邻面形成的单个面替换所选的面。
5. 点击鼠标按钮 2 表明您已完成选择面。Abaqus/CAE 替换所选的面。
6. 如果您已完成编辑部件，您应该通过在模型树中的部件上点击鼠标按钮 3 并从出现的菜单中选择 **Update validity**（更新有效性）来检查其有效性。
