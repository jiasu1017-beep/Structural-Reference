# 53.2.5 Customizing the labels in a ply stack plot







Click the **Labels** tab to customize the text and symbols that are displayed in a ply stack plot; for example, material and ply names, the state block, and integration points.

**To customize the labels in a ply stack plot:**

1. Locate the labels options. 1. Click the **Ply Stack Plot Options** button from the prompt area, or do one of the following: - In the Property module, choose ****View**![](../graphics/images/arrow.gif)**Ply Stack Plot Options****. - In the Visualization module, choose ****Options**![](../graphics/images/arrow.gif)**Ply Stack Plot**** or click the ![](../graphics/ico_plotPlyStackOptions.png) tool in the toolbox. 2. Click the **Labels** tab in the **Ply Stack Plot Options** dialog box that appears.
2. Select the font of all the text in the ply stack plot. 1. Click **Set Font for All Labels** to display the **Select Font** dialog box. 2. Use the font selector dialog box to set the font properties that you want. 3. Click **OK** to close the **Select Font** dialog box and to return to the **Ply Stack Plot Options** dialog box.
3. Toggle on an option to display it.
4. Choose the color of the text: 1. Click the color sample ![](../graphics/usi-colorsample.png). Abaqus/CAE displays the **Select Color** dialog box. 2. Use one of the methods in the **Select Color** dialog box to select a new color. For more information, see ["Customizing colors," Section 3.2.9](pt01ch03s02s09.md). 3. Click **OK** to close the **Select Color** dialog box. The color sample changes to the selected color.
5. Click **Apply** to implement your changes. The ply stack plot changes to reflect your specifications. By default, your changes are saved for the duration of the session and will affect all subsequent ply stack plots in that viewport. If you want to retain your changes for subsequent sessions, save them to a file. For more information, see ["Saving customizations for use in subsequent sessions," Section 55.1.1](pt05ch55s01s01.md).
