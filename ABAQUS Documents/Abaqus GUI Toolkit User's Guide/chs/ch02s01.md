# 2.1 Abaqus分析模型的组成部分

Abaqus模型由多个不同组件组成，这些组件共同描述了要分析的物理问题和要获得的结果。分析模型至少包括以下信息：离散化几何、单元截面属性、材料数据、载荷和边界条件、分析类型和输出请求。本章的讨论集中在结构应用上。类似的概念适用于流体动力学。

**离散化几何**

有限元和节点定义了在Abaqus中建模的物理结构的基本几何。模型中的每个单元代表物理结构的一个离散部分，而物理结构又由许多互联的单元表示。单元通过共享节点相互连接。节点的坐标和单元的连通性——即哪些节点属于哪些单元——构成了模型几何。模型中所有单元和节点的集合称为*网格*。一般来说，网格只是结构实际几何的近似。

单元类型、形状和位置，以及网格中使用的单元总数，都会影响从模拟获得的结果。网格密度越大（即网格中单元数量越多），结果越准确。随着网格密度的增加，分析结果收敛于唯一解，并且分析所需的计算机时间增加。从数值模型获得的解通常是被模拟的物理问题解的近似。模型几何、材料行为、边界条件和载荷中近似的程度决定了数值模拟与物理问题的匹配程度。

**单元截面属性**

Abaqus有多种单元，其中许多单元的几何不能完全由节点坐标定义。例如，复合壳的层或工字梁截面的尺寸不是由单元节点定义的。这种额外的几何数据被定义为单元的物理属性，对于完全定义模型几何是必要的（请参阅[第3章"有限元和刚体"](ch03.md)"）。

**材料数据**

必须为所有单元指定材料属性。虽然高质量的材料数据通常难以获得，特别是对于更复杂的材料模型，但Abaqus结果的有效性受到材料数据准确性和完整性的限制。

**载荷和边界条件**

载荷使物理结构变形，从而在其中产生应力。最常见的载荷形式包括：
- 点载荷；
- 表面上的压力载荷；
- 表面上的分布牵引力；
- 壳边缘上的分布边缘载荷和力矩；
- 体力，如重力；和
- 热载荷。

边界条件用于约束模型的某些部分保持固定（零位移）或按指定量移动（非零位移）。

在静态分析中，必须使用足够的边界条件来防止模型在任何方向上作为刚体移动；否则，不受约束的刚体运动会导致刚度矩阵奇异。求解器问题将在求解阶段发生，并可能导致模拟过早停止。Abaqus/Standard如果在模拟期间检测到求解器问题，将发出警告消息。学会解释此类错误消息非常重要。如果在静态应力分析期间看到"数值奇异"或"零主元"警告消息，应检查模型的全部或部分是否缺乏对刚体平移或旋转的约束。刚体运动可以由组件的平移和旋转组成。潜在的刚体运动取决于模型的维度。

| 维度 | 可能的刚体运动 |
| --- | --- |
| 三维 | 沿1、2和3方向的平移。 |
|  | 绕1、2和3轴的旋转。 |
| 轴对称 | 沿2方向的平移。 |
|  | 绕3轴的旋转（仅限轴对称刚体）。 |
| 平面应力 | 沿1和2方向的平移。 |
| 平面应变 | 绕3轴的旋转。 |

默认情况下，1、2和3方向与全局笛卡尔坐标系的轴对齐（稍后讨论）。

在动态分析中，惯性力防止模型立即无限运动，只要模型中的所有独立部件都有一些质量；因此，动态分析中的求解器问题警告通常表示其他建模问题，例如过度塑性。

**分析类型**

Abaqus可以执行多种不同类型的模拟，但本指南仅涵盖两种最常见的类型：静态和动态应力分析。

在静态分析中获得结构对施加载荷的长期响应。在其他情况下，可能对结构对载荷的动态响应感兴趣：例如，组件对突然载荷（如碰撞期间发生的情况）的响应，或建筑物在地震中的响应。

**输出请求**

Abaqus模拟可以生成大量输出。为了避免使用过多磁盘空间，您可以将输出限制为解释结果所需的内容。
