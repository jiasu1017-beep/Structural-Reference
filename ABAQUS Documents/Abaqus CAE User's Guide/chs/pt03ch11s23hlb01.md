# 11.23.1 添加草图线框特征





从主菜单栏选择**形状**![](../graphics/images/arrow.gif)**线框**![](../graphics/images/arrow.gif)**草图**，向当前视口中的零件添加草图平面线框特征。无论当前视口中零件的建模空间如何，平面线框工具始终可用。

您通过在选定平面上绘制特征来添加平面线框特征。Abaqus/CAE移除与现有面重叠的任何线框部分。

**添加草图线框特征：**

1. 从主菜单栏，选择**形状**![](../graphics/images/arrow.gif)**线框**![](../graphics/images/arrow.gif)**草图**。
2. 如果需要，指定要用于为草图线框特征的草图选择原点的方法。
3. 如果零件的建模空间是二维或轴对称，Abaqus/CAE会进入草图编辑器并对齐零件和草图的*X*和*Y*轴。
4. 使用草图编辑器绘制平面线框。在提示区域，单击**完成**表示完成绘制。
