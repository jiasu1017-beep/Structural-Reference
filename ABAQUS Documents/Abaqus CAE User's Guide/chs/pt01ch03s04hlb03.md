# 3.4.11 使用管理器对话框复制步骤相关对象







您可以使用管理器对话框中的**复制**按钮来复制步骤相关对象，例如载荷、边界条件、相互作用、预定义场、输出请求或自适应网格约束。您可以将对象从任何步骤复制到任何其他有效步骤。

**复制步骤相关对象：**

1. 如[「使用管理器对话框管理对象，第 3.4.9 节」](pt01ch03s04hlb01.md)中所述，打开相应的管理器。

2. 通过点击相应的表格单元格来选择要复制的对象。

3. 点击**复制**。相应的**复制*对象***对话框出现。

4. 为新对象输入名称。

5. 从**步骤**列表中选择目标步骤。默认情况下，对象会被复制到创建它的步骤中。

6. 点击**确定**。对象被复制到选定的目标步骤。

如果目标步骤类型与创建对象的步骤类型不同，Abaqus/CAE 可能需要修改对象以使其与新步骤兼容。例如，如果在静态（通用）步骤中创建了载荷，而您将其复制到稳态动力学（线性扰动）步骤中，Abaqus/CAE 将向载荷值添加虚部分量。相反，如果将稳态动力学步骤中的载荷复制到静态步骤中，则会移除虚部分量。

此外，当您将对象复制到创建它的步骤以外的步骤时，在其传播状态之一中对原始对象进行的任何修改都将在副本中被忽略。

Abaqus/CAE 阻止您将对象复制到对象类型无效的步骤类型。Abaqus/CAE 还会阻止您复制被抑制的对象或复制到被抑制的步骤。
