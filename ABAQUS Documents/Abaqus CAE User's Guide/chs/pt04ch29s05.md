# 29.5 创建组合紧固件





您可以在开发模板模型后在相互作用模块中创建组合紧固件。更多信息，请参见["关于紧固件，" 第29.1节](pt04ch29s01.md)。

**创建组合紧固件：**

1. 从相互作用模块的主菜单栏中，选择****特殊**![](../graphics/images/arrow.gif)**紧固件**![](../graphics/images/arrow.gif)**创建****。Abaqus/CAE显示**创建紧固件**对话框。**提示：**您也可以使用相互作用模块工具箱中的![](../graphics/ico_fastener.png)工具创建组合紧固件。
2. 从出现的**创建紧固件**对话框中，选择**组合**紧固件，然后单击**继续**。
3. 选择主模型中将具有模板模型控制点映射到的附着点（或参考点）集。使用以下方法之一：
   - 使用现有集合定义点或多个点。此方法允许您使用预定义的附着点集合轻松选择大量点。在提示区域右侧，单击**集合**。从出现的**区域选择**对话框中选择现有的点集合，然后单击**继续**。**注意：**默认选择方法基于您最近使用的选择方法。要恢复为其他方法，请在提示区域右侧单击**在视口中选择**或**集合**。
   - 在视口中选择一个或多个点。**提示：**默认情况下，**选择**工具栏中的![](../graphics/ico_filterVisible.png)**选择距离屏幕最近的实体**工具处于关闭状态。如果进行模糊选择，Abaqus/CAE会高亮显示该点，并在视口左下角显示该点的描述。使用**下一个**和**上一个**按钮循环浏览可能的选项，然后单击**确定**确认您的选择。单击鼠标按钮2表示您已完成选择。更多信息，请参见["在当前视口中选择对象，" 第6.2节](pt01ch06s02.md)。如果模型包含孤立网格实例和本机几何部件实例的组合，请从提示区域单击以下之一：
   - 如果要从本机几何部件实例或一个或多个参考点中选择点，请单击**几何**。
   - 如果要从孤立网格实例中选择点，请单击**网格**。
4. 完成附着点选择后单击**完成**。Abaqus/CAE显示**编辑紧固件**对话框。
5. 选择要使用的模板模型。模板模型应包含您要在主模型中复制的类紧固件结构。模板模型必须包含在当前模型数据库（`.cae`文件）中，且模板模型必须包含一个或多个引用命名表面的绑定、耦合或调整点约束。
6. 从模板模型中选择控制点集。此集合必须包含紧固件结构的控制点。
7. 在**表面**、**方向**和**属性**选项卡页面上，按照以下过程配置紧固件的设置：
   - ["将模板模型表面映射到主模型表面，" 第29.5.1节](pt04ch29s05hlb01.md)
   - ["从模板模型读取的几何定向，" 第29.5.2节](pt04ch29s05hlb02.md)
   - ["修改主模型中的紧固件属性，" 第29.5.3节](pt04ch29s05hlb03.md)
8. 单击**确定**创建组合紧固件。视口中会出现浅绿色小方块，显示组合紧固件的位置。

创建组合紧固件后，您可以使用**重新运行**选项卡页重新运行属性生成脚本；请参见["重新运行属性生成脚本，" 第29.5.4节](pt04ch29s05hlb04.md)。
