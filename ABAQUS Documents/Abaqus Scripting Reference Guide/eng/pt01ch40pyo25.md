# 40.25 DetailPlotOptions object







The DetailPlotOptions object stores values and attributes associated with a [Viewport](pt01ch11pyo04.md) object. The DetailPlotOptions object has no constructor command. Abaqus creates the *detailPlotOptions* member whenever a [Viewport](pt01ch11pyo04.md) is created.

**Access**

```
session.viewports[*name*].detailPlotOptions
```

### 40.25.1 Members

The DetailPlotOptions object has the following member:

*plyStackPlotOptions*

A [PlyStackPlotOptions](pt01ch40pyo26.md) object.
