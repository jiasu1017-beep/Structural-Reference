# 82.4 使用 Really Simple GUI (RSG) Dialog Builder 创建对话框







### 82.4 使用 **Really Simple GUI (RSG) Dialog Builder** 创建对话框

使用 RSG 对话框构建器是使用 Abaqus GUI Toolkit 命令和文本编辑器创建对话框的替代方法。**Really Simple GUI (RSG) Dialog Builder** 插件使您能够创建对话框并将其连接到内核命令，而无需编写任何代码。您可以从工具箱中选择项目以添加到空对话框并编辑其属性。

RSG 对话框构建器提供了对 Abaqus GUI Toolkit 命令子集的访问，但制作可用的对话框不需要任何编程经验。您创建的对话框会成为 Abaqus/CAE 的新插件。您可以将它们保存为 RSG 插件或标准插件，但只能使用 RSG 对话框构建器编辑 RSG 插件。必须使用文本编辑器来编辑标准插件。但是，将 RSG 对话框保存为标准插件允许经验丰富的程序员从完整的 Abaqus GUI Toolkit 命令集中进行选择来扩展对话框的功能。

RSG 对话框构建器插件包含一个"五分钟tour"，描述并展示了一些可用于构建自定义对话框的基本功能。tour 图标 ![](../graphics/plg-rsgtour.png) 位于 **RSG Dialog Builder** 中 **GUI** 标签页的左上角。
