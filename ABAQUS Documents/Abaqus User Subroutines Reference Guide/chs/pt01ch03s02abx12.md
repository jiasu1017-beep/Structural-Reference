# 3.2.12 许可工具





**产品：** Abaqus/Standard  Abaqus/Explicit  Abaqus/CFD  

##### **参考**

- ["Abaqus执行程序：概述，" 第3.1.1节](pt01ch03s01abo02.md)

### 概述

**abaqus licensing**工具为两种类型的Abaqus许可（FLEXnet和Dassault Systmes许可）提供管理和监控工具。执行**abaqus licensing**命令而不带额外参数会显示所有可用工具的命令用法摘要。

有关所有FLEXnet许可工具的详细说明，请参阅FLEXnet许可最终用户指南版本11.6.1。您可以从[www.3ds.com/simulia](http://www.3ds.com/simulia)上的"支持"页面的"许可"部分下载此文档。

有关更多信息，请参阅[Abaqus安装和许可指南第3章"Abaqus许可"](../sgb/sgb-link.md#sgb-chp-net-license)。

### 命令行选项

**lmstat**

此选项显示有关用于提供Abaqus许可的FLEXnet许可服务器的位置和功能的信息。可以使用此命令的额外参数来生成更多许可使用信息。

**lmdiag**

此选项显示有关各种FLEXnet许可功能的信息，并指示该功能是否可以检出。

**lmpath**

此选项可用于控制Abaqus查找许可的位置。额外参数用于打印、设置或添加许可位置信息。运行不带参数的命令将显示每个操作的命令摘要。

**lmtools**

此选项在Windows平台上启动FLEXnet许可工具集。此应用程序可用于调用大多数FLEXnet许可管理工具功能。

**dslsstat**

此选项显示有关Dassault Systmes许可服务器(DSLS)的位置和功能的信息。有关更多信息，请参阅["对Dassault Systmes许可服务器使用`dslsstat`实用工具，" Abaqus安装和许可指南第3.9节](../sgb/sgb-link.md#sgb-chp-net-lic-dslsstat)。

**reporttool**

此选项用于从许可使用历史记录生成报告。请参阅["使用`reporttool`实用工具，" Abaqus安装和许可指南第3.10节](../sgb/sgb-link.md#sgb-chp-net-lic-dslsreport)。
