# 4.5.2 链接视口





你可以控制 Abaqus/CAE 会话中哪些视口将被链接，以及链接视口将共享的特性。某些链接视口选项在所有模块中都可用；有些选项（如在所有链接视口中显示相同的绘图状态）仅适用于可视化模块。

**要链接视口：**

1. 找到链接视口命令。从主菜单栏中，选择****Viewport**![](../graphics/images/arrow.gif)**Linked Viewports****。**链接视口**对话框打开。**提示：**你也可以通过单击[**视口**工具栏](pt01ch04s02hlb02.md)中的![](../graphics/ico_viewportLink.png)来显示**链接视口**对话框。
2. 切换**链接视口**以启用会话的视口链接。
3. 在对话框的**选项**部分：
   - 切换**视图操作**以在链接视口中启用同步视图操作。
   - 切换**绘图状态**以在链接视口中显示相同的绘图状态或状态。此选项仅适用于可视化模块。
   - 切换**绘图选项**以在链接视口中显示相同的绘图选项。此选项仅适用于可视化模块。
   - 切换**场输出**以在链接视口中显示相同场输出变量的结果。当你更改其中一个链接视口的场输出变量时，Abaqus/CAE 会在所有包含该场输出变量的输出数据库的视口中显示新选择的场输出变量。此选项仅适用于可视化模块。
   - 切换**旋转中心**以在所有链接视口中使用公共旋转中心。
   - 切换**视口标注选项**以在所有链接视口中相同方式显示视口标注。当你在**视口标注选项**对话框中显示、隐藏或自定义视口标注（如图例）的显示时，Abaqus/CAE 会将该更改应用到所有链接视口。
   - 切换**视图切割**以在所有链接视口中显示相同的视图切割。此选项仅适用于可视化模块。
   - 切换**显示组**以在所有链接视口中同时执行所选的显示组操作。如果你按名称或标签为显示组选择特定项目，相同的显示组操作会在所有共享指定项目的链接视口中执行。此选项仅适用于可视化模块。
4. 在对话框的**链接视口**部分，选择要链接的视口，并取消选择那些要保持独立的视口。
5. 单击**确定**应用更改并关闭**链接视口**对话框。链接视口会表现出你在**选项**选择中指定的共享行为。
