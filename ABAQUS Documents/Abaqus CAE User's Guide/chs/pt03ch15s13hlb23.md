# 15.13.26 定义执行器/传感器相互作用







您可以在模型的单个顶点上创建执行器/传感器相互作用。执行器/传感器相互作用提供对用户子程序[`UEL`](../sub/sub-link.md#sub-xsl-uel)的接口。该子程序反过来表示线性或非线性用户定义单元。执行器/传感器相互作用必须在初始步骤期间定义，且仅对Abaqus/Standard分析有效。

**警告：**此功能仅供高级用户使用。在最简单的测试示例之外的所有使用将需要用户/开发人员进行大量编码。在继续之前，应阅读Abaqus Analysis User's Guide的第32.15.1节"用户定义单元"。

**定义执行器/传感器相互作用：**

1. 从主菜单栏中，选择****相互作用**![](../graphics/images/arrow.gif)**创建****。**提示：**您也可以使用相互作用模块工具箱中的![](../graphics/ico_interaction.png)工具创建执行器/传感器相互作用。
2. 在出现的**创建相互作用**对话框中，执行以下操作：- 命名该相互作用。有关命名对象的信息，请参见第3.2.1节"使用基本对话框组件"。- 选择初始步骤。- 选择**执行器/传感器**类型的相互作用。
3. 单击**继续**关闭**创建相互作用**对话框。
4. 从装配件中选择将应用相互作用的点。单击鼠标按钮2以表示已完成选择点。**提示：**您可以通过在**选择**工具栏中指定过滤选项来限制可以在视口中选择的对象类型。更多信息，请参见"使用选择选项"，第6.3节。Abaqus/CAE显示**编辑相互作用**对话框。
5. 在**编辑相互作用**对话框中输入必要的数据。所需的数据是用户定义单元子程序的函数。您可能需要创建实数和整数执行器/传感器相互作用属性。更多信息，请参见第15.14.8节"定义执行器/传感器相互作用属性"。下表指示**编辑相互作用**对话框中的字段与用户子程序[`UEL`](../sub/sub-link.md#sub-xsl-uel)中变量之间的对应关系。| 字段 | UEL变量 | | --- | --- | | **用户单元类型ID** | `JTYPE` | | **自由度** | `NDOFEL` | | **坐标分量数量** | `MCRD` | | **依赖于解的状态变量** | `SVARS`和`NSVARS` | 输入执行器/传感器相互作用属性的实数和整数值由`PROPS`、`JPROPS`、`NPROPS`和`NJPROPS`变量使用有关可以传递到用户子程序[`UEL`](../sub/sub-link.md#sub-xsl-uel)的所有变量的说明，请参见Abaqus Analysis User's Guide的第32.15.1节"用户定义单元"。
