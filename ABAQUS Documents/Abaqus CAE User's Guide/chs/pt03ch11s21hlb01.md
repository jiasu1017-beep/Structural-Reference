# 11.21.1 添加拉伸实体特征





从主菜单栏选择**形状**![](../graphics/images/arrow.gif)**实体**![](../graphics/images/arrow.gif)**拉伸**，向当前视口中的零件添加拉伸实体特征。您只能向三维零件添加拉伸实体特征。

您通过绘制二维横截面并定义拉伸距离来添加拉伸实体特征。

您还可以通过选择要拉伸到的单个面来定义拉伸距离。Abaqus/CAE会拉伸草图直到遇到选定的面。

此外，您可以选择一个中心点并指定pitch，Abaqus/CAE在拉伸时使用它来扭曲横截面。或者，Abaqus/CAE可以在横截面被拉伸时根据指定的拔模角度扩展或收缩横截面。

**添加拉伸实体特征：**

1. 从主菜单栏，选择**形状**![](../graphics/images/arrow.gif)**实体**![](../graphics/images/arrow.gif)**拉伸**。
2. 如果需要，指定要用于为拉伸实体特征的草图选择原点的方法。
3. 选择实体将被拉伸的平面。如果不存在合适的面，可以选择基准平面或孤儿单元面。
4. 如果选择了**指定**作为**草图原点**方法，通过在视口中单击点或输入原点的三维坐标来指定原点位置。
5. 选择一条边以及边在草图编辑器网格上的方向。
6. 使用草图编辑器绘制拉伸的二维轮廓。单击**完成**退出草图编辑器并打开**编辑拉伸**对话框。
7. 在**编辑拉伸**对话框中单击以反转拉伸方向（如果需要）。
8. 选择以下终止条件之一：**盲孔**或**至面**。
9. 如果需要，执行以下操作之一：切换**包含扭曲**并输入pitch，或切换**包含拔模**并输入拔模角度。
10. 切换**保持内部边界**以保持拉伸实体特征与现有零件之间生成的任何面或边。
11. 单击**确定**拉伸轮廓。
12. 如果选择了**至面**，Abaqus/CAE会提示您选择要拉伸轮廓到的面。
