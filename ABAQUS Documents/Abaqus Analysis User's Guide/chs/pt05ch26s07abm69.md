### 概述

Abaqus中用户定义的机械材料行为：
- 通过一个接口提供，任何机械本构模型都可以添加到库中；
- 需要在本构模型（或模型库）中编程用户子程序 [`UMAT`](../sub/sub-link.md#sub-xsl-umat)（Abaqus/Standard）或 [`VUMAT`](../sub/sub-link.md#sub-xsl-vumat)（Abaqus/Explicit）；以及
- 需要大量的努力和专业知识：该功能非常通用和强大，但其使用不是常规练习。

### 应力分量和应变增量

子程序接口使用Cauchy应力分量（"真实"应力）实现。对于土问题，"应力"应解释为有效应力。应变增量由位移增量梯度的对称部分定义（等同于速度梯度的对称部分的时间积分）。

用户子程序 [`UMAT`](../sub/sub-link.md#sub-xsl-umat) 中应力和应变分量的方向取决于局部方向的使用（["方向，" 第2.2.5节](pt01ch02s02aus15.md)）。

在用户子程序 [`VUMAT`](../sub/sub-link.md#sub-xsl-vumat) 中，所有应变测量都是相对于中增量构型计算的。所有张量量都在随材料点旋转的共旋转坐标系中定义。为了说明这对应力意味着什么，考虑图26.7.1-1中所示的杆件，它从原始构型 ![](../graphics/usb_eqn04408.gif) 拉伸并旋转到新位置 ![](../graphics/usb_eqn04409.gif)。这个变形可以通过两个阶段获得；首先对杆件进行拉伸，如图26.7.1-2所示，然后通过对其施加刚体旋转来旋转杆件，如图26.7.1-3所示。

杆件在拉伸后的应力为 ![](../graphics/usb_eqn00053.gif)，并且该应力在刚体旋转过程中不会改变。由于刚体旋转而旋转的 ![](../graphics/usb_eqn04410.gif) 坐标系是共旋转坐标系。因此，应力张量和状态变量直接在用户子程序 [`VUMAT`](../sub/sub-link.md#sub-xsl-vumat) 中使用应变张量进行计算和更新，因为所有这些量都在共旋转系统中；这些量不需要像在用户子程序 [`UMAT`](../sub/sub-link.md#sub-xsl-umat) 中那样被用户子程序旋转。

由速率形式本构定律预测的弹性响应取决于所采用的客观应力率。例如，在 [`VUMAT`](../sub/sub-link.md#sub-xsl-vumat) 中使用Green-Naghdi应力率。然而，内置材料模型使用的应力率可能不同。例如，与Abaqus/Explicit中的实体（连续体）单元一起使用的大多数材料模型采用Jaumann应力率。只有当材料点的有限旋转伴随有限剪切时，这种公式差异才会导致结果的显著差异。有关Abaqus中使用的客观应力率的讨论，请参见["应力率，" Abaqus理论指南第1.5.3节](../stm/stm-link.md#stm-int-stressrates)。

### 材料常数

用户子程序 [`UMAT`](../sub/sub-link.md#sub-xsl-umat) 或 [`VUMAT`](../sub/sub-link.md#sub-xsl-vumat) 所需的任何材料常数都必须作为用户定义材料行为定义的一部分指定。同一材料定义中包含的任何其他机械材料行为（热膨胀和Abaqus/Explicit中的密度除外）都将被忽略；用户定义的材料行为要求所有机械材料行为计算都在子程序 [`UMAT`](../sub/sub-link.md#sub-xsl-umat) 或 [`VUMAT`](../sub/sub-link.md#sub-xsl-vumat) 中编程。在Abaqus/Explicit中使用用户定义材料行为时需要密度（["密度，" 第21.2.1节](pt05ch21s02abm01.md)）。

### Abaqus/Standard中的混合公式

如果将混合单元与用户子程序 [`UMAT`](../sub/sub-link.md#sub-xsl-umat) 一起使用，默认情况下，Abaqus/Standard将用从拉格朗日乘数推导的压力应力替换从用户子程序返回的应力张量计算的压力应力，并适当修改Jacobian（["混合不可压缩实体单元公式，" Abaqus理论指南第3.2.3节](../stm/stm-link.md#stm-elm-hybridincompress)）。这种方法适用于使用增量公式的材料模型（例如，金属塑性），但与超弹性材料常用的总公式不一致。在后一种情况下，默认公式可能导致收敛问题。例如，当几乎不可压缩的非线性弹性用户材料受到大变形时，可能会观察到这种收敛问题。Abaqus/Standard提供了一种替代的总公式，在这种情况下更合适。总公式与Abaqus用于超弹性材料的原生几乎不可压缩公式一致（["超弹性材料行为，" Abaqus理论指南第4.6.1节](../stm/stm-link.md#stm-mat-hyperelastic)），在这种情况下比默认（增量）公式效果更好。

Abaqus/Standard还为混合单元提供了一种完全不可压缩的公式，用于定义完全不可压缩的用户材料响应。完全不可压缩公式与Abaqus用于不可压缩超弹性材料的原生公式一致。对于总混合公式，假定材料的偏应力和体积响应是解耦的，体积响应可以从应变能势函数推导。Abaqus中的所有原生超弹性材料都使用这个假设。对于不可压缩混合公式，假定偏应力可以从应变能势函数推导。

总混合公式对几乎不可压缩的超弹性响应有用。材料的体积响应假定用替代变量 ![](../graphics/usb_eqn04412.gif) 而非体积变化 ![](../graphics/usb_eqn02869.gif) 来定义。替代变量在用户子程序 [`UMAT`](../sub/sub-link.md#sub-xsl-umat) 中可用。更多细节在["UMAT，" Abaqus用户子程序参考指南第1.1.41节](../sub/sub-link.md#sub-rtn-uumat)中讨论。

完全不可压缩公式要求您只需在 [`UMAT`](../sub/sub-link.md#sub-xsl-umat) 中定义应力张量的偏应部分和材料的Jacobian矩阵。Abaqus/Standard根据拉格朗日乘数自动计算压力应力。

| **输入文件用法：** | 使用以下选项调用总混合公式： |
| --- | --- |
|  | ``` [*USER MATERIAL](../key/key-link.md#usb-kws-musermaterial), TYPE=MECHANICAL, CONSTANTS=*number_of_constants*, HYBRID FORMULATION=TOTAL ``` 使用以下选项调用增量混合公式（默认）： ``` [*USER MATERIAL](../key/key-link.md#usb-kws-musermaterial), TYPE=MECHANICAL, CONSTANTS=*number_of_constants*, HYBRID FORMULATION=INCREMENTAL ``` 使用以下选项调用不可压缩混合公式： ``` [*USER MATERIAL](../key/key-link.md#usb-kws-musermaterial), TYPE=MECHANICAL, CONSTANTS=*number_of_constants*, HYBRID FORMULATION=INCOMPRESSIBLE ``` |

| **Abaqus/CAE用法：** | Abaqus/CAE不支持混合公式的规范。 |

### 材料状态

许多机械本构模型需要存储解依赖状态变量（塑性应变、"背应力"、饱和度等，以速率本构形式或以积分形式编写的理论的历史数据）。您应在关联的材料定义中为这些变量分配存储（参见["分配空间" in "用户子程序：概述，" 第18.1.1节](pt04ch18s01aus104.md#usb-anl-usubrout-allocatespace)）。与用户定义材料关联的状态变量数量没有限制。

用户材料子程序在每个增量的开始时提供材料状态，如下所述。它们必须返回新的应力和新的内部状态变量。与 [`UMAT`](../sub/sub-link.md#sub-xsl-umat) 和 [`VUMAT`](../sub/sub-link.md#sub-xsl-vumat) 关联的状态变量可以使用输出标识符SDV和SDV*n*输出到输出数据库文件（`.odb`）和结果文件（`.fil`）（参见["Abaqus/Standard输出变量标识符，" 第4.2.1节](pt02ch04s02abv01.md)，和["Abaqus/Explicit输出变量标识符，" 第4.2.2节](pt02ch04s02xbv01.md)）。

#### Abaqus/Standard中的材料状态

用户子程序 [`UMAT`](../sub/sub-link.md#sub-xsl-umat) 在每个增量的每次迭代中为每个材料点调用。它提供增量开始时的材料状态（应力、解依赖状态变量、温度和任何预定义场变量），以及温度、预定义状态变量、应变和时间增量。

除了将应力和解依赖状态变量更新到增量结束时的值外，子程序 [`UMAT`](../sub/sub-link.md#sub-xsl-umat) 还必须为机械本构模型提供材料Jacobian矩阵 ![](../graphics/usb_eqn04413.gif)。如果本构模型是速率形式并在子程序中数值积分，这个矩阵也将依赖于所使用的积分方案。对于任何非平凡的本构模型，这些都将是具有挑战性的任务。例如，Jacobian矩阵定义的准确性通常是解决方案收敛率的主要决定因素，因此，对计算效率有很强的影响。

如果您在频域中指定材料的粘弹性行为，用户子程序 [`UMAT`](../sub/sub-link.md#sub-xsl-umat) 还必须提供阻尼（损耗模量）对材料Jacobian矩阵的贡献，以及刚度（存储模量）的贡献。

#### Abaqus/Explicit中的材料状态

用户子程序 [`VUMAT`](../sub/sub-link.md#sub-xsl-vumat) 在每个增量中为材料点块调用。当调用子程序时，它提供增量开始时的状态（应力、解依赖状态变量）。它还提供增量开始和结束时的拉伸和旋转。[`VUMAT`](../sub/sub-link.md#sub-xsl-vumat) 用户材料接口在每次调用时将一块材料点传递给子程序，这允许材料子程序的向量化。

温度在增量的开始和结束时提供给用户子程序 [`VUMAT`](../sub/sub-link.md#sub-xsl-vumat)。温度作为信息传入，不能修改，即使在完全耦合的热应力分析中也是如此。但是，如果在与比热和电导率结合的完全耦合热应力分析中定义了非弹性热分数，Abaqus/Explicit将自动计算由于非弹性能量耗散产生的热通量。如果 [`VUMAT`](../sub/sub-link.md#sub-xsl-vumat) 用户子程序用于在显式动力学过程中定义绝热材料行为（塑性功转换为热），则必须为材料指定非弹性热分数和比热，并且必须将温度存储为用户定义的状态变量并对其进行积分。在大多数情况下，温度通过指定初始条件（["Abaqus/Standard和Abaqus/Explicit中的初始条件，" 第34.2.1节](pt07ch34s02aus116.md)）提供，并在整个分析过程中保持不变。

#### 使用状态变量从网格中删除单元

可以通过用户子程序 [`VUMAT`](../sub/sub-link.md#sub-xsl-vumat) 或 [`UMAT`](../sub/sub-link.md#sub-xsl-umat) 控制Abaqus分析过程中网格中单元的删除。已删除的单元无法承受应力，因此对模型刚度没有贡献。您可以指定控制单元删除标志的状态变量编号。例如，指定状态变量编号为4表示第四个状态变量是用户子程序中的删除标志。删除状态变量应设置为1或0。值为1表示材料点处于活跃状态，而值为0表示Abaqus应从模型中删除材料点，方法是将应力设为零。在Abaqus/Explicit中，传递给用户子程序 [`VUMAT`](../sub/sub-link.md#sub-xsl-vumat) 的材料点块的结构在分析过程中保持不变；已删除的材料点不会从块中移除。Abaqus/Explicit将为所有已删除的材料点传递零应力和零应变增量。一旦材料点被标记为删除，就无法重新激活。只有当单元中的所有材料点都被删除后，该单元才会从网格中删除。可以通过请求输出变量STATUS来确定单元的状态。当单元活跃时此变量等于1，当单元被删除时等于0。

| **输入文件用法：** | ``` [*DEPVAR](../key/key-link.md#usb-kws-mdepvar), DELETE=*variable number* ``` |
| --- | --- |

| **Abaqus/CAE用法：** | 属性模块：材料编辑器：****通用**![](../graphics/images/arrow.gif)**Depvar****：**控制元素删除的变量编号：****变量编号* |
| --- | --- |

### 沙漏控制和横剪刚度

通常，Abaqus/Standard中减缩积分单元的默认沙漏控制刚度和壳、管、杆单元的横剪刚度基于与材料相关的弹性（["截面控制，" 第27.1.4节](pt06ch27s01aus113.md)；["壳截面行为，" 第29.6.4节](pt06ch29s06alm18.md)；和["选择杆单元，" 第29.3.3节](pt06ch29s03alm08.md)）。这些刚度基于材料初始剪切模量的典型值，例如可能作为弹性材料行为（["线性弹性行为，" 第22.2.1节](pt05ch22s02abm02.md)）的一部分在材料定义中给出。但是，在预处理输入阶段，对于使用用户子程序 [`UMAT`](../sub/sub-link.md#sub-xsl-umat) 或 [`VUMAT`](../sub/sub-link.md#sub-xsl-vumat) 定义的材料，剪切模量不可用。因此，当使用 [`UMAT`](../sub/sub-link.md#sub-xsl-umat) 定义具有沙漏模式的单元的材料行为时，必须提供沙漏刚度参数（参见["抑制沙漏模式的方法" in "截面控制，" 第27.1.4节](pt06ch27s01aus113.md#usb-elm-esection-hourglass)）；当使用 [`UMAT`](../sub/sub-link.md#sub-xsl-umat) 或 [`VUMAT`](../sub/sub-link.md#sub-xsl-vumat) 定义具有横剪柔性的杆和壳的材料行为时，必须指定横剪刚度（参见["选择杆单元，" 第29.3.3节](pt06ch29s03alm08.md)，或["壳截面行为，" 第29.6.4节](pt06ch29s06alm18.md)）。

### 将 [`UMAT`](../sub/sub-link.md#sub-xsl-umat) 与其他子程序一起使用

在Abaqus/Standard中还提供了各种实用子程序，可与子程序 [`UMAT`](../sub/sub-link.md#sub-xsl-umat) 一起使用。这些实用子程序在["在Abaqus/Standard分析中获取应力不变量、主应力/应变值和方向，以及旋转张量，" Abaqus用户子程序参考指南第2.1.11节](../sub/sub-link.md#sub-utl-utensor)中讨论。

用户子程序 [`UMATHT`](../sub/sub-link.md#sub-xsl-umatht) 可以与 [`UMAT`](../sub/sub-link.md#sub-xsl-umat) 结合使用来定义材料的热本构行为。在材料定义中分配的可解依赖变量在 [`UMAT`](../sub/sub-link.md#sub-xsl-umat) 和 [`UMATHT`](../sub/sub-link.md#sub-xsl-umatht) 中都可以访问。此外，用户子程序 [`FRIC`](../sub/sub-link.md#sub-xsl-fric)、[`GAPCON`](../sub/sub-link.md#sub-xsl-gapcon) 和 [`GAPELECTR`](../sub/sub-link.md#sub-xsl-gapelectr) 可用于定义表面之间的机械、热和电气相互作用。

### 材料选项

当材料的机械行为由用户子程序 [`UMAT`](../sub/sub-link.md#sub-xsl-umat) 或 [`VUMAT`](../sub/sub-link.md#sub-xsl-vumat) 定义时，可以在材料定义中使用多种材料行为。这些行为包括密度、热膨胀、渗透率和热属性。热膨胀也可以作为 [`UMAT`](../sub/sub-link.md#sub-xsl-umat) 或 [`VUMAT`](../sub/sub-link.md#sub-xsl-vumat) 中实现的本构模型的组成部分。

[`UMAT`](../sub/sub-link.md#sub-xsl-umat) 中可用的温度始终是单元积分点处插值温度场。自然，如果热膨胀行为在 [`UMAT`](../sub/sub-link.md#sub-xsl-umat) 中实现，它将根据积分点温度定义。当温度场在单元内的插值方式与Abaqus/Standard中位移场的插值方式不同时，在 [`UMAT`](../sub/sub-link.md#sub-xsl-umat) 中实现热膨胀行为可能导致与内置热膨胀行为的差异。这种情况通常发生在耦合温度-位移单元中。例如，对于一阶耦合温度-位移单元，内置热膨胀行为在整个单元上使用恒定温度场（参见["完全耦合热应力分析，" 第6.5.3节](pt03ch06s05at19.md)），而 [`UMAT`](../sub/sub-link.md#sub-xsl-umat) 中的行为将根据线性温度场定义。

对于由用户子程序 [`UMAT`](../sub/sub-link.md#sub-xsl-umat) 或 [`VUMAT`](../sub/sub-link.md#sub-xsl-vumat) 定义的材料，可以单独包含质量比例阻尼（参见["材料阻尼，" 第26.1.1节](pt05ch26s01abm51.md)），但刚度比例阻尼必须在用户子程序中由Jacobian（Abaqus/Standard仅）和应力定义。如果用户材料用于直接稳态动力学过程，则不能指定刚度比例阻尼。

### 单元

用户子程序 [`UMAT`](../sub/sub-link.md#sub-xsl-umat) 和 [`VUMAT`](../sub/sub-link.md#sub-xsl-vumat) 可用于Abaqus中所有包含机械行为的单元（具有位移自由度的单元）。
