### 概述

非结构质量：
- 是来自几乎没有结构刚度的特征（如汽车金属板上的油漆）对模型质量的贡献；
- 可用于将模型中一个或多个部件的净质量提高到已知值；
- 可以是正的（向模型添加质量）或负的（从模型中移除质量），并在Abaqus/Explicit分析中相应地增加或减少单元稳定时间增量；
- 可以以非结构特征的总质量形式指定，分布在模型中的一个或多个部件上；
- 可以以在弥散区域上增加密度的形式指定；
- 可以以每单位面积质量的形式指定，应用于由壳、膜和/或表面单元组成的弥散区域上；以及
- 可以以每单位长度质量的形式指定，应用于由梁、管和/或桁架单元组成的弥散区域上。

### 非结构质量

非结构特征的质量贡献可以包含在模型中，即使这些特征本身被省略。非结构质量被弥散在通常与非结构特征相邻的单元集上。此单元集可以包含实体、壳、膜、表面、梁、管或桁架单元。非结构质量可以按以下形式指定：
- 总质量值，
- 每单位体积质量，
- 每单位面积质量（适用于包含传统壳、膜和/或表面单元的单元集），或
- 每单位长度质量（适用于包含梁、管和/或桁架单元的单元集）。

当将总质量分布在单元集区域上时，可以根据底层单元"结构"质量的比例或根据初始配置中单元体积的比例进行分布。

"结构"质量定义为非结构特征之外所有质量贡献对单元的总和。这可能包括由于与单元关联的任何材料定义而产生的质量；壳、膜和表面单元的截面上给出的任何"每单位面积"质量；包含在壳、膜和表面单元中的任何钢筋产生的质量；以及梁/管单元截面上给出的任何附加惯性。如果单元没有结构质量，则不允许对该单元进行非结构质量贡献。

模型中的给定单元可以具有多个非结构质量规范的贡献。给定单元中的非结构质量将参与该单元上定义的任何质量比例分布载荷，如重力载荷。当非结构质量添加到具有活动转动自由度的壳、梁或管单元时，非结构贡献会影响单元质量和单元转动惯量。正非结构质量会增加单元稳定时间增量，负非结构质量会减少单元稳定时间增量。通常，使用非结构质量定义将额外质量引入模型比使用一组点质量来做同样的事情更容易。由于可能有更高的时间增量，在Abaqus/Explicit分析中也更有利。

指定为材料定义一部分的任何质量比例阻尼（请参阅["材料阻尼，" 第26.1.1节](pt05ch26s01abm51.md)）也将适用于使用该材料定义分配给单元或单元集的非结构质量贡献。

### 定义非结构质量

要定义对模型质量的非结构质量贡献，首先必须确定要添加贡献的区域。然后使用适当的单位指定非结构质量的值，如果已知非结构特征的总质量，则确定非结构质量如何在区域上分布。

#### 指定非结构质量的单位

非结构质量可以根据包含在指定区域中的单元类型以不同类型的单位指定。

##### 指定质量单位

具有"质量"单位的总非结构质量可以分布在包含实体、壳、膜、梁、管和/或桁架单元的区域上。

| **输入文件用法：** | ``` [*NONSTRUCTURAL MASS](../key/key-link.md#usb-kws-mnonstructuralmass), UNITS=TOTAL MASS *total mass of the nonstructural feature* ``` |
| --- | --- |

| **Abaqus/CAE用法：** | 属性或相互作用模块：****特殊**![](../graphics/images/arrow.gif)**惯性**![](../graphics/images/arrow.gif)**创建****：**非结构质量**：选择区域：**单位：总质量**：**幅值：** *非结构特征的总质量* |
| --- | --- |

##### 指定每单位体积质量单位

具有"每单位体积质量"单位的非结构质量可以分布在包含实体、壳、膜、梁、管和/或桁架单元的区域上。

| **输入文件用法：** | ``` [*NONSTRUCTURAL MASS](../key/key-link.md#usb-kws-mnonstructuralmass), UNITS=MASS PER VOLUME *added density due to the nonstructural feature* ``` |

#### 控制非结构特征总质量的分布

当已知非结构特征的总质量时，有两种方法可用于在区域上分布非结构质量。

##### 按单元结构质量比例分布非结构质量

如果您不想改变区域的质心，请按单元结构质量的比例分布非结构质量。此方法导致区域结构密度的均匀缩放。Abaqus默认使用质量比例分布。

壳、膜和表面单元中的单元结构质量包括钢筋（如果钢筋定义为钢筋层）的任何质量贡献（请参阅["定义钢筋，" 第2.2.3节](pt01ch02s02aus13.md)）。

| **输入文件用法：** | ``` [*NONSTRUCTURAL MASS](../key/key-link.md#usb-kws-mnonstructuralmass), UNITS=TOTAL MASS, DISTRIBUTION=MASS PROPORTIONAL *total mass of the nonstructural feature* ``` |
| --- | --- |

| **Abaqus/CAE用法：** | 属性或相互作用模块：****特殊**![](../graphics/images/arrow.gif)**惯性**![](../graphics/images/arrow.gif)**创建****：**非结构质量**：选择区域：**单位：总质量**：**幅值：** *非结构特征的总质量*：**分布：质量比例** |
| --- | --- |

##### 按单元体积比例分布非结构质量

或者，您可以按初始配置中单元体积的比例分布非结构质量。此方法导致在区域上添加到底层结构密度的均匀值。因此，如果区域具有不均匀的结构密度，则区域的质心可能会改变。

| **输入文件用法：** | ``` [*NONSTRUCTURAL MASS](../key/key-link.md#usb-kws-mnonstructuralmass), UNITS=TOTAL MASS, DISTRIBUTION=VOLUME PROPORTIONAL *total mass of the nonstructural feature* ``` |
| --- | --- |

| **Abaqus/CAE用法：** | 属性或相互作用模块：****特殊**![](../graphics/images/arrow.gif)**惯性**![](../graphics/images/arrow.gif)**创建****：**非结构质量**：选择区域：**单位：总质量**：**幅值：** *非结构特征的总质量*：**分布：体积比例** |
