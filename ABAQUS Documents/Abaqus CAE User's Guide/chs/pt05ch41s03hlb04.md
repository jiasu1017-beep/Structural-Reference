# 41.3.4 残差





**作业诊断**对话框中的**残差**页面显示Abaqus/Standard用于确定迭代是否产生了平衡解的信息。

残差代表作用在模型上的内力和外力之间的差异。如果残差很小，Abaqus接受该迭代为收敛。用于确定解是否收敛的容差非常重要。容差必须足够小以提供准确的解，但足够大以在合理的迭代次数内获得解。在接受迭代为收敛之前，Abaqus还要求主解变量和约束方程兼容性误差的修正也必须很小。

当平衡迭代不收敛时，在最后迭代中发生最大残差的节点通常是开始查找问题的最佳位置。有许多条件可能阻止平衡迭代收敛；诊断问题源需要一定的经验。

包括每次迭代中最大残差值在内的残差信息以表格形式汇总，用于包含残差诊断的每次尝试。您在**方程**和**变量**字段中的选择控制表中的数据。您可以通过选择**绘制所选列**来绘制表中的列。当您找到有问题的迭代时，在**作业历史**树中选择它。然后，您可以从该迭代的**残差**页面选择项目，并单击**在视口中高亮显示选择**来在当前视口中查看区域。
