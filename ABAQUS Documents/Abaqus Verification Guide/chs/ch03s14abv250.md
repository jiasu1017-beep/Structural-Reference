# 3.14.11 在导入期间更改材料定义





**产品：**Abaqus/Standard  Abaqus/Explicit

### 测试的单元

CPE4R

### 问题描述

这里考虑的问题展示了在导入后更改材料定义和继续分析的能力。在Abaqus/Standard分析中使用具有Mises屈服准则的弹塑性材料。通过使用剪切失效模型引入延性失效模型，在Abaqus/Explicit中继续分析。使用CPE4R单元对横向双轴拉伸载荷下的棱柱形棒的方形截面进行建模。由于几何和载荷的对称性，仅对域的四分之一进行建模，如图3.14.11-1所示。

**图3.14.11-1** 材料更改验证的模型。

![](../graphics/exximport-matchange.png)

在Abaqus/Standard分析中，对象进行加载，使得域的一部分开始屈服。在Abaqus/Explicit分析中继续加载，使塑性应变进入失效状态。将Abaqus/Explicit分析的结果导入回Abaqus/Standard，以验证失效单元不会被导入。Abaqus/Standard中使用的材料性能如下：

| 弹性模量 = 207.8 109 |
| --- |
| 泊松比 = 0.3 |
| 密度 = 7800. |
| 屈服应力 = 1220. 106 |
| 流应力 = 1440. 106 当 ![](../graphics/ver_eqn01320.gif) 1.0 |

在Abaqus/Explicit中指定延性失效，使得当等效塑性应变达到0.8时失效开始，当等效塑性应变达到1时完全失效。在Abaqus/Standard中指定载荷，使得最大牵引力 ![](../graphics/ver_eqn00746.gif) 是初始屈服应力的2.5倍；在Abaqus/Explicit中增加到初始屈服应力的4倍。材料状态被导入，参考构型未被更新。

### 结果与讨论

这个问题展示了在导入后明智地更改材料定义和继续分析的灵活性。发现在Abaqus/Standard和Abaqus/Explicit分析之间，应变、应变和能量量（如可恢复弹性应变能）是连续的。失效单元不会从Abaqus/Explicit导入到Abaqus/Standard。
