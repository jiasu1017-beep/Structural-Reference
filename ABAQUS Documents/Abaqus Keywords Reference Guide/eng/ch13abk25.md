# *MOISTURE SWELLING







### *MOISTURE SWELLINGDefine moisture-driven swelling.

This option is used to define the moisture-driven swelling of the solid skeleton in a partially saturated porous medium. It can be used in the analysis of coupled wetting liquid flow and porous medium stress.

**Products: **Abaqus/Standard  Abaqus/CAE  

**Type: **Model data  

**Level: **Model  

**Abaqus/CAE: **Property module

##### **Reference:**

- ["Moisture swelling," Section 26.6.6 of the Abaqus Analysis User's Guide](../usb/usb-link.md#usb-mat-cmoistureswell)

**There are no parameters associated with this option.**
