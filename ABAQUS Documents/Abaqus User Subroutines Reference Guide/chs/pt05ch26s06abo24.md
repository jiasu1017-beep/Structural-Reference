# 26.6.1 孔隙流体流动属性

Abaqus/Standard允许为流体充满的多孔材料定义特定属性。这种类型的多孔介质在耦合孔隙流体扩散/应力分析中考虑（["耦合孔隙流体扩散与应力分析，" 第6.8.1节](pt03ch06s08at26.md)）。以下属性可用：

**渗透率**：渗透率定义液体通过多孔介质的流速与该流体水头梯度之间的关系（参见["渗透率，" 第26.6.2节](pt05ch26s06abm64.md)）。

**多孔体积模量**：定义多孔介质中固体颗粒和流体的体积模量，以便在分析中考虑其可压缩性（参见["多孔体积模量，" 第26.6.3节](pt05ch26s06abm65.md)）。

**吸附**：吸附定义多孔材料在部分饱和流动条件下的吸收/解吸行为（参见["吸附，" 第26.6.4节](pt05ch26s06abm66.md)）。

**膨胀凝胶**：膨胀凝胶模型用于模拟在部分饱和多孔介质中膨胀并捕获润湿液体的凝胶颗粒的生长（参见["膨胀凝胶，" 第26.6.5节](pt05ch26s06abm67.md)）。

**湿胀**：湿胀定义多孔介质固体骨架在部分饱和流动条件下的饱和驱动体积膨胀（参见["湿胀，" 第26.6.6节](pt05ch26s06abm68.md)）。

### 热膨胀

对于土壤或岩石等多孔介质，可以定义固体颗粒和渗透流体的热膨胀。更多详细信息，请参见["耦合孔隙流体扩散与应力分析"中的"热膨胀"，第6.8.1节](pt03ch06s08at26.md#usb-anl-acoupdiffstress-thermexpan)。
