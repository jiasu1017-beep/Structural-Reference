# 31.1.2 创建裂缝





您可以使用Interaction模块中的**Special**菜单在模型中定义裂缝。裂缝定义了可以在分析过程中打开的区域。生成网格时，Abaqus/CAE沿着裂缝放置重叠的重复节点。裂缝不能沿部件边界延伸，必须嵌入二维部件的面内或实体部件的单元内。当您选择将用于轮廓积分分析的裂缝前缘和裂缝尖端时，可以使用裂缝。更多信息，请参见["什么是裂缝？"，第31.1.1节](pt04ch31s01s01.md)。

**要创建裂缝：**

1. 从Interaction模块的主菜单栏中，选择****Special**![](../graphics/images/arrow.gif)**Crack**![](../graphics/images/arrow.gif)**Assign seam****。
2. 从视口中的模型中选择表示裂缝的实体。实体必须嵌入在二维部件的面内或实体部件的单元内的边缘；您不能选择位于部件边界上的任何实体。
3. 单击鼠标按钮2表示您已完成裂缝的选择。Abaqus/CAE创建裂缝。
4. 要将裂缝包含在轮廓积分分析中，请按照["创建轮廓积分裂缝"，第31.2.9节](pt04ch31s02hlb02.md)中所述的过程进行操作。
