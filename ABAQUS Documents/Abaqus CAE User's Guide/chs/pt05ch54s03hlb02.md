# 54.3.2 选择场输出变量





您可以选择一个或多个场输出变量包含在表格报告中。可用的变量由您保存到当前步骤和帧的输出数据库中的那些变量组成;此步骤和帧在"**Report Field Output**"对话框顶部识别。

Abaqus/CAE可以计算和报告给定变量在各种位置的值。可能的报告位置包括:积分点、形心、单元节点和唯一节点。

单元节点和唯一节点位置都涉及在模型节点处报告结果;但是,唯一节点值的报告在每个节点处只产生一个值,而单元节点值的报告在每个对该节点有贡献的单元处产生一个值。对于基于节点的变量(如位移),只提供唯一节点值。对于基于单元的变量(如应力),必须执行外推和平均以获得唯一节点值。平均阈值的使用以及跨区域和材料边界的平均控制将影响为基于单元的结果报告的单元节点值(更多信息,请参阅 ["选择结果选项," 第42.6节](pt05ch42s04.md))。下表总结了可以根据保存变量到输出数据库时使用的位置报告哪些输出位置:

|  | 报告位置 |
| --- | --- |
| 输出数据库位置 | 积分点 | 形心 | 单元节点 | 唯一节点 |
| 积分点 | 是 | 是 | 是 | 是 |
| 形心 | 否 | 是 | 是 | 是 |
| 单元节点 | 否 | 否 | 是 | 是 |
| 唯一节点 | 否 | 否 | 否 | 是 |

接触表面变量(如CNORMF、CSHEARF、CPRESS、COPEN、CSLIP、CSHEAR1和CSHEAR2)只能在单元节点位置报告。

有关选择报告输出变量的截面点位置的信息,请参阅 ["选择场输出截面点," 第54.3.3节](pt05ch54s03hlb03.md)。

**选择场输出报告数据的步骤:**

1. 找到场输出报告变量的选项。从主菜单栏,选择 ****Report**![](../graphics/images/arrow.gif)**Field Output****;然后在出现的对话框中点击"**Variable**"选项卡。
2. 点击"**Position**"箭头显示可以选择变量的可能位置列表;然后选择所需的位置。只有当在相同位置报告时,多个变量才能[组合成单个表格](pt05ch54hla02.md)。变量列表会刷新,仅显示可以在所选位置报告的那些变量。
3. 使用变量列表中每个变量旁边的复选框(此方法是默认方法)或页面底部的"**Edit**"文本字段来选择要包含在报告中的场输出变量。 **注意:**如果您要指定报告值的单个截面点,必须使用复选框方法。 **使用复选框方法:** 1. 要选择变量及其所有分量,点击该变量的复选框。 2. 要在变量的各个分量中进行选择,点击该变量复选框旁边的箭头列出其分量;然后点击各个分量复选框进行选择。 **使用编辑方法:** 在"**Edit**"文本字段中,输入要包含在报告中的变量和分量的名称。要有效,变量必须在输出数据库上可用并且可以在当前位置报告。

要生成报告,请在"**Report Field Output**"对话框中的"**Setup**"页面上配置剩余选项;然后点击"**Apply**"。完成后,点击"**Cancel**"关闭对话框。
