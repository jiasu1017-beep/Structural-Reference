# 12.11.2 Defining dielectric material properties







The **Edit Material** dialog box allows you to define the dielectric properties (electrical permittivity) of a fully constrained material for use in a coupled piezoelectric analysis. For more information, see ["Piezoelectric behavior," Section 26.5.2 of the Abaqus Analysis User's Guide](../usb/usb-link.md#usb-mat-cpiezoelect).

**To define dielectric properties:**

1. From the menu bar in the **Edit Material** dialog box, select ****Electrical/Magnetic**![](../graphics/images/arrow.gif)**Dielectric (Electrical Permittivity)****. (For information on displaying the **Edit Material** dialog box, see ["Creating or editing a material," Section 12.7.1](pt03ch12s07hlb01.md).)
2. Click the arrow to the right of the **Type** field, and specify the directional dependence of the dielectric constant.
3. Toggle on **Use temperature-dependent data** to define the dielectric constant as a function of temperature. A column labeled **Temp** appears in the **Data** table.
4. Click the arrows to the right of the **Number of field variables** field to increase or decrease the number of field variables on which the dielectric constant depends.
5. Enter the applicable data in the **Data** table: **Dielectric constant** Dielectric constant for isotropic behavior. (Units of [C![](../graphics/usi_eqn00317.gif)1L1](../popups/usb-int-iconventions-unitsym.md).) **D11, D22, and D33** Three values to define orthotropic behavior, ![](../graphics/usi_eqn00324.gif), ![](../graphics/usi_eqn00325.gif), and ![](../graphics/usi_eqn00326.gif). (Units of [C![](../graphics/usi_eqn00317.gif)1L1](../popups/usb-int-iconventions-unitsym.md).) **D11, D12, D22, D13, D23, D33** Six values to define anisotropic behavior, ![](../graphics/usi_eqn00324.gif), ![](../graphics/usi_eqn00327.gif), ![](../graphics/usi_eqn00325.gif), ![](../graphics/usi_eqn00328.gif), ![](../graphics/usi_eqn00329.gif), ![](../graphics/usi_eqn00326.gif). (Units of [C![](../graphics/usi_eqn00317.gif)1L1](../popups/usb-int-iconventions-unitsym.md).) **Temp** Temperature. **Field *n*** Predefined field variables. You may need to expand the dialog box to see all the columns in the **Data** table. For detailed information on how to enter data, see ["Entering tabular data," Section 3.2.7](pt01ch03s02s07.md).
6. Click **OK** to close the **Edit Material** dialog box. Alternatively, you can select another material behavior to define from the menus in the **Edit Material** dialog box (see ["Browsing and modifying material behaviors," Section 12.7.2](pt03ch12s07hlb02.md), for more information).
