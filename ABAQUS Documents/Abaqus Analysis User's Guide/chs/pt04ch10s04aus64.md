# 10.4.2 将结果从对称网格或部分三维网格转换到完整三维网格





**产品：** Abaqus/Standard

**参考资料**

- ["对称模型生成，" 第10.4.1节](pt04ch10s04aus63.md)
- [*SYMMETRIC RESULTS TRANSFER](../key/key-link.md#usb-kws-maxiresulttransfer)

### 概述

对称结果转换：
- 可以降低首先经历对称变形然后在加载历史后期经历非对称变形的结构分析成本；
- 可用于将轴对称模型的解转换到三维模型；
- 可用于将三维模型对称部分的解转换到完整的三维模型；
- 必须与对称模型生成功能结合使用（参见["对称模型生成，" 第10.4.1节](pt04ch10s04aus63.md)）；并且
- 仅可用于将应力/位移、热传递、耦合温度-位移或耦合声-结构分析的解转换到新模型。

### 将解从对称网格或部分三维网格转换到完整三维网格

对称结果转换功能可用于将轴对称模型的解转换到三维模型，或将三维模型对称部分的解转换到完整的三维模型。["对称模型生成，" 第10.4.1节](pt04ch10s04aus63.md)中所述的对称模型生成功能必须用于生成三维模型。

对称结果转换功能不适用于定义为部件实例组合的模型。

转换到新模型的解包括变形构型和对相应的材料状态，包括应变和所有状态变量。节点以其原始坐标导入。此解成为新分析的初始或基础状态。

#### 指定读取原始模型中获得解的时间

您可以指定必须读取原始模型中获得解的时间。所需的步和增量或迭代必须在原始分析期间写入重启动文件。

| **输入文件用法：** | 如果解是从直接循环过程以外的任何分析转换的，请使用以下选项： |
| --- | --- |
|  | ``` [*SYMMETRIC RESULTS TRANSFER](../key/key-link.md#usb-kws-maxiresulttransfer), STEP=*step*, INC=*increment* ``` 如果解是从先前的直接循环分析转换的，请使用以下选项： ``` [*SYMMETRIC RESULTS TRANSFER](../key/key-link.md#usb-kws-maxiresulttransfer), STEP=*step*, ITERATION=*iteration* ``` |

#### 获得平衡

您必须确保模型在分析开始时处于平衡状态。建议包含一个初始步定义，使用与转换结果所来自的模型状态相匹配的边界条件 和加载。初始时间增量应等于总时间，以便Abaqus/Standard尝试在一个增量中实现平衡。如果需要，Abaqus/Standard可以在步上线性解决应力不平衡，从而使用多个增量。您可以选择让应力不平衡在步的第一个增量中解决。

| **输入文件用法：** | 使用以下选项让Abaqus/Standard在线性解决步上的应力不平衡： |
| --- | --- |
|  | ``` [*SYMMETRIC RESULTS TRANSFER](../key/key-link.md#usb-kws-maxiresulttransfer), UNBALANCED STRESS=RAMP ``` 使用以下选项让Abaqus/Standard在步的第一个增量中解决应力不平衡： ``` [*SYMMETRIC RESULTS TRANSFER](../key/key-link.md#usb-kws-maxiresulttransfer), UNBALANCED STRESS=STEP ``` |

### 识别重启动文件

对称结果转换功能使用旧分析的重启动（`.res`）、分析数据库（`.stt`和`.mdl`）、部件（`.prt`）和输出数据库（`.odb`）文件将解数据转换到新网格。运行新分析时必须使用**oldjob**参数指定旧分析的重启动文件名，或者通过命令过程的请求来指定（参见["Abaqus/Standard、Abaqus/Explicit和Abaqus/CFD执行，" 第3.2.2节](pt01ch03s02abx02.md)）。

### 验证新模型

建议在转换结果或执行任何分析之前验证新模型是否正确生成。对称模型生成功能只需要在数据检查运行期间存储在重启动文件中的信息来生成新模型，这使您可以在对原始模型执行分析之前验证新模型。通过在运行Abaqus的命令中使用**datacheck**参数来执行数据检查分析（参见["Abaqus/Standard、Abaqus/Explicit和Abaqus/CFD执行，" 第3.2.2节](pt01ch03s02abx02.md)）。

一旦模型经过验证，就可以对原始模型进行分析，并将结果转换到新模型。

转换的解可以通过请求步开始时的输出（零增量；参见["输出，" 第4.1.1节](pt02ch04s01aus38.md)）写入结果文件。此解也可以在Abaqus/CAE中查看。

### 方向系统

当结果从轴对称模型转换到三维模型时，应力、应变等的单元输出使用局部圆柱方向系统。如果原始轴对称模型中的材料不包含方向定义，则提供默认局部方向定义（["方向，" 第2.2.5节](pt01ch02s02aus15.md)）。此默认方向定义为系统的极轴沿旋转轴线，并额外绕局部1方向旋转90度，使得局部轴为1=径向、2=轴向、3=周向。如果使用壳或膜，则将局部2轴和3轴投影到壳或膜的表面上作为表面上的局部方向。假设材料属性在此系统中指定。另一方面，如果在原始模型中材料关联了方向定义，则新三维模型中的方向将是沿对称轴线旋转的方向定义。

当通过反射部分三维模型来创建完整三维模型时，基于部分三维模型中相应的方向定义在完整三维模型中创建局部材料方向。但是，如果部分三维模型中的材料不包含方向定义，且部分三维模型不是通过旋转轴对称模型创建的，则不会在完整三维模型中创建局部方向定义。完整三维模型使用全局坐标系。

当结果从三维扇区通过绕其对称轴线旋转三维扇区转换到周期三维模型时，应力、应变等的单元输出始终使用局部圆柱方向系统。如果在原始三维扇区中指定了方向，则新模型中的方向系统通过绕对称轴线旋转原始方向系统来定义。如果使用壳或膜，则将局部2轴和3轴投影到壳或膜的表面上作为表面上的局部方向。如果原始三维扇区中的材料不包含方向定义，则提供默认局部方向定义。此默认方向通过在原始模型中围绕新模型中的对称轴线旋转全局坐标系来定义。

### 节点处的坐标系

从原始模型获得的位移和旋转分量首先转换到全局矩形笛卡尔轴系统，然后再转换结果。如果新模型中需要局部坐标方向，则必须在 新模型中指定节点变换（["变换坐标系，" 第2.1.5节](pt01ch02s01aus09.md)）来定义此坐标系。

### 限制条件

从轴对称模型到3D模型的结果转换存在以下限制：
- 8节点减缩积分轴对称单元（CAX8R和CAX8RH）的结果不能转换到对应的20节点砖单元（C3D20R和C3D20RH）（当这些单元构成接触对中从属表面下面的单元时）。
- SAX2是有限应变壳，而S8R是小应变壳。当原始分析中变形较大时，请勿使用此组合。

从对称3D模型到完整3D模型的结果转换存在以下限制：
- 不支持具有每节点五个自由度的壳的结果转换（STRI65、S8R5和S9R5）。

### 初始条件

可以在所有节点和单元上指定初始条件（参见["Abaqus/Standard和Abaqus/Explicit中的初始条件，" 第34.2.1节](pt07ch34s02aus116.md)），包括使用对称模型生成（参见["对称模型生成，" 第10.4.1节](pt04ch10s04aus63.md)）生成的模型部分。然而，在大多数情况下，对称结果转换功能将使用从原始模型获得的解覆盖初始条件值。温度和场变量是例外。只有在原始模型中指定了温度和场变量时，才会覆盖初始温度和场变量。如果原始模型中只有部分包含温度或场变量规格，则认为模型的其余部分具有零大小的初始条件。此场分布将转换到新模型。如果原始模型中任何地方都未定义温度和/或场变量，则应用新模型中指定的初始条件。

### 边界条件

所有边界条件必须重新定义；对称结果转换功能忽略原始模型中指定的边界条件。您必须确保模型在分析开始时处于平衡状态；因此，应包括一个初始步定义，使用与转换结果所来自的模型状态相匹配的边界条件和加载。

### 载荷

所有载荷必须重新定义；对称结果转换功能忽略原始模型中指定的载荷。您必须确保模型在分析开始时处于平衡状态；因此，应包括一个初始步定义，使用与转换结果所来自的模型状态相匹配的边界条件和加载。

### 材料选项

原始模型中定义的所有材料定义都将转换到新模型。

### 单元

原始模型中活跃的任何单元或接触对移除/重新激活定义（参见["单元和接触对的移除与重新激活，" 第11.2.1节](pt04ch11s02aus66.md)）应重新指定。

### 输出

可用于应力/位移单元的所有标准输出变量都可以与对称结果转换功能一起使用。

转换到新模型的解可以通过请求步开始时的输出（零增量；参见["输出，" 第4.1.1节](pt02ch04s01aus38.md)）写入结果（`.fil`）文件。也可以在Abaqus/CAE中显示。
