# 12.7.6 选择和修改子选项或测试数据





如果当前行为有可用的子选项或测试数据，则**Suboptions**、**Test Data**或**Uniaxial Test Data**菜单将在数据区域的右上角可用。当您从菜单中选择其中一个选项时，会出现**Suboption Editor**或**Test Data Editor**，您可以在其中输入所需的数据。

**注意：**要显示**Suboption Editor**或**Test Data Editor**中特定按钮、文本字段和其他选项的上下文相关帮助，您必须选择感兴趣的选项，然后按**[F1]**。（显示编辑器时，主菜单栏中的**Help**菜单不可用。）有关使用**[F1]**获取帮助的详细信息，请参见["显示上下文相关帮助，"第2.6.1节"](pt01ch02s06s01.md)。

**添加材料子选项或测试数据：**

1. 点击数据区域右上角的**Suboptions**、**Test Data**或**Uniaxial Test Data**，并从出现的列表中选择您选择的选项。**Suboption Editor**或**Test Data Editor**（根据您的选择适用）会出现在一个单独的对话框中。
2. 在编辑器内输入所需的数据，然后点击**OK**返回材料编辑器。您可以使用键盘向子选项或测试数据表格中输入数据。或者，您可以鼠标右键点击表格中的任意位置来查看用于指定表格数据的选项列表。例如，存在一个用于从表格中的数据创建*X--Y*数据对象的选项；您可以在Visualization模块中绘制*X--Y*数据并直观地检查其有效性。另一个选项用于自动从文件输入数据。有关每个选项的详细信息，请参见["输入表格数据，"第3.2.7节"](pt01ch03s02s07.md)。
