# 20.10.9 在两条线之间绘制圆角


使用 Sketcher 工具箱中的圆角工具在两条线或圆之间绘制圆角。输入圆角半径并选择两条线或圆,如下图所示:

![](../graphics/ske-fillet2l-nls.png)

["构造几何," 第 20.5.2 节](pt03ch20s05s02.md) 展示瞭如何在两个构造圆上创建与圆相切的圆角。

在绘制时,您应该注意点的位置,因为这会影响网格的质量。草图中的点会成为您正在创建或修改的零件的顶点。反过来,当您在 Mesh 模块中对模型进行网格划分时,Abaqus/CAE 会将这些顶点转换为完全约束的种子,并在其位置放置节点。有关如何移动顶点的信息,请参见["拖动 Sketcher 对象," 第 20.17.1 节](pt03ch20s14hlb01.md)。

如果您创建了一个圆角,然后移动所选的线或圆,Abaqus/CAE 会移动该圆角并保持相切。

**在两条线之间绘制圆角的步骤:**

1. 从 Sketcher 工具箱中,选择圆角工具 ![](../graphics/ico_sketchFillet.png)。有关 Sketcher 工具箱中工具的图示,请参见["Sketcher 工具," 第 20.4.1 节](pt03ch20s04s01.md)。Abaqus/CAE 会在提示区域显示提示,引导您完成操作。
2. 在提示区域出现的文本框中,输入所需圆角的半径。
3. 选择圆角必须与之相切的两条线或圆(不是圆弧)。圆角会出现在两条线或圆之间。**提示:**当您从草图中选择一条线或圆时,Abaqus/CAE 使用光标位置来确定圆角的位置。要创建所需的圆角,在进行选择时,您应该将光标放置在圆角的预期位置附近。
4. 要创建更多相同半径的圆角,请重复上一步。
5. 完成创建圆角后,执行以下操作之一: - 在 Abaqus/CAE 窗口任意位置单击鼠标按钮 2。 - 选择 Sketcher 工具箱中的任何其他工具。 - 单击提示区域中的取消按钮 ![](../graphics/afxI_cancel.png)。
