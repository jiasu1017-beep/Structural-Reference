# 2.3 什么是模块？





Abaqus/CAE分为称为模块的功能单元。每个模块仅包含与建模任务的特定部分相关的工具。例如，网格模块仅包含创建有限元网格所需的工具，而作业模块仅包含用于创建、编辑、提交和监控分析作业的工具。Abaqus/Viewer是Abaqus/CAE的子集，仅包含可视化模块。

您可以从上下文栏的**模块**列表中选择模块。或者，您可以通过在模型树中选择对象的上下文来选择模块；有关更多信息，请参阅["模型树概述，" 第3.5.1节"](pt01ch03s05s01.md)。菜单和模型树中模块的顺序对应于创建模型的逻辑顺序。在许多情况下，您必须遵循这个自然的顺序来完成建模任务；例如，您必须先创建零件，然后才能创建装配体。尽管模块的顺序遵循逻辑顺序，但Abaqus/CAE允许您随时选择任何模块，无论模型的状态如何。

以下Abaqus/CAE中可用模块的列表简要描述了您可以在每个模块中执行的建模任务。列表中模块的顺序对应于上下文栏中**模块**列表和模型树中模块的顺序：

****零件****

通过绘制或导入几何体来创建单个零件。有关更多信息，请参阅[第11章，"零件模块"](pt03ch11.md)。

****属性****

创建截面和材料定义，并将其分配给零件区域。有关更多信息，请参阅[第12章，"属性模块"](pt03ch12.md)。

****装配****

创建和组装零件实例。有关更多信息，请参阅[第13章，"装配模块"](pt03ch13.md)。

****步骤****

创建和定义分析步骤以及相关的输出请求。有关更多信息，请参阅[第14章，"步骤模块"](pt03ch14.md)。

****相互作用****

指定模型区域之间的相互作用，如接触。有关更多信息，请参阅[第15章，"相互作用模块"](pt03ch15.md)。

****载荷****

指定载荷、边界条件和场。有关更多信息，请参阅[第16章，"载荷模块"](pt03ch16.md)。

****网格****

创建有限元网格。有关更多信息，请参阅[第17章，"网格模块"](pt03ch17.md)。

****优化****

创建和配置优化任务。有关更多信息，请参阅[第18章，"优化模块"](pt03ch18.md)。

****作业****

提交分析作业并监控其进度。有关更多信息，请参阅[第19章，"作业模块"](pt03ch19.md)。

****可视化****

查看分析结果和选定的模型数据。有关更多信息，请参阅[第五部分，"查看结果"](pt05.md)。

****草图****

创建二维草图。有关更多信息，请参阅[第20章，"草图模块"](pt03ch20.md)。

模块可以根据视口中显示的对象进行分类。当您在零件模块和属性模块中时显示零件；当您在装配模块、步骤模块、相互作用模块、载荷模块、网格模块和作业模块中时显示装配体；当您在可视化模块中时显示输出数据库结果。

当您在模块之间移动时，主窗口的内容会发生变化。从上下文栏的**模块**列表中选择模块，或通过在模型树中选择对象的上下文来切换模块，会导致上下文栏、模块工具箱和菜单栏更改以反映当前模块的功能。

当您在模块之间移动时，Abaqus/CAE会将当前视口与您选择的模块相关联。您可以有多个视口，不同的视口可以与不同的模块相关联。当您选择视口并将其设为当前视口时，与该视口关联的模块将成为当前模块。有关在视口之间移动的更多信息，请参阅["选择视口，" 第4.3.2节"](pt01ch04s03hlb02.md)。
