# 15.13.12 定义流体交换相互作用







流体交换相互作用允许您定义流体在空腔与周围环境之间或两个空腔之间的流动。流体交换相互作用不能在后续分析步骤中修改或停用。有关详细讨论，请参见Abaqus Analysis User's Guide的第11.5.3节"流体交换定义"。

**定义流体交换相互作用：**

1. 从主菜单栏中，选择****相互作用**![](../graphics/images/arrow.gif)**创建****。**提示：**您也可以使用相互作用模块工具箱中的![](../graphics/ico_interaction.png)工具创建流体空腔相互作用。
2. 在出现的**创建相互作用**对话框中，执行以下操作：- 命名该相互作用。有关命名对象的信息，请参见第3.2.1节"使用基本对话框组件"。- 选择将创建该相互作用的步骤**初始**。- 选择**流体交换**类型的相互作用。
3. 单击**继续**关闭**创建相互作用**对话框。Abaqus/CAE打开**编辑相互作用**对话框。
4. 切换打开所需的交换定义，**到环境**或**空腔之间**。
5. 选择主流体空腔相互作用，**流体空腔相互作用1**。
6. 如果您要定义空腔之间的交换，请选择**流体空腔相互作用2**。
7. 选择**流体交换属性**。如果需要，请单击![](../graphics/ico_interactionProperty.png)创建相互作用属性；有关更多信息，请参见第15.14.5节"定义流体交换相互作用属性"。
8. 输入**有效交换面积**的值。有效交换面积表示流体交换所通过管道的横截面积。
9. 单击**确定**创建相互作用并关闭编辑器。表示您刚创建的流体交换相互作用的符号出现在视口中。更多信息，请参见第15.10节"理解代表相互作用、约束和连接器的符号"。
