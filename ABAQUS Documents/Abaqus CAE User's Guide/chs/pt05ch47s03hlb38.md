# 47.4.40 转换X–Y数据对象使用的角度单位





### 47.4.40 转换X–Y数据对象使用的角度单位

使用**degreeToRadian**或**radianToDegree**函数对[之前保存的](pt05ch47s02hlb07.md) *X–Y*数据对象（有序对的集合）进行运算，生成一个新的*X–Y*数据对象。新的*X–Y*数据对象具有与原始*X–Y*数据对象相同的*X*坐标值。Abaqus/CAE通过将每个原始*Y*坐标从度转换为弧度或从弧度转换为度（取决于您指定的函数）来计算新的*Y*坐标值。

**要应用**degreeToRadian**或**radianToDegree**函数：**

1. 找到**Operate on XY Data**对话框。从主菜单栏，选择****Tools**![](../graphics/images/arrow.gif)**XY Data**![](../graphics/images/arrow.gif)**Create****。在出现的对话框中点击**Operate on XY data**；然后点击**Continue**。**Operate on XY Data**对话框出现。
2. 从列出的**Operators**中，点击**degreeToRadian**从度转换为弧度，或点击**radianToDegree**从弧度转换为度。相应的函数出现在表达式窗口中。
3. 从**XY Data**选项中，点击要运算的*X–Y*数据对象的名称，然后点击**Add to Expression**。您可以从本会话中之前保存的所有*X–Y*数据对象中选择（按**XY Data**字段中的字母顺序列出）。*X–Y*数据对象名称出现在表达式窗口中函数括号内。
4. 要继续构建表达式，将光标放在表达式窗口中，然后输入或选择要包含的函数、运算符和*X–Y*数据。
5. 要计算并显示表达式，点击**Plot Expression**。
6. 要[保存](pt05ch47s02hlb07.md)新的*X–Y*数据对象，点击**Save As**，然后在出现的对话框中提供名称。保存数据对象使其可用于本会话中的后续操作，以及包含多个数据对象的*X–Y*图中。
7. 完成后，点击**Cancel**关闭对话框。
