# 3.9 对 Dassault Systèmes 许可证服务器使用 dslsstat 实用程序



### 3.9 对 Dassault Systèmes 许可证服务器使用 `dslsstat` 实用程序

您可以使用 `dslsstat` 实用程序显示有关 Dassault Systèmes 许可证服务器 (DSLS) 的基本状态信息；您可以验证许可证服务器正在运行、配置正确并提供正确的许可证。要查看有关许可证服务器的更详细信息，您可以使用随 DSLS 安装提供的 Dassault Systèmes 许可证管理工具。请参阅*Dassault Systèmes 许可证服务器安装和配置指南*（`DSLS.pdf`）中的"启动许可证管理工具"。

如果您在与 Dassault Systèmes 许可证服务器所在的网络上安装了 Abaqus 产品，则可以使用 `abaqus` 命令运行 `dslsstat` 实用程序，如下所示：

```
abaqus licensing dslsstat [-server *machine*:*port*] [-usage]
```

**`-server`**

使用此选项指定运行 DSLS 软件的服务器计算机的名称和端口号；例如，

```
abaqus licensing dslsstat -server jupiter:4085
```

如果在命令行上省略了 `-server` 选项，`dslsstat` 会尝试通过两种方式确定要查询的 Dassault Systèmes 许可证服务器：
- 使用 Abaqus 被配置使用的任何 Dassault Systèmes 许可证服务器（通常在安装期间配置）。
- 在系统默认位置查找 `DSLicSrv.txt` 许可证客户端配置文件，该文件描述了要使用的许可证服务器。此文件的位置在*Dassault Systèmes 许可证服务器安装和配置指南*中的"文件位置、设置和注册表项"中有描述。

如果找不到服务器，则会发出错误消息。

**`-usage`**

使用此选项在输出中显示个别用户的许可证使用详情。请参阅下面的示例。

结果在命令/终端窗口中返回；例如，

```
Server:  jupiter:4085
  Name                : jupiter
  Port                : 4085
  Status              : Running
  Type                : Standalone
  Version             : 6.212.0
  OS Name             : Windows 2008
  OS Version          : 5.2
  OS Arch             : x86
  Computer Id         : TWK-4213101B783368E0
```
为每个找到的许可证服务器报告的信息包括：- *Status*：`Running` 或 `Not Running`
- *Type*：`Standalone` 或 `Failover`

还会报告在许可证服务器上找到的每个已注册许可证功能的信息；例如，

```
Licenses:                                                                      Server
| Feature | Version | Model          | Number | InUse | Expires              | Name
| --------| ------- | -------------- | -------| ----- | -------------------- | -------
| QAE     | 11      | Token          |     50 |     0 | 31-Dec-2013 19:59:00 | jupiter
| QAQ     | 11      | ConcurrentUser |      1 |     0 | 31-Dec-2013 19:59:00 | jupiter
| QEX     | 11      | Token          |     50 |     0 | 31-Dec-2013 19:59:00 | jupiter
| QSD     | 11      | Named User     |     50 |     0 | 31-Dec-2013 19:59:00 | jupiter
```
到期日期包括本地时区的时间。

如果包含 `-usage` 选项，则会提供额外信息，显示哪些用户（登录/用户名）正在使用特定的许可证功能；例如，

```
Licenses:                                                                      Server
| Feature | Version | Model          | Number | InUse | Expires              | Name
| --------| ------- | -------------- | -------| ----- | -------------------- | -------
| QAE     | 11      | Token          |     50 |     0 | 31-Dec-2013 19:59:00 | jupiter
| QAQ     | 11      | ConcurrentUser |      1 |     0 | 31-Dec-2013 19:59:00 | jupiter
| QEX     | 11      | Token          |     50 |     0 | 31-Dec-2013 19:59:00 | jupiter
    jsmith on bigbird, granted on 18-Oct-2012 10:06:13
    tjones on kermit, granted on 18-Oct-2012 14:02:47

```
每行用户信息的格式如下：
```
*username* on *hostname*, granted on *date*
```

有关 Abaqus 许可证执行过程的更多信息，请参阅["许可证实用程序，"Abaqus Analysis User's Guide 第 3.2.12 节](../usb/usb-link.md#usb-int-dlicproc)。
