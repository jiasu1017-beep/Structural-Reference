# 42.5.8 选择复数结果





默认情况下，Abaqus/CAE列出当前[步骤和帧](pt05ch42s01hlb01.md)的[输出数据库](pt02ch09s06hlb02.md)中所有可用的变量供您选择。如果当前步骤是稳态动力学分析，则输出变量（如应力（S）或位移（U））的值可以是同时具有实部和虚部的复数。使用**Field Output**对话框中的**Primary Variable**、**Deformed Variable**、**Symbol Variable**、**Status Variable**和**Stream Variable**选项来选择您要显示的复数变量。列表中每个变量的描述以星号开头，表示该变量包含复数结果。有关单个输出变量标识符的信息，请参见["输出变量，" Abaqus Analysis User's Guide第4.2节](../usb/usb-link.md#usbovsect)。

您可以以[变形](pt05ch43.md)、[云图](pt05ch44.md)或[符号](pt05ch45.md)图的形式显示复数结果；通过[模型探测](pt05ch51.md)；通过[沿模型路径获取结果](pt05ch48.md)；以及通过基于结果形成显示组或指定颜色编码。在每种情况下，您都可以从几种[数字形式](pt05ch42s04hlb05.md)中选择来显示复数结果的不同方面。您可以使用[谐波动画](pt05ch49s02hlb03.md)为复数结果设置动画。在谐波动画期间，复数的实部和虚部用于计算和显示表示半周期（0到180）或完整周期（180到180）角度序列的结果值。
