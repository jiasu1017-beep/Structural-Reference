# 3 Abaqus licensing







This chapter describes network licensing for Abaqus, including the following topics:
- Updating an Abaqus 6.14 FLEXnet license file.
- Upgrading a FLEXnet license server and tools.
- Redundant FLEXnet license server configurations.
- The FLEXnet license server manager, licensing administration tools, and options file.
- Using the `dslsstat` and `reporttool` utilities for a Dassault Systmes license server (DSLS).
