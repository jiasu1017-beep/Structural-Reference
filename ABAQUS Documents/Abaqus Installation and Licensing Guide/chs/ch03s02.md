# 3.2 更新 Abaqus FLEXnet 许可证文件



您可能需要更新 Abaqus 6.14 FLEXnet 许可证文件；例如，当您获取新许可证文件以替换过期的许可证文件或修改许可证功能时。您可能还需要替换以前版本 Abaqus 的许可证文件。例如，如果您手动将现有许可证服务器升级到高于 Version 11.6.1 的 FLEXnet Licensing 版本，则需要更新 Abaqus 6.14 的许可证文件，而无需重新安装和覆盖现有许可证实用程序。

您必须具有写权限才能更新许可证文件。如果您的许可证文件独立于使用 FLEXnet Licensing 的其他产品，您可以使用两种方法替换当前许可证文件：重新启动许可证服务器或强制服务器重新读取许可证文件。

**更新 Abaqus FLEXnet 许可证文件并重新启动许可证服务器：**

1. 通过使用新名称保存来备份当前许可证文件。
2. 使用原始许可证文件的名称将新许可证文件复制到与原始许可证文件相同的位置。
3. 运行 FLEXnet Licensing 工具 `lmdown`（请参阅["lmdown，"第 3.6.2 节](ch03s06.md#sgb-chp-net-lic-lmdown)）关闭当前运行的 Abaqus 许可证服务器。
4. 重新启动 Abaqus 许可证服务器。对于 Windows 平台，请使用["使用 **LMTOOLS** 启动 FLEXnet 服务器，"第 3.6.9 节](ch03s06.md#sgb-chp-net-lic-start)中所述的方法。对于 Linux 平台，请使用["Abaqus FLEXnet 许可证安装详情，"第 2.3 节](ch02s03.md)中所述的方法。

**更新和重新读取 Abaqus FLEXnet 许可证文件：**

1. 通过使用新名称保存来备份当前许可证文件。
2. 使用原始许可证文件的名称将新许可证文件复制到与原始许可证文件相同的位置。
3. 运行 FLEXnet Licensing 工具 `lmreread`（请参阅["lmreread，"第 3.6.6 节](ch03s06.md#sgb-chp-net-lic-lmreread)）强制许可证服务器重新读取新许可证文件。

您更新的许可证文件必须保持其文件名和路径名（位置）。如果文件名或路径名发生更改，您应该卸载许可证，然后运行许可证安装过程（请参阅[第 2 章，"安装 Abaqus"](ch02.md)"）。Abaqus 许可证文件的默认名称是 `simulialm.lic`。您可以通过运行以下命令来确定 Abaqus 许可证文件的名称
```
*simulia_dir*/License/lmstat -c [*port*]@*license_server_host*
```
其中 *simulia_dir* 是 SIMULIA 父目录，*license_server_host* 是许可证服务器的主机名。如果许可证服务器使用的端口不在 27000 到 27009 之间，则还必须指定端口号。

如果您的 Abaqus 许可证功能使用组合 FLEXnet 许可证文件与其他产品集成，请将组合许可证文件中的 Abaqus 许可证信息替换为更新的信息。您必须运行 FLEXnet Licensing 工具 `lmreread`（请参阅["lmreread，"第 3.6.6 节](ch03s06.md#sgb-chp-net-lic-lmreread)）强制许可证服务器重新读取新许可证文件。有关组合许可证文件的更多信息，请参阅 FLEXnet Licensing End User Guide Version 11.6，可从 **Support** 页面上的 **Licensing** 部分下载，网址为 [www.3ds.com/simulia](http://www.3ds.com/simulia)。
