# 21.5 分配粘聚建模数据





在Property模块中，您可以按如下方式完成粘聚建模数据：

1. 创建一个粘聚截面来定义两个粘接零件之间界面处的粘接层的截面属性。 您可以通过分别选择粘聚截面的本构行为为**Traction Separation**、**Continuum**或**Gasket**来对可忽略的薄粘接层、有限厚度粘接层或垫片进行建模。 有关更多信息，请参阅["创建粘聚截面，" 第12.13.16节](pt03ch12s13hlb12.md)。
2. 为粘聚区域定义材料。 如果您在粘聚截面编辑器中选择了**Continuum**或**Gasket**响应，您可以使用Property模块中可用的材料模型来定义连续体材料属性。 Abaqus/CAE不支持**Traction Separation**响应的材料模型。 在这种情况下，您必须使用**Keywords Editor**添加材料定义，如["向您的Abaqus/CAE模型添加不支持的关键字，" 第9.10.1节](pt02ch09s09hlb01.md)中所述。
3. 将粘聚截面分配给粘聚区域。 有关更多信息，请参阅["分配截面，" 第12.15.1节](pt03ch12s15hlb01.md)。
