# 20.21 重置视图







使用 Sketcher 工具箱中的重置视图工具![](../graphics/ico_sketchResetView.png)返回到原始视图。有关 Sketcher 工具箱中工具的图表，请参见["Sketcher 工具，"第 20.4.1 节](pt03ch20s04s01.md)。

如果您不确定零件或装配相对于草图平面的方向，可以使用视图操作工具检查草图平面和您正在草绘的对象。点击重置视图工具时，Abaqus/CAE 会将视图恢复到进入 Sketcher 时显示的原始视图方向。
