# 1.6.21 接触表面扩展







**产品：**Abaqus/Standard  

### 单元测试

C3D8    C3D10    C3D10I    C3D20    CPE3    CPE4    CPE6    CPE8    ISL21A    ISL22A

### 功能测试

小滑动和有限滑动的接触表面和滑线扩展。

### 问题描述

在小滑动接触中，扩展主表面允许从节点在分析开始时从节点略微位于主表面周长外时找到与主表面的交点。小滑动模型由堆叠块排列组成，其中从表面节点在分析开始时延伸到主表面周长之外。

在有限滑动接触中，扩展主表面可以防止节点"脱落"或被困在主表面后面。有限滑动模型与小滑动模型类似，只是从表面在分析开始时位于主表面周长之内。第二步将，从表面移动到主表面周长之外但在扩展区域之内。

**材料：**

| 杨氏模量 | 3.0 106 |
| --- | --- |
| 泊松比 | 0.2 |

### 结果与讨论

小滑动测试验证了找到交点并在分析开始时计算出正确的接触间隙。

当从节点进入扩展区域时，接触间隙、滑移距离和接触压力用于验证有限滑动结果。
