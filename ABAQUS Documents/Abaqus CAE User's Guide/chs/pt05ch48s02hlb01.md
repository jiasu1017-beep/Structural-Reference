3. 完成后，点击 **确定（OK）**。如果创建了新路径，Abaqus/CAE 会将路径名称添加到 **路径管理器（Path Manager）** 列表中。默认情况下，Abaqus/CAE 会保存您在会话期间指定的路径。如果您想在后续会话中保留路径，请将其保存到 XML 文件、模型数据库或输出数据库；有关更多信息，请参阅 ["管理会话对象和会话选项，" 第 9.9 节"](pt02ch09s08.md)。

# 48.2.3 创建或编辑边列表路径





路径是您通过在模型中指定一系列线段来定义的线。在边列表路径中，线段由单元边和连接单元边端点的线定义。有关在路径中包含边以外位置的信息，请参阅 ["在模型中创建路径，" 第 48.2 节"](pt05ch48s02.md)。

您可以通过将部件实例名称和边标签输入表格来创建边列表路径。每个边标签由单元编号、面编号、边编号和沿边的方向组成。使用三种方法的组合从视口选择边以输入表格：
- 选择单个单元边。
- 选择一个单元边和沿特征边的方向，并让 Abaqus/CAE 自动添加从您的选择到特征边终点的所有边。
- 选择一个单元边和一个节点，并让 Abaqus/CAE 选择它们之间沿单元边的最短距离路径。

您也可以直接编辑表格，使用视口来跟踪进度。但是，边标签的某些组成部分取决于单元类型、方向和所需的连通性，这些通过从视口进行选择更容易确定。

**创建或编辑边列表路径：**

1. 找到路径创建和编辑选项。**创建新路径：** 1. 从主菜单栏中，选择 ****工具（Tools）**![](../graphics/images/arrow.gif)**路径（Path）**![](../graphics/images/arrow.gif)**创建（Create）****。出现 **创建路径（Create Path）** 对话框。2. Abaqus 在 **名称（Name）** 文本字段中显示路径的默认名称。为了提供更有意义的名称，用您选择的名称替换此默认值（包括您希望的任何空格）。3. 在 **创建路径（Create Path）** 对话框中，点击 **边列表（Edge list）**；然后点击 **继续（Continue）**。**编辑现有路径：** - 从主菜单栏中，选择 ****工具（Tools）**![](../graphics/images/arrow.gif)**路径（Path）**![](../graphics/images/arrow.gif)**编辑（Edit）****。从出现的菜单中，选择您要编辑的路径。**提示：**您也可以使用 **路径管理器（Path Manager）** 来编辑路径。从主菜单栏中，选择 ****工具（Tools）**![](../graphics/images/arrow.gif)**路径（Path）**![](../graphics/images/arrow.gif)**管理器（Manager）**** 来显示管理器。选择您要编辑的路径，然后点击管理器右侧的按钮上的 **编辑（Edit）**。出现 **编辑边列表路径（Edit Edge List Path）** 对话框。
2. 要通过在视口中选择边来创建或编辑路径，请点击 **之前添加（Add Before）** 或 **之后添加（Add After）** 开始进行选择。Abaqus/CAE 提示您 **选择要插入路径的边（Select edges to be inserted into the path）**，并关闭 **编辑边列表路径（Edit Edge List Path）** 对话框。
3. 使用提示区域中的选择方法来选择边。提供三种边选择方法：****逐个选择（Individually）**** 按它们应出现在路径中的顺序选择单个单元边；所选边不必相邻。如有必要，点击提示区域中的 **翻转（Flip）** 以反转您选择的最后一条边的连通性。****按特征边（By feature edge）**** 选择沿特征边的一条起始边；Abaqus/CAE 将自动选择您的选择与特征边终点之间的所有边。点击 **翻转（Flip）** 以反转您选择的边的连通性以及用于自动选择的方向。****按最短距离（By shortest distance）**** 选择一条起始边和一个终点（节点）；Abaqus/CAE 将自动选择定义您的选择之间最短路径的单元边。点击 **翻转（Flip）** 以反转您选择的边的连通性以及用于自动边选择的方向。如果存在多条具有相同距离的路线，Abaqus/CAE 会选择其中一条。**注意：**您选择的边和节点必须在同一个部件实例上。如果所需边存在于多个部件实例中，请关闭 **选择最接近屏幕的实体（Select the Entity Closest to the Screen）** 工具（位于 **选择（Selection）** 工具栏中）。如果进行模糊选择，部件实例名称和边标签（不带方向值）将以下形式出现在视口左下角：`高亮显示单元边 *单元*.*面*.*边*（Highlighting Element Edge *Element*.*Face*.*Edge*）`。使用 **下一个（Next）** 和 **上一个（Previous）** 按钮循环浏览可能的选项，然后点击 **确定（OK）** 确认您的选择。Abaqus 在您选择时高亮显示边，并显示节点标签和路径连通性。此外，Abaqus 将路径中的第一个节点标记为 `起点（Start）`，将最后一个节点标记为 `终点（End）`。
4. 完成后，在提示区域点击 **完成（Done）**。Abaqus/CAE 显示 **编辑边列表路径（Edit Edge List Path）** 对话框；路径中的部件实例和边列在路径定义表中。
5. 如果需要，重复前述步骤添加其他边。**之前添加（Add Before）** 和 **之后添加（Add After）** 按钮决定您的选择将被添加到路径定义表中当前高亮行的什么位置。
6. 如果需要，您可以使用标准键盘和鼠标编辑技术在 **路径定义（Path Definition）** 表中插入、修改或删除部件实例名称和边标签。边标签的形式为（`*单元编号*, *面编号*, *边编号*, *方向*`）。**提示：**使用查询工具集查询视口中的节点以获取部件实例名称。使用绘图状态无关选项在视口中显示单元和面标签。（有关更多信息，请参阅 ["自定义模型标签，" 第 55.5 节"](pt05ch55s05.md)。）边从元素外部看时按顺时针方向编号。方向（1 或 1）取决于您所需的路径连通性。有关特殊的表编辑选项或从 ASCII 文件读取数据，请点击鼠标按钮 3。（有关更多信息，请参阅 ["输入表格数据，" 第 3.2.7 节"](pt01ch03s02s07.md)。）
7. 编辑 **路径定义（Path Definition）** 表后，按 **[Enter]**。Abaqus 更新视口中显示的路径。
8. 完成后选择边后，点击 **确定（OK）**。如果创建了新路径，Abaqus/CAE 会将路径名称添加到 **路径管理器（Path Manager）** 列表中。默认情况下，Abaqus/CAE 会保存您在会话期间指定的路径。如果您想在后续会话中保留路径，请将其保存到 XML 文件、模型数据库或输出数据库；有关更多信息，请参阅 ["管理会话对象和会话选项，" 第 9.9 节"](pt02ch09s08.md)。

# 48.2.4 创建或编辑圆形路径





路径是您通过在模型中指定一系列点来定义的线。在圆形路径中，所有点都是位于圆周上或沿圆半径的模型坐标位置。有关路径规范的替代形式的信息，请参阅 ["在模型中创建路径，" 第 48.2 节"](pt05ch48s02.md)。

您可以通过选择节点或输入坐标来定义圆来创建圆形路径。定义圆后，您可以使用圆的边或半径来定义弯曲的圆周路径或直的径向路径。

**创建或编辑圆形路径：**

1. 找到路径创建和编辑选项。**创建新路径：** 1. 从主菜单栏中，选择 ****工具（Tools）**![](../graphics/images/arrow.gif)**路径（Path）**![](../graphics/images/arrow.gif)**创建（Create）****。出现 **创建路径（Create Path）** 对话框。2. Abaqus 在 **名称（Name）** 文本字段中显示路径的默认名称。为了提供更有意义的名称，用您选择的名称替换此默认值（包括您希望的任何空格）。3. 在 **创建路径（Create Path）** 对话框中，点击 **圆形（Circular）**；然后点击 **继续（Continue）**。**编辑现有路径：** - 从主菜单栏中，选择 ****工具（Tools）**![](../graphics/images/arrow.gif)**路径（Path）**![](../graphics/images/arrow.gif)**编辑（Edit）****。从出现的菜单中，选择您要编辑的路径。**提示：**您也可以使用 **路径管理器（Path Manager）** 来编辑路径。从主菜单栏中，选择 ****工具（Tools）**![](../graphics/images/arrow.gif)**路径（Path）**![](../graphics/images/arrow.gif)**管理器（Manager）**** 来显示管理器。选择您要编辑的路径，然后点击管理器右侧的按钮上的 **编辑（Edit）**。出现 **编辑圆形路径（Edit Circular Path）** 对话框。
2. 选择路径类型，**圆周（Circumferential）** 或 **径向（Radial）**。Abaqus/CAE 更新对话框以反映您创建的路径类型。**圆周路径（Circumferential Path）** 沿圆的边缘，需要半径、起始角度和结束角度来定义其位置和长度。**径向路径（Radial Path）** 沿圆的半径，需要径向角度、起始值和结束值来定义其位置和长度。
3. 在对话框的 **圆定义（Circle Definition）** 部分中，选择 **原点和轴（Origin and axis）** 或 **弧上的 3 点（3 Points on Arc）** 作为定义圆的方法。原点和轴方法使用两个点来定义旋转轴，使用圆平面上的一点来定义默认半径长度和旋转起始点。弧上三点方法拟合三个边点成一个圆。
4. 点击 **从视口选择（Select from Viewport）** 选择节点或输入坐标来定义圆。Abaqus/CAE 关闭 **编辑圆形路径（Edit Circular Path）** 对话框，并在提示区域显示提示来指导您完成该过程。
5. 从视口中选择节点，或使用提示区域中的文本字段将每个点指定为 X、Y、Z 坐标，用空格或逗号分隔；例如，1.0, 3.4, 2.0。Abaqus/CAE 打开 **编辑圆形路径（Edit Circular Path）** 对话框；您选择的坐标出现在圆定义表中。如果您选择了节点，所使用的坐标对应于模型的当前变形状态。在当前视口中的模型绘图上，Abaqus 高亮显示路径中包含的所有点，并显示路径连通性。
6. 您可以按如下方式编辑 **圆定义（Circle Definition）** 中的单个点：1. 将光标放在点的坐标上。2. 点击鼠标按钮 3，并选择 **编辑选择（Edit selection）**。3. 在视口中选择一个节点，或在提示区域中编辑坐标。当您进行选择或输入新坐标时，Abaqus/CAE 返回 **编辑圆形路径（Edit Circular Path）** 对话框，并在视口中显示重新定义的圆。
7. 选择要包含在路径中的段数。Abaqus/CAE 修改视口中的路径以显示所选数量的等长段。
8. 按如下方式完成圆形路径的编辑或创建：**圆周路径** 选择 **半径（Radius）**，并选择旋转的 **起始角度（Start angle）** 和 **结束角度（End angle）**。默认情况下，Abaqus 使用您定义的圆的半径，并从您在圆上选择的第一个点开始完整旋转 360 度——如果您选择了原点和轴方法，则是圆上唯一的点。**径向路径** 选择 **径向角度（Radial Angle）**、**起始半径（Start Radius）** 和 **结束半径（End Radius）**。默认径向角度（0）是从圆心到您在圆周上选择的第一个点的方向——如果您选择了原点和轴方法，则是圆上唯一的点。默认起始半径为 `0.0`（圆心），默认结束半径等于圆的半径。
9. 完成后，点击 **确定（OK）**。如果创建了新路径，Abaqus/CAE 会将路径名称添加到 **路径管理器（Path Manager）** 列表中。默认情况下，Abaqus/CAE 会保存您在会话期间指定的路径。如果您想在后续会话中保留路径，请将其保存到 XML 文件、模型数据库或输出数据库；有关更多信息，请参阅 ["管理会话对象和会话选项，" 第 9.9 节"](pt02ch09s08.md)。
