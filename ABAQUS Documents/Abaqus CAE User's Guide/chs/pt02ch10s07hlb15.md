# 10.7.13 将子结构作为零件导入模型数据库





从主菜单栏选择**文件**![](../graphics/images/arrow.gif)**导入**![](../graphics/images/arrow.gif)**零件**，可以将`.sim`文件中的子结构定义作为新零件定义导入。您导入的`.sim`文件必须位于SIM数据库所引用的支持Abaqus文件所在的同一目录中；这些支持文件可能包括`.prt`、`.mdl`、`.stt`或`.sup`格式的数据。此外，您导入的`.sim`文件必须与您使用的Abaqus/CAE版本来自同一版本的Abaqus。

子结构导入还需要输出数据库（`.odb`）文件用于网格显示。

**从SIM数据库导入子结构：**

1. 从主菜单栏，选择**文件**![](../graphics/images/arrow.gif)**导入**![](../graphics/images/arrow.gif)**零件**。出现**导入零件**对话框。
2. 从**导入零件**对话框底部的**文件过滤器**菜单中，选择**子结构 (*.sim*)**。Abaqus/CAE列出所选目录中所有具有`.sim*`文件扩展名的文件。
3. 选择包含要导入的子结构的SIM数据库，然后单击**确定**。出现**创建子结构零件**对话框，显示子结构的名称及其唯一标识符（以字母Z开头）。
4. 如果需要，自定义要导入的子结构的**零件名称**。
5. 在**使用ODB文件中的网格**字段中，指定包含要与此子结构定义关联的网格的输出数据库文件的名称。
6. 单击**确定**。Abaqus/CAE将子结构作为新零件定义导入您的模型。
