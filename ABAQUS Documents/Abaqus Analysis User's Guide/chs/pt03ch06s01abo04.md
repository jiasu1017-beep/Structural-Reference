# 6.1.1 求解分析问题：概述





### 概述

一大类应力分析问题可以通过 Abaqus/Standard 和 Abaqus/Explicit 求解。这类问题的基本划分是静态或动态响应；动态问题是指惯性效应显著的问题。Abaqus/CFD 求解范围广泛的不可压缩流动问题。

在 Abaqus 中，分析问题历史使用步骤定义（『定义分析』，第 6.1.2 节）。对于每个步骤，您选择一个分析过程，该过程定义在该步骤期间执行的的分析类型。可用的分析过程列于下文，并在参考节中详细描述。

Abaqus 使用内置完全耦合过程、顺序耦合和协同仿真作为多物理场仿真的解决方案技术，提供多物理场功能。大量额外的分析技术提供了强大的工具来更高效、更有效地执行 Abaqus 分析；请参阅『第四部分：分析技术』。

### Abaqus/Standard 分析

Abaqus/Standard 在区分静态和动态响应方面提供完全的灵活性；相同的分析可以包含多个静态和动态阶段。因此，可以应用静态预载荷，然后计算线性或非线性动态响应（如旋转机器组件的振动或柔性海上系统在浮力和稳定洋流载荷作用下首先移动到平衡位置，然后被波浪载荷激发的响应）。类似地，可以在动态事件之后寻求静态解（通过在动态分析步骤后跟一个静态载荷步骤）。有关这些过程类型的详细信息，请参阅『静态应力/位移分析』，第 6.2 节，以及『动态应力/位移分析』，第 6.3 节。除了静态和动态应力分析，Abaqus/Standard 还提供以下分析类型：
- 『稳态传输分析』，第 6.4 节
- 『热传递和热应力分析』，第 6.5 节
- 『电磁分析』，第 6.7 节
- 『耦合孔隙流体流动和应力分析』，第 6.8 节
- 『质量扩散分析』，第 6.9 节
- 『声学和冲击分析』，第 6.10 节
- 『Abaqus/Aqua 分析』，第 6.11 节

### Abaqus/Explicit 分析

Abaqus/Explicit 使用显式直接积分过程求解动态响应问题。有关 Abaqus 中可用的显式动态过程的更多信息，请参阅『动态应力/位移分析』，第 6.3 节。Abaqus/Explicit 还提供热传递、声学和退火分析能力：请参阅『热传递和热应力分析』，第 6.5 节；『声学和冲击分析』，第 6.10 节；以及『退火』，第 6.12 节。

### Abaqus/CFD 分析

Abaqus/CFD 使用二阶投影法求解范围广泛的不可压缩流动问题。有关 Abaqus 中可用的不可压缩流动过程的详细信息，请参阅『流体动力学分析』，第 6.6 节。

### 多物理场分析

多物理场是一种耦合方法，用于数值求解多个相互作用的物理域。Abaqus 提供内置完全耦合过程、顺序耦合和协同仿真作为多物理场仿真的解决方案技术。

#### 内置完全耦合过程

原生 Abaqus 多物理场能力通过添加表示每个物理场的自由度的自由度并使用单一求解器来求解物理场。Abaqus 提供以下内置完全耦合过程来求解多学科模拟，其中所有物理场都由 Abaqus 计算：
- 『完全耦合热应力分析』，第 6.5.3 节
- 『耦合热电分析』，第 6.7.3 节
- 『完全耦合热电结构分析』，第 6.7.4 节
- 『压电分析』，第 6.7.2 节（电和机械耦合）
- 『涡流分析』，第 6.7.5 节（电磁）
- 『耦合孔隙流体扩散和应力分析』，第 6.8.1 节
- 『声学、冲击和耦合声结构分析』，第 6.10.1 节
- 『欧拉分析』，第 14.1.1 节

#### 顺序耦合

当模型中一个或多个物理场之间的耦合仅在一个方向上重要时，可以使用顺序耦合多物理场分析。一个常见的例子是热应力分析，其中温度场不太依赖于应力场。典型的顺序耦合热应力分析包括两个 Abaqus/Standard 运行：热传递分析和后续的应力分析。

您可以按照以下章节中描述的在 Abaqus/Standard 中执行顺序耦合多物理场分析：
- 『顺序耦合的预定义场』，第 16.1.1 节
- 『顺序耦合热应力分析』，第 16.1.2 节
- 『顺序耦合的预定义载荷』，第 16.1.3 节

#### 协同仿真

协同仿真技术是一种多物理场能力，用于运行时耦合 Abaqus 和另一个分析程序。Abaqus 分析可以耦合到另一个 Abaqus 分析或第三方分析程序，以执行多学科模拟和多域（多模型）耦合。

协同仿真技术在下文中描述：
- 『协同仿真：概述』，第 17.1.1 节
- 『准备用于协同仿真的 Abaqus 分析』，第 17.2.1 节
- 『结构对结构协同仿真』，第 17.3.1 节
- 『流体对结构和共轭热传递协同仿真』，第 17.3.2 节
- 『电磁对结构和电磁对热协同仿真』，第 17.3.3 节
- 『执行协同仿真』，第 17.3.4 节
- 『结构对逻辑协同仿真』，第 17.4.1 节
