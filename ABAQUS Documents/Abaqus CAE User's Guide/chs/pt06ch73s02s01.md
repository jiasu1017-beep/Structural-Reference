# 73.2.1 什么是集？







集是您可以对其执行各种操作的有名区域或实体集合。例如，创建集后，您可以将其用于以下任务：
- 在Property模块中分配截面属性。
- 在Interaction模块中使用接触节点集和曲面创建接触对。
- 在Load模块中定义载荷和边界条件。
- 在Step模块中请求从模型特定区域输出。

您可以创建以下类型的集：

**几何集**

几何集包含您从以下类型的部件或这些部件的实例中选择的几何对象（单元、面、边和顶点）：
- 原生部件（使用Part模块中的工具创建的部件）
- 从文件导入的部件。

您可以从当前视口中选择要包含在集中的实体。根据部件的形状和建模空间，您可以在一集中包括单元、面、边和顶点的任意组合。但是，某些操作只能与特定类型的对象一起执行。因此，您选择的集必须只包含对该操作有效的对象类型。例如，集中力只能应用于顶点；因此，施加集中力的集只能包含顶点。

**节点集和单元集**

节点集和单元集包含您选择的节点和单元。您可以从在Mesh模块中为部件生成的原生Abaqus节点和单元、从孤立网格节点和单元，或从任何部件实例上的节点和单元创建节点集和单元集。集可以包含来自单个部件或多个部件实例的节点或单元。使用集工具集创建的集只能包含节点或单元，不能同时包含两者。但是，您可以通过合并集或导入包含多个同名义的输出数据库或输入文件来创建包含节点和单元混合的集。集中的原生节点和单元允许您请求输出或将载荷添加到特定区域，而无需删除网格和分割几何体。但是，对网格的任何更改（包括从重放或日志文件重新生成网格）可能会使网格集的原生部分失效或更改。

有关网格部件和孤立网格节点和单元的更多信息，请参见["可以从Abaqus/CAE导入和导出的文件类型有哪些？," 第10.1.1节](pt02ch10s01s01.md)；["从Abaqus输入文件导入模型," 第10.5.2节](pt02ch10s05s02.md)；["从输出数据库导入模型," 第10.5.3节](pt02ch10s05s03.md)；和["创建网格部件," 第17.20节](pt03ch17s20.md)。

如果重命名或删除集，则与该集关联的对象（如截面或载荷）将变为无效。但是，如果将重命名的集的名称更改回其原始名称，或者使用其原始名称重新创建已删除的集，则与该集关联的对象将恢复。
