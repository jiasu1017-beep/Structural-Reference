# 62.6.4 在距两条边的指定距离处创建datum点







您可以通过在模型上选择两条边并输入datum点到每条边的距离来定位datum点。

![](../graphics/dtm-hpted-nls.png)

**通过指定到两条边的距离创建datum点：**

1. 从主菜单栏中，选择****Tools**![](../graphics/images/arrow.gif)**Datum****。此时将出现**Create Datum**对话框。该对话框概述了您可以创建的datum几何体类型。Abaqus/CAE在提示区域中显示提示以指导您完成此过程。**提示：**您也可以使用模块工具箱中的![](../graphics/ico_dtmPtOffsetEdges.png)工具创建datum点。有关工具箱中datum工具的图示，请参见["使用Datum工具集，" 第62.2节](pt06ch62s02.md)。
2. 从对话框顶部的类型列表中，选择**Point**。**Method**列表指示您可以使用的方法来创建datum点。
3. 从**Method**列表中，选择**Offset from 2 edges**。
4. 从当前视口中的零件或装配中，选择要在其上创建datum点的平面。
5. 在指定平面上选择一条直边，并输入datum点到该边的距离。
6. 在该平面上选择第二条直边，并输入datum点到该边的距离。datum点会出现。您可以通过从主菜单栏中选择****Feature**![](../graphics/images/arrow.gif)**Edit****并修改datum到每条边的距离来修改datum点的位置。
