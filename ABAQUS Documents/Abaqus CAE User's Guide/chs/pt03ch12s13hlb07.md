# 12.13.9 创建表面截面





表面截面用于定义空间中本身没有固有刚度且行为类似于零厚度膜的表面的截面属性。请参见[Abaqus Analysis User's Guide的第32.7.1节，"表面元素"](../usb/usb-link.md#usb-elm-esurface)。

**指定表面截面属性：**

1. 从主菜单栏，选择****Section**![](../graphics/images/arrow.gif)**Create****。出现**Create Section**对话框。
   **提示：**您也可以在**Section Manager**中点击**Create**，或在Property模块工具箱中选择创建截面工具![](../graphics/ico_section.png)。
2. 输入截面名称。有关命名对象的更多信息，请参见["使用基本对话框组件，"第3.2.1节"](pt01ch03s02s01.md)。
3. 选择**Shell**作为截面**Category**，选择**Surface**作为截面**Type**，然后点击**Continue**。出现表面截面编辑器。
4. 要将额外质量带入模型，请切换**Density**并输入单位面积质量密度。
5. 点击![](../graphics/ico_rebarLayers.png)以在表面截面中定义钢筋层，如["定义钢筋层，"第12.13.19节"](pt03ch12s13hlb15.md)中所述。
6. 点击**OK**保存更改并关闭表面截面编辑器。
