# 11.22.5 添加平面壳特征





从主菜单栏选择**形状**![](../graphics/images/arrow.gif)**壳**![](../graphics/images/arrow.gif)**平面**，向当前视口中的零件添加平面壳特征。无论当前视口中零件的建模空间如何，平面壳工具始终可用。

您通过在选定面上绘制特征来添加平面壳特征。

**添加平面壳特征：**

1. 从主菜单栏，选择**形状**![](../graphics/images/arrow.gif)**壳**![](../graphics/images/arrow.gif)**平面**。
2. 如果需要，指定要用于为平面壳特征的草图选择原点的方法。
3. 如果零件的建模空间是二维或轴对称，Abaqus/CAE会进入草图编辑器并对齐零件和草图的*X*和*Y*轴。
4. 使用草图编辑器绘制平面壳。在提示区域，单击**完成**表示完成绘制。
