# 10.7.10 从STEP格式文件导入零件





从主菜单栏选择**文件**![](../graphics/images/arrow.gif)**导入**![](../graphics/images/arrow.gif)**零件**，可以从STEP格式文件导入零件。您可以将草图和零件从Abaqus/CAE导出为STEP格式文件，但不能将装配体导出为STEP格式文件。

**从STEP格式文件导入一个或多个零件：**

1. 从主菜单栏，选择**文件**![](../graphics/images/arrow.gif)**导入**![](../graphics/images/arrow.gif)**零件**。出现**导入零件**对话框。
2. 从**导入零件**对话框底部的**文件过滤器**菜单中，选择STEP** (*.stp, *.step)**。Abaqus/CAE列出所选目录中所有具有`.stp`或`.step`文件扩展名的文件。
3. 选择包含要导入的零件的STEP文件，然后单击**确定**。Abaqus/CAE显示**从STEP文件创建零件**对话框。
4. 单击**名称-修复**选项卡设置以下内容：**名称** 将零件或零件导入Abaqus/CAE时，您可以使用STEP格式文件中指定的零件名称，也可以为导入的零件或零件提供名称。如果导入多个零件并提供零件名称，Abaqus/CAE将使用您的名称并向其附加数字；例如，如果您指定零件名称`ImportedPart`并创建三个单独的零件，新导入的Abaqus/CAE零件将命名为`ImportedPart-1`、`ImportedPart-2`和`ImportedPart-3`。**零件过滤器** STEP文件可以包含多个零件，Abaqus/CAE默认导入文件中的所有零件。如果导入STEP文件中的所有零件，您可以为每个零件创建单独的Abaqus零件，也可以将它们组合成一个Abaqus零件；此外，如果组合这些零件，Abaqus/CAE允许您使用用户指定的缝合容差值将这些导入的零件缝合在一起。如果要从STEP文件导入单个零件，可以切换**导入零件编号**并输入要从文件导入的单个零件的编号。
5. 单击**零件属性**选项卡设置以下内容：-正在导入的零件的建模空间。导入零件时，Abaqus/CAE会扫描文件并尝试确定建模空间。-正在导入的零件的类型。Abaqus/CAE始终假定零件类型为可变形。或者，您可以选择**离散刚体**导入离散刚体零件，或选择**欧拉**导入欧拉零件。更多信息，请参见"零件类型"，第11.4.2节。同样，在大多数情况下您应该接受默认设置；更多信息，请参见"什么是零件属性？"，第10.3.2节。
6. 单击**缩放**选项卡以在导入过程中缩放零件。-选择**不缩放**以保持文件中存储的尺寸。-选择**使用文件中的变换，包括缩放**以从STEP文件读取缩放因子、旋转矩阵和平移矩阵。-选择**将所有长度乘以**，然后输入缩放因子。Abaqus/CAE将缩放因子应用于文件中的所有坐标。因此，距原点的任何偏移都将按比例缩放。
7. 单击**确定**导入STEP零件并关闭**从STEP文件创建零件**对话框。Abaqus/CAE扫描STEP文件并根据所选选项启动修复过程。如果希望取消导入过程，请单击提示区域中的**停止**。导入单个零件时，Abaqus/CAE会显示一条消息，指示零件是否存在任何有效性或精度问题。如果从单个STEP文件导入多个零件，可以使用**零件管理器**检查任何零件是否无效或不精确。如有必要，使用查询工具集定位无效和不精确几何的区域。更多信息，请参见"在零件模块中使用查询工具集"，第11.16.4节。
