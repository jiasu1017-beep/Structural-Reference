# 62.8.5 通过一点并垂直于边创建datum平面







您可以通过选择一个点和一个边来定位datum平面；datum平面通过该点并垂直于选定的边。

![](../graphics/dtm-hppn-nls.png)

**通过一点并垂直于边创建datum平面：**

1. 从主菜单栏中，选择****Tools**![](../graphics/images/arrow.gif)**Datum****。此时将出现**Create Datum**对话框。该对话框概述了您可以创建的datum几何体类型。Abaqus/CAE在提示区域中显示提示以指导您完成此过程。**提示：**您也可以使用模块工具箱中的![](../graphics/ico_dtmPlanePointNormal.png)工具创建datum平面。有关工具箱中datum工具的图示，请参见["使用Datum工具集，" 第62.2节](pt06ch62s02.md)。
2. 从对话框顶部的类型列表中，选择**Plane**。**Method**列表指示您可以使用的方法来创建datum平面。
3. 从**Method**列表中，选择**Point and normal**。
4. 从当前视口中的零件或装配中，选择一个点。
5. 从当前视口中的零件或装配中，选择一条边。datum平面会出现。您无法修改使用此方法创建的datum平面；您必须删除旧平面并创建一个新的。
