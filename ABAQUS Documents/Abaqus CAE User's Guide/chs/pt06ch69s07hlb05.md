# 69.7.5 修复细长面







您可以将细长面视为面中的多余材料。您可以修复并移除三维实体或壳的面或二维平面部件中不需要的细长面。您必须选择包含要修复的细长面的面和面上的两个点。Abaqus/CAE 使用这两个点将所选面划分为两个区域。然后它删除较小的区域（细长面）并保留较大的区域。如果较小的区域与其相邻区域相比相对较大，则操作失败。修复细长面的操作作为部件的特征存储；因此，您可以使用模型树删除或抑制操作。

**修复细长面：**

1. 从主菜单栏中选择 ****Tools**![](../graphics/images/arrow.gif)**Geometry Edit****（工具→几何编辑）。Abaqus/CAE 显示 **Geometry Edit**（几何编辑）对话框。**提示：**您也可以使用位于 Part（部件）模块工具箱中编辑工具处的 ![](../graphics/ico_repairRemoveSliver.png) 工具来移除细长面。 有关工具箱中编辑工具的图表，请参阅["使用几何编辑工具集，" 第 69.1 节"](pt06ch69s01.md)。
2. 从对话框中选择 **Face**（面）类别和 **Repair sliver**（修复细长面）方法。
3. 选择要修复的面。您可以从视口选择面或选择包含面的现有集合。**注意：**默认选择方法基于您最近使用的选择方法。要恢复为其他方法，请点击提示区域右侧的 **Select in Viewport**（在视口中选择）或 **Sets**（集合）。**提示：**如果您无法选择所需的面，可以使用 **Selection**（选择）工具栏更改选择行为。 有关更多信息，请参阅["使用选择选项，" 第 6.3 节"](pt01ch06s03.md)。
4. 选择定义细长面角点的点。
5. 选择定义细长面相对角点的第二个点。
6. Abaqus/CAE 在两个选定点之间画一条线，并移除 resulting small face。
