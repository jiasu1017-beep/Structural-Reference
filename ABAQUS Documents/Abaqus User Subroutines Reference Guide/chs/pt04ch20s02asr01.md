# 20.2.1 aStudy.combine()





### 20.2.1 ***aStudy*.combine()**：为参数化研究组合参数样本。

**产品：** Abaqus/Standard  Abaqus/Explicit  

此命令用于组合参数化研究中的采样参数值。

##### **参考**

- ["参数化研究脚本，" 第20.1.1节](pt04ch20s01aus108.md)

### **命令：**

***aStudy*.combine** (*token*, *additional data*)

### **符号：**

CROSS

使用此符号从参数化研究中参数的采样值创建"十字形"模式的设计。

MESH

使用此符号创建"网格"模式的设计，其中参数的每个采样值与参数化研究中每个其他参数的每个采样值组合。

PRINT

使用此符号打印为参数化研究创建设计点。

TUPLE

使用此符号创建由参数化研究中参数采样值的元组组成的"元组"模式的设计。

##### **CROSS、MESH和TUPLE的额外数据：**

### **可选数据：**

**name**

将**name**设置为正在定义的设计集名称；此名称必须用引号括起来。如果未指定名称，则会创建默认设计集名称。

##### **PRINT的额外数据：**

### **可选数据：**

**name**

将**name**设置为正在打印信息的设计集名称；此名称必须用引号括起来。如果未指定设计集名称，则会打印参数化研究中所有设计集的信息。
