# 15.12.6 创建约束





您可以创建以下约束：
- 将两个独立表面绑定在一起的绑定约束，使它们之间没有相对运动。
- 允许您将区域集合指定为刚体的刚体约束。
- 允许您指定仅用于显示的部件实例的显示体约束。
- 允许您将表面的运动约束到参考节点运动的耦合约束。
- 允许您将一个或多个点移动到指定表面的调整点约束。
- 允许您将区域从节点的运动约束到单个点运动的多点约束。
- 允许您将壳边缘的运动耦合到相邻实体面的运动的壳到实体耦合约束。
- 允许您将模型区域嵌入到模型的"宿主"区域或整个模型中的嵌入区域约束。
- 描述单个自由度之间线性约束的方程约束。

**创建约束的步骤：**

1. 从主菜单栏选择 ****Constraint**![](../graphics/images/arrow.gif)**Create****。**提示：**您也可以使用「相互作用」模块工具箱中的![](../graphics/ico_constraint.png)工具创建约束。
2. 在出现的「Create Constraint」对话框中，执行以下操作：1. 为约束命名。有关命名对象的更多信息，请参见「使用基本对话框组件」，第3.2.1节。2. 选择所需的约束类型。
3. 点击「Continue」创建约束并关闭「Create Constraint」对话框。
4. 如适用，选择要应用约束的区域。更多信息，请参见第6章，「在视口中选择对象」。
5. 在出现的编辑器中，输入定义约束所需的任何数据。
