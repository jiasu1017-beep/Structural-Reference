# 3.8.13 压电子模型





**产品：**Abaqus/Standard  

### I. C3D8E单元

### 测试的单元

C3D8E    CPE8E    CPS8E    

### 测试的功能

子模型功能应用于平面应力、平面应变和三维实体压电单元。全局模型由一个单元组成；子模型建模全局模型的一半，由两个单元组成。使用静态和稳态程序。

### 问题描述

**模型：**

全局模型尺寸为7×7×7。子模型建模全局模型的右半部分。

**材料：**

| 杨氏模量 | 3×10^6 |
| --- | --- |
| 泊松比 | 0.3 |
| 密度 | 10 |
| 热膨胀系数 | 0.0001 |
| 介电常数 | 各向异性（见输入文件） |

**边界条件：**

在x=0平面上，电荷和位移![](../graphics/ver_eqn00528.gif)和![](../graphics/ver_eqn00529.gif)为零。在y=0平面上，电荷和![](../graphics/ver_eqn00196.gif)位移为零。

**载荷：**

初始温度在所有节点为10。在第一步中，3000的分布电荷施加在x=7平面和y=7平面上。在第二步中，40的温度施加在所有节点上。

### 结果与讨论

子模型中驱动节点的节点位移和电势从全局模型结果正确插值。

### 输入文件

[pgc38efm.inp](../eif/pgc38efm.inp)

全局分析。

[psc38ef1.inp](../eif/psc38ef1.inp)

子模型分析。
