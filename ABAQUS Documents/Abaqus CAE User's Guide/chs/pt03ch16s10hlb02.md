# 16.10.2 定义位移/旋转边界条件





您可以创建位移/旋转边界条件，以将所选自由度的运动约束为零，或为每个所选自由度指定位移或旋转。对于流动步骤，仅位移可用。

**创建或编辑位移/旋转边界条件：**

1. 使用以下方法之一显示位移/旋转边界条件编辑器：- 若要创建新的位移/旋转边界条件，请遵循"创建边界条件"，第16.8.2节中概述的过程（**类别**：`Mechanical`；**所选步骤的类型**：`Displacement/Rotation`）。- 若要使用菜单或管理器编辑现有的位移/旋转边界条件，请参见"编辑依赖于步骤的对象"，第3.4.13节。若要编辑应用边界条件的区域，请参见"编辑应用规定条件的区域"，第16.8.4节。
2. 如果您在屈曲步骤中创建边界条件，请选择**用于BC的选项**，以指定您希望边界条件用于其中计算的选项。更多信息，请参见"Abaqus Analysis User's Guide"第6.2.3节"Eigenvalue buckling prediction"中的"边界条件"。
3. 默认情况下，使用全局坐标系来定义边界条件。要更改应用边界条件的坐标系，请单击**CSYS**选项的![](../graphics/ico_selectBlue.png)，并执行以下操作之一：- 在视口中选择现有的基准坐标系。- 按名称选择现有的基准坐标系。1. 从提示区域单击**基准坐标系列表**以显示基准坐标系列表。2. 从列表中选择一个名称，然后单击**确定**。- 从提示区域单击**使用全局CSYS**以恢复到全局坐标系。此坐标系编辑选项仅在创建边界条件的步骤中可用。
4. 如果编辑器顶部附近出现**方法**字段，单击该字段右侧的箭头，并选择以下之一：- 如果要为特定自由度指定值，请选择**指定约束**。- 如果要将自由度固定在先前常规步骤中的最终值，请选择**固定在当前位置**。**方法**选项仅在两种方法对所选步骤都有效时才会出现在编辑器中。
5. 如果可用，单击**分布**字段右侧的箭头，并执行以下操作：- 选择**均匀**以定义均匀边界条件。- 选择**用户定义**以在用户子程序[`DISP`](../sub/sub-link.md#sub-xsl-disp)中定义边界条件。请参见以下章节获取更多信息：- "指定常规作业设置"，第19.8.6节- "Abaqus User Subroutines Reference Guide"第1.1.4节"DISP"用户定义的边界条件不适用于流动步骤。- 选择分析场（标记为(A)）或离散场（标记为(D)）以定义空间变化的边界条件。只有对这些边界条件类型有效的分析场和离散场才会在选择列表中显示。或者，您可以单击![](../graphics/ico_analyticalField.png)创建新的分析场。（更多信息，请参见第58章"分析场工具集"。）**分布**选项仅在您创建边界条件或在创建边界条件的步骤中编辑边界条件时才会出现在编辑器中。如果您选择了**固定在当前位置**方法，则此选项不可用。
6. 如果您选择了**均匀**、分析场或离散场分布选项，请执行以下步骤：1. 使用适当的方法定义边界条件：**如果每个自由度旁边没有出现文本字段：**- 切换打开一个自由度，以将值固定为零（如果您在初始步骤中定义自由度）或在先前分析步骤结束时的最终值（如果您在第二个分析步骤或后续步骤中定义自由度）。- 切换关闭一个自由度，以使该自由度不受约束。**如果每个自由度旁边出现文本字段：**- 切换打开一个自由度，以约束该自由度。文本字段变得可用，您可以在其中为该自由度指定值。如果您在当前步骤中创建边界条件，则文本字段中会出现默认值零。如果您在当前步骤中修改边界条件，则文本字段中会出现从前一个步骤传播的值。- 切换关闭一个自由度，以使该自由度不受约束。如果您在文本字段中修改了默认值或传播值后关闭该自由度，则修改后的值将丢失。如果您重新打开该自由度，则默认值或传播值将重新出现在文本字段中。- 如果编辑器底部出现**幅值**，请单击该字段右侧的箭头，并从出现的列表中选择您选择的幅值。或者，您可以单击![](../graphics/ico_amplitudeSmall.png)创建新的幅值。（更多信息，请参见第57章"幅值工具集"。）2. 单击**确定**保存您的数据并退出编辑器。
7. 如果您选择了**用户定义**分布选项，请执行以下步骤：1. 如果需要，使用以下技术定义位移：- 切换打开一个自由度，以约束该自由度。如果您在初始步骤中，则该自由度被设置为零。如果您在初始步骤以外的任何步骤中，则文本字段变得可用，您可以在其中为该自由度指定值。- 切换关闭一个自由度，以使该自由度不受约束。您在编辑器中为特定自由度输入的数据被传递到用户子程序中。2. 单击**确定**保存您的数据并退出编辑器。3. 进入作业模块并显示感兴趣的分析作业的作业编辑器。（更多信息，请参见"创建、编辑和操作作业"，第19.7节。）4. 在作业编辑器中，单击**常规**选项卡，并指定包含定义边界条件的用户子程序的文件。更多信息，请参见"指定常规作业设置"，第19.8.6节。**注意：**您只能在作业编辑器中指定一个用户子程序文件；如果您的分析涉及多个用户子程序，您必须将用户子程序合并到一个文件中，然后指定该文件。
