# 9.10.2 为您的Abaqus/CAE模型添加描述





在Abaqus/CAE中，您可以分别在**编辑模型属性（Edit Model Attributes）**对话框和**编辑材料（Edit Material）**对话框中为模型和模型中的材料输入描述。当您提交作业进行分析时，Abaqus/CAE会生成一个输入文件，并使用注释行将这些描述写入输入文件。模型描述的注释行紧跟在输入文件的标题之前，材料描述的注释行紧跟在材料定义之前。

有关使用描述的相关信息，请参阅["从Abaqus输入文件导入模型，"第10.5.2节"](pt02ch10s05s02.md#usi-imp-inp-desc)中的"导入描述"。
