# 6.2.9 从选择中排除对象





当你从视口中选择一个对象时，你的选择包括与该对象关联的所有较低维度的实体。例如，如果你选择一个单元体，你的选择包括与该单元体关联的所有面、边和顶点。同样，如果你选择一条边，你的选择包括与该边关联的所有顶点。在某些情况下，你可能希望从选择中排除较低维度的实体。例如，如果你选择一条边包含在集合中，你可能不希望集合包含边两端的顶点。从选择中排除较低维度的实体可能会解决你遇到的过约束问题。

**要从选择中排除对象：**

1. 使用选择、拖动选择、**[Ctrl]****+点击**和 **[Shift]****+点击**的组合选择所有对象。Abaqus/CAE 将选中的对象以红色高亮显示。
2. **[Ctrl]****+点击**一个对象以将其从选择中排除。Abaqus/CAE 将排除的对象以紫色高亮显示。
