# 3.2.30 将ANSYS输入文件转换为部分Abaqus输入文件





**产品：**Abaqus/Standard  Abaqus/Explicit

##### **参考**

- ["Abaqus执行程序：概述，" 第3.1.1节](pt01ch03s01abo02.md)

### 概述

ANSYS到Abaqus的转换器将ANSYS块编码数据库文件中的某些实体转换为其在Abaqus输入文件中的等价形式。

### 使用转换器

**abaqus fromansys**转换器可以将ANSYS块编码数据库文件（`.cdb`）转换为一个"扁平化"的Abaqus输入文件；即，不使用部件和装配方式编写的Abaqus输入文件。`.cdb`文件必须在ANSYS中使用以下命令创建：

`CDWRITE , , <jobname>, cdb`

`CDWRITE`命令的第二个字段可以包含`ALL`或`DB`。第八个字段可以包含`BLOCKED`。`CDWRITE`命令的任何其他使用方式都会给转换器带来问题。

### 命令行选项

**job**

此选项用于指定转换器输出的Abaqus输入文件的名称。它也是包含ANSYS数据的输入文件的默认名称。转换器生成的诊断信息将写入名为`*job-name*.log`的文件。

**input**

如果文件名与*job-name*不同，此选项用于指定包含ANSYS数据的文件的名称。
