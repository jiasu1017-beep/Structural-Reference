# 47.1.1 什么是X–Y数据对象，什么是X–Y图？









### 47.1.1 什么是X--Y数据对象，什么是X--Y图？

*X–Y*数据对象是Abaqus/CAE以两列形式存储的有序对集合——一列*X*列和一列*Y*列。*X–Y*数据可以源自输出数据库或ASCII文件，或者您可以使用键盘输入数据。此外，您可以通过组合现有的*X–Y*数据对象来派生*X–Y*数据。例如，您可以将包含应力随时间变化的*X–Y*数据对象与包含应变随时间变化的第二个*X–Y*数据对象组合，以产生包含等效时间下应力与应变关系的*X–Y*数据对象。您可以保存*X–Y*数据对象；并且您可以编辑、复制、重命名和删除它们。

默认情况下，您保存的*X–Y*数据对象仅在会话期间保留。如果要让*X–Y*数据对象可用于后续会话，可以将它们保存到XML文件、模型数据库或输出数据库，方法是使用****文件**![](../graphics/images/arrow.gif)**保存会话对象****选项；或者可以使用**XY数据管理器**中的**复制到ODB**按钮将它们复制到输出数据库文件。

Abaqus/CAE可以将*X–Y*数据显示为*X–Y*图。*X–Y*图是一个变量相对于另一个变量的二维图形。*X–Y*图的示例包括温度与时间、载荷与位移以及应力与应变。您可以在一个*X–Y*图中显示多个*X–Y*数据对象，并且可以使用自定义选项来控制每个数据对象以及*X–Y*图组件的整体外观。这些组件包括坐标轴；图例，它是关联图中每个*X–Y*曲线与其所代表的*X–Y*数据对象的密钥；以及*X–Y*曲线。[图47-1](pt05ch47s01s01.md#usv-xyplot-whatis)显示了一个显示三个*X–Y*数据对象的*X–Y*图。
