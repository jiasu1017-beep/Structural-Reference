# 8.2.4 选择图像目标





你可以选择将图像直接发送到打印机，或将图像保存到文件。如果你直接将图像发送到打印机，你可以选择要使用的 Windows 打印机。如果你在另一个平台上或你的 Windows 系统上没有打印机驱动程序，Abaqus/CAE 会选择 PostScript 格式，你可以指定打印命令。附加选项允许你选择份数、纸张大小、方向、边距、图像质量，以及是否包含日期和 SIMULIA 标志。

如果你选择将图像保存到文件，你必须提供文件名并选择以下文件格式之一：

**PostScript**

如果希望保存的图像与 Abaqus/CAE 打印到 PostScript 打印机的图像完全相同，请选择 PostScript (**PS**)。此格式的附加选项允许你选择纸张大小、方向、边距、图像分辨率，以及是否包含日期和 SIMULIA 标志。更多信息，请参阅["自定义发送到 PostScript 打印机或文件的图像，" 第 8.2.6 节](pt01ch08s02hlb06.md)。

**封装的 PostScript**

如果希望将保存的图像合并到单独文档中，请选择封装的 PostScript (**EPS**)；例如，字处理文件。此格式的附加选项允许你指定图像的大小和分辨率。更多信息，请参阅["自定义保存到封装的 PostScript 文件中的图像，" 第 8.2.7 节](pt01ch08s02hlb07.md)。

****TIFF****

如果希望将保存的图像合并到单独文档中，请选择 **TIFF**；例如，字处理文件。此格式的附加选项允许你指定图像的大小。更多信息，请参阅["自定义保存到 TIFF、PNG 或 SVG 文件中的图像，" 第 8.2.8 节](pt01ch08s02hlb08.md)。

****PNG****

如果希望将保存的图像合并到单独文档中，请选择 **PNG**；例如，用于在万维网上显示的 HTML 文件。此格式的附加选项允许你指定图像的大小。更多信息，请参阅["自定义保存到 TIFF、PNG 或 SVG 文件中的图像，" 第 8.2.8 节](pt01ch08s02hlb08.md)。

****SVG****

如果希望将保存的图像合并到单独文档中，请选择 **SVG**；例如，用于在万维网上显示的 HTML 文件。此格式的附加选项允许你指定图像的大小。更多信息，请参阅["自定义保存到 TIFF、PNG 或 SVG 文件中的图像，" 第 8.2.8 节](pt01ch08s02hlb08.md)。

**要选择图像目标：**

1. 从主菜单栏中，选择****File**![](../graphics/images/arrow.gif)**Print****。**提示：**你也可以单击**文件**工具栏中的![](../graphics/ico_filePrint.png)。**打印**对话框出现。
2. 从**设置**字段中的**目标**按钮中，选择以下之一：
   **打印机**
   选择**打印机**将图像发送到打印机。在 Windows 系统上，你可以从已安装打印机列表中选择，类似于其他 Windows 程序。在其他系统上，或在未安装打印机驱动程序的 Windows 系统上，你可以在**打印命令**文本字段中输入打印命令。此命令应该与你站在工作站上打印 PostScript 文件使用的命令相同。不要在打印命令中包含文件名；Abaqus/CAE 会自动将文件名附加到你的命令。关于站点有效命令的详细信息，请咨询你的系统管理员。无论你选择打印机还是输入打印命令，点击**份数**字段中的箭头来设置要打印的份数，或直接在文本字段中输入份数。你最多可以打印 100 份。如有需要，单击**页面设置**和**打印机属性**按钮（对于 Windows 打印机）或 **Postscript 格式选项**按钮（对于使用打印命令的系统）来指定页面大小、打印图像质量和其他选项。
   **文件**
   选择**文件**将图像发送到文件。有两种提供文件名的方式：
   ****文件名****
   在**文件名**文本字段中输入名称。你可以使用任何合法的 Windows 或 UNIX 文件名字符；例如，在 Windows 系统上：
   | `stressfield.png` |
   | `..\..\nozzle\presentation\injector_mesh` |
   | `~\pump\actuator\strainpattern.eps` |
   如果你没有输入文件扩展名，Abaqus/CAE 会将扩展名（`.ps`、`.eps`、`.png`、`.tif` 或 `.svg`）附加到文件名。
   ****选择****
   使用**选择**按钮使用标准文件浏览器提供文件名。有关文件选择的更多信息，请参阅["使用文件选择对话框，" 第 3.2.10 节](pt01ch03s02s10.md)。
3. 如果你选择将图像打印到文件，点击**格式**字段旁边的箭头选择 PostScript (**PS**)、封装的 PostScript (**EPS**)、**TIFF**、**PNG** 或 **SVG** 格式文件。如有需要，点击相应的选项按钮指定附加选项。
4. 当你完成**打印**对话框时，单击**确定**生成所需的输出。Abaqus/CAE 生成输出并关闭**打印**对话框。你的**打印**对话框设置会在会话期间保存。
