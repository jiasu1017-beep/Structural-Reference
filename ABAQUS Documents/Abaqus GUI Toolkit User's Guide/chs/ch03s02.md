# 3.2 刚体







在 Abaqus 中，刚体是节点和单元的集合，其运动由单个节点（称为**刚体参考节点**）的运动控制，如[图 3-7](ch03s02.md#gsa-rigidelem) 所示。 

**图 3-7** 形成刚体的单元。

![](../graphics/gss-rigidelem-nls.png)

刚体的形状可以定义为通过旋转或拉伸二维几何轮廓获得的分析曲面，也可以定义为用节点和单元网格划分的离散刚体。刚体在模拟过程中不变形，但可以经历大的刚体运动。离散刚体的质量和惯性可以基于其单元的贡献计算，也可以直接分配。

刚体的运动可以通过在刚体参考节点上施加边界条件来指定。刚体上的载荷由施加到属于刚体的节点上的集中载荷和施加到属于刚体的单元上的分布载荷生成，或由施加到刚体参考节点上的载荷生成。刚体通过与变形单元的节点连接以及与变形单元的接触来与模型的其余部分相互作用。

刚体的使用在[第 12 章，"接触"](ch12.md) 中有说明。

### 3.2.1 确定何时使用刚体

刚体可用于模拟非常硬的组件，这些组件要么是固定的，要么正在经历大的刚体运动。它们也可用于模拟变形组件之间的约束，并提供指定某些接触相互作用的便捷方法。当 Abaqus 用于准静态成形分析时，刚体非常适合建模工具（如冲头、模具、拉延筋、压料板、滚轮等），也可以作为约束方法有效。

出于验证目的，将模型的某些部分设为刚体可能很有用。例如，在复杂模型中，远离感兴趣区域的元素可以作为刚体的一部分包含，从而在模型开发阶段实现更快的运行时间。当您对模型满意后，可以移除刚体定义，并在整个模型中合并准确的变形有限元表示。

相对于变形有限元，用刚体表示模型部分的主要优势是计算效率。不对属于刚体的单元执行单元级计算。虽然更新刚体节点的运动和组装集中载荷和分布载荷需要一些计算努力，但刚体的运动完全由刚体参考节点上最多六个自由度决定。

在 Abaqus/Explicit 中，刚体在建模结构的相对硬的部分方面特别有效，因为在这些部分中跟踪应力波和分布并不重要。硬区域中单元稳定时间增量估计可能导致非常小的全局时间增量。由于刚体和属于刚体的单元不影响全局时间增量，因此在硬区域中使用刚体代替变形有限元表示可以产生更大的全局时间增量，而不会显著影响解的整体准确性。

在 Abaqus 中用分析刚曲面定义的刚体在计算成本上比离散刚体略低，并且可能产生更平滑的结果。例如，在 Abaqus/Explicit 中，与分析刚曲面接触往往比与离散刚体接触噪声更小，因为分析刚曲面可以是光滑的，而离散刚体本质上是多面的。但是，分析刚曲面可以定义的形状是有限的。

### 3.2.2 刚体的组成

要创建离散刚体，请使用 [*RIGID BODY](../key/key-link.md#usb-kws-mrigidbody) 选项作为形成刚体的单元的元素集的属性引用。使用 REF NODE 参数为刚体分配一个刚体参考节点。刚体参考节点同时具有平移和旋转自由度，必须为每个刚体定义。刚体参考节点的位置并不重要，除非向物体施加旋转或需要关于穿过物体的某个轴的反应力矩。在任一情况下，节点应放置在穿过物体的所需轴上。

```
[*RIGID BODY](../key/key-link.md#usb-kws-mrigidbody), REF NODE=*<node>*, ELSET=*<element set name>,*
PIN NSET=*<node set name>*, TIE NSET=*<node set name>*

```

除了刚体参考节点外，离散刚体还包括通过将单元和节点分配给刚体而生成的节点集合。这些节点称为**刚体从节点**（见[图 3-7](ch03s02.md#gsa-rigidelem)），提供到其它单元的连接。属于刚体的节点有两种类型：
- 销节点，只有平移自由度。
- 约束节点，同时具有平移和旋转自由度。

刚体节点类型由连接到刚体的单元类型确定。在直接将节点分配给刚体时，也可以指定或修改节点类型。对于销节点，只有平移自由度属于刚体，这些自由度的运动受刚体参考节点运动的约束。对于约束节点，平移和旋转自由度都属于刚体，并受刚体参考节点运动的约束。

定义刚体的节点不能对其施加任何边界条件、多点约束或约束方程。但是，可以在刚体参考节点上施加边界条件、多点约束、约束方程和载荷。

### 3.2.3 刚单元

Abaqus 中的刚体功能允许大多数单元——不仅仅是刚单元——成为刚体的一部分。例如，如果 [*RIGID BODY](../key/key-link.md#usb-kws-mrigidbody) 选项引用包含形成刚体的单元的单元集，则壳单元或刚单元可用于模拟相同的效果。管理刚体的规则（如如何施加载荷和边界条件）适用于形成刚体的所有单元类型，包括刚单元。

所有刚单元的名称都以字母"R"开头。接下来的字符表示单元的维数。例如，"2D"表示单元是平面的；"AX"表示单元是轴对称的。最后一个字符表示单元中的节点数。

**刚单元库**

三维四边形（R3D4）和三角形（R3D3）刚单元用于建模三维刚体的二维表面。另一个单元——两节点、刚梁单元（RB3D2）——在 Abaqus/Standard 中提供，主要用于模拟必须施加流体拖曳和浮力载荷的海洋结构组件。

对于平面应变、平面应力和轴对称模型，有两节点刚单元可用。在 Abaqus/Standard 中还可使用两节点平面刚梁单元，主要用于二维模拟海洋结构。

**自由度**

只有刚体参考节点具有独立自由度。对于三维单元，参考节点具有三个平移和三个旋转自由度；对于平面和轴对称单元，参考节点具有自由度 1、2 和 6（绕 3 轴的旋转）。

连接到刚单元的节点只有从属自由度。这些节点的运动完全由刚体参考节点的运动决定。对于平面和三维刚单元，唯一的从属自由度是平移。Abaqus/Standard 中的刚梁单元具有与相应变形梁单元相同的从属自由度：三维刚梁为 1–6，平面刚梁为 1、2 和 6。

**物理属性**

所有刚单元必须引用 [*RIGID BODY](../key/key-link.md#usb-kws-mrigidbody) 选项。对于平面和梁单元，可以在数据行上定义横截面积。对于轴对称和三维单元，可以在数据行上定义厚度；仅在向刚单元施加体力时才需要这些数据。默认厚度为零。或者，NODAL THICKNESS 参数基于节点处的厚度定义平均面元厚度。在施加体力时或当厚度需要用于接触定义时需要这些数据。

**公式和积分**

由于刚单元不是变形的，它们不使用数值积分点，也没有可选的公式。

**单元输出变量**

没有单元输出变量。刚单元的唯一输出是节点的运动。此外，在刚体参考节点处提供反作用力和反作用力矩。
