# 15.13.2 为通用接触指定和修改接触属性分配





通用接触相互作用的属性（如接触属性）在相互作用编辑器的「Attribute Assignments」部分独立于接触域指定。接触属性可以全局分配给通用接触相互作用，也可以单独分配给通用接触域内的特定区域。有关通用接触的简要概述，请参见「理解相互作用」，第15.3节。

您可以在原始相互作用定义传播到的任何步骤中更改分配给通用接触相互作用的接触属性。在Abaqus/Standard中，仅允许摩擦行为在原始接触属性定义和新接触属性定义之间更改（请参见「定义接触相互作用属性」，第15.14.1节中的「为机械接触属性选项指定摩擦行为」）——Abaqus使用从通用接触相互作用创建到分配新属性的步骤的原始摩擦定义；新摩擦定义用于后续步骤。

**指定接触属性分配的步骤：**

1. 使用以下方法之一显示通用接触相互作用编辑器：- 要创建新的通用接触相互作用，请按照「定义通用接触」，第15.13.1节中的说明操作。- 要编辑现有的通用接触相互作用，请从主菜单栏选择 ****Interaction**![](../graphics/images/arrow.gif)**Edit**![](../graphics/images/arrow.gif)***相互作用名称*****。
2. 点击相互作用编辑器「Attribute Assignments」部分中的「Contact Properties」选项卡（如果尚未选择）。
3. 如有必要，点击相互作用编辑器「Contact Properties」部分中的![](../graphics/ico_interactionProperty.png)创建接触相互作用属性；请参见「定义接触相互作用属性」，第15.14.1节，获取更多信息。
4. 使用以下方法之一指定相互作用的接触属性分配：- 要将接触属性全局分配给整个接触域，请从「Global property assignment」旁边的列表中选择一个属性。- 要为单个表面对分配不同的接触属性：1. 点击「Individual property assignments」旁边的![](../graphics/ico_edit.png)。此时会出现「Edit Individual Contact Property Assignments」对话框。默认情况下，当您从列表或表中选择一个表面时，Abaqus/CAE会在视口中高亮显示该表面；但是，高亮不适用于「(Global)」、「(Self)」和Eulerian材料表面。您可以关闭对话框底部的「Highlight selected regions」以关闭选择高亮。2. 从左侧第一列的现有表面列表中选择一个或多个表面。选择「(Global)」可以为整个接触域与单个表面之间分配接触属性。**提示：**您可以点击![](../graphics/ico_surface.png)定义新表面并将其添加到列表中。请参见「创建表面」，第73.3.3节，获取定义表面的说明。3. 从第二列的现有表面列表中选择第二个或多个表面以定义表面对。- 当在任一列中选择多个表面时，将生成所有可能的组合。- 要为自接触分配属性，请在第二列中选择相同的表面名称或「(Self)」。- 任何对于接触域之外区域的接触属性分配都将被忽略。4. 从第三列的现有相互作用属性列表中，选择要分配的属性。您可以在「Edit Individual Contact Property Assignments」对话框中创建新的相互作用属性（请参见「定义接触相互作用属性」，第15.14.1节）。5. 点击对话框中间的箭头![](../graphics/usi-itn-contactattribarrow.png)将您的选择转移到属性分配列表中。对话框右侧的表会更新以反映您的选择。6. 根据需要重复上述步骤以完成接触属性分配。如果您想删除接触属性分配，请选择行并点击![](../graphics/ico_delete.png)。**注意：**分配顺序可能是相关的；当属性分配重叠时，最后的分配将优先。7. 点击「OK」保存您的选择并关闭「Edit Individual Contact Property Assignments」对话框。相互作用编辑器重新出现，并更新了单个接触属性分配数量的信息。**注意：**单个接触属性分配优先于全局分配。
5. 点击「OK」创建相互作用并关闭编辑器。

**修改接触属性分配的步骤：**

1. 从主菜单栏选择 ****Interaction**![](../graphics/images/arrow.gif)**Edit**![](../graphics/images/arrow.gif)***相互作用名称*****。此时会出现「Edit Interaction」对话框。
2. 点击相互作用编辑器「Attribute Assignments」部分中的「Contact Properties」选项卡（如果尚未选择）。
3. 点击「Individual property assignments」旁边的![](../graphics/ico_edit.png)。此时会出现「Edit Individual Contact Property Assignments」对话框。
4. 从对话框右侧的表中选择包含要修改的属性分配的行。
5. 从对话框左侧第三列的列表中选择新的相互作用属性。
6. 点击对话框中间的箭头![](../graphics/usi-itn-modcontactprop.png)用新选择替换相互作用属性。
7. 点击「OK」保存您的选择并关闭对话框。相互作用编辑器重新出现。
8. 点击「OK」保存您的更改并关闭相互作用编辑器。
