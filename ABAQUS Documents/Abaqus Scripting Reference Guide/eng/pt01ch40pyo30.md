# 40.30 MdbDataStep object







The MdbDataStep object.It corresponds to same named step in the cae model.

**Access**

```
import visualization
session.mdbData[*name*].steps[*i*]
```

### 40.30.1 Members

The MdbDataStep object has the following member:

*frames*

A [MdbDataFrameArray](pt01ch40pyo28.md) object specifying the list of frames. The list is read-only. There is only one frame in a step.
