# 66.3 创建过滤器







从主菜单栏中选择 ****Tools**![](../graphics/images/arrow.gif)**Filter**![](../graphics/images/arrow.gif)**Create****（工具→过滤器→创建）以创建过滤器。有关过滤器的更多信息（包括由 Abaqus 自动创建的默认过滤器类型），请参阅["过滤场和历史数据，" 第 66.1 节"](pt06ch66s01.md)。

**创建过滤器：**

1. 从主菜单栏中选择 ****Tools**![](../graphics/images/arrow.gif)**Filter**![](../graphics/images/arrow.gif)**Create****（工具→过滤器→创建）。**提示：**您也可以通过点击 **Filter Manager**（过滤器管理器）中的 **Create**（创建）来创建过滤器。出现 **Create Filter**（创建过滤器）对话框。
2. 在 **Name**（名称）字段中，输入过滤器的名称。有关命名对象的信息，请参阅["使用基本对话框组件，" 第 3.2.1 节"](pt01ch03s02s01.md)。
3. 从 **Type**（类型）列表中选择以下之一，然后点击 **Continue**（继续）： - **Butterworth（巴特沃斯）** - **Type I Chebyshev（切比雪夫 I 型）** - **Type II Chebyshev（切比雪夫 II 型）** - **Operator（算子）** 出现 **Edit Filter**（编辑过滤器）对话框。
4. 如果您选择了 **Butterworth** 过滤器类型或其中一种 **Chebyshev** 过滤器类型，请执行以下步骤： 1. 输入 **Cutoff frequency**（截止频率）。**注意：**Abaqus 不执行任何检查以确保截止频率是合适的。如果截止频率设置过低，有效数据将从结果中被过滤掉；如果设置过高，将不执行任何过滤。 2. 输入过滤器的 **Order**（阶数），它必须是小于或等于 20 的正偶数。
5. 如果您选择了其中一种 **Chebyshev** 过滤器类型，请接受默认纹波因子或输入一个值。纹波因子决定您将允许的数据振荡幅度，以换取在已接受和已拒绝数据之间更快的过渡。有关更多信息，请参阅"输出到输出数据库"中的"Abaqus/Explicit 中的过滤输出和对输出进行操作"，Abaqus Analysis User's Guide 第 4.1.3 节../usb/usb-link.md#usb-out-odboutput-filter。
6. 如果需要，指定要过滤数据的边界值类型。 - 选择 **Maximum**（最大值）以输出输出请求中每个变量在每个输出区间内的最大值。 - 选择 **Minimum**（最小值）以输出输出请求中每个变量在每个输出区间内的最小值。 - 选择 **Absolute maximum**（绝对最大值）以输出输出请求中每个变量在每个输出区间内的最大绝对值。您也可以选择 **None**（无）关闭此过滤器的所有边界值相关输出请求。
7. 如果需要，通过在 **Bounding value limit**（边界值限制）字段中输入值来指定边界值请求的限制。此值限制 Abaqus/CAE 为使用此过滤器的变量记录的最高或最低值。如果达到边界值限制，您也可以停止分析。切换开启 **Stop analysis upon reaching limit**（达到限制时停止分析）以在最大或最小值等于使用此过滤器的变量的限制值时停止作业。
8. 如果您为边界值类型选择了 **Minimum**（最小值）、**Maximum**（最大值）或 **Absolute Maximum**（绝对最大值），您可以直接将过滤器应用于不变量。要过滤不变量，请选择 **First**（第一）或 **Second**（第二）作为 **Invariant**（不变量）。
9. 点击 **OK**（确定）创建过滤器。

要将过滤器应用于分析，请在为 Abaqus/Explicit 分析过程创建输出请求时选择 **Apply filter**（应用过滤器）。有关更多信息，请参阅["修改场输出请求，" 第 14.12.2 节"](pt03ch14s12hlb02.md)或["修改历史输出请求，" 第 14.12.3 节"](pt03ch14s12hlb03.md)。
