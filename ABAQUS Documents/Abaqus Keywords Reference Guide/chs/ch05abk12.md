# *ELGEN









### *ELGEN增量单元生成。

此选项用于增量生成单元。

**产品：**Abaqus/Standard  Abaqus/Explicit  Abaqus/CFD  Abaqus/CAE

**类型：**模型数据

**级别：**部件，部件实例

**Abaqus/CAE：**不适用；单元是在对模型进行网格划分时生成的。

##### **参考：**

- ["单元定义，" Abaqus分析用户指南第2.2.1节](../usb/usb-link.md#usb-int-ielement)

### **可选参数：**

ALL NODES

包含此参数以增加IRS型单元和拖链单元的刚体参考节点的节点编号，以及用于定义空间梁第一截面轴方向的节点。默认情况下，这些节点编号不会被增加。

ELSET

将此参数设置为单元（包括主单元）将被分配到的单元集名称。

### **增量生成单元的数据行：**

**第一行：**

根据需要重复此数据行。每行将生成 *N1* *N2* *N3* 个单元，其中 *N1* 是一行中的单元数，*N2* 是一层中的行数，*N3* 是层数。
