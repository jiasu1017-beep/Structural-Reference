# 14.14.2 指定ALE自适应网格约束





您可以为自适应网格域上的网格运动定义ALE自适应网格约束。ALE自适应网格约束可以规定自适应网格域中节点的独立网格运动，也可以指定必须跟随材料的节点。更多信息，请参见以下章节：
- 「ALE自适应网格概述」，Abaqus Analysis User's Guide第12.2.1节
- 「在Abaqus/Explicit中定义ALE自适应网格域」，Abaqus Analysis User's Guide第12.2.2节
- 「在Abaqus/Standard中定义ALE自适应网格域」，Abaqus Analysis User's Guide第12.2.6节

**指定ALE自适应网格约束的步骤：**

1. 从主菜单栏选择 ****Other**![](../graphics/images/arrow.gif)**ALE Adaptive Mesh Constraint**![](../graphics/images/arrow.gif)**Create****。Abaqus/CAE会显示ALE自适应网格约束编辑器。
2. 输入ALE自适应网格约束的名称。
3. 选择要创建约束的步骤。
4. 选择网格运动的类型，「Displacement/Rotation」或「Velocity/Angular velocity」，然后点击「Continue」。
5. 使用以下方法之一选择要约束的区域：- 使用现有集合来定义区域。在提示区域右侧点击「Sets」。从出现的「Region Selection」对话框中选择一个现有集合，然后点击「Continue」。- 使用鼠标在视口中选择一个区域。（更多信息，请参见「在当前视口中选择对象」，第6.2节。）点击鼠标按钮2表示您已完成选择。Abaqus/CAE会在视口中将所选区域以红色高亮显示。**注意：**默认选择方法基于您最近使用的选择方法。如需恢复为另一种方法，请点击提示区域右侧的「Select in Viewport」或「Sets」。此时会出现「Edit ALE Adaptive Mesh Constraint」对话框。
6. 如果您正在编辑现有网格约束，可以点击编辑器顶部的![](../graphics/ico_selectBlue.png)来编辑应用约束的区域。使用步骤5中描述的区域选择技术。
7. 如果要更改应用约束的坐标系（**CSYS**），请点击![](../graphics/ico_selectBlue.png)，并使用以下方法之一：- 在视口中选择一个现有的基准坐标系。- 按名称选择一个现有的基准坐标系。1. 从提示区域点击「Datum CSYS List」以显示基准坐标系列表。2. 从列表中选择一个名称，然后点击「OK」。- 点击提示区域中的「Use Global CSYS」恢复为全局坐标系。此坐标系编辑选项仅在创建约束的步骤中可用。
8. 点击「Motion」字段右侧的箭头，指定您是要规定独立于底层材料的网格运动，还是定义必须跟随底层材料的节点。如果您正在执行Abaqus/Standard分析，您还可以选择在用户子程序[`UMESHMOTION`](../sub/sub-link.md#sub-xsl-umeshmotion)中定义网格运动。
9. 如果您在步骤8中选择了「Independent of underlying material」，请切换您要约束的自由度。文本字段将变为可用，您可以在其中输入该自由度的值。关闭某个自由度可以使其保持未约束状态。
10. 如果您在步骤8中选择了「Independent of underlying material」，请点击「Amplitude」字段右侧的箭头，并从出现的列表中选择您想要的幅值。或者，点击「Create」创建一个新的幅值。（更多信息，请参见第57章，「幅值工具集」。）
11. 点击「OK」保存命名约束并关闭编辑器。
