# 49.1.1 时间历史动画





在时间历史动画中，Abaqus/CAE 通过为活动步骤和帧在输出数据库中找到的每个指定帧重绘变形形状、云图、符号或材料取向绘图来创建序列中的图像。时间历史动画中的每个图像显示实际分析结果。变形比例因子在时间历史动画的每个图像中保持不变。结果帧可以按顺序绘制，也可以根据时间间隔从活动帧中选择。

如果您正在为云图制作动画，场输出值的显示取决于您选择的选项。默认行为是图例中的值基于第一个和最后一个动画帧的最小值和最大值固定。要了解如何设置云图限制，请参阅 ["设置云图限制，" 第 44.5.7 节"](pt05ch44s02hlb07.md)。

# 49.1.2 比例因子动画





在比例因子动画中，Abaqus/CAE 从输出数据库的单个步骤和帧创建序列中所有变形形状、云图或符号绘图图像。Abaqus/CAE 根据您的规范生成从 0 到 1 或从 1 到 1 的动画比例因子。单个图像是通过将动画比例因子范围应用于绘图中显示的场输出值形成的。这给出了场输出随时间演变的 appearance，尽管所有图像都是从您的结果的单个步骤和帧生成的。（要查看结果随时间的实际演变，您必须生成时间历史动画。）

如果您正在为云图制作动画，场输出值的显示取决于您选择的选项。默认行为是图例中的值基于所选结果步骤和帧的缩放最小值和最大值固定。要了解如何设置云图限制，请参阅 ["设置云图限制，" 第 44.5.7 节"](pt05ch44s02hlb07.md)。要了解如何自定义图例，请参阅 ["自定义图例，" 第 56.1 节"](pt05ch56s01.md)。

# 49.1.3 谐波动画





谐波动画仅对包含复数的场输出有效。在谐波动画中，Abaqus/CAE 从输出数据库的单个步骤和帧创建序列中所有变形形状、云图或符号绘图图像。Abaqus/CAE 根据您的规范生成从 0 到 180 或从 180 到 180 的角度，并显示每个这些角度处复数场输出的值。这给出了场输出随时间演变的 appearance，尽管所有图像都是从您的结果的单个步骤和帧生成的。（要查看结果随时间的实际演变，您必须生成时间历史动画。）谐波动画可用于模拟特征值分析计算的振动模式。

如果您正在为云图制作动画，场输出值的显示取决于您选择的选项。默认行为是图例中的值基于实部的范围固定。要了解如何设置云图限制，请参阅 ["设置云图限制，" 第 44.5.7 节"](pt05ch44s02hlb07.md)。要了解如何自定义图例，请参阅 ["自定义图例，" 第 56.1 节"](pt05ch56s01.md)。要了解更多关于复数结果的信息，请参阅 ["理解复数结果，" 第 42.6.3 节"](pt05ch42s04s03.md)。

# 49.2 生成和自定义基于对象的动画





在时间历史动画中，您可以自定义 Abaqus/CAE 包含在动画中的分析步骤和帧。在比例因子或谐波动画中，您可以自定义模型的表观运动。对于所有三种类型的动画，您可以自定义底层变形形状、云图、符号或材料取向绘图。

在基于对象的动画播放之前或播放期间的任何时候，您都可以自定义动画底层的变形形状、云图、符号或材料取向绘图。例如，您可以切换图例、调整变形比例因子或选择不同的场输出变量。使用 **视图（View）**、**结果（Result）** 和 **选项（Options）** 菜单来自定义动画的底层绘图。

# 49.2.1 生成时间历史动画





通过时间历史进行动画处理会根据实际分析结果生成随时间变化的绘图序列。您可以选择为变形形状、云图、符号或材料取向绘图制作动画。

**生成时间历史动画：**

1. 使用 **文件（File）** 菜单打开包含您的分析结果的 [输出数据库](pt02ch09s06hlb02.md)。
2. 要绘制 Abaqus 默认选择以外的变量，请使用 **结果（Result）** 菜单。- 对于变形或材料取向绘图的动画，请选择 Abaqus 获取随时间变化的值的 [变形场输出变量](pt05ch42s03hlb04.md)。- 对于云图或符号绘图的动画，请选择要显示的 [主场输出变量](pt05ch42s03hlb03.md) 或 [符号场输出变量](pt05ch42s03hlb05.md)。如果您想在变形形状上显示动画云图或符号绘图，请选择 [变形场变量](pt05ch42s03hlb04.md)。有关场输出变量选择的更多信息，请参阅 ["场输出变量选择概述，" 第 42.5.1 节"](pt05ch42s03hlb01.md)。
3. 要自定义 Abaqus/CAE 包含在动画中的步骤和帧，请找到 **活动步骤/帧（Active Steps/Frames）** 对话框。从主菜单栏中，选择 ****结果（Result）**![](../graphics/images/arrow.gif)**活动步骤/帧（Active Steps/Frames）****。有关激活步骤和帧的更多信息，请参阅 ["自定义结果中步骤和帧的显示，" 第 42.4 节"](pt05ch42s02.md)。
4. 使用 **选项（Options）** 和 **视图（View）** 菜单来选择您想要的 [自定义选项](pt05ch40s03s03.md)。这些可能包括变形、云图、符号、取向和视图选项。
5. 要启动动画，请从主菜单栏中选择 ****动画（Animate）**![](../graphics/images/arrow.gif)**时间历史（Time History）****。**提示：**您还可以使用工具箱中的 ![](../graphics/ico_animTimeHistory.png) 工具来生成时间历史动画。动画在所有动画视口中开始，动画控件出现在上下文栏的右侧。Abaqus 现在处于时间历史动画绘图状态。在此状态下，每次您在 **结果（Results）**、**选项（Options）** 或 **视图（View）** 菜单对话框中点击 **应用（Apply）** 或 **确定（OK）** 时，Abaqus 会自动刷新您的绘图。所有动画视口保持时间历史动画状态，直到您在某种其他状态（如未变形或变形状态）中生成绘图。要了解如何控制动画，请参阅 ["控制动画播放，" 第 49.4 节"](pt05ch49s04.md)。
