# 65.3 调整特征再生







本节介绍如何使用 **Feature Options**（特征选项）对话框中的设置来调整特征再生。您可以通过从主菜单栏选择 ****Feature**![](../graphics/images/arrow.gif)**Options****（特征→选项）来打开 **Feature Options**（特征选项）对话框。在大多数情况下，默认设置将产生可接受的再生性能。只有在 Abaqus/CAE 再生部件或装配花费很长时间或内存消耗过大时，您才应该修改特征选项。

**Feature Options**（特征选项）对话框中的设置仅适用于当前模型，并保存在模型数据库中。如果您切换到不同的模型，Abaqus/CAE 会恢复为默认设置或与新模型关联的任何修改后的设置。涵盖以下主题：

- ["什么是自相交检查？" 第 65.3.4 节"](pt06ch65s03s04.md)
