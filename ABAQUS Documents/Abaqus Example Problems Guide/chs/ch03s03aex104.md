# 3.3.2 侧面帘式气囊冲击器测试







**产品：**Abaqus/Explicit

本示例问题说明了使用基于表面的流体腔在多室汽车气囊充气分析中使用Abaus/Explicit。气囊是补充安全装置，可在碰撞期间最大程度地减少乘员撞击车辆内部的机会。在检测到碰撞后的一小部分时间内正确展开气囊对于确保乘员安全至关重要。本示例描述了侧面帘式气囊冲击器测试模拟的建模和验证，与实验结果进行对比。

### 几何和模型

本示例中描述的侧面帘式气囊模型由Autoliv GmbH提供。模型设置如图[图3.3.2-1]所示，显示了冲击器撞击展开的侧面帘式气囊。气囊被分成18个室，以近似气体在整个气囊上的非均匀分布以及气囊复杂展开过程中观察到的压力梯度。每个室在任何时刻的温度和压力被认为是均匀的。每个气囊室建模为基于表面的流体腔。基于表面的流体腔能力提供了 contain气体对室边界施加的压力与充气室变形之间的耦合。各个室由用表面单元建模的虚构隔板分隔开，这些隔板代表不同室之间流体交换的区域。气囊有三个充气器，分别位于第1、第6和第9室，用于给气囊充气。充气器使用与流体充气器定义激活相关的充气时间振幅定义在不同时间激活。充气器注入6种气体物种的混合物，个别物种的分子分数、质量流率和温度定义为时间的函数。气体物种表现为具有绝热温度更新的理想气体。气体通过气囊织物泄漏到环境中的情况通过为每个气囊室的流体交换指定排放系数（有效泄漏面积与总气囊表面积的比率）为0.0003来建模。此外，还考虑了向环境损失的热能，方法是将热能流率提供为气囊织物两侧温差的函数。

气囊的外皮由3节点膜单元（M3D3）表示，并使用织物材料模型的非线性弹性定义。织物材料模型通过指定经纬方向纤维的单轴行为以及织物的剪切行为来校准。纤维的行为被定义为在一定压缩应变极限之前对压缩的阻力可以忽略不计，超过该极限后它们开始对压缩提供有限的阻力，以防止织物单元过度变形。定义双线性剪切行为反映初始剪切模量=9.44 N/mm²和60°以上角度处的模量=47.2 N/mm²，以防止严重剪切变形。此外，应用少量刚度比例阻尼。定义了参考网格（初始度量）以指定气囊的展开无应力配置。

气囊的未应力参考配置使用膜单元的初始参考网格定义。初始折叠配置和参考配置用于考虑气囊折叠过程中产生的褶皱。参考网格和折叠气囊配置分别如图[图3.3.2-2]和[图3.3.2-3]所示。织物材料被定义为沿纱线方向的任何初始压缩应变将导致纱线中的压应力。或者，可以恢复沿纱线方向的初始压缩应变而无需应力。

一般接触算法用于考虑气囊织物层之间的自接触以及气囊充气期间与周围结构的接触。

冲击器被建模为总质量为7.2 kg的刚体，并被给予朝向展开气囊的4500 mm/s的初始速度。冲击器的质量和速度分别反映了车辆侧面碰撞中通常观察到的人体头部质量和速度。

### 结果与讨论

气囊在不同展开阶段的状态如图[图3.3.2-4]（8 ms）和[图3.3.2-5]（16 ms）所示。[图3.3.2-6]、[图3.3.2-7]和[图3.3.2-8]显示了冲击器撞击气囊时的位移、速度和加速度的时间历史，以及与实验结果的比较。这些图表明Abaqus/Explicit模拟结果与物理测试结果之间存在密切相关性。
