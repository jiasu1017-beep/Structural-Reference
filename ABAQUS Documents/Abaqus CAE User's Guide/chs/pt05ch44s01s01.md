# 44.1 理解云图绘制





云图显示输出数据库中存储的结果，或显示当前模型数据库中模型的属性。每个云图显示以下值：对于输出数据库，云图显示分析在指定步骤和帧的特定场输出变量的值；对于当前模型数据库中的模型，云图显示分析中指定步骤的所选属性（如载荷）的值。根据您选择的自定义选项，这些值显示为模型表面的彩色线、彩色带或拼布型彩色面，或者显示为延伸到模型内部的等值面。如果当前视口中没有可见单元，则无法生成云图。线型、带型、拼布型和等值面型云图如图44-1所示（[图44-1](pt05ch44s01.md#usv-contourtype)）。

**图44-1** 从左到右：显示为彩色线、彩色带、彩色面和彩色等值面的云图。

![](../graphics/usv-cont-type.png)

与其他单元一样，线形单元（梁、一维单元、垫片链接单元和三维线垫片单元，以及二维接触表面）的云线默认沿单元绘制。不建议对线形单元使用线型云图。刻度线云图绘制提供了一种可视化梁和其他线形单元云图的替代方法。云图显示为绘制在单元法线两侧的线条之间的曲线，[如图44-2所示](pt05ch44s01.md#usv-tickmarkplot)。云图级别由这些法线上的"刻度线"表示。

**图44-2** 刻度线云图。

![](../graphics/usv-tickmarkplot.png)

解释云图的关键是绘图图例。图例指示云图值与云图颜色之间的对应关系。

默认情况下，Abaqus/CAE将图例中的最小值和最大值之间的范围均匀划分为12个区间。如有需要，您可以更改区间数。每个区间关联一种颜色。对于线型或等值面型云图，每条彩色线或表面对应模型中场输出变量值为图例中显示值的一组位置。对于带型云图，每个彩色云带对应图例指示范围内的值范围。对于拼布型云图，每个彩色单元面对应图例中该颜色指示的界限内的单个值。对于刻度线云图，每个彩色刻度线对应图例指示的单个值。

Abaqus/CAE对接触状态（CSTATUS）云图遵循不同的约定。此变量描述主表面和从表面上的单个节点是否接触，如果接触，这些表面在该点是粘结还是滑动。由于有三个设置可以描述每个节点的接触状态——粘结、滑动或张开——Abaqus/CAE使用三个云图区间显示接触状态数据，并根据节点的接触状态为该节点周围的区域着色。
