# 17.17.2 删除网格





您可以删除整个零件、零件实例或仅在选定区域内Abaqus原生网格。 要删除网格，请从主菜单栏选择 ****Mesh**![](../graphics/images/arrow.gif)**Delete Part Native Mesh****、 ****Mesh**![](../graphics/images/arrow.gif)**Delete Instance Native Mesh**** 或 ****Mesh**![](../graphics/images/arrow.gif)**Delete Region Native Mesh****。

**注意：**删除网格不会导致底层种子被删除，因此您可以修改种子模式并重新生成网格。

**删除网格：**

1. 从主菜单栏，选择 ****Mesh**![](../graphics/images/arrow.gif)**Delete Part Native Mesh****、 ****Mesh**![](../graphics/images/arrow.gif)**Delete Instance Native Mesh**** 或 ****Mesh**![](../graphics/images/arrow.gif)**Delete Region Native Mesh****。 Abaqus/CAE在提示区域显示提示以指导您完成该过程。 **提示：**您也可以使用Mesh模块工具箱中网格工具所在的 ![](../graphics/ico_meshDeleteInstance.png) 或 ![](../graphics/ico_meshDeleteRegion.png) 工具删除网格。 （有关更多信息，请参阅["使用Mesh模块工具箱，" 第17.15节](pt03ch17s15.md)。）
2. 执行以下操作之一： - 如果您要删除零件网格，请从提示区域单击**Yes**确认要删除网格。 - 如果您要从零件或装配中删除区域网格，请选择要从中删除网格的区域。 单击鼠标按钮2表示您已完成选择。 （有关选择对象的更多信息，请参阅[第6章，"在视口中选择对象"](pt01ch06.md)。） - 如果您要从装配中删除实例网格，请选择要从中删除网格的实例。 单击鼠标按钮2表示您已完成选择。 （有关选择对象的更多信息，请参阅[第6章，"在视口中选择对象"](pt01ch06.md)。） Abaqus/CAE会删除网格。
