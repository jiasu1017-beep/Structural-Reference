# 40.3.4 自定义多个视口





当您创建新视口时，它最初继承当前视口的自定义选项。例如，如果您为当前视口中的绘图设置了**填充**渲染样式，然后创建新视口，则新视口中的后续绘图将以填充渲染样式显示。新视口继承当前视口的绘图状态——直到您更改自定义选项、绘图状态或输出数据库，新视口看起来与创建它们时处于活动状态的视口完全相同。

在新视口建立之后，绘图状态和任何*后续*自定义选项默认独立于其他视口。多个视口可以各自处于单独的绘图状态；如果您使用多个视口，必须首先将特定视口指定为当前视口以更改其显示。您应用的自定义选择仅影响当前视口。当您将视口指定为当前视口时，选项对话框会刷新以显示与该视口关联的选项状态。有关使用视口的更多信息，请参见"使用视口"，第4.3节。

您还可以在会话中链接多个视口，以同时操作多个对象，并在不同视口中显示相同的绘图状态和相同的绘图选项。更多信息，请参见"链接视口以进行视图操作"，第4.5节。
