# 19.8.2 Configuring job submission attributes







To locate the job editor, select ****Job**![](../graphics/images/arrow.gif)**Edit**![](../graphics/images/arrow.gif)***jobname***** from the main menu bar. Use the **Submission** tabbed page in the job editor to configure atttributes related to job submission. You can do the following:
- Choose the type of analysis. (For more information, see ["Choosing the job type," Section 19.8.3](pt03ch19s08hlb03.md).)
- Choose the run mode. (For more information, see ["Choosing the job run mode," Section 19.8.4](pt03ch19s08hlb04.md).)
- Choose the submit time. (For more information, see ["Setting the job submit time," Section 19.8.5](pt03ch19s08hlb05.md).)
