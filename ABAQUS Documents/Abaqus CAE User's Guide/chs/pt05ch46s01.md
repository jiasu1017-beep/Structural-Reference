# 46.1 理解材料取向绘制

材料取向图允许您可视化模型中单元的材料方向。Abaqus/CAE在单元积分点处绘制三轴来表示材料取向。例如，材料取向图如图46-1所示（[图46-1](pt05ch46s01.md#usv-matorientplot)）。

**图46-1** 材料取向图。

![](../graphics/usv-matorientplot.png)

材料取向仅对在所选截面点请求场输出的那些单元可用。有效单元包括所有壳单元，以及您已定义局部取向的实体单元。材料取向显示在分析的当前步骤和帧以及壳单元的当前截面点选择处。如果为顶部和底部截面点位置启用了显示，则Abaqus/CAE仅为底部位置显示材料取向。对于小应变壳单元，材料取向不随模型变形而旋转（请参见["壳单元：概述，" Abaqus Analysis User's Guide第29.6.1节](../usb/usb-link.md#usb-elm-eshelloverview)，以获取更多信息）。材料取向图也可以为膜单元、壳单元和表面单元中的增强层创建。
