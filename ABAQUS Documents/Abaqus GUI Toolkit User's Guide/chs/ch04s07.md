# 4.7 小结







- 连续体单元中使用的公式和积分阶数对分析的准确性和成本有重大影响。
- 使用完全积分的一阶（线性）单元容易发生剪切锁定，通常不应使用。
- 一阶减缩积分单元容易发生沙漏；足够的网格细化可以最小化此问题。
- 在使用承受弯曲变形的一阶减缩积分单元进行模拟时，至少使用四个单元穿过厚度。
- 沙漏在 Abaqus/Standard 的二阶减缩积分单元中很少成为问题。当没有接触时，您应该考虑将这些单元用于大多数一般应用。
- Abaqus/Standard 中可用的不兼容模式单元的准确性受单元变形量影响很大。
- 结果的数值准确性取决于所使用网格。理想情况下，应进行网格细化研究以确保网格为问题提供唯一解。但是，请记住，使用收敛网格并不能确保有限元模拟的结果将匹配物理问题的实际行为：这还取决于模型中的其他近似和理想化。
- 通常，主要在您需要准确结果的区域细化网格；需要比计算准确位移精细得多的网格来预测准确的应力。
- Abaqus 中提供了子模型等高级功能，以帮助您为复杂模拟获取有用的结果。
