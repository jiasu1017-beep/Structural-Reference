### 概述

Abaqus/Explicit 接触对的接触控制可用于
- 缩放惩罚接触约束使用的刚度，和
- 调整跟踪两个表面之间运动的搜索算法。

### 缩放默认惩罚刚度

如果您使用惩罚方法在接触对中强制执行接触约束（见 ["在 Abaqus/Explicit 中的接触约束强制方法，" 第 38.2.3 节](pt09ch38s02aus182.md)），Abaqus/Explicit 通过向穿透节点施加"弹簧"刚度来抵抗表面之间的穿透。"弹簧"刚度将接触力与穿透距离关联，由 Abaqus/Explicit 自动选择，使得对时间增量的影响最小 yet 在大多数分析中允许的穿透不显著。如果存在以下任何因素，分析中可能会产生显著穿透：
- 位移控制的加载
- 接触界面上的材料是纯弹性或随变形硬化的
- 变形单元（尤其是膜和表面单元）只有相对较少的自身质量，并通过除边界条件以外的方法（如连接器）约束参与接触
- 刚体只有相对较少的自身质量或旋转惯性，并通过除边界条件以外的方法（如连接器）约束参与接触

请参见 ["Hertz 接触问题，" Abaqus Benchmarks Guide 第 1.1.11 节](../bmk/bmk-link.md#bmk-anl-hertzcontact)，了解这两个因素结合导致默认惩罚刚度的接触穿透显著的示例。

您可以指定一个比例因子来修改指定接触对的惩罚刚度。此缩放可能会影响自动时间增量。使用大比例因子可能会因维持数值稳定性所需的时间增量减少而增加分析所需的计算时间（见 ["在 Abaqus/Explicit 中的接触约束强制方法，" 第 38.2.3 节](pt09ch38s02aus182.md) 中的进一步讨论）。

| **输入文件用法：** | 使用以下两个选项缩放默认惩罚刚度： |
| --- | --- |
|  | ``` [*CONTACT PAIR](../key/key-link.md#usb-kws-hcontactpair), MECHANICAL CONSTRAINT=PENALTY, CPSET=*contact_pair_set_name* *surface_1*, *surface_2* [*CONTACT CONTROLS](../key/key-link.md#usb-kws-hcontactcontrols), CPSET=*contact_pair_set_name*, SCALE PENALTY=*factor* ``` |

| **Abaqus/CAE 用法：** | Interaction 模块： **Create Contact Controls**: **Name:** *contact_controls_name*, **Abaqus/Explicit contact controls**: **Penalty stiffness scaling ** **factor:** *factor* |
| --- | --- |
|  | Interaction 编辑器： **Mechanical constraint formulation: Penalty contact method**, **Contact controls:** *contact_controls_name* |

### 调整有限滑动接触跟踪算法

在有限滑动接触对中，在整个分析过程中持续进行搜索以跟踪两个接触表面之间的相对运动。接触跟踪算法包括昂贵的周期性全局搜索和较便宜的正规律局部搜索；搜索算法在 ["在 Abaqus/Explicit 中接触对的接触 formulations"中的"接触跟踪算法，" 第 38.2.2 节](pt09ch38s02aus181.md#usb-cni-aexpcontactpairform-track) 中详细讨论。您可以使用接触控制来调整这些搜索的频率和成本。

#### 指定更频繁的全局接触搜索

对于两个表面接触对，默认情况下，Abaqus/Explicit 每 100 个增量对每个从节点附近的主表面进行更彻底的搜索，这对大多数分析来说是足够的。然而，在某些有效的接触情况中，全局搜索需要在步骤中更频繁或更不频繁地进行。[图 36.5.5-1](pt09ch36s05aus164.md#acontact-fail-search) 展示了一种可能需要更频繁全局跟踪的情况。主表面是一个有效的表面，但它包含一个孔。所示的从节点在增量期间将阴影单元面片识别为最近的主表面面片。局部接触搜索查看这个主表面面片及其相邻面片。

**图 36.5.5-1** 局部搜索可能失败的示例。

![](../graphics/acontact-fail-search-nls.png)

如果从节点在相对较少的增量内跨孔位移，则不会检测到从节点与跨孔主表面面片之间的潜在接触，因为局部接触搜索仍在检查阴影面片。当从节点快速穿过主表面上的深谷时，也会发生相同的情况。解决方案是更频繁地进行全局接触搜索。如果需要非默认值 100，您可以为给定接触对指定全局搜索之间的增量数 *n*。

| **输入文件用法：** | 使用以下两个选项： |
| --- | --- |
|  | ``` [*CONTACT PAIR](../key/key-link.md#usb-kws-hcontactpair), CPSET=*contact_pair_set_name* [*CONTACT CONTROLS](../key/key-link.md#usb-kws-hcontactcontrols), CPSET=*contact_pair_set_name*, GLOBTRKINC=*n* ``` |

| **Abaqus/CAE 用法：** | Interaction 模块： **Create Contact Controls**: **Name:** *contact_controls_name*, **Abaqus/Explicit contact controls**: **Specify max number of ** **increments:** *n* |
| --- | --- |
|  | Interaction 编辑器： **Contact controls:** *contact_controls_name* |

#### 使用更保守的局部接触搜索

Abaqus/Explicit 使用的默认局部接触搜索使用允许它使用最少计算时间的技术。如果局部接触搜索在强制执行适当的接触条件方面有困难，更保守的局部接触搜索可能会解决问题。指定的接触搜索对使用自接触的接触对没有影响。

| **输入文件用法：** | 使用以下两个选项： |
| --- | --- |
|  | ``` [*CONTACT PAIR](../key/key-link.md#usb-kws-hcontactpair), CPSET=*contact_pair_set_name* [*CONTACT CONTROLS](../key/key-link.md#usb-kws-hcontactcontrols), CPSET=*contact_pair_set_name*, FASTLOCALTRK=NO ``` |

| **Abaqus/CAE 用法：** | Interaction 模块： **Create Contact Controls**: **Name:** *contact_controls_name*, **Abaqus/Explicit contact controls**: 关闭 **Fast local tracking** Interaction 编辑器： **Contact controls:** *contact_controls_name* |
| --- | --- |

### 高度扭曲表面的接触跟踪

计算高度扭曲表面沿的正确接触条件非常困难，特别是当接触表面的相对速度非常大时。默认情况下，Abaqus/Explicit 每 20 个增量监控由单元面形成的每个可变形主表面的方向，以检查表面是否高度扭曲；刚体面片表面仅在步骤开始时检查大的扭曲。如果表面变得高度扭曲，将在状态（`.sta`）文件中发出警告消息（见 ["Abaqus/Explicit 分析中的接触诊断，" 第 39.2.1 节](pt09ch39s02aus185.md)），并使用更准确的算法计算扭曲主表面上每个从节点最近点。替代算法提供更准确的解决方案，但使用稍多的计算时间。

#### 重新定义高度扭曲表面的标准

默认情况下，当面片节点处的表面法线之间的角度变化超过 20 时，Abaqus/Explicit 认为表面高度扭曲。面片上表面法线的最大变化称为面外翘曲角。您可以为模型中的任何接触对按步骤更改面外翘曲角截止值的默认值。

| **输入文件用法：** | ``` [*CONTACT CONTROLS](../key/key-link.md#usb-kws-hcontactcontrols), CPSET=*contact_pair_set_name*, WARP CUT OFF=*angle* ``` |
| --- | --- |

| **Abaqus/CAE 用法：** | Interaction 模块： |
| --- | --- |
|  | **Create Contact Controls**: **Name:** *contact_controls_name*, **Abaqus/Explicit contact controls**: **Angle criteria for highly warped ****facet (degrees):** *angle* Interaction 编辑器： **Contact controls:** *contact_controls_name* |

#### 修改 Abaqus/Explicit 检查扭曲表面的频率

您可以指定 Abaqus/Explicit 检查模型中任何接触对的扭曲表面的频率（以增量计）。频率可以从步骤到步骤更改。更频繁地检查扭曲表面（默认每 20 个增量）将导致分析的计算时间略有增加。

| **输入文件用法：** | ``` [*CONTACT CONTROLS](../key/key-link.md#usb-kws-hcontactcontrols), CPSET=*contact_pair_set_name*, WARP CHECK PERIOD=*n* ``` |
| --- | --- |

| **Abaqus/CAE 用法：** | Interaction 模块： |
| --- | --- |
|  | **Create Contact Controls**: **Name:** *contact_controls_name*, **Abaqus/Explicit contact controls**: **Warp check increment:** *n* Interaction 编辑器： **Contact controls:** *contact_controls_name* |
