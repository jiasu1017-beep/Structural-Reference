# FXTabItem







标签项放置在标签栏或标签簿中。当选中时，标签项向其父项发送消息，并使自己成为活动标签，并略微高于其他标签。在标签簿中，激活标签项也会使相应的面板置于顶部。
![](../graphics/gui-fxtabitem.png)

### FXTabItem(p, text, ic=None, opts=TAB_TOP_NORMAL, x=0, y=0, w=0, h=0, pl=6, pr=6, pt=DEFAULT_PAD, pb=DEFAULT_PAD)

构造标签项。
| **参数** | **类型** | **默认值** | **描述** |
| --- | --- | --- | --- |
| p | FXTabBar |  |  |
| text | String |  |  |
| ic | FXIcon | None |  |
| opts | Int | TAB_TOP_NORMAL |  |
| x | Int | 0 |  |
| y | Int | 0 |  |
| w | Int | 0 |  |
| h | Int | 0 |  |
| pl | Int | 6 |  |
| pr | Int | 6 |  |
| pt | Int | DEFAULT_PAD |  |
| pb | Int | DEFAULT_PAD |  |

### canFocus()

返回 True，因为标签项可以接收焦点。

从 FXWindow 重新实现。

### 全局标志

### **影响边框的标签项方向**

| **TAB_TOP** | 顶部标签。 |
| --- | --- |
| **TAB_LEFT** | 左侧标签。 |
| **TAB_RIGHT** | 右侧标签。 |
| **TAB_BOTTOM** | 底部标签。 |
