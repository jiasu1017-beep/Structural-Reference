# 50.2.1 在可视化模块中查询模型





您可以使用查询工具集来获取当前视口中模型几何的一般信息。

**查询模型：**

1. 找到"查询"对话框。从主菜单栏中，选择****工具**![](../graphics/images/arrow.gif)**查询****，或单击**查询**工具栏中的![](../graphics/ico_info.png)工具。"查询"对话框出现。
2. 要获取特定节点的信息，请执行以下操作：1. 从"查询"对话框的"常规查询"字段中选择"节点"。Abaqus/CAE在提示区域显示提示。2. 从视口中选择一个节点。该节点的未变形和变形X、Y和Z坐标以及其位移出现在消息区域中。消息区域中显示的相同信息也被写入重放文件。**注意：**要调整消息区域的大小，请拖动顶部边缘；要查看已滚出消息区域的信息，请使用右侧的滚动条。
3. 要获取两个节点之间的距离信息，请执行以下操作：1. 从"查询"对话框的"常规查询"字段中选择"距离"。Abaqus/CAE在提示区域显示提示。2. 从视口中选择两个节点。以下信息出现在消息区域：- 每个节点的未变形和变形X、Y和Z坐标，以及其位移。- 节点之间的绝对未变形和变形距离。- 两个节点之间未变形和变形向量的X、Y和Z分量。- 节点之间的绝对相对位移。- 两个节点之间相对位移的X、Y和Z分量。
4. 要获取由三个节点形成的角度信息，请执行以下操作：1. 从"查询"对话框的"常规查询"字段中选择"角度"。Abaqus/CAE在提示区域显示提示。2. 从视口中选择三个节点。您选择的第二个节点是角度的顶点。以下信息出现在消息区域：- 每个节点的未变形和变形X、Y和Z坐标，以及其位移。- 节点之间的绝对未变形和变形角度。
5. 要获取输出数据库中特定单元的信息，请执行以下操作：1. 从"查询"对话框的"常规查询"字段中选择"单元"。Abaqus/CAE在提示区域显示提示。2. 从视口中选择一个单元。以下信息出现在消息区域：- 单元的标签、单元类型、材料和截面。- 连接单元的标签。- 积分点位置的当前场输出变量。
6. 要获取网格的一般信息，请从"查询"对话框的"常规查询"字段中选择"网格"。以下信息出现在消息区域：- 当前输出数据库的名称。- 节点数量。- 单元数量。- 单元类型。
7. 要获取输出数据库中质量属性的一般信息，请执行以下操作：1. 从"查询"对话框的"常规查询"字段中选择"质量属性"。Abaqus/CAE在提示区域显示提示，并在消息区域显示默认选择方法"所有单元"的质量属性。**注意：**如果您在当前会话中已使用过"质量属性"查询并选择了不同的选择方法，Abaqus/CAE在提示区域显示您上次使用的选择方法，但不在消息区域生成新结果。2. 从提示区域中的列表中选择以下选择方法之一：- **所有单元**- **从视口选择单元**- **部件实例**- **单元集**- **截面**- **材料**- **单元类型**- **显示组**根据您选择的 选择方法，Abaqus/CAE在提示区域显示其他选择字段。3. 如下完成您的选择：- 如果您在 previous step chose **从视口选择单元**，请从视口进行选择（有关更多信息，请参阅[第6章，"在视口中选择对象"](pt01ch06.md)）。您可以在提示区域选择一个部件实例并输入单个单元编号以获取该单元的质量信息。- 对于其余选择方法，从提示区域中的列表或从视口中选择部件实例、单元集、截面、材料、单元类型或显示组名称。当您从提示区域选择一个对象或单击"完成"完成视口选择时，Abaqus/CAE在消息区域显示您的选择的质量属性。4. 如果您选择中的任何项目的密度或厚度定义不良，请单击提示区域中的"选项"以显示"质量属性查询选项"对话框，并指定一个值供Abaqus/CAE替代未定义的量。单击对话框中的"确定"保存您的条目并关闭对话框，或单击"重置"返回之前的设置。Abaqus/CAE在消息区域显示您所选项目的质量属性。以下属性可用：- 表面积（仅适用于壳单元）- 面积形心- 体积- 体积形心- 质量- 质心- 关于质心或关于指定点的惯性矩
8. 请求完信息后，关闭"查询"对话框。
