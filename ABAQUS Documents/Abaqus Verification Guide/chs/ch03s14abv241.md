# 3.14.1 在 Abaqus/Explicit 和 Abaqus/Standard 之间传输结果







**产品：**Abaqus/Standard  Abaqus/Explicit  

### I. 将应力/位移结果从Abaqus/Explicit传递到Abaqus/Standard

### 测试的单元

### 问题描述

本节中的验证测试由单单元模型组成，这些模型在Abaqus/Explicit中承受拉力、纯剪切或弯曲加载。Abaqus/Explicit中的分析之后是在Abaqus/Standard中进行的分析，其中从Abaqus/Explicit分析导入结果并移除荷载。几乎所有测试都涉及纯弹性材料。对所有参数组合执行测试，以检查参考配置是否被重置为导入配置，以及在从先前的Abaqus分析导入信息时是否导入材料状态。为了验证Abaqus/Explicit分析的结果正确导入到Abaqus/Standard，Abaqus/Standard分析的结果应显示单元返回到Abaqus/Explicit分析加载之前的原始配置，除非材料状态未被导入，在这种情况下单元保持变形配置。

Abaqus/Explicit中的加载和Abaqus/Standard中的卸载顺序如图3.14.1-1所示。

**图3.14.1-1** 加载和卸载顺序。

![](../graphics/exximport-loadsequence-nls.png)

对于承受拉力加载的S4R单元，当参考配置被更新为导入配置时，以及当前材料状态被导入时。

在Abaqus/Explicit分析中通过规定适当的位移来施加加载。在Abaqus/Standard分析中必须重新定义所有边界条件，在所有情况下仅定义固定边界条件。壳单元和膜单元加载，使最大位移为2。其余单元加载，使最大位移为0.2。

减缩积分单元的分析需要沙漏控制来消除奇异性（沙漏）模式。然而，在Abaqus/Explicit和Abaqus/Standard中计算的沙漏力差异影响导入问题的力平衡。对Abaqus/Explicit和Abaqus/Standard分析都使用增强沙漏控制可最小化导入时沙漏力的差异。包含了对Abaqus/Explicit和Abaqus/Standard分析都使用增强沙漏控制的验证测试，以测试导入问题的性能。

几乎所有测试使用的材料模型是各向同性线弹性。一个测试由用Mises塑性建模的塑性材料组成。使用的材料属性如下：

| 杨氏模量 = 200 109 |
| --- |
| 泊松比 = 0.3 |
| 密度 = 1000. |
| 屈服应力 = 500 106 |
| 硬化 = 500 106 |

对于使用内聚力单元的测试，一些使用带非耦合牵引行为的弹性，一些使用超弹性，一些包含损伤，纯剪切加载的测试还使用额外的横向剪切刚度。

### 结果与讨论

在所有涉及拉力加载、剪切加载和弯曲的弹性测试中，单元返回到其原始配置。在某些情况下，对Abaqus/Standard分析施加更严格的收敛标准控制以获得更准确的结果。

壳单元和膜单元的弹性测试在比较Abaqus/Standard计算的截面厚度与原始厚度时显示出差异。在Abaqus/Explicit中，壳单元和膜单元的厚度变化使用材料泊松比计算，而Abaqus/Standard中默认基于无体积变化假设计算厚度。在大多数实际情况下，卸载或回弹过程中的厚度变化很小：这些测试中观察到的差异是因为假定材料在非常大的变形下保持弹性。

### 测试的单元

B21    B22    B31    B32    C3D4    C3D6    C3D8    C3D8R    CAX3    CAX4R

### 问题描述

本节中的验证测试与第一部分执行的类似。单单元模型在Abaqus/Standard中承受拉力、纯剪切或弯曲加载。然后将这些分析的结果导入到Abaqus/Explicit，并移除荷载。几乎所有测试都涉及纯弹性材料。对导入能力的所有组合执行测试。为了验证Abaqus/Standard分析的结果正确导入到Abaqus/Explicit，Abaqus/Explicit分析的结果应显示模型在导入材料状态时围绕一个平均位置振荡。这个平均位置是Abaqus/Standard分析加载之前的原始配置。

在Abaqus/Standard分析中通过规定适当的位移来施加加载。在Abaqus/Explicit分析中必须重新定义所有边界条件，在所有情况下仅定义固定边界条件。所有单元加载，使最大位移为0.2。

包含了对Abaqus/Explicit和Abaqus/Standard分析都使用增强沙漏控制的验证测试，以测试导入问题的性能。

使用的材料模型与前一部分使用的相同。

### 结果与讨论

在所有情况下，当导入材料状态时，应力在相应的Abaqus/Standard和Abaqus/Explicit分析中连续。当参考配置未被更新时，位移、应变和能量量（如可恢复应变能）被验证在两次分析中连续。在Abaqus/Explicit分析开始时，当参考配置被更新时，位移和应变从零开始；而如果材料状态未被导入，则应力被设置为零。

### IV. 将声学结果从Abaqus/Standard传递到Abaqus/Explicit

### 测试的单元

AC2D3    AC2D4    AC3D4    AC3D6    AC3D8    

ACAX3    ACAX4    ACIN2D2    ACIN3D3    ACIN3D4    ACINAX2    

### 问题描述

兼容的实体单元和声学单元被绑在一起。实体单元在与绑定到声学单元的面相对的面被约束。声学单元承受具有正弦振幅的压力荷载。导入后，分析作为动力分析在Abaqus/Standard中继续。由于声学单元没有材料状态且只有压力自由度，导入材料状态将没有影响，而更新参考配置为导入配置将导入压力值。如果参考配置未被更新，压力值将被设置为零。

### 结果与讨论

通过将导入分析零增量的结果与前一个分析的最后增量进行比较来验证导入分析。进一步通过在导入后继续原始分析一段时间并将这些结果与导入分析进行比较来验证结果。
