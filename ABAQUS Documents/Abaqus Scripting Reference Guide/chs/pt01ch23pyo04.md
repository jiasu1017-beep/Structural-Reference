# 23.4 OdbModelFieldVarList 对象





只读的 OdbModelFieldVarList 对象列出当前 [OdbDisplay](pt01ch35pyo01.md) 对象中模型可用的所有变量。

**访问**

```
import visualization
session.viewports[*name*].layers[*name*].odbDisplay.modelVariableList
session.viewports[*name*].odbDisplay.modelVariableList
```

### 23.4.1 成员

OdbModelFieldVarList 对象没有成员。
