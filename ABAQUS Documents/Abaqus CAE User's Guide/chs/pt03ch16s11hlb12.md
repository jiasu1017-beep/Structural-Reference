# 16.11.12 定义饱和度场





您可以为耦合孔隙流体扩散/应力分析中的区域定义初始饱和度*s*（请参见["耦合孔隙流体扩散和应力分析"章节，《Abaqus Analysis User's Guide》第6.8.1节](../usb/usb-link.md#usb-anl-acoupdiffstress)）。您只能在初始步骤期间创建初始饱和度场。

**创建或编辑饱和度场的步骤：**

1. 使用以下方法之一显示饱和度场编辑器： - 要创建新的饱和度场，请按照["创建预定义场"章节第16.8.3节](pt03ch16s08hlb03.md)中概述的程序进行操作（**类别**：`其他`；**所选步骤的类型**：`饱和度`）。 - 要使用菜单或管理器编辑现有的饱和度场，请参见["编辑依赖于步骤的对象"章节第3.4.13节](pt01ch03s04hlb05.md)。
2. 单击**分布**字段右侧的箭头，然后从出现的列表中选择所需的选项： - 选择**均匀**来定义在该区域上均匀的初始饱和度。 - 选择解析场来定义空间变化的初始饱和度。或者，您可以单击![](../graphics/ico_analyticalField.png)来创建新的解析场。（有关更多信息，请参见[第58章，"解析场工具集"](pt06ch58.md)。）
3. 从**饱和度**字段中，为该区域指定介于`0.0`（无饱和度）和`1.0`（完全饱和）之间的初始饱和度值。
4. 单击**确定**创建饱和度场并关闭对话框。绿色方块出现在模型上，表示您刚创建的初始饱和度场。
