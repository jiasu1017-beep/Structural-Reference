### Results and discussion

Elements using reduced integration may have additional boundary conditions to those specified above. All elements yield exact solutions.

Section output requests to the results (`.fil`) file and to the data (`.dat`) file are used in some of the input files to output accumulated quantities on the face CD. The quantities are reported in a system that is local to the section.
