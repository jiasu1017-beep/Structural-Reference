# 10.2 文件工具集

文件工具集包含一个名为`getPrintForm`的方法，允许你访问显示"打印"对话框的表单。有关如何使用`getPrintForm`方法的示例，请参阅["打印对话框，" 第5.7.2节](pt03ch05s07.md#cus-dlg-dialogs-common-print)。

此外，Abaqus GUI工具包提供了两个虚方法，你可以修改它们来更改应用程序在打开数据库时的行为。通常，在打开输出数据库后，Abaqus/CAE将进入可视化模块。类似地，如果你在可视化模块中并打开模型数据库，Abaqus/CAE将进入上下文栏"模块"列表中列出的第一个模块。要更改此行为，可以覆盖`switchToOdbModule`和`switchToMdbModule`方法。这些方法如果成功则返回True。例如：

```
from abaqusGui import *

class MyFileToolsetGui(FileToolsetGui):

      def switchToMdbModule(self):

         # 始终切换到属性模块
         currentModuleGui = getCurrentModuleGui()
         if currentModuleGui and \
            currentModuleGui.getModuleName() != 'Property':
                 switchModule('Property')
         return True

     def switchToOdbModule(self):

          # 不切换模块
          return True
```
