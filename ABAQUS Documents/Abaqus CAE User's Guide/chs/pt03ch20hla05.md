# 20.20 撤销和重做草绘操作







使用 Sketcher 工具箱中的**撤销**工具![](../graphics/ico_undo.png)或**重做**工具![](../graphics/ico_redo.png)撤销您的上一个操作或重做您最近撤销的操作。有关 Sketcher 工具箱中工具的图表，请参见["Sketcher 工具，"第 20.4.1 节](pt03ch20s04s01.md)。

根据您之前的操作，点击撤销工具可能会有以下效果：
- 移除刚创建的对象。对象包括草图几何（直线、圆弧、圆、椭圆、圆角、样条曲线或点）、构造几何或尺寸。
- 将草图恢复到执行刚才的编辑操作之前的状态。编辑操作包括复制实体、删除实体、移动顶点、修剪边、延伸边、打断边和更改尺寸。
- 移除检索的独立草图，前提是您尚未定位检索的草图。一旦您定位了检索的草图，点击撤销工具会将检索草图的原点移回当前草图的原点。

您可以连续撤销和重做多个操作。例如，如果您绘制了一个圆并标注了其半径，您的第一个撤销操作会移除尺寸，第二个撤销操作会移除圆。您可以通过点击重做工具恢复圆，第二次点击重做工具恢复其尺寸。Abaqus/CAE 存储当前草图的操作历史，您可以指定要存储的最大草绘操作数（有关更多信息，请参见["设置记录的草绘操作的最大数量，"第 20.9.8 节](pt03ch20s09hlb08.md)）。

**撤销**和**重做**工具不执行视图操作。例如，如果您绘制了一个圆，放大视图，然后点击撤销工具，Abaqus/CAE 会删除圆（它不会调整视图大小）。
