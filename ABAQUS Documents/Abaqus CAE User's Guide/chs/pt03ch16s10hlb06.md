# 16.10.6 定义连接器速度边界条件





您可以创建连接器速度边界条件，以为您可能选连接器相对运动分量指定速度。

**创建或编辑连接器速度边界条件：**

1. 使用以下方法之一显示连接器速度边界条件编辑器：- 若要创建新的连接器速度边界条件，请遵循"创建边界条件"，第16.8.2节中概述的过程（**类别**：`Mechanical`；**所选步骤的类型**：`Connector velocity`）。- 若要使用菜单或管理器编辑现有的连接器速度边界条件，请参见"编辑依赖于步骤的对象"，第3.4.13节。若要编辑应用边界条件的区域，请参见"编辑应用规定条件的区域"，第16.8.4节。
2. 如果编辑器中出现**分布**字段，单击该字段右侧的箭头，并选择以下之一：- 选择**均匀**以定义均匀边界条件。- 选择**用户定义**以在用户子程序[`DISP`](../sub/sub-link.md#sub-xsl-disp)中定义边界条件。请参见以下章节获取更多信息：- "指定常规作业设置"，第19.8.6节- "Abaqus User Subroutines Reference Guide"第1.1.4节"DISP"**分布**选项仅在您创建边界条件或在创建边界条件的步骤中编辑边界条件时才会出现在编辑器中。此外，如果您正在执行Abaqus/Explicit分析，则此选项不可用。
3. 如果您直接在编辑器中定义边界条件（而不是在用户子程序[`DISP`](../sub/sub-link.md#sub-xsl-disp)中），请执行以下步骤：1. 使用适当的方法定义边界条件。只有所有选定线共用的分量才在编辑器中可用。**如果每个相对运动分量旁边没有出现文本字段：**您可以使用以下技术来定义边界条件：- 切换打开一个相对运动分量，以将相对运动分量固定为零。- 切换关闭一个相对运动分量，以使该相对运动分量不受约束。**如果每个相对运动分量旁边出现文本字段：**您可以使用以下技术来定义边界条件：- 切换打开一个相对运动分量，以约束该相对运动分量。文本字段变得可用，您可以在其中为该相对运动分量指定值。如果您在当前步骤中创建边界条件，则文本字段中会出现默认值零。如果您在当前步骤中修改边界条件，则文本字段中会出现从前一个步骤传播的值。- 切换关闭一个相对运动分量，以使该相对运动分量不受约束。如果您在文本字段中修改了默认值或传播值后关闭该相对运动分量，则修改后的值将丢失。如果您重新打开该相对运动分量，则默认值或传播值将重新出现在文本字段中。- 单击**幅值**字段右侧的箭头，并从出现的列表中选择您选择的幅值。或者，您可以单击![](../graphics/ico_amplitudeSmall.png)创建新的幅值。（更多信息，请参见第57章"幅值工具集"。）2. 单击**确定**保存您的数据并退出编辑器。
4. 如果您在用户子程序[`DISP`](../sub/sub-link.md#sub-xsl-disp)中定义边界条件，请执行以下步骤：1. 如果需要，使用以下技术定义速度。只有所有选定线共用的分量才在编辑器中可用。- 切换打开一个相对运动分量，以约束该相对运动分量。如果您在初始步骤中，则该相对运动分量被设置为零。如果您在初始步骤以外的任何步骤中，则文本字段变得可用，您可以在其中为该相对运动分量指定值。- 切换关闭一个相对运动分量，以使该相对运动分量不受约束。您在编辑器中为特定自由度输入的数据被传递到用户子程序中。2. 单击**确定**保存您的数据并退出编辑器。3. 进入作业模块并显示感兴趣的分析作业的作业编辑器。（更多信息，请参见"创建、编辑和操作作业"，第19.7节。）4. 在作业编辑器中，单击**常规**选项卡，并指定包含定义边界条件的用户子程序的文件。更多信息，请参见"指定常规作业设置"，第19.8.6节。**注意：**您只能在作业编辑器中指定一个用户子程序文件；如果您的分析涉及多个用户子程序，您必须将用户子程序合并到一个文件中，然后指定该文件。
