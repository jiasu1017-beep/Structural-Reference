### 概述

由于接触表面引起的阻塞而导致的腔体流体外流阻塞：
- 可以针对可能完全或部分引起阻塞的特定表面进行选择性定义；以及
- 仅在使用通用接触算法时才能考虑。

### 用于考虑接触阻塞的表面

要将在["流体交换定义"第11.5.3节"](pt04ch11s05aus72.md#usb-anl-afluidcavityexchange-blockage)中讨论的接触表面的阻塞视为障碍，您必须定义一个表面来表示流体腔边界上的泄漏面积。此外，您必须指定接触表面可能引起阻塞。所有表面（流体腔边界上的表面和接触表面）必须包含在通用接触域中。要考虑接触阻塞，表面上的节点必须是面对面接触。当流体腔边界上的表面上的节点与接触表面接触时，从节点被标记为接触阻塞的活跃节点。边缘对边缘接触也会考虑接触阻塞（参见["Abaqus/Explicit中通用接触的接触公式，" 第38.2.1节"](pt09ch38s02aus180.md)）。

| **输入文件用法：** | 使用以下选项指定两个接触表面可能引起阻塞： |
| --- | --- |
| | ``` [*CONTACT PROPERTY ASSIGNMENT](../key/key-link.md#usb-kws-hcontpropassign) *surface_1, surface_2, property_name* [*SURFACE INTERACTION](../key/key-link.md#usb-kws-hsurfaceinteraction), NAME=*property_name* [*BLOCKAGE](../key/key-link.md#usb-kws-hblockage) ``` |

### 确定阻塞面积

Abaqus/Explicit通过计算未被接触表面阻挡的流体腔边界表面面积分数来确定阻塞面积。对于表示泄漏面积的这个表面的每个单元面，阻塞面积是基于接触阻塞的活跃节点计算的。单元阻塞面积由以下公式确定

其中![](../graphics/usb_eqn06212.gif)是单元阻塞面积，![](../graphics/usb_eqn04609.gif)是单元面积，![](../graphics/usb_eqn01422.gif)是单元节点的总数，![](../graphics/usb_eqn05808.gif)是单元中接触阻塞活跃节点的总数。当所有单元节点都活跃用于接触阻塞时，单元被接触表面完全阻塞。总阻塞面积是所有单元阻塞面积的总和。如果流体交换未指定有效面积，则通过从表面总面积中减去总阻塞面积来获得用于流体交换计算的泄漏面积。如果同时指定了有效面积和表面（参见["流体交换定义，" 第11.5.3节"](pt04ch11s05aus72.md)），则通过使用总阻塞面积与表面总面积的比率乘以有效面积来获得用于流体交换计算的泄漏面积。在这种情况下，可以使用基于节点的表面，并且泄漏面积是通过使用活跃接触阻塞节点总数与表面中定义的节点总数的比率来获得的。
