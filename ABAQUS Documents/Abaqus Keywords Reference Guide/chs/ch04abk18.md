# *DETONATION POINT





### *DETONATION POINT为JWL炸药状态方程定义起爆点。

此选项用于为JWL炸药状态方程定义起爆点。当使用 [*EOS](ch05abk27.md)、TYPE=JWL 选项时需要此选项。[*DETONATION POINT](ch04abk18.md) 选项应紧跟在 [*EOS](ch05abk27.md) 选项之后。

**产品：**Abaqus/Standard  Abaqus/Explicit  Abaqus/CAE  

**类型：**模型数据  

**级别：**模型  

**Abaqus/CAE：**属性模块

##### **参考：**

- [*EOS](ch05abk27.md)
- ["状态方程，" Abaqus分析用户指南第25.2.1节](../usb/usb-link.md#usb-mat-ceos)

**此选项没有关联的参数。**

### **定义起爆点的数据行：**

**第一行：**

根据需要重复此数据行，以定义每个起爆点。
