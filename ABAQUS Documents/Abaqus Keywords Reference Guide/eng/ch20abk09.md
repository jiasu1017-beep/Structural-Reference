# *USER OUTPUT VARIABLES







### *USER OUTPUT VARIABLESSpecify number of user variables.

This option is used to allow Abaqus to allocate space at each material calculation point for user-defined output variables defined in user subroutine [`UVARM`](../sub/sub-link.md#sub-xsl-uvarm). If the [*USER OUTPUT VARIABLES](ch20abk09.md) option is used, it must appear within each of the relevant material or gasket behavior definitions.

**Products: **Abaqus/Standard  Abaqus/CAE  

**Type: **Model data  

**Level: **Model  

**Abaqus/CAE: **Property module

##### **Reference:**

- ["UVARM," Section 1.1.55 of the Abaqus User Subroutines Reference Guide](../sub/sub-link.md#sub-rtn-uuvarm)

**There are no parameters associated with this option.**
