# 15.13.25 定义集中辐射相互作用







您可以通过创建集中环境辐射相互作用来模拟装配件中由于辐射引起的一个或多个点与非反射环境之间的热传递。从主菜单栏中选择****相互作用**![](../graphics/images/arrow.gif)**创建****，然后选择一个或多个节点或顶点或保存的节点或顶点集合。有关辐射相互作用的简要概述，请参见"理解相互作用"，第15.3节。有关更详细的讨论，请参见Abaqus Analysis User's Guide的第34.4.4节"热载荷"。

**定义集中辐射相互作用：**

1. 从主菜单栏中，选择****相互作用**![](../graphics/images/arrow.gif)**创建****。**提示：**您也可以使用相互作用模块工具箱中的![](../graphics/ico_interaction.png)工具创建集中辐射相互作用。
2. 在出现的**创建相互作用**对话框中，执行以下操作：- 命名该相互作用。有关命名对象的信息，请参见第3.2.1节"使用基本对话框组件"。- 选择步骤。您只能在热传递、耦合温度-位移或耦合热-电步骤期间定义节点区域的辐射。- 选择**集中辐射到环境**类型的相互作用。
3. 单击**继续**关闭**创建相互作用**对话框。
4. 使用以下方法之一选择点：- 使用节点或顶点的现有集合定义区域。在提示区域右侧单击**集合**。从出现的**区域选择**对话框中选择现有集合，然后单击**继续**。**注意：**默认选择方法基于您最近使用的选择方法。要恢复为另一种方法，请单击提示区域右侧的**在视口中选择**或**集合**。- 使用鼠标在视口中选择节点或顶点。（更多信息，请参见"在当前视口中选择对象"，第6.2节。）如果模型包含网格和几何的组合，请从提示区域单击以下之一：- 如果要从几何区域选择顶点，请单击**几何**。- 如果要从本机或孤立网格选择节点，请单击**网格**。您可以使用角度方法从网格中选择一组节点。更多信息，请参见"使用角度和特征边缘方法选择多个对象"，第6.2.3节。
5. 在出现的**编辑相互作用**对话框中，执行以下步骤：1. （如果需要）指定集中辐射条件如何应用于自适应网格域的边界。此选项仅对Abaqus/Explicit分析有效。单击**自适应网格边界类型**字段右侧的箭头，然后从出现的列表中选择一个选项。更多信息，请参见Abaqus Analysis User's Guide的第12.2.2节"在Abaqus/Explicit中定义ALE自适应网格域"。- 选择**拉格朗日**以将集中辐射条件应用于跟随材料的节点（非自适应）。- 选择**滑动**以将集中辐射条件应用于可以滑动越过材料的节点。网格约束通常应用于节点以将其固定在空间上。- 选择**欧拉**以将集中辐射条件应用于可以独立于材料移动的节点。此选项仅用于材料可以流入或流出自适应网格域的边界区域。必须对欧拉边界区域应用垂直于区域的网格约束，以允许材料流过该区域。如果未应用网格约束，则欧拉边界区域的行为将与滑动边界区域相同。2. 在**关联节点面积**字段中，输入应用集中辐射条件的节点的关联面积。3. 单击**发射率分布**字段右侧的箭头，然后从出现的列表中选择您想要的选项：- 选择**均匀**以定义在区域上均匀的发射率。- 选择解析场以定义空间变化的发射率。只有对该相互作用类型有效的解析场才显示在选择列表中。或者，您可以单击![](../graphics/ico_analyticalField.png)创建新解析场。（请参见第58章"解析场工具集"获取更多信息。）4. 在**发射率**字段中，输入表面的发射率，![](../graphics/usi_eqn00171.gif]。5. 在**环境温度**字段中，输入环境温度，![](../graphics/usi_eqn00275.gif]。6. 如果要使环境温度随时间变化，请单击**环境温度幅值**字段右侧的箭头，然后从出现的列表中选择幅值。如果需要，请单击![](../graphics/ico_amplitudeSmall.png)创建新幅值；请参见第57.3节"选择要定义的幅值类型"获取更多信息。
6. 在**编辑模型属性**对话框中，指定正在使用的温度标尺上的绝对零温度，![](../graphics/usi_eqn00574.gif]，以及Stefan-Boltzmann常数，![](../graphics/usi_eqn00416.gif]，如第9.8.4节"指定模型属性"中所述。
7. 单击**确定**创建相互作用并关闭编辑器。
