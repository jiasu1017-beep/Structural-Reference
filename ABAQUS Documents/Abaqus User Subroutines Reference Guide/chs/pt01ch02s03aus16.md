### 概述

在Abaqus中，表面：
- 可用于定义接触和相互作用，包括声学-结构相互作用；
- 可用于定义预定义分布表面荷载的区域；
- 可用于绑定不同的网格；
- 可用于定义在Abaqus/Standard中进行腔体辐射分析的腔体；
- 可用于定义在Abaqus/Standard中预定义装配荷载中使用的预拉伸截面；
- 可用于定义在Abaqus/Explicit中跟踪表面平均运动的截面；
- 可用于定义输出量的截面，如通过表面传递的总力；
- 是具有相关面积但体积为零的几何实体；
- 具有由其法向定义的可识别方向；以及
- 通过指定节点或节点集、解析曲线或表面、欧拉材料实例或单元面、边缘或端部来定义；可以是可变形的、刚性的或部分可变形部分刚性的。

本节描述了在Abaqus中创建表面时适用的一般规则。

### 为什么使用表面？

表面可用于模拟两个或多个不同物体在机械、声学、耦合声学-结构、耦合热-机械、耦合热-电-结构、热、耦合热-电或腔体辐射分析中的相互作用。在机械或耦合热-机械分析中，刚性表面可用于表示比模型其余部分坚硬得多的物体，但缺点是不能向刚体传递热量。在声学-结构分析中，表面可用于定义阻抗边界条件，包括用于建模声学辐射的一阶条件。

表面可用于定义预定义分布表面荷载的区域；这可以方便用户对复杂模型输入分布表面荷载。此外，表面可用于定义多点耦合约束。表面还可以定义在Abaqus/Standard中预定义装配荷载中使用的预拉伸截面。

最后，表面可用于定义截面以获取累积量的输出；这提供了"自由体图"输出，允许分析通过静不定结构的"力流"。

可以在Abaqus中定义以下类型的表面：
- 基于单元的表面在单元的面、边缘或端部上定义。单元可以是可变形或刚性的，导致表面可变形或刚性。当形成表面的底层可变形单元是刚体的一部分时，表面将变为部分可变形部分刚性。在Abaqus/Explicit中提供了一个默认的基于单元的表面，其中包含模型中的所有物体，用于通用接触算法。
- 基于节点的表面在节点上定义，因此，本质上是不连续的。可以为表面上每个节点关联用户定义的面积。
- 解析表面直接以几何术语定义，始终是刚性的。
- 欧拉材料表面在欧拉截面中的材料实例上定义。这些表面在Abaqus/Explicit中可用于通用接触算法。

基于单元的表面比基于节点的表面或解析刚性表面包含更多固有信息。当基于单元的表面用于机械接触分析时，Abaqus可以为每个节点关联一个表面积，并可以计算作用于表面的接触应力。相反，当使用基于节点的表面（["基于节点的表面定义，" 第2.3.3节](pt01ch02s03aus18.md)）时，Abaqus可能无法计算准确的接触应力，因为与每个节点关联的实际面积可能不正确。此外，当使用由壳、膜或刚性单元形成的表面时，Abaqus可以在一些引用表面的应用程序中考虑这些单元的厚度以及可能是参考表面的偏移。例如，Abaqus/Explicit中的所有接触算法和Abaqus/Standard中的表面到表面小滑动接触公式都考虑了这些厚度。

不允许两个基于节点的表面之间或基于节点的表面与自身之间的接触；不允许两个解析刚性表面之间的接触；不允许在Abaqus/Standard中使用刚性单元定义的两个刚性表面之间的接触，在Abaqus/Explicit中仅允许惩罚接触。

表面定义不能从一个步骤改变到另一个步骤；但是，可以在重启时定义新表面。

### Abaqus/CAE创建的内部表面

在Abaqus/CAE中，许多建模操作是通过用鼠标选择几何来执行的。例如，可以通过在几何部件实例的面上选择来定义接触对。每个这样的面都必须被转换到输入文件中的表面。这样的表面由Abaqus/CAE分配名称并标记为内部。这些内部表面可以使用Abaqus/CAE可视化模块中的显示组进行查看（请参阅《Abaqus/CAE用户指南》第78章，"使用显示组显示模型的子集"(../usi/usi-link.md#uss-dgp)）。

| **输入文件用法：** | ``` [*SURFACE](../key/key-link.md#usb-kws-msurface), NAME=*surface_name*, INTERNAL ``` |
