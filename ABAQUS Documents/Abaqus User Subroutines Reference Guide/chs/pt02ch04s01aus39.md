### 概述

输出变量可用于：
- 单元积分点、单元截面点、整个单元和单元集；
- 节点；
- 整个模型；
- 基于模式动力学过程中的模态；
- Abaqus/Standard中的表面；和
- Abaqus/Standard中的截面。

所有输出变量在["Abaqus/Standard输出变量标识符，" 第4.2.1节](pt02ch04s02abv01.md)和["Abaqus/Explicit输出变量标识符，" 第4.2.2节](pt02ch04s02xbv01.md)中定义。来自单元、节点和整个模型的输出量可以写入Abaqus/Standard中的数据和结果文件，以及Abaqus/Explicit中的选定结果文件。在Abaqus/Standard中，特征模态、表面和截面的输出量也可以写入数据和结果文件。

对于根据零件实例的装配定义的Abaqus模型（请参阅["定义装配，" 第2.10.1节](pt01ch02s10aus28.md)），数据和结果文件中的输出以Abaqus内部生成的节点、单元、集合和表面标签给出。请参阅["输出，" 第4.1.1节](pt02ch04s01aus38.md)，了解如何将内部生成的编号和名称与您指定的对应起来的详细信息。

##### 输出频率

您可以通过指定步骤中写入文件输出的间隔数来控制特定步骤的所有Abaqus/Explicit结果文件输出的频率。数据总是在每个步骤的开始和结束写入，在该步骤中结果文件输出请求处于活动状态。写入结果的次数称为时间标记。

如果指定的间隔数为10，Abaqus/Explicit将写入11次结果：步骤开始时和步骤中10个相等时间间隔结束时的值。指定的间隔数必须为正整数。

默认情况下，结果将在每个时间标记后立即结束的增量写入。或者，您可以选择调整时间增量大小，以便增量恰好在通过将步骤划分为*n*个相等间隔而计算的每个时间标记处结束。

| **输入文件用法：** | 使用以下选项请求在每个时间间隔后立即结束的增量处写入结果： |
| --- | --- |
|  | ``` [*FILE OUTPUT](../key/key-link.md#usb-kws-hexpfileoutput), NUMBER INTERVAL=*n*, TIME MARKS=NO ``` 使用以下选项请求在精确时间间隔写入结果：``` [*FILE OUTPUT](../key/key-link.md#usb-kws-hexpfileoutput), NUMBER INTERVAL=*n*, TIME MARKS=YES ``` |

### 在多个步骤中请求输出

输出请求适用于定义它们的步骤以及所有后续步骤，直到它们被重新指定。

一个例外发生在步骤类型从一般变为线性扰动时（仅在Abaqus/Standard中可用）。一般步骤中定义的输出请求仅适用于后续一般步骤；在线性扰动步骤中定义的输出请求仅适用于后续连续线性扰动步骤。换句话说，一般步骤中定义的输出独立于线性扰动步骤中定义的输出。线性扰动步骤之间的传播仅发生在连续线性扰动步骤之间。如果在扰动步骤之间发生一般分析步骤，则第一个扰动步骤中定义的输出不会传播到下一个扰动步骤。此外，在Abaqus/Standard中，截面输出请求不会在线性扰动步骤之间传播。

### 单元输出

您可以将单元变量（应力、应变、截面力、单元能量等）输出到Abaqus/Standard数据（`.dat`）文件、Abaqus/Standard结果（`.fil`）文件或Abaqus/Explicit选定结果（`.sel`）文件。输出请求可以在步骤中根据需要重复多次，以定义不同类型单元变量、不同单元集等的输出。同一单元（或单元集）可以出现在多个输出请求中。

通常，单元输出请求适用于后续步骤，除非它们被重新定义；步骤中单个单元输出请求的出现会删除先前步骤中的所有单元输出请求。请参阅["输出，" 第4.1.1节](pt02ch04s01aus38.md)，了解在多个一般分析步骤或线性扰动步骤中请求输出的讨论。

在Abaqus/Explicit中，单元输出被写入选定结果（`.sel`）文件，必须按上述说明将其转换为结果（`.fil`）文件。

| **输入文件用法：** | 使用以下选项将单元变量输出到Abaqus/Standard数据文件： |
| --- | --- |
|  | ``` [*EL PRINT](../key/key-link.md#usb-kws-helprint) ``` 使用以下选项将单元变量输出到Abaqus/Standard结果文件或Abaqus/Explicit选定结果文件：``` [*EL FILE](../key/key-link.md#usb-kws-helfile) ``` |

#### 选择单元输出变量

为定义输出而识别的单元变量类型如下：
- "单元积分点"变量与执行材料计算的积分点相关联（例如，应力和应变分量）。对于在Abaqus/Standard中使用一般梁截面定义的梁和管道，仅当为截面指定了输出截面点时，积分点变量才可用（请参阅["使用一般梁截面定义截面行为，" 第29.3.7节](pt06ch29s03alm12.md)）。对于一阶热传递元素，积分点位于单元的角点，用于热容计算。
- "单元截面点"变量与梁、管道或壳的横截面相关联（例如，弯矩和膜力）。
- "整个单元"变量是整个单元的属性（例如，单元的总能量含量）。
- "整个单元集"变量是整个单元集的属性（例如，质心的当前坐标）；这些变量仅在Abaqus/Standard中可用。

可以写入数据和结果文件的单元变量在["Abaqus/Standard输出变量标识符，" 第4.2.1节](pt02ch04s02abv01.md)和["Abaqus/Explicit输出变量标识符，" 第4.2.2节](pt02ch04s02xbv01.md)中定义。

Abaqus/Standard仅允许完整的基本变量集（例如，所有应力或应变分量）写入结果文件。无法选择单个变量（如特定应力分量），必须通过后处理获得。Abaqus/Standard单元变量可以在积分点、质心、平均在节点或外推到节点处写入数据和结果文件。

在Abaqus/Explicit中，完整的应力或应变张量可以写入选定结果文件，或者可以写入单个标量变量（如等效塑性应变）。Abaqus/Explicit仅在计算它们的积分点处将单元变量写入结果文件。

##### 在每个单元的质心获取单元输出

您可以选择在每个单元的质心输出变量（对于壳单元，是参考表面的质心；对于梁或管道单元，是端节点之间的中点）。如果单元的积分方案不包括质心积分点，则通过积分点值插值获得质心值。

| --- | --- |

##### 获取在节点处平均的单元输出

您可以选择将变量外推到节点，然后对为每个节点贡献的所有单元进行平均。对于派生变量（如主应力），Abaqus/Standard将首先对连接到节点的的所有单元的外推张量分量进行平均，以在每个节点获得唯一的分量，然后基于平均分量计算派生值。

默认情况下，Abaqus/Standard将模型中的单元分区为平均区域。分区基于单元的结构：单元类型、截面点数量、材料类型、单层或复合等。分区不基于元素属性（如厚度）、材料取向或材料常量的值。平均仅发生在为节点贡献且属于同一平均区域的单元上进行。

在某些情况下，您可能希望平均区域考虑元素属性的值。例如，由于变量可能在具有不同材料常量的单元之间不连续，您可能不希望将具有不同属性定义的单元包含在同一平均区域中。在这种情况下，您可以通过将Abaqus环境参数**average_by_section**设置为`ON`来强制Abaqus/Standard考虑元素属性的值。但是，在具有许多截面和/或材料定义的问题中，默认值`OFF`通常比非默认值`ON`提供更好的性能。

| **输入文件用法：** | 使用以下任一选项： |
| --- | --- |
|  | ``` [*EL PRINT](../key/key-link.md#usb-kws-helprint), POSITION=AVERAGED AT NODES [*EL FILE](../key/key-link.md#usb-kws-helfile), POSITION=AVERAGED AT NODES ``` |

| --- | --- |
|  | ``` [*EL PRINT](../key/key-link.md#usb-kws-helprint), POSITION=NODES [*EL FILE](../key/key-link.md#usb-kws-helfile), POSITION=NODES ``` |

##### 单元输出变量的外推和插值

单元的形状函数用于输出变量的外推和插值目的。外推值在高应力梯度区域通常不如在积分点计算的值准确，特别是在改进的三角形和四面体的情况下。因此，需要在需要此类单元结果的准确节点值的区域周围进行足够的网格划分。如果为单元定义了圆柱或球坐标系（请参阅["取向，" 第2.2.5节](pt01ch02s02aus15.md)），则每个积分点处的取向可能不同。当积分点处的值被外推到节点时，不考虑取向的差异；因此，如果连接到节点的单元的取向差异很大，则外推值不会非常准确。如果材料取向在模型区域中真正是各向异性的，则需要更细的网格以即使在积分点也能获得准确的结果。在这种情况下，一旦整体解相对于网格密度的收敛，积分点处的插值或外推也可以被认为是相当准确的。不应将具有在二维中一个塌陷边或三维中一个塌陷面的二阶单元的单元输出外推到节点。

在耦合温度-位移和耦合热电-结构分析中，节点温度（变量NT11）比外推到节点的积分点温度（变量TEMP）更准确。

对于派生变量（如Mises等效应力），首先外推或插值分量，然后从外推或插值分量计算派生值。但是，在基于模式的线性动态分析过程（其中值作为模态响应幅值的非线性组合获得）（["随机响应分析，" 第6.3.11节](pt03ch06s03at16.md)和["响应谱分析，" 第6.3.10节](pt03ch06s03at15.md)），首先在积分点计算派生值。这些派生值被外推到节点或插值到质心。

#### 指定单元输出的方向

对于应力、应变和类似材料变量的分量，1、2和3指的是正交坐标系中的方向。如果未为单元定义局部取向，则应力/应变分量在["约定，" 第1.2.2节](pt01ch01s02aus02.md)中约定的默认方向中：实体单元为全局方向；壳、膜和垫片单元为表面方向；梁和管道单元为轴向和横向方向。

如果单元关联了局部取向，则单元输出变量分量在取向定义的局部方向中（请参阅["取向，" 第2.2.5节](pt01ch02s02aus15.md)）。在Abaqus/Standard中，如果为任何变量请求了分量输出，您可以请求将局部方向写入结果文件（请参阅["输出局部方向到结果文件](pt02ch04s01aus39.md#usb-out-oprintfile-results-directions)"下文）。在Abaqus/Explicit中，当为任何单元变量请求张量输出时，局部方向始终写入结果文件。局部方向自动从Abaqus/Standard和Abaqus/Explicit写入输出数据库文件。

在大位移问题中，参考配置中定义的局部方向通过平均材料旋转旋转到当前配置。请参阅["状态存储，" Abaqus理论指南第1.5.4节](../stm/stm-link.md#stm-int-statestorage)，了解更多详情。

#### Abaqus/Standard数据文件格式

在Abaqus/Standard中，变量打印输出在数据文件的表中排列。对于单元变量，表的每一行对应一个特定位置：单元、节点、单元内的截面点或积分点。特定表中出现的行由选择单元集以及可能集内每个单元中的位置来定义。

每个表由单元输出请求的一行数据定义，该行指定将出现在该表中的变量。可以定义的表数量没有限制。表的前几列定义位置——单元或节点编号、积分点编号等。您选择哪些数据出现在剩余列中；每个表最多可以出现9个变量（列）。例如，在三维分析中，输出变量S和E不能在同一数据行上请求，因为这将产生12列输出。如果一行中的所有条目都为零，则不打印该行。

每个表只能包含一种类型的输出变量（整个单元、截面或积分点）；一种类型的单元；和一种类型的截面定义。如果单元输出请求包含多种类型的输出变量、单元或截面定义，Abaqus/Standard将自动将输出拆分所需的表。所有由输出请求第一行定义的表将被打印，然后由第二行的所有表打印，等等。

#### 结果文件格式

为每个单元的每个积分点和截面点为每个请求行创建单元头记录（["结果文件输出格式，" 第5.1.2节](pt02ch05s01afi01.md)中所述的1类记录）。除了单元头记录之外，在Abaqus/Standard中，当请求完整应力或应变张量输出时，可以写入方向记录（类型85记录）（见下文）。在Abaqus/Explicit中，当请求完整应力或应变张量输出时，始终写入方向记录。

对于具有多个变量的Abaqus/Standard文件输出请求，优势是在每个数据行上指定尽可能多的变量（最多16个）。通过最小化请求行数，可以避免额外的类型1和类型85记录，并且结果文件的大小可以大大减少。这在Abaqus/Explicit中不是问题。单元变量必须属于相同的"类型"（单元积分点变量；单元截面变量；整个单元变量等）才能在同一行上输入——请参阅["输出，" 第4.1.1节](pt02ch04s01aus38.md)。在Abaqus/Standard中，如果文件输出记录中的所有结果都为零，则不将记录写入结果文件。

##### 输出局部方向到结果文件

默认情况下，在Abaqus/Standard中，局部坐标方向不会写入结果文件。如果请求了分量输出，您可以将局部坐标方向写入结果文件。方向记录（类型85）将在类型1记录之后写入。

在Abaqus/Explicit中，当请求完整应力或应变张量输出时，局部坐标方向始终作为类型85的方向记录写入选定结果文件。

张量分量输出在局部坐标系中给出，这可能对于单元是固有的（如壳和膜），也可能是用户定义的（["取向，" 第2.2.5节](pt01ch02s02aus15.md)）。

对于壳单元，为截面中请求分量输出的每个材料点写入方向记录，并为截面力和截面应变写入单独的方向记录。对于几何非线性分析，在Abaqus/Standard中，记录包含当前的更新方向，但对于小应变壳和垫片单元，给出原始方向。对于三维梁，仅当请求截面输出时才写入方向输出。

对于桁架、二维梁、二维垫片单元、轴对称壳、轴对称膜、轴对称垫片单元，或对于平均在节点的值，不提供方向输出。此外，对于GK*xx*N型垫片单元，不提供方向输出，它们没有膜或横向剪切变形。

| **输入文件用法：** | 在Abaqus/Standard中使用以下选项： |
| --- | --- |
|  | ``` [*EL FILE](../key/key-link.md#usb-kws-helfile), DIRECTIONS=YES ``` |

#### 默认表面输出

必须包含表面输出请求才能将与接触对相关联的表面变量写入数据和结果文件；不提供默认输出。

如果表面输出请求定义时未指定任何输出变量，则默认情况下将以下变量写入数据和结果文件：
- 对于接触分析，接触压力（CPRESS）、摩擦剪切应力（CSHEAR）、接触开口（COPEN）和相对切向运动（CSLIP）；请参阅["在Abaqus/Standard中定义接触对，" 第36.3.1节](pt09ch36s03aus145.md)。
- 对于热传递分析，热通量密度（HFL）、热通量（HFLA）、HFL的时间积分（HTL）和HFLA的时间积分（HTLA）；请参阅["热接触特性，" 第37.2.1节](pt09ch37s02aus174.md)。
- 对于耦合热电分析，HFL、HFLA、HTL、HTLA、每单位面积电流（ECD）、电流（ECDA）、ECD的时间积分（ECDT）和ECDTA的时间积分（ECDTA）；请参阅["电接触特性，" 第37.3.1节](pt09ch37s03aus175.md)。
- 对于耦合孔隙流体-力学分析，CPRESS、CSHEAR、COPEN、CSLIP、每单位面积孔隙流体体积通量（PFL）、孔隙流体体积通量（PFLA）、PFL的时间积分（PTL）和PFLA的时间积分（PTLA）；请参阅["孔隙流体接触特性，" 第37.4.1节](pt09ch37s04aus176.md)。
- 对于裂纹扩展分析，没有默认输出量；必须明确请求粘结失效量；请参阅["裂纹扩展分析，" 第11.4.3节](pt04ch11s04aus69.md)。

#### 数据文件格式

变量打印输出在表中排列。每个表由表面输出请求的一行数据定义，该行指定将出现在该表中的变量。每个表只能包含一种类型的输出变量（从节点或整个表面）。例如，输出变量CSTRESS和CFN不能在同一数据行上请求。对于从节点类型的输出，表中的每一行对应从表面上的一个节点。将出现在特定表中的行将限于输出请求中指定的节点集。表的第一列定义位置（节点编号）。其余列包含变量，如接触压力、摩擦剪切应力、接触开口和相对切向（滑移）运动。对于整个表面类型的输出，表中的每一行对应整个从表面。如果表一行的所有变量都为零，则不打印该行。

如果接触输出请求涉及多个接触对，则将为每个接触对生成一个单独的表。所有由输出请求第一行定义的表将被打印，然后由第二行的所有表打印，等等。

#### 结果文件格式

为每个输出请求创建接触输出请求记录（["结果文件输出格式，" 第5.1.2节](pt02ch05s01afi01.md)中所述的类型1503记录）。对于从节点类型的输出，此记录后跟几个节点头记录，每个记录包含从表面上的一个节点。每个节点头记录后跟包含输出变量的记录。输出将限于输出请求中指定的节点集。对于整个表面类型的输出，类型1503记录后只跟一个类型1504节点头记录，节点编号为零。节点头记录后跟包含请求输出变量的记录。

如果接触输出请求涉及多个接触对，则将为每个接触对生成一个单独的接触输出请求记录。

### 来自Abaqus/Standard的截面输出

在Abaqus/Standard中，您可以输出与用户定义截面相关联的累积量（请参阅["Abaqus/Standard输出变量标识符，" 第4.2.1节](pt02ch04s02abv01.md)）到数据或结果文件。此功能提供"自由体图"输出，允许分析冗余结构中的力流。输出请求可以在步骤中根据需要重复多次，以定义不同截面和不同截面输出变量。您可以为每个输出请求分配一个标签，用于标识截面的输出。截面输出不适用于特征频率提取、特征值屈曲预测、复特征频率提取或线性动力学过程，也不适用于使用多荷载情况的过程。

#### 定义表面截面

截面输出请求仅适用于使用基于单元的表面定义的截面（请参阅["基于单元的表面定义，" 第2.3.2节](pt01ch02s03aus17.md)。因此，截面必须使用连续单元的面定义，尽管其他类型的单元（梁、膜、壳、弹簧、阻尼器等）可以连接到截面。

截面上的累积量（如总力）的计算涉及仅来自截面一侧的单元的节点量。因此，表面定义应仅使用一侧的单元（"基础单元"，如[" prescribed assembly loads，" 第34.5.1节](pt07ch34s05aus127.md)中所定义），从而精确识别计算累积量的侧面。

由于在典型截面输出请求中截面通常穿过网格，因此无法使用表面的自动生成。通过指定单元面来精确控制形成表面的单元面，这是定义穿过实体的截面时的必要操作。

您必须指定正在请求输出的表面的名称。

在重启分析中定义的表面只能用于截面输出请求。新定义的表面不能用于任何其他目的（如接触对或预紧截面定义）。

#### 定义表面截面的局部坐标系

您可以允许Abaqus/Standard定义局部系统，也可以直接指定。

##### 默认局部系统

默认局部系统特别适用于截面是平面或几乎平面的情况。虽然它也可以用于定义的表面是弯曲的情况，但默认局部系统对于此类问题可能无关。

默认系统由在二维和轴对称情况下的直线或三维情况下的平面定义，通过最小二乘法拟合穿过属于截面的节点。局部系统的锚点（原点）是表面在拟合直线或平面上的投影的质心。局部方向由法线（1方向）和切线方向（二维和轴对称情况下的2方向）或切线方向（三维情况下的2和3方向）给出，指向拟合直线或平面。当可以同样好地拟合穿过定义截面的节点的多条直线或平面时（例如，封闭的圆形或球形表面），原始局部方向将与全局轴平行。

选择正的局部1方向，使其与截面的平均法线方向成锐角，通过对定义截面的单元面的正法线求平均计算。如果平均法线方向为零（封闭表面），则1方向将与全局*x*轴成锐角。如果在二维或轴对称情况中，1方向在全局*x*轴的法线0.1以内，则它将与全局*y*轴成锐角。在三维情况下，如果1方向在全局*X*–*Y*平面的法线0.1以内，则它将与全局*z*轴成锐角。

在二维和轴对称情况下，局部2方向通过围绕锚点逆时针旋转局部1方向90度获得。对于三维情况，表面上的局部方向使用Abaqus约定的局部方向定义（请参阅["约定，" 第1.2.2节](pt01ch01s02aus02.md)）。

| **输入文件用法：** | 使用以下任一选项使用默认局部坐标系： |
| --- | --- |
|  | ``` [*SECTION PRINT](../key/key-link.md#usb-kws-hsectionprint), NAME=*section_name*, SURFACE=*surface_name*, AXES=LOCAL [*SECTION FILE](../key/key-link.md#usb-kws-hsectionfile), NAME=*section_name*, SURFACE=*surface_name*, AXES=LOCAL ``` |

##### 用户指定的局部系统

用户指定的局部系统通过指定原点和轴的方向来定义。您可以通过给出节点编号或指定锚点坐标来指定原点（锚点）。

在二维和轴对称情况下，通过指定预定义节点编号或局部2方向上一点（点*a*）的坐标来定义局部2方向。然后局部1方向通过围绕锚点顺时针旋转局部2轴90度获得（请参阅[图4.1.2-1](pt02ch04s01aus39.md#oprintfile-localsys-usb-out-oprintfile)）。如果使用节点编号定义锚点或局部方向，它们必须连接到网格。

**图4.1.2-1** 用户定义的局部坐标系。

![](../graphics/oprintfile-localsys-nls.png)

在三维情况下，可以使用两个预定义节点或两个点的坐标来指定局部方向。然后通过其原点（锚点）和这两个点定义矩形笛卡尔坐标系。第一个点（点*a*）必须位于局部2方向上，第二个点（点*b*）必须位于局部3方向的侧面的局部2–3平面中。虽然不是必需的，但直观地选择第二个点使其在或接近局部3方向（请参阅[图4.1.2-1](pt02ch04s01aus39.md#oprintfile-localsys-usb-out-oprintfile)）。

#### 数据文件格式

打印输出在表中排列。表的第一行包含请求的输出变量的名称（请参阅["Abaqus/Standard输出变量标识符，" 第4.2.1节](pt02ch04s02abv01.md)），第二行包含相应的值。如果截面输出请求定义时未指定任何输出变量，则输出与当前分析类型相关的所有适当变量。

如果在特定步骤中遇到多个数据文件的截面输出请求，则将为每个请求创建单独的表。每个表有一个标题，表示截面名称和使用的表面名称。此外，如果在局部坐标系中请求输出，则输出锚点的全局坐标和局部轴的余弦方向。

#### 结果文件格式

为每个截面输出请求到结果文件输出几个截面输出记录（["结果文件输出格式，" 第5.1.2节](pt02ch05s01afi01.md)中的记录号1580–1591）。写入结果文件的实际记录集合取决于有效输出请求的数量。如果截面输出请求定义时未指定任何输出变量，则与当前分析类型相关的所有记录都存储在结果文件中。

#### 截面中的向量输出

与截面输出请求相关联的向量输出包括总力（SOF）、总矩（SOM）和力中心（SOCF）。输出变量SOF计算为表面中节点基于应力（内部） nodal forces的向量和。

输出变量SOM是相对于所考虑坐标系的原点计算的。因此，如果输出在全局坐标系中请求，则总矩相对于全局原点计算；如果在局部坐标系中请求，则相对于局部系统当前的锚点计算。局部系统是当前锚点坐标可以在分析期间改变（如果局部坐标系更新）。输出变量SOF和SOM都在所考虑的坐标系中报告。

力中心SOCF计算为总力SOF所通过的截面中离质心最近的点。SOCF始终在全局坐标系中报告。如果总力向量等于零，则报告截面的质心作为力中心SOCF。

总矩向量SOM不一定等于力中心向量SOCF和总力向量SOF的叉乘。作用在截面两个不同点上的力可能有沿相反方向作用的分量，这些力分量产生净矩而不是净力；因此，总矩可能不完全来自合力的结果。

#### 截面中的标量输出

与截面输出请求相关联的标量输出包括定义截面的面积（SOAREA）、热传递分析中的总热通量（SOH）、电分析中的总电流（SOE）、质量扩散分析中的总质量流（SOD）以及耦合孔隙流体扩散-应力分析中的总孔隙流体体积通量（SOP）。这些输出变量计算为表面中节点标量内部通量（与相关联的主求解变量的功共轭）的代数和。例如，在热传递分析中，总热通量（SOH）是表面上节点处NFLUX值的总和。

#### 使用截面输出请求的限制

截面输出请求受以下限制：
- 截面输出请求仅适用于由基于单元的表面定义的截面。因此，它们只能用于连续单元面的截面。
- 定义截面时，只能使用截面一侧的单元。Abaqus/Standard识别附加到此侧表面上的所有单元，并将截面输出变量计算为自由体图。
- 定义的截面必须完全穿过网格，形成封闭表面，或位于体外。[图4.1.2-2](pt02ch04s01aus39.md#oprintfile-valid)显示了一些有效表面的典型情况。如果截面仅部分穿过网格，则无法隔离有效的自由体图（请参阅[图4.1.2-3](pt02ch04s01aus39.md#oprintfile-invalid)），可能会计算错误答案。Abaqus/Standard将尝试识别无效情况并发出错误或警告消息。**图4.1.2-2** 有效的截面定义。![](../graphics/oprintfile-valid-nls.png)**图4.1.2-3** 无效的截面定义。![](../graphics/oprintfile-invalid-nls.png)
- 附加到截面的单元可以位于表面的任一侧，但不得穿过定义的截面。[图4.1.2-3](pt02ch04s01aus39.md#oprintfile-invalid)显示了一些无效情况。在大多数情况下，Abaqus/Standard将成功识别穿过表面的单元，并将发出警告消息。然后，这些单元将不考虑用于截面变量的计算。
- 对于附加到截面的单元，Abaqus/Standard将忽略它无法建立是属于截面一侧还是另一侧的单元（例如，SPRING1单元）。
- 截面输出请求不能在子结构中指定。
- 截面输出请求不能在随机响应分析中指定。
- 截面中的总力和总矩仅基于识别单元中的应力（内部力）计算。因此，如果这些单元中存在分布式体力载荷，则可能获得不准确的结果，因为总力在截面上的影响不包括在内。常见例子是动态分析中的惯性载荷、重力载荷、分布式体力力和离心载荷。在这些情况下，截面中的总力可能取决于用于定义截面的单元选择，如[图4.1.2-4](pt02ch04s01aus39.md#oprintfile-distload)(a)所示。假设重力载荷是唯一的活动载荷，则两个单元中的单元应力将不同。因此，如果首先使用单元1定义截面，然后使用单元2定义截面，则将获得不同的总力答案。类似地，识别单元中规定的任何分布体力通量（热、电等）的影响也不包括在内。**图4.1.2-4** 截面中的总力。![](../graphics/oprintfile-distload-nls.png)
- 根据用于定义截面的表面侧，在类似于[图4.1.2-4](pt02ch04s01aus39.md#oprintfile-distload)(b)所示的情况的分析中，将获得不同的答案。假设是静态分析，图中所示的集中载荷是唯一的活动载荷，如果使用单元1定义截面，则报告零总力；如果使用单元2定义截面，则获得等于集中载荷之和的非零力。
