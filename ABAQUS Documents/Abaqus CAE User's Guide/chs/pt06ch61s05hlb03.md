# 61.5.3 显示现有键盘快捷键





**Shortcut Listing** 对话框显示当前在 Abaqus/CAE 中分配的所有键盘快捷键及其映射到的功能的列表。

**显示键盘快捷键列表的步骤：**

1. 在任何模块中，从主菜单栏选择 ****Tools**![](../graphics/images/arrow.gif)**Customize****。

2. 从打开的 **Customize** 对话框中，单击 **Functions** 选项卡。

3. 要显示现有键盘快捷键的列表，请单击 **Show all assignments**。从出现的 **Shortcut Listing** 对话框中，您可以查看所有当前定义的键盘快捷键。
