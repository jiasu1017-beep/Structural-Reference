# 29.103 UserMaterial 对象

UserMaterial 对象定义用于子程序 [`UMAT`](../sub/sub-link.md#sub-xsl-umat)、[`UMATHT`](../sub/sub-link.md#sub-xsl-umatht) 或 [`VUMAT`](../sub/sub-link.md#sub-xsl-vumat) 的材料常数。

**访问**

```
import material
mdb.models[*name*].materials[*name*].userMaterial
import odbMaterial
session.odbs[*name*].materials[*name*].userMaterial
```

### 29.103.1 UserMaterial(...)

此方法创建 UserMaterial 对象。

**路径**

```
mdb.models[*name*].materials[*name*].UserMaterial
session.odbs[*name*].materials[*name*].UserMaterial
```

**必需参数**

无。

**可选参数**

*type*

SymbolicConstant，指定由此命令定义的材料行为类型。可能的值为 MECHANICAL、THERMAL 和 THERMOMECHANICAL。默认值为 MECHANICAL。

*unsymm*

Boolean，指定材料刚度矩阵 ![](../graphics/ker_eqn00381.gif) 是否不对称，或者当使用热本构模型时，![](../graphics/ker_eqn00382.gif) 是否不对称。默认值为 OFF。

此参数仅对 Abaqus/Standard 分析有效。

*mechanicalConstants*

Float 序列，指定材料的机械常数。此参数仅在 *type*=MECHANICAL 或 THERMOMECHANICAL 时有效。默认值为空序列。

*thermalConstants*

Float 序列，指定材料的热常数。此参数仅在 *type*=THERMAL 或 THERMOMECHANICAL 时有效。默认值为空序列。

**返回值**

UserMaterial 对象。

**异常**

RangeError。

### 29.103.2 setValues(...)

此方法修改 UserMaterial 对象。

**必需参数**

无。

**可选参数**

`setValues` 的可选参数与 [UserMaterial](pt01ch29pyo103.md#ker-usermaterial-usermaterial-pyc) 方法的参数相同。

**返回值**

无

**异常**

RangeError。
