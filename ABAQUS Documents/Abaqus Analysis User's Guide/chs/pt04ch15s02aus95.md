### 概述

光滑粒子流体力学（SPH）是一种数值方法，属于无网格（或网格自由）方法家族。对于这些方法，您不像在有限元分析中那样定义节点和单元；相反，只需要一组点来表示给定的物体。在光滑粒子流体力学中，这些节点通常称为粒子或伪粒子。

[图 15.2.1--1](pt04ch15s02aus95.md#sph-meshes) 中显示的示例对比了两种方法。两种离散表示都建模了瓶内流体体的初始配置，如 ["盛水瓶的冲击," Abaqus 例题指南第 2.3.2 节](../exa/exa-link.md#exa-dyn-bottledrop) 中详细描述。左侧的模型是流体占据体积的传统四面体网格。右侧，相同体积由一组离散点表示。请注意，在后一种情况下，没有连接点的边，因为这些点（伪粒子）不需要定义多节点单元连接性，如左侧传统有限元表示中所述。直接定义粒子单元的替代方法是定义常规连续体有限元，并在分析开始时或分析过程中自动将它们转换为粒子单元，如 ["有限元转换为 SPH 粒子," 第 15.2.2 节](pt04ch15s02aus96.md) 中所述。

**图 15.2.1–1** 有限元网格和 SPH 粒子分布。

![](../graphics/sph-meshes.png)

光滑粒子流体力学是一种完全拉格朗日建模方案，允许通过直接在分布在解域上的离散点集内插属性来离散一组指定的连续体方程，而无需定义空间网格。该方法的拉格朗日性质以及没有固定网格是其主要优势。与流体流动和涉及大变形和自由表面的结构问题相关的困难以相对自然的方式解决。

从根本上讲，该方法不是基于离散粒子（球体）相互压缩或如"particle"一词所暗示的那样在拉伸中表现出类似内聚行为。相反，它只是连续体偏微分方程的一种聪明离散化方法。在这方面，光滑粒子流体力学与有限元方法非常相似。SPH 使用进化插值方案来近似域中任意点的场变量。感兴趣粒子处的变量值可以通过从一组相邻粒子（用下标 *j* 表示）的贡献求和来近似，其中"核"函数 *W* 不为零

![](../graphics/usb_eqn02116.gif)

[图 15.2.1--2](pt04ch15s02aus95.md#sph-int-method-nls) 显示了一个示例核函数。平滑长度 *h* 决定有多少粒子影响特定点的插值。

自成立以来，SPH 方法已获得大量理论支持（[Gingold and Monaghan, 1977](pt04ch15s02aus95.md#asphanalysis-gingold1977)），与该方法相关的出版物现在非常多。下面列出了许多参考文献。

该方法可以使用 Abaqus/Explicit 中的任何材料（包括用户材料）。您可以像任何其他拉格朗日模型一样指定初始条件和边界条件。也允许与其他拉格朗日体的接触相互作用，从而扩大了该方法可用的应用范围。

当变形不太严重时，该方法通常不如拉格朗日有限元分析准确；在较高变形范围内，不如耦合欧拉-拉格朗日分析准确。如果模型中很大比例的节点与光滑粒子流体力学相关联，则在使用多个 CPU 时，分析可能无法很好地扩展。

### 应用

光滑粒子流体力学分析对于涉及极端变形的应用非常有效。流体晃动、波浪工程、弹道学、喷涂（如喷漆）、气体流动以及随后继发冲击的消灭和碎裂只是几个例子。有许多应用可以同时使用耦合欧拉-拉格朗日和光滑粒子流体力学方法。在许多耦合欧拉-拉格朗日分析中，材料与空隙的比率很小，因此计算工作量可能非常高。在这种情况下，优选光滑粒子流体力学方法。例如，在耦合欧拉-拉格朗日分析中，跟踪初级冲击碎片通过大体积直到二次冲击可能非常昂贵，但在光滑粒子流体力学分析中不会增加任何成本。

["盛水瓶的冲击," Abaqus 例题指南第 2.3.2 节](../exa/exa-link.md#exa-dyn-bottledrop) 包括使用光滑粒子流体力学方法对与冲击相关的剧烈晃动进行建模的示例。

### 单元

光滑粒子流体力学方法通过与 PC3D 单元相关的公式实现。这些 1 节点单元只是在空间中定义粒子的手段，建模特定的一个或多个物体。这些粒子单元利用 Abaqus 中的现有功能来引用单元相关特征，如材料、初始条件、分布载荷和可视化。

您可以以与定义点质量类似的方式定义这些单元。这些点的坐标位于被建模物体表面的内部或内部，类似于用砖单元网格化的体的节点。为了获得更准确的结果，您应该努力使这些粒子的节点坐标在所有方向上尽可能均匀地分布。

直接定义 PC3D 单元的替代方法是定义常规连续体有限元类型 C3D8R、C3D6 或 C3D4，并在分析开始时或分析过程中自动将它们转换为粒子单元，如 ["有限元转换为 SPH 粒子," 第 15.2.2 节](pt04ch15s02aus96.md) 中所述。

Abaqus/Explicit 中实现的光滑粒子流体力学方法使用三次样条作为插值多项式，并基于下面参考文献中概述的经典光滑粒子流体力学理论。

光滑粒子流体力学方法未针对二维单元实现。可以使用楔形扇区和平移边界条件来模拟轴对称。PC3D 单元没有相关的沙漏或扭曲控制力。这些单元没有与其关联的面或边。

#### SPH 核插值器

默认情况下，Abaqus/Explicit 中实现的光滑粒子流体力学方法使用三次样条作为插值多项式。或者，您可以选择二次（[Johnson et al, 1996](pt04ch15s02aus95.md#asphanalysis-johnson)）或五次（[Wendland, 1995](pt04ch15s02aus95.md#asphanalysis-wendland)）插值器。

该实现基于下面参考文献中概述的经典光滑粒子流体力学理论。您还可以选择使用平均流动校正配置更新，这在文献中通常称为 XSPH 方法（参见 [Monaghan, 1992](pt04ch15s02aus95.md#asphanalysis-monaghan)），以及 [Randles and Libersky, 1997](pt04ch15s02aus95.md#asphanalysis-randles) 的校正核，也称为归一化 SPH（NSPH）方法。

您可以控制这些设置，如 ["在截面控制中使用光滑粒子流体力学（SPH）的截面控制," 第 27.1.4 节"](pt06ch27s01aus113.md#usb-elm-esectioncontrol-sph) 中所述。

#### 计算粒子体积

目前没有自动计算与这些粒子相关联的体积的功能。因此，您需要提供一个特征长度，用于计算粒子体积，进而用于计算与粒子相关联的质量。假定节点在空间中均匀分布，每个粒子与以粒子为中心的小立方体相关联。当堆叠在一起时，这些立方体将用身体自由表面的一些小近似填充体的总体积。特征长度是立方体边长的一半。从实际角度，一旦您创建了节点，您可以使用两个节点之间的距离的一半作为特征长度。或者，如果您知道零件的质量和密度，您可以计算零件的体积，并将其除以零件中的粒子总数，以获得与每个粒子相关的小立方体的体积。这个小体积的立方根的一半是这个粒子集的合理特征长度。如果您请求将模型定义数据打印到数据（`.dat`）文件，您可以检查模型中各个集的质量（参见 ["输出中的模型和历史定义摘要," 第 4.1.1 节"](pt02ch04s01aus38.md#usb-out-ooutput-modelhist-sum)）。

| **输入文件用法：** | 使用以下选项定义光滑粒子流体力学体： |

| | ``` [*ELEMENT](../key/key-link.md#usb-kws-melement), TYPE=PC3D, ELSET=*particle_body* *element number*, *node number* ``` 根据需要重复数据行。 ``` [*SOLID SECTION](../key/key-link.md#usb-kws-msolidsection), ELSET=*particle_body*, MATERIAL=*material_name* *characteristic length associated with particle volume* ``` |

#### 平滑长度计算

尽管粒子单元在模型中每个单元使用一个节点定义，光滑粒子流体力学方法根据影响范围内相邻粒子的贡献计算每个单元的贡献。这个影响范围球的半径在文献中称为平滑长度。平滑长度独立于上面讨论的特征长度，并控制方法的插值属性。默认情况下，平滑长度是自动计算的。随着变形进展，粒子相互移动，因此给定粒子的邻居可以（通常会）改变。每个增量，Abaqus/Explicit 重新计算这个局部连接性，并基于以感兴趣粒子为中心的粒子云的贡献计算运动学量（如法向和剪切应变、变形梯度等）。然后，以与减缩积分砖单元类似的方式计算应力，然后用于基于光滑粒子流体力学公式计算粒子云中粒子的单元节点力。

默认情况下，Abaqus/Explicit 在分析开始时计算平滑长度，使得与单元相关联的粒子的平均数量大约在 30 到 50 之间。平滑长度在分析过程中保持不变。因此，每个单元的粒子平均数量可以取决于模型中的行为是膨胀还是压缩而分别减少或增加。如果分析主要是压缩性的，则与给定单元相关联的粒子总数可能超过允许的最大值，分析将停止。默认情况下，允许与一个单元关联的最大粒子数为 140。

您可以控制这些设置，如 ["在截面控制中使用光滑粒子流体力学（SPH）的截面控制," 第 27.1.4 节"](pt06ch27s01aus113.md#usb-elm-esectioncontrol-sph) 中所述。

#### 光滑粒子流体力学域

在分析开始时计算一个矩形区域作为边界框，粒子将在其中被跟踪。这个固定的矩形盒子比整个模型的总体尺寸大 10%，并以模型的几何中心为中心。随着分析进展，如果粒子在此盒子之外，它表现得像自由飞行点质量，不对光滑粒子流体力学计算做出贡献。如果粒子在稍后阶段重新进入盒子，它将再次包含在计算中。

您可以修改边界框的大小，如 ["在截面控制中使用光滑粒子流体力学（SPH）的截面控制," 第 27.1.4 节"](pt06ch27s01aus113.md#usb-elm-esectioncontrol-sph) 中所述。

### 约束

由于 PC3D 单元是拉格朗日单元，它们的节点可以参与其他特征，如其他单元、连接器或约束。由于这些单元没有面或边，因此无法使用 PC3D 单元定义基于元素的表面。因此，不能为粒子定义需要基于元素的表面的约束（如紧固件）。

### 相互作用

用粒子建模的体可以通过接触与 其他有限元网格化的体相互作用。接触相互作用与任何基于节点的表面（与粒子相关联）和基于元素或解析表面之间的接触相互作用相同。通用接触和接触对都可以使用。允许使用适用于涉及基于节点的表面的接触的所有相互作用类型和公式，包括内聚行为。可以通过常规选项分配不同的接触属性。默认情况下，粒子不是通用接触域的一部分，类似于其他 1 节点单元（如点质量）。粒子的默认接触厚度与截面定义上指定的特征长度相同；因此，为了接触目的，粒子表现得像球体，其半径等于与上述粒子体积相关的小立方体内接球的半径。

您不应该为与 PC3D 单元关联的节点指定零接触厚度，否则接触可能无法稳健地解决。建议的方法是使用默认值或指定合理的接触厚度。

允许不同体之间的相互作用都使用 PC3D 单元建模。然而，这种相互作用仅在碰撞的光滑粒子流体力学体由相同流体类材料制成的情况下才有意义，例如落入部分盛有水的桶中的水滴。在固体相关应用中，如建模子弹穿透装甲板，其中一个体必须使用常规有限元建模。

不能在粒子和欧拉区域之间定义接触相互作用。

### 输出

PC3D 单元可用的单元输出包括连续体单元的所有力学相关输出：应力；应变；能量；以及状态、场和用户定义变量的值。节点输出包括 Abaqus/Explicit 分析中通常可用的所有输出变量。

粒子可以通过圆盘在 Abaqus/CAE 中可视化。在轮廓图中，场输出变量的值显示为彩色圆形斑块。符号图也可用。

### 限制

光滑粒子流体力学分析受以下限制：
- 当变形不太严重且单元没有变形时，该方法通常不如拉格朗日有限元分析准确。在较高变形范围内，耦合欧拉-拉格朗日分析也通常更准确。光滑粒子动力学方法应主要用于常规有限元方法或耦合欧拉-拉格朗日方法已达到其固有局限性或执行成本过高的情况。
- 当材料处于拉伸应力状态时，粒子运动可能变得不稳定，导致所谓的拉伸不稳定性。这种不稳定姓与标准光滑粒子动力学方法的插值技术严格相关，在模拟固体拉伸状态时尤其明显。因此，粒子倾向于聚集在一起并表现出类似断裂的行为。
- 与使用连续体单元（如 C3D8R 单元）定义的相同体的质量分布相比，用粒子单元定义的体中的质量分布有些不同。当使用粒子单元时，该体中所有粒子的体积是相同的。因此，该体中所有粒子相关联的节点质量是相同的。如果节点不是以规则立方体排列的，质量分布有些不准，特别是在被建模体的自由表面上。
- 表面载荷不能在 PC3D 单元上指定。但是，分布载荷（如压力）可以施加到其他有限元表面，这些表面可以通过接触相互作用将压力施加到粒子单元上。
- 使用不同截面定义未定义的粒子建模的体不会相互相互作用。因此，您不能使用光滑粒子流体力学来建模不同材料体的混合。
- 此功能在 Abaqus/CAE 中不直接支持。但是，您可以执行以下操作：- 您可以使用 Abaqus/CAE 中的现有功能生成质量单元，写入输入文件，然后手动编辑输入文件将质量单元转换为粒子。- 您可以使用有限元转换为 SPH 粒子（参见 ["有限元转换为 SPH 粒子中的边界条件," 第 15.2.2 节"](pt04ch15s02aus96.md#usb-anl-ashpconv-bc)）在分析开始时（时间等于零的基于时间的转换准则）。- 您可以使用 C3D8R 单元创建网格，写入输入文件，然后使用脚本将这些单元转换为粒子，如 Dassault Systmes 知识库中"从实体网格生成粒子单元"所述，网址为 [www.3ds.com/support/knowledge-base](http://www.3ds.com/support/knowledge-base)。
- 在通过一个实体截面定义给出的给定体（零件）内，不能有选择性地对由此定义引用的单元子集指定重力载荷和质量缩放。相反，这两个特征必须应用于与实体截面定义关联的元素集中的所有单元。

光滑粒子流体力学计算在大多数情况下跨并行域分布；但是，对于具有以下任何特征的模型，它们都由单个域（具有单个处理器）执行（这通常会显著降低并行可扩展性）：
- 在分析开始后进行有限元到 SPH 粒子的转换（参见 ["有限元转换为 SPH 粒子中的边界条件," 第 15.2.2 节"](pt04ch15s02aus96.md#usb-anl-ashpconv-bc)）。并行光滑粒子流体力学实现仅支持在分析开始时（时间等于零）转换为 SPH 粒子。
- PC3D 单元的多个实体截面
- 指定为截面控制的归一化核
- 材料属性的预定义场变量（包括温度）依赖性

如果使用多个 CPU，光滑粒子流体力学分析受以下限制：
- 不支持光滑粒子流体力学从节点的接触输出。
- 不支持单元历史输出。
- 除了整个模型外，不支持能量历史输出。
- 无法激活动态负载平衡。
- 如果任何 SPH 粒子参与通用接触，则所有 SPH 粒子必须包含在通用接触定义中。
- 建议每个域至少 10,000 个粒子以实现良好的可扩展性。
- 如果使用大量 CPU，可能需要显著增加内存使用量。
