# 17.14.5 使用六面体单元对复杂实体进行网格划分





您可以使用Part模块创建包含沿旋转轴平移的复杂实体旋转零件，也可以创建包含围绕选定中心点扭转的实体拉伸零件。 自由网格划分允许您使用四面体单元对这些零件进行网格划分，如["使用三角形和四面体单元的自由网格划分，" 第17.10.3节](pt03ch17s10s03.md)中所述。 此外，您可以使用六面体单元对包含扭转的拉伸零件进行网格划分，如图 17-133](pt03ch17s14s05.md#mgn-complex-gear) 所示。

**图 17-133** 带有扭转的拉伸零件上的六面体扫掠网格。

![](../graphics/mgn-complex-gear.png)

但是，如果您想使用六面体单元对经过平移的旋转零件进行网格划分，则可能必须引入划分来使零件可进行扫掠网格划分，如下所述。

对于Mesh模块创建六面体单元的三维扫掠网格，连接源面和目标面的每个侧面必须只包含单个面（请参阅["三维实体的扫掠网格划分，" 第17.9.3节](pt03ch17s09s03.md)中的[图 17-102](pt03ch17s11s07.md#mgn-sweptconnect)）。 但是，当您创建的零件旋转超过180度然后沿旋转轴平移时，Abaqus/CAE会沿实体的长度插入环。 这些环仅存在于零件表面，它们不会创建穿过实体的面。 结果，连接源面和目标面的侧面现在包含多个面，除非您引入划分，否则该零件无法使用六面体单元进行扫掠网格划分。

例如，[图 17-134](pt03ch17s14s05.md#mgn-complex-partition) 显示了一个代表螺旋弹簧的零件。 该图还显示了创建零件时Abaqus/CAE插入的环。 这些环划分了源面和目标面之间的连接面； 因此，在您对零件进行划分之前，无法使用六面体单元的扫掠网格对螺旋弹簧的实例进行网格划分。

**图 17-134** 环将线圈分成多个段。

![](../graphics/mgn-complex-partition-nls.png)

划分操作使用N侧补片通过实体线圈引入划分以定义新的面。 您选择环来定义N侧补片，如图 17-135](pt03ch17s14s05.md#mgn-complex-sweep) 所示。

**图 17-135** N侧补片划分单元格。

![](../graphics/mgn-complex-sweep-nls.png)

您现在可以对零件实例进行种子布置并使用六面体单元生成扫掠网格，如图 17-136](pt03ch17s14s05.md#mgn-complex-patch) 所示。

**图 17-136** 该模型使用六面体单元进行扫掠网格划分。

![](../graphics/mgn-complex-patch.png)
