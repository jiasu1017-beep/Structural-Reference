# 11.13.8 定义扫掠路径和扫掠轮廓





要创建扫掠特征，请从主菜单栏选择**形状**![](../graphics/images/arrow.gif)**实体**![](../graphics/images/arrow.gif)**扫掠**、**形状**![](../graphics/images/arrow.gif)**壳**![](../graphics/images/arrow.gif)**扫掠**或**形状**![](../graphics/images/arrow.gif)**切割**![](../graphics/images/arrow.gif)**扫掠**，或从零件模块工具箱中选择等效工具。Abaqus/CAE显示**创建实体扫掠**、**创建壳扫掠**或**创建切割扫掠**对话框。

扫掠是两步操作：首先定义扫掠路径，然后定义扫掠轮廓。轮廓沿路径的长度扫掠以形成三维实体、壳或切割特征。扫掠路径可以是您使用草图编辑器创建的任何连续路径，或者是零件中任何连接的边或线框的组合。后者选项允许您定义三维扫掠路径，例如点对点样条线；草图方法提供更大的灵活性，但仅支持二维路径。

扫掠轮廓可以在草图编辑器中定义，也可以通过在几何中选择组件来定义。对于实体或切割扫掠特征，您可以选择零件中的一个面作为扫掠轮廓；对于壳扫掠特征，您可以选择零件中的一个或多个边作为扫掠轮廓。

您可以创建其扫掠轮廓与扫掠路径偏移的扫掠实体、扫掠壳或扫掠切割特征。

您可以控制扫掠轮廓在沿扫掠路径移动时其方向是否改变。

创建扫掠实体或切割特征时，扫掠轮廓必须是闭合的。但是，与扫掠轮廓不同，扫掠路径可以是开放的或闭合的，无论您是在创建扫掠实体、壳还是切割特征。如果扫掠路径闭合，路径的两端必须切向相遇。
