# 10.5.4 从Nastran输入文件导入模型





您可以通过从主菜单栏选择****文件（File）**![](../graphics/images/arrow.gif)**导入（Import）**![](../graphics/images/arrow.gif)**模型（Model）****使用Nastran输入文件将模型导入Abaqus/CAE。Abaqus/CAE将Nastran批量数据中的许多常见实体导入模型；有关支持的实体更多信息，请参阅["将Nastran批量数据文件转换为Abaqus输入文件，"Abaqus Analysis User's Guide第3.2.28节"](../usb/usb-link.md#usb-int-dnasabaproc)。

当您将Nastran输入文件导入Abaqus/CAE时，您可以控制Nastran到Abaqus转换的多个方面。这些选项包括：
- 从Nastran到Abaqus单元类型的单元映射。
- 选择将Nastran截面或膜数据转换为Abaqus截面的方法。您可以为每个Nastran PSHELL或PCOMP属性ID创建对应的Abaqus截面，为引用相同材料的所有均质单元创建截面，或者为Nastran输入文件中每个方向、偏移和/或厚度的组合创建单独的壳或膜截面。
- 使用预积分壳截面。
- 为Abaqus/CAE模型中创建的所有密度、质量和转动惯性值应用缩放乘数。
- 将使用Nastran库SOL 101、SOL 108或SOL 111的语句转换为具有载荷案例的Abaqus扰动步骤。
- 通过在偏移位置创建新节点、更改梁连接到新节点以及刚性耦合新节点和原始节点来耦合梁单元偏移。
