# 9.7.12 Compressing the file size of the current model database







Select ****File**![](../graphics/images/arrow.gif)**Compress MDB**** from the main menu bar to compress the current model database (MDB). Compressing the MDB attempts to reduce the file size. The change will be most noticeable if you have deleted multiple items from your model. 

Abaqus/CAE uses the compression function if you select ****File**![](../graphics/images/arrow.gif)**Save As**** to save a file with a new file name.
