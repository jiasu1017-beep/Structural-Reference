# 12.8.3 在Abaqus/Explicit中规范化用户定义的材料数据







作为独立变量函数插值材料数据需要在分析过程中进行材料数据值的表查找。表查找在Abaqus/Explicit中频繁发生，如果从规则数据进行插值则最为经济。

如有必要，Abaqus/Explicit使用误差容限来规范化输入数据。每个独立变量范围内的时间间隔数量被选择，使得分段线性规范化数据与您定义的每个点之间的误差小于因变量范围的容限乘积。

有关更多信息，请参见"材料数据定义"中的"在Abaqus/Explicit和Abaqus/CFD中规范化用户定义的数据"，Abaqus分析用户指南第21.1.2节。

**规范化用户定义的材料数据：**

1. 从**编辑材料**对话框的菜单栏中，选择**常规**![](../graphics/images/arrow.gif)**规范化**。（有关显示**编辑材料**对话框的信息，请参见"创建或编辑材料"，第12.7.1节。）
2. 在**Rtol**字段中，输入您希望Abaqus用于规范化材料数据的容限。默认值为0.03。
3. 指定您希望如何规范化应变率相关数据：- 选择**对数**使用对数间隔而非均匀间隔来规范化应变率数据。此选项通常能更好地匹配典型的应变率相关曲线。- 选择**线性**使用均匀间隔进行应变率数据的规范化。
4. 单击**确定**关闭**编辑材料**对话框。或者，您可以从**编辑材料**对话框的菜单中选择其他材料行为进行定义（有关更多信息，请参见"浏览和修改材料行为"，第12.7.2节）。
