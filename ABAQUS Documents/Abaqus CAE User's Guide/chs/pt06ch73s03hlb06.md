# 73.3.6 将对象（如载荷和截面）与集和曲面关联







当您选择要向其应用对象（如载荷或截面）的区域时，您可以在以下技术之间进行选择：

**在视口中高亮显示选择**

单击当前视口中的实体；例如，一个面或一组顶点。使用**[Shift]**+**单击**和**[Ctrl]**+**单击**将实体追加到您的选择中。

**提示：**您可以使用**Selection**工具栏限制可以选择的实体类型。更多信息，请参见[第6章,"在视口中选择对象"](pt01ch06.md)。

****Sets**或**Surfaces****

使用您使用集或曲面工具集创建的现有集或曲面。如果您选择使用现有集或曲面，将出现**Region Selection**对话框。该对话框包含适用于您正在应用的对象的集或曲面列表。

单击提示区域中的相应按钮在两种技术之间切换。更多信息，请参见["使用集和曲面工具集," 第73.3节](pt06ch73s03.md)。您还可以使用选择组来加快选择过程。您可以将选定的实体复制到选择组中，也可以将选择组中的实体粘贴到您的选择中。更多信息，请参见["什么是选择组？," 第6.1.2节](pt01ch06s01s02.md)。
