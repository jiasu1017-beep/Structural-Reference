# 16.11.8 定义流体速度场





您可以创建流体速度场来为Abaqus/CFD分析中的流体速度定义初始值。有关更多信息，请参见[第30章，"流体动力学分析"](pt04ch30.md)。"

**创建或编辑流体速度场的步骤：**

1. 使用以下方法之一显示流体速度场编辑器： - 要创建新的流体速度场，请按照["创建预定义场"章节第16.8.3节](pt03ch16s08hlb03.md)中概述的程序进行操作（**类别**：`流体`；**所选步骤的类型**：`流体速度`）。 - 要使用菜单或管理器编辑现有的流体速度场，请参见["编辑依赖于步骤的对象"章节第3.4.13节](pt01ch03s04hlb05.md)。**注意：**您只能在初始步骤中创建或编辑流体速度场。有关更多信息，请参见[第30章，"流体动力学分析"](pt04ch30.md)。
2. 默认情况下，流体速度场应用于整个模型。如果要更改流体速度场适用的区域，请执行以下操作： 1. 单击![](../graphics/ico_selectBlue.png)。 2. 从视口中选择一个或多个区域或集合。 3. 在提示区域中单击**完成**。Abaqus/CAE更新流体速度场编辑器以反映您的选择。
3. 输入速度分量的初始值。
4. 单击**确定**保存数据并退出编辑器。
