# 5.1.2 使用视图选项控制相机





使用视图选项控制当前相机模式和其他你无法使用视图操作工具设置的选项。

从主菜单栏中选择****View**![](../graphics/images/arrow.gif)**View Options****可以访问当前视口的视图选项，如图 [图 5-2](pt01ch05s01s02.md#viw-viewoptions) 所示。

**图 5-2** **视图选项**对话框。

![](../graphics/viw-viewoptions-nls.png)

你可以使用**视图选项**对话框控制：
- 视场角（实际上是当前视图的放大倍率）。
- 相机模式（电影模式允许你将相机移入模型并穿过模型）。
- 电影模式相机到近平面的距离（对象能够靠近相机且仍保留在视图中的最近距离）。
- 电影模式相机到远平面的距离（对象能够远离相机且仍保留在视图中的最远距离）。

**注意：**为大型模型指定近平面和远平面距离可以通过排除视图之外超出指定范围的任何模型部分来提高显示性能。

在可视化模块中，**视图选项**对话框还包括**相机移动**选项。你可以让相机跟随局部坐标系的运动，并选择相机是否也跟随所选坐标系的旋转。如果电影模式开启，你可以将相机定位在所选坐标系的原点上。有关可视化模块中视图选项的更多信息，请参阅["自定义相机移动，" 第 55.9 节](pt05ch55s09.md)。

**要使用视图选项：**

1. 从主菜单栏中选择****View**![](../graphics/images/arrow.gif)**View Options****。**视图选项**对话框出现。
2. 如有需要，更改**视场角**以调整视口中的视图大小：
   - 减小角度以放大视图
   - 增大角度以缩小视图
3. 切换**使用电影模式**开启或关闭，以在默认和电影相机模式之间切换。激活电影相机模式不会更改当前视图。但是，如果你应用默认相机模式（关闭**使用电影模式**），你会看到以下效果之一：
   - 如果模型完全在平截头体内，当前视图不会改变。
   - 如果模型的一部分被近平面或远平面裁剪，该部分会重新出现在视图中，并且 Abaqus/CAE 会重置近平面和远平面距离，以便下次激活电影模式时将整个对象包含在视图中。
   - 如果电影模式相机位于模型内部，相机将沿当前视图方向"后退"，使整个模型位于相机前方。Abaqus/CAE 调整视场角以在电影模式视图中以类似大小显示模型，并重置近平面和远平面距离，以便下次激活电影模式时将整个对象包含在视图中。
   **提示：**使用[循环](pt01ch05s06hlb07.md)视图操作工具返回上一个视图。
4. 如有需要，调整**近平面距离**。Abaqus/CAE 会从视图中移除电影模式相机和近平面之间的任何对象或对象部分。
5. 如有需要，单击**指定**并调整**远平面距离**。默认情况下，Abaqus/CAE 自动计算远平面距离，将其设置为超出模型中最远点的值，并在你操作视图时进行调整，以便模型的任何部分都不会被排除在视图之外。如果你指定了距离，Abaqus/CAE 会从视图中移除距相机超出该距离的任何对象或对象部分。
6. 单击**确定**实施更改并关闭对话框。你的更改会在会话期间保存。
