# 15.12.2 创建相互作用属性





您可以通过在相互作用属性编辑器中输入数据来创建相互作用属性。编辑器的格式因您正在定义的属性类型而异；创建属性时，您必须首先指定属性类型，以便出现相应的编辑器。更多信息，请参见「理解相互作用属性」，第15.4节，和「使用相互作用属性编辑器」，第15.14节。

**创建相互作用属性的步骤：**

1. 从主菜单栏选择 ****Interaction**![](../graphics/images/arrow.gif)**Property**![](../graphics/images/arrow.gif)**Create****。**提示：**您也可以使用「相互作用」模块工具箱中的![](../graphics/ico_interactionProperty.png)工具创建相互作用属性。此时会出现「Create Interaction Property」对话框。
2. 输入属性名称。有关命名对象的更多信息，请参见「使用基本对话框组件」，第3.2.1节。
3. 选择属性类型，然后点击「Continue」。会出现您指定的属性类型的编辑器。
4. 在编辑器中，输入定义相互作用属性所需的所有数据。**注意：**您可以通过从主菜单栏选择 ****Help**![](../graphics/images/arrow.gif)**On Context****，然后点击感兴趣的编辑器功能来显示该功能的帮助。
5. 点击「OK」保存数据并退出编辑器。
