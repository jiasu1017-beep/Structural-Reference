# 45.5.3 自定义向量和张量标签

您可以在符号图的每个向量或张量箭头旁边显示标签，以数值形式表示其值。您还可以通过更改颜色、字体、类型大小、数字格式和小数位数来自定义这些标签。

**要自定义向量或张量标签：**

1. 找到**Labels**选项。从主菜单栏中选择****Options**![](../graphics/images/arrow.gif)**Symbol****，或单击工具箱中的![](../graphics/ico_plotSymbolOptions.png)；然后在出现的对话框中单击**Labels**选项卡。单击**Vector**选项卡或**Tensor**选项卡；您选择的变量类型的符号图**Labels**选项变得可用。
2. 切换**Display value next to vector symbol**以为所选类型的向量显示向量标签。
3. 为所选类型的向量选择向量标签的颜色。1. 单击**Text color**颜色样本![](../graphics/usi-colorsample.png)。Abaqus/CAE显示**Select Color**对话框。2. 使用**Select Color**对话框中的方法选择新颜色。有关更多信息，请参见["自定义颜色，" 第3.2.9节](pt01ch03s02s09.md)。3. 单击**OK**关闭**Select Color**对话框。
4. 为所选类型的向量选择向量标签的字体。1. 单击**Set Font**。Abaqus/CAE显示**Select Font**对话框。2. 使用**Select Font**对话框中的方法选择新字体、大小和样式。有关更多信息，请参见["自定义字体，" 第3.2.8节](pt01ch03s02s08.md)。3. 单击**OK**关闭**Select Font**对话框。
5. 要更改所选向量类型的向量标签格式，请单击**Number Format**箭头并选择以下格式之一：- 选择**Scientific**以科学计数法显示图例值，即将图例值表示为1到10之间的十进制值乘以10的幂。当您为此格式选择数字`501,000`时，该值在图例中显示为`5.01e5`。- 选择**Fixed**以不使用指数值显示图例值。当您选择此格式时，数字`501,000`在图例中显示为`501000.00`。- 选择**Engineering**以选择工程计数法，以类似于科学计数法的格式表示值，但仅允许3的倍数的指数值。当您选择此格式时，数字`501,000`在图例中显示为`501E3`。
6. 单击**Decimal places**箭头以指定每个标签中的小数位数。
7. 单击**Apply**实施您的更改。当前视口中的符号图更改为显示带有您选择的格式的标签。默认情况下，您对符号图选项的更改在会话期间保存，并将影响当前视口和从当前视口创建的任何新视口中的所有后续符号图。如果您想在后续会话中保留更改，请将它们保存到文件。有关更多信息，请参见["保存自定义设置以供后续会话使用，" 第55.1.1节](pt05ch55s01s01.md)。
