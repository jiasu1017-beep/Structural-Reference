# 11.5 复制零件





从主菜单栏选择**零件**![](../graphics/images/arrow.gif)**复制**![](../graphics/images/arrow.gif)***零件名称***，可以将零件复制到新零件。您可以创建原始零件的完全相同的副本，或者可以在复制操作期间执行以下操作：

**压缩特征**

Abaqus/CAE将所有特征和参数信息简化为零件的简单定义。结果，如果您在随后修改零件，Abaqus/CAE会更快地重新生成零件；但是，您将不再能够修改零件的任何参数。更多信息，请参见"简化零件的特征列表"，第11.3.3节。

**按比例缩放零件**

Abaqus/CAE按您输入的比例因子缩放新零件。如果您选择缩放零件，Abaqus/CAE也会压缩其特征。您可以使用缩放来校正导入的零件。如果导入零件的比例不正确，您可以复制零件到新零件，并在过程中将其缩放到正确的尺寸。在某些情况下，您可以通过缩小零件、修复零件然后将零件缩放回原始尺寸来生成有效零件。您也可以在导入过程中缩放导入的零件。更多信息，请参见"导入草图和零件"，第10.7节。

**关于平面对零件进行镜像**

Abaqus/CAE关于选定平面（*X*–*Y*、*Y*–*Z*或*X*–*Z*）对零件进行镜像。如果您选择**关于平面对零件进行镜像**选项，Abaqus/CAE会选择**压缩特征**选项。

要关于除主平面之外的平面对零件进行镜像且不压缩特征，请使用**形状**![](../graphics/images/arrow.gif)**变换**![](../graphics/images/arrow.gif)**镜像**。更多信息，请参见"镜像零件"，第11.28节。

**将断开连接的区域分离为零件**

在某些情况下，当您导入IGES或VDA-FS格式零件并选择**缝合边缘**修复选项时，Abaqus/CAE会将单独零件作为单个零件导入。如果切换**将断开连接的区域分离为零件**选项并将导入的零件复制到新零件，Abaqus/CAE会将断开连接的区域分离为单独的零件。更多信息，请参见"控制导入过程"，第10.3节。

您可以复制网格零件并基于节点连接性将其分离为断开连接的零件。Abaqus/CAE假设所有连接的节点属于单个零件，不考虑单元类型。但是，Abaqus/CAE忽略具有非线性非对称变形（CAXA）的轴对称实体单元与某些线单元（连接器、弹簧、阻尼器、间隙和关节）之间的连接性。

您可以复制包含几何和孤儿网格特征的零件。压缩、缩放、镜像和分离断开连接区域选项可用于任何零件。

**注意：**在复制操作期间压缩或镜像零件时，不会复制参考点、点零件和基准。
