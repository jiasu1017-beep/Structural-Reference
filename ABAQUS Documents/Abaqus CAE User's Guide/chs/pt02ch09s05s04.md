# 9.5.4 创建和运行您自己的脚本





您在Abaqus/CAE会话期间执行的几乎所有操作都可以由包含一组Abaqus Scripting Interface命令的脚本（`*script_name*.py`）重复。相反，从Abaqus/CAE内部运行脚本相当于使用Abaqus/CAE提供的菜单、工具箱和对话框执行相应的操作。

您可以创建脚本来重复您在会话期间常规执行的操作；例如，您可以编写一个脚本来定义常用材料的材料属性，或者生成以特定视图方向显示的特定变量的等值线图。

Abaqus/CAE命令是用Python脚本语言编写的，您可以使用Python来增强Abaqus/CAE生成的脚本。命令以ASCII文本形式存储在重放、日志和恢复文件中，以及您创建的Abaqus/CAE脚本中。因此，您可以使用标准文本编辑器来编辑文件内容。有关命令的更多信息，请参阅[Abaqus Scripting User's Guide](../cmd/cmd-link.md#cmd)。

要运行脚本，请从主菜单栏选择****文件（File）**![](../graphics/images/arrow.gif)**运行脚本（Run Script）****，然后从**运行脚本（Run Script）**对话框中选择要运行的脚本。

**注意：**您应该使用Abaqus/CAE执行过程的恢复选项来运行日志文件并重新创建已保存的模型数据库。（输入`abaqus cae recover=`*model_database_name*.jnl``。）选择****文件（File）**![](../graphics/images/arrow.gif)**运行脚本（Run Script）****来运行日志文件可能会导致不完整的模型数据库。

您也可以使用Abaqus Python开发环境（Abaqus PDE）创建和运行脚本。脚本中的Abaqus Scripting Interface命令必须在Abaqus PDE的内核工作区中运行。有关Abaqus PDE的更多信息，请参阅[Abaqus Scripting User's Guide第7章，"使用Abaqus Python开发环境"](../cmd/cmd-link.md#cmd-pde-using)。
