# 16.10.13 指定温度





您可以创建温度边界条件来为选定区域指定温度。

**要在区域中指定温度：**

1. 使用以下方法之一显示温度边界条件编辑器：- 要创建新的温度边界条件，请按照"创建边界条件"中的步骤操作（第16.8.2节）（**类别**：`Other`；**所选步骤的类型**：`Temperature`）。- 要使用菜单或管理器编辑现有温度边界条件，请参见"编辑依赖于步骤的对象"（第3.4.13节）。要编辑应用边界条件的区域，请参见"编辑应用指定条件的区域"（第16.8.4节）。
2. 如果编辑器顶部出现**Method**字段，请单击字段右侧的箭头，然后选择以下之一：- 如果要指定温度值，请选择**Specify magnitude**。- 如果要将温度大小固定为前一个通用步骤的最终值，请选择**Fixed at current magnitude**。仅当所选步骤两种方法都有效时，编辑器中才会显示**Method**选项。
3. 如果编辑器中出现**Distribution**字段，请单击字段右侧的箭头，然后选择以下之一：- 选择**Uniform**来定义均匀边界条件。- 选择**User-defined**在用户子程序[`DISP`](../sub/sub-link.md#sub-xsl-disp)中定义边界条件。有关更多信息，请参阅以下章节：- "指定通用作业设置"，第19.8.6节- 《Abaqus User Subroutines Reference Guide》第1.1.4节"DISP"- 选择解析场来定义空间变化的边界条件。只有对该边界条件类型有效的解析场才显示在选择列表中。或者，您可以单击![](../graphics/ico_analyticalField.png)创建新的解析场。（有关更多信息，请参见第58章"解析场工具集"。）仅当您创建边界条件或在创建边界条件的步骤中编辑边界条件时，编辑器中才会显示**Distribution**选项。如果您选择了**Fixed at current magnitude**方法，则此选项不可用。
4. 如果您选择了**Uniform**或解析场分布选项，请执行以下步骤：1. 如果适用，请在**Magnitude**字段中输入温度大小。2. 如果适用，单击**Amplitude**字段右侧的箭头，然后从出现的列表中选择所需的幅值。或者，您可以单击![](../graphics/ico_amplitudeSmall.png)创建新的幅值。（有关更多信息，请参见第57章"幅值工具集"。）**注意：**如果您在初始步骤中创建边界条件（此时大小始终为零），或者将大小固定为前一个分析步骤的最终值，则**Magnitude**和**Amplitude**字段不可用。
5. 如果您选择了**User-defined**分布选项，请执行以下步骤：1. 如果需要，在**Magnitude**字段中输入温度大小。您在编辑器中输入的大小数据将传递到用户子程序中。（如果您在初始步骤中创建边界条件，则**Magnitude**字段不可用。）2. 单击**OK**保存任何输入的数据并退出编辑器。3. 进入Job模块，然后显示感兴趣的分析作业的作业编辑器。（有关更多信息，请参见"创建、编辑和操作作业"，第19.7节。）4. 在作业编辑器中，单击**General**选项卡，然后指定包含定义边界条件的用户子程序的文件。有关更多信息，请参见"指定通用作业设置"，第19.8.6节。**注意：**您只能在作业编辑器中指定一个用户子程序文件；如果您的分析涉及多个用户子程序，则必须将用户子程序合并到一个文件中，然后指定该文件。
6. 如果选择了壳区域，请在**Degrees of freedom**字段中输入要指定的温度自由度。关于壳温度自由度的标签方式，请参阅《Abaqus Analysis User's Guide》第29.6.2节"选择壳单元"中的"热传导壳单元"或"耦合温度-位移壳单元"。您可以通过用逗号分隔每个数字来输入多个自由度，范围从11到31。例如，如果您要指定自由度11、12和13，则必须输入以下内容：```11,12,13```
7. 单击**OK**保存数据并退出编辑器。
