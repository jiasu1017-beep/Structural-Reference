# FXScrollWindow





滚动窗口组件用于滚动任意的子窗口。当用户界面的某些部分需要滚动时（例如当应用程序需要在小屏幕上运行时），可以使用滚动窗口。
![](../graphics/gui-fxscrollwindow.png)

### FXScrollWindow(p, opts=0, x=0, y=0, w=0, h=0)

构造一个滚动窗口。
| **参数** | **类型** | **默认值** | **描述** |
| --- | --- | --- | --- |
| p | FXComposite |  |  |
| opts | Int | 0 |  |
| x | Int | 0 |  |
| y | Int | 0 |  |
| w | Int | 0 |  |
| h | Int | 0 |  |
