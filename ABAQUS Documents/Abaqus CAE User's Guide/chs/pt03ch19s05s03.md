# 19.5.3 后处理优化





要在可视化模块中查看优化过程的结果，您必须将优化结果和分析结果组合成单个结果输出数据库文件，如[「组合优化结果，」第 19.12.8 节](pt03ch19s12hlb08.md)中所述。

**创建基础结果输出数据库文件**

您选择作为基础结果的输出数据库文件成为组合结果输出数据库文件的起点。

**在初始设计周期和最终设计周期之间选择**

您可以指定基础结果取自初始设计周期生成的输出数据库文件，或取自最终设计周期生成的输出数据库文件。在大多数情况下，您将选择初始设计周期并查看从初始设计周期到最终设计周期的优化进程；例如，跟随刚度变化。如果您执行了频率优化并想要查看前几个模式的频率和模态形状，请选择最终设计周期。如果您执行了形状优化，您应该从第一个设计周期生成的输出数据库文件中获取基础结果。

**选择原始模型**

优化模块在运行优化过程的初始设计周期之前修改材料定义和截面分配。原始模型是在优化模块进行任何修改之前存在的模型。

您可以指定基础结果取自原始模型生成的输出数据库文件。但是，优化过程不会对原始模型运行 Abaqus 分析。因此，您必须先手动运行分析，然后才能选择原始模型作为基础结果。（原始模型的 Abaqus 输入文件与优化过程生成的所有输入文件一起存储在 `*优化过程名称*\SAVE.inp` 目录中。原始模型的输入文件名称为 `*优化过程名称*_org.inp`。）

您用于基础结果的输出数据库文件必须不包含零件和装配地生成，如[「编写不包含零件和装配的输入文件，」第 9.10.4 节](pt02ch09s09hlb04.md)中所述。优化模块生成的输入文件不包含零件和装配，无论 Abaqus/CAE 中的任何用户设置如何。但是，如果您从不是由优化模块或通过从 Abaqus/CAE 执行作业生成的输入文件创建基础结果，则必须确保生成的输出数据库文件不包含零件和装配。

**附加到基础结果**

在指定了将用作基础结果源的输出数据库文件后，Abaqus/CAE 将优化结果和 Abaqus 分析结果追加到组合输出数据库文件中。

**追加优化结果**

Abaqus/CAE 在每个设计周期后将优化变量作为场数据追加到组合输出数据库文件中，每个设计周期都显示为组合输出数据库文件中的一帧。同样，Abaqus/CAE 将优化值作为历史数据追加到组合输出数据库文件中。

**追加分析结果**

您可以执行以下操作来指定哪些分析结果数据被写入组合结果输出数据库文件：
- 指定应从哪些设计周期写入分析结果。
- 指定应从哪些模型写入分析结果。Abaqus/CAE 为优化过程中的每个模型创建一个组合结果输出数据库文件。
- 指定对于所选模型，应从模型的哪些步骤写入分析结果。
- 指定应写入哪些分析场变量。

在组合过程中，历史数据不会写入组合数据库文件。组合输出数据库文件仅包含来自基础结果输出数据库文件的历史数据。

建议您在创建优化过程后保存初始和最终设计周期的分析结果。优化完成后，您可以选择在初始设计周期期间生成的输出数据库文件作为基础结果。然后，您可以将基础结果输出数据库文件与每个设计周期的优化结果以及最终设计周期的分析结果组合，如[表 19-1](pt03ch19s05s03.md#usi-ana-opt-combine)所示。

**表 19-1** 在组合输出数据库文件中保存优化和分析结果。
|  | 设计周期 |
| --- | --- |
| 初始 | 第一个 | 第二个 | 第三个 | 第四个（最终） |
| 保存的数据 | 分析结果（场和历史数据） | 优化结果 | 优化结果 | 优化结果 | 优化结果和分析结果（仅场数据） |
| 操作 | 创建基础结果 | 追加到基础结果 | 追加到基础结果 | 追加到基础结果 | 追加到基础结果 |

例如，如果您执行了拓扑优化，则可以执行以下操作：- 查看模型的初始状态；例如，初始几何形状以及载荷和边界条件。
- 查看每个设计周期中优化变量（归一化材料分布变量 MAT_PROP_NORMALIZED）的变化，以观察优化的进程。
- 查看模型的最终状态；例如，优化后的几何形状以及位移、应力和应变。
- 创建跟踪目标和约束变化的历史图。

您可以对形状和尺寸优化的开始和结束状态以及进程应用类似的条件。

组合优化结果仅支持使用以下分析情况配置的优化：
- 简单分析（单个模型、单个步骤和单个载荷工况）
- 频率或模态分析
- 具有多个载荷工况的线性扰动分析
- 多模型优化
