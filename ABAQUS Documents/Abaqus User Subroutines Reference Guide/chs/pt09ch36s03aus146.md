### 概述

本节描述如何修改与接触对定义中表面相关的属性。

### 考虑壳和膜厚度

默认情况下，除有限滑动、节点-表面公式外的所有接触公式都考虑基于单元表面的初始壳和膜厚度。有限滑动、节点-表面公式不考虑表面厚度。无论连接到表面节点的单元类型如何，基于节点的表面都没有厚度。在接触计算中考虑单元厚度通常是可取的，但如果不需要，您可以选择不考虑厚度。

| **输入文件用法：** | ``` [*CONTACT PAIR*](../key/key-link.md#usb-kws-hcontactpair), NO THICKNESS ``` |
| --- | --- |

| **Abaqus/CAE用法：** | 相互作用模块：相互作用编辑器：**滑动公式：小滑动**或**有限滑动**，**离散化方法**：**表面到表面**或**节点到表面**，打开**排除壳/膜单元厚度** |
| --- | --- |

#### 示例

考虑壳被两个刚性表面夹持的情况，如[图36.3.2-1](pt09ch36s03aus146.md#pinchedshellcontact)所示。

**图36.3.2-1** 被两个刚体夹持的壳。

![](../graphics/pinchedshellcontact-nls.png)

在此示例中，使用小滑动、节点-表面公式的接触对定义在壳的顶表面与顶部刚性表面之间，以及壳的底表面与底部刚性表面之间。虽然壳表面定义在壳参考位置，但接触相互作用考虑了壳的厚度并从参考表面偏移。使用惩罚约束施加方法（见["接触压力-闭合关系，" 第37.1.2节"](pt09ch37s01aus166.md)）来避免过度约束从节点。使用了以下输入：

```
[*SURFACE*](../key/key-link.md#usb-kws-msurface), NAME=TOP_RIG_SURF
TOP_RIG_ELS,
[*SURFACE*](../key/key-link.md#usb-kws-msurface), NAME=SHELL_TOP_SURF
SHELL_ELS,SPOS
[*SURFACE*](../key/key-link.md#usb-kws-msurface), NAME=SHELL_BOT_SURF
SHELL_ELS,SNEG
[*SURFACE*](../key/key-link.md#usb-kws-msurface), NAME=BOT_RIG_SURF
BOT_RIG_ELS,
[*CONTACT PAIR*](../key/key-link.md#usb-kws-hcontactpair), INTERACTION=INTER_AL, SMALL SLIDING
SHELL_TOP_SURF, TOP_RIG_SURF
SHELL_BOT_SURF, BOT_RIG_SURF
[*SURFACE INTERACTION*](../key/key-link.md#usb-kws-hsurfaceinteraction), NAME=INTER_AL
[*SURFACE BEHAVIOR*](../key/key-link.md#usb-kws-hsurfacebehavior), PENALTY
```

### 指定表面几何校正

使用有限元方法，曲线几何表面自然地被近似为一组相连单元面的面组。使用面表面几何而不是真实表面几何可能会显著导致接触对中的接触应力不准确，特别是当面与真实表面的差异大小相对于接触组件的变形不小的情况下。克服与接触相互作用中面表面相关的收敛和精度困难的方法在["Abaqus/Standard中的接触公式，" 第38.1.1节"](pt09ch38s01aus177.md)和["Abaqus/Standard中平滑接触表面，" 第38.1.3节"](pt09ch38s01aus179.md)中讨论。
