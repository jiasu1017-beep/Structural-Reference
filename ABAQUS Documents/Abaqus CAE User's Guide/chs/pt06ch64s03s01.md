# 64.3.1 改进网格的策略





如果您有复杂的装配，应该选择****网格**![](../graphics/images/arrow.gif)**验证****来验证网格质量，然后再提交作业进行分析。网格验证工具可以执行以下操作：
- 高亮显示不满足指定标准（如长宽比）的所选形状单元。
- 打印网格统计数据，如所选形状的单元总数、高亮显示的单元数以及所选标准的平均值和最差值。
- 高亮显示未通过Abaqus/Standard和Abaqus/Explicit中包含在输入文件处理器中的网格质量测试的单元。

如果网格验证工具表明您应该尝试提高网格质量，在转向网格编辑工具集之前，您应该首先尝试以下操作：
- 更改种子分布
- 添加或修改分区
- 更改网格技术

此外，您可以尝试在"零件"模块中修改零件，或者尝试使用虚拟拓扑工具集并重新生成网格。您应该将网格编辑工具集视为网格划分过程的最后一步，仅使用它对节点和单元进行微小调整。Abaqus/CAE会尝试在您更改网格时保留属性（如载荷和边界条件）。如果您修改零件，Abaqus/CAE会在您返回"网格"模块时删除网格；因此，您将丢失对网格所做的任何编辑。
