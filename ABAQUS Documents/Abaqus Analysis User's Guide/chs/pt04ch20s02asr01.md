# 20.2.1 aStudy.combine()





### 20.2.1 **_aStudy_.combine()**: 组合参数化研究的参数样本。

**产品:** Abaqus/Standard  Abaqus/Explicit

此命令用于组合参数化研究中采样的参数值。

##### **参考**

- ["参数化研究脚本," 第20.1.1节](pt04ch20s01aus108.md)

### **命令:**

**_aStudy_.combine** (*token*, *additional data*)

### **标记:**

CROSS

使用此标记从参数化研究中参数采样值创建"十字形"模式的设计。

MESH

使用此标记创建"网格"模式的设计，其中每个参数的所有采样值与参数化研究中每个其他参数的所有采样值组合。

PRINT

使用此标记打印为参数化研究创建的设计点。

TUPLE

使用此标记创建"元组"模式的设计，由参数化研究中参数采样值的元组组成。

##### **CROSS、MESH和TUPLE的附加数据:**

### **可选数据:**

**name**

设置 **name** 等于所定义的设计集的名称；此名称必须用引号括起来。如果未指定名称，则会创建默认的设计集名称。

##### **PRINT的附加数据:**

### **可选数据:**

**name**

设置 **name** 等于正在打印其信息的设计集的名称；此名称必须用引号括起来。如果未指定设计集名称，则会打印参数化研究中所有设计集的信息。
