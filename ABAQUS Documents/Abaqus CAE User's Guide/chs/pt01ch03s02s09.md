# 3.2.9 自定义颜色





**选择颜色**对话框允许您自定义Abaqus/CAE中许多对象的颜色。有关可更改对象的更多信息，请参阅以下章节：
- ["自定义视图三轴，" 第5.4节"](pt01ch05s04.md)
- ["选择背景颜色，" 第7.7节"](pt01ch07s07.md)
- ["选择整体元素和表面边缘颜色，" 第55.3.3节"](pt05ch55s03hlb03.md)
- ["为无结果的元素着色，" 第55.12.4节"](pt05ch55s12hlb04.md)
- ["自定义图例，" 第56.1节"](pt05ch56s01.md)
- ["自定义标题块，" 第56.2节"](pt05ch56s02.md)
- ["自定义状态块，" 第56.3节"](pt05ch56s03.md)

当前颜色显示在**选择颜色**对话框左侧、吸管工具下方。您可以使用**选择颜色**对话框中的方法更新显示的颜色。在您单击**确定**接受更改并关闭**选择颜色**对话框之前，颜色不会在其他地方更新。您可以选择以下颜色选择方法：

**调色板**

对话框底部以框形式显示24种常用颜色。单击颜色进行选择；您不能修改调色板以显示不同的颜色。

**吸管工具**

吸管工具位于对话框左侧。单击吸管工具时，光标会变为十字线。下次您在计算机屏幕上的任何位置单击鼠标按钮1时，Abaqus/CAE会选择光标位置处的颜色。

**注意：**如果将光标移到Abaqus/CAE应用程序窗口外部，光标会恢复正常形式，但您仍然可以通过单击鼠标按钮1选择颜色。

**色轮**

色轮和亮度控制位于**轮**选项卡中。黑点表示当前位置选择的颜色，无论使用何种方法选择。单击轮上的任意位置选择新颜色。移动垂直滑块更改亮度；当您向下移动滑块时，Abaqus/CAE会向所选颜色添加黑色。

**RGB控件**

RGB（红、绿和蓝）控件位于**RGB**选项卡中。RGB设置与**选择颜色**对话框左侧显示的颜色匹配，无论使用何种方法选择颜色。您可以移动滑块或输入0到255之间的值来混合三种有色光并产生完整的色谱。0、0、0是黑色（无光）；255、255、255是白色（全强度、全色谱光）。

**HSV控件**

HSV（色调、饱和度和值）控件位于**HSV**选项卡中。HSV设置与**选择颜色**对话框左侧显示的颜色匹配，无论使用何种方法选择颜色。色调控制范围为0到360，更改设置对应于沿色轮边缘移动黑点（0和360都是红色）。饱和度控制范围为0到100，表示添加到背景颜色的所选颜色的量。值控制表示背景颜色；0是黑色，100是白色。

**CMY控件**

CMY（青、紫红和黄）控件位于**CMY**选项卡中。CMY设置与**选择颜色**对话框左侧显示的颜色匹配，无论使用何种方法选择颜色。您可以移动滑块或输入0到255之间的值来混合三种色调并产生完整的色谱。CMY控件的工作方式类似于添加色调到 paint；0、0、0是白色（无色调），255、255、255是黑色（全部色调）。

**颜色列表**

颜色列表位于**列表**选项卡中。您可以从数百种颜色中进行选择，包括灰色阴影。颜色列表为您提供比调色板更广泛的颜色范围，但不为您提供完整的色谱。

**自定义对象颜色：**

1. 打开包含要更改的对象设置的对话框。
2. 单击要自定义的对象的颜色样本![](../graphics/usi-colorsample.png)。Abaqus/CAE会显示**选择颜色**对话框。
3. 使用对话框中的方法之一选择新颜色。所选颜色的预览会出现在**选择颜色**对话框左侧、吸管工具下方。
4. 单击**确定**接受更改并关闭**选择颜色**对话框。Abaqus/CAE会返回原始对话框，并更新颜色样本以显示您选择的颜色。

当您在原始对话框中单击**确定**或**应用**时，Abaqus/CAE会在视口中更新颜色。
