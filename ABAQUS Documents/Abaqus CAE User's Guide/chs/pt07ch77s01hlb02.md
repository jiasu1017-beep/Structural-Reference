# 77.1.2 可视化模块中的颜色编码






可视化模块中的颜色编码过程与其他 Abaqus/CAE 模块略有不同。您可以使用 **通用绘图选项** 或 **叠加绘图选项** 对话框中的 **颜色和样式** 选项来控制模型的整体边缘和填充颜色。使用这些选项，您可以为所有单元和表面边缘选择单一颜色，并为所有单元面和表面选择单独的颜色。您选择的颜色统一应用于整个模型。

您也可以使用 **颜色编码** 选项来控制各个单元和零件实例的颜色。**颜色编码** 对话框允许您为各个项目选择单独的颜色。您必须使用 **颜色编码** 选项来执行任何复杂的非统一颜色方案。

默认情况下，各个项目颜色优先于整体通用或叠加[边缘颜色](pt05ch55s03hlb03.md)和[填充颜色](pt05ch55s02s01.md)。您可以使用 **通用绘图选项** 或 **叠加绘图选项** 对话框中的 **颜色和样式** 选项来更改此行为，以指定各个项目颜色或整体项目颜色应优先。（各个项目颜色不适用于等值线图。）

可视化模块提供比其他模块更少的颜色映射子集。当输出数据库显示在当前视口中时，可用于颜色编码的颜色映射包括零件实例、单元、节点、约束、材料、截面、显示组、平均区域、内部集合、铺层和层；当模型数据库中的模型显示在当前视口中时，唯一可用的颜色映射是按零件实例。但是，在自定义当前视口中各个对象的颜色时，您可以控制边缘和填充颜色，并且您可以选择将颜色编码应用于整个模型或仅应用于其节点或单元。此外，可视化模块中的设置取决于其他选项；当您在 **颜色编码** 对话框中指定单个项目颜色时，Abaqus/CAE 根据当前绘图的两个特征应用颜色：

**颜色优先级设置**

如果在 **通用绘图选项** 或 **叠加绘图选项** 对话框的 **颜色和样式** 页面上 **允许颜色编码选择覆盖此对话框中的选项** 被切换打开，则 Abaqus/CAE 会应用颜色。

**渲染样式**

在线框和隐藏渲染样式中，Abaqus/CAE 仅显示单元边缘。如果当前绘图使用线框或隐藏渲染样式，Abaqus/CAE 将各个项目颜色应用于这些边缘。

在填充和着色渲染样式中，Abaqus/CAE 同时显示单元面和单元边缘。如果当前绘图使用填充或着色渲染样式，Abaqus/CAE 将各个项目填充颜色应用于单元面，并将各个项目边缘颜色应用于单元边缘。

在填充和着色渲染样式中，线型单元（如梁）被当作线条处理，就好像它们是面一样。在填充和着色渲染样式中，Abaqus/CAE 将各个项目填充颜色应用于表示此类单元的线条。
