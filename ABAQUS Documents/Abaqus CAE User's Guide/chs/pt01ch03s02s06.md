# 3.2.6 使用以选项卡分隔的对话框





为了组织和方便起见，某些对话框以选项卡分隔。一次只能看到一个对话框。要查看特定的对话框，请单击其标签选项卡。

例如，[图3-2](pt01ch03s02s06.md#int-foldgen)显示了**通用绘图选项**对话框。

**图3-2** 以选项卡分隔的对话框。

![](../graphics/int-foldgen-nls.png)

如果单击**颜色和样式**选项卡，包含颜色和边缘属性选项的对话框会向前移动，遮挡其他四个对话框，如[图3-3](pt01ch03s02s06.md#int-foldtriad)所示。

**图3-3** 使用选项卡显示特定对话框。

![](../graphics/int-foldtriad-nls.png)

此外，分隔的对话框可以存在于单个对话框中。在这种情况下，分隔对话框的选项卡垂直排列，但工作方式与水平排列的选项卡相同。在[图3-4](pt01ch03s02s06.md#int-foldsub)中，**其他**对话框包含两个以选项卡分隔的对话框：**缩放**和**半透明度**。

**图3-4** 包含附加对话框的对话框。

![](../graphics/int-foldsub-nls.png)

对话框中的操作按钮适用于整组对话框，而不仅适用于您当前查看的对话框。如果单击**取消**，您在该组对话框中做出的所有未应用更改都会被取消，而不仅是当前对话框中的更改。同样，单击**确定**会保存您在任何对话框中所做的所有更改。
