# 7.5 选择透明度模式





透明度模式控制 Abaqus/CAE 在视口中显示半透明对象的速度和准确性。更改透明度模式的效果在你使用视图操作工具操作模型显示时最为明显。**快速**设置提供最佳图形性能，而**准确**设置提供最佳半透明对象渲染。

在大多数情况下，你不应该需要更改默认透明度方法设置。当你启动会话时，Abaqus/CAE 会检测系统上安装的图形硬件并选择适当的透明度模式。但是，如果你使用的是功能较弱的图形适配器，Abaqus/CAE 默认选择的设置可能不是你的模型的最佳选择。

**要控制透明度模式：**

1. 找到透明度模式选项。从主菜单栏中，选择****View**![](../graphics/images/arrow.gif)**Graphics Options****。Abaqus/CAE 显示**图形选项**对话框。
2. 将**透明度模式**滑块拖动到你想要的设置。对于某些图形卡，Abaqus/CAE 在**快速**和**准确**之间支持三个中间透明度模式设置；对于其他图形卡，Abaqus/CAE 在这两个选项之间仅支持一个中间设置。
3. 单击**确定**实施更改并关闭对话框。你的更改会在会话期间保存。
