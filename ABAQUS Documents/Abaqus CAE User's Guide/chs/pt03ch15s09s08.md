# 15.9.8 连接器截面分配编辑器





要创建连接器截面分配，从主菜单栏选择****Connector**![](../graphics/images/arrow.gif)**Assignment**![](../graphics/images/arrow.gif)**Create****。选择要分配连接器截面的线。

连接器截面分配编辑器包含三个选项卡页面，允许你指定要分配给线的连接器截面以及线端点的方向。例如，[图15-21](pt03ch15s09s08.md#itn-connectoreditor)显示了将连接器截面`Shock_Absorber`分配给所选线的连接器截面分配编辑器。

**图15-21** 连接器截面分配编辑器。

![](../graphics/itn-connectoreditor-nls.png)

你可以通过从主菜单栏选择****Help**![](../graphics/images/arrow.gif)**On Context****然后点击感兴趣的编辑器功能来显示有关特定编辑器功能的信息。有关创建连接器截面分配的详细说明，请参见["创建和修改连接器截面分配，"第15.12.12节](pt03ch15s12hlb11.md)。
