# *INCIDENT WAVE FLUID PROPERTY







### *INCIDENT WAVE FLUID PROPERTYDefine the fluid properties associated with an incident wave.

The preferred interface for defining the fluid properties for an incident wave is the [*INCIDENT WAVE INTERACTION PROPERTY](ch09abk11.md) option used in conjunction with the [*INCIDENT WAVE INTERACTION](ch09abk10.md) option. The alternative interface uses the [*INCIDENT WAVE FLUID PROPERTY](ch09abk09.md) option to define the fluid properties used to define an incident wave.

The [*INCIDENT WAVE FLUID PROPERTY](ch09abk09.md) must be used in conjunction with the [*INCIDENT WAVE PROPERTY](ch09abk12.md) option.

**Products: **Abaqus/Standard  Abaqus/Explicit  Abaqus/CAE  

**Type: **Model data  

**Level: **Model  

**Abaqus/CAE: **Unsupported; this option has been superseded by incident wave interaction properties.
