# 19.3.5 自适应过程管理器





**自适应过程管理器**与 Abaqus/CAE 中的其他管理器类似，允许您执行以下操作：
- 创建自适应过程并将过程与所选模型关联。
- 编辑所选自适应过程。
- 复制、重命名或删除所选自适应过程。

此外，**自适应过程管理器**允许您执行以下操作：- 在提交自适应过程之前对模型执行[数据检查](pt03ch19s09hlb02.md)。
- [提交](pt03ch19s09hlb03.md)自适应过程。
- 执行数据检查后[继续](pt03ch19s09hlb04.md)自适应过程。

您可以通过从主菜单栏选择 ****自适应**![](../graphics/images/arrow.gif)**管理器**** 来显示**自适应过程管理器**。[图 19-3](pt03ch19s03s05.md#ana-adaptivityman) 显示了**自适应过程管理器**的布局。

**图 19-3** **自适应过程管理器**。

![](../graphics/ana-adaptivityman-nls.png)

**自适应过程管理器**的三列显示以下内容：

**名称**

**名称**列显示自适应过程的名称。单击**重命名**可以重命名所选过程。

**模型**

**模型**列显示与自适应过程关联的模型的名称。

**状态**

**状态**列显示自适应过程的当前状态，并在过程运行时持续更新。状态可以是以下之一：

**无**

过程尚未提交进行分析。

**已提交检查**

模型正在提交进行数据检查。

**正在检查**

模型的数据检查正在运行。

**检查完成**

模型的数据检查已成功完成；您现在可以继续自适应过程。

**已提交**

过程正在提交执行。

**正在运行**

过程已提交进行分析并且正在运行。

**已完成**

过程已运行完成。

**中止**

由于模型中的致命错误或磁盘空间不足等问题，过程已中止。

**已终止**

过程已被用户终止。
