# 70.7.6 使用草图平面分割法分割单元体







您可以通过使用Sketcher绘制平面分割的几何形状来分割选定的单元体。当您想要创建的分割形状无法使用其他分割工具实现时，可以使用Sketcher。草图分割对于创建将用于断裂力学分析的裂纹面非常有用。

您必须在平面上进行草图绘制。在大多数情况下，您需要在与选定单元体相交的数据平面上绘制分割。下图展示了一个在数据平面上的草图平面分割：

![](../graphics/par-cell-sketch-nls.png)

**使用草图平面分割法分割单元体的步骤：**

1. 从主菜单栏，选择****Tools**![](../graphics/images/arrow.gif)**Partition****。将出现"创建分割"对话框。Abaqus/CAE在提示区域显示提示以指导您完成该过程。**提示：**您还可以使用位于模块工具箱中分割单元体工具旁的![](../graphics/ico_ptnCellSketch.png)工具来使用草图平面分割法分割单元体。有关工具箱中分割工具的图示，请参见["使用分割工具集，"第70.2节](pt06ch70s02.md)。
2. 从对话框顶部的**Type**单选按钮中，选择**Cell**。**Method**列表将显示可用于分割单元体的方法。
3. 从方法列表中，选择**Sketch planar partition**。如果部件或装配体只包含一个单元体，Abaqus/CAE将进入Sketcher。
4. 如果部件或装配体包含多个单元体，选择要分割的单元体并单击鼠标按钮2。**提示：**如果您无法选择所需的单元体，可以使用**Selection**工具栏更改选择行为。有关更多信息，请参见["使用选择选项，"第6.3节](pt01ch06s03.md)。
5. 选择要绘制草图的平面。如果不存在合适的面，您可以选择数据平面。选中的面将在视口中高亮显示。
6. 选择一条边以及该边在Sketcher网格上的方向。该边不能垂直于草图平面。默认情况下，所选边将垂直显示在Sketcher网格的右侧。要选择不同的边方向，请单击对话框右侧的箭头并从出现的列表中选择方向。**提示：**如果部件或装配体没有能提供所需方向的边，您可以创建一条数据轴。然后您可以选择该数据轴来控制部件在Sketcher网格上的方向。- Abaqus/CAE高亮显示所选边，进入Sketcher，并旋转部件直到所选平面或数据平面与Sketcher网格平面重合，且所选边以所需方向与网格对齐。Abaqus/CAE高亮显示所选面。
7. 使用Sketcher绘制平面分割。从主菜单栏单击**Done**以表示您已完成分割的绘制。
8. 在提示区域中，单击**Create Partition**。Abaqus/CAE创建分割。
