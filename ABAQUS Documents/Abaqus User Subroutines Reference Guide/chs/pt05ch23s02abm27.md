### 概述

双层黏塑性模型：
- 旨在用于建模观察到显著时间相关行为以及塑性的材料，这对金属通常发生在高温下；
- 由与弹性-黏性网络并联的弹性-塑性网络组成（与耦合蠕变和塑性能力相反，其中塑性和黏性网络是串联的）；
- 基于弹性-塑性网络中的Mises或Hill屈服条件和弹性-黏性网络中Abaqus/Standard中任何可用的蠕变模型（双曲蠕变定律除外）；
- 假定为偏量非弹性响应（因此，不能使用与压力相关的塑性或蠕变模型来定义两个网络的行为）；
- 旨在用于在各种温度下波动载荷下的材料响应建模；和
- 已被证明在热机械加载中提供良好的结果。

### 材料行为

材料行为分解为三个部分：弹性、塑性和黏性。图23.2.11-1显示了该材料模型的一维理想化，弹性-塑性和弹性-黏性网络并联。以下小节详细描述了弹性和非弹性（塑性和黏性）行为。

#### 弹性行为

两个网络响应的弹性部分使用线性各向同性弹性定义指定。Abaqus/Standard中任何可用的弹性模型可用于定义网络的弹性行为。参考一维理想化（图23.2.11-1），弹性-黏性网络的弹性模量（E~V~）与总（瞬时）模量（E）之比由下式给出：

f = E~V~/E

用户指定的比值f（作为黏性行为定义的一部分给出，如下所述）将指定用于弹性行为的总模量分配给弹性-黏性和弹性-塑性网络。因此，如果定义了各向同性弹性属性，两个网络中的泊松比相同。另一方面，如果定义了各向异性弹性，相同的各向异性类型适用于两个网络。假定为弹性行为指定的属性是瞬时属性（E）。

##### 时间相关行为

在"time hardening"幂律模型中，可以使用总时间或蠕变时间。总时间是所有常规分析步骤累积的时间。蠕变时间是具有时间相关材料行为的过程的时间之和。如果使用总时间，建议对于分析中蠕变未激活的任何步骤使用相对于蠕变时间较小的步骤时间；这是必要的，以避免在后续步骤中硬化行为的变化。

#### 热膨胀

可以通过提供材料的热膨胀系数来建模热膨胀（["热膨胀，" 第26.1.2节"](pt05ch26s01abm52.md)）。可以以常规方式定义各向异性膨胀。在一维理想化中，膨胀元素假定与网络其余部分串联。

### 材料参数校准

校准过程最好在材料模型的一维理想化背景下解释。在以下讨论中，假定黏性行为由Norton-Hoff率定律控制，由下式给出：

ε¯˙^v^ = A (σ¯)^n

在上述表达式中，下标V表示仅弹性-黏性网络中的量。这种形式的率定律可以例如通过为黏性行为选择时间-硬化幂律并设置C = 1来选择。对于这个基本案例，有六个材料参数需要校准（图23.2.11-1）。这些是两个网络的弹性属性E~V~和E；初始屈服应力σ~0~；硬化H；和Norton-Hoff率参数A和n。

需要进行的实验是在不同恒定应变率下的单轴拉伸。静态（有效零应变率）单轴拉伸测试确定了长期模量E~L~；初始屈服应力σ~0~；和硬化H。出于说明目的，假定硬化是线性的。材料模型不限于线性硬化，可以为塑性模型定义任何一般硬化行为。瞬时弹性模量E可以通过在非零、相对较高的应变率下测量材料的初始弹性响应来测量。可以在不同施加应变率下进行几次这样的测量，直到瞬时模量不随施加应变率的变化而变化。K和E~L~之间的差异决定了E~V~。

为了校准参数A和n，认识到在恒定施加应变率ε¯˙^0^下，弹性-黏性网络的长期（稳态）行为是大小为σ^ss^ = ε¯˙^0^ E~L~ K/(K - E~L~)的恒定应力是有用的。在假定硬化模量与弹性模量相比可以忽略不计（E~L~ << K）的假设下，整体材料的稳态响应由下式给出：

σ^ss^ = E~L~ ε¯˙^0^ + K E~L~/(K - E~L~) (A (K E~L~/(K - E~L~))^n)^(1/(n+1))

其中σ是在给定总应变ε下的总应力。为了确定是否已达到稳态，可以绘制量σ^ss^ - E~L~ ε作为ε的函数，并注意它何时变为恒定。σ^ss^ - E~L~ ε的恒定值等于K E~L~/(K - E~L~) (A (K E~L~/(K - E~L~))^n)^(1/(n+1))。通过在不同恒定施加应变率ε¯˙^0^值下执行几个测试，可以确定常数A和n。

### 不同分析步骤中的材料响应

材料在所有应力/位移过程类型期间都是活跃的。在请求长期响应的静态分析步骤中（见["静态应力分析，" 第6.2.2节"](pt03ch06s02at01.md)），只有弹性-塑性网络是活跃的；弹性-黏性网络不会以任何方式贡献。特别地，在长期静态响应期间，黏性网络中的应力为零。如果在耦合温度-位移过程或土壤固结过程中移除了蠕变效应，则假定弹性-黏性网络的响应仅为弹性的。在线性扰动步骤期间，仅考虑网络的弹性响应。

某些应力/位移过程类型（耦合温度-位移、土壤固结和准静态）允许用户通过用户指定的错误容忍度控制黏性本构方程的时间积分准确性。在其他当前没有此类直接控制的过程类型（静态、动态）中，您必须选择适当的时间增量。这些时间增量必须相对于材料的典型松弛时间较小。

### 单元

双层黏塑性模型不适用于一维单元（梁和桁架）。它可用于Abaqus/Standard中包含力学行为（具有位移自由度的单元）的任何其他单元。

### 输出

除了Abaqus/Standard中可用的标准输出标识符（["Abaqus/Standard输出变量标识符，" 第4.2.1节"](pt02ch04s02abv01.md)），以下变量对双层黏塑性材料模型具有特殊含义：

| EE | 弹性应变定义为：ε^e^ = σ/E。 |
| --- | --- |

| PE | 弹性-塑性网络中的塑性应变，ε^p^。 |
| --- | --- |

| VE | 弹性-黏性网络中的黏性应变，ε^v^。 |
| --- | --- |

| PS | 弹性-塑性网络中的应力，σ^p^。 |
| --- | --- |

| VS | 弹性-黏性网络中的应力，σ^v^。 |
| --- | --- |

| PEEQ | 等效塑性应变，定义为ε¯^p^ = √(2/3 ε~ij~^p^ ε~ij~^p^)。 |
| --- | --- |

| VEEQ | 等效黏性应变，定义为ε¯^v^ = √(2/3 ε~ij~^v^ ε~ij~^v^)。 |
| --- | --- |

| SENER | 单位体积的弹性应变能密度，定义为U^e^ = 1/2 σ^p^ ε^e^ + 1/2 σ^v^ ε^v^。 |
| --- | --- |

| PENER | 单位体积的塑性耗散能量，定义为U^p^ = ∫ σ^p^ dε^p^。 |
| --- | --- |

| VENER | 单位体积的黏性耗散能量，定义为U^v^ = ∫ σ^v^ dε^v^。 |
| --- | --- |

上述应变张量的定义意味着总应变通过以下关系与弹性、塑性和黏性应变相关：

ε = ε^e^ + ε^p^ + ε^v^

其中根据上述定义 ε^e^ = σ/E 且 ε^p^ = ε^p^(σ^p^)。上述输出变量的定义适用于所有过程类型。特别地，当为静态过程请求长期响应时，弹性-黏性网络不承受任何应力，弹性应变的定义简化为 ε^e^ = σ/E，这意味着总应力通过瞬时弹性模量与弹性应变相关。
