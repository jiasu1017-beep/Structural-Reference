# *ELCOPY









### *ELCOPY通过从现有单元集复制来创建单元。

此选项用于复制单元集以创建新单元。

**产品：**Abaqus/Standard  Abaqus/Explicit  Abaqus/CFD  Abaqus/CAE

**类型：**模型数据

**级别：**部件，部件实例

**Abaqus/CAE：**不适用；复制草图的部分和部件实例化具有类似目的。

##### **参考：**

- ["单元定义，" Abaqus分析用户指南第2.2.1节](../usb/usb-link.md#usb-int-ielement)

### **必需参数：**

ELEMENT SHIFT

将此参数设置为将添加到现有单元编号的整数，以定义正在创建的单元的单元编号。

OLD SET

将此参数设置为要复制的单元集的名称。被复制的单元是遇到此选项时属于该集合的单元。

SHIFT NODES

将此参数设置为将添加到现有单元节点编号的整数，以定义正在创建的单元的节点编号。

### **可选参数：**

NEW SET

将此参数设置为将分配由此操作创建的单元的单元集的名称。如果省略此参数，新创建的单元将不会被分配到单元集。

REFLECT

包含此参数以修改正在创建的单元上的节点编号顺序，这在某些情况下是必要的，以避免创建违反Abaqus逆时针单元编号约定的单元。此参数只能与连续体单元一起使用，并且通常仅在使用[*NCOPY](ch14abk01.md)选项生成节点时才需要。

**此选项没有关联的数据行。**
