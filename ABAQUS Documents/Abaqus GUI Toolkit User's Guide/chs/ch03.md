# 3 有限元和刚体







有限元和刚体是 Abaqus 模型的基本组件。有限元是可变形的，而刚体在空间中移动时不变形。虽然有限元分析程序的用户对有限元的含义有一定了解，但有限元程序中刚体的一般概念可能相对较新。

为了计算效率，Abaqus 具有通用刚体功能。任何物体或物体的任何部分都可以定义为刚体；大多数单元类型可以用于刚体定义（例外情况在["刚体定义，" Abaqus Analysis User's Guide 第 2.4.1 节](../usb/usb-link.md#usb-int-arigidoverview) 中列出）。刚体相对于变形体的优势在于，刚体的运动完全由参考节点上不超过六个自由度来描述。相比之下，变形单元有许多自由度，需要昂贵的单元计算来确定变形。当这种变形可以忽略或不需要时，将组件建模为刚体会产生显著的计算节省，而不影响整体结果。
