# 58.4 显示使用分析场的相互作用和规定条件的符号





当您将相互作用或规定条件应用于某个区域时，可以在视口中显示表示该相互作用或规定条件的符号。本节涵盖以下主题：
- ["表达式场显示符号详情，" 第58.4.1节](pt06ch58s04s01.md)
- ["映射场显示符号详情，" 第58.4.2节](pt06ch58s04s02.md)

默认情况下，使用分析场分布的相互作用和规定条件的符号会根据计算值进行缩放。对于除箭头以外的符号，在每个符号内部显示加号 (+) 或减号 ()，以指示在该位置相互作用或规定条件的幅值是正还是负。您可以使用**装配显示选项**对话框中的**属性**显示选项关闭符号缩放。有关更多信息，请参见["控制属性的显示，" 第76.15节](pt07ch76hla14.md)。

当分析场对某部分区域评估为零时，Abaqus/CAE 会显示缩小版的相互作用和规定条件符号。这些缩小版的符号明显小于默认符号大小。有关更多信息，请参见["了解规定条件符号类型、颜色和大小，" 第16.5.1节](pt03ch16s05hlb01.md)。
