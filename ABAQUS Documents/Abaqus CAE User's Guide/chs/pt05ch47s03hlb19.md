# 47.4.21 计算向量幅度







### 47.4.21 计算向量幅度

使用 **vectorMagnitude** 函数对最多三个[之前保存的](pt05ch47s02hlb07.md) *X–Y* 数据对象（每个对象是有序对的集合）执行平方和的平方根运算，生成一个新的 *X–Y* 数据对象。此函数本质上与 **srss** 函数相同（请参见["计算两个或多个X–Y数据对象的平方和的平方根，" 第47.4.19节](pt05ch47s03hlb17.md)），只不过该运算最多只能对三个数据对象执行。您可以通过计算三个向量分量平方和的平方根来绘制向量的总幅度。

新的 *X–Y* 数据对象以其 *X* 坐标作为第一个数据对象的所有 *X* 坐标以及对齐其余对象所需的任何其他 *X* 坐标。Abaqus/CAE 通过[插值和外推](pt05ch47s03s02.md)计算额外的 *X–Y* 数据对。新的数据对象以其 *Y* 坐标作为匹配 *X* 坐标处所有输入 *X–Y* 数据对象 *Y* 坐标的平方和的平方根。此函数的参数是可交换的。

[图47-18](pt05ch47s03hlb19.md#usv-xyplot-vect) 说明了可以使用 **vectorMagnitude** 函数生成的 *X–Y* 曲线图类型。

**图47-18** 使用 **vectorMagnitude** 函数生成的 *X–Y* 曲线图。

![](../graphics/usv-xyplot-vect.png)

**计算向量幅度：**

1. 找到 **Operate on XY Data** 对话框。从主菜单栏，选择 ****Tools**![](../graphics/images/arrow.gif)**XY Data**![](../graphics/images/arrow.gif)**Create****. 在出现的对话框中点击 **Operate on XY data**；然后点击 **Continue**。**Operate on XY Data** 对话框出现。
2. 从列出的 **Operators** 中，点击 **vectorMagnitude(X,X,X)**。**vectorMagnitude** 函数出现在表达式窗口中。
3. 从 **XY Data** 选项中，点击三个要操作的 *X–Y* 数据对象的名称，然后点击 **Add to Expression**。您可以从本次会话中之前保存的所有 *X–Y* 数据对象中选择（在 **XY Data** 字段中按字母顺序列出）。*X–Y* 数据对象名称以逗号分隔，出现在表达式窗口中 **vectorMagnitude** 函数的括号内。
4. 要继续构建表达式，将光标定位在表达式窗口中，然后输入或选择要包含的函数、运算符和 *X–Y* 数据。
5. 要评估并显示表达式，点击 **Plot Expression**。
6. 要[保存](pt05ch47s02hlb07.md)您的新 *X–Y* 数据对象，点击 **Save As**，然后在出现的对话框中提供名称。保存数据对象使其可用于本次会话中的未来操作以及包含多个数据对象的 *X–Y* 曲线图。
7. 完成后，点击 **Cancel** 关闭对话框。
