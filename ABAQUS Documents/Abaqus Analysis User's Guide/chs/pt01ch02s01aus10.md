# 2.1.6 调整节点坐标





**产品：**Abaqus/Standard  Abaqus/Explicit  Abaqus/CAE

##### **参考**

- [*ADJUST*](../key/key-link.md#usb-kws-madjust)
- [""定义调整点约束，"Abaqus/CAE用户指南第15.15.5节"](../usi/usi-link.md#usi-itn-helptopic-adjustpts)

### 概述

节点调整用于：
- 调整用户指定的节点坐标，使节点位于给定曲面上；和
- 指定节点移动的方向。

### 调整节点坐标

通常，用户指定的节点坐标在输入文件处理期间不会被修改。但是，在某些情况下，网格坐标仅以通用方式已知，并且为实际使用确定其坐标是不方便的。例如，在使用紧固件时，指定的参考节点应定位在关联曲面上的投影点。由于该位置可能仅大致已知，您可以使用节点调整自动将参考节点移动到该位置。有关节点调整功能的典型用法，请参阅["关于组装紧固件，"Abaqus/CAE用户指南第29.1.3节"](../usi/usi-link.md#usi-eng-fastener-overview-assm)。

使用此功能时，节点被调整到指定曲面上，不考虑壳厚度或壳偏移。因此，不建议将此功能用作校正接触或绑定约束的初始过盈量的方法。此外，在选择要调整的节点时应小心，因为此功能不尊重任何与调整节点与其他节点的相对位置相关的约束（例如，刚体定义）。

| **输入文件用法：** | 使用以下选项标识要移动的节点和要将节点移动到的曲面： |
| --- | --- |
|  | ``` [*ADJUST*](../key/key-link.md#usb-kws-madjust), NODE SET=*name*, SURFACE=*name* ``` |

| **Abaqus/CAE用法：** | 使用以下选项将耦合约束的控制点移动到耦合曲面上： |
| --- | --- |
|  | 相互作用模块：****约束****![](../graphics/images/arrow.gif)****创建****：**耦合**；**调整控制点位于曲面上** 使用以下选项将任意点或多个点移动到任意曲面上： 相互作用模块：****约束****![](../graphics/images/arrow.gif)****创建****：**调整点** |

#### 指定节点调整方向

可以使用法线调整或定向调整将节点移动到曲面上。默认情况下，节点沿曲面的法线调整到指定曲面上的最近点。您可以指定方向以沿给定方向而不是沿曲面法线将节点移动到曲面上。来自方向定义的局部*Z*方向向量用于将节点移动到曲面上（参见["方向，"第2.2.5节"](pt01ch02s02aus15.md)）。如果找不到投影，则节点坐标保持不变。

| **输入文件用法：** | ``` [*ADJUST*](../key/key-link.md#usb-kws-madjust), ORIENTATION=*name* ``` |
| --- | --- |

| **Abaqus/CAE用法：** | 在Abaqus/CAE中不支持方向投影选项。 |
| --- | --- |
