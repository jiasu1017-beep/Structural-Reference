### 概述

在为特定分析选择自适应重网格划分规则中使用的误差指标变量时，您应考虑：
- 误差指标变量的特征；
- 存在哪些场以及哪些是您感兴趣的；以及
- 载荷的性质。

### 误差指标特征

误差指标输出变量提供解精度的估计（参见 ["误差指标输出," 第 4.1.4 节](pt02ch04s01aus41.md)）。在自适应重网格划分的背景下，误差指标有助于确定应在何处细化或粗化网格以达到指定的精度目标（参见 ["自适应重网格划分：概述," 第 12.3.1 节](pt04ch12s03abo15.md) 和 ["基于解的网格划分," 第 12.3.3 节](pt04ch12s03aus85.md)）。本节讨论误差指标在如何适合影响各种分析类型的自适应重网格划分方面的额外特征。

### 存在哪些场以及哪些是您感兴趣的

某些变量自然地适用于某些类型的分析。例如，热通量指标（HFLERI）用于具有温度自由度的分析。在 Abaqus/CAE 的 **Remeshing Rule** 编辑器中选择误差指标变量时（参见 ["什么是重网格划分规则？," Abaqus/CAE 用户指南第 17.13.1 节](../usi/usi-link.md#usi-mgn-conc-adaptivity-rulerole)），您的选择将限制为所选过程类型可用的变量。

### 载荷的性质

某些误差指标变量仅指示当前分析时间——步骤中的特定增量——的离散误差。其他误差指标变量提供到当前分析时间为止的解历史记录。例如，如果您的模拟涉及非比例载荷或显著非线性响应，您通常会在使用记录解历史的误差指标变量时看到更好的自适应重网格划分结果。[表 12.3.2--1](pt04ch12s03aus84.md#usb-anl-aadperrorindicators-history) 列出了适用于自适应重网格划分的误差指标变量，并指示它们是否记录解历史。

**表 12.3.2–1** 记录解历史的适用于自适应重网格划分的误差指标变量。
| 解量 | 误差指标变量 (![](../graphics/usb_eqn00290.gif)) | 记录解历史？ |
| --- | --- | --- |
| 单元能量密度 | ENDENERI | 是 |
| Mises 应力 | MISESERI | 否 |
| 等效塑性应变 | PEEQERI | 是 |
| 塑性应变 | PEERI | 否 |
| 蠕变应变 | CEERI | 否 |
| 热通量 | HFLERI | 否 |
| 电通量 | EFLERI | 否 |
| 电势梯度 | EPGERI | 否 |

默认情况下，当您创建重网格划分规则时，误差指标是为分析的最后一步的最后增量指定的，自适应重网格划分基于此最后增量中的误差指标。当您选择记录解历史的误差指标时，此默认误差指标指定适用于几乎所有分析。但是，对于不记录解历史的其他误差指标变量，您可能认为合适（例如，在具有非比例载荷的多步骤情况下）为同一区域定义多个重网格划分规则，每个规则应用于不同的步骤。

以下示例提供了典型案例的简单说明，并显示了误差指标输出变量的适当选择。

#### 线性响应示例

[图 12.3.2--1](pt04ch12s03aus84.md#aadaptivity-proportional-example) 显示了最简单的载荷情况，其中载荷与步长时间成比例，模型的响应是线性的。在这种情况下，最后增量处的解将与任何其他增量处的解成比例。因此，基于任何误差指标变量选择，将重网格划分基于最后增量中的误差指标值是合适的。

**图 12.3.2–1** 比例加载、线性响应示例：悬臂梁的小挠度。

![](../graphics/aadaptivity-proportional-example.png)

#### 单调响应示例

[图 12.3.2--2](pt04ch12s03aus84.md#aadaptivity-nlgeom-example) 显示了更一般的情况，其中模型具有非线性响应——在这种情况下由几何非线性引起——并且载荷是单调的，但通常与步长时间不成比例。

**图 12.3.2–2** 单调响应示例：悬臂梁的大挠度。

![](../graphics/aadaptivity-nlgeom-example.png)

模型的响应稍微更一般，因为特定增量处的解与最后增量处的解不成比例。但是，最后增量处的误差指标输出值仍然反映模型对载荷历史的极端响应。因此，基于任何误差指标变量选择，将重网格划分基于最后增量中的误差指标值是合适的。

#### 一般响应示例

[图 12.3.2--3](pt04ch12s03aus84.md#aadaptivity-general-example) 显示了在分析过程中载荷特征发生显著变化的情况。

**图 12.3.2–3** 一般响应示例：承受刚性压头的块体。

![](../graphics/aadaptivity-general-example.png)

在这种情况下，您的误差指标选择将取决于材料模型。单元能量密度误差指标 ENDENERI 将考虑载荷历史的复杂性（并产生在整个分析过程中提供准确解的自适应网格），无论材料类型如何。如果发生塑性变形，您也可以选择使用等效塑性应变 PEEQERI 或塑性应变 PEERI 误差指标。塑性应变和塑性应变误差指标通常不能捕获历史影响；例如，它们不考虑对称应变逆转模型中的峰值应变。然而，此示例不涉及应变逆转；因此，PEERI 将是有效的误差指标选择。

#### 一般多步骤响应示例：模具成形和回弹

[图 12.3.2--4](pt04ch12s03aus84.md#aadaptivity-multistep-example) 显示了一般响应的进一步概括。这里，模拟了成形操作，不同阶段使用不同的步骤。

**图 12.3.2–4** 一般多步骤响应示例。

![](../graphics/aadaptivity-multistep-example-nls.png)

在这种情况下，模型的响应因步骤而异。您通常希望误差指标充分捕获模型对载荷历史的极端响应。但是，您不知道任何特定增量是否捕获了此极端。因此，您应该选择记录解历史的误差指标变量。
