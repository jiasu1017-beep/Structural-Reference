# 23.5 从复合铺层请求输出






当您创建将在分析过程中集成的复合铺层时，可以指定铺层中每层的积分点数量。默认情况下，对于在分析期间集成的壳截面，Abaqus/CAE 为壳或连续壳复合铺层的每层创建三个积分点，为实体复合铺层的每层创建一个积分点。对于预集成壳截面，Abaqus/CAE 为铺层中的每层创建三个积分点，并且您无法更改此值。[图 23-8](pt04ch23s05.md#adv-layups-sectionpoints) 显示了具有三层且每层三个积分点的复合铺层的积分点编号。

**图 23-8** 具有三层的复合铺层的积分点编号。

![](../graphics/adv-layups-sectionpoints-nls.png)

如果您未创建输出请求，Abaqus 仅从复合铺层的顶部和底部（最高和最低积分点）写入场输出数据，并且不会从其他层生成数据。因此，如果您的模型包含复合铺层，并且您需要来自各层或内部积分点的数据，则必须创建新的输出请求或编辑默认输出请求，并指定将输出变量的复合铺层。

当您创建输出请求时，Abaqus/CAE 默认仅从复合铺层每层的中间截面点向输出数据库写入场和历史数据。要更改默认行为，您可以使用场和历史输出请求编辑器来编辑输出请求，并将域更改为复合铺层。然后，您可以从每层的顶部、中部和/或底部截面点、所有截面点或指定的截面点请求场或历史数据。有关更多信息，请参见["修改场输出请求，" 第 14.12.2 节](pt03ch14s12hlb02.md)，和["修改历史输出请求，" 第 14.12.3 节](pt03ch14s12hlb03.md)。您可以请求以下内容：

**选定**

Abaqus/CAE 从每层选定的截面点（顶部、中部和/或底部）向输出数据库写入场数据。如果一层具有偶数个截面点，并且您请求从中间截面点输出，Abaqus/CAE 会从跨越该层中间的两个截面点中较高的一个生成数据。例如，如果一层有四个截面点，并且您请求从中间截面点输出，Abaqus/CAE 会从第三个截面点生成数据。

**全部**

Abaqus/CAE 从所有层的所有截面点向输出数据库写入场数据。

**指定**

Abaqus/CAE 从指定的截面点向输出数据库写入场数据。截面点从底层底部到顶层顶部按顺序编号，其中底层是铺层中的第一层。例如，如果您需要从[图 23-8](pt04ch23s05.md#adv-layups-sectionpoints) 中显示的每层的中间截面点输出，则应输入 `2,5,8`。

有关更多信息，请参见["理解输出请求，" 第 14.4 节](pt03ch14s04.md)。
