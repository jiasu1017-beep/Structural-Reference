# 3.4.9 使用管理器对话框管理对象





Abaqus/CAE为您提供一组管理器，列出当前模型或会话中定义的所有对象，如零件、独立草图、材料、截面、步骤、显示组和X-Y数据对象。此外，**模型管理器**列出当前模型数据库中定义的所有模型。

**注意：**有关特定管理器的详细信息及其位置，请参阅您感兴趣的特定模块的文档。

使用管理器对话框中的按钮管理对象列表。

**管理对象：**

1. 要显示管理器，请执行以下操作之一：
   - 要显示与管理器关联的模块，请从主菜单栏上的相应菜单中选择**管理器**。例如，要在属性模块中工作时显示**截面管理器**，请从主菜单栏选择****截面**![](../graphics/images/arrow.gif)**管理器****。
   - 要显示与工具集关联的管理器，请从主菜单栏选择****工具**![](../graphics/images/arrow.gif)***工具集***![](../graphics/images/arrow.gif)**管理器****。例如，要显示**集合管理器**，请从主菜单栏选择****工具**![](../graphics/images/arrow.gif)**Set**![](../graphics/images/arrow.gif)**管理器****。
   - 要显示与模型树中对象关联的步骤相关管理器，请在该对象上单击鼠标按钮3，然后从出现的菜单中选择**管理器**。
   - 要显示**模型管理器**，请从主菜单栏选择****模型**![](../graphics/images/arrow.gif)**管理器****。
   管理器会出现并显示当前模型或会话中的对象列表。该列表包含每个对象的名称，在某些情况下还包含有关每个对象的信息。例如，**零件管理器**列出每个零件的名称、其状态、其类型以及创建它的建模空间。
2. 要管理现有对象，请从管理器中的列表中选择感兴趣的对象或多个对象，然后单击相应的按钮。（例如，要删除对象，请从列表中选择该对象的名称，然后单击**删除**。）在大多数情况下，会出现一个对话框；例如，当您单击**重命名**时，对话框会询问所选对象的新名称。
3. 如果出现对话框，请提供请求的信息并单击**确定**。
4. 单击**关闭**关闭管理器。

**提示：**您也可以使用主菜单栏中的菜单来管理对象。有关更多信息，请参阅["使用管理器菜单管理对象，" 第3.4.10节"](pt01ch03s04hlb02.md)。
