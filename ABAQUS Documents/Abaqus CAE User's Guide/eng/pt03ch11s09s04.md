# 11.9.4 Cut features







A cut is a feature that removes material from a part. A cut can be a circular hole, or it can be any arbitrary shape. The sketched profile of any cut must be closed. In many cases the entire profile will affect the shape of the cut feature, even if it does not initially contact the surface being cut. To create a cut feature, select ****Shape**![](../graphics/images/arrow.gif)**Cut**** from the main menu bar or select one of the cut tools in the Part module toolbox.

**Note:**Most of the figures do not show a closed cut profile where it intersects with the part surface. These lines have been removed to show the shape of the cut feature.

Once you have sketched the initial profile, you perform one of the following operations to create a cut feature:
- To create an extruded cut, you extrude the profile through a specified distance (*d*), as shown in [Figure 11--32](pt03ch11s09s04.md#prt-extcut). **Figure 11--32** An extruded cut feature. ![](../graphics/prt-extcut-nls.png) In addition, you can apply either draft or twist to the extruded cut, as shown in [Figure 11--33](pt03ch11s09s04.md#prt-extcut-draft). You define the draft angle for an extruded cut with draft or the center of twist and the pitch (the extrusion distance in which a 360 twist occurs) for an extruded cut with twist. Select ****Shape**![](../graphics/images/arrow.gif)**Cut**![](../graphics/images/arrow.gif)**Extrude**** from the main menu bar to create this type of feature. **Figure 11--33** Extruded cut features with draft and twist. ![](../graphics/prt-extcut-draft-nls.png)
- To create a cut loft feature, you transition the shape from the initial loft section to an end section of a different shape or orientation, as shown in [Figure 11--34](pt03ch11s09s04.md#prt-cut-loft). Abaqus/CAE determines the shape between the start and end sections using tangency constraints, intermediate sections, and path curves. Select ****Shape**![](../graphics/images/arrow.gif)**Cut**![](../graphics/images/arrow.gif)**Loft**** from the main menu bar to create this type of feature. **Figure 11--34** A cut loft feature. ![](../graphics/prt-loft-cut.png)
- To create a revolved cut, you revolve the profile through a specified angle (*α*). A construction line serves as the axis of revolution. In addition, you can enter a pitch value to translate the profile along the axis of revolution as it is revolved and to create part details such as screw threads. [Figure 11--35](pt03ch11s09s04.md#prt-revcut) shows a revolved cut and a revolved cut with pitch. Select ****Shape**![](../graphics/images/arrow.gif)**Cut**![](../graphics/images/arrow.gif)**Revolve**** from the main menu bar to create this type of feature. **Figure 11--35** A revolved cut and a revolved cut with pitch. ![](../graphics/prt-revcut-nls.png)
- To create a swept cut, you sweep the profile along a specified path, as shown in [Figure 11--36](pt03ch11s09s04.md#prt-sweepcut). Select ****Shape**![](../graphics/images/arrow.gif)**Cut**![](../graphics/images/arrow.gif)**Sweep**** from the main menu bar to create this type of feature. For more information, see ["Defining the sweep path and the sweep profile," Section 11.13.8](pt03ch11s13s08.md). **Figure 11--36** A swept cut feature. ![](../graphics/prt-sweepcut-nls.png)
- To create a circular hole, you enter the diameter of a hole and the distance of its center from two selected edges, as shown in [Figure 11--37](pt03ch11s09s04.md#prt-holecut). Select ****Shape**![](../graphics/images/arrow.gif)**Cut**![](../graphics/images/arrow.gif)**Circular Hole**** from the main menu bar to create this type of feature. **Figure 11--37** A circular hole feature. ![](../graphics/prt-holecut-nls.png)

When you are sketching the profile of an extruded, revolved, or swept cut, you can sketch multiple profiles in a single sketch. Abaqus/CAE extrudes each of the profiles when you exit the Sketcher and creates a cut corresponding to each profile as shown in [Figure 11--38](pt03ch11s09s04.md#prt-cut-multiple). 

**Figure 11–38** Multiple profiles extruded from a single sketch.

![](../graphics/prt-cut-multiple.png)

The sequence of cuts is stored as a single feature, and you can edit it only as a single feature. For example, if you change the extrusion depth, the depth will change for all the cuts in the feature.

You can use the cut tools to add a cut feature to any deformable or discrete rigid part. You cannot add a cut feature to an analytical rigid part; you can only modify the original sketch that defined that part.
