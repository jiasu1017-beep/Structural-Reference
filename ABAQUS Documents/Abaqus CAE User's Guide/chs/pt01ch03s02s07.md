# 3.2.7 输入表格数据





某些操作需要输入表格数据。例如，XY数据工具集可以生成您在[图3-5](pt01ch03s02s07.md#int-spreadsheet)所示对话框中输入的数据的绘图。

**图3-5** X-Y数据表。

![](../graphics/int-spreadsheet-nls.png)

数据表由输入框或单元格组成，组织成行和列。您可以使用键盘在表中输入数据，也可以从文件读取数据。

以下列表描述了输入和修改表格数据的技术：

**输入数据**

单击任何单元格，然后键入所需的数据。您可以按**[Enter]**提交特定单元格中的数据。

Abaqus/CAE不允许您在需要数字数据的表中输入字符数据；如果您尝试在数字字段中输入字符数据，程序会发出蜂鸣声。（表示科学计数法的字母E（如12.E6）除外。）

**添加新行**

使用单击鼠标按钮3时出现的菜单在现有行之前或之后添加新行。将光标放在感兴趣的行上时单击鼠标按钮3；然后从出现的菜单中选择您需要的项目：
- 选择**在之前插入行**可在当前行上方添加一个空行。
- 选择**在之后插入行**可在当前行下方添加一个空行。

或者，您可以通过单击表中最后一行和最后一列中的单元格，然后按**[Enter]**来向表的末尾添加空行。

**从文件读取数据**

您可以通过从ASCII文件读取来输入数据。文件中的数据字段可以用空格、制表符或逗号的任意组合分隔；每个空格、制表符或逗号被视为单个字段分隔符。要从文件输入数据，请在目标单元格上按住光标时单击鼠标按钮3；然后从出现的菜单中选择**从文件读取**。会出现**从ASCII文件读取数据**对话框。在该对话框中，指定以下内容：
- 在**文件**文本字段中，输入要读取的文件名。
- 分别在**开始将值读入表行**和**开始将值读入表列**字段中指定目标单元格的行号和列号。（默认情况下，Abaqus将这些字段设置为您在单击鼠标按钮3时光标所在的单元格。）

单击**确定**。Abaqus根据您的规范从文件读取数据值到表中。

**在单元格之间移动**

使用**[Enter]**键在行的单元格之间从左到右移动。当您到达行尾时，按**[Enter]**将光标移动到下一行的第一个单元格。

此外，您可以使用**[Tab]**键和上下箭头键的组合在单元格之间移动。使用**[Tab]**向右移动，使用**[Shift]****+[Tab]**向左移动；使用上下箭头上下移动。您也可以直接单击感兴趣的单元格。

**更改数据**

如果单元格已包含数据，单击该单元格会突出显示数据；一旦您开始键入，单元格的内容就会消失，并被您键入的内容替换。您也可以使用**[Backspace]**或**[Delete]**键删除单元格中突出显示的数据。

单击单元格一次后，您可以第二次单击以删除突出显示并在单元格内定位光标。使用**[Backspace]**键和键盘上的其他键修改数据。

**剪切、复制和粘贴数据**

使用在单元格上单击鼠标按钮3时出现的菜单来剪切、复制和粘贴数据。您可以剪切或复制单个单元格中的数据，也可以剪切或复制整行或部分行、整列或部分列以及连续行或列系列中的数据。

首先，在包含要剪切或复制的数据的单元格上拖动鼠标。所有选中的单元格都会突出显示，但您首先选择的单元格除外。当您将光标移到数据表窗口外部或单击鼠标按钮3时，此单元格会变为突出显示。

选择感兴趣的单元格后，在选区上按住光标时单击鼠标按钮3；然后从出现的菜单中选择**剪切**或**复制**。要粘贴数据，请选择目标单元格，单击鼠标按钮3，然后从出现的菜单中选择**粘贴**。

**排序数据**

某些数据表提供排序功能。（要确定特定表是否可排序，请将光标放在表上；然后单击鼠标按钮3。如果可用，**排序**会出现在出现的菜单中。）

要对表数据进行排序，请在表上按住光标时单击鼠标按钮3；然后单击**排序**。会出现**排序表**对话框。在该对话框中，选择以下内容：
- 在**排序依据**文本字段中，选择要排序的列。
- 选择**升序**或**降序**排序顺序。

单击**确定**或**应用**。Abaqus根据指定列中的数据值对所有行进行排序。

**展开和收缩列**

您可以更改某些表中列的大小。要展开或收缩列，请将光标移至要调整大小的列标题的分隔线上；会出现调整大小光标。向左或向右拖动此光标以调整分隔线两侧两列的大小。

在某些表中，您也可以通过水平放大包含表的对话框来调整最后一列的大小。

**查看超出对话框边缘的数据**

使用水平和垂直滚动条查看对话框边界外表的部分。在某些情况下，滚动条可能不可用；请增大对话框的大小以显示更多数据。

**删除数据行**

在要删除的行中单击任何单元格，或选择连续行中的多个单元格。然后，在包含表的对话框上按住光标时单击鼠标按钮3，并从出现的菜单中选择**删除行**。行会消失；如果行有编号，Abaqus/CAE会自动对剩余行重新编号。

您无法从显示固定大小矩阵或张量的表中删除行，例如属性模块中各向同性或各向异性弹性数据输入表单中使用的那些表。

**从表数据创建X-Y数据**

在属性模块中创建材料时，您可以使用表中的数据创建X-Y数据。然后，您可以使用可视化模块绘制X-Y数据并直观地检查其有效性。要创建X-Y数据对象，请在表上按住光标时单击鼠标按钮3；然后从出现的菜单中选择**创建XY数据**。会出现**创建XY数据**对话框。在该对话框中，执行以下操作：
- 输入要创建的X-Y数据的名称。
- 指定包含X值的列号和包含Y值的列号。
- 单击**确定**。Abaqus从表中读取数据值到X-Y数据。Abaqus/CAE仅在会话期间保留保存的X-Y数据。

要查看X-Y数据，请执行以下操作：
- 从上下文栏上的模块列表中，选择**可视化**。
- 从主菜单栏，选择****工具**![](../graphics/images/arrow.gif)**XY数据**![](../graphics/images/arrow.gif)**绘图**，然后从弹出菜单中选择X-Y数据。

有关更多信息，请参阅[第47章，"X-Y绘图"](pt05ch47.md)。

**清除表**

您可以删除表中的所有数据。在表上按住光标时单击鼠标按钮3，并从出现的菜单中选择**清除表**。表数据会消失。
