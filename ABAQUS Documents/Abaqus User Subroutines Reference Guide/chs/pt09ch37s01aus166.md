### 概述

在Abaqus中，可以使用以下接触压力-闭合关系来定义接触模型：
- "硬"接触关系最小化从表面在约束位置穿透主表面，并且不允许界面传递拉应力；
- "软化"接触关系，其中接触压力是表面之间间隙的线性函数；
- "软化"接触关系，其中接触压力是表面之间间隙的指数函数（在Abaqus/Explicit中，此关系仅适用于接触对算法）；
- "软化"接触关系，其中通过逐步缩放默认惩罚刚度来构建表格压力-闭合曲线（仅适用于Abaqus/Explicit中的一般接触）；
- "软化"接触关系，其中接触压力是表面之间间隙的分段线性（表格）函数；和
- 一种一旦表面接触就不允许分离的关系。

此外，可以定义影响压力-闭合关系的粘性阻尼关系；有关更多信息，请参阅["接触阻尼，" 第37.1.3节"](pt09ch37s01aus167.md)。在Abaqus/Standard中，可以施加压力渗透加载来模拟渗透到两个接触体之间表面的流体；请参阅["压力渗透加载，" 第37.1.7节"](pt09ch37s01aus171.md)。

### 使用"软化"接触关系

Abaqus中有三种"软化"接触关系可用。压力-闭合关系可以通过线性定律、表格分段线性定律或指数定律来规定（在Abaqus/Explicit中，仅适用于接触对算法）。

对于涉及基于单元的表面和基于单元的接触（仅在Abaqus/Standard中可用）的接触，"软化"接触关系以过闭合（或间隙）与接触压力的形式指定。对于涉及基于节点的表面或节点接触单元（如GAP和ITT单元）且未定义面积或长度尺寸的接触，软化接触以过闭合（或间隙）与接触力的形式指定。对于Abaqus/Standard中梁单元上的从表面和Abaqus/Explicit中的接触对算法，将压力指定为每单位长度的力。如果Abaqus/Explicit中对梁单元上的从表面使用一般接触算法，则将压力指定为每单位面积的力。

在Abaqus/Explicit中使用在零过闭合时具有非零压力的软化接触关系（不允许与一般接触算法一起使用）时，您应该知道分析中可能存在初始未平衡的接触压力（见["在Abaqus/Explicit中调整接触对初始表面位置和指定初始间隙，" 第36.5.4节"](pt09ch36s05aus163.md)）。

#### "软化"接触与"硬"接触

"软化"接触压力-闭合关系可用于在一个或两个表面上建模软薄层。在Abaqus/Standard中，由于它们可以使接触条件更容易求解，因此有时对数值方面也有用。

#### 在隐式动态模拟中使用"软化"接触

在隐式动态冲击模拟中谨慎使用软化接触关系。如果在这种模拟中使用此关系，Abaqus/Standard将不使用冲击算法（冲击发生时破坏表面上节点动能），而是假设完全弹性碰撞。这种变化的结果是从节点在冲击主表面后会立即弹回；因此，可能导致广泛的"抖动"，从而导致收敛问题和小时间增量。

然而，在冲击效应不重要的隐式动态计算中，软化接触可能效果很好；例如，如果接触变化主要由沿着曲面（如低速金属成型应用中可能发生的）的滑动运动引起。

#### 在显式动态模拟中使用"软化"接触

在Abaqus/Explicit中，软化接触可以通过运动学或惩罚约束施加方法来施加（详见["Abaqus/Explicit中的接触约束施加方法，" 第38.2.3节"](pt09ch38s02aus182.md)）。对于惩罚施加，接触碰撞是弹性的，只是受接触阻尼的影响；而对于软化运动学接触，由于算法特性，一些能量将被冲击吸收：吸收的能量倾向于随接触刚度增加而增加。另一个考虑因素是对时间增量的影响：对于运动学施加，稳定时间增量与接触刚度无关；但对于惩罚接触，时间增量随接触刚度增加而减小。

#### 定义为线性函数的"软化"接触

在线性压力-闭合关系中，表面在它们之间的过闭合（以接触（法向）方向测量）大于零时传递接触压力。线性压力-闭合关系与具有两个数据点的表格关系相同，其中第一个点位于原点。

您需要指定压力-闭合关系的斜率*k*。

| **输入文件用法：** | ``` [*SURFACE BEHAVIOR*](../key/key-link.md#usb-kws-hsurfacebehavior), PRESSURE-OVERCLOSURE=LINEAR *k* ``` |

### 使用不分离关系

您可以指示Abaqus使用防止表面在接触后分离的接触压力-闭合关系。在Abaqus/Explicit中，此关系可以为一般接触指定，但只能用于纯主-从接触对，并且不能与自适应网格一起使用。

不分离关系通常与粗糙摩擦模型一起使用（见["摩擦行为，" 第37.1.5节"](pt09ch37s01aus169.md)）来建模非间歇性的粗糙摩擦接触。使用这种表面相互作用模型组合会导致表面在接触后保持完全粘合在一起（无分离和无切向滑动），即使它们之间的接触压力是拉伸的。

| **输入文件用法：** | ``` [*SURFACE BEHAVIOR*](../key/key-link.md#usb-kws-hsurfacebehavior), NO SEPARATION ``` |
| --- | --- |

| **Abaqus/CAE用法：** | 相互作用模块：接触属性编辑器：****机械****![](../graphics/images/arrow.gif)****法向行为****：**约束施加方法：默认**：**压力-闭合："硬"**，关闭****接触后允许分离**** |
| --- | --- |

#### Abaqus/Explicit中带有不分离关系的"软化"接触

在Abaqus/Explicit中，如果将软化接触关系与不分离关系一起指定，压力-闭合关系将包括拉伸行为。指数关系不能与不分离行为一起使用。对于表格关系，必须在零压力轴上指定一个点，斜率将根据前两个数据点相同的斜率继续进入拉伸区（见图37.1.2-6）。线性关系将具有与用于压缩行为相同的斜率的线性拉伸压力-闭合关系。

### 与接触压力-闭合相关的表面相互作用输出变量

Abaqus/Standard将间隙COPEN和接触压力CPRESS作为输出提供给数据、结果和输出数据库文件。请求输出到这些文件如["输出到数据和结果文件，" 第4.1.2节"](pt02ch04s01aus39.md)和["输出到输出数据库，" 第4.1.3节"](pt02ch04s01aus40.md)中所述。

Abaqus/Explicit将接触压力CPRESS作为输出提供给输出数据库文件（详见["输出到输出数据库，" 第4.1.3节"](pt02ch04s01aus40.md)）。

在数据、结果和输出数据库文件中，输出变量CPRESS对开放从节点给出粘性阻尼压力。此变量还对闭合从节点给出接触压力。在打印输出中，"VD"状态表示力是粘性阻尼力。

可以在Abaqus/CAE中绘制从表面上的接触压力等值线。
