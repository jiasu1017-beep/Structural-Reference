### 概述

ALE 自适应网格划分的输出：
- 可用于验证用户定义域的自动分割、拉格朗日边和角的形成、几何边和角的形成以及非自适应节点的确定；
- 必须仔细解释，因为网格特定位置处输出变量的值不再与特定材料点处的值相关联；
- 可包括示踪粒子的定义，这些粒子跟随材料点，并允许您检查这些点的轨迹并绘制这些点处所有单元和节点变量的材料时间历史；和
- 可包括自适应网格划分效率和对流准确性的诊断信息。

### 验证模型

可用于验证自适应网格划分模型的输出在数据（`.dat`）文件和输出数据库（`.odb`）中可用（有关这些文件的详细信息，请参阅 ["输出，" 第 4.1.1 节"](pt02ch04s01aus38.md)）。

#### 单元集

当用户定义的自适应网格域被 Abaqus/Explicit 分割时，组成新分割域的单元将打印到数据（`.dat`）文件中。

为所有自适应网格域创建新的单元集并写入输出数据库（`.odb`）。为每个域创建的单元集的名称是用户定义的名称加上细分编号（如果没有创建细分则为 1）加上步骤编号。例如，如果为元素集 *domain_name* 指定的用户定义自适应网格域跨越三个不相连的部分，Abaqus/Explicit 将把用户定义的域分割为三个域，并在输出数据库（`.odb`）中为第一步创建三个单元集：*domain_name*`-1-1`、`*domain_name*`-2-1` 和 *domain_name*`-3-1`。

Abaqus/CAE 可用于验证分割域的创建。

#### 边和非自适应节点

Abaqus/Explicit 根据自适应网格域的拓扑结构、与非自适应域的连接以及用户指定的边界区域自动形成拉格朗日边和角并识别非自适应节点。此外，几何边和角根据初始几何形状和初始几何特征角度的值自动形成。见 ["在 Abaqus/Explicit 中定义 ALE 自适应网格域，" 第 12.2.2 节"](pt04ch12s02aus78.md)。拉格朗日边、几何边和角以及非自适应节点（包括拉格朗日角）被输出到每个自适应网格域的数据（`.dat`）文件。可以通过请求输出到数据文件的模型和历史定义摘要摘要来获取此信息（见 ["输出中的模型和历史定义摘要，" 第 4.1.1 节"](pt02ch04s01aus38.md#usb-out-ooutput-modelhist-sum)），或通过监控自适应网格划分的进度（见下文 ["监控 ALE 自适应网格划分的进度"](pt04ch12s02aus81.md#usb-anl-aaleoutput-adaptive-prog)）。

此外，为每个步骤中的每个自适应网格域在输出数据库（`.odb`）中创建最多三个节点集。节点集的名称通过连接以下信息创建：
- 域单元集名称；
- 细分编号（如果没有创建细分则为 1）；
- LE 表示拉格朗日边、GE 表示几何边或角或 NA 表示非自适应节点（包括拉格朗日角）的字母；和
- 步骤编号。

例如，如果为元素集 *domain_name* 指定的用户定义三维自适应网格域自动分割为两个自适应网格域，Abaqus/Explicit 将在输出数据库中为第一步生成最多六个节点集：*domain_name*`-1-LE-1`、`*domain_name*`-1-GE-1`、`*domain_name*`-1-NA-1`、`*domain_name*`-2-LE-1`、`*domain_name*`-2-GE-1` 和 *domain_name*`-2-NA-1`。

由于在二维中边界区域由角分隔，而不是边，因此不会为二维自适应网格域中的拉格朗日边创建节点集。拉格朗日角包含在非自适应（NA）节点集中，与三维域相同。

Abaqus/CAE 可用于验证拉格朗日边和角、几何边和角以及非自适应节点的创建。

### 解释结果

当不执行自适应网格划分时，有限元网格跟随材料，从而能够直接解释分析结果。您可以通过研究网格的运动来可视化变形和材料运动。每个节点和单元输出变量对应于特定的材料位置，因为网格在整个时间过程中固定到相同的材料点。

一旦进行自适应网格划分，网格和材料点的位置就会偏离，分析结果必须相应地解释。自适应网格域内部的网格运动代表材料运动和自适应网格划分的综合效应。在拉格朗日和滑动边界区域上，网格和材料的运动在垂直于边界的方向上相同，但在切线方向上不同。

#### 节点变量

当执行自适应网格划分时，在步骤开始时与节点重合的材料点可能不会在该步骤中始终与该节点保持重合。位移和当前坐标的值代表节点的运动，而不一定是材料的运动。所有其他节点变量（包括速度、加速度和反作用力）代表材料粒子在节点当前位置处的变量值。这些变量的等值线或矢量图将显示其正确的空间分布，因此是有意义的。然而，对于经历自适应网格划分的节点，节点变量的时间历史通常没有意义。然而，在稳态问题中，基于固定空间位置而不是特定材料点的速度或加速度时间历史可能有用。

#### 单元变量

类似地，当执行自适应网格划分时，在步骤开始时与单元积分点重合的材料粒子可能不会在该步骤中始终保持重合。因此，单元积分点变量不一定代表整个步骤中相同材料点处的值。单元积分点变量的等值线或矢量图由于与节点变量相同的原因是有意义的。然而，时间历史基于单元积分点的空间位置，而不是特定的材料点。

整个单元变量具有类似的解释。

### 在材料点处跟踪节点或单元变量

可以定义示踪粒子来跟踪自适应网格域中的材料点。这些粒子也可用于获取对应于特定材料点处变量随时间变化的节点或单元积分点变量时间历史。示踪粒子定义如下（有关更多信息，请参阅 ["输出到输出数据库，" 第 4.1.3 节"](pt02ch04s01aus40.md)）。可以为示踪粒子集请求节点和单元变量输出，以检查材料粒子的轨迹或获取材料时间历史。示踪粒子的输出只能写入输出数据库（`.odb`）。

#### 在拉格朗日域中使用示踪粒子

在使用拉格朗日域的大多数自适应网格划分模拟中，域中的节点和单元既不对应于特定的空间位置，也不对应于特定的材料点或体积。因此，节点处和单元积分点处变量的时间历史在拉格朗日自适应网格域中通常没有物理意义。应定义示踪粒子来查看时间历史信息。示踪粒子也可用于可视化材料的运动。

示踪粒子的初始位置定义为与称为父节点的节点重合。示踪粒子通过定义多个父节点或节点集来定义集合。您指出其当前位置对应于示踪粒子初始位置的节点，并为示踪粒子集分配一个名称，以便在输出请求中使用。示踪粒子在其定义的步骤中以指定的间隔从其父节点重复释放。粒子在该步骤的剩余时间和所有后续步骤中跟随材料点。

示踪粒子通常仅在自适应网格域上定义，尽管可以在模型中连接到任何低阶实体单元的节点上定义。对于在后续步骤之前不执行自适应网格划分的分析，可以在分析开始时在非自适应域上定义示踪粒子，并将随着域变得自适应而连续跟踪。类似地，如果自适应网格域拓扑结构在步骤之间发生变化，示踪粒子将从域跟踪到域。

| **输入文件用法：** | 使用以下选项定义示踪粒子集： |
| --- | --- |
|  | ``` [*TRACER PARTICLE](../key/key-link.md#usb-kws-htracerparticle), TRACER SET=*tracer_set_name* *list of tracer particle parent nodes* ``` |

| **Abaqus/CAE 用法：** | Abaqus/CAE 不支持示踪粒子。 |
| --- | --- |

#### 在欧拉域中使用示踪粒子

在欧拉域中，节点和单元积分点处的时间历史在施加空间自适应网格约束的位置可能具有物理意义。例如，沿流出欧拉边界单元中等效塑性应变的时间变化充当该变量的空间时间历史，可用于评估过程是否已达到稳态解决方案。

可以定义示踪粒子来评估材料流过欧拉域时变量在材料点处的材料时间历史。示踪粒子也可用于评估材料点流过域时的轨迹和路径。

示踪粒子可以分配给欧拉自适应网格域中的任何父节点。如果示踪粒子到达流出边界且材料继续流出，则不再跟踪该示踪粒子，且与该示踪粒子相关的所有输出历史变量在停用后将为零。

当材料流过网格域显著时，可以在步骤中多次从父节点的当前位置释放示踪粒子集。每个示踪粒子释放称为粒子诞生。粒子诞生后，示踪粒子跟随材料的运动，而不管网格的运动。您可以指示步骤中的粒子诞生阶段数。这些阶段将均匀分布在步骤的时间周期内。

例如，可以定义示踪粒子集，使得沿流入欧拉边界的所有节点都是父节点。可以指定多个诞生阶段，以便在步骤期间定期从流入边界的网格释放一组示踪粒子。如果定义了足够多的诞生阶段，最终域将充满示踪粒子，因为材料从流入边界流到流出边界。

| **输入文件用法：** | 使用以下选项定义具有多个诞生阶段的示踪粒子集： |
| --- | --- |
|  | ``` [*TRACER PARTICLE](../key/key-link.md#usb-kws-htracerparticle), TRACER SET=*tracer_set_name*, PARTICLE BIRTH STAGES=*n* *list of tracer particle parent nodes* ``` |

| **Abaqus/CAE 用法：** | Abaqus/CAE 不支持示踪粒子。 |
| --- | --- |

### 监控 ALE 自适应网格划分的进度

可以将诊断信息写入消息（`.msg`）文件以跟踪自适应网格划分的效率和准确性。您可以选择要写入的诊断输出级别。

#### 在步骤结束时获取摘要

默认情况下，每个自适应网格域的自适应网格划分信息摘要将在每个步骤结束时写入消息（`.msg`）文件。此摘要信息包括：
- 移动节点的平均百分比，
- 移动节点的最大百分比，
- 移动节点的最小百分比，和
- 平均对流扫描次数。

每个值针对单个自适应网格域在所有自适应网格增量上计算。对流的成本与移动节点的百分比大致成正比，因为对于在自适应网格划分期间未重新定位的单元，不进行变量对流。

| **输入文件用法：** | 使用以下选项请求每个自适应网格域在每个步骤结束时的摘要： |
| --- | --- |
|  | ``` [*DIAGNOSTICS](../key/key-link.md#usb-kws-hdiagnostics), ADAPTIVE MESH=STEP SUMMARY ``` |

| **Abaqus/CAE 用法：** | Abaqus/CAE 不支持自适应网格诊断。 |
| --- | --- |

#### 获取每个 ALE 自适应网格增量的摘要

除了步骤摘要信息外，还可以为每个自适应网格域在每个自适应网格增量处获得以下诊断：
- 移动节点的百分比，和
- 对流扫描次数。

| **输入文件用法：** | 使用以下选项在步骤结束时和每个自适应网格增量处获取摘要信息： |
| --- | --- |
|  | ``` [*DIAGNOSTICS](../key/key-link.md#usb-kws-hdiagnostics), ADAPTIVE MESH=SUMMARY ``` |

| **Abaqus/CAE 用法：** | Abaqus/CAE 不支持自适应网格诊断。 |
| --- | --- |

#### 获取每个 ALE 自适应网格增量的对流准确性详细信息

以下详细诊断信息也可以写入消息（`.msg`）文件以跟踪对流的准确性：
- 对流前后的质量和动量，和
- 体积变化百分比。

| **输入文件用法：** | 使用以下选项请求最详细的诊断，包括每个自适应网格增量处每个域的对流准确性测量和摘要信息： |
| --- | --- |
|  | ``` [*DIAGNOSTICS](../key/key-link.md#usb-kws-hdiagnostics), ADAPTIVE MESH=DETAIL ``` |

| **Abaqus/CAE 用法：** | Abaqus/CAE 不支持自适应网格诊断。 |
| --- | --- |

#### 抑制 ALE 自适应网格诊断

您可以抑制所有自适应网格诊断信息的输出。

| **输入文件用法：** | ``` [*DIAGNOSTICS](../key/key-link.md#usb-kws-hdiagnostics), ADAPTIVE MESH=OFF ``` |
| --- | --- |

| **Abaqus/CAE 用法：** | Abaqus/CAE 不支持自适应网格诊断。 |
| --- | --- |
