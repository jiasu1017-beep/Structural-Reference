# 20.12.1 自动约束草图







向草图添加约束有助于确定草绘特征的最终形状。Abaqus/CAE 可以自动向您的草图应用约束。您可以使用 sketcher 选项选择 Abaqus/CAE 可以添加到草图的约束类型。（有关更多信息，请参见["在 Sketcher 中自定义约束的使用，"第 20.9.10 节](pt03ch20s09hlb10.md)。）

**自动约束草图：**

1. 从 Sketcher 工具箱中的工具中，选择自动约束工具![](../graphics/ico_sketchConstraintAuto.png)。有关 Sketcher 工具箱中工具的图表，请参见["Sketcher 工具，"第 20.4.1 节](pt03ch20s04s01.md)。Abaqus/CAE 在提示区域显示提示以指导您完成该过程。
2. 选择要约束的实体。**注意：**您可以选择整个草图；Abaqus/CAE 不会复制任何现有约束。
3. 在提示区域点击**完成**。Abaqus/CAE 在草图上创建约束。草图修改工具（如平移、旋转、缩放和镜像）可以覆盖约束。
4. 如有需要，使用删除工具移除任何不需要的约束。有关更多信息，请参见["删除 Sketcher 对象，"第 20.19.4 节](pt03ch20s16hlb04.md)。
