# 62.7.1 沿主轴创建datum轴







您可以通过选择三个主轴之一来定义datum轴。

![](../graphics/dtm-hapa-nls.png)

**沿主轴创建datum轴：**

1. 从主菜单栏中，选择****Tools**![](../graphics/images/arrow.gif)**Datum****。此时将出现**Create Datum**对话框。该对话框概述了您可以创建的datum几何体类型。Abaqus/CAE在提示区域中显示提示以指导您完成此过程。**提示：**您也可以使用模块工具箱中的![](../graphics/ico_dtmAxisPrincipal.png)工具创建datum轴。有关工具箱中datum工具的图示，请参见["使用Datum工具集，" 第62.2节](pt06ch62s02.md)。
2. 从对话框顶部的类型列表中，选择**Axis**。**Method**列表指示您可以使用的方法来创建datum轴。
3. 从**Method**列表中，选择**Principal axis**。
4. 选择datum轴必须通过的*X*、*Y*或*Z*主轴。datum轴会出现。您无法修改使用此方法创建的datum轴；您必须删除旧轴并创建一个新的。
