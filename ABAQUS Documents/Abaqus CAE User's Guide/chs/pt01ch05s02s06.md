# 5.2.6 自动拟合工具





使用**视图操作**工具栏上的自动拟合工具 ![](../graphics/ico_viewFit.png) 可以快速调整模型视图，使模型或模型绘图填充视口并在视口中居中。当你拟合模型视图时，方向不会改变，如视图 triad 所示。

当你自动拟合 *X–Y* 绘图时，自动拟合工具会将轴中的值重置为其指定的最小值和最大值；请参阅["自定义 X–Y 绘图轴，" 第 47.5 节](pt05ch47s04.md)。自动拟合工具不一定用 *X–Y* 绘图填充视口，因为图表选项可能规定绘图只占据视口的一部分；有关图表大小和定位选项的更多信息，请参阅["自定义 X–Y 绘图外观，" 第 47.7 节](pt05ch47s06.md)。

自动拟合会在你单击自动拟合工具时在当前视口中立即发生。如果你有多个视口，请在选择自动拟合工具之前选择要重新调整的视口使其成为当前视口。

当你从主菜单栏中选择****View**![](../graphics/images/arrow.gif)**Graphics Options****时，有一个单独的选项**旋转后自动拟合**。你使用此选项来控制 Abaqus/CAE 是否在旋转时自动将视图重新调整以适应视口。有关使用此选项的更多信息，请参阅["旋转视图，" 第 5.6.3 节](pt01ch05s06hlb03.md)。
