# 5.6.2 平移视图





使用**视图操作**工具栏中的平移工具 ![](../graphics/ico_viewPan.png) 在视口中水平和垂直移动视图。在使用平移工具时按住 **[Shift]** 可以访问替代平移模式。替代模式在平移视图时移动视图透视，创建更真实的图像（有关更多信息，请参阅["平移视图工具，" 第 5.2.2 节](pt01ch05s02s02.md)）。

如果当前视口链接到其他视口，Abaqus/CAE 也会移动会话中所有链接视口中的对象视图。有关更多信息，请参阅["链接视口以进行视图操作，" 第 4.5 节](pt01ch04s05.md)。

**要平移视图：**

1. 从**视图操作**工具栏中，单击平移工具 ![](../graphics/ico_viewPan.png) 进入平移模式。**提示：**你也可以从主菜单中选择****View**![](../graphics/images/arrow.gif)**Pan****，或按 **[F2]**。
2. 将光标定位在你要更改视图的视口中。光标变为四向箭头：![](../graphics/crs-pan.gif)
3. 沿任何方向拖动光标，直到获得所需的视图。拖动光标时，模型或 *X–Y* 绘图在视口中的视图位置会发生变化，橡皮筋线条表示平移量。**注意：**光标的初始位置不重要，只要你将其放置在视口内。光标移动仅受显示器物理边界的限制，即使你将光标移动到视口或窗口外部，平移也会继续。
4. 重复步骤 2 和 3，直到达到所需的视图。
5. 要退出平移模式，请执行以下操作之一：
   - 单击鼠标按钮 2。
   - 单击提示区域中的取消按钮 ![](../graphics/afxI_cancel.png)。
   - 单击平移工具。
   - 单击任何其他视图操作工具。
   **提示：**使用[循环](pt01ch05s06hlb07.md)视图操作工具返回上一个视图。
