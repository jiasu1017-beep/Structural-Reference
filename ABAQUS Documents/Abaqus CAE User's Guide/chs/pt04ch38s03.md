# 38.3 移除区域





从您不感兴趣分析的子模型中移除区域。只有感兴趣的区域应保留。您可以使用多种技术从子模型中移除区域。
- 使用Part模块中的切割工具。更多信息，请参见["添加切割特征"，第11.24节](pt03ch11s24.md)。
- 使用Geometry Edit工具集移除面。更多信息，请参见["编辑技术概述"，第69.2节](pt06ch69s02.md)。
- 您可以在Part模块中删除部件并创建具有相同名称的新部件。然后创建表示子模型的几何特征；但是，您必须将新特征定位在与原始部件相同的位置。有关子模型和全局模型之间容差的讨论，请参见["子建模：概述"，Abaqus Analysis User's Guide第10.2.1节](../usb/usb-link.md#usb-anl-asubmodeloverview)。新部件和原始部件必须在三维建模空间中创建。创建新部件是将壳全局模型转换为实体子模型的有用技术。您可以从复制的模型中删除壳部件，并在同一位置创建新的实体部件。您必须注意确保在实体子模型中定义的部件包含在壳全局模型中定义的部件内。
