# 5.2.4 放大工具





当你选择放大工具 ![](../graphics/ico_viewZoom.png) 和要工作的视口时，Abaqus/CAE 进入放大模式，如放大光标 **![](../graphics/crs-mag.png)** 所示。当你在放大模式下沿正方向拖动光标时，模型或绘图的视图会在视口中扩展，橡皮筋线条表示相对放大程度。同样，当你沿负方向拖动光标时，模型或绘图的视图会收缩，橡皮筋线条表示相对缩小程度。正负方向取决于你在视图操作选项中的设置（请参阅["使用视图操作快捷方式，" 第 68.2 节](pt06ch68s02.md)）。如果你使用默认的 Abaqus/CAE 配置进行视图操作，正方向是向右，负方向是向左。如果你使用非默认配置进行视图操作，正方向是向上，负方向是向下。为了反映配置设置，橡皮筋线条在默认配置下是水平的，在非默认配置下是垂直的。

拖动操作必须在视口中开始，但你可以继续在显示器限制范围内拖动。你也可以反复拖动以达到所需的视图。放大工具只识别你拖动动作的水平分量（对于默认配置）或垂直分量（对于非默认配置），如橡皮筋线条所示。因此，通过沿对角线拖动穿过屏幕，你可以获得更精细的控制，因为这在有效方向上产生比沿有效方向拖动相同距离更小的分量。

顾名思义，使用放大工具的默认模式会放大视图；如图 [图 5-8](pt01ch05s02s04.md#viw-magnify1) 所示，相对于视图中的对象，相机不移动。放大是通过更改视场角实现的，这与在固定相机上更改焦距时使用的方法相同。

**图 5-8** 放大视图。

![](../graphics/viw-magnify1.png)

放大工具的替代模式通过在执行操作时按住 **[Shift]** 来访问，它保持视场不变，并将相机移近或远离视图中的对象，如图 [图 5-9](pt01ch05s02s04.md#viw-magnify2) 所示。

**图 5-9** 将相机移近模型。

![](../graphics/viw-magnify2.png)

以这种方式移动相机在电影模式开启时最有用。然后你的视图不受限制；你可以将相机穿过模型移动，这样任何你不想看到的部分都会被近平面或远平面移除，或者实际上位于相机后面。如果你未使用电影模式，相机只能向前移动直到到达模型的外限。

当 *X–Y* 绘图显示在视口中时，你可以放大数据视图以聚焦于 *X–Y* 曲线的特定分量。Abaqus/CAE 会在你更改 *X–Y* 绘图的放大倍数时更新轴中的值。

如果你失去位置跟踪，可以使用[自动拟合](pt01ch05s06hlb06.md)工具 ![](../graphics/ico_viewFit.png) 将视图重新调整以适应视口。使用自动拟合工具也会将相机目标重置到模型中心。
