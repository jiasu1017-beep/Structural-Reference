# 57.1 理解振幅工具集的作用







振幅工具集允许您创建 Abaqus/Standard、Abaqus/Explicit 或 Abaqus/CFD 支持的任何类型的振幅。在振幅工具集中创建的振幅始终涉及相对数据，而直接在输入文件中定义的振幅可以涉及相对数据或绝对数据。（有关更多信息，请参见 Abaqus Analysis User's Guide](../usb/usb-link.md#usb-prc-pamplitude) 的"振幅曲线"第 34.1.2 节中的"指定相对或绝对数据"。）您还可以使用振幅工具集定义要在响应谱分析中使用的频谱。

从主菜单选择 ****工具**![](../graphics/images/arrow.gif)**振幅**![](../graphics/images/arrow.gif)**创建**** 可创建新的振幅定义；从同一菜单选择 **编辑** 可修改现有定义。任一命令都会打开振幅编辑器，允许您选择选项并提供定义振幅所需的数据。

您还可以用与绘制 *X–Y* 数据大致相同的方式绘制振幅数据。有关使用 **振幅绘图器** 插件的更多信息，请参见 ["绘制振幅数据，" 第 82.12 节](pt08ch82s12.md)。
