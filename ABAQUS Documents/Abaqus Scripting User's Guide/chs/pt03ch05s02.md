# 5.2 模态与非模态







对话框可以是模态的或非模态的。

**模态**

模态对话框在用户关闭对话框之前阻止与应用程序其余部分的交互。

**非模态**

非模态对话框允许用户在对话框显示时与 GUI 的其他部分进行交互。在 Abaqus/CAE 中，除提示外的所有辅助对话框都应该是模态对话框。

对话框本身并未定义为模态或非模态——该行为是从用于显示对话框的方法获得的。

对于由表单显示的对话框，您可以通过调用表单的 `setModal` 方法并提供 True 或 False 参数来设置模态行为。如果您使用 True 作为参数调用 `setModal`，表单将以模态方式显示下一个对话框。如果需要在一个表单管理的各个对话框之间更改模态行为，您可以多次调用 `setModal` 方法。

对于您自己显示的对话框，您可以使用 `showModal` 方法而不是下一节中描述的 `show` 方法。["文件/目录选择器"，第 5.7.1 节](pt03ch05s07.md#cus-dlg-dialogs-common-file) 包含一个使用 `showModal` 方法的示例。
