# 65.3.4 什么是自相交检查？







自相交检查是用于帮助确保您在 Abaqus/CAE 中创建的几何对于分析有效的测试。如果特征的面与同一特征的其他面相交，Abaqus/CAE 会显示一条警告，指出存在无效相交，并且不会创建该特征。与自身相交的特征可能无法进行网格划分，并可能导致分析问题。Abaqus/CAE 允许您在特征创建期间检查自相交；但是，检查过程会减慢特征的创建速度。此外，检查自相交会减慢特征的再生。完成测试所需的时间因您尝试创建或再生的特征的复杂性而异。

**Feature Options**（特征选项）对话框允许您开启和关闭自相交检查。**Perform self-intersection checks**（执行自相交检查）默认关闭。通常，只有在创建具有复杂几何的特征（如放样特征）时，才可能发生自相交（有关更多信息，请参阅["什么是放样？" 第 11.14 节"](pt03ch11s14.md)和["自相交检查，" 第 11.14.4 节"](pt03ch11s14s04.md)）。如果您知道您的特征不会自相交，或者您计划在后续操作中移除自相交，您可以保持 **Perform self-intersection checks**（执行自相交检查）关闭以加快特征再生。
