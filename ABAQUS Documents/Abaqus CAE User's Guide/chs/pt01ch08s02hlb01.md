# 8.2.1 打印到打印机或文件





Abaqus/CAE 允许你打印画布上一个或多个视口的快照，并将图像直接发送到打印机或文件以供以后使用；例如，包含在演示文稿中、嵌入打印报告中或显示在 HTML 文档中。打印的图像将再现画布上视口的层级；也就是说，如果一个视口在画布上遮挡了另一个，被遮挡的部分将不会出现在打印图像中。你可以选择的打印图像格式；附加选项允许你选择结果图像中视口的外观，以及图像的颜色、分辨率、方向和大小。

要创建打印图像，请从主菜单栏中选择****File**![](../graphics/images/arrow.gif)**Print****。要配置你的图像，请使用出现的**打印**对话框。有关对话框中各项目的详细帮助，请对各项目请求上下文相关帮助。

当你完成选择选项后，在**打印**对话框中单击**确定**将图像发送到所选目标。Abaqus/CAE 关闭**打印**对话框，将图像发送到所选目标，并保存你的打印选项以供会话期间使用。
