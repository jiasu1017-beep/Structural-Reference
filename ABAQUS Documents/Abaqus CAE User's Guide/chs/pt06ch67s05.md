# 67.5 自定义自由体切割显示







本节介绍如何使用 **Free Body Plot Options**（自由体绘图选项）对话框中的设置来自定义自由体切割的内容和外观。您可以通过从 Abaqus/CAE 可视化模块的主菜单栏中选择 ****Options**![](../graphics/images/arrow.gif)**Free Body****（选项→自由体）来打开 **Free Body Plot Options**（自由体绘图选项）对话框。

您对 **Free Body Plot Options**（自由体绘图选项）对话框中所做的更改适用于当前视口中的所有活动自由体切割，包括那些显示在视图切割上的切割。每个视口都有单独的自由体绘图选项，这为显示自由体切割提供了更大的灵活性。例如，您可以在一个视口中仅显示模型自由体切割的合力向量，而在不同视口中仅显示相同模型和自由体切割的合力矩向量。

您还可以通过编辑 Abaqus 环境文件（`abaqus_v6.env`）来控制 **Free Body Plot Options**（自由体绘图选项）对话框中出现的默认设置有关更多信息，请参阅["使用 Abaqus 环境设置，" Abaqus Analysis User's Guide 第 3.3.1 节](../usb/usb-link.md#usb-int-denvfile)。
