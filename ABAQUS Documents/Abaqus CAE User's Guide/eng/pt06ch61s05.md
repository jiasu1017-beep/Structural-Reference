# 61.5 Using the Customize toolset







All of the functionality of the Customize toolset is accessible through the **Customize** dialog box. Select ****Tools**![](../graphics/images/arrow.gif)**Customize**** or ****View**![](../graphics/images/arrow.gif)**Toolbars**![](../graphics/images/arrow.gif)**Customize**** from the main menu bar in any module to open the **Customize** dialog box.

This section explains how to use the features of the **Customize** dialog box. The following topics are covered:
- ["Hiding toolbars," Section 61.5.1](pt06ch61s05hlb01.md)
- ["Creating, modifying, and removing keyboard shortcuts," Section 61.5.2](pt06ch61s05hlb02.md)
- ["Displaying existing keyboard shortcuts," Section 61.5.3](pt06ch61s05hlb03.md)
- ["Creating and modifying custom toolbars," Section 61.5.4](pt06ch61s05hlb04.md)
- ["Changing icon assignments," Section 61.5.5](pt06ch61s05hlb05.md)
- ["Restoring toolbars to their default settings," Section 61.5.6](pt06ch61s05hlb06.md)
