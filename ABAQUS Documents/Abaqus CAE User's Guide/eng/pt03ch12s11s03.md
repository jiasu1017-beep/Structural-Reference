# 12.11.3 Defining piezoelectric properties







You can define piezoelectric material properties by providing the stress coefficients, ![](../graphics/usi_eqn00330.gif), or the strain coefficients, ![](../graphics/usi_eqn00331.gif). For more information, see ["Piezoelectric behavior," Section 26.5.2 of the Abaqus Analysis User's Guide](../usb/usb-link.md#usb-mat-cpiezoelect).

**To define piezoelectric properties:**

1. From the menu bar in the **Edit Material** dialog box, select ****Electrical/Magnetic**![](../graphics/images/arrow.gif)**Piezoelectric****. (For information on displaying the **Edit Material** dialog box, see ["Creating or editing a material," Section 12.7.1](pt03ch12s07hlb01.md).)
2. Toggle on **Use temperature-dependent data** to define piezoelectric properties as a function of temperature. A column labeled **Temp** appears in the **Data** table.
3. Click the arrows to the right of the **Number of field variables** field to increase or decrease the number of field variables on which the piezoelectric properties depend.
4. In the **Data** table, enter the piezoelectric stress or strain coefficient matrix. For more information, see ["Specifying piezoelectric material properties" in "Piezoelectric behavior," Section 26.5.2 of the Abaqus Analysis User's Guide](../usb/usb-link.md#usb-mat-cpiezoelect-coefficients). Enter values for temperature and field variables if applicable. For detailed information on how to enter data, see ["Entering tabular data," Section 3.2.7](pt01ch03s02s07.md).
5. Click **OK** to close the **Edit Material** dialog box. Alternatively, you can select another material behavior to define from the menus in the **Edit Material** dialog box (see ["Browsing and modifying material behaviors," Section 12.7.2](pt03ch12s07hlb02.md), for more information).
