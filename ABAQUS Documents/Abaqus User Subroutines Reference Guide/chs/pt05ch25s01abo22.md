# 25.1.1 流体力学行为：概述





**产品：** Abaqus/Explicit  

##### **参考文献**

- ["状态方程，" 第25.2.1节](pt05ch25s02abm50.md)

### 概述

Abaqus/Explicit中的材料库包含多个用于描述材料流体力学行为的状态方程模型。状态方程是定义压力作为密度和内能函数的本构方程（["状态方程，" 第25.2.1节](pt05ch25s02abm50.md)）。Abaqus/Explicit支持以下状态方程：

**Mie-Grüneisen状态方程**：Mie-Grüneisen状态方程（["状态方程中的Mie-Grüneisen状态方程，" 第25.2.1节](pt05ch25s02abm50.md#usb-mat-ceos-mg)）用于建模高压下的材料。它是能量线性的，并假定冲击速度与粒子速度之间存在线性关系。

**表格状态方程**：表格状态方程（["状态方程中的表格状态方程，" 第25.2.1节](pt05ch25s02abm50.md#usb-mat-ceos-tab)）用于建模在压力-密度关系中表现出尖锐相变的材料的流体力学响应。它是能量线性的。

**P–α 状态方程**：![](../graphics/usb_eqn00381.gif) 状态方程（["状态方程中的P--α状态方程，" 第25.2.1节](pt05ch25s02abm50.md#usb-mat-ceos-compaction)）专为建模延性多孔材料的压实而设计。本构模型在低应力下捕捉不可逆压实行为，并在完全压实固体材料的高压下预测正确的热力学行为。它与Mie-Grüneisen状态方程或表格状态方程结合使用来描述固相。

**JWL高能炸药状态方程**：Jones-Wilkins-Lee（或JWL）状态方程（["状态方程中的JWL高能炸药状态方程，" 第25.2.1节](pt05ch25s02abm50.md#usb-mat-ceos-jwl)）对炸药中化学能释放产生的压力进行建模。此模型以称为编程燃烧的形式实现，这意味着炸药的反应和起始不是由材料中的冲击决定的。相反，起始时间由使用爆轰波速度和距起爆点的距离的几何构造决定。

**理想气体状态方程**：理想气体状态方程（["状态方程中的理想气体状态方程，" 第25.2.1节](pt05ch25s02abm50.md#usb-mat-ceos-idealgas)）是对真实气体行为的理想化，可用于在适当条件下（例如，低压和高温）对任何气体进行近似建模。

### 偏量行为

由状态方程建模的材料可能没有偏量强度，或者可能具有各向同性弹性或粘性（牛顿和非牛顿）偏量行为（["状态方程中的偏量行为，" 第25.2.1节](pt05ch25s02abm50.md#usb-mat-ceos-deviatoric)）。弹性模型可以单独使用，也可以与Mises、Johnson-Cook或扩展Drucker-Prager塑性模型结合使用，以对具有弹塑性偏量行为的流体力学材料进行建模。

### 热应变

对于任何状态方程模型都不能引入热膨胀。
