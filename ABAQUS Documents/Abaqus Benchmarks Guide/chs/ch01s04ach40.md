# 1.4.4 带阻尼的自由振动和受迫振动

**产品：** Abaqus/Standard

本示例旨在提供 Abaqus 中可用频率相关弹簧和粘壶单元的基本验证。

系统中有几种不同的机制可以引起阻尼。在线性粘性阻尼中，阻尼力与速度成正比。在许多情况下，阻尼力的这种简单表达式不能直接获得。然而，可以通过将动能和应变能的损失等于能量耗散来获得等效粘性阻尼系数。滞变阻尼和粘弹性阻尼是比线性粘性阻尼更复杂的两种重要阻尼机制。在频域中，这些机制可以通过使用依赖于激励频率的粘性阻尼系数的粘壶来模拟。建模粘弹性阻尼还需要频率相关弹簧。

为了说明如何建模粘性、滞变和粘弹性阻尼机制，弹簧和粘壶（具有常量和频率相关特性）将用于单自由度和双自由度离散质量-弹簧-粘壶系统的频域动态分析。此外，通过使用常量粘壶系数在时域中对粘性阻尼进行建模。

Abaqus 还允许弹簧和粘壶特性依赖于温度和用户定义的场变量。这种依赖性提供了一种在时域分析期间改变弹簧和粘壶材料特性的简单方法。在使用 Abaqus 进行扰动分析（如频域稳态动态分析）时，不允许在分析步骤内改变温度和场变量。然而，由于每个扰动分析步骤的基准状态温度和场变量值可以更改，因此可以执行多步扰动分析，使用对应于基准状态温度和场变量值的不同温度和场变量相关材料特性。下面描述的分析 2 和 3 将说明这种依赖特性。这两种分析都采用 Abaqus 中的直接解和基于子空间的稳态动态过程。

单自由度和双自由度质量-弹簧-粘壶系统如图 1.4.4-1（[图 1.4.4-1](ch01s04ach40.md#sxmfreeoscdamp-systems)）所示。执行以下动态分析：（1）给定位移后释放的单自由度系统的自由振动；（2）带粘性阻尼的单自由度模型对施加谐载荷的稳态响应；（3）带滞变阻尼的单自由度模型对施加谐载荷的稳态响应；和（4）带粘弹性阻尼的双自由度模型对施加谐载荷的稳态响应。在所有情况下，激励函数都施加到最靠近锚点的点质量，数值结果与系统的精确解进行比较。

分析 3 在所有方面与分析 2 相同，只是建模的是滞变阻尼而不是线性粘性阻尼。滞变阻尼，也称为结构阻尼或固体阻尼，在许多固体材料的振动中观察到，可归因于内摩擦。这种形式的阻尼在每个加载循环的力-位移图上产生与振幅成比例的滞变回路，并倾向于随激励频率升高而保持恒定。对于粘性阻尼和滞变阻尼，能量损失都与位移振幅的平方成正比。这一事实表明，受到谐激励的结构阻尼系统可以建模为等效粘性阻尼系数与频率成反比的粘性阻尼系统：见 Denhartog（1985）。因此，这个单自由度系统的运动方程可以很容易地写成
