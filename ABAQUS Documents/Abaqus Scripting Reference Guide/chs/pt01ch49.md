# 49 Step commands (step)







The Step commands described in this chapter are used to create and configure analysis steps. [Chapter 51, "Step commands (output)](pt01ch51.md),” describes the commands used to create and configure output requests and integrated output sections and the commands to configure diagnostic printing, monitoring, and restart. [Chapter 50, "Step commands (miscellaneous)](pt01ch50.md),” describes the commands used to configure controls, damping, and frequency tables.
