# 14.13.2 配置重启输出请求





从主菜单栏选择 ****Output**![](../graphics/images/arrow.gif)**Restart Requests**** 可以配置写入重启文件的信息。更多信息，请参见「重新启动分析」，第19.6节，以及「重新启动分析」，Abaqus Analysis User's Guide第9.1.1节。

**配置重启请求的步骤：**

1. 从主菜单栏选择 ****Output**![](../graphics/images/arrow.gif)**Restart Requests****。此时会出现「Edit Restart Requests」对话框，其中显示当前模型中的步骤列表。
2. 在对话框中选择步骤，并指定Abaqus写入重启信息的频率。执行以下操作：**Abaqus/Standard和Abaqus/CFD** 对于Abaqus/Standard或Abaqus/CFD分析步骤，可以按增量或时间间隔输入频率：- 要在每若干个固定增量后写入重启信息，请在「Frequency」列中输入该数字。- 要在步骤中特定数量的等间距时间间隔写入重启信息，请在「Intervals」列中输入该数字。重启信息会在时间间隔指定的时间结束后的增量结束时写入。在Abaqus/Standard分析中，您可以切换「Time Marks」列，以便在时间间隔指定的精确时间写入重启信息。默认情况下，不会为Abaqus/Standard或Abaqus/CFD分析步骤写入任何重启信息（即频率设置为零）。**Abaqus/Explicit** 对于Abaqus/Explicit分析步骤，请输入写入重启信息的等间距时间「Intervals」数量。默认情况下，Abaqus/Explicit仅在每个步骤的开始和结束时写入重启信息（即「Intervals」设置为一）。您不能避免向Abaqus/Explicit分析步骤的重启文件写入数据；「Intervals」必须设置为一或更大。默认情况下，Abaqus/Explicit会在时间间隔指定的时间结束后的增量结束时写入重启信息。切换「Time Marks」列可以在时间间隔指定的精确时间写入重启信息。
3. 对于Abaqus/Standard或Abaqus/Explicit分析，您可以请求仅保留每个步骤中一个增量的信息，或保留每个指定增量的信息。- 切换「Overlay」列可以请求仅在重启文件中保留每个步骤中一个增量的信息，从而最小化文件大小。当步骤完成时，会保留最后一个增量。- 关闭「Overlay」列可以请求在重启文件中保留每个指定增量或时间间隔的信息。您只能在频率大于零时使用「Overlay」列。
4. 对分析的每个步骤重复步骤2和3。
5. 完成重启输出请求的配置后，点击「OK」关闭「Edit Restart Requests」对话框。
