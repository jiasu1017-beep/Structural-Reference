# 3.2.14 连接结果(.fil)文件





### 3.2.14 连接结果（`.fil`）文件

**产品：**Abaqus/Standard  Abaqus/Explicit  

##### **参考**

- ["Abaqus执行程序：概述，" 3.1.1节](pt01ch03s01abo02.md)

### 概述

**abaqus append**后处理工具：
- 用于将结果（`.fil`）文件连接成单个文件；
- 允许连接两个可以是ASCII或二进制文件或两者组合的结果文件，以便进一步后处理；并且
- 将以与**oldjob**选项指定的文件相同的格式写入结果文件。

类似工具**abaqus restartjoin**用于连接输出数据库（`.odb`）文件。请参阅["连接重启分析的输出数据库（`.odb`）文件，" 3.2.21节](pt01ch03s02abx21.md)，了解详细信息。

### 命令摘要

| **abaqus append** | [**job**](pt01ch03s02abx14.md#job1-usb-int-dappendfil)=*job-name* |
| --- | --- |
|  | [**oldjob**](pt01ch03s02abx14.md#oldjob2-usb-int-dappendfil)=*oldjob-name* [**input**](pt01ch03s02abx14.md#input3-usb-int-dappendfil)=*input-file* |

### 命令行选项

**job**

此选项指定执行期间使用的输出文件名。*job-name*值用作输出文件名。连接的输出文件将名为`*job-name*.fil`。

如果命令行中省略此选项，系统将提示您输入此值。

**oldjob**

此选项指定执行期间使用的第一个结果文件名。*oldjob-name*值用作结果文件名。

如果命令行中省略此选项，系统将提示您输入此值。

**input**

此选项指定执行期间使用的第二个结果文件名。*input-file*结果文件将被追加到*oldjob-name*结果文件。

如果命令行中省略此选项，系统将提示您输入此值。

### 示例

以下命令将把`fjoin003.fil`结果文件的历史内容追加到`fjoin002.fil`结果文件的末尾，并创建文件`fjoin001.fil`：

```
abaqus append job=fjoin001 oldjob=fjoin002 input=fjoin003
```
