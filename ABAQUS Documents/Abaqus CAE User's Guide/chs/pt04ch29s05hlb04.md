# 29.5.4 重新运行属性生成脚本





您可以使用**Re-run**标签页让Abaqus/CAE重新从模板模型复制属性，然后重新运行属性生成脚本。运行当前在**Properties**标签页上指定的脚本。操作在您单击**OK**关闭对话框时发生。

**Re-run**标签页仅在编辑已创建的组装紧固件时可用。

**注意：**每次您在**Edit Fasteners**对话框中进行更改时，重新运行操作会在您单击**OK**时自动发生。仅当没有进行其他更改时，才需要使用**Re-copy template model properties and re-run script**选项。

**要重新运行属性生成脚本：**

1. 从**Edit Fasteners**对话框中，显示**Re-run**标签页。
2. 切换**Re-copy template model properties and re-run script**。
3. 单击**OK**关闭对话框。
