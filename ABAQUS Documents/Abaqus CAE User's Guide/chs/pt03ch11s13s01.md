# 11.13.1 定义拉伸距离





您可以绘制二维轮廓并将其拉伸以创建以下内容：
- 三维拉伸实体特征。
- 三维拉伸壳特征。
- 三维拉伸切割特征。

Abaqus/CAE提供以下方法来定义拉伸距离：

**盲孔**

指定Abaqus/CAE拉伸草图的距离。草图和距离定义特征，可以使用特征操作工具集进行编辑。您可以在创建拉伸实体、壳和切割特征时使用此方法。

**至面**

选择Abaqus/CAE将草图拉伸到的单个面。选定的面不必与草图平面平行。选定的面可以是非平面面；但是，它必须完全包含拉伸截面。如果选择此方法来定义拉伸距离，则只能使用特征操作工具集修改草图；如果希望拉伸到不同的面，必须创建新的拉伸切割特征。您可以在创建拉伸实体、壳和切割特征时使用此方法。

**贯穿所有**

此方法仅适用于拉伸切割特征。Abaqus/CAE完全穿过零件拉伸定义切割轮廓的草图。如果选择此方法来定义拉伸距离，只能使用特征操作工具集修改草图。
