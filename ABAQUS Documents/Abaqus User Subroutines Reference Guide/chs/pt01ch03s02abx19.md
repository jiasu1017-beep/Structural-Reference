### 概述

**sim_version**实用工具可用于将SIM数据库文件从一个版本转换到另一个版本，查询SIM数据库文件的SIM发布级别，或确定您正在使用的代码的SIM发布级别。

### 命令摘要

| **abaqus sim_version** | **{**[**convert**](pt01ch03s02abx19.md#convert1-usb-int-dsimversion)=*old-sim-file-name*** | **[**query**](pt01ch03s02abx19.md#query2-usb-int-dsimversion)=*sim-file-name*** | **[**current**](pt01ch03s02abx19.md#current3-usb-int-dsimversion)**}** |
| --- | --- | --- | --- |
|  | **[**[**out**](pt01ch03s02abx19.md#out4-usb-int-dsimversion)=*new-sim-file-name***]** **[**[**level**](pt01ch03s02abx19.md#level5-usb-int-dsimversion)=*release-level***]** **[**[**help**](pt01ch03s02abx19.md#help6-usb-int-dsimversion)**]** |

### 命令行选项

**convert**

此选项用于指定要转换的SIM数据库文件的名称。

**query**

此选项用于指定要查询其SIM发布级别的SIM数据库文件的名称。

**current**

此选项用于确定您正在使用的代码的SIM发布级别。

**out**

此选项用于指定要由实用工具输出的SIM数据库文件的名称。此选项与**convert**选项结合使用。

**level**

此选项用于指定SIM数据库文件要转换到的发布级别名称。对于Abaqus 6.14，唯一支持的发布级别名称是`V6R2014x`。发布级别名称不区分大小写。此选项与**convert**选项结合使用。

**help**

此选项打印**abaqus sim_version**命令语法的摘要。

### 示例

使用以下命令查询SIM数据库文件的发布级别号：

```
abaqus sim_version query=*sim-file-name* 
```

使用以下命令确定与当前Abaqus版本对应的SIM级别：
```
abaqus sim_version current 
```

#### 降级由Abaqus 6.14生成的SIM数据库文件

使用以下命令降级由Abaqus 6.14生成的SIM数据库文件，以便与3DEXPERIENCE R2014x高性能后处理Physics Results Explorer应用程序配合使用：

```
abaqus sim_version convert=*sim-file-name* out=*downgraded-sim-file-name* 
   level=V6R2014x
```

以下示例在Abaqus环境文件中包含**onJobCompletion**函数，以在Abaqus作业完成后自动降级SIM数据库文件。有关更多信息，请参阅[Abaqus安装和许可指南第4.1.3节"作业定制参数"](../sgb/sgb-link.md#sgb-chp-env-jobcustom)。

以下**onJobCompletion**定义将文件降级到与3DEXPERIENCE R2014x Physics Results Explorer应用程序一起使用所需的发布级别：

```
def onJobCompletion():
    import os, osutils, subprocess
    print "Executing onJobCompletion"
    abaqusCmd = "abaqus"  # set this to the command used at your site
    simRoot = os.path.join(savedir,id)
    simName = simRoot + ".sim"
    if os.path.exists(simName):
       print "Found SIMDoc %s; running downgrade" % simName
       downgradedSimName = simRoot + "_R2014x" + ".sim"
       cmd = abaqusCmd + " sim_version convert=" + simName \
           + " level=V6R2014x out=" + downgradedSimName
       subprocess.call(cmd)
       if not os.path.exists(downgradedSimName):
          print "SIM Downgrade failed"
    print "End of onJobCompletion"
```
