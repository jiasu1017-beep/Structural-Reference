# 3.8.6 带扭转的轴对称应力/位移子模型





**产品：**Abaqus/Standard  

### 测试的单元

CGAX4R    CGAX4    CGAX6M    CGAX6MH    CGAX8H    

### 测试的功能

子模型功能应用于带扭转的轴对称应力/位移单元。子模型沿与全局模型规则几何成对角线的切割线切割全局模型。使用静态分析程序。

### 问题描述

**模型：**

全局模型在r-z平面中的尺寸为8.0×1.5。

**材料：**

| 杨氏模量 | 30×10^6 |
| --- | --- |
| 泊松比 | 0.3 |
| 密度 | 1.0 |

**步骤I边界条件：**

沿着模型的左面，![](../graphics/ver_eqn00099.gif)=0。一个节点进一步受到约束，使![](../graphics/ver_eqn00123.gif)=0。

**步骤I载荷：**

在所有模型中，1.0×10^6的压力施加在右侧面。（具有相同几何和载荷的图形显示在["二维连续体应力/位移子模型，"第3.8.2节](ch03s08abv205.md)中，只是使用直角笛卡尔坐标轴x和y而不是圆柱坐标轴r和z。）

**步骤II边界条件：**

底面在所有自由度上受到约束。

**步骤II载荷：**

在所有模型中，0.01弧度的扭转施加在顶面。

### 结果与讨论

在所有情况下，子模型中驱动节点的节点位移都从全局模型结果正确插值。在所示情况下，子模型中的单元和节点响应与全局模型中的响应匹配。
