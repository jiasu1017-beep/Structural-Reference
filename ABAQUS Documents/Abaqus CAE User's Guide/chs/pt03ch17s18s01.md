# 17.18.1 分配网格控制





**Mesh Controls**对话框允许您指定网格中单元的形状以及Abaqus/CAE用于创建网格的网格划分技术。 在某些情况下，您还可以选择过渡选项和重新定义区域角点。

**为区域分配网格控制：**

1. 从主菜单栏，选择 ****Mesh**![](../graphics/images/arrow.gif)**Controls****。 Abaqus/CAE在提示区域显示提示以指导您完成该过程。 **提示：**您也可以单击Mesh模块工具箱中的 ![](../graphics/ico_meshControls.png) 工具。
2. 如果您的零件或装配包含多个区域，请选择要查看或修改其网格控制的区域，然后按鼠标按钮2。 所有选定区域必须具有相同的维度。 **注意：**要查看或修改已分配自由网格划分技术和四面体单元形状的区域面的网格控制，或自底向上区域的面的网格控制，必须将实体选择类型在提示区域更改为**faces of solid regions**。 **Mesh Controls**对话框出现。
3. 选择您想要的网格控制。 有关特定网格控制的信息，请参阅以下内容： - ["选择单元形状，" 第17.18.2节](pt03ch17s18s02.md) - ["选择网格划分技术，" 第17.18.3节](pt03ch17s18s03.md) - ["重新定义区域角点，" 第17.18.4节](pt03ch17s18s04.md) - ["设置网格算法，" 第17.18.5节](pt03ch17s18s05.md) - ["在四面体网格边界添加楔形单元层，" 第17.18.12节](pt03ch17s18s10.md)
4. 如果需要，单击**Defaults**将**Mesh Controls**对话框中的设置更改回默认值。
5. 单击**OK**保存您的设置并关闭**Mesh Controls**对话框。
