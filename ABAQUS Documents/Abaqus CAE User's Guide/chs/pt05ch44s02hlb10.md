# 44.5.10 自定义云图颜色





默认情况下，Abaqus/CAE使用从红色（最大值）到蓝色（最小值）的**Rainbow**颜色光谱表示云图值。您可以选择替代颜色光谱或创建新的颜色光谱来使用。有关创建新颜色光谱的更多信息，请参见["创建新的颜色光谱，" 第44.5.11节](pt05ch44s02hlb11.md)。

如果您的结果包含正负极值等效的循环数据，请选择**Wraparound**颜色光谱。此光谱在两极都是绿色，正确显示诸如复数相位角的结果，其中+180和-180是相同的。

对于非刻度线云图，您还可以选择Abaqus用于表示超出您为绘图指定的限制的值的颜色。默认情况下，Abaqus以灰色阴影显示这些值。（对于刻度线云图，Abaqus仅在您指定的限制范围内显示云图曲线本身。）有关限制绘图范围的更多信息，请参见["设置云图限制，" 第44.5.7节](pt05ch44s02hlb07.md)。

**要自定义云图颜色：**

1. 找到**Spectrum**选项。从主菜单栏中选择****Options**![](../graphics/images/arrow.gif)**Contour****，或单击工具箱中的![](../graphics/ico_plotContourOptions.png)。在出现的对话框中单击**Color & Style**选项卡；然后单击**Spectrum**选项卡。**Spectrum**选项出现。
2. 要选择绘图限制内值的颜色，请选择光谱名称或单击![](../graphics/ico_spectrumSmall.png)创建和使用新的颜色光谱。有关创建新颜色光谱的信息，请参见["创建新的颜色光谱，" 第44.5.11节](pt05ch44s02hlb11.md)。**提示：**或者，您可以从Results Tree中选择或创建云图光谱。单击**Spectrums**容器创建新光谱，或单击Results Tree中任何光谱的名称以使用该光谱显示模型。
3. 要控制超出绘图限制的值的颜色，请执行以下操作之一：- 单击**Specify**；然后单击**Greater than max**或**Less than min**颜色样本![](../graphics/usi-colorsample.png)打开**Select Color**对话框。选择新颜色，然后单击**OK**关闭**Select Color**对话框并更新颜色样本。（有关更多信息，请参见["自定义颜色，" 第3.2.9节](pt01ch03s02s09.md)。）- 或，单击**Use spectrum min/max**接受此类值的最小和最大云图光谱颜色。
4. 单击**Apply**实施您的更改。当前视口中的云图更改为您指定的颜色。云图颜色与云图值之间的对应关系出现在云图图例中（如果激活）。有关云图图例的更多信息，请参见["自定义图例，" 第56.1节](pt05ch56s01.md)。默认情况下，您的更改在会话期间保存，并将影响所有后续云图。如果您想在后续会话中保留更改，请将它们保存到文件。有关更多信息，请参见["保存自定义设置以供后续会话使用，" 第55.1.1节](pt05ch55s01s01.md)。
