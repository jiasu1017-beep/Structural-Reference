# 17.3.10 独立和从属零件实例的网格划分







网格划分独立实例和从属实例的方法不同。 有关更多信息，请参见"独立实例和从属实例有什么区别？"第 13.3.2 节。

**独立实例**

要对独立实例进行网格划分，请使用上下文栏将 **Object** 更改为 **Assembly** 并直接对实例进行网格划分。您无法对用于创建独立实例的零件进行网格划分。

**从属实例**

要对从属实例进行网格划分，请使用上下文栏将 **Object** 更改为 **Part** 并选择与从属实例关联的零件。然后您可以对零件进行网格划分，Abaqus/CAE 会将相同的网格应用于装配体中的每个从属实例。当您有线性或径向模式的零件实例时，从属实例非常方便。您可以对原始零件进行网格划分，Abaqus/CAE 会将相同的网格应用于模式中该零件的每个实例。
