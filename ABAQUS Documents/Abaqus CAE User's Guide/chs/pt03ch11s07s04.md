# 11.7.4 什么是显示体？





显示体是仅用于显示的零件实例。您不必对零件进行网格划分，零件也不会被包含在分析中；但是，当您查看分析结果时，可视化模块会显示零件以及模型的其余部分。如果Abaqus/CAE报告导入的零件无效，您仍可以将零件作为显示体包含在模型中。更多信息，请参见"什么是有效和精确的零件？"，第10.2.1节。

您可以通过在相互作用模块中施加显示体约束来创建显示体。您可以对可变形和刚性零件施加显示体约束，也可以对包含几何和孤儿单元的零件施加显示体约束。您可以将零件实例约束为固定在空间中，也可以将其约束为跟随选定点。更多信息，请参见"理解约束"，第15.5节。有关使用显示体约束的模型示例，请参见第27章"显示体"。
