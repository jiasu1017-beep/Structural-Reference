# 64.9 在 Edit Mesh 工具集中撤销或重做更改







本节介绍如何撤销或重做使用 Edit Mesh 工具集中的工具对网格进行的更改。本节还说明如何配置存储 Edit Mesh 操作回滚信息的缓存。涵盖以下主题：
- ["撤销或重做网格编辑操作，" 第 64.9.1 节"](pt06ch64s09s01.md)
- ["启用撤销和管理缓存，" 第 64.9.2 节"](pt06ch64s09s02.md)

这些主题也适用于撤销和重做自底向上网格划分操作（有关自底向上网格划分的更多信息，请参阅["自底向上网格划分，" 第 17.11 节"](pt03ch17s11.md)。

Abaqus/CAE 支持 Edit Mesh 工具集中任何工具以及使用自底向上网格划分技术生成的网格的多级撤销和重做。这意味着单次撤销会移除您对网格的最新更改，第二次撤销会移除下一个最近的更改，依此类推。类似地，单次重做会恢复最近撤销的网格更改。下图显示 **Edit Mesh**（编辑网格）对话框的 **Undo**（撤销）部分在 **Undo**（撤销）和 **Redo**（重做）按钮旁边包含简短描述，高亮显示点击该按钮时将撤销或重做的网格更改。

![](../graphics/edt-undo-edit-mesh-win-nls.png)

当撤销被禁用时，描述区域会显示一条注释，指示在 **Settings**（设置）下撤销被禁用。要重新启用撤销，请参阅["启用撤销和管理缓存，" 第 64.9.2 节"](pt06ch64s09s02.md)。

Abaqus/CAE 通过在内存缓存中记录网格编辑和自底向上网格划分操作的历史来支持撤销操作。允许的缓存大小、网格大小以及您执行的网格编辑类型共同决定允许的撤销级别数。例如，如果您的网格有 100,000 个单元，并且您定义了 50 万单元的最大缓存大小，Abaqus/CAE 至少可以提供五级撤销。创建或移动节点等操作不需要大缓存来撤销。

**提示：**您可以通过运行 **Part mesh**（部件网格）或 **Instance mesh**（实例网格）查询来确定网格中的单元数。有关更多信息，请参阅["查询模型，" 第 71.2 节"](pt06ch71s02.md)。

大缓存会消耗大量内存，因此请考虑部件或部件实例所需的撤销级别数。如果您想确保任何网格编辑操作至少有一级撤销，请指定至少与部件或部件实例中的单元数一样大的缓存大小。
