# 11.12 什么是零件和装配锁定？





零件和装配锁定是Abaqus/CAE的一项功能，可防止对零件或装配的特征进行任何更改。您可以锁定零件或装配以防止意外更改，例如在与他人共享模型时或在使用包含许多相似零件的模型时。如果计划修改，必须解锁零件或装配。

**注意：**零件和装配锁定不是安全功能；任何用户都可以解锁和修改由其他用户锁定的零件和装配。

您可以右键单击模型树中的零件或装配，并使用出现的菜单来锁定和解锁特征。模型树中特征名称前的挂锁表示零件或装配已被用户或数据库升级锁定。更多信息，请参见"使用模型树管理特征"，第65.2节。

或者，您可以使用**零件管理器**来锁定或解锁模型中的任何零件。如果零件未锁定，**零件管理器**中的**状态**字段为空。如果零件被锁定，**状态**字段指示以下两种情况之一：

**锁定（数据库升级）**

Abaqus/CAE在从旧版本Abaqus升级模型时自动锁定零件。

**锁定**

用户使用模型树或**零件管理器**锁定了零件。

当从旧版本Abaqus升级数据库时，Abaqus/CAE会自动锁定装配和模型中的所有零件。锁定装配和零件使Abaqus/CAE能够比装配和所有零件也重新生成时更快地完成升级。如果解锁因数据库升级而锁定的零件，Abaqus/CAE会重新生成该零件。类似地，如果解锁因数据库升级而锁定的装配，Abaqus/CAE会重新生成该装配。

**警告：**如果零件因数据库升级而被锁定，您应该在更改造集合或属性定义之前解锁零件。如果在做出修改后解锁零件，当Abaqus/CAE重新生成零件时，您的更改可能会失效。

如果您解锁使用模型树或**零件管理器**锁定的零件，Abaqus/CAE不会重新生成该零件，因为它在锁定时无法被修改。类似地，当您使用模型树解锁装配后，Abaqus/CAE不会重新生成装配。如果解锁的零件重新生成失败，则会保留锁定版本和解锁版本。您可以在零件的解锁版本上重新创建缺失的特征，并使用它替换整个模型中的锁定零件。

您可以实例化锁定的零件并将其用于装配。此外，您可以向锁定的零件或锁定的装配添加或删除集合或属性定义。但是，必须先解锁零件或装配才能向其添加特征或编辑现有特征。
