# 47.9.1 显示或隐藏X–Y曲线图图例








### 47.9.1 显示或隐藏X--Y曲线图图例

*X–Y*曲线图图例提供了将曲线图中包含的每条曲线与其所代表的*X–Y*数据对象关联的键。每个图例条目包含一个线段，其颜色和样式与曲线图中显示的一致。图例条目还包括文本，默认情况下，该文本说明曲线所代表的*X–Y*数据对象的描述。您可以使用**Chart Legend Options**显示或隐藏图例。

**要显示或隐藏X–Y曲线图图例：**

1. 找到**Chart Legend Options**。选择****Options**![](../graphics/images/arrow.gif)**XY Options**![](../graphics/images/arrow.gif)**Chart Legend****；或点击![](../graphics/ico_xyChartLegendOptions.png)，该图标位于可视化模块工具箱的*X--Y*绘图工具中。**提示：**当显示*X--Y*曲线图时，您也可以通过在视口中双击图例直接打开此对话框。当活动过程生效时，您不能直接从视口中选择*X--Y*曲线图中的组件；在从*X--Y*曲线图中选择项目之前，请取消当前过程。
2. 点击**Contents**选项卡（如果尚未选择）。
3. 切换**Show legend**以显示图例，或将其关闭以隐藏图例。
4. 点击**Dismiss**关闭**Chart Legend Options**对话框。
