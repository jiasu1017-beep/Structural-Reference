# 7.3 使用抗锯齿





Abaqus/CAE 使用抗锯齿来改善模型上曲线和对角线的显示。抗锯齿在兼容系统上默认是启用的，你可以关闭此选项以提高某些系统的性能。

**要启用或禁用线条抗锯齿：**

1. 找到图形选项。从主菜单栏中，选择****View**![](../graphics/images/arrow.gif)**Graphics Options****。Abaqus/CAE 显示**图形选项**对话框。
2. 从**硬件**选项中，切换**抗锯齿线条**开启以启用抗锯齿，或关闭以禁用抗锯齿。
3. 单击**确定**实施更改并关闭对话框。你的更改会在会话期间保存。
