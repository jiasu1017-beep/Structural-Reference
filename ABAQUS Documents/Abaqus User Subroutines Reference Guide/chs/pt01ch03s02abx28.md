### 概述

从Nastran到Abaqus的翻译器将Nastran输入文件中的某些实体转换为其在Abaqus中的对应物。

### 使用翻译器

Nastran数据必须在扩展名为`.bdf`、`.dat`、`.nas`、`.nastran`、`.blk`或`.bulk`的文件中。翻译的Nastran数据条目列在下面的表格中。其他有效Nastran数据被跳过并记录在日志文件中。

翻译器被设计为翻译完整的Nastran输入文件。如果仅存在bulk数据，文件的前两行应是执行控制和案例控制部分的终止符，即：

```
CEND 
BEGIN BULK
```

为正常终止，使用以下行结束Nastran输入数据：
```
ENDDATA
```

Nastran解序列被翻译为[表3.2.28-1](pt01ch03s02abx28.md#table-execcontroldata)中列出的Abaqus过程。翻译器尝试根据Nastran文件中案例控制内容创建历史部分。

### 命令行选项

**job**

此选项用于指定要由翻译器输出的Abaqus输入文件的名称。它也是包含Nastran数据的文件的默认名称。翻译器生成的诊断信息将被写入名为`*job-name*.log`的文件。

**input**

此选项用于指定包含Nastran数据的文件的名称（如果与*job-name*不同）。

**wtmass_fixup**

如果**wtmass_fixup**=`ON`，则Nastran数据行`PARAM, WTMASS, *value*`上的值用作Abaqus输入文件中创建的所有密度、质量和转动惯量值的乘数。

此选项可在Abaqus环境文件中定义如下：

```
fromnastran_wtmass_fixup={OFF | ON}
```

**loadcases**

默认情况下，每个`SUBCASE`被翻译为Abaqus中的[*STEP*](../key/key-link.md#usb-kws-hstep)选项。如果**loadcases**=`ON`，对于线性静态分析，此行为会改变：每个`SUBCASE`被翻译为[*LOAD CASE*](../key/key-link.md#usb-kws-hloadcase)选项，并且所有此类[*LOAD CASE*](../key/key-link.md#usb-kws-hloadcase)选项被组合在单个[*STEP*](../key/key-link.md#usb-kws-hstep)选项中。

此选项可在Abaqus环境文件中定义如下：

```
fromnastran_loadcases={OFF | ON}
```

**pbar_zero_reset**

Nastran允许梁具有横截面积或转动惯量的零值；Abaqus不允许。设置此选项等于一个小的实数，以将*A*、![](../graphics/usb_eqn00271.gif)、![](../graphics/usb_eqn00272.gif)或*J*的任何零值重置为指定的小实数。如果省略此选项或未给出值，则使用默认值1.0×10²⁰代替零。要在翻译后的Abaqus输入文件中保留零，请设置**pbar_zero_reset**=`0`。

此选项可在Abaqus环境文件中定义如下：

```
fromnastran_pbar_zero_reset=*small-real-number*
```

**distribution**

此选项决定如何将Nastran数据中的壳和膜截面翻译为Abaqus。如果**distribution**=`OFF`，则为方向、材料偏移和/或厚度的每个组合创建单独的截面。如果**distribution**=`preservePID`或`ON`，则使用[*DISTRIBUTION*](../key/key-link.md#usb-kws-mdistribution)选项写入元素方向和偏移。如果**distribution**=`preservePID`，则创建与每个PSHELL或PCOMP属性ID对应的Abaqus截面。如果**distribution**=`ON`，则为引用相同材料的所有均匀元素创建单个Abaqus截面。

此选项可在Abaqus环境文件中定义如下：

```
fromnastran_distribution={OFF | preservePID | ON}
```

**surface_based_coupling**

某些Nastran刚元素在Abaqus中有多个对应物。如果**surface_based_coupling**=`ON`，RBE2和RBE3元素被翻译为具有适当参数的[*COUPLING*](../key/key-link.md#usb-kws-mcoupling)。否则，RBE2元素被翻译为[*KINEMATIC COUPLING*](../key/key-link.md#usb-kws-mkinematiccoupling)，RBE3元素被翻译为[*DISTRIBUTING COUPLING*](../key/key-link.md#usb-kws-mdistcoupling)。此翻译行为也适用于CBAR、CBEAM和CONM2元素上用于偏移的"隐含"RBE2型刚元素。

对于使用**surface_based_coupling**=`ON`创建的输入文件，翻译后的元素可以在Abaqus/CAE中可视化和操作。但是，大量这些元素可能会导致性能下降。

此选项可在Abaqus环境文件中定义如下：

```
fromnastran_surface_based_coupling={OFF | ON}
```

**beam_offset_coupling**

如果**beam_offset_coupling**=`ON`，则梁元素偏移通过在偏移位置创建新节点、更改梁连接到新节点以及刚性耦合新节点和原始节点来翻译。

如果**beam_offset_coupling**=`OFF`，则梁元素偏移被翻译为[*CENTROID*](../key/key-link.md#usb-kws-mcentroid)和[*SHEAR CENTER*](../key/key-link.md#usb-kws-mshearcenter)选项，这些是[*BEAM GENERAL SECTION*](../key/key-link.md#usb-kws-mbeamgensect)选项的子选项。

如果梁元素引用PBARL或PBEAML属性，或者如果梁偏移在梁轴方向上有重要分量，则此参数的设置会被忽略。在这些情况下，梁偏移始终按照好像**beam_offset_coupling**=`ON`那样翻译。

此选项可在Abaqus环境文件中定义如下：

```
fromnastran_beam_offset_coupling={OFF | ON}
```

**beam_orientation_vector**

如果**beam_orientation_vector**=`OFF`，则通过在定义横截面第一主方向的矢量的尖端创建新节点并更改梁连接到新节点来翻译梁横截面方向。

如果**beam_orientation_vector**=`ON`，则通过在[*BEAM SECTION*](../key/key-link.md#usb-kws-mbeamsection)和[*BEAM GENERAL SECTION*](../key/key-link.md#usb-kws-mbeamgensect)选项上定义矢量来翻译梁横截面。

此选项可在Abaqus环境文件中定义如下：

```
fromnastran_beam_orientation_vector={OFF | ON}
```

**cbar**

此选项用于定义从CBAR和CBEAM元素创建的2节点梁。默认值为B31。

此选项可在Abaqus环境文件中定义如下：

```
fromnastran_cbar=*2-node-beam-element*
```

**cquad4**

此选项用于定义从CQUAD4元素创建的4节点壳。默认值为S4R。如果选择减缩积分元素，则自动应用增强沙漏公式。

此选项可在Abaqus环境文件中定义如下：

```
fromnastran_cquad4=*4-node-shell-element*
```

**chexa**

此选项用于定义从CHEXA元素创建的8节点块。默认值为C3D8I。如果选择减缩积分元素，则自动应用增强沙漏公式。

此选项可在Abaqus环境文件中定义如下：

```
fromnastran_chexa=*8-node-brick-element*
```

**ctetra**

此选项用于定义从CTETRA元素创建的10节点四面体。默认值为C3D10。

此选项可在Abaqus环境文件中定义如下：

```
fromnastran_ctetra=*10-node-tetrahedron-element*
```

**plotel**

默认情况下，PLOTEL元素不被翻译。如果**plotel**=`ON`，则将PLOTEL元素翻译为名为PLOTEL_TRUSSES的元素集中的T3D2桁架元素。桁架的横截面积是输入到**pbar_zero_reset**的值，材料的杨氏模量*E*等于1.0。

**cdh_weld**

默认情况下，所有八个角节点都具有RBE3元素的CHEXA元素被翻译为**chexa**参数中指定的8节点元素类型。如果**cdh_weld**=`RIGID`，则所有八个角节点都具有RBE3元素的CHEXA元素被翻译为Abaqus中的刚性紧固件。如果**cdh_weld**=`COMPLIANT`，则所有八个角节点都具有RBE3元素的CHEXA元素被翻译为Abaqus中的柔性紧固件。
