### Problem description

The verification tests in this section consist of one-element models that are subjected to monotonically increasing tensile loads in a sequential import analysis. A slightly compressible hyperelastic material is used in the analyses. The sequence of tests involves transferring results from Abaqus/Standard to Abaqus/Explicit and then back into Abaqus/Standard.

Verification tests are also included for some first-order reduced-integration elements with enhanced hourglass control.
