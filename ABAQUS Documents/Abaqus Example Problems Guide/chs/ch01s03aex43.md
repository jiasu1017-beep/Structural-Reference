# 1.3.12 型材轧制





**产品：** Abaqus/Explicit  

本示例说明了在型材轧制的瞬态模拟中使用自适应网格划分。将结果与纯拉格朗日模拟进行了比较。

### 问题描述

此分析显示了对称I截面轧制的一个阶段。由于I截面的横截面形状，存在两个对称平面，因此只需要对截面的四分之一进行建模。四分之一对称模型（如图1.3.12-1所示）由两个刚性轧辊和一个坯料组成。轧辊1半径为747 mm，轧辊2半径为452 mm。坯料长度为850 mm，腹板半宽度为176.7 mm，腹板半厚度为24 mm，以及可变的凸缘厚度。

有限元模型如图1.3.12-2所示。坯料用C3D8R单元进行网格划分。在坯料的*y*和*z*对称平面上应用对称边界条件。轧辊建模为三维旋转解析刚性表面。轧辊1约束了所有自由度，除了绕*z*轴的旋转，那里指定了5 rad/sec的恒定角速度。轧辊2约束了所有自由度，除了绕*y*轴的旋转。向坯料施加4187.0 mm/sec的负*x*方向初始速度，以开始在坯料和轧辊之间的接触。此速度对应于轧辊在初始接触点处的速度。

使用可变质量缩放来缩放所有坯料单元的质量，以便最初达到期望的最小稳定时间增量，并且稳定时间增量在整个分析过程中不会降至该最小值以下。加载速率和质量缩放定义使得生成准静态解。

坯料为钢，建模为von Mises弹塑性材料，杨氏模量为212 GPa，初始屈服应力为80 MPa，恒定硬化斜率为258 MPa。泊松比为0.281；密度为7833 kg/m³。假定轧辊与坯料之间存在摩擦系数为0.3的库仑摩擦。

### 自适应网格划分

自适应网格划分可以改善涉及大变形的型材轧制问题的求解和网格质量。定义了包含整个坯料的单个自适应网格域。对称平面定义为拉格朗日边界区域（默认），坯料上的接触表面定义为滑动边界区域（默认）。所有自适应网格参数和控件使用默认值。

### 结果和讨论

[图1.3.12-3](ch01s03aex43.md#exxalesection-blank-def-adapt)显示了使用连续自适应网格划分时坯料的变形构型。为了进行比较，执行了纯拉格朗日模拟。[图1.3.12-4](ch01s03aex43.md#exxalesection-blank-def-lg)显示了纯拉格朗日模拟的变形构型。在拉格朗日模拟中，腹板-凸缘界面处的网格发生扭曲，但在自适应网格分析中，网格保持良好的比例。为了突出网格质量的差异，每个分析的坯料变形构型的特写视图如图1.3.12-5和[图1.3.12-6](ch01s03aex43.md#exxalesection-closeup-lg)所示。每个分析的等效塑性应变等值线如图1.3.12-7和[图1.3.12-8](ch01s03aex43.md#exxalesection-cntr-lg)所示。塑性应变分布非常相似。

[图1.3.12-9](ch01s03aex43.md#exxalesection-yforce)和[图1.3.12-10](ch01s03aex43.md#exxalesection-zmoment)分别显示了轧辊1的*y*方向反作用力和绕*z*轴的反作用力矩的时间历史图。自适应网格模拟的结果与纯拉格朗日模拟的结果密切一致。
