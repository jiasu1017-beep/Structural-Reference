# 12.4.4 创建复合铺层





您可以使用Property模块创建以下类型的复合铺层：
- 壳
- 连续壳
- 实体

要创建复合铺层，请从主菜单栏选择****Composite**![](../graphics/images/arrow.gif)**Create****。出现**Create Composite Layup**对话框，您可以在其中命名铺层、指定初始铺层数量，并指定要创建的复合铺层类型。完成此信息的输入后，点击**Create Composite Layup**对话框中的**Continue**以显示复合铺层编辑器，该编辑器允许您创建和编辑铺层。

复合铺层编辑器的格式因您定义的铺层类型而异。例如，壳复合铺层编辑器如图[12-7](pt03ch12s04s04.md#prp-compositelayup-editor-nls)所示。

**图12-7** 壳复合铺层编辑器。

![](../graphics/prp-compositelayup-editor-nls.png)

**注意：**您可以通过从主菜单栏选择****Help**![](../graphics/images/arrow.gif)**On Context****，然后点击感兴趣的编辑器功能，来显示有关此处未讨论的编辑器特定方面的帮助。将出现一个包含本指南相关部分的帮助窗口。

输入定义铺层所需的所有数据后，您可以点击**OK**关闭编辑器并保存复合铺层。
