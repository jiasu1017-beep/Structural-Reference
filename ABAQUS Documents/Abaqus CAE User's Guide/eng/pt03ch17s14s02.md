# 17.14.2 Meshing multiple two- and three-dimensional shell regions







["Meshing multiple three-dimensional solid regions," Section 17.14.1](pt03ch17s14s01.md), describes how the default meshing techniques applied to adjacent regions of a three-dimensional solid part or part instance may not allow you to generate a mesh that is compatible across the regions. In contrast, adjacent regions of a two- or three-dimensional shell part or part instance are always compatible.

[Figure 17--129](pt03ch17s14s02.md#mgn-compatible-shell-regions) illustrates a three-dimensional shell part instance with adjacent regions that Abaqus/CAE can mesh using the free, swept, and structured meshing techniques.
