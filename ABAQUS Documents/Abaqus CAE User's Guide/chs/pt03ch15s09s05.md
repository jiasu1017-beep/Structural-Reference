# 15.9.5 接触初始化编辑器





要为Abaqus/Standard中的通用接触相互作用创建接触初始化规则，请从主菜单栏选择**相互作用**![](../graphics/images/arrow.gif)**接触初始化**![](../graphics/images/arrow.gif)**创建**。会出现接触初始化编辑器，您可以在其中指定初始化定义的名称以及与该定义关联的规则。

您可以通过从主菜单栏选择**帮助**![](../graphics/images/arrow.gif)**上下文帮助**，然后点击编辑器来显示接触初始化编辑器的帮助。更多信息，请参见["创建接触初始化，"第15.12.4节](pt03ch15s12hlb04.md)。
