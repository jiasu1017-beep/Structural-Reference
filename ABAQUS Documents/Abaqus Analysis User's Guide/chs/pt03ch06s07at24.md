### 概述

涡流问题：
- 涉及电场和磁场之间的耦合，两者同时求解；
- 在低频假设下求解描述电磁现象的麦克斯韦方程，忽略位移电流的影响；
- 要求在整个域中使用电磁单元；
- 要求在整个域中指定磁导率，并在导电区域指定电导率；
- 允许时谐和瞬态电磁求解；
- 允许预定义导体的平移和转动；
- 计算与涡流相关的焦耳加热速率和磁体力强度作为输出变量，这些输出变量可从时谐电磁解转移以驱动后续传热、耦合温度-位移或应力/位移分析，从而以顺序耦合方式实现电磁场与热和/或力学场的耦合；
- 可在二维和三维空间中使用连续体单元求解。

### 涡流分析

当金属工件置于时变磁场中时会产生涡流。涡流流过工件时耗散的能量转化为热能，即焦耳加热。这种加热机制通常称为感应加热；感应炉是使用此机制的设备示例。时变磁场通常由紧靠工件放置的线圈产生。线圈通过已知总电流量，或在已知电位（电压）差下通过未知电流量。对于时谐涡流分析，线圈中的电流假定以已知频率交变；对于瞬态涡流分析，电流可随时间任意变化。

时谐涡流分析步基于以下假设：以特定频率的时谐激励将在域中各处产生相同频率的时谐电磁响应。换言之，电场和磁场均以与线圈中交流电相同的频率振荡。瞬态涡流分析不对线圈中电流的时间变化做任何假设；实际上，可指定任意时间变化，电场和磁场由时域中麦克斯韦方程的解给出。

涡流分析提供的输出（如焦耳热耗散或磁体力强度）可仅从时谐涡流分析转移，以驱动后续传热、耦合温度-位移或应力/位移分析。这允许以顺序耦合方式模拟电磁场与热和/或力学场的相互作用。详情参见["映射热载荷和磁载荷"，第 3.2.25 节](pt01ch03s02abx25.md)和["顺序耦合的预定义载荷"，第 16.1.3 节](pt04ch16s01aus97.md)。

涡流分析中必须使用电磁单元来模拟所有区域的响应，包括线圈、工件以及它们之间和周围的空间。为获得准确的解，建模空间的外边界必须在各侧距设备至少几个特征长度尺度以外。

电磁单元使用基于单元边的场插值，而非标准的基于节点的插值。用户定义的节点仅定义单元的几何形状；单元的自由度与这些节点无关，这对施加边界条件有影响（见下面的["边界条件](pt03ch06s07at24.md#usb-anl-aeddycurrent-bc)"）。

#### 定义磁性行为

电磁介质的磁性行为可以是线性或非线性的。但是，时谐涡流分析只支持线性磁性行为。线性磁性行为的特征是磁导率张量假定与磁场无关。通过直接指定绝对磁导率张量 ![](../graphics/usb_eqn01129.gif) 来定义，可以是各向同性、正交各向异性或完全各向异性的（参见["磁导率"，第 26.5.3 节](pt05ch26s05abm63.md)）。磁导率也可以取决于温度和/或预定义场变量。对于时谐涡流分析，磁导率也可以取决于频率。

非线性磁性行为仅适用于瞬态涡流分析，其特征是磁导率取决于磁场强度。Abaqus 中的非线性磁性材料模型适用于理想软磁材料，其特征是在 B-H 空间中的响应单调递增，其中 B 和 H 分别指磁通量密度矢量和磁场矢量的强度。非线性磁性行为通过直接指定一条或多条 B-H 曲线来定义，这些曲线提供 B 作为 H、可选温度和/或预定义场变量的函数，在一个或多个方向上。非线性磁性行为可以是各向同性、正交各向异性或横向各向同性的（这是更一般正交各向异性行为的特殊情况）。

#### 定义电导率

电导率 ![](../graphics/usb_eqn01098.gif) 可以是各向同性、正交各向异性或完全各向异性的（参见["电导率"，第 26.5.1 节](pt05ch26s05abm61.md)）。电导率也可以取决于温度和/或预定义场变量。对于时谐涡流分析，电导率也可以取决于频率。欧姆定律假定电导率与电场 ![](../graphics/usb_eqn01099.gif) 无关。

#### 时间增量

瞬态涡流分析中的时间积分使用向后欧拉法。该方法对线性问题无条件稳定，但时间增量过大可能导致不精确。所得方程组通常是非线性的，Abaqus/Standard 使用牛顿法求解。解通常通过一系列增量获得，每个增量内进行迭代以达到平衡。有时必须保持增量较小以确保时间积分精度。增量大小的选择也是计算效率问题：增量太大会需要更多迭代。此外，牛顿法有有限的收敛半径；增量太大可能阻止任何解的获得，因为初始状态离寻求的平衡状态太远——它在收敛半径之外。因此，对增量大小存在算法限制。

##### 自动增量

大多数情况下，默认自动增量方案是优选的，因为它将根据计算效率选择增量大小。但是，必须确保时间增量使时间积分产生精确解。Abaqus/Standard 没有内置检查来确保积分精度。

### 含电气非导体区域的涡流分析中的病态问题

在涡流分析中，模型的大部分通常由非导电区域（如空气和/或真空）组成。在这种情况下，众所周知，相关刚度矩阵可能严重病态；即可能有许多奇异性（[Br, 1999](pt03ch06s07at24.md#ref-usb-biro)）。Abaqus 使用特殊的迭代求解技术来防止病态矩阵对计算的电场和磁场产生负面影响。默认实现对许多问题效果良好。但是，在某些情况下，默认数值方案可能无法收敛或导致嘈杂的解。在这种情况下，向非导电域添加少量人工电导率可能有助于正则化问题，并允许 Abaqus 收敛到正确解。应选择人工电导率，使通过这些区域传播的电磁波几乎不发生改变，特别是不会经历当此类场撞击真实导体时典型的急剧指数衰减。建议将人工电导率设置为模型中任何导体的电导率的五到八个数量级以下。

作为在非导电域中指定电导率的替代方法，Abaqus 还提供了一种稳定化方案来帮助减轻病态效应。可通过指定稳定化因子为该稳定化算法提供输入，如果使用稳定化方案，默认假定为 1.0。稳定化因子越高，稳定化越多；稳定化因子越低，稳定化越少。

### 初始条件

可指定温度和/或预定义场变量的初始值。这些值仅影响温度和/或场变量相关的材料属性（如有）。涡流分析中不能指定电场和/或磁场的初始条件。

### 边界条件

电磁单元使用基于单元边的场插值。单元的自由度与仅定义单元几何形状的用户定义节点无关。因此，标准的基于节点的边界条件指定方法不能与电磁单元一起使用。以下段落描述了为电磁单元指定边界条件的方法。

Abaqus 中的边界条件通常指文献中传统上称为 Dirichlet 型边界条件的情况，其中主变量的值在整个边界或边界的一部分上已知。另一种，Neumann 型边界条件，指在边界的某些部分上主变量的共轭值已知的情况。在 Abaqus 中，Neumann 型边界条件在有限元公式中表示为面载荷。

在 Abaqus 中，Dirichlet 边界条件指定为（基于单元的）面上的磁矢量势 ![](../graphics/usb_eqn01119.gif)，这些面代表模型中的对称面和/或外边界；Abaqus 为代表性面计算 ![](../graphics/usb_eqn01145.gif)。在电磁场由靠近工件的载流线圈驱动的应用中，模型可能跨越的域最多为与线圈/工件组件相关特征长度尺度的 10 倍。在这种情况下，假设电磁场在远场已充分衰减，磁矢量势的值可在远场边界处设为零。另一方面，在导体嵌入均匀（但在时谐涡流分析中以时谐方式变化，在瞬态涡流分析中以更一般时间变化方式变化）远场磁场中等应用中，可能需要在外边界的某些部分指定非零磁矢量势值。在这种情况下，模拟相同物理现象的替代方法是在远场边界指定相应的唯一面电流密度值 ![](../graphics/usb_eqn01143.gif)（见下面["载荷](pt03ch06s07at24.md#usb-anl-aeddycurrent-loads)"）。![](../graphics/usb_eqn01143.gif) 可根据已知的远场磁场值计算。

#### 在时谐涡流分析中规定边界条件

在时谐涡流分析中，边界条件假定为时谐的，并同时施加于磁矢量势的实部和虚部。不能对实部指定 Dirichlet 边界条件，对虚部指定 Neumann 边界条件，反之亦然。即使只明确指定一个部分，Abaqus 也会自动约束实部和虚部。未指定的部分假定幅值为零。

在时谐涡流分析中规定基于单元面上的边界条件（参见["基于单元的面定义"，第 2.3.2 节](pt01ch02s03aus17.md)）时，必须指定面名称、区域类型标签（S）、边界条件类型标签、可选方向名称、边界条件实部的幅值、边界条件实部的方向矢量、边界条件虚部的幅值以及边界条件虚部的方向矢量。可选方向名称定义了磁矢量势分量所定义的局部坐标系。默认情况下，分量相对于全局方向定义。指定的方向矢量分量由 Abaqus 归一化，因此不影响边界条件的幅值。

在时谐涡流分析中，可按以下["时谐涡流分析中的频率相关边界条件](pt03ch06s07at24.md#usb-anl-aeddycurrent-freqdepbc)"所述规定频率相关边界条件。

| **输入文件用法：** | 在时谐涡流分析中，使用以下选项定义基于单元面上边界条件的实（同相）和虚（异相）部分： |
| --- | --- |
|  | ``` [*D EM POTENTIAL](../key/key-link.md#usb-kws-hdempotential) *面名称, S, bc 类型标签, 方向, 实部幅值, 实部方向矢量, 虚部幅值, 虚部方向矢量* ``` 其中边界条件类型标签（*bc type label*）可以是 MVP（均匀边界条件）或 MVPNU（非均匀边界条件）。 |

| **Abaqus/CAE 用法：** | Load 模块：**Create Boundary Condition**：选择 **Category** 为 **Electrical/Magnetic**，**Types for Selected Step** 为 **Magnetic vector potential**；**Distribution**：**Uniform** 或 **User-defined**；*实分量 + 虚分量* |
| --- | --- |

#### 在瞬态涡流分析中规定边界条件

瞬态涡流分析中边界条件的指定方法与时谐涡流分析基本相同，只是实和虚的概念不再适用。此时指定磁矢量势的幅值，后跟其方向矢量。指定的方向矢量分量由 Abaqus 归一化，因此不影响边界条件的幅值。

在瞬态涡流分析中，规定的边界条件可使用幅值定义进行变化（参见["幅值曲线"，第 34.1.2 节](pt07ch34s01aus115.md)）。

| **输入文件用法：** | 在瞬态涡流分析中，使用以下选项定义基于单元面上的边界条件： |
| --- | --- |
|  | ``` [*D EM POTENTIAL](../key/key-link.md#usb-kws-hdempotential) *面名称, S, bc 类型标签, 方向, 幅值, 方向矢量* ``` 其中边界条件类型标签（*bc type label*）可以是 MVP（均匀边界条件）或 MVPNU（非均匀边界条件）。 |

| **Abaqus/CAE 用法：** | Abaqus/CAE 不支持瞬态涡流分析。 |
| --- | --- |

#### 时谐涡流分析中的频率相关边界条件

可使用幅值定义将边界条件的幅值指定为频率的函数（["幅值曲线"，第 34.1.2 节](pt07ch34s01aus115.md)）。

### 预定义场

涡流分析中可指定预定义温度和场变量。这些值仅影响温度和/或场变量相关的材料属性（如有）。参见["预定义场"，第 34.6.1 节](pt07ch34s06aus128.md)。

### 材料选项

模型中各处必须指定磁性材料行为（参见["磁导率"，第 26.5.3 节](pt05ch26s05abm63.md)）。时谐涡流分析只支持线性磁性行为，而瞬态涡流分析也支持非线性磁性行为。线性磁性行为可通过直接指定磁导率来定义，而非线性磁性行为通过一条或多条 B-H 曲线定义。导体区域必须指定电导率（参见["电导率"，第 26.5.1 节](pt05ch26s05abm61.md)）。涡流分析中忽略所有其他材料属性。

在时谐涡流分析中，磁导率和电导率均可以是频率、预定义温度和场变量的函数。在瞬态涡流分析中，所有材料行为均可以是预定义温度和/或场变量的函数。

永磁体（参见["磁导率"，第 26.5.3 节](pt05ch26s05abm63.md)）可包含在瞬态涡流分析中。

### 单元

涡流分析中必须使用电磁单元来模拟所有区域。与使用基于节点插值的常规有限单元不同，这些单元使用基于边的插值，以磁矢量势沿单元边的切向分量作为主要自由度。

电磁单元在 Abaqus/Standard 中以二维（仅平面）和三维形式提供（参见["为分析类型选择合适的单元"，第 27.1.3 节](pt06ch27s01aus112.md)）。平面单元以面内磁矢量势为基础建立，因此磁通量密度和磁场矢量只有面外分量。平面单元的电场和电流密度矢量是面内的。

### 输出

涡流分析只向输出数据库（`.odb`）文件提供输出（参见["输出到输出数据库"，第 4.1.3 节](pt02ch04s01aus40.md)）。不能输出到数据文件（`.dat`）和结果文件（`.fil`）。对于以下前四个矢量量（从磁矢量势和本构方程导出）以及施加的体电流密度矢量，在时谐涡流分析步中输出实部和虚部的量级和分量。

单元形心变量：

| EMB | 磁通量密度矢量 ![](../graphics/usb_eqn01130.gif) 的量级和分量。 |
| --- | --- |

| EMH | 磁场矢量 ![](../graphics/usb_eqn01131.gif) 的量级和分量。 |
| --- | --- |

| EME | 电场矢量 ![](../graphics/usb_eqn01133.gif) 的量级和分量。 |
| --- | --- |

| EMCD | 导电区域涡流密度矢量 ![](../graphics/usb_eqn01151.gif) 的量级和分量。 |
| --- | --- |

| EMCDA | 施加体电流密度矢量的量级和分量。 |
| --- | --- |

| EMBF | 因电流流动产生的磁体力强度矢量（每单位体积每单位时间的力）。 |
| --- | --- |

| EMBFC | 因电流流动产生的复磁体力强度矢量（每单位体积的力的实部和虚部）。仅在时谐涡流分析中可用。 |
| --- | --- |

| EMJH | 因电流流动产生的焦耳加热速率（每单位体积每单位时间的热量）。 |
