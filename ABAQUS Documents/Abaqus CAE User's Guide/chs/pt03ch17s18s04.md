# 17.18.4 重新定义区域角点





结构化网格划分模式存在于具有特定拓扑的区域中。 例如，Abaqus/CAE为四边形区域应用特定的结构化模式，为五边形区域应用另一种模式。 但是，在某些情况下，您可以通过重新定义区域的角点来更改分配给曲面区域的结构化模式。 您只能为已分配结构化网格划分技术的曲面区域重新定义角点。

如果您在**Mesh Controls**对话框中单击**Redefine Region Corners**，可以选择要Abaqus/CAE在创建网格时考虑的区域的角点。 如果您未选择某个角点，Abaqus/CAE会在内部将该角点两侧的边缘"合并"为单个逻辑边缘（尽管区域的实际拓扑保持不变）。 例如，如果您将五边形区域的一个角点留未选择，Abaqus/CAE会认为该区域只有四条边而不是五条。 因此，该区域将应用四边形区域的结构化网格划分模式，而不是五边形区域的结构化网格划分模式。 有关更多信息，请参阅["二维结构化网格划分，" 第17.8.3节](pt03ch17s08s03.md)。

曲面区域只能使用结构化网格划分技术进行网格划分，前提是该区域由三到五条逻辑边界定。 但是，如果区域包含虚拟拓扑，Abaqus/CAE仅在区域有四个角点时才能应用结构化网格划分。 因此，要重新定义具有虚拟拓扑的区域角点，该区域必须由多于四个角点界定，您必须选择四个现有角点。

**重新定义区域角点：**

1. 从主菜单栏，选择 ****Mesh**![](../graphics/images/arrow.gif)**Controls****。 Abaqus/CAE在提示区域显示提示以指导您完成该过程。 **提示：**您也可以单击Mesh模块工具箱中的 ![](../graphics/ico_meshControls.png) 工具。 （有关更多信息，请参阅["使用Mesh模块工具箱，" 第17.15节](pt03ch17s15.md)。）
2. 如果您的零件或装配包含多个区域，请选择要重新定义其角点的区域，然后按鼠标按钮2。 您选择的区域应该有三个或更多顶点。 **Mesh Controls**对话框出现。
3. 如果尚未选择，请选择**Structured**作为网格划分技术。 **Redefine Region Corners**按钮出现在**Mesh Controls**对话框的右侧。
4. 单击**Redefine Region Corners**。 如果您选择了多个区域，此过程会依次考虑每个所选区域。 （Abaqus/CAE跳过无法应用结构化模式的所选区域或包含三个或更少顶点的区域。） 正在考虑的区域以品红色高亮显示。 当前为该区域选择的角点以黄色高亮显示。
5. 在提示区域，选择用于确定区域角点的选项。 - 如果您单击**Accept Highlighted**，Abaqus/CAE会接受当前高亮显示的角点。 如果选择了多个区域，会为下一个区域显示选项。 如果只选择了一个区域，过程完成，**Mesh Controls**对话框会重新出现。 - 如果您单击**Select New**，当前选定的顶点会变为红色。 您必须继续下一步。 - 如果您单击**Revert to Defaults**，区域的默认角点会高亮显示。 系统会提示您选择**Accept Highlighted**或**Select New**（如上所述）。
6. 如果您在previous步骤中单击了**Select New**，请选择您想要作为区域角点的区域顶点。 您可以选择三到五个顶点。 - **[Shift]****+单击**可选择顶点而无需取消选择所有其他顶点。 - **[Ctrl]****+单击**可取消选择单个顶点而无需取消选择所有其他顶点。 - 完成选择顶点后，单击鼠标按钮2。 所选顶点为红色，未选定顶点为黄色。 如果选择了多个区域，过程会从下一个区域重新开始。 如果只选择了一个区域，过程完成，**Mesh Controls**对话框会重新出现。
