# 16.10.17 定义磁矢量势边界条件





您可以创建磁矢量势边界条件来在表面上施加磁矢量势。（有关更多信息，请参见《Abaqus Analysis User's Guide》第6.7.5节"涡流分析"中的"边界条件"。）磁矢量势边界条件仅在电磁模型中可用。

**要指定磁矢量势：**

1. 使用以下方法之一显示磁矢量势边界条件编辑器：- 要创建新的磁矢量势边界条件，请按照"创建边界条件"中的步骤操作（第16.8.2节）（**类别**：`Electrical/Magnetic`；**所选步骤的类型**：`Magnetic vector potential`）。- 要使用菜单或管理器编辑现有磁矢量势边界条件，请参见"编辑依赖于步骤的对象"（第3.4.13节）。要编辑应用边界条件的区域，请参见"编辑应用指定条件的区域"（第16.8.4节）。
2. 单击**Distribution**字段右侧的箭头，然后从出现的列表中选择所需的选项：- 选择**Uniform**来定义在表面上均匀的边界条件。- 选择**User-defined**在用户子程序[`UDEMPOTENTIAL`](../sub/sub-link.md#sub-xsl-udempotential)中定义边界条件的大小和方向。有关更多信息，请参阅以下章节：- "指定通用作业设置"，第19.8.6节- 《Abaqus User Subroutines Reference Guide》第1.1.25节"UDEMPOTENTIAL"
3. 如果您选择了**Uniform**分布选项，请执行以下步骤：1. 在**Component 1**、**Component 2**和（如果适用）**Component 3**字段中，输入磁矢量势的实部（同相）和虚部（异相）。Abaqus/CAE计算磁矢量势的大小和方向。2. 如果需要，单击**Amplitude**字段右侧的箭头，然后从出现的列表中选择所需的幅值。或者，您可以单击![](../graphics/ico_amplitudeSmall.png)创建新的幅值。（有关更多信息，请参见第57章"幅值工具集"。）3. 单击**OK**保存数据并退出编辑器。
4. 如果您选择了**User-defined**分布选项，请执行以下步骤：1. 如果需要，在**Component 1**、**Component 2**和（如果适用）**Component 3**字段中输入磁矢量势的实部（同相）和虚部（异相）。Abaqus/CAE计算磁矢量势的大小和方向，并将该信息传递到用户子程序中。2. 单击**OK**保存数据并退出编辑器。3. 进入Job模块，然后显示感兴趣的分析作业的作业编辑器。（有关更多信息，请参见"创建、编辑和操作作业"，第19.7节。）4. 在作业编辑器中，单击**General**选项卡，然后指定包含定义边界条件大小和方向的用户子程序的文件。有关更多信息，请参见"指定通用作业设置"，第19.8.6节。**注意：**您只能在作业编辑器中指定一个用户子程序文件；如果您的分析涉及多个用户子程序，则必须将用户子程序合并到一个文件中，然后指定该文件。
