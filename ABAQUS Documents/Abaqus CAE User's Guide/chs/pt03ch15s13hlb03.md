# 15.13.4 为通用接触指定和修改接触稳定化分配





接触稳定化在两个表面之间引入粘性阻尼，以在建立接触之前稳定刚体运动。接触稳定化定义独立于接触域在相互作用编辑器的「Attribute Assignments」部分分配给表面。

接触稳定化定义可以在Abaqus/Standard分析中除初始步骤之外的任何步骤中分配。由于通用接触相互作用必须在初始步骤中创建，稳定化分配要求您编辑先前创建的通用接触相互作用。

稳定化效果通常仅在应用它们的步骤中强制执行。但是，如果选择非默认幅值作为接触稳定化定义的一部分，则稳定化效果可能会延续到后续步骤，如「Abaqus/Standard中通用接触的稳定化」，Abaqus Analysis User's Guide第36.2.5节中所述。要在这种情况下移除稳定化效果，您必须使用「Reset values from previous steps」选项创建单独的接触稳定化定义，并在后续步骤中应用该稳定化定义。

有关接触稳定化定义的更多信息，请参见「创建接触稳定化定义」，第15.12.5节。

**指定接触稳定化分配的步骤：**

1. 从主菜单栏选择 ****Interaction**![](../graphics/images/arrow.gif)**Edit**![](../graphics/images/arrow.gif)***相互作用名称*****。
2. 点击相互作用编辑器「Attribute Assignments」部分中的「Contact Properties」选项卡（如果尚未选择）。
3. 如有必要，点击相互作用编辑器「Contact Properties」部分中的![](../graphics/icoR_contactStabsSmall.png)创建接触稳定化定义；请参见「创建接触稳定化定义」，第15.12.5节，获取更多信息。
4. 点击「Stabilization assignments」旁边的![](../graphics/ico_edit.png)。此时会出现「Edit Stabilization Assignments」对话框。默认情况下，当您从列表或表中选择一个表面时，Abaqus/CAE会在视口中高亮显示该表面；但是，高亮不适用于标记为「(Global)」和「(Self)」的默认表面。您可以关闭对话框底部的「Highlight selected regions」以关闭选择高亮。
5. 从左侧第一列的现有表面列表中选择一个或多个表面。选择「(Global)」在整个接触域与单个表面之间引入稳定化效果。**提示：**您可以点击![](../graphics/ico_surface.png)定义新表面并将其添加到列表中。请参见「创建表面」，第73.3.3节，获取定义表面的说明。
6. 从第二列的现有表面列表中选择第二个或多个表面以定义表面对。- 当在任一列中选择多个表面时，会生成所有可能的组合。- 要为自接触分配稳定化定义，请在第二列中选择相同的表面名称或「(Self)」。- 任何对于接触域之外区域的稳定化分配都将被忽略。
7. 从第三列的现有稳定化定义列表中，选择要分配稳定化。您可以在「Edit Stabilization Assignments」对话框中创建新的接触稳定化定义（请参见「创建接触稳定化定义」，第15.12.5节）。
8. 点击对话框中间的箭头![](../graphics/usi-itn-contactattribarrow.png)将您的选择转移到稳定化分配列表中。对话框右侧的表会更新以反映您的选择。
9. 根据需要重复上述步骤以完成稳定化分配。如果您想删除稳定化分配，请选择行并点击![](../graphics/ico_delete.png)。**注意：**分配顺序可能是相关的；当稳定化分配重叠时，最后的分配将优先。
10. 点击「OK」保存您的选择并关闭「Edit Stabilization Assignments」对话框。相互作用编辑器重新出现，并更新了单个稳定化分配数量的信息。
11. 点击「OK」保存您的更改并关闭编辑器。

**修改接触稳定化分配的步骤：**

1. 从主菜单栏选择 ****Interaction**![](../graphics/images/arrow.gif)**Edit**![](../graphics/images/arrow.gif)***相互作用名称*****。此时会出现「Edit Interaction」对话框。
2. 点击相互作用编辑器「Attribute Assignments」部分中的「Contact Properties」选项卡（如果尚未选择）。
3. 点击「Stabilization assignments」旁边的![](../graphics/ico_edit.png)。此时会出现「Edit Stabilization Assignments」对话框。
4. 从对话框右侧的表中选择包含要修改的稳定化分配的行。
5. 从对话框左侧第三列的列表中选择新的稳定化定义。
6. 点击对话框中间的箭头![](../graphics/usi-itn-modcontactprop.png)用新选择替换稳定化定义。
7. 点击「OK」保存您的选择并关闭对话框。相互作用编辑器重新出现。
8. 点击「OK」保存您的更改并关闭相互作用编辑器。
