# 54.6 对场输出数据进行排序





对于表格场输出报告,您可以选择根据表格任何单列中的值对表中的所有数据进行排序,您可以选择按升序或降序排序。

您使用"**Report Field Output**"对话框中的"**Variable**"选项来控制Abaqus/CAE在表中包含哪些列。(有关更多信息,请参阅 ["选择场输出变量," 第54.3.2节](pt05ch54s03hlb02.md)。)表格为您选择的每个场输出变量有一列,还有额外的列来标识场输出的来源;例如,如果您的表格包含唯一的节点量,Abaqus/CAE会提供一个"**Node Label**"(节点号)列。

报告中使用负的单元标签来区分Abaqus/CAE创建的内部单元与您在定义模型时创建的单元。如果您的报告包含内部单元,并且您选择按单元标签排序,这些单元将被分组在报告的顶部或底部。

有关对探测值表格报告中的数据进行排序的信息,请参阅 ["输入表格数据," 第3.2.7节](pt01ch03s02s07.md)。

**对场输出报告数据进行排序的步骤:**

1. 找到场输出报告排序选项。从主菜单栏,选择 ****Report**![](../graphics/images/arrow.gif)**Field Output****;然后在出现的对话框中点击"**Setup**"选项卡。"**Sort by**"选项在页面中间。
2. 点击"**Sort by**"箭头显示列选择列表。
3. 从列表中选择要排序的列。
4. 点击"**Ascending**"或"**Descending**"来选择排序顺序。

要生成报告,请在对话框中配置剩余选项;然后点击"**Apply**"。完成后,点击"**Cancel**"关闭对话框。
