# 47.6.6 编辑X–Y图的自动颜色列表





### 47.6.6 编辑X–Y图的自动颜色列表

Abaqus/CAE通过引用*X–Y*自动颜色列表中的颜色定义来自动为*X–Y*曲线分配颜色。此列表以与通用**Auto Colors**列表为几何和网格单元着色相同的方式为*X–Y*曲线提供颜色，但这两个颜色列表是独立的实体。

本节说明如何编辑*X–Y*自动颜色列表以修改*X–Y*曲线的默认颜色分配集。您也可以从**Curve Options**对话框更改单个*X–Y*曲线的颜色分配。请参见["自定义X–Y曲线的外观，"第47.6.4节](pt05ch47s05hlb03.md)。

**要编辑X–Y自动颜色列表中的颜色：**

1. 在Results Tree中，在**XY Plots**容器上点击鼠标按钮3，然后从出现的列表中选择**Edit XY-Auto-Colors**。Abaqus/CAE显示**Edit XY-Auto-Colors**对话框。
2. 要将新颜色插入*X–Y*自动颜色列表：1. 在*X–Y*自动颜色列表中，高亮显示您要添加的颜色之前或之后出现的颜色。2. 点击**Insert Before**或**Insert After**以在*X–Y*自动颜色列表中的指示位置添加新颜色。**Select Color**对话框打开。3. 按照["自定义颜色，"第3.2.9节](pt01ch03s02s09.md)中描述的步骤选择颜色；然后点击**OK**关闭**Select Color**对话框。Abaqus/CAE在所选位置添加新颜色。
3. 要更改*X–Y*自动颜色列表中的颜色之一，双击该颜色并从出现的**Select Color**对话框中选择新颜色。
4. 要在*X–Y*自动颜色列表中移动颜色，高亮显示该颜色，然后点击**Move Up**或**Move Down**沿所选方向移动一个位置。
5. 要从*X–Y*自动颜色列表中删除颜色，高亮显示该颜色，然后点击**Delete**。
6. 继续添加、更改、移动和删除颜色，直到*X–Y*自动颜色列表包含您所需顺序的颜色。
7. 点击**OK**。Abaqus/CAE将修订后的*X–Y*自动颜色列表用于此后的任何*X–Y*曲线颜色编码。
