# 31.2.2 定义裂缝前缘





配置轮廓积分过程的第一步是通过从装配中选择实体来定义裂缝前缘。裂缝前缘是裂缝的前进部分。Abaqus使用裂缝前缘来计算第一个轮廓积分，使用裂缝前缘内部的所有单元和前缘外的一层单元。您可以请求多个轮廓积分的输出，在这种情况下，Abaqus/CAE通过向用于计算前一个轮廓积分的单元组添加一层单元来计算。[图31-4](pt04ch31s02s02.md#eng-crack-contourintegrals)说明了Abaqus/CAE如何通过添加单元层来计算二维模型的连续轮廓积分。

**图31-4** 通过添加单元层计算连续轮廓积分。

![](../graphics/eng-crack-contourintegrals-nls.png)

如果您的部件是三维的，Abaqus会沿裂缝线上的每个节点计算轮廓积分，如图31-5所示。更多信息，请参见["轮廓积分评估"中的"定义裂缝前缘"，Abaqus Analysis User's Guide第11.4.2节](../usb/usb-link.md#usb-anl-acontintegral-front)。

**图31-5** Abaqus沿裂缝线上的每个节点计算轮廓积分。

![](../graphics/eng-crack-contourintegrals3d-nls.png)

您可以选择的实体取决于裂缝前缘位于几何体还是孤立网格，以及部件的建模空间。

**几何体**

当您在几何体上定义裂缝前缘时，您可以选择的实体取决于部件的建模空间。

**二维几何体**

如果您在二维几何体上定义裂缝前缘，可以选择以下内容：
- 单个顶点
- 连接的边缘
- 连接的面

[图31-6](pt04ch31s02s02.md#eng-crack-front2dnative)显示了在二维几何体上定义裂缝前缘时可以选择实体的位置。

**图31-6** 从二维几何体中选择裂缝前缘。

![](../graphics/eng-crack-front2dnative-nls.png)

**三维几何体**

如果您在三维几何体上定义裂缝前缘，可以选择以下内容：
- 连接的边缘
- 连接的面
- 连接的单元

[图31-7](pt04ch31s02s02.md#eng-crack-front3dnative)显示了在三维几何体上定义裂缝前缘时可以选择实体的位置。

**图31-7** 从三维几何体中选择裂缝前缘。

![](../graphics/eng-crack-front3dnative-nls.png)

**网格**

当您在孤立网格上定义裂缝前缘时，可以选择定义裂缝前缘的单元或单元边缘或面。或者，您可以从相应区域选择节点。当您在孤立网格上定义裂缝前缘时，您可以选择的实体取决于部件的建模空间。

**二维孤立网格**

如果您在二维孤立网格上定义裂缝前缘，可以选择以下内容：
- 单个节点
- 连接的单元边缘
- 连接的单元

[图31-8](pt04ch31s02s02.md#eng-crack-front2dorphan)显示了在二维孤立网格上定义裂缝前缘时可以选择实体的位置。

**图31-8** 从二维孤立网格中选择裂缝前缘。

![](../graphics/eng-crack-front2dorphan-nls.png)

**三维孤立网格**

如果您在三维孤立网格上定义裂缝前缘，可以选择以下内容：
- 连接的单元边缘
- 连接的单元面
- 连接的单元

[图31-9](pt04ch31s02s02.md#eng-crack-front3dorphan)显示了在三维孤立网格上定义裂缝前缘时可以选择实体的位置。

**图31-9** 从三维孤立网格中选择裂缝前缘。

![](../graphics/eng-crack-front3dorphan-nls.png)
