# 4.2 选择连续体单元







正确的单元选择对于获得准确结果且成本合理至关重要。随着您在使用 Abaqus 方面更有经验，您无疑会开发出自己的选择单元的指南。但是，当您开始使用 Abaqus 时，此处给出的指南可能会有所帮助。

以下建议适用于 Abaqus/Standard 和 Abaqus/Explicit：
- 尽可能减少网格变形。带有变形线性单元的粗糙网格可能给出非常差的结果。
- 对于涉及非常大的网格变形（大应变分析）的模拟，使用线性减缩积分单元的精细网格（CAX4R、CPE4R、CPS4R、C3D8R 等）。
- 在三维中尽可能使用六面体（砖块形状）单元。它们以最低成本给出最佳结果。复杂几何可能难以完全用六面体网格划分；因此，楔形和四面体单元可能是必要的。这些单元的线性版本 C3D4 和 C3D6 是较差的单元（需要精细网格才能获得准确结果）；因此，这些单元通常应该仅在需要完成网格时使用，并且即使那样，它们也应该远离任何需要准确结果的区域。
- 一些预处理器包含用四面体单元对任意几何进行网格划分的自由网格算法。Abaqus/Standard 中的二次四面体单元（C3D10 或 C3D10I）适合一般使用；但当与接触一起使用时，它们应该仅与"表面到表面"接触离散化一起使用。这些单元的替代方案是两种分析产品中都有的改进的二次四面体单元（C3D10M）。此单元对于大变形问题和接触问题是稳健的，可使用传统的"节点到表面"或"表面到表面"接触离散化。但是，无论使用哪种单元，分析运行时间都比等效的六面体网格要长。您不应使用仅包含线性四面体单元（C3D4）的网格：除非您使用大量单元，否则结果将不准确。

Abaqus/Standard 用户还应考虑以下建议：
- 将二次减缩积分单元（CAX8R、CPE8R、CPS8R、C3D20R 等）用于一般分析工作，除非您需要建模非常大的应变或具有复杂、变化接触条件的模拟。
- 在可能存在应力集中的局部区域使用二次完全积分单元（CAX8、CPE8、CPS8、C3D20 等）。它们以最低成本提供应力梯度的最佳分辨率。
- 对于接触问题，使用线性减缩积分单元或不兼容模式单元（CAX4I、CPE4I、CPS4I、C3D8I 等）的精细网格。见[第 12 章，"接触"](ch12.md)。
