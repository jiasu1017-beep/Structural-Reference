# 34.1 什么是载荷工况？





载荷工况是用于定义特定加载条件的一组载荷和边界条件。您可以使用一个或多个载荷工况来研究结构在不同加载条件下的线性响应，涉及以下类型的分析：静态扰动、直接稳态动态、基于SIM的稳态动态（基于模态和基于子空间）以及子结构生成。载荷工况分析通常比等效的多步骤分析效率高得多，因为它利用了线性叠加原理。

您可以直接根据载荷和边界条件定义载荷工况，也可以根据先前定义的载荷工况组合来定义。您使用Load模块直接根据载荷和边界条件定义载荷工况。您使用Visualization模块在后处理期间定义先前定义载荷工况的载荷工况组合。载荷工况输出存储在输出数据库（`.odb`）中的单独帧中，您可以通过组合多个帧的结果来创建载荷工况组合的结果。

您使用Load模块中的**Load Case**菜单创建包含先前定义的载荷和边界条件的载荷工况。您可以使用非零缩放因子来缩放载荷工况内单个载荷和边界条件的大小。您只能在每个载荷工况中包含一次载荷或边界条件。如果步骤包含载荷工况，则必须将步骤中的每个载荷和边界条件包含在一个或多个载荷工况中。默认情况下，Abaqus/CAE会在您创建的每个载荷工况中包含从基态传播或修改的所有边界条件，但允许您修改单个载荷工况的此行为。您可以使用边界条件管理器来停用包含载荷工况的步骤中未使用的传播边界条件。

在包含载荷工况的步骤中，Abaqus仅支持场输出请求。Field Output Requests编辑器中创建的场输出请求适用于该步骤中的所有载荷工况。您可以使用Visualization模块查看和操作载荷工况结果（参见["查看载荷工况输出"，第34.4节](pt04ch34s04.md)）。

有关载荷工况的更多信息，请参见["多载荷工况分析"，Abaqus Analysis User's Guide第6.1.4节](../usb/usb-link.md#usb-anl-amultiloadcase)。有关在Load模块中创建载荷和边界条件的更多信息，请参见["创建和修改预定条件"，第16.4节](pt03ch16s04.md)。
