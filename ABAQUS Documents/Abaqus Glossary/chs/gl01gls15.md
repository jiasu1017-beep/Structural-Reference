# V







### view

位置、方向和比例因子定义了 Abaqus/CAE viewport 中对象视图。
更多信息：- [「了解视图操作工具，」Abaqus/CAE User's Guide 第 5.2 节](../usi/usi-link.md#uss-viw-under-viewtools)

### view orientation triad

指示当前显示模型的视图方向的三条互相垂直的轴。
更多信息：- [「自定义视图 triad，」Abaqus/CAE User's Guide 第 5.4 节](../usi/usi-link.md#uss-dsp-gen-triad)

### viewport

Abaqus/CAE canvas 上可以显示模型或分析结果的区域。您可以创建可以移动、调整大小和删除的多个 viewports。
更多信息：- [「什么是 viewport？，」Abaqus/CAE User's Guide 第 4.1.1 节](../usi/usi-link.md#uss-cnv-whatis-viewport)

### Virtual Topology toolset

Abaqus/CAE 工具集，允许您在网格零件或零件实例时忽略非常小的面和边等小细节。
更多信息：- [第 75 章，「The Virtual Topology toolset，」Abaqus/CAE User's Guide](../usi/usi-link.md#usi-vto)

### Visualization module

提供有限元模型和结果图形显示的 Abaqus/CAE 模块。它从输出数据库（`.odb`）获取模型和结果信息。Visualization 模块的主要功能包括未变形和变形形状绘制、结果等值线和符号绘制、*X–Y* 绘制和报告、现场输出报告、绘图自定义和动画。

Visualization 模块也可作为 Abaqus/Viewer 单独许可。
更多信息：- [第 40 章，「Visualization 模块基础，」Abaqus/CAE User's Guide](../usi/usi-link.md#usi-vis)

### volume fraction

在 Eulerian 分析中，以零到一之间的百分比表示的元素体积中任何给定材料的量。
更多信息：- [「Eulerian 分析，」Abaqus Analysis User's Guide 第 14.1.1 节](../usb/usb-link.md#usb-anl-aeuleriananalysis)

- [第 28 章，「Eulerian 分析，」Abaqus/CAE User's Guide](../usi/usi-link.md#usi-adv-eulerian)

### volume fraction tool

Abaqus/CAE 工具，创建离散场，根据该元素与模型装配中参考零件实例之间的空间重叠，为 Eulerian 零件实例中的每个元素指定体积分数。离散场可用于创建材料分配预定义场（在 Eulerian 零件实例中），对应于参考零件实例的几何。
更多信息：- [「体积分数工具，」Abaqus/CAE User's Guide 第 28.5 节](../usi/usi-link.md#usi-adv-eulerian-volfrac)
