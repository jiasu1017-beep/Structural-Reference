# 70.7.1 使用切割面方法分割单元





您可以通过用平面切割选定的单元来分割它们。您首先选择要分割的单元，然后选择三种方法之一来定义切割单元的平面。三种方法是：
- 选择一个点和任何直边。切割平面将通过该点并垂直于该边。
- 选择三个点。切割平面将通过所有三个点。
- 选择一条直边或曲边以及边上的一个点。切割平面将通过该点并垂直于所选点处的边。

以下过程说明了三种方法。

**使用平面方法分割单元：**

1. 从主菜单栏中，选择****工具**![](../graphics/images/arrow.gif)**分割****。将出现**创建分割**对话框。Abaqus/CAE在提示区域显示提示以指导您完成过程。**提示：**您也可以使用 ![](../graphics/ico_ptnCellCuttingPlane.png) 工具通过平面方法分割单元，该工具位于模块工具箱的分割单元工具中。有关工具箱中分割工具的图示，请参见["使用分割工具集," 第70.2节](pt06ch70s02.md)。
2. 从对话框顶部的**类型**单选按钮中，选择**单元**。**方法**列表显示您可以用来分割单元的方法。
3. 从方法列表中，选择**定义切割平面**。
4. 如果零件或装配体包含多个单元，请选择要分割的单元。您可以使用拖动选择、**[Shift]**+点击和**[Ctrl]**+点击的组合来选择多个单元进行分割。**提示：**如果您无法选择所需的单元，可以使用**选择**工具栏更改选择行为。有关更多信息，请参见["使用选择选项," 第6.3节](pt01ch06s03.md)。Abaqus/CAE高亮显示选定的单元。
5. 从提示区域中，单击以下方法之一来定义切割平面：
   - 使用**点和平面法**沿通过所选点并垂直于所选直边或基准轴的平面分割选定的单元，如下图所示：![](../graphics/par-cell-plane1-nls.png)
   - 使用**3点**方法沿通过三个所选点的平面分割选定的单元，如下图所示：![](../graphics/par-cell-plane2-nls.png)这三个点可以在零件或装配体上的任何位置，但它们必须是不同的且不能共线。
   - 使用**垂直于边**方法沿垂直于所选边并通过边上所选点的平面分割选定的单元。边可以是直的或曲的，不必是需要分割的单元的一部分，尽管您不能选择基准轴。当您想沿曲边垂直分割单元时，此方法很有用，如下图所示：![](../graphics/par-cell-plane3-nls.png)
6. 按照提示区域中提示的指导选择点和边。点可以是顶点、中点、圆弧中心或基准点。
7. 在提示区域中，单击**创建分割**。Abaqus/CAE创建分割。
