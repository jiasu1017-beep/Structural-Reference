# 70.7.5 使用N侧补片法分割单元体







您可以通过创建将单元体分割为多个区域的补片来分割单元体。您可以使用边或角点来定义补片。如果您选择一条边，Abaqus/CAE会搜索一条连续的连接边环来定义一个可行的补片。如果您选择角点，您可以选择三个、四个或五个点来定义分割。与其他单元体分割工具不同，N侧补片产生的结果分割不会超出补片范围。因此，N侧补片方法适用于创建不会延伸到单元体其余部分的孤立分割。

如果您选择一条弯曲边或选择由弯曲边连接的两个点，则结果N侧补片的相应边将沿边的轮廓。但是，如果您选择的点之间没有边连接，则补片只是简单连接这些点；补片不会沿单元体的轮廓，分割可能不完整。

**使用N侧补片法分割单元体的步骤：**

1. 从主菜单栏，选择****Tools**![](../graphics/images/arrow.gif)**Partition****。将出现"创建分割"对话框。Abaqus/CAE在提示区域显示提示以指导您完成该过程。**提示：**您还可以使用位于模块工具箱中分割单元体工具旁的![](../graphics/ico_ptnCellNSidedPatch.png)工具来使用N侧补片法分割单元体。有关工具箱中分割工具的图示，请参见["使用分割工具集，"第70.2节](pt06ch70s02.md)。
2. 从对话框顶部的**Type**单选按钮中，选择**Cell**。**Method**列表将显示可用于分割单元体的方法。
3. 从方法列表中，选择**N-sided patch**。
4. 如果部件或装配体包含多个单元体，选择要分割的单元体。**提示：**如果您无法选择所需的单元体，可以使用**Selection**工具栏更改选择行为。有关更多信息，请参见["使用选择选项，"第6.3节](pt01ch06s03.md)。Abaqus/CAE高亮显示选定的单元体。**提示：**使用**Previous**按钮(![](../graphics/afxI_backup.png))可以撤销步骤。
5. 从提示区域的按钮中，选择定义N侧补片的方法：
   - 使用**Select Edges**方法通过从选定的边序列定义N侧补片来分割单元体。使用提示区域中的文本字段选择选择边的方法：
     **Loop** 选择一条边，并让Abaqus/CAE搜索一条连续的连接边环来分割单元体，如下图所示：
     ![](../graphics/par-cell-patch1-nls.png)
     您选择的边必须在被分割单元体的边界上，且Abaqus/CAE必须能够从一系列连接的边形成一个闭合环。结果补片可以有任意数量的边。Abaqus/CAE高亮显示将形成补片的环。
     **Edges** 手动选择将分割单元体的边。您选择的边必须在被分割单元体的边界上。您可以选择任意数量的边，且所选边必须形成一个闭合环。
   - 使用**Select Corner Points**方法通过使用三到五个选定点定义N侧补片来分割单元体。Abaqus/CAE按照您选择点的顺序连接各点；因此，选择点的顺序很重要。这些点可以是顶点、中点或数据点，且必须在被分割单元体的边界边上。如果两个点之间由弯曲边连接，则补片沿曲线的轮廓延伸，如下图所示：
     ![](../graphics/par-cell-patch2-nls.png)
     Abaqus/CAE高亮显示所选点以及将形成补片的边。
6. 执行以下操作之一：
   - 如果高亮显示的边表明Abaqus/CAE将创建所需的分割，请在提示区域单击**Create Partition**。Abaqus/CAE创建分割。
   - 要更改边的选择，请单击**Previous**按钮(![](../graphics/afxI_backup.png))撤销上一步。
