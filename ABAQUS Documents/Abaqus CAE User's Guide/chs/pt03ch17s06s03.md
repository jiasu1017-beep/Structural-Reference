# 17.6.3 为什么需要在网格模块中进行分割？





您可以使用"分割"工具集将零件或独立零件实例划分为更小的区域。在网格模块中创建分割有三个原因：
- 将复杂的三维零件或实例划分为更简单的区域，使Abaqus/CAE可以使用结构化或扫掠网格技术主要使用六面体单元进行网格划分。（几乎所有三维零件都可以使用自由网格技术进行网格划分，但三维自由网格只能包含四面体单元。）
- 更好地控制网格生成。
- 获得可以分配不同单元类型的区域。

请参阅[第70章，"分割"工具集](pt06ch70.md)，了解如何使用分割工具集中每个工具的详细信息。

您只能分割零件或独立零件实例。如果您需要分割从属实例，可以分割创建该从属实例的原始零件。或者，您可以创建原始零件的副本，然后创建该副本的独立实例。然后您可以用新的独立实例替换从属实例，并分割该独立实例。有关更多信息，请参见["从属实例和独立实例有什么区别？"第13.3.2节](pt03ch13s03s02.md)。

默认情况下，自由网格技术与四边形单元将应用于所有二维零件和零件实例。当您使用此默认技术创建网格时，Abaqus/CAE隐式创建将零件划分为可使用结构化网格技术进行网格划分的区域的分割。（更多信息，请参见["使用四边形和四边形主导单元的自由网格，"第17.10.2节](pt03ch17s10s02.md)。）因此，所有二维零件都无需手动分割即可进行网格划分。

但是，当三维零件或实例无法使用六面体单元进行网格划分时，您必须采取以下步骤之一：
- 将单元形状从六面体更改为四面体，以便应用自由网格技术。
- 分割为可进行结构化或扫掠网格的区域。

当选择**网格默认值**颜色映射时，Abaqus/CAE使用橙色表示三维区域无法使用当前分配的单元形状进行网格划分。例如，图17-22展示了一个无法使用六面体单元进行网格划分的零件。

即使零件或实例可以在不分割的情况下进行网格划分，您可能仍希望进行分割以更好地控制网格生成。没有分割时，网格仅沿外部边缘对齐；有了分割，生成的网格将具有沿分割对齐的行或网格。也就是说，网格沿分割"流动"。例如，在图17-24中，分割矩形为两部分的分割导致网格沿分割以一定角度流动。

此外，您可以将不同的网格控制（如单元形状）应用于分割创建的区域。

进行分割时，请记住分割将成为单元边界。因此，请尽量确保分割与其他分割或边缘成接近90度的角度。此外，您应该避免创建会扭曲网格的不必要的短边缘。
