# 76.6 控制子结构零件的透明度





您可以指定模型中的子结构零件和零件实例是否以透明度显示。如果要控制零件和装配显示的透明度级别，请参见 ["更改透明度，" 第77.3节](pt07ch77s03.md)。

**以透明度显示子结构的步骤：**

1. 找到子结构透明度控制选项。从主菜单栏，选择 ****View**![](../graphics/images/arrow.gif)**Part Display Options**** 或 ****View**![](../graphics/images/arrow.gif)**Assembly Display Options****。在出现的对话框中，点击 **General** 选项卡。
2. 从对话框中间，切换打开 **Always show substructure with translucency** 从网格相关选项集中。
3. 点击 **OK** 实施您的更改并关闭对话框。您的更改会在会话期间保存。
