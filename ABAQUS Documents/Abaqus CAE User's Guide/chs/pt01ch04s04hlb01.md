# 4.4.2 创建箭头注释





您可以创建箭头来帮助注释视口的内容，并且可以将箭头放置在视口上的任何位置。通常，您可以使用[组合箭头和文本注释](pt01ch04s04hlb03.md)将文本注释连接到视口中的对象，但如果文本注释适用于多个位置，或者您希望在创建后完全控制锚点和位置而不编辑箭头，则可能需要添加单独的箭头。

**要创建箭头注释：**

1. 从主菜单栏，选择****视口（Viewport）**![](../graphics/images/arrow.gif)**创建注释（Create Annotation）**![](../graphics/images/arrow.gif)**箭头（Arrow）****。**提示：**您也可以通过点击[**视口**工具栏](pt01ch04s02hlb02.md)中的![](../graphics/ico_annotationArrow.png)来创建箭头注释。锚点符号会出现在视口中。默认锚点在您会话中创建新注释时会更改。如果是第一个箭头注释，锚点位于视口的左下角。
2. 如果需要，在提示区域单击**更改锚点（Change Anchor）**，然后执行以下操作之一：- 在视口中选择一个新锚点。- 从提示区域中的列表中选择视口角或中心点。- 在提示区域单击**从模型中拾取（Pick From Model）**，然后选择节点或顶点，或输入所需锚点的坐标。**注意：**如果您将注释锚定到节点或顶点，Abaqus/CAE使用所选点的模型坐标；注释不会在可视化模块中跟随模型变形。
3. 选择以下选项之一来定位箭头的端点：- 要将端点定位在当前锚点上，请在提示区域单击**在锚点上（On Anchor）**。- 要将端点放置在视口上的任何位置，请将光标移动到所需位置，然后单击鼠标按钮1。默认情况下，Abaqus/CAE在第一个端点处创建箭头；第二个端点没有符号。
4. 重复步骤2和3来锚定和定位箭头注释的第二个端点。Abaqus/CAE会打开**箭头编辑器（Arrow Editor）**对话框。
5. 完成**箭头编辑器**对话框中的信息以最终确定箭头注释。有关更多信息，请参阅["编辑箭头注释属性，"第4.4.6节"](pt01ch04s04hlb05.md)。
6. 单击**预览（Preview）**以在视口中查看您的更改。
7. 要完成箭头注释的创建，请单击**确定（OK）**。Abaqus/CAE会在视口中更新您的更改，关闭编辑器，并从步骤2开始创建另一个箭头注释。
8. 要停止创建箭头注释，请在视口中单击鼠标按钮2或单击提示区域中的![](../graphics/afxI_cancel.png)按钮。
