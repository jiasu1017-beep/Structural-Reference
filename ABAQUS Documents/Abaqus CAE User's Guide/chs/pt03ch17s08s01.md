# 17.8.1 什么是结构化网格？





结构化网格技术使用简单的预定义网格拓扑生成结构化网格。Abaqus/CAE将规则形状区域（如正方形或立方体）的网格转换到您要网格化的区域的几何形状上。例如，图17-42展示了三边形、正方形和五边形的简单网格图案如何应用于更复杂的形状。

**图17-42** 二维结构化网格图案。

![](../graphics/mgn-mapped1.png)

您可以将结构化网格技术应用于简单的二维区域（平面或曲面）或已分配**六面体**或**六面体主导**单元形状选项的简单三维区域。有关将单元形状分配给区域的更多信息，请参见["选择单元形状，"第17.18.2节](pt03ch17s18s02.md)。
