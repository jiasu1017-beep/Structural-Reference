# F.2 FLEXnet 许可证服务器操作





本节解答与 FLEXnet 许可证服务器操作相关的问题。这些问题的答案通常涉及运行 FLEXnet 管理工具之一。

### F.2.1 如何更新 FLEXnet 许可证服务器的许可证文件？

要更新 FLEXnet 许可证文件，您应该终止许可证服务器，用新许可证文件替换旧许可证文件，然后重新启动许可证服务器。如有必要，您也可以在不停止许可证服务器的情况下更新许可证文件，这样 Abaqus 用户就不会中断。两种技术都在["更新 Abaqus FLEXnet 许可证文件，"第 3.2 节](ch03s02.md)中有详细描述。

### F.2.2 如何判断 FLEXnet 许可证服务器是否正在运行？

查看许可证服务器是否正在运行的最简单方法是执行命令

```
abaqus licensing lmstat
```

此命令将列出为常规广播而运行的许可证服务器。要查看特定计算机上是否有许可证服务器正在运行，您可以输入命令

```
abaqus licensing lmstat -c [*port*]@*license_server_host*
```

其中 *license_server_host* 是您要查询的许可证服务器的主机名。如果许可证服务器使用的端口不在 27000 到 27009 之间，则必须同时指定端口号。

### F.2.3 如果 FLEXnet 许可证服务器未运行，如何重新启动它？

您必须登录许可证服务器主机才能重新启动服务器（在 Windows 平台上必须以管理员身份登录），然后运行 `lmgrd` 程序（请参阅["FLEXnet 许可证服务器管理器 lmgrd，"第 3.5 节](ch03s05.md)）。有关替代方法，请参阅["使用 **LMTOOLS** 启动 FLEXnet 服务器，"第 3.6.9 节](ch03s06.md#sgb-chp-net-lic-start)。

### F.2.4 如何终止 Abaqus FLEXnet 许可证服务器？

您可以使用 FLEXnet Licensing 管理工具 `lmdown` 终止 Abaqus 许可证服务器。如果使用 `lmdown` 终止作为冗余许可证服务器配置一部分的许可证服务器，该冗余配置中的所有三个许可证服务器都将被终止。如果您不想终止所有三个服务器，则必须使用操作系统实用程序（如 `ps` 和 `kill` 命令（Linux 平台）或 Windows **Task Manager**）来终止许可证服务器。使用 `kill` 命令时不要使用 `-9` 选项。有关详细信息，请参阅["lmdown，"第 3.6.2 节](ch03s06.md#sgb-chp-net-lic-lmdown)。

### F.2.5 如果作业或会话在完成前与 FLEXnet 许可证服务器失去联系怎么办？

客户端进入"假死"状态。客户端将尝试重新建立与许可证服务器的连接，直到超过用户定义的到期时间（默认值是无限期继续检查直到重新建立联系）。最短到期时间为 60 分钟。如果未设置预定义的到期时间，则可以使用标准的 `kill` 命令（Linux 平台）终止进程。在 Windows 平台上，如果作业在交互模式下运行，则可以通过在启动客户端的 DOS 窗口中输入 `<Ctrl>-C` 来终止客户端；否则，使用 **Task Manager**。

### F.2.6 如果客户端异常终止，FLEXnet 许可证会发生什么？

令牌通常会立即返回。如果令牌未立即返回，请确保作业未运行，并使用 `abaqus terminate` 或适当的操作系统实用程序（如 `ps` 和 `kill` 命令（Linux 平台）或 Windows **Task Manager**）停止分析的可执行文件。

不要使用 `lmremove` 工具来释放用于分析作业的许可证令牌，因为这不会产生预期效果。您可以使用 `lmremove` 来释放 Abaqus/CAE 或 Abaqus/Viewer 许可证令牌；您必须首先使用 `lmstat` 工具获取令牌信息（例如，请参阅["lmstat，"第 3.6.7 节](ch03s06.md#sgb-chp-net-lic-lmstat)）。

### F.2.7 我可以看到我的作业在 FLEXnet 许可证队列中的位置吗？

不能，只能使用日志文件信息。当分析作业在许可证队列中时，每 30 秒将队列状态信息写入作业的日志或状态文件。Abaqus/CAE 和 Abaqus/Viewer 交互会话不排队。

### F.2.8 我可以看到哪些用户正在使用 Abaqus FLEXnet 许可证吗？

命令 `*simulia_dir*/License/lmstat -a`（其中 *simulia_dir* 是 SIMULIA 父目录）将列出用户及其正在使用的许可证功能。有关详细信息，请参阅["lmstat，"第 3.6.7 节](ch03s06.md#sgb-chp-net-lic-lmstat)。

### F.2.9 安装新版本 Abaqus 时是否需要更新 FLEXnet 许可证服务器？

是的，如果新版本的 Abaqus 使用的 FLEXnet Licensing 版本晚于您现有的许可证服务器。Abaqus 6.14 使用 FLEXnet Licensing Version 11.6.1。如果您的系统上安装的 FLEXnet Licensing 版本早于 Version 11.6.1，您应该使用许可安装程序自动升级许可证服务器。如果您现有的许可证服务器使用的 FLEXnet Licensing 版本高于 Version 11.6.1，您应该使用["更新 Abaqus FLEXnet 许可证文件，"第 3.2 节](ch03s02.md)中所述的过程更新许可证文件而不覆盖现有许可证服务器。有关详细信息，请参阅["FLEXnet Licensing 安装，"第 2.1.3 节](ch02s01.md#sgb-chp-install-glance-lic)。如果您的 Abaqus 许可证文件与使用组合许可证文件的其他产品的 FLEXnet 许可证文件集成，您必须确保 `lmgrd` 版本符合 FLEXnet 版本兼容性规则（列在 FLEXnet Licensing End User Guide Version 11.6 的附录 I 中）。

从 Abaqus 6.13 开始，许可证密钥将继续在后续版本中工作两年，因此可能不需要在每个新版本中都安装新的 FLEXnet 许可证密钥。

您可以在 **Support** 页面的 **Licensing** 下检查 FLEXnet Licensing End User Guide，网址为 [www.3ds.com/simulia](http://www.3ds.com/simulia)。
