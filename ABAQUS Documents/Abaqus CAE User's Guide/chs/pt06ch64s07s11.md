# 64.7.11 查看和编辑网格-几何关联性







您可以通过从主菜单栏选择 ****Mesh**![](../graphics/images/arrow.gif)**Associate Mesh with Geometry****（网格→将网格与几何关联）来查看或编辑孤网格或自底向上网格单元与底层几何之间的关联性，并选择界定自底向上区域或一组孤单元的顶点、边或面。自顶向下技术生成的单元会自动与区域几何关联；您无法查看或编辑这些原生网格单元与相应区域几何之间的关联性。

**查看或编辑网格-几何关联性：**

1. 进入 Mesh（网格）模块。从上下文栏的 **Object**（对象）字段中选择 **Part**（部件），然后从列表中选择包含孤网格或自底向上单元的部件。
2. 从主菜单栏中选择 ****Mesh**![](../graphics/images/arrow.gif)**Associate Mesh with Geometry****（网格→将网格与几何关联）。Abaqus/CAE 在提示区域显示提示以指导您完成过程。**提示：**您也可以点击 Mesh 模块工具箱中的 ![](../graphics/ico_meshBottomUpAssociate.png) 工具，或者点击 ![](../graphics/ico_meshEdit.png) 工具显示 **Edit Mesh**（编辑网格）对话框——网格关联方法位于对话框内的 **Mesh**（网格）类别中。（有关 Mesh 模块工具箱的更多信息，请参阅["使用 Mesh 模块工具箱，" 第 17.15 节"](pt03ch17s15.md)。）
3. 选择要查看或编辑关联性的几何实体——顶点、边或面。Abaqus/CAE 以黄色高亮显示与所选几何关联的任何节点、单元边或单元面。如果没有黄色高亮显示，则没有网格部分与所选实体关联。
4. 使用以下选择技术更改网格-几何关联性（有关更多信息，请参阅[第 6 章，"在视口中选择对象"](pt01ch06.md)"）： 1. 点击节点、单元边或单元面以取消任何现有选择并开始选择新项目。 2. **[Shift]****+点击** 将节点、单元边或单元面添加到当前选择，或 **[Ctrl]****+点击** 取消选择项目。 3. 使用提示区域中提供的 **by angle**（按角度）选择方法根据它们之间的角度选择单元面。 4. 您可以使用 **Selection**（选择）工具栏中的工具来限制可以在视口中选择的对象类型。**注意：**在某些情况下，您可能需要选择内部网格面或边来创建正确的网格-几何关联。为了让您能够选择内部对象，Abaqus/CAE 不会将您的关联性选择限制为"屏幕上最近的对象"，即使此选择选项处于活动状态（有关更多信息，请参阅["根据对象位置过滤您的选择，" 第 6.3.3 节"](pt01ch06s03s02.md)。）。因此，您需要确保不会无意中关联不需要的单元，尤其是在使用拖动选择来选择多个对象时。
5. 完成选择要关联的网格对象后，点击 **Done**（完成）。Abaqus/CAE 保存新的关联性。
6. 重复第 3 步到第 5 步以验证或更改其他区域的关联性，或点击提示区域中的取消按钮 ![](../graphics/afxI_cancel.png) 结束关联性过程。
