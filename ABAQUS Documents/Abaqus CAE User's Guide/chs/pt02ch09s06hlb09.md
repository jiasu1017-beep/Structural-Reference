# 9.7.9 保存当前模型数据库





从主菜单栏选择****文件（File）**![](../graphics/images/arrow.gif)**保存（Save）****或单击**文件**工具栏中的![](../graphics/ico_fileSave.png)来保存当前模型数据库（如果是新数据库），或将当前会话期间所做的更改追加到先前保存的模型数据库。保存模型数据库后，Abaqus/CAE会在主窗口的标题栏中显示其名称。

在您第一次保存当前模型数据库之前，它仅存在于内存中且没有名称。当您第一次保存当前模型数据库时，Abaqus/CAE会显示**另存模型数据库为（Save Model Database As）**对话框，允许您输入名称；后续保存将使用此名称，并将当前会话期间所做的更改追加到先前保存的模型数据库。如果您省略文件扩展名，Abaqus/CAE会将`.cae`附加到文件名。

有关使用不同名称将模型数据库保存到新文件的信息，请参阅["将当前模型数据库保存为不同名称的新文件，"第9.7.11节"](pt02ch09s06hlb11.md)。有关保存文件的更多信息，请参阅["使用文件选择对话框，"第3.2.10节"](pt01ch03s02s10.md)。

您应该定期保存模型数据库。除非您执行明确的保存操作，否则Abaqus/CAE永远不会保存模型数据库；例如，没有基于计时器的自动保存。如果您尝试保存未修改的模型数据库，则不会采取任何操作。

Abaqus/CAE会在您退出会话前询问您是否要保存已修改的模型数据库。

****文件（File）**![](../graphics/images/arrow.gif)**保存（Save）****命令不会压缩模型数据库，即使您已从模型中删除了项目。要在删除模型内容后减小文件大小，请使用****文件（File）**![](../graphics/images/arrow.gif)**压缩MDB（Compress MDB）****命令或将模型数据库保存到新文件名（使用****文件（File）**![](../graphics/images/arrow.gif)**另存为（Save As）****命令）。
