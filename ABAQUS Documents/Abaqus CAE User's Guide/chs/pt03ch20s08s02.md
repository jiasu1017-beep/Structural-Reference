# 20.8.2 通过更改尺寸或添加参数修改对象







使用直线和顶点之间的尺寸来定义和维护特征之间的尺寸关系。尺寸可以应用于以下任何项之间的直线和顶点：
- 草图几何
- 参考几何
- 构造几何

参数是已与名称关联的尺寸或常量，以便您可以使用它们创建函数并定义特征之间的关系。例如，考虑一个具有拉伸圆形柱的三维壳体，如下图所示：

![](../graphics/ske-dimtech1.png)

此零件分两步创建：

1. 创建一个仅由壳体组成的零件。
2. 从壳体的面拉伸实体特征（圆形柱）。

如果您想保持壳体左边与柱中心之间的距离恒定，您可以编辑柱并在草绘圆的中心与代表壳体的参考几何的左边之间添加尺寸，如下图所示：

![](../graphics/ske-dimtech2.png)

如果您移动壳体的左边或右边，圆心与左边之间的距离保持恒定，如下图所示：

![](../graphics/ske-dimtech3.png)

如果您想将柱保持在壳体左边和右边之间居中，必须添加两个尺寸并将其与参数关联。然后您可以定义参数方程 *dim2*=(*dim1*)/2，将从左边到柱的距离设置为壳体宽度的一半，如下图所示：

![](../graphics/ske-paramtech1.png)

**注意：**与参数 *dim1* 关联的距离尺寸在壳体特征的草图中定义，无法在柱的草图中编辑。当您对参考几何进行尺寸标注时，Abaqus/CAE 会将尺寸显示为洋红色以指示过度约束。要清除过度约束，您必须使 *dim1* 成为参考尺寸。Abaqus/CAE 在参考尺寸的值周围放置括号（如果尺寸与参数关联，则在参数名称周围放置括号），如果尺寸量发生变化，Abaqus/CAE 会自动更新它们的值。有关更多信息，请参见["编辑尺寸，"第 20.13 节](pt03ch20hla01.md)。

您可以修改草图中的任何尺寸或参数。详细说明分别请参见["编辑尺寸，"第 20.13 节](pt03ch20hla01.md)和["添加和编辑参数，"第 20.12.5 节](pt03ch20s12hlb05.md)。
