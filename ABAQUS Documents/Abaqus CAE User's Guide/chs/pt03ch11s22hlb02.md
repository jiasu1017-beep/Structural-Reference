# 11.22.4 添加壳放样特征





从主菜单栏选择**形状**![](../graphics/images/arrow.gif)**壳**![](../graphics/images/arrow.gif)**放样**，向当前视口中的零件添加壳放样特征。您只能向三维零件添加壳放样特征。

您通过从选定边缘创建两个或多个截面并定义一个或多个放样路径来添加壳放样特征。

您可以允许Abaqus/CAE使用平滑路径定义单个放样路径，以连接每个放样截面的中心。

**添加壳放样特征：**

1. 从主菜单栏，选择**形状**![](../graphics/images/arrow.gif)**壳**![](../graphics/images/arrow.gif)**放样**。
2. 通过从视口中的零件选择边缘来创建放样截面。
3. 切换**保持内部边界**。
4. 完成创建放样截面后，单击**编辑放样**对话框中的**过渡**选项卡。
5. 执行以下操作之一：单击**选择路径**或**指定切线**。
6. 单击**预览**。
7. 单击**完成**创建放样并关闭**编辑放样**对话框。
