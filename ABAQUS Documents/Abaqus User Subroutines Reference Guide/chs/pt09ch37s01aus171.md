### 概述

用接触对模拟的压力渗透加载：
- 模拟流体在两个接触结构之间的穿透；和
- 允许流体从表面上的多个位置渗透。

### 定义接触体之间的压力渗透加载

分布式压力渗透加载允许模拟流体穿透到两个接触体之间的表面并对表面施加垂直于表面的流体压力。基于单元的接触表面用于模拟体之间的相互作用（见["接触相互作用分析概述，" 第36.1.1节"](pt09ch36s01abo33.md)）。表面被建模为从属和主接触表面（见["Abaqus/Standard中的接触公式，" 第38.1.1节"](pt09ch38s01aus177.md)）。

可以使用任何接触公式。

形成接头的体可以都是变形的，如螺纹连接器的情况；或者一个可以是刚性的，如使用软垫圈作为较硬结构之间的密封时的情况。您指定暴露于流体压力的节点、流体压力的大小以及低于该值流体渗透开始发生的临界接触压力。请参阅["基于表面的压力渗透加载，" Abaqus理论指南第6.4.1节"](../stm/stm-link.md#stm-ldc-presspenetrationsurf)，了解更多详情。

| **输入文件用法：** | ``` [*PRESSURE PENETRATION*](../key/key-link.md#usb-kws-hpressurepenetration), SLAVE=*slave1*, MASTER=*master1* *slave surface node or node set*, *master surface node or node set*, *magnitude*, *critical contact pressure* ``` |
| --- | --- |
| | 如果指定节点集，在二维中它只能包含一个节点；在三维中它可以包含任意数量的节点。 |

| **Abaqus/CAE用法：** | 相互作用模块：**创建相互作用**：**表面-表面接触（Standard）**，**名称**：*contact_interaction_name*；选择主表面和从表面**创建相互作用**：**压力渗透**；**接触相互作用**：*contact_interaction_name*，**主表面上的区域**：选择面、边缘或点，**从表面上的区域**：选择面、边缘或点，**临界接触压力**：*critical contact pressure*，**流体压力**：*magnitude* |
| --- | --- |

#### 指定压力渗透标准

使用单一的基于从节点的渗透标准。流体将从暴露于流体的一个或多个位置穿透到接触体之间的表面，直到达到接触压力大于指定临界值的点，切断流体的进一步渗透。

#### 指定流体压力的渗透时间

当满足流体压力渗透标准时，流体压力被施加到垂直于表面的方向。如果立即施加完整的当前流体压力，可能导致接触表面附近应变的大变化，从而导致收敛困难。对于大应变问题，也可能发生严重的网格扭曲。为了确保平滑的求解，流体压力在从零压力渗透加载到当前完整大小的时间段内线性斜升。

您可以指定流体压力渗透加载在新穿透表面段上达到当前完整大小所需的时间段。如果在渗透后立即测量的累积增量大小大于渗透时间，则将施加完整的当前流体压力渗透加载；否则，新穿透表面段上的流体压力将在渗透时间段内线性斜升到当前大小，可能跨越多个增量。当渗透时间等于0时，一旦满足流体压力渗透标准，就立即施加当前流体压力。默认渗透时间选择为总步骤时间的0.001。渗透时间在线性扰动分析中被忽略。

| **输入文件用法：** | ``` [*PRESSURE PENETRATION*](../key/key-link.md#usb-kws-hpressurepenetration), PENETRATION TIME=*n* ``` |
| --- | --- |

| **Abaqus/CAE用法：** | 相互作用模块：**创建相互作用**：**压力渗透**；**渗透时间**：*n* |
| --- | --- |

#### 指定暴露于流体压力的节点

流体可以从表面的一个或多个位置渗透。您必须在接触体的从表面上识别一个节点或节点集，以定义表面暴露于流体压力的位置。在二维中，如果主表面不是解析刚性表面（见["解析刚性表面定义，" 第2.3.4节"](pt01ch02s03aus19.md)），您还必须在主表面上识别一个节点或节点集，以定义表面暴露于流体压力的位置。如果表面上有多个位置暴露于流体，您可以指定多个节点或节点集。如果这些节点或节点集在从表面上，它们始终会受到压力渗透加载，无论其接触状态如何。然后流体开始从这些节点或节点集穿透到两个接触体之间的表面。

#### 指定施加的流体压力

您必须定义流体压力的参考大小。您可以通过引用振幅曲线来定义步骤中流体压力的变化。默认情况下，参考大小在步骤开始时立即施加或在整个步骤中线性斜升，取决于分配给步骤的振幅变化（见["定义分析，" 第6.1.2节"](pt03ch06s01abo05.md)）。

流体压力渗透加载将基于增量开始时的压力渗透标准应用于单元表面，并且即使在该增量期间流体进一步渗透，它也将保持在该增量期间恒定。在二维中使用节点积分方案将分布式流体压力渗透加载集成在单元上，而在三维中使用高斯积分方案；分布式流体压力在单元上的变化将由单元节点处的载荷大小决定。

| **输入文件用法：** | 使用以下选项定义步骤中流体压力的变化： |
| --- | --- |
| | ``` [*PRESSURE PENETRATION*](../key/key-link.md#usb-kws-hpressurepenetration), AMPLITUDE=*name* ``` |

| **Abaqus/CAE用法：** | 相互作用模块：**创建相互作用**：**压力渗透**；**振幅**：*name* |

##### 移除或修改压力渗透加载

在将压力渗透加载应用于单元表面后，即使表面之间的接触重新建立，它们也不会自动移除。然而，在每个新步骤中，流体压力渗透加载可以以类似于定义分布式载荷的方式进行修改或完全重新定义（见["施加载荷：概述，" 第34.4.1节"](pt07ch34s04aus120.md)）。

| **输入文件用法：** | 使用以下选项修改在前面的步骤中施加的流体压力渗透加载： |
| --- | --- |
| | ``` [*PRESSURE PENETRATION*](../key/key-link.md#usb-kws-hpressurepenetration), OP=MOD (default) ``` 在这种情况下，必须在数据行上指定暴露于流体压力的从节点。如果主表面不是解析刚性表面，对于平面或轴对称模型，还必须在数据行上指定暴露于流体压力的主节点。使用以下选项移除所有流体压力渗透加载，并可选择指定新的流体压力渗透加载： ``` [*PRESSURE PENETRATION*](../key/key-link.md#usb-kws-hpressurepenetration), OP=NEW ``` 当使用OP=NEW移除所有流体压力渗透加载时，不需要数据行。但是，当使用OP=NEW指定新的流体压力渗透加载时，必须在数据行上指定暴露于流体压力的节点。定义新暴露节点时必须使用OP=NEW。此外，当使用OP=NEW重新指定先前定义的压力渗透加载时，流体压力加载将首先恢复到最后已知配置，即使接触状态随后已更改。 |

| --- | --- |
| | 相互作用模块：**相互作用管理器**：选择相互作用，**编辑** 使用以下选项移除在前面的步骤中施加的压力渗透：相互作用模块：**相互作用管理器**：选择相互作用，**停用** |

#### 指定临界机械接触压力

为了考虑接触表面上的微凸体，引入了临界接触压力，低于该值流体渗透开始发生。该值越高，流体越容易渗透。临界接触压力的默认值为零，在这种情况下，仅在失去接触时才会发生流体渗透。

#### 在线形扰动分析中使用

在完全非线性分析期间，可以不时地通过在一般分析步骤之间包括线性扰动步骤来执行线性扰动分析。因为在线性扰动分析期间接触条件不能改变，所以流体不会进一步穿透到表面，它保持为在基础状态中定义的形式。然而，在先前的一般分析步骤中施加的流体压力大小可以在线性扰动分析步骤中修改。在矩阵生成（见["生成矩阵，" 第10.3.1节"](pt04ch10s03at32.md)）和稳态动态分析（直接或模态——见["直接解稳态动态分析，" 第6.3.4节"](pt03ch06s03at09.md)和["基于模态的稳态动态分析，" 第6.3.8节"](pt03ch06s03at13.md)）中，您可以指定载荷的真实（同相）部分和虚部（异相）。

#### 压力渗透加载的限制

承受压力渗透加载的每个从属表面必须是连续的，不能是闭合环。压力渗透加载不能与基于节点的从属表面一起使用。在任何增量施加的压力渗透加载基于该增量开始时的接触状态。因此，您应该谨慎解释在接触状态发生变化的增量结束时的结果。建议使用小时间增量以获得准确的结果。

当流体在解析刚性表面和变形表面之间的接触体之间渗透时，不会向解析刚性表面施加压力渗透加载。因此，解析刚性表面上的参考节点应在所有方向上被约束。为了考虑流体压力渗透加载对刚性表面的影响，解析刚性表面应该被替换为基于单元的刚性表面。

当具有不同压力加载的流体同时从表面上多个位置穿透到单元时，将施加流体压力加载的最大值。

在大位移分析中，压力渗透加载引入非对称载荷刚度矩阵项。使用分析步骤的非对称矩阵存储和求解方案可以提高平衡迭代的收敛率。请参阅["定义分析，" 第6.1.2节"](pt03ch06s01abo05.md)，了解更多关于非对称矩阵存储和求解方案的信息。

三维压力渗透仅支持实体、壳、圆柱和刚性单元。
