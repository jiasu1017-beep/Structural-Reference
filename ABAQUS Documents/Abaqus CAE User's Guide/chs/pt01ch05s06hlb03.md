# 5.6.3 旋转视图





使用**视图操作**工具栏中的旋转工具 ![](../graphics/ico_viewRotate.png) 在视口中旋转视图。在使用旋转工具时按住 **[Shift]** 可以访问替代旋转模式。替代模式围绕自身旋转相机，而不是围绕视图中心或所选旋转中心旋转相机。替代模式在你使用电影相机模式并将相机放置在模型内部使模型围绕相机时最有用（有关更多信息，请参阅["旋转视图工具，" 第 5.2.3 节](pt01ch05s02s03.md)）。你也可以在每个视口中指定一个点，或在每个视口中选择一个位置作为旋转中心。使用**图形选项**对话框中的**旋转后自动拟合**选项，你可以控制 Abaqus/CAE 是否在旋转时将模型重新调整以适应视口。

**注意：**你不能旋转 *X–Y* 绘图。

如果当前视口链接到其他视口，Abaqus/CAE 也会旋转会话中所有链接视口中的对象。有关更多信息，请参阅["链接视口以进行视图操作，" 第 4.5 节](pt01ch04s05.md)。

**要旋转视图：**

1. 从**视图操作**工具栏中，单击旋转工具 ![](../graphics/ico_viewRotate.png) 进入旋转模式。**提示：**你也可以从主菜单中选择****View**![](../graphics/images/arrow.gif)**Rotate****，或按 **[F3]**。
2. 默认情况下，Abaqus/CAE 围绕视口中心旋转视图。你可以使用以下方法之一更改旋转中心：
   - 将光标定位在模型上或视口中的任何位置，单击鼠标按钮 3，然后选择**设置为旋转中心**。
   - 在提示区域中单击**选择**。从视口中高亮显示的顶点中选择旋转中心，或输入坐标以指定一个点。在可视化模块中，你可以选择一个节点，旋转中心将保留在该节点上，无论是在未变形还是变形模型状态下。使用此方法时，只有当视口中存在现有几何体时才能选择旋转中心。如果你在草图器中工作，草图点将不可用于选择。
   - 单击鼠标按钮 3 并选择**使用默认旋转中心**，或在提示区域中选择**使用默认**以返回默认（视口中心）旋转方法。你选择的旋转中心会在视口中保留，直到你在视口中显示另一个对象、选择新的旋转中心，或返回默认旋转方法。
3. 将光标定位在你要更改视图的视口中。视口中会出现一个大圆，光标变为向右的箭头。如果你在视口中选择了旋转中心，它会高亮显示。
4. 沿任何方向拖动光标。拖动光标时视图会旋转，橡皮筋线条表示旋转量和方向。**提示：**通过执行一系列小旋转而不是一次大旋转通常更容易达到所需的方向。要围绕屏幕法向轴旋转视图，请将光标移到圆外并顺时针或逆时针拖动。
5. 重复步骤 2、3 和 4，直到达到所需的视图。
6. 要退出旋转模式，请执行以下操作之一：
   - 单击鼠标按钮 2。
   - 单击提示区域中的取消按钮 ![](../graphics/afxI_cancel.png)。
   - 单击旋转工具。
   - 单击任何其他视图操作工具。
   **提示：**使用[循环](pt01ch05s06hlb07.md)视图操作工具返回上一个视图。

**要在旋转时将视图重新调整以适应视口：**

1. 从主菜单栏中，选择****View**![](../graphics/images/arrow.gif)**Graphics Options****。**图形选项**对话框出现。
2. 切换**旋转后自动拟合**开启，以在旋转时自动将视图重新调整以适应视口；关闭它以禁用旋转期间的自动重新调整。
3. 单击**确定**实施更改并关闭对话框。你的更改会在会话期间保存。
