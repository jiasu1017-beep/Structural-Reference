# A.1 系统软件





本附录包含 Abaqus 可选和必需的系统软件先决条件的一般信息。支持平台和其他系统要求的确定列表见 Dassault Systmes Program Directory。

Program Directory 是类似于发布说明的附加文档，可从 [http://media.3ds.com/support/progdir](http://media.3ds.com/support/progdir) 获取。选择 SIMULIA Abaqus 作为产品**系列**，选择 Abaqus 6.14 作为**级别**，然后在左侧窗格中选择**先决条件**。访问需要支持合同和私有登录账户（参见 [https://iam.3ds.com/self_service/login/service/websupport](https://iam.3ds.com/self_service/login/service/websupport/)/）。

某些 Abaqus 功能需要额外的软件工具或组件：
- FORTRAN 或 C++ 编译器是编译用户子程序所必需的，但如果不在用户子程序的情况下运行 Abaqus，则不需要。
- Message Passing Interface (MPI) 组件是使用 Abaqus/Standard 中基于 MPI 的并行执行、使用 Abaqus/Explicit 中的域级并行化或运行 Abaqus/CFD 中的任何作业（无论 CPU 数量如何）所必需的。请参阅["Visual C++ 和 MPI 库，"第 2.4.1 节](ch02s04.md#sgb-chp-install-prod-libraries)，并查看 Program Directory 以获取支持的 MPI 软件和硬件互连。
- Linux 平台需要 X-Windows。Abaqus/CAE 和 Abaqus/Viewer 运行所在的计算机需要 OpenGL 共享库。
