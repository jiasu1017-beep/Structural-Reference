# 35.4.2 复杂部件的中面建模





此示例使用多种技术和工具为增强结构组件创建中面模型。

**实体模型**

本示例中的模型是[图35-11](pt04ch35s04s02.md#adv-mid-cmplxmodel)所示的结构梁。加固肋、不同的厚度和不对称形状不允许简单的梁截面表示。部件的复杂性及其薄截面使其成为用中面模型替换的良好候选。与前面的示例一样，通过使用壳模型进行网格划分而不是薄实体截面，将改善弯曲性能。

**图35-11** 加固梁的实体模型。

![](../graphics/adv-mid-cmplxmodel.png)

**分配中面区域**

使用Part模块中的**Assign Midsurface Region**工具从梁的活动表示中移除几何体，并创建原始实体几何体的参考表示，[图35-12](pt04ch35s04s02.md#adv-mid-cmplxrefrep)所示。

**图35-12** 梁的参考表示。

![](../graphics/adv-mid-cmplxrefrep.png)

参考表示是原始部件的抽象表示。它保留部件的原始几何体，但不能用于分析。参考表示默认显示在Part模块中；您可以使用主工具栏中与可见对象工具一起的**Show Reference Representation**工具![](../graphics/ico_referenceRepToggle.png)切换关闭和开启。更多信息，请参见["理解参考表示"，第35.2节](pt04ch35s02.md)和["分配中面区域"，第35.5.1节](pt04ch35s05s01.md)。

**创建壳表示**

您必须创建可由Abaqus分析梁的壳表示。创建此部件的壳需要多个步骤和工具。生产精确壳表示可能有多种同等有效的方法。以下步骤使用Geometry Edit工具集中的工具来创建新的壳面。

1. 使用偏移面![](../graphics/ico_offsetFaces.png)工具为梁的左上垂直侧创建壳面。从参考表示中选择面以创建偏移壳。使用**Auto Select**选择目标面。要偏移的面和目标面分别如图35-13中的红色和品红色所示。
   **图35-13** 选择面以创建左上侧的偏移。
   ![](../graphics/adv-mid-cmplxoffset1.png)
   要偏移的面是朝向后面的两个外部垂直面。目标面是九个大部分不相连的面，组成同一梁壁的内表面。使用**Fraction distance to closest point on face**方法和条目`0.5`来创建偏移，偏移量为较薄主要部分沿顶部厚度的一半。使用此选项可以防止左侧和底部较厚部分产生大于较薄部分的偏移。此步骤使用**Auto trim to reference representation**选项。此操作在偏移过程中延伸面，然后沿参考表示的边缘修剪它们。当选择多个偏移和目标面时，自动修剪可能会失败；如果自动修剪失败，Abaqus/CAE会显示指示失败的警告。您可以使用extend faces工具将新面延伸到参考表示的边缘。结果偏移面如图35-14所示。
   **图35-14** 结果偏移面。
   ![](../graphics/adv-mid-cmplxoffset1result.png)
2. 重复步骤1为梁的另一侧创建偏移面。结果壳模型现在包含梁的两个外表面，[图35-15](pt04ch35s04s02.md#adv-mid-cmplxoffset2result)所示。
   **图35-15** 第一和第二步的偏移面。
   ![](../graphics/adv-mid-cmplxoffset2result.png)
3. 为连接两侧的四个水平截面创建偏移面。您可以通过选择四个顶部面进行偏移并使用**Auto Select**选择相应的底部面来在单次偏移操作中创建面。在这种情况下，**Auto trim to reference representation**关闭，因此新面不会被延伸和修剪。水平面的壳模型如图35-16所示。（为清晰起见，已抑制步骤2中创建的面。）注意水平面与原始部件中垂直加固肋之间的间隙。类似的间隙存在于水平壳面与前两个步骤中创建的侧壁之间。
   **图35-16** 水平偏移面。
   ![](../graphics/adv-mid-cmplxoffset3result.png)
4. 重复步骤3为连接梁两侧的垂直肋创建偏移面。中面壳模型现在包含表示几乎所有实体几何体的面，[图35-17](pt04ch35s04s02.md#adv-mid-cmplxoffset4result)所示。但是，由于原始实体结构的厚度，大多数面之间存在间隙。
   **图35-17** ，面之间有间隙的中面模型。
   ![](../graphics/adv-mid-cmplxoffset4result.png)
5. 封闭垂直肋与梁侧面之间的间隙。使用extend faces ![](../graphics/ico_extendFaces.png)工具将六个垂直肋延伸以与侧面相交。所选面和目标面分别以品红色和黄色显示在[图35-18](pt04ch35s04s02.md#adv-mid-cmplxextend)中。
   **图35-18** 指定要延伸的面和目标面。
   ![](../graphics/adv-mid-cmplxextend.png)
   当您单击**OK**时，Abaqus/CAE更新高亮显示以指示面将被延伸的边缘，[图35-19](pt04ch35s04s02.md#adv-mid-cmplxextendedge)所示，并显示一个警告对话框，其中包含允许您接受选择、延伸所选面的所有边缘或取消延伸面过程的选项。
   **图35-19** 垂直面将被延伸的边缘。
   ![](../graphics/adv-mid-cmplxextendedge.png)
6. 使用extend faces ![](../graphics/ico_extendFaces.png)工具将水平面延伸到与梁侧面相交。在这种情况下，使用**Specify edges of faces to extend**方法和**Up to target faces**分别选择[图35-20](pt04ch35s04s02.md#adv-mid-cmplxextendedge2)中所示的边缘和面。当Abaqus/CAE高亮显示指示将被延伸的区域时，图中箭头所示的四个边缘从选择中移除，因为它们与目标面不完全匹配。单击**No**使用先前的边缘选择。
   **图35-20** 水平面将被延伸的边缘。
   ![](../graphics/adv-mid-cmplxextendedge2.png)
7. 封闭梁的水平连接截面和垂直连接截面之间的剩余间隙。您可以使用blend faces ![](../graphics/ico_blendFaces.png)工具或extend faces ![](../graphics/ico_extendFaces.png)工具来完成中面模型，封闭水平连接截面和垂直连接截面之间的剩余间隙。
   1. 使用extend faces ![](../graphics/ico_extendFaces.png)工具，使用[图35-21](pt04ch35s04s02.md#adv-mid-cmplxextendface)所示的选择。开启**Trim to extended underlying target surfaces**，以便Abaqus/CAE将垂直面延伸和修剪到与所选目标面的隐含交点。
      **图35-21** 填充垂直加固肋之间的间隙。
      ![](../graphics/adv-mid-cmplxextendface.png)
   2. 使用blend faces ![](../graphics/ico_blendFaces.png)工具填充剩余的三个间隙。该工具必须使用三次来填充模型中剩余的三个间隙。

**分配厚度**

所有原始实体几何体现在已被壳几何体替换。要完成模型，您应验证壳具有适当的厚度信息。单击assign thickness and offset工具![](../graphics/ico_assignThickness.png)。Abaqus/CAE高亮显示没有厚度数据的任何壳面。在这种情况下，由于壳面都是使用偏移、延伸和混合工具创建的，所有面都已分配了厚度数据。如果没有厚度的面，您可以选择每个面，然后在**Assign Thickness and Offset**对话框中使用**Compute thickness from opposite faces**方法，从参考表示中选择适当的顶部和底部面来创建缺失的厚度。

要使用壳厚度查看模型，您可以在**Part Display Options**对话框中开启**Render shell thickness**（更多信息，请参见["可视化壳厚度"，第35.5.3节](pt04ch35s05s03.md)）。如图35-22所示，结果视图包括原始实体模型中的厚度变化。

**图35-22** 显示厚度的壳面。

![](../graphics/adv-mid-cmplxthickassign.png)

**分配壳截面**

使用Property模块创建壳截面并将其分配给中面模型。当您创建壳截面时，可以输入壳厚度的任意值。当您随后将截面分配给壳时，您可以在**Edit Section Assignment**对话框中指定厚度和壳偏移从几何体计算。Abaqus/CAE忽略您为壳截面输入的厚度值，并使用您在Part模块中分配给面的厚度。更多信息，请参见["分配截面"，第12.15.1节](pt03ch12s15hlb01.md)。[图35-23](pt04ch35s04s02.md#adv-mid-cmplxsection)显示了在Property模块中截面分配后具有截面厚度的完整中面模型。几何体与[图35-22](pt04ch35s04s02.md#adv-mid-cmplxthickassign)中的相同。

**图35-23** 具有壳厚度的完整中面几何体。

![](../graphics/adv-mid-cmplxsection.png)

**为部件划分网格**

Abaqus/CAE在Mesh模块中将壳部件显示为粉色，以指示可以使用自由网格划分技术对其进行网格划分，[图35-24](pt04ch35s04s02.md#adv-mid-cmplxfreemesh)所示。

**图35-24** 自由网格划分可以应用于部件。

![](../graphics/adv-mid-cmplxfreemesh.png)

在设置种子和为部件划分网格之前，您可以应用自动虚拟拓扑以移除网格中不需要的小细节（更多信息，请参见["自动创建虚拟拓扑"，第75.6.1节](pt06ch75s06hlb01.md)）。默认自动虚拟拓扑设置应移除混合面边缘和其他会不必要地约束部件网格的小细节。

**注意：**如果相邻面的法线不一致，自动虚拟拓扑可能会失败。如果发生这种情况，返回Part模块并使用Geometry Edit工具集中的![](../graphics/ico_repairFaceNormals.png)工具来修复面法线。

应用默认种子和网格控制，并在部件上生成网格。生成的网格如图35-25所示，显示了壳厚度。

**图35-25** 具有壳厚度渲染的结果网格。

![](../graphics/adv-mid-cmplxmeshed.png)
