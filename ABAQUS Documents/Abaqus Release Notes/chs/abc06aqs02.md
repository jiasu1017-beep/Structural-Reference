# 6.2 具有各向异性材料的欧拉单元





**产品：** Abaqus/Explicit

**优势：** 现在可以使用各向异性材料进行耦合欧拉-拉格朗日分析。

**说明：** 对于涉及极端变形的分析，耦合欧拉-拉格朗日分析比传统拉格朗日分析更有效，因为单元不变形，材料可以流过单元面。该技术现在可用于涉及具有各向异性行为的材料的模型。
