# 20.9.1 Sketcher 自定义选项概述







使用 Sketcher 自定义选项来控制 Sketcher 的外观和行为。以下选项控制整个 Abaqus/CAE 会话期间 Sketcher 的行为：
- 光标是否[捕捉](pt03ch20s09hlb02.md)到网格。
- [预选](pt03ch20s09hlb03.md)是否处于活动状态。
- 是否显示尺寸。
- 是否显示参数的名称或值。
- 限制投影的共面实体数量。
- **自动尺寸**和**自动约束**工具可用尺寸和约束类型。
- 编辑或拖动对象时使用的[约束求解方法](pt03ch20s09hlb10.md)。
- 是否显示[草图约束](pt03ch20s09hlb10.md)。
- 是否在实体创建期间添加隐含约束。
- 是否显示[构造几何](pt03ch20s09hlb06.md)。
- 是否显示[背景图像](pt03ch20s09hlb11.md)。

以下选项随每个草图存储；它们控制当前草图和您创建的新草图的行为和外观：- 工作表大小、[网格](pt03ch20s09hlb04.md)间距和要显示的网格线数量。
- 工作表大小和网格间距是否自动控制。
- 网格是否[可见](pt03ch20s09hlb04.md)。
- 文本字体大小和[草图尺寸](pt03ch20s09hlb09.md)中显示的小数位数。

草图网格的原点和 *X* 轴的对齐也与每个草图一起存储。但是，网格原点和对齐设置仅控制当前草图的行为和外观；对于您创建的任何新草图，它们会重置为默认值。

**设置 Sketcher 选项：**

1. 从 Sketcher 工具箱底部，选择 Sketcher 自定义工具![](../graphics/ico_sketchOptions-nls.png)。出现**Sketcher 选项**对话框。
2. 在每个选项卡页面上设置所需的自定义选项。要获取详细帮助，请在对话框中的各个项目上请求上下文相关帮助。**提示：**您也可以点击**默认**将所有 Sketcher 选项恢复为默认设置。
3. 点击**确定**应用您的更改并关闭**Sketcher 选项**对话框。
