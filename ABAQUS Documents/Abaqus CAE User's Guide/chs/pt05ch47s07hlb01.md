# 47.8.2 自定义 X–Y 绘图标题的位置





### 47.8.2 自定义 X–Y 绘图标题的位置

使用 **绘图标题选项（Plot Title Options）** 中的 **位置（Position）** 选项将 X–Y 绘图标题放置在视口中的特定位置。

**要自定义 X–Y 绘图标题的位置：**

1. 找到 **位置（Position）** 选项。选择 ****选项（Options）**![](../graphics/images/arrow.gif)**XY 选项（XY Options）**![](../graphics/images/arrow.gif)**绘图标题（Plot Title）****；或点击 ![](../graphics/ico_xyPlotTitleOptions.png)，该图标位于可视化模块工具箱的 X–Y 绘图工具中。从出现的对话框中点击 **区域（Area）** 选项卡。
2. 切换 **嵌入（Inset）** 以在 X–Y 绘图内显示绘图标题图例。如果 **嵌入（Inset）** 关闭，Abaqus/CAE 会将绘图标题和 X–Y 绘图保持在视口的独立区域中。
3. 选择以下两种方法来定位绘图标题之一：**自动对齐（Auto-align）** 此选项使您可以将绘图标题定位在视口中九个位置之一：其中心、四个角中的任意一个，或沿四个边缘中的任意一个居中。展开 **对齐（Alignment）** 列表，然后选择列表中以图形方式描绘的选项之一。**手动（Manual）** 此选项使您可以将绘图标题定位在视口中的特定 X–Y 位置。拖动 **X** 和 **Y** 滑块可以在视口中水平或垂直移动标题。当这两个值都等于 `0` 时，绘图标题定位在视口的左下角；当这两个值都等于 `100` 时，绘图标题定位在视口的右上角。

# 47.8.3 为 X–Y 绘图标题添加边框和填充颜色





### 47.8.3 为 X–Y 绘图标题添加边框和填充颜色

使用 **绘图标题选项（Plot Title Options）** 中的 **边框（Bounding Box）** 选项在 X–Y 绘图标题周围绘制矩形边框，或用背景色填充绘图标题后面的区域。

**要为 X–Y 绘图标题添加边框和填充颜色：**

1. 找到 **边框（Bounding Box）** 选项。选择 ****选项（Options）**![](../graphics/images/arrow.gif)**XY 选项（XY Options）**![](../graphics/images/arrow.gif)**绘图标题（Plot Title）****；或点击 ![](../graphics/ico_xyPlotTitleOptions.png)，该图标位于可视化模块工具箱的 X–Y 绘图工具中。从出现的对话框中点击 **区域（Area）** 选项卡。**边框（Bounding Box）** 选项出现在对话框底部。
2. 从对话框顶部的 **图表（Charts）** 字段中选择一个或多个 X–Y 绘图。
3. 执行以下步骤来显示和修改边框：1. 切换 **显示边框（Show border）** 以在图例周围显示边框。2. 为边框线选择颜色。1. 点击颜色样本 ![](../graphics/usi-colorsample.png)。Abaqus/CAE 显示 **选择颜色（Select Color）** 对话框。2. 使用 **选择颜色（Select Color）** 对话框中的方法选择新颜色。有关更多信息，请参阅 ["自定义颜色，" 第 3.2.9 节"](pt01ch03s02s09.md)。3. 点击 **确定（OK）** 关闭 **选择颜色（Select Color）** 对话框。边框线变为所选颜色。
4. 控制边框填充颜色：1. 切换 **填充（Fill）** 以在边框中显示背景颜色。2. 选择要在边框背景中显示的填充颜色。1. 点击颜色样本 ![](../graphics/usi-colorsample.png)。Abaqus/CAE 显示 **选择颜色（Select Color）** 对话框。2. 使用 **选择颜色（Select Color）** 对话框中的方法选择新颜色。有关更多信息，请参阅 ["自定义颜色，" 第 3.2.9 节"](pt01ch03s02s09.md)。3. 点击 **确定（OK）** 关闭 **选择颜色（Select Color）** 对话框。边框背景变为所选颜色。

![](../graphics/images/black4rule.gif)有关相关信息，请点击以下项目：- ["选择要自定义的一条或多条 X–Y 曲线，" 第 47.6.2 节"](pt05ch47s05s01.md)

### 47.8.1 自定义 X–Y 绘图标题的内容

使用 **绘图标题选项（Plot Title Options）** 中的 **文本（Text）** 选项来为 X–Y 绘图创建和显示自定义标题。

**要自定义 X–Y 绘图的绘图标题内容：**

1. 找到绘图标题的 **文本（Text）** 选项。选择 ****选项（Options）**![](../graphics/images/arrow.gif)**XY 选项（XY Options）**![](../graphics/images/arrow.gif)**绘图标题（Plot Title）****；或点击 ![](../graphics/ico_xyPlotTitleOptions.png)，该图标位于可视化模块工具箱的 X–Y 绘图工具中。

5. 为 X–Y 绘图标题选择颜色。1. 点击颜色样本 ![](../graphics/usi-colorsample.png)。Abaqus/CAE 显示 **选择颜色（Select Color）** 对话框。2. 使用 **选择颜色（Select Color）** 对话框中的方法选择新颜色。有关更多信息，请参阅 ["自定义颜色，" 第 3.2.9 节"](pt01ch03s02s09.md)。3. 点击 **确定（OK）** 关闭 **选择颜色（Select Color）** 对话框。绘图标题文本变为所选颜色。

# 47.9.1 显示或隐藏 X–Y 绘图图例





### 47.9.1 显示或隐藏 X–Y 绘图图例

X–Y 绘图图例提供了图例，用于关联绘图中包含的每条曲线及其所代表的 X–Y 数据对象。每个图例条目包含一个线段，其颜色和样式与绘图中的显示相同。图例条目还包括文本，默认情况下，该文本说明曲线所表示的 X–Y 数据对象的描述。您可以使用 **图表图例选项（Chart Legend Options）** 显示或隐藏图例。

**要显示或隐藏 X–Y 绘图图例：**

1. 找到 **图表图例选项（Chart Legend Options）**。选择 ****选项（Options）**![](../graphics/images/arrow.gif)**XY 选项（XY Options）**![](../graphics/images/arrow.gif)**图表图例（Chart Legend）****；或点击 ![](../graphics/ico_xyChartLegendOptions.png)，该图标位于可视化模块工具箱的 X–Y 绘图工具中。**提示：**当显示 X–Y 绘图时，您还可以通过双击视口中的图例直接打开此对话框。当活动过程生效时，您无法直接从视口中选择 X–Y 绘图中的组件；在从 X–Y 绘图中选择项目之前，请取消当前过程。
2. 如果 **内容（Contents）** 选项卡尚未选择，请点击它。
3. 切换 **显示图例（Show legend）** 以显示图例，或关闭它以隐藏图例。
4. 点击 **关闭（Dismiss）** 关闭 **图表图例选项（Chart Legend Options）** 对话框。
