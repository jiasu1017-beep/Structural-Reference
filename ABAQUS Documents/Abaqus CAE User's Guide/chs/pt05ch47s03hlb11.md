# 47.4.13 对X–Y数据对象执行曲线拟合







### 47.4.13 对X–Y数据对象执行曲线拟合

使用 **fit** 函数对[之前保存的](pt05ch47s02hlb07.md) *X–Y* 数据对象（有序对的集合）进行运算，生成一个新的 *X–Y* 数据对象。新的 *X–Y* 数据对象包含沿最佳拟合之前保存的 *X–Y* 数据对象的曲线以规则间隔分布的点。**fit** 函数提供两种曲线拟合算法：线性最小二乘拟合和样条插值。

[图47-10](pt05ch47s03hlb11.md#usv-xyplot-fit2) 和[图47-11](pt05ch47s03hlb11.md#usv-xyplot-fit) 显示了使用 **fit** 函数（每种曲线拟合算法一个）生成的 *X–Y* 曲线图示例。

**图47-10** 使用 **fit** 函数（选择线性最小二乘选项）生成的 *X–Y* 曲线图。

![](../graphics/usv-xyplot-fit2.png)

**图47-11** 使用 **fit** 函数（选择样条插值选项）生成的 *X–Y* 曲线图。

**对X–Y数据对象执行曲线拟合：**

1. 找到 **Operate on XY Data** 对话框。从主菜单栏，选择 ****Tools**![](../graphics/images/arrow.gif)**XY Data**![](../graphics/images/arrow.gif)**Create****. 在出现的对话框中点击 **Operate on XY data**；然后点击 **Continue**。**Operate on XY Data** 对话框出现。
2. 从列出的 **Operators** 中，点击 **fit(X)**。**fit** 函数出现在表达式窗口中，默认选择线性最小二乘拟合选项。
3. 从 **XY Data** 选项中，点击要操作的 *X–Y* 数据对象的名称，然后点击 **Add to Expression**。您可以从本次会话中之前保存的所有 *X–Y* 数据对象中选择（在 **XY Data** 字段中按字母顺序列出）。*X–Y* 数据对象名称出现在表达式窗口中 **fit** 函数的括号内。
4. 要继续构建表达式，将光标定位在表达式窗口中，然后输入或选择要包含的函数、运算符和 *X–Y* 数据。您可以通过包含以下更改来自定义 **fit** 函数本身：- 如果要执行样条插值，请将 `typeOfFit` 选项设置为 `SPLINE_INTERPOLATION`。- 如果要指定为新 *X–Y* 数据对象创建的自定义点数，请追加 `numFitPoints=*number*`。默认情况下，Abaqus/CAE 使用2个点定义线性最小二乘拟合，使用100个点定义样条插值。
5. 要评估并显示表达式，点击 **Plot Expression**。
6. 要[保存](pt05ch47s02hlb07.md)您的新 *X–Y* 数据对象，点击 **Save As**，然后在出现的对话框中提供名称。保存数据对象使其可用于本次会话中的未来操作以及包含多个数据对象的 *X–Y* 曲线图。
7. 完成后，点击 **Cancel** 关闭对话框。
