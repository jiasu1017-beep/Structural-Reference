# 55.3.4 自定义单元和表面边缘样式





您可以自定义Abaqus/CAE显示单元和表面边缘的样式和粗细。例如,[图55-4](pt05ch55s03hlb04.md#viw-edgestyle)左侧显示了默认单元边缘的曲线图,右侧显示了自定义边缘的曲线图。边缘样式选项是曲线图状态无关的;也就是说,您为所有未变形、变形、符号和材料取向曲线图设置一次。您也可以设置Abaqus/CAE在变形形状叠加在未变形形状上时将应用于未变形形状的边缘样式选项。

**图55-4** 显示默认和自定义单元边缘的模型。

![](../graphics/viw-edgestyle.png)

**自定义单元和表面边缘的步骤:**

1. 找到公共或叠加"**Edge Attributes**"选项。公共边缘属性始终应用于变形形状;它们适用于在任何曲线图状态(未变形、变形、云图、符号或材料取向)中单独绘制未变形形状时。当未变形形状与变形形状一起绘制时,应用叠加边缘属性。从主菜单栏,选择 ****Options**![](../graphics/images/arrow.gif)**Common**** 或 **Superimpose**。在出现的对话框中点击"**Other**"选项卡;然后在出现的对话框中点击"**Color & Style**"选项卡。"**Edge Attributes**"选项在"**Color & Style**"页面底部。
2. 选择单元和表面边缘的样式: 1. 点击"**Style**"按钮显示边缘样式选项。 2. 点击您想要的边缘样式。"**Style**"按钮上出现指定的边缘样式。
3. 选择单元和表面边缘的粗细: 1. 点击"**Thickness**"按钮显示边缘粗细选项。 2. 点击您想要的边缘粗细。"**Thickness**"按钮上出现指定的边缘粗细。
4. 点击"**Apply**"实施您的更改。单元和表面边缘样式根据您的规格进行更改。默认情况下,您的更改会在会话期间保存,并会影响当前视口和从当前视口创建的任何新视口中的所有后续曲线图。如果要为后续会话保留您应用的更改,请将它们保存到文件。更多信息,请参阅 ["为后续会话保存自定义," 第55.1.1节](pt05ch55s01s01.md)。
