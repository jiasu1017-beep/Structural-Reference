# 4.6.2 显示背景图像





你选择的视口背景图像只显示在当前视口中，并且在切换模块时该图像仍保持可见。由于背景图像是视口特定的，你可以在每个视口中显示不同的图像，这些图像可以帮助你完成不同的任务，或使你更快地区分视口。

虽然模块范围的背景图像出现在所有模块中，但其外观可能会被草图器背景图像（在草图模块中）或背景电影（在可视化模块中）遮挡。当这些视口装饰处于活动状态时，它们会显示在所选模块中模块范围的背景图像之上。你也可以将背景图像包含在视口的打印输出中；有关详细信息，请参阅["选择要打印的图像部分，" 第 8.2.2 节](pt01ch08s02hlb02.md)。

**要在视口中显示背景图像或电影：**

1. 从主菜单栏中，选择****View**![](../graphics/images/arrow.gif)**Image/Movie Options****。**图像/电影选项**对话框打开，并选中**图像**页面。
2. 切换**显示图像**以在视口中显示或隐藏背景图像。
3. 选择要显示的图像文件：
   - 要显示为会话定义的图像，请展开**图像名称**列表并选择图像名称。
   - 要添加新图像，请单击![](../graphics/ico_imageSmall.png)；然后在出现的对话框中输入名称并指定文件位置。你可以直接在**文件名**字段中输入文件位置，或单击![](../graphics/ico_fileOpenBlue.png)在**选择图像文件**对话框中导航到它。
4. 单击**确定**应用更改并关闭**图像/电影选项**对话框。如有需要，你也可以自定义背景图像的比例、位置或透明度；请参阅["自定义背景图像和电影的外观，" 第 4.6.4 节](pt01ch04s06hlb03.md)。
