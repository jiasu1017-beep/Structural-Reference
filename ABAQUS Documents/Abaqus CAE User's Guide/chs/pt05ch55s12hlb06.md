# 55.12.6 控制壳厚度显示







如果您使用壳元素对分析中的相对较薄组件进行建模，则可以使用 ****视图**![](../graphics/images/arrow.gif)**ODB 显示选项**** 在后处理期间查看模型中这些壳元素的真实厚度。壳厚度渲染在 Visualization 模块中适用于所有绘图状态。对于变形和轮廓绘图，Abaqus/CAE 在更改变形比例因子时会缩放变形的位移分量，但不会缩放变形的旋转分量。对于轮廓图，Abaqus/CAE 会根据您选择的活动截面点位置应用颜色。如果当前活动的是顶截面点或底截面点，Abaqus/CAE 会显示整个壳厚度内该截面点的轮廓；如果顶和底截面点当前都活动，则 Abaqus/CAE 会通过壳厚度从顶部值到底部值创建线性轮廓渐变。如果您的模型中使用的壳截面使用元素分布或节点分布来定义可变厚度，Abaqus/CAE 不会显示可变厚度，而是使用输出数据库中所选截面指定的厚度值。

在尺寸优化后显示壳厚度可以让您查看优化过程中结构设计区域中壳厚度的变化。

Abaqus/CAE 使用模型在未变形状态下指定的壳厚度定义来确定后处理的壳厚度。如果您的模型在分析过程中经历了大变形或大旋转，壳厚度可能无法在变形绘图状态中很好地显示。

显示壳厚度使您能够检查壳几何相对于模型其余部分的厚度。您可以应用比例因子来减少或增加会话中壳厚度的显示。[图 55–20](pt05ch55s12hlb06.md#uss-dsp-shellthickness-example) 显示了模型的比例因子更改效果。 

**图 55–20** 从左到右：壳厚度比例因子设置为 0.5、1（默认）和 2。

![](../graphics/usv-gen-shell-thickness-graphic.png)

Abaqus/CAE 仅对三维壳元素渲染壳厚度；轴对称壳元素（如 SAX1 元素）不显示厚度。显示壳厚度时，Abaqus/CAE 还会渲染壳几何的边缘，除非视口中显示了视图切割。Abaqus/CAE 根据当前的颜色编码和透明度设置渲染壳厚度。当这些设置更改时，壳厚度的颜色和透明度也会更改。

**控制壳厚度显示：**

1. 找到 **渲染壳厚度** 选项。从主菜单栏，选择 ****视图**![](../graphics/images/arrow.gif)**ODB 显示选项****。在出现的对话框中，点击 **常规** 选项卡。
2. 从对话框底部切换 **渲染壳厚度** 以显示模型中壳截面的壳厚度。
3. 如果需要，应用 **比例因子** 到壳厚度以增加或减小其厚度。默认值为 1，它会产生壳厚度设置的逼真渲染。
4. 点击 **确定** 实现您的更改并关闭对话框。Abaqus/CAE 会以适当的厚度显示所选部件或装配中的壳截面。默认情况下，您的更改会在会话期间保存。如果您希望保留所做的更改以供后续会话使用，请将它们保存到文件。有关更多信息，请参见 ["保存自定义设置以供后续会话使用，" 第 55.1.1 节](pt05ch55s01s01.md)。
