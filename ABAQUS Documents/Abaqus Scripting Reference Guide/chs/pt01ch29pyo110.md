# 29.110 VolumetricTestData 对象

VolumetricTestData 对象提供体积测试数据。

**访问**

```
import material
mdb.models[*name*].materials[*name*].hyperelastic.volumetricTestData
mdb.models[*name*].materials[*name*].hyperfoam.volumetricTestData
mdb.models[*name*].materials[*name*].viscoelastic.volumetricTestData
import odbMaterial
session.odbs[*name*].materials[*name*].hyperelastic.volumetricTestData
session.odbs[*name*].materials[*name*].hyperfoam.volumetricTestData
session.odbs[*name*].materials[*name*].viscoelastic.volumetricTestData
```

### 29.110.1 VolumetricTestData(...)

此方法创建 VolumetricTestData 对象。

**路径**

```
mdb.models[*name*].materials[*name*].hyperelastic.VolumetricTestData
mdb.models[*name*].materials[*name*].hyperfoam.VolumetricTestData
mdb.models[*name*].materials[*name*].viscoelastic.VolumetricTestData
session.odbs[*name*].materials[*name*].hyperelastic.VolumetricTestData
session.odbs[*name*].materials[*name*].hyperfoam.VolumetricTestData
session.odbs[*name*].materials[*name*].viscoelastic.VolumetricTestData
```

**必需参数**

*table*

Float 元组序列，指定下述项目。

**可选参数**

*volinf*

`None` 或 Float，指定归一化体积值，该值取决于 [Viscoelastic](pt01ch29pyo106.md) 对象的 *time* 成员的值。默认值为 `None`。

如果 *time*=RELAXATION_TEST_DATA，*volinf* 指定长期归一化体积模量 ![](../graphics/ker_eqn00113.gif) 的值。

如果 *time*=CREEP_TEST_DATA，*volinf* 指定长期归一化体积柔量 ![](../graphics/ker_eqn00114.gif) 的值。

此参数仅对粘弹性材料模型有效。

*smoothing*

`None` 或 Int，指定平滑值。如果 *smoothing*=`None`，则不使用平滑。默认值为 `None`。

*temperatureDependency*

Boolean，指定数据是否依赖于温度。默认值为 OFF。

*dependencies*

Int，指定场变量依赖项的数量。默认值为 0。

**表格数据**

对于超弹性或超泡沫材料模型，表格数据指定以下内容：
- 压力，![](../graphics/ker_eqn00092.gif)。
- 体积比，![](../graphics/ker_eqn00022.gif)（当前体积/原始体积）。

对于粘弹性材料模型，这些值取决于 [Viscoelastic](pt01ch29pyo106.md) 对象的 *time* 成员的值。

如果 *time*=RELAXATION_TEST_DATA，表格数据指定以下内容：
- 归一化体积（体积）模量，![](../graphics/ker_eqn00401.gif)。
- 时间，![](../graphics/ker_eqn00093.gif)，![](../graphics/ker_eqn00121.gif)。

如果 *time*=CREEP_TEST_DATA，表格数据指定以下内容：
- 归一化体积（体积）柔量，![](../graphics/ker_eqn00402.gif)。
- 时间，![](../graphics/ker_eqn00093.gif)，![](../graphics/ker_eqn00121.gif)。

**返回值**

VolumetricTestData 对象。

**异常**

无。

### 29.110.2 setValues(...)

此方法修改 VolumetricTestData 对象。

**必需参数**

无。

**可选参数**

`setValues` 的可选参数与 [VolumetricTestData](pt01ch29pyo110.md#ker-volumetrictestdata-volumetrictestdata-pyc) 方法的参数相同。

**返回值**

无

**异常**

无。
