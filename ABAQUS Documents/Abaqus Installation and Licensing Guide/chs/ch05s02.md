# 5.2 配置打印机



Windows 平台通常不需要打印命令。Abaqus/CAE 和 Abaqus/Viewer 会自动检测并列出任何已安装的 Windows 打印机。如果您在 Abaqus/CAE 或 Abaqus/Viewer 中使用打印工具时遇到问题，请检查计算机上已安装的 Windows 打印机，并删除任何不再有效的打印机。

本节介绍 Abaqus 如何使用打印命令。Linux 平台上的打印命令应该与其他应用程序使用的命令相同。

### 5.2.1 Abaqus 如何确定和使用打印命令

当 Abaqus/CAE 或 Abaqus/Viewer 打印到 PostScript 打印机时，它会经过以下步骤：

**创建临时 PostScript 文件**

临时 PostScript 文件包含描述要打印的页面所需的所有 PostScript 代码。

**检索打印对话框中指定的打印命令**

打印对话框中指定的命令可以是任何向打印机发送文件的命令，其路径作为最后一个参数（允许某些 PostScript 修改，如 PostScript 打印管理器所做的修改）。

要自定义 Abaqus/CAE 或 Abaqus/Viewer 在显示打印对话框时使用的默认打印命令，请将以下行添加到主目录或 Abaqus 版本站点目录的 `abaqus_v6.env` 文件中：

```
def onCaeStartup():
    session.printOptions.setValues(printCommand=
    '*print_command_and_arguments_here*')
```

**追加临时文件名并调用生成的命令**

先前创建的临时文件的名称被追加到打印命令，PostScript 文件被打印。

**删除临时 PostScript 文件**

如果您的打印机不支持打印假脱机，则可能在打印文件之前删除临时 PostScript 文件。要防止删除临时 PostScript 文件，请将以下行添加到主目录或 Abaqus 版本站点目录的 `abaqus_v6.env` 文件中：

```
def onCaeStartup():
    session.printOptions.setValues(deleteTemporaryFiles=False)
```
