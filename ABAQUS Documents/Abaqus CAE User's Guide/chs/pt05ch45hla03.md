# 45.1 理解符号绘制





符号图允许您可视化向量和张量变量结果的大小和方向。此外，符号图可用于显示当前模型数据库中模型在指定步骤的属性（如载荷或预定义场）的大小和方向。每个向量和张量结果值显示为在模型中获得结果的位置或属性所在的位置绘制的箭头；表示节点量的箭头出现在节点处，表示积分点量的箭头出现在积分点处。（您无法创建存储在节点处的单元量的符号图。）

例如，主应力张量的符号图如图45-1所示（[图45-1](pt05ch45s01.md#viw-sym5)）。

**图45-1** 主应力符号图。

![](../graphics/viw-sym5.png)

当您使用视图切割来切割模型时，符号绘制行为会略有变化。如果您显示视图切割来研究节点输出变量的符号图，Abaqus/CAE会从视图切割表面最近的节点插值值，并在视图切割表面上绘制这些插值向量箭头。Abaqus/CAE仅对节点输出变量执行此插值，在您显示切割下方或上方的模型部分时，不会在切割表面上执行任何插值。有关视图切割功能的更多信息，请参见["理解视图切割，" 第80.1节](pt07ch80s01.md)。

箭头的大小表示结果值的大小。箭头的方向表示结果的全局方向。在张量图中，箭头指向箭杆内部的箭头表示压应力；箭头从箭杆指向外部的箭头表示拉应力。

符号图图例显示每个箭头颜色如何对应于特定范围的值。要了解如何显示与符号图关联的最小值和最大值，请参见["自定义图例，" 第56.1节](pt05ch56s01.md)。

默认情况下，Abaqus/CAE在符号图中绘制变量所有结果值。最大值显示为图中最大的箭头，所有其他箭头按该箭头大小成比例缩放。但是，您可以使用**Symbol Plot Options**对话框限制图中出现的值范围。
- 当您指定特定最大值时，图中仅显示表示小于或等于该最大值的绝对值的箭头。图中的最大箭头将表示小于或等于您指定的限制的最大绝对结果值。所有其他箭头将按该箭头大小成比例缩放。
- 当您指定特定最小值时，图中仅显示表示大于该最小值的绝对值的箭头。
