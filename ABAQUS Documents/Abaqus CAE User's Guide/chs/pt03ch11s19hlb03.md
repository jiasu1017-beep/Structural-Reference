# 11.19.3 选择新零件的类型





使用**创建零件**对话框中的**类型**单选按钮来选择在Abaqus/Standard或Abaqus/Explicit模型中创建的零件的类型。在Abaqus/CFD模型中，只能创建流体零件。Abaqus/CAE在建模过程中保留零件的类型；例如，您不能向刚性零件分配截面和材料属性，不能对分析刚体零件进行网格划分，并且只能向欧拉零件分配欧拉截面和材料分配字段。

在Abaqus/Standard或Abaqus/Explicit模型中，新零件的类型可以设置为以下之一：

**可变形**

您可以创建或导入的任何任意形状的轴对称、二维或三维零件都可以指定为可变形零件。可变形零件表示可以在载荷下变形的零件。

**离散刚体**

离散刚体零件与可变形零件类似，因为它可以是任意形状。但是，离散刚体零件被认为是刚性的，用于接触分析中建模不能变形的物体。

**分析刚体**

分析刚体零件与离散刚体零件类似，因为它用于表示接触分析中的刚性表面。但是，分析刚体零件的形状不是任意的，必须由一组草图线、圆弧和抛物线形成。

**欧拉**

欧拉零件用于定义材料可以在欧拉分析中流动的域。

创建离散刚体零件或分析刚体零件后，您还必须执行以下操作：
- 分配刚体参考点。您在载荷模块中向刚体参考点施加约束或指定运动，相同的约束或运动将应用于整个刚体零件。
- 如果零件是离散刚体零件或分析刚体零件，必须使用装配模块中的曲面工具集来选择代表外表面的零件侧面。

**选择新零件的类型：**

1. 从**创建零件**对话框中间，选择所需的**类型**单选按钮。
2. 完成选项选择后，单击**继续**关闭**创建零件**对话框。草图编辑器启动，您开始绘制新零件基础特征的轮廓。
