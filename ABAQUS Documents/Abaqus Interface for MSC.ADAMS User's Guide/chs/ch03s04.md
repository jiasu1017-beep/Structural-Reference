# 3.4 转换负特征值的模态





Abaqus Interface for MSC.ADAMS 使用组件模态综合法将固定界面正则模态和子结构恢复向量组合成将用于 MSC.ADAMS 动力学分析的模态自由度基础。该模态基础跨越的空间包括子结构的刚体响应。通常，对于三维中无预应力、未约束的物体，预期会发现六个具有零特征值的刚体模态。

对于预应力模型，情况通常不同，其中未约束结构可能具有少于六个具有零特征值的模态。预应力可能将预期的零值改变为显著正值或负值，这取决于预应力的符号。

默认情况下，Abaqus Interface for MSC.ADAMS 会删除具有负特征值的模态并重新正交化简化基础。如果您希望保留具有负特征值的模态，请定义环境变量。
- 在 UNIX 平台上，键入以下命令：``` setenv MDI_MNFWRITE_OPTIONS negative_roots_OK ```
- 在 Windows 平台上，键入以下命令：``` set MDI_MNFWRITE_OPTIONS=negative_roots_OK ```

在这种情况下，Abaqus Interface for MSC.ADAMS 将以与所有其他模态相同的方式处理具有负特征值的模态。

要确定模型在被 Abaqus Interface for MSC.ADAMS 转换时是否会有负特征值，您可以将无边界条件的 [*FREQUENCY](../key/key-link.md#usb-kws-hfrequency) 步骤添加到输入文件。如果将此步骤添加到创建 Abaqus Interface for MSC.ADAMS 所使用的结果文件的运行中，则不得将任何内容写入结果文件。
