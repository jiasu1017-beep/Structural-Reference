# 10.5.5 Importing a model from an ANSYS input file







You can use an ANSYS input file to import a model into Abaqus/CAE by selecting ****File**![](../graphics/images/arrow.gif)**Import**![](../graphics/images/arrow.gif)**Model**** from the main menu bar. Abaqus/CAE imports many common entities in the ANSYS input file into the model you create; for more information on supported entities, see ["Translating ANSYS input files to partial Abaqus input files," Section 3.2.30 of the Abaqus Analysis User's Guide](../usb/usb-link.md#usb-int-dansysabaproc).
