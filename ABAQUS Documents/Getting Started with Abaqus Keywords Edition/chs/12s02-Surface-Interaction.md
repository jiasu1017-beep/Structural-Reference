# 12.2 定义表面

表面是从底层元素的面创建的。下面的讨论假设表面将在 Abaqus/CAE 中定义。在 Abaqus 中可创建的表面类型的限制在《Abaqus Analysis User's Guide》第 2.3 节"Surface definition"中有讨论；在开始接触模拟之前请阅读这些内容。

**连续单元上的表面**

对于二维和三维实体连续单元，您可以通过在视口中选择部件实例的区域来指定构成接触表面的部件区域。

**结构单元、表面单元和刚性单元上的表面**

有四种方法可以在结构单元、表面单元和刚性单元上定义表面：单侧表面、双侧表面、基于边的表面和基于节点的表面。

使用单侧表面，您可以指定单元的哪一侧构成接触表面。与正单元法线方向相同的侧面称为 **SPOS**，而与负单元法线方向相同的侧面称为 **SNEG**，如图 12-1 所示。单元的连通性定义了正单元法线，如《使用壳单元》第 5 章中所讨论的。正单元法线可以在 Abaqus/CAE 中查看。

**图 12-1** 二维壳单元或刚性单元上的表面。

![../graphics/gss-surface-nls.png](graphics/gss-surface-nls.png)

双侧接触表面更加通用，因为 SPOS 和 SNEG 表面以及所有自由边都会自动作为接触表面的一部分包含在内。接触可以发生在构成双侧表面的任一表面或单元的边上。例如，从动节点可以在分析过程中从一个双侧表面的一侧开始，然后沿着周边移动到另一侧。目前，双侧表面只能在三维壳单元、膜单元、表面单元和刚性单元上定义。在 Abaqus/Explicit 中，通用接触算法和接触对算法中的自接触会强制在所有壳单元、膜单元、表面单元和刚性表面单元的所有面上实施接触，即使它们被定义为单侧。双侧接触表面不能与 Abaqus/Standard 中的默认接触公式一起使用，但可以与某些可选接触公式一起使用；有关更多信息，请参阅《Abaqus Analysis User's Guide》第 36.3.1 节"Defining contact pairs in Abaqus/Standard"。

基于边的表面考虑模型周长边上的接触。例如，它们可用于模拟壳单元边缘上的接触。或者，可以使用基于节点的表面（定义一组节点与表面之间的接触）来达到相同的效果，如图 12-2 所示。

**刚性表面**

刚性表面是刚性体的表面。它们可以定义为解析形状，也可以基于与刚性体关联的单元的底层表面。

解析刚性表面有三种基本形式。在二维中，解析刚性表面的特定形式是二维分段刚性表面。表面的横截面使用直线、圆弧和抛物线弧在模型的二维平面中定义。三维刚性表面的横截面在与二维表面相同的方式在用户指定的平面中定义。然后，这个横截面绕轴线旋转形成旋转表面，或沿矢量拉伸形成细长的三维表面，如图 12-3 所示。

**图 12-3** 解析刚性表面。

![../graphics/gsi-analrigidsurf-nls.png](graphics/gsi-analrigidsurf-nls.png)

解析刚性表面的优点是它们仅由少量几何点定义，计算效率高。然而，在三维中可以用它们创建的形状范围是有限的。

离散化刚性表面基于构成刚性体的底层单元；因此，它们可以比解析刚性表面具有更复杂的几何形状。离散化刚性表面的定义方式与可变形体上的表面完全相同。
