# 60.107 Viscosity 对象





Viscosity 对象指定机械粘度。

**访问**

```
materialApi.materials()[*name*].viscosity()
```

### 60.107.1 Viscosity(...)

此方法创建一个 Viscosity 对象。

**路径**

```
materialApi.materials()[*name*].Viscosity
```

**原型**

```
odb_Viscosity&
Viscosity(const odb_SequenceSequenceDouble& table,
          const odb_String& type,
          bool temperatureDependency,
          int dependencies);
```

**必需参数**

*table*

一个 odb_SequenceSequenceDouble，指定下述项目。

**可选参数**

*type*

一个 odb_String，指定粘度类型。默认值为 "NEWTONIAN"。

*temperatureDependency*

一个布尔值，指定数据是否依赖于温度。默认值为 false。

*dependencies*

一个 Int，指定场变量依赖项的数量。默认值为 0。

**表数据**

如果 *type*=NEWTONIAN，表数据指定以下内容：
- 粘度，![](../graphics/ker_eqn00143.gif)。
- 温度（如果数据依赖于温度）。
- 第一个场变量的值（如果数据依赖于场变量）。
- 第二个场变量的值。
- 依此类推。

**返回值**

一个 Viscosity 对象。

**异常**

RangeError。

### 60.107.2 成员

Viscosity 对象的成员与 [Viscosity](pt02ch60pyo107.md#ker-viscosity-viscosity-cpp) 方法的参数具有相同的名称和描述。

此外，Viscosity 对象可以具有以下成员：

**原型**

```
odb_Trs trs() const;
```

*trs*

一个 [Trs](pt02ch60pyo100.md) 对象。
