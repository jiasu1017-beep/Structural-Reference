# 70.6.7 使用投影边方法分区面







您可以使用它们与一条或多条边投影的交叉或连接来分区所选面（如下图所示）：

![](../graphics/prt-face-projedge-nls.png)

Abaqus/CAE 通过对要被分区的面进行垂直投影到所选边来创建分区；非平面面或在垂直投影中存在不连续性的尖锐边相交的面可能会显著改变投影边的形状。如图所示，您可以创建在投影边端点处结束的分叉，或者您可以延伸投影边以到达面边界。如果您延伸分区，Abaqus/CAE 会沿其切线延伸投影边的自由端点，以到达被分区面的边界边。

这些面和边不必属于同一部件；例如，在 Assembly（装配）模块中，您可以使用从第二个部件实例投影的边来分区属于第一个部件实例的面。要被分区的面必须是活动模型表示的一部分。边可以是活动表示的一部分，也可以是中面模型的参考表示的一部分。

**使用投影边方法分区面：**

1. 从主菜单栏中选择 ****Tools**![](../graphics/images/arrow.gif)**Partition****（工具→分区）。出现 **Create Partition**（创建分区）对话框。Abaqus/CAE 在提示区域显示提示以指导您完成过程。**提示：**您也可以使用位于模块工具箱分区面工具处的 ![](../graphics/icoR_ptnFaceProjectEdges.png) 工具来使用投影边方法分区面。 有关工具箱中分区工具的图表，请参阅["使用分区工具集，" 第 70.2 节"](pt06ch70s02.md)。
2. 从对话框顶部的 **Type**（类型）单选按钮中选择 **Face**（面）。**Method**（方法）列表显示可用于分区面的方法。
3. 从方法列表中选择 **Project edges**（投影边）。
4. 选择要分区的面。您可以使用拖动选择、**[Shift]****+点击**、**[Ctrl]****+点击** 和角度方法的组合来选择多个要分区的面。 有关更多信息，请参阅["在当前视口中选择对象，" 第 6.2 节"](pt01ch06s02.md)。**提示：**如果您无法选择所需的面，可以使用 **Selection**（选择）工具栏更改选择行为。 有关更多信息，请参阅["使用选择选项，" 第 6.3 节"](pt01ch06s03.md)。Abaqus/CAE 高亮显示所选面。
5. 选择要投影到面上的边。如果需要，切换开启 **Extend edges**（延伸边）以让 Abaqus/CAE 延伸投影边，防止新面分区中出现间隙。**注意：**您还可以通过在模型树中编辑已完成的分区特征来开启或关闭 **Extend edges**（延伸边）参数。Abaqus/CAE 高亮显示所选边。
6. 在提示区域中点击 **Create Partition**（创建分区）。Abaqus/CAE 投影所选边以创建分区。
