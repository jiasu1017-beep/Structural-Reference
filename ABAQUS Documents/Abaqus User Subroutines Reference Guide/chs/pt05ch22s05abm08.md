### 概述

弹性泡沫材料模型：
- 是各向同性和非线性的；
- 适用于孔隙率允许非常大的体积变化的多孔固体；
- 可选择允许指定能量耗散和应力软化效应（见["弹性泡沫中的能量耗散，" 第22.6.2节](pt05ch22s06abm11.md)）；
- 可以弹性变形到大应变，压缩应变高达90%；和
- 要求在分析步骤中考虑几何非线性（见["定义分析，" 第6.1.2节](pt03ch06s01abo05.md)和["通用和线性扰动过程，" 第6.1.3节](pt03ch06s01aus44.md)），因为它适用于有限应变应用。

Abaqus/Explicit还提供了一个独立的泡沫材料模型，旨在捕获低密度弹性泡沫的应变率敏感行为，如碰撞和冲击应用中使用的那些（见["低密度泡沫，" 第22.9.1节](pt05ch22s09abm16.md)）。

### 弹性泡沫的力学行为

多孔固体由相互连接固体支柱或板的网络组成，这些支柱或板形成细胞的边缘和面。泡沫由在三维中堆积的多面体细胞组成。泡沫细胞可以是开放的（如海绵）或封闭的（如浮力泡沫）。弹性泡沫材料的常见例子是缓冲垫、衬垫和包装材料等细胞聚合物，它们利用了泡沫优异的能量吸收特性：对于一定的应力水平，泡沫吸收的能量大大超过普通刚性弹性材料吸收的能量。

另一类泡沫材料是可压碎泡沫，它们经历永久（塑性）变形。可压碎泡沫在["可压碎泡沫塑性模型，" 第23.3.5节](pt05ch23s03abm34.md)中讨论。

泡沫通常在压缩中加载。图22.5.2-1显示了一个典型的压缩应力-应变曲线。

**图22.5.2-1** 典型压缩应力-应变曲线。

![](../graphics/chyperfoam-comp-curve-nls.png)

压缩过程中可以区分三个阶段：

1. 在小应变时（![](../graphics/usb_eqn02451.gif) 5%），泡沫由于细胞壁弯曲而以线性弹性方式变形。
2. 下一阶段是在几乎恒定应力下变形的平台期，由构成细胞边缘或壁的柱或板的弹性屈曲引起。在封闭细胞中，封闭气体压力和膜拉伸增加了平台的水平和斜率。
3. 最后，发生致密化区域，细胞壁压在一起，导致压缩应力快速增加。典型的最终压缩标称应变为0.7到0.9。

小应变下的拉伸变形机制与压缩机制相似，但大应变时不同。图22.5.2-2显示了一个典型的拉伸应力-应变曲线。

**图22.5.2-2** 典型拉伸应力-应变曲线。

![](../graphics/chyperfoam-ten-curve-nls.png)

拉伸过程中有两个阶段：

1. 在小应变时，泡沫由于细胞壁弯曲而以类似于压缩的方式线性弹性变形。
2. 细胞壁旋转和对齐，导致刚度增加。壁在大约拉伸应变![](../graphics/usb_eqn02452.gif)时基本对齐。进一步拉伸导致壁中轴向应变增加。

在压缩和拉伸的小应变下，泡沫的平均实验观察泊松比![](../graphics/usb_eqn00270.gif)为1/3。在较大应变下，通常观察到泊松比在压缩期间有效为零：细胞壁的屈曲不会导致任何显著的横向变形。然而，![](../graphics/usb_eqn00270.gif)在拉伸期间为非零，这是细胞壁对齐和拉伸的结果。

泡沫的制造通常导致细胞具有不同的主尺寸。这种形状各向异性导致不同方向的加载响应不同。然而，超泡沫模型没有考虑这种初始各向异性。

### 压缩和拉伸变形的差异

对于小应变（![](../graphics/usb_eqn02451.gif) 5%），泡沫在压缩和拉伸方面的行为相似。然而，在大应变下，压缩（屈曲和压碎）和拉伸（对齐和拉伸）的变形机制不同。因此，准确的超泡沫建模需要用于定义材料参数的实验数据对应于被分析问题的主导变形模式。如果压缩是主导的，相关的测试是：
- 单轴压缩
- 简单剪切
- 平面压缩（如果泊松比![](../graphics/usb_eqn02489.gif)）
- 体积压缩（如果泊松比![](../graphics/usb_eqn02489.gif)）

如果拉伸是主导的，相关的测试是：
- 单轴拉伸
- 简单剪切
- 双轴拉伸（如果泊松比![](../graphics/usb_eqn02489.gif)）

### 单元

超泡沫模型可用于实体（连续体）单元、有限应变壳（除S4外）和膜。但是，它不能与一维实体单元（桁架和梁）、小应变壳（STRI3、STRI65、S4R5、S8R、S8R5、S9R5）或Eulerian单元（EC3D8R和EC3D8RT）一起使用。

对于连续单元，弹性泡沫超弹性可用于纯位移公式单元，或在Abaqus/Standard中与"混合"（混合公式）单元一起使用。由于弹性泡沫假定为非常可压缩的，使用混合单元通常不会比使用纯基于位移的单元产生任何优势。
