# 68.2 使用视图操作快捷方式





您可以使用键盘和鼠标组合来访问平移、旋转和放大视图操作工具。这些快捷方式可以更轻松、更少中断地在执行其他任务时操纵模型的视图（例如，选择要定义集合的元素）。只要相关的键盘和鼠标按钮被按住，Abaqus/CAE就会激活指定的视图操作工具；要退出视图操作工具，请释放相关的键盘和鼠标按钮。

除了默认的Abaqus/CAE快捷方式外，您还可以将视图操作快捷方式配置为模仿五个其他常见的CAD应用程序，如[表68-1](pt06ch68s02.md#uss-viw-shortcut-configs)所述。通过从主菜单栏中选择****工具**![](../graphics/images/arrow.gif)**选项****并更改**视图操作**选项来更改快捷方式配置。您的配置会为将来的Abaqus/CAE会话保存。

**表68-1** 可用配置的视图操作快捷方式（"MB"代表"鼠标按钮"）。
| **应用程序** | ![](../graphics/ico_viewPan.png) [平移](pt01ch05s02s02.md) | ![](../graphics/ico_viewRotate.png) [旋转](pt01ch05s02s03.md) | ![](../graphics/ico_viewZoom.png) [缩放](pt01ch05s02s04.md) |
| --- | --- | --- | --- |
| **Abaqus/CAE**（默认） | **[Ctrl]**+**[Alt]**+**[MB2]** | **[Ctrl]**+**[Alt]**+**[MB1]** | **[Ctrl]**+**[Alt]**+**[MB3]** |
| **CATIA V5** | **[MB2]** | **[MB2]**+**[MB3]**；或 **[MB2]**+**[MB1]** | **[MB2]**+**[MB3]**，然后释放 **[MB3]**；或 **[MB2]**+**[MB1]**，然后释放 **[MB1]** |
| **SolidWorks** | **[Ctrl]**+**[MB2]** | **[MB2]** | **[Shift]**+**[MB2]** |
| **HyperView** | **[Ctrl]**+**[MB3]** | **[Ctrl]**+**[MB1]** | **[Ctrl]**+**[MB2]** |
| **Pro/ENGINEER Wildfire** | **[Shift]**+**[MB2]** | **[MB2]** | **[Ctrl]**+**[MB2]** |
| **UGS NX** | **[MB2]**+**[MB3]** | **[MB2]** | **[MB2]**+**[MB1]** |

**注意：**HyperView的旋转键组合（**[Ctrl]**+**[MB1]**）与Abaqus/CAE使用此组合在视口中取消选择对象的用法冲突。为了在复杂视口选择中进行取消选择单个对象可能很有用的选择，建议您选择其他快捷方式配置之一。

视图操作快捷方式的非默认配置会更改放大工具的行为。在默认的Abaqus/CAE配置中，放大工具通过水平拖动光标来操作；在所有其他配置中，放大工具通过垂直拖动光标来操作。

在默认的Abaqus/CAE快捷方式配置中，您可以通过除了其他快捷键之外还按住 **[Shift]** 键来访问平移、旋转和放大工具的替代模式（详情请参见["视图操作工具概述," 第5.2.1节](pt01ch05s02s01.md)）。此约定与某些非默认配置中的快捷方式组合冲突；因此，在使用非默认快捷方式时，您无法访问视图操作工具的替代模式。如果您使用标准方法（从主菜单栏的**视图**菜单中选择工具，或在**视图操作**工具栏中单击工具），则无论快捷方式配置如何，替代模式始终可用。

**要配置视图操作快捷方式：**

1. 从任何模块的主菜单栏中，选择****工具**![](../graphics/images/arrow.gif)**选项****。将出现**选项**对话框。
2. 显示**视图操作**选项卡页面。
3. 选择您要模仿其视图操作快捷方式的**应用程序**。请参阅[表68-1](pt06ch68s02.md#uss-viw-shortcut-configs)以获取不同应用程序中快捷方式的摘要。
4. 单击**确定**应用您指定的配置并关闭**选项**对话框。
