### 概述

吸附：
- 定义多孔材料在部分饱和流动条件下的吸收/解吸行为；以及
- 用于耦合润湿液体流动和多孔介质应力分析（["耦合孔隙流体扩散与应力分析，" 第6.8.1节](pt03ch06s08at26.md)）。

### 吸附

当总孔隙液体压力![](../graphics/usb_eqn01163.gif)变为负值时，多孔介质变得部分饱和（参见["多孔介质的有效应力原理，" Abaqus理论指南第2.8.1节](../stm/stm-link.md#stm-anl-poreffstress)）。![](../graphics/usb_eqn01163.gif)的负值表示介质中的毛细效应。对于![](../graphics/usb_eqn01188.gif)，已知饱和度在一定范围内，该范围取决于毛细压力![](../graphics/usb_eqn04379.gif)的值（参见["多孔介质中润湿液体相的连续性声明，" Abaqus理论指南第2.8.4节](../stm/stm-link.md#stm-anl-porcontstate)）。这些极限的典型形式如图26.6.4-1所示。我们将这些极限写为![](../graphics/usb_eqn04380.gif)，其中![](../graphics/usb_eqn04381.gif)是发生吸收的极限（因此![](../graphics/usb_eqn04382.gif)），而![](../graphics/usb_eqn04383.gif)是发生解吸的极限（因此![](../graphics/usb_eqn04384.gif)）。吸收和解吸之间的转换以及相反过程沿"扫描"曲线发生（见下文）。这些曲线由图26.6.4-1中所示的单一直线近似。

当在多孔介质流动分析中包含部分饱和时，应分别定义吸收行为、解吸行为和扫描行为（介于吸收和解吸之间）。以下讨论每种行为。如果完全未定义吸附，Abaqus/Standard假定对于![](../graphics/usb_eqn01163.gif)的所有值均为完全饱和流动（![](../graphics/usb_eqn01203.gif)）。

由吸附定义产生强非对称的部分饱和流动耦合方程。因此，如果请求部分饱和分析（即定义了吸附），Abaqus/Standard自动使用其非对称矩阵存储和求解方案（参见["定义分析，" 第6.1.2节](pt03ch06s01abo05.md)）。

### 定义吸收和解吸之间的行为

吸收和解吸之间的行为由用户指定的恒定斜率![](../graphics/usb_eqn04391.gif)的扫描线定义。该斜率应大于吸收或解吸行为任何段的斜率。

如果定义了吸收和解吸行为而没有扫描线，则扫描线的斜率取为吸收和解吸行为定义中给出的![](../graphics/usb_eqn04392.gif)最大值的1.05倍。

| **输入文件用法：** | ``` [*SORPTION](../key/key-link.md#usb-kws-msorption), TYPE=SCANNING ``` |
| --- | --- |
|  | 这必须是同一材料的[*SORPTION](../key/key-link.md#usb-kws-msorption)选项的重复使用。 |
