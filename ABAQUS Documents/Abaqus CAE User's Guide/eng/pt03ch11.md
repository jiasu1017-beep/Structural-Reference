# 11 The Part module







Parts are the building blocks of an Abaqus/CAE model. You use the Part module to create each part, and you use the Assembly module to assemble instances of the parts. The tutorial in [Appendix C, "Using Additional Techniques to Create and Analyze a Model in Abaqus/CAE," of Getting Started with Abaqus: Interactive Edition](../gsa/gsa-link.md#gsa-chp-caehinge) contains examples of how you create, modify, and manipulate parts.
