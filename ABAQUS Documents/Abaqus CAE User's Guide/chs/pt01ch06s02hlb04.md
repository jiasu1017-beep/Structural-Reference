# 6.2.4 使用面曲率方法选择多个面





除了通过它们之间的角度选择对象外，你还可以基于面的曲率从部件中选择多个面。当你执行允许你拾取多个几何面的任务时，Abaqus/CAE 会在提示区域显示一个字段。该字段允许你在三种选择方法之间进行选择——**逐个**、**按面角度**和**按面曲率**，如图 [图 6-7](pt01ch06s02hlb04.md#pic-facecurve) 所示。

**图 6-7** 从提示区域的字段中选择选择方法。

![](../graphics/pic-facecurve-nls.png)

角度选择方法在["使用角度和特征边方法选择多个对象，" 第 6.2.3 节](pt01ch06s02hlb03.md)中有描述。

面曲率方法在选择面的过程中可用。如果过程接受除面之外的其他对象类型，你可以在**选择**工具栏中将对象类型更改为**面**以访问面曲率方法。

从部件或装配体中选择一个面。Abaqus/CAE 选择所有在两个主方向上具有相似曲率并在小于 20 度角处连接的所有连接面。如果你选择一个平面，Abaqus/CAE 会添加位于同一平面中的任何相邻平面。不共享相似曲率的断开的面不会被选择，也不选择共享相似曲率但在相遇处边法线明显不同的面。[图 6-8](pt01ch06s02hlb04.md#pic-facecurvemethod) 显示了使用面曲率方法选择的两个圆形面。

**图 6-8** 选择单个曲面以选择具有相似曲率的相邻面。

![](../graphics/pic-facecurvemethod.png)

使用面曲率方法后，你可以在提示区域中单击**逐个**方法，并 **[Shift]****+点击**各个面以将其附加到你的选择。你也可以 **[Ctrl]****+点击**项目以取消选择它们。此外，你可以继续使用面曲率方法，并使用 **[Shift]****+点击**将面附加到你的选择。更多信息，请参阅["组合选择技术，" 第 6.2.8 节](pt01ch06s02hlb08.md)。
