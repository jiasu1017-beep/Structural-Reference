# 19.2.4 作业编辑器





您可以使用作业编辑器为新作业自定义设置，或为现有作业编辑设置。您可以通过从主菜单栏选择 ****作业**![](../graphics/images/arrow.gif)**创建**** 或 ****作业**![](../graphics/images/arrow.gif)**编辑**![](../graphics/images/arrow.gif)***作业名称***** 来显示作业编辑器。（您也可以在作业管理器中单击**创建**或**编辑**。）

作业编辑器包含以下选项卡页面：

**[提交](pt03ch19s08hlb02.md)**

使用**提交**选项卡页面配置作业的提交属性，例如作业类型、运行模式和提交时间。您还可以使用提交选项卡页面指定将作业提交到由本地 Abaqus 环境文件或系统管理员配置的远程队列。

**[常规](pt03ch19s08hlb06.md)**

使用**常规**选项卡页面配置作业设置，例如分析输入文件处理器打印输出、用于临时文件的目录名称以及结果输出格式。

对于与输入文件关联的作业，预处理器打印输出选项不可用；您必须在输入文件中指定这些选项。

**[内存](pt03ch19s08hlb07.md)**

使用**内存**选项卡页面配置分配给 Abaqus 分析的内存量。

**[并行化](pt03ch19s08s01.md)**

使用**并行化**选项卡页面配置 Abaqus 分析作业的并行执行，例如要使用的处理器数量和并行化方法。

**[精度](pt03ch19s08hlb08.md)**

使用**精度**选项卡页面为 Abaqus/Explicit 分析指定单精度或双精度。您还可以选择在分析期间写入输出数据库的节点输出精度。
