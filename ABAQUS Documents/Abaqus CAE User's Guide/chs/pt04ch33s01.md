# 33.1 定义惯性





可以为部件的Property模块中的部件或装配的Interaction模块定义惯性。您可以指定以下类型：
- **点质量/惯性**。您可以为部件或装配上的点定义集中质量和旋转惯性。您还可以定义质量和惯性比例阻尼。在Abaqus/Standard分析中，您可以定义复合阻尼。更多信息，请参见["点质量"，Abaqus Analysis User's Guide第30.1.1节](../usb/usb-link.md#usb-elm-emasses)和["旋转惯性"，Abaqus Analysis User's Guide第30.2.1节](../usb/usb-link.md#usb-elm-erotinertia)。
- **非结构质量**。您可以为部件或装配上的区域定义非结构质量。更多信息，请参见["非结构质量定义"，Abaqus Analysis User's Guide第2.7.1节](../usb/usb-link.md#usb-int-anonstructuralmass)。
- **热容**。您可以为部件或装配上的点定义集中热容。更多信息，请参见["点电容"，Abaqus Analysis User's Guide第30.4.1节](../usb/usb-link.md#usb-elm-ecapacitance)。

从Property模块或Interaction模块的主菜单栏中选择****Special**![](../graphics/images/arrow.gif)**Inertia**![](../graphics/images/arrow.gif)**Create****:*类型*来定义惯性。从同一菜单中选择**Edit**以对现有定义进行更改。

惯性定义出现在Model Tree的**Engineering Features**容器下（如果在Property模块中创建）或装配下（如果在Interaction模块中创建）。

当您定义惯性时，可以从视口中选择对象以标识要应用惯性的区域。默认情况下，会创建一个包含所选对象的集或表面。您可以通过关闭提示区域中创建集或表面的选项来更改此行为。提示区域中提供了默认名称，但您可以输入新名称。

当您将惯性应用到模型的区域时，可以选择在视口中显示符号以指示您应用惯性的位置。如果您将惯性应用到几何体，符号会出现在顶点上。如果您将惯性应用到孤立网格，符号会出现在节点上。有关图形符号类型的信息，请参见["表示特殊工程特征的符号"，第C.3节](ap03s03.md)。有关控制这些符号可见性的信息，请参见["控制属性的显示"，第76.15节](pt07ch76hla14.md)。
