# *GAP






### *GAP为 GAP 型单元指定间隙和局部几何。

此选项用于定义 GAP 型接触单元的行为。

**产品：** Abaqus/Standard  Abaqus/CAE  

**类型：** 模型数据  

**级别：** 零件、零件实例  

**Abaqus/CAE：** 不支持；通过建模连接器可以提供类似功能。

##### **参考文献：**

- ["间隙接触单元，" Abaqus Analysis User's Guide 第 40.2.1 节](../usb/usb-link.md#usb-cni-egap)
