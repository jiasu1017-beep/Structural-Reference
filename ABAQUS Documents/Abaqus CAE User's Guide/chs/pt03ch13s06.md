# 13.6 创建零件实例阵列

您可以在线性或径向阵列中创建所选零件实例的多个副本。您可以指定要创建的实例数量和阵列的结构，如下所述。这些命令不能用于模型实例。

**线性阵列**

线性阵列沿方向（例如X方向）线性定位新实例。所选零件实例的原点和新零件实例的原点位于方向指定的直线上。您可以指定实例数量和实例之间的间距。此外，您可以通过从装配中选择代表新方向的线来更改线性阵列的方向。

您可以通过在第二个方向（例如Y方向）创建副本创建副本矩阵；选项与第一个方向相同；您可以控制副本数量、间距和方向。默认情况下，第一个方向是X轴，第二个方向是Y轴。例如，[图13-16](pt03ch13s06.md#asm-conc-linearpattern)说明了零件实例如何沿X轴和Y轴进行阵列。

**图13-16** 在两个线性方向上阵列的零件实例。

![](../graphics/asm-conc-linearpattern.png)

**径向阵列**

径向阵列以圆形模式定位新实例。您可以指定实例数量，并且可以指定第一个和最后一个副本之间的角度，其中正角度对应逆时针方向。例如，[图13-17](pt03ch13s06.md#asm-conc-radialpattern)展示了与[图13-16](pt03ch13s06.md#asm-conc-linearpattern)相同的零件实例的径向阵列。

**图13-17** 零件实例的径向阵列。

![](../graphics/asm-conc-radialpattern.png)

默认情况下，Abaqus/CAE围绕Z轴创建径向阵列。或者，您可以从装配中选择一条线来定义圆形阵列的轴。

如果您创建接触的实例阵列，并希望将阵列视为单个零件，必须使用**合并/切割**工具将阵列中的所有零件实例合并为单个零件实例。例如，[图13-17](pt03ch13s06.md#asm-conc-radialpattern)中所示径向阵列中的实例彼此重叠，已被合并为单个零件实例。更多信息，请参见["对零件实例执行布尔运算，"第13.7节](pt03ch13s07.md)。如果您不合并零件实例，阵列可能在实例接触处包含重复的面或节点。

如果零件包含零件级集或面，Abaqus/CAE会为阵列中每个单独的零件实例创建单独的装配级集和面（有关零件级和装配级集和面的进一步讨论，请参见["零件集和装配集有何不同？"第73.2.2节"](pt06ch73s02s02.md)。例如，如果[图13-16](pt03ch13s06.md#asm-conc-linearpattern)和[图13-17](pt03ch13s06.md#asm-conc-radialpattern)中原始零件的顶面包含在零件级面中，Abaqus/CAE最初为图案化装配中每个零件实例的顶面创建单独的装配级面。将这些重复的集和面合并为单个集或面通常很有帮助。当您合并图案化零件实例时，Abaqus/CAE还会将任何重复的集或重复的面合并为合并零件和零件实例上的单个集或面。如果不合并图案化零件实例，仍然可以使用模型树中的**布尔**选项合并集或面（有关说明，请参见["对集或面执行布尔运算，"第73.3.4节"](pt06ch73s03hlb04.md)）。

当您创建实例的线性或径向阵列时，您会发现使用从属零件实例更方便。当您对原始零件进行网格划分时，Abaqus/CAE将相同的网格应用于阵列中的每个实例。相反，如果您创建独立实例的阵列，则必须单独对每个实例进行网格划分。更多信息，请参见["从属零件实例和独立零件实例有什么区别？"第13.3.2节"](pt03ch13s03s02.md)。
