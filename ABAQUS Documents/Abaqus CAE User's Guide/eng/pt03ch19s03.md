# 19.3 Understanding adaptivity processes







This section provides an overview of adaptivity processes. You use an adaptivity process to control a succession of analysis jobs that are adaptively remeshed by Abaqus/CAE based on the contents of your remeshing rules. For more information about adaptive remeshing and the other adaptivity techniques that are available in Abaqus, see ["Adaptivity techniques," Section 12.1.1 of the Abaqus Analysis User's Guide](../usb/usb-link.md#usb-anl-aadpchoicesover).
