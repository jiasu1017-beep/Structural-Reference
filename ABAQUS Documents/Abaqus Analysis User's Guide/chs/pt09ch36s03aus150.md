### 概述

Abaqus/Standard 中的接触控制：
- 对于大多数问题不应从默认设置进行修改；
- 可用于标准接触控制不能提供经济有效解决方案的问题；
- 可用于标准控制不能有效建立所需接触条件的问题；以及
- 可在某些情况下用于控制是否创建补充接触约束。

从 Abaqus/Standard 中的接触控制调整中受益的问题通常是具有复杂几何形状和众多接触界面的大型模型。

### 应用接触控制

您可以按步骤将接触控制应用于步骤中活跃的所有接触对和接触单元，或者应用于单个接触对。这使得可以将接触控制应用于特定接触对，以使模拟通过困难阶段。接触控制保持有效，直到它们被更改或重置为默认值。如果在任何给定步骤中为整个模型和特定接触对都声明了接触控制，则特定接触对的控制将覆盖该接触对的整个模型的控制。

此外，您可以如下的为单个接触对指定补充接触约束（参见下面的 ["补充接触约束"](pt09ch36s03aus150.md#usb-cni-acontacttrouble-supplementary-constraints)"）。

### 接触问题中刚体运动的自动稳定

Abaqus/Standard 提供接触稳定，以帮助在静态问题中自动控制接触闭合和摩擦约束此类运动之前的刚体运动。

建议您首先尝试通过建模技术（修改几何形状、施加边界条件等）来稳定刚体运动。自动稳定功能旨在用于明确会建立接触但在建模型时难以确定多个体的精确位置的情况。它不用于模拟一般刚体动力学；也不用于接触颤振情况或解决配合曲面之间初始紧密间隙的问题。

使用自动接触稳定时，Abaqus/Standard 在所有从节点处的接触对的相对运动上激活粘性阻尼，方式与接触阻尼相同（参见 ["接触阻尼," 第 37.1.3 节](pt09ch37s01aus167.md)）。与大多数接触控制不同（这些控制在后续步骤中保持有效，直到被修改或重置），自动稳定阻尼仅在指定该阻尼的步骤持续时间内应用。在后续步骤中，即使接触未建立或接触对后来完全分离导致刚体运动，阻尼也会被移除。如果需要，您还应为后续步骤指定稳定。

默认情况下，阻尼系数：
- 基于底层单元的刚度和步骤时间自动为每个接触约束计算，
- 在法向和切向方向均匀应用于所有接触对，
- 在步骤中线性减小，
- 仅当接触曲面之间的距离小于特征曲面尺寸时激活，以及
- 对于使用接触单元（如间隙接触单元、管-管接触单元等）建模的接触为零。

虽然自动计算的阻尼系数通常提供足够的阻尼来消除刚体模式而不会对解产生重大影响，但不能保证该值是最优的甚至是合适的。对于薄壳模型尤其如此，其中阻尼可能太高。因此，如果收敛行为有问题，您可能需要增加阻尼；如果阻尼影响解的准确性，则可能需要减小阻尼。前者很明显，但后者需要进行事后检查。有几种方法可以进行此类检查。最简单的方法是考虑粘性阻尼耗散的能量与模型的更一般能量度量（如弹性应变能）之间的比率。这些量可以作为输出变量 ALLSD 和 ALLSE 获得。通过将接触阻尼应力 CDSTRESS（具有单独分量 CDPRESS、CDSHEAR1 和 CDSHEAR2）与真实接触应力 CSTRESS（具有单独分量 CPRESS、CSHEAR1 和 CSHEAR2）进行比较，可以获得更详细的信息。如果接触阻尼应力太高，应减小阻尼。比较应在接触牢固建立后进行；接触未建立或仅部分建立时，接触阻尼应力总是相对较高。

增加或减少阻尼量的最简单方法是指定一个乘数，自动计算的阻尼系数将乘以此乘数。通常，您应该首先考虑更改默认阻尼（至少一个数量级）；如果这样可以足够地解决问题，您可以随后进行一些微调。在某些情况下可能需要更大或更小的因子；只要获得收敛解且耗散的能量和接触阻尼应力足够小，这就不是问题。

也可以直接指定阻尼系数。直接指定阻尼值并不容易，可能需要一些试错。为提高效率，最好在类似的小型模型上进行此操作。如果直接指定阻尼系数，则忽略为默认阻尼系数指定的任何乘数因子。

| **输入文件用法：** | 要使用默认阻尼系数： |
| --- | --- |
|  | ``` [*CONTACT CONTROLS](../key/key-link.md#usb-kws-hcontactcontrols), STABILIZE ``` 要指定默认阻尼系数的缩放因子： ``` [*CONTACT CONTROLS](../key/key-link.md#usb-kws-hcontactcontrols), STABILIZE=*factor* ``` 要直接指定阻尼系数： ``` [*CONTACT CONTROLS](../key/key-link.md#usb-kws-hcontactcontrols), STABILIZE *damping coefficient* ``` |

| **Abaqus/CAE 用法：** | 相互作用模块：Abaqus/Standard 接触控制编辑器：**稳定化**：**自动稳定**，**因子：** *factor* 或**稳定化系数：** *damping coefficient* |
| --- | --- |

#### 在增量内更改稳定

为了减少或消除接触稳定对报告解的重大影响的可能性，可以引入在增量迭代中变化的缩放因子。在增量的早期迭代中更多地使用稳定有助于在建立一些接触之前避免数值问题。在增量的后期迭代中减少或不使用稳定有助于提高增量最终收敛迭代的准确性。

您可以指定这些缩放因子。例如，指定"1,0"会导致缩放因子在初始迭代（直到各种收敛度量满足或接近满足）期间为统一值，然后缩放因子被重置为零（有效关闭稳定），直到收敛检查再次满足。

| **输入文件用法：** | 要在增量内更改阻尼系数： |
| --- | --- |
|  | ``` [*CONTACT CONTROLS](../key/key-link.md#usb-kws-hcontactcontrols), STABILIZE=USER ADAPTIVE ``` |

#### 指定稳定下降因子

您可以指定步骤结束时的下降因子。默认情况下，此值等于零，以便阻尼在步骤结束时完全消失。如果在步骤结束时刚体模式未完全约束（例如，如果问题是无摩擦的并且可以发生滑动但滑动方向上没有净力），则为此参数输入非零值可能很有用。在这种情况下，通常希望保持小阻尼进入下一步，方法是使用用于下降的值作为阻尼系数的乘数。如果需要，您可以通过将下降因子设置为一来保持此阻尼水平。

| **输入文件用法：** | ``` [*CONTACT CONTROLS](../key/key-link.md#usb-kws-hcontactcontrols), STABILIZE , *ramp-down factor* ``` |
| --- | --- |

| **Abaqus/CAE 用法：** | 相互作用模块：Abaqus/Standard 接触控制编辑器：**稳定化**：**自动稳定**或**稳定化系数**，**步骤结束时的阻尼分数：** *ramp-down factor* |
| --- | --- |

#### 指定阻尼范围

默认情况下，应用阻尼的开口距离（阻尼范围）等于特征从曲面面片尺寸；如果此尺寸不可用（例如，对于基于节点的曲面），则使用为整个模型获得的特征单元长度。阻尼在开口小于阻尼范围一半时为参考值的 100%，从那里线性减小到开口等于阻尼范围时为零。或者，您可以直接指定阻尼范围，覆盖计算值。如果阻尼应该仅适用于窄间隙，或者阻尼应该无论开口距离如何都有效，这可能很有用。在后一种情况下，应输入一个大的值。

| **输入文件用法：** | ``` [*CONTACT CONTROLS](../key/key-link.md#usb-kws-hcontactcontrols), STABILIZE , , *damping range* ``` |
| --- | --- |

| **Abaqus/CAE 用法：** | 相互作用模块：Abaqus/Standard 接触控制编辑器：**稳定化**：**自动稳定**或**稳定化系数**，**阻尼为零的间隙：指定：** *damping range* |
| --- | --- |

#### 指定切向阻尼

默认情况下，切向方向的阻尼与法向方向的阻尼相同。但是，如果需要较小或较大的值，您可以减少或增加切向阻尼或将其设置为零。

| **输入文件用法：** | ``` [*CONTACT CONTROLS](../key/key-link.md#usb-kws-hcontactcontrols), STABILIZE, TANGENT FRACTION=*value* ``` |
| --- | --- |

| **Abaqus/CAE 用法：** | 相互作用模块：Abaqus/Standard 接触控制编辑器：**稳定化**：**自动稳定**或**稳定化系数**，**切向分数：** *value* |

### 与法向接触约束相关的接触控制

这些控制允许您指定接触界面上的节点可以违反"硬"接触条件。此外，这些控制可用于修改"软化"压力-过盈关系以及增强拉格朗日或惩罚接触约束强制的行为。不能通过接触控制修改无分离压力-过盈关系。

节点可以通过两种方式之一违反接触条件。首先，Abaqus/Standard 可能认为该节点没有接触，即使该节点已穿透主曲面一小段距离。其次，Abaqus/Standard 可能认为该节点有接触，即使该节点处传输的正常压力为负（即，正在传输拉伸应力）。

#### 修改增强拉格朗日或惩罚接触约束强制的行为

对于增强拉格朗日接触，您可以指定允许穿透（直接指定或作为特征接触曲面尺寸的分数），以允许违反不可穿透条件。此外，对于增强拉格朗日或惩罚接触，您可以缩放 Abaqus/Standard 计算的默认惩罚刚度。增强拉格朗日和惩罚约束强制方法的控制在 ["Abaqus/Standard 中的接触约束强制方法," 第 38.1.2 节](pt09ch38s01aus178.md) 中讨论。

### 在线 性扰动步骤中修改切向惩罚刚度

在线性扰动步骤中强制切向约束的惩罚刚度通常与在一般步骤中强制粘附所使用的惩罚刚度不同。在扰动步骤中，当相应法向约束在基础状态中活跃且接触属性（曲面相互作用）定义包含摩擦模型时，Abaqus/Standard 会激活切向接触约束。默认情况下，切向惩罚刚度等于默认法向惩罚刚度。

您可以按步骤缩放切向惩罚刚度以模拟粘附/滑动条件。此缩放仅影响指定它的扰动步骤；它不会延续到后续步骤。如果您希望在同一系列扰动步骤中应用相同的缩放因子，则必须在每个步骤中明确指定缩放因子。

某些依赖频率分析的程序（如复频率分析和基于子空间的稳态动态分析）会受到先前频率分析有效的切向刚度缩放以及这些步骤有效的切向刚度缩放的影响。在这种情况下，建议为这些步骤使用一致的缩放。对于其他基于频率分析的模态程序，切向刚度的缩放被忽略，仅考虑先前频率分析的影响。

| **输入文件用法：** | 要在线性扰动步骤中修改所有接触对的切向惩罚刚度： |
| --- | --- |
|  | ``` [*CONTACT CONTROLS](../key/key-link.md#usb-kws-hcontactcontrols), PERTURBATION TANGENT SCALE FACTOR=*factor* ``` 要在线性扰动步骤中修改特定接触对的切向惩罚刚度： ``` [*CONTACT CONTROLS](../key/key-link.md#usb-kws-hcontactcontrols), PERTURBATION TANGENT SCALE FACTOR=*factor*, SLAVE=*slave surface*, MASTER=*master surface* ``` |

| **Abaqus/CAE 用法：** | 不支持在 Abaqus/CAE 中的线性扰动步骤中修改切向惩罚刚度。 |

### 线性扰动步骤中与压力相关的约束强制

在线性扰动步骤中，接触约束通常对于基础状态中封闭接触界面完全强制执行，与基础状态中的局部法向压力无关。您可以使用两个控制压力系数 ![](../graphics/usb_eqn02314.gif) 和 ![](../graphics/usb_eqn06124.gif) 来放松在基础状态中压力较低的约束，甚至完全移除它们。法向和切向约束都受影响。对于基础状态中压力小于 ![](../graphics/usb_eqn02314.gif) 的情况，法向和切向约束通过将约束刚度设置为零来有效移除。对于压力大于 ![](../graphics/usb_eqn06124.gif) 的情况，约束被完全强制执行。对于 ![](../graphics/usb_eqn02314.gif) 和 ![](../graphics/usb_eqn06124.gif) 之间的压力，约束刚度会降低，并在 ![](../graphics/usb_eqn02314.gif) 和 ![](../graphics/usb_eqn06124.gif) 之间线性增加。在此压力范围内，即使对于本来会使用严格拉格朗日乘数强制的接触约束，有限接触刚度也是有效的。初始应力刚度项也被缩放。

所有其他法向和切向接触惩罚的控制都适用。基础状态中开放的约束不受影响。压力相关约束强制不能在一般步骤中使用。

您可以按步骤指定与压力相关的约束强制。此规范影响指定它的扰动步骤；它不会延续到后续步骤。如果您希望在同一系列扰动步骤中应用相同的规定，则必须在每个步骤中明确指定它。

某些依赖频率分析的程序（如复频率分析和基于子空间的稳态动态分析）会受到先前频率分析有效的规定以及这些步骤有效的规定的影响。在这种情况下，建议为这些步骤使用一致的规定。对于其他基于频率分析的模态程序，规定被忽略，仅考虑先前频率分析的影响。

| --- | --- |
|  | ``` [*CONTACT CONTROLS](../key/key-link.md#usb-kws-hcontactcontrols), PRESSURE DEPENDENT PERTURBATION=![](../graphics/usb_eqn06124.gif) ![](../graphics/usb_eqn02314.gif) ``` 要在线性扰动步骤中为特定接触对指定基础状态压力相关约束强制： ``` [*CONTACT CONTROLS](../key/key-link.md#usb-kws-hcontactcontrols), PRESSURE DEPENDENT PERTURBATION=![](../graphics/usb_eqn06124.gif), SLAVE=*slave surface*, MASTER=*master surface* ![](../graphics/usb_eqn02314.gif) ``` |

### 与二阶面相关的接触控制

二阶单元不仅提供更高的精度，还能更有效地捕获应力集中，并且比一阶单元更好地建模几何特征。基于二阶单元类型的曲面与面-面接触公式配合良好，但在某些情况下与点-面公式配合不佳（参见 ["Abaqus/Standard 中的接触公式," 第 38.1.1 节](pt09ch38s01aus177.md)，了解这些接触公式的讨论）。

某些二阶单元类型不适合使用点-面接触公式和"硬"接触条件的严格强制作为从曲面底层，因为当压力作用在单元面上时，等效节点力的分布如图 36.3.6-1 所示。

**图 36.3.6-1**"硬"接触模拟中作用在二阶单元面上的恒定压力产生的等效节点载荷。

![](../graphics/adefsurf-3d-2nd-order-cont.png)

二阶单元中节点力的这种模糊性质可能导致 Abaqus/Standard 不适当地改变其内部接触逻辑。基于二阶四面体单元的从曲面对于点-面接触公式也可能有问题的，因为这些单元面上压力作用的等效节点力分布使得角节点处的力为零。

Abaqus/Standard 中可用于使涉及二阶从面的点-面接触对更容易使用的选项在下面讨论。您也可以通过使用面-面接触公式来避免潜在困难，这通常是优选的。

#### 手动或自动调整单元类型

修改的 10 节点四面体单元（C3D10M 等）不会对点-面接触公式造成根本困难，通常可以为具有点-面接触对的模型提供比 10 节点二阶四面体单元（C3D10、C3D10I 等）更可行的替代方案。修改的 10 节点四面体单元与二阶四面体单元之间特性权衡在 ["修改的三角形和四面体单元"中讨论，"实体（连续体）单元," 第 28.1.1 节](pt06ch28s01alm01.md#usb-elm-esolidcont-modifiedelems)。如果需要，您必须手动进行单元类型调整，因为它不会自动发生。

默认情况下，Abaqus/Standard 自动向与非绑定点-面接触对关联的大多数 8 节点从面底层（减轻型）单元添加面中节点。对于三维 18 节点垫片单元，如果元素连通性中没有给出面中节点，也会自动生成它们。面中节点的存在导致对接触算法不模糊的节点力分布。单元族 C3D20(RH)、C3D15(H)、S8R5 和 M3D8 分别转换为族 C3D27(RH)、C3D15V(H)、S9R5 和 M3D9。由于 Abaqus/Standard 不会转换二阶耦合温度-位移、耦合热-电-结构和耦合孔隙压力-位移单元，因此在这些情况下使用点-面接触公式中的减轻型单元的替代方法。当在任何用户定义的节点上指定值时，Abaqus/Standard 会在自动生成的面中节点处插值节点量（如温度和场变量）。如果从曲面用于绑定接触对，Abaqus/Standard 不会转换二阶减轻型单元。

默认情况下，Abaqus/Standard 不会自动向面-面接触对从曲面的二阶减轻型单元添加面中节点；但是，有选项可以启用与点-面接触对使用的自动添加面中节点的相同算法。

| **输入文件用法：** |  |
| --- | --- |
|  | ``` [*CONTACT PAIR](../key/key-link.md#usb-kws-hcontactpair), TYPE=SURFACE TO SURFACE, MIDFACE NODES=YES ``` |

| **Abaqus/CAE 用法：** | 您不能在 Abaqus/CAE 中启用自动转换面-面接触对从曲面底层减轻型单元的功能。 |
| --- | --- |

#### 补充接触约束

避免某些单元类型给点-面接触公式带来困难的另一种方法是在不更改底层单元公式的情况下添加补充接触约束。此方法仅适用于使用惩罚或增强拉格朗日约束强制或软化压力-过盈关系的点-面接触对，因为它会导致过度约束的情况（如果使用严格强制的"硬"接触条件）。补充接触约束有时有助于改善收敛行为或改善接触压力和底层单元应力的平滑度和准确性；但是，额外约束存在降低收敛行为的风险。默认情况下，对于具有非修改单元的 6 节点从面和 8 节点从面的点-面接触对，除非严格强制的"硬"接触条件生效，否则会选择性使用补充约束。您可以停用补充约束或为从曲面底层的其他二阶单元类型添加激活补充约束。

| **输入文件用法：** | ``` [*CONTACT PAIR](../key/key-link.md#usb-kws-hcontactpair), INTERACTION=*interaction_property_name*, SUPPLEMENTARY CONSTRAINTS=SELECTIVE *slave_surface_name*, *master_surface_name* ``` |
| --- | --- |
|  | 使用以下选项为其他二阶单元类型添加补充接触约束： ``` [*CONTACT PAIR](../key/key-link.md#usb-kws-hcontactpair), INTERACTION=*interaction_property_name*, SUPPLEMENTARY CONSTRAINTS=YES *slave_surface_name*, *master_surface_name* ``` 使用以下选项放弃补充接触约束： ``` [*CONTACT PAIR](../key/key-link.md#usb-kws-hcontactpair), INTERACTION=*interaction_property_name*, SUPPLEMENTARY CONSTRAINTS=NO *slave_surface_name*, *master_surface_name* ``` |

| **Abaqus/CAE 用法：** | 对于点-面接触公式： |
| --- | --- |
|  | 相互作用模块：**创建相互作用**：**面-面接触（Standard）**：选择主曲面；点击**曲面**；选择从曲面；相互作用编辑器；**使用补充接触点**：**选择性**、**始终**或**从不**；**接触相互作用属性**：*interaction_property_name* |

### 面-面接触对滑动时接触力重分布的平滑度

您可以控制面-面接触对滑动时节点接触力重分布的平滑度。默认设置（通常适当）导致节点力重分布的平滑度与从曲面底层单元的平滑度相同；即，对于线性单元为线性重分布平滑度，对于二阶单元为二次重分布平滑度。二次重分布平滑度通常有助于改善收敛行为并改善快速变化接触应力区域中接触应力的分辨率。然而，二次重分布平滑度倾向于增加每个约束涉及的节点数，这可能增加方程求解器的计算成本。线性重分布平滑度倾向于更好地解析活动接触区域边缘附近的接触应力，因此偶尔会产生更好的收敛行为。

| **输入文件用法：** | 使用以下选项指示对于面-面接触对，滑动时接触力重分布的平滑度应与从曲面底层单元的平滑度相同： |
| --- | --- |
|  | ``` [*CONTACT PAIR](../key/key-link.md#usb-kws-hcontactpair), TYPE=SURFACE TO SURFACE, SLIDING TRANSITION=ELEMENT ORDER SMOOTHING *slave_surface_name*, *master_surface_name* ``` 使用以下选项指示面-面接触对滑动时接触力重分布的线性平滑度： ``` [*CONTACT PAIR](../key/key-link.md#usb-kws-hcontactpair), TYPE=SURFACE TO SURFACE, SLIDING TRANSITION=LINEAR *slave_surface_name*, *master_surface_name* ``` 使用以下选项指示面-面接触对滑动时接触力重分布的二次平滑度： ``` [*CONTACT PAIR](../key/key-link.md#usb-kws-hcontactpair), TYPE=SURFACE TO SURFACE, SLIDING TRANSITION=QUADRATIC *slave_surface_name*, *master_surface_name* ``` |

| **Abaqus/CAE 用法：** | 您不能在 Abaqus/CAE 中更改默认接触力重分布。 |
| --- | --- |
