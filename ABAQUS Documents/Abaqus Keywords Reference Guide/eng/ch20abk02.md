# *UNDEX CHARGE PROPERTY







### *UNDEX CHARGE PROPERTYDefine an UNDEX charge for incident waves.

This option defines parameters that create the time histories of load, displacement, and other variables used to simulate an underwater explosion. This option must be used in conjunction with the [*INCIDENT WAVE INTERACTION PROPERTY](ch09abk11.md) option.

**Products: **Abaqus/Standard  Abaqus/Explicit  Abaqus/CAE  

**Type: **Model or history data  

**Level: **Model,  Step

**Abaqus/CAE: **Interaction module

##### **References:**

- ["Acoustic and shock loads," Section 34.4.6 of the Abaqus Analysis User's Guide](../usb/usb-link.md#usb-prc-pacoustic)
- [*INCIDENT WAVE INTERACTION PROPERTY](ch09abk11.md)

**There are no parameters associated with this option.**
