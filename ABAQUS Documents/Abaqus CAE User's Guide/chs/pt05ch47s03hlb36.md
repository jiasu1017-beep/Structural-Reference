# 47.4.38 追加两个或多个X–Y数据对象





### 47.4.38 追加两个或多个X–Y数据对象

使用**append**函数将两个或多个[之前保存的](pt05ch47s02hlb07.md) *X–Y*数据对象（有序对的集合）链接在一起，生成一个新的*X–Y*数据对象。新的*X–Y*数据对象包含您按指定顺序指定的数据对象的串联结果。

[图47-34](pt05ch47s03hlb36.md#usv-xyplot-append)显示了一个通过追加*Y*值从1变化到1的*X–Y*数据对象与*Y*值从0.5变化到0.5的*X–Y*数据对象而创建的*X–Y*图。

**图47–34** 使用**append**函数生成的*X–Y*图。

![](../graphics/usv-xyplot-append.png)

**要追加两个或多个X–Y数据对象：**

1. 找到**Operate on XY Data**对话框。从主菜单栏，选择****Tools**![](../graphics/images/arrow.gif)**XY Data**![](../graphics/images/arrow.gif)**Create****。在出现的对话框中点击**Operate on XY data**；然后点击**Continue**。**Operate on XY Data**对话框出现。
2. 从列出的**Operators**中，点击**append(X,X,...)**。**append**函数出现在表达式窗口中。
3. 从**XY Data**选项中，按您希望Abaqus连接它们的顺序点击要追加的*X–Y*数据对象的名称，然后点击**Add to Expression**。您可以从本会话中之前保存的*X–Y*数据对象中选择（按**XY Data**字段中的字母顺序列出）。*X–Y*数据对象名称以逗号分隔，出现在表达式窗口中**append**函数括号内。
4. 要继续构建表达式，将光标放在表达式窗口中，然后输入或选择要包含的函数、运算符和*X–Y*数据。
5. 要计算并显示表达式，点击**Plot Expression**。
6. 要[保存](pt05ch47s02hlb07.md)新的*X–Y*数据对象，点击**Save As**，然后在出现的对话框中提供名称。保存数据对象使其可用于本会话中的后续操作，以及包含多个数据对象的*X–Y*图中。
7. 完成后，点击**Cancel**关闭对话框。
