# 82.11 以 stereolithography (STL) 格式导出零件或装配





从主菜单栏选择 ****Plug-ins**![](../graphics/images/arrow.gif)**Tools**![](../graphics/images/arrow.gif)**STL Export**** 以将 Abaqus/CAE 模型中的零件或装配导出为 stereolithography (STL) 格式文件（`*.stl`）。您可以从单个零件或整个装配导出数据，也可以从选定的几何或网格数据导出。

要导出几何数据，请从 Part 模块或 Assembly 模块执行 STL 导出。插件在转换为 STL 格式期间将任何实体零件转换为 shell 几何。此外，插件使用零件的面表示来为导出的几何创建网格。生成网格的精度取决于 facet 化的程度，您可以使用 **Curve refinement** 选项控制 facet 化。有关更多信息，请参见 ["控制曲线细化，" 第76.4节](pt07ch76hla03.md)。

要导出网格数据，请从 Mesh 模块或 Visualization 模块执行 STL 导出。在转换为 STL 格式期间，插件在中等节点处分割二次单元以创建线性单元，并将任何四边形单元拆分为三角形单元。当从 Visualization 模块导出装配数据时，插件从视口中当前选择的绘图状态导出数据，而不管该绘图状态是否处于未变形绘图状态。

您可以 ASCII 格式或二进制格式导出 STL 文件。

**要将零件或装配中的数据导出为 STL 格式：**

1. 切换到 Part 模块或 Mesh 模块以导出零件数据；或切换到 Assembly 模块或 Visualization 模块以导出装配数据。
2. 在当前视口中显示要导出的零件或装配。
3. 从主菜单栏中，选择 ****Plug-ins**![](../graphics/images/arrow.gif)**Tools**![](../graphics/images/arrow.gif)**STL Export****。出现 **STL Export** 对话框。
4. 从 **STL file name** 选项中，指定要创建的 STL 格式文件的名称以及要创建它的位置。如果不指定位置，插件会在当前工作目录中创建文件。
5. 从 **STL file type** 选项中，指定要以 **ASCII** 还是 **Binary** 格式创建 STL 格式文件。
6. 点击 **OK** 导出模型数据。插件转换所选零件或装配中的模型数据，并在您指定的位置创建一个新的 STL 格式文件。如果任何 Abaqus 数据无法转换为 STL 格式，插件会在日志文件中记录有关未转换数据的错误或警告。
