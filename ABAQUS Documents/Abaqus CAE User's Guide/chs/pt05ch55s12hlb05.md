# 55.12.5 控制梁截面显示







您可以通过在 Visualization 模块中选择 ****视图**![](../graphics/images/arrow.gif)**ODB 显示选项**** 来查看结果数据的梁截面真实显示。梁渲染仅在 Visualization 模块中可用于未变形、变形和轮廓绘图。对于变形绘图，Abaqus/CAE 在更改变形比例因子时会缩放变形的位移分量，但不会缩放变形的旋转分量。

此外，由于梁截面的渲染需要输出数据库中存在梁取向数据，因此只有在使用输出数据库中请求了节点输出时，您才可以在后处理中显示梁截面。有关更多信息，请参见 ["使用分析过程中集成的梁截面来定义截面行为，" Abaqus Analysis User's Guide](../usb/usb-link.md#usb-elm-eusingbeamsection) 的第 29.3.6 节。

Abaqus/CAE 不会沿长度方向渲染梁截面的锥度。如果您的模型包含锥形梁截面，Abaqus/CAE 会沿整个长度使用梁的起始截面来渲染这些梁。有关锥形梁的更多信息，请参见 ["创建梁截面，" 第 12.13.11 节](pt03ch12s13s04.md)。

显示梁截面可用于检查正确的截面是否已分配给特定区域，以及分配的梁取向是否导致预期的截面方向。例如，[图 55–19](pt05ch55s12hlb05.md#cargo-crane-example-usi1) 显示了在 "示例：货物起重机，" Getting Started with Abaqus: Interactive Edition](../gsa/gsa-link.md#gsa-bms-exacargocrane) 的第 6.4 节中描述的轻型服务起重机上显示的箱形梁截面。

**图 55–19** 显示梁截面的货物起重机示例。

![](../graphics/usi-prp-cargocrane.png)

**注意：**如果您的结果包含二次梁元素，Abaqus/CAE 在渲染梁截面时不会考虑这些元素中的中节点。 

如果您为线指定了通用梁截面，Abaqus/CAE 会将梁截面显示为椭圆，其横截面积和惯性矩（![](../graphics/usi_eqn00695.gif) 和 ![](../graphics/usi_eqn00696.gif)）与您指定的值匹配。如果您为线指定了桁架截面，Abaqus/CAE 会将桁架截面显示为圆，其横截面积与您指定的值匹配。

Abaqus/CAE 根据当前的颜色编码和透明度设置渲染梁截面。当这些设置更改时，梁截面的颜色和透明度也会更改。显示梁截面时，Abaqus/CAE 会禁用模型的缩放和收缩。 

**控制梁截面显示：**

1. 找到 **渲染梁截面** 选项。从主菜单栏，选择 ****视图**![](../graphics/images/arrow.gif)**ODB 显示选项****。在出现的对话框中，点击 **常规** 选项卡。
2. 从对话框底部切换 **渲染梁截面** 以显示梁截面。
3. 如果需要，应用 **比例因子** 到梁截面以增加或减小其大小。默认值为 1。
4. 点击 **确定** 实现您的更改并关闭对话框。Abaqus/CAE 会以适当的尺寸和正确的方向显示梁截面。默认情况下，您的更改会在会话期间保存。如果您希望保留所做的更改以供后续会话使用，请将它们保存到文件。有关更多信息，请参见 ["保存自定义设置以供后续会话使用，" 第 55.1.1 节](pt05ch55s01s01.md)。
