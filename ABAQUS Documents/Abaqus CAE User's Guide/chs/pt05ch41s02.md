# 41.2 生成诊断信息





Abaqus在分析作业期间生成诊断信息；Abaqus/CAE读取存储在输出数据库中的诊断信息。因此，您可以使用**作业诊断**对话框仅查看为在输出数据库中生成信息的程序提供的收敛信息。诊断信息存储在以下Abaqus/Standard程序的输出数据库中：
- **耦合温度-位移**
- **地静力**
- **土壤**
- **静态，通用**
- **静态，线性摄动**
- **静态，Riks**
- **黏弹性**

其他Abaqus/Standard分析程序的诊断信息位于消息文件中，无法在Abaqus/CAE中查看。

Abaqus/CAE中的诊断信息可用于所有Abaqus/Explicit分析程序。退火程序不生成任何诊断数据，因此退火步骤不显示诊断。

默认情况下，会为Abaqus/Standard和Abaqus/Explicit中支持的分析程序将详细诊断信息保存到输出数据库。如果不想包含诊断信息，请使用**关键词编辑器**编辑输入文件，在模型数据中包含以下行：

```
[*OUTPUT](../key/key-link.md#usb-kws-houtput), DIAGNOSTICS=NO
```
您还可以使用**关键词编辑器**通过为关键词[*DIAGNOSTICS](../key/key-link.md#usb-kws-hdiagnostics)包含一行或多行并指定所需的参数值来更改Abaqus/Explicit的默认诊断输出参数。

有关使用**关键词编辑器**的更多信息，请参见"将不支持的关键词添加到您的Abaqus/CAE模型"，第9.10.1节。
