# 47.4.15 对X–Y数据对象进行插值







### 47.4.15 对X–Y数据对象进行插值

使用 **interpolate** 函数对[之前保存的](pt05ch47s02hlb07.md) *X–Y* 数据对象（有序对的集合）进行运算，生成一个新的 *X–Y* 数据对象。对具有非均匀数据的 *X–Y* 数据对象进行插值会产生一个 *X–Y* 数据对象，其新的插值 *Y* 坐标值出现在 *X* 轴上规则递增的位置。默认情况下，数据以源数据中最小 *X* 增量相等的增量进行插值。

[图47-13](pt05ch47s03hlb13.md#usv-xyplot-interpolate) 显示了使用 **interpolate** 函数生成的 *X–Y* 曲线图示例，叠加在原始数据上。

**图47-13** 使用 **interpolate** 函数生成的 *X–Y* 曲线图。

![](../graphics/usv-xyplot-interpolate.png)

*X–Y* 数据对象的名称是 **interpolate** 函数的唯一必需参数。可选参数的说明如下：
- 插值数据点之间的 *X* 轴增量（*xIncrement*）。如果不为此参数指定值或指定非正值，Abaqus/CAE 使用原始数据集的最小 *X* 增量。
- 指定插值方案的符号常数（*interpolation*）。此参数的有效值为 QUADRATIC（指定拉格朗日二阶插值方案）、CUBIC_SPLINE（指定三次样条插值方案）和 LINEAR（指定线性插值方案）。默认值为 QUADRATIC。
- 到达第一个数据点之前的原始数据曲线的斜率（*startslope*）。此参数的默认值为0.0（水平斜率），仅在 interpolation=CUBIC_SPLINE 时使用。
- 越过最后一个数据点之后的原始数据曲线的斜率（*endslope*）。此参数的默认值为0.0（水平斜率），仅在 interpolation=CUBIC_SPLINE 时使用。

**对X–Y数据对象进行插值：**

1. 找到 **Operate on XY Data** 对话框。从主菜单栏，选择 ****Tools**![](../graphics/images/arrow.gif)**XY Data**![](../graphics/images/arrow.gif)**Create****. 在出现的对话框中点击 **Operate on XY data**；然后点击 **Continue**。**Operate on XY Data** 对话框出现。
2. 从列出的 **Operators** 中，点击 **interpolate(X)**。**interpolate** 函数出现在表达式窗口中。
3. 从 **XY Data** 选项中，点击要操作的 *X–Y* 数据对象的名称，然后点击 **Add to Expression**。您可以从本次会话中之前保存的所有 *X–Y* 数据对象中选择（在 **XY Data** 字段中按字母顺序列出）。*X–Y* 数据对象名称出现在表达式窗口中 **interpolate** 函数的括号内。
4. 要继续构建表达式，将光标定位在表达式窗口中，然后输入或选择要包含的函数、运算符和 *X–Y* 数据。
5. 要评估并显示表达式，点击 **Plot Expression**。
6. 要[保存](pt05ch47s02hlb07.md)您的新 *X–Y* 数据对象，点击 **Save As**，然后在出现的对话框中提供名称。保存数据对象使其可用于本次会话中的未来操作以及包含多个数据对象的 *X–Y* 曲线图。
7. 完成后，点击 **Cancel** 关闭对话框。
