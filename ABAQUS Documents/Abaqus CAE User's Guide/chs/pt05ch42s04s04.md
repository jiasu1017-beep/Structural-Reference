# 42.6.4 了解结果缓存

缓存是系统内存的一部分。您可以在后处理期间选择将分析结果存储在缓存中，以加快屏幕上图像的生成。当结果被缓存时，Abaqus/CAE可以在不重新计算值用于显示或不经常访问输出数据库文件的情况下绘制结果。这可以大大提高性能。但是，缓存结果会增加内存使用。如果您发现Abaqus/CAE正在消耗工作站上的大量内存，从而阻碍系统性能，则可以禁用结果缓存选项以清除与结果缓存关联的内存。清除此内存可以提高整体性能；但是，结果的显示会变慢。切换到新的输出变量或输出数据库将自动清除当前结果缓存。有关的详细信息，请参阅["控制结果缓存，" 第42.6.10节](pt05ch42s04hlb06.md)。

Abaqus/CAE可以在分析作业仍在运行且数据库继续更新时显示来自输出数据库的数据。在大多数可视化模块操作和动画期间，Abaqus/CAE监控输出数据库的更新结果并相应地更新当前视口。如果您正在从远程输出数据库显示数据，如果通过网络的数据库监控时间很长，Abaqus/CAE的性能可能会下降。要提高性能，您可以减少Abaqus/CAE监控输出数据库更新的频率，或者可以禁用监控。有关的详细信息，请参阅["在远程计算机上访问输出数据库，" 第9.3节](pt02ch09s03.md)。
