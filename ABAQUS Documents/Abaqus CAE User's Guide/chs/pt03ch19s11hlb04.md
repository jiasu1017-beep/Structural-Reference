# 19.11.4 查看共同执行的结果







共同执行完成后，Abaqus/CAE 将每个分析作业的结果存储在输出数据库中。您可以使用可视化模块以图形方式查看组合结果。

**查看共同执行的结果：**

1. 从**共同执行管理器**中，选择共同执行名称并点击**结果**。可视化模块启动，并向您显示两个模型的叠加图。
2. 使用可视化模块来创建和自定义不同的结果图。有关使用叠加图的更多信息，请参见[第 79 章，"叠加多个图](pt07ch79.md)。"
