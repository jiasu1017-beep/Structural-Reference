# 16.9.7 定义体力





您可以定义体力以在物体上每单位体积指定载荷。

**创建或编辑体力：**

1. 使用以下方法之一显示体力载荷编辑器：- 若要创建新的体力载荷，请遵循"创建载荷"，第16.8.1节中概述的过程（**类别**：`Mechanical`；**所选步骤的类型**：`Body force`）。- 若要使用菜单或管理器编辑现有的体力载荷，请参见"编辑依赖于步骤的对象"，第3.4.13节。若要编辑应用载荷的区域，请参见"编辑应用规定条件的区域"，第16.8.4节。
2. 如果编辑器中出现**分布**字段，单击该字段右侧的箭头，并从出现的列表中选择您的选项：- 选择**均匀**以定义在物体上均匀的载荷。- 选择**用户定义**以在用户子程序[`DLOAD`](../sub/sub-link.md#sub-xsl-dload)（对于Abaqus/Standard）或[`VDLOAD`](../sub/sub-link.md#sub-xsl-vdload)（对于Abaqus/Explicit）中定义载荷幅值。有关更多信息，请参见以下章节：- "指定常规作业设置"，第19.8.6节- "Abaqus User Subroutines Reference Guide"第1.1.5节"DLOAD"- "Abaqus User Subroutines Reference Guide"第1.2.2节"VDLOAD"- 选择分析场以定义空间变化载荷。只有对该载荷类型有效的分析场才会在选择列表中显示。或者，您可以单击**创建**以创建新的分析场。（更多信息，请参见第58章"分析场工具集"。）
3. 如果您选择了**均匀**或分析场分布选项，请执行以下步骤：1. 在**分量1**、**分量2**和（如果您在三维空间中工作）**分量3**字段中，输入每个方向的每单位体积体力（单位[FL<sup>-3</sup>](../popups/usb-int-iconventions-unitsym.md)）：- 如果您在三维或二维空间工作，**分量1**、**分量2**和**分量3**字段对应于1、2和（如果适用）3方向。- 如果您在轴对称空间工作，**分量1**对应于径向方向，**分量2**对应于轴向方向。2. 如果需要，单击**幅值**字段右侧的箭头，并从出现的列表中选择您选择的幅值。或者，您可以单击![](../graphics/ico_amplitudeSmall.png)创建新的幅值。（更多信息，请参见第57章"幅值工具集"。）3. 单击**确定**保存您的数据并退出编辑器。
4. 如果您选择了**用户定义**分布选项，请执行以下步骤：1. 如果需要，在**分量1**、**分量2**和（如果适用）**分量3**字段中输入每个方向的每单位体积体力（单位[FL<sup>-3</sup>](../popups/usb-int-iconventions-unitsym.md)）。您在编辑器中输入的载荷幅值数据在Abaqus/Standard分析中被传递到用户子程序中，但在Abaqus/Explicit分析中被忽略。2. 单击**确定**保存您的数据并退出编辑器。3. 进入作业模块并显示感兴趣的分析作业的作业编辑器。（更多信息，请参见"创建、编辑和操作作业"，第19.7节。）4. 在作业编辑器中，单击**常规**选项卡，并指定包含定义载荷幅值的用户子程序的文件。更多信息，请参见"指定常规作业设置"，第19.8.6节。**注意：**您只能在作业编辑器中指定一个用户子程序文件；如果您的分析涉及多个用户子程序，您必须将用户子程序合并到一个文件中，然后指定该文件。
