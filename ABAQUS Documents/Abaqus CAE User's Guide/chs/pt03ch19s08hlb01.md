# 19.8.1 导航作业自定义选项







使用作业编辑器在提交作业进行分析之前自定义作业设置。要找到作业编辑器，请从主菜单栏中选择****作业**![](../graphics/images/arrow.gif)**编辑**![](../graphics/images/arrow.gif)***作业名称*****。作业编辑器包含以下选项卡页面：

**[**提交**](pt03ch19s08hlb02.md)**

使用**提交**选项卡页面配置作业的基本属性，如作业类型、运行模式和运行时间。您也可以使用提交页面将作业提交到[远程队列](pt03ch19s02s06.md)。

**[**常规**](pt03ch19s08hlb06.md)**

使用**常规**选项卡页面配置预处理器打印输出，并指定临时目录。预处理器打印选项不适用于与输入文件关联的作业；您必须在输入文件本身中指定这些选项。您还可以使用**常规**选项卡页面指定模型引用的任何用户子程序以及结果输出格式。

**[**内存**](pt03ch19s08hlb07.md)**

使用**内存**选项卡页面配置分配给 Abaqus 分析的内存量。

**[**并行化**](pt03ch19s08s01.md)**

使用**并行化**选项卡页面配置 Abaqus 分析作业的并行执行，如要使用的处理器数量和并行化方法。

**[**精度**](pt03ch19s08hlb08.md)**

使用**精度**选项卡页面为 Abaqus/Explicit 分析指定单精度或双精度。您还可以选择在分析过程中写入输出数据库的节点输出精度。

您也可以使用 Abaqus 环境文件（`abaqus_v6.env`）来控制作业编辑器选项卡页面中许多设置的默认值。有关更多信息，请参见["使用 Abaqus 环境设置，"《Abaqus 分析用户手册》第 3.3.1 节](../usb/usb-link.md#usb-int-denvfile)。![](../graphics/images/black4rule.gif)有关相关主题的信息，请点击以下任一项目：- ["理解分析作业，"第 19.2 节](pt03ch19s02.md)
