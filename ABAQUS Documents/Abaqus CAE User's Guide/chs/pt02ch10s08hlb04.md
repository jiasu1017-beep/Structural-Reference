# 10.9.4 将视口数据导出为VRML格式文件





从主菜单栏选择**文件**![](../graphics/images/arrow.gif)**导出**![](../graphics/images/arrow.gif)**VRML**，可以将当前视口中的模型导出为VRML格式文件。VRML格式文件可用于在Web浏览器或独立VRML客户端中显示三维图像。VRML文件对所有长度和距离使用米为单位。需要特殊的插件查看器（如Cortona或Cosmo）来查看VRML文件。

您可以在Abaqus/CAE会话期间随时将模型导出为标准VRML文件或压缩VRML文件。压缩VRML文件通常比相应的标准VRML文件小得多，但压缩比取决于模型的内容。

VRML文件以ASCII格式存储，可以在标准文本编辑器中查看。压缩VRML文件必须解压缩才能转换为ASCII格式。

动画和二维图像（如草图或图层）不会被导出。（有关导出动画的信息，请参见"保存动画文件"，第49.3节。）此外，视口方向或坐标系三轴以及视口箭头注释也不会被导出。视口图像中的文本字体在VRML文件中可能会显得较小；您可以在视口中增加字体大小以确保文本在VRML文件中清晰可见（更多信息，请参见"自定义字体"，第3.2.8节）。

当您导出版壳或膜表面并显示面标签的图像时，面标签仅相对于在Abaqus/CAE中显示图像的视图出现在VRML文件中。如果您旋转图像或在VRML查看器中更改视图，显示的面标签可能并不适合显示的面。例如，如果您导出版模型且显示正（`SPOS`）面，即使旋转模型显示负面的`SPOS`标签也会出现在VRML查看器中。

**将视口数据导出为VRML格式文件：**

1. 从主菜单栏，选择**文件**![](../graphics/images/arrow.gif)**导出**![](../graphics/images/arrow.gif)**VRML**。出现**导出至VRML文件**对话框。
2. 从对话框底部的**文件过滤器**菜单中，选择以下之一：**VRML (*.wrl)** Abaqus/CAE列出所选目录中所有具有`.wrl`文件扩展名的文件。**压缩VRML (*.wrz)** Abaqus/CAE列出所选目录中所有具有`.wrz`文件扩展名的文件。
3. 选择要导出图像的文件，或在**文件名**文本字段中输入新文件的名称。
4. 单击**确定**导出图像并关闭**导出至VRML文件**对话框。
