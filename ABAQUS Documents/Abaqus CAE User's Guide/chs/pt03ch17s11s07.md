# 17.11.7 为自底向上扫掠网格定义连接边





为自底向上扫掠网格定义连接边是一项复杂的任务。连接边仅在使用孤立单元面作为源侧创建扫掠网格时是必需的。但是，即使不需要连接边，使用它们也可以在创建网格后为您节省大量时间。连接边有助于控制扫掠网格的形状，并且根据您使用的是几何体还是网格，它们会强制网格与几何体之间的关联性或强制网格沿连接边匹配现有网格。换句话说，连接边强制自底向上扫掠网格更类似于自顶向下扫掠网格。如果没有连接边，您可能需要编辑网格形状、合并节点或以其他方式修改网格以符合相邻几何体或网格，并且如果有任何载荷或边界条件应用于这些边，您将需要手动将网格与周围的边关联。

自底向上网格的连接边可以是几何面、单元面和二维单元的组合。单元或单元面必须是四边形，单个连接边不能同时包含网格实体和几何面。下面的标准和要求将帮助您选择有效的连接边或从最初无效的边创建有效的连接边：
- 每个几何面必须有四个逻辑边。
- 每个几何面的角必须与网格扫掠操作一致。
- 连接边必须与源侧共享一条公共边。
- 几何面或网格实体必须形成规则的网格图案。

**每个几何面必须有四个逻辑边**

在图17-99中，顶面包含一个孔，前表面包含一个三角形面。由于这些额外特征，这两个面都不能用作连接边。

**图17-99** 连接边必须有四个逻辑边。

![](../graphics/mgn-buconn1.png)

如果您选择这样的边，Abaqus/CAE会显示错误消息，指示这些面在拓扑上不是四边形的。

除了拓扑测试外，Abaqus还会应用几何测试来确定具有四条或多条边的面是否应被视为四边形的。几何测试评估面顶点的角度，并确定是否可以生成高质量的结构化网格。您可以使用网格控制将这些面的结构化网格技术分配给这些面来覆盖几何测试。

例如，在图17-100中，前表面在拓扑上是四边形的；但是，由于各边沿曲线与顶边相接，Abaqus/CAE无法识别所有四个角。

**图17-100** 具有角半径的面可能需要编辑才能用作连接边。

![](../graphics/mgn-buconn2.png)

当您完成选择连接边时，Abaqus/CAE会高亮显示在几何上不是四边形的面并显示错误消息。您可以使用网格控制重新定义区域的角（如图中高亮显示），以便该面可用作连接边。有关在自底向上网格区域面上使用网格控制的更多信息，请参见["提高自底向上区域的边界网格质量，"第17.11.6节](pt03ch17s11s06.md)。

**每个几何面的角必须与网格扫掠操作一致**

图17-101中所选面上显示的四个角与相邻源侧（零件的底侧）的角不匹配。在这种情况下，您可以完成选择连接边；但是当您尝试应用自底向上扫掠网格时，Abaqus/CAE会指示它无法将连接边上的网格映射到规则网格。如果您使用网格控制移除半圆形切口处的角并在边的左下角添加一个角，则可以将该面用作连接边。

**图17-101** 角与源侧角不匹配的四边形面。

![](../graphics/mgn-buconn-notgrid.png)

**连接边必须与源侧共享一条公共边**

源侧和连接边之间共享的公共边不能超出源侧的边。此外，如果您使用二维单元或单元面作为源侧并选择几何面作为连接边，反之亦然，则单元边必须与几何边关联，否则Abaqus不会将它们识别为共享公共边。作为将单元与几何边关联的替代方法，您可以在几何面上创建自底向上边界网格，并手动合并源侧和连接边网格的节点（更多信息，请参见["为自底向上区域创建边界网格，"第17.11.5节](pt03ch17s11s05.md)）。如果连接边与源侧不共享公共边，Abaqus/CAE会指示它无法将连接边上的网格映射到规则网格。

**几何面或网格实体必须形成规则的网格图案**

您选择的几何面或单元面和二维单元必须形成规则的网格图案。图17-102提供了两个可接受的连接面图案示例。

**图17-102** 扫掠网格的可接受连接面图案。

![](../graphics/mgn-goodsweeppattern-nls.png)

图17-103中三个四边形面的组合使其不能作为连接边。在这种情况下，当您指示完成选择连接边时，Abaqus/CAE会显示错误消息，说明所选几何面不形成网格图案。

**图17-103** 不可接受的面图案。

![](../graphics/mgn-buconnbadgrid.png)

如果您无法创建可接受的连接边，可以省略连接边。生成的网格可能仍然可以接受分析。但是，您应该记住将区域边上的单元与几何体关联，特别是如果网格的边上有载荷或其他分析属性。更多信息，请参见["网格-几何体关联，"第17.12节](pt03ch17s12.md)。
