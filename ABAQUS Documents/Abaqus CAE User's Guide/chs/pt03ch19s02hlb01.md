# 19.2.6 监控分析作业的进度





**作业管理器**和**协同执行管理器**持续更新模型数据库中分析作业的状态。此外，Abaqus/CAE 将分析产品的错误消息打印到消息区域，并在当前工作目录中创建诊断文件。

您可以通过从主菜单栏选择 ****作业**![](../graphics/images/arrow.gif)**监控**![](../graphics/images/arrow.gif)***您选择的作业*****，或选择您选择的作业并在**作业管理器**中单击**监控**来监控有关已提交作业的信息。该作业的作业监控对话框出现，如图 19-2 所示（[图 19-2](pt03ch19s02hlb01.md#ana-monitorjob)）。您可以根据需要显示多个作业监控器，以查看多个作业的信息。

**图 19-2** 作业监控器。

![](../graphics/ana-monitorjob-nls.png)

为协同执行提交的作业显示在模型树的**作业**容器中的协同执行下的**协同执行**容器中。您可以通过在模型树中鼠标按钮 3 单击作业并选择**监控**来监控这些作业。

作业监控对话框的上半部分显示 Abaqus 为分析创建的状态文件中可用的信息。表标题根据从分析发送的设置针对每个作业进行自定义。对话框的下半部分显示以下信息：
- 单击**日志**选项卡显示分析的日志文件中出现的开始和结束时间。
- 单击**错误**和**警告**选项卡显示与分析相关的错误或警告。Abaqus/CAE 通过在**错误**和**警告**选项卡前添加感叹号来表示存在错误或警告。如果模型的特定区域导致错误或警告，将自动创建包含该区域节点或单元集合。节点或单元集合的名称与错误或警告消息一起显示，您可以使用显示组在可视化模块中查看该集合。（有关显示组的更多信息，请参见[第 78 章，"使用显示组显示模型的子集"](pt07ch78.md)。）Abaqus/CAE 在创建输入文件时可能不会执行所有一致性检查，这可能导致分析期间出现警告或错误消息。如果您的分析生成警告或错误消息，请考虑运行数据检查分析（请参阅["对模型执行数据检查，" 第 19.7.3 节"](pt03ch19s07hlb03.md)）来诊断和修复模型中的可能问题。作业监控器中显示的错误和警告消息数量受环境参数 **cae_error_limit** 和 **cae_warning_limit** 的限制（详情请参阅["作业自定义参数，" Abaqus Installation and Licensing Guide 第 4.1.3 节](../sgb/sgb-link.md#sgb-chp-env-jobcustom)）。如果错误或警告数量超过作业监控器限制，请查阅数据、消息或状态文件以获取完整的消息列表。
- 单击**输出**选项卡显示每个输出数据条目写入输出数据库的记录。此外，如果您请求 Abaqus 监控特定节点特定自由度向消息和状态文件输出的值，**输出**选项卡页面会记录每次写入此信息的时间以及分析中该点的自由度值。（有关更多信息，请参见["理解输出请求，" 第 14.4 节"](pt03ch14s04.md)和["自由度监控请求，" 第 14.5.4 节"](pt03ch14s05s04.md)。）
- 随着分析的进行，Abaqus 会创建以下文件：- Abaqus/Standard：数据文件、消息文件和状态文件- Abaqus/Explicit：数据文件、消息文件和状态文件- Abaqus/CFD：数据文件和状态文件Abaqus/CAE 会相应地激活**数据文件**、**消息文件**和**状态文件**选项卡。您可以单击任何这些选项卡来浏览或搜索相应文件以获取其他错误和警告消息。**注意：**Abaqus/CAE 仅对本地提交的分析填充**数据文件**、**消息文件**和**状态文件**选项卡页面；对于远程作业，此信息不会显示在作业监控器中。此外，尽管 Abaqus/CAE 会在分析运行时定期更新这些页面的内容，但数据可能并不总是与文件中最新数据同步。

有关 Abaqus 在分析期间创建的不同输出文件的详细信息，请参见["输出，" Abaqus Analysis User's Guide 第 4.1.1 节](../usb/usb-link.md#usb-out-ooutput)。

您可以在作业监控器中显示的选项卡页面中搜索特定错误或警告消息。选择所需的文件选项卡，在**查找文本**字段中输入搜索字符串，然后单击**下一步**或**上一步**在文件中从一个匹配位置跳到下一个。切换**区分大小写**以执行区分大小写的搜索。

作业监控器对话框中的信息会随着分析的进行而持续更新。如果作业失败，**错误**选项卡页面会自动出现在其他选项卡页面前面，以帮助您确定失败的原因。此外，如果输出任何错误或警告消息，选项卡上会出现一个感叹号。

如果您启动监控作业然后退出 Abaqus/CAE、关闭当前模型数据库或打开新的模型数据库，Abaqus/CAE 将停止更新作业监控器。作业将继续运行；但是，作业监控器不会报告作业状态或更新增量信息。

如果您为特定节点的特定自由度请求了 **DOF 监控**输出，Abaqus/CAE 提供了另一种通过随时间绘制自由度值来监控作业的机会。绘图出现在提交作业时自动生成的新视口中。如果画布的可见部分已经填充了一个或多个视口，新视口可能放置在画布不可见的部分上；在这种情况下，您应该平铺或层叠视口或放大画布以使视口可见。（有关为特定节点的特定自由度请求输出的信息，请参见["自由度监控请求，" 第 14.5.4 节"](pt03ch14s05s04.md)。）

如有必要，您可以通过单击作业监控器对话框底部的**终止**来终止分析作业。
