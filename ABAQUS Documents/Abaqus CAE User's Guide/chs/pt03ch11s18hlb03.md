# 11.18.3 复制零件





从主菜单栏选择**零件**![](../graphics/images/arrow.gif)**复制**![](../graphics/images/arrow.gif)***零件名称***以将零件复制到新零件。您可以复制包含几何和孤儿元素的零件。压缩、缩放和镜像功能对几何和网格都起作用。分离断开连接区域功能也可以处理同时包含几何和孤儿元素的零件；但是，只有几何区域被复制为新零件，因为孤儿元素不存在区域。

**复制零件：**

1. 从主菜单栏，选择**零件**![](../graphics/images/arrow.gif)**复制**![](../graphics/images/arrow.gif)***零件名称***。Abaqus/CAE显示**零件复制**对话框。
2. 在对话框中，输入新零件的名称。
3. 您可以创建原始零件的完全相同的副本，也可以从以下**复制选项**中进行选择：- **压缩特征**- **按比例缩放零件**- **关于平面镜像**- **将断开连接的区域分离为零件**。
4. 单击**确定**。Abaqus/CAE关闭**零件复制**对话框并将选定的零件复制到新零件。新零件成为当前零件。
