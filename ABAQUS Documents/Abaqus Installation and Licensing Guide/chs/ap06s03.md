# F.3 特殊情况





本节解答有关通常不经常发生的特殊操作的问题。

### F.3.1 如果我们的网络只有一个 FLEXnet 许可证服务器，我们如何添加冗余服务器？

您需要决定要使用哪些其他计算机作为冗余许可证服务器。冗余许可证服务器的总数为三个，必须始终在主服务器和至少一个其他备份服务器（shadow）之间保持可靠的通信，以便许可操作。三个服务器主机必须在同一子网上。

1. 将所有三个许可证服务器主机的 lmhostid 发送到您当地的办事处或代表，并请求冗余服务器的许可证文件。
2. 安装冗余许可证文件并手动启动 Abaqus 许可证服务器，如["冗余 FLEXnet 许可证服务器配置，"第 3.4 节](ch03s04.md)中所述。
3. 对于每个 Abaqus 安装，编辑 Abaqus 环境文件（`abaqus_v6.env`），使行 ``` abaquslm_license_file="[*port*]@*license_server_host*" ``` 变为 ``` abaquslm_license_file="[*port*]@*serverhost1*:[*port*]@*serverhost2*: [*port*]@*serverhost3*" ``` 其中 *serverhost1*、*serverhost2* 和 *serverhost3* 是运行许可证服务器的计算机的名称（引号是必需的）。如果许可证服务器使用的端口不在 27000 到 27009 之间，则必须包含服务器主机名以及端口号。列表中的项目在 Linux 平台上用冒号分隔，在 Windows 平台上用分号分隔。每个条目必须以 "@" 符号或端口号和 "@" 符号开头。确保用户对其主目录和本地目录中的任何环境文件进行类似的更改。

### F.3.2 如何判断我是否在使用冗余 FLEXnet 服务器配置？

查看位于 `*simulia_dir*/License` 目录中的 Abaqus 许可证文件（`simulialm.lic`）。如果您使用的是冗余服务器配置，许可证文件中将有三个 `SERVER` 行。服务器名称应该不同，但端口号可能对所有三个都相同。例如：

```
SERVER bifrost.simulia.com 69084992 27003
SERVER tank.simulia.com 69094954 27003
SERVER willow.simulia.com 69094344 27003
VENDOR ABAQUSLM
```

### F.3.3 如果我在批处理队列中运行分析作业会怎样？

FLEXnet 许可证直到分析实际开始时才被检出，因此批处理队列中的作业不会持有令牌。但是，如果作业在批处理队列中执行且没有足够的令牌可用，作业将被放入许可证队列直到令牌空闲，这将阻止队列中的其他作业执行。为避免批处理队列中浪费时间，建议在第三方批处理排队系统中将令牌注册为资源。有关令牌资源的 LSF 示例，请参阅["派生和使用自定义队列类，"第 4.3.6 节](ch04s03.md#sgb-env-exa-customclass)。

### F.3.4 许可证队列和批处理队列之间有什么区别？

批处理队列控制如何在特定计算机上运行作业以及何时运行（请参阅["定义分析批处理队列，"第 4.2 节](ch04s02.md)）。当 Abaqus 作业请求的令牌多于当前可用令牌时，许可证队列由许可证服务器创建。作业在到达批处理队列的前面并开始执行之前不会请求令牌。

### F.3.5 如果我们有多个非冗余网络许可证服务器，如何连接到特定服务器？

多个非冗余许可证服务器的情况与冗余服务器的情况明显不同。在冗余服务器中，一个服务器作为主服务器，其他服务器作为备份（shadow）以防主服务器出现故障。对于多个非冗余服务器，每个服务器都独立于其他服务器。要连接到特定服务器，您必须在本地 Abaqus 环境文件（`abaqus_v6.env`）中设置 **abaquslm_license_file** 参数，该文件位于您的主目录或当前工作目录中。参数定义的正确格式如下：

```
abaquslm_license_file="[*port*]@*license_server_host*"
```

其中 *license_server_host* 是运行许可证服务器的计算机的名称（应包含引号）。如果许可证服务器使用的端口不在 27000 到 27009 之间，则必须包含服务器主机名以及端口号。多个服务器可以在 **abaquslm_license_file** 参数中列出，如下所示：

```
abaquslm_license_file="[*port*]@*server1*:[*port*]@*server2*:
  [*port*]@*server3*:[*port*]@*server_n*"
```

Abaqus 作业将按列出的顺序尝试连接到服务器。

### F.3.6 在 DHCP 配置的网络中使用 FLEXnet Licensing 是否有任何特殊要求？

许可证文件服务器行上的主机名必须是计算机名称，而不是 IP 地址。客户端必须在 Abaqus 环境文件（`abaqus_v6.env`）中的 **abaquslm_license_file** 参数中通过主机名引用许可证服务器。
