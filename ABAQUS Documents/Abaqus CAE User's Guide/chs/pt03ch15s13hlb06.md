# 15.13.6 为通用接触指定主从分配





通用接触相互作用的属性（如接触公式）在相互作用编辑器的「Attribute Assignments」部分独立于接触域指定。默认情况下，Abaqus/Standard自动为相互作用对中的每个表面分配主和从角色；Abaqus/Explicit为所有表面（除了基于节点的表面和解析刚性表面）使用平衡主从公式。您可以覆盖默认分配并直接为通用接触相互作用中的表面对指定主从分配。

有关通用接触的简要概述，请参见「理解相互作用」，第15.3节。有关主从公式的更详细讨论，请参见「Abaqus/Standard中通用接触的数值控制」，Abaqus Analysis User's Guide第36.2.6节；以及「Abaqus/Explicit中通用接触的接触公式」，Abaqus Analysis User's Guide第38.2.1节。

**指定主从分配的步骤：**

1. 使用以下方法之一显示通用接触相互作用编辑器：- 要创建新的通用接触相互作用，请按照「定义通用接触」，第15.13.1节中的说明操作。- 要编辑现有的通用接触相互作用，请从主菜单栏选择 ****Interaction**![](../graphics/images/arrow.gif)**Edit**![](../graphics/images/arrow.gif)***相互作用名称*****。
2. 点击相互作用编辑器「Attribute Assignments」部分中的「Contact Formulation」选项卡。
3. 点击以下字段旁边的![](../graphics/ico_edit.png)：- 用于Abaqus/Standard通用接触定义的「Master-slave assignments」。- 用于Abaqus/Explicit通用接触定义的「Pure master-slave assignments」。此时会出现「Edit Master-Slave Assignments」对话框。默认情况下，当您从列表或表中选择一个表面时，Abaqus/CAE会在视口中高亮显示该表面；但是，高亮不适用于「(Global)」和「(Self)」。您可以关闭对话框底部的「Highlight selected regions」以关闭选择高亮。
4. 从左侧第一列的现有表面列表中选择一个或多个表面。选择「(Global)」为整个接触域与单个表面之间的接触分配纯主从权重。**提示：**您可以点击![](../graphics/ico_surface.png)定义新表面并将其添加到列表中。请参见「创建表面」，第73.3.3节，获取定义表面的说明。
5. 从第二列的现有表面列表中选择第二个或多个表面以定义表面对。- 当在任一列中选择多个表面时，所有可能的组合都会生成。- 对于Abaqus/Standard通用接触定义，在第二列中选择相同的表面名称或「(Self)」为特定表面上的自接触相互作用指定平衡主从公式。- 如果主表面和从表面重叠，则会排除重叠区域的自接触。
6. 点击对话框中间的箭头![](../graphics/usi-itn-contactattribarrow.png)将您的选择转移到主从分配列表中。对话框右侧的表会更新以反映您的选择。
7. 在「Master-Slave Assignments」表最后一列中为第一个表面指定类型。- 选择「SLAVE」表示第一个表面为从表面。- 选择「MASTER」表示第一个表面为主表面。- 选择「BALANCED」指定两个表面之间的平衡主从接触（仅在Abaqus/Standard中可用）。对于自接触表面对，您必须选择「BALANCED」。
8. 根据需要重复上述步骤以完成主从分配。如果您想删除主从分配，请选择行并点击![](../graphics/ico_delete.png)。
9. 点击「OK」保存您的选择。相互作用编辑器重新出现，并更新了直接指定的主从分配数量的信息。
10. 点击「OK」创建相互作用并关闭编辑器。
