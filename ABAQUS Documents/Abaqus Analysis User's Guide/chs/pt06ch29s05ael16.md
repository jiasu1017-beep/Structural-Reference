### 概述

本节提供 Abaqus/Standard 中可用弯头单元的参考。

### 单元类型

| ELBOW31 | 空间管道单元，具有变形截面，沿管道线性插值 |
| --- | --- |

| ELBOW32 | 空间管道单元，具有变形截面，沿管道二次插值 |
| --- | --- |

| ELBOW31B | 空间管道单元，仅考虑椭圆化，忽略椭圆化的轴向梯度 |
| --- | --- |

| ELBOW31C | 空间管道单元，仅考虑椭圆化，忽略椭圆化的轴向梯度。此公式与ELBOW31B单元类型的公式相同，只是忽略了管道圆周傅里叶插值中除第一项外的所有奇数项。 |
| --- | --- |

##### 激活的自由度

1, 2, 3, 4, 5, 6

##### 附加解变量

弯头单元有许多变量用于模拟横截面椭圆化和翘曲。变量数量取决于弯头单元类型、节点数量和选择的傅里叶模式数。在下表中，*p*是傅里叶模式数：

| 单元类型 | 变量数量 |
| --- | --- |
| ELBOW31 | 16, if *p*=0 |
| (16*p*+8), if *p*![](../graphics/usb_eqn04673.gif) 1 |
| ELBOW32 | 24, if *p*=0 |
| (24*p*+12), if ![](../graphics/usb_eqn04674.gif) 1 |
| ELBOW31B | 13+2*p*, if *p*=0,1 |
| 11+4*p*, if ![](../graphics/usb_eqn04674.gif) 2 |
| ELBOW31C | 13+2*p*, if *p*=0,1,3,5 |
| 15+2*p*, if *p*=2,4,6 |

### 单元输出

默认应力输出点在管道周围所有积分站点的内表面和外表面上。

#### 应力、应变和其他张量分量

对于具有位移自由度的单元，应力和其他张量（包括应变张量）可用。所有张量具有相同的分量。例如，应力分量如下：

| S11 | 沿管道方向的直接应力。 |
| --- | --- |

| S22 | 管道截面周围的直接应力。 |
| --- | --- |

| S12 | 管道壁内的剪切应力。 |
| --- | --- |

#### 截面力和弯矩

| SF1 | 轴向力。 |
| --- | --- |

| SM1 | 关于局部1轴的弯矩。 |
| --- | --- |

| SM2 | 关于局部2轴的弯矩。 |
| --- | --- |

| SM3 | 关于弯头轴的扭矩。 |
| --- | --- |
