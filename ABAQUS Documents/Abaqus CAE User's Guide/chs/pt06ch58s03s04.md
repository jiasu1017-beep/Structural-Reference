# 58.3.4 支持映射的单元





映射场只能用于网格中使用支持单元类型的模型。最常用的单元受支持，包括壳单元。

支持的单元类如下所列，根据代表性单元类型的拓扑结构（形状、节点数和积分点数）进行分类。任何具有相同拓扑结构的类似单元都支持映射。支持的代表性单元如下：
- S3、S4、S4R、S8R、STRI65
- C3D4、C3D6、C3D8、C3D8R、C3D10、C3D15、C3D20、C3D20R
- CPE3、CPE4、CPE6、CPE8
- CAX3、CAX4、CAX6、CAX8

以下单元类别不受支持，不能用于目标模型网格或输出数据库源网格：
- 表面单元
- 刚性单元
- 分析刚性表面单元
- 内聚单元
- 复合实体单元
- 连续壳单元
- 圆柱单元
- 欧拉单元
- 垫片单元
- 内部单元
- 使用修改的二阶插值的单元
- 具有扭曲的单元
