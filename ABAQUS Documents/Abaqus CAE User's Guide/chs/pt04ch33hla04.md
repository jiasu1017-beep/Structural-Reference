# 33.6 编辑应用惯性的区域





您可以编辑应用惯性的区域。

**要编辑应用惯性的区域：**

1. 从Property模块或Interaction模块的主菜单栏中，选择****Special**![](../graphics/images/arrow.gif)**Inertia**![](../graphics/images/arrow.gif)**Edit**![](../graphics/images/arrow.gif)***惯性名称*****以显示**Edit Inertia**对话框。
2. 在对话框顶部，单击![](../graphics/ico_selectBlue.png)。
3. 使用以下方法之一编辑区域：
   - 在视口中选择和取消选择对象。完成编辑区域后，单击鼠标按钮2。（更多信息，请参见[第6章，"在视口中选择对象"](pt01ch06.md)。）
   - 要从现有集列表中选择，请执行以下操作：
     1. 在提示区域右侧单击**Sets**。Abaqus/CAE显示包含可用集列表的**Region Selection**对话框。
     2. 选择感兴趣的集，然后单击**Continue**。
   **注意：**默认选择方法基于您最近使用的选择方法。要切换到另一种方法，请在提示区域右侧单击**Select in Viewport**或**Sets**。
4. 在对话框中，根据需要完成惯性定义的编辑，然后单击**OK**。视口中表示惯性的符号会出现在新编辑的区域上。
