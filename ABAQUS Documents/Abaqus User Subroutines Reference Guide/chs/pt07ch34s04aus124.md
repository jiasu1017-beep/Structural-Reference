### 概述

如["规定条件：概述，" 第34.1.1节"](pt07ch34s01abo31.md)所述，电磁载荷可以施加在["压电分析，" 第6.7.2节"](pt03ch06s07at21.md)；["耦合热-电分析，" 第6.7.3节"](pt03ch06s07at22.md)；["完全耦合热-电-结构分析，" 第6.7.4节"](pt03ch06s07at23.md)；["涡流分析，" 第6.7.5节"](pt03ch06s07at24.md)；和["静磁分析，" 第6.7.6节"](pt03ch06s07at25.md)中。

可用的电磁载荷类型取决于所执行的分析，如下所述。有关适用于所有载荷类型的一般信息，请参阅["施加载荷：概述，" 第34.4.1节"](pt07ch34s04aus120.md)。

### 定义随时间变化的电磁载荷

集中或分布电磁载荷的规定幅值可以随步中时间按照幅值定义变化，如["规定条件：概述，" 第34.1.1节"](pt07ch34s01abo31.md)中所述。如果不同载荷需要不同的变化，每个载荷可以引用自己的幅值定义。

在时间谐调涡流分析中，所有载荷都被假定为时间谐调的。

### 修改电磁载荷

集中或分布电磁载荷可以按照["施加载荷：概述，" 第34.4.1节"](pt07ch34s04aus120.md)中所述进行添加、修改或移除。

### 为压电分析规定电磁载荷

在压电分析中，可以在节点上规定集中电荷，可以在单元表面和表面上定义分布电荷面密度，也可以在单元上定义分布电荷体密度。

#### 指定集中电荷

要指定集中电荷，请指定节点或节点集以及电荷的幅值。

| **输入文件用法：** | ``` [*CECHARGE*](../key/key-link.md#usb-kws-hcecharge) *节点编号或节点集名称*,, *电荷幅值* ``` |
| --- | --- |

| **Abaqus/CAE用法：** | 载荷模块：**创建载荷**：为**类别**选择**电气/磁**，为**所选步的类型**选择**集中电荷**；**幅值**：*电荷幅值* |
| --- | --- |

#### 指定基于单元的分布电荷

您可以指定分布表面电荷（在单元表面上）或分布体积电荷（单位体积的电荷）。对于基于单元的表面电荷，您必须在电荷标签中标识规定电荷的单元表面。可用分布电荷类型取决于单元类型。[第六部分，"单元"](pt06.md)"列出了特定单元可用的分布电荷。

| **输入文件用法：** | ``` [*DECHARGE*](../key/key-link.md#usb-kws-hdecharge) *单元编号或单元集名称*, *电荷标签*, *电荷幅值* ``` |
| --- | --- |
| | 其中*电荷标签*为ES*n*或EBF |

| **Abaqus/CAE用法：** | 使用以下输入在单元表面上定义分布表面电荷： |
| --- | --- |
| | 载荷模块：**创建载荷**：为**类别**选择**电气/磁**，为**所选步的类型**选择**表面电荷**；**分布**：选择解析场，**幅值**：*电荷幅值* 使用以下输入定义体积电荷：载荷模块：**创建载荷**：为**类别**选择**电气/磁**，为**所选步的类型**选择**体积电荷** |

#### 指定基于表面的分布电荷

当您在表面上指定分布电荷时，基于单元的表面（见["基于单元的表面定义，" 第2.3.2节"](pt01ch02s03aus17.md)）包含单元和面信息。您必须指定表面名称、电荷标签和电荷幅值。

| **输入文件用法：** | ``` [*DSECHARGE*](../key/key-link.md#usb-kws-hdsecharge) *表面名称*, ES, *电荷幅值* ``` |
| --- | --- |

| **Abaqus/CAE用法：** | 载荷模块：**创建载荷**：为**类别**选择**电气/磁**，为**所选步的类型**选择**表面电荷**；**分布**：**均匀**，**幅值**：*电荷幅值* |
| --- | --- |

#### 在直接解稳态动力学分析中指定电荷

在直接解稳态动力学过程中，电荷以其实部和虚部分量给出。

| **输入文件用法：** | 使用以下选项在直接积分稳态动力学分析中定义电荷： |
| --- | --- |
| | ``` [*CECHARGE*](../key/key-link.md#usb-kws-hcecharge), REAL或IMAGINARY *（实部或虚部分量）* [*DECHARGE*](../key/key-link.md#usb-kws-hdecharge), REAL或IMAGINARY [*DSECHARGE*](../key/key-link.md#usb-kws-hdsecharge), REAL或IMAGINARY ``` |

| **Abaqus/CAE用法：** | 载荷模块：**创建载荷**：为**类别**选择**电气/磁**，为**所选步的类型**选择**集中电荷**、**表面电荷**或**体积电荷**；**幅值**：*实部分量* + *虚部分量* |
| --- | --- |

#### 基于模态和子空间过程中的载荷

由于"无质量"模态效应，电荷载荷应仅与特征值提取步中的残余模态结合使用。由于电位自由度没有任何相关质量，这些自由度在特征值提取过程中被基本消除（类似于Guyan约减或质量凝聚）。残余模态表示与电荷载荷对应的静态响应，这将充分代表特征空间中的电位自由度。

### 为涡流和/或静磁分析规定电磁载荷

在涡流分析中，可以在表面上定义分布表面电流密度矢量，也可以在单元上定义分布体积电流密度矢量。

#### 指定基于单元的分布电流密度矢量

当您定义分布体积电流密度矢量时，您必须指定单元或单元集、电流密度矢量标签、电流密度矢量幅值、电流密度的矢量分量，以及一个可选的方位名称，用于定义指定矢量分量的局部坐标系。默认情况下，电流密度的矢量分量是相对于全局方向定义的。

Abaqus对指定的电流密度矢量方向分量进行归一化，因此不会影响载荷的幅值。

| **输入文件用法：** | ``` [*DECURRENT*](../key/key-link.md#usb-kws-hdecurrent) *单元编号或单元集名称*, CJ, *电流密度矢量幅值*, *电流密度矢量方向分量*, *方位名称* ``` |
| --- | --- |

| **Abaqus/CAE用法：** | 载荷模块：**创建载荷**：为**类别**选择**电气/磁**，为**所选步的类型**选择**体积电流密度**；**分布**：**均匀** |
| --- | --- |

#### 指定基于表面的分布电流密度矢量

当您在表面上指定分布电流密度矢量时，基于单元的表面（见["基于单元的表面定义，" 第2.3.2节"](pt01ch02s03aus17.md)）包含单元和面信息。您必须指定表面名称、电流密度矢量标签、电流密度矢量幅值、电流密度的矢量分量，以及一个可选的方位名称，用于定义指定表面电流密度矢量的局部坐标系。默认情况下，电流密度的矢量分量是相对于全局方向定义的。

Abaqus对指定的电流密度矢量方向分量进行归一化，因此不会影响载荷的幅值。

| **输入文件用法：** | ``` [*DSECURRENT*](../key/key-link.md#usb-kws-hdsecurrent) *表面名称*, CK, *电流密度矢量幅值*, *电流密度矢量方向分量*, *方位名称* ``` |
| --- | --- |

| **Abaqus/CAE用法：** | 载荷模块：**创建载荷**：为**类别**选择**电气/磁**，为**所选步的类型**选择**表面电流密度**；**分布**：**均匀** |
| --- | --- |
