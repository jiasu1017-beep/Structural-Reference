# 11.27.1 圆化边缘







从主菜单栏选择**形状**![](../graphics/images/arrow.gif)**混合**![](../graphics/images/arrow.gif)**圆角/倒圆角**，对当前视口中零件的所选边缘进行圆化。您可以圆化凸边缘和凹边缘。带有圆化边缘的零件如下图所示：

![](../graphics/prt-round-select.png)

您定义倒圆角的半径，Abaqus/CAE将半径作为一个组应用于所有选定的边缘；因此，后续的特征操作（如编辑、删除和抑制）将应用于整个所选边缘组。因此，如果您选择多个边缘进行圆化，则不能仅修改其中一个圆化边缘。此外，结果边缘的形状可能取决于应用倒圆角的顺序，如下图所示。零件左侧的倒圆角是通过选择所有三个边缘并在单个操作中将圆角/倒圆角工具应用于所选边缘组来创建的。相比之下，零件右侧的倒圆角是通过分别选择每个边缘并按顺序将圆角/倒圆角工具应用于每个边缘来创建的。

![](../graphics/prt-round.png)

圆角/倒圆角工具仅在当前视口包含三维实体或壳零件时可用。此外，您不能对包含线框的边缘进行倒圆角。圆化边缘的半径定义了该特征，可以使用特征操作工具集进行修改。

**圆化边缘：**

1. 从主菜单栏，选择**形状**![](../graphics/images/arrow.gif)**混合**![](../graphics/images/arrow.gif)**圆角/倒圆角**。Abaqus/CAE提示您选择要圆化的边缘。Abaqus/CAE在提示区域显示引导您完成该过程的提示。**提示：**您也可以使用零件模块工具箱中位于混合工具旁边的![](../graphics/ico_partRound.png)工具来圆化所选边缘。有关零件模块工具箱中工具的图表，请参见"使用零件模块工具箱"，第11.17节。
2. 选择要圆化的边缘，然后单击鼠标按钮2提交您的选择。**[Shift]**+单击**添加更多边缘到您的选择，**[Ctrl]**+单击**已选择的边缘以取消选择。**提示：**如果您无法选择所需的边缘，可以使用**选择**工具栏更改选择行为。有关更多信息，请参见"使用选择选项"，第6.3节。提示区域中会出现一个默认半径。
3. 如有必要，在提示区域的文本字段中输入新的半径。单击鼠标按钮2提交半径。Abaqus/CAE使用圆化边缘后的零件重新绘制图形。
