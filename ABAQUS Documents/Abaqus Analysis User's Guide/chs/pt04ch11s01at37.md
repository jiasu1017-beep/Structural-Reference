# 11.1.1 几何缺陷

**产品：** Abaqus/Standard Abaqus/CAE

##### **参考**

- [*IMPERFECTION](../key/key-link.md#usb-kws-mimperfection)

### 概述

几何缺陷用于后屈曲分析。当结构在加载下发生屈曲时，缺陷会在结构中引入扰动，从而导致可预测的非对称响应。缺陷可以是预先定义的形状（来自屈曲模态分析），也可以是从先前分析结果中获取的变形构型。

### 定义几何缺陷

缺陷可以应用于模型中的单个部件实例或多个部件实例。

#### 从屈曲模态定义缺陷

缺陷形状通常来自屈曲模态分析。分析完成后，在**Step**模块中使用**Load**功能可以查看屈曲模态振型。

从屈曲模态导入缺陷时，可以按比例缩放模态振型。如果需要，可以同时应用多个屈曲模态作为缺陷。

#### 从先前分析结果定义缺陷

除了屈曲模态外，缺陷还可以从先前静态或准静态分析的结果中获取。这些结果可以是：
- 输出数据库（.odb）中的变形构型
- 结果文件中的节点位移

使用此选项时，应选择最能代表缺陷初始构型的结果帧。

### 指定缺陷比例因子

缺陷比例因子用于缩放缺陷振型的幅值。比例因子的选择取决于具体问题；通常使用1.0作为起点。

### 应用缺陷

缺陷在初始分析步（Initial step）中定义，并在第一个分析步开始时应用到模型中。缺陷的应用不会影响初始条件——它是在初始分析步开始时施加的扰动。

### 示例：屈曲后分析中的几何缺陷

1. 执行线性屈曲分析以获取屈曲模态
2. 从屈曲分析结果中导入所需模态作为缺陷
3. 在后屈曲分析中应用缺陷，指定合适的比例因子
4. 执行非线性静力分析

### 限制

- 几何缺陷仅在非线性静力分析（Static）中可用
- 缺陷振型必须来自相同几何构型的模型
- 如果使用多个缺陷模态，它们将线性叠加
