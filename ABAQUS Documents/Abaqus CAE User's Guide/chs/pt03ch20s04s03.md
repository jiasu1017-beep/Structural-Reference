# 20.4.3 Abaqus/CAE 如何定向您的草图







当 Sketcher 启动时，Abaqus/CAE 会定向零件或装配的视图，使您正在草绘的面或基准平面平行于屏幕；该平面称为草图平面。草图平面的方向取决于零件或装配的建模空间以及您正在创建的特征类型：

**二维或轴对称建模空间**

当您向二维或轴对称零件或装配添加平面特征时，Abaqus/CAE 会启动 Sketcher 并将零件的 *X* 和 *Y* 轴与 Sketcher 网格的轴对齐，无论您正在创建何种类型的特征。（Abaqus/CAE 从定义基础特征的草图的轴派生零件的 *X* 和 *Y* 轴。）

**三维建模空间**

当您向三维零件或装配添加特征时，必须使用以下技术来控制视图相对于 Sketcher 网格的方向：

1. 通过选择适当的几何来选择要草绘的平面；例如，零件的面或基准平面。
2. 选择一条边。默认情况下，所选边将显示为垂直且位于 Sketcher 网格的右侧。或者，您可以在选择边之前在 Sketcher 网格中选择边的不同方向；例如，水平且位于 Sketcher 网格的顶部。您可以选择零件或装配中任何不垂直于所选平面的边。您可以选择基准轴，也可以选择基准坐标系的一个轴；但不能选择基准平面的边。您可以选择弯曲边，但结果的方向取决于系统。

Abaqus/CAE 高亮显示所选边，进入 Sketcher，并旋转零件直到所选边与网格对齐为所需方向。Sketcher 网格上视图的方向也取决于您正在创建的特征类型：- 当您在零件模块中草绘切割特征时，Abaqus/CAE 定向视图，使结果特征向远离您并进入屏幕的方向切割。
- 当您草绘切割以外的特征时（例如，拉伸实体），Abaqus/CAE 定向视图，使结果特征突出屏幕向您。
- 当您添加扫掠特征时，必须草绘扫掠路径和扫掠轮廓。有关结果视图方向的详细信息，请参见["添加扫掠实体特征，"第 11.21.3 节](pt03ch11s21hlb03.md)；["添加扫掠壳特征，"第 11.22.3 节](pt03ch11s22hlb03.md)；和["创建扫掠切割，"第 11.24.4 节](pt03ch11s24hlb04.md)。

当您使用分区工具集中的草绘方法分区面时，Abaqus/CAE 会根据零件或装配的建模空间以及您选择分区的面的类型来定向视图。有关更多信息，请参见["使用草绘方法分区面，"第 70.6.1 节](pt06ch70s06hlb01.md)。

如果您不确定零件或装配相对于草图平面的方向，请使用视图操作工具检查草图平面和您正在草绘的对象。使用重置视图工具![](../graphics/ico_sketchResetView.png)返回到原始视图。
