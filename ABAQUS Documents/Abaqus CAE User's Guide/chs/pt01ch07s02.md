# 7.1 图形显示选项概述





当你启动会话时，Abaqus 会检测系统上安装的图形硬件并相应设置图形选项。如果你的图形硬件不受 Abaqus/CAE 支持，或者你想覆盖默认图形选项，你可以使用**图形选项**对话框调整显示性能。Abaqus/CAE 将设置应用到所有视口并保存会话期间的设置。要在每次启动 Abaqus/CAE 会话时使用自定义设置，请修改环境文件（`abaqus_v6.env`）。有关环境文件的附加信息，请参阅 [Abaqus 安装和许可指南](../sgb/sgb-link.md#sgb)。

**注意：**最近推出的图形适配器的推荐设置可从 [www.3ds.com/simulia](http://www.3ds.com/simulia) 的**支持**页面获取。

你也可以使用**图形选项**对话框执行以下操作：
- 在旋转、平移或缩放视图操作期间选择模型的外观。外观与渲染样式相关，可设置为**快速（线框）**或**原样**。
- 选择在旋转视图后 Abaqus/CAE 是否会自动将图像拟合到当前视口。自动将图像拟合到视口等同于单击**视图操作**工具栏中的自动拟合工具 ![](../graphics/ico_viewFit.png)。自动拟合会调整你对模型的视图，使模型填充视口并在视口中居中。方向保持不变，如视图 triad 所示。
- 选择 Abaqus/CAE 是否会优化半透明对象的显示以提高性能、准确性或介于两者之间的级别。
- 选择视口背景颜色。你选择的颜色将应用于当前 Abaqus/CAE 会话中的所有视口。

**要指定图形显示选项：**
- 从主菜单栏中，选择****View**![](../graphics/images/arrow.gif)**Graphics Options****。**图形选项**对话框出现，包含以下选项：
   - 使用显示列表、高亮方法和透明度模式选项调整性能。
   - 在视口中拖动对象时选择显示模式。
   - 启用或禁用旋转后自动将视图拟合到视口。
   - 选择视口的背景颜色。
