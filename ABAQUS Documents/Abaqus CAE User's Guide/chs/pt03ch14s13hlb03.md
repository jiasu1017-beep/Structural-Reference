# 14.13.3 配置诊断打印

选择**输出**![](../graphics/images/arrow.gif)**诊断打印**来配置Abaqus写入消息文件（对于Abaqus/Standard）或状态文件（对于Abaqus/Explicit）的诊断消息，并在分析期间报告给Job模块。如果您的模型分析失败或产生意外结果，您可以检查这些消息并查看分析的描述。（更多信息，请参见["诊断打印，"第14.5.3节"](pt03ch14s05s03.md)。）

Abaqus/Standard还将大多数诊断信息写入输出数据库以供可视化模块使用。但是，写入输出数据库的诊断信息不受诊断打印设置的影响。（更多信息，请参见[第41章，"查看诊断输出"](pt05ch41.md)。）

**要配置诊断打印：**

1. 从主菜单栏，选择**输出**![](../graphics/images/arrow.gif)**诊断打印**。**编辑诊断打印**对话框出现，其中包含当前模型中所有步骤的列表。
2. 在**频率**列中，为每个步骤指定您想要写入诊断打印数据的频率。（此选项不适用于Abaqus/Explicit分析。）
3. 点击其他列以在特定步骤期间请求特定的诊断信息。复选标记出现，指示该类型的诊断打印在该步骤期间已启用。再次点击列以关闭诊断打印。
4. 完成诊断打印的配置后，点击**确定**退出**编辑诊断打印**对话框。
