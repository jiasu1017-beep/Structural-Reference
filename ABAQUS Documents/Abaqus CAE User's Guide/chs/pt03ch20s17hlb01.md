# 20.22.1 管理独立草图


独立草图存储在您的模型中,独立于任何特定零件或装配。您可以在 Sketch 模块中工作时创建独立草图,也可以在绘制时将当前草图保存为独立草图。您可以在创建或编辑特征或分区时添加独立草图,并且添加的草图会覆盖当前草图,使它们的原点重合。

您可以使用 Sketch 模块来管理模型中定义的独立草图。要在 Sketch 模块中创建、检索、复制、重命名和删除独立草图,请使用以下之一:
- 主菜单栏上**Sketch** 菜单下列出的**创建**、**编辑**、**复制**、**重命名**和**删除**项。每个项目都包含一个子菜单,列出了当前模型中的所有草图。
- **草图管理器**对话框。**草图管理器**包含与**Sketch** 菜单下列出的相同功能,但提供了一个方便的浏览器,列出了当前模型中所有可用的草图。要显示**草图管理器**对话框,请从主菜单栏选择****Sketch**![](../graphics/images/arrow.gif)**Manager****。

**注意:****Sketch** 菜单和**草图管理器**仅在 Sketch 模块中可用。例如,当您在 Part 模块中绘制特征时,**Sketch** 菜单和**草图管理器**不可用。
