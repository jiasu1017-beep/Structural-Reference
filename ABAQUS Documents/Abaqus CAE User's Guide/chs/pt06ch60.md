# 60 CAD Connection 工具集





CAD Connection 工具集供 Abaqus/CAE 的可选附加关联接口使用，用于创建从 Abaqus/CAE 到另一个 CAD 系统的连接。您可以使用 CAD 系统修改模型或更改其位置，您可以使用已建立的连接在 Abaqus/CAE 中快速更新模型。您还可以在 Abaqus/CAE 中修改某些几何特征，并使用 CAD 连接更新原始 CAD 系统模型。本章涵盖以下主题：
