本节提供Abaqus/Standard中可用的管土相互作用单元的参考。

### 单元类型

#### 2D单元

| PSI24 | 二维4节点管土相互作用单元 |

| PSI26 | 二维6节点管土相互作用单元 |
| --- | --- |
|  |

##### 激活的自由度

1, 2

##### 额外求解变量

无。

#### 3D单元

| PSI34 | 三维4节点管土相互作用单元 |
| --- | --- |
|  |

| PSI36 | 三维6节点管土相互作用单元 |
| --- | --- |
|  |

##### 激活的自由度

1, 2, 3

##### 额外求解变量

无。

### 需要的节点坐标

2D: *X*, *Y*

3D: *X*, *Y*, *Z*
