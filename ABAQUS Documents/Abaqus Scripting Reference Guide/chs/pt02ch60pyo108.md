# 60.108 Viscous 对象





Viscous 对象为双层粘塑性材料模型指定粘性特性。

**访问**

```
materialApi.materials()[*name*].viscous()
```

### 60.108.1 Viscous(...)

此方法创建一个 Viscous 对象。

**路径**

```
materialApi.materials()[*name*].Viscous
```

**原型**

```
odb_Viscous&
Viscous(const odb_SequenceSequenceDouble& table,
        const odb_String& law,
        bool temperatureDependency,
        int dependencies);
```

**必需参数**

*table*

一个 odb_SequenceSequenceDouble，指定下述项目。

**可选参数**

*law*

一个 odb_String，指定蠕变定律。可能的值为 "STRAIN"、"TIME" 和 "USER"。默认值为 "STRAIN"。

*temperatureDependency*

一个布尔值，指定数据是否依赖于温度。默认值为 false。

*dependencies*

一个 Int，指定场变量依赖项的数量。默认值为 0。

**表数据**

如果 *law*=STRAIN 或 *law*=TIME，表数据指定以下内容：
- ![](../graphics/ker_eqn00010.gif)。
- ![](../graphics/ker_eqn00088.gif)。
- ![](../graphics/ker_eqn00089.gif)。
- ![](../graphics/ker_eqn00395.gif)。
- 温度（如果数据依赖于温度）。
- 第一个场变量的值（如果数据依赖于场变量）。
- 第二个场变量的值。
- 依此类推。

如果 *law*=USER，表数据指定以下内容：
- ![](../graphics/ker_eqn00395.gif)。
- 温度（如果数据依赖于温度）。
- 第一个场变量的值（如果数据依赖于场变量）。
- 第二个场变量的值。
- 依此类推。

**返回值**

一个 Viscous 对象。

**异常**

无。

### 60.108.2 成员

Viscous 对象的成员与 [Viscous](pt02ch60pyo108.md#ker-viscous-viscous-cpp) 方法的参数具有相同的名称和描述。

此外，Viscous 对象可以具有以下成员：

**原型**

```
odb_Potential potential() const;
```

*potential*

一个 [Potential](pt02ch60pyo83.md) 对象。
