# 17.14.4 参数化建模





Mesh模块的一个有用功能是能够在修改零件后重新生成划分和网格属性（如单元类型分配、种子和网格控制）。 （修改模型后，您必须始终重新创建网格本身。）

例如，[图 17-131](pt03ch17s14s04.md#mgn-parametric1) 所示的模型已被划分为四个区域，然后进行了种子布置以指定近似单元尺寸为3。

**图 17-131** 带有小孔的种子模型。

![](../graphics/mgn-parametric1.png)

您可以返回Part模块，修改模型中心孔，使其稍微变大。 当您返回Mesh模块时，划分和种子会被重新生成，如图 17-132](pt03ch17s14s04.md#mgn-parametric2) 所示。

**图 17-132** 修改零件后种子被重新生成。

![](../graphics/mgn-parametric2.png)

此外，**Mesh Controls**和**Element Type**对话框中的设置（如单元形状、单元类型和网格划分技术）也会被重新生成。 （您可以通过从主菜单栏选择 ****Mesh**![](../graphics/images/arrow.gif)**Controls**** 和 ****Mesh**![](../graphics/images/arrow.gif)**Element Type**** 来显示这两个对话框。）

**注意：**如果您对零件进行了大幅度修改，种子和划分可能无法重新生成。 在这些情况下，您必须在重新进入Mesh模块后创建新的种子和划分。
