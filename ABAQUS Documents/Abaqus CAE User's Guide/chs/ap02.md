# B 特殊单元类型





## 附录B：特殊单元类型

Abaqus使用特殊单元类型来指定不属于网格的模型特征，如解析刚体曲面。这些单元类型出现在输出数据库中。您可以在可视化模块中查看它们，同时查询模型或基于单元类型创建显示组。[表B-1](ap02.md#usv-app-odbelem)列出了Abaqus生成的单元类型。

**表B-1** 输出数据库中的特殊单元类型。
| ARSC | 解析刚体曲面（圆柱形） |
| --- | --- |
| ARSR | 解析刚体曲面（旋转） |
| ARSSE | 解析刚体曲面（拉伸） |
| ARSSS | 解析刚体曲面（分段） |
| LSURF1 | 二维曲面上的2节点线 |
| LSURF2 | 轴对称曲面上的2节点线 |
| LSURF3 | 三维曲面上的3节点线 |
| LSURF4 | 二维曲面上的3节点线 |
| LSURF5 | 轴对称曲面上的3节点线 |
| LSURF6 | 二次三角形曲面 |
| PSURF1 | 二维曲面中的节点（点） |
| PSURF2 | 轴对称曲面中的节点 |
| PSURF3 | 三维曲面中的节点 |
| QSURF1 | 三维曲面中的4节点面片 |
| QSURF2 | 三维曲面中的8节点面片 |
| QSURF3 | 三维曲面中的9节点面片 |
| RNODE2D | 参考节点（二维） |
| RNODE3D | 参考节点（三维） |
| TSURF1 | 三维曲面中的3节点面片 |
| TSURF1 | 三维曲面中的6节点面片 |
