# 79.2.2 重新排序叠加绘图中的层





层按照它们在**叠加绘图层管理器**中列出的顺序绘制在视口中。默认情况下，层直接叠加在一起，这有时会导致绘图看起来重叠的不良视觉效果。

您可以重新排列层绘制的顺序，并且可以沿屏幕Z方向（垂直于屏幕平面）对所有层应用小的偏移量以在显示中分离它们并消除任何不良的视觉效果。如果应用正偏移，层按照**叠加绘图层管理器**中显示的顺序绘制（管理器中最后一个层是视口中最上面的层）。如果应用负偏移，绘制顺序会反转。[图79-2](pt07ch79s02hlb02.md#offsetexample)显示了一个叠加绘图示例，其中应用了不同的偏移值。

**图79-2** 未变形和轮廓图的叠加：左侧正偏移，右侧负偏移。

![](../graphics/offsetexample.png)

**重新排序叠加绘图中层的步骤：**

1. 使用["生成叠加绘图，"第79.2.1节](pt07ch79s02hlb01.md)中描述的过程创建叠加绘图。
2. 使用以下方法之一重新排序层：
   - 点击**叠加绘图层管理器**中的**名称**或**对象**列选择要重新排序的层。点击**上移**将层在管理器中上移；点击**下移**将层在管理器中下移。
   - 拖动**层偏移**滑块到正值或负值，以沿正或负屏幕Z方向移动层。您可能需要尝试偏移值以获得所需的显示。Abaqus/CAE将您的设置应用到当前视口中的叠加绘图。
3. 当叠加绘图显示在视口中时，您可以点击**叠加绘图层管理器**中的**绘制单个**退出叠加绘图状态。叠加绘图从视口中消失，显示返回到之前的绘图状态；但是**叠加绘图层管理器**保持打开。点击**解散**关闭**叠加绘图层管理器**。**提示：**您也可以随时点击工具箱中的![](../graphics/ico_overlayState.png)工具在单个绘图和叠加绘图状态之间切换。
