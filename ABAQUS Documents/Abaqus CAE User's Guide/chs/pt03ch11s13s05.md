# 11.13.3 在拉伸中包含扭曲





您可以选择在创建拉伸时包含扭曲。扭曲可用于创建扭曲电缆、螺旋齿轮以及其他可以通过恒定横截面通过一系列平行平面形成的复杂形状。扭曲通过绕平行于拉伸方向的轴旋转草图轮廓来修改拉伸。扭曲中心是草图轮廓中的一个孤点；它是用于扭曲拉伸的轴穿过草图平面的点。Pitch定义轮廓被扭曲360的拉伸距离。您可以使用特征操作工具集修改拉伸轮廓、拉伸方向、扭曲中心和pitch。

您可以在创建拉伸实体、壳和切割特征期间添加扭曲。

如果您要创建复杂形状，其中草图轮廓被旋转而不是拉伸（例如螺纹或螺旋弹簧），可以在旋转实体、壳或切割特征中包含pitch。![](../graphics/images/black4rule.gif)有关相关信息，请单击以下任一项：- "添加拉伸实体特征"，第11.21.1节- "添加旋转实体特征"，第11.21.2节- "添加拉伸壳特征"，第11.22.1节- "添加旋转壳特征"，第11.22.2节- "创建拉伸切割"，第11.24.1节- "创建旋转切割"，第11.24.3节- "使用六面体单元网格划分复杂实体"，第17.14.5节
