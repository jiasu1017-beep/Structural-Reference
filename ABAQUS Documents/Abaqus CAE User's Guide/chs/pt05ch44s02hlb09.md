# 44.5.9 控制Abaqus/CAE如何计算云图限制





默认情况下，Abaqus/CAE自动计算云图中显示的最小值和最大值。如果您已为最小值和最大值[设置了限制](pt05ch44s02hlb07.md)，则自动计算不适用。

对于节点场输出变量的云图，限制是最小和最大节点值。对于[拼布型云图](pt05ch44s02hlb01.md)，限制是最小和最大单元面值。

对于基于单元场输出变量的[线型、带型或等值面型云图](pt05ch44s02hlb01.md)，限制基于从单个单元外推并平均在节点上的值。由于结果平均选项的变化，范围可能会发生变化。要了解如何控制平均选项，请参见["控制结果平均，" 第42.6.6节](pt05ch42s04hlb02.md)。

对于云图动画，您可以控制动画每一帧使用的云图限制。您可以选择始终使用第一帧和最后一帧的自动计算限制、始终使用当前帧的自动计算限制、重新计算每一帧的限制，或使用所有帧的计算限制。

**要控制Abaqus如何计算云图限制：**

1. 找到**Auto-Computed Limits**选项。从主菜单栏中选择****Options**![](../graphics/images/arrow.gif)**Contour****，或单击工具箱中的![](../graphics/ico_plotContourOptions.png)；然后在出现的对话框中单击**Limits**选项卡。**Auto-Computed Limits**选项位于页面下半部分。
2. 为动画云图限制选择以下选项之一：- 选择**Use first and last frame limits**以对动画的每一帧使用第一帧和最后一帧的自动计算限制。- 选择**Use current frame limits**以对动画的每一帧使用当前帧的自动计算限制。- 选择**Recompute limits for each frame**以根据该帧的结果值重新计算每一帧的云图限制。- 选择**Use limits from all frames**以从动画的所有帧重新计算自动计算的云图限制。
3. 单击**Apply**实施您的更改。当前视口中为云图生成的动画每一帧的云图限制将根据您选择的选项确定。如果激活，[云图图例](pt05ch56s01.md)会更改以显示Abaqus计算的限制。默认情况下，您的更改在会话期间保存，并将影响所有后续云图。如果您想在后续会话中保留更改，请将它们保存到文件。有关更多信息，请参见["保存自定义设置以供后续会话使用，" 第55.1.1节](pt05ch55s01s01.md)。
