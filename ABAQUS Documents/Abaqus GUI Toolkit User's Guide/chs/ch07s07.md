# 7.7 阻尼的影响





在此模拟中，我们在所有模态中使用了 5% 的临界阻尼。这个值是根据经验选择的，基于桁架和交叉支撑之间的螺栓连接可能因局部摩擦效应而吸收大量能量的事实。在这种情况下（如没有准确数据可用），重要的是研究您所做的选择的影响。

[图 7-11](ch07s07.md#gss-results-v) 比较了使用 1%、5% 和 10% 临界阻尼时顶部连接之一（点 C）处反作用力的历史。正如所预期的，较低阻尼水平的振荡比较低阻尼水平的振荡衰减得更快，而较低阻尼模型中的峰值力较高。即使阻尼比低至 1%，峰值拉出力仍为 85 kN，仍然小于连接的强度（100 kN）。因此，货运吊车在这种落载荷下应保持其完整性。
