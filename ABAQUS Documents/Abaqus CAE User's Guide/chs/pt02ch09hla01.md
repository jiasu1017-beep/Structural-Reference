# 9.6 使用文件菜单





使用主菜单栏上**文件（File）**下的项目可以执行以下操作：
- 选择****文件（File）**![](../graphics/images/arrow.gif)**新建模型数据库（New Model Database）**![](../graphics/images/arrow.gif)**使用Standard/Explicit模型（With Standard/Explicit Model）****来为Abaqus/Standard或Abaqus/Explicit分析创建新的模型数据库。您也可以单击**文件**工具栏中的![](../graphics/ico_fileNew.png)。有关更多信息，请参阅["创建新模型数据库，"第9.7.1节"](pt02ch09s06hlb01.md)。
- 选择****文件（File）**![](../graphics/images/arrow.gif)**新建模型数据库（New Model Database）**![](../graphics/images/arrow.gif)**使用CFD模型（With CFD Model）****来为Abaqus/CFD分析创建新的模型数据库。有关更多信息，请参阅["创建新模型数据库，"第9.7.1节"](pt02ch09s06hlb01.md)。
- 选择****文件（File）**![](../graphics/images/arrow.gif)**新建模型数据库（New Model Database）**![](../graphics/images/arrow.gif)**使用电磁模型（With Electromagnetic Model）****来为电磁分析创建新的模型数据库。有关更多信息，请参阅["创建新模型数据库，"第9.7.1节"](pt02ch09s06hlb01.md)。
- 选择****文件（File）**![](../graphics/images/arrow.gif)**打开（Open）****来打开现有的模型数据库或输出数据库。您也可以单击**文件**工具栏中的![](../graphics/ico_fileOpen.png)。有关更多信息，请参阅["打开模型数据库或输出数据库，"第9.7.2节"](pt02ch09s06hlb02.md)。
- 选择****文件（File）**![](../graphics/images/arrow.gif)**网络ODB连接器（Network ODB Connector）****来创建与远程主机的连接，您可以使用该连接读取远程输出数据库。有关更多信息，请参阅["创建网络ODB连接器，"第9.7.4节"](pt02ch09s06hlb04.md)。
- 选择****文件（File）**![](../graphics/images/arrow.gif)**关闭ODB（Close ODB）****来关闭输出数据库。有关更多信息，请参阅["关闭当前输出数据库，"第9.7.7节"](pt02ch09s06hlb07.md)。
- 选择****文件（File）**![](../graphics/images/arrow.gif)**设置工作目录（Set Work Directory）****来更改工作目录。有关更多信息，请参阅["设置工作目录，"第9.7.8节"](pt02ch09s06hlb08.md)。
- 选择****文件（File）**![](../graphics/images/arrow.gif)**保存（Save）****来保存当前模型数据库。您也可以单击**文件**工具栏中的![](../graphics/ico_fileSave.png)。有关更多信息，请参阅["保存当前模型数据库，"第9.7.9节"](pt02ch09s06hlb09.md)。
- 选择****文件（File）**![](../graphics/images/arrow.gif)**另存为（Save As）****将当前模型数据库保存为不同名称的新文件。有关更多信息，请参阅["将当前模型数据库保存为不同名称的新文件，"第9.7.11节"](pt02ch09s06hlb11.md)。
- 选择****文件（File）**![](../graphics/images/arrow.gif)**压缩MDB（Compress MDB）****来压缩当前模型数据库。有关更多信息，请参阅["压缩当前模型数据库的文件大小，"第9.7.12节"](pt02ch09s06hlb12.md)。
- 选择****文件（File）**![](../graphics/images/arrow.gif)**保存显示选项（Save Display Options）****来保存您自定义的零件、装配和可视化模块显示设置。有关更多信息，请参阅["理解Abaqus/CAE GUI设置，"第3.6节"](pt01ch03s06.md)和["保存您的显示选项设置，"第76.16节"](pt07ch76hla15.md)。
- 选择****文件（File）**![](../graphics/images/arrow.gif)**保存会话对象（Save Session Objects）****来将会话特定的对象定义（如视图切割、显示组或路径）保存到文件、模型数据库或输出数据库中。有关更多信息，请参阅["管理会话对象和会话选项，"第9.9节"](pt02ch09s08.md)。
- 选择****文件（File）**![](../graphics/images/arrow.gif)**加载会话对象（Load Session Objects）****来将之前保存的会话特定对象定义加载到当前会话中。有关更多信息，请参阅["管理会话对象和会话选项，"第9.9节"](pt02ch09s08.md)。
- 选择****文件（File）**![](../graphics/images/arrow.gif)**导入（Import）**![](../graphics/images/arrow.gif)**草图（Sketch）****来导入平面草图。有关更多信息，请参阅["导入草图，"第10.7.1节"](pt02ch10s07hlb01.md)。
- 选择****文件（File）**![](../graphics/images/arrow.gif)**导入（Import）**![](../graphics/images/arrow.gif)**零件（Part）****来导入零件。有关更多信息，请参阅["导入零件，"第10.7.2节"](pt02ch10s07hlb02.md)。
- 选择****文件（File）**![](../graphics/images/arrow.gif)**导入（Import）**![](../graphics/images/arrow.gif)**模型（Model）****来导入模型。有关更多信息，请参阅["导入模型，"第10.8节"](pt02ch10hla01.md)。
- 选择****文件（File）**![](../graphics/images/arrow.gif)**导出（Export）**![](../graphics/images/arrow.gif)**草图（Sketch）****来导出当前草图。有关更多信息，请参阅["将草图导出为ACIS、IGES或STEP格式文件，"第10.9.1节"](pt02ch10s08hlb01.md)。
- 选择****文件（File）**![](../graphics/images/arrow.gif)**导出（Export）**![](../graphics/images/arrow.gif)**零件（Part）****来导出当前零件。有关更多信息，请参阅["将零件导出为ACIS、IGES、STEP或VDA格式文件，"第10.9.2节"](pt02ch10s08hlb02.md)。
- 选择****文件（File）**![](../graphics/images/arrow.gif)**导出（Export）**![](../graphics/images/arrow.gif)**装配（Assembly）****来导出装配中的零件实例。有关更多信息，请参阅["将装配导出为ACIS格式文件，"第10.9.3节"](pt02ch10s08hlb03.md)。
- 选择****文件（File）**![](../graphics/images/arrow.gif)**导出（Export）**![](../graphics/images/arrow.gif)**VRML****来将当前视口导出为VRML格式文件。有关更多信息，请参阅["将视口数据导出为VRML格式文件，"第10.9.4节"](pt02ch10s08hlb04.md)。
- 选择****文件（File）**![](../graphics/images/arrow.gif)**导出（Export）**![](../graphics/images/arrow.gif)**3DXML****来将当前视口导出为3D XML格式文件。有关更多信息，请参阅["将视口数据导出为3D XML格式文件，"第10.9.5节"](pt02ch10s08hlb05.md)。
- 选择****文件（File）**![](../graphics/images/arrow.gif)**导出（Export）**![](../graphics/images/arrow.gif)**OBJ****来将当前视口导出为OBJ格式文件。有关更多信息，请参阅["将视口数据导出为OBJ格式文件，"第10.9.6节"](pt02ch10s08hlb06.md)。
- 选择****文件（File）**![](../graphics/images/arrow.gif)**运行脚本（Run Script）****来执行包含Abaqus Scripting Interface命令的文件。有关更多信息，请参阅["重放Abaqus/CAE会话，"第9.5.1节"](pt02ch09s05s01.md)和["创建和运行您自己的脚本，"第9.5.4节"](pt02ch09s05s04.md)。
- 选择****文件（File）**![](../graphics/images/arrow.gif)**宏管理器（Macro Manager）****来将您的操作存储为Abaqus Scripting Interface命令序列到宏文件中。您也可以运行宏并重命名现有宏。有关更多信息，请参阅["创建和运行宏，"第9.5.5节"](pt02ch09s05s05.md)。
- 选择****文件（File）**![](../graphics/images/arrow.gif)**打印（Print）****来打印全部或选定的视口和注释。您也可以单击**文件**工具栏中的![](../graphics/ico_filePrint.png)。有关更多信息，请参阅[第8章，"打印视口"](pt01ch08.md)。
- 选择****文件（File）**![](../graphics/images/arrow.gif)**Abaqus PDE****来打开Abaqus Python开发环境。Abaqus PDE是一个独立的应用程序，用于创建、编辑、测试和调试脚本。有关更多信息，请参阅["Abaqus Python开发环境概述，"Abaqus Scripting User's Guide第7.1节"](../cmd/cmd-link.md#cmd-pde-over)。
- 选择****文件（File）**![](../graphics/images/arrow.gif)**退出（Exit）****来退出Abaqus/CAE会话。有关更多信息，请参阅["退出Abaqus/CAE会话，"第2.1.2节"](pt01ch02s01s02.md)。
