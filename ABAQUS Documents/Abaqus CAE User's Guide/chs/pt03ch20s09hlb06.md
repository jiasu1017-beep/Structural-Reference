# 20.9.7 限制共面实体的投影







创建或编辑基于草图的特征时，您可以设置 Abaqus/CAE 自动从背景投影的共面实体数量的限制。默认情况下，Abaqus/CAE 自动投影 300 个共面实体。您为投影共面实体选择的限制适用于 Abaqus/CAE 会话中的所有草图。

**限制投影的共面实体数量：**

1. 从 Sketcher 工具箱底部，选择自定义工具![](../graphics/ico_sketchOptions-nls.png)。出现**Sketcher 选项**对话框。
2. 从**常规**选项卡页面，在**最大投影共面实体数**文本字段中输入一个值。如果超过此值，则会发出警告消息，并且不会自动投影任何实体。
3. 选择完所需的自定义选项后，点击**确定**应用您的更改并关闭**Sketcher 选项**对话框。
