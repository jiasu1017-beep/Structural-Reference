# 20.19.3 偏移 Sketcher 对象的边缘


使用 Sketcher 工具箱中的偏移工具来偏移边缘。您选择边缘,输入偏移距离,然后选择偏移方向。Abaqus/CAE 会沿边缘上每个点垂直应用偏移距离来创建偏移。

**注意:**包含与偏移距离大小相似的尺寸的特征的边缘将在偏移副本中丢失其细节。

**偏移 Sketcher 边缘的步骤:**

1. 从 Sketcher 工具箱中,选择偏移工具 ![](../graphics/ico_sketchOffset.png)。有关 Sketcher 工具箱中工具的图示,请参见["Sketcher 工具," 第 20.4.1 节](pt03ch20s04s01.md)。Abaqus/CAE 会在提示区域显示提示,引导您完成操作。
2. 选择要偏移的单个连接的边缘组。只能偏移草图几何;[构造几何](pt03ch20s05s02.md)、[参考几何](pt03ch20s05s01.md)和尺寸不能被偏移。**提示:**要选择端到端连接的边缘组,请从提示区域选择**按链**(更多信息,请参见["在 Sketcher 中使用链方法选择边缘," 第 20.4.6 节](pt03ch20s04s06.md))。如有必要,您可以使用 Abaqus/CAE 中的其他选择方法来增强您的选择。更多信息,请参见[第 6 章,"在视口中选择对象"](pt01ch06.md)。
3. 单击鼠标按钮 2 表示已完成选择对象。
4. 指定偏移距离或接受提示区域中显示的默认值。Abaqus/CAE 会在视口中显示偏移边缘的预览。
5. 使用提示区域中出现的按钮接受或更改偏移方向。
6. 要偏移更多边缘,请从第 2 步开始重复上述步骤。
7. 完成偏移边缘后,执行以下操作之一: - 在 Abaqus/CAE 窗口任意位置单击鼠标按钮 2。 - 选择 Sketcher 工具箱中的任何工具。 - 单击提示区域中的取消按钮 ![](../graphics/afxI_cancel.png)。
