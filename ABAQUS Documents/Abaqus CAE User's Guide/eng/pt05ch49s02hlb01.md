# 49.2.1 Producing a time history animation







Animating by time history produces a sequence of plots that vary over time according to actual analysis results. You can choose to animate a deformed shape, contour, symbol, or material orientation plot.

**To produce a time history animation:**

1. Use the **File** menu to open the [output database](pt02ch09s06hlb02.md) containing your analysis results.
2. To plot variables other than those Abaqus selects by default, use the **Result** menu. - For animation of a deformed or material orientation plot, select the [deformed field output variable](pt05ch42s03hlb04.md) for which Abaqus obtains values over time. - For animation of a contour or symbol plot, select the [primary field output variable](pt05ch42s03hlb03.md) or [symbol field output variable](pt05ch42s03hlb05.md) to display. If you want to display your animating contour or symbol plot on the deformed shape, select the [deformed field variable](pt05ch42s03hlb04.md). For more information on field output variable selection, see ["Overview of field output variable selection," Section 42.5.1](pt05ch42s03hlb01.md).
3. To customize the steps and frames that Abaqus/CAE includes in the animation, locate the **Active Steps/Frames** dialog box. From the main menu bar, select ****Result**![](../graphics/images/arrow.gif)**Active Steps/Frames****. For more information about activating steps and frames, see ["Customizing the display of steps and frames in the results," Section 42.4](pt05ch42s02.md).
4. Use the **Options** and **View** menus to select the [customization options](pt05ch40s03s03.md) that you want. These may include deformed, contour, symbol, orientation, and view options.
5. To initiate the animation, select ****Animate**![](../graphics/images/arrow.gif)**Time History**** from the main menu bar. **Tip:**You can also produce a time history animation using the ![](../graphics/ico_animTimeHistory.png) tool in the toolbox. Animation begins in all animated viewports, and the animation controls appear in the right side of the context bar. Abaqus is now in the time history animation plot state. In this state Abaqus automatically refreshes your plot each time you click **Apply** or **OK** in the **Results**, **Options**, or **View** menu dialog boxes. All animated viewports remain in the time history animation state until you produce a plot in some other state, such as the undeformed or deformed state. To learn how to control the animation, see ["Controlling animation playback," Section 49.4](pt05ch49s04.md).
