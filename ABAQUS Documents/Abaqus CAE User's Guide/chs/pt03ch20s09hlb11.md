# 20.9.11 在 Sketcher 背景中管理图像







您可以在 Sketcher 背景中显示图像，以帮助您在草图中绘制和对齐对象。此图像可以位于 Sketcher 背景中的任何位置，可以水平或垂直拉伸或压缩。当您调整视口大小时，Abaqus/CAE 会按比例调整背景图像大小。

Sketcher 背景图像仅出现在草图模块中。您还可以在每个 Abaqus/CAE 模块（包括草图模块）的视口背景中显示不同的图像；有关更多信息，请参见["在视口中使用背景图像和电影，"第 4.6 节](pt01ch04s06.md)。当 Sketcher 和模块范围的背景图像都已切换打开时，您进入草图模块时，Abaqus/CAE 会在顶部显示 Sketcher 背景图像。

在您可以在草图模块或任何其他模块的背景中显示图像之前，必须将图像添加到 Abaqus/CAE 会话中。要从**Sketcher 选项**对话框添加图像，请点击**创建**；然后输入图像名称并提供其位置。会话中的图像在所有模块中可用，仅在会话期间保留；它们不会保存到模型数据库或输出数据库。

Abaqus/CAE 支持以下格式的背景图像：Bitmap（`.bmp`）、PNG（`.png`）、TIFF（`.tif`）、XPM（`.xpm`）、PCX（`.pcx`）、ICO（`.ico`）、TGA（`.tga`）和 RGB（`.rgb`）。

**在 Sketcher 背景中显示和自定义图像：**

1. 从 Sketcher 工具箱底部，选择 Sketcher 选项工具![](../graphics/ico_sketchOptions-nls.png)。出现**Sketcher 选项**对话框。
2. 点击**图像**选项卡。
3. 切换**显示图像**以显示或隐藏 Sketcher 背景图像。
4. 选择要显示的图像文件：- 要显示已在会话中定义的图像，请展开**图像名称**列表并选择图像名称。- 要添加新图像，请点击**创建**；然后在出现的对话框中输入名称并指定文件位置。您可以直接在**文件名**字段中输入文件位置，或点击![](../graphics/ico_fileOpenBlue.png)在**选择图像文件**对话框中导航到它。
5. 使用以下技术之一拉伸或压缩背景图像：**沿任一轴缩放图像**您可以通过指定 **X 比例**和 **Y 比例**值来拉伸或压缩 Sketcher 背景图像。当这些比例值等于 1（默认值）时，图像保留其原始比例。您可以增加比例值以沿选定轴拉伸 Sketcher 背景图像，或者可以减小值以沿选定轴压缩图像。**校准图像大小**您可以通过校准图像上两点之间的距离来缩放 Sketcher 背景图像的大小。点击**校准**，点击视口中的两点；然后在提示区域输入两点之间所需的距离。校准图像后，Abaqus/CAE 会在**X 比例**和 **Y 比例**字段中反映更改。
6. 使用以下技术之一重新定位 Sketcher 背景图像：- 在**原点 X**和**原点 Y**字段中输入您要将 Sketcher 背景图像左下角移动到的草图坐标。- 点击**拾取**，然后在 Sketcher 背景图像上点击一个点。Abaqus/CAE 将图像的此点移动到网格原点，并用新原点位置填充**原点**字段。**注意：**图像的左下角用于固定背景图像的位置。如果在定位背景图像后更改缩放比例，图像中心将会移动。
7. 将**透明度**滑块拖动到您想要显示 Sketcher 背景图像的透明度级别。背景图像默认是不透明的，您可以从 0.00（透明）到 1.00（不透明）之间选择值。
8. 点击**确定**应用您的更改并关闭**Sketcher 选项**对话框。
