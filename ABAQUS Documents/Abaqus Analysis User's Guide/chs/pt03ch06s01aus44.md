# 6.1.3 常规与线性扰动分析步骤





**产品：**Abaqus/Standard  Abaqus/Explicit  Abaqus/CAE  



##### **参考**

- ["定义分析，" 6.1.2节](pt03ch06s01abo05.md)
- ["线性与非线性分析过程，" Abaqus/CAE User's Guide第14.3.2节](../usi/usi-link.md#usi-sim-conc-unprocs)

### 概述

在分析过程中，响应可以是线性或非线性的，这样的分析步骤称为*常规*分析步骤（general analysis step）。在分析过程中响应只能是线性的，这样的分析步骤称为*线性扰动*分析步骤（linear perturbation analysis step）。常规分析步骤可以包含在Abaqus/Standard或Abaqus/Explicit分析中；线性扰动分析步骤仅在Abaqus/Standard中可用。

在Abaqus/Standard中，常规分析与线性扰动分析有着明确的区分。两种情况下加载条件的定义不同，时间度量也不同，结果的解释也应该不同。这些区别将在本节中定义。

Abaqus/Standard将线性扰动分析视为关于预加载、预变形状态的线性扰动。Abaqus/Foundation是Abaqus/Standard的子集，完全限于线性扰动分析，但不允许预加载或预变形状态。

### 常规分析步骤

常规分析步骤是指可以包含模型中任何非线性效应的步骤。每个常规步骤的起始条件是上一个常规步骤的结束状态，模型的状态在整个常规分析步骤的历史中随着加载历史而演变。如果分析的第一个步骤是常规步骤，则可以直接指定步骤的初始条件（["Abaqus/Standard和Abaqus/Explicit中的初始条件，" 34.2.1节](pt07ch34s02aus116.md)）。

Abaqus始终认为*总时间*在常规分析过程中是递增的。每个步骤也有其自己的*步骤时间*，在每个步骤中从零开始。如果步骤的分析过程具有物理时间尺度（如动态分析），则步骤时间必须与该物理时间对应。否则，步骤时间可以是任意方便的时间尺度，例如步骤的0.0到1.0。所有常规分析步骤的步骤时间累积到总时间。因此，如果在多步骤分析中使用了依赖于总时间的选项（如蠕变，仅在Abaqus/Standard中可用），则任何没有物理时间尺度的步骤的步骤时间应该相对于具有物理时间尺度的步骤可以忽略不计。

#### 非线性来源

非线性应力分析问题最多可能包含三个非线性来源：材料非线性、几何非线性和边界非线性。

##### 材料非线性

Abaqus提供了多种非线性材料行为模型（参见["组合材料行为，" 21.1.3节](pt05ch21s01aus110.md)）。许多材料是历史相关的：材料在任何时刻的响应取决于此前发生在它身上的事情。因此，必须通过追踪实际加载序列来获得解决方案。常规分析过程的设计考虑到了这一点。

##### 几何非线性

在Abaqus中可以将问题定义为"小位移"分析，这意味着在单元计算中忽略几何非线性——运动关系是线性化的。默认情况下，即使对小位移单元 formulations 使用了分析，接触约束中也会考虑大位移和转动；即使用大滑动接触跟踪算法（参见["Abaqus/Standard中的接触 formulations，" 38.1.1节](pt09ch38s01aus177.md)，和["Abaqus/Explicit中接触对的接触 formulations，" 38.2.2节](pt09ch38s02aus181.md)）。小位移分析中的单元在参考（原始）配置中 formulated，使用原始节点坐标。这种近似的误差与应变和转动相对于1的量级相同。该近似方法还消除了捕获分叉屈曲的任何可能性，而这有时是结构响应的关键方面（参见["不稳定坍塌与后屈曲分析，" 6.2.4节](pt03ch06s02at03.md)）。在解释此类分析的结果时，必须考虑这些问题。

Abaqus中"小位移"分析的替代方案是包含大位移效应。在这种情况下，大多数单元在当前配置中使用当前节点位置 formulated。因此，随着变形的增加，单元会从其原始形状扭曲。如果变形足够大，单元可能会变得扭曲得不再适用；例如，积分点处的单元体积可能变为负值。在这种情况下，Abaqus将发出警告消息指示问题。此外，Abaqus/Standard将在进一步尝试继续求解之前削减时间增量。Abaqus/Explicit还提供单元失效模型，以允许在单元达到高应变时从模型中移除；详情参见["动态失效模型，" 23.2.8节](pt05ch23s02abm24.md)。

对于分析的每个步骤，您可以指定应使用小位移或大位移 formulation（即是否忽略或包含几何非线性）。默认情况下，Abaqus/Standard使用小位移 formulation，Abaqus/Explicit使用大位移 formulation。导入分析中 formulation 的默认值与导入时的时间值相同。如果在任何分析步骤中使用了了大位移 formulation，则将在分析的所有后续步骤中使用它；无法关闭它。

Abaqus中几乎所有单元都使用完全非线性 formulation。例外的是Abaqus/Standard中的 cubic beam 单元以及小应变 shell 单元（C3D20R、S4R等壳单元之外的那些壳单元），其中忽略截面厚度的变化，因此这些单元仅适用于大转动和小应变。除这些单元外，应变和转动可以任意大。

计算的应力是"真实"（Cauchy）应力。对于梁、管和壳单元，应力分量在随材料转动的局部方向上给出。对于所有其他单元，应力分量在全局方向上给出，除非在一点处使用了局部取向（["取向，" 2.2.5节](pt01ch02s02aus15.md)）。对于小位移分析，使用小应变度量，与应变输出变量E一起输出；使用输出变量LE和NE指定的应变与E相同。

##### 边界非线性

接触问题是应力分析中常见的非线性来源——参见["接触相互作用分析：概述，" 36.1.1节](pt09ch36s01abo33.md)。边界非线性的其他来源包括非线性弹性弹簧、薄膜、辐射、多点约束等。

#### 加载

在常规分析步骤中，加载必须定义为总值。在常规多步骤分析中应用加载的规则在["应用加载：概述，" 34.4.1节](pt07ch34s04aus120.md)中定义。

#### 增量控制

Abaqus中的常规分析过程提供两种增量控制方法。自动控制是一种选择：您定义步骤，并在某些过程中指定某些容差或误差度量。然后Abaqus在展开步骤响应时自动选择增量大小。直接用户控制增量大小是另一种方法，您可以指定增量控制方案。在使用Abaqus/Standard的重复分析中，直接方法有时很有用，因为您对问题的收敛行为有很好的"感觉"。选择自动或直接增量控制的方法在各个过程章节中讨论。

在Abaqus/Standard的非线性问题中，挑战始终是以最少的计算时间获得收敛解。在这些情况下，自动时间增量控制通常更有效，因为Abaqus/Standard可以响应您无法预见的非线性响应。当响应或载荷在步骤中广泛变化时，自动控制特别有价值，这种情况在蠕变、热传递和固结等扩散型问题中很常见。最终，自动控制使非线性问题能够在Abaqus/Standard中以较少的问题经验自信地运行。

由于显式动态分析产品特有的很小时间增量，Abaqus/Explicit中的强非线性通常不会带来困难。

### 线性扰动分析步骤

线性扰动分析步骤仅在Abaqus/Standard中可用（Abaqus/Foundation本质上是Abaqus/Standard中的线性扰动功能）。线性分析步骤中的响应是关于*基础状态*的线性扰动响应。基础状态是线性扰动步骤之前最后一个常规分析步骤结束时模型的当前状态。如果分析的第一个步骤是扰动步骤，则基础状态由初始条件确定（["Abaqus/Standard和Abaqus/Explicit中的初始条件，" 34.2.1节](pt07ch34s02aus116.md)）。在Abaqus/Foundation中，基础状态始终由模型的初始状态确定。

线性扰动分析可以在完全非线性分析过程中随时进行，方法是在常规响应步骤之间包含线性扰动步骤。线性扰动响应对常规分析的继续没有影响。线性扰动步骤的步骤时间（被任意取为一个非常小的数字）永远不会累积到总时间中。这种方法的一个简单例子是确定弦在增加张力下的固有频率（参见["张力下电缆的振动，" Abaqus Benchmarks Guide第1.4.3节](../bmk/bmk-link.md#bmk-anl-vibrationcable)）。弦的张力在几个几何非线性分析步骤中增加。在每个步骤之后，可以在线性扰动分析步骤中提取频率。

如果在线性扰动研究所基于的常规分析中包含了几何非线性，则线性扰动分析中会包含应力刚化或软化效应以及载荷刚度效应（来自压力和其他跟随力）。

载荷刚度贡献也由离心力和科里奥利载荷产生。在直接稳态动态分析中，科里奥利载荷产生一个虚数反对称矩阵。目前此贡献仅在实体和桁架单元中被考虑，并通过在步骤中使用非对称矩阵存储和求解方案来激活。

#### 加载

在线性扰动分析步骤期间，加载幅度（包括指定边界条件的幅度）被定义为加载扰动幅值。同样，任何输出变量值都仅作为扰动值输出——不包括基础状态中的变量值。

##### 多载荷工况分析

对于静态、直接求解稳态动态和基于SIM的稳态动态（包括子空间投影）线性扰动步骤，可以同时分析多个载荷工况。参见["多载荷工况分析，" 6.1.4节](pt03ch06s01aus45.md)，了解此功能的描述。

#### 限制

线性扰动分析受以下限制：

- 由于线性扰动分析没有时间周期，幅值参考（["幅值曲线，" 34.1.2节](pt07ch34s01aus115.md)）仅能用于将有意义地指定载荷或边界条件为频率的函数（在稳态动力学分析中）或定义基础运动（在基于模态的动力学过程中）。如果载荷或边界条件被指定为时间的函数，则将使用时间=0对应的幅值。
- 不能中断常规隐式动态分析（["使用直接积分的隐式动态分析，" 6.3.2节](pt03ch06s03at07.md)）来执行扰动分析：在执行扰动分析之前，Abaqus/Standard要求将结构带入静态平衡。
- 在线性扰动分析步骤期间，模型的响应由基础状态的线性弹性（或粘弹性）刚度定义。塑性和其他非弹性效应被忽略。对于超弹性（["橡胶类材料的超弹性行为，" 22.5.1节](pt05ch22s05abm07.md)）或次弹性（["次弹性行为，" 22.4.1节](pt05ch22s04abm06.md)），使用基础状态中的切线弹性模量。如果已经发生开裂（例如在混凝土模型中，["混凝土弥散开裂，" 23.6.1节](pt05ch23s06abm37.md)），则使用受损弹性（割线）模量。
- 接触条件在线性扰动分析期间不能改变。每个接触约束的打开/关闭状态保持与基础状态中相同。所有接触点（即具有"关闭"状态的点）如果有摩擦则假定为粘着，但由参考框架运动或传输速度施加速度差的接触节点除外。对于那些节点，假定为滑动条件，与摩擦系数无关。
- 对于依赖于温度和场变量的材料，温度和场变量扰动的影响被忽略。然而，温度扰动会产生热应变的扰动。
