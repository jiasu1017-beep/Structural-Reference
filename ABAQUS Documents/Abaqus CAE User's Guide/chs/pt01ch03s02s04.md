# 3.2.4 禁用警告对话框





某些对话框可以被禁用，以便在当前Abaqus/CAE会话期间不再出现。例如，如果您提交分析作业但具有相同名称的作业文件已存在，Abaqus/CAE会显示一个对话框，询问是否可以覆盖作业文件，如下所示。

![](../graphics/int-disable-nls.png)

如果切换关闭**下次显示此警告**，对话框将在当前Abaqus/CAE会话的剩余时间内被禁用。
