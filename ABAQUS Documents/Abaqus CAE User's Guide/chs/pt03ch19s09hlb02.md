# 19.9.2 对自适应过程执行数据检查







数据检查将模型的输入文件通过分析输入文件处理器运行，以确保模型的一致性并设置所有必需的模型选项。对自适应过程执行数据检查将创建作业并验证整个自适应过程第一次迭代的模型信息。

要对自适应过程执行数据检查，请从主菜单栏中选择****自适应**![](../graphics/images/arrow.gif)**数据检查**![](../graphics/images/arrow.gif)***自适应过程名称*****。您也可以通过在**自适应过程管理器**中选择过程名称，然后点击**数据检查**来执行数据检查。

数据检查创建的作业出现在**作业**容器中，使用["创建新的自适应过程，"第 19.9.1 节](pt03ch19s09hlb01.md)中讨论的命名约定。要查看数据检查的结果，请在您的工作目录中查看此作业的数据（`.dat`）文件。您还可以通过从主菜单栏中选择****作业**![](../graphics/images/arrow.gif)**监视器**![](../graphics/images/arrow.gif)***创建的作业*****来显示作业监视器对话框，以查看数据检查的结果。
