# 47.1.2 理解如何指定X–Y数据对象









### 47.1.2 理解如何指定X--Y数据对象

要指定*X–Y*数据对象，首先选择数据源，然后提供任何必要的细节。可能的来源包括：

**ODB历史输出**

选择此方法通过从输出数据库读取历史输出结果来指定*X–Y*数据对象。您可以指定要从输出数据库读取哪些变量、从分析的哪些步读取以及读取数据的频率；例如，您可以每三个数据点读取一次。您还可以为任何复数值输出数据指定数字形式。有关更多信息，请参见["从输出数据库历史输出读取X--Y数据，" 第47.2.1节](pt05ch47s02hlb01.md)。

**ODB场输出**

选择此方法通过从输出数据库读取场输出结果来指定*X–Y*数据对象。您可以指定要从输出数据库读取哪些变量以及为哪些元素或节点读取数据。Abaqus/CAE从当前活动的步和帧中提取结果；有关更多信息，请参见["激活和停用步和帧，" 第42.4.1节](pt05ch42s02s01.md)。您还可以为任何复数值输出数据指定数字形式。有关更多信息，请参见["从输出数据库场输出读取X--Y数据，" 第47.2.2节](pt05ch47s02hlb02.md)。

**厚度**

选择此方法通过从模型壳区域厚度的元素中读取场输出结果来指定*X–Y*数据对象。Abaqus/CAE从当前步和帧中提取结果。您可以指定要从输出数据库读取哪些变量以及为哪些元素读取数据。有关更多信息，请参见["通过壳厚度读取X--Y数据，" 第47.2.3节](pt05ch47s02hlb03.md)。

**自由体**

选择此方法通过从会话中所有活动的自由体切片读取场输出结果来指定*X–Y*数据对象。Abaqus/CAE从当前步和帧中提取结果，并显示节点力（NFORC）输出变量的结果。有关更多信息，请参见["从所有活动的自由体切片读取X--Y数据，" 第47.2.4节](pt05ch47s02hlb04.md)。

**对X–Y数据运算**

选择此方法通过操作先前保存的*X–Y*数据对象来派生新的*X–Y*数据对象。您通过将函数和数学运算应用于现有数据来指定新的*X–Y*数据对象。函数的一个例子是**组合**。如果您将包含应力与时间关系的*X–Y*数据对象与包含应变与时间关系的*X–Y*数据对象**组合**，则会产生包含等效时间下应力与应变关系的*X–Y*数据对象。有关更多信息，请参见["对保存的X--Y数据对象进行运算，" 第47.4节](pt05ch47s03.md)。

**ASCII文件**

选择此方法从包含由逗号、空格或制表符分隔的数值数据列的现有文本文件读取*X*和*Y*值。文件可以包含多于两列数据；您可以指定哪些列对应于*X*和*Y*轴数据。此外，您可以指定从文件中读取数据的频率；例如，每第三行。有关更多信息，请参见["从ASCII文件读取X--Y数据，" 第47.2.5节](pt05ch47s02hlb05.md)。

**键盘**

选择此方法手动将*X*和*Y*值键入简单的表格编辑器。在此方法中，Abaqus支持几种特殊的编辑技术，以及从文件读取数据的选项。有关此主题的更多信息，请参见["从键盘输入X--Y数据，" 第47.2.6节](pt05ch47s02hlb06.md)。

**路径**

选择此方法通过沿通过模型的路径在位置处读取场输出结果来指定*X–Y*数据对象。Abaqus从输出数据库获取结果。您可以指定构成路径的点、元素或边，以及要获取结果的步、帧和变量。有关更多信息，请参见[第48章，"沿路径查看结果"](pt05ch48.md)。

此外，您可以在使用表格编辑器创建材料时在属性模块中创建*X–Y*数据对象。有关更多信息，请参见["输入表格数据，" 第3.2.7节](pt01ch03s02s07.md)。

指定*X–Y*数据对象后，您可以保存它或将其显示为*X–Y*图的形式。保存*X–Y*数据对象允许您随后对其进行绘图、编辑、重命名、删除或运算；它还允许您将*X–Y*数据对象复制到输出数据库文件以供以后的Abaqus会话使用。对于源自输出数据库历史输出以外的来源的*X–Y*数据，您必须保存数据以生成包含多个数据对象的*X–Y*图。保存的*X–Y*数据对象仅在会话期间保留。对于跨会话持久化的*X–Y*数据对象，必须将其复制到输出数据库文件。

如果在早期会话期间已将*X–Y*数据对象复制到输出数据库文件，则可以在当前会话中打开该文件时访问该对象。您可以像任何其他指定对象一样以*X–Y*图的形式显示*X–Y*数据对象。要编辑、重命名、删除或操作在前一个会话中创建的数据对象，必须将其加载到当前会话。
