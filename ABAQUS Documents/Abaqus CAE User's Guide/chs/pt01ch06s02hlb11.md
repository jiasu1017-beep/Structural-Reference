# 6.2.12 选择内部表面





你可以使用选择工具选择模型的内部表面；例如，当你创建表面或当你使用实体偏移网格工具选择区域时。

**要选择内部表面：**

1. 在**选择**工具栏中，切换**从内部实体选择**工具 ![](../graphics/ico_filterInterior.png)。**注意：****从内部实体选择**工具默认是隐藏的。更多信息，请参阅["使用包含隐藏图标的工具箱和工具栏，" 第 3.3.2 节](pt01ch03s03s02.md)。
2. 从视口中选择内部表面。
