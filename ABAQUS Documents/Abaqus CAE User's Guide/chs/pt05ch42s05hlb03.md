# 42.7.3 组合多个帧的结果

要通过组合多个输出数据库帧的结果来定义新的场输出，您使用[**Create Field Output From Frames**](pt05ch42s05hlb05.md)对话框。

您可以选择以下涉及帧的运算：
- 对所有帧的值求和
- 查找所有帧中的最小值
- 查找所有帧中的最大值

**Create Field Output From Frames**对话框最初不包含输出。您选择要考虑的输出数据库帧；所有可用于这些帧的结果然后可以组合，或者您可以选择要组合的各个场输出变量。如果您选择对所选帧的值求和，Abaqus/CAE将创建表示所选结果线性组合的新场输出变量。或者，您可以让Abaqus/CAE确定所选结果在所选帧范围内的最小值或最大值。Abaqus不检查帧运算的物理有效性的一致性；例如，两个帧（包含具有不同边界条件的载荷情况的结果）的线性叠加是允许的，即使组合结果可能在物理上无意义。

张量运算在局部坐标系中执行（如果可用）；否则使用全局系统。Abaqus/CAE假定非一元运算涉及的张量的局部坐标系是一致的。
