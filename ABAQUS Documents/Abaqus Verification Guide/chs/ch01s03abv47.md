# 1.3.44 垫片元件装配





**产品：**Abaqus/Standard  

### 测试的元件

GKPE4    GKPE6    GKPS4    GKPS4N    GKPS6    GKPS6N    

GK3D6    GK3D6N    GK3D8    GK3D8N    GK3D12M    GK3D12MN    GK3D18    GK3D18N    

### 问题描述

本节测试将垫片连接到网格其余部分的不同方法。考虑一个1毫米厚的圆柱形垫片，夹在两个同轴圆柱管之间。内圆柱管的内半径为10毫米，外半径为24毫米，而外圆柱管的内半径为25毫米，外半径为50毫米。外圆柱的外表面承受300 MPa的压力。

![](../graphics/vergasketassembly-model-nls.png)

问题可以建模为平面应变问题、平面应力问题或三维问题。利用对称条件，对几何形状的四分之一进行建模。在所有情况下都建模了单位厚度的切片。所有垫片元件的厚度方向为正径向方向。因此，对于任何垫片元件，最接近圆柱轴的表面代表底表面，最远的表面代表顶表面。厚度方向通过对称平面上的用户指定法向定义来指定。垫片可以建模为单层或双层垫片。垫片通过共享节点、TIE和PIN MPC或无摩擦接触对连接到模型的其余部分。当使用接触对时，输入文件演示了通用接触和绑定接触条件的使用。

使用不同的元件类型来建模管道和垫片，并选择合适的方法来连接两种材料。例如，元件类型C3D27R和GK3D18与共享节点一起使用，而C3D20R和GK3D18N与接触对一起使用。用户定义的坐标系用于指定所有三维垫片元件的局部2和3方向。没有进行解的网格收敛性研究。

**材料：**

圆柱管：杨氏模量 = 2.0×10^5 MPa，泊松比 = 0.3。

垫片：厚度方向行为是线弹性的，因此对于单位厚度的垫片，当闭合量为0.002毫米时，压力为400 MPa。使用无卸载曲线的损伤模型来指定此行为。垫片的膜行为与圆柱管具有相同的杨氏模量和泊松比。根据所使用的垫片元件类型及其与圆柱管的连接方法，膜行为可能使用或不使用。

### 结果与讨论

垫片元件中的广义应变与其顶面和底面的位移一致，广义应力通过指定的垫片行为从广义应变正确获得。
