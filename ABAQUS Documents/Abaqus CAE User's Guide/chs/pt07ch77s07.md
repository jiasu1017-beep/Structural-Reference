# 77.7 在可视化模块中对约束进行着色





本节介绍当您在可视化模块中工作且选择了输出数据库时，如何对当前视口中显示的约束应用颜色编码。要对可视化模块中的所有几何图形进行颜色编码，请参见["在可视化模块中对所有几何图形进行着色，"第77.5节](pt07ch77s05.md)。

**对视口中显示的约束应用颜色映射的步骤：**

1. 点击**颜色编码**工具栏中的![](../graphics/ico_colorCode.png)工具。Abaqus/CAE显示**颜色编码**对话框，其中显示当前模块的默认颜色映射。
2. 从**颜色编码**列表中选择**约束**。
3. 在**约束类型**列表中，编辑任何分配的颜色。
4. 点击**应用**。Abaqus/CAE会根据**颜色编码**对话框中的选择对当前视口进行颜色编码。
