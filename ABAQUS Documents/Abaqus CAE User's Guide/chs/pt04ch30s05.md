# 30.5 为流体模型划分网格





流体分析中的场不由单元类型决定，而是由分析过程及其选项决定。单元类型的唯一目的是定义用于离散连续体的单元形状。三种流体类型单元可用。流体单元默认分配给网格。更多信息，请参见["流体连续单元"，Abaqus Analysis User's Guide第28.2节](../usb/usb-link.md#usbfluidcont)。
