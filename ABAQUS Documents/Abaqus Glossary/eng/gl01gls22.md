# U







### unmeshable region

A solid region that Abaqus/CAE cannot mesh automatically using hexahedral elements. An unmeshable region can be made meshable by partitioning, by assigning the bottom-up meshing technique, or by assigning tetrahedral elements to the region. See also [mesh](gl01gls14.md#gls-mesh).
For more information:- [Chapter 17, "The Mesh module," of the Abaqus/CAE User's Guide](../usi/usi-link.md#usi-mgn)

### user-defined view

A custom view to an Abaqus/CAE model in a selected viewport. You can save the current position, orientation, and scale factor as one of four user-defined views and subsequently apply the view to any viewport. A user-defined view is saved for the duration of the session. See also [view](gl01gls23.md#gls-view).
For more information:- ["Saving a user-defined view," Section 5.6.9 of the Abaqus/CAE User's Guide](../usi/usi-link.md#uss-viw-userviewsavebtn)

### user subroutine

Code you write to increase the functionality of several Abaqus options for which data line usage may be too restrictive. A user subroutine is typically written as FORTRAN  code.
For more information:- ["User subroutines and utilities," Section 18.1 of the Abaqus Analysis User's Guide](../usb/usb-link.md#usbusubutil)
