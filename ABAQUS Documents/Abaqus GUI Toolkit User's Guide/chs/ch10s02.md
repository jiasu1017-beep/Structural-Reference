# 10.2 延性金属中的塑性







许多金属在低应变幅度下具有近似线弹性行为（参见[图 10--1](ch10s02.md#gss-stress-strain)），材料的刚度（即杨氏模量或弹性模量）是恒定的。

**图 10–1** 线性弹性材料（如钢）在小应变下的应力-应变行为。

![](../graphics/gss-stress-strain-nls.png)

在更高的应力（和应变）幅度下，金属开始具有非线性、非弹性行为（参见[图 10--2](ch10s02.md#gss-nominal)），这称为塑性。

**图 10–2** 拉伸试验中弹塑性材料的公称应力-应变行为。

![](../graphics/gss-nominal-nls.png)

### 10.2.1 延性金属中塑性的特征

材料的塑性行为由其屈服点和后屈服硬化描述。从弹性到塑性行为的转变发生在材料应力-应变曲线上某个称为弹性极限或屈服点的点（参见[图 10--2](ch10s02.md#gss-nominal)）。屈服点的应力称为屈服应力。在大多数金属中，初始屈服应力是材料弹性模量的 0.05% 到 0.1%。

金属在达到屈服点之前的变形只会产生弹性应变，如果移除施加的载荷，这些应变会完全恢复。然而，一旦金属中的应力超过屈服应力，就开始发生永久（非弹性）变形。与此永久变形相关的应变称为塑性应变。弹性应变和塑性应变都在后屈服区域变形时累积。

金属的刚度一旦材料屈服，通常会显著降低（参见[图 10--2](ch10s02.md#gss-nominal)）。屈服的延性金属在移除施加的载荷时会恢复其初始弹性刚度（参见[图 10--2](ch10s02.md#gss-nominal)）。通常，材料的塑性变形会增加其后续加载的屈服应力：这种行为称为加工硬化。

金属塑性的另一个重要特征是非弹性变形与近乎不可压缩的材料行为相关。在弹性-塑性模拟中对此效果进行建模会对可使用的单元类型施加一些严格限制。

在拉伸载荷下塑性变形的金属在材料失效时可能会经历高度局部的延伸和变薄，称为*颈缩*（参见[图 10--2](ch10s02.md#gss-nominal)）。金属中的工程应力（每单位未变形面积的力）称为*公称应力*，其共轭*公称应变*（每单位未变形长度的长度变化）。金属在颈缩时的公称应力远低于材料的极限强度。这种材料行为是由测试试样的几何形状、测试本身的性质以及所使用的应力和应变测量引起的。例如，在压缩中测试相同材料会产生不具有颈缩区域的应力-应变图，因为试样在压缩载荷下变形时不会变薄。描述金属塑性行为的数学模型应该能够独立于结构的几何形状或施加载荷的性质来解释压缩和拉伸行为之间的差异。如果将公称应力，![](../graphics/gsk_eqn00240.gif)，和公称应变，![](../graphics/gsk_eqn00241.gif)（下标 0 表示材料的未变形状态的值）的熟悉定义替换为考虑有限变形过程中面积变化的新应力和新应变度量，则可以实现此目标。

**Abaqus/Explicit 中的数据规则化**

执行分析时，Abaqus/Explicit 可能不会完全按照用户定义的那样使用材料数据；为了提高效率，以表格形式定义的所有材料数据会自动*规则化*。材料数据可以是温度、外部场和内部状态变量（如塑性应变）的函数。对于每个材料点计算，必须通过插值确定材料的状态，并且为了提高效率，Abaqus/Explicit 用由等间距点组成的曲线来拟合用户定义的曲线。这些规则化材料曲线是分析期间使用的材料数据。了解分析中使用的规则化材料曲线与您指定的曲线之间可能存在的差异是很重要的。

为了说明使用规则化材料数据的含义，考虑以下两种情况。[图 10--5](ch10s02.md#gxi-regularized-data) 显示了用户定义了不规则数据的情况。

**图 10–5** 可以精确规则化的用户数据示例。

![](../graphics/gxi-regularized-data-nls.png)

在这个例子中，Abaqus/Explicit 生成了显示的六个规则数据点，用户的数据被精确重现。[图 10--6](ch10s02.md#gxi-dif-regularize) 显示了用户定义了难以精确规则化的数据的情况。在这个例子中，假设 Abaqus/Explicit 通过将范围划分为 10 个间隔来规则化数据，这些间隔不会精确重现用户的数据点。

**图 10–6** 难以规则化的用户数据示例。

![](../graphics/gxi-dif-regularize-nls.png)

Abaqus/Explicit 尝试使用足够多的间隔，使规则化数据与用户定义数据之间的最大误差小于 3%；但是，您可以更改此误差容限。如果需要超过 200 个间隔才能获得可接受的规则化曲线，分析会在数据检查期间停止并显示错误消息。通常，如果用户定义的最小间隔与自变量范围相比很小，则规则化会更困难。在[图 10--6](ch10s02.md#gxi-dif-regularize) 中，1.0 应变的数据点使得应变值范围很大，与低应变水平定义的较小间隔相比。移除最后一个数据点可以使数据更容易规则化。

**数据点之间的插值**

Abaqus 在提供的数据点之间（或者在 Abaqus/Explicit 中，规则化数据）线性插值以获得材料的响应，并假设响应在输入数据定义的范围内是恒定的（参见[图 10--7](ch10s02.md#gss-mat-svpe)）。因此，这种材料的应力永远不会超过 480 MPa；当材料中的应力达到 480 MPa 时，材料会持续变形直到应力降低到此值以下。

**在 Abaqus/CAE 中进行材料校准**

Abaqus/CAE 允许您从测试数据进行材料模型校准。通过此功能，您可以将材料测试数据导入 Abaqus/CAE，处理数据，并从数据中导出弹性和塑性各向同性材料行为。此功能在 ["创建材料校准，" Abaqus/CAE User's Guide 第 12.17 节](../usi/usi-link.md#usi-prp-calibration) 中进一步讨论。
