# 17.3.4 执行协同仿真





**产品：** Abaqus/Standard  Abaqus/Explicit  Abaqus/CFD  Abaqus/CAE

##### **参考文献**

- ["Abaqus/Standard、Abaqus/Explicit 和 Abaqus/CFD 协同仿真执行," 第 3.2.4 节](pt01ch03s02abx04.md)
- ["理解协同执行," Abaqus/CAE 用户指南第 19.4 节](../usi/usi-link.md#usi-ana-coexecutionconcepts)

### 概述

您可以从命令行执行以下类型的协同仿真：
- 结构对结构
- 流固耦合
- 共轭热传递
- 电磁到结构
- 电磁到热

您也可以在 Abaqus/CAE 中执行结构对结构、流固耦合和共轭热传递协同仿真。

### 从 Abaqus/CAE 执行协同仿真

您可以在 Abaqus/CAE 中交互式执行耦合分析，如 ["理解协同执行," Abaqus/CAE 用户指南第 19.4 节"](../usi/usi-link.md#usi-ana-coexecutionconcepts) 中所述。您不需要创建配置文件；Abaqus/CAE 自动创建该文件。

### 从命令行执行协同仿真

您可以按 ["Abaqus/Standard、Abaqus/Explicit 和 Abaqus/CFD 协同仿真执行," 第 3.2.4 节"](pt01ch03s02abx04.md) 中所述执行 Abaqus 作业。

#### 命令用法示例

使用以下命令提交两个 Abaqus 分析"job-1"和"job-2"之间的协同仿真：

```
abaqus cosimulation cosimjob=beam job=job-1,job-2
configure=config
```

### 使用 timeout 参数的注意事项

timeout 执行参数指定每个分析等待从运行中的另一个分析接收预期协同仿真消息的时间（以秒为单位）。使用命令行选项提交作业时默认超时值为 60 分钟，在 Abaqus/CAE 中执行作业时默认超时值为 10 分钟。当超时时间与典型分析增量实际时间相比很大时，您在启动作业和执行协同仿真分析步骤之前的操作方面具有更大的灵活性。需要这种灵活性的示例包括：使用队列提交作业、在协同仿真步骤之前具有长运行时间的分析，以及因输入错误而重新提交作业的情况。然而，当其中一个协同仿真作业在建立初始协同仿真通信之前失败时（例如由于收敛问题或计算资源可用性），大的超时时间可能导致问题。在这些情况下，您可能更愿意终止仍在运行的作业，而不是让它等待整个超时时间段。

### 限制

Abaqus/CAE 中不支持电磁到结构和电磁到热协同仿真。
