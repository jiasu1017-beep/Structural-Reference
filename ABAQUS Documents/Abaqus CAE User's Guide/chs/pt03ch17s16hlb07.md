# 17.16.7 删除零件或实例种子





通过从主菜单栏选择 ****Seed**![](../graphics/images/arrow.gif)**Delete Part Seeds**** 或 ****Seed**![](../graphics/images/arrow.gif)**Delete Instance Seeds**** 来删除零件或实例种子。

**删除零件或实例种子：**

1. 从主菜单栏，选择 ****Seed**![](../graphics/images/arrow.gif)**Delete Part Seeds**** 或 ****Seed**![](../graphics/images/arrow.gif)**Delete Instance Seeds****。 Abaqus/CAE在提示区域显示提示以指导您完成该过程。 **提示：**您也可以使用Mesh模块工具箱中种子工具所在的 ![](../graphics/ico_meshSeedDeleteInstance.png) 工具删除种子。 （有关更多信息，请参阅["使用Mesh模块工具箱，" 第17.15节](pt03ch17s15.md)。）
2. 执行以下操作之一： - 如果您要删除零件种子，请在提示区域单击**Yes**确认删除。 - 如果您的装配只包含一个零件实例，请在提示区域单击**Yes**确认删除。 - 如果您的装配包含多个零件实例，请选择要删除其种子的实例，然后单击鼠标按钮2提交您的选择。 Abaqus/CAE会删除种子。
