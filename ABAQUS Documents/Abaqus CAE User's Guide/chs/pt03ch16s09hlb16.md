# 16.9.16 定义惯性relief载荷





您可以创建惯性relief载荷以平衡作用在自由或部分自由物体上的外部力。惯性relief载荷应用于整个模型。对于每个常规分析步骤，您只能应用一个活动的惯性relief载荷。有关惯性relief载荷的详细信息，请参见"Abaqus Analysis User's Guide"第11.1.1节"Inertia relief"。

**注意：**您不能在子模型中应用惯性relief载荷。对于子模型，Abaqus会忽略通过在全局模型中包含惯性relief载荷计算出的惯性relief效果。

**创建或编辑惯性relief载荷：**

1. 使用以下方法之一显示惯性relief载荷编辑器：- 若要创建新的惯性relief载荷，请遵循"创建载荷"，第16.8.1节中概述的过程（**类别**：`Mechanical`；**所选步骤的类型**：`Inertia relief`）。- 若要使用菜单或管理器编辑现有的惯性relief载荷，请参见"编辑依赖于步骤的对象"，第3.4.13节。
2. 如果编辑器顶部附近出现**方法**字段，单击该字段右侧的箭头，并选择以下之一：- 选择**计算载荷**以继续计算指定方向的载荷。- 选择**固定在当前载荷**以将载荷固定为先前步骤中的幅值和方向。**方法**选项在创建惯性relief载荷的步骤中不可用。
3. 切换打开一个自由度，以定义您要应用惯性relief载荷的自由方向。显示的自由度取决于建模空间。
4. 如果您要更改惯性relief载荷的坐标系（**CSYS**），请单击![](../graphics/ico_selectBlue.png)并使用以下方法之一：- 在视口中选择现有的基准坐标系。- 按名称选择现有的基准坐标系。1. 从提示区域单击**基准坐标系列表**以显示基准坐标系列表。2. 从列表中选择一个名称，然后单击**确定**。- 从提示区域单击**使用全局CSYS**以恢复到全局坐标系。此坐标系编辑选项仅在创建惯性relief载荷的步骤中可用。默认情况下，使用全局坐标系来定义载荷。
5. 如果编辑器底部出现**X**、**Y**和**Z**文本字段，请输入定义刚体运动所需的附加点的坐标。对于某些自由方向组合，您必须定义一个附加点。更多信息，请参见"Abaqus Analysis User's Guide"第11.1.1节"Inertia relief"。
6. 单击**确定**保存您的数据并退出编辑器。
