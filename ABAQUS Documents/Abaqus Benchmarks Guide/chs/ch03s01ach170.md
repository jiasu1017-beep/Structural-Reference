### 材料数据的指定

在分析中指定材料数据需要以下步骤：
- 进行不同类型的测试以测量应力-应变数据。
- 将超弹性常数拟合到试验数据。
- 检查超弹性模型数值结果与试验数据之间的相关性。
- 如果满意，则继续进行有限元分析；否则，采取纠正措施，并再次尝试拟合过程。

在评估曲线拟合时，应使用以下标准：
- 如果有单轴、双轴和平面数据可用，计算曲线与测量数据的近似程度如何？
- 如果只有有限的试验数据可用，对除测量之外的其他变形模式的预测有多现实？在没有材料数据的情况下，这需要一些工程判断。在本例中，我们通过限制单轴拉伸数据的曲线拟合来模拟这种情况，即使所有数据都可用。
- 德鲁克稳定性准则是否满足？

对于多项式势，![](../graphics/bmk_eqn01398.gif) = 1（Mooney-Rivlin）的情况在低应变时给出合理的拟合，但无法再现橡胶在高应变时应变强化的响应。![](../graphics/bmk_eqn01398.gif) = 2的情况提供了高阶项，使所有应变水平都能与测试数据更紧密相关。类似观察适用于简化多项式，![](../graphics/bmk_eqn01398.gif) = 1（neo-Hookean）和![](../graphics/bmk_eqn01398.gif) = 3（Yeoh）；neo-Hookean模型仅提供第一不变量的线性依赖，因此无法提供对上翘的准确表示。相比之下，三项简化多项式（Yeoh）比具有五系数的![](../graphics/bmk_eqn01398.gif) = 2的完整多项式提供了更准确的表示。此外，Yeoh模型在拟合Treloar测试数据时没有表现出任何不稳定性。

Arruda-Boyce模型也给出了满意的拟合。在单轴情况下，上翘不如实验中那么陡峭；在中间拉伸范围内应力被高估。文献中已报道了其他曲线拟合；例如，Boyce（1996）报道了![](../graphics/bmk_eqn01492.gif) = 0.27 MPa ≈ 2.75 kgf/cm²和![](../graphics/bmk_eqn01493.gif) = 2.75。这些与我们的值不同，![](../graphics/bmk_eqn01492.gif) = 3.28 kgf/cm²和![](../graphics/bmk_eqn01493.gif) = 5.24。差异可归因于最小化的是相对误差。另一个可能的差异来源是Treloar测试数据的不同间距。

Van der Waals模型给出了比Arruda-Boyce模型更好的拟合，但不如Ogden模型好。应力-应变曲线的所有拉伸范围都以高精度拟合。我们的拟合与文献中报道的比较有利；然而，我们使用了更精细的模型，因为我们考虑了第二不变量的轻微依赖。

#### 拟合案例2——仅使用单轴拉伸数据

通常，用户可能只有一种或两种类型的测试数据可用。本例检查了仅使用单轴压缩数据拟合超弹性常数的结果。分析了以下案例，![](../graphics/bmk_eqn01398.gif) = 1（Mooney-Rivlin形式）和![](../graphics/bmk_eqn01398.gif) = 2的多项式形式；![](../graphics/bmk_eqn01398.gif) = 1（neo-Hookean形式）和![](../graphics/bmk_eqn01398.gif) = 3（Yeoh形式）的简化多项式形式；Ogden形式，![](../graphics/bmk_eqn01398.gif) = 2和![](../graphics/bmk_eqn01398.gif) = 3；Arruda-Boyce形式；Van der Waals形式，![](../graphics/bmk_eqn01494.gif)；和Marlow形式。

除了![](../graphics/bmk_eqn01398.gif) = 1的多项式模型（neo-Hookean和Mooney-Rivlin形式）外，单轴拉伸结果与单轴测试数据非常紧密相关。这是预期的，因为超弹性常数是使用单轴数据拟合的。然而，（一般）多项式和Ogden模型在等双轴拉伸和平面拉伸情况下显示数值和测试数据之间的很大差异。

对于![](../graphics/bmk_eqn01398.gif) = 1（Mooney-Rivlin）的情况，等双轴和平面拉伸情况的不稳定性立即发生。对于![](../graphics/bmk_eqn01398.gif) = 2的多项式情况，应力在高应变时非常快速增加。

对于Ogden势，![](../graphics/bmk_eqn01398.gif) = 3的情况在中等应变时开始显著偏离，但没有![](../graphics/bmk_eqn01398.gif) = 2的多项式情况那么严重。值得注意的是，Ogden ![](../graphics/bmk_eqn01398.gif) = 2的情况即使在高应变时仍给出相当紧密的拟合。额外测试数据的经验表明可以推广这些观察结果。

通过忽略多项式模型对第二不变量的依赖，可以对未测量的应力状态获得更好的预测。此观察与文献中报道的结果一致；见Kaliske和Rothert（1997）或Yeoh（1993）。特别是，neo-Hookean模型即使在仅从一个单轴测试测量系数![](../graphics/bmk_eqn00087.gif)时也对所有应力状态提供良好的第一阶近似，而在我们的示例中Mooney-Rivlin模型甚至无法正确预测定性趋势。Yeoh模型（或简化多项式，N=3）在所有应力状态下提供良好的三阶近似，在本例中未表现出任何不稳定性。不太可能需要更高阶的简化多项式，除了例如应力-应变曲线是"双S形"时。

当应变能势从单轴数据导出时，Van der Waals、Arruda-Boyce和Marlow模型给出了所有三种变形模式的最佳拟合。如果小拉伸范围内的测试数据更密集且S形更明显（如填充橡胶中所常见），Van der Waals模型可能表现出更明显的优越性，因为附加参数在表示复杂应力-应变曲线时创建了增强的灵活性。

### 结果与讨论

对于Treloar的测试数据，当考虑单轴、双轴和平面测试数据时，Ogden和Van der Waals形式比多项式形式给出更紧密的拟合。Arruda-Boyce和Yeoh形式也提供了准确的表示。（一般）多项式形式在![](../graphics/bmk_eqn01398.gif) = 2时表现出一些不稳定性，在![](../graphics/bmk_eqn01398.gif) = 1时仅提供一阶近似。

当只有有限的测试数据可用时，会得出完全不同的结论。在这种情况下，Van der Waals模型（带![](../graphics/bmk_eqn01494.gif)）和Arruda-Boyce模型明显优于Ogden模型。当忽略对第二不变量的依赖时，多项式模型显著增强。Yeoh模型即使对于未纳入曲线拟合的变形模式也给出了非常好的三阶表示。类似地，neo-Hookean模型即使在仅基于一种变形状态拟合时也对所有应力状态给出良好的第一阶近似。

Ogden拟合的高质量，与（一般）多项式相比，可以解释为Ogden势在符合测试数据方面的灵活性——指数![](../graphics/bmk_eqn00097.gif)可以取任何实数值，而多项式势只能有整数指数。

然而，对于最一般的模型——Ogden和（一般）多项式——如果实际要分析的弹性体模型将经历一般应力-应变状态，则使用多个且独立的测试数据类型来拟合超弹性常数是很重要的。

根据Yeoh（1993），我们建议在不完全或有限的材料数据可用时忽略对第二不变量的依赖；Van der Waals模型的曲线拟合应使用![](../graphics/bmk_eqn01494.gif)进行，应优先选择简化多项式形式而不是（一般）多项式模型。Arruda-Boyce模型根据定义独立于第二不变量。对于Ogden模型，无法抑制对第二不变量的依赖。

图3.1.4-15至图3.1.4-17显示了使用不同测试数据的Marlow模型的结果。可以看出，该模型可以在有测试数据可用的变形模式下精确表示材料的行为，并在其他变形模式下具有合理的行为。
