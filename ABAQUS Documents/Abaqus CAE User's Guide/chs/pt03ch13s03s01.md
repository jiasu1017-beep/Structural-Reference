# 13.3.1 理解模型、零件、实例和装配之间的关系







一个模型可以包含许多零件；但是，它只能包含一个顶层装配。装配由在全局坐标系中相对定位的零件实例组成，如"什么是零件实例？"，第11.3.4节中所述。顶层装配还可以包含从其他模型有效创建子装配的模型实例。

零件、零件实例和装配的概念贯穿于Abaqus/CAE建模过程：

1. 您在零件模块中创建零件；每个零件是一个独立的实体，可以独立于其他零件进行修改和操作。零件存在于自己的坐标系中，不了解其他零件。
2. 您在属性模块中定义截面属性，并将材料关联到截面。您使用属性模块将这些截面属性分配给零件或零件的选定区域。
3. 您在装配模块中创建零件实例，并在全局坐标系中相对定位这些实例以形成装配。您还可以在装配中添加其他模型的实例。Abaqus/CAE允许您创建从属或独立零件实例，如"从属和独立零件实例之间的区别是什么？"，第13.3.2节中所述。独立和从属零件实例都保持与原始零件的关联。当您在零件模块中修改原始零件时，Abaqus/CAE会在您返回装配模块时更新该零件的任何实例。您可以多次实例化一个零件并组装同一零件的多个实例。零件的每个实例都与分配给该零件的截面属性关联。
4. 您使用交互模块和载荷模块完成模型定义，例如定义接触以及施加载荷和边界条件。交互模块和载荷模块在装配上操作。
5. 您使用网格模块对装配进行网格划分。您可以执行以下任一操作：- 分别对装配中零件的每个独立实例进行网格划分。- 对原始零件进行网格划分。然后Abaqus/CAE将网格与装配中零件的每个从属实例关联。两种网格划分方法在"从属和独立零件实例之间的区别是什么？"，第13.3.2节中描述。

"创建零件或模型实例"，第13.10.4节包含创建零件实例的详细说明。
