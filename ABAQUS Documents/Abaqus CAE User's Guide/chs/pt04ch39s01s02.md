# 39.1.2 在分析中包含子结构





子结构的使用应在与子结构生成不同的模型中执行。您可以通过以下一般步骤在 Abaqus/CAE 中将子结构包含在分析中：

1. 从相应的 `.sim` 文件中将您要在模型数据库中使用的每个子结构导入。有关更多信息，请参见["将子结构导入 Abaqus/CAE，"第39.4节](pt04ch39s04.md)。
2. 在 Assembly 模块中，实例化要添加到装配中的每个子结构零件，并将子结构零件实例放置在装配中所需的位置。[在装配中使用子结构零件实例，"第39.5节](pt04ch39s05.md)解释了子结构零件实例的功能和限制。
3. 在 Load 模块中，通过创建**子结构负载**定义来激活子结构负载情况。有关更多信息，请参见["在子结构使用期间激活负载情况，"第39.6节](pt04ch39s06.md)。
4. 在 Step 模块中，创建以**子结构**为**域**的场输出请求，然后选择要恢复场数据的子结构集。有关更多信息，请参见["恢复子结构的场输出，"第39.7节](pt04ch39s07.md)。
5. 在 Interaction 模块中，应用约束将子结构零件实例连接到装配的其余部分。
