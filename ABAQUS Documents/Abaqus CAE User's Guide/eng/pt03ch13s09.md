# 13.9 Using the Assembly module toolbox







You can access all the Assembly module tools through either the main menu bar or through the Assembly module toolbox. [Figure 13--24](pt03ch13s09.md#asm-toolbox) shows the hidden icons for all the Assembly module tools in the toolbox. 

**Figure 13–24** The Assembly module tools.

![](../graphics/asm-toolbox-nls.png)

To see a tooltip containing a brief definition of an Assembly module tool, hold the mouse over the tool for a moment. For information on using toolboxes and selecting hidden icons, see ["Using toolboxes and toolbars that contain hidden icons," Section 3.3.2](pt01ch03s03s02.md).
