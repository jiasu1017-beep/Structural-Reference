# 31.2.7 为裂缝区域划分网格和分配单元





裂缝尖端会出现较大的应力集中。因此，您应该在裂缝尖端周围创建细化网格，以获得准确的应力和应变结果。相比之下，因为*J*积分是一种能量度量，所以您可以使用相对粗糙的网格获得准确的*J*值。但是，如果材料变得更具非线性，则必须在裂缝尖端创建更细的网格以保持*J*值的准确性。您可以通过分割区域和为生成的边缘分配网格种子来控制裂缝尖端的网格密度。更多信息，请参见["理解种子"，第17.4节](pt03ch17s04.md)。

对于Abaqus将允许几何非线性的的大应变分析，您应该使用四边形或六面体单元为轮廓积分区域划分网格。更多信息，请参见["传统有限元方法有限应变分析的断裂力学网格构建"中的"轮廓积分评估"，Abaqus Analysis User's Guide第11.4.2节](../usb/usb-link.md#usb-anl-acontintegral-finite)。

但是，对于不允许几何非线性的小应变分析，您必须通过使用一圈三角形或楔形单元对定义裂缝前缘的区域进行网格划分来容纳裂缝尖端或裂缝线的奇异性。更多信息，请参见["控制小应变分析中裂缝尖端的奇异性"，第31.2.5节](pt04ch31s02s05.md)。

您必须使用扫掠网格技术来创建楔形单元；但是，Abaqus/CAE可以使用扫掠网格技术划分的区域存在限制，如["三维实体的扫掠网格划分"，第17.9.3节](pt03ch17s09s03.md)中所述。因此，如果您无法使用扫掠网格技术，则无法创建楔形单元，并且无法在裂缝线上容纳奇异性。在大多数情况下，如果您的网格足够细化以模拟裂缝尖端或裂缝线周围的变形以及由此产生的高应变梯度，则可以忽略奇异性。如果只对轮廓积分输出感兴趣，也可以忽略奇异性。
