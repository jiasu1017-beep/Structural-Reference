# 12.2 常用方法





你可以使用以下语句访问应用程序对象：

```
app = getAFXApp()
```

以下列表显示了一些最常用的应用程序方法：

**getAFXMainWindow() **

返回主窗口对象的句柄。

**getProductName() **

返回产品名称。

**getVersionNumbers()**

返回一个元组(*majorNumber*, *minorNumber*, *updateNumber*)。

**getPrerelease() **

如果此应用程序是预发布版本，则返回True。

**beep() **

响铃系统铃。
