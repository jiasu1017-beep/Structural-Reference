# 12.13.16 创建内聚截面





内聚截面用于定义两个粘合部件之间界面处粘合层的截面属性。请参见[Abaqus Analysis User's Guide的第32.5.1节，"内聚元素：概述"](../usb/usb-link.md#usb-elm-ecohesiveoverview)。[第21章，"粘接接头和粘接界面"](pt04ch21.md)描述了涉及内聚截面的分析的整个建模过程。

**指定内聚截面属性：**

1. 从主菜单栏，选择****Section**![](../graphics/images/arrow.gif)**Create****。出现**Create Section**对话框。
   **提示：**您也可以在**Section Manager**中点击**Create**，或在Property模块工具箱中选择创建截面工具![](../graphics/ico_section.png)。
2. 输入截面名称。有关命名对象的更多信息，请参见["使用基本对话框组件，"第3.2.1节"](pt01ch03s02s01.md)。
3. 选择**Other**作为截面**Category**，选择**Cohesive**作为截面**Type**，然后点击**Continue**。出现内聚截面编辑器。
4. 为内聚截面选择一种材料。如有需要，点击![](../graphics/ico_material.png)创建材料；有关更多信息，请参见["创建或编辑材料，"第12.7.1节"](pt03ch12s07hlb01.md)。
5. 选择用于定义内聚截面本构行为的**Response**：
   - 如果响应直接以牵引力和分离来定义，请选择**Traction Separation**。使用此选项对可忽略的薄粘合层（脱粘）进行建模。
   - 选择**Continuum**以对涉及一个直接（开口应变）和两个横向剪切分量的应变状态进行建模。使用此选项对有限厚度粘合层进行建模。
   - 选择**Gasket**以指定应力状态为单轴。
6. 指定内聚截面的**Initial thickness**。
   - 切换**Use analysis default**以对**Traction Separation**或**Continuum**响应使用分析默认值。**Gasket**响应没有可用默认值。
   - 切换**Use nodal coordinates**以从节点坐标获取初始厚度。
   - 切换**Specify**，并输入初始厚度的值。
7. 为截面**Out-of-plane thickness**输入值。如果截面将用于二维区域，则必须指定截面厚度。如果区域类型不需要厚度信息，Abaqus/CAE会忽略厚度信息。
8. 点击**OK**保存更改并关闭内聚截面编辑器。
