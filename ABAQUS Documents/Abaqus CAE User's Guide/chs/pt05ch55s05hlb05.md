# 55.5.5 自定义节点符号







节点符号（圆圈、正方形、三角形等）用于标识每个节点的位置。例如，[图 55–10](pt05ch55s05hlb05.md#viw-nodesym) 中使用实心圆圈来表示节点。

**图 55–10** 显示节点符号的模型。

![](../graphics/viw-nodesym.png)

节点符号与绘图状态无关；即，您只需设置一次，即可用于所有未变形、变形、轮廓、符号和材料取向绘图。当变形形状叠加在未变形形状上时，您还可以设置 Abaqus/CAE 将在未变形形状上显示的节点符号。切换 **显示节点符号** 可显示或隐藏节点标签，并可选择其颜色。

**自定义节点符号：**

1. 找到通用或叠加节点符号选项。通用节点符号选项始终应用于变形形状；当未变形形状在任意绘图状态（未变形、变形、轮廓、符号或材料取向）中单独绘制时，它们也适用于未变形形状。当未变形形状与变形形状一起绘制时，叠加节点符号选项适用。从主菜单栏，选择 ****选项**![](../graphics/images/arrow.gif)**通用**** 或 **叠加**；然后在出现的对话框中点击 **标签** 选项卡。
2. 切换 **显示节点符号** 可显示或隐藏节点符号。当 **显示节点符号** 打开时，节点符号颜色、类型和大小选项可用。
3. 选择节点符号的颜色：1. 点击颜色样本 ![](../graphics/usi-colorsample.png)。Abaqus/CAE 显示 **选择颜色** 对话框。2. 使用 **选择颜色** 对话框中的方法之一选择新颜色。有关更多信息，请参见 ["自定义颜色，" 第 3.2.9 节](pt01ch03s02s09.md)。3. 点击 **确定** 关闭 **选择颜色** 对话框。颜色样本变为所选颜色。在您点击 **绘图选项** 对话框中的 **确定** 或 **应用** 之前，新颜色不会出现在视口中。
4. 选择节点符号的类型（圆圈、正方形、三角形等）：1. 点击 **符号** 按钮以显示节点符号类型选项。2. 点击所需的符号类型。指定的符号类型会显示在 **符号** 按钮上。
5. 选择节点符号的大小：1. 点击 **大小** 按钮以显示节点符号大小选项。2. 点击 **小**、**中** 或 **大**。指定的大小会显示在 **大小** 按钮上。
6. 点击 **应用** 实现您的更改。符号根据您的规格进行更改。默认情况下，您的更改会在会话期间保存，并会影响当前视口和从当前视口创建的任何新视口中的所有后续绘图。如果您希望保留所做的更改以供后续会话使用，请将它们保存到文件。有关更多信息，请参见 ["保存自定义设置以供后续会话使用，" 第 55.1.1 节](pt05ch55s01s01.md)。
