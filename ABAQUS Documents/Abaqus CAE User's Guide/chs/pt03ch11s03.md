# 11.2 进入和退出零件模块





您可以在Abaqus/CAE会话期间随时通过单击上下文栏中**模块**列表中的**零件**来进入零件模块。**零件**、**形状**、**特征**和**工具**菜单出现在主菜单栏上，如果存在当前零件，当前视口的标题栏会显示该零件的名称。

要退出零件模块，请从**模块**列表中选择任何其他模块。在退出模块之前，您不必采取任何特定操作来保存零件；当您通过从主菜单栏选择**文件**![](../graphics/images/arrow.gif)**保存**或**文件**![](../graphics/images/arrow.gif)**另存为**来保存整个模型时，零件会自动保存。
