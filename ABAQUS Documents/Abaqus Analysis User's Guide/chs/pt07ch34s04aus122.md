# 34.4.4 分布载荷









**产品：** Abaqus/StandardAbaqus/ExplicitAbaqus/CFD

##### **参考资料**

- [*DLOAD*](#)
- ["施加载荷：概述," 第34.4.1节](pt07ch34s04aus120.md)

### 概述

分布载荷（DLOAD）用于向有限元模型中的单元面、单元边或几何实体施加分布力。分布载荷包括压力载荷、线载荷和面载荷。

### 分布载荷类型

| 类型 | 描述 | 应用 |
| --- | --- | --- |
| P1 | 均匀压力 | 所有单元类型 |
| P2 | 非均匀压力 | 通过用户子程序 |
| LINE | 线载荷 | 梁/桁架单元 |
| SURF | 面载荷 | 壳/实体单元 |

### 均匀压力载荷

| **输入文件用法：** | ```
*DLOAD
element_or_element_set, pressure_magnitude
``` |
| --- | --- |

### 使用幅值曲线

可以使用幅值曲线定义分布载荷随时间的变化：

| **输入文件用法：** | ```
*DLOAD, AMPLITUDE=name
element_or_element_set, pressure_magnitude
``` |
| --- | --- |

### 非均匀压力载荷

对于非均匀分布压力，需要使用用户子程序 DLOAD：

| **输入文件用法：** | ```
*DLOAD, USER
element_or_element_set, pressure_label
``` |
| --- | --- |

### 线载荷

线载荷作用于梁或桁架单元：

| **输入文件用法：** | ```
*DLOAD
element_or_element_set, LINE, magnitude
``` |
| --- | --- |

### 旋转载荷

旋转结构上的离心载荷：

| **输入文件用法：** | ```
*DLOAD, CENTRIFUGAL
element_set, omega, R
``` |
| --- | --- |

其中 omega 是角速度，R 是参考半径。

### 体积载荷

体积载荷作用于整个单元体积：

| **输入文件用法：** | ```
*DLOAD
element_set, VOLUMEN, magnitude
``` |
| --- | --- |

### 重力载荷

重力载荷：

| **输入文件用法：** | ```
*DLOAD
element_set, GRAV, magnitude
``` |
| --- | --- |
