# 12.4.1 创建材料





要创建材料，请从主菜单栏选择****Material**![](../graphics/images/arrow.gif)**Create****。出现**Edit Material**对话框，您可以在其中输入材料的名称并创建或编辑材料属性。材料编辑器如图[12-2](pt03ch12s04s01.md#prp-editor)所示。

**注意：**创建材料后，无法使用材料编辑器重命名；您必须使用****Material**![](../graphics/images/arrow.gif)**Rename****来更改现有材料的名称。

**图12-2** 材料编辑器。

![](../graphics/prp-editor-nls.png)

材料编辑器由以下部分组成：

**Material Behaviors列表**

您已包含在材料定义中的行为列表。

**Behavior菜单**

行为列表下方的一组菜单，您从中选择材料行为。

**行为定义区域**

窗口的下半部分，其中显示与所选行为关联的参数、表格数据字段和子选项。

**注意：**您可以通过从主菜单栏选择****Help**![](../graphics/images/arrow.gif)**On Context****，然后点击感兴趣的编辑器功能，来显示有关此处未讨论的编辑器特定方面的帮助。
