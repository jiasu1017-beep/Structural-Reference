# 42.6.1 了解结果如何计算

显示存储在输出数据库中的结果所需的计算取决于结果是针对基于节点的量（如位移或速度）还是基于单元的量（如应力或应变）。

**基于节点的场输出结果如何计算**

基于节点的场输出变量与在模型创建期间应用的任何节点变换一起写入到输出数据库的每个节点。对于基于节点场输出的显示，Abaqus/CAE从输出数据库中读取包含在绘图中的每个节点的值。默认情况下，这些值显示在全局坐标系中；您可以选择将节点变换应用于结果，或应用用户指定的坐标系变换。然后使用最终值生成等值线、节点探测值、基于结果的显示组或颜色编码，或沿路径的*X–Y*数据。

**基于单元的场输出结果如何计算**

基于单元的场输出变量根据变量类型写入到输出数据库的积分点、单元形心或单元节点。对于基于单元场输出变量的显示，Abaqus/CAE从输出数据库中读取连接到包含在绘图中的所有节点的所有单元的值。然后对这些值应用计算以生成等值线、节点探测结果、基于结果的显示组或颜色编码，或沿路径的*X–Y*数据。

对于保存到积分点或单元形心的结果，第一个应用的计算是外推。（保存到单元节点的结果不需要外推。）仅对于等值线图，您可以选择"quilt型"外推，在这种情况下，以下讨论的剩余计算不适用。要了解有关quilt型外推的更多信息，请参阅["了解如何计算等值线值，" 第44.1.1节](pt05ch44s01s01.md)。对于所有其他结果显示方法，Abaqus/CAE使用适合单元类型和形状的加权将结果外推到节点。

外推值通常不如在积分点计算的值准确。因此，建议在需要精确节点值的此类单元结果周围进行足够详细的网格划分。在解释外推到节点的四边形单元（具有退化边的一侧，如在二维中）或六面体单元（具有退化面的一侧，如在三维中）的输出变量时，应特别小心。

单元张量量的外推在局部材料坐标系中的各个张量分量上执行。共同属于两个或多个单元的节点从所有贡献单元接收外推值。根据模型的特征，这些贡献可能来自多个[结果区域](pt05ch42s04hlb03.md)。如果节点处的所有贡献来自单个结果区域，则根据需要在进一步计算中组合这些值。如果从多个结果区域收到贡献，您可以选择遵守区域边界并在进一步计算中保持贡献分离，或者忽略区域边界并组合值。输出数据库中的默认结果区域复制了分析前用于为模型分配截面属性的区域。或者，您可以选择使用单元集或显示组作为结果区域。有关的详细信息，请参阅["控制区域边界处的计算，" 第42.6.7节](pt05ch42s04hlb03.md)。

如果请求了不变量或分量，您可以指定Abaqus/CAE是使用每个单元的外推数据还是使用所有贡献单元的组合数据来计算不变量。默认情况下，不变量在组合（平均）外推结果之前计算。等值线图的等值线或分量可能会受到Abaqus执行计算的顺序的影响。例如，von Mises应力的值可能超过非弹性材料的屈服应力；此外，不变量结果可能不会考虑材料方向在有限单元中以非等参方式变化的情况。如果在平均之后计算不变量，Abaqus通过平均贡献单元的方向来确定节点处的方向；如果贡献单元之间的方向不同，分量值将受到影响。

如果您选择单元集来定义结果区域，并且不变量将在平均之后计算，则选择的单元集必须包含兼容单元。兼容单元
- 共享相同的基本单元类型（连续体、壳、梁等），
- 使用相同阶数的插值函数（一次单元与二次单元），并且
- 具有相同的积分方案（减缩积分、完全积分等）。

最后，计算取决于您选择显示场输出值还是间断；间断是相邻单元之间场输出值的差异。
- **场输出**：对于场输出值的显示，根据贡献[结果区域](pt05ch42s04hlb03.md)的兼容性和您选择的选项，在共同属于两个或多个单元的节点处有条件地平均计算的不变量或分量。有关的详细信息，请参阅["了解结果值平均，" 第42.6.2节](pt05ch42s04s02.md)。
- **间断**：对于间断的显示，根据贡献结果区域的兼容性和您选择的选项，在共同属于两个或多个单元的节点处比较计算的不变量或分量以确定最大差异。仅与一个单元关联的节点和从所有贡献单元接收等值的节点将在间断图中显示为零值。

有关的详细信息，请参阅["显示场输出值或间断，" 第42.6.5节](pt05ch42s04hlb01.md)。

**结果变换如何计算**

基于节点和基于单元的结果都可以变换到用户指定的坐标系，并且可以对基于坐标和距离的节点矢量结果应用角度变换；请参阅["将结果变换到新坐标系，" 第42.6.8节](pt05ch42s04hlb04.md)，了解如何将变换应用于您的结果。

三维连续体单元的基于单元结果和所有基于节点的结果根据结果的位置变换到指定坐标系。变换结果的1-、2-和3方向对应于直角坐标系的*X*-、*Y*-和*Z*方向；圆柱坐标系的*R*-、![](../graphics/usi_eqn00065.gif)-和*Z*方向；以及球面坐标系的*R*-、![](../graphics/usi_eqn00065.gif)-和![](../graphics/usi_eqn00226.gif)方向。

二维连续体单元、壳单元和膜单元的基于单元结果通过在单元结果位置处围绕单元法向旋转结果来进行变换。变换结果的2方向由直角*Y*方向或圆柱或球面![](../graphics/usi_eqn00065.gif)方向在单元平面上的投影确定。如果用户定义坐标系的*Y*-或![](../graphics/usi_eqn00065.gif)方向与单元法向形成的角度小于30，则投影在单元平面上的下一个跟随*Y*-或![](../graphics/usi_eqn00065.gif)轴的轴（在轴的循环排列中）被投影以形成局部材料2方向，并显示警告消息。此结果变换方法不同于Abaqus/Standard和Abaqus/Explicit用于计算局部方向的方法（请参阅["方向，" Abaqus分析用户指南第2.2.5节](../usb/usb-link.md#usb-int-corientation)）。

梁和桁架单元的基于单元结果无法变换；它们始终以局部单元方向系统显示。此外，钢筋以及CAXA、SAX或SAXA单元的单元结果不会变换。

当您将结果变换到用户指定的坐标系时，您可以控制坐标系变换的以下附加方面：
- 您可以在变换计算中包含或排除当前变形的影响。变形效果不会缩放；Abaqus/CAE使用变形比例因子1.0执行这些计算。在变换中包含变形效果可以改变基于节点的坐标系统的方向、壳和膜单元上坐标系的投影，以及位置相关的圆柱和位置相关的球面坐标系统的方向。
- 您可以调整结果以考虑坐标系的刚体变换。您可以调整主变量结果或主变量和变形变量两者的结果的显示。刚体变换帮助您了解存在大刚体位移时的相对位移。刚体变换将变换应用于模型，以抵消跟随节点的指定坐标系（请参阅["在后处理期间创建坐标系，" 第42.8节](pt05ch42s06.md)，了解更多信息）从一帧到下一帧变形时这些节点的位移和旋转。刚体变换被确定为将指定坐标系节点从其当前位置变换回其原始位置所需的平移和旋转。

您可以对基于坐标和距离的节点矢量结果应用角度变换。角度变换计算圆柱坐标系的*R*、![](../graphics/usi_eqn00065.gif)和*Z*分量，以及球面坐标系的*R*、![](../graphics/usi_eqn00065.gif)和![](../graphics/usi_eqn00226.gif)分量。

您可以对包含场输出变量SORIENT输出的结果和复合截面应用铺层方向变换。此变换使用每个单独铺层上单元的方向而不是整个复合铺层的单一方向来计算张量和矢量场。
