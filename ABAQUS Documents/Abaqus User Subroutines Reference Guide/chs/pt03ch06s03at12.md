### 概述

模态动态分析：
- 用于使用模态叠加分析瞬态线性动态问题；
- 只能在频率提取过程之后执行，因为它基于结构的系统模态；
- 可以使用高性能SIM软件架构（见["动态分析过程：概述，" 第6.3.1节中使用SIM架构进行模态叠加动态分析"](pt03ch06s03abo07.md#usb-anl-alineardynamics)）；
- 仅在使用SIM架构时才能包含非对角阻尼效应（即来自材料或单元阻尼）；以及
- 是一个线性扰动过程。

### 模态动态分析

瞬态模态动态分析给出了模型作为给定时间依赖载荷函数的响应。结构的响应基于系统模态的一个子集，必须首先使用固有频率提取过程提取（["固有频率提取，" 第6.3.5节"](pt03ch06s03at10.md)）。模态将包括特征模态，如果在前面的固有频率提取步骤中激活了残余模态。提取的模态数目必须足以充分建模系统的动态响应，这需要您进行判断。

模态振幅通过时间积分，响应由这些模态响应合成。对于线性系统，模态动态过程比动态过程（["使用直接积分的隐式动态分析，" 第6.3.2节"](pt03ch06s03at07.md)）中对整个方程组进行直接积分的计算成本要低得多。

只要系统是线性的并且被正确地用所使用的模态表示（这些通常只是有限元模型全部模态的一小部分），该方法也非常准确，因为只要激励函数随时间 piecewise 线性变化，使用的积分算子就是精确的。您应确保激励函数定义和时间增量的选择是一致的。例如，如果激励是地震记录，其中加速度值每毫秒给出一次，并假定加速度在这些值之间线性变化，则模态动态过程中使用的时间增量应为1毫秒。

用户指定的最大增量数目在模态动态步骤中被忽略。增量数目基于为该步骤选择的时间增量禾总时间。

虽然此过程中的响应是线性振动，但先前的响应可以是非线性的，如果为固有频率提取过程的基态步骤定义包含了非线性几何效应，则应力刚化（初始应力）效应将包含在响应中，如["固有频率提取，" 第6.3.5节"](pt03ch06s03at10.md)中所解释。

##### 选择模态和指定阻尼系数的规则

以下规则适用于选择模态和指定模态阻尼系数：
- 默认不包含模态阻尼。
- 模态选择和模态阻尼必须以相同的方式指定，使用模态号或频率范围。
- 如果不选择任何模态，则将在叠加中使用先前频率分析中提取的所有模态，包括激活的残余模态。
- 如果不为选择的模态指定阻尼系数，则这些模态将使用零阻尼值。
- 阻尼仅应用于所选的模态。
- 所选模态中超出指定频率范围的阻尼系数是常数，等于为第一个或最后一个频率指定的阻尼系数（取决于哪个更近）。这与Abaqus解释振幅定义的方式一致。

#### 指定全局阻尼

为方便起见，您可以为所有选定的特征模态指定恒定的全局阻尼因子，用于质量和刚度比例粘性因子，以及刚度比例结构阻尼。结构阻尼是一种常用的阻尼模型，将阻尼表示为复刚度。这种表示对于频域分析（如稳态动力学）没有问题，因为解已经是复数的。然而，解必须在时域中保持为实值。为允许用户在时域中应用结构阻尼模型，开发了一种将结构阻尼转换为等效粘性阻尼的方法。只要投影阻尼矩阵是对角的，则在频域中施加的粘性阻尼与结构阻尼相同。更多详情，见["模态动态分析，" 第2.5.5节"](../stm/stm-link.md#stm-anl-modaldynamic)。

| **输入文件用法：** | ``` [*GLOBAL DAMPING](../key/key-link.md#usb-kws-hglobaldamping), ALPHA=*factor*, BETA=*factor*, STRUCTURAL=*factor* ``` |

### 初始条件

默认情况下，模态动态步骤将从零初始位移开始。如果定义了初始速度（["Abaqus/Standard和Abaqus/Explicit中的初始条件，" 第34.2.1节"](pt07ch34s02aus116.md)），则将使用这些值；否则，初始速度为零。

或者，您可以强制模态动态步骤继承紧接前一步的初始条件，该步骤必须是另一个模态动态步骤或静态扰动步骤：
- 在大多数情况下，如果紧接前面的步骤是模态动态步骤，则从该步骤结束处继承位移和速度，并用作当前步骤的初始条件。对于基于SIM的分析，您应该使用次基运动而不是主基运动（见["模态叠加过程中的规定运动"](pt03ch06s03at12.md#usb-anl-amodaldynamic-motion)"）来继承初始条件；如果使用主基运动，Abaqus会发出警告消息。
- 如果紧接前面的步骤是静态扰动步骤，则从该步骤继承位移。如果定义了初始速度（["Abaqus/Standard和Abaqus/Explicit中的初始条件，" 第34.2.1节"](pt07ch34s02aus116.md)），则将使用这些值；否则，初始速度为零。

### 边界条件

在基于模态的动态响应过程中，不能直接将非零位移和旋转（或声压）规定为边界条件（["Abaqus/Standard和Abaqus/Explicit中的边界条件，" 第34.3.1节"](pt07ch34s03aus118.md)）。在这些过程中，只能如上所述将运动规定为基运动。作为边界条件给出的非零位移或加速度历史定义在模态叠加过程中被忽略，并且特征频率提取步骤中支撑条件的任何更改都会被标记为错误。

#### 在模态叠加过程中规定运动

必须在固有频率提取步骤中对将在模态动态过程中规定的自由度施加边界条件。这些自由度被分组为一个或多个"基"（见["固有频率提取，" 第6.3.5节"](pt03ch06s03at10.md)）。未命名的基称为"主"基。命名的"次"基必须通过在频率提取步骤中指定边界条件来定义。可以为每个基规定不同的运动。

##### 指定自由度和运动的时间历史

与基关联的位移和旋转在模态动态响应过程中被规定。基运动完全由最多三个全局平移和三个全局旋转定义。因此，每个平移和旋转分量最多可以定义一个基运动。基运动始终在全局方向上指定，无论节点变换的使用如何。您指定定义基运动的方向（1-6）。如果指定的旋转原点不是坐标原点，则必须指定旋转中心。

运动的时间历史必须由振幅曲线定义（["振幅曲线，" 第34.1.2节"](pt07ch34s01aus115.md)）。

##### 指定次基运动

主基运动通过定义不引用基的基运动来指定。如果基运动要施加到次基，则必须引用在频率提取步骤中定义的基的名称。

对于命名基，您还可以使用自由度8作为声压自由度。声压变化只能与大质量缩放一起用于次基。在这种情况下，声学质量用于缩放大质量值。要指定声压，请使用位移基运动；要指定声压的一阶和二阶导数，请分别使用速度和加速度。

| **输入文件用法：** | ``` [*BASE MOTION](../key/key-link.md#usb-kws-hbasemotion), DOF=*n*, AMPLITUDE=*name*, BASE NAME=*secondary base* ``` |

##### 示例

为说明主基和次基的概念，考虑一个在节点1和4处有支撑的单跨框架。如果频率提取步骤之前的输入包括以下边界条件：
- 在节点1处约束自由度1到6
- 在节点4处约束自由度1
- 在节点4处约束自由度3到6

如果为节点1和4处的自由度2分配不同的基运动，则可以使用以下步骤定义：
- 频率提取步骤包含与`BASE2`关联的边界条件，约束节点4处的自由度2；以及
- 模态动态步骤包含两个基运动定义：分配给不引用基的自由度2的主基运动，以及引用`BASE2`的自由度2的次基运动。

如果未在频率提取步骤之前给出边界条件，则您必须在频率提取步骤中定义它们。同样，次基将由带基名称的边界条件定义。

#### 计算结构的响应

与主基关联的自由度在频率提取步骤中设置为零，主基运动通过将基加速度乘以模态参与因子引入。因此，Abaqus/Standard计算相对于主基的结构响应。如果在主基运动定义中引用了旋转自由度，则除非您提供旋转中心，否则默认围绕坐标原点定义旋转。

与次基关联的自由度在频率提取步骤中不设置为零；相反，向每个自由度添加一个"大"质量。在先前常规步骤中由常规边界条件约束的次基中的任何自由度将被释放，并向该自由度添加大质量。次基运动通过将基加速度（或声压）乘以大质量（或声学大质量）获得的节点力引入。虽然次基运动以绝对项定义，但在平移自由度上计算的响应是相对于主基的运动计算的。旋转次基围绕基名称定义中包含的节点集中的节点定义。因此，您不能更改次基的旋转中心。

有关基运动过程的更详细描述，见["基于模态过程中的基运动，" 第2.5.9节"](../stm/stm-link.md#stm-anl-basemotions)。

### 载荷

可以在模态动态分析中规定以下载荷，如["集中载荷，" 第34.4.2节"](pt07ch34s04aus121.md)中所述：
- 集中节点力可以施加于位移自由度（1-6）。
- 可以施加分布式压力载荷或体力；特定单元可用的分布载荷类型在[第六部分，"单元"](pt06.md)中有描述。

### 预定义场

不允许在瞬态模态动态分析中使用预定义温度场。其他预定义场被忽略。

### 材料选项

必须定义材料的密度（["密度，" 第21.2.1节"](pt05ch21s02abm01.md)）。以下材料特性在模态动态分析期间不活跃：塑性和其他非弹性效应、速率依赖性材料特性、热特性、质量扩散特性、电特性（压电分析中的电势![](../graphics/usb_eqn00092.gif)除外）以及孔隙流体流动特性。见["常规和线性扰动过程，" 第6.1.3节"](pt03ch06s01aus44.md)。

### 单元

除带扭曲的广义轴对称单元外，Abaqus/Standard中的任何应力/位移单元（包括具有温度和压力自由度的单元）都可用在模态动态分析中。

### 输出

Abaqus/Standard中的所有输出变量列在["Abaqus/Standard输出变量标识符，" 第4.2.1节"](pt02ch04s02abv01.md)中。在时域中模态动力学中的节点解变量U、V和A的值是相对于主基的运动给出的。因此，相对运动和主基运动的和给出总运动；可以通过请求输出变量TU、TV和TA获得此总运动。在没有主基运动的情况下，相对和总运动是相同的。

以下模态变量可以输出到数据或结果文件（见["输出到数据和结果文件，" 第4.1.2节"](pt02ch04s01aus39.md)）：

| GU | 所有模态的广义位移。 |
| --- | --- |

| GV | 所有模态的广义速度。 |
| --- | --- |

| GA | 所有模态的广义加速度。 |
| --- | --- |

| SNE | 每个模态整个模型的弹性应变能。 |
| --- | --- |

| KE | 每个模态整个模型的动能。 |
| --- | --- |

| T | 每个模态整个模型的外功。 |
| --- | --- |

| BM | 基运动。 |
| --- | --- |

元素能量密度（如弹性应变能密度SENER）和整个元素能量（如元素的总动能ELKE）在模态动态分析中不可用于输出。但是，整个模型变量（如ALLIE（总应变能））可以作为输出到数据或结果文件（见["输出到数据和结果文件，" 第4.1.2节"](pt02ch04s01aus39.md)）用于基于模态的过程。

可以通过减少请求的输出量来显著降低模态动态分析的计算成本。
