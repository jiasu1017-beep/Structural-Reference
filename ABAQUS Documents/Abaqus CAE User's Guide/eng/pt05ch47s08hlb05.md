# 47.9.5 Customizing the border and fill for the X–Y plot legend







### 47.9.5 Customizing the border and fill for the X--Y plot legend

Use the **Bounding Box** options in the **Chart Legend Options** dialog box to customize the appearance of the legend by wrapping it with a border or filling it with a specified color. 

**To customize the border and fill of the X–Y plot legend:**

1. Locate the **Chart Legend Options**. Select ****Options**![](../graphics/images/arrow.gif)**XY Options**![](../graphics/images/arrow.gif)**Chart Legend****; or click ![](../graphics/ico_xyChartLegendOptions.png), which is located with the *X--Y* plotting tools in the Visualization module toolbox. **Tip:**You can also open this dialog box directly by double-clicking the legend in the viewport when an *X--Y* plot is displayed. You cannot select a component in an *X--Y* plot directly from the viewport when an active procedure is in effect; cancel the current procedure before selecting items from an *X--Y* plot.
2. From the **Charts** field, select one or more *X--Y* plots whose legends you wish to customize. **Note:**To make an *X--Y* curve available for selection, you must first plot it.
3. Click the **Area** tab, if it is not already selected.
4. Perform the following steps to display and modify the bounding box: 1. Toggle on **Show border** to display a border around the legend. 2. Select a color for the bounding box line. 1. Click the color sample ![](../graphics/usi-colorsample.png). Abaqus/CAE displays the **Select Color** dialog box. 2. Use one of the methods in the **Select Color** dialog box to select a new color. For more information, see ["Customizing colors," Section 3.2.9](pt01ch03s02s09.md). 3. Click **OK** to close the **Select Color** dialog box. The bounding box line changes to the selected color.
5. Control the bounding box fill color: 1. Toggle on **Fill** to display a background color in the bounding box. 2. Select a fill color to display in the background of the bounding box. 1. Click the color sample ![](../graphics/usi-colorsample.png). Abaqus/CAE displays the **Select Color** dialog box. 2. Use one of the methods in the **Select Color** dialog box to select a new color. For more information, see ["Customizing colors," Section 3.2.9](pt01ch03s02s09.md). 3. Click **OK** to close the **Select Color** dialog box. The bounding box background changes to the selected color.
6. Click **Dismiss** to close the **Chart Legend Options** dialog box.
