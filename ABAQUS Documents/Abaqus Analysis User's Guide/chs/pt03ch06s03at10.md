### 概述

显式动态分析：
- 对于具有相对较短的动态响应时间和极不连续事件或过程的大型模型分析具有计算效率；
- 允许定义非常一般的接触条件（["接触相互作用分析：概述，" 第36.1.1节](pt09ch36s01abo33.md)）；
- 使用一致的大变形理论——模型可以经历大旋转和大变形；
- 可以使用几何线性变形理论——假定应变和旋转很小（参见["定义分析，" 第6.1.2节](pt03ch06s01abo05.md)）；
- 可用于执行如果预计非弹性耗散会在材料中产生热的绝热应力分析（参见["绝热分析，" 第6.5.4节](pt03ch06s05at20.md)）；
- 可用于具有复杂接触条件的准静态分析；以及
- 允许使用自动或固定时间增量——默认情况下，Abaqus/Explicit使用具有全局时间估计器的自动时间增量。

### 节点质量和惯性

Abaqus/Explicit中的显式积分方案要求所有激活的自由度上存在节点质量或惯性（参见["约定，" 第1.2.2节](pt01ch01s02aus02.md)），除非使用边界条件进行约束。更准确地说，除非所有激活的平移自由度都被约束且必须存在非零转动惯性，否则必须存在非零节点质量；除非所有激活的旋转自由度都被约束，否则必须存在非零 rotary inertia。属于刚体的节点不需要质量，但整个刚体必须具有质量和惯性，除非使用约束。属于欧拉单元的节点也不需要质量，因为周围的欧拉单元在模拟过程中可能在某些时候是空的。

当具有非零质量密度（如实体、壳、梁）的单元激活节点上的自由度时，来自集中质量贡献的集合自然会产生非零节点质量或惯性。

当具有无质量（如弹簧、缓冲器或连接单元）的单元激活节点上的自由度时，必须注意约束节点或适当添加质量和惯性。

### 时间增量

分析中使用的时间增量必须小于中心差分算子的稳定性限制。不使用足够小的时间增量将导致不稳定解。当解变得不稳定时，位移等解变量的时间历史响应通常会以增大的幅度振荡。总体能量平衡也将显著变化。

如果模型只包含一种材料类型，初始时间增量与网格中最小单元的尺寸成正比。如果网格包含均匀尺寸单元但包含多种材料描述，则具有最高波速的单元将决定初始时间增量。

在非线性问题——具有大变形和/或非线性材料响应——中，模型的最高频率将不断变化，从而改变稳定性限制。Abaqus/Explicit有两种时间增量控制策略：全自动时间增量（代码考虑稳定性限制的变化）和固定时间增量。

#### 缩放时间增量

为减少解变得不稳定的可能性，您可以按常数缩放因子调整Abaqus/Explicit计算的稳定时间增量。此因子可用于缩放默认全局时间估计、逐单元估计或基于初始逐单元估计的固定时间增量；它不能用于缩放您直接指定的固定时间增量。

#### 自动时间增量

Abaqus/Explicit中的默认时间增量方案是全自动的，不需要用户干预。使用两种类型的估计来确定稳定性限制：逐单元和全局。分析始终从使用逐单元估计方法开始，在某些情况下可能会切换到全局估计方法，如下所述。

##### 逐单元估计

在分析中，Abaqus/Explicit最初使用基于整个模型中最高单元频率的稳定性限制。逐单元估计使用每个单元中的当前膨胀波速度确定。

逐单元估计是保守的；它将给出比基于整个模型最大频率的真实稳定性限制更小的稳定时间增量。通常，边界条件和运动学接触等约束具有压缩特征值谱的效果，而逐单元估计不考虑这一点。

将稳定时间增量视为膨胀波穿过最小单元尺寸所需时间的概念有助于解释显式过程在逐单元稳定性估计控制时间增量时如何选择时间增量。随着步骤进行，全局稳定性估计（如果使用）将使时间增量对单元尺寸不太敏感。

| **输入文件用法：** | ``` [*DYNAMIC](../key/key-link.md#usb-kws-hdynamic), EXPLICIT, ELEMENT BY ELEMENT ``` |
| --- | --- |

| **Abaqus/CAE用法：** | 步骤模块：**创建步骤**：**常规**：**动态，显式**：**增量**：**稳定增量估计器**：**逐单元** |
| --- | --- |

##### 全局估计

随着步骤进行，除非指定逐单元估计方法、指定固定时间增量或阻止使用全局估计的以下条件之一，否则稳定性限制将由全局估计器确定。一旦算法确定全局估计方法的精度可接受，就会切换到全局估计方法。

自适应全局估计算法使用当前膨胀波速度确定整个模型的最大频率。该算法持续更新最大频率的估计。全局估计器通常允许超过逐单元值的时间增量。

Abaqus/Explicit监视全局估计算法的有效性。如果计算全局时间估计的成本超过其收益，代码将关闭全局估计算法，仅使用逐单元估计来节省计算时间。

##### 阻止使用全局时间估计器的条件

当模型中包含以下任何功能时，将不使用全局估计算法：
- 流体单元
- 无限单元
- 缓冲器
- 厚壳（厚度与特征长度比大于0.92）
- 厚梁（厚度与长度比大于1.0）
- JWL状态方程
- 材料阻尼
- 具有温度和场变量依赖性的非各向同性弹性材料
- 畸变控制
- 自适应网格
- 子循环

##### 三维连续单元和平面应力 formulation 单元的"改进"稳定时间增量

### 显式方法的优点

使用小增量（由稳定性限制决定）是有利的，因为它允许解进行而无需迭代，也不需要形成切线刚度矩阵。它还简化了接触处理。

显式动力学过程非常适合分析高速动态事件，但显式过程的许多优点也适用于较慢（准静态）过程的分析。一个很好的例子是金属板成型，其中接触主导解，局部不稳定性可能由于板的起皱而形成。

显式动力学分析中的结果不会像Abaqus/Standard那样自动检查准确性（Abaqus/Standard使用半增量残差）。在大多数情况下，这不是问题，因为稳定性条件施加了如此小的时间增量，以至于解在任何时间增量中仅轻微变化，从而简化了增量计算。虽然分析可能需要非常大量的增量，但每个增量相对便宜，通常会产生经济的解。Abaqus/Explicit进行超过105次增量的分析并不罕见。因此，该方法在计算上对于总动态响应时间仅比稳定性限制长一个数量级的问题具有吸引力；例如，波传播研究或一些"事件和响应"应用。

在准静态分析中，通过加速模拟或缩放质量来降低计算成本是方便的。在任何一种情况下，都应监测动能以确保动能与内能的比值不会太大——通常小于10%。

#### 通过加速模拟降低计算成本

为减少所需的增量数*n*，我们可以将模拟加速到比实际过程更短的时间——即，我们可以人为地减少事件的时间周期*T*。这将引入两种可能的错误。如果模拟加速太多，增加的惯性力将改变预测响应（在极端情况下问题将表现出波传播响应）。避免此错误的唯一方法是选择不要太大的加速。

另一个错误是，问题的某些方面（如材料行为）可能也是速率相关的。在这种情况下，模拟的事件的实际时间周期无法更改。

#### 通过使用质量缩放降低计算成本

通过因子 ![](../graphics/usb_eqn00632.gif) 人为增加材料密度 ![](../graphics/usb_eqn00347.gif)，将*n*降至 ![](../graphics/usb_eqn00633.gif)，就像将*T*减小到 ![](../graphics/usb_eqn00634.gif) 一样。这个称为"质量缩放"的概念在保持事件时间固定的同时减少了事件时间与跨单元波传播时间之比，这允许在分析中包括速率依赖行为。质量缩放对惯性力的影响与加速模拟时间完全相同。

质量缩放很有吸引力，因为它可以用于速率依赖问题，但必须小心使用以确保惯性力不会主导并改变解。可以调用固定或可变质量缩放（参见["质量缩放，" 第11.6.1节](pt04ch11s06aus74.md)）。

质量缩放也可以通过改变密度来实现；但是，固定和可变质量缩放功能提供了更通用的方法来缩放整个模型或模型中特定单元集的质量。

#### 通过使用选择性子循环降低计算成本

显式动态分析中的一个缺点是，少数非常小的单元将强制整个模型以小时间增量进行积分。您可以使用混合时间积分或"子循环"方法来减少此问题。在这些方法中，物体的运动方程仍然使用上述显式中心差分积分规则进行积分，但允许有限元模型中不同节点组使用不同的时间增量。如果大多数节点以较大的稳定时间增量积分，而只有少数节点以小时间增量积分，则计算成本可显著降低。

可以通过定义子循环区域来调用选择性子循环。详细信息请参见["选择性子循环，" 第11.7.1节](pt04ch11s07aus75.md)。

### 监测输出变量的极值

在某些分析中，监测每个增量中变量的值很有用。例如，在力驱动分析（如液压成型）中，足以模拟物理过程完成所需的模拟时间可能取决于模型中节点或节点组的位移大小。另一个例子是跌落测试模拟，其中不关心失效后的响应。监测关键变量的值并在这些变量超过给定准则时停止分析可以减少计算费用和周转时间。

对于此类问题，Abaqus/Explicit允许在分析期间监测输出变量，以验证它们是否超过或低于指定元素或节点集中用户指定的值。每增量执行指定单元积分点变量、单元截面变量或节点变量与用户指定值的比较。当变量首次超过用户指定的边界时，变量名、相关单元或节点编号以及增量号写入状态（`.sta`）文件。此外，您可以请求在变量超过用户指定边界的增量之后的增量中停止分析和/或写入输出状态。在每个监测变量的步骤结束时，将写出分析过程中每个变量达到的最大值、最小值或绝对最大值，以及发生极值元素或节点的编号。

- 可以指定用户定义场变量的值。这些值仅影响场变量依赖材料属性（如果有）。

### 材料选项

Abaqus/Explicit中的任何材料模型都可用于一般显式动态分析（参见["组合材料行为，" 第21.1.3节](pt05ch21s01aus110.md)）。

### 单元

Abaqus/Explicit中可用的所有单元都可用于显式动态分析。单元列在[第六部分，"单元"](pt06.md)中。

如果在显式动态分析中使用耦合温度-位移单元，温度自由度将被忽略。

### 输出

动态分析可用的单元输出包括应力、应变、能量以及状态、场和用户定义变量的值。可用的节点输出包括位移、速度、加速度、反力和坐标。所有输出变量标识符在["Abaqus/Explicit输出变量标识符，" 第4.2.2节](pt02ch04s02xbv01.md)中列出。输出类型在["输出，" 第4.1.1节](pt02ch04s01aus38.md)中描述。

当Abaqus/Explicit分析遇到致命错误时，预选的适用于当前过程的变量将自动作为最后一个增量的场数据添加到输出数据库中。

在显式动态分析中，检查解准确性的能量输出尤其重要。通常，总能量（ETOTAL）应该是常数或接近常数；"人工"能量，如人工应变能（ALLAE）、阻尼耗散（ALLVD）和质量缩放功（ALLMW），应与"真实"能量（如应变能（ALLSE）和动能（ALLKE））相比可以忽略不计。

在准静态分析中，动能（ALLKE）的值不应超过应变能（ALLIE）值的一小部分。

在涉及约束（如tie和紧固件）和接触的分析中，输出约束 penalty 功（ALLCW）和接触 penalty 功（ALLPW）是很好的做法。这些能量的值应该接近零。
