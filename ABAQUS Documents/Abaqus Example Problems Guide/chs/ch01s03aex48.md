# 1.3.17 不稳定静态问题：金属板的热成形





**产品：**Abaqus/Standard   

本示例演示了使用自动接触稳定来避免不稳定静态问题的方法。几何非线性静态问题可能因多种原因变得不稳定。接触问题可能发生不稳定，可能是因为颤振，或者是因为旨在防止刚体运动的接触最初没有建立。局部不稳定也可能发生；它们可以是几何的（如局部屈曲），或者是材料的（如材料软化）。

本问题模拟金属板的热成形；模具的形状可能使难以将未变形的板精确放置在初始接触位置，在这种情况下，初始刚体运动防止算法是有用的。金属成形问题的特征是相对简单形状的零件被相对复杂形状的模具变形。零件在模具上的初始放置或第二个模具的初始放置对于工程师来说可能是建模成形过程的简单几何练习。Abaqus 接受接触对中的初始穿透，并立即尝试解决它们；只要几何形状允许这样做而不会产生过度变形，零件的错位通常不会造成问题。另一方面，如果零件最初放置在远离模具的位置，则可能出现严重的收敛问题。除非有足够的边界条件应用或使用了稳定方法，否则会产生奇异的有限元方程组，因为一个或多个物体有自由刚体运动。当变形通过载荷而不是边界条件施加时，这通常会发生。接触稳定有助于在建立接触时避免收敛问题，而不会显著影响感兴趣的结果（参见["接触问题中刚体运动的自动稳定"在"Abaqus/Standard 中的调整接触控制"第 36.3.6 节](../usb/usb-link.md#usb-cni-acontacttrouble-stabilize)）。

本示例研究铝板的热成形。通过施加压力和重力载荷使板推向模具而产生变形。变形最初是弹性的。通过加热，材料的屈服应力降低，直到产生永久塑性变形。随后，组装件被冷却，压力载荷被移除，留下具有一些回弹的成形零件。尽管板最初是平的，但模具的几何性质使得难以确定板放置在模具上时的确切位置。因此，在两个物体之间建模了初始间隙，如图 [1.3.17--1](ch01s03aex48.md#sxmstable-thermmesh) 所示。

### 几何结构和模型

模型由一个梯形板组成，长 10.0 m（394.0 in），宽度从 2.0 m（78.75 in）到 3.0 m（118.0 in）逐渐变化，厚度为 10.0 mm（0.4 in）。模具是由半径为 13.0 m（517.0 in）和 6.0 m（242.0 in）的两个圆控制的直纹面，尺寸略大于板。板最初放置在距模具 0.2 m（7.9 in）处。板具有纵向对称边界条件，一个节点防止剩余节点发生面内刚体运动。模具在整个分析过程中是固定的。板网格由 640 个 S4R 壳单元组成，而模具由 640 个 R3D4 刚性单元表示。材料是铝镁合金，在室温下的流动应力为 1.0 × 10⁸ Pa（14.5 ksi）。在 400°C 时也提供了 1.0 × 10³ Pa（0.15 psi）的流动应力，基本上说明在较高温度下材料在任何应力下都会发生塑性流动。使用 0.1 的库仑摩擦系数来模拟板和模具之间的相互作用。

### 结果与讨论

分析包括三个步骤。在第一个步骤中，施加重力载荷和 1.0 × 10⁵ Pa（14.5 psi）的压力载荷，两者都将板推向模具。这一步骤得益于接触稳定，以防止板在与模具建立接触之前出现无约束运动。在这种情况下，垂直于附近接触表面的接触稳定提供了充分的稳定。如果可能，建议避免切向接触稳定，因为切向接触稳定更可能影响求解变量。接触稳定的有效间隙距离范围已在本示例中指定，使得接触稳定在板的初始分离距离时生效。Abaqus 逐渐减小接触稳定，使得在第一步结束时不再有接触稳定。这保证了黏 l力减小到零，从而避免在下一步开始时力出现不连续。模具的形状和相对较低的曲率使得该步骤结束时的变形是弹性的（图 [1.3.17--2](ch01s03aex48.md#sxmstable-thermpress)）。在第二个步骤中，对加载的组装件施加两小时的加热（从室温到 360°C）和冷却（回到 50°C）循环。由于流动应力的减小，产生永久（塑性）变形，如图 [1.3.17--3](ch01s03aex48.md#sxmstable-thermdeform) 所示。最后，在第三个步骤中，压力载荷被移除，计算变形板的回弹，如图 [1.3.17--4](ch01s03aex48.md#sxmstable-thermsprngbk) 所示。
