# 12.12.4 定义垫片行为





您可以使用**Edit Material**对话框来定义垫片行为。有关详细信息，请参阅以下章节：
- ["定义厚度方向的垫片行为"](pt03ch12s12s04.md#usi-prp-other-gasket-thickness)

- ["定义垫片的弹性横向剪切行为"](pt03ch12s12s04.md#usi-prp-other-gasket-transverseshear)
- ["定义垫片的膜行为"](pt03ch12s12s04.md#usi-prp-other-gasket-membraneelastic)

### 定义厚度方向的垫片行为

Abaqus/Standard将厚度方向变形测量为垫片元素底面和顶面之间的闭合；因此，厚度方向行为必须始终以闭合来定义。在所有情况下，您都可以将厚度方向行为定义为温度和/或场变量的函数。有关更多信息，请参阅以下章节：
- ["定义垫片行为的直接使用垫片行为模型"中的"定义垫片的厚度方向行为"，第32.6.6节Abaqus Analysis User's Guide](../usb/usb-link.md#usb-elm-egasketbehavior-thickness)
- ["定义垫片行为的直接使用垫片行为模型"，第32.6.6节Abaqus Analysis User's Guide](../usb/usb-link.md#usb-elm-egasketbehavior)

**定义厚度方向的垫片行为：**

1. 从**Edit Material**对话框的菜单栏中，选择****Other**![](../graphics/images/arrow.gif)**Gasket**![](../graphics/images/arrow.gif)**Gasket Thickness Behavior****。（有关显示**Edit Material**对话框的信息，请参见["创建或编辑材料，"第12.7.1节"](pt03ch12s07hlb01.md)。）
2. 点击**Type**字段右侧的箭头，并指定您希望如何定义垫片厚度方向行为：
   - 选择**Elastic-Plastic**以定义弹塑性模型。有关更多信息，请参见"定义垫片行为的直接使用垫片行为模型"中的"定义非线性弹塑性模型"，第32.6.6节Abaqus Analysis User's Guide。
   - 选择**Damage**以定义损伤弹性模型。有关更多信息，请参见"定义垫片行为的直接使用垫片行为模型"中的"定义带损伤的非线性弹性模型"，第32.6.6节Abaqus Analysis User's Guide。
3. 点击**Units**字段右侧的箭头，并指定用于定义厚度方向行为的单位系统：
   - 选择**Stress**以压力对闭合的形式定义厚度方向行为。此选项适用于所有垫片元素类型。
   - 选择**Force**以力对闭合或单位长度力对闭合的形式定义厚度方向行为，具体取决于此行为所使用的元素类型。此选项仅对链接元素和三维线元素有效。如果您选择此选项，可以从**Suboptions**菜单中选择**Contact Area**来定义接触面积或接触宽度对闭合曲线，以通过变量CS11输出平均压力。有关详细说明，请参见["为平均压力输出指定垫片接触面积或接触宽度"](pt03ch12s12s04.md#usi-prp-other-gasket-contactarea)。有关选择单位系统的更多信息，请参见"定义垫片行为的直接使用垫片行为模型"中的"选择用于定义厚度方向行为的单位系统"，第32.6.6节Abaqus Analysis User's Guide。
4. 显示**Loading**标签页。
5. 点击**Yield onset method **字段右侧的箭头，并选择定义屈服开始的方法：
   - 选择**Relative slope drop**将屈服开始定义为加载曲线斜率从到该点为止记录的最大斜率下降一定百分比的值。在提供的字段中输入相对下降值。默认值为0.1（或10%）。
   - 选择**Closure value**以指定发生屈服的闭合值。在提供的字段中输入闭合值。
6. 在**Tensile stiffness factor**字段中，输入定义拉伸刚度的初始压缩刚度的分数。默认值为0.001。有关更多信息，请参见"定义垫片行为的直接使用垫片行为模型"中的"厚度方向行为的数值稳定性"，第32.6.6节Abaqus Analysis User's Guide。
7. 切换**Use temperature-dependent data**将垫片厚度行为定义为温度的函数。**Data**表格中出现一个标为**Temp**的列。
8. 点击**Number of field variables**字段右侧的箭头，以增加或减少垫片厚度行为所依赖的场变量数量。
9. 在**Data**表格中，定义压力对闭合，力对闭合或单位长度力对闭合的加载。如适用，输入温度和场变量的值。有关如何输入数据的详细信息，请参见["输入表格数据，"第3.2.7节"](pt01ch03s02s07.md)。
10. 显示**Unloading**标签页，并根据需要切换**Include user-specified unloading curves**。用户指定的卸载曲线是默认卸载曲线的补充，默认卸载曲线是屈服开始点之前加载曲线的缩放部分。如果将此选项保持关闭，请跳至第13步。
11. 切换**Use temperature-dependent data**将卸载曲线定义为温度的函数。**Data**表格中出现一个标为**Temp**的列。
12. 点击**Number of field variables**字段右侧的箭头，以增加或减少卸载曲线所依赖的场变量数量。
13. 在**Data**表格中，指定卸载曲线：
    - 如果在第2步中选择了**Damage**，请提供压力（或力或单位长度力）对给定最大闭合范围内的弹性闭合的数据点。如适用，输入温度和场变量的值。请参见"定义垫片行为的直接使用垫片行为模型"中的"定义带损伤的非线性弹性模型"，第32.6.6节Abaqus Analysis User's Guide。
    - 如果在第2步中选择了**Elastic-Plastic**，请为每个给定塑性闭合按闭合升序提供压力（或力或单位长度力）对闭合（弹性加塑性）的数据点。请参见"定义垫片行为的直接使用垫片行为模型"中的"定义非线性弹塑性模型"，第32.6.6节Abaqus Analysis User's Guide。
    有关如何输入数据的详细信息，请参见["输入表格数据，"第3.2.7节"](pt01ch03s02s07.md)。
14. 点击**OK**关闭**Edit Material**对话框。或者，您可以从**Edit Material**对话框的菜单中选择另一种材料行为进行定义（有关更多信息，请参见["浏览和修改材料行为，"第12.7.2节"](pt03ch12s07hlb02.md)）。

### 为平均压力输出指定垫片接触面积或接触宽度

当您以力或单位长度力对闭合的形式定义垫片的厚度方向行为时，Abaqus/Standard将厚度方向力或单位长度力作为输出变量S11提供。在这种情况下，您可以定义接触宽度或接触面积对闭合曲线，该曲线将用于获得每个积分点处的平均"接触"压力作为输出变量CS11。

有关更多信息，请参见"定义垫片行为的直接使用垫片行为模型"中的"定义平均接触压力输出的接触面积"，第32.6.6节Abaqus Analysis User's Guide。

**指定接触面积或宽度：**

1. 如["定义厚度方向的垫片行为"](pt03ch12s12s04.md#usi-prp-other-gasket-thickness)中所述定义垫片厚度方向行为。
2. 从**Edit Material**对话框的**Suboptions**菜单中，选择**Contact Area**。出现**Suboption Editor**。
3. 切换**Use temperature-dependent data**将数据定义为温度的函数。**Data**表格中出现一个标为**Temp**的列。
4. 点击**Number of field variables**字段右侧的箭头，以增加或减少数据所依赖的场变量数量。
5. 在**Data**表格中，定义接触面积或接触宽度对闭合曲线。如适用，包含温度和场变量数据。有关如何输入数据的详细信息，请参见["输入表格数据，"第3.2.7节"](pt01ch03s02s07.md)。
6. 点击**OK**返回**Edit Material**对话框。

### 定义垫片的弹性横向剪切行为

Abaqus/Standard测量垫片元素底部和顶部之间沿局部2或3方向的相对位移来定义垫片中的横向剪切。您可以使用**Edit Material**对话框将弹性横向刚度指定为每单位位移的应力（或力，或单位长度力）。请参见"定义垫片行为的直接使用垫片行为模型"中的"定义垫片的横向剪切行为"，第32.6.6节Abaqus Analysis User's Guide。

**定义垫片横向剪切行为：**

1. 从**Edit Material**对话框的菜单栏中，选择****Other**![](../graphics/images/arrow.gif)**Gasket**![](../graphics/images/arrow.gif)**Gasket Transverse Shear Elastic****。（有关显示**Edit Material**对话框的信息，请参见["创建或编辑材料，"第12.7.1节"](pt03ch12s07hlb01.md)。）
2. 点击**Units**字段右侧的箭头，并选择您将定义横向剪切行为的单位系统：
   - 选择**Stress**以每单位位移的应力形式定义横向剪切刚度。
   - 选择**Force**以每单位位移力或每单位长度力每单位位移的形式定义横向剪切刚度，具体取决于此行为所指向的元素类型。您为横向剪切行为选择的单位系统必须与为厚度方向行为选择的单位系统一致（请参见["定义厚度方向的垫片行为"](pt03ch12s12s04.md#usi-prp-other-gasket-thickness)"）。
   有关更多信息，请参见"定义垫片行为的直接使用垫片行为模型"中的"选择用于定义横向剪切行为的单位系统"，第32.6.6节Abaqus Analysis User's Guide。
3. 切换**Use temperature-dependent data**将剪切刚度定义为温度的函数。**Data**表格中出现一个标为**Temp**的列。
4. 点击**Number of field variables**字段右侧的箭头，以增加或减少剪切刚度所依赖的场变量数量。
5. 在**Data**表格中，指定剪切刚度。如适用，包含温度和场变量数据。有关如何输入数据的详细信息，请参见["输入表格数据，"第3.2.7节"](pt01ch03s02s07.md)。
6. 点击**OK**关闭**Edit Material**对话框。或者，您可以从**Edit Material**对话框的菜单中选择另一种材料行为进行定义（有关更多信息，请参见["浏览和修改材料行为，"第12.7.2节"](pt03ch12s07hlb02.md)）。

### 定义垫片的膜行为

您可以通过提供杨氏模量和泊松比来定义垫片的线弹性行为。这些数据可以是温度和/或场变量的函数。如果您不指定垫片的线弹性行为，则垫片没有膜刚度。在这种情况下，您必须确保元素节点在垂直于垫片厚度方向的方向上被充分约束。

**定义垫片膜行为：**

1. 从**Edit Material**对话框的菜单栏中，选择****Other**![](../graphics/images/arrow.gif)**Gasket**![](../graphics/images/arrow.gif)**Gasket Membrane Elastic****。（有关显示**Edit Material**对话框的信息，请参见["创建或编辑材料，"第12.7.1节"](pt03ch12s07hlb01.md)。）
2. 切换**Use temperature-dependent data**将数据定义为温度的函数。**Data**表格中出现一个标为**Temp**的列。
3. 点击**Number of field variables**字段右侧的箭头，以增加或减少数据所依赖的场变量数量。
4. 在**Data**表格中，输入杨氏模量和泊松比数据。如适用，包含温度和场变量数据。有关如何输入数据的详细信息，请参见["输入表格数据，"第3.2.7节"](pt01ch03s02s07.md)。
5. 点击**OK**关闭**Edit Material**对话框。或者，您可以从**Edit Material**对话框的菜单中选择另一种材料行为进行定义（有关更多信息，请参见["浏览和修改材料行为，"第12.7.2节"](pt03ch12s07hlb02.md)）。
