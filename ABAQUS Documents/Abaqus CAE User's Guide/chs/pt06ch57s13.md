# 57.13 定义 PSD 定义







使用 PSD 定义方法定义随机响应分析步骤中随机载荷的频率相关性。有关更多信息，请参见 Abaqus Analysis User's Guide](../usb/usb-link.md#usb-anl-arandomresponse-psdfunction) 的"随机响应分析"第 6.3.11 节中的"定义频率函数"。

**定义 PSD 定义：**

1. 如 ["选择要定义的振幅类型，" 第 57.3 节](pt06ch57s03.md) 中所述显示 **编辑振幅** 对话框。
2. 点击 **规格单位** 字段旁边的箭头，并指定您希望如何定义曲线的单位。- 选择 **功率** 直接以功率单位定义频率函数。- 选择 **分贝** 以分贝单位定义频率函数。- 选择 **重力（基础运动）** 如果频率函数将用于以 *g* 单位定义基础运动。如果您选择这些单位，则必须定义重力加速度。
3. 如果您选择了 **分贝** 单位，请为 **参考功率** 输入一个值。
4. 如果您选择了 **重力** 单位，请为 **参考重力** 输入一个值。
5. 在数据表中，输入或导入函数的数据值：- 函数的 **实部** 和 **虚部**，以分贝或单位2/频率表示。- **频率**，以周期/时间表示，或频率带号（用于 **分贝** 单位）。有关如何输入数据的详细信息，请参见 ["输入表格数据，" 第 3.2.7 节](pt01ch03s02s07.md)
6. 如果您将通过用户子程序 [`UPSD`](../sub/sub-link.md#sub-xsl-upsd) 提供 PSD 函数，请切换 **在外部用户子程序中指定数据**。
7. 点击 **确定** 保存您的振幅定义并关闭 **编辑振幅** 对话框。
