# 53.2.1 铺层堆叠图选项概述





["铺层堆叠图概述," 第53.1节](pt05ch53hla01.md) 描述了如何创建铺层堆叠图。显示铺层堆叠图后,您可以通过点击提示区域中的"**铺层堆叠图选项**"按钮来自定义其外观。或者,您可以执行以下操作:
- 在"属性"模块中,选择 ****View**![](../graphics/images/arrow.gif)**Ply Stack Plot Options****。
- 在"可视化"模块中,选择 ****Options**![](../graphics/images/arrow.gif)**Ply Stack Plot**** 或点击工具箱中的 ![](../graphics/ico_plotPlyStackOptions.png) 工具。

Abaqus/CAE显示"铺层堆叠图选项"对话框,它允许您执行以下操作:- 选择在铺层堆叠图中显示哪些铺层以及如何显示铺层。有关更多信息,请参阅 ["自定义基本铺层堆叠图选项," 第53.2.2节](pt05ch53s01hlb02.md)。
- 选择是否在铺层堆叠图中显示纤维以及纤维的外观。有关更多信息,请参阅 ["自定义铺层堆叠图中纤维的外观," 第53.2.3节](pt05ch53s01hlb03.md)。
- 选择是否在铺层堆叠图中显示参考面以及参考面的外观。有关更多信息,请参阅 ["自定义铺层堆叠图中的参考面," 第53.2.4节](pt05ch53s01hlb04.md)。
- 自定义铺层堆叠图中显示的文本和符号有关更多信息,请参阅 ["自定义铺层堆叠图中的标签," 第53.2.5节](pt05ch53s01hlb05.md)。
