# 42.6.10 控制结果缓存

您可以在后处理期间选择将分析结果存储在缓存中，以加快屏幕上图像的生成。但是，缓存结果会消耗系统内存，并可能降低整体性能。切换到新的输出变量或输出数据库将自动清除当前结果缓存。

为了提高性能，您可以减少Abaqus/CAE监控输出数据库更新的频率，也可以完全禁用监控。如果您正在从远程输出数据库显示数据，如果通过网络的数据库监控时间很长，Abaqus/CAE的性能可能会下降。有关的详细信息，请参阅["在远程计算机上访问输出数据库，" 第9.3节](pt02ch09s03.md)。

**控制结果缓存：**

1. 找到"Results Caching"选项。从主菜单栏中，选择****Result**![](../graphics/images/arrow.gif)**Options****；然后在出现的对话框中单击**Caching**标签。
2. 使用"Caching Options"字段来加快屏幕上图像的生成。
   1. 切换打开**缓存变形变量结果**以将当前输出数据库的变形变量结果存储在内存中。
   2. 切换打开**缓存主变量结果**以将当前输出数据库的主变量结果存储在内存中。
   3. 切换打开**缓存切割场**以将用于显示切割模型的值存储在内存中。
3. 使用"ODB Access Options"字段在访问输出数据库时提高性能。
   1. 切换关闭**检查ODB文件更新**以停止Abaqus/CAE监控输出数据库的更新。
   2. 输入检查输出数据库更新的最小时间间隔（秒）。较大的数字将减少Abaqus/CAE检查输出数据库更新的频率。
4. 默认情况下，您的更改会在会话期间保存，并会影响后续的所有结果显示。如果您希望将更改保留到后续会话中，请将其保存到文件。有关的详细信息，请参阅["保存自定义设置以供后续会话使用，" 第55.1.1节](pt05ch55s01s01.md)。
