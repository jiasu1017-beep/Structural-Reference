### 概述

经典金属塑性模型：
- 使用Mises或Hill屈服面及相关塑性流动，分别允许各向同性和各向异性屈服；
- 使用理想塑性或各向同性硬化行为；
- 可在率相关效应重要时使用；
- 适用于碰撞分析、金属成形和一般坍塌研究等应用（包含运动硬化的塑性模型因此更适合于涉及循环加载的情况，Abaqus中也有提供：见["承受循环加载的金属模型，" 第23.2.2节"](pt05ch23s02abm18.md)。）；
- 可用于任何使用具有位移自由度的单元的过程；
- 可用于完全耦合温度-位移分析（["完全耦合热应力分析，" 第6.5.3节"](pt03ch06s05at19.md)）、完全耦合热-电-结构分析（["完全耦合热-电-结构分析，" 第6.7.4节"](pt03ch06s07at23.md)）或绝热热应力分析（["绝热分析，" 第6.5.4节"](pt03ch06s05at20.md)），使得塑性耗散导致材料加热；
- 可与Abaqus中的逐步损伤和失效模型结合使用（["延性金属的损伤和失效：概述，" 第24.2.1节"](pt05ch24s02abm41.md)）以指定不同的损伤起始准则和损伤演化定律，允许材料刚度的逐步降解以及从网格中删除单元；
- 可与Abaqus/Explicit中的剪切失效模型结合使用，以提供简单的延性动态失效准则，允许从网格中删除单元，尽管通常建议使用上面讨论的逐步损伤和失效方法；
- 可与Abaqus/Explicit中的拉伸失效模型结合使用，以提供拉伸层裂准则，提供多种失效选择并从网格中删除单元；和
- 必须与线性弹性材料模型（["线性弹性行为，" 第22.2.1节"](pt05ch22s02abm02.md)）或状态方程材料模型（["状态方程，" 第25.2.1节"](pt05ch25s02abm50.md)）结合使用。

### 屈服面

Mises和Hill屈服面假定金属的屈服与等效压力应力无关：这一观察在正压力应力下对大多数金属（多孔金属除外）得到实验证实，但在高三轴拉伸条件下可能不准确，此时材料中可能萌生和生长孔洞。这种情况可能出现在裂纹尖端附近的应力场和一些极端热加载情况下，如焊接过程中可能发生的情况。Abaqus为这种情况提供了多孔金属塑性模型。该模型在["多孔金属塑性，" 第23.2.9节"](pt05ch23s02abm25.md)中描述。

#### Mises屈服面

Mises屈服面用于定义各向同性屈服。它通过给出单轴等效塑性应变、温度和/或场变量的函数形式的单轴屈服应力值来定义。在Abaqus/Standard中，屈服应力也可以在用户子程序[`UHARD`](../sub/sub-link.md#sub-xsl-uhard)中定义。

### 硬化

在Abaqus中，可以定义理想塑性材料（无硬化），也可以指定加工硬化。各向同性硬化（包括Johnson-Cook硬化）在Abaqus/Standard和Abaqus/Explicit中均可用。此外，Abaqus还为承受循环加载的材料提供运动硬化。

#### 理想塑性

理想塑性意味着屈服应力不随塑性应变而变化。可以为一系列温度和/或场变量以表格形式定义；每个温度和/或场变量的单个屈服应力值指定屈服的开始。

#### 各向同性硬化

各向同性硬化意味着屈服面以均匀的大小改变在所有方向上，使得当塑性应变发生时，屈服应力在所有应力方向上增加（或减少）。Abaqus提供了一个各向同性硬化模型，适用于涉及整体塑性应变的情况，或者在分析中每个点的应变基本上在应变空间中沿相同方向进行的情况。尽管该模型被称为"硬化"模型，但可以定义应变软化或先硬化后软化。各向同性硬化塑性在["各向同性弹塑性，" Abaqus理论指南第4.3.2节](../stm/stm-link.md#stm-mat-isoelastoplast)中更详细地讨论。

如果定义了各向同性硬化，屈服应力σ~0~可以作为塑性应变以及温度和/或其他预定义场变量的表格函数给出。给定状态下的屈服应力简单地从此数据表插值获得，并且对于超过表格数据最后给出的塑性应变保持恒定。

Abaqus/Explicit将把数据正则化为以自变量偶数间隔定义的表。在某些情况下，当屈服应力以自变量（塑性应变）的不均匀间隔定义且自变量范围相对于最小间隔较大时，Abaqus/Explicit可能无法在合理数量的间隔中获得数据的准确正则化。在这种情况下，程序将在处理所有数据后停止，并显示必须重新定义材料数据的错误消息。有关数据正则化的更详细讨论，请参见["材料数据定义，" 第21.1.2节"](pt05ch21s01aus109.md)。

### 流动法则

Abaqus使用相关塑性流动。因此，当材料屈服时，非弹性变形率在屈服面法线的方向上（塑性变形体积不变）。这一假设对于大多数金属计算通常是可以接受的；最明显不适用的案例是对金属板中塑性流动局部化的详细研究，因为板材会产生织构并最终撕裂。只要这些效应的细节不令人感兴趣（或可以从不太详细的准则中推断出来，例如达到以应变定义的成形极限），Abaqus中与光滑Mises或Hill屈服面一起使用的相关流动模型通常可以准确预测行为。

### 率相关性

随着应变率的增加，许多材料表现出屈服强度的增加。当应变率在每秒0.1到1之间时，这一效应对许多金属变得重要；对于每秒10到100之间的应变率，这可能非常重要，这是高能动态事件或制造过程的特征。

有多种方法可以引入应变率相关的屈服应力。

#### 直接表格数据

可以将测试数据作为不同等效塑性应变率（ε¯~pl~）下屈服应力值与等效塑性应变的表格提供；每个应变率一个表。直接表格数据不能与Johnson-Cook硬化一起使用。管理此数据输入的指南在["率相关屈服，" 第23.2.3节"](pt05ch23s02abm19.md)中提供。

| **输入文件用法：** | ``` [*PLASTIC](../key/key-link.md#usb-kws-mplastic), RATE=ε¯~pl~ ``` |
| --- | --- |

| **Abaqus/CAE用法：** | 属性模块：材料编辑器：****机械**![](../graphics/images/arrow.gif)**塑性**![](../graphics/images/arrow.gif)**塑性****：****使用应变率相关数据** |
| --- | --- |

#### 屈服应力比

或者，您可以通过缩放函数指定应变率相关性。在这种情况下，您只输入一条硬化曲线，即静态硬化曲线，然后将静态关系表示为率相关硬化曲线；即，我们假设：

σ~y~ = σ~s~ × R(ε¯~pl~, ε¯~pl~)

其中σ~s~是静态屈服应力，ε¯~pl~是等效塑性应变，ε¯~pl~是等效塑性应变率，R是一个比值，定义为ε¯~pl~ = ε¯~pl~0~时的R(ε¯~pl~, ε¯~pl~)。此方法在["率相关屈服，" 第23.2.3节"](pt05ch23s02abm19.md)中进一步描述。

### 逐步损伤和失效

在Abaqus中，金属塑性材料模型可以与["延性金属的损伤和失效：概述，" 第24.2.1节"](pt05ch24s02abm41.md)中讨论的逐步损伤和失效模型结合使用。该功能允许指定一个或多个损伤起始准则，包括延性、剪切、成形极限图（FLD）、成形极限应力图（FLSD）、Mschenborn-Sonne成形极限图（MSFLD），以及在Abaqus/Explicit中的Marciniak-Kuczynski（M-K）准则。损伤起始后，材料刚度根据指定的损伤演化响应逐步降解。该模型提供两种失效选择，包括由于结构撕裂或扯裂而从网格中删除单元。逐步损伤模型允许材料刚度的平滑降解，使它们适用于准静态和动态情况。这与接下来讨论的动态失效模型相比是一个很大的优势。

### Abaqus/Explicit中的剪切和拉伸动态失效

在Abaqus/Explicit中，金属塑性材料模型可以与适用于真正动态情况的剪切和拉伸失效模型（["动态失效模型，" 第23.2.8节"](pt05ch23s02abm24.md)）结合使用；但是，通常首选上面讨论的逐步损伤和失效模型。

#### 剪切失效

剪切失效模型提供了一个简单的失效准则，适用于包括大多数金属在内的高应变率变形。它提供两种失效选择，包括由于结构撕裂或扯裂而从网格中删除单元。剪切失效准则基于等效塑性应变值，主要适用于高应变率、真正的动态问题。更多详情，请参见["动态失效模型，" 第23.2.8节"](pt05ch23s02abm24.md)。

### 塑性功产生的热量

Abaqus可选地允许塑性耗散导致材料加热。热量产生通常用于大量非弹性应变涉及的 bulk metal成形或高速制造过程的模拟，其中材料因变形而产生的加热由于材料属性的温度依赖性而成为重要效应。它仅适用于绝热热应力分析（["绝热分析，" 第6.5.4节"](pt03ch06s05at20.md)）、完全耦合温度-位移分析（["完全耦合热应力分析，" 第6.5.3节"](pt03ch06s05at19.md)）或完全耦合热-电-结构分析（["完全耦合热-电-结构分析，" 第6.7.4节"](pt03ch06s07at23.md)）。

### 初始条件

当我们需要研究已经经历了一些加工硬化的材料的行为时，可以提供初始等效塑性应变值来指定对应于加工硬化状态的屈服应力（见["Abaqus/Standard和Abaqus/Explicit中的初始条件，" 第34.2.1节"](pt07ch34s02aus116.md)）。

| **输入文件用法：** | ``` [*INITIAL CONDITIONS](../key/key-link.md#usb-kws-minitialcond), TYPE=HARDENING ``` |
| --- | --- |

| **Abaqus/CAE用法：** | 加载模块：****创建预定义场****：****步骤：**初始**，为****类别****选择****机械****，为****所选步骤的类型****选择****硬化**** |
| --- | --- |

#### Abaqus/Standard中的用户子程序规范

对于更复杂的情况，可以通过用户子程序[`HARDINI`](../sub/sub-link.md#sub-xsl-hardini)在Abaqus/Standard中定义初始条件。

| **输入文件用法：** | ``` [*INITIAL CONDITIONS](../key/key-link.md#usb-kws-minitialcond), TYPE=HARDENING, USER ``` |
| --- | --- |

| **Abaqus/CAE用法：** | 加载模块：****创建预定义场****：****步骤：**初始**，为****类别****选择****机械****，为****所选步骤的类型****选择****硬化****；****定义：**用户定义**** |
| --- | --- |

### 单元

经典金属塑性可用于任何包含力学行为的单元（具有位移自由度的单元）。

### 输出

除了Abaqus中可用的标准输出标识符（["Abaqus/Standard输出变量标识符，" 第4.2.1节"](pt02ch04s02abv01.md)和["Abaqus/Explicit输出变量标识符，" 第4.2.2节"](pt02ch04s02xbv01.md)），以下变量对经典金属塑性模型具有特殊含义：

| PEEQ | 等效塑性应变，ε¯~pl~ = ε¯~pl~0~ + Δε¯~pl~，其中ε¯~pl~0~是初始等效塑性应变（零或用户指定；见["初始条件"](pt05ch23s02abm17.md#usb-mat-cmetalplastic-initialcond)"）。 |
| --- | --- |

| YIELDS | 屈服应力，σ~0~。 |
