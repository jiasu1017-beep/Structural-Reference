# 2.1.4 从非活动会话保存模型数据







Abaqus/CAE 和 Abaqus/Viewer 包含一个非活动计时器。如果应用程序长时间处于非活动状态，许可证令牌将返回到服务器，以供其他用户使用。如果服务器连接丢失或无法获取新的许可证令牌，您的会话不会结束。相反，当没有可用许可证时，会出现一个对话框列出您的选项。对于 Abaqus/CAE 和 Abaqus/Viewer，您可以尝试重新获取许可证或退出应用程序。对于 Abaqus/CAE，您还可以选择保存当前模型数据库。保存模型允许您保留任何已完成的模型信息（如果尚未保存）；任何部分完成的信息（例如，在丢失许可证时处于活动状态的过程）不会被保存。保存模型数据库后，对话框中只剩下重新获取和退出选项。在 Abaqus/Viewer 中不提供保存选项，因为影响输出数据库的所有更改会在您进行更改时立即保存。

默认时间限制为 60 分钟。您可以使用 Abaqus 环境文件（`abaqus_v6.env`）中的 **cae_timeout** 环境变量更改时间限制。有关环境文件的更多信息，请参阅[「使用 Abaqus 环境文件，Abaqus Installation and Licensing Guide 第 4.1 节」](../sgb/sgb-link.md#sgb-chp-env-using)。
