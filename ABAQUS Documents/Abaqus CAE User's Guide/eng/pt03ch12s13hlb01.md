# 12.13.1 Creating homogeneous solid sections







Homogeneous solid sections are used to define the section properties of two-dimensional, three-dimensional, and axisymmetric solid regions. For more information, see ["Solid (continuum) elements," Section 28.1.1 of the Abaqus Analysis User's Guide](../usb/usb-link.md#usb-elm-esolidcont).

**To specify homogeneous solid section properties:**

1. From the main menu bar, select ****Section**![](../graphics/images/arrow.gif)**Create****. A **Create Section** dialog box appears. **Tip:**You can also click **Create** in the **Section Manager** or select the create section tool ![](../graphics/ico_section.png) in the Property module toolbox.
2. Enter a section name. For more information on naming objects, see ["Using basic dialog box components," Section 3.2.1](pt01ch03s02s01.md).
3. Select **Solid** as the section **Category** and **Homogeneous** as the section **Type**, and click **Continue**. The solid section editor appears.
4. Select a material for the solid section. If desired, click ![](../graphics/ico_material.png) to create a material; see ["Creating or editing a material," Section 12.7.1](pt03ch12s07hlb01.md), for more information.
5. Enter a value for the section **Plane stress/strain thickness**. If the section will be used with a two-dimensional region, you must specify the section thickness. Abaqus/CAE ignores the thickness information if it is not needed for the region type.
6. Click **OK** to save your changes and to close the solid section editor.
