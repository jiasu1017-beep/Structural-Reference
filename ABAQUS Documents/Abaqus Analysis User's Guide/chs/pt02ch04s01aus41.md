# 4.1.4 误差指示器输出





**产品：**Abaqus/Standard  Abaqus/CAE

**警告：**误差指示器输出变量是近似值，并不代表您的求解误差的准确或保守估计。如果您的网格粗糙，误差指示器质量可能特别差。随着网格的细化，误差指示器质量会提高；但是，您不应将这些变量解释为表示在进一步细化网格时求解变量的值。

### 概述

误差指示器输出变量：
- 表示求解量（基础解）中的离散误差，具有基础解的单位；
- 可以使用元素输出或接触输出选项请求，或作为自适应网格重划分规则的一部分；
- 可以通过基础解的形式进行归一化，以获得无量纲的（如百分比）误差指示器；
- 在某些情况下可能会显著增加分析求解时间；以及
- 在Abaqus/Standard中可用，但在Abaqus/Explicit中不可用。

### 求解精度

有限元分析对物理行为进行有用预测的能力取决于许多因素，包括：
- 几何、材料行为、载荷历史以及与描述所提出问题相关的各种其他建模方面的表示；
- 空间和时间离散化（网格细化和增量）；以及
- 收敛容差。

本节的主要焦点是空间离散误差。帮助理解和控制其他潜在误差源的讨论出现在["非线性问题的收敛准则，" 7.2.3节](pt03ch07s02aus51.md)、["瞬态问题中的时间积分精度，" 7.2.4节](pt03ch07s02aus52.md)、["评估超弹性和粘弹性材料行为，" Abaqus/CAE User's Guide的12.4.7节](../usi/usi-link.md#usi-prp-editor-evaluate)以及Abaqus文档的其他部分。您应该作为任何误差评估的一部分，对您的分析方法和假设进行详细研究。

#### 空间离散误差

模型域的有限元离散化产生了除 trivial 分析外所有分析的精确解的近似。为了帮助您理解有限元解中离散误差的范围和空间分布，Abaqus/Standard提供了一组误差指示器输出变量。理想情况下，误差指示器输出变量应通过其他技术（如网格细化研究）进行补充，以获得信心：离散误差不会显著降低有限元分析进行有用预测的能力。事实上，误差指示器可以通过Abaqus/CAE的自适应网格重划分功能帮助自动化网格细化研究；此功能使用误差指示器变量来确定在哪里细化或粗化网格（请参见["自适应网格重划分：概述，" 12.3.1节](pt04ch12s03abo15.md)）。

#### 误差指示器输出请求对求解时间的影响

Abaqus/Standard基于基础解的平滑和非平滑分布之间的差异确定误差指示器变量，使用Zienkiewicz和Zhu (1987)的补丁恢复技术等平滑技术。平滑计算偶尔会显著增加分析时间。如果您发现添加误差指示器输出请求会显著增加分析时间，减少此影响的策略包括降低输出频率以及将输出请求限制在特定感兴趣区域。大多数误差指示器变量的计算仅在将误差指示器变量写入输出数据库之前进行，因此降低输出频率将倾向于减少计算时间；但是对于元素能量密度误差指示器则不是这种情况，因为无论是否为给定增量输出此误差指示器，每个增量都会累积对此误差指示器的贡献。

#### 元素误差指示器变量输出请求范围的额外考虑

当您请求元素误差指示器输出时，该请求仅适用于支持误差指示器输出的元素。

用于计算元素误差指示器的补丁恢复技术假设解在指定的元素集上应该是连续的。Abaqus/Standard通过检查误差指示器域内的截面属性引用来确认您的误差指示器输出规范与此假设一致，并在提供的元素集中的元素引用不同截面定义时发出警告消息。如果截面属性相同，您可以安全地忽略此警告。

### 解释误差指示器输出

在解释误差指示器输出时，您应该记住，误差指示器是基础解局部误差的近似度量，其本身也受离散误差的影响。随着网格的细化，误差估计的准确性往往会提高。每个误差指示器变量具有与相应基础解变量相同的单位，这便于将误差大小的局部估计与基础解的局部估计进行比较。

#### 基础解及其相应误差指示器的感兴趣区域

将基础解变量和相应误差指示器变量的等值线图并排显示可以为求解精度提供有用的视角。例如，如果基础解以应力单位表示，则相应的误差指示器也以应力单位表示。[图4.1.4-1](pt02ch04s01aus41.md#aerror-hertz-cpress)显示了一个球压入刚性板的分析的CPRESS和CPRESSERI等值线图。这些图可以解释如下：
- 在活动接触区域中心附近，接触压力最大，接触压力解相当准确，因为该区域的误差指示器是基础解的一小部分。
- 在活动接触区域周边附近，接触压力解的局部变化最大（但接触压力明显小于最大值），接触压力解的准确性较低，因为该区域的误差指示器相对于基础解相当大。

如果在这种案例中主要感兴趣的是最大接触压力，则分析师可以判断网格细化的水平是否足够。如果活动接触区域明显小于[图4.1.4-1](pt02ch04s01aus41.md#aerror-hertz-cpress)所示，则需要局部网格细化来准确预测最大接触压力。

**图4.1.4-1** 可变形球与刚性板之间接触的CPRESS和CPRESSERI等值线图。

![](../graphics/acontact-hertz-cpresseri.png)

如果网格相对于局部解变化或所提出问题的精确解涉及应力奇异性，则误差指示器往往会对精确解的偏差给出粗略的、非保守的近似。以下是对误差指示器结果超过基础解结果约10%的定性解释通常是合适的：
- "此区域存在显著潜在求解不准确的可能性。"
- "网格可能太粗糙，无法对此区域的求解误差给出良好估计。"
- "也许在这个角落存在应力奇异性。"

### 限制

仅以下元素类型支持误差指示器计算：
- 平面连续体三角形和四边形
- 壳三角形和四边形
- 四面体
- 六面体

不支持具有变量节点的元素。

在以下情况下不支持误差指示器输出：
- 导入分析
- 重启分析
- 后处理输出分析
- 映射求解分析
- 对称模型生成分析
